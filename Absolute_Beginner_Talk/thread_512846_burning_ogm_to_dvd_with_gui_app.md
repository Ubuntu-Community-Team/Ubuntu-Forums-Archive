### Post by arpan on 2007-07-30
As far as I understand you want to burn an .ogm file on a video dvd.  If that is the case then you'll require to extract regular audio/video format streams from that .ogm file. Then you can burn those audio/video file to video dvd with GUI based tools like GnomeBaker or K3b.

I don't know any GUI based tool for extracting an .ogm file but following links may help you.

[http://filext.com/file-extension/OGM]("http://filext.com/file-extension/OGM")

[http://www.mplayerhq.hu/DOCS/HTML/en/index.html]("http://www.mplayerhq.hu/DOCS/HTML/en/index.html")

Mplayer has GUI as well.

Hope this will help.
