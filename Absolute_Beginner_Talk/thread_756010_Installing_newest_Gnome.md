### Post by Inxsible on 2008-04-15
Hardy Heron uses Gnome 2.22 by default. Hardy releases on the 24th of April. So in a little over a week, you would not only have Gnome 2.22 but also a newer OS in Hardy Heron.

If you still insist on using Gnome 2.22 in Gutsy, you will have to install from source which is not preferable even by Gnome developers (as in they recommend you wait for the official release by your distro)

[http://library.gnome.org/misc/release-notes/2.22/#rninstallation](http://library.gnome.org/misc/release-notes/2.22/#rninstallation)
