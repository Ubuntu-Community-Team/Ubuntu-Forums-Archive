### Post by p_quarles on 2007-06-19
I ran kubuntu 7.04 on an old machine for a couple weeks (it then got turned into a file server). In any case, I thought it worked fine. This was on a desktop, of course, but as far as I know, kubuntu uses the same power management scheme as the Gnome setup. As for roaming wireless, run the live CD and see if you can get it to work. You won't be able to use NDISwrapper, but if there's built-in support for your wireless card you'll be able to give it a full test drive. 

Of course, you could also just install Ubuntu, get everything working, and then install the KDE window manager.
