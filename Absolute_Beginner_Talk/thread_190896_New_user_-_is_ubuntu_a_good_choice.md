### Post by wastrel on 2006-06-06
You should boot the LiveCD and test it out a bit - you don't need to install to get a feel for things.

This will also help you determine whether all your hardware is supported before you install.

Ubuntu is a great Linux for beginners because of the automatic hardware detection, the excellent package management, and the community which is one of the most helpful and friendly I've found in more than 10 years of using Linux as my primary OS.

My one piece of advice, definitely get to know Synaptic, the package manager.  There's thousands of programs available pre-packaged to work with Ubuntu.  Following the Windows mindset of going to Tucows or somewhere and downloading an EXE can get you into trouble.  If Synaptic contains the program you need, don't download an installer from the web.

Good luck!
