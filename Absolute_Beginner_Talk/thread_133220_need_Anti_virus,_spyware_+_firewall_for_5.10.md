### Post by SickTwist on 2006-02-19
One of the best features of Linux is that you don't have to worry about those nasties. They exist primarily (if not soley) in the Windows world.

A firewall is never a bad idea but there are no network services running by default in Ubuntu. So adding a firewall to a default Ubuntu installation is kind of like putting locks on a house with no windows or doors.

Clamav scans for virii which are harmful to Windows. As far as I know, there are no Linux virii in the wild.
