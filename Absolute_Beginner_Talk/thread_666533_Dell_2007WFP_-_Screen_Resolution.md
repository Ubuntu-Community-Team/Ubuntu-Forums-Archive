### Post by useResa on 2008-01-14
First of all ... if you have a working environment that you would like to be able to revert back to, open a terminal and change to the /etc/X11 directory ```
cd /etc/X11
```
Next make a copy of your working configuration file
```
sudo cp xorg.conf xorg.conf.stable_backup
```
Of course you can give the backup file any name you prefer   ;)

Should you run into issues, restart Ubuntu in the recovery mode from the GRUB menu, login and then go to the /etc/X11 directory (as indicated above) and issue the command ```
cp xorg.conf.stable_backup xorg.conf
``` (in revovery mode you are logged in as root so need for sudo). Next continue the startup process by pressing <Ctrl>-<D>.

With regard to the resolution issue, you may find some helpful information in [this post]("http://ubuntuforums.org/showpost.php?p=4088859&postcount=29").

Good luck.
