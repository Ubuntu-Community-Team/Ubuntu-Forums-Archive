### Post by Rocket2DMn on 2007-07-27
There are open source alternatives to basically all programs you can get in windows.  Just browse the forums and you will find all sorts of suggestions.  If you want to escape the "withdrawals", you first start by breaking the habit.  But, like any addiction, it can be best to ease off of it - keep windows around for awhile, but with time you will realize you rarely use it.
As far as the visuals, there are many, many alteratives to using the default Human theme in Ubuntu.  There are different desktop managers like KDE and XFCE (basically, Kubuntu and Xubuntu), and plenty of skins to go on top of each of them.  These are referred to as Themes, and if you decide to install something like Compiz-Fusion (formerly Beryl), there are themes built in.  You can download your own of course.
Welcome to Ubuntu!
