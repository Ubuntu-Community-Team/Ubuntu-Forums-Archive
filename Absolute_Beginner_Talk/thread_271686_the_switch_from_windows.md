### Post by hyper_ch on 2006-10-05
That depends...

I'd say the hardest issue with a switch is the hardware and games...

If your hardware is supported by Ubuntu (or other Linux Distros) and you don't game much then the switch is (almost) painless... for certain software that you need and can't live without you can install VmWare Server (free as free beer) and install a windows there... it will be notably slower if you only have one cpu... if you run dual core it should not matter that much...

That way I still run photoshop and some other Windows stuff :)

P.S.: If you have questions just as here or you can go into the #ubuntu channel on irc.freenode.org --> You should get help in both places quite quickly :)
