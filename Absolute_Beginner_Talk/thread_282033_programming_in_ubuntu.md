### Post by jordanmthomas on 2006-10-22
to compile your c and c++ you will need to install the compiler
```
sudo aptitude install build-essential
```
will install the compiler gcc, which is what you'll use.

The actual code will be different too, as you are calling different OS stuff.  Just look up some stuff about c in unix and you should find plenty of helpful information.  Personally, I don't know c or c++ so I can't tell you all the differences.
