### Post by Zimmer on 2006-04-07
Has the Compaq got a Network port or expansion card?
Bt do a service which incudes a 2 port router, my daughter uses it and she has not reported any particular problems.
The good advice I took 3 years ago was to get an ADSL router/modem (in my case a Netgear DG814...the modern equivalent is a DG834 ) and NOT use a USB DSL modem.
I have been with my ISP (Nildram, now owned by PIPEX ) for those 3 years. They host the Jolt game servers, I believe. I have also set up a couple of friends with these routers without problems, one on Nildram one on Wanadoo.
The router has 4 ports, so you can immediately share the internet between 4 computers and set up a home network if you wish. The Netgear virtually sets itself up, auto detects most settings all you need is your ISP logon details and password and it is pretty secure. It acts as a hardware firewall.
I had no problems connecting to the net with Ubuntu when I installed.

### Post by damianubuntu on 2006-04-08
I've been using BT Broadband on Ubuntu for the last few months, BT supply a Voyager 105 USB modem which you can get drivers and dialer for (from sourceforge.net, I think - if you can't find it let me know)  Now I've got a Voyager 2500 router which works fine, either with USB or better through ethernet.  It does wireless too, which I'm just about to start playing with......
My impression of BT is that its not the cheapest, nor do they offer much help with linux (though some technical help guys are sympathetic), but the level of service provision is good, reliable, and almost always at full speed.  For what its worth the Voyager 2500 is a voice router, so that you can put all your landline calls through it, and over the broadband, for £7 a month....
