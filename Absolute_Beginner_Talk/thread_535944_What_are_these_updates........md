### Post by mlentink on 2007-08-27
Software is in constant development, and so is Ubuntu as well as all software that come with the distribution or which you install form the repositories. When a new release of any of your installed packages is available, update-manager will notify you of the availability of these updates. It is completely up to you whether you want to install them. As the previous poster said, if you want to wait until a next major release that&#347; fine.
