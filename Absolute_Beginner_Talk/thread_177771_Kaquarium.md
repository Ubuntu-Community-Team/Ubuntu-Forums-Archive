### Post by Sarisar on 2006-05-16
Yeah this has to be the most annoying thing for me - you install the program and it doesn't make any menu items.

I have found, however, that going into Synaptic you can right click on the package and get properties which gives you a list of installed files.  You can then try them in a console window to work out what to run to get the program working.... well most of the time.  Then you can make a menu item for it.

Oh and don't forget it's case sensitive so Kaquarium and kaquarium are two different commands...

### Post by FryerFox on 2006-05-16
I just searched for it, and it seems that Kaquarium is a KDE panel applet. You probably have to be running KDE to use it...
