### Post by Pragmatist on 2006-03-31
I would try this first:

[http://wiki.archlinux.org/index.php/Gnome_Custom_Menu_Icon]("http://wiki.archlinux.org/index.php/Gnome_Custom_Menu_Icon")

You can find some foot images by doing this:

type this in a terminal and wait for it to finish.
```
sudo updatedb
``` 

Then this:
```
locate foot |grep png
``` 
If that doesn't work, try this method:
[http://www.linuxquestions.org/questions/showthread.php?threadid=295217]("http://www.linuxquestions.org/questions/showthread.php?threadid=295217")

And if that doesn't work, try this:
[http://www.gnome.org/learn/]("http://www.gnome.org/learn/")
