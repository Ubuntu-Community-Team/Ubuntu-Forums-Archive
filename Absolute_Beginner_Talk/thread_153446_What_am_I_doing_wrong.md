### Post by jimrz on 2006-03-31
Welcome. No apologies needed...this is what these forums are all about.

Did you use the "burn as an image" option? If not, you may have just copied the .iso to your burned cd which will not work. To check open your burned cd in windows, you should see many files listed if done correctly. If you see only the one .iso file then you will need to go back and burn the file "as an image" (or words to that effect depending upon what software you are using.
