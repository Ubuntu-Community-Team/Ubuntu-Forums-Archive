### Post by lordlollo on 2006-07-10
The alternate install cd is for you if you have particular issues (low memory system, raid devices, ...) since it provides you with a textual tool for the installation with some features the graphical tool does not have. Anyway, when you finished with the alternate cd, you'll have a desktop environment as you performed an installation with the "regular" cd. The dualism is between a desktop or a server installation.

### Post by Crosbie on 2006-07-10
I believe it also allows a little more control over, eg, where Grub goes.  If you're new, want a desktop machine for everyday use, and have a normal, recent sort of system (eg, Windows Xp on one hard drive), you'll most likely want to use Desktop.

It's probably worth having as a Live CD anyway - so you can do any repairs by just running off the CD alone.  Live CDs are cool. :)
