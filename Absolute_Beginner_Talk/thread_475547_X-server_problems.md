### Post by w4ett on 2007-06-16
Here's sections from my xorg.conf...I had to specify the amount of memory the card has, and also to force it into pci mode...there is a bug in the OSS Driver which makes it actually run slower in agp mode.

Section "Device"

Identifier "ATI Radeon 9200SE"
Driver "ati"
BusID "PCI:1:0:0"
VideoRam 65536
Option "BusType" "PCI"

I believe it also helps to specify the actual card you have in the device section...seems to anyway, and has been recommended to me in the past.

### Post by w4ett on 2007-06-16
The easiest method to get back the original xorg.conf that I know of is to run the live cd again, copy the file to you HD or usb pen drive and then reboot back to your regular installation and copy and replace it via 

sudo gedit /etc/X11/xorg.conf

or  

sudo nano /etc/X11/xorg.conf

just copy and paste.  this will give you the Ubuntu default values for your system.
