### Post by poptones on 2005-12-10
You should always make an effort to install your home directory into its own partition so you can reinstall or upgrade without overwriting everything - so, you might plan ahead and install XP on, say, a 10GB partition and then when you remove it you can move your /home folder there and have a more maintainable system.
