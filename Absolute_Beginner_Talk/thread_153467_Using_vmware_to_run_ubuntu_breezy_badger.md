### Post by Qrk on 2006-04-01
If you want to try out breezy on your vmware, try this [http://www.vmware.com/vmtn/appliances/ubuntu.html](http://www.vmware.com/vmtn/appliances/ubuntu.html)

If you want to try out the liveCD, (ie by rebooting into Linux, not vmware) you need to burn the live cd iso file from this directory onto a cd, then boot from that disk.
[http://www.ubuntu.com/download](http://www.ubuntu.com/download)

You can burn the cd using any tool that allows you to burn iso's to disk (nero, [http://isorecorder.alexfeinman.com/isorecorder.htm](http://isorecorder.alexfeinman.com/isorecorder.htm) ) and configure your BIOS to boot from the cd drive. 

I'm not quite sure from your question which it is you want to do.
