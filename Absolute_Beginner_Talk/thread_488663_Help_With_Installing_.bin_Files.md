### Post by chamberlain2006 on 2007-06-30
Generally, you can run a .bin file like this:

```

chmod +x *bin-file.bin*
sudo ./*bin-file.bin*

```

Also, though, what is the program that you're trying to install, cause you may be able to install it from the Ubuntu repositories.
