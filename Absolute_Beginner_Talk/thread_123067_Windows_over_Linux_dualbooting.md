### Post by mostwanted on 2006-01-29
Install Windows, make room for an Ubuntu partition.

Install Ubuntu on the left-over space. During install when GRUB looks for alternative operating systems and locates Windows, select yes when it asks you to include it in the list.

Doing it the other way around is much more troublesome, why don't you just backup your files and reinstall?
