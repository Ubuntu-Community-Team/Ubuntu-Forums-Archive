### Post by SilentCacophony on 2005-10-12
Most people seem to find it easier to leave un unpartitioned space on thier drive, and let ubuntu install into that, but you can also set up the partitions first, and elect to manually edit the partitions.

These two links should explain it well:

[https://wiki.ubuntu.com/WindowsDualBootHowTo](https://wiki.ubuntu.com/WindowsDualBootHowTo)
[http://users.bigpond.net.au/hermanzone/](http://users.bigpond.net.au/hermanzone/)

Just be very careful when selecting to apply any changes during install.
