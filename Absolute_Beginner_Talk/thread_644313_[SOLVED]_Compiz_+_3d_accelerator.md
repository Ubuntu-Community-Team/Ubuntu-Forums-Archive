### Post by Nano Geek on 2007-12-18
Hello there,

Compiz is actually already installed by default. 
Yes, you do need the graphics-card driver provided by Nvidia, but it is very simple to install.

Just go to **Applications=>Add/Remove...** and search for **NVidia binary X.Org driver**. Install that, reboot, and Compiz should work for you. 

If you want to play with the advanced Compiz settings, then you'll need to install an extra program.```
sudo apt-get install compizconfig-settings-manager
```If you want them, you can also install the extra plugins for Compiz.```
sudo apt-get install compiz-fusion-plugins-extra
```

I hope you have a lot of fun using Ubuntu. :)
