### Post by mcduck on 2007-06-13
I'm studying multimedia design, and I'm mostly using Linux to do my work. Most of the time Gimp can replace Photoshop easily, and I actually like Inkscape a lot more than Illustrator so there's no problem. Scribus is very good replacement for Indesign.

I don't need any advanced features in PDF files so I usually just do them with either OpenOffice or Scribus, but there's some PDF editor available here: [http://www.getdeb.net/app.php?name=PDF+Editor]("http://www.getdeb.net/app.php?name=PDF+Editor").

Anyway, at least Photoshop, Illustrator and Flash can be run with Wine in Linux. Aybe others work too. You may need to install them on some Windows machine and then copy the directories and registry settings to Wine as I don't think their installer programs work correctly. I have only installed Flash this way, as it's the only program that has no good replacement in Linux.
