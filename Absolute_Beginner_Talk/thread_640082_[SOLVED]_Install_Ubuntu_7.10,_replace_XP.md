### Post by Dr Small on 2007-12-13
Just install Ubuntu over the entire drive. It won't make any extra partitions and setup a dualboot that way. It will erase the current drive, and install Ubuntu as the complete base system.

### Post by siciliancasanova on 2007-12-13
Yes you can just install right over Windows when you install Ubuntu.  It will give you that option.

On the other hand, I would still partition if I were you.  At least a separate /home folder or any other folder so when the next version of Ubuntu comes out you could if you wanted just install over your previous version and your settings and files will all remain intact.
