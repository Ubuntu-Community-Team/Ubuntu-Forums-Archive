Well no its not a bad thing.  But it wont help you at all unless you have Windows machines, as linux anti-virus checkers only look for windows viruses - because there arent any known Linux viruses which are still around "in the wild".

Even if one appears, it wont help to scan for it until the anti-virus makers have found it, understood it, and written a fix for it...  So you best defence is to simply keep an ear out in the community so that if any threat emerges you can know what to do about it.
