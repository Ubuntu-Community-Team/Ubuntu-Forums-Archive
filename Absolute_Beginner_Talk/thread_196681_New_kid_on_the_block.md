### Post by PriceChild on 2006-06-14
The best way to go is shove the Live cd in, or even install it properly if you've got the disk space for it, and then mess around!
 
Don't use automatix to get mp3 codecs for example... instead read the ubuntu wiki and install it all yourself! Read the excellent How-tos on this forum for installing your graphics drivers!
 
If you want your computer to do something, then just have a bash, and when that fails search the forums.
 
You'll never EVER be able to do anything much by reading documentation the net. Just try it yourself.
 
P.S. unless you have any indepth problems with specific software... then this forum should be all you ever need.
