### Post by FyreBrand on 2007-02-24
You can install kde-core and then you'll have to install the other components that you want or need to make your laptop work.  If you still want a text boot don't install kdm.

Check out this thread here: [**Ode to kde-core... a faster kde**]("http://www.ubuntuforums.org/showthread.php?t=277090") This article discusses how to install it and applications people mention that you might need (like ACPI) that won't be installed with a core install.  They did report that it is significantly faster.

Good luck.
