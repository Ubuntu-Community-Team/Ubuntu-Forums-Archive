### Post by llamakc on 2007-12-28
Go into the menu

SYSTEM > PREFERENCES > EMERALD THEME MANAGER

and you can select a different Emerald theme, or tweak the one you're using. Other themes can be found at various places online like [http://gnome-look.org]("http://art.gnome.org") and [http://themes.beryl-project.org](http://themes.beryl-project.org)
