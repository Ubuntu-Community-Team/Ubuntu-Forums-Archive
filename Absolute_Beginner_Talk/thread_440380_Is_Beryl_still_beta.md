### Post by Campingman on 2007-05-11
Beryl is in Synaptic and can be easily installed through system>administration>synaptic package manager and then search for beryl.  It does not say anything in the description about it still being beta.  
Compiz is installed by default in feisty and can be enabled by system>preferences>desktops effects.

Some people are experiencing problems with the desktop and some multimedia functions after installing and enabling either beryl or desktop effects though.
