### Post by alan cottrill on 2006-03-19
I just wanted to post and thank everyone who contributes to this forum.  I've been wanting to try linux for a while now,  but with all the distro options it was hard to make a decision.  when I found Ubuntu it seemed like the way to go.  anyway,  I started reading up on it friday,  and downloaded the install iso on my XP box while I assembled my linux box.  by the time I was done burning the install cd I was ready to drop it in my new machine.  I was up till 2:30 that night and was able to accomplish this...

Install Ubuntu
Install Automatix and all the goodies that looked necessary and useful
attached my linux computer to my windows network
setup network printer
download and install nerolinux (very cool that I was able to unlock the demo with my windows version serial)
download and install cedega 
install quake III and do some serious fraging

I should note that none of this was "easy"  and it didn't just all fall right into place.  I had to spend some time reading posts on this forum and did a lot of "copy and paste" work.  but with the help of this board it wasn't that hard.  

so anyway...Thanks again to everyone who contributes.  it really makes it a lot more fun when you have so many good people behind you.  I hope to be able to contribute soon!
