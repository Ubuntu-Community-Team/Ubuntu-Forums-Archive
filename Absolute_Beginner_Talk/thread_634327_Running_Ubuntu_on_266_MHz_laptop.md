### Post by alpage2 on 2007-12-07
Use the 'alternate' install CD. I have found that 128 MB of RAM is not quite enough for the normal install CD.

With limited resources, try to keep to lightweight versions of applications. For example, xubuntu will install Abiword word processor and the Gnumeric spreadsheets, as lightweight alternatives to open office.

Avoid too many browser plug-ins. You may find Epiphany a little lighter than firefox, and for an extra-lightweight browser, try dillo - very fast, but chokes on highly scripted sites.

For more ideas on lightweight apps, take a look at what other lightweight linuxes are using, like fluxbuntu.org.

Alan

### Post by KCPokes on 2007-12-07
There are a bunch of "lightweight" threads around.  I'd recommend looking into a different distro, because in my mind Xubuntu is still pretty heavy for such old hardware.  I have a Toshiba Tecra 8000, which has a 266 and 128MB.  I've ran Ubuntu and Xubuntu, with both being too heavy.  I've tried everything from Zenwalk, Mepis, and Puppy (Puppy doesn't like the video cards in the old Toshibas) before I finally decided on Arch.  Arch runs very well on the machine, but does take some patience to get it where you want to be.
