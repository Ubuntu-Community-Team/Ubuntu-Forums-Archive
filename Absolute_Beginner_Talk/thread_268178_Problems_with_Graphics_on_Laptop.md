### Post by ewl1217 on 2006-09-29
Use the following command to reconfigure xorg:
```
sudo dpkg-reconfigure xserver-xorg
```
This will guide you through creating your xorg.conf. I'm pertty sure it makes a back-up automatically, but you might want to make one manually just in case. Make sure you choose the correct graphics driver, resolutions, and other options. After that, logout and press CTRL+ALT+BACKSPACE to restart the X server. If all goes well, then you should get the resolution you want.
