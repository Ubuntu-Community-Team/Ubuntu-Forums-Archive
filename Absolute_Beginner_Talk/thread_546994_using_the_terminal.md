### Post by parvinder on 2007-09-09
A good start will be to learn the "vi" editor. It is very powerful editor and you may have seen  Unix/Solaris/Linux administrators to open file by typing "vi filename" and edit it....

Go to [http://www.unix-manuals.com/](http://www.unix-manuals.com/) and learn basics from Vi Manual... and basic unix commands like

cp - copy
mv - move
rm - remove files or directories
pwd - present working directory
cd - change directory
etc........

if you need help on any command open the help manual by typing "man <command-name>" e.g. "man rm"

Good Luck!
