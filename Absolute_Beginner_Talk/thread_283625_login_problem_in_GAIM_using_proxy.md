### Post by kkellner on 2006-10-24
to get gaim working behind my proxy, I had to set the network proxy in ubuntu: systems-preferences-network proxy, set it in firefox, and then set gaim to use "global proxy settings".  however it also seems to work when I have all those same settings, but "no proxy" set for gaim.  Basically I would just try setting every proxy you can find and then try several settings in the account preferences in gaim to try to get it to work.
