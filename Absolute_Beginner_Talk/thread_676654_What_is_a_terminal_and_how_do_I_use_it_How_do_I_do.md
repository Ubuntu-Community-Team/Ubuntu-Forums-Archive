### Post by tdashroy on 2008-01-24
I am new to Ubuntu, but a terminal is under Applications>Accessories>Terminal. The terminal is like a command line, but as I said I am new as well so I don't know that much about it.

To run things, you usually have to do it in a terminal, and you just type in the file name.  If it doesn't let you, you might try typing the following:

chmod +x whateveryourfilenameis

but, in order to run it you will have to be in the directory to run it.  You can change your directory by typing:

cd yourdirectoryname

to see all the directories you can change to type:

ls -a

As for the other stuff, I don't know.
