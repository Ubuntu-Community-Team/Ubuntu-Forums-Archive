### Post by Quillz on 2007-02-27
Go into Nautilus (the file browser installed by default on Ubuntu) and then go to File -> Connect to Server. You will get a pop-up with several different type of FTP connections. Pick the one you need, fill in the appropriate settings and then click OK. Ubuntu will save the FTP connection as a shortcut in the left sidebar of Nautilus, so you can quickly visit it again in the future.

There are also dedicated FTP clients for Ubuntu, such as gFTP.
