### Post by Bloch on 2007-05-27
Go to Tools > Customize and make sure you have the standard toolbars etc.

The icons on openoffice disappeared on me once when I changed the theme - go to the ubuntu System menu
System > preferences > theme 
and use the default theme.

Yet you say that the menu titles are missing too, so this might not be the problem.

You might have to try removing (and remove the configuration files) and reinstalling
