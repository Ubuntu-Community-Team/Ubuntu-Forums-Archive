### Post by Blairboy on 2006-09-06
Well, you could just use ekiga (formerly gnomemeeting) which comes with ubuntu.  Go to applications->internet->ekiga.  I use ekiga and gizmo project ([www.gizmoproject.com)](www.gizmoproject.com)).  Gizmo actually provides a .deb package that you can install by double clicking the file they supply.  The file you linked to was a tar.gz which is not executable.  It's source code that you have to compile.  I'd just use ekiga or gizmo.
