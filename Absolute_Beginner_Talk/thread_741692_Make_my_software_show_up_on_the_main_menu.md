### Post by wormser on 2008-04-01
Avast is a weird one for the menu.  This [thread]("http://ubuntuforums.org/showthread.php?t=530944&highlight=avast+menu") has the solution.

To run a program you can type in the full path like /bin/ls or if you are in that directory ./ls.  Sometimes you can just type in the command and it works.  That is because it is in the PATH.  Google linux path.
