### Post by elcu on 2006-12-18
If [this]("http://packages.ubuntu.com/edgy/gnome/gnome-pilot") is the package you are talking about, it is a Gnome applet.  Applets are the utilities you see on the panels.  For example, the battery monitor or the clock.  I think the trio of menus on top are actually an applet too.

Anyway, if the install went successfully, you can add the Gnome Pilot applet to the top or bottom panel by right-clicking on empty space on the panel you wish to add it to and then select "Add to Panel".  

Hunt through the available applets, or maybe type in 'pilot' into the search feature and hopefully the relevant applet will show up.

I'm not on my Ubuntu box atm so can't double check if my instructions are right, but let us know if that works.
