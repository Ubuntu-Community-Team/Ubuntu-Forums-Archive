### Post by AtinLango on 2006-03-12
I was most apprehensive about installing Ubuntu to share with my XP. I had read some really bad stories. But I think many such stories are unfounded. I have had absolutely incident-free installation (and re-installation) of Ubuntu. I didnt really need to re-install if not for adventuring!

I would advise, though, that you first repartition your hard drive (using Gparted of QTParted - I prefer GParted). If you have Partition Manager, better. Remember first backup your data, and defragment the drive. This is what I did so that by the time you install Ubuntu, you have a free partition to itself. Windows will be intact.
