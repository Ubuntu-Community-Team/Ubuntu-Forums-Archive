I also use Emerald because the window peel-back function did not work for me using GTK.  Once Emerald is installed, you can use the Emerald Theme Manager on the System -->Preferences menu.  There are a bunch of Emerald themes out there.  The themes listed under Beryl at [http://www.gnome-look.org/](http://www.gnome-look.org/) are Emerald themes and will work with Emerald and Compiz-Fusion.

It appears after a few clicks that many of the Compiz themes at gnome-look are emerald themes (the files end in .emerald), so they should be OK.
