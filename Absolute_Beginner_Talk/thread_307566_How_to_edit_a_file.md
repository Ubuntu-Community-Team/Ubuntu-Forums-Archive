### Post by taurus on 2006-11-26
You cannot edit a file if you cannot write to it but here it is anyway!!!  (Actually, you can edit but you cannot save the chances...)

```
nano filename
-or-
gedit filename
```

### Post by xpod on 2006-11-26
Hit "alt-f2" and type "gksudo gedit" into the box which will bring you up the text editor with the privalages you require.
Then just navigate to the file you want to edit..
