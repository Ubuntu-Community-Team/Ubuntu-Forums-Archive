### Post by sankarv on 2007-06-01
All u need to do is just open the console/ terminal and 

Set the permissions for execution for google earth and then run it.


> 
chmod 744 GoogleEarthLinux.bin

then


> ./GoogleEarthLinux.bin


(Give the full path of the file if u r not there in the directory containing the GoogleEarthLinux.bin file)



It will just ask you for installation , follow the instructions and then install it.


Also check whether u need to be root user for installing it. im not sure of that.


U can also see the link


Its given here
