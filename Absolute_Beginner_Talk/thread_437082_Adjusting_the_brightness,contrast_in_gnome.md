### Post by TomFumb on 2007-05-14
Are you using a laptop? If yes, (and possibly if no) go to System / Preferences / Screensaver, and click on the 'Power Management' tab, there you can change the brightness, not sure about the contrast though. Also, I'd have thought you should have some function keys on your keyboard, which Ubuntu is generally quite good at recognising.
