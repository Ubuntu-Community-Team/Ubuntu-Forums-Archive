### Post by ramjet_1953 on 2007-06-14
What speed did you burn the iso at?

Anything faster than 4 X will almost always cause problems.

Did you burn it as an iso image, or just copy the file to a CD?

For any iso, good quality media is essential.

To save the problem of burning coasters, I always use a high quality CDRW disk. 

That way, if something is not right, I can erase and re-burn.
