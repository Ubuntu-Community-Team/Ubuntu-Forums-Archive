### Post by Joe_lurker on 2005-11-02
I don't think you would want to automatically log ALL network traffic.  You would surly fill up your disk space in a hurry.  With that said there is a program called Ethereal.  You can install the program with the System | Administration | Add Applications program.  Scroll to Internet then More programs...  Place a check mark next to Ethereal and Ethereal (as root).  That will install its icons to Applications | Internet.

Play around with it a bit and post if you need more help.
