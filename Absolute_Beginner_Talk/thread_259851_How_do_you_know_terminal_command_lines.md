### Post by jordanmthomas on 2006-09-18
The best way to learn is to just use your computer.  Eventually, you will want to do something.  You may or may not be able to figure it out yourself.  You'll look it up and find an answer.  So, next time you will no just what to do.  Also, what you learned in that experience will help you to figure things out on your own later because you will know how things typically work.

As for knowing the options for a particular command, try
```
man command
```
or
```
command --info
```
or
```
command --help
```

Just read through even some of the basic commands like cp and ls
Everything is surprisingly well documented.

Also, I read this when I first started out.  It was a little over my head at first, but I found it to very thorough and I still go back to it for reference all the time.

[http://rute.2038bug.com/rute.html.gz](http://rute.2038bug.com/rute.html.gz)
