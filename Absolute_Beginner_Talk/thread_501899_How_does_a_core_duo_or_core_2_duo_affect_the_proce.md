### Post by gn2 on 2007-07-16
> **purplearcanist said:**
> What is the difference between a core duo and core 2 duo?

A Core Duo is primarily a laptop CPU, there are Core 2 Duos that are laptop variants as well. The Core 2 Duo was developed from the Core Duo laptop CPU, which was introduced because the older Pentium 4 based dual-core CPU's were extremely hot running and inefficient.
.
To confuse matters Intel have introduced some low-end E21xx dual-core CPU's and given them the model name "Pentium" they bear no relation whatever to a Pentium 4 and are actually Core 2 Duos with some cache disabled.
