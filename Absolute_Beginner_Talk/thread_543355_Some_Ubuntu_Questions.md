### Post by swisscow on 2007-09-05
Regarding keeping windows and then installing Ubuntu, lots of instructions out there - here's a link to one (NB I haven't tried this)

[HTML]http://www.howtoforge.com/dual_boot_windows_xp_vista_ubuntu_feisty?[/HTML]

Beryl is now merged with compiz to become the compiz fusion project - this is where the work is being done but it is still early days and can screw up your system (though a lot of people find it solid). However unless you've used linux before I think this is a case of running before you can walk. Get ubuntu running for a few weeks and then have a bash at installing Compiz Fusion - can be easy or a nightmare! Good tutorial though is:

[HTML]http://ubuntuforums.org/showthread.php?t=536149&highlight=compiz[/HTML]

As regards your hardware, lots of searching on the forums and with google can help sort things out. Probably 95% of stuff works fine, it's the other 5% that can be a pain - usually things like webcams, card readers, function keys. I'd say try the livecd and see what works - if it's enough then go for it!

### Post by anewguy on 2007-09-05
Welcome!   The install procedure from the LIveCD will allow you to shrink the size of your existing partition and thereby make room for the Linux partitions.  Before you do this, be sure to defrag the disk in Windows.  Also, I'm no expert, but I have always found it far easier to setup a dual boot system if you install Windows first, then Ubuntu.  That way the boot loader should automatically find Windows and give you an option for it on your boot menu.

Please be sure to post back with any concerns/questions!:)
