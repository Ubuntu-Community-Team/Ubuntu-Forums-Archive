### Post by lengau on 2008-01-15
Don't uninstall. It won't help, and it's not what you need to do.

At the command line, simply run rm -r ~/.kde4 (WARNING! THIS WILL REMOVE ALL OF YOUR KDE4 SETTINGS), or delete the .kde4 directory in your home directory from a file browser (you'll need to set it to display hidden files).

This sounds like a bug as, IIRC, it's supposed to always keep at least one desktop there for you.
