### Post by toidi on 2007-08-09
> **Ek0nomik said:**
> Absolutely not.

Just as ekonomik says. .  There are two types of software in the world, gnu and proprietary.

Linux prides itself on using gnu (public free licensing).

However there are some companies who write 'proprietary' software for linux and distribute it freely.

The big difference between them is, you can download gnu software and do what you like with it as the source code is available on the net (most of the time).

Proprietary software means that the company who makes the software does not usually allow you to get the source code, or modify it.  They will let you use it for free though, with Flash player being a prime example.

Other proprietary (or copyrighted) software requires you to purchase a licence to use it (photoshop for example).

That is the way I see it anyway.
