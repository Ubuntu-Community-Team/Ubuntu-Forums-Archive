### Post by v8YKxgHe on 2006-12-24
> Is Ubuntu right for me?
I don't know, is it? Try it and see, only you can say if Ubuntu is right for you.

Installing programs is different across different Linux distros, for example in Ubuntu you'd do "sudo apt-get install program" or in Fedora, afaik you'd do "yum install program" so things you learn from one distro may not always apply to the next distro.

Yes 6.06 is the LTS release, Long term support, and is very stable whereas Edgy (6.10) is not as stable, though still usable.

To dual boot use Grub but you must have installed Windows first then Ubuntu, other wise it can get messy :P

### Post by pseudonym on 2006-12-24
Hi,

if you want speed and minimalism, combined with new-user useability, I'd opt for Xubuntu. This is Ubuntu based on the [Xfce desktop environment]("http://www.xfce.org"). They say it's designed for use on older hardware, which really means it flies like the wind on newer machines. I know it does on mine.

I've tried Ubuntu with Gnome DE on my new machine and, while it's eminently useable of course, I find Xubuntu even more responsive. And when you're used to that extra responsiveness it's hard to give it up. Plus you can install any of the Gnome or KDE apps in the repositories if you wnat them, that way you don't miss out on anything.

All the best with it, anyway. :)
