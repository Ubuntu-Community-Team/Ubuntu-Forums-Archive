### Post by atria on 2007-07-07
Assuming that you're using Ubuntu, click Applications at the top right corner, Accessories and finally Terminal.

In the terminal window, use commands like cd and ls to enter the ies4linux folder. For example if the ies4linux folder is in your /home directory;

```

cd /home
ls -l /home
cd directory_to_ies4linux_returned_by_the_above_ls_command
./ies4linux

```
