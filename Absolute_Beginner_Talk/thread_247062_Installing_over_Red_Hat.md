### Post by tsiMental on 2006-08-30
There is no other way than to download either of the ubuntu/kubuntu/xubuntu install cds and install it. All three ubuntu versions installs directly from an icon on the desktop on the live-cd, which lets you to see what it looks like and try the most basic functions before you install it.

There is usually no point in keeping other files from a previous linux install than the files in your home folder.

/home/username

You can burn these to a cd or dvd or copy them to an external harddrive or a friends computer while you reinstall the Operating System (OS).
