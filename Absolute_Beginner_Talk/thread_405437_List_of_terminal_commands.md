### Post by x1a4 on 2007-04-09
Hi,

Just as with DOS/Windows, command-line commands are software tools without a GUI.  To find out what you have on your system look in the *bin* (*bin* as in binary, although these days scripts are often installed in those as well) directories.  Then, to get extensive info about an application run *man <application>* (*man* as in manual, and *<application>* is the application you want to learn about).  

Too see what you have in a directory run: 

*ls /directory/path/*
(eg: *ls /usr/bin/*)

To find out what *ls* is all about run: 

*man ls*

Primary *bin* directories are

[I]/bin/
/sbin/
/usr/bin/
/usr/sbin/
/usr/local/bin/
/usr/local/sbin/
/usr/X11R6/bin/
/usr/games/[/I]
