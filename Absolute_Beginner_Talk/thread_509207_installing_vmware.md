### Post by nhandler on 2007-07-25
Did you download it from the internet or did you install it using Synaptic? If you used Synaptic, it should already be installed.

Another way to install it is to launch a terminal (Applications->Accessories->Terminal) and enter
```
sudo apt-get install vmware-player
```

That will then ask you for you password. Enter it, and it should install vmware-player.
