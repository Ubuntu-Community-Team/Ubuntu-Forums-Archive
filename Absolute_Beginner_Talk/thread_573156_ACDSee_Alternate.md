### Post by Perfect Storm on 2008-04-15
No problem, here's what you do. Close synaptic and/or add/remove application first (vital to do so).

Go to System tab ---> Aministration ---> Software Sources

In Ubuntu software tab, you make sure you have enable "main","universe","restricted", "multiverse" and "Source Code".

In third party tab you can enable partner to get access to more software.

When closing "Software Sources" afterwards, you'll be ask to "reload". Do so.

------------------------------------------
Now you can use the search button in Synpatic Package Manager (System tab ---> Administration ---> Synaptic Package Manager) to find the stuff you want. Or use add/remove...but it's a good idea to learn to use Synaptic Package Manager.


If you havn't enable Medibuntu, I recommend you do so. The stuff you'll find in Medibuntu is close-source/non-free libs and software which with good reasons (legality/copyrights etc.) can't be in the normal repositories. (DVD-playback, closed media codecs, googleearth etc.)

Here's how you do it: [http://www.medibuntu.org/](http://www.medibuntu.org/)
