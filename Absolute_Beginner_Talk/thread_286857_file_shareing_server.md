### Post by Compact on 2006-10-28
You can install the ftp server in any ubuntu edition, but I recommend you to install the server version which hasn't got the X server installed. 
Almost every browser can download files from an ftp, if you are planning to upload files to ftp server you will need a ftp client, because not every browser can upload files to an ftp.
