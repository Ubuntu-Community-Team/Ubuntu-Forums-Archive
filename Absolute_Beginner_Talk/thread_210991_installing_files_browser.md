### Post by someusernoob on 2006-07-07
Open Places, grab the Home icon with your middle mouse button and place it where you want. Is that what you mean?

For the 'official' file browser icon go to:
Applications > Accessoires > Alacarte Menu Editor and check File Browser under Accessoires (it will appear in your menu now). Now you can drag and drop that icon to anywhere you want.
