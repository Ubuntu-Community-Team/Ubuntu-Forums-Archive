### Post by luisromangz on 2007-05-20
Pyqt doesn't depend on KDE, it just depends on python, X11, libgcc and some other standard stuff.

You can install it through Synaptic, just search for python-qt3 (or python-qt4, if the program you are triying to run uses QT4).
