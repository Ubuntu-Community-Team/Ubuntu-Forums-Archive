### Post by krokodil on 2007-08-28
You want something that plays music at a certain time or do you want something that goes just makes an annoying piercing noise at a certain time?

I prefer to wake up gently to music so I use a music player for an alarm clock. Currently I'm using Quod Libet's alarm plugin which simply allows me to set a time to start playing. You could create a playlist with screeching sounds, however it does not feature snooze.

The most advanced music player alarm I've seen is for XMMS. It allows for volume fade in and you can choose which playlist to play for an alarm.

For the more normal alarm clock try Kalarm (see a previous post) it seems to be similar to your Banshee Screamer thing.
