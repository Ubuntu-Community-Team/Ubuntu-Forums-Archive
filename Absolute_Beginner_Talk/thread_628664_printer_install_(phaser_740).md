### Post by paydaydaddy on 2007-12-01
Go to system, administration, printer. Follow the wizard and you should be able to see and set up your printer. There is a pretty good wiki about printers that helped me when I first started with Ubuntu. You should be able to find a link by doing a forum search.

### Post by halitech on 2007-12-02
When you are going through the install, select a network printer, use IPP or Printer on CUPS server and then enter the IP address of the machine. Then when it comes time to select the driver, just click on Install Driver and browse to where you extracted the files. From there it should be self explainatory.
