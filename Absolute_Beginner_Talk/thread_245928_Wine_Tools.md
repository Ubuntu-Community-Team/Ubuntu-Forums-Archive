### Post by Crosbie on 2006-08-28
You can use 'alien' to install .rpms, I believe:

sudo apt-get install alien

then just

alien winetools.rpm

or whatever the package is called.  Can't vouch for how well this works, but there's a useful howto [here](http://www.ubuntuforums.org/showthread.php?t=149585).
