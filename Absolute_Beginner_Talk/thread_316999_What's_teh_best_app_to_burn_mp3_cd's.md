### Post by mcduck on 2006-12-11
I'd go with Gnomebaker too.

If you want normal Audio CD's note that Gnomebaker uses Gstreamer0.8 instead of Gstreamer0.10 so you need to install those codecs..

But for mp3 cd's it doesn't even matter if you have any mp3 support installed, it's just meaningless data for the burning program anyway..

Just remember that mp3-cd is a data cd, not audio-cd.
