### Post by khgiese on 2007-12-26
If you have a CD burner download the Ubuntu iso image and create the Live CD. If not the best thing would be to order a Live CD. It has a partition application.

There are several ways to partition a hard drive, some easier the others.

A trick I use to use with a Windows XP CD was to boot off the CD and choose to do a fresh install. you will be prompted for the partition you want to install too. Delete all the partitions and create new ones. You have to go through the format step for the partition configuration to be set. Once the formatting is complete you you can exit the install. 
The draw back to it is you can only choose NTFS or Fat32 formats.
