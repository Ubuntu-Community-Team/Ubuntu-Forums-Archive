### Post by CCNA_student on 2008-01-27
I posted exact instructions on my Linux User's Group website at [http://wheatlandlinux.wordpress.com/instructions-and-how-tos/]("http://wheatlandlinux.wordpress.com/instructions-and-how-tos/")
Like you said, defragment your hard drive first. And please be sure to back up all of your data and have your recovery or Windows OS disc handy just in case something goes wrong. It is easy enough to shrink your NTFS partition. On step 4 of the installation, it will ask you how and where to install Ubuntu at. The second option is called something like "resize IDE 1 install Ubuntu in new unused space", this would be the easiest way to install Ubuntu. I hope that you like Linux.
