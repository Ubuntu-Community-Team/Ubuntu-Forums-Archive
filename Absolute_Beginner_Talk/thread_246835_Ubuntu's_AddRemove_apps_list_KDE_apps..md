### Post by teet on 2006-08-29
> **phr0ze said:**
> About: I just installed ubuntu for the first time. I have tried Linux on and off for years.

Yes you can run KDE programs in gnome and vice versa.  Many gnome users prefer programs like Amarok or K3B over gnome equivalents.

You'll have to download the KDE libraries as well (it does this automatically) though.

One downside is that KDE apps often don't look very pretty in gnome (and vice versa).
