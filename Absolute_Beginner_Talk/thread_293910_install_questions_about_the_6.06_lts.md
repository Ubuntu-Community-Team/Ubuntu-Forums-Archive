### Post by taurus on 2006-11-05
If you are using CDBurnerXP, fire it up and pick the first option in that window, burn an ISO image.

Click File and choose "Write Disc from ISO file."
Browse to where you have downloaded the image.
Click the Writing speed and pick 4x - 600kB/s.
Click Write Disc and just wait for it to finish writing to your CD-ROM.  

If you insert the CD-ROM back in the drive again, you will see a welcome message about Ubuntu.  If that's the case, reboot with the CD still in the drive and run it or install it...

It shows the ubuntu program on the screen, but in the open box it says *.dxp. What do I do?Do have NERO? That's how it worked for me:> Burning  Boot CD .ISO to a CD with Nero SmartStart and Nero Express 6 

1) Launch Nero SmartStart
2) Select the "Copy and Backup" icon 
3) Click "Burn Image to Disc" option 
4) "file selector window" opens up 
5) Here: Change in 'Files of type' bar to  "Image files (*.nrg, *.iso, *.cue)"
6) Navigate to wherever you extracted the *.iso file to
7) Select the .ISO file and click 'open'
8] If you have several CD-drives select  the CD-writer that you're going to use
9) Set writing speed to 4x or 8x
10) Click 'Next' and the burning process will start

When finished you have the bootable Ubuntu Live CD.
