### Post by Obor on 2006-11-14
I'm not sure what you mean by real ubuntu. Live CD gives you exactly the same as a basic ubuntu install. As far as I know the server software is not on live CD. You would have to install ubuntu on your HDD and then follow [this guide]("https://help.ubuntu.com/community/ApacheMySQLPHP") to install AMP (Apache, MySQL, PHP)

Otherwise you can use the ubuntu server CD which will install the server stuff ([guide with screenshots for dapper]("http://www.howtoforge.com/perfect_setup_ubuntu_6.06"), [for edgy]("http://www.debianadmin.com/ubuntu610-edgy-lamp-server-installation-with-screenshots.html")) and then you can add ubuntu-desktop to get the desktop applications etc. if required.
