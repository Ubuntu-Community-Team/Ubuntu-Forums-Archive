### Post by Metacarpal on 2006-09-13
You'll need to follow the steps [on this page]("http://easyubuntu.freecontrib.org/get.html").  The code in the first orange box needs to be entered, one line at a time in a terminal session.

To open terminal, go to the Applications menu > Accessories > Terminal

When you run the last line (sudo python easyubuntu.in) it will prompt you for a password.  Enter the same password you use to login.  Then a graphical interface will start for EasyUbuntu.

Edit: You can copy and paste the code to make sure you get it right.  In Terminal, paste can be done with shift+insert, or by right-clicking and choosing Paste.
