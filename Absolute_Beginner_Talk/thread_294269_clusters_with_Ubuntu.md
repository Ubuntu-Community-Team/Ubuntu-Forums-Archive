### Post by Marsolin on 2006-11-06
I haven't set up a cluster myself, but a couple monitoring programs that you might end up finding useful are [Ganglia]("http://linuxappfinder.com/package/ganglia-monitor") and [NetDirector]("http://linuxappfinder.com/package/netdirector").

I'd suggest reading the [ClusterMonkey]("http://www.clustermonkey.net/") website for info on clustering. They have some pretty good articles.
