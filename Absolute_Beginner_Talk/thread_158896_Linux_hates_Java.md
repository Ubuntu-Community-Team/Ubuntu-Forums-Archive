### Post by xhie on 2006-04-11
Linux does not "hate" java, its got more to do with keeping things open source. Ubuntu is a totally free open source distro, and in that spirit things that are not open source rarely, if ever make it into the distro, or any distro that follows the same ethics. mp3 support is another example of this. Both java and mp3 can be added by you the user later, but they will never make it into the distro out of the box.
