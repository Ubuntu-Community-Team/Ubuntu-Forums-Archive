### Post by Vogateer on 2007-09-22
I could be completely off on this, but I've noticed that changing your theme can seriously affect the speed of Nautilus. I had my computer set up with several different themes, and noticed a serious speed decrease, taking about 5 times longer to load folders, at least. I can't remember the icon set (which seemed to cause the problem) but it was something I loaded up from gnome-look.org.

Try saving your current theme as something like "My Personal Theme" and switch back to Human if you want to keep your previous theme's settings and give this a try. It really worked wonders for me.

Oh yeah, also, Konqueror was really responsive for me as well, though it was using the Crystal icon set, so I don't know how it would be affected otherwise.
