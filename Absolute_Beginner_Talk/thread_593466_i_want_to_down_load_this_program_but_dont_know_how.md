### Post by ed-koala on 2007-10-27
First things first.  The program you want is for Internet Explorer only, so you need to have a couple of things before you download the program.

Install Wine using Synaptic Package Manager
Next, use Wine to install Internet Explorer
Then, download your program, unzip the file, and use Wine to install it.

Some caveats - First, Wine is not perfect and some Windows programs do not run properly or at all.  You just have to try it and see.

Second - I had some trouble installing IE on my system, but it was most likely because I didn't google how to do it properly.  Look these things up on the web beforehand and you'll usually find step by step guides on how to use/install certain programs.

I didn't go thru how to perform each step simply because all that info is already posted in these forums somewhere. All you have to do if you don't know how to accomplish a step is look it up.  I'm not trying to sound mean, just don't want to go thru a long explanation that has already been listed.
