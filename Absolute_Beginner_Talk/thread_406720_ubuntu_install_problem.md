I did not personally like how you pick your partitions in the Ubuntu installer, probably Im just used to the typical fdisk command. :) 

If you dont like it either, you can always set up your partitions using de LiveCD. This page from the Gentoo Handbook can help you with the command needed.
[http://www.gentoo.org/doc/en/handbook/handbook-x86.xml?part=1&chap=4](http://www.gentoo.org/doc/en/handbook/handbook-x86.xml?part=1&chap=4)
