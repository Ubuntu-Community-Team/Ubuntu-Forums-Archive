### Post by Kobalt on 2007-03-04
There are several ways to change the splash-screen, the safer I would recommend is to add a package named gnome-splashscreen-manager :
```
sudo aptitude install gnome-splashscreen-manager
```
Then go over gnome-look.org or art.gnome.org to search and pick your theme... 
Other methods like GfxBoot I wouldn't advise unless you're pretty good in Ubuntu manipulating.
