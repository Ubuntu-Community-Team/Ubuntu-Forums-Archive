### Post by Kieranties on 2006-11-13
So long as you have the firefox plugin for flash then you can view it in firefox.

Right click the file and select "open with" think its that Im on a win machine at the mo.  If the option isnt there choose other and type in firefox as the program to use.  Firefox will be an option in the list from now on.

On the other hand, as it came with some htm files it probably is contained within one of them.  See if there is an htm file called index or default or something indicating it as the first file.  Open that with firefox in the same manner
