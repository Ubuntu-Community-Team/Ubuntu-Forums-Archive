### Post by thelatinist on 2008-01-02
No, dual booting means that you can boot to either operating system.  You have to *re*boot if you want to enter the other.

The only way to do what you describe is to install one OS in a virtual machine running under the other. The OS running in the virtual machine will be significantly slower than if you booted into it natively.
