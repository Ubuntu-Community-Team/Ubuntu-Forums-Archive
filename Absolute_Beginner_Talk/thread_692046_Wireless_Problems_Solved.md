### Post by visionthing on 2008-02-09
Finally after much stuggling to get my wireless to work probably (got it to run with ndiswrapper, but it never would start up properly and was always a headache) I have solved the problem with a small purchase. Instead of a wireless PCI card I connect my onboard network connection into this wireless access point (approx $50) [http://www.dlink.com/products/?pid=333](http://www.dlink.com/products/?pid=333)

So Linux only needs to know about my onboard controller. Plus this seems to connects to my wireless router a little more quickly.

Just wanted to pass this on as an alternative to getting an actual wireless card working.
