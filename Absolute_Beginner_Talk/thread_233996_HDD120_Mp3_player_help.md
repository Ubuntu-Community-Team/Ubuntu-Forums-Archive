### Post by Arisna on 2006-08-10
Without actually seeing what is in the script, general advice would be to mark the script executable first.  You can either right click it in file manager, click "Properties," and go to the "Permissions" tab.  Check the box for Execute across from Owner.  You can now run the script.  Simply double clicking it in Nautilus might work--otherwise, you might need to run it from a Terminal with parameters.  Try the permission and click strategy first.
