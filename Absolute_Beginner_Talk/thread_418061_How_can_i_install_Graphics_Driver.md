### Post by heimo on 2007-04-22
It may be unnecessary to install new drivers as your Xorg (the base for your graphical user interface) comes with support to Intel graphics. It's possible that your system is just not configured to use Intel driver, or that there is some other configuration needed. Actually, I think, Intel is best choice for graphics in Ubuntu (sadly, I don't have one).

Is the problem that you don't get correct resolution in System->Preferences->Screen resolution? Then try opening a terminal (Applications->Accessories->Terminal) or hit alt+F2 and type:
```
gksudo gedit /etc/X11/xorg.conf
```
It'll ask your password and open Xorg configuration file in a text editor. Don't make changes at this point. Just copy-paste us the sections Device, Monitor and Screen. My exampe is below.

```
Section "Device"
    Identifier    "nVidia Corporation G71 [GeForce 7300 GS]"
    Driver        "nv"
    BusID        "PCI:1:0:0"
#    Option        "RandRRotation" "Enable"
#    Option        "Rotate"    "RandR"
EndSection

Section "Monitor"
    Identifier    "CM752"
    Option        "DPMS"
    HorizSync    40-80
    VertRefresh    60-90
EndSection

Section "Screen"
    Identifier    "Default Screen"
    Device        "nVidia Corporation G71 [GeForce 7300 GS]"
    Monitor        "CM752"
    DefaultDepth    24
    SubSection "Display"
        Depth        24
        Modes        "1280x1024" "1024x768"
    EndSubSection
EndSection

```
