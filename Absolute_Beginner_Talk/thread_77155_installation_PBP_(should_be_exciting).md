### Post by SilentCacophony on 2005-10-16
If you choose 'erase entire disk', then you'll have only Ubuntu on the drive, which will indeed replace Windows. Sounds like that's your choice.

I believe the default way it'll set up that way is by making two partitions, one for /, and a small one for swap space. In that setup, you would use your home directory to store any files on that drive, which is fine for most people.
