### Post by PeterJS on 2008-01-12
That's what happens when compiz or beryl crash. To get things back to normal open up the run dialog with alt+f2 and run beryl and then emerald --repalce.

Which version of ubuntu are you running? if it's 7.04 you might want to upgrade the whole system to 7.10, if you're already running 7.10 you should be running compiz fusion instead of beryl. Compiz fusion, is the fusion of beryl and compiz.
