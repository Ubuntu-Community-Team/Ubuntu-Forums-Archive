### Post by davmac on 2006-08-31
Hi Griff,

I too use MS Project (and also Niku OpenWorkbench) for project planning. Whilst there are planning tools available, I've not yet found any way of opening up or importing MS Project into any of the tools that run on Ubuntu. Aside from this, I've not found any of the Linux planning tools approaching the "industrial strength" that I suspect you'll be after.

It'll be worth keeping an eye on OpenWorkbench ([http://www.openworkbench.org)](http://www.openworkbench.org)). It's windows only at the moment but it will open MSP files if you save them in XML format and now it's open source I'm hoping it's only a matter of time before it gets ported across.

The best I've been able to come up with is to MS Project within Windows XP within VMWare Server. :( 

Dave Mac
