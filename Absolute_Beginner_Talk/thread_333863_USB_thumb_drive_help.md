I have a 2GB USB thumb drive with about 800mb of pictures stored on it from windows. When I plug it into my USB port linux automatically opens it up but theres nothing there. It shows that theres about 1.9GB space left on the thumb drive, from 2GB so I know theres stuff on there (plus when I put it in my windows machine I can see everything on it). ### Post by HereInOz on 2007-01-08
I think you may have hit it with the NTFS formatting.  My only suggestion would be to copy the photos to a windows machine, re-format the USB drive as FAT32, put the photos back on it, and see if you can then see it.

If you can, that was your problem.  If you can't, bear in mind that many of the cheap USB memory sticks around are not particularly reliable, in that they work on some machines and not others, and it could possibly be something along those lines as well.
