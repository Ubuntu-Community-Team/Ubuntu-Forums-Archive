### Post by K.Mandla on 2007-03-08
You can test drive it if you want, by installing the *xubuntu-desktop* metapackage. You can do that from Synaptic or with this command.

```
sudo aptitude install xubuntu-desktop
```
If you find you don't like it you can cleanly remove it with 

```
sudo aptitude remove --purge xubuntu-desktop
```
or via Synaptic again.

But Sefrin is right: If you use a much lighter window manager and forego the entire desktop environment, you'll see a huge speed gain.
