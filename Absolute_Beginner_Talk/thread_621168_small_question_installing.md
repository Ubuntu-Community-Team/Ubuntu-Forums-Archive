### Post by banewman on 2007-11-23
The cd works as a "live" version of the OS.
Put the cd in and reboot and you will get the option to start or install - choose that and then the OS will run but not install anything to disk. A good way to see if it works with your hardware. Once running, there will be an icon on the desktop that you can click to install the software if you are happy to. :)

### Post by forestpixie on 2007-11-23
you need to firstly make sure that you've defragmented you're win installation.

Then you need to resize the partition to give ubuntu room to install - if you're running vista you need to use the vista tool. If not the livecd can do it for you when you install the os

As far as the monitor goes - if the live cd works then so should the installation.

What you're trying to do is a dual boot - there's plenty of info about here and generally on the net - psychocats seems to be ok - it helped me

When you do install - don't use the 'use entire disc' option or you will lose windows.

### Post by banewman on 2007-11-23
In the menu off the live cd is a program called "gparted". It will allow you to resize the windows partition to make room for ubuntu. Make sure you choose manual partitioning and follow the advice there.
I don't know the size of your hd but the cd advice is at least 2gig for the OS and 256mb for a swap partition.
Hope this helps some but ask for more if you need to. :)
