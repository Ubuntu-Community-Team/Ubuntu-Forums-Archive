### Post by Qrk on 2006-04-20
```
  wget -c ftp://ftp.nerim.net/debian-marillat/pool/main/w/w32codecs/w32codecs_20050412-0.0_i386.deb
  sudo dpkg -i w32codecs_20050412-0.0_i386.deb
```

Paste this into a terminal one line at a time. 

You can also look for automatix (on these forums) and/or read the wiki page. 

[https://wiki.ubuntu.com/RestrictedFormats](https://wiki.ubuntu.com/RestrictedFormats)
