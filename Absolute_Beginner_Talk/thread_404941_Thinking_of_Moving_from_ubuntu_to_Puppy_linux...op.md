### Post by feed_sparky on 2007-04-09
I actually just started using Puppy about a week ago, and I have it installed on a 1G usb flash drive. I am a linux noob, but was able to install Puppy to a usb and get wireless working (the tricky 4318 broadcom) in less than an hour. Puppy has its own package installer PupGet and DotPup packages you can download and have PupGet install, sometimes dependencies can be a little tricky but not all that hard to figure out. I would suggest dual booting, there are many options: install to HD, usb flash, external card reader with a CF card, or just use the cd and make a save file on a partition.

### Post by isTHEr3mOr3 on 2007-04-09
Puppy is a fantastic distro. It amazingly fast and has a great out of the box experience, with multimedia support.  
It's my most used desktop right now, browsing is fast, installing is fast and the latest build also looks great. It's also very easy to install. Don't believe people that say it's only a rescue distro, I've harddisk installed it on my 900 Mhz AMD and it is rocksolid for day-to-day use.  

There's only 1 but.. and that's installing new software. That can sometimes be a problem (especially for newbies), so if you want to use it as your primary distro, look at the puppy site first if all the programs you want to use are supported. There is a great forum as well with very knowledgeable people. 
If Puppy has got what you need, you really should drop all bloated distros and go for speed. If you have become used to the speed you will never want to go back. 
It's really way faster than Xubuntu or any other distro I know (and I've tried lots ;) )

I still use Ubuntu (Feisty right now) on my not so much used desktop, just to know what's happening and I guess because I like installing distros :)

Installing Utorrent: Don't know about that, but I've got azureus installed and that works perfect with Puppy. Wine is also available for Puppy.
Installing Gnomad: There is a thread on the puppy forum about that, and I think you should be able to install it.
