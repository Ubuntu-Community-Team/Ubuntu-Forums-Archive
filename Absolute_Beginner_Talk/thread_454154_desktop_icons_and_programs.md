### Post by ruscoe on 2007-05-25
Welcome to the community.

1) Ubuntu will be installed on a separate partition to Windows, if you go for a dual boot, and will not have access to your applications or current desktop settings. Adding shortcuts is a trivial matter and you should have your desktop set up as you like it in no time.

7.04, the latest release, does come with a migration assistant which should help you transfer your web bookmarks and possibly email settings from Windows to Ubuntu, but I have never used it and cannot tell you much about it.

2) Thunderbird, Firefox and OpenOffice.org all work on Ubuntu and Thunderbird is the only one that isn't installed by default.

### Post by Tux Aubrey on 2007-05-25
Welcome.

I think you will be pleasantly surprised.  The default "desktop" is called Gnome.  It  is very configurable and you can have icons on your desktop, on the default launcher bars or install other gizmos to launch anything.  Ubuntu gives you lots of options (which can be confusing or overwhelming when you first move from windows).

There are also several other window managers/desktops you can install and try.

I generally stick with the Gnome desktop but keep the desktop pretty clear - just a few icons to different drives and partitions where I keep my files.
