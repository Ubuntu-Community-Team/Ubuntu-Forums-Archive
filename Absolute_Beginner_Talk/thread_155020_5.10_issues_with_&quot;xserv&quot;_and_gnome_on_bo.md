### Post by izmaelis on 2006-04-04
Hit Ctrl+Alt+2 when you see that blue screen wich informs that xorg server and gnome failed to start. Then you should be able to enter user and password and login to command line interface. After that do
```
sudo nano /var/log/Xorg.0.log
```
and check for more detailed information why graphical interface failed to start. Search for the lines containing "(EE)" and show them (or whole file /var/log/Xorg.0.log) here.
