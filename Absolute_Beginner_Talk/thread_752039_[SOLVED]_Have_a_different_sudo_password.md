### Post by ajgreeny on 2008-04-11
This may not be quite what you meant, but it is possible, though not recommended to enable the root account, which means that the password for doing admin things, currently **sudo userpassword command **would become **sudo root password**.  As I say this is not recommended and as far as I can see gives no advantages over the current sudo system.  See [http://www.ubuntugeek.com/enable-and-disable-ubuntu-root-password.html](http://www.ubuntugeek.com/enable-and-disable-ubuntu-root-password.html)
