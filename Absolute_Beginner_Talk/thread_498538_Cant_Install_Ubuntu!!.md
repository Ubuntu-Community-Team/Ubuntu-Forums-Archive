### Post by GerryB on 2007-07-11
> **Skidpad said:**

Hi! I hope this helps. I'm running Ubuntu 7.04 (Feisty Fawn) on a 440 Mhz computer with 384 MB of RAM and a 10 MB hard drive - no problems. So you have enough to get it going. Once you download the ISO image, you must burn it as an image. To simplify things you can punch in "BurnCDCC" in Google. You can dowload this small utility to burn images. It can only burn images so you can't go wrong. Once you have the newly burned CD, turn everything off. Now you have to make sure your BIOS has the proper sequence. When you restart the computer you will see instructions on getting into your BIOS, most often by pressing the Del key, sometimes the Esc key. If booting options don't come up right away, look into each category at the beginning until you come to "Boot Sequence". Don't touch anything else. Usually you press the Page Up key to make the changes - go ahead and experiment - you can't do any harm. Once you have the right sequence, ex: CDROM, C, use Esc and go to where you exit. The computer will ask you to confirm with "Y" if you accept the changes. You're set. Leave it that way, because when you don't have a bootable CD in the drive, it will boot from C - you'll probably want to boot from the CD drive for other things later anyway. If this doesn't work, order the CD by mail - some computers are really quirky when it comes to burning CD's. There are fantastic experts on this forum - of which I am not - and someone will come up with the solution. Installing an operating system (when we buy computers, this has all been done for us)  is a big challenge but not insurmountable. Best of luck!
