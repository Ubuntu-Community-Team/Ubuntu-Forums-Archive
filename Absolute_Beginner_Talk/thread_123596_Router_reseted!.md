### Post by bernardfrancois on 2006-01-31
Try browsing to 192.168.1.1 (just enter that in the address bar of your browser).
Probably you will be asked for a password (and username, eventually), if you don't remember any password you can try 'admin', 'administrator', 'Admin' or 'Administrator' (for both username and password).

This is the default setting on most routers.
If it doesn't work, you can search for the manual of your router (on the net, if you don't have it any more).
