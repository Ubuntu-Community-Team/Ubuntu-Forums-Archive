### Post by m3ntor on 2008-02-13

---

### Post by emarkd on 2008-02-13
You can just use the Ubuntu LiveCD.  Boot from the CD and run gparted.  You can delete the partition and resize your Windows partition to take back the space.  Or leave it separate and reformat it as fat32 or something Windows knows how to deal with.  You don't really "uninstall" an OS, but you will have to use your Windows CD to reinstall the Windows bootloader.
