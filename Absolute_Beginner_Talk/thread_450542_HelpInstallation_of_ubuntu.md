### Post by aysiu on 2007-05-21
That can be a little too little RAM for the live CD. My laptop has 256 MB of RAM and also freezes up when I try to use the Desktop CD to install Ubuntu. The problem is that the Desktop CD runs an entire fully functional operating system  as well as the installer... all from the CD and your computer's RAM, so if you don't have a lot of RAM, your computer can't handle that much activity.

The Alternate CD allows you to install Ubuntu without having a live session going at the same time. More details here:
[http://users.bigpond.net.au/hermanzone](http://users.bigpond.net.au/hermanzone)
