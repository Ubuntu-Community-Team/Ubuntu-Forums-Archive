### Post by davarino on 2008-03-08
As you probably already know, OO.o password-saves only to OO.o native formats... although it can open MS Excel-created password-protected files.

Your fastest solution would be to save the file as an .xls file and then to "compress" the .xls file with an archiving program as a zip file with password. You could probably set this up as a macro or script if you do this quite a bit.

Good luck. OpenOffice.org is a wonderful tool, in my estimation. I prefer it to the so-called competition.
