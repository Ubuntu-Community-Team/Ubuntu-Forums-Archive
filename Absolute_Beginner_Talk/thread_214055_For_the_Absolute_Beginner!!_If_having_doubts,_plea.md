### Post by Upochapo on 2006-07-12
Right now I'm a little irrate and need a place to vent and couldn't find an informal place to vent here.  But for those that are absolute beginners this may be what you need to hear!

It so happens that I installed Ubuntu and got it set up and perfected the look of my desktop to my likings.  There seems to be a small problem with Ubuntu and my system concerning my favorite game NWN.  It just likes to crash unexpectedly and have yet to find a solution for this.  I did everything by the book and am at a complete loss.  Ok, see that little tid bit.  That's the bad news.  That's it.

Now comes the funny part.  In my venture to play my favorite game on linux because I just couldn't take windows any longer, I tried to install another distro alongside my ubuntu.  Well, dummy me accidentally selected format entire drive *whimper*.  Before I realized my mistake, it was gone, like a puff of smoke.  Now, just to use this as an example because this was the distro I was trying to install, PCLinuxOS seemed to be a fine distro. I installed it and this is where the nightmare for windows users making a switch begins!

What every window user needs to know when installing a Linux distro:

Keep an eye out for this doozy...The seemingly endless menu system.  There is an unexplained phenomenon in the linux community where it seems that the more things you have in your menu the greater your distribution must be.  What's even better, these programs have such funky names that you could only fathom a guess to their function and purpose.  Not only that, a lot of times they will put items in there that will run in the background.  But we the end user do not know this.  We click.  We see our screen blip.  We wonder, Did it execute?  Is there something going on?  Is the program broken?  We don't know.  All we know is that we clicked on flddjel, we saw a blip and we said a small prayer hoping that wasn't a no-no. Oh and if you happen to find something that you did find useful, good luck trying to find it again, especially if it was nine menus deep. All you old time linux users out there, can you explain this to me? I am posting this from pclinuxos as we speak and have a headache from navigating the menus.

The second unexplained phenomenon.  Why on God's green earth do you have to have 3400932 programs that do the EXACT same thing?  And, the real kicker is that it seems almost every program was included but 1 in linux distros.  Heck, if you're gonna go to that amount of trouble, what's one more?  And don't give me the ol' it's nice to have variety.  Ubuntu has it, you just have to install it through their package manager.

Finding Help.  I can not stress this enough.  You like the dentist?  You like having a root canal without a local anesthetic?  Well, boy do I have some great linux distros lined up for you!  Just pick one.  Because that is what it is like finding good help for trying to get your system up and running.

If you are seriously thinking about making a switch or trying out linux for the first time.  Ubuntu or any of its flavors is the absolute most perfect place for you to begin.  Is Ubuntu perfect?  No.  But neither is windows, any linux distro, or any other operating system for that matter.  I have been running Ubuntu for only one month and have had four other people that I know want Ubuntu set up on their system and then made a total conversion.  That's TOTAL conversion.  The only downside was getting the latest and greatest games to run but heck there are workarounds for getting windows based games to run.  And there are some great native linux games!

I have tested all of the different flavors.  Maybe I can help shed some light on which is right for you.

Ubuntu:  The cream of the crop.  It really takes talent and skill to muck this bad boy up.  My personal favorite flavor.  Sleek, easy to use, moderate amount of eye candy, great screensavers, and a wonderful selection of applications.  Themeing your desktop is a breeze.  This is a great distro to use for starting out easy.

Kubuntu:  Another very nice edition.  More of a windows type feel.  You may feel a little more at home here.  Plenty of eye candy to go around with nice KDE default desktop applications installed.  The Ubuntu control center that is incorporated into Kubuntu is a nice touch.  This is where my Ubuntu experience all started.

Xubuntu:  Great fast linux distro for older machines.  The desktop is light and loads quickly.  Can have a bit of a learning curve getting used to the way the XFCE desktop is setup and how everything works.  And it has that cute little mouse running around in a circle when your desktop loads.  Do you really need anything else?

Edubuntu:  I had a bit of a hard time getting this one to stay up and running.  But to put fears aside, that was my own fault.  But what I got to play with was great.  If you have kids, this is an absolute MUST!!  It comes with gCompris I believe which is a suite of programs geared towards kids.  I checked out those applications and way cool for kids.

Ubuntu isn't plagued with the big perils of other distros. You have simple menus.  You know what the programs in the menus do because they are properly labeled/named. And if you need more stuff, it's just a few clicks away with Synaptic. And help can be found a few clicks away!  Ubuntu is really that easy.  But be forewarned, you may have to cut and paste commands from the help wiki's.  But guess, what!  These commands actually work and do exactly what they say they're going to do.

I have to add this small important note.  Please be aware that when installing any operating system you may have some issues with getting hardware to run correctly.  As much as I have grown to love this operating system it does have it's fair share of users who do experience some difficulties.  Sometimes, it could be as simple as downloading a couple of files, sometimes it isn't.  I have yet to not find a solution.  After a little research, I have solved what few problems I had.  

Take care and thanks for reading.

Upo.
