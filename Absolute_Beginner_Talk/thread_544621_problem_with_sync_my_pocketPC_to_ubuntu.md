Prepare yourself for disappointment. You will ***not*** be able to install Windows Mobile XXXX programs through  Ubuntu, or Linux in general. ActiveSync is Windows, and Windows only. One of the very few areas in my opinion, where the empire is still a touch ahead of the rebellion.
