### Post by Bigbluecat on 2006-10-12
The problem is not necessarily the player but the fact that you do not have to correct codces installed. 

MP3, MPEG etc. are proprietary formats. As such they do not come bundled with Ubuntu open source software.

As a user you need to explicitly request these to be installed.

Please see the followin g link:

[https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)
