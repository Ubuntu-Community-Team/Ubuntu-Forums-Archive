### Post by LuisGMarine on 2008-02-24
I got this from banshee website.  Apperently there is a plugin that you can use, I'm not sure how to set it up and go about it, because I've never used such a thing.  But I'm sure you can find a tutorial out there on how to get it to work.   Here is the link to the banshee web page:

[**http://banshee-project.org/Guide/Share**]("http://banshee-project.org/Guide/Share")

This is the plugin that you need, I think you might need the extra repos that Ubuntuguide offers, but this is the command:

```
sudo apt-get install banshee-daap
```

Hope this helps you get started.
