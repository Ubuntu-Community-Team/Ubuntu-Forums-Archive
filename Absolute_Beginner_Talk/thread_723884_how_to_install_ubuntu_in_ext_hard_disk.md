### Post by baddog144 on 2008-03-14
This should not be hard...

Just go to administrative tools (in Control Panel), open Computer Management, go to Disk Management, and partiton it there. However, this won't work very well as you will havev to format it as NTFS or something, not Ext3. 

Alternatively, you can just use the Ubuntu installer, which will do the partitioning for you.

If you *really* want to do it yourself, you can use GParted on the Live CD.

Hope this helps ;)
