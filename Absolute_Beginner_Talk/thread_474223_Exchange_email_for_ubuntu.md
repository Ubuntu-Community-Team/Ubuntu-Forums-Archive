### Post by irish_flu on 2007-06-14
That depends on what services Exchange is running.  MS Exchange Server supports both IMAP and POP, which are the two common mail protocols used by non-Outlook mail clients (including Thunderbird).

However, there is no guarantee your server admin has those services turned on (I can't remember if they're on by default or not).  I'd ask whoever runs your mail server about it.
