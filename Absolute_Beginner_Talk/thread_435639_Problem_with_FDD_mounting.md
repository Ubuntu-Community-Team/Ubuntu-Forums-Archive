### Post by AAM on 2007-05-07
you seem to be bereft of answers. I'm not sure that I can help. Are you using a PII? There shouldn't be much of a difference between 5.10 and 7.04 (I run this on my daughters PIIx2 and its acceptable.

Do you have a formatted disk with written files in the drive?

I am aware that really new and really OLD hardware can cause problems. Search the UbuntuForums and Internet for FDD problems. Start by using the error message quoted and see what happens. I can only presume that you installed from the CDROM, so it is strange that its not mountable. Are you trying to mount it without a disk in it? It won't do this unless there are files to see - remember that the drive is actually seen as a 'file' by the opperating system, so the file doesn't exist unless there is a written disk in there.
