### Post by Inxsible on 2007-07-09
> **IDunno19 said:**
> Thanks!  One more question, what's a LiveCD?

LiveCD is a CD which will let you explore the Ubuntu OS without having to install it on your hard drive. Like, it was mentioned in an earlier post, you can see if you like the OS and then decide if you want to install it.

Running an OS from the CD drive will be slower than when it is installed. So don't let that be a criterion in your judgment of the OS. 

Most Linux Os'es now have Live CD options available.

you can download the Ubuntu LiveCD from [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)
This CD will also serve as an installation CD, if and when you do decide to install it.
