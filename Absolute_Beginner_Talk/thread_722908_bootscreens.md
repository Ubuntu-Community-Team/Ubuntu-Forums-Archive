### Post by angry_johnnie on 2008-03-13
There is a nice graphical utility for changing Usplash, called startupmanager.   Look for it in synaptic.  There are also some usplash themes available through synaptic.  You can install those, and then use startupmanager to choose whichever one you like.  There are also some usplash themes available at [www.gnome-look.org](www.gnome-look.org).  You can download them and use startupmanager to install them.
