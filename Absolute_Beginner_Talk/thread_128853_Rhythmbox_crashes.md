### Post by borisattva on 2006-02-12
it think i saw on this forum somewhere that its a known issue when trying to import folders with particular types of non music files, like images, .nfo files, and even unsupported music types. i would try doing it folder by folder (just drag and drop) QUIT rhythmbox after each succesfull import so it saves the importat tracsk thus far. if you reach a folder that causes the crash, try doing it file at a time to find out which is doing it.

also i had problems exporting over smb sometimes. could be a timing issue.
