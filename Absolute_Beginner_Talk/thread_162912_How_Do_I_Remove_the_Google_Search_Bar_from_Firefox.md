### Post by 5-HT on 2006-04-19
Right click just below the search bar (or elsewhere), select 'Customize', and drag the search bar into the customization window that pops up.

*EDIT: That searchbar need not have goole in there. It's quite handy as many different search plugins can be installed.

To remove only google: Go into the 'searchplugins' directory within the firefox installation directory, and delete the google one.
(This is the method that works for me using Mozilla's binaries, but aysiu's advice should work better if you are referring to the default Ubuntu Firefox package)

HTH
