### Post by bapoumba on 2008-01-12
Try with the compose key.
If using GNOME, there is a menu under keyboard prefs to enable the compose key.
Then <compose> + O + C = ©

With Xfce, you have to edit xorg.conf, with KDE, I do not know, there must be a menu somewhere too.
