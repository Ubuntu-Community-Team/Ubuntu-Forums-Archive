### Post by marx2k on 2007-02-06
in Gnome there's a button for it on the top right (or bottom right depending on how you configured) 

in a terminal or console you can type 'sudo shutdown -h now' to shut down completely
or to reboot you can type 'sudo shutdown -r now'

or if you're in a terminal, you can reboot with the old CTR-ALT-DEL

in a terminal, type 'man shutdown'

Has some really good info for you
