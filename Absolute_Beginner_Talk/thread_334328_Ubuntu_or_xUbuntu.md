### Post by Jussi Kukkonen on 2007-01-08
Ubuntu should install fine, and even run ok. My own experience includes a thinkpad 600 with 128MB (which was sluggish even with xubuntu) and a similarly specced HP laptop with 256MB, which run Ubuntu just fine. I'll warn you though, Openoffice is a memory hog -- I have no idea about Wine, 

The hard drive is a bit smallish. I believe the Ubuntu base install is more than 3 GB. You'll also need some swap... I'd suggest getting a usb disk for your /home-partition, but check the specs on the usb port first: it might not be full-speed.

Too bad you wrote this now -- two weeks ago I still had a 10GB original thinkpad hard disk just lying around. Unfortunately I already got rid of it.
