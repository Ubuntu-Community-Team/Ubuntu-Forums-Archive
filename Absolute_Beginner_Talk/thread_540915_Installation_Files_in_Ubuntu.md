### Post by forestpixie on 2007-09-02
have a look at apton cd - 
this is where downloaded files go - /var/cache/apt/archives

I'm not sure whether you can use them like setup.exe's, maybe they're all .deb files in the folder - which can be dbl clkd to install with GDebi installer - if you're going to be reinstalling have a go when it's a clean install 

If you are doing a clean install on another pc - did you have a seperate /home folder - that can make it much easier if you ever need to do a reinstall

Good luck
