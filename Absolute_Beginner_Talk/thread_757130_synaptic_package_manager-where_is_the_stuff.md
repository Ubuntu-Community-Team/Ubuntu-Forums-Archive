### Post by ace007 on 2008-04-16
it depends on what package you have installed.  If there is a package with a Graphical User Interface (GUI), then it will likely appear in the Applications/Systems Menu somewhere.  Not always though, if the package is more of a utility, it may need to be started/accessed from the command line.  You best bet is to try typing the name of the package at the command line, or typing a partial name and press the tab key until you think the correct package has been named.  

What packages in particular are you having trouble with? 

You can manually add a Launcher to a menu that will start the package if you know what command will call it at the command line.
