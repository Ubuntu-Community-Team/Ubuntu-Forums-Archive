### Post by Tomosaur on 2007-10-07
Kubuntu looks the most visually similar to Windows (for example, the main application menu is in the bottom left, the only panel is the bottom panel, the clock is in the bottom right, has a quick launch thing, the default theme is blue, and so on), but really, Gnome and KDE are their own thing and both very usable. You can make Gnome look like Windows if you really want to, but its default setup (menus on the top panel, application windows on the bottom panel) makes a lot of sense to me and is also very usable.

However, the only difference between the versions is the desktop environment (and a few applications) - if you download Ubuntu (which uses Gnome), then you can install the KDE desktop on top of that and just pick which one you wish to use when you log on to your machine, and vice versa. You aren't just limited to the desktop environment which comes with whatever flavour of Ubuntu you download, so you may aswell just install one, then install the other desktop environments and experiment to see which you like best.

All of the desktop environments strive to be easy to use (they wouldn't be much use at all if they were complicated!), so there's really none which we can recommend as being the best for newcomers from Windows. I'd say KDE would be more visually familiar to you, so I'd recommend download Kubuntu. Really though, the best choice is to just play around and see which one suits you best.
