### Post by FrozenFox on 2008-02-21
Yeppers.

Open synaptic, it should be under Administration in the start bar / start menu, IIRC. Press the reload button in the upper left. Search for flashplugin-nonfree via the upper right in synaptic. check the box, mark for installation. apply. reload firefox, and you should have flash working.

As for yahoo messenger, there is no yahoo messenger client for linux. However, multiple programs have been made that include yahoo messenger support. Most peoples' favorite is pidgin. Do the same as above, except search for pidgin instead of the flash plugin. Install it and it will be under the start menu under 'internet', i believe. You can go to accounts--> manage in pidgin (if it doesnt ask you on loading it the first time) to add a yahoo account.

As for ubuntu satanic, it tells you on their webpage **exactly how to do it step by step**, just **read it** and tell us where you're confused.. [http://ubuntusatanic.org/installation.php](http://ubuntusatanic.org/installation.php)
