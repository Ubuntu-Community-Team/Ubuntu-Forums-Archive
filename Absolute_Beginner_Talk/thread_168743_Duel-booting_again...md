### Post by philipgm on 2006-05-01
That should be the case, but it sounds like you've been having other problems.

For dual boot it's important to install windows first (it doesn't play well with other operating systems!). Then, install the second OS into the remaining space on the disk. At the end of the process the installer sets up GRUB, the boot loader, that will provide you with the menu to start either windows or ubuntu.

### Post by confused57 on 2006-05-01
The link Sef gave you has some excellent screenshots to help you set up a dual boot system and it's going to take a little reading and planning before you actually start installing Ubuntu.  During the manual partitioning, you can always "go back" to a previous step until you get the partitions set up how you like.
Then there will be a final screen of how the hd will be partitioned, if you like, start the partitioner.  The bootloader "Grub" is best installed on the MBR on WinXP & after Ubuntu installation is complete, you'll have the option of booting up Win or Ubuntu.  It's going to take a little patience, anything worth having is worth working for.
