### Post by Bachstelze on 2007-04-26
Yes, it is perfectly possible. Keep in mind though that if you already have Windows installed and it's partition spans the whole drive, you'll have to resize it and this can be risky. If you can afford it, do a clean reinstall of Win on a partition that does not take the whole drive, and install Ubuntu afterwards.
