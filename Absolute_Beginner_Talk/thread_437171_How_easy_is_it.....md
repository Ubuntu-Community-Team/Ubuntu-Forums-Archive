### Post by starcraft.man on 2007-05-08
The short answer is NO!

Linux is not a cheap replacement shell for Windows, it is its own OS. Just like if you buy a mac you can't run all your Windows only programs. Its a Mac, and this is Linux.

Read [this]("http://ubuntuforums.org/showthread.php?t=437113") thread for more discussion.

There are alternatives to windows programs, [here...]("http://linuxappfinder.com/windows") 

There is also WINE, an implementation of windows API. It will NOT run every windows program. It only runs the most popular and usually an older version.

If you want a PC that runs all your windows programs and configures with no effort, buy an XP computer from Dell. If you do want to free yourself from Microsoft, learn a new OS with new programs and never have to pay for a program again. The choice is yours. Take the red pill  :D

Anyway, make up your own mind.

PS: You can game with WINE (World of Warcraft works well for example) and other popular games have been implemented, it will however, like other things take time to set up. Cedega is a paid alternative with a GUI.

### Post by MOS95B on 2007-05-08
Being close to your situation, I would have to say, you kinda asked the wrong question.

Ubuntu is not hard.  Previous versions of Linux kept chasing me off, because they were hard to install.  Hard to understand. etc..

Ubuntu is not simple.  Having gone from DOS to Windows, Linux took some figuring out.  Installing was great.  it found all my hardware and fired right up.  Getting online was great, once I figured out where the network indicator was hiding.  What to do from there took some headscratching and investigating.  You can't just Google a program and install it as easily as Windows.  Linux actually tries to protect you from that by adding a step or two.  Still not hard, but it kinda forces you to know what you're doing (which after years of Desktop Support and customer service I consider a very good thing)

If you want games, you're in the wrong place.  As stated previously, games will run, but it's not what Ubuntu, or Linux in general, was intended for.  If you want to run newer, graphic intensive games on a laptop, I'm guessing you should run screaming from the room right now before you rip all your hair out.  You're now combining two items that were never intended for this purpose.

I would, in all seriousness, suggest doing what I did.  Play with the Live CD.  Read a **LOT**.  Think abouut you want and or need to do.  Try to fidure out what would actually require windows.  Play with the Live CD.  Read a **LOT**.  (Yes, I repeated that on purpose)  Then make a decision from there.
