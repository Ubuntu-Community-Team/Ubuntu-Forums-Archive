### Post by Onay on 2007-10-19
Partition 2 main drives... here's a basic process of what you'll do.

1. Delete all existing partitions
2. Make 2 new partitions...
[LIST]
[*]ext3 - Make your main partition ext3 should be entire disk minus amount of system memory
[*]linux swap - Make your swap partition the same size as memory + 1GB to be safe (for virtual memory too). e.g. You have 2 GB memory, make a 3GB swap.
[/LIST]

Then in the installer use "manual" partitioning. Select the ext3 partition, check the format box, and "edit" the partition info to mount at "/"

Good luck

### Post by spur on 2007-10-21
It looks like you are trying to repartition from within an active partition.
To get around that either use a third party partitioner or an install cd.
If you boot to an install cd you should be able to get around the cause of your error message.
Did you use the alternate install cd or the live cd?
I prefer to use a partitioner before I attempt an install as I ran across the situation where the install process did not seem to want to format my drive the way I wanted it too and the third party partitioner handled it. I was then able to install.
You may be able to find a trial of a partitioner or a free one if you google for it.
