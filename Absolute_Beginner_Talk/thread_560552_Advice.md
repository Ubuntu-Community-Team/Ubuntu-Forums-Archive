### Post by darc on 2007-09-26
I'd say it depends on your experience...

If you know Linux pretty well, know what you need, and how to configure it ... well, then I suppose you could use about any distro and tweak it how you want it : D
  
But, I'd probably go with Slackware minimal (personal choice) or perhaps Gentoo.  Or (since this is the Ubuntu forums : ) a minimal install of Ubuntu.

Mainly, you'll want to install a real lightweight GUI (if you're going to use one) like xfce, etc.  Avoid KDE and GNOME.
-darc


EDIT:  O yeah, damn small Linux is worth checking out as mentioned above.

EDIT:  Make sure to tweak your boot scripts and remove unnecessary stuff (or don't install it in the first place, hence a minimal install).
