### Post by tonym on 2005-12-08
No it won't.  The only impact you could have would be if you needed to resize/move the partition your existing system is in to make room and it became too small or inefficiently placed.

### Post by ChrisTheGeek on 2005-12-08
As already said, dual booting won't affect the other OS, as long as you have enough disk space.

If you're reluctant to dual boot however, there are two options you may like to consider:

1. Try cygwin ([http://www.cygwin.com)](http://www.cygwin.com)).  It's basically a unix command line for Windows and may do what you want.

2. Use an Ubuntu live CD and save all your work to a USB pen drive before turning the machine off.  That should allow you to work in linux, save your data but not repartition the machine.

Good luck :-)
