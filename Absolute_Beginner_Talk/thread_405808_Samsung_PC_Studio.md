### Post by smilingtim on 2008-03-26
I have the Samsung t-439; the default in the phone is that it is set to "modem" - which allows it to work with Samsung's "PC Studio" program - which should work with bluetooth or USB under windows, but doesn't seem to work with USB under linux (using WINE) - haven't tried bluetooth.  If you change the setting to "mass storage" in the phone, Linux should be able to see the memory card in the phone (if installed), though it can't see the phone's memory.  {The t-439 has a slot for a micro SD card.}

Go to "settings" -> "phone settings" -> "USB settings" -> "mass storage" to change the phone.
