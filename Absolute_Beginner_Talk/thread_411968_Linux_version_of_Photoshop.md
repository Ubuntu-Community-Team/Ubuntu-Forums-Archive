### Post by blackened on 2007-04-17
GIMP also supports CMYK. Double click the foreground or background color box. In the colour selection dialog select the second tab at the top (the one with the picture of a printer on it). In the eyedropper dialog you can use the drop-down lists to change to CMYK values.

### Post by thefoolisme on 2007-04-17
Don't get me wrong, I think the Gimp is a great application, and I use it regularly.  If the Gimp works for you, that's great.  It just takes a little getting used to if you're used to Photoshop.  However, The Gimp is not a clone of Photoshop; it is missing some things a diehard photoshopper would miss.  

You can actually run version of Photoshop up through CS on linux with Wine (free), or Crossover office (not free) if you find that Gimp is missing some tools for you.  Do a search through the forums and you can find good setup guides.
