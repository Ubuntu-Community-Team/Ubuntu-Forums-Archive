### Post by notwen on 2007-06-17
kTorrent offers you the ability to select what files are available in the torrent you are opening, upon opening the torrent a pop up window will offer you a check box besides each file and/or folder.  You can select what files from said torrent you wish to actually d/l.  I use this a lot if I'm missing certain albums from certain artists, search for artist discography and only d/l the needed albums.  kTorrent is available through the repos and works fine in Gnome.  Hope that makes sense. =p
