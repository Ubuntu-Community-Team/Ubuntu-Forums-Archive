This guy named Elmosan wants to know how to use terminal to "cd" in the directory of the .exe

Well, if it's in your home folder, it would probably look like:
```

cd /home/elmosan

```

If it's on your desktop, use:
```

cd ~/Desktop

```

"cd" means Change Directory.  So just use the file manager to figure out the path to what you want and put it after "cd".  Check out the link in my sig about the terminal for more information.
