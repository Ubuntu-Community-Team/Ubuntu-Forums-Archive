### Post by bobbob1016 on 2007-12-20
I'm not 100% sure how to fix this issue, but you could try typing "boot-game-os" in a terminal instead of rebooting.  That will get you back to the PS3 XMB after Ubuntu shuts down.  If you want Ubuntu back, just go to the Settings on the XMB and select Other OS by default, and switch back and forth this way.
