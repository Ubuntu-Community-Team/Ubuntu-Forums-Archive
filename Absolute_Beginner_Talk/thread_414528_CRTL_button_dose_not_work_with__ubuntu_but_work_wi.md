### Post by Austin_KW on 2007-04-20
Not all programs will use the shortcuts to which you might be accustomed, eg some terminal use shift-insert. Check the edit menu for key bindings. Or check a program's setting/preferences to see if you can change the key binding.

If you use a mouse, try using left button (1) to select text then middle button (3) to paste.
