### Post by mimada on 2008-04-15
> Also is there a way to get all my files and windows back? i think its gone for good though 

It depends on how badly you want to recover your files. It's pretty difficult and time consuming. To do a proper recovery, you'll need a second (empty) hard drive to image into. You'll also need a [rescue disk]("http://ubuntu-rescue-remix.org/").

I also recommend having the Ubuntu Live CD handy. Be aware that this is *not* a trivial task and there's no guarantee that you'll be able to recover anything.

BTW, if you're serious about recovering files, stop using the drive with the data you lost. The more you use it, the less likely you'll be able to recover the files.
