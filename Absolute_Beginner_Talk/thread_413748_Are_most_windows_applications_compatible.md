### Post by hyper_ch on 2007-04-19
Linux is a total different thing... think of Windows and Mac... can you use a windows game on a mac? Well, you will have to get the Mac version of it...

However there are some programs that will run some windows appz on linux.
You have for example wine (which is free) and runs a fair bit of software
Then for office products (and also photoshop and stuff) you have Crossover Office (that costs something...)
And for games you have Cedega which also costs something

However they can't run anything...

As further option you can install a virtual windows system within your linux with VmWare or Virtualbox (just think that you run Linux and then within you run a Windows). This will however require some cpu and ram and you have no 3d-acceleration - although I heard it's being worked on in VmWare....

### Post by trippinnik on 2007-04-21
He's afraid he'll loose his data, or any number of other things.  Setting up a dual boot will usually work, but there could be instances where something could go wrong and you loose the windows install.  Another thing is that you'll be splitting your storage into two drives and this means you'll have less space to install programs, download stuff, etc in windows.  Most likely your dad really doesn't want to risk having to do a lot of work to fix something that is currently working (and probably acceptably for what he needs).
If you have any old computer hardware around I'd recommend trying linux out on that, it typically can run on some pretty old hardware and still be acceptable.
