### Post by Nythain on 2007-05-13
not all applications have menu entries and the likes
as far as wine, in terminal type winecfg to set it up then to run anything or install anything, in terminal type
wine /path/to/the.exe
same goes with other software... if you cant find an entry in any of your menus open up terminal and start with just typing its name
