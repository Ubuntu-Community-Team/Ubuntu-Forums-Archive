### Post by UbUsOsOlik on 2008-01-18
First install wine on your machine. Go to add/remove and search wine, check the wine and apply the install option. After the wine is installed download skype and right click the install.exe(setup.exe) and select open with wine. Affter this the install is windows style. When finished just go to applications/wine/programs and start skype. The wine program is a windows program emulator. You can use wine for other windows programs too, like windows only games.

Hope i helped.
