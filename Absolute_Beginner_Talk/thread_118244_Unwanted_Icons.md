### Post by raul99 on 2006-01-16
I recently installed ubuntu on the second HD of a Windows XP machine, after the install there appeared on the desktop labeled icons for the drive partitions
of the first NTFS windows HD, hda1, hda2, hda3, hda4. How do I remove these from my Linux desktop?

---

### Post by jasay on 2006-01-16
Applications > System Tools > Configuration Editor 

On the left:
/ > Apps > Nautilus

On the right:
uncheck volumes_visible
