### Post by neemz on 2006-12-12
If you are low on space I would recommend not installing ubuntu at all and use the live cd instead.

The live cd will run from the cd-rom and not make any changes to your harddisk.

This means you can play with ubuntu, browse the web etc, without making any changes to your computer. You can also run the live cd inside vmware if you wish not to reboot.
