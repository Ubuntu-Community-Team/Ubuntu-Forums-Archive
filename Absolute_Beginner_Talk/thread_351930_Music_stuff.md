### Post by shareMenaPeace on 2007-02-02
You just need to install the codecs as you rightnow have just ogg format support.

To do this you can either add the repositorys to your sources, which contain such codecs manualy.
[http://ubuntuforums.org/showthread.php?t=182902&highlight=repository+mp3+codec](http://ubuntuforums.org/showthread.php?t=182902&highlight=repository+mp3+codec)

Or use an automatic instalation application such as 
[http://www.getautomatix.com/wiki/index.php?title=Installation](http://www.getautomatix.com/wiki/index.php?title=Installation)
Which let you easy install much more common known tools along.
