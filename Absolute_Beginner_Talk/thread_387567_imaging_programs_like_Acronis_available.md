### Post by Jose Catre-Vandis on 2007-03-18
Yes. get yourself a copy of system rescue CD  ([www.sysresccd.org](www.sysresccd.org))

This has Gparted and Partimage (basically partition magic and drive image clones, although partimage works like a graphical dos program.

partimage will allow you to make images of your partitions, but you will need a spare fomratted partition to save them to, and you have to mount this partition before you start.

System rescue Cd is a bootable linux toolkit.
