### Post by JeffoOfMetal on 2008-03-26
I don't think there is one, but you can go *System>Administration>Users and Groups* and change it. It will ask you for your user password, and from there you can set a root password. Also, make sure that if you plan to login to root from the login screen, you must enable that by going *System>Administration>Login Window*. In the *Security* tab, you can check the checkbox that reads *Allow local system administrator login.*
