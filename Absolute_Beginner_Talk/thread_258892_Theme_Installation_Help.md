### Post by IYY on 2006-09-16
If inside these folders on your desktop you have files that end with .tar.gz, you can just drag them into that themes selection window you have open. Otherwise, you'll need to copy the themes folder into the hidden folder ~/.themes and the icons to ~/.icons
