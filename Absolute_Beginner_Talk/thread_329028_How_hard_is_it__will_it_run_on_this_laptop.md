### Post by maxamillion on 2006-12-31
If you use the liveCD (Desktop Install) and select the guided partitioner during the installation process and make sure you don't delete the windows partition (either fat32 or ntfs) you will be fine.

Also, on a system with specs like that you may enjoy using Xubuntu simply to get a little extra speed out of the hardware but its up to you.

[www.xubuntu.org](www.xubuntu.org) for more information if you are interested.
