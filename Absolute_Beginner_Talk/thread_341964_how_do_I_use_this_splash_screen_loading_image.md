### Post by Sandlst on 2007-01-19
Are you using gnome?  If so, fire up a terminal (applications/accessories/Terminal) and paste in ```
sudo apt-get install gnome-splashscreen-manager
```Then, goto (System/Prefrences/Splashscreen) and click on "install", point it to your file, then click "Activate" with it selected.  Good luck, Post back if you have any problems.
