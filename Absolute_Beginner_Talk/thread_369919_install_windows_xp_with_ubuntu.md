### Post by temba on 2007-02-25
Try vmware.

The easiest is to register for the eval version of VMware Workstation, then install xp then after 1 month when it expires, install VMware player and keep your settings like eg SVGA graphics.

Remember to install VMtools when the os has been installed.

Note that (as far as I know) specific graphic card drivers and the like are not supported on vmware.

### Post by steve.horsley on 2007-02-25
If I understand right, you want the Beryl effects on XP windows. This is not possible. If you run XP in VMware or VirtualBox, the XP desktop is a single window inside the Ubuntu desktop. So dragging the whole XP desktop will make it wobble, but individual windows inside the XP desktop won't wobble, etc.
