I don't have this Netgear adapter, but I found a post here that should help you:
[http://ubuntuforums.org/showthread.php?t=51993](http://ubuntuforums.org/showthread.php?t=51993)

As you can see from that thread, this card requires ndiswrapper, which is a linux package that basically makes the Windows drivers work.
In order to get everything up and running, you'll need a wired internet connection (such as a cat5 cable to your router) in order to download what you need.

And my suggestion for learning linux... get everything (or most things) the way you want it with the CD you have, but then re-install from scratch with 6.06 LTS Dapper Drake. You should order the FREE cd's:
[https://shipit.ubuntu.com/](https://shipit.ubuntu.com/)

You certainly don't have to do that, but Dapper has several improvements over previous versions. If you really want to learn a lot, a couple installs is great practice!
