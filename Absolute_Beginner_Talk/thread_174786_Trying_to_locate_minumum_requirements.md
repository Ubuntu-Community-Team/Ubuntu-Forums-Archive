### Post by Jimmey on 2006-05-12
For a server install, it's around 64MB RAM. ( With this option, there's no graphical desktop ).

For a default install, you need at the least, 128MB RAM. But you can configure it once installed, to be more efficient ( XFCE, Fluxbox, etc. ).
