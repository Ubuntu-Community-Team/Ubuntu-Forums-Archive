### Post by starcraft.man on 2007-05-01
Easiest way is to just find the file you want VLC to open all the time (say an avi or mpeg or whatever), right click on it and select properties. Click on the Open With tab and then select the program you would want to open it all the time, in this case vlc, then apply. This works for any type of file with an extension.
