### Post by mahiyar on 2006-12-31
You cannot just copy the plugin to any folder. The lines, which are nothing but commands, given in the boxes are supposed to be fed in from the terminal window (something like DOS). You can access terminal from Menu given on the top row. After the terminal opens you feed in those commands given in the Wiki. All system related files are to be accessed by administrator or super user, when we log in Ubuntu we log in as normal users. From a normal login if we we want to access folders or run commands meant only for the administrator we use the command "sudo" before the actual command. So you start with
adding the extra repositories given in the link here [http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_apt-get_the_easy_way_.28Synaptic.29](http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_apt-get_the_easy_way_.28Synaptic.29) the steps are very easy to follow adding these repositories will help in the future also, repositories are nothing but links from where synaptic (program add/delete manager) gets the programs, after which you can follow the link here [http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_install_Flash_Player_.28Macromedia_Flash.29_Plug-in_for_Mozilla_Firefox](http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_install_Flash_Player_.28Macromedia_Flash.29_Plug-in_for_Mozilla_Firefox) to complete the flash installation. Hope this helps.
