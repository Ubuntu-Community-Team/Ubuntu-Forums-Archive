### Post by Maestro23 on 2007-03-29
> **rsnow said:**
> I used synaptic pkg mgr to install the Kino video editing application. After syn pkg mgr did its thing, how do I access Kino so that I can use it? It was not added to any menus as far as I can tell. The Add/Remove function shows it be installed. Is there anyway to put a shortcut on the desktop?:confused:

Open a terminal:

> whereis kino (gives the path to the bin file)

kino (runs the program from command line)

Use the path to the bin file to create a desktop launcher for it. Right click on desktop and create a launcher.
