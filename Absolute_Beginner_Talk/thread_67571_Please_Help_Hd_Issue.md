### Post by sonyack on 2005-09-20
Get a Win 98 Startup Disk.  Put it in your A: drive.  Start Computer.  Press DEL to enter Setup.  Change your boot sequence to have floppy as 1st boot device.  Press F10, Y, Enter.  You will get stuff, follow it until you get an A: prompt.  Type fdisk.  Delete all partitions.  It will probably reference a "nonDOS partition."  This will wipe out everything.  Then create a dos partition for Win98.  Then format the partition.  Then put Win98 on and then upgrade.  Remember to change the boot sequence to cd-rom as needed.

If you have no Win98 Startup disk, there is a chance that you can boot from the Win98 CD and do the partitioning and formatting with that, but you should use the startup disk if available.
