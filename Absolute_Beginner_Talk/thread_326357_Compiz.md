### Post by pseudonym on 2006-12-28
> **Griffongob said:**
> then which one should I use?
I don't use compiz but I can give you some advice about managing all the choice you have in Linux. Many people will say 'try both and see which one you like'. This sounds nice, but if you are new to Linux, expect to spend a LOT more time than you would like setting things up and troubleshooting problems if you take that approach.

Some learning and set up is inevitable of course, even with windows. Best bet is to read a little about the alternatives, then pick one and stay with it. If at some later time you decide you don't like it, you can always try the alternative.
