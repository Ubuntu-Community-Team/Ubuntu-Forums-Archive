### Post by Malta paul on 2007-01-27
Perhaps most people know this? I didn't and wondered why with only a few files on my pen drive, I had hardly any space left  :confused: 
When a pen drive file is deleted [Move to Trash] the file is not put into the trash bin on the desktop but instead goes to a hidden file that is created on the pen drive itself. To view this use  Ctrl-H. The trash file and or its contents can then be deleted if required, to make more free space available. If however a file is dragged from the pen drive to the desktop the pen drive hidden trash file stays empty :D
