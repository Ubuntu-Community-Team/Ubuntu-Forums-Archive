### Post by 3rdalbum on 2006-12-16
Burn them to CD in Windows, and then re-rip like any freedom-lover :-P

You can install iTunes through Wine (this only works if you install a Windows program called Quicktime Replacement first) and play your songs. Also, it's possible to find programs online that create unprotected AAC files from DRM'ed ones, but I don't know exactly where to find them or what their names are. In some parts of the world, doing this is illegal. You don't mention which country you're in :-)
