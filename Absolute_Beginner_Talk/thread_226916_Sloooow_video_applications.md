### Post by Ziox on 2006-07-31
did a little googling of my own :)

see if adding these two lines to xorg.conf will work:

Option          "NoAccel"       "false"
       Option          "DRI"           "true"

add this under the "device" section

so it looks like this:

Section "Device"
	Identifier	"Intel Corporation 82845G/GL[Brookdale-G]/GE Chipset Integrated Graphics Device"
	Driver		"i810"
	BusID		"PCI:0:2:0"
	VideoRam	8192
	Option		"UseFBDev"		"true"
      Option          "NoAccel"       "false"
      Option          "DRI"           "true"
EndSection

reboot or restart x by logging out and hitting ctrl + alt+ backspace
