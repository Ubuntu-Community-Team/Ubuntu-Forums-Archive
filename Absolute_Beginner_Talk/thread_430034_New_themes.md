### Post by hellfried on 2007-05-01

### Post by mahiyar on 2007-05-02
Instead of going in to a generalized install theme you might want to go to specific type e.g if it is a icon theme you want to install go to theme detail then icon tab and then install theme. Never ever install a mouse pointer theme, I learned that the hard way, just move the file to your ~/.icons directory.
