### Post by meat on 2005-06-11
It's been about a week since I installed Ubuntu and I continue to be amazed at what I can do with it and how most things work fine.  I installed it on an AMD machine and had a few issues getting my resolution right, but everything worked well, so I thought I would try a dual boot on my main machine.  I was a little hesitant, because I have an ATI PCI-e card and SATA hard drives.  Well after reading the guide and searching, I found fixes and got everything working great.  I even synced my Palm, got my mouse buttons all working and am able to play my mp3's from my Windows Partition.  I went 4 days without Windows.

Next I will install Ubuntu on a laptop and figure I will have some issues with getting the wireless NIC to work, but I see there is a lot of documentation on it, so I should be set.

People who are on the fence about trying Linux should go for it.  There is so much useful documentation here, that chances are, someone has experienced the same problems as you and posted a fix or work around.  There is more stuff that worked right out of the box than I thought there would be.

Thanks for the help and all of the documetation on this site.

### Post by ~J~ on 2005-06-12
Thought I'd add my little experience to this...

I installed Ubuntu over a week ago, installation went flawless and was pretty impressed with how it picked up everything (including my wireless network!) with no drivers at all!  The only thing it asked for in terms of installation was a partition, which ethernet socket to use (the inbuilt one, or the PCMCIA card) and the WEP key!

At first I didn't like Gnome, thought it a little to basic, and so installed KDE.  KDE was smart, very smart, but seemed a little slow in places and not 100% consistant.  For example when accessing certain admin options KDE would ask for the password and then nothing.  I had to select the option twice to start it running.  Or other times I'd click on the OK button and nothing would happen.

The other problem I had with KDE was that the mouse control was AWFUL!  The pointer would speed around the screen, seemed to judder and no amount of tweaking in the control panel would get it to react like the windows smooth version I'd been accustomed to.

As a test, I reinstalled Linux, 20 mins later was using Gnome again and finding new tools to make it a more pleasant experience.  None of the problems of speed or inconsitancy that I'd seen in KDE were present in Gnome.

So I'm sticking with Gnome for the foreseeable future.

As for linux itself.  Well...

Web browsing feels far far better that windows.  It feels part of the system rather than an add-on, and the speed is far faster on linux than on windows.

One of the primary reasons for switching is an app I use frequently called "Blender" which is a million times faster and more stable on linux than windows.  I also want to learn PHP and python which is a much better environment than the windows too.

I'm still new to it all, some areas are causing me concern such as why on earth the screen savers are soooooooo slooooooooow compared to windows ones, and why some of the installation packages on the synaptic installer are out of date and I can't use the "force version" to update it.

I'm not ready to make the switch 100% yet, I doubt I will for a while, but I'm certainly impressed and do seriously regret not switching sooner.
