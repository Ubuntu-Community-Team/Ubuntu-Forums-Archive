### Post by Tomosaur on 2007-01-24
Well, Firefox is not Ubuntu, that's pretty much why. The NoScript extension is to make **Firefox** more secure. If a script were to exploit a vulnerability in firefox, it would most likely be a browser based attack (ie, it might steal your web history, or passwords, or whatever). In this case, your OS is pretty much irrelevant, since the attack will go no further. In those rare cases where the exploit allows remote system access - Linux is inherently more secure than Windows. Unless you're stupid enough to run firefox as root, damage is going to pretty limited, if existant at all. If the extension was needed, then it would be in Firefox by default. It isn't, so it's not. If you want it, use it.
