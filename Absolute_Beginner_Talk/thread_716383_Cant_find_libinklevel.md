### Post by Vadi on 2008-03-05
The "lib" in front of a package means that it's a library. And libraries are meant to be used by programs; they're just a bunch of functions that a program can use. So you yourself can't use the library - need to find a program that does use the it.

So, searching for "inklevel" in synaptic for "description and name", I found the "ink" and "qink" packages. Give those programs a try; they're the ones that make use of the library.

(eh, that seems long. In short - get the ink and qink package, and use them)
