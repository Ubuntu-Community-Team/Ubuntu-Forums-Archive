### Post by Rocket2DMn on 2008-02-08
If you are dual booting, Vista is probably using this partition.  If it's not really a problem, I would just leave it.  If you ever go for a linux-only install, you could delete that partition and merge it with your root filesystem, otherwise Vista will probably yell at you if you delete it.

### Post by Pogeymanz on 2008-02-08
I have an HP with a recovery partition with WinXP, so it might not be exactly the same, but I can't see why not.

If you boot that partition, you have the option to restore your hard drive to the factory settings. In other words: it deletes all your stuff and reinstalls Windows. 

You can create a set of recovery disks from within Windows that are the exact same thing. I did that and deleted the recovery partition. Honestly, if my HDD screws up, I'd probably just install Ubuntu over the whole thing anyway.
