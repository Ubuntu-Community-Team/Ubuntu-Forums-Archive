### Post by bodhi.zazen on 2006-09-04
Open the file with nano and edit.

Did you backup?

In a terminal:
```
sudo nano /etc/X11/xorg.conf
```

Edit this section
> Section "Device"
        Identifier      "NVIDIA Corporation NV15 [GeForce2 GTS/Pro]"
        Driver          "nv"
        BusID           "PCI:3:0:0"
EndSection

To look like this
```
Section "Device"
        Identifier      "NVIDIA Corporation NV15 [GeForce2 GTS/Pro]"
        Driver          **"nvidia"**
        BusID           "PCI:3:0:0"
EndSection
```

to exit nano (after editing) -> Ctrl-X, type Y to save changes

Now Ctrl-alt-Backspace -> re-login -> viola, no?
