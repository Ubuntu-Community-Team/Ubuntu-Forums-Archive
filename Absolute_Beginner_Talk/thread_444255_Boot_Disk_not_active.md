### Post by taurus on 2007-05-15
1.  How did you burn the ISO image that you downloaded?  Did you burn it as an image instead of just copy it from your harddrive over to a CD?  If you look at the CD while in Windows, you should see a bunch of files and directories on it.  If you see one large file, then you've burnt it wrong.

2.  Did you go into the BIOS to make sure that you have CD-ROM in the first order in the boot order?

### Post by heimo on 2007-05-15
Two things to check
[LIST=1]
[*]You actually burned the .iso file as an image, did not extract files from archive or burn the .iso file as itself to CD or DVD
[*]Change boot order so that CD or DVD drive is the first boot device or selected manually, just right after you've turned your computer on[LIST]
[*]there's usually message on screen telling which key to press to get to the boot order / boot sequence / selecting boot device menu
[*]or you may need to hit some key to enter BIOS settings and select boot device order there - your computer manual / manufacturers support website may be helpful[/LIST][/LIST]Tell us how it goes, and hopefully we can help you to get Ubuntu installed! :) And welcome on Ubuntuforums!
