### Post by confused57 on 2006-05-11
Try Smart Boot Manager:

[https://wiki.ubuntu.com/SmartBootManagerHowto](https://wiki.ubuntu.com/SmartBootManagerHowto)

You can download the "smartbootmanager" file and "rawrite" to the same folder in windows.  Extract the rawrite file, double-click on the .exe file and it's pretty self-explanatory from there.  Rawrite will write the sbm file "as an image" onto a floppy.  Then bootup your computer with the floppy and install cd both inserted.
SBM will start. displaying a menu of boot options, arrow down to CD Rom. I think you press "F2" to save, then enter.  I usually get an error message the 1st time, you may have to repeat a couple of times to get it to finally boot from CD.

I don't know if you need it, but I already had WinZip installed to extract the rawrite zip.  Guess you could install the free trial version of WinZip, good for 30 days, then uninstall after you've used it.
