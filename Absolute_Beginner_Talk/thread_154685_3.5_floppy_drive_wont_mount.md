### Post by goatflyer on 2006-04-03
Insert the blank (unformatted) floppy into the drive

goto Places>Computer
double click Floppy icon to open it
right click, select Format
Choose DOS (FAT), not EXT2 if you want to share with PCs.

EDITED TO ADD THE FOLLOWING:

if it is already formated, try this:
open terminal window
```

sudo mount -t vfat /dev/fd0 /media/floppy0

```

### Post by meborc on 2006-04-04
[QUOTE=mr.champagne]crap now it says i dont have permission to copy into the floppy drive[/QUOTE]

of course you don't have the permission... as you mounted as **sudo** you must use sudo for copying... do it from the terminal with **sudo cp** command or open nautilus in sudo (**sudo nautilus**) to copy your files... BUT be careful when running sudo nautilus... you have the power of god over your machine then...

i usually don't use sudo mount... but just mount... it's just a floppy for garden's sake ;)
