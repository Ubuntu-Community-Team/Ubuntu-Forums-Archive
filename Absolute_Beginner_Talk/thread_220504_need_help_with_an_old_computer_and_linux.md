### Post by monkieie on 2006-07-21
yep make sure to keep the delete button held during startup - that's how to access the BIOS config screen. From there, depending on the BIOS type etc.. there will be a menu option where you can change the boot parameters

eg, 
A, SCSI, C
CD ROM, C, A

etc etc....make sure that the first option is "CD ROM".

Save your changes and then reboot.
