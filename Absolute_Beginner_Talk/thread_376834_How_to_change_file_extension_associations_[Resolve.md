### Post by ardchoille42 on 2007-03-05
> **radioouman said:**
> I am using Ubuntu (Gnome)

Right click the file, choose Properties. Go to the Open With tab and choose the app you want to open that file type by default. You can add apps in that list if the one you want isn't there.
