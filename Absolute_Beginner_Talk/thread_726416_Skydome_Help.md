### Post by Fate Reconciled on 2008-03-17
Under CompizConfig/Advanced Desktop settings make sure to:

-Enable Desktop Cube
-Enable Rotate Cube
-Enable Cube Caps

Also, go to Configure Desktop and click on the Multiple Desktops tab and set the number of virtual Desktops to 4. I think this is set to 4 by default, but make sure anyway.

This is all I really know, but I hope it helps to an extent. I'm not aware of any solid tutorials either. Maybe another user can help you with that.
