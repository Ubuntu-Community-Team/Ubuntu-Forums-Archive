### Post by kwilliam on 2007-08-25
Just go to "Control Center > Regional & Accessibility > Keyboard Shortcuts".  Under the "Command Shortcuts" tab, find the terminal program and add a keyboard shortcut.
You might just want to install [Yakuake]("http://yakuake.uv.ro/").  It is a KDE terminal that does exactly what you're asking for, plus has some spiffy features like transparency.

If you want to do some more advanced keyboard stuff...
Go to "Control Center > Regional & Accessibility > Input Actions"
Click "New Action" and you can define more advanced triggers, such as "If the active window is Kaffeine, when the laptop's Pause button is pushed, pause Kaffeine".
