### Post by mgmiller on 2007-12-07
If you can copy the video files to your hard drive, try renaming them and don't use a file suffix.  

For example, if the file is named videofoo.asf  try renaming it videofoo

Don't use a period or anything else after the file name.  

Totem may play it after doing that without giving the warning that it is a file type of one thing but appears to be something else and then refusing to play the file.

For most video and image files like .jpg .gif, etc.  you don't need any file suffix in Linux.
