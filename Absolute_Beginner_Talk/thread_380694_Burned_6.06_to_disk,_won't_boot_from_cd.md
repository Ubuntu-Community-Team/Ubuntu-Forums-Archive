### Post by rsambuca on 2007-03-10
Did you burn the cd as an image iso?  If you burned the CD as a regular data disk it will not work.  When in XP, put the CD in and open it up in explorer.  If you see a bunch of files and directories it should be OK.  If you see just one big file, you burned it incorrectly.
