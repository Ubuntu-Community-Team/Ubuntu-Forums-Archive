### Post by Pablo_Escobar on 2005-10-24
sudo - executes command with root (super user) privileges.
You type in terminal "sudo *command you want to run*". It's a good thing to use it carefully, because it keeps You from messing up Your system.
As for games - There is a emulator called "Cedega" which can be used to play games. You have to pay for it though (5$ per month). Google cedega - and it'll be the first link.
