### Post by nickle on 2006-02-17
Disadvantages:
[LIST]
[*]Gaming is restricted
[*]Not all hardware is supported. This is particularly so for some very new hardware. There is usually a lag time of weeks, mts or years depending on the hardware in question. It is always worth checking that your hardware is supported. Saying that neverthelss a vast amount of hardware IS supported
[*]You probably have to learn something about the OS.  This will involve following configuration instructions, entering commands on the command line and sometimes editing configuration files. I am not sure this is a disadvantage though, if you like that kind of thing
[/LIST]
Despite the above, desktop linux has greatly improved in the last years. For example, the only issue I had with installation was with my wlan card. Everything else worked straight from a menu.:p
