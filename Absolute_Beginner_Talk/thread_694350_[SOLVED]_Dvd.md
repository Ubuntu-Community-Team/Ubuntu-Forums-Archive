### Post by emarkd on 2008-02-11
Sorry about that.  Don't worry about it.  Click on System > Administration > Synaptic Package Manager.  This is the best way to manage the software installed on your system.  Linux software is kept in repositories and you install it by just searching the repository (with Synaptic) for the software you want to install.

First, you'll need to enable all of the repositories.  On the menu, select Settings > Repositories and click all the boxes except source code or cd.  Save it.  Then, back on the main screen, hit Reload so that Synaptic reads the new repositories to see what's in them.  

Now you can search for ubuntu-restricted extras and install it.  You may also need libdvdcss2.  That should take care of it for you, but if not, just ask and we'll try to help.
