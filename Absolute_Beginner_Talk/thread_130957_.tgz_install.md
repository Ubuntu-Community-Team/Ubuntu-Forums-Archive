### Post by steve.horsley on 2006-02-15
Firstly, check the multiverse and universe packages to see if you can install with synaptic. From Synaptic packafge manager, go to the repository settings and choose to show disabled packages. Then you can add the multiverse and universe packages as sources. Then search for your game. 

Second, look in the extracted tgz contents for a README file. The contents are probably a set of source code files that you need to compile with these commands:> 
./configure
make
sudo make install
or are a set of binary files that you put in a directory somewhere (eg. /opt/mygame) and just run from there.
