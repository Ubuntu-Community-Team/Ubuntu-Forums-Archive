### Post by public_void on 2005-11-28
This is how I did it.

apt-get install gcursor or use synaptic package manager to install gcursor. This will add Cursor Selection in Preferences. Download what even cursor you want as a .tar.gz. Now install it using Preference -> Theme. Going to install theme and point to the file you downloaded. It should say the theme was installed correctly. Now go to Preferences -> Cursor Selection and you should see the cursor you downloaded, select it and your done.
