### Post by mcduck on 2007-06-06
In windows that's common sense, in Linux you are being paranoid ;)

There is no spyware,  no viruses that would be any risk, and as long as you don't install any server software you don't even really need a firewall. Also the package management is smart enough so there really isn't much need for any cleaning software (just run 'sudo apt-get clean' every now and then to remove cached package files if you are low on disk space). And of course no need for defragmenting..

If you want a firewall, you could install Firestarter. It's a easy tool for configuring the built-in firewall of Linux kernel.

### Post by Happy_Man on 2007-06-06
Linux is very secure. There are no known viruses, any Windows viruses will not work, there is no need for defragging the HDD, very easy to recover your system, etc. There are firewalls and antiviruses, but these are mainly for if you are using your box as a server, to block outgoing viruses that are routed through.
