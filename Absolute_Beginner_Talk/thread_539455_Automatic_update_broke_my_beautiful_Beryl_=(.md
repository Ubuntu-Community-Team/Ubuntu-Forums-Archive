### Post by b0ng0 on 2007-08-31
Try a complete removal of Beryl and reinstall using the latest version. If this doesn't work, try enabling "Desktop Effects" from the Gnome Menu->Preferences and see if this works. If it does then you had a problem with Beryl, if you still have the same problem then it's something else. 

Sorry I can't be more precise but it's best to work out exactly what isn't working first (since you had 31 updates could be more than one thing..).
