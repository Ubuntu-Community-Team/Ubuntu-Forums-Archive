### Post by lucia_engel on 2006-09-08
The short answer is, you shouldn't expect to play windows game in Linux. You can check the WineHQ application list [http://www.winehq.com/]("http://www.winehq.com/")(?) and there will be reviews on how your game will run using WINE. However, it's usually very slow or even unplayable. There are also Cedaga but you will have to pay for the that.

Basically, if you want to run Window PC games with the same performance as you're used to, I'd suggest you to dual boot your machine with Windows/Ubuntu and switch to Windows whenever you want to play games.
