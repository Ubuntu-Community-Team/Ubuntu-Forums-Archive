### Post by az on 2006-05-27
[QUOTE=SZF2001] so I can complete this installation in the terminal:
[/QUOTE]
Install ndiswrapper-utils.  It is on the cd.  You already have the modules as part of your stock ubuntu kernel.

If ndiswrapper is not working for you, it is probably because you have an olf (or wrong) .inf file.  Get a newer one.  Your problem probably won't go away by recompiling ndiswrapper.
