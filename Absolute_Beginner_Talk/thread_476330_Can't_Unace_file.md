### Post by takuhii on 2007-06-23
I have figured it out, sort-of... It seems the ACE utility that comes with Ubuntu/Automatix can't cope with the newer versions of ACE, downloading this from WINACE.COM fixes the problem, but only from the command line, for some reason it puts the UnAce software (file roller) into some kind of loop. I fixed the problem by installing some software called Squeeze, this seems to handle newer ACE file fine...
