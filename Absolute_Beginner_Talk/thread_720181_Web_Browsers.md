### Post by erginemr on 2008-03-10
+1 for Opera. You can use it as a secondary web browser application alongside Firefox. Opera bookmarks is located in a single file "opera*.adr" probably inside a subfolder (".opera" maybe) in your home.

To speed up Firefox, you can get the "fasterfox" extension. Alternatively, you can give a try to Swiftfox, an off-shoot of Firefox aiming for a better performance.
