### Post by ubuntu27 on 2008-02-14
Azureus uses Java isn't it?
Then you will want to install Java:

```
sudo apt-get install ubuntu-restricted-extras
```

Then you can install Azureus:

```
sudo apt-get install azureus
```

Those commands have to be entered in the terminal which can be found at Applications menus/Accessories/Terminal


Also check out my link number four in my signature so you can learn how to install application in Ubuntu. (There are many ways to install)
