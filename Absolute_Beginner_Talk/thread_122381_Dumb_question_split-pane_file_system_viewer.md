### Post by Madpilot on 2006-01-27
You don't need to install anything to get a Windows-Explorer style split-pane file manager, Nautilus will do it just fine.

Open Nautilus (Places menu >> Home opens it) and look at the panel on the left-hand side. By default I think the title is "Places" and there's an arrow pointing down next to it - click that arrow, and select "Tree" - voila, one Explorer-style directory tree!

The lefthand panel in Nautilus can actually do a bunch of things - explore some of the other options in that dropdown menu. "Tree" really should be the default setting, though, not "Places".
