### Post by hums07 on 2007-12-29
If your Windows is working fine then you dont need a Windows install disc. You can install Ubuntu without affecting your existing system, well **IF the process runs smoothly** as it usually does.
You just need an Ubuntu install CD.
Just defrag your harddisk , in Windows: Start menu > All Program > Accessories  > System Tools > Disk Defragmenter. Or you can also use JkDefrag (download from [this site]("http://www.kessels.com/JkDefrag/")).
Install Ubuntu following [this guide]("https://help.ubuntu.com/community/GraphicalInstall").

Backup your data before starting. If something goes wrong, you can come back to this community which is more than pleased to help.
