### Post by Bartender on 2006-12-04
I tried Edgy Xubuntu for a week on my old test PC.  I haven't given up on it; thinking about using Xubuntu even if I settle into a newer PC.  The default desktop is pleasant.  Desktop icons are not a simple drag-and-drop affair.  Some icons can be brought over from - um - don't remember the folder.  I didn't figure out the "launcher" thingie but don't believe it's that difficult.  
Xubuntu's dial-up modem utility isn't as handy as Dapper's but if you're not on dial-up you won't care.  Xubuntu doesn't come with OpenOffice but you can always install from Synaptic if you wish.  OpenOffice is, relatively speaking, a resource hog so it kinds makes sense that it's not included.

CPU usage at idle was about 1% with Xubuntu.  It'd hung around 7% with Dapper.  Applications seemed to come up about 40 to 50% faster.  Feels snappier all around than Ubuntu or Kubuntu.

### Post by hyper_ch on 2006-12-04
well, best ist just to try. Open a terminal and enter:

sudo aptitude install xubuntu-desktop

Then you will have both installed :) and you can select the one you want to open a session with from your session manager...

If you don't want Xfce anymore just open again a terminal and enter:

sudo aptitude remove xubuntu-desktop
