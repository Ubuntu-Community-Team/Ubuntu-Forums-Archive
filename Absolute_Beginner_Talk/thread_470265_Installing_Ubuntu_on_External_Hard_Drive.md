I have just had some success installing Ubuntu on an external hard drive over USB 2.0. I can't tell any differences speed-wise between an internal and external drive, and I just wanted to let everyone know that this works great. I wanted to install Ubuntu for a while, but I couldn't afford a new computer. I also couldn't wipe my hard drive and install a new OS. This is a cost-effective alternative, so if you want to try out Ubuntu, go for it. It works much better than Live CD mode.
