### Post by inkybutton on 2007-01-05
Hi and welcome to Ubuntu forums! I suppose that you are trying to install Ubuntu:
Follow the steps:

1. Get the latest Ubuntu by downloading it in: [http://www.ubuntu.com/products/GetUbuntu/download?action=show&redirect=download]("http://www.ubuntu.com/products/GetUbuntu/download?action=show&redirect=download")
It's around 700MB, which is big, it might take long depending on your internet speed.

If you already have the installation disc go to step 3.

2. Burn the image file (ISO) downloaded from the above website to a CD [(see how to do it in detail )]("https://help.ubuntu.com/community/BurningIsoHowto")

3. Restart the computer and boot from the CD. (To boot the CD, you need to change some CMOS settings (also known BIOS settings). You can do so by hitting either ESC, F2, DEL or other keys depending on you machine when computer is still starting.

4. Choose "Start or Install Ubuntu".

5. Wait until Ubuntu has done its work.

6. You can either choose "Install" to install Ubuntu, or to look around and see what Ubuntu has got.

For more information, read [Switching to Ubuntu from Windows]("https://help.ubuntu.com/community/SwitchingToUbuntu/FromWindows").

If you need more help feel free to ask!

### Post by inkybutton on 2007-01-05
Java? I thought you want to install Ubuntu...

Anyway if you want to install Java Runtime Environment (which is for running Java applications such as Runescape :) ) in Ubuntu, go to Applications > Add/Remove Applications, type in the search keyword "Java". Tick the options "Sun Java 5.0 Runtime", and optionally, "Sun Java 5.0 Plugin". Click on "Apply", and you are done.
