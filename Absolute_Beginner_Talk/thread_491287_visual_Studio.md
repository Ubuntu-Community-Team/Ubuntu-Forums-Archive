### Post by mlentink on 2007-07-03
> Alcohol 120%, is a powerful ***Windows*** application that makes it easy to create backups of DVDs* and CDs. In addition, the program lets you store your most used CDs as images on your computer, so you can call them up at the click of a button. 
Visual Studio is a ***Windows*** dev environment. DirectX is for ***Windows***

Ubuntu is *not windows*, nor does it want to be. 

I would suggest running a dual boot setup so that you will continue to be able to run your favorite Windows apps in their native environment.
At the same time. there are excellent Linux alternatives for the apps you mentioned.
