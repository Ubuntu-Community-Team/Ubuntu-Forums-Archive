### Post by Brunellus on 2006-06-21
I'm not sure.  Does outlook have an "export" command?  If so, you'll need to export your previous messages in a format that Evolution (ubuntu's default mail client) can read.   that format is .mbx

So you'll need to run Outlook, export the files, and then import the resulting .mbx file into Evolution.
