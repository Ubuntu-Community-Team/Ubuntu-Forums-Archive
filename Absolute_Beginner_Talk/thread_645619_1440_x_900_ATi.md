### Post by Joshua on 2007-12-20
Well, I was trying to figure out how you could get it to work with a tool that's already built into Feisty, but I haven't had much luck.  The best thing I could come up with was [this howto]("http://wiki.cchtml.com/index.php/Ubuntu_Feisty_Installation_Guide").

In order to make sure it works, you'll probably have to do the manual install method (Method 2), but I've found that in the past that can make it more difficult to maintain than when everything is supported by Ubuntu.  When you find out that Ubuntu has support for the x1200 built in, you can remove the ATI stuff that you install from this howto and then use the Ubuntu tools to reinstall.
