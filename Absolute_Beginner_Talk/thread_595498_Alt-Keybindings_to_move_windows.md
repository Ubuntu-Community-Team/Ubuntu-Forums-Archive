### Post by Me887799 on 2007-10-28
I mean the binding where when you press alt, and then click, it moves the window.
[COLOR="Red"]
EDIT[/COLOR]

Haha, it was something to do with Compiz.  If anyone has this problem (Its a mean one in photoshop, where a lot things use the Alt key to select things for the clone tool, and other various things) I went to system -> preferences -> Advanced desktop effect settings -> Scroll down until you see Window Management and go into Move Window and go to the Actions tab and remap from there.  I set my button to <super>button1, be careful though, if you mess up then all you can do is move a window when you click something.  If that happened you have to tab down until you select the "Enable move windows" box and press the space bar and redo it from there.
