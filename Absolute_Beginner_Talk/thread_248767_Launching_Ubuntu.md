### Post by bodhi.zazen on 2006-09-01
> **Ghese said:**
> I just downloaded the latest Ubuntu (ISO) on to my laptop (XP), extracted it,  and ran the 'start.exe' but nothing happens.  I'm totally new to Linux.  How do I get this thing running?  Do I have to change some settings on WINDOWS?

Use the burn & boot technique:

1. Check the md5sum. (you can skip this for now if you do not know what a md5sum is).

2. burn the iso onto a cd.

3. put cd into cd-rom drive.

4. Re-boot computer.

5. If your box does not boot to CD B4 the hard drive (ie you boot to windows): Re-boot -> hit ? F2 or F8 or whatver -> enter bios -> set bios to boot to:

1. CD ROM
2. Floppy
3. Hard drive.
