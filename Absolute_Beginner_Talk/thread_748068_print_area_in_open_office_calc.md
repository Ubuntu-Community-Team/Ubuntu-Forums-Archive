### Post by drs305 on 2008-04-07
You can highlight the range, then enter a name leftmost window in the formula bar. If you don't see the formula bar, select View, formula bar. 
You can also select Format, Print Ranges, Define to create a range.
Once you have a range defined, you can type the name in the far left box of the formula toolbar and it should highlight the selected range.
To print the highlighted selection, you should be able to use the Print Selection function of your printer.

### Post by Jim_in_Germany on 2008-04-07
Hi,

Thanks for the tip.

I did what you said and got the solution. Here it is in case anyone else stumbles over this thread:

First highlight a section of your spreadsheet, then go to format -> page.
A dialog box will appear.
In the dialog box select sheet and under scaling mode select "Fit print range(s) on number of pages"
