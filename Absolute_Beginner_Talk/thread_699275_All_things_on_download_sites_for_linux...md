### Post by biohypnotix on 2008-02-17
In Ubuntu it's better to install software directly from the official repositories. If you want something that isn't there you could try [www.getdeb.net](www.getdeb.net).

Read more about installing software in Ubuntu: [https://help.ubuntu.com/community/InstallingSoftware](https://help.ubuntu.com/community/InstallingSoftware)

You can install Opera and BMPx on Ubuntu easily. Pidgin is already on the latest version of Ubuntu by default under the Internet menu.

Open up the terminal and type the following commands for Opera and BMPx:

> 
sudo aptitude update

sudo aptitude install opera

sudo aptitude install bmpx
