### Post by Schalken on 2007-04-28
You can open multiple torrents with the builtin torrent downloader. Just double click on a second one.

To set a different application to open .torrent files right click > properties > openwith and check the new app's option box. you can add applications that aren't there with "Add" below the list, and then "> Use a custom command" in the dialogue that opens.
