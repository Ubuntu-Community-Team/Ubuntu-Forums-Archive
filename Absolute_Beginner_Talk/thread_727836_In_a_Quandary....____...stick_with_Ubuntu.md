### Post by enchantedsky on 2008-03-18
I agree with much of the sentiment in this thread. However, I am against the idea of operating system exclusivity. It is almost like organized religion, where a religion may state that in order to follow it, you cannot follow anything else. No one should "switch from Windows to Ubuntu" and use Ubuntu alone. It's to your advantage to use both OSes to its fullest capacity.

Why should people just adopt one operating system for their computer? Granted, there are reasons to dislike Microsoft, but the vast majority of people in the world run Windows on their computer. We have to accept that as fact rather than fiction. There may be days when you need to run a Windows program, and it doesn't work in Wine. What if Ubuntu fails one day, you can have a secondary OS to run to check email, run applications, etc.

I advocate dual-booting Ubuntu and Windows XP/Vista, and assigning one to be the primary OS for a computer, and the second one to be the secondary OS.

Ubuntu is maturing more and more with each release, and people are relying less on the terminal to get things done. I no longer have to go to the terminal and edit some X-org file to get 1440x900 resolution on my Nvidia card. But the terminal will always exist for intermediate/expert users ------ that is the heart of a Unix-like operating system.
