### Post by p_quarles on 2007-07-05
First, you left-click on the network icon, and it will give you a list of available networks. If the router you want to use doesn't broadcast it's ssid, you'll have to manually enter that information. If it's encrypted, then a dialogue will automatically come up asking you for the passkey.

### Post by Mopana on 2007-07-05
are you using network manager applet? Can you click on an icon in your panel and select a new wireless network? You should be able to select the network, and the network manager applet will automatically ask you for the key. Otherwise, you could open Network Settings (System->Administration) and manually set the wired network up there.
