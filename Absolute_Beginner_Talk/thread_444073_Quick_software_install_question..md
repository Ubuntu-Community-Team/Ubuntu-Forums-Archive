### Post by RomeReactor on 2007-05-15
Hi. GNU Backgammon is available through Synaptic; you just need to enable Universe repositories. Both packages you mentioned from the site are RPM's, so the only way to install them would be to convert them to .DEB with a program called Alien. But there's no need. Just use Synaptic.
