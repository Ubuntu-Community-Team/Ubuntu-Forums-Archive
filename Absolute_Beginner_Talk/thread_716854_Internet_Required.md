### Post by Meatshield on 2008-03-06
It can't hurt to boot off the liveCD and check it out. You don't have to install anything if you don't want to.

And while it's completely possible to use Ubuntu without the Internet depending on what you want to do with it, if you want to install programs or packages that aren't on the CD (or update any of the packages that are on the CD), then you're going to need Internet access.
