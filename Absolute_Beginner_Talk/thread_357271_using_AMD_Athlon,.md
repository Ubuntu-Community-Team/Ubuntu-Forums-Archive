### Post by tonyr1988 on 2007-02-10
When looking at problems dealing with AMD64, the 64-bit is giving you the problem, not the AMD. If you're not too far into your Ubuntu install, I'd recommend going with the i386 version instead. I did on my AMD64 processor, and I haven't noticed a huge difference in performance (and TONS more applications run on it).

But if you want to keep running 64-bit, you may not be completely out of luck, but you'll need to do some more tweaking:

[http://wiki.winehq.org/WineOn64bit#head-56206e8bc74083807ffe06ccb471d3f964cb670a](http://wiki.winehq.org/WineOn64bit#head-56206e8bc74083807ffe06ccb471d3f964cb670a)
