### Post by jjalocha on 2007-12-18
The Epson Stylus CX3900 prints out of the box since Ubuntu Feisty. Previous to that, you have to set it up with the web interface ([http://localhost:631](http://localhost:631)). There's no CX3900 driver shipped, but you can use CX3805 simplified (recommended) or other similar, such as CX3700.

It is not hard to make the scanner work, either. Check out [thread 410220]("http://ubuntuforums.org/showpost.php?p=2656092&postcount=15").
