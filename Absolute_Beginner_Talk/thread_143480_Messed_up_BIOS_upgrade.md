### Post by bscbrit on 2006-03-12
This is not an Ubuntu question - but my only advice is:

If you kept a copy of the original BIOS (you _did_ keep a copy didn't you?) reflash it back and see if that regains your mobo.

If you haven't got the original BIOS, and your reflashing software runs from a floppy, try the procedure again.

If neither of these work, try saving up for a new mobo. :cry: 

Was there a particular reason why you had to upgrade the BIOS to get Ubuntu to work?

### Post by mcduck on 2006-03-12
I suppose reflashing isn't possible if you did reboot the machine after failed flash. You can't boot a machine with bad BIOS. But you could take your motherboard to some repair shop, so they can replace the BIOS chip or flash it for you. Or you could call to your motherboard manufacturers support and ask them to send you a new chip. I think there are also some web shops that sell pre-flashed BIOS chips.
