### Post by kesomir on 2007-12-23
1. try booting in recovery mode and see if you can get terminal access.

If you can - maybe it's compiz? If so do a search re: disabling compiz from the command line, or getting the gpu driver right for X.

2. Try using the non-live cd to install, or try out a different version - I'd recommend fluxbuntu for a machine with that spec since it whistles along on my dell inspiron 8200 pentium 4m with 256 meg of ram. It does take some learning though - if you're comfortable with the terminal you can still get everything done while going through the curve.


Sorry about my use of terminology - I've assumed you're fami.iar with them. If not, a quick google / wikipedia search should come through.

Best of luck.
