### Post by stuporglue on 2005-12-31
> wat is synaptic?

Synaptic is a nice click-and-point installer for almost any software you'd want for Linux. 

It's in your top menu:

System-->Administration-->Synaptic package manager


Synaptic gets the lists of software from servers called "repositories". You'll need to add the multiverse repository in Synaptic before you can install non-Free (ie. patent encumbered, or non-Open Source) software such as DVD  playing software and mp3 codecs.
