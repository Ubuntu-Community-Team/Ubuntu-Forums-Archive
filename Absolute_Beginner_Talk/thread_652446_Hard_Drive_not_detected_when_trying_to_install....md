### Post by mejy on 2007-12-28
Has he installed Windows on the laptop before?  It sounds very like a driver issue, which I've found to be a problem with Windows on laptops before.  Do you know what service pack the install disk is?  If it's any version prior to SP1, it will probably be fixed by updating the CD - look into [this]("http://www.winsupersite.com/showcase/windowsxp_sp2_slipstream.asp") article for instructions.

If that's beyond his reach, you could try and do it for him if you have an XP partition or buy an OEM XP SP2 CD (they're very cheap but difficult to find - perhaps ebay).  Of course, there are also less... legitimate means to acquire such a CD.

It's also likely that IBM supplied some sort of driver CD or floppy disk with the laptop that contained the driver, which it's possible to specify from the install CD.  If he can't find that or wasn't provided one, it may be possible to download it from the official website.

Oh, and it's quite possible I'm barking up completely the wrong tree - is it recognised by a Linux live CD?  If not, then it's probably something else entirely.
