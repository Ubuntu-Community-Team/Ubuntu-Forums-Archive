### Post by wagerrard on 2006-08-11
The difference between Ubuntu's single CD and Fedora's four is simply a result of how each distribution has decided to,ummm, distribute their product.  All Linux distributions draw from the same pool and make available essentially the same programs. 

Fedora's default install leaves most of the programs on those 4 CD's on the CD's.  

Both Fedora and Ubuntu expect you to flesh out your intallation by selecting from a long menu of programs.  Fedora offers (as opposed to forces) that during the install, Ubuntu assumes you'll be able to pull them down via the net. Neither distribution is bloated.
