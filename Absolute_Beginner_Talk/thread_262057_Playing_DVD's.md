### Post by benfindlay on 2006-09-21
You need to install various software an codecs (specifically libdvdcss I believe) for ubuntu to be able to access dvds. I'd recommend installing automatix or easyubuntu (or a combination of both) in order to get it to work. I personally have both installed, since automatix is good for other things too, but I use easyubuntu to play movies

### Post by nick_platino on 2006-09-21
In my case, I installed the following packages: libdvdcss2, libdvdnav4, w32codecs & libdvdplay0. After that, I was able to play encrypted DVDs exactly the same way I did in Windows with Xine. I Hope it helps.
