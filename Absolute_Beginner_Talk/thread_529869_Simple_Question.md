### Post by Armman on 2007-08-19
How do you change to a directory in terminal?

---

### Post by Golyadkin on 2007-08-19
With the 'cd' command, which is short for 'change directory'.

Example:
> 
/usr $ cd directory
/usr/directory$ cd ..
/usr $ cd
/home/user$
