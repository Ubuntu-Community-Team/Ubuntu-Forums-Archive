### Post by xelapond on 2007-11-03
Comment out is a term used by programmers to indicate that a certain piece of code should be temporarily(or not) deactivated.  In all programming languaes a comment is a part of the code that the compiler ignores.  It is a way to debug code, but deactivating parts of it in sequence to figure out where the problem lies.
 
In C++ and C you comment out a line with // infront of it. 

Which language are you using?  Each one has a different method of commenting out code.

Hope that helps,
