### Post by spiderbatdad on 2008-03-20
This wiki explains LTS vs standard releases. [https://wiki.ubuntu.com/LTS](https://wiki.ubuntu.com/LTS)
Hardy Heron is the upcoming LTS release, due in April, though available now for testing.
Ubuntu Linux will not run well, if at all, on 256 RAM.  512 is a good MINIMUM. 
Xubuntu or Fluxbuntu,
on the other hand,  are designed to run on systems with low memory.
 I would recommend Xubuntu to someone just starting out.

Taken from Ubuntu Documentation:

 Ubuntu should run reasonably well on a computer with the following minimum hardware specification.
 However, features such as visual effects may not run smoothly.

    *

      700 MHz x86 processor
    *

      384 MB of system memory (RAM)
    *

      8 GB of disk space
    *

      Graphics card capable of 1024x768 resolution
    *

      Sound card
    *

      A network or Internet connection

Note: All 64-bit (x86-64) PCs should be able to run Ubuntu. Use the 64-bit installation CD for a 64-bit-optimised installation.

### Post by mk1w86 on 2008-03-20
Ubuntu will run on a computer of those specifications and should be quite quick although if you find it sluggish you could try Xubuntu. 

As for the various versions a long term support release is supported 3 years on the desktop and 5 years for the server version as opposed to the normal 18 month support cycle.  The only drawback to running 6.06 is the packages will be older.  I would recommend the latest version 7.10 because it is a bit more user friendly than 6.06 and some things might work better.

Or if you wait the next release of Ubuntu 8.04 (due to be released next month) is also a long term support release.

### Post by Sukarn on 2008-03-20
LTS, as you know means long term support.

Once in 2 years, Ubuntu makes an LTS release, for which security updates are made for 3 years from the date of its release for desktop edition and for 5 years for the server edition, whereas the normal releases get security updates for 1.5 years from its release.

This is all there is to an LTS release that you need to know.


The version numbers of Ubuntu releases are of the form Y.MM, i.e. the version number tells when that version was released. Like Ubuntu 7.10 was released in October 2007.

[http://ubuntuforums.org/showthread.php?t=575456](http://ubuntuforums.org/showthread.php?t=575456) has a good list of OS'es for old computers, though I do think Ubuntu should run fine with 256 MB RAM.

Getting high speed from Ubuntu 7.10 on old computers - [http://kmandla.wordpress.com/2007/10/20/howto-set-up-gutsy-for-speed/](http://kmandla.wordpress.com/2007/10/20/howto-set-up-gutsy-for-speed/)
