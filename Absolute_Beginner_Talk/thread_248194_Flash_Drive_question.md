### Post by meng on 2006-08-31
I'm sure you can, if you run your USB drive as a virtual LiveCD. The basic idea is to copy the CD's files (ISO's files, you can skip the burning step that way) and then make sure there's a bootloader. Someone else posted on these forums about doing this with Xubuntu using Lilo as the bootloader, it included broad instructions. I didn't have enough specific knowledge to follow these directions.

Myself, I've been trying to find a good distro to boot from USB, and the obvious choices seemed to be Slax, DSL, Featherlinux and PuppyLinux. These are all in the 50-200MB range. In my opinion, if going to the 600-700MB range, consider Knoppix or PCLinuxOS since they have Java and other restricted format stuff enabled already.

[www.pendrivelinux.com](www.pendrivelinux.com) has some ideas for you.
