Have you updated the driver recently?? I've seen this happen with nVidia drivers on occasion, but it is not usually observed with every logout attempt. If you are consistenly getting this problem I would first remove the nVidia driver (use the xorg provided 'nv' instead), and then verify that this is indeed the problem by looking to see if the issue no longer occurs.
