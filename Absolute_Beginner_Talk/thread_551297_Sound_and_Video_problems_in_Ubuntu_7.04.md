### Post by lisati on 2007-09-15
I use Ubuntu on one of the A100 range of Toshiba laptops. In order to get the sound to work, I had to download the updates to the system before tinkering with the volume knob - no problems since. Ubuntu normally shows an orange icon on the screen when updates are available, clicking on this (if it's showing) will start the process. Failing this, you can start the update manager from the System->Administration  menu.

Getting videos to play can take a little more work - one tutorial can be found at [https://help.ubuntu.com/7.04/musicvideophotos/C/video.html](https://help.ubuntu.com/7.04/musicvideophotos/C/video.html)
