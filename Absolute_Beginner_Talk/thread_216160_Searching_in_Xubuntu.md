### Post by aysiu on 2006-07-15
I don't think Xubuntu comes with a GUI search tool.

You may have to install one. It doesn't have to be Beagle. I believe ```
sudo aptitude update
sudo aptitude install gnome-utils
``` should install the Gnome search tool.
