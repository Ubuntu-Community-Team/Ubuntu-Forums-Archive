### Post by aysiu on 2008-01-29
Right now, there is no way to do it in Firefox. I've heard rumors about this being fixed for next release, but we'll see.

In the meantime, you can use Galeon to view one thumb at a time (whatever file you have highlighted) or Konqueror to view all thumbs at once.

For more details, check out this bug report:
[https://bugs.launchpad.net/ubuntu/+source/firefox/+bug/89381](https://bugs.launchpad.net/ubuntu/+source/firefox/+bug/89381)
