### Post by aysiu on 2008-01-16
You'll find some useful documentation here:
[https://help.ubuntu.com/community/MacBook](https://help.ubuntu.com/community/MacBook)

The short answers to your questions, though are:
[list][*]It is possible to set up a dual boot [*]You will be able to boot into OS X or Ubuntu[*]You will need to partition the drive [*]The Ubuntu installer CD can repartition your drive [*]10 GB is plenty to run Ubuntu [*]x86 is fine[/list]

As for recommendations, I'd say don't be too hasty to install it. Play around with the live CD for a while. Also, know that installing Ubuntu on a Mac and getting everything to work is usually a lot more work than doing so on a previously Windows PC.
