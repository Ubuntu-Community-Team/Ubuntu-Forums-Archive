How do I change the splashes for applications like gimp and open office?

---

### Post by por100pre1 on 2007-10-22
For OpenOffice.org try replacing the /usr/lib/openoffice/program/openintro_ubuntu.bmp. If it is not present replace the /usr/lib/openoffice/program/intro.bmp image. For GIMP, create a ~/.gimp-2.2/splashes folder in your user folder and then put the splash in that folder.
