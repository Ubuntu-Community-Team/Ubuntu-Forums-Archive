### Post by Tomosaur on 2007-10-03
You seem like you don't really rely on Windows exclusive software, so yeah, Ubuntu should be perfect for you.

Linux doesn't have a native Photoshop version though. I think some older versions will run fine in Wine, but you'll have to check that for yourself. Luckily though, we do have a very nice image editing package called 'The GIMP'. The interface confuses many new users, and I personally think they should just stick to a more traditional 'one window' interface, but the current interface does have its upsides. The 'Stay On Top' button will be your best friend while using The GIMP. Interface aside, The GIMP is on a par with Photoshop in terms of what it can do, but you may need to learn some new tricks and such. There are a few galleries of stuff created with The GIMP floating around on the net - well worth a look.

If you don't rely on Photoshop for your job, then The GIMP will be fine for you. If you do rely on Photoshop, then you should know that it doesn't have CMYK support right now (if you don't know what that is then you can just ignore it).

As for instant messaging, Linux blows Windows away. Just use Pidgin, which supports all of those protocols (MSN, Yahoo etc), or Kopete.

### Post by Sunforge on 2007-10-03
Ubuntu makes Linux appear quite easy but you will run into quite a lot of rough edges. That said, to your questions:

Browsing the net - done deal with Firefox, Opera, whatever. Some sites and apps use Active X optimised for Windows which can cause a problem if you're an admin.

Talking to friends on aim, yahoo and such - I use Pidgin which connects to everything. It's as good as the default windows stuff.

Playing WoW - yes it can be done in Wine but people have had some upsets using it under Linux, if you want to find out more, post a question in the gamers section as I've never played WoW.

Listening to music - done deal, plethora of music players, too many options if you're not careful.

Some random editing in Photoshop - works in wine, depending on the version you have. If it's CS3 that could be a deal breaker. The Gimp is pretty good although a little bit of a challenge if you're not used to it.

It's worth reading through the "how to" sections of the Ubuntu site and if you're really keen and like books get a decent Beginning Ubuntu book (I like books so there you go).

Dual booting is a good idea if you're unsure. If you want to go the whole hog make sure that you have your original Windows installation media (and make sure it works) and licence key, so you can say goodbye if you wish to.

Oh - and back yo' stuff up!

### Post by Nehvrook on 2007-10-03
on board sound should be fine. Ubuntu auto detects drivers for things like that and usually does a really good job.

ATI does develop drivers for Linux but they're generally not as good as Nvidia's. You may run into problems there.

I play WoW and I have it running better on Ubuntu than it does on Windows, and it was really easy to get set up. If you're unlucky you might run into some problems but I'm almost certain it won't be anything un-solvable. If you head over to the Gaming & Leisure section ([http://ubuntuforums.org/forumdisplay.php?f=93](http://ubuntuforums.org/forumdisplay.php?f=93)) you can see a lot of people have WoW working in LInux and there's a lot of support out there if you do run across problems
