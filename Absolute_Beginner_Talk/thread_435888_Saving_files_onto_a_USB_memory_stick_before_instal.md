### Post by jrlii on 2007-05-07
Assuming that by NT, you mean Windows NT 4.0, ( or heaven help you, NT 3.x) forget it.

NT before Windows 2000 did not support any USB devices whatsoever.

There might be some third-party work-arounds, but Windows 98 was the first to support USB devices, and it requires a special driver for mass storage devices - and if your USB drive doesn't supply it, finding one which will work may be a pain.
