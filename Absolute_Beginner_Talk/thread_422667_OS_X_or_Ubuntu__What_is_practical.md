### Post by jem7v on 2007-04-25
Ubuntu, as with most Linux distrobutions, is optimized and best supported for x86 architecture PCs.  The available packages and software selection are probably going to be more limited for 64 bit computers and Apple computers.  Trying to run it on any Apple computer is probably going to be harder than using it on a PC.  Given your situation, Ubuntu *might* not be the best OS solution.

That said, did you go to the Restricted Formats page, as linked to in the Desktop User Guide?  [https://help.ubuntu.com/6.10/ubuntu/desktopguide/C/index.html](https://help.ubuntu.com/6.10/ubuntu/desktopguide/C/index.html)  I'm pretty sure most of the usual media codecs Are available for Apples running Ubuntu.

### Post by tgalati4 on 2007-04-25
Tiger is a great operating system.  However, it requires quite a bit of RAM (512 MB, 1 GB is better).  It also requires a large harddisk.  With Universal binaries, you are carrying both intel and ppc code.  20 GB is probably minimum, 60 GB is better--especially if you use iLife applications.  If your iMac is up to snuff then definitely run Tiger.  If you have an iPod, then iTunes and iPod integration is the best in class.  If you don't mind digital rights management and restricted Mac formats for all of your data, then Tiger is the way to go.

If Tiger doesn't run well on your hardware, or if you are trying to network your iMac with other Linux computers then you might consider Ubuntu.  For instance with iTunes 7 you can only share your music collection with other machines running iTunes 7.

On Linux, any machine running the appropriate DAAP daemons can see the music from any program that shares it through this protocol.

And yes, you are correct that Ubuntu takes time to get it set up the way you want it.  And yes, you are also correct that Tiger is a good value at (~$129 US) compared to another commercial operating system.

As I am writing this on a powerbook with Tiger 10.4.8, I have 5 other machines on my local network running Ubuntu.  I am also running XDarwin on it which allows me to run gnome-panel which gives me a remote desktop on any of those machines.  I can also run (Chicken of the) VNC to get virtual remote deskops on these machines when I need to see entire desktop.

I have run the Ubuntu Dapper 6.06.1 Live CD on this powerbook a few times, and most things work out of the box, but the fit-and-finish of Ubuntu PPC can't compare to Tiger.

The bottom line:  If you get an old Mac, and for some reason Tiger 10.4.x is not on it, then you have to weigh spending $129 to get a decent operating system plus $69 -$99 to get iLife (iPhoto, etc).  iTunes is "free".  Or, spend the time to get Ubuntu to work smoothly and with all the capability you expect.  This can take some time--no doubt about it.

The Ubuntu forums can help you find solutions, but you still have to do the work.

Now if you buy an old Thinkpad for $200 and it has Windows on it, then going to Ubuntu is a no-brainer.  With the Aqua themes, you can make it look like Tiger.  Just spray paint it white or silver and you have a pseudo-mac.
