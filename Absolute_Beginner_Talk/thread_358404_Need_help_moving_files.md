### Post by jpeddicord on 2007-02-10
The folder you will want to copy your profile to is /home/username/.mozilla/firefox. You may have to type that in manually, as .mozilla is a hidden folder. You can turn on hidden files from the View menu in the file manager if you like to see the folder.

Once you do that, copy the name of the other folder that was already there, and delete that folder. Rename the profile you want to copy to that so it works correctly.

Hope that helps!

Jacob
