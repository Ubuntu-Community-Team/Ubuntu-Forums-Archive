### Post by amsterdam on 2007-08-13
I installed the vmware server tools today on 7.04 with no problems. make sure you do a 

sudo ./vmware-install.pl

if that does not work make sure it is executable by

sudo chmod +x ./vmware-install.pl

if that still does not work paste any  error messages.
