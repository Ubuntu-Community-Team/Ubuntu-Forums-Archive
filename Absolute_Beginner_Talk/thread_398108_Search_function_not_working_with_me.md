### Post by Campingman on 2007-03-31
You have to select the right folder for it to search in, most of the files will be in file system.  
There is a good search function you can add on to your panel, right click on the panel, add to panel and add deskbar function, this a cool function that searches web, wikepedia and also files/folders, it will find most things.  It will also execute programs too.
It does not search for files by default, if you right click preferences you can configure it to search for files/folders too.
