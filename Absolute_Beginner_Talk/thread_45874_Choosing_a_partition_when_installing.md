 It is MUCH easier if you're willing to wipe the drive clean and start from scratch.  It's harder if you have to create and resize partitions.   If you are willing to wipe the drive clean (since everything is backed up), the key is INSTALL WINDOWS FIRST.

1) Restart with windows xp cd and reboot, it should take you to the install screen (if not check the bios for cd-rom boot).
2) At one point it will ask you about partitioning.  Erase all partitions, and recreate a partition that is the size you want windows to reside on-- what I mean is, don't create a partition of 100%.  Leave the rest of the drive "unparitioned space"
3) Install windows on this newly created parition
4) After windows is installed, restart with ubuntu cd in.  Install ubuntu in the unparitioned space (it will create the partition for you).
