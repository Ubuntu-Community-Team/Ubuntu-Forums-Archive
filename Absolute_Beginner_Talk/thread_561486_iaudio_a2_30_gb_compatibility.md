### Post by kelnage on 2007-09-27
From what I can see, the A2 actually runs on a modified Linux kernel, so that can only say good things about the compatibility. Another bonus is that is supports OGG Vorbis and FLAC - both of which are open source and widely supported by Linux so that's very good.

It's likely that it will act as a portable hard disk - it'll be just like plugging in a USB stick. However, I've never owned one and honestly, only first hand experience could guarantee it'll work. But all the signs are good, if nothing else.
