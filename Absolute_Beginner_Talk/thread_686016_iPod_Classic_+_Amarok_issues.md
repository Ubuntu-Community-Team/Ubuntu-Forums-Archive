### Post by Impulse666 on 2008-02-02
>  iPod nano 3G (fatty), iPod Classic and iPod Touch:

The song database is protected by a cryptographic hash. You need libgpod 0.6.0 (or newer) and the SysInfoExtended file as is detailed in [Common Problems]("http://amarok.kde.org/wiki/Media_Device:IPod#My_iPod_does_not_show_any_music") to compute it automatically. 

[Source]("http://amarok.kde.org/wiki/Media_Device:IPod#iPod_nano_3G_.28fatty.29.2C_iPod_Classic_and_iPod_Touch")
