### Post by Perfect Storm on 2007-03-02
No need for antivirus in Ubuntu/linux other than if you run email server or making a protection computer for Windows.

Firewall is installed by default "IPtablets". If you want a frontend for it you can install firestarter.

### Post by t0p on 2007-12-08
I've been using Linux on and off since Breezy Ubuntu... that's what, 2 years?  I've never used anti-virus software, and I've never configured iptables.  And I've never suffered from malware.

Some people say Linux is structurally safer from viruses.  Others think that when Linux is as popular as Windows is now, and there are lots of computers out there running Linux, that's when the virus writers will target Linux.

I don't know who's right on that.  But I *do* know that, right now, my computer's got more chance of growing legs and running away than getting struck down by a virus!!
