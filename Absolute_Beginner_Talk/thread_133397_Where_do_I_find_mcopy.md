### Post by bimberi on 2006-02-20
mcopy is available in the mtools package.  To install it type:

    [B]sudo aptitude install mtools
[/B]
in a terminal.  To open a terminal go to Applications -> Accessories -> Terminal.  This is where you type linux commands.

Cheers, Dave.
