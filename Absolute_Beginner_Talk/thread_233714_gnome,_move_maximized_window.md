### Post by Metacarpal on 2006-08-10
I'm pretty sure that by definition, a maximized window is taking up the entire available (non-panel) desktop area.  You'll have to un-maximize it to move it.  If you want a full-size window you can move (so some of it is off the screen), you'll want to take a non-maximized window and click-drag the lower right corner to resize the window to be full-screen.  Then you can move it wherever you want.
