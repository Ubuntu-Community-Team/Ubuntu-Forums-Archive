### Post by Inxsible on 2008-03-31
You should use the ```
sudo rm <path to file>
```You can also log into nautilus as root by doing ```
gksudo nautilus
``` and then browsing to the location and deleting the files. The first method is preferable since you can create havoc, if you are not careful in the second method
