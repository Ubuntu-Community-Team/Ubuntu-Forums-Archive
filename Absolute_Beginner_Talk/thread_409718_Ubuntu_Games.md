### Post by RomeReactor on 2007-04-14
Hi. There are many different ways of installing games and applications downloaded from web pages; it depends on the contents of the file(s) you download: maybe a compressed file containing a .sh file, or a .bin, or maybe a "make". If it's a .deb file, then just double-click on it to install with Gdebi. For Genesis emulation, I recommend DGEN; you can find it in Synaptic. For a Doom-type game, i really like [Alien Arena 2007]("http://red.planetarena.org/").
