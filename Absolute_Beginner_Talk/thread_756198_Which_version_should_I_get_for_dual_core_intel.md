### Post by dstew on 2008-04-15
> When i try to install the 64 bit version my computer says kernel alive, then a second line says something else about the kernel but it then reboots my pc.It sounds like you have the right version. Maybe the CD is bad. Run a CD integrity test (opening menu item).

EDIT: I see you tried this. It is probably a bad disk. You need to check the md5 checksum of the .iso after you download it, then burn at a slow speed (4x maximum) to get the best chance that the CD is good. Use good quality CD-R media.
