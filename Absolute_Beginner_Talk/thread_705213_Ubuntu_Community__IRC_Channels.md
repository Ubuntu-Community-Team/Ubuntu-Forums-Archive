### Post by p_quarles on 2008-02-23
I don't use Pidgin, so you'll have to find the exact menu items yourself, but here is the basic procedure.

1) You need to connect to the server. Once you've found the option to connect to an IRC server, you'll need to enter two things. The server, which should be:
```
irc.freenode.net
```
And a nickname, which can be what you want. You don't need a password to connect to this server.

2) Once you're connected, you'll see some text sent from the server, and a command line. In the command line, type the following:
```
/join #ubuntu
```
A new window or tab will open to the chat room.
