### Post by laidback on 2007-03-23
I've been running Ubuntu for a few months now, originally I had a Speedtouch 330 which worked with Mandriva, after some effort, but I couldn't get it to work with Ubuntu. I was advised to switch to a router/modem and use ethernet connections. Absolutely the right thing to do, much much better, I can also have up to 4 machines attached at any one time, I only use 2 but it saves all that cable swapping. The router also has a firewall. Mine cost around £30, money well spent.
Also, if you enjoy looking at other distro then an internet connection via a router/modem is so much easier and quicked than with the speedtouch microcode method, which even if it does work now in Ubuntu will probably be slightly different in another distro.

### Post by chopperfixer on 2007-03-23
I use a 3Com OfficeConnect ADSL Wireless Modem Router (3CRWDR100A-72).  Personally, I've never had any problems with this router.  Admittedly, it had been set-up and was working with Windows XP when I installed Ubuntu, but the Ubuntu installation process found the router and connected to the internet with no additional assistance from me.  I didn't need to install additional drivers or have to muck around trying to get the OS to see the router, it just worked.  Brilliant.

Original set up of the router was fairly painless.  One thing I've done is to make sure that the router firmware is the latest available.  This can change the router access interface, (usually via your internet browser), and make the router set-up easier.  I must admit I haven't tried wireless access with Ubuntu yet, as they who must be obeyed, would go nuts if I took Windows off the laptop.  One of these days!!!!!
