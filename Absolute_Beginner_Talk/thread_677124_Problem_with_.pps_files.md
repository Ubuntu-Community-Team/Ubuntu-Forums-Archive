### Post by Kevbert on 2008-01-25
Try installing an Openoffice extension.  
[LIST]
[*]Open any  Powerpoint (pps) file.
[*]Go to Tools - Package Manager
[*]Click on Get more extensions here
[*]In the search box enter 'Auto starter'
[*]Select Auto Starter for Presentation Documents.
[*]Click on 'Get it'.
[/LIST]
Hopefully next time you click on a PPS document it will autorun immediately.  To exit at any time press the escape key.

### Post by Hagar Delest on 2008-01-26
To automatically run a presentation, install that extension: [Auto Starter for Presentation documents]("http://extensions.services.openoffice.org/project/ImpressRunner").
*Edit: had not seen it had been advised already! It's not the package manager but the extension manager.*

For the sound, you need to install the JMF (Java Media Framework) but not all sounds format are supported IIRC. See here: [Can't embed movies in OpenOffice Impress]("http://www.oooforum.org/forum/viewtopic.phtml?t=26704").
