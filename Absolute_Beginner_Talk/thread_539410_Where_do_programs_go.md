### Post by panther_sn on 2007-08-31
If you can't find the program you have a couple of options, try pressing <alt><f2> and type the program name in ther and press enter, now if that doesn't work have a look in 
/bin or /usr/bin and see if you find your program in there, if you need more help let me know the program and I'll tell you how to start it

### Post by s26c.sayan on 2007-08-31
Typing the fast few letters of the program you want to run in the terminal, and pressing TAB auto completes the name! Then press enter to run it!
