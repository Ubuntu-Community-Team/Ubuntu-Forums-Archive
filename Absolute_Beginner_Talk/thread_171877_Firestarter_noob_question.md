Does firestarter work in the background after configured, or do you have to start it everytime you log in, and if so, is there a way to start it everytime the computer starts.

---

### Post by kaamos on 2006-05-07
It starts in the background. If you are feeling unsure you can use the command
```
sudo /etc/init.d/firestarter status
```
in terminal to check that it's running.
