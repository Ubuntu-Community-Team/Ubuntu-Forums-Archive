### Post by ClarkePeters on 2006-09-28
When I was a very green newbie, I didn't know the difference between Ubuntu and Xubuntu.  In short, Xubuntu is Ubuntu for smaller or outdated machines (it is xfce based instead of gnome, but like me, I bet you don't even know what that means--I didn't--so don't worry about it). Of course you can put Xubuntu on any size computer--as did I.

Here's what happened to me.  Ubuntu wouldn't install on my computer, it kept freezing up (maybe it was a problem with the sept 14 bug or a hardware config problem such as a video driver conflict or something, I'm not sure). 

At any rate, I happened to have a copy of the Xubuntu install CD.

I installed it and it worked beautifully. However, as I had gotten used to my Ubuntu on another system of mine, I still wanted Ubuntu. (You don't have to install Ubuntu if Xubuntu works well enough for you.)

So after installing Xubuntu, I did an update, and then after that I went to Synaptic Manager and checked off "ubuntu-desktop" and installed it through Xubuntu (or you can go to command line and type $sudo apt-get ubuntu-desktop). 

I'm a happy camper.

Good luck

Now I have a choice at login to which system (and language) I want to operate.
