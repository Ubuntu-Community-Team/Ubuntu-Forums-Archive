### Post by AFT on 2007-01-12
Thank you, 

Incase anyone comes across this, and needs hand holding like me....

(i use KDE)

Install Kxkb
Open System Settings
Under Regional & Accessibility > Keyboard Shortcuts
At the very bottom of Global Settings "Switch to Next Keyboard Layout"

DEFAULT IS CTRL+ALT+K

and if you'll want to switch back to English with the same key strokes, then make sure you program the "Alternative" option while in your 2nd language. For example mine shows "CTRL+ALT+Armenian_ken" as the alternative to "CTRL+ALT+K"

### Post by AFT on 2007-01-13
omg! after further research from the help files here it is for GNOME USERS

In the Keyboard 
      preference tool, you can set
      which key that you would like to use to switch groups on your keyboard. Choose
      Open Keyboard Preferences in the applet popup
      menu, and choose the key you want to use in the Layout Options tab, under Group Shift/Lock behavior.

IN OTHER WORDS what logically should belong under KEYBOARD SHORTCUTS is actually found under
   System > Preferences > Keyboard > Layout Options > Group Shift/Lock behavior
Under here, you select what preset key combination you want to use. The default is set to pressing the two ALT keys.
