### Post by stuporglue on 2005-10-27
> Q4. Most importantly, how to install applications? In windows, there's simple .EXE file for installing the program, but what to do in Ubuntu?

You'll use the program "Synaptic" which is a nice GUI for apt-get. Apt-get has a list of sources (ie. websites and/or CDs) with programs on them. It downloads a list of available programs from each site. You tell apt-get (via Synaptic) which program you want installed, and it installs and configures it for you. It's the greatest thing since the pentium. 

Top Menu Bar --> System --> Administration --> Synaptic Package Manager

---

For your MP3s, you might want to check out this wiki page: [https://wiki.ubuntu.com/RestrictedFormats](https://wiki.ubuntu.com/RestrictedFormats)

Welcome to Ubuntu!
