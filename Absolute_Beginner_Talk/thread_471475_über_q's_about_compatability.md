### Post by Kobalt on 2007-06-12
Ubuntu will run very fine on your system. But in my opinion you should dual boot. 
Wine will not allow you to run WoW actually *(which in fact it will)*, it is not aimed at running games, the performances would be too low. And consider you shouldn't find any other solution as easy and efficient for running Win games than running them...in Windows. 
Plus, in case you really mess up your Ubuntu or if something is broken you have your Windows as a safety net to access internet and get some help. 
Setting up a dual boot is really easy, see [this]("http://psychocats.net/ubuntu/partitioning") for more info.

Finally, yes you should really try out Ubuntu if you're interested in it. Of course there is a little learning before you feel comfortable, since it's a whole new world. But if you hang on you'll find it worthy :)

Welcome to Ubuntu !
