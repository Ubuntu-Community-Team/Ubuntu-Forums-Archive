### Post by irish_flu on 2007-02-20
There's a very simple piece of software that was included with my Ubuntu Install, called "sound recorder".  I found it in Applications --> Sound & Video.

If you want something with more advanced features (but still easy to use) check out "Audacity" (you can get it from Synaptic or from "Add/Remove Programs").

I love Audacity, it's very full featured (similar to Adobe Audition, but easier to use IMHO).  I use it in both Ubuntu and Windows (there's an identical version for 2000/XP).
