### Post by in_flu_ence on 2007-02-09
I have no issue with network card (Netgear wireless card). In fact, ubuntu is the only one that get it work right from start. Suse and Fedora don't. ATI bulit in chipset motherboard and I got it installed pretty smooth from one of the tutorials out there (sorry that I have lost the link).

Sound card is fine (Creative Audigy). Wireless USB keyboard and mouse set goes fine. Nevertheless, you might get some problem when u 'reset' the security code in the wireless keyboard set. I actually had to go to a windows station to reconfigure the set.

No problem with any USB device. Ipod, flash drive etc. they work fine for me. I can't get my Sony USB mp3 player working though.

I suppose get a cheap graphics card to play around. Your system might be old for any BIG investment in graphics card. NOt sure if the beryl and the fanciful video animation will work fine on the system in the first place.
