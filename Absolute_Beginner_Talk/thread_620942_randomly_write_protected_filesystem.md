### Post by ukripper on 2007-11-23
seems like few errors on your partition. Check logs for any specific error messages,  intermittent remounting could be occuring due to some errors on vfat partition. i recommend you check it using either windows inbuilt **chkdisk /F** or some other utility for fat as i don't use this filesytem therefore have a little knowledge how to check fat but i am sure windows utility chkdisk does that.

Personally I would convert this partition to NTFS using either windows utility** convert.exe **or some third party linux app as FAT32 is very limited when it comes to big files over 4GB
