### Post by zgornel on 2006-12-11
To create a shortcut to a Samba server (machine), create in ~/Desktop a file called smb-<your-workgroup>-<SERVER NAME> that contains:
```

[Desktop Entry]
Encoding=UTF-8
Name=SERVER NAME
Type=Link
URL=smb://IP OR SERVER NAME/
Icon=gnome-fs-server

```
