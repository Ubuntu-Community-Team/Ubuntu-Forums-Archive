### Post by DJ_Peng on 2008-03-17
The bottom line is that there is **no** release date set for Firefox 3, as they've said all along it'll come out "when it's ready." 

That said, I've seen several comments on Mozillazine that are guessing late May/early June for a release date. Either way, Firefox 3 will definitely *not* be officially released in time for Hardy.
