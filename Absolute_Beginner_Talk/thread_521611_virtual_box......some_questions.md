### Post by bodhi.zazen on 2007-08-09
You can check it out. You can either boot to Windows and run Ubuntu in a virtual machine or the other way around.

Either way, the virtual machine uses virtual devices (hard drive, videodriver, Ethernet card, etc) so performance is not the same as dual booting. Some games will work well, others will not ...

for example you can not run Beryl/compiz on a virtual ubuntu (yet) ...

Virtual box is quite fast and easy to use.
