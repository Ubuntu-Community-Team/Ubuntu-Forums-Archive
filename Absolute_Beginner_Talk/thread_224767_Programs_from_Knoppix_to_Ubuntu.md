### Post by benuski on 2006-07-28
Well, you have to make sure that you have all the repositories enabled in your package updater (Adept in Kubuntu, Synaptic in Ubuntu, I don't know what it is in Xubuntu).  If you don't know how to do that, search for "Repositories" on [http://wiki.ubuntu.com](http://wiki.ubuntu.com).  After that, just search for whatever program you want from Knoppix, and there's a good chance that its in the repositories and able to be installed.  

However, if its not there, that means that you're going to have to either find a .deb file, or build it from source.  You'll have to go to the website that each project maintains, and see what sort of downloads they have.  If they have a ".deb" file, you can just download it, right click on it, and there should be a thing that says install(not extract).  Click on that, and voila!, the program is installed.

If it only has a "tar.gz" or a "tar.bz2" file, you`ll have to build it from source, the prodecure for which is in this forum post:
[http://www.ubuntuforums.org/showthread.php?t=179797](http://www.ubuntuforums.org/showthread.php?t=179797)


Hope this helps!
