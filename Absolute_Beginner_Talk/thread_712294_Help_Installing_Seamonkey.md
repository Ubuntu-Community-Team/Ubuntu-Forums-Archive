### Post by scragar on 2008-03-01
once you know what file to run just right click on your menu and choose edit.
then click on internet and add item. In the box that appear select the seamonkey file in the command option and give the file a name, with any luck ubuntu will automaticly find the image, so you can just save it and close the edit menu screen. If ubuntu did not automaticly find the image click where it says no icon to search for youself.
