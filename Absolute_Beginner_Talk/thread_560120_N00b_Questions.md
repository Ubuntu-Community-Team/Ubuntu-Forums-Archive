I'm keeping Windows XP to run InDesign CS2. I think I remember seeing where somebody has gotten Dreamweaver to run under WINE. My clients are demanding work done with Adobe products, so I'm not willing to risk it. I run Ubuntu on a separate box. You can try dual-booting to give Ubuntu a try. The Live CD will take you right through dividing your hard drive for both operating systems. A new boot-loader, grub, will be installed which will allow choice of which OS to run.

Uninstalling Ubuntu can be a problem once done though. Not for the faint of heart or without solid data backup.

If you have a fast machine, you can run Ubuntu as a virtual machine under a hardware compatibility emulator program like QEMU, which is open source. Ubuntu will not run under MS Virtual PC. This will be easy to get rid of if necessary.

Give Ubuntu a try. There's a way to do everything I need to do except provide the working files for InDesign my clients demand. If end product is your only concern, there's a way to do everything I can think of, provided you have the hardware platform and peripherals to handle it.
