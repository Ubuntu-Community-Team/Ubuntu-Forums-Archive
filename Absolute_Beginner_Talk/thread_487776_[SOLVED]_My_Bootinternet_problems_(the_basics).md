### Post by ronocdh on 2007-06-29
Regarding the boot issue, if you suspect the monitor has something to do with it, unplug it and see how it boots. I don't think that's the issue, though. Have you verified the integrity of the burn? This is very, very important, because not doing it can result in errors. (Main screen of the installation CDs has the option to "check disc for errors.")

I'm not certain about your ISP, but shouldn't a cable connection be operating with DHCP, not a static address? I would also consider using a router, which for sure would be DHCP, and then Ubuntu can manage the rest.
