### Post by cotcot on 2007-09-12
It might be that you have to make the windows partition active. You can check it with a livecd and fdisk. If the windows partition is not active, set it active and reboot. 
You could use the SuperGrubCD [http://www.geocities.com/supergrubdisk/]("http://www.geocities.com/supergrubdisk/") to boot in windows or ubuntu untill the problem is solved.
