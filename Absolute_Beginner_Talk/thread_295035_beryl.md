### Post by turbojugend_gr on 2006-11-07
add this to **screen section **in your xorg.conf: 

# Enable 32-bit ARGB GLX Visuals
# does not support rendering into the Composite
# Overlay Window, you will need to disable clipping
# of GLX rendering to the X Root window with this
# option, or you will get a blank screen after
# starting compiz:
#    Option "DisableGLXRootClipping" "True"

*and this:*

Option         "AddARGBGLXVisuals" "True"

### Post by jordanmthomas on 2006-11-07
```
Section "Device"
Identifier "NVIDIA Corporation NV40 [GeForce 6200?]"
Driver "nv"
BusID "PCI:2:0:0"
EndSection
```

You need to change the driver to nvidia (assuming you have in fact installed the drivers...you would know if you have)
press Alt + F2
```
gksu gedit /etc/X11/xorg.conf
```
find the line and change it.  Save it and close it.

Now, before you log out...you will want to know how to fix it in case your xserver won't start.  In the off chance you get a nasty blue screen, press Ctrl-Alt-F1 and log in.
```
sudo nano /etc/X11/xorg.conf
```
and you can change it back, and then restart your xserver
```
sudo /etc/init.d/gdm restart
```
