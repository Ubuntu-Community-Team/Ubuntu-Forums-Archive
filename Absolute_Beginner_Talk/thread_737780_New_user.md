### Post by Rocket2DMn on 2008-03-27
The LiveCD does not have full support for video cards, but after you install you will be OK.  You can try the alternate install cd, too.  It is a text based installer, but it is very easy to use. You can get it at - [http://ubuntu.com/getubuntu/download](http://ubuntu.com/getubuntu/download) by checking the box at the bottom for the alternate cd.
