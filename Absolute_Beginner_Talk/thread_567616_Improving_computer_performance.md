### Post by nowshining on 2007-10-04
defrag - does automatically however fragmentation doesn't slow down Linux / Ubuntu like it does with Windows as it uses a diff. storing scheme and doesn't rely on the disk a lot for such things. -i'll get ur some links later :). So far all I did, was reduce gnome by going into gconf and reducing gnome resources which it does a bit. :) but does take some conviences away, etc.. which is minimal. Gconf is like the windows registry however Ubuntu doesn't depend on it like windows does.

Also I myself blacklisted unneeded modules from starting up, etc.. Used a how to someone wrote out and disabled some other things taht start up - disabled splash screen, etc..

I myself suggest waiting for Gutsy as the first install = easier, has more hardware support, etc.. Compared to Feisty. I did an update myself and have had minimal issues, however that's me. :)

Again it's ur choice tho - i myself would of waited if I didn't install feisty when I first had it.
