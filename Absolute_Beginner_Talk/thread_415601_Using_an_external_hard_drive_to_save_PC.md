### Post by steve.horsley on 2007-04-20
Yes, yes, yes. It is a very good idea to save all your work and data before trying to install. Although losing it is unlilely, it is possible and the pain if it happens is very bad. You should have backups anyway, just in case the hard disk dies and loses all your work, like mine did a few years ago. 

It is possible to find products that will make a backup of your whole Windows installation, but that is less critical because if hte worst happens, ypu can always reinstall from CD. 

If you replace Windows with Ubuntu, you could copy all your data back from the backup and probably read it all with Linux programs, except maybe for data from windows-only applications.  It might be wise to dual-boot instead until you are sure which side of the fence you want to live on though.
