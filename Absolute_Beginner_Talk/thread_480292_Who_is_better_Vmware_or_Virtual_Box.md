### Post by HeavyAl on 2007-06-22
After working with both of them a bit I'd say I prefer Virtual Box. Both of them gave me about the same performance in terms of startup times for the VM's and I didn't notice much difference in the memory requirements. Virtual Box though was able to actually run some low-end games that vmware had trouble with (Boggle Extreme, specifically, a very no-nonsense game that shouldn't have any trouble in a VM but did on vmware). Further, Virtual Box was MUCH easier to get connected to the rest of my Linux system and the additional tools installation was a cinch where as the VMWare desktop tools gave me all kinds of trouble.

The real clincher for me though was that Virtual Box is free!

### Post by bodhi.zazen on 2007-06-22
VirtualBox is under heavy development and so if you ask again in a few weeks opinions may change.

Virtualbox is very fast and easy to install. Innotek is maintaining a repo so you can install via apt-get/synaptic.

VMWare is a more mature product and so has more to offer in terms of stability and reliability. Vmware offers more advanced features and advanced features are easier to configure with VMWare.

Right now I use both. I find if I want to test drive a distro I boot the iso with Virtualbox. If I want to install and maintain a stable machine, VMWare.

Best to make a list of what you want from virtualization and go with the application that best suits you needs.

Intro to virtualization ~ Go with VBox

Stable virtual machine in a production environment - Go with VMWare.

The qeustion should be Which one is better *for me* since each has advantages and disadvantages.

Just my 2c. HTH.
