### Post by dptxp on 2007-05-06
If you are using Windows, download InfraRecorder (easy to find), install it.
Great for burning iso images.

Safest and fastest way. Never failed me.

Use burn image command and choose your iso.

Do nothing else. Wait till CD pops out. Put it back and restart with CD boot
as first option. Click on Install/Live when you see the page. You will be
running live CD. You can install when the desktop shows up.
There is an icon at top left

DO NOT use make boot CD or Data CD. You shall be left with a dead coaster.
