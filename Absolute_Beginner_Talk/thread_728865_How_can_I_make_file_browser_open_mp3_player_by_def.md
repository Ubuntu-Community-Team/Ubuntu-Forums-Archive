### Post by herbster on 2008-03-19
System > Preferences > Removable Drives and Media > Storage tab. Check the boxes accordingly to what you want, in this case you'd want to ensure that "Browse removable media when inserted" is checked.

### Post by Dan_472 on 2008-03-28
Good suggestions.  They're along the lines of what I was looking for.

I realized, too, that "music player" was actually a program called "rythmbox".  I went to System > Administration > Synaptic Package Manager and removed "rythmbox".  Now when I plug in the drive, an icon appears on the desktop that I can just click to open with the usual file browser.

Thanks again.
