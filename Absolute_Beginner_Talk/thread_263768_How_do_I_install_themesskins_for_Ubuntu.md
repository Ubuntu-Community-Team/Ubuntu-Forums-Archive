### Post by Omnios on 2006-09-23
There is a theme manager for Gnome

```

Menu:: System / Preferences / Theme

```

 In the theme program you can do a lot of things from choosing an exzisting theme or a custom theme. Now there is also a install theme button which allows you to add themes but you can also take the theme file and drag and drop it to the manager to install.
