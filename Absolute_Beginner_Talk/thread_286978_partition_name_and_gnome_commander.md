### Post by markymark on 2006-10-30
typing the command
gnome-commander
from the command line will start it
(to get a command line, go to applications | accessories | terminal)

you can also add a custom application launcher to the panel, by right clicking, then clicking add to panel, 
then choosing custom application launcher 

gnome-commander will have to go in the command box
