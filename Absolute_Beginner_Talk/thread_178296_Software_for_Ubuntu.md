There is no need to apologize. :) You can ask here whatever question will want, and you should get an answer pretty soon. Places where you can look for answers if you want to do this before asking the forums (which is of course recommended) are these:
1. System>Help
2. [http://help.ubuntu.com](http://help.ubuntu.com)
3. [http://wiki.ubuntu.com](http://wiki.ubuntu.com)
4. [http://doc.gwos.org](http://doc.gwos.org)
5. Searching forums for questions like yours.
6. Other webpages, which don't come to mind

When looking for software, try opening Synaptic (System>Administration>Synaptic package manager) and running a search, looking in description and name. Under System>Help you find instructions on how to activate several software sources.

Now, I wouldn't call your question basic, simply because there is quite a wide range to choose from in most areas of interest.

For mindmapping, the most recommended software, which I also like and use (and also works on windows, so a plus in portability) is [Freemind]("http://freemind.sourceforge.net/wiki/index.php/Main_Page"). I don't think it's in the repos, but instructions on how to install it are [here]("https://wiki.ubuntu.com/FreeMindInBreezy?highlight=%28freemind%29")

With pdf's, there are many alternatives, but I still recommend Adobe Reader, which can be installed from Synaptic as package acroread. Plus you might want mozilla-acroread package, which will enable you to view pdfs online embedded in Firefox.

As for media players, there are quite many. You already have a movie player totem and a music player rhythmbox Under applications>sound & video. They are pretty good. I suggest you try them. You might also want to look under system>help for instructions to install extra multimedia codecs.

Wellcome to Ubuntu and enjoy! :)
