### Post by Prodromoi on 2008-01-20
Q1. The two easy ways are to use the Add/Remove function in the "Applications" menu.  Make sure you select the "all available applications" option from the drop down menu at the top right.  Then just search for the app you want - the rest is obvious.

Alternatively use the Synaptic Package Manager (SPM) in the Systems > Admin menu.  It contains everything the first option does, plus extra stuff.  Not quite as "pretty" to use, but more thorough.

Q2.  You should have Mplayer and Totem installed as standard.  Another popular player is VLC Media player (installable as above).  VLC is ideal for playing DVDs whereas the first two options are restricted to computer file formats.

Q5. (If I understand the question correctly):
[http://www.gnome-look.org/](http://www.gnome-look.org/)
[http://art.gnome.org/](http://art.gnome.org/)

Hope these help with some of your questions.

This should be no problem. Use any of the installed players, or add others through the Add/remove software menu (e.g. Amarok for music etc).

OpenOffice has a version for Windows and Linux, and the files you create will work under any OS. Alternatively, if you're using MS Office under Windows, just save as .doc or .xls as usual, and open the files under linux using OpenOffice.
