### Post by Malibu Illusion on 2007-07-02
Compatible drivers....for what?

Ubuntu comes with many open-source and optional propriatory drivers for a multitude of devices out of the box.  I suggest running the LiveCD first and determining what, if any, devices you're having issues with and attempt to resolve driver problems one at a time.

Some devices have native Linux drivers, some run on generic open source drivers and others require a bit of tweaking to get running.  To even start this process, you need to know exactly what you experience problems with though and for that I think running the LiveCD would be a great start.
