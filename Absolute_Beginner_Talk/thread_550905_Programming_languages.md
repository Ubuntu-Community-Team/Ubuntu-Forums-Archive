Welcome to the Ubuntu Forums!

As far as I know, it is possible to use a number of languages to write programs for the GTK toolset. There is C, C++ (GTK+), C# (GTK#), Python (pyGTK) and others, as can be found here : [http://en.wikipedia.org/wiki/Gtk](http://en.wikipedia.org/wiki/Gtk)

AFAIK, most current gnome development is done in the above four I listed.

Now, as to running in windows, well, the GTK toolkit is supported on Windows, too, so it shouldn't be too hard to port..however, I'd leave this question to someone more capable than I am :P
