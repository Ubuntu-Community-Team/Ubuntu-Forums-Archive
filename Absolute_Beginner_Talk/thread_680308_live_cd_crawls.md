### Post by JoshuaRL on 2008-01-27
The Live CD will run much slower than the final system.  But Dr Small is right, if you are low on RAM than the Live CD can sometimes be unbelievably slow.  But if you can run it at all, you can install from it.  If not, you can always try the Alternate CD.  It runs in text-only mode and is much easier on system resources.
