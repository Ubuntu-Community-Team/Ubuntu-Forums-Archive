### Post by sawjew on 2006-11-28
> **nealklomp said:**
> Hey, Sawjew, could you tell me some specifics, like what driver is it using? I just bought the same printer, and I'll be using it as a networked printer, for both Ubuntu and Windows; although if all goes well I may finally dump windows on my desktop and go all Linux!

Nealklomp, I am now using Edgy with the default HPLIP drivers.  Edgy comes with HPLIP 1.6.9 and in the driver list is > Deskjet F300

make sure you select the HPLIP driver not the hpijs only driver if you want to use the HPLIP toolbox.  They both use hpijs to print but the hpijs only isn't recognised by the HPLIP toolbox.

I have selected the HPLIP Deskjet F300 driver for my desktop where the Deskjet F380 is attached as a local printer and for my laptop where it is attached as a network printer.  It works flawlessly on both although the HPLIP toolbox only works on the local printer connection.  If you use the HPLIP toolbox you can configure it to send you an email whenever something goes wrong such as running out of ink or paper jam which is excellent for a network printer.  The only complaint I have with this printer is the amount of ink it uses so make sure you find someone to refill your cartridges rather than buying new ones all the time.
