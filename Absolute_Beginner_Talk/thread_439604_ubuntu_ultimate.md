There isn't really any particular reason to use the Ultimate version. All it contains is a customized theme/icon set, and a bunch of preinstalled programs all of which are available via apt-get or debian installers. Theres really no reason to wait around while whoever is in control of the ultimate project compiles a new version. If you want feisty I'd get it now and install all the apps you want separately, it'll take less time than waiting.
