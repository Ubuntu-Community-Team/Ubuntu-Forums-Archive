### Post by jtwJGuevara on 2006-09-19
Hello,

My best advice would be to

a) download and burn an iso image of the latest stable release found here:  [http://www.ubuntu.com/download](http://www.ubuntu.com/download) .  This image will double up as a live CD and an OS you can install.  Just boot off of the CD and you will be into working a Ubuntu environment without actually having installed it to your harddrive.  Use this to play around with and determine whether or not you want to use Ubuntu or not.

b) If you like what you see, double click that installation icon and a wizard will walk you through installing.  When it comes time to partition your hard drive, you should have an option to resize your existing partition to make room to install Ubuntu.  Try this option.  This will allow you install Ubuntu right alongside with Windows.  You will be able to boot to both of them.
