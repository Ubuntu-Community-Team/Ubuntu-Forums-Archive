### Post by UltraMathMan on 2006-08-11
I suppose it depends on how the network is set up, but my kneejerk reaction is a yes. If it's set up with a router he shouldn't have any problem (other than prehaps configuring wireless [check out Ndiswrapper if he has problems configuring a wireless card]). If the connection is shared through a main computer (I remember seeing this as an option in XP network setup) - he may need to use Samba (haven't had to do this myself so I can't really give you direction here...)
^This assumes a wireless network^
He could always run the Live CD and test it out before committing to a full install.
