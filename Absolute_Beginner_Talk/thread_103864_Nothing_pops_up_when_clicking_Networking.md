---

### Post by CyberBob on 2005-12-15
It is possible that you are logged into your system as something other than the first user on your system, and do not have system admin privileges.  If this is the case, log out and then log in again as the first user on your system (the one that you created the login ID and password for during installation).

If this is *not* the case, and you are already logged in as the first user then look at this other thread that may help you ...

[http://www.ubuntuforums.org/showthread.php?t=104006](http://www.ubuntuforums.org/showthread.php?t=104006)
