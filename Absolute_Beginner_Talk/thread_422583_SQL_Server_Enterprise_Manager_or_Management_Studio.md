### Post by detheridge on 2007-04-25
If you are using the desktop version of Ubuntu I would recommend installing XAMPP 
[www.apachefriends.org](www.apachefriends.org)

This includes : Apache web server, MySQL, PHPMySQL (web based management front end for mySQL) and a host of extras.

If you have the server version of ubuntu you would have had the option to install LAMPP (the same as XAMPP before the project was renamed).
