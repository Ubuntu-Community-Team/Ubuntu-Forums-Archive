### Post by bremedios on 2007-03-13
This can be fixed by changing the xorg driver that is used to the sis driver instead of the vesa driver.  This can be done manually by editing /etc/X11/xorg.conf or more easily by running the command "sudo dpkg-reconfigure xserver-xorg" and selecting the sis driver instead of the vesa driver.
