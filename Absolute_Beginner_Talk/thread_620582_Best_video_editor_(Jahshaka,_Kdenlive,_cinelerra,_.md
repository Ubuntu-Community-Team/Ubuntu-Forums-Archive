### Post by lwhitmore on 2008-05-05
I've spent a while searching for decent video acquisition, NLE and compositing programs for linux.  Most of the free options seem a bit flaky when compared to their commercial counterparts.

I tried Cinerella, but it doesn't seem very stable on my latop.  Kino seems fine for acquiring video - but it's really pretty basic.  Still need to try out kdenlive.

Apparently a paid-for program called MainActor was a very good option up until mid-2007, but it's now been discontinued... there's a petition to GPL the source ([http://www.ipetitions.com/petition/mainactor?e](http://www.ipetitions.com/petition/mainactor?e)).  Would be worth signing this if you're interested.

I've just found out that Blender is really good for non-linear editing... if you're familiar with the program already, you should have little problem using it.  If you're unfamiliar with Blender, the interface will seem a bit confusing at first, but it's definitely worth persevering with it; manual pages are here... [http://wiki.blender.org/index.php/Manual/Using_VSE](http://wiki.blender.org/index.php/Manual/Using_VSE).

(Actually, I think blender used to be a commercial product which was abandoned and given to the FLOSS community.. maybe the same could happen to mainactor?)
