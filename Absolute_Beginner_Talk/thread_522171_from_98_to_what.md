### Post by Bartender on 2007-08-10
Get the exact brand of the laptop, and go to the internet with a running PC.  Visit Corsair or Kingston or wherever and plug into their Memory Configurator utility.  Find out the maximum RAM the lappy will accept.  
If you can bump it up to 512 that would be super.  Then you'd have the leeway to run LiveCD's.  LiveCD's won't run on less than 256.  they need more RAM.  
So it's an all-or-nothing proposition.  You'd have to download the "alternate install" version of Xubuntu, which does not let you test-drive it first.  With an "alternate install" CD you have to install the OS, then see if it works for you.
The "alternate install" will at least work on a PC with less than 256 RAM whereas a LiveCD won't.
