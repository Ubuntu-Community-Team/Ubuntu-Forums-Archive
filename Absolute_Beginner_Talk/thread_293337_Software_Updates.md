### Post by Madpilot on 2006-11-05
All those updates will be security or high-level bugfix updates of some sort; those are the only updates that are done on released versions of Ubuntu.

So yes, you probably need all of them. The ones that are "updates to programs that I don't even know what they are" are probably the important ones - low-level stuff that the user never really sees that holds the system together, that sort of thing...
