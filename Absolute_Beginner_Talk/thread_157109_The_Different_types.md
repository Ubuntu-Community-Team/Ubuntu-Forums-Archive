### Post by jpkotta on 2006-04-08
Those are just different versions.  Breezy is the current stable version, Hoary and Warty are past versions, and Dapper is the latest, due to be released in June.  They're all essentially the same, it's just that the applications get updated to newer versions, and extra features get added to newer releases.  I recommend installing the stable version, Breezy.  It is easy to upgrade without reinstalling (downgrading is harder).

### Post by mostwanted on 2006-04-08
[QUOTE=dylonium]Before I install again, can anyone describe the differences between Breezy, Warty, and Hoary? What's the best kind of Ubuntu? What should I use since I'm a beginner?[/QUOTE]

Those aren't kinds or flavours of Ubuntu, they are code names for the different versions of Ubuntu that have been until now. I think you are confusing them with Kubuntu and Xubuntu which are different kinds of Ubuntu (comes with other desktop environments than the standard Ubuntu).

> Warty - old version 4.10
Hoary - less old version 5.4
Breezy - newest stable release version 5.10
----
Dapper - bleeding edge, not released yet version 6.6

Basically, run Breezy.

### Post by confused57 on 2006-04-08
Breezy is pretty stable, I installed it in December and I can do about anything with Ubuntu that I can do with Windows.  You might want to consider putting your /home on a separate partition when you install, if you're feeling adventurous.  You might want to do this, so that when the final stable version,
Dapper Drake is released in June(which will be supported for 5 years with updates), Breezy will only be supported for another year; you can install Dapper over top of your Breezy without losing your data in the /home directory, which is where all your downloaded files, etc are stored.
After you install Breezy, I'd enable the extra repositories:

[http://help.ubuntu.com/starterguide/C/faqguide-all.html#fg-installing-applications](http://help.ubuntu.com/starterguide/C/faqguide-all.html#fg-installing-applications)

Then I'd do a search on Automatix, which will easily install about any needed programs to get Breezy up to par.  You'll need to do a lot of copy and pasting of commands in the terminal, it's really pretty easy; just highlight(a line at a time), copy, then paste in the terminal press enter, enter your user password and watch it work.  You'll find everyone in the forum is quite helpful and knowledgable, so don't be afraid to ask questions.
