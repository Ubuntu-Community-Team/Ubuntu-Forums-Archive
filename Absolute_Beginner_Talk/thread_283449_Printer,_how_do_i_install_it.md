### Post by Bartender on 2006-10-24
Matt, what have you tried so far?
Is it USB?  I'd try plugging it in and turning it on.  Go to System>Administration>Printing, click on "New Printer", wait to see if gnome-cups-add finds the printer and digs up some drivers.  If it does, just follow the steps.  Print a test page, open OpenOffice and type in some text then ask it to print, open GIMP and find a picture to print, etc.
The scanner program is Xsane, under Applications>Graphics. Ask it to detect a scanner and see what kind of functionality you get.
EDIT: I've got an old HP psc1350xi, with a memory card reader.  Didn't expect that to work, but it did.  Just try it and see if a USB icon pops up on the desktop.
