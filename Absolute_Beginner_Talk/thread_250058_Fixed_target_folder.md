### Post by steve.horsley on 2006-09-03
No.

Maybe.

You can install software wherever you want, but the pre-packaged software (.deb packages like you install using synaptic) is already preconfigured as to where it will go. And you probably want to stiick with the pre-made packages until you get a little more experienced.

In general, I install the pre-made packages, and they do tend to scatter files around somewhat. Any pre-compiled binaries (rather than in .deb packages) I install myself, I tend to put in /opt or /games.

P.S. 
No. You cannot pre-configure a destination that all installatinos will go into. You either go with the pre-ordained destinations in the .deb, or specify yourself when you unzip the software.
