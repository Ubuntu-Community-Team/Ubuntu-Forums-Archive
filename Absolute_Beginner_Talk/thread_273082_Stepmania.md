### Post by fisherman783 on 2006-12-05
> **Olly1234 said:**
> Hi everyone, I have downloaded stepmania binary, and run the executable file and it works fine, but how do I actually install stepmania so that, fr example, it will come up under my games tab. Thanks in advance.

You can manually add Stepmania to your Games menu.  Right-click on the Applications menu and select "Edit Menus" from the list.  Then click on Games from the list at the left, and click the "New Item" button on the right.  For the Command field, enter the path of the "stepmania" file in your main StepMania directory.

Is that what you were looking for?
