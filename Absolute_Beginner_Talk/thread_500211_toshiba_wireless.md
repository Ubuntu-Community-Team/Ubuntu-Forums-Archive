Hello, I have a Toshiba laptop with an Atheros card.  When you install Ubuntu you may have to go in an enable the HAL in the restricted drivers under Administration.  My Atheros cards works perfectly under Ubuntu.

Once this is done your wireless should work.

If not you may need to install the madwifi drivers.

[http://www.stchman.com/ath_drv.html](http://www.stchman.com/ath_drv.html)

Hope this helps.
