### Post by jerrylamos on 2007-05-07
I don't have Ubuntu up at the moment, but my Firefox font size got fixed by doing (this is from memory) System, Preferences, Font, Details and set the font resolution to 96.  That fixed my Firefox Bookmark sidebar font size and several other font sizes as well....
