### Post by dbd on 2007-01-11
> **geek_Man said:**
> Just a question, but what IS so great about vim? Don't you have to run it in the terminal? (BTW, I love the terminal, I have nothing against it.)

If you run gvim instead of vim then you can run vim in a window. As far as I know gvim is identical to vim other than the fact it has a window and you can therefore use the mouse to press buttons and stuff, and is generally slightly nicer to work with.

And what is so great about vim, for me, is that your hands never need to leave the keyboard, or even leave your typing position, since you can navigate with hjkl, so its quicker once you have it all worked out.
