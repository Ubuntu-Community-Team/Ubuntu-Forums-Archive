### Post by p_quarles on 2007-07-19
Ubuntu works very well. Unless you're doing something extremely specific in Windows (and I assume you would have mentioned this), it's a good replacement.

Migrating from Windows to Linux can present some problems, though:
1) MSWord .doc files are readable, but the formatting is usually screwed up.
2) Some hardware doesn't work very well. The biggest problems are wireless cards, printers, webcams, and recent ATI video cards. 
3) It's a pretty steep learning curve. If you get frustrated easily, go with XP. If you enjoy a challenge, go with Linux.

In any case, you should definitely research the compatibility of your system's hardware component's before deciding.
