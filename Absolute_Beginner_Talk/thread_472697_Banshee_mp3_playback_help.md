### Post by dptxp on 2007-06-13
Normally, if you open a file by right clicking on it and select the program using 'open with' you should be able to switchover to your preferred program.

To download codecs, click add/remove, select all from top right, and choose the gstreamer codecs (I use the first two) in audio video.
