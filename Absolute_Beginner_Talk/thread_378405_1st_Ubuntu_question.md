### Post by milton1 on 2007-03-07
difference between ubuntu and kubuntu is the desktop environment; gnome vs. kde. Which one you choose is personal preference. I like kde because it includes several of my favorite programs (kile, amarok,...), and because I like the interface better.

As for command differences, if your class is about unix, you most likely will be using commands that will not differ much at all from fedora to ubuntu. You may find small changes in things like log placement, etc, but the core functionality will be the same.

### Post by Bagnaj97 on 2007-03-07
The comments should be the same between Fedora and Ubuntu. The only commands that will be different that I can think of are the commands for installing new software, as Fedora uses a package management system called RPM, whereas Ubuntu uses the Debian dpkg package management program, with apt-get to handle dependencies and downloading the .deb packages.

The only difference between Ubuntu and Kubuntu is the desktop and default set of packages installed. Which one you use is matter of personal preference, and you can always install one on top of the other, which gives you a choice of desktop at the login screen.
