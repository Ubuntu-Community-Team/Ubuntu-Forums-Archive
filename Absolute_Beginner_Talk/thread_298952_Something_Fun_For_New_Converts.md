### Post by Orwell on 2006-11-13
Hi all, 

Being new to Ubuntu (and Linux!) myself, I've been trying all kinds of new things. Came across a fun game for those of you who may fancy yourselves the next Jimmi Hendrix. The games called Frets On Fire and can be downloaded for free from [http://games.internode.on.net/filelist.php?filedetails=4395](http://games.internode.on.net/filelist.php?filedetails=4395)
I've just given it a go myself and it is a whole lot of fun.

In order to get it to work, you will need to ensure you have various audio files installed. It's as easy as going into Synaptic Package Manager, searching for 'audio' and selecting: libaudiofile0, libaudiofile-dev, libaudio-dev and libaudio2. Once checked, install them.

Unpack the downloaded game to your desktop and enter **./FretsOnFire** in the desktop's Terminal directory and the game should start.

Hope this advice helps, just trying to pass on some help that has been given to me. Enjoy!:)
