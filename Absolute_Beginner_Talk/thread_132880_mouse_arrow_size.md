### Post by Madpilot on 2006-02-19
You'll have to install another application and find a new cursor theme with larger shapes, but it's quite straightforward. The UDFS has a [short howto]("http://doc.gwos.org/index.php/Desktop_EyeCandy#HOWTO:_INSTALL_CURSOR_THEMES_ON_UBUNTU.2FGNOME").

There are lots of cursor theme packs available - [gnome-look.org]("http://www.gnome-look.org/index.php?xcontentmode=36") is one good site.
