### Post by marpstar on 2008-03-09
eSATA and Firewire are two interfaces used for a number of different devices.  While Firewire is typically used for a wide variety of devices (external hard drives, professional soundcards, etc), eSATA is only external hard drives.  

[SATA]("http://en.wikipedia.org/wiki/Serial_ATA") stands for Serial ATA, and is a newer interface used for internal hard drives within a computer.  eSATA simply implies that it is **e**xternal SATA.  It is the fastest connection available for external hard drives as it rivals the speed of internal hard drives.

Firewire is a much more broad interface, but for longer file transfers, it has shown to maintain it's bandwidth longer than USB 2.0.  

I would use the eSATA for the drive, if your computer has an eSATA port.  Most newer machines don't have these ports yet.  If all else fails, I'd go Firewire > USB2.
