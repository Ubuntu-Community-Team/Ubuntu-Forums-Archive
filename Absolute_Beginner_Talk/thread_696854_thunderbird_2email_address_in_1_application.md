### Post by Thingymebob on 2008-02-14
I have 5 email addresses set up in thunderbird
goto Edit>account settings and you can set up as many as you want and configure different SMTP servers to use. You can set them all to store messages in 1 mailbox (Local Folders) or keep them seperate. I don't think you can set Thunderbird to switch between at startup though.
