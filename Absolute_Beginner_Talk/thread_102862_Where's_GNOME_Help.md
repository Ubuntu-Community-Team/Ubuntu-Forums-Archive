### Post by 5-HT on 2005-12-13
Sorry about your issue with gnome help.

My suggestion would be to try either: 
```
 yelp 
``` 
or
```
 /usr/bin/gnome-help 
```

In a terminal.

If either of them work, something may be wrong with your help launcher in the menu, but you could always modify that or create your own.

If that still doesn't work, unfortunately I'm at a loss. 

Sorry, but possibly something's wrong with your HTML rendering capabilities on the system or maybe the gnome-help files themselves.

One thing to try may be to install it again which may fix the issue (from searching the package cache it seems to be "gnome-help").

Then again, I'm extremely new to Linux, so someone else may have much better suggestions.
