### Post by ekarni on 2006-03-23
What kind of error are you getting when you try to access the router through a browser?  I assume you've got web access working through the router, so we can rule out any major connectivity issues?

Here are some things to try:
1)  Can you ping the router directly?  At the command line, type *ping YourRouter'sIP*.  (Router IP is usually something like 192.168.xyz)  If it won't ping, the IP address you're trying is probably incorrect.
2)  Use a tool like Shields Up at [GRC]("http://grc.com") to check if the ports are open from the outside.
