### Post by Axces on 2006-04-13
Well, I'll try and give a little insight. I have moved around from distro to distro almost on a monthly basis. I used FC 4 for the longest time, when I tried Ubuntu it stuck. This may be partially due to the fact that I only use my Linux box as a server and have a mac as a desktop, so access to it is limited and kind of a pain. From what I have heard about FC 5, it seems like they are trying to give you the bleeding edge of Linux with all the whistles that hinder what Linux is all about, performance. If you want to play around with little trinkets that take away from your actual task performance, FC 5 is probably where it&#8217;s at out of the box. On the other hand, Linux is Linux; you can install the same things on Ubuntu if you really cared. Ubuntu gives you the user friendliness and pretty darn good performance along with the best package management system I have ever used. I guess what it comes down to is totally preference, and you&#8217;re posting in an Ubuntu board, so my votes with them.

### Post by pikespeakhiker on 2006-04-15
I have been a Fedora user since FC3.  As mentioned above FC5 is a nice improvement over FC4.  I am not a zealot.  It is not perfect, but it suits me well.  I use it for a server, not a desktop.  I have found it easy to use, with nice support from repositories and active forums.

I have been using Breezy (in Ubuntu and Edubuntu form) for about a week.  I like it as well.  The install was nice, although it was not as pleasant to use as the graphical loader of fedora (which, I realize, some people have to disable if it isn't compatible...I personally have never had this issue on multiple computers).  The LVM and disk partioning is more straightforward with the graphical environment, at least for a first time user.  I really like Ubuntu as well, and look forward to Dapper.

Overall, it is way to early for me to pick one over the other.  But one first impression is that it seems kinda weird to have a system that is supposed to be much more "stable" and easy to use than FC, but have big articles about how manual work is required to get a basic item like Firefox upgraded.  I wish I had enough time to try Ubuntu or Kubuntu as a server, so that I could establish a more informed opinion on which is best for my purposes.  Maybe after dapper is out...
