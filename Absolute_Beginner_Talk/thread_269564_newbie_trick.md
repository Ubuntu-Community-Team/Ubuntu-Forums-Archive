### Post by new_foo on 2006-10-01
As a newbie, this trick is new to me, but perhaps not new to you all.

I don't load up flash player on my windows browsers or ubuntu browsers because I really wonder about what it is doing.  In windows it requires a restart when installed for the first time which leads me to believe it loads a device driver.  When uninstalled and then reinstalled, it doesn't require a restart which leads me to believe it doesn't remove the driver when unistalled.  Well, such behavior makes me think they are possibly up to "no good."  Just read what the opera web site has to say about plugins.  So I don't load it up even though many many web sites require it.  

So what I did is that in Ubuntu I created an additional account I use just for this kind of stuff.  I load up opera for windows in wine and then load flash player for that.  This makes sure that when not browsing it is not doing something bad.  Also, if it does something like mess up the .wine directory, I can just delete the directory and reload it.  Pretty cool.  Besides if I go to a site that loads something bad, it'll try to load some windows malware which hopefully won't go beyond the emulation.

Pretty cool.

Just a thought
