### Post by ugm6hr on 2007-12-21
> **skrribble said:**
> programs like "DeVeDe" and "ManDVD" let you take video files like .avi and burn them to a dvd to watch on a dvd player

I have used DeVeDe, which allows simple menu page creation easily for your DVD video.  It does need about 15GB HD space free to create a DVD (presumably dependent on the compression used), and takes a few hours to create the .iso.

Just make sure you select the correct options for screen format etc, since it can make widescreen images look squashed if you get it wrong (I speak from experience!)
