### Post by Paul820 on 2007-10-21
Did you burn the cd as an .iso image or did you burn it as data? You must burn it as an image file for it to work. Open the cd and have a look at the contents. If there is only one file than you burnt it as data. Brasero will burn iso's and so will k3b. A lot of software for windows will burn iso's aswell. You can download cdburner xp pro for free off the internet.
