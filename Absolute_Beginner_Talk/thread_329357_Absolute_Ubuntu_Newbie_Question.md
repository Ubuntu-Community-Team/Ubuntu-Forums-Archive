### Post by Balaam's Miracle on 2007-01-01
> **ricosuave412 said:**
> Okay, I just downloaded my ISO and I'm going to but it on a CD. Is it possible to mount the image to a Flash Drive or any other type of removable media? If not can I install the OS to a flash drive later on? Thanks.

I think you misunderstand something. You do not burn the ISO file to a CD, that's of absolutely no use.
An ISO file is an exact copy of a CD. Now you will need to recreate that CD so that you can boot from it.
To do this, use your CD buring program to open the ISO file and burn this to a CD.

If you do not have a CD burning program (besides the no good native WinXp stuff), then download the free version of Deepburner from [http://www.deepburner.com/?r=download](http://www.deepburner.com/?r=download). 
After starting the program it will give you the choice between 3 actions. The last one is "Burn ISO image", that's the one you will want. From there, just follow the yellow brick road.

### Post by Bartender on 2007-01-01
rico -
The easiest way to check this on a Windows PC is to simply open Windows Explorer, go to your download, and Explore it.  The download is one huge file.
If you Explore a properly converted CD, you'll see several folders and a handful of files.  If your CD is also one big file, you didn't burn it right.

For some people, converting the .iso is ridiculously easy.  The NTI burning software that came with my Sony DVD drive is so simple I was sure there was something wrong at first.   When I looked at the download, the icon next to it was the NTI icon instead of some generic Windows file icon.  I double-clicked on the file, NTI opened, asked me what speed I wanted to burn at (8X was the slowest option, fortunately that's worked so far), clicked OK, and the deed was done.
