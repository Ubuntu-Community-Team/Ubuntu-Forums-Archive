### Post by Rocket2DMn on 2008-02-02
That's a problem with X, you need to configure it.  Do CTRL+ALT+F1 and log into the terminal with the username and password you set.  This script will ask you a bunch of questions about your hardware, do your best to answer, defaults work for most things.  When asked about video driver, if you have an nvidia card, select "nv", and select "ati" if you have an ati card.  If you happen to have an onboard intel card, choose "intel", otherwise select "vesa" (this is like a backup driver that works on everything, but is limited in capacity).
When asked about screen resolutions, select your monitor's max resolution and everything less, using TAB to move and SPACEBAR to select.
```
sudo dpkg-reconfigure xserver-xorg
```
Then restart gdm (gnome desktop manager)
```
sudo /etc/init.d/gdm stop
/etc/init.d/gdm start
```
You should now be at the GUI login screen.

A lot of nvidia card users like to install the proprietary drivers afterward too (non open source drivers), ask if you have an nvidia card and are interested in that.
