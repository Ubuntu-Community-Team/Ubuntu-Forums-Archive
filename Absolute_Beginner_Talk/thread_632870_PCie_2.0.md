### Post by Radon on 2007-12-09
PCI Express 2.0 is backwards compatible to 1.0, just like Hypertransport 3.0 is to 2.0 and 1.0.  You can use the new graphics card (8800GT/HD3870) in an old MB, eg P35, and use an old graphics card (GeForce 6200) in a new board, eg X48.  

As you can see [here]("http://hkepc.com/?id=280&page=6#view"), there is negligible difference between PCI-E 2.0 and 1.1 for the 8800GT.
