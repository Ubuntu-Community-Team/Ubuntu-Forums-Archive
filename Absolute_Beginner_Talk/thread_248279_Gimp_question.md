### Post by robglinka on 2006-08-31
Ok, I know that kewldude606 has already answered your question, but I do want to add a WHY to this one.

If you look at the save menu, you'll see that "Select File Type(By Extention)" that means that it will try to automatically determine the file type you're trying to save as by what extension you put.

If you want to force it to use a file type, click the little arrow to expand that section, and you'll be able to pick a format from the options.

Hope this explains a little more.
