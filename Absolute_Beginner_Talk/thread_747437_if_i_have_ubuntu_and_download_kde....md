### Post by billgoldberg on 2008-04-06
Nope.

You will have kubuntu if you have ubuntu and install the kubuntu-desktop, not just kde.

I believe the command is 

```
sudo apt-get install kubuntu-desktop
```

Kubuntu is ubuntu with kde **and** the apps.
