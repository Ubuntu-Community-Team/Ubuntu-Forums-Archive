### Post by PmDematagoda on 2007-11-04
First burn the .iso image file to a CD, you can use this guide:-

[http://www.petri.co.il/how_to_write_iso_files_to_cd.htm](http://www.petri.co.il/how_to_write_iso_files_to_cd.htm)

Make sure you burn the CD at 4X or lower, a speed higher than 4X can corrupt the CD.

After you burn the CD, simply keep the CD in the drive and restart the system, when it starts up it will read the CD and will bring you to the Live CD menu. Select the first option.

Once the Live CD is loaded and you are at the desktop, simply click the Install icon to start the installation and follow the instructions given to install Ubuntu, enjoy:).
