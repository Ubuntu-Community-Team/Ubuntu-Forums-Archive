I have two sound devices, one as an onboard card, and the other as a normal card.  Currrently Ubuntu is set on the onboard sound card.  How can I change the settings to use the other card?  Oh, it has been detected by Ubuntu 6.06 - Breezy Badger didn't detect it.

### Post by Dr. Nick on 2006-06-24
you can disable onboard in the bios, that would probably be the easiest/best choice

Also look at System -> Prefrences -> Sound and see what the default card is

---

### Post by Denis on 2006-06-24
There is an option to change your default sound card. Go to System > Preferences > Sound. You can select your default souncard below. 

This should work for the Ubuntu sounds. Most programs however have options to configure the audio, so you have to set the right soundcard for each program.
