### Post by p_quarles on 2007-07-28
It's kind of difficult to say what's wrong without more information. But here's a basic checklist:
1) Is the system adequate for Ubuntu (256 MB of memory at the minimum, and probably no less than 800 MHz for the processor)? If the answer is no, your friend should try to install a lighter version of Linux.
2) How old is the hard drive? Those things don't last forever.
3) Did he check the MD5 after downloading? 
4) Did he burn the disk *slooowly *(like, no faster than 4x)?
5) Has he/can he try to use the alternate install disk instead of the live disk?

That's where I would start.
