### Post by dalani on 2005-10-05
Thanks The text below from that thread solved it
--------------------------------------------------------------------------------
> 
To change back to the location bar, go to Applications > System Tools > Configuration Editor.

Then, go to /apps/nautilus/preferences and click the checkbox next to "always_use_location_entry"
