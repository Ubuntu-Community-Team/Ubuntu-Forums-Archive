### Post by bluetracer on 2007-06-21
The cube should actually be activated by default.  Hold Ctrl + Alt and mouse button 1, then drag around.  See what happens.

To change themes you need to install Emerald.  There's a themes pack too.  Search for it on Synaptic.

### Post by bluetracer on 2007-06-21
Right-click the red Beryl icon in the notification area and choose "Emerald Theme Manager".

Or go to System / Preferences / Emerald Theme Manager
