### Post by Lunixfanboy on 2006-01-19
SWAT is short for Samba Web Administration Tool, and requires a web browser to access it. The Debian/Ubuntu package installed should do all the magic to get SWAT setup for you, so try pointing your browser to [http://localhost:901](http://localhost:901), as indicated in the man pages for swat, suggested by previous poster.
