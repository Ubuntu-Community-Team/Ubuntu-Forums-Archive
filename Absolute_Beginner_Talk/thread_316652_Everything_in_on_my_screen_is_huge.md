### Post by renzokuken on 2006-12-11
ok, lets try the vesa driver and see if it helps

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```

so we can replace the old file should it go wrong

```
sudo gedit /etc/X11/xorg.conf
```

go to the "Device" section and replace "savage" with "vesa"

save and close the file

press Ctrl + Alt + Backspace in order to restart X and log back in and see what happens.

if for anyreason you cant log back in or you get a black screen, simply restore your old xorg.conf and we can start again

```
sudo cp /etc/X11/xorg.conf.backup /etc/X11/xorg.conf
```
