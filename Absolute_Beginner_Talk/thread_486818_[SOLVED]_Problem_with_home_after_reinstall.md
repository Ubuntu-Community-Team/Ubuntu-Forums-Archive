### Post by Cypher on 2007-06-28
OK, it looks like your home partition of "/dev/sdb2" didn't mounted. So you'll want to do the following, at the terminal do
```

sudo gedit /etc/fstab

```
You will want to add something like the following
```

UUID=<big UUID string>             /home                   ext3    defaults        1 1

```
to the file. Get the UUID string by using 
```

vol_id -u /dev/sdb2

```

### Post by Cypher on 2007-06-29
Oh I'm sorry..I didn't read the information properly.."/dev/sdb2" is indeed being mounted under /media/hdb2. The location where a partition gets mounted can have any arbitrary name, so don't worry too much about that.

Go to "/media/hdb2" and see if it looks like it contains the contents of your "/home" directory from before the re-installation. If that is the case, then you will just need to modify the "'/etc/fstab" file and change the mount point of "/dev/sdb2" from "/media/hdb2" to "/home" and you should be in business..
