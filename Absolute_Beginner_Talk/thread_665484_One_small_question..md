### Post by monte48lowes on 2008-01-12
The file format is standard. It will work on either system. The only problem you might encounter is the format of the hard drive the pictures are stored on. Linux can read and write to a windows formatted hard drive. Additonally windows can read and write to a linux formatted hard drive with the proper program installed. 


[http://www.fs-driver.org/]("http://www.fs-driver.org/")

This installs in windows and allows you to read and write to linux EXT2/3 formatted hard drives.
