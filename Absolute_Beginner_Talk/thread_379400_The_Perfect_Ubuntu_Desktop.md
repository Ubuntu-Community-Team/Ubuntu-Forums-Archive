### Post by dptxp on 2007-03-08
Here are a few links that will get you through UBUNTU 6.10 (maybe later versions too).
I think that we should have all these put together somewhere.
I have the 64-bit installed and after long debate have decided to retain it. So if you got
AMD-64, I suggest you go for 64-bit. The LIVE CD ejects on shutdown, but the laptop
had to be powered off.

Select Manual Partition in STEP (4 or maybe 5)

If you got Windows, use the extended partition for logical drives. Delete the logical
drive (not extended partition) while installing in STEP 5 (use right clicks and choose),
use about 4 GB to 10 GB for UBUNTU and additional programs (set format to ext3),
from balance set 2048 MB or more for SWAP (swap format), rest as FAT32 for Linux
and Windows data. Do not forget to see that all is OK in next STEP, add the drives
missing from screen (enter  /media/hdaX) on left (X= the nimber you see on right, the new drive to be mounted) and proceed.

Windows to UBUNTU. Partitioning and Dual Boot:
[http://www.psychocats.net/ubuntu/installing](http://www.psychocats.net/ubuntu/installing)

After Installation :

In Case you do not see partitions in UBUNTU, you can edit:
[https://help.ubuntu.com/community/AutomaticallyMountPartitions](https://help.ubuntu.com/community/AutomaticallyMountPartitions)

The Perfect DeskTop shall teach you how to add and what to add:
[http://www.howtoforge.com/the_perfect_desktop_ubuntu6.10](http://www.howtoforge.com/the_perfect_desktop_ubuntu6.10)
