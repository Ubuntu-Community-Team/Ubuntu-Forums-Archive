### Post by Cypher on 2008-04-03
If you know what you're doing, then once you've logged in as the regular user by all means perform
```

sudo su

```
and you'll have access to everything on the system from the Terminal.

But typing SUDO before performing some Root function is a good thing and will prevent you from inadvertently messing up your system..
