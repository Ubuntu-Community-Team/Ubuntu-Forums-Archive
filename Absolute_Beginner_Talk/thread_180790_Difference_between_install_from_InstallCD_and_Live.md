### Post by aysiu on 2006-05-22
The live CD installs in a point-and-click way and is more system-intensive, as it also needs to run a live CD session. You cannot use the live CD to upgrade, only install.

The text-mode install CD is perfect for more tailored installations (OEM, expert, etc.) and for low-spec systems (128 MB of RAM or less). It's the same interface as the previous installers (Breezy, Hoary, Warty).

### Post by Jucato on 2006-05-22
From the download page: [http://cdimage.ubuntu.com/releases/dapper/flight-7/](http://cdimage.ubuntu.com/releases/dapper/flight-7/)

LiveCD Installer
> Desktop CD
The desktop CD allows you to try Ubuntu without changing your computer at all, and at your option to install it permanently later. This type of CD is what most people will want to use. 

CD Installer:
> Text-mode install CD
The text-mode install CD allows you to perform certain specialist installations of Ubuntu. This type of CD is intended for cases such as creating pre-configured OEM systems or setting up automated deployments; it can also be used as a package source for upgrading older installations.

So basically the difference is that the LiveCD just offers the basic install features for a regular desktop while the CD installer offers more features for a wider range of uses.

### Post by aysiu on 2006-05-23
[QUOTE=kuerbt]So, the InstallCD contains more packages than Desktop CD, while Desktop CD provide a more user-friendly installation interface. 
But, the Desktop CD contains enough application packages for an ordinary user; so Desktop CD is enough for a ordinary user, just like me.

Can I think so?[/QUOTE] It's not that the install CD has *more packages*--it just has *usable* packages--that is, you can use the packages on the install CD to upgrade from Breezy to Dapper in an existing Breezy installation.

When the live CD installs, it simply makes a copy of itself on to your hard drive. It doesn't have a repository of .deb files.
