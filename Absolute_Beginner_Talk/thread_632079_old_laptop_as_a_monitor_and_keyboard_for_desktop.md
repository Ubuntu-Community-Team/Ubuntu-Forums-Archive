### Post by mozetti on 2007-12-05
To answer your original question: Technically, it should be possible to use the laptop monitor, mouse, and keyboard as peripherals to a desktop machine. They all have to connect to the laptop's motherboard, after all.

But, this could only work if you are electronics/engineering-inclined. Basically instead of using standard cabling like external monitors/keyboards/mice, laptops use ribbon cables (very, very thin) with non-standard connectors on the end to plug into the motherboard. You'd basically have to trace the wires i the ribbon (or find a schematic that shows their functions) and make some sort of adapter to go from the ribbon cable interface to the standard desktop PC interface.

So, is it possible? Yes

Is it worth it? Probably not
