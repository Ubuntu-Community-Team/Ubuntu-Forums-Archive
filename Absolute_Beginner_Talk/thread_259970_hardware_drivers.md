### Post by CatKiller on 2006-09-18
You shouldn't need a driver for the router, since AFAIK that only puts out Ethernet packets that are widely understood. You might possibly need a different network card for wireless on the Ubuntu machine - I know nothing about USB networking, so I can't tell you if that should work with Linux or not. Search around and see if others have managed to get your network card working.

As to your scanner, I haven't tried to get one working yet, but the thing you need to look into is called [SANE]("http://www.sane-project.org/").
