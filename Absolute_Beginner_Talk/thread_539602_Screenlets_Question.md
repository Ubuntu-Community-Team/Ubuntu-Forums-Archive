### Post by FuturePilot on 2007-08-31
Go here
[http://forum.compiz-fusion.org/showthread.php?t=3460]("http://forum.compiz-fusion.org/showthread.php?t=3460")
and download the .rar file. It has many many extra screenlets in it including a couple weather ones. Unpack the .tar files inside and place the folder that has the screenlets name i.e. Weather, in /usr/local/share/screenlets/  then when you run the Screenlets Manager it should pick them up.
You might have to install unrar to unpack the rar file so
```
sudo apt-get install unrar
```
