### Post by Deadmode on 2008-01-05
7.10 Requires more space, because a lot of new features have been added.  384MB needed for 7.10 is absolutely bare minimum required to run ubuntu from a hard drive.  But I'm not quite sure what you are asking.  The system requirements page is talking about both requirements for installing and running.  There are no extra requirements.  I wouldn't recommend running ubuntu off of  live cd all the time though.  That just seems like it would be a pain.  If you need a linux distrobution for an older slower system you should look into puppylinux. 
[http://www.puppylinux.org/](http://www.puppylinux.org/)
