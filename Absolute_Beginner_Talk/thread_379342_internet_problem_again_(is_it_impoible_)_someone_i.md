### Post by Tomosaur on 2007-03-08
I looked though some of your posts. On the step where it says:

```

apt-get install build-essential

```

You need to put 'sudo' in front, so it looks like this:
```

sudo apt-get install build-essential

```

It will then ask for your password. It is the same password as you use when you first log on to your machine. You will not see it being typed however - so you need to be careful to spell it correctly.

If you've already done that step, I apologise, it's just that you said "It was impossible", so I thought you may have forgot to use 'sudo' too.
