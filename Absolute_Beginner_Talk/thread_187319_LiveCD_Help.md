### Post by confused57 on 2006-06-02
[QUOTE=Sazabirules]I finally was able to boot the Ubuntu image I put on a cd. Once I got to the main menu I have no idea on what to do. I want to run it on the CD until I know how to partition everything and not delete Windows. Can someone guide me through this?[/QUOTE]

The LiveCD is Dapper6.06-desktop i386 iso(if you downloaded the i386) and the install iso is Dapper6.06-alternate iso.  Check the iso you downloaded and see which one it is...if it is the desktop CD, at the main menu select the first option which says something about install.  This will load Dapper just into ram(probably need at least 512 Mb for it to run properly, with 256 it'll be quite slow), it won't install and will boot up from ram.  You can learn how to use Ubuntu using the LiveCD, you can browse the internet...just try to learn a little about the OS first.
There will be an icon on the desktop to actually install Dapper onto your hard drive,  if you decide to do so, read this guide first:

[http://psychocats.net/ubuntu/windowstoubuntu.html](http://psychocats.net/ubuntu/windowstoubuntu.html)

Don't do all this if you downloaded the Dapper6.06-alternate iso, this will install Ubuntu onto your hard drive.  I suggest using the Live CD until you feel comfortable, then go for it.

Note: Rather than completely shutting down, you can just insert the Live CD while running Windows and  restart your computer.
