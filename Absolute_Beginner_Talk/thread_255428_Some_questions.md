### Post by Metacarpal on 2006-09-11
Some answers:

#1: If you have at least 256MB of RAM, download the "desktop CD", which will run Ubuntu in your system memory without first installing anything to your computer.  That way, you can see if your hardware is compatible without having to install and find out the hard way.  With [so much hardware supported]("https://wiki.ubuntu.com/HardwareSupport"), I think you'll be pleasantly surprised.

#2: Your best bet is a program called [Cedega]("http://www.transgaming.com/index.php?module=ContentExpress&file=index&func=display&ceid=29"), available from TransGaming Inc.  Be aware that there is a $5/month subscription fee attached to this software, with a 3-month minimum.  After three months, you can cancel and the software will still work, you just won't get further updates.

Some Linux gamers have taken to calling this "extortionware", and encourage using Wine (a Windows-compatability layer) or CVS-Cedega (a free version of Cedega), but both are still in development and don't have a great track record for compatibility with the latest - or, indeed, any - games without serious tweaking.

#3. Ubuntu's installer is very friendly with installing Ubuntu alongside WinXP in a setup commonly called a "dual-boot".  [Here's a how-to]("https://help.ubuntu.com/community/WindowsDualBoot").  You can also find lots of help here in the forums if you need it.  :D
