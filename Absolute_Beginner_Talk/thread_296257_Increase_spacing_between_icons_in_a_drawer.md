### Post by hotbrainz on 2006-11-09
I use the panels and drawers on them and have set up a nice set of icons but they are by default very close to each other. Is there some way of 

1. Fixing the length of the drawer
2. Changing Spacing between the icons

Any help is most appreciated.

### Post by Ecthelion on 2006-11-09
You can do right-click > properties and adapt the number of pixels and so how big the things are in a drawer.

You can also use the normal menu separators. Putting them between icons increases the distance between two icons too...
