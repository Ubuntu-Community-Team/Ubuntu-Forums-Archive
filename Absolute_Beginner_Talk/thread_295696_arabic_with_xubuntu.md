Dear All,
I found this article about Switching the keyboard layout:
[http://linux.about.com/od/xubuntu_doc/a/xubudg24t01.htm](http://linux.about.com/od/xubuntu_doc/a/xubudg24t01.htm)

I edit the xorg.conf file , and change the "XkbLayout"= ar( no need for this step you can skip it, only you need to execute the command in the next step)

and every thing was great the problem that after execute the command 
**setxkbmap ar**

the terminal will write Arabic so you can't execute the  **setxkbmap us** again and switch back to the English.

so I add 2 launcher to my desktop the first one execute the  **setxkbmap ar**
to make the language Arabic and the second one execute the  **setxkbmap us**
