### Post by avik on 2007-07-06
You can get the .deb from [http://packages.ubuntu.com/breezy/otherosfs/wine]("http://packages.ubuntu.com/breezy/otherosfs/wine"), then install it using:

```
sudo dpkg -i *<name of .deb file>*
```

I'm not 100% sure it'll work, but it's a possibility.

And I'm guessing you're using 5.10 because you can't upgrade, but if you can, definitely upgrade.
