### Post by rbil49 on 2008-02-20
Looks to me that you've chosen to use xinerama as opposed to twinview to control your setup there. How did you come up with this xorg.conf?

I prefer to use the nvidia-settings to control this stuff and write the xorg.conf. Here's what I suggest you do ...

1) backup your existing xorg.conf file in case things screw up in the following steps:

open a terminal and do ...

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup

Now you'll have xorg.conf.backup to resort to if you need to.

2) in the same terminal launch nvidia-settings:

gksudo nvidia-settings

Setup for twinview and set your monitors to work the way you want. Can't remember where, but save to xorg.conf.

3) restart your GUI by doing:  ALT+CTRL+Backspace

HTH.

Cheers.

### Post by Hobo Joe on 2008-02-21
Do this:
```

sudo dpkg-reconfigure xserver-xorg

```
You don't need to be able to get to the GUI to run this command, and once you do it you SHOULD be able to get to the GUI. If you can't, run the command a second time and select 'vesa' as the driver.

Either that, or you could try 'sudo nvidia-xconfig'.
