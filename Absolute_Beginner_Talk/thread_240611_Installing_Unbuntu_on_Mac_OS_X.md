### Post by 3rdalbum on 2006-08-21
Jperez, for your future reference: The Ubuntu installer does not resize an HFS+ partition.

For resizing an HFS partition, see this thread: [http://www.ubuntuforums.org/showthread.php?t=89960]("http://http://www.ubuntuforums.org/showthread.php?t=89960"). It says "Breezy", but you should be able to do it with the Dapper Alternative CD, and possibly even with the Live CD installer if you alter the latter instructions a bit.
