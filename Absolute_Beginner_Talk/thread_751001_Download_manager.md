### Post by erginemr on 2008-04-10
If you want to stick to the wget backend, you can use **Gnome Transfer Manager (gtm)** as an alternative frontend, which performs better than Gwget in my system.
[http://gtm.sourceforge.net/](http://gtm.sourceforge.net/)

You can install it with:
```
sudo aptitude install gtm
```

Another good one, which uses multi-threading and has resume and scheduling capabilities: **wxDownloadFast**
[http://dfast.sourceforge.net/](http://dfast.sourceforge.net/)

Also the getdeb.net site has a copy:
[http://www.getdeb.net/app.php?name=wxdownload%20fast](http://www.getdeb.net/app.php?name=wxdownload%20fast)
