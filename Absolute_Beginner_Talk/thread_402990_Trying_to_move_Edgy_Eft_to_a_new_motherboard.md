### Post by houstonbofh on 2007-04-06
The system will boot fine.  However, what else is changing with the motherboard?  If you are useing the same video card, NIC, and sound, it will just work.  If those things change, parts may have "issues."  You can usually boot a live cd to access the net for help.  It will also have the correct config files to fix your problems.
