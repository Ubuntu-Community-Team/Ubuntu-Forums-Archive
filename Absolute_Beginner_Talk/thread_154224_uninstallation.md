Try using Synaptic (System>Administration>Synaptic Package Manager) and then search for that you want to remove then click on the box and select it for uninstall. You can also use terminal. For that type: "sudo apt-get remove <ap name>" Make sure you don't use quotes and you don't use the <> signs.
