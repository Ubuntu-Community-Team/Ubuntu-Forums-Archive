Emerald themes and compiz themes are the same (as far as I can tell). These give you the windows borders that can be transparent, where you can have the windows options on the left like on mac osx, ...

gtk 2.0 themes are the themes that give you the new panel theme, the windows look and feel themes, new borders, ... aka the themes that change everything.

Metacity is something like gtk 2.0 I think, I never used it.

all those themes can be downloaded on gnome-look.org

gdm themes are the login screen themes

beryl themes are the same as compiz themes but for beryl
