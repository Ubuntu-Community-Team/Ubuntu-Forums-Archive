### Post by HermanAB on 2008-02-25
GTK Apps will run on IceWM. (anything will run on IceWM).

A simple IceWM install will be about 850MB, so a 1GB / partition is possible without much sweat.

However, I'd suggest sticking 2GB RAM and a 8GB SD card in there and doing a full install - way more fun!
[http://aeronetworks.ca/eeepc-mdv-howto.html](http://aeronetworks.ca/eeepc-mdv-howto.html)

Cheers,

Herman
