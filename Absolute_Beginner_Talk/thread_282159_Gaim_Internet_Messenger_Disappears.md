### Post by Sef on 2006-10-22
A number of people have had problems with MSN and GAIM. Some of us (including myself) have switched to aMSN. It works with no problems.  If you want to download it, follow these directions:

A) Have you enabled Universe? (Universe is nonUbuntu GPL software, like aMSN.)  If yes, go to step 2.  If no go to step 1.

1) To enable the [reposititories]("https://help.ubuntu.com/community/Repositories/Ubuntu").

2) Go to step B.

B) Open the Terminal: Applications > Accessories > Terminal

C) Input these directions to install aMSN.

1) sudo aptitude update (enter)
2) enter password: the one that is used to login. (enter)
3) sudo aptitude install aMSN (enter)
4) aMSN should install with no problems
5) It is located here: Applications > Internet > aMSN
