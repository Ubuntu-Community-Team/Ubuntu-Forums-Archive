### Post by kinematic on 2006-11-26
if you store the torrent on a partition that ubuntu and windows can access you can resume the download in windows.
azureus will check the data that's already there and continue with the download because the data you've already downloaded isn't os dependant.
this offcourse also works the other way around in that you can resume a download you started in windows ;)
