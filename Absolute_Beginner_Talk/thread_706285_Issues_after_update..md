### Post by ajgreeny on 2008-02-24
I don't know if this is what you actually installed, but you now seem to have fairly normal looking xubuntu (xfce) desktop showing here.  If you started with a normal ubuntu with gnome, I'm a bit baffled as to how you've ended up with this now, unless you installed originally from a computer magazine disk, such as Linux Format, which usually has all three main desktops in its ubuntu iso files or live bootable dvds.

Log out and see if you can chose gnome from the options menu button on the login screen.  If so then you still have the standard gnome on your system and can set it as default when you login.
