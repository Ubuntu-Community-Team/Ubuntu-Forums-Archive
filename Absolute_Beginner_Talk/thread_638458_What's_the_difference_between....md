### Post by Rocket2DMn on 2007-12-12
Whenever you download an ISO for an OS, you should check the md5sum on the image file before you burn to ensure that it downloaded correctly.  Then you should burn the image to a cd at a slow speed like 4x, and use quality media if possible.
[HowTo md5sum]("https://help.ubuntu.com/community/HowToMD5SUM")
[Hashes to check against]("https://help.ubuntu.com/community/UbuntuHashes")
If that still doesn't work, then buying a cd or getting one from [ShipIt]("https://shipit.ubuntu.com/") would be a good idea.
