### Post by Majorix on 2007-04-02
You have to type in the correct values for the refresh rate of your monitor otherwise you will get the famous "no screens found" error. Look at the manual for your monitor or google the monitor's vendor and model and look for the refresh rates, both horizontal and vertical. Note them down. When it comes to the stage where it asks you about which mode you prefer to setup your monitor choose advanced. And type in those values you gathered. Now restart X and all should be fine.

> sudo dpkg-reconfigure -phigh xserver-xorg
This command works on all graphical environments in Ubuntu. The -phigh simply tells it to ask the "high" priority questions.

If when you say that you "can't start into Ubuntu," you mean that you can't see its graphical interface, do the following. Otherwise be more specific.

ok, now, if you can't see the X window system (shows some blue screen saying it couldn't start X) then press CTRL + ALT + F1. Now issue whichever command you like ( sudo dpkg-reconfigure xserver-xorg ... ) but make sure that you select "**vesa**" as the driver and any supported resolution.
Now, what you need to do is install the ATI drivers (since at the begginning, your post of you xorg.conf showed you were using the vesa driver)
Follow the instructions to do so here:
[http://wiki.cchtml.com/index.php/Ubuntu_Edgy_Installation_Guide](http://wiki.cchtml.com/index.php/Ubuntu_Edgy_Installation_Guide)

I've never really installed ATI drivers on an Ubuntu machine, and my guess is there's something like the nvidia-settings manager where you can easily change resolutions. If you're stuck on the driver isntallation, or done installing the drivers and still don't see where to change the resolution then get back to us.
