### Post by geek_Man on 2006-12-25
Ubuntu Server is pretty much Ubuntu only with server programs and no gooey desktop. Ubuntu Server comes with LAMP (Linux, Apache, MySQL, PHP), so if you were thinking about hosting a website or something, Server would be a good choice. If you are looking for a regular type of OS, Ubuntu Desktop is the one to get. Stuff like surfing the web, playing games and such. So unless you want to run a server, get the Desktop edition.

### Post by obrient on 2006-12-26
Yes, you can install from Synaptic and get Apache, PHP, and MYSQL up and running without much trouble. 

System>Admin>Synaptic Package Manager

Then choose Apache, PHP, and MYSQL. You should then see Apache and MYSQL running under System>Admin>Services.

Hope this helps.
