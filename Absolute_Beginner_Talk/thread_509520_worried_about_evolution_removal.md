### Post by tgoose on 2007-07-25
It's fine to uninstall. All that will happen is that you will lose a package called "ubuntu-desktop". Despite the name this is not vital to the running of Ubuntu! ubuntu-desktop just ensures that all default software is on your computer - since Evolution is installed by default, removing it means that ubuntu-desktop can't stay. But it's not a problem!

### Post by tszanon on 2007-07-25
If I recall correctly, it is advised that you have ubuntu-desktop (or kubuntu-desktop, xubuntu-desktop, or whateverbuntu-desktop) installed when upgrading the system. Since you will probably upgrade at most twice per year, I don't see any problems in reinstalling ubuntu-desktop just before upgrading, and then removing Evolution again.
