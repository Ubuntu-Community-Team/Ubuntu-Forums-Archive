### Post by rviccaro on 2007-11-16
I know this is an old thread but this might help someone:
if the .pub file is under 1MB, you can upload it to [http://convert.neevia.com/](http://convert.neevia.com/) and download it right back as a PDF for free...then you could try to find a linux tool for editing the PDF.
