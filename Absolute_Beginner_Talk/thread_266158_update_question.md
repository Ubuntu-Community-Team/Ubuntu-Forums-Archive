### Post by ewl1217 on 2006-09-26
It's entirely possible that there's just nothing being fixed at the moment (possibly with some more focus on the next release of Ubuntu, due next month). If you ever want to check for updates, and apply them if there are any, enter the following command at the terminal:
```
sudo apt-get update && sudo apt-get dist-upgrade
```
Just remember that you'll have to restart any updated programs for changes to kick in (Don't forget to reboot for kernel upgrades!). Now, if you don't get updates for a much longer period of time, then there's orobably another issue going on and you should post it here.
