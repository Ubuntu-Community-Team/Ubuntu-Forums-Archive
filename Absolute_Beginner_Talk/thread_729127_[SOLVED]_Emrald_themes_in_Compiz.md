### Post by alan34 on 2008-03-19
To install emerald go System - Administration - Synaptic Package Manager and search for emerald

right click and mark for installation ( should also install libemeraldengine0 )

Then go System - Preferences - Sessions click Add and in the Command box add

compiz --replace -c emerald &

Comment and Name boxes as you want and press OK. Logout/Restart and you should be away.

The Emerald Theme Manager is in System - Preferences
