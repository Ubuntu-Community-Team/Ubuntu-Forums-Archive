### Post by schiznik on 2008-01-26
which computer is the modem plugged into? if its the linux box, you will need [http://www.linux-usb.org/SpeedTouch/ubuntu/index.html](http://www.linux-usb.org/SpeedTouch/ubuntu/index.html) or similar to use it.

If its plugged into the windows pc, you will need to enable internet connection sharing so that the windows box becomes a router. on the linux box, you will just need to configure the ethernet interface
