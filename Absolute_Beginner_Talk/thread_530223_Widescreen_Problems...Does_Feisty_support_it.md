### Post by cawill on 2007-08-20
I use a widescreen monitor with the resolution at 1680 x 1050. Ubuntu does support widescreen monitors.

Use this command in terminal to select just the graphics driver and the resolutions you need.
```
sudo dpkg-reconfigure -phigh xserver-xorg
```

Then restart X (ctrl + alt + backspace) and you will now be able to use the resolutions you selected. It's not all that hard to do, though Gutsy Gibbon will be easier hopefully.

### Post by phyrewall on 2007-08-24
Change:

```
SubSection "Display"
		Depth		24
		Modes		"1024x768" "800x600" "640x480"
	EndSubSection
```

To:

 ```
   Subsection "Display"
        Depth       24
        Modes       "1680x1050" "1280x1024" ...
    EndSubsection
```

Add whatever resolutions you want to be able to use to the Modes line, in descending order.

Hope that helps.
