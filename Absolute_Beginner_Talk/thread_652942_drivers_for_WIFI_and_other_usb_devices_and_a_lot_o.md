### Post by espressopigeon on 2007-12-29
You may need to download a proprietary driver for your wifi card. Try System -> Administration -> Restricted Drivers Manager.

You can install software by going to Applications -> Add/Remove and choosing your application. You will receive security patches for the software installed, but the software is only upgraded to a new version when a new version of Ubuntu is released, which is every six months.

That's all I can help you with.
