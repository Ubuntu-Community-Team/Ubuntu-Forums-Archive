### Post by iain_colin on 2007-08-04
I trying to decide whther im ready for the big jump from XP to Ubuntu, and i just wondered whether anyone had any thoughts on what i decided as the big plus points and negative points for both?

**_Windows XP:_**

_Pro's_

Conditioning - I have basically been running Windows on all the computers i have had for the past 16 years, and have gotten used to the interface and the way things are done

Out Of Box Support: Almost every time i have installed XP it has recognised the drivers automatically, giving me a a fully functioning system. This is key for me as i have several components (wireless network, power management) that i need functioning. As well as this, it comes with the fully functioning codecs and media players, and will ven support more if i add the codecs.

Runs basically anything : From games, to office, to servers, if you want something, usually you can get it for M$
_Con's_

UNSTABLE!: I cannot leave my laptop on with XP for more than 2 days otherwise it completely crashes!

Cost: Most of the software for XP has to be paid for, unless i go for FOSS versions of paid programs (OpenOffice as an example, which im running Currently)

Lack OF Support: If something does go wrong, i always have trouble trying to get the problem fixed, if t is a new problem to me. Usually when this happens,  have to phone up M$ and stay on hold for a while.

**_Ubuntu_**

_Pro's_

Reccomendation: Most of my friends and colleagues have given up running Ubuntu, and so have recommended it to me. As well as this, i have run the live cd in a VMware box and, although it ran slow (only 768 ran, and mosst taken for xp) i was interested in the concept

Less CPU/Memory Usage: As far as i know Ubuntu uses far less memory than xp, and i have been able to run many more programs than i usually would.

Free (as in beer): I have no problem donating money to a project which i like and use, however i do have a problem with paying for one which doesnt work well!

_Pro's_

Lack of Media Support from Day 1.: The fact that you have to install the codecs with Automatix, and they are not automatically given to me. I run on a usage broadband plan, and the thought of downloading codecs when theyre already given in other OS's puts me off.

Programs off XP: I have quite a fe games and programs (such as VB6, DirectX games, and tools) that would have to be run under WINE or would not run at all. And as i said before, i am a complete novice to Linux and wouldnt rally know where to start.

### Post by robtg on 2007-08-04
You should run the operating system that runs the applications you use.  If you think of your computer as something that runs your applications, then your decision will be easier.  If you think of your computer as something that runs an operating system, your decision will enter the realm of a science-fair project.   Not that there's anything wrong with that, mind you.   

Warts and all, Windows XP generally works pretty well.   Ubuntu works pretty well too...if only I could go to Best Buy and buy a copy of Quicken for Linux.   I'd love to be able to wipe Windows off of every computer I own, but I know that, at least in 2007, I'm not going to find Linux replacements for all the applications I use.   

Make the jump to Ubuntu when it runs the stuff you need to run.    

-Rob
