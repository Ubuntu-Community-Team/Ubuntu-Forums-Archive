### Post by gautada on 2007-09-14
> **niceguy123 said:**
> And if they are, I just hookup via USB and then what?

1.) Add to your top panel the gpilot applet
2.) System -> Preferences -> Pilot? (I am not on my home machine that I use to sync to check the exact name): Use this to setup your palm pilot
3.) Connect the cables
4.) I have a button on my cable that I click to start the sync.

A couple of notes:

1.) Backup all data.  You have the option to just backup the device.  Do that first with all other conduits turned off.
2.) Make sure that either the PC or the device is clean or make the first sync from one to the other.
3.) Once the same data is on each device then set the conduit to synchronize
4.) Do this one conduit at a time.
