### Post by The Cog on 2007-12-17
It could be that your XP disc is not bootable. Perhaps try it and see if the windows installer boots (you don't have to actually do an install) on a different PC. If the XP CD is a copy, it may be an imperfect copy that can't boot. If it is an original MS disc then being faulty that way is much less likely. 

A virtual machine is where a computer runs a program that emulates (pretends to be) another computer, and runs programs inside that emulated computer. For instance, you might (on a PC) emulate an Apple computer and run Apple programs inside that virtual computer. Or you could run a virtual machine in Linux, and have that virtual machine boot XP and run XP programs. I do this for some programs that I need that don't have Linux variants.

### Post by The Cog on 2007-12-17
Opening the disk in another operating system is pointless. You have to be able to boot from the CD, and run the operating system (whatever that may be) that the CD provides. If the CD is not bootable, it is useless. So I suggest that you try and boot from it on another machine (without actually performing an install of course), to verify whether your booting problem is with the CD or the computer.
