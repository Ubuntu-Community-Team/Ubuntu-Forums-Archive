### Post by jaboua on 2006-10-07
> **chajuram said:**
> what is this game like? i want to play it, but dont seem to get time.
It's a strategy game - you have to fullfill a goal, usually wipe out the enemy leaders. To do that, you need to create units and take over cities. Cities will provide you with income, while units cost some money - both to create, and an "upkeep" fee. When your units fight, they gain XP - especially if they get to kill an enemy. When the unit gets enough XP, it will tranform into a higher level unit with more powers and HP. That should be the basics, find some time and install it!!!

Alt+F2 -> alacarte
Go to the Games menu, and press "New Item".
Enter a name, and the command "wesnoth" in these fields.
Then press "ok".
And a menu entry should get created.
