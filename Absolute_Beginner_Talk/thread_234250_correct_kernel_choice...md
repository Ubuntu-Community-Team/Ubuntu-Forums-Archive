### Post by kaffelars on 2006-08-11
If you want to use the 32-bit version of Ubuntu, the "best" kernel for you is the k7-smp. (Though SMP is enabled in the standard k7-kernel, the smp-package is a dummy package).

You can also use the 64-bit version of Ubuntu. The default 64-bit kernel also has SMP support built in. However, there are optimized 64-bit kernels as well, if you install 64-bit Ubuntu you probably want one of these (can't remember the names now, my workplace is Windows only :().
