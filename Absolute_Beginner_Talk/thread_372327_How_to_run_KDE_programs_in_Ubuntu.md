### Post by FyreBrand on 2007-02-28
You can definitely run KDE programs in gnome or gnome programs in KDE.  I use GIMP in KDE all the time.

Some KDE applications won't have full functionality in gnome because they interact with the K desktop environment (kio-slaves, etc) to provide further functionality.  So that might be a problem, but generally applications like Amarok and K3B work pretty good in gnome.
