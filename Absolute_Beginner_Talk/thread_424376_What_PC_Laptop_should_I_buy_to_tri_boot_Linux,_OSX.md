### Post by AusIV4 on 2007-04-26
> **Xarok said:**
> Did I not say "My school offers a Mac OSX intall CD for free" and is it not a fact that OSX can been run on a PC?
It can't be run on a PC easily or legally. If your school offers a free OSX install disk, it's either illegal, or only for the PPC architecture - there aren't any retail Intel OSX CDs because the latest Intel version of OSX has been included with every computer capable of running it. 

There is a hack floating around the web that allows you to run OSX on a PC, but as I understand it's fairly buggy and has problems with updates - and it's illegal.

If you want to run all three legally, your *only* option is a Mac. If you're okay with violating the DMCA to run OSX, it's going to be a pain in the *** to get it running and keep it running.
