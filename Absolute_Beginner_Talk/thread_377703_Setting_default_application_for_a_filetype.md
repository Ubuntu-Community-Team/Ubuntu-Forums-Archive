### Post by bash on 2007-03-06
I want to change the default application that opens when I click on a specific. In Windows I could do this pretty easily in a window where all the filetypes where listed and I could set the default application for the specific filetype. Where can I do this in ubuntu?

---

### Post by mikewhatever on 2007-03-06
Right click on a file, Properties, Open With, and choose the required application.

---

### Post by Obor on 2007-03-06
Just go to Nautilus (file browser) right click on the file of the type you want to change. Choose Properties - Open with  - and add application and mark it with the radio button to make it default.
