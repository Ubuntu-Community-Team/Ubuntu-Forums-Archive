### Post by goodfella on 2007-07-21
I think even if Ubuntu does not have equivalent programs to the ones you need you should still install it and switch to Ubuntu.  But as far as your needs are concerned here are my opinions:

Visual Basic 6.0 is a Microsoft Proprietary programming language so you most likely will not be able to use it in linux.

Depending on what you need to do in Adobe Photoshop you probably could do the same in GIMP.  It depends on your needs.

For editing and converting sound files, I use Audacity.  Although it took a little effort to get it working on my machine, it gets the job done.  Someone on this forum might know of a better sound editing program.

uTorrent is just a windows BitTorrent client and Ubuntu has an equivalent.

As far as gaming goes there is something left to be desired in Linux; however, some people as well as myself have been successful using wine to play games or run other software where there is no Linux equivalent.

You could also install vmware server on Ubuntu.  Here is the forum link that I followed:
[http://ubuntuforums.org/showthread.php?t=183209]("http://ubuntuforums.org/showthread.php?t=183209")

With vmware you could run windows while you are running Ubuntu and switch between the two as if you are switching between programs.  So if you need to do something in windows like program in vb just run windows with vmware and do what you need to do.  You can also dual boot Ubuntu and Windows if you want.  Hope this helps.

Like I said in the beginning, even if Linux does not have equivalents to anything in your list you should still install it.  Since you are learning VB 6.0 it leads me to believe that you might be a programmer.  Some people may disagree but learning how to maintain a Linux distribution on your own machine will make you a better programmer because by doing so you learn how to solve problems and learn how to understand manuals, and other forms of help which is key in programming.

### Post by anewguy on 2007-07-21
I'm relatively new at this myself, and I really think you should use a dual-boot system for a while.  Gradually work your way over to Linux, going for 1 thing at a time and getting used to it before moving on.  This may sound a little crazy to some, but it will save you the jumping in all at once and getting lost syndrome.  Also, I have looked for a Visual Basic type replacement for Linux myself, but so far haven't really found one.  Since it sounds like you are in a class on VB, I also would highly recommend using that purely in Windows, hence the dual boot.  You don't want to have potential problems introduced by the environment you chose when you are learning a language.  

I think this is the beauty of the LiveCD and the dual-boot - you can keep doing what you need/are used to in Windows as you work your way into Linux until you (hopefully??) can someday "turn-off" Windows and stick with Linux.  There really is no hurry to jump completely over - do things at your own time and pace, and in the way that is the most comfortable to you.:)

Good luck!!:)
