### Post by Delkster on 2007-05-17
> **Evegan said:**
> So do I need to get all the CDs before I do this? I'm pretty far back, meaning I'd need to do breezy, dapper, edgy and THEN fiesty. This scares me. 

No, you don't need the CDs if you have a (relatively speedy) internet connection. You can upgrade to a newer release over the network without having a single CD.

But yes, you do need to go through all releases. I've been upgrading my Ubuntu system since Hoary and it's now at Feisty, so it's possible, but there have been some changes in the system since then that may cause some headache after the upgrade unless you're at least somewhat familiar with Linux.

So, if possible, I'd suggest backing up your data, doing a clean install and restoring the data afterwards. You'd lose the configuration tweaks you've done but could keep all your actual data (photos etc.) if you can manage to fit that all onto some kind of backup media.

If you can't do that, see [upgrade notes in the wiki](https://help.ubuntu.com/community/UpgradeNotes) and the instructions it links to with regard to each single upgrade. The Hoary -> Breezy upgrade is a little complex and requires use of the command line, but there are quite good instructions for that in the wiki. Upgrades between more recent versions are generally friendlier.

Note that even a single step of the upgrade path (e.g. from Hoary to Breezy) can take quite a while particularly if your connection is slow. The Dapper -> Edgy upgrade required about six hours of downloading on my then 512k connection.

Also note that if you don't want to keep upgrading much in the future, you might want to just install or upgrade to Dapper, not all the way to Feisty. Dapper is a so-called LTS release (for Long-Term Support) and will receive security updates and support until 2009, so there won't be an immediate need to upgrade.
