### Post by fusiachi on 2007-03-18
Linux-based systems have a much smaller user base.  This translates into being a smaller target, as it were, for malicious code.  Furthermore, the transparency offered by open source software makes it easier to find and fix bugs/problems/holes/whatever.  As it stands, ubuntu/linux is considered to be pretty secure.  You're probably fine w/o a firewall or anti-virus, beyond the default.  Most end users don't run them, in my experience.  That said, there are a few available.



Edit: (Yes, I know "ubuntu" and "linux" are not interchangeable words, despite the above usage.  I don't care.)

### Post by johnc4510 on 2007-03-18
Anti-virus software is unecessary with linux. However a firewall is. Ubuntu comes with a firewall preinstalled. It is called iptables. It runs automatically everytime you boot. To configure iptables you have to access it through the terminal, and is difficult for new users. However, you can install a graphic interface called firestarter to ease configuration. It is available in synaptic package manager.
