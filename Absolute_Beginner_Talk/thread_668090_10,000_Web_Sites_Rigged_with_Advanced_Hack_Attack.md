### Post by fiddledd on 2008-01-15
I would think the **mebroot** rootkit can be defeated by write protecting your MBR from the BIOS, I only allow writing to MBR before partitioning or installing an OS, then I write protect it again.
The Javascript doesn't affect me as I only allow it on about 3 sites (including here), but I'm not the average user, I don't visit youtube etc, no online banking and all my downloading is done via Usenet. Also I have no personal details on any of my PCs, it's encrypted on Memeory Sticks and only mounted when needed. Yes I know it doesn't mean I'm safe :). 
On XP I use a LUA and don't allow any net access an the Admin account, any security updates are downloaded from MS using the LUA then installed with the Admin account. I don't have Antivirus software running, I schedule scans using ClamWin (it's free), I also run Rootkit Revealer and FSecure Backlight regularly. And AFAIK I've never had a Virus/Trojan.
But after saying all that I still don't feel secure, just a bit more secure than some other people. One security risk I haven't found a way to mitigate yet is ordering goods online. I'm at the mercy of the Websites security measures, and if they are no good then my personal details will end up in Russia probably.
