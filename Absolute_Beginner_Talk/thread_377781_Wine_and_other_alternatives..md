### Post by old_geekster on 2007-03-06
Another program that you can use is Cedega.  It is a commercial program that has a monthly fee; $5 per month.  There is a free version called Cedegacvs.  However, from my experience, it is as difficult to setup as Wine.

IMHO, gaming in Linux (any version), especially with Windows games, is extremely limited.  You can find native games if you read the lables.

Here is a link to a site that lists native Linux games:

[http://en.wikipedia.org/wiki/List_of_Linux_games](http://en.wikipedia.org/wiki/List_of_Linux_games)

Hope this helps.

### Post by Pikestaff on 2007-03-06
You can go to winehq and look up certain games to get an idea of how well they run under wine:

[http://appdb.winehq.org/](http://appdb.winehq.org/)

Although sometimes you may still have problems even with games that are given good ratings, so you have to be careful, but overall I think it will help you get a good feel for what Wine can do.
