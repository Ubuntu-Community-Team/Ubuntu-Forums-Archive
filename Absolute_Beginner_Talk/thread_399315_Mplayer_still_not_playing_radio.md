### Post by golovan on 2007-04-02
hi
I've had the same problem, maybe this will work for you:

First make sure vlc and totem mozilla plugins are not installed, if they are, remove them with synaptic. You only want mplayer-mozilla (or whatever it's called, can never remember:) )

Then go to a page with mediacontent, right click on the "mplayer box" in firefox, and click on configure. Untick "play media directly from site" and adjust the cache faders. Mine are around 4800 / 20. 

Hope this helps.
