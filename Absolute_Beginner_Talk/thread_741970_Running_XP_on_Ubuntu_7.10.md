### Post by lswest on 2008-04-01
Also, you may not be able to find drivers for the actual laptop you have for XP, but there is probably a thread somewhere in the world to install XP on that sony laptop, and you may just end up hunting down drivers for each piece of hardware in the laptop (i've done it with my HP laptop).  In the end though, XP just doesn't run well (battery-life wise) if you install it on a laptop that was meant to run Vista (especially with dual cores, etc.  Battery options are just rather limited).  Otherwise, if you run XP in virtualbox, it uses virtual hardware, so drivers shouldn't be a problem, however, few games, if any, will run in it.
