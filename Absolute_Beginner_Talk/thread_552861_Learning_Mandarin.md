### Post by por100pre1 on 2007-09-17
Some people use the QQ messaging network for learning Chinese languages. Read [here]("http://qq.bur.st/") for details and help setting up an account. Use Gaim/Pidgin or Eva to access the QQ network:

```
sudo apt-get install eva
```

**Do NOT use any QQ client under Windows, not even the official one!** They usually come with Adware and Trojans.
