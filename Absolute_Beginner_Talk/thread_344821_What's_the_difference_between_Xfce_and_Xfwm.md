... in more general terms, what's the difference between a desktop environment and a window manager?

---

### Post by IYY on 2007-01-23
A window manager is the actual engine that draws the window, allows you to drag them around, maximize, minimize... It manages your windows!

The desktop environment is a collection of applications, typically including a window manager, a file manager, panels and a desktop. These applications allow you to graphically work with your system.
