### Post by Paqman on 2008-03-05
The different architectures refer to whether your processor can handle 32-bit or 64-bit software. All 64-bit processors are backwards-compatible, meaning that they can run quite happily in 32-bit mode (this is done automatically, in fact)

So if you have a 32-bit CPU you have to use that kind of OS/applications, but if you've got a 64-bit you can use either.

32-bit will be referred to as x86 or i386. 64-bit will be referred to as x86_64 or amd64.

Hope that clears it up a bit.
