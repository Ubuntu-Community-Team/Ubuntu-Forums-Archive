### Post by equal on 2007-02-16
In fact, installing Edubuntu may be the best choice for you. It is Ubuntu, Gnome based and everything, with specific education-related packages installed by default. But yes, any software you might want to install is available in the repositories. Simply go into System > Administration > Synaptic Package Manager and do a search for what you need.

Alternately, you could install the Edubuntu-desktop package. Then when your computer gets to the login screen, you simply go into Session, and choose Edubuntu.

To install it, paste this into a terminal window one line at a time.

```

sudo aptitude update
sudo aptitude install edubuntu-desktop
```

Or just download the edubuntu .ISO and install it straight, seeing as I doubt you'll need both.

Good luck, and we're always here to help!
