Well clearly Wicd is too much for your hardware.  Unfortunately, the GUI scan for networks is not an option in Xubuntu - I don't know of a lighter app than WIcd (the one I haven't tried is wifi-radar).

If PuppyLinux works - then I'd suggest going with that.  

To be honest, I normally recommend Puppy for sub-128MB systems anyway.  Not quite as easy to install additional apps (Pupget is not as full-featured or as comprehensive as Synaptic), and you are forced to log-in as root automatically, but it will serve well as a spare browser machine.

Make sure you create a 256MB linux-swap partition, and do a "Full" (rather than "Frugal") install for best performance.  If you're not going to be saving files on that laptop, the difficulties with upgrading PuppyLinux versions is irrelevant (they tend to have a new release every 2 months!)
