### Post by petri on 2007-02-20
If you are just going to surf and e-mail you don't need to use Firestarter or a Guarddog or whatever (firewall). In fact, if you aren't sure what you're doing you might open up your system to hackers and that isn't the goal, right? 

If you are going to use some program to download torrents or setup an webserver then you will need the frontend to Linux iptables (so called firewall). Then you have to do some studying how to configure it. 

Antivirus - Linux does not have viruses, yet. Some do use antivirus program to protect the Windows machines but I think that is the Windows users job. 

Spyware - doesn't exist , yet.

There are allways some vulnerabilities in every OS but Ubuntu/Canonical is one of the fastest to fix those so I'm not worried.
