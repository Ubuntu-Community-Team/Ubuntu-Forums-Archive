### Post by Kobalt on 2007-04-04
The grub boot loader will be installed automatically with Ubuntu. What you actually need to worry about is your partitioning (first link). Once you've made space both for Ubuntu and Windows, you'll just need to specify Ubuntu where to install (third link) and it will install itself and the boot loader, configure it properly, and : voilà.
You'll reboot, have a menu from which you'll chose to boot whether on Ubuntu or windows.
