### Post by kakalaky on 2006-12-03
[http://download.macromedia.com/pub/labs/flashplayer9_update/FP9_plugin_beta_112006.tar.gz]("http://download.macromedia.com/pub/labs/flashplayer9_update/FP9_plugin_beta_112006.tar.gz")

Just extract libflashplayer.so, and throw it in /.mozilla/firefox/plugins/ to install for only yourself, or as root, extract it to /usr/lib/firefox/plugins/ to install systemwide.

### Post by Azakus on 2006-12-03
You could use [SwiftFox]("http://getswiftfox.com/") if you really want a stable Flash version. Swiftfox is 32-bit and runs on 64-bit systems, so you'll have full compatibility. I tried flash 9 on 64-bit firefox, but It never worked right (sound was a hassle). It is also optimized for each processor differently, so as to best be "swift" on any system.
