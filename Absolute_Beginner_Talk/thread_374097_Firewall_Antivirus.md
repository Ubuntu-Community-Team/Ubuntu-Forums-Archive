### Post by scxtt on 2007-03-02
antivirus:
[indent]if you aren't sharing files w/ windows users - i wouldn't worry about it (honestly, i wouldn't worry about it @ all, but that's just me) -- i've heard about ClamAV, check it out?[/indent]

firewall:
[indent]gnome: firestarter {sudo apt-get install firestarter}
KDE: kmyfirewall {sudo apt-get install kmyfirewall}
and if you're brave, configure your own iptables (not for the faint of heart :-p)[/indent]

also, if you use a router and don't forward any ports - a firewall isn't really necessary either ...
