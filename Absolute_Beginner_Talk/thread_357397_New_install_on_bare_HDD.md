### Post by etank on 2007-02-09
You will need to use some kind of CD burning software to extract the iso image to a CD. If you have access to another machine that has a CD burner then that is the way to go. If you do not have burning software and would like something that is free, I would recommend [burnatonce]("http://www.burnatonce.net/downloads/").
