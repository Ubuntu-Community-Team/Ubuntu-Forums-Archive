### Post by weblordpepe on 2007-12-19
Well firstly - you'll find that server stuff is very much glued to command-line. This is because not every server is going to have a monitor attached to it.
Anyway- you can either install each thing by using synaptic, or **apt-get install** or you can actually use Synaptic to tell your Ubuntu a role to play. Go to Synaptic, then Edit, then '**Mark Packages by task**'.

You'll have heaps of options for various different tasks. Eg a **LAMP** server, or multimedia box etc.

Although if you're serious about using Ubuntu as a server, grab the Ubuntu server edition or at least the alternate installer.
