### Post by kragen on 2007-05-10
do you have a nvidia graphics card? if so, have you installed the official nvidia graphics card drivers? I think that will probably fix it for you, and the official nvidia ones are better anyway (as long as you dont have anything against the fact that they aren't free).

To install them, I think you just need to use

```
sudo apt-get install nvidia-glx nvidia-kernel-common
sudo nvidia-xconfig
```

Look here for more info:

[http://ubuntuguide.org/wiki/Ubuntu:Feisty#How_to_install_Graphics_Driver_.28NVIDIA.29](http://ubuntuguide.org/wiki/Ubuntu:Feisty#How_to_install_Graphics_Driver_.28NVIDIA.29)

### Post by dbott67 on 2007-05-11
I had a similar problem with my old Toshiba laptop.  It's likely related to the refresh/modeline settings.  There is a very comprehensive HOWTO here by heimo:

[http://ubuntuforums.org/showthread.php?t=83973](http://ubuntuforums.org/showthread.php?t=83973)

_**Method #1 (from above thread):**_

[B]How to adjust position of your screen?
[/B][SIZE=2]open terminal[/SIZE](Applications->Accessories->Terminal), run xvidtune (type: "xvidtune"), adjust the screen and hit Show-button. You'll see a line with something like this on the terminal screen:
```
"1280x1024"   157.50   1280 1332 1492 1728   1024 1025 1028 1072 +hsync +vsync
```Next you should:
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup
sudo nano /etc/X11/xorg.conf

``` In Monitor section, add the above line with a prefix "Modeline", like this:
```
Modeline "1280x1024"   157.50   1280 1332 1492 1728   1024 1025 1028 1072 +hsync +vsync
```That should do it. There should be no need to restart X if you did make the change (hit Apply in xvidtune), but you should test that this new change works. Hit ctrl+alt+backspace to restart X. If it doesn't work, you can copy back the old configuration file using:
```
sudo /etc/X11/xorg.conf_backup /etc/X11/xorg.conf
``` and restart X using:
```
sudo /etc/init.d/gdm start

```_**Method #2:**_

Enter your desired resolution and refresh rate (both must be supported by the monitor, of course) at the modeline generator site:

[http://www.bohne-lang.de/spec/linux/modeline/](http://www.bohne-lang.de/spec/linux/modeline/)

Then paste the result into your xorg.conf file as per the instructions.

-Dave
