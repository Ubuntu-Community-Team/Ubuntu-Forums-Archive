### Post by xtacocorex on 2006-07-09
I'm hoping that this will inform new users of the differences between Ubuntu, Kubuntu, and Xubuntu since it seems that this comes up a lot in the forums. I don't know if someone has done this (an info page), but hopefully it will be of use to at least one new user.

It is hard to discuss this, without being biased, so I included the links to their respective websites as a way for one to get more info.

_Ubuntu:_
This is the default distribution put out by Canonical which uses the Gnome Desktop Environment (DE). Information about Gnome can be found here: [http://www.gnome.org/](http://www.gnome.org/)
Gnome uses GTK for it's GUI's, the GIMP Toolkit, which is written in C.
The link to the Ubuntu Wiki page is: [http://wiki.ubuntu.com](http://wiki.ubuntu.com)

_Kubuntu:_
Kubuntu is Ubuntu, but instead of Gnome, it uses KDE for the Desktop Environment. Information about KDE can be found here: [http://www.kde.org](http://www.kde.org). KDE uses Qt for it's GUI's, which was proprietary for a while, but released a license that allows for free use of the libraries. 
The link to the Kubuntu Wiki page is: [https://wiki.ubuntu.com/Kubuntu](https://wiki.ubuntu.com/Kubuntu)

_Xubuntu:_
Xubuntu is Ubuntu, but uses XFCE Desktop Environment instead of Gnome or KDE. XFCE uses GTK, just like Gnome, but is lighter and meant to run on older hardware, not saying that you can't run it on your spiffy new machine. Information about XFCE can be found here: [http://www.xfce.org/](http://www.xfce.org/).
The link to the Xubuntu Wiki page is: [https://wiki.ubuntu.com/Xubuntu](https://wiki.ubuntu.com/Xubuntu)

_Screenshots:_
For Gnome screenshots: [http://art.gnome.org/screenshots/gnome214](http://art.gnome.org/screenshots/gnome214)
For KDE screenshots: [http://www.kde.org/screenshots/](http://www.kde.org/screenshots/)
For XFCE screenshots: [http://www.xfce.org/index.php?page=screenshots&lang=en](http://www.xfce.org/index.php?page=screenshots&lang=en)

_Applications:_
Applications written for a specific DE will work in another DE, for example, K3B, a popular CD burning program will work fine in Gnome, and Gaim, a Gnome program for Instant Messaging, will work in KDE.

_ Installation:_
erik397 has a nice thread about installing the different meta-packages that will allow you to use any one of the DE's mentioned here. [http://ubuntuforums.org/showthread.php?t=205002](http://ubuntuforums.org/showthread.php?t=205002)

For help on the graphical differences between Gnome and KDE, aysiu has a nice page comparing them: [http://www.psychocats.net/ubuntu/kdegnome.html](http://www.psychocats.net/ubuntu/kdegnome.html)

This is the link to aysiu's thread on a Gnome/KDE comparision page: [http://ubuntuforums.org/showthread.php?t=196231](http://ubuntuforums.org/showthread.php?t=196231)

benplaut has a thread with a number based comparision of the different DE's and Window Managers, it may have some bias, but at least it's there for one to look at: [http://ubuntuforums.org/showthread.php?t=126618](http://ubuntuforums.org/showthread.php?t=126618)

It doesn't matter which one you install, if you have enough hard-drive space, why not try them all to get an idea of how they work. They each have their pro's and con's, but if you really want to know about something, it's best to try it out. Would you buy a car without test driving it?

For answers to some of your questions, you might try: [https://help.ubuntu.com/community/CommonQuestions](https://help.ubuntu.com/community/CommonQuestions)

In the end, it's all Linux and the choice is yours.

*Note:
I didn't include Edubuntu since I have not used it and I figure that it is more of a specialized Distro, then a general one. Not saying that it's bad, I just have no experience with it or know what makes it different.
