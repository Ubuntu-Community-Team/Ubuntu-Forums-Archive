### Post by LaurelLynn on 2007-05-04
Dear em11488,

If you really get stuck, consider trying an older release of the OS, Edgy 6.10, or even Dapper 6.06LTS (The version I and many others are still using).  Where windows has  on average of about 5 years before upgrages, there is another version of Ubuntu every 6 months.  That does not mean it becomes obsolete every 6 months. Dapper is still going to be supported till 2009.

The versions are not so different that you need the latest and greatest. Older versions are often more stable. Bottom line is find the one that works.  Maybe try another on a spare partition if you have one (I make a point of leaving one for that purpose). If it offers more of what you what switch.

There are at least three versions of Ubuntu worth considering. Some hardware works better with the newer versions. Some works better with the more mature versions. You deserve the one that works best for you.
