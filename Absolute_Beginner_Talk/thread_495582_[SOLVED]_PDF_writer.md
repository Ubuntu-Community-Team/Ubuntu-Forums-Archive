### Post by csimons on 2007-09-06
I've used PrimoPDF on Microsoft platforms, which is free and is very good quality.  This software is not available for Linux as far as I know, but they have an "online" version that will work with documents, pictures, and some odd data types.  I believe would be just as high quality.  You may want to check it out:

[http://online.primopdf.com/](http://online.primopdf.com/)

Additionally, a list of cross-platform PDF software, which descriptions, can be found here:

[http://en.wikipedia.org/wiki/List_of_PDF_software](http://en.wikipedia.org/wiki/List_of_PDF_software)
