### Post by ramjet_1953 on 2007-07-15
There really is no simple answer to your question here.

What I would do is backup your /home directory first.

There is a possibility that you may get kernel panic when you install this HDD into a new PC.

Certainly it is worth a try, but if you need to do a re-install, you can just copy across your /home directory.

You could also use the AptOnCD package to make a local repository of your downloaded applications, making a re-install much faster. The aptoncd package can be downloaded using Synaptic.
