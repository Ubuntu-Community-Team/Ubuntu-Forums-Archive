### Post by Ek0nomik on 2007-06-02
From the log on screen you can log in as root.  However, it isn't suggested unless you are positive about what you are doing.

If you touch a single file in a way that it doesn't want to be touched, you might find yourself with a system that is no longer working.

Having the sudo feature is a safety precaution.  Personally I don't find typing sudo to be a hassle, but that's just me.
