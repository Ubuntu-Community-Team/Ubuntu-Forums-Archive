### Post by 56phil on 2006-12-03
Welcome to Ubuntu.

Linux and Windows are not compatible. However, Linux does have a Windows emulator called wine. Go to [wine HQ]("http://winehq.org/") for information about any windows programs you want to run in Ubuntu. 

That said, there are Linux programs that will do every I need and most of the things I want. I use Open Office instead of MS Office, Firefox instead of Internet Explorer, etc.

Hope this helps.
