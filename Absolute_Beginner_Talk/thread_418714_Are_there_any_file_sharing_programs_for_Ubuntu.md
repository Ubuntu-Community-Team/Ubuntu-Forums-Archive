### Post by old_geekster on 2007-04-22
> **Neil_The_Newbie said:**
> I used to use Shareaza and Limewire when i had that OTHER operating system. Are there any  file sharing programs i can use with Ubuntu?

Thanks for your help in advance.
If you know how to use "Limewire", then, install "Frostwire".  It is a clone of Limewire for Linux.

I would suggest installing "Automatix2".  It has "Frostwire" in the "File Sharing" area.

Here is a link that will help you find and install many helpful programs:

[http://ubuntuguide.org/wiki/Ubuntu:Feisty](http://ubuntuguide.org/wiki/Ubuntu:Feisty)

See "Section 1.5.1" for help on installing "Automatix2".
