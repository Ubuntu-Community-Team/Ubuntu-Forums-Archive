### Post by SickTwist on 2005-12-20
Look into [MythTV]("http://www.mythtv.org/"). It is Free Software that has functionality similiar to Tivo.

Also be *very* careful about which TV card you purchase--especially if you don't have much Linux experience. Some cards can be a pain to set up.

The Hauppauge WinTV-PVR-250 card mentioned in the article that you linked works with Linux but I'm not sure if it is the best choice for MythTV. The PVR 250 has an onboard MPEG-2 encoder so it does not work with some TV programs like [TVtime]("http://tvtime.sourceforge.net/").

EDIT: I forgot to mention, there is a site called [Abarbaccia.com]("http://www.abarbaccia.com/") that is dedicated to using MythTV on Ubuntu.
