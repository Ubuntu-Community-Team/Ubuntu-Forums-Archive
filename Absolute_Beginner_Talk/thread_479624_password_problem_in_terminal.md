### Post by atria on 2007-06-20

### Post by FleetAdmiral74 on 2007-06-20
Above poster is correct. This is a security feature from what I understand, kinda like when you type in your password and only see *'s or dots. This time it just shows nothing, so that is more secure in some way or another, I do not pretend to know Linux security.

So just type in like normal and enter. If you think you made a mistake, just hit backspace enough times to clear the letters, and then retype and it all works.
