### Post by machoo02 on 2007-04-08
Hi,

Open up a terminal and enter```
wine /path/to/the/installer.exe
```Be sure to replace the "fake" path above with the actual path (i.e., if you have the installer on your desktop and your username is joe, the path would be /home/joe/Desktop/yahooinstaller.exe).

Out of curiosity, have you tried using Gaim or any of the other chat programs that are available natively?  I know for a fact that Gaim can connect to many of the chatting protocols such as ICQ, AIM, Yahoo, and Google Talk.  I've used it for the last two myself.

Hope this helps
