### Post by CarpKing on 2006-08-06
That message is nothing to worry about.  It just means that it is booting normally.  After this message, it should eventually come to a login screen.  If it doesn't, you may have a problem.  

The kernal is the part of the operating system that talks directly to the processor; if the install was successful, you already have it.  Ubuntu uses the Linux kernal (other operating systems have different kernals).
