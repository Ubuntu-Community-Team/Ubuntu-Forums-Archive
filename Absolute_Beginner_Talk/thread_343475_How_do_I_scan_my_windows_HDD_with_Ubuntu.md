### Post by mikewhatever on 2007-01-22
Scanning Windows partition from Ubuntu can be complicated, because, assuming it's NTFS, Ubuntu can't natively write to it. This means that if you do find that virus, you wouldn't be able to delete it. Besides, I am not sure AV software for Linux has the database to detect Windows viruses. I'd suggest getting help with Windows here [http://www.bleepingcomputer.com/forums/forum22.html](http://www.bleepingcomputer.com/forums/forum22.html)
