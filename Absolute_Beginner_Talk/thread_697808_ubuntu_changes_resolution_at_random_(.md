### Post by mbsullivan on 2008-02-22
Hi TheSeb,

You could try adding the following code to your xorg.conf underneath "DefaultDepth 24" in the "Screen" section:

```
SubSection "Display"
		Depth		1
		Modes		"1600x1050" "1024x768" "800x600" "640x480"
	EndSubSection
	SubSection "Display"
		Depth		4
		Modes		"1600x1050" "1024x768" "800x600" "640x480"
	EndSubSection
	SubSection "Display"
		Depth		8
		Modes		"1600x1050" "1024x768" "800x600" "640x480"
	EndSubSection
	SubSection "Display"
		Depth		15
		Modes		"1600x1050" "1024x768" "800x600" "640x480"
	EndSubSection
	SubSection "Display"
		Depth		16
		Modes		"1600x1050" "1024x768" "800x600" "640x480"
	EndSubSection
	SubSection "Display"
		Depth		24
		Modes		"1600x1050" "1024x768" "800x600" "640x480"
```

> However, when I check the resolution it tells me that it is at 1680x1050....

What entry is starred (with an asterisk by it) for the output of:

```
xrandr -q
```

Note: place the above command in a terminal window and press enter.

Also, are you using a laptop (attached to the VGA output) or is this a desktop machine? Do you have any sort of dual monitor setup?

Mike
