### Post by Tomosaur on 2006-12-20
Yeah, just plug in the cable and fire up the Live CD - if the internet is working while you're using the Live CD, then you should have no problems and you can go ahead and install properly. You may want to dual boot for a while to make sure you can solve any problems you may encounter. There are a whole load of guides on how to dual boot lying around on here, so I won't bore you with the details :P

Just a little FYI - linux without a net connection is like banging your head against a brick wall. If the net doesn't work with the Live CD, then DO NOT INSTALL until you are confident you can get it up and running. Some modems require Windows to be used properly, and although there are ways around this, it's always best to try the Live CD as fully as possible before committing to an install.

### Post by BarfBag on 2006-12-20
After you're finished installing, the first thing you should do is download and install updates.  If you have a DSL or Cable connection, this shouldn't take long.  A lot faster then the messy "reboot every 5 updates, take 30 minutes to install SP2, and continue with more restarts between updates" system that Windows has in place.

Linux doesn't come with most multimedia support, graphics drivers, Flash, etc.  That's because a lot of that is closed-source, while Linux is open-source.  I recommend you install a script called [Automatix]("http://www.getautomatix.com/") and use it to add support to those things.

Then, enjoy!
