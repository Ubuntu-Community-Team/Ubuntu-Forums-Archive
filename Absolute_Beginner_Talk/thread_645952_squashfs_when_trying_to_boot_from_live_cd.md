### Post by blueridgedog on 2007-12-22
squashfs is a compressed readonly file system, and I suspect that the boot CD is using it.  The fact that the CD works in another system is odd, as I would have guessed that the CD was bad.  Have you run the memory test option when booting off of the CD?
