### Post by aysiu on 2007-05-11
They just offer different graphical interfaces and different default programs.

All are equally good for graphics/music creation because they all have the same programs available for installation (even if they don't have the same default programs already installed).

I'd say for a novice, they're all equally "easy to use."

This will give you some sense of the difference between Ubuntu and Kubuntu:
[http://www.psychocats.net/ubuntu/kdegnome](http://www.psychocats.net/ubuntu/kdegnome)

If you still have doubts after reading that, just go with Ubuntu. You can always add or switch to Kubuntu later if you want.

Use Xubuntu if you have 128-256 MB of RAM. Or, if after playing around with Ubuntu and Kubuntu, you just want to try something different.
