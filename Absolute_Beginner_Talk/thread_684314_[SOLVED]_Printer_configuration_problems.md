### Post by Thelasko on 2008-02-01
How did you *upgrade* the hplip driver?  If it's something other than synaptic then there's your problem.

Remember: If synaptic has the software you need to get the job done USE IT!  If it doesn't and you use something else, you risk breaking your setup.  (note: there are some printers that won't work with what's in the repositories, particularly Lexmark z600 series)

If you used synaptic to install it, go back to synaptic and click on the package in question, click "packages" in the menu  and select "force version".
