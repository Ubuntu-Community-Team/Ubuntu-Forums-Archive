### Post by deadgobby on 2007-05-07
You can use Audacity. Install Audacity through synaptic all so install Lame, and liblam0 through synaptic too. Once installed. Open up Audacity and go to the edit>preferences and go to recording>channels and select 1 for mono. Now go to file open and open up a mp3 file. It should play back in pure mono.  You may want to try a different pair of speakers or head phones and see if it plays on both sides.
