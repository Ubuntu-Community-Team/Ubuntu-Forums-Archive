### Post by jqk on 2006-10-21
It also depends on the computer specification.

You can either install AutoCAD (Windows) by wine, or just simply install an alternative to AutoCAD for linux, such as QCAD (A Professional CAD system). You can get it right from the synaptic package manager or type in the terminal:

sudo apt-get install qcad
sudo apt-get install libqcad0
sudo apt-get install libqcad0-dev
