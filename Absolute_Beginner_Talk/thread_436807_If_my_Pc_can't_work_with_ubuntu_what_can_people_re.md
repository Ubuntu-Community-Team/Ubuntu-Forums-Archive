### Post by lintoolman on 2007-05-08
If Ubuntu 6.06 doesn't work, try Ubuntu 7.04.  If neither of these versions work, try to determine the cause.  You may find the same issues in other distros.  Looking at your specs I don't see why ubuntu won't run (assuming you WNIC and TV tuner work).  You can probably figure that out with a livecd, google search etc.

I also like CENTOS. I used to run openSUSE 10 on my server, but I had stability issues.  These issues were probably my fault.  :) 

As far as 64bit distros, I run 64bit with no problems on my Dell E520 and on my home grown 64bit server (both 7.04).  However, be aware there are some limitations, such as no easy way to install WINE.  If you want to try 64bit, go for it, I 'd recommend some research on the apps you would like to run before hand.  You may find 32bit is the way to go for now.

EDIT:  After reviewing the x86_64 forum it appears WINE can be installed.  My bad.
