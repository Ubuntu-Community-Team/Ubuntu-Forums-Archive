### Post by drascus on 2007-01-24
In ubuntu you really don't need to go about this procedure. This is because the administrative activities that lead to having things installed or settings change require a password. However you can create even more limited accounts in Ubuntu. This can be done through the Administrators drop down menu then go into users/groups there you can define a limited user account if you would like.
