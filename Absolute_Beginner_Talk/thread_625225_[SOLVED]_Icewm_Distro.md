### Post by jordanmthomas on 2007-11-27
You could use a distro that has no default window manager like Arch or Gentoo or something.

Then, just install it.

Also, distrowatch has a search function where you can choose to search by wm:
[http://distrowatch.com/search.php?category=All&origin=All&basedon=All&desktop=IceWM&architecture=All&status=Active](http://distrowatch.com/search.php?category=All&origin=All&basedon=All&desktop=IceWM&architecture=All&status=Active)
The list doesn't start 'til a ways down the page, but it's there.  I believe it's a list of distros that either have IceWM as default or have no default.

**edit**
This one seems to be active (based on slackware)
[http://www.pcbypaul.com/absolute/](http://www.pcbypaul.com/absolute/)
