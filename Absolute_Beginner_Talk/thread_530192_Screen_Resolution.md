### Post by Wim Sturkenboom on 2007-08-20
Install the drivers for your videocard. If you have an nvidia card, you can start with changing *vesa* to ***nv***. If you have an  Intel graphics card, you can start by changing to ***i810*** and you need to install ***915 resolution*** (not sure where to get it, might be in repositories, just search the forum). For ATI I don't know.
Please note that nv and i810 are generic drivers.

At the top of [this page](http://ubuntuforums.org/forumdisplay.php?f=138) are some stickies that give instructions how to install manufarcturer drivers.
