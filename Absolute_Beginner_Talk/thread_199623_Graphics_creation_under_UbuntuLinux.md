### Post by HereInOz on 2006-06-19
There is a graphics editing program with Ubuntu and that is GIMP.  I have to say that it is not my favourite graphics program, but that could be becuase I haven't really taken enough time to get to know it better.  It is reported, by those who use it regularly, to be very powerful with a very good rendering engine.

There is also the possibility of running Photoshop on Ubuntu using an excellent piece of software - "Crossover Office" from Codeweavers.  This allows you to install a range of "for Windows" apps on the Linux platform, which provides you with all the good things about Linux, but then allows you to run those apps that are made only for Windows, when there is no alternative.

Hope this helps.

In my view GIMP is all there is for a Paintshop Pro / Photoshop type program. Besides an unfortunate name, it also suffers from 3 separate windows when Paintshop Pro / Photoshop use 1. This is a major drawback as you invariably end up losing one or more windows and end up messing around resizing them, which becomes frustrating fast.

You can't run other applications as they get in the way. So a black desktop for the 3 GIMP windows is what I think works best, but not good enough (free or not).

The program itself is decent enough but the developers are off on a tangent with their 3 windowed idea. They like it and don't give the end user any option to have 1 window. It is a case of take it or leave it.

There is a mod called GimpSHOP which is little more than rearranging the menu order and names to be like Photoshop.

I used Paintshop Pro a lot and I wish we had more and better graphics programs for Linux.
