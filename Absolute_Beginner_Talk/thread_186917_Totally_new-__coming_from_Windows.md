### Post by mostwanted on 2006-06-02
Yeah, it'll allow you to just boot from the CD and run Ubuntu that way.

On the desktop there's an icon which you just double-click to open the installer (basically a next-next-next procedure). You can play with the Live CD while it's installing and it shouldn't take more than 20 minutes.
