### Post by mcduck on 2008-01-06
There is currently no system-wide EQ available, so you need to use a player that has one of it's own.

The best player for those used to Winamp is probably Audacious. It's a newer version of XMMS. Beep Media Player is another option, it's newer than XMMS but older than Audacious. Both have the same UI as XMMS/Winamp 2, and can use same skins.

To change the default program for your MP3 files just right-click one file, select Properties and then on the OPen With-tab set the program you want to use for those files.
