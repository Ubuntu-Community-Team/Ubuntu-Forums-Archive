### Post by airtonix on 2007-09-02
1) they dont go anywhere, unless you put them in a common place... 
 ie : /home/yourUserName/me/media/wallpaper

If you right clicked an image... and the chose " set as desktop background"

then the image being used for the wallpaper is the file you rightclicked.

delete the image, and watch your wallpaper dissapear.

2) also...."places" is a group of places....funny that.

3) the windows and linux thing your asking about....
  if the switch between ubuntu and windows is instant then what you saw was a virtual machine setup inside ubuntu running windows. 
 if there was a reboot in there then its just dual booting....

but if you cant figure out where to put images, be prepared for some complicated command line work.

packages you want to look at for virtual machines are virtualbox & vmware
