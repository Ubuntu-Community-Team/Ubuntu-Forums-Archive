### Post by pay on 2007-05-13
```
sudo nano /etc/X11/xorg.conf
```add```
Section "Extensions"
        Option  "Composite" "0" 
EndSection
```at the end```
sudo aptitude update
sudo aptitude install linux-restricted-modules-$(uname -r)
sudo aptitude install xorg-driver-fglrx
sudo depmod -a
```If that doesn't work, which you said it didn't before, we could just use the binary from ati. ```
wget https://a248.e.akamai.net/f/674/9206/0/www2.ati.com/drivers/linux/ati-driver-installer-8.36.5-x86.x86_64.run
sudo aptitude update
sudo aptitude install module-assistant build-essential fakeroot dh-make debhelper debconf libstdc++5 linux-headers-$(uname -r)
sudo bash ati-driver-installer-8.36.5-x86.x86_64.run --buildpkg Ubuntu/feisty
sudo nano /etc/default/linux-restricted-modules-common
```and add```
DISABLED_MODULES="fglrx"
``````
sudo dpkg -i xorg-driver-fglrx_8.36.5-1*.deb
sudo dpkg -i fglrx-kernel-source_8.36.5-1*.deb
sudo dpkg -i fglrx-amdcccle_8.36.5-1*.deb
sudo rm /usr/src/fglrx-kernel*.deb
sudo apt-get -f install
sudo module-assistant prepare
sudo module-assistant update
sudo module-assistant build fglrx
sudo module-assistant install fglrx
sudo depmod -a
sudo aticonfig --initial
```then startx
