### Post by aysiu on 2006-03-29
[QUOTE=etherealbeats]Synaptic package manager says "versatile and easy to use audio player for KDE", and i can't find one for GNOME. It is in the KDE section as well. Thanks for the quick reply.[/QUOTE] It's *for* KDE, but KDE applications work in Gnome, and Gnome applications work in KDE, too.

Some of the functionality may be weird. For example, if you drag-and-drop a song from AmaroK to your Gnome desktop, it'll *move* the song instead of *copying* the song. In KDE, it'll copy the song.

One way around this is to use Konqueror if you want to drag and drop songs to a folder in Gnome.

Other than that, it should work fine.
