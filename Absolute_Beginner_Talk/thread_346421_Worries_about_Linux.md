### Post by Shatrat on 2007-01-25
The only thing that could possibly harm the windows portion of the PC is the part of the install where you do the Partitioning.
The Ubuntu live CD installer can resize that partition and it isn't that hard to use or understand.
The key thing is to defrag before you install ubuntu and the installer can easily shrink the windows partition down and create a / and swap partitioni for linux in the leftover unallocated space.
