### Post by geokok1981 on 2007-01-23
Having many window managers does not affect your system at all. I believe (and please do correct me if wrong) that you load the manager you wish at the login screen. So if you but gnome you ll get gnome, ig you boot KDE you ll get KDE. The rest of them are not active at the same time.

Amarok is a KDE app. You can still run it through gnome using synaptic but it will install the KDE libraries as well. The drawback from running a KDE app on gnome and vice versa, is that when you launch the app the libraries of KDE in amarok's case, have to be loaded too, so it might be a bit slow and consume some resources.
