### Post by jargoman on 2007-05-09
you can't install just by opening it up in windows. You have to reboot your computer with an install cd in the drive. 

Your best option is to get an install cd somehow. I suggest you bring the flash drive with ubuntu on it to a computer with a cd burner and a burning program like final burner (it's free and non complicated). make sure you don't drag and drop. you need a program that "burn's iso's" and use that option.

It coud be possible to remove a cd rom  from one computer and add it to another temporarily just to install ubuntu then put it back. just make sure you remember how it was installed on the first computer.

you could also go with a network install but some computers don't support that and I've never done it myself.

running defrag doesn't free up any space on your computer it only rearranges the files in a better order. this can be usefull if you want to resize your windows partition.
