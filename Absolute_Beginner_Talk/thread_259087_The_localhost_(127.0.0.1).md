### Post by kerry_s on 2006-09-16
huh? Ubuntu's default is no open ports. 127.0.0.1 is your computer, so for instance you want to block a site you would add a line in your /etc/hosts file like> 127.0.0.1 google.com, so that any query for google.com will be routed to your computer thus blocking it.
