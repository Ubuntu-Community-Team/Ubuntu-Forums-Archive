### Post by dstew on 2008-01-15
It looks like the gutenprint driver will work. See [http://openprinting.org/show_printer.cgi?recnum=Epson-Stylus_CX4200](http://openprinting.org/show_printer.cgi?recnum=Epson-Stylus_CX4200). You can install the foomatic-db-gutenprint package in Synaptic, or maybe better the CUPS driver cupsys-driver-gutenprint.
