### Post by Tomosaur on 2007-04-21
Blender is the current most popular 3d modelling app at the moment. It's free, and available in the repos. It will take you a while to get used to however - as the interface is mainly keyboard-driven (which keeps the mouse free to do all the important stuff like modelling!) and can take a while to get used to. A film was released recently which was made entirely using open-source stuff such as Blender. You can check out it's website and read about how it was made, because it will undoubtedly help you :)

[http://www.elephantsdream.org/](http://www.elephantsdream.org/)

### Post by vnt87 on 2007-04-22
A couple of sites for Blender:

[http://www.blender.org/tutorials-help/tutorials/getting-started/](http://www.blender.org/tutorials-help/tutorials/getting-started/)
[http://en.wikibooks.org/wiki/Blender_3D:_Noob_to_Pro](http://en.wikibooks.org/wiki/Blender_3D:_Noob_to_Pro)
[http://www.blendernation.com/tutorials/](http://www.blendernation.com/tutorials/)

Also, major 3D package in Windows like Maya, Houdini etc. are available as native for Linux, and they do offer personal learning version which offer all functionality as in their full version except some limitation like watermarked and low res renders, non-commercial use only etc.
