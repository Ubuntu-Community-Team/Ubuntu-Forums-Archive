### Post by KingBahamut on 2006-06-04
CDparanoia is a command line interface  driven application. It isnt something that you would nessecarily run from an application icon (like Gimp or XMMS or other such application) 

For more information on CDparanoia and what it does go here -- [http://www.xiph.org/paranoia/](http://www.xiph.org/paranoia/)

or open up gnome terminal and type man cdparanoia 

What you need to be congisant of , is which apps have GUI interfaces to them and which ones do not. Once you understand that, then learning how to use these applications will become much easier. 

If you have other questions, cruise on over to the UDSF page ([http://doc.gwos.org](http://doc.gwos.org)) or the official Ubuntu wiki ([http://wiki.ubuntu.com](http://wiki.ubuntu.com))
