### Post by Joshua Hesketh on 2007-06-18
Hello and welcome to the Linux community.

Linux is very different from Microsoft Windows. One key difference is that there is no start menu. Instead if you look in the top left of your screen you will notice 3 menus. "Applications", "Places" and "System".

[B]Applications:
[/B]In here you will find a list of all your installed applications and a nice and easy way to add more from a great list of free and open source software.

[B]Places:
[/B]This menu contains a quick way to get to your home folder, hard drive, cd drive, thumb drive, ipod or music machine and much more. Any folder bookmarks and recent documents also appear here. This menu is primarily for accessing folders and files.

**System:**
This menu contains all configuration applications. There are two submenus: Preferences and Administration. Preferences contains settings for your user such as background and themes. Administration lets you change entire system settings and perform administrator tasks on your computer.

I hope this quick description helps you. For more information visit: [https://help.ubuntu.com/](https://help.ubuntu.com/)

Regards,
Josh
