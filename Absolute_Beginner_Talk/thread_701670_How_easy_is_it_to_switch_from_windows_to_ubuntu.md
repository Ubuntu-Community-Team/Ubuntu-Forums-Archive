### Post by steveneddy on 2008-02-19
Easy as:

1. Install CD in drive.

2. Reboot PC.

3. Make sure all of the hardware works while using the Live CD
   (Cd player, USB, firewire, Bluetooth, Ethernet, Card readers, etc)

### Post by neurostu on 2008-02-19
I transitioned from linux about 6 months ago. I would highly recommend it if you consider yourself as a person who is moderately computer literate.

One thing I would recommend is to find a linux users group in your area. They will be more then willing to help you out will be an invaluable resource when ever you have questions.

But man, yes, install ubuntu go for it!!! First you might want to pick up a copy of GParted and split your HDD. This will allow you to dual boot windows and XP. This way if you ever have a major problem you can always revert to XP if you have to.

### Post by bodhi.zazen on 2008-02-19
The hardest part is the installation. If I install Ubuntu for people they can use it out of the box for most things (web browsing, word processing, etc).

If you need to sys admin it will take longer, 3-6 months to start to feel comfortable with any new OS.

If you are expecting to run windows applications, like MS Office or games, you may struggle a little. Best to transition to linux native apps or dual boot for a while.

Virtualization can also help.
