### Post by norman_ on 2007-01-08
If your objective is to actually learn, I suggest you start with something else than ubuntu. I use ubuntu cause I don't want all the configuration trouble and maintenance. It is a very good distro for what it does. Maybe too good for learning.  

Try [www.archlinux.org](www.archlinux.org) read then installation manual and dive in. It doesn't do all the configuration automatically so you'll get to touch some of the files (xorg.conf, rc.conf, inittab...). It is really not that hard and it looks scarier than it is. Also arch is good cause, even if it does not do the configuration automatically, it has a great package manager so you won't have compile every single piece of software.

my 2 cents..
