### Post by Dngrsone on 2008-01-05
In this case, you are more limited by the amount of RAM you have than by the swap supporting it.

For that amount of memory, the allotted swap should be sufficient.

Keep in mind that Linux does not use memory and swap like Win does... for the most part, the available hard RAM will be used to its full extent before swap is, so as long as you are not trying to run a bunch of memory-intensive apps, you should be fine.
