### Post by Lasse_ on 2006-05-30
I would go with ZyXEL.

I bought a ZyXEL Prestige 660HW a few months ago and I'm very pleased with it. Althought despite the promises about the Prestige to configure itself automatically I had to manually input my ISP's VNC ( or VNP or whatever that acronym is ) value. After that it has worked like a charm. It comes with a use-to-use web configuration utility, there are four ethernet ports and the HW model has a WLAN capability. Prestiges own firewall is highly configurable and ZyXEL has made it rather easy to poke holes for SSH, HTTP, etc. In Finland the Prestige isn't the cheapest one ( about 90 euros ) on the marked but it's one of the best. It also supports ADSL2+ standard and there is some high speed WLAN solution if you got ZyXEL's WLAN cards.

 - Lasse

### Post by steve.horsley on 2006-05-30
I would say go for any ADSL router that gives you 4-port Ethernet. Wireless too might prove a useful extra. The one Lasse describes sounds nice. Mine is badged Mentor, nad doesn't have wireless - I had to buy wireless separately later.

The 4-ports comes in handy surprisingly often, and now it's there, I use wireless nearly every day.

Make sure it can be set up from a web browser - not dependent on some rotten windows-only software. Such a device may be more expensive than a win-modem thingy, but will prove well worth it in reduced hassle.
