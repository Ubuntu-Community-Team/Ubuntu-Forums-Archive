I just downloaded Thunderbird and set up my gmail account.  As soon as I finished my set-up Thunderbird started downloading all my old email. (more than 1GB). Is there any way that I could only allow Thunderbird to download only my unread mail from gmial??

### Post by lawentzel on 2008-03-27
Yes, I believe that is a setting in GMail.  In GMail, go to the Settings option at the top right side of the page.  Next select Forwarding and POP/IMAP.  There select Enable POP for mail that arrives from now on.  That will resolve your issue!
