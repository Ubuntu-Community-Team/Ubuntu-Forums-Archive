### Post by p_quarles on 2007-07-07
I can think of a few things to try. First, check the integrity of your live CD. If it loads as far as the first menu (where you get options like "Start/Install Ubuntu" and "Boot from first hard disk"), you can choose the option to check the CD. Second, you might try running the Ubuntu (Gnome) live CD, as KDE is apparently more buggy in some ways. You'll still be able to install the KDE desktop after you've got things configured. Finally, try the alternate installation CD.

It is very possibly your graphics card, though. ATI is the toughest of the three major vendors to get working properly, but it can be done. Search for tutorials, or ask here.
