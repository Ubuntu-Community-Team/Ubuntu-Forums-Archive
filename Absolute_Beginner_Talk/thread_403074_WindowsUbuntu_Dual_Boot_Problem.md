### Post by benerivo on 2007-04-06
How did you initially install XP? Ideally the partitions of the drive should be made during the installation, rather than installing XP on to the whole drive, then using a partition manager to divide the drive up afterwards. It sounds as if XP may be broken. If you want to reinstall I would do XP first and make a partition for it during the installation process.
