### Post by msak007 on 2006-08-29
First of all how much RAM does your machine have and how fast / slow is the CD-ROM? I recently had a similar issue trying to install Xubuntu on an old 500 MHz PC with only 128 RAM (I think...might be lower). A lot of factors (including RAM, CD-ROM speed, how good the burn was, and graphics) play into how well (or not so well) the OS loads and runs from the CD. Try the fail-safe graphics mode, and if that fails then there is an "Alternate CD" that you can download which has a text based installer. While not as pretty (or easy to some people), it gets the job done on machines that can't handle the Live CD installer.
