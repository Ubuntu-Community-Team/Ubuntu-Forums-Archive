### Post by LFC on 2005-12-17
Got problems installing ubuntu 5.10 on my old IBM Thinkpad 600x. I modified the bios so it starts from the cd-rom but it seems to ignore the iso file on the cd-rom and instead starts booting from the harddisk. Any suggestions ?

---

### Post by kaamos on 2005-12-17
You must burn the iso to cd as an image. Search your burning software for an "Burn disc image" or similiar option.

Also, use a low burning speed.

More info here: [https://wiki.ubuntu.com/BurningIsoHowto](https://wiki.ubuntu.com/BurningIsoHowto)
