### Post by meborc on 2007-09-16
if you like xfce, then stick to xubuntu... all the apps you are used to using in gnome, will run under xfce, but some of them may not be preinstaled, or some are switched . (mousepad instead of gedit, xfce4-terminal instead of gnome-terminal etc...)

just know, that when installing some gnome apps under xfce, a lot of gnome libraries are needed and therefore it might get a little bloated...

you can always install ubuntu and then install "xubuntu-desktop", now you have all the apps you have in standard ubuntu + the xubuntu ones... you just need to choose xfce in sessions during login
