### Post by bodhi.zazen on 2006-10-26
It depends.

IMO Windows runs fast after re-booting. Over time there is a performance degradation.

Ubuntu does not seem to degrade as much.

Ubuntu does run a number of daemons and large programs that degrade performance. You can try to speed ubuntu by removing what you do not need or starting with a server install and building up only what you use.
