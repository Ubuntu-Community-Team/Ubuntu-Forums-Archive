### Post by Midway on 2006-10-22
I can certainly sympathize with the poster.  I had been using dial-up for ten years, from 1200 Baud up to 56K.  I live too far out in the country to get broadband.  I used to have to take my tower to work to use it's T3 lines if I wanted to reinstall, download large programs, or doing dist-upgrades in Linux. Linux DOES really need broadband because of it's nature.  Updating Windows was not a problem for me since I used Autopatcher and I downloaded large files at work, burned them to a disk, and took it home to put on my computer.  Because of dependencies, you cannot do this with Linux.

I was so infuriated that I deleted Windows off my hard drive (I had been dual-booting with Ubuntu but screwed it up some weeks ago and was only booting into Windows) and installed Dapper so now I am pure Linux :)
