### Post by wormser on 2007-09-01
GnomeBaker is not bad for burning CD/DVD

Places> Home Folder and browse to where the tarball is.  You should just be able to double click it.

The best way to get software is with the repositories.  System> Administration> Software Sources.  Put checks in the top four repositories.  That will allow you to search and install from those locations.  System> Administration> Synaptic Package Manager.  Search for what you are looking for and install.

Welcome to Ubuntu!
