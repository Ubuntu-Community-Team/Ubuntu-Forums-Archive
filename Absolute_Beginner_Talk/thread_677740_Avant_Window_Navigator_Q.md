### Post by CJ56 on 2008-01-25
If you go to the System menu, then Preferences, then Sessions -

In the Sessions window, click the Startup Programs tab and then the Add button

That opens a New Startup Program box

Fill in the name of the application - i.e. AWN

Then the Command

(How do you get the Command if you don't know it? A cheesy but effective way is to right-click on the application in the Applications menu, put it on the desktop, right-click on the new desktop icon, go to Properties and find the command there. Then cut and paste it back into the Startup Program box. You can delete the desktop icon afterwards)

Click OK

Close everything...

It worked for me, anyway... :cool:
