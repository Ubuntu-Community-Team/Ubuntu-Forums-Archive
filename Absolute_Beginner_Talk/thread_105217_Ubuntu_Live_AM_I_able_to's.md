### Post by aysiu on 2005-12-17
Any programs you "install" while using the live CD will not be there any more once you log out or reboot. The installation will occur in your computer's RAM (temporary memory).

To have the installation of programs stick, you need to install Ubuntu. The live CD's just a "test drive."
