Hello,

I wish everybody a good day. This is my first post and instead of asking a question I would like to contribute with something extremely simple: creating  a shortcut to a directory through a launcher.

Note: I have only tried this in Ubuntu 6.06 with GNOME.

1) Right Click on the desktop
2) Select "Create Launcher"
3) On the command field, write "nautilus <directory>", where <directory> is the path you want to open.

A new window should open with the specified directory.
