### Post by mcduck on 2006-08-10
> **jincast90 said:**
> I would only really wanna install it, if it boots very fast. 
Does it boot very fast?
It depends on what you qulify as 'very fast'. It's not instant, but even the cd-version boots fast on my old K6-@500MHz, and I've seen some DVB-boxes that take longer to boot than Geexbox on my laptop.

Try the live-cd, it's not that big download (6,8MB) and you'll be able to see yourself :)

Geexbox is also available as ISO-generator, that allows you to customize it and add music and movies to the disk image before you burn it. So you can make MP3- or movie CD's that include player & OS and use those disks on about every computer as long as you can make it boot from a CD. I'm using one to turn my workmate's computer into music player when she's not around.. ;)
