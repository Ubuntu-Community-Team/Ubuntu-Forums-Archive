### Post by bobpur on 2006-06-05
How much RAM does it have? Hoary (v5.04) needs about 32 mb RAM to function.The newer versions of Ubuntu need more. 
I installed this OS in an older computer (IBM Aptiva Series E 550) that had 96 mb RAM and it worked fine.
You might want to increase the size of that hard drive to at least a 20 gb. I'm not saying it's necessary, but it sure couldn't hurt.
