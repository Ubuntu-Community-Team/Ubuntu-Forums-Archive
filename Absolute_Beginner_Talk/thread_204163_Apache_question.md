### Post by hw-tph on 2006-06-26
Apache is a http server. It has nothing to do with Windows domains. However, [Samba](http://www.samba.org) can act as a domain controller. The [wiki](http://wiki.ubuntu.com) has a [very basic piece on Samba](https://help.ubuntu.com/community/SettingUpSamba) and there are tons of tutorials and howtos on the web for setting up Samba to do different things.
