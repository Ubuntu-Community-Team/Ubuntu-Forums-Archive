### Post by pandemic on 2006-07-20
I issued the command uname -r and got back 2.6.15-26-386... ok I looked into the versioning scheme of the linux kernel. In the wikipedia article that I read it mentioned nothing about the "-26-386", so could someone please explain what this last part means?

---

### Post by yawnster on 2006-07-20
Well, the -26 is a subversion, a small snapshot of the code that will fix bugs and security issues, but will not add any features, think of it like the Windows Update feature for Windows platforms, it will be purely comprised of small fixes..

As for 386, that denotes the platform you are using, a 386 processor type..

Hope that answers it, Alex
