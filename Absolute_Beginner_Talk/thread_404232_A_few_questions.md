### Post by indytim on 2007-04-08
At the download site, the iso download is intended to:

1.  download the file
2.  burn the file as an iso to a cd.
3.  be able to boot your computer from a cd
4.  insert the cd into the cd rom and boot
5.  checkout the ubuntu operating system using the LiveCD.
6.  if you like it, come back here for additional information on preparing and installing.

Good Luck,
