Note though that it won't keep the user permissions from Linux, so I'd personally only do it with a /home partition if you're the only user of your computer or you don't mind if other users see it.  Setting up the / (root) partition on WinXP isn't recommended, as anyone who compromises your system through XP would be able to manipulate the Linux filessystem.
