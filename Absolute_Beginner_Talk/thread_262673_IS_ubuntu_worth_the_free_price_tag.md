"manual" in *nix circles often means the manual pages included with the distribution.  These are easily accessible from the command line.  for instance, if I wanted to find out about the program 'grep' (which allows you to search within a text file or a stream of text) I'd type

```

man grep

```

Of course, the man pages are pretty dry, technical reading, and not all that helpful if you're totally new to UNIX-like operating systems.
