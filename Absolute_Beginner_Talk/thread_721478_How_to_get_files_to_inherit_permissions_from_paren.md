### Post by dondad on 2008-03-11
> **techno-wiz said:**
> I've copied some files to a directory using nautilus and I want the files to have the same permissions as the directory I copied them to. How do I do this?


Assuming that it is a directory that you own (ie, not a system directory) try right clicking on the directory in Nautilus, selecting properties and checking the box to apply to the files contained inside.
