### Post by Kobalt on 2007-05-27
Kate isn't in version 4 yet, this is still in developpment. 
Ubuntu uses the repositories system to manage and update the applications. Once you installed a package, you automatically get the newest version available when it reaches the repos. It is safer and easier to leave it this way. However, you can manually install a newer version of an application if you'd like, through compilation or a package, but you won't be notified and able to have the automatic update for that program.
