### Post by qpieus on 2007-06-23
yes, it looks like you have a bad disc, therefore your installation will probably not be successful. A re-burn is definitely needed. Using a slower burn speed may be helpful if you are using discs of questionable quality. Give it another try, I wouldn't try the install with a disc that has errors.
