### Post by FuturePilot on 2007-11-05
Mplayer has a great Firefox plugin. And if you have all of the codecs installed it can handle just about anything you throw at it.
```
sudo apt-get install mozilla-mplayer
```
Then you also might want to get rid of the Totem plugin
```
sudo apt-get remove totem-mozilla
```
