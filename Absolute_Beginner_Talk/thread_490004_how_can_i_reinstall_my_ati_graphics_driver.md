### Post by Raineer on 2007-07-02
Not sure which one you were using, but

```
sudo apt-get install linux-restricted-modules-2.6.20-15-generic --reinstall
```

could help, this has a version of ATI driver.  If you can't get to a GUI at all you might type ```
uname -r
``` to ensure this matches your kernel.

If not, run ```
dpkg-reconfigure xorg-xserver
``` and choose either an ATI selection or "vesa" as the fall-back to at least get you running with graphics screens again
