### Post by Partyboi2 on 2007-12-07
Easiest way to install wvdial is to open a terminal (Applications>Accessories>Terminal)
then type or paste
```
sudo apt-get install wvdial
```That should install it and also take care of the libv6 problem.
If you run into any errors just post them back here and someone should be able to help
