### Post by Inxsible on 2007-07-02
You could use the 64bit OS. Do keep in mind the pros and cons of it however.
 
Not all software is supported in the 64bit version. Flash being the prime example(However, I did see a post which had listed the solution). 
The installation of 64 bit has become a breeze and is not as painful as it once used to be.
 
If you do not do very intensive graphics and use other processor intensive work, you will hardly notice any difference between the operation of the two OS'es.
 
You can however, have more than 4GB of RAM. 32bit OS only has enough address space to reference 4GB of memory(and swap is also counted)
 
So for eg if you have 1GB of swap, then you can use only 3GB of RAM. In a 64bit OS however, you have a lot bigger address space and so you can reference more than 4GB of memory.
 
There may be other advantages/disadvantages.
