### Post by Sef on 2007-01-02
> But I want to get a music/video player so I can listen to my music...most of which are .wmv's or .mp3's.....but I don't know anything....so what should I get that would work nicely


Rhythmbox will play your mp3s.  It is installed by default.  However, to get mp3s to play, you need to install the codecs.  Read [restricted formats]("https://help.ubuntu.com/community/RestrictedFormats") to install them.

> Also, when I download programs that are .exe's......my computer says it cannot view them? Help there too?


Read [Psychocat's installing software]("http://www.psychocats.net/ubuntu/installingsoftware") to learn how to install software on Ubuntu.  .exe is for Windows.  .deb is what is used by Ubuntu.
