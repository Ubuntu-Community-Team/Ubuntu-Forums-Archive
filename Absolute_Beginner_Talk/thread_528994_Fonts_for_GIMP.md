### Post by coffeecat on 2007-08-18
It's not the Gimp that's lacking the fonts, it's your system. Like most Linux distros, Ubuntu only comes with a selection of open source fonts - this for legal reasons. If you install more fonts to your system, they'll be available in Gimp. Search the forum for how to install more fonts, but two suggestions:

1 To get the Microsoft web-core fonts, look in Synaptic for a package called msttcorefonts. Or at least I think it's called that. I'm not in Ubuntu atm and I'm going from memory.

2 If you have some .ttf fonts, there are several ways to install them but the easiest is to create a folder called .fonts in your home folder (don't forget the dot at the beginning of the folder name - this is essential and makes it a hidden folder). Then simply drag the .ttf fonts into the .fonts folder. You may have to restart the Xserver before they are available.
