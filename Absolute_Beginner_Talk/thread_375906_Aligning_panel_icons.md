### Post by pveith on 2007-03-04
Yeah, sure. Just right click the icon you want to move around and select the move option. The icon has to be "loose" to do so (uncheck last option if it the move option is greyed out).

[Sorry if the name of the option in not correct, i have a german desktop, so the options here are in german.]
