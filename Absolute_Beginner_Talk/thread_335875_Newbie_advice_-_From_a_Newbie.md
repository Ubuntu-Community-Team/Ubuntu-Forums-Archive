### Post by mistypotato on 2007-01-10
This is about keeping your sanity (and desire to use Linux) in check....

I'm not new to computers but I'm new to Linux.

Several times in my illustrious career I've dabbled with Linux, always fascinated by it but never staying with it for too long.  One reason was that it never really seemed to have the "guts" I needed compared to Windows.  Another reason was because it was more complicated and I seemed to botch it up on a regular basis so that just about the time I got it configured fairly decently, BAM!....something would happen and I could no longer boot into it. 

Well, recently I obtained ubuntu and I must say things have changed.

There is an impressive array of tools and software to work with and it now feels so much like Windows (hmmmm, not sure thats a good thing) that it seems ready for mainstream.
In many ways, I find I prefer it to Windows....something I had trouble saying years past.

But the one thing that has not changed, at least for me, as a relative "newbie" to Linux is the propensity to "Botch" the installation so that it doesn't want to run.
Now I'm sure as I get "better" with it this will not happen as often, but until then, I knew that if I kept doing this, I would lose faith (and interest) again.

So first thing I did as a Newbie was to look into backing up my system.

What I REALLY wanted was a way to create a complete, easy to use restoration process to put my system back like it was before the boo boo.  I wanted a way to backup my system to CD's or DVD's easily so that all I had to do was insert the disks into the proper drive, turn on the computer and sit back while my botched Linux system was restored.

Forget Polly......Along Came MONDO.

There is a utility called MondoArchive that will save you from exasperation when things go sour.   If you're gonna spend time learning something, I suggest it's how to use Mondo.  Oh sure, you'll definitely want to learn the in's and out's of Linux and how to use sudo etc...but you'll get there quicker if you don't have to keep starting from scratch.

Now that I know how to use it, I back up my system to CD's (or DVD) once a month or whenever I have made a significant change to the system.

Now I don't fear disaster anymore.   I just laugh and grab my backups.

Check it out BEFORE you go nuts.   Trust me ;)
