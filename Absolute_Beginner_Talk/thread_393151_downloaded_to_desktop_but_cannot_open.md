### Post by wpshooter on 2007-03-25
If what you dowloaded is the ISO image of some flavor or Ubuntu then you need to **BURN** that image (not copy) to a CD-RW or CD-R and then upon rebooting your computer it will boot the the Ubuntu image, assuming that you have your CD drive set as the first boot device in BIOS.

If you can get the computer to boot to the Ubuntu image, make sure that the very first thing you do is:  check the integrity of the Ubuntu CD by running the verification function that is found on the initial Ubuntu boot screen menu.

Write back if you have further questions.

Good luck.
