### Post by bluetracer on 2007-06-20
1.  Try burning again, perhaps at a slower speed.

2.  Sounds plausible.  Once you've got a known good CD you'll find out, I guess.

3.  The Ubuntu installer is capable of creating a dual-boot setup.  See [this page]("http://www.psychocats.net/ubuntu/installing#partitioning") for a walkthrough.

As a general recommendation, I would suggest you run the Live CD and get everything working in that - especially your Internet connection - before you take the plunge and install.

### Post by Happy_Man on 2007-06-20
Make sure you burn the .iso to disk *really really slowly*, as Ubuntu does not play nicely with bad disks. 

You can always set up a dual-boot system, meaning you can choose whether to use XP or Ubuntu upon startup. That's automagically configured for you, if you choose that route when you install off the CD.
