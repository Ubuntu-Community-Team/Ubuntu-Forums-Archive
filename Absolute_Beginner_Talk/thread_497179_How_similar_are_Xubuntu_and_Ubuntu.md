### Post by Nythain on 2007-07-09
the base of it is the same... but the gui (desktop environment) is going to be different... so if he's asking questions like "Help, how do i change my wallpaper" or "I need to get programs to autostart when i log in" you might struggle a little or find yourself googling for answers... but if he's all like "I need to install something" or "how do i change permissions" or "how do i get my second drive to mount" then you will be fine, as most of those answers are the same in every ubuntu distro

how old is his machine that it cant run ubuntu well... im on a p4 2.0ghz 768meg DDR nvidia fx5200 and gnome/kde both ran faster than xp, and kde preformed almost flawlessly with Beryl...
