### Post by d3v1ant_0n3 on 2006-11-03
DethKnight, once you have downloaded and burned the Ubuntu .iso to a cd, you can then start it up as a 'Live CD'.

This means you can try out ubuntu without having to install any software to your computer- It runs straight from the cd. Amazed me when I first saw it!
The main limitation to this is you can't save anything from this session, it's also somewhat slower than the actual installed version of Ubuntu, so don't let the speed put you off.

If you like it (I did!) and want to install, first I'd recommend going back into windows, and backing up any important files you have as a precaution, and then defragmenting your hard drive. You should also make sure you have at least 5Gb of free hard drive space too (more is better).

Then pop in the live cd again, and chose the 'install' option on Ubuntu's Live cd desktop- It's a really simple process, and will handle the partitioning of your hard drive so you don't lose windows- it will recognise the Windows partition, and won't delete anything from it.

When the install is complete, you'll have a new menu at start up- you can now choose from Windows and Ubuntu!

Have fun!

If you do a lot of gaming on your pc, I'd recommend keeping Windows installed- it's so much more straight forward for games than trying to get things running through WINE (a program that lets you run Windows apps in Linux). I use our Windows box for games, but everything else thaht I used to do in windows, I now do in Linux:D  Except downloading cracked software. I don't need to do that anymore. It's all free!
