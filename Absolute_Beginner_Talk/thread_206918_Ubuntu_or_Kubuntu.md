### Post by IYY on 2006-06-30
Kubuntu is not really a seperate operating system, just Ubuntu with KDE instead of Gnome. If you want to keep your existing installation and just try KDE,

```
sudo apt-get install kubuntu desktop
```

And I'd suggest trying it if you have time. I prefer Gnome, but you should always see what the options are and choose on your own.
