### Post by ugm6hr on 2008-01-05
I am assuming you have wifi in that laptop.  If so, it will probably be a lot easier to use the phone as a "router" and access it via wifi.  I have never done this, but it should work in theory.  Connecting via USB to Windows Mobile devices is a bit hit-and-miss in Ubuntu.

This explains how to set up internet sharing via wifi on WM5/6: [http://msmobiles.com/news.php/6999.html](http://msmobiles.com/news.php/6999.html)

Provided your wifi works in your laptop, you should be able to access the phone from Netowrk Manager.
