### Post by p_quarles on 2008-01-30
The two operating systems are completely isolated from one another normally. If you were to download a virus-ridden file in Ubuntu, it won't affect your Windows installation unless you go out of your way to actually open or run the infected file. 

So, my advice would be to run some Windows anti-virus tools and make sure that your installation is clean.
