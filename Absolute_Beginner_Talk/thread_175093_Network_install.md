### Post by macdo on 2006-05-12
No, not officially, at any rate. 
Network Installl discs are kind of silly with Ubuntu, where a complete system weighs in at less than 700 meg (1 CD). This is because Ubuntu only installs one of each thing - one pdf reader, one office suite, one text editor, one browser,  and so on. If you want anything else, you're welcome to it, but you have to download it after install.
