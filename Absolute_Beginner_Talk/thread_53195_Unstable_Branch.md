### Post by Xian on 2005-07-30
There is always a balancing act between being advanced and stable. The two really don't co-mingle very well, so it requires much more work to provide a fresh set of packages that in general will run with few hangups. Ubuntu has done a great job of this to date, and with more experience and talent it has the capacity to improve even further.
