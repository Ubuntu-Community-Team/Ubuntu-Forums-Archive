### Post by asaz989 on 2007-11-27
The installation doesn't work as auto-run; to install linux, you have to boot from the LiveCD, which will let you try out linux without installing to the hard drive. For most modern computers, you should just need to restart the computer with the LiveCD in the drive to do that.

Then, if you like what you see, you click on an icon on the (Linux) desktop to install Ubuntu.

Here's the guide I like best:
[https://help.ubuntu.com/community/GraphicalInstall]("https://help.ubuntu.com/community/GraphicalInstall")
