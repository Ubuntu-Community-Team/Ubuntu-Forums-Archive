### Post by Sutekh on 2006-05-07
If you are new to Linux, and have bandwidth to burn I would suggest you download a LiveCD first.  

It runs from your PC's RAM, so it doesn't touch your hard disc.  You can determine then if Ubuntu will work on your hardware and if you want to install it for real.

As for 32bit and 64bit, try both CDs and see.  64bit doesn't have as much support for things like multimedia formats, java, flash.  They might work well, they might not. I haven't tried the AMD64 version for quite a while so I'm not really too sure.

So if you want to try a LiveCD get either the **ubuntu-5.10-live-amd64.iso** or the **ubuntu-5.10-live-386.iso**

If you decide you want to install Ubuntu get either the **ubuntu-5.10-install-amd64.iso** or the **ubuntu-5.10-install-386.iso**

### Post by tonyr on 2006-05-07
You really need to know what your processor is.  I don't know whether Asus A7N
series motherboards support AMD 64 processors.  The Asus motherboards
that that support **64** type processors are A8N series.  The **64** 
images are specifically for **64** type cpus.

The Breezy (Ubuntu 5.10) release is the latest stable release.  The current
development-and-almost-ready-for-release version is Dapper Drake,
(Ubuntu 6.06 LTS Flight-7).

I'm guessing that you need the 32 bit versions.  Breezy has a LiveCD
version; Dapper Flight-7 does not.  The Breezy images are at
[URL="http://cdimage.ubuntu.com/releases/5.10/release/"]
http://cdimage.ubuntu.com/releases/5.10/release/[/URL]

The Dapper images are at
[URL="http://cdimage.ubuntu.com/releases/dapper/flight-7/"]
http://cdimage.ubuntu.com/releases/dapper/flight-7/[/URL]
