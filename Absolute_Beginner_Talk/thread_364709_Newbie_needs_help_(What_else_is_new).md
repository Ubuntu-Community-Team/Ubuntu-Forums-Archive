### Post by g3k0 on 2007-02-18
Put the launcher in the menu by either right clicking on your menu button and select * edit menus* or press alt f2 and type alacarte.

Then click on the section you want it in (ie Accessories, education etc) 

Select New Item

type in the programs name in the command section and write a name for it (just use the same thing) and hit ok
