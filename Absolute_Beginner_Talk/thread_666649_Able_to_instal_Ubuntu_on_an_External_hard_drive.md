### Post by backflip on 2008-01-13
That's the set-up I've got on my desktop. I disconnected the internal drive first, then installed Ubuntu on the ext drive, that way  there is no possibility of your Windows being affected or compromised in any way. Make sure though that your BIOS allows booting from a usb drive, mine is now set to boot usb then hardd rive then cd etc. If I want to boot into Ubuntu I simply turn on the external drive first and then boot the compute, but  if I want vista (which I very, very seldom do), I leave the external drive switched off and the system auto boots into Windows. Never had a problem with it.
