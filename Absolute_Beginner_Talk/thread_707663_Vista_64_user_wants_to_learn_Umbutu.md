### Post by nofrendo on 2008-02-25
1. If you have command line experience with Unix, then using the terminal in Linux should be a lot easier for you.
2. I would suggest that you install 32 bit Ubuntu, because the 64 bit version has some problems with certain drivers.
3. Yes you need to "dual boot" to have Vista and Ubuntu, but the bootloader that comes with Ubuntu autodetects other operating systems and is fairly easy to configure.
4. Usually Ubuntu will detect your hardware on its own, something that any Windows can't claim to do. It works pretty well.

And yes, there are tons of guides both here and elsewhere on the interwebs. Do a google search for "dual boot vista and linux". 

Ubuntu is not too pleasing to look at at first, but with some work, you can make your desktop look like and do WHATEVER you want. The key is patience :)

Hope this helps.
