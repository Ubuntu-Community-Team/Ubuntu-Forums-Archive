### Post by daimaru on 2007-05-30
open ur /etc/X11/xorg.conf file 

sudo gedit /etc/X11/xorg.conf

scroll to section "screen" and add ur resosution to all the depth 1 - 24 like this:

SubSection "Display"
  Depth	1
	Modes	"1280x1024"	"1280x960"	"1152x864"	"1024x768"	"832x624"	"800x600"	"720x400"	"640x480"
EndSubSection

save and hit ctrl+alt+backspace to restart ur xserver. 
should be right res from now on .

### Post by daimaru on 2007-05-30
i would suggest just trying it. if ur xserver wont start anymore ull land at the console. 
to get back your olld settings (assuming you did a backup of ur xorg.conf file) you can type in console:
```
sudo cp /etc/X11/xorg.conf.yourBackupFile /etc/X11/xorg.conf
```
that overwrites ur xorg.conf file and restores the previous settings. so anything u did to ur xorg.conf file is rescinded.
bye the way if somehow u dont get a console during boot you can hit ctrl+alt+F2 to get into console mode.
