### Post by matthewv on 2006-03-22
To add a file to ur home directory, just type from terminal ```
nano ~/.xmodmap
``` and fill in the file. Press ctrl+x when ur done and save...

Or you could try using gedit (Applications --> Accessories --> Text Editor
