### Post by timcredible on 2007-06-19
i have a wireless broadband usb modem (verizon usb720), and gnome-ppp works it fine - i just clicked on autodetect, it found it fine, put in the phone number (which is kind of fake), username, password, checked the 'stupid' feature, and it worked.  not sure if a dsl usb modem will be different, but have you tried gnome-ppp?

warning about verizon's wireless broadband though - the first time you run it, it downloads something from the network and writes it to the modem, which means you have to run it once from windows (but it can be any windows machine with wireless broadband in the area).
