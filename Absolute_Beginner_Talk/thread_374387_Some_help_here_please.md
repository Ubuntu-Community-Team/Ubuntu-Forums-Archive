### Post by dannyboy79 on 2007-03-02
ok, well as I said. You are suppose to be able to use gparted and not worry about any data loss but it's always a good idea to back up your critical files first. I haave used windows backup tool before, it works pretty good, you can even compress if I remember correctly. the only way to get at your data is of course with winxp though cause it's like a .bk file or something like that.

or you could always use a windows tool to resize your current winxp hard drive. just make so that your current windows partition is resized to around 50gb (you need around 10gb for / and 1.5gb for swap. NOTE: these are just estimates, I am not exactly sure what the min for an ubuntu install is but I have installed ubuntu on a 10gb partition and I haven't had any problems.), then just leave the extra space unformatted. then you wouldn't have to worry about gparted (the installer's resizing tool) resizing your hard drive because you have already done it with a windows tool. then you could use the installer to create the 2 additional partitions, / and swap, and have the installer only format those to ext3 and you'd be set.
