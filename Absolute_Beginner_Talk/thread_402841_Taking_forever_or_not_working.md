### Post by steve.horsley on 2007-04-06
Try the "alternate" installer CD. This is a text-mode intaller rather than a full graphical environment. That said, the questions are about the same so it's no harder to use. It just has less video and memory requirements. It's a much more useful CD for installing, but it's slightly less useful for rescuing busted systems because you don't get a full Linux running to fix things with.
