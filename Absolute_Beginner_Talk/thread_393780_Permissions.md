I guess you are talking about file permissions. That would be whether the user (or root or group) can write, delete and/or execute a file. This is usually changed by the command 'chmod'. This guide explains everything:

[http://www.comptechdoc.org/os/linux/usersguide/linux_ugfilesp.html](http://www.comptechdoc.org/os/linux/usersguide/linux_ugfilesp.html)
