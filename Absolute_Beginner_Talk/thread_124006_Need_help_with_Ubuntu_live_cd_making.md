### Post by kaamos on 2006-01-31
He probably downloaded the .iso, but winrar by default associates with .iso files, so what you might see is the winrar file icon.

If you have the iso, you must burn it as an image. See [https://wiki.ubuntu.com/BurningIsoHowto](https://wiki.ubuntu.com/BurningIsoHowto)
