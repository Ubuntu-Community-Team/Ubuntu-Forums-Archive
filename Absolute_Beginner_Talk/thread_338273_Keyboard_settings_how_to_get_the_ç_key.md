### Post by Hasen_A on 2007-01-14
I am using a german keyboard layout "Germany Sun dead keys" with a "Compaq Easy Access Keyboard" and I would like to know how to access the following letter using a key combination or shortcut: ç

I am frequently now writing french and dont want to call up the character map every time I need this letter.

---

### Post by bapoumba on 2007-01-14
Hello,
check out the "compose" key : > System > Prefs > Keyboard > Layout options > Compose key position. I choose right ctrl key (because I'm on a laptop and there is no right windws key).

Then "compose" + "," + c = ç
With capslock on : Ç
Works for all letters with accents : "compose" + ~ + n = ñ (french keyboard and layout here, so no ñ or Ñ on keyboard for ex).
