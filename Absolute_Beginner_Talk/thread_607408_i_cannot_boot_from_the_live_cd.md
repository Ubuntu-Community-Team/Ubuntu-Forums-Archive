### Post by ubnewbie2 on 2007-11-08
When the screen goes black, is the hard disk flashing - indicating the system is working still booting.  I ask, because I know that sometimes the boot splash screen fails to show on some screens, however it still continues to try to boot, and when it get's far enough along that X windows starts, you will see a screen pop up and be able to use it.

It takes a long time to boot from CD, minutes, so if the disk light is flashing, be patient.

Also, you might be able to press ctrl-alt-F1 to bring up the text boot messages to watch, rather than a blank screen.
