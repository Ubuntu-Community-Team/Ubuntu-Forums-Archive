### Post by IYY on 2007-02-08


will take you one directory **up**.

```
cd -

```

will take you one directory **back**.

### Post by tbroderick on 2007-02-08
I'll just had a couple more useful cd's

```
cd ~/
```

Will take you to /home/username

```
cd "/path/to/My Folder"
```

Using quotes around the path is a nice time saver. It will translate to /path/to/My\ Folder. That way you don't have to remember to use all the "\" for spaces and the like.
