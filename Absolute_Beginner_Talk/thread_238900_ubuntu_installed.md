### Post by Opally on 2006-08-18
More pointless praise...

I got Ubuntu LAMP Dapper Drake 6.06 LTS installed on a Windows comp, an old Dell, and it went FANTASTIC. Not a single driver hitch.

I put Windows in a partition, made another bootable partition and a secondary partition (where I want to stash development files). The installer made a swap file partition by itself (!) and the extra partition *appears* to be mounted ... well I'm not sure where its mounted and if its got the right permissions ... understanding the file system is my next objective.

But that's not all! Due to a helpful post on these boards I was able to get KDE installed -- I am used to a GUI, but I'm not a total stranger to command lines either, my computer use pre-dates Windows.

And there's more ... I think I got Ruby and Rails installed too! Item number two on the agenda: check out Apache web server and RoR setup.

So, I am elated. I'll be hanging out here a lot more.

I did a little research after pondering what Linux distribution to use. Ubuntu seems well-supported and relatively friendly to the new-to-Linux-but-not-computer-newbie crowd.
