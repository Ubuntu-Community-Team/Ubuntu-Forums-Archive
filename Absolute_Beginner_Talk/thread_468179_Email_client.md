### Post by ccfjeff on 2007-06-09
Although I realize it isn't exactly what you're wanting, another possibility might be to view the mail online and selectively download the ones you would like.

It wouldn't automate the process for you nor delete the attachments from the emails you want to download before the download, but it might eliminate some of them.

I don't know which email clients allow that other than [Pegasus]("http://www.pmail.com/overviews/ovw_winpmail.htm") which is a Windows program.  I think that is an option with a lot of ISPs.

[Mailwasher]("http://www.mailwasher.net/") is a program that will allow you to get a list of incoming email and view them in text mode online.  It also shows which if any have attachments and has a number of other nice features.

You can mark them individually to either be deleted or downloaded.  You can also mark them as "friends" or "spam" so that any further mail from that source will be automatically marked for deletion in the future (though you can override it later if you like).

 It isn't exactly what you want, but it might be an improvement over your current situation.
