### Post by taurus on 2006-07-24
Actually, it's not that resource intensive at all...  Make sure you install both gdesklets & gdesklets-data and if you need more desklets, head over to [http://gdesklets.gnomedesktop.org](http://gdesklets.gnomedesktop.org) and download whatever you like there.

```

sudo apt-get install gdesklets gdesklets-data

```
