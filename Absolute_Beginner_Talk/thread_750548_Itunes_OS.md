### Post by aysiu on 2008-04-09
> **sirisaac87 said:**
> Is there a a linux operating system that will run itunes? Or is there a simple solution to getting itunes to work?
No, there isn't.

People always claim they can run iTunes in Wine, but there's always some functionality problem (can't sync with iPod or can't access iTunes music store, etc.).

Bottom line is that if you need iTunes, you should run Windows in some form (as a virtual machine, as a dual boot, etc.) or get a Mac.

There are plenty of iTunes alternatives that are native Linux programs, though.

Read more here:
[http://www.psychocats.net/ubuntu/itunes](http://www.psychocats.net/ubuntu/itunes)
