### Post by herbster on 2008-03-07
You should be using nvidia-settings, don't bother with the Screens and Graphics on an Nvidia card. Hit ALT+F2, type "nvidia-settings" and hit ENTER. It should also be in your Applications > System.

EDIT: overdrank beat me to it, and just to add if you run it as root as overdrank has instructed, you can then save your changes permanently to the xorg.conf file. If you run it from the Applications menu, it is not being run as root, and changes you make that would affect the xorg.conf will not be permanent.
