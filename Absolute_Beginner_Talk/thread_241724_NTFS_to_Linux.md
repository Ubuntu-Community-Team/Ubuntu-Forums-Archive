### Post by steve.horsley on 2006-08-22
Linux can read NTFS, see these notes: [http://ubuntuguide.org/wiki/Dapper#Windows](http://ubuntuguide.org/wiki/Dapper#Windows)

But it cannot write to NTFS. I believe that there are some experimental drivers but not mainstream well-trusted ones.

By what you say, reading only will be OK for you, so no problem.

Good luck.
