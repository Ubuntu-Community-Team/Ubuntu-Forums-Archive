### Post by LLRNR on 2006-12-02
There are of course alternatives, but it all comes down to what you really want. Some people might suggest an alternate app for Dreamweaver called Nvu (never used it myself) otherwise there are other means to make it run (using a compatibility layer program called Wine, it has an own implementation of the WinAPI, though it's not perfect) or you could as well run Windows in a virtual machine.

But BEFORE you do the complete switch, it would be better to set up a dual boot with your current Windows install. You'll be able to boot alternately, and when you'll finally decide to do the complete switch you'll do it... since then, though, I'd recommend you stick with a dual boot, it will save you from a lot of trouble.

LLRNR

First, you need to remember that Linux is not Windows. Once we've established that fact there are few options open to you ...depending on how willing you are to seek a change or challenge.

1. You can dual boot.

2. You can seek out alternatives to the Windoze software you are used to. ANd there are alternatives for nearly every piece of software.

3. You can get some Windoze apps to work in Linux using Wine or Crossover Office.
