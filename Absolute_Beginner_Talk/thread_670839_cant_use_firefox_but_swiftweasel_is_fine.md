### Post by cecure on 2008-01-17
Sweet that means its installed, if you prefer firefox over swiftweasel you can change the default browser by going System>Preferences>Preferred Applications, and then you can change the default from swiftweasel to firefox (if firefox is not in the drop-down click custom and type firefox in the command box below)

as for getting it on the menus Right click Applications, and then click Edit Menus.  
In this screen go down and click Internet on the left hand menu.
Make sure firefox has a tick next to it in the right and then click ok.
Firefox should now be in Applications>Internet.
cheers
