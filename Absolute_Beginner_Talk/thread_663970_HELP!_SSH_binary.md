### Post by PeterJS on 2008-01-11
How did you install the client?

Binary is another name for program, kinda. Binaries in the broadest referes to any file that isn't human readable, including pictures, music, movies, etc. However any binaries that falls in to one of those other categories are generally with the specific category name. Strictly speaking a picture is a binary, but most people would call it a picture. Also in the unix world the word binary has been used interchangeably with program for so long that you can safely assume that if somebody says binary they mean program.

And SSH stands for Secure SHell, and started out as a way to remotely connect to terminals on other machines, but has since been explanded to allow you to route pretty much anything over it like your thin client session.
