> I am looking for performance advice here. I should install the ...xfce ..is that it? window manager and then any tips to improve performance? Will the creative external modem work w/o problems with Ubuntu? i works fine now and she's running mandrake 7.x that i installed 2 years ago.

I would suggest you install xubuntu directly (that comes with the Xfce as default desktop environment. The main "performance influencing factor" on my system was the memory footprint: Xfce took about half of the memory Gnome is taking (and this means that you probably will use the swap less) --> more speed. On my system (with plenty of memory, so no swap in any case) I couldn't really appreciate a perceivable difference in speed (using the same applications).

I am unable to say if there are other Xubuntu optimizations for older machines "under the hood". I am not sure, as you can convert xubuntu into ubuntu or kubuntu just installing the relative package (e.g. ubuntu-desktop), so I imagine the system might be exactly the same, and only the desktop environment changes...

As for the hardware, sorry, I have no clue! :(
