### Post by seshomaru samma on 2007-06-01
Firefox may crash becuase of plugin issues like flash plugin
to know what causes the crash - start your applications from the terminal.
For example ,when you run firefox ,don't click on the icon but open a terminal and run
```
firefox
```
when firefox crashes it will leave an output in terminal which will help you understand what causes the crash.
