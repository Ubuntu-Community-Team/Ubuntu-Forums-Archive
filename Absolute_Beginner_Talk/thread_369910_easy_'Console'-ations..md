### Post by x1a4 on 2007-02-25
*kuake* is your best bet, which is in the repositories.  I've also tried *tilda*, *yakuake* and now I've settled on *kuake*.  IMHO it's the best implementation of the three.  The only problem I have with it is that it's based on Konsole and I would have preferred something like this based on Multi GNOME Terminal.  Oh well.  

*kuake* works in KDE, GNOME and Xfce (although in Xfce the transparency doesn't work).  If you don't want to install Konsole or any other KDE libraries, you're stuck with Tilda.
