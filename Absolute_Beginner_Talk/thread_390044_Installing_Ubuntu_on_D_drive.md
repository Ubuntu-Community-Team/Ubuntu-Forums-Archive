### Post by mikewhatever on 2007-03-21
It does not matter where you install Ubuntu, as long as it is not an A:\ floppy drive. All you need is some free=unallocated space to create Ubuntu partitions, format them, and then install. It does not matter which drive it is (pick any letter from C to Z). You should also be able to dual boot with Windows, although the more HDDs there are, the more complicating it can get.
