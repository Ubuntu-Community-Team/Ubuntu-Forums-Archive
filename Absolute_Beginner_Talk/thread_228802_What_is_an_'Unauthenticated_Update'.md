### Post by Average_Joe on 2006-11-22
It's really more of a 'mailbox' type of error than anything else.  Sometimes when new updates are released (and especially when several updates have been released at once), someone will forget to put on the appropriate keys to a piece of code for this or that small file.  Then (because many servers are used to send out these updates) when server X in Timbuktu is tasked with sending that particular file to your IP for inclusion in the update, then that file technically pass the security keys for the day's Ubuntu pushout and install of new software.

I haven't logged in to my Linux for a couple of days.  But I am willing to bet that when I do, later this evening, I will also get this type of error upon selecting to update my linux.  It sort of happens like that, where a jazillion people will receive this error on a given day.
