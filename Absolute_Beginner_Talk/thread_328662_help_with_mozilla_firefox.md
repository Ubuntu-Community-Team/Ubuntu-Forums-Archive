### Post by opticalfiber on 2006-12-31
once you've download the firefox it's a .tar.gz file, extract it where you want > then click on "run-mozilla.sh" execute. If you want to update it ubuntu-way (debian-way): open a terminal and run:
sudo apt-get update //to update the repositories
*if it's asking you a password it's the same of your user
sudo apt-get install mozilla-firefox
probably it would be already the latest 2.0.0.0 version
the repositories aren't updated as the official website, because the security team has to test it and create the .deb package!
