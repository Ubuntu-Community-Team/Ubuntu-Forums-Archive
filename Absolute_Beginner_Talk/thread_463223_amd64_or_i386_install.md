A few questions:  What are the apps (and drivers?) that you are having problems installing.  Perhaps we can help you out.

As far as performance goes, the performance gains will depend on what applications you are using.  If just using word processing and surfing the net, then no you will not notice any difference.  If you are ripping CD's, transcoding video, rendering, etc., then you can find upwards of 30% gain in performance.

### Post by PetePete on 2007-06-03
ive got a 64bit AMD, but use i386, mainly because i've had unneeded headaches with the 64bit software compatibility. If you are an advanced linux user i'd say go for 64bit and you'll probably have to get down and dirty with configs/CLI , if you are new to linux keep it simple and get the tried and tested i386!
