### Post by kpkeerthi on 2008-02-29
To answer your second question... 

Yes. You should be able to run VirtualBox. It works so well you hardly notice a difference in speed of your guest OS. VirtualBox does NOT support 3D yet. So it will not be able to utilize the capabilities of you host machine's graphics card. In short, you cannot play games in VirtualBox and videos may lag big time.

### Post by Paqman on 2008-02-29
> **3xpl0it said:**
> 
My question is, does Ubuntu support Intel Quad Core and does it also support 64bit for Intel processors?

Definitely. 64-bit support is excellent.

> 
will running Vista in it run smooth, or will it be choppy and slow watching a movie or using Dreamweaver/Microsoft Office for school work?
You've got a powerful machine, but you're always going to take a performance hit by running an OS in a VM. It should be usable though. You could always use XP instead of Vista.

You shouldn't need to use Vista to watch movies though. Ubuntu will do that nicely. Open Office can handle MS Office files, so you probably won't need to use that either. Dreamweaver is a slightly different story. It's still the best web design software on any OS. Kompozer is probably the best Linux equivalent. If you're doing simple stuff it'll be more than sufficient. It's fairly similar to Dreamweaver, actually.

> 
Third lol, Is there a torrent for whatever version of Ubuntu I would need to download?  (Is that even legal in the first place?  I figured Ubuntu was free so a torrent would be legal, but you never know nowadays)

Yep, torrents are usually the best way to get a download, IMO. They're often quicker than an html download (especially when new versions are out) and they don't use the Ubuntu project's bandwidth. It's a non-profit organisation, after all.

The [full download page](http://releases.ubuntu.com/7.10/) has the torrent files.

Ubuntu is released under the GPL licence. It's completely legal for you to download it, share it and even distribute it yourself.
