### Post by ThrobbingBrain66 on 2007-07-15
Are you saying that you want to have beryl running, but with no icon in your try?  If so, instead of running beryl-manager at startup, run
```
beryl --replace
```

for window decorations run ```
emerald --replace
``` or ```
heliodor --replace
```
depending on which WD you use.

EDIT: btw, Beryl forums are closed now.  To get support for beryl go here:
[http://www.opencompositing.org/](http://www.opencompositing.org/)
