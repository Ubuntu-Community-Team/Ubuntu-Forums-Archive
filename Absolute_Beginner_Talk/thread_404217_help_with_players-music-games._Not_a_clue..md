### Post by Famicommie on 2007-04-08
The first thing I do when I install an operating system is to make sure I get the necessary things I need to browse the web, play music/movies, etc. To that end, I recommend that you download and install a program called [Automatix](http://www.getautomatix.com/). It is a program that can automatically install the multimedia codecs that you need, install flashplayer for Firefox, and all kinds of other nifty things. For instance, it allows you to play .mp3 and .wav files. That should hopefully solve your radio problems (protip: use rhythmbox instead of realplayer).

A package is simply a program and it's required counterparts. There is a lot of boring history relating as to why linux has packages, involving the necessary "dependencies" that not all systems have on them. Packages include the programs and the dependencies, which will save you from a term that we call "[dependency hell](http://en.wikipedia.org/wiki/Dependency_hell)"

At some point, you will probably have to learn how to use the terminal. It's not very complicated, and with a little bit of practice it becomes second nature. Think of your computer like a car: the Desktop environment is like the driver's seat and the terminal is like opening the hood and tweaking the engine. There are quite a few [documents on the Internet explaining how to use the terminal](http://tldp.org/LDP/intro-linux/html/)

### Post by ayenack on 2007-04-08
If you would like to add packages to your Ubuntu desk top you will need to go to the Apllications menu tab at the top of the screen left click here and highlight Add/Remove tab and left click it this will bring up a window asking for your admin password this should be the password you log onto your system with this will open a window with many of the available packages for Ubuntu with a description of what they do. 

After you have installed the packages it will show you the path to open them i.e. Applications_Games_Blackjack. This is also an easy way to un-install programs just reverse the process.

If you would like to add more applications than highlight System tab menu at top of the screen Administration then Synaptic Package Manager (System_Administration_SynapticPackageManager).

In the new window click on search at top of window and type in what you are looking for e.g. "word processor" this will bring up a list of word processing programs available .
Click on the program that you wish for a brief explanation of what it does. And if you want to install it click the little square box to the left of the highlighted area and click install. This will tell the system that it needs to be installed click Apply at the top of the screen and this will download and install the program.

There will be some windows that pop up this is nothing to worry about.

As regards to media not playing try this. Synaptic Package Manager (as shown above) and click on search then type in Automatix this will install a program called Automatix. Then go to Applications, System Tools, Automatix (you will be asked for your Admin Password (this should be the password you log onto your system with) and go to codecs and plugins and choose AUD-DVD Codecs and Multimedia Codecs  then click on start at top left of screen.

You may also want to go to [http://www.youtube.com](http://www.youtube.com) to see if you have Flash Player installed. Once at the sight try to play a video if it does not play you should install Flash Player. This is easy type in [http://www.youtube.com](http://www.youtube.com) in the address bar of your browser this will take you to youtube home page again if you look to your right there should be a window with a little green jigsaw piece in it click it this will install Flash Player for you.

Finally you may wish to try these sites [www.opensourceconsortium.org](www.opensourceconsortium.org) and [www.openschoolsaliance.org](www.openschoolsaliance.org) these may be of some help.

Best of luck. ayenack.
