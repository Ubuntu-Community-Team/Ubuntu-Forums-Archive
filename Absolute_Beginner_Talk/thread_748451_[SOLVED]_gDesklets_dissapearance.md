### Post by ErusGuleilmus on 2008-04-07
You need to add gDesklets as to your session so that it will start with Ubuntu. I am not on an Ubuntu machine right now, but I think this may be correct. There is an option called "sessions" either under the Administration or Preferences tab, select that, then you should get the option to add to the session. I think the command for gdesklets it "gdesklets". If you add that, then it should run on start up. Sorry for the vagueness and possible inaccuracies.
