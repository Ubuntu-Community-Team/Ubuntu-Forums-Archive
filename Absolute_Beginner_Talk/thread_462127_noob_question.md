### Post by Tyke91 on 2007-06-02
easy way to do it in __ steps:


Downoad ImgBurn from this website
[http://www.imgburn.com/](http://www.imgburn.com/)

Download the Ubuntu Iso Image from this website
[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

use img burn to put the ubuntu iso image on a blank CD (click and drag the file into the img burn program)

restart your computer

hit whatever button makes your BIOS screen come up (you know... it'l say "hit F2 for setup" or whatever... go to that)

go to your boot menu, configure it so that you boot from the CD first.

restart your computer

select "Start/install Ubuntu" when promted



enjoy :popcorn:
