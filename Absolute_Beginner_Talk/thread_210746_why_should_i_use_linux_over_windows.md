### Post by Kilz on 2006-07-07
> **thunderfox933 said:**
> i am sick of windows but i need to know why i should use linux. Will my games work on?
Why someone wants to use Linux over windows is a personal choice. I [recommend reading this page]("http://linux.oneandoneis2.org/LNW.htm") to help you understand the diffrences, and problems of switching.
A lot of Windows games run under Wine or Cedega. Wine is free, but  it may take a lot of configuration and time to get them running. [You might want to look here]("http://appdb.winehq.org/") to see if the games are supported. Cedega isn't free, it costs $15 for a 3 month subscription. During that time you can download the program and any updates(updates add games). You can continue the subscription for $5 a month or let it end. The program will still work if you stop paying. The advantage of Cedega is that games install and run without a lot of headache. You will be playing in no time. The choice is up to you.

Well, for starters, Linux is MUCH more secure than Windows. Also, in Linux you have many choices, many different distro's and programs to choose from. Most of the Linux software is OpenSource, meaning freely available and distributable.

### Post by Bloch on 2006-07-07
There are many different distributiuons of Linux. Ubuntu is the most popular, but not the most user-friendly. The reasons for both of these is that ubuntu is true to the open source philosophy - it is and will always be a free system.
Other distros might begin to charge you extra for downloading software etc. (To be fair, most don't, but they are still developed by a priivate company and not a community)
Another difficulty is that ubuntu does not come with propietary formats - out of the box (believe it or not) it will not play mp3's, mpegs or dvds. These need to be downloaded.

 The forums here at ubuntu at the best help available - there is always at least 1000 users online.

Do you have any unusual hardware needs? Video cameras, combination scanner/printers? Ubuntu covers the vast majority of the most popular peripheries, but if you are used to plugging in your mobile phone, that kind of stuff is problematic
