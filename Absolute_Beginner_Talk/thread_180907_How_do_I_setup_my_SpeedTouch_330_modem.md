### Post by Wolfi3 on 2006-05-23
I managed it some months ago.. there are various ways out there, but I'm damned if I can replicate what I (finally) did in order to test other distros ](*,) 

This is the one that worked for me:
[http://steve-parker.org/speedtouchconf/](http://steve-parker.org/speedtouchconf/)

Basically it executes a script in terminal in order to logon.  Forget the connection software that comes with ubuntu - it doesn't use any of that.  To remain connected you must leave the terminal window open.  

Good luck with it :-)
