### Post by Kobalt on 2007-04-02
Installing Beryl with an Intel video card and a nVidia one is quite a very different thing... You will need to use AIGLX to enjoy Beryl. That's possible, but remember there is still a risk of messing up your system (Xserver mainly), so you might want to read a few posts on troubleshooting on Beryl.
Here is how to install Beryl so : [http://wiki.beryl-project.org/index.php/Install/Ubuntu/Edgy/AiGLX#How-to_install_Beryl_with_AIGLX_on_Edgy_Eft](http://wiki.beryl-project.org/index.php/Install/Ubuntu/Edgy/AiGLX#How-to_install_Beryl_with_AIGLX_on_Edgy_Eft)
