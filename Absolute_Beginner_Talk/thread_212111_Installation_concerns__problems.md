You can use the alternate CD to install using "text install":
[http://users.bigpond.net.au/hermanzone/](http://users.bigpond.net.au/hermanzone/)

Make sure to defrag your Windows partition before installing(may be why the installer couldn't resize).

You still might want to try out the Desktop(Live) CD to ensure it will run on your system, but you can install using the alternate(which many people prefer).

Note:  The other method was "OEM", which would be used by a pc manufacturer(no password is setup at install), so you can set up your
password later.
