### Post by Tautoa on 2007-09-14
There are a number of things already in place for old computers.
Xubuntu is Ubuntu with a lightweight window manager, so it can run on older systems with little memory.
There's also whole distros for older PCs, noteably Damn Small Linux, and Puppy Linux.
I'm planning on getting hold of a really old machine and trying Xubuntu, or maybe Ubuntu without a GUI at all... it would force me to learn how to use the CLI properly, and with a couple of hardware upgrades, maybe give me an excuse to build my first server :)
