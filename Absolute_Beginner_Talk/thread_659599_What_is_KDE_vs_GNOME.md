### Post by Bachstelze on 2008-01-06
> **Kedster said:**
> does installing kde interfere or make anthing harder to usein gnome 


No. the only way it will affect your Gnome in is that the KDE applications you install will also appear in your Gnome menus.

> **Kedster said:**
> and if i install kde is it exactly the smae a kubuntu

Yes, if you install the package *kubuntu-desktop*, which will give you the same thing as a Kubuntu installation (KDE plus all the Kubuntu applications and artwork). However, you can also install *kde-core*, which will install a very light and minimalistic KDE desktop, letting you add the applications you want afterwards.
