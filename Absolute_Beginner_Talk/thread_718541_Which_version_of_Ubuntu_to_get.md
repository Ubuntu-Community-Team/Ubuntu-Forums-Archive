### Post by insineratehymn on 2008-03-08
I suggest Gutsy Gibbon (7.10) for the time being.

There is a newer one, Hardy Heron Alpha (8.04) but as the name implies, it is just an Alpha release at this time.

I'm have no trouble doing anything I want on Alpha4, but I've gone through more than my share of headaches with Ubuntu so most of the problems seem overcomeable.

Although, if you are itching to install 7.10 right away, go right on ahead as there are droves of people that will help you. Buuuuutttt..... Hardy Heron is Ubuntu's next LTS (Long Term Support) so allegedly that will be the next bee's knees.

As for the Nvidia/Compiz thing... I think it's both of their faults. The nvidia drivers that come with Ubuntu don't like compiz, and if you do install the restricted drivers (something that Ubuntu doesn't support) then you are on your own, but compiz (possibly) will work.

In sumation, stability = 7.10
