### Post by paquito on 2006-03-11
applications>accessories>Terminal

you will start getting used to using the terminal... it takes a while to get used to some features, but really powerful and worth learning.

If you want easy methods to install software, use:

automatix ([http://www.ubuntuforums.org/showthread.php?t=114251]("http://www.ubuntuforums.org/showthread.php?t=114251")
or Applications>add application
or System>Administration>Synaptic Package Manager

unlike windows when u download a program, they often come in .deb format. from the console u can install using: sudo dpkg -i package_file.deb
([http://www.ubuntuguide.org/]("http://www.ubuntuguide.org/"))

it takes a while getting used to some things, I've just started getting into linux. u learn fast and people on this forum are really helpfull!
