### Post by JoshuaRL on 2008-02-02
That's a really good idea, and should save you money on the OS compared to Windows.

You shouldn't have too many problems with codecs.  Just make sure you install with it connected to the internet and dowload the package called ubuntu-restricted-extras.  And make sure you install the Java and Flash plugins so they work.  Other than that, I think it comes with everything the average user would need preinstalled.

I would suggest Nvidia graphics cards.  ATI's stink with open-source.  Either that or MBs with Intel integrated cards.  Those are really good too.
