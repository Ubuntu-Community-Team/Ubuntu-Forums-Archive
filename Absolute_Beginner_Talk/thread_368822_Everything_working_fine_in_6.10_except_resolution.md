### Post by AndyCooll on 2007-02-23
Nope, xorg.conf is still where it has always been.

Copy and paste the following into the terminal:
> sudo gedit /etc/X11/xorg.conf

That will open up the xorg.conf file ready for editing, You have to copy it exactly as it appears above, including the spaces and capitals. Linux is case sensitive, so "x11" isn't the same as "X11" for instance.
