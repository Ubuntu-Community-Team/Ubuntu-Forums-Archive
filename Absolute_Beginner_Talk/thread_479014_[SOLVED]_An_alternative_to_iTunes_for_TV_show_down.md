### Post by dfreer on 2007-06-19
You can check the main network sites (ABC, NBC, CBS etc), as most of these provide the shows free to watch online. However, as for iTunes there really is no suitable alternative in linux (yet!). Hopefully either apple will release a linux client, or wine matures enough to handle it. I hear you can run itunes 4.X on crossover office, but that won't help you as iTunes store requires at least 6.X if not 7.X to login/download.

What you can do is this: Get vmware server and install a copy of windows in that, or run a rdesktop to another windows machine so that you can use it's copy of iTunes. Sadly, those are about the only alternatives I know of :(

### Post by Concorde on 2007-06-19
I use a vm (virtual machine) of Windows XP within Ubuntu called VirtualBox. It's great for downloading Itunes songs within Windows. This way I have the best of both worlds without having to use a dual-boot setup. Then I can use Itunes and PE-Builder within Windows, copy files to a shared folder in Ubuntu and presco! I still can only listen to music files on Itunes within Windows. I haven't had time to mess with reconfiguring VB to see my CD-RW Drive for Itunes though. So putting it on a CD may be troublesome. As far as PE-Builder, I just copy the modified files over to a folder within the Home.directory and then I can access my CD-RW drive.
