### Post by PeterJS on 2008-04-16
The name of the flash package is: flashplugin-nonfree, it can be installed from synaptic, or from [here](apt:/flashplugin-nonfree). As for updating things, package management will keep you up to date with the leastest updates for your release, so as long as you update when update manager shows up in the tray you will always be up to date. And if you absolutely must install things stand alone, try and get them in .deb format.
