### Post by Alterax on 2007-06-07
I agree that you can turn it off, but not so much that it is irrelevant.

Many places that I have lived at had interesting quirks with their cable modems, in that you could access others' network resources through NetBIOS over TCP/IP.  Of course, that's mainly a Windows problem; I am not sure if Ubuntu uses NB, since it is pretty much obsolete anyway.

My recommendation is simple in regards to network services:  Keep it simple as you can.  If you don't need it, turn it off, because each service is a possible vulnerability.  Even though Linux distros like Ubuntu are more secure than Windows, they're not immune to attack, so it's always best to be safe.

--Alterax
