### Post by taurus on 2007-05-28
You need to install ntfs-3g if you want to write to ntfs filesystem.

Applications -> Accessories -> Terminal
```
sudo aptitude update
sudo aptitude install ntfs-3g ntfs-config
gksudo ntfs-config
```
and point it to your ntfs partition.
