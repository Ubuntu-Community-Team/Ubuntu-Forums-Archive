I personally think totem is a big heap of crap and haven't liked it from day one. I rarely get it working as well. No errors, just plain not working, so my honest advice is to leave it alone! Instead, I use gxine to play dvds and movies. I use mplayer-mozilla plugin (info is on above url) for media in my web browser, and if I want any other media played across a network or the internet, i use VLC. 

For more information on playing media in Linux, check out my Linux Newbie Guide website at the following address: [http://www.linuxnewbieguide.org/chap10.php#video]("http://www.linuxnewbieguide.org/chap10.php#video")

The above url contains links to all the specific homepages of all the video players mentioned and they should be installed through the Synaptic Package Manager for ease and best results.
