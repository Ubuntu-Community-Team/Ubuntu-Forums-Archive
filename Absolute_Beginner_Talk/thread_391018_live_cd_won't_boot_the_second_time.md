### Post by louieb on 2007-03-22
Sounds like you got lucky the first time. or the CD got scratched in between the first and second boot.  
as sad_iq 	wrote: Try burning the cd at a lower write speed (8x)...also check the md5 of the iso!!!
I always use 2x or 4x to burn. But first run the media check option of the CD. It does a checksum of important parts of the CD. If it reports errors then you have no choice  but to burn another.
