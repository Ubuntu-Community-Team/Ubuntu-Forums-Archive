 Hrm... That sounds a little worrying!

Do you mind if I ask what programs you deleted, and how? 

The best way to remove a program is to use the Synaptic Package Manager (System--->Administration--->Synaptic Package Manager). Any program with a green box next to it is installed. Right click and select 'Mark program for Removal' to uninstall it - and save your settings in case you reinstall it later -  or 'Mark Program for Complete Removal' to uninstall the program and any configuration files it created, too. 

If you install programs through any other method than Synaptic, then removing them can be a bit of a chore. Let us know what you did, and we'll see if we can help clean up.
