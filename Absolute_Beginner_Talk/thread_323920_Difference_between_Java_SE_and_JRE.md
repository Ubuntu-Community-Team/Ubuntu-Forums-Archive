### Post by jvc26 on 2006-12-23
The simple difference is that the JRE Update 10 is the java runtime environment for running java applications (1.5 version 10), the other one is the JRE **and** the plugin to run java apps with mozilla firefox. Entirely up to you which you'd like to install, the easiest to do so you have the most things you can do is to go for the one ith the plugin then you have mozilla set up too should you want to access java content on the web.
