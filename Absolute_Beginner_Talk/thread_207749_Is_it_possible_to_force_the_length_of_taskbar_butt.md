### Post by someusernoob on 2006-07-02
[QUOTE=Cherry Popper]I'd like to keep my taskbar buttons short like so -

[ATTACH]12087[/ATTACH]

instead of -

[ATTACH]12088[/ATTACH]

Would it be possible to force the length to stay fixed at all times?[/QUOTE]

Right click on the beginning of the windows list, select preferences. Go to the tab called "size" and set the minimum and maximum size to something like 750, or whatever size you're comfortable with.

### Post by noynac on 2006-08-04
> **someusernoob said:**
> Right click on the beginning of the windows list, select preferences. Go to the tab called "size" and set the minimum and maximum size to something like 750, or whatever size you're comfortable with.

This sets the maximum size of the window list not the individual buttons. By limiting the size of the window list, you will also limit the size of the individual buttons to a certain extent, but I'm not sure this is what the OP wants to do.
