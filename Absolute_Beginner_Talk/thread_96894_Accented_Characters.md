### Post by AgenT on 2005-11-29
You can either change the keyboard layout to your desired language (note that keys will probably be changed!) or set the compose key (or both). You can do this in System -> Preferences -> Keyboard.

If you set your compose key to the right control key, you can write an &#233; using this combo: Ctrl (the right control) + ', then pressing e. The + means you press both keys together. Or, better yet (so as to avoid mistakes), hold Ctrl and press the corresponding key, then release and press the remaining last key. For an &#232; you would use: Ctrl + `, then e.

### Post by SickTwist on 2005-11-29
There are two applets I know about they may help you with this. Right click on an empty area of a panel and click on "Add to panel...".  You will see a window with several applets that you can drag to a panel and position wherever you would like them. "Character Palette" allows you to quickly select from accented letters and "Keyboard Indicator" allows you to select different keyboard configurations and see what is currently selected. Hopefully one of those will help you.

Bonne chance!
