### Post by emarkd on 2008-02-04
The Nautilus side pane will show mounted locations by default.  You enable it by clicking View > Side Pane, and you can change the type using the drop-down at the top-left when it opens.

As for your drives, Linux doesn't really have a concept of "drives" like Windows does.  In Linux, there's only one file system.  Everything is in that file system and everything in there is a file (including that keyboard you're typing on).  It's a different perspective and it takes some adjusting for some Windows users.  Linux is different.
