### Post by annda on 2007-03-31
defragment your disk in windows, and install - when you are asked about partitioning, choose 'use free space'

a guide with screenshots is here:
[http://www.psychocats.net/ubuntu/installing](http://www.psychocats.net/ubuntu/installing)

EDIT: since you already have two partitions, you can see which one is the 'main' one, no the recovery (you don't want to install on that). while you are still on live cd, type

sudo df -h

the larger one using NTFS filesystem is the one to choose to resize on install (it's hda# or sda#)
