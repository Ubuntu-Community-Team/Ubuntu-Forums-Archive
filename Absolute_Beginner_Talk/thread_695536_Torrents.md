### Post by NightwishFan on 2008-02-13
You uninstall using System/Administrator/Synaptic (Assuming you got it with apt-get or add/remove etc)

Wine is an application which aims to allow Unix-like computer operating systems to run programs originally made for Windows.

It is buggy for most software, in my experience. However I have gotten a few games (Starcraft and Morrowind) to run almost flawlessly in it. Most common software from a year ago or more should work easy. You can always check the Wine DB. Install via Synaptic, or the Wine home page.
