### Post by gilforsyth on 2007-04-24
I'm at work and stuck with Windows, so I can't check this right now, but I think that if you right click on the drive under /media/cdrom0 and go to properties (with an audio cd in the drive) you should be able to select the program that it opens with.  If you do that once for audio cds and once again for dvds you should be able to start them just by clicking.  

Alternatively, if you can't do it for the whole drive, you could make VLC the default application for *.cda (using the above method on a regular audio cd track) and do the same for the *.vob files on a dvd.
