### Post by crimesaucer on 2007-04-18
I think you can basically have every app that is offered in ubuntu. To learn more about xubuntu you should read this link: [https://help.ubuntu.com/6.10/xubuntu/desktopguide/C/index.html](https://help.ubuntu.com/6.10/xubuntu/desktopguide/C/index.html)

You can install anything in the repos for xubuntu that you can in ubuntu. You can also compile anything for xubuntu that you can in ubuntu.

The major differences is that xubuntu works best when using apps that are built around the gimp tool kit. 

On a good machine that means your apps work really fast. On an old machine it helps make things quicker.

I started with xubuntu edgy and have stayed with it. 

For the few minutes I've spent using regular ubuntu, all that seemed different was that it came installed with more apps and games, and the menu and file system were different, and it was slightly slower.

Ubuntu's file system and menus also seemed a little easier for a beginner to install new themes into and to know where to access certain things.

Ubuntu seems easier and more beginner friendly, and most of the forum threads are for ubuntu so sometimes you have to figure out things on your own to make it work with xubuntu. But if you spend some time to figure out where to put things and how to access them in XFCE and Thunar, then you will realize that XFCE is a really nice desktop environment and Thunar is a nice file system.

It just takes a little time. 

The newest XFCE4.4 is out now and it looks really nice, you should check out the XFCE site to see what it offers because that is basically what xubuntu wiil have to offer plus everything that ubuntu has in it's repositories:  [http://www.xfce.org/about/screenshots](http://www.xfce.org/about/screenshots)

### Post by Tundro Walker on 2007-04-18
Xubuntu basically comes with a different, lighter-weight desktop, and installs less default software (EG: no OpenOffice.org, instead you get lighter apps like AbiWord word processor & Gnumeric spreadsheet).

Like CrimeSaucer said, you still have access to the repositories, so you can install whatever you want.  Heck, you can install the Ubuntu GNOME desktop and GNOME applications, along with OpenOffice after Xubuntu's on your machine and basically make it like an Ubuntu install.

Again, the difference is just the default desktop and preinstalled apps.  You still have access to the repo's, can install what else you want, and the System Updater still keeps it up-to-date like in Kubuntu & Ubuntu, because the core system is still Linux / Ubuntu distro.


If you want to use a lighter system, Xubuntu is actually pretty good and then trim some stuff off if you don't want to quite tackle a "server install" and work your way up.  On my old 800mhz, I started with Xubuntu (2nd install after a first one using Ubuntu), and then uninstalled the "xubuntu-desktop" meta package, which let me uninstall other XFCE apps / applets and other things I didn't want.
