If you like KDE, you will want to try out Kubuntu not Xubuntu. Xubuntu uses a GUI with the window manager XFCE and is designed to require less resources than Ubuntu or Kubuntu; Xubuntu might be a good choice for an older computer that doesn't have a lot of RAM.

A good discussion of some of the differences between Kubuntu and Ubuntu may be found [[COLOR="Red"]**_here._**[/COLOR]]("http://www.psychocats.net/ubuntu/kdegnome")
Also, there is a link on that site to [[COLOR="Red"]**_another page that has a guide on window managers and desktop environments_**[/COLOR]]("http://xwinman.org/").

When Ubuntu and Kubuntu are first installed, each begins with a different package of default applications. Some examples: the basic editor in Ubuntu is a program called gedit - in Kubuntu it's kate; for burning CDs/DVDs Ubuntu installs Gnomebaker - KDE installs K3b. There are many other examples, but you can always install a program you want to try from KDE to Ubuntu or the other way around. You can even install either Kubuntu or Ubuntu and then install the other window manager at the same time - then you can select which one you want to play around with at login time by choosing either a KDE or Gnome session. 

If you like the look of KDE, I would suggest you install Kubuntu and play around with it for a while. If you then later want to try out some Gnome applications or also play around with Gnome, you could search the forums or post for help on how to install Gnome alongside with KDE.

### Post by thefoolisme on 2007-04-06
I think you're trying to compare Kubuntu and Ubuntu.  Kubuntu uses the KDE desktop.  Xubuntu doesn't use KDE or the Gnome desktops, but used XFCE.  Xubuntu is a leaner OS, primarily built for systems with less resources.   

Between Ubuntu and Kubuntu, I think it is really a matter of personal preference in desktops.  From what I understand, apps for KDE will run in Gnome, and vice-versa for the most part, and you can install KDE on Ubuntu if you want.  Some of the original packages installed in the initial installation are different, but you can change that around anyway.  The speed of the two desktops is about the same, the system requirements are the same.  If you like the asthetics of the KDE, then go with the Kubuntu.
