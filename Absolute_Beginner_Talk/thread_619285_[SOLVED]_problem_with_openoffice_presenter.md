### Post by MegaJim on 2007-11-21
There is a known issue with the gtk extensions for open office, It is possible presenter is displaying a dialog on start which is causing it to crash.  You can find out more information about this problem here: 

[http://ubuntuforums.org/showthread.php?t=584088](http://ubuntuforums.org/showthread.php?t=584088)

To save you some reading, you can try to fix the problem quickly by removing openoffice.org-gtk using synaptic
