Or, you can just use parted which comes with ubuntu: sudo parted

### Post by MasterJS on 2008-02-18
Normally when people use a drive as backup, they format it to FAT32 or NTFS. I'm not sure what you want it as, I'm not recommending anything. I just wanted to point out that if you're ever going to use the drive on a Windows computer, I would recommend getting it formatted by a Windows computer because a lot of people get the drive saying it was unrecognized by Windows. I'm just pointing this out to save you a potential problem in the future.
