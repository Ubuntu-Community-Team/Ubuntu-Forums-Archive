### Post by MattJ100 on 2007-09-12
The quick answer is no - Windows applications do not work on Linux.

What many people don't realise however is that for every Windows program, there is nearly always a Linux equivalent.

For other applications you can use WINE, which will make a good attempt at running most Windows applications. Depending on the application itself, and how it works, your mileage may vary. For a guide check [http://appdb.winehq.org/](http://appdb.winehq.org/)

Also you should take a look at Ubuntu Studio, a version of Ubuntu for the creative type :)

[http://ubuntustudio.org/](http://ubuntustudio.org/)
