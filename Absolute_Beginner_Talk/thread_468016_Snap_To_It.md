### Post by Tomosaur on 2007-06-26
Somebody asked this same question a while ago, I said I'd look into adding it as a new feature for Gnome / Metacity, but I still haven't got round to it unfortunately. If anyone has some free time and experience I'm sure many users would be very grateful. Although Metacity would probably be what you would need to mess with to do it, I haven't looked as extensively into it as I would have liked. Added to this, it may be more worthwhile seeing if it could be done directly into X and thus available on other DEs. I also don't know how it would work if the user enabled the Desktop Effects - I would imagine you'd need to write the code for that too.

Anyway - as far as I'm aware, there is no current way to do this in the Gnome desktop environment, it seems to have been completely overlooked despite it being a very useful feature. You may, however, be able to find some third party software / modification which will do it for you, but I don't know of any.

I can't speak for KDE, but it could be worth checking it out to see if it's a feature present on that.
