### Post by b-atx on 2008-01-19
Is it possible for me to boot straight into my desktop without having to input my user name and password?

It would be great to just hit the power button and walk away, knowing that Ubuntu will boot, Azureus will automatically run and resume all my downloads, without me having to use the keyboard.

---

### Post by mister_pink on 2008-01-19
System --> Administration --> Login Window --> (password prompt) --> Security --> Enable Automatic login

---

### Post by b-atx on 2008-01-19
Perfect, thank you.
