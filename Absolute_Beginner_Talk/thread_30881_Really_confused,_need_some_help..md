### Post by tread on 2005-05-12
Linux != Windows :)

Windows programs will not work in Linux, though you will find equivalent programs for each utility you use. There is also a windows emulation mode which lets you play some windows games on Linux.

[This might help.](http://wiki.linuxquestions.org/wiki/Linux_software_equivalent_to_Windows_software) 

Welcome aboard, and I hope you enjoy the ride :)

### Post by az on 2005-05-12
If you are really hard-pressed, Linux can run some windows software using wine.  It is a windows emulator.  There is a version of it in the universe repository, but you should use the newst version found at winehq.

Once you are on the net, go here to read how to install wine from winehq
[http://winehq.com/site/download-deb](http://winehq.com/site/download-deb)

However, you should really use software written for linux.  There is usually no need to use windows software for much of anything in Ubuntu.
