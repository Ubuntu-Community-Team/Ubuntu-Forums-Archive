### Post by MasterOfDisaster on 2006-12-29
Nero (CD/DVD burning program):
Brasero, GnomeBaker

Nero Recode:
dvd:rip, Transcode (CLI), Auto-Transcode (Beta GUI for Transcode),

DVD Clone:
dvd::rip, XDVDShrink

AnyDVD:
dvd::rip, XDVDShrink (however, not sure if either can control drive speed)

SnagIT:
xvidcap

Macromedia Fireworks:
Spalah (not sure if maintained), Inkscape, GIMP

Linux takes a bit of getting used to, but many find that it can be a very usefull operating system.
