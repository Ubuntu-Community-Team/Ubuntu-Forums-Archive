### Post by _simon_ on 2006-09-30
1. Not many ISP's, if ANY at all will officially suport Linux (they're too lazy to train their staff) but Linux should work with ANY ISP.

2. I Can't help you with your dialup modem but you should be aware that when you install Ubuntu you'll have a fair few updates to download and on dialup it may take you a few days.

Also be aware that Edgy Eft (Ubuntu 6.10) will be released towards the end of October so you may want to wait for that rather than installing 6.06 (current version). - Edgy Eft is currently available as beta if you wanted to try that anyway.

### Post by Ven on 2006-09-30
You can easily dialup from the kde program **Kppp** from **Kubuntu** with almost all ISPs. The only info you need is:
1. Your ISP dialup #
2. user name
3. password

My suggestion is to give it a shot with the Kubuntu live CD with your current dialup ISP.

There is a much better program called **Knet,** but Kubuntu [doesn't come](http://releases.ubuntu.com/kubuntu/6.10/kubuntu-6.10-beta-desktop-i386.manifest) preloaded with it (**it should**), as of Edgy-Beta. If it doesn't have many dependencies, you can download knet separately and install the packages once the desktop loads. [Try it out](http://releases.ubuntu.com/kubuntu/6.10/).

Also, if you have a FAT32 partition, you can use the so-called "persistent" mode with the live CD, so that you don't actually have to install the distro to its own partition. Please see my posts [in this thread](http://www.ubuntuforums.org/showthread.php?t=268033).
