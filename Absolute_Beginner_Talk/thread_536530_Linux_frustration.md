### Post by anewguy on 2007-08-28
Ubuntu is wonderful, but there are a lot of common "problems" for which you should be able to find help here.  Some samples:

- wireless cards
- video cards
- blackscreen at boot or when it should be the log on screen
- resolution problems
- problems with using Beryl/Compiz/Compiz-Fusion

These things are common problems, and with the correct question(s) you can get help here to get all of these working, normally with very minimal effort.

Can it be frustrating?  Sure - but we are here to help.  I've gotten TONS of help from these forums, and learn more everyday by going through them.  I think most everyone who has changed to Linux in one flavor or another as run into some problem - the thing is that most of the time the answers can be found right here, either by reading a lot of the posts, or by asking your question(s).

Please post what some of your frustrations are and perhaps we can make Ubuntu and LInux the pleasurable experience it should be.  I was frustrated, too, when I first came to Ubuntu.  If you have no exposure to Unix or some flavor of Linux, there is without a doubt a learning curve.  It's up to you how hard you want to make that learning!:)
