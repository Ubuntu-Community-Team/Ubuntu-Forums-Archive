I think the Live CD fails on laptops very often, because of hardware compatibility issues. It sounds like your incompatibility is with the video display driver that the Live CD is using.

If you want to install Ubuntu, you can try the Alternate Install CD. It installs Ubuntu using a text-based installer that will work on almost everything.
