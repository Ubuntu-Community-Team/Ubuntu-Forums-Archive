### Post by kinghowdy on 2008-02-29
The Ubuntu Live CD or USB drive should work with most monitors/graphics cards. If it does not then I would try using Safe Graphics Mode but you should be able to use the default live environment most of the time. If it still doesn't work you may want to try the Alternate disk but that does not provide a live environment it is an install disc only. As far as I know there is not a way to have the live disc/usb pen cycle through drivers, but I may be incorrect.
