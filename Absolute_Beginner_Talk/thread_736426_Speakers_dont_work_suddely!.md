### Post by sirgogo on 2008-03-28
Hey,

My suggestions, check the following:
1. You have a login sound (ubuntu theme)
2. You have GStreamer and all related mp3/sound plugins installed. If not, you can go to synaptic package manager and look up mp3, you should be able to find a bunch of pluggins.
3. As stupid as it sounds, make sure your speakers are plugged in/and on.
4. If all else fails, you might want to try reinstalling/recompiling your kernel.


Good luck and let me know how it goes!
