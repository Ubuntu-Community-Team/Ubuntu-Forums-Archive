### Post by meborc on 2007-10-01
making the icons in the panel is simple... you need just the command you insert in terminal to get the program to work... for example... **firefox** starts Firefox... **oowriter** starts OpenOffice Writer... **pidgin** starts Pidgin etc etc...

if you are not sure... just start up the terminal... type the 2-3 first letters of the program (small letters) and hit the TAB key... it will then list all the commands starting with the same letters... the commands are in 95% of cases just the program name with small letters :)

hope you have fun with xfce4.4.1

EDIT: **xmms** is the command for XMMS and **xfce4-terminal** is the command for the terminal ;)
