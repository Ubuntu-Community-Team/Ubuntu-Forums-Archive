### Post by bobpur on 2008-01-01
Go to [www.distrowatch.com](www.distrowatch.com) and download Pmagic or Gparted and burn the ISO to a cd. Use this to format your drive. Forget the versions on the Windows or Ubuntu cd's. Although, if you choose to use one, I'd use the Ubuntu one. The stand alone partitioner is the best way to go.
Providing you have a good cd that you are trying to install from, a PATA drive will have issues being recognized if the jumper on the drive is set wrong for its position on the wide flat ribbon cable that these drives use (whew!).
