### Post by TwistesdTexan on 2007-06-02
Was firefox slow or did it crash? Firefox was extremely slow until I disabled the IPV6 using "about:config". Just type "about:config" in the address bar and hit enter. Then type IPV6 in the filter and the item you want to change will be there. Toggle the value to true. This has sped up things greatly for me.
