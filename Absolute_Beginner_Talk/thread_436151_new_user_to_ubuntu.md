### Post by uchuunoyanushi on 2007-05-07
[http://www.ss64.com/bash](http://www.ss64.com/bash) has a list of all the bash commands that you can use in Ubuntu.  You can also use the man command for more info on any command, which is particularly useful for third party programs (like mplayer, for example).

Also, [www.linux.org](www.linux.org) has a good tutorial which introduces the terminal and a lot of useful commands.
