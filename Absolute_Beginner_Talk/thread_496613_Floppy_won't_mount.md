### Post by ADT on 2007-07-21
Did you modprobe the floppy module?

The reason why your floppy disk wont mount is probably because the driver has not been loaded at start up. Modprobe should do this.

Open up a terminal emulator. I thinks its in applications>system>terminal

Then type:

```
sudo modprobe floppy
```



In your fstab file this entry:

```
/dev/fd0        /media/floppy0  auto    rw,user,noauto  0       0
```

deals with your floppy drive.

when you type:

```
sudo mount /dev/fd0
```

mount checks the fstab file for the directory to mount the device i.e. /media/floppy0.
But if that does not work try doing this:

Create a directory called floppy in your home directory just to test if it works.
And now try mounting the floppy:

```
sudo mount /dev/fd0 /home/username/floppy
```
