### Post by hiloguy on 2007-09-11
I don't know if there is a special place on this forum for recommending books, but BEGINNING UBUNTU LINUX by Keir Thomas is an excellent resource to keep close at hand.  It's actually written in English, gently introducing the newbie to UbuntuSpeak.  It is well indexed and nicely presented.  It covers everything from getting started up to advanced hacks.

If it were not for that book, I'd be on this forum a lot more.
