### Post by Sef on 2006-05-22
> Do I need an iso burner software program or will the Xp sp2 have that all ready? And if I need one what are the settings and are the freeware ones off of File Hippo like Deep Burner good to use?

For how to write an iso to a disc [click here.]("http://www.petri.co.il/how_to_write_iso_files_to_cd.htm")

> After successful image burning (700MB CD R/W will be sufficient, right?), how do I go about the boot install. I create a partition and a swap file right? My computer has 60GB total disk space, and I am planning on Dual booting with my Xp and I would like to keep a little more space on the Xp for my music/movie files.

Read this tutorial on dual booting:

[http://users.bigpond.net.au/hermanzone/]("http://users.bigpond.net.au/hermanzone/")

> And to start, say after all the above questions are answered, do I put the CD in to the tray, then restart and press (it's esc for my computer) the key to boot a CD?? Is that correct procedure.

If the BIOS is set to check the CD before the hard drive.  If not, you will have to go into the BIOS and change it.  Usually the keys are either esc, F2, or delete.  It will tell you what to push when your computer first boots up.  Best to keep pushing the button until you get into the BIOS.  Just don't start changing anything other than the boot sequence.  Otherwise you could render your computer inoperable.
