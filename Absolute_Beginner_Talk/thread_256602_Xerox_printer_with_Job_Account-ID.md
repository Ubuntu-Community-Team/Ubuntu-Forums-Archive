### Post by citybird on 2008-07-24
this dialog box that appears is an accounting/billing application that is installed on your windows machine. It probably reports to a central server and can be used to track printing costs and create monthly reports to help with billing.

If the developer of this software has a linux version of the application you may be in luck but most likely they do not.
