### Post by erginemr on 2007-12-07
Hello,

Could you please check the following thread:
[http://ubuntuforums.org/showthread.php?t=626835](http://ubuntuforums.org/showthread.php?t=626835)

and check my message over there to see if it matches your problem. (There are further links in my post there down the road.)

When you start the computer with the Live CD, you can bypass the Ubuntu boot splash screen and see the boot messages by pressing "Alt+F3", and if the screen is still blank, "Ctrl+Alt+F1 to F6".

If you detect an error message like "Retrieving libc6-udeb" and the computer stalls, then most probably, your cd/dvd drive is a type of Benq derivatives, notorious for having DMA (direct memory accress) issues with Ubuntu.
