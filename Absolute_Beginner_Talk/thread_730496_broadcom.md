### Post by Iandefor on 2008-03-20
Well, if it's anywhere in the Ubuntu Repositories, you can use [Ubuntu Package Search]("http://packages.ubuntu.com/") to find what you need. There are download links for the packages.

A better solution if you have Ubuntu/Debian on both computers is to install [AptOnCD]("http://aptoncd.sourceforge.net/") on the computer that has internet access, do > sudo apt-get -d install [package file]And run AptOnCD.
