### Post by moephan on 2006-04-04
Not really. There are some tools in the repositories for visually designing databases in mysql. LAMP is an application development stack. If you used the repositories to install apache, php, amd mysql, they should be running ready for you to just start writing some web apps using your favorite editor and command line tools.

You might want to consider checking out ruby on rails. If you search the web, you can find some good tutorials for getting started.
