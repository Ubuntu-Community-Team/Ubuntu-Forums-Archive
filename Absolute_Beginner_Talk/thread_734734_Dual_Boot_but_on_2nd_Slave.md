### Post by Squish on 2008-03-25
If it was me, 
I would setup install ubuntu on the sata along side xp, thus making an easy dual boot, and then I would set aside the extra ide as a universal storage used by both operating systems. I would suggest partitioning the storage then as a ext3, and installing a ext3 driver on xp, and mounting it under ubuntu as whatever you want in /media.
