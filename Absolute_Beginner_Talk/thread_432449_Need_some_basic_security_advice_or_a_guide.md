### Post by Billy McCann on 2007-05-04
A strong password is all you really need. I use clamscan to scan mail and flash drives, just so I don't pass anything along. 

Also, I like to have my ports stealthed instead of merely closed, which is the default.  I'd recommend Guarddog over Firestarter, because Firestarter doesn't work.  But be careful with Guarddog.  You have to enable certain protocol just to be able to access the internet.
