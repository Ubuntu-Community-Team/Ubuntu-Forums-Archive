### Post by paker on 2007-10-07
Wireless connection used to work for months. One day it stopped working.

1) It used to prompt me to enter key ring password. Not any more.
2) Instead of wireless connection icon at the top, manual connection icon shows up.
3) As I select manual connection, wireless connection is already check marked with as good a signal strength as it used to. Wired connection is "-" marked. I removed the mark. Nothing else I am prompted to do.

### Post by Beggar on 2007-10-07
Go to wireless connection -> properties make sure that the roaming mode box is checked.
