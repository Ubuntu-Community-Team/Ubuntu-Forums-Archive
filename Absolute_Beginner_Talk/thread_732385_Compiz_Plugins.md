### Post by NightwishFan on 2008-03-22
```
sudo apt-get install compizconfig-settings-manager
```
Above appearance will be the advanced desktop settings where you can configure the plugins.

If you mean the extra plugins there is a thread for how to get them on gutsy, but they arent worth it in my opinion.
