### Post by Jerry N on 2007-08-20
[SIZE="2"]OK so I am not only a noobie but I am a stupid old noobie.  My UBUNTU (7.04) machine has been able to see shared files on the XP machine for months but until now it has flatly refused all efforts to let the XP machine see files on the UBUNTU machine.  I also have Freespire 2 and Linspire 6 and they share files both ways with the Windows machine with no difficulty at all.  (By the way, I see no difference whatsoever between Freespire 2 and Linspire 6 except for the $49 I paid for the Linspire).  Anyway, I finally noticed that the "samba.conf" file says "workgroup=MSHOME" while my workgroup on the XP machines is "WORKGROUP".  Changed to "workgroup=WORKGROUP".  Viola!!  The windows machine sees the shared folders on the UBUNTU machine! Given enough time, even old geezers can figure it out.[/SIZE]](*,)
