### Post by mikewhatever on 2007-07-08
> **avynth said:**
> will all the stuff i put on ubuntu and do while i'm in ubuntu
I'm not sure what this means.
Running from a live cd is good for evaluating the OS, getting familiar with it and checking hardware compatibilities. You can't enable restricted drivers, because you'd need to reboot, and then all you have installed is gone. You can  use USB devices to save files, or get access to the HD, but the latter is not enabled by default, which makes sense, since Ubuntu 'does not want to touch' the hd with the files and the current OS.
Of cause you'll get better functionality and flexibility having installed it properly, but, if you do decide to, make sure to read the above posted guide several times, backup everything you can't afford to loose.
