### Post by aysiu on 2007-07-30
If you're really that determined to get rid of Ubuntu, this link should help:
[ HowTo: Remove Ubuntu (& Restore Windows)](http://ubuntuforums.org/showthread.php?t=508927)

The solution to your problems may be very simple, though, if you're using Ubuntu on an x86 computer (not AMD64 or PowerPC): Go to System > Administration > Synaptic Package Manager. Search for the phrase *ubuntu-restricted-extras*. Right-click the package and mark it for installation. Then click *Apply*. Java, MP3 playback, and a whole bunch of other proprietary codecs will be installed and ready for use.
