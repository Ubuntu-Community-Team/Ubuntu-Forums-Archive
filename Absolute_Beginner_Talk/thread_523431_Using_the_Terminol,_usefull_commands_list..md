### Post by macogw on 2007-08-11
man is the most useful command.   It shows you a manual page for whatever command you want to know about.  try it.  type "man apt-get" and it'll tell you what all can go after "apt-get..." to do different things.  there's even a "man man" (though "man woman" requires the funny manpages package).  check out "man man" because there's even a way to search the manpages to find out which command does what you want.  you use "man -k <searchterm>"

### Post by starcraft.man on 2007-08-11
Uh, Jake... I already pointed you to a place to read up on almost all parts of Aptitude/apt-get. Please read my blue guide (in my signature) this time (front to back) and you'll find it answers a lot of questions about ways to modify and install programs (especially with Aptitude in the terminal, I devote a whole section to it).

As for general terminal help, see the following:
[LinuxCommand]("http://www.linuxcommand.org/"): Start with "Learning the Shell" section and then read the other sections like writing scripts.
[Wiki docs]("http://doc.gwos.org/index.php/Main_Page"): Good wiki site, good lot of general things and I'm sure also terminal section.

I'm sure there are others people will bring up but those first two and the docs site should show you a lot.

There is also 

```
man command
```

where you replace command with the name of the command. This will show you a manual page for the command, not always the easiest to understand but mostly complete.

All the above are good posts and i don't disagree one pound or kilo. So you want a place to start that is not too complicated. 

sudo   A command that let's you be root for one instance
sudo su  turns you into root and only do this if you know what you are doing
cd  Change to another directory like cd /home/this/that
mkdir create a new directory like mkdir /home/new/debian/maybeishouldgobacktowindoze
mv move a file to another place (another dir)
cp the same as mv but it leaves a copy in the old dir
ls shows the contents of the current dir

That should get you started, oh yeah also 'file' that will show you what type of file it is . And use exit to get outta there. Don't stay in root. Your not that good yet. Someday, but not today. :)
Also 'clear' to clear the screen.
