### Post by Jagot on 2006-03-22
To be honest it really depends on what you want to use it for.

If you haven't used an Ubuntu installation before then I'd go ahead and install Breezy now.  Until June is a long time to wait.  

I know I was going to wait for Dapper (when the release date was still in April) but the urge got me in February so I went ahead with it then and have no regrets.  It will be perfectly easy to perform a clean install over Breezy when Dapper is released. and if you plan on storing documents on your Ubuntu installation (I just use mine for experimentation) then you can create a separate /home partition.
