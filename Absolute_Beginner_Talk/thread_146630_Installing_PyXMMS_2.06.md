### Post by bluevoodoo1 on 2006-03-18
I am not familiar with the PyXMMS plugin, but I can tell you how to install a .deb file. The following directions are for if the .deb file is in your home folder. 

Open up a terminal (applications > accessores > terminal) and type...

```
sudo dpkg -i thenameofyourfile.deb
```

(replacing 'thenameofyourfile' with the actual name)

that's all!
