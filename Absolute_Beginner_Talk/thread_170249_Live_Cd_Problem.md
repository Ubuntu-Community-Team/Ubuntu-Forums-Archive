### Post by browndog on 2006-05-04
The live CD is meant to boot as a fully functioning operating system.  If you go into the BIOS and make sure the computer can boot from the CD ROM drive, and you restart the computer with the live CD and it still won't boot into Ubuntu, I would try downloading a fresh ISO and burning the disc again.  Make sure the writing speed isn't too fast when you burn the image...I made the mistake of selecting too fast of a writing speed once and the disc came out wrong.
