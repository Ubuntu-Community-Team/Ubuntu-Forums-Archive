### Post by taurus on 2007-03-15
You need to make sure you have burnt the ISO image onto a CD as an image, not a data.  In other words, when you insert the CD into a Windows, you should see a bunch of files and directories.  Then, you need to check your BIOS to make sure that it boots from the CD first before the harddrive.  Then, boot from the CD and check out this guide.

### Post by nudnik on 2007-03-15
> **RolandDeschain said:**
> But I completely lack a OS of any kind. I bought a brand-new computer for cheap, and then found it had no OS. So no Windows, just the BIOS... I think...

No worries. The bios is accessable regardless at startup. Check the manual that came with the PC, or look at the start up screen as it whizzes by. There should be something on there like "Press F2 to enter BIOS Setup" On my machine it happens to be the Delete key, but manufacturers differ. 

If you received one of the SHIPIT CDs from Ubuntu you are good to go, otherwise, follow the wise moderator's instructions a post or two ago regarding ISOs.

If the boot order is correct, then it should boot up on its own thanks to the BIOS and you can install away.
