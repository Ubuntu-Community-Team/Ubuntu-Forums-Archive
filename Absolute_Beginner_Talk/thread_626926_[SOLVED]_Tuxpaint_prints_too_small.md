### Post by tjbonzo on 2007-12-30
Rather than having my 5 yr. old open Gimp to print, I made this simple script:

home: cat $HOME/bin/tuxprint
```
FILE=$(ls -rt $HOME/.tuxpaint/saved/*.png | tail -1)
pngtopnm $FILE | pnmtops | lpr

```
Then
home: cat $HOME/.tuxprintrc 
```
# Generated by tuxpaint-config version 0.0.7
windowsize=1280x1024
printdelay=10
printcommand=/home/*user*/bin/tuxprint
```

Now my kids just have to Save, then Print and it prints full size.

You will need the netpbm package for the pngtopnm program.
