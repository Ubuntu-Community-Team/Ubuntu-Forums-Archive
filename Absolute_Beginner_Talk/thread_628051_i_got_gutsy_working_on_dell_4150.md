### Post by scripteaze on 2007-11-30
Well, i tried to install gutsy desktop on my older laptop the dell 4150. Got a bunch of errors and it just wouldnt install..


What i did was this.

Install server edition, let it install only the base, then only select ssh  to be installed. When its complete, reboot as told..ssh into laptop or local do :

sudo apt-get install ubuntu-desktop

now about 407MB will be downloaded and installed onto the drive, taking anywhere from 30-45 minutes, when finished, it might hang, mine did..but reboot after about 5 minutes if you think its hanging..it should boot right up to the desktop..

i havent had anyissues with it as of yet and im running 3 ubuntu boxes in my house, so..just wanted to let someone know about it, maybe it could help someone out..later
