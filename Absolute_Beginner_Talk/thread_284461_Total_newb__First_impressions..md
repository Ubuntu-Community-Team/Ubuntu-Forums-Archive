### Post by thunderstorm on 2006-10-25
I've used Windows since I got my computer 10 years ago.  I'd tried Linux a few years back, but could never get things working right and gave up... but it's been something I've wanted to get into for a long time.

Not too long ago, I upgraded my computer.  I had to call Microsoft to get new authorization to use my operating system that I'd bought and paid for.  At that point, I knew I wanted something else.

So a few days ago, I looked around online and found Ubuntu.  It seemed to be somewhat newbie friendly, but with a learning curve you'd expect with any new OS.  After a few failed installs, I got it running.  (I had to use the alternate iso for it to work)

Now I've only had this since last night, but in that time I've done a lot of reading, and I've got most of what I could do in XP working on Ubuntu... and some of it works better than it did in XP.  I'm not having my system bog down when running apps, and I've tested it running way more than I could run in Windows, and that's nice.  Now that I'm getting more familiar with the interface, I'm finding it to be better organized than Windows as well, not to mention more flexible.

I am having a few issues though.  When I go to Screensavers and click through the previews, sometimes the system hangs.  That's the ONLY thing that's caused a system lock up so far.  I couldn't get Flash to install last night, but I haven't looked for any answers yet.  Other than that, no problems.

I don't see a downside to this.  So far, I'm very happy I switched, and more happy that I didn't have any major problems with upgrading.  And these forums have been a MAJOR help in getting my questions answered... so to you guys, thank you!
