### Post by Madpilot on 2006-09-24
I use the multiple workspaces (virtual desktops) all the time. 

In fact, I have 8 of them.

I run my browser in the first, XChat & Gaim in 2, File manager in 3, and then whatever else I'm using in the rest - usually one app per desktop.

Some applications like the Gimp or Gaim that spawn lots of windows become much, much more usable when they're given a desktop to themselves.

I do some graphics & DTP stuff, and when I'm doing DTP (desktop publishing) I'll be using those 8 desktops as follows: browser, XChat, file manager, Gimp, Inkscape, OpenOffice, Scribus - and still have two spare desktops for whatever else I need to start.

Virtual desktops rock, once you get used to them!
