### Post by petit.padavoine on 2006-11-17
You could install it thru Automatix but I would directly download and install the latest version as a .rpm package on the RealPlayer site, and convert it to .deb with Alien.
Alternatively, download the self-extracting install binary from the Real Website. The full path **should** end up being /usr/bin/realplay.
