### Post by xmastree on 2006-01-05
Welcome to ubuntu, and to the forums. :-)

The best thing you can do is to read [the guide]("http://www.ubuntuguide.org") first. whilst not explaining the ins and outs of linux, it will help you to learn how to install stuff, access your windows drives if you're dual booting, install codecs to play sound and video files etc.

For everything else, just remember to forget what windows has taught you... :rolleyes:

One big difference is the way programs are installed in linux. There's no "program files" directory, with each program having its own directory. Instead, most of them are split up such that the executable files (the .exe in win) are all in one place, /usr/bin which means that any program can be launched from a command line by just typing its name.

### Post by mcduck on 2006-01-05
for slow internet, you most likely need to disable ipv6 dns.

Try this: type 'about:config' in firefox address bar, then type  'ipv6' in the filter field and you'll get a line like 'network.disable.ipV6.dns' or something like that. right-click that line and select 'toggle' to set it's value to 'true'. Restart FF and it might be a lot faster now. (sorry about not-so-precise instructions, I'm at work now and forced to use WinXP and IE :()

Then install fasterfox extension, it will do some more tweaks to speed your browser.

If that helps, you can also disable ipv6 completely. Search the forum for instructions how to do that.
