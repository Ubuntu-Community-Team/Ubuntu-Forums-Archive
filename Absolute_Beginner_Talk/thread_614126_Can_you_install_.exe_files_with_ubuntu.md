### Post by Paulmd on 2007-11-16

If by MP3 player you mean:

Zune: not possible at the present time. Undoubtletly, many people are working on it. I have heard a rumor thay Microsoft was offering a substantial prize to anyone who succedded in creating Linux based software that would work with the zune. No takers so far.


iPod: [http://www.gtkpod.org/about.html](http://www.gtkpod.org/about.html)
 gtkpod is available  for installing in ubuntu

#sudo apt-get-install gtkpod

Other MP3 players: like many digital cameras, many work pretty much like an external hard disk. You can try plugging it in and see what happens, you may be pleasantly suprised.

Don't expect any music store type software to work, pretty much all you can do is read and write to the device.
