### Post by TwistableLime on 2008-04-20
If you are looking to install Windows XP, you need to tell the computer to boot from the CD-ROM drive. To do this you will need to restart your computer and wait for the splash screen to come up and hit one of the F buttons (its usually F2 or F8 ) to enter the BIOS. From there you will have to navigate using the arrows keys to change the boot order of your devices. You will want the CD-ROM to boot first.

### Post by gpilkay on 2008-04-20
To be honest about it, XP has ALWAYS been a pain to install, and I'm one of the black sheep on this forum who actually LIKES XP.  

First, make sure you have the XP disk in the CD-Rom drive, and the registration key on-hand.  Turn the machine off entirely.  During boot, it will give you boot options or something, usually, as one of the poster's said, by pressing one of the function keys to take you into the BIOS.  You want to boot from the CD.  From there it's just following the installer, though you should know that XP does NOT have a Live-CD mode like most Linux distros do, as such, you will have to just install before you know if anythings working.

Don't forget possible driver issues, which XP is infamous for.

What sort of computer is it (processor/memory wise)?  You could also look it up on Dell's site to see if there are any special special problems.

### Post by gpilkay on 2008-04-21
Also, nutty as it sounds, when I re-load XP on a Dell, I like to keep the Ubuntu CD at hand.  Reason being, even if you don't want to install it, it can boot into a fully functional system without actually installing, so if something botches, you can go in and save off your files to a USB drive and try again.  Dell XP installs are a pain in the neck.

While not really a listed feature, it's extremely handy.  I do recommend using 7.04 or 7.10 for that, though, as they tend to read NFTS better.  It's no system rescue CD but it does the job, and you can play with Linux with no worries about the host computer.  it's what I did for almost two years before I finally decided to install it.
