### Post by Delkster on 2006-11-21
> **gustav213 said:**
> hey this is a complete newbie questions, but is there somekind of like defragmenter, or other sytem tools like the ones for windows in xubuntu? do i even need them

You don't really need to defragment your drives. As long as you don't fill them completely (about 20% free space is good), the fragmentation shouldn't cause big trouble.

If you want to remove unneeded files to free up disk space, you could clean up the package manager cache (in Synaptic, open Settings > Preferences, then open the Files tab where you'll find a button titled "Delete Cached Package Files"). A disk usage analyser called Baobab can help you find out which of your own files and folders take up most space. It can be found in the gnome-utils package in Synaptic, and by searching for "baobab" in Add/Remove Applications (I'm not sure if there's such a thing in XUbuntu but at least there is in Ubuntu).
