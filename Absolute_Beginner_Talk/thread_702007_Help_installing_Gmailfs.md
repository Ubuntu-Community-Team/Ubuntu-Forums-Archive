### Post by jan quark on 2008-02-20
I have not used gmailfs.

But it can be found in the synaptic repositories.

Go to:

System > Administration > Synaptic Packet Manager,

type gmailfs into the search field,

check the box mark for installation,

click on apply.


Kernel module:
