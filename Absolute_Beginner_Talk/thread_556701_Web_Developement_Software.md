### Post by altonbr on 2007-09-21
You can search through this thread and see real programmer's discussions: [http://ubuntuforums.org/showthread.php?t=6762](http://ubuntuforums.org/showthread.php?t=6762)

Or you can see ubuntuguide.org's list of alternatives: [http://ubuntuguide.org/wiki/Alternatives](http://ubuntuguide.org/wiki/Alternatives)

I personally use gEdit (the text editor that comes with Ubuntu), but it's not an IDE. Now, in Gusty (the next version of Ubuntu), it can read php variables inside strings (which I found was the only feature missing). Screenshots are here: [http://ubuntuforums.org/showpost.php?p=3269797&postcount=1](http://ubuntuforums.org/showpost.php?p=3269797&postcount=1)

Also, if you run:
```
sudo aptitude install wine
```
you will be able to run Windows programs such as Notepad++: [http://notepad-plus.sourceforge.net/uk/site.htm](http://notepad-plus.sourceforge.net/uk/site.htm).

Lastly, Aptana (which is based on Eclipse), is slowly becoming a very popular, fully functional IDE for Web Development: [http://www.aptana.com/](http://www.aptana.com/)
