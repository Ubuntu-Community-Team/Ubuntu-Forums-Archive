### Post by Azakus on 2006-12-25
Just pop it in. You can then format it with the ultra-easy program Gnome Partition Editor, or Gparted. When you install it, it will usually have the name sda for SATA drive. Just install gparted and format it to whatever you'd like (I believe it also sets up mount points in fstab).
