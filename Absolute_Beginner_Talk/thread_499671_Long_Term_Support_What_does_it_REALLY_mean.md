### Post by p_quarles on 2007-07-12
LTS means that Canonical will continue to add security updates and bugfixes to the repositories for the designated amount of time. It's great for servers (which usually don't need the latest web browser or music player), and for those with slower internet connections. If you want the latest graphical apps, though, you should definitely upgrade the system.
