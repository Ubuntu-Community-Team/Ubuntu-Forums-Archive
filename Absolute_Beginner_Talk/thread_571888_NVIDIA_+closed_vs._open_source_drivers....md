### Post by Hospadar on 2007-10-09
when you install through the restricted driver manager, you are installing the closed-source nvidia drivers.  (these are the "nvidia-glx", nvidia-glx-new" etc packages)

I would highly reccomend using these drivers, because while yes, it's a bummer they are closed, the performace and functionality increase is huuugeee.  Even for loading websites with lots of graphics it's a noticable difference.  and you will not be able to run anything 3-d at all without the proprietary drivers, or even an accelerated window manager like compiz fusion or beryl.

So I suppose if you don't need the extra performance and you want to be all open source, by all means stick with the open source drivers.  but I would highly reccomend the nvidia drivers.
