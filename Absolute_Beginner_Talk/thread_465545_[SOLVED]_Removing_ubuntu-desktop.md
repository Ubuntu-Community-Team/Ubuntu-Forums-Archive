### Post by testube_babies on 2007-06-05
Removing ubuntu-desktop won't pose any immediate problems for updates, etc.

However, it is possible errors can occur when doing a dist-upgrade.  If I were in your position, I would have no problem removing the ubuntu-desktop package and then reinstalling it right before a dist-upgrade.
