### Post by bubba_169 on 2008-03-26
Do you mean the bootsplash? You can theme this using startup manager and usplash themes from various internet sites. To install startup manager open a terminal and type:

```
 sudo apt-get install startupmanager
```

then you'll be able to find it in System->Administration->Startup-Manager
