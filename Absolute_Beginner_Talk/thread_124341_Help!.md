Answers:  
(1) The ISO image you downloaded is just that, an image of a CD.  As such, you need to burn it to a CD as an image.  Most CD burning programs for Windows, Linux, OsX, etc. have this capability.  If after burning it and you look at a directory listing of the CD and see only the one ISO image file, then it's not burned correctly.  The CD should have lots of files on it.

(2) It is true that many recommend installing XP before any Linux.  This is because of the way that XP wants to take over the boot loader or process and wipes out the Linux Boot Loader.  It can be recovered, it's just a pain for a newbie to do.  Usually, most PCs have Windows already installed, so it's not an issue.

(3) The simple steps of installing Ubuntu is to boot off of the CD and follow the prompts.  However, there is probably some prep work you will have to do (particularly if there is another operating system on the hard drive that you want to keep).  Assuming you want to keep Windows and only have one hard drive, the first thing you will need to do is shrink the size of your Windows partition to free up space to install Ubuntu.  After that, you could basically just boot from the CD and follow the prompts.  If you don't care about your Windows installation, then just boot from the CD and when it eventually gets around to asking you if you want to use the whole disk, tell it yes.  For more specific and detailed info, I would recommend checking out the installation forum.

BTW, you've made a good choice with Ubuntu!
