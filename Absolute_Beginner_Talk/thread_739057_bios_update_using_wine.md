### Post by ramjet_1953 on 2008-03-30
Firstly, why do you need to update your BIOS?

The reason I ask this is because if your system is working fine and there are no new features that are required, updating a BIOS is like fixing something that isn't broken.

There are very real risks in flashing a BIOS.

If something goes wrong (and it does sometimes) you are left with a completely dead motherboard that usually requires a manufacturer repair.

Do not upgrade just because it is the "latest and greatest".

If you have overheating problems, it is often related to frequency scaling problems, where the CPU is running flat out all of the time.

Another thing can be where the cooling fan on the CPU is not operating properly, or the cooling duct to the CPU is clogged with dust. This is common with laptops.

I would do a search on this forum for any posts about over heating and frequency scaling.
