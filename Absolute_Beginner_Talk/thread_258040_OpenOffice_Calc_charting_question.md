### Post by spectrum42 on 2006-09-15
Are you selecting the cells before or after you click the chart button?  I just tried it and if I selected the cells first and then clicked the chart icon, it worked. I was able to select the multiple cells using the control key that way.

In case that's what you're doing and your computer isn't accepting it for some reason, maybe you could manually type in the range when formatting the chart.

As an example, when I selected the A1-E1 and A5-E5 cells, the range box shows it this way:

$Sheet1.$A$1:$E$1;$Sheet1.$A$5:$E$5
