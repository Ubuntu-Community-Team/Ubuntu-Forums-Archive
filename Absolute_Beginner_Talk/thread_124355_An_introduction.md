### Post by steve.horsley on 2006-02-01
Ubuntu and Kubuntu differ only in the desktop they install by default. In ubuntu, you can apt-get install kubutu-desktop, or in kubuntu you can apt-get install ubutu-desktop, and I think you end up with identical systems then, with a chioce of gnome or KDE at the login screen. Similarly, install xubuntu-desktop for XFCE desktop option. Brilliant!

I think the real thing here is that there is only enough space for one desktop on the installer CD. All 3 are in the online repositories.

Since you're unsure about the desktop you want, I suggest that you install all 3 desktops and try them all.
