### Post by ziroday on 2008-03-17
Hi,

To disable the "jelly" effect, also known as wobbly windows is a setting within compiz. You can disable wobbly windows by installing the Advanced Desktop Effects Setting ```
sudo apt-get install compizconfig-settings-manager
``` Once its installed you can access it in System > Preferences > Advanced Desktop Effects Settings.

To switch between virtual terminals you press CTRL+ALT+F(1,2,3,4,5,6) . CTRL+ALT+F7 takes you back to normal gnome desktop
