### Post by bodhi.zazen on 2007-06-29
You do know how to copy and paste those commands ? Saves on typing ;)

Open a terminal.

Highlight the command (with the mouse)

navigate to terminal ...

Push the mouse wheel down (works with almost all apps).

Also , check out tab completion.

Start to type a directory of file name, hit the tab key, and the command line will auto complete.

So rather then typing /etc/fstab as e-t-c-/-f-s-t-a-b 

Try /et<tab>fs<tab> :twisted:
