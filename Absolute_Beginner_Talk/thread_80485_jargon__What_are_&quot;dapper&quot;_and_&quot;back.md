### Post by heimo on 2005-10-22
Dapper refers to the next Ubuntu release, Dapper Drake 6.04 (April 2006), current one is Breezy Badger (5.10) and previous ones include Hoary Hedgehog (5.04) and Warty Warthog (4.10).


[Backports]("https://wiki.ubuntu.com/UbuntuBackports") is a project of [backporting]("http://en.wikipedia.org/wiki/Backporting") software to the older versions of Ubuntu.
