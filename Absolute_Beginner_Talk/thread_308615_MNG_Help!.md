### Post by qyot27 on 2006-11-28
If I remember correctly, the process of making one in the GIMP is exactly the same as making an animated GIF save for choosing MNG as the format from the save dialog (I've only gone through this process once, though, so I'm not completely sure).

Unfortunately, I've not run across too many things capable of handling them.  Konqueror can view them (as can Quicktime, which I seem to recall could potentially be used to get it to display within Firefox, but that's only effective on Windows - and presumably Mac), but aside from that it's mainly specialized tools meant to deal with the format, not general purpose software.
