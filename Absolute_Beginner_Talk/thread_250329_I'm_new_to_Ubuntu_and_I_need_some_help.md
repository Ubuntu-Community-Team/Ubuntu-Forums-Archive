### Post by aysiu on 2006-09-03
Go to System > Preferences > Themes and drag and drop the .tar.gz to the Theme Manager window.

The theme **must** be zipped. Do not double-click the .tar.gz and extract it. Do not run the *tar* command on it.

### Post by H.E. Pennypacker on 2006-09-03
As others may have said, first download the file from Gnome-Look.org.  Download it to your desktop.  Second, open up the Themes window (System > Preferences > Themes).  While it is still open, drag the file you just downloaded onto the Themes window.

It will take care of the rest.  It's really a matter of dragging and dropping.  Do not modify the file you downloaded in any way.  It should stay zipped.  I don't know why aysiu said it must be unzipped, because Themes requires that the files be in .tar.gz. format (which is a way to compress files).

> **aysiu said:**
> Go to System > Preferences > Themes and drag and drop the unzipped .tar.gz to the Theme Manager window.

The theme **must** be unzipped. Do not double-click the .tar.gz and extract it. Do not run the *tar* command on it.

If he's new, why would he run any commands?  The presumption should be that he's using GUI.
