### Post by Zuph on 2007-04-26
These forums, the wiki, and the main pages are the best places possible to learn about Ubuntu.  The way I ended up learning was by fixing my own problems, then starting to fix the problems of others.

As I've used Linux more and more, I've found myself more capable of fixing problems.  If you stick with it, you will figure it out.  Linux is just a different paradigm.  You have to learn new terminology and a new way of operating.  If you stick with it, you will learn.

You should be fine to run Beryl, though you might not be able to crank up all the effects.  I've run Beryl on a 1.1ghz tablet PC with crappy Intel graphics.  The easiest way is to just go to synaptic and search for beryl or do a "sudo apt-get install beryl" from the command line.  I think this installs the beryl-manager application.  At this point, just start beryl-manager and there will be an icon in the system tray and you can manager everything about it.  There are better guides than this in the forums and the wiki.
