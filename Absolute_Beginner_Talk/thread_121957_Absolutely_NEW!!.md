### Post by az on 2006-01-26
[QUOTE=MaXiMus](sorry for offending those who dislike Gates and his work) [/QUOTE]

This is a misconception.  People generally do not use linux because they hate another OS.  The difference between Linux and Windows is about software as property.

Windows is a proprietary OS, which means that the software is property.  Linux is a free-libre OS which means that it belings to everybody.  People are entitled to sit on either side of the intellectual-propert fence.

People also use linux because it works well for them and is available at no cost.  Those two qualities also spring from the free-libre developmental model which make it easy to study and improve.  It is also not subject to all the malware that is out there which afftect other operating systems.

Go ahead and hate Microsoft if you wish.  It is just not a prerequisite to running linux.
