### Post by halitech on 2007-03-03
Can you ping the printer or get it to do any internal prints?

to ping 
```

System - Admin - Network Tools

```
select ping and enter the ip address of the printer. If you can't ping but going into the printer menu will print a config page or menu map, then the issue is with your network.

Maybe by now you've gotten this working but if not, I hope I've helped.
