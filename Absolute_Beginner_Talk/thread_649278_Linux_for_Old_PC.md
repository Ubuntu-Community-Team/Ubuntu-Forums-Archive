### Post by angelsguitar on 2007-12-24
I have an old Compaq laptop Pentium II 266 Mhz 192MB RAM.  I tried Ubuntu (I don't why I did that?!), Xubuntu, Fluxbuntu, Puppy...and finally I decided to stick with Puppy Linux; the other distros were still a little heavy on my laptop.
Puppy Linux DOES HAVE an utility to install it to the hard drive.  Just run the live cd, and look for "Puppy Universal Installer" on the "Setup" Menu.  You can even install it on a Pen Drive and boot Puppy from it, if your computer supports it.
The basic installation of Puppy is somewhat limmited, yes, but it has it's own package manager and you can install a lot more packages from there (like Firefox, OpenOffice, etc).  Besides, it has an optional  utility to use deb packages too.
Hope this helps!:)
