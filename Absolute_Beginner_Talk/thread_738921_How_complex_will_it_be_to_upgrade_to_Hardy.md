### Post by bumanie on 2008-03-29
As far as I know, it can be done via update manager
System -> Administration -> Update Manager or
via the following commands
> sudo apt-get update
> sudo apt-get upgrade
> sudo apt-get dist-upgrade
Many users do a new reinstall. It's up to you which one you try.
