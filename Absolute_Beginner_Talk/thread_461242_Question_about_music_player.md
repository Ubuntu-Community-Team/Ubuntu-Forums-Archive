### Post by fastpakr on 2007-06-06
That's interesting - I still use the command line every day in Windows, and most of the other support people I interact with do as well.

I'm sorry you're having to deal with an issue with the modem driver, but keep in mind that this is primarily because the manufacturers (so far) are not writing drivers on their own like they do for Windows - on this side of the fence, individuals have to write their own for the time being.  Linux doesn't include the MP3 codecs for a very good reason - they're not open licensed.  It DOES include support for open formats like Ogg out of the box, though.  Windows can include them because Microsoft can handle the licensing issues with part of the money you lay down for their software - for obvious reasons, that's not possible here.

As far as malware goes, the limited audience certainly has a little to do with it, but the primary issue is the fundamental design differences with the use of a privileged superuser account, etc...  Windows is moving in that direction to some extent, but there are nearly always ways around the fences they put up between user accounts, where it's a core component of Linux that simply can't be overridden without the proper privileges.
