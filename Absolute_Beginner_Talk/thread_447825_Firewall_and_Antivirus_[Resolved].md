### Post by RichGuk on 2007-05-18
Tbh, I've never bothered with a Virus scanner on Linux and don't bother with firewall (apart from fileserver) becuase I'm behind a hardware firewall.

IPTables, which is built into the kernel is all the firewall loving you'll need.. you can use applications like gaurddog (kde) or firestarter (gnome I think) to manage theme. Ubuntu by default should come with no open ports anyhow, until you start install applications anyhow.

ClamAV - is maybe a anti virus, although not to sure :)
