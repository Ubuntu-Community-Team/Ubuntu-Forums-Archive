this is from the link you provided
> Description:
A Screenlet that shows a working clock wallpaper , you can get wallpaper clocks from [http://www.vladstudio.com/wallpaperclock/](http://www.vladstudio.com/wallpaperclock/) , the default version only include one wallpaper , however you can download them and extract them to the wallpaper folder inside the screenlet folder , just download the file and extract its content there

**Install**
Install as usual by extracting the contents of the screenlet's archive to the directory "$HOME/.screenlets".
So open your home folder, hit <Ctrl-h> to show hidden files then extract to there as posted above. I'm sorry I can't be more specific but I don't use screenlets. But there way should work. :)
