### Post by ryP on 2007-06-29
I noticed this also, I was using Totem, it was really bothering me.  I think I figured out how to fix it, but not exactly what causes it.  After this happened to me once again, I opened the Totem player separately (as opposed to having it run through a browser) and I went into Edit -> Preferences -> Display.  On the Display tab, I noticed that the Saturation bar was all the way to the left.  I hit the "Reset to Defaults" button and it set all the sliders back to the middle.  I then played another video and it was back in color.  After that, I found that manually sliding that Saturation bar all the way to the left makes the video black and white, while putting it in the middle gives you perfect color.  So that is how you can probably fix it if it happens to you.

As far as what is causing it, I'm guessing some movies files or websites (I am really bs'ing right now) must somehow set that saturation bar all the way to the left.  But manually putting it back should fix you up until it happens again.  Let me know if this works for you.  

ryP
