### Post by jkndrkn on 2005-05-22
Ubuntu can run on a broad range of resources, from older boxes with limited ram, all the way up to x86-64 processors with gigs of memory.

Hardware support is good, but not perfect. Before you start ordering parts, I suggest you look up all soundcards, sata drives, raid controllers, graphics cards, usb devices, etc.

Ensure that they have reliable performance in linux before you buy.
One can spend many hours trying to get a stubborn bit of hardware working
I suggest googling the devices, and i suggest searching for them in this forum as well to see if other users have had trouble getting them to work.

I have some experience with audio editing in windows, and have started to dabble in linux audio.
So far, i've found less glitching and more consistent performance from linux
however, i've yet to find replacements for all of my favorite software.

You'll find that configuring sound in linux can be a bit challenging, but the end results are pleasing and consistent.

The following website offers a good list of available software.

[http://linux-sound.org/](http://linux-sound.org/)
