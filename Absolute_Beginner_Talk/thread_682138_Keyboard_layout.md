### Post by fmbyv on 2008-01-31
I'm a native Spanish speaker using a US keyboard and do not recommend to configure your keyboard with a Spanish layout. It will drive you nuts trying to find out where each symbol is. You can easily type Spanish (French, Portuguese, German, etc.) symbols using the [US-International]("http://en.wikipedia.org/wiki/Keyboard_layout#US-International") layout, as shown [here]("http://www.barrancos.us/images/teclado_lnx.png"). You will find it so useful that you won't have to switch back and forth between different keyboard layouts.
