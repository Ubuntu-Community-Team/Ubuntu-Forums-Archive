i think kaffeine is a kde package. i'm gonna get slamed by the command line junkies for suggesting this. but go to [www.getautomatix.com](www.getautomatix.com) and install automatix (look for the install tab at the top, then select easy install for version 6.06) once it's installed you'll have no problems in getting the needed codecs, etc. to play video. don't let what you may have read scare you away from automatix. i use in on both of my ubuntu boxes, with no problems.
