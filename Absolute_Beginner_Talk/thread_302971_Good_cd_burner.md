### Post by jaboua on 2006-11-19
Screenshot of what app?

Gnome itself should be able to burn ISO's with no extra addons if I'm not mistaking (right-click the .iso -> "burn image" iirc, or if you need to create things from scratch, Places -> CD/DVD Creator). Apps like gnomebaker, brasero and graveman have some more options and are seperate apps (instead of being a part of the filemanager).

For lots of advanced configuration, try K3B - but that will require KDE (kubuntu) to be installed.

Nero4linux exists as well, but I read somewhere that you need a license for the windows version in order to get it, and I can't say how advanced it is as I've never tried it myself.
