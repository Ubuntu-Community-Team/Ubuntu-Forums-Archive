### Post by proalan on 2007-12-19
Flash 8, Dreamweaver 8 and fireworks 8 should install without any problems with the latest version of wine.

make sure you have wine installed via

sudo apt-get install wine

or if it is already installed do an update via

sudo apt-get update

you should be able to run the setup.exe as you would in windows
