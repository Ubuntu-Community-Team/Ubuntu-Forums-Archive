### Post by wpshooter on 2006-12-06
You would be much better off, if you would just go to the Kubuntu site for your country location and download and burn the CD yourself.  Make sure you burn the CD at 4X or slower.

After doing so, make sure you check the integrity of the CD by running the verification test found on the initial CD boot screen menu.
