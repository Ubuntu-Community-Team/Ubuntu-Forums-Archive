### Post by Ub1476 on 2008-02-03
Hi, you made a slightly confusing post there!

> Stated simply, I am a WINXP user and found an application that I need that runs faster, better with "Ubuntu 6.01 deb".

Ubuntu 6.01 is a REALLY old version of Ubuntu. It was released in 2006, January. There are two other version of Ubuntu which was released in 2006. 6.06, which is currently the LTS (long term support), and 6.10. This version is codenamed "Edgy Eft" and 6.06 is codenamed Dapper Drake. 

You should also know that the next LTS will be released in 2008 April, called Ubuntu 8.04 Hardy Heron.

Also, the "deb" is a execute file like Win "exe", so that's what you download when installing packages, not Operative Systems.

I saw you said you got Ubuntu 6.10 on a CD longer down in the post though, but what's with this key? As far as I know, there's no need for any authorization key (I supposed that's what you meant?), neither for installing on CD, USB etc..

So, you sort of name Ubuntu all the way here, install Ubuntu on this install Ubuntu on that. BUT, later down you say you need a very minimalistic OS, like Puppy?

Yes, Ubuntus goal is to be an OS for normal people, who uses word, mail, internet, games, msn etc. It's still a very fast OS compared to Windows, but it has most documentation, support and so on. I suggest going with Ubuntu, at least if you aren't experienced. 

Installing Ubuntu is easy. Just boot up the Live CD and click on the "Install" icon. It should give you a good walkthrough for setting up your Ubuntu. '

Here's some documentation for [SSH VPN]("https://help.ubuntu.com/community/SSH_VPN?highlight=(SSH)|(VPN)")

And if you plan on using Linux on an USB: [Pendrivelinux.com]("http://www.pendrivelinux.com/")
