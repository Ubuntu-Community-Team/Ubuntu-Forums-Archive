### Post by Bazirker on 2007-12-09
OK here's your stupid question for the day...

How do you get to the drop down menus of programs (file, edit, view, help, etc) using only the keyboard? ( I want to use this with Virtualbox so I can make it run full screen but still use the GUI to enable USB devices and stuff like that.)

---

### Post by jken146 on 2007-12-09
In most programs, File, Edit, View, etc. appear with an underlined letter (as they do in Windows).   You press Alt+<underlined letter> to get the menu.  For example, in Firefox I can see _F_ile, so I can press Alt+F to get the File menu.
