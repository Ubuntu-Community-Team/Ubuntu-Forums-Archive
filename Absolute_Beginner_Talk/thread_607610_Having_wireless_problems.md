### Post by bluespider on 2007-11-09
First off I am a complete newbie to linux so keep that in mind. I was having a lot of  problems getting my wireless internet connection working both in gutsy 7.10 64 bit and the 32 bit,Basically the problem was that while my wireless worked fine from the live cd , after I actually installed Gutsy It did not work at all,  after many days of trying a lot of  different things to get them working, this worked for me, unfortunatly to try this you have to completly uninstall Ubuntu from the drive you installed it to,and then format that drive to completly remove Ubuntu. I would suggest you try  this solution as a last resort simply because hey I dont even know why it works..but it does work every time I install.Here is what I do after I complete the format of the install drive:

Reboot your system with the Ubuntu 7.10 live cd in the cd drive.On the start up menu select the "start or install Ubuntu" option

After the live cd boots up the ubuntu kernel setup your wireless (if you cant get your wireless to work from the live cd then dont even try this technique)

Now if you got your wireless to work select the desk top install Icon to install Ubuntu,
at the end of the  install it will ask you to restart the system, select yes to restart.

Now here is where the live cd would normally eject itself, now you dont want to do that you want to restart with the live cd still in the cd drive (you might have to shut off the computer and restart with the live cd in the cd drive.

Ok now the live cd menu will come up, choose the boot from drive first selection,
after Ubuntu boots you should be able (if the trick worked) to get the wireless working,
Set it up and connect to the internet,just to check it out. Then shut down ubuntu and remove the live cd from the drive, and restart your system, and your wireless should work.(hopefully) If it doesnt ...uhh sorry... like I said this is a last resort kinda thing, Im only putting it up because it worked for me and if it says someone else from having to go through what I did Im happy...
