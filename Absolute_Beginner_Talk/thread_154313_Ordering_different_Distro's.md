### Post by catlett on 2006-04-03
You're in the right place now. Ubuntu and Kubuntu are at the top for ease of use and good for a desktop. Another easy one is Mandriva. They have a new Live CD out that can install from the Live session. Suse linux is another big one. This is the version from Novell. They are big in the server market but Suse 10 is an easy to use desktop. The other big server company is Red Hat. They are supposed to be the biggest. Their Red Hat Linux is a pay for use OS. But they have a free release Fedora. Fedora Core 5 was just released and is supposed to be very good. I would start with those. Ubuntu/Kubuntu, Suse, Mandriva and Fedora. They are big names in Linux and they have easy to use desktop OSs. There are literally dozens of Linux OSs so shop around. Not all are user friendly because the bread and butter of Linux is running servers. Have fun.
