### Post by virtuallinux on 2008-03-29
Only one thing about dual booting Vista: make sure you install Vista SP1 **before** you install Ubuntu.  I've read that the SP1 installer has to replace Vista's boot loader so that it can upgrade.  It has to do with bit locker technology, so if you're not using Vista Ultimate or Vista Business, it won't affect you.  You can repair the Ubuntu boot loader afterwards, but it's a little complicated.
