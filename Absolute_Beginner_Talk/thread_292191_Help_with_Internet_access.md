### Post by monkeydust on 2006-11-03
> **Desy said:**
> I presume that if I follow these steps, then use the same modem on my win xp drive everything should be ok

Should be. Practically everything you do under Linux affects just the Linux partitions you set up. Your Windows partition (unless you do something really silly) will be untouched.

In fact, by default if your Windows partition is NTFS, it'll be Read Only anyway, meaning that you cannot write to it.

I've had a similar issue with the Sagem Fast 800 USB modem with Tiscali Broadband. I followed similar steps online and it works a treat, both under Linux and Windows, although I'm sure it's slightly faster under Ubuntu...

All the best,

David
