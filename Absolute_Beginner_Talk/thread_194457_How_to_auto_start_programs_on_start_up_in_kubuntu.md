### Post by mrvgarg on 2006-06-11
Can someone please tell me how to automatically start programs (gdesklets, etc) when I start my computer? I am on kubuntu.

---

### Post by niviche on 2006-06-11
Put a link to the program you want to start in the \home\(your user name)\.kde\Autostart folder.

---

### Post by MartinG on 2006-06-11
Put a link to the relevant program in ~/.kde/Autostart (AKA /home/<your_name>/.kde/Autostart)

### Post by Jucato on 2006-06-11
Or just type it directly in your location bar (/home/username/.kde/Autostart)
Once you get to that folder, you can right click on any empty space, and choose Create New > Link to Location (url)  or Link to Application.

Btw, gdesklets is the GNOME version, while superkaramba is the KDE version
