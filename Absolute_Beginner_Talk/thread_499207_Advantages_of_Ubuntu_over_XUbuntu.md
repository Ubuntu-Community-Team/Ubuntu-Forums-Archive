### Post by Espreon on 2007-07-12
-GNOME (Ubuntu's DE)  is easier to use then XFCE (Xubuntu DE)
-More apps are developed for GNOME than XFCE (You can still use GNOME and KDE apps with Xubuntu)
-Wine and Wine-Doors have GNOME integration with the GNOME menu

But If you are conservative I recommend installing Enlightenment. It is a window manager that is light-weight and pretty. You install enlightenment after install Ubuntu or Xubuntu ( I reccommend Enlightenment with an Ubuntu install, since GTK and friends and other powerful apps will be installed.

Enlightenment will support icons and friends.

> **7Priest7 said:**

Was I mistaken?
It is. But software must be written for the 64 bit processor. One of the major advantages of 64 bit (not to bore you) is that it enables the system to utilize more than 4 GB of RAM. A 64 bit Processor can run both 32 bit and 64 bit OSes (infact, my dual core/64 bit system came with XP 32 bit). However a 32 bit system can only run a 36 bit OS (unless you want to go in to 16 bit, 8 bit, etc).
