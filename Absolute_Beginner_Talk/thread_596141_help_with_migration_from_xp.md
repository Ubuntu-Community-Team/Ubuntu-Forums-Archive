### Post by taurus on 2007-10-29
How much space do you need to Ubuntu depends entirely on what you plan to do with it.  It needs at least 2.5GB for Gnome so 5GB should be enough.  But if you plan to save a bunch of stuff on Ubuntu, then you need to increase the space.  So, if you can spare a 10GB, that should give you some room to grow.

And with Gutsy, you have the ability to read and write to ntfs filesystem using ntfs-3g driver.  Therefore, no need to copy stuff from your windows to Ubuntu if you need to access them.  Access them directly on window partitions.
