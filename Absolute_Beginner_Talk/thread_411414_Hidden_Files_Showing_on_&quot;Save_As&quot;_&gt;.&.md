### Post by skrimpy on 2007-04-16
I have my hidden files hidden in Nautilus.  However, when I go to "save as" a file into my home folder all of my hidden files are listed.  This has happened with both OpenOffice.org Writer and with the GIMP.  It's really annoying because it makes my folder list a mile long!  Is there an option somewhere to turn this off so I only see the directories I created for my home folder?  Thanks!

---

### Post by NeoLithium on 2007-04-16
When you have your mile long list; just right click with your mouse, and uncheck the "Show hidden files" box :)
