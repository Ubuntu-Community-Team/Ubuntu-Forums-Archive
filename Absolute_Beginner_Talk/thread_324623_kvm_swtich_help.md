### Post by tommcd on 2006-12-24
You should not install ubuntu while hooked up through the kvm switch. Ubuntu will have a hard time detecting your monitor that way. Hook up the PC directly to the monitor, install ubuntu, then put the kvm switch back in place. I don't know if this will fix the problem with the mouse moving around, etc. But you will have an easier time installing ubuntu this way.
