### Post by 56phil on 2006-12-02
Welcome to Ubuntu.

I may be barking up the wrong tree, but have you run the memory test from the install CD? Sometime intermittent problems, such as you describe, are traced back to faulty memory. Also, you may want to get more RAM. 256MB is a little tight. I have a 256MB system, but with a separate video card. When you run the memory test, let it run all night.

Note: If you ever decide to add a video card to your system, get one with a Nvidia chip. Nvidia supports Linux better than VIA or ATI.
