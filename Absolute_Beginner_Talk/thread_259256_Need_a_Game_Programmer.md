### Post by 3rdalbum on 2006-09-17
What part of the game are you actually making, if you're not programming it?

I suggest you learn some programming of whatever language and database management system you plan to be using (probably PHP and MySQL, considering this is a Linux forum), and get a simple beta that kinda works. Write up a document explaining exactly what will happen in the game, what features it will have, and why you think the public will like such a game. If applicable, explain how the game will make money.

Then distribute the document and a link to the beta around some communities, and put out a call for developers. Then you'll probably get some help.
