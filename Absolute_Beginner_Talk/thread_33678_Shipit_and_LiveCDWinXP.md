### Post by Arthemys on 2005-05-11
I know that atleast in Knoppix styled LiveCDs, any hard drives will be automatically mounted for read only access, and that you will need to unmount and remount with the proper switches to get read/write access.

Goal of a live CD, in my opinion is...

-Try out the distro
-Diagnostics / Security / Utilities and so forth
-NOT touch the host box in any manner that could change something
