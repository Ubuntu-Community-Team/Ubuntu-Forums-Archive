### Post by oracle2025 on 2007-03-31
Running Open Movie Editor in a VM would not work very well, because it uses OpenGL hardware-accelerated Graphics. So installing Linux from scratch should work better. But if you're using windows and don't want to remove it from disk, you might as well try a video editor for windows, like Avid FreeDV.

[http://www.avid.com/products/freedv/index.asp](http://www.avid.com/products/freedv/index.asp)
