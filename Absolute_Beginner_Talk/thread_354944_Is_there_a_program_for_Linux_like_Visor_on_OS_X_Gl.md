### Post by JayTee on 2007-02-06
If you are running Gnome you can assign a hotkey combination for launching terminal from the System | Preferences | Keyboard Shortcuts menu applet. It's fairly easy to setup. Click on Run a terminal in the list and then enter the keys you want to use to invoke the terminal and then click OK.
