### Post by Sutekh on 2006-04-09
There's no reason why you can't use the LiveCD to install things.  The Install CD doesn't contain the codecs either, you still have to install them.  

So long as there is room on your RAM (the LiveCD runs entirely from it) you can use the LiveCD to do pretty much anything an installation can.


All you need to do is enable the universe repository and then install the required package to get mp3's to play using your LiveCD

 - To enable the universe repository, just edit your apt-get sources.list and uncomment (remove the #) the lines containing **universe**
```
sudo nano /etc/apt/sources.list
```
 - Will open the nano editor and the apt-get sources.list

 - Look for these lines and uncomment them (this is a breezy example, for dapper the bolded text will be different)
```
# deb http://archive.ubuntu.com/ubuntu **breezy** universe
# deb-src http://archive.ubuntu.com/ubuntu **breezy** universe
```
 - Save the file (Ctrl+O) and exit (Ctrl+X)

 - For Breezy you need the [**gstreamer0.8-mad**](http://packages.ubuntu.com/breezy/libs/gstreamer0.8-mad) package
```
sudo apt-get update
sudo apt-get install gstreamer0.8-mad
```
 - For Dapper you need the [**gstreamer0.10-plugins-ugly**](http://packages.ubuntu.com/dapper/libs/gstreamer0.10-plugins-ugly) package
```
sudo apt-get update
sudo apt-get install gstreamer0.10-plugins-ugly
```
