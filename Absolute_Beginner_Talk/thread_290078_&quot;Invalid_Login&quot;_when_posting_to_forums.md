### Post by DaveBorealis on 2006-10-31
I used to have this problem running Firefox on Slackware 9.1. The browser had scripting enabled with no pop-up blocking, so this wasn't the cause...although you might want to double check and see if you do in fact allow scripting and pop-ups.

I never figured out what the problem was because I switched to Dapper, and logins suddenly all worked at such places as hotmail and YouTube, where they didn't before.

So, because Edgy is a little rough in spots, especially for a beginner, why don't you try a live CD for Dapper (6.10) and see if that doesn't work for all logins?

Something to try....
