### Post by ahaslam on 2006-06-12
If you can get your hands on the alternate (text based) install dics you should be ok. The live install cd is great if it works, but has been known to produce errors, especially on lower end machines. I've read reviews where it's not worked the first time, but did on a seccond attempt. I doubt that MEPIS has done anything to cause this, so try installing again and don't use your PC during its install, let it do its thing. If it still doesn't work, try the 'alternative' install disc.

Tony.
