### Post by Inxsible on 2008-04-07
> **lookmomimonlinux said:**
> I would like to be able to send a folder with files in it through pidgin but how do I zip. or rar. files to send?The  command line option was given to you by Jest3r in an earlier post. If you prefer the graphical way, then simply navigate to the folder which you want to transfer. Right click on it and then select Create Archive. Select which type of archive you want and it will create it for you. Then in pidgin, when you transfer files, make sure you select the one that you just created.
