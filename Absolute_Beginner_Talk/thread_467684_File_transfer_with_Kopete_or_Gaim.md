### Post by smolko on 2007-06-17
File transfer through ICQ works quite well in gaim, right now I'm not behind NAT but I've tried it some time ago and it worked . The trick is, that you can't receive a **folder** sent from the original client. You can** only receive single files**. So if you have more files, just compress them and it should work fine from the original client to gaim.
