### Post by oilchangeguy on 2007-03-15
if you're using 6.06lts (long term support) you don't need to do anything until the next lts version comes out. if you're using say 6.10 and are happy with it, keep it. however after the brief support period is over, no more updates will be available. so if you just have to have the latest but maybe not the best version, keep updating as new versions are available. if not stick with the long term support version. as far as for re-installing, of course it will wipe out everything. but as far as doing an upgrade, i'm sure linux is not much different than windows in that regard, the "experts" will tell you it's always best to start fresh (i agree). just back up your data, and re-install that on a fresh operating system.
