### Post by jordanmthomas on 2006-12-06
Generally, you need to reinstall the programs.  If it's a standalone program that doesn't really get installed:  ie, config files are all in one nice directory, no registry entries, etc....you may be able to just run it.  It can never hurt to try.  If it doesn't work, then you'll need to reinstall.
