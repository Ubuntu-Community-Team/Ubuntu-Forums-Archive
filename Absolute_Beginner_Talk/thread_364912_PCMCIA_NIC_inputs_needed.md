### Post by Skidpad on 2007-02-18
Greetings. You will see many threads about getting wireless to work in Ubuntu.  As a new user myself, I had a Linksys that had a Broadcom 4309 chipset in it.  It worked fine in Windows, but not in X.  After much research, I concluded that I wanted to learn X while using it, and that meant surfing this very forum while using my laptop...that laptop with the stubborn 4309 in it. I decided to forgo the headache, and purchase an X supported card so I could use X and surf at the same time.

For me, that card was an ASUS WL-107G.  It uses the Ralink RT2500 chipset, which is widely supported by X; I paid $30 for it online.

Here's a list of Ralink chipset cards:
[http://ralink.rapla.net/](http://ralink.rapla.net/)
