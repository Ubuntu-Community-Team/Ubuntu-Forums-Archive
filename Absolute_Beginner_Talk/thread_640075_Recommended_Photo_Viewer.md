### Post by RomeReactor on 2007-12-13
Hi. You could give [F-Spot]("http://f-spot.org/Main_Page") a try, since it's already installed. If you want something lighter, try [Gthumb]("http://gthumb.sourceforge.net/") (also included in the default installation). Then there's also [Picasa]("http://picasa.google.com/linux/").

According to [Wine's application database]("http://appdb.winehq.org/"), the only versions of ACDsee that run in Wine are [2.22 through 2.4]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=721").
