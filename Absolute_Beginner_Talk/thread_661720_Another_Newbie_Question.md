### Post by stump138 on 2008-01-08
for GDesklets, there should be an icon under applications->Accessories.  If you installed using synaptic, you should only have to type:

```
gdesklets
```
for it to go:)

typically if you have installed the application using synaptic, you are all set. There is no need to do anything with a tarball etc..  All you have to do is launch from a terminal by typing the name or hunt down what menu it ended up at:)
