### Post by candtalan on 2008-01-01
Welcome!
New applications are usually got by using either the 'add/remove programs' in the menu (when you are internet connected) and also using the package manager (synaptic). These both use the online Repositories of many thousands of apps. With more experience you might use a downloaded package which is pre-configured such as a .deb package (ubuntu is debian based of course), or later if you really want to fly, then get source and compile stuff for yourself. Although you will have to also maintain it yourself too.
For a messenger client use the multi client such as pidgin.
Verzion cell internet usb card: easy to plug it in and just try it, maybe it will work, maybe not. Just take stuff step by step, ask, discuss, and keep going, until you can make final decisions.
