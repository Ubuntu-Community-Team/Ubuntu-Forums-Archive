### Post by igknighted on 2007-03-01
KDE apps almost always work perfectly fine in gnome.  However, loading multiple libraries can cause the system to be sluggish.  If you prefer KDE apps, you should use KDE (as a general rule).  You can download and install KDE onto you Ubuntu system and log into that instead of gnome.
