### Post by jordanmthomas on 2007-11-07
If you're using gutsy, fonts are now configured under gnome-appearance-properties (should be under appearance in your preferences menu)

If you're using feisty like the thing on the side says you do, the fonts configuration is 
```
gnome-font-properties
```
hope this helps;
