### Post by MethodOne on 2007-12-09
Click [here]("apt:ubuntu-restricted-extras") if you're running Gutsy or enter the following in the terminal:
```
sudo aptitude install ubuntu-restricted-extras
```

You can play the file in Totem after that's installed.  That package also installs Flash, Java, and the common fonts found in Windows (Arial, Times New Roman, etc.)
