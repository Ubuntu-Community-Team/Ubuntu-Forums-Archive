### Post by legend2440 on 2008-01-18
I too had trouble getting tv out to work. I am using Gutsy
I have a Radeon 9600 so the first thing I did was install the latest drivers using Envy [http://albertomilone.com/nvidia_scripts1.html](http://albertomilone.com/nvidia_scripts1.html)

This set my card up to use the fglrx driver.  I am in the US so my tv format is NTSC-M. The reason I mention that is because when I first got tv out to work the television screen was flipping constantly like a vertical hold problem. Switching from Pal to NTSC-M fixed that.

If you type **amdcccle** in a terminal does the ATI Catalyst Control Center pop up? If it does maybe you won't have to reinstall drivers but I'm not sure. And if you type **aticonfig** in a terminal do the card options pop up? If it does there are a lot of useful options there.

Here is the section of my xorg.conf that pertains to tv out. This is all I needed to get tv out working.
My computer screen resolution stays at 1024x768 and I can watch movies on the tv and monitor at the same time using mplayer. For some reason vlc and totem only show a blue or black box where the movie should be. Also, in mplayer I have to choose the gl video driver in preferences or it doesn't work.
