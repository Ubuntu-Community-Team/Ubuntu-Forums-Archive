### Post by sethmahoney on 2006-01-04
You don't usually need to run explicitely.  If you're trying to install, say, a game (see note below), just insert the CD, wait for a Nautilus window to pop up, and doubleclick the installer.

Note: Here's the deal: If you're installing a game, it likely wont work under Wine unless it is OpenGL.  If you're trying to get a DirectX game running on Linux, you need to use Cedega.
