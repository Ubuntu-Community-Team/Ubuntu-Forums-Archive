### Post by Linux Lover on 2006-09-10
Tools --> Options --> Language Settings (Expand it) --> Languages

Now from the right pane set your User interface language. I have set it English (UK). Click OK to set it. Restart OOWriter. Now turn on Auto spelling check (the button between SpellingCheck and Cut).

Hope it will solve your problem.
