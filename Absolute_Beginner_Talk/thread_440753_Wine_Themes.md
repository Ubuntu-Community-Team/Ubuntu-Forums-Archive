### Post by ultrageeky on 2008-02-14
1, Search for the folder .wine (usually found at /home/(user name)/.wine;
2, add a folder called "Themes",
3 visit [http://search.deviantart.com/?section=browse&qh=boost%3Apopular+age_sigma%3A24h+age_scale%3A5&q=msstyles](http://search.deviantart.com/?section=browse&qh=boost%3Apopular+age_sigma%3A24h+age_scale%3A5&q=msstyles)
4, download a theme,
5, add the .msstyles file (into a folder of its own) under the (newly created) "Themes" folder,
6, use Katapult to load the program called "Configure Wine" (Press left Alt+Space, type "configure wine", press enter,
7, Under Desk Top Integration, select "Install Theme" then select the .msstyles file.

The actual file your looking for to add a theme is .msstyles.  Use a Google search if need be.

It's taken me a few days of wading through Google results to get the name of the theme files and to locate a download site.

I hope this has helped someone.:KS
