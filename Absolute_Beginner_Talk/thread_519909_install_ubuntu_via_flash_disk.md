### Post by Hospadar on 2007-08-07
you could try [this]("https://help.ubuntu.com/community/Installation/FromUSBStick?highlight=%28installation%29") guide, it's for 6.06, but you should be able to use it with fiesty (7.04) as well.  Note that they have you copying files from the cd, so unless you are currently using linux and can mount the .iso directly you will have to burn it or mount it with a tool like daemon to read the files out of it.  some compressors (maybe winzip or 7-zip) may be able to read the files out of the .iso as well, i'm not sure on that part.
