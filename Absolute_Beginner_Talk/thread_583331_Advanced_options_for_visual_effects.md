### Post by sgornick on 2007-10-20
The name of the application as it will appear in the list is "Advanced Desktop Effects Settings (ccsm)".

It does not appear in In Add/Remove Aplications until you change the "Show" option to "All Open Source applications"  (or "All available applications").   It will then appear under the "Other" category.

Once installed, the configuration can be launched with 
   System -> Preferences -> Advanced Dektop Effects Settings
