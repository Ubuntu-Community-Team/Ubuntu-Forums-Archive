Experience varies from one individual to the next. I personally chose Ubuntu over other Linux distributions for the following reasons:

1. Community: The Ubuntu community and documentation made it easier to learn how to maintain and use distro than any other.

2. Stability: Red Hat (Fedora) and Ubuntu (LTS) are the two least buggy and best conceived  of the free desktops. Although Fedora is as well designed as Ubuntu, some might even say it is superior, the community just isnt the same, resulting in a tedious, tiresome experience for the beginner.

Open SUSE was impressive as well, but only as a work in progress. It ran slow as molasses regardless of the kernel version, and something critical was always broken.

I dont have any experience with Feisty, so I'll leave it to others to comment.

### Post by NicoleM on 2007-03-26
I don't have a ton of linux experience so I can't attest to Ubuntu as being the "best" but I can say that I've done a lot of research lately and it seems like a less-scary way to learn some linux basics.

I haven't even installed yet, and this forum has been ridiculously supportive with some troubles I was having not to mention all the new bookmarks I've acquired just browsing the forums and finding links full of invaluable information.

i'm not sure if you're looking for a compelling reason that you should install or not, but I think it's important to note that the reasons some people use ubuntu are probably not going to be the same reasons you will use it.  at the very least pop in the livecd and mess around and see what you think.  it might be for you, it might not, but there's only one way to find out :)

i don't know if ubuntu will suit my needs in the future, but I'm hoping it will help me become comfortable enough with linux that later on if my needs change I can accurately evaluate the options available to me.

i'm not an ubuntu "fanboy" (fangirl?) by any means, and from what I understand linux in general CAN take some tinkering to get started (though not always).  The overall consensus that I've observed seems to be that it's worth the effort even if you end up finding out that it isn't for you...at least you learned something new.
