> I dont even know how to install stuff on here, i have a friend that knows the ins and outs of linux but he's a bit of a geek and wont explane it to me without getting all arsey about it :P


You need Synaptic for that. Start it from the System menu. (Can't tell you exactly how it's called in the english version, sorry, but you should be able to find it).

Don't go overboard with istalling programs, though. It'll have to save it to your RAM, and booting from a live cd already takes a lot of ram.

Oh, and you'll have to install the programs everytime you boot your computer. It's a bit annoying, but that's how live cds work since they don't use the hdd.

> 
Ive networked it up with the computer in the living room and i can access my mp3 collection (which is on the living room pc)

How do i go about playing them... Or is there no way to do anything like that using the live cd?

To play mp3s with Totem or Rhythmbox you need to install gstreamer0.8-mad from synaptic. If you can't find it you'll have to enable the universe repository. You can do this from Synaptic. Click on Settings -> Repositories -> Add and check universe in the dialog that opens.

mp3 support is not installed by default since it is patented and buying a license to distribute it with ubuntu would cost a lot of money, which then couldn't go towards paying developers or sending ubuntu cds for free.
