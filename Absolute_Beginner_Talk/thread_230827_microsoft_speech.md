### Post by philipacamaniac on 2006-08-06
You need to install a speech engine, such as festival (you can install it with Synaptic/Adept). Then you probably want a frontend - I don't know about Gnome, but KDE offers ksayit and kmouth, both available in the Universe component in Synaptic/Adept.
