### Post by Marsolin on 2006-10-18
Have you tried selecting Configure Konqueror from the Settings menu? Choose File Associations from the left sidebar and type in the extension for a shoutcast file. Once you select the file type there is a list on the right of apps that are associated with it. Add Kaffeine to the list and move it to the top.

I hope that works for you.
