### Post by an.echte.trilingue on 2007-03-07
There is no itunes for linux period.  However, there is a piece of software called wine that runs Windows software on Linux, with varying degrees of success.  You can download the Windows version of itunes, then "sudo apt-get install wine", and install iTunes with wine.

That said, however, wine is a royal pain to get working.  There is a commercial option for 20 dollars called CrossOver Office that will install iTunes almost insta-magically for you.  You have to factor in how much time you are willing to spend and how bad you want this to work.

By the way, a quick google search for "howto xubuntu ipod" brought up the program gtkpod.  Works just fine for me to move music on and off the iPod, but there is no itunes store interface.

Take care
