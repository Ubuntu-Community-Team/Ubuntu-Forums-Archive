### Post by mcduck on 2008-01-21
> **Sidewinder1 said:**
> No, Synaptic does not always have the very latest versions. I'm not really sure why; perhaps someone more knowledgeable can elaborate.

Synaptic has the latest versions available in Ubuntu's repositories. But Ubuntu's packages are not generally updated between Ubuntu releases for other than security reasons.

If you want more up-to-date programs you'll either need to install them by hand or move to some distribution that uses rolling version releases.

### Post by emarkd on 2008-01-21
Your package manager lists the newest versions of software PACKAGES.  These are compiled versions of the same software that is configured to run on Ubuntu (or whatever distro you're using).  New versions of software have to be packaged before they show up in the package manager.  See [https://wiki.ubuntu.com/MOTU]("https://wiki.ubuntu.com/MOTU") for more info on how Ubuntu's crew handles this.
