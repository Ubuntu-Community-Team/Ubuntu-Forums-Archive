### Post by FreewareFan on 2008-03-28
As far as Swiftweasel goes, it depends on which version you are using..  If it's version 2.xxx  the directory is at ~/.mozilla/swiftweasel , but if you are using the new 3beta4.5 version, it's at ~/.mozilla-trunk/swiftweasel .  

I wouldn't remove Firefox, because some other programs within the system depend on it somewhat.  It hurts nothing to leave it on the system, it doesn't get in the way at all...
