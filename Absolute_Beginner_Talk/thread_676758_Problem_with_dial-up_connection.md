### Post by AnonCat on 2008-01-24
I'm not sure if you've already done this, but you might also try this:

Open a terminal window and type:  

sudo wvdialconf

After doing that type: 

sudo gedit /etc/wvdial.conf 

Check to make sure the password and username is correct and add the following line if you're using a Smartlink modem:  

Carrier Check = no 

save and exit

then type:

wvdial
