### Post by kolinab on 2007-01-04
If you want to get back to defaule settings you can delete your user profile for t-bird. I would recommend making a backup first if you have any emails you would like to save. Your profile is in a hidden directory in /home called ./mozilla-thunderbird (if you are using the ubuntu T-bird) or ./thunderbird if you have installed from the nice people at mozilla directly. 

You can delete this folder and your thunderbird will be a clean slate as far as user settings.

As far as the SMTP problem - the one I use I have to set up the authentication steps pretty carefully. Check with your mail provider and they should have instructions for what info to put in as far as SMTP. There might be a tutorial for some other mail program but you should be able to glean the necessary information.

Good luck and post back if you have probs.

Edit > Preferences > Privacy > Passwords
Choose
View saved passwords
And press
