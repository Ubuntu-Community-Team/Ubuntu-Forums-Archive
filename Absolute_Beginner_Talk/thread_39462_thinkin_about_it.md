### Post by Seti on 2005-06-05
That's something you need to decide for yourself, simply by tring it. Use it for a while and see what you can do with it. Try setting up a dual-boot if you still fill you need a lifeline back to windows once in a while. 
If you go through this forum a bit you'll notice a lot of happy, liberated-feeling people who are for the first time really connecting with their computer; getting it to do things they didn't think possible. 
Put it this way, when I first started with GNU/Linux just over a year ago, (Mandrake) I could't believe sometimes the performance gain I was seeing on my system over XP on another partition. Copying large files around to other partitions, reading discs, network, sound...even KDE seemed way ahead of tired old NT. Plus I could see all my data on my C: drive; files inc. C:\Windows...I felt empowered. And it didn't fragment as I added data and apps...(did I mention no need for AV  and anti-spyware proggies?), well, you get the picture.
If you're getting into computer science and its something you'd like to do, would you really want to LIMIT yourself to only being a Microsoft guy? Or would you like to be versed in unix, Linux, Solaris, BSD, OSX...as well? 
Variety is nice.

### Post by Knome_fan on 2005-06-05
[QUOTE=kidgrey]how is it that better for me to switch totally to a new operating system with no knowledge... 
[/quote]
Why should you switch totally away? You can run Linux and Windows on the same computer without a problem, so if you decide to give Linux a try, you can still use Windows.
Further, you could also try the LiveCD first. That's an Ubuntu that runs completely from cdrom, so you don't even have to install it and it won't affect your Windows install at all. So if you just want to have a look at this Linux thing before deciding if you are interested, this might be a good idea. 

[QUOTE=kidgrey]
i know windows is for the novice.. but when i switch i will be a novice.. and i hear linux is not an easy operatin system[/QUOTE]
First off all, I don't think Windows is only for novices. Anyway, be that as it may, don't change to Linux because you feel you have to (you really don't, if Windows works for you, that's fine), but use Linux because you want to, because you think trying out new things is fun, because you want to learn something new, because you think it makes sense to know something else besides Windows if you want to work in IT.
And as I already said, this is not an either, or decision, as you can use both, Linux and Windows.
