### Post by hoges on 2008-03-05
1) Gamers Edition contains a whole lot of extra packages for games-oriented users. It may also contain the proprietary software that Ubuntu can't legally package (restricted drivers, adobe, etc), but don't hold me to it... Go with 7.10. It contains less drivers that could go wrong on install. You can always add the other stuff later.

2) Transfer files - yes. Motorola tools - unlikely. Depends on what what the software does, and whether it's compatible or not. Maybe search the forums for more info.

3) Common question that causes confusion. No anti-virus software is needed for linux, as there are [hardly any to combat]("http://en.wikipedia.org/wiki/List_of_Linux_computer_viruses"). Basically if you're careful about what you're doing, you'll never hear about them. Just put up a firewall to block any hacking/port scanning etc and you'll be fine.

4) Flash games you can download adobe flash player or freeware equivalent. I recommend adobe - it's more reliable and is compatible with more stuff.

5) There's a lot of free software on synaptic that will do this. Just look through the repositories for more info once installed.

I'd recommend the live cd first. It will give you an overview of what software ubuntu comes with to begin with.
