Well, I've herd of WUBI but never used it.
The process of obtaining and running Ubuntu is relatively simple and straightforward.

First go to [http://releases.ubuntu.com/7.04/](http://releases.ubuntu.com/7.04/)
Download the (x86) desktop CD
When its done downloading, download ImgBurn ( [http://fileforum.betanews.com/download/ImgBurn/1128426215/1](http://fileforum.betanews.com/download/ImgBurn/1128426215/1)) and use it to burn the .ISO file to the cd.
When you're done, reboot (make sure the cd is in the drive)
Select "Start or Install Ubuntu".

### Post by Wim Sturkenboom on 2007-08-07
The best way to go is to solve the problem, not to work around it. There can be several reasons why it does not want to boot from CD (that's the problem as far as I understand it from your post).
[LIST=1]
[*]Boot sequence; just verify and make sure that the sequence is CD first
[*]Bad CD/DVD player; use your Windows installation disk and boot from it; if it can't, it's either above or your player has died
[*]Bad CD; try CD in another computer
[*]Wrong CD; it might be that you took the wrong CD from the pack (e.g. CD 2). Only the first one has to be bootable, the others don't have to be bootable
[/LIST]
