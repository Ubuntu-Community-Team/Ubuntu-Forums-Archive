### Post by rsambuca on 2008-03-22
Although it is not a standard player, the program Audacity will allow you to slow down, speed up, change pitch, etc for any mp3 or other audio file.  It is in the standard repositories, so you can install it from the Synaptic Package Manager, or via command line:

```
sudo apt-get install audacity
```
