### Post by rajaram_s on 2008-04-21
ya.. the SHIP IT free cd scheme suits exactly users like you.... There are many alternatives for downloading packages without using internet in ubuntu. You can use:

1. Nonetdebs.homeip.net:

It is a website where you can register uploading your status.txt file, which contains the list of packages already installed in your computer, using which it shows all the packages that needs to be installed for a particular software and gives them all as deb packages.

2. There is a software called apt on cd, which lets you store all the packages you download usinf apt-get as a software repository on the disk. You can ask your friend with an internet connection to download it for you and get you the files burnt to a disk, which will serve you as a bundle of software repositories.
