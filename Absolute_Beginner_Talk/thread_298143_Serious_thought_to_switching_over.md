i swiched from fedora to kubuntu when dapper was released. main differences are:
- different package management. ubuntu is debian-based. instead of yum you use apt-get or (better) aptitude. 
- different networking. it's easy to set everything up un ubuntu, but i must say i'm still slighly confused when i need to do advanced stuff. the red hat system seems a bit more logical to me. but a have a fedora bible and no books on ubuntu, so maybe i should first buy one. 
- official kde support. 
- only one installation cd. i think this is a benefit. 
- great howto's. check [http://ubuntuguide.org/wiki/Dapper](http://ubuntuguide.org/wiki/Dapper)
- su & sudo. in ubuntu you don't use su, but the user that installs the system has full sudo permissions. it took me a while to get used to this, but in the end it works fine. 

my personal experience is that ubuntu is easier to install and configure than fedora. actually i feel like i'm loosing my linux-knowladge because ubuntu is almost too easy 8)
