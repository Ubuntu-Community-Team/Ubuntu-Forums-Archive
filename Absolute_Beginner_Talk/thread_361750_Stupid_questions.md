### Post by aidanr on 2007-02-14
1. ubuntu uses [gnome]("http://www.gnome.org"), kubuntu uses [kde]("http://www.kde.org"), kde and gnome are desktop environments, i guess they are roughly comparable to the windows shell, each provide a different look and feel and different applications and configuration tools. kde provides you with a lot of configuration options, while gnome tends to be simpler. it comes down to personal preference though, i suggest you try both out. there are also lots of other desktop environments to check out such as xfce, fluxbox, icewm etc

2. i think there is a project in the works for an installer that runs from within windows, but i don't know if there has been any release yet, so as far as i know you are stuck with the cd installation for the moment. possible things to try: you can check your iso download against the provided md5 sum to make sure it has fully downloaded and isn't corrupt, you can also try burning it at slower speed to be absolutely sure it has burned right, and of course make sure you burn the iso contents to the disc and not just the iso file itself (although you say it stalls at 31%, so i imagine you've got this covered already)

good luck
