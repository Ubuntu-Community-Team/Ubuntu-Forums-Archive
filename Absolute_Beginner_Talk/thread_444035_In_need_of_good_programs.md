### Post by Marsolin on 2007-05-15
I'm not familiar with Jet Audio so I don't know what to recommend there.  If I'm not mistaken Exact Audio Copy is a CD ripper. For your other problem you need to have [Samba]("http://linuxappfinder.com/package/samba") install and the service needs to be running.
