Hey all - I've had this problem with Ubuntu from the first time I installed it, in every version I've used.  I'm using a Thinkpad T60 laptop with a built in microphone.  When I install Ubuntu, the microphone defaults to "muted" in the Volume Control window, so naturally it does not work in any application.  However, when I un-mute the microphone, I hear all sounds over the speakers in real-time, and all sound-input applications seem to still not work. I had a similar problem when I installed Skype. I'm not sure what I did to fix it but finally Echo 123 started letting me hear my own voice. 

What I did was to find the standard application that all Ubuntu versions have called 'sound recorder'. Click 'file' and then in the drop down, click 'open volume control'. You'll then see a a control panel with numerous controls and tabs. Under 'edit' you see 'preferences' which allows you to make visible more tracks. I enabled them all and then started trying various combinations and finally was able to get the microphone working in Skype. It was simply trial and error, I haven't a clue what I changed now. Good luck.
