### Post by Jagot on 2006-05-02
Well obvioucly as you probably know, if you intend on sharing those documents you will need to save them from OOo as .xls and .doc files.  In terms of opening them with Microsoft Office, I generally don't have any problems except for an occasional formatting error - usually a problem with different margins - so you will need to set those up properly.

There shouldn't be any problems in sharing between OOo and Office unless you use drastically complicated macros or something along those lines.
