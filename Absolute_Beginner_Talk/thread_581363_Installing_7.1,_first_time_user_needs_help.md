Xubuntu is worth your consideration 
but if you want the fully fledged ubuntu then use the alternate cd, which is a text based install. The recommended RAM for the desktop Cd to install is now 320MB as per [http://releases.ubuntu.com/gutsy/](http://releases.ubuntu.com/gutsy/)
(It was previously 256MB for Feisty)

I am running Kubuntu Gutsy on 256RAM and have been for a while (during testing). No real problem if you just want to email, surf the net, play music/watch media (can't say that about DVD though as I don't have a DVD player)

link to the alternate cd download here if you need it
[http://releases.ubuntu.com/gutsy/ubuntu-7.10-alternate-i386.iso](http://releases.ubuntu.com/gutsy/ubuntu-7.10-alternate-i386.iso)
