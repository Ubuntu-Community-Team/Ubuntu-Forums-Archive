### Post by David Marrs on 2005-09-10
Take xmastree's advice and try the liveCD first. It's like a demo CD. It doesn't need to be installed in anyway. You put the CD in the drive and reset your PC. It'll boot into Ubuntu and you'll be able to use it as much as you like. When you've had enough, take out the disc and restart. You'll find yourself back in Windows.

More generally, I'd recommend getting used to Linux applications in Windows if you can by trying out the standard alternatives, like OpenOffice.org, Firefox and so on. The most popular ones can be installed from the LiveCD also, by running it in windows.
