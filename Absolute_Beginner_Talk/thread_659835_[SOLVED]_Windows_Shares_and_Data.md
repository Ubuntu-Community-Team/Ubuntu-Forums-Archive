### Post by beercz on 2008-01-06
Using your windows based documents in ubuntu should be fine,  I use windows based documents on network shares in ubuntu at work all the time, and I haven't had any problems myself.

Mind you I have very good backup systems, so should the worst happen, I can recover.

So I guess to be ultra safe, take a backup of you critical data first.

It's always good practice to have good backups.
