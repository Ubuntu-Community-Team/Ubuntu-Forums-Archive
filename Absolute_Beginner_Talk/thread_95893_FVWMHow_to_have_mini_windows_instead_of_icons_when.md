### Post by super on 2005-11-27
you mean like [this!]("http://www.linuxshot.org/index.php?page=detail&id_user=21&id_ss=74") (no that is not my screenshot!)

to get the window thumbnails in fvwm you need to include a thumbnailing script in your .fvwm2rc file. usually the script depends on either imagemagick libraries or imlib2 libraries so these need to be installed.

then just follow the instructions found [here.]("http://fvwm.lair.be/viewtopic.php?t=190")

'fvwm.lair.be' always has a ton of useful information of fvwm functions and scripts.
