### Post by melancholeric on 2006-11-21
System -> Preferences -> Font

default settings are (I think) 

Application Font | Sans | 10
Document Font | Sans | 10
Desktop Font | Sans | 10
Window Title Font | Sans Bold | 10
Fixed Width Font | Monospace | 10

Then there are the rendering options, no idea what the default settings are for those though.

It looks like you only need to change the document font + maybe rendering options, everything else seems fairly normal or so.
