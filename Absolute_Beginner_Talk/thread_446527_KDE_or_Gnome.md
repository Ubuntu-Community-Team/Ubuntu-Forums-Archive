### Post by V-600 on 2007-05-17
I used gnome when first moving from windows mainly because it is more different from windows than KDE and because it was "simpler" to use. Now I am used to gnome I find KDE a bit too much and slightly confusing.

Also you can install KDE as well as gnome but from experience you are probably better off with two separate installations. If you install both on one system then any changes you make in one will affect both. Also you end up with a messy menu system with apps from both desktops available. There are programs available to tidy up this though or a little patience works well.

Personally I say pick one, you will get used to it fairly quickly. If you don't then it is not difficult to switch.

Well, KDE is Qt based and KDE appz are streamlined towards KDE meaning the use a set of shared libraries...

Gnome and Xfce are GTK based
