### Post by JustAboutRealJAR on 2007-10-04
unfortunately this is a documented bug that has not been fixed (maybe an update in gutsy?)

what I would recommend is using abcde (a better cd encoder) for ripping, or another audio program, like banshee

to install abcde + lame (mp3 encoder):
```
sudo apt-get install abcde lame
```

to install banshee + lame
```
sudo apt-get install banshee lame
```

(note: abcde is a Command line app. That means it only runs in the terminal. If it's for a new user you probably want to go with banshee.)
