### Post by jw5801 on 2007-07-11
In order to change what Ubuntu does when you insert a CD, DVD or any other removable media, go to System > Preferences > Removable Drives and Media, and you can set the defaults.

### Post by jw5801 on 2007-07-11
The "%m" is a variable that contains the mount-point of the media (ie. /media/cdrom). So totem %m, equates to totem /media/cdrom.
