### Post by Teroedni on 2006-07-10
Yea thats possible
Just make sure to install xp First
Since xp comes with a fat/ntfs formatting tool, you should have no problems
In xp partition tool you make a 60gb fat32 partition or whatever you have in mind. Just make sure that you leave 20 gb as unformmatted;)

[COLOR="Red"]NB"!!I higly reccomend you to use fat32 and not ntfs for your xp as Ubuntu have problems with ntfs file system[/COLOR]. That way you can access your xp/data/music etc files from Ubuntu also



Then you can Install Ubuntu
Make sure to choice edit partitions,,, [COLOR="Red"]Dont let UBuntu automatically partition it!![/COLOR]
When you come to ubuntu partition it will showyour xp partition and a unused
unformatted space of 20gb. You then format this unformatted space into ext 3
