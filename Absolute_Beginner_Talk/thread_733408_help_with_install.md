### Post by JoshuaRL on 2008-03-23
Sure.  Go [here](http://www.ubuntu.com/getubuntu/download) and right under the Start Download button, you'll find the checkmark for the Alternate CD.  It installs with a text-based GUI that is much easier on resources.  The Live CD won't load correctly with that amount of RAM.  I would also suggest you use [Xubuntu](http://www.xubuntu.org/) since it uses the XFCE Desktop Environment and is easier on older computers.  [Here is the download page](http://mirror.anl.gov/pub/ubuntu-iso/CDs-Xubuntu/7.10/release/) and you can get the Alternate CD off of there too.
