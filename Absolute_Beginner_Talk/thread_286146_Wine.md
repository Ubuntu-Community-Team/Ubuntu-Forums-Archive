You use wine in the terminal e.g.
```
wine MyWindowsProgram.exe
```
winecfg allows you to tweak the settings for individual programs to get the best performance.

The wine docs give alot more help [http://www.winehq.com/site/docs/wineusr-guide/running]("http://www.winehq.com/site/docs/wineusr-guide/running")
