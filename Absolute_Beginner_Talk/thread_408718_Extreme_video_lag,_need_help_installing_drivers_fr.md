### Post by x64Jimbo on 2007-04-13
Depends on your video card. 
Is it nVidia? 
sudo aptitude install nvidia-glx
Is it ATI?
sudo aptitude install xorg-driver-fglrx
Then reboot.

### Post by taurus on 2007-04-13
After installing a driver for your ATI, you need to replace "vesa" in Driver with "fglrx" or it is still using vesa driver.

```
gksudo gedit /etc/X11/xorg.conf
```
Or you can try envy if you wish.

[http://www.albertomilone.com/nvidia_scripts1.html](http://www.albertomilone.com/nvidia_scripts1.html)

### Post by Scunizi on 2007-04-13
No need to reboot after changing the driver in xorg.conf.  Just CTRL-ALT-BACKSPACE.  That restarts the gui but is not a full reboot.  You'll still have to log in afterwards though.

 What is Envy?:

"Envy" is an application for Ubuntu Linux written in Python and PyGTK which will:
1) detect the model of your graphic card ([COLOR="Red"]**[SIZE="6"]ATI[/SIZE]**[/COLOR] and Nvidia cards are supported). However "Manual installation" is also available

6) restart the Xserver for you (if you wish so) (this feature is available only in the textual interface)
