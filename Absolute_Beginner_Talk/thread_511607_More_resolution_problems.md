### Post by drakos on 2007-07-28
I don't know that this will fix your problem, but it worked for getting 1440x900 on my integrated Intel card. Basically just go into Synaptic and search for xserver-xorg-video-intel and install it. This should update your graphics card driver. Then go to the terminal:

```
sudo dpkg-reconfigure -phigh xserver-xorg
```

This should allow you to pick *intel* instead of i810 and pick the resolutions you want. Then just reboot X and it should work.

Once again, this worked for me (with my 945G card), so hopefully it'll work for you.
