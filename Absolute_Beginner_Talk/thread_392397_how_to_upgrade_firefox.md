### Post by eljalill on 2007-03-24
You turn on synaptics, which is an update program, and search for firefox there. It should tell you, what version you have. 2.0.0.2 is available through that program (it's in the repositories).
Also you could type 
```
firefox --version
``` in a terminal, and that will also tell you the version of firefox you have installed.

Finally it would be a good idea to run apt-get update in a terminal, or install all updates available in synaptics, just to get you .. well, up to date ;)
