### Post by ltmon on 2005-10-05
I think what you mean by object dock is a theme for gdesklets (if you use the default Gnome) or KXDocker (if you use KDE).

Have a look at [http://ubuntuforums.org/showthread.php?t=31928](http://ubuntuforums.org/showthread.php?t=31928) for a howto on GDesklets.

As for mp3's....
Due to legal reasons the mp3 codec is not included by default... but don't panic yet you can download it seperately.  This link shows how: [https://wiki.ubuntu.com/RestrictedFormats](https://wiki.ubuntu.com/RestrictedFormats)
