### Post by rpgcyco on 2005-09-07
You'll need to use Cedega, as it was mentioned above. You can either pay a few dollars a month for it ([http://www.transgaming.com/](http://www.transgaming.com/)), or compile it from CVS. If you do compile it however, it won't have support for CD copy protection (I think).

WINE is not yet in a state to emulate the Source engine, but may be in the future.
