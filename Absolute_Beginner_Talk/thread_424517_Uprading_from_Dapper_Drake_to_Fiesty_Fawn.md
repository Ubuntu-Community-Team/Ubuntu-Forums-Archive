### Post by jerrylamos on 2007-04-26
Not sure how you tried to use the old partition.
Select manual partition
highlight the partition you want to use for Feisty 7.04 example the Dapper one
Edit the partition to change the mount point to /
Click on Format on the highlighted partition
Try to proceed from there....I'm not sure migration from the Dapper would work.  Migration doesn't do much for me.

By the way, I saved my old Dapper home & Desktop files example on another system on the LAN with shared files, or else copying to a USB drive.  That includes Firefox bookmarks file export.
