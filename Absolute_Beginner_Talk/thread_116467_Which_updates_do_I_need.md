### Post by SavageBeginner on 2006-01-12
If you don't want to install an update, and your tired of seeing the update icon all the time, you can lock a program at the current version.

Click System -> Administration -> Synaptic Package Manager and type in your password.

In Synaptic, click on the Status button in the bottom left corner.

In the left panel, click on "Installed (upgradable)". This will show you all the available updates.

Click on the program you do not want to update to highlight it.

In the menu, click Package -> Lock Version

Now any updates for that program will be ignored.

In all honesty you should just let the package manager update whatever it wants.  As has already been said in this thread, the package manager only updates programs which are already installed on your computer.
