### Post by daller on 2006-09-09
You basically have to replace every instance of "dapper" into 
"edgy" in the "/etc/apt/sources.list" file

Use your favourite editor...

In ubuntu, run this command:

```
sudo gedit /etc/apt/sources.list
```

Or if Kubuntu:

```
sudo kate /etc/apt/sources.list
```

And then find Edit -> Replace, and replace dapper with edgy! (everywhere...)

Or even easier... Send me your sources.list file, and I'll edit it!

I must warn you! - Edgy is still in it's development state!
