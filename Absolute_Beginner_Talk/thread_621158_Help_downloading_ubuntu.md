### Post by espressopigeon on 2007-11-23
Download Ubuntu from:

[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

then use a disc burning program to burn the file to a CD. Make sure you choose "Image burn" or "Burn as ISO" or another similar option when you burn! Put the CD in your computer and restart. When the BIOS is loading (where the computer manufacturer is displayed on the screen) press the key that activates BIOS setup. This key is most likely shown in a corner of the screen. Find a "Boot Sequence" or "Boot Order" option and set it so that the computer will look to a CD for an operating system before it looks to a hard drive. Exit setup (don't forget to save changes) and you will get an Ubuntu menu. Choose "Start or Install Ubuntu."
