[QUOTE=Hawkeye05]Im getting Ubuntu from a friend tommorrow, i mostly want it because im sick of the politics with Windows, and i want a good stable OS for my older PC.[/quote] Just how old is it? Because it may need a lighter-weight distro if it's too old.

> I have a few questions though before i install.  Im good with windows, but i have no programming expieriance, i do have some DOS/ Command Prompt expieriance. You don't need programming experience, but command prompt comfort is a good thing to have when you're first setting up Ubuntu.

> 
- Can i uninstall it easily if something goes wrong? Not easily, but you can install over it. There's a live CD you can try out that won't damage your hard drive.
> 
- Can i switch between Windows and Ubuntu? Yes, if you set up a dual-boot.
> 
- What are the system Requirements especially Video memory and Hard Drive I don't remember. But have at least 128 MB of RAM is recommended.
> 
- I have Netzero and im wondering if ill be able to connect while using it I don't know.
> 
- Will my XP drivers (external burner, Printer) work with it? The drivers themselves won't work, but the devices probably will, and if they don't, ask questions on the forums here, and people will try to help you get them working.
> 
- How can i sych my Palm Pilot with it? I don't know.
> 
- What kind of Windows Emulator is availible so i can play my games Look into something called Cedega.
>
