True and not true - ASUS uses the basic Intel chipset (G965) but still does its own thing for graphics and sound. There is no guarantee that an operating system that runs on one will run on the other.

 Thew best advice is that an installed feisty Ubuntu should run on the ASUS motherboard. But there is still a risk.
