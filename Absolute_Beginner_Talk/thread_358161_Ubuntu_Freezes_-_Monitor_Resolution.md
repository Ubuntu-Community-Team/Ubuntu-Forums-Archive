### Post by Happy_Man on 2007-02-10
Well, now Windows should show your ext3 formatted drive if it's hooked up to your computer. Find Z:/etc/X11/xorg.conf, (substitute Z for whatever drive letter you chose during install) and then edit it like I said. Then try rebooting. And for future reference, when someone gives you commandline edit instructions, they mean to type this into the terminal:

```

sudo gedit name_of_file

```

What that command does is tell the computer to open name_of_file with admin privileges in GEdit (a text editor like Notepad). Sorry if I didn't make that clear!
