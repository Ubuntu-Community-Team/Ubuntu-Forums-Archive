### Post by openix on 2007-03-30
System requirements for a 6.06 install from a standard Live CD is 256MB RAM and 3GB disk space. I think the alternate CD still takes 192MB, I would suggest you give the [Xubuntu Alternate CD]("http://cdimages.ubuntu.com/xubuntu/releases/6.10/release/xubuntu-6.10-alternate-i386.iso") a try as this installs on systems with less than 128MB RAM and is pretty fast due to the lightweight xfce desktop.
