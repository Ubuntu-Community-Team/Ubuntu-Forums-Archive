### Post by ndube on 2007-02-20
You can also run ubuntu in what is called "persistent mode" basically, you boot from the live cd, and any other programs, customizations, documents, etc. are stored on the external drive. This can come in handy if you want to be able to boot into ubuntu on several systems. Here is a link to the instructions.

[https://wiki.ubuntu.com/LiveUsbPendrivePersistent](https://wiki.ubuntu.com/LiveUsbPendrivePersistent)
