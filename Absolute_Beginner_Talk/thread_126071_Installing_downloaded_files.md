This link touches on what the formats are:
[http://www.psychocats.net/linux/installingsoftware.php]("http://www.psychocats.net/linux/installingsoftware.php")
In brief:
.deb = Debian package format.  Ubuntu is Debian based, so it also uses this package format.
.rpm = Red Hat Package Manager format.  Red Had, Mandrake, and some other distributions use this package format.
.tar, .tgz, .tbz2 = Tarballs.  This is a fancy name for archives (think zip-files for Windows).  tar = Tape Archive.  It basically takes a bunch of files and directories and glues them into one big file.  gzip and bzip2 are compression algorithms to shrink the resulting file.  Since compression is used with archiving so often, the program user to do the archiving tends to have the decompression part built in.  Hence, .tgz and .tbz2.
