Ok, you can download and install Wine, then download the Windows Firefox package and install it with Wine.

Then, go start up the newly installed Firefox (Menu>Wine> Programs>Firefox) and go install the Shockwave player that way.

That might work, might not, but it's how I use Roboform sometimes ;) 

Hope that helps

Edit: Dowload the shockwave.exe from Adobe, install it with Wine and you'll be all set.
