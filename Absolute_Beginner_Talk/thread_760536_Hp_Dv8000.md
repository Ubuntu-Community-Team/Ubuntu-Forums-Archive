### Post by bford16 on 2008-05-13
> **xcvcx said:**
> I don't see a restricted drivers,

To find the restricted drivers tool, click on System>Administration>Hardware Drivers (for Ubuntu Hardy). You will need an Internet connection in order to install the drivers.  Once you have this, the installation is almost automatic. Open Hardware Drivers, and Enable the drivers. You will be prompted to install the Restricted Drivers, and you will see a warning that this software is not supported.  Go ahead and install the drivers, then restart the computer.
