### Post by Nythain on 2007-06-11
there's your culprit... ASSUMING envy installed the nvidia drivers ok (envy and automatix arent to great at that) you need to change the section
```

Section "Device"
        Identifier      "nVidia Corporation G70 [GeForce 7600 GT]"
        Driver          "nv"
        BusID           "PCI:4:0:0"
EndSection

```
to read
```

Section "Device"
        Identifier      "nVidia Corporation G70 [GeForce 7600 GT]"
        Driver          "nvidia"
        BusID           "PCI:4:0:0"
EndSection

```
then restart x either by shutting down and rebooting, or hitting control+alt+backspace

i guess it helps if i tell you how to edit the file to make the changes eh
in terminal type
```

gksudo gedit /etc/X11/xorg.conf

```
it will ask for a password, thats your account password
then it should open up the file xorg.conf in gedit with root privelages.
edit that section, bascally changing nv to nvidia
save the file and voilla

### Post by Nythain on 2007-06-11
i really wish i would have mentioned how to revert back to the nv drivers via Command Line in case envy DIDNT install the proprietary drivers right... but judging from the xorg.conf it did, it just didnt make the important change...

well here we go, in case you read this before doing what i said...
if you cant boot back into x and are dumped into a command line, or a black screen for that matter

login
sudo nano /etc/X11/xorg.conf
change the "nvidia" back to "nv"
