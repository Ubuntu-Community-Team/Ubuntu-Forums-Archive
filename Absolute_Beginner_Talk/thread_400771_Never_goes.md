### Post by 3rdalbum on 2007-04-04
Since this post is 4 hours old, you've probably downloaded all the updates by now.

But just so you know, yes Ubuntu's alternate CD installer does download and install all available updates during this step. Unplugging the Internet connection while these are downloading is not a good idea - I did it once, and was left without a GUI. (I knew a little about Ubuntu so I figured out how to fix it in a few minutes).

If you unplug your internet connection before starting the install, then the install will happen without updating the system (you can choose to update later if you wish), and everything will go as quickly as possible.

On a seperate note, Virtual PC is quite buggy, and I wouldn't recommend trying to run a Linux desktop in it (VPC's emulated display hardware is too primitive and unrealistic). I suggest using Virtualbox or VMWare Server, both of which are excellent virtualisation programs and are available for Windows:

[www.virtualbox.org](www.virtualbox.org)
[www.vmware.com](www.vmware.com)
