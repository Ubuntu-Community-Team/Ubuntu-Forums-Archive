```

To be removed:
   kde
   kdeaccessibility

```

If I mark "KSayit" for "Complete Removal", I get this:
```

To be removed:
   kde
   kdeaccessibility

```

Same thing.

Can the difference between the two options be explained?  And will the removal of "KSayit" completely remove "KDE"?

### Post by bscbrit on 2006-02-20
Yes, 'mark for removal' removes just the packages, but 'Mark for Complete Removal' also removes the packages and any configuration files that might have been created.  I do not know which files might be removed if KDE is 'Marked for Complete Removal'
