### Post by Born-Thirsty on 2008-01-22
The colours are more for quick identification of what type of file/folder it is or what permissions are available for that file/folder. All of these are customizable by the user.

A folder will be a different colour to a file and a file that has executable permissions will have a different colour from one that does not.

I hope that answers your question.

### Post by tophcito on 2008-01-22
Hello!

The following command, run from a Terminal, puts out the different color codes (and their translation into english) that are used. So u can find a detailed list on what file gets what color (and change that color should u wish to do so.)

```
dircolors -p
```
