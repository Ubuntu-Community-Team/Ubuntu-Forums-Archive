### Post by Peepsalot on 2006-11-11
You shouldn't need a plugin for normal PNGs.

Firefox views all PNG files perfectly as far as I have seen.

And IE can view non-transparent PNGs just fine.

The problem is that IE does not support PNG transparency very well.  At least I know that IE6 didn't.

I haven't used IE7, but if you go to this page in IE7, you can see how it performs.
[http://entropymine.com/jason/testbed/pngtrans/](http://entropymine.com/jason/testbed/pngtrans/)

There is a reason why most web pages use gif for transparency, it's the only thing garaunteed to work in pretty much any browser.  I know it's not as good as PNG because it cannot do partial transparency, but those are the breaks.
