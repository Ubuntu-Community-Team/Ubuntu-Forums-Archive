### Post by elpichi on 2008-03-10
haha i had the same issue 
[http://ubuntuforums.org/showthread.php?t=710509](http://ubuntuforums.org/showthread.php?t=710509)
open winecfg and change the video options so the window manager doesn't control the windows, this will allow wine to take full control of the desktop covering the taskbars.

note: you can open winecfg from the terminal
or you can browse through the GUI,  Applications > Wine > Configure Wine
