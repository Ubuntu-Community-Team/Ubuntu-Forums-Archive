### Post by ridgeland on 2007-09-04
I would guess the safest way to have the drivers installed is to have the printer, usb drive etc plugged in and turned on when you first install Ubuntu.

If you have a 7.04 CD then you can test by turning on all the devices and boot the CD.  You'll see if they are recognized.
