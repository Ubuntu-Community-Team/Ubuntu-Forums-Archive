Hi everyone

Thanks for all the help - Ubuntu 7.10 now installs on new Mobo following crashed bios.

I ended up running memtest, found nothing but errors, so I replaced my single 1GB stick of RAM with 2 sticks of 512MB (as specified in the manual for the mobo) - if all else fails, read the instructions!

Seems to work fine now, although the screen flickers like mad when I first run it on the live CD - but then I change the screen resolution and it's ok.

However, I'm finding it difficult to burn a DVD of my old files before I reformat the HDD - the live CD doesn't seem to want to be ejected (so I can put in a blank one to record onto) - is this normal?
