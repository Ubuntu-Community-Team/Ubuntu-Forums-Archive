### Post by kpkeerthi on 2007-12-13
apt-get/aptitude did not gracefully exit last time. Either you force killed it or it crashed abruptly so it left the lock file (/var/lib/apt/lists/lock) locked. 

Reboot and see if the problem goes away. If not, remove the lock file
```

sudo rm /var/lib/apt/lists/lock 

```

### Post by kpkeerthi on 2007-12-13
Disable IPv6 and see if it helps.

```

gksudo gedit /etc/modprobe.d/aliases

```

Change

```

alias net-pf-10 ipv6

```
to
```

alias net-pf-10 **off**

```

[COLOR="Red"]**Reboot**[/COLOR]
