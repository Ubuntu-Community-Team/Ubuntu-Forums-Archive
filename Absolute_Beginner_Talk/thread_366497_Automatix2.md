### Post by tonyr1988 on 2007-02-21
Just copy / paste the command into your Terminal.

> gpg --export --armor 521A9C7C | sudo apt-key add -

Select in Firefox, right-click -> Copy.

Go to your Terminal, right-click -> Paste (or Ctrl+Shift+V, methinks)

WINE can help run some Windows applications under Ubuntu (not all of them, and a bunch of them don't run the same) - look at [http://appdb.winehq.com/](http://appdb.winehq.com/) for a list of the games and how well they work.

Automatix will install Wine for you, but you'll need to configure it - don't worry, it'll tell you the command you have to type in the Terminal afterwards (it's like winecfg or something similar).
