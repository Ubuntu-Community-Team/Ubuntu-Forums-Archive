### Post by rainwalker on 2008-02-10
Yes, you can dual-boot XP and Ubuntu. I don't do it myself, but I know there are tutorials and guides for setting it up.

There are some games that run through Wine (a program that lets you run some Windows programs on Linux by making them think they're running on Windows), I know WoW can, but it might take a little work to get the others working.

Those are the different flavors of Ubuntu. They're different in what desktop environment they use (basically the way everything looks/works).
Ubuntu uses the GNOME desktop environment, which is geared towards usability, whereas Kubuntu uses KDE which is geared towards looks and customization, and Xubuntu uses XFCE which is lightweight and more for older hardware. Edubuntu is geared towards education and schools, and Gobuntu is aimed at being COMPLETELY open-source. 
I would recommend Ubuntu, because (as far as I know) most people use it and it's easy to use, and because KDE can be really confusing at times.
After you download the disc image (it will end in .iso) you need to burn it to a CD. Check out [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto) for instructions, and [https://help.ubuntu.com/community/CommonQuestions](https://help.ubuntu.com/community/CommonQuestions) for general info about the *buntu's.

Good luck, and welcome to the world of Linux :)

### Post by JoshuaRL on 2008-02-10
I agree with Overdrank, there is a lot of good info there.  But a few quick and dirty answers:
> **Loneless said:**
> can I download ubuntu and still use Windows XP? Like switch between the two OS's when I see fit?
Yep, it's called dual-booting and many people do that.  There's plenty of instruction on it including here:
[https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall)
> Another question,  I've heard from friends that Linux is not geared very well towards gaming. I love playing games, Counter-Strike:Source to be exact, WoW, EQ, etc. Can Linux run any of those?
Some of the most popular are well-run in Wine.  Try the first link to install, second to check what programs run well:
[http://www.winehq.org/site/download-deb](http://www.winehq.org/site/download-deb)
[http://appdb.winehq.org/](http://appdb.winehq.org/)
> Also, at the main ubuntu page I saw different 'versions' I guess you could say, from what I remember there was "kubuntu" "gobuntu" "edubuntu" and "xubuntu" which would I choose to be more 'gamer' friendly?
The difference is what Desktop Environment it uses.  Ubuntu uses Gnome, Kubuntu uses KDE, Xubuntu uses XFCE.  Wikipedia is your friend here.  All will work well if Wine does, Ubuntu is a little better supported, but Xubuntu runs on less system resources so it's a little faster.
> Last question, once I do get ubuntu downloaded, what then? How do I set it up?.
That's kind of up to you, but I would suggest getting the Medibuntu repos, the ubuntu-restricted-extras package, the Helix players (for RealMedia), Amarok/Exaile for a media player, and turn off Compiz/desktop effects when gaming.  Other than that, it's kind of what you want.  Mess around a little in the Add/Remove app or Synaptic to see what there is out there.

Hope that helps.
