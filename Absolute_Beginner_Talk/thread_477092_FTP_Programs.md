### Post by wormser on 2007-06-18
If you are looking for a client then try gFTP.  It is a nice clean layout and looks similar to core ftp for windows.

```
sudo apt-get install gftp
```

You can always use the command line for quick access to an ftp server.

```
ftp mysite.com
```

Just type **help** to get a list of commands.
