### Post by Zill on 2008-02-16
If the Xubuntu Live CD runs OK then I would think that the PC spec is adequate.  If it only hangs while installing then maybe there is insufficient hard disk space available.  Are you trying to dual-boot with an existing Windows installation?

If the Xubuntu Live CD will not run then you could try a smaller system, such as [DSL]("http://damnsmalllinux.org/").
This is a very small OS, less than 50MB, that loads and runs very quickly on older PCs.
