---

### Post by aysiu on 2006-05-12
If you're in Gnome, install *numlockx*.
If you don't know how to install software, take a look at these links:

[http://www.monkeyblog.org/ubuntu/installing.html](http://www.monkeyblog.org/ubuntu/installing.html)
[http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)

If you're in KDE, it's an option in System Settings or the Control Center under Peripherals > Keyboard.
