### Post by catlett on 2006-06-21
Is "recycle bin" just a slip up or aree you talking windows? Well for windows it is easier to set up the application. Just download and double -click to run. [http://www.snapfiles.com/get/restoration.html](http://www.snapfiles.com/get/restoration.html)

There are ways of doing this in linux but it is definately for an advanced user. So justr in case you are an aol-hand at linux here are 2 ways. The first one uses "midnight commander". It is already installed . To call it up just enter ```
mc 
``` at the terminal (you might have to enter gksudo mc" if you have issues about permissions i.e. being root)
The other uses an applications that need to be compiled first. [http://e2undel.sourceforge.net/recovery-howto.html](http://e2undel.sourceforge.net/recovery-howto.html) If you are new to linux, this probably isn't going to be much help. If you or someone you know is linux able then you have a chance to get them.

Here's another how-to [http://www.stud.tu-ilmenau.de/~mojo/undelete.html](http://www.stud.tu-ilmenau.de/~mojo/undelete.html) Hopefully the recovery apps are compatable with ext3. Alot of times ext3 things are backwards compatable so hopefully an ext2 app will be "forward" compatable.

P.S. They are both free. I haven't used any of these so I can't verify if they work or don't. I just thought I'd throw them out there  just in case.
