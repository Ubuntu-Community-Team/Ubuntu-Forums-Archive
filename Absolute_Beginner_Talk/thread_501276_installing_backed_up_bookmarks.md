### Post by loudmouthman on 2007-07-15
Try renaming bookmarks.bak to bokmarks.html ( you might have to rename an existing bookmarks.html ) If Swift remains true to mozilla then your bookmarks file is bookmarks.html


Now whilst its not much help now you might be interested in how I have been handling my bookmarks under Linux/Windows and Mac.


I use Del.icio.us as a bookmarker; if you install the Bookmark toolbar from the site you can TAG and index your bookmarks on the web, such as I have done here  :  [http://del.icio.us/nikbutler](http://del.icio.us/nikbutler)

I then take the RSS feed for my delicious account and subscribe it to my Bookmarks in Firefox. Ths means that every desktop I work at has a live update of my current bookmarks.  

I back up the Delicious bookmarks into Google Docs as a precaution against Delicious folding at any point in the future.

I hope both of these have helped.

### Post by Malta paul on 2007-07-15
Hi,
You can make a backup copy of your 'bookmarks' as follows:
Go to  >bookmarks .Organize bookmarks >file >export   This will save a copy of your bookmarks as a .html file.
To recover your backup bookmarks file just: same as above but >import. 
You can do the same with your email address file.
Hope this is of some help :)
