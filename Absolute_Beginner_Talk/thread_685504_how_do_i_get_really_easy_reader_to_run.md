### Post by thelatinist on 2008-02-02
Nevermind.  It's a java application.  Locate the original (unextracted) jar file and try:

```
java jar your-application.jar
```

If that doesn't work, try right-clicking on the .jar file in nautilus and choosing "Open with 'Sun Java 6 Runtime.'"
