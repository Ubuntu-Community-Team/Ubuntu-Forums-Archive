### Post by John.Michael.Kane on 2008-01-24
> **glasgowk2 said:**
> I am a freshman college student majoring in Computer Science and I was thinking about installing Linux on my laptop but I had a few questions prior to actually going through the process.

1) Would I be able to run programs such as Microsoft Visual Studio 2005 on the new Operating System? I will need this for class.

No. you can not run Microsoft base software directly on Linux/Ubuntu. To do so you will require one of the below methods.

[wine]("http://www.winehq.org/")
 
To find out if your program is supported on wine 
[Working applications  ]("http://frankscorner.org/index.php")

[CrossOver Office]("http://www.codeweavers.com/")

The other options are.
1) Dual boot Ubuntu/windows
2) Run windows in a virtual environment using one of the programs below. 
[virtualbox]("http://www.virtualbox.org/")
[vmware]("http://www.vmware.com/")

> **glasgowk2 said:**
> 2) How efficient is the word processor? I will have plenty of papers to write on the computer.

Some users swear by Open office which is the default processor on Ubuntu, While others like koffice. Then you have those who write their technical docs etc using LaTeX. IMHO it will depend of the depth of the document you have to write. I would suggest you try them all, As they are all installable through synaptic, and see which one fits your needs.

For reference:
[latex-project]("http://www.latex-project.org/")
[koffice]("http://www.koffice.org/")
[openoffice]("http://www.openoffice.org/")

Dual boot. You will need Windows for class anyway, and as a computer major, it is good to be familiar with more than one OS.
