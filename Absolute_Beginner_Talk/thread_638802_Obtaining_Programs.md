### Post by thelatinist on 2007-12-12
That depends on what you're looking for.  Details?

For instance, nearly everything an average user will need can be installed using Add/Remove...from the applications menu (be sure to select all the repositories using the "Preferences" button).

If you know the name of your package you can install it using Synaptic.

Or you can type

```
sudo apt-get install *package-name*
```
