### Post by mrfelker on 2007-08-22
The answer is no (for Windows) Restore (first create a restore point - Windows Help will tell  ;you  how) will only restore system files - no programs or data.

However there is a utility on Synaptic called "testdisk" that's supposed to help with restoring partitions - but I haven't  used it.  Perhaps the best solution (which I have used) is Acronis which will make images of a partition or even an entire HD if you want.  Store the image on a second HD or external HD and you can restore anything.  Not a free program however.
