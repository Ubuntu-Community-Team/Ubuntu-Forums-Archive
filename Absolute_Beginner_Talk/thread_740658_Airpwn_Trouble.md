### Post by paultag on 2008-03-31
airpwn? I have never used it, but I have just downloaded it copy. The prerequs here are these:

  * libpcap ([www.tcpdump.org](www.tcpdump.org))
  * libnet ([www.packetfactory.net/libnet](www.packetfactory.net/libnet))
  * libpcre ([www.pcre.org](www.pcre.org))
  * lorcon (provided in the lorcon-current.tgz archive)
  * openssl ([www.openssl.org](www.openssl.org)) (only needed for the keygen)

so here it is for a compile.

[PHP]
sudo aptitude install libpcap-dev libnet-dev libpcre3-dev
[/PHP]

I was not able to find lorcon in the repos. Check out what he tells you. You need to find the library files that include dev headers, so the app can build from them. ( lib[x]-dev )

Be careful now....
