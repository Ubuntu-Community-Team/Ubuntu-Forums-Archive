### Post by alienexplorers on 2007-05-18
Number one: You do not unpack an .iso file.  You have to bun the .iso image to a CD.  In gnomebaker or K3B you select the option Burn iso image to CD.  If you do this correctly the CD will then be bootable.  Just make sure in your BIOS you have CD as the first item in the boot from menu.
