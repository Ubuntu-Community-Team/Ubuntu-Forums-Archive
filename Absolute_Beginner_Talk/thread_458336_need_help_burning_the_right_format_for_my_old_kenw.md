### Post by Sparkster185 on 2007-05-29
All CD players read the same type of data, regardless of age.

All you should have to do is burn your music as an "Audio CD" and the burning app should convert it to the proper format.  Do note that if you are going to go from MP3->CD Audio, you'll need the proper codecs installed.

In amaroK, assuming you have the codecs installed, you can right-click an album and it will have an option "Burn this album".  Select this and it will open K3b and burn the album (in CD player audio format) for you.
