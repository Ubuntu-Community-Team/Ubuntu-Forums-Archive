### Post by gilforsyth on 2007-06-18
To enable Japanese support in Ubuntu, follow the instructions here:
[https://help.ubuntu.com/community/SCIM](https://help.ubuntu.com/community/SCIM)

In terms of programs that support Japanese, SCIM will allow you to use Japanese with nearly any program in ubuntu.  Japanese is fully supported by OpenOffice for documents, except there is no native Furigana input function.  If you want furigana input, you should check out [Project Itadaki]("http://freshmeat.net/projects/itadaki/").  

In terms of just straight translation of words, Jim Breen's WWWJDIC is probably the best resource available.  You can use the [main mirror]("http://www.google.com/url?sa=t&ct=res&cd=1&url=http%3A%2F%2Fwww.aa.tufs.ac.jp%2F~jwb%2Fwwwjdic.html&ei=NWV2RuGgJZHisALQk8joCA&usg=AFQjCNFKjLAEXTzogrKjsb0s18j0gkWZ_w&sig2=BJRmfWMmn0ca9q8gDhBerQ") or choose one closer to you.  

Let me know if you need more info about using Japanese in Ubuntu.
