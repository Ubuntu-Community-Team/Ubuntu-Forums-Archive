### Post by scientist1971 on 2008-04-20
I'm on ubuntu GG but should be the same method.  There are3 ways to install/ uninstall:
1 type into terminal under the applications --> accesories menu (quick but can be slightly daunting for newbies - I include myself in that too!)
2 applications  --> add/ remove
3 (the method I normally use) goto system --> administration --> synaptic package manager (you will need to enter your password here)--> search (type in program name) --> when found right click on little grey box next to prog name --> mark for uninstallation --> apply
