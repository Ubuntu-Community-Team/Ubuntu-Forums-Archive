### Post by Arand on 2008-03-31
In ubuntu there is "file-roller archive manager", you can run it by typing "file-roller" into a terminal.

Easiest way to create an archive is probably to just select the files/folders (in Nautilus, the file browser) and right-click -> "add to archive".
