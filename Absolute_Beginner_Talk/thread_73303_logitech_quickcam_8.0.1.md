### Post by dean79 on 2005-10-10
it's not hard at all.

 just open the synaptic package manager, for warty it's under SYSTEM>ADMINISTRATION>SYNAPTIC PACKAGE MANAGER. once that's open on the right are loads of different programs you can install, just looked for the one called camsteam and mark it for upload (click on the box to the left of it).

 after that click on the apply button and it will install it all for you:)

 to run the program use APPLICATION>RUN APPLICATION than type in camstream and it should load.

 not sure if you need to install the extra repositaries. if you do try this webpage [www.ubuntuguide.org]("www.ubuntuguide.org") it tells you how to install lots of different things.

 dont know how compatable it will be. mine works but i cant get any of the special functions to work yet:(
