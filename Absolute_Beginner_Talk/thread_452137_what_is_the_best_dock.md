### Post by rstana on 2007-05-23
avant window navigator is quite a good one
more info here:

[http://code.google.com/p/avant-window-navigator/](http://code.google.com/p/avant-window-navigator/)

to install:

Add these lines to /etc/apt/sources.list

```

## Avant Window Navigator
deb http://download.tuxfamily.org/syzygy42/ feisty avant-window-navigator
deb-src http://download.tuxfamily.org/syzygy42/ feisty avant-window-navigator

```

Then do this in a terminal:

```

wget http://download.tuxfamily.org/syzygy42/8434D43A.gpg -O- | sudo apt-key add -
sudo apt-get update
sudo apt-get upgrade

```

### Post by sanicki on 2007-11-08
Kooldock benefits from not requiring Compiz. I use it right now on Kubuntu Feisty. However sampled Kiba and AWN on Ubuntu Gutsy. I think Kiba is further along (physics etc.) but found AWN to be easier to install and more reliable (Kiba seemed to crash a lot when left to sit). Both are more advanced than Kooldock.

If you're looking for a launcher Kooldock's all you need. AWN will get you more eye-candy without too much pain. Kiba seems to be the most popular and bleeding edge. Let us know what you choose.

(FYI, I used the script from [http://www.kiba-dock.org/index.php?option=com_smf&Itemid=30&topic=540.msg2700]("http://www.kiba-dock.org/index.php?option=com_smf&Itemid=30&topic=540.msg2700") to install Kiba...which got it working but could also be the reason its so unstable. Caveat emptor.)
