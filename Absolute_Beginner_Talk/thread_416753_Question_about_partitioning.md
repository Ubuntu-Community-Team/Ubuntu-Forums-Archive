### Post by seshomaru samma on 2007-04-21
If you want to keep XP as your main OS what you can do is this :


10 GB -Windows  , NTFS
10 GB- Ubuntu , Ext3
1 GB- Swap (you need a swap partition for Linux)
The Rest - Data , Fat32 (here you can keep all your files /music / movies or whatever)

Fat32 is a file system which both Linux and Windows are comfortable with, so you can view /edit you data when you are logged in Windows as well as Linux.
