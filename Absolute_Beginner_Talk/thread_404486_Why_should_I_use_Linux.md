### Post by Brightbelt on 2007-06-28
I want to share my experience because I NEVER thought I'd be using Linux. I assumed up front that it would be too hard, too code-oriented and too geeky. I've been using Ubuntu for almost 2 months now, but I also feel the need to qualify that.  

I'm still a dual booter for the most part. I still have a few critical, professional applications that only Windows can run, but there are also several pro apps that Ubuntu (thru CrossOver and Wine) is handling quite well. I've been pleasantly surprised more than a few times.

I have an older laptop that I still use for travel and I have only Ubuntu Feisty installed on that. But here's my sense of all this:

Look at it like a Project, perhaps a major one. Start off with a dual boot. That way you really don't lose anything.
There is a learning curve to this and it helps to enjoy trouble-shooting to some degree. Think of it as something you will whittle away at, over time. At least that's how I view it.

I still don't know if I'll make the TOTAL switch. I use Pro Audio programs for work on my computer; I do midi charts with the Finale program. On one of my web accounts (my only one), I still lean on Frontpage 2003 to do certain scripts because I don't know how to create ASP or CGI scripts, which I'd have to write using Dreamweaver. 

But when Vista came out and yes, I've got it installed on my dual-booting desktop along with Ubuntu, I saw more clearly how MS worked and how they've strong-armed the industry to match their standards or lack thereof. The amount on money I've put into Vista to make hardware and software purchases to be compatible with it is staggering. 

Also, they knowingly put out an OS that couldn't handle PDF files correctly. That blew my mind. The PDF file is a business industry standard. And their threat to sue corporations that use Linux was the final straw for me.

So I'm whittling away at this project and maybe, in a year or two, I can cut the chord. 

Good Luck, Frank

### Post by anewguy on 2007-07-11
I know you've seen a lot of answers, including those like mine, but here goes:

- to see if I could

Simple as that.  Then I decided I didn't like dual-booting so I found I could run a virtual machine (in my case VMware, but I've seen mention of VirtualBox and something else as well).  With the virtual machine I am still running Linux and then within that I run Windows in the virtual machine.  This gives me access to those things in Windows that I use.

I am not a game player, so I don't have problems that way.  I have read that some get their games working in Wine, others don't.  I'm pretty sure you probably can't run many games in the virtual machine Windows since it is emulating a video card.  I would imagine that first the game wouldn't know the exact video card you are running, and that secondly it would probably be slow and jumpy since it is emulation.  Wine might get around that, but I don't have any idea.

Once I got here, I decided to try staying for several months and see if everything goes okay.  If so, I'll probably just stick with Ubuntu since there would be no reason to go back to just Windows.  I may be one of the lucky ones, but my HP Deskjet 3915 and my HP 4200C scanner both worked in Linux without me having to do anything more than just add a printer and make it the default, just as you would in Windows.  It's all been pretty smooth for me.

As someone else said, though, if you are comfortable with Windows and what you are doing with it, there is no "real" reason to switch - it's all just personal view and preference.

Have fun no matter which way you chose!:)
