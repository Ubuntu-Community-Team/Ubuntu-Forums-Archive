### Post by sayakb on 2008-03-04
@OP
Either you can install kubuntu-desktop as indicated or make a large panel that would mimic the KDE panel. Download the KDE logo as png, and open the active theme folder you have in gnome "/usr/share/icons/themename" and under each icon size, open the Places folder and add/replace the logo as "start-here.png" (without quotes).. Now to the bottom panel, add the main menu applet to get a KDE-like menu..
