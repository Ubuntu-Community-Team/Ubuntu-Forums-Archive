### Post by MasterOfDisaster on 2007-02-23
In System -> Administration, you should find an application called Synaptic.  Run this, and you can access about 18000 programs install, all without the command line.

To access wine, run 'wine yourprogname.exe' in the terminal.  To change wine settings, run winecfg in terminal.  For more thorough information, read through the information on the wine website.
