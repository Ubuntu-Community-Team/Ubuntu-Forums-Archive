### Post by nocturn on 2007-07-04
> **moredhel said:**
> explain these agent things please?

Also, it's worth mentioning that there is little reason to use pidgin or gaim or kopete etc if you only use one protocol.

Kinda of obvious, but gaim is default, and so you could be using it when using a specific one like amsn for just msn would be better.

Agents are installed on the Jabber server and form bridges between Jabber and other networks.  You need to enable them and you need a login to any of those networks, but not a client that supports that protocol.  A jabber client will do.

I think this explains it better: [http://blogoscoped.com/archive/2006-05-02-n79.html](http://blogoscoped.com/archive/2006-05-02-n79.html)
