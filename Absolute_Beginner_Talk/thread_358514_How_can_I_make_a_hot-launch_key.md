### Post by jpeddicord on 2007-02-11
Right-click the panel and select **Add to Panel...**. Then click the Custom Application Launcher button.

Type in beryl-manager for the command, and Beryl for the name. Pick a custom icon for it (it doesn't have to be the Beryl icon).

### Post by Jussi01 on 2007-02-11
The other option is to make beryl start up with your computer. go to system -> preferences -> session, then the startup programs tab and add beryl manager to that.
