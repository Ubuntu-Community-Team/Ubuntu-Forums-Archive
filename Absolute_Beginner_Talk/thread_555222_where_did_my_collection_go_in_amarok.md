### Post by Maxtorm on 2007-09-19
You can hold down the Ctrl key to select multiple artists/albums to drag over to the Playlist. If you're intent on having all songs from the Collection in the Playlist, use Ctrl-A to select all of them and then drag them over.

### Post by mridkash on 2007-09-20
To see your entire collection like the library on iTunes, Go to the playlists tab in amarok, open smart playlists, then collection, then right click on all collection and select load.

This solves ur problem probably.
