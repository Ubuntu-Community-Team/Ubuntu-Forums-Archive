### Post by Shay Stephens on 2006-12-04
Make a backup before you install anything.

After making the backup, when you install ubuntu, it will ask you to setup the partitions.  If you mess that up, you can kiss the windows drive bye bye.  Then you will be very happy to have the backup :D 

If you do it right, you can make room for ubuntu and dual boot with windows.  Both can live happily on one hard drive.
