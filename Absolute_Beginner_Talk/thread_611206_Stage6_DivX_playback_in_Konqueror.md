### Post by FuturePilot on 2007-11-13
Try installing the mplayer plugin  or the VLC plugin. Whichever one you choose, just make sure that Konqueror knows where it is. Go to Settings>Configure Konqueror and go down to Plugins. There should already be an entry for /usr/lib/firefox/plugins/ which is where the plugins will be installed to.
