### Post by jerrylamos on 2007-05-05
Some people have found Feisty worked better with their wireless, some people found wireless broke.

I have a system with an older Sony LCD that crashes on boot with Dapper but runs fine on Feisty.  I also have another system that didn't boot Feisty until I recabled CD Rom and IDE hard drive.

Some people found Feisty Network Manager to be a help, many people (like me) are worse off.

There are a bunch of Feisty bugs - see my posting "Workarounds" in "Installation & Upgrades" that affect some people, but not others.

This is from a test system which is quad booted, Win 98, Dapper, Feisty, Gutsy.  

A typical thing might be to try CD Live Feisty to see how it goes with your setup.  This worked better with Edgy and Dapper which have a "persistent" mode which saves settings and package changes, however that's broke on Feisty.
