### Post by FuturePilot on 2007-10-08
gThumb. It's already installed. It does everything you're looking for. Possibly more;)
You might have to make it visible in the menu though. So right click on Applications and go to Edit Menus. Go the the Graphics section and check off gThumb. you can also make gifs open automatically in gThumb by right clicking on a gif and going to Properties and to the Open With tab and select gThumb

### Post by Cannaregio on 2007-10-08
If you like the command line as much as I do, or if you want to discover how mighty powerful a command line application can be, try and enjoy [COLOR="Blue"] feh[/COLOR].

> feh is a fast, lightweight image viewer which uses imlib2. It is commandline-driven and supports multiple images through slideshows, thumbnail browsing or multiple windows, and montages or index prints (using TrueType fonts to display file info). Advanced features include fast dynamic zooming, progressive loading, loading via HTTP (with reload support for watching webcams), recursive file opening (slideshow of a directory hierarchy), and mouse wheel/keyboard control.

This said, feh is good for almost every image, large or small, but is no good for animated gifs, lol :-)
