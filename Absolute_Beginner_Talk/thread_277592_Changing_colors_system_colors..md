### Post by NewRubberSoul on 2006-10-14
When I was using Xubuntu I had the same problem and it drove me nuts.  I'm sure that there's probably some way to fix it, though I'm not sure how to do it.  What I did was just found a new theme that worked better.  It's a shame to do that, but it would be a sure fix.

Otherwise, try this.  Open Firefox, go to Edit > Preferences. Click on the "content" tab at the top.  Under "Font and Colors" click the Colors button.  It should bring up your options.  Note that when I first opened the preferences window, I had to stretch it to make it larger because otherwise the colors button was barely hidden.  I don't know if you'll have the same issue.

I hope that this helps.
