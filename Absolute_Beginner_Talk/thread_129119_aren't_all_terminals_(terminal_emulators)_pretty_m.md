### Post by wrtrdood on 2006-02-15
Then there's those of us that use the terminal a LOT and can't stand the lack of configuration and ugly fonts the minimalist versions provide.  Xterm in particular.  They all basically emulate the ANSI standard so full screen editing and addressing work without complication.  Much easier to deal with than the actual alphanumeric terminals they are emulating, anyway.  Personally, I prefer the konsole from KDE because it allows me a lot of configuration flexibility.  I'm particularly fond of the the tabs so I don't have a lot of windows scattered about.
