### Post by NikoC on 2007-04-02
First remark: you don't need to restart at all after you've installed an application! Second: you are right, often there are no shortcuts placed into the main menu. If however you push Alt+f2 and type the name of the application (autocompletion here) and enter the app will startup. You can add it yourself to the main menu by clicking the right mouse button on the main menu and selecting 'Edit menus' --> new item can be added.
