### Post by jdong on 2007-10-21
> **RobertMills said:**
> Found MonoDevelop from Osalt.com, any idea how this compares to eclipse??


MonoDevelop is primarily for, well, developing on Mono (a "clone"/implementation of the ECMA C# / Common Language Runtime, similar to MS .NET), while Eclipse is a generic framework for developing in arbitrary languages.

Any of these tools would be good for programming, so pick whichever framework you are more familiar with, or feel more comfortable with.
