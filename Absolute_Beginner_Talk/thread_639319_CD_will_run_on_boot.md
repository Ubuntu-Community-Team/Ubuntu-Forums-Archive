> I think this is the reason that my Ubuntu CD won't run when I boot up my computer, but I don't know how to fix it. Any advice?

1) Getting into the BIOS:  Del, Esc, and F2 are the main ways to get into the BIOS.  F12 on Dell is a one time change.  Once you start up keep pushing the button until you find which one goes into the BIOS.   Once there, just set the CD to boot first.
