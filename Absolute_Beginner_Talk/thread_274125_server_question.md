### Post by bswilson on 2006-10-09
**Any** Linux distribution can run as a server.  And remember, Xubuntu is just like Ubuntu with some packages removed and a different window manager.  All you have to do is enable or install some server software on your box to make it a 'server'.

### Post by thornomad on 2006-10-09
YOu can also download the LAMP installation (That is, the actually SERVER installation disk) that will install Linux (Duh), Apache, Mysql, and PHP and get them all running for you.  

[http://www.ubuntu.com/download](http://www.ubuntu.com/download)

Choose a mirror, and then use the Server Install CD -- choose a LAMP installation.  This will not get you a GUI ... but you can get the Xubuntu graphic interface for certain. 

The other route would be to install Xubuntu, then add all the server parts you need one at a time.
