### Post by hyper_ch on 2007-08-24
well, I don't think you need a book to start with. I recommend having a read here:

[http://ubuntuguide.org/wiki/Ubuntu:Feisty](http://ubuntuguide.org/wiki/Ubuntu:Feisty)

and here:

[http://www.psychocats.net/ubuntu/](http://www.psychocats.net/ubuntu/)

and for the command line, I think this is a good list of the most common commands:

[http://fosswire.com/wp-content/uploads/2007/08/fwunixref.pdf](http://fosswire.com/wp-content/uploads/2007/08/fwunixref.pdf)

In order to know how else you can use those, use the man pages. E.g. if you want to know what else you can do with tar, just enter this in the terminal:

```

man tar

```
