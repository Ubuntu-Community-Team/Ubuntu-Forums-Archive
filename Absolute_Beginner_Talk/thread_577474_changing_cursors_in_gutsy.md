### Post by techno-mole on 2007-10-16
granted filing a report with launch pad would be a good idea, but from my experience more than a few people will have encountered the same issue and will have either started a new thread or posted in an existing thread on the forums.
as it turns out the cursors can be changed by using this method - system - preferences - appearance, then click on the customize button and select pointers, by selecting a different pointer the cursor is changed, this changes the cursors.
however i have noticed that when you run gcursor nothing seems to happen when you click to install a cursor set, the same goes if you click open go to theme folder, so how do you install new cursors ?
well ive been trying to change my cursor, and i tried the click to install and the click to go to theme folder and niether worked, but i found i had actually installed the cursor by dragging it to the gcursor window, it just didnt say anything had happened, but by doing that it had installed the cursor set.
hope this helps (just to let people know that a report has already been raised with launchpad regarding gcursor not working)
