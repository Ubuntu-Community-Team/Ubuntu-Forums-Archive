### Post by Jave27 on 2006-01-10
Learn C/C++ and the Windows API, and contribute code to the Wine project.   ;) 

Or, every time you purchase a game, email the vendor and ask for a Linux build or ask them to work with the Wine project or Cedega.  Or, refuse to purchase games that don't run under Linux.  But, don't just complain about it, make sure you request it so that vendors know there is a market for their extra efforts.  Quite a few game engines run in Linux now, so multiplatform isn't really tough to do if you plan for it early on in your development.
