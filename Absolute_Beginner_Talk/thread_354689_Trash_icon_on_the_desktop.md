ok, here is the long route (but it wil make your system menu visible too):
-right click on the application or main menu and select edit menu
-you will see all the menu categories such as accessories, graphics, etc. make the system menu visible. close the menu editor.
-in the system menu, you will see an application called configuration editor. click on that
-in configuration editor, do a search for "nautilus". it will bring up a list. the 2nd one down should be apps/desktop/nautilus.
 you can set the trash icon to appear on your desktop there.
