### Post by welsh_spud on 2005-12-12
Well, the first thing you need to do is buy a DVD burner (if you haven't already). websites like newegg.com are great for hardware such as that.

Once you have it and are ready to install the physical DVD burner, you need to open your computer case and plug the DVD burner into an open plug. I suggest watching someone do this first, as it is very hard to explain how to do so here.

Once your DVD burner is plugged in, turn your computer on and Breezy should have detected it automaticaly.

Software-wise you are spoiled for choice. You can just use the file manager (nautilus) to put things on DVDs (much in the same way as you put things on CDs). There is a good program called Gnome-Baker which can burn things onto a DVD or CD. If you are using Kubuntu you can try K3B which is a very good disk burning program.

To install any of these programs, open up Synaptic from System > Administration > Synaptic Package manager. Search for K3B, or whatever program you want to use then select it and click the Apply box.

Have fun
