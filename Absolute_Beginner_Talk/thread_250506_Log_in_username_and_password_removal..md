### Post by a.v.l on 2006-09-04
Since installing Ubuntu 6.06 I am prompted to type in my username and password.  It's no real problem, but I would prefer if I didn't have to do this each time I start my PC.  How can I change this?

Alan

---

### Post by Anonii on 2006-09-04
System -> Preferences -> Users

In one of the tabs on this window, you will find automatic login or something. (Sorry I'm on KDE, right now :/ ) Add your user, and you wont be getting the Username/Password dialog in the startup.

### Post by tilai on 2006-09-04
Click on System - Administration - Login Window. Type password. Click Security tab - Enable Automatic Logon
