### Post by timfrost on 2005-12-25
Use Partition Magic to shrink the size of  an existing partition, leaving free space at the end of the disk. 

The Ubuntu installation will detect the free space, and offer an option of using that space for the Ubuntu partitions.  Ubuntu needs two partitions, which it can create during the installation.  One partition will be the OS, and the second partition will be used as "swap space" - which is similar to Windows pagefile, except that it is a separate partition.

### Post by Mustard on 2005-12-25
Ubuntu will need probably 2.4GB to install just the operating system, plus you need extra space for storing whatever you download and install once you have ubuntu running.  That should give you a rough indication, if you are tight for space.  If you have lots of space, then as much as you want really.
