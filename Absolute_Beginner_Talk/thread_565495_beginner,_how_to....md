### Post by hyper_ch on 2007-10-03
> **sgtbob said:**
> I've been told by others that since Linux is not in as wide spread uses as Windows, that the hackers don't attack it as frequently - if this be the case isn't it just a matter of time before they begin to do so, if not already?

that's a littly myth.... if the # of installs would matter, than apache would be the most hacked webserver... however that is not the case. Apache is quite save... however IIS has a lot more problems. Furthermore, if you hack apache, you only get rights as to what apache can do.... if you hack IIS you basically have admin privileges to that computer as on windows everything is combined so heavily...

Windows was not build for a multi-user environment while Linux is...

Also have a read here:  [http://www.psychocats.net/ubuntu/security](http://www.psychocats.net/ubuntu/security)

Addon: I don't mean to say that Ubuntu/Linux is totally save... there is malware out there but by the nature of its design Linux is much less prone to become such an attractive target as Windows is... and in the end, you as user, can also contribute a lot to enhance/decrease security yourself...
