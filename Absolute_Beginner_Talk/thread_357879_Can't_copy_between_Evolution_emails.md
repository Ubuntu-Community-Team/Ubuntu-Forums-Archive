Also, as before, the 'paste' option is not available via the drop down menus.

I've found a way round this problem, which involves pasting text from an email into the text editor, highlighting and copying it a second time, and then pasting it into the target email. It works, but it isn't exactly elegant.

### Post by bapoumba on 2007-02-10
Hello,
highlight text from message B. Do not close the window. Go to message A, middle click with the mouse to paste ;)
If you want to close the message B window first, then CTRL-C to copy, CTRL-V to paste.
