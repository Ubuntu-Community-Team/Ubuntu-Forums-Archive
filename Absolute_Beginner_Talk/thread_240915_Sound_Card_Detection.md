### Post by Major_Stitch on 2006-08-21
Hope this helps!

Go System->Preferences->Multimedia Systems Selector (If you don't have it than use alacarte ["Right-Click" on Applications -> Edit Menus...]).

From there try the "Test" button underneath Default Output Plugin. If you get a sound than the the problem isn't in the hardware but a broken installation. If you don't hear anything than try to change the plugin or search the "Hardware" part of the forums. 

Good luck!
