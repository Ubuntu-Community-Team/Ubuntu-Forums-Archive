### Post by lila on 2006-02-05
Install what? Ubuntu or a program?
For a programm, use synaptic package manager which you find in system->administration.  You need to activate sources lists first, there is a good thread about that going on on this forum right now, unfortunately I've forgotten the name, just look around a bit, or search using the search tool at the top of this forum.  In synaptic, once your sources are updated, you just click what you want from a list of programs with short descriptions.

To install Ubuntu you need a CD which you download (I assume you have internet access, as you access this forum) as an .iso - burn as a CD image.  You can also get them by post, but apparently it takes a few weeks.  
You then boot on the CD and follow instructions.  It's really straightforward.  Except when, for example, the computer won't boot on CD, as I am experiencing right now...
I hope this helps, if you have problems please ask, people here are really helpful, and I've asked them a lot since joining this...
Lila
