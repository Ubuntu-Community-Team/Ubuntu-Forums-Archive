### Post by lloyd_b on 2006-10-21
I would recommend against your running the server version.

The desktop version DOES have somewhat lower performance.  However, it also has a LOT of easy-to-use tools (much like Windows) to handle both configuration and routine tasks.

The server version doesn't even have a desktop - everything is done from a command line.

For a Linux newbie, the ease of use of the desktop version far outweighs the performance advantages of the server version.
