### Post by yabbadabbadont on 2007-06-07
As far as I know, you can't do that unless you prevent nautilus from drawing the desktop, including the icons.  If you run the glmatrix program in a terminal window with the "--help"  (two dashes) option, it will show all the options.  I'm pretty sure that one of them is "window"  (don't know if it needs one or two dashes before it).  If you pass the word "root" as the parameter to the "window" option, it should display itself in the root window.
