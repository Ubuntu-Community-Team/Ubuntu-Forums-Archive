### Post by gruhland on 2008-01-19
For me it was the same. I turned down the windows software and looked for a photo management tool. The best I found was digiKam, especially if you are looking for keywords in your database it is easy to use.
The bad thing is (not only for digiKad but all photo databases) that all programs often use different fields in the IPTC informations. For example some programs use "author" for gthe photographer, others use the "copyright" field or whatever. The only field which is used most times in the same manner is the "keywords" field. There is no real standard for the usage of the information fields. But beneath these problems digiKad is useful.
