### Post by Forrest Gumpp on 2008-03-01
I note the 'Take Screenshot' option obtained through Applications--> Accessories--> Take Screenshot now has a delay function.  It didn't have this in Ubuntu 6.06.  I notice it defaults to 0 seconds (no delay) each time you close the application.

If that really bugs you, install KSnapshot, which works in both KDE and Gnome desktop environments.  With KSnapshot the delay stays set until you change it.
