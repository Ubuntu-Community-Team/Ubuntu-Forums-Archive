How can I find out what kernel architechture I have?

---

### Post by Xian on 2005-11-12
Run the command below from a terminal.
Then look for the suffix at the end of the kernel name.

```
$ uname -r
```

```
$ uname -r
2.6.12-9-k7
```

That is a K7 kernel.
