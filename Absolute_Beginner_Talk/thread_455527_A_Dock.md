### Post by starcraft.man on 2007-05-26
> **gamgee911 said:**
> I've seen a few docks but they require xgl/compiz to work and I am using my PowerBook with a Nvidia card so xgl currently isn't a possibility (unless there's something i don't know about).  Is there a dock that works on ubuntu without the xgl/compiz?

The docks work (I use that word VERY loosely, they are still in my experience unstable or at the worst very difficult to install/setup/use daily) with AIGLX I believe. So you can install  NVidia drivers and then put on Beryl/Compiz, and then install a dock. I don't recommend using any dock though, like I said, they all have problems. When I find one thats really stable and good, I will mention it (and yes, I've tried all 5 of the main ones).

If you really must get a dock, search forums for kiba or AWN. Be ready for serious time/effort put in to make sure it works (if you have problems). Do the dock only _after_ you have drivers/beryl installed.

This is a guide to install [beryl]("http://ubuntuguide.org/wiki/Ubuntu:Feisty#How_to_install_Beryl_.28Nvidia.29") Install Nvidia drivers before Beryl, and then do the dock if you want.

And use [Envy]("http://www.albertomilone.com/nvidia_scripts1.html") script to get Nvidia drivers installed, just follow instructions at top after installing deb 0.93 (double click to install, then go Applications > System Tools > Envy). Then select to automatically install the nvidia driver.

I reiterate, I strongly recommend against a dock. Try Beryl, you will be satisfied :). The docks aren't worth the frustration, the panel at your bottom of the screen will never crash/give you errors.

### Post by homergreg on 2007-05-28
Gdesklets provides a nice "dock" bar, with the icons growing as you scroll over them, no need to have compiz or beryl.   Gdesklets is in the add/remove software area, when you launch the gdesklets, you will be presented with all kinds of desklets you can launch, look under the toolbars/launchers area and there is a dock bar that you can configure.  I actually like it better than the compiz/beryl bars myself, plus I can't run google earth with compiz or beryl, but I can with this!  Also a lot of other eye candy for your desktop.
