### Post by Inxsible on 2007-06-02
Visit [www.gnome-look.org](www.gnome-look.org) and download a few themes. Installation procedures are often listed in files called INSTALL or README within the tar.

You can also install eye-candy programs like 
1) Beryl
2) Docks such as : KoolDock, AWN, KibaDock, GnomeDock etc
3) Gnome look will also have different wallpapers that you can download

It also depends on your creativity. Fire up GIMP and create something for yourself. There are lots of ways to customize a lot of things.

Grub can be customized too. [http://gag.sourceforge.net/](http://gag.sourceforge.net/)

 Depends on what you actually want to do.
