### Post by nickburns on 2006-11-28
Make a copy of your xorg.conf file just in case:

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.old

I think you need to add another input device in the file:

sudo gedit /etc/X11/xorg.conf

and add the following block after the Configured Mouse section:

Section "InputDevice"
	Identifier	"Synaptics Touchpad"
	Driver		"synaptics"
	Option		"SendCoreEvents"	"true"
	Option		"Device"		"/dev/psaux"
	Option		"Protocol"		"auto-dev"
	Option		"HorizScrollDelta"	"0"
EndSection

and add this line in bold to the end of the file:

Section "ServerLayout"
	Identifier	"Default Layout"
	Screen		"Default Screen"
	InputDevice	"Generic Keyboard"
	InputDevice	"Configured Mouse"
	InputDevice     "stylus" "SendCoreEvents"
	InputDevice     "cursor" "SendCoreEvents"
	InputDevice     "eraser" "SendCoreEvents"
**	InputDevice	"Synaptics Touchpad"**
EndSection

Save the file and restart x (ctrl+alt+backspace  --> then wait 10 seconds for restart)

Let me know how it goes...
 

if it makes no difference or something goes wrong you can get the backup conf with:

sudo cp /etc/X11/xorg.conf.old /etc/X11/xorg.conf

### Post by nickburns on 2006-11-28
you can:

sudo gedit /etc/X11/xorg.conf 

and undo everything we did. Or if you copied the file like I mentioned in the earlier post.

sudo cp /etc/X11/xorg.conf.old /etc/X11/xorg.conf 

will get you back to the previous file (before we started)
