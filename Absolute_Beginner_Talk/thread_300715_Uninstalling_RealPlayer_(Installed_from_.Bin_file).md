### Post by edmund on 2006-11-16
Yes, delete the folder
/home/milad/Desktop/RealPlayer

I have not used Automatix, but an easy alternative is:

Go to 
    [http://www.real.com/linux?pcode=rn&am](http://www.real.com/linux?pcode=rn&am)
and download the RPM package, rather than the bin file.
Then enter:
    $ sudo alien RealPlayer10GOLD.rpm
to convert the .rpm file into a .deb file.
Then double-click on the .deb file to activate the Package Installer and install realplayer in the right place and in the menus.
