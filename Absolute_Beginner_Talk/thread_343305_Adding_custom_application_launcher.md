### Post by CroEragon on 2007-01-21
Im a bit confused, do you want to start application with system or want to modify launcher? Maybe its because English is not my native language.
Anyway if you want to launch command that starts program on startup just go System -> Preferences -> Session -> Startup Programs tab -> Add -> Type in command that will start gdesklets (most likely gdesklets but maybe something different) -> OK
Thats it, gdesklets will be launched on startup from now on.
