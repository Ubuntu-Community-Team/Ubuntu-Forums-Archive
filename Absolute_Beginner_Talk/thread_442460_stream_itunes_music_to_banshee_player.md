Can't remember if Banshee does it, but Rythmnbox def does. You might hvae to enable it in the plugins section of Rythmnbox. The protocol is called DAAP if you have problems. If you have problems, you can install Tangerine on your Windows PC - this uses the same system to get music but doesn't need iTunes open.
