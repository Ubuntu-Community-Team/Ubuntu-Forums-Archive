I have been using Ubuntu on and off over the past year, so I'm not a total beginner, but just today, something really screwy happened. 

I went to System_>Preference->Session and checked off the option to "Automatically Save Changes to Session", well, shortly there after, my computer locked up for an unknown reason. 

Now, when I boot into Ubuntu, I have nothing on my desktop. My linux desktop folder in the home directory is still there and intact, but nothing is displayed. Additionally, I can not drag links to it from the file browser.

### Post by Buildscharacter on 2007-04-02
Well, if this happens to anyone in the future, just delete the default session in the session control panel and restart your computer. Bingo. That worked. Whew.
