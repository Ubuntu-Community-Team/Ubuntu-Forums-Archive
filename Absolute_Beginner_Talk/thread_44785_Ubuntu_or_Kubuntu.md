### Post by poofyhairguy on 2005-06-27
[QUOTE=thoffland]
Are there conflicts by using KDE with Ubuntu?[/QUOTE]

The biggest problem is that the apps of both will flood the menus of both. If that does not bug you everything else is fine.

> I thought Kubuntu was based on Ubuntu... yet they seem very different.

Thats because KDE is like a different world compared to Gnome. I like Ubuntu better and I like Gnome better and I think its best for new users....but KDE is more popular in the Linux community.

Our KDE is a very raw KDE. Most major KDE distro create many GUI tools to do things. As a side effect, KDE lacks some basic GUI tools (unlike Gnome). Ubuntu is big on using only the free tools that are availible (aka not special, closed stuff other distros have), so our KDE is a little harder to use.
