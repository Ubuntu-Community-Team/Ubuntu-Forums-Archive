### Post by aysiu on 2006-04-15
[QUOTE=babel]
I certainly do **not** want to erase my existing files, programs & Windows 98.[/quote] If you want to be 99% sure everything's okay, follow this dual-boot guide. If you want to be 100% sure everything's okay, back up all your important files and *then* follow this dual-boot guide:

[http://users.bigpond.net.au/hermanzone](http://users.bigpond.net.au/hermanzone)

> 
The envelope that the CD's came in says to use "expert installation mode" if I do **not** want to erase my existing files. Yeah, it's misleading. I advise you actually **not** to do the expert install. Follow the dual-boot guide aforementioned. If you do expert mode and you're not an expert, you'll end up with weird administrator problems. Trust me.
