### Post by Bachstelze on 2006-10-24
Try this :

```

sudo -i
mv /etc/apt/sources.list /etc/apt/sources.list_backup
echo "" > /etc/apt/sources.list
apt-get update
mv /etc/apt/sources.list_backup /etc/apt/sources.list
apt-get update
```
