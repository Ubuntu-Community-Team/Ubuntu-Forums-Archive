### Post by dstew on 2007-03-13
Hi Stormweaver. You are quite ambitious, and I wish you well in your adventures.

In my opinion, if you have never set up a linux system before, make a box and install an Ubuntu system to get used to it. If you want stability, use 6.06 Dapper, but 6.10 Edgy is getting pretty good. Feisty may be a bit much.

I think it is best to get a simple system going, and then start to add things to it, get more complicated. If you try to get a complicated system going right from the start, you tend to get frustrated. There is a lot to learn about Ubuntu/linux, especially if you want to do things like multiple video cards, or simultaneous desktop/server systems. Just start with a desktop system, learn the ropes, then go big.

### Post by Songwind on 2007-03-15
For NAS you mostly need SAMBA (if you're sharing with Windows boxes) or NFS (if only other Linux/Unix servers).

Apache is an HTTP server
MySQL is a SQL database
PHP is a server-side perl scripting language for the web.

So none of those are necessary for a NAS server.
