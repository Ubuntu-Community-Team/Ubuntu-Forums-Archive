### Post by mikewhatever on 2007-11-19
System>Administration>Login Window, Security Tab, tick Enable Automatic Login and specify the user.

---

### Post by Paul820 on 2007-11-19
System->Administration->Login Window, click the security tab and then check the **enable automatic login** box.

---

### Post by duruttye on 2007-11-19
System - Administration - Login window - Security - Enable Automatic Login, select the user you want to login.

And you're done.
