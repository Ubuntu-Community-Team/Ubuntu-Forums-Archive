### Post by jasay on 2006-01-27
I think the package 3ddesktop is a 3-D desktop switcher, rather than a 3-D desktop itself.  Run '3ddesk' in a terminal and use the arrows and space or the three mouse buttons to navigate the desktops and pick which one to work in.  I'm sure it would be more useful as a launcher on your panel than running it through the terminal.
