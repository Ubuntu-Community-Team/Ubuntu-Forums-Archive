### Post by srd on 2007-10-19
Live CD 
 
install menu click F6
 add this : noapic
click enter

install hangs:

click alt+f1 then
     sudo -s
     killall gdm
     rmmod bcm43xx
     apt-get update
     apt-get install xorg-driver-fglrx
     depmod -a
     aticonfig --initial
     exit
     gdm start

install ubuntu 
restart 
recovery mode
    rmmod  bcm43xx
    nano /etc/modprobe.d/blacklist

insert this : blacklist bcm43xx

    dhclient eth0   
or 
   dhclient3 eth0
    apt-get update
    apt-get install xorg-driver-fglrx
    aticonfig --initial
    reboot

Install ndiswrapper 
in ubuntu open console 

    sudo apt-get update
    sudo apt-get install ndiswrapper-common
    sudo apt-get install ndiswrapper-utils
    sudo apt-get install cabextract

Download this windows drivers from HP (sp34152.exe)
 
    mkdir sp
    cd /sp
    mv sp34152.exe /sp
    cabextract sp34152.exe
    sudo ndiswrapper -i bcmwl5.inf
    sudo ndiswrapper -l
    sudo ndiswrapper -m
    sudo modprobe ndiswrapper
    sudo gedit /etc/modules

insert this : ndiswrapper
    sudo reboot


suspend works, download and install latest ati drivers 8.01
method 2
[http://wiki.cchtml.com/index.php/Ubuntu_Gutsy_Installation_Guide](http://wiki.cchtml.com/index.php/Ubuntu_Gutsy_Installation_Guide)
