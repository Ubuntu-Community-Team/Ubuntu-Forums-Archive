### Post by tombott on 2007-10-30
For Cube Effect download CompizConfig

```
sudo apt-get install compizconfig-settings-manager
```

Then under System - Preferences - Appearance - Visual Effects

Select - Custom

Then click preferences and select Desktop - Rotate Cube

Hopefully that should do the trick.
