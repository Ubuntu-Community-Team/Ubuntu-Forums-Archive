```

sudo apt-get install nvidia-glx
sudo nvidia-glx-config enable
```

You will know they are installed when you'll have a white screen with the nvidia logo when the X server will load.

You could look in your /etc/X11/xorg.conf file to check that the Driver line in the Device section says **Driver    "nvidia"**. Or you could run **glxgears -printfps** to see if you're getting a decent framerate. Or you could check the scary-looking /var/log/Xorg.0.log to see if it lists any problems. Actually, if you're using the 64-bit version, you may struggle anyway.

### Post by dstan on 2006-10-29
What I did to get it working after installing nvidia-glx was in the section of the xorg.conf file change the "nv" to "nvidia"  like

Section "Device"
        Identifier      "NVIDIA Corporation NV31 [GeForce FX 5600]"
        Driver          "nv"
        BusID           "PCI:3:0:0"
EndSection

Section "Device"
        Identifier      "NVIDIA Corporation NV31 [GeForce FX 5600]"
        Driver          "nvidia"
        BusID           "PCI:3:0:0"
EndSection

Hope this helps
