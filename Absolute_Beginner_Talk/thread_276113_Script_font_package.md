### Post by The Mekon on 2006-10-12
I had this problem with Scribus and after a lot of searching found the following [site]("http://www.searchfreefonts.com/") with lots of Script fonts.  Not all of them free.

Download a free font, extract to a ".font" folder in your home directory (you will have to press Control H to see the hidden folders}

Now open Scribus without a document selected. Click Setting/Fonts and then Additional paths.

Type in your local font path - in my case /home/brian/.fonts and you should now be able to use your new font.

Unfortunately Edwardian is not a free font but Amaze is pretty nice.
