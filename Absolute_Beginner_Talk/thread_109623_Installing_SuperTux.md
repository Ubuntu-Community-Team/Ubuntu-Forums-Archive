### Post by Lord Illidan on 2005-12-29
Well, you could have installed supertux via apt-get, if you enabled the repositories.

However, if you have smeg, the gnome menu editor, you can add an entry, and make the command supertux, so that when you click it, you start the game.

### Post by mal on 2005-12-29
I would really encourage you to load your applications from the Ubuntu repositories using synaptic (or apt-get).  This will give you a better chance of a smooth installation with appropriate menu entries, etc.

Supertux is in the Universe repository.  I suggest you refer to the Ubuntu Starter Guide ([http://help.ubuntu.com/starterguide/C/faqguide-all.html]("http://help.ubuntu.com/starterguide/C/faqguide-all.html")) for information on how to enable the repositories and load various applications.
