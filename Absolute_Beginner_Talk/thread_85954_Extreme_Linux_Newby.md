### Post by Kyral on 2005-11-04
And just to spare you some confusion (maybe it will sound confusing, but trust me), this is Ubuntu Linux (Kubuntu, Edubuntu, and Xubuntu are basically the same as Ubuntu, for all intents and purposes). There are different flavors of Linux (Debian, Mandriva, Gentoo, MEPIS, Knoppix, Fedora, Slackware, etc etc etc)

Saying "Latest version of Linux" usually will refer to the Linux Kernel (version 2.6.14 right now), so you may want to qualify what you mean by "Latest version". Right now the latest version of Ubuntu Linux is 5.10 ( The Breezy Badger)

Just an FYI :P
