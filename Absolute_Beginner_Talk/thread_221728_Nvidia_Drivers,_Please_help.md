### Post by TravisNewman on 2006-07-23
the best way to install nvidia drivers is to use the Synaptic Package Manager-- it's in System -> Administration. When you open it, it will ask for a password. Enter that and then when Synaptic opens, search for the package nvidia-glx and install it. Be sure to enable the restricted repository. Full details on how to do that, and further details on how to install it are here:
[https://help.ubuntu.com/community/BinaryDriverHowto/Nvidia](https://help.ubuntu.com/community/BinaryDriverHowto/Nvidia)
