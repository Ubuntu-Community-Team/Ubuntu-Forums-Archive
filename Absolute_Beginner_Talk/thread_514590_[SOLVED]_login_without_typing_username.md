### Post by sancho panza on 2007-07-31
Is there a way I can ask the system to ask me select (not type) my username when I login so I have to only type my password? This is helpful because I'm the sole user on my machine (or at least I use it almost all the time) and there is no unauthorized local access. Occasionally if someone else wants to use the system, they can select their username from a list and type their password. It would be similar to how the lock-screen works.

I realise this is less secure than when both username and password are not displayed.

### Post by pollywog on 2007-07-31
You can easily enable automatic login. System>Administration>Login Window>Security. Click "Enable Automatic Login", and select your user name. This will bypass the login screen altogether, and boot straight to your desktop. -P.W.

### Post by pyros on 2007-08-01
> **sancho panza said:**
> That is not what I want. That would enable anyone to start my computer and login to my account even in my absence. I want the system to ask for my password only. Like is does in the "lock-screen" situation.

Try going to System > Administration > Login Window, Go to the "local" tab
look through the list for a theme titled "Human List" I'm pretty sure it is there on a default install. Make sure that circle on the left side of picture has a dot in it, then close out. On the next start, you should see a list of users on the system, then you can click on the user and type in the password. I think this is going to be as close to what you are asking for as possible.
