### Post by Hieronymus on 2007-04-07
Just go to Tools >> NAT / Firewall test and change the port to, for instance, 54322. The press test and if it says "Testing port 54322 ... OK !" then apply and in a few seconds you should see that there is no firewall active. If it still doesn't work something is blocking it.
