### Post by jem7v on 2006-10-22
That Compaq only came with 64 megs RAM when it was released, so if that hasn't been upgraded in the past that could certainly be what knocked out the LiveCD... LiveCDs *usually* need a lot of RAM (~256+ megs for Ubuntu distros) to run, and that holds true for Xubuntu, even though it's a lighter install overall.
