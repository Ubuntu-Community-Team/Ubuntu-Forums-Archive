### Post by fenian on 2007-12-13
On the computer connected to the printer go to the menu System>Administration>Printing under Server Settings check Share published connected to this system printers.Click apply.

On the other computer go toSystem>Administration>Printing and under Server Settings check  the Show Printers shared by others systems option and click apply.
