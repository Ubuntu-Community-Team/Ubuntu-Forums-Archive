### Post by greg_g on 2007-07-03
The second issue sounds like a defective cd.  Since you paid for it, I would go back and ask for another one.  

And also, before doing an install, always do the "check cd for defects" option.  yeah, it might take an extra 10 minutes, but it might save you 3 hours.

And, what errors did the server installation give you?
Do you need/want the server installation? (It doesn't have a graphical user interface)
Try the LiveCD, or if that doesn't work, the Alternate CD, all downloadable.  But make sure to burn at something lower than "maximum speed" and, again, check the cd for defects.

good luck
