### Post by hamishw on 2007-09-05
Hi,
About 5 months ago I got interested in Linux and specifically Ubuntu, I made a live CD and booted it up. I was impressed. I decided to install it as a dual boot and so backed up important stuff from our XP laptop. IMPORTANT - BACK UP YOUR STUFF, YOU NEVER KNOW!!!. I am very glad I did because when I decided to take the plunge several things went wrong! 
1. I didn't read the wizard properly and clicked to allow Ubuntu to install itself over XP and so managed to totally delete Windows from the system! DOH.
2. I didn't bother using the facility that allows you to check the CD ISO to make sure it is error free. I had assumed that it was error free because it was working as a live CD. It wasn't. It got to 68% installed and then froze. I tried a number of times, each time getting more and more worried! I didn't work. I checked the integrity of the CD and found that it did have errors. I had to get a mate to download and burn (on slow speed!) a new ISO disk. That worked. Wireless etc all worked perfectly. 
I had the usual problems making multimedia work but overcame all those problems and now have a system that does everything I want it to do, entirely free!
Bottom line is: Go for it, Ubuntu is great. Just make sure you do those back ups!
Hamish
