### Post by nickle on 2006-03-23
If you want beautifully set texts then use latex. Lyx or Kile are frontends for latex.  Latex takes a bit of getting used to, but I think is one of the most powerful free typesetters round. ### Post by Bloch on 2006-04-19
Abiword has improved a lot in stability over the last couple of years. For a writer, stability of the app is a crucial issue, and Abiword burned me a couple of times in some older versions.
I think it has improved too in the ability to open word docs.

OpenOffice has such features as marking a section of text as being e.g. German, and running the appropriate spellcheck on it. It's got all the features MS word has got except the grammar checker, which are all useless anyway.

If anything, it's too much of a clone of MS word - you have to know a bit about the menus just so you can turn off the auto-correct and capitals after a full-stop etc. I use it for my translations, where delivering the translated copy back with the same formatting, fonts, arrangement of tables etc is crucial. Once I think some auto-compilation of an index didn't work, so I lied to the client that I was using MS word on a mac.
