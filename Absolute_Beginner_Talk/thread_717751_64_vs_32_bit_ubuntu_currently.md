### Post by steveneddy on 2008-03-07
I currently use 64 bit Ubuntu on a System76 laptop.

Although I had to install it myself, I believe there are no drawbacks to the 64 bit system over a 32 bit OS.

Most of the software you need is already in the repos and if you need something, you can always compile it.

I haven't had any problems, there is a learning curve to install the extras.

Just follow [this guide]("http://ubuntuforums.org/showthread.php?t=368607") and you will be OK.
