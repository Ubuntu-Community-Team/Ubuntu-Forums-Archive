### Post by RSLxH on 2008-01-04
Those webcams are Java. You need to install the Sun Java package in Synaptic, or quicker, do it from the terminal with:
```
sudo apt-get install sun-java6-jre sun-java6-plugin sun-java6-fonts 
```
