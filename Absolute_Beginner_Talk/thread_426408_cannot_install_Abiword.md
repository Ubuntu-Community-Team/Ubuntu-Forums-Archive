### Post by Seisen on 2007-04-28
Do this Click on System>Administration>Synaptic Package Manager, it will ask you for your password enter it and Synaptic will open up. Clikc where it says search and enter abiword. When it finds abiword click on the box next to abiword and select mark for installation. Another box will pop up just click Mark.

After that click on Apply, looks like a green check mark in Synaptic. It will install Abiword for you and it should be located in the Applications>Office.

If you have any problems let me know.
