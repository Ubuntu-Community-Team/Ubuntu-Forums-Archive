### Post by LLRNR on 2006-12-03
Hi there !

Are you sure you enabled CD boot from your BIOS ? Depending on the BIOS you have, you hit a certain key at boot time in order to enter its settings (this key can be F1, F2, ..., F12, or Esc, or Del, it depends) and then you have to look for a menu entry saying something like "Change boot order" - you will have to enable CD boot or, in some cases, you will have to temporarily change the boot order - some CD ROMs are slower and if the first boot device is your HDD, the CD won't be able to boot unless it's the first boot choice and the second one is the HDD.

Good luck!

LLRNR
