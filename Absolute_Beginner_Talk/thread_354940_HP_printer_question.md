### Post by ^rooker on 2007-02-06
I don't know about your exact model, but if it's USB it usually shows up as seperate devices: printer / scanner.

I've got one of those HP-print-scan-combi things running under LFS (thanks to [HP Linux Inkjet Project]("http://hplip.sourceforge.net/")

Just plug it in and run "lsusb".
If you see 2 devices, you can use sane for the scanner and cups for the printer.
