### Post by Ronin324 on 2008-01-16
Well I have tried both file systems ext3 and reiserfs. I have used them both with Ubuntu and in my trials with Gentoo. In my opinion I think reiserfs might be a bit faster but it is pretty subtle. I have not had a problem with stability with either format. Since you are running internal drives with one file system and an external driver with another a direct comparison for you might be a little difficult with possible variations in transfer speeds.
