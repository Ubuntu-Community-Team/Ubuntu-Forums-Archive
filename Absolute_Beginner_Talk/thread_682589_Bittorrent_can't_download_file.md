### Post by p_quarles on 2008-01-30
Normally, you would need to forward ports 6881-6889 to the IP of your computer. If you're running any firewall software, you'll need to allow those ports with that as well.

Still, an unconfigured router shouldn't result in a tracker error, as far as I know. It would just make your download very slow. Try configuring the router anyway, though, and if that doesn't work, there's always the http download.
