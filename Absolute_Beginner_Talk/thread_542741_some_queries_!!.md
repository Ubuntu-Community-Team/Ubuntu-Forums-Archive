### Post by RomeReactor on 2007-09-04
Hi.
> **hydroweaver said:**
> hi guyz , im all new to linux and ubuntu , i have kubuntu fiesty 7.04 installd on my laptop dual booted with windows xp....
i had some queries ::confused::confused:
 
1. does every thing tht works in ubuntu works the same way  in kubuntu ??
The vast majority of Kubuntu programs will run in Ubuntu and viceversa. The underlying technology--libraries and system programs without graphical interfaces--is the same.

> 2.how do i open files in the root folder ?? and program them the way i want ( i guess thts wat ubuntu nd other open source OSs are for??)
Unless you have an imperative need to do so, root files and folders should not be altered; messing with them is extremely risky if you don't know what you're doing, and can lead to breaking your installation.

> 3.do i need to learn all the unix commands for better optimization of the OS ?
You don't *need* to; nowadays you can practically spend all your time without ever seeing the command line, provided all your hardware gets recognized and properly configured. There are graphical tools to tweak your system. However, you can learn the command line by using guides such as [Linux Command]("http://www.linuxcommand.org/"), but I strongly suggest you get used to your graphical environment first.

> 4.what shud i do to get into kernel programming nd shell scripting nd stuff ?
I don't know about kernel programming, but I imagine you would need to be proficient in C, at the very least. For shell scripting, Linux Command can teach you the basics.

> 5.do i need to go to the kubuntu forums ?
You don't *need* to, but it would probably be a good idea to keep your eyes on both forums.

> 6.the last wierd question : how do i compile the kernel , where is it ?
If you don't have a particular need to compile the kernel, it would be best to leave it alone.
