### Post by crispy_420 on 2007-02-11
No to all Windows players being supported. Some require special drivers and other win only software to add files. That is the case with Toshiba players (they look really nice), I was going to suggest one but required Windows Media Player 10 to add files. Looked through the user manual to find that out.

Just read through user manuals carefully before you purchase. Not just for linux support but it makes file loading easier in windows too. That is one major issue I have with my iPod and why I hardly use it anymore.

Also look at file support. Will it support ogg, mp3, aac, etc. ? If video, what formats (quicktime, avi, wmv)?
