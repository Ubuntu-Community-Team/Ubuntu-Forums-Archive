### Post by tmahmood on 2007-01-11
jEdit is the very nice if you don't have any problem with Java based Editor... with some extra plugins you can have a very decent web development IDE. I personally use this jEdit 4.3pre-beta with these plugins
SideKick, Project Viewer, CSSEditor, ErrorList, PHPPerserPlugin, Project Viewer, XML you can install them easily with Plugin installer built into jEdit.
Also you can use Geany which is basically gnome based. which is also very nice. Lists functions provides drop down list, good highlighting etc
you can use cssed for editing css file it have some minor issues though don't support recent files which actuall very big problem. gPhpedit is also good for PHP and CSS... IMO jEdit is the best.
