I'm trying to fill out a PDF tax form, but when I open the file in Adobe Reader, I get "plug-in required by this 'javascript' action is not available", and none of the fields are editable. ### Post by armitage1337 on 2008-02-09
I managed to find the solution: all I had to do was install the packages acroread-plugins and acroread-escript. I have the JavaScript functionality in Adobe Reader.
