### Post by pseudonym on 2007-01-01
This is possible, because Xfce4 uses what's called a 'compositing window manager', which basically means a window manager which handles special effects like transparencies. :)

Looks like someone wrote a [howto]("http://www.ubuntuforums.org/showthread.php?t=272890&highlight=xfce+transparency") for it on these forums. I haven't tried it myself yet, but it is on my to-do list.

One other thing you might want to do is subscribe to the [Xfce Mailing List]("http://foo-projects.org/mailman/listinfo/xfce"). Issues like this get discussed there quite a lot.
