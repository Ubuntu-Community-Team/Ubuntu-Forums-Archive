### Post by RomeReactor on 2007-05-27
Hi. I'd say, chances are that if a program is listed as "installed" in Synaptic, then it will automatically update (that is, when a new version comes up in the repositories, you will be asked if you want to upgrade). If it isn't listed there, then it's likely that program was installed from a downloaded file (such as .sh or .bin) and you'll have to locate the update through a website.

### Post by Kobalt on 2007-05-27
Ubuntu uses the [repositories]("https://help.ubuntu.com/7.04/add-applications/C/index.html") system. The new version of a program you have installed will be proposed to you automatically once it reaches the repos. 

Regarding Automatix (and such) I'd advise you to read this : [http://ubuntuforums.org/announcement.php?f=13](http://ubuntuforums.org/announcement.php?f=13)

### Post by RomeReactor on 2007-05-27

> Ubuntu uses the [repositories]("https://help.ubuntu.com/7.04/add-applications/C/index.html") system. The new version of a program you have installed will be proposed to you automatically once it reaches the repos.

Exactly. Also, to correct my previous comment, if you download and install a .deb file downloaded from a website (like [Opera]("http://www.opera.com") or [WICD]("http://wicd.sourceforge.net/")), they'll show up on Synaptic as installed, but since there's no repository including them, you'll need to either wait until the program tells you there's an upgrade (in Opera's case) or visit the site and download the update/upgrade when there's one available.
