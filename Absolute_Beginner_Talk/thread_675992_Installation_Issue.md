### Post by bumanie on 2008-01-23
If you only have 256mb ram you will need to use the alternate cd and also consider using xubuntu, which has a smaller resources footprint than ubuntu. I would even consider using the 7.04 version instead (still using alternate cd or xubuntu). If the install has not started (ie no partition changes, no grub install), you *should be able to take out the live cd and reboot to xp(* Can't guarantee anything as I can't see what is/has happened). Also with a 20g drive for both xp and ubuntu, you are going to be pushed for space.
