### Post by kostkon on 2007-06-27
You can easily add rar support to the default archive manager (*File Roller*), by just installing the *rar* package from *Synaptic*, and then doing this in terminal
```
sudo ln -fs /usr/bin/rar /usr/bin/unrar
```
