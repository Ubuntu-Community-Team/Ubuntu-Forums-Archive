### Post by steeleyuk on 2007-06-04
This might be a bug which I've come across at times with Azureus. This may not work but won't do any harm:

Go to your home directory. Press CTRL+H to view hidden files. Open .azureus and open the Logs folder. Delete the contents then try reopening Azureus.

I can really recommend using utorrent
1.install wine
```

sudo aptitude install wine

```
2. download utorrent (just the executable)
3. run utorrent with 
```

wine path-to-utorrent/utorrent.exe

```
It has a very small memory footprint and uses very little CPU resources.
