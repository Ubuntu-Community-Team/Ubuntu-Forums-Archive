ndiswrapper is a program that takes a windows driver, and uses it to make an attached device functional. 

It is designed to be used for wireless network card drivers, especially for cards that are not shipped with Linux drivers and the company refuses to help the community create their own.

ndiswrapper should be in the repositories, just do a quick search in Synaptic.

Also see [http://ndiswrapper.sourceforge.net](http://ndiswrapper.sourceforge.net)
