### Post by GreyFox503 on 2005-11-02
Kyral is right.  VMware costs money and it allows you to create virtual machines to run other operating systems (like Windows).  The free VMware player from their website only allows you to "play" existing virtual machines, not make them.  Hope that helps.  :)

I've never used it, but a free alternative to VMware is called QEMU, if you're interested.
