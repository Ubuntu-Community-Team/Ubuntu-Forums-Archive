### Post by The Mekon on 2006-05-23
I have a Speedtouch 510 which is set up using a web browser.  A Google search indicates that the 511e is set up the same way.

Connect the modem and switch it on.

Point your browser to 10.0.0.138 and you should be able to set it up.

See this [link]("http://www.dsl.net.pk/st511e1.html")
