### Post by Paqman on 2008-02-07
If you're not having problems with your Gnome behaviour, I don't really see that you'll get any great bonus. Gconf is a set of configuration files for the desktop and Gnome apps. It lives in your home directory, where most people have loads of space, so trimming it down won't matter much.

If you like things to be tidy, then go for it, but it's not an important maintenance task.
