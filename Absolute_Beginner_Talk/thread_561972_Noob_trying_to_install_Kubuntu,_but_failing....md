### Post by rug77 on 2007-09-28
I've had the same black-screen experience, in common with loads of others.

The trick seems to be to add the following at the end of the xorg.conf file ...

```


Section "Extensions"
        Option      "Composite" "0"
EndSection

Section "DRI"
        Mode    0666
EndSection


```

Hope this works for you - let us know .

Matt
