### Post by gn2 on 2007-06-21
I wouldn't attempt it. An 866mhz CPU will struggle to run XP on it's own, never mind Ubuntu at he same time.

You can access all the data on your Windows partition by using NTFS drivers.

My preferred method of adding this functionality is through [www.getautomatix.com](www.getautomatix.com), it's in the "Miscellaneous" category once Automatix is installed.

Or use Synaptic.

Automatix has it's own forum support, should you encounter any difficulty
