### Post by kanem on 2005-10-12
[QUOTE=Frosty]I'm sure there must be a way to do this. I set xmms to be my default audio player.[/QUOTE]
If you right click on the file you want to play and select options there is a tab called "Open With". This lets you select the default app. If vlc isn't in the list click the "Add" button. This presents you with a list of all available applications. Just scroll down and find vlc. If for some reason it's not in the list at the bottom is an option for "select custom command". click this and enter vlc.

Selecting vlc by right clicking and choosing 'Open with vlc' should only open it in vlc that one time. If you select vlc in the 'Open With' tab in the properties dialog it should open with vlc when you double click it. If it doesn't, it's a bug.
