EDIT: oh and a good partition plan would be:

Primary #1 = ntfs --  current windows partition -- leave a good amount of space if you want to keep downloading in windows
Primary #2 = ext3 -- "/" partition(ubuntu filesystem) -- a good 15 GB if want to add a decent number of apps
Logical #5 = ext3 -- "/home" partition -- 5 to 10 GB will do if you think of updating ubuntu in the near future
Logical #6 = fat32 -- "/windows" -- this can be a shared partition between both OS's :]
Logical #7 = swap area -- give this about twice memory of your RAM. that can improve stability i believe :]

### Post by ndefontenay on 2007-08-16
The good mindset to start with Ubuntu is to look for something that will do the job.
Not look for the same software exactly.

Ubuntu comes with Amarok and Open Office installed. It's a much more complete OS from start compared to windows or even Mac (I own one).

You should try to use Linux with what's already there. You will probably find out a few things you would like to have or use. There are certainly alternatives for that too.

When it's installed on your computer, go browse the add/remove program tool for the missing parts (firefox, thunderbird and the like).

In security terms, the fact that you need to input a password for any installation of program makes it harder for viruses to install without your will.

To me it's not only a low market shield that protects us from viruses. It's also a well designed OS from the start.
