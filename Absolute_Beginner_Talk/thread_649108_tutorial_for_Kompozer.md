### Post by erginemr on 2007-12-25
I don't have a direct answer to using FTP from within Komposer but, if you are also after a general tutorial, then; Kompozer 0.7 is just a quick-and-dirty update to NVU 1.0 with its bugfixes. NVU / Kompozer uses the same plugin / extension system that Firefox and Thunderbird uses. There is a tutorial plugin in the NVU website (down the page), which you can download and then install via its menu:
[http://nvudev.com/download.php](http://nvudev.com/download.php)

For a text-only tutorial:
[http://www.thesitewizard.com/gettingstarted/nvu1.shtml](http://www.thesitewizard.com/gettingstarted/nvu1.shtml)

For a video tutorial:
[http://www.designyourownweb.com/nuv-tutorial.htm](http://www.designyourownweb.com/nuv-tutorial.htm)
