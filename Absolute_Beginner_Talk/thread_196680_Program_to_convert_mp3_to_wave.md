### Post by someusernoob on 2006-06-14
Yes, there is a program (script) that can do batch converting for you.

[http://linux.softpedia.com/get/Multimedia/Audio/Audio-Convert-3104.shtml](http://linux.softpedia.com/get/Multimedia/Audio/Audio-Convert-3104.shtml)

Works very well.

You might need to install some codecs, but you can use Synaptic for that.

(When you follow the instructions in the readme file, there is an option under the right-click menu in Nautilus that says Script > audio-convert. When you select your mp3's run that script, and it'll prompt you for some questions. Maybe strange the first time you use it, but it's really easy and works great)
