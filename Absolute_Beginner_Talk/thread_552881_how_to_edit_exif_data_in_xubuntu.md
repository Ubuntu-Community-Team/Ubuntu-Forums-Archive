### Post by nathandumont on 2008-02-28
Open up Synaptic (Or Adept if you're in KUbuntu) and search for jhead and install it that way.  Alternatively you can run "sudo apt-get install jhead" at the command line.  This should install the package as it is in the default Ubuntu Repositories.

For other tools available try just running a Synaptic search for exif.
