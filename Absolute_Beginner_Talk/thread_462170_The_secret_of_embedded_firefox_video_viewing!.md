### Post by 4KvRMU7Nnv on 2007-06-02
The most foolproof way to play imbedded video of almost any format inside firefox is to use the "mozilla-plugin-vlc".  Make sure that you have removed all of the others that don't work as well like "mozilla-mplayer" or "totem-mozilla."  The only issue with the "mozilla-plugin-vlc" is that there is no controls that allow you to pause or skip parts.  If anyone knows of a better way without opening a new window to view the video, please tell us.
