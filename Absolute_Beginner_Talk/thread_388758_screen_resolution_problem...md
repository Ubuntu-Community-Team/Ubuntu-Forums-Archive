```

```

Finally, restart X with Ctrl+Alt+Backspace.

### Post by Kobalt on 2007-03-20
You will need to boot with the vga=771 option maybe. Here is the way to go :
First, boot normally. Once you get to your blank screen, press Ctrl+Alt+F1 to access a command line, log in from there with your username & password. Then enter the following command line to edit your menu.lst file : 
```
sudo nano /boot/grub/menu.lst
```
From there, go to the bottom part of the file, where you boot lines & kernels are listed. Pick the first one in the list (being the newer and the one you normally boot onto), it should look like this (though not the same figures) : 
> kernel          /boot/vmlinuz-2.6.20-12-generic ro single
At the end of the line add "vga=771" so that the line looks like this : 
> kernel          /boot/vmlinuz-2.6.20-12-generic ro single vga=771
Finally save with Ctrl+X, and close with Enter 
Reboot, and I hope it helped you...

You can just edit the /etc/X11/xorg.conf file manually and add the needed resolution to there.. I guess you want 1280x800

And to edit it just type in terminal:

save and reboot..
