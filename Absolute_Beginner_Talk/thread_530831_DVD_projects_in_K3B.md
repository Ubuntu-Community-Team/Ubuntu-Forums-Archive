### Post by jasonwatkins on 2007-08-27
> **chee said:**
> they are actually mp4 files that i downloaded using a torrent

i'm not entirely sure if mp4's will be compatible with DVD format, but if not, you can use avidemux to convert them to MPEG compliant files.

then, use DeVeDe to add each file and what I usually do is check the "create ISO" option and just burn the ISO with k3b.

if the episodes you want to add are larger than a 4.7gb disc, then select each one and click "properties" and play around with the bitrate and audio sample rate to reduce the size of each file.

when I used to use windows, using a combination of dvd fab decrypter and nero recode, I could get 8 episodes of "Scrubs" or "Friends" onto one 4.7gb DVD with no real loss in quality.

You can get 10, but then the quality does start to dip then.
