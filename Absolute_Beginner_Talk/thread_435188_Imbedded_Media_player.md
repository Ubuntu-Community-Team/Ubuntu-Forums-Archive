### Post by bobbybobington on 2007-05-06
yeah, this one can be a real pain. Try opening the file in the player program (as opposed to embedded) This is so you can figure out if you have the codecs to play the file. If you can't it should pop up a window to install the necessary codecs. As far as getting it to play in the browser window I haven't had to much success, but you can always open the video by pasting it's url in the video program. Hope this helps
