### Post by kevinatkins on 2006-11-13
Files with .sxw extensions are early (pre-2.0) OpenOffice.org word processor (Writer) files - if you click on such a file, Ubuntu should launch OpenOffice automatically for you.

OpenOffice.org 2.0 now uses open document format files by default - identified by .odt for word processor documents.
