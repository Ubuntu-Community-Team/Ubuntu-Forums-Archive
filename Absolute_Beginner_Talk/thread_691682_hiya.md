### Post by JoshuaRL on 2008-02-08
When you have Ubuntu up and runnning how you like, try the other desktop managers with:

```

sudo aptitude install kubuntu-desktop
sudo aptitude install xubuntu-desktop

```

Those will take a while, but they give you experience with KDE and XFCE.  As far as the iPod, probably.  It depends on which firmware version you have.  I believe the newest one has problems syncing with Linux because Apple wants it too.  I do not believe there is a way to get the Zune working currently.  Sorry.  You should consider dual-booting if you really need it.
