### Post by dcs3jah on 2005-12-04
Ubuntu uses GNOME for the desktop environment.  Kubuntu uses KDE.  Simple as that really.  If you don't know what that means, I'd recommend you just try each out as Live CDs (without installing to hard drive).  You'll see they look different, and act somewhat differently.  You can run the same applications though, as a rule.

### Post by canadianwriterman on 2005-12-04
Just to elaborate a bit further... GNOME and KDE are desktop interfaces with different look and feel. There are also some different applications (such as audio, editing, etc.) that are designed specifically to integrate with each of them (although most of them will work with the other). Check out:

[www.gnome.org](www.gnome.org)
[www.kde.org](www.kde.org)

for more information.
