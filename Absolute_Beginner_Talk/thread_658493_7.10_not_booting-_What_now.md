### Post by Xzallion on 2008-01-04
Then you burnt the iso file itself to the disk, instead of having nero burn the iso image.  You will need to burn another disk, and make sure that nero burns the iso image.

In case your wondering why you can view many files in ubuntu, an iso file is a compressed disc image, meant to be extracted and burnt to a disc.  Ubuntu can extract iso files so you can see the files compressed inside, just like unzipping a .zip document.
