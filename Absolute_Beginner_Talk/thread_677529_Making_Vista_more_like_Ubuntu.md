### Post by jon.reeve on 2008-01-24
There's no program that I know of in Windows that'll do what you're talking about. But (and forgive me, because this totally doesn't answer your question) if you haven't already tried, one way to get around all this is to use Wine, and that way you can run whatever windoze programs you need and not have to leave Ubuntu. I donno about Vista, but In XP I used to put together new keyboard shortcuts by first creating a shortcut (ie find the command you want, right click on the desktop & "create new shortcut") and then adding a keyboard shortcut for it in it's properties page, like Cntl + Alt + etc.   But this only really worked for launching applications and doing simple commands.
