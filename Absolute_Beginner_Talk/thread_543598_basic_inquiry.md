### Post by RobsterUK on 2007-09-05
If you need to use specific software your first stop should be the software vendor's website to find out if they provide a Linux version of their software.

As a rule you can't run windows software on Ubuntu, but some software can be run via WINE which is a windows emulator, but depending on the software this can be tricky.

### Post by Bethrine on 2007-09-05
Newb here. You have some options, looking for Linux equivelents, running in Wine, dual booting (putting two OS's on one computer or even one hard drive). All of which will take some time to set up. I personally dual boot, but wound up only using my Vista for AutoCAD, and it did take me some time and some education to do it. In the long run, for me, I find Ubuntu easier simply because I can get help when I need it easily and as I learn, I wind up able to manage easily the problems that do come up. Just my opinion.
