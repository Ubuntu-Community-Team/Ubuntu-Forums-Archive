### Post by jasay on 2006-06-11
Depends on what exactly you're trying to do.  

To change the size of one icon on the desktop (does not change font size):
right-click and selecte stretch icon.  Grab the corner and resize.

To change the size of all icons on the desktop and in nautilus (does change font size):
Open nautilus (the file manager) and selecet Edit > Preference.  Select a default zoom level in the drop down menu.

To change the size of all icons in a particular folder in nautilus (does change font size):
Control+mousewheel or Control and either + or - depending on if you want bigger or smaller.
