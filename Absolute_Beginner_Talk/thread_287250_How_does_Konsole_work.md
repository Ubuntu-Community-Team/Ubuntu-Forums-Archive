### Post by Toxicity999 on 2006-10-28
Okay heres the biggy for you. Konsole is only a front end for the Terminal. It's a way for you to run a terminal while still being in your desktop environment (Just like windows does.) It is a direct parrallel really. Only the linux Console is far more powerful, with windows only a select few commands are readily available where as with linux all binaries are stored per default in a few known locations so when you type xyz (title of your favorite game maybe) the OS starts it just like that, where as windows cd gfhdfhg, dir, *stare*, cd fgdfgf, crappywindows.exe

For simplicities sake, yes you will find it similar and for future reference the Ubuntu forums cover all derivations of Ubuntu K*, X*, and such. So we know about KDE, gnome, and XFCE all you need.
