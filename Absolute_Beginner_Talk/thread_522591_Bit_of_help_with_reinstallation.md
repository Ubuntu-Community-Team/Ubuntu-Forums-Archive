### Post by annda on 2007-08-10
if you want to reinstall quickly, i'd say copy all the contents of your home directory (thunderbird should be in .mozilla there, check it first!), /var/www and any databases (if you use them).

after install get the files you need and delete the rest - probably 90%

### Post by logos34 on 2007-08-10
hey, don't forget /var/cache/apt/archives -- that's where all your deb pkgs are you downloaded and installed!  It'll save you having to dl them all over again!  While you're at it, I'd back up /etc (configuration files) and /usr/local.

(use APTonCD if you can)
