### Post by russell.h on 2007-10-20
You can always press Control + Alt + F1 (or F2, or F3 up through I'm not sure what) to go straight to an oldschool fullscreen command line. Press Control + Alt + F7 to go back to X.

You can also go Applications > Accessories, then right click on Terminal and select "Add to panel" or something along those lines to add a terminal launcher to the gnome panel along with Firefox and whatever is already there. Thats always one of the first things I do on a new Ubuntu install.

### Post by russell.h on 2007-10-20
Certainly.

Also, while you're at it, if you have any interest in programming, or want to be able to compile programs from source (some newer or more obscure programs aren't available in the repositories) then you might want to do
```
sudo aptitude install build-essential
```

That will install all of the "essentials" for compiling stuff in C, C++ and some other languages too I think.
