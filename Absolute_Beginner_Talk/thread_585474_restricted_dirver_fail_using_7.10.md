### Post by mitchlourens on 2007-10-21
did you try using the command in the terminal: ```
sudo dpkg-reconfigure -phigh xserver-xorg
```

It will ask you a few questions and also let you select the resolutions your monitor supports.  When you are done, use ctrl+alt+delete to restart XOrg and then you will be able to choose higher resolutions.

### Post by magli on 2007-10-21
It appears that you are not using the restricted driver. Look at the device section of your xorg.conf file. There is a line:
```
Driver		"vesa"
```
This means you are using the generic vesa driver, not the restricted ati driver. I do not have an ATI card, so I am not sure, but I think that the proprietry driver is called fglrx. In other words, you should be seeing "fglrx" instead of "vesa".

### Post by Happy_Man on 2007-10-26
Right, here's an easy way to solve this: open a terminal (Applications --> Accessories --> Terminal) and enter this: ```
sudo dpkg-reconfigure xserver-xorg
``` When the wizard-type thing comes up, keep pressing enter until you get to a screen with a long list of resolutions on it. Scroll to select the ones you want, spacebar to select, enter until it finishes. Then log out, and press ctrl-alt-backspace at the login screen. Log back in again, and you should be able to choose a higher resolution.
