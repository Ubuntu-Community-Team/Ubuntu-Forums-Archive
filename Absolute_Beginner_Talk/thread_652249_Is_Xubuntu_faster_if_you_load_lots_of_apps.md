### Post by wh0rd on 2007-12-28
Xubuntu is based on the light weight Xfce environment as opposed to Ubuntu's default Gnome environment. Installing applications won't decrease the speed unless you're running them all at once or you have then running in the background. Overall you will notice a speed difference. Xfce does have support for nVidia and ATi Drivers, therefore you can dedicate your resources to Compiz GUI.

### Post by blueridgedog on 2007-12-29
The OP implied installing applications, not running them all at once.  As such Xubuntu will still have a performance edge.

In windows installing many application did indeed degrade the performance because it increased the size of the registry and frequently added shell integration.

In Xubuntu you will not have that problem, but installing many application may rarely create a scenario where two applications require different version of the same library.

You can always try both.
