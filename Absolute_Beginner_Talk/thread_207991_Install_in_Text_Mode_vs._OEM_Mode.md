### Post by Jagot on 2006-07-02
Text mode installs Ubuntu as normal.  It's best for installs on systems with less than about 192MB of RAM (as apposed to the installer on the LiveCD).  You can see it in action here:

[http://users.bigpond.net.au/hermanzone/p3.htm](http://users.bigpond.net.au/hermanzone/p3.htm)

OEM mode follows mostly the same process but includes some functions which enable you to set it up for another user.  You can see a bit more here:

[https://help.ubuntu.com/community/Ubuntu_OEM_Installer_Overview](https://help.ubuntu.com/community/Ubuntu_OEM_Installer_Overview)

If you were building a computer for another person then you'd install in OEM mode so they can set up things like language, user name, etc, themselves.
