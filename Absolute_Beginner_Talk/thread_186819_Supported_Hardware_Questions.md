A Win modem is modem that requires additional (Windows) software to function.

To create a dual system the easy way: reinstall Windows, preferably size it down to use a small partition and leave some free space (the Ubuntu installer can resize it too, though). Don't install Windows *after* Ubuntu, that's a lot more troublesome.

The Ubuntu installer is pretty much automatic next-next-next. It will detect Windows and add it to the boot menu so you can choose betwen the two on startup.
