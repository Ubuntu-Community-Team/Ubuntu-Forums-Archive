### Post by chili555 on 2008-04-13
vim is best. ```
sudo vim /etc/ur_text_file
```Press 'Insert.' Add or delete text. Press 'Esc.' Press :wq if you want to save the text or :q! if you want to quit without saving changes.

How hard is that? I know there are more advanced capabilities, but for the basic editing of .conf files, etc., this is all I need or want. You can even copy and paste into vim.

Be sure to save a backup copy you can restore in case all goes ker-flooey.```
sudo cp /etc/file /etc/file.bak
```Reverse the process to restore your original.

I love vim.
