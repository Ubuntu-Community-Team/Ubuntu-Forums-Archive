### Post by Herman on 2006-02-04
I don't know whether or not this would be your problem, but just make sure you only use the number keys across the top row of your keyboard.
If you are like me, and tend to prefer the number keypad over on the right, that's okay, but you need to keep checking that your numlock light is on each time. For some reason it will often go off while Ubuntu is being installed, and that can cause the above problem. I know because I did it myself a few times! :-D (Herman)
