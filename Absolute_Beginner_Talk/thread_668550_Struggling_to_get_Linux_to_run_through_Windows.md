### Post by zipperback on 2008-01-16
Boot the burned CD rom that you made, and select the verify media option from the boot menu if it gets that far. It might be that you've got some problems with the burned cd.

If it verifies, but it won't load into the livecd desktop for you to install, then I would suggest downloading and trying the Alternate installation cd for your installation of Ubuntu.
