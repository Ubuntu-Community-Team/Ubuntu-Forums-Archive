### Post by vishzilla on 2008-01-08
Firstly if you dont have the CCSM installed, install it by going to Applications->Add/Remove. Set Show: All available applications and in the search box enter 'ccsm'. Once you install it, go to System>Preferences>Advanced Desktop Effects settings. There on Desktop Cube, go to the Appearance tab. You shall find the Background images, remove the images. And you are done!
