### Post by 5trawberry on 2007-07-15
i don't have much confidence to reply but as you're in a rush i thought i'd give it a go. 
Download the same torrent and start it in the ubuntu torrent client.
Tell it to save to X (whereever you want) once you've done that, close the torrent application, replace the file it's just made at X with the file you've partially downloaded in windows.

Then restart the application and it should start from how ever much percent you already downloaded in windows.
