### Post by Majorix on 2007-12-30
Instead of trying to run a Windows app through wine, did you try native lightweight browsers like Swiftweasel?

Along with it, make sure that mozilla-mplayer and w32codecs packages are also installed and they will run the Windows Media Player media for you.
