### Post by Patrick K on 2007-02-15
I'm glad you are excited about Linux. I'm guessing the MoBo has a Nvidia chipset. I don't know how well they work so can't help there. It also probably has a nvidia video chip. Nvidia video cards work well. After you get everything stabilized, you can install nvidia drivers. The default "nv" drivers work well but don't take advantage of all the features of the nvidia chip. Something you'll probably want. 

As for sound, I've not had too good of luck with onboard systems. It might work for you, though, if playback is your main use. I record live music (layering tracks in a home studio) and went with a PCI M-Audio card that performs much better.  

I've not used Beryl but suspect it is a real CPU cycle burner that provides a lot of eye candy but not much else. But if you have the horsepower to burn, why not.
