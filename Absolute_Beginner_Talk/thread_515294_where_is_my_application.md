### Post by p_quarles on 2007-08-02
There are a few applications that are meant primarily for command line use, and these tend not to show up in the menu. In most cases, you can edit the menus (right-click on Applications, choose the edit option) and make sure the program you want is selected. 

That said, the archive manager that comes with Ubuntu is pretty good. Maybe you need to do something that it can't handle, but do give it a try if you haven't.
