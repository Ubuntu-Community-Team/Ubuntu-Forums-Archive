### Post by NilsE on 2007-06-03
If you have the i8k utilities installed you can enter the following in a terminal where X is either and l or r for left or right fan

i8kfan -X 0

0 will stop the fan, 1 will run it slow and 2 will be fast

However, if other fan control is running it will just be temporary and switch right back so you may want to consider using the i8k stuff to control the fan.
