### Post by ynef on 2008-01-25
Are you positive that it's not installed? Seeing how it comes by default, I suggest that you open a terminal window and type "nm-applet". That should bring up the Network Manager applet, and if there is any problem (missing package or so on), you will most likely get clues as to what you need to install. Most likely, if you haven't uninstalled the program (in which case, you'd be able to find in the "history" what it was called), there's just been some configuration error and your applet isn't starting on its own. See if bringing it up manually helps.
