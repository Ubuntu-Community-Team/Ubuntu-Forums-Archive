### Post by Arktis on 2006-01-09
I found a website with source for something called "amigadepacker".  [http://zakalwe.virtuaalipalvelin.net/~shd/foss/amigadepacker/](http://zakalwe.virtuaalipalvelin.net/~shd/foss/amigadepacker/)
It's a simple matter of running the configure script, then make and then make install.  I tested this; it compiles just fine.  I suggest using checkinstall (instead of the make install command) though, so you will have a package that can be cleanly removed/upgraded later.  I don't have any xpk files to test this out on.  Let me know if it works or link me to one of these old amiga archives, please... I may want to use this tool at a later date and so I wanna see if it works well. :D
