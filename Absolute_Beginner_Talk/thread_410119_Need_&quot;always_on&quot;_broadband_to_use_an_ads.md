### Post by e_james on 2007-04-15
Your question is not entirely clear but I think the following will cover it.

A standard dial-up modem uses audio frequencies to send data over the telephone. It basically uses the phone line as it was designed to be used and therefore it takes over the line for the duration of the session.

An ADSL modem uses radio frequencies, which explains the limitation in line length. The radio frequencies are separated from the audio by the microfilter. Because the two functions are separate, the line remains available for normal phone use and the ADSL function can be "always on". 

The ADSL modem / router / computer combination has 2 jobs to do. First it establishes the data connection between compatible digital devices (PC client to server) and then a login is usually required. An ADSL usb modem usually expects the PC to handle the login. An ADSL router usually does the login by itself, which makes it easy for several PCs to use the same ADSL connection since there are normally several ethernet sockets. The router can be used to network 2 or more PCs without an internet connection.

All the routers I know have a built in firewal between the internet and the local network.

Some older routers have an ethernet (RJ45) socket dedicated for connection to the internet by means of an external modem. In theory this modem could be dial-up, but I have never seen such a gadget.
