### Post by ltmon on 2005-11-04
If you get the Ubuntu disks there is no way to get Kubuntu without downloading more stuff -- it simply does not exist on the Ubuntu CD.

When you try and install kubuntu-desktop (like the parent suggested) you don't need to do it all at once.  If you cancel mid way (ctrl-c) any packages which have been already downloaded (completely) will not need to be downloaded again.  The biggest package size is < 20MB (i think?), so if you are prepared to spread your downloads over several weeks you could still do it.
