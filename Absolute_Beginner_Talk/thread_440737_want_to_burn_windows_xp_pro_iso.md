### Post by skwishybug on 2007-05-11
> **JosephBear said:**
> I want to burn windows to disk for my other computer can it be done while running linux and if so how i am running feisty and what is a key gen

If you have the .iso created and saved, try K3B to burn it to disk. It worked to burn the Feisty ISO for me and gives you the option right when it starts.

If you do not have the .iso yet, you'll have to create one, unfortunately others will have to give you direction on this one.

A Key Gen is generally a program that will generate a CD key for a program. Some work well, others work not at all. There are other places to get a key as well, it just takes a little looking (For obvious reasons I won't put them up here).

### Post by saphil on 2007-05-11
Just copying the windows installation you have running on your other partition on a dual-boot system won't work.  That version of Windows has been configured for the specific pc it is installed on.  If you have a windows install disk, you can diskcopy it using a linux-based burning application.  If you already have a windows install disk, why would you need a burned copy? I mean, aside from back-up in case the disk is damaged.
