### Post by rmjokers on 2006-02-06
If you downloaded a CD iso image (~600 MB), you need to burn it as an image onto a CD.  If you just write it as a file, the system will not recognize it as a bootable disc.

### Post by GreyFox503 on 2006-02-06
Yes, you don't want to burn it as a data disc (the normal way).  Burning disc images using requires a special option within the software, something like "Burn image to disc" or similar phrase.
