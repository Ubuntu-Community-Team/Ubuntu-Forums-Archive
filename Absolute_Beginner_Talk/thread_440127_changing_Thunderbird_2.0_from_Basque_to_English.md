Hi, I am from the automatix team. What happened was when you selected to install TB there is a dialog asking what language you want supported. You accidentally selected Basque when that dialog box came up. To fix the issue use AX to uninstall TB and then reinstall TB and select the proper language.

You can find the AX support forum [here.]("http://www.getautomatix.com/forum/")
