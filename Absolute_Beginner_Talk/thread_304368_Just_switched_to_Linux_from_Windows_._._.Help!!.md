### Post by qrm on 2006-11-21
well, i guess if you want to install stuff you have to install ubuntu on your harddrive, because you cant install programs on CD or in your RAM. And concerning your fear about command line ... youll find out that very often you are given a simple text input which you have to paste into terminal/konsole (mark the needed part with mouse and copy it with middleclick and paste it with anouther middleclick) because its way easier to tell a person to do that then to give the instuction of click there, and click there, and doubleclick there and do that stuff over there but you have to click there first and so one and so one. There are excellent tutorials for installing flash/java/etc in the wiki and you might  find some great tutorials on this forum too (subsection FAQs, Howtos and tips). If you happen to encounter a problem or a error message just search for it in this forum ... its covered by 99,9% already ;) To play the mp3-s you need mp3 codecs which arent legal everywhere in the world so they arent added by default. Automatix is a little program that take care for most basic needs for ubuntu desktop use, check out this link: [http://www.getautomatix.com/](http://www.getautomatix.com/)

good luck
