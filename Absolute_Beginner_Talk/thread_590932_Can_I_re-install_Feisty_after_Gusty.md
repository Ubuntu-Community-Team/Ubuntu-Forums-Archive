### Post by k3lt01 on 2007-10-25
> **soctu said:**
> how would you save the home directory from reformat other than just backing up the directory somewhere else (flash drive, cd, etc..:confused:? n3tfury half answers the question below.  One way for you to save it is for the /home directory to be a separate partition, if it is set up this way you go into "unguided" in the disc parttiions section of install and you do not tick the /home partition (it may be given another name but if you know how big it is then you will have an idea of what one it is, mine is 27 GB and is the only partition that big). If it isn't then the only way it can be saved is to back it up to another media as is pointed out in the quote below.

Once it is saved like this then you can reformat the hard drive and give it separate partitions making the /home partition the largest if you can. Then put all the files you saved from it before you reformatted the hard drive back into it and then all your personal files and settings are still there.
