1. Fix your graphics card ! ! !
2. Test the glxgears, make sure that the test is more than 500FPS
3. Install cedega 4.4.1
4. Install Point2Play, yes... gui is better /gg
5. Install Ragnarok pRO client
6. Update the game by copying an updated windows client.
7. Run the game.... (if see the login screen, then your config is near perfect)
