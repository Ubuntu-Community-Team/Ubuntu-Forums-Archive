### Post by Joeb454 on 2008-03-28
There's hardly any difference between the 64 bit and 32 bit repositories :)

I have flash working great on Hardy (8.04) x64 :) It installs using nspluginwrapper by default, which allows it to install the 32 bit flash (because that's the only version it comes in :()

So don't worry...Core 2 Duo's (and a lot of others) are actually x86_64, which basically means that they can run 32 bit **OR** 64 bit systems :)
