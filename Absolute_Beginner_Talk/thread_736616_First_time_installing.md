### Post by tylermoody on 2008-03-26
You'll definitely want to install 7.10 instead of 6.06. Make sure you get the 64-bit kernel; the name of the file you download should contain "amd_64" instead of "x86". You want the second option on [the download page]("http://www.ubuntu.com/getubuntu/download") for 64-bit AMD and Intel processors.
