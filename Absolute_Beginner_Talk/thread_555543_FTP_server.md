### Post by Jeroen Vernooij on 2007-09-20
Hi,
The nice thing about ubuntu is it's repositories and package manager.

Just open up 'synaptic package manager': system->administration->synaptic.

Now, search for 'ftp server' (in name+description) and you'll find many FTP servers you can directly install.

For example, you can try proftp, because it has a GNOME frontend.

Just install the package 'gproftp'  and all it's dependencies will be installed automatically.
