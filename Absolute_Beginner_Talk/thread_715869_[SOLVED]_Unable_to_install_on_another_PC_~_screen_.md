### Post by OffAxis on 2008-03-05
Try holding down the ALT key and left-click drag the window around (so you can get to the okay button, or whatever options are there).
This works in KDE; I hope it works for you in gnome.

Clarification: Using the ALT key + left mouse click lets you use any part of the window as a draggable 'grab' point; it doesn't just limit you to using the title bar.
