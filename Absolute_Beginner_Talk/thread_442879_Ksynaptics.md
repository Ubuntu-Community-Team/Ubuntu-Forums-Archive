### Post by DerArzt on 2007-05-14
I would recommend doing what the error message says. go to a terminal and type in

```
sudo gedit /etc/X11/xorg.conf
```

then find the section that is for your touchpad and insert <Option "SHConfig" "on"> (without the < and > symbols) somewhere between Section and EndSection on its own line. Then retry ksynaptic

### Post by floke on 2007-05-16
This bit is it:

Section "InputDevice"
    Identifier    "Synaptics Touchpad"
    Driver        "synaptics"
    Option        "SendCoreEvents"    "true"
    Option        "Device"        "/dev/psaux"
    Option        "Protocol"        "auto-dev"
    Option        "HorizScrollDelta"    "0"
EndSection

Add this to it somewhere
   
    Option        "SHConfig" "on"

and you're in business

Remember - back up your xorg.conf first - sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup (or something) 

To edit the file type - sudo gedit /etc/X11/xorg.conf
