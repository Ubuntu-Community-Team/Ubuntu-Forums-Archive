### Post by Foxmike on 2007-07-24
If you open-up the side panel in Nautilus, you can select thru a drop-down menu the "Emblems" menu.  You just select en emblem, you slide it over the folder you want to identifty that way and voila!:)
 
What happens here (I think) is that when you share a folder, you do it from root using sudo (or gksudo).  Doing it, root places some emblems on the folder (like in windows) but those emblems do not appear for the actual user, because this is not the actual user that did the job.
