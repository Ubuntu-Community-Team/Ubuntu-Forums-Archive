### Post by yabbadabbadont on 2007-12-20
If you mean that you want to dual boot (be able to boot into either windows or ubuntu) then it is usually best to install windows first, then ubuntu second.  The ubuntu installer handles configuring your boot manager (grub) so that you can load either OS.

If you want to run windows inside of VirtualBox, then there are tutorial threads about it here in the forums somewhere.  :D
