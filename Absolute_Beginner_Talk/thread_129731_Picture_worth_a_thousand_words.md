Try firing up Synaptic...
(and if that doesn't work, you'll need to open a terminal and apt-get)
Search for gtkpod, tick it and it will load...
But you might have to enable the universe repository...
And this might mean edititing sources.list...
And even then, it might not be the latest version, so you could add in the Debian sid repository as custom...
But you'll need to know about GPG keys...
Or else how to download and install the .deb...

...etc.

(Oh...and by the way...here are some other things which might work better for your iPod, like amaoK or Banshee...)
