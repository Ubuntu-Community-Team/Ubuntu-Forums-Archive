### Post by Paulmd on 2007-11-17
There is no tool to do both at once.

If it's just one shortish pdf, you might use Acrobat reader to select the text, and copy it onto one of those web based translation services.

None of the auto translators are great, for any language, just yet.

[http://babelfish.altavista.com/](http://babelfish.altavista.com/)

[http://www.google.com/translate_t](http://www.google.com/translate_t)
