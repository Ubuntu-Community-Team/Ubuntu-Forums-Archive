### Post by FrozenFox on 2008-02-11
Gparted is the gnome partition editor mentioned in jan's post above. You can download and burn that iso and boot into it as a livecd. Alternatively, I think the ubuntu/ubuntu-liveCD comes with it if you boot to the desktop and try to find it. If not, I know you can open the terminal from start-> accessories and sudo apt-get install gparted  to install it.

### Post by Moop on 2008-02-11
Gnome Partition Editor is available in the repos and is also on the live cd. You can't edit a partition that is mounted so it's usually better to use a bootable cd.
