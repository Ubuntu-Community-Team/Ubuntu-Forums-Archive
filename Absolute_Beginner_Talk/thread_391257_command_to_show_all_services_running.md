### Post by kostkon on 2007-03-23
Apache will run from root, so if you type this on a terminal, you'll see if Apache is runing. 
```
ps -fu root
```

or even better search if the processes that root runs contain the word apache somewhere
```
ps -fu root | grep apache
``` 

or if you do (high possibility that you'll have Apache 2)
```
sudo /etc/init.d/apache2 stop
```

if Apache is not running you'll get the appropriate message from this command.

Also, as an added measure, try to see if you get anything from [http://localhost](http://localhost) (or [http://127.0.0.1](http://127.0.0.1)) in Firefox.
