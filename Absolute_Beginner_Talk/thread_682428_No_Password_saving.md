### Post by jaytek13 on 2008-01-30
Well, first off, you don't need to "install" a firewall. A lot of linux newbies fall into this trap... A firewall is built-in, called iptables. The normal "firewall" that you would install (for ubuntu it would most likely be firestarter) acts only as a GUI frontend for iptables. Ubuntu, by default, comes with firewall settings for iptables that most users can live with.


That said. I've never found a need on any system to not save passwords in firefox. The passwords that are saved are encrypted, 

The fact of the matter is that iptables by ubuntu default provides a very secure system. The foremost concern with security is keeping every application up to date, so taking advantage of the automatic updates as soon as they are available. Otherwise, for a 'normal' system, I wouldn't be concerned with saving passwords in firefox.

As far as "I can't update anything," is that a seperate issue? If so, what do you mean? Do you get an error?
