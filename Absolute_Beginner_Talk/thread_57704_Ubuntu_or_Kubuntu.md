### Post by aysiu on 2005-08-17
[QUOTE=besby11]Im a n00b to linux and I need to know wuts easier  for a n00b kubuntu or ubuntu[/QUOTE] Either. It all depends on whether you like KDE or Gnome better. If you don't know which you like better, I'd go with Ubuntu, because there's better documentation and support for Gnome on Ubuntu than KDE on Ubuntu. For example, if someone says "Type *sudo gedit /etc/fstab* in the terminal," that's not going to help you edit the fstab file in Kubuntu because there is no gedit command in KDE.

I'm assuming it's possible, but is it more trouble than it's worth, especially for a Linux beginner?[/QUOTE] There's no rush on this. According to Kubuntu's website:

*Kubuntu is the first Ubuntu derived distribution. Our Kubuntu CDs are made up of Ubuntu's base plus KDE. You can get exactly the same effect by installing Ubuntu and adding the KDE packages (and removing the Gnome packages) from the Ubuntu archives.*

So, yeah... you can just install KDE.
