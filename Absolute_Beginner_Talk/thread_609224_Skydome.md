### Post by mcduck on 2007-11-11
The rule is that the dimensions of the picture need to be powers of 2. So for example 512x512 works, 1024x2048 works, 512x4086 works, but 1000x1000 won't work.

The maximum dimensions depend on how much memory your graphics card has, so you need to try yourself how high resolution image you can use. And as the image is mapped into spherical object from which some of the top and bottom are hidden, you'll need more horizontal than vertical resolution to get a good looking skydome.
