### Post by TheAlmightyNilz on 2007-01-26
The 32 bit version of Ubuntu has far more support for what you have most likely. I tried the 64 bit version the first time, and just was unable to install my graphics card drivers properly, also alot of plugins for Firefox are not supported, rumor has it.
So I was initially bummed about downgrading to 32 bit, but as far as I can tell, there's no sacrifice that's obvious. The speed is still there, AND I had no trouble installing my graphics drivers in the 32 bit version.

So I'd say don't worry about the 64 bit version right now, the 32 bit is the best bet.
