### Post by ropers on 2008-04-14
Warning: gThumb doesn't get the timing right. So if you're watching an animated GIF with gThumb, chances are the animation will run too quickly. 

As an example, watch _[this animated GIF]("http://commons.wikimedia.org/wiki/Image:Puffyanim.gif")_ with Firefox and with gThumb and compare.
