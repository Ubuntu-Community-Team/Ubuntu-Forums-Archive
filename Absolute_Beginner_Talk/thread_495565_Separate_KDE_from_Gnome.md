### Post by tarek on 2007-07-08
Hi, I have Ubuntu desktop and installed kubuntu-desktop but now the menu is too big to fit my screen :)

Is there a way to _not_ show KDE apps in Gnome and _not_ show Gnome's in KDE? Then if I want to run an application I can just use the terminal.

TK

### Post by chuckyp on 2007-07-08
The only way is to manually edit the menus yourself.  In gnome right click on applications then select edit menu.  Just go down the list and uncheck anything you dont' want to show up in your menus.
