### Post by matthewv on 2006-01-12
if you just install xubuntu on top of gnome, nothing from gnome will be erased. XFCE (the window manager used by xubuntu) will just be selectable from the login screen. If you select gnome, absolutely nothing will be changed. to install xubuntu, ensure that you have the universe repository enabled and then run ```
sudo apt-get install xubuntu-desktop
```

### Post by Sef on 2006-01-12
No, your Gnome files won't get erased.  Xubuntu, Kubuntu, and Ubuntu are just desktops.  What is underneath is not affected.  It's like changing from formal wear to a bathing suit.  You may appear different, but you are still the same person underneath the clothes.

### Post by jimrz on 2006-01-12
I have not yet tried xubuntu, though I understand that it is noticably faster than gnome or kde (which both run about the same on my system). There is no need tu lose your gnome stuff, just install "xubuntu-desktop" using synaptic (you will need to have the universe repositories). Once installed, you will have the choice at login to select which wm you use (this is under the "sessions" button on the gnome login screen, this will leave all of your gnome stuff in place and you will be able to use your gnome apps from xfce wm.
