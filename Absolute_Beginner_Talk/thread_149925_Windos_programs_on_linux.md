### Post by towsonu2003 on 2006-03-25
there are a number of options: wine (free, run win programs under linux, may fail), crossover office (run win programs under linux, not free, may fail, has trial version to try out), qemu (run windows itself under linux, free, you will need to reactivate your windows and compile qemu with kqemu support for better performance -there is howto in forum section tips&tricks), vmware (ru windows under linux, not free). 

I'll try in this order: 
1. wine (free)
2. qemu compiled (free)
3. crossover trial (not free)
4. vmware (not free)

PS. do dual-boot windows + linux for the first year, just in case...
PSS. ask maintainers of your specialized software to port them to linux.
