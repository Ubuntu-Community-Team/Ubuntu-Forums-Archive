### Post by tyggna1 on 2007-08-29
Believe it or not--most routers have a very small form of linux on them to run it more efficiently.  If anything--having another linux system would be a friend to your router and a blessing.  

Joking aside:  It is possible that you Linux machine is set to use wireless B instead of G.  Wireless B is the default on a number of linux drivers, and runs at 1/5th the speed of G.  Since B and G are on the same radio frequency, all the computers on your network would have to knock down from wireless G to accommodate for the single wireless B computer.  It would be worth looking into.
