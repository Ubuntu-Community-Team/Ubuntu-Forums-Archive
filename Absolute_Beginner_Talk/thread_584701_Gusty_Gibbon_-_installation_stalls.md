### Post by Qwerty_ on 2007-10-21
Excellent. Once you have installed and if you wish to install other packages (no doubt you will) I suggest you do the following to use another mirror.

System>Administration>Software Sources>Download From>Other>Select best mirror.

That way you will download from the mirror which you get the best speed from.
