### Post by ryanVickers on 2007-10-03
what do you mean by "application bar icon"?  The icon in the "Application, Places, System" bar?  If so, then find a good kubuntu logo, and then overwrite the file "start-here.png" in the human icon theme folder (/usr/share/icons) under every category you can find a "places" folder in.  The scalable folder will need an svg version however...
