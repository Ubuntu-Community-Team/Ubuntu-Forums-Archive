### Post by jordanmthomas on 2007-09-15
I'd just like to pop in here real quick and say that beryl is no longer being developed and there will be no more bugfixes or anything.

You're more likely to have success with compiz fusion (beryl merged into this project).
Just look up compiz fusion on the forums and you're bound to find some instructions.

**edit** Also, Ubuntu comes with compiz (not compiz-fusion) installed.  Look in system --> preferences --> desktop effects

### Post by por100pre1 on 2007-09-15
Go ahead and install beryl from the repos (in terminal):

```
sudo aptitude install beryl beryl-manager emerald emerald-themes
```

The next version upgrade will take care of removing beryl safely so go ahead and install it. :)

(Installing software versions out of the official repositories can cause troubles later, so be warned.)
