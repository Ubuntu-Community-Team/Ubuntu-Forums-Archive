### Post by Phantom784 on 2007-03-03
If you only have one hard drive and it is entirely taken over by your windows partition, you'll  have to shrink it to make room for Ubuntu.  To do this, I'd recommend a tool called Partition Logic.  This is a free program that you put on a floppy disk, boot off the floppy, and then resize your NTFS partition.  Before you use this program, you should defragment your hard drive from within windows.  After you're NTFS partition is shrunk, you can install Ubuntu and select the option not to delete any existing partitions (forget exactly what it says, haven't installed in a while, but I remember that it's pretty obvious what the right option is when you're installing)

Good Luck!!!

For more information about Partition Logic, visit [http://partitionlogic.org.uk/](http://partitionlogic.org.uk/)

### Post by Phantom784 on 2007-03-03
Installing Ubuntu is basically how you described.  Once you boot from the Live CD, there will be an icon on the desktop that says install.  Click it, and you'll be asked a bunch of questions.  Most of it is pretty simple stuff, like your name, a password, your time zone, the current time, a name for the computer (just pick something that sounds cool, or maybe just call it UBUNTU if you can't think of anything).  One of the questions is about partitioning your hard drive.  There should be an option to only use free space.  Choose this option (assuming you already made free space on your drive by shrinking windows).  After you answer all the questions, it'll install, then reboot.  If anything goes wrong, just ask on here (and if you somehow screwed up your computer so it won't boot at all, which I doubt you'll do, you can always use your Live CD to get online.)
