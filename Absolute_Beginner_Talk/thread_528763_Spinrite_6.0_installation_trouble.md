### Post by Jerry N on 2007-08-18
Spinrite does not have to be installed.  Just copy Spinrite to a floppy disk,   boot the computer from a bootable DOS disk (floppy or CD, any version of DOS including DR-DOS), insert the disk containing Spinrite, and run the Spinrite program.  And be very very CAREFUL!  Don't attempt a low level format unless your disk already appears to be a hopeless basket case.  

I have used Spinrite since its earliest days but I find little need for it now except when I suspect a hard drive is going bad.  

Jerry
