### Post by kleo skywalker on 2008-01-10
You can install the **sun-java6-jre** package from Synaptic. If you downloaded Frostwire (the .deb package) from its website then it should show up in Applications-->Internet-->Frostwire. If it doesn't, see System-->Preferences-->Main Menu-->Internet to see if the box next to Frostwire is checked.
