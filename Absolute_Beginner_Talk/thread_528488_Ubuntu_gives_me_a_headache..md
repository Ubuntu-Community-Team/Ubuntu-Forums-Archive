### Post by daveshields on 2007-08-18
I would suggest waiting until you have decent internet access before trying to learn more about Linux.

Almost all work on open-source, including Linux, is done in full public view on the internet, so there is a great deal of information available, of which this forum is but one example.

To go without this resource is to make learning Linux much more difficult, and definitely less fun.
