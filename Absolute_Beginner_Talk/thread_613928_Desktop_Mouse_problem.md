### Post by dajhorn on 2007-11-15
When you drag-and-drop the file, hold down the ALT key before you release the mouse button and you should get this menu:

[LIST]
[*]Move Here
[*]  Copy Here
[*]  Link Here
[*]  Set As Background
[*]  Cancel
[/LIST]

Choose "Move Here" from the context menu to get the desired result.
