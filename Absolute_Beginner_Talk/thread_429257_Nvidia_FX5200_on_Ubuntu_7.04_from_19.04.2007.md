### Post by mahiyar on 2007-05-01
No need to install automatix (though it is a good programme), simply go to >System>Administration>Restricted Drivers; there you will see Nvidia box, tick it, the drivers will get automatically installed and thats that. BTW I also have the same card. No problem in installation.

### Post by jjbean on 2007-05-01
I have the same card. This is how I did It. Go to the System menu, Administration, Restricted Drivers Manager. Like this: 
System > Administration > Restricted Drivers Manager. Than check the box for Nvidia and reboot, fast and easy.
