### Post by 3rdalbum on 2006-11-07
96 megabytes of RAM is unfortunately not enough to run Ubuntu. (for the record, virtual memory is space on the hard disk which is allocated by the operating system - so if your computer has no operating system on it, it has no virtual memory).

You'll have to upgrade the RAM to 256 megabytes, or download Xubuntu for PowerPC (PowerPC, often abbreviated to PPC, is the type of processor your Mac has). On these machines, upgrading the RAM by this amount is not something you can do on your own - you will need to go to an Apple centre. When you have done either of these, read on.

Insert the disc before booting up. Immediately after pressing the power on button, hold down the C key. If all goes well, the computer will start the Yaboot boot loader, and you will be able to hit the Return key to boot Ubuntu.

Note that Xubuntu doesn't have the same look as Ubuntu. Ubuntu/PPC (Mac) will look exactly the same as Ubuntu/x86 (PC). If you want to find out more about Xubuntu, visit [www.xubuntu.org](www.xubuntu.org).
