### Post by chewearn on 2008-01-03
> **kiwi_bloke said:**
> When I use the google search engine I get google.de (de=Germany) and not google.com. Even when I type [www.google.com]("http://www.google.com") into the address bar, Firefox changes it back to google.de. How can I correct this? I'm using Ubuntu 7.10 in English.

In the google main page, there is a "preference" link right side of the search box.  You can set the language to English.  One thing though; you need cookie to be enabled for google, else it will forget what you set.

Another method is to add a google country specific search engine to the toolbar.  Go to:
[mycroft.mozdev.org]("http://ubuntuforums.org/mycroft.mozdev.org")
Select Google link, scroll down the very long list to get to your chosen country.
