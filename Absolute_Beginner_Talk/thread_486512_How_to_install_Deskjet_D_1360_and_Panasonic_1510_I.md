### Post by randomnote1 on 2007-06-28
I'm not sure about the Panasonic printer, but for the HP you should be able to use one of the generic HP Deskjet drivers for it.

go to:

System -> Administration -> Printing

when the screen pops up select Local Printer 
then click Forward
The select the Manufacturer
and then under model select one of the Deskjet drivers.  If none of them work let us know what heppens.
