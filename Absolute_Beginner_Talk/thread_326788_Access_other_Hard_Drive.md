### Post by gh0st on 2006-12-28
The NTFS driver that comes with Ubuntu can only read, you need to update it to something like NTFS-3G or NTFS-Fuse. There is a really good How-To guide in th following thread that should help you.

[http://www.ubuntuforums.org/showthread.php?t=217009](http://www.ubuntuforums.org/showthread.php?t=217009)

This thread is written by an NTFS driver developer and he gives some really good advice. If you post a problem he will answer you personally which is cool. There is also a guide on Lunapark on how to install NTFS-3G for Ubuntu. You might want to look at that as well, here's the address - [http://lunapark6.com/?p=1710](http://lunapark6.com/?p=1710)

Hopefully that should sort it out for you. If you need anything else let me know :)
