### Post by ChaOConnor on 2007-07-16
The logic is sound, however as programs like Oblivion are not native to Linux (Ubuntu), it won't run smoother or even at all possibly.  You'd have to go through something like Wine or even run a Virtual Machine and install it there (if that would even work???) but both bring up the question of compromises.  For instance, Wine isn't complete, so Oblivion may not have all the bells and whistles or even be playable.
