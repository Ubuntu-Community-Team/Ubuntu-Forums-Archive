### Post by Arevos on 2006-10-12
You could try using the keyboard to navigate between buttons using 'tab' and enter. What happens when you push your cursor against the bottom right corner of the screen? Also, I believe you can move windows around by holding ALT and dragging, which should help you to get to the appropriate button.
