### Post by bobplano on 2007-09-25
oh this is going to be fun. install windows like you normally would but have the drive plugged in, and choose to install windows on the external drive. try to make windows doesn't write over the internal drive's mbr or that's just another step for you. now go to your BIOS and set usb devices or drives to boot first, then your internal drive. 

this means that you are going to have to unplug and plug the external drive, but GRUB doesn't like it when an OS goes missing, so i think that's out.
