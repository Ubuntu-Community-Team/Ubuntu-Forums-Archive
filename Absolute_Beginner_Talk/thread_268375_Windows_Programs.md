### Post by indytim on 2006-09-30
ProjectWhat,

I'd like to suggest that you take a cruise over to Wine at [http://www.winehq.com/]("http://www.winehq.com/") and check the database against your favorite app's.  There are a lot of good windows programs that will run in Wine.  Another choice would be CrossOver (I believe it is an app that is commercial - ie. cost money).  

I have found that not all app's run in Wine (I suspect the same in CrossOver also).  That being said, your choice is 

1.  Hunt for a suitable native "Linux" app as a replacement.
2.  Install VM and run Windows against it.
3.  Install Wine or CrossOver and run your app.
4.  Dual boot Windows and Linux.

I currently dual-boot because there is one remaining app that I cannot get to work in Wine.  All of the remaining work I do is in Linux (Ubuntu-Gnome & KDE).

Hope this helps.
