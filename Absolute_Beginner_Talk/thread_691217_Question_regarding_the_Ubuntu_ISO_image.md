### Post by taurus on 2008-02-08
Do not unpack the ISO image that you've downloaded; instead, use a program to burn it to a CD as an image file.

[https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28burn%29%7C%28iso%29](https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28burn%29%7C%28iso%29)

### Post by bumanie on 2008-02-08
An iso is an image of a cd. You need to burn the image to cd. In a burning program there should be a choice in the various menu's to burn an image to disc (or similar wording). Once you have done that, your live cd should be able to be booted. When burning any disc, [although most discs and burners operate at much higher speeds], it is advisable (to reduce burn errors) to burn your discs at 4x.
