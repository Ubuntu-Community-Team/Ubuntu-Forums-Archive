### Post by Anonii on 2006-08-31
Nautilus is the standard file browser. If you open your home directory, you probably open it with nautilus. You can also open it from the command line, with the command: nautilus .
Or even: Press "Alt + F2" and then type nautilus in the box, and then enter :)
