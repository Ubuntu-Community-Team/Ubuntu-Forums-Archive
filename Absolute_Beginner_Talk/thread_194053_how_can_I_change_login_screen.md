---

### Post by aysiu on 2006-06-10
Metacity and GTK themes will modify how your windows and controls will look *once you've logged in*.

GDM themes control what your login screen looks like (this is *before you log in*).

To install GTK and Metacity themes, go to System > Preferences > Themes and drag and drop the .tar.gz to the Theme Manager window.

### Post by aysiu on 2006-06-11
For all practical purposes, there's very little difference between KDM and GDM to the end-user.

The only tangible differences I can think of:

1. GDM themes are easier to install
2. In KDM, you can switch desktop environments without being asked questions. In GDM, you'll always be asked if you want to make the new environment the default or use it just for that session.
