Kubuntu comes with different apps for many things. The OS is the same, just the looks, and the difference in apps. You can use those apps on Ubuntu too.

Install it for yourself on Ubuntu:

For everything:
```

sudo aptitude install kubuntu-desktop

```

OR

For smaller package:
```

sudo aptitude install kde-core

```

You'll have both, and be able to choose which DE to use when you log on. You'll be able to use all the apps.

There are a few minor issues that are easily overcome.  You can still run almost all Gnome/KDE apps on both Ubuntu/Kubuntu. as a note : kubuntu has shutdown issues in some computers

if you are asking it in a sense of comaprison, then i think this question has been discussed a lot everywhere on internet, and the ultimate result is that it depends on your own prsonal choice, which you develop with time as you go on using the two DE.

in short, 

GNOME : simple, focus on usability, less configuration options in user hand, but default configuration are itself the bes, default apps are getting better day by day, cleaner & beautiful interface.

KDE : lots of options are there to configure, u can spend the whole day, apps are generally better than gnome's, definitely easier for people coming for windows, in other words, interface seems more like windows.
