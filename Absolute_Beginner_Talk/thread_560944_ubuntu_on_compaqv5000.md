Search around for the model of your motherboard (because I see that the Compaq Presario V5000 has integrated wireless networking) and see if it has given any other Linux users problems with their wireless networking. As for installing, if (as I think it is) the AMD Sempron is a 64-bit machine, you will need to install the AMD64 version of Ubuntu (as opposed to the x86 version).
