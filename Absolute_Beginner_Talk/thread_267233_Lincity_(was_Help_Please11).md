### Post by happy-and-lost on 2006-09-28
In Ubuntu, there's a huge central database of programs. Rather than sift through adware are crap you find on Google like on XP, you find what you're after on this database.

It's easily accessed via something called "Synaptic" (System-->Admin). In Synaptic you can search using keywords for what you're after. Open up Synaptic, type lincity into the search, and install the package. It's all done and set up automatically.

Much easier :)

Threads have been merged.  Pleas refrain from posting duplicate threads.  Also, please try to use your thread titles to briefly describe the TYPE of problem you are encountering so that users who know how to help you can know that they CAN help you.  Appealing to the mercy of the community is bad form and may not always result in good technical advice.

Software installation on Ubuntu is not, repeat NOT like software installation in Windows.   For general help on software installation, consult the [InstallingSoftware wikipage](https://help.ubuntu.com/community/InstallingSoftware)

Lincity is available from the 'universe' repositories.  Enable 'universe' and 'multivers' in the Synaptic Package Manager and search for Lincity.  Install it, and it should run.
