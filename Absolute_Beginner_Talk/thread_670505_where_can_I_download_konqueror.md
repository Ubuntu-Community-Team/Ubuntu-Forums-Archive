### Post by Cypher on 2008-01-17
The more definitive program is the Systems->Administration->Synaptic Package Manager. You can search for "konqueror" there..or from the command line
```

sudo apt-get install konqueror

```

### Post by DrMega on 2008-01-17
Or you could install kubuntu desktop (also from Synaptic). Slightly overkill but it will bring KDE and all the KDE apps that make up Kubuntu, while at the same time giving you the choice to use Ubuntu or Kubuntu when you login.

As I said, it is overkill, but if disk space is not an issue for you then it does give you plenty to play with.
