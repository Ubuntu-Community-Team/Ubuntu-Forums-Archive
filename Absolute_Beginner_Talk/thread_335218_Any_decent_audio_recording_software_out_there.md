### Post by justin_c18 on 2007-01-14
Ardour is a good studio suite, quite easy to use, a lot of plugins, and it uses JACK, which'll support many plugins. It's pretty much the hotrod of recording tools right now. At least I think so anyway.
I'm not too sure if alsa is supported. I'm sure it is. But just to make sure, check out; [http://ardour.org/]("http://ardour.org/")
Check out the system requirements before implementing any installation. [http://ardour.org/system_requirements]("http://ardour.org/system_requirements")

It might take a bit to get used to, but it rocks.
