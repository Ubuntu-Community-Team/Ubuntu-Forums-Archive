### Post by Flying caveman on 2007-12-01
cpufreqd and powernowd are two programs that do that.  

They pretty much the same thing you can install one or the other but you don't need both.

System>Administration>Synaptic Package Manager  search for either cpufreqd or powernowd to see if it is already installed (Green check box means its installed)

If one of them is installed you could try the other one.  

I would add the cpu frequency monitor to one of your panels so that you can see if its working, if you haven't already.  

If you still can't get it working, don't give up there's probably a way. I think I had to edit a line on some file but I can't remember which one right now.
