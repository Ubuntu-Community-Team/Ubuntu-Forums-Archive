### Post by Paqman on 2008-04-05
Try [Getdeb](http://www.getdeb.net/)

You'll have major hassles with dependencies though. Every package has a list of other packages that it depends on. These packages might in turn depend on other packages, and so on. This is a problem known as Dependency Hell. Normally your package manager deals with all this automatically when try to install something, but it can't do that if you're not online. You'll have to provide all the dependencies yourself.

Personally, I think it might be easiest to just plug your PC into the router, get everything set up how you want it, then move it to your room when you're happy.
