### Post by defrex on 2007-11-02
Right click on the desktop and select "Create Launcher...". Command is the same as what you would type into the command-line. You can drag the shortcut to a gnome panel if you want, too.

### Post by jpietari on 2007-11-02
FYI:  You should be able to find all the apps you need under "Add/Remove...".  This will handle the menu entry for you.

To add entries to your menu all you have to do is right-click the menu "Application" and select Edit Menus.  From the screen, you can create a "Launcher", which is a shortcut to your app
