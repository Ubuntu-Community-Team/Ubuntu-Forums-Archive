### Post by julian67 on 2007-10-03
If you installed your applications from Ubuntu and canonical repositories you should find everything is upgraded along with the OS. If you have any 3rd party repositories you might run into some problems but not necessarily. Check the 3rd party repositories to see if they have a Gutsy repo, but chances are it will all work. Applications you installed from source you will most likely need to recompile and re-install.
