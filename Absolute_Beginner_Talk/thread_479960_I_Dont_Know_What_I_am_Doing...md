An ISO file is a CD Image. You will need to burn the CD with a software package that supports Image Files (eg. Nero)

Instead of burning the ISO file onto a CD, what it will do is burn the CD Image, which is contained in the ISO file and has a full structure.

What operating system and CD burning software are you using?
