### Post by imaginos28 on 2007-07-05
> **Nekiruhs said:**
> Contrary to what everyone one else says here I will say this. Install Ubuntu Studio, that is designed with artists in mind. Then install VMWare Server. This allows you to install XP in a "fake" machine and run it while running Ubuntu Studio. This way you can have all your apps in both Ubuntu and windows at the same time. You can have your cake and eat it too.

Being an artist (and someone who can't live without Photoshop ;)) myself, I'd have to recommend this route also (with the addition of dual booting Windows with Ubuntu). If you could afford it, buying a Mac really would be the ideal solution, as most of the apps you listed above are native on the Mac. When I was in a similar position of hating Windows it's what I ended up doing...but that left me with this relatively new dual-monitor system that I didn't want to use anymore because Windows was on it. 

But for the past week I've been dual booting XP with Ubuntu Studio. On Ubuntu Studio (which I just love :D) I have VMWare Server installed and using a Windows 2000 virtual machine. Now, for all general computer usage...browsing,checking email, word processing, listening to music etc... I wouldn't dream of going back to Windows.

System Specs:
AMD Athlon X2 3800+ (2Ghz)
Biostar NF4U motherboard
1GB Patriot DDR2-6400 RAM
eVGA NVidia 7600GT
250GB WD Hard Drive

Virtual Machine Apps:
Photoshop (CS2) works fine, if only a bit slower. I have not tested any other Adobe applications. I can say, however, that modo refuses to run at all in the virtual machine. Otherwise, Windows runs pretty much like Windows always does, considering the limited amount of RAM (400MB) I allow it to use. You have 2GB so you could give it more and it should run faster. Also, it's pretty sweet being able to run Windows full screen on one monitor and still being able to use Ubuntu on the other.

Linux Apps:
Blender is preinstalled in Ubuntu Studio and ready to go. I'm sure Maya is available in a Linux version. If that's too expensive, I've heard of people running Cinema4D in wine, but apparently it's very complicated....:-?

In terms of 3d modeling, there is a version of Houdini for Linux that might be worth checking out, but neither ZBrush nor modo have Linux versions and modo doesn't really work on the virtual machine. (I've never used ZBrush, so I have no clue how that will work on the VM lol)

As for video editing, I've used Kino to capture some DV from my camera. Cinelerra (which has to be installed separately) looks pretty good so far. I've only edited one home video with it, but it did what it's supposed to do. It might take some time to get used to if you're accustomed with Premiere. I also needed to install Avidemux as I constantly find myself transcoding my mythtv recordings lol. I don't think anything could replace VirtualDub :( (which btw runs on the virtual machine..and not too slowly even :o), but Avidemux comes really close. If you do any sort of work with video on the web it's a must have.

In the case of audio...Well, my Mac is still the center of my home studio setup, so I really can't comment on the Ubuntu Studio audio apps.

All in all, I don't think you'll regret it. Go ahead and take the leap. Worst case: you can't find 3D apps that you like in Linux and you have to reboot into Windows when you want to work in 3D programs. Best case: You become a mad genius at using Blender :D

BTW, the native Windows installation might still be the way to go for gaming.....but I'd at least give wine a shot first. VMWare doesn't seem to have much in the way of 3D acceleration.
