### Post by useResa on 2007-09-11
Personally I do not use apt-spy, but it is -- just as in Debian -- available in the Ubuntu repositories. So you could install it.

However, doing a search on apt-spy I came across [this bug]("https://bugs.launchpad.net/ubuntu/+source/apt-spy/+bug/1780").
Therefore I am not sure if you should install and use it, based on the information in the bug I personally would not.

Information on changing your sources.list can be found in the Ubuntu documentation. For example [here]("https://help.ubuntu.com/community/Repositories/Ubuntu") (using a GUI) or [here]("https://help.ubuntu.com/community/Repositories/CommandLine") (using the command line).

Hope this helps.
