### Post by IYY on 2006-03-26
> -s -l -H

The meaning of the argumnts depends on the command in question. For example, ls -l means "long listing", ls -H means "follow symbolic links", ls -s means "print the size of each file listed". But if we're talking about a different command, these args could mean something else, or nothing at all.The best way to learn about the arguments of a specific command is to use the manual (man <command>).
