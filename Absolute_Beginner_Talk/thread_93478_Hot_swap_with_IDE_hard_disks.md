### Post by suRoot on 2005-11-22
No.

High-end / server class systems allow you to hot-swap, but with a SCSI disk.  So far as I know there is no way to hot-swap an IDE drive of any kind.

If you want an easy way to attach drives, check this out:
[http://www.newegg.com/Product/Product.asp?Item=N82E16812156001]("http://www.newegg.com/Product/Product.asp?Item=N82E16812156001")

That's just an example, but an IDE to USB converter is really nice to have.
