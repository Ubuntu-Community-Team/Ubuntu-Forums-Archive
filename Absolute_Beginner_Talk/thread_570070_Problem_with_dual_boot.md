### Post by J11Gyro on 2007-10-07
I have been running Fiesty since initial complete release and off and on have had a problem with XP shutting down even though it was on a seperate drive that is master. This has been going on every since I did the fresh install on a new build for both OS's, well I found what was causing it I think and it might be happening to some others that may not have figured it out. I have been going into safe mode to boot up XP every so often when it would freeze and reboot on its own. Well I noticed that the option disable auto restart on errors was open. Some how the option was either by XP default or got switched by family member but I turned it off and everything runs like it is supposed to now. It seems to have sped the system up alot also. Just post for FYI something you dual booters that have to have windows might check if having problems.
