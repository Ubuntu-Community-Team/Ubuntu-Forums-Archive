### Post by pseudonym on 2006-12-31
Hmm...if you're able to boot and burn after flashing the firmware, but still not play or read anything, it sounds like the problem is either with the discs you're using, or still with the drive itself, rather than any system configuration issue (I'm assuming you have a default install and have installed the relevant codecs for media playback).

another OS would be a good way to test this. But seeing you don't have windows you'll need a live CD linux which allows you to run it without the CD in the drive. A good candidate for this is [Puppy Linux]("http://www.puppyos.com/"). It's a small download, an easy setup, and should allow you to test your drive effectively.

HTH :)
