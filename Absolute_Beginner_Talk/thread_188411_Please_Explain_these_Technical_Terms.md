### Post by ape on 2006-06-04
/bin/bash is the path to the executable called 'bash' (a command processor or shell).  In the DOS/Windows world, this would be the equivalent of c:\bin\bash.exe (or something like that).

For a good starting off point, head on over to [http://www.linux.com](http://www.linux.com).

Good luck!
