### Post by paradoc on 2007-09-04
you can hide whole folders.........Properties->Permissions-> 'list files only'/none/none........can only be accessed via _your_ password.(gui).  Individual files can be preceded by a period(full stop) using terminal, viz---cp agraphic.mpg  .agraphic.mpg and delete the original.

This works for me at the basic level, and is quite good enough to deter the casual peeping tom:)
