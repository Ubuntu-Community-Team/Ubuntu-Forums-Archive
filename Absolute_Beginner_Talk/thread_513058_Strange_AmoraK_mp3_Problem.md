### Post by RomeReactor on 2007-09-30
Hi. This might be a configuration error in Amarok; try looking in your home folder for a hidden amarok folder (if you're using Gnome, press CTRL+H to see the hidden files and folders and look for **.amarok**, which is where the configuration would probably be stored), and move it someplace safe, like inside another folder. Then open Amarok again so it recreates the configuration folder again and see if you can play MP3's now. If it works after that, you can delete the .amarok folder you moved.
