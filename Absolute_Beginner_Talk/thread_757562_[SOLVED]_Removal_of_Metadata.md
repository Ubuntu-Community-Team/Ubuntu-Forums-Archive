### Post by erginemr on 2008-04-17
You will need an extra metadata tag editor. I suggest you to try **Cowbell Music Organizer** ([http://more-cowbell.org/index.php/Main_Page](http://more-cowbell.org/index.php/Main_Page))
```
sudo aptitude install cowbell
```

or **Audio Tag Tool** ([http://pwp.netcabo.pt/paol/tagtool/](http://pwp.netcabo.pt/paol/tagtool/))
```
sudo aptitude install tagtool
```
