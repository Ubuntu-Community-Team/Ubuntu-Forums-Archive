### Post by Bruce2 on 2007-12-19
Hi, I know very little about Linux - Ubuntu, but am wondering if you guys can help me out with a  question.

How do I get it set up so that 2 people can use the computer under different usernames and passwords, with only one having administrative capabilities?

Also once this has been set up, how do I go about changing the passwords for both?

Thankyou!

---

### Post by macogw on 2007-12-19
When you install, the user you create has administrative rights.  After installing, log in as that user and go to System > Administration > Users & Groups.  Add another user and make them a "Desktop User" instead of "Administrator".  You can set the user's password at the bottom of that screen.

### Post by spiderbatdad on 2007-12-19
very easy...use System--Administration--Users and Groups...just add new user and set this user up as a desktop user (rather than administrator) from the drop down menu: Profile
