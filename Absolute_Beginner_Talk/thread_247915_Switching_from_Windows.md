### Post by justin whitaker on 2006-08-31
> **dness said:**
> Before I install anything I would like to know a few things:
Which wireless cards are the best to use

Wireless tends to be a crap shoot: some cards work out of the box, others require some tweaking to run. Many times, ndiswrapper can be used to install/use a card if you have some esoteric card.

> Can I sync with an Ipaq

Do not know. There are several efforts to sync linux with MS PDAs, but I have no idea what sort of shape they are in.

> Do I need virus and spyware scanners

No, not unless you really want it and/or are paranoid.

> Is it as easy as Windows to install and setup

Yes, but it is something quite different.  

> I know nothing about Linux but would like to get from Windows

Ask yourself why you are doing it, then do some Googling. There are significant downsides to running Linux (lack of hardware support, spotty game suppport, etc.), but alot of advantages as well (well coded, secure, you can make it how you want it, etc.).

Good luck!
