### Post by magicfab on 2007-02-28
There are only security updates for any package already released in any given Ubuntu version. feature update will usually only happen in the next Ubuntu version.

For example, in Edgy we have Firefox 2.0.0.1. If Firefox 2.5 came out tomorrow, only its security updates would be "backported" to Edgy's version of Firefox. The full feature 2.5 version would only be updated once you update to Ubuntu 7.04.

A few exceptions exist, for example for maintenance updates of Ubuntu LTS where hardware drivers and limited new features may be included in 6.06.2 (when it comes out).
