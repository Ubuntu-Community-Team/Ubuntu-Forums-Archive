### Post by sciencectn on 2008-03-12
When you say the card is 'recognized' do you mean that xubuntu is able to output with 3d acceleration?

By 'recognize' meaning output with a vesa compliant card, it should be able to recognize almost anything using VGA. However, 3d acceleration is not always possible because some cards don't have a Linux driver.
