### Post by DariusS on 2008-04-06
dual booting with windows is a definite possibility, a solution I use to play my favorite games (HL2, Oblivion).
as far as web based embedded games, etc. the only thing I've seen people have any resistance with is flash, but that has certainly been resolved, even with the x64 architecture. 

In short, download and burn the latest live cd, pop it in and boot. check out the desktop before you install. it's like a test drive.
one word of the wise with dual booting, if starting from a fresh hard drive (i.e. no windows) install windows first, THEN Ubuntu. Windows' boot loader has a way of ignoring Ubuntu (although there are ways to edit the windows boot loader to see Ubuntu, and vice versa with grub, just a bit more digging and possible headache)
if starting with a Windows install, you'll be fine.
