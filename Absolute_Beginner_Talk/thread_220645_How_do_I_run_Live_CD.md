### Post by catlett on 2006-07-21
If you have the live cd installer, starting the live session is part of the installation. So you press enter and the startup process begins. Once in the live session there is an icon on the desktop that starts the installer. You do not have to use it. You can leave it alone and use ubuntu in the live session.
If you do not have the live cd installer you may have the text based installer. You will know after a couple of windows that it isn't going to start a live session. It will give you an option to quit before it does anything to change your system.
