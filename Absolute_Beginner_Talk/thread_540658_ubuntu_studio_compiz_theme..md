### Post by mikaelsnavy on 2007-09-01
Yes, (I do the same thing). Go to the package manager and install "ubuntustudio-theme" and, if you want the icons, ubuntustudio-icon-theme". Then go to preferences, theme. Hit customize and on each tab, choose ubuntu studio where you want it.
