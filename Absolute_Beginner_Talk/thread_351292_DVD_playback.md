### Post by PurplePenguin on 2007-02-01
You can also install [automatix]("www.getautomatix.com"), which will take care of this and a bunch of other useful programs/drivers (Google Earth, Opera, etc).  Of course, it does make it very easy... you might prefer to do things via command line to learn a bit.  The [Ubuntu Guide]("http://ubuntuguide.org/wiki/Ubuntu_Edgy") will walk you through the same kinds of things, all copy and paste into the terminal.
