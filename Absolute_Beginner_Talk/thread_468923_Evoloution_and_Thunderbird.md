### Post by beow on 2007-06-09
There is certainly an option in Evolution to leave messages on the server, so you can get your mails in both Evolution and Thunderbird! If Evolution access the server first it only downloads, but does not remove the mails from the server, which then Thunderbird would do.  Usually this option is found under "server settings" or similar.

Should be no problem for you.
