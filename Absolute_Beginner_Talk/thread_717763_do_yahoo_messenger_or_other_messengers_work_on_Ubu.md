### Post by Hospadar on 2008-03-07
There is a version of flash that runs on linux, Photoshop will not work although there is a good alternative to it (gimp) and there are some difficult ways to make photoshop work.

most IM services can be used through Pidgin, ubuntu's IM client.

Applications written for windows won't work in linux, although generally there are good linux alternatives, or, you can run windows programs through a special program loader (wine) which sometimes works.  Sometimes too for big applications (like flash) there are linux versions released from the original developer.

### Post by herbster on 2008-03-07
[http://strabes.wordpress.com/useful-links-for-ubuntu-beginners/](http://strabes.wordpress.com/useful-links-for-ubuntu-beginners/)

Stroll through the links there, you will begin to understand.

Yes, you can use your messaging services with programs like aMSN, Pidgin, etc.

And Ubuntu is a distribution of linux, a completely different operating system from Windows. The exact same software in most situations does not exist or run correctly; you will be more than likely installing what are deemed "equivalents" to those programs. For Photoshop, there's Gimp, which comes with an Ubuntu install.

Have fun and be patient, learning this system is well worth it if you just accept that it will take some time.

### Post by chipants on 2008-03-07
speaking of yahoo messenger, if you want to use yahoo messenger's webcam functionality, pidgin doesn't support it. there is a KDE IM application called kopete that *does* support yahoo messenger webcam usage. it will run fine under gnome, assuming you have the required KDE libraries installed.

small detail, maybe unimportant to you, but worth knowing.
