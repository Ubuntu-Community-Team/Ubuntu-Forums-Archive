### Post by magiceraser06 on 2007-09-21
Your window borders aren't showing up because the gtk-window-decorator isn't selected.  Compiz doesn't know which decorator to use, so it doesn't use anything.

go into the COMPIZ SETTINGS MANAGER.  Look for the WINDOW DECORATIONS effect and in the box that says COMMAND  type in:   gtk-window-decorator

then hit ALT+F2 and type in   compiz --replace.  This will reload your compiz and now you should have your window borders.

Another alternative is to install Emerald, but the themes-package file isn't found on the repositories anymore, at least to my searching.

hope that helps.
