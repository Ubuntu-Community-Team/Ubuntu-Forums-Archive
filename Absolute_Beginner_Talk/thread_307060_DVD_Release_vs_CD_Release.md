### Post by aysiu on 2006-11-26
The DVD has everything in the main and restricted repositories--that means it includes Xubuntu, Ubuntu, Kubuntu, and Edubuntu.

The CDs have only one desktop environment each (Xubuntu with XFCE, Kubuntu with KDE, or Ubuntu with Gnome).

the dvd contains more packages and all three installer types with the dvd you can pick between the **live install** that gives you a full gui to test out/play with before you install or use as a rescue cd.

the **alternative install** the gives you a text based install that gives a little more in depth control on the install, this is also great for lower specd ie machines with 128mb and lower amount ram.

and the **server install** this installs a minimal set of packages for server use this install type doesn't include a gui since it's just more overhead for a server do not use this if your installing it for a desktop/workstation 

the default install cd only has the **live install** method and and fewer packages included on cd however it does install the full Ubuntu distro, it just doesn't include the extra packages the dvd does because they are available for download through the repos
