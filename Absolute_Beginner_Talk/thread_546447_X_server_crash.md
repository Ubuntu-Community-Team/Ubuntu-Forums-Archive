### Post by Beggar on 2007-09-08
This would be a temporary fix, but try 

```

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.incase
sudo gedit /etc/X11/xorg.conf

```

Then find the section Device and change the driver back to the nv driver.,

```

Section "Device"
...
Driver "nv"
...

```

 the run startx. If this fixes it then the problem is the nvidia driver, if not then we need to look elsewhere.
