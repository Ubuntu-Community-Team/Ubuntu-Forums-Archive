### Post by starcraft.man on 2007-06-18
> **captgadget said:**
> What is the difference between installing with a live cd or using the alternate cd. .

The Live CD boots into an aptly named "Live" environment, essential you actually boot up the near full os. You can then test it out, browse the web and of course click install from the desktop. It is somewhat resource intensive though and in some specific hardware cases it will fail. The alternate CD is a text installer, it works on any machine (has less requirements than booting the disk and is less complex). My preference is the text based installers, that may be due to my dos days though... 

> Seems like my machine won't boot with the live cd and it will with the alternate except when the alternate is almost done it fails the installation

Neither should be failing, certainly not the alternate at the end. Please verify they are good burns like Raja suggested by following this [guide.]("https://help.ubuntu.com/community/BurningIsoHowto") Pay careful attention to [md5 sums]("https://help.ubuntu.com/community/HowToMD5SUM") at EVERY step, it will ensure perfect CDs are produced. If they continue to fail even after CDs check out good, post the specific output messages and we will diagnose from there if needed.
