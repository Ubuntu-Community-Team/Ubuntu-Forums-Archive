### Post by tommytom on 2007-01-06
Try the "Save As" option in Open Office, unselect "Automatic file name extension" box - it's the bottom box on the left. Now add ".txt" (with out the quotation marks) at the end of the file name - for example "spreadsheet.txt" and save. Thats should work, just let us know ;)
