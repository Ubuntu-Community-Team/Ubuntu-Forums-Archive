Of course biggest problem is small RAM - you cannot expect to run Firefox. I just read review DeLi linux [http://www.linux.com/article.pl?sid=07/01/10/1529228](http://www.linux.com/article.pl?sid=07/01/10/1529228) but never tried it. 

It does not look like a task to get know Linux - if it does not work, it is not because Linux is hard to install, but because your hardware is so obsolete and installing *anything* on it is a challenge!

Laptops are hard to opgrade, and after 10 years, hard disk could have been exposed to lots of abuse and might be ready to trip to bit heaven. If it will not install smoothly, try some hardware tests, like memory and disk test.
