### Post by justsomedude on 2008-02-21
> **Georgie.Mathews said:**
> I read somewhere that it has something to do with de interlacing, but I cant find a solid solution to this.

This is quite possible. You could try a player like VLC which is able to deinterlace on the fly.

Go to *Video --> Deinterlace* and try some of the different methods to see if it helps:

[IMG]http://i30.tinypic.com/91lcfr.png[/IMG]

To install VLC:

```
sudo aptitude install vlc
```


Edit:

You could also try Totems built in deinterlacing method: Hit CTRL + I while playing.

However, vlc offers more options. :)
