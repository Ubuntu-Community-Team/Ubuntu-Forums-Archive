### Post by @trophy on 2007-06-29
There's two ways to find out if there's a program you're looking for in Linux.

You should try the Synaptic method first, since if that one works out it'll be a really pain-free install for you and life will be grand.  To search for stuff using Synaptic, do the following:

(These steps only need to be done once)

1: Click System, then Administration, then Software Sources. (give it your password)
2: If any of the checkboxes aren't checked, check them.  You need to activate all the software sources.
3: Click "Close".

(These steps are what you do every time you want to search for something)
1: Click System, then Administration, then Synaptic Package Manager. (give it your password)
2: If the reload button is lit up, click it.
3: Click "Search" and type in a keyword that you're looking for.  In your case "Director player".  You can also search by category if you're looking for something specific: Games, Network Tools, etc.
4: Search through the list of matches, and if one looks good, right click it and choose "Mark for Installation". 
5: When you're done doing that, hit "Apply" and your new software will be installed!  Piece of cake!

I've already done those steps for you, and there isn't any Director player that I could find... so I did what any self-respecting hacker would do... I turned to Google... keeper of the knowledge of all mankind...  that's the other way to search for software.

So I googled "linux director player" and it didn't look like there is one yet.

So it looks like the only way you *might* get it working is to install a program called Wine using Synaptic (it's a program which lets you run some windows programs under Linux) and see if it will let you install Director that way.  That's beyond the scope of my expertise, though, so you'd have to read up on how to use Wine.

Hope that helps!
