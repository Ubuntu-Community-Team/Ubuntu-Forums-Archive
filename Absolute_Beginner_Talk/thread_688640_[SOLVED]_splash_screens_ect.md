### Post by spiderbatdad on 2008-02-05
login splash screens can be found in System>>Administration>>Login Window>>Local. To add one, click the add button and navigate to the tar.gz package. I think you can also just drag and drop the package into the window of installed themes.

### Post by tangibleorange on 2008-02-07
Search for a program called "startupmanager" in Synaptic Package Manager and install it. You can then access it from "Start --> System --> Administration --> StartUp Manager". It has many options for configuring GRUB-style boot options, like the usplash theme. One of the tabs has an option for it.

Good luck!
