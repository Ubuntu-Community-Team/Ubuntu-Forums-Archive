### Post by orange2k on 2007-05-03
It is recommended that you use ethernet instead of USB if anyway possible. I did use USB for some time on Dapper, but it just didn`t work in Edgy or Feisty, so I had to install a ethernet card and plug my modem via RJ-45 cable. Thats the best way to go, best of all you don`t need any additional drivers, because your ethernet card should be detected and installed automatically along with your ADSL modem...
