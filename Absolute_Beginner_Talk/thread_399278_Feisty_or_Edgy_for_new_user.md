### Post by tbraun on 2007-04-02
Hello!

Firstly, welcome to Ubuntu! Secondly, I would also recommend Edgy for the same reasons that have already been pointed out in the other replies. An additional point to make: Helper applications, such as Automatix (which can easily set up a lot of the more challenging features for you) are not yet available for Feisty. If you want to go that route, Edgy will be the right choice.

Finally, sometimes there is only a fine line between 'leading edge' and 'bleeding edge'. So, personally, I will also wait a few weeks after Feisty has been out before installing it.

One piece of advice: If you can, make a separate partition for your /home directory. That way, you will be able to install your entire Ubuntu distro without loosing any of your files. You can start with Edgy, upgrade to Feisty (or do a fresh install) and if there should be any issues, you can simply go back by installing Edgy again. Your entire user setup and all your files will remain untouched by this. A separate /home partition is very highly recommended.

In fact, it's actually so easy and straight forward that I never just upgrade my distro, I always do clean installs, which tends to result in the least complications.

Tom
