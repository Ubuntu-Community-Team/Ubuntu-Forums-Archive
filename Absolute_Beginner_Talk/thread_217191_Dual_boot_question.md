### Post by ovimunt on 2006-07-16
It might sound really dumb but... make sure you haven't wipped you windows partition in the process of installing Ubuntu...

This wouldn't be the first disaster I've heard of. I keep telling people they should have a VERY good understanding of computer hardware before even thinking to install any linux distribution. ### Post by annda on 2006-07-16
if you don't see the menu make sure you are actually dual-booting (and didn't erase your win partition by mistake whike installing): see if you have your win disc(s) under /media - any file manager will show you if there is something mounted there and what it is.

if you still have your win partition use the ubuntu boot/install cd to install the GRUB boot manager and access win.
