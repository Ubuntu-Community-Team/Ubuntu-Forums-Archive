### Post by urukrama on 2008-01-19
Try MPD and a graphical client. My favourite is Gmpc, but Sonata is neat too. There are some howtos on these forums to help you set up MPD. It takes a little longer to set it up, but it is extremely light and handles large libraries very well.

[Gmusicbrowser]("http://squentin.free.fr/gmusicbrowser/gmusicbrowser.html") claims to have been designed "with big (> 10,000 songs) libraries in mind". Perhaps you'll like that.

+1 for MPD. It's probably the fastest& lightest music player with great library. I recommend Sonata, although ncmpc is a great client as well if you don't mind the ncurses-based interface.

How to install MPD the easy way:
```

sudo apt-get install mpd ncmpc mpc sonata
sudo ln -s /path/to/your/musics /var/lib/mpd/music
mpc update

```
