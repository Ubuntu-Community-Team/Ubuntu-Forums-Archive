### Post by Kobalt on 2007-05-20
I've had many troubles as well trying to play streamed videos with Mplayer, and it all came down to a lottery. So I now use Totem and it works just fine. If you use the gstreamer plugins (basically if you're under Ubuntu) you can install this package in order to get Totem to play all types of stream (if you installed the [apropriate gstreamer codecs]("http://http://ubuntuforums.org/showthread.php?t=413624")) : *totem-gstreamer-firefox-plugin*
If you use Xine (like in Kubuntu), install this package : *totem-xine-firefox-plugin*
