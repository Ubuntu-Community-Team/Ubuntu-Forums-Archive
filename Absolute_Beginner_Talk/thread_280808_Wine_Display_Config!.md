### Post by CatKiller on 2006-10-20
You can normally hold down Alt to move a window when you click-and-drag. If that doesn't work for you, I think Wine keeps its settings in the pretend registry, that's just a text file. I don't know off-hand which file this is, but you could find out and then edit the file by hand to more sensible settings.
