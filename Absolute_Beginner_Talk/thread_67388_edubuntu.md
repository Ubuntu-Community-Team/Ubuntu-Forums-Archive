### Post by davmac on 2005-09-20
As you start up your machine go into the bios settings. Its different for each machine, but you'll probably need to press the delete, F2, F12 or similar key.

In your bios settings it specifies the order it searches for booting up. Just make sure that the CD is before the hard disk, save the changes and reboot.

If this doesn't solve it, it could be a problem with the edubuntu cd itself. If you browse the CD, what do you see?
