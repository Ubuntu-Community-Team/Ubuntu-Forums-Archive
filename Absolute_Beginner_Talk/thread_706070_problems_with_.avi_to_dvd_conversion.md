### Post by jpietari on 2008-02-24
I use version 3.6 and it works all the time.

[http://www.getdeb.net/app/DeVeDe]("http://www.getdeb.net/app/DeVeDe")

These are the steps I take:
1. Launch devede
2. Click VIdeo DVD
3. Add my videos and ensure "Create an ISO or BIN/CUE image..." is selected. (see screenshot).
4.  Once the file is created (by clicking Next button), I navigate to it using Nautilus, right-click it, and select "Write to Disc".
