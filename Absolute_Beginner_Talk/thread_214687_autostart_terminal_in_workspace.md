### Post by nanotube on 2006-07-13
that is a function of the window manager (in default ubuntu case, "metacity"). and afaik, metacity does not give you the capability to start things in specific workspaces. 

so you could switch WMs, or you could just set a kb shortcut to open terminal (i have set f12 - very convenient), and whenever you need a terminal, just hit f12, and there it is.
