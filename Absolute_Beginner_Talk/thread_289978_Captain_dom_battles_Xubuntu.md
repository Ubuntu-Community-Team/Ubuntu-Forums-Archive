### Post by devilsadvocate1987 on 2006-10-31
the repos can be browsed online. go to ubuntuguide.org, find the section about multimedia codecs in it, find the names of the packages, google for the packages (dont forget to include ubuntu and edgy/dapper in the search terms) and find the hits from the ubuntu multiverse / backports  or wherever the package is. Download the debs and then do dpkg -i <package file name> 

The repository interface also tells you the dependancies of the package. Get those as well
