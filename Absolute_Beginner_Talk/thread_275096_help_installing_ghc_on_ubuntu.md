### Post by podunk on 2006-10-10
ghc6 is in the repositories. If you're using Ubuntu look at the top of your desktop
System>Administration>Synaptic Package Manager

When it opens click search then type in ghc6.

If it doesn't show up in your search than you need to enable extra repositories, see this page:
[http://www.psychocats.net/ubuntu/sources](http://www.psychocats.net/ubuntu/sources)

To install from a tar ball see this page:
[http://monkeyblog.org/ubuntu/installing/](http://monkeyblog.org/ubuntu/installing/)
