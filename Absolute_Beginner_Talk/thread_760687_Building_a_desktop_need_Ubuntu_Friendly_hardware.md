### Post by theone85ca on 2008-04-20
Is it safe to assume that you'll be using this computer to play games on? Considering you want the high end GFX card. I have a system running Ubuntu on an old Radeon 9000 other wise and it runs great!

Other wise I'd go with AMD/ATI. Two resons, im a fanboi! lol
Secondly, they work great together and wil continue to do so in the future especially since the merger of the two companies. I just picked up an ATI 3870. I think its a great card and its going to give you a little bit of longevity (as much as you can with hardware now-a-days). If your so inclined you'll be able grab a second, third or fourth card and keep that single card and system running happily along. One down side is that they dont currently have linux specific 3870 drivers, although the 2900 version drivers are wrking with out a hitch right now.

Processor...its a tough call, theres Quad cord AMD's now in the form of the phenom series but id stick with a nice 64 bit 5600+ AM scoket.

Grab an AM2 scoket mother board, that will buy you a little more time as well as far as it goes. AMD are going to be using that scoket for a while now. My Asus board runs pretty good with Ubuntu, as well it should considering they have the EeePC's that run linux.

Anyway, processors should be a problem as far as compatability, nor should RAM. The Asus mobo should run great too as stated above and I can vouch for the ATI card. :) 

That system is going to run XP/Vista nicly too!
