Is there a way to set up terminal preferences like changing the default size of the window when it opens, I have looked in the profile settings this option is not in there.

Thanks.

---

### Post by adam.tropics on 2006-05-23
if the launcher is on your panel or desktop, right click, properties, command

something like

```
gnome-terminal --geometry=65x16
```

---

### Post by adam.tropics on 2006-05-23
you could also do the same in System-->preferences-->preferred applications.
