### Post by taurus on 2008-03-01
You need to enable repos in /etc/apt/sources.list before you can install anything else on your machine.  While you are still in Add/Remove, click Preferences and then put a check mark in front of the first four lines.  Close it and click Apply changes.  Now, see if you can install anything this time.
