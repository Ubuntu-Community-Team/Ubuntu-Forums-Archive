### Post by notquitemichael on 2007-06-06
before anyone'll be able to help you you'll need to stop ubuntu from hiding what it's doing. to do this get the ubuntu CD in your computer and restart to bring up the install menu.
with 'start or install ubuntu' highlighted you'll have to press either F2 or F4 (what ever is said to bring up additional, or boot options.)
from this long command it pulls out you need to delete the words 'quiet' and 'splash' 
now boot the installer.
this'll make ubuntu boot telling you everything that's going on, you'll need to come back and post what ever's causing your ubuntu install to freeze and i'm sure some'll be able to help.
