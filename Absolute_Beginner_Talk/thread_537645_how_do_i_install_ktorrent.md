### Post by NovaAesa on 2007-08-29
Or if you don't like to use the command line (like me), goto Applications -> Add/Remove then type "ktorrent" in the search box and ktorrent should come up in the top right pane. Click the check box and then apply.

In Ubuntu, most applications can be installed from here, straight from the repository without having to download and compile source code.
