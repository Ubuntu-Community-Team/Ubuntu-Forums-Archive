### Post by erfahren on 2008-01-11
that new flash (v. 9.0.115) just doesn't work with Opera 9.2x - I've tried it

I've heard that it'll work on Opera 9.50beta

If you like to use Opera the Flash 9r48 *will* work (although it still is a little buggy).

Flash 7 is real stable with Opera - I actaully have Flash 7 for Opera and Flash 9 for Firefox (for the sites that require it).

if you want to use the new Flash 9 you need to get Opera 9.50beta
