### Post by robertrej on 2007-02-05
I have tried many free linux  accounting software apps. like Gnucash but needed a full
accounting package with inventory control so I went to TURBOCASH  which is FREE
and works very well .  I had to install the newest version of wine 0.9.30 but thats
also FREE. Both are easy to get, easy to install and support docs can be found to aid
you along the way.I had no success installing Quasar ,sql ledger or  ledger smb as a
newbie so  I decided whats easy to install was best for me.
********* The only issue I had was It would not work with my hp 1020 laserjet printer driver installed
so I am still working around that problem however it is a good free program.

ubuntu desktop , wine 0.9.30 and turbocash  seems to be a great package once all
the components  are downloaded and installed.

P.S. I still believe Quasar ,sql ledger or  ledger smb are good apps just too much
for a simple person like me to install.

Turbocash is a Windows app. so do not be surprised, no mention of Linux on there site.
