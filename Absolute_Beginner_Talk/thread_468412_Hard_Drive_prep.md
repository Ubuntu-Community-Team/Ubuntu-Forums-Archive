### Post by drowner on 2007-06-08
Ubuntu by default uses the ext3 file system.

It can also work with ext2 (ext3 has superceded it to a large degree) or the Reiser file system.

I would just use the ext3, but don't worry, you can make it when you install the linux . You don't need to 'prep' the Hard Drive, depending on how you wish to install..

Do you want to leave a windows partition, or just totally install linux?
