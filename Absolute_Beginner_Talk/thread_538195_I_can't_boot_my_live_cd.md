### Post by st33med on 2007-08-29
Did you go into the BIOS and modify what boots first? To do this, you need to see what Key is required for you to go into 'Setup' when you turn on your PC. Search through the Menus to see if you can find a Boot order. Set CD/DVD-ROM to the top. It should boot after you save and exit BIOS setup.

If it doesn't, there might be an option to choose which device to boot up to when you turn it on.
