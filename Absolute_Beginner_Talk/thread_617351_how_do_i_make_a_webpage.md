### Post by asmiller-ke6seh on 2007-11-19
I like Kompozer, too. It's as easy to use as OpenOffice.org's Writer; when used by someone with good taste in composition it can create very nice web pages.

One of the features I love ... you can open someone else's existing web page and then see how it is structured in Kompozer, so that you can understand how you can create various design elements in your own efforts.

I think that Kompozer is available in Synaptic. Kompozer seems to have replaced NVU.

You can read more about Kompozer at [http://kompozer.net/](http://kompozer.net/)
