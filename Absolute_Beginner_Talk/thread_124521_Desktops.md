### Post by el3ktro on 2006-02-01
Both Ubuntu (with Gnome) and Kubuntu (with KDE) have the same base system, the only difference is that the first uses Gnome, and the last one uses KDE. In the default install, both use OpenOffice, but other than that, they each have their own applications for doing things. But you can always install Gnome apps in KDE and KDE apps in Gnome, in fact when you have Ubuntu and install the "kubuntu-desktop" package, you'll have both Gnome and KDE (and vice versa)
