### Post by Daveth on 2007-08-25
place the cd in the cd drive, but ignore the file window that opens. In a terminal type

 ```
 wine  /media/cdrom0/setup.exe
```

assuming guildwars has a setup.exe on the disk!

which should get the setup file spinnning off to a good start. 

[COLOR="Red"]BUT [/COLOR]before you do this, go to the link below and install the Installshield patch, as there is a risk that the installshield may not behave (before I loaded it the path to patches could not be seen, so you could not install them!)

I do recommend:

[http://ubuntuforums.org/showthread.php?t=497332](http://ubuntuforums.org/showthread.php?t=497332)

which is full of all sort of useful WINE stuff.
