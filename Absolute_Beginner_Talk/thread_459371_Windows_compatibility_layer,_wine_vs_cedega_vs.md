### Post by chamberlain2006 on 2007-05-30
Well, Cedega is made for games.  Wine can (unstabley) run many programs, but usually not very well.  Another option is Crossover Office, which is for office programs.  Cedega and Crossover Office are generally much more reliable, but both cost money.  If you want a better quality program, then go with Cedega (or CX office for office programs).  If you're cheap and don't mind performance loss, then use wine.
