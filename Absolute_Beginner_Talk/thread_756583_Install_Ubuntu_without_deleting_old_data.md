### Post by Atomic Dog on 2008-04-16
Well if there is plenty of free space on your hard drive then when the partitioner starts on install leave the ntfs partition Vista is installed on there and use the ubuntu partitioner to resize it down to accommodate Ubuntu.

This tutorial shows how to install and how to use the partitioner to shrink your vista partition: [http://apcmag.com/how_to_dual_boot_windows_xp_and_linux_xp_installed_first.htm](http://apcmag.com/how_to_dual_boot_windows_xp_and_linux_xp_installed_first.htm)

So to answer your question: yes.  You can install ubuntu and have access to your vista files still.  It's not the most elegant solution seeing Vista is messed up and not working, but it will allow access to your files any time, and you can shrink it down pretty close to the smallest possible leave at lease a few extra gig and if your drive is really large leave 10 or so.

or buy an 8 gig flash drive for $35 at Fry's and put all your files on that and do a clean install :)
