### Post by %hMa@?b&lt;C on 2006-08-30
create a folder called .fonts in your /home/user directory and move your .ttf fonts in to the folder.

via command line

```
cd ~
```
```
mkdir .fonts
```
then you can copy your fonts into that folder and viola! it will work!

### Post by reacocard on 2006-08-30
to install tt fonts:

open nautilus.
hit CTRL+L to get an address bar.
type fonts:/// into the address bar and hit enter.
drag the ttf file into that folder.

NOTE: this only installs fonts for you, not system-wide.
