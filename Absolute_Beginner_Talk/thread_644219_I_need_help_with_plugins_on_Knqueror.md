Hold on and I'll post the same here for you. I will edit this post with the info.

**Edit**

Be sure that you have the "Ubuntu Restricted Extras" from the Add/Remove under "Applications". After that is done, go to: [http://www.adobe.com/shockwave/download/download.cgi?P1_Prod_Version=ShockwaveFlash&ogn=EN_US-gntray_dl_getflashplayer](http://www.adobe.com/shockwave/download/download.cgi?P1_Prod_Version=ShockwaveFlash&ogn=EN_US-gntray_dl_getflashplayer)

Then...

Install the .tar.gz file, extract it to your desktop and do this in terminal:


```
cd Desktop/install_flash_player_9_linux
```

That will get you to the file where you need to be. Then use this code:
```


./flashplayer-installer
```

And follow the prompt.

After this is over, you can get rid of the folder and the download, unless you want it for the future. (It doesn't have to stay on your desktop after installation.)
