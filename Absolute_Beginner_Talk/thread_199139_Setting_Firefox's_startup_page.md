### Post by catlett on 2006-06-18
This is the easiest way. Open firefox and go to the page you want to be your homepage. Then go to the top of the screen to "edit" this will drop down a menu. Select "preferences". This will open up a window with a menu. Make sure you are in the "general" tab. There is a list of tabs at the top of the window. (general,privacy,content,tabs,diownloads,advanced) Click on general if it is not highlighted.
In "general" the first option is "home page".  It will have 3 boxes you can click on. Choose the first box "use current page" (since you already navigated to the page you wanted as your homepage). Hit on "close" in the bottom right hand corner and you're done.

### Post by aysiu on 2006-06-18
Go to System > Preferences > Preferred Applications.

Under **Web Browser**, select **Custom**.

In the **Command** area, if it says ```
firefox %s
``` change it to ```
firefox
``` If it says ```
firefox
``` change it to ```
firefox %s
``` Or if you're launching from an icon, right-click the menu or icon and toggle removing or adding %u.
