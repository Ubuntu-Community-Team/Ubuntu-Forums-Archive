### Post by kkellner on 2006-10-16
gDesklets lets you have some good widgets in Ubuntu,

sudo apt-get install gdesklets

I believe many good "gdesklets" come with the installation including system monitors/calender etc.  but you can download more at [www.gdesklets.org]("http://www.gdesklets.org")

hope this is helpful!

### Post by kkellner on 2006-10-16
also, if you want gdesklets to launch when you start up gnome, go to system-preferences-administration-sessions, click on the startup tab, and add the startup command for gdesklets to the list, which is of course

gdesklets

that way you won't have to start them up every time you log in.
