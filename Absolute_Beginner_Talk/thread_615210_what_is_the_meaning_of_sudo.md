### Post by plinydogg on 2007-11-16
Using "sudo" before a command give you "super user" privileges, which are necessary to make certain changes, open certain programs, etc. This is designed to make sure that those without "super user" privileges cannot mess up your system and/or compromise your security.

"cp" is the copy command.
