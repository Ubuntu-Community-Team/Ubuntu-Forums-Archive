### Post by khughitt on 2007-10-28
One easy way to find a program is to open a run dialog (alt+f2), or terminal, and start typing the name of the program "media", then his tab a couple times and you will see a list of all the installed software with a name starting with "media." The tab-completion will only work in the terminal, but if you start typing it in the run dialog, you should also be able to hit the down arrow I believe to see a list.

Goodluck!
