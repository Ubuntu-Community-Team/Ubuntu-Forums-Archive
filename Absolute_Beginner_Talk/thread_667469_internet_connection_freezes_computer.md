### Post by bwallum on 2008-01-14
I have experienced the same from time to time. I believe it to be some form of malicious net bot. When it happens (about 3 to 4 times a year) I turn off all my machines, then reboot my router, then when the router is up, turn on my machines again. It's always worked so far. I think IPv6 is less prone so if you can make sure your router is compatible with IPv6 then, trickier, get an ISP that uses it.
