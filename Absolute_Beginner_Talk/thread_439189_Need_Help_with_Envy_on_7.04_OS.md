### Post by [S|G] on 2007-05-10
I suppose you can't boot into your graphical interface, right?
If that's what is happening, then boot normally and login on the text terminal. Once you finish entering your password and gets to a command prompt, type this:
```
sudo dpkg-reconfigure xserver-xorg
```
Then try to reboot again and see if your  GUI is back :)
