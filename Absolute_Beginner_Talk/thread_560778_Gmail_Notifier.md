### Post by R.Bucky on 2007-09-26
The path to firefox is simply ```
firefox %u
``` or I believe that you can simply type firefox. Also, if you are using Gnome, right-click 'Applications' in the top-left and 'edit menus.' Find 'Internet' and firefox then right-click again to see the properties (i.e. application path).
