### Post by mmarshall on 2007-10-11
Maybe I've been lucky, but every single wifi card I have ever purchased has happened to have an Atheros chipset.  These things have been working out of the box in linux distros for years.  (using a binary blob, but that should change soon.)

On this page you can get a list of all PCI products using an Atheros chipset:
[http://customerproducts.atheros.com/customerproducts/ResultsPageBasic.asp](http://customerproducts.atheros.com/customerproducts/ResultsPageBasic.asp)
