### Post by notwen on 2007-04-26
You could try conky, it's more embedded in your desktop rather than sitting on top.  [Link]("http://conky.sourceforge.net/")  It is customizable to the extreme and can be very complex or really simple, it's totally up to you.  You can configure conky to your liking using a simple text editor.  [Here]("http://ubuntuforums.org/showthread.php?t=353362") is a link to a thread compiling conky from source, skip the first part fi you prefer my very basic install below, but some of the posts on this thread could be very informative.  Open up terminal and type:

```
sudo apt-get install conky
```

then type the following in terminal to get a peek at the default conky configuration.

```
conky
```

Hope it helps. =]
