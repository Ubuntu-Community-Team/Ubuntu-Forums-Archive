### Post by cocteau on 2006-09-01
I'm going to strike a note for Pure Data, real time music programming. 

If you need inspiration on what software to try go look up Planet CCRMA which is a Fedora repository based at Stanford. It has a ton of great rpms (which you can't use) and lots of useful information about the different software and various packages for them.

Also if you are looking for a pure audio distro keep an eye out for 64studio, which is also debian based.
