### Post by Happy_Man on 2007-09-09
If you are talking about the Windows drivers, you can't install Windows drivers in any Linux distro. Radically different OS's and all. 

What you can do, is plug the printer in, and use System Settings's Add New Printer function to quickly and easily select the printer from a list. It should take of drivers all by itself.
