### Post by HereInOz on 2007-09-30
Beryl is now, due to an amalgamation of Beryl and Compiz, Compiz fusion.  It provides desktop effects like shadows, window animation and stuff like that.  If you are running a server, with no desktop at all, not only will you not be able to use Compiz Fusion, but you will have no need for it in any case.

To run it you need a desktop installation such as Gnome, and a reasonably good video card - Nvidia GeForce 5300 or better is about the mark.
