### Post by Brunellus on 2007-08-04
> **sad_iq said:**
> Ok...I think you are below the minimum memory requirements to use the LiveCd..to install you would need to download the Alternate Cd. It is a text based install disk, fairly easy to use!!!
Agreed. 

The Live CD needs at least 256 MB of RAM to run properly. 

If you need a Live CD to try out Linux, I would choose [Slax](http://www.slax.org/)--quick, lightweight, responsive.  

If you would like to install Ubuntu, I suggest using the Alternate CD, which has the text-mode installer.  The text-mode installer will definitely work, but you will find that "full" Ubuntu/Kubuntu will be quite sluggish, given that you have 128 MB of RAM.  For an installed system, I recommend Xubuntu.
