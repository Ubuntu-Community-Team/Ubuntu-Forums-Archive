### Post by tszanon on 2007-09-22
> **LifeIsGood said:**
> First time poster and a complete novice with ubuntu..how do I install Pidgin?
You have 3 options to install Pidgin:
**1.** download the DEB file from [www.getdeb.net](www.getdeb.net), then double-clicking it;

**2.** download the source code from [http://pidgin.im](http://pidgin.im), extract the files to a folder, and, in the terminal:```

cd /path/to/the/folder/where/you/extracted/the/files
sudo ./configure
sudo make
sudo make install
```
Make sure you have the package build-essentials before doing this.

**3.** Wait until Ubuntu Gutsy is released next month. Pidgin is already part of the default Gutsy install.
