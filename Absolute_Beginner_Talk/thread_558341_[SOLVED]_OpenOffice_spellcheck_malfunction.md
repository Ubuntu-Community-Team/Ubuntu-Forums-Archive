### Post by ryanVickers on 2007-09-23
I was going to ask a question on how to fix this, but since after a lot of hard work I've fixed it, I want to post a simple solution.  I have had this problem with Open Office time and time again, and now I have a simple solution for fixing it!  When the spell checker doesn't work, go **File > Wizards > Install New Dictionaries... **(at the bottom) and it will guide you through the relatively simple fix.  Then restart open office and the quick starter tray applet (if open) and it should work now!!!

### Post by Hagar Delest on 2008-02-05
First, for spell check issues, see here: [[Tutorial] Spell check and Language configuration](http://user.services.openoffice.org/en/forum/viewtopic.php?f=7&t=67).

But you've other problems it seems.
Go to Synaptic and install the openoffice.org-style-... packages to get the icons back in your toolbar.
For the tiny wizard window, that's more serious. I've seen that but don't remember any fix.

Good luck.

Note that you can install the files manually if you still don't manage to run the wizard. You need to download the language files, unzip them and edit the dictionary.lst file.
