### Post by Kimm on 2006-06-08
gardara, same here.

Seems to work if I remove tcl/tk and aMsn from the system. And then install it from Autopackage

[http://prdownloads.sourceforge.net/amsn/amsn-0.95-2.x86.package?download](http://prdownloads.sourceforge.net/amsn/amsn-0.95-2.x86.package?download)

To install using Autopackage, run the package file as an executable from the terminal:

#sudo sh amsn-0.95-2.x86.package

that will download and install the package manager. Then in the future you just have to double click the package file to install it :)
