### Post by steve.horsley on 2006-01-03
I don't know what ET is, but I would worry about something that you "play" that has to be played with root provilege. It is of course normal that you keep using sudo while you're setting the syetem up, doing sysadmin-type work. 

It is not normal to keep doing that once the system is set up to your satisfaction and you are just doing day-to-day work as a normal user though. Try to keep a clear distinction between your user and sysadmin roles.
