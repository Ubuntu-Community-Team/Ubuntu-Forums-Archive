### Post by ajgreeny on 2007-09-11
And to confuse things you can have metacity, the default window manager for the gnome desktop environment running in kde instead of the default window manager, kwin.  All or at least most of these things are interchangeable.

There are many desktop environments, eg KDE, gnome, xfce, fluxbox, etc etc, and there are many window managers.  Most people simply use the window manager that comes with the desktop environment.  It's all part of the great big choice that linux gives you.
