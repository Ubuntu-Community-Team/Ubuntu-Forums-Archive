### Post by Cypher on 2007-04-23
The LiveCD usually contains many different drivers to allow for it work on all sorts of PCs. That being the case you might notice these kinds of error messages that can be ignored as long as Ubuntu fully boots up. Once you do the install, these errors should not show up because the drivers/modules/functionality that is loaded is more appropriate for your PC.

As an example, I have a PC with SATA drives and an IDE CD-ROM drive, during the LiveCD boot I kept getting errors relating to the CD-ROM drive, but once I install Ubuntu, it's all good..
