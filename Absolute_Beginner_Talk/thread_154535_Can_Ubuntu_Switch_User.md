### Post by mostwanted on 2006-04-03
Not sure if Breezy can do it but there's an applet in Dapper for doing just that.

Edit: That's the one: [http://www.gnome.org/start/2.14/notes/en/rnusers.html#rnusers-fusa](http://www.gnome.org/start/2.14/notes/en/rnusers.html#rnusers-fusa)

Dapper ship with Gnome 2.14, Breezy ships with Gnome 2.12. When Dapper is officially out, it should be a breeze (no pun intended) to update to the new version of Ubuntu from your Breezy installation.
