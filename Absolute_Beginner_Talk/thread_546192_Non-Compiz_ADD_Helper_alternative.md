### Post by neko18 on 2008-05-04
I would also like this. A partial workaround to clear your desktop is to run `nautilus -q` to quit Nautilus and remove all icons from the desktop (my main distraction). If you still need the file management capabilities but like having a clear desktop, you can run `nautilus --no-desktop` to open a file browser window but not have the desktop icons. If you would like to get the full desktop back, simply run `nautilus`.
