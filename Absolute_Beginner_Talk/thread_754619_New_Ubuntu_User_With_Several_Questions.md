_Short Answer:_ No.

_Long Answer:_ To answer that you need a reasonably basic understanding of how the internet works. There is no such thing as a "stream" of content that can be sent over the internet. Everything on the internet is transmitted in discrete packets, sometimes a connection and pathway is setup so as to simulate "streaming" however discrete packets are still all that's being sent. Thus "streamed" content is no different to any other content from a security perspective. The part of the internet that presents a security risk has very little to do with the content that is being transmitted and much more to do with how this content gets from the internet to your machine (which ports your machine listens on, presence of a firewall on either your machine or your router, etc). Due to the use of permissions in linux/unix, it is inherently more secure against an attempt to take over your machine than an OS where every operator has full administrative privileges such as Windows XP. There's also the fact that the user base for Linux is minuscule when compared to Windows, so there are incredibly few viruses or malware that are written for Linux.

> We also burn CD's a lot, and is there an easy to understand CD burning program on Ubuntu.
Brasero is incredibly easy to use and is the default in Hardy.

> I found the Mac OS X (Tiger 10.4) to be confusing, as just how to burn a CD.

Might Ubuntu be easier?

As with any new thing using a new operating system is going to involve a rather steep learning curve. You need to be aware that Ubuntu/Linux is not a free Windows clone, does not set out to be one and will never aim to be. That being said, after using Windows for many years I found GNOME (the desktop environment for Ubuntu) to provide a much more familiar interface than OS X did.
