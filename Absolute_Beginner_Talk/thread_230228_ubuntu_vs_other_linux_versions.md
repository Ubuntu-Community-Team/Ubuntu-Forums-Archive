### Post by Christmas on 2006-08-08
[quote=lwf]Sorry for trying to hijack your thread, but I would like to know what is actually making Ubuntu better than Debian? I mean, what is the difference (other than maybe that the installation is a bit harder to do)?[/quote]
I am no expert but I'll tell how was my experience with Debian. I installed the stable version, 3.1 Sarge. The installer is not graphical, it's only like Ubuntu's Alternate CD. It installs only the "base system" which is the equivalent with Ubuntu's Server Installation (correct me if I'm wrong). This means all you have after the installation of Debian is CLI. You can proceed and choose the Window Manager, as far as I could see if you choose it from the installation's menu it will install both KDE and GNOME (I did the installation from a 2CD Set). If you want only one, after the installation of the base system you have to type in:
For KDE:
```
su
apt-get install x-window-system-core kde-core kdm
```
For GNOME:
```
su
apt-get install x-window-system-core gnome-core gdm
```
Also, instead of kde-core and gnome-core you can choose the full installation of kde and gnome, just cut the "-core" part.

Debian is so famous for its stability. Well I might be stupid, but it made me the impression of a second Windows ME. The packages are old, the system itself is stable, but the applications crash like nuts. Compare that to Ubuntu's apps, which crash very rarely. This mentality Debian has over "stability" is wrong in my opinion. I am not of the opinion that newer versions of applications are more stable in all the cases, but in most cases (and Ubuntu prooved it, and others) they are.

Also, from what I could read on the internet, Debian doesn't offer commercial support, the support can be obtained from third-party companies. No ShipIt CDs, no official forums, only the mailing lists. There's no man in the back of Debian who would actually pay developers to produce it. It is maintained by people worldwide. Still, their website can be perfectly be viewable with a text-only browser, they won't make it more graphically because of this. Every distro with its own philosophy. Debian has been around since 1993, Ubuntu since 2004. If you ask me, Ubuntu seems  more reliable and more stable. Ubuntu is also well-known for its hardware support. For my PC, the hardware was detected correctly by both Debian and Ubuntu, no differences for me.

In conclusion the major differences would be:

Debian - Old packages -- but it doesn't proove the "stability"
Ubuntu - New packages, always new artwork and eye-candy

Debian - No free CDs
Ubuntu - Free CDs through ShipIt

Debian - No official forums, only mailing lists
Ubuntu - Great community, forums, mailing lists

Debian - Old look for the website (if someone cares about)
Ubuntu - Opened to new technologies

Tough we must not forget that Ubuntu is based on Debian, so APT, the packaging system, was made by Debian's developers. Also, the packages included in Ubuntu are taken (as far as I know) from the Debian testing repositories and they are worked out more by Ubuntu's developers.
