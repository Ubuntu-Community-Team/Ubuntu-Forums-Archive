### Post by compmodder26 on 2007-04-16
Did you burn this CD from an iso?  The reason I ask is that the install icon is performing an action that it isn't supposed to do.  It is supposed to begin the installation process.  If you burned it from an iso, I would suggest burning another copy, only this time at a slower burn speed.

As for the slowness, you are using the bare minimum amount of RAM to operate the live CD.  Once you install, your system should be a lot faster.  I would also recommend downloading the Alternate Install CD.  This is a text based installation CD.  Since you've tried the Live CD and like what you see, you can safely install from the Alternate Install CD.
