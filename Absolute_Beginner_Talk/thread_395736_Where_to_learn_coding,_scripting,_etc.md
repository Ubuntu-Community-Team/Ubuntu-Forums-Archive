### Post by annda on 2007-03-28
i'm a python beginner myself, but you can run it from the terminal in two ways:

to issue single commands, type
python [hit return]

or you can write executable scripts, just like in bash, but start them with
#!/bin/python

instead of 
#!/bin/bash

you can start here:
[http://www.python.org/doc/](http://www.python.org/doc/)

and see if you like it.

### Post by x1a4 on 2007-03-28
> **HgBoy said:**
> I wasn't trying to limit myself to bash, but can the other languages be run from the terminal? I have little to no coding experience.

Zsh and TCsh are excellent shells.  I use tcsh as my default shell and am loving it.  

Good books are: 

**Ubuntu**

"Moving to Ubuntu Linux" by Marcel Gagné

"Ubuntu Hacks: Tips & Tools for Exploring, Using, and Tuning Linux" by Jonathan Oxer, Kyle Rankin, Bill Childers

"Ubuntu Unleashed" by Andrew Hudson, Paul Hudson

**Shells**

"From Bash to Z Shell: Conquering the Command Line" by Oliver Kiddle, Jerry Peek, Peter Stephenson 

"Using CSH & TCSH" by Paul Dubois

**Tools**

"The GNU AWK User's Guide" by Arnold D. Robbins

"SED & AWK" by D. Dougherty

**General Linux**

"Linux in a Nutshell" by Ellen Et Al Siever
