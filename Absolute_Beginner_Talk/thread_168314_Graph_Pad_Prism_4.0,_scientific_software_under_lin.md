### Post by Sef on 2006-04-30
For graphpad 4.0, check out grace, which has gotten good feedback on its 2-D plotting.

[http://pdb.finkproject.org/pdb/package.php/grace]("http://pdb.finkproject.org/pdb/package.php/grace")

For some feedback on it, read the comments in slashdot.

[http://apple.slashdot.org/article.pl?sid=03/03/03/1435211&mode=nested&tid=134]("http://apple.slashdot.org/article.pl?sid=03/03/03/1435211&mode=nested&tid=134")

For SPSS, check out these two links.

[http://freshmeat.net/projects/spssutils/]("http://freshmeat.net/projects/spssutils/")

[http://www.gnu.org/software/pspp/]("http://www.gnu.org/software/pspp/")
