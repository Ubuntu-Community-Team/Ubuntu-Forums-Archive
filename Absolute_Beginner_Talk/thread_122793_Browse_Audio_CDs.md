### Post by Madpilot on 2006-01-29
Soundjuicer (in Breezy, at least) will also play CDs, as well as extracting them.

(not both at the same time, mind you... which would be cool, and AFAIK theoretically possible...)

The only CDs I can see needing to browse in Nautilus would be the ones with "extras" packaged, like video files or whatever. With those CDs you should get a popup window in Breezy asking if you want to use it first as an audio CD (in SJ) or as a data CD (in Nautilus).
