### Post by tgalati4 on 2007-10-10
Check for video memory in BIOS.  Try increasing it.  Sometimes video cards won't go to higher refresh rate when video memory is low.  If video memory is shared with RAM (as often the case with SIS chipsets) then increasing video memory in BIOS can fool the card into higher performance mode.

Also, what are the specs on the monitor?  Can it be driven at higher frequencies?

Most modern monitors can run 85 Hz to 120 Hz (very smooth) at 1024 by 768.
