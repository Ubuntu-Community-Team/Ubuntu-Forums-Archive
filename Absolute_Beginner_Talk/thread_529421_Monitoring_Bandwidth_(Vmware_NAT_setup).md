### Post by HereInOz on 2007-08-19
If you install Gnome Desklets, there is a facility in all of that which will allow you to display, on your desktop, the actual bandwith through the virtual network adaptor in the VMWare Ubuntu.

Install Gnome Desklets using Synaptic, and then go to the System/Net section and install the  FTB-Net-guage.  It might give you what you need.

Many of the Desklets are pretty ordinary at best, but this one is quite good, and works well.
