### Post by Ansible on 2007-07-22
Here's an easy noob question for you, 

Can you run KDE apps in regular (gnome) ubuntu, and do they integrate properly, etc?  I keep seeing apps that are 'for KDE', does this mean that they will not run properly in regular ubuntu?  Are there pitfalls I should know about when using apps for the 'wrong' environment?  I don't really want to run two window managers or whatever.

---

### Post by aysiu on 2007-07-22
Yes, you can run them.
No, they won't integrate properly.

**Pitfalls**:
Drag and drop may move instead of copying for some applications.
Loading applications in non-native environments uses a bit more RAM.

### Post by stmiller on 2007-07-22
Well they do integrate to some point. You can run a KDE app and if it has a task bar applet, the applet WILL work in Gnome's task bar.

But yeah you can run KDE apps in Gnome or vice versa and it doesn't make any difference.

### Post by JesterDev on 2007-07-22
Just to throw this into the loop. Kmail is about the only KDE app that I have found that does not show up in the applications menu. You'll have to add it in manualy, or just link to it. :) 

Just letting you know, just in case - I'm sure there are others that wont show up also.

You'll find it in:  /usr/bin/ along with others in case you need to link to them.
