### Post by srt5 on 2007-05-01
A quick search on Google lead me to [this website]("http://imsuden2.blogspot.com/2005/06/installing-firefox-theme.html"), which suggests that you take the JAR file and drag and drop it onto the Firefox 'Theme Manager' window. I haven't tried this myself, but that seems like a reasonable way of doing it.

Hope it helps :)

### Post by DJ_Peng on 2007-05-01
FWIW, that's the easiest way to install themes or extensions that you don't get from AMO. When I was developing my Fx theme that's how I'd install newer versions to test them.

This also works if you download an addon and then manually tweak it, such as to make it work in newer version of Fx/Tb.
