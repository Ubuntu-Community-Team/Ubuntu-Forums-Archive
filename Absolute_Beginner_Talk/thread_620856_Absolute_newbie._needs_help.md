### Post by kleo skywalker on 2007-11-23
I haven't been using Ubuntu too long, and have never used automatix because I keep reading everywhere that it breaks things. So why take that risk when there's Synaptic.
To play restricted formats, step one is to go to Synaptic, search for a package called ubuntu-restricted-extras and install it. This installs a whole bunch of stuff that's needed.
If you know what codec you want, you can search for it in Synaptic. Or, you can simply try to play the DVD, and Totem will prompt you about whatever is missing.
If you can't find something in synaptic, go to System-->Administration-->Software Sources and make sure you have the Main, Universe, Restricted and Multiverse repositories selected.
