### Post by drummingpariah on 2007-05-31
The system utility and nvidia raid are both just fancy ways to say configuration app.  You'll have these in Linux as well, they'll just look different.  Understand that Linux is NOT Windows, and will not have all of the same "features" that Windows has.  The benefit of Linux is the expandability and number of built-in utilities.  Forget about that CD for Linux though, it has no place in this world.  If you do decide to go back to Windows (I tried it last week, painful experience) you won't need that CD because everything can be downloaded automatically.  Linux takes care of all that for you, but with the Linux files, not the Windows ones.
