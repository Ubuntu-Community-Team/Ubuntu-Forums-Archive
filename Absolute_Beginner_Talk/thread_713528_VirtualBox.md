### Post by yaye on 2008-03-02
If you require a Windows based program that doesn't have a equivalent in Linux that you consider good enough and the program doesn't require 3D acceleration,  VirtualBox is a good choice.  It was very easy to install and configure.

I've needed to use it only once,  to view a webcast  that required Internet Explorer and Windows Media Player.  An added benefit was booting to the Windows XP login screen in under 20 seconds.   I tried to use it to run a web based game that my son uses, but it requires Direct3D which VirtualBox doesn't support.
