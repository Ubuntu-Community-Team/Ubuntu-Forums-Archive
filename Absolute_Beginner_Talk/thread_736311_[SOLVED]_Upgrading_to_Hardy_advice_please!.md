### Post by ahsile on 2008-03-26
The real advantage is that you will get newer versions of some software. For the most part software packages are frozen in a release with only the distribution of security updates. You can get new software using the backports repository, but I've never been a fan of that option.

If you want to upgrade though, wait until the official release comes out. Hardy is still in development so you probably won't be impressed!

(As anecdotal evidence... I upgraded last night and a) FF3 wasn't compatible with most of my add-ons so I had to downgrade, and b) I ran into an issue with python/bittornado version incompatibilites)
