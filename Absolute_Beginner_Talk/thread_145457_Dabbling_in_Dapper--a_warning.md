### Post by Brunellus on 2006-03-16
Just in case anyone reading this forum is thinking of upgrading to dapper (the next version) using an apt-get dist-upgrade....

Don't do it yet.    apparently, the dist-upgrade process is not terribly well-tested, and it has not proceeded for me with typical ubuntu smoothness.  

It should be great by June, but it's certainly not a place for new users.  

Note that I'm talking about upgrading from breezy to dapper here, not a clean install of dapper (which is not yet recommended for "production," i.e. "I really need this to work...NOW" use, either).  

Just a warning.

### Post by mstlyevil on 2006-03-16
I second that warning and want to add that Dapper is in the Alpha state of development. You should continue to use Breezy as your main operating system until Dapper is released as a stable operating system. If you would like to test Dapper, then may I suggest you download the ISO and set up a dual boot system. Just a warning if you decide to test Dapper, there is no Automatix for Dapper and there are still bugs that have not been fixed yet. Only do this if you do not mind configuring and troubleshooting Dapper manually.
