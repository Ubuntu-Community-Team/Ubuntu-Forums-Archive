### Post by mr.v. on 2007-02-10
The difference between Edgy and Dapper isn't beta vs non-beta; it's long-term support -vs- continually upgrading to maintain support every 6 months to 1 year.

If you're the type of guy who likes reinstalling the newest version of your OS on a yearly basis...choose edgy because it's got lots of neat features. You will have to upgrade when the next release is out if you want to keep getting security updates.

If you like a stable OS for a few years without reinstalling a newer version all the time, definitely stick with Dapper. If you, say, are still running Windows 2000 or Mac OS9, we Dappers are here for you =)

If you do upgrade firefox to 2.0 might I suggest not following the integration guide. It's a matter of time before there's a major conflict =).

Just install it as a standalone in like usr/local/firefox, update the globe at the top of the screen to point to it, and setup a firefox2 profile under the profile manager and load that with "firefox -P firefox2" so it doesn't fight with the 1.5's "default" profile and you're set =)
