### Post by 23meg on 2005-11-01
1. If you set them up to be, yes.

2. None that I know of.

3. Yes, if you run them on startup, but you may not feel the difference on a fast computer.

4. Why exactly would it be a backdoor application? AFAIK it doesn't actively listen on any ports. I don't think there are any serious security risks with Gdesklets.
