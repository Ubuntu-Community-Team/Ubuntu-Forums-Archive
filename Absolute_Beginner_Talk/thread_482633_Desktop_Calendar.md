### Post by arvevans on 2007-06-23
For that "gdeskcal" that you installed, you can go to a terminal screen (the feared Command Line Interface!) and type "gdeskcal" and it will pop up on your desktop.  If you want to be able to pull it up at will, you may have to build a launcher or menu entry for it.

### Post by BobCFC on 2007-06-23
To make an Icon on your desktop right-click choose Create Launcher and put gdeskcal in the Command box, you will also have to give it a name.  You can also pick an picture for the Icon.

To put this in the pannel at the top.. right click on a pannel, choose add to pannel.   There is a custom application button for anything which is missing from the menus.
