### Post by m.musashi on 2006-02-01
[QUOTE=aysiu][https://wiki.ubuntu.com/FirefoxNewVersion](https://wiki.ubuntu.com/FirefoxNewVersion)[/QUOTE]

This explanation looks good and similar to what I did. I had a friend walk me through it though and I don't recall it being too difficult.

It should be noted, however, that you don't need to install v1.5 as Ubuntu comes with Firefox preloaded and although a slightly older version it works fine. Although I installed 1.5 I haven't really been using it.

It might also be worth mentioning that apps don't install in Linux like they do in Windows - at least not in my limited experience. You cannot simply download an .exe type file and double click to install. It is a bit more involved than that. On the flip side, however, using the add application tool or synaptic is really easy - provided you can find what you are looking for.
