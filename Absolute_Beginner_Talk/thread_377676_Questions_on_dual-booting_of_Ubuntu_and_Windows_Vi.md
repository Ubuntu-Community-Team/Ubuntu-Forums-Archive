### Post by gitano1 on 2007-03-06
It has been my experience that Windows does no like sharing a hard drive with another OS. A much easier solution is simply to add another hard drive to your system and use it as the residence for Ubuntu. Grub will install the necessary software for the dual boot onto the C drive, but the second OS will reside on its own drive. I did that will my WinXP machine, and it works perfectly. I use a 20 GB hard drive for Ubuntu, an 80Gb for XP.
