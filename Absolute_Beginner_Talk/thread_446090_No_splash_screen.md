### Post by Kobalt on 2007-05-16
You should edit the /boot/grub/menu.lst file in order to adapt the splash screen resolution to your monitor. See this (the last part, from point 6) : [https://help.ubuntu.com/community/USplashCustomizationHowto](https://help.ubuntu.com/community/USplashCustomizationHowto)
