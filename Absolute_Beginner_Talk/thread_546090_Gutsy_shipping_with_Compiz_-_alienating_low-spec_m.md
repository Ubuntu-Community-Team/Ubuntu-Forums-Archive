### Post by AusIV4 on 2007-09-08
I'm pretty sure that if you attempt to run Compiz on a graphics card that can't support it, it just fails quietly and reverts to another window manager.

I have rather mixed opinions about distributing a composite window manager with Ubuntu. On one hand, I think the default Ubuntu install ought to just include the basics that people need to run a functional desktop. On the other hand, other major operating systems are going the flashy route, and it's something users are coming to expect - even if it's easy to install, some people will consider it a demerit that it isn't a "part of the operating system."
