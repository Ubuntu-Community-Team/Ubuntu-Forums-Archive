### Post by Daveth on 2007-07-08
and if you have not got a separate  /home (and you probably have not as it is not a default installation), then following the following will get you there:
[http://www.psychocats.net/ubuntu/separatehome](http://www.psychocats.net/ubuntu/separatehome)

I did exactly this with dapper, checked the data was safe with the dapper live cd, then over-wrote with the new feisty one. This is much, much easier than any other upgrade path - elsewhere in the forum is made the point that the internet upgrade path is very, very long and you cannot pop off to bed, as it reqiures questions answering. If you have a flaky internet connection as well..............doesn't bear considering.:)
