### Post by coolen on 2007-09-21
It depends what you want from a desktop.
To me, Gnome looks and feels nicer.
KDE has customisation options up to wazzoo.

KDE, generally speaking, has applications more suited for the power user (just look at Amarok), while Gnome's are simpler, although less configurable.
Even with gconf-editor, which gives you a tonne of options, some Gnome apps can't be configured the way their KDE equivalents can.

I use Gnome, mostly. I have KDE installed, and use it from time to time, but I keep coming back to Gnome. It seems comfortable to me.

### Post by PsycoGeek on 2007-09-21
Personally I like Gnome.  If you want to try out each interface (Gnome, KDE, Xfce), go the the respective links on the Ubuntu home page (Kubuntu and Xubuntu), download each of the distro's, burn them to CD, and try them in LIVE CD mode (no install, they will run from CD just like Ubuntu will), rather than install them.  All you have to do is set your PC to boot from the CD Rom in the bios, or it you have the option like I do on my laptop, set it to allow access to a boot menu and choose the CD Rom.

Good luck, and happy (interface) hunting!
