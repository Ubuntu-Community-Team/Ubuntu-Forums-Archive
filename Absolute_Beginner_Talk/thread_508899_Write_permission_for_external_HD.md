### Post by kngunn on 2007-07-24
You are using a drive formatted with FAT32.  FAT32 doesn't recognize file ownerships, so you can't chown them.

---

### Post by Rocket2DMn on 2007-07-24
You can, however, chown the mount folder, but I think the drive needs to be unmounted.  I believe you're supposed to set root as the owner of the mount point.  I'm not sure about group ownership, but I think it should just be the default group that root is part of.
Then you can ```
sudo mount -a
```
