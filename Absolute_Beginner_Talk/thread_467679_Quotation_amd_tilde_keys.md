### Post by freebird54 on 2007-06-08
I think what you have is the 'sticky keys' option set in your keyboard config.  You will find, if this is so, that you could press e then ' and get a french e with accent aigu.

Try a different keyboard layout in System-Preferences->Keyboard (US English ?) and see if that will do it for you.  If not, there are other ways to get there....

### Post by freebird54 on 2007-06-08
Yup - that's how it is meant to be.  It adds you second keystroke to the first.  If it is a space, you see only the original 'dead' key - if you hit a vowel, it combines them.  Handy if you have to write international correspondence with French or Spanish or German in it... :)  Glad you have the choice figured out now...
