### Post by Karl S. on 2007-05-19
I'm using a spare desktop (no monitor, no keyboard, just hooked up to the internet) running feisty for my music right now. I use xvncviewer to access it from my laptop. Just remember to set up remote desktop after you install (system, preferences: under 'security' I just use a password protect) and share whatever folders you want under system administration. Pretty simple.
