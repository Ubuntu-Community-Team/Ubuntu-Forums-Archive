### Post by ph1 on 2008-02-12
I'm not exactly sure what's going on in your thumbnail, but here are a few suggestions based on my experience.

If you're using Emerald, I find that sometimes, e.g. when a google talk window pops up while I am logged into gmail, Emerald "crashes" or flickers and disappears.  Closing the window and/or restarting the program fixes it.  (You can still pres 'alt' and click and drag the window around without the titlebar.)  I don't know if there's a fix that can be applied to prevent it in general, but if I find one I'll post.

If you want to adjust transparency in general, you can go  into the Emerald Theme Manager, click on the "edit themes" tab, then the frame engine, and then use the sliders to adjust opacity.

If you're not using Emerald, let me know--post more details about what exactly happens and what you're doing when it happens.
