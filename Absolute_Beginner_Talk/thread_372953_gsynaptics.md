### Post by Maestro23 on 2007-02-28
gsynaptics is in the repositories. You might need to enable extra repositories first.

Command line: [http://www.psychocats.net/ubuntu/sources](http://www.psychocats.net/ubuntu/sources)

GUI: [https://help.ubuntu.com/community/Repositories/Ubuntu](https://help.ubuntu.com/community/Repositories/Ubuntu)

Then:

> sudo aptitude update

sudo aptitude install gsynaptics


Good luck.
