Try clearing the keyboard shortcuts in gnome (I assume you are talking about System > Preferences > Keyboard Shortcuts and NOT Amarok > Settings > Configure [Global] Shortcuts), and then setting them *once*. I think when you set the same key to the same action more than once, it somehow maps to the action itself (XF86AudioPlay, for instance) instead of the keypress (0xb0). I don't know what script you used in amarok, so it's kind of hard to help you there, but why use a script when you could've set the keys in amarok yourself, using the Settings > Configure Shortcuts?
