### Post by starcannon on 2008-04-13
If you have the restricted drivers installed, then:
* CTRL-ALT-F1
* sudo /etc/init.d/gdm stop
*sudo nvidia-xconfig
*sudo /etc/init.d/gdm start
Alt-F2 
gksudo nvidia-settings
set things they way you like
save to xorg.conf option
exit
CTRL-Alt-Backspace or reboot either way will work fine.

If you don't have them installed then:'
* System-->Administration-->Restricted Drivers Manager
* Put a Check mark on appropriate devices, and follow on screen directions.
* After reboot if things are not as pretty as they should be, then go up and do the stuff at the top of this post.

GL and hope that helps
