### Post by RomeReactor on 2008-01-09
Hi. Give Rhythmbox (the default music player) a try; you can also check out [Banshee]("http://banshee-project.org/Main_Page"), [Listen]("http://www.listen-project.org/"), [Exaile]("http://www.exaile.org/"), or the already mentioned [Amarok]("http://amarok.kde.org/"). These you can find in the repositories, so look for them in Synaptic (System->Administration->Syantpic) or Add/Remove (Applications->Add/Remove).

For torrents, Ubuntu already has a BitTorrent client installed by default; just download and double click a .torrent file and it will come up asking where to save the file, or if you wish to resume a previous download. If you want something with more features, try [Deluge]("http://deluge-torrent.org/") or [Ktorrent]("http://ktorrent.org/"). You can find Ktorrent in the repositories, and you can get Deluge [from here]("http://download.deluge-torrent.org/index.php?dir=ubuntu/gutsy/0.5.8/&file=deluge-torrent_0.5.8-1_i386.gutsy.deb") (if you're running Ubuntu 64-bit, then [get this one]("http://download.deluge-torrent.org/index.php?dir=ubuntu/gutsy/0.5.8/&file=deluge-torrent_0.5.8-1_amd64.gutsy.deb")); once downloaded, double click on the package to install it.

Note that although Amarok and Ktorrent are KDE applications (which means they might look a little different than the rest of your applications) they''re exceptional programs and will work very well.
