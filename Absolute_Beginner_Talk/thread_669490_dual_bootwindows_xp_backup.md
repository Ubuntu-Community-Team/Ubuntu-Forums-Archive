### Post by EnergySamus on 2008-01-16
Hello!
So you don't have Norton Ghost... Well, most PCs come with a partition that has backup software. For example: My Acer has a backup utility in the BIOS that I can restore my original factory settings. 

Of course you should backup everything that is important on that external USB drive: documents, pictures etc. Plus, you have that systemrescue CD... use that if anything fails!

Also, if you want to get rid of the "temporary" OS after you are done, all you have to do is delete the partition(s) that it is on. If you have Vista, just do into the disk management utility and delete the partition(s). If you don't have Vista, Use GParted. Boot into the Gparted CD and delete the partitons, it is very simple!

Good Luck
