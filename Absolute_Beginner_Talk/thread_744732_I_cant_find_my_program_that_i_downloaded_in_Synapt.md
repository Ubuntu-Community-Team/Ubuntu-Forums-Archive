### Post by mmasroorali on 2008-04-03
If you want to install the latest firefox, a better option will be to install it from synaptic. Just enter Synaptic (System -> Administration -> Synaptic package manager), search for firefox, select it and then Apply. 

Firefox is usually installed by default, you can use the upgrade manager (System -> Administration -> Update Manager) to upgrade it.
