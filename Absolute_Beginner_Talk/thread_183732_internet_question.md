### Post by RRS on 2006-05-28
> then how do i do it or is there some thing i can down load to get free internet for ubuntu.

AOL is only free for a "trial period", then they start billing you and are almost impossible to get rid of. I'd suggest checking other providers first, often whoever provides you're phone service has "free trial packages" also, often with good deals for upgrading to dsl later.

Additionally you don't need to install any software from the ISP, you only need a username, password, and the appropriate phone numbers. The "setup cd" provided for windows simply makes it easy to enter this info, AOL and others pay the computer makers to preinstall this software so it's easy for the new user to get on-line (and eventually buy their service because changing is inconvient)
