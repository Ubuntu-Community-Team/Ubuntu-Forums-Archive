### Post by thelatinist on 2007-12-09
If you want to resize your system partition you will need to use the [GParted Live CD]("http://gparted-livecd.tuxfamily.org/"), which because it doesn't mount any drives will allow you to resize and move all your partitions.  Of course you should always back up vital data before you play with partitions.
