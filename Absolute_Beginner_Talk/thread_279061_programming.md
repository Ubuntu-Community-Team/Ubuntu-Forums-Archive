### Post by Tomosaur on 2006-10-17
If you're brand new to programming, then writing a game as your first project is going to be pretty difficult. It's not impossible by any means, but I think the problems you will encounter may turn you off the idea. It's probably a better idea to start off writing some simple programs / utilities first, so you can learn the fundamentals of programming. Writing graphical stuff is an art if its own, and one you can't learn overnight.
