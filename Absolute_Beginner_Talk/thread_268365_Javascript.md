### Post by 3rdalbum on 2006-09-30
Java is something different to Javascript.

Javascript is a scripting language that runs in your web browser. Java is a programming language and environment. The two have no relation to eachother; you don't need Java installed to use Javascript.

If you're having trouble installing Java, try Automatix ([www.getautomatix.com)](www.getautomatix.com)). It does all that stuff for you.

### Post by lebski88 on 2006-09-30
As above states Javascript only has the name Java in it for political reasons (well piggybacking off Java's success anyway). Normally you would use Javascript embedded into a webpage to make the page more dynamic. Try a google search for "Javascript tutorial" for more information on this. You don't need to install anything other than a web-browser to get this to work. It will have a javascript interpreter built in.

If you want to use Javscript outside of a web-page then your looking for something called ECMA Script which is the basis of Javascript. A google search for ECMA Script Shell Linux would probably get you the right results. However if your looking for a scripting language to work outside of a browser something like Python, Ruby or Perl would probably suit you better.
