HI here is my info on the same card
```
-desktop:~$ fglrxinfo
display: :0.0  screen: 0
OpenGL vendor string: ATI Technologies Inc.
OpenGL renderer string: ATI RADEON 9600 Series

-desktop:~$ glxinfo | grep direct
Xlib:  extension "XFree86-DRI" missing on display ":1.0".
direct rendering: No (If you want to find out why, try setting LIBGL_DEBUG=verbose)
-desktop:~$ glxinfo | grep vendor
Xlib:  extension "XFree86-DRI" missing on display ":1.0".
server glx vendor string: SGI
client glx vendor string: ATI
OpenGL vendor string: ATI Technologies Inc.
```

I took the steps to install and working 3d.
I went to synaptic manager, search for fglrx and installed 
```
xorg-driver-fglrx
xorg-driver-fglrx-dev
```
Then I edited the xorg and made sure the driver was fglrx instead of ati or vesa. Then restarted the system. 
```
gksudo gedit /etc/X11/xorg.conf
```
***And*** add at the end of xorg

Section "Extensions"
Option "Composite" "1"
EndSection

Then installed 
```
sudo apt-get install xserver-xgl
```
Rebooted ( you could try and just restart x with the ctrl,alt backspace) 
And enabled the effects and all is good. ( it is running on a low memory - have 2 gigs in route :) )
Hope this helps and good luck!
