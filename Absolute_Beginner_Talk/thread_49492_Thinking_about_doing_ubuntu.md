### Post by maruchan on 2005-07-16
Mike, it might be best if you tried out some LiveCDs first to see if you actually enjoy the different desktop environment, and if you do, spend some time learning the tools and commands. This will make the migration much easier, as it won't all be falling in your lap at once.

Most of the "code" you see is actually pretty simple, but you probably see it tossed around so often it gets confusing. There are only a few simple commands that get used a majority of the time.

Have you ever opened a document in a word processor before? Maybe typed a paper up? If so, the process is the same for editing configuration files. If you goof up, that's what this forum is for :)

If you do eventually install Ubuntu, it'd probably be best to

1) dual boot (install Ubuntu to a different hard drive or) so you can go relax in Windows whenever you feel overwhelmed
2) have a high-speed internet connection, just to make things easy on yourself.

Also, here are some links that may be helpful:

[http://www.linuxlinks.com/Beginners/](http://www.linuxlinks.com/Beginners/)

I can see the value of using a LiveCD first but speaking personally I think you should just go for it.  By that I mean install Ubuntu together with Windows as a dual boot system.  You should definately NOT get rid of Windows until you are very comfortable with Linux.  Once you have Ubuntu installed you can learn by using it.  IMO the trouble with using a LiveCD is you want to try everything at once.  By actually using Linux for your day to day computing everything you slowly but surely learn as you go along.  

When I installed Ubuntu it took about four days to get everything tweaked as I wanted it, but I think that compares favourably with Windows.  I have installed Windows on a lot of PCs and not everything works first time.

There are two points I would make about Ubuntu:
1. If something doesnt work (like playing MP3s) it is almost certainly because you need to download something first, don't assume you have done something wrong or got a major problem.
2. The section in the install where it partitions your hard drive is not very intuitive and IMHO needs tidying up, so take it steady when you get to that bit.

But don't let any of that put you off, Ubuntu is brilliant.  Go for it!
