### Post by notwen on 2007-10-23
For starters try out Debian, see what Ubuntu is based on.  Then possibly try out a RH based distro such as Fedora.  Maybe even try out SUSE's OpenSUSE.  Both Fedora and OpenSUSE are very polished and reliable distros.  You may even want to try out one of the distros that you have a hand in actually building from he ground up, such as Gentoo or Arch Linux.  Search the web, there are tons of distros out there, very possible you'll come across one you may like more than Ubuntu. =]

### Post by cfaulkner on 2007-10-23
Most people run away from stuff like Gentoo Installs..  Gentoo installs can take 12 hours or more, because Gentoo compiles everything for your system.  I use Gentoo for my production server installs because I like a system that compiles for the system i'm using considering i have many diffrent motherboard types and such.  I can sit down with the make menuconfig in the kernel and tweak it the way I want to.

As mentioned earlier, if you really wanna learn how Linux works, Gentoo is probably your best bet.  If you really don't care, Ubuntu is the best one in my opinion.

### Post by drama1981 on 2007-10-23
i dont have much to add since ive only used ubuntu, ubuntu studio and fedora. first thing though you dont actually have to reinstall to go from ubuntu to ubuntu studio since ubuntu studios base is ubuntu itself just some packages added thats all.

sudo apt-get update && sudo apt-get install ubuntustudio-desktop ubuntustudio-audio ubuntustudio-audio-plugins ubuntustudio-graphics ubuntustudio-video linux-rt

taken from here

[https://help.ubuntu.com/community/UbuntuStudio/UpgradingFromGutsy](https://help.ubuntu.com/community/UbuntuStudio/UpgradingFromGutsy)

as you can see it adds a realtime kernal as well as some audio/video utilities and extra graphics apps.

while im partial to ubuntu i would also give fedora a try. fedora 8 should be final in a few weeks. i run ubuntu gutsy (upgraded to ubuntu studio) and fedora 8 test 3 in a dual boot setup. for fedora only being a test version its actually pretty stable. ive only had minor problems. and contrary to popular belief yum (fedora version of apt-get) isnt actually that slow (at least not anymore). on the plus side if you like to be able to easily choose between gnome and kde fedora is a good choice as theya re both included on the install dvd however the live cd only has one version. either kde or gnome depending on the live cd you choose. id deffinitly give it a go.
