### Post by sup on 2006-09-09
Oh you can run SOME of that stuff no problem. It's easy to install, and will detect all your hardware automatically. 

However, gaming is a little bit of pain. Some games can be played via wine ([www.winehq.org)](www.winehq.org)), but it is not usually easy to setup (although if you do not mind a little bit of googling, advice is not that hard to find), for example Warcraft III. Only a minimum of games have natives ports (i.e. you can play them just as well as on Windows) Civilization, on the contrary, cannot be played. You have two options to continue playing it - install Ubuntu alongside Windows (it is not hard and should be done automagically If you happen to have more partitions on your harddrive and back up data on one) or, if your computer is fairly new and your processor has two cores, you could try and play the game in virtual machine (xen) - but I have nto tried it, it might be hard to set up and it might be too slow.

As for the other programs, especially for the purchased one, something similar applies, although there is much more linux ports of "normal use applications"(for example opera and skype). However, there is one big difference - many programs have fully or almost equally functional alternatives, most of them for free (for example GIMP instead of photoshop, if you do not do really advaced stuff).

I would say, think twice before you convert, you might miss many functions. On the other hand, you can give it a try on your other machine or dualboot (that is really easy to set up) and if you like it, you might find things can be done in other ways and that linux is better for you. For example you definitely will not need you antivirus software, because there is so few viruses for linux that they never spread (and it is harder for them).

### Post by TheEmb3rEgg on 2006-09-09
It's not exactly hack-proof, but it's a lot more secure than windows. People don't often write viruses for Linux, but anti-virus software is recommended. Ubuntu is probably the best bet for you as long as you are willing to get out of the windows mindset a little bit. Everything isn't going to be exactly the same.

I currently have two different computers with Ubuntu running on them. Drivers were immediately found for them (had a little problem with sound on one of them.).

A lot of your software has been either ported to Linux, has a Linux alternative, or isn't needed anymore.

-Gaim allows the use of text instant messaging for MSN, Yahoo, ICQ, and AIM.
-Thunderbird or Evolution easily replaces Outlook.
-Skype has a Linux version.
-The GIMP is a useful image editor that could replace Paint Shop Pro
-You probably won't be needing Spybot or PCTOols antivirus.
-You can't/won't need to use PCTools Registry Cleaner
-Ubuntu comes with Firefox for the default Internet browser
-Opera has a Linux version unless I am mistaken. 
-Gftp can be used for your ftp needs.
-You can find codecs for most of those A/V files you have.
-NVU is a useful web page editor
-Internet Explorer can be installed via a program called Wine.

I don't know what some of  your other programs do, but most likely, you won't be able to run them, without the help of a program called Wine ([www.winehq.com)](www.winehq.com)). Wine is not guarenteed to make your software work, but it's worth a try. You have to remember that most software companies think that only Microsoft matters. 

You can try to run your games under Wine, but it takes a lot of time, and may not work at all.

Your best bet (like the poster above me mentioned) is to either dual-boot or use it on your other PC. Linux (in my experience) is much faster and more stable than Windows, by a long shot. It's also more customizable than Windows is. You can actually get it to look like OS X on Apple if you want. You'll learn to love Ubuntu (I have)

You have to keep in mind, that this will take some time to get used to. Some things won't work, others will work better than it would in Windows.

The best thing about Ubuntu is the community. If you have a question, go on to the wiki or the forums here and tell them your problem. Chances are your question will get answered quite quickly and thoroughly.

It's worth a try; it really is.
