### Post by pellgarlic on 2006-06-20
ubuntu is the default, built-in .vmx file that comes with the free vmware-player - just download the player from: [http://www.vmware.com/products/player/](http://www.vmware.com/products/player/) and you should get the ubuntu virtual machine too.

you can alternatively download the ubuntu 6.06 dapper drake cd - it acts as an installation cd, and also a "live" cd - you can boot your computer from it, and get a "live" ubuntu environment to check it out that way.
