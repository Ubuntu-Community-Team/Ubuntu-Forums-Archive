### Post by Kvark on 2005-07-09
Old games has a higher success rate with wine then new games. Sometimes the install program fails with wine, while the game itself still works if you copy paste an already installed folder from windows.


To start off the suggestions on games for kids. I suggest the package junior-games-gl. With it you get tuxkart [COLOR=DimGray](simple gocart game)[/COLOR] tuxracer [COLOR=DimGray](very neat penguin sking)[/COLOR] armagetron & gltron [COLOR=DimGray](racing games from the disney movie Tron, only difference between them is that gltron looks better while the bikes in armagetron feels better to drive)[/COLOR].
