### Post by Fidelio on 2007-06-02
You seem to have two separate questions.
Can you run limewire on ubuntu? You can run frostwire, which is almost identical. I think Limewire runs as well, but I haven't tried it.
Can you keep all your windows files? Sure. You can dual boot and have the choice between windows and ubuntu. You can copy all your windows files onto the new linux partitionif you have hd space. If you decide to, you can eventually ditch windows (I did) but you can run them side by side for a while to see if linux does what you need.
