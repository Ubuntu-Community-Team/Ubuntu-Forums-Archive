### Post by glennric on 2008-02-26
You have the default emerald theme installed.  
Go to System->Preferences->Emerald Theme Manager
and select a different one.  
You can also install new themes from there.

### Post by Iehova on 2008-02-26
If you want the original window decorator back to alt+F2 and type "metacity --replace". If you want to change your emerald theme do alt+F2 and "emerald-theme-manager". :)
