### Post by Jussi Kukkonen on 2007-02-06
From *man metacity*:
> metacity is a minimal X window manager that tries to be as crack free  as  possible. It supports sloppy focussing and multiple desktops.


EDIT: 'crack' refers to anything the metacity developers consider non-essential for a window manager. Metacity is  also the default window manager for gnome, so you're already using it.,
