### Post by nimajiman on 2007-09-24
Maybe it's just me, but I am constantly frustrated by the layout of the default gnome file-select/save dialogue screen that comes up in firefox. It may be the theme I'm using or the size of my resolution, but to navigate to the desired folder quickly when I want to download a file just feels cumbersome and annoying.

Anyway, for anyone that's interested there's an easy way to change the default to look like the one in the screenshot. Sure, it doesn't look the prettiest and doesn't match the themes of my other dialogue windows, but if that doesn't bother you then you may find it easier to use like I do!

Just open firefox and type "about:config" in the address bar (no quotes). Find the string called "ui.allow_platform_file_picker" and double-click it. The status should change to 'user set' and the value to 'false'. From then on, you will get the new save file dialogue window (see pictures below).
