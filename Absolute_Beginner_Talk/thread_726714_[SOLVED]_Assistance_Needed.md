### Post by moephan on 2008-03-17
The best way to assess how Ubuntu will run on your hardware is to download it and burn a CD. Then simply boot from the CD and run Ubuntu in Live mode. If your tablet works in Live mode, you know it will work when you install Ubuntu. If it doesn't work when running from the Live CD, there may still be ways to get it to work.

In terms of running Windows applications, it's not just rumors, there are several good ways to run Windows applicaton on Linux. You can use things like Wine or Cedega, or using virtual machine software. You have lots of choices.

If you want answerws specicaly about your tablet or certain Windows software, you might consider reposting with your specific questions in the title, sich as "can I use my Adesso CyberTablet 12000 with Ubuntu", or "what is the best way to run <whatever Windows app> on Ubuntu>.

HTH

### Post by LeoSolaris on 2008-03-17
[https://wiki.ubuntu.com/HardwareSupport](https://wiki.ubuntu.com/HardwareSupport)

That will help you with figuring out if your hardware is directly supported. If you find a piece of hardware that is not in there, you should search the forums (with the name of the hardware at the search tool at the top of the main page).

And yes, there is wine, which stands for Wine Is Not an Emulator. (bit of geek humor) It is technically a Window's compatibility layer. It is limited and buggy as hell (the fact that anything works with wine is amasing), and can be somewhat unsafe as some viruses can infect it (although the bug will instantly be confused by the rest of Linux!)

Another option to run windows things, though it slightly slows down the Window's programs, is to install a full version of Windows as a virtual OS with something like VMware or Zen. There are others that do that too, I just can't think of all of the names. Doing that would boot up Windows in it's own little window on your desktop, which is kinda neat to tell ya the truth, but it has limitations, too.

I hope that all helps, and I wish you the best of luck in your Linux experence! 

Leo

P.S If Ubuntu doesn't support all of your hardware and you cannot find anything more in the forums, or it just flat out will not run, which happens, another distro may work instead. Check out Distrowatch.org if you need to. They maintain news and links to a massive number of Linux distros.
