### Post by MobiusNZ on 2008-03-20
I've noticed the live CDs often have garbaged screens on boot - it depends on how basic your vid card is, but it usually sorts itself out by the time the os has fully loaded.

As for the offset screen, this is quite common with D-SUB cables (the ones that have blue connectors/plugs) - it shouldn't happen at all with DVI (white plugs). If you have an LCD screen, use the auto adjust feature - if you have a CRT screen then use the monitor controls to move the image. In both cases, look in your monitor manual if you don't know how.
