### Post by Pragmatist on 2006-03-20
It might have to do with the fact that you showed it thru a screensaver.  Screensavers might provide a somewhat degraded image because it is trying to...um...save your screen? :)

Of course, you might be right about the LiveCD.  I would try **gthumb**
which almost certainly would be in a menu on the LiveCD.  There are MANY more such programs specifically for projectors, for slideshows, etc...but you probably need to have Ubuntu installed so you can install the software (which is just a touch of a button through a program called "synaptic").  Also, I found a whole bunch more for download using [www.google.com/linux](www.google.com/linux).

**gthumb** is probably simplest for what your trying to do.
