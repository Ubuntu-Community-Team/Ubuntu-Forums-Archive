### Post by _simon_ on 2006-10-08
The themes are the same, just the extension is different.

The new extension is .emerald so all you need to do is rename any existing ones you have.

Alternatively download emerald-themes.

If you already have Beryl then you have the repo so just do a sudo apt-get install emerald-themes or find it in synaptic.
