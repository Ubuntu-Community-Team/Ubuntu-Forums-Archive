### Post by Super TWiT on 2007-07-17
I think this can be done by putting the auto option in the fstab line so that it would look like this:```
/dev/sda1 /home/dragonneus/usbdisk1 ext3 defaults,auto 0 0
``` and then doing the same for the other device.  As for it getting the devices mixed up, I have no clue why that would happen.

### Post by bodhi.zazen on 2007-07-17
This is easily solved if you mount the device by Label or UUID

so replace the /dev/sdxx with LABEL=<label> or UUID=xxxxx

You cal list your drives (by uuid) with the command : ```
blkid
```

See here for details : [http://ubuntuforums.org/showthread.php?&t=283131](http://ubuntuforums.org/showthread.php?&t=283131)
