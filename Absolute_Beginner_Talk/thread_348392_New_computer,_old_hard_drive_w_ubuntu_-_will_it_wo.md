### Post by irish_flu on 2007-01-28
The "boot loader" is on the first hard drive, so it should work fine and let you pick your OS just the same.

However, I'd be very wary of both Windows and Ubuntu Linux if the machine is much different from the old one.  If it were me, I'd run setup in Windows (right over the old one, without erasing the old) and Ubuntu, too.

You can make both of them try to find the new hardware, but both will also have a bunch of "junk" left over from the old machine (the config files in Ubuntu, and the Registry in Windows).  Reinstall if at all possible.

### Post by irish_flu on 2007-01-28
> **Delta Factor said:**
> I may just replace my motherboard, processor, and memory, keeping all the old devices (video card, dvd, etc.) Do you think that would make things more likely to work without reinstalling?

Iffy.  Your motherboard has all sorts of resources attached to it that both Windows and Linux interact with.  Depending on what processor you're replacing, that might make a difference too (for instance, going from an Intel Celeron to an AMD X64x2 or an Intel Core2 Duo).

I'm not saying it won't work; I'm just saying that it might not work, and performance will probably suffer, in Windows and Linux both.
