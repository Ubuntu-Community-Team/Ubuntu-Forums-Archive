### Post by Toxicity999 on 2008-01-06
For AWN you can drag launchers form the main menu (Alt+F1) but for the record: /usr/bin/firefox

AWN cannot be moved at this time, the portability is there, all that's lacking is the code to do so.

What you're thinking of are GTK themes. See gnome-look or opendesktop for those.

When the prompt disappears it means something is running, closing the terminal will either terminate it, or just fork it to the background. If it's non-important, press Ctrl+C
