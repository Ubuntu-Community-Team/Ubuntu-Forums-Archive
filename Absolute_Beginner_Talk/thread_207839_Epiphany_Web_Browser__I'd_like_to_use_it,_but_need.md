### Post by aysiu on 2006-07-02
#3: Not all browsers do that--they do that by default, but that feature can be turned off in Firefox and Opera.

The truth is that Epiphany is a good minimalist browser that integrates well with Gnome and is a tiny bit faster than Firefox, but Epiphany lacks a lot of useful extensions and more-than-basic preference options.

I'd just stick with Firefox if I were you.
