A PS/2 mouse should be automatically detected. You're saying that it's *not*, or that it's conflicting with the USB wireless key? 

From what I've read, USB wireless devices are pretty hit-or-miss with Linux. It may work if you use NDISwrapper to install the correct Windows drivers for it, but you'll likely have better luck with a PC card or an internal device.
