### Post by sumguy231 on 2008-03-15
Try running kcontrol to see if it's already installed. If it isn't install it. Once it's started, go to Desktop -> Behavior, and under "Menu bar at the top of the screen", try checking "none" if it's on some other setting. I've never tested that feature in GNOME to see if it can work there, but it sounds like your problem. By the way, your screenshot is missing.
