### Post by neurostu on 2008-03-10
First you need to partition the drive into 3 partitions.

1- Partition for Ubuntu
2- partition for Red hat
3- parition for swap

There are plenty of posts on how to do this, I would recommend you read them extensively before you begin.

To partition your drive you can use the GParted utility on the Ubuntu live cd
