### Post by xequence on 2005-09-09
Limewire is good on gnutella, though its better on window then linux. Its really slow on linux...

Bittorent is in my opinion the BEST p2p program. You download a small .torrent file from a website. It tells the program what file to download, and it downloads small bits of the file from many people. Basically one person gets a part, passes it on, they pass it on, etc. It is very good for distrobuting large files like linux distros or movies.

You search on a torrent site for a torrent file. The site says how many seeders (people with the complete file, giving it to others) and peers (people with the file partially downloaded, and uploading the parts they have to others.) the torrent has so you can decide which to download. In bittorent you have to share the file back and in many exclusive sites you get banned if you dont have a good share ratio.
