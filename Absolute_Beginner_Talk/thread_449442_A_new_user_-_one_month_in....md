### Post by dpzektor on 2007-05-20
Well, it has been a month since I decided to go with Ubuntu. I was completely fed up with Windows (I was last running Vista) and decided to go the route most newbies do not...complete format/re-partition and run solely Ubuntu. I had everything backed up anyway, so what the hell.

I tried Linux years ago...Suse or Redhat I believe. It was not intuitive, and I couldn't get things going at that time. I didn;t give it much time at all, and back to Windows I went. Ubuntu was completely the opposite however. It is different, but the same in the way of simplicity for the first time user. 

I eventually found equivalents to all of the apps I used in Windows, and it most cases these apps function better. I have been able to get quite a few emulators working (I have been an emulation guy since about 1997), got DV video capture working (along with video converting...easy as pie here!), got seperate X screens working for my Nvidia TV out, and even run a few programs (Meka, Magic Engine) via Wine....which is incredible as well.

This community has been an excellent resource of knowledge, as well as a great help to little ol' me in a few times of distress. Thank you all for that.

Will I be the ultimate Linux geek and get to know the OS inside out? Maybe, maybe not. Will I be able to get my work done and have fun at the same time? Abosolutely!
