### Post by rax_m on 2007-12-07
HI, 

i've been having similar issues with Firefox on Gutsy (but with Gnome desktop). My solution has been to install SwiftFox ([www.getswiftfox.com](www.getswiftfox.com))  I find it much more stable and all my plugins do not have to be reinstalled :) 

Also if your system freezes with no mouse, keyboard etc. Try the Alt-SysReq-K combination ... it kills everything and GDM restarts and you can log in without restarting the box.

### Post by Riffer on 2007-12-07
It could be a problem with your nVidia card.  There has been issues around the drivers for these cards and one of the symptoms is firefox freezes.

I fixed this problem by installing the new beta driver from nVidia (169.04 I think) which isn't part of the Ubuntu repo's.  You can download and get instructions from the nVidia site.
