### Post by carlosqueso on 2006-06-02
You need to install the flash plugin.  You first need to enable the multiverse repository, which if you have dapper, can be done with these instructions: [http://ubuntuforums.org/showthread.php](http://ubuntuforums.org/showthread.php) Then install flash by typing ```
sudo apt-get install flashplugin-nonfree
``` If you need more help, or are using breezy, let me know.
