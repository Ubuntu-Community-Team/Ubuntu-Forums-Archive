### Post by Dr Small on 2008-01-26
The terminal can be launched a number of different ways.
For instances, you can go to Applications > Accessories > Terminal
which would undoubtedly launch your terminal.

By pressing ALT + F2 and typing in:
```
gnome-terminal
```
or
```
xterm
```

Or if you would rather use a Virtual Terminal, you can press CTRL + ALT + F1 - F6
which will take you to a fullscreen black and white terminal. To get back to your GUI, press CTRL + ALT + F7

Good luck!
Dr Small
