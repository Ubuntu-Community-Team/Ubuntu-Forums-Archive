### Post by eewald on 2006-12-07
i personally would not translate anything on rosetta, but use gnome's official translation project. this has two benefits. 
1) your work reaches to far more people, because all distros (including ubuntu) use that translation. 
2) translation is more homogenous, because all the work is done  and changes discussed in same plase. 

rosetta does more damage than good in my opinion. it duplicates work and scatters localisation communities, which are scarse anyway.
