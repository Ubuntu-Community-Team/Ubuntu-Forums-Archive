### Post by aysiu on 2007-01-06
There are two major versions of ID3 tagging--v.1 and v.2.

If you use a tag editor like TagTool (available in the repositories), you can easily see both v.1 and v.2 tags. Some players favor one version over the other. Some ripping applications favor one tag over the other.
