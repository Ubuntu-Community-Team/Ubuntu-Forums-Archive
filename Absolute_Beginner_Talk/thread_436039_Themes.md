### Post by lyceum on 2007-05-07
Wallpapers for.... Wallpapers (sorry :))
Metacity for the Window themes
Icons for... Icons (again, sorry :))
GDM for log in window

Just go to Systems, Preferances, themes and drag the unexported files over the theme manager and they will load. 

For GDM you need to go to System, administration, log in and use the install button.

For splash screens you need to use synaptic package manager to load the gnome-splash-screen-manager, then it will be in preferances to add splash screens and chose the one you want.
