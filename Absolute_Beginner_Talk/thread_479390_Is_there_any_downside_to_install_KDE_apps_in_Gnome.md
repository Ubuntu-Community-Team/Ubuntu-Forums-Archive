### Post by Adamant1988 on 2007-06-20
Naw, the only downside you might see is slower start times for KDE applications.  KDE keeps certain libraries on call all the time so that it can start applications very quickly.  GNOME doesn't do this, but the apps will behave fine, otherwise. :)
