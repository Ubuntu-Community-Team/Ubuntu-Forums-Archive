### Post by IronArjen on 2006-08-11
I just had a similar problem today that our pals from the forum solved for me. imagine we name your program myfile.

Typically you start your program then by typing in Terminal

      myfile

certain programs require that you are in the correct directory, You can search for your filelocation using the commandline, 

whereis 'myfile'

Always using the commandline is annoying. So once you know how to start you application start from your application menu the program accesories-> Alacarte In the left panel you see the icons that exist in your applications bar (if they have anything in them that is). Select the submenu in whihc you want to put your new softwar and click file, add. Type in name command etc.

Hope it helps
