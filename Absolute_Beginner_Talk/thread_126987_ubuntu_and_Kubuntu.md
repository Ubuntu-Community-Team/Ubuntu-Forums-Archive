### Post by Sokraates on 2006-02-09
If you download KDE, try the "kubuntu-desktop"-metapackage first. It will give you Kubuntu with all its tiny modifications concerning apps, default settings etc (e.g. kubuntu uses systemsettings instead of kcontrol). It's perfect to try which desktop-environment suits you best.

Installing the "kde"-metapackage will give you the full KDE, but you may find a lot of packages you don't need. Then again I found kubuntu-desktop lacking a few packages I wanted. Nothing you can't install afterwards, though.
