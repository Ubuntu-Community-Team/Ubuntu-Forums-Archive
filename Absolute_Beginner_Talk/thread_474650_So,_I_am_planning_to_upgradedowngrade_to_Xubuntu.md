### Post by jdusablon on 2007-06-22
Xubuntu from CD and xubuntu-desktop install after default ubuntu install are not the same thing.
If you want the lightness of Xubuntu, you should fresh install from Xubuntu CD. Otherwise, you've got all that gnome kruft still running. You can confirm this by looking at your applications menu.

Personally, I wanted the same thing... the speed of xubuntu. Turns out, they don't tell you alot of things about xubuntu... No built-in cd burning in Thunar and xfburn is garbage, really. Also the xfce4 panel and its plugins aren't very friendly to use, no dragndrop. I ended up sacrificing so many little features that I just switched back to gnome.
