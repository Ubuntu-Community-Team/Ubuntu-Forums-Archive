### Post by msak007 on 2006-12-01
I was initially using Thunderbird in Windows, continued using that in Kubuntu for a while (because it's all I knew), then switched to KMail as part of Kontact. I love how everything is integrated (mail, calendar, contacts, RSS reader) and how KMail can sit in the System Tray to let me know when I have new mail. I keep it running all day and it doesn't use many resources. Thunderbird is missing this feature and you have to install an extension for it, and it doesn't work so well. It doesn't even have a calendar feature built in and, for me anyway, seemed to experience the same memory leak issues that Firefox 1.5 suffered from. But on the other hand, I do miss all the extensions that are available for Thunderbird (especially the WebMail extension). I might give Thunderbird a shot once the new version comes out and see how I like it.
