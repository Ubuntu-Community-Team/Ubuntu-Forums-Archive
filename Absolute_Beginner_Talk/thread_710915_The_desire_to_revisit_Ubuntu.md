### Post by bumanie on 2008-02-29
There are many tutorials available. Personally, I would suggest that you choose manual at the partitioning stage, this way you can double check  that your windows partition is safe. As far as being able to choose which OS to boot, grub should recognize windows and give you a choice at boot as to which OS to load. This usually works very well in dual boot systems.
