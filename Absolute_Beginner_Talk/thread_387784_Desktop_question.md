### Post by Icemanyurt on 2007-03-18
Is it possible to remove mounted objects from the desktop and have them somewhere else (like a panel)?

I am using Edgy...

---

### Post by annda on 2007-03-18
run gconf-editor (in GUI: applications > system tools > configuration editor) and go to:
apps > nautilus > desktop > volumes_visible UNCHECK that.

you will still have your drives accessible through the places menu.
