### Post by mostwanted on 2006-05-13
First of all, if you're thinking of WINE, it's not an emulator. It's an open source implementation of the Windows APIs (win32, directX, etc.) for Linux.

Here's a good site for checking if your desired app will work on WINE:

[http://appdb.winehq.org/appview.php?versionId=147](http://appdb.winehq.org/appview.php?versionId=147)

Edit: Silver ratings are pretty good. The latest rating with the third most recent release of WINE is silver. Just an observation.
