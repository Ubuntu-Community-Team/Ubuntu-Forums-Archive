### Post by bodhi.zazen on 2006-09-04
1. What language? 

Did you try System>Admin>Language Support ?

If that down not work, open a terminal
```
sudo apt-get install language-pack-xy
``` where xy = 2 ditit code for your language.

2. Can't help with wireless.

3. Your keyboard may be quite difficult to solve.
Try google search "Linux <name_or_your_keyboard>

4. You are not likely to get any type of a "how-to" with pictures (sorry).
Linux has a robust command line and it is much easier to give instructions like:

Open a termainal -> type the following command(s)

Then it is to say:
Open this window <insert picture> -> click this box <insert picture>....

Also for some actions (sys admin tasks) there is no GUI solution, sorry.
