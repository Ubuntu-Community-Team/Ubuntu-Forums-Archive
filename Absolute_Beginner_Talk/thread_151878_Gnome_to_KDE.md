### Post by aysiu on 2006-03-28
Well, the default key combination in KDE for switching between desktop is Control-Tab.

Unfortunately, for Firefox users, that's also the key combination for switching tabs in Firefox.

So if you have Firefox and KDE defaults, Control-Tab will switch desktops. You can, of course, reassign this in KControl to some other shortcut if your'e a big Firefox fan.
