### Post by smartboyathome on 2008-01-19
You boot up your computer with the CD in your drive and select "start or install ubuntu". Then, when it fully boots up (ie, the whole desktop appears), double click the icon on the desktop that says "install" (by the way, you can play around on this if you want, since it is a livecd). Answer the questions until you get to a part with a lot of balloons (should look like [this screenshot]("http://architectfantasy.com/wp-content/uploads/2007/10/screenshot-install.jpg")). Select "Resize partition #1 and use freed space" and select how much space you want Ubuntu to have. Then, click next, fill out the rest of the questions, and click install at the end.
