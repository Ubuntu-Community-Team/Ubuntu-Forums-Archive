### Post by nickburns on 2006-11-23
You aught to start by replacing every instance of:

Modes "1280"

with...

Modes "1280x800" "1024x768" "800x600" "640x480"

and add any other resolutions that you may need / want.


You have two options for making the changes...

1. If x is running, go to the terminal and type 'sudo gedit /etc/X11/xorg.conf' and edit

2. If you are at a terminal, b/c x is not running type 'sudo vi /ect/X11/xorg.conf' to edit
