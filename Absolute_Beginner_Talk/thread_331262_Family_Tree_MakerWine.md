### Post by Merovius on 2007-03-25
As an alternative I use VirtualBox. I installed it with Automatix and used it to install WindowsXP. It runs Windows in a window and from there I installed Family Tree Maker 2006. I had to put my family file on a cd and have FTM open it from there first. Then I saved it to the virtual "C" drive and all works as normal as can be. I can even flip back and forth between Gramps and FTM at will. I set FTM up to start at Windows boot so I can launch right into FTM. Seems to good to be true, it works so well. :)

### Post by Merovius on 2008-03-30
Sorry so slow to respond. Didn't realize there was e post on this old thread. You can get Virtualbox through Synaptic. Just search "Virtualbox". I have the plain Virtualbox installed. Not the OSE one. It is pretty simple to set up Windows XP with it. Make sure you install the "Guest Additions" they are easy to get online and vital for a good experience.
