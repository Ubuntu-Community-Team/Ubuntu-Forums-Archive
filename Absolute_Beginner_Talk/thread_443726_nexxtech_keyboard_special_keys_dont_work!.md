### Post by jeffeb3 on 2007-09-20
Some keys on some keyboards use unorthodox protocols.  I ran into this problem with my logitech keyboard.  The media buttons, the extra function keys, and the extra special keys don't show up as standard keystrokes.  I installed a program called keytouch that was able to capture most of them.  The important ones for me were the volume controls, since I don't have volume control on my speakers.  You'll have to do some things to get it to start with ubuntu, but there is information out there for it.
