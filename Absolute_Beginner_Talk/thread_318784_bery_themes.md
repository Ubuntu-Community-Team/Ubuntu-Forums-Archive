### Post by Sqwishy on 2006-12-14
Ok make sure beryl is running. If it isn't, type beryl-manager in the console, and if it still isn't running right click on the beryl icon and select reload beryl (or something).
If beryl is running open emerald theme manager, then clicky on a theme! It should change, if it doesn't make sure emerald is managing your window-border.
You can get lotsa [beryl themes at gnome-look]("http://www.gnome-look.org/index.php?xcontentmode=103&PHPSESSID=bd5c76708386ed2bdd0b8d51f42c1ccf")
