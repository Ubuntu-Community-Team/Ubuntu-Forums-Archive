### Post by starcannon on 2008-04-13
Not sure if this is of any use to you, but first you may want to grab jre from the Synaptic Package manager. If that isn't an option that will work then create a folder, anywhere you like, drag drop the java package into it, right click and extract all, then open up a terminal, cd into that folder, and follow the directions from the Sun website.

If however there is no specific reason I highly recommend using the packages in Synaptic, it takes care of all the details for you.
