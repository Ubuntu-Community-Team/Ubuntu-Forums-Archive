### Post by machoo02 on 2007-09-28
Virtualbox and VMWare are two kinds of virtual machine software...they create a virtual computer that runs on top of your host operating system.  You would install an operating system (such as WinXP) within this virtual machine, and thus be able to have two different operating systems (Ubuntu+WinXP) running at the same time.  The virtualized operating system "thinks" that is is installed within a real computer.  This means that most, if not all, of your programs will install and run within this virtual machine as if they were native (because in a sense, they are!).
