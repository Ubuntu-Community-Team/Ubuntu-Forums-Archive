### Post by samir85 on 2006-01-20
It always depends on the files you are searching for.

If you are looking for smaller files (like single mp3s) I can really recommend the gnutella network, where gtk-gnutella is a very good client.

If you are looking for bigger files (like movies ...), I would choose the bittorrent network, here you can find various good clients. Personally my choice is bittornado and also the new official bt client (which is not avaible on breezy) improved a lot.

### Post by PsychoTrauma on 2006-01-20

but when it has finished, can you make it so that it is not shared?[/QUOTE]

You can stop seeding the torrent any time you want but depending on the tracker that might get you banned if you share less than what you download. If you dont share while downloading other clients will choke your download speed to almost nothing. It's morally wrong to download a file and not give anything back because if everyone did that no data would be avaliable for anyone. You're not adding risk by seeding a illegal file since you have already commited the crime in the first place.
