### Post by jjacks on 2008-03-14
I am stating the following not knowing what level of a user that you are. Please do not take offense if you are a seasoned user. 

One thing to remember, you are selecting a disk image to record, not creating a "data disk" and just writing the file to media.  (Something that I have done several times, without thinking about it.)  If you write the file as data, depending on your settings, it might not close the session and might wait for you to continue.  (These will not be bootable.)  If you look at your project selections (Under File on some apps) and select to write from a saved image (Record CD from CD image, or something like that), browse to the saved ISO file, it should burn a bootable image.

For reference I use Roxio Easy CD Creator in Windows, since it came with my workstation.

Good luck and I hope this helps,

Jonathan
