### Post by capdog on 2007-02-23
Simple file sharing? It depends on the files. 

There is a neat utility called rsync, which keeps things syncronised by only copying over the parts that have changed. You could probably use that to keep the folders nicely and quickly syncronised. I'm no expert on it, but Google it and you should find a howto... even search this forum for rsync... there may even be a gui if you're lucky! :)
