### Post by jken146 on 2007-12-09
The repositories will catch up with the newer version, eventually.  Each update has to be tested to make sure it doesn't break other packages etc. before it's included in the Ubuntu repos, so there's no telling how long this will take.  It really depends on how important the update is judged to be.

Here's a guide to installing everything: [http://monkeyblog.org/ubuntu/installing/](http://monkeyblog.org/ubuntu/installing/)

If you want my advice, stick to what's in the repos unless you really need a newer version.

### Post by mcduck on 2007-12-09
The new version is not likely to be ever available as update. Ubuntu only provides updates for security reasons and to fix serious bugs. 

It's quite impossible for us to tell exactly how the program should be installed, as there are numerous different ways. But the package should contain a text file telling you how to install it. Most likely the package you got contains source code for the program, and you'll need to compile it.
