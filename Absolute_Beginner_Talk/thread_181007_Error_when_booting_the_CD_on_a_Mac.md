### Post by linear on 2006-05-23
The error from the Install CD sounds like you ended up in Open Firmware.
 
The CD or iso image used to burn it might be bad. Check the MD5 checksum of the iso vs. what is posted at the download site. It's often recommended to burn the CD's at a slow rate (4x or less).
 
If still having problems, post in this forum (more Mac users there): [http://www.ubuntuforums.org/forumdisplay.php?f=95]("http://www.ubuntuforums.org/forumdisplay.php?f=95")
