### Post by CJ56 on 2008-01-04
Should work okay. OpenOffice - which comes as part of the Ubuntu package - handles .doc files quite happily. If you use a wired ethernet connection then no problems accessing Internet & Emails. If it's wireless, same applies assuming that Dell have set up the wi-fi stuff for you. 

If all your files are stored on, say, an external HDD formatted by Window$ into NTFS you might have trouble accessing them from Ubuntu. Could be worth burning them onto a DVD and transferring them that way (or, very laboriously, using a USB flash drive over and over again...).

I use Ubuntu and nothing else on my desktop. I use it for work (I'm a writer), Internet/Emails, music, family photos, one or two video clips & to play the odd DVD. I'm the opposite of a power user, in other words, but I find that Ubuntu is excellent in all respects - much better than Xp. 

I don't know if you get a price break on the Dell for running Open Source software, but either way, I don't think you'll regret the change...

### Post by SteveHillier on 2008-01-04
First response is go for it.
I haven't tried wireless with Ubuntu so cannot comment but it installed my Konica Minolta printer and HP printers without problem.
I am also not a games player so cannot comment.  You will be able to get hold of your Word files OK and access them in Open Office.  If you need to share files you have created in Open Office you may need to take care - Office 2007 should be able to access them native but I haven't tested it yet.
Support on Dell kit is pretty poor with Windows installed.   How they support Ubuntu I would like to check on first.  Also I would want to know how they do the pre-install.  On Windows machines they give you a recovery disk that takes everything back to factory set - which has the side effect of losing all user data and user installed programs and settings.  Better if possible to by a bare bones machine and get a copy of Ubuntu on CD (you can download and burn your own CD) then you know you have the same system as those of us on this forum rather than some interpretation that Dell might have put on it.
