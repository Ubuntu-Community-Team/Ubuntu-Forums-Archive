### Post by harun on 2007-04-30
> **wiggumon said:**
> I have never tried gedit with plugins, but codeblocks is great. Just my 2 cents...

Neither had I up to a few weeks ago. I saw a similar thread back then with all these people saying they liked using gedit for coding. I didn't see the attraction so figured I must be missing something. I was. If you look through the menu's and options there are a number of things you can change. For example, your file list can be open on the left side and a window open in the bottom that will show you the output of running make. So while in gedit you can build using the makefile in your currect directory with one hotkey and see the output while still in gedit. It is quite good in the scenarios I use it for. I will use KDevelop and Code::Blocks as well but for the majority of stuff I just use gedit now.
