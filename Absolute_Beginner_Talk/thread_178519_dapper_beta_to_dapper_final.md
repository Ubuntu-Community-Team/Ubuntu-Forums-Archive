### Post by xXx 0wn3d xXx on 2006-05-17
[QUOTE=guitarded]i am currently using dapper beta.  just wondering if we are going to be able upgrade to the final release using dist-upgrade or will we need to do a full install and overwrite all our preferences?[/QUOTE]
You will just need to run "sudo apt-get update" and then "sudo apt-get upgrade." You don't need to download Dapper and install it again or do a dist-upgrade, you just need to download and intall the packages that need to be updated. Just like a normal upgrade.
