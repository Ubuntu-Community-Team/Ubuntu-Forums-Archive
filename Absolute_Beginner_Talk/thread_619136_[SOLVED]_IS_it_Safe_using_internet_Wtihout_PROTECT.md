### Post by Sef on 2007-11-21
There are no viruses for GNU/Linux in the wild.   In Ubuntu, there is a firewall installed by default (IPTables.)   If you want a GUI for it, then download Fire Starter from Applications > Add/Remove.

### Post by Paqman on 2007-11-21
In a way, Ubuntu IS protection.

The vast majority of malware is written for Windows and IE. By using neither you neatly sidestep most problems. Coupled with Ubuntu's permissions system and the built in firewall you shouldn't have a problem.

You can get antivirus for Linux. You can use it to check any files you're sharing with Windows PCs.

Just to clarify what Sef said: Firestarter isn't a firewall, it's a configuration GUI for the built-in firewall. Don't leave Firestarter running after you're finished configuring.
