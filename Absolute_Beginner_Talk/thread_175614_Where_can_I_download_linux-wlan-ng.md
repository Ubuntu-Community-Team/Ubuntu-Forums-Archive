### Post by az on 2006-05-13
[QUOTE=rubrboots]Hello, I am trying to get my wifi-card to work, and I need linux-wlan-ng. My linux box is not on the internet so I need to download it with xp first. Doe anyone know where I can find this file?[/QUOTE]
It's on the install cd.  You don't need to download it.   Just go into synaptic and install linux-wlan-ng

or run

sudo apt-get install linux-wlan-ng


You may or may not be prompted for your cd, since it is probablly cached in /var/cach/apt/archives.
