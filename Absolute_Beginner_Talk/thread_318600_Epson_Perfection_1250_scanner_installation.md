### Post by jeremy on 2006-12-14
to print the best thing to do, IMHO, is to use gimp (and gimp-print - install it if it isn't already there). Open gimp, and go to
File -> Acquire -> XSane
then scan into gimp, then
File -> Print
You will have to set up your printer in the print dialogue.
