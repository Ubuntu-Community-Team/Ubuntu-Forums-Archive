### Post by hyper_ch on 2008-03-25
well, Xubuntu is a lightweight desktop environemnt... by default you don't ahve all those eyecandies and customizations provided by Gnome or KDE... also the default applications are more light-weight and not as feature rich.

If you want a fast setup, use xubuntu... however if you need desperately a gnome or kde app, you can still install it. For example k3b is (IMHO) the best burning program out there... it relies on several kde libs. While those libs will be loaded with KDE directly k3b will load quicker in KDE... however you can use it also in Xfce but then it will need to load those additional libs which makes this application load slower in Xfce.
