### Post by xtacocorex on 2005-12-27
```

aerospace
backups
bin
boinc
crons
Desktop
f2kcli
music
personal
pictures
school
tuxmag
utility
videos

```

The structure is pretty simple. Personal holds personal stuff, utility holds program sources I download and compile myself, crons holds my personal cron file.

Since I'm an Aerospace Engineer, I have a bunch of technical papers and stuff related to that in the aerospace folder.

bin holds scripts that I've written and or modified from the internet.

My pictures folder has a subfolder called wallpapers that I symlink to /usr/share/wallpapers so I can easily add my wallpapers to the desktop in KDE.

I realized a while ago that upper case letters and spaces (mostly spaces) are hard to use in the command line, so the only files that have those are in my music folder for band, album, and song names as that is how they are ripped.
