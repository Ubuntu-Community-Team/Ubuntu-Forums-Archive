### Post by Sandlst on 2008-01-21
Hello, if you want to have the complete xubuntu DE from a server, you should try installing the ```
xubuntu-desktop
``` meta package, it should set up the gui painlessly for you, requiring only a restart after installing.  As for other distros, I would give Damn Small Linux ([http://damnsmalllinux.org/]("http://damnsmalllinux.org/")) and Puppy Linux ([http://puppylinux.org]("http://puppylinux.org")) a try, they are both very good lightweight distros IMO.
