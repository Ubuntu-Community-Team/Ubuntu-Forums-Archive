HI Chapita,

Well, Mandriva as it is called now uses rpm (redhat package manager) to install packages. Ubuntu is a debian derivative (someone shoot me if i'm wrong) and usest apt-get or Synaptic to install  packages. This is MUCH easier. As it does everything for you. It resolves dependency issues and ensures the oveall integrity of your system as a whole. It will also update and upgrade for you. So, yes, in my humble opinion Ubuntu is better than Mandrivia, although i have to admit i have not tried it. but i was with SuSE for a while which used rpm aswell so i am speaking from experience in that regard.

Is there something easier still? Well, i have been told yes. Mepis is also debian derivative (i'm sure there is abetter word, but i hope people reading understand what i'm getting at) and it is supposed to be up and running upon install. I would definitely suggest going with a Debian-style system. Mepis is much more point 'n click apparently.

I don't want to kid you, but with Ubuntu you do need a little patience getting everything up to speed. It is not hard and there is so much support here in the forums and the wikis and [www.ubuntuguide.org](www.ubuntuguide.org) to help set you up right.
 A good idea might be to get a live cd for several distributions, (Ubuntu, Mepis, Knopix etc.) and by process of elimination see which suits you and appeals to your taste. Live cd's mean that you can see exactly how a distro looks and works without having to install it to your hard drive first. Its like an easy test-drive, Then if you like it you can go for the full install. 

Personally, i prefer the aesthetics of Ubuntu over Mepis for example and with the ready support i am happy to learn to do a bit more in command line and slightly less point 'n clicking. 

i hope that helps,

Either way welcome to the GNU generation, O:)

### Post by heimo on 2005-08-16
Try Live CD. Synaptic package management is easy to use and personally I'm a great fan of Debian based package management (dpkg, .deb). You could try to make some room and triple-boot Windows-Mandrake-Ubuntu and decide later if you want to scratch those other operating systems. :) No need to uninstall anything, just install over if you don't need some partitions anymore - make backups of your personal data before install.

Ubuntu-repositories are very comprehensive. [http://packages.ubuntu.com/]("http://packages.ubuntu.com/") I have Redhat-Debian background and so far Ubuntu is the best distribution to my desktop - it combines the best bits of stability, great collection of software and freshness. And don't forget about Ubuntu family, this board. :) That's unique.

But don't listen to me, go get that Live CD and decide yourself. (It's of course slower than using from hard disk.) Or ... if you're not in hurry, check after couple months next Ubuntu release, Breezy Badger (5.10). It will have Gnome 2.12 and lots of other fresh stuff. (Upgrade from Hoary, current 5.04 version, will be possible.)
