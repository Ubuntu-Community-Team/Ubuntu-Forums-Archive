### Post by oilchangeguy on 2007-04-11
ubuntu uses a different file system than windows, no need to defrag. virus and spyware problems are not an issue in linux (it can happen, but most likely not) ubuntu does a good job of internal housekeeping. there are ways to check for and delete orphaned files, but this is not an issue either, as they don't take up very much room. on my system i don't do anything special to it, just use it, and so far no problems.
