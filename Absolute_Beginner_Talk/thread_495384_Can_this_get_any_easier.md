### Post by zero244 on 2007-07-08
The scanner may not work at this time. Keep in mind that scanner was designed to run on Windows only.
Combination printers\scanners are not supported well in Linux. You could dual boot with XP and use it that way.
Linux supports  a huge amount of hardware, if the lazy manufacturers would get off their cheap a?? and come up with some drivers. It would make all of our lives easier.
A lot of hardware vender's will write drivers for Mac, it couldn't be much more work to include some Linux compatible drivers too.
Do a Google search for your printer......there are lists out there that will show if there is any way to get you setup to work with Linux.
