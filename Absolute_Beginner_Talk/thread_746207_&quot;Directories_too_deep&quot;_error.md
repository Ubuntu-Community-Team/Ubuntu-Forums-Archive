### Post by kleo skywalker on 2008-04-05
Mount an iso? Never heard of that. Just right-click on the iso in your file browser, choose "Open with CD/ DVD creator", hit "Burn as image" and choose the lowest possible speed.
And by the way, it won't work if you burn it as a data CD. Burning it as an image will give you a LiveCD - you can just boot from that and try out things. You can install if you like, or not.

### Post by Joeb454 on 2008-04-05
Yes you burn the iso to a CD (make sure you have the "Desktop CD" (also called a "Live CD") and burn that.

If you are burning from Windows, i would recommend [imgburn]("http://www.imgburn.com/") for this, I use it all the time on Windows machines :)

You then insert the CD and change your boot options to boot from the CD, and you will have a running working Ubuntu Desktop....though bear in mind it won't run as fast as an installed version :)
