### Post by jrusso2 on 2007-06-21
PII and PIII's are capable of running linux very well.  The limiting factor is memory.  To do a live boot install of Ubuntu you need 192 mb of ram.

If you use the alternate install cd you can install with 128 mb of ram.

I would consider using the alternate install of xubuntu or fluxbuntu if your running 128 mb for ram however.
