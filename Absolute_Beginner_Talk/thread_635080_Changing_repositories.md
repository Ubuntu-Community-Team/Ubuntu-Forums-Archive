There sure is.  Go to System --> Administration --> Software Sources
Use the "Download from:" pull-down menu to select a different server.

### Post by rsambuca on 2007-12-08
Go to System -> Administration -> Software Sources. Make sure the boxes are checked. Press the "Download from" Button and select "Other". Press "Select Best Server". It will test all of the software repository mirrors and select the quickest one to your location. After that you should be fine.
