### Post by nalmeth on 2006-05-12
Is it just DVD-playback that is slow?
totem is just a frontend to the media engine.
Have you installed totem-xine yet? Sounds like you have dvdcss already.
Also, [enabling DMA]("http://ubuntuguide.org/#speedupcddvdrom") on your DVD-ROM can help the playback.

When watching dvd's, I like to bust out xine :)
in the terminal
```
sudo apt-get install xine-ui
```
EDIT: Not a person fan of gxine. It's pretty fast, but I don't like it's controls/layout.
To each their own
