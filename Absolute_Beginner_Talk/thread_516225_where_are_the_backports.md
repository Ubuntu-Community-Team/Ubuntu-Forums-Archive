### Post by yabbadabbadont on 2007-08-02
You have to enable the "fesity-backports" repository.  System->Administration->Synaptic Package Manager.  Settings menu->Updates tab.  Select "feisty-backports".  Close the dialog and hit the Reload button.  Now the versions of software in the backports repo will be available.
