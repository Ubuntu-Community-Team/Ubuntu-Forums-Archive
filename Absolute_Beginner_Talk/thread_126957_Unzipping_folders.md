### Post by GreyFox503 on 2006-02-07
Although the description is a bit... vague, I'm going to take a stab and guess that viper just downloaded an Ubuntu iso and is trying to use it.  (Winrar associates itself with *.iso files for some dumb reason)

If you downloaded a *.iso file of Ubuntu, you need to burn it to a CD as an image file, either with your CD burning software, or like this method:

[https://wiki.ubuntu.com/BurningIsoHowto]("https://wiki.ubuntu.com/BurningIsoHowto")

Just remember to burn as disc image, not as data disc.  :)


When the disc is done burning and you want to use it, just reboot your computer with the CD in the drive boot from the CD-ROM.  If it doesn't work automatically, you will have to enter the BIOS options and choose to boot from your CD-ROM.
