### Post by Quillz on 2007-01-24
The default UI of Freespire is almost a clone of Windows XP. Ubuntu is also a great beginner distro, as is openSUSE.

Don't be confused by "beginner distro," though. In no way does it mean it is incapable of doing powerful things. This is not true at all. Theoretically, all distros are the same, because they all use the same Linux kernel. What matters is how each distro is packaged.

### Post by VoodooSteve on 2007-01-24
When I began with Linux,  I started with Ubuntu. I heard it was very easy to newcomers from word of mouth and reviews. However, I did not have your problem with printers and PDA's. Your solution might be a dual boot setup. That way, when you do not need your PDA etc, you can use Linux and get familiar with whatever distro you choose. When you do need them, you can boot into Windows. Once you get more familiar with Linux, then you might not need the simple plug and play functionality.

### Post by homh on 2007-01-25
Ubuntu is a friendly and easy to install distribution, as are some of the others mentioned above.  Perhaps your expectations are too high.  Some hardware wont work with any distro or not work well.  Printers can be a problem and some companies dont want to release drivers for Linux or make their hardware Linux-friendly.  As was mentioned you might want to try keeping a copy of MSWINDOWS on a dual-boot machine.  If thats not something your willing to do you might decide to stay with MSWINDOWS for now.  If Ubuntu isnt hardware compatible with what you want to use, I doubt another distro would give you better results.
