### Post by MonkeyBoy on 2007-12-02
The way I have always done this is to go to System>Administration>Network and make sure my wireless device is enabled and configure my SSID and security etc.  The Gutsy desktop network icon next to the date allows for some network searching etc.

I hope this helps.  It only works if your devices are all fully supported.  

If it doesn't help you may want to find the name of your wireless device (chipset etc) and search the forums using that.
