### Post by edm1 on 2008-01-13
I dont think you can recieve password hints, it would jeopardise the overall security of the OS. For a login screen with a browser go 'System>Admin>Login Window>Local' then select an appropriate theme. Login windows with customisable pictures for each user will be included in Ubuntu 8.04 i think.
