### Post by Hospadar on 2007-09-24
The places menu is generated a little different from the other two menus, the items there are just your nautilus bookmarks and places.  so if you want to add a folder to this menu, open up a nautilus window (the file browser) and navigate to the folder you want to add, and go to bookmarks->Add Bookmark.

You could also go to Bookmarks->Edit Bookmarks to remove or change around your bookmarks.

As for pdfs, you'll need to install the adobe pdf reader.  I just saw a guide about how to set it up on the forums today: [http://ubuntuforums.org/showthread.php?t=490459](http://ubuntuforums.org/showthread.php?t=490459)
