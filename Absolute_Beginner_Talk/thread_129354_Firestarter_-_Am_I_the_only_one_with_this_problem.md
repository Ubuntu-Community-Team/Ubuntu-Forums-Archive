### Post by TrendyDark on 2006-02-13
There really isn't a need for Firestarter if you're using dial up. Firestarter is a firewall program that generally just provides connectivity security, whcih isn't a big issue with dialup for two reasons. One, it's very slow, two, your IP will change everytime you reconnect.

### Post by cwaldbieser on 2006-02-13
[QUOTE=OtrMan]Thanks for the reply.  How can I protect my dial-up connection if I can't use a firewall?[/QUOTE]
I think the point is that unless you leave the dialup connection connected 24/7, you probably can get away without having a firewall.  You probably just aren't connected long enough for an attacker to find you and exploit the connection.

### Post by issueperson on 2006-02-14
if someone is looking for a computer to exploit, they will probably go for someone with a faster connection.  it just takes too long to mess with a computer on dial-up when there are so many others out there with faster connections.

also, Ubuntu comes with its ports closed as the default setting, so unless you do stuff to open your ports up, you should be pretty safe.
