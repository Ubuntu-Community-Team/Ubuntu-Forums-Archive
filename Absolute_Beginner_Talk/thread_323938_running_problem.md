### Post by Ecthelion on 2006-12-23
You could change the menu entry in the applications menu to run the correct file.
Just right-click, say edit menu's, go to the realplayer object and say properties.
Change the command to the path of the file you usually want to run to use realplayer
Normally you could just use the application menu again to run realplayer
