### Post by Extrapolation on 2007-02-01
What's the file type of the theme?  If it's *.deb you can just double-click and install it (if you have GDebi, which you probably do).  If it's *.tar.bz2 you should be able to drag-and-drop it on the Themes dialog and it'll install.
