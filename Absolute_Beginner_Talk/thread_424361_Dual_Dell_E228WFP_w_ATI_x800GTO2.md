### Post by maiios on 2007-04-27
I had pretty good results with this:

[http://ubuntuforums.org/showthread.php?t=221174](http://ubuntuforums.org/showthread.php?t=221174)

I had the best results with MergedFB with my Radeon card. I have a laptop, and it didn't always switch out of dual monitor mode so well, but all that you have to do is run 

```
 xrandr -s 0 
```

to go to single screen mode. Change the 0 to a 1 if you want to go to dual screen mode.
