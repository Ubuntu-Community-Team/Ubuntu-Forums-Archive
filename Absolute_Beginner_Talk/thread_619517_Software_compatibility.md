### Post by Inxsible on 2007-11-21
The system requirements there list only Windows and Mac and that too they say that the newer core 2 duo have not been tested.

You may be able to run it under Wine (Wine is not an emulator) which allows linux users to run software made for Windows. but such proprietary software may or may not run. there is no guarantee.
