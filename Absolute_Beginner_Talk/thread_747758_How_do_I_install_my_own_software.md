### Post by SunnyRabbiera on 2008-04-06
Most of the software you will need are in the repositories.
just remember this:
1: windows .exe files will NOT work by default, now you can get some windows software to work in linux but you will need wine.
but also keep in mind not all windows applications will work under wine.
2: most of the apps on the internet are targeted at windows, please make sure you search for apps that have linux compatibility.
3: the main installer format in ubuntu is .deb, if you do indeed find something on the net try to download something in that format.
.rpm and .tar will not install unless you do certain things like use something called alien, a terminal application.

but dont let this discourage you, if you need we can suggest some linux alternatives to your favorite windows software.
