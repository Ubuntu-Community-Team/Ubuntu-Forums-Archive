### Post by Compyx on 2007-09-01
Since the OP mentioned he/she doesn't have an internet connection under Linux, apt-get isn't very likely to work.

To the OP: you don't appear to have a compiler installed, With a working internet connection you should do:
```

sudo apt-get install build-essential

```
to install gcc, binutils and some other essential stuff.

Mind you, installing a package like VLC from source is asking for trouble, you'll soon find yourself in dependency-hell. Only install from source if the package you want isn't in Synaptic or if you need specific features/tweaks the package in Synaptic isn't providing.

My advice would be to get a working internet connection, without it you're pretty much screwed.
