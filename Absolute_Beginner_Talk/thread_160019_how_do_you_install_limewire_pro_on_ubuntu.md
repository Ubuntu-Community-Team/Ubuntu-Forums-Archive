### Post by Perfect Storm on 2006-04-14
Download the .rpm version of Limewire Pro and save it to your Desktop.

```
sudo apt-get install alien
cd Desktop
sudo alien -i XXXXXXXXXX.rpm

```

Where XXXXXX is the name of the package.
