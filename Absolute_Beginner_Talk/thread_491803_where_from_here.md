### Post by aquavitae on 2007-07-04
If you really want to learn Linux (Linux, not just Ubuntu)  I'd suggest installing Gentoo in a seprate partition.  Unless you use the install cd (which last time I tried, didn't really work) you have to install everything by hand.  Its a great way to learn linux basics, whcih of course carries through to ubuntu.  About the only new thing I had to learn when I started on Ubuntu was how to use apt-get, and really, one package manager is very similar to another!  If you really want to go hardcore you could try LFS (Linux From Scratch) which in more of a handbook than a distro and tells you how to build your own distro.  But it requires a lot of time.  If you do decide to do either of these, keep ubuntu on your pc, cos it will take a couple of days before iether is working fully.
