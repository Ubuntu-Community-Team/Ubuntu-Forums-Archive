### Post by GavinZac on 2008-01-13
You can use Add/Remove programs in your applications menu, or the more advanced "Synaptic Package Manager" in your system menu.

Both of these use Ubuntu's central 'database' of programs to find the right one for you and install it. They are the same as the terminal commands "apt-get install"

To answer your question, like everyone says, use add/remove or the Synaptic package manager.

By the way, to make use of KDE (K Desktop Environment) install the [COLOR="Blue"]kubuntu-desktop[/COLOR] package.  Be forewarned, when you do this, it will install KDE and a number of accessories that go with it.  During the process, it will ask you about which desktop manager to use kdm or gdm.  Either is fine as it only really affects the appearance of your login screen.  However, you are currently using gdm and you'll probably want to stick with it.
