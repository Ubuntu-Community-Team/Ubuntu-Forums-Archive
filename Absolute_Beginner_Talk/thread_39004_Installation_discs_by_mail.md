### Post by YopY on 2005-06-03
You could also 'borrow' someone else's CD player and install it in your own computer, plain CD players shouldn't cost more than ten bucks nowadays (if you can even find them, that is), and it's not that hard to install one into your computer if you know what you're doing.

Another alternative would be to rip out your hard drive, stash it in another computer, copy the Ubuntu install files onto your own hard drive, stash your hard drive back into your own computer, and run the install. There are howtos and documentations about how to install Ubuntu from your hard drive if i'm correct.

Another alternative would be to run an install from an USB drive, but i don't know for sure if the ubuntu installer allows this. 

But the best and cheapest solution would be to get a new CD player.
