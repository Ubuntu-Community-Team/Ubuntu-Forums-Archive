### Post by Qrk on 2006-09-24
ndiswrapper is included on the install cd, it just isn't installed by default. You can locate the deb on the install cd and install it from there or set up the cd as a source for apt.  You can also use another (connected) computer to download it from [http://packages.ubuntu.com/](http://packages.ubuntu.com/)
