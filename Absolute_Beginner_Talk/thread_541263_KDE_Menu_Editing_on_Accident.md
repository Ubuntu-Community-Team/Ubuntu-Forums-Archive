### Post by ridgeland on 2007-09-04
In Gnome items in menus have a checkbox on/off.  I don't know about KDE.
Just an idea on partitions.   I keep two Ubuntu partitions (8 GB each).  One is my main partition.  The second is a sand-box.  I'll try any software in the sand-box.  If the software is a keeper then I'll install it on the main partition.  So far the sand-box partition has not died, but if it does I'm still safe with the main partition.  Both access other partitions where data (images, music, writer files etc) are stored.  I even like use links for bookmarks so Firefox says in synch in both partitions.
