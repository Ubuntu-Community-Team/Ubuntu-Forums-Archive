### Post by RRS on 2006-06-19
[QUOTE=petgraveyard]Should I boot with the LiveCD of this program?

EDIT - Can I use this with Windows?  I'm trying to make a dual-boot between Ubuntu and WinXP[/QUOTE]

Yes to both your questions.
Booting from the gparted live cd will allow you to resize your current windows partition to make room and also create new partition(s) for your installation of Ubuntu.

The Ubuntu cd's also have a partitioning tool as part of the installation utility so it isn't neccessary to partition ahead of time, though some prefer to.

I'd suggest taking a look at these two sites;
[Moving from Windows to Ubuntu Linux: Getting Started]("http://www.psychocats.net/ubuntu/windowstoubuntu.html")
[Illustrated Dual Boot Site]("http://users.bigpond.net.au/hermanzone/index.htm")

They both contain excellent tutorials for installing Ubuntu to dual boot with windows.

Though this is a relatively safe operation it's always a good idea to backup any data you don't want to risk loosing, "better safe then sorry". Also run a couple defrags of windows to ensure no wasted space.
