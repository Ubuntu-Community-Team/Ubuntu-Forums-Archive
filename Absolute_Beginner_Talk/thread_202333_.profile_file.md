### Post by lkagan on 2006-06-23
Bash checks and loads settings from /etc/profile, /etc/bashrc, ~/.bash_profile, ~/.profile, ~/.bashrc.

They don't necessarily have to exist.   Always note that your local dot files are processed after the system (/etc) files and can override settings in them.  To have the 'final say', put things in .bashrc since it is the last configuration file that gets processed by bash.
