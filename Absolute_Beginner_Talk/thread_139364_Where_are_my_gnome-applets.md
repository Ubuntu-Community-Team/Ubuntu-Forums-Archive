I'm looking for a sticky note applet comparable to OS X's Stickies.  I've tried two in gDesklets, but have been unimpressed with both.  I don't like that you have to manually enter the 9 lines you want displayed.  I want something that is basically a floating resizable text box with word wrap.

A search in Synaptic shows me that a package called **gnome-applets** is already installed and includes Sticky notes.  But I can't figure out how to run it!   In fact, gnome-applets looks like it contains a few cool little gadgets that I want running.

- "gnome-applets" in the terminal tells me "command not found"
- anything that looks like a program name inside Synaptic's description of gnome-applets ("geyes," for example) returns "command not found"

Help?  Thanks.

### Post by Xian on 2006-03-03
Right-click on a clear portion of your gnome panel.
Select the option "Add to Panel", then select "Sticky Notes".
