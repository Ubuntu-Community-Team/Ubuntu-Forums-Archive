### Post by localuser on 2006-10-22
Keyring stores encrypted text and is used in particular to store encrypted passwords.

You can access the Keyring Manager at System | Administration | Keyring Manager.

[Edit] If Keyring Manager doesn't show up in Administration, you should be able to add it with Applications | Accessories | Alacarte Menu Editor, then select System | Administration on the left and Keyring Manager on the right.
