### Post by vedace on 2007-05-20
1. Yes
2. Yes, Bluefish.
3. Eh, not so much.  There is "The Gimp" and "GimpShop," but neither is quite as good as Photoshop.  Although for *most* common image editing requirements, Gimp should be sufficient.  You could also consider running an older version of Photoshop (e.g. photoshop 6) under CrossOver or wine, but this is of course a sloppy solution.  I haven't heard of anyone getting any of the recent releases - CS, CS2, CS3 - running in Linux yet.  You could alternatively (and optimally) install a windows emulator in Linux and install photoshop inside that.
4. See 3.
5. Yes, but it might take a bit of finagling.
