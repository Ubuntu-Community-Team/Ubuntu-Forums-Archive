I'm not 100% sure what your asking but downloading in ubuntu is done  using Applications -> add/remove or system -> administration -> synaptic package manager or the CLI (terminal)

You can download all those things there.

In ubuntu compiz is installed by default, to use all if it cable effects download this using synaptic:

compizconfig-settings-manager

For java: in synaptic search for "sun java" and pick this one:

sun-java6-jre

and 

sun-java6-plugin

For azureus go to add/remove (or synaptic) and search for it.
