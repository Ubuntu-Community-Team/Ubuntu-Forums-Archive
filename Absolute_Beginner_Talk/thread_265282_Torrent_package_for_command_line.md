### Post by skymt on 2006-09-25
rTorrent is a great text-based client. Many people prefer it to the graphical clients, because of the low resource use.

Be warned that the version in the repositories is kind of old. You'd be better off downloading and compiling the [latest stable versions](http://libtorrent.rakshasa.no/) of libTorrent and rTorrent.

### Post by andypaxo on 2006-09-25
The bittorrent package gives you a simple command line torrent downloader.

The bittornado package gives you a more interactive program, but still in text mode

Use 'apt-cache search torrent' to find more info

Hope this helps!
