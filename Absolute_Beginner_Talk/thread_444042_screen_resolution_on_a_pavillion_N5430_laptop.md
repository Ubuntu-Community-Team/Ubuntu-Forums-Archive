### Post by Kobalt on 2007-05-15
Open up a Terminal and enter ```
sudo dpkg-reconfigure -phigh xserver-xorg
```
Select the resolutions you want as available by hiting the spacebar then validate with Enter. Restart X with Ctrl+Alt+Backspace, and you're done.

### Post by renzokuken on 2007-05-18
well in that case use the "vesa" driver. i'm pretty sure it supports 1024x768.

open your xorg.conf

```
gksudo gedit /etc/X11/xorg.conf
```

and change the device section from this

```
Section "Device"
	Identifier	"Trident Microsystems CyberBlade/XP"
	Driver		[color=red]"trident"[/color]
	BusID		"PCI:1:0:0"
EndSection
```

to this

```
Section "Device"
	Identifier	"Trident Microsystems CyberBlade/XP"
	Driver		[color=blue]"vesa"[/color]
	BusID		"PCI:1:0:0"
EndSection
```

now save and close your xorg.conf and hit Ctrl+Alt+Backspace to restart X. let me know if it changes. you may have to select the new resolution from System -> Preferences -> Screen Resolution
