### Post by handy on 2007-01-01
If you right click on an icon & choose **Stretch Icon** from the menu that appears you can size each icon.

I'm sure that there is a more elegant way via a config' file! :) 

You may also find that by adjusting your font sizes you will be able to customise the size of title bar, panel & menu.

To do this go to ***Menu:* System / Preferences / Font**
