### Post by adamklempner on 2007-07-10
An mp3 cd is just a data cd with mp3's on it.  Start a new data cd project, put all of the mp3's on that you want and just burn it.  The default burning settings should work fine.

### Post by HereInOz on 2007-07-10
If it is a CD full of MP3 files that you want, you choose to create a data CD, and just burn the MP3 files to the cd as you would any other data file.

You should then be able to play those files in any compatible CD or DVD player (you may have to press the "go" button once, depending on the player).
