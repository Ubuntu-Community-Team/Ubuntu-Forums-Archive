### Post by angryfirelord on 2008-01-30
What are the specs on this computer? Anything with less than 256MB of RAM will have trouble running the LiveCD.
> -Does installing this completely wipe my old OS (WinXP) off it? allowing me to start from scratch with Xubuntu? if it doesnt, I only have 3GB left on this HD, would I be better reformatting it with Xp or something again?
You have the option of both. You can shrink Windows or get rid of it entirely. If you want to delete it. delete the Windows partition and create a / and a swap partition.
>  If it does the above, that is completely take off the old OS and allow me to run it completely on its own on the HD, erasing all the old stuff, is it possible to reinstall XP if i want to? A matter of putting in the xp cd again or...?
Yes. Just make sure when you use the Ubuntu or XP cd, your BIOS is set to boot off of the CD.
> Sorry if those sound completely nooby, I apologize, but you guys seem to be freindly enough with us ignorant souls, so be gentle
