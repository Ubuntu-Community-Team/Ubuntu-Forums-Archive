### Post by taurus on 2006-07-13
There are a bunch of programs would let you do that.  However, I always like k3b and you can access it from Applications -> Video & Sound (or something close to that name).  And if you don't have it, install it with

```

sudo apt-get install k3b

```

### Post by steve.horsley on 2006-07-13
Open the file browser (nautilus) and from the menu choose Go->CD/DVD writer. Open another browser window and go find the files, drag/drop them into the writer window.
