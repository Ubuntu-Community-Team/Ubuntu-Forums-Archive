### Post by taurus on 2007-01-28
Look in System -> Preferences -> Sound -> Sounds -> Default sound card: (from the drop down menu on the right) to see if you soundcard is on that list.

p.s.  If you decide to use the add on card, turn off the onboard in the BIOS.
