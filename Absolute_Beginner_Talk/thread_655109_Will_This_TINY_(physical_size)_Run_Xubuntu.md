### Post by simstim on 2008-01-03
I recently installed Xubuntu on a Zonbu and it works very well. With Openoffice and a few additional games added, and a 512MB swap partition, it takes up about 3GB on the 4GB flash card. Sound works "out of the box"and the VIA video driver can be installed using the very useful script from this thread:

[http://ubuntuforums.org/showthread.php?t=485646&highlight=openchrome](http://ubuntuforums.org/showthread.php?t=485646&highlight=openchrome)

The driver has stable support for 2D, but 3D is buggy.

Unfortunately not all of the units have space for a 2.5" drive (apparently the original prototype did, but the later ones don't). You can easily connect a USB drive for additional storage though.

If you're looking for something that can use an internal drive, then the [VIA Artigo]("http://www.via.com.tw/en/products/embedded/artigo/") kit may be a good option.
