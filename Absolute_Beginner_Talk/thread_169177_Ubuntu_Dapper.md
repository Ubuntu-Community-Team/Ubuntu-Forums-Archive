I suppose you just use the file manager to add/remove files from the mp3-player? Then those files you removed from the player are probably in a wastebasket in the player, still eating disk space. If you browse to the player with 'show hidden files' enabled in file manager you'll see the hidden wastebasket directory and you can remove it to free your diskspace :)
