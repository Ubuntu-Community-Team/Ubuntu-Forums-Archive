### Post by Sarisar on 2006-09-25
It is a little tricky (not really hard) but basically you need to make some space to put ubuntu on.  Think of it as a filing cabinet - currently you have all drawers marked as windows.  You need to move any files out from a couple of drawers before you can relabel them as ubuntu.

I'm not sure of any really good partition managers that will let you do that without losing data, so hopefully someone else can suggest one, but basically you need to partition the drive to reduce the windows partition and make an area for ubuntu before you can install it.  You will need to format _some_ of your hard drive though, but as I said you can do it on an unused area.
