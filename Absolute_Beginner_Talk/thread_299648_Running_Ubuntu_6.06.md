### Post by zytsef on 2006-11-14
> **Entase said:**
> 
okay, I need a straight yes or no anwser, will my computer run ubuntu, and will it run it strait off the CD, ( Live CD)? 

Ok, this isn't going to be exactly a yes/no answer for you but it's close:
If you have the Desktop CD (which it sounds like you do) it will probably run on your computer. It will not run well, and gnome (the graphical part) may not run at all.

If you download the server disc it will require you to install packages (programs and groups of programs) individually. It will allow you to install low resource programs, even install stuff off the internet. An easier option will be to install [Xubuntu]("http://cdimage.ubuntu.com/xubuntu/releases/6.06.1/release.1/") (which may or may not work well) or [fluxbuntu]("http://home.ralree.info/fluxbuntu/rev2") (which will almost definately work).

Bought it off ebay when it's available for free? That's unfortunate. I think you got taken for a ride. Anyway, search the forums and the wiki for info about downloading and burning an iso image to a CD. Hope this helps. Also, that's a really, really small hard drive. The desktop cd will definately not work if you try to do a standard installation. You won't be able to have very many programs on your machine at all. Trying upgrading to at least a 2GB drive; should be pretty cheap.
