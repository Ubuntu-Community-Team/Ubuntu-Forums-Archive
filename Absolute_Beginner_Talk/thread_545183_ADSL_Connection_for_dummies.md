### Post by ayenack on 2007-09-07
Type this in terminal.

sudo pppoeconf

This will open the config window that lets you set up an adsl connection. For a little more info have a look at [this]("https://help.ubuntu.com/community/ADSLPPPoE"). It's pretty straight forward.

Best of luck. ayenack.
