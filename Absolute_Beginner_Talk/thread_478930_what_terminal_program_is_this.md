### Post by RomeReactor on 2007-06-19
Hi. That is indeed Konsole, running [Midnight Commander]("http://www.ibiblio.org/mc/"), a command line file manager. The picture you posted shows a file being edited in Midnight Commander's [integrated text editor]("http://www.ibiblio.org/mc/images/mc-edit-perl.png"). That is the default color scheme for the whole program; you can find it through Synaptic (search for **mc**, or to install from the terminal:
```
sudo apt-get install mc
```
