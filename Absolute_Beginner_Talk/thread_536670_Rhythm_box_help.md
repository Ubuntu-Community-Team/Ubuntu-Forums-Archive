### Post by benhall on 2007-08-28
To add album art you can do several things- the first thing I would do is to go to the Edit menu and click on plugins. In the dialog box which comes up, tick the checkbox by "Cover Art" to enable searching for album art if it isn't already checked. This should make rhythmbox deal with cover art, either by looking for jpg or png files with the name cover (ie cover.jpg) in the folder with the music, or by downloading album art from Amazon if it can't find the art in the folder. If it cannot find the art, you can add it manually using a jpg or png either by renaming it cover.png or cover.jpg and copying it to the music folder.

I'm not sure that you can add your playlists to your ipod in rhythmbox- I haven't used this feature, but I think that it can handle simple playlists if you drag them from your library onto your ipod, but it cannot handle smart playlists (i.e. any track by Bon Jovi released in the 80's). GTKPod is an excellent program- I think it can copy anything, including videos and photos, but it is less easy to use (in my opinion).
