### Post by mukiex on 2007-05-29
You might need to compile the latest linux-wacom BETA driver.

This should do it for you :

sudo apt-get build-dep wacom-tools
sudo apt-get install tcl8.4 tk8.4 build-essential

wget [http://internap.dl.sourceforge.net/sourceforge/linuxwacom/linuxwacom-0.7.7-9.tar.bz2](http://internap.dl.sourceforge.net/sourceforge/linuxwacom/linuxwacom-0.7.7-9.tar.bz2)

tar xvf linuxwacom-0.7.7-9.tar.bz2

cd linuxwacom-0.7.7-9

./configure --enable-wacom --enable-wacomdrv

make

sudo ./prebuild/install

#tricky part, I'm assuming a default Ubuntu Feisty install :

sudo cp src/2.6.19/wacom.ko /lib/modules/2.6.20-15-generic/kernel/drivers/usb/input/wacom.ko

# Props if someone can make a script that does all this, or even better, using stuff like 'uname -r' for seamless integration :3
