### Post by floke on 2007-04-29
Don't worry about uninstalling XP.
Just put the LiveCD in the drive. Restart the PC.
Select 'start or install' from the menu.
Double click the install icon on the desktop when it boots up.
Follow the instructions on the screen.

This assumes you have a LiveCD of ubuntu to use. If not then you need to download and burn one from the website. It also assumes that you want to zap XP altogether, which is a good thing.
