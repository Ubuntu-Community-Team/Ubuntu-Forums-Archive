### Post by teamkiller87 on 2008-03-14
Concerning your bottom panel issue. You can create a new one by right-clicking on the one at the top and selecting Add Panel. To enable the windows list, right click on the newly created panel and select Add to Panel. You should find an option Window List.

About the so-called "burn-in" window issue. I experienced the same thing when I first started using Linux... Ended up I had to re-install video drivers. Don't take my word for it though...
