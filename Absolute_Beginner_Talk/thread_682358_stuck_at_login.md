### Post by amingv on 2008-01-30
You can go into recovery mode (spam the ESC key before ubuntu boots) and try:

```
passwd username
```

Set the password to whatever you want (and I dearly recommend to use a secure one).
Also don't be fooling around in recovery mode, that's a full root session.
