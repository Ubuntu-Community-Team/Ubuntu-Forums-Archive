### Post by tompickles on 2007-05-17
Bit torrent basically allows you to download bits of the same file from tons of different machines (P2P) across the world. Once you have finished downloading completely, your box automatically makes available to the rest of the people who are logged on to the torrent file the download you want. Ubuntu  i beleve has a bit torrent app alreayd built into it, and when you download a .torrent file and open it, it starts to download in this way. firfox will download all other files normally - ie one connection to a server.

### Post by mikewhatever on 2007-05-17
Bittorrent is one of the many p2p (peer to peer) file sharing networks.
[http://en.wikipedia.org/wiki/P2p](http://en.wikipedia.org/wiki/P2p)
Ubuntu 6.10 has a bittorrent client installed, but it is very basic. Lots of better options are in the repositories, such as bittornado.
Firefox is a web browser. It does not have a bittorrent client integrated, but opera browser does.
