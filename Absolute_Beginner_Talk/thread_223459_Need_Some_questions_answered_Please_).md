You don't need Partition Magic, but there have been some reports that the Desktop CD's new graphical installer is a little buggy sometimes. To be safe, I'd use the Alternate CD, even though it doesn't look as pretty.

Guide for the Alternate CD:
[http://users.bigpond.net.au/hermanzone](http://users.bigpond.net.au/hermanzone)
