### Post by camilluk on 2006-11-20
From newbie to newbie...

[https://help.ubuntu.com/community/](https://help.ubuntu.com/community/)

That's your starting point.

I've done a few installations so far, and I knew nothing about Linux when I did the first. What I found is that unless you have weird hardware, it should all be straightforward. The installer will take care of itself.

If you do have less usual configurations, in my case they were about larger than normal screen resolution, bluetooth, cellphone dial-up etc., it can still be done... it just takes a few how-to's and a bit of patience. This forum and the community documentation are extremely helpful. So far, I eventually figured out all that I needed to know, and this linux thing is just growing on me, including the community experience.

### Post by steve.horsley on 2006-11-20
I advise using the "alternate" installer rather than the live CD installer. Because it runs in text mode, it has less dependency on getting the video right straight off, and has less memory requirements. The questions are the same for a basic install like yours (the alternate installer has more options for advanced installers).
