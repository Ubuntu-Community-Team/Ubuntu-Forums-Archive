### Post by PeterJS on 2008-02-28
Under a dual boot only one OS can be loaded at a time. There are methods to run virtualization software based off of real disks (rather than images) but that still leaves you with on OS as the host and one as a virtual guest. Running windows as the guest OS for playing games isn't going to work because 3d acceleration doesn't virutalize, so if you ran windows as the guest OS it would have a generic virtual video card with no 3D support.
