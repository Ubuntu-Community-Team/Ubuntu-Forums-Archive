### Post by orb9220 on 2006-08-28
What he means of booting up is you have to set the cdrom drive as first boot device in the bios.

Then the cd should boot up to what is called the liveCd enviroment where you will see a install icon on the desktop. But before installing you have to make decisions like to I want to keep my windows. Or do I want to just install and have ubuntu on this system.

When in the liveCD do I have sound which you should here on booting into the desktop. Can I browse the internet by clicking on the blue world icon which is Firefox web browser and go to google or another web site?

You can test drive ubuntu without installing first.

Hope this helps.
