### Post by BatPenguin on 2007-08-07
> **bullgr said:**
> for the ProEngineer CAD software, if you have a dual core cpu mashine, you can run winblows from vmware server.
it's really fast and stable (if you have a dual core cpu mashine). 

I'd just like to point out that a dual core is not needed for VMWare. Although I'm sure a dual core is very good for that and fast, like you said, but it's not a requirement - Vmware works fine with an older single core CPU too. I run Vmware Server with a lowly AMD Athlon 2800XP (2.1 Ghz or something). As far as I know, Vmware needs plenty of RAM (enough for both systems), the cpu doesn't have to be dual core. I have 2 gigs of RAM and Vmware works great for me...I use to play poker under WinXP and my machine still has no problems doing its other things (browsing, playing music, recording shows under mythtv etc.) at the same time.

So yes, I'd recommend vmware. Unfortunately 3D games don't work. But that might change too, there's experimental support for DX 8 I believe, hopefully it'll work soon with DX 9. I do use Cedega, great for Civ IV but not so great for many others. Battlefields work but less than great, much slower than in Windows. I still dual-boot for intensely graphical games like Oblivion and the Battlefields, FEAR etc. Hopefully in a year or two we can all be Windows-free :)
