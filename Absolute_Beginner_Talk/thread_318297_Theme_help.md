### Post by 23meg on 2006-12-13
> I want to get some more themes for Ubuntu. I'm using Ubuntu 6.06 and running Gnome. What should I look for when I go to get themes? I downloaded a few today, but they didn't work.

Metacity themes change the appearance of your window borders, GTK themes change how widgets look, and icon themes change the icons. You can drag and drop themes on the theme manager (System / Preferences / Theme) to install them. The two best known sources are: 

[http://gnome-look.org](http://gnome-look.org)
[http://art.gnome.org](http://art.gnome.org)

> is there any way I can increse the font size in the windows? It is so small I can barely read it.
System / Preferences / Font

### Post by d3v1ant_0n3 on 2006-12-15
If drag/dropping the file doesn't work, unpack the file (right-click over it, and 'Extract here'). Then open Nautilus, and go to your home directory. Select 'View' and then 'show hidden files'

There should be a folder named '.themes' (the . means its a hidden folder)-you can copy the folder you extracted into here and it should now appear in the themes menu.

### Post by igknighted on 2006-12-15
Make sure you have the right themes... GTK 2.x themes and metacity themes are what you want.  Also, I wouldn't recommend the extract and drag/drop method... try using the install theme button if drag/drop doesn't work.  If neither of these work you either have a corrupt file or the theme is the wrong kind.
