### Post by dstew on 2007-05-22
It sounds like you have a display problem. There can be several things that cause this. It can be an incorrect display driver, or an incorrect configuration. To work on this problem you will need to use the command line. You can get to the command line by pressing Ctrl-Alt-F1 if the machine is running, or by booting from Recovery mode. Once there, we can examine your system and try to see what the problem might be. To start, post the result of the command```
lspci -l
```That will tell us what video card is installed. The other thing to do is look at your /etc/X11/xorg.conf file to see what driver it is using. It may not be the best available driver for your display, because Ubuntu only ships with open-source drivers. You will probably need to install a closed-source driver to get the display working properly.

I don't see what lilo would do, it is only a boot loader like grub.

### Post by ugm6hr on 2007-05-23
On my Dell Inspiron5000e, I had to edit the Monitor section in xorg.conf, because without this edit, the screen was split into 3 sections, with random gaps.  I think you can just try increasing the HorizSync /VertRefresh ranges, unless you know the manufacturers recommended settings. 

Section "Monitor"
	Identifier	"Generic Monitor"
#	Option		"DPMS"
	HorizSync 28-90
VertRefresh 50-160
EndSection
