### Post by nexu56 on 2007-07-28
I'm not sure if Ebox will have the functionality to set up your PC as a router (i.e. set up NAT and DHCP)

However you might want to have a look at IPCop. It installs a very minimal linux kernel so it is extremely fast and works well on old hardware. It has an easy use web interface and loads of features including firewall, VPN, traffic monitoring graphs, web proxy, and there are plugins such as CopFilter available that let you do other cool stuff like Content Filtering, Antivirus, Ad-blocking and spam filtering.

IPCop: [http://www.ipcop.org/](http://www.ipcop.org/)
