### Post by mikewhatever on 2007-04-18
You can not install programs while running the Lice cd. First install Ubuntu on the HDD, then install the programs you want.
A faulty memory stick should be changed. It will probable make Ubuntu installation impossible or unstable. I am not familiar with that particular error.
HDD partitions are not mounted, so as to prevent a user to damage the data on them by accident.
You do not need to format everything to install. Find a good tutorial to help you before the installation.
An article on linux file system [http://www.freeos.com/articles/3102/](http://www.freeos.com/articles/3102/)
