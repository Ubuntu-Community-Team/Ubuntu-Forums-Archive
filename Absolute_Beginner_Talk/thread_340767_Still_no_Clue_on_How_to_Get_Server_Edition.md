### Post by kebes on 2007-01-17
It's actually not that obvious (but then again most people don't need the server edition). On the Ubuntu download page:

[http://www.ubuntu.com/products/GetUbuntu/download?action=show&redirect=download](http://www.ubuntu.com/products/GetUbuntu/download?action=show&redirect=download)

Pick your mirror location from the list, and then pick "other installation options"...

This brings you to a page ([e.g. this one]("http://ftp.wayne.edu/linux_distributions/ubuntu/edgy/")) which lists all the architectures (x86, AMD64, PPC) and all the types (Desktop, Server, Alternate).

Hope that helps...
