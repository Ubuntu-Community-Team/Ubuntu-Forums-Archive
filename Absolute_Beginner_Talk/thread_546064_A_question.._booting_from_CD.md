### Post by -SpaZ on 2007-09-08
I don't know if you can have it ask you to boot Ubuntu or Kubuntu, but you can install KDE on an Ubuntu machine with ```
sudo apt-get install kubuntu-desktop
```

After you do that on a fresh install of Ubuntu you can use [Reconstructor]("http://reconstructor.aperantis.com/") to make your own custom *buntu with KDE already installed.

I know it's a long process, but it should work pretty well.
