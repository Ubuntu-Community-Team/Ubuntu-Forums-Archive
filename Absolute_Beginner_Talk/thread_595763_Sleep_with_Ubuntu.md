### Post by Bigbird999 on 2007-11-28
About a zillion threads on this issue with a zillion different solutions that work for some, but not all.  It seems very system specific (laptop model, video card and wireless card all seem to play a role) Tried everything and the best I can get is Sleep to work only when clicking on the Logout icon and selecting Suspend in the menu.  If I push the keyboard Sleep/suspend button, set power manager to sleep after a certain time or set the lid closing button to Sleep,  it won't resume without a hard shutdown (remove battery)  For some users it works fine, some it never works and some partly work like me.  

From what I can see, the Ubuntu guys don't even respond to these requests for help. Not a single response to my post and I have seen many others with no response.  If I were you, I would title a new post "Help with Laptop_modelxxx won't Suspend" or similar and hope that somebody with exactly the same model can get you going.  I question if anything will even be done on this issue before the next release in April/May.
