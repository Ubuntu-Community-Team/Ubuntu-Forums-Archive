### Post by deNoobius on 2005-12-08
Well, one possibility is that the computer's BIOS is not set to boot first from the CD drive.  When the computer starts, there should be a message that comes up early in the Windows boot process that says something like "Press DEL to enter setup."  (it may be another key).  So do that, and there should be a way to set the boot order.  Make sure the CD drive is first, then try again.

If that doesn't solve the problem, re-post.  It may be that something was wrong with the way the CD was burned.
