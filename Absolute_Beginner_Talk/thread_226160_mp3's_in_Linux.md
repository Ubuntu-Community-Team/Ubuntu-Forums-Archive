### Post by Indras on 2006-07-30
Get Automatix:
[http://www.getautomatix.com/](http://www.getautomatix.com/)

Install the AUD-DVD and Multimedia Codecs packages from within Automatix, you should then be able to play just about any multimedia file using Totem (.mov, .rm, .mp3, .avi, .mpeg, etc)
