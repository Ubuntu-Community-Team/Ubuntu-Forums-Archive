For a start you could read over this page on the wiki.  The information that you want is probably at the bottom of the page.

[https://wiki.ubuntu.com/DialupModemHowto](https://wiki.ubuntu.com/DialupModemHowto)

You would need to check your /etc/wvdial.conf for the X3 initialisation string most likely for problems with no dialtone.  As for the carrier problem, there are some settings you can change using the pppconfig command.  I would read over the guide and see if you can find some clue to guide you anyway.
