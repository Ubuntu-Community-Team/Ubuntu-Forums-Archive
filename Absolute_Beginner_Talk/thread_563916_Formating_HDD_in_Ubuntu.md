### Post by candtalan on 2007-09-30
ubuntu includes parted, or at least the live cd does and anyway it is easy to install from the repositories (synaptic) I think.

Using something like parted or gparted whatever, you can do many things including format a partition. If the partition is filling a hard drive completely up then you can consider it to be formatting the drive in a general sense, although stricltly speaking it is the partition being formatted I believe.

However - an important point - you will not be able to do this if you are using the partition before you start to do this work. In other words, the partition must not be 'mounted'.

You also cannot do much if anything at all with any partition you are running as part of ubuntu.
