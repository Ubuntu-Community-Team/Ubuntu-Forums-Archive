### Post by nothingspecial on 2008-02-28
Beginner myself but,
 Try looking for things in applications - add remove first.
 Make sure you`ve made the show box (top right in add/remove) say all available applications.
 If that fails look in synaptic (system - administration - synaptic package manager)

Also make sure you have universe, multiverse and restricted ticked in system-administration-software sources.

Anything you download in this way will be installed automatically. 
Compiz is there.:)
