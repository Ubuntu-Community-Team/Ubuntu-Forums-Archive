### Post by freebird54 on 2007-06-04
A number of possibiolities come to mind.  One is that you have it plugged into the wrong spot.  There are different 'levels' of output available (depends on sound card) and putting headphones into a speaker output can be unpleasant.

Another thing that can happen (related) is the 'gain' is too high.  In Ubuntu you can set the level of the 'drive' to be lower using alsamixer (try 70%), not sure how you fix Windows - probably a setting in the card's own drivers.

A third possibility - if the annoying sound is a hum - is that you;re getting 60Hz (cycle) hum coming through your system from the power lines.  (Of course, elsewhere this can be 50Hz hum).  In this case, you can try placing tinfoil between the most likely source of trouble and the headphone, or wrapping the connectors in tinfoil - or getting better headphones.

Hope one of these thoughts helps...
