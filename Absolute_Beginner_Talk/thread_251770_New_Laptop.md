### Post by disciple on 2006-09-05
seems cool.. but if you plan on using it for games, forget it.. it uses an intel graphics chipset.. which is more than fine if you're just gonna be listenin to music , writing reports and sending email..but not for running the latest games at any decent frame rate.
you'll want something from ati or nvidia for that 

and it's a bit heavy.. 6.7 pounds.. which, again, is fine if you're fine with that.
but it takes a toll on the shoulder

other than that, it has a basic design ( which i like ). 512 ram is a bit pedestrian these days, but you can always buy another stick from a vendor ( not from dell..  try crucial. might save a fair amount of cash)

 luckily, the  512 comes in one stick( as opposed to two sticks, which would occupy both ram expansion slots)

how about waiting till vista comes out, tho? or do you need it now?
