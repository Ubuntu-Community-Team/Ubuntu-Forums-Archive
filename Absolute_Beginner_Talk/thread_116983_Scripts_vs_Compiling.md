A script is code that is generally run by an interpreter, so there is no compilation step.  The rule is not exactly hard and fast-- python is script, but the code gets automatically compiled to bytecode when you run it.  However, it is not translated into machine instructions that your processor can run.  Rather, it is transformed into codes the interpreter can understand for efficiency.
