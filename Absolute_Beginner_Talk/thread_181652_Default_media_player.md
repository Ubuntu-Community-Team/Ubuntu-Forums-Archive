I downloaded everything for VLC to play most things but, how do i make it my default media player instead of mplayer and totem?

---

### Post by Nonno Bassotto on 2006-05-24
Just right-click the file you want to open (say mickeymouse.avi) and choose "Properties". Under the "Open with" tab you can select the program which will open it by default. You do this once and it applies to all .avi. Then repeat with every file type you want to be opened with vlc (.mpeg, .wmv etc.)
