### Post by an.echte.trilingue on 2006-12-04
It depends what you want.

I use nothing but GNU/Linux because it meshes rather well with the way my mind works, and I have become dependant on some applications that do not exist for windows.

My wife has some specialist linguistic software that does not run right in emulation on linux, so she stays with windows.

In short, Ubuntu includes (for free) programs that will do almost anything you may may have done on windows, but you will need to take the time to learn the new interfaces.  If there is a windows program that you absolutely have to have (most games fall in this category), there is a chance you can use it on linux, but don't bet on it.

Just try it and see if you like it.
