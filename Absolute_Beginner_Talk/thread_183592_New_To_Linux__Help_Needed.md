If you want to try getting an .exe to work, you need Wine (.exe is a Windows extension. In Ubuntu Linux, more often than not you should look for .deb or tar.gz, if you know how to compile software). Else, if you want to install a program, there's an easier way to do that; in your menu: system -> administration -> (Ubuntu) Synaptic or (Kubuntu) Adept. ### Post by brt on 2006-05-28
the easiest way to install software on ubuntu is to use one of the user-friendly gui-interfaces to the package manager like synaptic. 
(you do have an icon in your system menu) 

unlike in windows where you have to look at tucows or google for your needed software, in ubuntu it much easier, as the linux distributions architecture especially debian and ubuntu allows all the availiable software to concentrate on the distributions-servers and mirrors well integrated straight to your operating system.

thats why you should always use e.g. synaptic when looking for software, because you download and and install it with 2 clicks :)

i am using debian/ubuntu for years now and i really very rarely use a manual install by downloading source-code as a .tar.gz file. but these always come with a good README or INSTALL file containing all needed information on how to build and install it.

and you can install rpm's by converting them using the command-line program called "alien" and install them using "dpkg -i blabla.deb". this way the package also appears within synaptic :)
