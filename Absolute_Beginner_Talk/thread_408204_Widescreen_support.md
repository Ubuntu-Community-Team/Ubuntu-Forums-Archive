EDIT:

To lepht - you might want to open an Application->Accessories->Terminal and type the following command into it:

```
cat /etc/X11/xorg.conf
```

then highlight the result, copy it, and paste it into a reply here.  Highlight it in the editor, and click the # in the button bar above to 'put it in a box'.  This will also help to figure out what's up!

Oh - and in case you are going to end up editing it directly, also do this in the terminal (which will copy the file to a backup in case things get worse before they get better).

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup20070413
```

Always good to be prepared....

### Post by spinflick on 2007-04-13
open up a terminal and type

sudo gedit /etc/X11/xorg.conf

put in your password

scroll to the Display section and put in "1440x900" at the beginning of each of the mode lines

save and exit.

control alt backspace

and then log back on
