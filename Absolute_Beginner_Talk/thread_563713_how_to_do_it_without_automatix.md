### Post by mcduck on 2007-09-30
> **QUEEN HIPPOLYTA said:**
> Can anyone list where for find windows fonts and other things that are commonly installed by automatix such as google earth and codecs? I would love to break away from using Automatix once I do a clean install if possible.

The easiest way to get most of that stuff is to use the Medibuntu repository, and then just install everything through the package management.

[http://medibuntu.sos-sts.com/]("http://medibuntu.sos-sts.com/")

Windows fonts are available in Ubuntu's own repositories. Same goes for Flash etc, and if you want the ubuntu-restricted-extras -package will install most of the included non-free stuff for you (Of course you can just install things you want individually if you don't need everything that the ubuntu-restricted-extras contains).
