### Post by John.Michael.Kane on 2007-07-02
Open synaptic

Search for each file you want to roll back 

Highlight the package. you will have to do each package in question one by one.

Next click on package in the menu bar at the top.

Select force version a option box will pop up, and give a list of versions you can back down to. click force version.

Last step select select lock version this will prevent you from updating that particular package till the fix is release or problem is sorted out
