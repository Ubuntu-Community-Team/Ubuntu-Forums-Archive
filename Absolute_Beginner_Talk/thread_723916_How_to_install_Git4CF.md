I'm not sure if it's wise to use some 3rd party application to update your Compiz, since there's a version of it that is optimized for your version of ubuntu in the repositories. Check your system -> administration -> software sources and enable all sources, and remove the cd-rom. This way you get the latest updates for your applications.
