### Post by aysiu on 2008-03-25
> **DarkSaga70 said:**
> I was just dragging the download link into the apearance preference window like my brother told me to do. So do I need to download it and then do something else ?
Yes.

Forget about dragging the download link.

*Click* the download link, which should download the theme to your desktop.

Then open your file browser (Places > Desktop) and keep that window open (in the window, you should see the theme you downloaded, and it'll have a .tar.gz extension--do not double-click the file).

Go to System > Preferences > Appearance and the Gnome Appearance Properties window should open.

Drag the .tar.gz file from the file browser window to the Appearance Properties window and it should install.
