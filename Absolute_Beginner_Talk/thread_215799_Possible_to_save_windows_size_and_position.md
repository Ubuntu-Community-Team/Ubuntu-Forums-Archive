### Post by gThree on 2006-07-14
Depending on how you've configured windows, right-click on title portion of window frame or select the window menu (on top far left by default ... I think).  You should be able to figure it out from there.  I'm not in Ubuntu right now, but in Kubuntu it's:
```
Advanced >> Special Application Settings >> Geometry Tab >> Position = Remember.
```
Hopefully Ubuntu is about the same.
