### Post by jimbren on 2007-02-23
You can't really "uninstall," but you can do a dual-boot configuration so you can preserve Windows if you want to.  You should also keep in mind that you can run Ubuntu from the live CD without touching your hard drive to test it out.  You can also install software while and try it out when in the live distribution, just keep in mind that anything you isntall will be lost when you reboot out of the live CD, and that installing a lot of software will slow the system down, because it all sits in RAM.

If you decide to do a dual boot configuration, the installer is intelligent enough to know how to do this for you.  It is *very important *that you defragment your hard drive from Windows before doing a dual-boot.

But if you just say forget Windows and do a full installation of Ubuntu, give yourself a week-long learning curve, spending some time here on the forums, and you'll never want to go back.

Good luck, and let us know how it goes.
