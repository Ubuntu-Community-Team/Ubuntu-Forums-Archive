Welcome to the wonderful world of Linux, Open Source and finally, Ubuntu!

Xandros is a fine distribution, and it has a number of similarities to Ubuntu and I would say it's up there with the best of them (i'm not the only one - check [http://www.distrowatch.com]("http://www.distrowatch.com")

However, for absolute click and play pleasure I must say that Ubuntu almost hits the nail on the head every time. I've never had to set up any hardware, almost all software can be clicked and installed through the Synaptic Package Manager, and there is lots of great support on places like here, Ubuntuforums, and my site [http://www.linuxnewbieguide.org]("http://www.linuxnewbieguide.org"), so I would say it has the edge, yes - Do try it, I really do think you'll like it.

If you end up preferring KDE over Gnome as well, don't despair - simply install kubuntu-desktop from synaptic package manager, and you can have KDE as well!

### Post by AndyCooll on 2006-01-14
The reason that you won't get folks on these boards slagging Xandros off is that we all recognise that each distro has its strengths and weaknesses. And the best distro really is the one that suits your personal needs. Some prefer prefer Xandros because it has Windoze similarities, others prefer SUSE because it installs loads of software choices from the start, others prefer the clean install of Slackware etc etc. Obviously on these boards we prefer Ubuntu!

If you have the ability, I'd download the Ubuntu Live CD stick that in your CD drive (either on your Linux box or your Windoze PC) and have a look at Ubuntu without actually changing any of your current settings.

If you really like it then install Ubuntu, if having tested it you prefer Xandros then stick to that.
