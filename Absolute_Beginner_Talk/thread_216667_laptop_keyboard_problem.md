### Post by ubumebme on 2008-03-20
I figured it out... I have an HP Pavilion zv6000 and had the worst problems with the keyboard.  I would type things in and the pages in Firefox would scroll down and do all kinds of weird stuff.  It was a real easy fix.  Go to System > Preferences > Keyboard > and select the Layouts tab.  In the keyboard model section select the model most appropriate to your computer (in my case I chose the Hewlett-Packard Pavilion ZT1XX).

On a side note, this same laptop had a problem with slow boots and wasn't able to load the ubuntu splash screen.  Those were easy fixes as well (found the solutions here in ubuntu forums).  All in all Gutsy now works fantastic.

One last thing, disable the trackpad using the button right above the actual pad to further prevent keyboard problems when typing in a lot of text.  I noticed that sometimes when I type something in, my right thumb will accidentally touch the pad and cause the pointer to move or scroll.  I kind of wish they didn't have that scroll section of the track pad in such a bad spot.
