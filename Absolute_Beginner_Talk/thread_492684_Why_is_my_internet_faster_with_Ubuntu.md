### Post by coolen on 2007-07-06
> **lee292 said:**
> Just wondering... Could it be because Linux is very similar to Unix, which the Internet uses, or at least it did in the early days?

The Internet doesn't use Unix, specifically. It uses a standard to which any system can comply (and if it wants to survive, it damn well better :)).
The Internet is "run" by a series of servers and routers. Most of these are running Unix-like OSs, such as Linux, BSD, and Solaris, and a fair chunk uses Microsoft solutions (although not nearly as many as comprise the client base).
The purpose of the standard is to level the playing field. The Internet is the one place where it shouldn't matter at all what system you're running, as far as setting up connections goes. In practice, different systems use different methods to achieve these ends, and some work better than others.
In short, it doesn't matter if you run a *nix system. What matters is how smart you build your networking software.
