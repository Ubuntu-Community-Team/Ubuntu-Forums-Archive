### Post by l951b951 on 2007-05-06
> **matty14 said:**
> can i have the link to the download for the ndiswrapper for ubuntu please

Ubuntu is different than windows:  lots of your software can be found through Synaptic Package Manager, instead of downloading from a website.  Synaptic will go to trusted (as defined by you) servers and download the package (software) and install it.  

(I'm doing this part on a Windows Machine by Memory, so you might have to poke around a bit to find it)
Go to System --> Administration --> Synaptic Package Manger

then do a search for ndiswrapper.  

Also, if you're looking for a walkthrough on any topic (ndiswrapper, Synaptic, Belkin USB) do a search through the forums.  It has an amazing wealth of knowledge.  

As an aside, I had a wireless card go bad and bought a belkin notebook adapter with the Atheros Chipset.  It worked out of the box after months of wrestling with ndiswrapper.
