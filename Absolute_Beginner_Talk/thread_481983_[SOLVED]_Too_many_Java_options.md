### Post by Inxsible on 2007-06-23
If you do not need all the previous versions of java you can safely remove the java 5's and 1.4. Go to Synaptic and search for Java. Search it only in the 'Installed' section. That should list all the java jre's installed in your system. Keep the latest one - 6 and remove the 1.4 and 5 if you like

### Post by Inxsible on 2007-06-23
Java jres are always backward compatible. So unless you are coding java programs specific to that particular JRE version, you can make do with only the latest JRE version.
