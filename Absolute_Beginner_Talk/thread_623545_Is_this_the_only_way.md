### Post by Slakker on 2007-11-26
Agreed.  It may seem like a hassle, but in my opinion it's far easier to type in
```
sudo apt-get install sample
```
than it is to go [www.sample.com](www.sample.com), browse to the downloads section, download the file, wait for the file to download, run or extract the downloaded file, mess with the installation wizard, and finally get it installed.  With a terminal command, you can do all of it hands free.

I know it's a little daunting at first, maybe even a little scary, but you'll get the hang of the Linux terminal sooner or later.
