### Post by RomeReactor on 2007-03-07
Hi. Do you mean the default player in firefox? If so, close Firefox, then open Synaptic and do a search for **mozilla** and uninstall any other **video** plugin you have installed there (totem, vlc, helix, kaffeine, etc.). Then restart Firefox.

after he/she removes those, they will also have to make sure they have installed and configured mplayerplug-in before mplayer will be the default embedded player for firefox.

also if you want mplayer to be the default player for certain types of media stored on your computer, you can right click on the file type and set mplayer as the default.

if you don't like the player window embedded in firefox, you can also set defaults in firefox by going to Edit/Options/Content/Manage and set the media file types  to open with 
/usr/bin/mplayer then video will open in an external mplayer window.
