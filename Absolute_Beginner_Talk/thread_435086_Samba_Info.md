### Post by Nythain on 2007-05-06
its mainly used to browse windows share (the client side of it) and set up shares for windows users to see (the server/daemon side of it)... in a nutshell if you have a network with windows and linux machines, samba is your solution

its located in the repos... sudo aptitude search samba, if you want to install sudo apt-get install samba... as for setting it up... there are many good guides on these forums and the rest of the net
