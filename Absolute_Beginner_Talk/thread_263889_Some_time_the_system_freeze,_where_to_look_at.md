### Post by seshomaru samma on 2006-09-23

you can bind it to a keyboard combination (like ctrl alt or something) 
so you can bring it upwhenever there is a problem
another option is to bind 'xkill' to a keyboard combination 
xkill is an application that allows you to instantly terminate any programme simply by clicking on it
