### Post by blendmaster on 2006-10-12
Well, in my experience, users really don't know how the bittorrent client will work with their computers. If you have another OS and it works (Windows, and I mean on the same computer), then it could be either your client (which is unlikely, but possible), or your connection settings.

If you're using dial-up, the connection could be so slow that it just "gives up".

Download bittornado (if not using it already), as it seems to be the best client, then go [here]("http://www.bittornado.com/faq.html"), as you might be able to solve your problem using that guide.
