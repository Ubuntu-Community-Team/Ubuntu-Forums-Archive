### Post by PmDematagoda on 2008-01-26
A good IRC is XChat for GNOME, it can be installed using:-
```
sudo apt-get install xchat-gnome
```

For making flowcharts you can have a look at OpenOffice which comes pre-installed on Ubuntu and can be found in Applications>Office.
