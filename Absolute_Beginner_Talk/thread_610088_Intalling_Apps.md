### Post by elmosera on 2007-11-11
Like most OSs, Ubuntu installs stuff all over the place. Most applications are installed via Debian package (.deb), and they will then appear in your applications menu. Ubuntu offers lots of Debian packages via Synaptic Package Manager or Add/Remove Programs.

### Post by Palmyra on 2007-11-11
If you're new to Ubuntu, don't worry about this.  If you have a .deb file, just double click it, and install.  If you are using a package manager such as Synaptic, go through the process, but don't worry about where things end up.  You can always uninstall software via Synaptic.

### Post by atlfalcons866 on 2007-11-11
Most files are stored in /usr /lib. /bin is where the execuitbles are. /Home is where the programs settings go. /Var is usually logs like fsck logs system logs.
