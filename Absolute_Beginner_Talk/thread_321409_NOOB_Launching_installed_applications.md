Warning, complete noob question:
When I install new applications, whether from synaptic or binary, I never know how to launch the application afterwards. This has caused many a headache with fglrx-control and other essential apps. ### Post by aysiu on 2006-12-18
Tab completion in the terminal (type partial name of application and hit **Tab** twice) or Alt-F2 and start typing the name of the application--in Gnome, you should get auto-completion.
