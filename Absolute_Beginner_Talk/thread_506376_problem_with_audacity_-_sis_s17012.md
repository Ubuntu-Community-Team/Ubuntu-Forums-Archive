Do you have Jack Running? It seemed to work pretty good for me by installing the meta package ubuntustudio-audio. That package includes everything you will need for an Ubuntu recording studio.

You can read more here: [https://wiki.ubuntu.com/UbuntuStudio/PackageList](https://wiki.ubuntu.com/UbuntuStudio/PackageList) or at [http://ubuntustudio.org/](http://ubuntustudio.org/)
