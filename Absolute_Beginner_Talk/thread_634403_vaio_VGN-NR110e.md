### Post by lazarous64 on 2007-12-12
Hello. I have the same laptop.  I got it at BB on Black Friday.
I also had trouble running the live cd.  At the selection menu that pops up when u put in the cd, choose the OEM install option.
Linux should now boot properly. You can now install to some free space.
I reformatted using the recovery disks and shrunk the vista disk to 40GB.
Then I used 20GB of the remaining free space for linux(linux will automatically use this space so dont worry about it, just use the guided option).
After it installs and reboots, you should either be able to boot into linux(ubuntu) or vista.
Happy Holidays!

### Post by PtPub on 2008-03-16
There is a thread on UbuntuForums explaining exactly how to install Ubuntu 7.10 on Sony VAIO VGN-NR110e. I have curently Windows XP Profesional, and Ubuntu 7.10 running with everything working. Im using GRUB to dual boot XP and linux. 

Ubuntu 7.10 WIFI is  working without a problem using NDIS Wrapper. If you need more information on how to get it all working search for Chalewa's guide here on these forums. That guy really knows what he's talking about and that's what I used to get my lappy working. 

I would recommend NOT deleting or formating your VISTA installation since you will need some drivers from VISTA that will work in XP. Unfortunatelly i tried doing everything by myself at first, and deleted my installation, which made the rest of the setup much longer than it should be...:popcorn:
