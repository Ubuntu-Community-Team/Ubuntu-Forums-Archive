### Post by lamego on 2007-01-26
You should have no problems with the hardware drivers. The major problems with 64bits are the closed sources apps (like flash player) for which there are no 64 bit versions yet.

If you care more about using all the applications that you can, run 32 bits, using 64 bits means you are willing to be a power user and you want to contribute to the 64bits development.

There more 64 bit users the faster companies will care about producing their software for 64 bits.
