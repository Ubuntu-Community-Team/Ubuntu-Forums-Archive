### Post by crimesaucer on 2007-05-02
I had a similar problem for xubuntu a while back, and I pressed alt + F2 and then:

```
xfce-panel &
```

I don't know what the proper gnome code is (probably gnome-panel &), but if you check the forums with the search option, many people have gone through this so just search a bit and you'll find the correct procedure.
