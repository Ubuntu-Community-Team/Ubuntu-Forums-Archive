### Post by Adrian on 2005-12-22
[QUOTE=The Hedgehog]If I do [this]("https://wiki.ubuntu.com//InstallingKDE"),

You can use both Gnome and KDE. Just select which one you want to use from the login screen.

Some people don't like that the KDE programs will show up in the Gnome menus and vice versa. This is easily solved by editing the menus (removing programs from the Gnome menu will not remove them from the KDE menu and vice versa). You can also easily remove the KDE programs from the Gnome menu by using a simple script, as described in this HOWTO:
[http://www.ubuntuforums.org/showthread.php?t=59455](http://www.ubuntuforums.org/showthread.php?t=59455)

The real problem is if you want to uninstall any of the desktops. See these threads for more information:
[http://www.ubuntuforums.org/showthread.php?t=96046](http://www.ubuntuforums.org/showthread.php?t=96046)
[http://www.ubuntuforums.org/showthread.php?t=96048](http://www.ubuntuforums.org/showthread.php?t=96048)
