Try ```
sudo aticonfig --initial --input=/etc/X11/xorg.conf
```If that doesn't work try changing this line```
Section "Device"
Identifier "ATI Technologies, Inc. Radeon Mobility 9100 U3 (R200 IGP)"
Driver "ati"
BusID "PCI:1:5:0"
EndSection

```to this```
Section "Device"
Identifier "ATI Technologies, Inc. Radeon Mobility 9100 U3 (R200 IGP)"
Driver "fglrx"
BusID "PCI:1:5:0"
EndSection

```But back it up first with```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_bak
```
