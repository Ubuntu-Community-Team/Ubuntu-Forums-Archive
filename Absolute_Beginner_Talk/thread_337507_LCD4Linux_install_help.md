### Post by ajmorris on 2007-01-13
go into synaptic and go to settings and then preferences. The first checkbox in that window should be checked. Click OK then go to the package you are trying to run. Select the "installed files" tab and find where the programs exicutible is. (i would assume /usr/bin, but maybe different) Then click the run command (alt+F2) or run a terminal. If the executable is in /usr/bin then just type the executable name. If it is in another folder navigate to the folder then run the executable. :)
