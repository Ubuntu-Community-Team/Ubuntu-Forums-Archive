### Post by kpkeerthi on 2007-12-18
Easytag scales very well if you have huge collection. Its very easy to use. If you are looking to tag same album and artist to a list of files, here is what you need to do:
1. Make sure the files are selected on the left pane. 
2. Enter the Album and Artist name on the right pane. 
3. Select the check box beside the artist and album name field. (You tick the checkbox beside a field if you want that field value to be tagged to multiple files you have selected. Its as simple as that.)
Done.

The are many good alternatives you could try - **ex falso, cowbell, tagtool**. 

There is no need to wine some windows application for mp3 tagging when there are powerfull native counterparts.
