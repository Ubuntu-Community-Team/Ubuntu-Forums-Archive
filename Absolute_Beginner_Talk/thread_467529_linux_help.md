### Post by RomeReactor on 2007-06-07
Hi morphius1, welcome!

Those effects are easily achieved enabling **Desktop Effects** on Ubuntu, which by default uses Compiz, and by also installing **gnome-compiz-manager**. After installing this package, you can find it under "System-->Preferences-->GL Desktop"; there you can set different levels of transparency, make the windows wobble when you move them, enable water effects, etc. You can also install Beryl if you want more variety, though. Just remember to only enable one or the other, not both. The terminal transparency can be set in the terminal's preferences, even if you don't have Compiz or Beryl.
