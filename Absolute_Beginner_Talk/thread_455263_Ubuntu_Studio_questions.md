1. You should be able to just install Studio over Ubuntu, I'm not sure I haven't tried. Either that or you're going to have to use a partitioner on a Live CD. The other option is to keep your Feisty install and "upgrade" to studio. The link in my sig will tell you how to do that.

2. It's pretty much the same just with a different theme and different programmes installed by default. There is one major difference, which is the low-latency kernel, which is for audio work.

3. As far as I'm aware, as long as Vista stays on the partition it's on (my XP had a hissy fit when it went from the 2nd partition to the 1st).

4. I use it and it's pretty damn good. You can get all the programmes on a normal Feisty install (Though some in Studio, like Ardour, is newer) but the theme is really good and the low latency kernel is a godsend.
