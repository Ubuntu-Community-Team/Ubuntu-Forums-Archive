### Post by ArchangelSWAT on 2007-12-24
I also play Assault Cube. The problem with different maps is planned to be solved in the next release of the game, beta 0.94 - but in the current version, to get a map, type "/clearsecuremaps" and then "/getmap". It might be advisable to bind these commands to keys, so that you don't have to type them out every time. For example - type ingame /bind "F1""[clearsecuremaps] [getmap]". Then, when there is a custom map that you don't have, just press F1. Hope that solved your problem ;)
