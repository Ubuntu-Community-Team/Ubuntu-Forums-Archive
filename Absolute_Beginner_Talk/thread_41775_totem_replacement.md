### Post by bored2k on 2005-06-14
0. [http://ubuntuguide.org/#extrarepositories](http://ubuntuguide.org/#extrarepositories)

1. VLC
sudo apt-get install vlc-esd (after its installed, set it up to use ESD sound)

2. GXine
sudo apt-get install gxine

3. Mplayer
sudo apt-get install mplayer-386 mplayer-fonts mozilla-mplayer

4. Totem-xine
sudo apt-get install totem-xine
