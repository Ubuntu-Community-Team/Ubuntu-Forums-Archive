### Post by obrient on 2007-07-28
I found some useful information in an online magazine when I was learning. Grab issue [20]("http://www.tuxmagazine.com/node/1000238") which talks about ls and cd. The mag is not printing anymore, so grab the back issues. 

Also I know that a few helpful things when I first started were finding about man and apropos.  man is a help page so you can type in for example  ```
man cd
``` and it will tell you about the cd command. They are criptic but can help out if you are reading forums and wand to know what something means. Simply hitting ```
q
``` get you out of the page.

Also ```
apropos
``` is a great command. I use it all the time. It basically gives you a list of commands related to a key word. For example: ```
apropos internet
``` lists half a dozen things related to the internet. Not always the best way to find things but I have learned a lot. 

```
ctrl d
``` is a quick way to exit out of the terminal. Not even thinking about it till now go to the Ubuntu wiki they have a page called [Using the terminal]("https://help.ubuntu.com/community/UsingTheTerminal") :) 

Good luck and don't stop learning.
