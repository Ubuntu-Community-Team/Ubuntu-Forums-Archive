### Post by az on 2007-01-09
> **whee said:**
> Aha ok.
I thought maybe some road map has been laid out for LTS releases.

Thanks guys.

There is no set roadmap because no one knows how things will pan out.  For sure, there will be a stable release with set goals every six months.  You can also bet that the next LTS will come out before the current LST is no longer supported.

Other that that, it depends a lot on what happens upstream.  If a few major projects can come up with stable milestone releases at about the same time, that would probably be a good time to aim for an LTS.

Edgy set the groundwork for some big infrastructure changes.  Feisty should showcase some serious bling.  How long it will take after that to put together a solid candidate for an LTS is too far to predict.
