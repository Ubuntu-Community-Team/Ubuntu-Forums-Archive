### Post by laidback on 2007-08-19
I have same problem and posted a thread but no answers.
I found that the cursor focus is on the Chat tab, so go there are kill off cursor blinking on data line by pressing Chat Rooms, if that doesn't work try any other button on Chat tab. Once cursor stops blinking on Chat's data entry line you'll be OK in Search etc. 

Nuisance, it's like it every time I start the program, and sometimes during use.
