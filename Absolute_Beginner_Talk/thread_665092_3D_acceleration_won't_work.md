Ok if you add these to your device section 
Option "AddARGBVisuals" "True"
Option "AddARGBGLXVisuals" "True"
This hopefully will correct the issue. If you can use gedit then you use this command ```
gksudo gedit /etc/X11/xorg.conf
```
