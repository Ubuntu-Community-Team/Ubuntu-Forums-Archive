### Post by LeAstrale on 2008-02-22
in the most cases nothing will happen... if you're burning a dvd i would however hold back on the use of intensive programs (could be wine, cedega, maybe a big pic in gimp) until your done burning. 
This does have a lot to do with your recorder though.. if it has a built-in 32mb cahce of its own (like mine) you wont have trouble doing pretty much anything you want while burning but if you have an older drive that only uses the system ram as a buffer.
