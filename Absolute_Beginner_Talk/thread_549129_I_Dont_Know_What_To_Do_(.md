### Post by AtrejuT on 2007-09-12
Basically what you have to do is just insert the CD into the drive and reboot your computer.
It should automatically run ubuntu for you to try out, without yet deleting anything. Once Ubuntu is running you can install it simply but double-clicking the 'install'-icon on the desktop.

Should your computer not run Ubuntu upon rebooting you have to change the boot order in the BIOS. You can usually access the BIOS by pressing something like 'del', 'F1' or 'F2' when booting the computer. A message on the screen should tell you which button to press.
Once in the BIOS you have to find a place where you can set the order in which devices are searched to boot. Put your CD Drive on the top of that list.

Good Luck, and Welcome to Ubuntu!
