### Post by startherepodcast on 2007-10-07
I am also a noob at running servers, but you might want to put a harddrive somewhere else in trhe house, maybe in a really secure place and back up via lan every few hours so if your pc screws up you can still get it back up pretty fast.

I have no idea about the hardware you should use, as i don´t run small company websites myself, but if this website is not some huge thing that needs loads of computing power a normal home pc should do the trick, you can always scale it up by either clustering two home pcs or if u want it to be more professional a real server that companies sell especially for this purpose and use the home pc for something else
