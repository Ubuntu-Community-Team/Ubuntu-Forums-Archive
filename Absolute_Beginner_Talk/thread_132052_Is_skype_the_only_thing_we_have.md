### Post by kaamos on 2006-02-17
[http://www.openwengo.com/](http://www.openwengo.com/)

Wengo is a cross platform GPL licenced voip app. In my understanding it supports sip so you can use (for example) [ekiga](www.ekiga.org) (the new gnomemeeting) to call your friends that use wengo if want to use a program that integrates better with the gnome desktop.

### Post by theturner on 2006-02-17
Ekiga is excellent. It's easy and it uses SIP, the *VoIP* standard. If you want to call to / be called from real phones, there are many providers for SIP gateways, i.e. Telephone numbers for your VoIP phone.
