### Post by igknighted on 2007-02-14
Dual booting is extremely easy, don't worry about that.  If the Dell warrantee is a big deal, I would call them and inquire.  I think it might invalidate it.  However, as a workaround, I might install another HD (you can get an average size HD for like 20/30 dollars US now).  If you plug in a new HD and install ubuntu to that, you can dual boot ubuntu and XP without fear of damaging your XP install at all.

As for the benefits of Ubuntu... where to start?  It looks way better for one.  If you (like many others) were insulted that MS made XP look like a fisher-price toy then you will like Ubuntu.  Security isn't even comparable.  In XP you need to install firewalls, anti-virus, anti-spyware, etc. and even then (after all the programs are sapping your computers power) you are not as secure online as you are with XP.  Windows was designed as a single use OS, without thought to the internet (the core of windows was designed before most people used the internet).  Unix (and by extension linux and Ubuntu) was designed for networking and networking security.  Price is also a factor.  Linux is free, while Windows Vista + Office + a few other programs can very easily run you over 1,000.  Not only do you get the OS with Ubuntu, but you get an office suite comparable to MS Office 2003, and tons of other free software (music/video players, development tools, email/chat clients, the GIMP, and much much more).  Finally, you get choice.  Linux is all about choice.  You are not entrapped by the OS, you can dictate to it (once you get good enough) how you want things to look, feel and behave on linux.

I strongly recommend at least downloading the CD and trying it in Live mode and giving it a shot.  Be aware that there is a learning curve.  Things you knew how to do easily in XP will seem hard at first, but in the end it will make sense and you will learn a lot about linux and computers, and probably be a more aware computer user.  Hope you give it a try!
