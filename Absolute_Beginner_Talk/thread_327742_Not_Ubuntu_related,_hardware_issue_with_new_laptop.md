### Post by oilchangeguy on 2006-12-29
what are you trying to do? upgrade the laptop to a duo core? it can't be done with a motherboard designed for single core cpu's. check your computer makers website for any bios updates. to completely flash an older bios with a newer version (still no duo core cpu on a single core mobo) go to the bios makers website, and you can get the latest and greatest bios for a price. and if you've never done a bios flash (and even if you have) there's a good chance you'll kill the computer. so to that end, if it ain't broke don't fix it.
