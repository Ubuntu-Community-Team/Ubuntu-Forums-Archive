### Post by HereInOz on 2006-11-13
To use the existing Windows XP with Ubuntu, you will need to set up a dual boot situation.  You will not be able to access the existing XP installation by using VMWare.

However, should you wish to set up a new installation of XP, in a virtual machine, you will have no problems at all.  Click here:

[http://www.ubuntuforums.org/showthread.php?t=183209](http://www.ubuntuforums.org/showthread.php?t=183209)

for instructions how to do that.
