### Post by Knomefan on 2005-12-22
The recomended and best way to install software is to install it from the repositories.
I just checked and 7zip seems to be in the repositories, so in order to install it, simply open up synaptic and search for 7zip (it's called p7zip). Select it and install it.

You first need to setup universe:
[http://help.ubuntu.com/starterguide/C/faqguide-all.html#addinguniverse](http://help.ubuntu.com/starterguide/C/faqguide-all.html#addinguniverse)
Then you should be able to install it easily:
[http://help.ubuntu.com/starterguide/C/faqguide-all.html#fg-installing-applications](http://help.ubuntu.com/starterguide/C/faqguide-all.html#fg-installing-applications)

Hope this helps.
