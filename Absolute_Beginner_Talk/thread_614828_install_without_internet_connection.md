### Post by ruibernardo on 2007-11-17
Hi jpinks,

you can use [http://nonetdebs.homeip.net](http://nonetdebs.homeip.net) for this. Check how to use it [here]("http://ubuntuforums.org/showpost.php?p=3742162&postcount=11"), search the package you need [here]("http://packages.ubuntu.com") and find out which are the packages you don't have installed on your computer and you need to download on the nonetdebs website (dependencies, etc). 

You can use APTonCD to make them available to your computer like overdrank said, creating an ISO file or a CD with the packages you downloaded, or use a script listed when you ask for packages in nonetdebs website.
