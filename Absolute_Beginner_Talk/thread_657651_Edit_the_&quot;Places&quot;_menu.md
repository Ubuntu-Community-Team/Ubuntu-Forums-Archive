### Post by willmcgree on 2008-01-04
The Places menu mirrors the sidebar in Nautilus, the file manager, so it's possible you have a duplicate Desktop shortcut there.

Open any file manager window and check the sidebar. The default (from top to bottom) looks like this:

* your Home folder
* Desktop
* File System
* Deleted Items
--------
* Documents
* Music
* Pictures
* Video

Hope that helps
