### Post by cskeide on 2006-05-30
When you're browsing you are only selecting a directory. So browse to your selected directory and click "open". Then in the window displaying the icons, you might have to select the directory path in the "address field" and hit enter.

I don't like this behaviour myself. It's not very intuitive. Sorry if my explanation is a bit vague. Hope it helps though.
