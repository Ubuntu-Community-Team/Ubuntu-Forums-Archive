### Post by djf_jeff on 2006-12-06
The one I can thing right off my head now is the problem with 32 bits plugins (like flash, win32codecs, java, etc).

You cannot run these plugins in the 64 bits version of firefox. You can try some hack like nspluginwrapper but it's not considered stable for the moment.

Apart from that, I think it's really a good distribution for a 64 bits platform.
