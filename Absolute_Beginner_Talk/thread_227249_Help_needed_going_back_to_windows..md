### Post by petersjm on 2006-08-01
I can only hazard a guess at why the OP is returning to Windows from Ubuntu - Same thing (if I'm right) happened to me many times in the past: Linux was too new (read: *different*) and I wasn't ready to change. I tried RedHat, Suse, and something else that I can't remember. But then I stumbled across Ubuntu when I wasn't even interested in trying out Linux again, and hey presto, I got hooked almost immediately!

If you want to reinstall Windows and completely remove Ubuntu, make sure you can boot from CD, stick your Windows CD in the drive and reboot. When you get to the part in the installation process about partitions, Use your arrow keys to highlight each partition (one at a time) in the list and select delete (or format) from the options. When you've done that, continue with the installation process and Windows will set up a brand new partition for you and install correctly. Good luck and here's hoping we see you again in the future!
