### Post by hyper_ch on 2007-11-23
(1) Overwrite: That pretty much depends on your install/partitioning selection... you can choose to use the existing partitions for the root, boot, home etc. partition and you can even select if you want to format that. If you have a seperate /home partition you can just select that as /home again but have not format it. So the config files and data files stay the same... but that's manual partitioning....

(2) you can vista make smaller but for that I advice to use the partitioner within vista to shrink that partition.
