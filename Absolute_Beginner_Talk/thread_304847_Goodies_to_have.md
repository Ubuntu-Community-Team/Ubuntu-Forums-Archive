### Post by turbojugend_gr on 2006-11-22
The better routine to follow when you need to install an app is: 
1. search in syanptic and install it using synaptic. 
2. search the official desktop guide and ubuntuguide.org if there is a how-to there 
3. search this forums if there is a how-to 
4. google up for a how-to (likely in psychocats, and similar sites)
5. Visit the app's official site and try making out how to install it (usually reading the Faq, documentation, how-to instal etc).
6. google up for a .deb package (which is usually distro sepicific or at least debian specific).
7. download the tarball (source code) and compile it yourself. (not easy for new users)

I hope this helps.

Deric,

We need to take a step back. 

With windows, you get the core operating system, some fairly useless apps, and that's it. Right? You need to go to Tucows, Beta News, Filehippo, and everywhere else to get what you need to make the system useful.

With Debian type Linux operating sustems, there is a software repository that is one command line away, or via the Synaptic GUI. 

By looking in Synpatic, you can select any number of applications, from Developement tools to Multimedia, and even grab dependencies where they exist. Last number I saw was upwards of 15,000 applications in the repository.

You tell it to install these applications, and go make yourself a cup of coffee, watch the Simpsons, or go to the gym (depends of the speed of your connection, and how much you are installing.)

When you come back, all the applications are installed, and where applicable, they are added to your menu. 

Next time you update, the system will also grab the updates to the applications you have installed.

There are times when you need to go outside the repository to install something: an application that you want may only be available as source, or only for another distribution.

Some good guides for those situations are here:
[http://psychocats.net/ubuntu/installingsoftware](http://psychocats.net/ubuntu/installingsoftware)
[http://monkeyblog.org/ubuntu/installing/](http://monkeyblog.org/ubuntu/installing/)

Hope that helps,
