### Post by varunus on 2005-07-17
Don't install from source unless you have to.  Use synaptic instead.  Open up System->Preferences->Synaptic and search for licq, and install it from there.

Installing things individually is the windows way of doing things, and is far inferior in my opinion.   You'll like ubuntu's one place for everything approach.  Make sure to enable the extra repositories as stated at [www.ubuntuguide.org](www.ubuntuguide.org).

Most things you need should be in there.
