### Post by ACGarland on 2008-04-13
I'm in the process of converting my wife over to Ubuntu (from an old copy of W2K). She used to use an old copy of Print Artist.  I couldn't get it to run under wine. And in VirtualBox it was so old that it wouldn't handle finding data from a shared folder mapped to the Z: drive. (I didn't want to bloat my VM image with megabytes of graphics images.)  I had similar problems with (admittedly older) version of Print Shop.

I finally dug out  a copy of PrintMaster Greeting Cards Deluxe (5,5,0,1202) my sister-in-law had given us. To my amazement, it installed under wine, runs reasonably, and allowed me to load all the CD content wherever I wanted under Ubuntu.

I don't have all the glitches worked out yet (haven't run the print layout wizard to get folded card layouts set up with my printer, program still harasses me about registration).  But based on what I'm seeing so far, I'd say look into PrintMaster as something that can run under wine when others might not.
