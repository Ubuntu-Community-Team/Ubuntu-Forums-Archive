### Post by ramjet_1953 on 2007-07-09
Probably the easiest thing to do is to dual-boot, until you feel comfortable with Ubuntu.

The installer will take you through the steps and providing you take it nice and slow and make sure that you are following the directions, all will go well.

However, before doing this, make sure that you boot into Windows first and run the defrag utility at least twice, before installing Linux.

Another option is to buy another hard drive and if you are using a desktop, use one of those HDD plug-in caddies in a spare drive bay. Of couse, this give you complete sepeartion of operating systems, with no chance of corrupting your Window OS.


If you have a laptop, like myself, you can easily swap hard drives, as most laptops just have a trapdoor underneath and the HDD is usually screwed into a metal caddy.

Yes, you can virtualise Windows under Linux. There are a few options:

1. Wine. This is a compatibility layer and can run quite a few (but not all) Windows applications. They are continually developing it and it is getting better and better.

2. VirtualBox is a virtualisation engine that can be used to install any OS under Linux. I have used this myself in the past for an XP install and it ran all of my applications flawlessly.

3. VMWare is similar to Virtualbox. I have never used it myself, but there are lots of people who love it.

As you can see, there are several options, which is what Linux is all about - freedom to choose.
