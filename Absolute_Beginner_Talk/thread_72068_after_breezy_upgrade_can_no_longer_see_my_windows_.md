### Post by Zwack on 2005-10-15
I have (had) the same problem.  It seemed the fact that I was sharing  a directory (read only) caused issues.  I went into a terminal, typed "smbpasswd" and set a new password for my account.  Then I went into the places->Network Servers window, used my new password to connect to my own share, and saved it...

Now I can access other shares...

I hope that this helps,

Z.
