### Post by Terry of Astoria on 2006-01-08
Maybe you ought to test the RAM in your machine. 
When you boot up the CD type 
memtest
- I don't know if it's on the installer CD but it's on the live CD.
If it's not on the installer, just download the newest ver - google memtest86
to find a tiny iso you can burn to CD containing just Memtest86 - simply boot it up and the test is off and running. I have experienced similar problems to yours and found that my problems arose from bad RAM. By the way Windows will often run fine (or not so fine) while linux refuses to run on a machine w/bad RAM.
