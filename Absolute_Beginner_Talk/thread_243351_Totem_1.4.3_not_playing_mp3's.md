### Post by yager on 2006-08-25
> **steve789 said:**
> now where and how do i get the plugins

When I first started, I had the same issue until I installed Automatix.  Arnieboy has recently moved this project on to its own forum so here is a link to the how to install page.

[http://getautomatix.com/wiki/index.php?title=Installation#Installing_Automatix](http://getautomatix.com/wiki/index.php?title=Installation#Installing_Automatix)

This script has two options to provide all of the codecs that are available for Linux.  Now I only have an occassional issue with not being able to access multi-media files.  When this happens, I ask automatix to re-install the codecs and sometimes this clears up the problem.

If you just want to read about Automatix, go to [http://getautomatix.com](http://getautomatix.com) .  This script file not only installs codecs, but automates the installation of additional applications that fill in all of the "holes" on usability in Ubuntu.
