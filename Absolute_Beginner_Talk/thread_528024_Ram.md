### Post by ugm6hr on 2007-08-17
> **poetofzwan said:**
> Ok I used Wubi and installed Xubuntu and that thing is smokin'!  I notice that alot of software (like open office) is not installed however this isnt a major prob.  I am curious however what drawbacks Xubuntu has over the normal distro of Ubuntu?  It seems to look just as nice and work faster, so why is everyone not using Xubuntu?

You can just add OpenOffice to a Xubuntu install if you want (I have).

Drawbacks - none of any significance to most users.  I have found a few things though:
1. No built-in WPA support via GUI for wifi (I use Wicd)
2. No built-in Network Browsing with Samba shares (I use pyNeighborhood)
3. Big fat panels that don't shrink very well (I moved all stuff apart from Task List to top panel, and shrunk just the bottom panel).

My current laptop would happily run either - but I stick with Xubuntu/XFCE to keep a familiar interface for all of my family's laptops (which I have to install / maintain).
