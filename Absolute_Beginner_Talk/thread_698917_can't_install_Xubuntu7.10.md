### Post by xeth_delta on 2008-02-16
> **3076 said:**
> no, but why, my memory is emough,i don't wanna download it

The reason a previous user suggested trying the alternate cd is not because of lack of memory, but because the graphical installer from the live CD can sometimes have problems with some hardware configurations. I would recommend you to give it a try.

Some more advice. Before burning the installation CD, make sure the MD5 checksum is correct, in other words that the downloaded image is not corrupted.
It is also recommened to burn the installation CD at 4x or less.
For more information see: [https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28burn%29%7C%28](https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28burn%29%7C%28)
