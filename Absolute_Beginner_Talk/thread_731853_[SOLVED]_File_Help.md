### Post by marco123 on 2008-03-22
Well first of all welcome to Ubuntu. :) tar.gz is an archive file kind of like .zip and .rar.  In Ubuntu you just right click on them and select "Extract Here".

As for other file types, there's a good little list here: [http://www.debianhelp.co.uk/fileext.htm](http://www.debianhelp.co.uk/fileext.htm)
