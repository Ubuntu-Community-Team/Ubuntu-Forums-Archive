### Post by jvc26 on 2007-01-21
The issue will probably be RAM - a live cd runs entirely from the cd and your RAM, so by having less RAM your system will be hugely slower. Installing will make the process much faster. You could try the alternate install cd which doesnt require the live function. It uses a text based installer and hence isnt so resource heavy.
