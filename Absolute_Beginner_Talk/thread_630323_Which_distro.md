### Post by Mario247 on 2007-12-03
Hi,

A lot of distros are livecd's so you can try them out without overwriting your existing system.  Also distrowatch have a list of the most popular distros - if you use a popular distro you are more likely to be able to get free internet support than for an obscure distro.  One more factor to consider is the power of your PC - if you are using older hardware you want to go for a lighter distribution.
