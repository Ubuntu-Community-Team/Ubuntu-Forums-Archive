### Post by forrestcupp on 2008-03-26
There should be an installation icon on your Live desktop.  You can install it permanently that way.  In the installation, there is an option to use your free space.  You can tell it how much of the unused space to use and it will create a separate partition to install Ubuntu on.  When you're done, you will have a GRUB boot menu where you can choose if you want to boot to Windows or Ubuntu.
