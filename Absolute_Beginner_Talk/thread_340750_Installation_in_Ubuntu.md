### Post by SunnyRabbiera on 2007-01-17
remember:
windows .exe files will not work by default...
Package installation in ubuntu is very easy though, you need files with .deb extentions but they will install new packages with little effort most of the time.
Now with open office, dont bother getting the latest version as Open office does not support .debs... just stick with the one that is included in the ubuntu pool.
there is a program in ubuntu that is dead easy to learn, it is called synaptic package manager that will help you grab a vast majority of programs like firefox, mozilla, open office, gaim that sort of thing...
dont expect internet exploreror or other windows programs, keep in mind that windows and linux are two different OS's and what works in one may not work in another.
Just keep to it though and linux can become your best friend.
