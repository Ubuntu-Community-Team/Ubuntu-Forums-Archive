### Post by mcduck on 2007-02-20
Just like in any other OS, if application isn't made to use multi-core setups then it will only run on one core. Of course even then you get the benefit of having one core free for all other applications and your system.

So, Linux makes as good use of dual core as is possible. Why wouldn't it, when even some of the fastest super computers run Linux and you can be sure they have more than one CPU ;)

Honestly, I wouldn't even think about buying a single-core CPU any more.. Dual core makes running many apps at the same time so much nicer.
