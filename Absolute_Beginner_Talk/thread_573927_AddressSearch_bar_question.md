### Post by FreewheelinFrank on 2007-10-12
Hi,

Been using Ubuntu/Kubuntu for a few weeks now (settled on Ubuntu) and I have a little niggle.

In Windows if I click on the address or search bar in any web browser, the contents of the bar (if any) is automatically selected, so hitting backspace will delete the previous entry and I can type my new entry.

In Ubuntu, clicking the bar only places the cursor in the bar and I need to right-click and select all>backspace or select 'clear' from the menu if present.

Is it possible to change a setting somewhere so that clicking the address or search bar will select the contents of the bar?

### Post by swisscow on 2007-10-12
Yes - 2 ways

One just click on the little icon on the left of the address - that selects it all

The other os to type about:config in the address bar

Then find browser.urlbar.clickSelectsAll and double click on the value to make it true

The about:config change now means clicking on the address bar* and* the search bar now highlights existing contents!
