### Post by dalee on 2006-07-02
Hi,

First, I'd check to see if you are having a firewall problem, (if you've set one up). Second make sure you have permission to access the network, (that one has got me a couple of times). Look in the Users and Groups in Sytem- Admin- Users and Groups.

If those fail you, then boot up with your Ubuntu CD, and test to see if your internet connection is working. Then go to System - Admin - Networking, the write down the info you see under all the tabs. Be sure to look at the Properties tab too. Chances are pretty good that you need to enable DHCP. Then boot back into to your harddrive install, open the Networking there and compare. Make them look the same. It should then work.

If not, then we need more info:)
