### Post by p_quarles on 2008-02-19
Quanta and Aptana are the most similar to Dreamweaver in terms of number of features, but neither of them will have the same kind of WYSIWYG editor that DW has. 

From the perspective of writing pages to W3C standards, however, this is a good thing. The best WYSIWYG editor in the world will not produce optimized HTML code.

### Post by wolfear on 2008-02-19
I agree with Aysiu on GIMP and Inkscape. You might find more browsing the repos.
If you are using Synaptic, try looking for the Ubutunu Studio metas. That might give some ideas of what is available.

As far a Dreamweaver, all I can say is "Why"?
If you can't be troubled to learn and really need the WYSIWYG, Quanta is decent but you'll still have to do a bit of coding. Quanta does have some great capacity for using ftp to work remotely and some of the CSS tools are pretty neat. I use Quanta quite often if I need to throw together an HTML page or find a missed bracket in a php page in a hurry (ok..laff it up you guys..like you never missed one)

I found Nvu to be a bit of a PITA, but I know some folks who love it.

Personally, I would suggest using Eclipse (download the Europa, not the repos version) and add the development tools you need and use in combination with the web developer toolbar for firefox.
Since this is Linux, you can easily work locally before you put it live (LAMP is simple install if needed). No muss, no fuss.
Just my .02 worth.
EDIT: At Ubuntu Studio in Synaptic..I meant the "mark by task"..you can see what all is included
