Hi everyone, I need to find some photos on me laptop, but I'm stupid and can't remember where they are. Is there a way to search for just photos in Konqueror? Coz I don't know any of the file names either.

---

### Post by neurostu on 2008-02-15
Go to the directory you want to search (Ideally your home directory) then click on the magnifying glass and search for *.jpg  or you can open a termial cd your home directory, or where you think you stored them and type
```

$find . -name *.jpg

```
