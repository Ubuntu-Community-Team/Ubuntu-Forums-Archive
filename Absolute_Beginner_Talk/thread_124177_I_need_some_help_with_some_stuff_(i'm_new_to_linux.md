I can't answer your first question, sorry...

But, amaroK is really easy to install so that it plays iTunes music. Look at this guide to installing the new [AmaroK]("http://www.ubuntuforums.org/showthread.php?t=80492&highlight=amarok+svn").

As for your third and fourth questions, the answer is a resounding YES! Just get [Automatix]("http://www.ubuntuforums.org/showthread.php?t=80295"). Ask it to install both Firefox 1.5 and anything else you may want. For windows .exe files, check the box next to Wine in Automatix. This will install Wine on your computer. All you have to do is run winecfg in the terminal (Applications-->Accessories-->Terminal) to be ready to run .exe files.

(Note that Wine will work only for some .exe files. For many it will only kind-of work or not even work at all. But it's a start, at least.
