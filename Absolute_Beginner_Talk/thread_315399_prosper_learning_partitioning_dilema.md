### Post by Crakie on 2006-12-09
Ubuntu comes with a full-featured partition tool during the installation, so there is no need to pre-partition. If you insist, you can pre-partition with a piece of software of your choice, of course. Partition Magic (assuming you run Windows now) is probably the best known, although I fail to see why one would purchase it if Ubuntu gives you everything you need for free.

Partitioning with whatever software you use (including resizing partitions) has inherent risks to lose files. The usual recommendation is to backup first. It is my impression that it is generally safe though.

### Post by beefcurry on 2006-12-09
If you are NOT using windows, or if you do NOT have a OS currently installed just boot with the Ubuntu LIVE CD. Use Gparted which is included on the LiveCD, that is a full featured partition magic clone that should do just about anything.
