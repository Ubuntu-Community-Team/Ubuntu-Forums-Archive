### Post by trent dillman on 2006-04-01
Install Firestarter, and you'll have a pretty GUI to help you out:

Either find it in Synaptic (System > Administration) or open a terminal:

```
sudo apt-get install firestarter
```

It will be in Applications > System Tools, or you could open it with:

```
sudo firestarter
``` from a terminal.
