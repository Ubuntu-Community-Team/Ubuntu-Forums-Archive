### Post by DJ_Peng on 2006-11-17
As one of Firefox's testers I can say that 1.5.0.8 is the version of Firefox in the 1.5 line that was made in part to allow updates to 2.0. If you can't find 2.0 in the official Ubuntu packages you can always do what I did. I downloaded Firefox 2 directly from Mozilla, extract it to a folder of your choice (I put it in my Home folder) and simply create a link to run the firefox\firefox shell script.

If you add "-profilemanager" at the end of the command to run the script you'll get the Mozilla Profile Manager which will allow you to create a profile that uses your current profile folder without losing any of your current settings. I've done this as I installed several newer versions of Firefox and even when I moved my profile from one computer to a new one, and then as I moved from Windows to Linux through a couple different flavors of Linux as I tried to find the flavor I would stick with. You can get more information on moving a Mozilla profile at [http://kb.mozillazine.org/Migrating_settings_to_a_new_profile](http://kb.mozillazine.org/Migrating_settings_to_a_new_profile)
