### Post by Aelfric5578 on 2006-09-18
You will need an extermal program capable of doing this.  If you have Nero, if you open the Smart Start Panel, you will see a tab marked "Backup" within that tab you will see an option "Burn Image to Disk"  Choose that and Nero shoud ask you for a file.  This is the .iso file that you've just downloaded.  Open that file, and burn the CD.

If you do not have Nero, there are a number of free programs that will also burn disc images.  I have never used any of these, but the Ubuntu Wiki reccommends: CDBurnerXP Pro ( [http://www.cdburnerxp.se/download.php]("http://www.cdburnerxp.se/download.php") )  Instructions on how to use this program can be found [here]("https://help.ubuntu.com/community/BurningIsoHowto") on the Wiki. 

I hope this helps.  If you have any other questions, just ask.
