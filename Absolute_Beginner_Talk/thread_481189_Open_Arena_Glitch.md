### Post by rne1223 on 2007-08-17
I found the solution. The problem is the screen saver. I notice that if I set the screen saver to 1minute Open Area would crash in one minute. So to fix that you need to go to System->Preferences->ScreenSaver and set the settings however you like. I uncheck the Activate screensaver when... and now I can play Open Arena for a long amount of time with no problem. Hopefully this helps you guys. :)
