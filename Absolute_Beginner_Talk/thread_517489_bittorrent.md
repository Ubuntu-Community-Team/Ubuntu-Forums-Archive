### Post by SOULRiDER on 2007-08-04
Deluge if you use Ubuntu
KTorrent if you use Kubuntu

Now, to answer your question: To resume download simply open the torrent file again, if you didnt download it, youre gonna have to go to the website where you found it and re-open it, but remember to save it to your hard drive first.

Everytime you close bittorrent and want to resume simpli open the torrent file and BitTorrent should pop up. If you use Deluge or KTorrent, you dont need to save the torrents manually, and they both have nice feature rich interfaces.
