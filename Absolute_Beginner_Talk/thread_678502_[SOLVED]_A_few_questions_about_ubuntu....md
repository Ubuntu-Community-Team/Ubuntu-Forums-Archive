### Post by knopper67 on 2008-01-26
You need the python-gpod package. It isn't installed by default though so you'll have to install It through synaptic package manager. The package allows you to read write songs from/to an iPod. Once it's installed Amarok will see your player and you can play the songs.

As for testing your video card I think that you just type glxgears into the terminal. I'm not sure though.
