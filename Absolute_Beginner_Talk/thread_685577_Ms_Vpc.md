### Post by gn2 on 2008-02-02
This was run as an item in a UK magazine recently.

There are problems with MS Vpc and Linux due to the default colour depth of 24 bit.
MS Vpc only supports 16 and 32-bit.

My suggestion would be to use VMWare Server or Virtualbox instead.

[http://www.vmware.com/download/server/](http://www.vmware.com/download/server/)
