### Post by catlett on 2006-06-02
Open the terminal and put in```
 gksudo "update-manager"
``` Hit enter and this will run the update manager. It will prompt you that dapper is available for upgrade. Follow the prompts to upgrade and install. 
If an error is coming up for some reason, post it.
