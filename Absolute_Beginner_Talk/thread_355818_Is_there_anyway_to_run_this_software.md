### Post by old_geekster on 2007-02-07
PE5 will no doubt run in "wine".  Wine is a program that allows Windows programs to run in Linux.  It is a bit complicated to setup, but does work.  You can install it from "Synaptic" Package Manager.

Once you have it installed,  I believe that you can place the CD/DVD in the drive, right click on the .exe file and click install using wine.

As for vmware player, I had it installed and couldn't decide what to do with it, so I uninstalled it.  Over my head for now.
