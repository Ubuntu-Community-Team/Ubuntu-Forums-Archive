### Post by jordanmthomas on 2008-01-29
Part of your problem is that /dev/hda{5,6} are NTFS, but your fstab mounts them as ext3.

Try commenting out those lines in your fstab and then rebooting to see if they get automounted.  If they don't someone can help put proper entries in fstab to mount them on boot.

### Post by zetetic on 2008-01-29
> **Dave.B said:**
> the problem is I cant see ANY of the drives/partitions.  either ont he deskto OR under places


Dave

Well, that's a completely different story!

Well, do this:

sudo -i
Enter your adminstration password
Use nano or gedit for editing.

Remove your actual hda5 and hda6 entries on /etc/fstab

Add these entries to /etc/fstab:
/dev/hda5 /media/WorkingFiles ntfs defaults 0 0
/dev/hda6 /media/MediaFiles ntfs defaults 0 0

If you are running Edgy Eft or later you may add instead these entries:

/dev/hda5 /media/WorkingFiles ntfs nls=utf8,umask=0222 0 0
/dev/hda6 /media/MediaFiles ntfs nls=utf8,umask=0222 0 0

Reboot

Then open "Configuration Editor", on the Menu, and go to this place:
/apps/nautilus/desktop
See if "volumes_visible" is selected.


Note: If you want your Windoze partions to be read/writable, then read this Howto:
[http://www.debianadmin.com/mount-your-widows-partitions-and-make-it-readwritable-in-ubuntu.html](http://www.debianadmin.com/mount-your-widows-partitions-and-make-it-readwritable-in-ubuntu.html)
