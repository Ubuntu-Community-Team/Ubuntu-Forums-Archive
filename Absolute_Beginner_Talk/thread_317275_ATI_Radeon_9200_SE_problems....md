### Post by Kobalt on 2006-12-12
Hello,
first of all, there is no point using nvidia packages with an ATI card... 
You Xorg sever is not yet configured to use an ATI card, you have to run the following command, answer the questions then reboot.
```
sudo dpkg-reconfigure xserver-xorg
```

Cheerio ! :rolleyes:

### Post by Dusty545 on 2006-12-12
I'm not entirely sure why you're using nVidia drivers on an ATI card.  The best bet I can think of is to run:

'apt-get install xorg-driver-fglrx'

This is the ATI driver and is listed as working with the 9200.

Also, if when typing 'startx' at the command line you get an error saying that an instance of X us already running, type 'gui'.

Hope this helps.
