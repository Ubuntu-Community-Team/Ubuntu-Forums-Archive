### Post by monte48lowes on 2008-01-23
Let's start with what wine isn't... Wine Is Not an Emulator... (sorry, couldn't resist)

Wine is basically a 'wrapper' for a windows application which allows the windows system calls to be interpreted to linux system calls. Since wine has been written by reverse engineering windows it is not perfect. Although it has improved quite a bit over the years.

In Windows, I'd say if you have a broadband modem without hardware firewall, i.e. just a modem, then I would run a firewall.
