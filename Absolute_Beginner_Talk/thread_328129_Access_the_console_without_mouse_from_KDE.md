### Post by capitalistpiglet on 2006-12-30
alt+f1 will bring up the menu then system>konsole.
If you go to system settings then keyboard & mouse > mouse> then the mouse navigation tab> check the box saying move mouse with keyboard and you will be able to use the numpad as a mouse.
