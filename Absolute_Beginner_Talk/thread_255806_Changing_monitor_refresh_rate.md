### Post by orb9220 on 2006-09-12
If the file was empty then that means you did not type it in right. Copy and paste in terminal.

gksudo gedit /etc/X11/xorg.conf

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_bak  first to back up your xorg.conf file first.
