### Post by blendmaster on 2006-10-16
Right-click on the applications menu which will bring up the option to edit menus.

click that, then select the section wanted to edit. Click file >> new entry, and add the program with the filepath. 

Then you may have to add the picture manually, but that's pretty easy.
