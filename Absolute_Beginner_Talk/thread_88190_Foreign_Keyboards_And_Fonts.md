### Post by sbkcu on 2005-12-28
Not sure if this is what you're asking but if you want to type a french accent, hold CTRL and SHIFT and then type the unicode hexadecimal (for instance á is 00E1).  See Applications->Accessories->Character Map for a guide to the codes.  The CTRL SHIFT represents the U+.  Don't mean to insult your intelligence; I only just figured this out.
