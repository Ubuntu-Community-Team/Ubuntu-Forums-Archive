### Post by Linux_Man on 2007-09-22
Ok, Im assuming this is a desktop computer. It shouldn't matter if you have Intel or AMD, If you are waiting, a bit, ATI will open-source some drivers and it will be easier to use that, but Intel has free drivers for its graphics cards but they are quite lacking for gaming, and NVidia has great binary drivers (and ATI's are horrible) I don't know about motherboards as I have never fully built a computer from scratch, generic, name brand souncards work well, Intel soundcards are well supported and Linksys/dlink cards when installed internaly usualy work with restricted drivers, if you use USB ones, though your asking for trouble.
