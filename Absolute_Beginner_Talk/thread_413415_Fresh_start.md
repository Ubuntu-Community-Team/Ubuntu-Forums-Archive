### Post by Nik_Doof on 2007-04-19
If you mean "restart" your user account, backup your documents to another folder then remove your home folder (/home/username), Probably best if your not logged into that account at the time

If you mean "restart" your install, like it was a freshly installed version, Easiest way i can think of is to install debfoster

> sudo apt-get install debfoster

Then run it, and select what you want to keep, if you want a fresh-fresh install, then keep ubuntu-desktop and N everything else :)
