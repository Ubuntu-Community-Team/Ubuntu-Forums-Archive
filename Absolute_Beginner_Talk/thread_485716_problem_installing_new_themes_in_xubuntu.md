### Post by phizikal on 2007-06-27
Ok, download your theme and open up the folder that it downloaded in. Mostly firefox will open up archive manager to get you to extract the files, but exit out of it. Go back to the folder it is downloaded on, and open up system> preferences> theme.

And click on the theme file and drag and drop it into the theme manager, Click install and it should install it and click OK. This should work, it got too.

And also make sure they are metacity themes.

Unless somehow you have compiz or beryl installed, with I can doubt.
