### Post by sguart on 2006-07-28
> **Dingobret said:**
> Has anyone got any ideas why I am unable to execute this task?

This is quickly answered by googling thunderbird import outlook. thunderbird relies on outlook to import mail. you need a copy of outlook installed and selected as default mail client in order for thunderbird to import outlook mail. 

Last time I check, there's no outlook linux client. Therefore, it's probably better to perform the outlook mail import step in a windows environment. Then later export your windows thunderbird mail to be imported into the linux thunderbird client.

I hope this answers your question.
