### Post by jfinkels on 2007-03-02
to add or remove entries from the menus, use the "alacarte" menu editor. It's under System > Preferences > Menu Layout, or in the terminal, type
```
alacarte
```
If you know the name of the command you want to run, though, you can always just run it using Alt+F2, then typing the command you want to run. Faster then searching for the icon.
