### Post by tonyr1988 on 2007-02-08
Break things.

Get another computer (or partition yours in two...backup first!), install Ubuntu on it, and just go crazy changing stuff. Go through the FS and look at / edit some of the files. [Compile your own kernel.]("http://www.ubuntuforums.org/showthread.php?t=43065") Compile some programs (and mess with the ./configure options - see what they can do).

Or, if you get really bored:

Open a terminal. Without typing anything, hit the Tab key twice. When it says:

> Display all x possibilities? (y or n)

Hit y. You'll see a list. Go through each item and learn what it does. :D:D:D

But seriously, a lot of people (including myself) learn things the hard way (breaking their system) - you might as well learn it on a testing box.

And, of course, read the forums and the wiki often.
