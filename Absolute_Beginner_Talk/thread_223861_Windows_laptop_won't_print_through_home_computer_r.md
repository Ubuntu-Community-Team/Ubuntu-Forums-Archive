### Post by Sef on 2006-07-27
Here Ubuntu's documentation on setting up [Samba.]("https://help.ubuntu.com/community/SettingUpSamba")

From the Samba page:

> Do you need Samba? Maybe not:

  If you want your Windows computer to be able to use (via a network) a printer that is attached to a Linux computer, you don't need Samba. CUPS can be configured to make the printer accessible to the network.
