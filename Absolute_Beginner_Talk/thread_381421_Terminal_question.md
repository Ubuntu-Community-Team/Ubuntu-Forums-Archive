### Post by IYY on 2007-03-10
These programs are just terminal emulators. They all run the same **shell**, namely Bash, and are therefore exactly the same. Of course, some terminal emulators will support antialiased fonts, scrolling, and GTK copy/paste, but these are just superficial differences.
