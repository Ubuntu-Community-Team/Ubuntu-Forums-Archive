### Post by rsambuca on 2007-09-06
Yeah, you can triple, quadruple, quintiple boot, or whatever you wish.  You can also just install kubuntu on your current setup and select either gnome or kde when you login.  ```
sudo aptitude install kubuntu-desktop
```

### Post by maruchan on 2007-09-06
If you install Kubuntu-desktop it'll appear in your "sessions" menu at the login screen. Kubuntu isn't treated as a different OS, just a different desktop environment. So this is a simple request on your part :)
