### Post by ugm6hr on 2007-06-21
Any wireless router that allows setup via a webpage (i.e. from any web-browser) should work.

Which is basically any router branded with a networking manufacturer rather than an adsl provider.

For example, I have a Toshiba, a D-Link and Linksys router that all work with Linux just as they would in Windows (i.e. independent of OS), whereas BT and Apple branded routers often require Windows (or MacOS) to install their setup software.  I personally don't find there is much difference between the various manufacturers with regard to ease of setup - you need to have a little understanding of wireless security to get them up and running.

It's more important that the wireless cards in the desktop / laptop are carefully selected to work with Linux.
