### Post by thelatinist on 2007-12-16
It depends.  If these are DRM-enabled files purchased through iTunes, they will only play in iTunes and on devices synchronized with iTunes.  If they are files ripped from CD in aac format you should be able to play them if you install the ubuntu-restricted-extras package.  Instructions are here: [https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)
