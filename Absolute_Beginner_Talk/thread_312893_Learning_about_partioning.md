### Post by az on 2006-12-05
> **SteelCore said:**
> Is there a document that gives detailed explanation about hard disk partitioning and how to apply it on new disks using the Ubuntu cd?

The installer should be straightforward.  It can partition your drive safely for you.  All you need to do is tell it what size to make the partition you are shrinking.  It will then use the free space to create the root and swap partitions without you needing to do it yourself.

If you are using a blank hard drive, again, it will use the free space to make it's own partitions.

You can use manual partitioning to create extra partitions if you want....

some interesting links here:
[https://help.ubuntu.com/community/DrivesAndPartitions](https://help.ubuntu.com/community/DrivesAndPartitions)
