### Post by mocoloco on 2007-09-16
Do you have the PPC install disc of Xubuntu?  PPC is PowerPC, which is a different processor architecture that Apple used until fairly recently when the switched to using Intel processors.  The PPC installer is different from the regular desktop install disc.
PPC isn't "offically" supported anymore, but that just means it's community supported, so it should still live on.  You can get it [here]("http://cdimage.ubuntu.com/xubuntu/ports/releases/7.04/release/").
