### Post by Inxsible on 2007-11-01
you would have to change the images in the theme for it to take new icons. OR 

simply use a different icon theme in your current gnome theme. This could be iffy, because it might have some other icon(like for folders) that you don't like.

I normally change the icons that i don't like right within the theme itself.

For changing the names : Click on Understanding fstab in my signature, search for 'How to label' without quotes. Be careful of not using make2fs unless you want to wipe all data.
