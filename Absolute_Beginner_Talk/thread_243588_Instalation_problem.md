### Post by Haegin on 2006-08-25
It may be your computer is not powerful enough to run the live cd at a decent speed. Your best bet is probably to download the alternate install and use that to install ubuntu on your computer.
[http://www.ubuntu.com/download](http://www.ubuntu.com/download)
Go there and select whichever country you are in then scroll down to find the alternate install.
You probably want the x86 cd (if you were using the red normal install cd then you do want x86 alternate install).
