### Post by ajgreeny on 2006-10-09
How much ram do you have; you really need at least 256 to run gnome or kde, and preferably 512 if you want some speed.  If ram is a problem get the alternative install CD instead of the live CD, and once installed download and use xfce instead of gnome or kde.  Or you could just download the alternative install CD of xubuntu instead and install that.
