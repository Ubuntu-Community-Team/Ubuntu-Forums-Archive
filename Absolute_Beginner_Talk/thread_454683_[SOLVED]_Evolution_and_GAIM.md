### Post by teknosexual on 2007-06-01
Well, I'm using Edgy, but I assume the steps are about the same for Feisty.

Open Evolution, go to Edit > Preferences > Mail Preferences > Automatic Contacts. 

There should be an option there that says Instant Messaging Contacts and a checkbox to synch with GAIM buddy list. Also a button to synch immediately should be there if you don't want to wait for the synch to occur on its own.
