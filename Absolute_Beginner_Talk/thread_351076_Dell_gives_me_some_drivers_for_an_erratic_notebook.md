### Post by josh420 on 2007-02-03
Theoretically, you should be able to stick the BIOS exe on a DOS Boot disk of some sort and run it like that.

Unfortunately, more and more vendors are using self extracting EXEs for their stuff, which means you'd need to use WINE or open it on a windows box.

Most of the time it's just a setup.exe program that automates copying the needed boot files and update stuff to your floppy drive. Which you then boot to.
