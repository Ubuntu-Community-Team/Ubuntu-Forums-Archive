### Post by MikeMLP on 2007-07-27
The desktop effects are plain compiz with none of the fancy things you want.  This came with Feisty by default.  Did you install compiz from a separate repository?  Or did you just use the Ubuntu repositories?  All of the new things like Burn and firepaint are in a project called Compiz Fusion ([http://forums.opencompositing.org/]("http://forums.opencompositing.org/"))
If you want those new features, you'll have to install a git repository or install from source.  You also could install Beryl, which is no longer developed or bug fixed (But is probably more stable right now).  Beryl does not have the newest additions such as firepaint, however.  Good luck, and ask around in the forum that I linked to.
