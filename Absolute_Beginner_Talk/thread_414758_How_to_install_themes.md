### Post by AAM on 2007-04-20
ajmannen, some of the items that are installed with themes have to be selected, others can be drag'n'drop. Either way you open the menu choice 
> System | Preferences | Themes
Grab your icon file with the mouse and drag it on the window, if it loads, you are then able to select the icon set from **Theme Details** - there's an **Icons** selection tab there.
If it doesn't load then use the **Install Theme...** button.

Hope this works for you!
