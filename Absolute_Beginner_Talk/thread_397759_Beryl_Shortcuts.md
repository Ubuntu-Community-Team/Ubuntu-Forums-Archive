### Post by Campingman on 2007-03-31
Apologies, should have googled before asking, I found these that all work ok, are there any more, Is there a comprehensive list, how do you create your own?

    * CTRL+ALT+Mouse click and hold: Freely move desktop cube
    * CTRL+ALT+ARROW KEYS: Turn the desktop cube
    * CTRL+ALT+SHIFT+RIGHT or LEFT arrow: Carries active window to the next desktop
    * ALT+Mouse click and drag on a window: Drags the window
    * ALT+Mouse Wheel on a window: Alters transparency of the window
    * F8: Scales and shows all windows on all desktops
    * F8: Scales only the windows opened on the active desktop
