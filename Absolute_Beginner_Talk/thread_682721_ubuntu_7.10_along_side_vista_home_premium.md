### Post by ColdAir on 2008-01-30
May be that you want to have a look at [http://neosmart.net/wiki/display/EBCD/Ubuntu](http://neosmart.net/wiki/display/EBCD/Ubuntu) for Dual-Booting Ubuntu Linux and Windows Vista. This is only if you intend to install Ubuntu on its own partition and want to use Vista boot loader.
I have a working Dual boot Vista+Ubuntu laptop configured using the instructions from the above website. 
The point here is Vista uses a BCD configuration file instead of the old boot.ini . Also give a search in this website for your question. You can find  plenty of help.
