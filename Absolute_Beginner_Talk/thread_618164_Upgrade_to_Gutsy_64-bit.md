### Post by PmDematagoda on 2007-11-20
You shouldn't have much problems with Ubuntu 7.10 x64, Frostwire and all the other applications work well in 64 bit (in some cases even better), the installation of Flash is also very much simplified as Firefox has been well modified by Ubuntu so that the plugins are installed automatically without much of a hassle.

Skype works on Ubuntu 64 bit using this [guide]("http://ubuntuforums.org/showthread.php?t=432295") and Epiphany should be easily installed using the repos or by using the command "sudo dpkg -i --force-architecture".
