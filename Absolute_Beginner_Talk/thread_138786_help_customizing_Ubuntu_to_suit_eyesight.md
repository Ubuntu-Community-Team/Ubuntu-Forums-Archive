### Post by Pragmatist on 2006-03-03
Are you using GNOME? KDE? XFCE?

For the maximum control over your window manager and overall appearance of your desktop, take a look at alternatives like fluxbox:
[https://wiki.ubuntu.com/Fluxbox?highlight=%28fluxbox%29](https://wiki.ubuntu.com/Fluxbox?highlight=%28fluxbox%29)

You can install it from synaptic: ```
sudo synaptic
``` then hit search and type in **fluxbox**

AFAIK, you should be able to set the panel background by Right-Clicking and selecting **panel properties**
