### Post by Obor on 2007-03-29
yep, just a normal update will bring you up to speed. It should be showing you if you have updates available. try the update manager (you might have beta already if you ran the update manager in last few days)
or in terminal 
```
sudo aptitude update && sudo aptitude upgrade
```
