### Post by 505 on 2007-08-29
If there is noting on the 500GB drive, you don't have to format it, just put a file system on it. You can use GParted for that (you might need to install it first, it is in the repositories). If you only plan to use Ubuntu you can put an ext3 file system on that drive.
