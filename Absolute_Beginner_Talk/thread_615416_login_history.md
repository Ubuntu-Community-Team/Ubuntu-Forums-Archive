### Post by wooster on 2007-11-17
A couple commands might help you that i'm not sure you are familiar with. First is the "last" command which will show a list of users and times they login/logout at, the "lastbg" command will show the last failed login attempts, and "finger" command which can look up user information. You can also look in the file /var/log/auth.log and see user logins. Hope that helps.
