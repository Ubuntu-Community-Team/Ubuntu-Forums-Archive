### Post by ComplexNumber on 2006-05-03
type in 'gksudo system-config-display' on the command line. i don't know what the entry is called in the menu, but there should be something in the sytem/admin menu called 'display'. this will change the maximum resolution for you. there is another option to change the resolution in the preferences section of the system menu. use that to select the display for each individual user.
