### Post by stawnster on 2007-07-15
I have Ubuntu Feisty, fresh install with all the updates. Installed wireless card WDA-2320 AFTER install and updates. Ubuntu recognized it upon bootup and downloaded/installed the Atheros restricted drivers. I restarted, then switched network from "wired" to "wireless". It prompted for WPA password, and then asked for a passphrase to store on Ubuntu for some kind of internal process. I entered a different password than the WPA and it connected me in about 5 seconds. CONFIRMATION: WDA-2320 works out of the box!
