### Post by nicedude on 2008-04-21
I can't say for sure about your laptop but mine ( Acer5520-5716 ) wouldn't fully work with the 64bit Gutsy as there are no 64bit wifi drivers for it so I was forced to use 32bit OS and I don't see any performance issues. I believe it would only make a large difference if you were doing really intensive things like editing video for example. So while I don't know for sure about your setup I would say I think that in general there is more of a chance that you could run into a lack of 64bit support for a given thing than 32bit. I think both will use your dual core but that only matters when you are doing two things. And if you never got your flash to work you should search in the forums for how to add the Medibuntu restricted repositories to your sources list as it will allow you to install flash, video playback codecs & other cool things that aren't included in the other flavors of ubuntu.
