Under SYSTEM >> PREFERENCES >> SESSIONS, on the session preferences tab, is an option to automatically save sessions on logout.

This answer might be useless as I don't actually know what this does, but I *think* it just saves your system state - which means next time you log in, Ubuntu will load all the programs you had open last time.

AFAIK this option is not enabled by default - perhaps you've turned it on by mistake? My system never asks me about saving sessions .....

### Post by 3rdalbum on 2007-11-08
Some Gnome programs can save their state, so if you log out while those programs are running, Gnome can restore them to exactly what they were doing when you log back in.

Useless? Not quite, but most people don't work that way. You can turn it off from System > Preferences > Session > Session Options.
