### Post by yota on 2006-10-20
Hi, you can start from this page:

[http://wiki.linuxquestions.org/wiki/Linux_software_equivalent_to_Windows_software](http://wiki.linuxquestions.org/wiki/Linux_software_equivalent_to_Windows_software)

(self-explaining url, isn't it?)

But please be open-minded too! Sometimes there are things that on linux are done in different way rather than just with different applications, and many habits have to change: experiment and try to learn, there's a lot of very good documentation on-line on every subject.

Most of program are already in ubuntu repositories and are very easy to install:

[http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)
[http://www.monkeyblog.org/ubuntu/installing/](http://www.monkeyblog.org/ubuntu/installing/)

the equivalents IMHO would be like these:

webserver -> apache
webeditor -> nvu (easy), bluefish (advanced)
network   -> ?!? please detail better
file storage -> nfs (unix clients), samba (windows clients)
