### Post by stinger30au on 2007-09-03
After bashing my head against a brick wall trying to get my scanner to work in Ubuntu i have tried something different.

I have been toying around with "Virtual Box" for a little while now. You can install Virtual Box via Automatix 2 very easily.

[http://www.getautomatix.com/wiki/index.php?title=Installation](http://www.getautomatix.com/wiki/index.php?title=Installation)

I then installed my old version of Windows XP. configured it so it would use the DVD drive and installed XP and also told it to use USB devices. It detected my Canon USB Lide 20 scanner and i installed the drivers that came with it. It works just fine in the virtual machine.

Now... in the virtual box there is an option for bidirectional copy/paste via clipboard. this is not working. dunno why. so to get myself out of trouble i have shoved a 256Meg USB memory stick in the usb port. any thing i scan i stick on it. i can exit the virtual machine and access these files from linux.

i would love to have a shared directory on the virtual machine to copy/paste data to/from linux, but i dont know how this is done.

hope this may help someone out there. it is a little bit of mucking round, but it does the trick for me
