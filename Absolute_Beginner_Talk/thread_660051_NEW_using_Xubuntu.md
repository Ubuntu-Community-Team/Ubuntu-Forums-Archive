There isn't any disadvantage to running Xubuntu.

Xubuntu is a version of Ubuntu that is designed for lower spec PCs. Ubuntu requires a bit more processing power and RAM that Xubuntu. This makes Xubuntu perfect for older computers.

Found this on Wikipedia.org

[QUOTE=Wikipedia.org]Ubuntu uses the GNOME Desktop Environment as well as GNOME applications. Xubuntu uses the Xfce desktop environment which, using fewer system resources, works better with older computers. It is intended for users with less-powerful computers, or those that require a highly efficient desktop environment on faster systems. Xubuntu also features primarily GTK+ applications that are designed to use fewer resources than their GNOME counterparts.[/QUOTE]

### Post by sub2007 on 2008-01-06
My computer is perfectly able to comfortably run GNOME. I prefer and use XFCE because I think it is much more customisable than GNOME. I do find that for me it is more responsive than GNOME but I wouldn't be prepared to declare that is **significantly** faster on my hardware nor uses fewer resources (especially if you, as I do, run KDE applications and Compiz Fusion on it). The deal breaker for me was customisation and I prefer the way that XFCE works over KDE, therefore use XFCE.
