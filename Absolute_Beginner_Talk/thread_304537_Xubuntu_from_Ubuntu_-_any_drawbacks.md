### Post by ewl1217 on 2006-11-21
Xubuntu uses different applications from Ubuntu, but it is fairly similar in functionality. It uses Abiword for word processing and Gnumeric for spreadsheets. Unfortunately, it has no presentation program. However, you can install all your favorite programs from Ubuntu, even though they might run a bit slow.

### Post by seshomaru samma on 2006-11-22
The way I see it there are two options:
Installing Xubuntu
or
Installing XFCE on top of Ubuntu

In my opinion the second is preferable, first tweak your Ubuntu well, if you need to connect to other computers or printers then do it in Ubuntu . Then install XFCE and you have a Xubuntu with the same programmes as Ubuntu and slightly faster. It can run SCIM easily and you can switch the menues to Chinese in the login screen.
The first option will give you a fster machine but some of the twaeking will be quite difficlut and will involve much more terminal work.
If you are interested in a faster machine, you might want to give ice-wm a try. It is very vey fast.
[This post]("http://www.ubuntuforums.org/showthread.php?t=171203") explains how to set IceWm with Rox for a fast and easy desktop environment.
