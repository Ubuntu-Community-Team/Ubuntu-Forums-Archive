### Post by Vengeful Cynic on 2006-02-15
[QUOTE=dearanna]Hey all,

I was just wondering if it is possible to run AOL on ubuntu. I've used Ubuntu before, but never with Internet. Now I want to use it but I can't fugure it out? Another, better question would be, What internet does work with ubuntu??[/QUOTE]

Ignoring for a minute the question of "why?" (hey, I'm being nice... never said I had to give up certain personal disdain for AOL), it would appear that it is possible to run AOL under Ubuntu, it just takes a bit of doing.

From what I understand, and I haven't tried this, there is a package called penggy available in the universe repositories (networking subsection) that should allow you to connect to AOL.  All you have to do is make sure you have the universe repository in your repositories list (in Synaptic go to Settings->Repositories and if it's not there, click add select "Ubuntu 5.10 'Breezy Badger'" from the drop-down and check "Community Maintained (Universe)"), install penggy, and read through their help file.  They have a webpage at [http://savannah.nongnu.org/projects/pengfork](http://savannah.nongnu.org/projects/pengfork) and while it might be a bit thick with tech-speak, it seems fairly informative.  If you're feeling more adventurous, yolinux.com has an old article on how to connect to AOL using Linux under PengAOL located [here]("http://yolinux.com/TUTORIALS/LinuxTutorialAOL.html").

As to your question about other ISP's, most other ISP's provide nothing more client-side than an internet connection, making it very easy to transparently connect to most dial-up and high-speed internet providers.  In short, pick an ISP and odds are almost 100% they're easily connected to using Ubuntu, especially if they don't have flashy end-user software like AOL.

-Vengeful Cynic
