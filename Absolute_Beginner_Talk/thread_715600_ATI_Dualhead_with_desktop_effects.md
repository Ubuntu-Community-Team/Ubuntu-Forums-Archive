In order to get desktop effects to work you will need the restricted driver and xgl.  I would suggest trying that first and testing the screen resolution to see just how far it will go before it crashes.  I should warn you that xgl is a tad buggy with desktop effects but I'm only saying this based on my personal experience with my mobility radeon x300 graphics chip.  Best of luck to you.
