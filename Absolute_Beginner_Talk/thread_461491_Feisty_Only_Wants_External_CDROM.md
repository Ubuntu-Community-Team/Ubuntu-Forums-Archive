### Post by taurus on 2007-06-02
Maybe your internal CDROM is /dev/hdc.

Edit /etc/fstab

```
gksudo gedit /etc/fstab
```
and place a # sign to comment out this line.

```
/dev/scd0 /media/cdrom0 udf,iso9660 user,noauto 0 0
```
Then, add this new one for your CDROM.

```
/dev/hdc /media/cdrom0 udf,iso9660 user,noauto 0 0
```
Save it and reboot.  See if it reads your internal CDROM when you insert a CD now.
