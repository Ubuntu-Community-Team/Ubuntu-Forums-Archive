### Post by kopo88 on 2006-10-26
Believe it or not, that's how it's supposed to be by default.
If you want to change it, go into System > Preferences > Fonts, and change the resolution from 96 dpi to 72 dpi. Alternatively, you could just make the system font sizes smaller.
Keep in mind that this will probably make a lot of things illegible.
If there's one thing Ubuntu/GNOME/Linux still haven't gotten right, it's decent font rendering.

### Post by IYY on 2006-10-27
Looks fine to me. But if you want it to be more compact, first change the fonts (you might even like to turn off anti-aliasing; it doesn't do much good in small sizes).

Then if it still feels too ``bulky'', you can edit the gtkrc file in your theme. The clearlooks theme has easily modifiable values for the sizes of various components (like menus and buttons).
