### Post by alienexplorers on 2008-01-01
In Firefox under the main menu have you checked to see if:
View>Sidebar>Bookmarks is checked.   If not, select it.

If that does not work you can create a new profile using:
> firefox -profilemanager

Then you can import you data from the old profile to the new profile by following this:
> [http://kb.mozillazine.org/Migrating_settings_to_a_new_profile](http://kb.mozillazine.org/Migrating_settings_to_a_new_profile)
