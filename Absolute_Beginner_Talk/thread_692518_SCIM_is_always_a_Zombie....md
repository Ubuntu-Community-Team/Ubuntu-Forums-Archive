### Post by tyggna1 on 2008-02-10
Smart Common Input Method.  It provides language support.  Probably want it running.  if you really want it dead, though, you can use a pkill -9 <pid> from the command line to send it an "unignorable kill".  If it still is running after that, then you'll have to edit something in your /etc/init.d directory.
