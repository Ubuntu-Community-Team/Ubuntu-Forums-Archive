### Post by Dinerty on 2006-07-27
Ubuntu has a built in firewall, but if you want to configure it by gui you can use **"Firestarter"** [http://www.fs-security.com/](http://www.fs-security.com/) 

To install Firestarter type:

```
sudo apt-get install firestarter
```

Anti virus is not really needed unless you are running a mail server, like mentioned there is AVG and ClamAV I believe
