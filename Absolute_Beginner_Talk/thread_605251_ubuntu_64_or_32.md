### Post by sawjew on 2007-11-06
Despite what you hear there is little difference between the usability of 64 bit and 32 bit.  I have been using 64 bit Ubuntu for about 18 months now and I used 32 bit for about the same time before that.  I also have 32 bit Xubuntu installed on an old laptop.  The only issue I have had with 64 bit Ubuntu is with the java plugin for firefox, everything else works fine.  As a general rule if drivers are available for Linux they are available for both 32 and 64 bit versions.

I would recommend using 64 bit because the more people who adopt 64 bit systems the more people will develop on 64 bit systems.  The speed difference is marginal to unnoticeable for most common apps but if you compile software or run modelling software the difference is considerable.  The other advantage is that you can use more than 4GB of ram with 64 bit.

Give the 64 bit OS a go and support the new wave of computing.  If you have any problems with java (the only issue as far as I know) then use the script provided by Kilz (in the sticky thread at the top of the 64 bit forum) to install 32 bit java and 32 bit firefox and use that.  

Come to think of it, check out those stickies for some more info on the pros and cons of 64 bit.  Bear in mind though that some of that info is pretty old and 64 bit Ubuntu has improved dramatically since Dapper.
