### Post by justin whitaker on 2007-07-26
> **4Real said:**
> the ISO , its in Rar format.

No, it's LHA compression. 

[http://en.wikipedia.org/wiki/LHA_(software](http://en.wikipedia.org/wiki/LHA_(software))

Its a little bit different. 

Actually, you don't need to extract it, just change the bios so you can boot from CD, and run Puppy in RAM. If you like it, you can install it from RAM, no decompression necessary.

And actually, there is a ton of stuff you can install. They use a compressed format called a .pup, which basically acts like one of Slax's modules: you can add applications to the running system on the fly, including a compile suite.

Pretty slick.
