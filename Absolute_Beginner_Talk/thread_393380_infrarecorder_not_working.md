### Post by annda on 2007-03-25
burning a cd means that you save/record/write something to it. if your cd drive cannot record/burn cds, no program will help you.

if so, you might have a really old computer and it is not sure ubuntu will run nicely. still, you can order a cd free of charge here:
[https://shipit.ubuntu.com/](https://shipit.ubuntu.com/)

and try it out. it is a live cd, so you won't have to go through installing ubuntu to try it.

however, if your computer is really old, it's possible that it can't even boot from cd. please read your computer's manual to see if you can change the BIOS settings to boot from cd. even if you don't know what BIOS is, it is mentioned in your manual. read the section on boot, boot sequence, boot options, or something like that.
