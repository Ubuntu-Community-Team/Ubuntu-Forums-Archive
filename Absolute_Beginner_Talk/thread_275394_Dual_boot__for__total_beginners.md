### Post by podunk on 2006-10-11
No problem. :-0

One way to do it so your windows partition is completely safe from mistakes is 
Unplug the computer.
Unplug the Windows hard drive
Install the Ubuntu hard drive jumpered for master.
Replug your computer
Install Ububntu from the CD.
Unplug the computer. Rejumper the hard drives.
Use Bios to set up the boot drive order

The real simple way to do it is install the new hard drive
Stick the cd in and boot into Ubuntu.
Tell Ubuntu to install into the largest free space.
At the end of the install you will have a boot up menu that lets you chose between Ubuntu and Windows.

### Post by CatKiller on 2006-10-11

Set BIOS settings as you wish.
