### Post by Paul820 on 2007-08-02
You ever notice how windows is constantly using your hard disk? Never hear it grinding away even when you are not doing anything? When windows is idle, it uses that time to write gods knows what to it. In ubuntu, the only time my hard disk is getting used is when i open an application or when i save something. The rest of the time it is quiet. Maybe windows grinding away all the time increases the heat.
