### Post by mcduck on 2008-01-07
Sure. Just go to System/Administration/Language Support and install the languages you want.

Users can select the language they want to use at the login screen. Ubuntu will also remember the language they have selected, so they don't need to change it every time they log in.

I think it's also possible to start only a certain application with different language, but I can't remember how to do that.. :)
