### Post by bobplano on 2007-07-10
if you just took the screenshot you can just change the extension in the window. otherwise open it up in gimp and then resave it as a .jpeg.

### Post by jordanmthomas on 2007-07-10
It'd be fastest to change the extension in the window as bobplano suggested, but if you install imagemagick, you can convert faster than opening up gimp (just adding another way here).

```
  convert shot.png shot.jpg
```
