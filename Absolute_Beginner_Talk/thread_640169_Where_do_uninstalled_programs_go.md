### Post by PmDematagoda on 2007-12-13
When you use apt-get or any normal package manager to install applications, it downloads the deb file and saves it in an archive automatically, this makes it easier whenever you want to reinstall the same application again. But that will only happen if you **install** the package, not in any other case.
