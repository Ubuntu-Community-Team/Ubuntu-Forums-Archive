### Post by Spin Doctor on 2006-11-28
This is what I did to be able to include a logo signature with outcoming mails in Evolution 2.8.1:
[LIST]
[*]Go to Settings menu, Select the menuitem underneath the emailsettings, I am running swedish locale, but I guess translating it  to "Edit preferances"
[*]Check the option: Format Messages as HTML
[*]Click on the second tab, "signatures"
[*]Click add new button
[*]From menu, select Format>>HTML
[*]Include your graphics and save footer with name
[*]When writing message, select footer[/LIST]For me this worked fine.
