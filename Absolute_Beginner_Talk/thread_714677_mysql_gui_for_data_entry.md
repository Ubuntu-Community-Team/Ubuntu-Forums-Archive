```

```

### Post by niteshifter on 2008-03-04
There's MySQL Administrator (in the repositories as mysql-admin) which is a GUI that will let you do anything with MySQL - including data entry.

I've also used MySQL Navigator (in repositories as mysql-navigator) in the past, it's not as full-featured as the admin program above.
