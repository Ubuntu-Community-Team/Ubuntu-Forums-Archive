Well, since the archiving method uses gzip, you're not going to get much better than that from any other compression program. I'd guess the problem is that the pdf is already internally compressed or contains a lot of compressed stuff. If it's got lots of pictures, for example, it won't compress because graphics are usually already compressed as jpegs or whatever.
