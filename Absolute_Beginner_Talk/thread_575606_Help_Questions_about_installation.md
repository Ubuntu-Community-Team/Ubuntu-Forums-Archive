### Post by HermanAB on 2007-10-25
I think the correct answer is a Definite Maybe.

Most trouble concerns WiFi drivers.  However, you can use a Windows driver with ndiswrapper.

Stick the CD in and boot.  If most things work, then go ahead and install.  If more than two things don't work, then shop around for another Linux distribution.  The only difference between Linux versions is the install wizards.  Everything else is the same.
