### Post by MikeMLP on 2006-08-13
I'm assuming that you are trying to install from the live CD.  You could try installing from the alternate CD, however if the live CD does not provide you with a functional desktop, there probably aren't any guarantees that things will be any different once you install Ubuntu to your hard drive.  Did you run the MD5 checksum before burning your disc?  Did you run the CD check utility that comes on the live cd?  If both of these check out, than it is probably a driver incompatibility issue, which may be hard to troubleshoot.  Hopefully, it is a faulty download or CD burn.
