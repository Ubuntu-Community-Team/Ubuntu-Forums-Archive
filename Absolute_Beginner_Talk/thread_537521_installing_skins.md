### Post by mcduck on 2007-08-29
Go to System/Preferences/Themes and drag the theme packages into that window to install the themes. Then click the "Customize"-button and now you can select the application, window and icon themes you want to use.

Sometimes the theme packages are in wrong format or contain more than one theme. In those cases you can just extract the package by hand, icon themes into ~/.icons and window- and GTK2-themes into ~/.themes (Those are hidden directories inside your home dir, press Ctrl-H to show hidden files in Nautilus).
