### Post by catlett on 2006-09-02
If you have an ethernet cable connected then all you should have to do is go to System<Administration<Networking. When the window comes up it should display an "Ethernet Connection". It will most likely say "not configured". Click on the ethernet connection entry to highlight it and to bring up the options on the right. Then select "activate". If you get any errors, post.
