### Post by FuturePilot on 2007-11-12
Chances are you're already up to date:)
You can check this by making sure you have all of the repos enabled. I think it's K Menu>System>Adept. Go File>Manage Repositories. Make sure you have all of the ones enabled in the first tab. Then go to the Updates tab and make sure the first 2 are checked. You can also check the Backports one too, if you want to enable the backports.
