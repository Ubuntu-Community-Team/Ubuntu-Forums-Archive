### Post by outofnicks on 2007-02-28
You are not screwed. You can download all the packages in the repositories on the web here:
[http://packages.ubuntulinux.org/](http://packages.ubuntulinux.org/)
Then you need a way to move the debs from Windows to Ubuntu, like a USB stick or whatever. Many ways to do that, so I won't go into that unless you need the help there.
Pick your version, browse the packages, each package has it's dependencies listed so you may need to download some of those. You can open Synaptic Package Manager to see if any are already installed, then just download the ones you don't have.
Next, in Nautilus, at least in Dapper is the right-click option to use GDebi Package Installer on a local deb package. I'm pretty sure GDebi will handle all the dependencies also.
