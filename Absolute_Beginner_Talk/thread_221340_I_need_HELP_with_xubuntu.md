You can always, easily, install the games from the repositories using Adept after the installation.  It only takes a minute.

*Do not* use OEM mode unless you are building a PC for someone else.  With the alternate installation, text mode can be Expert or Normal.  Unless you have a real reason for doing so *and a firm understanding of Linux*, I suggest you use the Normal mode.  Alternate is available via CD.
