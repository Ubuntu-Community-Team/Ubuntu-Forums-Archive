### Post by Jaearess on 2007-06-10
A few things:

1) Compiz is not installed by default in Kubuntu, only Ubuntu. You can install it in the terminal by using sudo apt-get install compiz. As far as I remember, this tries to pull in a lot of GNOME dependencies, so I didn't end up installing it and went with Beryl instead (sudo apt-get install beryl.)

2) System-> Preferences is only in the GNOME menu found on Ubuntu, not on Kubuntu at all.
