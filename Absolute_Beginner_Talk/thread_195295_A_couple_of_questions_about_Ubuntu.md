### Post by HeavyAl on 2006-06-12
Viruses are pretty much a non-issue since you aren't running as root. The ones that are out there usually take advantage of sillyness on the users part and since Ubuntu uses sudo to do everything in root your unlikely to run across one that will actually get a handle on your system.

Still, if you're worried you can install ClamAV which is a top-notch antivirus program for linux. Do a search for it in Synaptic.
