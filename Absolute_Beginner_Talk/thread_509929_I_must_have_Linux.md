### Post by voided3 on 2007-07-26
I run damnsmalllinux on a 200mhz pentium one comp with 72mb of ram and it works just fine. My only gripe with it is the vesa video drivers have slow response for scrolling and opening new windows even with the Radeon 7000 I stuck in it, but otherwise it's very light on ram and lets me actually use the computer for something. I once did an experiment in VMware with DSL and it still booted with 12-16mb given to it (though choppy) and only 24-32mb of ram to be smooth. I also have used/use Puppy Linux on a bunch of computers, primary a 700mhz celeron computer w/ 192mb of ram I built into a briefcase. It is VERY easy to get up and running (even with wifi if the card is supported) and only needs 64mb to be smooth and 128mb or more to run entirely in RAM. In my experiences though, it's best on anything that's faster than 500mhz; I had it on a 450mhz K6-III computer w/ 192mb for a while and was still very usable, but it was obvious that the CPU was a bottleneck (though it beats the pants off of Win98!). It's lightning fast on a 800mhz PIII comp I have and it still blazes on that Celeron briefcase comp I made. I had Puppy on that 200mhz comp I mentioned above and it ran fine on that, but bear in mind still what the system is and was designed for. With 266mhz, it still will be slow, but you have way more than enough ram and hard drive space w/ 256mb plus it will be much more stable. Just make a primary ext2 partition and leave about 300mb or so for a swap partition using gparted on Puppy.

 In conclusion, try the latest DSL and Puppy releases (BTW, you can boot off of a CD-ROM with that system right? I had to make a boot floppy for my 200mhz comp to select it since it's a Compaq and the BIOS suck; while you're at it, make sure the CD-ROM works!). I also know someone is going to say it eventually, but if this project doesn't turn out the way you want as a result of the hardware, older computers are very cheap. My 800mhz PIII computer with an Asus mobo, cd-rom, 20gb hard drive, 64mb of ram, and an 8mb SIS vid card was only $15 at a flea market! I stuck in 640mb of ram and an nvidia mx440 and it flies, so if you want a more modern system just for the basics, it's VERY affordable. I understand though the draw to playing with old comps; my oldest is a Compaq Contura laptop with a 25mhz 386 in it and 8mb of RAM still running win3.1! Anyways, good luck and most of all have some good fun!

### Post by bromix on 2007-07-26
4Real....   Although there many streamlined distributions of Linux available for use on an older not so powerful pc, I would really encourage you to look around a bit and find a good deal on a pc with a little more power behind it.    The computer geeks [http://www.geeks.com](http://www.geeks.com)  has a bunch of desktop systems for affordable prices.  Spending just $100-$200 would get you something with which you could take advantage of all that the open source community has made possible in a Linux Operating System.  I while having an older computer, I was able to learn and familiarize myself with the Linux environment.  But, once I upgraded, and now have something that supports XGL, CompizFusion, meets system requirements of popular windoze games to run under Wine/Cedega....I now have something that I can use to show other people what Linux can do.  So, go ahead and get xubuntu...familiarize yourself with the operating system, but don't be satisfied with that :)  Don't "cage the beast"   Save some money and buy yourself a box where you can use the OS to it's full potential.
