If you want to have your gDesklets automatically launch when you start up Ubuntu, then go to System -> Preferences -> Sessions -> Startup Programs -> Add, and type in "gdesklets", and assign it some boot sequence, probably 50, which worked for me.

### Post by jasay on 2005-10-19
From the look of your screenshot it seems as though you have installed gdesklets and are working on the starterbar applet.  If that's the case there are a few options to add programs to starterbar.  For the "command" line enter whatever you would in a terminal to run the program you want (eg. firefox, gnome-terminal, gaim, etc.)  

Or, instead of creating new launchers, you can drag any launcher from your panel into the starterbar (right-click on the program from the applications menu and add to panel.  then drag into starterbar and remove from the panel if you want) .  Be warned that people have had firefox open to odd pages after doing this drag.  If it happens to you, reopen the launcher properties and remove the %u from the command line.
