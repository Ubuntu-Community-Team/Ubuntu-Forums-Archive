Yes. When you first boot up the Live CD, you will see an "Install" icon on the desktop.  This is for when you are ready to install to your HDD.

This link will explain partitioning/dual booting: [http://www.psychocats.net/ubuntu/partitioning](http://www.psychocats.net/ubuntu/partitioning)

Use the Live CD to see if Ubuntu likes all your hardware first, before you take the plunge.  Good luck and welcome to Ubuntu:)
