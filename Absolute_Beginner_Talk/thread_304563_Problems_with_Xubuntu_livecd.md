1) Have you tried the Live CD in another computer to see if the same problem occurs?

2) If it does, then either the CD, the burn, or the download is bad.

3) Did you [md5sum]("http://www.beeblebrox.org/hashtab/") the download?

4) Did you [burn]("https://help.ubuntu.com/community/BurningIsoHowto") the iso at 4x or less?

5) Did you check the CD to make sure it is good?

6) With your processor, I would use Xubuntu.
