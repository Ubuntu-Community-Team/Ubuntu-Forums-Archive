### Post by bogolisk on 2007-07-30
> **khris said:**
> What is* Wine*?

Wine an independent implementation of the windows system dlls. Wine implement Win32 API calls using linux/X API. It let's u run some windows program under unix/linux.

It can run things such as Photoshop 7, Photoshop CS2, ConvertXtoDVD, NeatImage, various Photoshop plugin. Usually it'd have more success with number-crunching programs (transcoding, photo processing, sound encoding, etc.) than hw control program (dvd burning, etc.)

Although, I've managed to upgrade my dvd-drive firmware using the windows program under wine.
