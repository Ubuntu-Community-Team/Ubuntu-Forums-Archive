### Post by zettberlin on 2008-01-08
> **chewit said:**
> From what I understand. Is that Ubuntu Studio is your Windows Media Center

Absolutely Not.

Ubuntustudio is a way to tune a standard Ubuntu for audio/video-PRODUCTION and to install a bunch of capable Linux-applications for such WORK.

Win-Mediacenter is for entertainment AFAIK no musician/multimedia artist would use it. The equivalent Linux-variant for mediacenter would be something like mythTV.

Ubuntustudio does not care much for MP3 or AVI usage or TV-viewing and it does not care at all for DVD-viewing or Gaming. 

If you install it on a modern PC with a decent soundcard(2-channel audiophile or 4,6 10 etc  Monotracks - NO 6.1-"home theater" cheapos), you have a system, that can be used to produce CD-ready music, broadcast-ready radiofeatures and motion pictures with real soundtracks.

On the other hand it is still your average Ubuntu: ready for office/internet and for entertainment and gaming also if you add some extra-stuff like a usefull version of libxine (not the Ubuntu-package that is), mediacodecs, 3D-drivers and some more that cannot be named here but downloaded from a french project.

I use Ubuntustudio to work with productions I primary make with 64Studio (another Debian-derivative, that is even more tuned for music) and for all my daily work. I use it with XFCE instead of GNOME and a lot of KDE-stuff and it works just great.

I agree, that the Ubuntustudio website is not as it should be. There was much discussion about the question, if there should be a central place for information and forums on the web for UbuStudio. The decision was made, to litter such infos all over the Ubuntu-webworld. Not wise but done so far....
