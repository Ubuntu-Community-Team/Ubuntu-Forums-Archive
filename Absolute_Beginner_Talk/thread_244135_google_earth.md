### Post by it.henrik on 2006-08-26
After you have downloaded, you must change the permissons of the file so it can be executed.

The GUI way its done like this:
Open the folder where you saved it with Nautilus (the File Browser).
Right clik on the file.
Click on properties and then the permissions tab..
Now mark the checkbox for Owner - Execute.
Close.

### Post by 3rdalbum on 2006-08-26
Good, you're learning to give more information when you ask for help.

Now that the file is executable, you need to open it in the terminal.

Let's say that you downloaded it to your desktop. You would open a terminal and type:

```

cd Desktop
./googleearth.bin

```

The first line changes the current directory to the Desktop folder. The second line runs the file "googleearth.bin".

See how that goes. I think Google Earth puts an icon in the applications menu?
