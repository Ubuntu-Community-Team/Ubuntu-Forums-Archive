### Post by Qrk on 2006-03-28
Sure, you can run it in any desktop environment you'd like.

Konqueror is a KDE app, so installing it over Ubuntu is going to install a lot of things you may not otherwise need, such as qt, kde-base etc. Also, as a KDE app it won't fit in with your gnome theme. 

But these are only cosmetic issues, it will run fine.
