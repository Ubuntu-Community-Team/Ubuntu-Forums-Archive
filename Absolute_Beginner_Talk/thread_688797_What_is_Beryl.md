### Post by emarkd on 2008-02-05
Beryl was a project that split off of Compiz.  They've combined back into one project, now called Compiz Fuzion.  You already have that installed if you have Gutsy.  If you want more control over your Compiz settings, search in Synaptic for CompizConfig-Settings-Manager and install it.

Synaptic is at System > Administration > Synaptic Package Manager, by the way.

### Post by emarkd on 2008-02-05
First, install the package I mentioned.  Then, enable advanced effects by going to System > Preferences > Appearance, click on the Visual Effects tab and select "Custom".  Then you can click the preferences tab to set your options.  There will also be a shortcut to those custom options at System > Preferences > Advanced Desktop Effects
