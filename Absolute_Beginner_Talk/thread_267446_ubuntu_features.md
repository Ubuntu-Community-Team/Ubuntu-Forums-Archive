### Post by aysiu on 2006-09-28
You can do all the basic things and a little more.

For example, I use Ubuntu for the following tasks:

1. Checking email (Thunderbird)
2. Web browsing (Firefox)
3. Ripping CDs (GRip)
4. Organizing MP3s (TagTool)
5. Listening to music (Rhythmbox)
6. Updating my website (Gedit)
7. Uploading to my website (FireFTP or FileZilla)
8. Organizing pictures (KRename)
9. Installing new software (Synaptic Package Manager)
10. Creating screenshot tutorials (Gnome-Screenshot)
11. Burning CDs (K3B)
12. Image editing (GIMP or Kolourpaint)

That's all I can think of for now. For most tasks, Ubuntu should suit your needs. If you need specialized programs like Dreamweaver, AutoCAD, Adobe Creative Suite (you design for a living, not just for fun), and commercial games, then Ubuntu will probably not suit your needs.
