### Post by iris-n on 2008-03-27
It would not be confortable, but your problems would be mainly with installation, i.e., you wouldn't be able to run the Live CD.

I dearly recommend using Xubuntu, which is a lighter version of the system, and using an Alternate CD, which uses a text version (tested personally, quite easy) of the installer, thus not demanding too much RAM.

Good luck,
