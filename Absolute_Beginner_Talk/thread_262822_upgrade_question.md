### Post by jcrnan on 2006-09-22
You can upgrade by using sudo apt-get dist-upgrade

But its reccomended to do a clean install when updating to the new version, altough it should work with dist-upgrade as well. If you want to start from scratch it can also be nice to make a own partition and put the home folder on (it saves you user configurations, etc)
