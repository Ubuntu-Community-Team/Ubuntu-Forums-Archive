### Post by Nikusan on 2006-05-27
Try Kopete, it works great for webcam on msn.
```
sudo apt-get install kopete
```

To check if your webcam is working, open Ekiga (Applications > Internet > Ekiga), skip the wizard, and then select the video tab. If you see  your camera's display then it works. :)
