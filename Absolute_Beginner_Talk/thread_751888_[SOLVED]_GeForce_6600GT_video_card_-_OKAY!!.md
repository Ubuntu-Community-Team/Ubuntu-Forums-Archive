### Post by anewguy on 2008-04-11
For anyone who had seen my previous posts on problems with this video card, the problem has been isolated.  It is NOT a problem with the video card - if you have one and are contemplating switching to Ubuntu, PLEASE DO SO!!!  Some of these posts are old now and no one has been following them for several days, so I wanted to post this thread and mark as solved so you know you will be fine using a GeForce 6600GT with Ubuntu!!

My problem happened to end up being an old (Biostar M6TZA) motherboard whose BIOS doesn't set up the video card correctly (possibly because it is only AGP 1.0).

The card worked great on a different motherboard ( a used one I bought that unfortunately shot craps about an hour after I installed it).
