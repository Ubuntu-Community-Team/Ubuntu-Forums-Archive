### Post by Sudo Sumo on 2008-01-26
There's a ton of e-books out there in PDF format if you're willing to hit up a few torrent sites (google searching torrent linux books ought to do it), but if you head on down to your local mega-bookstore and are willing to drop about $50 or so, you can pick yourself up a dummies-to-expert guide on ubuntu server.

I can appreciate where you are coming from, I just recently installed a LAMP server on my home network and added Samba support on it for file and print serving.  Thanks to forums like this, got it figured out.

If you are facing a time crunch, the best advice I can offer is to install a full (read: GUI) version of the same distro on another machine and piece it together; sometimes the automated config programs can move you along the learning curve and allow you to copy the resulting .conf files on over... just don't forget to be mindful of the permissions.

GOOD LUCK!
