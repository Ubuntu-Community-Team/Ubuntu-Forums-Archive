### Post by Chuck_W on 2006-09-23
A long time ago I had one of those. It was ok at the time but forget trying to run a modern OS on it. There are only 2 memory slots for 72 pin SIMM's which means that 64mb is the max. Also it had a kind of lousy motherboard design that actually slowed down some processes.  If it has been sitting unplugged for the last 3 years the PRAM battery is probably dead. Radio Shack used to carry them but is it worth $15-$20 on a machine that old? Ethernet cards are proprietary and hard to find. My 6300 went to the Great Recycler. You may want to do the same.

### Post by liquidtenmillion on 2006-09-24
Keep the PC.  It's not exactly weak.  However, you will not be able to run Linux on it.  I mean you would, but you'd have to use the 2.2 kernel, and run without X.(You could still use it as a server, or as a media player with a framebuffer).

You may be able to use a tiny variant of an x server and twm, but 48mb of ram is going to be a tough fit, and it certainly won't be fast.

However, Mac OS7 isn't that bad to begin with.  There are quite a few good apps for it.
