### Post by Hospadar on 2007-08-07
It may be that other apps are window-izing your game, like for example gaim will sometimes minimize or windowize a fullscreen app when it opens a new message window.  if this is the case, you might either disable the offending app or see if it has some option to open windows below other windows.

If that is not the apparent problem, it may be that some other application is doing it, but not opening obvious new windows.  Try closing extraneous applications that are running on the taskbar and in the system tray.

If you are running the games under wine, it may just be a bug with the game, in which case you might be able to find a fix for it, but maybe not.

Also if you are running beryl/compiz this might be affecting that, try switching back to metacity and see what happens.
