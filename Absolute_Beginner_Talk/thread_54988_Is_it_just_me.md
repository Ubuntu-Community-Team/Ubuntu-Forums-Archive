### Post by varunus on 2005-08-07
Well, one interesting thing to note, is that you don't actually need to refresh the gnome-panel usually.  Its there in the ubuntuguide as a precaution just in case the menu item isn't autodetected.  Most things in the ubuntu guide can be done with a GUI, but it saves bandwith to do everything from a terminal (they only have to post text on the website).

You can install multiple things at once; everything the ubuntuguide says uses the console "sudo apt-get install something."  Instead of this, go to System->Administration->Synaptic Package Manager.  You can select as many programs as you want to install from there, and then once you have selected everything, just hit apply and everything will install for you.  (Power of GUI.)

Another thing to note is that Linux does require tweaking.  That's part of its design; so everyone can have exactly what they want.  And once you tweak a linux system to work, it won't break unless you go doing something weird.  No random failures, no viruses, no spyware, and no updates that break your computer (Service Pack 2, cough cough).

Me, I like to tinker, so Linux is right for me.
