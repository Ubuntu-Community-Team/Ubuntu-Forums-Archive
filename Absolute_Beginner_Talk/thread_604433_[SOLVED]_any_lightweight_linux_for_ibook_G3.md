### Post by motokitn on 2008-03-13
I am currently running Xubuntu on my iMac Blueberry 350. It's a little sluggish, so I've been looking at alternatives. I have found:

Debian PPC - runs from an install cd and fetches packages over the net, or you can download several cd's and install from them. Sort of advanced.

Gentoo PPC - you can install from the cd or build a customized system from scratch- more advanced

Crux PPC - very advanced, you must build from source code. However, there is a disk on their homepage you can just install from but it's not as current.

None of these are as easy to install as Xubuntu PPC, but it's heavier and the last release was 6.10. Hope this helps somebody!
