### Post by zeeR on 2007-06-15
> **locke.dragon said:**
> i've got a few themes i want to uninstall.  where's the folder where they're kept?

I found that they can be on two folders...Do this, on the terminal:
```

sudo nautilus
```

Now, navigate to the folder /home/~YOURUSERNAME,  and tell nautilus to show hidden files. Go to the .themes folder, and delete what you want.

The other folder is /usr/share/themes, delete the themes you don't want from there too!

I don't know why Gnome doesn't keep all the themes in one folder.

Hope that helps!
