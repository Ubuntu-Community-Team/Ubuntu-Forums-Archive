If you want to create a swap manually and try again - go to GParted (GNOME Partition Manager) - in  the menu somewhere.  It is very straightforward, and essentially self-explanatory, but:

[http://gparted.sourceforge.net/larry/generalities/gparted.htm](http://gparted.sourceforge.net/larry/generalities/gparted.htm)

Gives a bit of advice on how to do things.

Of course, the Ubuntu AlternateCD or Xubuntu LiveCD would be reasonable alternatives if this doesn't work.
