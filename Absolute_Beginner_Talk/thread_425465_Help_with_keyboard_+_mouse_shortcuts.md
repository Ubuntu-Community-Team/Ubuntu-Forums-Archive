### Post by firefeather on 2008-03-18
Sorry to see that you've been waiting for a solution since August. My experience is with Gutsy's Compiz Fusion, so your mileage may vary. However, I think that Beryl's configuration manager is one of the big things merged into Compiz Fusion, so the approach should be almost the same.

This is how I'd do it: Go to the settings manager and find the place where you can change keyboard shortcuts for switching workspaces. Set the equivalent to "Change Workspace Right" as <Alt>Button4 which means alt+scroll up and "Change Workspace Left" as <Alt>Button5 which results in alt+scroll down. 

As for workspace switching; if keyboard settings for Beryl match up with the keyboard shortcuts in the System -> Preferences menu, then you should be able to set ctrl+alt+1/2/3/4 to get to the different workspaces there, if you can't find it in the settings manager.

I really hope this helps. Let me know if I can help you more on this subject.
