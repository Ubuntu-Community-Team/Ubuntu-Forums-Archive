### Post by davmac on 2006-09-04
Very easy to do. I'm sure there will be more detailed guides around, but here's what I do.

1) Prepare windows first by defragmenting the disk and then doing a full system backup.

2) Pop the Ubuntu 6.06 disk in and reboot

3) Double click the install icon on the desktop and follow the prompts.

The "scariest" part is where you have to partition the drive. Make sure you select the option to keep what you've got and reallocate some of the free space for Ubuntu.

...and that's it!

Hope this helps,

Dave Mac

### Post by Amablue on 2006-09-04
Also, I'd recomend making an extra partition for your files that you want both OS's to access. I think I heard there was a program released recently that let Ubuntu access the Windows side, but I set up my computer a little differently. 

I made three partitions, Windows (10 gigs), Ubuntu (10 gigs), and shared (35 gigs). I keep all my music and homework and stuff on the shared drive which makes both OS's more functional for me.
