### Post by Brunellus on 2006-09-18
NOTE:  thread title slightly altered to reflect actual content of help request.  In the future, please use the thread title to give an idea of the TYPE of help you need, so that users who know how to help can help you.

short answer:  

YES.

Longer answer:

The installation CD will take care of all the hard drive preparation for you.  If this will be DUAL BOOT computer--booting both Windows and Ubuntu-- a little extra care should be taken. For dual booting, follow these instructions:

[https://help.ubuntu.com/community/WindowsDualBoot?action=show&redirect=WindowsDualBootHowTo](https://help.ubuntu.com/community/WindowsDualBoot?action=show&redirect=WindowsDualBootHowTo)

If your router uses ethernet, you should be good to go.  Networking via USB (wired) is Not Ready for Prime Time (in *any* OS, as far as I'm concerned).
