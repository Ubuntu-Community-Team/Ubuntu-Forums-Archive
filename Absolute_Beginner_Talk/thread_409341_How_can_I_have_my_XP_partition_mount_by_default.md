### Post by brennydoogles on 2007-04-14

> Right now I can mount it by:

You could add it to your fstab by typing ```
sudo gedit /etc/fstab
``` 

Your fstab should look something like this:
```
# /etc/fstab: static file system information.
#
#  -- This file has been automaticly generated by ntfs-config -- 
#
# <file system> <mount point>   <type>  <options>       <dump>  <pass>

proc /proc proc defaults 0 0
# Entry for /dev/hdb1 :
UUID=6f7e9e85-2804-4047-90c3-7c283e15eb6a / ext3 defaults,errors=remount-ro 0 1
# Entry for /dev/hdb5 :
UUID=9e7518a5-123e-4953-af40-ec7327132458 /home ext3 defaults 0 0
# Entry for /dev/ !! UNKNOW DEVICE !! :
UUID=d2d4beaf-6d06-4f74-84b3-aef9f1552dbe /media/hda6 ext3 defaults 0 0
# Entry for /dev/hdb6 :
UUID=bec95c15-749a-47c7-b421-1f4a88980f6c none swap sw 0 0
/dev/hdd /media/cdrom0 udf,iso9660 user,noauto 0 0
/dev/hdc /media/cdrom1 udf,iso9660 user,noauto 0 0
/dev/ /media/floppy0 auto rw,user,noauto 0 0
/dev/hda2 /mnt/hda2 ntfs-3g defaults,locale=en_US.utf8 0 0
/dev/hda1 /mnt/hda1 ntfs-3g defaults,locale=en_US.utf8 0 0

```
and you would add to the bottom
```
/dev/sda2 /mnt/windows ntfs-3g defaults,locale=en_US.utf8 0 0
```
or something similar to this.
