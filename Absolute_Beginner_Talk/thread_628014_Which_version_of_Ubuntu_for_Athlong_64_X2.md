### Post by Dr Small on 2007-11-30
You can get the x64 if you want, as your system can handle it, but as a beginner, I would suggest using the x86. I am even using it with a 64bit processor.

It is really up to you, but the x64 is a little bit more difficult to install Flash and stuff, but it can be done.

### Post by avik on 2007-11-30
> **Dr Small said:**
> It is really up to you, but the x64 is a little bit more difficult to install Flash and stuff, but it can be done.

Flash is easy to install even on amd64 starting from Gutsy, though Java applets don't work for me as of yet.  Depending on your needs, you can either the standard or the amd64 version, though both are more or less equivalent.  The amd64 version may be slightly faster, and it allows you to have more than 4GB RAM.  That probably doesn't affect you though.
