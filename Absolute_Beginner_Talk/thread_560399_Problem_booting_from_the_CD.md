### Post by Ozeuss on 2007-09-26
It is probably a problem with either the cd or the iso file. refer to[ this guide ]("https://help.ubuntu.com/community/HowToMD5SUM"), to check if the iso downloaded correctly. If yes, burn the cd again at slow speed. You also have the option of running a cd check at the ubuntu boot screen.
Enjoy, Ubuntu is a wonderful OS!

### Post by alienexplorers on 2007-09-26
