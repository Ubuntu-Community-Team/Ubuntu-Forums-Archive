### Post by bbqsandwich on 2008-01-27
Why don't you set up a Windows/Ubuntu dual-boot? Then you can use Ubuntu when you want, and if you find out you must use IE or MS Office for something work-related, you can switch over to the Windows partition.

Also if you decide Ubuntu is not for you, it is easy to remove (though you must be careful to do it correctly, esp. if you have Windows Vista, I believe) and then you are back to where you were.

The Live CD is nice but sometimes there are some things you only get once you have installed Ubuntu on at least part of your hard drive - in my case, a working wireless card, for example.
