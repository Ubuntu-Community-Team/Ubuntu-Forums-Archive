### Post by wieman01 on 2006-08-11
If you don't mind deploying "ndiswrapper" then I can recommend **"Linksys WUSB54G V4"** (version number is important!). Works great with all levels of encryption & authentication (WEP, WPA, WPA2).

The flipside of the coin is that you have to use the latest native Windows driver in connection with "ndiswrapper" as mentioned above. But that's a minor hassle if you ask me.
