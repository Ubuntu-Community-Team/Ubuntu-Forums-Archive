### Post by viniosity on 2006-09-10
Postfix is a Mail Transport Agent (MTA): [http://www.postfix.org/](http://www.postfix.org/)

It's useful and needed if you are planning to use your ubuntu box to send mail on your behalf, but it's not usually required unless you are running a website from your computer.  (It does not affect you using an external mail service like yahoo, hotmail, gmail, etc or using an external imap or pop3 service)

Still, if it's crapping out you should investigate why and try to fix it.
