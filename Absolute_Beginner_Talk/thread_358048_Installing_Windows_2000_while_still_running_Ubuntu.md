### Post by mikewhatever on 2007-02-10
Windows will overwrite GRUB in the MBR, so that you will not be able to boot into Ubuntu. You'll need to reinstall GRUB from a Live CD, or from Super Grub CD. [Here is the guide]("https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows?highlight=%28windows%29")
