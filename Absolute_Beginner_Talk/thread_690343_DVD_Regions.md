### Post by MonctonJohn on 2008-02-07
According to the documentation regionset will only work 4 times then the region code will remain on the last region it was set to. This is not a limitation in the software however this was designed into the hardware to only allow you to set the region so many times. It is the industy's attempt at keeping people from watching movies in different regions.

I know... not very fair but we have to live with it.
