### Post by schorsch on 2007-08-24
Debian is a linux distribution like Ubuntu (which is debian based) whereas KDE is a desktop environment inside of the distribution. You can get KDE on many distributions but you cannot get any distribution based on KDE.
Ubuntu and Kubuntu are both Debian based, the only difference between them is the desktop environment:
Ubuntu uses Gnome
Kubuntu uses KDE.
