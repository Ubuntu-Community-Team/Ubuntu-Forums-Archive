### Post by PartisanEntity on 2007-11-04
Try booting Ubuntu in recovery mode and issue this command to manually reconfigure your xorg.conf file:
```

sudo dpkg-reconfigure xserver-xorg
```

Remember to back up your xorg.conf file beforehand though with:

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup
```
