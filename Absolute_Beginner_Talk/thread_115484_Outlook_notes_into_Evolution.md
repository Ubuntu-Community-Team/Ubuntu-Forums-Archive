### Post by nocturn on 2006-01-12
If you go to outlook-notes, and select all your notes.  Hit File -> Save as, select .txt and save the file.

You now have a text file like this:
Modified:	do 12-01-2006 09:26

test
Modified:	do 12-01-2006 09:27

test2
test

Which are the two notes I created to test.
