### Post by Wolki on 2005-09-24
[QUOTE=aftermath]Are these compliers not the default options for Ubuntu? If no, how can I install them?[/QUOTE]

Ubuntu doesn't include compilers by default, since most people don't have to compile stuff. You get the most important things if you install build-essential, available from synaptic or "sudo apt-get build-essential".
