### Post by RomeReactor on 2007-12-12
Hi. If you're using Desktop Effects, you could try disabling **Application Switcher** in compizconfig-settings-manager in favor of **Shift Switcher**; if you don't have compizconfig-settings-manager installed yet:
```
sudo apt-get install compizconfig-settings-manager
```
You would then find it in "System->Preferences->Advanced Desktop Effects Settings". Launch it, go to the Window Management section, uncheck **Application Switcher** and check **Shift Switcher**. I know this isn't a direct solution to your issue, though.

Hope this helps.
