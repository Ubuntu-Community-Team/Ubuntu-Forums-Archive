### Post by annda on 2007-03-26
i suppose you were using word on windows.

the only thing you can hope for is restoring the last auto-saved file, NOT the final version. recovery programs won't help, because the file was NOT deleted, just overwritten.

use the search function to find any files containing '~' in the filename. make sure to check the options for hidden files and system directories.

i haven't used word for ages, but i vaguely remember it automatically saves temp files and they have the ~ in the name.

but really, you will get more info if you google for your problem - lots of people experience it - until they learn (like me) to save a file right after i finish a sentence.

### Post by NicoleM on 2007-03-26
I'm not sure how long autosaved versions of files are saved for unfortunately, but searching for stuff with a ~ in the name seems to be on the right track.  I think these "~" files only exists while the document is open as that's the only time it's autosaving so you may be able to pull these files up in your recovery program, but that's obviously hit or miss.  

I know you probably realize this now, but any document that could cause you to have enough of a "panic attack" when deleted to hit yes to save changes should probably be backed up somewhere :(  hindsight is always 20/20 though right?  Also "edit>undo" = my savior for accidental edits (which it seems like this was).

unless someone has developed a utility specifically to counter this issue, I can't think of any windows tricks off the top of my head to help.  google would be the pest place to check to see if a utility that exists to retrieve previously saved versions of word files.
