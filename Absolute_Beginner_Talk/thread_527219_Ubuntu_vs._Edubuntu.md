### Post by Seisen on 2007-08-16
You can install Edubuntu as a stand-alone client, I dual-booted it with Windows so my kids could play the games and so I didn't have to worry about Windows getting messed up. If the administrators like what they see then you look into setting Edubuntu up as a server and the other computers as thin clients.
