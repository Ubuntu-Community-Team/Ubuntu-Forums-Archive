### Post by jvc26 on 2007-01-17
By the sound of it could be one of three things:
1/ A bad download, do the md5 checksum to confirm that its all good. If it isnt you will need to re-download, try using a torrent as that reduces the chances of download errors.
2/ A bad burn - there is a fault in burning (plausibly as you were using too high a burn speed, try again at 4x probably to minimise the chance.
3/ You didnt burn the image to the disc as an image but as a file. You need to reburn the disc as an image not a data file :)
Hope that helps.
