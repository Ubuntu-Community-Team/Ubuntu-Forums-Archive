### Post by z-vap on 2008-02-02
I have this happen occasionally.  

What I do is when I'm back in my normal viewing mode (just at the desktop, no wine programs running) and I am certain I have the resolution set properly (ie 1280x1024), I use my  monitor buttons to correct the overscan that it's currently doing.  

Specifically my monitor has an AUTO button that will auto-adjust the Horizontal and Vertical positions and size for me.  Other monitors may have this.  If not you will have to adjust it using the button controls.
