### Post by pappapump on 2007-06-03
You certainly can.
Make sure your XP is completely installed then set your bios to boot from CDROM first.
When that happens make sure your Ubuntu live-CD is in the drive.
Follow the defaults to install by clicking the install icon on the desktop.
Best scenario when it comes to partitioning is not to use the entire drive, but divide the partitions equally or with more on the windows side since its such a drive hog.
After install you will be given the option to either boot into Ubuntu or boot into XP.

### Post by Tyke91 on 2007-06-03
BIOS is the screen that will come up if you hit the button for setup on startup (you know... it'l say press XXXX for setup... do that)

Grub boot loader is installed by ubuntu and lets you switch between Operating systems.


ALWAYS INSTALL THE NON UBUNTU OS FIRST or else the grub loader will be deleted and you will not be able to switch.


you will need to restart your computer and use grub to get onto windows (grub comes up automatically) this is how you will play your games.
