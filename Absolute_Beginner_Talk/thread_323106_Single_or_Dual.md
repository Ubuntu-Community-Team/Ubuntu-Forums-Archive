The last time I did the check it took a very long time. You could just boot up and see if it works in stead of it takes too long. 

It doesn't matter if you dual boot. You only boot one OS at a time, the other one has no influence other than taking up hard disk space.

You can use your preferred partitioning tool, but the liveCD comes with it's own partitioner you can use to wipe the disk clean and install Ubuntu on it (or to set up a dual boot if you choose to do so). I prefer creating a separate /home  partition (similar but not the same as /My Documents in Windows), it makes it easier to reinstall if you ever have to.

Read [this]("http://www.psychocats.net/ubuntu/installing") for more info.

Have fun.
