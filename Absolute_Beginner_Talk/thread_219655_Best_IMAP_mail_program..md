If your mail provider offers IMAP, then you can run Evolution in Linux and XP (I don't know about Vista) and your mail for both programs is stored on the mail server and is accessed from Evolution (windows installation package and separate Linux installation package).  However, if you have an existing offline store of mail, then you'd have to place it on a disk segment - possibly FAT 32 - usable by both Windows and Linux.  There are people here who can tell you more about that than I can.
