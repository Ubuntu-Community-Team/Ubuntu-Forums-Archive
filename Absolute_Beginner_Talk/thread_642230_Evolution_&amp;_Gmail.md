### Post by bazzbc on 2007-12-16
I had a similar problem with Thunderbird/Swiftdove , it started about 3 months ago and nothing I tried including creating a new user profile seemed to solve it.

What I did was change from using pop3 to IMAP. I'm pretty sure that evolution supports IMAP. It does mean reconfiguring your settings and downloading all of your mail again. I had about 500 MB and it took ages. But since then I have had no login problems.

The main advantage of IMAP (for me) is that you can set it up on more than one PC which is really useful if you use a laptop and a desktop on a regular basis and want all of your mail on both devices.
