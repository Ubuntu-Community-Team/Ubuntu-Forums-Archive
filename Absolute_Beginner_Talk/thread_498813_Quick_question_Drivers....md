If you plan to install Ubuntu and get rid of XP, you will not need the drivers anyway except maybe the wireless one (to be used by ndiswrapper- in case your wireless device is not supported by Ubuntu).

Dont worry about all that now. In short, your XP drivers would be of no use in Ubuntu anyway. So I guess you are good to go.

All drivers / applications for Ubuntu are mostly free. There are very few proprietary software...and even those have alternatives most of the time.
