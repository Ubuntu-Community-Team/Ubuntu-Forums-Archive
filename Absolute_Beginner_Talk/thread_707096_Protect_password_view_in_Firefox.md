### Post by mdsmedia on 2008-02-25
The only way I can think of is that you only have to input the master password once per session. Once input it remains current for that session. If Firefox is closed down, however, the master password would have to be input again to access the saved passwords.

On the other hand if you don't have a master password the passwords can be used, and can be seen if the users go looking for them.

If you don't trust them with the passwords, why trust them with the access?
