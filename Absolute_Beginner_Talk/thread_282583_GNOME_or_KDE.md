### Post by aysiu on 2006-10-23
As others have pointed out, what you've just asked makes about as much sense as "Which is better--Coke or Pepsi?"

This will explain some of the differences (may not be 100% accurate as of Edgy Eft--to be released this Thursday, but it'll still give you a basic idea of the types of differences): [http://www.psychocats.net/ubuntu/kdegnome](http://www.psychocats.net/ubuntu/kdegnome)

An amendment to tubasoldier's recommendation, if you install KDE in addition to Gnome, please use *aptitude* instead of *apt-get*. Full instructions, with screenshots, are here: [http://www.psychocats.net/ubuntu/kde](http://www.psychocats.net/ubuntu/kde)

In terms of graphical installers for Linux programs, you can't get better than Synaptic Package Manager, which comes standard with Ubuntu. Instructions for how to use it:
[http://www.monkeyblog.org/ubuntu/installing](http://www.monkeyblog.org/ubuntu/installing)

### Post by JayTee on 2006-10-23
Why not both? You can install Ubuntu with Gnome or Kubuntu with KDE and later add the other desktop and switch between them. Just do a sudo apt-get install ubuntu-desktop or sudo apt-get install kubuntu-desktop for whichever desktop you don't have and when you logout you can click the session button on the login screen to switch desktops. It uses more disk space but with hard drive prices what they are you can always add more for cheap.

I think that you need to get kubuntu sources
```
sudo nano /etc/apt/sources.list
```then add these lines at the bottom
```
# Automatically generated sources.list
# http://www.ubuntulinux.nl/source-o-matic
#
# If you get errors about missing keys, lookup the key in this file
# and run these commands (replace KEY with the key number)
#
# gpg --keyserver subkeys.pgp.net --recv KEY
# gpg --export --armor KEY | sudo apt-key add -

# kubuntu.org packages for the latest KDE version (packages, GPG key: DD4D5088)
deb http://kubuntu.org/packages/kde-latest dapper main

# kubuntu.org packages for the latest KDE version (sources, GPG key: DD4D5088)
deb-src http://kubuntu.org/packages/kde-latest dapper main
```
