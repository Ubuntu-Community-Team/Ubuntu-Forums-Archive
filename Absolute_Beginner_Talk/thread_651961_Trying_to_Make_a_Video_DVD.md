### Post by bone2006 on 2007-12-28
I know you asked about another program, but I use DeVeDe to create a video DVD and just starting playing.
It opens, I select Video DVD, then select add from the top right and add your file and I leave everything else alone and click NTSC for north american then ok and take a checkmark off anything about menu and create ISO file ready to burn.

I open the ISO file with VLC to play it before burning it to make sure it's right.  If everything is ok I right click and select burn.  
It's the way I get some xvids on a DVD so they just play when you pop it in a DVD player without any menu
