### Post by erginemr on 2007-12-26
Based on your configuration, I believe  your computer can handle Ubuntu Gutsy (7.10) and the Compiz Fusion goodness. Although Dapper Drake (6.06) is renownfor being rock solid, each new version of Ubuntu gets better than before in terms of new features (such as NTFS writing) and supported hardware. 

Besides, you will want to use the latest software in, say, web development and the latest Debian packages of those software are only supported in Gutsy (with the exception of backported software). Linux software are developed at a pace difficult to follow, and each new version require installation latest library files, which are not compatible with Dapper and will break Dapper's instrinsic Gnome library system if force-installed. So eventually, you will have to upgrade anyway.

So, I definitely suggest installing Gutsy Gibbon.
