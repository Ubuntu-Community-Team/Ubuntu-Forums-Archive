### Post by ugm6hr on 2007-07-30
[http://neosmart.net/wiki/display/EBCD/NeoGrub](http://neosmart.net/wiki/display/EBCD/NeoGrub)
Follow the instructions there, and when you come to the NeoGrub Menu File section - make the C:\NST\menu.lst (assuming Vista is on C:\) look identical to the previous menu.lst auto-generated by Ubuntu (without the Windows entry).

To do this, it will be easiest if you reboot with the Ubuntu LiveCD, and copy the file /boot/grub/menu.lst (from the hard drive - e.g. /media/Disk1/boot/grub/menu.lst - if the Ubuntu partition is seen as Disk1 - not the CD!) to a USB stick.  Make sure you save it as a .txt or .doc file.  Then you will be able to open it from within Vista to copy and paste the section between (just paste it to the bottom):
> ## ## End Default Options ##
### END DEBIAN AUTOMAGIC KERNELS LIST

Hopefully that should now work.  

Be aware that when you update Ubuntu with a new kernel - the new kernel will not necessarily be available from the boot options though.  If you want to use the new kernel - you will have to manually add the extra lines again by following the same advice for NeoGrub Menu File section.
