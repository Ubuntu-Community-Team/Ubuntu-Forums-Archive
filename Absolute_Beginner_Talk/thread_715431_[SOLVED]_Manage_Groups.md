### Post by neurostu on 2008-03-04
Many different programs on your computer have their own group. this is mainly for security reasons.... I would highly recommend that you not worry about that.  

I remember when I would fresh install Win XP I would began the task of removing bloat ware. Ubuntu doesn't have any of this. Unlike windows, everything that comes with the basic Ubuntu installation is there for a specific purpose. Granted there are some applications that not everybody uses but I would leave them alone.

I can't tell you how many posts I've read where people have said " I just installed ubuntu and I tried to remove software "XYZ" and now my pc is broken!!" most programs only take up a few megabytes so unless you are REALLY pressed for space there is no reason to uninstall them.

Sorry about the rant... back to your question...

So the easiest way to find out about a specific group would be to google it, but again any group that you aren't sure about I would leave alone.
