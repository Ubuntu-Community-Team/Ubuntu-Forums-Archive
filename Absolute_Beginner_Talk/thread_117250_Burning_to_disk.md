### Post by me3ohm on 2006-01-14
Hi,

Are you just burning the ubuntu.....iso file to the disk?  The reason I ask is that you need to use the "Burn image to disk" wizzard in Nero to make the disk useable.

If you use Nero Express, on the main wizzard that shows on startup - look around the menus for "Burn/Write Image".  If you use the full Nero package, look for that option on the menus.

The .iso file you have is a disk image and needs to be burned correctly.

Let me know how you get on.
