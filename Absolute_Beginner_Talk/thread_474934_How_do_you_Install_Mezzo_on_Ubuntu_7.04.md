### Post by justin whitaker on 2007-06-15
> **dylan623 said:**
> I took a look at SymphonyOS, and I have to say that I am very interested. However, I don't want to install a whole new operating system, so I'm looking for a way to just install Mezzo on Ubuntu. The guide at [http://fak3r.com/?p=44](http://fak3r.com/?p=44) doesn't work. Can anyone help?

The install actually picks up with metatheme. 

1. Install metatheme (the instructions look about right).
2. Copy over the config files for metatheme. 
3. Add FVWM to your install. 
4. Mount the symphonyOS disk, and dump it to a folder.
5. Copy the mezzo and orchestra somewhere.
6. Copy the configuration files as directed, but edit them for the location you put mezzo and orchestra.
7. Add mezzo to GDM. The directions are similar to the tutorial to Fluxbox [here]("http://fluxbox-wiki.org/index.php/Howto_add_fluxbox_to_gdm"). 

Good luck.
