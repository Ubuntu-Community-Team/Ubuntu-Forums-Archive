### Post by vpalexander on 2008-07-21
> **iClouseau88 said:**
> Another question if you don't mind.  How do I install an icon or a menu item for it on the desktop?

On an empty area of the GNOME Desktop you can right-click and Create Launcher. Supply a name (whatever you like), the program name without any path, and finally an optional description or comment. This places a Windows-like shortcut on the desktop.
