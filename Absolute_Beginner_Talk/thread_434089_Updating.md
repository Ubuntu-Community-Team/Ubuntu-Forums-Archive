### Post by meng on 2007-05-05
Note that if you previously had 6.06 fully updated, then there's no point installing 6.06.1
If, however, you want a clean install for other reasons (something on your current system is borked beyond repair), then 6.06.1 is the way to go, as it will reduce the number of updates required post-install. Mind you, many have complained that there are still too many updates and that 6.06.2 should be released.
