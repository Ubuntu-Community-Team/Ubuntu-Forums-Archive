### Post by GGLucas on 2007-12-11
gedit can be completely transformed to be a web developer's wet dream (IMHO), it's light weight, fast, has very good syntax highlighting (dunno about ruby on rails, though), and a load of plugins that give you various extensions (like a file browser tree pane that has direct access to all your nautilus FTP bookmarks).

[I used this guide]("http://www.micahcarrick.com/09-29-2007/gedit-html-editor.html") to customise it to work nicely, and I've been using it for PHP/HTML/CSS/Javascript as well as Python. I attached a screenshot of how it looks for me.

Another good IDE you might want to try is BlueFish, it's nice, but nothing beats gedit :P
