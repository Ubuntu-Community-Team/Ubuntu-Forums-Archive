### Post by MighMoS on 2008-02-10
Each release with two exceptions are supported for 18 months. The release number is when it was released, so 7.04 ("Fiesty Fawn") was released in April, 2007 while 7.10 was released in October 2007. 

The two exceptions were 6.06 (released June 2006) and the upcomming 8.04.  These are considered "LTS" (Long Term Support) releases and will be supported for 3 years.

Upgrading is very easy.  When a new release is released update manager will notify you that a new version of Ubuntu is out, and you can choose to update -- at that point you may be asked a few options for some new settings, but otherwise it works like any other update.  However, I'm not sure "downgrading" to a previous release is supported at all.

Go with the latest release 7.10.
