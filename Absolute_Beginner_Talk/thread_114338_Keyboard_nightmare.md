### Post by jamb on 2006-01-08
Hi,
check that you have the correct keyboard layout. I guess it would be 105-keys (Intl).
When you have configured it correctly you should be able to type the country specific charcters:
- in any editor window
- at the console window

To check your keyboard layout, pick 'System Settings' from the menu, and then pick  
'Regional and Accessibility' and finally the keyboard laout option. I'm running Kubuntu Breezy (KDE desktop). I hope that you can follow my direction in the case you running with Gnome.

Cheers
