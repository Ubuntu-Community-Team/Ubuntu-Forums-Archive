### Post by Paul820 on 2007-11-13
To install a program with wine, just right click on the .exe file and select properties then **open with**, then click **add** at the bottom. Look for wine in the list, if it's not in the list use the little arrow where it says **use a custom command** and type **wine**. Close the dialogue and double click the program and it will start the installer. You only have to do this once as any other .exe files you try to run will automatically be assigned to wine. 

You program will appear in the menu under Applications->Wine->Programs
