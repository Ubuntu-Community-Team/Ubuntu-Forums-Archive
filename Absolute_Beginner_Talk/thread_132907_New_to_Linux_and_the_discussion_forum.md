### Post by Bartender on 2006-02-19
Ant -
Take a good read thru the first posts in The Beginner's Forum.  Look at Beginner's Rules, Read this before Posting.
I've probly violated most of the rules, which is not good.  But the Ubuntu Forums are among the more forgiving.  
This really is a community, with a broad, well-lit Main Street where many folks reside who want to help.  Then there are some back streets where cranky old Linux users lurk, waiting to bite your head off if you should have the temerity to wander down their alley.
Unless you have a degree in Unix, you're going to have questions.  Try to google your questions, check out the Wiki, type some keywords into "Search", etc. before posting.  The Private Message function can be handy if you find someone willing to discuss newbie stuff off the forum.
This is a well-equipped forum.  Somebody bought the Cadillac version.  The Search functions, Tools (for such things as printable versions of a thread), etc. are all pretty nice.
