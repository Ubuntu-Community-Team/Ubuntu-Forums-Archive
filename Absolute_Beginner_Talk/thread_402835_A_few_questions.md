### Post by jem7v on 2007-04-06
1. As long as your computer can boot from its CD-ROM (the option is in your BIOS) it'll be fine.
2. The GCC (Gnu C Compiler) is actually installed by default, and there are MANY free IDEs in the Add/Remove Software list.
3. If you have really strange hardware that isn't well supported it might take a couple days of tinkering to get everything to work.  On the other hand, if the hardware is well supported it might only take 40 or 50 minutes to install and set everything up.
4. You'll almost certainly be able to use Ubuntu, but in a similar vein to what mikewhatever said, [http://distrowatch.com/dwres.php?resource=major](http://distrowatch.com/dwres.php?resource=major) talks about what they consider the current major distributions.  Personally, I'd stick with one that uses the Gnome Desktop, because it is (in my opinion) more fully developed and stable than the other desktop environments available, and less resource intensive than KDE.
