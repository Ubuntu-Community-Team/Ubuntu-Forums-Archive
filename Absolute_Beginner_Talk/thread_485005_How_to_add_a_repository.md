### Post by supersonicdarky on 2007-06-26
[quote=Google]A repository is a central place where data is stored and maintained. A repository can be a place where multiple databases or files are located for distribution over a network, or a repository can be a location that is directly accessible to the user without having to travel across a network. [/quote]

repositories can be added through the **synaptic package manager > Settings > Repositories > Third-Party Software > add**. usually its just one line of text such as ```
deb http://www.virtualbox.org/debian feisty non-free
```
