### Post by Madpilot on 2006-10-27
You don't need to; Ubuntu is set up by default to run with sudo & your own user account/password.

[https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo) will give you more information, and even tell you how to set up a 'real' root account if you really want to - and tell you why it isn't needed or recommended...
