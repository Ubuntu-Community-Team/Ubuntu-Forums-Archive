### Post by Gen2ly on 2007-07-18
what does /dev/cdrom or /dev/dvd point to?

I had an issue not too long back where something was broke and an update repaired it.  That probably isn't much help, but I was able to burn CD's via commandline.

```
growisofs -Z /dev/hda -lrJ /path/to/files
```

If it already has a filesystem on it you'll see an error like this:

*WARNING: /dev/hda already carries isofs!*

you'll need to reinitialize the filesystem:

```
growisofs -Z /dev/hda=/dev/zero
```
