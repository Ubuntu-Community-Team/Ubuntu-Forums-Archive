#1) You gotta give things some time, lots of people go through each thread and even if it gets bumped down a page or so you will likely get a response

#2) Check your WINE configuration (Applications -> Wine -> Configure Wine), in the Display tab there will be a checkbox for 'Virtual Desktop' - make sure it is unchecked to allow games to go full screen.
