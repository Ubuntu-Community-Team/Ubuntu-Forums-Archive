### Post by neotasic1 on 2007-11-22
HI all, I have upgraded my desktop to Gutsy using a clean install with a separate home partition. Got pretty well everything working, but I do have a question for the knowledgeable.  When I had Feisty installed with Beryl, it gave the option of actions associated with moving the mouse to the screen edges.  This was a very useful function especially with the cube desktop as moving the mouse to the top right corner of the screen displayed all the open windows on all the desktops, and selecting the window moved the cube to the desktop concerned and gave the selected window focus.  Likewise, by moving the mouse to the bottom left corner of the screen all the windows on the current desktop were displayed and selecting one brought it to the top and gave it focus.  This was really quick and neat, but seems to be missing in Compiz Fusion. Can it be retrieved or is it a casualty of the fusion of the Beryl and Compiz projects.  Does anyone know who to contact to see if this functionality either exists or can  be put back into Compiz Fusion?

Thanks again.

### Post by bromix on 2007-11-22
Yes, you can.  Do you have the compizconfig-settings-manager package installed.  After this you can access the config manager via System==>Preferences==>Advanced Desktop Effects Settings.   From there you can configure the "Scale" option under Window Management  heading.  Choose it...choose the "Actions" tab.  And configure your edge by clicking the "Screen Edge" column.   You can configure other actions the same way and assign them to the edges you want for them as well.
