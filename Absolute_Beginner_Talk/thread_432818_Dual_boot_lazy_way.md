### Post by David4 on 2007-05-04
Hi, want to share my solution to the dual boot.
I did not want to make changes on the MBR file because I am computer challenged did not want to repair it for any problems may develop.
I purchased to Mobileracks (Removable trays) from Newegg, selected the cheapest since was not sure it will work.
Have XP installed on one HD and Ubuntu latest ver 7.4 installed on a second HD.
Removed both HD from the internal desktop case and reinsalled each one on its own Mobile rack, with each one independent power/IDE cable.
I can now select what OS system I want to use and powerup the computer.
Looks it may be the solution for the use I need.
Any comments or suggestions to improve this alternative appreciated.
