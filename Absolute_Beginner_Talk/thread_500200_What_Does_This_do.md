### Post by Malibu Illusion on 2007-07-13
nautilus-gksu adds a context menu option into Nautilus file manager that allows you to open things as root.

nautilus-image-converter adds a "Resize Images..." menu item to the context menu of all images

---

### Post by Rocket2DMn on 2007-07-13
nautilus-gksu gives you the ability to act as root from nautilus (the system file/folder browser) rather that having to type "gksudo" in terminal to run a program with superuser privileges.
nautilus-image-converter looks like an extension to nautilus that allows for mass photo manipulation.

the "sudo apt-get install" downloads and installs these extensions.
