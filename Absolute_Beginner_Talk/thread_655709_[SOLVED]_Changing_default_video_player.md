### Post by t0p on 2008-01-01
I've  installed VLC, which I want to use as my default movie player.  But whenever I insert a DVD into my computer, totem starts to play the movie.  How do I go about changing this?  I have got some flash video files on my hard drive, and I was able to right-click on them and change default player in Properties to VLC - but how do I do this for DVDs?  If I right-click on the DVD icon on the desktop, Properties does not contain the option for changing default player.

I thought it might be under System > Preferences > Preferred Applications, but it's not.

---

### Post by shad0w_walker on 2008-01-01
You can change the command used when a DVD is inserted from Removable drives and media

Find it in:  System > Preferences > Removable drives and media

The setting for DVDs is in the Multimedia tab.
