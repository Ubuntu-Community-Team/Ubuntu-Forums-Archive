You can go to System > Administration > Synaptic Package Manager and search for programs to install.

If you know exactly what you want to install, you can use this command (Applications > Accessories > Terminal)

Type in:

```
sudo apt-get install packagename
``` 

Where packagename is the name of the package to install. You will be prompted for your password. Type it in (no *'s or anything will appear in the terminal) and hit enter, and it will install it for you automatically.

If you want something more similar to .exe's like in Windows, .deb files are what you're looking for; [http://www.getdeb.net](http://www.getdeb.net) is a good site, and so is [http://packages.ubuntu.com](http://packages.ubuntu.com)

I'm not on Ubuntu right now, so the menus that I listed (like the System > Administration stuff' may not be 100% correct, but they should at least be really close.
