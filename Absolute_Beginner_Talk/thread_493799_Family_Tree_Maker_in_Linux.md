### Post by macogw on 2007-07-07
I don't think there is a way to export with multimedia between different formats whether for PAF, GRAMPS, FTM, or anything else.  

GRAMPS is the best genealogy program on Linux.  I love it. Got a professional genealogist all jealous of its reports ;)  

Only other thing I can suggest is if you wanted to install an Apache2 server on your computer and install PHPMyGedView which will let you work through Firefox on your stuff as if it's a website.  You can use it online as a website too (I do so on yourbeginnings.com).  It only takes GEDCOM though.
