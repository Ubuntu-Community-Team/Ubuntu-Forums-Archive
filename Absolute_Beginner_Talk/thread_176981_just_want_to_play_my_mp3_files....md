### Post by Kvark on 2006-05-15
Don't ever use a file in a lossy format as source for a new file. mp3 to ogg and even mp3 to mp3 (for example a remix) will become double lossy which is bad. mp3 to cd audio will still be lossy but in a much larger file which is a waste of space and if you re-rip then it becomes double lossy.

So let the files that already are mp3 stay mp3. If you want them in another format then rip them from the CDs again. Personally I'd let them be and just use ogg for any new rips (unless I had an mp3 player without ogg support in which case ogg wouldn't be an option). Ogg is lossy too so it's not much different from mp3 anyway. Most say ogg sounds better but the difference is small so it could perhaps be imagined.

Flac is another open source audio format and flac is lossless. So flac files are true CD quality and thus just as good sources for conveting to other formats, making remixes and whatever as the original CD. But they are a couple times bigger then lossy files.
