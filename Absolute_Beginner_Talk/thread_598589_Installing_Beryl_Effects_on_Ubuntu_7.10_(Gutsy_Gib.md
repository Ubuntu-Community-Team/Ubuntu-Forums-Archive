> **eldara5 said:**
> Ive used Ubuntu before, and i liked it. Now ive come back for more i previously used Feisty but now ive got Gutsy. But i was wondering how can i install Beryl/Fusion on Gutsy all guides seem to be for Feisty

On Gutsy, just:

> sudo apt-get install compiz compizconfig-settings-manager 

Will make an Entry in your menu.

System>>Prefs>>Advanced Desktop Effects.

OR

System>Prefs>Appearance>>Visuals Tab
Then enable your level of Visual Effects.
