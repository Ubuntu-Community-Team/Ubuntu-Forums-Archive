### Post by nickpaton on 2006-11-20
When you run the cd, and you come to the menu choices, press f6 for Other Options.

A line of command-like text will appear, and you will not see the end of it.

simply type:

```
noapic
```

and this will be added to the end.

this should speed up your install speed considerably, but let us know if it doesn't
