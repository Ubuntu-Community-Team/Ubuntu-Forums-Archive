### Post by rockin_goliath on 2007-12-26
Happy holidays everyone! I thought I'd give you all a present: my first bash script! It installs all the things that I usually do when I install Ubuntu. It adds the Medibuntu repository and then installs Flash, Micro$oft fonts, Java, the totem-xine backend, commercial dvd playback, and all the other media codecs. It also installs Gnome Partition Editor and Advanced Desktop Effects Settings. The script does not contain any malicious code, but you can look it over yourself if you want. It is written for Gutsy, but to modify it for any other version, just change the Medibuntu repository section to the appropriate information for your version (as stated on the Medibuntu website). To run the script, download the script to your desktop, then at a terminal type:
```
cd ~/Desktop
```
```
sudo sh quicksetup.sh
```
then type the appropriate responses when prompted.

Happy holidays and thank you everyone for a wonderful year of Ubuntu!
