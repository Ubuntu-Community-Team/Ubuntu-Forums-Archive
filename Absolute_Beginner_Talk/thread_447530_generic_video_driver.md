### Post by vikramsharma on 2007-05-18
You are having problems in display because of the changed monitor.  The easy way to solve this problem is to boot with the Ubuntu install cd.  Once you are running Ubuntu live through cd you can go to /etc/X11 directory and copy the xorg.conf to your hard drive.  You mount the ubuntu hard drive and replace the xorg.conf file.  You basically would have to replace the xorg.conf file on the hard drive with one on the cd.

Once you have the display running go to System -> Administration->Resistricated Drivers Manager and install the Nvidia driver throgh that.  I hope this helps

### Post by ramjet_1953 on 2007-05-20
I know that it's a bit late, now that you have bought an new nVidia card, but did you try installing the new Intel driver for your on-board graphics?

The older 915resolution driver has been updated.

It may be worth a try, if you haven't already done it.

copy and paste this into a terminal:

[COLOR="Red"]sudo apt-get install xserver-xorg-video-intel[/COLOR]

after this has done its thing, just re-boot and see if it has done the right thing.

 It should automatically select the optimum settings for your system and all available resolution settings should now appear in the screen resolution menu.

Roger :cool:
