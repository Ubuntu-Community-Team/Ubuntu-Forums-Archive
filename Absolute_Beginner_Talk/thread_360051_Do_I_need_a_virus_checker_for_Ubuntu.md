### Post by Zuuswa on 2007-02-12
Nope!  And you dont even need to install a firewall!  Ubuntu automatically blacklists all ports.  Some people suggest installing some virus software so that you dont mistakenly send your Windows friends any viruses, but thats rather unlikely.  And as long as you dont install any suspicious software (as in not from the official repositories) you should be just fine.
