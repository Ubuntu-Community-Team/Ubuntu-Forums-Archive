### Post by Maestro23 on 2007-09-13
> **aristos said:**
> Do I have to have a OS, like Windows XP installed on my computer before I can install Ubuntu?  Can I remove completly my Windows XP then load in Ubuntu?

Before you completely remove your Windows, use the Live CD to try Ubuntu out.  Also to see if it likes all your hardware from the start.  Then if you like it and you have no serious hardware issues. install it to the hardrive.  Or you can dual-boot with XP.
