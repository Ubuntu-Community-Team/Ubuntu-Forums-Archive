### Post by kerry_s on 2007-09-15
press> alt+F2
type> gksudo mousepad /etc/X11/xorg.conf

change screen section to>

```
[B]Section "Screen"
	Identifier	"Default Screen"
	Device		"Generic Video Card"
	Monitor		"Generic Monitor"
	DefaultDepth	16
	SubSection "Display"
		Depth		16
		Modes		"1024x768"
	EndSubSection
EndSection
[/B]

```
also recommend you disable the fancy crap
add this to the bottom->

```
[B]Section "ServerFlags"
        Option  "AIGLX" "off"
EndSection

Section "Extensions"
        Option "Composite" "Disable"
EndSection
[/B]
```

make sure you leave a blank line at the bottom.

when you done, logout and press> ctrl+alt+backspace

### Post by VraiChevalier on 2007-09-15
Might want to make a backup copy of that /etc/X11/xorg.conf file first.

```
gksudo mousepad cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup
```

That way if anything goes haywire you can restore the settings with the backup of the original file.

```
gksudo mousepad cp /etc/X11/xorg.conf_backup /etc/X11/xorg.conf
```
