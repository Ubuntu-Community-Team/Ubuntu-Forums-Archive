### Post by arkangel on 2006-08-02
for installing  c++

open  synaptic  
and look for  build essential

or in a console 
sudo apt-get install build-essential



for java  open  synaptic,  i am not sure but java 1.4.2  is  found easily
if you want to install java 1.5  you have to enable  Multiverse or Universe (not sure)

open synaptic  Settings>>repositores >> add and  add both (or all you can add  :) )

then  press the reload button and look for java 1.5


BTW  synaptic is found in System>>Adminitration>>Synaptic Pakage Manager  in the desktop
