### Post by Pragmatist on 2006-03-09
To check if you have **gdesklets-data ** package installed do this:
```
sudo synaptic
``` then use the search feature to search for **gdesklet** and see which packages are installed (have a filled green box)

### Post by Kapre on 2006-03-09
grim - gdesklet doesn't stay in your desktop (unless you made a shortcut for it). If you did not put gdesklet everytime you startup, the desklet (eyecandy) that you want to appear on your desktop will not show when you restart your pc. Regarding your pc running slower, check some of the apps you're running (as there maybe some in the background) that makes your pc to run slow.
