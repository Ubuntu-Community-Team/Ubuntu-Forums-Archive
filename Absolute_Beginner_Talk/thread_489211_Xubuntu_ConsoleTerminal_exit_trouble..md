### Post by ugm6hr on 2007-07-01
Have you done a complete install of Xubuntu (or installed xubuntu-desktop from Ubuntu)?

If it is a fresh install - Applications->Accessories->Terminal should bring up Terminal.

If this doesn't work, try Alt+F2, and type "xfce4-terminal" then Return.

And Appfinder is in Applications->Accessories->Appfinder (or Alt+F2->"xfce4-appfinder").

If you use Alt+Ctrl+F2 to exit X (to get to CLI), I'm afraid I have no idea how to get back to GUI (maybe startx).
