### Post by localzuk on 2005-10-16
Breezy Badger is version 5.10 of Ubuntu. It was released on the 13th October 2005. Kubuntu is the ubuntu distro with KDE as its desktop instead of gnome.

You don't 'need' either of them. However, if you want to stay on the 'bleeding edge' (in terms of Ubuntu) then you should upgrade to Breezy.

To have a look at kubuntu, all you need to do is ensure that you have the universe repo's enabled and install the package 'kubuntu-desktop' (apt-get install kubuntu-desktop or select it in synaptic).

### Post by erikpiper on 2005-10-16
Ok- consise version 

Breezy= nickname for ubuntu 5.10
Hoary= nickname for ubuntu 5.04

Ubuntu= Default distro, uses gnome.

Kubuntu= Ubuntu, but KDE and KDE apps instad of Gnome and Gnome apps. (I don't like KDE myself..)

Xubuntu= Ubuntu, but with XFCE. The "lightweight" version- much faster. (Good for old machines)


You can convert any of these to the other by installing (respectively) ubuntu-desktop, or kubuntu-desktop, or xubuntu desktop. 

Then you can switch between window managers by pressing "sessions" on the login screen.
-------------------------------------------------------------------

BTW- Breezy rocks! I had no problems whatsoever. You have to do a fresh install though- don't just upgrade. More people have had succes with breezy that way.
