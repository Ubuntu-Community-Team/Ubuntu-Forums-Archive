### Post by simosx on 2007-05-25
Right-click on some empty space on the top panel and choose to add the Keyboard Indicator applet.
In the Keyboard Preferences/Options, go to Layout Options (&#917;&#960;&#953;&#955;&#959;&#947;&#941;&#962; &#948;&#953;&#940;&#964;&#945;&#958;&#951;&#962;) and under Group Shift/Lock Behaviour choose "Alt-Shift changes group" ("&#932;&#959; Alt-Shift &#945;&#955;&#955;&#940;&#950;&#949;&#953; &#959;&#956;&#940;&#948;&#945;"). Deselect any other options in the same group. Once done, Alt-Shift should get you to switch between English/Greek.

Visit [https://launchpad.net/~ubuntu-greek-users](https://launchpad.net/~ubuntu-greek-users) for the Greek Ubuntu Users group.
