### Post by lucidite on 2006-01-21
Have you gone into the desktop defaults and made the changes there?

(When you're talking about clicking on a hyperlink, I'm assuming you're talking about a link that you've encoutered when not browsing with Firefox, like a link in a text. Right?)

If you use KDE, you can open up KControl > KDE Components > Component Chooser & then select Firefox as your default browser.
