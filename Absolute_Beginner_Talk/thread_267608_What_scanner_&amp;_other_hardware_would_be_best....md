### Post by fjm03 on 2006-09-28
Try this before spending money for something you already own.

Download VMware Server and install via the instructions on this forum.

Install your old copy of Windoz on VMware Server.

Run your present scanner on Dapper from a virtual M$ machine.

There are only two twists in the process. 1) Where to store the scanner output. I'd suggest a cheap NAS if your Dapper box isn't part of a network that includes one SAMBA share. 2) You'll need adequate GBs of free space on hda1. More if you keep a backup copy of the vm (XP Pro w/ minimal apps @ 30GB).
