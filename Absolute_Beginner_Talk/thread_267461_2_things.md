### Post by aysiu on 2006-09-28
I don't know how to *recover* the password, but you can always change the password by booting into recovery mode and then typing ```
cd /home
ls
``` seeing the username (let's say it's *oldindian*) and then typing ```
passwd oldindian
reboot
```
