### Post by CroEragon on 2007-01-26
You need codecs, built-in players can handle all kinds of media filetypes.
Because media filetypes are property of their inventers, only they can give permission to use it with Ubuntu. Since they don't wanna make it open source it is illegal for Ubuntu developers to put support for them in Ubuntu. But dont worry,  you just install few codecs and built-in media players like Totem and Xmms will able to play them. You just need Automatix to install codecs for them. It actually installs all many freeware programs that are not Open Source (like Skype). You can find how to install it [here]("http://www.getautomatix.com/wiki/index.php?title=Installation#Installing_Automatix2_with_Apt")
When you get it installed just read what it says and find one that is media support codecs. Install them with Automatix.
Thats it, you can play mp3 or wmv and many other media filetypes.

BTW 
Dont worry, English is not my native language also. If you have problems with English, try to ask your local Ubuntu group (LoCo Team). They translate Ubuntu and do all kinds of stuff to make Ubuntu available on your language. Also you could say what is your native language here, you may be lucku and somebody whyo nows your native language can help you.

### Post by HereInOz on 2007-01-26
To open a terminal:

Applications > Accessories > Terminal.

VLC and its plug-ins can be downloaded using Synaptic - just search for VLC.

You might also like to have a look at this page:

[https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)

It will help you to install what you need to play restricted media such as DVDs and MP3 files.  There are pages in other languages for this as well so click on the Wiki tab up the top right, then put "restricted formats" in the search box, and it will show you all the available pages.
