What is difference b/w these,  Read this:
[https://wiki.ubuntu.com/Releases](https://wiki.ubuntu.com/Releases)

These are code names for the various releases. 
Ubuntu 6.06 = **D**apper Drake
Ubuntu 6.10 = **E**dgy Eft
Ubuntu 7.04 = **F**eisty Fawn
Ubuntu 7.10 = **G**utsy Gibbon
Ubuntu 8.04 = **H**ardy Heron

Yes, the code names follow the Alphabet ...
