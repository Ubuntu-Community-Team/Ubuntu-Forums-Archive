### Post by landotter on 2005-10-21
Ensure that the .bin file you downloaded is executable. You can make the .bin file executable by right clicking on it and choosing properties, tick the "run box"

Assuming you downloaded to the Desktop, open a terminal and:
$ cd Desktop
$ sudo ./RealPlayer10GOLD.bin
$ password: ****

When you launch the player for the first time, a set-up assistant will take you through configuring your player.  To run Realplayer, get a run dialog by pressing alt+f2 and type in "realplay".

This information was readily available on the RealPlayer for Linux page. ;) I just made it even simpler. :p

Have a read of the wiki at the Ubuntu website--if you've only used linux a couple days, you need to do your homework to get comfortable.

have fun.
