### Post by Dr Small on 2007-08-23
To do a warm boot, after you save the bios settings, it should automatically load the settings. If for some reason you need to do a "reboot", hit CTRL + ALT + DEL and it will reboot the computer, as a warm boot (because the system is still running.)

A cold boot is where the system is NOT running.

Dr Small
