### Post by ubuntu27 on 2007-08-21
You can use [Amarok]("http://amarok.kde.org/") which is the most powerful media player out there. It has built in library, iPOd support, last.fm support, it has extension manager, etc. 

I believe to be better than iTunes.

Just look for the package [amarok]("http://amarok.kde.org/") in the Add/Remove or with Synaptic Manager.

If you want to use the terminal:

```
sudo aptitude install amarok
```
