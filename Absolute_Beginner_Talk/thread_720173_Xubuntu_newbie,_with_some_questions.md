### Post by angry_johnnie on 2008-03-10
Well, first of all, welcome.  And, no,  you don't have to be a programmer to use Linux.  It's easy enough, really.  You won't lose any functionality, but keep in mind that, while many Windows programs have a Linux equivalent, not all of them do. 

Xubuntu is Ubuntu with the XFCE desktop environment.  Long story short, xfce is lighter than GNOME (ubuntu's default) and runs better on older hardware. 

gOS is another Ubuntu-based distribution, but as far as I know, it includes only web apps.  Google web apps, to be more precise.  I'm not sure whether that's something you would want.

You can buy a computer with Linux preinstalled.  It's the easiest way to go --not much fun, though :-) -- if you want something that just works.  

For more info on that particular subject, go to this site:

[http://www.dell.com/content/topics/segtopic.aspx/ubuntu?c=us&cs=19&l=en&s=dhs&dgc=EM&cid=21690&lid=511380](http://www.dell.com/content/topics/segtopic.aspx/ubuntu?c=us&cs=19&l=en&s=dhs&dgc=EM&cid=21690&lid=511380)

:lolflag: long link!

If and when you decide to install Linux, whether it's xubuntu or whatever, it would be better if you came back to the forums for instructions on dual-booting (if that's what you want), or just to make sure you're on the right track.

Xubuntu is a good OS, but why don't you try a few others as well?  Download a few ISO images.  Burn the cds and see which distribution better fits your needs.  There's thousands of them out there, and I think it would be a good idea to try out a few, just to make sure.

### Post by Arkenzor on 2008-03-10
Not forgetting Kubuntu, Ubuntu with the KDE environment. I find it interesting for beginners because of its excellent (and huge) graphical control panel, a bit like the WIndows configuration panel only... better.

At any rate, since it's so easy to try several distributions you should just do it. Unless you want an OS quickly, in which case you may simply install Ubuntu/Kubuntu (I find Xubuntu to be a bit sparser on easy configuration options), and keep trying other distros after that. The reason I usually recommend Ubuntu to people trying Linux is that it will (almost) always work out of the box and allow you to explore what Linux actually is. Then you can try a few other distributions and make a weel-informed choice.

### Post by erm4gh on 2008-03-10
I wouldn't expect to see any real performance difference between Ubuntu, Kubuntu and Xubuntu if you have 1GB of memory. I think its more a question of what windows/file manager do you like, and the launcher. I settled on Xubuntu mainly because I prefer Thunar.

I suggest you dual boot XP and (one of the versions of) Ubuntu, but configure Ubuntu to have a separate home partition. That will make it easier to keep your data if you switch to another Linux distro later on. After you've used it for a couple of weeks you'll get a better handle on what really matters to you, and whether you want to keep dual booting.

If you decide you want to switch windows/file managers (thats the biggest difference between the different variants of Ubuntu) see [http://www.psychocats.net/ubuntu/index](http://www.psychocats.net/ubuntu/index)
