### Post by Jussi Kukkonen on 2006-10-14
For C development: installing *build-essential* package will give you the gcc compiler and other necessary tools. If you want a graphical environment, you could try Anjuta.

The first user that was created during install has sudo rights. Read more here:
[https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)
