### Post by woedend on 2006-07-22
yes, you can put them both on the same hard drive.  When I had windows still, I made one partition for ubuntu, one for windows, and one for all my mp3's and media so I could play them from both.  It can do everything windows can do (in general)...but it will not run windows programs(IE - it won't run adobe photoshop or most high end games) but yes you can play all of your media,mp3s, web browse.  I really can't tell you every difference.  In general they are very much the same, you have a point and click interface, minimize/exit in the top right corner..etc etc.  But linux by default you will find more secure and best of all you won't have the spyware problem windows is plagued with. I really think youll love it, download the desktop install cd, put it in, and reboot.  It will let you test the waters without installing anything.  It will run much slower than the real ubuntu, though.
