### Post by johnc4510 on 2006-06-07
Do this:
System>Administration>Synaptic Package Manager
Click on settings, Click on repositories
Put a check mark in all the empty boxes and close that window
Click on Reload, Click on Mark all Upgrades, Click on Apply
Now you should be able to install firestarter
