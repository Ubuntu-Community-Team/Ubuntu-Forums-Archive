### Post by monkeydust on 2006-11-03
> **Nitrouzzz said:**
> Ubuntu started to lag constantly

The delay was due to running from CD. It's far slower than running off a Hard Disk.

> 
should i NOT use any Linux version if i want to play games?

As for games, I know that some people use WINE or VMWARE to run *some* Windows-only games, but I'm afraid my knowledge of this is limited as I use Ubuntu mainly for office stuff.

Ubuntu runs GNOME desktop environment. Useful for most desktop use.
Kubuntu is the same but running KDE desktop. 
Xubuntu runs a slightly more lightweight desktop that would be better for slower machines.

Honestly? If you want to play with Linux, I'd "dual boot" - ie, a menu offers you the choice of Windows or Ubuntu when you start up. That way you can try out Ubuntu without missing out on your Windows stuff.

Just my £0.02.
