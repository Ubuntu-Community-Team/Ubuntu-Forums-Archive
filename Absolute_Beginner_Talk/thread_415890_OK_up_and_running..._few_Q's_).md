### Post by avik on 2007-04-20
> **ljcasey said:**
> 
What is the best app to show system resources etc up the side of the screen


If you're using GNOME, right click on one of the panels and go to "Add to Panel."  System monitor is what you're looking for.  If you *really* want it up the side, drag one of the panels to the side, or create a new panel.

> **ljcasey said:**
> How about people list there top 5 apps they install on a fresh install!

The great thing about Ubuntu is that it already comes with so many useful tools.  As for me, I like Opera for my day to day browsing, and Mozilla Thunderbird, as I don't need all the features of Evolution.

If you're a software developer, definitely install build-common and friends.

If you're into bittorrent, I've taken a liking to KTorrent (yes, it's a KDE app, but it's still pretty good).

Inkscape is nice for those into vector graphics.

But really, I just install software as I need it, simply because Ubuntu already sets you up with a great deal of what you need to get up and started.
