### Post by Kateikyoushi on 2007-03-16
I would recommend to install all of them, the old packages replace the new ones so won't take up more space neither do they bog down your pc.
If you are on dial up then would stick with the cd and upgrade every 6 months when a new version comes out. Maybe use only security updates, Can set it in system > administration > Software sources then uncheck all except the security updates checkbox.
