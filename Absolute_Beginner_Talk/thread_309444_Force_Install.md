### Post by LLRNR on 2006-11-29
Hi,

To get VMware Server installed properly you indeed need to remove any previous (attempts of) installs. I'd suggest you take a look at [this HowTo](http://ubuntuforums.org/showthread.php?t=183209) - if you scroll down a little you'll notice there's a heading reading **Common Problems and Solutions** (though a full-read is highly recommended! - this is why I don't want to give you the solution myself, it's not good to spoil your fun) - where you will find the way to remove previous installs.

Since you can't have both the player and the workstation (server) installed at the same time, I think you'll take the time to consider what's best for you - the player only allows you to use precompiled OS images and thus you cannot change them... while the server edition actually lets you install and configure any OS you want, it the way _you_ want it. (In my case, I find the Server a lot more useful for my purposes, but maybe for you it's different, I don't know).

LLRNR
