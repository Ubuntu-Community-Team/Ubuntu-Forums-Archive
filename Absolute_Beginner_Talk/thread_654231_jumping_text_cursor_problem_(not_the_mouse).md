### Post by bwtranch on 2007-12-30
This sounds like some sort of wiring problem to me. If you have another keyboard laying around, try that one and another mouse if possible, etc.
But, there are other possibilities...could be an intermittent short at the CPU itself, like at where the cable connector enters the CPU or the connector itself. Internal wiring issues are not out of the question either. See if you can try to make it happen by jiggling things around and stuff. Let us know what you find. Cheers.
