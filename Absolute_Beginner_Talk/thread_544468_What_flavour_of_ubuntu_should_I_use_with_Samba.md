### Post by bongobonga on 2007-09-06
It makes no difference what so ever what version you use, they will all work equally as well. ubuntu/kubuntu etc only really change the user interface, such as using gnome instead of kde as the primary user interface. 

Samba is a server, which runs in the background, as such it is completely independent of the user interface. Actually, you don't need any user interface running at all on you computer to use samba, just the low level stuff.
