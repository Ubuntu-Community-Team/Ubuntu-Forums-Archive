### Post by alienexplorers on 2008-01-02
What exactly are you trying to do.  If you want firefox to access IE only pages you would need to setup wine and load IE in it.
You could also use the User Agent Switcher add-on to emulate IE.
Firefox should be able to access all web pages whether IE or not.

### Post by aysiu on 2008-01-02
IETab won't work in Linux, even if you have IEs4Linux installed.

As w4ett points out, the User Agent Switcher extension will try to trick sites into thinking you're using IE.

If that doesn't work, try [IEs4Linux](http://www.psychocats.net/ubuntu/ies4linux)
