### Post by afonic on 2007-09-04
The official release date for Gutsy (7.10) is October the 18th.

No need to do anything, just stay clear of automatic installation apps like Automatix2 and when the new Ubuntu is out the release announcement will contain update instructions.

It is very easy to do through update manager.

About the question if a clean install is better, it depends. If you have a clean Ubuntu system, installed software only from the official repos and stayed away from stuff like Automatix, the upgrade should be really smooth and you will be in the next version in no time. However in some cases (mostly stuff like the ones I mentioned above), problems may occur in the upgraded system. Those are most of the time solvable, however some prefer just doing a clean install.

If you ask me, backing up my settings and files seems like a nightmare so I prefer doing the upgrade! Also I had upgraded from 5.10 -> 6.06 -> 6.10 -> 7.04 just fine with very very few problems. (I did a clean install of 7.04 to use the AMD64 arch but even now I have updated to 7.10 alpha and everything works fine).
