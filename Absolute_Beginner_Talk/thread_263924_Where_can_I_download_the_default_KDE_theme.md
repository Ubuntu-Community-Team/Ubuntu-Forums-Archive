### Post by SunnyRabbiera on 2006-09-23
KDE is not a theme, it is a desktop enviroment.
Unlike windows XP you have a multitude of enviroments, its like installing a mac interface in XP if it was possible and each one has its flaws and benefits.
KDE is pretty good, I have used it and generally like it better then Gnome though I am getting used to gnome and find that gnome is not as big on resources as KDE.
KDE has a lot of nice eyecandy though, nice interfaces and all that but it doesnt make it better.
as for installation the best way to install KDE is through synaptic, just search for KDE and select the appropriate packages.
Just remember there are a LOT of packages for KDE and it can possibly break your system... not your computer mind you but your linux setup.
