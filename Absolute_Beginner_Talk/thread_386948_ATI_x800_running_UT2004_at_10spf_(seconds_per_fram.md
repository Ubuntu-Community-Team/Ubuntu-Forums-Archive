### Post by igknighted on 2007-03-24
Don't worry about commenting them out.  Its fine to leave them because in the ServerLayout section it points to the proper ones.  Aticonfig --initial sets it up this way so you can just edit the ServerLayout instead of backing up the whole file.

I think you need to remove the ATI drivers you have installed and reinstall them.  You could try the Envy script if you want to automate it to cut down on the possibility of errors.  How did you install fglrx initially?  Did you use the version in the repo's or from ATI's site?

EDIT: to edit the file you need to use the sudo command.  You do not have the permission to write to that part of the file system without obtaining root privileges.  Type "sudo nano /etc/X11/xorg.conf" if you want to edit it, not just nano /etc/X11/xorg.conf.  You can substitute your editor of choice for nano (vi, emacs, gedit, w/e).
