### Post by Pobega on 2006-12-16
```

sudo apt-get wine
```

To run a program in Wine, for an example on your desktop you have "install.exe" you type this in your terminal:

```

sudo wine ~/Desktop/install.exe
```

Wine doesn't emulate DirectX well (if at all?) but it does emulate a good amount of applications pretty well.
