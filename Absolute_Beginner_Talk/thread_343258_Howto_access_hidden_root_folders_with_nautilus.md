### Post by JamieC on 2007-01-21
CTRL + ALT + F2 is command line in tty2, CTRL + ALT + F7 will return you to X.

You can set up a keyboard shortcut for 'Run a terminal' in System -> Preferences -> Keyboard Shortcuts.
