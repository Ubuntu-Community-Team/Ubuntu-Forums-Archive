### Post by senning on 2006-02-24
From what I understand, iTunes uses DAAP to share music accross a network.  In Dapper Drake (the current experimental version of Ubuntu), Rhythmbox and Banshee (and Muine, I think) should be able to pick up DAAP shares.  If you're using Breezy (you right click on the panels->About Panels and it has a version number of 2.12.*, and this is most likely) then [this]("http://gridpt1.fe.up.pt/mlopes/blog/index.php/2005/12/24/rhythmbox-092-deb-for-ubuntu-breezy/") may work for you.  Be sure to read it all.
