### Post by az on 2006-02-12
Java is proprietary.  There are free implementations, but they do not work 100 per cent.

Java was a cross-platform language initially developed for embedded devices.  At the time (late eighties) they imagined that your toaster would like to talk to your oven.

Java was just about completewhen the internet became useful and so it was pimped as the language that would unify computers all over the world.  "Java drives the web".  Bullshit.

It is horribly slow.  Did I mention it is proprietary?  It is just about as difficult to write something in Java as it is to write it in C.  Just write it in C.

Nothing that runs on ubuntu needs Java.  Some web pages use java so there is a browser plugin for it.  Most people only ever hear about java when they visit a page with really insignificant and useless games on it.  Then they install the java runtime and the web plugin and play for a minute or two.

Get rid of it and continue on happily.  If you ever visit a web page that asks for the java pluggin, you can consider reinstalling it...
