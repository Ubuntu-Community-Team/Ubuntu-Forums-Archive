### Post by yabbadabbadont on 2007-08-04
You will find that you will receive much more help if you use proper English and not "IM speek".  :D

Anyway, here are the steps:

1)  Go to [www.vim.org](www.vim.org) and read the documentation and tutorials so that you will know how to use vi/vim.
2)  Take a class, read a book, or work through an on-line tutorial on how to program in C.  (assuming that you don't already know that language)
3)  ```
gcc -o HelloWorld HelloWorld.c
./HelloWorld
```
