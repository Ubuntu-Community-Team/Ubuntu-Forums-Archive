### Post by c-ron on 2008-04-03
In Firefox's address bar type: 
> about:config

Make sure that the:
> browser.display.use_document_color value is TRUE
and that
> browser.display.use_system_colors value is FALSE



If you need a new Firefox theme.
Go to Tools > Add-ons > Themes and select a different one, or click 'get more themes' to see even moar.
