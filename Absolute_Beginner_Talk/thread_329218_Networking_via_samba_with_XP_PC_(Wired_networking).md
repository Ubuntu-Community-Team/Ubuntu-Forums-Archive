### Post by kelbizzle on 2007-01-02
Yea I finally got off my lazy butt and did this. I followed the same guide you did, ran into the same problem you did, and I found out that whoever wrote that neglected to tell us to restart samba. I already fixed the wiki entry Just do 
```
Sudo /etc/init.d/samba restart
```

After you should be able to connect to the share using the password you specified.
