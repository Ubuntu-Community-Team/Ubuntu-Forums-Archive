### Post by HermanAB on 2007-09-30
Unless this is an incredibly old computer, the boot sequence is set in the BIOS.  So start the machine up and press F2 or Del or whatever it requires to get into the BIOS settings, look for the boot sequence and make the CDROM the first and HDD second.

'Hope that helps!
