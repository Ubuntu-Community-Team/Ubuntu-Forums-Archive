### Post by Quite Interesting on 2006-12-23
What you want to do is first go to system > preferences > theme. Then open up the folder you downloaded the tar.gz theme archive to. To install the theme just drag and drop the tar.gz file from the file manager to the theme preferences window and that should install it for you. Hope that helped
