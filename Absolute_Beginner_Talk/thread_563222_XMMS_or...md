Audacious handles it all. :biggrin:
[http://audacious-media-player.org/](http://audacious-media-player.org/)

Ogg and MP3 naturally plus the playlist formats... 
If you use the one in the Ubuntu repos, make sure to get plugins-extra.
It has a port of xmms-sid for sids, sexypsf plugin for psf, spc included in Game Console Audio plugin.
What format is the megadrive stuff? VGM files? If so it handles it.
See below:
[http://audacious-media-player.org/Features](http://audacious-media-player.org/Features)

Winamp *was* my favorite player, used it since version 1.9... 
But now it is Audacious :KS

P.S. Amarok, VLC, and mplayer won't handle these console/computer formats. ;) Except maybe sid, as there is a gstreamer plugin.
