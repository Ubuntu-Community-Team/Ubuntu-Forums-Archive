I've just seen this - and uncertain if this is an independent question, or linked to this: [http://ubuntuforums.org/showpost.php?p=3177811&postcount=1](http://ubuntuforums.org/showpost.php?p=3177811&postcount=1) (quote below):
> 
I was wondering how much space Ubuntu 4.07 takes up on a full install (will it fit and work within 2GB)

If you are planning on installing within 2GB - I would advise installing OpenOffice with caution.  It requires a lot of space both to download the .deb, and even more space to install.  You may find that everything crashes (possibly unrecoverably) if you try....  Just a bit of advice from someone who ended up reinstalling after breaking Synaptic trying something similar (although that probably wasn't necessary - it was the simplest solution for me).

VLC works fine (I use it with Xubuntu), as does Azureus (I have it installed, but much prefer KTorrent for ease of use & speed).

Just to clarify - Synaptic Package Manager is in Xubuntu too, so ALL of the Ubuntu packages are available for easy search, download and install with a few clicks of the mouse (or *sudo apt-get ...* if you prefer), provided you have internet from Xubuntu.

Hope that's useful (if not - please disregard).
