### Post by msak007 on 2007-04-08
Another option if you know someone with an internet connection and want to continue using Ubuntu is [APTonCD]("http://aptoncd.sourceforge.net/"). If you have an Internet connection already, you can back up packages you've already downloaded. In your case since you don't, if you know someone that does and can burn you a CD / DVD they can back up an entire repository for you. It's a great app, but unfortunately you must have some sort of Internet connction to get the packages in the first place. I would take others' suggestions and see if there is a local LUG (Linux Users Group) in your area that you can join. LUG members are usually very helpful and will do anything to help you out, including burning CDs and sending them to you. Depending on where you're at, there might even already be an Ubuntu LoCo (Local Community) group / subforum that you can join (look in the "Ubuntu LoCo Teams" part of the forum) and ask for help there.
