### Post by aysiu on 2006-07-04
Well, this is the problem with graphical (point-and-click) instructions. Too many variables.

If instructions are command-line ones, they can appear intimidating to new users at first, but the command-line is interface-independent, so it doesn't matter if you use Kubuntu, Xubuntu, Ubuntu, or Edubuntu. Just copy and paste the commands, and you're good to go.

That said, yes, a lot of tutorials here assume you're using Gnome, so I would say Gnome is a good place to start. If you're feeling dissatisfied later, you can switch to KDE again.

Instructions on how to install Ubuntu:
[http://www.psychocats.net/ubuntu/gnome.html](http://www.psychocats.net/ubuntu/gnome.html)
