### Post by awkwardmoment on 2007-06-02
Right click on a video and choose properties. Click the open with tab then choose vlc from the list.
If vlc is not listed, then click add and choose vlc from the add application window. click add to close the window, then make sure vlc is selected in the properties window. finally click close, and all videos of that type should open with vlc, however you will need to do this with each type of video (mpg, flv, avi, etc.).

I'm not an expert but I have discovered that there's no real reason that **[COLOR="DarkRed"]VLC[/COLOR]** need be in the "Add Application" list. It is worth noting, too, that if you want to completely remove an installation (files and everything), [COLOR="Purple"]**Synaptic Package Manager**[/COLOR] seems to be the place to go.
