### Post by freebird54 on 2007-06-02
As the install works with the hardware it sees, you could well have lots of problems trying this method.  If you aren't sure about the computer you will end up on, then try the CD "live' session first.  If the computer is not handly (and this is the reason for the question, you could install for 'practice', the redo it for real when you have access to the final system.

Unlike certain other OS's the install is pretty easy - and in most cases you can surf the web (and even drop in here with questions) WHILE you install - so an install does not take long, nor is it as painful.  But - I would certainly recommend installing on the final destinaation directly - and save yourself a lot configuring after the fact..

Hope this helps!
