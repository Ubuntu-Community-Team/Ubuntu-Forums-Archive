### Post by Zack88 on 2008-02-02
ok I have a ati xpress 200m integrated graphic card and it's in the restricted driver area when i click to enable it I get an error that reads "The software source for the package   xorg-driver-fglrx   is not enabled" how to I go about enabling this?

---

### Post by PmDematagoda on 2008-02-02
Go to Software Sources in System>Administration>Software Sources and enable the required software sources in the Ubuntu Software tab. After that is done close the window and reload the sources, that should allow you to install the ATi driver.
