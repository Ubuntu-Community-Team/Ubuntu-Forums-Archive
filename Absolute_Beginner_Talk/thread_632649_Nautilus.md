### Post by jken146 on 2007-12-05
If you have Ubuntu (not Kubuntu or Xubuntu), Nautilus is the default file manager (the only one installed).  When you go to Places > Home Folder, for example, you see a Nautilus window.  If you want to launch nautilus from a terminal or from Alt+F2, the command is ```
nautilus
```

### Post by ArtInvent on 2007-12-05
Nautilus is also referred to as File Browser and it's generally installed by default under Accessories menu. 

I use Nautilus constantly so I generally copy this launcher to my main panel and advise just about everyone to do the same. You can just go to the menu and drag the icon to your panel or to the desktop or both. You can also right click on it and rename it to Nautilus if you want.

### Post by JillSwift on 2007-12-05
Also, if you right-click on a volume Nautilus has on the desktop, and select "properties", it gives you a nice pie graph of used versus unused. You can do this for / by pointing a nautilus window at / and right-clicking in an empty part of the file display and selecting properties, too.

More detail can be had with Applications -> Accessories -> Disk Usage Analyzer
(A _*lot*_ more detail!)
