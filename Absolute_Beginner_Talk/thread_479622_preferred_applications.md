### Post by bodam on 2007-06-20
I know about "preferred applications" under "preferences" but is there a way to tell the system preferences for other file types.  For example, how to I configure Adobe Acrobat Reader to open PDF by default rather than the native tool (I think it's called "Document Viewer")?

Thanks in advance

### Post by BlackMeTaL on 2007-06-20
Find a file you want Adobe Acrobat to open (I assume pdf). 
Right click on it and select Properties. 
Go to the tab "Open with".
Select there the application you want to open files of this type. (Click the bullet in front of the name of the application)
If the application is not listed there you can click the Add button of course.
