From what I gather about the original post, the printer is connected to a computer running Windpws XP.  If that is the case, and the OP is asking how to share this printer over the network, then Lexmark/Linux compatability doesn't matter (fortunately!).


To the OP:  First thing to do is tell Windows to share the printer.  Then see if you can pick it up in the tool for adding printers in Ubuntu (you have to tick "Show printers shared by other systems" and then click on "New Printer").  If Ubuntu doesn't find it then maybe you'll have to install some Samba stuff (I don't know anything about Samba, sorry).
