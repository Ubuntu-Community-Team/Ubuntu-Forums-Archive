### Post by chriscando on 2007-05-08
Can you see if you are using any cpu scaling (lower the cpu frequency when not under load)? There is any easy way to see by adding the 'CPU Frequency monitor' to your panel.
Right-click on the top panel > Add to Panel > Add the cpu frequency monitor
See if it just runs at the max speed all the time, if so, that will make a difference.

Also, wireless and screen brightness greatly effect my battery life, perhaps this is making a difference?
