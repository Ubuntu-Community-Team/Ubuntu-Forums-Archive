### Post by AlanRogers on 2006-12-27
> **james2nd said:**
> I wondered if I should get a router so I can run both computers. My modem is a BT voyager 105James, take heart and yes, buy a router. As Jonthan.Lees says, a wireless one will give you utmost flexibility and they're cheap as chips these days. It will give you the following advantages:[LIST=1]
[*]The router manages the internet connection, not your computer
[*]You have a hardware firewall between you and the rest of the world, as well as your software one if you're using
[*]You set the router up once and forget about it - an occasional reboot is all it should need 
[*]Configuring a network connection to the router is far quicker than a PPPoA on every machine!
[*]Your computer won't be slowed by the modem drivers, firewall, etc
[/LIST]I used to work for one of the subsidiary companies that are now owned by Pipex and use one of their other subsidiaries as my provider. If you need any help setting up, catch me on IM some time.

Oh, and for the time being, keep your Ubuntu machines wired, not wireless. That's not newbie territory yet, trust me - tried and failed.
