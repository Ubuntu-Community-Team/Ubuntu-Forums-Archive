### Post by nickburns on 2006-12-06
When you are on the Live CD, open up a terminal (Applications >> Accessories >> Terminal ) and type in:

top

this will show you all the programs that are running and put the one with the most resource use at the top.  See if there is one program that is causing the trouble, perhaps we can find the problem and convince you that all will be ok for the install.

BTW you use CTRL+C to get out of top.
