### Post by ByteJuggler on 2008-04-11
Firstly, I would suggest you install a newer version.  6.06 is several revisions away from latest.

Secondly, have you checked the CD for errors?  There's an option on the boot menu for checking it.  If it fails, go back to your downloaded ISO image file, and check that it's MD5 checksum verifies correctly (see here: [https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM))  If it fails, your download was corrupt and you need to re-download the ISO.  It is advisable to use a download manager to download large files.  If it passes, you need to re-burn the ISO to CD, but probably at a slower speed to help prevent burn errors.

Do not attempt to install until your install CD passes its verification check.
