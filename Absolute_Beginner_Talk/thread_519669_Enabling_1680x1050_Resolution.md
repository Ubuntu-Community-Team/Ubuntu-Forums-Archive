### Post by The_Falconer on 2007-08-07
This post if for users iwth the i915 intel on board graphics card.

I just started using ubuntu a month ago, so I'm still a beginner. I recently had to reinstall ubuntu, and get 1680x1050 resolution working. After modifying (and almost completely destroying) my xorg.conf file to no avail and pouring through the tomes of the internet looking for a solution I found it by looking in synaptic. Ubuntu by default installs the "ubuntu okay-ed" driver. The default driver does not allow you to enable the 1680x1050 resolution, no matter how hard you try. You have to install (through synaptic) the other i915 driver, even though it doesn't have the ubuntu symbol of approval.

I have no idea if everyone else already knows this or not, I just figured that I'd try and help take the burden off the backs of those who suck with linux as much as me :p
