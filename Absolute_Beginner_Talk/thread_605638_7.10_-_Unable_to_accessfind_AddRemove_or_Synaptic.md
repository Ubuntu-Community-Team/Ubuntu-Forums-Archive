### Post by taurus on 2007-11-07
Try adding **a** to group admin as well.  To do that, you need to boot into recovery mode from GRUB menu since you can't use sudo right now.  Then at the prompt, type

```
adduser **a** admin
shutdown -r now
```
Now, see if a can use sudo with

```
sudo apt-get update
sudo apt-get upgrade
```
