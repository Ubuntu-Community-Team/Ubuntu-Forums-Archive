### Post by LaRoza on 2007-04-16
The only book I have on Ubuntu is "Moving to Ubuntu" by Marcel Gagné. It comes with a copy of an Ubuntu DVD which can be run live, and installed.

This is how I got my copy of Ubuntu in the first place.

The book is very nice. It is specific to Ubuntu and gives shell commands for most things while also giving the GUI way of doing things.

[http://www.amazon.com/Moving-Ubuntu-Linux-Marcel-Gagn%C3%A9/dp/032142722X/ref=sr_1_8/103-0404600-7810250?ie=UTF8&s=books&qid=1176743059&sr=1-8](http://www.amazon.com/Moving-Ubuntu-Linux-Marcel-Gagn%C3%A9/dp/032142722X/ref=sr_1_8/103-0404600-7810250?ie=UTF8&s=books&qid=1176743059&sr=1-8)

It is not very technical, nor is it comprehensive.
