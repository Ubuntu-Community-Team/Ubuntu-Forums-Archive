### Post by daou on 2007-02-09
HP printers go great with GNU/Linux OSs. But try installing a Canon printer... :) 

One of the hardest things about Ubuntu and GNU/Linux in general is overcoming hardware problems. This is because drivers are usually closed source. If you ever have to decide what hardware to buy, always check if the manufacturer supports open source. This will encourage others to release drivers as open source as well.
