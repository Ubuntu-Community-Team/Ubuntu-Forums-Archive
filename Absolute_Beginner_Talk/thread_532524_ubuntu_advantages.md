### Post by dwhitney67 on 2007-08-22
I've been using Ubuntu for about 2 months myself.  Prior to this my exposure to Linux was Fedora Core 5, and many moons ago Mandrake and Slackware.  I still use FC5.

Ubuntu is nice simply because it's interface tends to be idiot proof --- well almost --- the universe keeps expanding along with the population of idiots.

As for comparing it to FC, well the both can run Gnome, support the same version of the kernel, development tools, etc.  So frankly there is not much difference... only the difference in the applets provided with the Gnome interface and the support given on the web.

P.S.  When a system is made "idiot proof", a lot of the tooling of the OS is hidden from the average user, and occasionally from the advanced user as well.  Sort of reminds me of Windows.

### Post by neoguy2012 on 2007-08-22
Hmmm, the following keep me on Ubuntu:

1) Great Hardware Support: Unlike other distros I've tried, this one detected all those fancy buttons on my laptop... and some other cool things.

2) Wonderful Community: Very fast responses and most people aren't here to make you feel stupid, but to help you get familiarized with the linux environment.

3) The installation process was a breeze (although many other distros also have great installers now...).

When push comes to shove, it's pretty much all the same.  Ubuntu probably just made the initial hurdle a lot smaller (with a bit of a price... as it's a bit slower than those super-difficult-to-install distros).
