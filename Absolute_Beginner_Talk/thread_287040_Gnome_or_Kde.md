### Post by dbott67 on 2006-10-28
I prefer Gnome because of it's layout & structure --- it seems very intuitive to me.  I also find that the Gnome/Ubuntu installation installs a 'best of breed'* set of apps, rather than everything under the sun (15 different e-mail clients; 10 different ways to burn a CD; etc.)

As others suggest, start with Gnome and then install the other desktops and try them:

```
sudo apt-get install kubuntu-desktop
```
```
sudo apt-get install xubuntu-desktop
```
[I]
*Note: 'best of breed' is obviously a matter of taste & preference.  Some people may hate Totem for watching movies, while others may love it.  My point is that you don't have to make 15 choices to get started.  If you don't like a particular app, you're always free to try others, but the developers include a pretty decent app to get started.[/I]
