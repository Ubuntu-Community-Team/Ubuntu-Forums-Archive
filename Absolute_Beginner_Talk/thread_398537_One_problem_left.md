There is just one problem left

I've found that I can connect to the internet via Firefox & Opera fine but updates will not work. From these forums I found that replacing the internal DNS with the one the ISP gives you via the 'K-menu>Settings>Networking' solves all these problems.

This includes the issue of Evolution email client connecting to my Gmail account which I was just about to give up on. Again, Thunderbird connects with no problems. When I realised that Evolution is integrated with the desktop I just tried the networking route and whoooosh... of we go with a connection.

I'm just asking is there a way of automatically reassigning the ISP specified dns value after boot or router/ISP's own reset without doing it manually via networking?
