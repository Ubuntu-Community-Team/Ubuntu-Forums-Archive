Don't forget that first slash ;)

### Post by TheMono on 2006-10-12
Press Ctrl+Alt+F1 - this will dump you to a terminal. Then, put in 

sudo /etc/init.d/gdm stop

Then enter your password. You now have no X server... In theory lol.

This also means you have no graphics and no firefox, so if you have a list of instructions you have to go through, you'll want to print them first...

To start it again,

sudo /etc/init.d/gdm start
