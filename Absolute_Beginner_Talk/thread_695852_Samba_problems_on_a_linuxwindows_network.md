### Post by OffAxis on 2008-02-13
you can also run the smbclient command from the command line and it'll connect you to that share.
```
smbclient //servername/directory
```


To view all the shares of your WinXP from the command line...
```
smbclient -L <servername> -W <workgroupname>
```

**smbclient **= Samba client
**-L** = Host name (NETBIOS name of the Windows XP machine)
**-W** = Workgroup name (MSHOME in your case)

It'll prompt you for a password, too. If you don't have one just hit the **Enter **key. Alternatively, if you don't have a password you can just add a **-N** on the tail of the previous command.

```
smbclient -L <servername> -W <workgroupname> -N
```
