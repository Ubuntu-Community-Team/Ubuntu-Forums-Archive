### Post by Hospadar on 2007-11-04
If you are going to just completely wipe out the windows stuff, you don't need to do anything to xp.  The linux installation will (if you choose) format the entire drive, removing all windows data.  So as long as you are comfortable with the data you have backed up, you will be fine.

Note also, that no windows applications or accounts will transfer over (although I think there is a tool in the installer to import windows accounts if you wish, you can have a look at that before it wipes the system)

If you want to save some settings for specific applications that happen to run in linux (probably very few of your apps fall into this category, maybe firefox, or openoffice) you could look into those specific apps and see if they have import/export tools for settings (I know with firefox it's easy to export and import bookmarks for example)
