I started out with Ubuntu, but I wanted a intermediate distro to help me learn the basics faster. There are easier distros to install and configure like Mepis or Suse 10. If starting with a intermediate distro does not scare you and you are willing to take the time to learn how to configure it then Ubuntu will suit you just fine. Ubuntu will help you learn the actual process of everything faster than one of the easier distros.

If you choose Ubuntu, you will have these forums and this great community here to help you every step of the way. I suggest you dual boot with Windows until you get comfortable with Ubuntu/Linux.

Edit: I agree with the above poster for you to try out the live CD first. It will give you a good idea if your hardware will work with Ubuntu before you install it.
