### Post by testube_babies on 2007-07-05
Updating GNOME is a difficult procedure that is simply not meant to be done on Ubuntu unless it's fed to you through the repositories.

If you don't break your system whilst updating GNOME, you will surely break it in the future if you ever try to install something from the repos, since all of the version numbers will be nonofficial and hence dependencies won't be met.

That being said, it *has* to be possible to update GNOME, but it's something the average user would never do, so I doubt you'll find much help on the subject.
