### Post by [S|G] on 2007-05-13
It is possible, but it would be probably better if you got a distribution that was created with that in mind. I'd suggest Damn Small Linux ([http://www.damnsmalllinux.org/](http://www.damnsmalllinux.org/)) which is a 50mb distro in a live-cd or, if you're going to boot from an USB flash stick, Slax ([www.slax.org)](www.slax.org)).

And yes, choosing boot from a cd in the BIOS will boot from a DVD drive.
