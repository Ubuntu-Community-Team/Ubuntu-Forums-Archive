### Post by 45acp on 2008-03-31
First you need to burn the iso to a cd. Then you can boot Ubuntu from the cd and take it for a test drive without affecting your current operating system. If you like it then click the install icon and follow the instructions. Before installing be sure to back up your important files for safety. You can set up a dual boot system if you wish to keep your current OS. That way you can choose which OS    you want to use at start up.
