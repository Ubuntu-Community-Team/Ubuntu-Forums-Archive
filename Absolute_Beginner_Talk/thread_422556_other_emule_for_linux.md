### Post by mikewhatever on 2007-04-25
If the problem is that Kad is firewalled, the solution should be to open the ports it uses. Here is how
[http://ubuntuforums.org/showthread.php?t=331161](http://ubuntuforums.org/showthread.php?t=331161)
The ports are given for the Bittorrent client, so change it to the numbers you have in connections.
Frostwire seems to be a nice program, but, it uses completely different protocols then emule. In particular, it does not seem to be using Kad.
[http://en.wikipedia.org/wiki/Frostwire](http://en.wikipedia.org/wiki/Frostwire)
