### Post by jordanmthomas on 2008-02-17
Just extract that anywhere you like (I prefer /opt)
and run the binary inside it.  Mine is at
```
/opt/mozilla/bin/firefox3
``` 
Yours will likely differ since I installed mine from source and I am not using Ubuntu, but after extracting it look for a bin folder and for an executable called either firefox or firefox3
