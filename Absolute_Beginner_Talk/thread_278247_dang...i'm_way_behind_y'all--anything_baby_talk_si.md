### Post by Ryupower on 2006-10-16
I'm a total beginner too.
here are some basic terms;
**Gnome:** the desktop environment. You notice how your desktop menus and all look different? That's a GNOME desktop. 

**KDE**: There's ANOTHER desktop environment called KDE. It's a little more windows stylish. it has a menu on the bottom left of the screen, like windows. But it's made up of mostly icons.  This is what separates Kubuntu and Ubuntu ( right? ).

**GUI**: Graphic user interface.

Synaptic: The package manager, an easy way to download and install .deb files.

**Package**:In Linux you don't install things with an .exe file( that's windows ), but you download packages,  you install them in different ways depending on the packages. For instance: .deb files you can just install with the synaptic package manager. A quick and easy way to do it. But a .tar you need to install via terminal.
[B]
WINE[/B]: Stands for "Wine is not an emulator". It's a special program designed
for Linux to run .exe files.

Anyways,
Wikipedia.org is always a good place to look up stuff if you don'T know it. ;) 

I  say you spend some internet time searching for infos on using Ubuntu. If you don't have enough info for Ubuntu, you can just look up anything having to do with Linux or Unix, with exception of some minor things that are special for a certain distribution ( like certain programs: RPM, Yast, etc..) it all should work on your platform. :)
Or take a look at any Unix/Linux book.

I had to spend alot of time to know the little things I already know... so don't feel bad if there's anything you don't get.
