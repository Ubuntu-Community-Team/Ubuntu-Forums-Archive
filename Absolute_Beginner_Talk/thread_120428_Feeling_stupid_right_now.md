### Post by GreyFox503 on 2006-01-21
If you want a shortcut on your GNOME desktop or panel, then just right-click and choose the launcher option.

If you mean create a symbolic link, the command is :

```
ln -s <target> <linkname>
``` 
type "man ln" for more information.
