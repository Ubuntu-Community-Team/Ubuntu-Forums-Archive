### Post by Delkster on 2007-01-18
> **atlfalcons866 said:**
> so basicly i should download the dapper drake image?

I have dapper on a work laptop (it doesn't need the latest features but needs to work, preferably without tweaking) and edgy on my home desktop (slightly better support for some video file formats, for example, and other such more recent features).

Both will probably work for you. If you want something that Just Works and aren't very interested in future improvement and new features, take dapper. If you want to get the newer features not only now but also in the future (i.e. you plan on upgrading to later releases when they come out every six months), take edgy. Upgrades are, in general, only supported from an immediately preceding release, not skipping any releases, so if you plan on upgrading to the next release when it comes out, with dapper you'd first need to upgrade from it to edgy and then again from edgy to feisty (the next release). If you install edgy now, you'll have one less upgrade to do.
