### Post by nalmeth on 2006-05-07
If your windows apps are supported with wine, you could probably use wine to launch them in linux off of your windows drive. You have to mount your windows drive first, and of course you need to have wine installed.
Do you know how to mount your windows / install wine?
VMware costs money, but if you have it, I think you have to install either windows/ubuntu through vmware, running in either windows/ubuntu. If that makes sense.
There is a guide on how to install windows with Qemu, but don't expect great performance (like running games) whilst running two OS's at once. Do you have a decent computer?
You should look into linux equivalents though. If your windows system is NTFS, you will likely run into problems when running those windows apps, as you can *only read *ntfs *not write to it*.
