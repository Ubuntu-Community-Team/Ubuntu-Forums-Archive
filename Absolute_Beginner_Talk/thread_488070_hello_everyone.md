### Post by Rocket2DMn on 2007-06-29
Yo, welcome to linux.
If you are starting from scratch (which is best), Windows XP should be installed first.  You can partition your HD while setting up XP or just use the whole HD and repartition when you setup Ubuntu.
Boot from your XP cd and do the format/install.  When you're done with Windows, boot from your Ubuntu live cd and install!  You can use as much or little hard drive space as you want.
I have a dual boot set up on my laptop and with ntfs-3g you can share files between both OSes seamlessly.  There is a little addon program to Windows you can get for reading/writing the linux partition.

If you have further questions, let us know!
