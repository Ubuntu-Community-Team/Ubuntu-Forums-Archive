### Post by Cypher on 2007-06-01
Install the app and it will bring in the most basic KDE libraries needed to run without bringing in the full KDE desktop environment.

Most apps will work within Gnome without any issues, but there might be some extra features that are only available throught the KDE desktop that will not work with Gnome..
