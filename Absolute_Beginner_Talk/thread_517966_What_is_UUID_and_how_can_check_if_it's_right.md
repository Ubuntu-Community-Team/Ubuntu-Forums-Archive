### Post by ron999 on 2007-08-05
That fstab file keeps a record of where your partitions are.
But instead of using a term such as hd(0,1) it uses a UUID number.
To find out where the definite locations of the partitions are then copy and paste this into the monitor:-

**ls -l /dev/disk/by-uuid**

Then make sure that the numbers are the same as the ones in the fstab file.
If they're different then change them in the fstab file.

### Post by logos34 on 2007-08-05
To check the UUIDs:

**ls -l /dev/disk/by-uuid**

Or just take out the UUID's in fstab.  Try this:

> /dev/hdb5 /media/hdb5 vfat iocharset=utf8,umask=000 0 
/dev/hdb6 /media/hdb6 vfat iocharset=utf8,umask=000 0 
/dev/hdb7 /media/hdb7 vfat iocharset=utf8,umask=000 0 

Or instead use:
> ... vfat defaults,umask=0000 0 0
