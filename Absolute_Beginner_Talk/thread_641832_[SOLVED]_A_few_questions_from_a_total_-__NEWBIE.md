1. Use something like Kubuntu.  It is still Ubuntu, but instead of using an OSX like interface, it uses a Windows like interface.  Start menu, etc.
2. I agree with a poster above - Ubuntu (and I think any derivatives of) come with Firefox preinstalled.  It's faster, more secure, and does the job a lot better.
3. tar.gz files are usually for building applications from source - something usually done by those who have more experience with linux.  Ubuntu is based on Debian, which uses APT, which is etremely simple to use, even at a command line interface.  For example: to install BZFlag, you would do: "apt-get install bzflag".  It would download, install, and set up BZFlag for you.  Also, for Ubuntu, since it is based on Debian, you can use .deb files, which are little bundled applications that do the same thing as the command line above.  
4. Im not too sure what you mean by this one.  My apologies.
5. Screenlets are like Widgets.  [http://www.screenlets.org/](http://www.screenlets.org/) has a lot of info for you on what they are, and how to get them.
6. Security.  Linux is a very secure environment, and to make sure the user understands potentially destructive actions to their software, passwords are required to reinforce this.  In Windows, the user is an administrator by default.  Not so in Ubuntu, and most Linux in general.  You are a regular user, and when you need administrator access, you provide a password to get it.  More security.  
7. CRTL-ALT-D in Ubuntu.  
8. Ubuntu is free, due to a man named Mark Shuttlesworth, who is pretty wealthy, and he founded (I beleive) Canonical, to develop Ubuntu. You can still donate to the cause if you want, as is with most Linux distributions.  Linux itself is free, as that is the philosophy of the software.  Open Source.  Anyone can use it, take it, change it, and give it away, as long as its free and credit is given to whom it needs to be given.  Its a little more complicated than that, but...its a start.  

Hope weve answered some of your questions, and welcome to Ubuntuforums.  Enjoy the stay, and welcome to freedom.
