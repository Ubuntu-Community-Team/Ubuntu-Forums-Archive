### Post by edm1 on 2008-04-08
Do you just want it so that it doesnt mount your pen when inserted? If so you need to System>Preferences>Removable Media and Drives, then untick the option 'Mount removable drives when hot-plugged'. I use this in conjunction with the 'Disk Mounter' gnome panel applet to manage my external drives.
