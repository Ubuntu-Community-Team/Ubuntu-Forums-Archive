Flash 9 in Opera is just plain buggy, there's really no way around it. Oftentimes when I return to a page with flash from another tab the flash section (or whole page) will just be a grey box. With Flash 7 I didn't seem to have that problem.

### Post by erfahren on 2007-12-02
just for info: a button can be added to Opera's main toolbar to open the current page in Firefox

the ~/.opera/toolbar/standard_toolbar(1).ini file just needs edited to include it:

(under the section):
```

[Browser Toolbar.content]

```
mine looks like this:
```

Button8, "Execute program"="Execute program, "firefox", "%u", "Firefox", "Firefox""

```
an icon can be added for it as well by modifying the skin. for more info on all that see: [http://www.aimwell.org/Help/Buttons/buttons.html](http://www.aimwell.org/Help/Buttons/buttons.html)
