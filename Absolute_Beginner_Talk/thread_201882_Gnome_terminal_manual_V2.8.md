### Post by Kimm on 2006-06-23
You can download and install any Ubuntu you like. Its all about taste.

Konsole is very similar to Terminal, atleast: You can do the same things with them (as in: use the exact same commands), since they are both "frontends" for bash (the real interpreter).

Edit:
But if you want Gnome, you would want Ubuntu (just plain Ubuntu), if you want KDE, its KUbuntu, and if you want XFCE, you want XUbuntu. The package database is the same (you can install the same programs in all of them), but they all install a different set of software by default, and looks somewhat different.
