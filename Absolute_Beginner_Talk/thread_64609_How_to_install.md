### Post by CMIVXX on 2005-09-11
Unfortunately, unless your running VirtualPC or VMWare, you will need to burn the ISO to CD.  However assuming you have downloaded the install CD and ont the Live CD, the GUI installer program that you will get once you reboot with the CD in will guide you through the process.  However you will need to make sure that you either have some free unpartitioned space to install this on, or you risk losing your windows xp installation in the process.

On note, once you install Linux on a seperate partition and use Lilo as the boot loader, press "Shift" or "Tab" when the Lilo message appears at the top to be presented with a menu of what OS to load.  I hadnt had much experience with Lilo in the past, so I thought I lost my XP install completely in the process.  Boy was I relieved.

Good luck.
