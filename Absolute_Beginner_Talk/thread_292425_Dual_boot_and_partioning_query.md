### Post by podunk on 2006-11-03
Back up any important data before you start. Copy it to a USB drive, a DVD, whatever. 99.99% of partition resizes go without a hitch - but when things go bad they go real bad.

De-fragment your hard drive several times. To be very honest - when I still had Windows I would use the Windows disk management tools to manage my partitions and then let Ubuntu install into the largest free space (it will be an option in the set up). If you decide to let Ubuntu shrink your partition you will be able to decide how much to give what.

Yes, you will be able to network - it takes a bit more setting up than you''re used to - but it works good.

Welcome to Ubuntu!
