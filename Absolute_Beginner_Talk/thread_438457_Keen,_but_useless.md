### Post by p_quarles on 2007-05-09
I, too, am not real fond of the username, but I'll give you the benefit of the doubt.

Save the file where you can find it, because like drowner said it doesn't really matter beyond that. 

The best free Windows software for .ISO burning is ImgBurn ([http://www.imgburn.com/)](http://www.imgburn.com/)). Install it, run the program, insert a blank CD, then get it to copy ("burn") the Ubuntu download onto a CD. This will take a while to finish.

Once it's done, you can boot your computer from the CD. Just place the disc in the drive and restart the system. If you choose the main option (something like "live CD"), you can try Ubuntu without changing anything on your computer. Hope this helps.
