### Post by Tweedicus on 2006-08-08
I have an Ibm Thinkpad T23 which I recieved yesterday from ebay for £200. No probs at all, everything works, install super smooth, and the thinkpad is faster than my Toshiba P4 3.2 ghz running windows. Would highly recommend Thinkpad T23 of you want a cheap laptop that looks good and does everything you want. If you're in England there is a Linux shop in Sutton Coldfield which sells new Thinkpads, good range, with Ubuntu installed, and they can configure it how you want. The shop is called Linux Emporium:

[http://www.linuxemporium.co.uk/products/laptops/](http://www.linuxemporium.co.uk/products/laptops/)

I'm new at Linux and do not know computers, but had NO problems with Ubuntu and my thinkpad. I installed automatix also for the multimedia formats/codecs  and  even watched a smooth running DVD last night on it. Windows has gone forever.
