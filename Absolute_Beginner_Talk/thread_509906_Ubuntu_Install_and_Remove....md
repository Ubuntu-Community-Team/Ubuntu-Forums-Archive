### Post by RomeReactor on 2007-07-26
Hi. You can see where all the program's files are stored by searching for the already installed program in Synaptic (System->Administration->Synaptic Package Manager). Once the package shows up, right-click on it and select "Properties", and go to the "Installed" tab. [This site]("http://www.tuxfiles.org/linuxhelp/linuxdir.html") explains how the directory structure in Linux works.
