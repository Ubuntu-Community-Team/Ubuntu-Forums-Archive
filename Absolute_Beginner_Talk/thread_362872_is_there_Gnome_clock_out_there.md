### Post by ComplexNumber on 2007-02-16
> **docetes said:**
> just wondering i can only find a KDE clock
if you put in "clock" into the search engine on gnome-look, you can find various clocks. some will be gdesklet clocks (these run on python and require all the gnome-python libraries), cairo-clock(this requires you to have transparency, so you either have to have xcompmgr installed or to have beryl/compiz running), and screenlets (these are, again, python programs that require beryl or compiz to be running).
