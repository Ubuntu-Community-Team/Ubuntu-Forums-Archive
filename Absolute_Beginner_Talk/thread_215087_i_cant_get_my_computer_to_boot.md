### Post by T700 on 2006-07-13
First step is to *make sure* that you burned the ISO as an image.  Don't just copy it to the disk and don't make a "bootable disk"; it has to be burned as an ISO image to the CD.

Next, make sure that your computer's BIOS is set to allow booting from the CD drive.

If that's done, I'd try reburning it at a slower speed.  The install process seems very vunerable to errors on the CD.

Good luck,
Paul
