### Post by Jimmyfj on 2008-03-04
The easiest way of getting around that is to download & install the Ubuntu server edition. Chose LAMP install and from the CLI type:

```
sudo apt-get install ubuntu-desktop
``` 

The command will install the Ubuntu desktop and give you the GUI to play with.
