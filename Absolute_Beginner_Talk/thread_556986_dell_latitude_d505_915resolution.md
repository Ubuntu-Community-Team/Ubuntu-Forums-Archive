### Post by ramjet_1953 on 2007-09-23
Before fiddling with your xorg.conf file, try this:


[COLOR="Red"]sudo 915resolution -l[/COLOR]

This will list all available resolutions.

Now choose a resolution that you will never use: ie mode 30

type this in:

[COLOR="Red"]sudo 915resolution 30 1400 1050 24
[/COLOR]
It will come back and tell you that is has been patched.

Now type this in:

[COLOR="Red"]sudo gedit /etc/default/915resolution[/COLOR]

This will open the file and you now need to change these entries:

1. MODE=auto     Change to [COLOR="Red"]MODE=3[/COLOR]0

2. XRESO=     Change to [COLOR="Red"]XRESO=1400[/COLOR]

3. YRESO=     Change to [COLOR="Red"]YRESO=1050[/COLOR]

4. BIT=     Change to [COLOR="Red"]BIT=24[/COLOR]

Now re-boot and hopefully all will be well.

Regards,
Roger 8)
