### Post by HansWilhelm on 2007-12-20
To answer my own question I bought an excellent book "Windows Linux Migration Toolkit".  This book covers pretty much everything from the Server to the desktop.
1. Migration Planning
2. Migrating MS-DNS/DHCP to Linux
3. Designing Linux -Based Directory Services
4. Windows and Linux Authentication - Migrating from NT/Exchange/Active Directory to OpenLDAP
5. Windows to Linux file system migration - Moving shares to Samba
6. Migrate Windows Print Services to Linux
7. Spam & Anti-Virus services
