### Post by K.Mandla on 2007-03-23
> **thefoolisme said:**
> I've been searching now, but I can't find the answer to this question:  What's the minimum recommended processor speed to run Ubuntu?  And Xubuntu?  Thanks in advance.
I think to run Ubuntu comfortably you'll want a machine that's at least 1Ghz, maybe more. For Xubuntu, I'd probably point at 750Mhz or higher.

Which isn't to say you can't run Ubuntu on a 400Mhz machine, or Xubuntu on a 233Mhz computer, only that at lower and lower speeds you'll see more lag, more pauses and a general sense of slowness that otherwise wouldn't be noticeable on a faster machine.

But in theory, 1/0 is right. Anything 386 or up should run Ubuntu, in some form or another. ;)
