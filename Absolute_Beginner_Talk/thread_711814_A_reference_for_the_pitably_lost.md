Probably, the difference between Ubuntu and Xubuntu is essentially just the desktop, Ubuntu uses Gnome and Xubuntu uses Xfce (a "lightweight" desktop with less features overall than Gnome).

Any Gnome package will require the Gnome framework, but "normal" packages should work as part of the base Linux system.
