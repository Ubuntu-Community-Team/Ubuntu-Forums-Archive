### Post by herbster on 2008-03-12
If you want a good plugin for Firefox that will allow you to set which programs open which types of media files and more, I'd grab Mediaplayer Connectivity plugin here: [https://addons.mozilla.org/en-US/firefox/addon/446](https://addons.mozilla.org/en-US/firefox/addon/446)

As for your crashing issues and whatnot, run Firefox from the terminal, see what the output is when it crashes. If your system locks up and you have to reboot, you'll be able to pull up the log and potentially see what the issue is, which you can paste back here.
