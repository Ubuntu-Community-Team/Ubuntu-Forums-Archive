### Post by aysiu on 2005-10-25
[QUOTE=dbzzx]I go into the theme manager thing and i try to incert it into it and it says : this file format is invalid"[/QUOTE] Well, there are three possibilties:

1. It's the wrong kind of theme. Maybe you're trying to install a GDM theme as a Gtk theme or vice versa. Or, worse yet, you're trying to install a KDE theme in Gnome or vice versa.

2. It's not an actual theme, but it's a collection of themes. In that case, you can't "install" it *per se*--you may have to extract it and sort out the parts into their respective folders (the icon folder with the icons, the theme folder with the themes, the wallpapers with the wallpapers, etc.) manually.

3. You actually un-tarred the .tar.gz before trying to install it.

### Post by aysiu on 2005-10-25
[QUOTE=dbzzx]Do you know any good theme sites that are for ubuntu?[/QUOTE] There are no theme sites for Ubuntu. You need a theme site for your desktop environment--KDE, Gnome, XFCE, etc. If you got the default general Ubuntu, you probably are running Gnome. If you got Kubuntu, you're probably running KDE. These two sites may be of interest to you:

[http://www.gnome-look.org](http://www.gnome-look.org)
[http://www.kde-look.org](http://www.kde-look.org)
