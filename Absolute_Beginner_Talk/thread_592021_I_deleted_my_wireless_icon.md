### Post by RomeReactor on 2007-10-26
Hi. I think you're missing the Notification Area; to add it back, right-click in an empty space on the panel and select "Add to panel..." and choose "Notification Area". If the network manager applet still doesn't show up, press ALT+F2 and write:
```
nm-applet --sm-disable
```
then press enter.
