### Post by Tux Aubrey on 2007-01-04
Yep. Repositories are web sites with programs etc. that work on Ubuntu.  There are official and unofficial repositories - the official ones are sort of guaranteed to work.  You enable different repositories in Synaptic or by editing a file called sources.lst.  Synaptic, the apt-get command and easy ubuntu use the sources in that file to find programs ("deb packages") for you.  

Libraries are just files that come with the packages and contain part of the code.  Some libraries are shared by several programs.  Windows DLL files are much the same.

That's my (non-geek) take on it anyway.
