### Post by Klaidas on 2006-03-08
Go to System -> Administration -> Users and Groups
There, click Add User
Entir info and click OK

For installing software, simply use sudo ( [https://wiki.ubuntu.com/RootSudo](https://wiki.ubuntu.com/RootSudo) )
