### Post by EnigmaStigma on 2007-03-21
VMware works great. I'm actually running a VM of ubuntu on my computer right now. You can get VMware server for free here:
[http://www.vmware.com/download/server/](http://www.vmware.com/download/server/)

And then just get the VM for Ubuntu here:
[http://www.vmware.com/vmtn/appliances/directory/693](http://www.vmware.com/vmtn/appliances/directory/693)

All you have to do is remember where you've placed the Ubuntu VM and there should be a few icons in the folder. The only one that you're worried about is the one that has three blue squares. Short cut that to your desktop or whever else you want it, double click and boom you're ready to go.

You won't be able to log into the root but you can sudo around, and change the root password if you want. I'm sure there's a couple posts about that topic, so it shouldn't be to hard to find.

Enjoy,
Enigma

### Post by Tomosaur on 2007-03-21
Hello :)

VMWare player will let you run Ubuntu in Windows. You may also want to look into Microsoft's very own Virtual PC, which is pretty much the same thing.

You are aware, however, that Ubuntu comes on a LiveCD? You don't need to install it to see what it's like. Be warned though, the LiveCD WILL run very slowly. Installed Ubuntu runs much, much quicker.

And of course, you can dual boot Ubuntu with Windows, so you can have both, but enjoy maximum performance. A virtualized Ubuntu (ie, Ubuntu running within Windows) will be quicker than the LiveCD version, but still won't be running at the best possible speed. To get the real performance benefits, you need to install Ubuntu.
