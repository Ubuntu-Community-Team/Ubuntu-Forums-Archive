### Post by NiceGuy on 2006-06-03
kelster - these 'strange files', did the file names end in a ~ ? If so then they may just have been innocent backup files. Whenever you edit somthing using a program like gedit it will make a backup of them. BTW you say they were hidden on the desktop? If you press Ctrl + H when you are focused on the desktop (ie. you have no other windows open) this should show any hidden files.

"like the mouse pointer acting strange etc" - strange in what way? The reason I ask it that I one had a mate play a practical joke on me (this was in windows) where he installed a VNC server on my pc and remote controled it (windows remote assistance style). I must admit it was a bit freaky seeing my mouse pointer moving about on its own.

Having said that I wouldn't worry about "security flaws over the net". Ubuntu's default settings are locked down pretty tight (ip tables setup correctly, no external logins etc.), so unless you/someone else with physical access to you machine has changed something then your probably ok.

I'd have a look at your mouse - does it need cleaning, or if its an optical mouse does your mouse mat have dark/shiny patches on it?
