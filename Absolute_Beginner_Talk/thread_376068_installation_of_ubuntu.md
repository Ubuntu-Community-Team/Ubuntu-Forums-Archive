### Post by toasterofirony on 2007-03-04
I'll presume you downloaded a .iso for Ubuntu.

In which case, you need to use your CD burner of choice to burn the .iso onto a disk. It's not like just dragging and dropping, but it is not difficult. The option to do it should be easy to find when selecting the kind of disc (audio, data etc) you want to make.

Then just boot from said disc and you can play with the Live version of Ubuntu - if you like it, you can install it from the desktop on the Live disc.
