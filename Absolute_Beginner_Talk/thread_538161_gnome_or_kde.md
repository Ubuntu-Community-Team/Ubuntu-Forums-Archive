### Post by fflarex on 2007-08-29
One thing I've found is that gnome apps will run better in KDE than KDE apps in gnome. This is because KDE will re-theme them to look more native, while gnome has no such feature (so, for example, using the default themes KDE apps will have blue menus rather than orange and a different icon set). Amarok also seems to crash on me when I run it in gnome, but I've never heard of anyone else having this problem.

Having said that... I use gnome, because KDE has issues with USB audio devices (would switch in a heartbeat if anyone could fix that issue for me, but nobody seems to respond to those threads).
