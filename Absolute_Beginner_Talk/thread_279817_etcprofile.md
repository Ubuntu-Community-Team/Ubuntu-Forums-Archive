Hey guys I was wondering how do i edit my /etc/profile file i tried opening it and editing it but it won't let me save changes because it is read only can someone plz help me?

---

### Post by Anonii on 2006-10-18
**gksudo gedit /etc/profile**

You didnt have root access. With the "gksudo" in front of the command, you will be able to edit it!
