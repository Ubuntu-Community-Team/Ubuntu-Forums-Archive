### Post by dsmorgan6922 on 2008-01-08
You can enable the 3d graphics (if your card can support it) by going to System -> Administration -> Restricted Drivers Manager and enabling the 3d graphics for your driver.

You can add compiz fusion by going to add software, enabling the search for All Available Applications (top right of the add software screen) and then search for "Advanced Desktop Effects" and install that.
