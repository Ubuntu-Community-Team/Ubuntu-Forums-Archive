### Post by Bjerrk on 2007-05-30
1. Metisse wouldn't work with Beryl, as they're both Window Managers (Metisse uses a patched version of FVWM), and running to window managers along side in the same X session just doesn't work.

2. Yes, Metisse can be used with GNOME. Mandriva does that. They autostart Metisse just at the beginning of the GNOME-session, letting it replace metacity (the default GNOME window manager). I'm not sure _exactly_ hw they do this, but i figure that it wouldn't require much, beside putting metisse in the GNOME application autostart dialog (i don't know it's exact name as i don't use GNOME).
