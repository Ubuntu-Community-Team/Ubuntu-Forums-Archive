### Post by bartcramer on 2008-04-28
Hi!

If you go into Synaptic package manager, search for kubuntu-desktop. This is a metapackage that will install the complete KDE environment. Before logging in, you can choose which kind of session you'll be starting. 

The funny thing is, as Spamzilla said: although Ubuntu comes with GNOME, and Kubuntu with KDE, it is so easy to turn either an Ubuntu or Kubuntu installation into one that is both KDE and GNOME. 

BTW: the difference between the two is not only the appearance. There is a set of standard applications that come with both graphics environments. For file management: GNOME->Nautilus, KDE->Konqueror, for instance. 

Good luck!
