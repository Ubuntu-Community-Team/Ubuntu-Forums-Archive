### Post by Dual Cortex on 2007-04-01
Simply boot the liveCD, run the installer then when selecting the partitions to install Ubuntu on, format the 15GB partition, create a partition with ~1GB for swap, and another partition for Ubuntu + /home, in other words for '/'.
It's pretty straightforward. Make sure to see the links mac.ryan posted, though, just to get an idea of what it's like and ask any questions beforehand.
