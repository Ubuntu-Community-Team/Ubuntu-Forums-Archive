### Post by Oki on 2006-11-24
Hi Cardmaster91 &#8211; let the noob help you:-)

Some applications are missing the icons and so on when you install them; Linux should be better with this. 

Here is what you can do;
Open Alacarte Menu Editor via Applications menu. Mark the Games entry and go to File(at top left) and choose New entry. Under name add &#8220;3d chess&#8221; and under Command press Browse. Go to File System> Usr>games and mark the file called 3Dc, press Ok and close the Alacarte Menu Editor. 

You can also add a icon on the Desktop if you want to. 

You can now start the game from the applications menu under games. 

Hope that works for you!
