When you install Windows apps, you usually do so from some installer, either a .msi, .exe, etc., right?  Well, Linux has something very similar -- packages.

You simply install a single file, a package (.deb for Ubuntu) and everything is taken care of. Compiling/installing from source is a pain in the rear, but it would also be the same way on any O/S you use.
