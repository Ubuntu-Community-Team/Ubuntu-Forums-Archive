### Post by mattismyname on 2008-01-14
You might have a couple options even if they don't offer binaries.

1) Fire up "Synaptic package manager" in Ubuntu and use the search feature to search for jahshaka.  It's very possible that an indepent party has built the package and made it available for Ubuntu.

2) Build it from source.  You can probably find instructions on the Ubuntu wiki to tell you how to build something from source.  For any modern SW it's usually a matter of just installing the compilers on your system using the package manager and then "./configure; make; make install"
