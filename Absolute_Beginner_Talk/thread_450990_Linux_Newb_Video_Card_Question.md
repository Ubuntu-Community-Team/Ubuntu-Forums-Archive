### Post by vtel57 on 2007-05-21
Hmm... ATI, huh? 

Well, I can tell you one thing right off... the only file you need to be concerned with is "xorg.conf". The other one with the number string after it is a backup that was created at some point.

I am not an expert on ATI accelerated graphics on Ubuntu (I use Nvidia). However, I do know that the generic "nv" drivers provided by Ubuntu for Nvidia do **not** provide optimal accelerated graphics. I had to use the Nvidia-provided compiler script to create a custom kernel module for my system. Sounds difficult. It wasn't. Especially after you do it a few times.

Now, being that you're using ATI, you may need to install ATI provided drivers for Linux. Here's where we enter the realm of my sketchy knowledge. You can read more about this at the ATI site.

[http://ati.de/support/driver.html](http://ati.de/support/driver.html)

Wait for further opinions from others here who may be more knowledgeable about ATI. 

Luck! :)
