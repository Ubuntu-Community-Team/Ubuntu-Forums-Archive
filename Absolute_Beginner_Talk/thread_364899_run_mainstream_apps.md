### Post by shano26 on 2007-02-18
Is there a reason your company 'has' to upgrade to Vista or macs at all?? If these apps are mission critical to the business it could be a pretty big move to switch to Ubuntu (or even Mac).

In saying that Ubuntu/Linux has some great alternatives to the three apps you mentioned (Openoffice/Crossover Office for Word, GIMP for Photoshop, Unsure about a vector editing program similar to illustrator but I'm sure there's one, maybe GIMP does this as well). These programs usually do pretty close to what the Windows equivalents can do but compatibility with your existing data files for these apps may be poor or non-existent is some cases (I've had a lot of success with opening my Word files using OpenOffice but the Word files were pretty simple and didn't have many features Word specific, others mileage will vary).

Another option could be running these apps through WINE (a program that simulates Windows to an extent allowing certain apps to be run, some with greater success than others... [http://appdb.winehq.org/](http://appdb.winehq.org/) has a list with a lot of applications and how well they run through WINE) which in essence would be running these apps in Ubuntu (if they work... :-k) .

### Post by srunni on 2007-02-18
For Photoshop, follow this tutorial to get a flawless installation in WINE of it: [http://blog.publicidadpixelada.com/how-to-adobe-photoshop-cs2-on-ubuntu-10-steps/](http://blog.publicidadpixelada.com/how-to-adobe-photoshop-cs2-on-ubuntu-10-steps/)
Since Illustrator is also made by Adobe, a similar process may be possible for it.

As for Word, there are still some bugs when running it through WINE, but OpenOffice.org Writer has more than satisfied my word-processing needs. However, as you would be running Ubuntu in a business environment, your needs will probably be greater than that of the average home user. Of course, if your computers are fairly new, you can run Microsoft Office in a virtual machine (VMware Server is free) and then display just the Word window directly in Ubuntu ([http://searchopensource.techtarget.com/tip/0,289483,sid39_gci1238129,00.html](http://searchopensource.techtarget.com/tip/0,289483,sid39_gci1238129,00.html)). Of course, you could run all 3 of your programs in a Windows virtual machine, instead of locally installing some of them through WINE.
