### Post by varunus on 2005-07-06
Edit: filemanager:  actually, Ubuntu Hoary can resize NTFS and FAT32 partitions in the installer.

Sure.  I did this, except I didn't replace Windows XP.  :)

To do this, you should boot the Ubuntu installer, proceed normally until it asks you about partitions.  Choose advanced (or something similar, I don't remember exactly), and it should show your windows partition.  Choose it, and then choose "size" and reduce it to whatever you want to.  It will ask you about commiting changes to disk once you do this, go ahead and do it.  Then, you will have some "Free" space; select it and choose Guided Partitioning on it.  Then proceed.  Ubuntu will have created partitions for itself and will install there.

This is just from memory, but I remember seeing a post in another forum on the internet that listed the exact steps; if you're not feeling confident, you can try and find that.

Good luck.  Ubuntu is a big step up from Windows ME.
