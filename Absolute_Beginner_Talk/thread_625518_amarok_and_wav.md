### Post by smartbei on 2007-11-28
Try this:
```

sudo aptitude install amarok-xine

```
This should change the sound engine in amarok to xine, which from what I know can play .wav files. Try and report back :p.

If this does not work, please post your amarok version, and whether you are able to play .wav files in player or rhythmbox (sp?).
