### Post by CatKiller on 2006-11-06
Sometimes, with Gnome, entries **are** added to the menu, but they don't show up straight away - the panel has to be restarted, or you need to log out and log back in. Perhaps XFCE is the same (I haven't used Xubuntu)? Also, programs that have no graphical interface are not added to the menu - they are run from the Terminal. Typing the name of the program into a Terminal window is normally sufficient to start the program, although it differs in some cases.
