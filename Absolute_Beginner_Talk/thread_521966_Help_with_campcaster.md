### Post by ferenc on 2007-12-15
Campcaster is not supported by Ubuntu.  Please ask questions on the Campcaster Support mailing list, either using the [forum]("http://code.campware.org/phorum/list.php?20") or by [subscribing to the list]("http://lists.campware.org/mailman/listinfo/campcaster-support").

It helps if you include information about what version of Ubuntu you are using, and which version of Campcaster you installed.

This error message could mean that you had an older installation of Campcaster, and you have an old [FONT="Courier New"].campcaster[/FONT] directory in your home directory.  In this case, removing the [FONT="Courier New"].campcaster[/FONT] directory will solve the problem.
