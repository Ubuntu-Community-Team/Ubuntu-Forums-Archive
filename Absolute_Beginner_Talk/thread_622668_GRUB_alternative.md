### Post by LaRoza on 2007-11-25
> **patchido said:**
> you are telling me that if i take my external drive to any computer and then i start the computer ubuntu will start??? that would be great

When the computer is first turned on, it (usually) looks for boot devices. First it usually looks for a floppy, then a CD, then the hard drive. You can set the bios to look for that device, or manually select it at start up. Usually you press a key to choose a boot device, each bios is different.

It is like a live cd in that respect, but may not boot unless the computer is told to boot it.
