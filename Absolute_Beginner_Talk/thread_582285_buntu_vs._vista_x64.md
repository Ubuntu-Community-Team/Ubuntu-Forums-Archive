### Post by p_quarles on 2007-10-19
Unless you're compiling code or ripping/encoding media, the 32 vs. 64 bit OS won't make much difference. Just about everything else will perform close to the same, in Linux or in Windows.

To answer your question, though, Kubuntu will run great on a P4 with 512 MB RAM. Vista will barely run on that. If you add speed and memory to either OS, they will only run quicker.
