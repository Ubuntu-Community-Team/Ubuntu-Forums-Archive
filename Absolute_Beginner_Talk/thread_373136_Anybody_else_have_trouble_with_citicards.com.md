### Post by skillllllz on 2007-04-21
It is because of bad flash. I use Adblock plus and I manually entered these to my filter and it instantly solved the problem:

```
http://www.citicards.com/cards/wv/FlashJavascriptGateway/installation/JavaScriptFlashGateway.swf
http://www.citicards.com/cards/wv/swf/savecards.swf
http://www.citicards.com/cards/wv/swf/filter1.swf
```

Please do not let this end here. Use this workaround in the meantime if you like. But please, also report the error. Go to "Help" in the Firefox menu bar, select "Report Broken Web Site."
