[B][FONT="Comic Sans MS"][SIZE="3"][COLOR="Blue"]You can dual boot anything you want, as only one HDD or partition thereof can be booted to at any one time.......but why run server and desktop of the same distro?
Seems a waste of resources to me.
The server boots to a command line, and unless you have your own seperate one to install, the choice is Gnome or KDE desktop, and gnome comes up looking EXACTLY the same as a normal Ubuntu install.KDE comes up similar in style to Kubuntu i expect.
Try dual booting Ubuntu and say Fedora 7 or openSuSE, both these have their own merits and drawbacks, but SuSE is again Gnome, so looks and works almost the same as Ubuntu.Just a lighter install on the harddisc.[/COLOR][/SIZE][/FONT][/B]
