### Post by wersdaluv on 2006-12-23
There are lots of things that newbies really need to know. I have been spending about a month learning about Linux and there is still much for me to learn about.

As I have read, most linux users are very intelligent people and do not need very comprehensive instructions. In the case of newbies like me, that is not the case. I need some more sensitive help!

After searching the net, I have seen some web pages that newbies really have too see. 


Here is the list:

[http://www.pcmech.com/show/os/917/](http://www.pcmech.com/show/os/917/) - Windows to Ubuntu transition guide (this is really good!!!)

[http://monkeyblog.org/ubuntu/installing/](http://monkeyblog.org/ubuntu/installing/) - this page claims that it preaches of how one can install anything in Ubuntu (and it really seems to be that way)

[http://thelinuxnewbie.blogspot.com/](http://thelinuxnewbie.blogspot.com/) - Yes! As the url says, it's for newbies like me!!

[http://www.linuxforums.org/forum/](http://www.linuxforums.org/forum/) - another good forum site

[http://video.google.com/videoplay?docid=-6104490811311898236](http://video.google.com/videoplay?docid=-6104490811311898236) - Ubuntu / Linux Dual Boot instructional video (I just don't know if it really works... I never tried it)

[http://www.tuxmagazine.com/](http://www.tuxmagazine.com/) - Official website of TUX Magazine, "The first and only magazine for the Linux user"

[http://www.linuxquestions.org/questions/index.php](http://www.linuxquestions.org/questions/index.php) - another good forum site

[http://lifehacker.com/software/ubuntu/hack-attack-top-10-ubuntu-apps-and-tweaks-195437.php](http://lifehacker.com/software/ubuntu/hack-attack-top-10-ubuntu-apps-and-tweaks-195437.php) - Top 10 Ubuntu Hacks and Tweaks


I lost the urls of some of the good newbie sites that I have seen but I'll just try to add them later.
