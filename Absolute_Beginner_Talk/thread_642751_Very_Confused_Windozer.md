### Post by bump_ on 2007-12-17
If you are using Ubuntu 7.10 Gutsy Gibbon, it has compiz fusion installed by default. To tweak many of the settings, install the compiz settings manager like this:

```
sudo apt-get install compizconfig-settings-manager
```

Then you could either type 

```
ccsm
```

in your terminal or go to System>Preferences>Advanced Desktop Effects to change many aspects of compiz fusion to you liking.
