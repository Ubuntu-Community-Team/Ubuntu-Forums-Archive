> Is connecting to the internet easy? Via a wireless router? How about adding a wireless pci card? Easy or a big deal?

This is the most important question to which you'll need an answer. Use the link recommended above, google around to make sure that you'll buy one that works with GNU/Linux.

A DHCP (automatic IP) connection over an ethernet (RJ45 cable) modem is the easiest as far as I know.

For wireless, all I know is that  **in the case of Wifi cards, it is best to buy the ones based on the Atheros chipset**. I bought one myself and it worked out of the box in a public wifi spot. I only have some problems connecting to password protected wifi networks, but since I usually connect over cable, I never searched for a solution to this problem.

> I see in the forum that Skype can be used. Do I need add-ons to get it going? Is it easy?
Yes, Skype can be used.
The installation is easy since Skype offers an Ubuntu installer:
[http://www.skype.com/intl/en/download/skype/linux/]("http://www.skype.com/intl/en/download/skype/linux/")

But I'm not sure how good audio/video support is. Audio might be Ok, but video could be an issue (again, make sure to buy a GNU/Linux compatible webcam).
I only use it for chatting.

If you want an equivalent of MSN, use **Pidgin** or **Kopete.**

I recommend **AmaroK** and **XMMS** as music players.

### Post by andb on 2007-07-14
The good: 
For what you are talking about, yes, ubuntu would make a great platform. The great thing about it is that once you have the machine set up the way you want it to be, it will pretty much work until there is a hardware failure.

The bad:
It probably wont be as easy to set us as you are used to in Windows. But if you look at it as a learning experience and you enjoy "do it yourself" projects, this can be even seen as a benefit.

I can speak from experience. I've set up Ubuntu machines for people that knew they needed certain functions / programs and where happy to use a computer that I just set up for them. In my own home I have a media center and a simple internet machine, much as you described. My advice - install windows first on one partition. Make it work the way you like. This way, you always have a fall back. Then install Ubuntu on another partition. Then you won't feel time pressure to just make it work. 

About the specifics, there is an Ubuntu Media Center project out there being developed. [https://wiki.ubuntu.com/UbuntuMediaCenter](https://wiki.ubuntu.com/UbuntuMediaCenter). The base is Elisa, which I found processor intensive, so I switched to Freevo [http://freevo.sourceforge.net/](http://freevo.sourceforge.net/), which is anything but beginner friendly setup  and installation. But now that it's set up, it's a beautiful, simple music and video jukebox.

Wifi - No problem if you get the right card. I highly recommend Atheros chipsets. The same card may use different chips, so buy it from a store that has a good returns policy. This is going to be one of your biggest problems. I've always used MSI products and alwasy had the right chipset. You'll find tons of howtos, the best is a sticky post in the networking forum. For home use, its better not to use network-manager.  

itunes. I dont think this is going to happen too easily, you'd probably have to use windows emulation to get it to work. I know there are other programs that can handle the ipod, but I have my doubts about functioning with itunes, specifically the apple sales of music. And I am quite sure that none of the music jukeboxes will work well with this. However, the jukeboxes could be set up to use the ipod when connected. Its in this area of "corporate functionality" that you'll probably be most frustrated.

What else do you need to get started? A love of learning. Would you consider building your own stereo or doing repairs on a classic car? If you love to dive in and learn more, then here's a great project for you. Do you just want a machine for internet, music, and other stuff and  just have it work? Yes, Ubuntu is great but have someone else set it up for you. Im not joking, you can install the base system and a program like secure shell, and if you allow, someone can connect securely over internet, and set up everything remotely for you. Thats a great thing about linux, you can very easily replicate work. I can send you the package and source list that I have on my mediacenter, and in one command line (a long one, I admit) you can install all of the software that I have. Add to that the config files that I have, and everything will just work. Try that with the Windows Registry :) 

Summary: What you are talking about is very possible and when finished will be very reliable and usable. Setup will take some work. Wifi and mediacenter will be the most work now, but these things are being actively developed.
