### Post by FuturePilot on 2008-04-13
I wouldn't recommend enabling the Proposed repository. That contains software that needs testing before it comes through the regular updates. Unless you're helping with testing, I would stay away from that one.

But it's pretty safe to enable the Backports repository. That contains newer versions of some pieces of software. I've always enabled that one and I've never had any problems with it.
