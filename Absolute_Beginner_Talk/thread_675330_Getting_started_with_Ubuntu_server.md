### Post by HermanAB on 2008-01-22
Uhmm, if you are new to Unix, then you should likely not install a 'server' version.  The Ubuntu server version is meant for use in a deep, dark and scary data centre where all the machines are headless monsters...

Rather install Ubuntu/Kubuntu/Xubuntu, since you likely want to have a keyboard, mouse, screen and GUI.
