### Post by xpod on 2006-08-08
You have to have a new partition created for ubunto to reside on.

Once you click the "install" icon on the desktop of your live cd you will go through the different stages of selecting your country,keyboard layout etc.
Then the installer will ask you where you want to install ubunto.

If you only have one xp partition it will offer to resize an existing partition or to create partitions manually.

If you choose the resize option it does it all for you and sets up the dualboot for you too.

I might have the presise order of things recalled incorrectly but thats about how it all happens.

Just make sure you defraf your c drive first to tidy it uo and you should be fine......And if not.........Well you`ll have great fun fixing it!
