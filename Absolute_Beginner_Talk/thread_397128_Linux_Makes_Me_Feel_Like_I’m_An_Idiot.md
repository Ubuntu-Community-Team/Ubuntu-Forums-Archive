Hello PhillD,

You should not feel that way :-) Switching from an Operating System to another is never simple, and we always need to learn a bit. When we go to school and learn new things we do not feel stupid, we just now that we will have to learn many things but we also foresee us when we will know them.
This is the same for switchers :-) so there are many things you have to learn now, but in a few weeks/months you will be able to help others too!

Machine in general are not easy, that is why we have people studying ergonomic and trying to conceive machine that human could use without having to learn. However, this discipline is kind of new, and so lots of effort has to be done so things will get better in the future. For the time being we all have to learn and adapt to the machine to use them. And computer are no exceptions!

So congratulation to have embarked on the X/K/Ubuntu ship, that is courageous of you. But you will see that soon you will know lots of things and can help people. :-)

How old is your hardware? Because a entry-middle range PC bought in 2000 is good enough already to run Ubuntu with Gnome :-) As long as you have more than 256MB of memory, you can run Gnome without too much trouble. Ubuntu is perhaps more integrated than its variants. I mean no offence to them, I started with Kubuntu, but I had bugs using the system tools in Kubuntu that were not existing in Ubuntu. So perhaps you could use Ubuntu. It has a very simple application to install software. It is in the Applications menu, you just click "Add/Remove..." select the software you want and click OK, that's it :-)

PS: Ubuntu/Gnome runs perfectly on a Duron 600MHz with 384MB RAM.

Hi Phill,

I can understand your feelings, because when I started using Ubuntu about a year ago, I felt exactly the same as you did. The beginning is a little bit rough because some things have to be done by hand which you get pre-configured on Windows or Mac OS X. But once you figured things out, things will get better - definitely.

I am far from being a Linux expert but I got my Ubuntu to do nearly everything I want it to do, and I think you will be able to reach that point, too.

My advice is a little general (as your complain is, too :-)), but as it helped me it might be helpful to others, too.

[LIST=1]
[*]**Get a book on Linux or Ubuntu.** Even though we have the net, a quality book on the topic can provide you with the information and put the single bits of information in relation to another. [*]**Look for the right things on the net.** More easily said than done, but try to feed Google (or the search engine of your choice) with as detailed as possible information - and don't forget to have it search for "ubuntu", too. :-)
[*]**Learn to describe your problem tecchy-style.** ;-) Learn from this forum, what kind of information you have to provide the experts here to let them find a solution.
[*]**Be persistent!**
[/LIST]
