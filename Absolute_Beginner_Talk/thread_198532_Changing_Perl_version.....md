### Post by catlett on 2006-06-17
I don't particularly know about these two versions of Perl. But open synaptic package manager and enter perl in the search box. It will list the versions of perl available. Look for the version you have installed. Right click on it and choose "mark for removal". Do the same with the version you want except choose "mark for installation".
The only thing to watch out for is a dependency issue. You might have other programs that rely on the newer version of perl.  Removing perl may cause them to  be removed as well or may create a "broken" package. If things like this happpen you'll have to reinstall the newer version of perl again.
