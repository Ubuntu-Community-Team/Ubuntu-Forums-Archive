Mepis or Ubuntu....i'll know my opinion in about a half an hour \\:D/[/QUOTE] Until just about a week ago, I was quadruple-booting. I had one partition for Windows XP, one for Mepis, one for Ubuntu, and one for the "distro of the moment."

I switched things around, though, in light of the recent Mepis repositories problems and my growing tired of trying out new distros. So now I triple-boot Windows XP, Kubuntu Hoary and Ubuntu Breezy.

My advice--if you're serious about trying Ubuntu--would be to start with Ubuntu, not Kubuntu (as Ubuntu has better documentation at the moment). Right after installing Ubuntu, look carefully at the Ubuntu Guide and decide what you need (codecs, programs, whatever). Then copy and paste exactly what the guide tells you to.

Once you do that, you'll have a very stable and capable system, and you can start tweaking it.

What I like about triple-booting is that I have one distro to use (Kubuntu), one to test things out in (Ubuntu Breezy), and Windows in case I need it for something Windows-only.

I think when newbies are starting out (I'm only in the my fifth month), it's important to have a distro to experiment on with all sorts of reconfiguring and installing of new software. That way, if you mess up, you still have a good, functioning distro to actually use.

one more question....setting my monitor settings.....i haven't gotten the chance yet but is there somewhere easy where i can set my monitor settings or do i go to xorg config like i did manually with mepis?[/QUOTE]
 Well, there's the 'Screen Resolution' program under System--->Preferences on the menu. But if it doesn't display the resolution/refresh rate you want, you have two choices.

You can either edit xorg.conf directly. (It sounds like you already know how to do this, so I won't go through it again unless you want.)

Or, from the command line, you can type
```

sudo dpkg-reconfigure xserver-xorg

```
Then... try not to freak out at all the scary questions it asks. If you read each screen right through, it usually tells you the best option to pick. When you get down to the bit where it asks you how you want to configure your monitor, [Advanced, Medium, Simple], pick 'Medium', and then just select your best resolution/refresh rate from the list. That's usually 1208x1024@60hz for a 17" flatscreen, and 1024x768@75hz for a 17" CRT.

But honestly, if you know how to edit xorg.conf safely, that's probably the quickest way.
