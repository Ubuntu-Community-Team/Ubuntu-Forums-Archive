### Post by akai_kenshi on 2007-12-26
When I have a conversation window minimized in Pidgin, I'm used to it's taskbar icon blinking brightly to catch my attention when I receive a new message. Lately I've been missing messages for a long time, and I noticed that the taskbar icon does blink, but it's a slow fade-in of a light color...not enough to catch my attention. Is there any way to tweak this?

I'm using the glossy theme with extra Compiz visual effects enabled.

---

### Post by adam.tropics on 2007-12-26
The guifications plugin will grab your attention with little popups. In the preferences, you can also set sound notifications, and lastly if you have the system tray icon enabled, that will flash like a trooper when you have unread messages.
