### Post by -Phi- on 2006-11-09
If you have the standard keyboard shortcuts, ctrl+alt+del should lock the screen. Alternatively you can attach xflock4 to a different key in the Keyboard settings. Or you can change the screensaver settings (in the Settings Manager) to lock the screen when you're idle.

To change user Quit > Log out. Then sign in again (you might have to sign in twice. For some reason it sometimes doesn't work?).
