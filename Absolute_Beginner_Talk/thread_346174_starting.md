You can NOT use native Windows application directly in Linux/Ubuntu, it will not work.

You do not need your ISPs CD in order to get the internet working. Depending on your internet setup (modem/router etc) it is done differently in linux but should not be a problem.

For your windows applications the best bet is to find linux equivalents. If you really have to run windows applications in linux then you must look into WINE or the commercial version called Cedega.

As sugested above maybe you should read up a bit more about linux/ubuntu.

If you need specific help post one problem at a time in the appropriate forum giving a detailed explanation of your problem. For example with the internet issue post in the networking forum, explain what hardware you are using, LAN card/Modem/Router/Cable/Wireless etc and you should receive help.
