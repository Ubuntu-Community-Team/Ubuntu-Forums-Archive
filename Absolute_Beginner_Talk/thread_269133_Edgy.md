### Post by Malac on 2006-10-01
The dates for the release of Ubuntu’s new bleeding edge operating system - Edgy Eft, are out. Here’s the release cycle -
 **September 2006**
  	September 7th 	- Feature Freeze
  	September 14th 	- Documentation String Freeze
  	September 21st 	- Beta Freeze
  	**September 28th 	- Beta Release**
 **October 2006**
  	October 5th 	- Kernel Freeze, String Freeze
  	October 12th 	- Non-Language Pack Translation Deadline
  	October 19th 	- Release Candidate, LanguagePack Translation Deadline
  	**October 26th 	- Final Release**
