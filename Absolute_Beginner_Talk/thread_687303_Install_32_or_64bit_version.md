### Post by p_quarles on 2008-02-04
Either one will work. You'll get noticeably better performance in *some* tasks with the 64-bit version. The gain will be seen mostly in CPU-intensive tasks, and not in normal desktop uses like web browsing or word processing. 

The 64-bit version has become very easy to set up and use at this point. The only difficulties are with some closed source apps, but there are workarounds for these if you're willing to put the time into looking. 

If you don't want to deal with workarounds, and are willing to use less than the chip's full power, go with the 32-bit version. For what it's worth, I switched to 64-bits about three months ago, and haven't regretted it once.

EDIT: simonalpha is correct. the Pentium M is a 32-bit processor, so you do not have the option of using the 64-bit system.
