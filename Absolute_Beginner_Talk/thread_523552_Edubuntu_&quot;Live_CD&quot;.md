### Post by henriette_holm on 2007-08-12
Hi.
All the installation cds work as a live cd, so you just have to boot the cd, pick the 1st option in the boot menu and wait. This wil NOT install the OS on your disk.

### Post by Jimmyfj on 2007-08-12
The Second item on the Live Cd is for lesser heavy load on the graphics card - Use the "Start Ubuntu in safe graphics mode" option if you cant start up Ubuntu with the first option. Also there's an alternate install for all Ubuntu versions too. This is for less powerfull systems with small amounts of system RAM.
