### Post by Michl on 2007-03-01
Well, this is resolved and maybe this will save some
grief to another rank beginner like me.

On dial-ups and when loading large IMAP accounts,
Evolution loads slowly.  During this time the window
freezes.  If you force quit, and reload, you return to
the frozen window.  But if you give it time to load, maybe
two minutes or even more, everything returns to normal.
After the first time, loading is faster, but the window
still is frozen during this time and will frighten nervous 
first-time users like me.

Michl
