### Post by Tatty on 2008-03-31
I think you will need compiz enabled to do that.  

Im not totally sure how, but i vaguely remember setting beryl up to do that a long time ago, so im assuming compiz should do that now.

You will need to install the compizconfig-settings-manager with this:

```
sudo apt-get install compizconfig-settings-manager
```

and then go to System->Preferences->Appearances and switch on compiz.  There is also a button on there which will let you change the compiz settings, the option you want will be somewhere in there.

**EDIT:  Also, by default i can swich desktops using the mousewheel on the desktop when i have compiz enabled**
