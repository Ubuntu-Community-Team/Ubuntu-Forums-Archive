factory default partition? What is that?

You can use the Kubuntu (and Ubuntu) install disks to resize and edit existing partitions. No, the entire drive doesn't need to be erased, you can shrink the Windows partition and create new ones.

Make sure you defrag in Windows and backup anything you need. The install sequence is reliable, but if something goes wrong, problems can occur.

I use two small partitions for Windows and Ubuntu and a large one for data to be shared between them.
