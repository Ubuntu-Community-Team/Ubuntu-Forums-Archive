### Post by lennyjames on 2006-07-14
I am making a transition at my home office from Windows to Ubuntu. I have been using The Bat for over a year and it has been critical for handling all my business mail. Now I am uncertain how to export all my mail boxes and which program would be best to handle my business mail from now on. Any suggestions?

### Post by bohboh on 2006-07-14
I went through the same exercise about 2 months ago, i was using the bat for about 5 years so had loads of emails, i wanted to be able to share between xp and ubuntu.

1. What i did was to install Thunderbird in ubuntu and to create a profile.
2. Then in the bat go through each folder, select all emails and export to unix mailbox. This will create a file with no file extension. Do this for every folder. You will end up with a file for every folder. (Name the file the same as your folder)
3. Copy the files to your Thunderbirds profile directory0 (Which since it is in ubuntu you will probably need to somehow transfer onto something else , cd, usb key). 
4. Open TB and you will have all your folders with all your emails.
5. You can drag the folders around as you please.

If you need any more info then ask.

HTH
