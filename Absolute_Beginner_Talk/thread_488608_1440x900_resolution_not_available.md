### Post by NilsE on 2007-06-30
Try either using the 915resolution utility to force the resolution you want.

or

Since you have an intel chip try to install the following in synaptic

```
xserver-xorg-video-intel
```

It will likely ask you to uninstall the i810 driver which is installed by default. You can always reinstall the i810 if it gets no better.

you can also go into a terminal and run the following to recreate the xorg file and select the resolutions you want.

```
sudo dpkg-reconfigure -phigh xserver-xorg
```

If you install the intel driver you may also have to run the above command.

### Post by alienexplorers on 2007-06-30
If these suggestions above don't work you might try a modeline in your "monitor" section of you xorg.conf...  The mode line would look like:

>   # 1440x900 @ 60.00 Hz (GTF) hsync: 55.92 kHz; pclk: 106.47 MHz
  Modeline "1440x900_60.00"  106.47  1440 1520 1672 1904  900 901 904 932  -HSync +Vsync

This is a modeline for 1440x900@60hz.  To make another modeline goto:

> [http://www.sh.nu/nvidia/gtf.php](http://www.sh.nu/nvidia/gtf.php)
