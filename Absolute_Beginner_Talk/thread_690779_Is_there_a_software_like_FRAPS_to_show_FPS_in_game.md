### Post by kesman on 2008-02-08
Most games have an option to show fps. For linux native games, the config file is usually somwhere under your /home/usernam/.gamename/ -directory. For example nexuiz for me, the config would be /home/kesman/.nexuiz . From there you may find a line similar to "printfps 0" or "showfps 0" which you should change to "printfps 1" to get fps to be shown. Also browse trough the ingame-menu for fps options. Google also helps, try with "gamename show fps"
