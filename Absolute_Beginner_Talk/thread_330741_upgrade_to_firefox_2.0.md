### Post by aysiu on 2007-01-04
Yes, it can be used for future versions, but once you have 2.0 anything you can update to 2.0.0.1 or 2.0.0.2 or any other 2.x release by launching it as ```
gksudo firefox
``` and checking for updates.

Once 3.0 gets released, you'd have to run the script again.
