### Post by ewl1217 on 2006-07-17
In the installer, there is no default account made. It prompts you to create your own account, so you're out of luck if you can't remember the password. You'll be stuck doing a clean install most likely. If you need to recover anything from it, you could try using a live CD to retrieve what you need and save it to some external storage device (flash drive, floppy disk, CD, etc.).
