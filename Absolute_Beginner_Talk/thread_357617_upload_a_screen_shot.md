### Post by Atreus12 on 2007-02-09
You could open the screenshot up in gimp and save it as a jpeg. That should make it smaller. Or you could shrink it in gimp.

Go to appications -> graphics -> gimp
hit 'ctrl+o' to and open your file
go to file -> save as ->select file type (jpeg)
click save and choose the % compression.

85% compression should be roughly half the size of the origional png file.

alternately
open your file as before
image->scale image
and choose a smaller width or height (it keeps the image ratio by default)
and save (as png or jpeg)



There may be other, better, faster ways to do this as well.
