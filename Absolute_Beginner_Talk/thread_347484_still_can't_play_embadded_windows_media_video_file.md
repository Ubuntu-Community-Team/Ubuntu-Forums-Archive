### Post by mtalexan on 2007-02-06
Unless you have extra codecs install, it's my understanding that the only player that can even handle a wmv file is VLC.  I notice you do have the plug-in installed, but as far as I can tell there's a problem with the plug-in not working with streaming media (VLC won't even play media across a network connection for me).  The only solution I found was to install the firefox add-on "MediaWrap".  It allows you to run streaming media in an external application of your choice.  The screen of the embedded media comes up black with a button to run it externally.  When you first install the plug-in you can select which players you would like to automatically handle the file, based on the file type, or you can have it ask you each time.  With this add-on, all file types have worked just fine for me.

[Link to Firefox MediaWrap Add-on]("https://addons.mozilla.org/firefox/1879/")
