### Post by TrevorNet on 2006-02-23
I've been trying to switch over to Linux for a few years now. It's been hard because I am *NOT* a hardware person. Configuring my different computers to work under (place distro name here) linux has been a never ending nightmare. In fact, this past weekend I attempted to install Gentoo as my friend raves about it. Alas... I couldn't even get it past the install, much less into any sort of a configuration set-up. I've bought books, including the Bible series, that included CDs on GNU, RedHat, Fedora, Corel, and a few others... nada...

While I was messing with Gentoo on my desktop, I decided to play the PCLinuxOS card with an old laptop. You can boot it as a "Live CD" without having to actually install.  This was a good thing... I have to say that I was rather impressed with the Live Boot. So much so that I went ahead and did the actual install. You can even start the install from the Live Boot via a desktop icon. The install wizard helped me though the entire process. I have to say that PCLinuxOS was very straight forward. A few notes though :

1. Start the download, go out to eat, go to a movie, shag your significant other... check the download... repeat. Even though the iso fits on one CD, the hosting server is incredibly slow. :neutral: I was able to Quake on-line with 0 lag. (Wife was.... somewhere else.... probably watching tv as I neglected my husbandish duties)

2. My PCMCIA ether-card was not detected. A check of the wiki will provide the steps nessecary to fix this. The laptop is alive and kicking on my network.

3. Without knowing too much about the file structure of (distro name) linux, I have come to realize that PCLinuxOS's habbit of dropping everything into a /Union folder is a bit odd. I'm not really sure what the ramifications are.

4. KDE Desktop is the default. Personally, my limited experience has given me more time with Gnome, so I like it better.

Other than that... give it a try. The Live CD with option to install is nice. On the other hand, Ubuntu has become my classroom of choice. I have yet to really understand how to install applications outside of the "Add Applications" wizard, but with only 5 days under my belt, I am starting to feel more comfortable with Ubuntu.
