### Post by aktiwers on 2007-02-15
> hi
I was wondering how to change icons too, so thanks for the answer
but I have another question, how can I change the 'recycle bin' icon for a custom icon?

Go to your Home folder and press CTRL+h
then find the folder called .icons   (the . infront of a folder means its a hitten folder)

In there, you can find the folder that contains all your icons. If there is more than 1 folder, pick the one that you currently use as icon theme.

the trash icons are called:

user-trash.png (in Edgy)
gnome-fs-trash-empty.png (before Edgy)

user-trash-full.png (in Edgy)
gnome-fs-trash-full.png (before Edgy)
