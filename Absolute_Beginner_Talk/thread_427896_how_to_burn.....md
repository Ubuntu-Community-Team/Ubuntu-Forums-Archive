### Post by starcraft.man on 2007-04-29
> **ginnie6 said:**
> a bootable feisty cd in ubuntu? I see the option to burn the iso file but in windows nero gave the option of making it bootable. How do i do this is Ubuntu?

Ummm, if you are using any of the ISO files of Ubuntu or any other distro, you only need to burn them straight as Data CDs.

A simple gnome program to burn is gnomebaker, type into the terminal:

```
sudo aptitude install gnomebaker
```

Then just select to burn an iso from the tools menu, burn at a low speed, 12 or less to prevent errors.

k3b is also a good alternative. Its a bit more advanced in what it can do.
