Yes, you do need either his IP address or hostname.

> If so, how would I be able to get it via telling my friend how to find this information(his IP address, etc.). I've already downloaded gFTP, I just need some help figuring out how to use it.

If you want to use FTP for transfering the files, one of you will need to have an FTP server and the other one will need an FTP client. gFTP is a client, so your friend would need to have setup an FTP server on his computer.

I don't know how to obtain an IP address or how to set up an FTP server on a Mac, but somehow I suspect that if he doesn't know how to find out his IP address, he doesn't know how to set up an FTP server either.

Which IM protocol/service are you using? Kopete might have better file transfer support than Gaim, although it's for KDE and I abhor its user interface. It's possible to use it in Gnome, though, just like you can use more applications across desktop environment borders. For MSN there's also kmess which at least claims to have file transfer support, and for ICQ there's Licq which might support file transfer. I can't remember if it does but it might.
