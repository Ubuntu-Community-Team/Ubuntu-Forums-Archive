### Post by Bias on 2007-04-16
Normally, once an image is on a USB stick all you need to do is tell the BIOS to boot from it.

Some older systems don't have the boot from USB option in the BIOS but that's a limitation of the BIOS. If that's the case then you may be able to flash the BIOS with a newer version downloaded and put on a floppy (normally). Sometimes this functionality is disabled in the BIOS in which case just enable it.

Hope that helps.
