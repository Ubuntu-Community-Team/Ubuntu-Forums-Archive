### Post by nvteighen on 2007-10-15
Are you using a GDM theme that supports faces? Go to System -> Administration -> Login and select the "Local" tab. There you can select either a theme with faces-browser or deactivate themes and use a standard login box with faces. Play a bit with the options... Or, if you don't like any theme, you can install a new one from there (search at [http://www.gnome-look.org/](http://www.gnome-look.org/) there are hundreds of them).
