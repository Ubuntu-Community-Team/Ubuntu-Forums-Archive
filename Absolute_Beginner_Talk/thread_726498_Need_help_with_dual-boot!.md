### Post by neurostu on 2008-03-16
Yes there are hundreds of tutorials. If you google for "Install Ubuntu after XP", "Dual boot Ubuntu XP", "Install Linux after XP", "Resize XP partition", it should come up with a couple good guides.

A good place to start it to boot the Ubuntu Live CD then under Applications-->System Tools--> Gparted. You can use this to resize the XP partition before you install ubuntu.  

Also the ubuntu install process can also be used to partition the hard drive.

Good Luck and welcome to the fun world of ubuntu.
