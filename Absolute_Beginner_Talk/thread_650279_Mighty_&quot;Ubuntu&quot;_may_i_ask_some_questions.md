```
sudo apt-get install totem
```

The commands listed do several things:

"sudo" means run as root. It allows you to administer the system

apt-get and aptitude are used to install and remove packages. They can also be used to update your system and upgrade to the newest version of Ubuntu.

"install" means install, and whatever follows it is the package name. You type them in the terminal. You can also do it graphically in Synaptic, but that takes longer.

The programs listed are media players, I use Vlc myself, which is also available for Windows.

The commands can be used to install an application, but they also look for other packages that are needed. For example, Opera (the web browser I use) needs a shared library that is not on Ubuntu, the installer for Opera doesn't have this package, but when you install Opera, Ubuntu will see that it needs it and finds it.

The ISO image is a sort of compressed file that contains the file for the Ubuntu CD/DVD, it is very essential that you do not tamper the image in anyway, that is to modify it or anything else.

Here is a good [guide]("http://www.petri.co.il/how_to_write_iso_files_to_cd.htm") you can use to burn the image file.
[B]
IMPORTANT:- It is recommended to burn the ISO image to the disc at a speed of 4X or less, a greater speed could corrupt the Live CD/DVD and cause some unpleasant problems.[/B]
