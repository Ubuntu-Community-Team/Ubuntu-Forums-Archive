We need to know a little more about your computer, I'm afraid. The easiest option is if you have an external modem that's connected to the computer by the serial port. Most likely, though is that you have an internal modem and a cable that connects the PC directly to the phone line. In that case we have to work out what modem you haver and see if we can find software (drivers) for it. [http://www.linmodems.org/]("http://www.linmodems.org/") is the place to start a journey of discovery into how your computer works  said:**
> also  what  is KDE what makes  diff  from gnome ? 
KDE and Gnome are both [window managers]("http://xwinman.org") and they decide what your Desktop, toolbars, window frames and other visual interfaces look like and how they work. Both have features which are better than the other, although KDE is usually easier for a beginner from a Windows background. [ducks the flames]
