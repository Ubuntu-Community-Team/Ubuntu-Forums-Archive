### Post by Qrk on 2006-06-30
I'm afraid you won't be able to just install a AMD 64 kernel like you would a K7 or i686 kernel. You'll have to get an AMD64 ubuntu CD if you want the 64 bit version.

However, there aren't any exciting "features" in the 64 bit Ubuntu. Many people don't think its worth it. In fact, my friend just bought a 64 bit laptop but I was surprised to find 32 bit windows on it.

### Post by Qrk on 2006-07-01
Yup, x64 isn't compatable. i386, i686 and K7 are all 32 bit, so you can run 386 binaries on all. But with 64 bit, you'll have to build from the ground up.
