### Post by Neobuntu on 2007-03-25
Yes, just tell it to take that partition and swap partition.

OR just delete both (unwanted and swap) partitions (the right ones please if you also run other OS's) with qtparted (or gparted) from a live CD and simply tell the Kubuntu Dapper installer (or Edgy if you are not as concern about maximum usability and stability. it's quite fine now but just not as rock solid as Dapper.), to install in the blank space left over.

For 128MB to 256MB use the alternate non-live install (Kubuntu) CD. Then (see forums) tweak Kubuntu for less memory.

If you give this an honest try and ask for help (free online) where needed, then you will be VERY thankful you did. 

I can't tell you how many myths this will clear up. Just do it!:KS
