### Post by papabean on 2005-07-05
I can't recall if I changed this preference myself or not, but if I press F11 the game will go into full screen. What's another game that opens in a window? 

If I did change that keypress, I did it by going to the menu and selecting System -> Preferences -> Keyboard Shortcuts. There is an option in there for Full Screen. And you can Full Screen just about anything. :grin:
