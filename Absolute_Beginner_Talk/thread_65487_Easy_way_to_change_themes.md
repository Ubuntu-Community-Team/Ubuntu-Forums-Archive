### Post by frodon on 2005-09-14
Open the themes window in System>Preferences>Themes, then go to the [http://www.gnome-look.org/](http://www.gnome-look.org/) site to download a gtk/metacity theme. To install the theme just drag and drop the downloaded theme in the theme window.
