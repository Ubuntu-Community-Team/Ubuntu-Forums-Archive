### Post by bollix47 on 2006-06-30
Places - Search for files... 

Change Look in folder to filesystem

Enter Name contains...

Edit:  If it's always a particular folder that you're searching in then you can add that folder to places and use that in the search criteria.
To save a search in Nautilus click on File - Save search as
after doing a search.  Option doesn't show up until you do a search.

Nautilus search is only intended for use in finding, for instance, a document; it searches your home recursively for that.

If you're looking for, say, a library in /usr/lib, it's more useful to use locate + grep.

And, yes, Nautilus' search can search in specific folders. From Search Results, click the little + button, and in the location menu go to "Other..."

### Post by aysiu on 2006-07-01
I think people are missing the point here.

Yes, you can use locate and grep.
Yes, you can go to Places > Search for Files and specify a location.

The OP, however, is talking about functionality that should exist but doesn't. If you're already in Nautilus--say, in /usr/lib--and you press Control-F, Nautilus will allow you to search... your home directory... instead of searching the directory you're already in, which would make far more sense.
