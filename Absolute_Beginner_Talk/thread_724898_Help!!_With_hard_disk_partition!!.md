### Post by uberlube on 2008-03-15
Download PartedMagic and burn it. It is a live CD just like Ubuntu but it's primary purpose is for HD recovery and partitioning. Boot it just like ubuntu and use Visparted to reformat the Ubuntu partition into whatever you want. This is a great tool that no geek should be without! :)
[http://partedmagic.com/wiki/PartedMagic.php](http://partedmagic.com/wiki/PartedMagic.php)
