### Post by muggins on 2007-07-03
When installing frostwire I also got sun java 5 installed with it. However I already had sun java 6 installed, do I need both versions or can I uninstall one of them and if so which one?

---

### Post by jrib on 2007-07-03
You only need one version.  I would uninstall the older one.

You can configure your default java using 'sudo update-alternatives --config java'

---

### Post by Raineer on 2007-07-03
You **can** keep more than one if you need, if you have apps that need the lower versions. It's as easy as setting your PATH to point to that Java dir for a terminal.  If you don't have that need then it's fine, but it's another point of flexibility I love about Linux.
