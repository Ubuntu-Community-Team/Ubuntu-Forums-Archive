### Post by HDave on 2007-12-08
Go to System->Preferences->Main Menu to launch a program that will allow you to turn menu items on and off via checkboxes.

That said, there is no "Device Manager" under Adminitration that I have ever seen (in Gutsy).

There is something called "Hardware Information" under preferences however.  There is also the commands lsusb and lspci which you should read up on.
