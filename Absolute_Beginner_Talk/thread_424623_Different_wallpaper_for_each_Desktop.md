### Post by Scheater5 on 2007-04-26
I'm not sure about the specifics in GNOME, but it's under your system settings.  In KDE, that's System Settings (K-menu -> System Settings or K-Menu -> KControl).  In GNOME I believe it's under the menu Adminstration -> System, and there you will find an icon for Desktop Background, or something to that effect.  There will be an option, most likely already selected, to use the selected background on all desktops.  Deselect that, and then choose backgrounds for each desktop in turn.  Perhaps someone else can give you more detailed instructions if necessary in GNOME.  In KDE, under system setting, click on the icon desktop and change All Desktops to Desktop 1, and select a wallpaper, and then change it to Desktop 2, and then select another wallpaper.
