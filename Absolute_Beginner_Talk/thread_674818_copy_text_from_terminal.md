### Post by jd65pl on 2008-01-22
You could output to a file.

```
cat file1 > ~/Desktop/file2.txt
```

e.g. I want to post my xorg.conf so I would

```
cat /etc/X11/xorg.conf > ~/Desktop/my_xorg_conf.txt
```

You don't need to put the files on your desktop you could put them anywhere you have write permission.
