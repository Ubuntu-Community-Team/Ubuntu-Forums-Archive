### Post by Celegorm on 2007-05-26
To compile C code, type ```
gcc filename
``` at the command line, for C++ ```
g++ filename
``` This will create a file called a.out. Type ./a.out to run your compiled program.
You can use whatever program you wish to make and edit text files for the source code. Emacs and vim are pretty popular for programming.
