### Post by catlett on 2006-05-13
Xubuntu is the ubuntu choice for an older computer. It has XFCE as it's window manager as compared to kde for kubuntu and gnome for ubuntu. This is the install wiki [https://wiki.ubuntu.com/InstallingXubuntu](https://wiki.ubuntu.com/InstallingXubuntu) This is it's site for a litle more info  [http://www.xubuntu.org/](http://www.xubuntu.org/)
