### Post by oakgrove on 2008-03-03
Hold down the alt key then press the f2 key to open up a Run Application box.  Type gnome-terminal then press enter.  A terminal will popup.  In that terminal, type vmware and press enter.  Copy and paste the results from that into a new post in the forum here so we can help you.
