### Post by aJayRoo on 2007-09-30
I am using Sun Java 6 and with this it is very easy to do what you want. Simply right click on the jar file in Nautilus and choose properties, then select the open with tab and select the Sun Java runtime option to be the default. This will allow you to just double click on any jar file to launch it properly. I do not know how to do this or even if it is possible with the other JREs available, so I am sorry but my advice is limited to Sun Java Runtime!
