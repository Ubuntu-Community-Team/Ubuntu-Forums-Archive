### Post by forestpixie on 2007-08-24
once you have you're ubuntu installed I would let it get all of it's updates and then to get Edubuntu

sudo aptitude install edubuntu-desktop

similarly you could replace edubuntu with kubuntu or xubuntu

to remove 

sudo aptitude remove edubuntu-desktop
