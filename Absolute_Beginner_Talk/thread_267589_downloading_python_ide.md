### Post by DirtDawg on 2006-09-28
I'm not completely clear what you're asking, but I'll try to help you out here. 

The IDE that's an official part of Python is called 'Idle'.

There's a great tutorial called "[How to install anything in Ubuntu]("http://htmldesign.dk/blog3/ubuntu/installing/")". Take a stroll through that and make sure to read the part about enabling extra repositories, if you haven't already.

Of course you can also use the "sudo aptitude" command, which you appear to have a good handle on.

Idle is very good, but also very basic. If you decide to try something a little more powerful, I recommend [SPE]("http://stani.be/python/spe/blog/"). It's also available for free in the repositories.
