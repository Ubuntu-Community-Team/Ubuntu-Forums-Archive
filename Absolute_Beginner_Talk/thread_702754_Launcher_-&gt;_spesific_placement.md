### Post by drs305 on 2008-02-20
One program that can do that is devilspie. It can open an app in a particular workspace/viewport or with a specific location (I haven't been able to do both at the same time).  Caution: devilspie can be frustrating to set up but works pretty well once you get to know what it can and cannot do.

You can put the commands in System, Preferences, Sessions, Startup Programs (in Gutsy at least) to get them to open when the system starts.

### Post by spiderbatdad on 2008-02-20
You can also open a nautilus window Edit>>Preferences and de-select the open in browser option. Nautilus will then remember window sizes and locations. No need to install devilspie (crap) IMHO. And really I shouldn't put the software down. A lot of work went into creating it, and it can be fun.
