### Post by mediamack on 2007-03-25
I think you mean being able to put gaim on the side of the screen and then be able to maximize any other window and it will only maximize up to the side of gaim. 

I've looked forever for a solution and couldn't find one so what I did was put an extra taskbar on the right side and make it the same size as my gaim window. Then just set your gaim window to be on top of all others so it stays on top of the extra taskbar. Now you can maximize your other windows and they'll only go to the edge of the gaim window :) 

This is in kde without a window border, not sure if it would work otherwise. Might be a similiar way to accomplish it in gnome but I'm not sure.

If there is a cleaner way, feel free to tell.
