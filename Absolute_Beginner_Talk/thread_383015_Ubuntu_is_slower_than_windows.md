### Post by russell.h on 2007-03-12
In my experience Linux takes longer to boot than windows, but thats because when Linux is done booting its ready to go right away. I dual boot, and Ubuntu takes probably 1.5 times as long to boot as windows does, but when Ubuntu reaches the login screen its ready to go, whereas windows shows you the login screen while its still loading all sorts of crap in the background.

Thats why if you boot to windows and log in everything will go really slow for another 15 seconds to a minute. From the time that I press the power button the time before the computer is pleasantly usable is about 5-10 seconds less in Ubuntu than in windows (also note that in Ubuntu since most services aren't attached to a user, login times are less than in windows). I think that on older machines that time might be smaller, or depending on how much stuff you have installed in windows it might even be faster on some machines. I know when I first installed XP it was surprisingly "snappy", but its gotten very bogged down over time.
