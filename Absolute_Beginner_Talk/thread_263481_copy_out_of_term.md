### Post by steve.horsley on 2006-09-23
If this is a text console like you get when not running in a GUI, you cannot. The best you can do is to pipe the output of a command to a file like this:
**ls -l > listing.txt**

If you are running a terminal in the GUI there are 2 ways to copy text.

1) Highlight the text you want to copy by dragging the mouse over it (with the left button held down) and then use the manu and select Edit->Copy. This can be pasted elsewhere with the normal paste copmmand.

2) Highlight the text you want to copy by dragging the mouse over it (with the left button held down). Leave the text highlighted and go to the destination window. Paste by clicking the middle mouse button (or both the left and right mouse buttons at the same time if you don't have a middle button).
