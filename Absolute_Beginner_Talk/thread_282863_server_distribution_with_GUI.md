### Post by Bachstelze on 2006-10-23
Either install via the Server cd and then install the GUI manually with

```
sudo aptitude install ubuntu-desktop
```

(or kubuntu-desktop, or xubuntu-desktop, depending on the DE you want).

Or install the "full" system with a Desktop or Alternate CD and then install Apache, MySQL and PHP manually :)
