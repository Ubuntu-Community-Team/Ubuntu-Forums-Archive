### Post by camini on 2007-11-15
Add ccsm (compiz settings manager) from add/remove applications.
Then use the manager from system/pref.

---

### Post by bulldog on 2007-11-15
You need to install a package named compizconfig-settings-manager.
Open synaptics and search for this package,install and you can find it in your menu--system--preferences

---

### Post by Yoooder on 2007-11-15
...to add a note to bulldog's advice: after adding the package there will be a fourth menu item in the Appearance tab called "Custom" that will have a button that will take you to the advanced settings manager.

Doesn't matter if you use that to get to the manager, or through the menus--just an FYI.
