### Post by neurostu on 2008-02-15
I'm not familiar with Joomla, but it is pretty easy to put a website on a CD.  Simply create a directory with all of your files, and make sure that all of your links are relative not absolute links (img src="images/image1.jpg") not (img src="c:\dir\image\imag1.jpg").

Once you have your entire site built and saved to a directory simply burn that directory to a CD and then open the site in a web browser with File->Open File-> Select the index page of your site.

That should do it!

Good luck!
