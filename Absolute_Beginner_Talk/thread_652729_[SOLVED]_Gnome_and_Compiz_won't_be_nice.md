### Post by chewearn on 2007-12-29
> **juniah said:**
> I've noticed that regardless how many times I do

```
metacity --replace

```1.  The terminal box where I type that command, the prompt is never returned

Press ALT+F2; you will get a "Run Application" dialogbox.  Then you can type in "metacity --replace".

> 
2.  That Compiz is loaded on startup and not gnome.

How do I fix this?

Go to Top Panel Menu > System > Preferences > Appearance > Visual Effects tab > Select "None".

That will turn off compiz.
