### Post by mcduck on 2007-01-16
Desktop CD works as a live-CD allowing you to test Ubuntu without installing. It also has a graphical installer, and some free software you can install in Windows. Desktop CD needs at least 256MB of RAM to run.

Alternative CD has a text-mode installer (just as easy as the graphical). It only needs 64MB of RAM and also has option for OEM-install and other install options not available on the desktop-cd.
