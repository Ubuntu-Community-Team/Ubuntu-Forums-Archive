### Post by tribaal on 2006-09-02
Hi there.

I'm really not into 3d modelling so I guess i cant' really help you on that one.
But for the rest I'll give it a go :)

Ok, so basically ATI drivers aren't near as good as Windows ones. So if you're of the gaming type, you're likely to be disapointed by your FPS count.
Most ATI graphic cards work however, even not to their full capacity because of the drivers.

For the installation you'll need a partition, yes.

The Desktop CD (the liveCD) will prompt you where to install linux during the installation. It will detect any other OS on the hard drive, and will give you an option to resize a partition (an already existing one) to make room for Ubuntu. So you don't really need any particular skill to install Ubuntu.

For your data's security however, I recomment you backup any sensitive data before touching your hard drive's partitions. Also not that the default install option is to wipe the whole hard drive and use it. Read choices carefully during install, and don't "Click next" all the time like in windows.
