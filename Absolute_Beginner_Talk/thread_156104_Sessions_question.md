### Post by carlosqueso on 2006-04-06
For you, default is GNOME, if you were to set another window manager (say.....XFCE or IceWm like me) as your default, that would select it.   Then you'd need to select GNOME to get into GNOME.  The GNOME failsafe is a backup GUI, I'm not really sure how it differs.  The failsafe terminal has no GUI, in case you royally screw up GNOME (which I have done).
