### Post by Jussi Kukkonen on 2007-09-25
You can't install a PPC version on x86. So you can't really skip the downloading... Installing without CDs is possible (search for netboot in the wiki), but the amount of downloading will likely not be any smaller in the end...

As a side note: My advice is to buy more RAM and install the newest version of Ubuntu... (EDIT: 128MB will be fine for non-GUI use though)
