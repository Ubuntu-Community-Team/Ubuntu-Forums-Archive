### Post by kleo skywalker on 2007-11-22
Search for flashplugin-nonfree in Synaptic. If it shows up green, uninstall it.
To install Gnash, search for Gnash in Synaptic, and install whatever you need. From what I can see, you only need to choose mozilla-plugin-gnash, Synaptic installs whatever else is needed.
