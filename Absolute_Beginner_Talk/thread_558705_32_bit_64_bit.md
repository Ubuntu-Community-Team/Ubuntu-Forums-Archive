### Post by mali2297 on 2007-09-24
In the terminal, run
```

uname -m

```
If it says *x86_64* then you have a 64bit OS, otherwise it says something like *i686*.

### Post by misfitpierce on 2007-09-24
Depends if you have 64 bit or 32 bit ubuntu installed. Did you install amd64 ubuntu or Ubuntu i686/i586/i386 and so on. Can check with program hardinfo under software info also. 

Terminal: sudo apt-get install hardinfo
