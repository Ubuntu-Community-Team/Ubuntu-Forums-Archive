### Post by bulldog on 2006-12-13
Well,ask yourself what you want to do with the distro.
How is the feedback when things go wrong,is there something you can do with another distro,and not with ubuntu.

Basically the difference between distro's isn't that significant,it's more the ease to install applications.
I have used Suse and Fedora for a while,and they can do things well.
But there were always little things which annoyed me,and then I ran into Ubuntu.
It does all I want to do with it,I can experiment with it,install applications and there are Howto's for everything you can imagine.
And if you get in trouble,you can ask help on the forums.

So I stick with Ubuntu,because all I want is right here.
