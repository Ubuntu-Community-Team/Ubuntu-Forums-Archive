### Post by FuturePilot on 2008-03-15
Gcursor is kind of broken right now. But you can use the Appearance Properties to install cursor themes, it has the same functions of Gcursor. Go System&#8594;Preferences&#8594;Appearance and drop the theme archive in the window. If it asks you if you want to use the new theme, **don't do it**. Select Keep Current Theme. (it thinks you installed an icon theme and it will try to set the cursor theme as the icon theme and if you happen to log out you'll have a lot of trouble trying to log in again) So click on Customize Theme and go to the Pointers tab and select your theme. Sometimes it doesn't show up right away and you need to close and reopen the entire theme manager.
