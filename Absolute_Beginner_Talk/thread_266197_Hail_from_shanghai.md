"Is ubuntu really lighter and faster than XP ? in term of resources allocation ?? is it really more Stable than windows ??"

YES it it is lighter. Altho if you add more functions ( install kde pack, use compiz and so on it will get more rescource consuming. As with all OS'S it will depend on what you do with your system.

Stability is a dubious issue. There are still a lot of hardware issues ( the same goes for windows, altho some might argue there which OS is worse ), but linux's name for stability comes more from the software side. As it does not use the mess of a registry system as windows does Linux's stability will not deteriorate when you add/remove programms. In addition Windows systems tend to "tainted" by malware more often, thus tempering with the stability ( and not unimportant either: SPEED )

So in general one COULD say that Linux is more stable.
