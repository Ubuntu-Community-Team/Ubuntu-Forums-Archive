### Post by 3rdalbum on 2007-01-04
trevorpsucks, have you double-clicked the icon on the desktop that says "Install"?

A lot of new users don't realise what the Desktop CD actually does. When you boot the Desktop CD, all it's actually doing is booting up the operating system into your memory, so you can try it out first. If you like it, then you double-click the Install icon and it installs to your computer.

I hope this clears up some of it.
