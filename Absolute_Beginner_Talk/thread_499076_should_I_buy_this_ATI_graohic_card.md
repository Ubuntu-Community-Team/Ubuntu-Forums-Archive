### Post by Zannax on 2007-07-12
I've got a ATI Radeon 7000 and have had some minor problems using it with Feisty, nothing serious but the bigger issue is that ATI proprietary driver only support cards **newer** than the 7000 one!

I use the opensource "ati" driver and DRI acceleration works, but some of the problems I've noticed include 3d games running slow (i.e. flightgear, scorched3d has some glitches, quake2 runs fine but has some problem with textures), problems with composite extension (desktop effects, they work but hang Xserver when switching user, I had to disable compositing)...

Maybe there's a simple way to fix everything (I'm stil looking for) BUT the bad thing is that I can't check if proprietary driver work better since this card is not supported...

So my suggestion is to buy a newer one, even if a used one, but at least a model which is supported by proprietary drivers too.
