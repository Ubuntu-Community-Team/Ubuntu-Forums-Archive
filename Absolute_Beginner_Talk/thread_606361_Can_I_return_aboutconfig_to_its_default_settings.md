### Post by ntu on 2007-11-07
Is there a way to get back to the default settings for about:config? I made some changes, can't recall what they were, and now wish to change them back.

---

### Post by Hospadar on 2007-11-07
you could probably just delete the ".mozilla" folder in your home directory.  Note that this will also remove your bookmarks, so you should export those first (Bookmarks->Manage->Import/Export) so you can reset them.

If you do this, I would just change the name of the .mozilla folder so if stuff gets screwed up you can easily just change it back

Note that this folder is hidden, so go to View->Show hidden folders.
