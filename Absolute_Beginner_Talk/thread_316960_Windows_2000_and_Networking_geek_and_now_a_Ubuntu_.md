More than possible - quite easy, in fact.
Fire up the install CD and follow instructions, resizing your Windows partition to make room for Ubuntu when prompted.

> 2.  There are already existing dell hardware on my machine, do I need drivers for all of them to work in Ubuntu? Where do I get these and how do I go about installing them also?
First it always helps to know exactly what hardware you have in your machine, so you should check that out - it will make things much easier later on.
Having said that, you don't usually need to **get **drivers for Linux - Linux **has **drivers.
Built-in, as it were.
Try it first, worry later :)

> 3. Since I am really really really new (and probably blind) about terms and the Linux OS, where do you think I could start reading and learn from first before I do venture into this world.... Im really lost and dont kow where to start but I'm trying my best to understand things... which Im not apparently successful with... if anyone can point me in the right direction, what to do and what not that would really be great!!! 
A few good (if not essential) tips:
1. Get comfortable with using the command line ASAP; there are many good guides out there, of which the **rute user's guide** is hands down the most complete _and_ made *especially* for beginners.
This will also explain concepts that are important on Unix.
2. Don't run as root unnecessarily - thankfully, Ubuntu actually enforces this for first-time users, and encourages it everywhere.
3. Don't worry about messing things up - experiment! 
(**Permanently** screwing things up is actually not that easy - certainly not as easy as it is on WindBlowz)
