### Post by jw5801 on 2007-08-10
Firstly, does the keyboard work in your bios? If not, then you have an issue. :p

If it does then you may have the same issue I had, where it won't work in any pre-OS menus (ie. the one that loads with the LiveCD), but it will work once it actually reaches the OS. Or it may still not work in the OS. In either case the way to fix the issue lies in your bios: there should be a setting in there somewhere to "enable support for usb keyboards" or "enable legacy support for usb input devices" or something along those lines, which you need to enable in order to get your keyboard working properly.

EDIT: Apologies, I assumed that it was a USB keyboard/mouse, if it's not then disregard the above.

EDIT 2: Google reveals that the Logitech diNovo is a bluetooth device! Which means that the solution remains the same, there should be a setting in the bios that will enable support for bluetooth devices, just look for options referencing bluetooth rather than USB.
