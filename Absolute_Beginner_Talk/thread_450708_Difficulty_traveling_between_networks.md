### Post by steve.horsley on 2007-05-21
In the default network manager, after you have all the settings working nicely, you can then save the settings as a particular location name. I forget the wording in Dapper, it was something like add a location, which was somewhat misleading.. But it functions as a **Save As...** for the current configuration, and then you can recall that config just by re-selecting that location.
