### Post by public_void on 2006-11-08
Segmentation fault is a problem with the program trying to access a memory location that its not allowed to. It isn't your fault that the program is doing that.

aMSN is available via Synaptic. Try either marking it for reinstallation, or for complete removal, then installing it again.
