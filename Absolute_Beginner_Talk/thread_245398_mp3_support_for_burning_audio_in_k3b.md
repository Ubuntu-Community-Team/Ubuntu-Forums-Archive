---

### Post by Kilz on 2006-08-27
> **cptjaben said:**
> I recently installed k3b after having problems with gnomebaker, when I try to add .mp3 files for an audio cd it says, 'Unable to handle the following files due to an unsupported format' Does anyone know how to make .mp3's a support format?

You need to install the libk3b2-mp3 package

```
sudo apt-get install libk3b2-mp3
``` 
should do it
