### Post by Edgeworth on 2007-12-01
I assume you are talking about login splashes. Gutsy, by default, has none. Just make a directory anywhere and use gTweakUI's session menu to enable them. Here's the code for getting gTweakUI:
```
sudo apt-get install gTweakUI
```
Hope this helps.
