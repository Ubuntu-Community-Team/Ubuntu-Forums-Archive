### Post by Bartender on 2007-12-07
Also, get used to checking things out with the "top" command in terminal.

I have a couple of 450 MHz/500MHz PC's with at least 512 of RAM.  It's interesting to start up various programs and see how hard they worked the CPU.  

Don't use the System Monitor!  Just the act of starting System Monitor on an older PC will push the CPU usage up way higher.

As an example of some of the odd things you'll find, Sound Juicer would only use 7 to 15% CPU on the old PC's when playing a CD.  Amarok went up to 70% or more to do the same task.

I have a PIII 1GHz now for a test PC...huge difference over 500MHz.  Amarok only uses about 7%.  1GHz is still not enuf to do anything I want when I want.  Tried ripping a CD last nite - 80 to 90% CPU usage!  No multi-tasking until it was done.

So you can use an older PC, but it's not very useful beyond the act of seeing Ubuntu run and figuring out which buttons to push.  You'll need a newer PC to start having any fun with it.
