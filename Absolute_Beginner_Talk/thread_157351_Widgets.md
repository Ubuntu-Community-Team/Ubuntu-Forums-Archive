[QUOTE=dreamsINdigital]gDesklets[/QUOTE]
Yes, there is.  Just install it with apt-get...
```

```
Then, head over to [http://gdesklets.gnomedesktop.org/categories.php](http://gdesklets.gnomedesktop.org/categories.php) and download yourself some nice desklets.  :)
