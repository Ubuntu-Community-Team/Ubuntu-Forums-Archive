### Post by Patrick K on 2007-01-09
I solved this issue, at least in Gnome. Go to <System-Preferences-Keyboard> click on the Layout Options tab then "Alt/Win key behavior." Put a tick on "Alt and Meta keys are on the Alt key (default)". Now the right atk key works the same as the left. 

I don't know how to fix this in the console. I've read on it but nothing I've tried works.
