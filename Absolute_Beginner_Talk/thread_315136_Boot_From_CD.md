### Post by Henry Rayker on 2006-12-08
You'll need to set the boot priority in the BIOS to boot from cd first. The BIOS can be accessible from any number keys during the boot sequence. One mobo I have requires you to tap the F2 key (holding it won't do). I believe another requires F4...an my laptop requires the Delete key be tapped...

It may sound silly, but try "raking" the F keys...just swipe your fingers across them (unless, of course, you have a screen telling you which key to press). I use this method all the time and it works. If the F keys aren't the ticket, try the Delete key.
