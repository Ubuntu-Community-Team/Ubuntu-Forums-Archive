### Post by jjsonp on 2008-01-19
When I right-click on a file I'd like the option of 'Open with text editor' to always appear in the list of the popup window. Right now I frequently have to choose 'Open with Other Application' and then scroll all the way down the list to 'Text editor'.

How can I add this as a default option in the popup when I right-click? Thanks.

---

### Post by Lord Illidan on 2008-01-19
Right click on the file, select Properties -> Open With, and +Add text editor (The command is gedit)
