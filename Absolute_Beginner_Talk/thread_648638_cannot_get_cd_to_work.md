### Post by Rocket2DMn on 2007-12-23
You may need to try using the alternate cd - it doesn't have a live session and is a text-based install but it avoids a lot of problems that sometimes arise during installation with the live cd.  If you wanna give that a go, here's how:
[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) - check the box the Alternate cd at the bottom.  After you download, don't forget to check the md5sum
[HowTo]("https://help.ubuntu.com/community/HowToMD5SUM")
[Hashes]("https://help.ubuntu.com/community/UbuntuHashes") to check against
Once that checks out, burn the image to a cd at a slow speed, like 4x to prevent write errors, and use quality media if you can.
