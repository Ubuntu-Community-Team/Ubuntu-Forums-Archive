GNOME uses Nautilus, KDE uses Konqueror, Fluxbox uses Thunar by default. Ubuntu comes with GNOME, but you can install the others if you wish.

After installing Ubuntu, you can get the kde packages with this command:

```

sudo aptitude install kde-core

```

To change your desktop image(s), you right click on the desktop (usually). You can configure your Ubuntu in the System->Preferences menu. It is quite easy.

You can use Partition Magic, although I prefer GParted. The repositories are enabled, so you don't have to do anything if you are connected to the internet.

If you have any problems, let us know!
