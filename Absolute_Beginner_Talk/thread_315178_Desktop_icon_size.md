### Post by Dr. Nick on 2006-12-08
Open up a file browser and go to Edit -> Prefrences and look through their to set size. You can auto arrange them by right clicking on the desktop, but new ones pop up in  random spots

### Post by stalker145 on 2007-07-23
> **musky said:**
> Where might this file browser be?

I believe he's talking about Nautilus (in Gnome) or Konqueror (in KDE).

Another option (slightly more work) is to use gconf-editor (typed into the run box) and navigate through Apps~>Nautilus~>Icon_Size (if I recall correctly). You can change the size there as well.
