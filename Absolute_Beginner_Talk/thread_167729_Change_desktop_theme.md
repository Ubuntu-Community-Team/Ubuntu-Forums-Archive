You do not need to extract the files.

When you are going to install the theme, simply select the the tar.gz file that contains the theme you want. :)

Details:

1)System/Preferences/Theme
2) Install Theme
3) Browse and "open" your tar.gz or tar file :) 
4) Click "Install"

Voila! You got new theme now :)
