### Post by ramjet_1953 on 2007-06-20
Did you burn the downloaded iso image as an iso?

If you just copied the file to a CD it will not work and the picture you showed tends to make me believe that this is the case.

Most CD burning software has the option to burn an iso9660 image in the dropdown menus somewhere.

Two other things that are VERY important with burning iso images.

1. Burn the image SLOWLY, no faster than 4 X

2. Use HIGH QUALITY media. Cheap stuff does not "cut the mustard" with any iso image.

### Post by atria on 2007-06-20
It doesn't matter if you use a dvd. It should work fine (works for me at least). The key is to burn it as an image, dont extract the files before burning because the boot sector will be omitted (which is required for the livecd to boot). 

Anyway from the image above it looks more like corruption to me. Maybe the media is bad? Try burning it on another disc?

Finally, you might want to use Nero instead of poweriso.
