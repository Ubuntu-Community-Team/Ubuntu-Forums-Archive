A codec is used to decompress (on the fly) multimedia files that are compressed using a certain algorithm (I think).

If you have a look [here]("http://www.ubuntuguide.org/#codecs"), the site explains how to get them.

Be aware that this was written back before the repository guys got a legal warning from Microsoft, so some of the links won't work as they have been removed.  The main one you'll notice is w32codecs.  If you do a search in the forums I'm sure you'll find a link to them.

I hope this helps.
