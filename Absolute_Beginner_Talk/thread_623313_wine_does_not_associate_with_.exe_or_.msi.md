### Post by rouge568 on 2007-11-25
I would like to associate all .exe and .msi files with wine so that they open with a doubleclick. I know to put in a custom command for opening them, but it then only applies to individual files, not file types. Any advice? Tanks.

---

### Post by aktiwers on 2007-11-26
Right-click on the exe or msi file and pick Properties. Then go to the "Open With" tab and pick Wine. Now all exe and msi files should open in wine by default.
