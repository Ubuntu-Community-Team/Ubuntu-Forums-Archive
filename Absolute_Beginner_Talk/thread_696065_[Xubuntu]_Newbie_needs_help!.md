### Post by Arcnsparc on 2008-02-17
Q3 - I had the same problem, I reinstalled Compiz and Emerald and that helped.  If that doesn't work, try rooting around in the Gconf-Editor (run it by pressing alt & F2 to open a run command).  the editor has all kind of little hidden preferences for a lot of applications.  Some may have options in more than one place.  Just find the options that relate where to open the windows, it should be something like window x and window y.  I hope this helps, but im just a newbie so I cant vouch too much.
