### Post by bumanie on 2008-04-11
Beryl as a project is no longer developed nor supported. For the gusty release, it was joined with the compiz project and is now called compiz-fusion. Follow this [http://forlong.blogage.de/article/2007/10/2/Desktop-effects-by-default-in-Gutsy---how-Compiz-Fusion-enhances-Ubuntus-desktop-of-version-710](http://forlong.blogage.de/article/2007/10/2/Desktop-effects-by-default-in-Gutsy---how-Compiz-Fusion-enhances-Ubuntus-desktop-of-version-710) to enable desktop effects.
