### Post by engla on 2006-03-18
[QUOTE=LaBOOdha]
Is there a mac emulator?
[/QUOTE]
Not really, not what you want. There is Basilisk for (very)oldworld macs, you can run System 7 and stuff like that. It's fun if you're curious but if you want to try OSX, that doesn't work.

QEMU is a multi-platform emulator project, and they are aiming for being able to run OS X on their emulator; so far it doesn't work though.

If you run Ubuntu on a mac, you can use virtualization software, mac-on-linux that works just because there is no emulation needed. Now that OS X is available for intel however, this type of thing should be possible soon with VMWare or similar on x86 computers.
