### Post by prabbit237 on 2008-02-04
If the modem has ONLY a USB interface and not an ETHERNET interface (which is rare. Usually, they have either both or just enet.) then you might be SOL here. The other person linked to how to setup PPPoE on linux but that does nothing to get the USB port to work. If there's no enet port, you might need to check the manufacturer's website for drivers for linux (if it does have an enet port, just use that.)
