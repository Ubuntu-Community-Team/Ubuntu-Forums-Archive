Do you want to be the root user?

If a few people reply to this thread I will put the procedure here.

I didn't even know Ubuntu had a root user.  I was using the normal "initial" user.  In the process of atempting to make my Ubuntu system join my home network, I changed the name of the computer.  POOF!  I lost all privileges, and had to re-install Ubuntu.

I don't think this is any worse than if I had been the root user and had overused my privileges.  If I had a root user account I could have logged in as such, re-named the computer back to its original name, and be happy.

So, the warning are fine.  Don't operate as the root user unless you either know what you are doing or don't care anymore.  However, for Pete's sake, you are the owner of the machine and should be allowed to blow it up if you feel like it.

Let me know.
