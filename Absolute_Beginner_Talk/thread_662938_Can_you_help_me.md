### Post by kellemes on 2008-01-10
Well, I've been browsing the website of "Password Manager XP" and haven't seen anything KeePassX can't do.
KeePassX also can open multiple db's, you simply open multiple instances of the application in order to do so.. I have 3 KeePassX db's I use for different sorts of data, and for different purposes.

Like I said in one of my previous posts.. you are using a different operating system so you need to take the time to learn there are different ways of handling things, eventhough it's not like you're used to.
And if you're really depending on "Password Manager XP".. and can't get it running with Wine.. use a vm or dual-boot.
For setting up a vm see.. [https://help.ubuntu.com/community/VirtualBox]("https://help.ubuntu.com/community/VirtualBox")
For dual boot see.. [https://help.ubuntu.com/community/WindowsDualBoot]("https://help.ubuntu.com/community/WindowsDualBoot")
