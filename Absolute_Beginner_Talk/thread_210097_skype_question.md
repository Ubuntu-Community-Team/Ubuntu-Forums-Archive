### Post by nocturn on 2006-07-06
Skype works (and the new beta isn't too bad), but there are Free alternatives using the SIP protocol.  Skype uses a closed protocol, so interoperation is not possible.

I recommend ekiga or openwengo or even gizmo (which is closed source, but uses SIP anyway).

The up-side is that each sip application can talk to others.
