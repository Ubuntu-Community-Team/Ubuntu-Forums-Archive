### Post by ron999 on 2007-08-03
Hi
This sometimes happens to me too.
Try this:-
System > Administration > System Monitor

When System Monitor opens select 'Processes'.
Then right-click on ktorrent and 'kill process'.
Then go back to the menu and start ktorrent again. The icon should then appear in the top panel.
