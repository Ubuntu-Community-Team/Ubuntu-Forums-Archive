### Post by steve-shinn on 2008-02-15
If you are looking to build from scratch, the the starting point should be deciing upon the CPU you require, and then decide upon a Mobo that is compatible with the CPU. At the moment, the Intel core 2 duo CPU's offer the best bang for buck.
If you are not intending running games on the new build, then an AMD Am2 would be more than adequate.
