### Post by Odd-rationale on 2008-02-08
Have you installed compizconfig-settings-manager?

If not, install it via synaptic. Search for "ccsm".

Or
```
sudo aptitude install ccsm
```

Then open ccsm and go to "General Options." In the "Desktop Size" tab, set
Horizontal Virtual Size = 4
Vertical Virtual Size = 1
Number od Desktops = 1

Then restart compiz.
