### Post by petri on 2006-09-06
To get your mp3 and dvds running you need codecs. There is Automatix at [www.getautomatix.com](www.getautomatix.com) it can install some programs too.

The preferred starting programs are included in Ubuntu ;) but in Synaptic you can find more, some of the Kubuntus programs are quite nice and mostly even works in Ubuntu. They can have some problems with their depedencies or libraries because they are written to KDE which is Kubuntus desktop. And Gnome for Ubuntu of course.

These forums can give you several hints about programs but there is a free (?) linux magazine at [www.tuxmagazine.com](www.tuxmagazine.com) which is quite nice. Only problem with them is that they are KDE fans, I haven't found any magazine on internet for Gnome.

Hope this helped you a bit.

BTW Welcome to Ubuntu.
