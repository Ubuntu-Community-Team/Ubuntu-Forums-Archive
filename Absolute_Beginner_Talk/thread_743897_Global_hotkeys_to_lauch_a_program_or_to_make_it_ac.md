### Post by SunnyRabbiera on 2008-04-09
well your issue lays more with Gnome then with ubuntu linux itself.
Gnome doesnt fare so well with hotkeys, its a feature that is lacking in it I am afraid.
You may want to use KDE instead of gnome, now that has some good hotkey support.
Now if you are wondering what Gnome and KDE are they are two of the most popular desktop environments for Linux.
As opposed to OSX and windows linux offers you choice on what sort of interface you use.
Both KDE, Gnome and XFCE (another desktop interface) do the same thing but in different ways.
KDE seems to be targeted at windows users, it has many similarities to windows in its base layout.
Gnome is targeted at simplicity, the least amount of tools to do the same job.
XFCE seems targeted at lighter computers, it has a low memory footprint compared to Gnome and KDE.
If you want to give KDE a shot it is in the repositories, look for it in synaptic which is under system>administration>synaptic
you can also use it without messing up your system, linux supports using many desktop interfaces at the same time.
