### Post by ajgreeny on 2006-08-25
When I see there is an update available I open synaptic and mark all updates, then go to Custom in the bottom left hand corner where you can chose to see just the marked packages.  From there you can then see in the description what the package is and what it does.  I use kubuntu but never use adept updater as I much prefer synaptic as it gives me more details.

If you go to settings >preferences you can chose to "Show package properties in the main window" which gives even more info, and can be very useful, as in the broken xorg business last week.
