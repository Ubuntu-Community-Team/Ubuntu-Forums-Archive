### Post by LHZ on 2005-10-29
The 'easy way' is to open an editor like gedit, select HTML syntax highlighting (it's in preferences somewhere), and have a firefox browser window in which you open the page at the same time. Then you can edit the page, save it, and refresh your browser window to see the results, though you'll have to do some fiddling with the windows to get them next to each other.

I'm not sure there's an all-in-one program for HTML editing. Linux as an OS tends to favor lots of little tools over few big ones.
