### Post by dcstar on 2006-03-25
[QUOTE=Compact]I have heard if you install ubuntu 32 bits in place of 64 bits, you will have more software to choose. Can 32 bits software run on 64 bits system?
If I've got a 64 bits processor, is better to install 32 bits version or 64 bits version?[/QUOTE]
All 32 bit software can run on a 64 bit processor - but not necessarily on a 64 bit O/S.

Some 32 bit packages need to be recompiled to run on a 64 bit O/S, that is the current issue (not all have been recompiled).

If you have a 64 bit AMD processor and want to be able to use all currently available software, just install a 32 version of Ubuntu and then upgrade the Kernel to the K7 version.
