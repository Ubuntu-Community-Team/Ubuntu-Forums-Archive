### Post by skippy81 on 2006-05-30
I would get the cheapest.  Both 754 and 939 are going to be phased out, so unless you know you will be getting a processor upgrade at some point then I wouldnt care.  Dual channel makes little difference as far as I know.  

The main difference between 754 and 939 is that while 754 boards almost exclusively use agp, 939 is a pci-express platform.  Therefore if you have an existing AGP graphics card you should go for 754, while if you will be buying a card in the future go for 939 (pci-express cards are actually cheaper than AGP).

The other main difference is that 754 boards work with the sempron budget 64 bit processors, which means that you can get a comparable system $50-60 cheaper than on 939.
