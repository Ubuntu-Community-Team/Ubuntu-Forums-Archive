### Post by tyggna1 on 2007-08-29
my card has issues with Beryl--but you might want to just stick with Metacity for a few boot ups before attempting to reinstall the packages.  I kept Network manager off my computer for a week or so, and used Wicd, and when I installed it again a week later:  BAM! network manager would suddenly see my wireless card.  

It's worth a shot.  Also, you might want to post additional information about your graphics card here, so some of the more experienced Beryl users could let you know what tweaks are helpful.

if you type in lspci in the command line, then look for something that says VGA compatible controller, post the name and we'll go from there.
