---

### Post by jswan on 2006-05-08
you go into a terminal.   ALT-CTRL F1
then you type
```
sudo /etc/init.d/gdm stop
```
or
```
sudo /etc/init.d/gdm start
```
or
```
sudo /etc/init.d/gdm restart
```

what ever you want to do
