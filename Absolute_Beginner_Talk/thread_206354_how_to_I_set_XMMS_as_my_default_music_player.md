### Post by Kilz on 2006-06-29
[QUOTE=babyboss]How do I set XMMS as my default music player under ubuntu. Thanks[/QUOTE]
Im not sure if you can set it for the default for all music files in one place. But if you right click on a music file, then Properties, select the Opens With tab, and click the circle next to xmms. Xmms now should be default for that type of file untill you change it. If Xmms isnt listed in the Opens With tap, click add, select the application in the box that pops up and click add.
