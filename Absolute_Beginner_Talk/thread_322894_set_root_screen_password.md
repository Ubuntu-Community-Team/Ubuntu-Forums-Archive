### Post by aysiu on 2006-12-21
The screen is already locked for the root user. Ubuntu defaults to not allowing you to log in graphically as root.

If you want to lock it for the current user, you can go to System > Preferences > Keyboard Shortcuts and then define a keyboard shortcut for **Lock Screen**. I use Control-Alt-L, as you can see.
