### Post by asmiller-ke6seh on 2007-12-14
I think that you, as a new user, will be happier using the 32-bit verion of Ubuntu 7.10 (Gutsy Gibbon). There is not that much speed advantage in the 64-bit version, and there are some potential issues that you are not likely to want to deal with.

Yes, you will be better off doing a fresh install of Gutsy instead of updating twice after an install of 6.10 -again, there are fewer issues.

Also, you might want to consult the New User's guide (use the link below, in my SIG) for some good hints, tips, and resources.
