### Post by foxholeunder on 2007-08-09
After you install Xfce and have it set as your prefered desktop environment you can uninstall the Gnome GUI by:

```

sudo apt-get remove ubuntu-desktop

```

Although if you remove the Gnome desktop many installed applications will no longer function as they may require dependencies that are included with a Gnome configuration.

Gnome will not run while using Xfce and the amount of hard disc space it will keep hold of is pretty minimal so you may not want to uninstall it entirely.
