### Post by Shatrat on 2007-01-13
There are two different installers, a live CD installer and an Alternate CD installer that isnt so graphical and works better on older stuff.
The DVD has both of these and also has more packages and stuff.

I wouldnt bother with the dvd unless you are gonna download and burn on a computer with broadband and then install on one that only had dial up or something.
It's easy enough for apt to download packages as you need them, and you dont really need 2 different installers as long as the first one you try works.
