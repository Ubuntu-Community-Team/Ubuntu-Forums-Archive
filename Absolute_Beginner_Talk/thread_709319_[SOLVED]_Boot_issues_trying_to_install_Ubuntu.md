### Post by taurus on 2008-02-27
If you have access to another machine, see if you can boot your DVD with that second machine.  If it boots, then there is something wrong with your DVD drive.  But if it doesn't boot, then there is something wrong with the DVD.  

Meanwhile, put the DVD into the drive while you are in Vista or a windows machine and see what's on the DVD.  Do you see one large file or do you see a bunch of files and directories?
