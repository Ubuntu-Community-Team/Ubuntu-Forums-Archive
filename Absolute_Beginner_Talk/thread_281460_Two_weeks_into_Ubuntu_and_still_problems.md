### Post by dbd on 2006-10-21
If you select your media player (by choosing "change how I view or hear this") to be Windows Media Player then it will work fine (you may need some [codecs ]("https://help.ubuntu.com/community/RestrictedFormats") installed, but you probably have them already).

A terrifying solution I know, but I'm not sure if real player can work from within your browser.
