### Post by meng on 2007-01-10
You can't have Windows and Linux (Ubuntu) on the same partition, they must be on separate partitions. Basically you will have to have a partition formatted to a LInux-friendly format (choose ext3 unless you have a specific preference). You will need to burn the file you downloaded to CD.
[www.psychocats.net/ubuntu/iso](www.psychocats.net/ubuntu/iso)
and then boot from the CD.
