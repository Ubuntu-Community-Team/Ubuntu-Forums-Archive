### Post by h0ax on 2008-01-30
You can burn the distro on a CD-RW; but when you load into Ubuntu, you are running a combination of the CD, and memory (RAM).
In order to save your firefox bookmarks, the easiest way would be to put in a flash drive(USB thumb drive) and Go to Bookmarks -> Organize Bookmarks; then File->Export
Save this file to your jump drive, or to another computer if you have networking and have mapped a shared drive.
Then, on the computer you want to retrieve these bookmarks from, Go to Bookmarks -> Organize Bookmarks; then File->Import and locate the file.

### Post by ugm6hr on 2008-01-30
Honestly - Ubuntu is probably not the best option for LiveCD use.  Unless you have a top-end computer with plenty of RAM, it will be horribly slow.

The LiveCD is really just to get a brief feel of Ubuntu, prior to installing it.  If you actually want to *do* anything, I would suggest you install it to HD.

As you say - there are some distros designed for LiveCD use (DSL / PuppyLinux are examples), which are much "lighter" than Ubuntu, and can run entirely in RAM.  For those distros, it can be useful to have the CD-RW functionality.  The creators of Ubuntu obviously fell that is not the case here.  Also - the Ubuntu LiveCD almost fills a CD - there is no room for saving settings anyway!
