### Post by mcduck on 2007-05-04
No, you don't need to uninstall Gnome, or download any disk image. Just ru 'sudo apt-get install kubuntu-desktop', or install the kubuntu-desktop package using Synaptic Package Manager.

If you wish to create customized install disk, you may want to try Reconstructor, alhough I think it can only do CD images currently: [http://reconstructor.aperantis.com/index.php?option=com_frontpage&Itemid=1]("http://reconstructor.aperantis.com/index.php?option=com_frontpage&Itemid=1")
