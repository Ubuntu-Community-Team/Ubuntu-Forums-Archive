### Post by shearn89 on 2007-08-06
Welcome! As to your questions:

1. You should be able to easily repartition your drive using a livecd if anything went wrong: boot using the livecd, open partition manager, wipe ubuntu, reassign to windows. Although ubuntu should install fairly easily.

2. I don't think you can install anything during a live session, so unless your card is natively supported by ubuntu, then you won't be able to install it without installing ubuntu properly.

Good luck!
