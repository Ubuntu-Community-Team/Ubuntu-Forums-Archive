### Post by hobbit1983 on 2006-02-06
Hi,

I can't help you too much with the running of photoshop under WINE, but the answer to your question re: burning CD's is one I can help with.

Nautilus (file manager application) allows you to burn ISO images to CD.  Insert a blank CD into your CD drive.  When the pop up turns up asking what kind of CD you would like to burn hit cancel

Go to the folder that contains the ISO that you would like to burn to CD Right Click on the ISO and there should be an option to "write to disc" - select this option.

However I would like to also agree with s6dalane gnome baker is a great application

Hope your WINE question gets answered

### Post by facefur on 2006-02-06
I have WINE installed, but I don't use it a lot, because I found the number of applications that run smoothly is still a little limited.  There is a compelte website for WINE, which includes access to a listing of applications and how well they run under WINE at [http://www.winehq.org](http://www.winehq.org).

I just looked over the database, and there are mixed reports on the functionality of Photoshop in WINE.  Earlier WINE releases and ealier version of Photoshop seem to work OK< but the latest Photoshop (7.0, 8.0) and newer WINE releases are apparently not compatible.  

I'd recommend you read the reports on the web site.

If "Office" = MS Office, you should have no troubles at all.  Personally, I would just use OpenOffice and save in a Windows format if you need it.

Good luck.
