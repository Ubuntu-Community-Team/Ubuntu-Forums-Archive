### Post by jken146 on 2007-12-03
Hi there.

I strongly suggest that you give Ubuntu a go.  You can dual boot with Windows (or any other OS) if you want, some people find this helpful.  Your system sounds ample.

As for those programs:
-Azureus - in the Ubuntu repositories
-Firefox - installed by default
-Winamp - no linux version bet there are loads of good music players in the repos.  Everyone has their favourite, so the best advice is to try some out for yourself.
Spybot, antivirus - not necessary
Codecs - there are loads of codecs available, but not installed bu default as many are non-free.  The package ubuntu-restricted-extras will enable many things like that.
VLC - this is probably the best thing for playing lots of different formats
Photoshop - I believe some older versions work in wine, but not the newest one.  The GIMP may do as a replacement, but that depends on your needs.  Photoshop is a bit of a sticking point for many people.  You can always keep Windows though just for that!
ISO - there is a CD/DVD burning app already installed, but if that isn't good enough the best one is probably k3b.
WinRAR - Fileroller, installed already, will archive anything.

As for differences with Windows, there are many.  The first thing you should learn is that Linux is not Windows, and nor should it be.  Things work differently, and that is often a good thing IMO.  If you need help in your transition there are plenty of good guides out there (try aysiu's [www.psychocats.net](www.psychocats.net)) and there are always these forums!

Welcome to Ubuntu!
