### Post by RobMunda on 2007-07-30
Sure thing. I'm doing this from memory, so if I miss a step, forgive me. I'm assuming you're using Firefox, but it should be a similar process if you're using one of the other browsers.

-Click on the Gallery.
-Click on a wallpaper you like.
-Click on the medium-sized version that appears.
-Click on the larger version that appears to enlarge it fully. (This step might be unnecessary, but it's always good to be thorough.)
-Right click on the now very large image.
-Choose "Save Image As..." (The wording might be different, again this is from memory.)
-If you don't have a place to store wallpaper already, I'd suggest making a "Wallpaper" folder somewhere in your /home directory. Save the image/wallpaper in that folder.

It sounds like you already know how to set images as wallpaper, so I won't go into that unless you need assistance. You can also do it from Firefox by right clicking the image and choosing "Set as desktop background".

Hope this helps,
-Rob.

### Post by RobMunda on 2007-07-30
You can save them on any computer, it won't hurt anything. The nice thing about image files is that they will display on any computer. You can save them on one and then put them on a CD, DVD, flash drive (one of those nice handy USB keyrings), or anything that will allow you to store and move files from one computer to another.

A suggestion for you, especially if you have a broadband Internet connection. If you don't have one already, get yourself a router. They usually have them on sale on the weekends at computer stores (Staples, Fry's, CompUSA, etc.). A router will allow you to connect all the computers in your house to the Internet. Ubuntu (and Linux in general) flies on the Internet. My connection using Ubuntu is 10 times faster than what it was when I was using the same computer on the same connection in WinXP.
