### Post by taurus on 2008-02-25
First, get into the BIOS and make sure you have CDROM as the first bootable device.  Then, boot the CD and at the first screen, pick the Scan CD for Defects option to make sure the CD is good.  

Now, boot into LiveCD session and click the install icon on the desktop to install it.
