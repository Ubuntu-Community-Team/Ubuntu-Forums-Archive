### Post by Bachstelze on 2007-02-15
The only difference between them is the desktop environment they install by default. Ubuntu installs Gnome, Kubuntu installs KDE and Xubuntu installs XFCE. The choice is not absolute though, you can very well install Ubuntu and then add KDE to it afterwards.
