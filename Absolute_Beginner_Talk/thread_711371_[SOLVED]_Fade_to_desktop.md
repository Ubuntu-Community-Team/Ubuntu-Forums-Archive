### Post by AmericanYellow on 2008-02-29
The plugins that cause this behavior are"Opacify" or 'Trailfocus" or both. Disable those two and you shouldn't have the problem.

Personally, I love those plugins. When I first started using Compiz, they were very annoying and now I can't be as productive with them.(less mouse clicking using them). Whenever I'm using Windows now, I get frustrated. :lolflag:
