### Post by thelatinist on 2008-01-14
You should be able to follow the instructions at [http://kubuntu.org/announcements/kde-4.0.php](http://kubuntu.org/announcements/kde-4.0.php).

 I'm not quite sure what you mean by "the more desktop managers the less comapatbility issues and such," though.   For me the more desktop environments, the more cluttered the menus and the more likely something will conflict (despite everyone's best efforts).  If an application needs a component of KDE 4.0, it will install it and you needn't install the whole desktop.

Of course, if you just want to try KDE 4.0 out, by all means.  I suggest using a test partition for it, though.  KDE 4.0 is new, and not everything is perfected.  There is lots of missing functionality in in this first release, too...
