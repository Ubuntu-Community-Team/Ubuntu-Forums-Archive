### Post by arochester on 2007-11-19
1) You can dual boot both Windows and Linux
2) You can run Linux as a Windows app through Wubi
3) You can keep Windows a and run Linux as VMWare
4) You can install Linux and (more complicated) run Windows through VMWare
5) You can try Wine (free as in beer)
6) You can try Cedega (not free)
7) (I think) you can try Win4Lin (not free)
