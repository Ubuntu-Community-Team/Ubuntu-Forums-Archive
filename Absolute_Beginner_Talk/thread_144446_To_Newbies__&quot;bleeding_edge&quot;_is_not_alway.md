### Post by Brunellus on 2006-03-14
I've been noticing that a LOT of new users--many with no previous Linux experience--have been installing Dapper.

This is emphatically NOT recommended for newbies.  The "newest" version of Ubuntu at any given moment is the development/testing version (as of this writing, Dapper).  It is therefore guaranteed to have bugs, and some of them will be quite unpleasant.

As a rule, when installing open-source software, remember that NEWER is not necessarily BETTER.  In most cases, NEWER also means "bug-testing."  If you are at all in doubt about which version you should install, go for the latest "Stable" version (as of this writing, Breezy Badger, 5.10)
