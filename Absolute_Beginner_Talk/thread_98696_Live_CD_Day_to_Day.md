### Post by aysiu on 2005-12-03
It's not dangerous--either with a live CD or a dual boot.

The only exceptions I can think of is mounting an NTFS partition with Ubuntu (live or dual booted partition) and modifying files. Otherwise, it's safe. A dual boot runs Ubuntu on its own separate partition. A live CD runs in RAM.

### Post by aysiu on 2005-12-03
[QUOTE=bvsciguy]But I can save pictures and documents with a live CD, and that wouldn't cause conflicts with Windows and its files, Right?[/QUOTE] I guess it really depends where you're saving them. 

If you're using the live CD to save files on to an NTFS partition, then, yes, it will damage the NTFS partition. 

If you save it to a FAT32 partition, there will not be damage in all likelihood.

If you save it to RAM, there will be no damage, but I don't see the point of saving it to RAM.

### Post by aysiu on 2005-12-04
Yes, a USB drive would be an excellent (and safe) way to share files between the OSes.

By the way, I like how cautious you're being. It's really for the best. A lot of new users dive right into a dual boot (or even a clean erase of Windows and install of Ubuntu) and end up regretting it either because Ubuntu ends up being not what they expected or because they didn't really know what they were doing and screwed up the installation. There's no need to rush, and I'm glad you're taking it one step at a time.
