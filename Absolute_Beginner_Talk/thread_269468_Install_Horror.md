### Post by StanE on 2006-10-01
Here's what happened when I tried to install Ubuntu 6.06

1.  Downloaded OK, burned the LiveCD disk, checksums matched.
2.  Put into computer, it started to load, and when I got to partitioning, I tried automatic.  Seven hours later, it was still percolating, so I killed it and tried again.
3.  This time, it wouldn't get past the language selection page.  The first time, hitting ENTER when the language was selected worked.  This time, it just sat there.

So, I tried it on a second computer.
2.  Put into computer, it started to load, but when it got to the line mentioned GNOME, everything went black.  It sat that way for many minutes.  Tried again, same result.  

Minor problems:  My ATI card on the first computer locked up on a very small resolution, chopping off some of the rolling list of tasks.  There's a published fix for this, but of course it doesn't work on the LiveCD.  
There's a difference between the two computers on the rolling taks list. The second computer lost the pretty display and switched to a DOS-type display halfway through the list of the things it got through.

In short, lots of wasted time, zero progress, very disappointing.

I tried to install Linux six or seven years ago, same thing: horror install - never worked.  Actually, I got further then than now.  

I'll try again next decade, to see if anything works.
