### Post by taurus on 2007-10-06
Edit /etc/fstab

```
gksudo gedit /etc/fstab
```
and replace these two lines 

```

**UUID=22ACE255ACE222D1** /media/sda1 ntfs-3g defaults,locale=sv_SE.UTF-8 0 1
**UUID=28709BB0709B836C** /media/sda2 ntfs-3g defaults,locale=sv_SE.UTF-8 0 1

```
with these two new lines.

```

**/dev/sda1**  /media/sda1  ntfs-3g  defaults,locale=sv_SE.UTF-8  0  1 
**/dev/sda2**  /media/sda2  ntfs-3g  defaults,locale=sv_SE.UTF-8  0  1

```
Save it and mount them with

```
sudo mount -a 
df -h
```
