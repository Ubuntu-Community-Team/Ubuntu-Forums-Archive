### Post by freebird54 on 2007-05-13
OK - here is the section that needs to be changed:

```
Section "Monitor"
        Identifier      "Generic Monitor"
        Option          "DPMS"
        HorizSync       28-64
        VertRefresh     43-60gksudo gedit /etc/X11/xorg.conf
EndSection
```

should instead read:

```
Section "Monitor"
        Identifier      "Generic Monitor"
        Option          "DPMS"
[B]        HorizSync       30-68
        VertRefresh     50-120[/B]
EndSection
```

This should enable a setting of 1280x1024 @ 70Hz or so.  Perhaps this will remove the artifacts from your viewing!

To enable the editing, if you haven't done this yet, first create a backup file
```

cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup_20070513
```
gksudo gedit /etc/X11/xorg.conf
then

```
gksudo gedit /etc/X11/xorg.conf
```

make the changes, save, exit - then hit <ctrl><alt><backspace> to restart the X Window system (will stop all running graphical programs alive at the time).  When you login again, hopefully the problem is gone...

EDIT: if it isn't - the maybe we need to see if it's the correct drive for your video - but try this first.
