### Post by bored2k on 2006-01-26
Linux has a media player for everyone:
- For trend-setter winamp fans, we have the now ugly xmms and the sleek but not as powerful beep-media-player.
- For library-based media player fans (hey, that's me!), we have a lot of promising applications. Rhythmbox, amaroK, QuodLibet/Ex-Falso, Muine, Banshee, it's all there.
- For those that prefer to have music daemons running, we have MPD and its gazillio frontends.

And heck, we can even run an emulated foobar2000,  for those who are into that kinky wine stuff ;).
