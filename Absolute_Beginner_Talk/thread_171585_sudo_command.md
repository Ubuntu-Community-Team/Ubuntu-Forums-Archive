### Post by ncappel1 on 2006-05-07
the wiki page will explain very well what sudo is and does, but in a nutshell here it is:

to ensure that you or anyone else doesn't inadvertantly mess up your computer, administrative tasks must be run as the administrative user. Originally, there was a "root" account, which could do whatever it wanted with out question. the sudo command lets you, a regular user without administrator priviliges, make administrative commands.
