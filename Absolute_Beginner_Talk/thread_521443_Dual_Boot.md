### Post by splintercellguy on 2007-08-09
Dual-booting means you can pick what OS you want to start when you boot up. If you install Ubuntu after XP, then you will basically be able to pick between the two out of the box. As for how, partition the XP so you get some unallocated then you can use the installer to partition the space for Ubuntu how you would like. Some people have mentioned giving /home its own partition so it's easier to reinstall Ubuntu.
