### Post by viciouslime on 2007-04-22
Any type of router will work with ubuntu. Make sure you get one with an adsl modem built in as some are designed for cable customers who have their own external modem with an ethernet port supplied as part of their cable package.

If you are getting an ADSL Modem/Router any router will be able to work with Ubuntu, but if you are going to be using wireless with Ubuntu then you should probably look into the compatibilty of the wireless adapter (probably USB). If you are planning to connect your router to your PC by a network cable then it should just work straight off.
