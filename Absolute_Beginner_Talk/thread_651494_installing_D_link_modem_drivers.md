### Post by Ertai87 on 2007-12-27
Welcome to the world of Linux.  Linux does not have a such thing as "autorun install".  Most drivers you use on Windows are not compatible with Linux anyway.  As such, you have to look for Linux equivalents to your Windows drivers online.  Ubuntu is good in that most of these drivers are ready-packaged, so you shouldn't need the driver in the first place most of the time.  Try using the router without the driver, and if it doesn't work, ask about what the driver is.  Be sure to include the router model number next time.  Good luck!
