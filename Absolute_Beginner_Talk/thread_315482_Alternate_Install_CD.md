### Post by cookfromfrozen on 2006-12-09
"Install in text mode" is the one you want.  This installs the same thing as the live CD.

I'm not sure what the OEM option does, but there is also an option called "Install a base system" or "Install a server" depending on which version you have.  This just installs enough for a console system without a GUI (i.e. no GNOME, KDE, Xfce etc...) so you can mix and match and generally configure the new install more precisely if you know what you're doing.

I personally prefer the alternate CDs; they're much faster.

### Post by matt_risi on 2006-12-09
I actually find it easier, and moreover more reliable. Much less a chance of the installer crashing, because it doesn't have a GUI to deal with. Also, it doesn't ask you any confusing questions, if you know what your keyboard layout is and what timezone you're in, that's about it.

The server/text method has become my favourite way of installing.
