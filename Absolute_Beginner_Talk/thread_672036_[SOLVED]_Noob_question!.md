### Post by blueridgedog on 2008-01-19
Weil, if it didn't put an icon into your menu, then you cant simply try and run it from a terminal:

```
superkaramba
```

If that fails you cant try and find it:

```
locate superkaramba
```

Then use the path/location that is produced if it finds it.
