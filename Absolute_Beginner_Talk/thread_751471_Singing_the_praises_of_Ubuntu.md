### Post by gear_head on 2008-04-10
OK, so I am an Ubuntu Noob. I just wanted to state that right off the bat. But I have to say, though there is a sizeable learning curve (as things are waaaay different than windows, and even some slight diffrences from fedora) Ubuntu ROCKS!!!!!

I've been running the live cd for a few weeks and finally bought another drive to install it on and I finally got it up and running the way I like it. One of the main features to me was to be able to get some kind of MP3 playback going on so I can have my music going on while I am surfing the net. It took some doing but thanks to the wonderful online documentation, as well as some of the docs that were installed with the OS, i was able to get it set up with little effort.

I also like synaptic. It's a totally different way to find and install software. I think that's awesome. Much better than fiddling around with .rpm's and debian packages that I'd have to spend more time in a console trying to install and configure. 

I look forward to using open office more for my productivity needs. I may still have to use wine to run MS Office as the OO Word processor just doesn't have certain features that I am used to having (like draw). :(  Oh well, I'm definitely in a transitional phase though, and I look forward to moving away from microsoft and more into the linux realm.

Great work guys!!
