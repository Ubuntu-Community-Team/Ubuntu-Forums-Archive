### Post by grte on 2006-01-02
You can install it on the second hard disk.  In fact, if you've got two, it's a better idea to install it on the second hard disk then it is to mess with the partitions on your first.

---

### Post by ardchoille on 2006-01-02
For a dual boot system, you can install Windows on one drive and Ubuntu on the other. However, it is much easier in the long run to install Windows first, then Ubuntu.. otherwise you will be left fixing the MBR.

---

### Post by Sef on 2006-01-02

### Post by jimrz on 2006-01-02
You certainly can put each on separate drives. In fact, I recommend doing so as I had some real issues trying to install both to the master drive and no problem at all putting each on separate drives.
