---

### Post by rok3 on 2007-03-18
Running Gnome applications under KDE and vice versa is done all the time.   Aptitude or apt-get should automatically install the libraries that are needed.  It does take a little longer for the applications to startup in the non-native enviroment but after that there is little to no speed difference.
