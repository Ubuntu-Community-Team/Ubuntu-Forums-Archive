Now I'm trying to get my Thunderbird mail off of the old drive.  I am able to mount it ok, but when I try to access my old thunderbird profile folder, I can't because I don't have permission.  Any help would be greatly appreciated.

### Post by ziggykg on 2007-03-28
try mounting the drive as root.  If it's already mounted as root, you can right click the mount point from explorer (Nautilus) and try and change permissions from there.  Hope this helps.

---

### Post by Compeau on 2007-03-29
Thanks for the help.  The owner of the folder was root, so I was able to get to the files by logging in as root.
