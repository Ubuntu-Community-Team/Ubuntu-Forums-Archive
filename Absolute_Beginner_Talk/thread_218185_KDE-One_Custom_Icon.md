### Post by T700 on 2006-07-18
It's a horrible kludge, but I think this will work:

1.  Take your text file (let's call it, "myfile.txt") and change the name to "myfile.ggr.  I chose the .ggr at random.

2.  Change the icon to whatever you like.

3.  Using the right click/properties, set .ggr files to always open with the text editor of your choice.
