### Post by Smandola on 2006-09-16
Welcome to Ubuntu.  If you want you can continue to use the Live CD to get the hang of what Ubuntu is like.  In order to use the Live CD, just put it in your CD, and boot your laptop.  Ubuntu should load with the Live CD.  This does not affect your win installation on your hard drive.

If you want to use Ubnutu and Win. Then yes, a dual boot is required.  I did this with a my pc and it works very well.  Although I find I never go back to win.  

I used this 
[http://www.crhc.uiuc.edu/~mjmille2/howtos/dual-boot-linux-and-windows/](http://www.crhc.uiuc.edu/~mjmille2/howtos/dual-boot-linux-and-windows/)

Although I'm sure there are several threads in this forum that explain Dual Boot.  If you choose this route, make sure that you do a back up of your system first, and if possible verify the back up.  

Here is another url for Dual boot
[http://users.bigpond.net.au/hermanzone/](http://users.bigpond.net.au/hermanzone/)

And for a more visual experience
[http://video.google.com/videoplay?docid=-6104490811311898236](http://video.google.com/videoplay?docid=-6104490811311898236)

Hope this help, post back to the forum if you have other question, or something specific.
