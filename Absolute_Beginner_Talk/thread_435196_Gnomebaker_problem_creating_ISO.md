### Post by cypherzero on 2007-05-06
Creating an ISO creates an image of the entire DVD - all 17Gb or however large your disk is* which understandably takes a long time.
Copying the files copies only as much information as the total size of the files, which usually don't take up the entire disk.

*though for a mostly empty disk with lots of empty bytes this will compress to a significantly smaller ISO.
