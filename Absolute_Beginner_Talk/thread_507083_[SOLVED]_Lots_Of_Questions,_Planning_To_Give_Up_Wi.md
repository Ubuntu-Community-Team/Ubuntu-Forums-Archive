### Post by jvc26 on 2007-07-22
Q1
> After being a windows xp user for years which version of Ubuntu should i go with, regular, xubuntu or kubuntu (please give pros and cons of each if you can)

I'd go with Ubuntu. Mainly because, I personally find gnome much easier to use than XFCE (Xubuntu) and to have more functions there by default - I dont want flaming for this, its a personal preference, I like Gnomes clean interface and it basically being exactly as I like it from the word go. Kubuntu I found (using KDE) to be too busy for my liking, I prefer a much more clean lined and simple desktop and general layout, which Gnome gives me.

Q2
Fraid I'm not sure about the printer.


Q3.
> I have two hard drives, an 80gig running and a 300gig storage. will my files on the 300gig still be accessible? i think i know the answer but just checking.

Yes, well this mainly depends on the format of the partition, if ntfs, then you will get read only access by default, but can get read-write by installing the ntfs-3g drivers for ubuntu. If fat32 then it will be read-write by default.

Q4
> I am going to list some of my favorite programs with windows (and programs i would like to have), could someone post linux equivalences?
CoreFTP= Not 100% sure about this app, but I've used gftp with no issues.
Bittorrent= I've used uTorrent under wine and Azureus. Also have used Deluge - again it tends to be a personal preference which to opt for.
A video editing program= Quite probably Kino, at least I've used it for some editing.
Winamp= My personal preference at the moment is Exaile, but there are so many players out there - XMMS, Listen, Banshee, Amarok which offer great functionality, but are very much down to personal preference - try a couple and see what you like best.

Hope that helps, welcome to Ubuntu.
