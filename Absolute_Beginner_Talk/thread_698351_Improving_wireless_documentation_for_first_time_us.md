### Post by JPWhite on 2008-02-16
As a first time user of Ubuntu I'd like to suggest an improvement to the official documentation regarding Network Manager and Wireless Networking.

As a first time user I kept going to System -> Administration -> Network and not seeing a list of wireless networks as the instructions indicated I would. What I should have done is simply click on the Network Manager Icon in the notification bar. The instructions for doing this come second in the instructions and there is NO GRAPHIC showing what it will look like.

I would have saved a lot of time if the instruction to click on the network manager icon came FIRST in the documentation and there was a graphic showing where to find it and what it looks like. People are visual and will see graphics before reading a bunch of prose. It is very easy to start clicking on links and performing complex procedures for troubleshooting when in fact nothing is wrong.

Initially I thought Ubuntu would be too difficult to show a friend how to use it until I realized how to use it. Without clearer documentation, many may leave Ubuntu behind thinking its too difficult when in fact its as easy as falling off a log.

JP
