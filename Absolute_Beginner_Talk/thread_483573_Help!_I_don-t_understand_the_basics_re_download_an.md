### Post by anon32 on 2007-06-24
You need help installing software?
Ubuntu and many other Linux distributions don't follow the Windows "google, download, install, run" style of application installation. Instead, packages are located in a central "repository". Ubuntu provides tools to access them, namely the "Synaptic Package Manager (System -> Administration)" and "Add/Remove (Applications -> Add/Remove)" on GNOME/Ubuntu, or "Adept" on KDE/Kubuntu.
In any of these package managers, you search the database of available applications and select the ones you want for installation.

Because of legal and support issues, you'll need to enable some sections in the repository before all applications are available to you (see [https://help.ubuntu.com/community/Repositories/Ubuntu](https://help.ubuntu.com/community/Repositories/Ubuntu)). For your multimedia problems, I suggest also reading the RestrictedFormats page linked to above (I recommend mplayer for a media player, VLC kind of sucks).
