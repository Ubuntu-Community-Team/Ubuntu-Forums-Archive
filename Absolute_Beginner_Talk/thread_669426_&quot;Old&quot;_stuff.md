### Post by Arthur Archnix on 2008-01-16
As I understand it, the older stuff is what Ubuntu has tested and decided was ready for inclusion at the time of release, which was back in October. After that, most programs only receive incremental updates. You may find that newer versions of these programs are available in the 'backports' repository. In addition, you can download and install any .deb file that has been made for Ubuntu.

For example, I want the latest version of Skype so that I have video chat. I downloaded the .deb file from their website for Feisty, and installed it. When I installed it said "there is an older version available in the channel, and it is recommended you install this version", because maybe one dependency is a newer version of some library, that another program on your computer isn't compatible with. This is the risk of installing newer versions. 

Generally speaking though, if you take a quick look through the forums and the homepage of the program you'll see if someone is commenting that the new version breaks something.
