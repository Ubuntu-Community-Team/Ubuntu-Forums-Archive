### Post by {BzF}~JOKesTER on 2008-03-04
If You And Me Are Thinking The Same Thing - It's Very Simple --------------------------------------------
Wine Does NOT Auto Register Itself As The Defualt To Open .exe Extensions!!!

Ok - So On A Test .exe Anyone Of Your Choice - Right Click It - Select Properties - Goto The "Open With" Tab - Remove All The Programs In That List By Selecting Each One End Clicking "Remove"

Now Click The +Add Button - In The Bottom Section Of The Dialogue You Will See "Use A Custom Command" - Select It And In The Box Type "wine" {Without The Quotes} Now Click "Add" Close The Properties Box And Double Click On The .exe That You Just Edited - It And All Other .exe Files Should Now Open Using Wine

Hope This Helped
