[COLOR="DarkRed"]hi. my guess is that you downloaded the server version of Linux, hence the lack of a graphical user interface, a.k.a. GUI or the boxes where you can click through your options. my suggestion is that you download Ubuntu Linux again, and choose the **desktop** version this time... but if you chose the **server** version deliberately, and you just want to learn the commands, you could google "linux commands". i think i did that one time and several sites that listed common linux commands came up. but the problem is, what is your version of Linux?
