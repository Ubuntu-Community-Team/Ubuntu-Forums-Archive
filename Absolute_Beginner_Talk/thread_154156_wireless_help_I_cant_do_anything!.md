### Post by briguy on 2006-04-02
I may be mistaken, but I don't think you can install the drivers when running from a live CD.  You might try and see if the package ndisgtk is installed, which is a graphical program to install windows drivers for wireless cards to get them running (you can search the forums for ndiswrapper if you want to learn more).  However, nothing of course will be saved when you shut the computer down!

Your best bet would most likely be to do a dual-boot.  It is pretty straight-forward, and there is lots of info on that on this forum and the wiki.
