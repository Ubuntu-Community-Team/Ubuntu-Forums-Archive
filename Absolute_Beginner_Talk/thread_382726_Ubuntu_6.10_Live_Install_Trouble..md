### Post by taurus on 2007-03-12
192MB of RAM may not be enough to run the LiveCD.  If you want to install Ubuntu on that machine, try using the alternate CD but better yet, try Xubuntu (with XFce4 window manager) since it would run a little faster on that machine then Ubuntu (with Gnome window manager).

[http://www.xubuntu.org](http://www.xubuntu.org)

### Post by benfindlay on 2007-03-12
First things first, if possible, only 192 meg of ram is gonna struggle with the lvve cd. Its probably running out of memory and just packing in.

I would also recommend installing xubuntu over ubuntu or kubuntu, as it uses the xfce desktop environment rather than kde or gnome. It is much more strealined and lightweight, but still looks as good! Some would say your computer looks more like a Mac with xfce and I would be inclined to agree with them!
