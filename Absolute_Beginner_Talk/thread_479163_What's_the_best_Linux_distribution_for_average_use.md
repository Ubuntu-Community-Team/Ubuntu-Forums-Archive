You're very correct about one aspect of the terminal, its ability to download *and install* packages (packages = roughly translated as "programs" in your Windows experience).

The terminal can also let you control everything happening in your Linux system through its command interface.   Poke around with the terminal for even just a bit, and you'll start to understand that you are now in control of a computer as opposed to merely using the computer.  It's great learning and so powerful.

You should challenge yourself to do something specific - for instance, you want to (in Linux) edit some home video, or simply copy a DVD onto a blank DVD -- and go through the process of installing the correct packages for doing that.  In the process, you'll learn about Linux and open source software and you'll be much more comfortable with the terminal afterwards.
