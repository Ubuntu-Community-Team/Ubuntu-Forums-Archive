### Post by kavon89 on 2007-01-06
We need more information about your computer, do this:

In Windows XP,

Click Start, then click Run

In the Run window, type ' dxdiag ' (no quotes, no spaces, just the letters between the quotes) and press Enter

If a window pops up asking a Yes or No question, click Yes

A large window should open named DirectX Diagnostic Tool and many things should load. In the middle of the window there is basic computer hardware information, please tell us the exact Processor name written there. Then how much Memory is listed on that same window.

Finally, at the top of that same DirectX Diagnostic Tool window, click on the Display tab (3rd from the left) and find in that window the Name of the device listed. It is located in the upper left side of the window, first line it should say Name: then the name we are looking for. Tell us the exact name of that also.
