### Post by dgbauer on 2006-01-02
I have successfully had Counter-Strike source run under Ubuntu, as well as Half-Life 2.  For both of those, I am using a program called Cedega that is available from [http://www.transgaming.com.]("http://www.transgaming.com")  (Not free)  Without having to pay for Cedega, you can use Wine which is freely available and you can use the command "sudo apt-get install wine" to install it on Ubuntu.  There are tons of howto's and such available in the Ubuntu forums as well as at [http://www.winehq.com]("http://www.winehq.com")

Serious Sam 2...  Well, I haven't tried that one, but I do have it and will try for you.  

Doom 3 works natively under linux, but you do need to get the Doom 3 linux installer from [ftp://ftp.idsoftware.com/idstuff/doom3/linux/doom3-linux-1.3.1302.x86.run]("ftp://ftp.idsoftware.com/idstuff/doom3/linux/doom3-linux-1.3.1302.x86.run")

Good luck!  And I'll let you know how that Serious Sam 2 thing works out.
