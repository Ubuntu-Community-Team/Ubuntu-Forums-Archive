Play that movie in (g)mplayer. If a/v is out of sync too then it's a problem of mplayer. In that case try to update it.

### Post by rvm4000 on 2008-04-12
mplayer or gmplayer.

Open a console and type:

```

mplayer file_to_play.avi

```

or

```

gmplayer file_to_play.avi

```

where *file_to_play.avi* is the file you want to play, including the full path if necessary.
