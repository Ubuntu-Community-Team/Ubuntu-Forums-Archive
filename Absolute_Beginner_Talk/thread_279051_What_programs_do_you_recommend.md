### Post by n0dl on 2006-10-17
xmms: lightweight, dependency light, music player
openbox: use it stand alone like I, or use it with gnome and replace it with metacity. Lightweight window manager
fluxbox: same as above, but with more eyecandy.
nethack: Awesome game
naim: If your into instant messaging naim is lightweight, and if your X breaks you can still sign onto aim in a tty console
irssi: Great IRC client... Personally I think its better than Xchat

links/links2/elinks: any of em... There all text based browsers
build essentials: gotta compile things from source
gcc & g++: same as above
conky: awesome sys monitor.
k3b: Although I dont necessarily like KDE stuff, k3b has gotta be one of the best cdwriting software... Of course you could learn to use cdrecord (but thats a bit of a hassle if your not comfortable operating bash)

### Post by Garyu on 2006-10-17
Many people here are listing Nvu for webpage editing. There is a new project called Kompozer which is Nvu with a few bugfixes. [http://kompozer.net/](http://kompozer.net/)

Inkscape is great to play with and do some vector graphics.
Scribus is excellent for desktop publishing.
The gimp of course for those bitmapping things.
gThumb - for batch renaming of image files from creation date and so on (DC0001.jpg -> 2006-10-17001.jpg).
OpenOffice.org for those wordprocessing, calculating, presentation needs.
Gaim for MSN and ICQ.
aMSN for using webcam with MSN.
Evolution for mail and calendar and contacts.
Firefox for surfing the web.
GoogleEarth for fun.
Valknut for peer-to-peer / directconnect file servers.
Rhythmbox for playing music.
EasyTAG for managing MP3 tags.
MPlayer for watching movies.
GnomeBaker for burning CD's and DVD's.
QGIS and Grass for making maps (still learning, maybe not essential software yet).
Blogger panel applet which connects to my wordpress blog.


Games:
Americas Army ([https://help.ubuntu.com/community/AmericasArmy](https://help.ubuntu.com/community/AmericasArmy))
Frozen Bubble
Neverwinter Nights (commercial license)
SuperTux
Wormux


Also, on my server Ubuntu machine, still in evaluation though:
Postgres database
Apache2 webserver
SQL-Ledger (accounting)
PHP5
Joomla! web content manager
Wordpress blogg
SMF Forum

I think I got a pretty nice list there of the best of the best. I heard there is a new image editing software in Edgy (F-spot?) that might be better than gThumb, I haven't tried it yet. But the best thing about gThumb is the batch renaming of files, can be used on non-image files as well. :cool:
