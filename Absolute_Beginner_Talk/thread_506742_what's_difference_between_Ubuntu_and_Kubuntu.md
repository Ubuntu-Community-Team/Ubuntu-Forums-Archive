### Post by gn2 on 2007-07-22
The software list is also different.

KDE and Gnome can be added to Ubuntu or Kubuntu, so if you have one you can swap to the other very easily, or even try XFCE.

The relevant packages are:
ubuntu-desktop
kubuntu-desktop
xubuntu-desktop

Add those packages either through command line or package manager and select Gnome, KDE or XFCE in Sessions from the boot splash log-on screen.

### Post by TeaSwigger on 2007-07-22
Different programs and interfaces. In my limited tests I had these vague impressions:

ubunutu had the easiest user interface - more sort of "it just works" than the others.
kubuntu had the more "sophisticated" software (or appeared as such). More involved.
xubuntu was leaner, more sort of "no nonsense."

All three impressed me greatly. My PIII box was happiest with xubuntu speed wise. It was refreshing and clean. It sort of gets out of your way and I really like Thunar (the file manager). ubuntu was charming to me; gets a lot of heat for the color scheme (and the sighting of the occasional foot) but I found it a nice difference that has to be good for some folks. Probably best for just using in comfort. Nautilus (the file manager) was a bit sluggish on my box. kubuntu was in between speed-wise and "feel" wise. As I enjoy the features such as k3b (the excellent burning software) I figure I'll enjoy it most as I get more familiar with it.

Can't go wrong with either of the 3, IMH. The install was flawless on one PC - a lovely fawn - and is being, well, feisty let's say ;) on the other... such are these things. Luck to ya.
