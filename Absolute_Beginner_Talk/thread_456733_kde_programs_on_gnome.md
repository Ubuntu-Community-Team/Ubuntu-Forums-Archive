### Post by spin2cool on 2007-05-28
Yes.  When you install something that requires KDE libraries, whatever package manager you use should install the appropriate libraries.  This is only really a problem if you're very low on hard drive space.

### Post by Tux Aubrey on 2007-05-28
You will probably end up installing much of the desktop anyway because of dependencies (not that you have to use the actual KDE desktop).  In the past I have used KDE apps (Amarok and a few others) under Gnome but always ended up with crashes or slow performance.  I suspect the newer versions of the desktops and the newer apps work better because there is a real effort towards having consistent standards.  

There's no harm in trying!
