### Post by LaRoza on 2007-12-27
I don't know exactly what you are asking.

You can dual boot, easily. So when you start the computer you can chose Windows or Ubuntu.

You can virtualize an OS, and run Windows and Ubuntu in a Window (or the other way around, VirtualBox can work in Windows and Ubuntu)

If you are new at this, and want the easiest method, chose the option from the Ubuntu disk that will install Ubuntu with Automatic partitioning and use freed space. It will resize the Windows partition to what you specified and do everything automatically. Just move the slider to the new Windows partition size and hit ok, answer the questions and wait.

[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)

If you are talking about virtualizing [http://www.virtualbox.org/](http://www.virtualbox.org/) is very good. It has an installer for Windows and Ubuntu (among others)
