### Post by quanct on 2006-08-08
To install Limewire, I followed these README.txt instructions:

                        LIMEWIRE  README

              ====================================



To run LimeWire, you will first need to install the 

Java2TM Runtime Environment, v 1.4.2 or greater.  You can find these 

downloads at:



    [http://java.sun.com](http://java.sun.com)

    

LimeWire is tested against Sun's Java Runtime Environment's and may not

work with other JREs.  To determine what kind of JRE you are using, type



    java -version

    

You must place Sun's JRE in the path prior to launching LimeWire.



If you are using this from the LimeWireOther.zip file, you can launch

LimeWire by typing:



    sh ./runLime.sh

    

If you are a Windows user and using LimeWireOther.zip, you can run

LimeWire by double-clicking on LimeWire.exe.



We sincerely hope you enjoy using our product.



Best Wishes,

LimeWire Team
