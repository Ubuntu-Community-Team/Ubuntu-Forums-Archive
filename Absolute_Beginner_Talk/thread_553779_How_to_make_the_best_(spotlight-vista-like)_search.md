### Post by mocoloco on 2007-09-18
Deskbar plus an indexing seach is the closest you'll get.  I've been using Beagle but I know Ubuntu 7.10 (Gutsy) will use Tracker and have it installed by default.
I've read a bit about it, and the general consensus is Tracker is much more efficient on resources, but not quite as good as Beagle or Google Desktop for Linux just yet.  However development is fast.
There was a great article that compared the 3 but now I can't find it.  I think I'll uninstall Beagle and install Tracker though, because I've noticed beagle sometimes bogs down my system.

As far as mapping your WinKey (or as I like to call it, TuxKey) to open the deskbar, Linux sees that key as a meta-key, meaning it does nothing by itself.  I'm sure that could be changed, but not sure how easily.
