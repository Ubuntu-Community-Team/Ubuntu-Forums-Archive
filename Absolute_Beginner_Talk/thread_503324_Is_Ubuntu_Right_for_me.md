### Post by Rocket2DMn on 2007-07-17
I don't know anything about WarRock, but I know counter-strike can be run under wine (a sort of windows emulator - though it's not a true emulator), but there is some loss in quality, namely in your FPS rate.  Newer games tend to run slower since they have to be run in wine.
You can certainly IM in linux using programs like Pidgin which can connect to a whole host of different IM protocols.  Also, you can make youtube videos.
The command line interface is an integral part of linux, but don't let it scare you too much - it may seem cryptic but there is a method to the madness which you can learn.  You will need it often to set things up and do configurations.

### Post by raja on 2007-07-17
Depends on a few things, but it is always worth giving it a shot!

Why are you tired of windows?
I am not into games, but your gaming experience is definitely going to better in windows. You may check to see how your favourite games are doing on linux with wine.
Again making youtube videos means -- do you need to do a lot of video editing? There is some scarcity of easy-to-use tools in that regard in linux.
For IM, if it is text-based, you will be at home with pidgin or kopete. If you need sound or video, though, it is going to be difficult.
For most every-day things, you wont need to use the command line in Ubuntu. It is an extremely newbie-friendly form of linux.

You should plan on either dual boot or run virtualization (VirtualBox, VMWare, qemu) untill you learn the ropes.
