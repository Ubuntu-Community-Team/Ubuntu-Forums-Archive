### Post by il-luzhin on 2007-12-11
update:
if [[color=blue]devilspie[/color]](http://ubuntuforums.org/showthread.php?t=202249&highlight=devilspie/) is installed 

```

sudo apt-get install devilspie

```

you can make a text.ds file in the directory /home/YOUR USERNAME/.devilspie that says simply

```

(debug)

```

now when you type **devilspie** in the terminal you should get window geometry for all open windows.
