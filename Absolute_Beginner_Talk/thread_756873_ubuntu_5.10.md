### Post by ByteJuggler on 2008-04-16
Ubuntu 5.10 is comparatively old.  It really would be good if you could install a newer version.  I don't offhand know if you can ugrade from 5.10 to a newer release like you can with the newer releases, but it should be possible for you to upgrade, failing all else, from the command line.  (But, you'd have to upgrade to each intermediate version in turn - not a pleasant experience.)

However, it is also possible to install Ubuntu from USB key, there are instructions/howto's that explain how to put an Ubuntu ISO image onto a USB flash drive and boot from it.  Do you have a USB flashdrive that's large enough (1GB or bigger) and would this not allow you to directly install the latest version?  (You would need a PC with a BIOS that allows booting from USB devices for this to be viable.)

### Post by northern lights on 2008-04-16
From the "Alt+F2" dialog, run ```
gksu "update-manager -c "
```
and select the distribution upgrade.



On a side note: Ubuntu (and some other distros) can be installed without a CD drive - from either Windows or any other Linux OS via [unetbootin]("http://sourceforge.net/project/showfiles.php?group_id=198821&package_id=243411") (W/Lubi)
