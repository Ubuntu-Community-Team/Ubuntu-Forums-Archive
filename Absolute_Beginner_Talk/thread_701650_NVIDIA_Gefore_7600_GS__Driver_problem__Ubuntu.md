### Post by herbster on 2008-02-19
I had a 7600 GS before. The problem I'm sure is where it says "Driver "nv"", it should be "nvidia" instead of "nv" I do believe.

Run nvidia-settings and it will tell you if your nVidia driver is in use or not. If you get many options and can configure the screen, etc. you're using it. If not, it's not using it, so you most likely have to change "nv" to "nvidia".

### Post by akschnare on 2008-02-19
I found these instructions when I was having problems and it worked for me. I can't find the original link or who wrote them. Hope this helps.




"Ok, some people have had problems and others haven't. For those of you who can't get Nvidia working, here is how I did it.

Get the driver (I used the Beta version)
Download the driver from here: [http://www.nvidia.com/Download/index.aspx?lang=en-us](http://www.nvidia.com/Download/index.aspx?lang=en-us)

Move it into your home directory.

Preinstall
I tried a usual install and got a "no libc header files installed" error message, so go ahead and install both files:
Code:
sudo apt-get install libc-dev
You will get "libc6-dev"

Install(Command Line out of X environment)
Hit Ctrl+Alt+F1

You will now be at the terminal, but we still need to kill the xserver, in our case it's the GDM "Gnome Desktop Manager"
Code:
sudo killall gdm

To install the driver you have to be root, so you need to type su and enter your root password.

You are still in your Home directory, so type ls to list the files and you will see the Nvidia installer. (NVIDIA-Linux-x86-100.14.23-pkg1.run)

Next you need to type the sh command to run the installer:
Code:
sh NVIDIA-Linux-x86-100.14.23-pkg1.run

Use your arrow keys and the Enter button to answer yes or no to the Nvidia dialogue. Just say yes to everything, I am going to give you the benefit of the doubt here, and pray that screenshots really aren't necessary 

There won't be a precompiled kernel available for your driver (there never is) so just say yes to compiling one.

That should be it, the driver will install. 

Now type exit to get out of root and then you just type startx to fire up the xserver and hopefully into your aligned, pretty looking, Nvidia desktop.

I got a "Fast User Switcher Desktop Applet has Unexpectedly Quit" error message, so I tried reload, but it didn't want to start, then I hit cancel, and it was loaded in the top left of the panel anyway    Ubutnu...pffft!

Then I rebooted and got my nice shiny Nvidia logo pop-up. Happy days !"

### Post by roaldz on 2008-02-20
Please folks, first try to use the repositories for all these things. It will most of the time do the job.
Here´s what to do:

Let envy remove everything it installed. You should install nvidia-glx-new. I think the -new package goes for the 7300 too (if not, its just nvidia-glx). The same thing is done by restricted drivers manager, so why don´t you try that first?
At the end of the line:
You should have nvidia-glx(-new) installed, and the video card driver should be set to ¨nvidia¨ in /etc/X11/xorg.conf.
```
gksu gedit /etc/X11/xorg.conf
```
For me, it looks like this:
```
Section "Device"
	Identifier	"nVidia Corporation G70 [GeForce Go 7600]"
	Driver		"nvidia"
	Busid		"PCI:1:0:0"
	Option		"AddARGBVisuals"	"True"
	Option		"AddARGBGLXVisuals"	"True"
	Option		"NoLogo"	"True"
EndSection
```

Then log out, press CTRL+ALT+Backspace, this will restart the X server (graphical interface). Now it should load the good driver. If you have anymore problems, post them here, I´ll answer them

Roald

First of all, the apt-get remove --purge nvidia* is not needed, just removing nvidia-glx-new will remove the driver.
About the refresh rate thing - thats weird, and I have the same thing. You actually have 60Hz, but it reports 50Hz. Go to CCSM --> general options --> display settings and set the refresh rate to 60Hz. That should fix it. (It did for me).
Also, movies are 24/25 fps, so you wont notice the 50/60hz difference, in fact, with 50Hz, it should be better in sync. Some movies are 29/30 fps. They are viewed better with 60Hz, but the difference is small.

The restricted drivers manager just installs the nvidia-glx-new driver from synaptic, so you can choose, it won´t make any difference.
