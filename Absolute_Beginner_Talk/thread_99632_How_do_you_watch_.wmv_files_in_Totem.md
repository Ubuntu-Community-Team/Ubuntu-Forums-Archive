### Post by Nequeo on 2005-12-05
.wmv is a restricted Microsoft format... You can't play them by default, because in some countries and states it would be breaking the law!

It's a thorny legal problem that you probably don't care about. So for an easy way to just get on with you life, try running 'Easy Ubuntu'

[http://placelibre.ath.cx/keyes/index.php/2005/10/27/65-easy-ubuntu-24-beta](http://placelibre.ath.cx/keyes/index.php/2005/10/27/65-easy-ubuntu-24-beta)

Visit that website and follow the instructions there to run the program. Then just make sure 'media programs' is ticked, and hit 'apply', and away you go. It will take quite a while to run if you're on a slow internet connection, but just let it do its thing, and when its finished you should be able to play WMV files. If not in Totem, then with Mplayer at least.

Cheers,
