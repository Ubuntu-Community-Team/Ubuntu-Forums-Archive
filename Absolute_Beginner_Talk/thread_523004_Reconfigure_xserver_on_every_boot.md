### Post by eapache on 2007-08-11
So it isn't really a problem with x then. In grub, edit the normal ubuntu boot option and remove splash and quiet. To do this:

- press escape if ubuntu is the only os, otherwise it should appear automatically.
- select the main boot option and press e
- select the second option in the new list and press 'e' again
- remove splash and quiet
- enter, and then 'b' should do it.

This boots without the splash screen, so you should see what the error message is.
