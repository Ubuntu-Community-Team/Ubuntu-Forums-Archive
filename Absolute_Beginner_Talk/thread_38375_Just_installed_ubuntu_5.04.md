### Post by Leif on 2005-05-31
Unless it's giving you error messages, it sounds like you installed it in server mode and not normal. Not a problem tho, type "sudo apt-get install ubuntu-desktop" and enter your normal password when prompted for gnome. For KDE, type "sudo apt-get install kubuntu-desktop". Restart and things should be ok.
