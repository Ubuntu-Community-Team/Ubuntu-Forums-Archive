### Post by Marsolin on 2006-09-18
For anyone using [Konqueror]("http://linuxappfinder.com/package/konqueror"), [MountISO]("http://linuxappfinder.com/package/mount-iso") is a cool plug-in for create, mounting, and unmounting iso files. It shows up as a menu anytime you right-click an iso or a directory.
