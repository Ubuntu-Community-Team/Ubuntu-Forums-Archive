I too have twin view with dual monitors.  I run each of my monitors in 1680x1050 mode.  
You can right click on a blank area of the desktop and select "Change Desktop Background" -- then select Style to eiither Fill Screen, Scaled, etc.

You can fill both monitors with one image or stretch it to fill the screen but the image is distorted.

I tell you what I did.  I found a great image of a nighttime view of a bridge in Tokyo - it was 1600x1200.  I used Gimp to scale it to 1680x1050.  This was great for one monitor.  Then, using Gimp, I took the image and flipped it horizontally so that it as a mirror image.  Now these two images could be joined together, again using Gimp, as one big image - i.e. 3360x1050 which I used as my desktop wallpaper.

Carl
