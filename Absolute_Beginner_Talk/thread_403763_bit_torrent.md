### Post by Sammi on 2007-04-07
The best site there is on port forwarding: [http://www.portforward.com/](http://www.portforward.com/)

You can find instructions for almost any router in there.

You can then use this page to see if the forwarding is working: [http://www.canyouseeme.org/](http://www.canyouseeme.org/)

ATM the best torrent client for Linux is Ktorrent, which is made to work best with KDE. A Gnome equivalent is in the pipes, but won't be here for some time. I myself use uTorrent through Wine. It may be a Windows app, but I hear that it's programmed to run well in Wine.

Howto on Wine/uTorrent: [http://ubuntuforums.org/showthread.php?t=295134](http://ubuntuforums.org/showthread.php?t=295134)

---

### Post by Sammi on 2007-04-07
Oh and you should also make sure that the port is open on your computer. There is a very simple app called Firestarter that you can use to do this. It's sort of a front end to the firewall that's built in to the Linux kernel.
