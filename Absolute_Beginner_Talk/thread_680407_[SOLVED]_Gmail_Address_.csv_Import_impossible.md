Hi Moezy,

This is easy to do using the "Older Version" of Gmail.
If you are using the new version (as you must be) there is a link to
older version.

Then you simply paste the emails into the [add] box.

The format required is
name <email>, name <email> etc

You will have to enclose the name in quotes if it contains special
characters, eg.
"surname, firstname" <email>

To get the list to paste, I generally use Word.
A useful trick there is Replace ^p with ,^p (comma paragraph marker)
to add a comma at the end of each line.
