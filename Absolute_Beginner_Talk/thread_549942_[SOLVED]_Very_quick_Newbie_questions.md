### Post by s26c.sayan on 2007-09-13
1) You can run all the programs installed from the terminal/command line.
Try opening Accessories -> Terminal and enter there the first few letters in the app name you are trying to run, and press TAB to autocomplete.
You can use the alacarte Menu Editor to edit your menu to include all the apps you want!

2) Ubuntu Security : Read [this]("http://ubuntuforums.org/showthread.php?t=510812").
I recommend Firestarter, which is a GUI configuration tool for the firewall inbuilt into the linux kernel.
