### Post by jag164 on 2006-01-19
One way to do this is to add the "Character Palette" applet to your gnome panel.   Then select a pallette, select a character, and then middle click (or ctlr-v) where you want to insert your character.

### Post by Azriphale on 2006-01-20
I'm busy investigating this, but I'll tell you about what I have come up with so far (nothing useful for you right now, tho...)

On many European keyboards, there was an "AltGr" key in place of the right Alt key on the US kyboard. This supposedly stands for something like Alternate Group. This was used to access a third set of characters from certain keys on the keyboard. 
I read [here]("http://wiki.linuxquestions.org/wiki/Accented_Characters") that the AltGr key can be used to produce the accents.
i.e. "AltGr + ]" accents the following character with a ~. so pressing AltGr + ] followed by "n" would yield &#241;.

All that remains is to determine how exactly to map a key to it. Right now I am at work on a windows machine, so I don't have access to a Linux box to play with.
