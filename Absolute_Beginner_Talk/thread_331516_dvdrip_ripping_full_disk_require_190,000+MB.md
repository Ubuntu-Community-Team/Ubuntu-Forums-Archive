### Post by altonbr on 2007-01-05
I'm just starting to use this program myself, but I noticed that in the "Storage" tab it has two bullets: "Copy data from DVD to harddisk before encoding" and "Encode DVD on the fly". You might want to select "Encode DVD on the fly" so that the file is only 4.5GB or so. What is happening is that the program is copying the video completely uncompressed (so it's at it highest quality) and then compressing it later to fit back onto a DVD.
