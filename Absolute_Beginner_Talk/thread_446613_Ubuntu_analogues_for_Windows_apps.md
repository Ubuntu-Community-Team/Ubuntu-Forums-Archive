### Post by Terl on 2007-05-17
If you are going to be new to linux, the first thing I would suggest is setting up a dual boot arrangement so that you still have access to your windows apps.  This is especially a good idea as some of the available linux apps may not be to your liking.

Some of the programs on your list are available for linux as they are (opera is one) while others have similar counterparts and some may have none.  There exists also virtual machines and Wine to allow you to run windows apps in linux.

I would suggest a bit of googling to learn more.

[Some ideas here]("http://www.linux.ie/newusers/alternatives.php")

[More here....]("http://www.foogazi.com/2006/11/10/alternatives-to-windows-programs/")

[And from the Ubuntu guide...]("http://ubuntuguide.org/wiki/Alternatives")

### Post by knightnet on 2007-05-17
> **Kobalt said:**
> 
* Acrobat Pro (for pdf creation and editing)  [COLOR=Red]- OOo or KOffice[/COLOR]
    * Photoshop :** Gimp  [COLOR=Red]- Yep, GIMP if you really need the power though this still does not match the ease of PS[/COLOR]**
    * Excel : **OpenOffice   [COLOR=Red]- Or Koffice for simpler requirements. Or try Excel under Wine. Or use WMware, etc.[/COLOR]**
    * Wordperfect :** OpenOffice****  [COLOR=Red]- Or Koffice for simpler requirements[/COLOR]**
    * Opera : **linux version available**
    * The Bat   [COLOR=Red]- Depends on your needs. Personally I find Kmail (also part of Kontact) one of the very best mail clients I've ever used.[/COLOR]
    * Visio  [COLOR=Red]- Dia is the closest[/COLOR]
    * utorrent : **Transmission, Deluge, KTorrent,...**
    * Comodo firewall (if firewall is still needed) : **Ubuntu built-in firewall is far enough  [COLOR=Red]- Yep, it is IPtables and there are a number of GUIs available. Not usually needed though unless you are using your machine on an untrusted local network (unlike Windows).[/COLOR]**
    * NOD (if anti-virus is still needed) : **no need  [COLOR=Red]- YES THERE IS!!!! If you are working with other peoples documents, you should use some kind of AV. AVAST is great and works with Windows and Linux.[/COLOR]**
    * Google Earth : **linux version available [COLOR=Red]- Works fine on Linux[/COLOR]**
    * Faxamatic (for sending scanned multi-page faxes)  [COLOR=Red]- I think this is built in as one of the special printer types though you will need to configure an appropriate modem I guess.[/COLOR]
    * Womble MPEG Video Wizard (editing video)  [COLOR=Red]- Too much choice! I'm afraid you will have to try a few packages and see what fits your requirements.[/COLOR]
    * Personal Stock Monitor   [COLOR=Red]- Don't you do this online? I would download the appropriate data (or get it sent to you via email or RSS) and load it into a spreadsheet - indeed this is exactly what I used to do (under Windows).[/COLOR]
    * Skype : **linux version available**
    * Yahoo Messenger with conferencing and webcam reception
    * Cobian backup  [COLOR=Red]- Plenty of choice here. "SUDO aptitude search ~dbackup" from a command prompt will list a load of stuff. The rdiff stuff is good or just an rsync or two in a script attached to a cron schedule.[/COLOR]
    * Winzip : **File Roller  [COLOR=Red]- Select some files in a file manager and right-click, you might be surprised![/COLOR]**
    * and the real challenge: Alpha IV (DOS database) which is so full of scripts I could never re-write due to brain cell loss. Maybe Alpha IV would be WINEable. : [B]I'd say use Wine...
[/B]
