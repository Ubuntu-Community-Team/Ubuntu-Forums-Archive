### Post by julian67 on 2008-02-26
Not exactly. You'll need to learn some new tools. If you're happy using proprietary software check out [LightZone]("http://www.lightcrafts.com/products/index.html").  Windows and Mac users have to pay, but Linux users get it for free (but without support).

There are several photo management tools for Ubuntu, probably you're familar with Digikam, F-Spot and gThumb. For batch processing have a look at Phatch. There are others, like RawStudio, RawTherapee and GQview. The best thing is to try a few and see if you can make them work for you.
