1.  You don't need automatix to get rar support.
2.  Just install this package:
```

```

Automatix is not supported at Ubuntu Forums so if you decide to install it and have troubles, you'll need to go to their forums for support at [http://www.getautomatix.com](http://www.getautomatix.com)
