If you reboot into the Live CD you should be able to change it.

**[COLOR="DarkRed"]!!NOTE!![/COLOR]:**You may lose data while trying to resize the partitions.  The best solution is to start from scratch because data is written all over the drive.  However, if you haven't been using the newly installed operating systems for very long, then I don't think you will be at much risk.  If you have had the Windows installation or Ubuntu installation for awhile though (and have been using it regularly) you are at risk for losing some data.
