### Post by steveneddy on 2007-10-18
If you are Feisty or Gutsy, go to 

System --> Preferences --> Power Management

and select Suspend when laptop lid is closed.

When you have Gutsy installed, the login dialog on your screen allows users to leave you a message that will be in the bottom right corner for you to read when you get back.

:popcorn:

EDIT:

If it's a desktop, go to the start button and push it and select Lock Screen. Lock screen works on laptops also.

### Post by ayenack on 2007-10-18
You can also set the screen saver to lock the screen when the screen saver is active. If you do that then you can set the screen saver to a low value maybe 5mins and then when someone tries to deactivate the screen saver the comp will ask for athe password.

EDIT:

To do this got to.

**Systems>Preferences>Screensaver**

And make sure that activate screen saver when computer is idle and lock screen when screen saver is active are ticked.
