I have a DV9000 and have noticed the same thing in both Windows and Linux.

The fan is not controlled by the operating system.

It is possible that opening the program might cause the fan to start up because the processor's frequency has changed to perform better when using applications. You could check the CPU Frequency Scaling Monitor.

Is it especially bad? I mean, does it sound very loud? It could be a problem that has occurred for some other reason.
