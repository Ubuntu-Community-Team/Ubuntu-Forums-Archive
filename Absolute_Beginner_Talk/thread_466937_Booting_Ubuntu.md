### Post by taurus on 2007-06-07
Perhaps you need to tell winrar NOT to screw around with .iso when you download because you are not supposed to unpack it before you burn those files to a CD.  It will not boot that way.  You need to save it as one large file with .iso extension.  Then, use your burning software to burn it as an image file.

[https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28burn%29%7C%28iso%29](https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28burn%29%7C%28iso%29)
[http://www.psychocats.net/ubuntu/iso](http://www.psychocats.net/ubuntu/iso)
