### Post by Ub1476 on 2008-03-18
You can install themes from gnome-look.org.

They usually come in a tar.gz archive, which you can open and install through System>Preferences>Appearance.

Some GTK2 themes (those are colors and widget, scrollbars etc), requires a special engine. If you read the notes it shold say what engine is required, which can be downloaded from Synaptic.

If you want to use a new icon set, you should look at "latest" or "highest rated", cause those at "most downloaded" often are quite old and will not work with the latst version of Gnome.
