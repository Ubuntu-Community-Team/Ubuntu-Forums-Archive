### Post by Ek0nomik on 2007-08-10
Do you get any output whatsoever?  No error messages?  Just a blank screen?

If no errors were found on the CD, I'd suggest trying to install with the Alternate CD instead of the Desktop (Live) CD.

[http://www.ubuntu.com/getubuntu](http://www.ubuntu.com/getubuntu)

Below the green 'Start Download' button there is a checkbox to download and install the Alternate CD.  Burn it to a disc, and you can install via a text-based installer.

The issue is most likely related to your video card.
