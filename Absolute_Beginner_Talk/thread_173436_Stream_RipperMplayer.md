### Post by jazzmuzik on 2006-05-10
I've not used stream ripper, but but I did come across an example using mplayer and saved it to my cheat sheet:

```
mplayer -dumpstream -dumpfile stream.wmv mms://lang.stanford.edu/courses/ee380/050216-ee380-100.wmv
```
