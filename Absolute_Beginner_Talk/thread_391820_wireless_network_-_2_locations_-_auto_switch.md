### Post by Kobalt on 2007-03-23
The best tool you could use to switch easily and in a graphical way between wireless networks would be using [network-manager]("https://help.ubuntu.com/community/WifiDocs/NetworkManager"). It's easy to install since it's in the repos, it's very powerful and it will be included natively in the next ubuntu release.
