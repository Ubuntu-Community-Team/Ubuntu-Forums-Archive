So, as I stated I'm new to Linux, I figured that it would be a good idea to dual-boot this aging Windows box. I had heard Ubuntu was a good distribution, and so far I like it. (7.10, all the latest updates installed)

### Post by atomkarinca on 2007-12-29
Open Nautilus (e.g. Places > Home Folder), Edit > Preview > Sound Files and select None. This may fix your problem.
