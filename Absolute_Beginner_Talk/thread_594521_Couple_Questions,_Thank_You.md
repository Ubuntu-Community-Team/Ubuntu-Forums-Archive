### Post by Rinzwind on 2007-10-28
Ubuntu comes with 2 names. 1 with a cool name and 1 in digits.

Ubuntu 7.10 = Gutsy Gibbon. So if you have indeed 7,10 you are allready using Gutsy Gibbon. 

The 7.10 is the release date (2007, 10t month).

The last few releases are:

    * Ubuntu 6.06.1 LTS (Dapper Drake)
    * Ubuntu 6.10 (Edgy Eft)
    * Ubuntu 7.04 (Feisty Fawn)
    * Ubuntu 7.10 (Gutsy Gibbon) 

LTS means long term support and has a longer support cycle (it's more stable but lacks the super cool feature). 

In Gnome System -> About Ubuntu shows what you are using.
In Gnome: Systen > Administration > Synaptic you can update from 7.04 (Feisty Fawn),


I never use mplayer and have always been using VLC. But I do think mplayers 'just works' if you install the codes for playback (in Gutsy the system will do this by itself when you click a file without codec (like with MP3)).
Here is a large webpage about mplayer: [http://www.ubuntugeek.com/install-mplayer-and-multimedia-codecs-libdvdcss2w32codecs-in-ubuntu-710-gutsy-gibbon.html](http://www.ubuntugeek.com/install-mplayer-and-multimedia-codecs-libdvdcss2w32codecs-in-ubuntu-710-gutsy-gibbon.html)
I bet this forum's ubuntugeek made this page :D 

IF you are using 7.04 Feisty Fawn this is an excellent 1st start for getting the most basic stuff installed
[http://ubuntuguide.org/wiki/Ubuntu:Feisty](http://ubuntuguide.org/wiki/Ubuntu:Feisty)
The same for 7.10 Gutsy Gibbon: [http://ubuntuguide.org/wiki/Ubuntu:Gutsy](http://ubuntuguide.org/wiki/Ubuntu:Gutsy)
