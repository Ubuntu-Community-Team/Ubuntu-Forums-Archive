### Post by hw-tph on 2006-05-28
There are several Ghost-like imaging tools for Linux - g4l ("ghost 4 linux") is one of those. It uses a curses interface - a GUI in a terminal, but you should be able to deal with it.

I prefer having /home on a separate partition. This way it is easier to back up, and if I mess up the system I can reformat and start over again but use my /home partition with my user data.
