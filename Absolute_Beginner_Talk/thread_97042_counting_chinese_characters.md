### Post by Sutekh on 2005-11-30
Go to Tools -> Word Count

I tried it with a MS Word file (.doc) that had 10 characters and had no problems

It says 10 words and 19 characters.  It obviously counts each character as a "word", don't know what characters is counting, but it seems to work.

**Edit**: I just realised that wasn't a fair test, because if the file includes English words as well as characters, the word count will be incorrect.  It doesn't distinguish between English words and Chinese characters.  Sorry, there must be another way though... 

**2nd Edit**: I recall a function in Word that allows you to select all instances of a special type of formatting, if there was something like this in OpenOffice you could select the font that is used for the characters and then use the word count to calculate the number of charaters selected.
