I think to contact a real fax machine you need to make a phone line connection at some point, either using a modem (phone-line faxmodem that is) or something like VOIP. Winfax (GTI Faxmaker?) allows a company's users to send faxes as emails to the central server, but I believe the server still has to have old-style modems to do the faxing over regular phone lines.
