### Post by ds[de] on 2006-04-17
Well you could either use ICQ/Gaim/SIM/GnomeICU or whatever messenger you prefer and send him the file.

Or one of you could set up an ftp server on his machine, make an account and give that information to the other person. Then you can connect to his computer via the ft-protocol (ftp) with programs like gftp and upload the file or vice versa if you open the ftp server on your machine.

An example for an ftp server for linux is proftpd.

Regards,
ds[de]
