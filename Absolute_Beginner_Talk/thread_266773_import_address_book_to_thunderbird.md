### Post by bodhi.zazen on 2006-09-27
Yes, it works great.

In outlook just export your address book. Move to to Ubuntu. Import with Thunderbird.

It is all done with pull down menus. It worked without a hitch when I converted my wife from Windows 2K to Zenwalk.

You can install Thunderbird in windows, import your address book, and then remove outlook. You will be more secure if you do.
