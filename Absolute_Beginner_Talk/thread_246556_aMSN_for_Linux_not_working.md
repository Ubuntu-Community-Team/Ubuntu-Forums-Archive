aMSN is in universe.  First, you'll need to enable the universe repo - [https://help.ubuntu.com/community/Repositories/Ubuntu](https://help.ubuntu.com/community/Repositories/Ubuntu), then install aMSN with ```
sudo aptitude install amsn
```
