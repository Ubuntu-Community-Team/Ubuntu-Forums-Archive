Thanks. I appreciate the linking, the more people point to it the less questions about installing X we got to answer (and more people will know about their OS) and we can then get to the more problematic/serious ones.

Now to edit your signature go to the top bar, click "User CP" then edit signature. You'll be taken to the box with your signature and you can preview it before saving (both buttons on bottom).

URL's work with the forum's formatting. You just need to pick the text you want it to read, for example "Complete Guide to Installation" and then you find the url you want to link to (in this case it's [http://ubuntuforums.org/showthread.php?t=500020](http://ubuntuforums.org/showthread.php?t=500020)). Then all you do is select the text and push the link button (in your menu it should be the button with a world globe on it and a link at bottom right), then paste in the url you want and it will be formatted for you.

The final formatting should read like so (I moved the bracket in red to break the coding so you'd see it, if you put it next to the URL it will be read properly) :

[COLOR="Red"][[/COLOR]
URL="http://ubuntuforums.org/showthread.php?t=500020"]Complete Guide to Installation[/URL]

Feel free to apply other formatting too like bold and other such things, or link to more guides. You are limited in characters though so be careful not to put too much.
