### Post by Campingman on 2007-04-27
In terminal
Sudo aptitude install wine.


It is also in applications/add remove (search for wine)

It is also in system/administration/synaptic package manager (search for wine)

Here are a couple of good wine resources once you have it installed
[http://frankscorner.org/](http://frankscorner.org/)
[http://www.winehq.org/site/docs/wineusr-guide/index](http://www.winehq.org/site/docs/wineusr-guide/index)
