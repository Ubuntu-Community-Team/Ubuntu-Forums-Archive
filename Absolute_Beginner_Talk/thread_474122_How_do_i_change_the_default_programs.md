### Post by NeoLithium on 2007-06-14
You can right click on the movie, and select Properties.  Then go to 'Open With' and select VLC instead of the totem option. :)

### Post by locke.dragon on 2007-06-14
right click on a file (.wmv for example), hit properties, open with, and choose.  if the program you want isn't there, just add it.
