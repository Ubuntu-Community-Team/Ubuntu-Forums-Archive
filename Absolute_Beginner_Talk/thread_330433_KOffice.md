### Post by viciouslime on 2007-01-03
Jut the same way you would install anything else; through synaptic, synaptic will automatically install the kde libraries for you. That's how everything works in ubuntu, if a program needs something, like the kde libraries, then it's deb file will "depend" on whatever it is it needs. So, just goto synaptic and mark koffice for installation, then click apply.
