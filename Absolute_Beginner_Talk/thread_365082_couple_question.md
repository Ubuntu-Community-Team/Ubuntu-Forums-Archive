### Post by ShareBuntu on 2007-02-19
Hi Michael,

Glad you like Kubuntu. I use it too. I've addressed your questions below, hopefully you find it useful.

> **greynails said:**
> 1. as a beginner, I looked at the forum and get this message: "To convert RPM packages to Debian package format use command ALIEN. New users: don't use alien method, If there are DEBs available you should use them or you might severely screw up your system."
but say if I got a rpm file, how should i install it?   ie. flashplayer.

I see no reason why you shouldn't use Alien. Look at [http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_convert_.rpm_files_to_.deb_files]("http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_convert_.rpm_files_to_.deb_files") on how to make .deb packages using Alien.

> **greynails said:**
> 2. what is a sudo?

Sudo is a program that allows you to run programs as your system's super user (root). It's a security measure and makes using your system a lot safer since you won't be able to make changes to certain areas using the account you use on a daily basis. Check out [http://en.wikipedia.org/wiki/Sudo]("http://en.wikipedia.org/wiki/Sudo") for more about sudo.

> **greynails said:**
> 3. there aren't enough programs that i wish to find in the add/remove program, say if i want a dictionary, how could i download it from ubuntu?  (some linux distributions like suse or redhead provides it, but i don't know whether ubuntu has).

I find using Google the best way to locate additional software for Kubuntu. Check out [http://ubuntuguide.org/wiki/Ubuntu_Edgy]("http://ubuntuguide.org/wiki/Ubuntu_Edgy") for a comprehensive guide to installing just about anything on your system. Make sure you enable additional repositories: [http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_add_extra_repositories]("http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_add_extra_repositories"). The guide is geared towards Ubuntu/GNOME but most of it works with Kubuntu/KDE as well.

If you have any other questions, feel free to ask.

Welcome to Linux. :)
