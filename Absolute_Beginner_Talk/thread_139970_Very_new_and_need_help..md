### Post by steve.horsley on 2006-03-05
Are you sure you burned the CD as an **image**? This seems to be the most common mistake. You do NOT put the .iso file on the disk as a large data file, and you do NOT open the .iso file and put the contents on the disk as data.

Nero has an option in the menu to burn an image file, and I guess other burning software will too. The important thing is that an iso image file is a complete image if a CD, including the file system and a boot sector. It must be burned back as an image.
