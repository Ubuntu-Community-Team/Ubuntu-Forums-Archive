### Post by Haegin on 2006-01-07
no - all the functionality from winzip is in either file-roller (gnome) or ark (kde). If you download a .zip file and try to open it the appropriate program will open automatically. Also you cannot use winzip on linux as it is a windows program. Windows programs work on windows and linux programs work on linux. All .exe files are windows programs. Linux programs can be .deb files, .rpm files or source archives (.tar.gz or .tar.bz2). To install things on linux use synaptic which will do all the downloading, installing and everything for you. Trust me - its alot easier than anything else. Don't even touch source archives or .rpms until you know more about linux and how they work.
