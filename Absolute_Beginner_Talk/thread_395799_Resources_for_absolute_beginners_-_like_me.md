### Post by akbob on 2007-03-28
I'm a very recent Ubuntu devotee.  Trying to get up to speed quickly I've been scouring the Internet for information, ideas and resources for all things Ubuntu, Linux, or Open Source.  I've compiled a list (attached) and created a Custom Search Engine with Google.  I've included 65 sites so far, which I'm sure is just the tip of the iceberg. 

[http://google.com/coop/cse?cx=001341933570257451639%3Ay8gwl467cqw](http://google.com/coop/cse?cx=001341933570257451639%3Ay8gwl467cqw)
I've also created a redirect page on one of my sites because the CSE link above is ridiculously long and has a tendency to break in some email clients [http://alaskanbest.com/ubuntusearch.htm](http://alaskanbest.com/ubuntusearch.htm)

If you know of other resources  that should be included in the search list, you could post them here, email them to [email]searchubuntu@gmail.com[/email] or add them directly as a volunteer.


I've added it to my search providers in FireFox so it's real handy when I'm looking for an answers, information and solutions.  I hope others find it helpful as well.
