### Post by wpshooter on 2006-11-29
Even if the CD was sent to you by Ubuntu that does not necessarily mean that it could not be defective.  

You still need to verify the integrity of the CD by running the verification check that is found on the initial Ubuntu boot screen menu.

Also, it is very advisable when installing this O/S, to make sure that you allow your CD drive to spin down between each installation prompt.

If you have means to do so, you would be much better off downloading, and burning the ISO image from the Ubuntu website.  Make sure that your computer is strictly dedicated to the tasks of downloading and burning the image.  Make sure you burn the CD at the slowest speed possible.
