### Post by Kateikyoushi on 2006-10-14
In gnome you could use [gnomebaker]("http://gnomebaker.sourceforge.net/v2/"), if you use kubuntu the [k3b]("http://www.k3b.org/") in kde. You can also use [graveman]("http://graveman.tuxfamily.org/").

To install it copy this to terminal.
```
sudo aptitude install gnomebaker
```
