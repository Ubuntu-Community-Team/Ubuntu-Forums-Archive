### Post by maruchan on 2005-07-15
If you are using Windows XP, you can download the ISO Burner powertoy and use that. Or, if you have Nero or Roxio, you can use one of those.

[http://isorecorder.alexfeinman.com/isorecorder.htm](http://isorecorder.alexfeinman.com/isorecorder.htm)

ISO burning is a specific CD-burning operation, so you can't just copy the iso file to a CD and burn it like that. You need to make sure you tell your burning software that you are making an iso image CD. Then point it to the Ubuntu iso.

If all goes well, you can restart your comp with the CD in the drive and install Ubuntu.

Edit: Things to know when installing:

MAKE SURE you have a free drive partition or a freed-up hard drive to install it on.
It's good to have a high-speed connection
Be patient and follow directions
If it looks broken, it's usually easy to fix
It's different than what you've used before
It is absolutely friggin' awesome once you get it all setup and tweaked right :D

### Post by wylfing on 2005-07-15
To answer your other question: Yes, you can play Windows games. At least, some of them. You need to obtain and install Cedega and its Point2Play utility. That will get a lot of the most popular games up and running with few problems. You can check the game compatibility database at [http://www.transgaming.com/](http://www.transgaming.com/) and see how well the game(s) you are interested work.
