### Post by erginemr on 2008-01-21
+1 for Python, because it is very easy to learn, cross-environmental (will run in both Windows and Linux), powerful and very productive (you can develop applications very fast with it). I have used Visual Basic, Pascal and Delphi (a godsent RAD) and had a short engagement with C in the past, and now am also learning Python, and I can recommend Python heartily to a new programmer.

Once you learn the basics of the language (whose syntax is very simple), you can start developing native graphical applications via Python extensions.

For an easy crash course on the Python language and how to program in general:

[http://openbookproject.net//thinkCSpy/](http://openbookproject.net//thinkCSpy/)
