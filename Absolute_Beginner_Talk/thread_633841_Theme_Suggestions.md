### Post by crimesaucer on 2007-12-07
You can also learn to make your own.

I make my own gtk-2.0 theme, my own Emerald window theme, my own GDM, my own panel images for xfce4-panel, my own icon packs, and I usually search for a nice image on flickr for my wallpaper/skydome/cube-cap themes.

This is my winter theme: [http://ubuntuforums.org/showthread.php?p=3870525#post3870525](http://ubuntuforums.org/showthread.php?p=3870525#post3870525)


... I have already found a new wallpaper that I like better, but my gtk and icons are still the same.


Just find some gtk themes/icon packs/emerald themes that you sort of like, then change them to use your own colors, and sizes. Learning how to modify a gtkrc is pretty easy. So is re-packaging an icon pack after changing them using Inkscape... and Emerald Theme Manager is super simple to use to make nice window themes.
