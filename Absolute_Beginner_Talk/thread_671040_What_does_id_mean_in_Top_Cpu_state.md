### Post by mcduck on 2008-01-18
> **pfwd.tech said:**
> what does `id` mean in the cpu states in top?
any one knows

'id' means idle CPU time, the time when your processor is not doing anything.

So if you are using 10% of your CPU, you'll get about 90% idle time.
