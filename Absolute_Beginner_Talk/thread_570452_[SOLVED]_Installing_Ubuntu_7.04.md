### Post by ThrobbingBrain66 on 2007-10-08
Yes, go with the x86 version.  You'll also have to use the alternate CD.  The LiveCD won't run on a computer with less than 256MB of RAM.

Edit: Lastly, you might want to try Xubuntu.  It's quite a bit lighter on resources, so it'll be better for your system specs.

[http://mirror.anl.gov/pub/ubuntu-iso/DVDs/xubuntu/7.04/release/](http://mirror.anl.gov/pub/ubuntu-iso/DVDs/xubuntu/7.04/release/)
