### Post by Kobalt on 2007-04-19
If you save the /.evolution folder that's in your /home directory then you'll keep all your mails, addresses, memos, calendars, etc. To see it press Ctrl+H (in order to view hidden files/folders).
It has proven to be efficient as far as I'm concerned...
