### Post by H.E. Pennypacker on 2007-06-17
You will not need Spybot or anything like it, because unlike Windows, the spyware that you have been used to simply cannot be effectively used on Ubuntu.  First of all, the applications that you used that were spyware cannot be natively run on Ubuntu.  Second,  their effects could only be very limited, if anything at all, because they'd be dealing with an entirely different operating system.

Even if you were to run spyware in a virtual/emulated area, the spyware still would have either no effect or limited effect.

In any case, the very first point I raised eliminates all the spyware you may have had: they cannot be run natively.  Don't worry about your browsing habits too much (other than using common sense, of course, such as not giving your credit card information to others), because software alone will not hurt you.

### Post by stchman on 2007-06-18
Here is how I understand viruses and spyware in Linux.

Viruses need to be able to have free reign of an operating system to do what they need to do.  The permissions structure of Linux means that viruses can only affect the person's /home folder.  Diversity is also a big defense for Linux as there are MANY distributions.

Secondly viruses are fairly specific to the OS they run on.

Spyware on the other hand can be in the form of tracking cookies (they are pretty harmless) to installing the robot type programs that cause your system to be a drone.

Software cannot be installed without the proper permissions.  So therefore viruses and spyware are pretty much a non-factor in Linux.
