Don't know if this has been posted before, apologies if it has (I have searched).

But if anyone is having some teething problems setting things up (like I've done in the past), I've stumbled upon a great little site...

[http://placelibre.ath.cx/keyes/index.php/2005/09/29/45-easy-ubuntu-23](http://placelibre.ath.cx/keyes/index.php/2005/09/29/45-easy-ubuntu-23)

All in all a nice little site with a handy little tool.

Hope some of you total beginners benefit from it.
