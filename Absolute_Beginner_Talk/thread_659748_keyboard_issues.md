### Post by Rocket2DMn on 2008-01-06
Older computers don't tend to recognize USB devices on boot (which means you can't use it at the GRUB menu).  I'm not sure about the FF problem, but it could be related to having 2 keyboards.  You should see if you can get your hands on a USB to PS/2 adapter, they often come with USB mice and keyboards.  Then you can use your newer USB KB through the PS/2 port and only need to KB.
Why don't you post the contents of xorg.conf, they have keyboard entries in there
```
cat /etc/X11/xorg.conf
```

### Post by Rocket2DMn on 2008-03-06
OK, let's try to auto-configure X and see if it picks up any changes, first we'll make a manual backup of your xorg.conf
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
sudo dpkg-reconfigure -phigh xserver-xorg
```
Then save and close any open files and restart the GUI with CTRL+ALT+BACKSPACE.  If for some reason you get stuck at a terminal, run
```
sudo cp /etc/X11/xorg.conf.backup /etc/X11/xorg.conf
startx
```
