### Post by mysticmatrix on 2007-08-29
Would run like hell out of box(well usually).
Intel is only graphics for which linux has out of box support, and if you have made a fresh install and not messed too much with /etc/X11/xorg.conf, just use Add/Remove from Applications Menu on top-left to add Beryl and Beryl-Manager. 
Then start Beryl Manger from Application--> System Tools.

If you face some problem, come back here.
