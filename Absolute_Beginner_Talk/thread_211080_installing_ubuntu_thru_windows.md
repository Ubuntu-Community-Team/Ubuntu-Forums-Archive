### Post by Bloch on 2006-07-07
The standard ubuntu disk is both a "live" disk and an installation disk. 

> The desktop CD allows you to try Ubuntu without changing your computer at all, and at your option to install it permanently later. This type of CD is what most people will want to use. You will need at least 192MB of RAM to install from this CD.

What you do is, you press F12 (or ctrl+bckspace) to enter BIOS. Change your bios to boot from the cdrom and save settings and reboot.

Ubuntu will start and there will be an icon on the desktop for you to install it.
Running ubuntu from the CD is slow and annoying, but it is a great way to test that all your hardware will work in ubuntu
