### Post by sumguy231 on 2007-09-09
Xscreensaver is the screensaver application which Xubuntu uses. You could try checking Applications -> Settings -> Settings Manager -> Screensaver and making sure 'Lock Screen After' isn't set to something ridiculously low which could cause it to happen while the desktop is loading. It's prompting you for a password, right?
