### Post by Jeek Elemental on 2007-08-15
well there is alot of information at

[http://www.winehq.org/](http://www.winehq.org/)

if you have a particular windows app youd like to run, search for it in the wine app db.

a good companion for wine is winedoors, which lets you install common windows programs in much the same way as synaptic:

[http://www.wine-doors.org](http://www.wine-doors.org)

note that both wine and wine doors are beta projects, expect rough edges. that said, both have worked great for me 8-)

generally for primers, google is your friend, there is usually a solution out there.
