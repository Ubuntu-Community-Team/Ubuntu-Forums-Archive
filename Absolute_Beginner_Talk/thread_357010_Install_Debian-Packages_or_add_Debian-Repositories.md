### Post by DirtDawg on 2007-02-09
Hi. Ubuntu alters .deb packages for their [customized repositories]("http://packages.ubuntu.com/"). Official Debian packages are often not compatable. If you haven't already, you should [enable the Universe and Multiverse repositories]("https://help.ubuntu.com/community/Repositories/Ubuntu#head-5bbef89639d9a7d93fe38f6356dc17847d373096").
