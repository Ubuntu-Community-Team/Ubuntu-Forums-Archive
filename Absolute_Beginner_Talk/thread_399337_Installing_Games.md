### Post by justleen on 2007-04-02
im not sure that Cube (Sauerbraten) is in the repositories.

You can go into System > Administration > Synaptic
Search for cube/ sauerbraten.. 

if its there, just mark it for install, and apply.

If its not there, go to the [cube site]("http://sauerbraten.org/"),  download it from the site (sauerbraten_2006_12_04_gui_edition_linux.tar.gz  is the file you want probably) and then follow the install instructions here [http://sauerbraten.org/docs/config.html](http://sauerbraten.org/docs/config.html)
