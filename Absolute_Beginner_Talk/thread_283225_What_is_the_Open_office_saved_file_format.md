### Post by lucia_engel on 2006-10-23
OpenOffice can save to a variety of file types. For example, text doc is odt, doc. spreadsheet is ods, xls. powerpoints is odp, ppt. and so on.

One can also export files as PDF (File > Export as PDF).

To save as Post Script I believe you have to go to print and then check "Print to File". To check if a file is a post script file (or any kinds of file), right click on the file and go to properties. Look under Type.
