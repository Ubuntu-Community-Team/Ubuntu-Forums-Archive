How would I install a game from a .bin file.  It doesn't seem like i can do anything with it.  Help please!

### Post by [S|G] on 2006-07-15
Open a terminal and type this:
```

cd DirectoryYouDownloadedTheBin
chmod +x game.bin
./game.bin

```
