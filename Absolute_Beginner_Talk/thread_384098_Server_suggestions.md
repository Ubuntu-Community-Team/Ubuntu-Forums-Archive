### Post by maniacmusician on 2007-03-14
I would go with Debian, but I guess I'm biased. For me, Debian based systems are easier to use due to their package management...especially when you're stuck with a CLI instead of GUI. 

Ubuntu would be even easier, as it is Debian based but tries to simplify things a lot.
