Download the disc image that you need from [http://www.ubuntu.com/](http://www.ubuntu.com/), burn the image to a disc (note that you will have to use a program that can write a .iso image to a disc, see here for more info on that [http://www.psychocats.net/ubuntu/iso](http://www.psychocats.net/ubuntu/iso)) and boot from the disc. You will be able to try out Ubuntu with it's Live CD environment, so you don't even need to install it to your hard drive.
