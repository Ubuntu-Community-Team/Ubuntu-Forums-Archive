### Post by IdolizingStewie on 2005-12-06
[QUOTE=thavard]I have loaded VLC onto the computer 
How do I get to be the default program for video and audeo
Alos it does not see the DVD player in disc start?[/QUOTE]

If you're playing content already on the computer, go into Nautilus and right-click on, for instance, an mp3 file and pull up the Properties menu. Go to the last tab and you can select which program you would like to be the default player for all mp3s. You will have to do this for each file type you wish to change.
