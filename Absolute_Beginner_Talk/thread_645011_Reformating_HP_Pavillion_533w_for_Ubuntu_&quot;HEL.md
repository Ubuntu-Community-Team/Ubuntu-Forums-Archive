### Post by ftaylor on 2007-12-19
Hello,

if you download the Ubuntu live cd image and write it onto a disk, then put it in your cd drive and restart your computer, it gives you all kinds of options for changing the partitions you already have. If you dont want to keep windows at all the ubuntu install process has an option to use the entire drive and scrap any existing partitions. There is also an option that will automatically install ubuntu in free space. You can also manually decide which drives to keep and which to remove, but this is slightly more complicated. You have to assign all the drives you want yourself.
