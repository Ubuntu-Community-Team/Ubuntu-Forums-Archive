### Post by scxtt on 2007-06-08
"Is there a program that allows a windows PC user to remotely operate an Ubuntu machine over a local area network?"
-- you can use Putty (SSH client) and VNC to go from Windows --> Ubuntu [ for administration ]

"Can Ubuntu store and share large amounts of files with Windows over a Windows local area network using USB 2.0 external hard drives?"
-- you can get Windows and Ubuntu to share files using samba - pretty simple to set up ... i've got NFS working b/t XP and Linux using Exceed for Windows, so even that is possible ...
