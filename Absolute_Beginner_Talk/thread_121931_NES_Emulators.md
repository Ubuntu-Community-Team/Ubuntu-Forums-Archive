### Post by lha on 2006-01-26
[QUOTE=BruschiOnTap]I would love an NES emulator for my Ubuntu system.  Would I be able to get one with apt-get?  I am dying to play Tecmo Super Bowl!!![/QUOTE]

Fceu is a good one. You can find it in universe. Here's [a how to on adding repositories]("https://wiki.ubuntu.com/AddingRepositoriesHowto") if you need it. If you use Synaptic, reload package lists (click Reload or press <control>-R) and install fceu. If you prefer apt-get, do
```
sudo apt-get update
sudo apt-get install fceu
```
