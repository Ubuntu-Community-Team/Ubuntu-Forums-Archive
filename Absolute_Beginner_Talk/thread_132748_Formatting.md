### Post by drmcclure on 2006-02-18
How do you format a floppy?  If you format a floppy in linux, then can you use it on XP and vice versa?  How are the two systems compatible.  Thanks!

---

### Post by 5-HT on 2006-02-18
If you are using Gnome (standard for Ubuntu), go to Applications > System Tools > Floppy Formatter.

If you format as FAT, you will be able to read and write on both linux and windows.

If you format as ext2 however, unless you install some special software in windows, it'll only be accessible to *nix systems.
