### Post by kpkeerthi on 2006-07-30
Which burning util are you using? I would recommend K3B that just supports multisession disks out-of-the-box, so you can use the same CD over and over again to copy files till it is full.

K3B is the only KDE app that I use in GNOME. There are GTK-based burners like GnomeBaker, Bonfire, etc. for GNOME but getting multi-session discs work with these need some command-line tinkering. I just prefer K3B over these as the overall burning process is very 'natural'.
