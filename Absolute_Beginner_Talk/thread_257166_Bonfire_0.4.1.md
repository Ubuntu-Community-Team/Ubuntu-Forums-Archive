### Post by slimdog360 on 2006-09-14
Try the firefox2 beta2 [http://www.mozilla.org/projects/bonecho/releases/2.0b2.html]("http://www.mozilla.org/projects/bonecho/releases/2.0b2.html")

you just have to extract it and type ```
bash /extracted_location/firefox
``` into a terminal. Of course you should change the extracted_location bit to where to actually extracted the file to eg. /home/bobanshirl/Desktop/

If you want to use the one you downloaded try a similar thing.
