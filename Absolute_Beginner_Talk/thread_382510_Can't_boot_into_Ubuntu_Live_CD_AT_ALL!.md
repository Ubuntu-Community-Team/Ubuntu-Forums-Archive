### Post by HereInOz on 2007-03-12
Make sure that your bios is set to boot from the CD first, then the HDD second.

F2, or Del or F10 will usually get you to the BIOS.  It should be in the instructions for the notebook somewhere.

Once you set the CD as the first boot device, it ought to work.

As Taurus said though, the alternate CD is not the Live CD, and will NOT give you the opportunity to view ubuntu running from the CD - it will take you direct to the install.

If you want to run Ubuntu from the CD to have a look at it, you will need the Live, or Desktop, CD
