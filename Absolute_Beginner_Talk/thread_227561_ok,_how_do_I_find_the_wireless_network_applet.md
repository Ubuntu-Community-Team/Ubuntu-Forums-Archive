### Post by GreenHawkIA on 2006-08-02
If I understand the question right you are looking for a program to switch back and forth between wireless networks graphically and easily??  Hands down the best program (Windows or Linux, IMHO - at least according to me and my non-tech, Windows, girlfriend) is Network Manager.  Just open a command line and type sudo apt-get install network-manager-gnome (or search for network manager in Synaptic).  If it doesn't add itself to the automatically starting programs go to System-Preferences-Sessions go to the Startup Programs tab and add nm-applet.  It will appear in the Notification Area in the upper right hand corner, and should handle all your needs.  For pics, etc, go to [http://www.gnome.org/projects/NetworkManager/](http://www.gnome.org/projects/NetworkManager/)  If this is not what you need, let me know.
