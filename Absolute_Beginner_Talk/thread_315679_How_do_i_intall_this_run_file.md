### Post by NESFreak on 2006-12-09
go to the terminal.
go to the directory where the file is by using the cd command just like on the windows command line.
first do this so you are sure it will run```
chmod +x et-linux-2.55.x86.run
```
that run the file
for local install (only you)```
./et-linux-2.55.x86.run
```
for system wide installation use```
sudo ./et-linux-2.55.x86.run
```

good luck
NESFreak
