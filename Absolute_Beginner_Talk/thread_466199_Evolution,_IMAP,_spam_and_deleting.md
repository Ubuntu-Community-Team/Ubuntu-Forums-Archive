### Post by asandor on 2007-06-06
I use Evolution as my mailer of choice. Select Edit then preferences. On the left pain, choose mail preferences and about half way down you will see  Delete Mail. Make sure that Empty Trash Folder on Exit is checked and choose the drop down of your choice. I have mine set to Every Time. This will purge the trash folder  when you exit Evolution. It will take a few seconds to empty if your mail server is remote.

Hope this helps.
