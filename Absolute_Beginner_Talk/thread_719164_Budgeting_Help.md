### Post by Red Shift on 2008-03-08
I use KMyMoney to keep track of income and expenses.

KMyMoney allows the user to create accounts (assets, liabilities, income, expenses, etc), add categories to those accounts (employment income, auto expenses, household expenses, credit card balances, etc), add payees (people and businesses), and add transactions in a ledger.  KMyMoney also generates reports such as monthly income/expenses, net worth, etc.

You can learn more at their website:
[http://kmymoney2.sourceforge.net/index2.html](http://kmymoney2.sourceforge.net/index2.html)

To install KMyMoney from the repositories, in a Konsole, type:

```
sudo apt-get install kmymoney2
```
