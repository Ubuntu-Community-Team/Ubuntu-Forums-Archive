### Post by stijngysemans on 2007-10-26
If you want to use use "C" drive for ubuntu do the following:
Start up the live Cd and the install
Choose for "manual partitioning"
Now you've got an overview of all your paritions (2).
Choose the 15 gb partition and press "Edit"
Choose "/" as mount point
Select the "format" checkbox on that 15 gb parition.

What that will do is install ubuntu on that 15gb partition. When you boot up your pc after the installation, you're other parition will be available to you!
