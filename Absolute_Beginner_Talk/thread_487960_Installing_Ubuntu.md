### Post by tarek on 2007-06-29
Hi, 
I had exactly the same problem, and the solution was to use the safe graphics mode in the installation screen.
I think it's the second option, choose it. The monitor refresh rate will be 60, which is not good but once the installation is finished, reboot and install the ATI driver.

I used Envy: [http://www.albertomilone.com/nvidia_scripts1.html]("http://www.albertomilone.com/nvidia_scripts1.html")
to install my ATI card driver and it's great, it takes care of everything plus it adds an ATI icon under Accessories to manage the card. I got the TV-out working too.
