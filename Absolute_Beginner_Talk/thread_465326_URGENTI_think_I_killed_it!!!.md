### Post by aysiu on 2007-06-05
I have no idea about e2fsck, superblocks, or exit statuses, but I can tell you how to post your /etc/fstab if you have a live CD.

Boot up the live CD, open up a Konsole and type (or paste): ```
sudo mkdir /recovery
sudo mount -t ext3 /dev/hda1 /recovery
kdesu konqueror /recovery
```
