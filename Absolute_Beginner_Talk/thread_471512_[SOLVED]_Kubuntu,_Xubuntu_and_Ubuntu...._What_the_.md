### Post by anaconda on 2007-06-12
The differences..
kubuntu comes with KDE
Xubuntu comes with Xfce
Ubuntu comes with Gnome

Xfce is the lightest and KDE and Gnome are about as heavy. They all have a bit different programs installed. I use Gnome, but often prefer the KDE applications, like kaffeine, krita, ktorrent etc..

The easiest way to instal KDE (or xfce) on top of your ubuntu is to simply write (in terminal)
sudo apt-get install kubuntu-desktop
(or sudo apt-get install xubuntu-desktop)
