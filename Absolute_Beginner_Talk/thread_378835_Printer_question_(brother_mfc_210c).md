### Post by luke411 on 2007-03-07
Go to "System" - "Administration" - "Printer" and try to add the printer using the wizard that is there. If there is a driver already installed it will use it after you select the model number and manufacturer of the printer. If you can't find something there that works, or a model that is close to the one you have, you may have to google on the internet for a driver for your printer. Brother is pretty well supported though and it might be there.
