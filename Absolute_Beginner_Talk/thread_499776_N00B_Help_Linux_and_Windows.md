### Post by Ek0nomik on 2007-07-13
You have two hard drives by the sound of it?

You could simply install Ubuntu and Windows onto one hard drive, and then use the second as a sharing device.  You would format the second as FAT32 so Linux and Windows can read and write to it.

Or did you want Linux specifically on one hard drive and Windows specifically on another?
