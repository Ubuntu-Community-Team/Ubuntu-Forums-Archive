### Post by blendmaster on 2006-10-10
Here's how I got my IP4000 working with no problem (obviously in same series, so it should work for you too).

Open up a text editor.

Type test (or whatever, it doesn't matter)

Make sure your printer is connected

Go to file >> Print (In the basic text editor anyway)

In this box, click add (the button on the right just below the list of printers)

It should take you to a printer connection wizard. Click the radio button "use a detected printer", and it should be there

If not (mine was there, though) you'll probably be able to manually install it

Edit: After installed, print the document. Make sure you have the ip 4200 selected as your default printer though, otherwise it'll use a different driver to run your printer, and the document might print in really tiny ink (it did in mine the first few times until I figured this out).

Hope I helped!
