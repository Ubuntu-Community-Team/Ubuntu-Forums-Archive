### Post by mcduck on 2007-08-10
> **kavon89 said:**
> I think Icon themes are slightly diffrent to install vs a regular window theme. I know for a regular window theme you just point the theme manager to the archive you download.

You can do the same with icons.

However with theme packs containing several versions of the theme or icons and metacity themes etc. it's often best to install themes manually (Dropping the package into Theme window will only install the first theme contained in the archive).Just extract the theme package and drop GTK2- and Metacity themes into ~/.themes and icons theme into ~/.icons.
