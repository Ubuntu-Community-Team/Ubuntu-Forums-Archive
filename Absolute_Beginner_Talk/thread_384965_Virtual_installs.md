### Post by hyper_ch on 2007-03-15
Did you install the VmWare Tools? Once Windows has booted you go to the VmWare Tools (or Options) menu and install it.

Have you tried to run it with Wine? According to their AppDB Forte Agent v4 works fine on Debian Sarge 3.1... since Ubuntu is a debian derivative it may also run well on Ubuntu :)
