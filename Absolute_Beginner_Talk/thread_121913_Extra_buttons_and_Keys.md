### Post by Ultimo Aliento on 2006-01-26
Hi ! 

I got a genius keyboard with many options , here is the way im using it :

First, go to System>keyboard , there , go to distributios tab, and select you keyboard if is available

Next , System>Keys combinations , there you can find a list of actions for key's , just go to the apropiate section (if you want to configure the media key's , just go to the the media section), select a action and press the key you want for that.

With that you should be fine, if you want to configure more options, in the config editor,  i can post you here, how i do that :D, but this should be enough ;)

Good luck !
