### Post by Seisen on 2007-06-26
Its more of a user preference than anything. Some people like having the new release and some people don't. Upgrading can be done either by using synaptic/apt-get or some people have resorted to backing up their information and using disk to completely wipe out the old release and install the new one. I know upgrading by using synaptic/apt-get it always seems like the servers are getting pounded and sometimes you will get errors. So it is usually best to wait about a week to upgrade using synaptic/apt-get.
