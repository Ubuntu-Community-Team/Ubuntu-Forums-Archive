I know how to make a launcher point to a website... however I would like the website to open in a new window rather than opening a new tab in an existing window.... I am sure it can be done... how do you do it?!?!?!

### Post by spiderbatdad on 2008-04-02
I think that is a browser setting. For example Firefox opens your home page in a new window. Then in Edit>>Preferences>>Tabs. the default setting is "open new pages in" a new tab...changing that to new window, I believe will do what you want.
