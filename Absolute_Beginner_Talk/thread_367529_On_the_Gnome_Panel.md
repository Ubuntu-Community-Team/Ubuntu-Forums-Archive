### Post by orb9220 on 2007-02-22
The network manger in applets is Network monitor NOT the network-manger that gives you multiple connects to ssid's.

That is started from sessions manger. Look for > nm-applet --sm-disable  in the startup programs tab. If it is there check that it is enabled. If not add it there.
