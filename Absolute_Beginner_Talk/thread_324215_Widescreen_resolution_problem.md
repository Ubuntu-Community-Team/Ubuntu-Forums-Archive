### Post by Kobalt on 2006-12-23
Hello,
To add more resolutions avaliable in your configuration menu type this command line : 
```
sudo dpkg-reconfigure -phigh xserver-xorg
```
Then select the resolutions you want avaliable with the **spacebar**. Restart X by pressing Ctrl+Alt+Backspace, and here you go !

### Post by bodhi.zazen on 2006-12-24
I advise you add vertical and horizontal refresh information in your monitor section, like this:

> Section "Monitor"
	Identifier	"Monitor1"
	VendorName	"DEL"
	ModelName	"DELL P1110"
	HorizSync	29-121
	VertRefresh	50-180
EndSectionYou need to know the numbers for your monitor :p

You may also need to add a line:

> Option "UseEdidFreqs" "1"Add this line just before the EndSection (actually location does not matter, anywhere in the monitor section will do).
