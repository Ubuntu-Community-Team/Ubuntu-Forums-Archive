### Post by bluefrog on 2007-01-02
Some programs from universe/multiverse do not create (if they do...) "correct" menus entries.

You can either create your own menu by using alcarte menu editor or sudo apt-get install menu menu-xdg and activate the debian menu entry in alacarte after a while (then all programs will have an entry in the debian menu)
