### Post by billgoldberg on 2008-03-20
If you have compiz fusion on, turn it off, it might do the trick

A little app that might help you with that:

[http://forlong.blogage.de/article/pages/Compiz-Switch](http://forlong.blogage.de/article/pages/Compiz-Switch)

If you really want to upgrade I would install some extra ram memory.
