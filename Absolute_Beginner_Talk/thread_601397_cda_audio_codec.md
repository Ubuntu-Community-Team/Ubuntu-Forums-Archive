### Post by Nikitas350 on 2007-11-03
CD Audio (.cda) tracks are audio files that can be stored on CD media. The .cda files are representations of CD audio tracks and do not contain the actual pulse code modulation (PCM) information. Cda files can be played only from a CD-ROM. You probably copied this file directly from an audio cd. To extract tracks from an audio cd use sound juicer (Application--->Sound & Video---> Sound Jouicer). Enjoy.....
