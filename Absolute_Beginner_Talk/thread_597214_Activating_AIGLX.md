### Post by jordanmthomas on 2007-10-30
I'm not sure if this will fix you up, but my "Modules" section is different than yours (also on Intel)

```

Section "Module"
        Load  "dbe"
        Load  "extmod"
        Load  "xtrap"
#       Load  "GLcore"  #you do NOT want this
        Load  "glx"
        Load  "dri"
        Load  "type1"
        Load  "freetype"
EndSection
```

EDIT
Also, I can't remember exactly why, but I was having troubles with compiz and I added this to /etc/profile (you'll need to logout and back in for it to take effect)
```
export LIBGL_ALWAYS_INDIRECT=1
``` and it fixed whatever my trouble was.
