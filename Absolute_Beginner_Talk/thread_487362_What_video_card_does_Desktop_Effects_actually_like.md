### Post by cogadh on 2007-06-29
Nvidia runs great, regardless of model. Nvidia is a huge supporter of the Linux community and provides excellent, robust drivers for their cards. I don't run Desktop Effects, but I do run Beryl (it looks better, IMO) and I had no problems getting it up and running.

Oh, if you are going to use Linux, you are going to need to use a little of that "terminal witchcraft" sometimes. Linux has made great leaps as a desktop OS in recent years, but at its core it is still an old school console OS.
