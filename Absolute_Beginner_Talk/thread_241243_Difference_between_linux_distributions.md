### Post by aysiu on 2006-08-22
The differences between distributions:

Package management
Default packages
Installation method
Hardware detection
Community support and documentation
Release cycle
Cost

But once you get it up and running with KDE, Gnome, or whatever and all the applications you like... it's very difficult to tell the difference.

### Post by stig on 2006-08-22
If you are new to Linux an important distinction between distros is that some are more difficult than others if you are unfamiliar with Linux, and some will look and behave more like Windows than others.

Some distros try to make it especially easy for people migrating from Windows. This might seem an advantage but it can mean that you don't learn about Linux, which is a shame (and perhaps then it would be better to stay with Windows!).

But Ubuntu is somewhere in between and is a great distro because you can learn some Linux but it's still user-friendly.

Of course, you have to pay money for some distros and that might be an important factor in choice too!

There is lots of info on distros at the DistroWatch site:
[http://distrowatch.com/](http://distrowatch.com/)

### Post by kwalo on 2006-08-22
What about philosophy? Each distibution aims at something different.

- Ubuntu is for human beings.
- Debian tries to be as stable as rock.
- Gentoo's main targets are speed and full control.
- Trustix is stability and security.
- DSL whants to be as small as possible.

It is worth to check what are the distribution's main goals, before you decide which to use.
