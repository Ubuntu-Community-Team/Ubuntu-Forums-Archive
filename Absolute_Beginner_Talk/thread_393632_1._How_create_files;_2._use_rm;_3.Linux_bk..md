Make a symlink to what?  The command to make a symlink is "ln -s {source} {destination}", where {source} is a file somewhere else on the system, and {destination} is the name for the new line ("10-local.rules", I think for this case).
Note: for that particular directory, you'll need "sudo " in front of the "ln" command.

> Q2. How do i use rm command to remove the typos made above such as 10.local existing in /etc/udev/ .
"cd" to the directory, then "rm {filename}" (or "sudo rm {filename}", in this case).
