### Post by AndyCooll on 2008-03-05
That is a request for a GPG key. To access (almost any) repository you need authorisation, which is normally supplied in the form of a GPG key. For the standard repositories like the one you seem to be having problems with these keys are usually installed by default as part of the installation process so you never notice the "authorising" process taking place.

Although it says its having a problem, have you been able to access that repository before? If so it may be a temporary problem with the repository itself. 

Otherwise you may need to download the key again.
