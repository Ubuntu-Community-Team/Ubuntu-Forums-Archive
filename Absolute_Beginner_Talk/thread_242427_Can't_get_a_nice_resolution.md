### Post by BatteryCell on 2006-08-23
To find out the driver you can just do:
```
nano /etc/X11/xorg.conf
``` 
and then look for the part that begins with:
Section "Device"
and find the line in that section that starts with driver, that's the driver.  

Im running kubuntu(and kde) so I dont know how you can check this with a gui in gnome, but if you happen to be running kubuntu then you can do the same thing by going to:
System Settings->Display->hardware 
and its in there as well.

Edit:
Sorry, the reason why I'm asking is because for me I was using the vesa driver and could only get three resolutions (same as you), but after I switched the driver to nvidia after installing the nvidia-glx packages I had alot more.

### Post by halfvolle melk on 2006-08-23
You could run
```
sudo dpkg-reconfigure xserver-xorg
```
and at some point select the desired resolution, or edit /etc/X11/xorg.conf bij hand. Important bits:
```
Section "Monitor"
        Identifier      "Generic Monitor"
        Option          "DPMS"
        HorizSync       27-96
        VertRefresh     50-160
```


```
SubSection "Display"
                Depth           24
                Modes           "1280x1024_85" "1024x768" "800x600" "640x480"
        EndSubSection
```
You'll have to adjust these to your needs offcourse.
