### Post by testube_babies on 2006-07-14
There are plenty of extra screensavers hiding in the repositories.  First, make sure you have extra repositories enabled: [http://ubuntuguide.org/wiki/Dapper#How_to_add_extra_repositories](http://ubuntuguide.org/wiki/Dapper#How_to_add_extra_repositories)

Then, open up a terminal and enter these commands:
```
sudo apt-get install xscreensaver-data-extra xscreensaver-gl-extra
sudo rss-glx_install
```
