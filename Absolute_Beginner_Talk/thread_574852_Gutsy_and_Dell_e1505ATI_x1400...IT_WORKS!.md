### Post by raydeen on 2007-10-13
Finally! Ubuntu is playing nice! I had to d/l a modified ISO from [www.mylittleubuntuguide.com](www.mylittleubuntuguide.com) to get 7.04 to work and up until the current RC of Gutsy, that was the only solution. Now I have all the Compiz/Fusion eyecandy (well, maybe most of it) and every thing seems really stable. There is a bit of a hit in 3D performance with whatever ATI drivers Gutsy is using (Lattice screensaver isn't as smooth) but so far the only real stumper I've run into is that Savage:Battle for Newerth won't launch again. It wouldn't launch under Feisty until I found the trick where you have to add *setsave vid_mode 9 * to the startup.cfg in ./game but that doesn't seem to be working now (unless I have to change it to a different number). But in any case, I'm a happy camper. Thanks Ubuntu. Best. Distro. Ever.
