### Post by RomeReactor on 2007-08-15
Hi. I'm also using wireless, and it does work. Read these lists of supported wireless cards from the [Free Software Foundation]("http://www.fsf.org/resources/hw/net/wireless/cards.html") and the [Ubuntu Wiki]("https://help.ubuntu.com/community/WifiDocs/WirelessCardsSupported"). Try to get a laptop with one of these cards.

Please note that many cards that are not supported can also be made to work using a program called NdisWrapper.
