In my experence you wont be able to do that. Vmware, even with vmware tools installed is not capable of that resolution. The reason is it isnt using your video card, but a virtual svga video card. If you did manage to set it, say using 
```
sudo dpkg-reconfigure xserver-xorg
```to reconfigure the x server. Odds are the x server will crash.

### Post by myoungf1 on 2007-07-20
From a terminal type lspci.  Then look for you card name and at the front of the card name it will give you a number like 02:00.9.  Convert this in you xorg.conf to BusID PCI:2:9:0.  Below I copied what my lspci looks like and highlighted where my card is in the list.

00:00.0 Host bridge: Intel Corporation 82845 845 (Brookdale) Chipset Host Bridge (rev 04)
00:01.0 PCI bridge: Intel Corporation 82845 845 (Brookdale) Chipset AGP Bridge (rev 04)
00:1e.0 PCI bridge: Intel Corporation 82801 PCI Bridge (rev 05)
00:1f.0 ISA bridge: Intel Corporation 82801BA ISA Bridge (LPC) (rev 05)
00:1f.1 IDE interface: Intel Corporation 82801BA IDE U100 (rev 05)
00:1f.2 USB Controller: Intel Corporation 82801BA/BAM USB (Hub #1) (rev 05)
00:1f.3 SMBus: Intel Corporation 82801BA/BAM SMBus (rev 05)
00:1f.4 USB Controller: Intel Corporation 82801BA/BAM USB (Hub #2) (rev 05)
00:1f.5 Multimedia audio controller: Intel Corporation 82801BA/BAM AC'97 Audio (rev 05)
[COLOR="Red"]01:00.0 VGA compatible controller: ATI Technologies Inc RV280 [Radeon 9200 PRO] (rev 01)[/COLOR]
[COLOR="red"]01:00.1 Display controller: ATI Technologies Inc RV280 [Radeon 9200 PRO] (Secondary) (rev 01)[/COLOR]
02:00.0 USB Controller: ALi Corporation USB 1.1 Controller (rev 03)
02:00.1 USB Controller: ALi Corporation USB 1.1 Controller (rev 03)
02:00.2 USB Controller: ALi Corporation USB 1.1 Controller (rev 03)
02:00.3 USB Controller: ALi Corporation USB 2.0 Controller (rev 01)
02:00.4 FireWire (IEEE 1394): ALi Corporation M5253 P1394 OHCI 1.1 Controller
02:01.0 Ethernet controller: D-Link System Inc RTL8139 Ethernet (rev 10)

This is the section of your xorg.conf that you need to put the above number to.
Section "Device"
	Identifier	"ATI Radeon 9250"
	Driver		"ati"
	BusID		"PCI:1:0:0"
	VideoRam	128000
EndSection
