### Post by mrehorst on 2006-10-03
It isn't what linux gives to the former windows user, but what it does not give them that makes the big difference.  No more rebooting 3 or 4 times per day, no concern about viruses, no need to reinstall the OS every 6 months, and etc.

Of course, it doesn't give them MS apps that the rest of the world is using- OO is good, but it isn't 100% compatible.  Like windows, linux requires a certain amount of geek knowledge to solve problems.

Linux is great for us geeks, but it still isn't quite ready for the rest of the world.  When it is it will install and configure itself, it will have things like wine built it so it can run windows apps, the browser(s) will have all the plugins to view any web page.
