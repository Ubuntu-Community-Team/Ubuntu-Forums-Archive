### Post by vexorian on 2007-09-10
all right

first, install abiword, go to synaptic package manager and find the abiword package.

Then you open up a terminal , go to your folder (using cd , etc and you can type this:

```

for i in *.doc ; do abiword --to=pdf $i ; done

```


wonders come from: [http://opensource.weblogsinc.com/2005/06/29/use-abiword-to-convert-filetypes-on-the-command-line/](http://opensource.weblogsinc.com/2005/06/29/use-abiword-to-convert-filetypes-on-the-command-line/)
