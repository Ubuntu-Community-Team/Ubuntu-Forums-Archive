### Post by julian67 on 2008-03-01
Antix is excellent for old computers. Its default desktop is fluxbox which might not be to everybody's taste but it's done really well and is surprisingly easy to use. It has tools like Synaptic and should feel familiar to anyone used to Ubuntu or Debian. Antix is based on mepis so it has all the mepis config tools (easy samba set up, cpu scaling, power management, suspend, simple wep/wpa wireless config etc). I found it very easy to follow a guide and install Xfce and have it as the default desktop environment on booting. I started using it on an old centrino laptop (pentium-m 1.6 GHz) a couple of months ago and like it enough that I just installed it on my Core Duo centrino Lenovo laptop. It's impressive for speed and low resource use. I found that booting into Xfce even with wicd tray icon and and beagle indexing & icon started by default it uses less RAM than a completely default Xfce desktop on Arch on the same machine. It's a really fast set up and you can choose to run it using Debian testing or unstable repositories so you can have it right up to date with all the latest applications.
