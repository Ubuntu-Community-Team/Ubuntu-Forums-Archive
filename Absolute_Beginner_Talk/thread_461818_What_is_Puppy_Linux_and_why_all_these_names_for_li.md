### Post by Haegin on 2007-06-02
They are different distributions of Linux. The Linux kernel is not enough on its own to get everything you can see on your desktop going, you also need various libraries and programs (such as Gnome or KDE) to get what you recognise as a fully functioning desktop. With other operating systems such as Windows XP or Mac OSX the whole set is installed at once and the user isn't aware of the distinctions. On Linux there is more than one choice for almost everything but as it is very hard for an inexperienced user to know what they need to install the different distributions provide a working desktop environment for you to start with.
There are many different distributions because different people want to use Linux for different things. Some want a full desktop (like Ubuntu and Kubuntu) whereas others want a very lightweight and small desktop for older computers, a media center, a web server or any one of many other things. Linux is also used on live CDs for testing hardware support, fixing computers (even Windows computers) or for security testing. Some people have even used Linux for very specific applications such as a dedicated router and firewall.
Obviously they all need different names to tell them apart and most of the different projects have a reason for that name. Debian is named after the creator, Ian Murdock, and his wife (then his girlfriend), Debra. Ubuntu comes from the Zulu and Xhosa concept of ubuntu, and can be roughly defined as, "the belief in a universal bond of sharing that connects all humanity".
Puppy Linux is a very lightweight but full featured Linux distribution that runs from a live CD (or other external media such as a zip drive or USB pen). For more information on Puppy Linux check out wikipedia, distrowatch or the Puppy Linux website:
