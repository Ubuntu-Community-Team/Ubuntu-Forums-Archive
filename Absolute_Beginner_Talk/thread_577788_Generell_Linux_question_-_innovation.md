### Post by haldean on 2007-10-16
Some innovations?!

Well, the largest supercomputer in the world (Seaborg) runs Linux.

Almost every small electronic device you use (including mobile phones and those TV screens in the back of the airplane seats) run Linux.

Linux is the only major OS which can run off of a CD or a USB stick without being installed to the hard drive. (Yeah, BSD can too, but it's hardly a major OS)

Almost every computer cluster is based off of Linux, and (unless I'm mistaken) the first set of parallel clusters was created through Linux. Try getting parallel computing using a Mac, I dare ya.

This will make some people cringe, but Linux is also the leader in visual desktop effects. Compare Compiz Fusion (or even its predecessors, Beryl and Compiz) to the effects on Windows or Mac. Ballmer goes on about how the little titlebars on the top of their brand-new, 5-years-in-the-making Vista are translucent. Well woop-de-doo. I can press Alt and use my mousewheel to make an entire window translucent. I can flip my desktop around like its a cube. I can stick windows to the backs of each other. And I'm using the old version of the current software

Also, I think it's pretty amazing that a group of like-minded volunteers came together and created an OS which is a strong competitor for Windows, which is created by a multibillion-dollar multinational company. That's pretty innovative.

There's a huge list, and I find it hard to believe that you couldn't find these anywhere else. Go to a site like digg.com and you'll be astounded by the number of "Top Ten Things About Ubuntu" lists you'll find ;)

BTW: Hard to have PR without PR people... I think most of what you've heard is satisfied users. I know it's a hard concept for Windows users to get used to, but people who use Linux generally enjoy using their computer :)
