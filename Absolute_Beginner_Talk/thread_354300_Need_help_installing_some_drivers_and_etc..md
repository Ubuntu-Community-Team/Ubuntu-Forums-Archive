### Post by shareMenaPeace on 2007-02-05
Hello,
what exactly cou want to install for drivers?

And a tip for starters try this automatic install script (mp3 codecs, browser plugins, video codecs and much more).

[http://getautomatix.com](http://getautomatix.com)

To run this just downlaod the installer for your ubuntu version and click to install - than it appears in

Applications -> System Tools - > Automatix

Now you can install with a few clicks many usefull stuff.

For running windows software (exe) you can try out wine.
Use automatix to install it or choose

System -> Administrator -> Synaptic Package Manager and search gere for wine and install.

Basicly if you want to install or uninstall stuff first check synaptic or use automatix.

Tar.gz files are archives (packed data) like zip or rar under windows.
