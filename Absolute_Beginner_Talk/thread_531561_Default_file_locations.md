> And what locations should new applications be installed to?

Different parts of a program are typically put in different places; the binaries, libraries, images, and config files all go in separate places. The good news is that you don't actually have to worry about where anything goes because your package manager will do that for you.

P.S. If you're looking for binaries, those are typically in /bin, /usr/bin, or /usr/local/bin.
