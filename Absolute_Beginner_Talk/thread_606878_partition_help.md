### Post by cotcot on 2007-11-08
Download the Gparted LiveCD. Boot from it and you get a graphical interface allowing you to extent the ubuntu partition with the unallocated space (supposing the unallocated space is just behind the ubuntu partition. (if not post what you get wit "sudo gparted")

You might do this with the ubuntu install/live CD as well. I always use the Gparted LivedCD for partition jobs and had never had an issue with it  till now. I avoid commenting or proposing things I have not experienced myself. Maybe somebody else could comment on the ubuntuCD.
