### Post by est on 2007-10-21
I know this is an old post, and noone cares any more, but if you want to be pedantic, you could all say they stand for GNU:

GNU = GNU's not Unix
Gnome = GNU Network Object Model Environment (originally)
GTK = GIMP toolkit = GNU Image Manipulation Program Toolkit

So anything starting Gnome or GTK also starts GNU.
