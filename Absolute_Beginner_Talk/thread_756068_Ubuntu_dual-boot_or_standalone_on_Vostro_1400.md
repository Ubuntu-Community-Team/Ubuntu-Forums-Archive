### Post by Inxsible on 2008-04-15
You cannot select 32 bit or 64 bit from the Live CD. You have to download the appropriate version beforehand.

7.04 is also an older version. I would recommend that you wait for another week and then install 8.04 which is coming out on the 24th of april.

If you are keen on installing it now, then try and install the beta version of 8.04 (updates will make sure that you have the latest once 8.04 releases)

or you can also install 7.10 for now, and then later upgrade.
