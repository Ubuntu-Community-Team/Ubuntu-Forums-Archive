### Post by ayenack on 2007-08-02
What is it that you want to do with your keyboard? If it's simply adding custom short cut key's this can be done from System>Preferences>Keyboard_Shortcuts. Dead simple. If you want to change the key map then use System>Preferences>Keyboard and choose whatever suits you best.

Good luck. ayenack.
