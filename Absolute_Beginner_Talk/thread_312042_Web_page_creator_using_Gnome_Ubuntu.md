### Post by houstonbofh on 2006-12-03
Since you are a GUI guy, run "synaptic" and enable the extra universe and multiverse repositories.  Now "reload" in synaptic, and you will find easy packages for a lot of stuff.  I use NVU, and it is a one click install with synaptic.  Apache is also there.
