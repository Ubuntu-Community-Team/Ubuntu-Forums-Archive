### Post by Jimmyfj on 2008-04-03
If you have a router with a build-in firewall what more do you want? 

GNU/Linux come with a build-in firewall at kernel-level, and all its ports closed by default, so you have no need for a firewall.

Antivirus program is only needed if you forward a lot of E-mails to Windows users just to ensure yourself that you don't send those not-protected-users any virus through an E-mail. 

GNU/Linux is the most trustworthy system around. The only system more safe than this is a not yet turned on computer.
