### Post by cilynx on 2007-01-09
Officially, i386 == Intel 386 (32-bit) Architerture or compatible.  amd64 = AMD 64-bit architecture or compatible.  For all intents and purposes, use i386 for any 32-bit system and amd64 for any 64-bit.  Just FYI, the 32-bit version of Ubuntu will run on 64-bit processors.
