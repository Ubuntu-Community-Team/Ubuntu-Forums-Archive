### Post by scrooge_74 on 2007-01-17
Like in windows you need the correct drivers. Some card manufacturers only have close source or binary format drivers.  Ubuntu only ships with open source drivers, that is why out of the box you dont have 3D acceleration on Nvidia cards.

You should buy cards that are well supported in Linux so you dont end with a $150.  supper dooper card that can only work at 600x800 because of bad driver support in linux
