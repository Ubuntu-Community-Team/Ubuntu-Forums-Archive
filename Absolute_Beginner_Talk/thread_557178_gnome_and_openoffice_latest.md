### Post by southernman on 2007-09-22
Software isn't allowed into the repos until it's been thoroughly tested by the developers... testing for security holes, bugs and whatnot.

You still have the option to install the "latest & greatest" if you need them, but you'll need to do it yourself, with either source files, or .deb packages that may be available from their website.
