### Post by CatKiller on 2006-11-08
> **doh224 said:**
> what to use insteed of direct x ?

The most direct (sorry) replacement for [Direct3D]("http://en.wikipedia.org/wiki/Direct3D"), the part of [DirectX]("http://en.wikipedia.org/wiki/DirectX") that draws 3D bits, is [OpenGL]("http://en.wikipedia.org/wiki/OpenGL"). There is also [SDL]("http://en.wikipedia.org/wiki/Simple_DirectMedia_Layer") to handle some of the other sections of DirectX.

[Wine]("http://www.winehq.com/") is available from the [Universe repository]("https://help.ubuntu.com/community/Repositories"). You can find some more information on how to use it [here]("http://doc.gwos.org/index.php/HowToWine").

There are quite a number of games that run natively in Linux without the need for Wine. You can find a list of some of them [here]("http://doc.gwos.org/index.php/Native_Games"). Or you can find out quite a lot from the [Gaming & Leisure forum]("http://www.ubuntuforums.org/forumdisplay.php?f=93").

Welcome to the community.
