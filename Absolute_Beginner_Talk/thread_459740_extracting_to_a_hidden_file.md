### Post by Ek0nomik on 2007-05-31
If you are doing it visually (with a file browser such as Nautilus) then you can hit Control + H and it will show hidden files.  Then you can copy and paste.

Or, if you want to do it via terminal:

```
mv skin_luis.zip /wherever/the/beginning/path/is/.vendetta
```
