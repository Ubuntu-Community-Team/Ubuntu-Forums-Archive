Ask them why they said it.

Ubuntu is used for many reasons, a few are:

* It is free as in beer
* It has all the functions and more that people use.
* It is free as in speech
* It is flexible, and can be suited to anybodies taste (don't let the defaults fool you)
* You don't need new hardware to run it (like Vista)
* It is secure, and you don't need to use Anti-Virus software

A few things before you use it:

* Linux (Ubuntu) is not Windows, so you will have to learn new things
* Linux can use some Windows software under Wine, but not all
* Linux has thousands of applications available for free, and can be easily added
* Linux can be configured to your tastes, as I said before, but it takes time to learn it

You can try Ubuntu without installing with the live cd. You can run it of the cd without installing it. See the Ubuntu wiki and homepage for more.
