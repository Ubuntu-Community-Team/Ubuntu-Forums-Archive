### Post by tonyr1988 on 2007-02-19
GDM = your login prompt. The default has a brown background, the Ubuntu logo in the middle, and the username / password box below the logo.

When you download the GDM theme somewhere (It doesn't matter where you put it...I believe it's a .tar.gz file), don't untar it.

System -> Administration -> Login Window -> Add

Browse to it, Install, and you should be good to go!

### Post by potterzot on 2007-02-19
Under system >> Preferences >> Themes

That opens up a dialog.  You can download themes from gnome-look or gnome-art.  Do a google search for those.  once downloaded you can just drag the downloaded file into the theme window and it will install

good luck and have fun :)
