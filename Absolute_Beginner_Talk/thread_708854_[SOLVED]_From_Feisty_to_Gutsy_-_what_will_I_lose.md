### Post by mytwobears on 2008-02-26
Hi:

I don't know about those specific packages,  but when I used the upgrade function to upgrade from Edgy to Feisty and from Feisty to Gutsy, all my settings were retained after each upgrade.  

The only thing is, if a package is no longer a part of the new OS (it's no longer being maintained nor a part of the repository) it will remove it if you don't tell it otherwise.  During the upgrade process it will alert you and I think it may ask you if you want to continue.
