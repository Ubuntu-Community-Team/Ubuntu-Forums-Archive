### Post by zhinker on 2007-05-14
The Terminal basically Ubuntu's version of Window's Command Line prompt.  If  you don't know what that is then in windows go to Start->Run and type in cmd and hit 'ok' to open it up.  In ubuntu you use it to do various things like installing certain programs and changing the system settings.  It's really helpful when you're following instructions because then you just need to copy and paste the commands (code) that others give you into the Terminal.

### Post by rich.bradshaw on 2007-05-14
If you know what to type you can perform any command that the operating system can do.

A lot of advice here uses it, as it is much quicker to use and describe.

i.e.

sudo apt-get install htop

is quicker than

Click on System, go to Administartion, go to Synaptic Package Manager. Wait for it to open. Search for htop. Mark it for installation. Then click Apply Changes, then wait then close.

They do the same thing, namely installing htop.

If you copy and paste commands into the terminal there is much less chance of messing things up! However it makes it harder to learn what's going on.
