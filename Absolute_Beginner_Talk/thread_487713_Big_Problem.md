### Post by ronocdh on 2007-06-29
I don't think this has anything to do with changing to 64-bit. First, an obvious question: are you sure your hardware supports 64-bit? Beyond that, It may have been an error resulting from an improper burn. Did you verify the integrity of the burn by checking the CD for errors? The installation CD has an option for this on the main screen. I'd suggest you try installing from the "alternate" installer disc, as this is text-based. Then before installing, delete your current partition(s), create new ones, format them, then proceed with the installation. This is a safer way to install, as it ensures data from the previous installation will not be present and thus give you trouble.
