### Post by IYY on 2006-08-27
It's not that XFCE is "faster", it just uses less memory. This means that you have more memory left for other programs, and the window manager itself will run well on low memory systems. So really, it depends on the amount of memory you have to begin with, and how many programs you usually have open at a given time. If you have only, say 128 MB of memory, XFCE will be *much* faster than Gnome.
