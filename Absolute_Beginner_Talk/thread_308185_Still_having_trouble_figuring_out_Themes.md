### Post by taurus on 2006-11-27
System -> Preferences -> Theme -> Install Theme...  Then, navigate to where you downloaded the theme.  You don't even have to unpack it first.  The installer will do that for you.

Luckily most themes would just need to drag 'n' drop your theme tar (the file you have downloaded without even extracting it. You smiply drag 'n' drop it to the Themes window and it should instaqll.
