### Post by Jimmey on 2006-02-13
Open up the terminal. You need to change the terminal's current directory from your /home directory, to the directory where the .run is located ( if the .run is in /home, you've no problems. )

To change to the directory of the .run, in the terminal, type 

> cd /home/Desktop  ( Obviously, you'd replace '/home/Desktop' with the directory where the .run is. )

To make sure that you CDd to the right directory, try 
> ls, which just lists the contents of the current directory. If you see the the name of the .run file among those listed, you're all set.

Here's how to execute a .run ( I'm pretty sure )

> sudo sh nameOfTheFile.run
