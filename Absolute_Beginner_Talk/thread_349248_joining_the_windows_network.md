### Post by nhandler on 2007-01-31
I'm not sure what you mean by join the network. You can share/access shared files in the windows network using samba. About the static IP, try no-ip.net. You get to choose a domain name like: yoursite.something.com. They have a list of about 50 different sites to replace something.com with. They also have a small script that you can download. This script, when run, will keep your domain name pointing to your ever changing ip.
