### Post by quicksilver1234 on 2007-03-29
I'm in command line. I type a command like "ls" and it give me more than a screen worth of data. I'm sure the ls manual page gives the switch, but I don't see anything like "-[character] one screen at a time" 
What should I use?

---

### Post by taurus on 2007-03-29
```
ls | more
```
Then space bar for the next screen.
