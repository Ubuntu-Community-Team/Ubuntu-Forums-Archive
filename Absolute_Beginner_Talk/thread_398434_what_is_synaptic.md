### Post by h0ax on 2007-04-01
Synaptic is your package management system.
Think of it as "Add/Remove Programs" in Windows.

You don't need to download it to be able to use it as it's included with the Ubuntu installation.

To see it go to System -> Administration -> Synaptic Package Manager

You will have to enter your password, then you see all the things you can install.

Be careful removing things if you're not sure what they do.

Hope this helps.
