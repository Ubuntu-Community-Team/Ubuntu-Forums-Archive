### Post by timlewis242 on 2007-02-09
The most obvious solution is to install NDIS Wrapper and try the windows driver that comes with the card. You can get it from the Ubuntu repositories and if you don't want to read the command line, there is also a GUI front end that can be downloaded from the same place.

Some cards are simply more friendly than others and Belkin claims to have a lot of LInux support but the few I've tried in the past were no easier than any other brand. Just make sure you keep the receipt and if you can't get it running, take it back and complain to the store that they need to support ALL OS, not just Windows.

### Post by Bartender on 2007-02-09
You're talking about adding a plain old NIC, not a wireless card, correct?  Turn the PC off, protect from static electricity, install the NIC, close 'er up, turn it on.  
That's all we did when we added a Belkin 10/100 NIC to a friend's Dell.  On the next boot, Ubuntu recognized the card and went online via his router.
