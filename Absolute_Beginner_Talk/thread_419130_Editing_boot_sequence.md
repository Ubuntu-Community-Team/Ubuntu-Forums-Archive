### Post by avanrens on 2007-04-22
After I installed the new updates Kernel 2.6.20-15 won't boot and I have to use Kernel 2.6 20-12.
I would like to edit Grub so that Kernel 2.6 20-12 boots first and remove 2.6 20-15. I have highlighted the first line and then pressed "e", but what I get is nothing like the line I want to edit.

---

### Post by johnc4510 on 2007-04-22
You need to edit your grub menu:

/boot/grub/menu.lst

The command to edit is this:

sudo gedit /boot/grub/menu.lst

Where it says:

defaut      0

Change it to the position of the kernel you want.

0 is 1st position, 1 is 2nd position

Look farther down the list to determine the position.
