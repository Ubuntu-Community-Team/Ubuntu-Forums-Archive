### Post by oscar on 2006-04-15
Yes you can. Just download the app you want through synaptic and the dependencies needed to run it will be installed automatically. You dont need to install KDE you just need the qt libraries, which will automatically be downloaded.

### Post by aysiu on 2006-04-15
[QUOTE=Tim Thumb]Are there any issues or problems that arise from using KDE apps in Gnome? For instance I'm looking at Quanta for web development in Gnome but I don't want to be investing time in it if I face problems down the line.

Apologies for the thread hijack.[/QUOTE] I've encountered only a couple of problems using cross-environment applications, and I don't even know if they really qualify as problems.

1. In AmaroK, you can drag-and-drop songs to your desktop, and they'll copy in KDE. If you do this same thing in Gnome, they'll *move* instead of copying.

2. KDE applications tend to load up more slowly in Gnome. Gnome applications tend to load up more slowly in KDE.
