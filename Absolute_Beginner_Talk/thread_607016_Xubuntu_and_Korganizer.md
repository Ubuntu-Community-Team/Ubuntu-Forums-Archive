### Post by julian67 on 2007-11-11
You can easily make a menu item using an existing one as a guide/template. In Xubuntu they are mostly stored in /usr/share/applications/ and are named yourapplication.desktop. Open some with mousepad or vim and you'll see they are simple enough and it's easy to make your own. 

You can create them using any text editor.
