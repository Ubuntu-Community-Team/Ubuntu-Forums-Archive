### Post by engla on 2006-02-21
You could have to choose a distro, there are some differences regarding if you just want a complete system or if you want something more -- a system full of extras, alternatives etc.

Ubuntu comes on only one cd but it gives you a complete installation -- you don't have to install anything else. However, someone has  made a choice for you so you don't have many alternatives..

If you rather want a system that's not like the default ubuntu install you perhaps have to use something like Fedora which comes on 4(?) CDs -- more software bundled, probably x3 of Ubuntu.
