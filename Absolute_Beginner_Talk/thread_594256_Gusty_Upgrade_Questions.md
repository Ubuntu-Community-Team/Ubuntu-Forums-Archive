### Post by bcrowell on 2007-10-27
>I don't know if all my apps currently installed in Edgy will work/update still. Will they?
They should.

>I don't know if all the work I did to get Totem to embed and open all formats (QT, AVI, MOV, ect) will be lost. Does the new release make changes/easy to play these type of formats?
In general, if you've hand-edited configuration files, the upgrade manager may ask you whether you want to keep the old one or replace it with a default version for the new distro. Make sure and save copies of your old config files.

Fiesty is fairly mature and reliable. Gutsy is brand new. I'd suggest you upgrade to fiesty now, since edgy is nearing the end of its life, and fiesty is likely to be reliable.
