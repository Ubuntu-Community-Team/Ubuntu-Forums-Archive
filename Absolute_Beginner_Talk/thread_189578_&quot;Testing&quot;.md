### Post by NiceGuy on 2006-06-05
The problem with 'bleeding edge' .debs is that you tend to find yourself in dependancy hell. ie. can't install this because it depends on this and this depends on that etc etc.

Have you conisdered going to the programs websites, downloading the source and compiling it yourself? If you want to have a 'tidy' system (and don't want the source files hanning arround so you can uninstall something later) I'd suggest you look in to [CheckInstall]("https://wiki.ubuntu.com/CheckInstall")
