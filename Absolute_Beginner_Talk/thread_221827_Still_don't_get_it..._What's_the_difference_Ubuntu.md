### Post by GStubbs43 on 2006-07-24
Ubuntu - Gnome
Kubuntu - KDE
Xubuntu - XFCE
Edubuntu - Gnome, with educational apps for kids (I think)

Xubuntu is really for older computers with little RAM and a small HDD.
It's up to which you think looks better, Gnome or KDE, you can install K/X/ubuntu, then use ```
sudo aptitude install K/X/ubuntu-desktop
``` and you can choose between KDE/Gnome/XFCE, depending on which you installed, on the login screen. But ultimitley, it depends on whether you like thhe setup of Gnome, KDE, or Ubuntu
