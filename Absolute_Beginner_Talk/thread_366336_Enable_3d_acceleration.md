### Post by RomeReactor on 2007-02-20
Hi. Try this:

```
gksudo gedit /etc/default/linux-restricted-modules-common
```

Put a **#** before **DISABLED_MODULES="fglrx"** so it looks like this:

```
# DISABLED_MODULES="fglrx"
```

Save, close and reboot.
