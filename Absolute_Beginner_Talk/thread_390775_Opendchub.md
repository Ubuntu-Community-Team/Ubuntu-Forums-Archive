Eh? Opendchub is server software that allows you to run a hub. You need client software in order to connect to hub (like [linuxdcpp]("http://www.ubuntuforums.org/showthread.php?t=193984&highlight=dcpp")). To get around the opendchub bug, set your connection type to 0.5 or 5 (I can't remember which) in the prefs. Then ask the hub operators to upgrade to 0.7.15.
