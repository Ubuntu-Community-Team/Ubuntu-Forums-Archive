### Post by WiZzkEy on 2007-06-16
Just a brainwave;

I recently changed my Ubuntu login screen to a "blackish" image. I also changed the Ubuntu splash screen to a "darkish" image, as well as my desktop background.

However, when I log in, I still see a brown flash on my screen (that is, after logging in and before the splash screen appears). I assume this has got something to do with some background color setting. Note that the brown color matches the original Ubuntu default background-color.

Changing the desktop background color doesn't seem to remove the brown screen flash, so I was wondering is there anyone that knows how to get rid of it (or change the color)?

Thanks in advance!

---

### Post by Lapino on 2007-06-16
This should solve it:
1. Go to System>Administration>Login Window
2. Choose the "Local" tab
3. Change the background color over here too.

2. Count to 1000, go get a drink, listen to classical music, sing I'm a little teapot
