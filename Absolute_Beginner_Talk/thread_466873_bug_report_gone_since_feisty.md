### Post by John.Michael.Kane on 2007-06-07
You can open synaptic, and see if bug-buddy is installed this allows for bug reporting on gnome based issues.

For filing bugs to launchpad you need make sure launchpad-integration is installed.

To use the launchpad-integration tool open the offending program,and select 'Help > Report a Bug' you should be sent to the following URL: [https://launchpad.net/ubuntu/+source/name](https://launchpad.net/ubuntu/+source/name) of program/+filebug

Hope this helps.
