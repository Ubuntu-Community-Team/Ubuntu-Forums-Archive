### Post by tonyr1988 on 2006-11-03
Did you install Ubuntu onto your hard drive, or only run the LiveCD? The LiveCD will run MUCH slower than if you install it to your hard drive (there's no way WinXP would be able to run as a LiveCD reasonably, to put it in perspective).

Which games are you wanting to run under Wine that aren't supported? Sometimes you can do just a few extra steps to get them working fine.

Also, to improve speed, you could run XFCE (Xubuntu) instead of GNOME (Ubuntu).

If you've already installed Ubuntu, you can download the XFCE package without having to reinstall your entire OS.
