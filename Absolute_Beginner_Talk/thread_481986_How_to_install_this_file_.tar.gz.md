### Post by sambion on 2007-06-25
Just a quick pointer. It's "cd /" to get back to the root directory. (without the quotes of course. Also notice the space between the cd and the /.) 
"cd .." moves up one level in the directory. Notice again the space between the cd and the ..
It's a little different than windows for moving around but not much.

Good luck on the install.

### Post by kevdog on 2007-06-25
if you want to make it more windows like just put an alias in your .bashrc file.  For example

alias cd..="cd .."
alias cd/="cd /"

That way you dont have to getting confused between windows and linux-- I rather like cd.. :p
