### Post by Kvark on 2005-12-04
Nothing, only a background image.

The only ways to get to the desktop is to close/minimize all windows or switch to an empty workspace. In other words, it is the hardest folder to get to and therefore the worst place to put shortcuts and folders on.

Use the panel and menues for shortcuts instead. Put the music, image etc folders in your home folder and use a file browser window (that you can switch to without minimizing all other windows) to get to them.

### Post by mcduck on 2005-12-10
One way is to right-click on a launcher in Applications menu and select 'Add this launcher to desktop'.

You can also right-click in desktop and select 'Create Launcher'. And launchers can be drag-and dropped to desktop from panels, and to panels from desktop..

I keep my desktop empty, I only use it for those files I'm currently working with and stuff downloaded from the Internet. I made a new autohiding panel to store all my application launchers. That way I can have my panels and desktop clean and uncluttered.
