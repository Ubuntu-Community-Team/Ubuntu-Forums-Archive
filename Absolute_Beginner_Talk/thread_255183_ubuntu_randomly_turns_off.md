### Post by Metacarpal on 2006-09-11
So when you boot normally (without the boot disk), it will start up and run for a while, then suddenly shut off?

When it shuts off like that, does it sometimes take a short time to get it to start back up again at all?  And does it seem to die faster when you do restart?  Does it last longer if you wait longer before rebooting?

If so, it sounds like an overheating issue.  You should listen to make sure your system fans are running and that there isn't dust or cat hair clogging the ventilation...
