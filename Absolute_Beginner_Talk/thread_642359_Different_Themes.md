### Post by Achetar on 2007-12-16
download moomex and HumanStrange from [www.gnome-look.org]("http://www.gnome-look.org") and use those with the ubuntu-darkblue2 background image. Perfection! (looks even better with the moomex Compiz Fusion theme and kiba-dock!) Will post screens later.

Unzip the moomex theme to /home/<username>/.themes and HumanStrange to /home/<username>/.icons

If you use the moomex Compiz Fusion theme, just download the .emerald file from the website, open the theme manager, and click import theme. Then navigate to where you save the file. 

You should be able to figure out how to change the desktop background.

Cheers!!
