### Post by Paul820 on 2007-08-26
To install xmms skins all you need to do is

1. Download the xmms skin archive. DON'T EXTRACT IT.
2. Open your home folder.
3. Press CTRL+h.
4. Browse to the xmms folder, open it, open the skins folder.
5. Put the unopened arcive file in there.
6. Start xmms.
7. Go to the options inside xmms and click skins browser.
8. Select skin.

### Post by glennric on 2007-08-26
Install the package xmms-skins in synaptic to get several of the basic skins.
For other downloaded skins, assuming they are in .tar.gz form, copy the file into the directory ~/.xmms/Skins/
