### Post by Tatty on 2008-04-03
Firstly i would reccomend waiting until 8.04 is released as a stable version at the end of this month.  At the moment it is still in Beta.


But yes i would install the 64-bit version... you have a 64 bit computer, why not use it to its fullest?  The reality is that you arent really going to notice any speed improvements if you just do typical day to day computing.  But if you do anything that does intensive number-crunching such as graphics editing then you might notice a big improvement.

There have been much talk of issues with 64-bit compatibility, but they seem to have been mostly ironed out by now.  I am running 64-bit gutsy on a laptop and I have not come across anything that I have had problems with.  If you do run into any problems however, there is a 64-bit section to this forum which might prove useful.

### Post by Jeremy Jackins on 2008-04-03
The biggest thing about choosing a 64 bit operating system is whether you want to be able to use 4 or more GB of ram. If not, most people will recommend 32 bit for compatibility (though it hasn't been an issue at all for me on 8.04).
