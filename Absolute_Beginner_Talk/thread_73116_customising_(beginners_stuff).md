### Post by darkmatter on 2005-10-08
The program you want for the widgets is gdesklets, install it from Synaptic.

For various monitors, launchers, etc., [http://gdesklets.gnomedesktop.org/](http://gdesklets.gnomedesktop.org/) is the one stop answer for all your shopping needs.:)

### Post by jipke on 2005-10-08
The widgets you are looking for are called desklets. To use them you will need to install gDesklets:

sudo apt-get install gdesklets

As you've noticed, there are many displays and sensors. They can be found on:
[http://gdesklets.gnomedesktop.org/](http://gdesklets.gnomedesktop.org/)

good luck!
