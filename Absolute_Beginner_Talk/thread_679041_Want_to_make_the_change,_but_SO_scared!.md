Ubuntu has a so-called package manager, which lets you install something like 20,000 applications easily by simply selecting them in a graphical user interface -- no more hunting for programs all over the net and wondering whether you can trust the source. Basically, if you've ever seen that there's a Linux version of something, you'll have it installed within seconds using the package manager. That being said, VLC is of course one of these programs. Media Monkey seems to be Windows only (you could of course try it in Wine -- a program that lets you run Windows programs in Linux), but I have no doubt that Amarok or Rhythmbox will win you over quickly. Check them out!

Security programs are plentiful -- Linux (and thus Ubuntu) has tons for every single need you might have. Heck, we even have a virus scanner for Windows viruses so we can make sure to not let viruses get into the network in the first place (clamav). ;-)

### Post by sajro on 2008-01-26
Welcome to the community! I hope you'll find (as I did when I first began) that most people are extremely gracious and helpful. 

If you're unsure about it, try using a liveCD. You don't have to install to do it and it will get you used to Linux and help decide if it's for you. If you think you like it, then you'll be on your way to a truly secure, stable, and customizable system.

Almost all software for Linux is "freeware". However, as a Linux user you'll learn most of it is free in both price and freedom. If you ever get into programming you'll enjoy being able to legally open up and observe the innards of your favorite programs. As mentioned before it's easy to install most programs through the package manager or Add/Remove. However, some will need command line commands. Don't let that scare you; it's mostly copy and paste you'll be doing for those

If you have any trouble just come here. It's very active and you're likely to be answered quickly and accurately.

### Post by ExpatPaul on 2008-01-26
It's surprisingly easy to take the plunge. Best thing to do is burn yourself a live CD and boot up from it - this lets you use Ubuntu without actually changing anything on your actual PC. And once you're happy that you have all the essential stuff, installing from the CD is very easy.

If you install from the Live CD it will give you the option to partition your hard drive so you dual boot between Ubuntu and Windows (if you do this, it's a good idea to defrag your hard drive first).

As for software, there's tons - and it covers everything that the vast majority of people need to do. Unless you are a gamer or have some **really** specific requirement, you can pretty much rely on finding what you need simply by clicking on the Add/Remove button in the Applications menu.

Welcome aboard, good luck and have fun. :)
