### Post by dfreer on 2007-06-26
> **tcoffeep said:**
> Is there any way to tweak internet speed, as in to speed up response times in Firefox, and get torrents downloading faster (I use Ktorrent, is that the best one available)?
You might want to look into blacklisting IPv6, either in firefox or in the kernel. neither method is very hard at all, although this may not fix the problem.
I've heard deluge is a great torrent downloader, back before I started using torrentflux (probably not an option for you) I used Azureus which worked fine. Make sure your ports are open, as most linux bittorrent apps have problems with UPnP in my experience.

> **tcoffeep said:**
> 2) Which games are playable on WINE, as I seem to be unable to find a reliable list.
winehq has the best list, IMO. 

> **tcoffeep said:**
> 3) Are there any good tutorials for Ubuntu? I had found a book, Ubuntu Linux Bible, but I don't know if that is good enough.
This forum is a great source for information, ubuntuguide.org has great step-by-step how-to's, and 
[http://www.psychocats.net/ubuntu/index.php](http://www.psychocats.net/ubuntu/index.php) has some decent starter tutorials. Also check the built-in Help system.

> **tcoffeep said:**
> 4) What are good anti-viruses/anti-spyware? Or, better yet, do I even need these? I've tried downloading
The main thing is to download a firewall, and firestarter is a great one. The main thing is to watch what you do as root, there is really no need for anti-virus unless you need to scan windows files (or have a windows file server), and spyware is virtually unheard of. In fact, I don't know of many anti-spyware tools for linux, although Clam Anti-virus does exist, along with rootkit checkers.

> **tcoffeep said:**
> 5) What is this BIOS error I keep coming across before Ubuntu starts? Should I ignore it?
If it is a true BIOS error, it probably has nothing to do with ubuntu. Try going into BIOS and look for some options, like "Halt on all errors" or "Display Error Information". You could also try hitting the "Pause" key when it flashes and see if you can write it down.
Welcome to Ubuntu and the Ubuntuforums, btw :D
EDIT: Check the manufacture of your mobo, or the makers of your BIOS (award for example), as error codes are very likely different across different BIOS and mobos.

### Post by ugm6hr on 2007-06-26

Is there any way to tweak internet speed, as in to speed up response times in Firefox, and get torrents downloading faster (I use Ktorrent, is that the best one available)?

I have an Acer 5050, which I believe is fairly similar in hardware terms to the 5100.

Your touchpad may be either ALPS, or more likely, Synaptics.  If so, I would suggest you edit xorg.conf to reflect your personal preferences, then install GSynaptics (seacrh for it from Synaptic Package Manager - no relation).  If you want, I can post my xorg for you to have a look at (I use vertical scrolling and touch tap features).

This is pretty informative:
[http://gentoo-wiki.com/HARDWARE_Synaptics_Touchpad](http://gentoo-wiki.com/HARDWARE_Synaptics_Touchpad)

As for Ktorrent - I suspect it will be the firewall that is slowing things.  Try installing Firestarter (from Synaptic) - and run it while KTorrent is downloading - if it generates events - then KTorrent is being blocked by the internal firewall, which can be edited by Firestarter.  You can check / set which ports it uses in the Ktorrent menu (settings->configure ktorrent), and then add those to the forwarded / accepted ports in the firewall.  If you use a DSL / ADSL router for internet, you will have to forward those ports on the the router too.

### Post by kvonb on 2007-06-26
The first thing I would suggest is that you will get a better response by asking individual questions!

If you run into a room and start firing off questions like a Fox "news" reporter on acid, you might give the wrong impression :).

Relax, slow down, enjoy doing the simple things and you will see how everything falls into place.

First off, which version of Ubuntu are you using, the standard (32 bit), or the 64 bit?  There are still some issues with the 64 bit version, so I would recommend using the standard version.

If you don't know, simply re-visit the page that you downloaded Ubuntu from and look for clues.

If you need help on any error, you will have to give more information.  People on here can't see your screen, and things like: "what's this error on my screen" are ambiguous, and will result in no help at all.

Fasterfox is something I would stay clear of, I have heard that a lot of websites ban users of Fasterfox because it is a bandwidth "pig", it downloads a whole site just to view one page, be careful.

If Firefox is running slow, then there could be a network problem somewhere, maybe ask in the network section.

For torrent files, I use "Azureus", it is a little more complicated as you have to also install Java, both are available from the add/remove programs icon on your menu.

Ubuntu won't be slowing your torrents downloading, again it sounds like a network problem.  Ask a question in the network section of the forum and give details of your network card and your Internet connection, if you have a wireless connection then that could be the source of a lot of headaches unfortunately.

Games, have a look in the gaming section of the forum, you will find several threads with lists of games.  Also you will need to tell us what type of game you enjoy, there are several genres.

Search the forums for an answer before asking a question, you will probably find that it has been answered a thousand times before.

And lastly, Linus is NOT Windows, it doesn't pretend to be Windows, and is NOT a replacement for Windows.  Things are done differently, there are things you can and can't do on Linux, the trick is to open your mind and be willing to learn new things.  Patience is a virtue.

Just keep in mind that a lot of people have put their time and effort into making Linux and Ubuntu for no reward at all, be polite and respectful and you will benefit, be rude and demanding and you will get nothing :).


Regards, Kev :)
