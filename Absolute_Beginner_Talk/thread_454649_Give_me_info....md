### Post by starcraft.man on 2007-05-25
Themes for [Ubuntu]("http://www.gnome-look.org/") can be found here.

GDM is a theme for your login screen.
GTK 2.0 is a theme for the whole look of your OS (Fonts, icons, windows, panels, everything).
X11 cursor, is only mouse themes (need to install gcursor via apt to use them).
Metacity is a window theme that can be used over whatever you select with GTK.
The rest are individual parts of GTK or another.
Beryl and Compiz themes are for the seperate window managers, unless you have them installed you don't want them.

If your running Kubuntu, go to kde look in the bottom left of links. GDM is changed to KDM for them, everything else is more or less the same.

As for sounds... go System > Preferences > Sound > Sounds Tab. Then pick any you like for the actions you want. I believe that is question answered, bye bye have fun customizing. You can import sounds too of your own choosing, I forget which folder they go in >.>.
