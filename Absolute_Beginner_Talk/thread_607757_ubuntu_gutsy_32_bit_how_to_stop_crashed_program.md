### Post by drs305 on 2007-11-09
One of the easiest is to add "Force Kill" to the panel. Right click on the panel, Add to Panel, and select Force Kill. It will put the icon on the panel. When a program freezes, click the icon, then click anywhere in the frozen program. A dialog box will appear asking if you want to kill the program. 

There are terminal commands as well but this GUI is probably the fastest/simplist.
