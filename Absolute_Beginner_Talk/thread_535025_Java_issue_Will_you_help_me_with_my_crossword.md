Don't know about the java, but your keyboard issue sounds like a keyboard that is, well just plain bad. Not a software issue. To test this, just open up a text editor and see if the arrow keys work. 

Keyboards, even for laptops are generally cheap, and easy to replace. Another thing you can do is try an external USB keyboard (you can borrow one from a friend).

I have a number of methods of fixing laptop keyboards--- sometimes, but the itty bitty scissor  lifts are so fragile, you can end up breaking them instead.
