### Post by Anonii on 2007-06-18
.exe files are for Windows. They _won't_ work in Linux. There are ways to emulate Windows programs, and run exe files, but that's a whole different story. 

DeSmuME comes out for Linux, too. So you will want to install the Linux version. To do that, follow this link:
[http://downloads.sourceforge.net/desmume/desmume_0.7.1-1_i386.deb?download](http://downloads.sourceforge.net/desmume/desmume_0.7.1-1_i386.deb?download)
You will get to download a .deb file. That's basicaly, an application installer for Ubuntu. Open it, after it downloads, and it should dosmemu should get installed. You will also get a menu entry in the games menu. If you get any errors (obviously, Linux and Ubuntu related, and not DeSmuME problems), please post them here.
