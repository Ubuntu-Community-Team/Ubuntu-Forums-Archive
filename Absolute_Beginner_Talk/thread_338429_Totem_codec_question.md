### Post by bruenig on 2007-01-14
That is not the only reason they don't include it, there are legal issues with some of these codecs. Also, the reason they don't have closed source installed by default is a philosophical one. Some closed source stuff they could include, but that goes against their philosophy. They don't want to force someone who is very anti closed source to have to install closed source if they want to use ubuntu.
