### Post by obrient on 2007-05-24
In regards to the Chinese translation go to System>Administration>Synaptic Package Manager and search for language-pack-gnome-zh and that should be the package you need.

GNOME translation updates for language Chinese
Translation data updates for all supported GNOME packages for:
Chinese

Please note that you should install language-support-zh
to get full support for this language.

Hope this helps.
