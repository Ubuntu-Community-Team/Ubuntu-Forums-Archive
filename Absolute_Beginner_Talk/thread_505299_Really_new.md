### Post by kiv on 2007-07-20
Alternatively you can access your BIOS by pressings the BIOS key at startup, this is usually either, DELETE, F2 or INSERT.

Once in your bios looks for boot priority options. Set your CD/DVD Drive to be the first boot device and your hard drive second. This way the computer will look for bootable media in your drive before booting from the hard drive. 

Once you've done this the Ubuntu  Live Cd menu will appear, run the live cd and then install by clicking install on the desktop.

Enjoy!!
