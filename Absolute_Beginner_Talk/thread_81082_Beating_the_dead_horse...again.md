### Post by blueturtl on 2005-10-23
> I mainly use my pc for gaming, web browsing, and e-mail. I don't do very much video editing or cd ripping and very little as far as music is concerned. With that said, here is a few questions of the top of my head.

Web and email are fine, but if you're at all serious about gaming - I wouldn't jump straight into Linux. Despite some of us getting lucky - you really can't expect your games to work. Not even those that work in some cases. If you have purchased Windows games be prepared to let go of a few, and tweak the rest like hell if you want them to work. That said, there's nothing in the realm of web or email that you couldn't effortlessly do in Ubuntu.

> 1.) I've tried Ubuntu and am currently downloading Kbuntu to get a feel for the differences between the two. If I understand it correctly, they are basically the same thing, just different interfaces...Is that the gist of the differences? Will I be able to get the same updates with Kbuntu as you do with Ubuntu?

Kubuntu is the same as Ubuntu but with KDE as the default desktop manager (Ubuntu uses Gnome). Gnome focuses on simplicity, aesthetics and usability. KDE is a bit more overflown with visible features, but some prefer it over Gnome hence the project Kubuntu was born. You get the same Updates for both as Kubuntu is really just "Ubuntu with KDE installed as the default desktop environment".

> 2.) I use a firewall and am behind a linksys router. I also use adaware and an online AV scanner frequently to keep the nasties off my machine, but from what I have read on the forums, some say that you really don't need to run a firewall with Ubuntu. So without a firewall, will there still be a need to run AV and spyware checkers? Or does Firefox stop them from getting on the machine?

There is really no serious spyware / viruses for Linux. You pretty much have to, download one and install it by yourself to have your system infected. Malware and virus scanners are void. I never needed them in Windows and I don't think they really even exist for Linux. As for the firewall, if you're not going to be running some kind of service that requires you to handle traffic in a precise manner, you won't be needing one as all the ports are blocked by default in Ubuntu. I've had my system online for closer to four months now without a firewall. No problemo.

> 3.) And lastly, there isn't much info I would need to port over from my Windows install except emails that I'd like to keep (which I could re-mail to myself) and all of my email addresses and web links... Is there a way to install Ubuntu, import the Windows info, and then completely remove said Windows install?

I don't think Ubuntu offers such advanced import functions. It's just that there are so many potential different programs that might need import/export info. It's just not going to work. I think you can use your email program's spesific export function to export either emails or your address book into a file and then try importing that under Ubuntu, but even that depends a bit on your client.

> Again, I thank you for the info and can't wait to get going on the switch from Windows to Ubuntu. I just wish that I had found out about it earlier lol.

Well look at it this way: Ubuntu is a miracle to those who know their tech. Learn to use it and you will make the most of it. Most spyware and viruses are spread by unknowing users and as your computer's master you are responsible in the end. Ubuntu does make managing security fairly easy in the end, but it doesn't mean you shouldn't be responsible. I hope you enjoy your new system. ;)

### Post by davmac on 2005-10-23
I managed to get all of my emails from windows. I first installed Mozilla Thunderbird on windows, imported all of my stuff from Outlook Express, and then, once Ubuntu is installed copy the emails across. Just shout if you need more info about doing this.

If you're using Firefox on windows, and you really should be :), You can open the bookmark manager, export your bookmarks and save them somewhere. Once you get Ubuntu installed, open up the bookmarks manager in firefox and import this file and hey presto. You might also make use of the bookmark import/export facilities at [http://www.mybookmarks.com/](http://www.mybookmarks.com/) which I use to keep my bookmarks on my home machines in sync with those on my work machine, where I'm forced to used IE (bleeeeach).
