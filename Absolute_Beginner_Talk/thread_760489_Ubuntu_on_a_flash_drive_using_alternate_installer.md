### Post by george9233 on 2008-04-20
Alternate cd is very much like the live cd, but just without a graphical user interface.

You can find more [here]("https://help.ubuntu.com/community/GettingUbuntu#head-40b5bcdbc1d4ec7b8149519dfd4f08c7fa274559").

And it needs at least 4 GB of disk space as said [here]("https://help.ubuntu.com/community/Installation/SystemRequirements").

Full installation details can be found [here]("http://www.pendrivelinux.com/2008/04/14/ubuntu-804-usb-hard-drive-install/").

### Post by Sef on 2008-04-20
> Alternate cd is very much like the live cd, but just without a graphical user interface.

If you are going to use the whole disk, then the alternate cd is very easy to use.   The only hard part would be manually partitioning if you are using only part of the drive.
