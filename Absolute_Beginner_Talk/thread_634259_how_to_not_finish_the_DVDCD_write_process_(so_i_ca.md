### Post by thelatinist on 2007-12-07
It's different in different programs.  In K3B, right-click on the project and choose "properties."  On the "Misc" tab there is an option to set "Multisession Mode."  You have to set it to "Start Multisession" for the first time you write to a disk or "Continue Multisession" to add data.  You probably won't be able to use the DVD without finalizing it during your last burn for that disk using "Finish Multisession."
