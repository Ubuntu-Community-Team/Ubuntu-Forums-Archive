### Post by justleen on 2007-04-03
Yes, the programs are interchange able. The Ebuntu comes with some educationals applications installed, but you can install those on Ubuntu as well.

They are listed by categorie, so you can go to the Edu. section, and install what you like (or what your daughter likes ;) )

### Post by Obor on 2007-04-03
You install them the same way as any other application in ubuntu. There are few different ways. 
1. Add/remove in you applications menu
2. Synaptic package manager in system - administration - synaptic
3. in terminal by typing sudo aptitude install name_of_the_program

more here
