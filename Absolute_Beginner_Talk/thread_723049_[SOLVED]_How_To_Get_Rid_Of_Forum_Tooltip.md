### Post by angry_johnnie on 2008-03-13
Is swiftweasel some sort of modified firefox?  If so, this might help you.  Type   "about:config"   in the address bar.  Then go to "filter"  and type  "tip."   You'll then be presented with some options.  One of them says  "browser.chrome.toolbar_tips"  which is set to "true"  by default.  Double click on it to set it to "false"  and you'll get rid of tooltips.
