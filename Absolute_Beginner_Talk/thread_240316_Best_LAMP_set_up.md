### Post by garner_nc on 2006-08-20
I don't mean this in a negative way but, if you want a real learning experience, install all MySQL, Apache, and PHP form source. Learning how to get this all configured and working together is a GREAT skill to have.

 Also,I installed MySQl and PHP via Synaptic originally. However a later upgrade completely fried my installation to the point I couldn't just un-install and re-install via Synaptic. Now that I install and upgrade with source it all works, I avoid the upgrade problem, and it all works the way I want it to.

### Post by stormblue on 2006-08-20
You have a number of opitions:

The first one is to use a webhost such as [Host Gator]("http://www.anrdoezrs.net/click-1877237-10408505").  I use them and they are very helpful.  They've helped me with an obscure compatibility problem between lftp and their ftp server.  This will let you host your website(s) 24/7 without having to have your computer on.  This is one one to go.

The second way to go is to run a LAMP server on your desktop, which has to be turned on for you to access it.  You could leave it running 24/7 as well.  You can install a package such as Xampp or you can install the LAMP stack. For this option with some configuration you can also get a domain name from no-ip.com or similar service that will make it easier to connect to.  For example you can get a domain name like whatver.zapto.org.
