### Post by nutz on 2007-06-19
> **zach12 said:**
> it's a dell

On Dell machines the F12 key will bring up the one time boot menu to allow you to boot to a different device for that one session. It reverts to the default device as soon as you reboot...

If your Bios supports booting to a flash drive then you should see it listed there. But you will need the device connected when you power the PC on in order for the one time boot menu to detect it and offer it as a boot option.

I run Ubuntu from an external HD on my D620 this way...
