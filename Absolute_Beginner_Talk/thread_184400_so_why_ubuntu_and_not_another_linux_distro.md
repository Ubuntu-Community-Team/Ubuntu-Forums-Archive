All the tools are there in the background if and when you need them.[/QUOTE] I'm on that wavelength.

I get confused when people say Ubuntu is a good "newbie distro." Most "newbies" I know are point-and-click people and get frustrated when they need to use the command-line or edit configuration files to set things up. For those people, Mepis or PCLinuxOS are better choices.

Ubuntu is a perfect intermediate distro. It doesn't require you to know as much about Linux as Gentoo or Slackware require you to know, but it gives you just enough point-and-click stuff while also acquainting you with the command-line.
