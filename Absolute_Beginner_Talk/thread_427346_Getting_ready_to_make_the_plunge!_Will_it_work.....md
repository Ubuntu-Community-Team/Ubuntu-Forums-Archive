Try the LiveCD. If it works, then it'll work when you install it. The 64 bit versions of Ubuntu are still a bit buggy, if I remember correctly, but the last time I checked was a while ago so things may have improved.

Wine works just like Windows (sometimes even better) on many programs, and then falls on its face with others. Best way to be sure is just to try it out and see. You can also check how well your desired programs work in wine by checking the [Wine Application Database](http://appdb.winehq.com/). However, don't misinterpret no results for bad results - people tend to only report there when things go wrong, it may be that if your program is missing, then that's a good sign. Wine is also updated pretty regularly, so even if your program has bad results, it's well worth checking anyway, just to be sure the situation hasn't gotten better.

Even if Wine doesn't work for you, you can still just dual boot with Windows, so you get to keep those programs. You could even just virtualise Windows so you can run it through Linux - but it may not be a perfect solution.
