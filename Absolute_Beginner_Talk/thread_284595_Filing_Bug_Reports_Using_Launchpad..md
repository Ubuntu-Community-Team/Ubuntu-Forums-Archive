### Post by John.Michael.Kane on 2006-10-25
**This will outline the Filing of bug reports using launchpad.**

**Step1:** Make Note of all error messages shown on screen,and inside your /var/crash folder.

Also make note of the program including version number's.

[COLOR="Red"]**Note: Should you not know the program causing the issue/s**[/COLOR]

**Option 1:** Try posting the problem with the error in the appropriate section of the forums making sure to use code# tags to see if you can get the program named by other members or even a work around/fix posted.
[B]
Option 2:[/B] Post you report selecting "I don't know" in regards to the program causing the issue.
[COLOR="Red"][B]Note: This method may not get your report seen to as quickly thus slow you getting your bug fixed.
[/B][/COLOR]
**Step2:** Sign up for a user name at [https://launchpad.net/distros/ubuntu](https://launchpad.net/distros/ubuntu) [B]click on log in/register 
[/B]
**Step3:** Search to make sure a report on your issue/s has not been open already
[COLOR="Red"][B]Note: Should there be an open report on your issue add you error messages/issues,and try to get the bug confirmed.
[/B][/COLOR]
**Step4:** Should no report be open you would file your report for the application or issue in question at this link [https://launchpad.net/distros/ubuntu/+filebug](https://launchpad.net/distros/ubuntu/+filebug)

**Step5:** Post the link in your ubuntu forums thread so that others with the same issues can add to your report,and getting it confirmed, and fixed.

Note: You will receive emails giving the status of your report,and if others have added to it.
