### Post by treesurf on 2007-06-12
I have an NX570X that I bought back in January.  So far I am very happy with it.  Only two points of contention:  the microphone jack will not work, and the brightess function keys are a bit wonky.

I have heard that the newer versions of this series are coming with the newer Intel 4965 wireless.  I don't know if there are Linux drivers for this yet, so you may want to check on that.

I'd also check out the laptops that Dell is now selling with Ubuntu installed.  Guaranteed to work with Linux and they're pretty reasonably priced as well.
