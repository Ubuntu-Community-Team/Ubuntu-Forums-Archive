### Post by Can+~ on 2008-04-19
You can enter a preview mode on the CD when you boot from it, it will copy all that needs to the ram and leave your hard disk untouched, on layman terms, a desktop "on the fly". Also, I suggest you to install when the latest version of ubuntu comes out (on April 24, 5 days).

Your problem about vista starting first, is that your boot order is set to start the harddisk first, then the CD/DVD reader. Enter the bios (depends on the model, try F12, Supr, anyone with a toshiba laptop knows?), and change the boot order to set the CD first, which is not a difficult task.
