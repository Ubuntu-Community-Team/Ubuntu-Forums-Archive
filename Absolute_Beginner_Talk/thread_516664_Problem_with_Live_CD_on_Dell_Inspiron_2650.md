### Post by Ek0nomik on 2007-08-03
You are dropped into a shell?  Meaning, you get a command prompt so you can type things in?

You may want to check out the Alternate CD to install Ubuntu.  The Live CD worked fine for me on Ubuntu Edgy, but once Feisty came out, the Live CD's no longer worked with my hardware.  I used the Alternate CD's to install instead and it worked perfectly.

[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

Below the green 'Start Download' button you can check a box to get the Alternate CD instead.
