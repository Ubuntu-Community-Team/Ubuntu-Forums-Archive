### Post by Celegorm on 2007-06-28
You'll need to go into your computer's BIOS and set the cd drive to boot before the hard drive. Usually when your computer is booting up, it will tell you which key to press to go into the BIOS menu. Once this is done, simply put the Ubuntu cd in the drive and reboot the computer. It will then boot up into a live version of Ubuntu (unless you are using the alternate cd) which will give you a taste of what Ubuntu is like, and also allow you to check if it works on your hardware. It will run much slower than an actual install, however. Once you are ready, just double click on the install icon, and it will guide you through the installation.

Also, if you haven't already, you should verify your download using the md5 check sum (prior to burning to cd), as a bad download will definitely give you troubles.
