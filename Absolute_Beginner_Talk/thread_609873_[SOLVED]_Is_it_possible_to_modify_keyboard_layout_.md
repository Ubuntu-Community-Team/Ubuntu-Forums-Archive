I was wondering if it was possible to modify the keyboard layout because I am from the Faroes and we use this letter "ð/Ð"

### Post by Paul820 on 2007-11-11
System->Preferences->Keyboard for the layout.
System->Preferences->Keyboard Preferences for mapping the windows key.

### Post by geirha on 2007-11-11
Does the keyboard have a Ð key? 

Have you set the farao islands keyboard layout in system->preferences->keyboard?

In most layouts you can get those letters by typing ALT Gr+d and ALT Gr+SHIFT+d.

ALT Gr+Right Shift seems to go into some sort of weird mode though, right CTRL turns it off again. It's probably some accessability feature of some sort. Using the left shift seems to work fine.
