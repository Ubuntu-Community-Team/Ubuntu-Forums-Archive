### Post by chriscando on 2006-05-02
Hopefully you wont need to install a driver to get it working in Linux. You said that you recently installed Ubuntu, can you tell if it is already working by going to System > Administration > Networking > Configure. See if it lists a Wireless connection.

Unlike windows, I find that most things just work in Linux without installing more drivers.
