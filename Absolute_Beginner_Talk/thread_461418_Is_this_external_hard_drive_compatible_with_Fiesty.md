### Post by Ek0nomik on 2007-06-01
Of course it works with Feisty!

If it comes formatted as an NTFS filesystem, you can either download support for it or reformat it to a FAT32 filesystem, which is supported out of the box in Linux.

It will definitely work.  I myself use a MyBook 500GB on my Desktop which uses Feisty Fawn.  ;)
