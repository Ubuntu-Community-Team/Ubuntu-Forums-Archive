### Post by erginemr on 2007-11-15
Hello, 

The wallpapers are under:
/usr/share/backgrounds

With a Live CD and a usb memory stick, you can easily copy them when you boot the Live CD to the pen drive. You can then boot into the real Ubuntu and copy the pictures in the pen drive to the correct folder above. Of course, you will need the root permissions.

I will not give you the commands because you are not a newbie, I am sure you can find your way.

### Post by erginemr on 2007-11-15
Stop, I have found a better way!

Even if you remove the background from the list, the actual file is kept intact (file permissions again). So, all you need to do is click on Add..., find the folder /usr/share/backgrounds, add the wallpaper, and that's all there to it!

Good Luck.
