### Post by abn91c on 2007-06-01
if you have windows XP and wubi providing your card is working and you have internet access, just run the setup. No Cd is needed, wubi will download the requierd iso for the install and proced with the install. make sure you defrag windows XP first. Also when you get to the reboot part, allow the PC to do it by itself(be patient) DO NOT use the reset button or manually reboot, this may cause the ubuntu installation to fail. After the PC reboots, select Ubuntu from the boot screen and follow the easy setup instructions. You may also need to go to windows xp, my computer, properties, hardware,  hardware profiles set the time to 30 seconds. Good Luck
