### Post by Hospadar on 2008-04-10
well those packages are probably only designed to put shortcuts in the gnome start menus.  You could either: make your own shortcuts, or, use the gnome panels so you can just have a gnome start menu.  Try searching in synaptic for the package that has the gnome panels and the panel applets, and maybe google "manually start gnome panels" and I'd bet you could find how to start them up outside of gnome.

The problem is that different wm's will have all kinds of different start menu setups, so depending on how they set it up, it may not be easy to generically add a start menu.  At least in gnome, you won't need to reboot to see things in the start menu.

Also note too that a lot of apps may not have a start menu item.  For most command-line apps, this would be a pretty meaningless feature.
