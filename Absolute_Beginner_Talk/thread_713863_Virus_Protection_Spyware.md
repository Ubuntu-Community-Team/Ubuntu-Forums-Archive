### Post by lswest on 2008-03-03
are you talking about virus protection and malware in Linux? if so, you can pretty much just forget about it, there are hardly any linux-viruses and none of them are wide-spread if in any public area whatsoever.  There is virus detection available, but only for windows viruses, if dual-booting with windows or if you're sending/sharing files with windows computers.  Feel free to ask any more questions, but rule of thumb: linux doesn't need virus protection or malware scanners.
