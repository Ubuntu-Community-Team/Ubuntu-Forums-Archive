### Post by BobSongs on 2006-06-05
It's a feature that I personally don't like. Especially when I accidently leave the pointer over a stray file on the desktop and the speaker volume is too loud.

I'm not sure which destktop environment you're running. For Ubuntu switching it on is like this:

Start Nautilus

Places -> Home Folder

Edit -> Preferences -> Preview Tab -> Sound Files: Preview Sound Files.

Hope it works.

Gnome has this option. In a Nautilus window, go to Edit > Prefrences. Click on the "Preview" tab. Under "Sound Files" select always.
