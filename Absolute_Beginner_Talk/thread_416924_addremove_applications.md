Sounds like it's frozen for some reason. When this happens, you can just try leaving it for a while and seeing if it comes back to life. If you just want to close the application, you can use the System Monitor (it should be in System > Administration), right clicking the process, and selecting kill. I would personally just leave it alone for a while.

Did the password box pop up? If not, then the problem could lie here. Check whether the box is hidden behind other windows (although it shouldn't be) - you need to input your password before new software can be installed.
