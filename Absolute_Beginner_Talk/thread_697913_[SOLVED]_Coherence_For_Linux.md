### Post by macogw on 2008-02-15
Crossover will run certain Windows programs.

Virtualbox is a free virtual machine like VMWare or Parallels, and it *can* do seamless mode just like Parallels.  There's an open source version in the repositories named virtualbox-ose, but it doesn't have USB support.  If you need USB support, go to [http://virtualbox.org](http://virtualbox.org) and download their regular closed source version.  It's free for personal/home use.
