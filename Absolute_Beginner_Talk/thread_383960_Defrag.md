### Post by Adam_GUI on 2007-03-13
You don't need to defrag Ubuntu.  ext3 is a tidy file system.

But, if you want to do maintenance on Ubuntu, you can run fsck from the live cd.
DO NOT MOUNT THE HARD DRIVE YOU ARE RUNNING fsck ON!!!!!

fsck is run automatically run every 28 mounts anyhow, so don't sweat it.

For other file system information you'd use mkfs.

For more info, open a terminal
Applications > Accessories > Terminal
and type: mkfs -help

But you don't really need to do that either.
