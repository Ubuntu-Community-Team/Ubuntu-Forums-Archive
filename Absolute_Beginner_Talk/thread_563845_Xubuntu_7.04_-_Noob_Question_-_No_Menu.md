### Post by julian67 on 2007-09-30
It's a known bug with Xubuntu on certain hardware. Try this: alt+F2 to open a run dialog and type in there ```
xfce4-panel
``` and click run. You should now have a menu bar. Btw this problem is only present when running from CD, if you install to hard disk then all is OK.
