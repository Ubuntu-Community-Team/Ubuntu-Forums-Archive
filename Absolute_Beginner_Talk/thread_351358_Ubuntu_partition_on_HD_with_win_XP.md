Hi, 
when you install from the live cd it lets you decide with gnome partition manager (gparted) where to install ubuntu, it even let you resize existing partitions and than it establish a boot loader(grub) for ubuntu and including other existing OS's.

I read somewhere it is not the best idea to use eg. partition magic to prepare a linux install -- instead just use the shipped gnome tool for it and all should work.
