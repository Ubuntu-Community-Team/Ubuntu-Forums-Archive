### Post by macogw on 2007-10-25
Go to System > Preferences > Keyboard Shortcuts to set a keyboard shortcut for "home folder"  That'd open a window for you to browse.  Alternatively, if you're using Compiz, set one in its "Commands" section (in the general settings) with the command being "nautilus"

Some combination of Application Switcher and "Group and Tab Windows" in Compiz may work for the second one.  Incidentally, the fact that OSX switches whole apps at a time on Alt+tab (instead of keeping individual windows on alt+tab and putting apps on the standardly-means-nothing alt+`) annoyed me to know end when I was forced to use a Mac this summer.
