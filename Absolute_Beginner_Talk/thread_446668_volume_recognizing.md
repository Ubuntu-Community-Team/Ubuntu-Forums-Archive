### Post by Terl on 2007-05-17
Using synaptic, download ntfs3g and right underneath it ntfs3g-config.  Install these and then run the config tool.  It will be in the apps menu under system tools.  Remember to give write permissions when asked if you want to allow it.

This will also set icons on your desktops for the drives and connect everytime you log in.
