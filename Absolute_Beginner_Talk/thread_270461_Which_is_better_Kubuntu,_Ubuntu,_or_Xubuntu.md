### Post by lkagan on 2006-10-03
> Kubuntu is based on debian also correct?
All *buntu versions are based on debian.

Gnome seems to be the top distributions main choice (Red Hat, Ubuntu) as a default desktop so I use it solely for that reason.  However, Kopete kicks Gaim's butt as far as messengers go.  I can use Kopete or any other KDE apps on Gnome and vice versa, *buntu makes this very easy.  KDE's pim suite isn't quite as nice or feature rich as Evolution but I'm sure you can run Evolution on KDE.  KDE has always been known to have a bit more eye-candy, but it's negligible.  Xfce (Xubuntu) is very Windows 2000-ish.  I personally don't like it since since there's much better desktops available.  Xfce is generally for older hardware since it uses less resources.  But if I'm going for less resource usage, I'll use fluxbox or jwm on DSL (Damn Small Linux).  A good example of a modern day usage for DSL is within virutal machines like Virtual PC or VMware where hardware use is shared with the 'host' system.  By the way, DSL is also debian based.

Larry Kagan
