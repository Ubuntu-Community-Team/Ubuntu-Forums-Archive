### Post by mendingo84 on 2007-09-29
that 478 is the Socket. that is the pin configuration ( on the processor )  that fits on the motherboard. so the only configuration that matches with your motherboard is that one. so when you buy a new CPU you must make sure that it has the same pin configuration, this meaning that it must have the Socket 478 implemented, in your case..otherwise, if you want to change the CPU and the new one has another Socket, you also have to buy a new motherboard.

### Post by mivo on 2007-09-29
For the video card, you can look here:

[http://en.wikipedia.org/wiki/Comparison_of_NVIDIA_Graphics_Processing_Units](http://en.wikipedia.org/wiki/Comparison_of_NVIDIA_Graphics_Processing_Units)

You're chiefly looking for an increase of memory bandwidth and tact. Decide  how much money you want to spend and what the best card is that you can buy with it. :) A 6600 would do nicely -- already 2-3 years old, so it should be inexpensive by now. "More memory" is not always better, also depends on the type of memory and tact/bandwidth. If you need an AGP card, the choices are more limited.
