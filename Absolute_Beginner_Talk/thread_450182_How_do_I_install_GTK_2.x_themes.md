### Post by mcduck on 2007-05-21
Just go to System/Preferences/Themes and drag&drop the package you downloaded to that window.

If there are many themes in one package you need to extract them by hand into ~/.themes (a hidden directory inside your home dir)

### Post by mcduck on 2007-05-21
GTK2- and Metacity-themes go to ~/.themes, the rest (menu icon, panel backgrounds etc.) go anywhere you wish to keep them (and you need to configure them by hand, as they aen't part of the GTK2-theme).

You can just drag any image to your panel to set it as panel background image.
