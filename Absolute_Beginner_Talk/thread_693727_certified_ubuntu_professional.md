### Post by jaytek13 on 2008-02-11
At the core the certifications are essentially be the same for any Linux distribution. Aside from package management and some GUI differences (which generally aren't addresses on these tests), and perhaps some differences in the locations of the configuration files, any Linux certification would serve the purpose. Red Hat offers some free practice exams for RHCE, and they give a pretty good idea of what would be required to pass the test.
