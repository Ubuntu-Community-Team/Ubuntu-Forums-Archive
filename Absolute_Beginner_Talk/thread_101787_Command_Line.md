[QUOTE=Gary Lambert]How do I get a command line?  I am trying to use Wine for window apps.[/QUOTE]
For an Ubuntu command line: Applications -> Accessories -> Terminal

For a Windows command line: Start -> Run -> type "cmd.exe" or "command.exe" if that doesn't work.

### Post by Qrk on 2005-12-10
Wine treats a windows .exe as a document, not a program. So you can't use the basic program name for a wine'd windows app. Instead, do 

wine program.exe
