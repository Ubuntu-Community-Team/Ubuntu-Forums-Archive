Welcome!

The file you downloaded is an .iso, this means that putting it onto a CD media needs to be done as  'burn *image*'. This is not the same as the usual file-onto-cd operation, which is done in iso format, and created at the time of burning. The ready made .iso file has to be mirrored to the CD as it is, with no extra packaging by your burner.

So - check that you are using a 'burn as image' or similar burner function, and not your usual one.

Also, you do not mention how your external drive gets connected to your PC, I am guessing it is USB? Your PC is controlled by the BIOS in how it will be controlled at boot. If you want to boot from a live CD in your internal CD drive, you will need to have the bios set to allow machine booting from the CD drive. Sometimes this is already set ok, sometimes not. However, if you want to boot from a USB connected device, it is likely that the BIOS of the PC will need to allow this and it has to happen before the hard drive boots of course. 

If you downloaded a live CD then it is quite safe to boot in your internal CD drive, it will run fully (if a little slowly) but not change your PC, unless of course you ask it to install.
