### Post by CatKiller on 2006-11-25
You can make it so that nautilus doesn't send things the the Wastebasket (Edit -> Preferences -> Behaviour -> "Include a Delete command that bypasses Wastebasket") which should stop it happening. Other than that, I don't know of any.
