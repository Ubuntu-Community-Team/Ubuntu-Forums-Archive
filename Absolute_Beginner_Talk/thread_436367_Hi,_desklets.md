### Post by audax321 on 2007-05-07
To install the desklet you have to open up the gDesklets program. Should be under Accessories in the menu. If not, open up a terminal and type 'gdesklets'

Then once the window pops up (or if it doesn't, look in the system tray for an icon and click on it to bring up the window) drag the desklet file you downloaded onto the program window and a prompt should pop up asking if you want to install it. Otherwise, there should be something in the program's menu to the tune of 'install package' or something.

Hope that helps.. (I'm using version 0.35.3, so if these directions make no sense, they may have changed something in a later version).
