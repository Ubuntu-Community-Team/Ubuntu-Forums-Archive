### Post by MichaelSM on 2008-01-29
Having made a fool of myself with a previous post ...try again!

There is a new version of cairo-dock on the web. If I managed to find it, you can. It's dated January this year.

So don't use the cairo-dock in Synaptic unless you've done a lot of updates.

Download the .debs from Berlioz (I think) which  has the main program plus plugins.

If you've already installed xcompmgr from repos, uninstall it.

I just wrote, in terminal: 'sudo apt-get remove xcompmgr' without the inverted commas. Enter password.

Back to Desktop. 

Download the 'new' xcompmgr deb file I've attached at the bottom of this post and save it to Desktop. 

Restart your computer.

In order ..install cairo-dock first (right-click>Gdebi installer), then the cairo-dock plugins the same way, followed by the xcompmgr deb file you got from me (and a lot of other people!)

Restart again. (This may not be necessary)\

Anyway, on Desktop, hit Alt+F2. A box will open. Type in   cairo-dock   then hit Run.

Same again. Alt+F2. Type in   xcompmgr  then hit Run again.

Assuming your graphics card drivers are up to date, you should have a nice little dock at bottom middle-screen. If it has a black surround, it means that your card drivers aren't updated.

I'm happily running Cairo-dock on a 450 mhz PIII with a 16meg Nvidia card and Envy's latest driver download. It's fully functional with transparency.

There are lots of things which can go wrong.

But that's the basics.

From beginner to beginner.

Mike.
