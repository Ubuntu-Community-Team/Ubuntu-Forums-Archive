### Post by drdnl on 2007-05-09
open beryl-manager -> General Options -> Main -> browse down for -> Horizontal Virtual Size -> set to 4

Also, make sure the vertical virtual size is set to 1 and number of desktops is 1. I know this answer is brief but it should be enough to fix your problem.
