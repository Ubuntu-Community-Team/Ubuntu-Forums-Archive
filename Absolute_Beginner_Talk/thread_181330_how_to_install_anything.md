### Post by Phlosten on 2006-05-24
If you are entirely new to the world of Linux and computers you may want to see if you can seek out local help from a Linux User Group or similar.

First thing to do to install Linux is download an install cd image or order one from shipit.
[http://www.ubuntu.com/download](http://www.ubuntu.com/download)
[https://shipit.ubuntu.com](https://shipit.ubuntu.com)
Boot it up and install. If you need to retain a currently installed operating system you will need some more advanced help.
