### Post by Ecthelion on 2006-12-17
> I changed the display fonts in ubuntu, but now I want to change them back to their default settings. What are the default font settings in Ubuntu Dapper?

I don't know, but I see there's a font dir: /home/username/.fonts
I wouldn't be surprised to see that it is there that your font settings are saved.

What you could try is to change the name of that dir (so your system doesn't find it, but that you can change it back if there's a problem), restart your pc and see if the fonts are changed (to whatever the defaults are).

When you remove one of those dirs, it gets (often, not always, don't start deleting all those dirs) created back with the default options.
