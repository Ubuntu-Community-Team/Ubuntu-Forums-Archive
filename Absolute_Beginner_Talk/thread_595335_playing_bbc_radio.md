I'm able to listen to the bbc movies and whatnot with the xine backend totem.  I'm not sure if i'm using the real player stream or the windows stream, but you can try either one ( if you click preferences in the video player it will give you an option)

So the things to do would be to install some extra packages.  first, install "totem-xine" this is the same movie player, but it uses different codecs.  Next, go to medibuntu's website (you can google them) and add their repositories, and install the needed stuff from there.  I'm not sure exactly what packages you'll need from them, but there arn't that many so you can always just install them all if in doubt. (I know you need "w32codecs" and i think there are some others for real)
