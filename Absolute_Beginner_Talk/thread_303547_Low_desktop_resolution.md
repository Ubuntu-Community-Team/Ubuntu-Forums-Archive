### Post by dbott67 on 2006-11-20
Not a video card issue... we've got the exact same card! :)

You could just change the xorg.conf file.  Everywhere you see **1152x864** change it to **1600x1200** (look at my xorg.conf file for reference.

After that, you may want to install the ATI binaries (aka fglrx drivers):
```

### Post by CatKiller on 2006-11-20
You could try to put the extra resolutions in the Modes sub-section of you Screen section. Also, in the Monitor section, you can put ```
    HorizSync   30-70
    VertRefresh    50-160
``` - this is sometimes necessary if the graphics card isn't properly reporting the monitor information. I got those values from [this page]("http://support.dell.com/support/edocs/monitors/e773c/En/specs.htm"), so if you think they're wrong please put values for your monitor in instead.

Also, dbott67, you could consider chaging your values to ```
    HorizSync   30-81
    VertRefresh    56-76
``` I got those values from a [similar source]("http://support.dell.com/support/edocs/monitors/2007FP/en/about.htm#Specifiications"). I don't know if it will actually help anything for you, though.

EDIT: You don't need to change the resolution values, you can just add more in. X will go through them in turn and select the first one that is successful.

### Post by azkehmm on 2006-12-10
Ok, I got this fixed now.

It turned out I needed to change the device section from this:
```

Section "Device"
        Identifier      "ATI Technologies, Inc. RV350 AP [Radeon 9600]"
        Driver          "ati"
        BusID           "PCI:1:0:0"
EndSection

```
to this:
```

Section "Device"
        Identifier      "ATI Technologies, Inc. RV350 AP [Radeon 9600]"
        Driver          "radeon"
        BusID           "PCI:1:0:0"
EndSection

```

and add 1280x960 to the screen resolutions. 
 Just figured this might be usefull for other people having a radeon card :)
