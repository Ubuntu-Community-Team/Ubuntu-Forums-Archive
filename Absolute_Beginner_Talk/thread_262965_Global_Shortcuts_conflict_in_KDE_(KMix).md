### Post by jordanmthomas on 2006-09-22
Right click on Kmix in your tray and go to select master channel
Choose the one you want your volume controls to errr...control.

If you have your keyboard model selected correctly (kcontrol --> Regional & Accessibility --> Keyboard Layout) the buttons should control this master channel selected in Kmix and you shouldn't have to set the buttons manually.

Post back with the results, and we can see what else can be done if this doesn't work.
