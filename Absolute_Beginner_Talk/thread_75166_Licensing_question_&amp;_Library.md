### Post by mostwanted on 2005-10-13
Of course it would be okay :)

As an end user, you're pretty much free to do whatever you want with GPLed software.

As a programmer, you can only publicise programs derived from the source code if they too are GPLed or under a similar license. That's the real "limitation" of the GPL; It prevents GPLed code from ever becoming proprietary.
