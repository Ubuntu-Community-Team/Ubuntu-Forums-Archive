### Post by Strid on 2006-08-09
This is not a bug fix, but a way to get around that bug.

If you go the the "Help" panel and select "About Azureus", then an about-azureus-window pops up. With this window open, the buttons on the annoying 'Warning' window work, so you can now hide the warning properly.

Cheers.
