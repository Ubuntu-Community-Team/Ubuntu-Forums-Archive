### Post by impeachw on 2006-07-30
If you want a dual boot, don,t even go here. But assuming you were using windows, now want to clean install Ububtu, at startup go into setup, esc, f1, f2, del, or whatever gets you there and find "reset bios defaults". Ususlly an f# like f7. Hit it, and then go to boot and select device, cd first, hd second. F10 save and exit, with cd in drive. If you have more than 1 cd or dvd drive disconnect the other first. Make sure the cd is secondary master. My Linux experience is very limited, but I've done many xp installs on used computers for low income students, and I found that sometimes for machine happiness you just have to start at the beginning. If that doesn't work and you want a clean install (recommended) get a win 98 or me setup floppy and fdisk the hd. Then install. If a hd fdisk's it's probably ok. If not, probably trash. I fdisk all my hd's and then run dban as a matter of course, before any install. And no one brings one back, but this is probably overkill for you.  :cool:
