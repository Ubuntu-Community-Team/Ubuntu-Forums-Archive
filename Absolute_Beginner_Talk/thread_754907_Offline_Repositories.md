### Post by EXCiD3 on 2008-04-14
Offline repositories are all the packages for the software bundled in your specific distro (ubuntu for example). Instead of downloading the packages off a server online you have a set of CD/DVD's or a server at home that you can install packages from.

You can find the package list at [http://packages.ubuntu.com](http://packages.ubuntu.com)

APTonCD is a tool you can use to create a limited offline repository so that you can burn packages onto a DVD and install from there.
