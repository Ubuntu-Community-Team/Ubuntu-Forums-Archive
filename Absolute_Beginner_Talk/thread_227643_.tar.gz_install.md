### Post by mlind on 2006-08-02
You can install bittornado straight from **main** repository
```

sudo aptitude install bittornado bittornado-gui

```

To extract a gzipped tarball: tar xzvf *package.tar.gz*.

### Post by aysiu on 2006-08-02
Step #1 Delete your .tar.gz. It's absolutely unnecessary.

Step #2 [Make sure you have extra repositories enabled](http://www.psychocats.net/ubuntu/sources).

Step #3 Install BitTornado by pasting this single command in the terminal: ```
sudo aptitude update && sudo aptitude install bittornado-gui
```
