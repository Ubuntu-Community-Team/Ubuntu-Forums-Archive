### Post by santiagoward2000 on 2008-03-22
> **VyperX said:**
> Thanks, how do I use the ''cube''? (the one that has 4 sides)

Hi!
First, make sure you have **compizconfig-settings-manager**. If not, look for it on Synaptics or open a terminal and type:
```
sudo apt-get install compizconfig-settings-manager
```
Then, open** Advanced Desktop Effects Settings**, go to **General Options** and under the **Desktop size** tab, set the **Horizontal virtual size** to 4. Finally, on the main menu, make sure the **Desktop cube** plugin is enabled.
Cheers!
