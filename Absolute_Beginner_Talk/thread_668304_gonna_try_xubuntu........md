### Post by tgalati4 on 2008-01-16
Linux Mint 4 XFCE uses XFCE4 and contains compiz fusion which is the mashup of compiz and beryl.  XFCE4 has compositing built-in which can be used independently of compiz.  So far, all of the effects that I have tried in compiz work.  Compiz is not turned on by default, you can run it from the terminal (compiz &) or activate it in the desktop settings.
