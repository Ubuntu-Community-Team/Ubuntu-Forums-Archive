### Post by Yoooder on 2007-06-27
There are a lot of solutions available, and I think gpartimage might be a good fit for your needs.  It is basically a partition-backup utility.  There is also a bootable CD utility called g4u, which works similar to ghost--however I'm unsure if it will work when copying from a 20gb to a 120gb drive, I've only ever used it on identical machines.

Research gpartimage and try to find a tutorial that matches your situation, mainly migrating a whole smaller drive's contents to a larger drive.

Regarding formatting, you probably will need to wipe the drive (or if you don't, the drive will be wiped when cloning).  In any event the data on it will be lost to some degree, so make sure you backup anything you need prior to performing the copy.
