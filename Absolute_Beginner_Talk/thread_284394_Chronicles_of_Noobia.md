Remember home users, your gateway (the DSL/Cable router) is not nessicarily your DNS server. in ubuntu, go to System>Administration>Networking to check your list of DNS servers. if you see a local address like 192.168.1.1 and another, external address (yours will be different, but mine was 65.220.x.x) you can go ahead and delete the 192.x.x.x one if you are only getting access to parts of the internet. apparently, ubuntu chooses one of these DNS servers based on.. well, i have no clue how it decides which DNS server to use, but if you have an invalid one in there, you'll get errors. :( good luck, fellow convertees!
