### Post by Jasper Houtman on 2006-07-10
If you want to test drive Ubuntu than you should just install it next to XP.
Just resize the XP partition to make room for ubuntu and let the install cd partition the free space for you and set up the boot menu.

As for support. Most of the things you want/need to know can be found on these forums  (there's probably a mexican forum aswell, try looking it up).
Just keep in mind that most problems you will come across have already been solved by other users who most likely will have posted the solution on these forums. So search the forums before posting a new thread.

As for a source of help, most of the members here will help you with any problem you might come across. I think you'll find the Ubuntu community a very friendly and helpfull one :)
