### Post by quietglow on 2005-11-15
I've used both of those, but you know what's the best (assuming you're running gnome--safe bet if you're using gftp)? Nautilus! Under "Places" choose "Connect to server" and then select which type of ftp (public or not) you're trying to use. Easy! And the best thing is that you stay with the same file manager you're used to. This will work for ssh, WebDav and others.
