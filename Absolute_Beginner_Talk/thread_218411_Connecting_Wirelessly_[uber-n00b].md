### Post by steve.horsley on 2006-07-18
Go for System->Administration->Networking. Look now to see if the wireless card is shown in the Connections tab. If not, you need to install drivers. If it is, then when you are in range of a base station, you should be able to see it in the wireless configuration - you get a drop-down list of stations to choose from.
