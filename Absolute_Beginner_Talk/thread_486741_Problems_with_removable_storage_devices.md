### Post by Inxsible on 2007-06-28
you can use pmount to mount your external hard drives.
 
If you do not already have pmount installed,
 
```
sudo aptitude install pmount
```
 
then use pmount like so.
 
```
sudo pmount /dev/Xd*? MOUNT_POINT
```
 
where X = s or h
* = a,b,c,d.......
? = 1,2,3,4.....
 
MOUNT_POINT could be any name of your choosing like Seagate, WD etc. pmount creates the folder under /media for you. so you should use a name which is not already a folder under /media
