### Post by msak007 on 2006-08-08
I agree in that this isn't a Windows forum, but I'll answer anyway. By nature Windows cannot have anything like Automatix because (almost) nothing is free, and there is no single source you can download apps from like the Ubuntu repos. My brother wrote a script to automate his XP installs to grab files from his file server. If you have a setup where you have all the apps you want installed on a local fileserver, either stored as EXEs or ISOs, then it is possible to write a script to automatically mount the ISOs or run the EXEs, install the apps with your registration info / keys, and move on to the next app. Very laborious and time consuming, and even then it would only be for personal use as nobody else (outside of your home network) would be able to use it. +1 for Ubuntu :)
