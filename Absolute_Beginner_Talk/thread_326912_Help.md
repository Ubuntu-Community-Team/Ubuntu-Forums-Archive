I also could not get onto the Internet at first. After doing the stages below, I got onto the Internet. If you would like to, try opening Firefox and typing "about:config"

A long list will appear in the browser. At the top of the browser is a filter box. Type in "ipv6".

You will see there will be one line in the main window with the title "network.dns.disableIPv6". Double click on the line. The text under the heading "value" will now say true.  Close all the browser windows and restart Firefox. You will now be able to use the Internet *fingures crossed*
