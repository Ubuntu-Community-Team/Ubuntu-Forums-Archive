### Post by p_quarles on 2007-10-02
While that computer certainly won't run Vista, the specs are really good for Linux. The RAM is fine, and 30 GB is actually still a lot of space if you lose the MS bloat. 

Of the major distros, Ubuntu and PCLinuxOS are the easiest to get up and running if you don't already have a lot of experience. I prefer Ubuntu, but nothing should stop you from giving both a try.

The only trouble you might run into is with the wireless. You didn't say what kind of card you have, and I'll warn you that certain cards can take a bit of work to get running. 

If you do find that you have performance issues after you've installed the system, you can always switch to a lighter window manager. Gnome and KDE can both be kind of doggish on older computers, so you should know that there are (easy-to-install) alternatives if you run into this kind of problem.
