Debian is a kind of Linux. It is an operating system that uses a Linux kernel (I think of it as the heart). Ubuntu is a fork of Debian that is very close to the original.

I don't think Ubuntu has its own books yet, as it is pretty new. But Debian sources will work great as they are close enough.
