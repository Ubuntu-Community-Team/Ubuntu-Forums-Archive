### Post by caspian on 2006-10-22
Hi, steven8.

We're all very glad to hear your excitement about Ubuntu!

As for importing your bookmarks into Firefox, boot into Windows, then visit this page:
[http://www.linkagogo.com/go/Convert?informat=aol&outformat=netscape](http://www.linkagogo.com/go/Convert?informat=aol&outformat=netscape)

Save the .html file to something like a USB disk. Than, boot into Ubuntu, and launch Firefox. In Firefox, go to Bookmarks > Organize Bookmarks... (or "manage bookmarks") > File > Import. Select the .html file you saved when you were in Windows.

If all goes well, Firefox will now reflect your AOL bookmarks.

Hope this helps!

~Caspian

### Post by nickpaton on 2006-10-24
> **steven8 said:**
> 
I do not know how to check the md5 sums.  I am running winxp sp2.

Download and install Fast Sum free edition from:

[http://www.fastsum.com/]("http://www.fastsum.com/")

Go back to the exact site where you downloaded ubuntu-6.06.1 and whether x86 or alternative, and find the section md5 sums or checksums and copy all of the contents for the various versions into Notepad and save.

Run Fast Sum - File>Calculation Wizard>Browse to where you stored  your 6.06.1 download (make sure it's the only file in the folder)>Run (or OK).
It will now run through the 6.06.1 and generate a checksum file.

Open the folder via my Computer, and you will see an additional checksum file in there; open using notepad.
Open the original checksums you saved up above and compare the letter/number combinations which should be identical.

I ahve tried using the verification way, but doesn't seem to work for me, so do it this way.
