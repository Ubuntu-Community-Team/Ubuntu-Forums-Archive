### Post by drs305 on 2008-01-24
You need to get a cd from some source. You can order one although that will take some time or download it via another computer and burn it to cd. 

Read the psychocats info on burning the cd. You need to ensure you have the correct version for your machine and should ensure the download is verified using mdsum or some other verification method before you burn the cd. 

Since you will probably be using a windows machine for this, here is a link for md5sum for windows (how to use it and downloading it):
[http://www.pc-tools.net/win32/md5sums/](http://www.pc-tools.net/win32/md5sums/)

Finally, make sure you are burning the image as an ISO. When you are done, you should be able to see the files on the CD.

You can search this site for just about all the information you need.

We're here if you can't find the answers to your quesitons.
