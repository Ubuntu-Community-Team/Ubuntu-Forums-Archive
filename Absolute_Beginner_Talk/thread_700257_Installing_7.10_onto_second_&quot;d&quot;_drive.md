### Post by justleen on 2008-02-18
ubuntu doenst use "drive letters" as you are familiar with.
Instead is uses number do designate drive and partition.

For example, if you have one drive it wil (mos tlikely) call is drive HDA, and then number any partitions.
So C: would be hda1, D: would be hda2 etc
if D: is a physical drive, it will be hdb1

edit:  you could offcourse delete the D: partition from your windows install, then boot from the LiveCD and choose the option to use freespace..
