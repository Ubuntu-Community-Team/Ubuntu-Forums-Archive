### Post by mdsmedia on 2007-12-07
I've tried using GAIM for IRC before. GAIM is designed as an instantant messenger client and IRC is not instant messaging.

If you're new to IRC I'd definitely suggest using X-Chat which I believe is installed by default in Ubuntu. If not, install it using Terminal, as suggested above (Applications menu, Accessories, Terminal), or through Add/Remove Programs in your menus (Applications menu - "Add/Remove") or through the Synaptic Package Manager(System - Administration - Synaptic Package Manager).

Open X-Chat and it gives a dialog where you can input your choice of nicknames. Some will be "taken" so it gives you several choices. Select FreeNode from the Networks list.
