### Post by Bakerconspiracy on 2007-03-15
The keyring is what you can use to remember passwords for other programs and login information for GUI ssh sessions etc... basically you just have to enter one password for the keyring, and from then on in the rest of your login session it will automatically enter in the passwords you have the key ring configured for. This came is really handy managing a few websites. Pretty cool eh?

---

### Post by mcduck on 2007-03-15
Keyring is used to store passwords, keys for wireless networks and other network connections. It's automatically used when needed. You don't have to really do anything to use it. The manager is useful for changing these passwords and removing them if you don't need them any more.
