### Post by distroman on 2006-09-09
yes, basically you need privoxy and tor. Don't install tor from the repo, since it's outdated.
[http://tor.eff.org/](http://tor.eff.org/)

Install privoxy as you would any other app. 

You can either compile or
[http://tor.eff.org/docs/tor-doc-unix.html.en](http://tor.eff.org/docs/tor-doc-unix.html.en)
-you can use the repo from here
[http://tor.eff.org/download-unix.html.en](http://tor.eff.org/download-unix.html.en) 

If you install with packages, I think, tor will run as a service, nothing wrong with that, I guess. Otherwise you can simply run it with tor. 
Remember you'll need to configure privoxy, no matter what.

If it's for irc, which is the only sane thing, I can think of using it for, and you want to use x-chat, you'll need at least v. 2.4. which is as fare as I know, not available in the repo, so you'll need to have a look at there site. 
If any of this is available though automatix, I do not know.
