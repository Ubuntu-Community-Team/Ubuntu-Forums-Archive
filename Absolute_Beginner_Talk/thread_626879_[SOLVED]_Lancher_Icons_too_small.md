### Post by Bigbird999 on 2007-11-29
Noob here!  Just found it yesterday.  Right click on the launcher select "stretch icon and use the handles to increase the size  And if you want to change it click on properties then the icon and you will be taken to the pixmaps directory where you can select a different icon.
