### Post by uclalinux on 2007-12-28
I have PVR500, so it should be the same. I use Mythtv you can install it from synaptic, you will need to install the back and front ends. Once it is install you will need to set up the back end, this done all with a GUI under systems. You will need to select the device and some other small items, its fairly simply.  Once that is done you will need to setup the front end, when you start the front end you will need to select a device for an input device. If you don't do this step, when you go to watch TV an error message will return saying no input device, even thought the back end has told you you have done it correctly. I am sorry I so vaige, but I am not on my computer right now so I can look it up. there Are alot of good web sites out there to help.
