### Post by jken146 on 2008-01-11
First I would try installing VLC player (it seems to play nearly everything straight away).  Then install ubuntu-restricted-extras, which includes some codecs.  Then add the medibuntu repository and install the non-free codecs there if they're of any use).

Oh yeah, there is also helix player -- a real player clone or similar.  You might want to try that.
