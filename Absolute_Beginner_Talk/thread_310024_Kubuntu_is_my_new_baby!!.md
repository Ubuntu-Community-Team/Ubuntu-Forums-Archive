### Post by LLRNR on 2006-11-30
> **bxlinux said:**
> What is the difference between Kubuntu and Ubuntu? Is Kubuntu free as well?

Yes, every *ubuntu is free (Ubuntu, Kubuntu, Xubuntu, Edubuntu). While Ubuntu is the brown calm thing, Kubuntu is a little more blue by default :D Kidding.

Ubuntu uses Gnome as its desktop environment (DE), while Kubuntu uses KDE and Xubuntu uses XFCE. Each DE comes with its apps and specific things, but the best thing is that you can install each one of them on your system without losing the original DE you have. At login time, you can select your session type (Options -> Sessions).

If you want to know more on the differences between Gnome and KDE for instance (Ubuntu and Kubuntu), take a look [here](http://www.psychocats.net/ubuntu/kdegnome).

If you want to install KDE (kubuntu-desktop) look [here](http://www.psychocats.net/ubuntu/kde), if you want XFCE look [here](http://www.psychocats.net/ubuntu/xubuntu)...

And in general, look at [this guide](http://psychocats.net/ubuntu) for learning more stuff abount Ubuntu.

LLRNR

This house has been a Windoze free zone for the last year. I've got a few old boxes (and one newer one) and some old laptops and they all run Ubuntu Edgy. They're not fast necessarily, however they're run perfectly ok. This laptop (a P3 700 with 192mb RAM) runs everything I need without any hassle.

I have an XP VMware image that I keep for playing one game, Football Manager 2007. It hasn't been ported to Linux, and I haven't been able to run it under Wine. Having said that, even my slower laptops will run VMware server and some images (the ones that don't ask for much memory, e.g. Minix).

About KDE. Personally I prefer GNOME, however there are a few KDE apps that I always install since they're superior (IMHO) to the GNOME equivalents. Amarok, K3B, KTorrent immediately come to mind.
