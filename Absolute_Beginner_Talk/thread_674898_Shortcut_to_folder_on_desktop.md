### Post by Born-Thirsty on 2008-01-22
I use Kubuntu, but I am sure it is similar in Ubuntu.

Right-click on the desktop and (under the create new option) select link to location/url. Then browse to the folder you want the link to point to, say Open and give it a name. 

As easy as that. It worked for me.

### Post by b0rka7a on 2008-01-22
Drag`n`drop the folder to your desktop with pressed Alt.
Or create a launcher. If you want to make a shortcut of the folder ~/folder you will type for command:
```
nautilus ~/folder
```
Hope I helped.
