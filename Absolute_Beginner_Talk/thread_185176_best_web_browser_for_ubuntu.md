### Post by lkagan on 2006-05-31
Unfortunately, plugins are still not as easy on any linux distro as they are on Windows and Mac, but it seems that Firefox has come a long way.  One note, on Firefox 1.0x (Breezy) installing the Flash plugin from the repositories caused my browser to continuously crash.  That's when I discovered epiphany.  It has the Gecko rendering engine (Firefox, Netscape, Mozilla, Camino, bla bla bla).  It's a nice tool to get the job done and follows standards but Firefox is still my favorite.  By the way, to resolve the Flash problem, I got the plug-in directly from Macromedia.

Also, if you all haven't tried the 'Live Bookmark' feature of Firefox, I strongly suggest you check it out.  It's a news aggregator but as bookmarks.  More info can be found [here]("http://www.mozilla.com/firefox/livebookmarks.html").

Also, you can import those live bookmarks just like any other bookmark.  To try it out, go to my links page, download it via the 'download link for import' link and see for yourself.  See my links [here]("http://larrykagan.com/links").
