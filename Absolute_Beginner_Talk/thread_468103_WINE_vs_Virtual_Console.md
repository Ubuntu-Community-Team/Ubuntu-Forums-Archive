### Post by Ek0nomik on 2007-06-08
```
lspci
```

will output hardware for you.

You can also go to System in your preferences to see it.

If you want to run Quake 4, and other newer games, you will probably want the actual Windows O/S.  Wine and other programs like Cedega probably will be quite buggy, and they will slow down your gameplay quite a bit.  However, for older games, an emulator like Wine might work out alright.
