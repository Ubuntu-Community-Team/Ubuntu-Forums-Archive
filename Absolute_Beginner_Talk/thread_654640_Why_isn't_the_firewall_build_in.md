### Post by fiddledd on 2007-12-31
Ubuntu uses iptables, this means there are no open ports after you install. A good GUI for iptables is Firestarter, it's in the repositories.

---

### Post by Steveway on 2007-12-31
There is a Firewall built in, it's called IPtables.
If you really want to configure it then you can install Firestarter, it's a gui that makes configuring easier.
But a normal user doesn't need to do this, you only need to do this if you want to set up a server of any kind.
