### Post by scholzilla on 2007-08-15
To enable the universe repository:

System --> Administration --> Synaptic Package Manager

Then click Settings ---> repositories

and check universe.



You can install ndiswrapper from the same program. Simply use the search tool and type ndiswrapper and it will pop up and allow you to check a box to install it.

good luck!
