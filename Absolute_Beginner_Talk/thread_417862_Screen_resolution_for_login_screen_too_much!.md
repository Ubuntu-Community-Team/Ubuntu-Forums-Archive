### Post by leibowitz on 2007-04-22
You need to edit the /etc/X11/xorg.conf file by hand.

Go to the Applications menu > Accessories > Terminal, and paste this

```
gksudo gedit /etc/X11/xorg.conf
```

Find the line beginning with **Section "Screen"**, and check what is the number after ```
DefaultDepth
``` that is four or five lines down. Now you know what is your Default Depth. I will suppose you have the number "24" after DefaultDepth. Please adapt next step to yours.

Go down till you find this.
```
SubSection "Display"
           Depth           24
```

Please adapt to the number you see after DefaultDepth.

And now delete everything you don't want on the next line, that would be something like this.
```
Modes           "1280x1024" "1024x768" "800x600" "640x480"
```

In my case, I could delete "1280x1024" as I don't want to use this on my little screen (poor eyes).

This is my result:
```
Modes           "1024x768" "800x600" "640x480"
```

Now save the file and exit Gedit.

You can now log out, and press CTRL+ALT+BACKSPACE when you are logged out. This should reload your login manager.

### Post by R.Deckard on 2007-04-25
Apparently you have to manually set the screen resolutions for the ATI drivers separately.  This command seems to have fixed my problem:

```
sudo aticonfig --resolution=0,1280x1024,1024x768,800x600,640x480
```
