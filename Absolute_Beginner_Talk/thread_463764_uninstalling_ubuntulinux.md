### Post by Catsworth on 2007-06-04
If your box will boot from a Linux CD, in order to either run it as a live CD or to install it, then there's absolutely no reason why the same shouldn't be true of the Windows CD.

The only reason that CD might not work is if:

1.  The options are wrong in the BIOS.  This is unlikely because you say that other CDs boot, or
2.  There is some problem with the Windows CD that you are using.  ie.  that it is either not a bootable CD, or it's corrupted/damaged somehow.

Is this an original windows CD or a copy/something you've downloaded?  Is it an installation CD and not an upgrade CD?  Have you tried the CD on another machine (just to see if it boots from it that is).
