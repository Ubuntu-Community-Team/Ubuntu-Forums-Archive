### Post by rosegarden78 on 2008-01-25
You need a shareware program to mount the EXT2 or EXT3 file system.  I have another post with a link.  I'll check back when I find it.  Maybe try [www.tucows.com](www.tucows.com) for an ntfs-to-ext2 program to mount your Ubuntu partition from your Windows partition.

As far as running Ubuntu programs in Windows you'll need a compatibility layer or virtual computer.  You may check QEMU  for Windows if not try VMware.  Maybe a compatibility layer like WINE exists for Windows-to-Linux but I haven't heard of it.
