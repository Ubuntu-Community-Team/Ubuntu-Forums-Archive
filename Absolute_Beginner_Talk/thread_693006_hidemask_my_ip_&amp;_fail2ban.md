### Post by max renn on 2008-02-13
I guess you would need to elaborate on what type of file sharing you are interested in.

MoBlock is good to deny known bad file sharing client ips from accessing your computer.

There are privacy networks used for masking your identity while browsing, emailing.  There are various proxy servers used to hide your ip, for both browsing and file sharing.  Along these lines, you can setup a shell account.  There are plenty that don't require verification.  Some are free.
