### Post by igknighted on 2007-04-03
Did you read the installation instructions?  They should have came with it as a text file or are on the website you downloaded from.  In general, anything binary (.run, .bin, etc.) is run by ./filename.run, or /path/to/filename.run.  Note that if you cd to the directory it is in first, there needs to be a period (./), while if you use the path in the command there is no period.  There may be other stipulations (needs to be run in a specific place, needs dependencies, etc.), so always read the install instructions though.

### Post by Maestro23 on 2007-04-03
In terminal:

> chmod a+x filename.run

./filename.run



*Also, this is probably in the INSTALL/README doc as well.
