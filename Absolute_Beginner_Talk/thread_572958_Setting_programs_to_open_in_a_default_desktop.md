I was wondering if there is some way to make a program open, by default, on a certain desktop. For example: let's say I have 4 desktops set up, and always want Amarok to open on Desktop 3, and Evolution to always open on Desktop 4...is there a way to do this in Ubuntu? ### Post by bluenova on 2007-10-11
With KDE you can certainly do that, with the application open goto the window menu > Advanced > Special Application Settings > Desktop > Force > Desktop 1, 2, 3 ...

Gnome has very few window placement options, so I very much doubt you can do it there.
