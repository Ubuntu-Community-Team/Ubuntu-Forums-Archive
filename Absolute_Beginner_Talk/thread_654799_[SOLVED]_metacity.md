### Post by tgilber1 on 2007-12-31
Nautilus is the default file manager for Ubuntu Gnome.  With this file manager there are two modes, browser and spatial.  Browser mode has two frames, left frame is labeled "Places, which has a list of devices and trash.  Click on Ubuntu Help and Support for more information:  #System#Help and Support#  Search for spatial or click on the link below

file:///usr/share/gnome/help/user-guide/C/user-guide.xml#gosnautilus-FIG-naut-spatial-mode

If your help files are not installed, install the following:  sudo apt-get install ubuntu-docs

In any event, click on  the following to stop opening multiple windows with Nautilus.

[LIST=1]
[*]From the menu panel, click on #Places#Home Folder#
[*]Select #Edit#Preferences#
[*]Click the Behavior tab and select the checkbox labeled "Always open in browser windows"
[*]Click "Close" button and close Nautilus window
[*]Open Nautilus window again described in step 1
[/LIST]
