### Post by Metacarpal on 2007-04-29
The server installation does not have a GUI by default.  You can install one at any time by typing any one of the following:
```
sudo aptitude install ubuntu-desktop
sudo aptitude install kubuntu-desktop
sudo aptitude install xubuntu-desktop
```
(for Gnome, KDE, and XFCE, respectively)
