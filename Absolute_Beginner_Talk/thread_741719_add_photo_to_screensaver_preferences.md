### Post by ubuntu27 on 2008-04-01
Here is one way to do it. Open F-spot, the Photo Manager, import your photos, and mark your photos or pictures that you want to display on the screensaver by adding a tag called "Favorites"
You can use a slideshow of your photos as you screensaver. 

Then go to System -> Preferences -> Screensaver [The menu structure will be different if you are using the latest Ubuntu]

Select the F-Spot photos screensaver theme 


If you would like to change which tag is used for the slideshow: 
Open F-Spot 
Press Edit -> Preferences 
Under Screensaver, select the desired tag 
Press Close
