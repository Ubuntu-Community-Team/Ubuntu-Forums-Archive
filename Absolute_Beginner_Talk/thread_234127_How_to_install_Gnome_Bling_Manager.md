### Post by Jagot on 2006-08-11
Download the .deb:

[http://www.gnomefiles.org/download.php?soft_id=1395&where=https%3A%2F%2Fwiki.ubuntu.com%2FGnomeBlingManager%3Faction%3DAttachFile%26do%3Dget%26target%3Dgnome-bling-manager_0.1-1_i386.deb](http://www.gnomefiles.org/download.php?soft_id=1395&where=https%3A%2F%2Fwiki.ubuntu.com%2FGnomeBlingManager%3Faction%3DAttachFile%26do%3Dget%26target%3Dgnome-bling-manager_0.1-1_i386.deb)

Then you should just be able to double click on it and it will install.

Failing that, from Terminal:

```
sudo dpkg -i gnome-bling-manager_0.1-1_i386.deb
```
