### Post by Melonhead on 2006-10-31
> **cptjaben said:**
> the link says,

Open about:config search for browser.backspace_action change it to 0.

how do I open about:config, do I use Chromedit? orr.

Thanks.

You actually type 'about:config' in the address bar (it's like any other url, like [www.yahoo.com]("http://www.yahoo.com")).

Once that's loaded, type in browser.backspace_action in the filter box, and you should be able to find the field to change.

--Ryan
