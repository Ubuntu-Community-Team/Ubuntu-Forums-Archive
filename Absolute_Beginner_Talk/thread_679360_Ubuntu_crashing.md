### Post by pelish on 2008-01-27
I had similar problem with pidgin on fresh installation of Gusty. At first it worked perfectly, but then when i configured it like i always do (msn, xfire, gtalk + startup link + some plugins) it started freezing. 
So i started to do some testing, because i suspected it had to do something with startup+compiz fusion, but then i noticed that if i listen to music and then start pidgin that it doesn't crashes.
It was MusicTracker plugin (ver.:0.4.1), in fact all i had to do was change it's Player setting from Auto-detect to a player i use (in my case XMMS).
It also freezes if the setting is set to a player that isn't installed.

If you have MusicTracker plugin enabled, this might help you!
