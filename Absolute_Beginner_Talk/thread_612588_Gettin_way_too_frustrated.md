### Post by Shinbu-Otaku on 2007-11-14
what are the specs of your computer? and what version of Ubuntu are you using? Because if you have older hardware and the newest Ubuntu (7.10) it's not going to be as fast as it should. Its like trying to put Vista on a 1ghz CPU 512mb RAM system, yes it can handle it, but my god its gonna be slow.

Regarding your dual boot issue, when you first install ubuntu you can manually edit partitions (thus taking some available hard drive space that Windows doesnt necessarily need i.e 20gb of space from a 120gb hard drive where you only have 40gb of stuff on Windows) and then they would have both been installed 'side-by-side'. Admittedly, i do know my way around computers quite a bit, so it may seem like im talking down to you but im not, and i apologise if it sounds that way.

In my opinion, if you have older hardware, try to download an older version of Ubuntu (6.06 works incredibly on 5yr old technology) and see if that helps. Also, if you do install Windows again, when installing Ubuntu for the dual-boot make sure to select the 'Manually Edit Partitions' option of the installation and create space for it so it doesnt wipe windows.
