### Post by 23meg on 2006-12-15
Open source in general means that the source code of the software is available for review. The Open Source Initiative have a document called the [Open Source Definition]("http://www.opensource.org/docs/definition.php"), which is more specific, and it's the generally accepted definition of open source.

Free software by the [FSF]("http://www.fsf.org/") (Free Software Foundation) definition means that the software is both open source, and gives you freedom to modify and redistribute it, as well as use it for any purpose you like, under the terms of the [GNU GPL]("http://www.gnu.org/licenses/"). See [here]("http://www.gnu.org/philosophy/free-sw.html") for more information.
