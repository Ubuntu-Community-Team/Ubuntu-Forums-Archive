### Post by Alterax on 2007-08-13
I recommend any of the beginner level Linux certification books to get you started with the terminal commands (specifically the BASH shell, which is what Ubuntu uses by default).  There is also a lot to be learned from Linux in a Nutshell (O'Reilly press), which has already been recommended.

Another noteworthy book is "Unix Power Tools".  Although it's written from a Unix power-user's viewpoint, the commands are not all that dissimilar from Linux commands, and it can teach you tons of things to do with the terminal.  I've barely scratched the surface of it, but the small amount I learned from it has been well worth the purchase.
