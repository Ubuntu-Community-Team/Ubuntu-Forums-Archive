you can use the ubuntu live cd to resize partitions, just boot from the live cd and run gparted set the partition size and filesystem type and apply. im running three distros on one drive, i usually set a 1.5gb for swap on the first partition and three ext3 partitions for the installs and use the same swap partition for all the distros.
