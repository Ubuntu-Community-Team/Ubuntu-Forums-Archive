### Post by IYY on 2006-09-16
> 1. So if it is alright I'd like to know what in your opinion would be the ten cool things that a newbie should learn to do using Linux?

1. install a few applications from source
2. learn the shell, and how to write shell scripts
3. set up an apache server on your machine and host a website on it (if your ISP allows this)
4. get your desktop to look just right for you by changing window managers, themes, shortcut keys, etc.
5. learn one of the difficult text editors: vi or Emacs
6. learn LaTeX for typesetting

Some of these aren't too easy for newbies, but these are very useful skills and will gain you a lot of respect in the open source community.
 
> 
2. What all should I be capable of doing to graduate out of the newbie class?

Everybody has his/her own definition of "newbie". Some people still see themselves as newbies after years of experience and a lot of knowledge, while others consider themselves to be experts after one month. It all depends on your goals. 

> 3. At a more general level, I'd like to know how similar Linux and Unix are, because I know very little about Unix too in other words could I learn some fundamental everyday Unix commands using Linux and if I could what would they be?

Very similar. From a newbies perspective, identical. If you compare something like solaris or FreeBSD (they use kernels that came from Unix) to Linux, they will have the same functionality: commands that do the same things, the same programs, but very different kernels. So, they do the same things, but don't work in the same way underneath. 

And of course, some things are distro-specific, like RPMS, apt-get, etc.

But if you learn the Linux commands, you could say you know Unix as well.

### Post by bakert on 2006-09-16
Here's some fun things you can do in Ubuntu.  Just a very quick run through.  You'll have to google or experiment for the details (that's the fun bit - the computer IS the game).

1. ssh.  Control your machine remotely, or log onto your webserver from home.  You need two machines to try this one out!  Also check out scp for moving files between machines.

2. Upload to flickr with kflickr, resizing on the fly so you don't use up too much space in your flickr account.

3. Try out amarok or banshee for your music collection.

4. Install abiword and gnumeric.  Faster Word and Excel clones than the OpenOffice versions.

5.  Do some programming.  Python, perl, bash scripting are all possible on the default install.  Ruby is in the repositories.  I'm sure there's loads more.  C, PHP.  Anything you care to name.  Sometimes I think Unix is basically the operating system for programmers.

6. Learn the commandline tools.  Programs like cut, sort, uniq, xargs, find can be so useful.

7. Set up a local webserver.  Run Apache on your machine.  Develop a website locally and then use scp (see 1. above) to make it live.
