### Post by phr0ze on 2007-04-13
Do not make a new partition and install files if the hard drive is failing. If the windows OS is screwed up, thats a different story. But unless you are sure the drive is fine, only run it to recover the files. 

If you have another windows machine laying around, you can easily pop the bad drive in as a slave or secondary and use the working windows to read the files. If you need to do datarecovery (failing drive) I recommend R-Studio. Its a windows program. I don't know of anything as good at NTFS recovery in linux. There is cheaper software and probably free stuff in windows and linux, but any software you try which isn't good enough will make it less likely the good software will recover your data. (ie. Cheap software might do more harm than good.)
