### Post by sumguy231 on 2007-09-09
Is there any particular reason you're trying to install 5.10? It's no longer supported and no longer receives updates, so you should either install the latest version (7.04) or the 6.06 LTS version, which is older and more outdated but supported for longer. Ubuntu 6.06 and higher combines the install cd with the LiveCD which allows you to run the system without installing, while 5.10 has a separate liveCD which you would need to find if you really wanted to.

As far as 64-bit stuff goes, if you don't want to deal with some proprietary stuff like Flash not working automatically, you might want to just stick with the 32-bit version. Caveat: I have never owned a 64-bit system so I'm just speaking from what I've heard.
