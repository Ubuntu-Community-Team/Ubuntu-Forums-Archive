### Post by xequence on 2005-09-20
The live CD is normally much slower then the hard drive install.

I have 192 MB RAM and the live CD was super slow, but the hard drive install wasnt.

Knoppix uses a different desktop environment, called KDE. Ubuntu uses gnome. From my experience KDE is slower then gnome, but people dont often aggree. You can install other desktop environments that will go faster for you, like XFCE, Fluxbox, Openbox, icewm, enlightenment, and many others. I would recomend you use another desktop environment other then gnome or kde. 128 MB RAM just isnt alot for them...

XFCE is a fast DE. It is quite graphical, though faster then gnoem or KDE.

Openbox is VERY minimal. It is super fast.

Fluxbox is like openbox, but with some extra features. It runs slower on ubuntu (and red hat) then other linux distros because of some UTF-8 encoding or something. It is still faster then XFCE, not as fast as fluxbox.

IceWM is fast and is good with themes in my opinion. It has a similar bar to windows at the bottom.
