### Post by markbusu on 2008-02-27
Hi

You can use Win Emulators such as Wine which is free... or Cross Over from Code Weavers which comes at a cost...

I managed to get a number of Win Games and Applications with Ubuntu using Wine, the best start is to check which software has been tested with Wine at the following link:

[http://appdb.winehq.org/](http://appdb.winehq.org/)

With regards to games, you just need to ensure that you configure your GFX card correctly for 3D Accellaration.
