### Post by catlett on 2006-09-04
I use TV Time. It is in the repositories (i.e. synaptic package manager). I have a basic Hauppauge tv card I got for $40 dollars at circuit city.
I have cable tv and I just run the coaxial from the cable box to the tv card. Then I use TV Time like my regular tv. I put it on channel 3 and change channnels with the cable box using the remote.
