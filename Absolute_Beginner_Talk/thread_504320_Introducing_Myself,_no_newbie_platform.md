### Post by macogw on 2007-07-19
The live cd will boot you into the OS itself and you can run it to test out all the hardware for compatibility, then it has an installer. If it's an old comp, the alternate cd is better to install from, but it doesn't let you test drive it.  The live cd runs slower than an installed system will because it is limited by your CD drive speed and everything has to be loaded into memory.

Most drivers are right in the kernel and should work automagically.  New ATI and Nvidia cards require getting drivers elsewhere, as might some wireless cards.
