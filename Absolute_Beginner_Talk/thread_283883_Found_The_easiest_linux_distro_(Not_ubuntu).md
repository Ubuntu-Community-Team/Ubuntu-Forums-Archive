### Post by leech on 2006-10-24
Well, from my own experience (which I must say I've tried just about every linux distribution out there at one time or another) Freespire is good, but lacks good Gnome support.  If Gnome is what you like (which is what I prefer) then Freespire just doesn't cut it.

Now mind you, perhaps this is only my opinion, but I think what would make Freespire and Ubuntu a lot better is if they didn't try to go too far from their ancestors (Debian).  By this I mean as far as package dependencies go.  This way we don't have .deb files floating around the Internet everywhere that aren't compatible with all distributions (like RPMs have already gotten)  For the most part, with a little tweaking, you can usually get any deb package to work on another Debian based distribution.  Especially if you have a deb-src repository for it (for example, there has been several cases where I had the deb-src line for standard debian in ubuntu and grabbed a program from there and packaged it.)

Freespire is easy in that it installs the codecs for you, but that is only because Lindows/Linspire/Freespire won a lawsuit with Microsoft that allowed them to license the codecs from them.

Freespire is easy also because, much like Windows, it really only has one DE, and a set of software that is much more limited (initially, I know you can always add more repositories, much like Ubuntu).

Ubuntu I find better simply because it is more innovative, whereas Freespire is trying to be the Linux for Windows users.

### Post by janbockaert on 2006-10-24
I tried linspire back in the days i choosed my first distro, and i did not like it. It was already out-dated (i hear freespire is also anything but bleeding edge). It had a lot of branded programs that looked cool at first, but were inferior to existing linux programs (like lphote and lsongs). You logged in to the system as a root! and it did not supported dutch. And at that time, i had to pay to install evolution.

Freespire is based on linspire, so i will not give it another trie. Maybe   at a later time. I do think it has one big advantage over other linux distro's, and that is the availability of commercial software in their Click n run warehouse. It is (as far as i know) the only distro that lets you install commercial software via repositories. But then again, i do not use commercial software in ubuntu.

If you want a beginner-distro, opensuse is a very nice one. It doesn't ape windows to the ridiculous (root acces!) but you also do not need the command line for configuring your system. And 10.1 just got repacked with the latest fixes.

What makes ubuntu the distro of my choice, is this forum, where you will find an answer to every question you could ask. It is unmatched by any other distro.
