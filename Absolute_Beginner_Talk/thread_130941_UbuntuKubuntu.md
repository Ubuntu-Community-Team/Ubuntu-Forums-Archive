### Post by WolfJay_83 on 2006-02-15
The safest answer is to try both.  If you have both installed, than you can simply pick one at the login screen (or set one as default).  All of your files and programmes will be present in both desktops and work (nearly) eaqally well.

To install both.  Either install Ubuntu.  Then once installed, open a "Terminal" and type in 
Sudo apt-get install kubuntu-desktop

or, you can install kubuntu, then open a Terminal and type in
Sudo apt-get install ubuntu-desktop

Either way will leave you with both installed, and the option to select either at the login screen.
