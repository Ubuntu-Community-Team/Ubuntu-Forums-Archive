### Post by mr bob on 2007-05-03
I've posted this reply on another thread.

I was having trouble with firefox plugins. But I finally got it to work with MPlayer plugin, installed with Automatix.

The BBC videos didn't work at first but I right clicked on the window and selected configure.

It seemed to work once I selected "Use HTTP instead of RTSP".
I also selected X11 video output and alsa audio, although I don't know if that was necessary.

### Post by mr bob on 2007-07-04
I seemed to have got it working with the MPlayer plugin for Firefox. See my post above. MPlayer needs a to be configured properly then it plays fine.

If you do use MPlayer, remember to click on "preferences" in the BBC Player and select Windows Media rather than RealPlayer.

However some audio and video is still RealPlayer only on the BBC, such as local stations. In which case I found the easiest way to install video software such as RealPlayer is to use the Automatix installer. Check any posts on Automatix.
