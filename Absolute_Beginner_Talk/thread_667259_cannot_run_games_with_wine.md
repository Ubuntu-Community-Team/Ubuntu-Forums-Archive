### Post by AndyCooll on 2008-01-14
Installing Wine isn't a guarantee that a Windows game will work. 

You should start by visiting Wine HQ's [Application DB]("http://appdb.winehq.org/") and see if anyone else has got the game to work, and if they did what modifications they had to make if any.

You could also check out our own  [Wine]("http://ubuntuforums.org/forumdisplay.php?f=313") forum and see if anyone has had any success in getting these games running, or even our own  [Gaming & Leisure]("http://ubuntuforums.org/forumdisplay.php?f=93") section.
