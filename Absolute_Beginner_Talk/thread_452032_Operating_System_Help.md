### Post by BarfBag on 2007-05-22
By themselves, Windows games (or any Windows program) will not run in Linux.  The only way is through Wine (or a Wine-based application such as Crossover or Cedega).  Wine is free, but it doesn't run much without errors.  The best way to run Windows games is through Cedega, but you have to pay a monthly fee.

### Post by Scunizi on 2007-05-22
There are lots of links for games that run on Linux and Ubuntu.  I couldn't begin to provide them all here.  However there are not a huge amount of windows games that run.  UT2004 and up have a native linux install and work great on Ubuntu.  There are some others as well.  Doom wad files can be used with an opensource backend that's easiest enough to install.  If you're really stuck on some games you might ask youself what kind of games are they, FPS, stratagy etc.  What draws you to that particular game.., friends, fun, smoothness of play.  Once you think about that you might just find a number of linux games that will do the same for you even though they aren't exactly the same.

Your restore CD's for your other computer will probably not work on the new machine at all.  If you had a genuine copy of xp that you bought it would install on the new machine just fine.  However, Microsoft check through the authenticaiton process to see how many machines your windows copy is installed on and will disable it if it's more than one (1).  You can copy Ubuntu all day and hand it out with no penalty.
