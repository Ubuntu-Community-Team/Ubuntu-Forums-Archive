### Post by Sourcey on 2007-09-23
I had the same problem, I used envy to make sure the Nvidia driver had been removed.

Then I deleted my xorg.conf, but remembering to make a backup first.

```
cd /etc/X11

sudo cp xorg.conf xorg.conf.back

sudo rm xorg.conf
```


Then you can install the driver manually, or use Envy to install the driver.

Then reboot.
