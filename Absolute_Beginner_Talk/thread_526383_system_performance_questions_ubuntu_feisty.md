### Post by RedSquirrel on 2007-08-15
> **myk.dinis said:**
> I would like to improve the efficiency of my desktop...

The minimal install would serve you well.

[https://help.ubuntu.com/community/Installation/LowMemorySystems](https://help.ubuntu.com/community/Installation/LowMemorySystems)
[http://www.psychocats.net/ubuntu/minimal#barebones](http://www.psychocats.net/ubuntu/minimal#barebones)

Using that, you get to build up the system to suit your needs rather than having a whole bunch of stuff installed that you'll probably never use.

If you do a search for "lightweight apps" (or similar), you should be able to find some threads with suggestions for applications. For email, I use webmail or Thunderbird (which is not really light, but I happen to like it ;)).

Have fun. :)

PS - Xubuntu is designed to run well on older systems, but it comes with a lot of programs pre-installed. Xfce is available on its own as a metapackage (xfce4). I prefer Fluxbox myself.

### Post by p_quarles on 2007-08-15
I agree with the suggestion that you might want to install a lighter GUI, like Xfce or FluxBox. 

If you just want lighter apps, though:
Epiphany is *definitely* faster than Firefox.
Sylpheed is lighter than Evolution or Thunderbird
Beyond that, it's entirely possible to use terminal-based applications for everything. Links2, for instance -- while it is a graphical web browser -- is considerably lighter than anything similar. The text editor Emacs can be extended to perform most internet-related tasks. This last option involves a much steeper learning curve, so it depends on what you want to get yourself into. :)
