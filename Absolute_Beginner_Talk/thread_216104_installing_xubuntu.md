### Post by jvdowney on 2006-07-14
If you are using a standard Ubuntu setup, the easiest way is to open Synaptic, hit the 'Search' filter and type in xfce. There is an entry for xfce4 that will only install the core and setup scripts without giving you the whole desktop metapackage.

### Post by H.E. Pennypacker on 2006-07-18
This is exactly what I was looking for: how to install *only* XFCE.  Installing Xubuntu will likely get you a lot more than you're asking for.  JVDowney's answer worked, and I was able to log in using XFCE.  You can still use your Gnome applications (gEdit, Gnome Terminal, etc.), but you now have a  new Desktop Environment.  You can always switch back to Gnome (as I did - only testing out XFCE).
