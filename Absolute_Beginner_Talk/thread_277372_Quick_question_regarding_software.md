### Post by Delkster on 2006-10-14
If you can find the software you want in the Ubuntu software repositories, it's a good idea to install it from there. The software packages included there generally integrate well with the system, are easy to install and remove, can be automatically updated with system updates, and are digitally signed so their authenticity can be guaranteed when installing so it's also more secure than installing software packages from random websites.

The searching functionality in **Applications > Add/Remove** is a little limited but sometimes you can find just what you want with it. If that's not enough, you can click on **Advanced** in Add/Remove Applications and **Synaptic Package Manager** will open (it can also be found in **System > Administration > Synaptic Package Manager**) which allows you to browse and search a larger variety of software (all of it still from the Ubuntu repositories), although it's also more complex to use.

If the searching functionality in these tools isn't adequate, a third option is to use web search engines to find the application you want. Of course most of the software you find that way may only be available for Windows, so it's a good idea to remember that when searching. Even when you find the Linux software you want that way, I'd recommend first taking another look at the repositories for example with Synaptic and seeing if the software (whose name you now know) is available there after all. Sometimes Google may be very valuable as a search tool for finding the package you need even if the package itself is actually available in Ubuntu.
