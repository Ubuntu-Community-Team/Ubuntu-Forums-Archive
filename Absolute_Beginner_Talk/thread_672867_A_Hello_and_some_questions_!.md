### Post by Christmas on 2008-01-20
I'd recommend [ur=http://www.tuxfiles.org/]http://www.tuxfiles.org/[/url] which is just great for beginners. It is a must read, trust me. A good introduction to Bash (you'll use this one often) is [http://linuxcommand.org/](http://linuxcommand.org/). And then, the topics in the Wiki ([https://wiki.ubuntu.com/](https://wiki.ubuntu.com/)) which usually resolve the first questions and problems a newbie has:

[https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)
[https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)
[https://help.ubuntu.com/community/InstallingSoftware](https://help.ubuntu.com/community/InstallingSoftware)

When you advance, I strongly recommend reading the Debian tutorials (Ubuntu is based on Debian) from [http://www.debian.org/doc/](http://www.debian.org/doc/).

It's better if you get used with the command line in the first place instead of doing things using a GUI. CLI is very powerful and quite modern in my humble opinion. It is often faster to do things like that. Good luck reading!

### Post by candtalan on 2008-01-20
Welcome!
The amount of RAM will be one of the factors to have a significant effect on performance. Also keep in mind that a modern full graphics OS will use the PC resources - including time of course. There is a great flexibility in open source and so there are many linux based distributions which are deliberately intended for less use of resources. In the Ubuntu family, the one that comes to mind is Xubuntu. Although distros such as DSL, Puppy, Vector are some of the smaller ones.
When I moved from windows I looked for differences, and boot speed was one. However its significance decreased as I used it more.
