### Post by Hospadar on 2007-08-13
If you don't know, .mozilla is a hidden folder in your home directory.  if you go to view->show hidden files in the file manager in your home directory you can find it.  just right-click and rename it to ".mozilla-old"  this way firefox will think you have no settings and create a fresh set.  if that fixes the problem, you will have to reset any options/bookmarks you have and then you can delete the old settings folder.

You may want to look into exporting/importing firefox bookmarks to/from file before you do this so you don't have to redo them all if you have to clear your settings.
