### Post by kinematic on 2006-07-27
probably a bad burn or download.
when you boot the cd there's an option to check the cd for errors.....run the check and see what it comes up with.
if the cd has errors check the md5sum of the iso image to see if that's okey.
if it checks out ok burn at a lower speed(say x4)if it doesn't you'll have to redownload it.
