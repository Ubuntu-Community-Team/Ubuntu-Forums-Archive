### Post by y-lee on 2008-02-23
> Oh, also...can you install other programs on it? Can you use Synaptic Package Manager? 

I don't use gOS but it is based upon [the Enlightenment (window manager)]("http://en.wikipedia.org/wiki/Enlightenment_%28window_manager%29") known to be light on resources, therefore any program that works with Enlightenment would work in it. If you install it with your Ubuntu installation I would guess that about all your Ubuntu programs would also work. If you make a clean install of gOS you might have to install additional stuff to make KDE or gnome apps work. Also  according to wikipedia

> gOS 1.0.1 had Ubuntu's easy to use "add/remove" application installer, but it is missing in gOS revision 2.0.0. However the synaptic package manager is available in both versions.
