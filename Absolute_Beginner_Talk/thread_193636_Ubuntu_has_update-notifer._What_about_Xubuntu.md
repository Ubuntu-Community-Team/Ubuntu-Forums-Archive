### Post by Dr. Nick on 2006-06-10
that depends, I think with xubuntu you may have the synaptic application. If you find it and run it then hit "reload" and then the "mark all upgrades" button then 
"apply" to update it.

If you dont have synaptic then open a terminal and type

sudo apt-get update
sudo apt-get dist-upgrade


to update it. I dont know of any way you can be notified though, BUt I have not used xubuntu in awhile so it may still be possible
