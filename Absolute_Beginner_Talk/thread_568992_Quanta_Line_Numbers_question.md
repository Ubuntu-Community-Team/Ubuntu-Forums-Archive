Does anyone know if I can have line numbers in Quanta Plus showing by default without having to press F11 each time?

Thanks

---

### Post by Crenfinkle on 2007-10-07
sure thing....

go to **Settings -> Configure Editor -> Appearance ** in quanta+

Find the "Borders" section and X the "show line numbers" option.
