### Post by mysticrider92 on 2007-11-13
Try:
```
sudo gedit /etc/X11/xorg.conf
```

Find the line that says "VertRefresh 43-60" and make it "VertRefresh 75". Then save and close the file and press Ctrl>Alt>Backspace. Login, and the refresh rate should be at 75.

For the overscan problem, I don't know what to tell you.

ATi Linux drivers are kind of cheesy, and Compiz (used by desktop-effects) is instable, so that is what causes program crashes. There is no real way to fix this as far as I know.
