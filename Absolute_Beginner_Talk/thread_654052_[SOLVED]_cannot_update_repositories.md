### Post by dondad on 2007-12-30
> **rickcam said:**
> problem is resolved, Changed the dns from 10.1.1.1 to the broadband providers actual dns(in this case aapt) 203.8.183.1 up in wired network connection icon top right of the desktop screen. All boxes are now ticked in software sources, thanks muchly.

If you go to software sources in the admin menu, on the first page there is a box that says "download from". Select the dropdown, then "other". This will bring up a page of servers. There is a button that says select best server. Clicking this will check all of the servers and determine which has the best connection for your area. This can speed up downloads a lot if you are using a marginal server connection.
