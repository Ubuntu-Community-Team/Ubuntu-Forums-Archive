### Post by bluepowder7 on 2007-11-26
first - is your BIOS set to boot from the CD first (or at least before the hard drive)?  if it is, then it's likely the CD wasn't burned the correct way.

an .ISO file can't be put onto a disk as a file.  the program doing the burning has to burn it as an image.

now, before you actually install ubuntu, make sure that you have defragmented your windows system a few times since you're gonna need part of that drive for the ubuntu install.  we're assuming that you want winxp and ubuntu to be on that 40gig drive.
