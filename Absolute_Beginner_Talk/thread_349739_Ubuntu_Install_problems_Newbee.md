Ubuntu doesnt come with as standard .mp3 format support or for that matter all other proprietary codecs. Dont worry about it - you can install of them once you have ubuntu up and running - see [http://ubuntuguide.org/wiki/Ubuntu_Edgy](http://ubuntuguide.org/wiki/Ubuntu_Edgy) for lots of helpful stuff on setting up your system once installed.
