### Post by durrell on 2007-06-13
Close gaim out, and then try to start pidgin. You said pidgin did install fine, right? Well for me, if I'm running pidgin and then try to start GAIM, it switches pidgin to gaim because gaim = pidgin, just with a different name. Gaim had to get a name change due to trademark issues with AOL.

Edit: And also, make sure when you start pidgin, you look up in your taskbar. The buddy list doesn't appear at startup, look for the green ball with a conversation box over it..that'd pidgin.

Edit2: And actuall starting gaim closes pidgin and then reopens it, so I dunno.
