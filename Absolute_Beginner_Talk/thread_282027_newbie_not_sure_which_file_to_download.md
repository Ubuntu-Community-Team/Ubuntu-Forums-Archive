### Post by bikeboy on 2006-10-22
The general iso you want is the Desktop i386. There is also an alternate cd which uses a text based installer for older machines, but your should do fine with the desktop one.

PPC and AMD64 are for Apple G3, G4 & G5 and AMD Athlon64 respectively.
