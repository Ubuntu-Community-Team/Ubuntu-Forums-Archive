to boot the cd, you hold down the option key when booting and then select the cd icon.

Airport Express cards (in powerbooks) do not yet work that i know of, since the company that makes the chipsets (Broadcom) has not released linux drivers for the BCM4301.  there is an [online petition]("http://www.petitiononline.com/BCM4301/petition.html") about this issue.
