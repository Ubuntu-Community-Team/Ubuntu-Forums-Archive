### Post by mahiyar on 2007-02-25
If you are very new and if this is going to be solely a linux box then just follow the default instruction while installing from CD (assuming that the HD has been in use before and is not brand new), the ubuntu installation will take care of formatting etc. Try keeping it simple with few partitions and remember to make swap partition double your RAM size (default is equal).
