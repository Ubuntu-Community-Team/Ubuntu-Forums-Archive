### Post by pjkoczan on 2007-12-29
Adding a Firefox translation plugin might be a good place to start ([https://addons.mozilla.org/en-US/firefox/browse/type:1/cat:37/sort:popular](https://addons.mozilla.org/en-US/firefox/browse/type:1/cat:37/sort:popular)).

I don't know how well it works since I haven't tried it. Part of the problem of language translation is that some grammar constructs are different and it doesn't handle idioms or proper names perfectly (it tries, though).

You should also check out some of Google language services ([http://www.google.com/language_tools](http://www.google.com/language_tools)), I hear they're making some very interesting and powerful developments in the field of computational linguistics.
