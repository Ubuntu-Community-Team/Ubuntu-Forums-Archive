### Post by az on 2006-03-06
1. Remove all accessories (pci cards) except the video card.
2. Unplug the power and remove the motherboard's battery for about a minute.  
3.  Put the battery back and try booting.  The MCOS memory may be corrupt.

4.  Do you have another vidoe card handy?  Some video cards do not play nice with some motherboards.  The same goes for any pci hardware, but videocards are more so.

5.  If you have several sticks of ram, try booting with only one plugged in.  Some boards insist you use the first ram socket first.
