### Post by mvniekerk on 2006-12-11
Whaddap my homies!

I've created a script for my noob friend to install kxdocker without worrying about what to download and so-on.

This was created because the kxdocker-1.1.4a deb file in the universal repos is busted, so the script downloads, compiles and installs the tarballs from the kxdocker main website.

You can go to [the gnome-look website]("http://gnome-look.org/content/show.php?content=49964") and download it from there.

It will open a new window also that links to an ubuntu post on how to prime the thing to be schweet.

Hope you enjoy it!
Pop me a message of it's not working for you.
