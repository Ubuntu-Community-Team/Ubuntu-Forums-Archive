### Post by ramjet_1953 on 2007-04-18
I have found[COLOR="Red"] wxDownload Fast[/COLOR] to be absolutely brilliant!

Much faster than just clicking on a links and using your browser to download it.

It segments the file and simultaneously downloads each degment, speeding-up the process considerably.

It also atomatically does an MD5 checksum test at the completion of the download.

You can download it using Synaptic. It is listed as [COLOR="Blue"]wxdfast[/COLOR]

Here's the SourceForge link, if you want to check it out: [http://dfast.sourceforge.net/](http://dfast.sourceforge.net/)
