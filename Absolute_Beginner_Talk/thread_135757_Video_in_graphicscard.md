---

### Post by Stormy Eyes on 2006-02-24
That depends on what exactly you want to do. If you just want to watch TV through your ATi X600 card, you can try [tvtime](http://tvtime.sourceforge.net) (sudo apt-get install tvtime). If you intend to do movie editing, try [cinelarra](http://heroinewarrior.com/cinelerra.php3) (sudo apt-get install cinelarra).
