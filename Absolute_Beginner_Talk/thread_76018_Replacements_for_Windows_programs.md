### Post by LHZ on 2005-10-14
What do you use iTunes for? If you need an media player, you can get XMMS (or rythmbox, or amaroK). If you need iPod syncing, gtkpod (though I haven't ever used it). If you use iTunes to buy music, you're out of luck for the moment.

[www.ubuntuguide.org](www.ubuntuguide.org) has very helpful pointers on how to install XMMS. It also explains how to install Limewire (which has a Linux version). If the big list of questions daunts you, use the search function of your browser to find 'XMMS' and 'Limewire'.

If you want to play MP3's, you'll have to download the necessary codecs (the MP3 format is proprietary, so the Ubuntu people can't give it away). Again, this is covered on [www.ubuntuguide.org](www.ubuntuguide.org).

Have you ever used Synaptic (System->Administration->Synaptic)?
