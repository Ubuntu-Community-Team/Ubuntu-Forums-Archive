### Post by raja on 2007-04-10
As you restart your computer, you can press a certain key to enter the bios settings. The exact key varies - try esc or del or F1. The name of the key should also be displayed in the first few seconds as the computer starts. Pressing this key takes you to bios setup. There you will be able to change the order in which the boot device is selected. If your hard drive is higher in the order, your pc will boot from it even though you have a bootable cd in the drive. So set it so that your cd drive is first on the list. Now continue to boot and you should boot from the cd.
For more details, check the manual that came with the pc or search online for tutorials on BIOS.
