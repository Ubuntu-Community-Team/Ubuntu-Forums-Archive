### Post by misconfiguration on 2007-08-17
Generally you only run into issues with drastic differences in clock frequencies on the RAM.

Now days the boards are smart enough to realize that and operate the RAM evenly (work at the lower clock-ratio).

There is not much difference betwixt PC100 and PC133, you should be just fine :).

### Post by WebSiteGuru on 2007-08-17
You should be able to use both. Put 512MB in the first slot, and 128MB in the second slot.

If you run into the memory problem, then remove the 128MB and use only 512MB.

You can also get another 512MB and make it 1024MB. The PC133 shouldnt cost that much more to get.

Hope this help.
