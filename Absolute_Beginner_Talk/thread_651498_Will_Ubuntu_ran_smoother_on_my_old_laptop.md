### Post by cjm5229 on 2007-12-28
Try installing Wubi in Windows, It will install Ubuntu into a file in windows then you can reboot into Ubuntu and try it out, if you don't like it you just boot back into Windows and uninstall Wubi, it won't mess up your hard drive. Now when you want programs in Ubuntu, Don't try to install Windows apps until you know what you are doing. Go to Applications> Add/Remove  and look for apps that will do what you want. If you don't find what you are looking for there, use Synaptic Package Manager "System>Administration> Synaptic Package Manager. Make sure all Repositories are enabled "Settings>Repositories" reload and then use search for what ever you want an app to do. There are Ubuntu Apps to do just about anything you could do in Windows, They are a little different but once you get used to them, they will work as well or better than what Windows has. Once you get used to using Ubuntu you can just remove Windows and Install Ubuntu.:) 
Good Luck, Carl
