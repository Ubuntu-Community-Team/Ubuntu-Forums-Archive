### Post by candtalan on 2007-12-04
(Welcome!)

You are correct, install xp first, and then ubuntu, which will pick up the install control elegaantly.
Ubuntu installer I think will find it easy to offer to install into unused drive space - unpartitioned unformatted space(!) so if you want an easy install, put windows in and do not actually partiton for ubuntu, just leave the space, then choose that option from the ubuntu installer (install into the largest uncommittted space).

btw after burning your Image to cd (do not 'make iso') then the check cd option in the cd boot up options is useful to verify the image and burn and your cd drive are all ok.

good luck
