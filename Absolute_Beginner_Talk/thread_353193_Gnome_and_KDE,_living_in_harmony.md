### Post by taurus on 2007-02-04
If you install Ubuntu, then install KDE with

Applications -> Accessories -> Terminal
```
sudo aptitude update
sudo aptitude install kubuntu-desktop
```
And choose which window manager you want to use in the Options (bottom left) at the GUI login screen.

Check [http://www.gnome-look.org](http://www.gnome-look.org) for the OSX theme.
