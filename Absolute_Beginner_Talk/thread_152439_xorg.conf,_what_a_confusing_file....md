### Post by jdong on 2006-03-29
The flat panel will have the most restrictive resolutions, etc. Disconnect the Samsung, then run **sudo dpkg-reconfigure -phigh xserver-xorg**

Just for fun, if you want, run **sudo dpkg-reconfigure -plow xserver-xorg** to have it ask you every possible configuration option about your display setup. Try that both with one and two monitors connected; I don't know if the xserver-xorg setup script can handle dual screens.

Let's isolate only the stuff that we are concerned with right now. A line starting with # is just an explanation what the previous line does:

> Section "Device"
        Identifier      "NVIDIA Corporation NV34 [GeForce FX 5200]"
        **Driver          "nvidia"**
# This means I am using the official nvidia drivers rather than the nv driver
        BusID           "PCI:1:0:0"
# The location of your card. This is already set up correctly on your machine

        **Option "TwinView" "on"**
# Turning dual-display support on
        **Option "MetaModes" "1280x1024,1024x768"**
# The resolutions of the two monitors.
        **Option "TwinViewOrientation" "LeftOf"**
# Meaning that the first monitor (1280x1024) is physically located to the left of the second
[B]
        Option "SecondMonitorHorizSync" "31.4-68.6"
        Option "SecondMonitorVertRefresh" "59-85"
[/B]
# These two lines are important. They are most likely the reason for your monitor not working. The exact values differ from one monitor to the next, and you can find yours either in the monitor's manual or somewhere in the monitor's settings (that's where I found them).
        
        Option "RenderAccel"    "true"
# Makes things faster :)
        Option "NoLogo" "True"
# Don't display the nVidia logo every time you start an x server
  **      Option "ConnectedMonitor" "CRT,CRT" **
# Saying that both my monitors are CRT's
EndSection


The next important section is the "monitor" section. There is only one, and it's for the primary monitor. The most important part here is, again, the horizontal sync and vertical refresh. Find these in your monitor's manual for things to work.

> Section "Monitor"
        Identifier      "DELL M781mm"
        Option          "DPMS"
        [B]Option "HorizSync" "30-85"
        Option "VertRefresh" "50-160"[/B]
EndSection

And the screen section:
(note that we are pretending to only have one screen, and only one monitor, the primary):

> Section "Screen"
        Identifier      "Default Screen"
        Device          "NVIDIA Corporation NV34 [GeForce FX 5200]"
        Monitor         "DELL M781mm"
        DefaultDepth    24
        SubSection "Display"
                Depth           24
                Modes           "1024x768" "800x600" "640x480"
        EndSubSection
EndSection

This is all. Obviously, you shouldn't copy my file but try to understand it and apply the ideas to your own.
