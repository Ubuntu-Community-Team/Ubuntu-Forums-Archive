### Post by FizDev on 2006-03-18
Oh, I just saw that you have Hoary... then in that case, I believe you can use the Alacarte Menu Editor. To install and use it, open a terminal and do :
```
wget http://dev.realistanew.com/alacarte/releases/0.8/alacarte_0.8-0ubuntu1_all.deb
```
```
sudo dpkg -i alacarte_0.8-0ubuntu1_all.deb
```

Now it should be installed in your menu in :
Applications --> Accessories --> Alacarte Menu Editor
