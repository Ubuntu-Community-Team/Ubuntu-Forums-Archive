### Post by forrestcupp on 2008-03-28
It looks like Ubuntu CE doesn't even have a Gutsy version.  They had a conversion script for Feisty, but it appears that they haven't even updated to Gutsy.

What you can do, is just install all of the software it includes.  One of the great softwares that it comes with is Gnome Sword from The Sword Project.  It is a great Bible study tool.  You can install it from the repos along with a lot of modules.
```
sudo apt-get install gnomesword
```
Then open up Synaptic and search for **sword-** to find all of the modules that are in the repos.  You can visit [their modules site](http://www.crosswire.org/sword/modules/index.jsp) to see if there are more modules you might want.

I'm sure most of the other apps that Ubuntu CE includes are in the repos, too.
