### Post by jfinkels on 2007-04-29
You may want to try "Frostwire", which is exactly the same, but it's better because it's free, open source, you don't get annoying requests to try LimeWire Pro, etc. Type in the terminal
```

sudo aptitude install frostwire

```
and it will be installed. Since you are a novice, you should only be installing programs from the official repositories, either by typing "sudo aptitude install whatever" or by using the graphical package manager (which you may prefer), Synaptic. Synaptic Package Manager is in the "System > Administration" menu.
