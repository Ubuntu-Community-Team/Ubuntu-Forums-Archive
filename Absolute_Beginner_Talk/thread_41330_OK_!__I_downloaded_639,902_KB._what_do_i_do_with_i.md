What you need to do is use it to make a CD from the ISO file you already have. How this is done depends on your particular burning software.

### Post by poofyhairguy on 2005-06-13
[QUOTE=MrClarke]
does ubuntu work ok with the NTFS?
[/QUOTE]

It can read NTFS fine. But it can't write to ntfs drives. Only Fat32 drivers can be written to.

If you need to resize you NTFS drive to make space for Ubuntu (drive needs at least 5 gigs free and is defraged), then follow this advise:

[http://mlf.linux.rulez.org/mlf/ezaz/ntfsresize.html](http://mlf.linux.rulez.org/mlf/ezaz/ntfsresize.html)
