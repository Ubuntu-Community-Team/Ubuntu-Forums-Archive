### Post by Kobalt on 2007-05-21
You can download the packages from any OS from this website : [http://packages.ubuntu.com](http://packages.ubuntu.com)
The packages you should need are these ones : ubuntu-restricted-extras libxine-extracodecs gstreamer0.10-plugins-base gstreamer0.10-plugins-good gstreamer0.10-plugins-bad gstreamer0.10-pitfdll
