### Post by catlett on 2006-09-20
You can install automatix without using the repositories. Open a terminal and enter these commands
```
wget http://www.getautomatix.com/apt/dists/breezy/main/binary-i386/automatix_6.4-6-5.10breezy1_i386.deb
```
```
sudo dpkg -i automatix_6.4-6-5.10breezy1_i386.deb
```
```
automatix
```
