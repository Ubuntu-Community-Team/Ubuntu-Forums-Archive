### Post by Ragazzo on 2006-07-13
If you have a file "longfilename", you can start typing "cd l" and press TAB and it will autocomplete it. If there are multiple choices for completion, pressing TAB twice will show them. Also, Midnight Commander could be similar to CDir (mc in Synaptic).
