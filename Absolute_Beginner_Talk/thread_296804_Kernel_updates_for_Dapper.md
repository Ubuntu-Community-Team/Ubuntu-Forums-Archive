### Post by Sef on 2006-11-10
> 'm a little confused about all this talk with Dapper, Edgy, Feisty etc. upgrades and so on.

Dapper is current stable Ubuntu version.  The packages will work well, but not the most up-to-date.  However, less likely to have crashes and bugs.

Edgy is the current unstable Ubuntu version. The packages are more up-to-date, and more likely to have have crashes and bugs.

Fiesty is the next unstable Ubuntu version.  It is due out in April 2007.  

Ubuntu releases an update every six months.

> Will 6.06 still get regular Kernel updates etc. for new hardware, bug fixes ? 

Yes, for the desktop, there will be upgrades for till 2009 and dor the server till 2011.
