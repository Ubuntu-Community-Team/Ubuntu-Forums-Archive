### Post by wersdaluv on 2008-05-23
Actually, you don't need to install Bittorrent. If you are using Ubuntu Hardy Heron, it has Transmission installed by default.

Open Transmission in Applications->Internet->Transmission Bittorrent Client.

Hardy Heron opens Torrent files with that application automatically by default.
