### Post by mssever on 2007-04-21
The Ubuntu CE forum is here: [http://ubuntuforums.org/forumdisplay.php?f=168](http://ubuntuforums.org/forumdisplay.php?f=168). You might be better off asking questions about CE there.

You might want to consider downloading the ISO and burning a live CD so you can "try before you buy."

Also, I noticed that the Feisty version of CE is a week or two away. It would be smart to wait for it. You should probably upgrade to standard Feisty first. Then, on the Ubuntu CE website, there's a script you can run to convert standard Ubuntu to Ubuntu CE. No need to backup and re-install.
