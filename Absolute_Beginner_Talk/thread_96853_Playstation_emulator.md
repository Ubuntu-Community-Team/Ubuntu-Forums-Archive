### Post by phanboy_iv on 2005-11-29
ePSXe [http://www.epsxe.com]("http://www.epsxe.com") has a Linux version, but you'll have to install it manually(i.e., no synaptic)

PCSX [http://www.pcsx.net/]("http://www.pcsx.net/") also has a Linux version, has a more gnomey interface, and is open-source, unlike ePSXe, but again, you'll have to do a manual install.

I use ePSXe. I had some plugin issues with PCSX. But it's a matter of personal preference, so try both.
