### Post by wellery on 2005-08-08
The ubuntu guide is a good reference for setting things up. Here's how you add the codecs:

[http://www.ubuntuguide.org/#codecs](http://www.ubuntuguide.org/#codecs)

It may also be better if you install xine, totem-xine, vlc or mplayer. I use totem-xine.

To install these just add extra repositories as described here:

[http://www.ubuntuguide.org/#extrarepositories](http://www.ubuntuguide.org/#extrarepositories)

and then use synaptic to install the media player.
