### Post by e^(i*pi) on 2007-06-19
Thank you, that was exactly what I needed, everything is working perfectly now.  There was one hiccup along the way where things went terribly wrong, but I was able to reboot with no permanent damage done.  For anyone who is reading this trying to solve the same problem, here is what worked for me.

First, I installed the resolution altering tool
```
sudo apt-get install 915resolution
```
Then, I used this command
```
sudo apt-get install xserver-xorg-video-intel
```
Finally, I restarted X (control-alt-backspace) and selected the correct resolution under System->Preferences->Screen Resoltion

This might not work for everyone with this problem, but it worked great for me.  Thanks again for the help.
