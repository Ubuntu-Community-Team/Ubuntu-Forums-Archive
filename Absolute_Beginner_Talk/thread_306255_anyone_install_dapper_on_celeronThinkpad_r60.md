### Post by bodhi.zazen on 2006-11-25
Put the CD into the CD tray, close the door.

Reboot. It should boot to CD.

If not you will have to set up your laptop BIOS to boot from CD.

As the computer is booting you have to hit a key. It may be F2, F8, Enter, Esc, you will see it flash on the boot screen, something like to enter setup....

Every BIOS is slightly different. Look under the boot menu, change to boot form CD first, Floppy second (if you have a floppy), and HD last. Save and exit....

You will have to follow the on screen directions....

### Post by bodhi.zazen on 2006-11-25
> **Zenmind said:**
> Sorry Boddhi - you lost.... can you please explain more?

gparted is a program.

You can run it from a Live CD (Ubuntu, Gparted Live, others).

You can not manipulate a partition that is mounted. When you boot to HD you mount your root partition, as well as any listed inf fstab with the "auto" option...

Thus, IMO, it is easier to run gparted from a live CD then starting it after booting from hard drive....

Same with fdisk.....
Although fdisk will give you information at any time....
sudo fdisk =l will list all your partitions.

You will see, it should be very easy for you....
