### Post by Rocket2DMn on 2007-12-11
Is there a question in there?  If you're asking if it's possible to install a program or script to automatically turn your computer on, I don't believe it is possible.  You can always schedule the computer to shutdown or reboot, but when the computer is off, there is no power and no software running to recognize that it needs to do something (boot the machine).
The only way I know of to get a computer to turn on without pressing the power button is to setup Wake on LAN which some motherboards allow.  I don't think this is what you're looking for though.
