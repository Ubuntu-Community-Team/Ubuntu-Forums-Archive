### Post by arsenic23 on 2008-04-06
Sounds like your ubuntu system was told to use UTC when you set it up.  So more then likely every time you boot into linux your system clock is set to UTC.  Ubuntu then displays the time based off the notion that your clock is running UTC.  Windows doesn't do this, so you get an unmoddified UTC time when you boot into it.

I don't know how to change it on the fly though.

### Post by dorath on 2008-04-07
Try right-clicking on the clock and selecting Preferences.  From there you can un-check 'Use UTC'.

Same thing happened to me. :)
