### Post by p_quarles on 2007-07-24
1) Yes, install the updates. It's good to keep your system up to date.
2) It's not a web site, but it's a great source for themes:```
sudo apt-get install gnome-art
```
After you install it, you can open the program from System >> Preferences >> Art Manager. You can install themes, wallpapers, and splash screens. 
3) You mean Beryl and Compiz. Go to the Desktop Effects & Customization sub-forum to ask about that:
[http://ubuntuforums.org/forumdisplay.php?f=223](http://ubuntuforums.org/forumdisplay.php?f=223)
4) Just get the standard Adobe Flash player plugin, available in the Add/Remove Applications menu.
5) The problem is not fixed -- but the most you will need to is "force-quit" your web browser and then restart it. 
6) Audacity is available for Linux. ```
sudo apt-get install audacity
```
