### Post by oilchangeguy on 2007-03-04
if you burned the iso file to cd, you've made a nice drink coaster. if you have nero cd burning on a windows machine, find the option to burn iso files to cd, then you'll create a readable cd. also set it to burn at the slowest speed. after you do that you should be able to boot the cd and install ubuntu. (make sure the bios is set to boot from cd drive first)
