### Post by Cypher on 2007-05-10
They're talking about the command line. Within Windows, you can get away with never having to use the command line for anything.

However, with Linux, you can do most of the things without the command line, but there will be times when you'll have to type a few commands and if you take the time to learn, the command line is considerablly more powerful..

### Post by tommcd on 2007-05-10
If you really want to learn to use linux, you should learn basic command line skills. It will help you get the most out of your system, and will pay off well if you ever try another distro, as the command line stuff is (more or less) consistent no matter what distro you use.
For now though, start here with ubuntu:
[http://www.psychocats.net/ubuntu/](http://www.psychocats.net/ubuntu/)
Then once you get started, here is command line 101:
[http://www.linuxcommand.org/learning_the_shell.php](http://www.linuxcommand.org/learning_the_shell.php)
It might look daunting at first, but it's not really that hard.
