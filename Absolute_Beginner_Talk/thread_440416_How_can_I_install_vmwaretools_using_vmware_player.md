### Post by deepwave on 2007-05-11
VMWare tools only comes with VMWare Workstation if I remember correctly.  VMWare player only lets you play pre-made images.  If you had Workstation, and installed the vmware tools through it, then probably you could play a vmwaretool enabled image in Player.  Not sure if you would gain anything from having vmwaretools when running Player though.

### Post by Acksys on 2007-05-11
You can install vmware tools through the vmware player by grabbing them from the vmware workstation ISO. You'll have to register for the VMWare workstation free trial to download the ISO first.

Here are the instructions I used:

[http://www.brandonhutchinson.com/Installing_VMware_Tools_with_VMware_Player.html](http://www.brandonhutchinson.com/Installing_VMware_Tools_with_VMware_Player.html)

VMWare tools makes the image run MUCH better.

### Post by jhansonxi on 2007-05-11
When you install the tools with Workstation it mounts them as a CD volume.  Just dump the virtual CD contents to an archive and move it to your host system so that you can install them without Workstation in the future.
