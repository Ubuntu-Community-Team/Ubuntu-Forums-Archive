### Post by mikeyphi on 2008-04-01
> **babisgamatos said:**
> I try to connect to the internet but I don't know where to go to activate a Lan port.When Firefox says it is in Offline Mode means that I am not connected?

In Firefox select File - uncheck work offline.

If still no connection - open System/Administration/Network and confirm that wired/wireless connections are enabled.

If still no connection - check your ISP/router and then ask again!
