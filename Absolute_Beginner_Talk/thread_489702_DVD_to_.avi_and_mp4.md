### Post by dfreer on 2007-07-02
Just install acidrip, it is pretty self-explanatory. It defaults it's video output as .avi, and you can change the codec to Xvid if you wish, although the default settings work well (I normally increase the resolution and file size myself for a higher quality rip).
dvd::rip does just about everything, however it takes quite alot of reading to use it as it has a TON of features. It should be able to output xvid or divx, although I'm not sure about .mp4
ffmpeg and mencoder should be able to re-encode the .avi files to .mp4 if you need it to, although they are CLI based and so you will need to google a guide on how to use them.
