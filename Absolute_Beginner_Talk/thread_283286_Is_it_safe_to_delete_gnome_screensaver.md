### Post by CatKiller on 2006-10-24
> **cairnsww said:**
> Thanks - does that mean that installing xscreensaver just installs the software and does not actually install the screensaver itself? (You can tell I am trying to get hold of some concepts here).

Installing xscreensaver installs xscreensaver.

You still have to make it run at startup yourself.

You don't want to have two screensavers running at the same time, so you'll either have to uninstall one, or turn it off. You can either uninstall gnome-screensaver, which will uninstall ubuntu-desktop IIRC, then re-install ubuntu-desktop before you upgrade, or you can clear a tick from a box.

Guess which I prefer :cool:
