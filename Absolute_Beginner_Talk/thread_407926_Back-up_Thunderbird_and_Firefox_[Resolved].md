### Post by aysiu on 2007-04-12
Your settings and emails are all inside your home directory.

Thunderbird is either /home/iopo/.mozilla-thunderbird (if you're using Ubuntu's Thunderbird) or /home/iopo/.thunderbird (if you're using Mozilla's Thunderbird)

Firefox is /home/iopo/.mozilla

Any file or folder with a . in front of it is hidden. If you want to see hidden files in Ubuntu or Xubuntu, press Control-H. If you want to see hidden files in Kubuntu, press Alt-V, then H.
