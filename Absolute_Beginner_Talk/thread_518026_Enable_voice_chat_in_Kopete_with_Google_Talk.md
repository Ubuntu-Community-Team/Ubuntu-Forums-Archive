### Post by ravenwere on 2007-08-05
Hi Helino,

If I understood your question correctly.

I use Kopete to chat using googletalk. There is a howto on the google account website somewhere.

If you have installed Kopete using the package managers (synaptic or adept) or apt the dependent files should have been installed automatically.

To set up your google talk account on Kopete. Go Settings->Configure->Accounts->New.
Choose Jabber in the account wizard. For the jabber ID and password use your own full google email address (????@gmail.com) and add in your own password. Complete the wizard and you should be up and running.

Hope this helps,
