### Post by FrozenFox on 2008-04-18
Edit: Curse you fast posters.

Yes, there's a way to make your own livecd with your own stuff. However, Wine is not intended to (and will probably fail miserably 98% of the time trying to) install drivers users want. As such, I ---highly-- doubt doing that would work. Wine is meant only to install and run actual programs and games, not system stuff. IE: You are more than likely going in the wrong direction.

You are better off giving your friend's modem model info and asking how to get that working. Using the normal ethernet plug most modems come with is practically foolproof, but if you must use USB for any given reason, please post the model information.

 I'd suggest instead that you find .DEB files for the modem drivers and its dependencies (if there are any that dont come with ubuntu, which i dont imagine would be the case) if they are available and putting them on a separate cd (though I don't think making your own repository cd will be necessary), as the above people suggested, then just install them after ubuntu is installed.

FYI, creating a live cd info can be found here [http://howtoforge.com/ubuntu-linux-mint-livecd-with-remastersys](http://howtoforge.com/ubuntu-linux-mint-livecd-with-remastersys)  but it isn't really for those who have no clue what they are doing.
