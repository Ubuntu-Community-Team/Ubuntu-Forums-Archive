### Post by Severun on 2007-12-17
I actually have the same card you mention the Mobility ATI X1400

I've gotten dual head monitors to work, but it's a bit of a pain, and there appear to be some problems with the current fglrx driver so I can't run the full-on Compiz 3D desktop.  Here's how I got it to work.

The other note is when logging in and choosing a session I choose "Gnome Fail Safe".  If I choose anything higher, it will run OK for a few minutes, then the session semi-hangs, i.e. the mouse moves, but I can't grab Windows and drag them, etc.  Hopefully this will be fixed soon.  I haven't done much testing to see if I can run w/ Compiz when not using dual head monitor, but it might work.  I am using the BigDesktop feature of the fglrx driver to make both screens act like one desktop.

When I want to switch form Two monitors to one, I just use System->Preferences->Screen Resolution from the desktop menu,  I use 1280x800 when I want just the laptop and 2560x1024 when I want both the laptop and my secondary monitor (My monitor is a bit taller than my laptop screen).

Make sure you stay away from trying to use PairModes in your xorg.conf, it causes the X server to crash with a segmentation fault.

Here is my xorg.conf file, there is a bit of extraneous garbage in their but it seems to work. With BigDesktop.

My config is:
ATI Mobility Radeon X1400
Centrino Duo CPU
Toshiba TravelMate 4670
2gb Ram
Running Laptop screen at 1280x800 and monitor on expansion port at 1280x1024
