### Post by PhatBoy on 2007-07-01
Well I have my Myth box all set up and ready to go... but just a couple of little issues left. The one up for discussion here is to do with the backend.

On boot the backend does not appear to be starting correctly. The frontend gives the "cannot connect to master backend server" error.

The backend is in the list of running processes so it has started.

Restarting the backend with

```
sudo /etc/init.d/mythtv-backend restart
```

results in everything working perfectly. I would just prefer not to have to do this every time I start the machine up.

Cheers.
