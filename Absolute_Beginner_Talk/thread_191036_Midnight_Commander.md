### Post by rcarring on 2006-06-07
This is a text mode file manager that you run from terminal with the command 
```

mc

```

If you need to runas root then
```

sudo mc

```

I like to use this to copy files from one place to another quickly, to view conf files, amongst other things.

Its also useful to have if X bombs out and you are at the command line trying to figure out how to look at files buried five levels down.

Two ways to get this:
```

sudo apt-get install mc

```

OR

Use synaptic, and find mc listed in the All section.

I mention this program as when i was a linux newbie many years ago, it was a big help to me.
