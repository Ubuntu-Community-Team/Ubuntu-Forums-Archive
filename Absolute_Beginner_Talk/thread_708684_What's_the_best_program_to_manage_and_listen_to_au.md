### Post by odiseo77 on 2008-02-26
I remember having played video and audio streams from the BBC on Ubuntu. It depends on which player you've selected when you first tried to play the stream (Windows Media Player or Real Player). If I'm not wrong, the procedure I took was this: Installed w32codecs and the mozilla totem plugin (or mozilla xine plugin, depending on which one you want; I can't give you the exact package names because I'm not on Ubuntu now, but you can search on Synaptic). After that, the stream played fine. You can also install RealPlayer from it's site (or from some repo, if it's available), and see if it works.
