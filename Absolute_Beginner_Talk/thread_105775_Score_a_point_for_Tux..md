### Post by GoldBuggie on 2005-12-19
If you are into the latest games and play alot then linux is not for you. There are ways to use some windows games but it isn't the most pleasent of experience. There are some nice linux games thou. Not many but still.

If you use your computer for surfing, email, chatting, writing documents then linux has some really nice things for you.

If you don't have the time to get accustumed to a new OS. New shortcuts, command, filesystem hierchy etc ... then linux is not for you. If you are just a bit intrigued then linux is for you.

If you want to have good access to neat eye-candy for the desktop or the ability to just change the color of your theme then linux is for you.

If you do any kind of development(programming etc) then there is nothing else then linux.

If you want to feel like you are using something legal and free of charge then this is the place. If you actually count the amount of money all the software costs that ppl used. yeeez..it is alot. I know many don't buy everything, but it is a nice feeling coming away from something that actually more or less is unusable without buying extra software. Ubuntu comes preinstalled with all those thing. I'm talking about CDburner/write, create pdf's, image viewer, scan software etc.

If you want a really nice, friendly and good place for asking questions and sharing knowledge then this forum(& distro) is for you. (It is hard to find a forum which is this dedicated) Almost all your problems will mostl likely have been solved already so remember..the search button is your friend.

Remember that linux is still not as major as windows and there might be some hardware/driver issues. One can solve almost anything but it can take time. If you are lucky then everything will work almost directly at the beginning. My stationary did that, my laptop was a whole other thing. If you are installing it on a laptop then be prepared to tweak. Or buy a linux approved laptop:D

Only downside about changing is that I feel that some software is still a bit premature and needs some work. But most are damn close.

One thing that you need when using linux is internet. This is important since all upgrades and software come from there. But the ability to use a program which has a database of software which by a click of a button it downloads, installs and configures is a GREAT TOOL.

Well I can take up more things but since I don't know what you are looking for. But I do recommend giving it a shot. Linux has matured ALOT the last 2 years and with you and others coming to the community it will only grow stronger and hopefully better.

Hope you enjoy it.
