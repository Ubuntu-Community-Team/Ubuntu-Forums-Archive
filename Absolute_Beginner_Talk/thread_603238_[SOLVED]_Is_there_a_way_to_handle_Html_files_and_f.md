### Post by blastus on 2007-12-16
> **thelatinist said:**
> Why would anyone want their file manager to move folders without being told to?  That's idiotic.

IMO the feature is intrusive...

1. It makes the assumption that a file called x.html and a directory called x_files are related--this could be a WRONG ASSUMPTION (i.e. you could delete the x.html file and it would automatically delete the the x_files directory)

2. As far as I know renaming the html file or folder breaks the relationship--Windows Explorer does not know how to fix the links in the html file so there's not much point to the feature.

3. As far as I know it only works in Windows Explorer.
