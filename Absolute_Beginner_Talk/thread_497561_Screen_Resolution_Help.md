### Post by Pragmatist on 2007-07-10
> **rookshop said:**
> No. I did not backup anything. I just followed Johans advice word to word.

That is not a good policy.  In the future, always backup any configuration files you are changing.  It is easy:
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak
```

You can name the backup file anything you want, but it is best to just add the word backup to it, or append with .bak

You can cut-and-paste your posted xorg.conf  Then copy the file to your main system (how are you connecting to the internet right now?  On another computer?).  You can access the problem computer with a LiveCD and, if you make a new xorg.conf by cutting-and-pasting, you can bring it over to the problem computer using a floppy.  In sum, you need two things:

1.) your old /etc/X11/xorg.conf (which you get by cutting-and-pasting from here)
2.) a LiveCD to access your broken system.
