### Post by energiya on 2007-02-14
1) It depends on what you're still using windows for, and what your future plans for linux are. If you have windows on the 1,8GHz machine, you could just live it there, and install Linux on the slower machine. That, of course, will make it run slower (don't even think of installing a KDE based distro). Try Ubuntu (+) or even the lighter XUbuntu (or Zenwalk but a little hard to digeste for a newbie).

2) If command line isn't a problem for you and you are willing to learn, it will not take much to get used to it. Soon you will forget to start the windows machine.
