### Post by irish_flu on 2007-08-26
> **Controlpanel said:**
> Yea ok thanks. I think thats what I will do. Learn about computers more. Could you maybe tell me a place to start as far as using the internet on Linux goes?

If you want an easy "no commitment" way to try out Linux without worrying about setting up a dual-boot, and without worrying whether or not your hardware will be easily detected (which seems to be the problem you were having earlier), you might try a "virtual machine".

What's a virtual machine?  In this case, it's software that lets you run (if you want) entire operating systems in a window inside another operating system.

One company that provides (free of charge) Virtualization software is VMWare.  Their free player can be downloaded here:
[http://www.vmware.com/products/player/](http://www.vmware.com/products/player/)

Then, you can go and download an already-built Linux environment to run inside that player.  Kind folks have already created many distributions, including some perfect for the new user (including Ubuntu).  Here is a link to their "full operating system" pre-built virtual machines (which they call "appliances"):
[http://www.vmware.com/appliances/directory/cat/45](http://www.vmware.com/appliances/directory/cat/45)

Boom, you're running Ubuntu inside a window that you can minimize and rock right back to Windows XP (without rebooting or anything).  It won't perform quite as well as loading it on "bare metal", but it's an excellent way to try out Linux.  The other nice thing is that if you really screw it up (and mistakes are always part of learning), you can just keep yourself a copy of the zip you downloaded, so that you can revert back to the way you got it.

All that, without spending a buck.
