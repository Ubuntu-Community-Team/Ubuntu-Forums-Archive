### Post by Hendrixski on 2007-11-03
If it's a good app then it's cross platform.  If it doesn't work on Mac and Linux just simply don't buy it.

In fact, there are many VPN and anonymous SSH solutions already included in Ubuntu, so you don't have to spend money on that.  After all, having everything you need already included is what makes a great operating system great, right?

If you go through Synaptic and search for VPN you'll find a few that meet your needs.  Or if you're a command line junky type in "apt-cache search VPN".
