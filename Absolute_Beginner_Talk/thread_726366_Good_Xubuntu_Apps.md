### Post by Darganot on 2008-03-16
where to start...

Okay perhaps your most useful program is called "Synaptic Package Manager" or just synaptic.  It's probably under 'Administration' in the menu.  You can use this program to install other programs from the internet.  You also use this to update all the packages on your system (essentially update your machine).  You'll need to put your password when you start because you will be modifying your system.  Anytime linux asks for your password, you're making SERIOUS system changes so make sure you think before you click apply or OK.  So once your password is in click the arrows at the top of the screen where is says 'update'.  This will update the package lists, or the actual list of programs you have to choose from.  This needs to be done first so you know you are selecting the most recent program out there.  Then click the button with the computer next to it 'upgrade'.  This will select all the packages on your machine which have possible updates.  (packages is just another word for programs).  After clicking ok to select those packages click the check mark next to it called 'apply' to install all the packages that are marked.  You can select packages before or after you apply, but if you do it after you will have to apply again.

After clicking apply your programs should be downloaded and installed, automatically.  That's it!  You should see the program you installed in your menu.  If not you may need to type the name into a terminal to get it to run.

Synaptic has a search function you can use but I still find it interesting just to browse the repositories in different categories.  You never know what port of a game you might find or a new media player to try out.

You can also add more repositories, or sources, to the lists of packages that synaptic gets.  There's many, many guides floating around on how to do this.
