### Post by Yasumoto on 2007-06-15
I got a Dell E1505 during October of last year (my first year at college), and I'm really happy with it. If you go with Dell, head over to [dealcatcher]("http://www.dealcatcher.com/dell-home-coupons"), because they've got some pretty sweet coupons. Make sure you get an Nvidia graphics card (won't be an issue if you go with a laptop with Ubuntu pre-installed), since ATI has driver problems. (I found that out the hard way) :-|

But except for gaming, I actually like using Ubuntu so much more than Windows, so long as you keep an open mind, and read a lot. And the fact that you get one without having to even bother with Windows is pretty nice.
