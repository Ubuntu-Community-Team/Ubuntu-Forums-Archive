### Post by Samurai Penguin on 2008-03-29
65 posts and you never selected a theme? Well, I suppose it's possible.
So ...

Go to  [www.gnome-look.org](www.gnome-look.org) and check out some Themes
and download all you want for free.

open Terminal and copy/paste each line one at a time (press
the enter key after each entry)

sudo apt-get install emerald
sudo apt-get install libemeraldengine0
sudo apt-get install compizconfig-settings-manager
sudo apt-get install compizconfig-settings-manager python-compizconfig

Close Terminal window (type exit and press return)

then ...

Right click on the Desktop > Change Desktop Background > Visual Effects

If you can select NORMAL (or any of the other two) you can
then run Emerald (I think it's in System > Preferences) to change
window boarders and other neat stuff and you can also change
THEME when you Right click on the Desktop > Change Desktop Background > Themes and import the Themes into Emerald.

Highlight a Theme, place the mouse cursor on the top bar of
the Emerald window and leave it there. Then use the up/down
keypad arrows to select a Theme. It should change right away.

If you need more help, post Q's here
