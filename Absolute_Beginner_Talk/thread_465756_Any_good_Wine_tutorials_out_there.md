### Post by Brightbelt on 2007-06-06
I'm not sure I can speak to your bluetooth/phone issue. From what I understand about Wine, it's mainly for running software. When you get into hardware and drivers, that's another area entirely. 

I'm a Noob too bit I think I read somewhere that there are some drivers and stuff that Wine can run but it's very tricky and touchy. 

As for using Wine, I put my (windows-oriented) software CD into my CD-rom (say Photoshop 7 which I installed recently). I look for the Setup.exe file and right-click that file and choose "run with another application" and then the window comes up listing all my programs. At the bottom, there's an option to type in your own custom command. I simply type "wine" and click OK.  "Wine" may already be on the menu as an option for you, but read on....

Sometimes this does it and wine starts the install right away. Other times, I have to right-click the setup.exe file again, but then every time, Ubuntu "learns" what I did before and the option is there at the top of the menu 'Open with "wine"'. 

That's it as far as I know. Photoshop 7 has been running smoothly for the most part. It did kick me out of the program once recently which was annoying. But my PS CS3 froze in Vista the same day. So go figure. 

I hope this helps,...Frank B.
