### Post by Brunellus on 2005-08-26
[QUOTE=locomatrix69]Where can I get java for the lime wire[/QUOTE]

read the relevant instructions at [www.ubuntuguide.org](www.ubuntuguide.org)

or alternatively, check the restricted-formats wikipage at

[https://wiki.ubuntu.com/RestrictedFormats?highlight=%28restricted%29#head-55315677ab8f9890825549fa2ecebdde4bc68087](https://wiki.ubuntu.com/RestrictedFormats?highlight=%28restricted%29#head-55315677ab8f9890825549fa2ecebdde4bc68087)

incidentally, you don't need limewire.  Limewire is only a client for the gnutella network;  any gnutella client will do.  I use gtk-gnutella, which is in the universe repositories.  find it on synaptic, or open up a terminal and 

$sudo apt-get install gtk-gnutella
