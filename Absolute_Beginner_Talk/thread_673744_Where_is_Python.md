### Post by Majorix on 2008-01-21
Python is already installed on your system.

If you are going to program using it, invoke it by typing
```
python
```
into the terminal, as advised.

If you have a script you want to run with python, type
```
python /path/to/script
```
into the terminal.

If neither of these mean anything to you, then just leave it be. Python will be invoked by Ubuntu whenever it is needed.
