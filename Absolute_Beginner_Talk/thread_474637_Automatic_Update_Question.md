### Post by Cypher on 2007-06-15
All of the package updates show up in the list in the main part of the update manager which details about them. But when there is a full version upgrade, you will just see a button on the top of the window which says "Upgrade Ubuntu", clicking that will upgrade the whole system.

As for upgrading. If you do that when Gutsy comes out, I recommend removing any proprietary blobs (they usually aren't expected and thus sometimes cause trouble).
