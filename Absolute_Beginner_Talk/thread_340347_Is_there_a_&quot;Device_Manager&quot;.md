### Post by InTheFlow on 2007-01-17
I've read that one of the best things about using the Live CD before actually installing it is that you can find out if your computer's hardware will work with Ubuntu.

My question is: How do you find out if everything is supported once the CD has gotten you into Ubuntu? Is there a 'Device Manager' type place in Ubuntu that would show a list of things that are not recognized or working properly?

If so, where?

---

### Post by steven_mckenz on 2007-01-17
If you go to System/Administration/Device Manager, it will show you a list of the hardware, and you can ensure that everything is detected and working properly.
