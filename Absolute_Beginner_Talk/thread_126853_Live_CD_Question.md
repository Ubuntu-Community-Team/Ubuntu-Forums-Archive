### Post by GreyFox503 on 2006-02-07
As far as I know, yes.  You can download new programs and change the operating system in most normal ways while the livecd is running.  Although I haven't installed anything *really* big while running it, I have installed several other programs, both from the repository and third-party.

As long it whatever it is doesn't need a reboot (and nothing you download should) you can just keep using the system.
