There is Picasa for Linux which you can get here, but it uses wine which I hate waiting for to load (once it is up and running everything is pretty fast though). You can also use a number of other native linux programs that can import pix (some are KDE....like digiKam which is my favorite).  DigiKam (really KDE) also handles things like modifying the timestamp and datestamp for EXIF & file data. There are others which can be decent, like gthumb and f-spot.
