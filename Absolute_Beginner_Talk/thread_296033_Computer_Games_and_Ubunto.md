### Post by Perfect Storm on 2006-11-09
Windows game are made specfic for windows, so it's natural it runs better on windows (if you see past all the programs that slow windows down that runs in the background).
Populare FPS games does have native installer for linux, like thte UT series and quake series. (also ut2007 and quake wars will be released for linux) <insert big w00t smiley here>.

As other members stated here, some windows game can be run via cedega, wine etc. but don't get you hopes up. Though very populare games like WOW and GW is possible in Cedega and (Wow in wine with a bit of tweaking).

I know alot of serious gamers which do dual boot where they use windows as gaming platform and then everything else on linux OS. Perhaps you should considering it.

If you're into Dosgames, you can run it perfect in Ubuntu/linux with Dosbox and ScummVM for old windows adventure games and alike.
