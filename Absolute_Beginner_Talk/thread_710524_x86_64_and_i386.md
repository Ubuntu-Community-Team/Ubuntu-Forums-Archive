### Post by Rocket2DMn on 2008-02-28
i386 is for 32 bit processors and will also work on a 64 bit.
x86_64 is for 64 bit processors ONLY and will not work on a 32 bit system.

If you tell us your processor make/model we can tell you which you should use.
