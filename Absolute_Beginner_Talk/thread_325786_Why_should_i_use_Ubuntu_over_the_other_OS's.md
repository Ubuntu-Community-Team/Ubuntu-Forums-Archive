### Post by Quillz on 2007-01-01
Ubuntu Linux is comprised of entirely open source software. Many of the included open source applications, such as Mozilla Firefox and OpenOffice.org are comparable to many bundled Windows applications, such as Internet Explorer and Microsoft Office.

One of the biggest advantages is that Linux distros tend to have very low minimum system requirements, meaning that even older machines can run modern Linux distros fairly easily. Windows Vista, for example, has minimum system requirements that most newly manufactured machines will match, but older ones may have some trouble. On those older machines, you may prefer to run Ubuntu Linux as opposed to an older version of Windows, such as Windows 2000 or Windows XP.

My advice is to try out Ubuntu Linux, and see what you like and dislike about it. If it doesn't work out for you (and it may very well not,) then keep using Windows. But for me, I can work productively on Linux just as well as I can with Windows, simply because Linux does what I need it to do in the manner that I need it to be done. Windows is a great OS in its own right, but sometimes it can be frustrating to use. Linux, I feel, has more of a "user-first" policy than the "Microsoft-first" policy that Windows can sometimes seem to force upon you.
