### Post by theratster on 2006-07-27
Two things.

If you want to install kubuntu completely, install the "kubuntu-desktop" item.

... and since KDE is already installed, you can switch back and forth at your login screen. There should be a sessions button or tab: click it and select "KDE" for the next login and login with the same username. You should now be in a KDE desktop environment.
