Unix is not Linux but it's close.  Some of the commands for a particular flavor of Unix will not work unless you add some aliases.  

For instance, if I remember correctly, on HP-UX (HP's version of Unix) you could type ll for a long listing of the files and directories.  However, with an unmodified Ubuntu install you need to type ls -l.  Little things like that can make it difficult to learn Linux using a Unix book.

You might want to pick up a book that deals directly with Ubuntu.  Once you learn Ubuntu, then return to the Unix book and you should be able to see the differences.

Hope that helps.
