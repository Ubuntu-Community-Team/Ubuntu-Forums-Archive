### Post by taurus on 2007-10-31
DO NOT unpack the iso image before you burn it.  Leave it as one large iso file and burn it with your burning software as an image file, not a data file.

[https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28burn%29%7C%28iso%29](https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28burn%29%7C%28iso%29)
