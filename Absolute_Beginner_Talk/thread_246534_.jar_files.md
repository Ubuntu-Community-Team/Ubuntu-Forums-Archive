### Post by Bloch on 2006-08-29
A .jar file needs java installed to execute it. The same .jar file will run on an operating system that has java installed. Ubuntu automatically associated .jat files with the java runtime environment (the 'java virtual machine') so all you have to do is double click on it. 

From the command line however you have to enter java ***.jar
and the program will run.

There are two java machines: the "official" Sun one, and the opensourse Blackdown one. I have not heard of any problems with either version and I think they are just aboput 100% compatible.

Open synaptic. Search for 'java'
Install this:   j2re1.4

When it is installed type java -version to see if it is there. Also do a search for "jars" to check it all works - this page has some:
[http://mathsrv.ku-eichstaett.de/MGF/homes/grothmann/java/](http://mathsrv.ku-eichstaett.de/MGF/homes/grothmann/java/)
