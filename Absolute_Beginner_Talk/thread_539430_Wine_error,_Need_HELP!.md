### Post by popch on 2007-08-31
If I read the description of your problem correctly, you are running Ubuntu on your desktop and trying to install a driver made by Intel. That drive is made for Windows. It is not possible to install it in Ubuntu Linux, even when using Wine. 

Even if it was possible (which is not), it would not help you in any way because the driver would still not be available to you when running Windows. 

Why do you attempt to install that driver? Is it because your laptop computer will not boot when a video cable is plugged in?

When your computer boots fine without the external monitor and does not boot at all when an external monitor is connected, I would assume that either the monitor or the cable was defective. In that case, installing a driver would not help in any way.
