### Post by aysiu on 2005-12-16
[QUOTE=Deaf_Head]Like, in windows you can say:

Windows folder = OS stuff
Program Files = Default Directory of many programs
Documents and settings = Useless XP setup of stuff[/quote] Sure.
/home/deafhead = where all your documents and settings live
/boot = where the boot configuration lives
/etc = random config files
/usr = themes, applications, icons, and all sorts of other fun goodies
the rest... = OS stuff

> 
Also, is there a way to update firefox through ubuntu or is taht somethign i gotta download and deal with seperatly? Update in general? Just ```
