### Post by spur on 2007-09-12
First- don't presume any similarities between xp and Ubuntu.
Second- Normally you should have no trouble if you dual boot as you only boot into that os so the other one should not effect the one you have booted into.
Third- You may find Ubuntu runs faster than xp, but it depends on what you are doing.
Fourth- When using Ubuntu you are not using Windows.So don't expect things to be in the same place or behave the same. You have to learn it the same way you learned windows.

### Post by spur on 2007-09-24
Have you done this yet?
If not the first thing is to install your windows os before your linux os.
Second if you haven't done so make sure you have enough free space for your second os. Including enough for your swap partition.
Then install you linux os. You can use the installation process to create and format your linux partition and it should detect your win partition and os and create grub entries to allow booting into either.
It should go smoothly but if you can't get into one of the os then ask here as that problem can be fixed.
