### Post by dbott67 on 2007-01-24
1) If I download and install Ubuntu will it take over XP totally so that I can just use Ubuntu (much like a reinstall)?

[COLOR="Blue"]The Installation CD is a "Live" session, which means it runs completely off of the CD without touching your hard drive.  If you choose to install it to your hard drive (by clicking the INSTALL icon on the desktop) you will (potentially) destroy all existing data on your hard drive (Operating System, files, MP3s, videos, etc.)[/COLOR]

2) Can I still use applications such as Winamp/real player etc?
[COLOR="Blue"]
Yes.  There are plenty of apps that can play these types of files (although, they're not called Winamp ;).  Try Beep media player or Amarok).  Keep in mind that Ubuntu supports the FLOSS philosophy and that certain proprietary codecs and applications are not included in the default installation (such as the capability to play MP3 files and DVDs).

Fear not, as there are programs such as Automatix ([www.getautomix.com](www.getautomix.com)) that can make the installation of these codecs and applications such as "Real Player" & "Beep" a breeze.  Just install automatix, select which programs you want, sit back and have a coffee and in no time you'll be singing and dancing.[/COLOR]

3) Will I lose all my MP3's by installing Ubuntu?

[COLOR="Blue"]You might.  [/COLOR]**[SIZE="5"][COLOR="Red"]BACK UP YOUR FILES TO CD or DVD first.[/COLOR][/SIZE]**

[COLOR="Blue"]Ubuntu can dynamically and non-destructively re-size your XP partition to make room for itself.  Then, you could leave the MP3s on the drive and access them from Ubuntu (plus, you can always boot back into Windows if you desire.  

Personally, I have never had a problem using the non-destructive re-partitioning, but things sometimes can and do go wrong, so backup, backup, backup! :)

If you wish to completely remove all traces of Windows (after backing up, and verifying existance of said files on backup media), then insert the CD, boot into the Live Session of Ubuntu, double click the INSTALL TO HARD DRIVE icon and in 15 minutes or so, you'll be Windows-free.

After rebooting into your new Ubuntu installation, go get Automatix and install your desired applications.[/COLOR]

-Dave

Although Automatix has been mentioned a couple of times and is a fast and easy way to add propiatory (sp?) codecs and other things, you may want to enable the same capability in other ways.  Automatix may create some issues for you down the road.  There's plenty of threads on this in the forum. 

Try running the live cd first and see if everything works, ie wireless and whatnot.  You may have to do a little configuring.  Keep in mind Ubuntu & Linux in general are much different from the Win environment.  Some of what you currently know will carry over but you'll have to get use to new commands and ways of doing things like installing programs.  A little patients goes a long way and there are lots of people here willing to lend a hand.  Search for your issue(s) on the forum, read then ask.  

I run a dual boot, xp & Ubuntu Dapper LTS for work & play.  XP is there for games and Ubuntu for everything else --- almost.  I have a couple of propiatory things for work that demand Windows.  So for those few things I don't bother booting into the other partition.  I just installed windows in VMWare Server and open it (in a window) within Ubuntu. Ofcourse, you have to have a valid CD of windows to install it.  EBay could be your friend there.
