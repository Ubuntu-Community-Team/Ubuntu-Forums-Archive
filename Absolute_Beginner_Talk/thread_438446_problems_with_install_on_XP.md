When you use a LiveCD, you have the ability to boot from the CD without changing your current OS setup.  I would download a LiveCD image which will be an .iso file, burn it to the CD, and boot from the CD.  You will have a number of options to choose from.  If you just want to look around Ubuntu, choose the first option, which is something to the effect of "Use Ubunut LiveCD" (the verbage may be different, I cant remeber exactly).  IF you decide that Ubuntu is something that you want to use permanently, reboot and at the Ubuntu boot menu, choose the "Intall Ubuntu" optiona and go from there.
