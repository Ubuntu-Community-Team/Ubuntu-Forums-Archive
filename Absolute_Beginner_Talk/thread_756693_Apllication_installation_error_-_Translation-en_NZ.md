### Post by Pasty-Flawss on 2008-04-16
I was googling for hours looking for the answer to this problem who many people encountered and no one that i saw actually posted a conclusion to what they did to fix this problem so i would like to quickly state what i did.

I went into System Sources (System > Administration > Software Sources)
Then i clicked drop down menu called Download from, then i chose other. Then i clicked Best server available(or something on those lines). then it pinged all the countrys and chose one from me. Then my problem was fixed.

NZ Is now free from windows! :)
