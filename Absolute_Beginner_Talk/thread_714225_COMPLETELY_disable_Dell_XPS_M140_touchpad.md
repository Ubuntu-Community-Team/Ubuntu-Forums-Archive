### Post by OffAxis on 2008-03-04
Interesting. I thought that would've done it.
Okay.
Uncomment that section and try adding this line to your touchpad's section in xorg.conf
```
Option "TouchpadOff" "1"
```
Note: That's a number one.

```

 Section "InputDevice"
         Identifier	"Synaptics Touchpad"
	 Driver		"synaptics"
	 Option		"SendCoreEvents"	"true"
	 Option		"Device"		"/dev/psaux"
	 Option		"Protocol"		"auto-dev"
	 Option  	"SHMConfig"   		"True"
         **Option "TouchpadOff" "1"**
# EndSection

```

 I think that'll do it. The TouchpadOff parameter can take a "1" (Disable) "0" (Enable) or "2" (disable scrolling and tapping) value.

I know you can use (from the command line)
```
synclient TouchpadOff=1
```
to turn it off temporarily (for that session) if the above doesn't work for some reason.
