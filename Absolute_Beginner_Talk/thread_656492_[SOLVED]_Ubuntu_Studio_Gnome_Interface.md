### Post by Lacrimstein on 2008-01-02
Yes, the only difference is software.

go to gnome-look.org, and search for ubuntu studio theme, then just install it by dragging the downloaded file into the Appearance dialog (System->Preferences->Appearance). Then just select the correct window border/widget themes in other tabs of the dialog

### Post by shafin on 2008-01-02
You can get the the whole ubuntu studio look by going to synaptic and searching for "ubuntustudio". install **ubuntustudio-gdm-theme ubuntustudio -icon-theme ubuntustudio-look ubuntustudio-theme **and** ubuntustudio-wallpaper**s packages
