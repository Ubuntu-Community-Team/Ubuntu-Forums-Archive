### Post by DarkOx on 2007-10-11
My Dad is using Ubuntu (well, Kubuntu) as he picked up an old laptop from work, and it didn't make sense to drop 250 bucks for a home license of Windows.

Internet, Photos, and music are all easy, but make sure you install all the restricted stuff for him (java, flash, mp3 codecs), since he'll probably need it. For a Quicken replacement, there are options (gnucash, kmymoney, etc.) but I doubt they'd be anywhere near as robust as a commercial package. According to [wine]("http://appdb.winehq.org/appview.php?iAppId=107") it looks like some versions of Quicken run better than others. Alternatively, you could try [CrossOver Office]("http://www.codeweavers.com/compatibility/browse/group/?app_parent=2318;") but it costs money.

Realistically, there's no reason not to try Linux, since if it doesn't work you'd be looking at buying a Vista computer anyways. So why not see if it works? Just remember to back up your data first.
