### Post by aysiu on 2006-06-08
FAT32 can be accessible by both, but so can Ext3 if you use [http://www.fs-driver.org](http://www.fs-driver.org)

I have Xubuntu, Ubuntu, and Kubuntu on one partition, and they total about 2.8 GB.

Read more here:
[http://www.psychocats.net/ubuntu/partitioning.html](http://www.psychocats.net/ubuntu/partitioning.html)
