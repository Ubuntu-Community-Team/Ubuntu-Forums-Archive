### Post by 3rdalbum on 2007-04-01
If you just click on "Clearlooks" in the list, it should update everything to use the Clearlooks theme. If it doesn't immediately, try logging out and then logging back in again. Clearlooks is already installed, as is every theme listed in the panel.

The "install theme" button is for themes that you've downloaded off the Internet, for example from [www.gnome-look.org](www.gnome-look.org).

### Post by carloslosgrande on 2007-04-01
Hi Counterpoint.
In the Theme Preferences option box where you find 'clearlooks' it is already installed and you can make changes to it relating to borders and icons in Theme Details.
If you want more themes you can visit art sites such as gnome art and get more themes that you can download and then just drag into the Theme Preferences box, where a dialogue will ask if you want to install it.
