Hi,

With all that was posted above, I'm guessing you've already managed to install Opera... But, if not, I'd like to point two things out.

1. The Opera website does have a help section that explains to you how to install the .deb file you've downloaded... Just look around a bit, remember that the install instructions are probably for Debian-based Linux distros in general... Ubuntu is Debian based.

2. There is actually a how-to for installing Opera, here on the forums. Check this link, it shows you all the HOWTOs on the forum...

[http://ubuntuforums.org/showthread.php?t=53050&highlight=installing+opera](http://ubuntuforums.org/showthread.php?t=53050&highlight=installing+opera)

It should even have something on your ATI-card.

I know Opera isn't available through the Ubuntu repositories, but before you go and download more stuff off of the net.... Please check if you can't use Synaptic to install the application. It will save you a lot of hassles in the future.

Hope that helps.
