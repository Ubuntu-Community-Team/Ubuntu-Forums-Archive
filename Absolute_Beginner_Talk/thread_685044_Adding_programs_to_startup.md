### Post by eapache on 2008-02-01
To find out the filename of a program that is already in your normal menu, go to System>Preferences>Main Menu, and find the item. Then right-click and select properties (or edit, not sure), and the box that pops up should contain the program name. For pidgin it is just pidgin, but I'm not sure for Tomboy.

### Post by ronmarley1 on 2008-02-01
> **woli said:**
> Hi. I want to know how to add programs to the startup? I mean, what script do I choose to run the program (from preferences > session > startup).

Specificaly, I want to run pidgin and tomboy. Without having to add it to my panel

On the Startup Programs tab, click the +Add button.  Under name, type "Pidgin" (or whatever you want) and under command type "pidgin".  You can add a comment also.  Mine reads, "Instant Messenger Client."  I don't use tomboy, but the command "tomboy" starts the app.
