### Post by njparton on 2007-12-10
That can be done very easily.

Disable onboard VGA in BIOS, put in new card and tell BIOS to initialise PCIE graphics first.

I run a HTPC that has onboard VGA switched off and a PCIEx16 ATI X1600PRO graphics card.
