### Post by phread59 on 2008-04-13
Stupid question, The display issue is while running Windows, correct.  If so then I would do the card removal as you intended.  Reboot and reenable the on board video.  If the problem is repaired then you know the card is at fault.

If the problem still persists I would obtain a copy of the Ultimate Boot CD (it's freeware).  It has a lot of diagnostic tools on it.  Run the motherboard and memory diagnostics first.  If they pass I'm not sure where to go then.  May be a monitor issue then.

As far as the machine's ability to run Ubuntu.  I am by no means an expert here.  But I would say emphaticly yes.  I have seen posts where people installed Ubuntu on way less hardware.  You should be golden on the hardware issue.

I came here as a long time Wyndoze user.  I heard about Linux back when the earth was still cooling (I have been playing around with computers since before the Commodore 64).  Never really wanted anything to do with it.  Off late I've really been disenchanted with Windows.  So here I am.  I am currently dual booting XP Pro 64 and Ubuntu 7.10 32 bit.  I hardly ever have to boot into XP anymore.  I'm customizing my install here.  And it is getting better and better all the time.  I've really come to like Ubuntu a lot.  I just have so much to learn.  Anyway as a new user to Linux I'll say I'm impressed and happy.  You will be too.  I hope you get you're problem ironed out.
