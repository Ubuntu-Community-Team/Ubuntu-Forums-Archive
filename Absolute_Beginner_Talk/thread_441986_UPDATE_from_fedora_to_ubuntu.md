### Post by Kobalt on 2007-05-13
I've read many post (search and you'll find) : you should be able to get your usb wifi stick to work.
If you want to migrate from fedora to Ubuntu then you will be facing a fresh install, there is no upgrade possible. Run the LiveCD and select which partitions you want to use. The installer will format it and install Ubuntu over it. If you have a separate /home partition you need not to format it, you can use it with Ubuntu.
