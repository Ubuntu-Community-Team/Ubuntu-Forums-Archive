### Post by ddrichardson on 2007-09-09
There may be an underlying problem with the drive, which when automounted is causing the system to lock up (Linux used to be prone to this with scratched CDs).

If you have access to another one try it first to confirm that the disk is not the problem, then try in another port to clarify the ports are OK.
