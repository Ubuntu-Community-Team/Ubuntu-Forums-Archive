### Post by dreadlord_chris on 2007-04-26
> **darthphume said:**
> oh and btw i burned the download to a cd using nero - make bootable cd :mad:  :lolflag:

There's your problem right there - the iso you downloaded was already bootable. By using Nero's "make bootable cd" option you added a new bootsector to the CD - this is why you are getting that weird [DR-Boot]A:/> prompt (Nero's default bootsector *mimics* a floppy drive).

Just reburn the iso as a **data cd** and **don't** tick the option to make it bootable - it already is.
