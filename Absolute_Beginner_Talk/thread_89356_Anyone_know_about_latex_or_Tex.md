### Post by LHZ on 2005-11-12
You can install LaTeX trough Synaptic. Search for 'tetex', and install the basic package. Then you can use

latex bla.tex to make a .dvi file, or
pdflatex bla.tex to make a pdf file.

(You might have to install pdflatex seperately, I'm not sure. If you do, you can find it though Synaptic by searching descriptions for 'latex")

As for tutorials, I suggest you browse around and use google. There's plenty of material floating around on the web, but writing quality differs (as always). Googling for 'latex symbols' will yield some very use tables of symbols.

Hope this helps.
