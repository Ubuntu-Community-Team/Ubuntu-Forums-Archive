### Post by Dr Small on 2007-12-10
Install Xchat, then find freenode (irc.freenode.net) and then connect to #ubuntu:
```
sudo apt-get install xchat
xchat
```

Once you connect to freenode, then run:
```
/join #ubuntu
```

And there you will be :)
