### Post by benerivo on 2008-01-02
Have you tried remastersys? [This]("http://www.howtoforge.com/ubuntu-linux-mint-livecd-with-remastersys") is a good guide, and you can download the deb installation file from [here]("http://www.linuxmint.com/repository/"). I've tried it and it worked.

Also, some applications that are installed with Ubuntu can't be removed due to dependency problems, so you may want to look in to a minimal install from the alternative installation cd, from which you can install a 'barebones' ubuntu and add the apps you want to it.
