I have a garden variety DVD-ROM internal drive.  In windows, it works fine.  In UBUNTU, I'm able to browse whatever disk happens to be in the drive but the eject button doesn't work!!  This only happens when UBUNTU is loaded.  I find this very strange!  What is UBUNTU doing to disable the open/close button on the drive??

Breff

---

### Post by eentonig on 2007-05-07
Right-click on the icon on you desktop representing the drive and chosse 'eject'

Linux blocks the manual ejecting of CD drives while they are mounted, to prevent problems if problems are still using it.
