### Post by FleetAdmiral74 on 2007-06-15
I don't think it is possible *while* you are reinstalling, but there are a few methods I can have ya research

1) using the Ub LiveCD, then mounting the old filesystem. If you are fairly new, just search for instructions. Once you find the files, you can just save em to whatever.

2) burning something like knoppix, should mount file systems and you can copy the files.

3) if you think you know what the prob is, use something like SUperGParted (thats not what it is called, ...SuperGrub maybe?) to try and fix it. kinda advanced, but can give it a shot, lots of instructions floating around.

Good luck!
