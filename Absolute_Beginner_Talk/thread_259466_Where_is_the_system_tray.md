### Post by aysiu on 2006-09-17
It's called the "notification area" in Gnome.

Right-click on the panel and select **Add to Panel**.

### Post by Haegin on 2006-09-17
In gnome the system tray is the section in the top right next to the clock. You should be able to see the volume control in there already and if everything is working ok there should be a kopete or amarok icon up there which you can use to close the program (or perform other tasks with the program) when you right click.
If they dont appear then i can only assume it is because they are looking for the KDE system tray which obviously isnt there as you are running Gnome.
