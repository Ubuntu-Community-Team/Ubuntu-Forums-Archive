### Post by aysiu on 2006-12-06
> **adewale said:**
> due to some patent issues ubuntu by default doesn't play those files you have to download some extra codecs i can't remember the link for that wiki try searching this forum for restricted formats.
It's not just patent/copyright issues. It's ideological as well. Ubuntu doesn't want to include proprietary software by default (it makes an exception for proprietary drivers required to run certain hardware, but that's a different issue altogether, though not for some people). It believes in promoting Free software as much as possible.

Other distros like Mepis, Blag, and PCLinuxOS have no problems including those proprietary formats and are also cost-free distros.

### Post by tonyr1988 on 2006-12-07
> **jrstubs said:**
> Oh, thanks for the quick replies, but how about running windows programs in linux, like counterstrike?
There is a program called WINE (stands for Wine Is Not an Emulator...yes, confusing :D) that lets you run some Windows programs in Linux. However, it is SOME, not ALL, so be careful.

They have what is called an App Database that has a bunch of Windows programs and how well they work under Linux. Here's the appDB entry for Counterstrike:

[http://appdb.winehq.com/appview.php?iVersionId=3731](http://appdb.winehq.com/appview.php?iVersionId=3731)

It appears that it works great under Ubuntu Edgy, but someone said that it's Garbage in Dapper, so you may want to opt for Edgy, as long as you don't experience any big problems with it (I haven't made the upgrade, so I'm not sure).

If you haven't heard about it, there's a great program called Automatix. It allows you to install a lot of common programs easily, including those audio and video codecs you were talking about. It can also setup Wine and some other helpful programs for you.

One more thing - make sure you read that AppDB entry for Counterstrike. It doesn't look like you'll be able to just install it like you would in Windows - you'll have to take a few extra steps to get it working smoothly. The steps are listed if you scroll down - it doesn't look too complicated.

I hope your Linux experience is great, and don't be afraid to post threads if you're having problems.
