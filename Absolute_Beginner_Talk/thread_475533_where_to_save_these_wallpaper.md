### Post by starcraft.man on 2007-06-16
Well first make any folder wherever that you want to use to store them, a simple one in home will do. Then right click on background of your desktop and select "change desktop background". Push add wallpaper, navigate, and find your folder with em. Thats it :).

### Post by Golyadkin on 2007-06-16
If you want to automatically cycle really cool photos from Flickr as your background, you can install Webilder ([Website]("http://www.webilder.org/")), which works very simply and lets you specifiy which tags the photo's you want to use must have and some more options. 

For instance, you can supply the tags: "wallpaper bikini 1600x1200" to only get large sized wallpapers of scantly clad ladies ;)
