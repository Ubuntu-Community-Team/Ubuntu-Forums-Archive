I've found NoMachine to be a nice substitute for VNC. Not only does it run over SSH (port 22), but it encrypts *and* caches session information so that it runs smoother.

I've run VNC and remote login from my wireless laptop before while my remote machine (the one I'm logging into) was downloading a bunch of torrents and it still ran acceptably.
