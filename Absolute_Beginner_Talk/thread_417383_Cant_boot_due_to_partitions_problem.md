### Post by Zillion on 2007-04-21
Sudo fdisk -l

Than if for example ur fat32 is sda2

$ sudo vol_id -u /dev/sda2
bdf8786e-fa96-40e3-93af-9dfd386d7f29

use this uuid for mounting in etc/fstab
1st make a folder in media, for example winfat32

than add this line with ur uuid

UUID=bdf8786e-fa96-40e3-93af-9dfd386d7f29 /media/winfat32 vfat iocharset=utf8,umask=000 0 0

sudo mount -a

and it should work :) 

took me some time to figure that out :)
