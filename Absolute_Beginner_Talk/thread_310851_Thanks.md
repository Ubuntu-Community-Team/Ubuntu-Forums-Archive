### Post by ityro on 2006-12-01
My system: HP/compaq nx9010 Notebook with Pentium 4 2.66GHz, 40GB HDD and 702MB RAM, Printer: HP Deskjet 5740, Scanner: HP Scanjet 3770, and an Iomega 80GB removable USB HDD. Windows System: XP Pro (SP2).


I have successfully installed Ubuntu (dual-boot with XP on System Drive).   Maybe my experience will help someone else.

After many failures, I went back to step one and downloaded Live CD ver 6.06.1 LTS using HashTab to create/verify an ISO file and, ImgBurn to copy it to a CD-R. This method of creating the installation disk was suggested by a member the Ubuntu Tribe.
 
When I went to install I changed methods so I can't be sure my earlier problems were due to the installation disks. This time I disconnected all devices including my cable connection to the internet.  This is recommended (by Windows Forums) for an XP clean install so I decided to try it with Ubuntu. Installed without a hitch (finally got to see the slider to resize the partition). In a few minutes Ubuntu was ready to go.

I transferred "My Documents" from XP to Ubuntu via the removable USB HDD where I formatted a partition in Fat32 to use with Ununtu. I installed Thunderbird, Firestarter, Adobe, and did a backup using the How To: by Heliode, which is impressive. All in a few hours after the installation.

I have some questions but will use another thread as the purpose of this message is to say THANKS!
 
So, MY MOST SINCERE THANKS to the Ubuntu Community for all the great documentation and Forums. Installing and using Ubuntu without you would have been impossible for me.


THANKS AGAIN.
