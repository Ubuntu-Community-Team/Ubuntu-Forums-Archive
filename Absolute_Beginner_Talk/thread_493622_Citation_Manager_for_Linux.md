### Post by sybille on 2007-08-17
I'd suggest Zotero as well, especially since the new Zotero release includes support for an extension for OpenOffice.org Writer. 

There's a lot of great documentation about Zotero at the home page for the extension, including various useful demos of its features:
[http://www.zotero.org/](http://www.zotero.org/)
If you want the newest version of Zotero, it's better to install it from that page rather than the official Firefox add-ons site.

Once you've installed Zotero, you can install the extension for OpenOffice.org as follows:
First download the extension from the Zotero site: [http://www.zotero.org/documentation/openoffice_integration](http://www.zotero.org/documentation/openoffice_integration)
Unzip the file so that you have a file called Zotero.oxt somewhere where you can find it.
Then open a new text document with OOo Writer, go to Tools -> Extension Manager, choose "Add."
Navigate to the directory where you put Zotero.oxt and choose it.
Once the extension installs, restart Writer, and now you'll see a new toolbar for inserting references and bibliographies into your Writer documents.

I prefer Zotero because it makes it so easy to enter bibliographical information into the database it manages. Generally it's just a matter of locating the publication or resource you want to cite in an online library or journal service (some newspapers are currently supported as well) and then clicking on an icon in the Firefox address bar. Maybe some small edits are needed depending on how the information is imported, but that's it.

Zotero also allows for taking notes and adding and annotating attachments - links to files, copies of files, snapshots of web pages, etc., etc. There are some new full-text search capabilities, also.

The OOo extension does the same kind of job as Bibus, using a UNO service to insert citations and bibliographies from Zotero into Writer documents in any number of styles. These references can be updated when the Zotero database changes.

Anyway, I'm finding the Firefox/Zotero combination very useful for my research. I've prepared a book-length translation using Zotero without the OOo integration. I exported references from Zotero to the clipboard and then pasted them into my OOo (sub)document(s). This was not nearly as good as the direct integration of Zotero and Writer with the newly released extension, though.

So anyway, I think this setup is worth a look for anyone doing research. :)
