### Post by anewguy on 2008-04-20
First thing you will need is to understand that the 2 types of modems you are talking about are most likely "winmodems".  This means that there is software (the driver) that handles a lot of the functions of a stand-alone modem.  This software was most certainly written for Windows, not Linux.

So, take the names and models of both the modems and do a search on Yahoo! for something like:

ubuntu <your modem/chipset here> 

This will tell you if anyone has gotten your particular modem to work with Ubuntu.

I'd start there, post back, and see if we can help from there.  :)
