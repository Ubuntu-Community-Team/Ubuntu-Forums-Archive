### Post by dbd on 2006-11-07
One way to go about this is to learn Qt, which is what KDE, and all KDE apps are written in. This is a set of C++ libraries which allow you to easily write programs with GUIs.

The best way to learn is first to learn the basics of C++ and then Qt.
A good place to start learning C++ would be with this:
[http://www.cplusplus.com/doc/tutorial/](http://www.cplusplus.com/doc/tutorial/)
And a good basic Qt tutorial is this one:
[http://doc.trolltech.com/3.3/tutorial.html](http://doc.trolltech.com/3.3/tutorial.html)

Also, Thinking in C++, as mentioned above, is very good. But goes into a lot of detail, and is not aimed at someone learning their first programming language. Probably worth a download though, since it would make for a good reference book when you want some more details.

Good luck :)
