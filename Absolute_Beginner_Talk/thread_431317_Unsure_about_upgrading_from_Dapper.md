### Post by euler_fan on 2007-05-02
Feisty is still pretty new. As a personal preference I tend to like to upgrade about half way to the next release, so I would suggest if you really feel you need to upgrade go to Edgy. I am running it on an HP with a comparable ATI card (X200M) and it works great.

Dapper is a fine release with long term support. Unless staying completely current is a necessity, there is really no need to worry about making a change until the next LTS release comes out.

Good luck!

### Post by Jawbreaker4Fs on 2007-05-04
I'd stick to dapper on the Inspiron 6000... I'm using one in feisty, and this laptop has seen significant drops in default functionality in every release since Dapper. These are the default out of the box things that have worked and not worked by Ubuntu version:

Dapper:
Front media buttons work
Screen resolution fine

Edgy:
Only volume up/down and pause/play buttons work
Screen resolution fine

Feisty:
Only volume up/down button works
Screen resolution limited to 1024x768, 800x600, and 640x480

I've been a long time Ubuntu user on this laptop, and I have to say that if the next version sees another drop in functionality I might be switching distributions.
