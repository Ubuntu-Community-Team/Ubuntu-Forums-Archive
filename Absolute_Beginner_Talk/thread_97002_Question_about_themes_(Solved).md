### Post by TeeAhr1 on 2005-11-30
Where are the GNOME themes located, and how can I install new ones, e.g. from GNOME-look.org?

---

### Post by aysiu on 2005-11-30
Download the theme. Do **not** extract it.
Open up System > Preferences > Themes.
Drag (with your mouse) the theme.tar.gz into the Theme Manager window.
Click Install.

### Post by Sutekh on 2005-11-30
Copy aysiu's method except this time go to System -> Administration -> Login Screen Setup.

Inside go to Themed Greeter and either drag your tarball as aysiu said or click install theme and direct the browser to the file.
