### Post by Gustav on 2006-02-17
Either use file-roller that has a gui or type this in the terminal:
```
unrar e <filename>
```
You'll might need the unfree unrar, install it by typing:
```
sudo apt-get install unrar-nonfree
```
Maybe you need to add the multiverse repository for this.
