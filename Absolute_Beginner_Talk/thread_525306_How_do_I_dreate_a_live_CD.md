### Post by Hospadar on 2007-08-14
It may be that the disc wasn't burned properly, you can use the disc checker utility on the installer (it's an option when the disc starts up, at the same time as choosing to install) 

to verify the download you can check the md5 checksum on the .iso you downloaded. This is a hash key designed to verify file integrity, there are some instructions on how to do this [here]("https://help.ubuntu.com/community/HowToMD5SUM").

Probably you just burned a bad disc, try burning it again on a really slow write speed like 4x.

Also, just to be sure, could you give us some hardware info and which version of xubuntu did you download (64 bit, x86).  If you arn't sure which version you downloaded, the filename of the .iso will tell us.
