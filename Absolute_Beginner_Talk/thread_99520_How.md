### Post by Rinzwind on 2005-12-05
Hello,

In your menu there should be a programm called **Synaptic Package Manager**. With that you are able to install programms, There are several(!) programs that install software and all use different styles. The Synaptic Package Manager uses ikon and a graphical display and therefor is easy to use and very newbie-friendly.  

What it does tho is call upon a command called apt-get, So it's also possible to do it commando-style :)
Something like 
**apt-get install mozilla-firefox**
would get you Firefox 1.07(!) and will install it in  the same run.

A HOW-TO on installing  multimedia can be found here:
[http://www.linuxforums.org/tutorials/1/tutorial-25783.html](http://www.linuxforums.org/tutorials/1/tutorial-25783.html)
It even starts with the basics so it should be fairly ok to use. You should be aware tho that the brand and chipset of your card are very important and you should  not just go about installing all kinds of codecs and drivers.

Oh and I'm a very newbie too tho I understand alot about the basics of coding and it seems to be a bit eassier for me to 'get into' Debian and it's procedures. 

Learn about apt-get so you understand the things that happen underneath and you'll enjoy (k)Ubuntu even more :)

Good luck!!
