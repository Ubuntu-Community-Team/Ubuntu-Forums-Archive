### Post by tgalati4 on 2007-06-13
Plug the phone into the computer with the USB cable and it should show up as a mounted drive on your desktop.  Then you should be able to drag and drop music into that drive.  Wait several minutes for the files to transfer.  Nautilus file manager will falsely report that the files have been transfered.  The phone takes much longer to receive the files.  You may have to cycle the power on the phone to reindex the music files.  I need to do this using my SE w810i to see the new music files.

Don't fill up the phone's memory completely, used the extended memory whenever possible.
