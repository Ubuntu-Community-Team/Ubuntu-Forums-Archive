### Post by scooper on 2006-05-20
An exotic option is switching to the Fluxbox window manager.  It can auto-tab any application(s) based on your specification.  E.g. it can automatically make all xterms attach to each other in a single tabbed window.  So it can do the same with any file manager you like.  I'd recommend Rox Filer, but there are other good ones.

This will give you the tabbing functionality with greater flexibility, but requires entering the world of alternative window managers.  You can also tab apps together arbitrarily with the ion wm, and probably others.
