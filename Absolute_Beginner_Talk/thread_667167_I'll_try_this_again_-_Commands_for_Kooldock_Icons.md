### Post by CJ56 on 2008-01-14
This is probably about the most inelegant way of doing it but it works for me -

Find the application you want in the Applications menu

Right-click on it and put it on the desktop

Right-click on the desktopped icon, go to Properties

Click on the tab marked Launcher

In there you'll find the command

After that, I just copy & paste the command to wherever it needs to go....
