### Post by mtalexan on 2007-11-29
1) Goto System->Administration->Synaptic Package Manager.  
2) Search for "wacom" and select the "xserver-xorg-input-wacom" package for install.  
3) Click the "Apply" button and accept everything until the package begins installing.  Once it's done, hook up your tablet and restart your computer.

If the tablet still isn't working you should check out the [Linuxwacom Project]("http://linuxwacom.sf.net") for further details and can reference questions back here.  For reference, a lot of wacom support is included in the kernel used by Gutsy, which is 2.6.11+.
