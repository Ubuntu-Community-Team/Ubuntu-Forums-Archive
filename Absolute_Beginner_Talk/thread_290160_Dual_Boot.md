### Post by bobpur on 2006-10-31
Install Windows first on one hard drive and then install Ubuntu last on the other drive. If you don't, Windows will overwrite GRUB and you won't be able to get to your Ubuntu drive. You just install Ubuntu by following the prompts it shows and GRUB installs and you're done.
This might be a little over simplified; but, it's really not that hard.

                                      Good Luck!
