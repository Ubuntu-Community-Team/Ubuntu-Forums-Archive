### Post by sprok8 on 2006-02-01
In general, you do not need to install drivers for your motherboard under Linux. Drivers do exist for motherboards, graphics cards, sound cards, etc if the default installation doesn't work. The most common upgrade is graphics card drivers, which are provided as binary-only software by Nvidia and ATI.

If you want to check your hardware will be compatible, try a Live CD first.
