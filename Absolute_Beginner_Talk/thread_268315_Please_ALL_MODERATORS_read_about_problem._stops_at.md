### Post by VoodooDali on 2006-10-02
This sounds like the *exact* same problem I was having for two days or so. I don't think the problem is your CD-ROM *or* the Linux distribution.

Are you trying to install to a disk that is formatted NTFS? If so, then you'll want to use a DOS boot disk and fdisk, to deallocate and unformat some space, or else Ubuntu will have no place to go.

This turned out to be exactly what was preventing me, but I only tried it on a weird hunch, based on some very *vague* clues.

The Ubuntu documentation, forums, etc. take for granted that Ubuntu installs, period - all available literature seems to yada-yada past it. It is not a safe assumption.  :(
