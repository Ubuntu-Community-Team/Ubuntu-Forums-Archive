### Post by mssever on 2007-11-30
> **meanburrito920 said:**
> is there a way to have apps automatically start up in their own desktop when I log in? for instance, have my email open on the main desktop when I log in, and Pidgin open up in a different desktop, etc.

That depends on what window manager you're using. If you're using Beryl, it's simple. Since I'm not using it anymore, I don't remember how to do it, but if you liik through the settings app, you'll find it.

If you're using Compiz Fusion (e.g., Ubuntu Gutsy), you need to install compizconfig-settings-manager. There's an option in there. Unfortunately, that configuration is quite cryptic and there's no documentation; I've never figured it out (I haven't tried too hard). I think that in many ways compiz fusion is quite a regression.

If you're using Metacity, I don't think it's possible.
