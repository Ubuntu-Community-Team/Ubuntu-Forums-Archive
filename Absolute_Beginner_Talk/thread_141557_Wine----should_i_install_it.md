### Post by LordBug on 2006-03-08
If you want to play Windows based games you have 2 choices:  Wine and Cedega.

The only burden these have is the hard drive space they take up (a few megs each, nothing huge).  They don't load up on boot or anything like that, so they only use up CPU and RAM when you're actually running them.
