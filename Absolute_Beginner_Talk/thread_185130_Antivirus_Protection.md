### Post by Brynster on 2006-05-31
the general consensus is at this time virus for linux exist in labs as a proof of concept. There has not been a wild one in a good few years to the best of my knowledge. Simply put the core differences between say my Ubuntu box and yours would mean its incredibly hard to implement a virus as nobodies two boxes are indentical. Unlike Windows and OSX where the same flaws in security appear on many boxes and many different versions of the OS (in windows, OSX being the better but still recently a couple have sufaced.)

That being said if you sit on a network where a windows box is present, you may not be a victim of a windows based virus but if you forward an attachment from an email which contains a windows virus they the windows user could contract the virus.

So clamAV is available and others for the sole purpous of us the great unwashed linux community from passing a windows virus to other windows users. Or not if you just don't care.

### Post by az on 2006-05-31
[QUOTE=hotbrainz]Just wondering if any firewall would be necessary for the Ubuntu box. [/QUOTE]

A firewall can be useful if you have no control over what your box does and you don't want it talking to the outside world.  You don't have that problem with Ubuntu, in general.  Just be mindful of what you install.  Always use the official repos.

As for viruses, if they ever do become a threat, it probably will be handled promptly.  Keep up-to-date with your security updates....


It it not neccessary to run any anti-virus sfotware in Ubuntu.  If that ever changes, I'm sure everyone would hear about it through the notification applet and the updates-notifier.
