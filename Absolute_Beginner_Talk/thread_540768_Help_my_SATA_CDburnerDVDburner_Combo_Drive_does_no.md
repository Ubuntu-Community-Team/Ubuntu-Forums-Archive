### Post by psychicist on 2007-09-03
This might have to do with an unsupported piece of hardware. You could try building a newer kernel but you wouldn't want to do this if you're very new to Ubuntu and Linux in general.

You could try the Gutsy Gibbon alpha prerelease version but it isn't recommended for production use. So if you want to run or install it just to see if your hardware works, be my guest. But if you want to use it for production you'd better wait for 2 months until it's released.

Ubuntu 7.10 (Gutsy Gibbon) testing versions can be found [here]("http://www.ubuntu.com/testing/tribe3").
