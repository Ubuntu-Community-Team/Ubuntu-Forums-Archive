### Post by nalmeth on 2006-05-07
gnomebaker/k3b is good if you want a whole app to do it.
.iso burning is built into nautilus though, you should just download the image, browse to it's directory in nautilus, right-click the .iso, and click burn to disk.
Easy!

### Post by cgjones on 2006-05-07
Put a blank CD-R into the CD Burner. A dialog box will pop up asking what you would like to do, click cancel (I think). Open up the directory where you saved the iso, right click on it and click write to cd (or burn?). Sorry I couldn't be more specific, but it's pretty easy.
