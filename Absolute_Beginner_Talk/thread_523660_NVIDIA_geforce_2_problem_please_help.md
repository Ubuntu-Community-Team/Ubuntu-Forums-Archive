### Post by hamburglar6 on 2007-08-12
are you sure you downloaded the right nvidia drivers from the nvidia website?  it looks to me like this is probably the one you want:

[http://www.nvidia.com/object/linux_display_ia32_1.0-7184.html](http://www.nvidia.com/object/linux_display_ia32_1.0-7184.html)

if you had already tried the correct one, what were the problems you encountered trying it?

also, in your xorg.conf did you change the driver to 'nv' because 'nvidia' did not work, or did you never set the value to 'nvidia'?  if it is the latter, before you try anything else go to the section labeled 'device' and change the 'driver' option to 'nvidia', and press alt + ctrl + backspace to restart x.

```
sudo dpkg-reconfigure xserver-xorg
```

Just use the default anytime you don't know the answer, make sure you choose 'nvidia' as the driver, and select the refresh rate/resolutions that you want to use.
