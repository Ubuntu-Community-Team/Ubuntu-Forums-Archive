### Post by Sef on 2006-05-14
> On scanning disks in the partitioner the computer hangs at 41%. Any ideas on how to get past this?

1) Bad CD - burn a new one.

2) Bad burn - reburn the disk

3) bad download - redownload (did you do an md5sum check?)

4) See if a live cd works.  If it does, then the problem is most likely one of the above.
