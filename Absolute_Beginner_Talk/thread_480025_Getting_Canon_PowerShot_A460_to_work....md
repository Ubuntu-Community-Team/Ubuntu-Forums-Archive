### Post by FleetAdmiral74 on 2007-06-21
Sure thing. All I had to do was make sure the camera was in playback mode (where you view already recorded pics, I think its the triangle symbol...) before I turned it on (already usb connected). Ubuntu then picked it up and was able to import the photos. I could not access it as a disk drive right away, but the import photos worked great.

Tell me if this works for you.

EDIT: You don't need to wait to turn your camera on actually. If it is in the play back mode, but the import window does not auto show up, then you can go to the f-spot photo manager (app - graphics) and go file - import, then chose the input source as your camera.
