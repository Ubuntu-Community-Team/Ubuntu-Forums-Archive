### Post by gatorbrit on 2008-03-28
Well I'm back.  After about a year and a half away I thought I would give linux another shot.  

A few observations.
1. Open office is much more compatible with word than it was.  The equation function works very well.
2. Wine seems more stable, and I am able to run my favorite statistical program - stata.

So I am loving ubuntu on my toshiba laptop!!

The bad:
1.  I had a very large window title bar fonts.  These go away when I disable the special effects for the appearance. 
2.  My other PC - 4 year old dell with dual monitors works fine until I try to select the dual monitor function - then it all goes bad and I get terrible resolution.  There are lots of posts on this and I need to decide how much time I want to spend trying to get it to work.

Finally, wubi was a nice way of getting back into ubuntu, But I ended up doing a full install from the ISO to reinstall it.

### Post by gatorbrit on 2008-04-01
Wubi is an installer that installs UBuntu from windows.   Its not a VM set up - you boot directly into ubuntu, and it pretty much runs like a full install, except that all the ubuntu files live in one folder in windows.  You can move your wubi install to a full install at a later date.

One nice thing about it, is that reinstalling ubuntu is very quick if you are testing things out on a new machine - which is what I was doing.

[http://wubi-installer.org/]("http://wubi-installer.org/")
