### Post by carlosqueso on 2006-08-02
I'm not 100% sure...but there is an easy way to check.  Download the desktop CD and burn it as an iso.  Then you can boot your computer with the cd and see if it works, as well as play with any tricks you find on the forum.  If you can't get it to work, then don't install.  If you do decide to try it, make sure your wireless card is attatched, and from a terminal, run ```
lspci
``` which will give you what the computer thinks it is.  Search for help with that instead of the card brand, you'll have much more success.
