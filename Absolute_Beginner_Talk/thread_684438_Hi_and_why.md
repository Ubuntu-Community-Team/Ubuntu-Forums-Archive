### Post by indytim on 2008-02-01
I can't speak for WUBI, but it sounds like you may have a hardware incompatiblity.  I'd suggest:

1.  Download the Ubuntu-Kubuntu-Xubuntu image file from Ubuntu.
2.  Burn the iso as an image onto a CD at a slow speed.
3.  Boot to the CD.
4.  Run the CD check option.  If it's good-to-go, then go ahead a boot up in the LIveCD.
5.  Exercise the LiveCD ops a bit to make sure it's playing nice with your hardware.
6.  If it's ok, then go ahead and install.

If the LiveCD doesn't work (assuming you have at least 192M of RAM), then you're getting into the hardware element that I referenced above.  Report back your hardware config on a separate posting for additional assistance.
