### Post by aysiu on 2007-08-01
**Cleaning House**
Tools > Clear Private Data in Firefox will remove temporary internet files.

You can also delete stuff in the /tmp directory

And you can clear out the old installation .deb files in /var/cache/apt/archives by using the command ```
sudo apt-get clean
```
