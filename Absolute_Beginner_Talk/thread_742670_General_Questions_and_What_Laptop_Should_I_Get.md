### Post by Mogurijin on 2008-04-01
If you get a dell laptop preloaded with Ubuntu, then you shouldn't have many issues. Also, Dell constantly provides the latest version of Ubuntu for download in their Linux wiki. 

As for wallpapers, any image will do, just like Windows.

I believe the "minimal" install lacks a desktop manager (ie: Gnome, KDE, xfce, etc). You are fine going with the regular live cd install. It is very easy to remove software through synaptic and the add/remove software in the main menu thingy.

As for the interface, you can use compiz and emerald. I believe both are pre-installed now on 7.10. But I don't really like 7.10 since it doesn't play nice with my internet speeds. However, there are plenty of howtos, just search "compiz-fusion" on these forums and something should come up.

Enjoy ;)
