### Post by the_nomad on 2007-03-01
i myself i use xmms and its good enough for me :) to install it go to terminal and run

```

sudo apt-get install xmms

```
or 
```

sudo aptitude install xmms

```

you may also install it from synaptic manager :) go to System > Administration > Synaptic Manager and check xmms then click Apply to apply changes.
hope it helps. good luck.
