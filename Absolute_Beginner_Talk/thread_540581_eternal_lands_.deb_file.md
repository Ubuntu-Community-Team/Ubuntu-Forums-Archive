### Post by nonewmsgs on 2007-09-01
i looked it up and did not find one but it doesnt look too painful to install.

download the zip file
make a folder call it eternal on your desktop
put contents of zip into folder eternal
then in a terminal type cd ~/Desktop/eternal
sudo sh el.x86.linux.bin
gedit el.ini and find the line where it says the game is locaed and replace it with 
~/Desktop/eternal
