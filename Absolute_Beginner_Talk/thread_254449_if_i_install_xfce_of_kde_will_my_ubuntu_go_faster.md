### Post by aysiu on 2006-09-10
> **purple unichorn said:**
> but does KDE run faster than GNOME?
KDE and Gnome will run about the same speed... or one may be faster, depending on your situation or biases. Seriously, [as you can see from this thread](http://ubuntuforums.org/showthread.php?t=172238), you're just as likely to get someone saying "KDE is faster" as you are to get someone who says "Gnome is faster."

The only way to know for sure **on your computer** is to try both **on your computer**.

That said, at least I and another user here (nalmeth) have found *kde-core* to be faster than *kubuntu-desktop* (Ubuntu's default KDE set of packages), so if you like KDE but think it slower than Gnome, you might want to do ```
sudo aptitude update
sudo aptitude install kde-core
``` XFCE will almost always be faster than both Gnome and KDE, though.
