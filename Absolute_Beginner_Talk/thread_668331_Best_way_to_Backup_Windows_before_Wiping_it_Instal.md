### Post by fiddledd on 2008-01-15
Just image the entire Hard Drive, Acronis True Image (not free), Clonezilla, Partimage (both free) or any of a number of imaging apps available. Then just restore the image if you ned to later.

### Post by gbold on 2008-01-15
Maxtor and Seagate both offer a free backup program based on Acronis True Image (great program by the way).  All you need is a Maxtor or Seagate drive to install it...  It will have you make a boot CD so you can restore the image at a later time if need be...  It will let you back up to DVD/CD as well as individual partitiions...  A removeable drive is quite handy for this.
