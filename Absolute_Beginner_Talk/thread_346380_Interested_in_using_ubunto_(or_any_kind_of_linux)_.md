### Post by Quillz on 2007-01-25
Linux, believe it or not, is actually much more compatible with hardware than is Windows. Whereas Windows is used almost exclusively on the computer, Linux has been engineered for use on cell phones, cash registers, etc.

Anyway, desktop distros like Ubuntu have very good hardware support, because there are a lot of open source drivers available. However, other hardware, like NIC, tend to be more difficult to set up because the developers work in a Windows-only world, and fail to provide drivers for Linux support.
