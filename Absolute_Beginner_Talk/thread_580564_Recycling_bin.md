### Post by Dr Small on 2007-10-18
Applications > System Tools > Configuration Editor
apps > nautilus > desktop
[check]trash_icon_visible

Or, to run Configuration Editor, go to the terminal and type:
```
gconf-editor
```

### Post by Paul820 on 2007-10-18
You might not have anything in that menu, the menu won't show if there is nothing there. Go to > System->Preferences->Main Menu
Click on System Tools in the left box and activate configuration Editor in the right box. You will now have a system tools menu.
