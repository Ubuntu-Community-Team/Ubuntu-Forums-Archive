### Post by billgoldberg on 2008-03-14
VLC will play .srt, I use it all the time.

Press open file -> then tick the "subtitle" box and add the subs.

You can even adjust the subtitles if they are a bit off.

If I remember you can use "alt + j" and  "alt + k" to adjust them. (could be ctrl instead of alt)
