### Post by bodhi.zazen on 2007-03-24
A meta package is not an actual program, but more a list of all the programs to be installed.

For more information (new user friendly) [http://www.tuxmachines.org/node/14383](http://www.tuxmachines.org/node/14383)

### Post by Circus-Killer on 2007-03-24
to put it simply, a meta-package points to a whole bunch of packages.

for example, ubuntu-desktop is made up lots of packages, including a LOT of gnome packages, including gnome itself, X, etc.

in order to make peoples life easier, meta-packages were invented so that you dont have to specify all the packages seperately, but can rather install all the necessary packages by specifying just the meta-package.
