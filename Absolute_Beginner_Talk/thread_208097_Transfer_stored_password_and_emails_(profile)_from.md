### Post by HereInOz on 2006-07-03
I don't know about the profiles, but as far as the bookmarks are concerned you can export them and then import them to Firefox on the ubuntu machine, but you probably already knew that.

With the emails in Thunderbird, you should be able to copy the mail store over to the Ubuntu box and then set up the path to that mail store in Edit > Account Settings > Server Settings > Local Directory.  This should work, but certainly try it before deleting anything in the Windows system.

The Thunderbird Address book can be exported in Windows and imported to Thunderbird on Ubuntu.  You can't export the mail settings, unfortunately.  I don't know about the passwords being exported but I have not found a way to do it.

There is a settings import facility in Tools in Thunderbird, but that mainly applies to importing from other mail clients that already exist on the machine.

I would like to help more but this is the extent of my knowledge in this area.
