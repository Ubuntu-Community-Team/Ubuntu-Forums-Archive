### Post by sub2007 on 2008-01-16
I don't have any experience with download managers but it sounds like your problem should be solved if you install FlashGot: [https://addons.mozilla.org/en-US/firefox/addon/220](https://addons.mozilla.org/en-US/firefox/addon/220)

The developer says that it works best with WxDownloadFast: [http://dfast.sourceforge.net/](http://dfast.sourceforge.net/)

You could also try DownThemAll, which builds a download manager into Firefox: [https://addons.mozilla.org/en-US/firefox/addon/201](https://addons.mozilla.org/en-US/firefox/addon/201)
