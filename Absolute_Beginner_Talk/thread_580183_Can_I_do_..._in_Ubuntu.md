### Post by azad on 2007-10-18
1. Yes, you can run media formats with x264 encoding with Totem-xine Media Player. Once you install Ubuntu, search the repository for **Totem** and you'll find it. Make sure you get Totem-xine and not Totem-gstreamer.

2. You can burn .iso and .dvd easily. There are many CD/DVD burners for you to choose from.

3. Windows Media Video is well supported by Totem-xine.

4. Absolutely! The Streamzap remote is fully compatible with Linux Infrared Remote Control (LIRC)

You'll find LIRC package in Synaptic or just use apt-get. Use MythTV as your PVR.

You can use Elisa if you want a complete Media Centre experience.

5. Why not?

I hope that helps. Just remember, everything is possible in Ubuntu!
