### Post by jw5801 on 2008-04-10
Unless you have alot of RAM (ie. need to be able to address more than 3GB) or run alot of simulation or heavy calculations, or use it for graphic/audio/video encoding/editing, you probably won't notice a preformance difference between the two. If you're really struggling to get things running on 64-bit then it probably wouldn't be a bad idea to give 32-bit a go and see if it makes life a little bit easier.
