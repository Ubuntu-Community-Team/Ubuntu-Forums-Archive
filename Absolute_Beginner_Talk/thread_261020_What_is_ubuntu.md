### Post by Brunellus on 2006-09-19
The Shorter Catechism

1) Ubuntu is a Free, Open-source operating system built on Linux.

2) Ubuntu is derived from the Debian project, a project aimed at building a Free and Open-Source operating system for a number of environments.  Ubuntu differs from Debian in that it is primarily focused on individual user desktops.  

3) Ubuntu represents a Free and Open-Source alternative operating system for hardware that currently runs Microsoft Windows.  It is stable, secure and reliable.  It has no spyware, suffers from no widespread virus problem.  It has no onerous licensing agreements.  

Ubuntu is an ALTERNATIVE to Windows, not a 1-for-1 REPLACEMENT for Windows.  It is NOT Windows.  It is DIFFERENT, and in some respects BETTER.

### Post by darrenm on 2006-09-19
I am starting to think that if you force someone to use Ubuntu then they end up not using it anyway. They have to be curious in the first place and want to see what the fuss is about.

In the beginning, way before Windows there were Unix type operating systems. They built the foundations of computer world we live in today, hence why we use forward slashes on the internet instead of backslashes.

In the mid 90's a Finnish computer student called Linus Torvalds started work on a kernel (the part of the operating system that converts what a program wants to do to actually making the hardware do it.) that was Unix compatible. He and a group of others built the first Linux kernels and used some GNU (a foundation set up to make free software for Unix) programs to actually make the kernel do something useful.

Over the years Red Hat, Suse and Debian became the major players in the Linux distribution market. They sold pre-packaged box sets of the operating system with limited support or you could download it for free.

Around this sort of time Mark Shuttleworth was developing Crypto software and made Thawte who produce and sign digital certificates. He made a lot of money from this and decided to start his own GNU/Linux distribution based on Debian.

Mark Shuttleworth uses his own money to fund parts of the Ubuntu project and it is now by far the most popular Linux distribution with a very exciting future.

It is very, very important for people to have a choice of what operating system they run on their computer. If everyone only run Microsoft products there would be no reason for them to improve, no innovation, no freedom. Ubuntu and GNU/Linux brings that freedom. And that is essentially the ideal behind Ubuntu, that everyone should have the freedom to do whatever they want with their computer.

All the bugbears I currently have with Ubuntu don't really bother me. I know they are there because of either the jealous underhanded tactics of Microsoft or the economics of the free software model.

I am very excited about the future of Linux within Ubuntu. Edgy Eft is due for release in 6 weeks and is already looking fantastic.

I enjoy the personal freedom Ubuntu gives me. I use software others are giving to me for free. I don't consider Windows XP to be worth the license fee they charge and I don't want to use something illegally. The fact I consider Ubuntu dapper/edgy to be better than Windows XP by leaps and bounds is a bonus.

### Post by aysiu on 2006-09-19
I love Ubuntu, but I would recommend Mepis for new ex-Windows users.

Mepis is based on Ubuntu, so it has a solid base and a good set of repositories (Ubuntu's repositories), but it comes with proprietary software (stuff Windows users generally care about--Flash playback, Java, etc.) and automounts partitions with a click (with the correct permissions)--no editing of the /etc/fstab.

It can also do a point-and-click restoration of Grub to the MBR.
