### Post by kpkeerthi on 2008-03-18
Right click on **Applications** and select Edit Menu. Choose the menu item you want to edit, right-click, select **Properties** and change the command to your liking.

If **Edit Menu** does not appear install **alacarte**
```

sudo aptitude update && sudo aptitude install alacarte
```

### Post by Sef on 2008-03-18
> If Edit Menu does not appear install alacarte

It is installed already.  System > Preferences > Main Menu   Simply if an application is listed there, but not on the Applications menu, then find it and click in the box next to it.  It will then appear in the Applications menu under the correct submenu.
