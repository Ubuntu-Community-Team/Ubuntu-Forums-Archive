I think Recent Documents should be in your home folder. Click it on desktop.

I htink this will work not sure though. To hide recent documents follow these instructions. In terminal type:

    **sudo chmod 400 ~/.recently-used**

To enable it again, run the command:

    **sudo chmod 600 ~/.recently-used**
