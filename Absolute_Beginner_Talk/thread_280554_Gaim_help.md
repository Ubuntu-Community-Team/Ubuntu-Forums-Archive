### Post by Evergrey on 2006-10-19
I found solution for my problem... I was too lazy lol ](*,) 

One account for each protocol needed ofc.

Before you can use Gaim you need existing accounts with one of the following clients. Gaim supports:

    *

      AIM ([WWW] Create an account )
    *

      ICQ ([WWW] Create an account)
    *

      Gadu-Gadu (create an account using the official client)
    *

      GroupWise (See your Novell server administrator)
    *

      IRC (see XChatHowto)
    *

      Jabber ([WWW] Create an account)
    *

      MSN ([WWW] Create an account)
    *

      Napster
    *

      Yahoo ([WWW] Create an account)
