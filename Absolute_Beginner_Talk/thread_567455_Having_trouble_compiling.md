### Post by Paulmd on 2007-10-04
1) Do you have gcc installed?
2) Do you have the build-essential package installed?
3) it should be ./configure, and not ./config... usually
4) if you use bastille or other security programs, you may need root permissions to compile


5) many times, you are required to install additional packages, ending with -dev, in order to compile a particular program. You havn't got to that point yet, but you may.
