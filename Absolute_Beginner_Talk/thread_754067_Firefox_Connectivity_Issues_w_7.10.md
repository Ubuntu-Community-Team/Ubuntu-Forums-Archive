### Post by rklauco on 2008-04-14
Wired connection usualy do not create problems.
Try to check System -> Administration -> Networking, if the ethernet connection is enabled.
Try to get properties and determine, if the connection properties are set to DHCP (usual choice in cable modem connection).
If not, set the connection to DHCP and try some browsing.
Let me know - if it does not help, we will do some digging into the system to identify the problem.
