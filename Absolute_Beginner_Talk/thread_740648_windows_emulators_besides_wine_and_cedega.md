### Post by JoshuaRL on 2008-03-31
Well, if you ever have any problems with Wine always try running it from the terminal.  That will give you any errors so you can track down the problem.

So lets say that the name of the program is program.exe:
```

wine program.exe

```

Also, its a good idea to check on the [applications database.](http://appdb.winehq.org/)  If a program needs a certain install or run method, it will usually be there under it's entry.
