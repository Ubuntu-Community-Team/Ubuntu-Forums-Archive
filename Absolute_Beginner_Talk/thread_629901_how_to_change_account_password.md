### Post by Dr Small on 2007-12-02
In the terminal, type:
```
passwd
```

It ought to be very self explanitory from then on out.

---

### Post by mouseboyx on 2007-12-02
You have to login to root
```

su
```
and then type 
```

passwd root
``` to change the root password
```
passwd yourusername
``` to change your password
