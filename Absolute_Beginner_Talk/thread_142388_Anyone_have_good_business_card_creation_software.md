### Post by arman.haghi on 2008-04-16
I use Inkscape for vector illustration then import into Scribus for publication (I use this professionally), though if its a simpler design it can all be done in Scribus (recommended to begin with).

Mate, have a go and try taking it to the next step:

a) Create an artboard (new page)

If there's any elements touching or extending past the edges, then Leave **5mm bleed all round**,

i.e. if the finished size is 90mm by 50mm, then create a file 100mm by 60mm and draw crop marks on a new layer or just tell the printer you know and trust that you've added 5mm bleed (easier).

b) Colours

- Trick is to start by **removing all unused colours** then
- Create colours individually
- Making sure **all colours are CMYK** (or spot depending on printing)

c) DESIGN!

d) Save art, then **outline** all text.

e) export

**Export it to PDF**, and best practise is to make it **X-3**, but this option is only available when **colour management is on** (try little CMS).

---

That's a fair bit, I know, but give it a go, there's plenty of documentation out there, and if you need a hand with any of the above give me a yell.
