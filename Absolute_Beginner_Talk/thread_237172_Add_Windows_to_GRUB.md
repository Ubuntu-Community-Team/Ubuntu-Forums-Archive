### Post by mlind on 2006-08-15
After automagic kernel list

```

# This is a divider, added to separate the menu items below from the Debian
# ones.
title           Other operating systems:
root


title           Microsoft Windows
root            (hd0,1)
savedefault
makeactive
chainloader     +1

```
