### Post by rouge568 on 2007-12-09
1) To get the top and bottom transparent, go to Desktop Cube -> Appearance -> Cube Color. Drag opacity all the way to the left.
2) The background is called a "skydome". Go to Desktop Cube -> Appearance -> Skydome. Note that you should only use .png images with a resolution of a power of 2. (e.g. 1024x2048, NOT 1000x2000). Also, If you want the skydome to rotate too, click on "Animate Skydome".
3) For the cool effects, got to Animations -> everything in there.

Hope this helps!
