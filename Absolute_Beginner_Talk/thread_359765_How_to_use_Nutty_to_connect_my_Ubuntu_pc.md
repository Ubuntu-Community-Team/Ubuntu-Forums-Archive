### Post by dannyboy79 on 2007-02-12
nutty is merely a modified version of putty for windows. all you would have to do is run the .exe in windows. are you already running an ssh server on your ubuntu box. if not, than you need to set that up first. 

sudo aptitude install ssh, this command within a terminal should install and start your ssh server so that you can then use nutty from windows. good luck. the nutty .exe. is here. [http://www.groehn.net/nutty/](http://www.groehn.net/nutty/)
