### Post by jimmp226 on 2006-02-05
I just wanted to let all the noobs (myself included) know about a book I found that has helped me a lot. The "Linux Bible 2006" just came out and has a lot of information in it that is easy for someone with a limited understanding of Linux to    use. There is still a lot of information in it that goes right over my head but at least they make an attempt to explain it to you. I've only read about 1/3 of the book so far but I'm impressed so far. There is even a whole chapter devoted to Ubuntu. The only other book I have on Linux is "Linux in a Nutshell".I have almost read this book cover to cover and 99.9 percent of the time I had no idea as to what they are talking about. Most of it went right over my head. Now that I'm starting to get an idea as to whats going on I can see how this book will help in the future (it does list all the commands and there options), but for a noob it might as well have been written in Chinese. Also there is another book coming out March 6th that is devoted entirely to Ubuntu. I already ordered it so if anyone is interested I will post my thoughts on it as soon as I read it if anyone is interested. ### Post by deadgoon on 2006-02-05
I haven't read the Linux Bible, but here are some things that really helped me.

[http://www.linuxfromscratch.org/](http://www.linuxfromscratch.org/) - If you really want to understand the concepts, this is a great way to learn.

Old Suse Manuals - Back at release 6.2 Suse had huge manuals that had tons of info, you might be able to find one, but I image most of them are long gone.

Online forums - Forums like this one often have just the answer you need.

Gentoo manual - The online Gentoo install manual has tons of info.  I recommend everyone do a Gentoo install at least once just to see how things fit together and work.

Ye Olde Trail & Error - Nothing works better than trial and error.  Buy a $40 crap box off eBay, a good KVM switch, install Linux, and...  Have a lot of fun!
