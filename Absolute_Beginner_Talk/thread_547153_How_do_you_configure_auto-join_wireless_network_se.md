### Post by julian67 on 2007-09-11
If you're using network-manager (the default in Ubuntu) then when you have multiple networks available it will automatically try to connect to the one you selected previously. If you then manually select a different network then that becomes the default for the future. 

You could also try [wicd]("http://wicd.sourceforge.net/") which is also a very good tool. It lets you select which networks you connect to automatically. I'm not sure what happens if there are several available that you have chosen to connect with automatically, whether it prefers strongest signal or previous selection.Try it and see!
