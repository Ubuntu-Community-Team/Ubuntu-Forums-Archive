### Post by bsussman on 2005-09-20
OK - now a more complete answer...

1.  cd works ALMOST the same way in linux as in Uncle Bill's world.

2.  The big difference is that 

 ```
> cd
``` 

means

 ```
cd toyerhomedirectory
``` 

to do what cd in DOS does, type

 ```
> pwd
``` 

(**p**resent **w**orking **d**irectory)

There are also shorthand things - 

 ```
> cd -
``` 

(tothelast diryouwere)

 ```
> cd ~/whatever
``` 

(cd to /yourhomedir/whatever)
