### Post by tompickles on 2007-06-12
sudo allows you to temporarilly run a command as the root user - which you can't log into by default in Ubuntu, as root's password doesn't exist - rather than blank - so root can't be cracked. You need your user password when you run a command with sudo.

### Post by phr0ze on 2007-06-12
In any terminal you can type 'man' without quotes and the command. For example

```
man sudo
```

This will explain the command better and tell you additional options. Man is short for manual.

Type a q to quit man.
