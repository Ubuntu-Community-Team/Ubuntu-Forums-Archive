### Post by SGC on 2005-05-29
Only extensions require approval , search plugins does not.

Most sites use javascript to add the search plugin, so make sure that the javascript is enabled   from edit/preferences/web features.

Alternatively, you can install the "Top 30 plugins" as extension which probably will work. To do that go to the following page:
[http://mycroft.mozdev.org/download.html](http://mycroft.mozdev.org/download.html)

scroll down to the section that say "Mycroft goodies" and click on "Install Top 30 search plugins"

later you can remove the plugins that you don't like
