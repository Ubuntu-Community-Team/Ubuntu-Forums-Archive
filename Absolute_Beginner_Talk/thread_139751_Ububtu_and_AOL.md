### Post by %hMa@?b&lt;C on 2006-03-04
if you have broadband, then you dont *need* AOL, if it is the Always on type, then you should be good to go. but if not, you may have PPPoE, (point-to-point over ethernet), try running 
```
sudo pppoeconf
```
if that doesnt work try [http://yolinux.com/TUTORIALS/LinuxTutorialAOL.html](http://yolinux.com/TUTORIALS/LinuxTutorialAOL.html)

### Post by emalyse on 2006-10-29
AOL broadband, which has the BT router does not connect automatically as most of us would imagine.1st the router is locked so you can't really mess around with settings. There is no internet until you login with the AOL software each time (On windows)which I presume is basically logging in via PPoA therefore running linux will not instantly give you broadband as most routers would (where the username and password are supplied by that router). I have heard that replacing the router with a PPoA capable router and using your AOL email address and password could work.
