### Post by mali2297 on 2007-10-08
Open a terminal and copy/paste:
```

cd Desktop
chmod +x RealPlayer10GOLD.bin
./RealPlayer10GOLD.bin

```
You will be asked some questions, just hit ENTER and the defaults will be chosen.
