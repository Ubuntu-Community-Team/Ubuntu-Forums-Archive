### Post by aysiu on 2008-04-14
Gutsy is the shorthand for the nickname (Gutsy Gibbon) of the latest stable release of Ubuntu, which is Ubuntu 7.10.

Compiz fusion is some special desktop effects (wobbly windows, spinning cube, etc.).

### Post by northern lights on 2008-04-14
compiz is a desktop effects manager and ships with Ubuntu. There are alternatives out there, one of which was/is called Beryl. Beryl introduced some of the fancy stuff you've probably seen, such as the cube. It is discontinued and integrated into compiz, which since is called compiz-fusion.

Gutsy (7.10) is the current stable version of Ubuntu, but in two weeks Hardy (8.04) will be out. It already exists in beta state, but has not received the "stable" label yet.

In general, you might want to check [Psychocats]("http://www.psychocats.net/ubuntu/index.php").
