### Post by fatality_uk on 2008-01-19
If the card has dual output, try your monitor lead in another socket. If not, it might be a bent pin on the lead. Look closely to see if there are any pins out of alignment. It thats ok, try another leade between PC and monitor. If that still doesn't work, then try reseating the gfx card. It could just have come lose. If all that fails, try restarting x server
