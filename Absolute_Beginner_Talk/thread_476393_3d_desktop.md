### Post by swisscow on 2007-06-17
hmmm......I tried it 3ddesktop myself and it ran fine for me - which is of course absolutely no help for you.

The problem I suspect is your graphics drivers, which can be a pain in the *rse with ati. If it helps, I have installed the ati fglrx drivers through synaptic which went ok and then made my xorg look like below. Your device name will be different. It may be an idea to do a quick search for ATI X200 and drivers in these forums before you do this.

I recommend that first you backup your xorg before playing around with it using

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bup
```

Then if all stuffs up you can restore the original from the command line using

```
sudo cp /etc/X11/xorg.conf.bup /etc/X11/xorg.conf
```

Then restart x using ctrl-alt-backspace or reboot and you will be back to where you start.

I'm afraid that's about the limit of my knowledge:(
