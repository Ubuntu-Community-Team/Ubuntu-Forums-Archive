### Post by LuisGMarine on 2008-03-07
Try running this command from terminal:

```
sudo dpkg-reconfigure xserver-xorg
```

Make sure that select your correct resolution, and one or two below that resolution.  So say if my normal resolution was 1280x1024, then I would pick that along with 1024x768 and 800x600.

### Post by spiderbatdad on 2008-03-07
sounds like you've enabled advanced desktop effect without having a theme selected in emerald. Some Radeon cards, it seems, dont display the window controls when the window is full-sized. You may find that if you mouse into the right spot, you still have control of the window. You should be able to reset your window manager by pressing Alt-F2 then enter ```
metacity --replace
``` You wont be able to use compiz with metacity, but should get you stabilized until you can figure out what you're trying to do.
