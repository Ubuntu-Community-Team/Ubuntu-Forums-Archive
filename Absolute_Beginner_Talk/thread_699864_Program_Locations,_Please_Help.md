### Post by herbster on 2008-02-17
AFAIK, binaries are in /usr/bin or /usr/local/bin. Have you tried hitting ALT+F2 and typing "yak" and seeing if autocomplete shows it? Also, you can open a terminal and type the first few letters, hit TAB and see if it completes the name. If so, it's installed and that's all that's needed: type the command.

To find just try this:

```
whereis yakauke
```

in terminal.
