### Post by xjgnsdof on 2008-05-15
When I clocked my CPU down a bit, all freezing behavior ceased. From what I can understand, Ubuntu utilizes memory differently than Windoze does, so it picks up errors that Windows just doesn't notice; when a CPU is overclocked, the memory is overclocked too, and errors can arise. Based on my experience, CPU heat or clock frequency may be at issue for some of your rigs.
