### Post by chaplanger on 2006-12-29
For the "How to" The issue was letting the Conversion process complete (there is no signal that a process is underway, only a box that  appears when the process is completed telling you to reboot).  Once this is done you can access the proxy settings from System>Administration>Configure Parental Controls.  Unlock Firefox proxy settings and you are golden!
