This gets asked a lot. 

The short answer is no.

The long answer is there are virtual machines that allow you to run the mac OS.

The longer answer is there have been projects to emulate macOS in linux, but as far as I can tell, no one's been very successful. It requires re-implementing proprietary libraries.
