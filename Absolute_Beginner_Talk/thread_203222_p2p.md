Yes [frostwire]("http://www.frostwire.com/index.php?title=Main_Page") is a good gnutella client that is the same network limewire uses. It even has Ubuntu install packages. There is a CD++ client called Valknut you can install with synaptic. 
But be carefull none of them is safe if you are trading copyrighted works.
