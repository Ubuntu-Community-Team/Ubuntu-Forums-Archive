you have to download a ppd file from [http://www.linuxprinting.org/printer_list.cgi](http://www.linuxprinting.org/printer_list.cgi) relating to your printer and use that as your driver when you install it in system-administration-printer i did that with the canon bj 250 after a long time trying and it fixed it straght away when you run a test print it prints the ubuntu logo
