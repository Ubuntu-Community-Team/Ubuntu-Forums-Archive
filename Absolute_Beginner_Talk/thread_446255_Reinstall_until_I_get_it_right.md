### Post by Martin on 2007-05-16
If you are using standard Ubuntu you can uninstall or install anything using Synaptic which you can find under System -> Administration -> Synaptic Package Manager.
 Once Synaptic is started hit the reload button to refresh everything and then by using the search button you can find software packages that are a.) available for installation and b.) already installed. Installed packages are marked with a green square, by clicking on the square you can choose to uninstall it. Blank squares denote progs that are available for installation
