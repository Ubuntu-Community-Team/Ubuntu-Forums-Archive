### Post by quandary on 2007-11-29
I think this is because Linux is capable of using Microsoft-protocols, but Microsoft can't (=doesn't want to) use Linux Network protocols.

What you need to do is to install Samba file server on your Linux computer, and configure it.
Should be no problem, you should find all the information on Google.
