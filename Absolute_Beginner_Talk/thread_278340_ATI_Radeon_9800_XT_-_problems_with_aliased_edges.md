### Post by caravel on 2006-10-17
FSAA is Full Scene Anti Aliasing (a type of AA).  This is enabled to remove the 'jaggies' (aliasing) you see on 3D image, especially on the edges of 3D objects.

I'm not sure if it will help you, as it's only relevant to the 3D API (in this case OGL).

AF is Anisotropic Filtering. [http://en.wikipedia.org/wiki/Anisotropic_filtering](http://en.wikipedia.org/wiki/Anisotropic_filtering)
