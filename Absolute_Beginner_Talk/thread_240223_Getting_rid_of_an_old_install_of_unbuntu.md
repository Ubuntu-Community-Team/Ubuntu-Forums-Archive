### Post by harisund on 2006-08-25
No, the regular Ubuntu 6.06 LiveCD is all you need, since it uses Gparted anyway. 

Start with the LiveCD, and then start the installation by double clicking the install icon in the desktop. Along the way, you will be asked where you want to install, and whether or not you want the partition formatted. Go ahead and format what you want (all the previous Ubuntu installations). 

When you reboot, you will have a nice option of Windows XP or Dapper.
