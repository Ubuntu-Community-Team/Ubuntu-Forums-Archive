### Post by Carrots171 on 2006-08-19
You can download either version, but I think that the 64-bit version will take better advantage of your 64-bit processor. The Dapper install CD is also a LiveCD - when you boot it up, you can try the Ubuntu OS without actually installing it on your hard disk. You can use the live cd to find out whether or not wireless works automatically - usually it does, but sometimes you'll need to install a driver or do some configuration.
