### Post by costoa on 2006-06-17
1. DSL compatibility. No problems there. It's more like they can't deal with Macs (lol). Verizon likes pppoe so any cheapo depot router can handle it. 
2. DSL speeds. Yeah, the key is "up to". The closer to the CO you are the better chance you have of getting the higher speeds. Run the land line telephone number though dslreports (or alike) and check the distance. Also ask the neighbors if they have dsl and (if so) fire up speakeasy.net's flash based speed test. The latter is the best of the two.
3. AFAIK verizon uses some off the shelf linksys combo dsl modem / router unit. Nothing special. I suspect you might be stuck using it.

As for cost I kept the cable, dropped verizon and got voip but that was good for me. Also verizon dsl deals normally have a one year hook with a $325US out (ouch).
