[QUOTE=MatsB]When I boot up with Ubuntu live CD I only get to the desktop. Doesn’t the installation process start automatically or am I missing something here?[/QUOTE]
The live CD doesn't install the operating system, you have to dowload the install disk, which will start the installation process. When Dapper Drake (6.04) comes out in April, you will be able to install from the live CD, but it's not stable yet. For now, just use the install CD.
