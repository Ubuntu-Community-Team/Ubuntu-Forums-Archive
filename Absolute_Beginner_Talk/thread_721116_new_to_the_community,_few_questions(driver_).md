### Post by srhlefty on 2008-03-11
Hi aldyguy,

Welcome to the forums!  I think you'll find the people here very supportive and willing to help you, as many are in the same boat.

When it comes to drivers, there's a good chance that you will not need to download anything prior to installing a linux distro such as K/Ubuntu.  Basically, if the livecd runs ok, then the distro it comes with will run in exactly the same way.  Pretty much all hardware is autodetected.  3D graphics drivers for modern gpu's may take a bit more effort, but on first run you'll still have a usable system (as per your livecd tests).

I would personally recommend downloading a more up-to-date live cd if you do in fact have a dapper disk.  Dapper is from two years ago, so won't exhibit as much polish as say Gusty would.

### Post by hyper_ch on 2008-03-11
there are various releases and different flavors of ubuntu

releases are like: Win95, Win95, WinME, Win2000, WinXP, Windows Vista... they come one after another and have new stuff....
In Ubuntu it's: Dapper Drake (6.06), Edgy Eft (6.10), Feisty Fawn (7.04), Gutsy Gibbon (7.10), Hardy Heron (8.04 --> still alpha). The numbers indicate the release year and month. So Dapper was release in june 06, Gutsy in october 07 and Hardy will be in april 08.

So the newest stable version is Gutsy Gibbon right now.

As to the flavors: Unlike Windows there are different desktop and window managers on linux. The best known ones are Gnome, KDE, Xfce... those flavors all come with a different set of preinstalle applications, mostly polished to the according desktop environment. Kubuntu (KDE) uses only KDE appz, Ubuntu (Gnome) uses only GTK appz... If you install ubuntu and want to test out kubuntu, you can easily install that also:
```

sudo apt-get install kubuntu-desktop

```
Same goes for Xubuntu and Ubuntu.... the underlaying system however is all the same.

So in your case you should grab one of the Gutsy Gibbon downloads and try taht.

### Post by Mogurijin on 2008-03-12
Linuxs generally support a wide range of architectures. I don't know what your machine uses, but seeing as it is IBM it might be a ppc. What ever the case you should be able to find a i386 (32-bit), AMD64 (64-bit) and ppc (PowerPC I think) version of what your downloading. If you can't find a ppc version search Google for psubuntu. Psubuntu is a version of Ubuntu adapted for the PS3 (which has a ppc architecture).

As for the flavors of Linux (as people seem to be calling them). I like Ubuntu, but that is because I prefer Gnome to KDE. You won't have to mess with Edubuntu (it is for educational environments like schools). Also, unless you have an old rig I would recommend staying away from Xubuntu. Kubuntu and Ubuntu look nicer ;).

EDIT: It appears that most Thinkpads were x86 machines. So you'll probably have more luck getting the i386 version. Also, if you're relying on wireless, you might need to mess with drivers before hand.

### Post by InfinityCircuit on 2008-03-12
IBM thinkpads are probably the best supported Linux hardware out there.  Many linux developers use thinkpads, and just about the only thing you will find as unsupported is 1) the winmodem, if it is present, or 2) the infrared port, as it is accessed through the proprietary SD card reader.

Note that Lenovo Thinkpads are still very well supported but not as well as IBM.
