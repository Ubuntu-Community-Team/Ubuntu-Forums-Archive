The Logitech Revolution MX is awesome in XP though...tried some stuff on here, but I kept messing up X and forgetting to back it up (I know...dee dee dee)  Any ways welcome aboard...don't get too frustrated quitting cold turkey  is tough,but oh so worth it  :)
