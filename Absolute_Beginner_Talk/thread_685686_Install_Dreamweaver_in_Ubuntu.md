### Post by amazingtaters on 2008-02-02
if you are running a virtual machine with windows on it (via VMware or similar) or if you have WINE you can run .exe's. However, there is no guarantee that a program will work in WINE. Photoshop CS2 works via WINE. I'm not sure about Dreamweaver. Take a look at the WINE website, [http://www.winehq.org/](http://www.winehq.org/), to see if dreamweaver will work and to get WINE.
