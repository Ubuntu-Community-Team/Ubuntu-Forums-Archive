### Post by jfinkels on 2007-10-29
You could start here: [https://help.ubuntu.com/community/Wacom](https://help.ubuntu.com/community/Wacom)

Read the first link in my signature for info on how to install software in Ubuntu.

As for "restoring", you can use the program "PartImage" to make a backup of your system, and then restore it if you don't like the changes you have made. You could also just remember what packages you install and what configuration files you edit (i.e. make a backup of files before editing them) and just reverse the changes you make to your system.
