### Post by szf on 2006-12-19
> **Obi-Wan said:**
> Second, where can I go to get a list learn all the commands needed for this OS?


If you have cash - go to the bookstore, alternately, the local library may carry some titles - there's nothing like having dead tree copy for the more complex command line interface (CLI) programs/tools.

Try:[LIST=1]
[*]Linux In A Nutshell (O'Reilly)
[*]Linux/Unix System Administration Handbook (PH/PTR)
[*]Linux Cookbook (No Starch)
[/LIST] 

There's also a new small format series available from Sam's Publishing that covers Linux Commands. Costs around $15 USD.

In the terminal *man* and *apropos* are two helpers.
