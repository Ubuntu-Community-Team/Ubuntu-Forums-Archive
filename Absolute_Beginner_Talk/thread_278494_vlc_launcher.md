### Post by Bigbluecat on 2006-10-16
You could save the link in a playlist.

Select View -> Playlist

Once the playlist manager is open select add and paste the stream in. Then  save as a name and location you want.

Not sure how to launch VLC with a particular stream but VLC documentation is quite extensive:

[www.videolan.org]("http://www.videolan.org")

EDIT: Just had a look at the VLC documentation on the videolan website and found this:

 To receive a HTTP stream, start VLC with    :
% **vlc [http://www.example.org/your_file.mpg](http://www.example.org/your_file.mpg)** Hope that helps
