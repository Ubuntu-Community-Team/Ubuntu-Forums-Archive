### Post by Sef on 2007-01-31
> 1.Should i download 6.06 or 6.10?

6.10 has newer software, but less stable.  Also it is only supported until April 2008.

6.06 has older software, but is more stable.  (It's Long Term Stable: LTS.)  It is supported until June 2009.

> 2.my computer has a AMD 64 chip. So now im stuck between downloading either 32-bit or 64 bit. Which one would be better for me?

Get 32-bit.  Not all apps are made now for 64-bit.

> 3.Does Ubuntu have a partition maker built in the disk, or am i going to need to use partition magic before hand?


Ubuntu uses an older version of GParted as its partitioner.   Do not use Partition Magic as it and Linux tend not to get along well.   If you want to partition ahead of time, get [GParted]("http://gparted.sourceforge.net").  It is a newer version than what's on the Ubuntu cd.
