Hi there,

I don't have any idea on how to synchronize the mails spooled for each Thunderbird install but I can tell you how I deal with this.  What I do is use IMAP and spool the mail on one machine (a server actually but it just needs to be a machine that is always booted).  This way all the mail is located on this machine but other installations of Thunderbird can get at the mail.  All 5 of my computers can now get at the same mail and there is no need to synchronize.
