### Post by HereInOz on 2007-09-26
If you are running Ubuntu, open a terminal and type:

btmaketorrentgui

This will give you the facility to package the torrent and create the torrent file.  You then upload the torrent file to a tracker.  Most tracker sites have instructions on the site how to upload the torrent file once you have created it.
