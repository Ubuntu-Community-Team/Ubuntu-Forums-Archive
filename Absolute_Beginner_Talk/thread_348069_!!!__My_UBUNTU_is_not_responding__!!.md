1) Actually, Linux is the one of the **most** secure - whoever told you that it was less secure doesn't know what they're talking about. Linux has a firewall already built in, called IPTables. It requires no extra configuration, but you can install a graphical frontend for it. I recommend 'firestarter'. To install it, open up the Add/Remove program, and then search for 'Firestarter'. Check the little box, and then click OK.

2) I see no reason why not, that should be perfectly ok.
