### Post by mcduck on 2007-03-26
The biggest difference between torrents and other P2P software is that torrent breaks the files into small pieces, allowing you to download parts of the same file from many people at the same time. That makes it much faster in transferring large files than other P2P networks are. The other thing is while other P2P systems have their own program that you use to find the files, you usually get torrent seeds from websites.

I've never seen any need to replace the torrent client that comes with Ubuntu.
