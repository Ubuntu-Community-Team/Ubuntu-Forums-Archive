### Post by arjun.shankar on 2006-08-09
No. Unless you want to share directories on the linux machine. What this means is that you already have a samba client, but you will need a samba server only if you want to share yourself. In my hostel, we usually ask the windows guys to give us write permission, and copy in whatever they want ;).
