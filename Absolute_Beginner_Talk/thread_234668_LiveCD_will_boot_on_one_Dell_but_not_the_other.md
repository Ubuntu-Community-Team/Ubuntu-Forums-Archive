### Post by steve.horsley on 2006-08-12
My D600 at work just had to have a replacement CD drive because it lost the ability to read rewritable CDs, although it could read the silver factory CDs perfectly. The first I knew about the problem was when it failed to boot an Ubuntu live CD. Then I found that it could still boot the windows installer CDs, but could not read backup CDs that I had previously burned on that drive. It would not even spin up to speed for a rewritable CD. 

So, try reading other recordable CDs to prove whether it's the same problem I had.
