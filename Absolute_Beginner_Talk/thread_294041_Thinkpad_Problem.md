### Post by ReaderRat on 2006-11-06
According to the specs, T21 shipped with only 128 Megs of RAM. This is a good bit less than the 192 --> 256 Megs needed for a full install of (K)(X)Ubuntu. I would suggest using a less demanding OS like [**Damn Small Linux**](http://www.damnsmalllinux.org/download.html) or [**Puppy Linux**](http://www.puppyos.com/download/downpage.htm)
