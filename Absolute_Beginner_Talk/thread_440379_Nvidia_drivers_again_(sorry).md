### Post by pizzutz on 2007-05-11
back up your xorg.conf 
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.backup
```

and then 
```
sudo nano /etc/X11/xorg.conf
```
to edit.

Change 
```
Depth 24
Modes "1024x768" "800x600" "640x480"
EndSubSection
```
to read
```
Depth 24
Modes "1280x1024" "1024x768" "800x600" "640x480"
EndSubSection
```

save and restart X (ctrl+alt+backspace) and it should appear on your resolutions drop down box.

### Post by joft on 2007-05-11
Well, the resolution of 1280x1024 does not appear in "Modes" lines. So the first step could be adding "1280x1024" in front of "1024x768" to at least the last Modes line.

If this does not work, you might need to look up again what your monitor supports regarding horizontal/vertical refresh rates. The current maximum of 70 kHz (see line "HorizSync") might be too less - your monitor should support more.
