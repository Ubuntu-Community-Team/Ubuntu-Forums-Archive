### Post by poofyhairguy on 2005-07-07
[QUOTE=filemanager]
So that would allow me to install 32-bit apps as well as 64-bit apps?[/QUOTE]

Bascially what that does is allow you to run the 32 bit version inside the 64 bit version. SInce the 64 bit version is lacking in a few ways (say....firefox pluggins and codecs) its the only way to enjoy this stuff in 64 bit Ubuntu.

As I have said before, the 64 bit version is really meant for advanced users. If you need the certain software this is basically the "you have to be this nerdy to ride" line for the 64 bit version.
