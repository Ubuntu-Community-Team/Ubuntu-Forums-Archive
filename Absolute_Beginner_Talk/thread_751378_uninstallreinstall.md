### Post by artblakey on 2008-04-14
Assuming you downloaded the main installation CD image from the Ubuntu site then it should be the live CD. Burn it to disc and then restart your computer with the disc in the drive to enter the Live CD environment, and install the new version on the same partition where 5.10 is now (after you've backed up any important files on that partition). 

It might also be a good idea to read some of the guides and info available on these forums and the main Ubuntu site to give yourself a better idea of what you're actually doing before you do it, because while partitioning a hard drive is pretty easy now it's still possible to loss all your data straight away if you do the wrong thing.
