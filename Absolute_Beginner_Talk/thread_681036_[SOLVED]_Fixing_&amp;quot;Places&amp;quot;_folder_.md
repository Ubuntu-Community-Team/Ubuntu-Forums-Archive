### Post by jpeddicord on 2008-01-28
> **Echo35 said:**
> I was wondering if there was any way to change the folder listing under "Places" in 7.10? I used my own folders for Pictures and a few other things, so under Home Folder and Desktop, there is nothing else listed, save for one folder. Is it possible to add other folders from my home directory here as well?Yep! Open your Home folder (or any other folder) and navigate to the directory you want to add to the menu. Once you find it, hit Bookmarks > Add Bookmark. It will be added to the Nautilus sidebar, the Places menu, and all Save/Open dialogs.
