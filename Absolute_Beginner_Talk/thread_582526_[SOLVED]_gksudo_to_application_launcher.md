Is there a way to do a launcher that, automatically, executes some program wiht root permissions (prompting you for password)?

### Post by Hobo Joe on 2007-10-19
Yes, if it is in the menu, you need to go to System > Preferences > Main Menu, and find the entry that you want, then edit it, and add the 'gksu' prefix.

That should do it.
