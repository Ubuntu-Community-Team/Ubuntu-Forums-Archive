### Post by noynac on 2006-08-13
Boot-Up Manager [BUM] is a Perl-Gtk2 application to handle runlevels configuration of any debian derivative system. With this program the user will easily start and stop boot-up scripts, without the necessity to handle thru complex links and permissions.

[http://www.marzocca.net/linux/bum.html](http://www.marzocca.net/linux/bum.html)

### Post by catlett on 2006-08-13
```
sudo apt-get install bum
```Then select it under System<Administration<Boot-Up Manager. When the window opens you will have a list of applications that run at boot. Un-check anythinmg you don't need.
