### Post by Sentinel83 on 2006-09-21
[FONT="Arial"]Haxer,

If you want to make it so that you have a launcher in KDE for Firefox, right-click on the menu (in the KDE "start" menu) that you would like to have the launcher and select edit menu.  Then choose "Add Item" and put in Firefox for the description.  Then have it launch from /usr/bin/firefox.  Save that setup and you're all set.  I am pretty sure though, that it may already be in the KDE menu under Internet > Firefox browser. [/FONT]
