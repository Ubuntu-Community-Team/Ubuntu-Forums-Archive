### Post by Afkpuz on 2008-02-26
Yes you can.


System>Administration>Login Window

Enter your password

Select the "Security" tab

Check "Enable Automatic Login"

Click the drop down arrow on the right and select your user name.


Thats it!  It's actually quite easy isn't it?
