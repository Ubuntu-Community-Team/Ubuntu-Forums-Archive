### Post by buntu_hugenewbie11 on 2007-05-29
So I IM all the time and sometimes I connect through yahoo messenger and talk over the mic and even use the vid cam with nothing but a click.
But those were the Windows days...
Now in linux I need to compromise: I can use Kopete for camera- voice I can't use anything other than Ekiga. Yes skype exits but not for 64bit.
So what are my options: wait and see??
Thank goodness for dual boot. But overall it seems so much time has been spent in just getting my machine in linux to have even half the functionality it does in Windows. Now this is likely because I'm a novice you'd say, but the truth is you lose functionality with this op system and though I like it's stability it does come at a sacrifice. So a further question is: when does this compromise stop?
Yes until more gets developed, but until it does dual boot will remain first best choice for many newbies such as myself.
