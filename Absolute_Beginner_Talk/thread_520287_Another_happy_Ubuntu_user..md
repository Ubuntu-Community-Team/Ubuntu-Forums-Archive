### Post by wak_wak on 2007-08-08
I was running Windows XP Media Center for some time.  Decided to finally make the move to Linux.

Threw up a VM on my laptop and started testing everything.  Eventually wound up on Ubuntu for nothing more than personal preference.  Also had read some articles and what not that indicated it as being pretty well supported.  Besides which, Dell was distributing it and that bode well (thanks Michael Dell).

Alright!  So I'm running a Dell Inspiron E1705.  It's got an ATI 256mb graphics card and an Intel ProSet wireless card.

It works GREAT!  It's solid.  Very solid.

I've got it running KDE, Gnome and Gnome with XGL.

Also running is Compiz Fusion.  Awesome!

I had some problem with the wireless under Ubuntu.  So I went with Kubuntu and then installed Gnome and then Gnome with XGL.

Bottom line is it's fast and dependable.  

Thanks a lot to the community.  Beyond amazing help and support!  I'm a very happily converted Linux user.
