### Post by aysiu on 2007-04-10
Hope you don't mind--I moved your post to its own thread in support. Thought it might get more useful responses here.

I'm not sure what your background is with applications, but one easy way to smooth your transition to Linux is to use as many open source applications as possible. This page has a great list of open source applications for Mac OS X:
[http://www.opensourcemac.org/](http://www.opensourcemac.org/)

Once you get used to using Firefox, Thunderbird, VLC, AbiWord, Audacity, and GIMP on OS X, trying out Ubuntu (or any other Linux distribution) won't be such a culture shock.

As for the differences between KDE and Gnome, this should give you *some* idea:
[http://www.psychocats.net/ubuntu/kdegnome](http://www.psychocats.net/ubuntu/kdegnome)

I would start with Gnome, as you're less likely to be confused (most of the help on these forums is for Gnome, and most users here are Gnome users). If you are curious about KDE later, you can always install that and switch back and forth between the two.

You may want to check out [our subforum for Intel Mac users](http://ubuntuforums.org/forumdisplay.php?f=211). There's also [a HowTo on the Wiki about installing Ubuntu on a Macbook](https://help.ubuntu.com/community/MacBook). And, finally, [a tutorial on burning an ISO in Mac OS X](http://www.psychocats.net/ubuntu/maciso).
