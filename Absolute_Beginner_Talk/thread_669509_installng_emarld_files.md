### Post by Hospadar on 2008-01-16
you'll probably need manually start up emerald the first time after you install it as above.  After you install some themes in System->Preferences->Emerald Theme Manager use the "import" button, or get themes from the theme repositories on the next tab do an Alt-F2 and type "emerald --replace"
