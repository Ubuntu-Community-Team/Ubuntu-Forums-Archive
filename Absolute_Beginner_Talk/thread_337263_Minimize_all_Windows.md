### Post by joshzam on 2008-02-03
> **dondad said:**
> Right click on the panel and select "add". From there, there is an icon for show desktop. This will minimize everything and bring you to the desktop.

No! This button has the exact same functionality as CTRL+ALT+d. It does not minimize everything unless all windows are already restored. If you have just one window minimized and you click this button (or use the keyboard shortcut) it RESTORES all windows. You'll have to click it AGAIN to get them all to minimize and be able to see the desktop.
