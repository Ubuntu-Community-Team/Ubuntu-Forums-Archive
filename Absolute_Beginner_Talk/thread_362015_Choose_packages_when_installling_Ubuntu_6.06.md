I'm not sure ubuntu lets you do this.  If this is important to you either (a) install debian instead, or (b) do the install, but promptly delete everything and build it up again.  There is only 1 CD's worth to install, so its not too bad, but ubuntu assumes you want everything and installs it (I can only assume the rationale is not to scare new users away with lots of confusing choices during install).
