### Post by ubuntu-freak on 2008-01-15
Hope you get it working. If you do and your resolution is still wrong, try going to system>preferences as you can select a resolution there. If the correct one isn't there, paste this in the terminal (applications>accessories)
 
sudo gedit /etc/X11/xorg.conf
 
Then scroll down to the resolutions section and type in the res you know your screen can do.
 
If anything goes wrong (write this down) paste/type this

sudo dpkg-reconfigure xserver-xorg

To get multimedia working (streaming, java, flash, playing/ripping etc) give my how-to a try
 
[http://ubuntuforums.org/showthread.php?t=661833](http://ubuntuforums.org/showthread.php?t=661833)
