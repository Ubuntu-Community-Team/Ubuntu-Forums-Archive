### Post by igknighted on 2007-04-03
:) Java is pretty straight forward as long as you have the login password.  Just go to System->Admin->Synaptic Package Manager and open it up (you will need to enter the password again).  Then search for this package: sun-java6.  You will want to mark the jre and the browser plugin for install.  The click install and it will be ready to go.

As for your printer, in the same menu there should be a printer wizard.  Just run that and click OK to accept the defaults and it should work no questions asked.
