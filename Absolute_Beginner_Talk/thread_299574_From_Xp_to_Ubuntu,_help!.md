### Post by gitano1 on 2006-11-14
I started with a computer with XP on it. I decided to use a dual boot system since there are some functions, DVD ripping for one, that linux does not yet have the software for. I initially installed the 6.06 and upgraded to 6.10 when it came out. Both forms worked faultlessly on my system. They found the various hardware and installed drivers. The one exception was the more advanced drivers for my Nvidia card. Those were easy to find using Automatix2. 
I strongly recommend setting up a dual boot system which will allow you to experiment with Ubuntu while keeping one foot in the World of Windoze. The software is designed to walk you through the process of setting up the a dual boot system. It is very easy. When the computer boots it defaults to Ubuntu, but you can use the arrow keys choose Windows if you want. You have several seconds, plenty of time, really, to make the choice. With a fully functional Ubuntu in place it is only using about 6 GB of disk space. If you set aside 10GB of space for a partition you should have plenty of room. This would be my recommendation.

### Post by gitano1 on 2006-11-14
The applications that I have in mind are for decrypting commercial movies. DVD Decrypter and DVDShrink both work in Linux, but they are not updated for the new encryption algorithms. There are several commercially available programs which are continuously upgraded to deal with the new encryptions. They function only in a Windows environment. Wine is not powerful enough to run them. That is the only reason that I maintain a Windoze XP(U) OS. I am finding that I boot into XP about once a week. The rest of the time I am either using Ubuntu or my iMac. Automatix2 and Wine are really essential additions, but the basic Ubuntu programs are fantastic all by themselves. I am running the dual boot on a computer custom built for me by Infotech in Seattle area. It has Asus MOBO, Pentium 4 2.0GHz, 1 GB of RAM, 80GB HDD, and a 20 GB HDD. I have the Linux loaded on the 20 GB drive mainly because I have a lot of music files on the 80. I use K3B to copy decrypted and shrunk DVDs and CDs. It is superb software far faster than Nero.
