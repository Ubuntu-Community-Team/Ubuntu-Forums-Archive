### Post by Sef on 2007-05-08

No, it isn't.  The alternate install is a text-based installer.   It is fairly simple to install with it: mostly common sense.  The [Illustrated Dual Boot]("http://users.bigpond.net.au/hermanzone/") has great tutorial on how to install with it.  If you aren't dual booting, you can still get a lot out of it.
