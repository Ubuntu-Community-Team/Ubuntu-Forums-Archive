### Post by ricardisimo on 2006-10-06
How do I select my preferred app for listening to music? I kind of like both XMMX and Listen Music player. How would I set one or the other as my default music app? I'm assuming that this question will also answer other default app questions, but please elaborate if this isn't the case. TIA.

---

### Post by aysiu on 2006-10-06
Preferred in what sense? Like when you double-click a music file?

Right-click the music file, select Properties > Open With
Make sure the black dot is next to the application you want to be the default.

---

### Post by GlennB on 2006-10-06
You can do this by opening nautilus, then navigating to a file of the type you are interested in (e.g. an mp3 file, ogg file etc).

Then right click the file in the nautilus window. You should see a 'open with other application' option. At this point, you may find the app you want in the list. If not, you can choose 'custom application' and browse to the application file name.

Regards,

Glenn.
