The Ubuntu Dapper 6.06.1 can be ran from CD, this is known as the "LiveCD" it basically enables you to test the Operating System before you install it, you can also see what it detects e.g sound, internet, resolution.

You may need to go into your bios and boot from CD first.

Always try to remember windows and linux are totally different, linux can run .exe, but not by default, you would have to install a program called wine, which then enables you to run SOME, not all window programs.

Ubuntu can install programs through synaptic package manager, you search for the program you want and then it not only downloads it for you, but installs it to.

You can also install programs manually :)
