### Post by bulldog on 2006-09-11
When you opened Network you can see your ethernetcard.
Select it and see if its set to on.
If not set it on.

Fill in if you use configuration with DHCP [router] or fill in your IP and the rest off the fields.

But if it worked before without filling in these fields try to restart your computer.
It should be automaticly configured then.
