### Post by PmDematagoda on 2007-12-16
I would suggest going for Ubuntu X64, it can mean some performance boosts and also would mean that the memory is used more efficiently, also a lot of things have been made more easier on Ubuntu 7.10 X64, especially the installation of Flash. A more detailed thread concerning this topic can be found [here.]("http://ubuntuforums.org/showthread.php?t=368607")

### Post by jrharvey on 2007-12-16
I agree, you should go with the 64bit version. It is the same as asking the question "should I put low grade or high grade gasoline in my Z06 corvette" only they are both the same price :) As long as you get your apps on synaptic then you should have no problem with finding 64bit compatable apps. The only problems you may run in to is if you are trying to download apps that are only for 32 bit that arent in the repository yet. There is usually always a way to work around it is frusterating to see that "wrong architecture" error. Dont worry about it though because like i said, there is usually always a fix for this.
