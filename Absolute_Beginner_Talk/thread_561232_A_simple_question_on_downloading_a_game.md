### Post by hyper_ch on 2007-09-27
Actually I would try to install the debian etch .deb ( [http://six.retes.hu/download/debian/etch/six_0.5.3-1_i386.deb](http://six.retes.hu/download/debian/etch/six_0.5.3-1_i386.deb) ). Ubuntu is based on Debian and a .deb file is a precompiled file for installtion (just like an .exe in Windows). Download that file to your desktop, then double-click it and check if you can install it. If it works, all fine... if not then you will have to compile the program from source.
