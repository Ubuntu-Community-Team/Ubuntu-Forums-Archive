### Post by Carbonfish on 2006-09-14
The odds of someone "slipping something in" are so tiny as to be insignificant. Someone might sniff the network, and might even get some info. regarding your system, but they would have to actually upload code to the repositories for it to wind up on your machine and even then it probably wouldn't do them any good as you would have to allow them administrative pivileges for them to exploit your machine.

One of the big differences between Linux and Windows is that to make system changes on a Linux (or Unix) box you have to be root (or a user who has been given root permission). Windows lets anybody make system changes even to the registry, so Windows is much easier to exploit. Linux (and Unix) are simply safer by design.

There are definately things to worry about with open nets, but I don't think that updates through Synaptic are one of them.

There are alot of Sysadmins out there in the shadows who know alot more about it than I do, but I think you're pretty safe.
