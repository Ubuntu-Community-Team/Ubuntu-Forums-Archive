### Post by 3rdalbum on 2008-02-03
If you are running NetworkManager, just go to the little wireless applet icon on your top panel, left-click and select the unsecured network. Connection should then happen automatically.

Otherwise, if you're not running NetworkManager or it doesn't work for any reason, you could connect to your network using an Ethernet cable, secure the network, and then connect wirelessly.
