### Post by jordanmthomas on 2007-09-22
Mostly the commands are the same, though many distributions have specific tools installed as well.

Fedora and Ubuntu both have sudo installed, but Arch doesn't by default (easy enough to install it though) and Fedora's default user isn't allowed to use sudo.  This is because most distros have you set up a root password, so you don't really need sudo.

Fedora also has redhat configuration programs that Ubuntu doesn't.

Each distro has its own package manager.  Ubuntu -- apt, Fedora -- yum (can use apt, but still has rpm, not deb), Arch -- pacman.

But probably 95% of the commands are the same from distro to distro.
