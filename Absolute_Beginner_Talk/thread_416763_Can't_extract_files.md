### Post by rillip on 2007-04-21
It's not exactly clear from what you posted what you're tyring to do and how, but you don't "extract" an ISO.  It's not a compressed format you expand and use; it's a cd image.  You burn it as an ISO/CD Image with burning software and it's just like a factory manufactured CD, with all the files on it, instead of having to burn the files all individually, it burns the whole thing in one file.  With the Ubuntu ISO's, you're meant to burn them to a CD and boot from that CD, which is a live installer.

Hope this helps; if not, you'll need to clarify exactly what it is you're trying to accomplish so we can help better.
