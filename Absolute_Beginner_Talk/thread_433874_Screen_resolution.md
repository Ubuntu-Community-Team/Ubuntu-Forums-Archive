### Post by orange2k on 2007-05-05
Open up the console and there type:

sudo nano /etc/X11/xorg.conf

Scroll to the bottom of the file and there you will find what you showed us:

Section "Screen"
Identifier "Default Screen"
Device "nVidia Corporation NV43 [GeForce 6600]"
Monitor "Generic Monitor"
Defaultdepth 24
SubSection "Display"
Depth 1
Modes "1024x768" "800x600" "640x480"
EndSubSection
SubSection "Display"
Depth 4
Modes "1024x768" "800x600" "640x480"
EndSubSection
SubSection "Display"
Depth 8
Modes "1024x768" "800x600" "640x480"
EndSubSection
SubSection "Display"
Depth 15
Modes "1024x768" "800x600" "640x480"
EndSubSection
SubSection "Display"
Depth 16
Modes "1024x768" "800x600" "640x480"
EndSubSection
SubSection "Display"
Depth 24
Modes "1024x768" "800x600" "640x480"
EndSubSection
EndSection


Now add the resolutions you want to be there, for example add "1600x1200" before "1024x768" for every screen depth there is, save the file by clicking control+O, and thats it!!!

Restart and you will have additional resolutions for your monitor!

### Post by jfinkels on 2007-05-05
First read the thread suggested by the post below mine.

The summary of it says that you should try the following:

Make a backup of your xorg.conf file:```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
``` Edit the file to add whatever your desired resolution is (i.e. the suggested resolution for the monitor/video card combination):```
gksudo gedit /etc/X11/xorg.conf
```
in the "Screen" section of the file, change each line that looks like this:```
Modes "1024x768" "800x600" "640x480"
``` so that it looks like this:```
Modes "1600x1200" "1024x768" "800x600" "640x480"
```
Save the file, exit, and restart your X session by pressing <Ctrl>+<Alt>+<Backspace>. (This will log you out of your session, so save anything that your working on!).

Tell us how it goes!
