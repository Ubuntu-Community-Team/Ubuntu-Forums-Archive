### Post by nimbuscott on 2006-10-13
You said Powerbook so make sure you downloaded the PowerPC version of Ubuntu. You can burn a bootable CD from the downloaded .iso file using Disk Utility found in Applications/Utilities. Launch Disk Utility and simply drag the .iso file into the left pane then select it in the left pane and go to Images/Burn it will prompt you to insert a blank CD. (This is the procedure on OSX 10.3.9 anyway). Once the CD is burned restart your Mac holding down the "C" key at startup and it should boot from the CD.

The Ubuntu Mac support forum can be found here: [http://ubuntuforums.org/forumdisplay.php?f=133]("http://ubuntuforums.org/forumdisplay.php?f=133")

I eventually gave up trying to get Ubuntu running on an old Wallstreet Powerbook and started experimenting with older PC's laying about. That was much more rewarding and less of a headache and Ubuntu is now my second favorite OS running on an old Thinkpad I picked up on ebay. Sorry Ubuntu, I my heart belongs to my MacBookPro running OSX but Ubuntu has been fun in a "garage mechanic" sort of way and I have learned alot about Linux in the process. Have Fun!
