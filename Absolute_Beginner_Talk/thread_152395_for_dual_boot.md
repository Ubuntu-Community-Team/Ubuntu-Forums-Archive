### Post by WoodyMahan on 2006-03-29
No offense to your teacher but they couldn't be more wrong.  Installing Ubuntu adds a program called GRUB which lets you choose which OS you want to boot into.  If you go back and re-install Windows, then it destroys GRUB and you can no longer boot Ubuntu.  Always install Windows First.
