### Post by Foxmike on 2007-07-27
I got this issue as well. This is because Nvidia drivers do not handles memory well. This usually happens when you have several windows open and some (like web browsers) that takes a lot of memory. Then, there is not enough memory to draw the maximised window. The only work-around is to de-maximise your window and resize it smaller or de-activate Beryl for intensive work.
 
Hope it helps!:)
