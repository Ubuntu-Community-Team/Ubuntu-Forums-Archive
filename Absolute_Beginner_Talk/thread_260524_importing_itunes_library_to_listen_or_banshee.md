### Post by jonlatane on 2007-02-21
If you use Banshee, there is a plugin available that will import your iTunes music library, so long as it's in the folder you select to import.  Unfortunately, the version of Banshee included on Edgy is too old to use the plugin.  However, if you manually upgrade to 11.3 and use it, you can then downgrade back to the original version.  I already wrote a HOWTO and you can find it here:
