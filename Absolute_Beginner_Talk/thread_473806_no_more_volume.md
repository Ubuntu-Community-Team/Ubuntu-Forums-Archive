### Post by NeoLithium on 2007-06-14
Hello :)
Open up a terminal (Applications -> Accessories -> Terminal) and type in the following command:
```
alsamixer
```
Inside that, use the directional arrows to increase the volumes, and unmute those which are muted (They have an M in the button, and can be unmuted by pressing M when the device is highlighted)

Hope this helps!
