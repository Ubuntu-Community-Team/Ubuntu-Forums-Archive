Depending on where you placed the file depends on how to navigate to it. Start out by typing ```
pwd
``` which will probably give you ```
/home/yourusername
```. Chances are, the file was placed right there in the home directory. By typing ```
ls
``` you can see the files within your present directory. If you see the directory there, type ```
cd directoryname
``` and you're in.
