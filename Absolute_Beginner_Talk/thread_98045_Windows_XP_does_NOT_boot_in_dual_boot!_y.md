Generally speaking, installing Ubuntu as described would not have affected your Windows install (unless you did something odd with the partition manager during install).

The fact grub knows Windows is installed and that Windows tries to load makes me think that this may not be Ubuntu related.  Did you ever boot Windows Ok with the second harddrive installed?  Have you changed anything else?

Are you sure Windows isn't just being *really* slow to boot due to the changed hardware?
