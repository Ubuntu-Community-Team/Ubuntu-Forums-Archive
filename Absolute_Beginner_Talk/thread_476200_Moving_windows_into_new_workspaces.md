### Post by kevinlyfellow on 2007-06-17
Thats normal.  But if you have a good graphics card you can "enable desktop effects" and perform this.  If that is not an option, you can drag windows inside the workspace switcher to the desktop you desire.  Alternatively, right clicking on the top you your application will bring up a menu that lets the window be moved to any workspace.
