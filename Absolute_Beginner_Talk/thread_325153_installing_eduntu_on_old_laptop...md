I would not upgrade to Edubuntu since it would require 256 mb ram minimum.   512 mb really works better.  

I would install the apts through synaptic.   To install you would need an internet connection.   To learn to install, read [How to Install Anything]("http://monkeyblog.org/ubuntu/installing/") in Ubuntu.  

To increase your usb ports, you could get a multiport device that plugs into your one usb port.
