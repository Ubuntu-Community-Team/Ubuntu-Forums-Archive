### Post by ravenwere on 2007-08-29
If you want a dedicated games PC then unfortunately stick to Windows XP. That is the default platform games are created for and supported in today. Linux games are catching up but the best games for Linux are about the early warcraft levels.

Wine doesn't automatically mean the windows game will work on Linux. You have to check the database to see which games play, which you can tweak wine for and which games will never be played on Linux.

You have a couple of options. 1. Keep the games PC as XP (personally I will never upgrade to vista for games) and enjoy.

2. Install Linux and dual boot Windows and LInux. Play most games on XP and explore Linux. Do this if you have the space, and also if you are prepared to have to restore the boot section of your PC if you go back to a windows only PC.

3. Get an older PC from somewhere. Install Linux /Wine and/or a DOS emulator. Play around with some classic free DOS based games. Have a look at some of the Linux games and play the games that work using wine such as Counterstrike, Deus Ex, WoW etc

Cheers,
