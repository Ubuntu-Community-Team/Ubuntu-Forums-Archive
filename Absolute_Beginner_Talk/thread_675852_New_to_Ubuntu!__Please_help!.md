### Post by kleo skywalker on 2008-01-23
If you've downloaded Ubuntu, burn it on to a CD as an iso image. See [this]("https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28iso%29%7C%28burning%29") for more details.
Check you BIOS to make sure that the CD/ DVD drive is the first boot device.
Pop in the CD and click the "Install" icon on the desktop when it loads. [This]("https://help.ubuntu.com/community/GraphicalInstall") is a simple installation guide.

Internet Explorer is a web browser, the default one in Ubuntu is Mozilla Firefox . MSN messenger doesn't work in Ubuntu. There are other IM clients though; Pidgin is the one installed by default and it can connect to your MSN account. If you're looking for something similar to MSN, aMSN is an option.
Other than the default applications, you will have  host of options to choose from.

Good luck!
