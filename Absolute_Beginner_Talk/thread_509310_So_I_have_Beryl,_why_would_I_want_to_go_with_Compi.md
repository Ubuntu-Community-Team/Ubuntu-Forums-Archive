There's no need to go immediately to CF. But my reasons were

- Compiz Fusion is faster. They managed to streamline the code so it works better on my low end GPU (Intel)
- More features available
- Niver graphical effects
- They are both beta, but Compiz Fusion is the only that is still under active devellopment. So bugs will be fixed (and introduced :mrgreen:) faster.

### Post by technikalKP on 2007-07-25
I moved from Beryl to Compiz-Fusion, then back to Beryl.

I did notice Compiz tended to be a bit faster,  but the two things that sent me back to Beryl were:


- Inability to resume from Standby with Compiz:  Under Compiz, I'd often have the machine fail to resume from standby (Dell D600 laptop with ATI card).  I tried a few different suggestions, but nothing resolved the issue.  Beryl never had a problem with this.  

- Blur Effect didn't work - I'm not sure why, but even though I had the option enabled the 'blur' effect to make text on transparent pages more readable wouldn't never work under Compiz.  It worked fine under Beryl.  

I see constant updates for Compiz, so maybe these issues have been resolved.  I'll have to try it out again sometime soon.
