### Post by lindyslack on 2006-07-13
During the Ubuntu install you will have the option to edit partitions. If I remember corectly Fedora uses one partition by default and Ubuntu can do the same so... If you just choose to install Ubuntu on the fedora partition then fedora will be gone. Grub should reinstall without any issues and "bobs-your-uncle" no more RPM's :) Welcome to the world of GNU/Linux, Synaptic, and apt-get... :D
