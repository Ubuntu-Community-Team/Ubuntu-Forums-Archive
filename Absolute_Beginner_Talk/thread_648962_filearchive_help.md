### Post by Maxtorm on 2007-12-24
As a beginner, you're better off using the Add/Remove from the Applications menu. 

Some Synpatic packages don't have the menu configuration included with them, so you'll need to start them from a command line. The easiest way to do this is Alt-F2, then type the command. Generally, the command will be the same as the name of the package.

The archive manager allows you to open compressed files and archives such as gzip, bzip, zip, and tar. What specifically are you trying to download? Perhaps walking through a specific example would be instructional?
