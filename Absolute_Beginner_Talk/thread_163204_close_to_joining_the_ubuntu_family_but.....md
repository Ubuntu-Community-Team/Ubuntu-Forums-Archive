But I suggest to just run the LiveCD and see what happens!

### Post by localzuk on 2006-04-20
The current live cd is a seperate disc from the install cd. You have to download it seperately.

There is a program called 'wine' ([http://www.winehq.org](http://www.winehq.org)) which allows for execution of *some* windows applications. It is by no means perfect. For a list of applications and information go to the appdb ([http://appdb.winehq.org)](http://appdb.winehq.org)). You will most likely not be able to use hardware such as webcams with msn and yahoo messenger through wine as these aspects of the system are not implemented (as it would require huge amounts of work, and not even all webcams are supported natively on linux yet).

There are lots of books on linux, take a look at amazon.com and see what reviews say. For ubuntu, take a look at the documentation at [http://help.ubuntu.com](http://help.ubuntu.com) and [https://wiki.ubuntu.com](https://wiki.ubuntu.com)

### Post by localzuk on 2006-04-20
To re-enforce this. Linux is a culture shock when coming from Windows. This is due to most users having spent many years using Windows and as such learning its intricacies. Jumping into a new OS means that you want to be able to do everything you have learned to do on Windows over those years, straight away - which is not going to happen.

I would advise reading the documentation I mentioned earlier - it will save you a lot of time (for example, to get mp3's to play back it is as simple as following the instructions at [https://wiki.ubuntu.com/RestrictedFormats](https://wiki.ubuntu.com/RestrictedFormats) or using one of the scripts 'Automatix' or 'EasyUbuntu').

### Post by Mustard on 2006-04-20
I'd emphasise what has been said earlier.  A Live CD will give you a good idea of how well your linux install will work.  If the live CD has problems, then you can be sure that the install will be similarly problematic.  In learning to overcome problems to get the live CD to work, you should gain some insight into how to get the hard drive installation to work.

You definitely sound like you would need to setup a dual boot system, so that you can fall back to windows at any stage.  I'm not totally certain that linux is the way for you to go.  As mentioned earlier, its a bit of a steep learning curve at the start, and the simplest things can seem impossible (when you don't know how to do them).  It's a completely different operating system and that makes the whole 'language of linux' different.  Keep your windows install and dual boot into linux, and you will be able to take on the challenges of learning linux at your own pace.
