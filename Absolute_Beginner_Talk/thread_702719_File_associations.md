Find a .mp3 file and right mouse click on it.  Go to Properties and find the Open With tab.  Select a difference application.  I use Audacious, but select whichever one you want.  It will become permanent.

Rythmbox does not play .mp3 files by just clicking them.  Rythmbox will play them when it compiles all the music on the selected folder.  Use either VLC or Audacious.

### Post by ajgreeny on 2008-02-20
>  In the past I've just selected properties and changer the radio button here, now I can't change it here.Is that because the program you would like to use isn't listed in the "Open With" tab of properties, or simply that the change will not work.  Usually a program is not listed here unless you have already opened a file type by right clicking and selecting "Open with" and then "Other appliacation" and added the one you want.  From that point it should be listed in the "Open With" tab of properties, where you can make the change the default.

As to your "problem" of a .txt file opening in gedit, that is the system default for ubuntu and text files;  it doesn't matter which program was used to create it, a txt file is always a txt file and will be opened with gedit unless you change the system default.
