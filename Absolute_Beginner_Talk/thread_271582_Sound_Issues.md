Your post indicates you are using Ubuntu 5.10, which is outdated. You should seriously consider upgrading to 6.06.1. 

Second, it might be a good idea to copy the files onto a disk which has native linux formating, like the file system Ubuntu uses.

Third, do make sure you have everything you need installed to play music properly. Consult this webpage to make sure:

[https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)
