### Post by PmDematagoda on 2007-12-15
You can install Pidgin by entering:-

```
sudo apt-get install pidgin
```
in a terminal.

But if you have Ubuntu 7.10 then it is already installed and it can be found in Applications>Internet.

Why do you want the latest version? If there aren't any fixes for Pidgin that concerns Linux then it most likely will not be put onto the repositories.
