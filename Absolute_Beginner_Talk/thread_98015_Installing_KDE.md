### Post by 23meg on 2005-12-02
```
sudo apt-get install kde
``` should do it. If you want the whole Kubuntu suite of apps on your normal Ubuntu installation, do this instead: ```
sudo apt-get install kubuntu-desktop
```

### Post by linbetwin on 2005-12-02
Hello, Pisces (from a Libra)

If you want to install KDE you don't need Konstruct. Open Synaptic, enable universe and multiverse repositories and then search for a package called kubuntu-desktop and install it.

Alternately, you could just download the Kubuntu 5.10 ISO image. Kubuntu is Ubuntu with KDE instead of Gnome.
