### Post by aeiah on 2007-06-29
there is an ubuntu ultimate, which is an unofficial ubuntu dvd with everything on it, i believe. im not exactly sure whats on it, however.

i think as far as reducing filesize and whatnot goes, you are going to be stuck to downloading stuff. there's a nice program called aptoncd which you can use to make an iso cd image of all the stuff you've installed via synaptic / apt-get. so once you've done this once and got everything set up the way you want, you can burn the cd and have the normal ubuntu cd + a cd with your apps / drivers / codecs

this still doesnt avoid the use of the internet unfortunatley but if you could get to a university connection or something else thats fast, maybe you can have all the big stuff downloaded in one go
