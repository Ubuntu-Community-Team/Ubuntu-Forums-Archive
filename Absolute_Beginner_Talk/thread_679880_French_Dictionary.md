### Post by ubuntu27 on 2008-01-27
```
sudo apt-get install ifrench-gut
```

That will install ifrench-gut which provides French dictionary for ispell. (Which in turn can be used in other programs like OpenOffice.org

I also found this utility called ["Magic-Dic"]("http://magic-dic.homeunix.net/")
[http://magic-dic.homeunix.net/](http://magic-dic.homeunix.net/)
