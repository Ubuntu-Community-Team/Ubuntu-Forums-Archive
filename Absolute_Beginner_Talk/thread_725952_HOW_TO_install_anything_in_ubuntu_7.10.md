### Post by jken146 on 2008-03-16
A guide: [http://monkeyblog.org/ubuntu/installing/](http://monkeyblog.org/ubuntu/installing/)

Software that comes in tar.gz is usually just source code which you'd have to compile yourself.  Installing from the repositories is recommended whenever possible.

If you want VLC, search for it in Synaptic or open a terminal and type ```
sudo aptitude install vlc
```
