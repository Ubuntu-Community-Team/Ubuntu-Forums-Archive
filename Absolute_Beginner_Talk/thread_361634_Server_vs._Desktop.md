### Post by lamego on 2007-02-14
> Anyway, I was wondering if there was a list that compares what is installed with the server vs desktop versions.
The server version only contains server related packages (apache, mysql, etc), no GUI.
The desktop version includes the window manager and several desktop graphical applications.
You can always install graphical or server applications regardless of your base install.
