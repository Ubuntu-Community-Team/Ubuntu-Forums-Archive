### Post by enzomatrix on 2006-09-30
I've never used adept to uninstall VMware.
Just download the server from the official site:
[http://www.vmware.com/download/server/](http://www.vmware.com/download/server/)
Remember to get the serial numbers by clicking on the REGISTER NOW button.
Then just uncompress the .tar.gz downloaded, run the installation script and if it installed it will be uninstall it.
If it doesn't, there is an uninstallation script which depends on where did you install vmware.
