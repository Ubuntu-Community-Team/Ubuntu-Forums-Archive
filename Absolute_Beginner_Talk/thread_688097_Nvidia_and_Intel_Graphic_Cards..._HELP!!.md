### Post by kpkeerthi on 2008-02-05
Not sure about about intel cards but with nvidia its pretty easy if it has dual vga/dvi outputs.

Install the nvidia card, disable your onboard intel card in the BIOS. Install nvidia binary driver using restricted driver manager. Then open Applications -> System Tools -> Nvidia Settings (a.k.a nvidia control center in windows). You should be able to detect your second monitor, choose dual-display mode (clone, roght-of, left-of, etc.) all graphically.
