### Post by KIAaze on 2007-07-30
You can also just double-click it. :)

If it opens up as a text file instead of running it, just right-click->Properties->Permissions and make it executable.

When double-clicking it (and it is executable), you get the option to run it in a terminal, display as text file or run without terminal.
It's always better to run scripts in a terminal the first time, so that you can see error outputs.
Some scripts can even only be used in a terminal anyway.
