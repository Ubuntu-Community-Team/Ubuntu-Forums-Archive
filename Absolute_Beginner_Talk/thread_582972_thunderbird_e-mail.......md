### Post by JohnCub on 2007-10-20
I have Tbird 1.5

Edit menu
--Account settings (choose the account you wish to have a sig file for)
 about midway in that dialog box is an area that says "Attach this signature"

Check the box and tell it where your sig file is (which I think you can use plain text or markup files)
