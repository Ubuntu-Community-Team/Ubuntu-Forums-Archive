### Post by Hospadar on 2007-10-25
The older version is just fine.  The only real reason to install a newer version would be if you want added features (which will probably degrade performance on an old system, and arn't that big a deal anyways) or if you had hardware issues.

So if you hardware is working fine, and you have no major problems with the features, I would just stick with the older version.

Also, with 6.06, you might be getting older versions of certain softwares, so you'd have to manually acquire newer versions if you had issues with the old ones.  But again, if you are happy with how your system is running, this should be no big deal.
