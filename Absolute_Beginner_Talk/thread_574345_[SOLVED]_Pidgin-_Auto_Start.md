### Post by wil313 on 2007-10-12
Is there a way to make pidgin start up automatically when Ubuntu starts up?

---

### Post by aspen_dv on 2007-10-12
System-Preferences-Sessions
Start-up programs
New
Name : Pidgin
Command: pidgin
Then go to Sessions Options and save it.
