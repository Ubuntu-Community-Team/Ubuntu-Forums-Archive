### Post by kostkon on 2006-08-13
> **true_friend said:**
> nice.
i would like to upgrade my system then unlikely reinstalling ubuntu edgy.
is it possible to upgrade system from cd?????

Yes you can upgrade from CD. If you download the *alternate install cd* you'll be able to do an upgrade from that.

You'll need to change your repositories, add the cd as a source and then update and dist-upgrade. You can even use the iso of the *alternate install cd*, you don't need to burn it.

There is a thought for edgy, I think, for automating even more the upgrade from CD. We will see.
