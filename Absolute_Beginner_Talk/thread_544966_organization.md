### Post by Ubuntiac on 2007-09-07
You could install Thunderbird and Lightning (the calendar addon). Just open Synaptic (or Adept Manager in Kubuntu), select Thunderbird and choose to install it. Then visit:

[https://addons.mozilla.org/en-US/thunderbird/addon/2313](https://addons.mozilla.org/en-US/thunderbird/addon/2313)

And follow the 3 steps to install lightning. Voila!

BONUS! :) You can also get this to sync with google calendar using this rather natty free extension:
[https://addons.mozilla.org/en-US/thunderbird/addon/4631](https://addons.mozilla.org/en-US/thunderbird/addon/4631)
