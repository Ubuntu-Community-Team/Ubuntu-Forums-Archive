### Post by linux_believer on 2007-04-07
After an update frostwire crashed at start... we all know this. There was a frostwire update that included java6. 
I uninstalled frostwire and java 6.
installed Java 5 from repository
Installed frostwire from the frostwire site
 -- If you install from the repository it automatically  includes java6

Not really a fix, rather a work around to roll back the update. Now I make sure I dont do an automatic update for frostwire because that will include java6

Not really a solution but it gets you up and running
