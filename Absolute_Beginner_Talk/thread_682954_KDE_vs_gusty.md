### Post by Rocket2DMn on 2008-01-30
KDE is a more intensive Desktop Environment build on top of the X-server.  GNOME is what comes preinstall on a standard Ubuntu installation and is what most people use.  The interface is different, some people tend to say KDE is a little more like using Windows.
KDE also has some cool apps like K3b (for burning discs) and Amarok (for music), but these can be run under GNOME as well if you have the libraries installed.  Indeed, you can install both GNOME and KDE on your system to try them both out, though in the end you will want to settle on one since you would have to configure both separately which could get annoying to keep track of if you regularly use both.  I suggest you try both if you're interested!  If you have GNOME right now, you can download KDE:
```
sudo apt-get install kubuntu-desktop
``` and select it from the GUI login screen before you log on.
