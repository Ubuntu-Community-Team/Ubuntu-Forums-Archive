### Post by ugm6hr on 2008-04-02
Hang on.

Before you go giving up on Xubuntu, let's try and work out what is making it soooo slow.  192 MB RAM is not great, but I've had it faster with 128MB RAM (albeit v7.10).

What do you see on screen when you boot?  Is it a black screen for several minutes?  Or do you see a splash animation?

Is it responsive after getting to the GUI? Or is it still really slow?

If the latter, then you are right that you need an alternative.

It is worth trying adding gOS: [http://ubuntuforums.org/showthread.php?t=602640](http://ubuntuforums.org/showthread.php?t=602640)

Otherwise, PuppyLinux is a very good alternative.  IceWM is slicker, but will require menus editing.  IceBuntu looks good for this kind of thing (but I haven't tried it).

The problem with fluxbuntu and a direct gOS install is that they are LiveCD only, so may not install properly.  But, you can modify Xubuntu to be like them (see gOS above).
