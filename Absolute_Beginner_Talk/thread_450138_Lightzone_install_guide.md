### Post by arkiedan on 2007-05-26
Okay, I'm as new or newer than either of you but I've got LightZone running so here goes.

I'm assuming you've downloaded the compressed file from LightZone. Just right click on the file and it'll ask you if you want to extract the file. Click on extract and it'll unpack (I think that's the word) the file into a folder full of program files. Open the folder and look for a single file called the LightZone shell script. Click on that script and it'll start LightZone. There's really nothing more to it. 

I did have a problem that wasn't related to LightZone. I had to disable Beryl to get it straightened out. For me that was no biggie since I had grown tired of the Beryl 3d effects. 

I now use LightZone almost exclusively for editing my raw photos. It's an outstanding application.

Good luck,

arkiedan

The name of the file you are looking for is Lightzone, not "Lightzone shell script"
It's inside the folder you extracted. shell script is the "file Type".

I suggest you open that folder and from the "View" menu select "View as List", so that you can see the information better. Once you find that file you just double-click on it and Lighzone will run. Also you can make a launcher to that file on the Applications menu to run it from there.
