### Post by dmizer on 2006-09-29
actually the airport extreme is a wireless router.  at this point, all you need to do is choose a network card.

go here: [https://help.ubuntu.com/community/WifiDocs/WirelessCardsSupported](https://help.ubuntu.com/community/WifiDocs/WirelessCardsSupported)
and make sure you pick a wireless card that works in linux.  it will save you a lot of trouble.  also, you'll want to see if wpa or wep works as well.  some cards work perfect if you're not using wireless security, but have trouble with encryption.

so what it all comes down to is ... pick your wireless card carefully.
