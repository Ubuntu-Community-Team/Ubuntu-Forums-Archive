### Post by ecs_pf5 on 2007-06-26
When you download the CD, there's a checkbox:

> 
Check here if you need the alternate desktop CD. This CD does not include the Live CD, instead it uses a text-based installer.


If you put a tick in that, you will get the 'dead' (lol) version.

The normal one is the version that contains the live CD *plus* the normal installer.

The alternate CD is very useful if you have problems installing from the 'standard' CD. It uses a text-driven menu to install the OS, but does not have the run-from-CD capability.
