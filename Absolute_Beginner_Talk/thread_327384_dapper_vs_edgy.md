### Post by HereInOz on 2006-12-29
If edgy causes you no difficulties, then probably stay with it.  The main difference between the two apart from the goodies that edgy has that dapper does not, is that dapper is going to be supported with updates for 3 years (5 years for the server version I believe) whereas edgy support will run out in (I think) 18 months from the release date (correct me if I am wrong with the time) so people wanting a stable and long term installation would probably stay with dapper.  

However, if you want the latest stuff that is in edgy, and it all works for you, then it is probably the one to go for.

Hope this helps.

### Post by mcduck on 2006-12-29
Every new Ubuntu version takes the latest stable versions of apps and puts them together. After that usually only security updates are made for the packages in Ubuntu repositories, so now Edgy has the latest stable versions and Dapper's packages are a bit older.

Usually going for the latest Ubuntu version is a good idea, but since Dapper is supported for longer than usual time Ubuntu developers introduced a lot of new and a bit experimental things in Edgy. Because of this it's a bit unstable release and might or might not work for you.

But if it is working for you, go for it as getting the new versions of apps in older Ubuntu is not always that easy and smooth thing to do..

In my case the new kernel in Edgy brought better support for my laptop, now my wired network card and memory card reader work out-of-the-box, and the kernel also handles my Core Duo CPU better than Dapper's did. Also there seem to be some new default apps, like Tomboy panel applet and the Disk Usage Analyzer are included by default. And the boot is definitely faster, and higher resolution boot graphics. And Gnome's panels are fixed and now transparent background images work again.. So there are lots of small changes that are not so clearly visible to every user.
