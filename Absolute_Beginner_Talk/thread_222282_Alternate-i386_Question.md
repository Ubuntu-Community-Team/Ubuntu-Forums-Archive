### Post by Jagot on 2006-07-24
Yep it is a standard installation, but with more options thrown in if you need them.  It is a text based install rather than the live cd graphical installer you have on the desktop cd.  You can see the process of installing with the alternate install cd here:

### Post by GuitarHero on 2006-07-24
The live cd is concidered the standard install, but the alternate will give you the same outcome.  The live may be easier because it is graphically based and lets you try it out with a live session first, but if you dont want to try it first without altering your system the alternate cd is just fine.
