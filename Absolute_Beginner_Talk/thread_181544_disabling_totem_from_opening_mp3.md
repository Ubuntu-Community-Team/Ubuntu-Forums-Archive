### Post by mlind on 2006-05-24
if you're using gnome open nautilus (file manager) and find a .mp3 file

right click that .mp3 file and choose Properties
then choose Open with and select desired application.

### Post by glinsvad on 2006-05-24
Eeh, it works for me. Using amaroK though, but same difference right?
Rightclick mp3 -> Properties
In Open With-tab click Add and select program
Leftclick radiobox (white circle) left to the program you added to select it (black circle)
Done
