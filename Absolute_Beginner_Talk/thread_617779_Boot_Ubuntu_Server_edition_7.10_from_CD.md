### Post by TheOrangePeanut on 2007-11-19
No, you can't use the server installation CD as a live CD.  You'll have to use the desktop version.  If you want to see what using the server CD as a live CD is like, boot up the desktop CD and when you're at the login screen press ctrl-alt-F2 :)
