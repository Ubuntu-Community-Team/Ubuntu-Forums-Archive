### Post by Tyke91 on 2007-09-16
depends on how badly they have crashed.

if it's just the program that's not responding, you can set a minimize window shortcut by going to System>Preferences>Keyboard Shortcuts

if it's had a very bad crash, i find that restarting the computer is often the only way to fix it (crashes have a nasty way of affecting the computer even after the offending program has been closed
