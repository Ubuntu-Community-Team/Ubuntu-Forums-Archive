### Post by zachariah on 2006-01-19
The X Server is the program which allows the graphical user interface (GUI) to appear and make everything look pretty and user-friendly. You may have heard of GNOME and KDE - these are associated programs which present the screen to you. Without them you're stuck with the command line.

When I installed the live CD I did a complete install, which installed and configured the X Server automatically. If yours is not doing that, you may have an obscure graphics card which is not supported.

More info is needed - what is your hardware, and what is this 'something' it says?
