### Post by Paul820 on 2007-11-20
A **.bz2** file is just like a zip or rar file in windows. You can right click on it and extract it on your desktop. I don't think you will be able to install on your system though, you will have to run it from that folder. In fact i think it would be better running it from the folder just in case something goes wrong with it. It is a beta after all.

### Post by sandpaperback on 2007-11-22
vDub, nice to see you.

Running the beta from the extracted folder doesn't work for me.  It just starts the stable 2.0 version.

If you're intent on using a testing version for daily use, you can type this from the terminal:

```
sudo apt-get install firefox-granparadiso
```

This is not the newest beta, but I believe once (if) they upgrade the version in the repos, you'll get an update notification.
