### Post by Aircooledmadness on 2007-03-20

Adding programs to startup?  I can show you how to add them to Gnome startup...

System > Prefs > Sessions
Click on Startup Programs and add away!!!

### Post by alienexplorers on 2007-03-20
To load a program at startup you can put it's name in System>Preferences>Sessions.
Go to the startup programs tab and click add.  Browse to the program you wish to start and enter it.  On next reboot it should start from Ubuntu load.
