### Post by skymt on 2006-09-26
You can find lots of native games at [The Linux Game Tome](http://www.happypenguin.org/) or [the UDSF](http://doc.gwos.org/index.php/Native_Games).

Windows games are a bit trickier. Some games you can play with a program called Wine (Wine Is Not an Emulator). It's free, but kind of tricky to use. To check if Wine can play a game, look at the [Wine Application DB](http://appdb.winehq.org/).

If your game doesn't run on Wine, you might want to look at [Cedega](http://www.transgaming.com/), which is a commercial product based on Wine. It's not free, but it's easier and supports more games.

### Post by UncleOwl on 2006-09-26
Of native games, there is for example Battle for Wesnoth ([http://www.wesnoth.org](http://www.wesnoth.org)) which is a free, multiplatform strategy game (thus works on Win and Mac too). There are some 3D action games which are starting to develop on Linux, cannot recall examples now. For more childish stuff :) there are Frozen Bubble and Planet Penguin Racer (aka Tuxracer), both of which can be installed from Ubuntu software repositories (see [http://www.ubuntuguide.org](http://www.ubuntuguide.org) for the former). 

For older DOS games, there is Dosbox ([http://dosbox.sourceforge.net)](http://dosbox.sourceforge.net)). Works with ancient stuff but also with more complex games like Master of Orion 2, Jagged Alliance or Darklands. AFAIK it can also be installed straight from Ubuntu repositories.

For newer things, as already mentioned, are Wine (works out of the box for e.g. Diablo 2, but not so well with others; [http://www.winehq.org](http://www.winehq.org)) and Cedega (proprietary with monthly subscription fee; [http://www.transgaming.org)](http://www.transgaming.org)). Some folks at our home play WoW with it...

There is also a bunch of  older proprietary games which were ported to Linux by a software company called Loki Games. The firm went bankrupt due to some people running away with the money, but the games are still sold in some places online (some years ago, I bought Heroes of Might and Magic III which can still be run on Ubuntu). See [http://www.lokigames.com](http://www.lokigames.com) for more infomation.

Hope this helps a bit.
