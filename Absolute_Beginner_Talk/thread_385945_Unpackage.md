### Post by Jensor on 2007-03-16
I down loaded Adobe Flash player tar file and it I have these instructions.  I'm okay up to step 3 which says unpackage the file. How do I do this? What does unpackage mean?

Jack Ensor

.tar.gz installation

   1. Click the "Download .tar.gz" link. A dialog box will appear asking you where to save the file.
   2. Save the .tar.gz file to your desktop and wait for the file to download completely.
   3. Unpackage the file. A directory called install_flash_player_9_linux will be created.
