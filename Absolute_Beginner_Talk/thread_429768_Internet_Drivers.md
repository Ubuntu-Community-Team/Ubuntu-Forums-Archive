### Post by KCormier on 2007-05-01
I'm new to linux but networking i've done enough of.  I recommend a linksys router.  For the price, the wrt54g is a great router and you can just shut the wireless off for now.  There is no way to turn it back on unless you're on the network (that means no one can drive by and turn it back on!).  On top of that, you always have the option to go wireless in the future if you so choose :).  My only reocmmendation if you do that, is upgrade the firmware on it from the regular linksys firmware to the dd-wrt firmware!  [http://www.dd-wrt.com/dd-wrtv2/index.php](http://www.dd-wrt.com/dd-wrtv2/index.php)  It adds all sorts of options you will never use ::rolls eyes:: but it's also much more stable.  The nice thing about the wrt54g, is that it runs on linux as well!  This means other people are able to write their own firmware for the router!  Yay for embeded linux!  lol.
