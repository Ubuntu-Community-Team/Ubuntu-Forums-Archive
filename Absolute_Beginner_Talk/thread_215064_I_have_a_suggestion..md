### Post by iceshaft07 on 2006-07-13
This isn't as big a problem on Ubuntu as it is on Windows but I would like to throw my idea out there.

Quite frequently, I find myself typing something, whether an e-mail, program, or instant messenger, and a popup will come, and before I realize that anything has popped up I have already pressed space and answered "Yes" or "Okay" to the question I don't know was asked. Granted Ubuntu doesn't have many/any viruses, it could be a problem if, for example, someone I don't know sends a file via GAIM, that I would have normally said "No" to.

I simply propose that if you are typing, it will not recognize the keyboard until it realizes you have stopped typing. Or, the popup will come up, but without focus.
