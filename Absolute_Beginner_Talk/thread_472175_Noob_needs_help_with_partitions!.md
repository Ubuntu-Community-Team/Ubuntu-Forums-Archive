### Post by dpar on 2007-06-12
When you boot from the LiveCD, it should give you these options.
Strart or Install Ubuntu
Start Ubuntu is safe graphics mode
check cd for updates
memory test
Boot from first hard disk

This is taken from the Feisty LiveCD, so yours might be slightly different.
Pick Start or install ubuntu.
When its done doing its thing, you will be on the desktop which will have an icon for Install Ubuntu.
DOUBLE CLICK IT!
