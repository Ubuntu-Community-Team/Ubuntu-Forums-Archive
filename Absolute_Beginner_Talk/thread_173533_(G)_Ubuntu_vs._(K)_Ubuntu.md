### Post by Kvark on 2006-05-10
If you have a fast connection then install the package called kubuntu-desktop to get KDE then you can choose between Gnome and KDE at the login screen. Those who installed from the Kubuntu disk with KDE need the ubuntu-desktop to get Gnome instead if they want to try both. Try both and see for yourself.

The only downside of trying both is that you may find some great KDE programs and some great Gnome programs so you end up using programs from both. That is perfectly fine but they don't look exactly the same which annoys some people when using programs from both side by side.

Which one is better is entirely a matter of opinion, like Coke vs Pepsi. So don't listen to anyone who says the one is better then the other. One common way to describe the difference is that Gnome is cleaner while KDE has more features.

### Post by niviche on 2006-05-10
> I've only been using Ubuntu for 2 days now, but I absolutely love it so far. Apparently I have the Gnome version of it - I apologize if my terminology is incorrect, as I said I'm just a noob.

The easiest way to tell is the following: the default installation of the Gnome version is brownish, while the one for the KDE version is blueish.

>  Anyway, with reading through other forums and websites about Linux I discovered that there is a new distro - KDE 4

KDE is a Desktop Environment, not a distro. Let's say that a distro is a car, the Desktop Environment is the appearance and design of the stirring wheel and accessories.

KDE 4 will be the next version of KDE. It is planned for very late this year or 2007. The current version of KDE is 3.5.2

>  this KDE is integrated with Ubuntu in an OS called Kubuntu.

Kubuntu is a version of Ubuntu where KDE is installed by default, while Gnome is not.
(_)Ubuntu is the version with Gnome installed by default.

However, note that:
1. you can install KDE or Gnome from the repositories. This means that you don't have to reinstall everything to try one or the other.
2 you can use KDE software while on Gnome, and vice-versa.

Kubuntu is not an OS. It is the same thing as Ubuntu (the same car), but another Desktop Environment (the stirring wheel looks and feels different).

> which is better, Ubuntu or Kubuntu, and why? What are their advantages/disadvantages?

You are opening the gates of Hell. :) Gnome and KDE have their fans and their myths. Both are good, and both can get the job done.

The best one is the one you prefer. [COLOR="Red"]Don't listen to anybody who tells you that one is better/slower/nicer than the other[/COLOR], but just try them out.

For example, if you want to try KDE, open a console window, and type:

```

sudo apt-get install kubuntu-desktop

```

then you can log-out, and choose to log-in in a KDM (KDE) session. The job is done. :-)

Have fun, and good luck with Ubuntu, whatever flavor you prefer. :)
