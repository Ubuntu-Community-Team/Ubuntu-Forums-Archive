### Post by Barge on 2007-11-06
Yeah what you want is Screenlets. Me giving help is like the blind leading the blind a bit :D but  the instructions I followed are at
[Screenlets Howto for Gutsy]("http://www.screenlets.org/index.php/FAQ#Installation_instructions_for_Ubuntu_Edgy_Eft.2FFeisty_Fawn.2FGutsy_Gibbon_users")
Basically following that howto will get it installed so you can set them up from System>Preferences>Screenlets.

If you want to install more from gnome-look.org or whatever I found that clicking install and pointing it to the tarball download of a new screenlet download wouldn't work for me, however the manual method they mention of creating a .screenlets directory in /home/<username> and moving the extracted screenlet folders into that worked a treat.

Hope that helps.
