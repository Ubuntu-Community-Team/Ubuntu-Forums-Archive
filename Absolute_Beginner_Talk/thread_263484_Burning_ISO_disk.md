I use gnome but I still got k3b because I like it.
```
sudo aptitude install k3b
```
or via Synaptic. I think it's also in Add/Remove. Not sure!
If you install k3b under Gnome, you will get unnecessary dependencies...but that's how the cookie crumbles!
K3B also runs an automatic md5 check prior to burning the image.

### Post by Sef on 2006-09-23
If you're using Gnome, then put your cursor on the icon and right click. Next just go down to 'Write to Disc'.  Have used it numerous times and never had a problem burning an iso.
