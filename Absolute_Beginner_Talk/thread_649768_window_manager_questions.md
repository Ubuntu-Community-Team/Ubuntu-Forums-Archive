### Post by taurus on 2007-12-25
You just install fluxbox from synaptic, Add/Remove, or apt-get/aptitude.  And at the GUI login screen, you can choose which window manager you want to use for that session in the Sessions, bottom left.

So, you can have both Gnome and fluxbox on the same machine if you want.
