You can install HL via steam, and it will run. Get the latest version of wine, and grab the installer for steam from Valve's website. 

Wine will prompt you for a Mozilla control...let that install. I think there is also a font dependency that you need, but I don't recall which one...I want to say Tahoma. 

More here:
