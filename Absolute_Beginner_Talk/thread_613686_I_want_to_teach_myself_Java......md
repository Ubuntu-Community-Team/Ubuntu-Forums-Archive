### Post by ByteJuggler on 2007-11-15
Yes, for a full featured development environment, I would also suggest Eclipse.  

However, I would strongly suggest you actually start out initially using only a basic text editor and the java command line compiler, so you actually understand how things work "behind the scenes" before letting the IDE hide all that from you.  In my view it's quite important to understand how things work behind the scenes and not just learn the ways of the IDE of the day.

### Post by xiombarg on 2007-11-15
Hi,

Do you have a specific reason for learning Java? If your aim is to step into modern programming, then a version of object oriented (OO) C may be more useful to you.

I learned Java in uni, and it's OK, but if you want good run time performance, C will generally be a lot faster.

Java does have plenty of good documentation, and programming tools, though. It can certainly serve as a reasonable intro to OO as well.

Just thought I'd mention this, in case you spend a lot of effort learning Java, only later to find there may have been a more useful alternative.
