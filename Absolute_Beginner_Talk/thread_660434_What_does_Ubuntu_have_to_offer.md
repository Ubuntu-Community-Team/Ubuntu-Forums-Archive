### Post by eapache on 2008-01-06
GIMP definitely supports JPEGs out of the box. Please explain your problem here more thoroughly.

To use your m4a files, you need to go into Add/Remove Applications and install Ubuntu Restricted Extras (they can't be included by default because of copyright issues).

Pidgin does support those things (if I understand what you're referring to). Try browsing under Tools > Plugins.

Games you'll have less luck on simply because most developpers write their games purely for Windows. You can try wine ([www.winehq.org](www.winehq.org)) but it won't work for everything. Your best bet here is to set up a dual-boot and use Windows for games. Windows generally only gets viruses if connected to the internet, so if you only use it for gaming, delete you internet settings and you'll be trouble-free.

You tried amsn, it offers those features that you want (offline messaging and audio), but you didn't like it.

Try other IM like ["emesene"]("http://emesene.org/trac/wiki/WikiStart"),  
[http://emesene.org/trac/wiki/WikiStart](http://emesene.org/trac/wiki/WikiStart)

Install isntruction:

Open the terminal (Application menu/Accessories/Terminal)
and type:

```
sudo gedit /etc/apt/sources.list
```

A Text Editor called "gedit" will be opened. Copy and paste the following text to the last part

> deb [http://apt.emesene.org/](http://apt.emesene.org/) ./
deb-src [http://apt.emesene.org/](http://apt.emesene.org/) ./

Now, save it and close it.

in the terminal type the following (you can copy and paste):


```
sudo apt-get update && sudo apt-get install emesene
```

That's it. Now, you can launch it from your internet Menu.
