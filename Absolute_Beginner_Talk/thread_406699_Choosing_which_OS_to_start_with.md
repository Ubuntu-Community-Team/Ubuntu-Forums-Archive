### Post by aysiu on 2007-04-11
> **doncasteel8587 said:**
> Thanks to everyone so far for your help.

Something I came across that I'm not sure I understand is the "Live Trial CD"?

From what I've read, it sounds like I can make a bootable CD from an ISO file that will actually allow me to boot my computer with the Ubuntu OS test drive it, with no changes to my current system? Yes, that's it exactly! Just keep in mind that, depending on your system specifications, the live CD (also called the Desktop CD) could run a lot more slowly than a regular installation.

> If I like it... the installation is right on the CD? Yup. Just click the *install* icon on your desktop.

> Is this the same download as the regular installation or a different download? This is the same as the regular download. If you have lower system specs or some special need (a server or OEM installation), there's another installation CD that doesn't have a live session--it's called the Alternate CD.

**My advice**
Play with the Desktop CD's live session for two weeks. Really decide if Ubuntu is for you before you commit to an installation. You can never be too cautious when switching. If you don't believe me, read [this thread](http://ubuntuforums.org/showthread.php?t=405610).

And, more importantly, back up any important files you have and defragment Windows before attempting a dual boot.

If you back up your files, you're assured 100% that your data will be safe.
If you don't back up, and you know what you're doing, your data is probably about 99% safe.
If you don't back up, and you don't know what you're doing, your data is probably about 50% safe.

### Post by aysiu on 2007-04-11
I wouldn't exactly call Xubuntu a bare-bones installation. It's a lighter desktop environment, but it is still a full desktop environment (with a taskbar, a file browser, taskbar applets, etc.). Xubuntu is good for systems with 192 MB-256 MB of RAM. It's also good if you just happen to like Xubuntu and you have more than 256 MB of RAM.

But of 128 MB of RAM or less, you'd want a real bare-bones--a window manager like IceWM or Fluxbox.
