### Post by Nikusan on 2006-05-21

In an ideal world it would appear in Applications > Games. If it's not there, select the package in synaptic, right click > properties > installed files. You'll want to find something in /usr/bin probably. For example if you find /usr/bin/3dchess, the command you want to run is 3dchess.
