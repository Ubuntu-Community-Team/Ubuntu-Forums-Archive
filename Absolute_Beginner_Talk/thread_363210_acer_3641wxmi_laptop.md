### Post by annda on 2007-02-16
don't worry. just defragment D: (or even do it twice). then, when you install ubuntu, make sure you don't check the option that uses/erases the entire disk, but use free space on D:. you will be fine.

but save all your important data/documents etc. first. the worst thing that can happen is that you will need to reinstall windows (EXTREMELY unlikely) and this will clean your windows.

### Post by annda on 2007-02-16
ubuntu install will not damage any files when resizing your existing partitions (unless something goes terribly wrong, but i can't think what that may be).

however, after resizing the partitions etc. windows may report some errors (again, not likely, since you won't be using the system partition on C:). in that case, don't worry and click OK when windows asks you to check for errors. it is normal and healthy behavior after disk partition changes. that will not damage ubuntu.
