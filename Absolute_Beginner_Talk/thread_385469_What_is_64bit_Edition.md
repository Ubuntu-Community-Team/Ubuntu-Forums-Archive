### Post by mikewhatever on 2007-03-15
Yes, the flash player only exists in 32 bits version. Hopefully things get better in the future, but for now, the 64 bits Ubuntu is harder to setup, some programs will not work etc. The advantage is a faster desktop, especially for programs that use alot of RAM.
