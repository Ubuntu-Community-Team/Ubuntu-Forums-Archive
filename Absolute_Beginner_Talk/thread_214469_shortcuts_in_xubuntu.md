### Post by Brunellus on 2006-07-12
you will want to look into the idesk package, which is in the universe repositories:

[http://idesk.sourceforge.net/wiki/index.php/Main_Page](http://idesk.sourceforge.net/wiki/index.php/Main_Page)


Note--what you mean by "shortcut" is really called a "launcher."

It's a personal pet peeve of mine that Windows has made users believe that all of the following are "Shortcuts":

1)  Application launchers--double-click this icon to launch a program.

2)  Hyperlinks of any sort.  All URLs are "shortcuts" in microsoft-land.

3)  Linked objects using the Object Linking and Embedding software.

4)  Soft or symbolically-linked directories, which point to other "real" directories.

Each is a very different thing, which does somethign different.... and using the same vague term ("shortcut") makes things annoyingly imprecise.  Good for marketing, bad for tech support.
