### Post by jordanmthomas on 2008-02-16
Ubuntu server edition is much less than Xubuntu as it doesn't have any desktop stuff installed.  Upon installation, you'll be greeted with a command prompt and nothing else.

1.  Yes, it's possible to do for free.  Be careful, you need to make sure CenturyTel allows you to run webservers.  I know Comcast has a rule against it.
2.  Sure, it's likely to work.  Ubuntu server edition comes basically ready to host a website.  Just add your webpages and you're done.
3.  Depends if you're willing to put in the time to set it up.

could you have a wiki?  Sure thing, it's pretty easy to set up.
have games?  Certainly, especially if it's just flash games you serve up or something.  Keep in mind actually MAKING games is pretty involved.
You're not limited to anything.  Take any site on the internet, and you could likely do something similar with only free and open-source software.  Most of the Internet runs on Apache on *nix computers.
