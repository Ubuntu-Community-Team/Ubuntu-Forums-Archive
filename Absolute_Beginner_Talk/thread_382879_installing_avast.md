### Post by mikewhatever on 2007-03-12
All you need to do is copy and paste the commands. During the installation, the following directory was created /usr/lib/avast4workstation/share/avast/desktop. The cd /... command takes you to that directory. The sudo ./install-desktop-entries.sh install command creates a launcher in the menu.
