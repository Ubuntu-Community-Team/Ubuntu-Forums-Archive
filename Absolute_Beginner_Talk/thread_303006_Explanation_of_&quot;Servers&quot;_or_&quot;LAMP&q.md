### Post by Tomosaur on 2006-11-19
A server is a computer which accepts incoming connections from other computers, and sends ('serves') them information. Websites are run on servers, your school/college/university probably has a network of servers. A client is a machine which connects to a server. So, your computer is a client, and the websites you visit are stored on servers.

A server is a pretty broad collection of things really. A popular web server is Apache. This lets other computers connect to yours, and depending on what you have stored in your /var/www directory, will show the a web page, or let them download files, or anything really. A web application lives on a server.

You may want a server if you want to be able to play your music files from another computer, or you just want to be able to connect to your computer from somewhere else, for whatever reason.
