### Post by KIAaze on 2007-07-09
Well, it fits the minimum system requirements, so I would suggest trying it out. (you haven't gotten anything important on that 2GB disk, right?)

If it doesn't work, you could also try the Debian net-install CD (without connecting to the internet since you want a minimal system).
Debian is also quite user-friendly, altough the install procedure might scare you (text based).

If both don't work, then yes, go for a small distro like DSL or Puppy Linux.

edit:
And to help you keep the distro small, check out those programs:
-localepurge
-debfoster
-deborphan
