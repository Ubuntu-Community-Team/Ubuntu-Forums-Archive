### Post by petersjm on 2006-11-08
In case anyone's interested, I got myself a new printer and it whizzes like some fantastically futuristic thing! I opted for the [HP PhotoSmart C3180](http://www.shopping.hp.com/webapp/shopping/product_detail.do?storeName=storefronts&landing=printers&category=photosmart_printers&orderflow=1&a1=Function&v1=Print%2C+copy%2C+scan&product_code=Q8160A%23ABA&catLevel=2) which I picked up in the UK for about GBP-70. I'm still having some problems with HPLIP (every time I try to access hp-toolbox it tells me I need to restart HPLIP!) but the printer works well and I'm more than impressed :D

### Post by bonzini on 2006-11-08
We have a recent HP, a Photosmart 2575 all in one, bought about a month ago at FNAC in Grenoble.  A great unit so far, does a nice job of everything.  Builtin network interface works well.

I gave it a fixed IP address as it seemed to be annoying our Livebox and this has been the only problem.  Our two windoze computers run a load of HP gunk that talks to the printer most of the time, but my Ubuntu laptop just has a standard printer connection, no software installed or anything.

Haven't tried scanning from the Ubuntu desktop.

I have an older Epson photo printer back in Vancouver.  Impossible to get it to behave on the network, using a Linksys printer adapter.  Nice output though, when it works.  Absolutely burns through the ink, might have to get a second mortgage if we continue using it :-k
