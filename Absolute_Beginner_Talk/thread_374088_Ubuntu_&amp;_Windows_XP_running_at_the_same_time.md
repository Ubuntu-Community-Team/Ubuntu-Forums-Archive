### Post by docshawnc on 2007-03-23
Hi -
   I don't know about VM Player, but VM Server (free) works very well running out of Ubuntu.  An XP installation in VM server is not slow in the least, and you can flip/copy back and forth between Linux and Windows without problem.  The only draw back comes if you are trying to play games which require proprietary drivers for your video card - they unfortunately don't exist for VM machines (at this time) so you won't be able to do any serious fragging.
