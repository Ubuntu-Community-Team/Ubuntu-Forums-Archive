### Post by aysiu on 2006-04-09
[QUOTE=chimerla]
I have an iPod mini. Is it possible to use an iPod with Ubuntu?[/quote] Yes, but the newer the iPod, the newer the version of Gtkpod you need. I think Mini's are a little old, so you should be good. > I also have a Canon digital camera, is there software I can use along with this as well? Digikam? Your digital camera should just show up as a removable USB device. > I'm also wondering how drivers will work. If I were to install Ubuntu, would I still be able to install the drivers I need for sound, video, etc? It should be able to detect most stuff. If it doesn't, you usually end up editing configuration files instead of installing drivers. The best thing to do is to use a live CD to see how well Ubuntu detects your sound, video, and internet. The live CD doesn't affect your hard drive. It uses only the RAM and the CD to run, so it'll also run a little more slowly than a regular Ubuntu installation. > I was also wondering if there's a way to run Windows-only programs on Ubuntu without actually needing an additional installation of Windows on the system. It depends on the program. Some programs can be run in Wine. Others need Crossover Office, which costs money.

### Post by incubus on 2006-04-10
The way I usually convince people to use Linux is to first try it on a virtual machine, because:

1. If anything goes wrong or you don't like it, you can just delete the file; and try again.

2. People are reluctant of even trying a Live CD (which doesn't touch the hard drive). A virtual machine can run in your current OS, similar to a normal application.

If you are in windows right now, I'd suggest you to try VMware, QEMU, or even Microsoft Virtual Server (which is free now, competition does miracles). You can install one of them even in your laptop and try.

There are good tutorials in this forum, in case you like the idea.
