### Post by ramjet_1953 on 2007-05-25
A good alternative to Skype is WengoPhone.

[http://www.openwengo.com/](http://www.openwengo.com/)

It supports video calls, allows you to pre-pay, like Sykpe for landline calls, and best of all, is open source and uses the SIP protocol.

Another interesteing feature is that it allows you to make encrypted calls.

All versions are identical, regardless of platform.

All you have to do is to convince your friends using Skype to switch to WengoPhone.
