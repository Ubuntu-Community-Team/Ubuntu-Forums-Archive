### Post by erm4gh on 2007-10-26
I have no idea how well Ubuntu detects changes to the hardware. One possibility would be to create a separate home partition (so that you can keep your data/account) and then re-install Ubuntu off of a live CD telling it you want to manually partition drives - reformatting your root and swap partitions. 

See [http://www.psychocats.net/ubuntu/separatehome](http://www.psychocats.net/ubuntu/separatehome)

I'm using a AMD-64 with 2GB memory and decided it wasn't worth the hassle to deal with 32 vs 64 bit issues so I installed the i386 version of Ubuntu. In part thats because I saw too many people upgrade from XP to Windows Pro x64 for the wrong reasons. I wouldn't immediately assume you should choose the build for "64bit AMD and Intel computers"
