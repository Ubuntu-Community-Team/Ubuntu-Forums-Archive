### Post by jken146 on 2007-12-02
The volume control is on the top panel by default.  If for some reason it isn't there, hit Alt+F2 and type gnome-volume-control.

If you install the package gnome-alsamixer then a powerful-ish mixer GUI can be accessed under Applications>Sound & Video.  You probably won't need this though.
