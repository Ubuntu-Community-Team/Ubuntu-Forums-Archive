### Post by dpzektor on 2007-05-24
I noticed today that when I enabled to view hidden folders/files and then navigated to a directory (in Nautilus) that I wrote a text file, there was two other copies of the same text file, normally hidden. For example, I wrote a readme for something, aptly named "Readme". When viewing hidden files, there is a "Readme~" file there too. Why is this?

---

### Post by vtel57 on 2007-05-24
Usually a file with a ~ in the name is a backup created by the text editor you used to save the file. Note: this does not occur with command line text editors like Vi or Vim, only with graphic editors like Kate or Gedit.
