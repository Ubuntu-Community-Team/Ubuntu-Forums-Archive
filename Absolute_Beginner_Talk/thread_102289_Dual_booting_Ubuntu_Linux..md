### Post by MonsterBox on 2005-12-11
Depends on how you have everything set-up. You need two things for Ubuntu to run effectively:
1.) Partition for Linux and
2.) Partition for Swap Space

If your current configuration has some free space (another drive letter that is used for back-up/etc) then you could install Ubuntu to that drive and break it up into 2 partitions (OS/Swap) right from the Ubuntu installer tool. This is how I set-up my desktop.

If you only have 1 hard drive and it is all Windows (or whatever your current OS is) then I think you would need to get a third-party partitioning tool in order to create a partition of "empty space" that you could use to set-up Ubuntu.

Hopefully this makes some sense. ;)
