### Post by suomalainen on 2008-02-06
Ubunteros,

I get email from a few sites that is automatically generated.

I'm experiencing the problem whereby NOT ALWAYS the entire email is viewable. Although some emails have this link, i.e. :

"If you do not see the graphics below, click here to view in a new window."

Is there a way to get my Evolution mail client to work so I can see the graphics instead of having to click a link???

Thanks!

---

### Post by Golem XIV on 2008-02-06
Yes, select "Load images" from the View menu or press Ctrl-i.

This is a standard feature in all new mail clients to protect from spam.  The images are not in the message body but are linked to a site somewhere.  This is a possible security vulnerability, which is why outside images are never displayed by default.

If you don't mind the possible risks, go to the Edit menu and select Preferences; under Mail preferences -> HTML mail set the "Load images" to what you like.
