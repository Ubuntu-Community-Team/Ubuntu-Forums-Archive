1.  I think you will have to see if your router can be set to B&G rather than G only (mine can). I don't think you can receive from a G transmitter using a B receiver!
2.  WPA is for wireless security - not required if your PC is directly connected. So if you can set the router to G&B you should also set wireless security to WEP.
