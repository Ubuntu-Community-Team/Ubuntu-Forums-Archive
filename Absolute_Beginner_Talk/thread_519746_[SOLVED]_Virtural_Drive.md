### Post by Hospadar on 2007-08-07
If you are wanting to mount .iso files, you don't need any special application, you can just mount the .iso file as if it were a drive.  try right clicking on it, and if that doesn't work, use the "mount" command through the text terminal.

If you need some form of copy-protection emulation/cracking as provided by alcohol or daemon tools, I don't think there are really any linux softwares that do that, but you might try running alcohol or daemon under wine (I have no idea if/how well that would work though)
