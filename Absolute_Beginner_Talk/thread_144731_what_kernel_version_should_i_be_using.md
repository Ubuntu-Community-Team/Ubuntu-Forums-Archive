### Post by mcduck on 2006-03-15
[QUOTE=SD-Plissken]You should use k7, and install the k7 headers as well.[/QUOTE]
You might want to install 'linux-k7' instead of installing any specific version of the k7 kernel. This way you'll get headers and restricted modules and everything, and all those will also get automagick updates too.

Version go like this: 386 works for everybody, 686 works for both new Intel and AMD CPU's  and K7 is for Athlons and later AMD CPU's. SMP kernels are for computers with multiple CPU's, multicore CPU's or hyperthreading.
