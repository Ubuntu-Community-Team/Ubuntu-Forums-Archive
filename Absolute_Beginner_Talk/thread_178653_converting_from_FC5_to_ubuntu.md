### Post by tribaal on 2006-05-18
In short, the is no "easy way" to do that.

The major problem is that FC and Ubuntu don't use the same packaging system... And it's tricky even when trying to "copy" an ubuntu setup onto another machine, so...

If I were you I'd patiently install the applications you need in Ubuntu using apt-get whenever you need them, and copy all the files (including hidden files!) from your FC home directory to your Ubuntu home directory.

Hope this answers your concerns?

- trib'
