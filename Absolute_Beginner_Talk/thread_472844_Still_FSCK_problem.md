### Post by DBStevens on 2007-06-13
I think you'll need to comment out that UUID line for /media/sda6 and possibly change the nodev,nosuid on the /dev/sda6 to default.  The error message provided by fsck is because of the UUID entry not being findable.  So fsck couldn't continue.

fsck.ext3: Unable to resolve 'UUID=57904835-2f33-41a7-84ad-f673a7fca109'
