### Post by O-pos on 2007-05-12
Hello,

I don't like to have menu bar in my GNOME Terminal. there is an option View>Show menubar ,which can enabled/disabledm but it works only as long as that particular terminal window exists. After closing and reopening terminal, there's again that menubar. Any ideas how can I disable it forever?

Thanks in advance,

O-pos

---

### Post by Metacarpal on 2007-05-12
Go to Edit > Current Profile
In the General tab, there is a check-box for "Show menubar by default in new terminals"

Uncheck that, and you'll be good to go.
