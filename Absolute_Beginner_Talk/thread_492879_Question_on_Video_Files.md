### Post by igknighted on 2007-07-05
> **Darth_Maul said:**
> I am making a Dvd with Devede and the original clip is in avi format 170mb  with it converts the file to mpeg it is now 738 mb  it dont understand why it does this?

Different formats tend to do this.  As an example, find any .jpg file.  Now convert it to a .bmp.  File size jumps by over 100% (my test file went from 770KB to 1.7MB).  Jpegs are a compressed format (instead of specifying each pixel's value, it says "ok, the next 5 pixels are all red" to cut down on size).  This is the same with video formats.  I don't know the specifics about mpeg and avi (hence why I explained it with jpeg/bmp), but I am quite sure it is the same phenomenon.
