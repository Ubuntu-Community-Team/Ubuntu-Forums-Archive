### Post by Ek0nomik on 2007-08-20
Ubuntu comes with a web browser if that's what you're looking for.  It's called Mozilla Firefox.

There have been a lot of people who have gotten WoW to run on Ubuntu, but it will require other software.

Installing WoW:

[http://www.desktoplinux.com/news/NS4169902864.html](http://www.desktoplinux.com/news/NS4169902864.html)

[https://help.ubuntu.com/community/WorldofWarcraft](https://help.ubuntu.com/community/WorldofWarcraft)

[http://ubuntuforums.org/showthread.php?t=312482&highlight=world+of+warcraft](http://ubuntuforums.org/showthread.php?t=312482&highlight=world+of+warcraft)

What each person does after an install of Ubuntu is really subjective to each individual.  I personally installed an audio player, Amarok, an e-mail program, Thunderbird, and a bittorrent client, Deluge.

If you want to install software graphically:

System/Administration/Synaptic

You will find a **ton** of software in the repository.  :)

If you have more questions feel free to ask...
