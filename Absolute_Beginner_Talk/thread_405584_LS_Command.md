### Post by ardchoille42 on 2007-04-09
> **brennydoogles said:**
> What is the best way to get a tree view output from terminal of a directory, all subdirectories and their contents in terminal?

There is an app in universe called tree. It outputs a treeview of a dir and its subdirs.
Enable the universe repo and then:
```

sudo apt-get install tree

```
