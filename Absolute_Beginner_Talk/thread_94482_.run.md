### Post by Pablo_Escobar on 2005-11-24
Open terminal
CD to the directory where You have the .run file f.e. 
> cd /home/pablo/games/et
then type in 
> chmod +x nameofthefile.run
and then
> ./nameofthefile.run

Remember that bash is case sensitive so use TAB for autocomplete.
