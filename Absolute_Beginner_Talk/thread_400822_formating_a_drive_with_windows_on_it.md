### Post by Tilex on 2007-04-04
If Windows takes the whole 40gig of your hard drive (i.e. it's not already partionned), then you will erase Windows upon installing Linux.

What I would suggest is if you really want to try out Linux, start by dual-booting Windows and Linux.  For this, you will need to partion your hard drive and you can do so by re-installing Windows.  So you could backup your files, re-install Windows and make sure to create 2 partitions (like 2x20gig).  Then, once Windows is installed, you can install Linux and tell it to install on the free partition.  This way, when you boot your system, you will be prompted to choose between Windows and Linux, and both operating systems will share your machine.
