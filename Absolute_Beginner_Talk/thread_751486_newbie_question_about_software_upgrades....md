### Post by arochester on 2008-04-10
I have a G3.

You can only update one version at a time e.g. Feisty to Gutsy, Gutsy to Hardy. Not Feisty to Hardy.

Just because a piece of software isn't in repositories doesn't mean you can't install it. For Firefox 3 go here: [http://linuxappfinder.com/package/firefox](http://linuxappfinder.com/package/firefox)

Should you install a .deb. I do. If if won't work it will tell you.
