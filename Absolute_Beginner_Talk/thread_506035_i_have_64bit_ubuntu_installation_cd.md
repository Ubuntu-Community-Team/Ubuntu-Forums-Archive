### Post by bigfox on 2007-07-21
If you have a 64 bit capable processor, it will work.

If not, it will not.  It won't even install, so it won't hurt anything to try.  If your machine is only 32 bit, the installer will not even be able to load.

In that case, get the 32 bit version of Ubuntu.



If your machine is 64 bit, it will be faster, however there are 2 known caveats.

Adobe hates Linux, and only provides an ancient version of Flash that is only 32 bit.  It can be beaten and hammered to run under 64 bit, but it is a royal pain.  (I hate you Adobe, I hate you so much.)
