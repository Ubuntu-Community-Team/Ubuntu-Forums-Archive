Nope, as long as it has a DHCP server and a web interface, you make all of your changes that way and any computer with an ethernet cable and a web browser can connect through it.

### Post by shareMenaPeace on 2007-02-02
Select the "?" icon on top of the Desktop panel, to start the Ubuntu Help Centre - UHC.
Select Working with your desktop.
Select Internet.
Select Connecting to the Internet.
Select ADSL Connections.
