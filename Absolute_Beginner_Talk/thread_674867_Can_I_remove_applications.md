### Post by thelatinist on 2008-01-22
Yes.  And I removed Evolution without any problems, too.  It will tell you you have to remove a package called "ubuntu-desktop" which seems scary, but don't worry.  Ubuntu-desktop is an empty package that makes sure you get all the default apps when you first install.  You don't need it unless you do a version upgrade, in which case you can just reinstall it.
