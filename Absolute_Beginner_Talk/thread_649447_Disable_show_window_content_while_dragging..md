### Post by AmericanYellow on 2007-12-25
These options are located in Compiz Config Setting Manager which can be gotten for Synaptic Package Manager or by typing this in the terminal:

sudo apt-get install compizconfig-settings-manager

Once installed, you go System--> Preferences--> Advanced Desktop Effects Setting

You can add and remove all the effects you want from there. You will also be able to enable effects that you would never know exist.

Disabling Maximize/Minimize can be done in "Animations" effect or "Minimize" effect.
