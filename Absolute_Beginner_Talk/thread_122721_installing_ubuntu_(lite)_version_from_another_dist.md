Try using UNetbootin at [http://lubi.sourceforge.net/unetbootin.html](http://lubi.sourceforge.net/unetbootin.html) download and install the "Ubuntu" version (use the .sh version), when prompted for the desktop environment just deselect them all, then once done, you'll have a bare-bones console-only environment, from there you can install the various packages for Ubuntu Lite (IceWM) via apt-get.

If you need more instructions, see [http://ubuntuforums.org/showthread.php?t=427540](http://ubuntuforums.org/showthread.php?t=427540)
