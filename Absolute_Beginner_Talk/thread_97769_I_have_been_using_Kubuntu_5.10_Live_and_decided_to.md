Besides the other responses to your question, even if you are running gnome under Ubuntu, you can also install K3B.  Assuming you use synaptic to install it (or the command line), any KDE dependent libraries will also be installed so that when you run K3B, all will function just like it was running under KDE.

Likewise, if you were running KDE and wanted specific gnome apps, say evolution, the required libraries would be installed so that you could run it under KDE.

Before going the K3B route, though, you might want to try graveman or simply the drag and drop method through nautilus (which, if I'm not mistaken, is how Apple's OSX works, too).
