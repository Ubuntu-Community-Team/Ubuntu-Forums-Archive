### Post by quirks on 2008-04-19
Try configuring the proxy settings in Synaptic:

1. Go to System -> Administration -> Synaptic Package Manager.
2. In Synaptic, go to Settings -> Preferences.
3. Switch to the Network tab and enter the proxy settings.

I'm not sure, if this works, but I wish you good luck. Don't know why there are separate proxy settings for Synaptic. They really need to consolidate that.
