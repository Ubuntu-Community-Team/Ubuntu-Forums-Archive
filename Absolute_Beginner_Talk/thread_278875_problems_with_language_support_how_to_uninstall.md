### Post by dmizer on 2006-10-17
i'm going to make a few asumptions here because you don't say.  so if this is incorrect, please provide more detailed information.

first, i am assuming you are using gnome as your window manager.  previously, to get language support to work correctly, you should have followed these directions here: [https://help.ubuntu.com/community/SCIM](https://help.ubuntu.com/community/SCIM)

second, i'm assuming you've chosen to use scim to enable chinese input.  in which case ... your settings can only be turned off but not removed.  according to everything i can find, the only real way to get rid of scim and chinese input is to reinstall.  which ... if you've made a seperate home partition, shouldn't be a problem.
