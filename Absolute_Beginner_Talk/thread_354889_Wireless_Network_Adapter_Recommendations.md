### Post by grendel38 on 2007-02-06
I just got a Belkin F5D7010 v.2 (using the rt2500 chipset)

It seems to get generally good reviews.  Just make sure to get one with the rt2500 because it's one of those annoying cards that use different chipsets for the same model #.
