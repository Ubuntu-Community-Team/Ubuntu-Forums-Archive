### Post by gerrymoth on 2007-04-23
It can run with even less than 10GB, most threads I've read state a partition size of 8GB is adequate.

I've got a dual boot system on a 60GB HDD and have Photoshop for WinXP, but don't use it much now cos Gimp for Linux is just as good and its free.

I created a Live CD from the downloaded Ubuntu 7.04 ISO file and booted up from the CD to try Ubuntu before I installed on my PC, that way you can check if your Wifi works and try the pre-installed applications (which includes Gimp).

I'm totally using Ubuntu now for surfing the web, watching DVD's and playing mp3's. The only time I boot up using WinXp is when I need to sync my Nokia phone, but I'm trying to find a solution for that.
