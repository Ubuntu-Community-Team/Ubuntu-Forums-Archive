### Post by vexorian on 2007-08-07
I just heard of kxdocker which seems to look well and has the drag and drop you need.

Anyways for the docks I really have tested:

- Kiba : compoziting required, no reflections, great plugins
- AWN : compoziting required, now stacks and reflections which are the new cool things, but still very bad desktop switcher and no menu / clock.
- simdock: Very lightweight, reflection, no compoziting required. Only that it has no plugins and still looks unpolished.
