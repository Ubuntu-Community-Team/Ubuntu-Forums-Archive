### Post by GhostZrydA on 2007-10-13
Right click on desktop, create new launcher, call it compiz, then under the command put "compiz --replace" without the quotes.

Run that and it should start up compiz.

Put that same command into the startup option under "sessions" on the system/preferences menu, and compiz will start when you run ubuntu.

I got it setup really well by by following this blog how to [http://forlong.blogage.de/article/2007/8/26/The-best-way-to-install-Compiz-Fusion-on-Ubuntu-Feisty](http://forlong.blogage.de/article/2007/8/26/The-best-way-to-install-Compiz-Fusion-on-Ubuntu-Feisty)

After following that first entry, make sure you setup the plugins as well, it's well worth reading to see what the plugins do.
