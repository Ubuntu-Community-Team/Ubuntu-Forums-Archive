However, as most people will tell you, the only reason to have anti-virus is to protect Windows users from you. ### Post by ramjet_1953 on 2007-06-26
You already have a firewall automatically configured and all ports are closed. It is called iptables.

There are packages available, like Firestarter and GuardDog, which allow you to manually configure iptables for special setups, like port forwarding, etc.

However, iptables is best left alone, unless you have speacial reqirements.

As far as anti-virus software is concerned, you really don't need it, unless you are forwarding-on emails that have attachments to Windows users. Even this is really becoming unnecessary now, as most isp's scan all attachments, anyway.

Loosen-up and enjoy the security of Linux.
