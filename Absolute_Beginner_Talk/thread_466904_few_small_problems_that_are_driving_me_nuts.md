### Post by Inxsible on 2007-06-07
You can fix your Cd problem in kernel -16 :
 
1)
```
ls -l /dev | grep cd
```
 
This will list the permissions on the drive and also an alias at the end of the line. Do the same with dvd
```
ls -l /dev | grep dvd
```
 
If you have only one drive, they would all map to the same name. Mine for eg. was hda
 
2)Then edit your fstab:
```
gksudo gedit /etc/fstab
```
 
Change the line for the CD from```
 /dev/scd0 /media/cdrom0 ..blah blah
```
to ```
/dev/<alias name you found in step 1> /media/cdrom0 ..blah blah
```
