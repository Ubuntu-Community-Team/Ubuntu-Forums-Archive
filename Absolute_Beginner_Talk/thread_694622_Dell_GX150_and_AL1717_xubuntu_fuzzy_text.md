### Post by dstew on 2008-02-12
There are several things to try. Enter the command **sudo dpkg-reconfigure xserver-xorg** and try different settings. See if there is a i810 driver choice. You can also experiment with the horizontal and vertical refresh rates, and with the resolution choices. You can also try the vesa driver. If you want to be able to revert to your old settings, save your /etc/X11/xorg.conf before reconfiguring.

### Post by dstew on 2008-02-15
> Horizsync 30.0-81.0The upper limit seems excessive. Try 28-51, or other values.> Vertrefresh 55.0 - 76.0The values are probably right, but there are spaces around the hyphen. I don't know if that makes any difference, but it might. My xorg.conf doesn't have spaces. Also, 55-76 might be a little high, my laptop LCD has values 43-60.> Virtual 1280 1024
Modes "1280x1024@60"I am not sure if the Virtual statement is a good idea. Also, why put the @60 in there? Let the display pick a refresh rate. If that forces it to use 60, maybe the resolution won't be good.> screen 0 "Default Screen" 0 0Why are those zeros in there?

Your xorg.conf has lots of things that look like you have hand-edited it several times. Maybe save this one as a backup, and try dpkg-reconfigure xserver-xorg to make a fresh one.
