### Post by Majorix on 2007-12-01
Try the open source radeon drivers. I use them with my Radeon X800. They are worth at least as much as the closed source ones.

To do that, boot Ubuntu in recovery mode, and do this:
```
dpkg-reconfigure xserver-xorg
```
and choose "ati" for the video driver when asked. The "ati" is just a wrapper that will load radeon whenever possible.
