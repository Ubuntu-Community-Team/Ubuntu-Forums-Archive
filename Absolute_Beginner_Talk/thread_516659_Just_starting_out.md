### Post by p_quarles on 2007-08-03
I'm not sure about the joystick. If the one you have doesn't work, I'm betting that you can problably find one that does control the mouse pointer. I've just never tried.

The sticky keyboard is a definite yes, though. Go to the System Menu, select Preferences, then Accessibilty, and finally Keyboard Accessibility. There, you'll be able to set the repeat timeout to whatever is needed.
