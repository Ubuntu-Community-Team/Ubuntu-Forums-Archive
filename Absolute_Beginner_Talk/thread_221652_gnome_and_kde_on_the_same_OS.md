### Post by deanjm1963 on 2006-07-23
If you have Ubuntu installed and you want KDE, use Synaptic and look for Kubuntu-Desktop, and install it, and if you're using KDE and want Gnome, look for Ubuntu-Deskstop. It will install all the necessary files, and you can chose which desktop you prefer to log into as default, and you can switch between environments at the log-on screen.
