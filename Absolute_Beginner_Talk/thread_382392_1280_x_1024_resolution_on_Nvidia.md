*Open a terminal
*Run:
```

gksu gedit /etc/X11/Xorg.conf

```
*Enter your Password.
*Edit the file like following  ( I changed the resolutions for other depth subsections too.)
*Save it
*Restart X using Ctrl+Alt+Backspace
*Login again.
