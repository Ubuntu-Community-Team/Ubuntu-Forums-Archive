### Post by dlpfmVfH on 2006-06-16
the live cd tries to figure out which resolution to use...
sometimes this makes the screen out of line....

to change this, you need to use the buttons on you're monitor...
when installed, you can change the resolution, and refresh rate to match you're monitors and the problem should dissapear,

It is just a problem with right resolution, wrong refresh rate... or the otherway round...

hope this helps.

### Post by Jagot on 2006-06-16
Yep, as aboe says using the monitor settings is the only way to go with the live cd.  If you do go ahead and install then make sure you install the correct drivers for your video card.
