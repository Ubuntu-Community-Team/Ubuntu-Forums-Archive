### Post by t0p on 2007-12-21
When I started using Ubuntu (back in the days of Breezy) I had no internet access at home.  So I would go to the local library and use one of their computers to visit [http://packages.ubuntu.com]("http://packages.ubuntu.com") or [http://archive.ubuntu.com]("http://archive.ubuntu.com") and download software that I wanted.  I'd save it to CD, then take the CDs home to install to my computer.

Unfortunately, this method often led to **dependency hell** - I couldn't use Synaptic's ability to resolve dependencies automatically.  But still, I was able to access the vast amount of software that is available for Ubuntu.
