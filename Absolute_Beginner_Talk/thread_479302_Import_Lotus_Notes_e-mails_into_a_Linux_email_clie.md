### Post by orange7 on 2007-06-28
The only solution I know to this problem is to use Transend Migrator (or a variant) on a Windows system to convert your mail files from the Lotus Notes format to the Thunderbird format. You can then copy the Thunderbird files onto your Linux system and read them directly. 

It's not a clean solution and you need special software on a Windows machine but it worked for me and I haven't had to think about Lotus Notes since then.
