### Post by mc4man on 2007-12-09
An R1 drive will play R1 titles only, R2 for R2 titles ect.
The region setting of your drive or disc's doesn't really matter in linux, the software player should bypass it. You do need to have a region set i_nitially_ in the drive. If your mainly viewing R1 titles set it to that. There's a package called regionset for setting the region of the drive
note: to be clear you only need to have set a region once
reading  [https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)
