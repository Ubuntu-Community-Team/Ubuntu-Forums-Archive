### Post by frup on 2008-01-21
I understood it as him calling us his teachers so maybe no need for Edubuntu.

Note that what ever you want to achieve from the server edition you can learn on normal Ubuntu. The server edition just has no gui and comes with certain common software installed such as Apache, MySQL and PHP. These can all be on normal Ubuntu so while you are learning you could set up 2 machines as client and server with both normal Ubuntu and practice and use the server edition once you know why you need it.
