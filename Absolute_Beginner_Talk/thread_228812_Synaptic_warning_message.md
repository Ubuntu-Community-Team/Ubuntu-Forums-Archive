### Post by linuksamiko on 2006-08-03
This message allway acourse when ever you don't add the public gpg-key of the server to you system. This is only a warning that your computer doesn't know the server from which you try to download. 
You can install those packages!

But you should add the gpg-key to your system that your machine can check if that is realy the server you like to connect to.

This is how you might add a gpg-key:
1. download the gpg-key provided by the server
2. sudo apt-key add /here/is/the/file.gpg
