### Post by mdsmedia on 2007-01-14
At the risk of adding another post to a lot of good advice, please take it all into account before taking one piece of advice and running with it.

I've read through the 5 pages here, and I'm confused!! I've installed Ubuntu several times on 2 machines, and yet I'm confused with the advice that's been given. Thankfully I know enough now to extract the wheat from the chaff.

From the advice given above, you need to download a .iso CD image. The .iso is a burnable file which will create the CD to install from. Once downloaded, it doesn't matter too much what Windows thinks the file is. If it has a .iso filename extension it's probably OK. BUT you DO need to save it to disk rather than open it from the website.

The "desktop" version of the installation files allows you to run a LiveCD of the OS, meaning you can try it out before installing. Then you can use the same CD to install the OS. The "alternate" version I believe is the recommended version for effective installation. I used the desktop version and had no problem installing Dapper Drake (the 6.06 version).

The installation guide at hermanzone was the one I used to first install Ubuntu a year and a half ago. It's very good. The one at psychocats.net was the one I used most recently when reinstalling. It's also very good.

When reading advice on these forums please remember that you will usually get lots of good advice, but you need to read it along with other advice, because people who are helping here, including, and especially myself, are just users helping out. Our advice comes from personal experience and doesn't necessarily apply to your situation. It also doesn't take into account every issue that may have come up or may come up.
