### Post by aJayRoo on 2008-01-03
On my laptop the key is on a sticker on the base. The operating system files are stored in a folder on the C: drive called i386. If that folder exists and is around the size of a CD then that is likely to be the Windows installation disk, in which case you can copy it (as data, it is not a CD image) to a blank disc. I would of course recommend testing it out before overwriting Windows! By the way my laptop was from Dell, it didn't come with any installation media however there was a program on the Start Menu to create backup media and all this did was burn the files from the i386 folder to a disc.

Note that I am not advocating pirating software in any way, but suggesting you legally obtain the media to install the software for which you have a valid key for.  If you don't have a copy of Windows, and do not have a valid key, you need to go buy one of each at your local electronics store.
