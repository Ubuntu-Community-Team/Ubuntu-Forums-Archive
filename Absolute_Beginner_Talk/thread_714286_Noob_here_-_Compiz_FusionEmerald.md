### Post by justsomedude on 2008-03-03
Compiz is a 3D window manager.

You can change your window borders in Emerald, which is a window decorator.

Icons, wallpaper can be changed under System --> Preferences --> Appearance.

For the transparent terminal, open it and create a new profile under Edit --> Profile.
