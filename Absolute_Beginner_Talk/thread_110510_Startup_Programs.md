### Post by cmcgeecc on 2005-12-31
One way you could get it to start when you log in is to save current set up when you shutdown or logout.  Here's how.  When you logout or shutdown close everything except the program you want to run at start up and then make sure the box is check that says save current set up above the choices of shutdown or logout or suspend or whatever choices it gives you.  This should work as long as the program is compatible with save setup.  Let me know if it works for you.
