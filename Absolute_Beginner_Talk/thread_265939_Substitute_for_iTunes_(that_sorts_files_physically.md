### Post by aysiu on 2006-09-26
You can rename your files based on their tags using a program like TagTool.

That's what I do, and I actually find it a lot more convenient to see them all in one folder in the file manager than digging three folders deep to find a song (that's the way iTunes does it).

I name my songs *artist* - *track number* - *song name*.
