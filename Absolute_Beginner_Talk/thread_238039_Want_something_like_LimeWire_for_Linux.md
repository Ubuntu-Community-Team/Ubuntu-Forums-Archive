### Post by abu-fatu on 2006-08-17
i also found that frostwire slowed my system down.
it seems to me that limewire is faster.
this is how i learned to install it from this forums
1.install alien from synaptic package manager
2.download LimeWireLinux.rpm from their web site to desktop.
3.open terminal and type: cd Desktop
4. then type in terminal: sudo alien -i LimeWireLinux.rpm
when finished limewire would be in applications-internet-limewire

there is also Apollon.
