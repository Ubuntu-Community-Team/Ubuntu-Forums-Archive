### Post by Psquared on 2006-11-04
**Pros:**

font rendering is superior on laptops
battery module now works
package updates are timely
fast starting and running apps
uses GTK libs (in addition to its own)
themes are modern and most are useable
does not crash
comes with latest versions of Ooo
uses Thunar (Xfce) as file manager (evidence too, but not being actively developed)

[B]
Cons:[/B]

engage is very difficult to configure
sound is a little tough to configure but doable
wireless (ipw2200) is borked by updates (good news is NDISWrapper works)
firefox 2.0 not yet available
needs more working modules
skip the eyecandy for now; its useable as a workstation out of the box.

I have it on my laptop and I would abandon everything else if I can get wireless to work.
