### Post by JO3Y on 2008-02-17
I don't know if there is a plug in for rythymbox that lets you edit ID3s with AAC but you can use a free program called [Easytag]("http://easytag.sourceforge.net/") to get this done. 

to install easy tag just open a terminal and type
```
sudo apt-get install easytag
```

use is fairly straightforward, hope this helps
