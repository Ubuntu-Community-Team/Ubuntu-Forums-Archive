### Post by KIAaze on 2007-05-09
Since you are concerned about data loss:

I have resized my windows and GNU/Linux partitions a lot of times now (2 times for GNU/Linux install on a full windows disk, and maybe 3 or 4 times to delete, add, move, resize partitions (testing distros, resizing a too small /usr, adding space again to windows to install a game, reducing it again...)), and I have never had any data loss until now. :D

I used the Debian installation dialog for my first resize and since then, always the GParted Live-CD.
The Ubuntu installation disk uses GParted, so I think it's quite reliable.

All you have to do to be able to resize an NTFS partition is defragment it. All other partitions, you can move & resize as you wish (but moving can take quite some time)(and it's only a translation, you can't exchange the position of two partitions).

**Back-up of important data is of course still recommended.**
But as long as you don't format or delete any partition, the risk of data loss is rather minimal I think.
The most dangerous is probably resizing the NTFS partition, but many have done it before and I don't know a lot of people who had data loss because of it.

### Post by Carlos Santiago on 2007-05-09
First i recommend you to prepare your hard drive in windows.
For example using the hard disk management (C:\windows\system32\diskmgmt.msc)

Delete one or two partitions and _leave them unformatted_.
I recommend you to free at least 8 GB.

Then run ubuntu install, and select to use all free disk space. 
You just have to be careful to _NOT select entire disk_.
Select 'All free space', you will have no problem.

And yes, at every time you boot your PC, you will be able to select 
which Operation System you want to boot in: Ubuntu or Windows.
Dont worry about it.

before installing. And no changes to anything will be made while running in **LIVE** mode.

Once you are ready to install , a 7 step process as I recall, make sure to **read** the choices given at each step and you should have no problems. I just installed FF on one of my laptops last night.  It took about 30 min.
( I was also watching TV at the same time.)

You will also gain the ability to run most windoze documents in FF out of the box.  I just had an experience
where I created an inventory document on my desktop but when I tried to run it on my laptop at work I
lost the formatting (both xp pro) I found that this was because I had neglected to install that application on my
laptop(office 2k).  But when I tried it in Linux ( both xubuntu 6.06 and FF 7.04) it ran with no problems
and I could do all that I needed to do.

You will loose some things once you install and start to use FF.  You will loose the need to purchase and use overpriced winblows software.  While there some commercial linux software , all that I have seen is reasonably
priced and most has a free version you can try.

You will loose ( at least for the most part) the **need **to have anti-virus software and the need to update it daily.  In addition you will loose malware, spyware, and most if not all other *** wares.  I also think that as long as you dont run as Root you might be able to get by without a firewall, but that is iffy.

So as you become more experienced and comfortable with the terminal and command line you will gain
a wealth of tools, tweaks and possibilities.  If you should experience difficulties with hardware make sure
to contact the manufacturer.  As more and more people are saved from windoze and convert to  OSS,
if each one contacts the manufacturer of errant hardware they might just see $$$$$ slipping away and provide the necessary information for their product to work.
