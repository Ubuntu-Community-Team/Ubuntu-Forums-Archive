### Post by rkd on 2008-02-05
r4ik: I think he is trying to access a memory card (SD flash card, or something similar), not a sound card, so I think the pointer to a guide for getting sound to work isn't going to help.

kurian: Do you have other memory cards to try? It would be helpful to know whether the problem is happening with just that one memory card or with any memory card.

Maybe it would be useful to look in the system log to see whether there are any messages at the time you insert the memory card. Messages might give clues about why it isn't working. You can look at the syslog by going to System->Administration->System log. When the window opens, select syslog and scroll down to the end (or to the time when you inserted the memory card). I don't know what to look for. If you aren't sure how to interpret messages you see there, copy them from the syslog display and post them here so we can help you interpret them.

I don't have a memory card reader in my computer, so I don't have any direct experience with them, so I might be making unhelpful suggestions.  If someone who has experience using a memory card reader can respond, that would be better.
