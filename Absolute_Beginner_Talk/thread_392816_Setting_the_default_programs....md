### Post by schwascore on 2007-03-25
Right click on a file of the type you wish to change and go to Properties.

Click on the Open With tab and select the program you wish this file type to be opened with.

That should do the trick.

### Post by elcasey on 2007-03-25
In the preferences menu under "Removable Drives and Media" you can set what occurs when you do things like plug in an iPod or other MP3 player, insert a blank CD, etc. 

Because I don't like Rhythmbox and don't use Nautilus' CD burning capabilities, it was nice to be able to change those, too.
