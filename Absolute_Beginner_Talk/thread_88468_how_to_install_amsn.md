### Post by Darrin on 2005-11-10
Go to Applications>Add Applications.  The Add Applications screen will launch. At the bottom of the screen type in amsn. CLick search. It should show up in the list. Check mark the box next to it. Click apply in the lower right section of the window. It will install it for you.
