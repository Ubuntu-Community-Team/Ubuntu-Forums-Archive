### Post by JReagan1990 on 2008-02-12
Ubuntu accepts code written in Python, but there are pratically an unlimited amount of programming languages you can use.  

You can use C, C++, C# (Mono) and others as well.  

Check the add/remove applications, and search for any particular programming language, and you are bound to find something.  Eclipse is a good multi-language IDE for java, python, C, C++, etc.

(if add/remove apps does not have what you are looking for, you can also use synaptic, via System > Administration > Synaptic Package Manager)

### Post by Blutack on 2008-02-12
Python would be a good place to start.
Dive into python is included with the Ubuntu Help - hit F1 at the desktop and search for python.
Otherwise what you need depends on what language you want to learn!
You may also want an editor.  GEdit (the built in text editor) is ok but I prefer Geany.  Editors are a very personal thing so try a few and see which you prefer.
