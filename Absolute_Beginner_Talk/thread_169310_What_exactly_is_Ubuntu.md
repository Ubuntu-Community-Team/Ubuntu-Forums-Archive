### Post by aysiu on 2006-05-02
A Linux distribution is an operating system based on the Linux kernel.

Every distribution is its own operating system, even though they may appear similar and have similar sets of available software. A binary compiled for one distribution will not necessarily work on another distribution.

Each distribution is different from others in usually these respects:

1. Installer
2. Number of CDs
3. Package management (how software is installed and removed)
4. Default interface, default logos and graphics
5. Support community
6. Price
7. File structure
8. Live/install
9. Included packages

### Post by aggiechemist on 2006-05-02
I think everything so far is great answers, but I have one to add:

Ubuntu is a philosophy.

So are all Linux distros in some way or other.

Ubuntu believes in being universal, easy to use, having a reasonable amount of control for most users, having total control for advanced users, having a very strong community, having regular updates, etc. etc.

Other distros have their own philosphies. Gentoo is known for believing all software should be compiled from source. Knoppix was one of the first to make live CDs a big part of what they do. You could probably descrive a philosophy for most distros.

Hope that helps :)
