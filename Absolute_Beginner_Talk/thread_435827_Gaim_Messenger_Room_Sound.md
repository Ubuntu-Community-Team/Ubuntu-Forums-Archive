### Post by nightskywind83 on 2007-05-07
You can assign sounds in Gaim using preferences.

Go to Tools menu, then Preferences, then click the Sounds tab.  Then use the check boxes to assign sounds to the events you want.  Default sounds will be used but you can specify others if you wish.
