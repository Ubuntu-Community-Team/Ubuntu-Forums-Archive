### Post by tangibleorange on 2008-03-06
Well both Xubuntu and Ubuntu use the same package system, but the packages you wish to install may need additional packages called dependencies. These are automatically installed by the package manager if you have an internet connection, but if you don't, it can prove to be a problem, as you may have to search the internet for the required dependancy packages.

In short, there is really no issue if you have an internet connection on the Xubuntu machine :).
