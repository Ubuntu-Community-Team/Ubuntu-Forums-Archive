### Post by boydjj on 2006-12-30
Having recently installed Kubuntu, I've managed not to use Windows for three days. There are a lot of guides out there, but the simplest of them that I found is at [http://www.cs.cornell.edu/~djm/ubuntu/]("http://www.cs.cornell.edu/~djm/ubuntu/"). It's simple, and there's not much trouble-shooting, but it will get you off the ground. If you see weird portions, or portions you're not comfortable with, find a guide that deals with that step in more depth. (If you don't want to use Edgy Eft, there is also a link to a similar Dapper Drake install guide.)

A word of advice: The CDs you burn can be used as a live CD before installing. Boot into the CD and look around to see what hardware is/is not supported out-of-the-box with Ubuntu. Then, *before* you install, go back to Windows (or your OS of choice) and start looking for information about the drivers and such you need. This is *especially* important if your network connection isn't supported out-of-the-box or your monitor's resolution is wacky. If you have those two working, you can search for info from within Linux. These forums will be invaluable help at that point. Cheers and good luck.

### Post by Sef on 2006-12-30
> i just want to install ubuntu onto my computer, not burn it onto a cd.
how do i go by doing that?

If you don't want to burn it from a cd, then you have two choices:

1) Order from Ship-it.  Go to [ubuntu.com]("http://ubuntu.com") and click on ship-it and follow the instructions to have your Live CD sent to you.  This will be a Live CD, so you can check your hardware to make sure it is compatible with Ubuntu.

2) Do a [net install]("https://help.ubuntu.com/community/Installation/Netboot"). However, you have to have a floppy drive.

If you decide to burn a cd, read these sites:

3) [How to write an iso file to a cd]("http://www.petri.co.il/how_to_write_iso_files_to_cd.htm").

4) No cost [iso burner]("http://isorecorder.alexfeinman.com/isorecorder.htm") for Windows XP.
