### Post by meng on 2006-10-25
What txt file are you talking about?
If you want to install flash player, I would recommend enabling multiverse/universe repositories:
[https://help.ubuntu.com/community/Repositories/Ubuntu](https://help.ubuntu.com/community/Repositories/Ubuntu)
then in the terminal, the command:
sudo aptitude install flashplugin-nonfree
or these instructions for more detail:
[https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)

PS Your terminal is definitely working, by the way, it's a case of "garbage in garbage out"!
