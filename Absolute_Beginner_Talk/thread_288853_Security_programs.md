### Post by igknighted on 2006-10-30
Ah, the beauty of linux:  None of those programs are needed!  I know many people feel safer with a firewall, and there is a program called firestarter which you can get out of the repositories I believe (I am behind a router, so I don't bother).  As for antivirus, you can try clamAV, but really all it is useful for is scanning documents so you dont accidentally pass a virus along to another windows PC.

EDIT: When looking for programs for ubuntu, always check add/remove programs or synaptic or another package manager before you search google.  Your package managers search the official repositories so the software integrates with the OS better and if you ever want to unistall it is much, much easier.

### Post by mblinux on 2006-10-30
Avast is also available for linux. Your ubuntu already has a firewall running which is called iptables. Firestarter is a gui for iptables. It helps you view events and set up easily some basic rules.
