There are several distros that work well for Linux newbies. Linspire, Ubuntu, maybe Mandriva... but since you're here already, I'd recommend Ubuntu.
At the moment, there is very little gaming support for Linux. Gaming companies just don't think it has a large enough userbase.
There's a few games ported to Linux though. UT2004 and NeverWinter Nights come to mind.
I'm not exactly sure what Quick Par is, but you might be able to find an open source alternative to run on Ubuntu.

[QUOTE=Athlongamer]
There is so much i don't know and so much i want to ask but idk where to start....

That's good. The first step is a desire to learn. Just keep in mind that Linux isn't Windows, and you'll be okay.

> 
Can i emulate Windoze and then play CS:Source threw the Windows emulation...Will it be laggy?

Possibly, and probably. If you still want to run CounterStrike, I'd recommend running a dual boot.

> 
Can i just copy all of someones 'code' and paste it into mine and i'll have the same setup as that person???


Err...you may be thinking of source code, which doesn't exactly work like that. In order to install programs on your computer in Ubuntu we use a program called apt-get or Synaptic (a graphical user interface for apt-get). Apt-get downloads "packages" to your computer from a central repository and installs them.

### Post by wylfing on 2005-10-19
This is a lot of good advice, but I want to counter-argue something: Cedega is quite mature and does one heck of a good job running Windows games. I do not dual-boot. I am Linux-only. 

Some of the games I have installed with Cedega include Morrowind, Dungeon Siege, Diablo II, Red Alert (and variants), and Dune 2000. Now what's remarkable about this list isn't the games themselves, but the fact that I pulled random Windows games from my CD case, installed them, and played them without a single problem.

Transgaming maintains a list of games and how well they work [right here](http://transgaming.org/gamesdb/). From a quick glance it looks like CS:Source runs great. Far Cry is a supported game, so even though it has a few (correctable) issues right now I'd expect it to get a lot better very soon.

Also, Unreal Tournament 2004 has a native Linux binary, so there's no need for Cedega to run that. You should be able to use your existing install CDs and do a "real" Linux install.
