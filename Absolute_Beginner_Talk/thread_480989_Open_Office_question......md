### Post by Brucevdk on 2007-06-22
OpenOffice.org doesn't seem to include "convert text to column" functionality by default, however after a quick Google I found [an article describing the installation procedure for an OpenOffice.org Extension]("http://www.xaprb.com/blog/2006/06/26/how-to-convert-text-to-columns-in-openofficeorg-calc/") which does.

The installation procedure is:

[LIST=1]
[*][Download "text to columns" for OpenOffice.org Calc]("http://sourceforge.net/project/showfiles.php?group_id=87718&package_id=104183") and unzip it.
[*]Start OpenOffice.org Calc and use Tools -> Extension Manager to start the manager.
[*]Click add and navigate to the unzipped .oxt.
[*]Restart OpenOffice.org Calc (the Quickstarter in the notification area if enabled too).
[/LIST]

After doing so there'll be an icon in your toolbar which allows you to convert text to columns.
