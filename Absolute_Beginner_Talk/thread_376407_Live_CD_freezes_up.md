### Post by ramjet_1953 on 2007-03-04
OK, let's have a look at a couple of things, shall we?

Firstly, assuming that you downloaded the LiveCD, what speed did you burn the iso at?
If you burnt it at any speed faster than 4 X, you are headed for disaster. ANY iso burnt at high speed is not reliable.

Secondly, is the media that you burnt it on low cost? Again low cost media is not good for burning iso images.

If the above is all OK, you may need to download the alternate CD. Some systems don't like the graphical installer that the LiveCD uses and the alternate CD uses a text-based installer. It is still quite intuitave and you should have no problems in using it.

### Post by 3rdalbum on 2007-03-05
That's a laptop, right?

As laptops all use different ACPI methods, they can be problematic for Linux until their implementation is reverse-engineered. My advice is to get a cheap, outdated desktop and try with that; alternatively, buy a machine with Linux preinstalled. (System76)
