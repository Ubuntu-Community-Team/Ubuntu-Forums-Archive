### Post by Majorix on 2007-07-23
Why don't you use the graphical interface?

If you are using Gnome, right click a file or folder, select Create Archive, then choose .zip and give a name and place and you are done.

However .zip is not the best compression method. .tar.gz or .tar.bz2 are better ways to do this.
