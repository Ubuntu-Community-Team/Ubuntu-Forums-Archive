### Post by vinnn on 2007-06-02
Editing that /etc/X11/xorg.conf file is the way to get whatever range of screen resolutions that you require in most distros.
The newest versions of Xorg do away with the need to edit this file. This feature is just starting to appear in the latest distros such as Fedora 7 (released 2 days ago)


Bring up the "Run Application" box in Gnome by hitting [Alt] + [F2].

Type in the following; ```
gksudo gedit /etc/X11/xorg.conf
``` and press the Enter key or click on the **Run** button.

It's good practice to make a backup before you make changes so do a 'File' >  'Save As..' and save the file as **xorg.original** or something.

Then in gedit change **all** the lines that go...
```
Modes "1024x768" "800x600" "640x480"
```
to...
```
Modes "1600x1024" "1024x768" "800x600" "640x480"
```

**Make sure there are no typing errors**, then save the file as **/etc/X11/xorg.conf**


Once you're sure it's correct, simply hit **[Ctrl] + [Alt] + [Backspace]** to restart the X server and return you to the login prompt. Your display should be now 1600x1024 or you should at least have the option in **System > Preferences > Screen Resolution** when you login
