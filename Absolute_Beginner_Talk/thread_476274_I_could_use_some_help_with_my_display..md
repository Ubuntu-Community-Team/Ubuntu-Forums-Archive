### Post by H.E. Pennypacker on 2007-06-17
You will need to change the resolution, regardless of what you have been told by Ubuntu.  Try System > Preferences > Screen Resolution.  I am assuming there is only one option there.  If there is only one option, you will have to add more resolutions.  Start off here:

[https://help.ubuntu.com/community/FixVideoResolutionHowto](https://help.ubuntu.com/community/FixVideoResolutionHowto)

### Post by H.E. Pennypacker on 2007-06-17
If you go with a lower resolution, buttons and other things on the screen will appear bigger, but I don't think a lower resolution will actually fill up more of screen space.  You should look for higher resolutions, which will make desktop items appear smaller, but will take up for screen space.

Whatever screen resolution you use on Windows should be used on Ubuntu.
