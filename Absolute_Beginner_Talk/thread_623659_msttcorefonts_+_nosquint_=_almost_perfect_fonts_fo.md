after reading allot over fonts i came up with salution for better fonts in ubuntu. If you have firfox install the plugin -nosquint- my settings is permantly on 125%, i have a 24" lcd monitor so for other people this setting may be different.
After that open termanel and copy paste this code and run
> sudo apt-get update   this is to get a good font,,and now run the following command to install it
> sudo apt-get install msttcorefonts
The affect will take place after you logout and login again. 

This is very simple way to get better results,,after a while if you and i have a better understanding to install different fonts and so this method is good enough for now.

This helpt me allot and now i can cancel my appointment with my eyedoctor.
