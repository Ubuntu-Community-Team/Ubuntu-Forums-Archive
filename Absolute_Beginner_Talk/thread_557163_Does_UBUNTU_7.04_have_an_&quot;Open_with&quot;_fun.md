### Post by mikewhatever on 2007-09-22
I believe what you are looking for is not an option in Ubuntu proper, but rather an option in Firefox. Check out under Edit-->Preferences-->Content Tab-->Manage. You can change file type association with a program there. To some extent, this is already the case, as flash files open in the flash player, and most of the rest of multimidia in totem. The 'open with' function to open files on the hdd is present by default, but, once again, that's not what you want.
