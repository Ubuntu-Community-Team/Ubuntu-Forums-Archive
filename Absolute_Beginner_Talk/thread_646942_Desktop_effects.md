### Post by RomeReactor on 2007-12-21
Hi. Beryl is no longer being developed; Ubuntu has Compiz Fusion installed by default, so you should use that. If you meant Fusion, then maybe you need to install xserver-xgl, at least until the newer ATI drivers are available in the repositories. Go to "System->Preferences->Appearance" and on the "Visual Effects" tab select "Normal".
