### Post by nalmeth on 2006-06-22
Yeah, play around with the liveCD to get acquainted, and do some reading. Remember that Ubuntu doesn't come with non-free stuff that you might be used to like mp3 and video codecs, so you might want a script like Automatix to get all that setup.

For explanation see: [http://wiki.ubuntu.com/RestrictedFormats](http://wiki.ubuntu.com/RestrictedFormats)

I have a feeling that it will make things easier to install vista first, then Ubuntu. 

Windows isn't very friendly to non-windows systems. It won't corrupt your install in anyway, but it will most definately overwrite the GRUB boot menu (from which you pick which OS to boot), and it definately won't include ubuntu on it.

If you have any data on your old XP you want to save, BACK IT UP
Just in case something goes wrong and you overwrite your windows partition.

I would suggest the alternate install CD for installation, aparently a lot of people have had problems with the Desktop install CD. It worked fine for me though.
