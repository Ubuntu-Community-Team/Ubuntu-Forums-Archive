### Post by rsambuca on 2007-08-20

### Post by funrider on 2007-08-20
you need to burn the cd image to a CD. once you are done, place the CD in your drive and reboot your machine. 

Once the machine is power on, make sure you have your bios setting's boot order configured so that CD-ROM Drive is the first booting device.

If thing is all set, the ubuntu CD should boot up your machine with the on-screen menu.

let us know if you need more help!
