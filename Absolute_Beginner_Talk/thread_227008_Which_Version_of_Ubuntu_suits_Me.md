### Post by frodon on 2006-08-01
You can receive your ubuntu CD for free if you wish : [https://shipit.ubuntu.com/](https://shipit.ubuntu.com/)
You can also download them here : [http://www.ubuntu.com/download](http://www.ubuntu.com/download)

As for the version the 64bit version, it is good but you have to forget some package like wine, libdvdcss (needed to play dvd movies) and some other apps which don't exist in 64bit version.

I myself own an AMD64 3700+ and use the 32bit version, so it's really a matter of feeling, but for sure you have better performances with the 64bits version.
