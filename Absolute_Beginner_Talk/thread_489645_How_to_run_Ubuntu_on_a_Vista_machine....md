### Post by steve.horsley on 2007-07-01
I asume you usse windows? You mat do well to have a look at VMWare, a product that allows you to run one OS inside another OS. Google for it. They may well provide an image of Ubuntu alreadu installed inside the virtual machine. Google for VMware.

Alternateively, you could install Ubuntu in a dual boot arrangement. The Ubuntu installer CD can shrink a windows partition to leave free space for instlling Ubuntu's partitions (needs 2 partitions). Or just install Ubuntu on an old PC - it's not as hardware hungry as XP.

The "desktop" CD boots into a wowrking Ubuntu, running purely in RAM, and this might even be enough for you (if it has internet access - you will be wanting to download and install compilers etc). This live CD also has an install icon on the desktop where you can begin installation to hard disk. You need ot download and burn the ISO image to CD and then boot it. Here'e the download page:
[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

Do not be tempted by the server edition - sitck with the 386 desktop version.
