### Post by Mornedhel on 2007-08-29
Sometimes you can spot a tiny Tux on the box (I have the UT2004 box in my hand right now, and there's a small penguin in a corner on the back side, and lo and behold, there was an install shell script on one of the CDs). For other games you should check the editor's website.

And if somewhere the specs say the game uses or may be used with OpenGL instead of DirectX, chances are the developers also wrote an installer for Linux - if they didn't, at least it will be easy to WINE it. (For instance, UT2004 uses DirectX by default, but an OpenGL version was quickly written near the end of the development cycle : it's quirky, but works as well as the DirectX version would on XP.)
