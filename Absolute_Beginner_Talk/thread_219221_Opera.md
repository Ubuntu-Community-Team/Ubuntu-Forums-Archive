### Post by danpre on 2006-07-19
If you have already installed JAVA you can enable plugin for OPERA.

Go to: Tools > Preferences > Advanced > Content > 

there you will have ENABLE JAVA and button JAVA OPTIONS - press it and insert a patch to your java destination (on my computer it is: /usr/lib/jvm/java-1.5.0-sun-1.5.0.06/jre/lib/i386)
