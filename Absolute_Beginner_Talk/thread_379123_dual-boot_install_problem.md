### Post by Kateikyoushi on 2007-03-08
Before resizing the partition check the disc for errors then defragment few times, if hibernation or swap is in the second half of the disc disable them till you resize, and do not forget to backup important data because power surges, lightnings and other acts of God can happen any time.
