### Post by punx45 on 2007-01-23
i haven't used fedora (yet) so i can't speak to the differences.   i know major ones are package managers (apt for Ubuntu and yum for fewdora).   and fedora i believe uses kde by default rather than gnome.  also, if you are interested in a more server type environment, fedora is a free version of red hat, which is geared towards network administration, so it may come with more useful apps for that.   check out distrowatch.com for the lowdown and comparisons of different distros.

as for apache, i have it running on the desktop version.  (the difference between the desktop version and the server version is what apps and daemons you start out with, after that you can add or remove stuff to your liking) i havent used the server version but i would imagine you can configure the server to include apache etc. in the initial install.  

whether you go with server install or desktop install, the configuration of apache remains the same. (the difference being you can edit apache conf. files with gedit etc. rather than with vi(m) or emacs or whatever.

i encourage you learn to use the console as much as possible as it can become very useful, especially with servers.
