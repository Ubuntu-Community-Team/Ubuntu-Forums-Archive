### Post by xcat442 on 2007-03-13
Hi,

When the Ubuntu CD first boots up try "Start Ubuntu in safe graphics mode" See if this will get your computer started. If possible, start the computer this way, then do the install. If by chance this does not work, press F4 when the computer comes to the Start Ubuntu screen and try a lower screen resolution 800 x 600 x 16.

Hang in there, once you get Ubuntu installed there are many cool things to explore.

Peace.

P.S. There's a lighter version of Ubuntu that can run on less powerful hardware, this one may be worth a try.

[http://www.xubuntu.org/](http://www.xubuntu.org/)
