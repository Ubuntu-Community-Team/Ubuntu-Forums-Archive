The best way to learn is to jump in.  Your requirements will be filled by the basic install of Ubuntu/Kubuntu. ### Post by Murrquan on 2007-11-06
> **rioja said:**
> Ubuntu seems to offer the widest choice, but will it be able to cope with my above requirements?

Definitely! You'll be able to get software to do all that and more, and you'll love the Add/Remove Programs interface. It's like an enormous free software catalog ... just select what you want to install, and it'll automatically download everything and keep it all updated, too!

The real question is, will it support all your hardware? Lexmark printers and laptop wireless cards, for instance, have been a bit of a sticking point for some people. But if your hardware's not supported, there's often a way around that.

> **rioja said:**
> Secondly, is Linux a primarily visual interface (point and click), or a command line interface; where lots of code typing is involved?

I had to type a few commands to set up my install of Ubuntu, because it didn't support all my hardware. But the people here on the forums helped me out a lot, and I was able to see where people had already had my problems and dealt with them. It was mostly cut-and-paste. And everything else is done using the Graphical User Interface, which as the YouTube videos indicate is pretty sweet.[/QUOTE]

> **dward526 said:**
> Any questions you may have, you already know the right place to find answers.  YOU WILL BE FINE:)

Biggest thing to remember, right there. ^.^
