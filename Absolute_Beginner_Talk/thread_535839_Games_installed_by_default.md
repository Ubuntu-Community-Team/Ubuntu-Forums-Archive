### Post by KIAaze on 2007-08-27
[LIST]
[*] You should be able to find most of what you need here:
[http://ubuntuforums.org/showthread.php?t=359842]("http://ubuntuforums.org/showthread.php?t=359842")

[*] If you want the binary/source of a specific game, you can usually find it on its official site or on the sourceforge page.

[*] If you want any .deb package from the repositories, you can get it here:
[http://packages.ubuntu.com/]("http://packages.ubuntu.com/")

[*] You should also check out the "File->Generate download script/Add downloaded packages" function of synaptic. Very practical for installations on a PC without internet access. :)

[*] If you want the source code, you can always do the following from a PC with internet access and a Debian based distro on it:
```
apt-get source <game>
```
[/LIST]
