### Post by AndyCooll on 2006-02-04
You don't need to get it from that site, it's in the repositories and easier to install that way. So, either open up Synaptic (System->Administration) and do a search for it there, or type in a CLI:

```
sudo apt-get install vlc
```

You're probably best going through Synaptic because you can then also easily install all the plugins.
