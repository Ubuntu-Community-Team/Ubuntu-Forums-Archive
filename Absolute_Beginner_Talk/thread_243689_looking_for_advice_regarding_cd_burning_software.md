### Post by taurus on 2006-08-25
Yes, you can install and run KDE apps even if you are using Gnome and vice versa.  It will install all the necessary KDE libs for you so when you install k3b with aptitude, you will see it downloads and installs a few other apps as well.

```

sudo aptitude install k3b

```

### Post by steve.horsley on 2006-08-25
Although k3b will probably work when all other burning software fails (I don't know why - it just seems robust to me), the easiest to use if it is working is the burner that's built into Gnome's file exprorer (called nautilus). Just pull down the menu Go->CD/DVD Creator. Drag in the stuff you want to burn and then click burn. It hasn't failed me on Dapper.
