### Post by T700 on 2006-08-03
There is not.  Linux does not use the central registry/hive model, so there is nothing to repair.  Modern Linux file systems are journaled, so the OS keeps track of changes and errors, and corrects most of them automatically.  

Honestly, the lack of constant upkeep was one of the toughest things for me to get used to with Linux.  Sort like when I went from a manual to an automatic in my car--I had nothing to keep my left foot busy!
