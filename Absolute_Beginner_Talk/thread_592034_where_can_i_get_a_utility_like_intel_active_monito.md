### Post by Devlin67 on 2007-10-29
Have you ever tried Gkrellm. It's a utility that you can view all temperatures and memory uses etc. You must first add lm-sensors and then open a konsole and type "sudo sensors-detect" and answer yes to all and Enter where asked.
Once restarted you can start Gkrellm and setup what info you need to see.
Hope that explains it all.
