### Post by Nythain on 2007-05-30
if this fuction is really important, there's always the option of installing wine and running utorrent...

[http://www.winehq.org/site/download-deb](http://www.winehq.org/site/download-deb)    instructions on how to install the latest version of wine

after installing wine, make sure to run winecfg in terminal to set it up

then install utorrent like this
```

wine /path/to/utorrent/setup.exe

```
running it is just as simple
```

wine ~/.wine/drive_c/Program\ Files/uTorrent/utorrent.exe

```
due to random crashes with ktrorrent, and a lack of features and speed with most other clients, i run utorrent instead
