### Post by Zelut on 2006-03-19
Currently write support for NTFS is "experimental".  You'll be able to read but editing/writing is a different story.

I'm sure someone here has a suggestion as far as a write-to-NTFS app.  In the meantime can you use a windows-based app to re-format to FAT32?  If its FAT32 you'll have read-write support in Ubuntu.
