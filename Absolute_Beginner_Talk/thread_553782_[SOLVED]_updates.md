### Post by Seisen on 2007-09-18
> **nikef said:**
> how often should i get updates
also whats the difference between automatic and choosing update manager
sorry if it seems a daft question,but i am a newby to linux.
thanks

You can set update manager to check for updates daily and it will show you what is being updated and but it won't install them until you tell it to. We have only had a few major problems with updates and those were fixed within a day or two if not within 24 hours. Personally I think it is better to let update manager let you know when their are updates available but not to install them, that way if you have a problem with an update you will have an easier time of figuring out what caused the problem.
