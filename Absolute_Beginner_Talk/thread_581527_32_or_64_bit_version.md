### Post by loganm10 on 2007-10-19
you should for sure go for the 64 bit version

the problem with windows is *almost* everything is closed source, and most software vendors only put out 32 bit versions of drivers and programs

on linux *almost* everything is open source, and pretty much every version of every driver and program is available for both 32bit and 64bit computers

I run gutsy 64 bit, and everything works just as it would on 32bit gutsy (I dont actually really notice a performance difference, but whatev, it works fine)
