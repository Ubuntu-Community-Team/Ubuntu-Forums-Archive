### Post by Delkster on 2007-05-03
There's absolutely no need for Automatix or other strange tricks just to get Scribus on Feisty. It's in Add/Remove Applications. Just search for it there. There will actually be two of it -- check the package descriptions, one of them is a development version with more features.

It's a KDE application but should work fine under Gnome as well.

### Post by Delkster on 2008-02-28
> **Station said:**
> is it possible to use adobe page maker files with scribus

Judging by a quick search, it seems that no proprietary formats are supported by Scribus. Check the [Scribus FAQs](http://www.scribus.net/index.php?name=FAQ) for more information -- there's something about file format support in the "Miscellaneous" category of FAQs.

If you can export from PageMaker in EPS or SVG formats, though, Scribus should be able to import those.
