### Post by melancholeric on 2007-02-03

It is possible to get errors (like installing amarok some people get arts errors) but these are usually trivial.  Overall, the only negatives to doing this are (a) they can look out of place at times) and (b) especially if you have an older system, the performance can suffer because more backend libraries needed to be loaded into memory (gnome and kde apps each have their own set of libraries).

I would recommend trying both out, seeing which apps you prefer, then choosing the desktop that has more of the apps you like.  This will give you optimum performance.
