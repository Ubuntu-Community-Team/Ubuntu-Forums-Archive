### Post by podunk on 2006-11-02
I use the Live DVD not the CD and I'm able to "install" from Synaptic, compile programs (pack a lunch!) with Ubuntu using memory as a virtual hard disk. Of course space is sort of limited.

Is "gt7800" a Nvidia graphics card? If so I'd bet you 5 cups of extra caffeine Ubuntu it will work just fine. The monitor configuration may take some fidgeting and manual editing of your xorg.conf file.

You could try downloading and "installing" the Nvidia video drivers using this guide:
[http://ubuntuguide.org/wiki/Ubuntu_Edgy](http://ubuntuguide.org/wiki/Ubuntu_Edgy)

and see what happens. :) I doubt you would be able to edit the xorg.conf file and have the changes show up though
