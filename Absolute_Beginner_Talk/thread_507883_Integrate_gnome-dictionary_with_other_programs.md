### Post by zanglang on 2007-07-24
Not sure about gnome dictionary, but another software, 'stardict', has a scanning feature that can read words pointed to by your mouse and pops up a definition in-place. It supports pronunciations, offline dictionaries, fuzzy searching, and offers Oxford and Longman dictionaries on its website as well. Give it a try! ;)

If you'd rather setup Gnome dictionary to work offline, you'll need to install  'dictd' (essentially running your own dictionary server) and a few other dictionaries like dict-gcide and dict-wn.
