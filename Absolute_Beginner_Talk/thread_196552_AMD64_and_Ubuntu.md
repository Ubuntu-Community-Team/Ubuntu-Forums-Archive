### Post by Kilz on 2006-06-14
[QUOTE=sgbeamer]Yes no problem.  I have an Athlon 64 (about a year old).  I initially put on the 64 bit version but then no java plugin, no real player, etc so I shifted to the 386 version and have been running happily ever since.[/QUOTE]

That may have been true a year ago. Breezy had problems running 32 bit versions of software, to do so you needed a complicated chroot setup. But in Dapper is is completely possible to install 32 bit applications without a chroot. That way you have the 64 bit system, and the ability to run 64 bit versions of applications. But if you find some proprietary piece of software like flash or java that do not have 64 bit versions yet. You cab install the 32 bit version. It isn't automatic in some cases. But there are simple howto's that should only take 10 mins or so in most cases. Edgy will take it a step further I have read and be a true mixed 32/64 bit OS.
