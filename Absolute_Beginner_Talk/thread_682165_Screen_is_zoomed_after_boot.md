### Post by Crumpets and Jam on 2008-01-29
If it is to do with Compiz Fusion you should go to System --> Preferences --> Advanced Compiz Fusion Management (or something like that, I'm not on Ubuntu right now). You should be able to configure the zoom settings from there.

EDIT: If the Advanced Compiz Manager is not on the menu, you may have to install the package. Do this by running this in the terminal.

*sudo apt-get install compiz compizconfig-settings-manager*
