### Post by PeterJS on 2008-02-14
They're called scripts, running a single static command with only one parameter is a bit over kill for writing a script.

To create a launcher on the desktop, right click and select Create Launcher, for the command box just put in ushare -x, and fill in the rest of the fields as you see fit.
