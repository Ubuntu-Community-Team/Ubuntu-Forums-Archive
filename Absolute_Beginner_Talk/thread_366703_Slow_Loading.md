### Post by freelancer1995 on 2007-02-22
i actually know that feeling, i have a celeron 1.7ghz, 640mb, nvidia mx4000 and i wrote a similar post about a month ago, my main problem was web pages in firefox 2 and that turned out to be a known bug in its handling of large javascripts.

it is important to remember that xp is based around the central code, thus operations are preload, ie; internet explorer is used to open helpfiles, outlook express uses internet explorer to view files, etc.  however it is very dangerous having so many products linked it this way, eg; exploit IE and you can exploit outlook express, etc.

linux isn't windows and thus will need to load each program individually to protect the environment, thus slower but safer and more stable.

hope that makes sense and doesn't put you off :)
