### Post by aJayRoo on 2007-09-26
I don't know anything about the upgrade personally as I have never done one, I usually do a fresh install of a new version and keep my home directory (it is on a separate partition).

What I can say is that in my opinion the 64-bit version of Gutsy is considerably improved over Feisty. I have been using the 32-bit version of Feisty on my 64-bit machine because the the 64-bit version was unable to correctly identify my graphics card and I was unable to install Nvidia drivers other than the legacy drivers (which don't work well with a new card!). This has been fixed in Gutsy. Also installing flash player in 64-bit Gutsy is now as easy as it is in the 32-bit version which will probably increase its popularity. Considering that the two main things that held me back from using the 64-bit version have now disappeared I would suggest that you give it a go on your system to see if the same holds true for you. Try the live CD first when it comes out and see if it feels any better for you.
