Hey, welcome to the Linux community. About the "create" a live cd, well you do not have to create a Live CD, it is already created you just have to burn it. ### Post by Orbital75 on 2007-10-26
Well, the Live CD is going to be your best bet for now.  Download the .iso file from
the Ubuntu Site then use your CD burning program ( Nero, Roxio, ect. ) to Burn
the Disc.  You need to look for Burn from Image.  aysiu put up a good link for you
to follow to do this.  The Live CD as said above will allow you to run linux off the
CD and not touch your hard drive. This is a good way to test your computers
compatibility with Ubuntu. If the CD boots right into Ubuntu and everything
seems to open and close and not crash then your good to go.  After you determine
whether or not Ubuntu works on your system, you can install it from the little icon
on the desktop that says install. If you click this, it will start the installer and walk
you through all the steps.  This is very easy and you'll also be able to keep windows.
The Installation process will set up the duel boot for you.  Just a note, Windows
programs don't run natively on Linux. You can use a program called Wine that
may or may not work with some of your programs.... All in all there are tons
of programs that equal or surpass programs you use in windows.

Hope this helps......
