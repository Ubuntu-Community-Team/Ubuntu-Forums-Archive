### Post by ^rooker on 2007-02-22
About gnutella:

1) The version in the repository is probably outdated. I haven't seen gnutella for a while, so maybe you might consider using "aMule" instead? If I'm not mistaken, aMule's connecting to gnutella's network anyway...

2) Firewalling warning: Are you behind a router? That could cause those warnings, because your local ports cannot be directly reached from outside, which is something a P2P application has to consider when negotiating transfers.
Haven't had problems using aMule behind a NAT'ing router.
