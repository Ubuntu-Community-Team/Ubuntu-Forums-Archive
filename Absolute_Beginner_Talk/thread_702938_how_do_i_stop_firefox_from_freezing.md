### Post by y-lee on 2008-02-20
Start firefox in a terminal 

```
firefox
```

when it acts strange like that and you force quite it post any errors reported in the terminal here. If no errors tell us that too.

You can also start firefox in safe mode and see if it still has the same problem.

```
firefox -safe-mode
```

If it doesn't have the same problem in safe mode it is your extensions or something  like that.
