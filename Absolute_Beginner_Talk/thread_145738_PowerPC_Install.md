### Post by linear on 2006-03-16
If you don't want to lose OS X, you'll need to re-partition the hard disk. You can either: back everything up, use Apple's Disk Utility to partition the disk (destructively), and restore; or back everyting up (just in case), and use the instructions [here]("http://www.ubuntuforums.org/showthread.php?t=89960") to resize the existing partition.

Also read these, if you haven't seen them yet:

[https://wiki.ubuntu.com/Installation/PowerPC](https://wiki.ubuntu.com/Installation/PowerPC)
[https://wiki.ubuntu.com/YabootConfigurationForMacintoshPowerPCsDualBoot](https://wiki.ubuntu.com/YabootConfigurationForMacintoshPowerPCsDualBoot)
