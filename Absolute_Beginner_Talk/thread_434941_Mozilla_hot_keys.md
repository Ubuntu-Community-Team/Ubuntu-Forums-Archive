### Post by rygar on 2007-05-06
Ubuntu ate it!

I had the same problem the other day.  Here's how to fix it:

In the address bar in Firefox, type "about:config" (without the quotes)
In the Filter field, type "browser.backspace_action"
Modify it's value so that it is set to 0.

And you're set!
