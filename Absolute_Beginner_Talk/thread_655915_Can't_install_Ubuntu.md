### Post by bump_ on 2008-01-01
The Os will boot from the CD first. WHen it has booted to a desktop, you will see an "Install" icon which you can click to install the OS to your HD. 

In case you did not know, the purpose of booting first from CD is to try the OS out and check hardware compatibility before taking the plunge and installing it.

### Post by RedSquirrel on 2008-01-02
s3a is right, you need 256 MB of RAM to use the LiveCD installer. You will need the alternate CD.

With only 128 MB of RAM, it will probably feel quite sluggish. Xubuntu might be a better choice. 

[http://www.xubuntu.org](http://www.xubuntu.org)

You might want to have a look at the following link:

[http://www.psychocats.net/ubuntu/minimal](http://www.psychocats.net/ubuntu/minimal)

If you really want a fast system with your specs, give the "barebones" option a try (it's on the same psychocats page).

Good luck.
