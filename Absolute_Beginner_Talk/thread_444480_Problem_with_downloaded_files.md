### Post by vtel57 on 2007-05-16
Open your Theme Manager (Settings -> Appearance -> Themes Manager). Where it says "Install New Theme", click on that, then navigate within that window to where your new theme is located. Select it and go.

Open Mouse Configurator (Settings -> Peripherals -> Mouse), choose Cursor Theme tab, click "Install New Theme", then navigate within that window to where your new cursor theme is located. Select it and go.

Have FUN! :)
