### Post by mlomker on 2005-11-03
> 686 is for anything that is a Pentium 2 or newer.

According to the package notes, it supports anything from the old Pentium Pro's onward.

I'd imagine that there is a modest performance boost but the big difference is in memory capacity--the 386 kernel only supports one Gig of RAM, whereas the 686 kernel will go to four.

> Sorry for my ignorance, but I just wanted to make sure: does (K)Ubuntu by default use a 386 or 686 kernel?

By default it installs the 386, in order to be as compatible as possible.  You can see what kernel you are using with the **uname -r** command in a terminal.
