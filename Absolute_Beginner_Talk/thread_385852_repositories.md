### Post by Kobalt on 2007-03-16
Yes, enabling the repos through synaptic or through terminal has the same result (and is the same process actually : uncommenting the lines in sources.list).
For the repos : you should enable the ones that might be useful to you. If you want to learn about Ubuntu repos head over here : [https://help.ubuntu.com/community/Repositories/Ubuntu](https://help.ubuntu.com/community/Repositories/Ubuntu)
