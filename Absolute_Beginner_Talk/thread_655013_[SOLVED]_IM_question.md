### Post by barrack on 2007-12-31
I think by IM you mean Internet or Instant Messenger? If so, Ubuntu comes with one by default called gaim. If you installed Gutsy, then it is called pidgin. Both are multi-protocol programs. I now manage a couple chatting accounts all through one program. The main ones are covered, AIM, MSN and Yahoo are what I use it for. It's pretty easy to set up.

Check it out: [http://www.pidgin.im/](http://www.pidgin.im/)
