### Post by rubbsdecvik on 2007-03-15
Linux is a whole different system in its self.  It needs some hard drive space to call its own.

It can not be directly installed on a windows formated hard drive, however it can "push" over some of the free space and claim it for it's own.  If you went through all the default settings it could delete your windows partition, but it is possible to install linux without getting rid of your windows.

The amount of memory you give to linux doesn't have to be very big.  I'm running a partition that is only 7 gigs, however if you want to do some real computing in Linux you might want to give it more.  The actual system doesn't require much.  I'm not sure off the top of my head, but it will take about a gig or so, plus whatever free space you want to give it.

Most games are not compatible with linux natively.  However many work by using an application called WINE.  WINE is free, but has some bugs.  There are other things like Cedega as well, but you have to pay for them.  You can do a little research on both with google.

Wine will also run many windows applications, but most of the time you can find a program that runs in Linux natively that will do what you need.  For example Microsoft Office -> Open Office.  Both will open .doc files.   

Given a few setting tweaks, you can access your windows partition in Linux and use files and folders.  However getting Windows to see Linux partitions can be a little more tricky.  There is lots of documentation on how to do all of it.

Linux I know can seem intimidating, but it's like getting used to the water in the pool.  Some like to go in slow and take their time getting each part wet.  Others like to dive in and get the whole thing over with.  I did the second one and I don't regret it.  If you are willing to be patient and learn, then I say it can be worth the effort.

Hope that helps
~Rubbs

### Post by annda on 2007-03-15
> If you install Linux, Will it Delete Windows?
no, if you choose "use free space" - DEFRAGMENT your disk under windows first!

> How many memory It requires my Hard Drive to Install Linux?
memory: at least 192 MB, hard disk: at least 2 GB, 5 GB is a much better idea.

> Will most of my games compatiable to Linux?
generally, no. linux programs are linux programs, and windows programs are windows programs. do you try to run Mac application on windows? there are some workarounds (search for wine, cedega and crossover office), but they cannot be expected to be 100% reliable.

> Can Windows program compitable to Linux?
see above.

> Can Linux share Files or Folders I have in Windows? 
of course.
