### Post by djbsteart1 on 2008-03-14
For starters as you are new to the whole idea of Linux, stick with files that end in .deb. These are debian packages, and are very easy to install. You could if you are feeling very brave start to install tarballs, these end with something like tar.gz or tar.bz2, these are harder to install and require the use of the terminal. As you don't have access to internet on this laptop, it is possible to purchase the whole of the ubuntu repo on dvd and use that as a software source. I'm sure it can be got free somewhere I just don't have the link. 
Otherwise, find the program you want, download in windows and transfer over. Then double click for .deb files of see here for[ tarball]("http://www.tuxfiles.org/linuxhelp/softinstall.html")s.
