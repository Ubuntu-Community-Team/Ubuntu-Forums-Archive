### Post by netrattler on 2005-07-17
To do any administration task you use the sudo command and enter your user password. Root under Ubuntu is disabled by default. Under gnome you can use the command gksudo. For example " sudo apt-get update " without the quotes of course. Hope this helps you out, another good source of information if you haven't already found it is the ubuntuguide at [http://ubuntuguide.org/](http://ubuntuguide.org/)
