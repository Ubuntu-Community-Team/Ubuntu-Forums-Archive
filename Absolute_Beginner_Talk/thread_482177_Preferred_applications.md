### Post by jleckie on 2007-06-23
To make an automatic change in a default media application, do the following:
1. Right click on the preferred application's icon.
2. Click on Properties.
3. Write down the information in the Command box.
4. Click on System --> Preferences --> Removable Drives and Media
5. In the dialog box, click on Multimedia
6. Replace the application name with the name you copied in #3 above.
7. Click Close and you're done.

Here's an example: I changed the default DVD player from Totem to VLC, by changing the application command under Video DVD Discs in System/Preferences/Removable Drives and Media/Multimedia from totem %m to wxvlc %m.
