### Post by DirtDawg on 2005-08-09
Run runs programs from wherever you point the command to. 
For example, if you have [Blender](http://www.blender3d.org) installed, you'll have a file in your 'bin' directory that runs it (probably called "blender" and it will appear green in your terminal). In the run command, type "/usr/bin/blender" and the Blender application will start!
