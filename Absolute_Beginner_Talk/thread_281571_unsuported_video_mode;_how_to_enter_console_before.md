### Post by grim918 on 2006-10-22
When your machine boots up go into the grub menu and go into the recovery mode. Then you can log into a command line and make changes to files. If you don't get the grub menu on boot up then wait until your monitor shows a countdown timer. When you see the timer counting down hit the esc key. Then you should see the grub menu.
