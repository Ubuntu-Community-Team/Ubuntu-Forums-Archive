### Post by FakeOutdoorsman on 2008-01-20
What are the hardware specs for the computer, and do you get any error messages?  Here's what you can try:

1. Check cd for defects
The disc itself might be corrupted.  The install disc has an option to check itself in the initial menu.

2. Use memtest to test your ram
The install disc can also check for bad ram.  Let it run for awhile.

3. Use the alternate Ubuntu install disc.
This is good for some hardware issues or for computers with lower memory.
