Compiling from source is not recommended for a new person to linux.  If you really must have the latest version, be prepared to get you hands dirty.  Consult the "Installing from Source" section in the following link.
[http://www.psychocats.net/ubuntu/installingsoftware#source](http://www.psychocats.net/ubuntu/installingsoftware#source)

*Also, when you extract the tarball, it should contain a README/INSTALL doc in the folder with instructions and any dependencies the program may require.  These dependencies must be resolved for the program to install correctly.

*I would uninstall the older versions 1st thru Synaptic.

Good luck.:guitar:
