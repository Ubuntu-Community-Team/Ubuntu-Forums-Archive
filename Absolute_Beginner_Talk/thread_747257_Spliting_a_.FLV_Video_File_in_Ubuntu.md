### Post by FuturePilot on 2008-04-06
Can it be archived? You can archive it and then split the archive.
There's also a program called HJsplit that can split any file.
[http://www.freebyte.com/hjsplit/]("http://www.freebyte.com/hjsplit/")
The Java version is the easiest. You only need to have Java installed.
[Direct link]("http://www.freebyte.com/download/hjsplit/hjsplit_g.jar")
it's a .jar file so you just run
```
java -jar /path/to/file.jar
```
