### Post by tbresson on 2006-07-04
Of course you could try to remove all the hardware you can, e.g. audio cards, tv-cards, ekstra usb cards, and whatever you might have added to your pc.

Try with the minimum configuration. Perhaps run a ram-test, and the cd-test to see if everything is in order.

Then perhaps try a server install see if you can get that far. If that works, try running the live CD.

If it's an old computer some people suggest using xubuntu  ([http://www.xubuntu.org/](http://www.xubuntu.org/)) - perhaps you want too?
