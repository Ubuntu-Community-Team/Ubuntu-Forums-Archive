### Post by randy78 on 2008-01-29
Go to the Synaptic Package Manager (Menu>System>Administration>Synaptic Package Manager) and search for "ubuntustudio", that has a full suite of audio apps and tools.

If you are interested in something like Fruity Loops, check out LMMS and Hydrogen Drum Machine ;)

Fruity Loops 7 actually works fairly well through Wine, so you might want to check that out too...
