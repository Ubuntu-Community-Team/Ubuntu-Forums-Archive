### Post by amohanty on 2005-11-27
You can install Ubuntu as a server without X (the nice GUI thingy) just type server at the install prompt (which you will get if you boot off of the install CD).

A web server to try out would be apache.

If you need a GUI control panel (not quite like IIS - but will ease transition) I would suggest that you try webmin.

To install apache and webmin you will need to use System->Administration->Synaptic Package Manager
