### Post by Ziox on 2006-08-01
open up menu and then go to settings => autostarted applications, check to see if it is located there, if yes, then delete it

if you don't see it then answer this:

do you automatically save your sessions?

---

### Post by WADemosthenes on 2006-08-01
It's not in auto started aplications.  How do I look and see if I'm saving sessions?  

It wont' let me close the dang thing :P

---

### Post by Ziox on 2006-08-01
go to system => settings => sessions and startup, and then uncheck the box that says "automatically save session on logout"

close xfmedia

and then logout then login again, see if it still starts...
