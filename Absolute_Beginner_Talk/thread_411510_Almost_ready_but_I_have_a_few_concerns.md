### Post by PENGUIN-PC on 2007-04-17
In response to your AutoCAD concern, and its availability in Ubuntu:
Unfortunately as of now AutoCad does not yet work in Ubuntu. There are a few options like QCAD, Varkon, and a few others, but none of them have the same functionality as AutoCAD. Specifically there is no 3D modeling in either of those programs. Also WINE as of yet does not work with enabling AutoCAD to run on an Ubuntu system. A program called crossover is supposed to support AutoCAD 2000 on Ubuntu but I have yet to test that out as I run AutoCAD 2006. Other than that I love Ubuntu! I am sure that very soon AutoCAD will be releasing a Linux based version. If I read the news correctly SUsE (another Linux based Distro) and Autodesk are trying to implement something that will be Linux based. I preffer Ubuntu to SUsE quite a bit because of ease of use, and the online community help here on the forums. I hope I was of some help happy computing!
