### Post by tadashi on 2006-03-31
Method one is to install Kubuntu.

Method two would be to load the KDE windows manager (using the package manager).  I think you want the core files and artwork.  The desktop environment will loadup all the KDE apps also.  I preferred to go through and pick what I wanted.  Then when you login you choose the KDE or gnome session type.

### Post by dan828 on 2006-03-31
From the terminal you can do:

sudo apt-get install kubuntu-desktop

which will install KDE and all of the Kubuntu related packages with it.  And you can pick either KDE or Gnome when you log in.  :)
