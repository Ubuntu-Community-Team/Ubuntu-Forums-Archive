### Post by BrendanM on 2007-11-02
1) Try using VLC. It should be in the repositories. VLC was designed as a network streaming client, and usually handles streams very well. If you're still having problems, it may be a problem with the sound system, try searching for 'ALSA' which is the main Linux sound system. There's an older sound system you can switch to (I forget the name -OSS?) that sometimes fixes issues.

2) The custom effects are driven by beryl or compiz, search for those terms. This tutorial might help: [http://kevin.vanzonneveld.net/techblog/article/enable_compizfusion_in_ubuntu_feisty/](http://kevin.vanzonneveld.net/techblog/article/enable_compizfusion_in_ubuntu_feisty/)
