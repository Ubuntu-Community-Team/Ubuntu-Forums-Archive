### Post by dwhitney67 on 2007-11-07
1) Make sure that you specified the correct IP address to your Windows system when you configured the printer on Linux.

2) Make sure that the printer is being shared on Windows.

3) And if it still does not work, ensure that the Windows firewall is not interfering with the communications.
