### Post by djbsteart1 on 2008-03-11
No, KDE4 is only installed if you select it to be installed. Also KDE4 isn't mature enough for everyday desktop use so it is not the default for 8.04.

Note, there is a community supported edition of 8.04 that has KDE4 by default if you want it, but as you say that this is for a friend, it was just a add on piece of knowledge.

---

### Post by forrestcupp on 2008-03-11
You will either have to download the KDE4 version when it comes out and do a fresh install, or upgrade then install the KDE4 desktop afterward.  The upgrade will not be to KDE4.
