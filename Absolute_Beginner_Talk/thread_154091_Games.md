### Post by moephan on 2006-04-02
What have you tried so far for finding games, and what kind of games do you like? You can get a slew of games in Add Applications, if you add all the repositories. Just add al the repositories, and look under games. All these games will work easily. I like super tux and tux racer, but i'm old ;) There are even more games you can get by going through package manager, but some of them take a little more tweaking to make work. You can quake there. You also might like egoboo.

Cheers, Rick

Directions for adding repositories to Add Applications
1. Applications->Add Applications
2.  When the Add Applications window comes you you go Settings->Repositories
3.  Click the add button and when the add repositories window comes up check all the unchecked checkboxes then 
4. Say Yes to all the message boxes 
5. After the new repositories are set, open the tree control under games and then under Games open More Programs ....
