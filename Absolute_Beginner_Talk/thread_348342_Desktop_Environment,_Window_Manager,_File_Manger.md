### Post by bruenig on 2007-01-28
Desktop environment is the basic graphical environment. This is sort of an all encompassing thing. Gnome, kde, xfce, etc.

File manager is a particular application that resides within the desktop environment. Nautilus for gnome usually. Thunar for xfce. Konqueror for kde. Think explorer in windows

Window Manager is basically the engine that runs basic things like windows and the desktop. Gnome uses metacity by default. This is what beryl and compiz are.
