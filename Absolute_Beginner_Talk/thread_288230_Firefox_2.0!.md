### Post by aysiu on 2006-10-30
You're already using Edgy? If that's the case, Ubuntu's Firefox should be 2.0 already, but if you're using the 1.5 installed to /opt, your choices are to either [remove the /opt Firefox and use the Ubuntu build](https://help.ubuntu.com/community/FirefoxNewVersion#head-51054869b8a40ea50859b0163c806e76f882499c) (which is already 2.0), or use [this script](http://www.psychocats.net/ubuntu/firefox) to install 2.0 over your current 1.5 in /opt.
