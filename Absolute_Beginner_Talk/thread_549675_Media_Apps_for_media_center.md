### Post by Tautoa on 2007-09-12
I've got to recommend Amarok...
You can set it up to use a mysql backend, so it can cope with thousands of songs, the interface is easy to pick up, you can easily get information on the artists (via Wikipedia pages embedded into the sidebar), lyrics for the song you're playing, album art straight from amazon (you can even pick the Amazon locale to use), and you can look up the song on MusicBrainz if you don't to tag all those songs yourself.

Unfortunately, it doesn't do video, which is a real shame, but I spose it's better to be really good at one thing than to be average at several :)
