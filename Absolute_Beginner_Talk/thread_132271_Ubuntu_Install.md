How exactly can a person install Ubuntu permanently and still choose which OS Ubuntu or windows at Boot?

---

### Post by Coelocanth on 2006-02-18
You can choose to install GRUB to the MBR when you install Ubuntu. This will give you the option of which OS to boot whenever you start your computer. This is what I did when I installed Ubuntu as a dual-boot with my WinXP install and it works great (although I find I hardly boot to Windows at all anymore... :mrgreen:  )

### Post by Coelocanth on 2006-02-18
[QUOTE=jasrog]How do you install GRUB to MBR? Are you given that option from the permanent Ubuntu install cd?[/QUOTE]

Yes. At some point in the installation  there will be a prompt something like this:

Install GRUB to Master Boot Record?

Just choose 'Yes' and carry on with the instal. Easy as pie. You should check out Herman's most excellent tutorial (complete with tons of screenshots) for creating a dual-boot system.

[Herman's Guide](http://users.bigpond.net.au/hermanzone/). I used this as my guide when I installed Ubuntu. It's really easy to follow and works great.
