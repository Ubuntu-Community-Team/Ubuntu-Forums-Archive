### Post by str8line on 2007-09-02
For the most part with the installation of Ubuntu most devices are covered.  Some things do need tweaking (wireless, graphics and 3d sound) but on multiple different setups in my own home (currently have installed on 5 pcs here) and the only thing that did not work out of the box was the Realtek 3d sound on my HP Media Center PC.  

If you are running the live CD now, try out the different aspects of the laptop.  Can you connect to your network wired, can you connect wirelessly etc) If things just work, like most things will then go for the install by double clicking on the Install Icon on your desktop.
