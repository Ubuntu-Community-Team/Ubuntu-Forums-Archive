I uninstalled a program and want to remove it's icon from my Applications bar, how do I do this?

---

### Post by qpwoeiruty on 2007-03-16
If it's still in the menu, you can delete/disable it by going to System/Preferences/Menu Layout, clicking on the category that the icon is in, and unchecking the box next to the app or by right-clicking on it and choose delete.

---

### Post by raja on 2007-03-16
You can use the Alacarte menu editor to do this easily. Type alacarte from the terminal to start this. (Or you can launch it with Alt-F2).
