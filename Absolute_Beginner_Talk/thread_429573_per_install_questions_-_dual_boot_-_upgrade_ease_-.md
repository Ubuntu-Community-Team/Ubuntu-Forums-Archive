### Post by Bachstelze on 2007-05-01
> **silent1643 said:**
> i assume you can do dual boot with an xp machine just by making another partition for ubuntu correct?

**correct**

Also i noticed a new version is released every 6 months, how easy is it to update ubuntu?

**It's a bit of a Russian roulette : for some people, it's a breeze and for others, it's a nightmare, IMO, the best way to go is to do a fresh install when you want a newer version. Keep in mind also that always running the latest version of everything is not a wise thing to do if you aim for stability.**

And my last question, how easy is it to setup all my hardware if i dont have every driver cd there is.

**It mostly depends on what hardware you have. For the most part, you will have nothing to do. Problematic drivers are often graphics or Wi-Fi**

Welcome to Ubuntu :)
