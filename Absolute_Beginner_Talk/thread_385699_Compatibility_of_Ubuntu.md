### Post by kolinab on 2007-03-16
I was totally new a few months ago. Didn't know Mac really much, used Windows all the time.

I think if you are used to Mac or XP that you will find Ubuntu a pleasure to work with. I don't do much tweaking - I find the default configuration to be pretty nice. I played with the theme a little bit and brightened up the background image. I think you will be happy with the way applications work with each other. It's snappy fast - much faster than XP was on this same machine. It kind of feels like instead of being in a car and someone else is driving, you are driving the car where you want - and it's a nice car! lol

I don't know much about gaming with Ubuntu. There is a section on the forums here devoted to it, and I know a lot of people are having pretty good success running games - with some tweaks. You could always dual boot (just choose your OS at startup) if you want to keep XP for gaming.

There are substitutes for iTunes in Ubuntu. Search the forums and you will find them. I don't think you'll have to do much tweaking - there are great HOWTOs here if you need them and everyone is quite helpful in the forums.

MSN - no problem. There is a program included in Ubuntu called Gaim that will work with all the various instant messaging services.

That is a lot of general, not too much in the particular. Try using the search feature of the forums to find out on your iPod etc., and post questions when it doesn't give you what you're looking for!

I would recommend downloading an .iso and making a Live CD if you haven't already. That way you can test drive it and see how you like the look and feel. You can install if you like and use it as is, or tweak to your heart's delight. Either way, I think you'll have a good experience.

Post back if you need help downloading and burning the Live CD and getting it working.

Sorry for the long post!
