I will use Bootpart and  edit my Windows boot.ini file so that I can load Ubuntu from the Windows boot menu.

According to some guides I've read, installing from alternate CD will set the Linux root partition as active, which then prevents Windows from loading.

### Post by 5-HT on 2007-04-20
You have a choice of which partition to make active in the partitioning phase of the install using the alternate CD: just double check that the Windows system partition is active and you have nothing to worry about. :-)
