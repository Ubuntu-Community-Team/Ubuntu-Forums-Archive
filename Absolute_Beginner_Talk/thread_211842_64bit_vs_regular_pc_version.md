### Post by Qrk on 2006-07-09
Ubuntu's 64 bit doesn't have the XP 64 problems. (I remember when I bought a turion laptop, I was surprised to find 32 bit XP on it.)

The main problem with Ubuntu's AMD64 architecture is that Windows apps and codecs won't work on it. Things like win32codecs (mplayer) aren't made for x86_64 systems. 

If you want these applications and codecs, you'll probably better off using the 386 ubuntu.
