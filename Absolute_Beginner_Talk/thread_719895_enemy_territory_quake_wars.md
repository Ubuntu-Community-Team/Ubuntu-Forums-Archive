### Post by taurus on 2008-03-09
Were you in the directory where you saved that download file?  Did you download it to ~/Desktop?

```
cd ~/Desktop
chmod 755 _filename.run_
sudo ./_filename.run_
```

### Post by smartboyathome on 2008-03-09
You need to right click on it, select properties, select the permissions tab, and select allow to be run as a program. Then, you can run it in a terminal.
