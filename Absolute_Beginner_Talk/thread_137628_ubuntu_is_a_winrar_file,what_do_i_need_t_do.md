### Post by kaamos on 2006-02-28
The file is a .iso, but winrar by default associates with iso files so that is why you see the winrar icon. Just burn it as an image to a cd. [https://wiki.ubuntu.com/BurningIsoHowto](https://wiki.ubuntu.com/BurningIsoHowto)

If you have problems, you could have a faulty cd or a corrupted iso. Try checking the md5sum of the iso file and when burning the cd use a low (like 2x) burning speed.

Hope this helps.
