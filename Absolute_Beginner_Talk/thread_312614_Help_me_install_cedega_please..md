From their site download the .deb package to your Desktop. Doubleclick and it will install or else:
```
cd Desktop
sudo dpkg -i XXXXXXXXXXXX.deb
```

apt-get/aptitude is for package in the repo and not manual downloaded packages. From the screenshot I can see you downloaded the source which means it need to be compiled (in most cases).
