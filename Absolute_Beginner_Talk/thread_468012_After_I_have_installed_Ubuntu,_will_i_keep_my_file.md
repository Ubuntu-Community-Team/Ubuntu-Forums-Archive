### Post by Rocket2DMn on 2007-06-08
If you are setting up a dual-boot system and you don't overwrite or format anything, all your files should be intact.  However, before you do anything, I do suggest you make a full detailed backup of all your files - music, photos, documents, etc.  Using an external HD works best, esp. if you have lots of stuff.
Since you mention wine it sounds like you just want Ubuntu to be your only system, and you want to simulate Windows in wine.  Be aware that not all programs run or run smoothly in wine.  If you want to setup a dual boot, there are plenty of articles around about that since it is very common.
As far as not wanting to get your programs again, if you are completely switching to Ubuntu, none of your Windows programs will run, except maybe under wine, but there are open source alternatives to pretty much everything in Windows that run in Linux.  Ex: OpenOffice replaces MS Office.  Some stuff like firefox and azureus is available on windows as well as linux.
Good luck!
