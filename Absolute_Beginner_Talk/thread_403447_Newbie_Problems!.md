### Post by e_james on 2007-04-07
If I remember correctly, the 6.10 live CD has an option to install in text mode at the first menu. It won't look very pretty but, if you have done a linux install before, it shouldn't ask you any questions you haven't already seen. The main advantage here is that you will see the various stages running and may get a clue as to what is not working.

I just thought of something. If your internet connection is available during the install, there may be lengthy periods where nothing much seems to be happening. I believe this is because the installer is downloading more up-to-date files.
