### Post by Peepsalot on 2006-10-23
> **knightinarms said:**
> Just a quick question here, is there a program in/for Ubuntu that lets you burn iso files to DVD.
If I right-click on an ISO file, there is an option to:
Open with "CD/DVD Creator"
All that is necessary after that is to click to start the burn process.  About two clicks total.  It has worked great for me using CD or DVD isos.

Although, this is on edgy, I don't know if this is setup the same for dapper.

The actual program that this invokes is called "nautilus-cd-burner"
