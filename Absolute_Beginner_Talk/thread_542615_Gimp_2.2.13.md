### Post by James Fitzpatrick on 2007-09-04
The Help files, for some reason, don't come with the Feisty package.  You'll have to download them, using the Synaptic Package Mgr.  Click on System>>Synaptic Package Mgr.  Then do a search for Gimp.  Browse through the available packages until you find the Help files, and download them.
