### Post by Hospadar on 2007-08-07
Most programs (especially programs that have a gui) will have all of their options accessible through some menu in their gui somewhere just like any windows or mac program.  Also all of the installing and removing of standard software can be done through the gui via synaptic (the package manager).

That said, you may often see in forums and help guides text commands simply because they are much easier to type and explain then a guide with screenshots for which windows to go to and which buttons to click on.

Because of this, you will probably want to learn about some basic commands in the terminal such as cd, ls, cp, rm.  these are all very straightforeward and they all have very helpful manuals that can be accessed with the "man" command by doing "man <the name of the command>" i.e. "man ls"

If you google "basic linux commands" you will turn up a ton of helpful sites.

If that sounds like a lot, don't worry, you will rarely need to do it unless you really want to hack up your system a lot, and even then generally guides are very straigtforeward since you can just copy and paste their text commands.

oh also for wine, check out the wine appdb for detailed instructions for individual games on common bugs for those games and how to fix them.  most major games have some good info on how to get them running with wine.
