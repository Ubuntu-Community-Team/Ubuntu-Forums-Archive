### Post by jpeddicord on 2007-02-12
VMware does not have decent video card support, and so it probably will not run some Flash or Java apps too well or at all.

If you want to install Ubuntu to your PC (outside of VMware) then just make sure you have backup your data and follow the installer instructions carefully. Check here if you need help:
[https://help.ubuntu.com/community/Installation/I386](https://help.ubuntu.com/community/Installation/I386)
