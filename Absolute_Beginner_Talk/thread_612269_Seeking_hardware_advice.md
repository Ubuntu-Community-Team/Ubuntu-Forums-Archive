### Post by SteveHoffmanUK on 2007-11-13
Inxible wrote:
> most people simply buy laptops/desktops with Windows on it and then dual boot or simply remove Windows altogether and install Linux.


If you go that route and you really don't need Windows, I would always try to get a discount. The supplier won't have to pay Microsoft for installing Windows if you're not having it, so he should be willing to knock something off the price. Of course the discount won't be as large as the retail price of Windows because he will get it at a much reduced OEM rate.

Any computer will run Linux. The real problem is the peripherals (printer, scanner, modem,  and other USB devices) that you intend to use, so be sure if you're buying a package deal that the peripherals are Linux-friendly. There are a number of websites that list these, with assessments of how Linux-friendly particular models and manufacturers are.
