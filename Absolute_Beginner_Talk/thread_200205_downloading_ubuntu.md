### Post by IYY on 2006-06-19
If you're using "Visual C++" or "C++.NET" or "C#" then yes, it's very different. These versions were created by Microsoft specifically for Windows. But the language itself, C/C++, is the main programming language used in this operating system and the programs written for it. So, there are many IDEs you can use to  program.

Of course, I suggest to start from writing code in plain text files, and compling it with the gcc (g++ for C++) complier. It's the best way to learn the language itself.
