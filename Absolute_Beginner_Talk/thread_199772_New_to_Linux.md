### Post by Winblowz on 2006-06-19
1. Yes!:D This is done very easily, although I have never done it myself. Many Ubuntu users dual-boot with XP. You may want, to be on the safe side, back-up your XP data just in case though.

2. What is the model of your Linksys G card? In linux, compatibility depends on the chipset the card is using. You can find out what chipset it is using by doing a quick google search for the model of your card or by searching in the Ubuntu forums. It is very likely though that you will find if it is supported or not without even having to worry about the chipset it uses. 

NOTE: Not all versions of the same model of a wireless card MIGHT not use the same chipset. If they don't all use the same chipset, then you have to find out what version of the card you're using, the chipset it uses, and if that chipset is supported. 

If you already own the wireless card though, then you could just try it out first and see for yourself if it works.

3. Mplayer is probably the best, and is my favorite. It is very good and supports virtually every type of media format. There are others though, which are also very good, and are preferred by some.

4. I don't know much about this, although I think both Gnome (Ubuntu) and KDE (Kubuntu) come pre-bundled with system monitor's for their respective desktops. There may also be others, but I have little knowledge about them.

5. I'm not too much of a pc gamer, and although games can sometimes be hard to get working in linux, I think World of Warcraft is very much supported. Most pc games only have Windows versions, except for Quake 4 and Doom 3. In order to get a Windows game working in Ubuntu, or linux at all, you must use either WINE, or Cedega. WINE is 100% OpenSource, but Cedega is commercial software and warrants a fee to use. Cedega is built upon WINE, but is specifically for gaming. Some people think the price for Cedega is worth it, some don't. I'm not sure though, so I'd check the Ubuntu Gaming forum to be sure.
