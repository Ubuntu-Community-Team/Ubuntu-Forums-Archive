### Post by christhemonkey on 2006-10-11
```
sudo apt-get install *bla*
```
Will install a program for you (and if your online, will install all the things it depends on to install)

You say your doing everything from the command line?
Does that mean you are using the server edition?
If you are not using the server edition then, gedit can do syntax highlighting if you just switch it on in the options.
