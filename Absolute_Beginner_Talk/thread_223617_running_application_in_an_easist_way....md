### Post by Paerez on 2006-07-26
It is called an "Application Launcher" you can add them to the panel and the desktop and the menu through the "Alacarte Menu Editor".

For example, right click the desktop and click "Create Launcher" then give it a name and set the command to the path followed by the command, such as:
/usr/bin/command
or
/home/YOU/script.sh

then you can just double click the launcher.
