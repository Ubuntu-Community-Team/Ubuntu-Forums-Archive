### Post by Metacarpal on 2006-08-04
Hi Radioguy,

There are several available media players for Ubuntu and for Linux in general.  Everyone has opinions on which one to use, most of which are based on personal preference.  The default player is called Totem, and some swear by it while others swear at it.

Other popular players are Rythmbox, Xine, VLC, Beep, Helixplayer... the list goes on.  You can try different ones and see which ones you like.  Just go to Applications > Add/Remove, and look in the "Sound and Video" category.  Just check which ones you want to try!

If you're planning on listening to MP3s or WMA, or other proprietary or non-free formats, see [this documentation]("https://help.ubuntu.com/community/RestrictedFormats") or, if that all seems intimidating, try using [EasyUbuntu]("http://easyubuntu.freecontrib.org/") or [Automatix]("http://www.getautomatix.com/").  I prefer EasyUbuntu, but some people swear by Automatix, so I leave the choice to you.

*(Asking which media player or installer-helper [like EU or Automatix]is the best choice is the easiest way to get a 20+ response thread going. ;) )
