### Post by mitch.c on 2006-09-29
> **EddieA said:**
> Anybody got any ideas -  I would liketo be able to use Firefox again.
Could this be caused by an extension (or combination of)? When I've had problems like this, it's almost always come down to extensions or something in my profile. I have always solved it by creating a new profile, copying bookmarks.html from my old profile, and installing extensions one at a time.

You can create a new profile by typing...
```
/path/to/firefox --profilemanager
```
