### Post by giox on 2007-09-26
I download the drivers for your card. Then i unzip it. To see what files you have to use. Assuming you have the same files do the following.
Now you have to make a folder name it what ever you want. Then grab this to files from a zip file in the folder Windows XP2KME98SE driver folder. The files you need are net8185.inf and rtl8185.sys. Take those and put them in that folder that I told you to created. Then that folder needs to be in the same folder that you put the ndiswrapper folder.

Now that you have the right files do the following:

Change directory to the folder that you put those files.

Then type 
sudo ndiswrapper -i net8185.inf

sudo ndiswrapper -l

You should see a message that says driver present, hardware detected

Now finish installing the driver
In a terminal type:
sudo ndiswrapper -m
Then:
sudo modprobe ndiswrapper

NOW REBOOT

Then test you wireless with 
sudo iwlist scanning.

Let me know if it works. Good Luck!
