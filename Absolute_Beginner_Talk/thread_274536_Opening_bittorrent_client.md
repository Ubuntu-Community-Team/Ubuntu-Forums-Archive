### Post by uchuunoyanushi on 2006-10-10
> **zerg zega said:**
> There isn't any icon that I can click to open the bittorrent client. It's like the client is hidden somewhere on my computer.

If you go into the Alacarte Menu Editor (under Applications -> Accessories), you can unhide the bittorrent client; it's in the internet menu, it's just not shown by default. When you open the client, it will prompt you to select a torrent file to open. If you've saved the torrent file on your computer and re-open it, bittorrent will automatically detect the incomplete file and ask you if you want to continue that session.
