### Post by notoriousdbp on 2007-07-15
Ubuntu is my only OS - quick tip make sure you have a separate /home partition from your / partition in case things go wrong in the future.  That way you will only re-install the OS without losing your e-mails, docs etc.  I recently made the gross error of trying to install PCLinuxOS as a second distro and it wrecked my partition table.  Fortunately I was able to do a clean install of Ubuntu and re-link it to my home partition - even your desktop settings are stored there.

Networking - not a problem using ndiswrapper for my Broadcom wireless card.  I copied the Windows drivers onto a USB stick ages ago.  My wife hapily surfs the net and since I bought an Epson printer photo printing has been really easy. Oh and I replaced network manager with wicd - far more reliable.
