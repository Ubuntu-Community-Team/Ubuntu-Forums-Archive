### Post by arjun.shankar on 2006-08-10
Most programs you compile from source will not install the way you indicated. The process is usually as given here:

[https://help.ubuntu.com/community/forum/software/OpenSource/SourceCode](https://help.ubuntu.com/community/forum/software/OpenSource/SourceCode)

Also, once you unpack the source, you should read the INSTALL, and README (these are their usual names) files to get an idea of how to go about installing.
