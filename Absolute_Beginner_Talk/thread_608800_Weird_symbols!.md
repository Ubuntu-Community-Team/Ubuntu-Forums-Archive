### Post by steve.horsley on 2007-11-11
Uninstalling and reinstalling is unlikely to make any difference. Whatever the problem is will most likely be stored in the user settings in ~/.mozilla, and a reinstall of the application won't affect those. He could try renaming ~/.mozilla and then starting firefox. This will start it without any saved personal info or settings, and show for sure if the problem is a personal settings issue.
