Since you are new I recommend either, SLED, Ubuntu, or if you really want to get down and dirty with linux you should try Slackware (there are no package mangers, so dependency issues arent resolved which also means that youll really learn to administer a linux system, compiling things from source, editing scripts galore). I hear the SLED is really nice, but I think you have to shell out a couple of dollars. If you choose ubuntu I can tell you that the community around here is really nice. Apt, the package manager in debian based distros, truly is robust and makes software installation (as well as upgrading) a snap. Furthermore, Ubuntu makes a great launchpad for trying other distros (It was my first :) ). In any case it would be great to have another member to add to the ubuntu community, and linux community in general. You also might wanna check [www.distrowatch.com](www.distrowatch.com). That has a plethora of distros to choose from, all of which link to their main site. Happy hunting :)

### Post by snakyjake on 2006-09-29
I'll try not to be biased...

Once a distros is isntalled, they are pretty much the same.  The only difference might be application management and system tools.  They all do the same thing, some just have a different perspective on accomplishing the same thing. 

Some distros have certain goals.  One being most like Windows, or really lightweight, totally free, or try to make it extremely easy right out of the box, and there's many other philosphies that you can read about on [DistroWatch]("http://distrowatch.com/").

Ubuntu is my favorite mainly because of these reasons:
1. The _best_ community for help and support. Because of the community support, Ubuntu is the easiest, less strees, happiest distro I've used.
2. Lots of flexibility.  I can add/remove features that I want.  Not stuck with the distros way of doing things.  I love experimenting with the different features that Linux offers.
3. I truly cannot over estimate the tremendous help from this forum.  There are all types of Linux users here.  Everyone is very helpful.  And you can get help on ANY aspect of Linux. This is the best place to learn about Linux.

Not to be biased, but I found Freespire to be less intimidating  for someone new to something different and for the "out of the box" experience.  Freespire has assembled a nice "quick start" package that gets you going quickly.

Disadvantages of something proprietary like Freespire is that it is very easy to get stuck.  For example, you may want to try different "desktops GUI interfaces"  like Gnome, Kde, Xfce, and others.  Or maybe you want to install something that isn't in Freespire's "application store" (aka Click N' Run).  You may want an application that Freespire does not yet have in their repository (kinda like a store).  And at one time Freespire charged money to access their store, but they quickly changed it to be free.  Who knows, they may start charging money again...and you just don't know if they will or won't.

Freespire has a Live CD you can demo.  Burn the CD, boot up to the CD, and try Freespire without having to install.  

Just remember, you can make Ubuntu (and any distro) look and function the same.  So don't think the eye candy is just for one distro and not the other.

My adive, when you switch over from Windows it is going to be slightly different.  It is like driving a new car or living in a new home.  Take some time to get used to Linux.  And then if you still want more...then there's a lot more than Windows will ever offer you.
