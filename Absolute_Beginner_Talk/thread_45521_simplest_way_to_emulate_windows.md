### Post by chestnut on 2005-07-17
For my personal requirements, after trying all solutions, I have settled with win4lin home (running windows 98)

* Wine - free, to buggy
* Crossover office $ - still to many bugs for the Excel spreadsheets I have to use from work (macros being the main issue). Far to many crashes on critical work (and before anyone screams - bloated openoffice 2 beta calc slows to a standstill with the exceedingly large spreadsheets that I use; the relatively unbloated gnumeric works far better in this respect, no VB macro support though..)

* VMWARE - excellent software, $$$, way too slow on my old laptop, does run well on high spec machine - may be overkill if you do not need the all the functionality

* QEMU - free, speed issues on my low spec laptop, no win98 with accelleration module

* Win4lin home - best $ value for my requirements (about 30 bucks US), runs better than my native windows 98 install, & can run Excel with no probs whatsoever, and more importantly, the Australian Tax offices tax return software that you can only get on windows platform :( Also seems to run anything else designed for win98 bar activex games.
