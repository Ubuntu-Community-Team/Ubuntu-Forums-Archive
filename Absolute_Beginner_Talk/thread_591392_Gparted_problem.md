### Post by qpieus on 2007-10-25
Are you using the gparted that is installed with ubuntu, or are you using the gparted livecd? I've had numerous problems with the installed gparted not working correctly, so I always recommend using the gparted livecd. The livecd doesn't mount any drives and seems to work much better. Google it to find the link, burn the cd, boot from it, then do whatever you want with your drives.
