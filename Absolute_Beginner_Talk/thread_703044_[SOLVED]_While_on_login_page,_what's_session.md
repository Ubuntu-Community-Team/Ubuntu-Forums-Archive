### Post by luisito on 2008-02-20
I can tell you that you have Xubuntu.

The "session" option typically lets you choose between the available window managers and desktops. Since you probably installed only Xubuntu you have only xfce and a failsafe (boring) one. If you (hypothetically) installed Kubuntu-desktop and/or Ubuntu-desktop on top of your current installation (watch out, this would be a big install) then you would be able to choose also KDE and/or Gnome as sessions.
