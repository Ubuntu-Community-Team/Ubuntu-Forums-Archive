Ubuntu came with a panel containing the "Show Desktop" panel and the workplaces. I want to remove the Show desktop panel,place it in the other panel and delete the panel wiht the workplace icon.

---

### Post by Mornedhel on 2007-07-28
All widgets can be added/removed from either panel (right-click on widget, then Remove from panel ; Right-click on panel, then Add to panel). Panels can be deleted as well (Right-click on panel, Delete this panel)
