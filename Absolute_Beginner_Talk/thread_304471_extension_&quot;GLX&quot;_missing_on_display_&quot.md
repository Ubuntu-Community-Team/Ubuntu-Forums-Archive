### Post by LLRNR on 2006-11-21
I'm surely not an expert in xorg.conf messing up, but 2 things occur to me:

1. if you did install the nvidia drivers, then your Section "Device" should be : ```

Section "Device"
	Identifier	"NVIDIA Corporation NV34 [GeForce FX 5200 Ultra]"
	Driver		"**nvidia**"
	BusID		"PCI:1:0:0"
EndSection
```

2. surely you don't want 16 bit as default color depth !? I (with an older nVidia than yours) have 24, so this shouldn't be an issue:

```
Section "Screen"
	Identifier	"Default Screen"
	Device		"NVIDIA Corporation NV34 [GeForce FX 5200 Ultra]"
	Monitor		"Generic Monitor"
	DefaultDepth	**24**
        .................................
```

Be sure not to change anything else! But these things won't hurt, I modified them myself manually and it turned out just fine.

LLRNR
