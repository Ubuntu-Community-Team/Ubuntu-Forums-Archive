### Post by Paul820 on 2007-07-30
If you are very happy with XP then stay with XP. If you really do want to try it out then defrag windows, BACK UP you important data and give ubuntu a go. Make sure your first boot option in the bios is set to your cd drive stick the cd in and boot your computer. You can allocate about 10Gb for ubuntu to try it out. If you don't like it take it back off. To take it off, insert your windows cd, choose repair windows, then type 'fixmbr' without the quotes. Then in windows just delete the partition that ubuntu was on and allocate it back to windows.

If you find you like ubuntu then just give it more space.
