### Post by one2one on 2006-01-04
The disk install was pretty easy. Only place I found confusing was after using the "Create partition in existing disk (1st option)" was the "New Partition Size." This does NOT mean the new Ubuntu partition. It means the NEW size of your WindowsXP Partition. First look at the size of you Windows Partition and make it the size that will give what you want to Ubuntu.
