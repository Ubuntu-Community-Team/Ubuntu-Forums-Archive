### Post by forestpixie on 2007-08-07
I'm not sure if the data can be merged - it might depend on how you set it up in the first place, ie which collection database type you use, you can change it in 

**settings >configure > configure collection** 

You can find the database it produces in a hidden folder

If you open nautilus and show hidden files you'll find it in

**/home/yourhomename/.kde/share/apps/amarok**
