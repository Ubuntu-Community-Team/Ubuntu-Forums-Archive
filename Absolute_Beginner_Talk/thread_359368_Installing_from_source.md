### Post by tbroderick on 2007-02-12
> **mdecler2 said:**
> 
Any help is appreciated!

Like I said do you have **libc6-dev** installed? libm.so.6 is found in libc6, but  Debian/Ubuntu splits libraries and headers/development files. So if you want to compile a program that links to libm.so.6, you will need to install libc6-dev which contains libm.a and libm.so.
