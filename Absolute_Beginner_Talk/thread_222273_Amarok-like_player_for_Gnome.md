### Post by Jagot on 2006-07-24
It has it's own repo for Ubuntu Dapper:

```
deb http://theli.free.fr/packages/ dapper listen
```

Add that to your sources.list, then:

```
sudo aptitude update && sudo aptitude install listen
```
