### Post by LaRoza on 2008-02-11
Kubuntu uses KDE and KDE applications and Ubuntu uses GNOME and GNOME applications.

They both are official versions, and KDE applications and GNOME applications will work fine on the other versions.

You can install all the Kubuntu packages on Ubuntu (and vice versa).

[http://en.wikipedia.org/wiki/GNOME](http://en.wikipedia.org/wiki/GNOME)

[http://en.wikipedia.org/wiki/KDE](http://en.wikipedia.org/wiki/KDE)
