### Post by Mr. Electric Wizard on 2005-06-30
Printers --> New Printer
You have to put in the PrinterName, IP, login credentials, etc.
It is really similar to browsing the network with Nautilus.
After you get all of that sorted out, it will ask what type of printer it is, etc.
