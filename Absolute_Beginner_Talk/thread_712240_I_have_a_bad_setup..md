### Post by herbster on 2008-03-01
You can't download linux updates on your windows machine directly, AFAIK. What you could do is use a virtualization app like VirtualBox and run Ubuntu from within Windows, but of course that's defeating the purpose of getting away from Windows, which is what you want to do (?).

This is the FS driver you can use on Windows to see linux filesystems: [http://www.fs-driver.org/download.html](http://www.fs-driver.org/download.html)
