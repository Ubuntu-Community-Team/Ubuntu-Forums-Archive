### Post by str8line on 2007-09-02
I had a similar issue when I first installed Ubuntu with my system currently connected to my living room TV.  I had to configure the X-Server to register lower resolution options to work with the S-video.  Try removing the 1280x800 resolution options in your xorg.conf.  Not an Ideal solution but sounds like it is a similar limitation of the lack of open source drivers from ATI.
