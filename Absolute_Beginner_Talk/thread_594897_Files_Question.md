### Post by por100pre1 on 2007-10-28
NO, if you install Ubuntu using all the hard disk the installer will reformat the disk deleting everything! **It doesn't matter which install method you use, you should backup your files to removable media or external hard disk drive!** Backup everything EVEN if go dual boot.

---

### Post by Leonivek on 2007-10-28
You can keep your data files on one partition, whether it be an NTFS or a Linux Filesystem, and access it from Ubuntu regardless of the partition.  If you have 2 drives, keep all your data on one and install Ubuntu on the other.  If you don't have 2 drives, partition the one you have, keeping your data on one partition and installing Ubuntu on another.
