### Post by CupofDice on 2008-02-03
The only way to use it in ubuntu is by running xp (or maybe vista) in vmware. The transfer speeds will be USB 1.1. Some people (well, 1 at virtualbox forums) have gotten it to run with an older version of virtualbox.
