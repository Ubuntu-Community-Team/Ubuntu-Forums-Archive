### Post by neurostu on 2008-02-23
You probably have both of them  installed. When you get to the boot screen there is a button in the bottom left that says "Session" if you click on that you can select XFCE or KDE or Gnome (depending on which ones are installed). 

What probably happend is when you install XFCE is was set as the default session manager.  Using that "session" button you can set the default session manager to what ever you want it to be.

 As far as changing the login screen from Kubuntu to Xubuntu I'm not sure how to do that.
