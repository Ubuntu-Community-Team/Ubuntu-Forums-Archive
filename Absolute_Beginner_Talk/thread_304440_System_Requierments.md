### Post by ReaderRat on 2006-11-21
Don't know about Macs, but it should install on a P III with 256M RAM (the minimum for Ubuntu). If the X86 is having trouble (like video), try the X86 Alternate CD. It installs in text mode and only needs 192M RAM for install.If you are using the X86 LiveCD, when you boot into the boot menu, pick the option to check MD5sums, to be sure you have a good burn on the disc.
Good Luck...

### Post by ewl1217 on 2006-11-21
If Ubuntu runs slow, you could always use Xubuntu. Xubuntu is a sister-project of Ubuntu, but uses the faster Xfce desktop instead of GNOME. I've had it run very well on computers with as low as 128 MB of RAM (I've never tried using anything with less.).
