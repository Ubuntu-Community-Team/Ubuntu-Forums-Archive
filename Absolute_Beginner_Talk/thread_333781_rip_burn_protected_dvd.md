### Post by Hallvor on 2007-01-08
I have made a lot of backups in Windows, but I guess you can use wine. You only need dvdshrink and a burning programme. Dvdshrink will rip it and shrink it to one 4,5 gb dvd. Then, burn the ripped directories to a dvd. I have only used Nero for this, but any dvd burning programme will probably do. You can configure dvdshrink to start Nero and burn automatically. Don`t think dvdshrink can use other burning software like that. Just make sure you don`t burn the dvd as multisession or forget to finalize the dvd. If else, it won`t play.

Hope this helps.
