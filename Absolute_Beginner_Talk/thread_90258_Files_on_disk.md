### Post by Kvark on 2005-11-14
What file extension does the winrar archive have? .rar? .zip? .iso?

If the extension is .iso then choose "burn from image" or something similar with the word "image" in it in your burn program. Then choose the .iso file when it asks which image you want to burn.

If the file extension is something else then .iso then you got the wrong file.
