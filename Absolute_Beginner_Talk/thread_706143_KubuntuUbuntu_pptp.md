Vpnc:
try sudo aptitude install vpnc
openvpn:
Again, try typing 'sudo aptitude install openvpn' at the terminal command prompt
PPTP:
Is just a protocol that is used and configured within a VPN connection. It stands for 
point to point tunneling protocol.
