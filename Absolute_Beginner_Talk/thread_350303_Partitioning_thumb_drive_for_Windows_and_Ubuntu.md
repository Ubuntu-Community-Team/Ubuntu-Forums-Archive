### Post by Wartooth on 2007-01-31
I have a couple of Kingston DataTraveler 1GB USB sticks.  I am wondering what it would take in order to have two separate sections on the same stick in order to have Windows-useable apps as well as Linux-useable apps (portables of Firefox, OO.o, etc.).  Also curious if there's a way to store files on it that can be seen by both Windows and Linux, such as .odt or .doc or image files.  

I've tried searching but haven't found anything yet. :/  

Thanks.

---

### Post by Paerez on 2007-01-31
If you format the whole thing in the FAT32 Filesystem, both linux and windows can read and write to it easily. Then you can store your programs for both linux and windows on the same partition.

### Post by Wartooth on 2007-01-31
Thank you so much!  For some reason I was convinced it couldn't possibly be that easy!  I put an .odt on there to test it after partitioning, and sure enough, both XP and Ubuntu saw my test file! :D
