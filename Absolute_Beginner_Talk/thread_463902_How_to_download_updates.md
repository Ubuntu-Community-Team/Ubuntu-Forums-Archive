### Post by Obor on 2007-06-04
If it was me I wouldn't be worried about patching ubuntu install if its not connected to the internet. 

The way ubuntu works you cannot download one file that will patch your whole system. The update manager will find which packages have newer versions but for that you would have to be connected to the net. I'm not sure how you want to find which packages you need.

That said if you do know which packages you need you can download them from [http://packages.ubuntu.com/feisty/](http://packages.ubuntu.com/feisty/)
