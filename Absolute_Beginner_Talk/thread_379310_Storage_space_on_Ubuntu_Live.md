So, you're just using the LiveCD and you want to save things from it? The LiveCD isn't really meant to be a day-to-day OS. AFAIK the LiveCD is just used to test out Ubuntu and try it out for a little while and if you like it go with the install and dual boot with windows or if you really want to Format and just use Ubuntu as the OS on the computer. If you need to save things you should be able to plug in your Flash drive or thumb drive and save packages or files to that from the LiveCD until you get ready to do a real install.

Hope this helps.
