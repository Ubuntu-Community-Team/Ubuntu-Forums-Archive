### Post by nhandler on 2007-11-22
It would help if you gave more info about what you want the book to cover. When I first made the switch to Ubuntu, this guide was a big help. It showed me how to get a lot of the programs I used on Windows running with Ubuntu. It also helped with many other Ubuntu related things. It isn't an actual book, but you can always print it out or download it to your hard drive.
