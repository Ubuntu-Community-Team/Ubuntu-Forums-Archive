### Post by freebird54 on 2007-06-06
The command for creating a directory is **mkdir**.  Like this:

```
mkdir directoryname
```

When you first open a terminal, you are in your home directory, and that is a good place to put such things.  If you are NOT in your home directory, just entering **cd **will get you there.

You might want a look through this as go along:   [http://www.linuxcommand.org/lts0010.php](http://www.linuxcommand.org/lts0010.php)

Lots more than you need now - but a good wat to find out what you CAN do... :)
