I was looking for a way to Encrypt and Decrypt a file using right click of mouse

Install
1- libgpgme11
2- seahorse

restart X (Cnt+Alt+Backspace)

re-log in and try it. It gives me the menu on right click.

If not make sure then that gnupg is also installed.

Oh, yes install the above by going to System->Administration->System Package Manager and perform a search for each file.
