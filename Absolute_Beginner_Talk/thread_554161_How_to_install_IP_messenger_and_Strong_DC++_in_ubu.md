### Post by ijason on 2007-09-18
the easiest way to install programs for ubuntu is to check either of the two repositories to see if there is a version already in them. you can check one with the _system>admin>synaptic package manager_. pop up menu. the second is _application>add remove_

if the program isn't listed in either of those sources, my preferred method is to actually search the forums for the name of the program to see if anyone else has needed help installing.

good luck!
