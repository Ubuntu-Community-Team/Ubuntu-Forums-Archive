### Post by Dylnuge on 2007-02-25
> **oasiao said:**
> Another question, anyone know how much python code is Ubuntu?

Ubuntu's config files are primarily in three types:

Binary C Files
Bash Script Files
Plain Text Files

You should not have much problem figuring out the later two if you know python.

Since most Ubuntu users rely on community support, your assistance will be most greatly appreciated helping them out. If you are looking around the forum, you can find many great projects to work on. Just remember that Ubuntu is the operating system. It is the complex underlying layer that controls the computer. Even graphical frontends like GNOME and KDE are separate projects. Decide what aspect you like working on most, and lend a hand there.

Welcome to the open source community.
