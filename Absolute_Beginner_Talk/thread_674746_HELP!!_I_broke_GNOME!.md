### Post by staticvoid on 2008-01-22
when it boots hit ESC and boot into restore mode... i think there is an option for that... or just reinstall and this time use ubuntu ;) which is based on debian.

to recover your files use these commands:

cd - change directory
ls - list files
cp - copy

e.g.

cd ~/
ls
cp sound.mp3 /media/usb
