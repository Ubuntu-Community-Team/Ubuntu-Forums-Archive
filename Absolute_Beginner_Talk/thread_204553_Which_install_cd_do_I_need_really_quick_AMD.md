### Post by steve.horsley on 2006-06-27
[QUOTE=Apple 101]What is the actual difference between the desktop and the alternate version?[/QUOTE]
The desktop installer is a live CD that has a graphical installer  with it. The Alternate installer is not a live CD, has a text mod e installer (as all previous versions have had), extra inatall options, raid support, and has a rather useful rescue mode with it.

Unless you are doing an absolutely standard default install, you should probably use the alternate installer.
