### Post by Martin on 2007-05-09
I'm not sure how much time you spent looking at the wine compatible software list. But at this point there is a lot more stuff that does not run under wine than stuff that does. If your software is listed as supported on the wine site then you have a problem that might be fixable with more info. Otherwise your software probably won't work. Look for a native linux alternative.
