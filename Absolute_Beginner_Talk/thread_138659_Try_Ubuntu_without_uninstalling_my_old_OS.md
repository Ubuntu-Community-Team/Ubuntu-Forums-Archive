### Post by Sandlst on 2006-03-02
yes- on the download page, select "Live CD", this will allow you to try ubuntu out without doing anything to your harddisk, however it is considerably slower than a real install; if you like it, another option is to dual boot-dual booting (provoided you have enough disk space) will allow you to keep your old OS, and choose when you turn your computer on which OS to boot into, hope this helps!

*EDIT* to use the livecd, just put it into your cdrom drive, restart your computer, and it should automagically start; if it loads windows instead as usual, you may have to change your BIOS settings to boot from cd first.

### Post by Coelocanth on 2006-03-02
Yes. You can do it a few ways:

1) Get a copy of the Live CD and you can run Ubuntu from the CD without having to install it at all.

2) DL and burn the Live CD ISO and run it from the CD.

3) Install Ubuntu as a dual boot with Windows (this is what I've done). This may involve resizing your Windows partition, which would mean you should back up all your data.

### Post by AtinLango on 2006-03-02
After downloading the .iso, you don't just copy it! You have to create an image onto the CD.

Easy. Lets assume you downloaded the .iso onto some folder of your hard drive. If you have a good CD burning software that can create images (for me I use Sonic RecodNow, and I guess nearly any other can do), just doubleclick on the .iso file. It may prompt you to insert a CD. Do it. It will create the image in the CD.

Once completed, restart your computer with the CD inside. If your BIOS is set right to boot from CD, you are home and dry.

Try and let us know.
