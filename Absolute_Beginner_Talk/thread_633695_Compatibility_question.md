### Post by CaptainInsaneO on 2007-12-06
I absolutely agree with shadow. If you're not a gamer, there's really no viable reason to keep Windows around (unless using Ubuntu for you is more painful than Windows, but I hope you'd give it a valiant try in those cases). Ubuntu can do everything Windows can do, plus some (including gaming).

Personally, I keep Windows on a small drive on my machine, and Linux is on another larger physical drive. I LOVE gaming, and that's why I keep Windows around. HOWEVER, I haven't booted into Windows in months. The games in Linux are fairly good if you know how to find them, and for me, they were a fantastic replacement to a lot of my Windows games (and some games for Windows also run natively on Linux ;))

Earlier I had made the suggestion that you read up on dual-booting Windows and Linux because a lot of new-comers to the Ubuntu community accidently delete their Windows partitions during installation of Ubuntu and hopelessly lose all data on said partition. It's a very good idea to get some help and knowledge on installing Ubuntu next to Windows if you are not knowledgeable on the subject. If you are AT ALL confused about the process, never hesitate to ask for help. That's what this forum is here for. :)
