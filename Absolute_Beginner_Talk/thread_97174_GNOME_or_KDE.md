### Post by phanboy_iv on 2005-11-30
You can install Kubuntu via synaptic by searching for "kubuntu-desktop" and installing that. Or, if you're running Kubuntu and want to install Gnome, just do "ubuntu-desktop" instead. Then you can pick the one you want at the login screen with the "sessions" menu.

I recommend you try the Kubuntu/Ubuntu live CD's though. 
That way you don't have to install anything for a test run.

Also, someday you might want to try XFCE. It's a bit more lightweight/minimalistic that Gnome/KDE, but still very nice. The package to install for that is "xubuntu-desktop"
