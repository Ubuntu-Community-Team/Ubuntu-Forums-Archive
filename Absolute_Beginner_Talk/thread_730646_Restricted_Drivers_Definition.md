Restricted drivers are drivers that are not released under a license that is compatible with Ubuntu and thus cannot be shipped on the CD.  This is mainly for many wifi cards, and most graphics cards.  These companies keep the workings of their drivers and firmware closely guarded secrets and won't release them open source to be integrated into the kernel like every other driver, so we are stuck with this less than optimal situation.

As far as the end user is concerned... you should install them, your system will almost certainly run better with them installed.
