I have been using Linux for years and I use it for:

11. IRC - irssi, xchat, ksirc, mozilla-chat, kopete

Total Cost of Ownership:
-----------------------------------------
Windows - $$$
Linux - $0.00 (not counting blank CD's)

Is Linux free? Yes, absolutely.

Not really true.
Qt is widely used.  Skype uses it.  You have to pay to develop comercial applications with Qt

[http://www.trolltech.com/products/qt/pricing.html](http://www.trolltech.com/products/qt/pricing.html)

Quote:

"When your team buys commercial Qt licenses, you receive the following: 
 The right for one, named and registered developer per license to use and develop commercial applications with Qt. All developers in your company using Qt will need individual licenses.  
 Maintenance: One year of upgrades to all new releases of Qt.  
 Support: One year of email-based support for the registered developer. Find out more about Trolltech support.  
 The complete source code of Qt for the platform(s) you have purchased, with the ability to modify it for your needs.  
 A subscription to our technical magazine Qt Quarterly, bringing high quality technical content from Trolltech and partner developers, documentation and support people. "




As I mentioned, the same goes for a lot of Java technology. 

The question is whether things in the linux world are free.  I think a better description is not to call it the linux world, but the free-libre open-source world.  You can run Qt, Java, Python, PhP, Perl, and a whole lot of other technologies on many platforms (Microsoft, Linux, Mac, BSD, etc...)

Now, if you chose to use a non-free (proprietairy) cross-platform technology, it is what it is.  You probably will have to pay somebody for the right to use what they own (proprietairy software is software that is not owned by everyone - not free)

If you chose to use a free-libre cross-platform technology, you will avoid the catches of proprietairy software.  You will find that there are more than enough option in the free-libre world to satisfy your needs.  Free-libre does not mean of lesser quality than proprietairy.
