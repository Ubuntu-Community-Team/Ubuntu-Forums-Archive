It is a question of what is 'mounted' from the live CD. It is easy (if you know how) to mount windows and other partitions using ubuntu live CD, but knoppix intends to be something of a general purpose toolkit, while ubuntu live cd intends to be a serving of the user distro - with some simple safeguards in place.
If you were to install ubuntu I think you find that the other stuff (eg windows) is mounted ready for you, but not from the live CD.
