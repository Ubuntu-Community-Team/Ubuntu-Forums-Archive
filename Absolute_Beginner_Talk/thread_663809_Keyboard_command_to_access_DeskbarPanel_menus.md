### Post by jonkulp on 2008-01-10
If you type Ctrl+Alt+Tab, you can select the top or bottom panel or the desktop, then use the right-left arrow keys to navigate to the app or menu you want.  If you get stuck in a loop on the Applications/Places/System side of the panel, press Ctrl+Tab to move over to the next area of the panel.

### Post by jonkulp on 2008-01-10
I think you can remap the keystroke, but I haven't tried it yet.  It's a very awkward keystroke if you try to do it all with one hand, so I always hold down hold down Ctrl with right hand while doing the Alt+Tab with the left (that way the left hand is doing the familiar keystroke for toggling among open windows).  I have the right Alt key assigned as the "compose key" for conveniently doing foreign characters with keystrokes (such as à and á).  You might be able to re-assign keys to suit your purposes in the same way I assigned this one?  Go to System>Preferences>Keyboard, then choose the Layout Options tab.  There are a number of things you can customize there, but I've only assigned the compose key so I can't comment on anything but that.

Just noticed something else, btw.  You can access the Applications/Places/System menus by pressing Alt+F1.  It doesn't seem like you can get to the rest of the panel that way though.  There's a list of a bunch of default keystrokes in System>Preferences>Keyboard Shortcuts.  I don't see one for moving focus to the panel.  If you figure out an easier way than the Ctrl+Alt+Tab let me know!  JK
