### Post by BobCFC on 2007-08-01
Try clicking on System->Preferences->Preferred Applications

Then on the Multimedia tab  change Rythmbox to Amarok or anything else.


Recently I am trying Listen Media Player.. seems to be a good iTunes replacement for my podcasts.  But I change all the time if anyone else has good suggestions.  Tried Songbird but it doesn't works with my rss feeds properly.


EDIT:  if you want to change the default app for a particular file type it is slightly different.. For instance, if you want to open MP3s with a certain app, right-click on a file, choose properties->open with... then use add remove to set the application.
