### Post by Paqman on 2007-10-09
> **OneMoreOfThem said:**
>  I need to download any files I will need beforehand from this computer at work.

If you're using GNOME instead of KDE then you'll have a quite a few files to install to get Amarok. That's because Amarok is designed for KDE (don't worry, as long as you get all the dependent files, it'll run fine)

One thing you could try:

If you boot up your LiveCD into a machine at work you should be able to fire up Synaptic and search for Amarok. Select "download package files only" before hitting Apply. That will download Amarok and any dependant packages. If you put them on a USB flash drive you should be able to take them home and install them.

You might want to also try Exaile while you're at it. It's a GNOME clone of Amarok.
