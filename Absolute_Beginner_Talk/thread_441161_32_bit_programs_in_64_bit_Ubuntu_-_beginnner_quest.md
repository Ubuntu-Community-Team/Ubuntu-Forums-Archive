### Post by starcraft.man on 2007-05-12
Just a question right off the top, but why did you choose the 64 bit? I mean, I've yet to really see any compelling reason for the average user to use it. It will cause incompatibilities/problems with programs, quite a few apps don't even write a 64 edition, not to mention I don't think theres any speed improvement or other such bonus for the trouble. Just cuz you have a 64 bit chip, doesn't mean you have to use it for such. I'd recommend ya go install the 32 bit, life will be much nicer to ya, and you can install all the regular programs real easy. :)

Thats my two cents. 

As for the windows programs, if you absolutely need them, check the [wineHQ]("http://www.winehq.org/") site and see if its in the appDB, if not you might find it a bit of trouble/not working under WINE. As for opera, check my sig, theres an install guide there for 64, it works for most people I think >.>.
