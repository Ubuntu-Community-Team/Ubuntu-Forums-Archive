### Post by rcatman on 2006-09-14
CHeck in your cd-burning program if there is an option "Burn Image". If you burn an .iso file as a data file it will burn the image onto the cd as on file. Usually there's an option in the cd burning application to burn an iso/image file. 

Also make sure to change your bios to boot off the cd first. 

I would also recommend downloading the 'alternate' cd rather than the 'desktop'. The desktop cd boots up in live mode, then you have to install, for me its a waste of time. The alternate cd installs in a simple command interface and takes less time.
