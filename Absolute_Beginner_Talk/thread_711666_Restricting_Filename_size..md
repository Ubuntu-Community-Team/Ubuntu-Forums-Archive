Hmm, really not... beautiful. The problem gets really boring when there is/are more icons down the big-file-name one... he simply takes all space available, huh.

I don't know a full solution to this, but, on nautilus, you can do the following:

Open a window, edit -> preferences -> on "Icon view defaults", mark "Text beside icons".

This way the text will look *less ugly*, let's say. :)
