### Post by Sef on 2006-07-27
> I have been reading around ad apparently Adobe does not support linux in flash development. Does this mean that all flash sites are useless for me?

Adobe's last linux version is version 7.  There was not a version 8  for Linux.  There is version 9 that is being developed, but it is not scheduled for release until early 2007.

To get flash, you can install it from Multiverse.  To add the multiverse repository, [click here.]("https://help.ubuntu.com/community/Repositories/Ubuntu#what")

If you have trouble with flash after install, [click here.]("https://help.ubuntu.com/community/RestrictedFormats")
