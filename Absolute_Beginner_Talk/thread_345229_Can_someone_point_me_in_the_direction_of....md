### Post by jpkotta on 2007-01-24
This is pretty good for shell stuff: [http://rute.2038bug.com/index.html.gz](http://rute.2038bug.com/index.html.gz)

As far as shortcuts go, each program has it's own, so you just have to be curious and learn them as you go/read the docs.

I didn't look too carefully, but didn't see this trick mentioned: ```
. /etc/bash_completion
```

It defines a whole bunch of specialized completions. Add that line to ~/.bashrc to automatically enable it.
