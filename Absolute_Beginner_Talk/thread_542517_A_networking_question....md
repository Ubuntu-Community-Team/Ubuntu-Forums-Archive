You can use this here for a feisty server:

[http://www.howtoforge.com/perfect_setup_ubuntu704](http://www.howtoforge.com/perfect_setup_ubuntu704)

Or this here for a debian server:
[http://www.howtoforge.com/perfect_setup_debian_etch](http://www.howtoforge.com/perfect_setup_debian_etch)

I'd go for a debian one as debian has been proven to be rock-stable as server (my sarge server had an uptime of over 250 days before I reinstalled it to etch ;) )

The howtos are quite easy, most of it is copy'n'paste ;)

You will get a full LAMP server, DNS-Server and a "webhost" software (ISPConfig) that will allow you to setup multiple websites.
