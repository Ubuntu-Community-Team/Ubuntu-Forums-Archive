### Post by aktiwers on 2007-09-11

```
sudo apt-get install xchat
```

Then Look under Applications => Internet=> Xchat

### Post by abhilash82 on 2007-09-11
You don't have to install any irc client as ubuntu has one in-built. you can invoke it by typing xchat in your terminal. For more details, go to this link [here]("https://help.ubuntu.com/community/XChatHowto?highlight=%28xchat%29").
