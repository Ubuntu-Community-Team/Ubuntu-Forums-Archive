### Post by starcraft.man on 2007-05-16
Ummm, I don't know that much about the basic burner, just looked at it and I don't see the burn speed option at all. I would recommend that you use burning programs with a bit more control, the basic one is well designed to be really basic >.>. Try gnomebaker if you want something simple and gnome based, I however prefer k3b it offers a few different functions not found in gnomebaker, but its also kde based so if your on gnome/Ubuntu you will be running it with that (not that there are ever usually problems, simply pointing it out). Both are in the repos, you can install them as always "sudo aptitude install x", where x is k3b or gnomebaker.

That should be it. If btw your speed burning option is greyed out in either/both of those, then you may have some kinda hardware issue, I find it doubtful though, enjoy :)
