### Post by david_e on 2007-09-03
It seems that the video card has been configured correctly. The problem is in the configuration of the monitor. The fastest way to fix this is by editing the xorg.conf file, but you have to know which resolutions your monitor supports. For example 1600x1200, 1280x1024 or 1400x1050. If I am correct your monitor should run at 1280x1024, but you can try something more. 

Close gedit if it's still opened.

Let's get to work. First we make a backup copy of the file.

```

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup

```

then we open it with gedit:

```

sudo gedit /etc/X11/xorg.conf

```

then find this section:
```

Depth 1
Modes "1024x768" "800x600" "720x400" "640x480"
EndSubSection
SubSection "Display"
Depth 4
Modes "1024x768" "800x600" "720x400" "640x480"
EndSubSection
SubSection "Display"
Depth 8
Modes "1024x768" "800x600" "720x400" "640x480"
EndSubSection
SubSection "Display"
Depth 15
Modes "1024x768" "800x600" "720x400" "640x480"
EndSubSection
SubSection "Display"
Depth 16
Modes "1024x768" "800x600" "720x400" "640x480"
EndSubSection
SubSection "Display"
Depth 24
Modes "1024x768" "800x600" "720x400" "640x480"
EndSubSection
EndSection

```

And change it like this:
```

Depth 1
Modes "1024x768" "800x600" "720x400" "640x480"
EndSubSection
SubSection "Display"
Depth 4
Modes "1024x768" "800x600" "720x400" "640x480"
EndSubSection
SubSection "Display"
Depth 8
Modes "1024x768" "800x600" "720x400" "640x480"
EndSubSection
SubSection "Display"
Depth 15
Modes "1024x768" "800x600" "720x400" "640x480"
EndSubSection
SubSection "Display"
Depth 16
Modes "1280x1024" "1024x768" "800x600" "720x400" "640x480"
EndSubSection
SubSection "Display"
Depth 24
Modes "1280x1024" "1024x768" "800x600" "720x400" "640x480"
EndSubSection
EndSection

```

Save and exit gedit. Now you can restart your Xserver by pressing CTRL+ALT+DELETE (all the opened applications will be closed without saving anything so save your work before doing this.). The resolution should be fixed now.

Before trying other resolutions instead of "1280x1024" (that will work on your monitor) remember that a wrong config will make your system unable to go graphical and you will need to fix your xorg.conf, by pressing CTRL+ALT+F1, loggin in and use nano to fix the file:

```

sudo nano /etc/X11/xorg.conf

```

HorizSync 30-70

Like this:

<code>


Section "Monitor"
Identifier "DELL E770s"
Option "DPMS"
HorizSync 30-70
VertRefresh 50-160
EndSection

</code>

in your /etc/X11/xorg.conf file

Linux always starts with your default rate and the first resolution in the line under the default rate, so in your case:

Depth 24
Modes "1024x768"

... which is why your resolution is set to 1024x768.


After you log in, you should be able to go to system>preferences>screen resolution to check your settings or change them.
