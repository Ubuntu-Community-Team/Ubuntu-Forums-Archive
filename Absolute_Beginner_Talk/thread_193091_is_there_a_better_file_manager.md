### Post by u.b.u.n.t.u on 2006-06-09
The default "windows browser" for ubuntu is called "nautilus". It is every bit as good as "windows explorer". Better in many ways, as it isn't 5 years old.

I store all my data on a different HDD.

With the default install of ubuntu, your "home" is equivalent to XP "My Documents". So you can throw everything in there. If you have a separate partition or another HDD you can have a dedicated storage area.

Creating folders and links is easy. Just state what it is you wish to do. It is worthwhile keeping nautilus and learning how to use it.

Every other directory (folder in windows terms), is equivalent to the "Windows" and "Program Files" of say XP. You don't want to mess with those unless you have a specific reason.
