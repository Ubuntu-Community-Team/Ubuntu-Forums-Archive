### Post by SickTwist on 2005-12-12
[QUOTE=giantweta]I guess I'm SOL with my Pentium MMX 166 with 64Mb RAM then? :([/QUOTE]

If part of the memory is shared with an onboard graphics adapter then yes. If your system has a separate graphics card with its own memory then it *might* be possible to install Ubuntu on that machine. You will definitely need to use a light desktop (e.g. Blackbox, Fluxbox, IceWM, FVWM, FVWM95, XFCE, etc.) if you want a GUI.

If you do not wish to use that box as a workstation, it might work well as a file server or a dedicated firewall.
