### Post by steve.horsley on 2007-10-24
I use VirtualBox. Two steps to install it. First, install build-essential from Synaptic (System->Administration->Synaptic). 

Second, download the appropriavte .deb:
[http://www.virtualbox.org/wiki/Downloads](http://www.virtualbox.org/wiki/Downloads)

Third, double-click the downloaded .deb file to install it. Then start it from System Tools menu.
