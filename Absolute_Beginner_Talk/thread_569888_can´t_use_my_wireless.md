### Post by hollerith on 2007-10-07
You have a driver loaded for this hardware?  If ubuntu install doesn't detect it and you cannot find a suitable driver then you are going to have to use the Windows driver and ndiswrapper.  If you think you do have a working driver and it sounds like you have investigated this - it may just be you need to reconfigure network, essid and password and such.
