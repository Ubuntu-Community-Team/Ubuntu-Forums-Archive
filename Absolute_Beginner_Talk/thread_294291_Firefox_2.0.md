### Post by Average Joe on 2006-11-06
Maybe there is still an incomplete file in your Firefox cache folder. My first try would be to clear the cache folder, and make sure that it is really empty, and then start Firefox again.

Your Firefox cache folder is probably here: ```
~/.mozilla/firefox/profiles/whatever.default/Cache/
```
where *whatever.default* is some ramdon name.
