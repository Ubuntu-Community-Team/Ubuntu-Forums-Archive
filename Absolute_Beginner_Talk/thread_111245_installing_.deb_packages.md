### Post by mazelado on 2006-01-01
I've used Debian packages before without a problem. I don't think there's any real reason not to, but then again I'm far from an expert on packages and distros. You can even use RPMs if you use alien. The biggest reason that I know of not to use converted RPMs is that the directory layouts in Red Hat-based distros and Debian-based distros are slightly different. That's also the only reason I can think of that Debian packages might not work exactly right in Ubuntu.
