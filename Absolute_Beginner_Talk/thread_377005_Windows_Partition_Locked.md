### Post by jpeddicord on 2007-03-05
Try defragmenting the partition in Windows in safe mode to reset the filesystem. When your system starts up, hold F8 before the Windows logo appears. Select Safe Mode, then let the system boot. Go to Start > Programs > Accessories > System Tools > Disk Defragmenter, and let it run for as long as it has to. It may take a few hours, depending on how much you used Windows.
