### Post by TeaSwigger on 2007-11-22
Hello parm289 and welcome :)

Give that Live CD a spin! Be patient and methodical, and bear in mind that to a great extent everyone has to "learn as you go" just like we did when we started with windows. Search a lot, ask a lot - this is a great spot to get information and answers. Others include:

[http://www.psychocats.net/](http://www.psychocats.net/) (various info geared to new & newish users)
[http://users.bigpond.net.au/hermanzone/](http://users.bigpond.net.au/hermanzone/) (this one is about dual-booting with windows if you are interested in that)
[http://ubuntuguide.org/](http://ubuntuguide.org/)
[http://doc.gwos.org/doku.php](http://doc.gwos.org/doku.php)

Linux is very modular and highly configurable - choice is a fundamental concept, really. There's not much you can't change it's just how long you're willing to plug at it. ;) It put the interest and fun back in using the computer for me, by giving me control.

The LiveCD will give you an idea if your harware will "just work" - although if it doesn't, that doesn't mean you're plain out of luck. There are often "ways and ways" as they say.

Installing stuff is amazing - a startling amount of things you can install from ubuntu's own repositories (think on-line storehouses of ready to install pre-screened programs & stuff) with incredible ease and speed. Being free sure doesn't hurt either ;)

I do hope you can get to be comfortable with the command line in time; it's very handy and powerful in Linux. There are a surprising amount of things you can do with it. Oh and you can dress it up to look nice too ;) But you may not absolutely need to use it much if at all if you really, really don't like it.

Don't worry too much about which *buntu to pick; you can mix and match peices later if you want. To me the surface difference is:

Ubuntu is easiest to use, tending to intentionally simplify interfaces. By default it uses warm colors. It may have the highest chance of "just working" since it gets the most developer focus.
Kubuntu has the higher number of "best" programs by default, and tends to be more specific and featured in interfaces. By default it uses bright blue / silver colors.
Xubuntu is a bit "cleaner" with an interface somewhere in the middle of the other two and a bit of a no-nonsense feel. By default it uses blue and flat light grey colors.

In the future you may also like to know about Fluxbuntu:
[http://www.fluxbuntu.org](http://www.fluxbuntu.org)
who are developing a variation that's more "esoteric" to some than the main *buntus, but is different, distinctive looking and faster since it uses less of the computers' resources. It'd probably be helpful to get familiar with things in ubuntu first though. It's something to keep in mind if you like. :)

There are numerous variations of Linux. I do think Ubuntu is a fine full-featured one, especially to start with. Luck & enjoy :)

### Post by parm289 on 2007-11-25
Well guys, I got it - my Linksys usb card works!!  What I did was (in case anyone else has the same problem):

1.  Download the ndiswrapper build to a hard disk
2.  Follow theses directions to install:
```

Copy the appropriate files over to a directory on your Ubuntu computer (e.g. your Home directory) and install them in this order: 



  sudo dpkg -i ndiswrapper-common_*.deb

  sudo dpkg -i ndiswrapper-utils*.deb

  sudo dpkg -i --force-depends ndisgtk_*.deb



The commands listed above are a general example of how to install a .deb package from the command line. You need to be in the directory where the files were copied to.

```
3.  Copy the Linksys WUSB54G drivers for your respective hardware to your hard disk.
4.  Go to System>Administration>Windows Wireless Drivers
5.  Select the driver that you want to install.  Make sure the .inf and .sys files are in the same directory.
6. From the ndiswrapper window, configure your wireless connection.

-----------------------------------------------------------------------------------------------------------
This is what got it working for me.  Now that I know how to set it up (and it was QUITE the learning experience), I'm installing ubuntu at the next chance I get.
