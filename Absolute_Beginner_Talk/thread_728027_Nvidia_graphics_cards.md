### Post by Xyphus on 2008-03-18
Depends on what you want to spend...

The 8600GT is a nice budget card that is plenty powerful enough to run just about anything you'd want to...

If you want a little more power and want to go with a mid-grade card, the 8800GT is a great bang-for-the-buck card...

Both are fully DX10, so if you dual-boot with Windows, you should still be able to play some games with decent settings...

### Post by forrestcupp on 2008-03-18
I got an 8600.  I'm pretty happy with it.  You can get one for around $90.  If you want to spend more, go with the 8800.  Don't go with the 9xxx series, unless you want driver headaches.
