### Post by endersshadow on 2005-12-16
For Flash and other movie codecs:
[https://wiki.ubuntu.com/RestrictedFormats](https://wiki.ubuntu.com/RestrictedFormats)

For other programs:
Hit your System menu, go to Administration, and then to Synaptic Package Manger.  Search for the program you want to install, mark it for installation, and click "Apply."

You can also install from the command line with:
```
sudo apt-get install programname
```
