### Post by jken146 on 2008-03-29
> **ssulaco said:**
> Next question......(remember im only used to windows)......What about utuilities, Defragmenter,Ccleaner,AVG antispyware,....Spybot S/D....etc,etc.....what utilities will I need?
.......and mostly what about UPDATES

I like answering this question.  The answer is: you don't need to do anything.  Defragmentation is not a problem on the filesystems that Linux uses, there is no registry like in Windows (so no need for CrapCleaner), you won't get a virus, there isn't any spyware, and there are no services listening on any ports by default (so no need for a software firewall (although there is one -- iptables)).  You'll get updates automatically for all your programs.

### Post by forrestcupp on 2008-03-29
Yeah, that's the great thing about Ubuntu.  In Windows, you get automatic updates for Windows.  But in Ubuntu, you get automatic updates for everything you install from the repositories.  Another beauty is that if you want to install an app, you just have to open Synaptic and either search by category or by keyword for the type of app you want to install and it downloads and installs it for you.  If you have supported hardware and use the tools that are given to you, it's much easier than Windows, not harder.
