### Post by dfreer on 2007-05-18
I'm pretty sure that you want to use Vista's repartitioner to shrink it's partition, I hear bad things happen to good people when you try using Ubuntu/Partition Magic. Lemme look up some links really quick.

Actually, I can't find the link explaining why you shouldn't use partition magic or ubuntu installer (although most people seem to think the ubuntu installer can hadle the repartitioning of the Vista partition, IDK about that). But you *can* use vista to repartition it's own partition, I would recommend doing it that way, then install ubuntu to the free space.

About some of the original questions:
CS and WoW *should* work using WINE (Which stands for "Wine Is Not an Emulator", btw). Battlefield 2134 *should* work with the newest version of Cedega (which is a fork of wine that costs around $5 a month to use, but it may be worth it, I hear F.E.A.R. may work in it now too :D ).
Note that in order for these games to run, you will need to install the correct video driver for your system, mainly to get 3D Acceleration/direct rendering working. When you install Ubuntu you can find out if it is already working by typing in terminal:
```
glxinfo | grep rendering
```
It should say direct rendering: yes if it does.


My buddy's Dell Inspiron 1505 (I'm pretty sure that was the model he had) used an ATI Radeon card (a bit trickier to get driver installed then nvidia), and also a broamcom wireless card (works fine once you install the native windows drivers using ndiswrapper). You will need to be aware of these things, but all in all you should have a great time in Ubuntu! Welcome!
