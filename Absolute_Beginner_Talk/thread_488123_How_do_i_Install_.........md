### Post by starcraft.man on 2007-06-29
Open any terminal and make sure firefox is closed. Also make sure you have the extra repositories active (I forget which, I have them all :D) Then simply paste this into the terminal:

```
sudo aptitude install flashplugin-nonfree
```
