### Post by GrueTamer on 2007-06-08
For me, the main difference between BSD and Linux lies deep down below the programs and GUI and stuff, but at the core kernel coding.  BSD was made to BE unix, Linux was made to be LIKE unix.  But beyond that, they're a lot alike, with BSD's linux compatibility layer, as well as all the native software.  Other people's links will answer your question more directly, but it's always important to know that when you use BSD, you're getting a more Unix-like experience than when you're using Linux.  That's really all I have to add, everyone else input good links and information.
