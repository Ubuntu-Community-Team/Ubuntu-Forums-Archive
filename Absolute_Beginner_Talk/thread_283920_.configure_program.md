### Post by bwobbones on 2007-03-11
All thats going on here is that in the ./configure script, the c program that they use to check if the SDL library exists doesn't compile.  If you take the line (142):

```
echo "int main(void) { if (SDL_Init( 0 ) < 0) { printf( \"SDL_Init(): %s\\n\", SDL_GetError() ); return 1; } return 0; }" >> "$FILE"
```

and make it:

```
echo "int main(void) { return 0; }" >> "$FILE"
```

everything works just fine (assuming you have libSDL-dev installed).
