### Post by Rocket2DMn on 2007-07-03
internet: firefox (comes with Ubuntu?)
ntfs: ntfs-3g with ntfs-config
messaging: pidgin - the next generation of GAIM (this may not be in the repositories yet)
OTHER STUFF:
music: there are a million players, I use rhythmbox (came with Ubuntu)
video: vlc
email: mozilla-thunderbird

Most programs you need can be downloaded and installed with apt-get:
```
sudo apt-get install programname
```

### Post by NeoLithium on 2007-07-03
First things first.  Welcome to Ubuntu :)

Second, there's a lot of good beginners guides that are kicking around.  ones that I found helpful and still hand out to those that want to learn more are [http://www.psychocats.net](http://www.psychocats.net) and [http://www.ubuntuguide.org](http://www.ubuntuguide.org)  They're easy walkthroughs for a lot of common programs and some other things that make life a little bit easier.

Of course, you can always search around the forums for things as well; there's quite a lot of info in the Tutorials and Tips section that you might like.  It's located here: [http://ubuntuforums.org/forumdisplay.php?f=100](http://ubuntuforums.org/forumdisplay.php?f=100)

Some good advice, don't get discouraged and never feel stupid. Everyone starts out somewhere and it just takes practice to get everything down to a science. :)
