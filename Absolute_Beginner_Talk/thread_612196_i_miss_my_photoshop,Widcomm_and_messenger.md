### Post by SteveHoffmanUK on 2007-11-14
NoSmokingBandit wrote:
> virtual machines are a whole lot slower than just rebooting. Unless you have a really fast pc youll probably want to set up a dual-boot.

Not in my experience, but it may depend on your machine resources and the application. I run Virtualbox on a 1.8Ghz processor with 1.3 Gb of RAM for one XP application (a WYSIWYG web authoring program), and it seems to run as fast as in native XP mode. Maybe Photoshop would be a lot slower, because it's graphics.

Virtual machines are really useful if you need to switch quickly between your Linux app and your Windows app. For example, I'm getting Linux emails from my website customer asking for changes in his website (XP app), so I'm reading the email (Linux), then making a change (XP), then reading the next bit of the email, then making the change etc. I'd go crazy if I had to reboot every time I needed XP.

If you're working for hours just with Photoshop, then the switching isn't so important. If you must continue to use your Windows applications, I'd try to limit them to offline ones because of the viruses and other security issues.
