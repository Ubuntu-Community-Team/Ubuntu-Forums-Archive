Hi, welcome to Ubuntu :)

I can't really recommend any books for you but if you are searching Google for answers there is a dedicated Linux area.

[http://www.google.com/linux](http://www.google.com/linux)

It might still be a like hunting for a needle in a haystack but at least it will be a smaller, Linux related haystack :)

Just as another quick pointer about the other machine you tried to install Ubuntu on, the one with the 19" Widescreen display, you probably could have got the right resolution with the proper graphics driver for the video card. I have the same display myself and running on Ubuntu with an Nvidia card. There is a really great script for setting up both ATI and Nvidia driver on Ubuntu called Envy.

[http://albertomilone.com/index.html](http://albertomilone.com/index.html)

Alberto Milone does a great job and Envy sorted my drivers out. You might wanna give it another go sometime and see if you can get the right resolution. Just a suggestion, it's up to you obviously.

[http://www.tuxfiles.org/](http://www.tuxfiles.org/)
