### Post by d@emon_us3r on 2008-01-13
I ordered the Ubuntu installation disc from the official site. It was the 7.04 version. At first I couldn`t boot from the disc, so I set the boot order to boot from the CD drive first. It still didn`t work. I used the Smart Boot Manager from the disc, put it on to a floppy using rawwrite and then tried to boot from the disc using that floppy. It still didn`t work. The only logical conclusion that the boot sector of the disc or the disc itself is damaged, but I know it isn`t because I used a software called Virtual PC and with it, I suceeded in booting from the disc. ### Post by Malta paul on 2008-01-13
Hi, Welcome, Does your CD boot to the disk menu start screen? 
If not, you could try changing the 'Integrated Peripherals' in your BIOS to Legacy Mode.
If you can boot to the start menu, select 'Check CD for defects' if the disk checks OK then try 'F4' and set you VGA resolution. then try the live disk boot
Good luck hope this is of some help :)
