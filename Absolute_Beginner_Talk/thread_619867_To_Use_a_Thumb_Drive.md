> **Lawtonfogle said:**
> I have tried using search, but one of two things always happen, either they are discussing installing from a thumb drive, or the discussion goes right over my head.

So I ask, for someone who knows cd, ls, gcc, and not much else, how do I use my thumb drive?  More specifically, I would like to transfer a file over to it, so I can transfer that file onto a different computer.

Well all I do is plug it in and it appears on the Desktop, then just copy your file to it (either
via drag and drop or using cp from the command line). To remove the drive, right-click on the drive's
icon to bring up a menu and I think you choose "eject" to unmount it, if not it is something fairly
obvious like that, then the icon will disappear and you pull the drive from the USB socket.

Ron.
