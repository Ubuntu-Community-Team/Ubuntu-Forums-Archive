### Post by dstew on 2007-03-10
It is better to install Windows first, then Ubuntu. Windows still does not make allowances for dual-boot operating systems, at least not easily. Ubuntu was born to dual-boot. You can install Windows after Ubuntu if you want to try, but it will be a little more difficult to get the system to dual boot correctly.
