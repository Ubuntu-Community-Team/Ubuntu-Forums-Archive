### Post by axle_foley00 on 2006-12-10
You will need to change a BIOS setting for your computer so that it attempts to boot from the CD-ROM/DVD drive before the Hard Drive. Usually when you just boot up the computer you can press F1 or DEL or something like that and it goes into your BIOS Menu. Look for a tab called "Boot" and it should allow you to set it to boot from the CD-ROM drive before the Hard drive. Once you have the CD-ROM first and then the Hard Drive second, you can save your changes and exit the configuration menu and it should work.

Hope that helps.
