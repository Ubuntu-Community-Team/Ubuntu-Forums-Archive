### Post by shad0w_walker on 2007-10-21
I would recommend doing a backup of any thing you want to keep and just reinstalling it on the new system. 

However, it is possible that just moving the hard drive to the new system will work, it is quite possible that it will be able to handle the hardware changes as long as you don't have anything compiled specifically for that type of processor (swiftfox, specific kernel, that sort of thing) you might get lucky.

P.S. Backup before you try moving things to the new system, I believe its possible for the swap out to work however I haven't done anything like that in quite a while and mine didn't end to well.
