### Post by Sutekh on 2006-04-06
Nope so far so good.  

You download the torrent file then use a bittorrent client to download the Ubuntu installation ISO.  Have you used bittorrent before?

Once you have that, you can burn it to a CD.  But the key is burning it as an **Image** not a data disc not a bootable disc.

What software are you going to use to burn it?  You should be able to find the option to burn an image pretty easily.  In Nero its in the Recorder menu (skip the wizard when it starts)
