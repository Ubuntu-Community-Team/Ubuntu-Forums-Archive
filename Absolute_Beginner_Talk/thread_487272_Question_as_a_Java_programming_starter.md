### Post by bukwirm on 2007-06-28
Install one of the JDK (java development kit) packages - I used the 'sun-java5-jdk' package. You probably want java6 so install the 'sun-java6-jdk' package.

I would not start programming using an IDE (integrated development environment like) Eclipse - it hides some of the details.
