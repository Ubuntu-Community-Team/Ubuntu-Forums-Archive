### Post by perfecttao on 2007-10-12
Active Directory is a MS term to describe a user authentication and permissions based network server architecture.....there are other options available using Ubuntu (Samba/LDAP), but do not confuse this with Active Directory....this is free!!

There is more info on this available here: 

[https://help.ubuntu.com/7.04/server/C/configuring-samba.html](https://help.ubuntu.com/7.04/server/C/configuring-samba.html)
