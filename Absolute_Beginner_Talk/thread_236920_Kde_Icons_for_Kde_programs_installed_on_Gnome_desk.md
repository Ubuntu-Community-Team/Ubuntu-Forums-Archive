### Post by siacs on 2006-08-17
For a desktop icon - right click > Properties > Select Custom Icon

For a panel icon  - right click > Properties > click on the icon to change it

For a menu icon - Open Alacarte Menu Editor, find Ktorrent then right click > Properties > click on the icon
