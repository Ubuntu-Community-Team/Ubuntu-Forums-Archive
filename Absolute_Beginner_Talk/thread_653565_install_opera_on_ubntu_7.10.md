### Post by elithrar on 2007-12-30
Just download the [Opera 9.5 beta](http://www.opera.com/download/index.dml?opsys=Linux%20i386&lng=en&ver=9.50b&platform=Linux%20i386&local=y) for Ubuntu 7.10 (or use the link there to install the current 9.2 stable) - or open a terminal and type:

```

apt-get install opera

```

(which will install the current stable release)
