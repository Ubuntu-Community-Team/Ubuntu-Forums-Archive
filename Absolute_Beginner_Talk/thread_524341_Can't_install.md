### Post by jrusso2 on 2007-08-13
You can't run two installation applications at the same time.  If you have synaptic open then you try to install something using apt, it will complain.  So close down synaptic if you using apt

### Post by sysikon on 2007-08-13
Ubuntu has System Monitor. *System>Administration>System Monitor*

Then kill synaptic.
