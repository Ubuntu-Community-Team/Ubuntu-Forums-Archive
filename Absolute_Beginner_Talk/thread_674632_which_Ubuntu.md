Gutsy is the newest version. No linux distro based on US soil can legally pre-install flash, which is pretty much the standing explanation for why any version doesn't include something media related. Gutsy will automatically mount ntfs for reading, if you install the ntfs-config tool you can set it up with write access. And all versions for years have had firefox pre-installed.

Ubuntu studio has a munch of media editing stuff pre-installed if that's what you're looking for. The big difference between the different version is the default desktop that is installed, but they all feed from the same network of software sources, so you can install anything on any base.
