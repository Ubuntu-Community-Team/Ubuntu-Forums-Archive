### Post by coggins on 2008-01-21
I'm not too familiar with DigiKam, but I think you might be misinterpreting that message.
My reading of that message is that DigiKam wants to delete the *entries* from the database.  It will not delete any actual picture *files*. 
Probably those files have been moved or renamed or deleted since the last time you used DigiKam.
It's also possible that some pictures DigiKam previously knew about have been moved to the top level of the Photo Album.  Go ahead and remove them from the database, then go to your root album in a file manager and put any image files in there into a folder.  Reopen DigiKam and all should be well.
