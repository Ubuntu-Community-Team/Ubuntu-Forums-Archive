### Post by Whiffle on 2007-10-10
What I would do is change the Monitor section to this: (found the numbers w/ google)

```

Section "Monitor"
Identifier "FPD1975W"
HorizSync 31.5 - 100.0
VertRefresh 59.0 - 75.0
Option "dpms"
EndSection

```


and the Screen section to this:

```

Section "Screen"
Identifier "Default Screen"
Device "Intel Corporation 82915G/GV/910GL Integrated Graphics Controller"
Monitor "FPD1975W"
Defaultdepth 24
SubSection "Display"
Depth 24
Modes "1440x900" 
EndSubSection
EndSection

```

All you have to do is copy the old /etc/X11/xorg.conf to /etc/X11/xorg.conf.backup (so you can get it back if this doesn't work), edit xorg.conf, and then log out, hit ctrl+alt+backspace, and hopefully it should come back.

If it doesn't, press ctl+alt+F1, log in, do:
```

sudo cp /etc/X11/xorg.conf.backup /etc/X11/xorg.conf
cp /var/log/Xorg.0.log ~/Xorg.0.log
sudo /etc/init.d/gdm restart (or kdm if you're using that)

```

and it will be back to normal.  After that post the Xorg.0.log file I had you copy...
