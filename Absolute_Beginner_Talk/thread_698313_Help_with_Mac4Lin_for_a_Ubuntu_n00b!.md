### Post by Dennis on 2008-02-16
~ that is a tilde which denotes your home directory and the . in front of .purple means it´s a hidden directory.
In your file manager goto your Home directory and in the View menu click show hidden files and if .purple is not there then come back and someone will show you how to create it, I´m sure. ;-)
