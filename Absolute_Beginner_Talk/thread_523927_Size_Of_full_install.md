### Post by ugm6hr on 2007-08-12
> **JasonBourneLinux said:**
> is xubuntu a slimmed down version?

Sort of.... It has the same Ubuntu base, but uses XFCE instead of Gnome.  

If that means nothing to you, they just have slightly different looks and the default applications are different too.  Xubuntu tends to use smaller (and lighter / faster) applications as default (which is why it takes less space on the HD), but you can easily install all the Ubuntu applications if you want them (and you have enough space!!!!! - be wary - it will *allow* you to try and install things that you don't have enough HD space to install - which will then break your system).

If you can't get something to work in Xubuntu - the majority of help here in the forums applies, but some will be slightly different.  Just mention Xubuntu whenever you ask for help - there are enough of us using it to help you out.
