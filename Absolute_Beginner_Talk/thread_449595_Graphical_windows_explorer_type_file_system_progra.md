### Post by Kvark on 2007-05-20
On top of the side panel to the left (select side panel in the view menu to show it.) there is a drop down list that you can change from places to tree.

Also if you choose show as list in the view menu you get a list in the main view where you can also expand folders like in a tree. So handy for expanding several folders and moving files between them using only one window I changed the default view to list in preferences.
