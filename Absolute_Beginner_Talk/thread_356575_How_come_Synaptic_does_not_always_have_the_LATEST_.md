### Post by astoltz on 2007-02-08
I wish I could find the "official" policy statement from Ububtu but a quick search didn't turn up anything so I'll have to paraphrase:  

Ununtu is released every six months with the latest software at the time.  After the release, it is maintained with SECURITY updates only.  New versions of software are NOT included in the official repository.  

That said, there are several ways to get the latest version without waiting 'till the next Ubuntu release.  Backports, compiling from source, un-official repositories all come to mind.
