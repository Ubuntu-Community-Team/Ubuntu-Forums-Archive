### Post by alphaniner on 2007-12-02
There is a plugin for Firefox called flashblock which will prevent flash content from loading.  It puts a blank 'place-holder' where the flash video would be, and if you want to view it, you just click the button in the middle.  You can also set it up to always allow certain sites.  I use it in Xubuntu, so I reckon it'll work in Ubuntu

[https://addons.mozilla.org/en-US/firefox/addon/433](https://addons.mozilla.org/en-US/firefox/addon/433)
[http://flashblock.mozdev.org/](http://flashblock.mozdev.org/)
