### Post by crossedout on 2007-02-15
Try ctrl+alt+d

If 3 keys is too many, go to System -> Preferences -> Keyboard Shortcuts and scroll down to the 'Hide all windows and show desktop' option.  You can highlight it and make a custom shortcut.
In particular if you want it to be a single keystroke, highlight that line and press the windows key and it will assign it.
