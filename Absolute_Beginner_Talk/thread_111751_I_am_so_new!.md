I would recommend putting Ubuntu on the internal HDD.  If you are willing to give it half, that would be more than enough room.  There would be no need to convert the other half to FAT32.  Ubuntu can read NTFS, but cannot write.  If you plan on using the big drive for storage, I would put a FAT32 partition on it so that you can access files on it from either WinXP or Ubuntu.
