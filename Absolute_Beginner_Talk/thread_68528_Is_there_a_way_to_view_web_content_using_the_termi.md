### Post by SilentCacophony on 2005-09-23
Hello. There are several programs to do that:

lynx
w3m
elinks
links
links2 <- also does graphical, and has more dependencies

The top two should already be there, the others you can apt-get. For more info on them, read the manpage (i.e. *man lynx* in a terminal.)
