### Post by Ub1476 on 2007-11-07
Well, SwiftFox is an optimized Firefox for Linux. To install codecs, java etc just do..:

```
sudo apt-get install ubuntu-restricted-extras
```

Most useful software is already installed for the common user, but there's 20.000 + more in Add/Remove and Synaptic.
