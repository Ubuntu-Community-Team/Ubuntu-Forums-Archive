### Post by xtacocorex on 2006-06-28
For adding the NTFS partition, see here: [http://www.psychocats.net/ubuntu/mountwindows.php](http://www.psychocats.net/ubuntu/mountwindows.php) (Credit: aysiu)
You may need to modify what he has a bit for it to work with the captive-ntfs, but it shouldn't be hard to grasp.

As for skype, go to System -> Preferences -> Sessions and then add it to the list of programs to start up with on, I think, the 3rd tab. (I'm currently at work using XP, so I can't confirm the tab number, but it's one of them)
