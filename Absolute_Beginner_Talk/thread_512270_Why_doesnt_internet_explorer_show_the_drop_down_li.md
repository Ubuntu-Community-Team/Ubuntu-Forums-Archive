### Post by RKCole on 2007-08-02
A Windows XP Virtual Machine will probably solve your problem with Internet Explorer.  I had a Lexmark All-In-One Printer/Scanner/Copier/Fax that would not work with Ubuntu (not the fault of Ubuntu or Linux, though.  Some hardware and peripheral manufacturers just like to leave out the Linux community.).

See...a Windows XP Virtual Machine is pretty much the same as a full Widnows XP installation, but it si restricted to certain drivers for audio and video.  The Virtual Machine can get viruses and spyware (I tested this one :) ), and ti can be corrupted.  I mean...it IS Windows after all. :D

There is a good tutorial for using VirtualBox (that's what I use here at home...well...I used to.  don't need it anymore) here:
[http://doc.gwos.org/index.php/VirtualBox](http://doc.gwos.org/index.php/VirtualBox)

Just remember, you will have to go through the full Windows XP installation in VirtualBox.  It will not wreck your hard drive or anything as ti is allocated a virtual disk image.  worst case scenario, you could use the Windows XP VM for IE-related and printing-related tasks.

Hope this helps.

Take care.
