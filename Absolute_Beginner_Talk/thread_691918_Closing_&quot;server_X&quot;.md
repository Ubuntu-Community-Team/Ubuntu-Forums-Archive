### Post by PmDematagoda on 2008-02-09
Closing the X-Server means killing your GUI so that you would have to use the CLI(Command Line Interface) until you are done with installing the Nvidia driver.

But as a suggestion, why don't you install the Nvidia driver through the Restricted Driver Manager in >System>Administration>Restricted Drivers Manager and enabling the Nvidia driver? It would be more easier and at times more reliable.
