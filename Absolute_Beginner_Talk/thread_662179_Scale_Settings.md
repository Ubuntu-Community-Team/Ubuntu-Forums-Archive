### Post by Hospadar on 2008-01-08
I'd assume you're talking about compiz?

You need to install the advanced settings manager, it's called "compizconfig-settings-manager" you can install it in synaptic, or at the command line ```
sudo apt-get install compizconfig-settings-manager
``` the go System->Preferences->Advanced Desktop Effects Settings

I'm not sure exactly where the setting to change that is, but if you browse through all the plugins i'm sure you'll eventually find what you're looking for.  there's some other cool stuff in there too.
