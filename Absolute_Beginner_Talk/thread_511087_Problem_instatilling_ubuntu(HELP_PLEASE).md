### Post by Nevis on 2007-07-27
Well, what you downloaded was likely an ISO image of the installation CD. 

1 You need to burn that to CD 
2 Configure your PC so it boots from CD
3 Restart the computer with the Ubuntu install CD inserted
4 Double click the install icon on the Ubuntu desktop
5 Follow the instructions

The specifics for the above depend on what OS you're using now, what tools are installed, what computer you have, etc.

### Post by MikeMLP on 2007-07-27
Yes, that is the first step.  Make sure that the iso you downloaded has "Desktop" in the name, as opposed to "alternate"  And unless you are running a PPC mac, make sure it also has i386 in the name as well.  The "Desktop" image is a live CD which you can use to test Ubuntu's compatibility with your hardware without having to install ANYTHING to your hard drive at all.  There is no risk unless you actually click "install."  Make sure you read the info on psychocats though.  It walks you right through the entire process.
