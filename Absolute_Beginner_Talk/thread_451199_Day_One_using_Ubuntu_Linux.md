### Post by Znupi on 2007-05-22
In my case, perhiperals work actually better :). For example, I have four memory card slots built in my PC. Windows detected them all as "Removable Drive". Ubuntu detects each one what type it is, SD/MMC, MS/MS-Pro, etc. And that's not the only thing. When I insert a card, on Windows nothing happens, and I have to double-click each of the Removable Drives to see which one is the one I'm looking for. Ubuntu senses that I inserted someting, changes the icon of that drive to signify it has been mounted, and automatically opens up nautilus so I can browse the contents of the card. Excellent!
