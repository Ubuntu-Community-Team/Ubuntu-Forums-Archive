### Post by nosrednaekim on 2007-09-03
you can get the whole KDE desktop to try out with "sudo apt-get install kubuntu-desktop" you can select between KDE and gnome when you log in.

If you want  have all of the  master of all KDE apps, get amarok, the media player. Also make sure to get libxine-extracodecs to make sure you have mp3 support for it.

Generally, any application from any desktop manager will work on another if you have all the libraries required by that app installed (which will be done automatically)

### Post by saxonjf on 2007-09-03
I have installed both (and I installed KDE using Synaptic, not using the terminal), because there are a couple of really good K applications: KTorrent and Konquerer.

You can get into synaptic from the Systems -> Admin menu, type in your password, and then search for KDE.  When you get he results, choose KDE desktop.  Install all of the packages synaptic recommends with it.  Then, just log out, and log in as you wish.

Logging into KDE or Gnome is a personal choice.  I prefer Gnome, personally, because I started with it, and I think that its GUI is simpler.  Plus, KDE reminds me too much of Windoze.

Good luck

Kubuntu = Ubuntu + *kubuntu-desktop* - *ubuntu-desktop*

Or, from [the Kubuntu FAQ](http://www.kubuntu.org/faq.php): > **What is Kubuntu?**
Kubuntu is the first Ubuntu derived distribution. Our Kubuntu CDs are made up of Ubuntu's base plus KDE. You can get exactly the same effect by installing Ubuntu and adding the KDE packages (and removing the Gnome packages) from the Ubuntu archives.
