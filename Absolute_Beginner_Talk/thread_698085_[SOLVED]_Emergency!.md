### Post by Presto123 on 2008-02-16
Okay, then, do this:

In that Edit Menus box, click on "New Menu" and type in "Accessories".
Click on the box afterwards so that it will show in the menu list. Then click on "New Item" and do this for each program that follows and name each accordingly.
```
bluetooth-analyzer
gcalctool
gucharmap
gnome-dictionary
baobab
/usr/bin/system-config-printer-applet --no-tray-icon
gnome-screenshot --interactive
gnome-terminal
gedit %U
```

Baobab is the Disk Usage Analyzer, gedit %U is Text Editor, and "/usr/bin/system-config--printer" is Manage Print Jobs.
