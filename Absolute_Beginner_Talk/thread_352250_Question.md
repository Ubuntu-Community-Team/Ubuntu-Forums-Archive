### Post by Bofur on 2007-02-03
I have XMMS installed, and when I want to play an .mp3 file, it automatically opens it up with Movie Player, so I have to right click, then choose xmms from the list. Is there a way to automatically set .mp3 files to open up with XMMS? Thanks!

---

### Post by Gerard Barberi on 2007-02-03
Right click any .mp3 file and select "properties"
Select the "open with" tab and check the circle next to XMMS.  If XMMS is not there, click add at the bottom and locate it.

All .mp3 files should now be associated with XMMS.
