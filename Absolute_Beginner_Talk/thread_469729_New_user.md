### Post by Kobalt on 2007-06-10
We need more info : which wifi card do you have ? 
If your don't know precisely, open up Applications > Accessories > Terminal. Enter the following command line : ```
lspci
```
You should see a list of your hardware in which you'll find your wifi card. Eventually, post here the corresponding line. Starting from that we'll help you set up your wireless. 

Welcome to Ubuntu :) !
