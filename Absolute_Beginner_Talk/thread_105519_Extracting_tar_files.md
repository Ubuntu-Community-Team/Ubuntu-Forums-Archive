### Post by dcraven on 2005-12-18
For tar.gz, the command is "tar xvzf whatever.tar.gz".

For tar.bz2, the command is "tar xvjf whatever.tar.bz2".

For more information on the tar command, type "man tar" in a console.

HTH,
~djc

### Post by matthew on 2005-12-18
For plain tar the command it "untar e whatever.tar"

You will probably have to install the package "untar-nonfree" to use this one.
