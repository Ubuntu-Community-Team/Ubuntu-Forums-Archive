### Post by hopstah on 2006-11-01
linux runs differently than windows.  initial load times will probably be longer on linux than on windows.  that's just kind of the way that it is.  internet explorer starts almost instantly on windows because it is integrated so heavily into the operating system.  but non-microsoft programs don't have that benefit in windows, so they load more slowly.  load times after the first time should go more quickly because of how linux handles memory, however.  what specifically do you feel is slow on your computer?

### Post by adam.tropics on 2006-11-01
If a little explanation helps. and sorry I am not providing any associated benchmarks! The way it was explained to me is that much of the presumed speed difference is merely visual. Windows apps will seem to load faster, but you will note for example if you start word, there is that sometimes brief, sometimes less so, period of time, when whilst you can see the application window, you cannot do anything with it as it's still loading. With Linux, you don't see the application window so fast, but when you do, it's ready to use. That's the theory that was explained to me anyway!
