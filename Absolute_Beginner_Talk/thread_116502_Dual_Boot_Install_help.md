### Post by jimrz on 2006-01-12
Welcome,

   Look [**[COLOR="Sienna"]here[/COLOR]**]("http://users.bigpond.net.au/hermanzone/") for an excellent guide to doing a dual boot install. You should have no problem installing ubuntu over your old win98 on your second drive, as the installer will re-format the partion that you point it at for the ubuntu install. I followed this guide a few months ago on my first ever linux install and had no problems then or since. I, also, have ubuntu on my secondary drive (xp on the master). Once you have it installed you will need to edit the GRUB bootloader to make win the first boot choice. I have left mine with ubuntu as the default os, but have seen some posts here explaining how to change it around. will take a look and see if I can find you a link to a good one.

Good luck..........and have fun with it.
