### Post by PriceChild on 2006-09-30
Gaim is installed by default. Find it at applications>internet.

Its pretty self explanatory to add an msn account there.

Amsn may be installed via the synaptic package manager or by typing this into a terminal:
```
sudo apt-get install amsn
```

### Post by aysiu on 2006-09-30
Ubuntu comes with GAIM already. 

You can also install aMSN from the repositories. Just enable extra repositories and then install it using Synaptic Package Manager.
