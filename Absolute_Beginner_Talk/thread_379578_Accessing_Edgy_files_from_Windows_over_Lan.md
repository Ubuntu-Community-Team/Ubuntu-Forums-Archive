### Post by jkeyes0 on 2007-03-09
Did you create an SMB password for your username? If not, type
```
sudo smbpasswd -a USERNAME
```
where USERNAME is your username in Ubuntu, and give it a password for you to access it from the outside.

To share folders, click System ==> Administration ==> Shared Folders, and set up a folder to share out. You will have to uncheck "Read Only" if you would like for it to be writable.

### Post by lamalex on 2007-03-09
do this and then post again
```
sudo smbpasswd -a randy
sudo smbpasswd -e randy
sudo /etc/init.d/samba restart
```

now try going in through your windows box. if it doesnt work make an ubuntu user and samba user with the same name as your windows account.
