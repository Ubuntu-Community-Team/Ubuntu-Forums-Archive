### Post by Arkenzor on 2008-02-25
Actually, icon themes and panels have nothing to do with compiz, which only manages the decorations of your windows and their behavior.

You can find lots of icon themes (well, basically themes for anything) at [www.gnome-look.org](www.gnome-look.org), with - usually - sufficient installation instructions.

As for the panel, I suppose since you're running Gnome you'll want to stick with the Gnome Panel. I don't know much about configuring it, but you should be able to find a bunch of options by simply right-clicking it.

### Post by Trail on 2008-02-25
sudo apt-get install emerald

Launch the emerald theme manager, hit 'Import', point to a .emerald file that you downloaded off the internet, and hit Ok.

And just to make sure, be warned that Emerald handles the WINDOW themes - as in window title, window buttons, transparencies and fancy stuff like that. Not desktop icons, cursors and the like.

See [www.gnome-look.org](www.gnome-look.org) or [www.kde-look.org](www.kde-look.org) etc.
