### Post by Orochi on 2007-07-18
You might want to at least back up your /home directory (if it's not on a separate partition) or any config files you've heavily edited (like if you had to mess around with your xorg.conf file a lot to get your screen to display correctly, perhaps make a backup of it before upgrading). 

From what I've seen the actual upgrade goes smoothly, but sometimes not all the programs you want to run have been upgraded for compatability (like VMware doesn't run right on Feisty, but does on Edgy) so that's really what you want to check before upgrading. If some piece of software you use frequently says it's not compatable with the new version yet, you might want to wait before upgrading.
