### Post by wolfen69 on 2007-08-05
synaptic is your package manager where you get your programs from. go to: system>administration>synaptic package manager. then you can search for whatever you need. before that, go to: system>administration>software sources and make sure you have all boxes checked. also go to [http://medibuntu.sos-sts.com/repository.php](http://medibuntu.sos-sts.com/repository.php) to add extra repos for more packages to download.
