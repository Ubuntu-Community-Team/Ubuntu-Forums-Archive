### Post by jcwmoore on 2008-03-03
most software people use object oriented programming, so you should use C++, Java, or C#, all of these languages are objected oriented, and all of these are based on C, so if you are really good with one of these, you should be able to move to either of the others without too much difficulity.  I would recommend you start with Java, because java is based on C++, and C# is based on Java, Java is right in the middle.  I actually started with Java, now I use C# at work, and C++ in a class, and I have no trouble going between any on those three languages.
