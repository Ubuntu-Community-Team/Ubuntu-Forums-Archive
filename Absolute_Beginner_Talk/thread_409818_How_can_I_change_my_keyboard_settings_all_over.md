### Post by daynah on 2007-04-16
Okay, when I go to System> Pref> Keyboards > Layouts and I have Dvorak on Top, but English as the default, Opera will type Dvorak and Terminal will type English.

BUT when Dvorak is on top AND default, both apps do dvorak. I think this has something to do with the "seperate groups" check box below.

If I could find a big list of which apps were going to act which way, maybe I could help you more, but I was just doing that by trial and error.

So if you want it all Dvorak, Dvorak on top AND checked default. Same thing for  English. If you want some one way and some another, that's when you should just fiddle and twiddle and test. :)
