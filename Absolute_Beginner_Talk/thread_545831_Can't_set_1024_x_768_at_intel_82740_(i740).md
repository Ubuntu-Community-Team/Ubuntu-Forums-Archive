### Post by overdrank on 2007-09-08
Ok I have a couple of suggestion if you would like to try. First backup your current xorg with this command in the terminal
 ```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup
```
If any thing goes wrong then you can alway restore your backup with this command
```
sudo cp /etc/X11/xorg.conf_backup /etc/X11/xorg.conf
```

In this section I would try and change the default to 16 instead of 24
Section "Screen"
Identifier "Default Screen"
Device "Intel Corporation 82740 (i740) AGP Graphics Accelerator"
Monitor "StudioWorks"
[COLOR="Red"]DefaultDepth 24[/COLOR]
SubSection "Display"
Depth 1

If this doesn't work then I would try and reconfigure your xorg with the command 
```
sudo dpkg-reconfigure -phigh xserver-xorg
```
And make sure your resolution that you want is check and you can do this with the space bar to check the resolution desired. I hope this helps and good luck!
