### Post by Jimmyfj on 2007-08-14
Is there a F12 option for boot-menu? Some newer computers have an F12 option which allows the user to choose which device to boot from first, Cd or HDD. Try hitting the F12 key on startup, then select to boot from Cd or portable device.
