[QUOTE=Nevermore]i want to use with my athlon 900 380meg ram
and something to use with my oldie P100 64meg ram..

could you explain me the difference between knoppix and ubuntu??
![/QUOTE]

Sure. Knoppix is the original Live CD that was created to make an OS run off a CD. It is thought by many to be the best Live CD and works well.

Ubuntu is a new desktop OS that is designed to be installed and run the same way you would run any other installed os. The Live CD is mostly so that people can "try out" Ubuntu before they install it.

I must tell you that both Live CDs will run slower than if you installed the OSes. The athlon would make a very nice Ubuntu machine if it was installed on there, but the other computer is so old that I only know of two things to do with it:

[http://www.damnsmalllinux.org/](http://www.damnsmalllinux.org/)

or turn it into a firewall:

[http://www.smoothwall.org/](http://www.smoothwall.org/)
