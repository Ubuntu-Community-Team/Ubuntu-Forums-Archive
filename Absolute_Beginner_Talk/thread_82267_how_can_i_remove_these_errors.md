### Post by SilentCacophony on 2005-10-26
Hi. It sounds as if it's the gnome-panel complaining about a missing icon. If you have a menu entry or launcher for amarok on the panel, try deleting the launcher if you don't want it anymore, or right-click and change the icon in properties for the launcher. If it's a menu entry, you can edit that using *smeg.*
