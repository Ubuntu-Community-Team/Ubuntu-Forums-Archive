### Post by Enverex on 2007-06-09
Try removing (or commenting out) these lines:
    HorizSync       28.0 - 51.0
    VertRefresh     43.0 - 60.0

That should help. You can normally also comment out the "Modes" lines too. These things are all auto-detected on remotely "not ancient" hardware.
