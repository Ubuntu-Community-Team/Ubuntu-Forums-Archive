### Post by yarlson on 2006-06-14
I found the solution!

I added next line:

```
Option         "HWCursor"     "Off"
```

to Device section of my /etc/X11/xorg.conf.

Finally this section looks like:

```

Section "Device"
    Identifier     "NVIDIA Corporation NV43 [GeForce 6600 GT]"
    Driver         "nvidia"
    Option         "HWCursor"     "Off"
EndSection

```

And after X restart my cursors stop flckering!
