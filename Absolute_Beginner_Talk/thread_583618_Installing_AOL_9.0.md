The best I can do in the way of instruction is this:

1. Download your programs.

2. Are they a single file or a folder? If former skip to 2a. if latter skip to 2b.

    2a. Right click and select Open With> Wine Windows Emulater

    2b. Open the folder and find Setup.exe than right click and select Open With> Wine Windows Emulater.

Hope that helps!
