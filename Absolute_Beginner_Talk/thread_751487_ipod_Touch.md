I don't think you'll get iTunes to run under wine.

Gtkpod's page says:

> There's also preliminary support for the iPhone and the iPod Touch but they must be jailbroken to work

[http://www.gtkpod.org/about.html](http://www.gtkpod.org/about.html)

So it will presumably work with gtkpod to some extent, but you'll have to "jailbreak" it -- get into the device via a hack and modify it.  I guess it would be fun if you like hacking stuff, but you risk problems with the device, so you might want to think twice about doing that:

[http://chris.pirillo.com/2007/11/19/what-is-iphone-jailbreak/](http://chris.pirillo.com/2007/11/19/what-is-iphone-jailbreak/)
