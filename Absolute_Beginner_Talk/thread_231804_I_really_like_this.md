My name is Mark Shields and i just successfully installed Ubuntu. I am running a Amd athlon xp 2800 with a pc chips sis chipset mobo . From the CD without installing the Ubuntu, my NIC worked, my ati radion x1600 pro worked, my aopen sound card worked; I was blown away. I didn't have to install any drivers even after i installed the program. The software that came with this program was great. I was able to log in and complete some spreadsheet stuff for work  *we use office 03 pro at work*, log into my messenger accounts, and surf the web, from the start. Some of the negatives have to do with Multimedia. Mozillia based browsers have always had some problems streaming music, video, and flash like say on My space. IE is worlds most compatible browser. So far this OS is a real contender against windows. The ease of installation is incredible. A friend of mine booted up the CD and his mouse didn't work so this isn't perfect but it's a real contender. I intend to get a laptop and see if can't run it with this. 

thanks for the great OS
