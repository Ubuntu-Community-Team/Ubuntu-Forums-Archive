### Post by houstonbofh on 2006-12-16
Welcome to a new world!  But your problem is very old world.  Active X is a Microsoft environment.  Wine is a fake Microsoft environment, but it is not perfect.  To make thins worse, the repositories are way behind in Wine versions.  Start by going to [www.winehq.com](www.winehq.com) and looking around.  You application might be in the database.  Also follow the steps at [http://www.winehq.com/site/download-deb](http://www.winehq.com/site/download-deb) to get the latest version.  If no luck, you will need to post what app you are trying to run for us to help your further.
