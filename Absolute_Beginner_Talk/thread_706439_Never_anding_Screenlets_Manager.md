### Post by bobbybobington on 2008-02-24
type the following in the terminal
```
screenlets-manager
```

If it starts up normally from the terminal, but still doesn't start normally from the menu, you will have to edit Screenlets Manager's shortcut in the menu. 

Look up the program under System>Preferences>Main Menu. Once you find the entry for Screenlets Manager, right click on it and select preferences. Make sure that under Command, it only says **screenlets-manager**. 

If all the above fails, try reinstalling the program.

Hope this helps you out :)
