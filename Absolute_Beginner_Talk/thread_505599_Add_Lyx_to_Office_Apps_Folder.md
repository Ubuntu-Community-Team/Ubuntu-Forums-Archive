### Post by kngunn on 2007-07-20
I *think* you're want to add this to the Applications menu under "Office", yes?  If it is another location under "Applications" the process is still the same...

System | Preferences | Main Menu

Click "Office" then "New Item"

Fill out the friendly form and you're done!
