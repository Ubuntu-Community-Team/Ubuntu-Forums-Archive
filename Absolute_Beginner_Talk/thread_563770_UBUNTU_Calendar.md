### Post by eph1973 on 2007-09-30
Ubuntu calendar is a series of Desktop wallpapers for each month (at least that's what I get when I install it).  You install it by opening the terminal and using:

```
sudo apt-get install ubuntu-calendar
```

Then you should see the different wallpapers available if you go to System>Preferences>Desktop Background

The Ubuntu calendar pictures are the naked people shot artistically  with the tan background.
