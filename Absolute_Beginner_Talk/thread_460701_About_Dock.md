### Post by luciferin on 2007-06-01
Well, if you don't mind a bit older version you could download the package in this [linked thread]("http://ubuntuforums.org/showthread.php?t=312789&highlight=kiba+dock").

Then just run the following simple command: 
```
sudo dpkg -i kiba-dock_0.1cvs20061110matt-1_i386.deb 

```

If you want something newer though you could just search the forums, I believe there are tutorials on compiling from the source, if you're brave, or an apt repo with it is listed somewhere.
