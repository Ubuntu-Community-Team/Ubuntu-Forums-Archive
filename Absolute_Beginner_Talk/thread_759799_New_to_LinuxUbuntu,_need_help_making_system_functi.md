### Post by quirks on 2008-04-19
I only know the answer to one of those questions:
> While I am here I might as well ask how I can partition a drive from command line?
The tool is called **parted** ([http://www.gnu.org/software/parted/manual/html_chapter/parted_2.html](http://www.gnu.org/software/parted/manual/html_chapter/parted_2.html)). But you don't have to use a command-line tool. You can use the graphical counterpart GParted, too. That is way more comfortable.
