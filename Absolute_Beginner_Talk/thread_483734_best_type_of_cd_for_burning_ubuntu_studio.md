### Post by ramjet_1953 on 2007-06-25
There is no reason why you couldn't burn the Ubuntu Studio iso onto a DVD recordable disk.

Most burning packages allow you to burn an iso image onto DVD.

Just make sure that it is good quality and that you burn it slowly.

Also if in doubt, you could always use a DVD/RW, which will allow you to re-burn, if there are errors.
