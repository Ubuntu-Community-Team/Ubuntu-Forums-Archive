### Post by jimcooncat on 2007-01-31
Burn live CDs of several distros (like Ubuntu, Kubuntu, Xubuntu!) and give them a spin. See which environment you like best.

Once you've decided on a flavor, take the next step -- but you don't have to commit to a dual boot install. For now, you'll run Linux in a window, and you can switch back and forth easily. You can explore other options after you fall in love.

Instead, download the free-as-in-beer VMware Player:
[http://www.vmware.com/download/player/](http://www.vmware.com/download/player/)

Download and run the Edgy Server Edition virtual machine:
[http://www.vmware.com/vmtn/appliances/directory/694](http://www.vmware.com/vmtn/appliances/directory/694)

Update your repositories:
sudo apt-get update

And load your desktop flavor:
sudo apt-get install ubuntu-desktop
or
sudo apt-get install kubuntu-desktop
or
sudo apt-get install xubuntu-desktop

### Post by angryfirelord on 2007-01-31
If your considering using Linux again, I would dual-boot it with Windows so you can transition at your own speed. 

Provided you have net access, software is distributed in things called packages. Ubuntu uses deb packages for software management. Dapper have a little over 18000 & edgy is a little over 20000 (with all the repos enabled). You can use synaptic to get them (a gui tool for apt-get).

Linux is no longer the insanely difficult mess it once was. While you don't need to know the terminal, even the basic 2 to 3 letter commands are highly useful.

GIMP is probably the best replacement to photoshop.

Windows gaming can be done under Linux, but it's still tricky to get it running. Try a couple of the free games that are out there. (bzflag)

### Post by Audimage on 2007-01-31
For your adobe programs I suggest checking VMware out. It is a virtual machine utility. Basically, you have a second computer inside of ubuntu. It functions exactly like a separate machine. I use it quite a bit, and haven't found any problems running it so far. As far as I know it will work with any OS (probably not MacOS though). However, it will not work for games, they haven't gotten graphics acelleration worked out yet.
