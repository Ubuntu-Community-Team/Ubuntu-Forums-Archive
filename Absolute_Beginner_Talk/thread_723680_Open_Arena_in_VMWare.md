### Post by ByteJuggler on 2008-03-13
Most games (particularly using 3D acceleration) will not run very well on a virtual machines, as VM's don't emulate 3D accelerators and do not pass the hosted operations through to actual hardware accelerator present in the system.  You need to run the game on the native OS, whether that is Windows or Linux.
