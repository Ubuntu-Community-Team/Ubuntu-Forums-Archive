### Post by john_c on 2005-12-02
You can use the dd comand to do this.  Basically you copy the floppy contents to a file on the harddrive, put in another floppy and copy the file to the new floppy.

See this reference on the web for further info:

[http://www.momonga-linux.org/docs/Removable-HOWTO/C/autogen-8.html](http://www.momonga-linux.org/docs/Removable-HOWTO/C/autogen-8.html)

John_c
