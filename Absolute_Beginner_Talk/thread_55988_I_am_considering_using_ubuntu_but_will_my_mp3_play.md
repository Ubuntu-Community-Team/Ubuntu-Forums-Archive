### Post by heimo on 2005-08-10
Most of the stuff you list is possible on Linux. Your computer should be able to run Ubuntu or Kubuntu just fine - a bit more memory would be good, KDE and Gnome and their applications eat a lot of memory.

About your mp3-player, I don't know. If it requires that piece of software to transfer files, you'll need something similar on Ubuntu too. Most players show up as a generic USB disk drive and "just work". I would guess your player will not work - iPod on the other hand, works (and requires software to load files).

It's possible to play all the video files on Ubuntu, but some codecs are not available from Ubuntu repositories, but it's still quite straightforward to setup. Check: [http://www.ubuntuguide.org/](http://www.ubuntuguide.org/)

Is it easy to install software? As long as it's in repositories [http://packages.ubuntu.com/](http://packages.ubuntu.com/) it's as easy as it gets. Open Synaptic (or Kynaptic), select software to install and hit Apply. Software is downloaded and installed for you with couple mouse clicks, if it requires some other software to run, those software packages will be installed too. In my honest opinion, Debian package management which is what Ubuntu uses, is best package management system in Linux world (and Windows world has nothing comparable).

Go get the Live CD and run it. It wont touch your HD and so it's safe way to check hardware support and take a glimpse of Gnome. If you're satisfied, do your search on dual booting (sorry, I can't help on this, but only say that it's definitely possible and all the tools, even resizing partitions are there) and install. You need a install CD for that, not yet possible to install from Live CD.

### Post by varunus on 2005-08-11
FAT32 is microsoft's older filesystem; its nice to use with Linux since linux can both read and write it.  So your ME partition will just be another hard drive in Linux.  NTFS is the newer filesystem; Linux can only read this one, no writing (so far).  Microsoft is being very closed-mouthed about the specs for it.  If ME uses FAT32 (which I think is correct) you'll have read/write access to all your files; no need to even copy them over if you don't want to.

KDE apps and GNOME apps will have no problem running in the other desktop environment.  The interoperability is huge these days thanks to the nice people over at freedesktop.org.  The only thing that's different is that KDE apps will follow the KDE theme and GNOME apps will follow the GNOME theme, so you should try to stick to one set or the other just for aesthetic purposes.  (Since having one app following the KDE theme and the rest following the GNOME theme is ugly.)  You should just pick your favorite Desktop Environment and only run apps from the other if you have to; if you don't have a preference, you can always get both the Ubuntu and Kubuntu liveCDs and see which one you like more.
