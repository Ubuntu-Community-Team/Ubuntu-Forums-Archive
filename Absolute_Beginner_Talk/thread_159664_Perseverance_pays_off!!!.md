[QUOTE=Reaver]I'm discovering the same thing, a little more time consuming initially, but well worth it in the end.

Out of interest, did you use any particular guides / threads for the setting up of vnc and windows file sharing?[/QUOTE]

The VNC was fairly self explanatory - it seems to be the 'built in' remote desktop. I did run the synapic packet manger (system -> administration) first, as I was concerned the version included with Ubuntu 5.1 used VNC 3 protocol, rather thatn 4 which is running on my windows systems. but after tht System -> preferences -> remote desktop entered a password and I was able to log in from work (realvnc on windows) straight away.  

The windows share was not so straight forward. Make sure Samba is installed and up to date through the package manger again and you should be able to browse other windows machines fairly easily. It was much harder getting the windows machines to see the linux files though i found ubuntuguide.org had step by step instruction though, but this was my first entrance into command line working and editing config files!

Great feeling when I got it all going tho!!!!
