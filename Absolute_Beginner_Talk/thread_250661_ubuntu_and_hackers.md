### Post by jd65pl on 2006-09-04
If you wish to check the security of your setup here are a few programs

nmap - Port scanner shows up ports which may
 have been opened by installing programs such as FTP, apache, mySQL

tiger - A unix sercurity checking prog

chkrootkit - a root kit scanner

firestarter - as previously mentioned

Aegis Virus Scanner - as described

All should be found in the repositories
