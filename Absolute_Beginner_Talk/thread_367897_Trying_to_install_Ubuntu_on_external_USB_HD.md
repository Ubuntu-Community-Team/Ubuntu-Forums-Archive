### Post by aysiu on 2007-02-23
Many people have done it with Damn Small Linux, so booting from USB (as Sony and your BIOS told you) is possible.

I don't know if it's going to work for Ubuntu, though, without doing something special. Usually, the installation wants to install Grub (the boot loader) to the Master Boot Record of the computer.
