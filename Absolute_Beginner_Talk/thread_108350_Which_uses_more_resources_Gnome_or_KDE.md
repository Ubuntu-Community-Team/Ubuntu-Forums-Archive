### Post by Estariel on 2005-12-26
There was some bechmarking tool around some time ago, but I can't remember its name...

You should get pretty trustworthy numbers if you use UT or Quake for benchmarking. Both have native Linux versions, and you can just run it in benchmarking mode...

If you want very untrustworthy numbers, you can run glxgears (just type it in a shell). A window with some opengl gears will pop up and start spinning, and in the shell you will see its fps. However this is really unreliable, but it is ok for getting a general idea as to how fast one's system is...
