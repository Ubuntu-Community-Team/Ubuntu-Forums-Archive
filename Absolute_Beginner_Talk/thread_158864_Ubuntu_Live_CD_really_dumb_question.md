### Post by Stranger678 on 2006-04-11
Burn the iso to a disc, (Make sure you burn it as an image disc, nero makes this easy, and they have a free trial) 

Insert burned disc and reboot....chances are this will launch the ubuntu live installer, all you have to do then is press enter....But if it doesn't you will need to enter your BIOS and set your primary boot device to CD-ROM and then reboot again. 


Clear as mud?

### Post by YuHoo on 2006-04-11
Though the live CD is wonderful, I'd recommend that you check for a second computer, probably really cheap one from previous years. Install on that a full version of Ubuntu if you feel like beginning to transition. If you install it as a dual boot, power to you, but you strike me as slightly computer illiterate and in that case you will want to have a computer that has access to the internet if you have something go wrong with Ubuntu. It's something that all newly install OSes have when first installed, not just linux. Live by the rule better safe than sorry, or you'll really be sorry.
