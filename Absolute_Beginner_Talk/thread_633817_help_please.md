### Post by Presto123 on 2007-12-07
I doubt that if you don't know where it is you probably don't have it installed. It is not covered by Ubuntu, so it is not on the disk.

Be sure to go into your "Restricted Drivers" under "System/Administration" and turn it the Nvidia card off.

Okay...Go into your add/remove programs under the "applications" list. First off, select 
"Installed Programs" and see if a driver might already be installed...just to be on the safe side. We need to remove this in order to not have driver conflicts. We can do this by unchecking that driver before moving on and hitting "Apply". Let it uninstall that driver.

If it is NOT there go back and click on "All Available Applications" and in the search box, type in "Nvidia". Let it pull it up and then click on it, then click apply.

Go back into your "Restricted Drivers" and enable it again...

Once you are done with this, type in the code that follows below in the terminal or make a "Launcher" on your desktop:

```
gksudo nvidia-settings
```

With that code, I would actually make sure to create a launcher on you desktop for later use. I have used it SOOOO many times on my desktop computer. 

That code brings up the actual Nvidia window that you can change everything with...

Hope that works! :)
