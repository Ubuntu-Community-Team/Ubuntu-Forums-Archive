Quick simple question, what is the terminal term for a blank space " " in a file name,?

---

### Post by dwhitney67 on 2007-09-02
If you have a file or folder with a space in its name, for instance "My Pictures", then you would enter something like the following to change directory into it.

[FONT="Courier New"]$ cd My\ Pictures[/FONT]

Notice the "\ " between the two words that make up the folder name.  I hope this answers your question.
