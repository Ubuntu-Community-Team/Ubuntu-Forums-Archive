### Post by Roosta21 on 2008-03-21
Hi this bugged me and took me a little while to fix so i thought id post a fix..

I was attempting to use my universities citrix client.. but i keep getting ***the sign you have not chosen to trust "GlobalSign Root CA" - Citrix***

so if this happens to you, google GlobalSign Root crt the link at the top should be for globalsign certificate download, if not the address is below

secure.globalsign.net/cacert/root.crt

right click and save to any location.. then copy the file to the youfolder/ICAClient/linuxx86/keystore/cacerts/

thats it! all done
