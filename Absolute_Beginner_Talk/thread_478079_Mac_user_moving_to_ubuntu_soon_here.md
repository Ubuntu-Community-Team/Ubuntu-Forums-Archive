### Post by 3rdalbum on 2007-06-19
I moved from Mac OS 9 to Ubuntu 5.10. I'd never used a command-line before, and yet I got through it pretty well!

The the OP: I think you should post the question into the Apple PPC forum, as you're more likely to get Mac-specific advice.

Unfortunately, Skype and Limewire will not work on a PowerPC Ubuntu machine, as Skype and Java have not been compiled for the PowerPC processor and they are closed-source. (Java's source h

as not been opened yet). You'll find some good open-source programs that will work with Ubuntu PowerPC - for example, Ekiga allows you to VOIP-call to Windows NetMeeting users.

In terms of hardware detection, you should be pretty right! Ubuntu supports a lot of Mac hardware. You don't say what machine you have, but the only problems you're likely to have are with 3D acceleration and dialup modems.

You can run the "Desktop CD" on your computer, which is a version of Ubuntu that doesn't require installing to your hard disk. It's good to use as a trial, so you know a bit about the system before you install it perminantly. I ran it for a couple of weeks before doing a perminant install, but as long as you've got permission from your parents you can install it as soon as you get it.

Be aware that Linux is a much more different operating system than you're used to. Mac OS 9 and Mac OS X are quite a lot like Windows, compared to how OS X is to Linux. Unfortunately, the PowerPC Linux is a second-class platform - proprietry programs are always compiled for x86 and not for PowerPC, so there's no official Flash, no Skype, and no 3D acceleration. If you run into anything you're confused with, do some research online and if you're still stumped, feel free to PM me and ask for guidance.

Good luck! Remember, you can't damage your computer or existing data just by running the Desktop CD. If you decide to install, you should take the installation process very seriously and perform it with due caution. Oh, and this may seem like an obvious thing, but make sure you download the PowerPC release, not the x86 release.
