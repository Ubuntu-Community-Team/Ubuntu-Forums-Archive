### Post by BarfBag on 2007-05-21
There is a version of Flash available for Linux.  No problems there.

Photoshop CS2 can be ran via WINE or Crossover.  From what I've heard, the speed shouldn't be much slower then it is in Windows.  A good open-source alternative is The Gimp.  It has a lot of the same features, and it's free.

Edit - it appears that I'm not as right as I thought I was about Photoshop.  It's possible to install it, but it looks like there's a lot of problems.  [http://appdb.winehq.org/appview.php?iVersionId=2631](http://appdb.winehq.org/appview.php?iVersionId=2631)  My advise is to install The Gimp in Windows (there's a version for Windows) and see if it works for you.

Don't know about Dreamweaver.  It might be able to run in Crossover.
