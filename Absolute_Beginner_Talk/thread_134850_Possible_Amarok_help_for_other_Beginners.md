### Post by Cesium on 2006-02-22
Not sure if this is will help others to run Amarok better in Gnome or if this is "common" knowledge... but I thought I would share in case it helps other beginnners.  I was having problems with Amarok crashing often, but have had no problems with crashes after turning off "watch folders for changes" and "scan folders recursively."  After setting up my music library I changed these two settings under Settings->Configure Amarok->Collection.  Maybe this is only helpful for people with large libraries.
