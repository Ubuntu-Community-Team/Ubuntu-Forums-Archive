Ah, I think I see what you mean.  If I understand you correctly, they aren't different versions of Ubuntu in the sense that you're thinking.  They're just names for the releases.  So Feisty Fawn (the newest) is Ubuntu 7.04, Edgy Eft is 6.10, Dapper Drake is 6.06, etc.  The next will be Gutsy Gibbon (Ubuntu 7.10).  The different versions of Ubuntu either use different window managers (Kubuntu uses KDE instead of GNOME, Xubuntu uses Xfce) or are for special purposes (Edubuntu).  The releases are the same in those versions, though.  I'm using Xubuntu Feisty Fawn, for example.

### Post by macogw on 2007-07-10
Yes, hardware changes are part of it.  Maybe you don't know how "stable" works in Linux, but a "stable" release for most distros (I know there's a few with rolling stable, I'm ignoring those) has the majority of bugs worked out and will no longer be adding anything new at all, unless it is to fix a security bug.  Dapper is getting security updates still, but no new software because new software will introduce bugs.  That means no new drivers either.  A computer with a graphics card that was released 2 monthss ago will not work on a year-old distro without a LOT of extra work using drivers which are not supported by that release, so for people who need new drivers, they need a new OS.
