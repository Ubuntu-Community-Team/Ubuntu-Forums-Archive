### Post by mocoloco on 2008-03-12
Your title is "password dir protection" so I'm guessing you want to keep a set of passwords in directory?  Or you want to password protect a directory?  Making it hidden does not provide any protection.  You may be looking to set permissions on that directory, so only your user can access it.  More info on file permissions at [https://help.ubuntu.com/community/FilePermissions]("https://help.ubuntu.com/community/FilePermissions")
