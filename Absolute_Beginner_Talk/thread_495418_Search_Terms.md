### Post by pyros on 2007-07-08
In reguards to restarting x:

*yes*, ctrl+alt+backspace will restart x, but it is hardly a clean way to do it. Think of it like using Ctrl+Alt+Delete on a dos/windows3/95/8/me box. It isn't very nice to your programs and can cause problems for you. If I am not mistaken, by default an Ubuntu install is set to restart x when you log out. So, to restart x, you can just log out and back in.
