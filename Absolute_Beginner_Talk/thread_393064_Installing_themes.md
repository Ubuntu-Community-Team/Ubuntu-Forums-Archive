### Post by ThrobbingBrain66 on 2007-03-25
if you want to change the window borders (with the close, maximixe and minimize buttons) you look under the metacity category

to change window color, widget style (buttons, etc) go with gtk2.x category.

*note: some gtk themes require extra gtk engines to be installed from the repos to work correctly.
