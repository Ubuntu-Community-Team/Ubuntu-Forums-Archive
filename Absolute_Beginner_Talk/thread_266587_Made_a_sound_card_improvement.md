### Post by Unterseeboot_234 on 2006-09-27
Hey, my Dapper install is 1 week old. I can't stand Win98SE on the other 15gig. But one thing I noticed while clicking around on my ubuntu software was my SoundBlaster Live! 5.1 audio was weak. Very puny. Herbert Gronemeyer mensch.wav had no punch.

Did a search on all ubuntuforums for SoundBlaster, found a link to ALSA -- Advanced Linux Sound Artchitecture, read a bunch of the technical stuff at their website, decided NOT to do anything as Fedora-install, Redhat, or kernel-compile. 

Glad I read the user-posts attached to the suggested driver download -- [COLOR="Red"]"if you've got a distro use your Add/Remove". [/COLOR]-- Got some ideas from those user posts on how to tweak and add .midi later using the .midi library that came with the SoundBlaster CD.

Sure enough, by using Applications | Add/Remove I found GNOME ALSA Mixer graphical control. The Installation was smart enough to add SoundBlaster features. There were other choices for ALSA. I might try them later.

I had fun, got to surf the web, I learned something and now my SoundBlaster is better. This ubuntu ... it's all pretty positive stuff.
