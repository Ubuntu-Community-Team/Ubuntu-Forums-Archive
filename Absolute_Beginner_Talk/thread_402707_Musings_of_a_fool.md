### Post by adam.tropics on 2007-04-06
Well if you can boot the live cd then, you can use gnome-partition-editor from the live cd and delete the old partition, then resize partitions as you like. Be sure you delete the right ones though! The options sound like grub is seeing windows, then all installed kernels from both ubuntu installs as well. Once you have the disks as you like that's a fairly simple clean up of /boot/grub/menu.lst but one step at a time!
