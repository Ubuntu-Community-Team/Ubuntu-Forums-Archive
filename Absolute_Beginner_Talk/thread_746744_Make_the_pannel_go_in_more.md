### Post by joshrobinson on 2008-04-05
right click applications in the top left of your screen, go to edit menu's
click system tools, then put a check next to configuration editor

now click applications, go to system tools and launch configuration editor

running ```
gconf-editor
``` launches configuration editor, but its handy to have it in your menus

now expand
apps > panel > top levels

youll see bottom and top, do whatever ones u wana change the autohide on
click bottom or top(dont expand it click the text)
on the right you will see "auto hide size"
click the number after it and change it to 0

do the same for the other panel if you want both to do it
