### Post by ugm6hr on 2008-03-28
> **NikS said:**
> is using nonetdebs.com safe??


It was set up by a community / forum member.

I can't see how sending a copy of your ubuntu packages status could be "unsafe".

nonetdebs doesn't install anything on to your computer at all.  All it does is give you links to all the official packages from Ubuntu for you to install manually.

Hence, it merely tells you how to do everythingmanually, without having to dig through dependency lists.
