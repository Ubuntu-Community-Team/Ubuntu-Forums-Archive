### Post by KhaaL on 2007-05-16
this is easily fixed by going into:
View -> toolbars -> customize

From there, click on the toolbar button (under new) and choose icons & text - or icons only depending on what you had. :)

### Post by paulvandenberg on 2007-05-16
You might need to install the appropriate OO.o theme. Look in Synaptic for OpenOffice.org packages. There are a handful of theme packages. If you select a given theme without the corresponding theme actually being installed, you will only get words where the icons should be.
