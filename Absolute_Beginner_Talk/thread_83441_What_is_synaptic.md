### Post by raublekick on 2005-10-28
In the top gnome panel, click on System -> Administration -> Synaptic Package Manager. You need to have the sudo(root) password to use it. 

If you set up your computer and you're the only user, then that password is your password.

Synaptic is a package manager. Instead of downloading programs yourself, pretty much anything you need is online under Synaptic, and it makes installing programs in Linux much easier than compiling from source.

### Post by xequence on 2005-10-28
Synaptic is a front end for apt. Basically, apt rocks, but it is text based. Synaptic is graphical.

Apt downloads and installs something from a repository (repo). For example...

Sudo apt-get install inkscape

installs inkscape. You can use synaptic to search and stuff ;)
