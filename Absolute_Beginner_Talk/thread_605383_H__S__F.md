### Post by bluepowder7 on 2007-11-07
hardware is physical parts, like drives and motherboards and ram chips

software is programs, like windows and firefox and ubuntu

firmware is special programming for a particular chip, and it sets up how that one particular chip behaves (how it handles signals coming into it and out of it).  BIOS is kind of like that.
