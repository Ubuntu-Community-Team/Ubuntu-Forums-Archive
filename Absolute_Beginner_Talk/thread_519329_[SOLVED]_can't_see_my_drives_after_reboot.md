### Post by Inxsible on 2007-08-06
open fstab again using ```
 gksudo gedit /etc/fstab
```Change the last 2 lines to be

```
/dev/sda1 /media/WinXP ntfs-3g defaults,auto,locale=en_US.UTF-8 0 0
/dev/sdb1 /media/Storage ntfs-3g defaults,auto,locale=en_US.UTF-8 0 0
```save the file and close it. then do ```
sudo mount -a
```

### Post by Wiebelhaus on 2007-08-06
You must shut down windows properly , if you don't the ntfs log will be flagged as dirty and ntfs-3g won't touch it until you either run a chkdsk or shutdown windows correctly so that the ntfs log is labeled as clean & healthy.


In other words , boot into windows then shut down properly.
