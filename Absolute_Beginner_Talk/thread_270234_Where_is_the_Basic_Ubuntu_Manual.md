### Post by aysiu on 2006-10-02
> **dolphinsonar said:**
> 
1. Where do things go after you install them?  I have installed a few programs that I cant find graphically, and have no idea how to run using the command line. If it's a graphical program, you can almost always launch it by using the name of the program. To launch Firefox, *firefox*. To launch Inkscape, *inkscape*. To launch Banshee, *banshee*. If it's a command-line program, you may want to read the manual page for that program to see how to launch it properly. *man mp3wrap*, for example.

> 
2. The basics of using the command line.  From simplist commands to most complex, so that a newbie can take her time to learn the stuff without being overwhealmed. The easiest way to learn commands is to ask people how to do tasks you want to do. Look at the commands and ask questions about parts you don't understand.

> 
3. Keyboard shortcuts. System > Preferences > Keyboard Shortcuts

You can define your own, too:
[http://doc.gwos.org/index.php/GnomeKeyboardShortcut](http://doc.gwos.org/index.php/GnomeKeyboardShortcut)
