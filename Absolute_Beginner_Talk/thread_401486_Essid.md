### Post by benfindlay on 2007-04-04
your essid, is the same as the ssid in windows, it is the name given to identify your wireless network. You can identify this by logging into the router and seeing what it is set to, or if you are able to wire yourself in, install wifi-radar in synaptic and then use that to scan for the available wireless networks. If you are using kubuntu, there is a program called kwifi manager which will do the same thing, it should be pre-installed

Hope this helps!
