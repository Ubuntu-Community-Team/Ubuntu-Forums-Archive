### Post by ugm6hr on 2007-07-14
Text editor: *mousepad*

It's in the Accessories menu (or use it from Terminal).

And to add custom actions to *thunar* (file manager):
Open Thunar (either click on Home, or Accessories->Thunar)
In the Thunar Menu: Edit->Configure custom actions
Click on the + Sign on the right, and put the location of the script in the *command* box with necessary options, and put appropriate things in the other boxes. Make sure you edit the* Appearance Conditions* tab entries too (e.g. if script searches directory - appears if contains Directory).
