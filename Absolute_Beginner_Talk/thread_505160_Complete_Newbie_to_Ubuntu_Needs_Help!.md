### Post by ramjet_1953 on 2007-07-20
Being a newcomer, I suggest that you stick to loading packages from either [COLOR="Blue"]Applications>Add/Remove[/COLOR], or using Synaptic package manager ([COLOR="Blue"]System>Administration>Synaptic Package Manager[/COLOR].)

The applications that you want are there and will install an icon for you in the menus.

You may want to enable the extra repositories, to give you a wider selection of packages. This is done in [COLOR="Blue"]Add/Remove[/COLOR] by clicking in the drop-down menu in the upper right corner and selecting [COLOR="Blue"]All Available Applications[/COLOR].

In Synaptic Package Manager, it is done by clicking on the drop down menu [COLOR="Blue"]Settings>Repositories[/COLOR] and putting a tick in every box, except Source Code.

Hope that this helps you.
