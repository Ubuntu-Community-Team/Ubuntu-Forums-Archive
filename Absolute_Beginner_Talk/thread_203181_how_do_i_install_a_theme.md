### Post by theredcross on 2006-06-24
if you open System > Preferences > Theme
then click on Theme Details and there should be three tabs, each one opens a box with selectable themes, drag and drop the tarball into the box that associates with it (i,e, icons into icons, etc)
then select it, and all should be well.
