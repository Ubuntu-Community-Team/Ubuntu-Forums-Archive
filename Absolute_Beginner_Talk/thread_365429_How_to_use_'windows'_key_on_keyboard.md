### Post by fusiachi on 2007-02-19
go to "System>Preferences>Keyboard"

under **layout options**---alt/win behavior, see that "Super is mapped to the Win-keys (default)" is ticked.  

Then, set the shortcut you'd like from "System>Preferences>Keyboard Shortcuts"
