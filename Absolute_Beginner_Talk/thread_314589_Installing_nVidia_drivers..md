### Post by igknighted on 2006-12-07
in this section:
```
Section "Device"
Identifier "NVIDIA Corporation NVIDIA Default Card"
Driver "nv"
BusID "PCI:1:0:0"
EndSection
```

change "nv" to "nvidia", then hit ctrl-alt-backspc.  X will reload to your login screen, but before it gets there you should see an NVIDIA splash screen.  If you see this it worked.  If you dont, X probably wont load, and you will get a command prompt.  type:
```
sudo nano /etc/X11/xorg.conf
```
then change "nvidia" back to "nv", then type 'startx' to get the GUI back.
