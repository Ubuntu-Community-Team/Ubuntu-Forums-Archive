### Post by shirilover on 2008-01-13
If by connect to different servers, you mean connect to different networks, it's simple.
Open a new network tab(Ctrl+t), use the command /server irc.newserver.net
If you want to connect to multiple networks at startup, then open the network list(Ctrl+s).
Select the network from the listbox on the left and click the edit button.
Place check in the box for connect to this network at starup.
You can also add specific servers for the network by clicking the Add and Edit buttons.
