### Post by aysiu on 2007-04-22
If you need WYSIWYG, try Kompozer:
[https://help.ubuntu.com/community/Kompozer](https://help.ubuntu.com/community/Kompozer)

And if you consider learning some hand-coding HTML (it's not as hard as you might think), there are a few more options:
Quanta
Screem
Bluefish

I use Gedit, which has syntax highlighting.
