### Post by taurus on 2008-03-01
Gutsy, 7.10, is the latest stable version so download the desktop CD (aka LiveCD) and check it out.  The whole thing runs entire from the CD so no need to install anything yet until you click on the install icon on the desktop.  And if you can wait until near the end of April,  the next release, 8.04, will be out.  

So for now, I would go with Gutsy to see if this is something you want to install on your machine.  Remember, it will be slow because everything is on the CD so don't expect it to run fast but once you install it on your machine, it should fly.
