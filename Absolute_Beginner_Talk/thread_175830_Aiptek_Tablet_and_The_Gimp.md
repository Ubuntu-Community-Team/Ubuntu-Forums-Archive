### Post by Herman on 2006-05-14
> I did a search for "gimp" but nothing with "grokking" comes up. A search for "Grokking" returns no results.

Hello, georgetoon,[Grokking The GIMP]("http://gimp-savvy.com/BOOK/") is a both a book and a website by Corey Bunks. If you go to the website, you can download the HTML version of the book as a .tar.gz file.
I have it and I highly recommend it, I'm still learning lots from it.

> what do I need to get my Aiptek graphics Tablet working with the Gimp? All Ubuntu standard Ubuntu installs should feature GIMP 'out of the box'. What happens when you click on 'Applications' Graphics,' GIMP Image Editor?' Is the GIMP not found there? Or are there error messages? 
Maybe try 'fix broken packages' under 'Edit' in Synaptic? Is this a new install? How much RAM does your machine have, etc?
P.S. Have you installed Ubuntu or Kubuntu? Kubuntu has 'Krita', which is like GIMP, but for KDE.
