### Post by shadow-of-sin on 2008-03-26
Yeah there are quite a few. Pidgin comes by default in Ubuntu and it can be used to chat in MSN however it doesn't support webcams. If you want to use a webcam or audio chat, use amsn, its; feature rich, but IMO its ugly and its a bit buggy at times. You can install both of these programs using Synaptic

The client I'm currently using and recommend is: emesene, which you can get from here: 
[www.emesene.org/](www.emesene.org/)

If you would like 2 chat in msn, you could use Pidgin Internet Messenger. To open this, go to Application>Internet>Pidgin Internet Messenger.

Then, you create your account using wizard shown on the screen. In fact, you could add more IM account ie:Yahoo Messenger, GTalk, ICQ,etc..
