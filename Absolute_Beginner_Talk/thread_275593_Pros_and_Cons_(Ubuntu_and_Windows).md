### Post by aysiu on 2006-10-11
Since my software needs don't involve obscure programs found through Google searches, **software installation and upgrades** are far easier for me in Ubuntu.

**Customization of the interface** is easier for me in Ubuntu. Yes, I know there are free things in Windows like LiteStep, but I can't figure those out.

**Security** is better implemented in Ubuntu. When I don't run as administrator all the time in Windows, my OS experience is crippled. The *Run as* command can't be run on any of the Control Panel applications, Explorer, or Windows Update. In Ubuntu, I can run as limited user most of the time and easily temporarily become administrator for administrative tasks.

Ubuntu gives me **freedom** from complicated and restricting license agreements, activation keys, and trial periods.

Ubuntu has **better documentation** than Windows has--at least that I've been able to find.
