### Post by Third Thoughts on 2006-08-05
I second the recommendation to move away from Ubuntu for your media server. Ubuntu is designed to be a fully featured desktop, you'll want something slimmer. DSL is a nice distro, but you may also want to check out dyne:bolic, which is a liveCD designed for multimedia playback and editing. You can dock the CD on your hard drive, which is easier than a full installation. I don't know if it comes with out of the box DVD support though, and since it was designed primarily as a Live CD, I'm not sure if it has any sort of package manager, which would make installing new things difficult. If it does have DVD support, and a decent manager though, it would be a good distro to use.
