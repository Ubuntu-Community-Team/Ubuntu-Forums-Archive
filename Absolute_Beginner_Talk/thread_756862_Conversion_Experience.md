### Post by stands2reason on 2008-04-16
Just put Ubuntu 7.10 on my laptop. Originally it had Windows XP, with so much bloatware that it was even slower than my 256MB Celeron laptop. I wiped the disk, but when I tried to put XP on it, it woudn't work. Turns out that XP doesn't support booting from new SATA drives because of the BIOS, so I gave up and put Ubuntu on it.

At first, I wasn't too happy about this. My last attempt at using Linux was Fedora Core 5 a couple of years ago. It seems like everything had to be done through the terminal, and there were no drivers. I couldn't even get wireless internet.

But Ubuntu was a surprisingly pleasant experience. Synaptic is easy to use for installing everything. I've already got my Thunderbird profile up and running, and I'm working on getting autoplaylists set up for my MP3 player.
