### Post by aysiu on 2007-04-10
I would take it a lot more slowly than that.

**Which version to download?**
First of all, I think you have a 64-bit processor. Is that correct? Ubuntu has a 64-bit edition, but [a lot of 64-bit users still install and use the 32-bit version](http://ubuntuforums.org/showthread.php?t=403064) for better functionality (some programs are simply not available for 64-bit). That's your choice, though. Performance v. functionality. More info here:
[url=http://ubuntuforums.org/showthread.php?t=368607]Please Read First Advantages and Disadvantages of 64bit.
[/url]

**Downloading and Burning the CD**
Use this guide to download and burn the CD:
[http://www.psychocats.net/ubuntu/iso](http://www.psychocats.net/ubuntu/iso)

**What does the CD do?**
If you download the Desktop CD (as opposed to the Alternate CD), you will have a Ubuntu CD that allows you to run a "live" session off the CD and your computer's RAM without affecting your hard drive. This is a safe way to test out Ubuntu and see if it's something you really want. The "live" session is a fully functional Ubuntu operating system. The only differences between that and an actually installed Ubuntu are it running slightly more slowly than a regular installation, and none of the changes you made saving once you reboot.

**Before committing to an installation...**
I would play around with the Desktop CD for a couple of weeks--get really familiar with using Ubuntu, seeing what hardware detection problems you might have to deal with after an installation.

The other reason to wait a couple of weeks to install--Feisty Fawn (7.04) will be released by then, and you can install the latest version of Ubuntu, instead of the version that's six months old (Edgy Eft, 6.10). That'll involve downloading another ISO and burning it, though.

**Setting up a dual-boot**
After you're sure you want to set up a dual-boot, this guide should help you do that:
[http://www.psychocats.net/ubuntu/installing](http://www.psychocats.net/ubuntu/installing)

You may also want to read this to figure out the best partitioning scheme for your computer:
[http://www.psychocats.net/ubuntu/partitioning](http://www.psychocats.net/ubuntu/partitioning)

### Post by darrenm on 2007-04-10
Programs from the Windows side - Its a whole separate issue that you will need to look into and be specific on which programs you want to use etc.

Files from the Windows side - You should have an icon on your desktop for windows or in the Places menu at the top?

Dont worry about the 2 entries for Ubuntu, it's obviously updated and installed a new kernel but not removed the old one. If it gets on your nerves open Synaptic package manager from System > Administration. Choose status in the bottom left hand corner then choose Installed (local or obsolete) in the top left corner. It should show something like linux-image-2.6.17-10 or something like that. Right-click it and choose to remove it.
