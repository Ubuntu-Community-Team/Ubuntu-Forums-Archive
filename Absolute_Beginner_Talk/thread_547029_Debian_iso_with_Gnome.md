### Post by shae on 2007-09-09
Debians basic installer is NOT based on kde.  Rather it is based on Gnome.  The debian-40r1-i386-xfce-CD-1.iso and debian-40r1-i386-kde-CD-1.iso disks are aparent, but gnome is contained on the debian-40r1-i386-CD-1.iso disk.  You do not really need the rest of the disks to install a working system.

I find debian more to my liking because I am in control of more by default.  This, however, may be difficult to a first time linux user.  Anyone with good command of basic linux commands and google should do fine.  Debian Stable is just that, extremely stable.  Debian testing is pretty stable and provides up-to-date packages for the most part.  Personally I use etch + backports + repo for pidgin(until pidgin hits backports) + custom kernel/latest fglrx/madwifi from testing.  

I recommend trying out the installgui feature of the debian CDs, I found it very easy to use.
