### Post by David Corrales on 2006-02-14
First, Ubuntu already comes with Eclipse enabled on it's repositories so it's just and apt-get away... Why don't you try Netbeans? I prefer it to eclipse because of the amazing GUI builder.
Ubuntu also comes with the open source GnuJava which is *somehow* like the standard 1.4 JDK from Sun. If you want to get a more current JVM, I'd recommend getting the version 1.5 from Sun, converting it to a .deb package and selecting it as the default JVM for the system.
