### Post by Hendrixski on 2007-06-07
BTW ... yeah, that's an old computer alright!   You may want to run Xubuntu instead because it uses less resources.

It's very easy to switch: click applications-->accessories-->terminal then type in sudo apt-get install xubuntu-desktop then log out chose to switch session by clicking the icon on the bottom left, pick xfce out of the list, and log in.  TADA.  your comp will run a lot faster (and it may even fix that sound problem for you).
