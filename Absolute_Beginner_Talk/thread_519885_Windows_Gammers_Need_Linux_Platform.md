### Post by st33med on 2007-08-07
Well, in order to run .exe games in ubuntu, you need WINE, Cedega, or Crossover Linux. 

WINE is a free windows emulator, but some games will not run perfectly or not at all. 

Cedega and Crossover is better, but you have to pay for them. Cedega charges as a subscription and Crossover charges a one-time fee.

You could also get Windows virtual machines, the best one to choose is Bochs. But, you need a Windows disk and a good computer because virtualization takes a big hit on performance.
