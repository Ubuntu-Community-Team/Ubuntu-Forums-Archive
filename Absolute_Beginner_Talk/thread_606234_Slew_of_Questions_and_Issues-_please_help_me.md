### Post by Scunizi on 2007-11-07
Cool Stuff?   It's everywhere.  One of the best little titbits I picked up was [www.google.com/linux](www.google.com/linux).  It's the /linux on the end that helps alot.  Sourceforge.net is a great resource for programs.  Linuxquestions.org is tough to navigate but has good referance material.  

Partitions can live well together.. I don't know how you did your install but Ubuntu usually uses the Grub boot loader.  Perhaps grub is messed up.  Check out [http://users.bigpond.net.au/hermanzone/p15.htm](http://users.bigpond.net.au/hermanzone/p15.htm) for redoing grub.

As for Thinkvantage, you might find that the included Network manager will handle most everything you need for switching networks both wired and wireless.  Security like wpa and wep are a different matter.  I'm not wireless so I don't have experience there.

This forum is excellent.  Another great resource is IRC.  Install xChat, Opera, irssi or your flavor of IRC chat client and go to chat.freenode.net port 8001. Register your nick then log into one of the many Ubuntu channels.

/join #ubuntu
#kubuntu (kde oriented)
#ubuntu-effects
#ubuntu-server
#ubuntu-offtopic 

and others..

Welcome!
