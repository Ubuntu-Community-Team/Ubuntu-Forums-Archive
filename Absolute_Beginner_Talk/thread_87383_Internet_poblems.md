### Post by az on 2005-11-08
[http://packages.ubuntu.com/breezy/web/flashplugin-nonfree](http://packages.ubuntu.com/breezy/web/flashplugin-nonfree)

You need to enable the multiverse repository in synaptic (three or four clicks)  and then install the flashplugin-nonfree package.  It will fetch the latest flash player for you and install it.  Restart firefox and you are good to go.


You can install java, too this way (j2re1.4 and j2re1.4-mozilla-plugin) to enable java applets in your web browser.
