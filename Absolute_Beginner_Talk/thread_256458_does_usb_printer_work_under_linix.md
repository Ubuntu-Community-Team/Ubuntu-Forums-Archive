### Post by coffeecat on 2006-09-13
Linux (and Ubuntu) supports a large number of printers without you having to install drivers. The fact that it is a USB printer is not an issue. It is the make and model of the printer you need to check. Have a look in the printer listings at [linuxprinting.org]("http://linuxprinting.org/").
