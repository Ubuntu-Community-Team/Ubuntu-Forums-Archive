### Post by mostwanted on 2006-05-05
AAC/M4A/MP4 (all boils down to MPEG-4 Audio) uses a patented decoding algorithm. This means that it, like so many other media formats, is illegal to play without a paid license (if you live in the US, I don't know of any other country with software patents). Because of the nature of free software and because there are so damn many Americans in the world, including support for patented formats is not possible (some distros don't care about this, others, like Ubuntu, do). You can, however, enable support for these formats quite easily:

[https://wiki.ubuntu.com/RestrictedFormats](https://wiki.ubuntu.com/RestrictedFormats)

Please support free formats, start building an Ogg collection... today! :) I've done this myself just because I like using free formats, not because of any legal issues. My old collection still exists, but it's only used to populate my iPod's harddisk.
