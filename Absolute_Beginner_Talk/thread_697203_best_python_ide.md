### Post by roofer on 2008-02-14
I'm trying out DrPython, eric, and PyPE.  In addition to IDLE for just plain checking stuff...  As far as an IDE, eric holds promise, as it is a full-fledged environment.  Dr Python and PyPE don't really qualify as full-fledged development environments, though they both are quite handy for coding python.

I've heard/read good things about some of the other true development environments, but haven't tried them out.  'Course, good ol' emacs can hold it's own with vim if you just want to code.  I use DrPython on both Linux (Ubuntu and fedora 8) and Windows XP.  Oh, I also use IDLE on both Linux (I.B.I.D.) and Windows XP.

Really, try a couple and see what fits your needs.  The free, open source packages are very powerful/useful, and can be had for the right price :-)

-roofer
