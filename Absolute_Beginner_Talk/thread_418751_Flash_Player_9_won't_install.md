### Post by aysiu on 2007-04-22
A few things:

1. Your profile says you're using Kubuntu 5.10. Is that true? 5.10 doesn't have the latest Flash available. So you may have Flash 7 installed but a lot of sites are requiring Flash 9 these days.

2. It also has a different plugins directory from 6.06, 6.10, and 7.04 (all later versions than 5.10).

3. If you're using Kubuntu, you may be using Konqueror as your web browser. It's possible that the plugin is installed properly in the Firefox plugins directory but Konqueror hasn't detected it.
