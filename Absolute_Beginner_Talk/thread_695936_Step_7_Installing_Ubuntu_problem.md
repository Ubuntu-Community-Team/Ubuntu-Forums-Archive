### Post by dstew on 2008-02-13
How much memory does your Dell have? Are you trying to install using the Live CD? Make sure that computer has enough RAM to handle the Live CD. I think you need at least 256 Mb, and probably 384 Mb is better. Its funny, but the Live CD won't tell you it can't run if it runs out of memory, it just gives up. Same goes with disk space. You should have at least 5 Gb of free space.

If RAM is your problem, try installing using the Alternate Install CD. It uses a text-based installer. You get it from the same [Ubuntu Download page]("http://www.ubuntu.com/getubuntu/download"), just check the box below the Start Download button. If disk space is a problem, you can install smaller distributions. Maybe Xubuntu uses less space.
