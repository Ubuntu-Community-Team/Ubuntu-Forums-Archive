### Post by IYY on 2007-01-30
> 1. With Dapper Drake is there such thing as a waste of hardware resources? I know there is a hardware forum, but the description for that forum my question did not fall under.

I am not sure what you mean, exactly. Ubuntu tries to use as large a chunk of your system resources as possible (unlike Windows), so the more memory and CPU frequency you have, the faster it will function.

> 1. What is a ideal machine hardware wise... Just basic ideas.....

I'd say that a very good machine would include 1GB of memory, 2GHz CPU, and a 128 MB video card if you want Beryl, but probably 256 MB if you're into games as well. 

However, I run Ubuntu on 512 MB of memory, a 1.25 GHz CPU and a 128 MB video card, and it runs *very* smoothly. I can play games like Quake 3 and Unreal Tournament 2004, but not Doom III. Beryl runs perfectly. 

The very minimum for comfortable operation would be something like a high-end PII machine: 400 MHz CPU, 128 MB memory. A PIII is certainly good enough.

If you are buying a new machine, I would suggest to go 32 bit rather than 64, nVidia rather than ATI, and generally Intel rather than AMD. Do a quick google search for each component to see if it's compatible with Linux to save yourself potential frustration later on. Things to be particularly careful about are wireless network cards and video cards.
> 
2. Does Ubuntu use Xine as well?

As well as any other distribution.
