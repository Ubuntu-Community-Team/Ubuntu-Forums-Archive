### Post by az on 2006-05-05
Since there is not just one company making one software product with one deadline to meet, free software is distributed along several different schedules.

Kind of like a bunch of developers who harvest the work of other developers and make it all work together.  The individual software projets are all evolving as the days go by.  Since they all work together at the source level, but the end user only uses things at the binary (precompiled) level, you only get the new software after it has been tweaked to work together properly as well as debugged.

In Ubuntu's case, the releases happen every six months.  Most other distributions do not have a fixed release date they just release when it's time.
