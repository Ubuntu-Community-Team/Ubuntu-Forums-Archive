### Post by shanepardue on 2008-02-26
> **hyperyoung said:**
> Hello

Open synaptic from System > Administration > Synaptic Package Manager. Search for 
flashplugin-nonfree and left-click the box and mark the package for installation. Click apply 
changes and let Synaptic install this plugin. Close it out when it's done and open firefox to 
see flash working properly.
