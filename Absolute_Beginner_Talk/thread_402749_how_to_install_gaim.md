### Post by mikesmith36 on 2007-04-06
You should just click applications and then at the bottom add/remove, this gives an easy list of available applications which install with a couple of clicks.
I am a Ubuntu novice and have found this really easy, much easier (and cheaper) than Windows, and there seems to be a program for everything.
Incidentally my kids are avid MSN chatters and they prefer kopete which is another chat program which certainly covers MSN and I believe ICQ as well.
Another benefit of the add/remove option above is it can be searched and it gives you a description of what the program does.

### Post by gh0st on 2007-04-06
Yeah mikesmith36  is right, thats probably the best way to install things in Ubuntu. Just think of "Add/Remove" in Ubuntu as "Add & Remove Programs" in Windows but the add bit actually works :)

Another thing you might wanna check out if you're an Ubuntu newb is Automatix. Some people don't like it and may tell you not to use it but I found as a new user it really helped me out. It can install all kinds of software for you easily that isn't available in the standard repositories. I recommend you check it out at - [www.getautomatix.com](www.getautomatix.com)

You can install software on any Linux or Unix-like system by getting the source code and compiling it as well but this really isn't necessary for most people using Ubuntu. I find everything I need can be installed with a couple of clicks. I would even say it's easier than Windows once you get the hang of it thought no doubt others will disagree.

Welcome to Ubuntu and good luck :)

Most people use Synaptic as a package manager, you could call it the equivalent of 'Add/Remove software'  in Windows. You don't need any cd's or dvd's though, packages are available via so the so called online repositories (these are kept up to date by the ubuntu developers).

Main menu -> System -> Administration -> Synaptic Package Manager

Play around with it a little bit,
