### Post by rillip on 2007-05-02
Another consideration is if he has a restore CD.  I'm guessing not if his copy of Windows is dodgy, but installation can be tricky with the semi-hidden recovery partitions.

I would give him a live CD, say try it out, understanding it's a little slower on here than it will be installed, and see if he likes it.  You also need to understand what he's doing with it.  If he's planning to run a lot of Windows software, then the best thing is to run Windows.  You'll also have to consider whether or not he's going to want to use a lot of proprietary formats.  If so, you'll have to install them all, and don't be suprised to be getting calls "this video isn't playing" etc.

Knowing more about how he uses it would help form our opinions.
