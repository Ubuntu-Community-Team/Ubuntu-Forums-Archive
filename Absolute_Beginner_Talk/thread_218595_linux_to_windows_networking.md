### Post by moshuptrail on 2006-07-18
Ubuntu will share files with Windows XP right out of the box!
Just click on Places, then Network Servers, and keep drilling down.
It seems to "see" whatever workgroup you have defined. It WILL ask for a name and password to log in to the other machine.  (Something Windows XP won't do)
Have fun!
