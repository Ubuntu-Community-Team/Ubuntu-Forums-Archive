### Post by SlappyPappy on 2008-04-18
I was trying to play Letters from Iwo Jima, a store bought DVD and it kept locking up Totem so that I had to reboot to get the damn DVD out. I kept trying to beat Totem before it locked up the drive but no luck.

So I found out how to change the default DVD to another, at least to VLC which I like a lot and seems to be able to play almost all disks.

Go to System/Preferences/Removable Drives and Media

Then click on the multimedia tab and change the Video DVD Discs command to "wxvlc %m" without the quotes. The next time you put in a DVD VLC should start up if you have it installed.

BTW VLC was able to read Letters from Iwo Jima no problem. I don't know what Totem's problem is but it gets a brain fart with some discs.

Good luck and hope this helps someone out there.  :)
