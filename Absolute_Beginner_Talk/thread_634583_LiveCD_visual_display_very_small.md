### Post by ramjet_1953 on 2007-12-08
You haven't mentioned what video card / chipset that you have.

My guess is that with the LiveCD, you are running on a legacy driver, which won't give you native resolution.

By the way, when running the LiveCD, did you go into System>Preferences>Screen Resolution and check if your native resolution is listed?

When and if you decide to install Gutsy on your hard drive, you can sort out video driver issues after install, if you need to install a restricted driver.
