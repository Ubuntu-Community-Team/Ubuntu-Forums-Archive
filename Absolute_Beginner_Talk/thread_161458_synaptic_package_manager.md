### Post by sitedesign on 2006-04-17
In the properties of the package in synaptic it will list the installed files. Find the file that is used to start the app (normally in /usr/bin/ or something). 

Then you can use SMEG or Alacarte (Gnome menu editor) to add it to you gnome menu. 
Or just right click on a panel or desktop and create a new application launcher.

Point either of these to the app file and give it an icon to look posh then away you go.
