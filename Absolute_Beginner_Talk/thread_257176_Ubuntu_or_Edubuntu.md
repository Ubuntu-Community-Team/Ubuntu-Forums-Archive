### Post by Architeuthis on 2006-09-14
Hello,
Edubuntu is practicaly the same as ubuntu (it has the same repositories); whenever there is an update for ubuntu the same update is available for edubuntu.

Edubuntu is actually for teachers and school computers, it has a different management system, made escpecially for teachers, and other standard school-focused software then ubuntu; and it has support for a ltsp server system.

Ubuntu is for normal Desktop users, it has standard desktop software installed and a desktop user management system.

### Post by Najand on 2006-09-14
It is always possible to switch between Ubuntu, Kubuntu, Edubunut, Xubuntu. You just need to run this command in terminal:
```

sudo apt-get install <Project>-desktop

```
change project with any of: Ubuntu, Kubuntu, Edubunut, Xubuntu

### Post by Leeghoofd on 2006-09-14
Just to make things clear: edubuntu is NOT really a different distribution. It is actually more of a "theme" around Ubuntu, just like the "ubuntu Christian edition". 

You'll have full ubuntu functionality when installing edubuntu and you'll be able to switch back with great ease.
