### Post by jken146 on 2007-11-19
Go to System>Administration>Printing then add a new printer.  Select "Print into PDF File" and create a generic PDF printer from there.  You can then choose this instead of a real printer when printing in any program.  By default it saves the PDFs created in ~/PDF.
