### Post by arijarot on 2007-08-05
When about to do an administrative task in x, say use synaptic, you get a password prompt that is accompanied by a darkening of the screen. Is there a way to change he password prompt's behaviour from blackening the screen to simple putting the input box up without all those graphic changes (because they flicker and become block like)?

---

### Post by por100pre1 on 2007-08-05
I think there are some Accessibility Preferences under System with an option for that... check that out. :)

```

gnome-at-properties

```

### Post by arijarot on 2007-08-05
FYI. It's system>preferences>accessibility>assistive technology preferences; check the option **password dialog as floating window**

:)
