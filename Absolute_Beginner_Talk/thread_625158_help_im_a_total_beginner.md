### Post by PeterJS on 2007-11-27
No problem, did you get 7.10 or 7.04? The first thing you need to do is burn the image to a cd using nero or something like that. After the cd is burned just stick it in the computer and reboot, it should pop up with a menu the default option of which is to boot from the CD. Do that, it'll bot up, you can poke around a bit when the CD is loaded, that's more or less what it's going to look like when you're done. When you are ready to install there should be an icon on the desktop that will start the installer. It'll ask you a few questions about set up and off it will go, lucky for you since it's a bare machine that's going to take most of the hard part, setting up partitions, out of the equation.

### Post by thane1 on 2007-11-27
I'd say go, go, go with Ubuntu.  It's great.  However I'm just wondering...you said your windoze disk is bad.  Did you mean your hard drive is bad because of getting the virus?  Some viruses get down into the boot sector and a normal format c: won't get rid of them.  If this is the case you could do a low level format.  You may need to google the hard drive manufacturer's website for the low level format program to completely wipe the virus (and everything else) out of your HDD.  Then you can start from scratch.  This is assuming you'd really like to reload the evil empire system onto your computer.  The learning curve is somewhat high at first with linux, but its worth it in the long run.  I don't run Windoze now unless I absolutely have to.  If you're sure the drive is ok, I'd use a burner program, which will burn iso files to cd.  Make a disc copy of the Ubuntu program and then boot up with the disc in the default drive.  Cheers.
