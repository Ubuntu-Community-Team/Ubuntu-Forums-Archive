```

```
You should see the line you need to change, you can do so by editing the file with root privileges:
```
gksudo gedit /etc/fstab
```

### Post by Rocket2DMn on 2007-09-05
you need to make sure there is a directory called Windows under /media/
run
```
sudo mkdir /media/Windows
``` then ```
sudo mount -a
``` which will mount everything in your fstab file.  An icon may not appear on the desktop at this time, but after you restart it should.
