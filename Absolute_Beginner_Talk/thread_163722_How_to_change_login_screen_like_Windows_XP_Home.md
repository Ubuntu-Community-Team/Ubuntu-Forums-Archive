### Post by gingermark on 2006-04-28

Press Alt+F2 and type "kcontrol"
Now click "Appearence and Themes" and select "Splash Screen"

The "Redmond" theme seems to be the one you want. Click "Test" to make sure you are happy with it, and if so, click "Apply".

And you're done. If you want to add other splash screens that you download, click the "add" button to add them.
