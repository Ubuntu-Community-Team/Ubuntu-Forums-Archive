### Post by aysiu on 2007-01-08
Any time you mark an application for removal, Synaptic will let you know what other programs that removal affects. If one of those programs seems like something you'd like to keep, don't remove the first program.

You also can at any time restore all the default applications by reinstalling *ubuntu-desktop*.
