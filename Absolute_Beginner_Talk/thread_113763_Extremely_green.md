### Post by Madpilot on 2006-01-07
The .iso file you downloaded needs to be burnt a bit differently than a regular data CD in order to work - have a look [here]("https://wiki.ubuntu.com/BurningIsoHowto") for information, including WinXP info.

Also, when you want to run the LiveCD, you can't just pop it in while Windows is running - put it in the drive, then restart the computer with the LiveCD in your drive - it should boot from the CD instead of your harddrive, and start Ubuntu instead of Windows.

It won't affect your harddrives or install of Windows at all.

If this doesn't work (if it always just starts Windows) you probably need to adjust the boot order in your computer's BIOS, which is fiddly but not hard!
