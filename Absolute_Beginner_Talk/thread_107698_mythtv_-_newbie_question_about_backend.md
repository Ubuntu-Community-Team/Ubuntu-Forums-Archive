### Post by andrewsawyer on 2006-01-13
Hiya,

I've just installed MythTV on my system.  I have it running on a 2.6Ghz Celeron with 512Mb ram.  I've just upgraded to 1Gb, however it was running ok with the 512.  Alongside MythTV (both backend and frontend), the system is also running Apache 2, which has a blog site on it, as well as access to the Mythweb through https, and proFTPd for if I want to give my mates access to anything on my computer.

Other than that, it also streams movies (.avi) and mp3's to 3 other computers in the house (2x laptops and 1x office).  It seems to be holding its own so far, so I wouldn't say that you would have a problem having the backend on the office machine.

I hope all goes well!
