Kubuntu and Ubuntu have exactly the same sources.list.  The only difference is the packages installed.  ex: KDE vs. Gnome which are both availible in the repos.  For instance you can install a ubuntu core system and make it a kubuntu system..

### Post by mcduck on 2007-09-14
> **hyper_ch said:**
> What's the difference between the group "Dapper Drake, Edgy Eft, Feisty Fawn, Gutsy Gibbon" and the group "Ubuntu, Kubuntu, Xubuntu, Edubuntu"?

Dapper Drake, Edgy Eft etc are the code names for development versions of Ubuntu. So, Dapper Drake is Ubuntu 6.06, EdgyEft is 6.10 and so on.

The only difference between Ubuntu, Kubuntu and Xubuntu is the desktop environment and default installed applications. Behind the desktop they are all the same Ubuntu, and you can actually just add different desktops to any of them. As they are all technically the same operating system they use the same repositories.

Edubuntu uses Gnome as desktop but it's more designed for use in schools so it comes with a bunch of educational software by default and other stuff useful in school networks.
