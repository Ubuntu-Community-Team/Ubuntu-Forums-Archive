### Post by sirhalos on 2007-04-04
Try making the IP address manually sometimes when you use DHCP it will try to obtain the same IP address for whatever reason of another computer on the network. Usually when I occur this problem I set the IP addrdess manually and get it working and then switch it back to DHCP. After that point DHCP will usually always give you that same IP address again (the one you selected manually).
