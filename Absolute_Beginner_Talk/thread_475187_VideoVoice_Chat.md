### Post by ramjet_1953 on 2007-06-15
Unfortunately, Skype does not have the video function in the Linux version.

If you want equivalent to Skype, WengoPhone is roughly the same. ie video, calling landlines, etc.

However, you will need to get your friends who use Windows to download the Windows version of WengoPhone also, as it uses SIP and is not compatible with Skype.
