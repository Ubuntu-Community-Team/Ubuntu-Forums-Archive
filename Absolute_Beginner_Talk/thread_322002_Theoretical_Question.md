### Post by macogw on 2006-12-19
Also because Ubuntu's install disk is 692 mb (I think), and Windows is 2.5 gb.  There's a huge size difference.  There's a lot of backward compatibility worked into Windows.  Now back-compatibility is good, but the way MS did it was to never delete old, unused code.  There's a whole LOT of crappy old code sitting there taking up memory, and it really needs to be cleaned up, but there's just too much to wade through and clean up in any reasonable amount of time.  They haven't maintained their code much, if at all.
