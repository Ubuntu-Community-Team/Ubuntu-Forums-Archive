### Post by illuminaris on 2008-02-26
My computer is still having annoying problems from time to time where it decides to reboot in low graphics mode because it can't work in the resolution it's trying to load in, which is something ridiculously high.

I keep reconfiguring the xserver-xorg but I always get to the screen where it asks me to select all the resolutions I want to keep. The menu where it has the list of resolutions, with little bracket boxes next to them, and the ones you have selected have stars in them.

It looks like this

[*] 600x800

[*] 1024x768

[ ] 1280x1024

etc.. etc..

So, my question is, when I get to that menu, what command do I use to REMOVE the resolutions that I do not want? I've tried removing the resolutions that are too high from my xorg.conf but I really don't know what I'm doing with that and it doesn't seem to have helped. I figure it's better if I let Ubuntu fix it for me, but in order for that to happen, I need to remove it in the xserver-xorg reconfigure screen. People keep referring me to guides, but the guides don't show you what key to press in that menu to remove or add different resolutions, it just goes into a bunch of complicated jargon about config files.

### Post by illuminaris on 2008-02-26
I figured it out. Spacebar is the button to add and remove those little stars. I'll leave this up here in case anyone else runs into the same silly problem. : )
