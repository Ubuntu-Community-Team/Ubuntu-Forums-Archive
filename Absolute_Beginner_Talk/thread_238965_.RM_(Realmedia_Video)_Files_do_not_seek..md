### Post by alan yeates on 2006-08-18
Not quite sure that this is the prob, but if you bring in .rm files from elsewhere, rather than downloading them there would not be the .ram file there that launches them in realplayer. When you click on a download, this .ram file installs itself rather like a cookie, and then the .rm plays. If you want to play it again, the .ram is still there in the player, so no probs. If you want to pursue this have a look at the Helix Community website, it doesn't seem to make much difference which O/s you use. RealPlayer handles a lot of file types and I have not seen another app play .rms, but I stand to be corrected.
