```
sudo apt-get install phpmyadmin
```

---

### Post by Average Joe on 2006-09-28
You can uninstall it with:
> sudo apt-get remove phpmyadmin
or try to reinstall it with:
> sudo apt-get reinstall phpmyadmin
I'm not sure if everything will back to normal then though, but I think it is worth a try.
