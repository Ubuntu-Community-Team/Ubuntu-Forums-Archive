### Post by master_kernel on 2007-10-22
No. Unless you resize it from the beginning. Make sure you resize the XP partition from the end of it. If you resize it from the beginning, it will screw the XP boot sector, causing the XP partition to become unbootable (it's happened one too many times to me :)).
