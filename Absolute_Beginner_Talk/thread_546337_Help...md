### Post by Jimmyfj on 2007-09-08
During boot up, or power on you hit either Del, F2, F10 or any F-key the startup screen shows. Look for a section called Advanced BIOS setup and the for First Boot Device. Make sure this is set to be your cd drive.

### Post by annda on 2007-09-08
when the computer boots, for a short time you can see what key you need to hit to change the BIOS settings (every computer has one). it might be F2 or F12 or DEL or something else.

if you feel uncomfortable with the menus, check the documentation that came with your computer - it should tell you how to change the boot order (which disk you want to boot from).

by the way, you must have set your computer to boot from cd to run ubuntu...

### Post by ts51 on 2007-09-08
Get into your BIOS setup, and make sure your boot order has CD Rom (or the similar) at the top. Hit ESC and save settings. Then put your Ubuntu cd in drive and boot from CD. After you install just take out the CD. 

Also, no one answered but YOU HAVE TO TAKE OUT THE CD IN ORDER TO BOOT NORMALLY IF YOUR CD DRIVE IS SET TO BOOT FIRST.. (by the way I'm not yelling just emphasizing.... There is no highlight feature
