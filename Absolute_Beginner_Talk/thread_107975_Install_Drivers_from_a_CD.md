### Post by bscbrit on 2005-12-24
You can do an install from the install disk without requiring access to the internet.  You will not have access to all of the possible packages but you should have sufficient to give you a working system.

Burning with XP and then using the disk in ubuntu shouldn't make any difference.

It is possible that a problem during installation could result in you not being able to boot XP, but it is not too much of a risk. You will see some people asking for help here but the ones who don't have a problem (i.e. the vast majority) do not write to the forum!

Probably best if you keep your master XP CD handy just in case, but if you haven't got one I wouldn't let it stop you from installing Ubuntu.
