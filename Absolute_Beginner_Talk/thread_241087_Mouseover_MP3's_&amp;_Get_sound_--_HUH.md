### Post by BitTorrentBuddha on 2006-08-21
You should have music playing when you hover over a music file in *nautilus* not rhythmbox. If you don't, go into nautilus, select Edit -> Preferences, then under the preview tab set 'Preview sound files' to either 'Local files only' if you only want it to preview files on your hard drive, or 'Always' if you want it to preview files over a network.
