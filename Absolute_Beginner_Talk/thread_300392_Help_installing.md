### Post by ReaderRat on 2006-11-15
> **Ronfar said:**
> Ok after comparing that is acctually the one I am using to try and install Ubuntu. Should I try and use the LiveCD to try and install it?
Actually, you ***should*** be able to install from either, but that is not always the case. The LiveCD takes 256 Megs of RAM to install graphically (GUI). The Alternate CD only takes about 192 M RAM and installs in text-mode (no GUI). It uses fewer resources, so, sometimes it installs easier.
