### Post by 23meg on 2007-01-04
A virtual machine is actually preferred for security reasons in many cases, since any malfunction that happens remains within the virtual environment, as long as you don't give the virtual machine access to your critical native resources.

I'd worry more about the privacy breaching "features" of Windows if you'll be giving it access to your internet connection.
