### Post by NIT006.5 on 2007-10-24
Also if there's anything you can't find a reference for, you can just look at the launcher for the app to see what command is running. i.e. if you see it on the menu, right-click and add the launcher to the desktop, then look at the properties of the launcher icon on the desktop to see what command is being run.

Or, if you can locate the *.desktop launcher file for the app, just look at what the Exec line is set to.
