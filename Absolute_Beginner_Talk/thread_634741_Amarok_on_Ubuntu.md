### Post by kodoku on 2007-12-09
Pull up a Terminal (You can find this in Applications -> Accessories -> Terminal)

In the box that comes up, type
> sudo apt-get install amarok

When it prompts for the password, type the password you use to log in.

Hit enter

When it asks if you want to install *insert list of dependencies*, type "Y", hit enter.

Done.

You can find the launcher for Amarok in Applications
