### Post by BadSquishy on 2007-05-15
I found a website recently called [www.virustotal.com](www.virustotal.com) where you can submit a file, either through the web interface or by sending them an email with the correct subject line and the suspect file attached.  Virustotal scans the file with 32 different scanners and sends you back a report of the results.  

Sounds pretty cool if you just have a suspect file now and again.
