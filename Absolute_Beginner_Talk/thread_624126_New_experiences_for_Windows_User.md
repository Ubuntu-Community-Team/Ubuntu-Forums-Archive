### Post by JohnLM_the_Ghost on 2007-11-26
Well, I had Ubuntu 6,06 LTS (Dapper Drake, if i'm not wrong) installed a while back. A month now i guess.

Despite being a hardened Windows user and having all rumors around of Linux being full of bugs and problems, I was pretty surprised - in a good way.

------------------- You may skip this part if you dont like reading. I have a short summary of this below.--------------

First I liked is the LiveCD, what allows you to try before you use.
Then -- of course -- Linux is free and most software are Open Source.
Ubuntu's Gnome GUI was a bit strange at start but easily understood after just a few minutes.

When I started installing I was dazzled by it's easy-to-use Install procedure, and the most not Windows-like ability to listen music and Browse internet with ease during installation process.

Later I did download some Linux software but compiling source code was one thing I wasn't ready for. I thought there must be some easy way to do this, like install.exe :)
Not long I found Package Manager. And Installations was as simple as checkbox in the list. It downloaded, installed, configured automatically - no stupid questions asked.
I really like the repository system, though sometimes list is just way too long.

One more thing would be this forum. Never had this friendly and helping forum while I was using Windows. There's tutorials and HowTOs on "every corner".

Well as you would (or maybe not?) expect everything wasn't as smooth, rosy and shiny.
First off - I ran into a problem having software to use instead of my Windows'.
Not entirely solved but wasn't as bad.
After checking out [http://ubuntuforums.org/showthread.php?t=33183](http://ubuntuforums.org/showthread.php?t=33183) and installing [Wine]("http://www.winehq.org/")

Secondly - Extensive using of bash (Command Line Utility)
This one was the least bad of my problems. My experience while using Windows 3.1 and 9x ways back, where the DOS was still backbone of system, proved itself once again. Just searching internet for HowTOs and lists of available commands like [this one here]("http://www.ss64.com/bash/"). Even newbie user could copy-paste from HowTo!

Thirdly - Though all my hardware worked more or less, with and without some configuring, I cannot say I'm very happy of everything's performance.
Like my audio perhaps - My Hi-end Audigy 4 sound card with some 5.1 acoustic system did not sound the way it sounds on Windows with proper drivers and configuration.
Actually after tinkering too much with sound card's drivers and configs I managed to crash them. -- EDIT: The bug is solved, but still want to improve quality of sound. --
My Printer also doesn't work as good with Ubuntu, but thats probably lack of good Drivers for Canon printers for Linux. At start it didn't work at all, but after some digging I found decent drivers, but the print is still kinda blurry (not as much dpi as it should have)
though I noticed it is a bit better when printing vector graphics.

And Lastly - Ubuntu doesn't seem to work as fast as Windows for me. But before some of you start to discard Ubuntu as slow and others to start telling me Ubuntu is faster, note I am experienced long-term Windows user and I customize, configure, disable and delete the unnecessary to sqeeze the most out of Windows.
Generally I still think Ubuntu is faster comparing to average Windows equipped machine. It's just I don't know how to improve Ubuntu yet. Because of my high standards on speed I had to reinstall the system six time within first four days I was using it. I always managed to crash something. But I always say 'You have to crash a system a few times before you can understand how it works.' (Newbie users! Don't be afraid. If you stay away from system settings and stick to HowTOs when you cannot, you will do just fine)


------------------------- Skippers can continue from here! :D ---------------------------------

Let's summarize all this to get the big picture!

Ubuntu and Linux in general can be used by anyone (except for stubborn Windows Lovers, who don't want to learn anything new and are ready to pay money not to :lolflag: )
I can recommend Ubuntu for Newbie and Experienced users alike.

Newbies can have a free and good OS and for them it might actually be easier to switch than for a long-term Windows users.

On other hand Advanced Windows (or other OS) users can have a challenge of learning something brand new for yourselves, extend your knowledge outside the boundaries of Windows and add an additional degree of freedom by using and customizing Open Source programs to suit your needs.

There is good and bad things about Ubuntu and Linux, but so there is for any other OS and Windows definately won't an exception.
Let's make a list of those thing in Ubuntu then:
What there is good:
[LIST]
[*]LiveCD - Try first! Then Take it or leave it!
[*]Package Manager - No problem installing software anymore!
[*]Friendly Forums - They will help you if you run into trouble!
[*]It's Damn FREE! - Just like that!
[*]Loads of mostly Open Source or Free Software - Not only you can use it! You can customize and share!
[*]Severe Lack of Security Problems - YEAH!
[*]Compatibility with Windows File systems (more or less) and Networks
[*]and more...
[/LIST]

Not so good things then:
[LIST]
[*]Windows users, might have Windows software problems - Try Linux Native Software or Wine!
[*]Command line - if you don't like typing.
[*]Linux Driver Shortage - Nevertheless, all your hardware should work!
[*]Linux Native Game Shortage - If it is your priority.
[/LIST]

Well I myself still do dual-boot with Ubuntu, like many of us still do.
I'm not sure I will be able to switch to Ubuntu completely ever. At least not now. But I'm not discarding it either. Ubuntu is one of my best experiences while working with PCs.

With Dual-Booting you can pretty much overcome point one and three.
One more thing I want to mention for those migrating from Windows. Wine is not really Emulator, cause it doesn't emulate anything, It just loads Windows system Libraries (some DLLs) to run Windows software. [Read this!]("http://www.winehq.org/site/myths")

On ending tone: 
TRY UBUNTU, YOU WON'T REGRET EVEN IF YOU NEVER SWITCH TO IT!
I TRIED AND I HAVE ENJOYED IT!
