### Post by gamerchick02 on 2007-01-25
Go "View-->View Mode-->Tree View".

Then go "Settings-->Configure View Profiles".  A "Profile Management" window will pop up.  Save that as your "Kubuntu File Manager" view.

Hope that helps!
