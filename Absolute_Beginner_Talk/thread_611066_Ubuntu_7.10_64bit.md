### Post by jay1097 on 2007-11-12
I have my laptop with 64bit gusty and i find that sometimes it can be a pain to install certain apps like flash and what not. If you want an easier user experiance use 32bit, however if u want to try 64bit you could use it for a few weeks and if u decide you don't like just install the 32bit version. Also it should be noted that you will not notice a real differnace between 32bit and 64bit unless you are using more than 4gigs of ram, and if u have less than 1 gigs of ram i would strongly recommend using the 32bit version because the 64bit version requires slightly more ram. So my 2cents is don't use 64bit for the sake of using 64bit.
