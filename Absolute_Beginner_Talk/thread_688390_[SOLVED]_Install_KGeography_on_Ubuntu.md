### Post by emarkd on 2008-02-05
It's in the repositories.  Search for it in Synaptic, or if you prefer the command line:

```
sudo apt-get install kgeography
```

LOTS of things are in the repositories.  Synaptic should be your first stop whenever you want to install anything.

### Post by gunksta on 2008-02-05
Open up Synaptic and search for kgeography. Tell Synaptic to install it. If you are using Ubuntu and don't have any other KDE applications installed, this application will appear to be a rather large installation because it will also force the computer to install the basic KDE libraries. This is perfectly OK unless you have an small / mostly full hard-drive.
