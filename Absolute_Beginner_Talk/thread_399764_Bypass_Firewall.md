### Post by Sammi on 2007-04-02
The Linux kernel already has a firewall built in that by default has all the ports closed for incoming data. If you want to open up some ports then you can use Firestarter, which is in the Ubuntu repositories. You can find it in Synaptic, or install it with this command: sudo apt-get install firestarter

and then you find it in system -> administration on you top panel

Here's a quick tutorial for it: [http://www.fs-security.com/docs/tutorial.php](http://www.fs-security.com/docs/tutorial.php)
