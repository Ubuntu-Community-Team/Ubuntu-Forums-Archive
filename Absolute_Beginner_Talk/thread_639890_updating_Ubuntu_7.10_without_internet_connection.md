### Post by ruibernardo on 2007-12-13
Check this thread: [Howto: NoNetDebs - upgrade Ubuntu without Internet (or with low-bandwidth connection)]("http://ubuntuforums.org/showthread.php?t=572819")

To set up an account, look at [this post]("http://ubuntuforums.org/showpost.php?p=3742162&postcount=11"). 

You will need a file from the offline Ubuntu: /var/lib/dpkg/status (~1MB).

You also can install new packages from the repositories that aren't on the CD/DVD.

There is an upgrade link on the website. You will get many links (packages) to download if you never upgraded.

Install aptoncd to ease the installation of packages.
