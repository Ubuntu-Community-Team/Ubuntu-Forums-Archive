### Post by p_quarles on 2007-09-21
Generally speaking the only groups you would ever add a user to are those that you have manually added and defined. 

As for user privileges, in Ubuntu, the first user you create has full permission to change anything. Users created after that have very limited permissions. 

Someone here can probably give you a more specific answer if you describe what you're trying to do.
