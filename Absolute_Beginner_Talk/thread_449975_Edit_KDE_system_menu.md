### Post by swisscow on 2007-05-30
Have a look at this thread:

[HTML]http://ubuntuforums.org/showthread.php?t=325062[/HTML]

I went into that directory as sudo, made a copy of one of the .desktop files with a new name e.g.

```
sudo cp home.desktop work.desktop
```

then edited the new file using Kate to give a destination and a new name (it's at the top)

Logged out and in and Bingo!
