### Post by Dusty545 on 2006-12-13
In the upper right-hand side of the screen you should see a symbol looking like a PC monitor.  Double-click on this and you'll probably see the Name as 'lo' (loopback).

Select the text and type 'ath0' and click 'configure'.  You can then set up the preferences easily (e.g. - encryption, use DHCP, etc)
