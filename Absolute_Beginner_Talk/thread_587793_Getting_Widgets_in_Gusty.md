### Post by mcduck on 2007-10-23
Compiz doesn't have any specific widgets, but instead it allows you to define any program to be considered as a widget, hiding it by default and only showing it after a certain key has been pressed, like the OS X dashboard.

If you want some nice widget engine to use with it, I recommend Screenlets: [http://www.screenlets.org/index.php/Information]("http://www.screenlets.org/index.php/Information"). There's Ubuntu repositories from them in here: [http://hendrik.kaju.pri.ee/screenlets/?q=node/5]("http://hendrik.kaju.pri.ee/screenlets/?q=node/5")
