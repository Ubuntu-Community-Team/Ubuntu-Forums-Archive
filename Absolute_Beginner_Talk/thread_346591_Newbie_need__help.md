### Post by miseljt on 2007-01-25
When I wanted to use different themes I downloaded a bunch of new ones from the repositories using Synaptic. System>Administration>Synaptic Package Manager. Once that's loaded I searched for 'theme' and downloaded just about anything that I could find. I downloaded all of the Gnome and KDE ones. Once you get done w/ that, System>Preferences>Themes.
