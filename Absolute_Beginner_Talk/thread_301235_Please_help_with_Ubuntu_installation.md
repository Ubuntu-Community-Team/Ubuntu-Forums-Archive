### Post by Swab on 2006-11-16
Burn a CD from the file you downloaded. [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

Boot from the CD (need to make sure your BIOS is set to boot from CD).  

Launch the installer.

Follow the on screen instructions.
