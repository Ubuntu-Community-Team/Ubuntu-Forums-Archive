### Post by ncappel1 on 2006-12-31
4 gigs will work with a standard Desktop installation of Ubuntu. The reccomended minimum requirements at the [release notes]("http://www.ubuntu.com/download/releasenotes/606#head-e15a51f7ff4cac464dfd54cbed7506ef13814de3") page say 500 megs for a server install and 3 gigs harddrive space for a desktop install. 

If you need more space to maneuver Xubuntu might be an option you may want to think about. it only needs 1.4 gigs ([xubuntu release notes]("https://wiki.ubuntu.com/XubuntuDapperReleaseNotes?action=fullsearch&value=linkto%3A%22XubuntuDapperReleaseNotes%22&context=180"))

hope this was helpful!
