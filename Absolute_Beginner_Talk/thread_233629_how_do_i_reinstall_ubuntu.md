### Post by Paerez on 2006-08-10
Put the ubuntu cd in the drive.

Boot it.

Double Click "Install"

When it asks you about partitioning, say "erase entire disk" (If you have windows on the disk too, let me know and I'll help you with that).

This will wipe ubuntu clean and install it again.
