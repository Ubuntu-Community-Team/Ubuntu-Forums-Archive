### Post by ske1fr on 2006-08-04
I saw this several times when trying to install other versions of Linux.  Did you download and burn the CD yourself, or is it an official disk or a magazine disk?  Magazine disks can occasionally be faulty.

If you downloaded the CD image (.iso) then before burning it you should check that the file is a true copy by checking the MD5 sum of the download against the official sum.  There's a useful guide at [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto) that links to an OpenOffice tutorial if you've not heard of MD5 sums before.  Always check the MD5 of your download matches the official one before you burn.
