### Post by Waappu on 2007-03-15
Hi

Goto
system -> Preferences -> Emerald Theme Manager

Or type in terminal ```
emerald-theme-manager -i
```

In Emerald Theme Manager goto Emerald Settings tab and there you can change Titlebar Duble-click Action
