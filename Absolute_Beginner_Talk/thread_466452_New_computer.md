### Post by meatpan on 2007-06-06
A few things you might want to consider:

o Do you want to use the video-conferencing and audio utilities from both operating systems?  If so, try checking to see if the hardware you want to buy is compatible in both operating systems.
o Linux has support for a ton of filesystems, so see if you can successfully mount and read the filesystem of your other OS.  This has helped me quite a bit in a dual boot scenario, especially with playing music and looking at files I *forgot* on the other OS partition.
