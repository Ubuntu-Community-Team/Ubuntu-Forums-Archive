### Post by FyreBrand on 2007-02-15
It does depend on your system.  Memory seems important to Linux even more so than CPU power.  I just installed Xubuntu on a laptop at work that had Win98 on it.

Specs: PIII (around ~700Mhz), 120MB RAM, 12GB hard drive.

It works, but it's a little slow.  If you are comfortable with the terminal you could do a command line install and then install a desktop environment (xfce4, openbox, fluxbox) if a complete Xubuntu install is just too much for the system to handle.

Try doing a forum search for Xubuntu, fluxbox, openbox, IceWM.  I read through several threads when deciding what to install on that notebook.  See what worked and didn't for other people really helped me figure out what I wanted to try first.
