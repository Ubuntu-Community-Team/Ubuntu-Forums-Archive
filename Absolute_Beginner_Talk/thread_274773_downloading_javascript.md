### Post by meng on 2006-10-10
Java and javascript are different. Firefox can already interpret javascript without you needing to do anything, except possibly enable it in the Firefox options/preferences.
Java is a different kettle of fish.
In Ubuntu, you don't use "su" but "sudo".
You need to read these guides:
[https://wiki.ubuntu.com/Java](https://wiki.ubuntu.com/Java)
[https://wiki.ubuntu.com/RootSudo](https://wiki.ubuntu.com/RootSudo)
