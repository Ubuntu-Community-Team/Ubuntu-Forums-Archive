### Post by jpeddicord on 2007-02-10
This is most likely dur to the screensaver. If you have one, try turning it off from System > Prefrences > Screen Saver. Wait for your desktop to lock again (or manually lock it from System > Quit) and then see if that improves the problem.

If you are not using a screensaver, are there any open programs in the background that might be making it slow when it's locked?
