My Opinion:
The whole system is set up for security/stability purposes. One reason for password to log in is because of concerns for privacy.Not everyone is in a position where they are the only user.
 The root mode is to keep people from accidentally hosing their system. Since Linux is set up to give more control to users than conventional home systems, it opens the door to do more damage to the system. I have boken Linux systems from SOT-Linux 2003 through Xubuntu 6.X just tweaking it until it drops. It taught me a lot, but I'm sure there are less frustrating ways to learn.
 The way Ubuntu and Derivatives are set up is sweet for students of this OS, since it keeps the user from unintentionally tweaking it to death. Since I built my grandchildren's boxes, I have been looking for something that is easier for them than FC5, and I think Kubuntu is it.
