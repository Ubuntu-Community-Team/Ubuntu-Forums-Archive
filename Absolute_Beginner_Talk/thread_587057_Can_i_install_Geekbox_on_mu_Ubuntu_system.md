### Post by Maestro23 on 2007-10-22
You can download the packages you need from Ubuntu packages.  Save them to a disc or flashdrive and them install them that way on your Ubuntu box with no internet connection.

Restricted Formats: [https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs?highlight=%28formats%29%7C%28restricted%29](https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs?highlight=%28formats%29%7C%28restricted%29)

Ubuntu Packages: [http://packages.ubuntu.com/](http://packages.ubuntu.com/)

*Be advised, you are going to run into dependency issues which must resolved before you can completely install what you need.
