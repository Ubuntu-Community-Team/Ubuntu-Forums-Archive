### Post by Seisen on 2007-07-25
For themes go to System>Preferenes>Appearance and on the first tab click install. Don't untar the file keep it the way it is.

To change the login screen go to System>Adminstration>Login Window select the local tab and then click on add.  Don't untar the file keep it the way it is.
