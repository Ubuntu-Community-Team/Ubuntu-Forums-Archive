### Post by adam.tropics on 2006-07-09
in firefox preferences you don't want the advanced tab for fonts, check the 'content' tab. Also you may want to install the tahoma font (not included in ms fonts package, there is a howto but i am in a rush and don't have time right now to find it for you). But try fiddling with font choices already available, some of the linux fonts are very good. Different to windows granted, but good. Finally check system>>preferences>>fonts and play with the settings there wirh regard to the font smoothness etc.
