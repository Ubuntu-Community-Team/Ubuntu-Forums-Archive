### Post by aysiu on 2007-04-13
> **carl Yeomans said:**
> hi all. ive been thinking of getting away from MS. i  have only been using a pc since xp was new, 5 years now.
i have a few  questions about ubantu or any linux, if anyone can these id be most grateful .

1:   i use my pc mainly for my photos. can i install photoshop and all my other photo related apps  on a linux OS Probably not Photoshop, but you can use GIMP, Krita, Kolourpaint, and F-Spot to manage and edit your photos

> 2 :  will i be able to inport all my bookmarks ? If they're Firefox ones, yes. If they're Internet Explorer, you should probably install Firefox in Windows first to import the Favorites to Bookmarks.

> 3:   i use a wired network thru my router, will both my pc's still work the same as they do now ? Wired should work automatically. Wireless can sometimes take some tweaking.

> 4:  can i use peer 2 peer software ? Yes. Frostwire is very popular, and often recommended on these forums.

One piece of advice, though--take your migration *slowly*. 

Try Firefox, GIMP, and Frostwire in Windows first. Get used to using open source applications. You can find a lot of great open source Windows applications here:
[http://www.opensourcewindows.org/](http://www.opensourcewindows.org/)

Once you get used to those apps, you can try and use Ubuntu for literally weeks without installing anything to your hard drive. Get the Desktop CD. It runs off the CD itself and your computer's RAM. It can also install Ubuntu to your hard drive, but it doesn't have to until you click the *Install* icon.

Once you've used the live CD for a couple weeks and feel comfortable with Ubuntu, you can install a dual boot between Windows and Ubuntu.

Then, after a few months, you may just stop using Windows altogether. You can delete your Windows partition and use the extra space for something else... in Ubuntu.

### Post by pkarjala on 2007-04-13
If you're brand new to Linux, I recommend downloading the 6.10 CD of Ubuntu, burn it to a CD, then boot it on your current computer.  This is known as a "Live CD".  You don't have to install the OS, and the currently installed OS on your computer remains untouched.

In this way, you can mess around with some of the common programs without the risk of destroying any data on your hard drive.  The drawback is that it won't run at a very high speed, since the OS is essentially loading from your CD-ROM drive.

I also highly recommend installing it to a second computer if you have one available.  This allows you to tinker on that computer without worrying about destroying any data on your main one.  You'll be able to see how it works, what it can do, and if it's right for you.  :)
