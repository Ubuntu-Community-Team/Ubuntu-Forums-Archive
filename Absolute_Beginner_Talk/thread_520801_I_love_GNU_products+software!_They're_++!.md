Ubuntu compared to Windows and Mac OS
1.Efficient, Faster, More Features -- Believe it or not, GNU software is faster, more efficiently coded, and simultaneously is simpler and offers more advanced features – windows software is just bland
2.More Creativity -- GNU/Linux software allows – by nature – for more creativity, total creativity with OpenSource set-up
3.Total Mutability and Customization-- You can change ANYTHING from menus, to commands to the entire program layout of anything you want in ubuntu; you can't change anything but preprogrammed preferences in windows.
4.Universal File System -- If you're interested in the more universal file system, you have to learn unix/linux – it's the file system that's been around forever.
5.Universal Learning --Learning about Ubuntu means I'm learning about the core, computer universals – windows, even make, operates in its own bubble, if you just learn about windows or mac, and then try to experiment with other operating systems, you'll be in for a jolt. In other words, learning commans in ubuntu/linux means you're learning about ALL computers, learning commands in windows means,well, that you're just learning about windows!
6.Universal Purpose --EVERYTHING in ubuntu has a purpose in the file structure that you can tweak and learn about; there's a lot of “mysterious unknown” system files in windows. I don't like that
7.Cooler GUI -- Ubuntu's GNOME GUI is just so much cooler than any windows version and with gDesklets you can make it cooler than the Mac OS dock and dashboard combined.
8.Users Create and Utilize the Software – Ubuntu has a community of people intrinsically interested in making the software better and have a curiosity of how it works as well as using it; mac and windows users simply “use” the software and stand apart as creators of it.
9.Exciting – It's just more exciting to realize that your operating system has similar attributes to all the high-end server operating systems which run unix/linux, too.

This isn't David Letter men ; it's ubuntu! We don't need a top 10 list, top 9 is just fine!

So there you have it.
