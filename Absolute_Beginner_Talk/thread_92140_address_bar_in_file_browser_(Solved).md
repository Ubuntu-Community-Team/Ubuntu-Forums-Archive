### Post by darkmatter on 2005-11-18
If you want Nautilus to use the traditional location bar instead of the pathbar without using Ctrl+L, open the configuration editor.

apps -> nautilus -> preferences : check always_use_location_entry

whenever you open an instance of the filer after that...it will use the location bar instead of the pathbar
