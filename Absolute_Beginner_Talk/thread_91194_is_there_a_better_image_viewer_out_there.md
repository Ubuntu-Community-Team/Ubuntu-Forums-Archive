### Post by ow50 on 2005-11-16
[QUOTE=apjone]gthumb is great , comes as standerd[/QUOTE]
The version in the Breezy is **not** great for image browsing because it has this bug:
[Can't go to fullscreen view in 1 step when opening a folder from nautilus](http://bugzilla.gnome.org/show_bug.cgi?id=315356).

As this is the beginner forum, an easily installable alternative would be GQview (Install package "gqview" with Synaptic). You can hide all the extra stuff in the GUI and browse with PageUp and PageDown.

I'm using gThumb 2.6.8 myself and that works good.
