### Post by inoxllor on 2007-08-29
Get rid of Beryl and Install COMPIZ-FUSION!
check here: [http://ubuntuforums.org/showthread.php?t=481615](http://ubuntuforums.org/showthread.php?t=481615)

Also use the COMPIZ-FUSION tray icon
download the deb here: [http://ubuntuforums.org/showpost.php?p=3163821&postcount=8](http://ubuntuforums.org/showpost.php?p=3163821&postcount=8)

You can add the COMPIZ-FUSION TRAY ICON to the sessions, it will start automatically compiz-fusion on every login (in the Gnome XGL session.)

NOTE: I personally prefer the trevino's repositories they are always updated with the latest compiz-fusion git version. :D
