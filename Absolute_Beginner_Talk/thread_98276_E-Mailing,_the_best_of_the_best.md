### Post by greymoose58 on 2005-12-02
I've mainly used Evolution since switching to Linux so I can't comment on the other aps. The one exception would be Mozilla Thunderbird which I used for a few days while figuring out Evolution. I didn't get into the bells & whistles but it seemed quite similar to Evolution for the short time I used it. I mainly stick with Evolution for the calendar and task options.

To group my e-mails I create a filter by sender (right click on message, create rule from message, filter on sender) and send it to a folder with the senders name.  I view all of my new e-mails in a v-folder for unread messages so I don't miss any messages when they are shifted to sub-folders.

Have fun.
