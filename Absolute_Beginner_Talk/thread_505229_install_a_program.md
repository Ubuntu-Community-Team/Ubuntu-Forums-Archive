### Post by Motoxrdude on 2007-07-20
Just go to in your top right corner "Applications>>Add/Remove". Once you find an application you want to install, check the box next to it and press apply, easy as that.
If you want to install applications from a command line, open your terminal (Applications>>Accessories>>Terminal). Enter in
```
sudo apt-get install name-of-program
```
So if i wanted to install evolution from the terminal, i would type
```
sudo apt-get install evolution
```
Simple as that.

Also, there are packages called "debs" that you can install by simply double-clicking them. They are kinda like the linux version of window's EXEs.
