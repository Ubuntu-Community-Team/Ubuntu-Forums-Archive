### Post by CCNA_student on 2008-01-27
I was not able to get the function key to work with the sound, but it worked with the screen brightness. If you are using Ubuntu 7.10, Compiz-Fusion, the desktop effects you mentioned, is installed by default. Use the command *sudo aptitude install compizconfig-settings-manager* to configure compiz-fusion. If you have an ATI video card like me, you have to use the command *sudo apt-get install xserver-xgl*. You would configure the settings then by going to System>Preferences>Advanced Desktop Effects Settings.  And I believe that Evolution mail should work with Yahoo and Google mail. Godspeed.
