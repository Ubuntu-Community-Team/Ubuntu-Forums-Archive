### Post by mpettitt on 2005-10-11
You can have both installed at once (and XFCE, if you want) without any problems, and choose which to use at login (Click the "Session" button on the login screen, and choose from the list that appears). You _might_ be able to run both at once, but this tends to be a bit resource heavy, so I won't say how to do that just now (well, the fact that I'm in KDE without a "Login as New User" option also contributes).

### Post by asimon on 2005-10-12
[QUOTE=Lord Illidan]What I don't like myself is that when I arrange the icons on the desktop in KDE, they come out different in GNOME, sometimes even off the screen... I think it is because they share the same .desktop files?[/quote]
Yes, they use both the same .desktop files. But each desktop environment stores the coordinates for your desktop icons separately.

That some icons appear off-screen is clearly a bug. That should never happen.

[QUOTE=Lord Illidan]How can I arrange that?[/QUOTE]
Just arrange the icons on each desktop like you want. But you have to do it two times, once for KDE and once for Gnome.

Maybe that gets solved in the future by some new standard... but so far, both desktop store the desktop icon coordinates separately.
