### Post by Shazaam on 2008-03-13
Thats a bug in gnome-ppp that I thought was fixed. You need to install gnome-pppfixedforgutsy. 

[http://launchpadlibrarian.net/10692409/gnome-pppfixedforgutsy_0.3.24-1_i386.deb](http://launchpadlibrarian.net/10692409/gnome-pppfixedforgutsy_0.3.24-1_i386.deb)

From this launchpad post by nowshining (ubuntuforums member).....
```
 nowshining  wrote on 2007-12-02:  (permalink)

    * patched version of gutsy for 32bit computers (53.7 KiB, application/x-debian-package)

i made a I386 deb file 32-bit computer as that is what i have :) and learned what and how to do it..


download it, remove old one, don't worry all info. will/should remain when u install this version..

install it, :) and it's found under in synaptic gnome-pppfixedforgutsy cause if u do NOT remove the gnome-ppp it will show that old one and this one..
```

We will deal with the server problem once you get gnome-ppp running correctly.
