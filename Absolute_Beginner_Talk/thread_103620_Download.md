### Post by Qrk on 2005-12-14
Try downloading using Bittorrent or wget (if you have Linux presently). These methods are not prone to quiting before the whole file gets downloaded. Also, be sure to check the md5 sum of the image. (on Linux the command is **cksum**) This gives a string of characters which you then make sure is the same as the md5 sum provided. (its a text file in the directory where Ubuntu is hosted)

Another thing that might help is buring the CD at 4x.
