### Post by OliverN on 2007-07-05
[EDIT: Man, this OS is weird. I've just added
```
Section "ServerFlags"
   Option "BlankTime" "0"
   Option "StandbyTime" "0"
   Option "SuspendTime" "0"
   Option "OffTime" "0"
EndSection
```
to my xorg.conf to solve another issue in which a screensaver can't be turned off, and upon reboot I find that this fixed my desktop issue as well!]
