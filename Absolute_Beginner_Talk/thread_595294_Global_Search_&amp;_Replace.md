### Post by michaelzap on 2007-10-28
[Geany]("http://geany.uvena.de/") currently does global search and replace in all open docs (session) and global find in files (folders and subfolders). It doesn't yet replace in files, but I expect that will be added eventually. Development seems to be active, and the latest build includes a plugin interface to add new features.
