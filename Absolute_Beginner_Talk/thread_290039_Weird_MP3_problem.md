### Post by RudolfMDLT on 2006-11-01
Hi,

There really isn't enough info to solve this right away(i think). Try converting one of the currently screwed up mp3's again to mp3(re-encode) using a different bitrate and using standard encoding, not lame and/or VBR. Upload it to your player and see if this mp3 plays. Whether it plays or not convert a mp4 file to mp3 ans see if that works. This waa we'll be able to see if your converter doesn't like converting mp3-2-ogg or if it's a genuine mp3codec issue.

Hope this helps,

Cheers.
