### Post by the_purulent on 2005-08-06
How is it that I can change which application is the default for a certain file type?
(Example - I want mplayer to launch by double clicking a movie file instead of totem)

---

### Post by johannes on 2005-08-06
Right click on the file icon, choose "Properties". Click the "Open With" tab and choose the application you want. If it's not on the list click the "Add" button.  :)
