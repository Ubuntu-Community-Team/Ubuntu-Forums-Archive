### Post by BrendanM on 2007-03-11
With very few exceptions, all Linux commands are going to work the same way on Ubuntu for PPC (Mac) as Ubuntu for x86 hardware. Try the command suggested above, and then search around these forums for threads about the G3 Mac. One nice thing about Macs is they share common hardware, so somebody else with that same model of mac has almost certainly resolved this issue before you. Good luck, and don't give up.
