### Post by shearn89 on 2007-11-02
It should be much quicker and easier than a windows install (it may not be, so don't quote me).
Grab the desktop iso (you probs want the 32bit version) from the website.
Burn at LESS THAN 4X SPEED. Otherwise you may get disc errors, which suck.
Put in cd drive, reboot. If you don't get to the startup screen, reboot and change your BIOS order.
Try it a bit, then if you like it, click the install link. Follow the wizard (he's magic).

### Post by shearn89 on 2007-11-04
As mlentik said, we're all volunteers - just other users like you. No need to be offensive. You need to know which of the partitions is the windows one you want to keep. Then, you need to use gparted to remove the others, and use the free space to create a swap partition (around 256mb, or roughly the same as your ram. Opinion varies), and the rest of the free space should go to the "/" or root partition. Format both of these as ext3, and then you can use the installer (as you were) to select them as swap and root.
