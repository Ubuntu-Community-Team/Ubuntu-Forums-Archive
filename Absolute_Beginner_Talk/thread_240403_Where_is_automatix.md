### Post by Psquared on 2006-08-20
Be aware that in Xubuntu, Automatix does not (or did not for me) install all the necessary files for Thunderbird. I had to manually install using apt-get, the following files to get Thunderbird to work properly.

mozilla-thunderbird-dev 
mozilla-thunderbird-inspector 
mozilla-thunderbird-typeaheadfind

No problem in KDE or Gnome - just Xubuntu.
