### Post by mofokuban on 2007-04-22
Here's what I usually do when I do a completely fresh install (I wipe the hard drive before):

Set up partitions through the Ubuntu installer.

Before Ubuntu installs, I take the CD out, after the partitions are created.

I then install XP, and then update it (SP2, bells and whistles)

Then I install Ubuntu on the other partitions/free space.

Works like a charm every time.

### Post by az on 2007-04-22
It's easy, ubuntu does this by default.

Install XP.  Then install Ubuntu.

See here:
[https://help.ubuntu.com/community/Installation](https://help.ubuntu.com/community/Installation)
and here:
[https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall)

You will be asked abotu resizing the windows partition.  All you have to do is select the size you want to windows partition to become and then the Ubuntu installer will create the partitions you need for ubuntu.  There is no need for you to do this yourself.
