### Post by 505 on 2007-10-24
I had the same problem. It turns out that the name of the program was changed, not Thunderbird removed. Anyway, if you have a shortcut on the panel or on the desktop, rightclick it and choose Properties. If it is in a menu, rightclick the top menu and choose 'Edit menu' first, then navigate to thunderbird.

In the properties dialog, see if the name say 'thunderbird', not 'mozilla-thunderbird'. Change it to 'thunderbird' if necessary.
