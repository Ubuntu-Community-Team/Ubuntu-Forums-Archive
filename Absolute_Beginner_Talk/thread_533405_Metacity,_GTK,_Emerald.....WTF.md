### Post by macogw on 2007-08-23
GTK controls the colors in the menus and on the backgrounds behind icons on toolbars in programs.  Icon themes are obviously the icons.  Metacity themes are the window borders if you use the default Metacity window manager.  If you use Compiz, there's GTK Window Decorator, but that's the same as Metacity themes (just makes them work with Compiz).  If you use Beryl, you can use Heliodor to let you use Metacity themes with it, or you can use Emerald themes for the transparent/shiny Beryl themes.  KWin (the window manager in KDE) also has themes, and to use them in Beryl, you use Aquamarine.

### Post by WinterWeaver on 2007-08-23
Metacity --> This is the standard gnome window manager. The default that comes with your Ubuntu Install

Emerald --> This is a Window Decorator (window borders) that only works if you have another window manager, like Compiz Fusion installed.

GTK --> Hopefully someone else will answer that knows more... but afaik this is the library to create the standard window elements like buttons, scrollbars, radio buttons etc.
