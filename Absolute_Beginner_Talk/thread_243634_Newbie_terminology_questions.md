### Post by towsonu2003 on 2006-08-25

Xorg is what enables you to have windows. without it, you would have the command prompt like in DOS. 

gtk is a toolkit to create gui (graphical user interface, whatever you see in a program's window). [http://en.wikipedia.org/wiki/Gtk](http://en.wikipedia.org/wiki/Gtk)
