### Post by plexi50 on 2006-09-02
Try the notebook with a live CD if you can to be sure about hardware. Some notebook hardware is quite unique and ALL of it might not work. I have a Dell M140 and every single thing works except the card reader which is not supported by Linux at all according to all of my research. One note, Intel graphics drivers are somewhat poor compared to their Windoze cousins. I would look for Nvidia graphics if you can (for any Linux box, notebook or desktop). A lot of notebooks seem to had the Intel 950 graphics this days, it is ok but not so great for DVD, etc.
