### Post by martinw89 on 2007-03-25
Flashgot is available as an extension to Firefox, go to the Firefox add-ons website to get it.  I don't believe D4X is in the add/remove application, but it would be in the Synaptics Package Manager.  It is also available by typing```
sudo apt-get install d4x
```Sudo means you will run the command as the super user, apt-get is the name of the package manager, install installs the package, and d4x is the name of the package you want to install.
