### Post by theoldgit on 2006-12-20
You cant just change the file ending to .doc you have to sve the file as the correct format. So if you are using Open Office you would create the document then go to File - Save As then click on file type and select the version of word you want to save as. This will apply the .doc file extension and create the correct format. Then word will open it without a problem (probably).

If you are using a different word proccessor then there will probably be something very similar.
