### Post by ugm6hr on 2008-04-03
Infrarecorder is a CD-burning app for Windows.

Ubuntu has lots of its own alternatives...

The default burner which is embedded in Nautilus file manager is good enough for what you want to do.

Just right-click on the .iso image, and select Open with CD/DVD creator.

As mentioned - k3b is a more feature-rich CD-writing program: Brasero and Gnomebaker are other options.
