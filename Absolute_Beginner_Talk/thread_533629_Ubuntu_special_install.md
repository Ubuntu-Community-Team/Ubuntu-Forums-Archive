### Post by K.Mandla on 2007-08-24
> **ritterrav said:**
> Ok, thats not bad, whats the difference in fluxubuntu from regular ubuntu? and if you install that can i install regular ubuntu later?
Fluxbuntu is has the Ubuntu underpinnings, but uses Fluxbox as the default window manager. That means it's a whole lot lighter and more efficient than regular Ubuntu, but looks just as good (some would say better).

If you install Fluxbuntu and decide later you want the full Gnome desktop, you can add that on later.

Xubuntu is similar in that it's an official Ubuntu project, but it's not much different from regular Ubuntu these days. (It started out as a lightweight version of Ubuntu, but even the developers acknowledge that it's no longer a light distro.) You could give that one a shot too if you wanted, and again you'd have the option to install the entire Ubuntu desktop later.
