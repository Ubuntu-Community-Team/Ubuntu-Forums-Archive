### Post by bulldog on 2007-04-22
You need the top button,boot or install ubuntu.
It boots the live cd ,so you can have a look if your hardware is properly recognized and working.

On the desktop is the button to install ubuntu if you're satisfied with the performance of your hardware.

You have to set the boot priority in your BIOS to boot from cd first,otherwise it will just boot from the hard disk,and that's something you don't want to do,when you plan to reinstall.
