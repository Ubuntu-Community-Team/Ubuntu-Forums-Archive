Maybe this is a silly question, but here I go:

I installed Ubuntu with Gnome as desktop environment. Now, I would like to try KDE, can I just install the 'kde' package with synaptic/apt-get? And what's the difference between the kde package and the kubuntu package?

---

### Post by matthew on 2006-01-05
You can install either the kde package or the kubuntu-desktop package. Either will give you a working KDE desktop. I believe the second one just makes brings the whole setup a bit more in line with Ubuntu standard stuff--like changing the bootsplash to say "Kubuntu" in blue instead of "Ubuntu" in brown and stuff like that. My understanding is that in terms of functionality they are pretty much identical.

---

### Post by Lord Illidan on 2006-01-05
I think that pretty much covers it. kubuntu-desktop is a meta package which installs everything kubuntu related, while KDE is a package which installs (by means of dependencies) everything KDE related. I like KDE.. you might like it too!
