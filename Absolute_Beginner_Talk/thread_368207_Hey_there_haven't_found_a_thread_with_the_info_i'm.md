### Post by tonyr1988 on 2007-02-23
First, you shouldn't have the problem of endlessly searching for drivers. Unless you have some crazy hardware, most things are supported out-of-the-box (and if they aren't, there's usually a simple, quick fix).

I would recommend wiping your computer clean of XP and installing Ubuntu. You can actually do it all in one step. When you put in the Ubuntu CD and click Install, it will have an option to "Erase entire disk and install Ubuntu." It then does it all for you. :) (Just make sure you pull anything important off of it first, of course)

Networking between Ubuntu and Windows is a very common thing to do, and there's a popular program to do so. It's called Samba, and it's pretty easy to set up (see [here]("https://help.ubuntu.com/community/SettingUpSamba")).

Most importantly, if you run into any problems, post on the forums (after searching :D). This is a great community and someone will definitely be able to help you out.

### Post by aysiu on 2007-02-23
My advice:

1. Play with the live CD (also known as the Desktop CD) for a few days--see how well it detects your hardware. The live CD (I don't know if these were around six years ago) runs a working version of Ubuntu off the CD itself and your computer's RAM. It doesn't affect your hard drive. This is a nice, safe way to try Ubuntu and see what problems you might run into later. I wouldn't run the live CD with less than 256 MB of RAM. 512 MB or more is preferable.

2. Once you're pretty sure you won't run into too many problems, you can set up a dual boot by following this tutorial:
[http://www.psychocats.net/ubuntu/installing](http://www.psychocats.net/ubuntu/installing)
