As far as I know you need a separate wireless card for each 2-way connection. So I don't think you first option is possible. However if could you find a USB device that works (check out the supported hardware site, towards the bottom of the page):  [https://help.ubuntu.com/community/WifiDocs/WirelessCardsSupported](https://help.ubuntu.com/community/WifiDocs/WirelessCardsSupported)

The speed of USB 1.1 is an issue that can only be judged subjectively!

I have a wired and a wireless connection: They won't both connect to the Internet at the same time because I only have one router and one OS running. However, a different computer is also connected to the Internet through the router and I can use one of my cards to connect to a different (local) network at the same time as being connected to the Internet. The other thing about the router is that all computers logged in can interact with each other.

So - if working through a router it is possible for a device, with a card, to connect both to the internet and other computers but the connection has to be through the router (wired or wireless) and not through another device.

I've not tried, in Ubuntu,  to use my wireless card to give Internet access to another device however, I did do just that a long time ago in the windows OS - where sharing an Internet connection (before routers were readily available) was a feature.
