### Post by bernardfrancois on 2006-01-30
Nvidia drivers will never be installed automatically. Instead, an open-source driver called *nv* is used.
If you want the nvidia drivers, you need to install the driver from the nvidia website (And carefully read the readme file, after the installation you first need to activate the driver. It will also explain how to configure twinview if you want.)

I think there's also an package on synaptic I think. If you want more information; there are definitely some howto's on this forum and on the [wiki](wiki.ubuntu.com).
