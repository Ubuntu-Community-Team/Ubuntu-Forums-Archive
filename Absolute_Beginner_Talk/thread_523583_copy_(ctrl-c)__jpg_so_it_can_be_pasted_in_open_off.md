### Post by charlesbrooking on 2007-08-12
> **ssaamon said:**
> I was wondering if there is a command or terminal-based program that would allow me to copy an image into the clip-board so that I could paste it (ctrl-v) in open office word or any other software that accepts clip-board images... that is, if I had a picture: myPicture.jpg, I could run the program then hit ctrl-v in open office and see the picture in my text document.

Try this: Click on the 'Insert' menu, then 'Picture', then 'From File' and select myPicture.jpg from there.

I haven't heard about any program to copy an item to the clipboard; using menu items like the one I just mentioned seems a better way to me. Hope this helps, and don't be afraid to click on the different menus to find new features!
