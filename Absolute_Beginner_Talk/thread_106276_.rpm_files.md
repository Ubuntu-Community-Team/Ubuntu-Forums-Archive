### Post by Perfect Storm on 2005-12-20
To do the .rpm file. Open the terminal

```

cd /where/you/saved/the/package/
sudo alien XXXXXXX.rpm
sudo dpkg -i XXXXXXXX.deb

```

XXXXXX = the name of the package.
Alien tries to convert the .rpm file to .deb so an .deb file of the program will appear.
