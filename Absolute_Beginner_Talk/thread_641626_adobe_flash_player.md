### Post by RomeReactor on 2007-12-15
Hi. You can install the Flash plugin for Firefox by visiting a page that requires it, like YouTube; you'll be prompted to install the missing plugin.

However, there's a bug with the flashplugin-nonfree package where it gets installed, but is not recognized by Firefox; [this thread]("http://ubuntuforums.org/showthread.php?t=636397") has a corrected package you can install, or you can wait until the fix is available in the repositories.
