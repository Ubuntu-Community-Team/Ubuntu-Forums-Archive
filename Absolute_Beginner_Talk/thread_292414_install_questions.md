> **hiffenscgaupher said:**
> I'm looking to change my os. But do I need Linux already to install Ubuntu?
no - ubuntu is a packaging of a distribution of linux.  Is is complete and includes automated manager(s) for install and maintenance of various components.
> Will Zone Alarm work with Ubuntu, or is there a recommended firewall for this program?
Is there alot of virus, spyware, worms, etc.that can interupt Ubuntu?
There are fewer hazards but you must protect yourself anyway.  Part of the protection consists of how ubuntu is configured on install with fewer open ports.  But every time you install something, it might require an open port and therefore you need to be prudently careful.  Also, there are folks who would like to break in and will eventually find vectors to do so.
There is a software firewall maintainer called firestarter for Zone Alarm.  If you have spare hardware, a real firewall (separate pc/router) is always more secure.  IPCop is a good example of such a system.   It is yet another version of linux, hardened and designed for firewall use.  You do not run applications on it.

Ubuntu is linux; you do not need anything else to begin but the install CD.

A free firewall can be downloaded and installed with linux.

There are no "in-the-wild" viruses that affect linux.  One of the big advantages is the lack of virus, spyware, worms, etc that can affect linux.

READ CAREFULLY!  Before you install Ubuntu read a lot in these forums.  Determine if your computer will run on the live CD; if it does it is a safe bet it will install and run properly.  There are some good guides and Wikis for installation tips and techniques.  If you have two hard drives it is easy to install Windows on one HD and Ubuntu on the second HD to dual boot, or you can repartition your single HD if you don't have two, but remember that you should back up all important files before you try to do any installation.

### Post by SomeGayDude on 2006-11-03
Think of Ubuntu as being a customized Linux.  When you install Ubuntu, you are installing Linux and other prepackaged software.  I think this is the best way to explain Ubuntu.  

ZoneAlarm and other Windows based programs will not run on Ubuntu with out a cross over program.  99% of the time their is a Linux version or a program that does the same thing.  Their is a great program called Firestarter that is a firewall.  I have a antivirus program on my machine, but never got a virus.  Viruses and worms are very rare for Linux based systems.  I don't think their is any spyware for Linux.

I would suggest downloading the LiveCD and running that first to get a feel for it.
