### Post by skymt on 2006-09-27
* Download it. [GNOME-Look](http://www.gnome-look.org/) is a good source.
* Go to your home folder.
* Select View > Show Hidden Files
* If there isn't a folder called ".themes", make one.
* Open the theme you downloaded.
* Drag-n-drop the theme folder to the ".themes" folder.

EDIT: xpod, that doesn't always work in my experience.

### Post by ferebee on 2006-09-27
You can install downloaded theme files by going to
System>Preferences>Themes and clicking Install Themes. 
Just navigate to wherever you keep your downloaded files 
and select your file.

Depending on what you downloaded (eg: gtk theme,metacity theme, window border, icon set) it might not appear in the main list, but in Theme Details.
