### Post by SpikeyB on 2007-06-07
For anyone who is interested I managed to suss it out.

I decided to install gnumeric again.  Ubuntu told me that it had to remove Xubuntu desktop (some problem with dependencies) before it could install gnumeric.

I had forgotten that I had installed xfce.  Anyway, that solved the problem.
