### Post by sep1318 on 2007-08-05
I'd either go for 7.04, or if you can wait three months, do a fresh install of 7.10 when it comes out. The differences between versions is that the updates that happen to programs within the versions are limited to bugfixes only. For example, Thunderbird 2.0.0.0 came out on the same day that 7.04 did, so the default version in Fiesty is still 1.5.0.x, whatever the latest version of 1.5 is. There are backports available, but not for everything and not of every version, so I'd always go for the latest one. I'm also an update addict, so maybe its just me.
