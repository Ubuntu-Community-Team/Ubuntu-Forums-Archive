### Post by mcduck on 2008-03-24
Yes, you can. If you have enough free disk space just run defrag in Windows safe mode, and then start the Ubuntu install CD. Ubuntu's installer will then give you option to resize the Windows partition, making some free space for Ubuntu. The installer will then install Ubuntu into that space and add a menu where you can select between Ubuntu and Windows when starting your computer.

There is also the option of using Wubi to install Ubuntu into a file on your Windows partition with installer that runs as program in Windows. This way you can try Ubuntu easily and if you decide that you don't like it you can just uninstall it like any other Windows program. The are some downsides to installing Ubuntu this way, like slightly slower disk reads and writes, but it still works very well and is  a nice way to test Ubuntu if you are unsure about making moving to Linux.
