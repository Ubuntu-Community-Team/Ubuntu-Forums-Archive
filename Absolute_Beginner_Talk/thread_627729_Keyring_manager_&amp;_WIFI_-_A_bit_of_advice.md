This might be of help to a lot of folks who are having issues with Keyring Manager not picking up their password for the WIFI network, and so always having to enter the password to logon. I found a great little app called "Wicd". Google for it. Simple install and great support for WEP/WPA 1&2 etc. More effective than nm-applet imho!
