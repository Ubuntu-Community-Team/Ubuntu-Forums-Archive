### Post by Steve H on 2007-05-02
Try typing the following in a terminal (command line)

```
winecfg
```

This will bring up the Wine Config page, you should be able to select which drivers you want to use, OSS or ALSA. OSS works better with WINE in my experience, but you might have to play around for good results.

Let me know how you get on. Good luck!
