### Post by RobZoet on 2008-01-20
SUCCEEDED IN SOLVING THE Firefox/Java WINDOW SIZE PROBLEM.
After tips from the Dutch forum.

As service for those with the same problem.

Installed version 6 of Java Virtual machine (JRE) (with applications add/remove). This did at first not help.
Than I found that old versions are not automattically removed. I disabled java in firefox (preferneces).
Witt applications add/remove and with the package manager I removed everything related to SUN Java JRE which smelled , version 6.
After enabling Java in firefox again, the problem was solved.
