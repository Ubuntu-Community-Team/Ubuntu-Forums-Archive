### Post by kellemes on 2008-01-21
Not sure if you can call this a bug but the filedialog of KeePassX restricts the opening of files to files with the .kdb extension..
The program itself doesn't care what the extension is.

The solution is to rename "file" to "file.kdb".. assuming it's a valid KeePassX-db it will be opened.

Still, I think you have a valid point and you can file a bugreport with the developers of KeePassX. Maybe in there forum?
