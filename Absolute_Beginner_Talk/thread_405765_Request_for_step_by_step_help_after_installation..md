Hi Alan,
Programs such as VLC and Thunderbird are pretty simple, there is an app called add/remove, you can check the box next to the the app you want like VLC and Ubuntu will download and install it for you! If you want something and it's not there you can try Synaptic which has a larger selection, but a bit slower because of the size. And the third way, if the app has a Ubuntu .deb file to download from the website, double clicking will install it like you would for an .MSI/.EXE in Windows and put links in the menus etc.

The post [HERE]("https://help.ubuntu.com/community/PortableDevices/iPod") can probably help you with getting your Ipod working under Ubuntu.

If you fetching files files from around a network with Windows machines you'll need SAMBA which comes as part of Ubuntu so you can use Nautilus and enter the computer name in the title bar to connect to the machine. Nothing special for you to do, the only one that might require effort is your Ipod. Hope that helps.
