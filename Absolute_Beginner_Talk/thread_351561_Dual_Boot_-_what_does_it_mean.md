### Post by CroEragon on 2007-02-02
Just to explain it in more details, one OS can be booted in same time. VMWare thing guys are mentioning is program that runs on top of existing OS and acts as virtual machine in witch second OS is running. That means you get just what you want, switching between OS-es same as programs. The first OS is still main one, it can control second one, it is one that runs on physical machine, second one is just program that acts mach9ine and runs other OS in it.
Just Google VMWare, and fire away if you have more questions.
