### Post by user1397 on 2006-05-08
There is a great application that can search all of your installed applications.  It's called Xfce 4 Appfinder - and don't be scared by the name, it's available for Ubuntu, Kubuntu, and Xubuntu.  You can download it via the repositories ( not sure if it's universe or multiverse, so just in case, follow this guide to enable multiverse repositories: [http://wiki.ubuntu.com/AddingRepositoriesHowto](http://wiki.ubuntu.com/AddingRepositoriesHowto) )
First, 
```
sudo apt-get update
``` Then,
```
sudo apt-get install xfce4-appfinder
```  It'll be in Applications->Accessories in GNOME, Kmenu->Utilities in KDE, and Xfce Menu->Utilities in XFCE.
