[wicd]("http://wicd.sourceforge.net/") is a good alternative to network manager, it can sit in the tray but if you don't have a system tray it doesn't matter, you can still use the gui. Pics: [http://wicd.sourceforge.net/screenshot.php]("http://wicd.sourceforge.net/screenshot.php")

[IMG]http://wicd.sourceforge.net/main.png[/IMG]

[IMG]http://wicd.sourceforge.net/preferences.png[/IMG]

You'll need to download the deb or use the feisty repo described on the wicd download  page, it should work fine in Gutsy. Network-manager will be uninstalled. I use wicd on 2 laptops very happily, it's fine for managing wired and wireless wep & wpa.
