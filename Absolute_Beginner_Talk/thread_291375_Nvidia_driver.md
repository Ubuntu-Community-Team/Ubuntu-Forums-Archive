### Post by dannyboy79 on 2006-11-02
what do you think is already installed? the point of finding out your kernel version is so that when you install the linux-restricted-modules you will install the correct one. The example omn the guide your following is installing linux-restricted-modules-amd64-k8 but if you kernel version is 386 or 686, then you want to install linux-restricted-modules-386 or linux-restricted-modules-686, now do you get it? Then after you have installed the restricted modules for your kernel version than you can install the nvidia driver. post your lspci -v so we can see what video card you have. that'll depend on whether you need to install the legacy driver or the current driver.
