> **Warpnow said:**
> I intend to run my pc as both a server and a workstation, with the vmware running windows for when I need it for various things.

Is one of the ubuntu distros more supported than the other?

Does one lend itself better to being BOTH a server and a workstation?

Ubuntu has corporate support and a paid development team from Canonical where as Kubuntu, just like Xubuntu, is community based/supported/developed.

If support is what you want, Ubuntu, otherwise ... download the live cds from each and see which user experience you prefer. Also, something to think about is that if you are going to be running a server and VMware (which are very resource intensive) you might want to look into Xubuntu due to its speed advantages and smaller memory footprint (just a thought).
