### Post by hyper_ch on 2007-02-01
(1) Install apps
Sure, it's like an own machine :) you can install apps from the default repositories... you can add more repositories... you can download .deb files (for Ubuntu) and install them and you can download source code and compile it and hence install it :)

(2) Well some PC games can made run on Linux... Please see [http://appdb.winehq.org/](http://appdb.winehq.org/) or if you want better gaming support have a look at CeDeGa:  [http://www.transgaming.com/index.php?module=ContentExpress&func=display&ceid=2&meid=-1](http://www.transgaming.com/index.php?module=ContentExpress&func=display&ceid=2&meid=-1)
(that's commercial, you'll have to pay per month...) or if you want to run office software have a look at CrossoverOffice: [http://www.codeweavers.com/](http://www.codeweavers.com/) (also commercial)
However the 3D does not run in VMWARE!!!!

(3) You can, in the vmserver setup, assign a physical cd/dvd-drive to vmware and it will then behave as usually :)
