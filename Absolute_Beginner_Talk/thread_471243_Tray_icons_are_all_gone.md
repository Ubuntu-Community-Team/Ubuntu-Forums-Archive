### Post by odiseo77 on 2007-06-11
Are you using KDE or Gnome? if you're on Gnome, simply right click on an empty space of the upper bar, where the trayicons should be, then click on 'Add to panel' then got o 'Utilities' on the next window and select 'Notification area' (or something like that, my gnome is in spanish, so the terms might not be exactly the same). If you're on KDE there should be a similar procedure (I don't remember right now, but I can switch to KDE if it's necessary).
