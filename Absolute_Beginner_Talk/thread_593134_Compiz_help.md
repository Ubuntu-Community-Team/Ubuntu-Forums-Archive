### Post by amaroKer on 2007-10-26
Go to Applications > Add/Remove Programs...

Search for compiz

Ensure "All available applications" is selected at the top

"Advanced Desktop Effects Settings (ccsm)" should appear.  Checkmark it and apply changes.  You can find that tool in your system menu.  

I think you can also get to it by doing what you did before to enable it, but there is a new option it adds to the menu.

To enable the cube,  "Cube", "Rotate Cube", and "Viewport Switcher" need to be checked.  Some recommend that you make a 2nd profile to test so that you can always go back to old settings.  It's buried in the config tool...
