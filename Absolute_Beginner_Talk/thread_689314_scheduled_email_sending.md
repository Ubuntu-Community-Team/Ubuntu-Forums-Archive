### Post by bilge.tutak on 2008-02-07
There is an Add-on for Thunderbird called "SendAt", but it is pretty old. It is not workign with the current version of Thunderbird. If you can make it work, it will do what you are asking for. You will prepare the mail and set the sending time. It will send it when it is time. However, you have to keep Thunderbird open to do that. It is not exactly a good solution, but it worked for me..
