### Post by ajgreeny on 2006-04-20
If you just have one file on the cd (xxxx.iso) then I'm afraid you did not burn an iso image but simply copied the file to cd.  You need to burn the image using nero or whatever you have on your other operating system so that you end up with several different files on the cd, I can't remember exactly what, but I think it includes a boot folder, a linux image file and about 5 or 6 others.
Have a good look at your cd burning software under the file menu and I think you;ll find an appropriate menu item, then burn the iso image at a slow speed, max 4X, and I think you may be OK.
Good luck;  I think you'll love ubuntu.

### Post by ajgreeny on 2006-04-20
No, don't burn your cd as "bootable cd" as you will still not have a bootable image but just a bootable cd, (sorry but I can't explain that any better, but I think it would be a bit like the bootable floppies that win98 etc let you burn to start your comp if things go wrong. You don't need that)

If you can't work it out in Nero, which I also found difficult as I only have NeroExpress, and you're on windows, get a copy of cdburnerxp, (Google to find it) which is the best windows cd burner software I've ever used, is free, and when you start it up it gives you the opportunity to burn an iso image.  Of course when you've got linux running you will have even better cd burning software available to you.
