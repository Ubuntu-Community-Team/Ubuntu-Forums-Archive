### Post by Sutekh on 2006-05-11
Xubuntu Dapper is sensational!  I'm using it full time now.  

The shipped version of the window manager xfwm is capable of composite effects. You can make your panels and applications fully transparent.

It's very easy to setup.  All you need to do is edit your X server configuration file
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup
sudo nano /etc/X11/xorg.conf
``` Then include these lines at the bottom of the file

```

Section "Extensions"
         Option  "Composite" "Enable"
EndSection 
``` Save the file (Ctrl+O) and exit(Ctrl+X) and then restart the X server. Log out and in, or press Alt + Ctrl + Backspace

When that's done, if you open the Settings Manager and choose Window Manager Tweaks. You should be able to adjust the transparency of windows and panels.

If you are able to, you should also get the latest video drivers for your card.
