### Post by kebes on 2006-12-11
Taurus is correct: if you have both KDE and Gnome installed, you can just pick the one you want each time you login. You can switch between them very quickly and painlessly. Applications will retain their settings and preferences from one GUI to the other. Of course settings that are specific to each desktop environment (background image, shortcuts, etc.) will be different and you'll need to modify them to suit your tastes.

The above assumes that you have both Gnome and KDE installed. A default Ubuntu install only has Gnome, but if you want KDE you simply install "kubuntu-desktop" using Synaptic or on the commandline:
$ sudo aptitude install kubuntu-desktop


Similarly you can install "xubuntu-desktop" if you want to try that GUI.

The point is that it's easy to try switching and to switch back if you don't like it. It's also easy to use both, switching between them whenever you want.
