### Post by aysiu on 2006-10-11
A repository is a place where application installers are stored.

When you "install" a program through Add/Remove programs or Synaptic Package Manager, the package manager looks to see what installers it needs, downloads them, and unpacks them.

**More Info**:
[http://psychocats.net/ubuntu/installingsoftware](http://psychocats.net/ubuntu/installingsoftware)

Universe is community-maintained and has a lot of open source software

Multiverse is community-maintained and has a lot of closed source software

**More Info**:
[http://www.ubuntu.com/ubuntu/components](http://www.ubuntu.com/ubuntu/components)
