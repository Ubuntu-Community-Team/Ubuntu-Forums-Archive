### Post by vanadium on 2007-08-15
You will need to convert both the audio and the video in the mp3 file to DVD compliant formats. After that, the footage can be authored to a DVD. I am not sure if Avidemux can open mp4. If it can, that would be an easy solution to convert to DVD compliant mpg. DVDStyler is an easy to use DVD authoring solution. qDVDauthor is another graphical solution that allows much more tweaks, but consequently is a bit more difficult to use.
