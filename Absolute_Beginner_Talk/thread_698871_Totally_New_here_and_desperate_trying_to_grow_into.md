### Post by kaiju on 2008-02-16
the repositories accessed by synaptic don't necessarily contain the latest software. filezilla probably reports the new update based on a connection with the filezilla makers' server.
while you could compile the latest version from source, i'm not sure it would be worth it, the software available from the repositories being more secure as it's constantly being tested by all the ubuntu users. are you sure the new version offers enough new features and improvements over that?

edit:
if you are looking for the latest versions of your apps, [getdeb]("http://www.getdeb.net") is a pretty much trustable source for ubuntu packages. they don't have any filezilla packages though.
