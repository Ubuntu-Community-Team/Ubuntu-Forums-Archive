### Post by Bachstelze on 2007-05-14
Just grap the ZIP file, unzip it and install the TTF like any other font. For example you can do in a terminal

```
mkdir ~/.fonts
```

If will create a (hidden) dir called .fonts in your home dir, just copy the TTF in it.
