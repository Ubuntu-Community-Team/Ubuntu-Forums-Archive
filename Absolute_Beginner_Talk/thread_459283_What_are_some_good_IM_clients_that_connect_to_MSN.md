### Post by Wudugast on 2007-05-30
There's a package called "extended preferences" for both Pidgin and GAIM designed to implement features the designers either don't like or never got around to. Anyhow, when I was using Windows, installing extended preferences gave me the option to have blinking message windows in the taskbar, etc., so you might want to check it out. I'm not sure what the Linux options are as I am having a difficult time getting extended preferences to install.

You can find it here:  [http://gaim-extprefs.sourceforge.net/](http://gaim-extprefs.sourceforge.net/)
