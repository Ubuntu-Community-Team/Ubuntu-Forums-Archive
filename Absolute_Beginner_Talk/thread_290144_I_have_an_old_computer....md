### Post by devilsadvocate1987 on 2006-10-31
To start off with - do NOT try to run the live cd. Anything below 256 mb it turns into a non-responsive slideshow, and probably wont even boot. Get the alternate CD.

I remember of hearing a distribution called Ubuntu-light or something that is less resource hungry. Xubuntu with XFCE is also a possibilty but it will be reasonably sluggish as well. 

Try googleing around and see if you can find the ubuntu light distribution - I remember seeing a wikipedia page with the less resource hungry ubuntu derivatives. I'd find it for you if I wasnt in a hurry right now... 

In case that doesnt turn out too well, you could try installing xubuntu from an alternate install cd, and if things are responsive enough you could try some other window manager like fvwm or fluxbox. Those will take a little while to get the hang of though, especially if you're coming straight from windows.
