### Post by mrchrisblau on 2006-04-14
I'm in almost the exact same position you are in, mcletter.

What I'm doing is slowly merging over from windows to linux (ubuntu) with a dual boot system. I try my best to do most, if not all, of my work in linux but when there is something that would just be easier/faster/better on windows, I reboot to windows. This has worked great for me so far. 

I've found that there is a linux app that does almost exactly the same thing as its windows counterpart (for lack of a better word). Gimp = Photoshop, Office = OpenOffice.org, etc. 

My recomendation is to install a dual boot for the first bit and then, when you are doing all of your work/design/whatever on your linux install, then you should wipe the windows install. Theres no harm in it and you get an easier transition period.

Here is a list of popular windows programs and their linux counterparts (I've reffered to this often in the past): [http://ubuntuforums.org/showthread.php?t=33183](http://ubuntuforums.org/showthread.php?t=33183)

Good luck!
