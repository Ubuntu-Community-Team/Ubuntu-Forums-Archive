### Post by beniwtv on 2006-07-31
Normally updates shouldn't make things undone, like your modem configuration. This can only happen in very remote cases where an update replaces the configuration file because the author changed it or something.

But if you're still in doubt, write down what you have done to get your modem working to easily restore it if necesarry.

Hope it helps.
