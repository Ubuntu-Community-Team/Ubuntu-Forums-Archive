### Post by kelbizzle on 2007-11-25
I'm not sure about your windows key issue as all the shortcuts I use with the windows key works. To get your right ALT key to work. System -> Preferences -> Keyboard ->Layout options -> Third level choosers. I set it so that the right windows key would choose the third level. Basically that key is used for special characters one might use if you spoke another language.
