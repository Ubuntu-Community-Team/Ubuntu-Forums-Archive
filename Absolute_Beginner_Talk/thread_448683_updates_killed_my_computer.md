### Post by shae on 2007-05-19
Oh you were trying to upgrade versions, not update packages.  That is somewhat different.  If the upgrade does not work, which sometimes just happens, I recommend downloading the latest version, Feisty Fawn, and install it from scratch.  When you are installing it, create a separate home partition so the next time you want to upgrade, you do not wipe out all of your personal files and such.  If you have any questions, please ask.  

Upgrading between releases is a hit or miss kind of thing.  Either it will work perfectly, or it could end up rendering the install unbootable.  That is why when they release a new version, I usually wait a day or two to make sure they have the bugs worked out, then I reformat.  

Good luck.
