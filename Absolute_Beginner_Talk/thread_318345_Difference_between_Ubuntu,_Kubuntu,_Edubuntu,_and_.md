> **wersdaluv said:**
> What is the difference between Ubuntu, Kubuntu, Edubuntu, and Xubuntu?

I want to download KDE's BasKet app but the packages are only for Kubuntu. 

Will the packages work with my Ubuntu 6.10?They are different "flavors" of Ubuntu.[LIST=1]
[*]Ubuntu = original 'buntu, uses GNOME as its desktop environment
[*]Kubuntu = first spinoff, uses KDE as its destkop environment
[*]Edubuntu = gears towards schools and institutions, uses GNOME
[*]Xubuntu = uses XFCE4 as its desktop environment
[*]There are a few others as well, but these are the main ones that are supported by Canonical.[/LIST]You can use KDE apps in GNOME, and vice versa.  And really that does for any app that is "made" for a certain DE (desktop environment).  I use Ubuntu Edgy 6.10, which uses GNOME, but I run many KDE apps.  "KDE" apps will typically run a little bit quicker in their "native" environment of KDE, and vice versa with GNOME and GNOME-geared apps.  KDE apps are written using Qt and GNOME apps are typically written using GTK.  But that's probably more info than you wanted. :D
