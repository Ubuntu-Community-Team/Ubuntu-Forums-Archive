### Post by coskierken on 2007-11-18
I am the one who had, don't anymore, problem.  Install, via Synaptic, the Gnome ALSA mixer.  It will install to the Sound & Video applications menu.  Once inside, find the mic and make sure the REC button is checked.  I thought there would be some sort of software switch, and this is it:)
