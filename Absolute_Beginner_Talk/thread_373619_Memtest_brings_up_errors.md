### Post by P235 on 2007-03-01
Hello, for the past month or so I have experienced problems starting up my computer for the first time of the day on my [HP Pavilion N5495]("http://www.if.pw.edu.pl/~kisiel/laptop/hplinux.html") laptop.  It was always a challenge to start it up because once it reached the point just after the login it would freeze.  But, once the laptop got going, there were no problems.

I brought my problem up in irc and took the suggestion to run memtest over night to check for errors.  The test produced no errors, but I start memtest a few times a week before the first start of the day just in case.  Lately, for the last two of these morning memtests, errors have appeared within the first three minutes.  I typically shutdown the computer for a minute and start it back up again when errors occur, but the errors will continue to show up on the second memtest.  I repeat the shutdown again and the third memtest usually produces no errors within the same three minutes.  I then continue to boot up my computer.  On good days, my laptop will start smoothly, but on bad days, I may end up wrestling with it for half an hour.  

Does this mean my RAM has gone bad?  Is bad?  Must be replaced?

### Post by V-600 on 2007-03-01
I had a similar intermittent problem with my computer. I attributed it to faulty RAM. However (i'm sure someone with more technical skills will correct me if i'm wrong) if RAM is faulty the it is all the time. It turned out that one of the sockets on my motherboard was actually to blame. Shifting the RAM to a different one helped.

This may be of less help in a laptop but if you have two sockets try various combinations with what you have first.

### Post by Patrick K on 2007-03-01
Moving the memory around the slots is a good idea. Also, you can clean the contacts on the mem board with an eraser, go lightly. Sometimes that helps. Over time oxides can form on the contacts. Worth a shot.

### Post by old_geekster on 2007-03-01
I am a computer enthusiast and overclock my my CPU, RAM and GPU.

Memtest is for exactly what the name says, to test the memory for errors.  If you are getting errors, if you don't overclock (this can cause errors if it is oc'd incorrectly), then the RAM is no doubt faulty.

If you have access to some other RAM, you can install it to see if it makes a difference.

Good luck!
