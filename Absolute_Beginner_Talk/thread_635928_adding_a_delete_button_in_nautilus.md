### Post by mcduck on 2007-12-09
No, not without modifying the source code of Nautilus.

But if you wish to add a delete (that doesn't use Trash but deletes immediately) option to right-click menu in Nautilus, just go to Edit/Preferences and in the bottom of the Behavior-tab enable "Include a Delete command that bypasses Trash".

Also, Shift-Delete removes files without moving them to Trash.
