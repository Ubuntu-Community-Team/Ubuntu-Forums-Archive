### Post by steve.horsley on 2006-02-20
There are a number of ways.

**wine** is available from the Ubuntu repositories. It can run a number of windows applications. This is not "windows in a window" but windows applications in Linux.

**vmware** is a commercial offering that allows thw complete windows OS to be installed and run under Linux - windows thinks it's on a normal PC. I think this really does give you windows in a window.

There are also products called Crossover Office and Cedega. I don't know much about these but I understand that they are for running windows programs under Linux. I think Cedega is primarily for running games.
