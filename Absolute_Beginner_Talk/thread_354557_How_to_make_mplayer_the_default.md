### Post by corstar on 2007-02-06
Firstly, I am assuming you are using Ubuntu, I think it can be done the same way in Kubuntu

Yes. You need to do 2 things now.

1/ to make a application the default, right click and select properties, then the open with "tab"

Here you can select on the movie player of choice. Or image viewer or whatever file you happen to click on is.

2/ visit the "Restricted Formats" site [https://help.ubuntu.com/community/RestrictedFormats]("https://help.ubuntu.com/community/RestrictedFormats")
and you will, find a wealth of information on all the formats and proprietary formats 

Also, visit the Ubuntu Document Storage Facility here to find even more cool info.

[http://doc.gwos.org/index.php/Main_Page]("http://doc.gwos.org/index.php/Main_Page")
