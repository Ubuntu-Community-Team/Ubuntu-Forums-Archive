### Post by finer recliner on 2007-05-09
if you are trying to copy/paste in a terminal use ctrl+shift+c and ctrl+shift+v (this is for your own safety, but i believe it can be changed in the terminal's prefrences).

if you are copying between 2 different applications, make sure you do not close the first application, because each application is responsible for its own clipboard buffer in ubuntu. 
i recommend getting glipper clipboard manager to get around this problem
