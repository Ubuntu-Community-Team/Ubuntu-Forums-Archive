### Post by catlett on 2006-07-29
[http://doc.gwos.org/index.php/DapperGuide](http://doc.gwos.org/index.php/DapperGuide) I install all my firefox plugins through the Dapper Guide. I watch You Tube without any problems with Ubuntu.Flash Player 7 is the only one available but only a few sights are not compatable and require 8 or abaov. Supposedly flash 9 is being released for linux (don't hold your breath though)
KDE applications can run under gnome in Ubuntu. As long as you have the extra disk space to spare. When you install a kde applicatin, most of the kde libraries get installed as well. Therefore the application has everything it needs. I personally have  gnome and kde installed. Although only one window manager can run at a time, I have access to all the other applications that are normally associated with the other manager i.e. I run konqueror and Amarok from Ubuntu without issue.
Wine runs most apps but if you are getting a system with good specs and plenty of ram, vmware is an option as well. If your system has the resources then running a virtual XP machine inside of Linux will solve any windows only issue because you will be running windows, just virtually from inside Ubuntu.
Just a few thoughts. I am sure others have stuff to add or to correct.
