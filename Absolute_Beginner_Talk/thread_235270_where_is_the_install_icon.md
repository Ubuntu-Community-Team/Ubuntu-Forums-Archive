### Post by 23meg on 2006-08-12
You need to boot with the live CD to do the install. Insert the CD, make sure boot priorities in your BIOS are adjusted so that the computer checks if there's a bootable CD in your drive, restart your computer and you'll get the Ubuntu boot menu. Once you get to the GNOME desktop you'll see the install icon.

### Post by mostwanted on 2006-08-12
> **Jimmyd said:**
> well, you may as well have told me to build some dna, lol. thanks anyway.](*,)

Well, you *are *installing a new operating system. That has never been a simple operation.

You can't install Ubuntu from within Windows, you need to boot the CD itself. If it doesn't boot when you restart with the CD in the drive, you can usually enter your BIOS setup by holding down F2 when the bootup logo is shown. From there you can change the boot order to boot from a CD before booting from the harddisk.
