### Post by starcraft.man on 2007-05-03
Yes you would be able to install a 64 bit version of Ubuntu. Almost all chips in the last few years support 64 bit architecture, most people just don't know it.

The truth is though, unless you constantly deal with enormous file sizes, like converting movies and editing... there is no reason to really install 64 bit. 32 does everything just fine for most people and is well more supported through software and drivers, and it just works :)

Choice is yours though.
