### Post by Bachstelze on 2007-03-19
*gnome-terminal* in Gnome, *konsole* in KDE. There's also *xterm* which is a more minimalistic terminal emulator.

You can install other terminal emulators but those are the ones you have by default.

### Post by mcduck on 2007-03-19
In Gnome I recommend installing the 'nautilus-open-terminal'-package. This will add 'Open Terminal' into your right-click menu on desktop and 'Open In Terminal' into right-click menu in Nautilus file browser. Very useful feature :D
