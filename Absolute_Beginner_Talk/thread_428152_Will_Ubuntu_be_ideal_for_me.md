### Post by Nythain on 2007-04-30
if you are used to the optimization and customization of gentoo then ubuntu might not be for you... you get no real options of what software is and isnt installed during installation, and everything pretty much uses a generic 386 architecture... so there's no real custom compiling per system...

now what ubuntu does offer is a great, easy, user friendly installation process... debian repositories for all your software needs, and excellent hardware support (in my opinion) with probably the best user community for a linux distro out there
