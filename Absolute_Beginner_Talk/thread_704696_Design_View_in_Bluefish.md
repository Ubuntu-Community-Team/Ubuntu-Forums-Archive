### Post by daniel_szollosi-nagy on 2008-02-22
From what I can gather, Bluefish is a text editor and has no option for graphical preview. If you are looking for a web editor with HTML syntax highlighting and WYSIWYG editing, I recommend Kompozer.

It's in the repositories, just type 

sudo apt-get install kompozer

### Post by macogw on 2008-02-22
Bluefish is for web app development...like for writing dynamic stuff in PHP, Python, Perl, or using CGI scripts.  It's not for web design.  It's an IDE, like Eclipse or Visual Studio, but for the internet.
