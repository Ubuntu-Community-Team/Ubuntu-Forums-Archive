### Post by shanepardue on 2007-02-12
Winetools is very outdated and no longer being developed. That's probably why it's not working with a recent wine version. You are able to use wine without it, but I don't know of another helper program like winetools, sorry!

### Post by machoo02 on 2007-02-12
Supposedly there is a version that will work with the most recent version of WINE...search the forums here or check it out at [http://download.formationos.net/]("http://download.formationos.net/").  Just be aware that Winetools severely changes the configuration of WINE, so much that if you have problems with an application and want help from the WINE developers they _will not_ help you until you uninstall winetools.
