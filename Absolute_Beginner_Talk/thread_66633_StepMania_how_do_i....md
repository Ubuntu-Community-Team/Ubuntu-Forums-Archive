### Post by rsay on 2005-09-17
Unzip the file and place it under the ../stepmania/songs directory and the program will find it when it boots. My stepmania is under /usr/games/stepmania but I'm not sure if that is universal.

My first mix looks like this: /usr/games/stepmania/Songs/Dance Dance Revolution 1st Mix

You can place other add-ons under the appropriate directories. Announcers, RandomMovies, NoteSkins, Characters, BGanimations, Themes and Visualizations. I found a bunch of add-ons at StepManiaStuff.com

### Post by gerbman on 2005-09-21
[QUOTE=Eibo]ok where is that file? the stepmania file?[/QUOTE]

My Stepmania directory is at /usr/local/games/Stepmania/

Within that directory is the "Songs" folder.  This is where your mixes need to be.  If you got a .zip file (like one from [here](http://www.ddruk.com/downloads.php?subcat=57)) then you need to unzip the file and put the resulting folder in the Stepmania "Songs" folder.  Things *should* work after that.
