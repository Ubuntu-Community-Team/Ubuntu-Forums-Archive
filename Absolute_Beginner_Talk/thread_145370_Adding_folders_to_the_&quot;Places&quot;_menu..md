### Post by mcduck on 2006-03-16
Open nautilus (the file manager in gnome), browse to folder you want to add to the menu, select Bookmark/Add Bookmark. That's it. Shortcut is Ctrl-D.

(If you are not using browser-mode nautilus, it's in Places/Add Bookmark)
