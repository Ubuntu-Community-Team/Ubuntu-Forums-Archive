### Post by lastredoubt on 2007-03-12
The best way to deal with the fact that Ubuntu isn't Windows is to sit back and smile.  Think again before you feel it necessary to have a Windows specific application -- there are many alternatives in Ubuntu that work just as well and in some cases better.  There are many easily found resources for typical alternatives.  If you can't find what you are looking for, then try asking in these forums -- many people have gone through similar searches and have found satisfactory results.  If, however, you are compelled despite all of that to use that must-have windows application, then wine is a great place to start, to which there are many resources available for usage of all sorts of applications, including wine tweaks.  As has already been mentioned, the wine database is the start.  

Also, wine is not there to be windows, but to emulate it, nor is it there to 'counter balance' a suspected deficiency in Ubuntu.  If you feel strongly about the given applications, perhaps you should attempt to have a dual-boot machine.

### Post by gh0st on 2007-03-12
I have to say that I'm glad Ubuntu isn't Windows, it's one of the main reasons I use it :) 

VMWare is no good for gaming because it doesn't support 3D graphics, you are only using a VMWare virtual video card which is only basic. Wine does work well for some games but it's not a replacement for Windows, I don't think it's supposed to be.

The new Linux Kernel has virtualization built in and there is talk of that giving better hardware integration and allowing hardware video accelleration. I'm not sure what the situation is with that so you'll have to read up on it if you want to know more.

For me, I have a VMware Windows machine that I keep as a backup just in case I need it but I hardly ever boot it. The only thing you can't do with VMware is gaming because of the 3D situation. If you really need a windows app you can run it on a virtual machine as long as there are no heavy graphics requirements. I use my PS2 for gaming and it does me fine. If you need Windows gaming or something you should run a dual-boot system really, it's the best way I think.

I suppose it's a personal thing but I think Ubuntu does everything I need from a PC and I do quite a bit of programming and techy stuff, I don't do a lot of video rendering/encoding or graphics stuff I have to admit but I think there are good open-source solutions for most tasks these days. It's getting better all the time.

Just my 2 cents worth, I'm sure some people will disagree, a computer is only what you make of it in my opinion. If you can make it do what you want and get the results you want, who cares how you do it.
