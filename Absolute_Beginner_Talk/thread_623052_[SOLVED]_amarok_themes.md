### Post by paradoxni on 2007-11-25
install kcontrol

```
sudo apt-get install kcontrol
```

then run it

```
kcontrol
```

Appearance > Colors

Then choose from one of the color schemes or import one which you can download from [http://www.kde-look.org/index.php?xcontentmode=28](http://www.kde-look.org/index.php?xcontentmode=28)

I got Amarok to look like the pic above using a slightly adjusted version of [ this theme](http://www.kde-look.org/content/show.php/Ubuntu+studio?content=61890)
