I've been a long-time kde fan, but Ubuntu's version of gnome is really growing on me.  Anyway, here's my question:

I really like how gnome uses the two small panels to keep program launchers/notification areas separate from running programs.  In kde, I tended to put a lot of shortcut icons in the single panel, leaving the taskbar area very crowded.  I've tried putting in new panels to serve as a dedicated launcher or taskbar area, but it seems the configuration menu then only lets me configure the original panel.  I could use a dock instead of a launcher panel, but so far none of the ones I've seen come close to the dock in OSX or the free docks available for Windows in terms of configurability and graphical polish.  

So, how do you guys usually customize your kde desktop to help make managing and launching programs smoother and more efficient?

### Post by Sparkster185 on 2007-05-22
Have you tried using SuperKaramba?  There's a bunch of widgets out there that simulate the OSX dockbar.  Off the top of my head, there's TuXbar, Kroller (the one I use, currently), as well as others.

I recently moved all my application shortcuts (and K-Menu plus Home) off my desktop and taskbar to the Kroller.  This way I can only have open applications and a few other things on the taskbar. It makes things a lot less cluttered.

[http://www.kde-look.org](http://www.kde-look.org) hosts a bunch of Karamba themes.
