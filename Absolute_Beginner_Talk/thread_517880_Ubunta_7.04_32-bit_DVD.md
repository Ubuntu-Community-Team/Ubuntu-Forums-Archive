### Post by TBerben on 2007-08-05
The base installation of Ubuntu needs about 4GB of disk space, so 8GB will be fine (just don't expect to put much data on the rest of the partition ;)). The Ubuntu dvd is bootable, pop it in the dvd drive, reboot your computer (optionally have a look in the BIOS to configure it to boot from dvd) and you'll enter the livecd environment: a complete Ubuntu OS run from the dvd. From there you can install Ubuntu to your harddrive.

During the installation Ubuntu installs GRUB, this so-called boot loader allows you to choose either Ubuntu or windows at boot time. The default choice is Ubuntu, which boots automatically if you don't do anything after 10 seconds. You can change this though in the grub configuration once you have installed ubuntu
