### Post by -grubby on 2007-12-08
like window borders? if so :> sudo apt-get install emerald and navigate to system>preferences>emerald theme manager and install the theme from the theme file you downloaded ([http://www.gnome-look.org](http://www.gnome-look.org) has some) and it will (or should) work with your old beryl themes. Oh yes, some people find it doesn't work until you restart (after clicking on the theme you installed it doesn't change, so you can always try restarting)
