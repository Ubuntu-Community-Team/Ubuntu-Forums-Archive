### Post by ScottMorris on 2006-07-16
There are a few programs that can burn isos.

In Windows, Nero Burning ROM, Roxio easy media creator, iso creator ([website]("http://isorecorder.alexfeinman.com/isorecorder.htm")) and others, google iso burning.

In Linux some include K3B ([Tutorial]("http://www.iceteks.com/articles.php/burniso/3")), NeroLINUX (but you either need a Serial Number for Nero 6 Retail or you can puchase one from Nero, [www.nero.com]("http://www.nero.com")). and others
