### Post by utilitarian on 2007-10-22
instructions for downloading veoh movies from gusty gibbon

1.open package manager

2. search for and install libsqlite3-dev,libssl-dev, and subversion.

3. open command prompt type: "cd /home/your_user_name"

4. type "svn co [https://openveoh.svn.sourceforge.net/svnroot/openveoh](https://openveoh.svn.sourceforge.net/svnroot/openveoh) openveoh"

5. type "cd /home/your_user_name/openveoh"

6. type "make socket"

7. type "make"

8. type "make install"

9. type "cd /home/your_user_name/bin"

10. type "vget url_of_veoh movie"

11. type "cd ../Video/Veoh"

12. type "your_favorite_player nameOfMovie.avi"

### Post by DrMega on 2007-12-13
> **aonegodman said:**
> WoW! is there a way to "flag" threads like this or save to a favorites folder or something so you can find it again on the forum? This is great info. Thanks to all contributors. ;)

From the search menu, choose "Find all your posts". That's how I keep an eye on stuff.

If you like FPS, have a look at Nexuiz and Sauerbraten (both in Synaptic) and World of Padman (not in Synaptic but easy enough to install).

[http://padworld.myexp.de/index.php?startlang=en]("http://padworld.myexp.de/index.php?startlang=en")

If you download it, it is a .run file. Having downloaded it, open a terminal window, cd to the directory you saved it to and then type:
```
sudo bash worldofpadman.run
```

The installer will take care of the rest. It is still in its fairly early stages but it looks amusing, and you can get a deathmatch on the go. You need to choose multiplayer and then create a server (all really intuitive from the game's main menu).

If you like internet based video, a tool I like is the video download helper addon for firefox.

[https://addons.mozilla.org/en-US/firefox/addon/3006]("https://addons.mozilla.org/en-US/firefox/addon/3006")
