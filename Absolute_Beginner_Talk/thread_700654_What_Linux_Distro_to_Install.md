### Post by julian67 on 2008-02-18
btw partedmagic is probably the best graphical partition tool available, the new version is superb.  It includes firefox and a file manager so you can browse your partitions, locate & recover data easily as well as do the fundamental tasks of creating/deleting/resizing/formatting/labelling partions and so on. It's definitely a step up from Gparted, in fact it's a fork from Gparted with many enhancements.

distros for your hardware: as above, also Antix and Zenwalk are great on older hardware. Antix, Zenwalk and Vector are all considerably faster than Xubuntu Gutsy.  Antix is the easier install if you're fairly new to all this as it uses the extremely  simple Mepis installer. Antix is based on Debian so will feel very familiar to an Ubuntu user (once you get used to the fluxbox desktop). For eye candy and a friendly looking environment maybe Vector is better.
