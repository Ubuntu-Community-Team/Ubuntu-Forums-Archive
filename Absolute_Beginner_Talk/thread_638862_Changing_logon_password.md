When I highlight any of the user names, including root, and then click on properties, the password spot, which I assume is for logon (is it?) has eight dots. So I am not sure whether this really is the place to change the logon password. ### Post by kevinatkins on 2007-12-12
I'm assuming you're using Gnome.

Go to System -> Preferences -> About Me.

You can change your password from there (not really the most logical place I guess, but there you go!)
