### Post by balaknair on 2008-01-22
OK here's a short HowTo
In Amarok open Script Manager(Tools>Script Manager)
In script manager choose Lyrics

If you have already installed any Lyrics scripts, choose it and and click Start

If not choose the 'Get more scripts' option to install a Lyrics script, and then in Script Manager, start the Lyrics script.
When you start playing the song, open the Lyrics tab in the Context Side Panel and the script will get the lyrics for you from the net.
