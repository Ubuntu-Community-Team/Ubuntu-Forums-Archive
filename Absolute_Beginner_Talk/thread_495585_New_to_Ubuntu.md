### Post by Old Pink on 2007-07-08
> **Th0rn0 said:**
> Another big one for me is... Will I be able to get Xfire and Windows live messenger working on it. These are musts :). Also Fraps and Daemon Tools.

You'll be able to use Windows Live Messenger services, but not the actual client you're used to.

We have much better, open source, ad-free, annoyance free equivalents. Included with Ubuntu is GAIM (now [Pidgin]("http://www.mbhoy.com/14-06-2007/pidgin-201")) which will connect you to AIM, ICQ, Jabber/XMPP, MSN Messenger, Yahoo!, Bonjour, Gadu-Gadu, IRC, Novell GroupWise Messenger, QQ, Lotus Sametime, SILC, SIMPLE, and Zephyr. :) 

There's also options like Kopete and aMSN, which are closer to WLM feature wise, and allow video chats, etc :) 

Have a look at aMSN: [http://www.amsn-project.net/](http://www.amsn-project.net/)
