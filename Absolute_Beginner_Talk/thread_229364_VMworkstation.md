You should probably get the VMWare Server. It's free (as in beer) and it's quite well supported. 
Go to this webpage:
[http://www.vmware.com/download/server/](http://www.vmware.com/download/server/)
Download the software, then check out this thread:
[http://ubuntuforums.org/showthread.php?t=183209]("http://ubuntuforums.org/showthread.php?t=183209&highlight=howto+vmware+dapper")
It details how to get Windows XP up and running in VMWare, but once you get VMWare installed, you can install whatever OS you want on your VMs. If you run into trouble, post in that thread.
