### Post by foxmulder881 on 2007-06-13
Right click on the top panel and select +Add to panel. Select the CPU Frequency Scaling Monitor. There should now be a small applet in the panel which should allow you to change to CPU frequency. Underclock or overclock.

NOTE: Some CPUs do not support frequency adjustment using this applet.
