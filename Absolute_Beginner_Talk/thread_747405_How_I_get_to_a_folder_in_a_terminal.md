### Post by viciouslime on 2008-04-06
If you mean make a folder, you use the command mkdir. So:

mkdir test

Will make a folder called test in the folder you are in at the moment. To change into a folder, use cd, so:

cd test

will change into the folder test, as would cd /home/you/test assuming that is where you made test.

I think this is what you mean? If you were getting at something else, let me know...
