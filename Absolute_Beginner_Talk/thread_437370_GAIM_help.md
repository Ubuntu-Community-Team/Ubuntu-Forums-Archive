### Post by dutchman72 on 2007-05-08
In the buddy drop down list, if you uncheck the show buddy details and show idle times, it shrinks to a smaller font size than what is used at first. Other than that there is no real option to use a different font size in the buddy listings.

In the preferences, there is a font changing setup, but that is only for the conversations.
