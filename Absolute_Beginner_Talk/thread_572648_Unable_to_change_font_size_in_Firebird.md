### Post by kyphi on 2007-10-10
Re Firebird, Firefox and Thunderbird.

I assume that you meant Firefox, the browser and Thunderbird, the email programme.

In Firefox you can increase the size of the text by going to View and then Text Size.

In Thunderbird, you can do the same thing but you can also increase the default text size after selecting 'Write', to start a new email and going to 'Edit" and then Preferences, Display and then Fonts.

As for the updates, yes you can get them from the source directly but sometimes this causes another programme to malfunction because each programme has dependencies to be satisfied.  A new version that is not yet in the repositories may have dependencies that you are not aware of.

> Do you have any idea why changing font sizes through Preferences is not working?

When you go to Edit -> Preferences -> Display -> Fonts, you will see 3 buttons to adjust font sizes.  Adjust all 3 to the (same) size that you want.  Perhaps you only adjusted one of them before.
