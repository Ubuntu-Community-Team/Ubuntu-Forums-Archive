### Post by Brunellus on 2007-08-04
Fluxbox and blackbox are very light weight, relatively minimalistic window managers.

The Ubuntu Fluxbox howto is here:

[https://help.ubuntu.com/community/Fluxbox](https://help.ubuntu.com/community/Fluxbox)

The main page for the Fluxbox project is here:

[http://www.fluxbox.org](http://www.fluxbox.org)

I use Fluxbox for lightweight installations, or whenever I want to run a clean, uncluttered system.

Note that unlike GNOME (whose window manager is metacity) and KDE (whose window manager is kwin), fluxbox is ONLY a window-manager.  It doesn't come with anything else--no disk mounting/unmounting tools, no file browsers, nothing.
