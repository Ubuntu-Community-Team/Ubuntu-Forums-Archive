### Post by deadgobby on 2007-09-10
Ubie does see all yours cards and that is fine. What it does is getting confused on which one to use. You'll need to go into BIOS and set up a default sound card you want to use all the time. Or you'll get different defaults all the time. As I know of there is no easy way to set up multi sound devices or it is going to do it. So if you want your PCI card as default. As I stated to to going to into BIOS and make that card default.
