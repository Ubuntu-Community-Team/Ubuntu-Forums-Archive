### Post by seventhc on 2008-02-21
go to *System>Administration>Software Sources* click on the *Updates* tab and you can change it there. It's probably set to check daily, and should check without a reboot.
You can also manually check by going to *System>Administration>Update Manager *from there you can check for new updates anytime you want[I].
[/I]
