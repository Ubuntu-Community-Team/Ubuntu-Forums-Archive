### Post by bongobonga on 2007-11-10
You can download the latest drivers from the Nvidia website, but there is a much easier way to do it. 
The easiest way to upgrade your nvidia driver is to use an application called 'envy', which can be obtained from the Ubuntu repositories. Envy will automatically download the latest driver from the nvidia server and compile/install it onto your system. All you have to do is follow the simple on-screen instructions.
