### Post by lebski88 on 2006-10-01
Hello,

When you say chip what exactly do you mean? The only problem I can see you having is that your bios might refuse to boot off it. So go into bios (del or f2 or something when booting) and make sure that you can boot off whatever device you are thinking of using.

Then load the live CD and select that device to instal it onto. However this wont work like a live CD that can be moved around from computer to computer.

You can get a linux distro that runs of a USB key or similar; I think there is a light version of knoppix that will boot live off a USB stick but I don't think you would be able to do it with a standard ubuntu distro. Check out:

[http://rz-obrian.rz.uni-karlsruhe.de/knoppix-usb/](http://rz-obrian.rz.uni-karlsruhe.de/knoppix-usb/)

For details on booting knoppix.
