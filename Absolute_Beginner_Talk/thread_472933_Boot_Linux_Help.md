### Post by scott_g on 2007-06-18
You could try shutting your computer down with the CD-ROM door open, then restart (it doesnt matter if there's a cd in the tray or not).  If the computer closes the CD-ROM door when it starts, then it is trying to boot off the disk.  If it doesn't, its a BIOS problem.  If it does, you may need to re-burn the disk.
