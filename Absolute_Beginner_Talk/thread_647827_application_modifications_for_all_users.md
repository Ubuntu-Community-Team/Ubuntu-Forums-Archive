### Post by miltdewey on 2007-12-22
Installed flash plugins for firefox under the "administrator account" to play youtube videos. Other users' version of firefox does not have the plugins and these users are not able to install the plugin. How do you make an application modification available to all users?

---

### Post by taurus on 2007-12-22
The flash plugin should have been installed in /usr/lib/firefox/plugins.  Then, everybody on the system can use it.
