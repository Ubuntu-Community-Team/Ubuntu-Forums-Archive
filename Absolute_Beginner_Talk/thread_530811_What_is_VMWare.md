### Post by Jverse on 2007-08-20
VMWare is a virtualization tool that will let you create multiple machines on one computer. It basically tricks the computer in beleiving that you have that many computers running using virtualization. The only downfall is it will use the memory that you have as well as the disc space. So if you are thinking of using it, you better have a hefty machine with alot of memory and disc space.
