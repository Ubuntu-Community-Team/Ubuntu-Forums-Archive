### Post by Drakkor on 2006-06-25
MwSnap Current version is capable of capturing the whole desktop, a highlighted window, an active menu, a control, or a fixed or free rectangular part of the screen. MWSnap handles 5 most popular graphics formats and contains several graphical tools: a zoom, a ruler, a color picker and a window spy. It can be also used as a fast picture viewer or converter. ### Post by joshuapurcell on 2006-06-25
I don't know of a complete replacement for the program you mentioned, but the printscreen button comes very close to what you are talking about. Just press the button for the full desktop to be captured, and hold down the ALT key before pressing PrtScreen to get only the focused window. Once the Save To window pops up, I believe you can just change the extension to whatever image type you want. I'm not sure about what various formats are supported or anything.

### Post by obadiah on 2006-10-10
In addition to doing screen captures, MWSnap also has some other nice features like a color picker and a screen ruler. You can get very similar functionality on Ubuntu by installing gcolor2 and gruler. gcolor2 is in the repository and the easiest way to get gruler installed is doing an alien install of the Fedora 4 RPM. Then to make it easy to use, I map some keyboard shortcuts to launch gcolor2 and gruler. I also really like the ability to capture an arbitrary part of the screen so I wrote a tiny script that uses imagemagick and import.
