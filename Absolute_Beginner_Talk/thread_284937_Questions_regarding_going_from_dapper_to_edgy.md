### Post by d3v1ant_0n3 on 2006-10-26
As standard, Edgy doesn't come with flash- same as dapper in that regard- you can install it through automatix/automatix2 tho, or just grab the flash 9 beta from adobe.

Your sources.list file shouldn't need a whole cd! It's only a text file. Unless, of course, you have a whole bundle of sources. Like millions.

If you have specific 3rd part sources, I'd either make a note of them and add them to sources when you install edgy, or email the /etc/apt/sources.list file to yourself or something, and then copy/paste any 3rd part sources you need.
