### Post by Cypher on 2007-04-26
Is Feisty running enough for you to load up Firefox? If so, Bookmarks->Organize Bookmarks->File->Export and it will export to an HTML file. 

I'm sure you can find something similar with Thunderbird..

Because of saving bookmarks contanstly, I ended up switching over entirely to [Del.icio.us]("http://del.icio.us/") for my bookmarking needs. Firefox intergrates VERY well with it and I have the same bookmarks whereever I go..
