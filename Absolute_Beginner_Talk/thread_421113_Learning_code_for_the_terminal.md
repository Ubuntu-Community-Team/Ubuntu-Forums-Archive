### Post by 3rdalbum on 2007-04-24
[www.linuxcommand.org](www.linuxcommand.org) is a really good tutorial.

But really, you do eventually learn stuff just by listening to what commands other people post, and if necessary look at the manual pages.

For instance, if you want to know how the Apt-get system works, you would type "man apt-get" into the terminal, and it will tell you all the options for Apt-get and how everything works.
