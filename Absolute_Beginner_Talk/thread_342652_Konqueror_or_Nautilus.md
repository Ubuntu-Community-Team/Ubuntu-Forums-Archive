### Post by Pobega on 2007-01-20
Nautilus is the default Gnome file manager, and it also doubles as a CD-Burner. Nautilus by default is not great, but in the Ubuntu package it's default are nice.

Konquerer is KDE's file manager, and it's very quick within KDE but I find it to be pretty slow outside of KDE.

All in all, since Konquerer works alongside all other KDE apps, you probably only gain usability if you're using Konquerer from **within** KDE. If you're using Gnome I'd go with Nautilus, if you're in KDE definitley use Konquerer.
