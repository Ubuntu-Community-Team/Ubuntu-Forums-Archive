### Post by soulfly7x on 2007-05-01
I assume you have both installed. When you get to the login screen click Options>Sessions>KDE. Finish logging in. It will ask if you want to set it as a new default or just that session. Select whichever you want and you should be in Kubuntu instead of GNOME Ubuntu
