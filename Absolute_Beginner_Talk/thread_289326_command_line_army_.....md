### Post by IYY on 2006-10-31
The reason I can't suggest a script for you is that most problems solved by scripts are personal ones. If there is a big problem that needs solution by many users, you can be sure that somebody has already written a program to solve it. But the problem with personal problems is that *you* may not have such problems at the moment. This is where the following point comes in:

The best way to learn shell scripting is to read these forums and try to solve other people's script problems. I've written many medium-length scripts for people and it taught me a lot. Try to answer even if you can't figure it out immediately. Do some research and write the script.

If you need some help, feel free to IM or contact me through MSN.
