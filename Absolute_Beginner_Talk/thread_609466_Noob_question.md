### Post by taurus on 2007-11-11
Boot from the CD.  Then, you would see an install icon on the desktop.  Double click on that to install.  Make sure you burn the ISO to a CD as an image file if you want to boot from it.  Don't just unpack and copy the files over because it is not going to boot that way.
