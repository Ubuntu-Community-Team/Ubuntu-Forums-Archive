### Post by llamakc on 2007-11-09
```

# udev rules file for libgphoto2 devices (for udev 0.98 version)
# Created from this library:
#   libgphoto2      2.4.0          gcc, ltdl, EXIF
#   libgphoto2_port 0.8.0          gcc, ltdl, USB, serial without locking
#
SUBSYSTEM!="usb_device", GOTO="libgphoto2_rules_end"
ACTION!="add", GOTO="libgphoto2_rules_end"

```


is how mine on Gutsy is now. Notice the difference between yours (usb*) and mine (usb_device). If you edit yours to test it, don't forget to restart udev.

sudo /etc/init.d/udev restart
