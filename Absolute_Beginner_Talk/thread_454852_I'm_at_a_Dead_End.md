### Post by Happy_Man on 2007-05-25
From Konsole: (System -> Console)

```
sudo apt-get install firefox
```

will install firefox for you. Try from there.

EDIT: Sorry, didn't read it all. My bad. From a computer with working Internet, go to [http://packages.ubuntu.com](http://packages.ubuntu.com). Look for Firefox, and download all the .debs for it and its dependencies. Install the .debs for the dependencies BEFORE you install Firefox on your comp. Hope this helps!
