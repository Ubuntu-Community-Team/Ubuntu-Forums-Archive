### Post by louis_nichols on 2006-05-08
Well... a software with a powerfull media library in my opinion would be [amarok.]("http://amarok.kde.org/") Very powerful media library. It can edit tags also. As for managing dvd collection, too, that it cannot do, nor do I know an app that does all these things. As a media catalog I use [gwhere,]("http://www.gwhere.org/home.php3?idLanguage=en") partly because it will run in win too so I can share the catalog. It's not in the repos, but compiles smoothly with gtk2 support.

mp3-editing would be pretty well done with [audacity]("http://audacity.sourceforge.net/")

As for a good cd extractor... I have to say that Linux, in my experience, lacks a REALLY nice and good one. Lame is available, so you can use that from CLI, but I'm guessing is not what you want... I think sound juicer can use lame as a backend, via liblame.
