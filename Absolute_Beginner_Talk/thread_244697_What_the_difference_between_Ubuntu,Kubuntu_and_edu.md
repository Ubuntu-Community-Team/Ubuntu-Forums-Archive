### Post by HereInOz on 2006-08-26
Essentially, the main difference is in the type of desktop used.  Ubuntu uses the Gnome desktop, Kubuntu uses the KDE desktop and Xubuntu, designed mainly to run on low spec machines, uses the XFCE destop, which is more basic, and uses less resources.  

There is no real difference in the base system of Ubuntu and Kubuntu, and probably most Ubuntu users will end up installing applications from the KDE destop in any case, as Gnome and KDE co-exist quite happily.  I run primarily Gnome desktop, but use K3b, from the KDE desktop, as my favourite CD/DVD creator.

I haven't looked at Edubuntu, but I believe it is essentially a KDE desktop with emphasis on educational applications.  I will be happy to be corrected on this though.

### Post by Jerome36 on 2006-08-26
You really can't ask what the best is, because you're going to get a variety of answers, because it's all a matter of opinion.  Some people like using the KDE desktop enviornment (Kubuntu), while others like Gnome (Ubuntu).  You can find all sorts of discussions on why people prefer one over the other, on this forum.  It's been discussed a LOT, including differences, similarities, and features.

As far as which is faster, jISh already answered that.  Xubuntu will run better, especially on machines with lower hardware specs.  It uses Xfce, which is lighter and less demanding then Gnome and KDE.  If you were to install Ubuntu, which uses Gnome, you can still add the other desktop enviornments afterwards, and then decide which one you want to use, whenver you boot up, or log back into Linux.
