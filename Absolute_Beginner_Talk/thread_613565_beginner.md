### Post by erfahren on 2007-11-15
hermanzone's site also has comprehensive info on dual-booting. [http://users.bigpond.net.au/hermanzone/](http://users.bigpond.net.au/hermanzone/)

before shrinking your Windows partition I suggest:
Run scan disk ("Check this disk for errors" under the Tools on the drive's Properties)
run it two or three times, it doesn't fix everything the first time through in my experience. You can check the results in the Windows Event Viewer see: [http://www.housing.hawaii.edu/resources/support/chkdsk.htm](http://www.housing.hawaii.edu/resources/support/chkdsk.htm)

Then defrag the drive using JKDefrag (freeware), it will defrag and move date to the beginning of the drive better than the Windows one will. (Shut down as many running processes as you can before running - Anti-virus, etc. - you'll get better results. [http://www.kessels.com/JkDefrag/](http://www.kessels.com/JkDefrag/)

Personally, I set up a FAT32 partition as well to exchange files between the two OSs. Gutsy may have support to be able too read/write to a Windows NTFS partition now, but I'm not sure. Having a storage partition comes in handy in any event.

good luck
