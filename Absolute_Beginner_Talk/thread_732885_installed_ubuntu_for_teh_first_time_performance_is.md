### Post by seshomaru samma on 2008-03-23
Welcome to the forums

it's not a qusetion of drivers
(notice that you are using an old version of Ubuntu , a new version will come out next month , I think, and will probably be faster)
Ubuntu is not the fastest Linux around but you can speed it up considerably
two of the easiest ways are:

1.installing a lighter graphic environment=the default is GNOME which you are using now, install XFCE and it will double the speed. to do that you run the command:
```

```
log out and choose XFCE from the "sessions" menu
2.using lighter applications, instead of Firefox use Opera (or Firefox 3 beta) instead of Nautilus (the equiviland of Explorer in Windows) use Thunar etc...
