### Post by outofnicks on 2006-11-05
Bluefish is a great way to wean yourself from GUI apps, if you feel inclined to go in that direction. Tags are entered with the click of a button after highlighting a block of text, and there's also auto-closing for tags which can be toggled on or off with Ctl-T. So you can type <table> and </table> automatically appears with the cursor between them, and then you do the same with <tr> then <td>. It has a lot of helpful features that I'm still discovering, which makes working with the source much easier than a plain text editor. Like a Tidy cleanup filter, and built-in HTML validator.
