### Post by mcduck on 2008-01-02
Gnome Splash screen? It was enabled in previous Ubuntu versions but disabled in Gutsy. You can enable it with the Configuration Editor: Press Alt-F2 and run 'gconf-editor', then browse to apps/gnome-session/options and enable 'show_splash_screen'.
