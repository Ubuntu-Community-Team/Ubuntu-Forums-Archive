### Post by ESPOiG on 2007-06-24
try making your entry in /etc/fstab but back this up first

sudo cp /etc/fstab /etc/fstab-bak
sudo mkdir /media/sda1
sudo gedit /etc/fstab

add this line

/dev/sda1 /media/sda1 ntfs defaults,nls=utf8,umask=007,gid=46 0 0

and find the old one that has #/dev/sda1 look beneath it and there should be a line like UUID=786etc like so and uncomment that out by putting a # infront of it

then do
sudo umount -a
sudo mount -a

and see if it is mounted or not i dont think it will appear anywhere straight away but you can check /media/sda1 and see if the files are in there and then reboot if you want to see it in the places menu or computer

hope this helps
