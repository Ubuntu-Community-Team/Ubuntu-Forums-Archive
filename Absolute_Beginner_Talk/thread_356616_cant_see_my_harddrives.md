### Post by rai4shu2 on 2007-02-08
I think that's working as designed. Nautilus should only ever show drives that are removable or that do not properly support checking whether the media is removable.

[http://mail.gnome.org/archives/nautilus-list/2006-October/msg00014.html](http://mail.gnome.org/archives/nautilus-list/2006-October/msg00014.html)

If you want to make those drives easy to access, simply put bookmarks to them in Nautilus or the file selector window.
