### Post by Stex on 2006-11-03
You just need to run the following command from a terminal (Applications > Accessories > Terminal) and the update manager should do the rest. Though there have been a few problems with this in the past, consider everything you do with caution. Back-up everything.
```
gksu "update-manager -c"
```

Remember, however that 6.10 is potentially unstable. ([http://www.ubuntuforums.org/showthread.php?t=286209](http://www.ubuntuforums.org/showthread.php?t=286209))
