### Post by forestpixie on 2008-03-25
> Hey, but how do u recognise a beta release??

You recognise a beta - by everyone saying - this doesn't work, that doesn't work - and when the answers are - it's beta :)

When it's not a beta - you will get an option in your update window to update to it - that only happens once it's been released, shortly after that there will be a new version being released- it will start as alpha, then beta, then RC (release candidate) and then a new version gets out.

The new version will be long term support - LTS. In October the new version will be apparently called Intrepid Ibex. 

There are 2 types of Ubuntu, normal releases which happens about every 6months which are supported for less time and the LTS versions which last for 3 years depending on whether its a desktop or server version.

[https://wiki.ubuntu.com/Releases](https://wiki.ubuntu.com/Releases)
