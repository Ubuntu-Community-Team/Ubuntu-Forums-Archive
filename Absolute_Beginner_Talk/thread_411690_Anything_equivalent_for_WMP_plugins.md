### Post by Marsolin on 2007-04-17
They don't officially support [Opera]("http://linuxappfinder.com/package/opera"), but Opera is compatible with Mozilla plug-ins and will go use it anyway. I use Opera and have gotten [MPlayer]("http://linuxappfinder.com/package/mplayer") working. If you just want the plug-in and not the MPlayer GUI there is a mplayer-nogui package available.
