### Post by az on 2006-08-06
> **JohnJSal said:**
> But how do I access just that part of the disc?

The live cd can run the installer, but you don't have to run the installer to use the live cd.

You can safely press enter when you boot the live cd - it won't install ubuntu on your hard disk until you press the "install" button.
