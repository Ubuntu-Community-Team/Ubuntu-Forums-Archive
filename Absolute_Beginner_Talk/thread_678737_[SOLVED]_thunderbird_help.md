### Post by mortsahl on 2008-01-26
>> the error comes up "failed to connect to server pop. .com"

Unless you made a typo, the answer to the problem is in the error message.  There should only be one . (period) between pop and com, not two.  Double check your Thunderbird setup for pop and smtp server names.
