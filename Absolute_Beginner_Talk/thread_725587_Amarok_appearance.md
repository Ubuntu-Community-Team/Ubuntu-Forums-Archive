### Post by AndyCooll on 2008-03-15
Isn't that the default setting almost?

All that's been done is that the user has populated all their songs (manually) into the right-hand pane probably by drag and drop. You can drag and drop as many songs as you want into the right-hand pane, and if you choose in the options to remember the last list (which I don't think is the default setting) they'll still be there next time you restart Amarok. 

The left hand pane is simply displaying the "Context" tab. And the "suggested songs" is an Amarok plugin IIRC.
