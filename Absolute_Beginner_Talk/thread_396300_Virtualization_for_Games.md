### Post by compmodder26 on 2007-03-29
As I said, 3D in virtual machines is a no-go right now.  As far as I know, there is no virtual machine program (VMware, Virtualbox, etc.) that can access your video card, currently.  Without, that, the best you can do is software rendering, which is not advisable.
