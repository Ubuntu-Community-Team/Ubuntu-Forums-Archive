### Post by seventhc on 2008-02-16
For vlc you can go to *Applications>Add/Remove* set it to *All Open Source Applications* in the upper right corner then you can just search vlc. It should show up just check it off and click [I]Apply Changes.
[/I]I don'tuse realplayer, so not sure on that one.
I think you will have to set it to default if you want it to be. Otherwise vids will try to open with Totem.
To do this you can right click on any vid file, choose properties and Open With, add vlc and tell it to open with vlc.

### Post by oldos2er on 2008-02-16
To install Realplayer, go to [http://www.real.com/linux](http://www.real.com/linux) , download RealPlayer10GOLD.bin to your desktop. Once it's downloaded, right-click on it,  go to  Properties, Permissions, and check the box  'Allow executing  file as program.'  Close the Properties window, double-click RealPlayerGOLD.bin, and select 'Run in terminal.'
