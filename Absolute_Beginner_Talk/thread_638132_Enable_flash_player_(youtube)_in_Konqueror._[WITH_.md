### Post by Whiffle on 2007-12-11
You should be able to install it with apt-get/aptitude/synaptic etc.  Once you've got it installed, just go to (in konqueror)  Settings > Configure Konqueror > Scroll Down to Plugins > Scan For New Plugins.  

After that it should work... i think.

### Post by Seisen on 2007-12-11
To install Flash run this in the terminal in the install_flash_player_9_linux directory.

```
./flashplayer-install
``` which will run the installer.
