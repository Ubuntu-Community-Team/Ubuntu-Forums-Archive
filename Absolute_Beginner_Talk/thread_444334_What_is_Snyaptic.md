### Post by lebe0024 on 2007-05-15
It's the package manager - which allows you to install/uninstall programs listed in the ubuntu repositories.  It's like the add/remove software section in the windows control panel - except you can install programs with it that are listed in the ubuntu online software repositories and any custom software repositories you add.

So under "applications" on your toolbar, there's an add/remove program.  This is a very simple program that does't do everything, appearently.  Synaptic is under System->Administration.  It's a more advanced version of the add/remove program. 

I don't understand why there's two different programs to add/remove stuff from your system - There should only be one.  It's confusing to new users like me.

### Post by HereInOz on 2007-05-15
Synaptic is one of the utilities that you can use to install and uninstall programs on your Ubuntu system.  Synaptic keeps a list of programs that are available in the repositories, and allows you to search for them, and install them.

Probably one of the most useful pieces of kit available if you want to install software.

Don't worry about it - just use it!

### Post by SomethingCronic on 2007-05-15
Synaptic is the GNOME GUI deb package manager/installer. If you've used apt-get you've been using the command line version of synaptic already.

Load it up and have a look, it lists all the programs available for you  in the ubuntu repositories and makes them avaialble for download. It's not something to 'worry' about, but without it ubuntu could become quite boring very quickly :) 

If you use Kubuntu you might be used to seeing it as 'aptitude'
