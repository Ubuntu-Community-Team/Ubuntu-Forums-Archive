### Post by teeleef on 2008-02-27
Yes Ubuntu comes as an iso file.  You can burn that using Nero, I think nero has has burn iso/image options.

Then load the CD into a PC change the bios of the PC to boot from CD (this is done by holding down the delete key or the F2 key?) and the Ubuntu CD should load.


Try the 'Live' CD first Brian to get a feel of things.  Should you decide to be brave you have some options, either install directly or dual boot with Windows.  I would suggest backing up all data before doing either option.   

Good luck!   

Patience and an open mind is all that's required.  Remember Bri "Ubuntu IS NOT WINDOWS".

Trial it for at least two months (I think it deserves that), that should give enough time to ease the withdrawal symptoms enough for you to be able to have a more rational judgement of GNU/Linux.


Teeleef
