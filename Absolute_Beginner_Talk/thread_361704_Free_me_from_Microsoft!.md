### Post by freebeer on 2007-02-14
I think the general response to your situation is to install in safe mode.  I personally haven't had to do that, but I believe this mode is in text mode.  The problem you're encountering, I think, is a conflict with your video hardware.  It's trying to install in graphical mode, but due to the hardware/driver conflict, it can't.  In text mode, you bypass this particular issue.  Once installed (via text mode), you'll be able to configure the installation to work with your card and you'll have a graphical interface.

I haven't tried 6.10, so I'm unfamiliar with it, but I think this is the standard procedure.

You could also try downloading and burning the Alternate CD.  I've read many posts where people used this version to achieve the same results.

### Post by igknighted on 2007-02-14
I don't know if the live CD can boot from a USB drive.  My laptop has a PCMCIA cd drive that the live CD cannot boot from.  I needed to use the alt. CD.  Xubuntu is an option, but be aware that from a looks standpoint you may be dissapointed, as standard Ubuntu with gnome has many more graphical goodies.  I run Ubuntu with gnome on a laptop with very similar specs (256mb ram) and it runs fine, so I think that should be no issue.  Try the alternate CD, and try burning it at 4x, because faster burn speeds often lead to disc errors that could make it not boot, or cause issues once installed.
