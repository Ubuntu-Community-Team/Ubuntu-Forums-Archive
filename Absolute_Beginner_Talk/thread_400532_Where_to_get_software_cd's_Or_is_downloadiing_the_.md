### Post by pppetter on 2007-04-03
the repositories are on the internet, and no, sorry, there's no cd version. Right now I believe that the repositories contains about 170GB with software. 

Even thou you don't have broadband, you only need to download a program once to use it :) If theres alot of things you want to install you can always go to a friend, to work or school and download the packages you want there and burn it to a CD. 
everything is available on [http://packages.ubuntu.com](http://packages.ubuntu.com)
Just remember that you will need to download the packages dependencies also.

btw. most packages aren't really that big.
