### Post by aberry5555 on 2007-01-03
Test under Firefox and IE under wine and I would also strongly reccomend using the W3C validator at [http://validator.w3.org/](http://validator.w3.org/). As long as your coding conforms to this the you can shout at microsoft very loudly if it doesn't work, as IE's meant to conform to this too :p. both mozilla and IE conform to this, but use an occasional difference in terms of embedded objects, scripts and such though, generally, they both should read HTML fine if the syntax is correct. If you are using a linux wysiwyg program I would suggest learning how what you do on the screen converts in to code, by checking the code as you go along, as it makes it alot easier to fix your code to make sure it works properly ( I even do this on dreamweaver, you would have thought something that expensive would create error free pages but far from it! ).
