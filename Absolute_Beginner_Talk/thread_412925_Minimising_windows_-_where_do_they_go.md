### Post by jjbean on 2007-04-18
Do you have a Window List on the bottom tool bar? It is the white rectangle with the black line, right next to the desktop icon. See the attached picture. If not. Right click on the tool bar. Choose add to panel. Place a Window List, it is in the middle section titled Desktop & Windows. I figured this out when I nuked my tool bar. If the bottom tool bar is missing, right click on the tool bar that you have and select New Panel to add one on the bottom.
