### Post by atria on 2007-06-20
Azureus is good for torrents, though you need to install java6-jre-bin for it to work.

For edonkey networks, you have 2 choices. Either use emule with wine, or download amule with 

```
sudo aptitude install amule
```

amule does not support protocol obfuscation currently so if you need any form of protocol obfuscation, go with emule.

The other windows p2p programs should work fine with wine too. Good luck with it ;)
