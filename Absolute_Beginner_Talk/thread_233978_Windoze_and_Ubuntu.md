### Post by Tom Brokaw on 2006-08-10
Windows will write over your Ubuntu boot information.  You have two options that I can think of.  The first is to reinstall Ubuntu after you reinstall Windows.  If you don't want to reinstall Ubuntu, you could buy a new hard drive, install Windows on one partition, and then clone your Ubuntu install to the rest of the drive.
