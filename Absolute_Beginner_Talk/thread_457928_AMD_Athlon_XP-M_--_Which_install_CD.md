### Post by Tux Aubrey on 2007-05-29
If I was you, I would start with the Xubuntu 32 bit desktop live CD.  You are most likely to have a pleasant experience and be able to do everything you want.  The 64 bit version will work - but not everything is optimised for that platform and the performance boost is unlikely to be worth the hassle - unless you like that kind of thing.

I assume you have a reason for choosing Xubuntu?  With that processor and, say 500mb of RAM, you would have plenty of power to run Ubuntu or Kubuntu.

Welcome to the forums.
