Sounds like a problem with the CD then... so, things to do.... check the MD5 sums, info [here]("https://help.ubuntu.com/community/BurningIsoHowto") on that and basic burning. Long as the ISO is correct and matches, open up your burner and burn it at lowest speed, should be 4X. Then check for errors again with the sum. If the sums match up again and you reboot with no success, I guess its a hardware issue and you need the alternate CD.
