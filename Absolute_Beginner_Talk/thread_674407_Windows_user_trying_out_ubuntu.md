### Post by PurposeOfReason on 2008-01-21
With older hardware like that I would use fluxbuntu or xubuntu. Both of these version have a livecd (which I would recommend) and all you will have to do is put in the disk and click the install icon on the desktop. It will give you seven steps to install. The only one to worry about is partitioning. For a beginner, just use the option with the slider so you don't override any other OS on your computer. If you would like to do such a thing, just choose the use full HDD option.

### Post by ZylGadis on 2008-01-21
Ubuntu will run perfectly well on such a machine, and it is the officially supported version. You will have the least trouble with it (if at all - depending on your hardware). I would not recommend Xubuntu or Fluxbuntu to a complete newbie. You can install from the livecd, or, if you are not afraid of non-flashy graphics, you can install from the alternate cd (it will run better on your machine).
