### Post by Dr Small on 2007-02-08
> reinstalled firefox and reinstalled java (although i cannot make a shortcut from java to the firefox plugins, permission denied??  )
You need to have root access in order to create a shortcut to your java in the Firefox Plugins.
Try opening (Alt+F2) {Run} or the Terminal and typing:
```
gksudo nautilus
```

And then browse to your firefox/plugins folder to make the shortcut ;)

Dr Small
