### Post by psusi on 2006-01-05
If you install ubuntu packages through synaptic or apt-get then you can simply remove the package with those tools and all traces of it will be gone.  There is no need for something like "Go Back" for that reason.  

If you are compiling programs yourself, or messing with config files and screw thigns up, then you should have backups to restore from.  There's plenty of info between the forums and the wiki on backing up your data.
