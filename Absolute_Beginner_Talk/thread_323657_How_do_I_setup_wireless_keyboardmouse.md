### Post by morphet on 2006-12-31
This worked on an apple, but it was so beautiful and easy, it's worth a try for you as well. Go to System -> Administration -> Synaptip Package Manager, and search for "bluez", a package called Bluez-utils is installed. Click the box to mark it for removal and apply. It solved all my bluetooth issues. If it doesn't work for you, you can just click it to install it again. Goodl luck!
