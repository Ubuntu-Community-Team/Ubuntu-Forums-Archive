I might matter, it may not.  If you put hardly any data on the hard drive before resizing, than you are probably fine, which it sounds like you are.

If you can boot into Windows, than I guess you didn't lose anything "too" important.  However, you might run into trouble in the future if a file is missing that you need to complete a task.  Or, you may never run into any trouble whatsoever.

I would *guess*, based on you saying you barely put any data on the hard drive, that you are probably fine and won't run into any troubles.
