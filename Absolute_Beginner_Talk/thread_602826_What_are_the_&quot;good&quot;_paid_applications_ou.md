### Post by Frak on 2007-11-04
Xen is OSS, Professional version is not free.
Zimbra is OSS, Professional version is not free.
Wine is OSS, Professional version is not free. (Crossover is the Professional version)
WineX is OSS, Commercial version is not free. (Cedega is the Commercial version)
Fedora is OSS, Professional/Commercial version is not free. (RedHat is P/C version)
OpenSUSE is OSS, Professional/Commercial version is not free. (SUSE is P/C version)
Pixel is Proprietary, Current versions are free, stable versions will not be.
etc.

Not all Linux software is free.
