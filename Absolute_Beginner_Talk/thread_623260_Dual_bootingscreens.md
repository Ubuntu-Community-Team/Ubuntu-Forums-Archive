### Post by jken146 on 2007-11-25
Dual boot is completely supported.  You'll need to install both operating systems on different partitions (I believe Vista has a tool to shrink itself, which may be useful to you), but it is very very easy to dual boot.  You should plan for:
* a windows partition
* a shared partition for your data
* a / partition for ubuntu
* a linux swap partition at the end of your drive.

There is a partitioning tool on the Live CD and in the Ubuntu installer.

Dual screens are also supported I believe but I have no experience of that personally.  I expect there will be many threads in these forums about that though,

Welcome to Ubuntu, have fun!
