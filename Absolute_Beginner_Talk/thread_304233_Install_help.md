### Post by dbott67 on 2006-11-21
The good thing is that you don't have to 'remove' anything.  During the installation process, Ubuntu will ask if you want to use the entire disk.  If you wanted to keep Windows, you would answer 'no', however, just answer 'yes' and Ubuntu will take care of the rest.
