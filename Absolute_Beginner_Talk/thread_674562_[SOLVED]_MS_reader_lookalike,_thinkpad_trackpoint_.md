### Post by jleaker01z on 2008-01-21
1.  .lit reader:  Read this thread: [http://ubuntuforums.org/showthread.php?t=49232](http://ubuntuforums.org/showthread.php?t=49232)

2. Trackpoint: [http://ubuntuforums.org/showthread.php?t=49232](http://ubuntuforums.org/showthread.php?t=49232) (for gutsy)

3. USB card:  Should work no problem

4.  IM Client;  Yes...GAIM comes preinstalled with a default Installation.  There are others you could use also.  That would be a personal preference.

5.  Skype:  Yes

6.  Ubuntu default installation will have a firewall installed and enabled.  You wont find much need for an anti-virus running a Linux desktop, but I and most others who do have one use ClamAV.

7.  Burning CDs:  Yes

8.  Ripping CDs:  Yes, I use GRip

9.  Visual customization:  Yes, alot more than Windows.  There are 3 desktops (Gnome, KDE, and XFCE), the default Ubuntu install will have desktop effects (wobbly windows, transparency etc), or you could go with Compiz-Fusion, or Beryl for their effects.

10.  You will already have a Windows looking desktop.  There are 3 default 'start' buttons in the default Ubuntu install.  Applications, which is what starts installed programs, Places, which shows file managers, my computer, etc, and System, which is where you can set preferences and has a 'control panel'.  It makes more sense once you see it, much more sense than Windows throwing everything under the one 'start' button.

11. OSx Dock:  Yes.  Its called Gdesklets.

12. FF,  OO: Yes, all installed with a default Ubuntu install.  Thunderbird is optional.  Evolution is the default email program for Ubuntu.

13.  GUI vs CLI (Graphical User Interface vs Command Line Interface):  Yes...99% of things can be done with a GUI.  If you ask for help here, often you will get a CLI solution (its easier and faster for the person helping you, and is guaranteed to work regardless of how you have your GUI customized), but if noone else offers a GUI solution, specifically ask for one.  If you want GUI, I would recommend sticking with the default desktop, Gnome.

Welcome to the club!

### Post by jleaker01z on 2008-01-21
Oh and by the way, most any program you need can be installed from within Ubuntu.  It is quite different from the Windows way (visit a website, download the file, start the setup file, answer all the questions, click next, next, next, next, next, install, finish).  To install things in Ubuntu just go to Applications>Add/Remove.  Find what you are looking for, check the box beside it and click ok.  Much simpler than Windows.

> Actually, I don't want Microsoft, but here's the first problem: I'm in love with **Microsoft reader** and have quite a library of .lit files

Your problem is interesting in that it demonstrates the problem of getting oneself unknowingly locked into proprietary software.  Just think of all those folks downloading copy-protected music from the iTunes store and inadvertently locking themselves into a true **need** of iTunes. :-|

> My main concern though: I use an **IBM Thinkpad and rely on the TrackPoint**
(on an old machine, T20)
Does Ubunto automatically support that or do I need a separate driver? Where can I get it?

Your Trackpoint should work out of the box.  You might also be interested in [this]("http://ubuntuforums.org/showthread.php?t=645208") thread.

> Can I use** Trillian or another multi-messenger client?** What about Skype?
If possible, not Miranda... I've tried that before and wasn't too keen.
I need to run multiple instances of MSN, yahoo, ICQ (if possible, with webcam function), and now run Skype as a standalone application. 
If Trillian doesn't work, are there similar alternatives?

You'll be using Pidgin (used to be called Gaim) if you install Gutsy with a Gnome environment.  With KDE you'll be using Kopete.  Both work similarly to Trillian and allow you to sign into multiple accounts at the same time.

> What about a **firewall and anti virus software?** While Linux is safer, I'd feel naked without, what can I use?

Not that it's currently necessary, but I think Gutsy comes preinstalled with some sort of antivirus software.  Can someone else speak to this?

> Can I burn CDs like I do now- with the inbuilt function in Windows Explorer? If I need a separate program- which one?

With a default Ubuntu installation, you will be able to burn CDs straight from Gnome's file manager.  Serpentine (installed by default) can be used to burn audio CDs.  If you prefer to use a Nero-esque program instead, there are plenty of options to choose from.  Many hail K3B as the best, but you may not want to use it if you're operating in a Gnome environment.

> Next question is rather silly, but I'd like some answers anyway:
How much visual customisation is possible with Ubuntu? (for someone who doesn't yet quite know what they're doing)

Much **much** more than Windows allows.  Check out [Gnome-Look]("http://www.gnome-look.org/") for some ideas/tips on customizing your Gnome desktop.  There's also a KDE-Look and Xfce-Look if you find yourself eventually settling with either of those GUIs.

> Can I create a windows-looking desktop with icons for my main programs? Something like the start menu? Can I use custom icons and keyboard shortcuts? I find my way around easier with those.

Of course!  All of this is very easy to set up.

> Alternatively, can I have something that looks and works like the Mac Dock?
(I know someone whose obscure linux distribution looks entirely like a Mac...but he has way more of a clue than me :o) )

There are programs out there that, for all practical purposes, emulate the OS X Dock.  There are plenty of people in these forums who run such programs, so you shouldn't be short on resources.

> I guess I won't have to worry about Firefox, Thunderbird and Open Office running on Ubuntu, right?

Nope.

> In daily use, I'd definitely prefer much clicking and dragging and dropping over the command line- would be best if any program didn't "feel" much different from Windows.

The need for the command line in an average user's desktop environment diminishes more quickly every day.  You'll eventually find, though, that you'd miss it quite a bit if it were suddenly gone.

One more piece of advice...it sounds like you've thought a lot about making "the switch", but it doesn't sound like you've done as much research as you probably should (the questions about Firefox and icons/"start" menus are giving me this impression).  While it's always fun to dive headfirst into a new project and figure things out along the way, be careful not to get yourself stranded in unfamiliar territory without a lifeline.  You might want to dual boot with your Windows installation, at least for a while, in order to preserve a comfortable working environment.
