### Post by sleepingdragon on 2008-01-05
Ubuntu is gonna be nasty on that configuration. I'd personally go for something a lot lighter. DSL (Damn Small Linux) will run quite well on that setup and is supplied with Monkey Web Server to get you started on your project. 

I've seen DSL on 300Mhz, 64Mb RAM, 2Gb of hard drive - and not much else, so you should be OK on your specs. 

Ubuntu is based upon the 2.6 kernel, which is likely to be a bit heavy going for such an older machine. DSL runs on the 2.4 kernel, which is a lot more friendly for slower processors.

You can find DSL here: [www.damnsmalllinux.org]("http://www.damnsmalllinux.org"). There's a new version - 4.2. Nice interface :)
