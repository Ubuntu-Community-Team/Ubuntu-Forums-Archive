### Post by jamadagni on 2006-08-13
The development forum is closed except for the Edgy forum, but this is a general Ubuntu question so I put it here.

With SUSE, [a program called applydeltaiso]("http://en.opensuse.org/Download_Instructions#Applying_Delta_ISOs") is available which can convert ISOs of development versions - i.e. alpha, beta and RC - into later versions of the same release. For examples see for  [SUSE Linux 10.1]("http://download.opensuse.org/distribution/SL-10.1/delta-iso/") and [openSUSE 10.2 alpha 3]("http://download.opensuse.org/distribution/openSUSE-10.2-Alpha3/delta-iso/").

So I believe it would be useful for testers who have downloaded ISOs of development versions to simply upgrade them using such a method. Does Ubuntu have such a method or shall I file a bug report at malone?
