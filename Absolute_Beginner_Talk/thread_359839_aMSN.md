### Post by lucia_engel on 2007-02-14
Ok, I'm not sure if this is the default setting or not but here's how mine's setup.

Under Session:
select Reconnect on loss

Connection:
select direct connection

Advanced:
Connection - select Keepalive (NAT)

Also try posting your log file to the amsn forum.
