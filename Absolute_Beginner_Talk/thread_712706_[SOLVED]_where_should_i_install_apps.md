### Post by Shazaam on 2008-03-02
Did you use the Synaptic Package Manager to install it? System>Administration>Synaptic.:)

It depends on where you got the file. When I download a file from a website for installation I usually download it to my desktop. Others make a special download folder while others use other folders. As long as you know how to cd (change directory) in terminal you should be fine with the desktop. Here is a good read on installing different files......

[https://help.ubuntu.com/7.10/add-applications/C/index.html](https://help.ubuntu.com/7.10/add-applications/C/index.html)
