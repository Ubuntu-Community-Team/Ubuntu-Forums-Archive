### Post by jordanmthomas on 2006-12-23
in a terminal, cd to the correct directory
then, type
```
chmod +x filename.bin
```
Then, 
```
./filename
```

Alternatively, you may right click on the file, go to properties and then its permissions tab and set it to be executable.
Then, it's just a double click away.
