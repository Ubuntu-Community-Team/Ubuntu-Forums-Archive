### Post by NikoC on 2007-04-03
Not all programs get added to the menu automatically, you may create a new shortcut by pressing alt+f2 and typing:
```
alacarte
```
This will open the menu editor for gnome via which you can add and delete entries or select which programs are viewable in the menu or not!
