### Post by aysiu on 2006-11-05
There shouldn't be multiple graphical text editors in a default Ubuntu installation.

If you're using Ubuntu, there should be only one graphical text editor--Gedit.

If you're using Kubuntu, it's Kate.

If you're using Xubuntu, it's Mousepad.

You should also have Nano available as a text-only text editor for when you're using the terminal.

### Post by hearnden on 2006-11-05
Text editors are often something of a religious issue.  Emacs and vi are more pervasive than any Linux distro or flavour of UNIX, and while gthis and kthat come and go, emacs will live forever...

I think it's funny that now I use basic text editors (mostly emacs) for creating documents (with LaTeX), and powerful and bloated IDEs (Eclipse) for programming.  It's quite the reverse of what it was 5-10 years ago...
