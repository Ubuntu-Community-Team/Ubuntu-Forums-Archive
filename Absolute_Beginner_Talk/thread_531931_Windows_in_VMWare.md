### Post by AndyCooll on 2007-08-22
As one of the earlier posters mentioned, it depends on a number of factors. VMware (or VirtualBox for that matter) are excellent if you have a decent processor and enough RAM because they use a portion of your processing power. 

I have a VMware XP image and it runs as smooth as if it were the only OS on the machine. That's because I have 2gb of RAM and dedicate 1gb to the XP image. It even runs my game (Football Manager) nice and smooth. However this game isn't graphics intensive, it's a number cruncher. Most games run extremely poor in virtual environments because the virtual environment creates a "virtual graphics card" which is usually only bog standard level, it doesn't use the one that's in your machine.

Wine is a better alternative provided the app can run under Wine. Thankfully my game can so I now use this method instead and the XP image sits gathering dust. Otherwise, for games you'll probably need to dual-boot.
