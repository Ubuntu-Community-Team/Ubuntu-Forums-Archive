### Post by ThrobbingBrain66 on 2008-01-13
System > Preferences > Appearance

Drag and drop the .tar.gz file into the theme window and it'll ask you if you want to use that theme or keep the current.  To choose between the metacity themes that are already installed, click the Customize button and navigate to the Window Border tab.
