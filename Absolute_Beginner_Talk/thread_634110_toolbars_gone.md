### Post by i586 on 2007-12-07
Try what bumanie suggests, you've nothing to lose...

To install any 'official' nVidia driver do the following:

1. Download the driver from the nVidia site (it should be a *.run file).
2. Rename it to 'driver.run' (try to save it either in your /home or in your desktop). (There's no need to rename the file but it's faster to type).
3. Copy the following lines to a new file.

```

#!/bin/bash

chmod +x driver.run
sudo apt-get install -y linux-headers-$(uname -r) build-essential
exit 0

```4. Now, save it as a script file (myscript.sh) on your Desktop and in a terminal type:

```

cd Desktop
chmod +x myscript.sh
./myscript.sh

```Those lines will execute the script and install everything you need to compile the nVidia driver.

Once you've done that, please post back your results so we can go further.

Regards & Good Luck.
