### Post by Jucato on 2006-05-27
Could it be possible that one of those 4 or 5 apps that you wanted to install happened to be a package that is needed by the all the other KDE packages? Or it might be possible that removing some of the apps removed a crucial dependency upon all other KDE apps depended upon (a lib, perhaps). 

Anyway, Adept (Kubuntu-only, no other KDE distro has this), unlike Synaptic, doesn't really warn you with a dialog box regard things that will be installed/uninstalled. You can only find that info on the status bar or if you click on "Preview Changes". That's why I don't like it and actually use Synaptic on Kubuntu. Much safer that way. :D

(Don't know if Adept 2.0 on Dapper is any better. I hope it is).
