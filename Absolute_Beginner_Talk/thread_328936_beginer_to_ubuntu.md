### Post by meng on 2006-12-31
Looks like you need Crossover Office (a commercial product which emulates Windows) to run iTunes, you can't run it in Linux as such. Firestarter helps to configure your firewall, but you won't necessarily need this as port access is not granted by default anyway. I don't use antivirus software, many other users don't either; some do, but mainly to protect other Windows computers on their network.

### Post by iand675@gmail.com on 2006-12-31
Many users have reported success in running iTunes through the installation of Wine, which allows many Windows programs to be run in a linux environment, but there may be compatibility issues with the Windows programs you try to use. On the other hand, there are plenty of excellent music programs that have the same capabilities as iTunes except the large music store. Some users choose to use ClamAV as their antivirus program of choice, but I personally have never needed to use it. Of course I imagine that it would probably be a smart choice to have it just in case :)
