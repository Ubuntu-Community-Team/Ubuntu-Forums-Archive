### Post by think13 on 2006-11-03
I would say 'most likely' that it will install on your machine.  You can also look here [https://wiki.ubuntu.com/HardwareSupportMachinesDesktops](https://wiki.ubuntu.com/HardwareSupportMachinesDesktops)
One thing to note is that hardware from older machines is more likely to be supported, so if you have an old computer, I would recommend playing around with it on there.

And if you are an absolute newbie, it would be a good idea to at least try out Ubuntu a bit before you go all the way to a full install.  There is somewhat of a learning curve associated with Linux of any type.
