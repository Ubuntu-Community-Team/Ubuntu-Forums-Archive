### Post by aysiu on 2005-10-08
Yes, create a FAT32 partition and put all your files there. If you follow these directions

[http://www.ubuntuguide.org/#automountfat](http://www.ubuntuguide.org/#automountfat)

that partition will be mounted in Ubuntu every time you boot up. Windows will recognize the FAT partition as probably D: or E:
