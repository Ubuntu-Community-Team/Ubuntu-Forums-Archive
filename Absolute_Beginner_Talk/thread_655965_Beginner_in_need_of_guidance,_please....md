> **acosmichippo said:**

Up until now, i have just assumed that this would not work in ubuntu because of the MSN software needed to connect... is this true? You should be able to create a normal dialup connection if you know the number it dials, your username and your password.  I think I read that you have to add MSN/ before your username, so MSN/myusername.  You can get the MSN number local to you from [https://support.msn.com/accessnumbers.aspx](https://support.msn.com/accessnumbers.aspx).  p_quarles is right that it may be hard to set up a winmodem in ubuntu (many modern modems now rely on some features of windows to function, so special work-arounds have to be found for linux users).
