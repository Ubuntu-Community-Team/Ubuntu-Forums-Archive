### Post by irish_flu on 2007-10-02
Yeah, it's much easier than the manuals might lead you to believe.

These instructions assume you're using Ubuntu (and not Kubuntu or Xubuntu):

- Download (from Synaptic) "xchat-gnome".

- Start it up, it will be under the "Internet" menu in "Applications".

- Go to Edit ---> Preferences ---> General and set a nickname for yourself.

- Find out what server you want to connect to (such as irc.freenode.net, for the Ubuntu channels let's say).

- Go to --->IRC --->Connect and find or enter the server's name.

- Figure out what channel you want, and enter that into the command area at the bottom with the "/join" command.  For instance,

/join #ubuntuforums

- start pretending you're an expert about anything people might be discussing :lolflag:
