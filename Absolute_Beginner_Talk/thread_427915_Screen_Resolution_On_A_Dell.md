### Post by Happy_Man on 2007-04-29
There are two ways: manually editing config files so that 800x600 is an option, or using a terminal. Personally, I would recommend the former method, although it does require a good bit of trial and error (at least for me).

Enter this into your terminal:

```
sudo mousepad /etc/X11/xorg.conf
```

Scroll down until you find a bit of text that looks like:
```
Section "Screen"
```

It will have a list of resolutions under it. Just add "800x600" (Quotes and all) into the file wherever there are resolution numbers. Once you finish, save and exit. Then log out and log back in, and see if you can change the resolution. Hope this helps!
