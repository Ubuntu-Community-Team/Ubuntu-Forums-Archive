### Post by dsauxier on 2008-04-01
Any MP3 player that acts as a Mass Storage Device should work. 
(it should say so in the spec, it may say "usb drive" or "flash drive" instead of "mass storage device").
I have one from MeElectronics (the RockMe) that not only works on linux as a mass storage device, it also plays ogg files too.  (some of the earlier meelectronics devices don't play ogg.

I also had a Coby that worked, but it was poor quality so I took it back.

There is also an opensource project that writes replacement firmware for proprietary systems that don't support linux natively.  The [RockBox]("http://www.rockbox.org/") firmware can co-exist with the native firmware in a "dual boot" setup.
