### Post by Inxsible on 2007-06-09
Create a folder first
```
sudo mkdir /media/MyDrive
``` You can use any other name instead of MyDrive. Not using spaces will be a better idea or you will have to use quotes around them everytime.

Then add this line to the end of your /etc/fstab file
```
/dev/hdb1 /media/MyDrive ntfs-3g defaults,locale=en_US.UTF-8 0 1
```

Use the same name that you use in step 1
