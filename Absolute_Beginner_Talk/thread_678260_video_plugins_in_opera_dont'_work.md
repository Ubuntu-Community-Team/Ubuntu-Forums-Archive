### Post by obscur156 on 2008-01-25
Opera is known to have problems handling FLASH.

The only way i managed to make flash working is to use opera beta.
You can try if you dont mind using beta software.

Here is a version that i know that is working.
[http://snapshot.opera.com/unix/snapshot-1766/intel-linux/opera_9.50-20080118.6-shared-qt_i386.deb]("http://snapshot.opera.com/unix/snapshot-1766/intel-linux/opera_9.50-20080118.6-shared-qt_i386.deb")

Just make sure that you have the right plugin installed.
I am using "ubuntu restricted extras" for all the plugin and codec needed on ubuntu.You can install it with synaptic manager or add/remove or even the terminal.

Good luck and give feedback if you succeed or not.
Best regards.
