### Post by agger on 2005-10-05
I'm running Ubuntu, Hoary with GNOME.

I thought I was smart and it would be cooler if I removed the GNOME panel in the bottom in the default install, but now I find it very difficult to navigate between windows. How do I get it back?

Yes, I know I can make a new panel, but how do I enable the small pane for each window which one can use to switch between windows and check which windows are open?

---

### Post by Ampersand on 2005-10-05
Right click on the top panel, and then 'new panel'. Right click on that panel, 'add to panel' and add a Window List.
