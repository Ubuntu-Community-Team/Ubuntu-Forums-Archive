### Post by sYs^ on 2006-04-10
My Documents is perhaps /home/your_username/

ProgramFiles are installed to different locations:

/etc/ = configuration files
/bin /sbin /usr/bin = executable files
/lib /usr/lib = like dlls in windows
/tmp = like temp in windows

More informations:
[http://www.freeos.com/articles/3102/](http://www.freeos.com/articles/3102/)
