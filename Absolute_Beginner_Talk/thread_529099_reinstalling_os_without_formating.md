### Post by zero244 on 2007-08-18
One way to make a fresh install easier to setup is when you first install Ubuntu create a separate partition for your home folder. In other words create a partition for Root and a second partition for /home and you will have to create a 3rd partition for the swap file.
Now when you do this, if you have to reinstall Ubuntu much of the configuration stuff will remain intact.
You will still have to setup and install some things but not as much if you did a format and then a clean install.
If you set it up this way just make sure when you do a reinstall to not format the /home partition.
By setting up Ubuntu this way you can have music, photos etc in the /home partition and they will remain intact.

Good Luck
