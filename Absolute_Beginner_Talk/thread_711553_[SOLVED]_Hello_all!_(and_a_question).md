### Post by glennric on 2008-02-29
The ubuntu-desktop package is a meta-package that depends on many packages that are part of the default ubuntu installation.  It can be removed if you want to remove some of those packages that it depends on.  It will not uninstall any of those packages if you remove it.  It is not recommended to remove it for upgrade purposes.  I do, because there are several things it depends on that I don't want installed.
