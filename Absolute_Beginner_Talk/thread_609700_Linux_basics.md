### Post by PmDematagoda on 2007-11-11
1) If you literally mean Services, they can be found in System>Administration>Services.

2) Usually Ubuntu does that automatically when it concerns .deb's or installing through the repositories, I'm not really sure about compiling them from the source code but I believe I have seen some cases where the manually compiled software is automatically added to the menu or the desktop. 

And if you wanted to put things on the desktop, just remember, the desktop itself is just a simple folder which is found in your Home directory, so anything you put into the Desktop folder is automatically added to the desktop.

3) You can check the amount of memory you have using a Windows Task Manager-like(But better) software called System Monitor which can be found in System>Administration>System Monitor.

I hope this helps you:).

By the way, welcome to the forums and to the world of Ubuntu:).
