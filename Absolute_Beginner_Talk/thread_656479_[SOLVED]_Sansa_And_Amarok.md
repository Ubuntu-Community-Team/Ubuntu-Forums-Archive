### Post by JackS on 2008-01-08
Mount point:
Before you connect the Sansa to the computer, make sure it is
in the MSC mode.  (Or Rhapsody mode for an e250r model.)

Node location:
Ubuntu auto mounts it as a usb storage device at "/media/SANSA E250".
cd to the /media directory and find the actual name of your device.

I've been using a Sansa e250r for a couple months.  It plays nice with
ubuntu and Rockbox.


This url came up when I did a web search for "ubuntu sansa".  While
it does not answer all questions, it does cover quite a bit of ground.
