It's not a good idea to give yourself full permission to everything, it can break things pretty bad sometimes try this instead
[http://ubuntuguide.org/wiki/Dapper#How_to_browse_files.2Ffolders_as_root_user_in_Nautilus](http://ubuntuguide.org/wiki/Dapper#How_to_browse_files.2Ffolders_as_root_user_in_Nautilus)
follow the instructions in the link this will create a root launcher in the menu and when you need to create/delete/move files as root you can just use the launcher.
