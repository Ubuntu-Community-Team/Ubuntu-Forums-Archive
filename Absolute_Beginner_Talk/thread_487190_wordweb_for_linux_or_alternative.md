### Post by xelnaga666 on 2007-06-28
well im not sure about ubuntu or any linux distrobution for your question. But I tend to use google for things like this.

Go to [http://www.google.com](http://www.google.com) and type

```

define WORDTODEFINE

```

Im thinking, if your using firefox as your browser, the google search in the top right would be a method of doing this.

EDIT: Theres a program called "dict". Its in the ubuntu repos so do a ```
sudo apt-get install dict-gcide
```& ```
sudo apt-get install dict
```
You run the app by doing a ```
dict WORDTODEFINE
```

if you do a ```
sudo apt-cache search dict
```You will see the various extra packages for it too.

Hope this helps.

### Post by p_quarles on 2007-06-28
Ubuntu does come by default with a dictionary program in the Applications >> Accessories menu. If it's just not comprehensive enough, there are larger versions. For American English, for instance, there's the package wamerican-huge. Equivalents are available for British and Canadian English, and you can install all of them if you want to.
