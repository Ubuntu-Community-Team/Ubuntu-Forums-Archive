### Post by filam on 2008-03-09
What version of Ubuntu are you using? Make sure you have *Advanced Desktop Effects Settings* installed. This can be found under System > Preferences. Once it is installed (check Synaptic) you can access the transparency under Desktop Cube > Transparent Cube.
