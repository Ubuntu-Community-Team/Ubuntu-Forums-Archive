### Post by FuriousLettuce on 2006-08-30
Lad, have you done this in a painting program rather than a programming language? Because if you have, and all you want to do is to make a nice theme / look for an existing OS, then go for it, and good luck.

Writing an OS from the ground up is very hard work. You need a very thorough understanding of the relationship between hardware and software, as well as an intimite knowledge of the inner workings of the software itself.

It's a good a noble dream, but you're not going to get there by painting some pretty windows. If you really want to look into it, start by looking at basic kernel code, or anything along those lines.
