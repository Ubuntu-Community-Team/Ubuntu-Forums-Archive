### Post by Rocket2DMn on 2007-07-13
You can copy in the whole file by opening it graphically and copy/pasting:
```
gksudo gedit /etc/X11/xorg.conf
```
or just view the whole file in terminal, selecting it all, and right clicking to copy:
```
cat /etc/X11/xorg.conf
```

Please post that and include information on your video card and monitor please (make/model).
