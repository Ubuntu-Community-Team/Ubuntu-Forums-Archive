### Post by Ocxic on 2007-07-03
try going to System->prefrences->screen resolution
and changing it there, it also has the option to make a default res.
I'm thinking that the nvidia-settings, are seperate from the actual settings on your computer, as nvidia's application allows real-time interaction with the video card/drivers, and doesn't acctually make all changes permanent.
