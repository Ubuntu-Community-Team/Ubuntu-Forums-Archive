### Post by Bachstelze on 2007-03-11
Does the nvidia logo appear when X starts ? If so, we can move further, run this :

```
sudo dpkg-reconfigure -phigh xserver-xorg
```

You should have a list of all available resolutions. Select the one(s) you want then when it's done updating your config, save your work if any, close all opened apps and press Ctrl+Alt+BkSpace to restart X. If all went well, you'll have the desired resolution.

### Post by Bachstelze on 2007-03-11
As I thought... See those lines ?

```
		Modes		"1024x768" "800x600" "640x480"
```

That's the resolutions your X server uses. As you can see, the one you want is not in there. Edit them like this :

```
		Modes		"1650x1080" "1024x768" "800x600" "640x480"
```
