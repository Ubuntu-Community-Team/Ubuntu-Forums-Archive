### Post by btoone on 2007-05-10
Katapult seems to grab the first thing that matches what you are typing because if I start typing "firefox" I get "Customize Firefox".  I found that typing "web" will bring it up for me.  I'm pretty sure you can type any part of this string "Firefox Web Browser" and you'll get it.  So for example you could also type "firefox web" and you'd get it or even "browser".  It would be nice if there was an easy way to configure/customize Katapult.

--update --
I just found this [thread]("http://ubuntuforums.org/showthread.php?t=199226&highlight=katapult+catalog").  Author mentions that Katapult will display the items in the KMenu.  This made me realize that the reason I have to type "firefox web browser" is because that is how its listed in my menu.  So I edited the item in my menu and changed its name from "Firefox Web Browser" to just "Firefox".  Now I can launch Firefox from Katapult by typing "firefox".  Horray!
