### Post by HermanAB on 2007-10-14
For dual booting, install Windows FIRST and Ubuntu second.

For dual monitors, use Xinerama.

Dual booting is however rather 20th century.  The modern way is to install some kind of virtual machine: Virtualbox, Qemu or VMware and then install Windows inside that.  The advantage being that you can run everything at the same time.  This also allows you to experiment with various operating systems, without damaging your base system.
