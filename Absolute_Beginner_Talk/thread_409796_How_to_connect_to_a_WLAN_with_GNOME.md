### Post by ayenack on 2007-04-15
All you have to do is go to System>Administration>Networking type in the name of your network in Network Name ESSID (SSID) select password type, type in password (your Network Key) in Network Password. That's it, easy.

Best of luck. ayenack.

### Post by ayenack on 2007-04-16
If you want a network icon on the task bar do this.

Right Click on task bar a menu will pop up Left Click on Add To Panel Scroll down there should be a Network Monitor Double Left Click it (it will appear on task bar looks like two little monitors) Right Click this a menu will pop up Left Click Properties and in Name type in the name of your Adapter or Interface e.g. rausb1 and that should be that.

Best of luck. ayenack.
