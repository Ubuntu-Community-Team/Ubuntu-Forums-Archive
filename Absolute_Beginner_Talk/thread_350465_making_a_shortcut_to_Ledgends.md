```
cd /usr/games/ledgends
./runledgends

### Post by Buck2348 on 2007-01-31
On the Desktop, right-click and select "Create Launcher."  In the dialog that opens, type whatever you like for "Name," and the command with full path in "Command." ```
 /usr/games/ledgends./runledgends
```  Click on "No Icon" and choose an icon--you can find more icons in /usr/share/icons.  Click OK.

If you want the launcher in the panel rather than the Desktop, right-click on an empty area of the panel, select Add to Panel --> Custom Application Launcher and use the same procedure as above.

Buck

### Post by Brunellus on 2007-02-01
one last thing:  these things are called "launchers."

I discourage the use of "shortcut," because it's one of those terms that confuses more than it clarifies.  Microsoft, particularly, has used it to mean:

* a URL
* a bookmarked URL
* a pseudo-symbolic link
* an application launcher

and those are only four instances.  There are likely more.
