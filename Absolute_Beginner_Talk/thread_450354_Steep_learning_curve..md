### Post by LaRoza on 2007-05-21
If you are just learning, as every one did, try using the live disk first, then dual-booting, then installing.

For Ubuntu, use 7.04 Feisty. You can install, run live, and dual-boot easily.

When you do install, you can edit partitions and format without trouble (it does it automatically), it will also let you dual-boot without hassle.

### Post by Lucifiel on 2007-05-21
> **LaRoza said:**
> If you are just learning, as every one did, try using the live disk first, then dual-booting, then installing.

For Ubuntu, use 7.04 Feisty. You can install, run live, and dual-boot easily.

When you do install, you can edit partitions and format without trouble (it does it automatically), it will also let you dual-boot without hassle.

Okay, to clarify on what LaRoza means by "dual-booting", this means that:

Windows XP has to be installed first and then, Ubuntu. :) Also, if I were you, go download the Live CD first and then play around with it to get a feel for Ubuntu. Based on your own learning style, adjusting to Ubuntu can take between less than 20 mins to a few hours or days. 

And don't worry, the Live CD runs off your system's memory so nothing will be installed. :)

Also, finally, I'd make a list of all installed hardware and find out which is compatible with Ubuntu and which is not.  And also, a list of all software I use within Windows and the possible replacements for it. If I were you, save all your results into a .doc file so that you can look it up after having installed Ubuntu.
