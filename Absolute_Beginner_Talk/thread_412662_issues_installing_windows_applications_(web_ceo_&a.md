### Post by lotacus on 2007-04-18
From one windows user to the other: 

I agree. To ease your transition, I would install windows under a virtual machine such as Vmware or VirtualBox.

You can also try using CrossOver. They have a handful of supported win32 apps which include the microsoft office suite and IE6. However, firefox runs just great compared to IE after you get all the plugins installed.

However, if there are applications which you require that are only available to windows, I would stay away from emulators and use a virtual machine.

Good luck on your linux adventure :) You will pat yourself on the back when you realise how much you've learned, and just how much you took windows for granted. :)
