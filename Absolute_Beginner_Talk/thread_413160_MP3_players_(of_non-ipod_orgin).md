### Post by jem7v on 2007-04-18
Honestly, pretty much any USB-connected MP3 player is going to work with Ubuntu.  All I do with mine is plug it in and it's recognized.  You shouldn't even have to install any extra drivers.  As for getting .wma files to work, DRM could be a problem but if it ain't there all you need to do is install the codecs for them... and luckily, all the codec help you'll probably ever need is condensed to the same how-to page!  Hurrah!  [https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)

Amarok is a good program for managing and playing music in Linux.  I think you might be able to do the whole "syncing" thing with it, though I've never tried so I can't honestly be certain.
