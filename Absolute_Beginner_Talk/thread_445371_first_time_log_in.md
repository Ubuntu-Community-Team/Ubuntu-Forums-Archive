### Post by Dr Small on 2007-05-15
When at the grub bootup menu, after you turn the computer, it should allow you choose Ubuntu to boot up in. If it doesn't press Esc or something, (it should tell you) to access the boot menu. Once there, select Ubuntu Recovery and follow the instructions above from there.
