Hey fodder,

Never fear, there's an easy answer for everything you want to do.  May I suggest Gnomebaker as the tool you might want to use to burn ISO CD's?  I'm kind of a newbie myself and I've had nothing but good luck with it.  To get Gnomebaker, use the Synaptic Package Manager... on the top menu bar of Ubuntu click on System --> Administration --> Synaptic Package Manager.  Type your password in when it asks you and then you're in.  Once inside click on settings --> repositories to get into a screen where you can enable all the online repositories for Ubuntu to look for Gnomebaker.  To enable a repository highlight it and click on "Add".  Once you've added all the repositories go back to the main Synaptic Package Manager page and click on the Search button.  Type in Gnomebaker and click search.  When Gnomebaker comes up, click on its box in the right hand panel and select "mark for installation", then click "apply" at the bottom.  When it finishes installing close the package manager...you will find an entry for Gnomebaker under applications --> sound and video.  Start gnomebaker and click on Burn ISO CD Image at the top.  Post back if you have any problems.  Good luck!!
