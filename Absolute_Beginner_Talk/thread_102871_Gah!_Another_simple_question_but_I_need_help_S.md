### Post by 23meg on 2005-12-13
Not a regular CD or DVD project from scratch; you should use the option to burn an ISO file as it is. In other words, you don't just put the ISO file into the project as the only file there, you choose the appropriate option in your program to "burn an image". You won't need to specify if it's bootable, etc.
