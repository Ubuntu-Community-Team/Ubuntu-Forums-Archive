### Post by Happy_Man on 2007-05-18
You can dual-boot.... that way, you can have both Windows (with all your Windows apps) and Ubuntu on the same drive... makes transferring so much easier. Or, there is WINE, it allows Windows programs to run in Ubuntu.

### Post by chamberlain2006 on 2007-05-18
