### Post by p_quarles on 2007-08-11
> **Malevol3nce said:**

Okay, I getcha now. (FWIW, the current version of Ubuntu names hard disks sda-sdz = SATA disk a - z). 

Honestly, my best guess here is that your CD drive is failing. I mean, it sounds like you've installed before, and you've tried multiple disks . . . so I'm guessing that's the problem. 

To find out, though, you can try a couple things. One is to load the installation .ISO onto a USB flash drive and install it from there. Another is to install from a network drive. There are guides to both of these methods in the "Tutorials" sub-forum. Of course, both of these require that your BIOS is compatible with those boot options.
