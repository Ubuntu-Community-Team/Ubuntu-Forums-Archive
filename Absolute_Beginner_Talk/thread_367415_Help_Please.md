### Post by george29 on 2007-02-22
hmmm... what exactly have you done? are you currently using windows and wish to try ubuntu?

have you just downloaded the ubuntu liveCD image and are having problems burning it to a CD?

what you need to do is download the image - this will have a .iso file extension. Burn this onto a CD using the burn image option if you are using nero at 4x speed (too high a burn speed can result in a bad burn).

Then reboot your pc, make sure the bios is set to boot from cd and the pc should then boot  from the CD into an ubuntu desktop.  From here you can then install ubuntu onto your harddrive if you wish.

VERY IMPORTANT - make sure you have backed up all your important data, files, music, pics etc - otherwise if you make a mistake you could lose the lot.

there  are lots of other guides which you can look at on the forums and on the web for how to prepare hard drive partitions for you to install ubuntu... read them carefully and think twice before you do anything.

### Post by george29 on 2007-02-22
> **errorz said:**
> So, I need to download it..Then burn onto a CD and then run it?

yes run it by rebooting your computer with the bios set to boot from CD

 - it's the liveCD - that means you can try ubuntu without changing anything on your harddrives.

Be aware that it will run more slowly from the CD than from a Hard drive... this is because the data read speed for CD is much slower than that for a hard drive. if you like ubuntu you can install it to your harddrive using the built in installer utility... there are how to's on the forum and web for this. 

Again - make sure you understand how partitions etc work if you want to keep your windows installation and make sure you back up all your data etc before you start installing ubuntu or playing with partitioning your hard drive.
