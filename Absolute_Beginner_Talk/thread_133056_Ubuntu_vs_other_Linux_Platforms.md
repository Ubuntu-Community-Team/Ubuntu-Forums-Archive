### Post by az on 2006-02-19
[QUOTE=Ant01] Do they all run the Linux software programs or are each specific to different applications. Please can someone give me advise so to short cut this learning curve?[/QUOTE]

Big picture - yes they do.  They are all developed in an open source fashion which means the code belongs to everyone and it evolves as people use it.  The interface for development is at the source code level.

All programs are written by people as text and this text (the source code) gets turned into the running program (binary).  

You can always get the source code for any application (as well as the stuff that it uses to run) and compile an application for your box.  An application that has been compiled and packaged for distribution in binary form is not usually able to run on another linux distribution.  This is because they use different versions and options of the shared libraries.

However, if you stick with the packages in the repositories (It's all built for you) you will be hard-pressed to find anything you need to compile yourself.  Ubuntu is based on Debian and most of the software that anyone could want to use (over 13000 packages) is packaged for debian by someone (one of the thousand debian developers) and is available to you in Ubuntu.

For software, use synaptic.  Read add the Universe repository to your list of repositories in use (see synaptic) and you will have access to more free-libre software than you know what to do with.  You install them by pointing and clicking on the,  Synaptic downloads them and installs the packages for you.
