### Post by floke on 2007-02-24
Hi there,

(a) have you checked the CD (select 'check CD' option from the menu)
(b) unfortunately the LiveCDs don't work on all PCs - this says nothing about the ability of the PC to run Kubuntu, it's just the way of the world - my PC runs ubuntu LiveCDs fine, for instance, but other LiveCDs (eg Mandriva, some versions of Knoppix etc.) do nothing.

If (b) is the case for you, then you can try running a different LiveCD - eg Xubuntu - or to install you will have to use the alternative installer - from the download site select the alternative install iso.

If you want to use a graphical partitioner before doing the install, you can get this

[http://www.sysresccd.org/Main_Page](http://www.sysresccd.org/Main_Page)

...which has the GParted tool on it (the same one that's on the LiveCD) - so you can resize your partition etc. and then use the alternative install to put Kubuntu on the PC.

Be sure to run a thorough defrag and scandisc in Windows before resizing it though! And work out how much space you want to give Linux before partitioning - i.e. how much space you want to leave Windows and how much for Linux (check 'properties' on your My Computer 'C' drive option to see how much space Windows is using and how big the hard drive is).

Post back if you have any problems

Good luck
