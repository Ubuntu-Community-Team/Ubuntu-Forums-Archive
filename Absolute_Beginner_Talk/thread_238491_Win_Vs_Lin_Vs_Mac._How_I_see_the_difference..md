### Post by Solver on 2006-08-17
In Windows, it's often a good thing that you can be sure other people will open files in Microsoft formats - .doc, .xls, etc. Whereas you can't make a document in OpenOffice and expect a random person to open it.

I disagree about your Linux weak point listing. Windows, in fact, requires codecs just like Linux does, so you have to install something for most media anyway. Being not compatible with Windows programs isn't a weakness, it just means it's not Windows. The problem usually is incompatibility with games - except games, there are Linux alternatives for almost any program.

As I see it, strong points of Linux, as opposed to Windows:

Command line. Once you learn to use the console and the most useful commands, you begin to appreciate how useful it is. Plus there's Bash scripting. On Linux, you can easily do tasks such as appending "-2006" to the name of each JPG file in a folder. In Windows, you would have to do it manually, or write your own program.

Ease of backing up settings. You can tar your home folder, then untar it on a fresh installation of Linux, and you will have your settings back, including application-specific stuff such as emails, browsing history, etc.

Customization. Well, this goes without saying. Linux not only is very customizable through different programs and config files, it's open source - meaning, if you have the knowledge, you can change absolutely anything.

Stability & security. Linux is a very stable OS, and when programs do crash, they very rarely take anything else with them. Linux is immune to Windows viruses, spyware and security exploits - hence, a Linux user running a standard desktop workstation is practically secure from Internet threats with a default install.

Weaknesses of Linux would be:

Necessity of using the console & config files. This is not a weak point for everyone, definitely not, I like having to use the console, but for some people, it can be a weak point. Even the desktop-oriented, user-friendly Ubuntu require some console interaction. This is mainly a problem for people new to computers in general - otherwise, anyone who has copy & paste skills should be able to cope.

Problem diagnostic. This is probably because I'm not an advanced Linux user yet, but it seems to me that Linux problems are harder to diagnose. There's excellent logging, but there's a lot of commands or log files to check if something is wrong. I'm sure advanced users will disagree.

Poor gaming support. A known fact, and a real weakness. There are few popular games that have Linux-native versions, and having to work with Wine or Cedega can be hard.

### Post by statue on 2006-08-17
Windows Pros: 

1. Very small learning curve, basically anyone with average intelligence can do just about anything Windows is capable of with minimal effort.

2. Hardware/software manufactuers typically aim at building/programming their products solely for Windows thereby ensuring the greatest "out of the box" detection and usability.

Windows Cons:

1. Requires a fairly decent setup just to run smoothly. Older computers with small amounts of ram,hard drive space etc will not suffice.

2. Security is a joke. Not only viruses and spyware but Windows gives the equivilent of root access to it's users so just as Windows is easy to learn it's equally easy to screw up. It doesn't protect the user from himself.

3. Updates must come from Windows alone as their code is not open source which means spans of time between someone finding a usable exploit (there have been many) and Windows getting a patch out.

Linux Pros:

1. Free...obviously

2. Open source, everyone who wants to alter anything, can, which helps speed up the overall development.

3. Much more powerful from server point of view. Any type of server: web,mail,database, etc can be run from Linux...Windows XP wasn't made to, nor should it, run services such as these.

4.Can be run on older systems. Distros such as Puppy and DSL can run fine on bare minimum of hardware.

5. Choice...don't like Gnome? use KDE...don't like Slackware...use Ubuntu. 

Cons:

1. Let's be honest, support from hardware manufactuers is lacking as well as commercial application companies. Major applications such as Photoshop were not made to run under Linux and until Adobe along with other companies finally start to port their widespread products to Linux, this will continue to be a big issue...and as mentioned above, that goes for gaming companies as well.

2. Steep learning curve depending on distro, but even more user friendly ones like Ubuntu still take more time to learn than Windows for the average user.

3. Addictive...I need a support group...help me.

So all in all I would say the best thing Windows has going for it is it's wide influence over software and hardware manufacturers to make things easy for Windows users and the best thing Linux has going for it is it's flexibilty. ie : choice
