### Post by msak007 on 2006-08-26
Like they said Firestarter is a GUI front-end for iptables (the kernel's built-in firewall), it lets you configure the firewall graphically. You don't need the actual program running or the System Tray icon unless you want to make changes, as the firewall itself is always running in the background. If you do want application and the System Tray icon to always show up on startup, follow the [FireStarter FAQ]("http://www.fs-security.com/docs/faq.php").
