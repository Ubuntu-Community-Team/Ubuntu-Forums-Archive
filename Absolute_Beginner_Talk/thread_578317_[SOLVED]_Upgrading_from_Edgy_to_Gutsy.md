### Post by ramjet_1953 on 2007-10-17
Upgrading is always a minefield anyway.

Much better to do a clean install of any new release.

It is always a good idea when doing an install to set up a separate partition for /home.

That way you can do a clean install and still have most of your settings, mail info, etc inact for the new install.
