### Post by BarfBag on 2007-02-10
> **a.v.l said:**
> I've made the move to Ubuntu on two of my computers and the last thing I need to do before installing Ubuntu on a third computer is to create my windows XP website onto Ubuntu using NVU. As long as it's pure HTML code (hand coded - not done by a program like Frontpage or Dreamweaver), all you have to do is [mount your Windows partition]("http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_mount.2Funmount_Windows_partitions_.28NTFS.29_manually.2C_and_allow_all_users_to_read_only") and copy and paste the code into NVU.

FTP works just like it does in Windows.  It's in Places -> Connect to Network Server.
