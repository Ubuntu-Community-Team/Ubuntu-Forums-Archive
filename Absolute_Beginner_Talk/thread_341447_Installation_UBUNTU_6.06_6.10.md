### Post by kebes on 2007-01-18
It's certainly worth a try. If you download/burn the Ubuntu install disk, it also acts as a LiveCD. So you can try the LiveCD and if it boots up properly (and recognizes hard drives, etc.) then you can be quite confident that the final install will work too.

The main thing is that your motherboard BIOS needs an option to boot from CD-ROM.


The thing that I would worry about is that such an old computer might not be up to the task of running Ubuntu. It may install but it will be painfully slow... You might also want to consider xubuntu (which is optimized for being lightweight) or maybe even "damn small linux"... which is a very trimmed-down Linux distribution.
