### Post by H.E. Pennypacker on 2007-07-02
I can't address your performance issues, but Ubuntu should definitely be as fast as Windows, if not faster (since Ubuntu doesn't need all the resources that Windows needs).  Something has to be wrong if that is not the case.

What do you mean double-clicking is not working?  If you double click on a folder, it should open.

To learn how to install stuff, see this:

[http://www.monkeyblog.org/ubuntu/installing/](http://www.monkeyblog.org/ubuntu/installing/)
