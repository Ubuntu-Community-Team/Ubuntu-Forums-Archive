### Post by Sparkster185 on 2007-05-29
These forums are going to be a little biased ;), just so you know.

I would honestly recommend going with Ubuntu Feisty.  Your decision should be if you want GNOME (Ubuntu) or KDE (Kubuntu).

Removing programs (as long as you install them with a package) is as easy as uninstalling them.  You simply tell the package manager (Synaptic, Adept, etc) to remove the Program and it will.

Try searching the Ubuntu package lists (packages.ubuntu.com) for the applications you want.  I know that Firefox, Thunderbird and Gimp are in the repositories.  A few mouse clicks and they are installed.

If you don't find the app. you want in the official repos, just try searching for a .dpkg installer.  It will work on Ubuntu, since it's based on Debian.

### Post by floke on 2007-05-30
Apparently it was created from scratch! Uses PiSi package handling ('packages install succesfully as intended'), a python-derived boot script (the fastest boot up I've ever seen!); and a security/task handling system called Comar (pronounced chow-mar) - in which you don't have to enter passwords for GUI operations (eg add/remove progs), but only in the terminal. It uses KDE and looks great (see screenshot). There are seperate Live and install CD's, so you cannot install off the live version. Its also one of the easiest (if not the easiest) installations I've ever done. Gives you the option to install grub to seperate partiton rather than overwrite mbr, which is nice! Also uses the SUSE-style kickoff, which works well (and can be changed to K-menu). Only real downsides are the small community, and the amount of applications in repositories, although both are growing all the time.

Definitely worth trying IMO.

** EDIT ** Forgot to mention that its a state-sponsored distro too - destined for use by the Turkish government! It's made by the Turkish Scientific and Crypotology Institute.

sudo apt-get remove PROGRAMNAME
```
