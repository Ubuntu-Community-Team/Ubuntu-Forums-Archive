### Post by taurus on 2007-02-05
You need to change to the directory where you downloaded it.  Then, you can unpack it and change to the new directory.

```
cd ~/Desktop
tar xvzf filename.tar.gz
cd filename
```

Look at Section 4 of this guide.

[http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)
