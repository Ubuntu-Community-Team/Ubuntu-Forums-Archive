### Post by bobpur on 2006-03-15
I used two different Harddrives when I installed Ubuntu. When you put in the install CD, it'll show a screen that should showw both drives. One as master and one as slave. In this computer, windows was on the master drive. When prompted, I selected erasing the slave and installing ubuntu there. It was just a matter of pushing the right buttons in response to questions asked during the install process.
