### Post by ChrisTheGeek on 2005-12-02
Different drivers have different performances.  It could simply be that the linux driver isn't as optimised as the Windows one.  However, it sounds more likely that the linux driver is printing in a higher default resolution than you were on Windows.

Unfortunately the built in print options are a bit limited, although I'm sure you can change the resolution.  Try playing with the built in printer options and see if you can speed it up.

If that doesn't work, check out a program called TurboPrint that provides Linux printer drivers with more options.  There is a free demo of the program available for download.  The full version of the program does however cost a bit of money.
