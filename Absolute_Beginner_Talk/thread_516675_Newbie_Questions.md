### Post by Hospadar on 2007-08-03
you should look into Gimp, as someone mentioned to replace photoshop, the versions in the ubuntu repositories are a little old, but i hear the ones from the gimp website can easily replace photoshop.  Try inkscape to replace illustrator, does great vector graphics editing.  for web editing, you might look into bluefish, quanta, or n|vu

Flash works fine natively in linux, and if you want to run stuff under wine, you should google "wine appdb" and have a look to see if it is posted there, a lot of popular stuff is listed with good walkthroughs to get it working.  Other than games though, I generally find it's better to find native linux replacements than to use things through wine, although for some specialty apps this is not an option.  I've generally found most normal apps to run just fine with wine.  Ususally it's when you get into strange 3d stuff and really huge or hardware dependant apps that stuff gets wierd.

I run team fortress and hl2 through steam and I'm sure counterstrike will run great as well (assuming you get your graphics driver working which should be no biggie with a modern nvidia like you have).
