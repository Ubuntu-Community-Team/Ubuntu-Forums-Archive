### Post by Kurt` on 2006-03-21
Make sure you burn the ISO file as an IMAGE. If you burn it as a data disc, you will end up with the just the ISO file on the CD.

If you are using Nero, select Image/Saved Project, pick the ISO file and hit Burn. For other programs, look for a Burn CD Image option.

### Post by Kurt` on 2006-03-22
Go to [http://www.ubuntu.com/download](http://www.ubuntu.com/download)

Pick a location close to you.

Assuming you're using a 'standard' PC, click **PC (Intel x86) install CD**

Download and burn a disc, then simply place the disc in your drive and reboot. Ubuntu should boot up into a graphical environment right off the cd. It's a good way to make sure all your hardware will be automatically detected and installed in Ubuntu. :)
