### Post by benuski on 2006-08-07
I know that you can hook up your ipod and Rythmnbox will read it, and AmaroK might be able to put songs on, I'm not sure.  The program that I like the best, even though its still in beta, is Banshee.  It allows you to import music in ogg vorbis (yay free formats), but then it will put the music onto your ipod or onto a CD in mp3 automatically, without you having to convert it.  Although the convertor is slow, I really like that built in functionality.

### Post by ciscosurfer on 2006-08-07
Try Rhythmbox.  You can look at its homepage [here]("http://www.gnome.org/projects/rhythmbox/").  If you want the latest *stable* version plus some frills, enter this into a terminal```
wget http://people.ubuntu.com/~seb128/deb/rhythmbox_0.9.4.1-0ubuntu1_i386.deb http://people.ubuntu.com/~seb128/deb/rhythmbox-dbg_0.9.4.1-0ubuntu1_i386.deb
sudo dpkg -i rhythmbox_0.9.4.1-0ubuntu1_i386.deb
sudo dpkg -i rhythmbox-dbg_0.9.4.1-0ubuntu1_i386.deb
``` Consider [adding these optional components]("http://www.gnome.org/projects/rhythmbox/download.html") for the ultimate Rhythmbox setup.  Enjoy!
