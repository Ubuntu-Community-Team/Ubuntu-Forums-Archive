### Post by DBStevens on 2007-06-10
And MD5 is one of many possible message digests.

Think of it as a single large number that represents the entire contents of a file (message).

If the MD5 of a file is provided it can be used to verify if a file is correctly downloaded.  It is a means to detect an error in file down loads. 

Programs are availible that caclulate MD5 values, programs like K3b will verify that .iso etc get correctly burned by using a MD5.
