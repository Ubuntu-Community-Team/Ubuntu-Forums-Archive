### Post by taurus on 2006-06-11
There is only one version for x83 and it comes with a default i386 kernel.  After you have finished installing it, you can upgrade your kernel to i686 if you wish with
```

sudo apt-get install linux-686

```
However, using i386 on your i686 doesn't hurt a thing.  Runs just as good...
