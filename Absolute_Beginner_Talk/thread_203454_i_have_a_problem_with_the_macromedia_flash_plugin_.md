### Post by harishg on 2006-06-25
There are two kinds of flash software macromedia and the gpl flash.The problem might be due to the installation of both the softwares.To check if this is the case open firefox and type 
 
about: plugins (no space between : and p)

It will list the plugins installed.Check if there is a mention of GNU flash.Remove it using synaptic if you find it.

Another problem might be due to the way the site was coded.I have faced some flash sites displaying wrongly in linux but working fine in windows.

Hope this helps.
