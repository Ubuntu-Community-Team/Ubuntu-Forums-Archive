Customizing the look and feel can be all done through the GUI for the most part.  See that "System" menu on your gnome panel(I'm assuming you are running gnome, the default desktop environment)?  Well, click that.  Now click the "Preferences" sub-menu.  You can play around with these things all day long.   The main ones for changing the appearance would be "Themes" and "Windows".

Once you get bored with that stuff, you can try your hand at [installing beryl](http://www.ubuntuforums.org/showthread.php?t=263851) if you so desire.


> I don't know how to command an installation. 
And as for learning how to install things.  There's pretty much just one command that will install 99% of the apps you could ever want on Ubuntu.  
```

sudo apt-get install _____

```
Run this command replacing the blank with the package name you wish to install.  Usually the package name corresponds exactly with the name of the application.

And the most powerful command of all
```
man ____
```
man is short for manual. Run man followed by a program name that you want to learn more about.

Also, you can go through the list of things on
[http://ubuntuguide.org](http://ubuntuguide.org)
for some ideas of new things to configure or install.
