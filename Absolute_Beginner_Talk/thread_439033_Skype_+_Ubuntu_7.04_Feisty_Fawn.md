### Post by Killtown on 2007-05-18
the Public Chat feature doesn't work which is a total drag.

You can't log on to Skypecasts the traditional way, but there is a back door way:

> You can 'join' a Skypecast by entering it's 'room' number as a phone number. Going to [https://skypecasts.skype.com/skypecasts/ongoing]("https://skypecasts.skype.com/skypecasts/ongoing") select the Skypecast you are interested in with a right click on the 'Join now' link and open the page in a new browser. The address of this new page, which you will see in the address box at the top contains a 17 digit number, that currently start with +9900111000. Enter this number as a phone number and you will join the room.

[http://forum.skype.com/index.php?showtopic=72088&st=0&p=346526&#entry346526](http://forum.skype.com/index.php?showtopic=72088&st=0&p=346526&#entry346526)
