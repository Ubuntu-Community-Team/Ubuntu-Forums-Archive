### Post by liamwithers on 2007-10-20
WINE is a Windows Emulator, you can download and install Windows .exe (executable) Files with it. You can download it by going to Applications --> Add/Remove Applications and then searching for WINE.

It allows you to run anything from games to calculators :)

Hope this helps, Liam
