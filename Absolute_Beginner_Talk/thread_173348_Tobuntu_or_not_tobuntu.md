### Post by 3rdalbum on 2006-05-11
These are very strange problems that you're reporting. I've just installed Ubuntu on a new computer; it works almost perfectly without any extra setting up. Funnily enough, it came with Windows, which I found quite difficult to get my broadband working on; and I even had to use the command-line to encode some WMA files!

The passwords are due to Linux's security systems (for the record, you type your user password, not the root password). Try going into System > Administration > Users and Groups. Click your username in the list that appears. Hit the Properties button, go to the User Privileges tab, and tick all the items there.

Just wondering: When you installed it, after you rebooted and Ubuntu started loading its packages, did it tell you that it couldn't complete the installation? If it did, try going to a terminal and typing:
```

sudo apt-get update
sudo apt-get install ubuntu-desktop

```

That might solve your problems.
