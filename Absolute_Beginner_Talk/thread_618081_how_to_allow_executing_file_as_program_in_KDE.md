### Post by jordanmthomas on 2007-11-20
Right click the file and go to properties.
Then, go to the permissions tab.  Here you will find a checkbox called "Is executable" 
Did you really look that hard?  :)

I believe this is what you're looking for.  If all else fails, you can alway chmod +x the file.
