### Post by JT673 on 2007-04-20
I thought file-roller is installed by default...

Go to Applications (top-left) -> Accessories -> Terminal, and type:

```
sudo apt-get install rar
```

And it will ask you for your password.  Then everything should work :)...

You can still run [some Windows apps]("http://appdb.winehq.org/") like WinRAR if you like:

```
sudo apt-get install wine
```
