### Post by muep on 2006-09-18
If your system is still in a good condition, there is nothing stopping you from just dist-upgrading. On the other hand, if you have a lot of stuff installed from source or otherwise bypassing the package management, the upgrade system might not be able to handle it that well. I don't know if Automatix or Easy Ubuntu are some of those breakers, I've never used them. Most systems should dist-upgrade quite nicely.
