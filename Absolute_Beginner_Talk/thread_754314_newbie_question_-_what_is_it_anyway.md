### Post by ibutho on 2008-04-13
It really depends on what apps you run on Windows. From your list, only Photoshop does not have a native Linux version, but it can be used via WINE which is a tool for install Windows apps on Linux (not all Windows apps work).

If you used Linux, you would be less vulnerable to viruses and malware, but you should still be vigilant about security.

Welcome, 

Ubuntu is an alternative to Windows. You can dual-boot it along with another os, like windows. This way you can benefit from both. You could also install it as you main system, like I have on my laptop.
It comes with Firefox, Openoffice and a PDF reader. Photoshop CS2 is installable, you have to figure that out with ¨wine¨.
Ubuntu comes with ¨the GIMP¨ Some say it´s as good as photoshop, others say it´s not. It sure is a good free alternative to Photoshop.
More info on Wine: [www.winehq.org](www.winehq.org).
Wine application database: [http://appdb.winehq.org/](http://appdb.winehq.org/)
Wine can also be used to play some windows-based games.

You are not vulnerable for windows virusus/trojans/malware. There is Little to None risk that you´ll ever get a virus on ubuntu, or any other GNU/Linux distribution.

It comes as a ¨livecd¨ which you can pop in to check for a first impression. If you like it, you can install it very quick and secure. If your hardware isn´t fully supported, you should try get some help here. If it´s totally not for you, you remove the cd, and nothing happened.
