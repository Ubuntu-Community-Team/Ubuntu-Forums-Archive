### Post by Pablo_Escobar on 2005-10-18
It slower then in Hoary because in Breezy Synaptic default search option is "Name and Description" (or something like it - I use Polish version) which searches through descriptions of packages.
In Hoary the default was only "Name" which is a lot faster because it only searches the required word in the packages names.

I hope this clears it up.
