### Post by arochester on 2006-10-15
I've got vmware installed on both Windows and Linux.

When it is starting you will see icons for floppy and CD. If they are "on" the program looks for vm there. Click on them, make sure they are "off", but be quick, to make sure they are disabled - and the program will then look for vm on the computer.

### Post by djsroknrol on 2006-10-15
You need to run vmware-uninstall.pl script in the vmware folder you installed from to remove all traces of the original install and then reinstall it.
