### Post by YokoZar on 2006-11-20
> **BarfBag said:**
> Wine is slower then Windows because it's emulation.  The only way to improve your performance is to install your graphics drivers and ajust the brightness on your monitor.
Wine is not an emulator.

Slowness in Wine is either due to bugs in Wine's implementation of the Windows API or, more rarely, due to shortcomings of the Linux operating system.
