### Post by Ek0nomik on 2007-08-01
I would try booting with the Live CD again.  If the Live CD spits out errors, or doesn't boot, you have nothing to worry about, because it doesn't write anything to your hard drive.

You can also install Ubuntu using the Alternate CD.  It's a text based installer that can rid a lot of troubles that relate to video/graphic issues.

[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

You will see a checkbox under the green button that allows you to download the Alternate CD instead of the Live CD.  It's just as easy to use to install.  Give it a whirl.  ;)
