### Post by Bloch on 2006-06-23
Can you see the ubuntu desktop? Or do you mean there is no signal to the monitor? What exactly is on the screen?


You are probably running ubuntu from a CD boot. The first time you start ubuntu from the CD it runs "from the CD", but does not install anything on your hard drive. There is an icon on the desktop called "Install" and you click that to begin real installation. You will be asked questions about langage, timezone, and paritioning the hard disk.

But you say your laptop is old. Don't install from the standard CD. Download the alternate install CD. It is better for installing on an older system.

You do not need floppy disks. Where did you get that idea?
