### Post by daemonoid on 2007-01-23
It can be a pain switching monitors in ubuntu if the new one isn't automatically recognised. I'd suggest first of just try it, if it doesn't work you just power down plug the old one back in along with the new one, boot up and set up the new one.

You may need to fiddle with your /etc/X11/xorg.conf to change which output on the graphics you're using... but as daemonoid said... best to try it first and see if it works without any meddling then come back and ask again :)
