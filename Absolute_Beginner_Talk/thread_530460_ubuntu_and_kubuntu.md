### Post by tszanon on 2007-08-20
Both (Ubuntu and Kubuntu) have install CDs (700MB) and install DVDs (4 GB). The difference, as far as I can recall:
CDs:
1. Two versions: live and alternate.
2. only support one distribution (ubuntu or kubuntu, not both)

DVDs:
1. they cover the live-cd, the alternate cd, both ubuntu and kubuntu (this must be checked, I'm not so sure), and a lot more packages that you can find in the repositories, but not in the cds.


Which one (ubuntu or kubuntu) is better? If you prefer Gnome, get Ubuntu. If you prefer KDE, get Kubuntu. If you have no idea what I'm talking about, then you have some alternatives:
1. search the forums for some threads like "Gnome vs. KDE", so you can see some opinions about both;
2. try both and see what suits you better;
3. roll a dice. 1-3 you choose gnome, 4-6 you choose KDE. :) :p :mrgreen:
