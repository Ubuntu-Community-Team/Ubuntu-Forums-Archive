If you right click the file and hit properties, there is a checkbox that can make a file executable.

### Post by trak87 on 2007-06-07
While it's a bit daunting to learn at first, I still think the command line is best.

However, you can change file permissions from Nautilus. For example, select Places and then Home Folder. Right click on any file and select Properties. Select the Permissions tab. See the "Allow executing..." check box?
