### Post by digitalbenji on 2007-06-07
Howdy,
   To my knowledge, the Add/Remove list gets automatically updated when updates are rolled out for it.  The difference between it and Synaptic is that it contains a smaller subset of packages that are in Synaptic, so Synaptic has everything in Add/Remove (called EasyBuntu I believe) plus some more.  However, the stuff in Add/Remove is all/mostly stuff that integrates well into the Ubuntu desktop.  Also, I've found that lots of stuff I install through Synaptic does not make menu entries, but everything I install through Add/Remove does.  I hope this helps.
