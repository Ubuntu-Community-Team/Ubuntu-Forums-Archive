### Post by st33med on 2007-05-12
1- You will have a hard time trying to get Windows games and a few programs running. Try WINE or use virtualization software to run Windows XP, such as VMware or Bochs.

2- That depends. It might take a bit of meddling around in the terminal to get drivers working (if Linux doesn't support it), but, once you get around to it, it is really easy to use stuff.

3-First off, You need to know about Beryl. It is a program that makes the desktop have neat effects  and look pretty, but, IT IS BETA! It is supposedly unstable, so I would not recommend installing it because, if it breaks Ubuntu, you keep both pieces. Only use it if you have Ubuntu backed up and ready for the onslaught of problems. Next, Terminal is in the Ubuntu menu under applications. Finally, don't worry about something not working, we are here to help! :)

### Post by aeiah on 2007-05-12
ubuntu is very easy to use, but not always easy to set up. what is your system like? my girlfriend doesnt care about tech stuff, but she liked beryl so i installed linux for her. she's got a dell with an intel graphics card, and a wired connection, so everything worked out of the box and she loves it (and so did i, since it was so easy to set up). if you have a wireless card, an ati or nvidia graphics card, or a printer, then you will need to do some setting up (unless you have a wireless card that 'just works'). some is a little complicated, other bits are a breeze.

Disadvantages:
Linux has a higher learning curve than Windows- but if you think about it- it probably took you a while to get used to Windows too (it was just so long ago that many of us forgot how frustrating that time was). Most people jumping into linux give it an hour or two and then give up. But when I was a beginner on Windows (3.1), it took me more than just an hour or two to feel comfortable. You will need to learn to make the best of the awesome community support in wikis, forums, and even blogs- and get used to googling when you run into problems. 

Ease/Simplicity:
-Ubuntu is a very friendly system that any basic user can pick up on. Doing basic computer stuff like surfing the web, reading email, instant messaging, office applications--- are no more difficult than Windows. Where you may find difficulty is when you venture out of the basics. But it's really fun once you get into it.

Misc:
The best advice I can give a beginner is to play around in a live CD. Many distros have live CDs that you can pop in your computer, restart, and then boot up linux without making any changes to your hard drive. They run a little slow compared to 'the real thing', but they can give you a neat preview into the unknown.

Good luck!
