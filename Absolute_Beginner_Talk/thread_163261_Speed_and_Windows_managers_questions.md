### Post by hw-tph on 2006-04-20
First and foremost: Responsiveness almost purely subjective.

Second, load times are more or less equal. You're using the same packages in KDE and Ubuntu, except for the desktop related ones. If you want a responsive and usable desktop go for a lighter alternative. Enlightenment isn't really one of them, it is actually rather heavy on resources. Have a look at the *box series of window managers - the original Blackbox, Fluxbox and OpenBox. I prefer OpenBox because it is very easy to configure (XML configuration files; there are graphical configuration programs as well if you're a point and click fanatic) and very responsive . Simplicity at its best, IMHO. For a rundown of window managers, check out [xwinman.org](http://xwinman.org/).

To further increase responsiveness you use one of the alternative IO schedulers (these can greatly alter the way the computer "feels") and you could also build a custom kernel with some performance enhancing patches.


Håkan
