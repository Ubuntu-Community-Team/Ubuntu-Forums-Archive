### Post by Marsolin on 2006-09-14
I just wrote an article about this last Friday called [Switching from Windows to Linux using VMware Player]("http://linuxappfinder.com/blog/switching_from_windows_to_linux_using_vmware_player").

It describes how to set up the free [VMware Player]("http://linuxappfinder.com/package/vmwareplayer") to run Windows within Linux.

To your original question, while it is possible to run Linux from within Windows, I would only suggest it if you cannot change the main OS. The ideal situation is to run Linux natively and then only run Windows for specific applications that you want to keep. The only other reason you would want Windows to be the native OS is if high-end 3D games are required.
