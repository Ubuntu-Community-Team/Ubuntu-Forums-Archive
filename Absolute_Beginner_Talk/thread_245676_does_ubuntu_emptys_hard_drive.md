### Post by amo-ej1 on 2006-08-28
if you say it should, it will. When you install ubuntu you can select the partitions where it should install itself on and you can choose to format these partitions. If you hard disk is partitioned in two pieces, you can choose to format one partition and install ubuntu on that one and the installer will not touch the other partition.

I guess you want to know is if when you install Ubuntu, everything in your hard drive disappears.
The answer is it depends on you. During install you will be prompted for different options. You can chose to erase the entire disk, to use the free space or for customized partitioning. You can  perfectly leave a Windows partition untouched, or a /home partition from a previous linux installation (Ubuntu or not), to preserve your documents or anything you may want to keep there.
