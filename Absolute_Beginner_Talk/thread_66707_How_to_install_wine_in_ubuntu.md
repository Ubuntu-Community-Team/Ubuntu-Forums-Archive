### Post by Leif on 2005-09-18
first, follow the instructions here to enable extra software repositories : [http://ubuntuguide.org/#extrarepositories](http://ubuntuguide.org/#extrarepositories)

afterwards, open system->administration->synaptic. search for wine. scroll down and find wine, right click on it, choose install. click apply and it will be installed.

now when you want to run a windows program preface it with wine, ie wine notepad.exe.

p.s. i also have winesetuptk and winetools installed. i'm not sure why i installed these, but if you have problems using wine try and see if they help.
