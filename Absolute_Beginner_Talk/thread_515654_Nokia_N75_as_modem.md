### Post by jermschevy on 2007-08-05
i've been using my nokia 6230b as a modem for quite some time. Just got the N75, it works too.  Get your data cable ready.  But first, go to [www.nokiausa.com](www.nokiausa.com) and download Nokia PC Suite. I'm using Nokia PC suite version 6.82.22.0.  My N75 came with a data cable and the cd with the software on it. There is an option to connect to the internet, open it up and go to settings,  select Nokia USB Modem, configure connection manually, access point is:  wap.cingular      username is: [email]WAP@CINGULARGPRS.COM[/email]     password is:      cingular1           
that should do it, good luck,     repost, let me know how it works out for you,    Jerm

### Post by jermschevy on 2007-08-05
oh, don't forget, you might want to upgrade your media.net package to unlimited.... in my area you can get unlimited media.net and 200 text messages a month for $20.00.  "MEdia(TM) Max 200 Bundle 
UNLIMITED: Cellular Video & MEdia™ Net. Plus 200 Text, Video, Picture, Instant Messages (5cents/add'l)."   
When i first started calling cingular about doing this, they told me it couldn't be done, and I'd have to get a pc card, and an unlimited package would cost $50 or $60 a month.  Then I found the upgraded nokia pc suite software with the connect to the internet option.
been working great for me... i have no trouble with it in non-3g areas.  In New Orleans I get 3g signal but it seems to be less reliable than non-3g areas.   Doing bandwidth test on speakeasy.net i get about 165 kb/s download and 95 kb/s upload in New Orleans 3g.   Not near as good as I expected.
