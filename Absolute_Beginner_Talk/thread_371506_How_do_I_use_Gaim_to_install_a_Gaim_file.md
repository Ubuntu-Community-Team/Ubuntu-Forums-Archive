What you downloaded is called an autopackage.

From the [autopackage.org sit]("http://autopackage.org/docs/howto-install/")e:
> **How to install Linux autopackages in 4 easy steps**

   **You only have to follow these instructions once**. After the first time, you should be able to   open autopackages directly from your web browser. 
  **Step 1: Download & save**
Download the packages you want to install and save them on your computer.  Go to the folder where you saved the software package to.                   
**Step 2: Open Properties dialog**
Click on the package with your *right mouse button*. Then click on the *Properties* menu item. [B]

[/B]**Step 3: Enable permissions**
Go to the *Permissions* tab. Check the *Execute, Exec or Is Executable* checkbox.  If there are more than one *Execute/Exec* checkboxes, check the top-most one. Click *Close* or *OK*, depending on the dialog.     

**Step 4: Uninstall previous and open**
If you already have the software installed from your distributions packages or from the source, remove it  first. Then *open* the package file. If you are asked whether you want to display or run the package, click *Run*.

**Done**
The installation will now begin. Follow further intructions that are displayed on the screen, if any.

Just be warned that *installing some software that is NOT in the repositories can break your system*.  Be sure to search synaptic for a program you are looking for.  In this case try "gaim" or "paltalk" and see if it comes up with anything.

Good Luck!
