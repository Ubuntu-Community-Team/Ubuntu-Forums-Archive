Sorry I can't help much on your LiveCD, just 1 piece of advice if you intend to install Ubuntu as dual boot in the same computer. Do NOT allow Ubuntu install disc to re-partition your hard disk, your Vista will not boot after that.

Instead, go into Vista disk managment, and shrink the Vista partition to the new size. When you install Ubuntu, chose "guided- use largest continuous free space" option.
