### Post by superm1 on 2007-03-07
The easiest way would be to install vmware tools inside the VM.  They aren't distributed with vmware-player, but are obtainable on the web.  

[https://help.ubuntu.com/community/VMware/Player?action=show&redirect=VmwarePlayer](https://help.ubuntu.com/community/VMware/Player?action=show&redirect=VmwarePlayer)

Describes this, grabbing the latest version of workstation and taking the vmware tools ISO image out of it.  Just make sure you grab the right one (there are vmware windows and linux tools avail as ISOs)
