### Post by Hospadar on 2008-02-24
I'm trying to create a non-standard size pdf from an openoffice spreadsheet.  It's a big spreadsheet and I want it to show up on all one page.  It also has some images and non-standard formatting so I can't really distribute it in a microsoft format that I could count on people being able to use.

Openoffice's export pdf tool doesn't allow for any other paper sizes, I tried adding a pdf printer in System->administration->Printers and changing the paper size there, as well as changing the paper size in openoffice when printing to that printer, but no dice.

If anyone knows how to do this, I'd really appreciate it.  Even if I could just get a bitmap of the entire spreadsheet that would be really useful.

Thanks!

---

### Post by Hospadar on 2008-02-24
Never mind, figured it out!

went to File->Page Preview, then Format Page, changed the page size there, then exported the pdf.
