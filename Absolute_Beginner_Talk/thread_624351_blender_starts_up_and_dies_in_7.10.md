### Post by erfahren on 2007-11-26
I doubt the one from the Blender site will do any better. Blender works fairly well for me (occasional crash).

do you need to use a restricted graphics driver? (enabled through: System > Administration > Restricted Drivers Manager)

the reason I ask is after a quick Google search (ubuntu Ignoring Xlib error: error code 158 request code 146) I came up with this which talks about a messy xorg.conf
[http://blenderartists.org/forum/showthread.php?t=110651](http://blenderartists.org/forum/showthread.php?t=110651)

(they're saying it can be ignored but that's not going to work in your case!)

Anyway, it sounds like a graphics driver problem. You might check to see if your using the correct one.
