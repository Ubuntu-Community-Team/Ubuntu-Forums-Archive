### Post by rich.bradshaw on 2007-10-11
Easiest way for future ease of use is:

Reinstall using Gutsy official release. Make a separate partition for /home.

After that, you can reinstall whenever you like without losing documents or settings.

Not sure if you can update from Feisty using the Gutsy CD, I'd just do a clean install if you have limited internet access.
