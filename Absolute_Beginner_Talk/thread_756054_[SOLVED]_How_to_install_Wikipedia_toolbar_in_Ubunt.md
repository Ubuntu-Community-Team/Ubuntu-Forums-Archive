### Post by qamelian on 2008-04-15
You don't extract it. It is a Firefox / Mozilla add-on. When you left-clicked on the link to download it, Firefox should have prompted you to install it. Alternately, if you are using Firefox 3, you should be able to install the original xpi file through the add-in manager found under the Tools menu.
