### Post by blendmaster on 2006-11-06
Go to Synaptic Package Manager, and search for gdesklets (or in the the terminal, sudo apt-get install gdesklets).

Gdesklets is a great little program for adding eye candy to your desktop that does things like that!

Hope that helps!
