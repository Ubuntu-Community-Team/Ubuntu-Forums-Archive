### Post by Kubunteando on 2007-06-17
You are using the Open Source MESA driver. But it is not configured properly for HW acceleration, so 3D games will be slow, and no Beryl...
You have to get Direct Rendering as Yes to have HW acceleration.

You have to check what is the model of your ATI card and decide if you want Open Source drivers of you want the ATI proprietary ones.

### Post by Kubunteando on 2007-06-18

OpenGL vendor string: ATI Technologies Inc.

And Direct Rendering says "Yes", meaning you have 3D HW acceleration enabled.

So you have the drivers from ATI (the proprietary ones) correctly installed.

Make a backup up of the file /etc/X11/xorg.conf just in case you need it someday.

Now you can test some 3D applications to check how they are doing.
