### Post by NightwishFan on 2008-04-19
There are the same applications, only made for 64bit with added performance.  In the rare case there is no 64bit version you can still use the 32bit version. You will find most applications perform better in 64bit but take slightly more memory which is no problem. I have 895mb of ram and I barely use half. Programs that do audio/video/pictures/gzip will see great improvements.
