### Post by Klaidas on 2006-03-11
> Skype Internet Telephony

    *

      With Skype software, you can talk over the internet using your computer. Skype uses a peer-to-peer technology, so your calls do not go through a central server. In addition, all communications are encrypted so others cannot listen in. The Skype software is free to use, but it is not free software; the source code is proprietary and not available for modification.
