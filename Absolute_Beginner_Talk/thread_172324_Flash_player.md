### Post by aysiu on 2006-05-09
Synaptic Package Manager, whether you know it or not, uses a version of *sudo*--*gksudo*, which asks for your password graphically (instead of in the terminal).

This is why most Mac OS X users don't realize that OS X uses *sudo*, as almost everything is point-and-click on Macs.
