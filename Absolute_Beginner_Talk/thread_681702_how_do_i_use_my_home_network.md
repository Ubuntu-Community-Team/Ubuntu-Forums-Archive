### Post by mgm2rr on 2008-01-29
As you have probably already seen, you can share a file on Ubuntu by right clicking and then selecting the Share option on the dropdown menu, but you need to be sure to share it as a windows share (smb) not as a unix share.  Also, sometimes the windows machine will show you a prompt asking for your username and pw, in that case be sure to use your linux username and pass.
