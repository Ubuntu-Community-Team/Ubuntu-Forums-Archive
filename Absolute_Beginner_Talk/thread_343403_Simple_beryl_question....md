### Post by floke on 2007-01-21
The windows follow the mouse because this is how you have it set up!
In your system/preferences/windows (I think) is an option for click window to focus, or focus follows mouse - yours is set up for the latter.

For Beryl themes see eg. Gnome-look.org and search for beryl themes.
To get them - dowload and save the theme you want as a tar file - then use emerald manager (under system/preferences) -click import and then click to the tar file of the theme.
