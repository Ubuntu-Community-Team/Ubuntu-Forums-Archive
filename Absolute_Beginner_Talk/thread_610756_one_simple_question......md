### Post by az on 2007-11-12
What makes your box vulnerable is what's running on it.  Who's logged in is pretty irrelevant if you have insecure software running on it.

If you are running an up-to-date system with software from the repositories and no third-party software, and are not running some way for user to log in easily (like runnning the ssh deamon along with having users on your system with easy passwords) you should be fine.

If your ktorrent is the only thing listening to the network, then there is no way for someone to harm your computer, regarless of whether you have a (useless) firewall or not.
