### Post by taurus on 2007-05-28
Make sure you have java running (Sun version) on your system first before you can run frostwire.

Applications -> Accessories -> Terminal
```
java -version
```

Then, download the Ubuntu/Debian version from here.

[http://www.frostwire.com/?id=downloads](http://www.frostwire.com/?id=downloads)

After saving it to your computer, desktop, double-click the icon to install it.  Then, there should be an icon in Applications -> Network.
