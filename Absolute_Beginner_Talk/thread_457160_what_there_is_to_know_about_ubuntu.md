### Post by aysiu on 2007-05-28
Linux is the basic core of the operating system that talks to all the devices. There are many operating systems that use the Linux kernel, and these operating systems are called *Linux distributions* (sometimes just *distros*). Ubuntu is one distro, possibly the most popular one--depends whom you ask.

Really, Ubuntu can do a bunch of different things. 

If you want, it can be a live CD that is a **fully functional operating system** that does not affect your hard drive. It runs completely off the CD itself and your computer's RAM (so the more RAM you have, the faster it'll run--I'd recommend 512 MB at least).

Alternatively, you can use VMWare to "install" Ubuntu on a virtual drive within Vista to try it out.

If you want, you can instead set up what is called a *dual boot*, which allows you to install Ubuntu and Vista together on one drive. When you boot up your computer, you'll be given an option to boot up Ubuntu and use Ubuntu or boot up Vista and use Vista. While the process has been greatly simplified over the years, it's still advisable for you to understand partitioning in order to use this option, and you may (if you don't know what you're doing) accidentally delete your Vista partition, so you *must* back up your important data before attempting a dual boot.

Another option is to use [Wubi](http://www.cutlersoftware.com/ubuntusetup/wubi/en-US/index.html) (currently in beta testing) to install Ubuntu as a Windows application. It sets up a non-destructive dual boot with Ubuntu as a large .img file within Windows (so no need to mess with partitioning). It also uses the Windows' boot loader instead of the Ubuntu one and allows you to uninstall Ubuntu easily. This is another kind of dual-boot. The downside to this approach is that it can be difficult to move from this kind of dual-boot to a single boot of Ubuntu, since Ubuntu lives inside of Windows instead of next to Windows.

The last option is to just erase Vista completely and install Ubuntu in place of it. This is the easiest option. No messing with partitioning. No dual booting.

As for *whawt will be my downfals and advantages to going ubunto style*, read [Is Ubuntu for You?](http://ubuntuforums.org/showthread.php?t=63315). I'd also suggest reading [A home user&#8217;s successful migration strategy from Windows to Ubuntu](http://ubuntucat.wordpress.com/2007/05/25/a-home-users-successful-migration-strategy-from-windows-to-ubuntu/)
