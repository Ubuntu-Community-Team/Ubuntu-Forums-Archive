### Post by Kilz on 2006-07-09
> **jpazindustries said:**
> I was just wondering what the difference was between a 32 bit processor and a 64 bit processor, 32 bit ubuntu and 64 bit ubuntu. :confused:
A 64bit processor can handle more information than a 32bit processor. 64bits of info vs 32 at a time.  64bit chips can run 64 and 32bit applications, 32bit can only run 32bit. 64bit processors also have an on chip memory controller. IMHO 64bit processors will eventually replace 32bit versions. Just as 32bit replaced 16.
I run 64bit Ubuntu Dapper drake. I will tell you from a 64bit point of view what the difference is. Some people may later tell you what they think the difference is from the 32bit side. 
When you ask someone what the difference is make sure they are running the latest version of the 64 bit Ubuntu. Because there is a huge difference between 64 bit Dapper and 64bit Breezy.
For the most part the difference is with the 64 bit version there is nothing. I would say for 97% of the applications there is a 64bit version in syanptic. You will install the application just like on  the 32bit version. 
For 2% there is a little more setup for some applications. For most part its a simple cut and paste howto you will have to follow. I have written 2 howto's you will find in my signature. I am presently learning how to write install scripts, so it will get easier on my howto's.
For the remaining 1% it may be a little harder. But those applications are rare, older, or proprietary. 99% of the users will not use them. They can be installed, but you may have to do a lot more work.
All 32bit applications can be run on 64bit Dapper. It just depends on what percent they are in. With 64bit you can run 64bit applications. Some of them, encoders, rippers, and 3dmodeling and other programs written for 64bit you can see a 30% speed improvement.
Now the negatives. 
64bit - You cant run some automatic setup scripts, or at least you have to make 100% sure you use the 64bit version. Some applications run better as 32bit and you need to follow a howto to install them. They use slightly more ram to run programs.
32bit - Some people running 64 bit systems report that when they were running the 32bit OS they had problems. Slow running, some applications crashed, etc.
