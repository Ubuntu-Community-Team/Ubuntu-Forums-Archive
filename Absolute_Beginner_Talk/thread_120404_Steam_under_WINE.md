### Post by claydoh on 2006-01-21
You can use regular wine (at least the latest versions 0.90 and up) to run Steam. [Look here]("http://appdb.winehq.org/appview.php?versionId=1554&iTestingId=740") for the best (most current) instructions.
I would suggest using "Option one" as it is simpler

This requires you have the latest wine from [winehq]("http://www.winehq.org/") , good directions for obtaining that via apt/synaptic are found  [here]("http://www.winehq.org/site/download-deb") .

I do find ( and recommend) cedega to be better and is easier to tweak (necessary for a lot of games), but the gap is closing quite a bit lately.
