### Post by Brunellus on 2006-09-28
> **Strike! said:**
> but i can update 5.10 and be as strong as 6.06 right?
so nothing changes if i install 5.10 ?
5.10 remains supported, so you're not missing out in terms of security updates.

6.06 made one tremendous change in the way it handled devices, though, abandoning hotplug entirely.  It's quite a bit faster on my hardware...but there was an issue with the default kernel on the 6.06 CDs on some older hardware that fails to detect PCI cards.  This has been fixed in subsequent updates.

YOu can definitely install 5.10 (breezy) and always dist-upgrade to 6.06 later, if you wish.
