### Post by eluzi on 2007-11-26
Two suggestions:

1 - change fstab, trading UUID for the directions to your devices/partitions, like:

# Entry for /dev/hda1 :
/dev/hda1 /media/hda1 ntfs-3g defaults,locale=en_IN 0 1

2 - In case u wanna try something before changing system files, you could type:

# mount /dev/hda1 /media/hda1

This will manually mount the device in /media/hda1, if it works the problem is in fstab. When you type only "mount /dev/hda1" the system will look for where to mount it in fstab, so it wouldn't work if you had a problem there...

Hope it helps ;)
