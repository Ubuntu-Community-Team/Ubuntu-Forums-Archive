### Post by MasterOfDisaster on 2006-11-30
Download this file for normal computers:

[http://www.stanford.edu/group/pandegroup/release/FAH504-Linux.exe](http://www.stanford.edu/group/pandegroup/release/FAH504-Linux.exe)

Or, this for 64-bit, dual core processors (won't work if you don't have one).

[http://www.stanford.edu/%7Ekasson/folding/FAH_SMP_Linux.tgz](http://www.stanford.edu/%7Ekasson/folding/FAH_SMP_Linux.tgz)

Then, cd into the directory you saved it to, and then make sure that it is excecutable (sudo chmod +x FAH5-Linux.exe), and run it by typing ./FAH5-Linux.exe in the terminal.

Cheers.
