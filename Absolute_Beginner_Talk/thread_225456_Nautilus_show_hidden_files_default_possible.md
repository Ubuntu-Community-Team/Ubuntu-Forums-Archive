### Post by qamelian on 2006-07-29
> **orb9220 said:**
> Like the thread says is there a startup flag to toggles this automatically?

Open a Nautilus window and go to the Edit menu. Select "Preferences". On the Views tab, about a third of the way down, put a check mark beside "Show hidden and backup files". Close the Preferences box.

### Post by ovimunt on 2006-07-29
At the moment I'm running KDE so I can't run you throught the process step by step. However, I remember that in one of the Menus you have a Preferences option. Go there and somewhere around there you'll find an option to enable this.

***NOTE!*** You'll have to do this for every user. If you change the options while running as root it will remember it but only for this user. You'll have to run Nautilus for each of the other users and change this setting.
