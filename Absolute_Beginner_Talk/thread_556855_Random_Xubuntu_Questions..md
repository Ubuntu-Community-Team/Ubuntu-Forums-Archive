### Post by Orbital75 on 2007-09-22
Please don't take this the wrong way..... If I were you, I would switch to Gnome.  In my opinion 
Gnome seems easier for a novice.  I'm a Linux novice myself and have Ubuntu installed on my desktop 
and Xubuntu on an old Laptop.  I find Gnome more user friendly than Xfce. Xfce is the desktop that Xubuntu uses.
 I've used both so I know. 

It's a pretty simple task to use Gnome if you want.
Just open the Synaptic Package Manager located
in Application ---- System.
Now do a search for Gnome and apply it.
Logout and then choose Session. Choose
Gnome and your all Set...

It will ask if you would like to use it all the time
or just this session. If you choose "Just this Session"
it will use Xfce the next time you log back in.  In other
words the settings will just run once. 

Now if you have a lower end PC Xfce is what you will need. Gnome and KDE take more resources.

I'm pretty much a novice but have had to learn all this just as
you so it's fresh on my mind. Changing the time, Wallpaper,
mp3 support, even adding a docking bar.

I hope this helps a little.
