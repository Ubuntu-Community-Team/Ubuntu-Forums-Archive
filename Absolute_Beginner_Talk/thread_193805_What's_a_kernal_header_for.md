In Synaptic, in the development section, there's something called a linux-header, and like kernals, there's one for every type of processor. So what is it for?

---

### Post by taurus on 2006-06-10
You need the kernel header if you want to build something from source that requires you to have your kernel source on your machine!  If you don't know what it is, chances are you don't need it.  And if you ever need it later, then you just install it using synaptic.  Remember to install the right version for your kernel...
