### Post by Raineer on 2007-07-25
Several very good methods... WINE is the best all-around windows "emulator", which is in quotes because WINE stands for "Wine Is Not an Emulator". Many, many windows apps can be run this way.

Crossover works well on a good deal of games, as does Transgaming Cedega.

You can always use VMWare to run an actual installation of Windows inside Linux if you need. Gaming is very short this way but you get a complete install seperate of Linux.

As for Win32 and third-party apps, I'm not sure what you mean. WINE can run the programs, but the windows drivers are useless unless you are using VMWare.
