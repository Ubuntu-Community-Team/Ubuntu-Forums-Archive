### Post by jordanmthomas on 2007-04-02
dapper -- long-term support release, came out before edgy
edgy -- current Ubuntu release
feisty -- next Ubuntu release

Which is best?  Depends on how cutting-edge you want to be...stable --> dapper, bleeding edge --> feisty

**a definition real quick:  package managers manage the software you have installed on your computer.  This is one of the main diffrerences between different Linuxes.

My personal quick reviews of other distros you may be interested in:
fedora -- quick, except package management.  I love its administration / networking tools
red hat -- haven't really used it.  I imagine it's basically the same as fedora.
gentoo -- if you choose it and succeed in following the instructions to install it, you'll be happy.  Almost everything is built from source for your specific computer.
SuSE -- for me, it's slow.  Its package management is retardedly slow.  Besides, it's from Novell and I can't suggest going anywhere near them right now.  :(
Arch Linux -- for me, it's nearly perfect.  It's fast and only what you want installed is installed.  However, because of this, some people suggest it may be too advanced for novices.  Personally, I think diving all the way in is the only way to go.

Ubuntu -- fast package manager, has too many services enabled my default for my tastes.  It's good though and it has excellent free support.  I would probably reccommend going with edgy at this time, and upgrading to feisty when it comes out if you have decently fast Internet.  Otherwise, I'd say just go ahead and install feisty.

** beaten, and disagreed with.  Good luck!
