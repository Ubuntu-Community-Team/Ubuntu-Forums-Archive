### Post by Leif on 2005-07-08
MP3 playback requires a non-free codec, which is not included by default. See [http://ubuntuguide.org/#codecs](http://ubuntuguide.org/#codecs) to fix this

---

### Post by Mr. Electric Wizard on 2005-07-08
I was able to play MP3's with XMMS (ubuntu 5.04), without installing any additional codecs...
You have to go into XMMS -->Options--> Preferences --> then select eSound for the output codec, then it should work fine.
