### Post by ultimut on 2006-12-13
This is more of an FYI than a problem I am currently experiencing.  Since I was able to solve this problem and I know of other people who also had it, I figure that it was my duty to at least post it on here so others would have access to my solution.

I have been using Ubuntu for the last couple years and I love it.  I recently wanted to install Ubuntu on my Mac G4, so I downloaded the image and off I went.

I attempted to install the machine a whole bunch of times, but it always stopped during the install (and it was stopping in different places).  I burned new copies of the cd's and downloaded other versions of Ubuntu and I continued to have the same problem.

I was able to boot from the live cd, but the install would not complete.

A friend of mine at work had also had the same problem with the same symptoms and we had discussed pretty much every possibility from bootstrapping all the way up as to why this was not working.  We decided that the problem had to be with the reading of the data from the cd, so we approached it from that perspective.

He had an old cd drive in a junk machine that he switched in to his problem machine and then tried the install again.  The install went through beautifully and he was up and running.

With that info I went back to my problem and remembered that I had taken the original cd drive out of my Mac and had replaced it with a CD-RW one.  I found my old cd drive and replaced the CD-RW one with the old one and guess what, everything installed beautifully and I was off and running...

The moral of the store... (Sorry it took so long for me to get to this.)
We believe that the Ubuntu install MAY have a problem with the read/write speed of some cd drives.  If you having a similar problem and you have an old cd drive kicking around somewhere, you may want to throw it in and try it.  I cant guarantee it is going to fix your problem, but it worked for us.

It is probably worth noting that we both went to older drives to get our machines working.  I would kind of expect that the newer hardware would be handled better than the older stuff, but that may not be the case.

Note:  I don't have the problem drive in front of me right now, so I cant give details specs in regards to the problem drive.  I am also unsure what the drive specs were for my friends drive.

I hope this helps someone and maybe if one of the developers has a free minute he/she may want to look into why this may be...

Cheers...
