### Post by PurplePenguin on 2007-03-07

> Azureus is a ram hog, most people who use uTorrent dont use Azureus any longer because of this.

Just as an aside (ie. I don't mean to start any kind of flame wars!) :)

I think I'm the only guy in the world who finds that Azureus uses far less in terms of system resources than the others I've tried.  Keep in mind, I have an old computer (AMD 1200 with 768 MB RAM).

I usually have about 8-10 torrents downloading/seeding at any one time, I'm burning DVDs, have Evolution open for email, and I'm on the net at the same time... and I've got RAM and CPU cycles to spare.

I tried using ktorrent (too many bugs), official bittorrent client (too few options, such as selecting a single file in a large torrent), bittornado (I think it was bittornado that I was using - it loads a separate instance of the program for each torrent downloading, right?) which was the worst of the bunch.  As soon as I'd have four torrents downloading at once, my CPU would be at 100%.  

I haven't tried utorrent under wine... maybe I should give it a shot.  Can't hurt, can it? :)  I am very happy with Azureus, though.  Seems to be the best when large numbers of downloads are going on simultaneously.
