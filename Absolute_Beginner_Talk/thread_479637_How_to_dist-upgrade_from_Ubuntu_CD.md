I have just downloaded the Ubuntu Feisty-ISO and burnt it to a CD. I put it into my CD-ROM drive, which automatically added it to my software sources. Then I opened the Update-Manager and clicked on the Upgrade-Button. However, the tool is still downloading all the packages although they are on CD. How can I make it use the CD instead?

quirks

### Post by EagleRock on 2007-06-21
You can manage the sources for your packages in the Synaptic Package Manager.  The CD and the online source should already be in the list.  You can either make the CD a higher priority (higher in the list) or uncheck the online source in order to force installation from the CD.
