Let me start by saying that I am not an expert in this matter, but I'll try to help you out based on some "searching" I have done.

From what I understand dvdauthor does not come with a GUI but is a command line program. As such I think it is logical that it does not appear in the Applications menu.
The gui is a separate package that is called qdvdauthor.
BTW: An earlier version of dvdautor and qdvdauthor is available in the repositories (can be found by searching on name for dvdauthor in Synaptic).

The sources I have used to write this answer are:
1. The thread [Gui for dvdauthor]("http://ubuntuforums.org/showthread.php?t=53243")
2. This page [Dvd Videos from An AVI File]("http://doc.gwos.org/index.php/Dvd_Videos_From_An_Avi_File")
3. Searching Synaptic
Additionally you may find some useful information on this page on [DVDAuthoring]("https://help.ubuntu.com/community/DVDAuthoring")

This may not entirely solve your problem but hopefully put you on the right track.
