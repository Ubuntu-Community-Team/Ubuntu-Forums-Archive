### Post by Pizzarth on 2007-06-26
well how about some first hand experience? ^^

I use ubuntustudio, and beyond the hot black gnome and plethora of editing applications, it's not too different :P You'll have to install the same codecs for restricted things, like dvd playing as such. but the essentail apps are still the same as the standard ubuntu installation. However, I do find that it handles more multimedia right outta the box. I prefer it to both normal ubuntu and ubuntustudio. not much, if anything, at the core of the system is different. I chose to install from a DVD, but mainly because I had some around and like convenience. If there are alternatives that you're comfortable with, go for it. but I think a clean install is nice.

Oh, and by the way. You might experience an Xserver problem. For some reason the graphics card driver never changed from vesa in the xorg.conf file. As soon as I changed it to i810 it works. so just remember to check there first and adjust to the appropriate driver if the problem comes up.
