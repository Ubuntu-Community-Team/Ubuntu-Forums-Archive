### Post by Tyke91 on 2007-10-25
i would suggest installing anyway (if you like the way ubuntu works on a live cd)

there is an 80%+ chance that desktop effects will work for you, but you will need to download something called the CompizConfig Settings Manager in order to make full use of it.


in order to get it, you should type this into a terminal

```
sudo apt-get install compizconfig-settings-manager
```
then to use it, just look in System>Preferences>Advanced Desktop Effects Settings

Ubuntu isn't for everyone... but most people who have used it agree that it is t3h R0xXor :guitar:
