### Post by benson444 on 2007-02-03
I've just installed Ubuntu on my brothers laptop and he asked me the same question. Maybe there's an easier way but this is what I told him. First just try typing the name of the program in the terminal. A lot of the time this works. If it doesn't then open synaptic, search for the package you want to start, right-click it's entry and click properties. Now click the installed files tab.

When you get to understand the file system you can work out what the command might be. It won't be in a path with doc, as they are documents, pixmaps are icons. Usually it will be in /usr/bin which are where binary files go. So then you can try the whole path in a terminal

/usr/bin/firefox

Then you can put a launcher on the panel or edit the menu and add the program launcher.
