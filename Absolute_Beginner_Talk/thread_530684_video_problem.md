### Post by Hobo Joe on 2007-08-20
Install VLC media player.
```

sudo aptitude install vlc

```
Paste that in the terminal and it will install.

Then go to Applications > Sound and Video > VLC media player, and open it.
Then go to File > Open disk > Ok.

That should do it.
