Photoshop doesn't work very well in linux, although I have heard of people getting PS7 to work under wine.  However, Gimp is included with Ubuntu.  Gimp is a powerful and easy-to-learn photo editing program much like photoshop.  While I'm on the subject, wine is an application layer that allows you to run _some_ windows programs in LInux.  It's not perfect, and it has a fairly limited repertoire.  That being said, it does work for WC3 and steam pretty well for your games listed.

iTunes can be replaced with amarok, which can be installed easily from the repositories.  Rhythmbox, which I believe comes installed with Ubuntu, also works quite well with most ipods.  AIM is incorporated into a program called Gaim, included with Ubuntu.  It can be used with most of the chat clients in existence, including AIM, MSN, Yahoo, Jabber, IRC, ICQ, etc.  VLC can be installed in Linux easily

For torrents, I recommend ktorrent or azureus.  Both can be installed from the repositories easily.  Powerpoint can be made to work through crossoveroffice, another application layer like wine that is specialized for office programs.  Or, a better way to work it is to use the included presentation software called OpenOffice Impress.  It can open powerpoint presentations and export/save as powerpoint presentations.  

Finally, in the US, DVD programs are technically illegal.  HOwever, if you look around the forums or at the Ubuntu beginner's guide, you can find how to install the codecs for DVDs.  Basically it's very easy, especially in feisty fawn (7.04).  Oh, and many Google software programs have a linux version that you can download and instlall directly from Google.  

Good luck, welcome to the forums and have fun with Linux!

### Post by ramjet_1953 on 2007-05-08
The best advice that I can give is to download the LiveCD and give it a try.

That way, you can see if your hardware is detected OK and if you like the feel of Linux. 

When running from the LiveCD it locks your hard drive, so that no changes can be made to your current Windows installation. 

One thing to remember with the LiveCD is that it runs far slower than a full install, as it has to read from the CD, which is much slower than a hard drive.

When burning the CD don't forget to burn it as an iso. Most burning programs have the option to burn as an iso image. Also burn it SLOWLY, no faster than 4 X on good quality media. Budget media just does not cut the mustard with iso images.

Almost any piece of software either has a Linux equivalent or there is a direct Linux version.

Some games have been ported to Linux, others run in Wine, or you can install a virtual Windows environment like VirtualBox to run Windows software that you can't do without.

The other option is to set-up a dual-boot environment, where both Windows and Linux are on your hard drive and at boot-up, you select which OS you want.

Also, the Ubuntu forums are there to help you with any configuration issues that you may have.

One final point. Linux is not Windows. You have taken years to learn Windows, so don't expect to be able to pick up Linux over a few days.
