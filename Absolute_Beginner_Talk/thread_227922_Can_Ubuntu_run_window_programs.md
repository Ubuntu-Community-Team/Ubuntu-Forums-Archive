### Post by mithras86 on 2006-08-02
There is a program, called Wine, which could emulate Windows programs. 
Actually, wine isn't an emulator (Wine Is Not an CPU-Emulator;) ), but it works the same. You have to install wine (through synaptic or apt-get install wine) and start the program with wine [program], for example ```
wine /path/to/calculator.exe
```
