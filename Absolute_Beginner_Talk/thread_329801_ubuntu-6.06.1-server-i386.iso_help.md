### Post by pseudonym on 2007-01-02
Hi, in your buring app you need to make sure to set it to 'burn disc image' (or similar). Copying the .iso file as if it were a data CD will not work.

Also, if you are having trouble booting from CD, check in your system's BIOS to make sure that the boot order is set to boot CD first.
