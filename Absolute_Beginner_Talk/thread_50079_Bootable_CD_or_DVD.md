### Post by UbuWu on 2005-07-19
The iso is a special file that contains all the files and information that will be on the cd. Just putting it on the cd won't work. You have to go to burn image (I believe it is in the file menu in Nero), then select the iso and burn it to cd. You won't be able to put the other things on the same cd. After this, it should work.
