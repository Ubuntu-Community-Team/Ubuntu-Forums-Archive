### Post by tagra123 on 2007-03-24
A word of advice for the future is to make the backup first.

Backup the MBR and partition table. then either the entire partition or the data that is important.


Wierd characters?   

How are you mounting the partition?

Are you sure it is a fat system and not NTFS?


Mounting NTFS
```
sudo mkdir /media/windows
sudo mount /dev/hda1 /media/windows/ -t ntfs -o nls=utf8,umask=0222
cd /media/windows
ls
```

Mounting VFAT
```
sudo mkdir /media/windows
sudo mount /dev/hda1 /media/windows/ -t vfat
cd /media/windows
ls
```

Paste the terminal output from the ls and other messages here.
