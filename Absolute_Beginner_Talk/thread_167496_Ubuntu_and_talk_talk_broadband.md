### Post by _simon_ on 2006-04-28
It will require windows because they will have supplied you with a USB ADSL modem that ONLY comes with Windows drivers.... 
Most ISP's don't support linux - support as in help you if you have a problem with settings. It doesn't matter which OS you use, in theory you will be able to connect, it's just a matter of *getting* your modem to connect.

If you provide all the modem details someone may be able to help you get it up and running or alternatively you could purchase a (wired) ADSL router with modem built in which does not require any drivers - a wireless one would however.

I use Pipex with a Linksys ethernet DSL modem and Linksys router.

### Post by jazzmuzik on 2006-04-28
I see, a USB modem. Very often these devices are using standard protocols and thus it's possible to run them in Linux whether the manufacturer supports Linux or not. For example, when I was on BellSouth DSL they sent me the modem with Windows software. But I didn't need any software because Linux was fully capable of dealing with the modem on its own. This may not be the case with this particular modem, but it's worth finding out. I did a quick search and found nothing. Doctor B many need to ask around.

My point above about software for windows is that often these programs are just fancy install programs that work without the user having to do anything. But often it's still possible to set things manually without the included software.

If Dr. B does decide to get a differrent modem, get one with an Ethernet connection. That is preferred over USB.
