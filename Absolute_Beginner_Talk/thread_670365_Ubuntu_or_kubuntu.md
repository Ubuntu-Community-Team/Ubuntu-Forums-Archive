### Post by eolson on 2008-01-17
For the most part just the desktop.
  Ubuntu uses Gnome
  Kubuntu uses KDE

Gnome "tends" to be a little cleaner while KDE "tends" to be a little more windows like.  "Tends"  because either can be changed.

There's also a differnece in applications,  but most can run under either.

### Post by aonegodman on 2008-01-17
):P  As already stated, the basic diff is the desktop and menu structures.

I have tried a variety of the linux distros based on debian as well as others and for the most part have settled on Ubuntu. Not only that, I have left Windows almost entirely. Linux is not pefect it has its issues, but compared to the issues with Windows it is superior.

I have tried Linux several times over the past 10 years. Each time I had problems installing or getting it to work in a dual boot setup. Next it was always the driver issues with graphics, printers, monitors and internet access. But when I tried Ubuntu using the LiveCD I was immediately impressed by the fact that it recognized and installed most, it not all, of the needed drivers and I was able to use it right off.

I have noticed that several specialized distros, mostly multimedia focused, tend to lean towards the KDE desktop for some reason. i.e. Studio To Go, 64 Studio, ect.

Get some various LiveCD distros and try them out. Different strokes for different folks!!!

Best to you.
