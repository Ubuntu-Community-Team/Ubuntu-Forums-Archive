### Post by PuleX on 2006-07-12
Syncing pics can be done with Gpixpod. 

The homepage for it: [http://www.gpixpod.org/wordpress/](http://www.gpixpod.org/wordpress/) 
You can download the .deb here: [http://prdownloads.sourceforge.net/gpixpod/gpixpod_0.5.0-0ubuntu1_all.deb?download](http://prdownloads.sourceforge.net/gpixpod/gpixpod_0.5.0-0ubuntu1_all.deb?download)  
Just install it from teh command line with: 
```
sudo dpkg -i path/to/gpixpod_0.5.0-0ubuntu1_all.deb
```
...and it will show up in your gnome Applications menu under Graphics.
