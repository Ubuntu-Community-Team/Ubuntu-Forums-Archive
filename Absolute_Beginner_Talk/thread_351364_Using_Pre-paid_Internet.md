### Post by psyne on 2007-02-01
If it requires special software you could try to install it using Wine. Or you could have her setup the account on a windows machine then use the dial up access from ubuntu to connect. Other than that I would say there are not too many options. 

Cable or adsl is extreemly easy, just get a broadband router like a linksys configure it with the correct settings and then connect to it using dhcp and your network manager.
