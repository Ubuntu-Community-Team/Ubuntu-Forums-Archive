### Post by mcduck on 2007-06-13
> **a.v.l said:**
> This is what happens.   I create a folder on the desktop, the icon for that folder appears but is to large for the desktop and I  have to manually resize it down.   I wondered if this process can be done automatically so that all icons which appear on the desktop are of the same size.
As Nautilus handles the desktop, changing default icon size in it's settings will also change size of desktop icons.

Just open your file manager and go to Edit/Preferences. On the 'Views'-tab there is setting for default zoom level for icon view. Change that to 50% or 75% and you'll get smaller desktop icons.
