### Post by starcraft.man on 2007-06-27
> **ktown said:**
> is imo, gimp, or krita on my kubuntu currently compressed or archived, or do i need to download them?

IMO = In My Opinion :p

gimp and krita are both in the repositories. Please open any terminal  (in Kubuntu, push Alt + F2 and then type in konsole and push enter) and paste this in :

```
sudo aptitude install gimp krita
```

Push enter and that will install the two programs and then they will be in the menus.
