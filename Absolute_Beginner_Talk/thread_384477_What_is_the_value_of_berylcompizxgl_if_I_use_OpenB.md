Openbox is a window manager; so are Beryl and Compiz (though they do compositing as well), and you can use one window manager at a time. XGL, not sure; if there's an addon to Openbox that does compositing, you may take take advantage of it. Or you can probably use xcompmgr, which is an old (and abandoned, as far as I know) compositing manager with it.
