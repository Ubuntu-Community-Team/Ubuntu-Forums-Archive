### Post by msak007 on 2007-02-23
The user "ubuntu" is the default user used for the Live CD environment. The only time I've seen that timer is when I set auto-login with a specified timeout, usually 30 seconds. That seems like an awfully long time for the load, and it is probably due to your system resources being too low for the Live CD environment.

As Taurus suggested, you can try Xubuntu or Fluxbuntu. Xubuntu is Ubuntu using the XFCE Desktop Environment instead of Gnome (Ubuntu's default DE). It is supposed to be lightweight and less resource intensive, making it ideal for machines with lower specs. Fluxbuntu is Ubuntu using Fluxbox (another lightweight DE). There is also Ubuntu Lite, which uses IceWM (but I can't try it or get more info as the site has been down for quite some time). There are many DEs to choose from, so if you're just starting I would recommend Xubuntu as it is the only one of those that is an official distribution.

Another alternative is to get the Ubuntu alternate installer CD, which is a text based install. It will be easier on your laptop because it is not booting into the Live CD environment which is what taxes those resources. I can't guarantee what kind of performance you'll get once it's installed though if you want to go that route.
