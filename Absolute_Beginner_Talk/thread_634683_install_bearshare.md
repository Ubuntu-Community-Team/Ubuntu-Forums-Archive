### Post by awthomp on 2007-12-08
You can install Nicotine, which runs off of Soulseek (another peer-to-peer program).  To install this, type:

```
sudo apt-get install nicotine
```

Also, you can check out the Synaptic Package Manager and search by description in the future.

As for Limewire, you can install it on Linux.  Visit: [http://www.limewire.com](http://www.limewire.com)
