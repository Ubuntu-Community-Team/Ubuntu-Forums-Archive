### Post by prismpirate on 2008-04-03
why would you need to add wine from a cd?

To install wine, just go to

Applications > Accessories > Terminal

A white box comes up

In that box, you paste:

```

sudo apt-get install wine

```

There! You get wine.


And yes, wine can help run window programs, but not all of them, just some.
To see a list of them, follow the following link:

[http://appdb.winehq.org/]("http://appdb.winehq.org/")

### Post by ichi@YUKI on 2008-04-03
Are you using Ubuntu now? I'm using Gutsy Gibbon (Ubuntu 7.10) and though I'm new to linux, that doesn't mean I never tried installing wine. I know two ways to do it.

If you are using Gutsy Gibbon as well, the easier way would be to use the Synaptic Package Manager (System > Administration > Synaptic Package Manager). You'll find that this is the easier way of installing packages in Ubuntu. I think that wine can be found in one of Ubuntu's Repositories (Universe, I think). In any case, just click on search and you'll find it.

The other way would be to use the Terminal:

-sudo apt-get install wine

Just type that, and it will install for you. Installing windows programs in wine can be quite different though, and so far the only programs I ever tried installing are uTorrent and VisualBoyAdvance. However, since both of them didn't run smoothly, I eventually got rid of wine and started looking for Ubuntu alternatives..
