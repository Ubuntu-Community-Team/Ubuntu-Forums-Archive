### Post by PartisanEntity on 2007-03-09
Find out how large the c,d,e partitions are. Note down the sizes.

Place the ubuntu cd in your computer and boot from the cd. Select the first choice "run/install ubuntu" this will load the live cd desktop environment into your memory (it will not touch your hdd).

Then the ubuntu desktop will load and on the desktop you will see the Install icon, double click it.

You will be asked for language settings, time settings, name, password, computer namr.

You will then also be asked how you want to partition your HDD. You must then select the partition you noted earlier and install ubuntu on it.
