### Post by Steve1961 on 2007-08-20
> If I'm running, say, Dia, will it allow me to save a file in .vsd format, so that I can send it to my work machine, or to other people, that use Visio?

The answer to that one is no, but you may be able to get visio working in Ubuntu with wine or crossover office.

Other than that, you shouldn't have any problems transferring files with a flash drive, openoffice supports all MS formats, and you can make either Gnome or KDE look however you want - whilst KDE has one panel by default you can run as many as you want, and whilst Gnome has two panels you can use one if you want to.  You can also place the menus and buttons wherever you want.  Personally, though, I like the default look of Gnome.

As for ease of use, there's certainly a learning curve involved because things work differently, but nothing that anyone who wants to learn couldn't get to grips with - my wife, a self-confessed technophobe, found the transition from windows to Linux easy, as she only uses the computer for web browsing, email and some office stuff.
