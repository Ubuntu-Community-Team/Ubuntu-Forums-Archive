I know how to install Thunderbird, but I was hoping to install it to the home directory so that if i really screw up Ubuntu and have to reinstall it... which is very possible... i won't lose my email (I put home on a separate partition).

### Post by aysiu on 2006-03-24
That's how it already is.

The executable binary for Thunderbird lives in /usr/bin, but your emails actually live in your /home directory:

/home/morwyn/.mozilla-thunderbird

or

/home/morwyn/.thunderbird
