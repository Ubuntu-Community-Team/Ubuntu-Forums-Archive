### Post by Snakob808 on 2008-03-06
I have a dv6000, I had a beast of a time getting it all up and running.  My advice for 7.10...  use madwifi or ndiswrapper for wifi... you'll need the Atheros ar5211.inf windows driver (for ndiswrapper).  

When you first try to install 7.10, use the "safe graphics mode."  There are lots of threads in the forums about the "noapci" boot parameter... but safe graphics was much easier.  Also, use Envy to get the nVidia drivers to work.  You can get them here... [http://www.albertomilone.com/]("http://www.albertomilone.com/").   The only other problem that I had, and am still having, is the speakers will not shut off while the headphones are plugged in.  

Let me know if you have any questions.
