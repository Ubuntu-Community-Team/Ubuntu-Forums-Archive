Xubuntu will run just fine on your laptop, and i think that Ubuntu will run pretty smoothly as well. I have a p3 669 Mhz with 256mb of ram with ubuntu installed, and it is running fine. 

As for 64bit, i've read on other threads that it seems that you have to do extra steps to install 32bit apps on 64 bit OS, and most of the times, the good software, such as firefox, opera, aren't customized for 64bit.

But if you don't mind a bit of tweaking around, then go ahead and install 64bit (people say that it is the future...:D)
