### Post by UltraMathMan on 2006-08-09
The easiest method would be to just create a panel, deselect expand in its properties, set the size at something like 50 pixels, and set the background as transparent. However this panel would behave as the top and bottom panels do, that is, any maximized window would stop at its [the panel's] border.

I know there are some things you could use to do something more like an OS X dock, but I don't really have much experience in them...
