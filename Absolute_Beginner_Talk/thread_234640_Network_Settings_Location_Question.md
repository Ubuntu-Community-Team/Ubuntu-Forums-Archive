### Post by ebob on 2006-08-11
You can enter anything you want, or nothing at all.  Supposedly, this allows one to have different sets of network settings.  This is mostly for using with a laptop.  At home, one may use a wireless router with different settings than one would find when one goes to a wireless hotspot.  By setting up different "locations", you can easily switch from home settings to a different set of settings.  Think of it as a profile for your network settings.  For example:  At home, I use WEP encryption, so I need to have a key entered.  When I am out and about, there is no encryption on public hotspots, so I have another "location" set up for the different hotspots I frequent.  If I didn't do this, I would have to re-enter the WEP key when I returned home (which is tedious).

I say supposedly because it sometimes messes up and loses settings.  The method for saving the settings is not intuitive.
