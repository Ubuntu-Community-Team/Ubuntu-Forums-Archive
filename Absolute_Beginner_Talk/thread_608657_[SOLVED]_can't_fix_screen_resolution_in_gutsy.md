### Post by kshane on 2007-11-10
> **kleo skywalker said:**
> Thanks for your patience.

No prob...  Did you make a back-up copy?

If not, do so now.  

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak
```

If you did, in terminal, try this:

```

sudo dpkg-reconfigure xserver-xorg

```

When it runs, make sure to give your resolution you want (1024 x 768 ).

***Another way to do it*** would be to pull up your xorg.conf and remove the higher resolutions listed towards the bottom.  Making sure the resolution you want remains in the list.

After ward, reboot.  If you have problems getting back to your graphical screen, from the command prompt, enter:

```

sudo cp /etc/X11/xorg.conf.bak /etc/X11/xorg.conf

```

then reboot.

Kevin
