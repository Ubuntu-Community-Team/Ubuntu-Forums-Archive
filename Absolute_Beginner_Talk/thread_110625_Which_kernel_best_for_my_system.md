### Post by dalee on 2005-12-31
Hi,

I think the best kernel is the one that works:) . Honestly, if all your hardware is working as you want it to, and you have no security issues with your present kernel, then there is little reason to change it!

The only times that I've ever recompiled was to get a piece of hardware working better or for security reasons.

But if you want to fool around, then pick the latest and greatest and have some fun!
