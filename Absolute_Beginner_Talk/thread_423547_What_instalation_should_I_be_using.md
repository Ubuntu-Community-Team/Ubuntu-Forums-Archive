### Post by Sef on 2007-04-26
> I have an old 233 MHz Intel Pentium with a 4 GB harddrive, and 128 MB ram

I would recommend [Xubuntu]("http://xubuntu.org").  It is made for older laptops.  With 128 mb ram, it should do well.  Xubuntu is a lightweight window manager.  

These are the system requirements from their website:

> Minimum system requirements

To run the Desktop CD (LiveCD + Install CD), you need 128 MB RAM to run or 192 MB RAM to install. The Alternate Install CD only required you to have 64 MB RAM.

To install Xubuntu, you need 1.5 GB of free space on your hard disk.

Once installed, Xubuntu can run with 64 MB RAM, but it is strongly recommended to use at least 128 MB RAM.
