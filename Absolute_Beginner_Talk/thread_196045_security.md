### Post by mlind on 2006-06-13
You don't need anti-virus software when you're running linux. I think there's no
existing viruses for linux. Just don't run unknow scripts as root and you'll be
safe. If you really really need a virus protection for some reason, try ClamAV.

Linux kernel contains built-in firewall called **iptables**, but for default, you don't
need it either. There's no open ports on default Ubuntu install. See *firestarter*
if you want easy tool for configuring iptables.
