### Post by xopher on 2007-02-18
Not yet.

Id recommend using x86 for now. The speed increase is marginal, if that even. And the problems you will run into with the 64bit install aren't that pleaseant.

Eg. Flash won't work, without a workaround.
Java 1.5 won't work in firefox. The list goes on.

It's getting better. But it's still not there yet. It will be, but it isn't :)

Personally I've been using 64bits since Breezy, and Ive got everything running quite well. But I would have saved many headaches if Id use the x86.
