### Post by Bloch on 2006-07-04
I know that the prboom game can have X11 and sdl versions. However most software just has one version.

SDL = Simple DirectMedia Layer. It is software which will make use a your graphics card if you have one. The X11 version does not use this newer technology.

The game (usually it's a game or multimedia app) will run faster if SDL is used. Maybe on an older computer the X11 version would work better.

GTK is used to design a user interface. It controls window borders, how menus look etc. Check to see if the alternative version has the ubuntu logo in synaptic, or if it says it is for the gnome desktop. (presuming you have gnome)

GTK apps can look a little old-fashioned and clunky, but they will look the same and work on all linux systems.

Choose the "normal" version - use your own "fuzzy logic" to guide you!
