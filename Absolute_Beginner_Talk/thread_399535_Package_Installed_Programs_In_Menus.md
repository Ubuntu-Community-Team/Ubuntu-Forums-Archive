when you install a program using synaptec package manager or a .deb file, is there any way to put a shortcut to the program in the menu bar like it does in  add or remove programs?

### Post by theicyj on 2007-04-02
You can right click the menu bar, then click "Edit Menu".  A window will pop up in which you can edit and add to the menus to meet your needs.
