### Post by shirilover on 2007-08-05
Well, I have found them.
The dagger is U+2020
[FONT="Times New Roman"][SIZE="7"]†[/SIZE][/FONT]

The double dagger is U+2021
[FONT="Times New Roman"][SIZE="7"]‡[/SIZE][/FONT]

and the pilcrow is U+00B6
[FONT="Times New Roman"][SIZE="7"]¶[/SIZE][/FONT]

You can either use the character map and look in the 'Common' section, or use the unicode input method of Ctrl+Shift+U, and while holding down Ctrl+Shift, the unicode hex values above.
