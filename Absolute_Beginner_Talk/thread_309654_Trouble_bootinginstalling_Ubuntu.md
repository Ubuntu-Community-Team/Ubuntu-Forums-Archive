### Post by wieman01 on 2006-11-29
Alright... You need to know nothing about the mentioned programs to install Ubuntu either. See? No difference at all.

Now, I presume you have downloaded the Ubuntu CD, correct? If so, make sure you are able to boot from CD ROM. That's a setting in the BIOS. That done, you need to insert the CD & let Ubuntu boot first. It is a Live CD which means it runs a functional system without installing any programs or occupying harddisk space. Try this first to test your hardware... If the Live CD does its job, you can begin to install (double-clicking the icon on the desktop). Tried that?
