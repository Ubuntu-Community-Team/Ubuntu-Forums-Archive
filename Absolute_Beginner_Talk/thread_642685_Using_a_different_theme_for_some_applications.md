### Post by avik on 2007-12-17
It is possible to change the color of the form widgets (or any part of the interface) on Firefox, but you have to know CSS.  Create a userContent.css in your /home/*<username>*/.mozilla/firefox/*<stuff>*.default/chrome folder and add in any CSS you want applied to all the pages.

If you don't know CSS, I don't know what else to do.
