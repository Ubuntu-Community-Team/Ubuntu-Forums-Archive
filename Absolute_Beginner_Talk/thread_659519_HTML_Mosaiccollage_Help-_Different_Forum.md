### Post by LaRoza on 2008-01-05
The HTML is easy for a link:

[html]
<a href="http://laroza.freehostia.com/home" title="LaRoza's Home Page"><img src="images/laroza.png" alt="LaRoza" /></a>
[/html]

I see you want an Image Map

[http://www.kasparius.com/tutorials/imagemap/index.htm](http://www.kasparius.com/tutorials/imagemap/index.htm)

It is valid (I didn't fully review the tutorial above, so it may not be.)

Don't use a WYSIWYG editor, learn XHTML, see my wiki.

### Post by frup on 2008-01-05
There is a feature you can use with hrefs and images that allows you to make a map and link to co-ordinates in an image. I have always found it much more trouble than it is worth and tend to cut and image up and use css to position the images instead (i don't even know if the maps are w3c standard html in fact). The signature you have made is a pretty simple layout and just using the odd div to position the elements should be incredibly simple. HTMLDog.com is a great resource for html and css and it would be wise to learn from there and understand HTML rather than just using a WYSIWG editor.
