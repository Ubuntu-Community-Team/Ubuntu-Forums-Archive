### Post by Jeldert on 2007-02-02
Maybe you turned on "Proposed updates". This are updates which are released to be tested, before being turned into a 'real' update. When turned on you receive more updates.

You can turn these updates on and off by going to System > Administration > Software Sources. On the tab "Internet Updates" you should uncheck "Proposed updates"
