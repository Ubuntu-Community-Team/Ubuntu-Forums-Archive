### Post by mcduck on 2008-01-22
When you have the video open in Firefox just open the file manager and browse to /temp. You'll find the video file there. No extra tools needed.

If you want, you can then install ffmpeg2theora and use that to convert the video to .ogg.

### Post by billgoldberg on 2008-01-22
Farout the most easy way is installing the program "youtube-dl".

Basicly you install the program using synaptic.

You copy the url from a youtube video. Open a terminal and press "youtube-dl url". And it's downloaded to your home directory.
