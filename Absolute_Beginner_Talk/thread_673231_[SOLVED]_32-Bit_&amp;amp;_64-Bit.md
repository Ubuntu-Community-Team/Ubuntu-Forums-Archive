Every single (user-market) 64bit processor, available today, is also a 32 bit native processor. Except something like (Itanium processor) 

This means that the performance when running 32 bit OS and apps is exactly the same than another equivalent 32 bit processor.  Of course you won't enjoy any of the 64 bits performance you CPU can offer. But nowadays this is still something difficult to feel if you don't need tons of GB to redirect specially if you use programs like MAYA.  

I would  recommend  you to use 32 bits os. 64 bits native software are still coming.

About hardware, every single device will work perfectly.
