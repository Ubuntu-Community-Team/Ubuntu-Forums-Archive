I don't remember the website where I saw this tip and I've tried to get back to it, but I can't find it.

Anyway, this guy suggested that instead of using "rm" (remove) in a command, first replace it with "ls" (list).  This will tell you everything the "rm" will do do your system before it does it.

He says to do this especially if you're experimenting with wildcards (which is how I found it...I was looking for wildcards).

I hope this helps somebody.
