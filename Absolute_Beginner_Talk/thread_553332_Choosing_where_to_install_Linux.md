### Post by Majorix on 2007-09-17

### Post by 505 on 2007-09-17
In the setup you can choose where to install Ubuntu. By default it chooses the largest free partition, but you can choose an advanced option. There you can choose how to partition your hard drive and where to install Ubuntu.
