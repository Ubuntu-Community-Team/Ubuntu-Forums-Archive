### Post by Omnios on 2008-03-02
Hi as for vista My install is about a month old or so and did a diskcleanup with the remove all but the last backup. Anyways this removed shadow copies which cleaned up about 15gigs. 
 
 Also with that much storage you might want to research partition laout. 
 
 The main root file is the root directory / . There is also a /home directory within the file system. For the / 30G or so is god for that. The /home is where all your personal files will go such as download music video etc. When using the partiton editor you make partitions for the root / and /home seperate from the / . Normally /home is in the root / directory so all files will be in the same partition. By making a seperate partition for /home it is seperate from the / and if the / craps out your personal files should be ok.
 
 In the partition editor you can shrink expand and make partitions. Also in advanced you can set the mount points for the different partitions. Another benifit is that you can use a windows ext3 program to mount you /home drive in windows and even possibly use it as the documents drive.
 
 One note it is not recomended mounting the root / partition in windows and windows does not use linux permissions so they would be accessable in windows and windows problems.
