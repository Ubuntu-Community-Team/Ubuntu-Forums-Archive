### Post by mejy on 2007-04-30
You'll generally have to disable and exit all virus scanners/firewalls etc., disable hibernation in power preferences and turn off the page file in advanced system properties.  If that still doesn't fix it, try rebooting into safe mode and defragging (hold f8 on boot to do that).  It's all fairly well documented if you wish to look further into it.
