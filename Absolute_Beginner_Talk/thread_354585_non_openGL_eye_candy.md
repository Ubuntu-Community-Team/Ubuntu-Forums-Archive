Something I found that adds a little spice to Ubuntu is something called Murrine.  I don't know a lot about it, but I went to gnome-look.org and saw a bunch of cool themes that required it.  Here's some more information on it:

[http://cimi.netsons.org/pages/murrine.php](http://cimi.netsons.org/pages/murrine.php)

I don't *think* you can run beryl if you don't have opengl support.  And murrine isn't a replacement by any means.  But it spices up the standard install a little more than normal.  If you install it let us (me) know what you think!
