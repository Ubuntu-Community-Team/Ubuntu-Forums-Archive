### Post by Bachstelze on 2006-08-30
Very well known issue, and easily fixed ;)

Open your sources.list in gedit, delete everything so the file is blank, save, run *sudo apt-get update*, pste your sources.list back, save, run *sudo apt-get update*, voilà :)
