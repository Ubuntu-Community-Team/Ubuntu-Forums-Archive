### Post by Paqman on 2008-01-27
Checkout the [Nautlius Image Converter](http://packages.ubuntu.com/gutsy/gnome/nautilus-image-converter) plugin. It'll allow you to resize images from a right-click. Uses Imagemagick. Dead handy.

### Post by Jerry N on 2008-01-27
Xnview and Irfanview both do batch size conversions, among many other things.  These are Windows programs but both run under Wine.  I have had better luck with Xnview in this respect.  Xnview has a Linux version but it is useless (they claim to be working on a new Linux version but I am not holding my breath).  I have not been able to find a Linux equivalent of Xnview.  Xnview is not a substitute for Gimp.

Jerry
