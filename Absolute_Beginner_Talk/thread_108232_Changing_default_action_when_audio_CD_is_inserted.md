### Post by AndyC_772 on 2005-12-25
When I put an audio CD in the drive, Ubuntu automatically launches Sound Juicer. I'd prefer it to run GRip instead.

How do I change it?

---

### Post by kaamos on 2005-12-25
In a terminal, type:
```
gnome-volume-properties
```
And choose the action for an audiocd under multimedia.
