### Post by charles.g.moore on 2007-04-03
if you want to see the desktop, you could use gnome-rdp which is a remote desktop app. or you could use VNC both are in the repo's, you could try tightVNC, just do a google search and take a look at them first.  I would proly go with tightVNC.

If you want to transfer files you could install samba and that would let you see each other on the network for filesharing.
