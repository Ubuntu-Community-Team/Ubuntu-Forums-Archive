### Post by bumanie on 2008-03-25
I believe vlc player will play rmvb files. Open vlc, click on file --> click on Quick open file --> find the file you want to play and click open. Vlc will play it. You could also try helix player, it's an open source player that is meant to play real media files. Using automatix won't teach you anything - it does everything for you, hence its name auto....
