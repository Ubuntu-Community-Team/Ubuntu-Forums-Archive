### Post by Pablo_Escobar on 2005-10-18
1. Rumour - they work both very well installed together
2. Breezy has some new features like editing Services and other things
3. Faster, hmmmm, I haven't noticed any significant speed boost, but the desktop feels more snapy :)
4. Backports are the upto date packages, which are not 100% tested and therefore they are not in official repos (it's somewhat safe to install them if You want the latest software). Breezy backports will be available soon.
5. Starter Guide for Breezy will be completed soon
6. As for bootup I have to say that Hoary seemed to boot a bit faster, but maybe it's a thing about configuring services in my system (it's a pretty fresh install :) )
