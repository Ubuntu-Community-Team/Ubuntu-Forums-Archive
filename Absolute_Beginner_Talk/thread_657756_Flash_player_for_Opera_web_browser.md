### Post by RomeReactor on 2008-01-03
Hi. The stable version of Opera at the moment (9.25) has trouble detecting Flash; you can use the [latest beta (9.50)]("http://www.opera.com/download/linux/?ver=9.50b"). If you have Flash working in Firefox, then installing 9.50 will let you have Flash in it.
