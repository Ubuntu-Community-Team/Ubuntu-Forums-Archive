 Hi Roland,

I'm not familiar with Macs myself, but try rebooting with the CD in the CD-ROM drive. If that doesn't work, reboot and hold down "C" as it boots.


Possible problems that might crop up if you did something wrong:
* Did you download the PPC (not Intel x86) version?
* Did you burn the image to a CD (good) or burn the .iso file to a CD (bad)
* Are there BIOS settings on Macs? If so, make sure it is set to boot from CD

Good luck,
Damon
