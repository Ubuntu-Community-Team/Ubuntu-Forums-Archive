### Post by encompass on 2007-07-28
Even a new distro might be nice.  But odds are a simple install is ok.
Installing most software in linux is rather simple.  You can add repositories and use the ones you have to install thousands of peices of software to help you.
Any easy way to get right into it is to use synaptic, located in System--- Administration--- Synaptic Package Manager.
In there you can search for software you like.  Don't search for the exact same name of the old software, but rather, search for what you want to do!
For example... bible
turned up MANY entries... one that I use is gnomesword.  If you don't see it there, tell me.

### Post by ramjet_1953 on 2007-07-28
When you go into synaptic, you need to enable the extra repositories.

[COLOR="Blue"]Settings>Repositories[/COLOR]

Then put a tick in every box, except Source Code.

If you do this and then do another search using bible, you will see quite a few packages.

### Post by Pencil Head on 2007-07-28
Don't know if it's available for Dapper, though I guess it would be, but, if you haven't already. you may like to have a look at '**Sword**' and '**Bibletime**' which is a front end for Sword. Just search for 'Sword' in synaptic, and they both should show up. For Sword you would want the package called 'libsword6' or whatever version is available for Dapper (if any), all the other Sword stuff that you will see available to download from within Synaptic is also available to be downloaded from within Sword/Bibletime itself, which is the way I prefer to do it. You can see what the individual modules are by checking the web site.

Sword
[http://www.crosswire.org/sword/index.jsp](http://www.crosswire.org/sword/index.jsp)

Sword Modules
[http://www.crosswire.org/sword/modules/index.jsp](http://www.crosswire.org/sword/modules/index.jsp)

Bibletime
[http://www.bibletime.info/](http://www.bibletime.info/)
