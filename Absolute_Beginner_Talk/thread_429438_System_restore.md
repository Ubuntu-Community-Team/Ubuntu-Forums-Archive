### Post by starcraft.man on 2007-05-01
You probably wanna look at a drive imager, that way you make an exact clone of a drive before you modify and then put it back if you mess something up. 

[Here]("http://linuxappfinder.com/backupandrecovery/driveimaging") is a link to some applications for doing such. 

On a related note, you may only want something a bit easier like a back up software in Ubuntu, for that browse through [these]("http://linuxappfinder.com/backupandrecovery/backup") 

Read through the documentation before using any of the imagers, don't wanna mess up your install :)

I myself use Acronis but it is paid and truth is had I known about all the free ones available I wouldn't have gotten it, so look through those.
