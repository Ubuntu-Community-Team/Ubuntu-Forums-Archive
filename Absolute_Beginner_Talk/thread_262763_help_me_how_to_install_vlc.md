### Post by benfindlay on 2006-09-22
VLC is listed in one of the repositories that aren't turned on by default (sorry, don't know which one off the top of my head)

If you launch synaptic and look in the menus for repositories, it brings up a list of different ones to pick from. Just turn them ALL on, and click OK. Then you'll get some message about needing to reload. So just click the reload button (top left I believe). Now VLC should be listed!

### Post by dmizer on 2006-09-22
you're likely going to need more than vlc to play internet media unless you've already done some configuring.

two options here for you.
1) you can browse through the wiki on restricted formats and manually configure your media: [https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)

2) you can use an automatic configuration script like easy ubuntu (i recommend): [http://easyubuntu.freecontrib.org/](http://easyubuntu.freecontrib.org/)

it depends on the level of interaction you want with configuring your system.
