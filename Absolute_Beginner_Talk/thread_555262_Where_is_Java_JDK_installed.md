### Post by Daveth on 2007-09-21
well, have you actually downloaded the development kit (JKD) from Synaptic? 

All you will have as standard is the Java Runtime Environment (JRE), which is often bundled with JKD but is also available on its own.
As to where it lives, ask 

```
whereis java
```
