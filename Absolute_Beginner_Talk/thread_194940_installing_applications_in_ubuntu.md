Ok to install applications you have synaptic with Gnome and Adept with KDE.
(On the toolbar at the top) System-Administration-Synaptic package manager.

You can use this program to search for programs/games etc.

Remember that you can get 18,000 packages here, unlike windows where you have to go searching for stuff to download and install.

To enable root, you open the Terminal (Applications-Accessories-Terminal)
and type :-
> sudo passwd root
Just invent the password (write it down !!)
Now you have root access when you need it :)
