### Post by BaffledMollusc on 2007-08-08
You mean you get pop-ups while running Ubuntu? If so, since you're running linux, I'd guess it's highly unlikely that malware has been installed.

Could it just be advertising pop-ups from a specific web site? If so, it's browser related. If you're using Firefox have you blocked pop-ups? If you have, another option would be to install the Firefox Adblock extension and block whatever site is serving that ad.
