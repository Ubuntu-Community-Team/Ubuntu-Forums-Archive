### Post by igknighted on 2007-03-14
Ubuntu is based on Debian, and hences uses Debian packages (.deb).  Mepis, Freespire, Mint, Debian and many others also use this package type.  RPM is used by distros such as Fedora, Red Hat, Suse and Mandriva (among others).  Before you install that program, however, take a good look in Synaptic for it.  Installing through Synaptic (or APT) is always preferable to downloading random packages from the internet.
