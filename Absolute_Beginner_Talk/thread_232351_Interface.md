### Post by carlosqueso on 2006-08-08
[http://xwinman.org/](http://xwinman.org/) is where you can see a quick overview of the window managers and desktop environments.  They all do pretty much the same thing, although the Desktop Environments (near the bottom) tend to have a more complete setup.  Ubuntu uses GNOME, Kubuntu uses KDE and Xubuntu uses XFCE, but you can put almost any of the window managers on top of that (they're in the repos.)
