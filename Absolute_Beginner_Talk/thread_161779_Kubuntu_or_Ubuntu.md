### Post by catlett on 2006-04-17
Just my two cents to the new guys who want to try out "all" the linux distros.  They are not going to look very different. The major distros that you are going to bump into all have Gnome or KDE as their desktop. When you put in Fedora FC5 it is Gnome with a Fedora backsplash. If you get the Knoppix live cd and throw it in to see if it is easy and have everything, it is KDE with racing cars. All the desktops look   basically the same (at least to a newbie like me)..
The difference is in the upkeep. How you download programs and upgrades. How the terminal interacts. The difference is in the "system" not the "desktop". It's the ubuntu system with Gnome running on top. You're going to compare it to the Red Hat system with Gnome running on top. So there isn't this great variety of desktops. You should look for the one that you can interact with the best.
The only people who can really tell the difference are the ones who know their kernels (so to speak). The people who use the computer as a tool and not a toy. They use the other dektops (fluxbox etc) because it works with what they are doing. People who know the command line in and out don't want to point and click, it's too time comsuming. And they don't want pretty icons and shadows becasuse it is slowing down what they are doing.
Antways, all your going to see is Gnome and KDE. The diffderence will be in command etiquette, your package management system and the splash screen. It's still fun to install new distros. I average at least one a week. If I have insomnia on the weekends it can get up to three. I've been through the main ones and some lesser known ones but they aren't very different(at least to my amateur eyes). I always end up in front of Breezy at the end of the day anyways. Just my two cents (more like a quarter I suppose):D

### Post by Sutekh on 2006-04-30
To find out how much size they will take up, open Synaptic, and mark for installation the **kubuntu-desktop** meta-package.  

Click Apply, and Synaptic will tell you how much the download will be and how much space it will take up once installed.  If you are happy with the numbers than OK it.  If not then cancel the installation.  

You can do the same with the **xubuntu-desktop** package.

As for swapping between interfaces, simply logout of one and when you are at your login screen, click Sessions and choose the new interface.  Then you can log into that one.

### Post by gingermark on 2006-05-01
First off, if you are going to try out other environments then use aptitude to install them, as this will make their removal easier if you don't get on with them.

For example, if you installed ubuntu, then wanted to try kubuntu, you would do ```
sudo aptitude install kubuntu-desktop
``` and the many packages that make up kubuntu would be installed. ```
sudo aptitude remove kubuntu-desktop
``` would then remove all those packages if you then changed your mind.

If I was to generalise, which is never a good idea, I'd say Gnome is good for beginners and experts, whilst KDE is probably better for intermediate users. The reason I say this, is that you can configure more from the GUI in KDE, which is why I use it. But there are lots of options.

Gnome has less config options via the GUI, which might suit newer users better, whilst I'm sure experts can just edit configuration text files if they want to change something.

Obviously this is big generalisation though, and the only real way to figure out which one is suitable for you is to try them both.
