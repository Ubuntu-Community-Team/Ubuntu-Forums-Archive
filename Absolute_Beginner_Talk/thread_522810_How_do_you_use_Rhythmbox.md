### Post by aysiu on 2007-08-11
You throw them into a music library folder (in the Rhythmbox preferences, you specify *which* folder is that folder).

Rhythmbox will scan subfolders as well, so you can organize things however you want.

If you're importing CDs, you can also specify in the Rhythmbox preferences the folder hierarchy for newly added songs (Artist / Album or Artist, etc.).

### Post by aysiu on 2007-08-11
Well if you're asking how people use Rhythmbox, here's my answer:

I use Rhythmbox the way it was intended. If I have a CD, I rip it, and Rhythmbox uses the prescribed folder hierarchy to import the songs. If I have a single song not from CD, I move it to the music folder.

I like generally being able to find stuff through the folders, but the easiest way for me to copy songs to my MP3 player is to drag and drop straight from the Rhythmbox interface to the MP3 player window instead of navigating through all the folders to find the songs I'm looking for.

Actually, I create extra work for myself because Rhythmbox doesn't allow you *not* to have a folder hierarchy. I prefer to have no subfolders--just one music folder with a whole bunch of songs (the file names include the artist and song names)--so I use Rhythmbox to rip the CD and then move the songs out of the subfolder into the larger folder.
