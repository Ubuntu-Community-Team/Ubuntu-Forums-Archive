### Post by tubasoldier on 2007-11-22
There are other editors out there besides gedit. 

VIM is by far my favorite, although a new user would be completely confused. It is used in programming all the time because once you know the keyboard shortcuts it can be very efficient compared to other text editors. 

other editors would include Nano, joe, emacs, kate, kwrite, jedit, the list goes on and on. You can open up synaptic and search for editor and get quite a bit.

Since you are editing web pages you may want to give Kompozer a shot. it is not in the Ubuntu repositories but easliy found on google. They have an ubuntu package and it is easily installed. 

Good luck.
