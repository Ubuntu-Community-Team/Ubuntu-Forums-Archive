### Post by Paul820 on 2007-08-02
If you want to open a .pdf file, right click the file, go to properties, click the 'open with' tab and select the program you want it to be opened with. If the program you want to use is not there, click the add button, a big list of applications will come up. If the program you want is not in there, then you can click the little arrow at the bottom that says 'use a custom command'  and enter the name of the program you want to open the particular file with.
