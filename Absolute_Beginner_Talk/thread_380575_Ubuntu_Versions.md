### Post by Bachstelze on 2007-03-09
```
uname -r
```

2.6.15 = Ubuntu 6.06 (aka "Dapper Drake")
2.6.17 = Ubuntu 6.10 (aka "Edgy Eft")

Edgy is the latest but some people have problems with it and prefer to stick with Dapper, which is very stable. The next version is Ubuntu 7.04 (aka "FEisty Fawn"), to be released in mid-April.
