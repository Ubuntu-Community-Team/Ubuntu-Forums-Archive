Deleted file go to a folder called .Trash in you home directory. Since it starts with a dot, it is notmally hidden and you have to go View->Show hidden files to see it.
Alternatively, if you are seeing folders called Recycler, these are created by Windiows if for instance, you share a USB memory stick with windows.
