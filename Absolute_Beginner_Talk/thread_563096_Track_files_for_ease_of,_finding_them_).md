### Post by ryanVickers on 2007-09-29
I just had an idea the other day when I was going "open recent" in a program, and then remembered I moved/renamed the file.  The idea is this:

Wouldn't it be a good idea to have some kind of database, like a FAT but better :) that's simply a file in /etc or something that tracks changes of any file.  ex. If you make a file, and then move it, it would store the fact that it was moved, and to where in the database!  Then when a program goes looking for it and can't find it, the first thing it does is refer to the database, and search up the file, realize the new location/name, and open it!

I don't think this would by ***too*** hard, and it would be very useful!!  Also, I'm pretty sure no other OS does this yet, so that would be pretty cool too! :guitar:
