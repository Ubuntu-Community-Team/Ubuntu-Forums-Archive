### Post by dstew on 2008-02-07
Your computer has to be able to boot from the USB drive. If it cannot do that, you can use the floppy drive to install a tiny system, that will let you install the rest over the internet (or from a non-bootable USB stick). I think you can make a grub boot floppy, and use that to boot the USB stick.
