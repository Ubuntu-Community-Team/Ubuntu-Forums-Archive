### Post by stroyan on 2007-11-21
You can get the common essential packages with 
```
sudo apt-get install build-essential
``` 

You can get packages required to build a specific ubuntu package with 
```
sudo apt-get build-dep pkg
```
where pkg is a package name.

If you are missing a header file for building you can search for it with
```
sudo apt-get install apt-file
sudo apt-file update
apt-file search /usr/include/foo.h
```
