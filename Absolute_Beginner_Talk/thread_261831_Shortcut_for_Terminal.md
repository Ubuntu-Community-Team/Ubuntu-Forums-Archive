### Post by cstudent on 2006-09-20
You can set a keyboard shortcut by going to the System menu > Preferences > Keyboard Shortcuts.


You can also install the package nautilus-open-terminal. This will give you a menu option when you right click on the desktop or in a nautilus folder.

```

sudo aptitude install nautilus-open-terminal

```
