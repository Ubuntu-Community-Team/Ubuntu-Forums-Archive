### Post by Jagot on 2006-07-15
The way the files are fragmented should not pose any problems.  You are aware that you do not need the GParted live cd - the Ubuntu installer has it's own partitioner with which you can perform all of the actions you require.

You do not need a 2GB swap.  There's a point where it just wastes space that you could be using elsewhere.  The general rule is the swap should be twice the size of your RAM, so with 512MB the maximum you should make your swap is 1GB.  I'm running with ~600MB and it has never caused a problem.

6GB should be fine for / as long as you don't install humongous amounts of packages.
