### Post by greg_g on 2007-06-29
My suggestion is to make sure that you don't depend on any software that is windows only.  I know some people really like QuickBooks or whatever for managing finances etc and those aren't available for Linux.

The only problem I have had so far with ditching Windows completely is convincing people that YES, if you say Netscape Navigator on WIndows will work then Firefox on Linux will work also!  But it really isn't an issue because I haven't had any problems with usability, just educating people  :).

Before you throw out or microwave your Windows install CD I would work with Ubuntu for 1 month and make sure you don't run into any little snags where you think "Man, if I could only boot into Windows for 5 minutes and do this one thing, I would be so happy."  If you don't have any of those moments, they ditch it!

Now, when you think to yourself that it might be easier to just boot into Windows to do something, make sure you haven't missed an OSS alternative, usually there are.

Anyways, Welcome Aboard!


greg

### Post by LaRoza on 2007-06-29
The command line is the same for all distros. If a distro has different programs, you will need to install them if you want to use them. I use a Fedora command line reference book for my commands in every distro.

The shell can be different, but it is not usually significant.

Doing a search for "Linux commands" will give you a lot, and if you have a question about a particular command, type "man commandName". "man" gives you the manual.
