### Post by jw5801 on 2008-04-01
There's a few alternatives, the one I remember reading about most recently is:
[http://www.linux.com/feature/54191](http://www.linux.com/feature/54191)

You'll probably need to assess what remote desktop software is available on your college PCs first however, and set up your home PC accordingly. I assume if they're running Ubuntu, your college computers will have rdesktop which uses RDP (the Windows remote desktop protocol), I'm sure there's a way to set up your computer to use that if all else fails.

Here's a more complete article:
[http://blog.lxpages.com/2007/03/13/remote-desktop-for-linux/](http://blog.lxpages.com/2007/03/13/remote-desktop-for-linux/)
