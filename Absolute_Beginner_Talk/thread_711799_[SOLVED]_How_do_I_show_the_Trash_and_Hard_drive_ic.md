### Post by baxterdog on 2008-03-01
Not to impose my views here,.... but why do you want these things on your desktop? My preference is a nice clean desktop so that my wallpaper can shine. The icons to link to these things are still where they always were. 

BTW, try using the keyboard shortcut options in System -> Preferences -> Keyboard Shortcuts to create your own methods of bringing these things up.

For example;

Terminal = ctrl-alt-t
browser = ctrl-alt-b
home folder= ctrl-alt-h

etc... use what feels comfortable to you!
