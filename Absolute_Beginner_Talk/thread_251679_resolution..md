### Post by croak77 on 2006-09-05
Are you using the nvidia driver or the nv? Make sure to install the nvidia driver or else the nv might not be able to get a higher resolution.

[https://help.ubuntu.com/community/BinaryDriverHowto/Nvidia](https://help.ubuntu.com/community/BinaryDriverHowto/Nvidia)

You can run from a terminal
```
 sudo dpkg-reconfigure xserver-xorg
```
It willl ask you a bunch of questions about mouse,keyboard,driver, and resolution. 
Or you can edit /etc/X11/xorg.conf.
```
 gksudo gedit /etc/X11/xorg.conf
```
And put the correct values in yourself. You'll need to know your monitor's HorizSync and VertRefresh, make sure the values are correct, then you change the "Modes" lines to;
```
Modes "1280x1024" "1024x768" "800x600" "640x480"
```
