### Post by wishyjr on 2005-10-26
ok, welcome to the land ubuntu!

1. [http://www.ubuntu.com/download/](http://www.ubuntu.com/download/) go here and download an install .iso file (depending on your sytem) and burn it to a blank CD.

2. place the CD in your pc and reboot it- the installer should start and if you follow the instructions carefully, you should been running soon enough. :)

Also, If you just want to try ubuntu out, download a 'LiveCD' version. This loads up ubuntu from the CD but does not install it so you cant break your system! ;) Follow the two steps above to get a LiveCD running.

If you run into problems try going here:
[http://www.ubuntu.com/support/supportoptions/freesupport/](http://www.ubuntu.com/support/supportoptions/freesupport/)

good luck!!! :)

### Post by wishyjr on 2005-10-26
well it doesnt have to be, but truth be told - it is probably the hardest part of the install process. My advise to anyone wanting to dual boot is..

make sure you choose to manually edit the partition table when it asks you!!

then it'll show you all of the partitions on your hard drive. select a partition for your ubnutu installation (use the ext3 filesystem). If you need to create partitions (if you only have one partition) then the installer can sort that out pretty simply. also, make a another partition called a swap partition. This only has to be a small size about double the size of your RAM.

Ubuntu is clever enough to not bother with the partition that windows is on
