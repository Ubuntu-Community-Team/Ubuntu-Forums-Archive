### Post by ubuntu27 on 2007-05-09
You can use the terminal/konsole to download youtube videos:

Download [YouTube Downloader]("http://www.arrakis.es/~rggi3/youtube-dl/") by:

```
sudo apt-get install youtube-dl
```

now run this command:



```
youtube-dl <insert URL web address here>
```

that will download the video automatically for you.
