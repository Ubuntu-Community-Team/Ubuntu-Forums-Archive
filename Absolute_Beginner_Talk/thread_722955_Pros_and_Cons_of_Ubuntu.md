### Post by scragar on 2008-03-13
you can try before you choose to install ubuntu with the liveCDs if you are feeling in any way aprehensive about installing, and you can run windows and ubuntu in what is called a dualboot, where you have the option of which OS to run when you start/restart your computer.

### Post by Tatty on 2008-03-13
Excellent.  Youre going to have to be prepared to learn though.  The biggest problem most purely-windows users have (myself included when i first started using ubuntu) is that they expect it to work just like windows... only looking slightly differently.

The reality is that Linux is not Windows... they are two different things so be prepared to face the possibility that at first you are going to feel lost.

But stick at it, and eventually you will become as comfortable with Ubuntu as you are with Windows.  Infact i am more comfortable in Ubuntu now... there are so many things I cant do in windows that i could do easily in ubuntu,

And it is wonderful knowing that I no longer feel tied down to windows...
