### Post by Austin_KW on 2008-01-14
If I understand you, you want to use a windows printer that has no linux driver.

You can do this by creating a virtual postscript printer on the windows box and then printing to it using any postscript driver from ubuntu.

Hav a read of this howto [http://www.stat.tamu.edu/~henrik/GSPSprinter/GSPSprinter.html](http://www.stat.tamu.edu/~henrik/GSPSprinter/GSPSprinter.html)
