### Post by RaZer0r on 2007-09-04
you can run the program from the alt + f2 menu
Just press alt + F2, put gmail-notify in there and it will run.

To add it to your autostartup items, go to:
System, Preferences, Sessions.
Add a new shortcut, name: Gmail Notifier
command: gmail-notify

Hope that helps :)
