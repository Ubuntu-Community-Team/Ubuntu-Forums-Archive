### Post by Dr. Nick on 2006-10-15
Their is a folder called /tmp that holds some temp internet and system data. It should be automatically deleted on reboot though.

Other web browsers all have their way of doing things, epiphany which is the default in gnome also has ways to delete history and cache if thats what is used from now on.

I dont think thier are any logs that store private data, just be aware of the "recent documents" in the panel that stores some stuff across reboots.
