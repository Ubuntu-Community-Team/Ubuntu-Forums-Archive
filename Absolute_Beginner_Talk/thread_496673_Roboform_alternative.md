I searched for the same thing and settled on using KeepassX. I believe it's available through Synaptic Package Manager if you have "universe" enabled.
It's not as functional as Roboform...  but what it does it does well. 
(it takes 2 steps to launch a url, and enter username/password)
I also installed Keepass on a USB for the Windows side of things. The same database file works for both.
