### Post by steve.horsley on 2007-06-22
I can't help with the modem. Let's hope someone else can.

If you get no help at all, I would suggest getting an external router with a built in ethernet switch (maybe wireless too). This will then support any old OS, playstation, print server etc. It costs money, but it's less hassle for now and for a long time to come. Also serves as a hub for a home network. Choosing a router+4-port switch instead of a modem was one of the best decisions I ever made, except maybe getting married.
