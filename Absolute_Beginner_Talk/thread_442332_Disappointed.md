### Post by Acksys on 2007-05-13
Did you try booting with the VGA option (you press one of the function keys when the disc starts up, can't remember which one, but it says at the bottom of the screen)?

This will hopefully get you into a graphical interface with a vanilla VGA driver, where you can install the system and troubleshoot further. Search the forum and you'll find lots of suggestions for solving video card problems.

Video problems can be tricky to solve, but you'll succeed if you persist. It can take some tenacity to get a Linux system working perfectly!
