### Post by K.Mandla on 2006-07-22
Judging by the specs I found [here]("http://germain.umemat.maine.edu/faculty/hiebeler/480cdt.html#specifications"), it's going to be tight. Memory is the biggest problem (I'm assuming you haven't added any). I keep a slower P3 with 512Mb in it, and with a bare-minimum installation, it uses about 58Mb just to boot. If you can add more, you'd be doing yourself a huge favor.

Aside from that, you should be okay. A 4Gb hard drive is more than enough for Xubuntu. If you can get your hands on an out-of-the-box wireless card (I use a Linksys WPC11 -- no tweaking needed, but only 11b speeds), you'll be set. Give yourself plenty of time to install it, though. ;)

If you get it on there and it seems sluggish, consider an installation for [low memory systems]("https://help.ubuntu.com/community/Installation/LowMemorySystems"). In that case, you're starting with the bare essentials and building on to it, rather than dumping an entire suite of applications on at once. You only add what you want and need. :)

Good luck!
