### Post by jm2003uk on 2008-03-30
If you installed using the server/minimal CD then there will be no GUI by default. If you want to just upgrade your install to the default Ubuntu system, then you can use:

sudo apt-get install ubuntu-desktop

I guess if you just wanted to GUI then you could just install gnome, but i'm not sure on what other dependencies it will need:

sudo apt-get install gnome-desktop-environment


Or download the normal livecd.


Hope this helps.
