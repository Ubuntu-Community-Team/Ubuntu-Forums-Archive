### Post by Kobalt on 2007-05-28
> Section "Device"
Identifier "VMware Inc [VMware SVGA II] PCI Display Adapter"
Driver "vmware"
I'm not an expert about the VMware, but I don't think, regarding how it loads its configuration, that you can install such drivers. 
If you wanted to make it happen you could try.
First you'd make a backup of the existing xorg.conf : ```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```Then you would need to change the identifier of your card to its name (for ex. ""nVidia Corporation GeForce Go 7200") and set the Driver "nvidia". Then you restart X with Ctrl+Alt+Backspace.
