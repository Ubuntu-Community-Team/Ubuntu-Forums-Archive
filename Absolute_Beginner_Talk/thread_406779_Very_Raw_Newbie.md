### Post by locke.dragon on 2007-04-11
i don't know about importing firefox data, but here's how you get files from a windows partition...

[http://technical-itch.co.uk/2006/11/06/how-to-access-your-windows-hard-drive-from-ubuntu/](http://technical-itch.co.uk/2006/11/06/how-to-access-your-windows-hard-drive-from-ubuntu/)

i was in a similar situation and feel great now that i've ditched my last operating system.  i hope you make the choice to stay with ubuntu and more power to ya!

as for the dual-boot, when you run the installer, it'll give you the option to make it a dual boot system.  just use gnome partition manager to shrink the windows section and then choose to install to the largest contiguous free space.  you're all set!

and the keyboard problem should be fixed once you install to your drive.
