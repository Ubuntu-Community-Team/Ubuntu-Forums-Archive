Hello new and old linux users alike 

I'm writing this little guide out of experience and results may vary but  will explain the steps I took to get my wireless to work.
first you'll need a few program so fire up a teriminal and copy paste these commands if you don't already have them:

sudo apt-get install ndiswrapper ndisgtk kwifimanager 

now type:

sudo ndisgtk

Now you will need the apropriate widozes driver for the hardware to work
locate the driver with the .INF extension and ndswrapper will install the it.


The next thing I did was get a good informative wireless manager so I installed kwifimanager

I was able to see the networks and I was able to connect. That simple 

If anyone has any comments or additional info that they would like to add please do so.
Some time ago it was a pain to get wireless to work but now it all just worked for me.

good luck and power to the people or, should I say wireless.
