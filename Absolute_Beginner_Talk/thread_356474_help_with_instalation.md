### Post by benuski on 2007-02-08
You probably don't have your BIOS set to boot from CDs before it boots from the hard drive.  When your computer first turns on, before Windows starts to boot, look up in the upper right hand corner of the screen.  There should be an option, usually by hitting F12 but sometimes F10, to load up a one time boot menu.  Go there, and click to boot from the CD Drive.  If you want to change this permanently, as I have, you'll have to hit F1, go into the BIOS setup, and I don't remember the exact steps from there.
