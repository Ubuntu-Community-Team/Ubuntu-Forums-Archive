### Post by carlosqueso on 2006-01-20
a symlink is like a shortcut in windows, except it works in the terminal as well.  For example, I have a symlink in my home directory to the documents on my windows hard drive, so I just have to go to the link instead of going through all of those directories.
 
You make one with ```
ln -s <file/directory to link TO> <name of link>
```

They're pretty cool...enjoy your symlinking.
