### Post by asmoore82 on 2007-08-09

there should be a startup program called "Network Manager" and it should be enabled.

IF it's not there ...
click "New"
name it "Network Manager"
command is 'nm-applet --sm-disable'
you will probably have to logout/in to activate that change
