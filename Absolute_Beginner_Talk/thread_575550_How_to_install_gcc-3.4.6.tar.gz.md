### Post by Vixis on 2007-10-14
You can search deb files at [packages.ubuntu.com]("http://packages.ubuntu.com/")
or in Synaptic choose to install what you need, then File -> Generate package download script
it will give you a script for download with all dependencies. Script can run on *nix systems or you can see what packages you need and download manually. 


When you got deb files you may double click on them to install or in Synaptic go File -> Add downloaded packages.
