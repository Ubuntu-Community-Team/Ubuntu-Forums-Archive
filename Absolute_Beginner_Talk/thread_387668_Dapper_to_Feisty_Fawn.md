### Post by dhughes on 2007-03-18
If you are getting rid of your dual boot setup you should use the new extra space and make a separate /home partition so it's safe, when you upgrade you won't lose anything if you don't touch /home (just make sure you **don't** format /home). 

 I use hda1 for / and hda2 for /home (plus hda4/hda5 for Swap).
