### Post by PeterJS on 2008-03-12
Yep ubuntu will be fine for three reasons:
1) it asks permission before attempting to autorun anything, and you can just tell it not to.
2) By default files in ubuntu aren't marked as having executable permission, so you have to explicitly give them executable permission before they will.
3) Barring all of the above, windows and ubuntu don't use the same executable format, so with out additional software installed (wine) windows programs won't even run on ubuntu.
