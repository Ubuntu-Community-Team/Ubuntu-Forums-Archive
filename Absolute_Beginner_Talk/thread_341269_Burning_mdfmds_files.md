### Post by rai4shu2 on 2007-01-18
That's an Alcohol 120% type file. As such, you either need that special CD burner program or use a converter:

[http://mdf2iso.berlios.de/](http://mdf2iso.berlios.de/)

The converter, however, will rip out everything but the bare ISO data.
