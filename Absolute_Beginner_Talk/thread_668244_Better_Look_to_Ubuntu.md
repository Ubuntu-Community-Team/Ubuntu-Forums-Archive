### Post by Yoke &amp; Chung on 2008-01-15
If you want the gadgets like Vista, you can do the followings:

sudo apt-get install gDesklets

Goto System--> Preferences--> Session, click "Add"
Key in "gdesklets" in command and name with the quotes. Reboot.

You will find gDesklets under Applications--> Accessories

Do whatever you want :)
