### Post by tizza10 on 2008-01-29
Generally (unlike windows) you do not have to find each driver with a linux distribution, the kernel and the installed packages usually provide superior hardware support. There are a few exceptions...mostly ATI and NVidia graphics cards and a few wireless NIC's.
Welcome to the world of Linux.
For AIM, use Pidgin its great. Or use synaptic or another package manager to install something else if you prefer.
