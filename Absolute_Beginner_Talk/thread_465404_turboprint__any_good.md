### Post by zero-9376 on 2007-06-05
I have been using turboprint with my MP510 for a couple of months now and it works very well for most needs. I dont do photo printing so I set turboprint quality to medium (or low- cant recall right now) which gets rid of the banner. If you set the quality back to the higher setting you are warned that the banner will be printed.
