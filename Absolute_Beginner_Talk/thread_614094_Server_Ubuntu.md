### Post by Lord_Dicranius on 2007-11-15
If you're trying to install Ubuntu for everyday use on say a laptop or a desktop machine, I'd stick with the desktop version.  You can find more info on the server version [here](http://www.ubuntu.com/products/WhatIsUbuntu/serveredition).  The biggest differences I've found: doesn't install a dekstop environment by default, and it allows for easy setup of webserver, ftp, etc etc (it prompts during installation asking what kind of server, then installs all the packages needed for it to work).
