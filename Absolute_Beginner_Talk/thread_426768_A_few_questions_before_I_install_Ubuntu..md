### Post by [S|G] on 2007-04-28
1 - No, they won't get fried
2 - Yes, NVidia has released drivers for your video cards
3 - No. Linux doesn't use the same codecs that windows does. You'll probably want to install w32codecs to get some movies to work.
4 - What is your motherboard? Probably you won't have to touch any drivers CD to get things working, as chances are that your hardware will be automatically dected and installed.

You should try running Ubuntu from the live cd to see how it'll be like beforehand :)
