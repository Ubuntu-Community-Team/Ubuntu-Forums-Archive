### Post by SirGrant on 2007-07-31
Here is what you want to do.  To export your bookmarks open up firefox and go to the bookmarks tab.  Then select organize bookmarks and a box should pop up.  Then click on file --> export.  Then put that file that you created on your floppy and you can go to any computer and do the same process (except click import instead of export) and bam you have your bookmarks!  Good luck!  This is easier because it just backs up all your bookmarks into one easy to carry single file instead of having to digg it out of the hidden folders.
