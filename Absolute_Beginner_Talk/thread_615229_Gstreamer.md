### Post by frank002 on 2007-11-16
For MP3 you install the Fluendo codec. Make sure to have the Universe repository activated. Open Synaptic and click Search. Type gstreamer0.10-fluendo-mp3. Mark it for installation and click Apply. You now should be able to play mp3s in Rythmbox and Totem. Good Luck.

### Post by AndyCooll on 2007-11-16
Using the search box in Synaptic and typing gstreamer will bring up all the gstreamer codecs. You can install as many as you want, however the fluendo one is the one specifically used by Rhythmbox to handle mp3's.
