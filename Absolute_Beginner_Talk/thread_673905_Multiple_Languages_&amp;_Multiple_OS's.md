### Post by housam on 2008-01-21
If you can connect the Internet through the live cd , do sys>> admin. >> language support .it'll download / install the language package.

If not , boot normally in edgy and connect to the Internet and then do sys. >> admin. >> software sources . this will download the univers and the multivers packages. then do sys>> admin. >> language support . this will download the any languages you want. tick the square in front of ( i.e French ) to start installing it.

after that go for the upper panel and right click >> add to panel >> utilities >> keyboard indicator.>> add.

after that you will find a word 'USA' appears on the panel. right click on it >> keyboard preference >> layouts >> add >> ( i.e French.)
this will ease the switch between languages during writing.

Good luck
