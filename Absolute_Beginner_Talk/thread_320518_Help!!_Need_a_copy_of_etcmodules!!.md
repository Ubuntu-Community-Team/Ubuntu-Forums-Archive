### Post by aidanr on 2006-12-17
> # /etc/modules: kernel modules to load at boot time.
#
# This file contains the names of kernel modules that should be loaded
# at boot time, one per line. Lines beginning with "#" are ignored.

lp



# Generated by sensors-detect on Wed Oct 25 07:08:02 2006
# I2C adapter drivers
i2c-i801
# modprobe unknown adapter NVIDIA i2c adapter 0 at 1:00.0
# modprobe unknown adapter NVIDIA i2c adapter 1 at 1:00.0
# modprobe unknown adapter NVIDIA i2c adapter 2 at 1:00.0
i2c-isa
# I2C chip drivers
eeprom
lm90
w83627hf

leave out everything below "lp" though

also if you deleted part of the file in gedit, check for the file /etc/modules~ it would be a hidden file, a backup
