Edit /etc/apt/sources.list

```
gksudo gedit /etc/apt/sources.list
```
and place a # in front of the first line to comment out the CD repo so it won't ask you to insert the CD from now on.  

Then, remove the # in front of all the lines that start with **deb** and **deb-src**.  

Save it and run

```
sudo apt-get update
sudo apt-get install sun-java6-bin sun-java6-jre sun-java6-plugin sun-java6-fonts
java -version
```
