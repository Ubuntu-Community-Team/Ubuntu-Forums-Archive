### Post by RequinB4 on 2008-03-03
I assume you mean on the dropdown toolbar?  go to that directory with nautilus (the directory browser) and go to bookmarks --> add bookmark.  You can edit/delete it from the menu at any time via bookmarks --> edit bookmarks.

If it is on a seperate hard drive, the hard drive will need to be mounted before you can view it with nautilus.
