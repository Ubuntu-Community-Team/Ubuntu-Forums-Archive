### Post by TheTux on 2006-11-17

For Ubuntu menu, I don't know how to the icons smaller, but if you don't want it, you can remove it. Do this System > Preferences > Menus & Toolbars. Untick show icons in menus. Anyway the icons in windows are smaller but the menu is not as organized as in Ubuntu. Just my personal opinion though.
