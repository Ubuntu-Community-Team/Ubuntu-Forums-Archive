### Post by cliff0108 on 2006-04-20
Does gFTP have a 'resume download' function in the event a transfer is broken ? I can't seem to find it.

---

### Post by izmaelis on 2006-04-20
Yes it has.
1. connect to any ftp server
2. go to the file that you want to download and select it by clicking mouse over it
3. hit a button with an arrow witch shows to the left and start a download process
4. now exit gFTP (or cancel the download in progress)
5. try repeating steps 1-3 and you will see a dialog window asking you to overwrite, resume etc.
