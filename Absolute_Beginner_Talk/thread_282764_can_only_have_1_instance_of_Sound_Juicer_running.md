### Post by xyz on 2006-10-23
I like
```
sudo aptitude install goobox
```
> CD player and ripper for GNOME
Goobox is an CD player and ripper for GNOME 2 environment.
It follows the "Just Works" principle so its interface is
beautiful and easy-to-use.

It uses GNOME/GTK+ for its user interface, GStreamer framework
for CD playing and ripping operations and GnomeVFS to search
album cover images with Google.


[http://ftp.gnome.org/pub/GNOME/sources/goobox/](http://ftp.gnome.org/pub/GNOME/sources/goobox/)

There are many others...it's up to you.
