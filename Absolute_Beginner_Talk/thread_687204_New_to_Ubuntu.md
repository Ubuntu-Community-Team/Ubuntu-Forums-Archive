### Post by jordanmthomas on 2008-02-04
[http://infrarecorder.sourceforge.net/?page_id=5](http://infrarecorder.sourceforge.net/?page_id=5)
This is a free (and open source) cd burning program for Windows that I've successfully used to burn CDs.

Be sure to use its "burn iso image" option.

And yes, an iso file is similar to a zip file, but DO NOT EXTRACT IT.  After you burn the CD, all the files will be on there.  If you burn the CD and there is only one file on it, you have done it wrong.

edit:  see this if you get confused.
[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

Good luck, and don't hesitate to post anything else if you need more help.
