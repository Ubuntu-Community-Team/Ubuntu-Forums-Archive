### Post by Kyral on 2005-08-20
1) The CDROM drives should be mountable by user. Same with floppy. But if you need to use root (for anything really), preface the command with sudo.

2) They should show up as they are mounted

3) Ubuntu doesn't use the root account, it uses sudo for gaining root power for the single command. The "root password" is your user password.
