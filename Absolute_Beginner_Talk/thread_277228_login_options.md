### Post by CatKiller on 2006-10-14
What you're after (I think) is a GDM Theme with a Face Browser. You can change the theme that's used with System -> Administration -> Login Window. I can't remember what themes are included in there by default, but you can find more from places like art.gnome.org and gnome-look.org
