### Post by jimrz on 2007-12-19
Netgear WG 511T (the "T" is critical) which uses the Atheros 5212 chipset, works "out of the box" with ubuntu and gives pretty much the same performance that you get under win.

 If you use WPA encryption, you will need to upgrade the firmware as the one that comes with the retail ver only supports WEP. This ,of course applies to using it under win, as well. This should not present any problem, as installing the updated firmware (available ford/l on the Netgear site) is pretty simple under either OS.

I have been using one of these on my old laptop for several years (been running under ubuntu since breezy) and, also recently installed ubuntu on an old Dell laptop for a friend and got her one to use with it. Both have given flawless performance.
