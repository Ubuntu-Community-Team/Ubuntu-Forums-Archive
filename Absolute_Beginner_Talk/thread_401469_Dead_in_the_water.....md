### Post by newlinux on 2007-04-05
> **roho76 said:**
> Thanks for the reply Mr. Wiley. Ha!

Anyway. Isn't Myth TV what's running in the Linux MCE?

Baby steps to the Linux.

LinuxMCE is basicallly a combination of MythTV and Pluto. It is very early in its development and if you are new to Linux (heck even if you aren't) I would recommend you stay away from it at this point and stick with getting to know Linux and installing MythTV. Any issues you run into you can just ask us on the forum...

I strongly recommend you run one of the LiveCDs for a little while before installing Linux. It will be a bit slower than it will be when it's installed, but it gives you a chance to get used to it and to make sure everything works with your hardware. If you can't wait to install it I strongly also recommend that you don't completely erase Windows -- run it as a dual boot Windows and Linux machine. You already paid for MCE... Unless you are starved for hard drive space there is no harm in keeping both available.

If you are really very intereted in MythTV I recommend you wait until Feisty Fawn (Ubuntu 7.04) will be released - which will be later this month. A lot of work has been put into making Myth install easier with Fiesty, and early reports are very good.

Also - which live version did you have problems with? There is probably an xorg issue if you got sound but just a black screen. Your video card should be well supported so it is probably a matter of configuration.
