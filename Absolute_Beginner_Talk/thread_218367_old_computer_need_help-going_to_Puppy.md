### Post by Eddie Wilson on 2006-07-19
Hi Akash799,

Sorry that you can't get Xubuntu to run on your system. The largest problems I've seen with installing on older systems is the memory and the hard drive space. I've have also tried to installed several flavors of Ubuntu on older systems. When I run across an older system that *ubuntu will not install on I install the Puppy distro. You can install with 64 megs of ram and that's hard with a lot of distros. Don't get me wrong. I really like Ubuntu and will install it when possible, but I will install any distro I can so people will learn and use Linux. Puppy  2.01 is a very nice distro. It takes a little getting use to. You can add packages, you can update, you should be able to do anything you need to on an older system. The Puppy website is a good place for you to start. Good luck with your linux adventure.

Eddie

### Post by 3rdalbum on 2006-07-19
Puppy is a great little distro. Graphically, it's not very flashy... at all... but it has package management, Live CD persistance and a great selection of apps. There are a number of people who build DotPup packages of things which aren't in the DotPup or PupGet repositories. And most importantly, it runs fast!

I've never quite understood why people prefer DSL over Puppy...

This is the first time that Xubuntu has been on a CD, so I'm not surprised if it has a few show-stopper bugs.
