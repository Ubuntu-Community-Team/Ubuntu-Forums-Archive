### Post by K.Mandla on 2006-11-12
I think perhaps you may need to adjust the settings in your Thinkpad's BIOS to boot from the CD.

If you're seeing a message that says, "Boot from this CD to try Ubuntu without affecting your system," you've started Windows with the CD in the drive, and Windows is showing you that message when it spins up the drive.

You're on the right track, though. Try, try again! :D
