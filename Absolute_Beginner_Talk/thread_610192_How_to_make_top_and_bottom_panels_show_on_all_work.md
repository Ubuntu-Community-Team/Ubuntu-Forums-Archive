### Post by Jake90 on 2008-01-20
> **bettyhills said:**
> What is "cube".  How did you set it up?

Go to System>Preferences>Appearance then click on the last tab (Visual effects). Go to the bottom where it says custom and select it. Next to it you should see a button for preferences. Click on it then you should see cube as on option. Then it should work
