### Post by flossgeek on 2007-06-02
Try going to "**System>Administration>User and Groups**". Select the user name you want sound for and click the "**properties**" button. Then in the properties dialog box click the "**User Privileges**" tab and ensure "**Use audio devices**" is checked.
