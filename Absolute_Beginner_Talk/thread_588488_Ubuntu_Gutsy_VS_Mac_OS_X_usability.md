### Post by bschopman on 2007-10-23
I have been using Mac OS X for two years now after using WIndows since msdos & win3.11. Using WindowsXP felt like a leap back into time, since I had been using MacOSX for two weeks. However, unlike the so-called progression of the Windows platform, Ubuntu is imo extending the Linux platform with reasonable usability.

Operating Systems have inspired each other since the beginning of time... there is absolutely no shame in "stealing" features. We can flame each other with endless lists of who did what first, but I think that wouldn't help anyone.

Instead, I would like to discuss simple/fundamental features that could be implemented in Ubuntu. 


First of all some PRO's of Ubuntu Gutsby:
- Desktop effects (compix?) are not too much and give a slick feel to the GUI.
- the installation is super easy
- os-runs-from-installation-cd is mandatory for modern OS, ergo great!


My inspiration for the topic start was primarily the alt-tab functionality. I use it a lot on every computer I use. In all OS's except macosx one can only hold alt and press tab again to go the right in the list. I love that in mac os x after pressing alt-tab one can hold alt and press the arrows left and right to go to the left & right in the list. 

Furthermore, as I use the keyboard as primary input device, I would like to have something like [Quicksilver]("http://quicksilver.blacktree.com/"). It's a productivity tool, but I use it primarily as an application launcher and sporadicly as *control-panel launcher*. Right now, I've set some keyboard shortcuts to launching the web browser, terminal & file browser, but if I were to be using Ubuntu seriously, I would definitely need some better app-launching functionality then icons or menus in the top panel.

### Post by radioaktivstorm on 2008-03-08
Hello bschopman,

I just encountered your message on the forum and thought about what you've said. As a Ubuntu/Mac OSX user, I can see where you are coming from. However, I think that these features already exist in ubuntu and hopefully i have some solutions for you.

1)Moving to the left in the window list: with compiz fusion off, by default after you click alt tab, to open the window list, shift alt tab can be used to iterate to the left through the windows. In compiz fusion, this may be inactive, but you can turn it on in the compiz fusion configuration application. 

2) a quicksilver-like tool for launching applications: alt + f2  allows you to call programs by name and run commands in the terminal as well. this feature is enabled on Xubuntu, Kubuntu and Ubuntu. if you want something that is more quicksilver with respect to interface, katapult (a kde application) may be appropriate; you can obtain it from the repositories with a simple apt-get install katapult from the terminal. it will run on GNOME, XFCE, and KDE, but you might find that you will have to move around some of the shortcuts in gnome, as it may collide with your existing ones.

Also, as a hint, the alias for the control panel for ubuntu proper is gnome-control-center.

Hopefully this will help you in your quest to be more productive with ubuntu.
Cheers!
~radioaktivstorm

It's not in the Ubuntu repositories - you have to add Google's own repositories. Full instructions are [here]("http://www.google.com/linuxrepositories/").
