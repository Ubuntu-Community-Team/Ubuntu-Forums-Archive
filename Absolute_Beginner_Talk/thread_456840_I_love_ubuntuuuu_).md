### Post by Kobalt on 2007-05-28
The first thing you should do, to send/receive files with mobiles or such, install the *gnome-bluetooth* package (with Synaptic package manager). Once you did this, you can send a file with a right click on it, select Send To > Via bluetooth.
The bluetooth mouse will be a different set up, depending on which mouse you pick.
