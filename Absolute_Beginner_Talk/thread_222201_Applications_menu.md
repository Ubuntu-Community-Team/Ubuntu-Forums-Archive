### Post by aysiu on 2006-07-24
If it's in there but just needs to be refreshed...
Alt-F2 ```
killall gnome-panel
```

If it's not there at all, right-click the **Applications** button and **Edit Menus**. Then go to **File** and select **New Entry**. The command is usually the name of the program (in all lowercase letters).
