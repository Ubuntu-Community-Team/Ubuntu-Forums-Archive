### Post by aysiu on 2005-10-29
Follow the instructions here:

[http://www.psychocats.net/linux/sources.php](http://www.psychocats.net/linux/sources.php)

Then, ```
sudo apt-get update
sudo apt-get install wine
``` Both sets of instructions assume you know what a terminal is. If you don't, click on Programs > Accessories > Terminal. That's where you type the commands in.
