### Post by nalmeth on 2007-01-31
> **unklejman said:**

Wine will not help you with these things. Its good, but not that good. 
i.e. you can run Photoshop CS with wine by installing it on windows, then copying all the files directly into the ~/.wine/drive_c/Program\ Files/ folder on your Ubuntu install. Flash is supposed to work on wine.
It won't however run 3D max, no idea about after effects, or director.

You can research how well any given app will work from here: [http://appdb.winehq.org/](http://appdb.winehq.org/)
Doesn't look like your game works too well: [http://appdb.winehq.org/appview.php?iVersionId=6422&iTestingId=7674](http://appdb.winehq.org/appview.php?iVersionId=6422&iTestingId=7674)
You would need Cedega to run the game, or dual-boot (wintendo)

If you're serious about going linux (but not about using different programs), Crossover Office is what you'll need. Check with them if your apps are supported. It's certainly cheaper than going Mac. It's not free, but thats the overhead price of closed software.

What do you use for music, and what type of music do you do? There is a lot of studio software for linux, but if you insist on proprietary apps like Protools/Cubase don't bother. However, if you're open-minded, you'll find theres a lot to be found. :-)
