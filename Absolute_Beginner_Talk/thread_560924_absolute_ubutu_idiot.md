### Post by bobplano on 2007-09-27
for some ubuntu tutorials you might want to use [http://psychocats.net/ubuntu/index.php](http://psychocats.net/ubuntu/index.php). 

to install ubuntu you first need to burn an image cd. i don't know if nero can do this but just make sure its burned as a image at no more than 4X. you need to then go into the BIOS of the other computer (press F2 or something right when it starts) and change the boot order so the cd boots first.

if you don't really get my instructions or something look at [this site]("http://psychocats.net/ubuntu/installing") and if you're not using windows on the other computer, then don't worry about the stuff dealing with windows, like defragmenting

this is a really good partition guide from the psychocats site so read this over before you install: [http://psychocats.net/ubuntu/partitioning](http://psychocats.net/ubuntu/partitioning)

### Post by eph1973 on 2007-09-27
Nero can burn an image.  Pretty much any CD burner you use in XP should be able to burn an image.  Remember: an image is NOT a data file.  You want to burn an image file.  It may be also called a .ISO file on your CD burning software.  Either way, it's the same thing.  You might want to click on this [how to burn an image file]("http://www.petri.co.il/how_to_write_iso_files_to_cd.htm") link.  Also, once you have the image file on your CD, everything you should need to install Ubuntu including the formatting and partitioning software will be on the disk.  Just follow the instructions of the sites that bobplano refered you to.  Hope this helps.

### Post by nvteighen on 2007-09-27
I actually recommend you to learn about Linux in general first and then, about Ubuntu. There's a great tutorial about Linux basics at [http://www.linux.org/lessons/beginner/index.html](http://www.linux.org/lessons/beginner/index.html) It will show you command line basics, but more important than that is that shows you the system structure, teaches you the concepts ("mount", "root", etc.)... and it's really funny to read. It also teaches you to install Debian, if you're interested on Ubuntu's mother-distro.

After having clear the concepts and basic commands (you'll have to use them from time to time), you'll easily understand the links the people gave you before and will learn Ubuntu specific things or problems to be cautious, etc.

EDIT: And feel free to ask questions here!
