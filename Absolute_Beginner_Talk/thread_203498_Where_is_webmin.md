### Post by Randomskk on 2006-06-25
Yea, the webmin script was removed from the repos for security reasons / the maintainer wanted it gone, so if you want it you need to get it from the webmin site. Make sure to keep up to date with new releases and such, you should probably subscribe to the release mailing lists.

The alternative is to learn how to configure your system without it; but depending on your needs it may make life a lot easier.
