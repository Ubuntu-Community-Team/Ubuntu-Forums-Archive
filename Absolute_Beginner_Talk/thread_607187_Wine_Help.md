### Post by TeaSwigger on 2007-11-08
Well some sage advice you probably don't want: WINE and especially compiling stuff from source isn't exactly something a person should do yet until they've used ubuntu / linux a while. They get complicated fast if you're not incredibly lucky.

That said; I suggest you use a .deb of WINE from the ubuntu packages site:
[http://packages.ubuntu.com/](http://packages.ubuntu.com/)
and install that rather than compile wine from source. They've done the work for you.

Second, for games you may have to use Windows in a virtual box or dual boot, as it can be hard to impossible to run many games in WINE. Short of Microsoft cooperating with WINE and/or linux users (yeah right). To run windows in virtualization the computer needs more than 512mb ram. I'm not saying you have to give up, just that it's going to take some doing.
