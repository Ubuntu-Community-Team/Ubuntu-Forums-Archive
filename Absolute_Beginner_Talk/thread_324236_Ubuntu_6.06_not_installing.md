### Post by az on 2006-12-23
> **gustasonfrever said:**
> I have a Pentium III 753mhz Dell with 128 mb of RAM.

You need more ram.

The minimum recomended amount is 256, but you can run the live cd without installing with only 196 megs of ram.  If you already have a linux distribution installed on your computer and have a swap partition set up, the live cd will use that, so you can get by with even less, but it will be slow.
