### Post by kripkenstein on 2006-07-18
All programs installed from the repositories will notify you when they can be updated (unless you disable the notifications). Basically, when any package you have installed is updated in the repositories, you will be notified. 

If all you do is install stuff from the repos (and nowhere else), then keeping up-to-date is very easy, just click to update when you are told.
