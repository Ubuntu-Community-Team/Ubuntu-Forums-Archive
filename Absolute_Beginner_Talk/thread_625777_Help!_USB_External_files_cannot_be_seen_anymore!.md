### Post by mtalexan on 2007-11-29
Linux and Windows generally store data in two very different formats and Windows really doesn't like to play nice with the Linux filesystems directly.  Can you still see your files when you hook the drive up to your Windows machine?  If not, I'm guessing Windows did something like automatically beginning to format your drive to something it prefered and that's why it was taking so long for the copy.
