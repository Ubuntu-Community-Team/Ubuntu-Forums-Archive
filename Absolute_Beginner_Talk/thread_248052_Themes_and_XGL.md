### Post by Kobalt on 2006-08-31
Hello,

If you install XGL, you will probably want to install Compiz as well to get the visual effects. When you'll install all this, you will install a package called cgwd themer, that will handle the themes. If you had any gnome gtk2 themes, they won't be working anymore when you install XGL+Compiz. To get any cgwd themes, I suggest you surf around gnome-look and the compiz forums. 

Cheerios !
