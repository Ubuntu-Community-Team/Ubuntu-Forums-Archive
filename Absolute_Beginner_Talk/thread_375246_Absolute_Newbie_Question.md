### Post by Ek0nomik on 2007-03-03
You can dual boot Windows and Ubuntu.  I was able to do it rather easily.  Just make sure you install Windows first, because if you install Windows second, it will want to take over your MBR (Master Boot Record).

You can run Windows programs in Ubuntu.  You can use a few different programs to do it.  Wine, VMWare are the two big ones that people name.  Wine seems to do smaller applications, while VMWare can do larger ones, including running Windows itself.

Hope that helps!
