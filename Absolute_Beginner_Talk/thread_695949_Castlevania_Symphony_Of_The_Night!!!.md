### Post by bump_ on 2008-02-13
I'm not sure if this is the right forum for that question, but I'll try to help. If it is a multi-part .rar file, just put them all in the same directory and extract the first part. The extraction will put all the parts together for you and the resulting folder should hold an image of some sort. In epsxe, go to File -> Run ISO. The go to the folder that was just created by the extraction and click the image file, .iso, .img, .bin, or whatever it is.
