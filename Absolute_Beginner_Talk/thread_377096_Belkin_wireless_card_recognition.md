### Post by AndyCooll on 2007-03-05
The problem with this card is that there is more than one chipset version. I use this PCMCIA card. 
The older version works "out-of-the-box". However the newer version uses the RT61 chipset. There's a good HOWTO here: [ HOWTO: RT61 on Egdy Eft with WPA]("http://www.ubuntuforums.org/showthread.php?t=296822")
