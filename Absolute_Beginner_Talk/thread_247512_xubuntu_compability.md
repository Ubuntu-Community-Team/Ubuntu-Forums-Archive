### Post by LadyDoor on 2006-08-30
The first place I'd suggest you look at the Ubuntu wireless documentation ([https://help.ubuntu.com/community/WifiDocs)](https://help.ubuntu.com/community/WifiDocs)). If it doesn't look like it's natively supported (all the *buntu flavors should support the same wireless hardware), you may be able to use NDISWrapper ([http://ndiswrapper.sourceforge.net/)](http://ndiswrapper.sourceforge.net/)), a program that allows the use of many wireless cards with non-free drivers. You can get ndiswrapper from the Ubuntu repositories; just make sure you keep a copy of your driver if you need to use it.

Good luck!
--Door
