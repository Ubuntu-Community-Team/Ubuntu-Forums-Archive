### Post by adam.tropics on 2006-07-13
Should you ever decide to give Xgl/Compiz a go, the 'switcher' plugin, has a temporarily_unminimize option, which gives some this capability to <alt><tab>. I say some of, as it will show you minimised windows (and then if not selected return to minimise), but it will not show stuff that minimises to the notification area, such as gaim, amarok etc
