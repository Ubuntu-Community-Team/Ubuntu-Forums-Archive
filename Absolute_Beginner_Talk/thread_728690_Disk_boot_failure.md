### Post by chewearn on 2008-03-19
1. Check the harddisks jumpers are properly set to master and slave.
2. Check your BIOS harddisks detection, verify the correct harddisks are detected.
3. Check your BIOS boot order.  Make sure the correct harddisk is selected as the first boot device.
