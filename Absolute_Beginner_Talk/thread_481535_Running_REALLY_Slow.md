You might want to download a program called "Conky" to monitor your system, it sounds like this is a resource issue, you might also want to go into terminal and type:
```
top
```
You can see what processes you have going. Then you probably want to install Conky or Gkrellm, it doesn't really matter:
```
sudo apt-get install conky
```
Your RAM seems to be pretty low too, that could be an issue as well.
