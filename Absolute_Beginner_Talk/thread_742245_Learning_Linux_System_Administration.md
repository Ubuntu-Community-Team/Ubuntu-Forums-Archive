### Post by Calash on 2008-04-01
Gentoo is quite a difficult first venture into Linux, but worth wile if you take your time and learn as you go.

A basic Ubuntu setup will give you a nice starting point, with the benefit of having a GUI to work from.  If you are really looking to learn networking, setting up a virtual environment is not a bad thing...and if you break it nothing is really broken.  I like Virtualbox for this type of setup, but VMWare is very good, and the server platform is also free.

For books, I cannot recommend any current ones but there are many that focus on CompTIA's Linux + certification.  That would be a good path to go, should not be that difficult to get and any CompTia cert is nice to have under your belt.
