### Post by Brunellus on 2005-11-24
1)  fglrx is a restricted module with a non-free license.  it is therefore not included by default.  You will need to install it yourself after installation.  I run an nvidia card, so I don't know what the procedure is, but if you search these forums or the RestrictedFormats page on the wiki, you will probably find an answer.

2) if your camera behaves like a regular USB mass storage device, then you will be able to see it in nautilus.

3) I have no experience with video editing/rendering, but I'm pretty sure you'll be able to see your camera plugged in via firewire.

4) No experience with that game.

Time is not a major investment.  Factor an hour for the install and maybe another hour for post-install tweaking (notably for the new video driver).  This assumes you have no oddball hadware that you are trying to make work.
