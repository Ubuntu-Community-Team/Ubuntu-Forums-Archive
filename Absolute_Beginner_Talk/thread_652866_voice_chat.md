### Post by Sef on 2007-12-29
Pidgin does not have voice chat.  [Kopete]("http://kopete.kde.org") does and so does [Gyachi Enhanced]("http://gyachi.sourceforge.net/"), a yahoo clone.  (Kopete is like Pidgin, except it is bundled with Kubuntu.  You can download to Ubuntu if you like, via Applications >Add/Remove)

Update: Also there is [OpenWengo]("http://www.openwengo.com/").
