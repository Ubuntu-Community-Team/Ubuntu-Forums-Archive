### Post by finer recliner on 2007-11-18
more importantly. if you ask for help on any forum, make sure you understand what you're doing before you go ahead and blindly type in a command that some anonymous user gave you to patch your problem. This is important not only so you dont get tricked by the few a$$holes out there, but so that you can learn yourself. Now you can undo whatever you did to your computer if it didn't work as expected. If it does work, you now have the knowledge to help other new users who have the same issue.

if you dont know what a command does, at least man it first:
```

man command

```
where 'command' is something you dont understand (i.e. rm, ls, cd, chmod...etc)
