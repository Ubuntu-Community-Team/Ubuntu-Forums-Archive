### Post by christhemonkey on 2006-01-24
Whilst the pc is doing its thing before windows, it should say something along the lines of, Press Escape to enter setup.  To enter your BIOS just follow its instruction.
To set it to boot from CD you will need to find the boot order option and set it to CD first.

### Post by Klaidas on 2006-01-24
When the system boot, try pressing the delete button. You should get into BIOS. There, set your computer to boot from a CD-ROM device. (1st boot device - CD-ROM). Then, insert you CD and reboot computer :)
If there are any questions - ask ;)
