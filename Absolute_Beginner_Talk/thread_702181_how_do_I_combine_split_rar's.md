You may need to install rar first.
open a terminal and type 
```
sudo aptitude install rar
```
Then click on the first , or unnumberd split rar and file roller should open up and allow you to extract. You dont have to actually rejoin the rar files themselves. The file roller should extract the contents as if it were one big rar.
