### Post by markyb86 on 2008-01-21
The live cd basically loads generic options, some might work and others might not, but once its installed it gives a more solid set of options that will pretty much all work for the hardware it is running on.

plus if you are using the restricted driver, it has a different set of options then the nv driver that loads on the cd
