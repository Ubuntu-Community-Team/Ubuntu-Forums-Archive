### Post by LHZ on 2005-11-04
Browse around for some screenshots, and look which one is prettier ;)
Seriously though, it all comes down to preference. Both are very robust, very customisable and stuffed full of applications. Some arguments often heard are:

GNOME
- more lightweight
- one application per function
- MacOS X-y

KDE
- more complete
- many applications per function
- Windows XP-y

As has been said, it's also easy to just install one of them and then install the other window manager. Then you can fiddle all you want.

If after all the remarks here you still don't know (which I could imagine), I recommend you get GNOME (so "regular Ubuntu"). Not because I'm on the GNOME side of the window manager wars, but because more people here run it, so it'll be easier to get help if something goes wrong.

*Edit*: You could also download one of them, then order the CDs for the other from the website (for free!). Then, by the time you'll have adjusted to Ubuntu a bit, CDs full of fresh options/challenges will arrive!

### Post by GreyFox503 on 2005-11-04
I came from a Windows background, but still liked GNOME better.  It looks a little different, but the menus are simpler to navigate and frankly just seem less cluttered.  To me, KDE menus seem like they just have too much "stuff", more lack of a better word.

A while ago they were giving away copies of Linspire for free ("freespire"), so I got in and installed it in a virtual machine.  The installation and configuration options are VERY minimalistic, and it ran slowly compared to other machines.  That can be good for newbies.

It *does* have you run as root by default, a bad idea.  It makes one little mention of creating a another user during the tutorial, but doesn't explain why that's a good idea or why running as root is bad.   (Linspire calls it "Administrator").

In fact, the first time I booted into it, it started playing a video tutorial (complete with a narrator) that explained how to use it.  It was very basic and did not assume you knew anything.  Good for people new to computers or linux in general.

You still have to pay to use their CNR service, which is their way of downloading and installing software (synaptic has more stuff and is free anyways...)

If you want to use linux but don't want to learn anything about it, I would try linspire.  However, you actually are interested in learning about unix/linux, you will quickly tire of Linspire as it tries to baby you around.

Those were my impressions, take them with a grain of salt.
