### Post by Patrick K on 2007-06-10
VMServer, from VMware, along with others, can run Windows in Ubuntu in a virtual machine (VMserver works in windows to run Linux, too). I hear it mostly works but no VM I know of support 3D graphics, so games are iffy. Since you are a beginner you might try Wine first. As 444 said many Windows programs refuse to work with wine. Others work fine (font size seems to be a problem, though). 

You must remember Linux, all flavors, is a totally different Operating System than Windows or Mac. Programs written for one OS will not work in another unless you use a software layer, like VMserver, to emulate the other OS. If you can find Linux SW that can fulfill your needs, that is the best way to go. Many Linux programs can read and write data from other OS files. For instance, Open Office can read and write MS Word, Excel, and PowerPoint files.
