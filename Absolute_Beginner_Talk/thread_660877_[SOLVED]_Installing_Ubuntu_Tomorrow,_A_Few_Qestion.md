### Post by jba6511 on 2008-01-07
as far as the dual booting goes, create 2 partitions, one for windows and one for ubuntu. Install windows first to one partition and then ubuntu to the other. GRUB will then recognize the windows install and give you the option of which OS you want to boot into when starting up. You can run virtual box, vmware, and wine without creating seperate partitions as indicated earlier.
