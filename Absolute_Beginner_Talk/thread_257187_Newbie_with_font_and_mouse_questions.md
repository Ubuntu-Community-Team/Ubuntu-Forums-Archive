### Post by Najand on 2006-09-14
Try to edit fonts:
System -> Preferences -> Font
Usually  size 10 to  13 is enough but it exactly depends on your screen resolution. If you have selected a too big resolution for your screen change it at:
System -> Preferences -> Screen Resoultion
