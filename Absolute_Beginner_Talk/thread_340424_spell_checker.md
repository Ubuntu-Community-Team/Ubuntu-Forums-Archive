> I don't know if you still check this post, but incase you do:

1. Right click the misspelled word.
2. Select "Check word spelling..." from the popup menu
3. Select the correct spelling from the list in the next popup. Make sure that the correct language is shown at the bottom of the popup window.

If the spell checker uses more than one language and this is a problem (some like it that way!) then go to Edit>Current Language... in the main menu and select the language for the current message.
