### Post by jem7v on 2007-04-03
DSL-N is like DSL, but with more stuff installed by default. It doesn't have OpenOffice, but you could probably install it because I think you can use .debs with it. On the other hand, it has Abiword installed by default and that Does open .doc files.

ChubbyPuppy has OpenOffice installed by default and is a variant of Puppy Linux, another good tiny distro.


Either would run on your system specs.  Neither should take more than ~100 MB space on your hard drive if you choose to install them.
