### Post by BarfBag on 2006-12-23
You don't have the win32 codecs installed.  They need to be installed before you can listen to .MP3's.  There's a bunch of ways to do this.  The easiest way is probably [Automatix]("http://www.getautomatix.com/").

If you don't want to use Automatix, there's a Debian Multimedia repository you can add that has everything you need.
