### Post by oeolycus on 2007-12-06
If I'm understanding you correctly, you are using the guided partitioning by shrinking the Windows partition. That means you're creating the NEW PARTITION, which is a percentage of the old one. If it's set to the "lowest" then I think you're saying the new partition is only a small percentage of the Windows one. This should not delete your Windows partition, but I would make sure of the following:
[LIST]
[*]You have backed up any critical documents on Windows (in case something *does* go wrong)
[*]Have you defragmented your HD from Windows yet?
[*]You have at least 10-15 gigs in the new partition for Ubuntu.
[/LIST]
I hope this makes sense.

### Post by oeolycus on 2007-12-06
Defragmenting
[INDENT]In XP, Start > Programs > Accessories > System Tools > Disk Defragmenter[/INDENT]

Backup
[INDENT]Photos, Music, Videos, Quicken(?), e-mails, video-game saves, etc. If you have an external HD, I'd just take the whole My Documents folder just in case (and from each user if there is more than one)[/INDENT]

I keep my XP installation pretty slipstreamed but I would want to make sure to have copies of all my music, videos, financial documents, pictures, my Photoshop Elements library file, Firefox user settings (cuz I've made some major tweaks), but that would be it for me.

I don't think you'll run into any hiccups if you're safe with the Xubu installation. I've never had any bad luck using a good copy of the LiveCD.
