### Post by _simon_ on 2006-11-04
If you have downloaded the Desktop CD ISO then Ubuntu will run from that.

Remember to burn it as a CD image, not as data.

All you do is make sure your CD or DVD drive is set as the first boot device, stick the CD in and wait for the prompt on screen to boot the CD.

Ubuntu will then boot without any further prompting.

If you choose to install Ubuntu then you can do so via the install option on the desktop.

I am not sure whether your USb pen will be detected by the Desktop CD or not, it would definitely be detected with a full install. Just give it a try.

Remember as it's running from CD it will be a lot slower than running from your hard drive.

Just to confirm, the Desktop CD will not install anything to your hard drive unless you go through the install process.
