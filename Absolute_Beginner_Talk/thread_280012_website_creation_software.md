### Post by coder_ on 2006-10-18
I would highly recommend you not go with Frontpage, Word, or any other "WYSIWYG" editors (Or using frames in your site), but rather learn [X]HTML/CSS and code it by hand so you can produce standards compliant sites that have a better chance at rendering correctly in all browsers.


**EDIT (Oops, too slow, this part was posted before me):**
But if you really don't want to learn how to code, then I'd suggest Nvu (I guess).  I think you need multiverse/universe to get Nvu though, so make sure you have that enabled.  Then just do a 'sudo apt-get install nvu' to get it.
