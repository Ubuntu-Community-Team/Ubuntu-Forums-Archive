### Post by rolheika on 2007-10-30
Hi, I am obviously quite new with Ubuntu, I was wondering, is there a way to configure Ubuntu so that I can toggle between typing in Russian, Arabic, and English?  (That is, switching between the Cyrillic, Arabic, and Latin alphabets?).  My install is in English and everything, I would just like to have the option to type in other non-Latin keyboard layouts if possible.

### Post by mikewhatever on 2007-10-31
First, go to System>Preferences>Keyboard, and add the required languages in the layout tab. Then in the next tab (layout options), expand Group shift/Lock Behavior and choose the keyboard shortcut to toggle languages. I think it is set to Alt+Alt by default.
You can also add Keyboard Indicator to the panel. Right click on the panel (top or bottom) where you want the indicator, select Add to Panel, scroll down to Ubtilities section and select Keyboard Indicator.
