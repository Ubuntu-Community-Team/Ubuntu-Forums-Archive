### Post by kane77 on 2007-01-18
first of all you have to make sure its executable ("ls -l" in the directory where the file is... if it is executable it should look something like this
-r-xr-x-r-x (only the x's are realy important... its enough to have only one x...)
if you are unsure, just use "chmod +x yourfile.bin"
then run it "./yourfile.bin"
