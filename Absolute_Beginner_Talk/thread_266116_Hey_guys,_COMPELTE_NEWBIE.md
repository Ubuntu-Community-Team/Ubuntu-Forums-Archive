### Post by D10 on 2006-09-26
Are you going to connect to a windows network with this laptop?

If so I would recommend starting with ubuntu (gnome desktop manager) or Mepis (KDE desktop manager) or kubuntu (kde as well). I have had better luck with mepis over kubuntu, but that is a personal opinion.

xubuntu would run really well on your laptop, but doesn't connect to windows shares as easily out of the box so to speak as ubuntu or mepis/kubuntu.

You can read about and get mepis at [http://www.mepis.org/](http://www.mepis.org/) Mepis uses the ubuntu repositories so you get the same packages that you would get with *ubuntu.

If you don't need or don't care about mounting windows shares I would go with xubuntu, would be faster than the others on your laptop.
