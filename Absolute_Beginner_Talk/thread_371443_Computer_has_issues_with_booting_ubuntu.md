### Post by shellhrs on 2007-02-28
Azoth, I don't know what version of Ubuntu you're using. I suggest you try Ubuntu Desktop Install CD first to see if it's running well. If it does, then try installing it directly from the desktop. It should work (if you have enough free RAM :-))

Alternatively, use the Ubuntu Alternate Install CD. It will not boot into a desktop, but it will give you a choice for different installation options.

I'm not familiar with your system, but if it were one of the older systems, or you want to have faster performance while maintaining Ubuntu ease of use & support, you may want to try different Ubuntu flavor (Fluxbuntu, Xubuntu, or Dreamlinux). They all came in LiveCD, I think.

As far I have tried (which is not much :-)), Dreamlinux has a better capability to recognise a system (unusual screen or audio etc.).
