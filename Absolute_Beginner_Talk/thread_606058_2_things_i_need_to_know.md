### Post by aysiu on 2007-11-07
I think it might make sense to post a link to the exact .tar.gz file you're trying to install.

.tar.gz just means that it is zipped up or compressed. The contents can vary widely from a packed theme file, to software source to be compiled... or a precompiled binary.

It's like asking how to install a .zip file in Windows. Sometimes .zip means it's just a bunch of files packed together. Sometimes it means it's a setup file that's zipped up together. There's no standard procedure.
