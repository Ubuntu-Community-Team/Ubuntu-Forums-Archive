### Post by yagood on 2007-06-20
It's because this particular file doesn't contain GTK Theme (skin for windows, buttons etc). Quote from the description:

> It consists of themes for GRUB, bootsplash, GDM, desktop splash screen and a desktop background

So you can just unpack it and use images directly for wallpapers etc.

You can also go to System > Administration > Login Window > Local > Add and select this file - it'll then add GDM theme - this is the theme for the login screen.

EDIT: If you want to skin windows, controls etc. you need to go to "GTK 2.x" category and download stuff from there and drag'n'drop it to Theme Preferences.
