### Post by pchachte on 2006-06-09
It all depends upon how you work and whom you're writing for. I'm a historian and I wrote my dissertation first in Latex and then I had to switch to OpenOffice in order to deal with others. And then I came across a wierd formatting bug in openoffice that forced me last minute to work in Microsoft Word. 

I personally don't like word processing software, because I've found Word and OpenOffice to be clumsy and buggy.  Latex is perfect, but you need to be comfortable with emacs (or another editor) and the syntax for latex formatting. And you ultimately need to worry about what format you will need when you submit your manuscript. Most publishers, alas, require doc format, so OpenOffice/Word might be your only choice.  You can easily work on OpenOffice throughout the writing and switch to Word for the final submission if you run into troubles.

If you plan to publish it yourself through lulu.com, then you're free to make your own decisions!

If you have any specific questions, I am fairly knowledgeable about this subject.

### Post by gr0kzer0 on 2006-06-09
I've had a novel bubbling away at the back of my brain for a couple of years, but I never got any further than very basic plans/synopses.

Now though, since I got a laptop, I've decided to get real with this project.  I only started very recently, so I still haven't written very much - but I've been using Mousepad to start planning the structure of the novel and also to write out ideas/scenes/character descriptions as they occur to me.

I use the xubuntu-desktop, so the text editor is Mousepad.  When I start actually writing the novel itself, I expect I'll be using Abiword.  I have toyed with the idea of using Mousepad for typing up the manuscript, but I don't know if its formatting is good enough to use instead of a word processor.

Another idea I've had is to write chapters/scenes/events to a blog, in the hope I get feedback from readers of the blog.  I know that authors can fall in love too much with their work, and getting feedback from a neutral reader can help keep you real.

### Post by fast1 on 2006-06-14
It depends which features you want to have, If you need just a spell checker, then an editor like emacs will work fine. If you need to emphatize words with italics or boldface, then you need a word processor like open office or Abiword. If you need to write math formulas then you should use Latex.
