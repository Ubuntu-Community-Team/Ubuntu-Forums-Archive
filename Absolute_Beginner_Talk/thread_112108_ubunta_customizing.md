### Post by meborc on 2006-01-03
i guess you mean the splash screen... the easiest solution for you would be to install **gnome-splashscreen-manager** from **synaptic** (or from terminal using **sudo apt-get install gnome-splashscreen-manager**)...

but if you mean the login window, go **system > administration > login window** (i don't remember the correct path in breezy) in [www.gnome-look.org](www.gnome-look.org) there are a lot of nice login screens to chose from or you can compile one yourself using some tutorials from [art.gnome.org]("http://art.gnome.org")
