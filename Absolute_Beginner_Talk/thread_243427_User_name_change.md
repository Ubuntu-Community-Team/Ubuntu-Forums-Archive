### Post by eeGhoong0ais on 2006-08-25
Changing the "real" name associated with a user is easy:[LIST=1]
[*]Go to System > Administration > Users and Groups from the panel menu
[*]Select the user whose name you want to change
[*]Click the Properties button
[*]Edit the Real Name field[/LIST]I don't know of any way to change the "login" username wihout editing /etc/passwd - which could easily get you into trouble. It would probably be easier just to add a new user with the name you want.
