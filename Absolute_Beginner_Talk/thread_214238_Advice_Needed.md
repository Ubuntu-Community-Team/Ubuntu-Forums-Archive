### Post by jsc1328 on 2006-07-12
Hi,

Unless you have set up seperate parts of your hard drive, ubuntu will overwrite your XP.

You can test out the broadband connection with the live/install CD you burned.  Just click on the firefox icon and see if you can surf around.  If you have a wireless LAN at home you might have some troubles.

Be sure and check the media before installing ubuntu.  In the boot menu you'll see something like "test CD".  Choose that option to make sure that the distribution got on to the CD correctly.
