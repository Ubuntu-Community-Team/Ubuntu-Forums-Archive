### Post by Zuuswa on 2007-03-30
Although you will want to make sure that you burn it at a slow speed (8x or less) because errors in the install disc can be very serious and frustrating (and a waste of a cd).  Also, make sure that you select some sort of option like "Burn image to disc"  and not just put the .iso file onto the disc, as that will not work either.  Although, I would not suggest running Ubuntu if you do not have a decent internet connection, as it does not come with proprietary codecs and such (.mp3 and windows media files, for example) will not work unless you download them from the repositorys, and as you know this may take some time on a dialup connection.  Otherwise, you should be all set and welcome to Ubuntu!!
