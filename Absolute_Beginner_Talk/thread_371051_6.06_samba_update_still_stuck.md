### Post by v8YKxgHe on 2007-02-26
Hey,

Not sure if this will work, but try:

```
sudo apt-get remove --purge samba
```

```
sudo apt-get clean
```

Then maybe,

```
sudo apt-get autoclean
```

After that try to install Samba again. Seems like the samba package was downloaded before, but got corrupt, and now when you're trying to install it again it's using the cached package (which may be corrupt) So by doing "sudo apt-get clean" it should remove all cached packages, then when you go to install Samba it will download a fresh package.
