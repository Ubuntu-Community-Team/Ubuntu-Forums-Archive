### Post by eapache on 2008-01-06
Just enter the command "nautilus ~/" for a custom launcher.

Nautilus is the name of the file browser, and ~/ specifies the users home folder.

### Post by phidia on 2008-01-06

Select location and type in your /home/(actual username) pick and icon and you should be finished.

### Post by scorp123 on 2008-01-07
> **brucewagner said:**
> Is there some way to Add a Launcher for my "PLACES-->HOME FOLDER" to my quick launch "Panel" at the top...? Drag & Drop. Click on the "Places" menu, locate the "Home Folder" icon, click on it and don't release the mouse button, and now drag it over to the panel where you want the icon. Now let the mouse button go. Voila, the icon is there. :)
