### Post by aysiu on 2008-01-03
It doesn't really matter what distro you use.

It matters more the window manager/desktop environment and applications you use.

DSL and Fluxbuntu both use Fluxbox as the window manager.

Puppy Linux uses IceWM as the window manager, but you can also use IceWM with Ubuntu.

Ultimately, though, if you stick with heavier applications (OpenOffice, Firefox, Evolution, Gedit), the benefits of a window manager over Xfce or Gnome will be negligible.

If you have 128 MB of RAM, I'd use lightweight apps like AbiWord, Dillo, Sylpheed-Claws, and Leafpad.

### Post by aysiu on 2008-01-03
> **~LoKe said:**
> So I suppose that means that the only reason to use such a light weight distro like DSL would be due to a lack of hard drive capacity (old drives, flashdrives, etc)?
If you're comfortable with DSL and/or like its default applications, then you should use DSL.

But it offers no advantage over Fluxbuntu or a custom Ubuntu installation with Fluxbox in terms of performance or hard drive space use.

As long as you're using the same applications and the same window manager, the hard drive use should be roughly the same. I guess even the barebones installation of Ubuntu takes up more than 50 MB, though. You might have to uninstall a bunch of system applications to get it down to the size of DSL.

So Ubuntu by default takes up more room, but it does not constrain you from having an equally (to DSL) barebones installation.
