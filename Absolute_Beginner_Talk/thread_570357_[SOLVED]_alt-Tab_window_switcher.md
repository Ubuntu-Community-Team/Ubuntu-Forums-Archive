### Post by macogw on 2007-10-08
The right key isn't broken.  It's just not really an alt key.  It's AltGr (Wikipedia it).  AltGr is the key you use to get á.  If you press AltGr and a together, you get á.  AltGr and e gets é.  AltGr and z gets æ.  AltGr and C gets ¢, while AltGr with c gets ©.  AltGr has been around for *ages* and was the standard use of that key on all systems until, well...Windows.

It's not really awkward to do Alt+tab though.  Dvorak puts those keys in the same place as Qwerty, doesn't it?  Do it with your left hand instead of trying to use two hands.  I tend to use my middle finger on tab and my thumb on alt.

The control, alt, home, end, delete, and backspace are under the left thumb and the only tab can reached only with the left little finger. ### Post by srf21c on 2008-01-09
I use a Kinesis Advantage USB keyboard, and was having the same problem with the awkward alt-tab sequence described.

I was able to solve this problem by changing the layout from the "Windows" configuration, to the "Non-Windows PC  configuration".   This changes the right "Windows" modifier key to an additional "Alt" key, and can be accomplished in a second using the "=p" keyboard sequence, as described in page 25 of the user manual.  This also aids firefox web navigation using the ALT-<arrow key> keyboard shortcuts immensely.

The difference between the two layouts can be seen here:

[IMG]http://www.kinesis-ergo.com/images/layout_contour-win-usb.gif[/IMG]
[IMG]http://www.kinesis-ergo.com/images/layout_contour-pc-usb.gif[/IMG]

Again this works with the Advantage USB, not sure if your particular model supports the hot switching features described.

To make the right alt key on the Kinesis Advantage work like left alt (not AltGr).

Run "xmodmap .Xmodmap" on the following file and make sure that you are in =p mode with xev if you have to:

```

! Xmodmap file for Kinesis Advantage MPC USB/XD + xgl/compiz
!
! =p mode:
! L ctrl alt -- alt ctrl R
!    37  64     113  109
!
! =w mode:
! L ctrl alt -- win ctrl R
!    37  64     116  109
!
! useful commands/files:
! xev
! xmodmap -pm

! keep Shift+BS from killing X
keycode 22 = BackSpace BackSpace

! fix right alt
clear Mod1
keycode 64 = Alt_L
keycode 113 = Alt_R
add Mod1 = Alt_L
add Mod1 = Alt_R

```
