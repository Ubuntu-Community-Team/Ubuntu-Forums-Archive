### Post by benner on 2007-06-04
if you are running ubuntu 64 bit you can't install wine (basically a windows emulator) but if you are running a 32 bit version you can.  the other option is to create a 'virtual machine' which is like another computer inside of your computer.  then, you install windows in there and it opens windows as though it were another program.  pretty handy.  it is called 'vmware'.  you need to get the server version to create a virtual machine but it is pretty easy once you have it installed.  just say yes to every question it asks as you install it and it will probably work fine.  to install programs, you can go 'applications > add/remove' or you can go 'system > administration > synaptic' an then run searches for stuff you want.  i don't know about halo but there are a million linux messengers that you can connect to your windows messenger friends and also connect to yahoo and many other networks.  there is one called 'gaim' that i think was installed when you installed ubuntu.  check it out.  good luck.
