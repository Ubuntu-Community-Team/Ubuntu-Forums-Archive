### Post by jimv on 2008-03-06
Go to Gnome-look.org and do a search for "Vista" or "Aero".  You'll find plenty of themes.

I used Murrina Aero for the GTK theme
[http://gnome-look.org/content/show.php/Murrina+Aero?content=54573](http://gnome-look.org/content/show.php/Murrina+Aero?content=54573)

An Emerald theme called Vista_q for the window borders
(I think it is in the Emerald non-GPL repos)

### Post by jimv on 2008-03-06
Also, make sure you get all the GTK theme engines:

sudo apt-get install gtk2-engines-* gtk-engines-*
