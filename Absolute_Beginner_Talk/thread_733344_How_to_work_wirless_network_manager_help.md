### Post by mikeyphi on 2008-03-24
> **Stjames said:**
> So i think i now have my BCM 43xx wireless card running in Ubuntu but i cant find where or how to enable the network manager to manage the wireless singles.  I think in 7.04 this is when you need to disable the network manager but i have no clue what to do.  any info would be great Thanks.

Open System/Administration/Network - that will provide access to the Network Manager where you can enable/disable; select roaming mode; set static dns etc.

If you need to disable/remove Network Manager then search under the Add/Remove application and 'untick' the box/apply.
