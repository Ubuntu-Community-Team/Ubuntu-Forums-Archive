### Post by jkeyes0 on 2007-10-24
In my experience, the most common problems people experience hardware-wise are with video cards, wireless network cards, and printers. For the most part, ubuntu works very well with a lot of hardware.

ATi graphics cards in particular can be problematic. If you just run with the default driver, you will be able to use Compiz (the cube, and the wobbly windows, etc.), but if you want to run any real 3d applications, like video games, you almost certainly have to enable the restricted driver, which (in my experience) doesn't like to work with Compiz. However, ATi is supposed to fix all that in the upcoming days. They have a new driver out that is supposed to work with Compiz by default, so hopefully it will get ported into Ubuntu in the next few months, but who knows?
