> I hate to say it, but it's the one thing Microsoft got right - installing things is EASY. I don't understand (I do realize that this is almost certainly gross ignorance on my part) why Linux programs can't come with that magical "setup" icon.
I don't particularly want to argue about this, but I don't think you're taking into account the nature of this release - they just dusted off this old source code they had laying around. It's just not packaged for end users yet.
On a bigger scale, you're right that "Linux" doesn't have a single simple way to install software. But Ubuntu does. There are a billion different Linux distributions in existence and getting them all to agree on how to install programs would be like trying to make everyone convert to the same religion. Meanwhile, in Ubuntu you can use Synaptic or just click on a .deb file and have it install a program about as easy as Windows can.
