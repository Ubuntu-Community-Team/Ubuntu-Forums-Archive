### Post by bodhi.zazen on 2007-05-02
It is exactly the same as installing into a single hard drive.

The only thing that changes is the designation of the partition to install into.

So, yes, assuming you have the space this is quite easily done. Yes you will be able to play MP3.

Defragment your second hard drive. Boot Ubuntu,  resize the partition, install ubuntu, install gurb to the MBR (master boot record).

BUT YOU NEED TO UNDERSTAND PARTITIONS BEFORE YOU INSTALL

so yo do not overwrite your data.

Partitioning : [http://ubuntuforums.org/showthread.php?&t=282018](http://ubuntuforums.org/showthread.php?&t=282018)

How to install : [https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall)
