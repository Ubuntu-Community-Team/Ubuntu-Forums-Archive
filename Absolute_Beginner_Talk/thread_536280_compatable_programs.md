> **jakebur01 said:**
> Will I be able to run most programs on ubuntu as i can on windows? Not exactly, Windows and Linux are two totally different Operating Systems and you shouldn't expect the installation or running of apps to be exactly the same. But with intuitive GUIs these processes are more relegated to the background and invisible to the user. Having said that, you will still need to get accustomed to the new OS and have a will to learn it.

You will also find that using the CLI (Command Line) to run a few things turns out to be far easier than doing a bunch of things in the GUI. Of course there are also a few things that require some basic knowledge of the CLI and Unix Commands. Nothing too difficult tho :)
