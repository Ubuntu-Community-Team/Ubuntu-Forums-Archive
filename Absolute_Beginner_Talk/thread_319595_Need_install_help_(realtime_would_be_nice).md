Well, if the GUI is not working for you, anything else will be faster. :)

Now this is a common, but very annoying problem for me.  The video detection doesn't work, and you get stuck at time zones.  To fix it, you need to look at the files on your system, which aren't there yet.  The alternate install is MUCH faster since it doesn't need to load an entire GUI, and runs in less ram.  Once it is up, you can right click on the desktop and change resolution.  You can open a terminal and edit your config file as well if needed.  Also, if you need more help, we need to know what video card (or video chipset) you have.
