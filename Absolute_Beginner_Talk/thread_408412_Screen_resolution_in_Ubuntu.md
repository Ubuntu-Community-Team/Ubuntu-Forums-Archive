### Post by ubukool on 2007-04-13
Hi there,

Do you have an intel graphics card?   If you do, there is a great package called 915resolution which you can download using the synaptic package manager which enables you to hack the intel BIOS and create the resolutions you want.

Here is the website for the package:

[http://www.geocities.com/stomljen/](http://www.geocities.com/stomljen/)

I recommend that you have a look at the README document which you can find here: [http://www.geocities.com/stomljen/readme.html](http://www.geocities.com/stomljen/readme.html)

I've got a resolution of 1200 x 800 running now with this package, no problem at all.  I had 1024 x 768 before, and this is much better  Even better, I didn't have to hack around with the xorg.conf files to get it!

Please don't hesitate to write back if this package can help you and you need some guidance with it. :D
