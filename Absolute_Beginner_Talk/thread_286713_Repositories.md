### Post by DirtDawg on 2006-10-28
Yes, you can. 

Go to the [Ubuntu packages page]("http://packages.ubuntu.com/") and enter whichever version you have  (I have dapper). Inside are software categories where you can choose programes. For example under the games category, the game Cowsay. Clicking the Cowsay link will bring you to a page that shows everything you need. Anything marked with a red dot must be installed. For example, Cowsay requires perl, which is already installed. Then the package can be downloaded from the bottom of the page and burned to CD. 

Sometimes things get ugly when dependencies require dependencies require dependencies ad hoc until Dependency Hell.
