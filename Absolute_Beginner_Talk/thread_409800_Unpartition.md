### Post by taurus on 2007-04-15
You can resize or merge partitions using gparted from the LiveCD if you have the LiveCD.  Otherwise, you can use GParted LiveCD, [http://gparted.sourceforge.net/livecd.php](http://gparted.sourceforge.net/livecd.php)
