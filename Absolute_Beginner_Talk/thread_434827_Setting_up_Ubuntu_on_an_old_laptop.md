### Post by Happy_Man on 2007-05-06
Xubuntu isn't that much different, just that it runs the XFCE desktop manager instead of GNOME. That's the only major difference. All packages are the same, everything else is exactly like Ubuntu. No worries!

### Post by freebird54 on 2007-05-06
With only 128 it will be SLOWWW - but it should be possible to run Ubuntu.  Not, however, recommendable.  The biggest difference in Xubuntu is that the desktop environment is much lighter, and it doesn't come with OO.o office stuff as a default.  A bit less configurable too, but quite usable.

An easy way would be to try Xubuntu.  If you want to add Gnome (standard Ubuntu desktop) you could easily add it later, and see if the speed is acceptable - if not then remove it again quite easily.  This link has some info and methods for implementing the above...  [http://www.psychocats.net/ubuntu/index](http://www.psychocats.net/ubuntu/index)

Hope this helps!
