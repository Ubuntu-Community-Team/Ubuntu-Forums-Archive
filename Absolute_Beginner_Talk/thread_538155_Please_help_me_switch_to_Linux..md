### Post by earl gray on 2007-09-01
Hi there,

No OS is perfect, but I enjoy Ubuntu for three things: the philosophy of sharing, the high quality of implementation and the infinite tweakability. I hope the community can help you on your road to Linux - just as they helped me a few years back when I was an inexperienced user and struggling with an incompatible modem.

In general, you should be aware that Linux is different from Windows. Getting used to differences in any field is frustrating. My advice is to perservere. Once you learn your way around Ubuntu, you'll find it's actually much easier to use than Windows.

Now, onto your questions:

> will Ubuntu work on this system?
Ubuntu has excellent hardware support, so you should be fine. To double-check, boot from a LiveCD if possible (for example, if a friend has the same components.)

> 
Is it really compatible with the same range of software/hardware as XP, Vista or OSX? Software i use includes, Cubase, autocad, adobe photoshop, iTunes (currently media player).
Commercial software is as a rule not available on Ubuntu, but there's open source software for all your needs. Check[ this thread]("http://ubuntuforums.org/showthread.php?t=141735") for a discussion of CAD software. I'm not sure about Cubase - would Audacity or Lmms compare? I'm too tonedeaf to use such software :-)

You can emulate Windows to run Windows-specific software - in theory. 

> 
Is it as safe to use online, security wise as windows?
MUCH safer.  Safety is one of the key reasons to switch to Linux.

> What internet browser comes with it?
Mozilla Firefox. 

> Does iTunes work on Ubuntu/Linux and will my iPod behave the same as it would on PC or Mac?
A friend of mine uses iShuffle on his Ubuntu machine, but not with iTunes. I'm afraid that's all I know.

> Will it recognize my digital camera, firewire soundcard and external hard drive like windows does and how do you install drivers etc?

It recognizes all my peripherals without any problem. If unsure, you can test your hardware with a LiveCD on your present machine.

If you need to install anything, your best bet is Synaptic, the tool that browses, downloads and installs software packages from repositories. As you grow in experience, you'll  also use Debian packages (Debian is the Linux that Ubuntu is built on) or even source code which you can compile yourself.

> I use NOD32 anti virus, does this work on Ubuntu/Linux?
No. And neither do viruses :-)

> Will it manage my on board sounds, graphics, keyboard (even if i use a new apple keyboard on my PC), mouse, usb 2.0 ports, firewire, HDMI port etc.

Yes.

> Does it have equivalents to plug and play, control panel, system restore, diskeeper.
Plug and play - you plug it in, it works. Control panel - it's not called that, but you get GUI tools for administration and tweaking. System restore - you get it through session management. Diskeeper - no, and no need to. Read the section titled Defragmenting a disk [on this page]("//http:/www.linux.com/articles/32002").

> Basically will it do all the good things my old OS did?
Yes - and more. if you take the time to learn it and get past the initial frustration of things working differently than they used to :-)
