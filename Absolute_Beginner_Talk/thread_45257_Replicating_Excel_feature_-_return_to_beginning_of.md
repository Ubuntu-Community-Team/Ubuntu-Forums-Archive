### Post by cwaldbieser on 2005-06-29
Well, don't know if it helps or not, but in KSpread (part of KOffice), you can configure the enter button to work how you describe:

Settings -> Configure KSpread -> Misc -> Press Enter to Move to -> Bottom, first cell.

Don't know how you'll feel about the other features, though.  I rarely use a spreadsheet for much more than a glorified table, or very rarely making a really simple chart.

### Post by El Jamo on 2008-07-01
Well, here's an answer three years later (I found this thread because I was having the same problem).  To get gnumeric to return to the column you want after hitting enter, click the cursor on a cell in that column and then save (ctrl+s).

That worked for me.
