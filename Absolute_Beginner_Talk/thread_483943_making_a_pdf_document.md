### Post by mcduck on 2007-06-25
Yes, PDF documents created with OpenOffice work just fine. You can't do forms or other advanced stuff that can be done with Acrobat but as long as you don't need that kind of things OO will work great.

I usually send all my reports and stuff as PDF's and create them with OO. This far I haven't had any problems.

### Post by ncappel1 on 2007-06-25
I have had experience, with this, I have found that I have not had any problems either with the print to pdf dialogue or the export to pdf. There are several options, the default works well for me, it is a good idea to test anyway though, just make a document and create a pdf in different ways. The default export to pdf works well, I could read a document in any pdf viewer, on Windows or Mac (or linux!). I should add, however, that I have not had any experience with printing out forms and other complex things, just wordprocessing.

### Post by kevdog on 2007-06-25
If you need to make pdf documents from other programs besides OO, I find install kprinter to be the easiest method.  I know this is a KDE package (dreaded by the Gnome Loyalists) but you dont have to install the full kde desktop to use this tool.  It works in much the same way as PDF Creator in windows, in that it installs a virtual print driver that prints directly to pdf.
