Making this a little easier to understand for you.  Ubuntu has servers called repositories that store thousands of packages (programs).  You can access the servers in the Terminal (like Command Prompt in Windows), or you can use a graphical program that Ubuntu has called Synaptic Package Manager.  Go through this - System > Administration > Synaptic Package Manager.  Click on the search button, and search for Ubuntu Sane.  You'll see it list a bunch of packages.  Right click, and hit install on the one you think is Sane (it's clearly marked).  Then, hit Apply.  It'll download and install it for you.
