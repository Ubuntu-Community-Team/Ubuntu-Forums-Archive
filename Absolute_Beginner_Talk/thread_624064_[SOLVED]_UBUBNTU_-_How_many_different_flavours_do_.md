### Post by rsambuca on 2007-12-01
> **sandysandy said:**
> How does one add the MINT repository to the sources of ubuntu to get the MINT working. i have edubuntu 7.10 and would like to try converting it to MINT.

regards

If you are using Gutsy, you have to use the Daryna repos.  To add this to your sources, use 'gksudo gedit /etc/apt/sources.list'

```
deb http://www.linuxmint.com/repository daryna main upstream import 
```

You will also want to create an apt preferences file that looks like this (gksudo gedit /etc/apt/preferences):

```
Package: *
Pin: release o=linuxmint
Pin-Priority: 700

Package: *
Pin: release o=Ubuntu
Pin-Priority: 500
```

The to get mint, there is a start menu program and a art/theme pack, which I don't know the names of.  Sorry.  If you search in Synaptic for them, I am sure you should find them easily.
