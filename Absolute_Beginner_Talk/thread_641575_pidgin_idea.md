### Post by llamakc on 2007-12-15
Yes: if you move your .purple directory (there isn't a .pidgin) to another machine all of your settings would be preserved.

You wouldn't even have to overwrite somebody else's .purple directory. You would just start pidgin from the command line with:

```


pidgin -c /PATH/TO/YOUR/DOTPURPLEFOLDER


```
