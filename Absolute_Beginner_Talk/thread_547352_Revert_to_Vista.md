### Post by wheredidrealitygo on 2007-09-09
vista comes with it's own partition manager, you just have to backup all of your important files from your linux partition (documents, videos, music, etc), then insert the Vista disc.

If it's an OEM (original equipment manufacturer) supplied disc (HP, Acer, Toshiba, Dell, etc) then it should load a ramdisk image and image your computer back to it's out of the box state.

If it's a retail version of Vista, than it will allow you to delete and remake your partitions to NTFS which is Vista's default file system.

### Post by wheredidrealitygo on 2007-09-09
Yeah that is a problem.

There's a workaround, though it takes a long-ish time.

delete both the partitions that you see (with the vista disc), and clean install vista.

do not put your product key in at the point of the install, but make sure you install your version of Vista that you purchased (so if you have Home Premium upgrade version, install Home Premium).

It'll allow you to install without the product key, but not activate.

Then, when it's fully installed and you boot into Vista, from WITHIN Vista, (on the desktop), insert the Vista disc and "upgrade" from your current version (clean installed home premium) to home premium again.

It's imperative that the upgrade be done WHILE the OS (Vista) is running, otherwise it won't work.

You're basically faking the OS into believing you're upgrading it.

Then, put the product key in during the upgrade, then the activation will work.

If you try to put the product key in while clean installing, it won't work, and it won't activate either.

(I used to work at the french activation centre, trust me, this works).
