### Post by ashmew2 on 2007-02-15
I dunno exactly how to fix ur problem , but if u like the command line (like me) , just open a terminal and do :
```

sudo shutdown -h now
```

u can replace now with the number of minutes u want the system to shut down in 

and for restarting :

```
sudo reboot now
```

again u can replace now with number of minutes after u want process to take place.
