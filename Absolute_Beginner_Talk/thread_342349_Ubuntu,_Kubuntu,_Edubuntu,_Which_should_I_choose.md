### Post by enopepsoo on 2007-01-19
Ubuntu uses the gnome window manager, and kubuntu uses the kde window manager.

Gnome looks more like an older version (but up to date if that makes sense) of macos, while kde looks more like windows.  You can configure either to look like anything you want really though.

xubuntu uses the xfce window manager which is lighter weight, but still compatible with the others.  :guitar:

### Post by Spr0k3t on 2007-01-19
You sound like you are very new to the Linux approach so let me detail what I've learned.  I've not worked with Linux very much outside of the days of the command line, so I'm having to relearn everything I knew about Linux... and it's quite different from the last time I played with it.

The main three are the ones you should consider imho.  Edubuntu is geared for a learning(sic) classroom environment.  I haven't tried it, but you can read up about it in more detail on the [http://www.ubuntu.com](http://www.ubuntu.com) web site.

So, down to the three.  Ubuntu uses a Gnome interface, Kubuntu uses a KDE interface, and Xubuntu uses Xfce.

Gnome is what some would consider a "tried and true" interface that has withstood the test of time.  KDE enthusiasts would possibly differ in response saying that KDE is superior.  However, until you try each for yourself, there's no way to tell which is the better interface.  The last of the bunch is Xfce which approaches the interface of Gnome with a minimalist approach making it fantastic for the older computer systems.

Hope that helps.

### Post by kinson on 2007-01-19
Well, it would also depend on your hardware.

If your pc has slightly older specs, such as 256ram, then I'd suggest Xubuntu, which is geared for lower spec pc's(not that it doesn't work fine in higher spec machines ;) ).

Like others have already mentioned, KDE looks a bit more like windows, and shows you more menu's, while gnome opts for the more simplistic approach. But what you can do in Ubuntu, you can also do in Kubuntu or Xubuntu. Not to mention you can turn Ubuntu into Kubuntu/Xubuntu(or whichever way) at a later time if you please.

But if you're asking me, as a newbie(much like myself), and if your pc hardware specifications are ok, go for Ubuntu. Because most of the help files are for it. And I'm assuming most people here are using Ubuntu, so it might be a little easier when people give tips, or you read articles(tips and tricks) on other sites, they normally talk about Ubuntu first.

My 2 cents: Go for Ubuntu :)
