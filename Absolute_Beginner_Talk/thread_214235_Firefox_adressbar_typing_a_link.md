### Post by mech7 on 2006-07-13
i found it..

about:config and set browser.urlbar.clickSelectsAll to True
Why this is default in windows and not ubuntu is strange ?

### Post by chickengirl on 2006-07-13
> **mech7 said:**
> i found it..

about:config and set browser.urlbar.clickSelectsAll to True
Why this is default in windows and not ubuntu is strange ?
Actually, it's the default for Windows and *only* Windows. For Mac OS and all versions of *nix it defaults to double click.

It's the same deal with tools > options vs edit > preferences. Windows uses the former, Mac and all *nix use the latter.
