### Post by macogw on 2006-12-24
You can install the Server version (so LAMP is there) and install the GUI over top.  If you want to work command line like a server, ctrl alt F1 is all terminal.  If you want to work on it like a desktop, run the GUI.  You can always admin over an internet connection using ssh tunneling.
