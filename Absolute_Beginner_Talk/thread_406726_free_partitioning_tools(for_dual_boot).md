### Post by xpod on 2007-04-11
Your ubuntu cd already has a version of "gparted" for your partitioning needs but you can go one step further and download the actual gparted live cd which is a great tool to have lying around.

[http://gparted.sourceforge.net/](http://gparted.sourceforge.net/)

Some people think it`s a lot more reliable on it`s own cd but i cant say i`ve ever had issues either way.

Defrag you XP a couple of times before you do any resizing and be sure to back up any imprtant data

[http://www.psychocats.net/ubuntu/partitioning](http://www.psychocats.net/ubuntu/partitioning)

This site will explain it better than i ever could..good luck:)
