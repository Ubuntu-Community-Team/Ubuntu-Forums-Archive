### Post by Jagot on 2006-08-09
You can copy all your Windows fonts over to your fonts directory in Ubuntu so it can use them.  Just go to a Nautilus window and press ctrl+h.  Now type in the address bar fonts:/// - you can drag any fonts you want to into there and they will be installed.  Bear in mind that this method will install for your user only.  If you want to install system wide fonts then you can copy them to your /usr/share/fonts directory.
