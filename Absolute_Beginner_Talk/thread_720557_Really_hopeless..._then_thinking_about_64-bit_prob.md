### Post by lswest on 2008-03-10
no idea, quad cores are kinda a gray area these days.  However, i can tell you you'll have less problems running a 32x Ubuntu on it, and the virtualization within quadcores will make pretty good use of the 64x capabilities, while running 32x (at least, this is true for AMD dual and quad cores, because there is hardly any noticeable speed difference between my AMD dual core 64 bit processor in 32 or 64 bit systems, due to the hypermemory or w/e they call that technology).  The only issue would be if you have 4GB or more of RAM.

*EDIT* amd calls it "hypertransport" and claim their AMD64 processors run the same on either 32 or 64 bit, so i dunno about quad-cores.  You can always give it a try.
