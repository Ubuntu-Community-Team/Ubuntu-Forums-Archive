### Post by yetanothersteve on 2006-07-29
I use the weather report adesklet that uses data from weather.com. I also have the weather report panel applet, but they are pointing to different parts of the country.

If I needed more desklets I would probably look at gdesklets, but adesklets does the job I wanted. 

To autostart the adesklet, I added this command to Session Start Up Programs of System Preferences Sessions:
```
adesklets -d 10 --nautilus
```
