### Post by pdlethbridge on 2007-09-23
I have, for many years kept to the KISS principle, namely, Keep It Simple, Stupid, and "if it ain't broke, don't fix it". Applying that to feisty helps keeping it working. I haven't had a crash or X server problem for a long time.
   The x server problem was solved by NOT using Automatix or Envy to load Nvidia drivers. It seemed that every time I installed the driver I would end up with X server issues. Letting the system use its own restricted driver made everything run just fine.
   With a litle site search here, I've been able to get my canon n1220u scanner operating in Xsane and my canon pixma 1500 printer working as well.
   A simple little program called tvtime allows me access to my cable tv here at the desk top. A little more searching or questioning will probably help me getting my generic remote going too.
   Sometimes it's better to keep it simple, and not go for the latest and greatest especially with installing software.
