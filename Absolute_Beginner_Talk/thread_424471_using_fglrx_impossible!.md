### Post by cstrippie on 2007-04-28
Ok, the guide you followed is a newer version of the guide I use.  After install I assume you did the "sudo aticonfig --initial" and "sudo aticonfig --overlay-type=Xv"

If the aboive assumptions are true, then the only difference between your X11.conf and mine should be the composite disable instructions.  Mine are as follows:

Section "Extensions"
        Option  "Composite" "Disable"
EndSection

Section "ServerFlags"
        Option  "AIGLX" "off"
EndSection

Verify all of the above have been done, then restart the machine and run fglrxinfo.  You should see something like: 

$fglrxinfo
display: :0.0  screen: 0
OpenGL vendor string: ATI Technologies Inc.
OpenGL renderer string: MOBILITY RADEON 9700
OpenGL version string: 2.0.6400 (8.35.5)

Though obviously with your x1600 and the version of fglrx you installed.

If this fails, post the results you get from fglrxinfo.
