### Post by ajgreeny on 2006-07-04
Hi Lepoja.  What did you mean exactly by:-

"I recently instaled Ubuntu 5.10 on my PC. I have Asus p4pe2-x motherboard that comes with installation cd (chipset utilities, drivers for sound & lan etc).

I'm unable to start instalation!"

Do you simply mean that you want to add other programs to your installed Ubuntu system with synaptic?  There is a lot of info her in the forum and in various parts of the wiki on how to install programs, and if you search you'll find it easily.

However, as a quick intro, here goes.

1  Open up synaptic.

2  Search for the prog you want to install using the search icon (probably best to search in name only to start and then name and description if that doesn't find anything.  You acn just search a term like mp3 if you want and then look through the long list of programs that mention mp3 in some way.

3  Click on the program's small square box at the left of the window and click "Mark for installation".  You can do this for as many programs/packages as you want.  Any dependencies will also be marked, so accept them if other required packages are listed.

4  Click apply icon at the top of window and that's all there is to it; everything else is done for you automatically.

Who said it was easier to install programs in Windows, where you have to find them first and download them, then go through all sorts of license restrictions etc etc.

Debian is just another linux distribution upon which Ubuntu is based.  All debian based distro's use apt, the backend of synaptic, for installation of programs.  Just forget your windows methods and systems and start reading about linux and Ubuntu, search these forums and the ubuntu wiki for info and you'll find it is quite easy and fun.

Remember!  It's not Windows, so don't try to use Windows knowledge.  You'll have a great time.  Welcome to the world of Linux and Ubuntu
