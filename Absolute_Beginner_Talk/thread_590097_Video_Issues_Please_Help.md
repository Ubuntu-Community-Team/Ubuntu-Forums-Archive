### Post by Hospadar on 2007-10-24
If you're having trouble with youtube, you need flash.  Install "flashplayer-nonfree" from synaptic.

If you have issues with other formats, go to Applications->Add/Remove and install "Ubuntu Restricted Extras"  make sure you have selected "All Available Applications" from the top right drop-down menu.

Also, just so you know, Automatix isn't the devil, but it can mucky up your software indexes and cause some issues, so generally it's better to avoid using.  Most of this codec type of stuff is resolvable without the use of automatix.

You could also try using a different media player like mplayer or vlc (which are installable through synaptic)
