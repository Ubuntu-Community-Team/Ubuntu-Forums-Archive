### Post by marco123 on 2007-09-14
The AMD64 version will work fine on an Intel 64bit.:)

As a reply to post #2: There is a simple script here for setting Flash up in 64bit Firefox: [http://ubuntuforums.org/showthread.php?t=476924](http://ubuntuforums.org/showthread.php?t=476924)  - Takes a few seconds.

64bit is definitely the way to go if you want to enjoy the advantage that 64bit provides, e.g. Transcoding video/encoding audio will be a LOT faster.
