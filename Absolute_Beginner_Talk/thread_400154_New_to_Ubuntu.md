### Post by 11hjpphty76lkjj on 2007-04-05
The installer has to connect to the internet to download and install dependencies for the application to run.  It's not so much that our software is 'missing' something, it's that Ubuntu needs to have several things installed to make the software work. 

You could look over the manual install procedure on the hplip.sf.net website and search for each package, and find the .deb file for each package and copy those to your flash drive.  But you may run into a situation where some dependencies requires an additional dependency and so forth.

Although i have not tested this so I don't have much more info beyond that.
