### Post by kebes on 2007-01-30
Dual boot will automatically appear as an option after installation, **provided you don't erase Windows!**

You said that you have a partition ready for Linux, so as long as you install to that partition, it will be fine. During the install you will reach a "partition my hard drive" step. At this step, **do not** select "erase the entire hard drive." Instead, select "use free space" or select "manually" and pick the partitions you've preppared for Linux.


So, if you set up your hard-drive properly during the install, then after the install finishes your computer will give you the opportunity to select what OS you want (Windows or Linux), each time it boots up.

It's always a good idea to back up important files before installing a new operating system, however!


I hope that answers your question. If not, feel free to post more questions.
