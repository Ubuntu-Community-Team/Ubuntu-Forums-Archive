### Post by az on 2007-07-10
> **7Priest7 said:**
> I think xp is the same min. specs

I will be fine...

Now for the other questions 
1. Is Ubuntu Install known for Freezing?


You won't be fine.  The installer is too busy swapping buffered ram that it's not going to get anything done.  The live cd installer needs 256 megs of ram to work.  Period.  The alternate cd can work with 192 megs.

Even once you install, if you try to boot with less than 256 megs of ram, it will be so slow it is unusable.

With adequate ram, linux is faster than windows XP (with the same amount of ram).
