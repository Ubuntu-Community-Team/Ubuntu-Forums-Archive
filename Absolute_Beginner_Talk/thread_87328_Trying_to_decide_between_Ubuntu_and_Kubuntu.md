### Post by raublekick on 2005-11-07
Gnome and KDE are vastly different, but for the most part equally as good.

Ubuntu and Kubuntu are the exact same base, they just use different desktop environments. If you install Ubuntu you can install Kubuntu as well (kubuntu-desktop) and visa versa (ubuntu-desktop). 

I'd suggest starting with Ubuntu, and then when you get the hang of things (or maybe not :) ), install Kubuntu in addition.

To install Kubuntu, or Ubuntu, in addition to which one you choose initially, type this in the terminal:
```
sudo apt-get install kubuntu-desktop
```


Additionally, applications from Gnome will work in KDE and apps in KDE will work in Gnome, although they may look a little funky sometimes.

### Post by Adrian on 2005-11-07
[QUOTE=Noah0504]There has to be more to it than just settling on Gnome or KDE.[/QUOTE]
Well, actually not. The only difference _IS_ the desktop environment, i.e. Gnome or KDE.

Of course, they come with different installed applications by default. For instance, Kubuntu uses amaroK and Ubuntu uses Rhythmbox. However, you can run KDE programs in Gnome and vice versa, so you can as well use amaroK in Gnome (or Rhythmbox in KDE) if you want to.

It really IS a matter of personal preference and I definitely recommend you to try both Gnome and KDE. You can have them installed at the same time and choose which one to use from the login screen.

### Post by Cufishing on 2005-11-07
In my limited experience, from a nutshell.

1. Gnome is flat, well laid out, and expects more user interface. It seems to respond a tad bit quicker, and is very straight forward. 'Most' programs are interchangeable through either sudo apt-get or the repositories. I too miss itunes, but use streamtuner as the launch point for internet radio, 'mo-better' than iTunes. You will also need XMMS as the engine. Once stream is up and your station is selected, it can be closed. There is also a plugin for XMMS to play MP3 files. 

2. KDE is very graphical, and very familar for those not wanting the extreme of gnome. I find KDE to be a bit slower, but light years ahead of windoze, I don't really care for the built in utility manager 'Konqueror'. 

It really boils down to personal preference. If you have broadband, download both of the 'live' CDs and give em a whirl.

Either way you will have to add 'codecs' in order to play MP3 files.

### Post by Brunellus on 2005-11-07
Gnome and KDE approach the desktop from slightly different angles.

KDE has a much more coherent feel to it, in my opinion.  Everything KDE seems to fit together very well.  Konqueror does double duty as a file manager and a web browser, and the usual KDE apps--Kopete, Kmail, the KOffice suite--all great.  The corrolary is that KDE seems to be at its very best when your'e living entirely in KDE-land.  The default KDE setup also seems rather cluttered and busy.

GNOME, on the other hand, is a little less polished than KDE;  there's a lot less visual chrome in gnome by default.  I have, however, found it to be a remarkably neat environment to work in--easy to tweak and tinker with.  Once you understand how all the little constituent parts work together, you can combine and recombine them in almost any way you see fit.  It's great fun.

Why not run the Kubuntu and Ubuntu live cds and see where you're more comfortable?  They're both good environments.

### Post by dtfinch on 2005-11-07
I recently (few weeks ago) switched from gnome to KDE on Breezy. Contrary to my past experiences, KDE has been very nice and responsive. Gnome 2 in recent years has been dumbing down the user interface and features a little too much for my tastes, and gnome-vfs and nautilus are full of annoying bugs and quirks. For example, copying a bunch of files would randomly fail with no error, as though it completed successfully. And several times in the past, nautilus has somehow crapped itself so badly that even reinstalling gnome and logging in as a new user would not fix it, each time encouraging me to reinstall Ubuntu from scratch.

Firefox takes a bit longer to start on KDE, but Konqueror works well for nearly all browsing needs. KDE is not without its own quirks, but the change of scenery is nice. Despite all the eye candy, KDE runs very quickly on my system, whereas gnome, with no eye candy, always ran very slowly, sometimes taking 10-20 seconds to display the gnome menu after clicking on it. Probably the most annoying KDE quirk so far is that with Konqueror maximized, there is a bar about 4 pixels wide to the right of to the vertical scroll bar where dragging it would resize my browser window rather than scrolling. Almost every other desktop and program in the universe fixes this sort of problem by placing the scrollbar exactly against the edge of the screen. I whine a lot more than I should.
