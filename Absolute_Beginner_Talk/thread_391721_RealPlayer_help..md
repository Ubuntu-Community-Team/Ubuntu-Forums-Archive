### Post by camz on 2007-03-23
Hey guys, I have downloaded the Linux version of RealPlayer 10 onto my desktop (I use the root account). The file name is RealPlayer10GOLD.bin What do I do with it now? When I click it it says it can't be opened.

---

### Post by taurus on 2007-03-23
Open a terminal and change to wherever you have saved it.   Then, run

```
cd ~/Desktop
chmod 755 RealPlayer10GOLD.bin
sudo ./RealPlayer10GOLD.bin
```
