### Post by nitro_n2o on 2007-07-02
You might need to edit your xorg.conf file 

type: 

```
gksudo gedit /etc/X11/xorg.conf
```
This will open the xorg.conf file. Scroll down till you see Section "Monitor" and add 1024x768 in the Modes, you'll find it in the Monitor section 

Then restart your xserver ALT+CTRL+backspace
