### Post by steve.horsley on 2006-11-22
It is for connexcting to "terminal server"s. These are devices that serve a GUI out to clients, such as Windows Terminal Server - many people all connect, get given a Windows GUI, but all the processing happens on the server - just keyboard and mouse stuff up and pictures back.

It's used for sharing computers, and also for remote conotrol of PCs such as servers in a server farm.

The client is easy to use - just bang in the protocol and address of the server. MS terminal servers accept RDP protocol. A very popular open source protocol is VNC.
