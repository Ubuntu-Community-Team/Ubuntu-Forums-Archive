### Post by gingermark on 2006-04-23
I am running the 32-bit version on my 64-bit computer. Right now getting some proprietery things to work (live drivers, codecs, flash etc) is far harder with the 64-bit version.

If you are a beginner I'd recommend using the 32-bit verison. If you put your home directory on a separate partition you can then install the 64-bit version when things improve.

That said, if you are in "learning" mode, using the 64-bit version will give you far more to get your head around, but you might be up for that.

As the previous poster said, it's all a matter of choice.

Do keep in mind though that the majority of the users on this forum won't be able to help you out on any 64-bit specific problems you might encounter.

### Post by steve.horsley on 2006-04-23
My advice would be to go with the 32 bit version. Currently, there are problems with several binary add-ons under 64-bit, such as the mp3 codecs, flash player. They can be made to work, I gather, but it's not easy. I also gather that the performance difference between 32 and 64 bit is minimal, so you don't lose anything staying with 32-bit.

You can install the k7 kernal after installing the main system if you like. This has some Athlon specific enhancements, and may improve performance. This is what I do on my A64.

There is nothing different to "learn" between 32 and 64. Same programs, same commands etc.
