### Post by ola on 2007-09-16
Hi! It sounds like your SATA controller is unsupported (just a wild guess) in Fesisty (7.04). You might want to try to boot up Gutsy (7.10 the next Ubuntu release). You can download it from here: [http://cdimage.ubuntu.com/releases/gutsy/tribe-4/](http://cdimage.ubuntu.com/releases/gutsy/tribe-4/)

Remember that Gutsy is not production ready yet (which means that you might have problems with some programs, system stability etc) but it might be worth a shoot.

You can also try to use the alternative install cds for Feisty (or Gutsy) and see if that makes you be able to boot up.
