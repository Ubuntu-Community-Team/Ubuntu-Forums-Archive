### Post by Inxsible on 2007-11-20
Depends. If it has the entire source, you would have to extract it first using```
 tar -xvzf *filename*
``` Then to install the software, cd into the directory and do```
./configure
make

sudo checkinstall
```If however, it is not a source code, then simply extracting it using the tar function would suffice.
