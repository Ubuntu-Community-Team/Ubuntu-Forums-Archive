The version that you have is a release canidate. It is not the final stable version of Dapper. Go ahead and install it, you can upgrade your system to the latest version of Dapper after it is installed.

Edit: I would not recommend the 64 bit version even if you have an AMD processor. There are fewer packages avaible and it is difficult to get things working. (flash, java, etc.) Download, burn, and install the i386 version.

### Post by catlett on 2006-06-01
Even if you don't have the newest iso. You will immediately get the updates via the update manager to make it the newest version. You will always have an up to date version of Dapper if you have an internet connection. The update manager will notify you of any updates, patches etc. and offer to install them for you. (you can even enable the update manager to install security and critical updates automatically without prompting)
