### Post by banewman on 2007-10-13
If you open system - admin - synaptic package manager then choose from the top left - settings - repositories - updates    you can be notified of security updates and/or recommended updates. Then , when they are available you get a desktop notification (a "balloon")  and can choose to download them or not. :)

### Post by drinkpepsi on 2007-10-13
System -> Administration -> Software Sources
Click the updates tab
Click the radio button next to install updates without confirmation.

---

### Post by banewman on 2007-10-13
If the "balloon" says there are updates available then one click will install them. I have mine set up for the security updates only. Works out to be once a month on average. There is an option to install security updates automatically. Check out settings - repositories   in synaptic package manager. :)
