### Post by nvrpunk on 2007-08-14
ESC in a terminal doesnt have a function.  Ctrl+C kills the running process you commanded, Q is the way to quit out of top and man.  The only application that uses esc in terminal I know of is alsamixer.

Try typing alsamixer and hit esc.  It probably works.

### Post by Dan_472 on 2007-08-14
Thanks for the suggestions.  The esc key does indeed shut down alsamixer.

It turns out this is some kind of problem with angband and is talked about here: [http://angband.oook.cz/forum/showthread.php?t=77&highlight=esc+key](http://angband.oook.cz/forum/showthread.php?t=77&highlight=esc+key)

So, for anyone who's interested:  the short answer is that the " ` " key (just to the left of the number 1) does what esc is supposed to do in angband.
