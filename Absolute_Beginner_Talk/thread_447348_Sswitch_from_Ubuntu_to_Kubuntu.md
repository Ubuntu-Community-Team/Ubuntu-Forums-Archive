```

```

All your files will remain. Enjoy!

### Post by louieb on 2007-05-17
> **darkrose said:**
> You can uninstall ubunttu-desktop if you want to get rid of Gnome.
Well actually no. ubuntu-desktop is a meta package. Which means that is sole purpose is to install multiple packages when its installed.  Nothing depends on it so when its removed. nothing else is removed with it. You will still have access to Gnome if you need it.
