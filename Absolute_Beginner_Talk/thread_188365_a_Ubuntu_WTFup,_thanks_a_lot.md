### Post by grsing on 2006-06-04
It's relatively trivial to dual boot Windows and Ubuntu (I'm a student as well, and have been using Ubuntu for most things, with Windows almost soley for games for several months).  Besides futzing with games, Ubuntu is usually quite easy to use (you didn't mention if you also use wireless internet, that *can* be a bit of work to make work, sometimes).  Unfortunately, there is always some risk when changing the partitions on a drive with data on it, and you just got unlucky.  It's easier to install the dual boot if you install Windows first and then Ubuntu, but just install Windows and don't bother installing other stuff until you have Ubuntu installed, so that if something goes haywire again, it's not much to reinstall.  I don't remember the Windows install, but if it will let you make different partitions, just do it then and you won't have to worry about it during the Ubuntu install (just leave two partitions unformatted, one for swap and one for root, since Windows won't format them for Linux).
