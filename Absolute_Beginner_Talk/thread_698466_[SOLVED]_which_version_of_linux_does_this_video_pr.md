### Post by Arthur Archnix on 2008-02-16
It could be virtually any Linux distribution, since many now use Compiz. Compiz is a window manager that does all that cool stuff. Then there are different desktops, Gnome, KDE, XFCE, the various *boxes. Then there are distributions, Ubuntu, Suse, RedHat, PCLinuxOS.

If you look at Ubuntu, it uses Gnome and Compiz Fusion in the latest version. So it can do everything in that video. Kubuntu is the same, except it uses the KDE desktop. And so forth.

So, to answer your question, maybe it's Suse, maybe it's Ubuntu, maybe it's something else. But if you're wondering which one to get so that you can do that, the answer is pretty much all of them. You can head over to distrowatch to find out more, but all those in the top 5 or 10 will be able to do this. And ubuntu Gutsy does it all out of the box (well, you have to install the compiz settings manager, but otherwise it's ready to go out of the box).
