### Post by bscbrit on 2005-12-22
The config.log says that it cannot find a file called crt1.o.

Are you sure that this is an Ubuntu compatible program?  My version of crt1.0 lives in /usr/bin/crt1.o and not in 

/usr/bin/ld: crt1.o: No such file: No such file or directory

Secondly, I cannot find a 'xvidcap' program in my repositories which suggests to me that you have downloaded this program from elsewhere.  If this is true, then you might have downloaded the wrong version.  

Alternatively, you might be trying to install an Ubuntu compatible program but haven't installed its dependencies.  This is the problem with working at such a low level and why programs such as synaptic or apt-get are so useful - they take all the difficulty out of installing new progs.
