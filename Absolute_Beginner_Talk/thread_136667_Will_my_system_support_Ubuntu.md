### Post by stuporglue on 2006-02-26
> If I go with Ubuntu I have read that I can then turn this PC into a file and print server, is this correct. 
Yes. I have a machine acting as a file server using Samba. Samba uses Windows file sharing, so you can access files from Win/Lin/Mac. Samba can also serve printers, but I used CUPS instead for that. My file/print-server is actually Debian, but Ubuntu is debian based, and can do the same things.

> Also can someone direct me in reading more step by step info on install and configuring as far as drivers for Sound and video cards.

If you're wiping windows, you can just pop in the CD and boot. You should be able to just accept all the defaults. Ubuntu will try to auto-detect the sound/video cards and usually does a fabulous job. If by chance it doesn't, come back once Ubuntu is installed, and we'll fix you up then. :-)

btw. I have installed Ubuntu on a couple of ~400Mhz PII machines and it works very well. If you can get more RAM, it'll be worth it, but 384 is good enough if you can't. 

Good luck!
