### Post by Big Dave on 2007-04-01
> **Psyik said:**
> If I install Ubuntu, then install KDE and remove GNOME, does that make it Kubuntu? Or XFDE; does that make it Xubuntu?

If you install the kubuntu-desktop or xubuntu-desktop metapackages on top of Ubuntu, you do essentially get Kubuntu or Xubuntu. I believe there are minute differences, but not much to be concerned about.

> **Psyik said:**
> Or can I install all 3 and choose which one to load up at boot?

On Ubuntu, you can install kubuntu-desktop and xubuntu-desktop. At login, you can choose if you want to boot into Gnome, KDE or XFCE.
