### Post by mostwanted on 2006-07-31
You will need all dependencies as well, so it's a hard job.

You can actually [download the Multiverse and Universe repositories on DVD's]("http://cargol.net/~ramon/ubuntu-dvd-en"), Main is already on the install CD, Restricted is largely [available on the Automatix CD]("http://www.ubuntuforums.org/showthread.php?t=145889") I think.

But honestly, you really need to have an Internet connection when you want to install something. Linux has always been distributed over the Internet and due to its constantly transforming nature, the only way to make sure software will work, is by using package managers.
