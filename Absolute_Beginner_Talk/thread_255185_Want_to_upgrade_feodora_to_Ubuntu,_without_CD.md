The CD image is designed to be burned onto a CD-R or CD-RW disc; a CD ROM is the finished product, ie, "Compact Disc Read Only Memory".

I suggest buying a pack of CD-R media and downloading the image directly from Ubuntu. The Alternative install image would be best.
