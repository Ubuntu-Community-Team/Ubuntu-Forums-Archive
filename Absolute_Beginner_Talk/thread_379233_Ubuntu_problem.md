### Post by etank on 2007-03-08
I would say that is probably not the best version to start with. The Windows based installer is still in beta I believe. If you want to try Feisty (also still in beta) I would recommend downloading the .iso file. Feisty Herd 5 has been quite stable for me. I am running it on two different machines. It is really pretty easy to set up a dual boot install too if that is what you intend to do. If you have any questions then feel free to ask.

### Post by etank on 2007-03-08
Yes it is. That is called dual booting. When you run the CD and are ready for the install you have a chance to shrink the Windows section. This will create an empty area on the hard drive where you can install Ubuntu.Ubuntu will then take care of creating the grub menu (boot loader) that will give you a choice at boot time.
