### Post by chamberlain2006 on 2007-11-17
To fix that, go to Account>Preferences, and click the Others tab.  Type "gnome-open $file" (without quotes) in the Open file command box, and change the File Manager box to read "nautilus $location" (without quotes).  Then click Save.
