### Post by skymt on 2006-10-27
Too much of Ubuntu 6.06 is tied into Firefox 1.5 to make a real upgrade easy. Someone made a [script with howto](http://ubuntuforums.org/showthread.php?t=284317) that installs 2.0 alongside 1.5.

If you want 2.0 *instead of* 1.5, you'll need to upgrade to the new version 6.10, which just came out. You should see the [wiki](https://help.ubuntu.com/community/EdgyUpgrades) for instructions on how to upgrade.

I'd wait on upgrading for now, for two reasons. First, because everybody's upgrading right now, so the download will be very slow. Second, because of the short development cycle (6.10 was made in four months, compared to 6.06, which took ten), people have been reporting some fairly serious problems with the upgrade. My upgrade went well, but some others haven't been so lucky.
