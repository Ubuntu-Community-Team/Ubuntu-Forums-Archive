```
 sudo mkdir /media/c:\\ 
```

then

```
 sudo gedit /etc/fstab 
```

change where sda1 mounts to /media/c:\\

### Post by logos34 on 2007-08-15
> i just started playing around with fstab so this line "/dev/sda1 /media/ ntfs-3g defaults,locale=en_GB.UTF-8 0 0" had "/media/c:\\" somewhere in it. I tried deleting it to see if it would work.
... 

**/dev/sda1 [COLOR="Red"]/media/[/COLOR] ntfs-3g defaults,locale=en_GB.UTF-8 0 0**

Edit:

Try a more orthodox mountpoint name like /media/windows, /media/ntfs or simply /media/sda1 .

**sudo mkdir /media/windows**

then edit the fstab line:

**/dev/sda1 /media/windows ntfs-3g defaults,locale=en_GB.UTF-8 0 0**


Reboot.

Then remove the old c:\\ mount folder
