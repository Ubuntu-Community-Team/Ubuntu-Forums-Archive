### Post by KIAaze on 2007-06-19
The important thing is to "burn image"/"burn ISO", i.e not burn as data.
[http://www.psychocats.net/ubuntu/iso]("http://www.psychocats.net/ubuntu/iso")

edit:
> it came up as an unknown file I duno if this is normal or not. 

It should come up as a ".iso" file. If you have no CD burning program with which .iso files are associated, it is normal that Windows doesn't know it.
.iso files are not meant to be opened, they are CD images meant to be burned as is on a CD. (not as a file on a CD, but as I said "burned as an image")
