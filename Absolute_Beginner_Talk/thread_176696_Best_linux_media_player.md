### Post by darkninja on 2006-05-15
Totem in Dapper dosn't support DVDs and VCDs yet. However GStreamer 0.10 is much better (faster and reliable) then 0.8

I like using Totem-Gstreamer for most things and VLC for DVDs and stuff Totem borks on.

For GStreamer make sure you install the win32codecs package (which isn't in the official repos) and the gtreamer0.10-pitfdll package or the gstreamer0.8 version depending on what you're running. It should be able to handle most kinds of video formats that way.
