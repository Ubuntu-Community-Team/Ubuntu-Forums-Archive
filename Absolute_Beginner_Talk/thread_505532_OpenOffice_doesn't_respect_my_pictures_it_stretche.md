### Post by swisscow on 2007-07-20

As for setting metric, I'm not sure how to change it by default but if you want say a width of 10cm, type in "10cm" (this is from the help file.

### Post by grishenko2000 on 2007-07-20
To change the measurements to metric
tools > options > OpenOffice.org Writer > General
and then change the measurement unit.

to set the picture to original size right click on it > picture > type tab click original size
