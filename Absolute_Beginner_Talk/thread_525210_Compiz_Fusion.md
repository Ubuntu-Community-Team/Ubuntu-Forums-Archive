### Post by proalan on 2007-08-14
yea it annoyed me too this is how i tweaked

go to

```
System->Preferences->CompizConfig Settings Manager
```then click on 'General'

then the tab labelled 'opacity settings'

double click the line in the textbox "window opacities" and change it to whatever you want

set the value to 100 for no transparency
I think its set at 70 by default

You can also customize settings for other elements using your own expressions, perhaps they will make it more GUI friendly in future versions
