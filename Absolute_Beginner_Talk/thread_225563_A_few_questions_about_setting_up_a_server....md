### Post by T700 on 2006-07-29
> **Oxin said:**
> 
1. I have a USB Motorola wireless adapter. Will I be able to use it?

Maybe.  USB adds a whole additional layer of complexity to getting wireless to work.

> **Oxin said:**
>  2. Can I set up my server to rebroadcast the wireless signal, like an ad-hoc network?

Yes, but it would be *much* easier to buy a piece of dedicated hardware to do that.  Doing it off your server would be almost the same cost, as you will need to buy add hardware to rebroadcast.

> **Oxin said:**
>  3. Can I setup the server to stream music?

Yes.

> **Oxin said:**
>  4. Is there a program I can use to remotely connect to the computer and have full control of the mouse and keyboard?

Many; for example, VNC.  Be extremely careful with these and understand the security implications.  There are few quicker ways to get your server owned, then a poorly setup remote access program.

> **Oxin said:**
>  5. Will the server ask for a password on boot? In case of power failure or accidental reboot.

It is possible to set it to not ask for a password, but you should understand the security implications of this.

Welcome to the forum and good luck!

Paul
