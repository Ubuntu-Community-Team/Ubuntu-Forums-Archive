### Post by expatCM on 2007-08-19
I can understand your caution but ....   I do not understand your problem.

Why not change your BIOS settings so that the CD drive is the first device in the boot sequence and simply boot your Ubuntu CD as a Live CD and then have a look round and see if this is what you want?

Having booted to the Live CD you may then, if you wish, if you like what you see, simply install from that session, there is an icon on the desktop which makes it all simple.

The installation instructions are well set out and you are not likely to get lost.

However, please note and note well that this is NOT Windows.  With Windows you can click and hope.  With Ubuntu (and other Linux distros and Mac) you do need half a brain to at least get some idea about what you are doing.  It is harmless when doing a desktop install but it is also an opportunity, an opportunity to learn a lot more about computers.   If you do not want to do any learning at all then stay with XP.

No matter what you do ....  it is a good idea to back up your data :)
