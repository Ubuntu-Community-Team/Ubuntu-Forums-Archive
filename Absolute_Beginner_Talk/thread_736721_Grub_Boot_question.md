### Post by Jorge32 on 2008-03-26
i have xubuntu 7.10 and Windows XP in my computer, and I was wondering if there was a way (hopefully an easy way) to change the order of the booting options? I mean, changing Windows to the first option instead of the last one above all of Ubuntu's options?

### Post by smartboyathome on 2008-03-26
Not unless you want an update to take away your Windows boot option, because it would erase your windows line and put only Ubuntu when the kernel does update (which can happen when you upgrade from one version to another, or within the same version of ubuntu).

### Post by glennric on 2008-03-26
> **jken146 said:**
> There's a very easy way.  Change the entries around in /boot/grub/menu.lst

There is an even easier way.  Change the line
```
default 0
```
near the top of menu.lst to
```
default #
```
where # is the number of the boot option you want to boot to by default.  Note that the first boot option is 0.  So if you have 5 boot options, and windows is the last one change # to 4.

Edit:  If your menu.lst has a dummy boot option like "Other operating systems" and the windows boot option is after that, you might have to count that too.  Not sure.
