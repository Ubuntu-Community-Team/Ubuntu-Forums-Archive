### Post by jordanmthomas on 2007-11-20
There is a package called bsdgames that has a command you may like:
```
sudo apt-get install bsdgames
```
Then, run this
```
wtf is programname
```
and you'll get something like this:
```
**wtf is lspci**
lspci                (8)  - list all PCI devices

```

Also, you can run
```
man commandname
```
and the first thing it tells you (generally) is what the program is for.

Man pages are great.
