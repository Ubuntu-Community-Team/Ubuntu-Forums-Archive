### Post by ewl1217 on 2006-09-29
Windows managers simply manages windows (thus the name). A complete desktop environment includes a window manager in addition to other programs (which vary by each desktop environment) such as a panel, a desktop manager, applets, and so on. Most desktop environments (such as GNOME, KDE, and XFCE) also have many applications made just for them (like Rhythmbox in GNOME, Amarok in KDE, and Xfmedia in XFCE).
