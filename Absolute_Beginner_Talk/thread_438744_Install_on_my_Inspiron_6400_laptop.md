### Post by wormser on 2007-05-10
I have a 6000 and everything besides the broadcom wireless card worked on install.  If you have an Intel card it should work fine.

You can dual boot XP and Ubuntu if you want to keep XP on it.  I would recommend backing up all your data and install XP over again.  Set up a partition for XP and leave the rest blank.  Install XP then install Ubuntu.
