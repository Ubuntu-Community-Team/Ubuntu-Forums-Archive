### Post by ruy_lopez on 2008-03-13
This starts the GUI version.

```

gvim

```

If you want a launcher, right-click the toolbar -> add to panel -> custom application launcher -> name="gvim" command="gvim &". There should be a vim icon you can add by clicking on "no icon" button.
