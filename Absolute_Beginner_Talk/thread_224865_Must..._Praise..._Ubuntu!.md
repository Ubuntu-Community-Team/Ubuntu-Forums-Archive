### Post by tocleora on 2006-07-28
Oh my gosh... Ok so this is my first time to post here... But I just have to tell someone my story and I figured if anyone appreciated it the Ubuntu group would...

I've been using Linux since Red Hat 5.x.  I've been running Fedora Core 4  on my desktop since probably November or December (can't remember exactly when I installed it).  I've used most versions of Red Hat past 5.x, Fedora Core 2, Fedora Core 4, Knoppix and even an earlier version of Ubuntu.  I've never had 100% luck with any version of Linux, there was always something that was missing or not working correctly, but since I was using Fedora Core 4 at work and because it's associated with Red Hat, I've been using it on my desktop, but I used Windows primarily because the biggest problem with FC4 was I couldn't get my Wi-Fi working, but there were other problems that I just couldn't commit no matter how much I wanted to.

So I decide to try Ubuntu 6.06 last night.  It was a nice touch that I could actually play games, use Ubuntu I guess while installing Ubuntu on my hard drive.  That was the easiest installation process I've seen yet, and it didn't take forever (5 disks?  I think FC5 is 6 if I remember correctly).  One major note is that my wi-fi worked!!!  Even during installation!!!  I had tried every howto and man entry available for fedora with no luck, it worked instantly in Ubuntu!  Another problem I had with FC4, for some reason my headphone jack wouldn't work on my laptop.  But found two different options in Fedora for controlling my laptop speakers and the headphone jack so that worked as well!  Firefox 1.5 wasn't available for FC4 last time I checked,  Ubuntu came with it preinstalled.  I was also able to get MySQL 5 for Ubuntu which I wasn't able to get for FC4.  Sound has always been crackly for me, especially the sounds when you startup and shutdown. no crackling during Ubuntu... So far (knock on wood) everything I couldn't do in another linux version I can do in Ubuntu!

So I dared to take the ultimate test... I've used multiple linux's on multiple computers using multiple configurations... And I've *never* been able to play tuxracer. It's always been really choppy, sluggish... What are the chances it will work in Ubuntu?  So I download it and install it, get some other things working like dual boot thunderbird (great) firewall (awesome), apache/php/mysql (no problemo)... and I decide to start it up... and for the first time ever.  I play a game of tuxracer.  Wow.  

So needless to say, I'm very happy with Ubuntu. :)
