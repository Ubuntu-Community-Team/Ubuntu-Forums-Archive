### Post by megamanXplosion on 2008-02-13
Recently I've been using Ubuntu, and I quite like it. I've found a slight annoyance I hope someone could alleviate. In Windows, one writes the em dash using Alt+0151 but this doesn't work in Ubuntu. How do I type an em dash?

---

### Post by Jussi Kukkonen on 2008-02-13
System - Preferences - Keyboard - Layout Options - Compose key position

Check "Right Alt is Compose"

Now "AltGr + ---" will give you a &#8212;.
