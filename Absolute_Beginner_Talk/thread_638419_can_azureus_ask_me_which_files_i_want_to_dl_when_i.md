### Post by rahimveron on 2007-12-12
Yes Azureus would pop-up a small window where you can select the destination folder and the files you want to download.
Just make sure onder Tools>Option>Interface>Display "Add Torrent Silently" is not selected.

### Post by RomeReactor on 2007-12-12
Hi. In Deluge, go to "Preferences", and on the "Downloads " tab check **Enable selecting files for torrents before loading**.
