### Post by ramjet_1953 on 2007-05-27
One of the nice things about Linux, is that it uses RAM that is not being utilised for programs as cache.

This means that things happen much faster, as the HDD doesn't need to be accessed as much. 

If you run one of those RAM monitoring applets, you will see what I mean.

So to be specific, the more RAM you have (within reason) the better.
