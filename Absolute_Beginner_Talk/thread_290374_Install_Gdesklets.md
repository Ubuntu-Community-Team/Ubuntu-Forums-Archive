### Post by Stex on 2006-11-01
You probably need to add the extra repositories. Gdesklets is in Universe.

Open up synaptic and go to Settings > Repositories. If you're using dapper then there should be a big list and you need to select the one that says Universe - source. If you're on edgy / have updated synaptic then you just need to check the first option ("Community maintained Open Source software (universe)").

Remember to hit reload afterwards then search for gdesklets again.
