### Post by bdash on 2005-10-20
For compatibility with Windows applications, search in Google for:
- wine (the general project)
- crossover (focused on some soft, such as MS Office)
- cedega (for games)

But be aware that there are very few applications fully supported (maybe none?), and it will never be as good as a Linux application.

As for VB.NET, I don't think you can install the Microsoft version on Linux. But you can develop in VisualBasic.NET using Mono (not sure it is fully implemented)
[http://www.mono-project.com/VisualBasic.NET_support](http://www.mono-project.com/VisualBasic.NET_support)
