### Post by phr0ze on 2007-12-12
32bit OS's can natively address only 4GB of ram. There are ways of memory management to access move than 4GB though. 64bit OS can address 128GB of ram. As far as speed, I'm not sure.

Although my new system has 5GB of ram in it, I'm going to stick with the 32bit OS (of any flavor) because I don't want to deal with any compatability problems.
