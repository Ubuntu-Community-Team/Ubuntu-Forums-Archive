### Post by TheOtherShoe on 2006-08-18
The only way I know of to get complete transparency is to use Compiz with either XGL or AIGLX. The software is still experimental and setting it up is not a trivial operation. But if you are interested you can find a lot of howtos in the forums.

To point you in the right direction: if you have Intel graphics hardware then you will want to use AIGLX because Intel is the only graphics chipset manufacturer that has released drivers that are compatible with AIGLX. If you have graphics hardware by ATI, Nvidia or any other chipset maker, then you will want to use XGL.
