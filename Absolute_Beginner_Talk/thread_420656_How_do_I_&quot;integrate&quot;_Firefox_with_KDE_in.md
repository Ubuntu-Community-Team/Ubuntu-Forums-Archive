### Post by igknighted on 2007-04-23
Well, firefox IS a gtk app, so it will always look more like it belongs in Gnome... however if you browse through the firefox themes there are some KDE like ones.  I would look int he firefox preferences menu to to try to set different default apps.  Finally, go to the kde control center and under appearance there is a tab "gtk styles and fonts".  Change this to show the qt fonts, and install the firefox/thunderbird fix as well.

I would even more strongly recommend trying to love konqueror.  It is a much faster browser, and while it doesn't have ALL the features, it comes close.  Opera is another good choice because it is built on qt and fits in better as well.
