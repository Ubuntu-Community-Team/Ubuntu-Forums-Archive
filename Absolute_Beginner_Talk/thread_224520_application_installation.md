### Post by 3rdalbum on 2006-07-28
It is safe to delete the ubuntu-desktop metapackage when you uninstall something that came with the system.

Thunderbird is in the repositories, and Yahoo Messenger for Linux has fewer features than GAIM or the KDE IM program.

Programs that aren't in repositories have all sorts of different ways to install... best to read the instructions that come with the particular program. For instance, Google Earth uses a binary installer like you would find on Windows, Skype uses a .deb file, and most other programs use source code.
