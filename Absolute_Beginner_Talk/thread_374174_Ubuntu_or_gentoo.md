### Post by igknighted on 2007-03-02
Gentoo would be a trial by fire.  In fact, if you don't know linux at all, and your computer is not super new, be ready to be without your computer for a day or so as it compiles EVERYTHING from source.  Personally, that appeals to me, then again, my processor is fast and can compile stuff fast.  Gentoo can be a challenge, but the install process is very well documented.  I would strongly recommend having another computer nearby to access the net so you can get help from their forums or browse the manual.

Basically it comes down to:
1) If you want a system up and running linux in 30 minutes all set ready to  go, I'll get my feet wet slowly and learn as I go, use Ubuntu (or some other binary distro like Mepis or Suse).

2) If you want a trial by fire, sock it to me now, I'm gonna learn how this all works approach, then go with gentoo, do your HW well, have the install guide with you as you go, and I think you'll be OK.  But the compiling can take up to a day to complete, so be aware.

Personally I think a gentoo (or other source based distro) install is something every linux user should do at some time... you learn so much.  But I would not suggest starting with it.  You will understand what you are doing so much better if you know the basics of the OS first.

EDIT: Gentoo might run games better because (a) there is a lot of support in their forums, its where a lot of linux gamers are, and (b) its all optimized for your system, so there will be some performance gains there.
