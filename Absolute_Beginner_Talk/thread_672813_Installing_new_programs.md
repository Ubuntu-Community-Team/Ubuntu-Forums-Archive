### Post by jleaker01z on 2008-01-20
A .exe file is a Windows file.  Ubuntu is not Windows and will not natively run anything for Windows.  You need to find alternatives for what you want.  For example, Pidgin or GAIM is a suitable alternative to MSN, and VLC or RhythmBox for Winamp.  Note that Gaim and RhythmBox are already installed in a default Ubuntu installation.

You can find GAIM by clicking Applications>Internet, and RhythmBox is under Applications>Sound & Video
