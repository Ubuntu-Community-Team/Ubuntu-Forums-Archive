### Post by qpieus on 2007-11-16
Most system-wide config files are in the /etc directory. Personal config files are usually hidden files in your home directory. So, if you back up those 2 directories, you'll catch most, if not all, of the important config stuff. My /etc directory is only 6 MB, so I backup the whole thing.
