### Post by rygar on 2007-05-07
Ubuntu has no need for boot disks.  The Ubuntu Desktop cd is a "live cd", meaning you can run a fully functional version of the operating system entirely from the cd, without installing anything to your hard drive.  This means that you can run any sort of system utility to diagnose and fix your computer problems, and/or demo the operating system itself.  if you choose to install Ubuntu, just double click on the "Install" button on the desktop.

### Post by NilsE on 2007-05-07
It's really just 3 steps or 4 if you choose to install to a hard drive.

1) Download the ISO of your choice from the main page download section

2) Burn the ISO file to a CD. Noy copy the ISO file but use a cd burner to create the bootable CD from the ISO file.

3) Boot from the Live CD you created and run Linux

4) This is the optional - Install to the hard drive but read up on installing first.
