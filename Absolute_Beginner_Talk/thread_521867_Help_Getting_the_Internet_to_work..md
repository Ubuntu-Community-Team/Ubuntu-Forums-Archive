### Post by p_quarles on 2007-08-09
Wireless can be a pain. The way that many of the adapters actually work is a tightly kept secret by the companies that manufacture them, so it's difficult for developers to create Linux based drivers for them.

In the vast majority of cases, you'll be able to get it to work using an application called NDISwrapper, which allows you to install the Windows driver for the wireless adapter. I've never done this, so I can't help you with it, but if you go to the "Tips and Tutorials" sub-forum, and search for "NDISwrapper," you'll find some guides.
