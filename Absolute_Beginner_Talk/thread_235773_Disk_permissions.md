### Post by em3raldxiii on 2006-08-13
Ubuntu (Linux in general) cannot natively write to an NTFS partition. There is an experimental system that will allow you to write to your NTFS partition, but it is NOT supported, so it is use-at-your-own-risk. I don't have the link just now, but I am sure if you do a quick search of the forum that you will find it. In the case of your USB drive, if it is also formatted in NTFS, the same problem will occur. It is not actually a permissions problem (well, it MIGHT be in the case of the USB drive). I'll see what I can dig up.
