### Post by Ziggy72 on 2007-03-27
You haven't given much information about how you want to use virtualization, nor have you given any details about your computer system, but assuming that you want to run Ubuntu as a windows guest on a standalone PC then I suggest you consider VMWare Player or VirtualBox. Both of these are free. 

VirtualBox is still in development and there are a number of issues with it which aren't very satisfactory and the VirtualBox forum support is poor. That said, VirtualBox is still quite useable.

I haven' t used VMWare Player to create a Ubuntu guest on Windows but I'm led to believe that it's good as long as you don't have 2 CPUs because only systems with one CPU are supported. Using this, you're probably much more likely to get support, if needed, because the user base is much larger.

Of course the program of choice if you want to create a Ubuntu Guest machine on a Windows host is VMWare Workstation. That is what I use and it is excellent. However, it has a price tag of nearly USD 200 and so it's not for everybody.

If you are booting into Ubuntu and want to create a virtual Windows guest then VMWare Workstation for linux works well.

I hope this helps.
