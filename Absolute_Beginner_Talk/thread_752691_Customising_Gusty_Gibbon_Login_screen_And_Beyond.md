### Post by NightwishFan on 2008-04-12
If you downloaded the mac4lin, wallpapers, and icons from the link I gave you:
Extract the Mac4lin.tar.gz file. Then find the folder gtk-metacity theme. Open appearances where you theme gnome. Drag one of the files inside that folder into the appearances window. It should say, theme successfully installed, nothing will change.  Now drag the mac4lin icons into the same window with similar results. (Remember you have to download them both on the site) Extract the wallpapers tar.gz and set the blue-tiger or leopard.jpg (for now) Close and reopen appearances. Hit customize. Set the mac borders and icons. It should now look very much like a mac os x except the dock. I will tell you how to get awn, and install the theme for it later as I am busy, but perhaps someone else can help you with that.
