### Post by crispy_420 on 2007-04-06
Dual boot is actually easy to do. Dual boot is basically installing two OSs on the same system. When you boot, you choose which to use. They run independently of each other and basically unaware of each other. Is that what you want?

Or do you mean run ubuntu within windows? Say like in it's own window as an application? If so that will require a virtualization program. That will mean spending a few bucks on software like VMWare or Parallels Workstation. There are free options but they are not as easy.
