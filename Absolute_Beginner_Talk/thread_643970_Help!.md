### Post by Cypher on 2007-12-18
You'll want to add the following
```

Option "AddARGBGLXVisuals" "True"

```
to the Screen section of your /etc/X11/xorg.conf file to make the window decoration appear.

### Post by Cypher on 2007-12-18
If you are doing things on the terminal then..
```

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak
sudo vi /etc/x11/xorg.conf

```
VI can be a little scary initially...but just use the arrows and scroll down to the Screen area in the file and after the "DefaultDepth 24" line, hit the 'i' (eye) key to Insert, type the text that I suggested. Now hit the ESC key to leave 'edit mode' and then type in ":" (colon, shift+semi-colon) and follow that with "wq" and ENTER.

This should now add the line, safe the file and quit. You can now restart X and see if it behaves better..

If that doesn't work as well, we can rever to the .bak (backup) file, disable compiz for the time being and use a nice GUI editor (gedit) to edit the file..
