### Post by forrestcupp on 2008-04-13
It depends on what kind of DSL you have.  Older DSL services had to have a dedicated line run just for internet.  Now, usually you can use any phone line for DSL internet, but you have to plug your phones into filters to filter out the internet signal.

If you have the type of service where you can use any phone line for your internet, you should be able to plug each modem into their own jack and not have any trouble.  If you have the type that has to have a dedicated line, which usually uses a larger jack, you may need to get with your phone company to see if they have some kind of splitter.

But it should be possible to do what you want to do.
