### Post by steve.horsley on 2007-07-07
After an Ubuntu version is released, only security updates or fixes for  really, really horrible bugs are released. This is to reduce the number of changes (and therefore breakages) in the installed user base. Nothing upsets users so much as releasing a "fix" that breaks a working system.
