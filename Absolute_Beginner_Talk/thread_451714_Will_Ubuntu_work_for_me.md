### Post by Sparkster185 on 2007-05-22
> 1. I need to learn how to add plugins to make stuff like videos work
2. I need to figure out how to make the os recognize my wireless network and periphials (printer, etc).
3. I need to learn how to run windows based programs.

0.  I switched to Ubuntu about last September.  I've been very pleased.  Granted I do use Linux servers at work, so I had a little experience with the terminal, commands, etc.

1.  For me, Feisty solved these problems I had with Firefox.  Flash worked out of the box and installing codecs is really easy with Automatix.  There are other ways, but Automatix did the trick for me.

2. Feisty on my laptop recognized the wireless card & router out of the box.  The Network Manager (mine is KNetworkManager) works great.  I don't have a printer installed right now, but I haven't read about people having problems.

3. There's an application called WINE (WINE is Not an Emulator) that can run Windows programs.  However, it's still .9.X, so it's not 100% stable.  If you absolutely need to use Windows programs, just dual-boot and boot to Windows when you absolutely have to.
