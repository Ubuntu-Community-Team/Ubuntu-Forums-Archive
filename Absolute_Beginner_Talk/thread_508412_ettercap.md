### Post by benx009 on 2007-07-24
install the **ettercap-gtk** package from synaptic (this will uninstall the regular **ettercap** package, but that's ok).  then enter a terminal and as root do "**ettercap -G**"  (case is important). this should give you a gui to use w/ ettercap (rather than using text mode).
