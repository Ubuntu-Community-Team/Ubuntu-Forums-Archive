### Post by Tweedicus on 2007-04-22
I have a Belkin Wifi Card using the Ralink RT61 chip. It has never worked on any previous version of Ubuntu, nor any other Linux distro. A lot of people are reporting the same problem -- it's quite a common wifi card chip.

I upgraded to Feisty from Edgy and saw the green light go on on the card, got excited, only to see it go off again and it not work. I could not get it to work on Feisty 7.04.

However, I've just installed Feisty 7.04 as a clean install from the CD iso and the install, inc. the Live CD, has recognised the card straight away and made it work. Thanks to the new wireless manager in 7.04 I can see all the wifi networks in my street. Connecting to them is as simple as entering the pass key.

So, it seems this particular card will not work when upgrading to 7.04 via Software Update, but it does work when you do a fresh CD install. 

For those of you with problems with wifi it might be worth doing a fresh CD install to see what happens.

Tweed.
