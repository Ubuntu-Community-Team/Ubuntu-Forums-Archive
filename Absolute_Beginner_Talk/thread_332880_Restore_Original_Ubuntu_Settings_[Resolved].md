### Post by Praxicoide on 2007-01-06
I found a way!

I opened Mozilla, went to File--->New, and created a new folder which matched the name of the custom evolution folder.

I then used nautilus to find and open the original folder, copied everything, and pasted it in the new one (which was blank).

You can do this using the teminal too, of course.
