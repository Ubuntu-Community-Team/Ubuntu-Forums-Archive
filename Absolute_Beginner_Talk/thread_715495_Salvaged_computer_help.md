### Post by JoshuaRL on 2008-03-04
I would suggest you install Ubuntu Gutsy Gibbon (7.10).  I don't believe that 5.10 is supported anymore.

P4 should be just fine, and you should be able to install and run with 256mb of RAM.  But I would suggest you upgrade to 512mb.  It will run much better.

If you install fresh, we can show you how to add software sources without having to manually edit the sources.list.

### Post by DrMega on 2008-03-05
I've successfully had 7.10 working on a machine with much lower spec than yours, so you should be OK. Just don't enable any good desktop effects unless you upgrade the memory and graphics card.

If you try it and performance is a bit poor, then as others have suggested install Xubuntu over the top. There is a package that will enable you to do this without having to reinstall called 'xubuntu-desktop'.
