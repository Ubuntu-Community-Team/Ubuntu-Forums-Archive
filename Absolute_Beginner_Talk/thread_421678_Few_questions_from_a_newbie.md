### Post by jordanmthomas on 2007-04-24
For beryl, make sure you have the following things set and the window list should work like you want:

Horizontal Virtual Size:  however many virtual desktops you want
Vertical Virtual Size:  1
Number of Desktops:  1

and right click on your window list part of the panel and go to properties.  Make sure it's set to show windows from current workspace and not from all.

That should fix things for you.
