### Post by aJayRoo on 2007-09-18
Personally I would just use a gnome panel as a dock in a 2D desktop. Create a new one, open up its properties screen and select the height you would like and un-check expand. you can also set the panel to autohide. You can choose a nice background for it or make it transparent too. Than all you need to do is add launchers to the panel, you can either drag them there from your applications menu or use the 'Add to Panel' option on the right click menu. I actually find this to be a good solution as it is very simple and doesn't really go wrong since it is part of Gnome! Hope that helps.
