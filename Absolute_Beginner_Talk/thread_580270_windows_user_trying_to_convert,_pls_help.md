### Post by Hobo Joe on 2007-10-18
1. Yes, dual boot is the best option. VMware is ok, but dual-booting is a much better way to do it.

2. Yes, you can run it through WINE, I've done it before, and I got pretty good performance. About 10 or 20% less than in Windows, but I don't that was because of WINE, but because of the Linux nVidia drivers.

### Post by jpietari on 2007-10-18
#1
I use Ubuntu Gutsy on my PC at home and via Vmware on my work PC.  I recommend you partition your HD and do a dual boot.  Running Ubuntu in a virtual machine is just not responsive enough.  Virtual machines aren't responsive enough for desktop hardware (unless you have a very powerful computer).

### Post by Dr Small on 2007-10-18
1.) You can dual-boot with Windows and Gutsy, and this can be selected when you run installation. At bootup, GRUB will load and you can choose between Windows or Gutsy.

2.) I have no idea about CounterStrike, as I am not a big gamer myself, but you could try it on wine and tweak it to get it to work, or just run it on Windows since you most likely to dual-boot. Running games over VMware or VirtualBox is just not that fast.
