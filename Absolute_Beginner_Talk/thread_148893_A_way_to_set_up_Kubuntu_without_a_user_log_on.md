### Post by C_P on 2006-03-22
I would like to set up Kubuntu to load without having to log in with a user and password because I am the only one using my system. Is there a way to do this so each time it loads, I do not have to log in?

---

### Post by aysiu on 2006-03-22
Yes.

Press Alt-F2 and type ```
kdesu kcontrol
```

Then go to System Administration > Login Manager > Convenience > Enable Auto-Login.
