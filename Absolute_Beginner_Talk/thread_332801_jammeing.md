Most mayour linux distributions, such as Ubuntu,Fedora, and so on, come pretty much with the same applications. Ubuntu is not targeted to developers so much as to end user. Fedora and Debian are. Packaging is anther difference. In Ubuntu you can use .deb packages and install them really easy. In fedora you use .rpm packages and i do not rally know how to install those.
