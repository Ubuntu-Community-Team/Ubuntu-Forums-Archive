### Post by Coinnach on 2006-10-20
Have you already physically connected the hard drive in the machine?

If you haven't then you will need to ensure it is on the same ribbon cable as the primary hard drive. You will need to set the jumpers on the hard drive to either cable select or slave.

You should then be able to check in your BIOS to see if you can see the new HD - if you can then you should be able to continue and I assume you should be able to use a disk manager application in Ubuntu to format it. I am completely new to Linux so I am sorry but I can't help you with the Logical side of the install.

### Post by Bartender on 2006-11-03
If it was me I'd recycle the 9GB and just replace it with a new one.  A 750GB Seagate would be nice ;)   
220W ain't much, which is OK if you don't start plugging in extra drives, the fanciest AGP video card you can find, etc.  Not much point in working that PSU any harder by installing a second drive, and 9GB doesn't do much for you anyway.
If you buy a retail HDD it'll have an 80-wire data cable included.  It's a good idea to avoid the old 40-pin cables.  80-pin gives you the option of simply plugging 2 drives onto the cable and setting the jumpers on both to "Cable Select".  The default first device will be the one on the end of the cable, not in the middle, but you can go into BIOS and tell it to boot from the middle one if you want.
