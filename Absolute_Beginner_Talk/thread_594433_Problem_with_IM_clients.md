### Post by genbuntu on 2007-10-28
1) did u install any extra plugins like auto-away ... ?

2) try using a webclient like 'meebo' etc , to make sure its not an internet/server problem.

3) try logging into two of your accounts and message yourself both ways. (I mean for e.g. if you have 2 gmail accounts , log one from pidgin and other from gmail, add each other to your buddy list and try messaging from gmail to pidgin and vice-versa). Do this with meebo also?

4) If all else fails, how about making a clean fresh install of pidgin .

5) If still no success, then the last thing I can think of is to check using some messenger in windows (if you have it) or run  yahoo/msn msngr thru wine and see?

Good luck!
