### Post by Anonii on 2007-06-16
Yes, you can. You will have to install the .deb of it, tho, which can be found here: [http://packages.ubuntu.com/](http://packages.ubuntu.com/)
The problem with that, is that Add/Remove (Synaptic) will also install the nessecary dependencies, when the .debs wont and you will have to download every dependency from the site I posted (The site is quite user-friendly, so you won't have any problems.) (You will also have to download the dependencies of the dependencies if its nessecary (You understand, I guess.).).
