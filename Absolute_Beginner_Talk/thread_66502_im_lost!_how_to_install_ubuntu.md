### Post by mouldy on 2005-09-17
[QUOTE=rykle777]i just downloaded the new version and i dont know how to install it.doi need to reformat my pc?anybody who is kind enought to teach me how, thanks guys. :???:[/QUOTE]
 Did you download the LiveCD version or the install version? You can only install with the install version - I haven't tried yet because it's still downloading - but you probably just need to boot with it. With my PC, I need to press F11 when my computer is booting to bring up a boot menu, then I choose my CD drive. It might be the same for your PC, on the first screen you see when you turn your PC it'll tell you what key to press for a boot menu. If you don't have that, press the key in your BIOS and mess around with the boot order so your CD drive is before your hard drive.


You shouldn't need to reformat your pc though :p
