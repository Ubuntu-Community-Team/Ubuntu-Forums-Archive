### Post by markyb86 on 2007-12-12
once you get it the way you want it, get some web space. I would recommend 110mb.com. free, no ads, high bandwidth and alot of space.

( I assume you are running dreamweaver from windows xp/vista so i will explain how to upload from windows.)

When you get your account, goto My Computer
type this in the address bar:

```
ftp://username:password@yourdomain.110mb.com/
```      (if you use the site im giving you. if you choose another host tell me and ill assist with it.) Replace all the user name, password and your domain with appropriate info.

Now all you have to do is copy/paste your website into this directory. I even reccomend adding the ftp site as a web folder, that way you can just click on it like any other folder.

To view it on the web, your main page has to be either index.htm or index.html, and you would view it by simply ```
http://yourdomain.110mb.com
```
(or whatever domain you choose)

Hope this helps?
