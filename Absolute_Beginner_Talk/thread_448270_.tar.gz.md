### Post by ha1f on 2007-05-19
a tar file (called a tarball) is like a zip file.  the .gz means it has been compressed with gzip.  you can manpage them both.

if im understanding you correctly, you want to install winrar from the tarball you downloaded.  in that case, your going to want to untar it, then cd into the created directory.  then you just run ./configure && make && make install.  you can use new lines instead of &&'s.  that will install the application for you.  though, id be suprised if ubuntu didnt have rar in its repositories.
