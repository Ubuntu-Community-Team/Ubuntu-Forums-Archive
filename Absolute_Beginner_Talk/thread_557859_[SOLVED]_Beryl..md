Okay, as some of you know, I managed to get Beryl working with an ATI Radeon Xpress 1100.

What I'd like to know is, the default window manager Metacity, is there a way to change it so the default manager is Beryl?

Thanks

### Post by ayenack on 2007-09-23
Do this.

**System>Preferences>Sessions**

Then.

**New**

For name.

**Beryl**

For command.

**beryl-manager**

This will start beryl at boot so re-boot. Then right click on the beryl icon on the panel. Then.

**Select Window Manager**

**Beryl**

That should do it. ayenack.
