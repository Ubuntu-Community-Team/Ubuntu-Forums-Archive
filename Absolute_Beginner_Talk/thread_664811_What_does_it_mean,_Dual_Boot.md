### Post by kellemes on 2008-01-11
> **Redptc said:**
> What do people mean by a dual boot? My computer fires up and, after a few seconds, I get an option to select Ubuntu or Windows. Is this 'Dual Boot'? 

Yes, indeed..
If you want both os's (or more) to function at the best of there ability, this is the way to go..

> Or does 'Dual Boot' mean two operating systems running at once? Do you toggle from one system to the other?

This you can do with a VirtualMachine, software like VMWare, Qemu or VirtualBox will do this for you. You run one or more os's in a window, just like any other "program".
Cons.. you are working in a "virtual"-machine, this includes "virtual" devices, may not be offering the best of performance. Also.. running 2 os's at the same time demands enough resources to be able to handle it, memory etc..
