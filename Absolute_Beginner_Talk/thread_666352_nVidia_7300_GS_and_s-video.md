Hi and if you are using the restricted drivers for the graphics card then you can use the nvidia-settings to use your tv as a monitor  ```
gksudo nvidia-settings
``` It will be located under the x server display configuration. This option will use twinview. As for watching tv and use the computer at the same time I don't think this is possible.
