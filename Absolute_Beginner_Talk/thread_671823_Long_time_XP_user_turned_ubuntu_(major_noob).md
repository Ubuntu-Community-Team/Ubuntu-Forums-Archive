### Post by rzrgenesys187 on 2008-01-18
Welcome to Ubuntu.  I apologize as I am probably not the best source on emulation.  Anyways, the way to install wine is to either go to the Applications menu and select Add/Remove Programs and search for wine that way.  Another way would be to use synaptic package manger through the terminal with the code:
```
sudo apt-get install wine
```
With wine installed you should just be able to run the executable for the program you want to install.  Your best bet however would be to google wine + the program you are looking to install as there is probably a how to somewhere online.  I hope this helps, let me know if I can give you a hand any other way.

### Post by JoshuaRL on 2008-01-19
Just a couple ideas, although I have no personal experience with either Adobe Apps or Steam.  Try Gimp for Photoshop, and I believe Steam does in fact work with WINE, but I may be wrong.  I would always suggest installing the newest version of WINE instead of the one in the repositories, since that is an older one.  The newer version will have more dlls and more apps supported.  To install from them, follow this:

[http://www.winehq.org/site/download-deb](http://www.winehq.org/site/download-deb)

Also, check whether your apps are supported.  Although some that aren't on there just happen to be.  But that's a good starting point:

[http://appdb.winehq.org/](http://appdb.winehq.org/)

Hope you find what you need.
