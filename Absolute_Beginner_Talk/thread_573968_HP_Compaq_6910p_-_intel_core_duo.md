### Post by frogotronic on 2007-11-18
Hello,

Yes, Gutsy Gibbon installs cleanly on this machine with no problems at all.  There are even drivers for the thumb reader that you will have to manually install (google).

I don't know about Debian, generally Ubuntu is far more along than Debian (thus Debian is more stable).

****Currently there is a problem with Suspend and Hibernate using the new ATI drivers on GG.  They will be patched in about a month.  If you need Suspend and Hibernate, good idea to intall Fiesty Fawn and then manually install the ATI binary (proprietary drivers) not the ones from the restricted drivers & then upgrade to Gutsy once the ATI drivers have been patched.*****
