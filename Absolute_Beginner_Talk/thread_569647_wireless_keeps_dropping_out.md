### Post by EtniesBMX on 2007-10-11
I have the same problem using a Netgear wg111 wireless USB adapter. When I plugged it in, Ubuntu automatically used the prism54 driver, which is a Linux driver. I disabled it and used the Windows driver, and things have been better, but the internet still occasionally goes out on me. So maybe trying the appropriate windows driver (by using ndiswrapper) would help/solve your problem.
