### Post by neurostu on 2008-02-27
It probably has to do with your graphics card. Try booting in "Safe Graphics Mode" or something like that it should be the 2nd or 3rd option on the disc.

The live CD doesn't contain all the drivers for all the various video cards. If you have a advanced video card (nvidio, ati) it probably isn't supported by the CD.  Once you install though, ubuntu will find the correct driver and give you the option to install it.

### Post by Rocket2DMn on 2008-02-27
If you are unable to get the LiveCD to install even in safe graphics mode, use the alternate cd which is a text based installer without a live session, but it is very intuitive and easy to use.  Get it at [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) by checking the box at the bottom for alternate disc.  As neurostu said, the graphics problem is easily fixed once you install (if it doesn't go away automatically).
