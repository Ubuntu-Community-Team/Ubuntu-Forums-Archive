### Post by kebes on 2007-01-08
If you want a WYSIWYG editor (like FrontPage or Dreamweaver) then "Nvu" is a similar program that runs on Linux.

If you want to do more complicated web development, and know how to code in html/php/etc. then you may want to look at installing "Quanta Plus" which gives you a very nice interface to the webpage code. It also gives you a quick preview mode very similar to what you would get with a WYSIWYG editor.

### Post by kebes on 2007-01-08
In Ubuntu, the easiest way to get the correct version that works with Ubuntu is to install using the built-in "package manager." If you're using Ubuntu, there is a program called "Synaptic" that lets you do this. Just search in it for "nvu" and then click "install" and it will take care of the rest.

(If you prefer the commandline you can go "sudo aptitude install nvu".)
