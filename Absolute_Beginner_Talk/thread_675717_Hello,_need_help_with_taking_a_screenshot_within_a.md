Hi there,
Unfortunately in a lot of games, keyboard shortcuts do not work, such as alt+tab and printscreen. The only way to take a screenshot, might be to use the screenshot utility within the game itself, so check options and keyboard sequences within the game, to find the keys to press to take a screenshot.
