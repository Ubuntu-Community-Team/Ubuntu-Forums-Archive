### Post by aidanr on 2006-12-23
mplayer and totem play dvd iso's 
as far as i remember so does kaffeine
a quick google pulls up this handy tip

[quote=googled]After 
inserting the file I can right-click on an ISO, Open with..., choose Kaffeine,
and have the movie start playing.
-------------------------
# this file is "/usr/share/applications/kde/kaffeine-iso.desktop"
[Desktop Entry]
Exec=kaffeine dvd:%f
Icon=kaffeine
MimeType=application/x-iso
Name=kaffeine
Name[en_US]=kaffeine
StartupNotify=true
Terminal=false
Type=Application
--------------------------[/quote]
