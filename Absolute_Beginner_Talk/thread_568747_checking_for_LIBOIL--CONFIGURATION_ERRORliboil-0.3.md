### Post by Maxtorm on 2007-10-06
The current repositories contain liboil-0.3.10. To install, open the Synaptic Package Manager (System -> Administration). Click Search and type in "liboil". It should return liboil0.3 and liboil0.3-dev. Depending on what your ultimate goal is, you may need the liboil03.-dev package as well. Right-click on each and "Mark for Installation" then click the Apply button.
