I installed KDE, and whilst it seems quite good I personally prefer GNOME, but now I'm back using GNOME again I can't help but miss two of KDE's features:

1. The use of the Win Key "Super" in combination with another key to launch something, eg. WinKey + T to launch a terminal.

### Post by laidback on 2007-04-06
For qu 2)
Right click on any file with the extension needed, e.g. my_file.pdf for pdf files. Then select Properties> Open With, use the radio button to chose the application default. That should set the selected software to be the default whenever you double click on any pdf file in future. Repeat as needed for any other extension.
