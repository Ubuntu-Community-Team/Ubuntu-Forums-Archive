### Post by Paqman on 2008-02-12
> **Beaker Boy said:**
> Now I'm fully connected, I seem to have a plethora of downloads being offered up!

Have you just recently installed? If so there may well be a lot of updates available.

The only updates the system will try and download are ones that are relevant to your machine. Have you installed any KDE apps? (such as Amarok?) If so, you'll have KDE components installed, and that's why they're asking to be updated.

In general, if bandwidth isn't an issue you might as well go and download everything. You can always roll back to an earlier version through Synaptic if you need to.
