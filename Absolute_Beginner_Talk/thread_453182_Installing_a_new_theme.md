### Post by Simran on 2007-05-24
System > Preferences > Theme

Once the theme window is open, just drag the *.tar.gz into the window and it will install it

Then just select it from the list.

Job done :)

### Post by NilsE on 2007-05-24
It looks like it is an icon theme so you will have to go into themes then "customize" to select it once it is installed.  

Sometimes with icon themes it will not install with the drag and drop (rarely) so you can extract it to either usr/share/icons or /home/username/.icons where user name is the name you log in with.
