### Post by smartboyathome on 2008-01-10
If you weren't able to do ctrl+alt+backspace, then you wouldn't be able to do Alt+Tab either. It is the same keyboard shortcut in ubuntu as it is in Windows.

### Post by Vixis on 2008-01-10
