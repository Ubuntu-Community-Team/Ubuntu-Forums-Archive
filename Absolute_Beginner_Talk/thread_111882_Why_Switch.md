### Post by mostwanted on 2006-01-03
You seem interested in free software based on those applications you mentioned, wouldn't joining the free software community be a good incentive to switch? It certainly has been for me.

Also, development on Linux is so fast, every couple of months something new amazing happens in the software world, you just don't get that excitement on Windows IMO.

Other good reasons to switch:

* Xterms
* A nice desktop environment like Gnome or any of the other very different window managers and desktop environments on Linux
* Beagle, maturing faster than ever at the moment
* A nice unified multimedia backend so you don't need 3 different media players to play the most casual formats
* Better themes :P
* Apt-get and synaptic for easy installation
* Better versions of Gtk software like Inkscape, GIMP or Gaim like you mentioned (they're all Gtk+ programs)
* Loads more

### Post by akniss on 2006-01-03
[QUOTE=thezookeeper]Thanks all for your replies.  I'm still curious though -- Is ubuntu any faster than windows xp?  Are there any real speed comparisons out there?[/QUOTE]

I guess that would depend on how you define speed... GIMP loads much faster on Ubuntu than on XP, probably because it is GTK.  Openoffice loads (a little) faster on Ubuntu than in XP.  The only *real* computing I do is statistical analysis using R, which I can detect no differences between Ubuntu and XP.  So 'speed' is probably a relative thing, and not sure how one woule go about testing it.  

I only started using Ubuntu a couple months ago.  I had tried mandrake, and didn't really like KDE.  Not knowing much about linux in general, I assumed that it was Linux I didn't like, not KDE.  I then heard about Ubuntu, and tried the LiveCD.  I thought GNOME had a really nice layout.  So I tried the dual boot thing, and started playing around with Ubuntu.  Before long I had moved Ubuntu to the default OS in GRUB, and now I only boot into Windows once per day to sync my pocket PC.  I started out just 'playing' with Ubuntu, since I like to play around with computers.  I now find myself annoyed when I'm in windows.  I've become much more productive with simple things like multiple workspaces, and lots of configurable keyboard shortcuts.  

So its up to you.  There's no real 'need' for me to use Ubuntu rather than XP.  I was happy with XP, and just tried this linux thing out of sheer curiosity, but now I prefer it hands down.  Just play with it.  If you like it, great.  If not, no harm done.  If you need help, come back here... I'm constantly amazed at the Ubuntu community and their willingness to help new folks like me!!
