### Post by Joeb454 on 2008-04-08
Search either Add/Remove or Synaptic for unzip as **hyper_ch** suggested, or if you fancy tackling the CLI, open up a terminal, and type (or copy!) ```
sudo aptitude install unzip
```

It'll ask for your user password, so you type that in, but be aware that it won't show that anything is being input...even though it is going in :)
