### Post by suRoot on 2006-02-25
Support for certain audio & video types is not enabled by default in Ubuntu.  This is generally due to the fact that some of these formats don't comply with the GPL (not "free" software) & would therefore taint Ubuntu.

Anyway, easiest thing to do is download & run Automatix - a script which will install & configure all this stuff for you, plus a lot more.

Here's a thread that explains where to get it & how to use it:

[http://ubuntuforums.org/showthread.php?t=66563&highlight=automatix](http://ubuntuforums.org/showthread.php?t=66563&highlight=automatix)
