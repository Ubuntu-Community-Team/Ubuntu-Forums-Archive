### Post by Angus77 on 2008-02-21
System > Administration > Language Support

Check the languages you want to use, then next time you log in, you can choose to switch languages for the current session.

I don't think yo can do it on the fly, though.  I think you have to log out, switch languages, and then log in again each time you want to switch language environments.
