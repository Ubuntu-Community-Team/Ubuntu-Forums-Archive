### Post by rosegarden78 on 2008-01-25
You can use Compiz Fusion with Ubuntu GG 7.10 to get a 3x3 viewport.  Desktop icons from your Gconf editor still appear on all viewports.  But you can stretch windows like GIMP and Opera across a 2x3 viewport 2048X2048.  I have instructions [here]("http://ubuntuforums.org/showthread.php?t=677959") and [here]("http://ubuntuforums.org/showthread.php?t=385981") if you can't get Compiz working.
