porsher - If you want to change the icon of Amarok in your applications menu, click on Applications-------scroll down and select -----System Tools -------Applications Menu Editor (click on it) ----------on the left window you'll see the list of apps -----look for Sound and Video then click on it ----------the right window will show all apps that is on this menu -----look for Amarok and double click on it -------a new window will open and there you can click on the icon to change it. Just make sure you know where the location of icon you want to is.
