### Post by pppetter on 2007-04-11
Hi there!

With 192mb ram I would also recommend downloading the alternate cd, instead of loading ubuntu it will run a text-based installer instead.

About what version to use...
I always recommend using the latest release, due to the simple fact that for every release, Ubuntu just gets better and better. The latest stable release is edgy(6.10). However now there is a beta release of the next Ubuntu version, Feisty (7.04), which you could also try.

Here you can find all Ubuntu releases, and the Alternate CD, which I recommend:
[http://releases.ubuntu.com/](http://releases.ubuntu.com/)


Of course, you could try Xubuntu, which is a bit more light-weight than Ubuntu, and therefore might feel a bit more responsive. Just remember that it's quite different in looks, and there's another set of standard applications.
You'll find Xubuntu downloads [Here]("http://www.xubuntu.com/get"). Just remember to download the alternate CD instead.
