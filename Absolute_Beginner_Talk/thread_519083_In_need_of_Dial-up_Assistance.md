### Post by Bartender on 2007-08-07
> **revolutionaction said:**
> Perhaps there is a more simplfied way of connecting via dial up??


 Well, yeah, there is.  Get an external hardware serial modem.  Used is fine.  ebay or craigslist or garage sales...
The US Robotics modems are the most reliable.  Check the little switches on the back.  3, 5, and 8 should be "down".
There are some USB modems that work but I have no experience with them.  if you can try some for free that'd be one thing but I wouldn't buy one unless I knew it was gonna work.
The Amigo serial modem, sold at newegg and other places, is the cheapest new external I've seen that appears to work with Linux no problems.

The Modem How-To is a bit confusing.  After you find a modem that works, check this [thread]("http://ubuntuforums.org/showthread.php?t=480717"), see if it helps some.  The info there will help you with the software part, but not the hardware part.  If you have an internal winmodem that doesn't work well with Linux, and you find all that linmodems/scanmodem/etc/ stuff too confusing, the first step is getting a hardware modem that just works.

Also check out the Zoom 3095.  I don't know if this works, but Zoom claims it does.  A bit expensive but it's so cool
