### Post by Kobalt on 2007-03-19
nVidia and ATI cards both have drivers (official and opensource ones) that are functional in Ubuntu. 
The linux Terminal (the console) is the best one you'll find out there.
About your game pad, and maybe gaming, concerns : I would recommend you to keep Windows in parallel of Ubuntu to keep your games running on windows. It should be simpler than installing wine, configuring everything, and it will in most case be faster.
It's very easy to do, here is some doc on how to do that : 
[http://www.psychocats.net/ubuntu/iso](http://www.psychocats.net/ubuntu/iso)
[http://www.psychocats.net/ubuntu/installing](http://www.psychocats.net/ubuntu/installing)

### Post by russell.h on 2007-03-19
Yes, if you plan to play games I would keep a windows partition. Thats what I did, although I hardly ever boot it anymore. After using Ubuntu for a few months trying to use windows is excruciatingly painful. Operations that are simple in Ubuntu are slow and tedious in windows. Of course when I first began using Linux it was the other way around, so give it time.

There is a database which keeps track of how well various programs work under wine. It won't load currently, but I think the address is [http://appdb.winehq.org/](http://appdb.winehq.org/)
