### Post by NilsE on 2007-05-05
Not sure about the no password fuction but you can go into the top panel and select System/Administration/Login window then under the "local" tab you can pick one of the login menus that has list in it.  With that you get to pick the user by clicking on the name then entering a password for that user.

You can also add a picture under the users tab called a "face" but you still have to click on the name and not the picture.

### Post by zvacet on 2007-05-05
You can solve it by going to the system>administration>login window>security and choose automatic login for user(your son in this case).He will still have user name and password but he will not have to type it because he will login automaticly.
