### Post by Sinkingships7 on 2008-01-18
middle click only works on the desktop, so be sure that's where you're clicking. 

System -> Preferences -> Advanced Desktop Effects Manager.

go there, and be sure you have the following enabled:

under accessibility:  Enhanced Zoom Desktop
                             Negative

under Desktop:       Desktop Cube
                             Expo
                             Rotate Cube

under Effects:         Animations
                             Cube Reflections
                             

and as long as you haven't touched anything else there, that should do it for you.
