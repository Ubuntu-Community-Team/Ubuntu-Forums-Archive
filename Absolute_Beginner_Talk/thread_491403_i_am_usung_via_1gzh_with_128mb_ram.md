### Post by Sunforge on 2007-07-03
There is an alternative method which is called the "alternative install CD". 

You can find it in the same place you got the Live CD from, check the box with the text "Check here if you need the alternate desktop CD. This CD does not include the Live CD, instead it uses a text-based installer." next to it.

NB: As the text says this is a straight forward installation CD, so be sure that you're sure if you know what I mean.
