### Post by bodhi.zazen on 2007-09-25
Linux = the kernel : [http://www.kernel.org/](http://www.kernel.org/)

GNU = (c) compiler and additional basic programs : [http://www.gnu.org/](http://www.gnu.org/)

Distro = Linux kernel + GNU + additional (default) applications + repository of additional apps + forums/support.
