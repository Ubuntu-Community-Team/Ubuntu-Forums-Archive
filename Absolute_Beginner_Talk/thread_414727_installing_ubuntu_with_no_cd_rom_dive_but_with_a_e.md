### Post by TURNER on 2007-04-20
> could i download the instaler to the externall hdd and install it to the .. laptop

I will presume that the laptop currently has a version of windows installed? If so you can simply download the ISO, and mount it from the external HD (using a tool such as VCD), a quick search of google provides the following for VCD/DVD:

[GOOGLE VCD SEARCH]("http://www.softpedia.com/get/CD-DVD-Tools/Virtual-CD-DVD-Rom/")

I believe that M$ also provide a free version from their site.

> and can i do a dual boot .. i already have windows installed .. in the laptop

Sure no problem, during the Ubuntu install it should detect the Windows install, and work around it by creating a partition in available space....Windows shouldn't be affected by this process.
