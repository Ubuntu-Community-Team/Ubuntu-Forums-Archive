### Post by decoherence on 2006-09-24
in a file browser window, press F9 to get the side-pane if you don't already have it. Click the "Places" button and select "Tree."

---

### Post by decoherence on 2006-09-24
oh, also in a file browser window select the menu item Edit>Preferences click the "Behavior" tab and make sure "Always open in browser windows" is checked. i forget what the default is in ubuntu 6.06.
