### Post by confused57 on 2006-07-30
The Xubuntu Desktop LiveCD won't run with so little ram...you could probably install using the Dapper Xubuntu alternate CD, but you wouldn't know if your hardware is compatible.
I'd definitely backup any data you don't want to lose.
You can always do a server install from the alternate cd, then(if you have an internet connection configured):

[https://help.ubuntu.com/community/Installation/LowMemorySystems](https://help.ubuntu.com/community/Installation/LowMemorySystems)

also, you may want to consider Ubuntu Lite(again, internet connection needed)

There's also Damn Small Linux.
