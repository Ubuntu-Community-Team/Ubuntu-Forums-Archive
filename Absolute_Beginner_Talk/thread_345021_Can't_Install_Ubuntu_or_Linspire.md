### Post by wpshooter on 2007-01-23
Did you burn your Ubuntu CD at 4X or less ?

Did you check the integrity of your Ubuntu CD by running the verification function that is found on the initial Ubuntu boot screen menu ?

Did you try the video safe mode parameter ?

Have you taken XP completely off of the computer ?

If you have means of getting XP back on the computer if you really need to, then you might try getting the **KILLDISK** program and **WIPE** your drive completely clean before you attempt installing Ubuntu. ** Sometimes** this can make a difference.

If all of the above does not allow you to get Ubuntu installed then try downloading and burning the ALTERNATE (text based) CD instead of the DESKTOP (live) CD.

I assume your are trying to install either Dapper or Edgy and not Feisty which is still in development.  For now I prefer and recommend that you stick with Dapper.

Good luck, let us know how it goes.
