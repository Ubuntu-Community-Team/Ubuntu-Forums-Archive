### Post by Trebaruna on 2007-07-22
Hello Simon,

The thing is DVDs do not work like removable drives: they need to be 'finished' as it is called, to be able to be normally accessed. This means that in order to write more to it, you need to erase the whole disc, and start again.

It is possible to make a multi session disc, whereby you finish each section, and then add new ones, but I am not sure how.

The simplest way would probably be -if the data isnt too large- to copy all the files to the hard disk, add the new ones and burn it to DVD again.
