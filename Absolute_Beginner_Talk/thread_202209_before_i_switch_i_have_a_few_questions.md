### Post by aysiu on 2006-06-23
[QUOTE=Stromham]
1.i have a wireless card, its a linksys wirless notebook G card and i want to know if i can install it and use it with ubuntu.[/quote] I don't know, but if you use the Desktop CD, which runs a live Ubuntu session from the CD and your computer's RAM, you can test out and see how well it detects your wireless. You can also consult this: [https://help.ubuntu.com/community/WifiDocs/WirelessCardsSupported](https://help.ubuntu.com/community/WifiDocs/WirelessCardsSupported)

> 
2.will my computer be faster or slower than windows? is there a big differnce? I think the general consensus is that boot time will be longer (1.5 to 3 times as slow) but that general desktop functioning will be equally as fast if not faster. It also depends greatly on how many services you run and what desktop environment you use. If you use Xubuntu (instead of Kubuntu or Ubuntu), it will almost definitely be faster than Windows.

> 
3.i need the computer for webdesign and programming and other computer arts, my dad just needs to use the email and he wants to look at adult sites (please be mature), will we both be able to do what we want? Yes! That's what Ubuntu's made for--safe web browsing and email and programming and website design.

> 
4.i am very much into computers and computer science, but using windows all my life, will ubuntu confuse me and my windows like thinking? how bad is the learning curve? You may have to do some command-line stuff in the beginning to get it set up properly. And even if you don't *have* to use the command-line most people on these forums will give you commands instead of saying "Click this and point here and click that." If that's a problem, I'd advise using a more point-and-click distro like Mepis, PCLinuxOS, or SuSE.

> 
5.will we be able to reformat and wipe the drive then install ubuntu? i dont want traces of windows left behind. Why don't you try the live Desktop CD before you commit to erasing Windows? Once you're sure you want to install a dual-boot or a complete overwrite, follow these directions: [http://www.psychocats.net/ubuntu/installing.html](http://www.psychocats.net/ubuntu/installing.html)

> 
6.protection etc. will we be protected? we i have to go looking for good quality firewalls and such? You'll be fine, but there are firewalls or iptables configuration programs like *guarddog* and *firestarter* available in the repositories.

> 
7.is ubuntu worth the time what are its pros and cons? will i have a good amount of programs to put on my computer? Yes. The repositories are full of random programs to suit many needs. >  is programming on ubuntu easier than windows? No idea. I don't program.

> 
8.will ubuntu be able to run on my computer will i need special equipment? It depends on whether your hardware supports Linux or not. If it doesn't... well, you'll need to replace it with Linux-compatible hardware... or recompile your kernel... or install proprietary drivers. Ubuntu could work like magic... or it could suck up your life trying to get it to work. It all depends on your hardware. I lucked out, and Ubuntu detected everything except my monitor's optimal resolution, but that was an easy fix.

> 
9. will i be able to customize my user interface with backgrounds and such? Yes, and not just the backgrounds--much, much more.

> 
10. is there alot of resources and books on ubuntu? will i be able to figure this all out? There are a lot of resources but only one book so far. The book is not terrible, especially for absolute beginners, but I would recommend online resources instead. These forums are top-notch for support.

### Post by Brunellus on 2006-06-23
Answers in detail:

1)  Tough to say, unless you tell us the exact model and chipset.  One thing you should learn:  never, ever believe the marketing name of a device.  Ever.  Only the model number and its specifications are really useful, but then, they're not sexy.  There are a number of Linksys cards that work "out of the box" in Ubuntu;  check to see if your card is listed as working here:

[https://help.ubuntu.com/community/WifiDocs/WirelessCardsSupported#head-751ff8f98030df97347576e45ed468055b1e3fe3](https://help.ubuntu.com/community/WifiDocs/WirelessCardsSupported#head-751ff8f98030df97347576e45ed468055b1e3fe3)

2)  The machine will likely not run much slower.  If your computer was running a lot of spy/ad/malware in the past...well...none of that exists on Ubuntu, so the system overheads will be lower.  Some users report better performance, but it's tough to tell without specific benchmarks.

3)  Yes.  very yes, unless his porn sites are ActiveX only, or your developing work requires you to use Visual Basic or Visual Studio.  As you might expect, there's no Microsoft software for Linux....

4)  Repeat this after me:  LINUX IS NOT WINDOWS.  WINDOWS IS NOT LINUX.  They do things differently, and each has a certain internal consistency.  Whatever you think you know about how things work or should work in Windows...well...that's not really all that relevant to Linux.  After a while, you get used to things.  I myself have begun to think of things more in Linux/Unix terms lately, and find certain Windows conventions to be odd. 

5) You will be pleased to note that the default option in the install formats your hard drive and installs ubuntu.  Back everything up if you really do intend to make this a permanent move.  The option to boot both operating systems exits, though, so consider your options carefully, especially if you have critical tasks that can only be accomplished with software that runs in Windows.

6)  Unless you do things that are stupid, you are far far safer on Linux than on Windows.  Ports are closed unless you open them (rather than the reverse being the case, as on Windows).  Firewalling at the level of the individual host is possible using iptables and its graphical frontend, Firestarter.  But security in general is good, again, provided you do nothing stupid.

7)  This is a broad question, and only you can answer it fully.  I think Ubuntu is worth it.  It's my preferred operating system;  it has almost all the software I could possibly want (and a bit more, besides).  The Universe and Multiverse repositories contain literally thousands of programs, and I have discovered a number of excellent ones.  But your needs will be different.

Also, the extent to which Ubuntu is "worth it" depends largely on how well-supported your hardware is.  

As far as programming;  I'm not a coder, so I don't know.  But once you install the build-essential package on Ubuntu, you have a full set of development tools right there-- compilers, debuggers, and so forth.  The GNU project couldn't live without 'em.

8)  That depends on what your hardware is.  Again, Ubuntu "just works" on a lot of hardware.  

9)  The user interface is customizable to a degree that Windows users could only dream of.  Never mind backgrounds and themes--you can change the whole interface and desktop environment. Have a look at the "Desktops" thread in the Community Cafe area and see what many forum users are able to do.

10)  You are only limited by your capacity to learn, and your capacity to connect information.  I would recommend a good, non-specific book on Linux in general.  My favorite is *How Linux Works:  What Every Superuser Should Know* By Brian Ward (San Francisco, No Starch Press 2004).  Admittedly not a "for dummies" book, but it has good, solid information, doesn't insult your intelligence, and encourages you to get "under the hood."  As far as other resources go, well, you're looking at one.  These forums are remarkably good.

Recommendation:  burn a LiveCD and see how you like it.  If you decide you can't live with it, you can pass the LiveCD to someone else.  If you decide you want to take the plunge, we'll be waiting here.
