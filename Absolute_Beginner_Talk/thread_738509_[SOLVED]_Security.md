### Post by Inxsible on 2008-03-28
You were right. The answer is No.

One OS has got nothing to do with the other in a dual boot as only one can run at any given time. They are independent of each other. And there are not many viruses for Linux. So you can be pretty safe while in Linux. When you are in Windows, however, all bets are off :D

You should have some antivirus software installed in Windows to give you some degree of protection.
