### Post by cwmoser on 2007-12-14
I've used vmWare and it works very well.  You will not though be able to run the Joost application in vmWare.

I've successfully tested vmWare running windows 2000, XP, Vista, and Server 2003.  All seem very close to native speed except for Vista and Server 2003.  In addition, I did not find that Ubuntu ran all that responsively in vmWare.

But, I think most will find that Windows 2000 and Windows XP as the more useful uses of vmWare.  I've tested Office 2003, Visual Studio, Quicken, and Quickbooks as running very well on a Ubuntu native OS running vmware Windows XP.
