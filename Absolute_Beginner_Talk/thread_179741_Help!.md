### Post by Jagot on 2006-05-20
[QUOTE=Zepplinne]is there any way to make the screen brighter that can be done with one of the ubuntu functions, or is there an ubuntu/linux program I can download?[/QUOTE]

You can use xgamma from the terminal.

```
xgamma -gamma 1.0
```

1.0 is the default value - you can increase the value to make it brighter, decrease to make darker.
