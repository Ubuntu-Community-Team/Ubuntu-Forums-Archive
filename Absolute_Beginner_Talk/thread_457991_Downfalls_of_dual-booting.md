### Post by Shay Andrews on 2007-05-29
Hey. I'm a newbie to Linux, and think Ubuntu is the right distro for me. However, I want to keep Windows for my Lexmark printer and a few other programs. I have a supergeek friend who is willing to partition my hard drive for me and set up a dual-boot (he's done it many times), but before I commit to anything, I wanted to know, are there any possible cons of this? Slower performance, etc.?

---

### Post by Inxsible on 2007-05-29
The two OS'es will be completely independent of each other. Only one can run at a time, so even if you have performance issues, they wont be because of dual boot. Only because of your hardware.

Dual boots are preferred because you can have access to your machine even if one or the other OS crashes. Other adantages being you can run your programs natively instead on using emulators or WINE etc.

Windows may still be required for a few proprietary games that you might own or some specific software that you own. Most others have Linux counterparts, or work in both environments 

Hope that helps !

### Post by LaRoza on 2007-05-29
Pros:
1. More choices
2. Two OSs, one computer
3. Easy to share information
4. Both OSs are just as fast as they normally are

Cons:
1. Less disk space, only an issue if you have a really small disk
2. You have to restart to use the other OS
