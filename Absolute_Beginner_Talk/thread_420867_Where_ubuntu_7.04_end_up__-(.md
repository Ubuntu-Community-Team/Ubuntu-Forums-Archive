### Post by digital_sabotage on 2007-04-24
i'm assuming you're on windows since you're using ImgBurn ...try this ...i might not be exact but close
 ...go to "Start" button in windows ...select "Search" ...search window pops up ...in the left pane select "All files and folders"  ...make sure "Look in... Local harddrives" ...and search for > ubuntu-7.04-desktop-i386.iso
this should find it if you have it ...and show you what folder it is in ...then run ImgBurn again and browse to that folder ...great program for burning .iso files in windows ...ubuntu does with simple right click on file and write to disk
