if you want to be completely sure, physically break the hard drive and burn the pieces. if you don't need to be that sure, you can use Gparted from a liveCD to delete all the partitions. then if you install windows or whatever after that, only a very experienced data recovery technician would be able to find anything from the previous system. using Boot and Nuke would probably be slightly safer than Gparted, as Gparted doesn't actually destroy any data.
