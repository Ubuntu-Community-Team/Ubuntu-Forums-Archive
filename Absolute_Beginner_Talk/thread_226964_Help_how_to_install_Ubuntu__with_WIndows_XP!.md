### Post by meney on 2006-08-01
Ubuntu can see and write to FAT32. I have mine set up with windows on one side, uduntu on the other, and a FAT32 share in the middle. Now if your just installing Ubuntu as is, bare bones, you won't need more than 4 GB. Now if actually want to have some room, I would give about 10-15 GB. Windows is a pig and requires 8 GB for the base system, but Ubuntu is nice on space (you could run it on a flash drive ;) ).
