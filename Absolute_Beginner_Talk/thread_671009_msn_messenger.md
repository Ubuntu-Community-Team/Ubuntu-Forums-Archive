### Post by Sef on 2008-01-18
> how can i install msn messenger to linux

With WINE, you could run it, but not sure what version.

Alternatives: Pidgin, which is the default im manager (Applications > Internet > Pidgin), or download amsn from the repositories.  Applications > Add/Remove > set top right box to 'All Available Applications' > search: amsn > click on the box next to amsn > apply changes > apply > close.
