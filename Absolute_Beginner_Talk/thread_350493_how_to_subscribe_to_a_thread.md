### Post by nhandler on 2007-01-31
First, check and see if you are still subscribed to the thread. Go to the thread, and click on Threat Tools. If you are subscribed, it should prompt you to unsubscribe. If you are unsubscribed, it should prompt you to subscribe.

If you are unsubscribed, simply subscribe again. If you are subscribed, and you aren't getting notified by email of new replies, Try unsubscribing and subscribing again.
