### Post by dca on 2008-02-08
You may be able to try these guides on setting up file sharing on WinXP
 
[http://www.wellesley.edu/Computing/FileSharing/Windows/facstaffxp.html](http://www.wellesley.edu/Computing/FileSharing/Windows/facstaffxp.html)
[http://www.sophos.com/support/knowledgebase/article/11738.html](http://www.sophos.com/support/knowledgebase/article/11738.html)
 
 
Once it's enabled, you should be able to click Places on the menu bar and 'Connect to Server...'.  From the drop-down, select 'Windows Share'.  Server should be the IP address of the XP box with no slashes or anything.  Share would be something like c$, I don't think you need anything in folder area unless you config XP that way, user name is the name of the Windows user or Windows profile, and then click 'Connect' button.  You should see an icon on your Ubuntu desktop after that...
