### Post by wastrel on 2006-11-26
You can change the backspace key to "back" instead of "page up" by editing a config in firefox

put ```
about:config
``` in the firefox address bar.

Put "backspace" in the filter bar, and change the value (double-click it) to 0 (zero).

Dunno about the rest, good luck :]
