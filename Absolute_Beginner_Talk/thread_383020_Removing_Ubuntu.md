Right. Well, in previous versions of windows (i.e. XP) you could put your XP setup disk in, go to the recovery console, type FIXMBR, and that would totally nuke grub, and replace it with the XP bootloader.

Now, I presume you have vista from what you have typed, so, [and i have read this, and seen good reviews, but its not guarenteed to work, as i have not used it], but you can do what I said, with an XP disk, if you have one to hand... and then stick your vista disk in. That has an option to repair an existing instillation.

Now, vista cant repair grub directly, but it can repair the XP bootloader.

So, if you follow that, that should work.

If not, you can always re-install vista...
