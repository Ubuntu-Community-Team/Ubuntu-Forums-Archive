### Post by ramjet_1953 on 2007-07-10
This may be a problem with the image.
First, a few questions:

1. Did you burn the image as an iso9660 image, not just an ordinary data CD? Most CD burning programs have this option in one of the drop-down menus.

2. What speed did you burn the image at? Anything faster than 4 X for any iso image is asking for problems.

3. Did you use high quality media? Cheap media often causes problems for iso images.

4. Did you make sure that your downloaded image is OK by verifying the checksum?
