### Post by Kvark on 2005-09-10
[QUOTE=nikita12]what if i want to use windows again what should i do?
i have mandrake 10.0 how will i install it? :([/QUOTE]
If you want to keep your files on the hard disk then...

Before replacing Ubuntu with Windows you could shrink your Ubuntu parition and put in a fat32 partition to put your files on, that way you can keep them.

Before replacing Ubuntu with Mandrake you could make your /home directory into it's own ext3 partition and then tell Mandrake to use that parititon as /home, that way you can keep both your files and most of your settings, bookmarks, etc.

It's prolly easy to find howtos for both situations.
