### Post by Anonii on 2007-06-16
> **SnakeHips said:**
> I was thinking of buying the "dummies guide to linux" or similar. What book would you recommend to get me started. I'm pretty well versed in dos/windows and spent a good few years in support so i'm no stranger to desktops but want to get a handle on basic linux commands etc...
Hello there, sir.

You _have_ to learn about the Linux system architecture. I bet you don't know what /usr, /etc, /usr/bin are for. 
I'd also give a look to: [http://www.linuxcommand.org/learning_the_shell.php](http://www.linuxcommand.org/learning_the_shell.php), since the terminal is really important (You used DOS; you know it.).
I wouldn't suggest reading Ubuntu specific guides; they will narrow your knowledge. But if you want to use Ubuntu, you _must_ read this: [http://www.us.debian.org/doc/user-manuals#apt-howto](http://www.us.debian.org/doc/user-manuals#apt-howto) . Not all of it, at least for now, but at least the basic points since APT is the package manager of Ubuntu (and Debian), and you will use it everytime you have to install something (There are of course graphical tools for that, but I would recommend the use of the terminal for better control, safety and performance)
Check this site out: [http://tldp.org/index.html](http://tldp.org/index.html) It's one of the oldest and most useful sites for linux documentation. Many and awesome guides for advanced and newbie users.
Also, lurk a lot in these forums, and ask any question you have. That way not only you solve them, but benefit others that have the same ones (Of course there is the search function, which should be used.)
Finally, remember that you can access the manual page of each terminal command using the man command. For example, man top.

(I wouldn't suggest that advanced stuff to other people, but since you mentioned that you are experienced with the CLI... I did it.)
