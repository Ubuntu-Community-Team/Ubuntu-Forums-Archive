### Post by IYY on 2006-08-20
> 4. I'm a coder. Good HTML apps? Notepad app?

There are several HTML apps. Notepad apps are even more common, since most Linux coders like to code "the old fashioned way".

Mousepad is a clone of notepad.
gEdit is installed by default, and is quite a bit better: has syntax highlighting, tabs and other cool features.
Many users like vi or Emacs. These editors take some time to learn because of their untraditional interfaces, but once mastered they are incredibly fast and easy to use.

> 6. Can I have mutiple users on the machine? (Like WinXP?)

Unix was build as a multi-user system from the very beginning, unlike Windows which only recently added this feature. 

> 7. Can I logout, and password protect my computer?

You can, and in fact this is the way it acts by default.

> 8. Are there games that have multiplayer, online and LAN?

Certainly. The Quake games are the best examples.

> 10. I have 64MB ram, 6gig HD. Supported? How much space will I have after the installation?

It's lower than the system requirements for Ubuntu. First of all, you'll have to use the alternate CD (no LiveCD functionality, and a modest text-based installer) rather than the default one. Also, you should use Xubuntu instead of Ubuntu. You may also switch to an even lighter window manager like Fluxbox or IceWM after the installation.

### Post by reacocard on 2006-08-20
> I just rechecked the computer, and I actually have 127 RAM
You'll still want to go for Xubuntu. Xubuntu and Ubuntu share the same 'repository' of software, so any app that's avavilable in ubuntu is available in Xubuntu, and vice-versa. You can even have both interfaces installed n the same computer, so you could install Xubuntu, and then try installing Ubuntu's interface in it, although 128MB really isn't enough for Gnome (Ubuntu's interface). Then if Gnome doesn't work, you can always switch back to Xfce (Xubuntu's interface) without needing to reinstall.
