### Post by Hospadar on 2008-03-18
If you want a requirement-light installation, you might want to look into xubuntu (google it) it's an official derivative of ubuntu which runs xfce instead of gnome, it's very similar to use, but is much lighter in terms of hardware requirements.

Depending on how comfortable you are configuring things, and the specifics of your machines hardware, you might also want to try:
Damn Small Linux - it's a debian-based distro (ubuntu is also debian based) designed specifically for low-power machines

Debian - it's what ubuntu is based off, the only reason I suggest it is that it comes with less stuff installed by default, so it's not as end-user friendly, but it also will run lighter by default.  They have a version that runs xfce as well, which is probably what you would want to use
