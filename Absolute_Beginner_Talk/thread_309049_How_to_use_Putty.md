### Post by celloandy on 2006-11-29
Putty doesn't have built in file transfer.  PSCP is, indeed, the right tool from the Putty suite to do file transfers, but it's a command-line utility, so you have to open open a Windows command line, and run things from there.  It sounds, though, like you're probably not super-comfortable with command line stuff, so I'd suggest Googling for WinSCP, instead.  It's a full, Windows GUI SCP client, and it's fairly intuitive.
