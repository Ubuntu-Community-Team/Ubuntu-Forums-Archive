Good choice. Extra RAM really helps system performances, at least in my experiences...

Installing KDE is as easy as 'sudo apt-get install kubuntu-desktop' or using Synaptic. The first time you log in with KDE, it'll give you a slider for the effects. Just set it somewhere near the middle and you'll be fine.

Good luck!

Oh, and I must say, even if Ubuntu wouldn't load from the CD, you can install it by installing the 'ubuntu-desktop' package with apt or Synaptic.
