### Post by aquavitae on 2007-08-07
What exactly is the problem?  I don't know much about ubuntu ultimate, but from what I've heard its quite bloated, and not the best idea for a beginner (I reserve the right to be wrong :))  These are the step's I'd follow; tell me where you're going wrong:
1. Download and burn the ubuntu cd from the ubuntu website (for a very old computer, e.g. a P3 I'd get xubuntu cos its lighter on resources)
2. Boot of the cd.  At the boot menu, choose Check disk for defects.
3. Once you're sure the disk is ok, boot off it to make sure it will see all the hardware
4. Backup important files from windows.
5. Install ubuntu (you have the option during the install to overwrite windows completely or to keep it - its your choice.)
