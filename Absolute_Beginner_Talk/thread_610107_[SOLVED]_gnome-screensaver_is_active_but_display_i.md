### Post by unebaguettesvp on 2007-11-12
fixed it! add this to your xorg.conf:

```
Section "ServerFlags"
	Option "blank time" "0"
	Option "standby time" "0"
	Option "suspend time" "0"
	Option "off time" "0"
	Option "screen blank" "0"
EndSection
```

and restart X. <CONTROL><ALT><BACKSPACE>

found this here:

[http://ubuntuforums.org/showthread.php?t=341617]("http://ubuntuforums.org/showthread.php?t=341617")
