### Post by aysiu on 2006-09-24
> **Josh1 said:**
> And where is thememanager, everytime i try and import it into themes i get an error. :(
I don't think you're reading what we're trying to tell you.

A standard theme or icon theme is a .tar.gz within which is a theme configuration file and possibly some icons in various folders.

What you have is a *pack*--not a *theme*. It's a .tar.gz full of other .tar.gz files. Inside those .tar.gz files are random icons. No configuration file--no organization.

If you don't know what I'm talking about, compare the contents of that pack to [this one](http://www.gnome-look.org/content/show.php?content=32146) (a valid icon theme).
