### Post by Patrick K on 2007-03-28
WMware's VMServer is a free virtual machine. Their site has lots of info. VirtualBox is another choice. There are others out there. A quick google on virtual machines should turn up a number of options. With these you have to install Windows to the VM. VMware has a conversion program that can virtualize an existing Windows install and run it in Linux. Very handy if you have a complex Win install and want to keep what you have. 

Wine is a different approach to running Windows apps. [www.winehq.com](www.winehq.com) However, checking the apps DB for Publisher shows it's a no go for wine..
