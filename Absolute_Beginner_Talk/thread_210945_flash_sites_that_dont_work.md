### Post by deanlinkous on 2006-07-08
This will get you past where it checks for the version, it does not help with the content. So if it truly requires newer features then you are still out of luck.
------------------------------------------
This is what I do on debian etch
install firefox, download flash, unpack flash, install to /usr/lib/firefox

Open the file ~/.mozilla/firefox/pluginreg.dat
change 
Shockwave Flash 7.0 r63:$
to
Shockwave Flash 9.0 r63:$
