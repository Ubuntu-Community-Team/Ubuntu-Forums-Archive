### Post by anthonyellis82 on 2007-02-07
I've got the T22 and it's set up to dual boot to Ubuntu and to slackware. I got mine off of ebay with no OS installed which is why I went with Linux, I wanted to learn it and it's only cost is time. Mine has a 20 GB drive and 256 MB RAM and it runs beautifully. Slower then my main computer, but that's a P4, so I don't expect it to keep up.

I've personally never had an issue with the video, just graphics load slower because it's a weak chipset. Not a big deal, I don't use it for new 3d games. Because of this (and probably the RAM), the LiveCD that you load Ubuntu off of runs incredibly slow. Don't get scared off by that, once I had it installed I was very please overall. 

I think my only real frustration with it is because I haven't figured out how to get it to run DVD video from the DVD-ROM / CD-RW drive, but I'll figure that out with some more time I'm sure. 

I'd say keep WinXP on there and dual boot to Ubuntu, I think if I remember the installer correctly, you can do it easily in there. Section off a few gig, like 5 at least, for Ubuntu, and about a gig for the swap file if you are looking to just play around with it. That's all really easy to do with the installer. If you like it you can either take windows off or just expand the Linux partition to get more space. 

I'm not familiar with the apps you are talking about, that's why dual booting may be ideal. You can test out the linux equivalents, or you can use Windows for those and Linux for everything else.

Hope that helps, if and when you install Ubuntu, there are many things you can do to speed up the boot time and things like that. I found all of that in these forums.
