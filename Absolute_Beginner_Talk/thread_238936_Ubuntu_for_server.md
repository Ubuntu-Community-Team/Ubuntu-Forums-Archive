### Post by dca on 2006-08-18
This isn't gonna' help you at all but...  Fedora is a huge install, one DVD or a crap-load of CDs.  Granted it comes with everything you could ever need as far as setting up anything (PC w/ raid or honest to goodness file server) you could possibly need on a server.  The install gives you options if you want to go all out w/ Apache and everything else.  However, just starting out, I would probably go w/ Ubuntu 6.06.1 LTS.  I'm starting to roll out a few dummy servers (file dumps & file sharing devices over a network basically) using the Ubuntu 6.06 server install which is Ubuntu but w/o the desktop gui...
