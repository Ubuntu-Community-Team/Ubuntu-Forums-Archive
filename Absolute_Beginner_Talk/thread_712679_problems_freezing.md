I had this same problem with the BIOS chip getting hot. The machine would shut down. I think if it were memory boards that were loose you might get erratic crashes but the machine migh not turn off. On my machine, the tell tale sign that it might be heat is that after the machine had a chance to cool off, it ran ok but then in time it would begin dying. At the end, I guess when the heat sink had separated from the BIOS chip, it would run for only about 15 minutes or so. That is why I suggest that you check all heat sinks. If you have a way of monitoring correctly the temps of the various points on your board, it might give you a clue. Unfortunately on my board, the BIOS did not have temp monitor and I was running Windows XP.
