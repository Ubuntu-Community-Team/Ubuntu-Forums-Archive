### Post by Bill_58503 on 2008-01-26
Just a thought...

I had similar issues with a much more powerful system (AMD 64, 3 gb ram, nvidia 8600 gt). After starting the install from the first screen,  live CD my monitor would go blank.  Perhaps you monitor/video card does not support the resolution the installer or X is  trying to run by default. This was not an issue in 7.04 on the same system?

You say "after the splash screen"? After the CD boots and you get the install or start screen, assuming you get that screen,  try different settings for the VGA (F4 option bottom of screen). I had to set mine to at least 800X600 for it to work?:)
