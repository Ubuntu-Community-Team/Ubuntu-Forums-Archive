### Post by drinkingmouse on 2006-05-05
Let me share with you some of my feeling between windows and linux.

I used windows since WIN95, followed by WIN98 up toWINXP.
My first try in linux was RedHat9, then changed to Mandrake 10.0 up to Mandriva 2006, which is the version that I am using now. I also tried Fedora Core 1, Debian Woody and Sarge, and will going to try Ubuntu in the coming June.

I would say, linux is not windows, if you expect everything to be the same as windows, then I advise to use windows directly, that's no point to change to linux in this sense.

Linux is a completely different system compare with windows. Windows uses FAT or NTFS, while Linux uses EXT3 and some other formats. Windows stores the system in one partition, while linux probably uses many partitions.

In the sense of the working as Desktop, both windows and linux can do the same.

You asked about which distro is the best. I would say, you better try some of them to find out which one you like most.
Mandriva 2006 will be quite good, easy to install, easy to use, stable enough and the softwares included in the CDs are quite new, but you may need to install something by yourself, e.g. java in firefox (instead of using the auto installing plugin in the firefox), this is because some softwares and plugins are "enterprised" (and this is the only thing I feel dissatifactory).
FC comes from RedHat, but I don't like it personally, because I think that it is used to provide a testing platform for the RedHat enterprised.
Debian Sarge probably is the most "difficult" one, even in the way of installation. For newbie in Linux, I don't suggest to try it out.
I don't know much about Ubuntu, and now trying to grasp more information about that before I really get a try in the LTS version coming in June. However, I suggest that, if you can wait, try the 6.06 version directly instead of trying the 5.10 version, as it is only a month left for the release of 6.06, and 5.10 only have about 12 months support left.

Hope this help.

### Post by nudnik on 2006-05-05
I had the same feelings as the last poster regarding Fedora. Although I beleive it is a fine OS, it is indeed a test bed for Red Hat which brings along undesirable corporate entanglement. The corporate culture rears its ugly head rather quickly when you set about customising Fedora and run into needless roadblocks.

Ubuntu is a pure product in this regard, thoroughly open throughout its design. Once you learn the basics, you find it is infinitely customisable. This is made possible for the begginer by excellent resources such as this forum. Not all distros are as open and supportive. 

Since my initial education, I have come to prefer the Gnome enviroment which Ubuntu offers as default. Once you master the general architecture of Ubuntu Linux, Gnome is easier to manipulate, in my opinion.

Being relatively new to Linux, I am only now, after countless hours of use, (relative to my countless hours with Windows) starting to notice the benefits of Ubuntu's stability as I boot between the two daily. It seems to me at least anyway, that data integrity and stability are much improved with Linux. Some people have declared me a nutter, but I see fewer "hiccups" in the data stream when it is processed with Ubuntu. 

I am currently using the most recent version of Dapper Beta. I prefer it to Breezy because it is more compatible with my PC, a fairly recent build. 

I reccomend Ubuntu for three reasons: 

1. Community support.
2. Infinitely customisable once you get the hang of things via the community.
3. Stability
