### Post by drowner on 2007-06-18
Welcome

I assume the disks you received were the LiveCDs. The beauty of these is that you can run them and see if everything works before you go ahead and install without damaging your system.

Ubuntu and Kubuntu share the same basic 'guts' as it were, they just use a different desktop manager, Ubuntu uses the gnome desktop manager, kubuntu uses the KDE. Both have their fans. Asking which is better will only spark a war ;) They have a slightly different 'feel' and look, (but they both can be completely customised). They come with slightly different applications, but you can install any applications you like, so its not a big deal. 

try them both out, and choose your favourite

To boot the LiveCD (and then install, if you choose) you need to make sure your computer BIOS is set to boot from the CD drive, rather than the Harddrive first. Forgive me if you already know this, but the computer BIOS has an order of things to boot, eg floppy then CD then Hard drive. it will check them all in order.

Put the ubuntu cd in the drive, and reboot. If it doesnt load ubuntu, it means its not set to boot from CD before Hard drive.

you need to check that your BIOS boots from CD first. To do this you will need to press a 'magic button' as soon as your computer turns on. Its normally Delete, or esc, or f1 or f12, or f2, just tryhitting a few ;). Sometimes your computer tells you which it is.

Then  you can find the settings for the boot order, and make sure it says Cd before hard drive, put the CD in, and reboot
