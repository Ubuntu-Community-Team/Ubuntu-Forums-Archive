### Post by wolfen69 on 2008-04-20
first go to system>admin>software sources and make sure everything is checked off. you could also go to the third party tab and check that off too. (not source code)
close and reload. then go to system>admin>synaptic package manager and search for wine.

### Post by wormser on 2008-04-20
Make sure you have the repositories open.  System> Administration> Software Sources: enable Main, Universe, Multiverse and uncheck CDrom.

System> Administration> Synaptic Package Manager: search for wine or use the command:

```
sudo apt-get install wine
```
