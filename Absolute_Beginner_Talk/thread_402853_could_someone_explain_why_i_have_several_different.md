### Post by Maestro23 on 2007-04-06
> **trent dillman said:**
> OK, Automatix just has stuff not in the repos. Adept, Synaptic, aptitude, and apt-get all use the repos, and therefore should have the same programs available.

And if the repo's don't have a program you want, you go to that program's website and download the program.  It might be a tarball, .rpm, or .deb  It will come with a README/INSTALL doc for instructions on installing.  Be advised that when installing from source(tarball), you may run into many dependency errors which must be resolved before the program will install correctly.
