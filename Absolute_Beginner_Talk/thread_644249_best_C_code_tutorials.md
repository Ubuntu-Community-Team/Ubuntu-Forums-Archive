### Post by rich.bradshaw on 2007-12-20
[http://www.physics.drexel.edu/courses/CompPhys/General/C_basics/c_tutorial.html](http://www.physics.drexel.edu/courses/CompPhys/General/C_basics/c_tutorial.html) 

seems pretty good.

To compile code in Ubuntu you need to

```
sudo apt-get install build-essential
```

first. This won't be mentioned in most tutorials.

To compile programs:

```
cc program.c -o programname
```


Where program.c is your source code, and programname is what you want it to be called.

To run the program, go to the directory where it is stored and type

```
./programname
```

This will run it.

Again, that likely won't be in many tutorials.
