### Post by Bakerman on 2007-06-05
Hi,

There is some bluetooth software that you can install from Synaptic:

bluez-utils & gnome-bluetooth (plus any libs that need to be downloaded with these two packages)

My bluetooth dongle works ok, allowing me to receive photos from my nokia 6230i anyway. I've not tried it with anything else yet.

Not sure if you need both apps though, being new to Linux & gnome, I just installed the most likely looking ones from the list. One of these apps will put a bluetooth file sharing icon into your accessories folder on the main menu.
