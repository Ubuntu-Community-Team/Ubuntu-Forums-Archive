### Post by starcraft.man on 2007-05-26
Uh, simple and short answer is you need windows and not Linux. .exe files do not execute or run in any distros. [WINE]("http://www.winehq.com/") is a free application layer (they hate being called emulator), but only a handful of programs work perfectly (I don't think either yahoo or MSN run that well).

Try aMSN, don't use the repo version, try the newer 0.97 one, its linked to [here.]("http://ubuntuforums.org/showthread.php?t=416062&highlight=new+aMSN") Download the 4 first links, and then scroll down to a poster with two more, download the bottom one. Install them one after another, just double click and put the pass in. They depend on each other so try em all, until you get the one that can be installed.

If you really have to have some of that windows software and don't want to try [alternatives listed here]("http://linuxappfinder.com/"), then (not trying to be mean) go back to windows. There isn't any emulator that runs all of their apps perfectly. The bottom line is simply that Ubuntu/Linux is not a cheap/free shell for you to run windows programs, it has its own to be used.
