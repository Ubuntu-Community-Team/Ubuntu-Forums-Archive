### Post by Atomic Dog on 2007-01-04
> **jawknee530 said:**
> how do you cd into a directory? say your desktop so i can run the .exe with wine

click Applications-->Accessories-->Terminal.  Then you will be in your home directory.  Lets say you put the windows install file in a folder called "Halflife" off of your home folder.  You would type "cd halflife" at the terminal.

If you want to drill farther you would type something like "cd halflife/setupdir/bin"
