### Post by urukrama on 2007-08-05
I run a dapper server install one my laptop, with Openbox as its window manager. It is very fast, because it just installs the base and lets you add whatever else you want. If this is your first server install, expect it to take some time, or prepare well (and know what you want to install).

Fluxbox and Openbox are both very light. If you like a more standard Windows feel, try IceWM (though you could accomplish something very similar with Openbox and a panel (perlpanel, fbpanel, lxpanel, xfce4-panel, etc.)

If you are interested in newer packages, go for Feisty. If you do a server install, you won't benefit from all the improvements in gnome in Feisty, but will have the same base and repositories. Hardware (wifi?) recognition might also be differ between Feisty and Dapper.

If you want something stable and LTS, go for dapper. You can also always compile later versions of apps from source in Dapper.
