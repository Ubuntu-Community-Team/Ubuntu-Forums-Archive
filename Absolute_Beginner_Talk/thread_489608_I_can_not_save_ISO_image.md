### Post by majickmann on 2007-07-01
I'm guessing the "system" is the new Ubuntu that replaced MS Windows...

Sometimes it's easier to describe one way to do something (that works), so you can compare it.

I always select the ftp link over an http link when possible.  Whether it is at that point or after I connect to the ftp site, I right click on the file and select "Save link as...".

At the next prompt, I always select save to disk.  

There should never be an issue regarding the name or file extension (e.g. iso).  The "system" doesn't care.

Unless there are "other demons" at work, you should have a perfectly good file.  :-)
