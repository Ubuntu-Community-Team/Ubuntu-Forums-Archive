### Post by ugm6hr on 2007-05-13
To enable Universe and Multiverse on Xubuntu:
1. Applications -> System -> Add/Remove...
2. Select "Preferences"
3. Select "Ubuntu software" tab
4. Ensure all boxes are ticked (apart from "Source code")
5. Select "Close"

The other thing to do is to make sure you are connected to the internet with the computer when trying to add programs.
