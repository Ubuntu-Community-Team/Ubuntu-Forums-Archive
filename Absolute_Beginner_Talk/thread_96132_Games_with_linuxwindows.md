### Post by livingtarget on 2005-11-28
I would estimate only 5% of your games collection will survive.

You'll most likely need/want to buy (yes buy) [cedega](http://www.transgaming.com/).

You can get [wine](http://www.winehq.com/) as well which is the free solution. However don't expect your games to just work. I've heard World Of Warcraft runs quite well on wine, but not much else besides applications.

If you have any games that support linux natively (they are build for linux) then you are much better off.

Games that are known to run natively on linux:
# Most Unreal Tournament games
# Most ID software games, doom3, quake3/4, return to castle wolfenstein

For a large list of free and commercial games that run natively on linux, go to this thread: [http://ubuntuforums.org/showthread.php?t=5153](http://ubuntuforums.org/showthread.php?t=5153)

2. If you are new to partitioning i'd advice to get a seperate harddisk, +-20gigs will do, and install linux on there. I always find messing around with your existing windows partition dodgy.

3. Linux will install grub automatically unless told otherwise, it's easiest to install grub as bootloader. HOWEVER windows might complain about this quite a bit. This is always a risk you should be aware off.

As for system commander i'm not sure.

Hope this helps you out to make a decision. If you are completely new to linux you might want to read this article: [Linux is ready for the desktop--but whose desktop?](http://www.psychocats.net/essays/linuxdesktop.php)

To decide which linux distribution is best for you can take this short quiz which will point you to the best suitable distribution for your needs:
[http://www.zegeniestudios.net/ldc/index.php](http://www.zegeniestudios.net/ldc/index.php)
