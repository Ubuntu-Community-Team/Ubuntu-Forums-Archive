### Post by Scheater5 on 2006-10-15
All very true, but to answer the original question, no, it's not really "the prefered way to install applications."  "The prefered way" is a matter of personal preference.  Automatix is great for getting things up and running because it does things that many people really want but are sometimes difficult to a user new to Linux - or that are really tiresome to do repeatedly every time you do a new install.  Add/Remove programs is a great way to get packaged out of the official repositories easily, and Synaptic is a similar, if more complex, method.  Generally once you get sed to Linux, most people use command-line methods where possible simply because of the speed of it.
