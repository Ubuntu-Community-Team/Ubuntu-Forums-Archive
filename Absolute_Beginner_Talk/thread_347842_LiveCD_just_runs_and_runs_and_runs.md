### Post by Threbus5 on 2007-01-27
Hi,

Add RAM (RAM for the older machines runs about $14 or less per 128 Mb.)

Xubuntu is less resource intensive, however, even Xubuntu requires (per the Xubuntu site specs) 192 Mb RAM in order to install from the live CD. (I tried to install it with 64 Mb and had to add RAM).

One machine that I am running is 333 Mhz with 192 Mb RAM and it works well with full Ubuntu.

If you really do not want to spend any more money, you may be able to use the Xubuntu alternate install with less than 192 Mb - possibly with the 128 that your computer already has. 

One note. I tried an install an a very old and even slower machine with 128 Mb RAM and in frustration interrupted the install. Bad Idea. Take the patience to wait for the machine. I found that installing a spare (faster) CD drive increased the performance significantly.



Take care
