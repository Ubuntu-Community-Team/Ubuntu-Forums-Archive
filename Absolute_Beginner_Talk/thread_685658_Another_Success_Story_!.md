### Post by Presto123 on 2008-02-02
I would agree with you jan, but, it does look a bit like there are some places here that we could help, too.

Wine is fairly easy, just look at it if you were running through your "My Computer" thing on Windows. Another thing is that you can configure it to run as any type of Windows environment via the "Configure Wine" application. Lastly, if Wine Windows Emulator (the defaut) doesn't always work, you can right click, "Open With" and "Custom Launcher". There are several different things in Wine that you can use to launch programs like, wineconsole, winelauncher, etc. I found this useful in 7.04 when I had the lower version of Wine and it actually worked trying different methods. Now, I don't know if it is worth the effort. ;)

For higher resolutions, if you are using a proprietary added-on video device (IE: Nvidia), you would definitely need to get the driver and use this command to play with resolution settings:

```
gksudo nvidia-settings
```

That would be best to set up as a launcher on the desktop. Give it a name (I just simply name it Nvidia) and put that code in the command line.
