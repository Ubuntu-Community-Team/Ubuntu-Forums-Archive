### Post by johnc4510 on 2006-05-06
Any time you have a fix it's good to let others know. As to the virus/firewall issue, you don't really need any virus protection. Ubuntu comes with the firewall firestarter already install and running. You might want to install the graphic interface for configuration. sudo apt-get install firestarter. Then launch and set your preferences.
