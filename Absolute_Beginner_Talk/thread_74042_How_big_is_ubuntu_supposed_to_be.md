### Post by apollo2011 on 2005-10-10
That sounds about normal.  My Windows XP when I got it 4 years ago was 6GB...

One thing you could do, is go into Synaptic, select Settings>Preferences>Files, and select the "Delete Downloaded Packages After Installation".  This will delete the package files after they have been installed.  Once a package has been installed, the package file is not needed anymore.  You can then click the "Delete Cached Package Files" button.  This will delete package files that have already been downloaded and are cached on your system now. I am not sure how much space that will clear up (I don't think you can get much back) but it should help.
