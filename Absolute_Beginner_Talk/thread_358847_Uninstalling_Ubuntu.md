### Post by darrenm on 2007-03-04
It sounds like you have Ubuntu installed on a 2nd hard drive. If so just boot from the XP CD, choose to go to a recovery console and run fixmbr, this will re-write the boot sector with a Windows one.

Then in windows go into disk management and choose to delete the partition on the second drive, re-create it as NTFS, format it and choose a drive letter to use for it.
