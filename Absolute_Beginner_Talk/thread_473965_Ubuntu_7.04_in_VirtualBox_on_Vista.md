You have to install the additions package like any other VM.

1) Open up terminal of any kind and paste this in:

```
sudo aptitude dist-upgrade
```

then

```
sudo aptitude install build-essential
```

If you need a restart do so.

2) When your ready go to the top of the VM and click Devices > Install Guest Additions... Copy and paste the contents to your desktop. Open the terminal again, paste the following in.

```
cd Desktop
```

Then

```
sudo sh ./VBoxLinuxAdditions.run
```

Restart again.

Thats it to getting the additions in, then go to the menu Machine > and the top 3 let your resize the window I believe. I haven't had need but that should do it... any problems post back.

Ok, now a little tutorial for you. See the mode sections I highlighted in blue, this lists the possible resolutions you can have. First one in list (inside the "")is the default that will be used. Depth (there are different modes for each depth) refers to the color (32 bit is what its called in windows, though its really only 24 bit color and 8 bits monochrome). Now, I could edit this to read 1280X1024 which is your max resolution, but this makes a problem. This is the actual size the VM will display at INSIDE the window, that means the window would be actually outside of the borders of your monitor. Not good. So, you actually have to use a size smaller than your max resolution, I did some testing yesterday and "1260x900" fits perfectly  for me. Because I have 2 taskbars (top and bottom I take off 100 pixels from the height).

Now, here is what you want to replace that small section (just the screen section) with.

