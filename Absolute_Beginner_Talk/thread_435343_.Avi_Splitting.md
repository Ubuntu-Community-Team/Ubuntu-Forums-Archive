```
tar -cvM -L 90000 -f file0.tar  -f file1.tar -f file2.tar -f file3.tar
```

to make 4 files with a maximal filesize of 90MB

```
tar -xvM -f file0.tar  -f file1.tar -f file2.tar -f file3.tar
```

to extract them.

winrar and 7zip should be able to handle the volumes under windows.
