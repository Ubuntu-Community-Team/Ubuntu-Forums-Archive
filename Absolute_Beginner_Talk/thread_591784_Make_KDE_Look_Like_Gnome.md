### Post by Happy_Man on 2007-10-25
Well, I'm speaking from the other side of the fence here, or so it seems. I've been a die-hard KDE user since I first used it a couple years ago. If you want to make it more like GNOME, he're a few tips. 

To put a panel on the top of the screen: Right-click the existing panel, and say Add New Panel --> Panel. Then you can edit it settings as you wish. To add applets, it's much the same as in GNOME: Right-click and say add applet to panel. 

Although, I'd much rather prefer that you mess with KDE's settings yourself using Kcontrol and see how you like it. KDE is very powerful and very customizable, and if you use as you would GNOME, you'll only scratch the surface. Please, take the time to delve into the settings for everything. I guarantee you'll find a setup that works for you. If you want, you can start with a GNOME-esque setup, but please experiment from there. You won't regret it. :)
