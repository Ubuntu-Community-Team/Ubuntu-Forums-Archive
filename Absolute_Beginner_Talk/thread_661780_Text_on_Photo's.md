### Post by billgoldberg on 2008-01-08
Right click the image you want to add text to and pick "open with the gimp".

You can find lots of fonts [here]("http://www.dafont.com/").

Go install the, right-click them and "extract here." Then go to your home folder. Press"ctrl+h" and look for .gimp. Double click the folder and past the font in the "fonts" folder. Restart the gimp.

Some basic gimp text tutorials can be found [here.]("http://www.pixel2life.com/tutorials/Gimp/Text_Effects/")

Brushes can be found [here]("http://browse.deviantart.com/resources/applications/gimpbrushes/?order=9&alltime=yes").
You install them the same way as the fonts but then in the "bushes" folder.

Have fun!
