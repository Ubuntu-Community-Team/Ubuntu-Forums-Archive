### Post by bumanie on 2008-02-12
There are a number of options available. There is dvdrip, which I found did a good job, but was rather slow (But I have an older system - ie slow). You could shrink the dvd in something like k9copy and then put the output into devede which outputs to avi (+ other formats). I don't know about all the other encoders, but there are a number of linux encoders and authoring programs. Check them out Tovid, dvdauthor, avidemux, kmediafactory come to mind. I'm not sure of all their functions, but try a few and see which you like best. I personally use devede, due to a simple interface. If the file is within the 4.3gb range, devede should output that to avi as is with no other messing around. Apart from dvdrip, I don't know of any other 1 step programs, but there may be some.
