Open a terminal, Applications -> Accessories -> Terminal, and change to where you have downloaded that <filename>.bin which is usually in Desktop...

```

cd ~/Desktop
chmod 755 <filename>.bin
./<filename>.bin
-or-
sudo ./<filename>.bin

```

It depends. A file that ends .bin is really ambiguous. It could be some binary file to be installed - I understand Google Earth comes as a .bin file - which is what taurus was saying: you need to **run** that kind of file.

It could also be a cd image. It's Nero that insists on making BIN/CUE files, isn't it? In which case, you'd want something like **bchunk** (in the Universe repository) to  convert that file into an .iso. Which you could then mount, view or burn.

Or it could be something else entirely. If you're more specific about what the file is, someone will be able to tell you what to do with it.
