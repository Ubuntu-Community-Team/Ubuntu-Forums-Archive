### Post by slughappy1 on 2007-11-23
Ok, well I found a plugin for rhythmbox. It was on the original, launchpad, website for AWN. There isn't a link on the new website so maybe this will help somebody else. I found a plugin for Banshee so I tried that, but it constantly crashed so I switched back and found the plugin for Rhythmbox. 

1) Unzip this file to your /home/username/.gnome2/rhythmbox/plugins (There will be 2 folders and 2 files)
2) Restart the computer
3) Open Rhythmbox and go to edit -> plugins, and choose Art Display-awn and the Awns Menus for Rhythmbox.

I edited the wiki for awn. The plugin and howto is now at [http://wiki.awn-project.org/index.php?title=RhythmBox](http://wiki.awn-project.org/index.php?title=RhythmBox)
