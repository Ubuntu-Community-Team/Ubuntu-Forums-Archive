Go *Applications => Wine => Configure Wine*. Try changing the Windows version and other settings. It may also help to enable *backports* and *proposed updates* in *System => Administration => Software Sources*. There's also a program called CrossOver that runs some Windows programs that don't work correctly in Wine. Click [here]("http://www.codeweavers.com/compatibility/"). CrossOver costs money.
