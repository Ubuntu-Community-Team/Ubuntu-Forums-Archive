Simply uncompress the downloaded archive. I did this in my home folder. It will create a folder called avast4workstation-1.0.4

You can run the program from there. In the avast4workstation-1.0.4 folder is a "bin" folder containing the executable files "avast", "avastgui", and "avast-update".
