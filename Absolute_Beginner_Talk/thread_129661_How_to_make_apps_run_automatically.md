### Post by nursegirl on 2006-02-14
I'm not in front of my Ubuntu box, but this is what I remember:

-Click System tab in the upper panel. Then Preferences ---> Sessions.
-Click the Startup Programs tab and click the Add button.
-In the startup Command write your program name and click OK.

Hope this helps!
