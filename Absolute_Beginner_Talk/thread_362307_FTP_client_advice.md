### Post by DirtDawg on 2007-02-15
Aye this question has been asked many-a-time. Search for "ftp client" in these forums.

That said, I use Nautilus (the Ubuntu file browser). Go Places>ConnectToServer, then change 'Service Type' to 'Public FTP'. Tadaa!
