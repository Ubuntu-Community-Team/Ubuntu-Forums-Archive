### Post by moberry on 2006-05-07
Pretty much any regular old window manager will do. Windowmaker, Fluxbox, fvwm, etc. Unfortunately all web broswers now need a minimum of 64mb. I had this issue several years ago using linux (back then netscape navigator was standard) Try using the basic mozilla suite, make sure your swap is on, and get ready to wait. 

a 700mhz desktop with 256 ram will go for less than a hundred on ebay :p
