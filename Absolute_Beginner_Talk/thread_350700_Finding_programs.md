### Post by Zuuswa on 2007-01-31
1.) Click on System ->  Administration -> Synaptics Package Manager.

After that, click on the Search button and type in a few words to describe the type of program you want to find.  The results should appear and you can right click on them and select 'mark for installation'.  When you have finished selecting the programs you want, click 'Apply' and you will be all set!

2.) Search for them using step 1 above.  If you cant find one that suits your taste, try enabling more repositories . . . there is also a very good guide [here]("http://monkeyblog.org/ubuntu/installing/") if you find you need a more in depth explanation.

3.)  It's really a matter of taste.  They are differant window managers, or desktop environments.  While there are some differances in performance, they are both equally adequate at getting your work done.  You can try out KDE if you want, simply install the kde desktop base package through synaptic!

If you have any other questions, you should search the forums for answers!  Many others have probably asked the same questions and had the same difficulties, and the community is very large and supporting!

### Post by Stochastic on 2007-01-31
> **Ion101 said:**
> Srry for the noob questions just got Unbuntu today,

1.  Where can u find programs for Unbuntu

Programs can be added and removed through a few different applications:  the easiest is through Applications>Add/Remove Programs but if there aren't enough in there, try System>Administration>Synaptic Package Manager and then go to Settings and choose Repositories.  There you can add more programs to the available list.  Lastly, when you get more experienced check out the command line program apt-get or aptitude these have lots of features that Synaptic doesn't.

> 2.  Can u get developer tools like Visual C# w/ XNA Framework to make games

I'm not a game developer myself but there are MANY developer tools and languages available in Ubuntu.  Do some research and I'm sure you'll find what you're looking for.  But if memory serves correct, Visual C# is a Microsoft program/language so the chance that they've released a linux version is very slim and it's probably closed source so that Linux developers aren't able to adapt it to Ubuntu.

> 3.  What is the difference between Unbuntu and Kubuntu

Ubuntu uses the Gnome desktop environment and Kubuntu uses the KDE environment.  They're very similar under the surface but look very different and use different default programs for some tasks.  There are some pages that compare all the differences so maybe do a google search.
