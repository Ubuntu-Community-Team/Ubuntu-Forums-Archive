### Post by Daveski on 2007-11-26
You can still move your windows by using the Movement key when you click and drag. By default this is the Alt key but can be changed in System -> Preferences -> Windows. Sounds to me like you have a theme with missing window borders. System -> Preferences -> Appearance.
