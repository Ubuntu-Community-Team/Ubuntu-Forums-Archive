### Post by thelocust on 2007-09-06
The loading screen is no big deal. If you want to change it back just enter kubuntu in adept/synaptic and there will be a kubuntu splash package, just uninstall it. You might have to reinstall your ubuntu one. As for media make sure you have totem or whatever other player you want installed. They might of gotten removed when you installed kubuntu. And I would use Automatix2 to make sure you have all of your codecs.
