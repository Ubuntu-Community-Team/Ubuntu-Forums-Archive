### Post by bodhi.zazen on 2007-02-07
Open a terminal

enter :

```
nvidia-settings
```

What version of the nvidia driver are you using (under "X Server Information") ?

You may be able to configure twinview under the "X Server Display configuration"

Hit the configure button, select twinview ...

If you do not have that option or it dies not work, upgrade your nvidia driver from here

Envy : [http://albertomilone.com/nvidia_scripts1.html](http://albertomilone.com/nvidia_scripts1.html)

Use the beta driver ...
