### Post by aberry5555 on 2007-01-11
If you have a CD burner I would simply download the new iso yourself, as you wont want to have to reinstall or upgrade when the time comes to do so. If you're not sure how to properly just go to this website [http://isorecorder.alexfeinman.com/v2.htm](http://isorecorder.alexfeinman.com/v2.htm) and install this program. Then go to the Ubuntu.com site and download the CD Image (.iso file). Once it's finished simply right click on the file, wherever you save it, and there will be an option to burn it to CD :). Probably the reason that CD doesn't work is, unlike the newer distributions, it wasn't really designed for laptops and such, therefore it won't have the proper drivers set up in the Linux Kernel.
