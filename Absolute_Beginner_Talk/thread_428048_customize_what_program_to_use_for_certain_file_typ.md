### Post by bobbybobington on 2007-04-30
Sure it's pretty simple. Go to any file that is the file type in the file browser. Right click on it and select properties. A properties window will pop up and select the "Open With" tab. You can select which program you want to use, or you may need to add it to the list. This can be done with the "add" button. If your program still isn't listed expand the "Use a Custom Command" option and click on the "browse" button. A folder called "bin" will come up, and you can select the program from there. Hope this helps!
