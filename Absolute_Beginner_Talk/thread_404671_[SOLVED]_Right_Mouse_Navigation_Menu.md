### Post by Najand on 2007-04-08
there is a package for that in Universe Repositoried: nautilus-open-terminal
You can install it using:
```

$ sudo apt-get install nautilus-open-terminal

```
After installation you need to restart X using Ctrl+Alt+Backspace
Then Login and then you will find it when you right click.
Note: If you have not already enabled Universe/multiverse repositories do it using System>Administration>software Sources
