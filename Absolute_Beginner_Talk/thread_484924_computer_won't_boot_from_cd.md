### Post by drowner on 2007-06-26
You need to enable your BIOS to boot from CD.

This is normally done by pressing a key as soon as your computer starts up, before it boots Windows.

Sometimes it will tell you what key to press, other computers you have to guess. Its usally F1 or F2 or DEL or F10... try it and you should get  a menu which well have an option for 'boot order'... make sure the CD Drive is BEFORE the hard disk.
