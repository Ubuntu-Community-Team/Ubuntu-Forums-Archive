### Post by finferflu on 2007-03-26
I say aMSN! I think it's the most decent one and most compatible with the original MSN. You can find it in the repositories, but the beta version is much better, you can get it by adding an extra repository:

```
deb http://download.tuxfamily.org/3v1deb edgy 3v1n0
```

for more info, see: [http://3v1n0.tuxfamily.org/blog/lista-repository-sourceslist-ottimizzata-per-ubuntu-kubuntu-linux/](http://3v1n0.tuxfamily.org/blog/lista-repository-sourceslist-ottimizzata-per-ubuntu-kubuntu-linux/)
