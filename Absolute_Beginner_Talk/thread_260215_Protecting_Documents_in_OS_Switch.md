### Post by ajgreeny on 2006-09-18
Is your windows partition ntfs or fat32?  If it is fat32, you are lucky as you will be able to read and write to it with linux, once you have mounted it.  If it's ntfs, you will be able to read but not write (well not with complete confidence, anyway), but you could edit files in linux and save them to your ubuntu partition easily enough.
