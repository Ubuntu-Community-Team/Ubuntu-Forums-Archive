### Post by rapiscan on 2008-02-25
Here is the detailed walkthrough that you're interested in:
[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)

Specifically, on partitioning, you would following the "Automatic" process in the "Resizing Partitions Using the Ubuntu Installer" section of this How To.

Also note that it's recommended to defragment your windows install before resizing a partition, as it will potentially make resizing the partition a lot faster.
