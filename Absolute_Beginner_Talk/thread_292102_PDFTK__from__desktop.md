### Post by chriscando on 2006-11-03
Sure, if you are using gnome, just right-click on the desktop and choose 'Create Launcher...' Add the information, importantly the name and the command you use to run PDFTK.
Another tip if you like to run it from the command line is to add & after the command (eg. firefox &), this will run it in the background and will free up your terminal

### Post by Stex on 2006-11-03
More tips:
You can entirely bypass using the terminal for running commands by using the run application dialog (default: alt+f2 on gnome)

To edit your (gnome) menus you can use a program called alacarte. It is installed by default in Breezy I believe or get it from the package manager in any other ubuntu version.

There is a gui for pdftk called guipdftk. Googling you can find links for downloads. It does not need to be installed. You can create a gnome menu item with alacarte using a link to where you have unpacked it.
