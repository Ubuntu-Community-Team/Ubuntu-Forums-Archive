### Post by nb123 on 2007-09-27
Hi, I have windows on my primary hard disk as well, and use my external USB hard disk for all my linux distros. I'm very comfortable working on the USB hard disk speedwise. Having ubuntu on an internal drive would definitely be faster, but in my opinion if you're just starting to get into linux like me then you're probably not gonna be using ubuntu for everything that you do, so the speed is not really an issue I feel. However, if you like it and want to make the change permanent, then definitely install it on your internal hard disk! But for now, my advice is just install it and get started, don't worry about the speed, unless you have an older computer? what are your computer specs?

### Post by nb123 on 2007-09-27
Oh, ok, so you're moving your internal hard disk to another computer? Then definitely install it as an internal hard disk! Or perhaps partition it so that you can put windows on it as well. The only reason why in my mind you'd want to go for USB installation if you have the option of an internal hard disk installation, is if like me, you'd like to be able to use your linux at different locations. I installed my bootloader on the external hard disk, so traveling isn't an issue, as I can just bring my USB hard disk with me and boot it on any computer (well anything that allows USB booting!). But if this is not what you're looking for then return that disk and get yourself an internal disk!:)
