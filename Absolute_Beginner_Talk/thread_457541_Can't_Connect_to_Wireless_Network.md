### Post by shamreez on 2007-05-29
Is your wireless adaptor drivers properly installed..

do a wsconfig and see if you can see your wireless card set up..

If it is setup then you can go to add remove programs and install a an application to let you connect to your wifi network (There are a few available like wifi-radar...)

If you can see your wifi network there then you can connect to it...

If you still have any problems just try logging to the network using a static ip
