### Post by ArgentSilver on 2007-01-05
Unfortunately, Linux is a little behind the curve on wireless access, especially WPA access. I'm a relative novice at Ubuntu myself, but I do have two systems working with WPA. I never managed to set it up from the live CD. You can try clicking the network manager icon at top right of the screen (on a standard Ubuntu setup - not Kubuntu or Xubuntu). Assuming that your wireless card has native support, you may be lucky. But my path was to manually configure wpa_supplicant and then install Wifi-radar as a gui front-end to start it up.

What I'm hinting at is that you probably won't get WPA working from the live CD, and it can be a thorny project even after hard-disk install. Sorry!
