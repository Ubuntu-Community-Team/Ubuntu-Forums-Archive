### Post by -deadcats on 2006-08-09
The .ISO shows as a Winrar file because your friend has Winrar installed. .ISO is a compressed file, and Winrar is a utility that can compress/decompress those file types.

Ignore Winrar. Burn it to disc as an .ISO image. Don't try to decompress the file. Don't burn it as a data file. If your CD burning software lacks that function, Google for some Windows freeware that can.
