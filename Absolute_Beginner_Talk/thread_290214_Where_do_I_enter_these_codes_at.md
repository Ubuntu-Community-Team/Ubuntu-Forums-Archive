### Post by ciscosurfer on 2006-10-31
You need to place this code in your userChrome.css file and you can locate this file by going to your home directory > username > (hit CTRL-H to see hidden files) > go to your .mozilla directory > go to your firefox directory > enter the directory with all the letters and numbers > go to your chrome directory > modify your userChrome.css file with the code you want placed there.
