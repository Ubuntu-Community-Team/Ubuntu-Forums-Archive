### Post by Gimbo on 2006-01-01
I think pretty much any modern OS would be suitable once set up right.

If you say she uses NO apps (other than a web browser, email client and games), then any differences in the way apps are installed (ie Windows: Google>download .exe>keep clicking "Next"; Ubuntu: Synaptic>search>mark for installation) are irrelevant.

With Ubuntu, there's obviously far fewer security issues to worry about, and if you think she'd have trouble managing anti-virus and anti-spyware apps, it might be the best option.

If her requirements are really as simple as you've described, it doesn't sound like there would be much difference. My advice would be to think about the problems you've tried to help her with over the phone, and decide whether these problems would exist on Ubuntu and whether they'd be easier or harder to solve.

### Post by pbb on 2006-01-01
I've been helping many people, over much greater distances.

I would say any system what is able to run VNC will do, I've used it with people having dial-in connections (56k/s) and it was no problem if you're patient.

Linux has the added advantage that you can login as another user and fix her problems for her.

However, if she is willing to learn a bit about her computer but following along with you solving her problems, then VNC is a great tool.
