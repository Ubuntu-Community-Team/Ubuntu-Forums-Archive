### Post by podunk on 2006-10-03
You most likely need the “non-free codecs” they don't come built in to Ubuntu.

There is a program called Automatix which you can get here:
[http://www.getautomatix.com/](http://www.getautomatix.com/)

Install it, and then look in Applications>System Tools>Automatix. Click it, select what you want to install - it will ask for a password and go to work.

The media codecs are about halfway down the page.

While you have Automatix up – download “streamtuner,” it has just tons of radio stations built in. On my machine it opens a player called xmms – it looks and works just like Winamp.

Enjoy! :-)
