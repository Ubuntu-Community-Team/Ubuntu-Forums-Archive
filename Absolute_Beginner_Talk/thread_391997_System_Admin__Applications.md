### Post by buuntuu! on 2007-03-24
control "alacarte" (accessories > applications), maybe the progs you installed are just untagged there.

if not, from the terminal try:
```
apropos name_of_the_programme
man name_of_the_programme
```
to find out about the commands to start the programmes you installed
if you want to start them without using commands, just create starters with alacarte.
have fun
