Generally speaking there should be instructions somewhere on the boot-order screen, or a button to press to bring up a help screen.

One thing you could try is to highlight the device you wish to move then hit PgUp or PgDown to move it.

### Post by plucky on 2007-08-07
Try Page Up and Page down buttons to change the sequence.

1. Leave the floppy first
2. Change HDD0 to Cd
3 Change Cd to HDD0

This should then try to boot the CD if there is a disk loaded.
If no disk loaded, will go on to boot had disk drive

Good Luck
