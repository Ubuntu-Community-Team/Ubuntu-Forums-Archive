Mr. N - I think Metacity was removed (by accident) on your session. Could you please check it out by clicking on Systems ----then Preferences -----then Sessions (just above the Sound) --- select it ---then click on the Current Session Folder ----- look if the Matecity is on that list. If not, then add it by selecting the Startup Programs folder ----- click on Add and then type metacity and place order as 20.
