### Post by janipewter on 2007-08-13
You'll most likely find that your CD burning software has a "create disc from image" feature. This will allow you to select the ubuntu ISO (image file) which you downloaded, and burn it to a disc.

Remember though - you'll need a 700MB CD as it will not fit on a 650MB CD.

Once you've got the image burned to CD, start up the machine you wish to install it on and make sure the CD is in the drive. You should receive a message saying "To  boot from CD press any key.." at which point you should press any key (no, not the "any key). If you don't get this, and it doesn't boot from the CD, hit the reset button and enter the BIOS to check that the CD drive is the primary boot device.

Hope this helps.
