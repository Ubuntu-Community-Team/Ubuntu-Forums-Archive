### Post by chinocracy on 2007-04-12
Hope you were able to dig up further on how to use an internal modem. But internal modems are difficult to use in Ubuntu, or Linux for that matter. Since Linux started out from Unix, a system used often on networks, I guess it's expected for users to use LAN or broadband rather than dial-up. However, serial external modems work very well in Ubuntu. You might be pressed to get one, if the internal modem does not work well with Ubuntu.
