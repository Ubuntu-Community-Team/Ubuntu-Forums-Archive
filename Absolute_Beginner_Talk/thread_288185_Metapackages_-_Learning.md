### Post by blendmaster on 2006-10-29
If you want an educational Ubuntu system, then you should really get [edubuntu]("http://edubuntu.com/"). It's specifically made for these programs.

However, for installing them in SPM, you should just check the box and it will ask if you want to mark it for installation, which it may also have to install other programs too. Then click apply, and it should be done.

Hope that helps!
