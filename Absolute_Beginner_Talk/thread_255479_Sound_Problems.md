### Post by Najand on 2006-09-11
When using skype the "Skype Testing User" is always helpful. Her User ID is "echo123". You can call her and see if all your sound is working fine or not... 
Well, for the begining, click on:
Tools -> Options -> Sound Devices
And check if in Audio System to use "ALSA" has been selected.
Then in Sound Device Box for both calls and ringing select your sound device name.
Save and then check your settings with "echo123"
