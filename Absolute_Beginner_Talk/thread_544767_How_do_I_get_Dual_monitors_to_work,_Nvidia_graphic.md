Easiest way to install it is with Envy : [http://albertomilone.com/nvidia_scripts1.html](http://albertomilone.com/nvidia_scripts1.html)

### Post by graben3 on 2007-09-12
I searched around the web on forums how to get my 2 video cards (both connected to one screen) working like in windows XP. I finally found something all of you can be happy to find :
displayconfig-gtk

This is a small utility for ubutu that is working on feisty fawn. You can download the binary packages from :
[http://glatzor.de/filesink/displayconfig/feisty](http://glatzor.de/filesink/displayconfig/feisty)

Install the guidance backends package first then the other one.

then, run
sudo displayconfig-gtk
in a ternimal window.

The utility pop up and you can test the configurations you want. I got one configuration working well with this so I saved the location.

Now the program just creates a xorg.conf file but does not write it to the /etc/X11/xorg.conf file.

Just get the configuration file generated at :
/var/lib/displayconfig-gtk/locations

then copy it over your existing /etc/X11/xorg.conf file and hit ctrl+alt+backspace to restart X server.

Everything should now work as the testmode in the utility. IE. if your 2 screens were working in test mode it should now too out of test mode.



Here is the output generated by this program for my xorg.conf file :
I Have a Nvidia FX 5200 and a Nvidia Riva TNT2 64 card. Both are on a LCD display which handles a resolution of 1280x1024
