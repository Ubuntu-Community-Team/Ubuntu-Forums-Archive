### Post by danhm on 2006-02-10
Notepad++ is a Windows application (being open-source has no bearing on the operarting system a program can run on). gedit (in GNOME) or kate (in KDE) are pretty similar to Notepad++, so you might want to give them a try. They are installed by default, and are the default editting programs.


Should you require Notepad++, install WINE and then you'll be able to install Windows applications.
