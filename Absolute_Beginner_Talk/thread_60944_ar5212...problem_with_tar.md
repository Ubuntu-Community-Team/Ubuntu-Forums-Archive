 You must be in the same directory as the file for that command to work. Either "cd ~/Desktop" before to change working directory, or move the file to where you are at, or use "tar xvzf ~/Desktop/sharutils-4.3.80.tar.gz" to unpack it in your current dir.

Note that unpacking archives might sometimes create a lot of files/directories, so you might want to do this in a temporary directpry somewhere instead. =)
