### Post by boudreaujr on 2007-03-29
Never mind.  I finally got things working.  For anyone else with a similar problem try this...  Download the free utility testdisk which is available for many OSs.  Use this program to clear out any and all partition table information and possibly rewrite the MBR if necessary.  I used this to clear out everything and then reinstalled xubuntu and everything worked like a charm.  This will become one of my permanent disk utilities that stays on my systems.

Good luck to all.
