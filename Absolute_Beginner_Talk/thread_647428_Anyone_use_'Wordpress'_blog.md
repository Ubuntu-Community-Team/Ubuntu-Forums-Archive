### Post by mdebusk on 2007-12-22
> **poeticscot said:**
> I used to use 'Blogdesk' when I was with windows - but for some reason - it appears not to be compatible with Ubuntu - hence this request.

You won't find *any* Windows apps that are "compatible" with Linux. Different operating system. Unless you"re talking about running it under Wine.

The two offline blog editors I see most frequently recommended for Linux are [drivel]("http://www.dropline.net/drivel/") and [scribefire]("http://www.scribefire.com/"). I've tried both and prefer the latter. I prefer WordPress' post editor to anything else I've tried, though.
