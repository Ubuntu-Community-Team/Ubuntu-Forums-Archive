### Post by ReaderRat on 2007-07-20
If he swaps PCs to install 7.04, the installer will load drivers for the wrong PC . An ideal .iso burn is at around 2X to 4X (as low as you can go with the burner) . Faster burners sometimes make unusable install disks. 

If you get a good burn (check the disk), and it won't install, do try the Alternate (Text Install) CD.
