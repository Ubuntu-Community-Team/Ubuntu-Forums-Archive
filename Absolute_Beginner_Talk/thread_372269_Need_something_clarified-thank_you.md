### Post by 23meg on 2007-02-28
The latter is the shell root prompt, which says you're running as user "kittyhawk63" on a computer with the same name. The first part is your user name, and the second is the computer name.

/home/*username* is your home directory. In the case of user "kittyhawk63", it would be /home/kittyhawk63. 

The home directory of the active user is also referred to as "~/". Which means in your case, "~/foo" would be the same thing as "/home/kittyhawk/foo".

### Post by 23meg on 2007-02-28
Since / is the uppermost level in the hierarchy, any path that starts with / is right under it. When you click Places / Home Folder in GNOME, the folder displayed is your home folder, in this case "/home/kittyhawk63".
