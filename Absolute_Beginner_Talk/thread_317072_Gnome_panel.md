### Post by muep on 2006-12-14
Kde has a different panel than Gnome, called Kicker. Kicker can be configured to act somewhat like gnome-panel, but I don't think there is an exact clone of Gnome's menu available for it. 

You can use Gnome's panels inside Kde as well. It should work like it does in Gnome, but the performance and integration might lack a bit.

XFCE can use as many panels as you like. Using gnome-panel in XFCE instead of KDE is also a bit more sensible, imho. XFCE is also more lightweight than Gnome, so you might like it.

### Post by goatflyer on 2006-12-15
>In XFCE, is it possible to make more than one menu like in Gnome? As in, a menu for Applications, a menu for System...

Yes.  On a new or existing panel, right click, Add New Item, XFCE Menu, Add, Button Title: mycustommenu, Use Custom Menu File, mymenu.xml

And yes, you'll have to create your own mymenu.xml

And no, it won't automatically update itself when you install new apps, you'll have to manually add a new menu item to your .xml file each time you want something new in the menu.
