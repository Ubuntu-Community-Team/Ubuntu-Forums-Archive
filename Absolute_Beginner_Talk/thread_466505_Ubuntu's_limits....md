### Post by meng on 2007-06-06
It's not that hard, it's just that not everything is done the same way as in Windows.

examples of things that are the same: alt-f4 closes a window, alt-tab cycles through open apps
examples of differences: you can choose to keep a window on top of other windows even if it doesn't have focus, mouse scrolling works even if a window doesn't have focus

Some users don't like learning new things; Ubuntu certainly isn't an identical interface, so you'd have to learn some new tricks.

### Post by Feba on 2007-06-07
In general, every modern OS with a GUI has the same basic feel, you click on a menu to open apps, you press one of a set of buttons on the task bar to manipulate the window, etc. Honestly it's easier for a windows user to use Ubuntu than it is OSX

Ubuntu is not limited though, in fact it is much more capable than XP because of the way it is made. Open Source software can be manipulated any way you like it,

### Post by ramjet_1953 on 2007-06-08
I think that we need to define a few of things here.

1. Linux the operating system is capable of anything that any other OS can do. In fact, the vast majority of super-computers use it as an OS.

2. However, if you have a particular application that runs under Windows, that application may not have a direct alternative in Linux, or even OsX, for that matter.

3.Nevertheless, most users are very happy with Linux and the applications that are available. Certain self styled "Power Users" may complain that Open Office doesn't have all of the macro capability of Microsoft Office, but they would be less than 1% of users. Most people would never notice any difference.

4. One area that definitely does need some improvement for professional users is that there is no real Photoshop equivalent. However, for most of us, who just want to spruce up a few photos that we stuffed-up, GIMP does the job well. Again, this is not Linux, it is just that Adobe hasn't seen fit to port it to Linux.
