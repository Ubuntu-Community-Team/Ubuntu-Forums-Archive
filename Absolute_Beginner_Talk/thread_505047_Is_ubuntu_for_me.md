### Post by deadlikeoscar on 2007-07-19
Yes it should be fine. First you will have to install all of the restricted multimedia codecs and stuff. There is a sticky here that is good. Do you use torrents? If so, use Deluge. It is a lot like uTorrent, works with almost ALL private trackers (unless they have specifically banned it do to a lack of information--i.e. BitMe) and is great all around. The link to the newest version is in my signature. When you download movies if they are .iso when unzipped, right click and select "Write to Disc." Problem solved. If they say .img, rename them to .iso and do the above step. If they are nrg. there is a program that will convert them--not sure why people use nrg format. Honestly, Linux is a little lacking in coverting avi to DVD format. I installed ConvertXtoDVD through Wine (think Windows emulator but it actually isn't) and it works great. I don't think it will burn automatically but you can do that will either K3B (which is amazing) or install Nero for Linux and use a Windows product key. DVD Shrink and DVD Decrypter work well in Wine as well or you can use K9Copy to shrink with if you want a native Linux app. Really, if you look hard enough there is a program for every need and at worst you can find a Windows one to fill the gap if need be. I prefer not to use Windows apps but sometimes it makes the job easier. There are guides to run uTorrent in Wine as well if you would be more comfortable but really Deluge is great and gets even better with every release.

Also, Ubuntu may be ugly to you now but once you customize it with things like transparency, new themes, icons and Compiz-Fusion--Vista users will be jealous of your OS. Trust me, I know first hand.
