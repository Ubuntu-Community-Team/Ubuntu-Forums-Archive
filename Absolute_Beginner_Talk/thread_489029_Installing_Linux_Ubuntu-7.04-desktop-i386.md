Pop a CD in, pick a low speed, finalize the disk and let it burn. Thats it, I hope thats the right directions. Haven't opened Nero in some months.

### Post by ubun2noob on 2007-06-30
If you are dual booting, it is much easier if you install Windows OS from oldest to newest OS and then anything like Ubuntu or other OS after.  You can use the GRUB bootloader instead of Windows boot manager, but make sure you use / as location and bootable.

Look for the option in nero for burning an.iso. It also may say "Burn CD Image".

I think it is

File --> Burn Image

Ah - here it is -[ look here]("http://www.weethet.nl/english/cdrw_usingnero_iso.php").

Just remember to burn it slow - like 4X or slower. The data will be better at a slow burn. And use good CD's where you can.
