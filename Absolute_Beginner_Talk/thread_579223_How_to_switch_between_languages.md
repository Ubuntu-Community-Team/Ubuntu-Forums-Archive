### Post by solikos on 2007-10-26
You can create a shortcut in order to switch languages directly.Go to:

-System->Preferences->keyboard->layou options->Group shift/Lock behaviour

Mark your preferable choice.Be careful!choose an option which mentions 'change group' and not 'switch group'.For example  a 'Caps Lock key changes group',if you want Caps lock to be your shortcut.This does really work!!!:)
