### Post by Mustard on 2006-01-21
[QUOTE=olddave]Mustard the link was great on line now do you no how to download packages in ubuntu.I think it is done in the terminal window.
olddave.:razz:[/QUOTE]
For most of your software needs you will use Synaptic Package Manager which is in your System>>Administration menu.  Open Synaptic put in your user password when it asks you to, then read this guide below to enable the Universe and Multiverse repositories....

[https://wiki.ubuntu.com/AddingRepositoriesHowto](https://wiki.ubuntu.com/AddingRepositoriesHowto)

Here is a HOW TO on using Synaptic too...

[https://wiki.ubuntu.com/SynapticHowto](https://wiki.ubuntu.com/SynapticHowto)

Once you have those 'extra' repositories enabled, you can use the search functions in Synaptic to find different packages to install.  You can search by  package name or description or both.  I find searching by description to be the easiest method (sometimes name and description).

Synaptic is a front end gui for Apt-Get.  You can do all the things you do in Synaptic via the command line.  The main thing to remember is that you should have Synaptic closed when you are using apt-get commands in the command line, as you can't run both together.

It's possible to install KDE as well as Gnome.  I would get a bit settled into finding your way around the machine before you get to adventurous with your installation goals. :)  There is a bit of stuff to do at the start in terms of getting mp3s and movies playing.  I would concentrate on that at the start.  For movies and music I would be look at this link here....

[http://help.ubuntu.com/starterguide/C/ch03.html#sect-music-and-movies](http://help.ubuntu.com/starterguide/C/ch03.html#sect-music-and-movies)

Included in the above link are instructions to go here (see link below), but just in case you miss that part, here is the link for some common stuff that has restricted licences on them and for that reason can't be included by default with ubuntu, but are installed by the user afterwards...

[https://wiki.ubuntu.com/RestrictedFormats](https://wiki.ubuntu.com/RestrictedFormats)
