### Post by Chappy7777 on 2008-02-19
I have an AMD Athlon XP, with 1 Gig DDRRam. I installed 7.04 on here months ago ( a dualboot setup with XP) and had absolutely no problems with the install.

before I did the install i tried running the CD live, to see if I'd have problems. Prior to putting it on this machine, I'd loaded Dapper and the Feisty on an older AMD 1 Gig CPU. It worked fine. So fine that I decided I'd run it on this, my main machine. 

I have had no issues. This PC has a VIA motherboard, and all of it works great with Ubuntu. 

Try the live CD before doing the install. To do the install, just follow the "nexts" and remember to write down your username and password.
