### Post by crispy_420 on 2006-07-17
Did you want all files of a certain type(mpg, avi, etc.) to open from only one player(vlc, xine, etc.)?

If so right click the file.

Select properties.

In new window, select 4th tab("Open With").

Choose your player, there will be a list of known players.
If the one you want is not there, select "add" at bottom. This will bring up a list of all programs installed, you choose the one you want.

Hope that helps.
