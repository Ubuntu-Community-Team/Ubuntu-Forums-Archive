### Post by Kosimo on 2007-10-15
System > Preferences > Appearance > Visual Effects

If your computer is able to use Compiz Fusion so, is going to be enabled by default.

Gutsy uses Compiz fusion in a basic feature mode: So, as far as I know there is no settings manager and not emerald themes selector. You should install those packages separately.
