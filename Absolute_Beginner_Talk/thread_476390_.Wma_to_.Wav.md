### Post by allix on 2007-06-17
Use soundconverter(install it using synaptic if you have to), make sure you have the gstreamer0.10-plugins-ugly package installed too.

> SoundConverter is a simple sound converter application for the GNOME environment. It reads sound files in any format supported by GStreamer and outputs them in Ogg Vorbis, FLAC, or WAV format, or MP3 format if you have the GStreamer lame plugin.

### Post by dptxp on 2007-06-17
In audio editors /converters, all files open in Raw format (similar to .wav). You can save them in any format provided you got the encoder for it.
WMA is lossy compression, and the WAV files shall be about 10 times larger.
