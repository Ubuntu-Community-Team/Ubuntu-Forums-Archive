### Post by JerMe on 2006-08-05
Welcome to Ubuntu Linux ;)

To change the default formatting for gaim (text size, color, etc.), go to Preferences > Interface > Conversations > Message Text.

What do you mean by "display picture"?  A buddy icon?  For buddy icons, click Accounts, and either add or modify an account.  In that window, you'll see an option to add a buddy icon.
