Hi all just making a post for people that are looking for a Daemon Tools like application in Ubuntu. There is a couple of threads that list the application but none i think with the obvious search string....

so heres an attempt.


Anyway...if your looking to mount iso files in a virtual cd device like daemon tools in windows or alcohol....look no further than:

***Gmount-iso***

This is Gmountiso, a PyGTK GUI to mount your cd images  
Gmount-iso is a small tool written using PyGTK and Glade. It allows you to easily mount your cd images. This is a frontend to the 'mount -o loop -t iso9660 foo.iso /mountpoint' command
Homepage:

 [http://www.crans.ens-cachan.fr/Syst%C3%A8meLinux/GmountIso](http://www.crans.ens-cachan.fr/Syst%C3%A8meLinux/GmountIso)

AVAILABLE IN THE ADD / REMOVE APPS TOOL.

**_Archive Tool_**

If its files you want to access the Archive Manager that is installed in a default installation can open an iso in a similar manner as a .tar or .zip file....and you can drag files and folders out
