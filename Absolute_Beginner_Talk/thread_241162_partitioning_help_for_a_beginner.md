### Post by Marsolin on 2006-08-22
I'd use GParted ([http://linuxappfinder.com/package/gparted](http://linuxappfinder.com/package/gparted)) if you are using Ubuntu or QtParted ([http://linuxappfinder.com/package/qtparted](http://linuxappfinder.com/package/qtparted)) if you are using Kubuntu.

Both are graphical programs to allow you to create, resize, or delete partitions. Don't forget to backup anything critical on the drive first though.
