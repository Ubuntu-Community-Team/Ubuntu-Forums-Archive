### Post by Obor on 2007-06-30
Just go to synaptic package manager, search for Google Desktop and mark it for removal, apply changes, that should be it.
You might wanna have a look in System-prefs-sessions-start up sessions and if the link to google desktop is still there remove it from start up sessions.
