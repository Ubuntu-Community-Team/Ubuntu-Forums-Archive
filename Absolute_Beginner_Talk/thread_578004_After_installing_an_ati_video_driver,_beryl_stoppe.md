### Post by strabes on 2007-10-17
ATI's binary driver (called fglrx) does not support AIGLX, which is generally required in order to run compositing software like beryl. It has a whole lot of other problems as well. This is a problem with ATI, not linux. They are expected to release acceptable drivers this month, but based on their past performance I would seriously doubt this. If you **really** want beryl, you could install the xserver-xgl package. XGL is slower, less stable, and basically all around worse than AIGLX, but it works with fglrx.
