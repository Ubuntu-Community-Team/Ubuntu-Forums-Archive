### Post by The Cog on 2007-12-03
No, do not decompress it.

If you are already running Ubuntu, just find the .ISO file in nautilus (the file explorer) and choose to write to disk.

But I guess you are on windows or you wouldn't be calling it a compressed file, which is actually the wrong word. You don't  treat this file like you do other files. You must burn it to disk **as an image**. I know that nero has a menu option for burning **image** files, and I guess most other applications do too. You know you've done it right if:
a) You burn a single file (without unpacking the contents first)
b) After burning, there are lots of files on the CD

### Post by Niniel on 2007-12-03
Make sure you do a WRITE IMAGE to disk or something like that, do not just burn the file like you would other data files or the CD structure won't be set up correctly and it won't work. 

And before you start burning, you may want to check the MD5 code... if you are indeed using Windows right now, there's a program called MD5Win that can compute the code of your downloaded iso-file for you, you then compare it to the code posted on the Ubuntu servers. 

Good luck installing.
