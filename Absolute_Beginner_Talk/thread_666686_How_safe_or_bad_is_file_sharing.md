### Post by steeleyuk on 2008-01-14
[Deluge]("http://en.wikipedia.org/wiki/Deluge_(software)") is a BitTorrent client.
[Limewire]("http://en.wikipedia.org/wiki/Limewire") is a Gnutella based client.

They both use different protocols to share files so are not compatible with each other.

If you're going to use Limewire, I'd take a look at the Linux version [Frostwire]("http://en.wikipedia.org/wiki/Frostwire").
