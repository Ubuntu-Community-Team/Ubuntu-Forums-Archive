### Post by phersotty on 2007-02-19
Are you connecting to it to your computer with an ethernet cable or usb?  If you are using usb then you probably need a usb driver, but with ethernet i don't think  you need any special drivers if your network card is detected.

the file install.exe is most likely a windows executable file.  If this is the first time you are setting up your cable modem then you might need to use a windows machine to configure it with your isp.  Once its configured then it should work on any computer it connected to through the Ethernet.
