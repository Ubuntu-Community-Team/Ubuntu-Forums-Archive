### Post by adam_kimber on 2007-11-16
The network icon with two monitors is the network-applet, Gnome Network Manager. 

It can be added back to the panel (the bars on the screen) in the following manner press ALT-F2 and then type nm-applet. It will now be back. To keep it there you need to go to Ubuntu Menu --> System --> Preferences --> Sessions and make sure Network Manager is checked in the startup programs. 

As for the cursors check out [Gnome Look]("http://www.gnome-look.org/") and then X11 Cursors. At a glance I noticed there is one there based on the Aero theme in Vista there is probably one for XP cursors too.
