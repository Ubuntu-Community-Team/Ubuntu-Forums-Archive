### Post by bodhi.zazen on 2006-10-12

IMO aptitude is better.

```
sudo aptitude update
sudo aptitude install <program_name>
```

Details: ```
man aptitude
```

```

```

Alternatively, you can open up synaptic package manager, search for 'typing' or something, and probably find alternatives.
