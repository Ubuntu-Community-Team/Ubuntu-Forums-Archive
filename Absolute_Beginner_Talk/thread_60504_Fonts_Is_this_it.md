Hi! I have a question on fonts - sorry if this is obvious or simple. I'm a font collecter. Big time. I love fonts, can never have enough. But... am I stuck with just the core ms fonts & what comes preinstalled, or is there a way to use my collection of windows true type fonts, too? Thanks!

### Post by xmastree on 2005-08-28
In your home directory, make a folder called .fonts (if it isn't there already). Copy your ttf fonts into it.

That's it.

 :smile:
