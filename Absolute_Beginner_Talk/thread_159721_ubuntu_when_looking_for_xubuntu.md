### Post by m.musashi on 2006-04-13
[QUOTE=hoarythehedgehog2009]Is dapper supported by the package manager? Do Gnome and KDE apps work in Xubuntu? Will Dapper mess up my windows partition?[/QUOTE]
Yes, it is supported. I believe gnome and kde apps will also work but I have not been able to get xubuntu working on my laptop so that is only an assumption. I did an ubuntu install and then used apt-get to add xubuntu so I can't say where the problem lies. If you download the xubuntu .iso it should run fine. Maybe someone else who has actually used it can add some thoughts.

As for messing up your windows partition there is never total certainty, but I would say Dapper is about as close as you will get. You are more likely to mess it up by making a wrong selection on install. To be safe you should certainly back everything up that is important.

There are several good guides that can walk you through a dual boot setup. I found [this site](http://users.bigpond.net.au/hermanzone/) to be very thorough.

You should also be aware that Dapper is still not officially released. It works quite well for many people. However, there is always a chance that something will get borked. That being said, I really haven't had or heard of too many problems.
