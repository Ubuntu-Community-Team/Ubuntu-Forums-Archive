### Post by Sparkster185 on 2007-05-23
I know you can get dual monitor support with Ubunutu, but I'm not sure about three.

You can most certainly use your graphics card with Ubunutu.  There's this application out there called Envy that installs/configures your nVidia drivers for you, it's pretty easy.

As for games, you cannot run those games natively on any Linux distro.  If you want to still play your games, you need to either keep your windows partition around and boot to that when you want to game, or you need to use a program called WINE.  It basically allows you to run Windows apps on Linux.
