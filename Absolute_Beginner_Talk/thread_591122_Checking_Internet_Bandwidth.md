### Post by evilregis on 2007-10-25
Make sure no one else in your house is connected.  If you have a roommate/family member downloading last season of Rome (or anything for that matter) you will not get an accurate reading.

If you have an open wireless connection, make sure no one is using your connection.

Also remember that you are only as fast as the server you are connecting to.
