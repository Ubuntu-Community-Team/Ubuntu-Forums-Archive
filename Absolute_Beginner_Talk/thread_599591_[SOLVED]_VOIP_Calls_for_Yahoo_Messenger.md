In short, NO.

Yahoo uses a proprietary software and so OSS cannot make use of the protocol that they use. So PC-2-PC calls are not supported.

Having said that, Gyachi shows a lot of promise, where the developer has made huge progress and now has voice chat (not pc-pc calls) through ekiga (a sip based pc-pc caller installed by default in Gnome)
