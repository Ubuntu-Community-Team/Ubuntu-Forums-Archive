### Post by Marsolin on 2006-11-03
[Anti-Virus]("http://linuxappfinder.com/security/anti-virus") programs in Linux typically protect you from transmitting Windows viruses through email, however there are a small number of native Linux viruses.

[Firestarter]("http://linuxappfinder.com/package/firestarter") and [Guarddog]("http://linuxappfinder.com/package/guarddog") are both front-ends to the [iptables]("http://linuxappfinder.com/package/iptables") firewall that is built into most Linux distributions. Which you pick is strictly a matter of personal preference for one interface over another.
