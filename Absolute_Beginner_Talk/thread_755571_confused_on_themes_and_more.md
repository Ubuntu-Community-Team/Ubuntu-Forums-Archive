### Post by Robertorps on 2008-04-15
If you go to System -> Preferences -> Appearance, you can choose the themes on the first window in front of you. If you look the buttons on the bottom of the screen, you can choose many options to personalize the desktop.

I reccomend you to download a program called "gnome-art" (look for it on System -> Administration -> Synaptics... ) This program downloads all options from a webpage quite similar to gnome-look.org. Once you find something you like, click the INstall button and it will be available in the appereance window.
