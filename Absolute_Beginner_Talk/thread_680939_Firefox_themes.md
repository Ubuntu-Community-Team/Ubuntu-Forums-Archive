### Post by disturbedite on 2008-01-28
i can use ifox original in ff trunk (3.0).  the issue prolly lies with the maxversion.  maxversion means the maximum version which the extension or theme works with.
it is generally recommended that extension/theme authors do it this way:
for ff 2.x:
2.0.0.*
this is so that it will work with any future version of ff2.
or for 3.x:
3.0.0.*
for the same reasons, except ff3 is the difference, obviously.

the author of the theme might have done it differently.  like:
2.0 instead of 2.0.0.*.
there are ways to "fix" this.  you can manually edit the extension's rdf file that contains the maxversion entry or you can get the mr tech local install extension that can do it for you.  (override maxversion is the feature).
