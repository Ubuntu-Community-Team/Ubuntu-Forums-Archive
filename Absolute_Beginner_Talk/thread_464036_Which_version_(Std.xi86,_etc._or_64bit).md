### Post by rich.bradshaw on 2007-06-04
If you choose the 64bit version, you can only used 64bit applications (without some messing around) - The notably annoying one here is Flash - as far as I know there is no 64bit Flash for Linux.

Personally, I wouldn't bother - the only real difference is that 64bit architecture can support more than 4GB Ram - unless you are a server, this doesn't make a lot of difference for most people.
