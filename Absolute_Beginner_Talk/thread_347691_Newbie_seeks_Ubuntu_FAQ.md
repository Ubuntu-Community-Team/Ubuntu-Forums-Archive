### Post by Sklasko on 2007-01-27
Firefox is multiplatform and comes with Ubuntu, so that's an instant win. Plus, you can *try* to use WINE (A Windows Compatability Layer) to run the others Windows apps that you use.

However, if your new to Linux, I would not recommend scrapping your Windows installation just yet. Go for a dual-boot and try it out for a while.

Edit: Also, I would recommend 6.06 Dapper LTS as opposed to 6.10 Edgy.
