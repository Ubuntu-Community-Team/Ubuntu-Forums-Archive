### Post by nickle on 2006-05-09
Installing is really easy. Eithe you use a command line statement by opening a terminal in the menu Applications > Accessories > terminal

and type sudo apt-get install <package> where package is for example  digikam

Or you use snyaptic or adept. These are two package management tools located in the menu:System > Administration and search for the packages you want. The should be easy to install by clicking on the packages you want and then press apply changes button. It might sound difficult, but believe it is very easy. 
The beauty of linux is that you have loads of tools to install for manipulating photos and mp3 files. Search on this site for howto's, they should sort you out.
