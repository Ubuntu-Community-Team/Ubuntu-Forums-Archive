### Post by PmDematagoda on 2007-12-23
The Frostwire launcher is found in Applications>Internet>Frostwire.

Or you can simply execute:-
```

frostwire
```
in a terminal.

### Post by rsambuca on 2007-12-23
You have to make sure you have a recent version of java installed as well.  

To run Frostwire, you should see a Frostwire launcher/icon in your Applications -> Internet menu.  As an alternative, you can launch Frostwire from a terminal just by typing the word 'frostwire' and then enter.
