### Post by DarkN00b on 2006-12-16
I use OnlineBookmarkManager. The extension is available on the Firefox add-ons page, and you can sign up for the free service at [http://www.onlinebookmarkmanager.com/]("http://www.onlinebookmarkmanager.com/") It can be set to sync your bookmarks so that you import them when you open FF and export them when you close it. I've been using it for a while and am very satisfied.
