### Post by davebgimp on 2006-09-01
Are you checking the repositories first? There's a ton of programs available (especially if you enable the universe and multiverse repositories) that makes installing programs really very easy.

Repository help:
[http://ubuntuguide.org/wiki/Dapper#How_to_add_extra_repositories](http://ubuntuguide.org/wiki/Dapper#How_to_add_extra_repositories)

You should try searching the repositories first using Synaptic (Adept on Kubuntu) to see if the program is there. if so, it's basically a one-click install. other than that, yeah, you'll have to build.
