### Post by Jucato on 2006-07-02
Currently, Gmail does not support Konqueror (neither does it support Epiphany). 

However, you can use Konqueror's Browser Identification feature to sort of "spoof" Gmail into thinking it's some other browser. Go to Settings > Configure Konqueror and scroll down to Browser Identification. Click on "New" and enter the following data:

When browsing the following site: google.com
Use the following identification: Safari on Mac OS X

That should do the trick. It's not Konqueror's fault, though. It's Google's decision not to make their code compatible with some browsers. Maybe they think Konqueror, Epiphany, etc. are probably too insignificant to support directly. At least we can use spoofing. :(
