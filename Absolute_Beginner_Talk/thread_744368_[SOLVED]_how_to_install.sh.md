### Post by stchman on 2008-04-03
> **Biggy said:**
> hi

help to install.sh file !!!

The .sh extension usually indicates a shell script.

Do this:

Make script executable
```
chmod 755 ./<shell script file>
```

run the script
```
./<shell script>
```
