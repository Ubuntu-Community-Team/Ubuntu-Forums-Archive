### Post by lloyd_b on 2007-05-05
It's usually a matter of messed up ID3 tags.  Right-click on the song, and select "View File Info".  Correct the Artist/Title information, and click "Save".

Note: if the song is currently playing, you'll need to restart it before you see the changes that you've just made.

Lloyd B.
