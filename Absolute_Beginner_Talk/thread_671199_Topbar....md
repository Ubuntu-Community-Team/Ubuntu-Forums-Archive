### Post by mcduck on 2008-01-18
There are some different menu applets available, just install one you like, add it to your panel and remove the menu bar applet (the one with Applications/Places/System).

For example, USP is worth a try.

The menu bar applet itself doesn't support skins. It can only be changed by using a different GTK2-theme. There's also Main Menu-applet (already installed) that's more like WIn2K's start menu, with only single menu. It has support for different button images.

The panel itself can also have different backgrounds, just drag&drop a picture to the panel to use it as panel background.

2. Go to the bottom panel. Remove the icon on bottom left corner.

4. Add the following items:
     Main Menu
     Notification Area
     Volume Control
     Clock

to Move items, right click and select "Move"

Place the main menu of the left corner

### Post by drs305 on 2008-01-19
Just back to your original question, I don't use skins but you have some options of setting the panel background color, transparency, or you can use your own image for the background.  To change these settings, you right click on an empty portion of the taskbar and select Properties, Background.

Note, when you right click on the taskbar make sure you are not selecting to change a shortcut's properties. If you see "Add to panel" as the top choice after right clicking, you are about to enter the panel properties options. If the top option is "Launch", you've selected to modify one of the panel's shortcuts.
