### Post by Miguel on 2007-12-05
First, you have to open an account in launchpad.net

After that, you should search to see if your issues have been already noted. For that you can use google, launchpad's own search engine, the forums, whatever. It is also a good idea to see  if others see the same behaviour you are experiencing.

Once you are sure (or almost) that your bug is original (i.e. not already reported) and that it genuinely is a bug, you report one in launchpad. Please note that since summer, launchpad will perform a search pointing you to possible duplicates. This is a nice tool to avoid duplicity. You should put there as much info as is relevant such as:[list]
[*] package affected (this is mandatory)
[*] version of ubuntu and package you are using
[*] steps which lead to reproducibility of the bug
[*] crashlogs, tracebacks or similar (if available)
[/list]

In any case, I think there's a nice post in the development forum, which link now seems is hidden so that newbies don't feel the temptation to run alpha distros.
