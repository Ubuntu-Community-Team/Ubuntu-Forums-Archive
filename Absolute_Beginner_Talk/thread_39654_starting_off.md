### Post by bored2k on 2005-06-06
- Adjustments ? I always say this: take Linux one step at a time (like a 12 step program). We understand you know how to do everything in your Windows environment, but don't try to learn everything at once, or else you will get frustrated and leave the beautiful world of Free software. The best you could do is ask on our forums or any other you feel more comfortable with. At least here people are friendly and will be more than glad to help you.

- Step 1: You will need a hard drive partition** specially formatted for Linux. You can do this with the install disc, but since you are better using Windows, you better divide your HDD using PartitionManager (dont worry about the filesystem, Ubuntu install disc will take care of that).

- Get comfortable with the installer. Since you have used the live disc, you have never seen it. Ubuntu installer has no pretty graphical installation like other linux distributions do, but it is in no way any harder than the other. Get ready: [**[COLOR=DarkRed]click here[/COLOR]**](http://shots.osdir.com/slideshows/slideshow.php?release=305&slide=1) to see the installer run. Better safe than sorry ;).

- At first, MP3s won't play and you won't be able to access your Windows drive/files on Ubuntu. Do not panic, this gets easily solved in about a 3 minute process. We would help you on this in due time (remember, one step at a time) ;).

- Have an open, positive mind. If you install it thinking you will fail, you will fail. Remember applications in Linux are not 100% clone copies of Windows's applications (some are), so don't get frustrated if you don't get things done.

- Use the forums.

** Hard drive partitions: [http://en.wikipedia.org/wiki/Hard_disk_drive_partitioning](http://en.wikipedia.org/wiki/Hard_disk_drive_partitioning)

### Post by Seti on 2005-06-06
Of course you can play games using linux. Right now though, its just a lot easier to do it with windows, because you will need Cedega/WineX to run those windows games you like. Setting this up is a bit involved, so it would definitely be a learning challenge to get going. 
Of course, you need to ask yourself what you want get out of using linux. If you are telling yourself "OK, I want to start using linux to play games" you are going to be dissapointed, period. Linux is not a gaming platform. Yes there are many games available, but just not as many as for windows. We wonder why. 
Many linux-heads are also big gamers, so these people keep a Windows partition around. 
But if you want to get more out of your computer than just playing GAMES, then linux is worthwile.
