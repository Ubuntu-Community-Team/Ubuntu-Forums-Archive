### Post by ByteJuggler on 2008-03-10
Yes.  If you want the full shebang, then install the (meta) package "kubuntu-desktop".  Then it'll install all the applications of the normal kubuntu desktop install alongside whats already there.  If however you only want a subset of things you may want to install one of several other packages (which I forget the names of now, probably "kde-desktop" or such.)  

Anyway, after installation you can pick whether to log into KDE or Gnome on the logon screen. (Click on "Session".)

(Warning: Uninstalling all the packages installed by the Kubuntu desktop if you decide later on you don't want it is not as straightforward as installing it. However, there are pages on the internet that list the packages in kubuntu-desktop and thus make it at least possible to uninstall them with a little effort , should you ever wish to go back to a Gnome only system.)
