### Post by alan34 on 2008-03-28
If you are running compiz, emerald is a "shiny" window decorator. To install 
go System - Adminstration - Synaptic Package Manager and search for emerald.

To start with compiz and emerald go System - Preferences - Sessions click Add then
In Name put Compiz and in command put

compiz --replace

Then to start emerald go System - Preferences - Sessions click Add then
In name put Emerald and in command put

emerald --replace
