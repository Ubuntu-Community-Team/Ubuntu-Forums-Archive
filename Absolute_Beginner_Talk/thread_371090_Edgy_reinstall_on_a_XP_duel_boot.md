### Post by annda on 2007-02-26
the same as installing :-)

just choose "manual partitioning" using the installation cd and choose your ext3 partition to be mounted as root filesystem ("/"), and the existing swap to be mounted as swap. make sure you check them both for formatting.

p.s. save any files created under lunix that you want to rescue first, because everything on your linux partition will be DELETED!
