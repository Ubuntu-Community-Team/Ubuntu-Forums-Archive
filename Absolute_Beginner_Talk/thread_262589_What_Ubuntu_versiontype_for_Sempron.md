### Post by dmizer on 2006-09-21
the amd kernel is for use with the 64 bit kernel.

either the amd or the i386 install will work with your processor.  you might like to start off with the i386 kernel as it's a bit more friendly to get working properly.

i do suggest at least trying out the amd64 kernel at some point in the future, compare and see how you like it.  but if you're brand new to the linux world, i'd suggest avoiding it until you're past some of that learning curve.

### Post by 3rdalbum on 2006-09-22
The Desktop CD can be run live, but it also has an installer on it.
