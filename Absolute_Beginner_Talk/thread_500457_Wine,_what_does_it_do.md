### Post by Alex&amp;Linux on 2007-07-13
Havent really delved into it, but, as far as my understanding leads me:
Its not an "emulator" but more of a compatibility layer that can be applied between your fav distro, and all those .dlls out there.
Apparently its performance with many apps is the same, or close to what it would be if run directly on windows.
Should be able to run Office or any other MS app easily with it, as well as print, share ect.

### Post by bobplano on 2007-07-13
wine lets you run programs from windows in ubuntu. it partially recreates the windows API so if you're worried about legal things you might not want to use it.
