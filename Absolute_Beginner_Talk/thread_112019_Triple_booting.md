### Post by Griff on 2006-01-03
Yep.  Have at it.  :p  I'm not sure about edubuntu, but ubuntu and kubuntu are the same thing.  They just have different desktop managers.  You shouldn't have to boot them seperately.  Just install ubuntu and if you would like to try kde (kubuntu) just install kde through apt-get.  Then when you start ubuntu you will have the option of using either Gnome or KDE.
