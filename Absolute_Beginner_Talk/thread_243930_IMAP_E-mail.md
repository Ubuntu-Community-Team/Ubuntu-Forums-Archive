### Post by benfindlay on 2006-08-25
Theres fetchmail, which is available through the Synaptic Package Manager or via the terminal with:
```
sudo apt-get install fetchmail
``` (I think, anyways ;))

It's SSL enabled, and does POP3, APOP and IMAP
Oh yeah, and under Synaptic it has the Ubuntu logo next to it, so it looks like it's designed specifically for ubuntu

Hope this helps
