### Post by Patrick K on 2007-03-13
Wine is probably best described as a shell that houses windows apps. Apps are installed into wine. Wine tries to emulate Windows and fool the apps into running. 

VMServer creates a virtual machine, running in the Linux OS, that allows you to install a Windows operating system. Once Windows is installed, it thinks it is running directly on the hardware. Apps are installed into real Windows running in the VM. 

There are several other VM solutions, too. VMware's VMS is the one I'm most familiar with, although, I haven't ran it myself.
