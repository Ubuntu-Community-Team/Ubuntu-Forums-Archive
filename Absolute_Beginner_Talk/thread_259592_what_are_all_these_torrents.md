### Post by jordanmthomas on 2006-09-17
Bittorrent itself is perfectly legal.  However, many of the sites you can find torrents at have torrents that link to illegal material.  Just use good judgement...you know what is and isn't right.

Anyway, a file with a torrent extension tells a bittorrent application where to go to start downloading the file(s).  These files are on other user's computers (read:  p2p).  They upload parts to you and you in turn upload parts to others.  Eventually, you will have the whole file downloaded. 

Say you have something you want to share.  Using a bittorrent app, you can create a .torrent file and upload it to some website.  Then, when people open that .torrent file, they will connect to you and start to grab your files. 

If the torrents you are looking at are multiple gigabytes, chances are it is many videos in one package.  You should be able to select only the file you want as to keep some of your bandwidth.

Hope this clears it up a little bit.
