### Post by intangible on 2005-06-24
Unfortunately, just like in the Windows world, each software manufacturer can decide how to create their own installer anyway they want, so it may be different for each application.  The creators of the software you're trying to install should have included installation instructions with them.

You can install a lot of other software through **System->Administration->Synaptic Package Manager**.

If you're having a problem with installing a specific peice of software, just ask about that and someone will try to help you. 

Hope I helped some.

Well....the best way to install programs in Ubuntu is using synaptic package manager (system, administration, synaptic) and download and install the programs from the internet.
 
For that to work well, you need to do this first:

[http://ubuntuguide.org/#extrarepositories](http://ubuntuguide.org/#extrarepositories)

Ubuntu really works best only with software installed this way.

Since you are so excited about Linux I'll be a little more honest with you than I would be with most people. If you want to use Ubuntu, but can't stand to download all the software on dial up, then you should pay some enitity to ship you an Ubuntu install DVD.  It has KDE on it (it would be over 100 mb to download and install KDE otherwise) and many other packages.

Let me even be a little more honest. If I was on dial up, I wouldn't use Ubuntu. The DVD is nice, but it doesn't have but maybe 1/3 the packages synaptic has. If I was on dial up, I would use Debian- the version of Linux on which Ubuntu is based on. Why? Because even though Debian is harder to set up, you can order ALL its packages on a CD. What I'm saying is that in Ubuntu, even if you have the dvd, you can only access about 1/3 the programs you could if you downloaded everything you wanted to use.

But Debian ships on many cds, so EVERY SINGLE PIECE OF SOFTWARE is availible on cd...no downloading except for illegal stuff like media codecs.
