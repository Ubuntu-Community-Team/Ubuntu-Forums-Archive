### Post by galvatron1983 on 2007-05-03
To download files via BitTorrent you need two things, a BitTorrent client (the main program like KTorrent or Azureus etc...) and a bittorrent(.torrent) file that you get from a Torrent website like [www.isohunt.com](www.isohunt.com) or [www.mininova.org](www.mininova.org). 

The BitTorrent client opens the .torrent file and uses it to connect to the swarm of users who have the file you want. Basically at this stage you join a swarm of users all connected who have a piece of this whole file that you want and you begin downloading all of the pieces from them. As long as there are seeds (people who have the complete file downloaded) connected your client will download the file to wherever you specified as a save location. 

Thats BitTorrent in a nutshell.....
