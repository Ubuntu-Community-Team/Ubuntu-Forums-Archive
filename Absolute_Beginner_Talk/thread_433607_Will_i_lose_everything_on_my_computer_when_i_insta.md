### Post by Pobega on 2007-05-05
If you install it over Windows then yes, you will lose all of your data.

If you partition your computer to be 50% Windows and 50% Ubuntu then no, you shouldn't lose any data.

If I were you I'd use the Ubuntu 6.10 LiveCD to resize your Windows partition, because the NTFS support in 7.04 is crippled from what I've heard.

What do you have on your system that is valuable to you? 
If it's your main machine with photo's movies, music and documents, and you use it for email and stuff, then ask yourself the question how much would I pay to get this data back if I lost it?  Try and put a value on it as this can then help you justify spending money on something like an external hard disk that you can use to copy your data to.  I have seen an external disk 320GB for £56 odd.

If you are in any doubt, then don't install (yet!) get your data secured with all your favourites, and passwords, address books, emails etc, etc, ...  Have a play with the live CD while you get yourself sorted out, and check out the forums here to see if you may have any hardware problems when you install.
(Under windows have a look online for belarc advisor, it's a windows app that will list your system to the nth degree, it's free for home use. install it, and then print a copy of the report off.  It may save your bacon when looking for drivers and asking questions here. (you could even post the entire report, but it does put all the installation keys you used for things like windows, office + other apps.

The answer really is that you can install it so that it SHOULDN'T mess up your system, however you are adding a new oprtating system and this does carry an element of risk.  Nothing to do with what operating system you are installing.  If you were to dual boot with Vista, or 98, or XP, same level of risk really. There is the potential for bad things to happen.  

Good luck on the install, I hope we hear from you on progress and how it went.
