### Post by The Joe on 2007-01-24
I've used Kubuntu before and I can tell you it's worth it if you want a new look, KDE is a lot more closed in, whereas in GNOME most things are scattered. IE, all the desktop options (wallpapers, virtual desktops, themes etc) are in the same place, in GNOME they are in completely different areas to eachother. However, the panel and desktop are easier to clutter than GNOME.

Kubuntu is as maintained as Ubuntu, Edubuntu and Xubuntu are, so you'll still get everything you need like in Ubuntu.  If you want to try it, go for it. I'm pretty sure it will have the 3D Desktop treat as well as Edubuntu will, not sure about Xubuntu as it's for lightweight systems.

Also if your new to Linux, you might want to try it as it has a more Windowsey feel to it.

One last thing: Don't make the mistake I made, if you only want to try Kubuntu and you have Ubuntu, simply go into Terminal and type ```
sudo aptitude kubuntu-desktop
``` Using aptitude instead of apt-get makes it a whole lot easier to uninstall.
