### Post by paulyche on 2006-05-24
Do you know what you've started!?

In my opinion:
Take the time to learn vi or emacs only if you think you're going to be doing a fair bit of  programming/scripting. I use vi but people get zealous so someone will disagree with me. Once you've learned one of these editors text editing jobs can usually be done MUCH faster than with gedit or something.

Else: gedit and kedit are perfectly functional notepad-like GUI editors. They get the job done and are perfectly intuitive whereas VI and emacs are powerful but not in the least intuitive.
