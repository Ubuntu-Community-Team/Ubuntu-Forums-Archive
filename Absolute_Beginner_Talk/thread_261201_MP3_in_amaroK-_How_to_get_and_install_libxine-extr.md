### Post by mongooseman1128 on 2006-09-19
I was having that same trouble a while ago. I finally went with using Rhythmbox and got the codecs from automatix. Go to [www.getautomatix.com](www.getautomatix.com) then click the link to install and find the instructions in there. After it installs for go Applications> System Tools> Automatix open that and it goes through a few dialogue boxes and runs a script. After that it gives you a big list of things to install (avoid the nvidia drivers I think something happened when I installed them last time and it caused some GUI problems) and choose w32codecs. That should get you your codecs and you can open with rhythmbox
