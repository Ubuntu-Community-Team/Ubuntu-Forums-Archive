### Post by p_quarles on 2007-06-18
> **dunklegend said:**
> Is there something similar to Microsoft's Clear Type Tuner?
I would like something so the text shows more clear in the LCD displays?
A couple things to try:

1) Go to System/Preferences/Font. Make sure that subpixel smoothing is selected.

2) then enter this ```
sudo apt-get install msttcorefonts
```

Hope this helps.
