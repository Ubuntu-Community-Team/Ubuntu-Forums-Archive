### Post by mormor on 2008-05-21
FM08 is possible in wine in two ways. The first one I used in 7.10, the second one in 8.04.

1: Copy the folder from a windows xp/vista install and put it in a botle of Wine. Worked without CD-Crack. Set wine to vista.

2: Install under wine from cd, use XP as os when you install it, find a no-cd-cracked exe, run it from wine under vista.

Works like a charm for me.
