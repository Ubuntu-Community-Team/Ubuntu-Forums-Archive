### Post by Tomosaur on 2006-09-01
Usually you just type the name of the program into the command line (terminal), or something very similar if it has a long name or contains multiple words. Many programs aren't something you'll need to run manually though.

To add programs to the Applications list, you should use Alacarte (I presume you're using Gnome rather than KDE, not sure how you do it in KDE you see). To get to alacarte:

Applications > Accessories > Alacarte Menu Editor
