### Post by Inxsible on 2007-11-05
Its probably the entire DVD install which may include most of all the apps available in the synaptic (or whatever the Debian equivalent is)

If you look into the parent folder and then into iso-dvd, there are 3 DVD which more or less equals to 21 CDs (7 CDs for a DVD ~ 700MB *7 = a lil over 4 GB )

### Post by bumanie on 2007-11-05
A download page with mutiple iso and torrent files is purely reflecting various servers that are offering ubuntu for download. All you have to do is choose one of them. It is good practice to check the md5checksum of the iso file after download.

### Post by macogw on 2007-11-05
You only need the first CD to install Debian.  If you get all 21 CDs (or all 7 DVDs), you will have the entire repository on CD/DVD and be able to install things without an internet connection.
