### Post by Sonic Reducer on 2007-10-22
ok, i got it. i modified my /etc/fstab to look like this:
> # /etc/fstab: static file system information.
#
# <file system>	<mount point>	<type>	<options>	<dump>	<pass>
proc		/proc		proc	defaults	0	0
# /dev/sda2
UUID=cf3dbb90-6e5d-4560-a89a-6b1ebdc01697 /               ext3    defaults,errors=remount-ro 0       1
# /dev/sda1
# /dev/sdb2
UUID=b648bb8d-c7b6-4447-9f57-bd323e0a4fea none            swap    sw              0       0
/dev/hda	/media/cdrom0	udf,iso9660 user,noauto	0       0
/dev/hdb	/media/cdrom1	udf,iso9660 user,noauto	0       0
/dev/fd0	/media/floppy0	auto	rw,user,noauto	0       0

[I]#I Added This To Mount The Media Drive At Bootup (10-22-07)
/dev/sdb1	/media/Media   vfat	defaults**,user,noauto**	0	2[/I]

# Generated by Automatix
/dev/sdc1   /media/sdc1   vfat    iocharset=utf8,umask=000  0    0
/dev/sda1   /media/sda1   ntfs-3g  defaults,locale=en_US.utf8    0    0
## End of Automatix mounted partitions

the italics are what i added when following what Overdrank posted, and the bolded part is what i think solved it. i dont know which option did it, but i figured i could read/write/rename/delete/whatever files on /dev/fd0 (floppy drive) so i'd borrow the options. this works, but please let me know if i should modify it further because of a security risk or something.
