### Post by aysiu on 2006-08-27
Go to System > Preferences > Themes. A Theme Manager window will appear.

Drag and drop the theme .tar.gz file into the Theme Manager window.

That's it.

P.S. Do not unzip the .tar.gz or modify it in any way! Just drag and drop it exactly as you downloaded it.
