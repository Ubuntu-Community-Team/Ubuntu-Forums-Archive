### Post by Frank Golden on 2006-06-06
Hi All,
   I posted this possible solution on the Laptop support subforum
in response to a 5672WLMi owner having problems playing DVD movies
in Ubuntu. I think it bears repeating here. I am a 5672WLMi  owner and
had been having trouble with DVD movies not playing even after installing
all the restricted codecs etc. using Easy Ubuntu.
   The problem with my machine and I bet all new 5672's is that the drive has no region set out of the box. Simply setting a DVD region was all it took to
allow DVD playback. This is with protected disks. Of course setting a region 
uses one of your 5 choices. I also found that Xine worked better than Totem.
Since I have Ubuntu 6.06LTS/XP-SP2 Pro dual boot I used device manager in XP to set the region. Don't know how in Ubuntu. This is a firmware setting so it doesn't matter what OS you use.
    Apparently this is an issue with the Panasonic (Mitshitu) DVD-RAM UJ-845S
that ships with this laptop model. Anyway hope this helps. 
BTW there is not supposed to be region free firmware available for this DVD
drive.
