Hi there and welcome!

1. Download package to Desktop
2. Open terminal - cd Desktop
3. untar package
4. cd to untarred package
5. run the install script

Those are the actions required - you will have to find the correct package on your system and confirm the correct name  of the install file (install.sh?) (Right click on package - open with archive manager and have a look!)
