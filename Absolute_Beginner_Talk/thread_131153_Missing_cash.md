### Post by Turtle.net on 2006-02-15
In synaptic, you can select the packet you need info about, and look to the properties.
There is a list of installed files where you can find where exactly was installed the software you are looking for.
Normally the exec file is in /usr/bin
You can launch it from a console, or use smeg to create a launcher in your Applications folder.
Hope this helps :)
