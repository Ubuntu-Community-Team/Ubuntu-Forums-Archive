### Post by Sef on 2007-05-06
> Now first of all I want to know what is the difference between Ubuntu and Kubuntu?
What is Gnome and KDE?
What is a 'LiveCD'?

1) Ubuntu uses Gnome and Kubuntu uses KDE.

2) Two different desktop environments.  The backends are the same though.

3) A live cd is an operating system that runs off of your cd (or your ram sometimes.)  It won't install to your hard disk unless there is an option to install.

4) Start with Ubuntu because most people on here use it and answers tend to come quicker and more people can answer them.   Once you get comfortable with Ubuntu, then you can try out Kubuntu.

5) With the Live CD you can check out your hardware and see if there will be any problems.

Additional Questions:

> Can I find similar software to work with ubuntu??

Most of the time yes.  What software are you looking for?

> Will my USB devices (Printer, MP3 Player, Camera etc) work with Ubuntu?? I mean can i access their memory after i plug them into the USB port?

Could you list what you have please.  Some hardware works OOTB (Out of the Box), some doesn't.

> I know how to set up internet in Windows XP...i.e by entering the IP address and stuff, is setting up the internet similar in ubuntu??

If you have broadband, it will most likely just pick it up.  If you have dial up, then you will have to set it up.

Again what hardware do you have.

> I would like to dual boot XP and Ubuntu as I'm not sure Ubuntu will work properly as I'm running it for the first time.

Read [Psychocat's installing]("http://www.psychocats.net/ubuntu/installing") for the Desktop install.

Read the [Illustrated Dual Boot]("http://users.bigpond.net.au/hermanzone/") for the alternate (text-based) install.

> Can ubuntu work with NTFS drives??

Yes, there is some software that you can download from the repositories that reads and writes to NTFS.

> Some of my friends told me that Linux was for people with a lot of grey matter inside their head (If you know what i mean)..is this true?? Is it difficult to set up Ubuntu??

Ubuntu Linux is very easy to install and use.  If you have any more questions, please ask.

> Should i go for 64-Bit Ubuntu or 32-Bit?? 

Go with the 32-bit.  There are more software for it right now.

> Can I get an antivirus software for Ubuntu??? Can ubuntu be affected by viruses?

You can get an antivirus for Linux, but in the real world there aren't any viruses for Linux in the wild.

### Post by MoxJet on 2007-05-06
Regarding Daemon Tools, mounting .iso files is very easily done in Linux. If the file isn't iso you can generally convert them without any hassle with programs like nrg2iso , mdf2iso , ccd2iso etc. You can also mount how many iso files you want at the same time.

When you finally want to try it, this is what you write in your soon-to-be best friend, the Terminal:

```
sudo mkdir /media/iso/
sudo mount file_to_mount.iso /media/iso -o loop -t iso9660
```

And that's it! Viola!

Good luck with your Ubuntu and Linux experience. You seem computer orientated; this will probably be a wow-experience for you :)

@asdfgh2: Assuming that the basic linux are all same. Use this script:

Make a new file anywhere (on the desktop maybe)
[Desktop Entry]
Name=RP-PPPoE
Comment=RP-PPPoE
Exec=gksudo /opt/rp-pppoe-3.6/go-gui
Icon=pppoeconf.xpm
Terminal=false
Type=Application
Categories=Application;Network;

SAve the file and click on it.

Run: Applications -> Internet -> RP-PPPoE


THis worked fine in ubuntu 6.06. I cont guarantee that this will work fine in RH. Anyways Best Of Luck.
