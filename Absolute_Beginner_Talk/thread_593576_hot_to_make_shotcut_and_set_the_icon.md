### Post by WinterWeaver on 2007-10-27
If you are editing the icon on the desktop or panel, then the above method will work. If however you want to change the icon for you app in the Main Menu, then you need to do this by clicking:
System >> Preferences >> Main Menu

Search for you app in that app, and then Right Click >> Properties on it

Hope it helps.
