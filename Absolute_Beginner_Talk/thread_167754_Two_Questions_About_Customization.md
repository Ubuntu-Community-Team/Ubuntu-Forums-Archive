### Post by htinn on 2006-04-29
In GNOME, you right-click on an empty part of the panel and select Properties. That will let you change some things about the panel.

When you select an icon for a launcher or menu item, you will have to navigate a folder browser to select a custom icon. Once you find the folder you want, you will need to click the text at the top of the "Browse icons" window and press Enter. That will refresh the icon window, and at that point you can select a new icon.
