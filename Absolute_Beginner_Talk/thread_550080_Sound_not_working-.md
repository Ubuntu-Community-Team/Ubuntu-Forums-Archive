### Post by startherepodcast on 2007-09-13
Ok try do the following:

Go into Synaptics Package Manager and check if the Packages

    * "GStreamer ffmpeg video plugin"
    * "GStreamer extra plugins"
    * "GStreamer plugins for aac, xvid, mpeg2, faad"
    * "GStreamer plugins for mms, wavpack, quicktime, musepack"
 are installed. If not, install them. Try if it works now.

If that does not work go into your sound settings and check if there is a device U can set Settings for (Sound Playback: Something)

Test it and check what happens (click Test)

If it still doesnt work post results!
