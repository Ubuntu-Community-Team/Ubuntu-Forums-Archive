There is a setting in Firefox that will let you simply click once on the address bar, and the entire line of text will be highlighted, ready for you to either copy and paste for whatever reason, or you can start typing in a new address to go to, and the old address will have disappeared.  You change the setting by doing this:

In the address bar, type ```
about:config
```  

When that page show up, in the search bar, type in   ```
browser.urlbar.clickSelectsAll
``` 

When that option shows, double click on that line to change the option to True.

Then close that page, and the next time you single click on the address bar, the entire line of text will be highlighted.

Hope this helps some of you that had this question, as I really didn't see this answer within this thread.
