The apple thing is
Kxdocker
Which is in the repositories.
Here is very in depth tutorial, with picture by picture help, on how to install and custumize your ubuntu desktop, including kxdocker.
[http://www.freesoftwaremagazine.com/articles/change_ubuntu_look](http://www.freesoftwaremagazine.com/articles/change_ubuntu_look)

another thing you can do to really macify your desktop is installing 
XGL/Compriz desktop effect.

The safest way to I have found to installing that is first, trying automatixbleeder.
Automatixbleeder will check your hardware to see if you even install XGL/Compriz, and if you do have the hardware, it will not install it.  
Here is the automatixbleeder install howto:
[http://ubuntuforums.org/showthread.php?t=225967](http://ubuntuforums.org/showthread.php?t=225967)

When you are done with automatixbleeder, you can delete it, or keep it.  
If you like automatixbleeder, then you will probably like automatix.
they are both in the same repository.
The install for automatix is here:
[http://ubuntuforums.org/showthread.php?t=190025](http://ubuntuforums.org/showthread.php?t=190025)

Automatix has alot more apps.  
NOTE:
When you are done with automatix or automatixbleeder, you will be asked to restore you old repository list (sources.list). CLICK yes.  

have fun.
