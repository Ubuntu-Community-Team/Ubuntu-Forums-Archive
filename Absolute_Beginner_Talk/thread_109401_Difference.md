### Post by az on 2005-12-28
You can install both.  You select which one you want when you log into your session.

In Ubuntu, to get kubuntu, install the kubuntu-desktop package

In Kubuntu, to get the ubuntu desktop, install ubuntu-desktop

You can also try edubuntu-desktop (for kids and schools) and xubuntu (xfce is a lightweight desktop for older lower powered systems).  Gnome is simple and useable.  KDE is .... full.  Gnome is based on the GTK libraries, which were always GPLed free software.  KDE is based on QT libraries which is a commercial toolkit which is free, but was once non-free. 


In free software, there is often more than one way to get things done.
