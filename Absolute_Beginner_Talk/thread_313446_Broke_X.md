Try you back up:```
sudo cp <backup> /etc/X11/xorg.conf
```

Where <backup> is the full path to your backup, I presume /etc/X11/xorg.conf.bak or some such....

### Post by aysiu on 2006-12-06
Let's say your backup is called xorg.conf.backup

In recovery mode, type ```
sudo mv /etc/X11/xorg.conf /etc/X11/xorg.conf.beryl
sudo mv /etc/X11/xorg.conf.backup /etc/X11/xorg.conf
reboot
```
