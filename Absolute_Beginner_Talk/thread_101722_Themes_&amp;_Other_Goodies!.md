### Post by mcduck on 2005-12-10
How about installing Gnome Art Manager. It will let you browse and install themes from art.gnome.org without even opening your web browser..

Run 'sudo apt-get install gnome-art', and you'll can find it in System/Preferences/Art Manager :)
