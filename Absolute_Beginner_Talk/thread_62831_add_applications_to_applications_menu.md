### Post by tehwa on 2005-09-06
There is a menu editor named 'Smeg' that you will love:```
sudo apt-get install smeg
```Once installed, press Alt+F2 and type 'smeg' to start modifying menus.

GNOME panel menus are controlled through text files I believe, which I guess is flexible if you know them well, but useless if you dont  :-| .
