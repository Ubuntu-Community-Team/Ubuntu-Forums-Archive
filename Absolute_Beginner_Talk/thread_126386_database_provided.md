### Post by ysse on 2006-02-06
Oracle 10 Express Edition (free) will run on Ubuntu. There is a thread about it here in Absolute Beginner Talk. 

Last time I looked, "ordinary" Oracle 10g was certified only on SuSE, Red Hat and United Linux. With a bit of trickery it could be fooled to think it ran on Red Hat (create /etc/redhatversion and some option to the installer).
