### Post by ZenobiaBE on 2007-07-05
Hello Firfin,

I can remove the remote servers from my list by opening the 'Network' folder (Places >> Network), right-clicking the server I want to remove and then clicking unmount volume.

The difference between unmounting and moving to trash is that unmounting simply cancels the connection to that remote folder, moving to trash would try to delete the contents.
