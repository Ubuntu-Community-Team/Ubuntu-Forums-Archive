### Post by uchuunoyanushi on 2007-04-28
I've been using Ubuntu for about a year.  Word processing is my secondary reason for using a computer (after the internet, of course), and the first time I used Open Office, I was amused to see that Ubuntu had what I assumed to be open source versions of the fonts on Windows that I was used to (Verdana, Times New Roman, and so on).  Well, it was no skin off my nose, so I found a font I liked, set it as the default and continued happily banging away on my keyboard.

A few days ago, I made some comment to my Mum - who's something of a computer person - about the difference in fonts, which sparked a little discussion/argument about fonts.  Mum's argument was that if I type a document on Open Office on my computer using, say, Deja Vu Sans, it shouldn't work on a computer that doesn't have that font installed.  Mine was, "well, it does, so meh".

Oddly enough, if I open a document on my computer that I originally wrote on my windows computer, it says that the font is in Verdana, even though I don't have Verdana installed on my computer that I am aware of.  I sent a test document to Mum's windows computer which displayed fine, but said that the font was Deja Vu Sans, which I know she doesn't have installed on her computer.

So it got me wondering.  It was my assumption that it works a little like webpages, Open Office defines a font set (like Sans) as well as the font within the .odt file and when it's opened if OOo can't find the exact font it picks a similar one to display.  But we couldn't figure out why OOo said that the font was Verdana instead of whatever comparable font it was actually using.  Or for that matter, if there's a potential compatibility issue should I one day do business on this computer and start sending .doc files written in non-MS fonts.

I searched around a bit (on google, here and the OOo forums) and didn't really find anything about how fonts work on word processors.  Since I'm particularly lazy today, I thought I'd throw this out here and see if anyone knows anything about fonts or has any links on the subject.

Thanks!

### Post by freebird54 on 2007-04-28
I suspect that if you don't change them, it will leave the names alone - reagardless of what it has to use to display them.  That way the file is 'unchanged' if it goes back into an Office environment.  Things you create in OO.o, however, use fonts from the system by their names.  You may see this behaviour change if you install the ms fonts pack - but it makes sense as it is (sort of).
