### Post by vigleik on 2007-02-04
You are **not** supposed to extract the iso file you downloaded. Instead, find a CD burner program that will allow you to burn the iso file as an image (not all windows cd burner programs will do this, google for one that will or search on the forum). Do not burn it as a file, then you won't be able to boot from it.
