### Post by kyphi on 2007-08-30
If you right click on a file in your home directory you can click on the Help button that will take you to a full description of emblems and icons, including how to make your own emblems.

Emblems are an additional identifier to icons.

I have a collection of icon files on my XP drive and access them from Ubuntu.  These are .ico files and Ubuntu has no problems with that.  All the icons in Ubuntu are .png files.

If you want to make an icon collection, you can create a folder called 'icons' in your home folder rather than adding them to the system files.

Is that what you wanted to know?
