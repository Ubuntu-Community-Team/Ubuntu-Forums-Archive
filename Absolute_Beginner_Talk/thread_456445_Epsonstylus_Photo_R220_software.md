### Post by throntax on 2007-05-28
Those options should be available through the gimp-print driver interface, from memory (i'm at work, no GNU/Linux here) these are available obviously through The Gimp's print setup, but also are available in more general print setup tools. 

You could do a search for CUPS (Common Unix Print System, I think) in your package management software, maybe some configure software will appear, or just search fro gimp-print, but it's probably already installed. 

I have an Epson Photo R210 and it works well under most conditions (it used to be reeeally slow on Hoary Hedghog and High res, but more recent versions of Ubuntu don't seem to have that problem...) .There's generally good support for Epson products.

hope this helps
