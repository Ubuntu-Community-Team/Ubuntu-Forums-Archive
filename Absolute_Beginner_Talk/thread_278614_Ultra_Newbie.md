### Post by pbaehr on 2006-10-16
If you don't want to use Windows ME and you don't want to upgrade to Windows XP then a dual-boot system is not necessary. You don't need to buy a new hard drive, you could simply format the existing one.

The advantage to dual-booting between your existing Windows ME install and Ubuntu is that if you have problems with Ubuntu you can always go back into Windows while you're trying to resolve them.

If you can afford to have the machine out of commission for a few days while you fix any problems (hopefully they'll be few and far between) then I'd recommend you just format the existing drive during the install and let Ubuntu overwrite it.

Of course, it would be handy to have another computer available in case you can't get on the internet to get help with any problems you encounter.

Also, keep in mind any **personal data or files will be destroyed** in the process, so be sure to copy that off the drive first if you want to save anything.
