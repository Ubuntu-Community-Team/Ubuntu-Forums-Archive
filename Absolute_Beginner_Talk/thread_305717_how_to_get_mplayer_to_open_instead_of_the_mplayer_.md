### Post by turbojugend_gr on 2006-11-23
OK that's not for sure but give it a try: 

--> in firefox -> edit -> preferences -> (third tab the one after "tabs", I can't recall how is that in english version, sth like content though) -> file types or sth, near the bottom -> manage. There add a rule for mp3 and mplayer should open mplayer (or any app you wish, like rhythmbox or sth, instead of using hte plugin)

A better(or smarter) way linux-wise : create  a launcher on your desktop with the command "gmplayer "%1" and simply drag and drop the mp3 link over that launcher. That's nice ha?

Finally **I would suggest** you to take a look at [this video]("http://www.songbirdnest.com/screencast/"), if you like it you can install the amazing songbird easily, search for songbird keyword (press CTRL+F and a search bar will appear at the bottom of firefox) in [www.ubuntuguide.org](www.ubuntuguide.org).
