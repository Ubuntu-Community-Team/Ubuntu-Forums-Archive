### Post by swisscow on 2007-09-11
I can do this.

I did right click on the bottom panel, choose new panel, panel

I moved the new panel to the top. Right click, choose configure

Make sure settings for panel are selected (not main panel) - this is in the arrangement tab

In the size section choose custom, change size as required.

I think this is "normal" behavior - if yours isn't working something isn't right.
