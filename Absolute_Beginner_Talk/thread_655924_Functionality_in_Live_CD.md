### Post by R15I23D05D14Y on 2008-01-02
The only difference between a LiveCD and a proper installation is the speed that everything happens at, and the fact that settings don't reset on reboot.

Installing Ubuntu permanently on your computer shouldn't make it more easy to tweak hardware - unless you install additional drivers and such, eg, ndis-wrapper.
