### Post by kyphi on 2007-09-28
Under Applications, Crossover select Configuration.

You can either go to Manage Bottles, highlight the application you want to delete and press the delete button on the screen ... or...

you can go into Add/Remove, expand the application entry, highlight the application and click on Repair/Remove.
