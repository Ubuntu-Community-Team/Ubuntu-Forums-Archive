### Post by locke.dragon on 2007-04-26
for those of you who would also like to do this, follow these instructions...

before you do anything, backup xorg.conf 
```
 
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup 

``` 

now open up xorg.conf in gedit
```

sudo gedit /etc/X11/xorg.conf

```

scroll down until you see the "synaptics touchpad" entry.  replace that section with this...

```

#To re-enable touchpad, change "SendCoreEvents" to "true"
Section "InputDevice"
        Identifier      "Synaptics Touchpad"
        Driver          "synaptics"
        Option          "SendCoreEvents"        "false"
        Option          "Device"                "/dev/psaux"
        Option          "Protocol"              "auto-dev"
        Option          "HorizScrollDelta"      "0"
EndSection

```

that should do it.  if you have any problems, just punch this into the terminal...
```
 
sudo cp /etc/X11/xorg.conf_backup /etc/X11/xorg.conf 

```
