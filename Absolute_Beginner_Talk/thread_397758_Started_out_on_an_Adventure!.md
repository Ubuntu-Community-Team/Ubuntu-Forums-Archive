### Post by ramjet_1953 on 2007-03-31
Linux installs a firewall automatically.
It is called IP tables.

However, if you want to see what is going on and if you need to make some alterations to permissions, it is a good idea to install a package called "FireStarter"

It is in the Synaptic repositories and is GUI based.

You may want to consider adding a virus scanner if you decide to run Windows software under Wine.

Also, if you regularly forward on received mail to Windows users you may also want to consider using anti-virus software to protect them.

I find that Avast is quite good.
