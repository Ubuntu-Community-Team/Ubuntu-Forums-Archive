### Post by tophcito on 2008-01-21
Just my opinon:

I can confirm that OpenOffice.org is quite stable. It used to complain a lot about a misconfigured java install on my system, but that was my and not OO.o's fault.

Last time I checked, doing multimedia presentations, including video clips were quite a hassle. Because my hardware is rather outdated, I use OO.o only for viewing MS Office documents, people keep sending me. I had never any import problems either. 

IMHO, OO.o is quite resource hungry, so if u like a very responsive application, perhaps one of the smaller ones is more to your liking. Espescially if you are stuck with older hardware.

AFAIK, Ubuntu comes loaded with Accessibility features like screen reader or speech output, and these should work then together with OO.o
