### Post by Iowan on 2006-03-23
[QUOTE=clyndab]My question therfore is when do I get that option? Please make any answers in simple language and not linux speak at this stage.[/QUOTE] After selecting keyboard, timezone, and a few other things, the install will eventually get around to partitioning the hard drive.  THIS is when your spider-sense should start to tingle.  Be VERY cautious about any default selections, since the default is (probably) to install on the first (master?) drive. If you select either guided or manual, you can see what's on the drive before it gets committed - also lets you adjust sizes if you wish.
