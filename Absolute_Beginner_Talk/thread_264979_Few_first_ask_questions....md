1.Being 14 will I be able to use the program efficiently without getting confused and cross? A lot of people get confused and cross using Linux, or any operating system for that matter. Being so young, you may be at an advantage having not developed bad habits from decades of use of other operating systems. From my own experience, and from reading about others' problems on this forum, the absolute hardest part about using Ubuntu is **unlearning** the bits that you think you already know.
> 2.In linux ive been told on numerous occasions that you can program your own applications, programs, games. Is this true?
Yes.
> 3. If the answer to number 2 is yes could learn it or is it too hard.
I've not tried it myself. If you have an aptitude for programming, then it will probably be easier to write applications for Linux than for any other operating system, since the tools to do so are freely available, and there is a vast amount of code that you can look at for ideas. Also, a higher proportion of the people that you'll find in Linux fora to discuss your programming problems with will be similarly inclined. A random Windows forum, for example, will have very few programmers.

If you don't have an aptitude for programming, you'll find it hard. On any platform.

> 4.Will I be able to do most normal things on there just only using diffrent programs to perform them with?

If I understand your question correctly, then yes, you can do pretty much all of the things that you can do in Windows, but using different applications - Totem instead of Windows Media Player, for example. In my case, I found that I was using the exact same applications - Firefox, Thunderbird, Open Office.org, ScummVM and so on.

Welcome to the Ubuntu forum, and good luck with your experimentation. I hope you find it as rewarding as I have.

### Post by Aelfric5578 on 2006-09-25
I already wrote this in one of the other identical threads you started, but since this one has more replies, I figured you'd be more likely to read my answer here...

[QUOTE="Aelfric5578"]Well, I started using linux a few years ago, and I was only two years older than you are at that time, and I've managed to use it.

1. Your age has nothing to do with whether or not you'll be able to learn to use Linux. If you're 14 now and you've been using Windows since you were very young (I had been using various versions of Windows since I was 5 or 6), you will find that you will have to learn a different way of doing things. It isn't any harder or easier. It's just what you are used to. My advice is to start small though. Don't expect to be able to use it as well as you use Windows the first time you start your computer. I've been using Linux for two years now, and I would definitely not consider myself an expert. As something comes up, I search the forums or I use google, and I learn something new.

2. Yes. You can program applications, games, and stuff with Windows too though if you have the right tools (in this case, programs).

3. Is programming easy? Well, it is and it isn't. I started programming when I was about 10 or 11, and I picked up the basics quickly. But the first few programs I wrote were very, VERY simple. (For example, my first really interesting program was this two frame animated alien that 'spoke' whatever you typed in. By 'spoke' I mean the mouth moved and the message appeared one character at a time.) Now, I'm studying programming in college, and I'm writing more impressive programs. You're probably not going to read a book on programming and make the next great first-person shooter all in one day, but given a few weeks, you could make a fun but simple game. When one of my friends ws fifteen, he made this game where apples fell from the sky and you had to catch them in a basket. Two days after he showed it to someone, everyone in our class was playing it.

4. That depends on what you mean by normal things. Unless you have some hardware problems, using the internet is easy. If you use Microsoft Office for school, you can probably do everything you do now using OpenOffice. If you like listening to music or watching movies, things become a bit more difficult--but not impossible. The only real problem I could see is if you like PC Gaming. Most of the games you may have bought for Windows won't work properly in Linux, but there are some that will (at some point you'll probably learn about something called Wine or Cedega) and there are games written specifically for Linux.

I hope I've answered your questions. If you want to know anything else or run into any problems, just ask. In my experience this community is very friendly and helpful.[/QUOTE]
