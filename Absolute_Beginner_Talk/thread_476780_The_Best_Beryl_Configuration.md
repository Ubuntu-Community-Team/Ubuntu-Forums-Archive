LOL! To each his own. Everyone likes it different. Spend an hour or so and try every single setting. That will get ya to the best configuration.

I do have a few tweaks for beryl that make it run better:

Open up Beryl settings manager from the icon and change the following settings on the general options tab (main one selected).

Texture Filter > Fast (you won't notice much graphical difference, becomes smoother on older cards)
Level of Focus Stealing Prevention > None
Detect Refresh Rate > Uncheck
Refresh Rate > Input yours manually, whatever it is, beryl sometimes assigns it wrong.
Sync to VBlank > Uncheck

Those make it run better, as for options turn on and off whatever plugins you like. [WIki to them here.]("http://wiki.beryl-project.org/wiki/Main_Page")
