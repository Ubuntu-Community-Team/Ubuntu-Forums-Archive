I know, I know!

What it is is one of these fancy U3 drives. If you use this drive in a windows machine, you'll see there is a fancy little program in the drive that is supposedly there so you can install actual software to the key and carry it all installed and everything over to other windows machines. Myself, I think this is kind of a silly idea. What I did was go to the Sandisk website, and did a little searching for a program that uninstalls this U3 feature. I can't remember if it's possible to reinstall it at a later date if you want to, so you might want to check.

I just got rid of the U3 thing and never looked back - formatted my whole stick to FAT32 I think and it works great. No more CD icon and I have use of the whole memory stick.

I guess the CD icon is just Ubuntu's way of recognising that U3 gadget - I don't know the technical specifics of how the whole thing works.

Try a search for 'uninstall sandisk U3' to find out more about getting rid of it if that's what you want.

Regards,

Kolin
