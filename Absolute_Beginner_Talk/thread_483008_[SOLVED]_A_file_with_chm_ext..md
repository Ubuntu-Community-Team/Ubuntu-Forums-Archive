In a terminal, type: ```
sudo aptitude install xchm
```

### Post by Unterseeboot_234 on 2007-06-29
Many times I find I have to fire software from the terminal. I remember this eDoucment reader program with a play on words: "xchairman" and think of Bill Gates leaving MS in 2010. In a terminal, at the prompt:

```
myUsername@myUserName-desktop ~$ xchm
```

that brings up a graphic GUI with buttons that let you navigate to the file. If the file still won't open, then you might have a password issue. I know I downloaded some free eBooks and they come with the password "spyderEdocs". Good Luck!
