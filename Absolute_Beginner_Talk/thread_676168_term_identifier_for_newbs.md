### Post by emarkd on 2008-01-23
Gnome is the desktop environment in Ubuntu.  It's what you graphically interact with.  It's actually separate software than Ubuntu.  See [www.gnome.org](www.gnome.org) for more details.

awn stands for avant window navigator.  it's a toolbar launcher like the mac osx dock.  see [http://code.google.com/p/avant-window-navigator/](http://code.google.com/p/avant-window-navigator/) for more details.
