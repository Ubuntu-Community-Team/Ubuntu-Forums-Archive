### Post by drummer on 2005-09-07
To run KDE apps in Gnome, you don't need the whole Kubuntu-desktop and all the kde desktop packages. If you install a KDE app such as Amarok or some such, synaptic/apt-get will also install the KDE packages it needs. These are usually just some libraries of data, I installed Amarok recently and the extra libraries for KDE apps was about 60mb I think. The same goes the other way around (Gnome/gtk apps in KDE)

Hope that helps and is kinda clear...
