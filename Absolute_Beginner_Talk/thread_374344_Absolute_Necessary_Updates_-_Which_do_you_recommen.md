### Post by Sitix on 2007-03-02
There is no such thing as unrecommended updates. If you ask me, all of them are necessary, since all programs might contain bugs and therefore all programs COULD cause your system to miscalculate or loop/freeze.

So it's no use to just install a part of all updates, as you will need all of them. Especially if you want to be able to use all new stuff surrounding Ubuntu. For example, you don't need tlc/tk 8.5, since 8.4 works fine too. But if you try to run aMSN with 8.4, the interface is really ugly and the letters are barely readable. Compiling with 8.5 makes it a smooth appearance, though.

It is not a matter of not needing updates, because in my opinion you need them all.

Anyway. I'd update Nautilus, download the Linux Headers, use Automatix to install the video/audio codecs and use XMMS as the only audio player. Then I would just update everything having to do with the internet, as that is the most vulnerable point of your computer. The rest would be unnecessary, I suppose.
