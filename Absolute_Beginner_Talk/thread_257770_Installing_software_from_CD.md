### Post by 3rdalbum on 2006-09-15
Have you extracted the .gz file? If you double-click on it, it will open in Archive Manager.

Extract the file directly into your home directory. Now find its icon, right-click it, and go to Properties. Go to the "Permissions" tab, and turn on the Execute checkbox next to the "Owner" row.

Go into a terminal and type:

```
./scanModemSept15
```

I'm not familiar with Winmodems, so I don't know what you should do next. Maybe that program will tell you.

[http://linmodems.technion.ac.il/first.html](http://linmodems.technion.ac.il/first.html)
