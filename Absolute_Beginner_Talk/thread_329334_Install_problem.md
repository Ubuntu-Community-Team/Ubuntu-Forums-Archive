### Post by meng on 2007-01-01
Bzzzt you burned it wrongly. That "burn bootable disk" option is a very common mistake (so don't feel bad, you are in good company!) You need to burn FROM the iso file to create a complete filesystem on the CD, that filesystem is inherently bootable so you don't need to specify this. What happened was the infrarecorder installed a version of Caldera DOS onto the CD in addition to the iso file.
[www.psychocats.net/ubuntu/iso](www.psychocats.net/ubuntu/iso)
