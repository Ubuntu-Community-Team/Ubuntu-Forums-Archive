### Post by jordanmthomas on 2007-11-24
DIdn't you already post this once?  It is rude to post the same problem in multiple threads.

Anyway, to install a gdm (login) theme, go to system --> administration --> login window and then to the themes tab.

To change the ubuntu logo to an apple one, you have to go into your theme's folder and find the distributor-logo.png and replace it with the apple logo.
The icon folders are found in 
```
~/.icons
```
and ```
/usr/share/icons
```
(you'll need to use sudo / gksudo to access the ones in /usr/share
