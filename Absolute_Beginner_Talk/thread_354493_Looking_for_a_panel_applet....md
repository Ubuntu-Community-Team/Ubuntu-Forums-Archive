### Post by gradedcheese on 2007-02-06
you might want to try 'deskbar' (right-click a panel, choose Add to Panel, select Deskbar).  It does what you want (and a bunch more).  If you want it to only do the URL thing, right-click it and choose Preferences and uncheck everything except for the web page feature.
