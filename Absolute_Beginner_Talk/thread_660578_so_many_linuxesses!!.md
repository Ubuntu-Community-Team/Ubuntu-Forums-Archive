FYI : The server edition of Ubuntu (and Centos) has no GUI by default, command line only.

I suggest you install a desktop and then add the server software you need. Once you are comfortable with the new OS (ie Linux) and the command line you can lose the GUI.

I have experience with both Centos and Ubuntu as a server and both are rock solid.

Last, under the surface, once you learn the command line, most distros are fairly similar.
