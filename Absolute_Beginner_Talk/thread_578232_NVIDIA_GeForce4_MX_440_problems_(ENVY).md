### Post by southernman on 2007-10-17
Post the output of "sudo nano /etc/X11/xorg.conf"

My graphics card is an older ATI so I don't recall what it should show for a nvidia card... it should have detected the card properly for that MX440 though.

As to your refresh rate issues, you could do the following:

> 
sudo cp /etc/X11/xorg.conf /etc/X11/sorg.conf.backup10172007

Now run

sudo dpkg-reconfigure xserver-xorg


Before running the reconfigure command above, google (or go to your monitors manfufacture website) your monitor model number for the specs of it. Use the refresh rates and resolutions for each.

Typically, if your unsure about what to use as settings during the reconfiguring process, using the defaults is safest to work with. Although, since you first backed up the conf file... experimentation is ok. Just be prepared to boot into recovery mode or from a live cd to reverse the backup procedure above.

Hope that helps some...

In reply to your email, Envy is not available in Ubuntu's repositories but only on my website.

If you want to use Envy you will have to follow these easy steps:
Dowload [this package]("http://albertomilone.com/ubuntu/nvidia/scripts/ubuntu/envy_0.9.8-0ubuntu7_all.deb") (Envy New, which works on Gutsy) and then double click on it. [make sure you have an Internet connection available]
Select "install"

Then you will be able to launch Envy from the "Applications/System" Tools menu.

You can use Envy's "Install the Nvidia driver" function and let Envy do the work for you. It will also ask you to restart your computer.

If, after the restart you still don't have the resolution you need, you will have to type:
```
sudo nvidia-settings
```

and set the resolution from there.

[B]NOTE: if something goes wrong, you will have to boot in Recovery Mode (which you can select from the GRUB menu almost as soon as you turn on your computer) and type:
```
envy --uninstall-all
```
and then type "reboot".
In this way you will undo what Envy did.[/B]
