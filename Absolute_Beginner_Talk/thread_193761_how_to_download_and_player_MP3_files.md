Okay, I can't help you with peer to peer (that's what Kazaa is) as I've never used it. But I can give you some advice on finding mp3 files with google.

Enter a search argument like this:

```
intitle:index.of .mp3 reggae
```

That would turn up directories on the web containing mp3 files to do with the word reggae.  You can look for genres, or particular bands, by changing the search keywords.

I've got a pretty crummy internet connection, using a mobile phone as a modem, so this is how I find music.  There's a lot out there.

### Post by nalmeth on 2006-06-10
If your son wants to use the kazaa network, then try apollon, with the extra network plugins. You may see a small slowdown on your system if you're using gnome, as it is a KDE app. I'm satified with its performance though, and I love the app.

here's install instructions.

Open a terminal (applications - accessories - terminal)

sudo apt-get install apollon gift libfasttrack-gift

when you run apollon, it will ask where your gIFT plugins are. They are in /usr/share/gift

Satisfy yourself as to the content that is available on these networks. There a lot of porn that comes up if you search for video. I don't think apollon has a built-in filter to filter that crap out. Gnutella is the limewire network, fasttrack is kazaa.

If you'd like an app with a filter on it, try limewire or frostwire or something, but you'll find that there is a lot less available on these singular networks.
