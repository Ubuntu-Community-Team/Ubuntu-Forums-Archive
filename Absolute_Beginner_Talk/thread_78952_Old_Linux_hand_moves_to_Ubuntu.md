### Post by skar42 on 2005-10-19
Hi all,

I've been using Linux way back since kernel 1.1.59. When Slackware came on floppy's and hand writing your own X86Config file was the only way to get X windows running. <Ahhhh... the bad old days!>:rolleyes:

I've been using Debian and Redhat for the last 6 years. Running a server/firewall plus two Cobalt Qubes. Just moved over to Ubuntu for a look and boy am I happy. Great looking, easy to install and the hardware detection was fab. I had a little problem with my Nvidia 6800 card on the initial startup, but once I'd changed the X conf and installed the latest Nvidia drivers all was sweet.

I run the 6800 over DVI to twin Dell 2001 FP LCD's. The twinview worked as soon as I got the drivers installed. Nice....:D

Anyway, its all good news from me. I'm about to trash my Dell X300 laptop and install Ubuntu. I'll post up my results into the laptop forum.

Bye for now...
