### Post by UnixAnt on 2007-07-10
The first place I would start is at your router's config page.

In a nutshell, you will need to find out within your torrent software which port is being used, then forward this to the IP address of your PC via the router's configuration page.

An excellent resource for help and guidelines on this is:

[http://portforward.com/]("http://portforward.com/")
