### Post by Shay Stephens on 2007-08-01
I'm a photographer and currently I am using the following:
Bibble Pro - RAW editing
Gimp - batching web galleries and general editing
Photoshop 7 - Layouts requiring text manipulation (run via wine)
Memories on tv - Slideshows (run via wine)
Qdvdauthor - Authoring DVD's (really unstable right now)
K3b - Burning DVD's 

As Gimp makes progress, I hope to use it more an more.  Keep an eye open for improvements in ufraw, which works with gimp to do raw editing.  It is too rough for me to use currently, but I hope to use it in the future.  So for now I am using Bibble Pro to do my raw editing.  Photoshop 7 has all the tools for making page layouts with text that I need.  Gimp treats text in a very bizarre way making it extremely labor intensive to make and work templates.  But Gimps scripting support does make it a good solution for batching web galleries, so I use it primarily for that right now.

Similar with slideshow creation.  So far I have not worked up a working solution for making a slideshow and rely on Memories on tv, a windows based program, to create my slideshows.

### Post by bogolisk on 2007-08-02
> **undertakingyou said:**
> I like the gimp, and I have installed "gimpshop" because the menus make more sense.  One thing to remember though.  The gimp is not photoshop.  It is not equal.  It is comparable and the gimp can be just as powerful as photoshop if you learn to use the tools there.  

In all though, I like the gimp a lot, and I use both CS2 and The Gimp.

About the UI: I'm more familiar with gimp interface so I felt really lost in CS2.

About the "power": No Gimp is not in same league as CS2. But for the majority of users, it would make absolutely no difference. What's not avail in GIMP: good shadow-highlight tool; 16bit channels; easy to use LAB mode, etc.

About the "ease of use": there's a lot more hand-holding in CS2 than gimp. There's also more plugins for CS2 than for Gimp. And before anyone would mention it: pspi stopped working with gimp 2.3.1x on linux.

Again, for the avg Joe, gimp is more than good enough, IMHO.
