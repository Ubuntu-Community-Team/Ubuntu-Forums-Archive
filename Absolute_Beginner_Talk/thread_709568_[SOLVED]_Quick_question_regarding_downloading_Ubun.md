Application or program management is done almost entirely off the internet. So if you're installing a new program, Ubuntu is fetching the software packages off the internet. If you're getting security updates, Ubuntu is fetching the updates off the internet.

Ubuntu without an internet connection leaves you with the default installation and not a whole lot else.
