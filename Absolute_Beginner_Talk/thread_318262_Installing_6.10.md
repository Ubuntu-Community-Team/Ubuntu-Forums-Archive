### Post by Kiwi Si on 2006-12-13
Hi there,

I installed v6.10 yesterday onto my PC that already had XP installed on it.  I have the 200GB hard disk partitioned into two.  100GB of it I was already using for XP and the other 100GB was free/unpartitioned.

When I went to install Kubuntu it gave three options when it came to where to install it.

1. Delete/Clear the entire disk and install it on this freshly formatted disk.
2. Install on the largest unassigned space on the disk **This is the option I went for.
3. Manual - didn't go into this too much though am guessing you could go in and remove/add partitions, etc.

After going through the entire install process and then rebooting you are then given a boot menu selection screen where you can either select booting into Kubuntu (3 different types of options) or Windows XP.

Hope this helps,
