Every Linux distribution I ever tried (which total more than two dozen) ran significantly faster than Windows on this hardware. The GIMP (Graphics Image Manipulation Program) is a powerful image editor that rivals Photoshop.  It's interface takes some getting used to but I've been using it for years to great effect.  I now prefer it to Photoshop since it's faster.  Furthermore, there are packages available for both Windows and Mac.  If you prefer Photoshop's layout, then you'll find GimpShop easier to learn.  It's a fork of GIMP that has been rearranged to mimic Photoshop's menus and overall look and feel.

### Post by aysiu on 2007-05-09
X is the graphical server that allows window managers to run. Pretty much any program that will run in any desktop environment (Gnome, KDE, Xfce) will run in a window manager (IceWM, Fluxbox, OpenBox, Enlightenment, WindowMaker) and vice versa.
