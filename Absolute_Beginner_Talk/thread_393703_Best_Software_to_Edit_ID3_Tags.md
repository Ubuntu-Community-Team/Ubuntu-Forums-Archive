### Post by gmendoza on 2008-07-07
> **euthyfro said:**
> it reads id3 v.2 tags, which easytag is set 2 write.  the strange thing is that the mp3s on the player that have tags it recognizes show up in easytag in red right off the bat (as it: this tag has been modified would you like to save it?)  i've been saying don't save for fear that if i do it will make them unrecognizable as well.

EasyTag has a default setting under "Settings... Preferences... ID3 Tag Settings" named "Automatically convert old ID3v2 tag versions".  Also by default, EasyTag is set to tag in the 2.4 format.  So when you browse any collection using 2.3, EasyTag queues up a modification and displays your tracks in red letting you know there's a change to be made.

If you don't want either of these settings, feel free to adjust.
