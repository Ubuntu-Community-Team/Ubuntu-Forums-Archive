### Post by El Lance-O on 2007-08-10
To install Steam under WINE, follow [these instructions]("http://appdb.winehq.org/appview.php?iVersionId=3731").

And as for the font problem, you have to install tahoma.ttf into the WINE font directory, which is in ~/.wine/drive_c/windows/fonts I do believe. You can find that font using Google, and for future WINE tutorials for certain programs, be sure to check out the [WINE Application DB]("http://appdb.winehq.org/") which will help you with pretty much anything.

As for Cedega, I have no experience with it because the whole paying money thing made me lean more towards WINE, and I've never had a problem with it so far, aside from a few hours here and there trying to tweak things just right. :)

Good luck!
