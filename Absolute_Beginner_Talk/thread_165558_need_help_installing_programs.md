### Post by Bloch on 2006-04-24
Hi Proboller86;
Welcome to ubuntu. As a general rule, to see what software is available for ubuntu first check the Applications --> Add Applications menu and see if there's anything you fancy. All this software is part and parcel of ubuntu and more or less guaranteed to work.

Then (taking a look at the recommended guide first) open synaptic and enable the universe and non-free repositories. These are all well-tested progs, but might have problems, might want to uninstall other packages etc. And they might not place an entry in the menu, which is a real bummer. But generally they install no problem.

Only then look farther afield. At the beginning it was a bit of a turn-off for me that programs found at random on the web rarely worked. I was trying to compile stuff, and there was no "warning" that it might be difficult. So to stock up with good software, check out the first two methods, and only then stray farther afield.
Here's a brand new illustrated guide: 
[http://monkeyblog.org/ubuntu/installing.html](http://monkeyblog.org/ubuntu/installing.html)
