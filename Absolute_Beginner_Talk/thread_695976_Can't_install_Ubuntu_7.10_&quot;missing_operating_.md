### Post by Therion on 2008-02-13
So you want to say goodbye to both your Windows XP and Vista partitions?  Forever?

If so I would suggest you reformat and reinstall Ubuntu.

Reboot from the LiveCD and when prompted, choose the "Guided: Use Entire Disk" option like you see in this [screenshot]("http://aycu03.webshots.com/image/32522/2001738602340396146_rs.jpg").  Your drive configuration will look different, so don't worry about that.  It just sounds like you need a thorough reformat and reinstall to get Ubuntu up and running.
