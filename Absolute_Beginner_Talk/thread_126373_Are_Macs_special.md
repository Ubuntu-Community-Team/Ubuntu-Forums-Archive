### Post by dickohead on 2006-02-06
It's not so much the tech specs of a Mac that make them great, rather the user experience, the user base and the types of applications being used on them.

I think we can safely assume three things:
  Linux users are more tech savvy
  Windows users vary from n00bs the the very 1337
  Mac users are more artistic/in touch with design (standards based web developers anyone?)

Because of the nature of Apples target audience, the system has been built to accommadate their needs and functions, for example: Since Mac users have classically been into art/design they require programs like: Premier, Dreamweaver, Photoshop, CorelDraw, 3DSMax, Maya etc etc etc, so you will find that these programs are built on Mac, for the MAc and then ported/rewritten for Windows.

You might also be aware of the fact that the Intel Pentium Processors have very large MHz ratings when compared to AMD, but run the processors along side one another and you soon realise that MHz these days means stuff all.
