### Post by GreyFox503 on 2006-01-16
If you burn the file 'ubuntu-5.10-install-i386.iso' to a CD as data (i.e. the normal way), you will end up with a CD that has one really big file on it:  the one you downloaded.  Try popping in that disc you burned into your computer (with Windows already running).  You should just see that one file "ubuntu-5.10-install-i386.iso" sitting there.  That doesn't do you much good. :)

However, that thing you downloaded wasn't just a file.  It was a disk image, which is a complete picture of how a finished Ubuntu disc should look, all wrapped up in one file.  So when you burn it as an image (most all burning programs have an option for this), it will create an entire, fully bootable Ubuntu install disc for you.  After you do this, try putting the new disc in your computer.  You should then see a whole bunch of directories, files, and subdirectories.  (The disc might even auto-run under Windows, I'm not sure).

The disc image is an easy way for you to get a genuine Ubuntu CD.  Other than a pretty label, the disc you burn should have the same contents as the ones shipped in the mail.

Have fun with your computing!
