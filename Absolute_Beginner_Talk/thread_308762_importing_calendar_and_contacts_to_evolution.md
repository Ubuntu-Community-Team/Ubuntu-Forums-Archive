### Post by Handssolow on 2006-11-28
I'm sure others will give a fuller reply but reading in the Forum I learn that it can be done using Thunderbird mail as a tool.

My wife wanted her Outlook Express contacts transfer from XP to Ubuntu. I downloaded Thunderbird when in XP but I didn't configure it to take the emails. Lots of boxes which I had cancel or quit. I could then in Thunderbird import the addresses from Outlook Express then give a name to the file and save them onto a USB memory stick.

A reboot into Ubuntu's  Evolution then import the Addresses from the memory stick.
