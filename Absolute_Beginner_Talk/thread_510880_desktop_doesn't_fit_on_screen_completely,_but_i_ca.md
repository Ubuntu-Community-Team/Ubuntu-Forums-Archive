### Post by Mornedhel on 2007-07-27
Just saving the xorg.conf is not enough, you have to restart your X server for the changes to take effect. The safest way is to log out (no need to reboot), then press Ctrl Alt Backspace, then pray your changes have not hosed everything. If they have, Ctrl Alt F2 takes you to a console session where you can restore your xorg.conf, then Ctrl Alt F7 takes you back to the X session (don't forget to kill the X server again so it takes the changes into consideration). In my opinion, xorg.conf is one of the things that most needs additional user-friendliness.
