### Post by aysiu on 2007-03-06
> **icelechi said:**
> Is there any way to remove rhytmbox or any of the programs that come with Ubuntu Dapper without removing ubuntu-desktop?
No, because *ubuntu-desktop* is defined as the default set of applications that come with Ubuntu.

If you don't want the default set (i.e., you remove one of the default applications--like Rhythmbox), then just get rid of *ubuntu-desktop*.

Keep in mind that if you're planning to upgrade, you should keep *ubuntu-desktop* installed, though.
