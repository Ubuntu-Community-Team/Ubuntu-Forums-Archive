### Post by sonofusion82 on 2007-06-21
if you just need run a few apps, perhaps you could try out Wine if it works well with your windows apps, I am not sure about Office 2007 (hey, why not use OpenOffice?), but you can easily find resources on the net on how to install IE6 on Wine.

for full virtualization, there are many options available such as VirtualBox, QEmu, VMWare and others. if you have a newer machine with processor that supports hardware virtualization, look out for emulators that support them as it provides great speed improvements.

personally, I have mostly used M$ VirtualPC 2007 (free for Windows) at work for software development testing. I have not really tried much virtualization stuffs at home on linux as the processor is a little old thus doesn't take virtualization too well.
