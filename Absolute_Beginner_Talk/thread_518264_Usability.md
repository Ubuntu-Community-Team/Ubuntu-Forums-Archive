If you tell us what you plan on using your computer for, we can tell you whether Ubuntu will suit your needs :)

You can try downloading Ubuntu right now, though - the download works as a LiveCD, so you can boot from it and try Ubuntu out without having to install it. The variations are as follows:

Ubuntu - uses the Gnome desktop environment, which has two panels (one for menus, one to keep track of windows / programs you have open).

Kubuntu - uses the KDE desktop environment, which is a bit similar to Windows in terms of first appearances, but is pretty different once you get used to it (different in a good way, though :P)

Xubuntu - uses the XFCE desktop evironmnt, which is similar to Gnome but has some key differences. It's fairly lightweight, which makes it great for older computers.

However, if you fancy trying out each of the environments, you can just install them too and choose which one you wish to use when you log in. There are many different types of GUI for Linux, so you can keep searching and tweaking til you find one perfect for you.
