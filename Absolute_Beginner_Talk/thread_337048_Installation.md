### Post by meng on 2007-01-12
Your system is a little underpowered to run Ubuntu or Kubuntu. However, Xubuntu would probably be okay, and Fluxbuntu almost certainly. You need to make sure you download the Alternate Install CD, not the Live/Desktop Install CD. The installer can preserve Windows partitions if you wish. However, if you don't want to keep the Windows partition then you can overwrite it with a Linux partition, but you will lose your data. So backup, backup, backup!
