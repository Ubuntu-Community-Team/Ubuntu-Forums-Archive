Ubuntu Forum Readers,

Let me first say thank you to all who are involved with this project and what it stands for. I am a huge fan of all of the OpenSource projects. I used Red Hat early in its production around 2000 and was impressed but not ready to jump off the boat/island that is MS. Discovered Ubuntu 5.10's Live CD and thought boy if this only worked on my laptop I would love to use it... I honestly forgot about Ubuntu until early 2007 when our MS Small Business Server pooped after a failed Service Pack update and then after three days of beating myself over the head I got it going with help from outside support $$. I downloaded Ubuntu server but because I am not a great command line type I sat and stared at a blinking curser and gave up after a couple of days trying to create a backup mail server to replace the MS one should something happen again... 

To my surprise the open source world was still out there working on great projects and gaining momentum to beat the band and Ubuntu was leading the charge so lets try again with a desktop install... downloaded the 6.x version and since that was so close to the release of the promising 7.04 Feisty Fawn (love the names) I will also download that tomorrow... Downloaded the live CD version of both and messed around trying to get them to run on my 1.13 ghz IBM Stinkpad A30 w/384 mb ram XP Sp2 40GB WD HD blah blah with little luck getting the new kernel to even boot... so I grabbed the old 5.1 disk and it ran OK so maybe 6.x will... nope. Poop now the XP won't boot... Message boards... downloaded Partitioning and SuperGrub... got the xp back but its the weekend and I don't have DSL at home besides it is going to be Spring this weekend... Monday came and I tried again... Failure to load... Message boards... tried numerous boot options and no luck getting the kernel to load... Alternate download ... loading it yippy... corrupted files??? ... messege board ... try another burn check sum ... load again ... corrupted files ??? ... Message boards ... redownload overnight so not to use all the bandwidth work has left ... Tuesday try again ... corrupted files?? Arghh! ... No XP so used magic Supergrub disk to fix ... did some actual work on XP and tried again ... Corrupted Files ??... Redownload different source burn extra slow on Wednesday morning ... reinstall ... Corrupted ... Give up for now until I can get some different CDs had been trying with RWable ones and not getting anywhere... Message board to write my dissertation... end.

So, although I didn't have a ton of luck the boards have been great and I have learned more in my failure than I would have if things went perfectly. I was able to use the Live CD on three other desktops just not the laptop, oh well. This operating system is so clean and easy that I will be moving my mom and dad's win 98 machine to it at some point in the near future. They don't use anything other than Solitaire (Dad) and MS Word with photo software and printing of pictures. This operating system is built for this from what I can tell just going to need the correct drivers for the printer scanner hardware and they will be ready. Oh and I haven't stopped trying the laptop either. I will not fail at this! Thanks for taking the time to build this and best wishes all for the overtaking of the OS world should you so choose or even if you don't choose and it just happens because your product is better!
