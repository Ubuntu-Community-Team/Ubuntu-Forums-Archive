### Post by louis_nichols on 2007-02-26
You can just install the desktop version. Then, installing LAMP is as simple as 5 clicks and typing a password. :) The desktop version can be installed via the liveCD installer or the alternate installer. Installers are [here]("http://www.ubuntu.com/products/GetUbuntu/download#currentrelease")

Specifically, in synaptic, there is an option Edit>Select packages by task. There you can just check LAMP and it will select all relevant packages.
