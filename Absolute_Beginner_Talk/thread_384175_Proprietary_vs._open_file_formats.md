### Post by nalmeth on 2007-03-14
I grabbed this link in like a second, there is likely a better reference directly for open/closed formats, but its a good start: [http://en.wikipedia.org/wiki/List_of_codecs](http://en.wikipedia.org/wiki/List_of_codecs)

Essentially, the legal, good free + open formats are:
OGG lossy audio
FLAC lossless audio (my favorite)
OGG Theora Video 
[http://en.wikipedia.org/wiki/Open_source_codecs_and_containers](http://en.wikipedia.org/wiki/Open_source_codecs_and_containers)

mp3, aac, wma, wmv, avi, mpeg, etc, are non-free

### Post by ferg on 2007-03-14
Well MP3 is a lossy format as well, so it's just a case of compounding the quality problems. I prefer to use FLAC, as does nalmeth, due the the fact that there is no loss in quality. I keep lossy versions of all my music at low bitrates for my iPod, however. 
You might like to check [http://www.openformats.org/](http://www.openformats.org/) ( [http://www.openformats.org/en9](http://www.openformats.org/en9) in particular) as it provides some useful information about common file formats and the nature of their open or closed source. It also provides about which formats are the best to use. 
 
Hope this is of some help! :)
