### Post by 3rdalbum on 2008-04-15
You can actually use Nautilus (which is Ubuntu's built-in file manager) to access FTP servers.

Places > Connect To Server. Change the type to "FTP (with login)". Put in the relevent details, and type "web page upload" under "Name for the connection". 

Hit Connect (or is it Browse Network?) and in a few seconds you'll see a new file manager window open with your FTP site in it. Just drag and drop files onto it as though it was a local folder.
