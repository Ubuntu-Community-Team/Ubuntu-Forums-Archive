### Post by dfego on 2007-07-13
Yes, your wireless should work if it worked on the LiveCD.  It's actually a good practice to test a particular machine with a LiveCD if possible before installation, so you can be sure hardware will work.  If it doesn't work on the LiveCD, that doesn't mean it won't when installed (with tweaking), but if it does work on the LiveCD, it should work on the install just fine.  Not to mention I have that wireless card and have used Ubuntu on it before and it worked fine. :)
