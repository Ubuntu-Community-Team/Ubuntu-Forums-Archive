### Post by happyhamster on 2008-03-05
Yes, memtest just runs forever. Note that it's recommended to let it pass the entire set of tests a few times (because there are long-term tests running in the background). I once had a bad stick of ram that would only show errors after testing at least 5 whole cycles (but it could take even more time than that).
