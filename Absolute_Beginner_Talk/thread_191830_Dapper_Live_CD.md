### Post by mcduck on 2006-06-08
1. download the image
2. check the MD5 checksum for the image
3. Burn the image to disk (with 'burn image', not as a file) with low speed
4. boot the machine, and enter to BIOS settings
5. Make sure that CD drive is the first in boot order, before your hard drives
6. insert the Ubuntu CD and enjoy.
