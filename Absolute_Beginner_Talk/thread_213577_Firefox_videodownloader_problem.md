### Post by bruenig on 2006-07-11
Try [download embedded](https://addons.mozilla.org/firefox/1993/). It does the same thing as video downloader. I've used it for a long time and never had any problems. If there is an embedded video in a website, a red arrow appears on the status bar. You click that and hit view embedded items and then it will show the embedded item, just select it and click download.
