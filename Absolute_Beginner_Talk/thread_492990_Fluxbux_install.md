### Post by RedSquirrel on 2007-07-05
The fluxbox package is in the repositories. You can install it with Adept or just do:

```
sudo apt-get install fluxbox
```

IMPORTANT:

After you have installed Fluxbox be sure to create your basic Fluxbox menu with this command:

```
sudo update-menus
```

Check out the links in my signature for more help with Fluxbox.
