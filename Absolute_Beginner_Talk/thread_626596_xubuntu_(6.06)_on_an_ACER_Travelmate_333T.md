### Post by luisfcup on 2007-11-30
Ok, change in Section "monitor":

```
Option "MonitorLayout" "LVDS,**AUTO"**
```

Then in the Section "Screen" add the resolution you want in SubSection "Display" like

```
Section "Screen"
Identifier "Default Screen"
Device "Trident Microsystems Cyber 9525"
Monitor "Generic Monitor"
DefaultDepth 24
SubSection "Display"
Depth 1
Modes **"1024x768"** "800x600"
EndSubSection
SubSection "Display"
Depth 4
Modes **"1024x768"** "800x600"
EndSubSection
SubSection "Display"
Depth 8
Modes **"1024x768"** "800x600"
EndSubSection
SubSection "Display"
Depth 15
Modes **"1024x768"** "800x600"
EndSubSection
SubSection "Display"
Depth 16
Modes **"1024x768"** "800x600"
EndSubSection
SubSection "Display"
Depth 24
Modes **"1024x768"** "800x600"
EndSubSection
EndSection

```

Save and restart the X with <Ctrl>+<Alt>+Backspace
Don't forget you need root access to change the xorg.conf file.

### Post by luisfcup on 2007-12-04
The restricted drivers are proprietary drivers made by the cards manufacture, they are not open source so they cannot be changed. Some times the restricted drivers work fine, and they can give us better performance, but other times there are conflicts with the restricted drivers.
Normally by default the open source drivers are the ones installed. You can try the restricted drivers for you card to see if these solve the problem. In the "System -> Administration -> Restricted Drivers Manager” See if you have your card name there and check the box to use the restricted drivers and see what happens, you may have to edit again the xorg.conf file to see if the modes you want still appear in there.
