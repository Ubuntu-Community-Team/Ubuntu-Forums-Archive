### Post by aysiu on 2008-02-17
It is not possible to *upgrade* with the Desktop CD (or "live CD"). You can, however, do a fresh reinstall with the Desktop CD.

To upgrade from the Alternate CD, follow the instructions at the bottom of [this page](http://www.ubuntu.com/getubuntu/upgrading).

### Post by RomeReactor on 2008-02-17
Hi. It is recommended that you use the Alternate CD for upgrades, but the Live CD should also work. When you get it, insert it on your system's CD drive and Ubuntu will notify you that you can perform the upgrade. Note that you will most likely also need to download at least some packages, depending on how many other programs you've installed besides the defaults.
