After downloading non-synaptic applications, should they be unpacked and installed from a specific folder? I want to eliminate having bits and pieces of unzipped files scattered about in incorrect locations. 

In Windows I keep the zip archive and/or setup file in a companion folder to the working program in case I need to re-install at some future time.

### Post by bodhi.zazen on 2007-10-21
I save all this stuff in ~/src (~ = short hand for your home directory)

```
mkdir ~/src
```

I then make a directory in src for each app and save the code in the event I need to update, patch, or remove.
