### Post by wpshooter on 2006-09-29
Try this.

First of all this is assuming you have another computer to work with.

Go this site and download KILLDISK program and WIPE your hard drive clean on the computer on which you want to install Ubuntu.

[http://www.killdisk.com/](http://www.killdisk.com/)

Then make sure you check the integrity of the Ubuntu Cd by running the verification test on the initial Ubuntu boot screen menu.

Assuming it passed that test, then go to the Ubuntu desktop and double click on the install icon.

Make sure as you are given the install parameters prompts that you let your CD drive spin down and stop between each parameters.  Also, when you get to the parameter regarding the keyboard language, make sure you click on the typing box at the bottom of the page (you don't have to type any thing in it - just click on it) and then click back on the language box at the top of the page, then click on forward.

Make sure you give your CD drive plenty of time to spin down between each prompt.

If the above does not work then WIPE your drive again but this time try the ALTERNATE Ubunut CD instead of the DESKTOP version.

Good luck.
