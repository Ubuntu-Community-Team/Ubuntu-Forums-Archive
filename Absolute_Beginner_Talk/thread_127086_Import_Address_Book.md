### Post by davmac on 2006-02-08
It is a long time since I did this, but, from memory, the steps I followed were;

1) Install Thunderbird on Windows
2) Start Thunderbird and ask it to import everything from Outlook
3) Go trawling through the Thunderbird profile directory looking for the relevant info (in something like c:/documents and settings/userid/application data/mozilla thunderbird/profiles/XXX.XXXXXX)
4) Back up the .mozilla-thunderbird directory on Ubuntu
5) Copy across the relevant data from Windows to Ubuntu.

Took a bit of trial and error (which is why the back up step is so important, but I eventually got everything from Outlook to Windows Thunderbird and then from Windows to Ubuntu. Mind you, I had a six month gap between moving to Thunderbord and moving to Ubuntu.

Hope this helps,
