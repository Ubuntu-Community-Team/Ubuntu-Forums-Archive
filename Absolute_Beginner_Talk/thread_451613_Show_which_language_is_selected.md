### Post by codesplice on 2007-05-22
In GNOME, try right-clicking on an empty space on the panel.  Select "Add to Panel".  Scroll down to the "Utilities" section, select "Keyboard Indicator" and click the "Add" button.  

That should indicate what keyboard layout you have active. 

Hope this helps.
