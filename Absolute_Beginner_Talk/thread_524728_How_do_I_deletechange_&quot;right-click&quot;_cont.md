In Ubuntu 7.04 Feisty whenever I want to open an HTML document that is stored in one of my Tutorial folders or any other HTML document that is stored on my computer and I just "left-click" to open the file the default "open-with" program is Opera.

I would like to change this and make the default "open-with" program Firefox.

I already have on the "right-click" drop-down context menu Firefox and others as options, so if I remember, I will "right-click" and use the appropriate application.

My main reason for wanting to delete/change the "right-click" context menu is that I have created a Launcher to an HTML document and when I click the Launcher Opera fires up.  This is where I want Firefox to be the default "open-with" program for this Launcher.

### Post by jspangler on 2007-08-13
Right click on the file and go to "Properties"

Click the tab "Open With"

Choose which program you want to be the default. If it's not there, add it.

You can also change the "right-click" list here by adding or deleting programs. But the selected one is the default.

---

### Post by jspangler on 2007-08-13
Also, under System > Preferences > Preferred Applications, you can choose your default web browser for html files opened by other programs.
