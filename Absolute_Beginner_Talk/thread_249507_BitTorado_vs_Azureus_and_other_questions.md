### Post by nudnik on 2006-09-02
Bittornado is the only client that works properly for me. Although its great for Windows, there are Java issues with Azureus in Ubuntu that can be irritating to solve, so its best avoided. I am told Azureus works well with the latest Java runtime installed, but I haven't experimented with it. 

It does indeed sound like your router is blocking the connection. Unfortunately, I have no experience with wireless routers.

### Post by jISh on 2006-09-02
As far as native Linux clients, Bitornado is my choice.

As for port forwarding, check [this]("http://www.portforward.com/").
