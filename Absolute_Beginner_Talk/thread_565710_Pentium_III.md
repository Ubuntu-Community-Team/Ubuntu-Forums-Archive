### Post by matthew on 2007-10-02
My old laptop has similar specs, but with 384 Mb of ram (128 + 256). I have Edubuntu running on it, which is basically a GNOME desktop, just like a standard Ubuntu installation. It does okay, but is occasionally slow. It's perfect for my kids to play games on (ages 4 and 5). If I needed to do my work on it I would strip the installation down a little for speed, starting with Xubuntu. K.Mandla, one of our site staff, has done several installations on computers well older than this. Look up some of his threads for more ideas.

Bottom line: try Xubuntu, and in any case, the computer should work fine.

### Post by dwhitney67 on 2007-10-02
I ran Ubuntu on a Pentium MMX with 256MB RAM.  Of course I did not use the Gnome interface for day-to-day use; I only used it briefly until I setup the SSH-server.  From that point on, I connected to it from another system.  I used that Pentium MMX system to compile software packages for the i586 architecture.

So if you are looking to use your system for graphical purposes, I would recommend like the others have suggested to consider Xubuntu.  You can also choose another Linux distro that includes twm (tom's window manager), which I think is even lighter than XFCE.
