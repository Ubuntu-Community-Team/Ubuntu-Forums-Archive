### Post by Paul820 on 2007-10-23
You can set the font rendering in System->Preferences->Fonts in feisty, System->Preferences->Appearance in gutsy. You can also install the msttcorefonts( Microsoft true-type core fonts )  and use them to make you feel more at home. Set the fonts to 'best contrast'

EDIT: If you want smoother fonts set it to 'subpixel smoothing'
