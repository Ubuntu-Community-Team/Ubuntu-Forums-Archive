### Post by LuisGMarine on 2007-04-26
I take it that you are using the Gnome Desktop.  If so all you have to do is right-click on the icon, select ' Stretch Icon ' from the menu and resize the icon to what ever size you want.

Another option would be to increase your screen resolution. I think the default for a Ubuntu install is 832*624 which is outdated for most users. Especially those that spend a lot of time online. I suggest going to System > Preferences > Screen Resolution , and choosing a larger size until finding one your monitor can support, and you are comfortable with.
