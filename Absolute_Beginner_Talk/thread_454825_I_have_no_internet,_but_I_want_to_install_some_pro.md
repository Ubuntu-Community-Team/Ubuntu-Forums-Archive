### Post by chamberlain2006 on 2007-05-25
Simple.  On a PC that has an internet connection go to packages.ubuntu.com, and use the search tool to find the program you want.  Download the program and bring it to you Ubuntu computer.  Double click on the file to install it.  For some programs you may need to install other programs that the program you want needs.
