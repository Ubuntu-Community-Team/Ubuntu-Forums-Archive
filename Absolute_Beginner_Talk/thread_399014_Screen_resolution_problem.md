### Post by Dual Cortex on 2007-04-01
Lets try to make it simple (this is to be followed by users with the nvidia bin proprietary drivers installed):

Open up a terminal and type
> gksudo nvidia-settings

Click on X Server Display Configuration. Are you able to change your resolution there?
If you are then change it to 1280x1024 and clock apply. Then click on "*Save to X Configuration File*". 
Press CTRL + ALT + DEL and see if everything's OK.
