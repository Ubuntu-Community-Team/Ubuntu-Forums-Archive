I think that the best bet would be a clean install. The dist-upgrade process has not been 100 % perfected yet. If you do a dist-upgrade there might be alot of clutter left, your system will be slower then a clean install, and the dist-upgrade process may leave your system broken so then you would have to do a clean install. Back up any valuable data and then do a clean install.
