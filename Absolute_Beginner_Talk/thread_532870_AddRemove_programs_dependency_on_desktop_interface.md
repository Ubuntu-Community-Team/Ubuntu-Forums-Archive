### Post by Calash on 2007-08-23
You can pretty much install anything.  When you install something that was designed for another Desktop Environment (For example, I run Amarok (KDE) on my gnome based system) it will install the necessary library files for it to run.

The apps do tend to run better in the native environment but they work fine in others.

To take it a step further, you can install multiple Desktop Environments and switch between them during boot-up by selecting the session you want to login to.  At one point I had Gnome, KDE, XFCE, and Fluxbox on my laptop to see what one ran best.

Went with XFCE on that system :)
