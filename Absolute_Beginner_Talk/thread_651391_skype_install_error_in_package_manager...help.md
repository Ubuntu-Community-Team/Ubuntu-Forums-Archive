### Post by anewguy on 2007-12-27
It sounds like you are missing the ALSA sound package, which seems odd.  At any rate, go in to synaptic package manager (System/Administration/Synaptic Package Manager) and do a search for libasound2 - you will see a few entries come up (it takes a while to search).  Be sure you have "libasound2" checked, and it probably wouldn't hurt to check "libasound2-dev" either, then "apply".  When it all finishes, exit the package manager and try your Skype install again.:)
