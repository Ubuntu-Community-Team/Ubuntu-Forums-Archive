### Post by benerivo on 2008-01-19
The desktop wall feature isn't really ever activated by a mouse or keyboard shortcut. By itself it just allows a slide transition between desktops. To  give it some functionality, activate the 'Expo' feature and in Expos options, set the Screen Edge in Bindings to a screen corner of your choice.

Dragging windows in the panel view will not be used by many people, but it is there if you want it.

You can add desktops by right clicking in the desktop view in the panel and increasing the number as rows or columns as neccasary.
