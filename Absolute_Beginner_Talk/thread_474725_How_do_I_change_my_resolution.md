### Post by BarfBag on 2007-06-15
Drop into a virtual terminal (Ctrl + Alt + F1) and type:

> sudo dpkg-reconfigure xserver-xorg

This will give you manual configuration options for your monitor and video card.  It's pretty straight forward.  You shouldn't have any trouble.

Well, I could edit your xorg to add the  mode (I assume you just did it wrong) but its easier for you to just use the following command in any terminal:

```
sudo dpkg-reconfigure xserver-xorg
```

follow it along and keep selecting defaults (they will be set to same as your current xorg), then when you get to the resolution section enable every mode you wish to be able to select. Then restart and you will have it :).
