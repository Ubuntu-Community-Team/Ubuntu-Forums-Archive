### Post by PeterJS on 2008-02-01
Just a heads up you might want to shy away from AMA Desktop. I installed it on a VM, and as a verteran linux and windows user it didn't act or feel like either OS. The main menu they have just isn't easy to use, and I really had to dig to find the ubuntu underneath. Gave the live support a go thinking that might be a key feature, it was like a ghost town in there, I was the only one in the support channel, no so helpful. It's old and I wonder about the state of the project, they never made a final release, the last release they made was an RC based on fiesty which is now well over 9 months out of date.

If you really can get support for it, or have a compelling reason to use it, go for it. But you might want to try a more stock version of ubuntu, it will be newer, have better support (ie at least all of us), will default to having a different loading splash, and has easier tools to change the loading splash:
[http://www.getdeb.net/app/Startup%20Manager](http://www.getdeb.net/app/Startup%20Manager)

Good luck, and enjoy.
