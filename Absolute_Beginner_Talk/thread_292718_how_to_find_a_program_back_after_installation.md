### Post by Ecthelion on 2006-11-04
Usually you can just type the name of the program in command-line to make it work...
Or add it to your menu if it's not in it automatically... rightclick on the menu and choose the right option....

Other hint:

When you go in synaptic, search for that program, right click on it... properties...
Then you get a list with all the installed component...
If you really want an Icon it should be among those installed components I think.
