Hello!

Whenever I double-click on a .mov icon (to open it), Ubuntu tries to open that file using "Totem".
Unfortunately, I get an error message & the .mov file can not be opened with "Totem"...
I then need to right-click on that .mov file & select "open with Mplayer Movie Player".

Only then, can I really view the .mov file...

I "hate" doing this all the time!

I would like to instruct Ubuntu to open all .mov files with Mplayer instead of Totem...

Does anybody know how can I do that?

### Post by mcduck on 2006-08-03
right-click a .mov file, select 'properties' and then choose the program you want to use to open those files..

(so don't choose the 'open with' that is in the right-click menu but the one that is in file's properties window)

I did the following:

> 1, Right-click on the icon & select "Properties"

2. Click on the Tab named "Open With"

3. Select to open with "Mplayer Movie Player"

4. Click on the button named "Close"

Now all .mov files will always open with Mplayer!
