### Post by halfvolle melk on 2006-07-07
I'm not exactly sure what you mean but you can connect both to the internet at the same time using a switch or a router. Something like this:
```
                                            -----
                                    |-------|PC1|
----------   -------   ---------------      -----
|internet|---|modem|---|router/switch|
----------   -------   ---------------       -----
                                    |--------|PC2|
                                             -----
```
[An ethernet switch](http://image.compusa.com/prodimages/18/8c7d9589-a6a3-11d4-b41e-00805fe6e270.jpg)

### Post by harisund on 2006-07-07
Wait a minute, if you are looking for a switch, I think you will have to pay the ISP for 2 internet connections. What you are looking for is a router with an integrated switch. This way, the router will be a gateway in the middle, the ISP will think there is one machine only (for it will see the router alone) but you will see 2 machines connected behind the router.
