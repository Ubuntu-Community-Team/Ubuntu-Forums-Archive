### Post by Paul820 on 2007-07-30
Extract the folder to your desktop, then open the terminal Applications->Accessories->Terminal from the menu. In the terminal type ```
cd ~/Dekstop/nameOfTheExtractedFolder/
```
Now you should be inside the folder on your desktop, then follow the rest of the instructions: make disclean press enter, and then sudo make install

### Post by Nikron on 2007-07-30
Not trying to be mean, but do you know what Ubuntu's packaging system is?  What a packaging system is?  

Let me try to explain.  What you did was go to a website and download the source code.  Then you tried to compile the source code.  You do not need to do this.  This is a long and boring process that someone else will do.  You instead get these programs using your package manager.  The the package manager for Ubuntu is called apt. 

Apt has 2 command line front ends.  apt-get and aptitude.  When I used Ubuntu, I used apt-get.  To install ndiswrapper, you would have to
```

sudo apt-get install ndiswrapper

```

That is all.  Apt will do all the heavy lifting.  There is also a graphical interface for apt.  It is called sypnatic.  It is in System menu, somewhere. (It's been a while since I've seen the default Ubuntu layout).

I think you will benefit from the following wiki guide. [here]("https://help.ubuntu.com/community/WifiDocs/Driver/Ndiswrapper")
