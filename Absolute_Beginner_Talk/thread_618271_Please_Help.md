### Post by bumanie on 2007-11-20
There is a general consensus that people who have not previously had linux experience are usually better off starting off on the 32 bit version, even if they have a 64 bit processor.
There are lots of native linux programs that are capable of doing the same things that windows programs do. You can also run windows programs under wine (this can apparently be a bit hard to configure at times). 
Just take it slowly - you have to crawl before you can walk.
Don't know why you are having download problems. Are you downloading via http or bittorrent? Both are available. If one mirror/bittorrent site is not working, try another, there are plenty to choose from.
