### Post by rillip on 2007-06-02
If you're going to be running a lot of Windows made games, you may find that the best OS for you is in fact Windows.  Wine is the typical, free suggestion on how to make games work, and while it works quite well with some games, with many there are slight or annoying bugs, or out right unplayable behavior.  Some games, like Doom, have a Linux port. That's your best bet, as they run natively just like a Windows game does in Windows, but there aren't very many.
