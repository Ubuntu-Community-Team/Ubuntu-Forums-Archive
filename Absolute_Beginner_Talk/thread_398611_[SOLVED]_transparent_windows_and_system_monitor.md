### Post by irish_flu on 2007-04-01
As far as I know there currently isn't support in Gnome (and its default Window manager called Metacity) for transparent windows without some help from some extra program (actually a Window Manager that runs instead of Metacity) like Beryl or Compiz.

EXCEPT for the terminal.  open up the terminal window, click the "Edit" menu, then click "Current Profile".  Under the "effects" tab, you can do stuff to the terminal like making it transparent (with whatever background color you want).
