### Post by Tomosaur on 2007-06-26
Distrowatch is the place you need. It provides descriptions for many of the different distributions to help you make your mind up. Distrowatch will charge you if you want install CDs directly from them, but in most cases, the distribution is actually free of all costs, so you can just visit the relevant website, download the disk image, and burn it to a CD.

A LiveCD is a fully featured version of the operating system which allows you to get a feel for how it works, along with some of the applications that come with it. LiveCDs generally run very slowly (which is to be expected) so don't use the performance of the LiveCD as an indication of how fast the operating system runs when you actually install it. Linux is generally considered to be much faster than Windows when installed, normally much more reliable, and free of security worries. You can use the LiveCD to make sure that your hardware works correctly before committing to installing the OS. Don't be put off if one distribution doesn't seem to work with your wireless modem, for example - different distributions are tailored for different needs, and the chances are you can find a distribution which works best for you.

Ubuntu is a great distribution which many consider to have the best hardware support around. It's also extremely user-friendly, has great performance, and a great user community. If you run into problems, just ask here and we'll be happy to help :) You don't have to use Ubuntu to ask for help on these forums, but you may find that it takes a while longer for your questions to be answered if your problem is specifically related to whichever distribution you're using.

### Post by eeeek on 2007-06-26
I have an old laptop at home (64 mb RAM, 366 mhz CPU, etc) that I've experimented with. The best "mini-distro" that I've found out there is Damn Small Linux (DSL). I also tried Puppy Linux and Feather Linux with success. 

In general, I've found these smaller distros harder to learn on because they're trying to keep it small and they have to cut out a lot of stuff, making it harder for folks new to Linux. 

The minimum system requirements for most "full" distros seems to be around 256 mb RAM and maybe 500-700 mhz CPUs. I know the base of Ubuntu seems to be about 2 GB on the HDD.
