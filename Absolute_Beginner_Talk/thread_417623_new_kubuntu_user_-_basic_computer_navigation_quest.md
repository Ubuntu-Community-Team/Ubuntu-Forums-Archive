### Post by SOULRiDER on 2007-04-21
Kubuntu should have added your NTFS partitons when you installed, but only as read-only. Try opening konqueror and type ```
media://
``` in the menu bar Also, check out this guide: [https://help.ubuntu.com/community/AutomaticallyMountPartitions](https://help.ubuntu.com/community/AutomaticallyMountPartitions) on how to write to them.

To edit the shortcuts in your menu, right click the k menu icon and then hit edit menu:
Heres a screen of how i amde an entry to open konqueror on my home directory:

[IMG]http://img214.imageshack.us/img214/4825/editorqh4.png[/IMG]

If you go tp the /media/ directory, you can see where everything is mounted. If you have any questions just PM me or come by the IRC channel.

Good luck!
