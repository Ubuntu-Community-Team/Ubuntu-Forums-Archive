### Post by Sorivenul on 2007-10-29
I can tell you two ways to install flash player:
1.) sudo apt-get install ubuntu-restricted-extras
     This will install the Mozilla flash player plugin as well as other extras, like Java 6, etc.
2.)sudo apt-get install flashplugin-nonfree
     This will install the Mozilla flash player plugin

There may be other ways, but these two have worked best for me.  Good luck!
