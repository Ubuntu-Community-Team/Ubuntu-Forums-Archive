*/ad/*
*/ad.*
*/ads*
*advert*
*adtech*
*adclick*
*.ard*
*kokoku*
*i1.yimg*
```

I think the best way to adblock is to create such a list tailored for your own favourite sites.

### Post by benerivo on 2008-01-19
Install the adblock plus extension in firefox. It should ask you to restart firefox, and when you do, it will load a list of adblock subscriptions that you can subscribe to. Select cancel. Go to Tools > Adblock Plus, and you will see the list is empty. Cancel from this and go to a site that has ads you don't want. If you right click on them and select adblock image... In the address string it displays, look for a part that suggests it is an advert, such as this```
http://m1.2mdn.net/viewad/951243/41530/0113_YMA_FurnBtn_300x100.gif
``` and edit this line so that it only reads *viewad*

If you want you can add

*/ad/*
*/ad.*
*/ads*
*advert*
*adtech*
*adclick*

 as a start, which will remove the vast majority of ads.
