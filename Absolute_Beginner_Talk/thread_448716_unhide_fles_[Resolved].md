### Post by earobinson on 2007-05-19
In ubuntu if a file name starts with a "." it is hidden if not it is not, so simply remove the . from the start of the name.

### Post by aysiu on 2007-05-19
Remove the . from the front of the filename.

Hidden: ```
.nameoffile
``` Unhidden: ```
nameoffile
```
