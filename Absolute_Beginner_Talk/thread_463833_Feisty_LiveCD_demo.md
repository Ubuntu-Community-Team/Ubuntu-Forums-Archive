### Post by candtalan on 2007-06-04
The live CD is fully functional. I have 'installed' things into the running Live CD system by downloading from the web (ubuntu repositories) with no problem - all 'live' only. Unless you take special action to save things onto the hard drive then the live CD session is entirely in ram, even though the install  of - say - a game, or apps,  looks real.

Real Player that I use is a binary download from Real site, and it is installed using a simple command line use. If this is what you want to demo, you certainly can, also any codecs etc, afaik, all can be downloaded and installed via live CD. With a little expereinmce you could maybe get this stuff ready available on a usb stick and take it from there. You can certainly write to usb sticks etc.
