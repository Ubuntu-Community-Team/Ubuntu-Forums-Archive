### Post by emarkd on 2008-02-05
If you're not going to be using the GUI, there's no reason to install the desktop and you save yourself the disk space.  If you're going to be administering the system locally, the desktop may make that easier if you're not as comfortable with a commandline-only environment.  Besides, installing the server stuff you're talking about on the desktop version is as simple as marking the LAMP task in Synaptic and clicking install.

In the end, it's up to you.
