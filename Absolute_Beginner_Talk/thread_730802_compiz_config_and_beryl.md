### Post by bumanie on 2008-03-21
You don't need to install beryl, it is a project that is no longer supported and was joined with the compiz project, (prior to the release of 7.10) to form compiz-fusion. As you have compizconfig installed, you don't need to worry about beryl. Read this blog to help set up desktop effects.
