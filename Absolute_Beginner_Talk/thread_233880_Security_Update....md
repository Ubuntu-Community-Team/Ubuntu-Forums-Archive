### Post by davebgimp on 2006-08-10
> **spiral777 said:**
> I saw on Digg that there was an Ubuntu security update today, is this something that happens automatically, or do I need to do something?

Ubuntu should notify you, but there's no reason you can't:

**sudo aptitude update**
**sudo aptitude upgrade**

But if what you saw was the 6.06.1 announcement, you should be fine as long as you've been updating regularly in the past. This new release is more for fresh installs to save time downloading patches and bug-fixes, which you likely already have installed.
