### Post by UbieWan on 2006-06-26
Hi

Here is what I do: From the Applications Menu go to Terminal and right click on it. Select Add this Launcher to Panel. You will need a Terminal A LOT in Linux. Its like the Command Prompt in Windows. Search the forums here for Automatix and download it. Open the terminal by clicking the icon that is now in the launch panel. At the command prompt type cd Desktop and press enter. Youi should now be at your desktop and type ls and enter.  To see where you are you can type pwd (print working directory) there are two ways to LiSt files with ls that is with -aolF this gives you all the detail of your files. When lloking to go somewhere at the terminal you type a single letter and press tab and it will complete some of it for you. Anyway, at the desktop type aptitude install auto (now press tab). If it does not install type dpkg --install aut(now press tab so it completes the file name for you). Good now you have automatix installed. Go to the Application Menu System tools and run Automatix, it installs a bunch of useful stuff for you automatically. have fun.
