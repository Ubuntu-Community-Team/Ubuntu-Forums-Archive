### Post by LaRoza on 2007-10-31
There is also Edubuntu.

@OP There are many desktop environments and window managers. You can install any of the them on Ubuntu (or any version of it). You can choose which one to use when you log in. It is under sessions. 

On Ubuntu:

```

sudo aptitude install kde-core

```

This will get you the KDE setup. If you want the entire Kubuntu package (it is large!), use ```
sudo aptitude install kubuntu-desktop
```

I use Fluxbox, by the way.
