### Post by gigi1234 on 2006-02-12
As a new Linux user myself I can sympathize with your position. Learning to function in the Linux/Ubuntu world takes some time and some work. Things aren't as easy as plug-and-play and auto-install. 

As a first step please try reading through this starter guide:

[http://www.ubuntuguide.org](http://www.ubuntuguide.org)

I don't know anything about your particular issues but hopefully someone will come along to answer your specific questions.

Try using Synaptic (or apt-get) to download and install a program called Wine. This program allows some WIndows programs (such as your games) to be run in Linux. There is also a commercial (read: NOT FREE) program called Cedega that many gamers use to run their Windows games in Linux. I am not a gamer myself so I can't vouch for the usefulness of these programs.

Welcome to Ubuntu! Just keep at it. Read a lot. Google or look at the Wiki. Ask questions when you hit the wall. It really is worth it, in my opinion. 

Good Luck!
