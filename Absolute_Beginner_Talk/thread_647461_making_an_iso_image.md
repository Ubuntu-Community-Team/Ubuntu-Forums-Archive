### Post by alexj.powell on 2007-12-22
1. Insert disk
2. Go to desktop
3. Right click the cd on the desktop, select copy disk
4. on the drop down menu select copy disk to file image
5. burn the iso to disk using cd/dvd creator

Have fun selli... "making backups" of your ps2 games :p
