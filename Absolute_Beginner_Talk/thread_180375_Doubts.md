### Post by Klaidas on 2006-05-22
I'll try to explain some of of those questions :)
> what is exactly deb and rpm files?
Those are the files which you use to install programs. Ubuntu/Kubuntu use .deb files.
> how do i see what files a package had installed when i use apt-get install?
You will see it's dependencies. Those are the files it will install.
> when i install software with apt-get what kind of installation it uses?
If you're using apt-get, it will handle your package's dependecies (the files that are needed to run your application). So it takes care of pretty much everything :)
> how do i configure a package installed with APT?
Not sure what you wanted to ask. Program are configured in different ways
