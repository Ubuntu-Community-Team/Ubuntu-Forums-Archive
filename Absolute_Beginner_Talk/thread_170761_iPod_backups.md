### Post by astoltz on 2006-05-05
Backing up is probably still a good idea, gut the iTunes and gtkpod databases should be identical.  I transfer songs with both all the time with no trouble.

As far as backing up.  The file names on the ipod are all different - they are not named with the song title etc.  So if you just copy everything to your harddrive, you'll have to use something to grab the ID tags and re-name the files.  I don't think It would work to copy everything back to the ipod with the same weird file names as before.  gtkpod has an option somewhere to copy music to the harddrive and then you wouldn't have to worry about the renaming thing.
