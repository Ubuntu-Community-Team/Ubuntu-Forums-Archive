### Post by w7kmc on 2007-01-19
Hi,

If you are interested in just viewing a history of  what is typed in the terminal...simply type:

history

and your last 500 commands will be displayed.  These are all stored in .bash_history in your home directory (hidden file). 

These can be retrieved by using the up/down arrows as well.  To increase the number of commands logged...then you can adjust the HISTSIZE shell variable in .bashrc

See here:

[http://gentoo-wiki.com/TIP_Some_pretty_.bashrc_hints](http://gentoo-wiki.com/TIP_Some_pretty_.bashrc_hints)
