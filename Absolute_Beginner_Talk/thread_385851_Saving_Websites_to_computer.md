### Post by haelters on 2007-03-16
If you use firefox, you can use the FlashGot extension, which can be found here: [https://addons.mozilla.org/firefox/220/](https://addons.mozilla.org/firefox/220/). Just click the install now button. Once installed, you must restart firefox and when it does, you should find the help of flashgot inside one tab. You will also need a download manager (can also be found as an extension for firefox)

Hope it helps...
