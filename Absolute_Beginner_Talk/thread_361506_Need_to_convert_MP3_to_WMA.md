### Post by Marsolin on 2007-02-15
The Konqueror plug-in gets installed automatically. Right click on a folder or file, go to Actions and the PAC Convert menu. It will convert from WMA, but I don't think it will convert to it though.

In Amarok select Script Manager from the Tools menu. Click install script and select pacx from /usr/share/apps/amarok/scripts/. It will appear under General in the tree after you install it.  You'll need to select it and click Run to get it working. After you do that right clicking any file will show the convert menu.
