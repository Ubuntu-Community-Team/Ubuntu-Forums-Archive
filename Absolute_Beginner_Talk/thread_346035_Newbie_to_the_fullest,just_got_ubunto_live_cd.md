### Post by Tux Aubrey on 2007-01-25
Welcome moseefus!

Have you booted your laptop with the LiveCD in the drive?

That does NOT install Ubuntu - it gives you a fully working preview without touching your own windows setup.

Use it that way for a while so that you can test whether all your hardware works and whether you like it or not.

For most people it works fine (there wouldn't be so many people here if it didn't)

Use it, test and ask questions here.

When you have decided to install it you will find everything you need on the LiveCD - there is an "Install" icon on the desktop that will do it for you.

But read a few guides and tutorials before you start - the wiki here is great and some of the Mod Squad have put together their own documents - I really like these ones:

[http://users.bigpond.net.au/hermanzone/](http://users.bigpond.net.au/hermanzone/)

and

[http://www.psychocats.net/ubuntu/index.php](http://www.psychocats.net/ubuntu/index.php)

But read around until you feel confident to partition your hard drive for dual booting (assuming you want to keep windows too at this stage).

See you around.

### Post by Famicommie on 2007-01-25
When you first run the live CD, one of the icons on the desktop says "Install Ubuntu". The installer will help you decide how you want to partition your hard drive.

After you have it installed, your very first step should be to read the system documentation. You can find it under System->Help->System Documentation. That will give you all the basic information you should need to feel more comfortable.

After that, I don't think there's any easy way to learn all there is to know about running a Linux machine. The forums are -very- helpful, and there is an excellent book available which can explain how to do many ordinary tasks.

Finally, I recommend that you install a program called [Automatix](http://www.getautomatix.com/). It automatically installs flash, Azureus, and many other useful programs that you'll find yourself wanting before you're done learning.

### Post by 3rdalbum on 2007-01-25
When the Live CD is booted up, you double-click the Install icon.

Proceed through the install. Then you will be asked how you want to install Ubuntu - by erasing the hard disk, erasing a partition, or resizing a partition. Choose the option about resizing the partition.

A slider will appear, where you can specify what the new size of the old partition should be. If, say, you want to give 25% of your hard drive space to Ubuntu, you would drag the slider so it's three-quarters of the way to the right.

Then continue the install. Everything will be done for you.

The best way to learn Ubuntu and Linux is to set up all your codecs and proprietry software "by hand" - don't use EasyUbuntu or Automatix. If you get stuck at any point, look up a HOWTO on Google or on these forums, and if you get REALLY stuck, ask here.

But really, the best way to learn Linux is simply to use it as much as you can!
