### Post by cbudden on 2006-05-08
First of all, you should be using Sauerbraten (aka Cube 2) from here - [http://www.sauerbraten.org/](http://www.sauerbraten.org/).

Once you have downloaded the tar.gz, right click, extract here and inside the folder you will find another folder called sauerbraten and within that a file called sauerbraten_unix .  Double click that and then click run and you should be off.
