### Post by GreyFox503 on 2006-02-10
I haven't used KDE in other distros, but you might not have all the applications/options you're expecting if you just select KDE looking stuff from Synaptic.

If you want the full KDE/Kubuntu effect, install the kubuntu-desktop metapackage.  It will give you everything that comes in a Kubuntu installation.

If invoked from a regular Ubuntu installation, it will hundreds of new packages, so be warned.
