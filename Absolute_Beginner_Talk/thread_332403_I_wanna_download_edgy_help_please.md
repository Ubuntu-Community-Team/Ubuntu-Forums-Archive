### Post by Frak on 2007-01-06
You can only download one desktop manager at a time, whenever you have one of them, just run this command.

```
sudo aptitude install ubuntu-desktop kubuntu-desktop xubuntu-desktop

or seperately

sudo aptitude install ubuntu-desktop
sudo aptitude install kubuntu-desktop
sudo aptitude install xubuntu-desktop
```

But for you, I recommend Dapper more than Edgy.
