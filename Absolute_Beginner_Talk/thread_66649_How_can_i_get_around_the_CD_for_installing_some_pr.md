### Post by fishfork on 2005-09-17
Assuming you've got internet access, this should be easy. You just have to de-activate the CD as a repository.

Click on: System / Administration / Synaptic to get the package manager
Open: Settings / Repositories
In settings, select 'show disabled software sources', and close
Untick 'CD Ubuntu 5.04'
Press OK, and yes, to reload the package lists.

Hope this works!
