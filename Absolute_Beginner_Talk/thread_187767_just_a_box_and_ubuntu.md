### Post by IYY on 2006-06-03
If you want to connect your machine to a printer directly, you may run into some trouble (not all printers are supported). If you want to copy the file to a floppy, you shouldn't worry about it being in "linux format", since both Open Office and AbiWord can save to the standard .doc format. The floppy itself can also be formatted with the FAT filesystem which is understood by all Windows machines.

Just be sure to check the specs on that computer. You'll need a minimum of 64 MB of RAM to run Ubuntu, and that's if you're willing to switch to IceWM or Fluxbox as your window manager. For Gnome, you need 128 MB.

### Post by CompShrink on 2006-06-04
About saving in .doc format, as mentioned above both OpenOffice and ABIWord can do it. I have not used ABIWord before, but I know in OpenOffice, it is an option, not the default. Just remember to change it, or the first paper you write and try to print may be a hassle. You can go into preferences in OpenOffice and change the default to .doc, which I would reccomend.

Also, make sure you install before you go. You might find it's too slow using Gnome as the GUI (graphical user interface) manager, and want to switch to something that works better on slow hardware, such as ICEWM or Fluxbox which were mentioned before.
