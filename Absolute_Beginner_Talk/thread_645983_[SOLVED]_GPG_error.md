### Post by Seisen on 2007-12-20
Enter this in the terminal to download the GPG key

```
wget http://hendrik.kaju.pri.ee/ubuntu/hendrikkaju.gpg -O- | sudo apt-key add -
``` then ```
sudo apt-get update
``` and you should have no more errors.
