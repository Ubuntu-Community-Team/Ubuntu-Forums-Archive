### Post by Malibu Illusion on 2007-07-08
Great place to start would be taking a look at the WINE application DB:

[http://appdb.winehq.org/appview.php?iAppId=1922](http://appdb.winehq.org/appview.php?iAppId=1922)

After consulting that and seeing how it runs, then it should just be a matter of launching the Windows executable for installing from within WINE:

```
wine /path/to/setup.exe
```
