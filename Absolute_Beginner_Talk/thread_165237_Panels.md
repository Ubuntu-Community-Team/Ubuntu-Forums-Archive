How can I make panels and windows transparent?

---

### Post by frodon on 2006-04-24
For panels it depends on the theme you use, have a look to the gnome-look.org site to get more themes (or kde-look.org if you use KDE).
To get transparent windows you have to enable the composite display, here is a good guide about it for gnome : [http://doc.gwos.org/index.php/CompositeManagerGuide](http://doc.gwos.org/index.php/CompositeManagerGuide)

### Post by ncappel1 on 2006-04-24
In Gnome, you can right-click on the panel and choose properties. Once there click on the "background" tab and select the radio button to make the panel a "solid color" then slide the bar all the way to the left to make it transparent. 


Hope this helps!
