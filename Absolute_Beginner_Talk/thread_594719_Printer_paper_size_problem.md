I've installed it OK using the printer setup thing under Gnome and I can print a test page OK.  However, I can't print from Open Office.  When I click print I set the printer to the correct one (I only have one) and check that the paper size is set to A4 under the properties tab. Like I said, I can print a test page OK on A4 paper, but get this problem with Open Office.  I didn't get this problem under Feisty.

### Post by mikeyphi on 2007-10-28
Open the OO document - select 'Format' from the menu - select 'page' then the page tab then select A4 in the 'Format' window!! 
Unfortunately OO comes with 'letter' set as default - you might want to change that to A4
