### Post by groggyboy on 2006-11-17
The Ubuntu installer is looking for a place to install.  That's what that error means.  Linux starts at the root ("/").

What does previous screen of the installer look like?  Also, what program did you use to partition your hard drive and how did you format the new ubuntu partition?  By default, Ubuntu uses ext3.  The partition program in the installer is pretty powerful.  If it can't find the partition you made, you can always try again from within the installer.

And don't forget to include a linux swap partition!

Go back one step with the installer and post a picture of that screen.  That'll help to figure out what's going on.
