### Post by mcduck on 2006-08-28
You can install programs when running the Live-CD, as long as you have enough RAM to keep those programs. But you'll loose all installed stuff when you reboot. Installing programs to hard disk is not possible with Ubuntu's Live-CD, but there are some other Live-distros that can do this.
