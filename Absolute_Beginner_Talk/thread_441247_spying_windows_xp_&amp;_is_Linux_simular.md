### Post by [S|G] on 2007-05-12
Linux keeps ALL individual user data (such as email, configuration settings and documents) in the user's home folder (usually /home/username). Deleting this directory will most likely erase all data that the particular user had, unless you specifically allowed user data to be stored somewhere else.

Also, unlike Windows, recovering deleted data on an ext3 filesystem (which is linux default filesystem) is not as easy as on Windows, but it IS possible with the right tools. This is mostly due to the fact that once you delete data it is not phisically erased from the hard drive, but only the references that point to where the data was stored is erased. It takes more than one program to do that, though.
