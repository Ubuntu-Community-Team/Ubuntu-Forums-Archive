### Post by gautada on 2007-09-10
This comes from some spam detection system.  If you are using an ISP that is probably where it was set.  If are hosting your own mail server then you should know where it came from.  You can turn it off at the mail server or use it to create a filter to put detected spam in the junk folder.  I have a whitelist with my ISP that removes this spam message for people I know/trust.
