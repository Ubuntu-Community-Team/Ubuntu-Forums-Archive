### Post by eentonig on 2007-06-04
1. Installing/Enabling additional Repositories  
--> [https://help.ubuntu.com/7.04/add-applications/C/index.html](https://help.ubuntu.com/7.04/add-applications/C/index.html)
+ You'll get some background as to what are repo's. And why and how they are split up like this.

2. Installing Multimedia Codecs 
--> [https://help.ubuntu.com/7.04/musicvideophotos/C/index.html](https://help.ubuntu.com/7.04/musicvideophotos/C/index.html)
near the end of the page is the howto to install extra codecs.
3. Installing Video Players 
--> See above
4. Installing Audio players
--> See above
5. Installing Macromedia Flash-Plugin
--> See above

6. Installing beryl and other eye candy (but I have heared that it makes alot of problems since it's not stable yet)
--> I've already given you the commands to do this in a previous reply.If you want more info regarding the subject: [http://wiki.beryl-project.org/wiki/Install_Beryl_on_Ubuntu](http://wiki.beryl-project.org/wiki/Install_Beryl_on_Ubuntu)
Make sure you have a graphics card that supports it. And that X is set up to use it.
For me, it installed flawlessly and I never had any real problems. 
Hint: In Beryl manager, make sure you select to fall back to Metacity if Beryl crashes. That way, you'll 'alway' have a backup-path

7. Installing Java Runtime environment v6.0 With Mozilla Firefox plugin
Haven't done it.
8. Installing Google Applications - Google Earth and Picasa
Haven't done it.

9. Installing Real Player and Adobe Reader
I can read pfd out of the box, so didn't install Acrobat Reader.
Realplayer doesn't interest me.

10 . Installing Downloading & P2P tools - amule , Downloader for X ,azureus
ftp and wget for direct downloads are available from scratch
torrents are supported via bittorent after installation. Allthough there are probably better client, I didn't bother and use the default.

11. Installing Skype
Don't use skype

12. Installing Microsoft True Type Fonts
> sudo apt-get install msttcorefonts
If you want more fonts. Use 
> apt-cache search ttf
You can then install them by typing 'sudo apt-get install " and append the package names.

13. Installing Miscellaneous Other Applications
Obviously, at your choice.

I'm sorry if I was to lazy to only provide the CLI way of installing. Please feel free ti subsitute every occurance of 'sudo apt-get install' with

1 Go to System\Administration\Synaptic Package Manager
2. Click search to find your desired package
3. Click on the square in front of the package and choose 'install'
4. Click on the 'Apply' box

### Post by Dylnuge on 2007-06-04
1. Change around your themes. Get your desktop to look the way you want. Check [art.gnome.org]("http://art.gnome.org") for some awesome wallpapers and themes for GNOME.

2. Install the software you want. Check out [this list]("http://www.linuxrsp.ru/win-lin-soft/table-eng.html") for some Linux equivilents.

3. Get the things that make your computer yours up. Specific background? Files? Fonts? etc.

4. Start learning the new software. Evoloution, OpenOffice, Firefox, GIMP, and more may be new to you.

5. Visit [www.linuxcommand.org](www.linuxcommand.org) and learn the terminal (You don't need to do the scripting lessons unless yoiu want to). I recommend learning this early on so you will understand it. In Feisty GNOME, the Terminal is located under Applications -> Accessories -> Terminal.
