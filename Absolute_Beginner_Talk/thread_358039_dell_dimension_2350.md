### Post by negated on 2007-02-10
The memory can report it's total amount correctly and still be bad. 

I highly suggest you get yourself a copy of Memtest+ to test for bad memory. It's FOSS just like Ubuntu.

[Memtest+]("http://www.memtest.org")

There you will find downloads for ISOs (to make a bootable CD), bootable floppy, and USB keys.

Basically, if you can run a couple of complete passes with that without any errors, you probably don't have bad memory...

Hope that helps!
