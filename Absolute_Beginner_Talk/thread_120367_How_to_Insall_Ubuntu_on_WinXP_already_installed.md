### Post by JAwuku on 2006-01-21
If you don't want to trouble your Windows XP installation, you could choose to run Ubuntu indirectly via virtualisation software, which emulates another PC on your PC, enabling you to run another OS at the same time as Windows.

There is a free version, called QEMU, available from [http://free.oszoo.org/download.html]("http://free.oszoo.org/download.html")

Alternatively, Parallels.com provides a free 45-day trial version of Parallels Workstation, which can be downloaded from [Parallels.com]("http://www.parallels.com/en/download/")
