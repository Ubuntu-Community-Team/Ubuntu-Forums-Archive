### Post by benuski on 2006-08-04
An alternate CD is just another copy of the ISO that you burn, because if you 're doing other things while burning a cd, sometimes you can get small errors on it, that can be fatal.

There should be an option at the bottom of the screen that pops up when you put in the livecd and it loads to the main menu that says "Check disc for errors" or something like that.  Just select that option, and it will do it all by itself.
