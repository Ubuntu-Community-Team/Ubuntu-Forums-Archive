### Post by lbriner on 2007-06-01
Easiest way is probably to insert the image into an Openoffice word processor document, resize it with the handles and make it as big as you like. As Inxsible correctly pointed out, making the image larger just makes all the pixels larger so if the image is really quite small, it will look pretty grim when enlarged. No commonly available software can guess how to increase the picture detail and even expensive software can only guess!!!
