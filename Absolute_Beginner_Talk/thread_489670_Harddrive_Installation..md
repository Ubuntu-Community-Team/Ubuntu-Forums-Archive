### Post by indytim on 2007-07-01
As Arochester mentioned, you will need to burn the iso to a CD (not copy but actually burn the iso).  Also, you will need to be able to boot to a CD.  If you cannot currently do that, you will need to access your BIOS at boot initiation and adjust your boot sequence.

If you need help with any of the above, do a reply and someone from the collective should be able to provide the support you need.
