### Post by Jagot on 2006-08-12
System -> Administration -> Login Window.

Then go to the Security tab and tick Enable Automatic Login, then select the user you want to login automatically.

> Yes thats exactly what i'm after, but i'm asking, how do i make an image i've got become the login screen?

From the login window, go to first tab "Local" Then you can customize it any way you wish. To put in a custom background click where it says theme and either choose "Plain" or "Plain with Face Browser" when the next window comes up choose image and browse to the file with the image you want.
