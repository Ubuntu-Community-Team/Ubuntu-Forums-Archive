If it works with Wine, you should just right click on the .exe file and pick "open with Wine".

### Post by Paul820 on 2007-10-21
Right click on the tunebite.exe file and select open with, choose wine. If it is not there in the options, click 'add' and a bigger list will open. If it's not there again, click 'use custom command' and type in 'wine'.
