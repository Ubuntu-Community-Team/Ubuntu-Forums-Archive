### Post by az_human on 2006-08-30
Applications -> Accessories -> Alacarte Menu Editor

Scroll down on the left and choose 'System Tools'.  Put a checkmark in 'Configuration Editor' then close Alacarte.

Applications -> System Tools -> Configuration Editor

Expand 'apps', then 'nautilus' and choose 'desktop'.

Put checkmarks in 'computer_icon_visible' and 'home_icon_visible' (and network, trash, etc etc).
