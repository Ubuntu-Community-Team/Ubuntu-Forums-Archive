Try the alternate (install) CD, and if that doesn't work, try to install a server from that disc. The desktop/live CD doesn't always work just right. 

If you do find yourself installing a server, use the *sudo aptitude install ubuntu-desktop* command to install the graphical desktop. A server install basically leaves out all the pretty stuff. ;) Good luck!
