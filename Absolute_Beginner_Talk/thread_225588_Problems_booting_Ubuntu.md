### Post by casperlingus on 2006-08-18
Sorry I'm a bit late on this.  I hope I can still help.  I also have a Thinkpad T22 and had the identical problem.  But the fix is trivial!

Simply, Thinkpads don't like running Dapper in 16-bit color depth.  You have to change the default color depth to 8-bits or 24-bits.

So right after you install, you need boot into recovery mode and either modify /etc/X11/xorg.conf if you know how, or run
```
sudo dpkg-reconfigure xserver-xorg
```
and select all the default values except for the color depth option (you want 24-bits, 8-bits looks terrible).  This worked flawlessly for me.

Casper...
