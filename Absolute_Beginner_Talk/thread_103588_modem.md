### Post by localzuk on 2005-12-14
Hi, welcome to Breezy.

First one: There is some information available from [http://www.lack-of.org.uk/viewarticle.php?article=114](http://www.lack-of.org.uk/viewarticle.php?article=114) for the modem. Linux does not have much support, as yet, for USB adsl modems due to manufacturers not being helpful and providing any useful information about their devices.
Second: Synaptic is a software management system that allows you to install any of the packages available in the 'repositories'. Green ticked boxes are packages that are currently installed. Ones that are unticked are not installed. More information about synaptic can be found at [http://www.nongnu.org/synaptic/](http://www.nongnu.org/synaptic/)
Finally: Terminal Server Client is for connecting to remote servers running RDP or VNC servers. If you do not have a network connection running then you will not get any connections. Take a look in the admin tools (I cannot remember where they are, either in Preferences or System in the menus at the top) for the Network tool. It will show if you have a working connection.
