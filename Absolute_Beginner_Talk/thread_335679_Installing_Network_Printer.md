### Post by braddcadd on 2007-01-10
1. Go to system (pull down menu), admin, network.
2. Browse to the printer of choice.  Determine the network address.
3. Go to stystem, admin, printing.
4. New Printer.
5. Network printer with the above address.


Here is a help page on printing:
[https://help.ubuntu.com/6.10/ubuntu/desktopguide/C/printing.html](https://help.ubuntu.com/6.10/ubuntu/desktopguide/C/printing.html)
