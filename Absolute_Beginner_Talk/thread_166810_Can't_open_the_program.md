### Post by gingermark on 2006-04-27
The files you have downloaded are CD-images. You don't "open" them in Windows. As rado_london said, use a burning program (Not Windows' own one - that is useless) and make sure you select an option like "burn from CD image".

DO NOT just burn the iso file itself.

You then need to boot off the CDs on the computer you want to use them on. You might need to edit your boot order in your BIOS settings so that the computer boots from the cd/dvd drive first.

You are usually instructed on how to enter BIOS when you turn your computer on (usually you press Del, or F2, or something).
