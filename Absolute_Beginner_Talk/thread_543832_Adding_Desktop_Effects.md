### Post by kellemes on 2007-09-06
At the bottom of xorg.conf you should add..
```

Section "ServerFlags"
  Option "AIGLX" "off"
EndSection

```

AIGLX will not work, instead you should install XGL, I can't explain how to, but [this thread]("http://ubuntuforums.org/showthread.php?t=488385") may help.
