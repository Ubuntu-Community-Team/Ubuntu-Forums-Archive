### Post by Lexyboy on 2006-10-30
I take it you mean programs to run lab instruments from ubuntu?  I would imagine then that if you have the software for windows you could probably run it through wine, since such programs don't usually do anything too fancy, though there might be problems in dealing with the necessary ports (I'm no expert here, just a guess).  In terms of other software, there seems to be quite a lot of small apps, but nothing terribly powerful.  I use Scifinder under wine with no trouble, though no luck yet with Mestrec or ChemDraw.  There's a fairly extensive list of chemistry-oriented (though some more generally useful) programs here: [Linux4Chemistry]("http://www.redbrick.dcu.ie/~noel/linux4chemistry/").

I think there are a few pretty good graphing-type apps around, can't remember names at the mo though...

### Post by Bloch on 2006-10-30
Canon cameras in particular are well supported by ubuntu. You should have no problem there.

However you will not be able to use the extra programs on the accompanying CD - for example my camera had software for making panoramas etc.

Small scientific programs are the kind of thing that works well using wine. If the program doesn't use the internet connection, and is not hugelt complex in its UI then chances are it will work fine. Another indication is if the program is not too fussy about which version of windows it requires.
