[QUOTE=Lost-in-North-Dakota] Any advantages of Ubuntu?[/QUOTE]

The PPC edition of Ubuntu has as big of universe of software pacakges as the x86 version. Therefore the biggest advantage of PPC Ubuntu over Yellow Dog is that fact that there are thousands more software packages ready to be installed (no compiling programs from source for you!).

### Post by froogle on 2005-06-14
Right, and it's a heck of a lot easier to install apps using Ubuntu than it is with Yellow Dog. You simply run up a tool call Synpatic, search for what you want and click install. The package then downloads, installs and configures itself automatically. In fact, it's even easier to find and install new software on Ubuntu than it is on Mac OS X. 

Now, to be fair to Apple, and I'm not saying this in an Ubuntu forum to get flamed or anything, it's going to be about 2009/2010 before Apple stop supporting the IBM processor based Macs. They are switching to Intel yes, but that won't be complete for all desktop Macs until 2007, and based on Apple's history with processor changes I would expect them to keep supporting the older processors for about 2 to 3 years after that, by which time the Mac you currently have will probably have gone to the great hardware graveyard in the sky, and the software you currently use will most likely be obselete anyway.
