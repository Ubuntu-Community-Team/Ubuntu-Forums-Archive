### Post by cobrn1 on 2007-08-15
I'm assuming you're on broadband with a router...

If so then you don't actually need the AOL software to connect to the internet - the router does it for you automatically...

You can use firefox for webbrowsing and thunderbird for e-mail - there are guides on how to set thunderbird up for your aol account, but it's really easy... you just set the incoming and outgoings servers, and tell thunderbird not to save to the sent folder and you're all done - look it up and it'll become apparent.

You're only problem however (as I'm sure you're aware) is that you use AOL to view your billing information, and that *can't* be done outside AOL's software, and you do need AOL for that...

Maybe you could consider a dual boot situation, given that there's only rare cases where you'll actually *need* to use AOL's own software...
