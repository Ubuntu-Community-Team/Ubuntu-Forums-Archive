### Post by AndyCooll on 2008-02-09
> **antgul3382 said:**
> ok i installed gaim and pidgin and cant find either in the menus what am i doing wrong???

You've *installed* Pidgin on your Linux box? Why? It should have already been installed as part of the original Ubuntu installation process and should be under Applications > Internet  > Pidgin Instant Messenger.

If it isn't displaying you can edit your menu entries using System > Preferences > Main Menu. You can also open up a terminal and enter "Pidgin" (without the speech marks) and that should open up the app.

Gaim and Pidgin are the same app, Gaim simply being the old name for PIdgin.
