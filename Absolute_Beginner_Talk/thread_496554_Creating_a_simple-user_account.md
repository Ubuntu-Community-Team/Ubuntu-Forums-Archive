### Post by ROUBOS on 2007-07-09
Hi people.
I have placed my Ubuntu Box in my brother's library for students to be able to serf the web and use OpenOffice.

So what I want to do is create a simple user account. With this account a  student will be able to only serf the web, use OpenOffice  and all the programs that are installed already. I want them to be able to save only in user's home folder and not to be able to browse and alter anything outside that folder.

How do I go about creating such an account please?

---

### Post by Old Pink on 2007-07-09
Go into **/home** in nautilus, and make sure your home folder is set to permissions you like, so he can't access/read/modify/execute anything in there. :) 

Then go to **System > Administration > Users and Groups **adding a new user with only the privileges you mentioned. :)
