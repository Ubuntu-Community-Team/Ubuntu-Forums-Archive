### Post by aysiu on 2006-09-02
First of all, I don't know what Linux Genealogy 2.0 is, and I can't help you set up your internet, as Ubuntu just auto-detected mine...

But... I will say that to set up your internet a good place to start might be System > Administration > Networking. You may also want to mention if you have dial-up or broadband and what kind of network card you have (if you know), and if you're going through a router--basically, supply as much information as you can.

The terminal where you paste commands (save yourself some keystrokes and don't retype them) is Applications > Accessories > Terminal in Ubuntu or KMenu > System > Konsole in Kubuntu.
