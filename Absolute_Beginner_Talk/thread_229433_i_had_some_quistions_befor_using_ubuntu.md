### Post by greybirds on 2006-08-04
Two things to keep in mind when trying out Ubuntu on the cd:

(1) Because it has to detect your computer's setup and configure itself each time you use the cd, it takes a long (long) time to start. Once installed it'll boot much, much more quickly. With my system the cd takes about a minute, but booting from my hard drive takes 15 seconds. 

(2) Video and sound files come in different formats, and each format requires a codec (compressor/decompressor - it lets the media player understand the format and play it). Ubuntu comes with some codecs preinstalled, but not all. If a movie you want to see doesn't play, it's most likely because it's format's codec needs to be installed. Doing that's easy, but it can be frustrating if you don't know you need to install those extra codecs.

Have fun playing with Ubuntu, and I hope you like it well enough to join the club!
