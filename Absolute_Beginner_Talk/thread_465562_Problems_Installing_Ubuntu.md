### Post by ramjet_1953 on 2007-06-05
A few important things to remember, when burning any iso.

1. Burn slowly, no faster than 4 X.

2. Use good quality media for iso's.

3. Remember to burn it as an iso image, not a data CD. Most burning software has this as an option in one of the drop-down menus.
