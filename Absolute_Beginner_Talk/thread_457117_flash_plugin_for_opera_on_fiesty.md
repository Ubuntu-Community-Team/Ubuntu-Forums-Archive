### Post by yabbadabbadont on 2007-05-28
Use synaptic (System->Administration->Synaptic Package Manger) to install flashplugin-nonfree.  From reading Opera's documentation for linux on their website, it appears that the plugin should then work.  You will need to restart Opera after installing the flash plugin before it will be available in your browser.
