### Post by mivo on 2007-11-23
I got my new desktop a few weeks ago, and I put Linux on it right away. I never even installed any Windows version on it.  The guy who built it for me (well, he had the motherboard and more skill than me!) had put OpenSuse on it, so I knew the hardware was basically supported by Linux. When it arrived, I put the Gutsy CD into it and ... nothing happened. Turned out I needed some extra boot options, after which things worked fine. Everything was detected, X properly found the 1680x1050 native resolution of the monitor, and the machine was running just fine. (It currently runs Arch, which was an equally painless affair, minus the boot parameters because it uses a newer kernel version.)

So, if you plan to put Ubuntu on the laptop, go ahead and do it right away. :) No point in waiting a few weeks -- it's not like headphones that you need to "burn in" for the best quality. Since it propably comes with Windows, do a test-run to see if all components work, and then tackle the Ubuntu installation!
