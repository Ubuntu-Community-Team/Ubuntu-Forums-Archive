### Post by akiro.yamamoto on 2006-03-25
What you probably did was COMPILE a new custom kernel.
But you could have selected a pre-compiled kernel from Synaptic with basic defaults.
Ex. For a Pentium 4 CPU you could use the i686 series kernel image.
For an AMD Sempron you could use the k7 series kernel image, etc instead of the default i386 image.
Each of those images are compiled to operate best on a certain Class of CPU.
