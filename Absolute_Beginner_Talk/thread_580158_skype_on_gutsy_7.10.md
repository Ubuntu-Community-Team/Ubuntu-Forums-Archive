### Post by ronmarley1 on 2007-10-18
Hey Dude,
Is your sound still working?  I upgraded from Feisty to Gutsy about two weeks ago and Skype still works fine.  I'm using version 1.4.0.118, which is just a slight rev. behind the current one at [http://www.skype.com/download/skype/linux/](http://www.skype.com/download/skype/linux/).  I think you should be able to download the one listed for Feisty on that page and it'll work fine.  I don't think that one is specific to Feisty, it's just how Skype listed it on their site.

EDIT: If you want, you can add the Skype repo.  It'll tell you how at [http://www.skype.com/download/skype/linux/repositories.html](http://www.skype.com/download/skype/linux/repositories.html)
Then, you can install Skype using Synaptic, and it'll give update notices whenever Skype posts a new version.
