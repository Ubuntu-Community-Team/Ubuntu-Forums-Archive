### Post by Bachstelze on 2006-11-14
Uncompress the archive :

```
tar xzvf filename.tar.gz
```

browse to the dir, you should have a README or INSTALL file that will tell you what to do.

### Post by igknighted on 2006-11-14
Just because it is a tarball does not mean you are compiling it.  It could very well be a binary installer, shell script, etc.  There are different ways to install all of these, so the best you can do is follow the instructions included in the archive or on the website you got it from.
