### Post by Indras on 2006-09-17
Your best bet is to go to [http://www.getautomatix.com](http://www.getautomatix.com) and get that installed, then run automatix, Applications->System Tools->Automatix.  When the package list comes up, install either Firefox and Firefox Plugins, or Swiftfox and Swiftfox Plugins (or both, they're compatible).  It's also a good idea to install the AUD-DVD and Multimedia Codecs packages as well, just to be sure you have them all.

I've done this with a fresh install of Ubuntu, and I have yet to find a video or song that I can't play (realaudio, quicktime, windows media, mp3, xvid, etc).
