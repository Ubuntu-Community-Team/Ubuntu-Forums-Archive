### Post by dark_harmonics on 2008-04-14
I was in a similar situation to you about 6 months ago. I had the advantage of having run Ubuntu in a virtual machine for a few months before installing on all my home computers. I would recommend the book: "Ubuntu Unleashed". Its rather large but has real-world applications of different features of Ubuntu. 

As in any computer adventure, I found that experience is really the best teacher. when i started out i just kinda jumped in, experienced the problems that arose from my ignorance of the inner workings of the OS, and researched solutions to the problems I encountered. 

I think the most important thing anybody can say is to not give up! Keep plugging away and you will learn it. The biggest hurdle is getting past that learning curve. Once you re-adjust (we are all used to windows and think in the windows mindset) that perspective you will never want to go back to supporting microsoft's crappy contribution to the computing world.

Some suggested packages that i know i would want to install:

smbfs samba -- allows access to NTFS shares
nfs-common -- linux to linux file sharing
ubuntu-restricted-extras -- Java MP3 support stuff like that...
comipzconfig-settings-manager -- allows you to edit the detailed settings of compiz fusion.
build-essential -- needed to compile most things.
wine -- natively run some windows programs. This can be hard to use sometimes though. 
Some non-repository items: 

Packages from mediubuntu should be added to support DVD playback.
Cairo-dock, AWN, or kiba-dock are my favorite docking programs. 
Gnome-look.org has tons of cool customizations for your linux OS
K3b or Brassero for CD/DVD burning. I prefer k3b personally.

Other misc info:

aticonfig tool will be helpful for your video card. You are also gonna wanna check out the proprietary driver installs. Do a search for bianary driver ati install for more info. 

creating directories for you mounts under /media makes it so that when you mount the drives, they auto-populate to your desktop. 

Ok thats all the time i had for posting today :) Hope some of it helps!

### Post by hyper_ch on 2008-04-18
you do need another partition (except if you want to try wubi I think)... however partitioning should for you not be a problem...

If you buy a second disk you might even try something else - just to make sure you don't screw up anything:

(1) unplug your first harddisk
(2) plug your new harddisk
(3) install ubuntu on that

That way you won't screw up at all...

However if you want to install both oses on the same disk, then

(1) first install windows and leave some space unpartitioned for Ubuntu
(2) install ubuntu and make sure to select "largest continuous free space" (emphasis on "free"

Besides that, run first the Desktop CD as it is a liveCd. I see that you have an ATi card - those are sometimes a bit of a problem but with the forums here you should be able to manage that.

As for reading I recommend
(1) [http://www.psychocats.net](http://www.psychocats.net)
(2) [http://www.ubuntuguide.org](http://www.ubuntuguide.org)
(3) [https://wiki.ubuntu.com](https://wiki.ubuntu.com)
(4) This forum and goole ;)

On a sidenote: The new version Ubuntu 8.04 Hardy Heron will be released in a few days. I'd say you can try the beta of it right now (it will be auto-upgraded to the release one) as the hardware support is better on 8.04. So you might not even have small problems with your video card.
