### Post by Xian on 2006-03-04
They play fine using RealPlayer...
Use Synaptic to install the 'realplayer' application.

Or download it [directly](http://archive.ubuntu.com/ubuntu/pool/multiverse/r/realplayer/realplayer_8.0.11_i386.deb) and install on your system.
If using Synaptic you'll need to enable the [extra repositories](https://wiki.ubuntu.com/AddingRepositoriesHowto).
