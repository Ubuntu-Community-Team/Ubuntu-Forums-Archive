### Post by BarfBag on 2007-04-18
If I'm thinking straight, Belkin wireless cards don't play well with Linux.  Feisty Fawn (7.04) is going final tomorrow.  They improved wireless compatibility.  You should try it out.  If it doesn't work, [SimplyMepis]("http://www.mepis.org/") has even better wireless compatibility (or so I've heard).

Yes, you can install the nVidia driver.  I find that the easiest way is through [Automatix]("http://www.getautomatix.com/").  It's a one click installation.  Feisty is going to have a one click installation feature for the driver as well, so again; you should check it out.
