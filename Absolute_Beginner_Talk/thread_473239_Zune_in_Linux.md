### Post by starcraft.man on 2007-06-13
> **Aiello said:**
> So you can only use one app at a time with Wine?

I don't think you'll be able to get the Zune to work natively at all with Linux. I just checked [WINE DB]("http://appdb.winehq.org/") here. The zune software hasn't even been tested, I assume that means it won't work at all and windows media player 11 is rated as garbage in WINE (thats bad).

The only solution, is to install a copy of WIndows into a VM (Virtual Machine) and then you can use it to sync the songs on the Zune from inside linux. I don't know how well it will work though, probably no problems long as the VM shows your USB Zune inside it. Short of doing that, you'll be stuck with a dual boot of windows to get the Zune working.

The zune quite frankly is a DRM ridden product, it may be nice in its UI but its squirt feature is idiotic (some songs don't even squirt, and the few that do are 3 days or 3 plays... lame) not to mention the fact that Windows doesn't support the Zune on any distro other than windows I assume. IMO, its a worse player than the iPod. I stick to folder driver players that allow drag and drop (PSP is one), they run super well in Linux with no software. Sorry, but I'm afraid you just made a bad purchase if you want to use Linux full time, thats how it goes. Unless someones been able to hack the Zune.....
