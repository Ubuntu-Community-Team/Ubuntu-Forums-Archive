### Post by vanadium on 2007-12-13
The message says there is no connection with the server. Thus, check the name of the server carefully, and obviously it should be on line (it probably will). Perhaps you also need to specify a port to connect. If that is the case, then the syntax is, for example

pop.tiscali.co.uk:432

to connect to port 432. Substitute 432 in tis example by the port specified by your provider.
