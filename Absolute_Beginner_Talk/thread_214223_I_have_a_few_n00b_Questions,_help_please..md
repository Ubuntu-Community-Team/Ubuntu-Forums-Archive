### Post by shinythings on 2006-07-12
For some general info about setting up the network have a look at [https://help.ubuntu.com/community/InternetHowto](https://help.ubuntu.com/community/InternetHowto)

Normally it should work out of the box, but if it doesn't some of the settings may be wrong. You can adjust them by going to System -> Administration -> Networking
Wireless networking shouldn't be a problem either (unless you have very obscure hardware), but it also needs to be set up properly.

Hope this helps. If you still have problems I might be able to help if you provide some more information about how you connect to the internet.
