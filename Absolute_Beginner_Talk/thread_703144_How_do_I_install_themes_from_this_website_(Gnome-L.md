I never used compiz-fusion, or compiz theme before. But I have read a lot of articles about it. It seems that what you need to to is to install an package called compiz manager, which you can install it by inserting the following command in the terminal

```
sudo apt-get install gnome-compiz-manager
```

And search for the app in one of the menus. I believe it will be installed in System menu.
