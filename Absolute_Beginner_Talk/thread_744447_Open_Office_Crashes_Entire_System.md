### Post by TeraDyne on 2008-04-03
Koffice is a great alternative. It's a KDE application, so if you're using Ubuntu and don't have the Kubuntu or KDE desktop installed, it'll have to install the basic parts of KDE as well.

If you're using Kubuntu or KDE, it's much faster than OOo, as far as my experiences go. I suggest giving it a try.

Edit: Whoops. I forgot to mention: Koffice is in the repositories, so you can use the Add\Remove Programs option in your menu to install it. Abiword, which kellemes mentioned, is there as well.
