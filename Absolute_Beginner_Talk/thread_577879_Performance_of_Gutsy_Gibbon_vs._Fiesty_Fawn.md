### Post by spxero on 2007-10-22
> **jazz1 said:**
> Downloading Xubuntu now. Is there a way to install on the same computer as Ubuntu and boot one or the other just to compare?

This seems like a dead thread, but in any case- to install Xubuntu on the same machine (or Kubuntu, if you like) just do this:
sudo apt-get update
sudo apt-get upgrade     <-- update/upgrade to the latest first
sudo apt-get install xubuntu-desktop <-- or kubuntu-desktop

Then, on restart, change the session type from Gnome (or Last Session) to Xubuntu or Kubuntu.
