### Post by pjkoczan on 2007-12-18
I generally use emacs, mostly because it's what I learned first. I think the graphical version of (x)emacs has a more forgiving learning curve than vi, but vi has a lot easier metakeys and escape-sequences than emacs. I haven't used gedit or kate extensively, but many people seem to like them.

Give each an honest chance, and pick whichever one you like best. All the major editors are very well-supported and have enough of a user base to help when you have problems. I don't think this about everything, but picking an editor is truly a choice of "whatever you like best."
