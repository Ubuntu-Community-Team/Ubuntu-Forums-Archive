### Post by Shatrat on 2006-12-15
The different versions of ubuntu differ primarily in the packages they come with by default.
Ubuntu, kubuntu, and xubuntu are pretty much the same but have different window managers as default.  Ubuntu CE is Ubuntu with some christian themed applications and stuff bundled with it.  They have their own forum here if you want more details.  Either Ubuntu or Ubuntu CE would be a nice place to start with for you, and I recommend Dual Booting.  There are eleventy billion threads on dual booting and the 6.10 installer sets it up pretty smoothly.  Since you havent used linux before you wont have a preference for gnome/kde/xfce or other window managers so just install Ubuntu and try out the other window managers if youre curious.

I don't recommend installing Edubuntu as it is really a server distribution with thin clients in mind.
