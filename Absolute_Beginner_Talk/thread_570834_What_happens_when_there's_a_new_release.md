### Post by scapalexis888 on 2007-10-08
When a new release of Ubuntu comes out, you can do two things: upgrade or do nothing. in Windows language, that is like having a new service pack being released.
You can do a simple upgrade or completely reinstall the new version, or not do anything. It's up to you. Linux gives you the freedom to do that. The new release is has more features but older releases are more stable

### Post by shad0w_walker on 2007-10-08
As said in the previous post, upgrading is very different from Windows upgrades. You will be able to install all the new packages onto your existing system and with any luck, should be able to just upgrade with out any real problems. You can of course reinstall completely if you want but its very optional.

Also, Ubuntu continues to get security upgrades after the next version is released (Can't remember precisely how long for) so if you are happy with the system you have and don't want/need any of the new features, there is no real reason to upgrade.

### Post by mivo on 2007-10-08
Also, your settings and configurations are saved in your /home directory, so these are generally not affected. If you have a separate /home partition, you could even install a completely different Linux flavour, and the installed programs would usually still find their settings and data. :) It's really very different from Windows, in a good way.

Upgrading from Feisty to Gutsy should be very painless. Might just take some time to download the new files, depending on your internet connection.

### Post by HermanAB on 2007-10-08
The trick is to always configure a separate /home partition.  Then when a new version comes along, you can re-install and keep /home intact - just don't format /home.  If you don't have a separate /home, well, now you know why you should have one...
