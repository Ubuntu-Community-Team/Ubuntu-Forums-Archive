### Post by Kimm on 2006-06-26
There are several things you can do.

You can try other Desktops, that look and feel different.
A Popular desktop for low-end computers, is XFCE4, for high-end computers, there Gnome or KDE.

In both Gnome, and KDE, you can move everything around. Did you notice that you can make the panels transparent? That you can shrink and enlarge them? that you can make them shorter or longer?
Try right-clicking on something on your panels, you will notice that you can move it (you need to uncheck "lock to panel" first).

You can also mix desktop environments... you can have the desktop from XFCE4, the KDE taskbar, the GNOME applications menu, and the windowmanager from OpenBOX.

The possibilites are pretty much endless.

### Post by Kimm on 2006-06-26
[QUOTE=maddbaron]how do u do that? i have kda desktop and ubuntu desktop installed if i add xfce4 how do i change it up?[/QUOTE]

If you want it exactly the way I said... do this (after installing XFCE4) in Gnome:

Remove the bottom panel (Right Click -> Remove Panel)
Run this command (from terminal, or ALT + F2): xfdesktop
Run this command (as above): kicker

If you do this, I also suggest that you start using Thunar as your filemanager, instead of Nautilus. Its faster, and nautilus will take over the desktop when you run it.

Commands:

xfdesktop                 For the XFCE4 desktop
kdesktop                  For the KDE3 desktop
xfce4-panel              For the XFCE4 panel
gnome-panel           For the Gnome2 panel
kicker                       For the KDE3 panel
xfwm4 --replace       For the XFCE4 windowmanager
kwin --replace          For the KDE3 windowmanager
metacity --replace   For the Gnome2 windowmanager

These are only a few commands... you can mix even more DMs if you want.

PS.
Note that mixing Qt and GTK will cause higher RAM usage (mixing KDE and Gnome/XFCE)
