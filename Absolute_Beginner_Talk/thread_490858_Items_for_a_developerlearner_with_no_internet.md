### Post by Cypher on 2007-07-03
The Ubuntu CD will have all the necessary development tools, so you wouldn't need to get those from the Internet. Of course, to get the updates to these packges, you'd need the Internet access.

Get the Kernel sources at [http://kernel.org](http://kernel.org). The latest stable Kernel is 2.6.21.5, click on the link marked F which is the full Kernel sources (linux-2.6.21.5.tar.bz2), do not click on the the version name which will just give you a patch from the previous released Kernel.

Hope that helps..
