### Post by davidgarcin on 2007-12-04
The website you link to uses a Windows Media Player plugin to display.  Obviously, this won't work in Ubuntu, so you have to install the mozilla-mplayer package using synaptic.  For how to install packages using synaptic, see [http://monkeyblog.org/ubuntu/installing/#installing_with_synaptic](http://monkeyblog.org/ubuntu/installing/#installing_with_synaptic)

Hope that helps.
