### Post by Cypher on 2008-01-21
This would seem to indicate a more basic issue with your Mobo..first and foremost..you will want to make sure that you are running the latest BIOS..

A lot of new Mobo's have the capabilty of upgrading the BIOS using a Flash stick. Check your manual to see if it can do that. Next, you might want to contact the customer support for Foxconn to see if they can tell you anything about how to get into the BIOS.

If all else fails and you are still within the return timeframe of the place you bought this Mobo from, you might want to return it or exchange it..

Personally, I've never heard of Foxconn and I've found that sticking with well known namebrands as far as key components go in the PC works out the best..so Asus, Abit, MSI, Gigabyte..can't go wrong with these guys..
