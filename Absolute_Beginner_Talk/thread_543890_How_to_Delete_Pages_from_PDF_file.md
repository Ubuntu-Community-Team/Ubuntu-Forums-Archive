I haven't manage to run the program, but I'm not at home with my Ubuntu machine right now. You should be able to follow the instructions given in the accompanying documentation:

> 
INSTALL AND RUN

Prerequisites:
pdfsam need a working Java Runtime Envirnment. It has been developed using Sun Java(TM) 2 SDK,
Standard Edition, Version 1.4.2 and tested on many Microsoft Windows version and on Ubuntu using
J2SE 5.0 and J2SE 1.4.2 .

pdfsam-0.7-b1.zip
Unzip the archive in a directory and double click the file pdfsam-0.7b1.jar or type the command "java
-jar INSTALL_DIR/pdfsam-0.7b1.jar" in a console.


You might need to install Sun Java from the repositories, look in Synaptic.

You downloaded the source, that shouldn't be necessary. The archive you need is named pdfsam-0.7sr1-out.zip.
