### Post by jordanmthomas on 2007-08-20
from their page:
**We Add CNR**   This is true.  There has been talk of offering CNR on Ubuntu as well.  Personally, I would never use CNR since apt-get (or pacman in my case) is pretty simple as it is.
**We Add KDE**    True I guess.  You could choose Kubuntu to get KDE out of the box or a simple one-line command will get you KDE in Ubuntu.
**We Add Enhanced Applications**     Not sure here, since I've never used Linspire.  I am not sure what they add to firefox or to nvu, but I imagine it's just something to make them work better in KDE.
**We Add Proprietary Software**    Also easily installable in any Linux distro...it's just a matter of, "do I feel like installing this myself?"  Also, many would argue this is a reason to NOT use a certain distro...just sayin'
**We Add Codecs**   same as proprietary software...not really that big of a deal to install yourself.
**We Add Drivers**   I suppose this means they install proprietary drivers.  A myriad people seem to have trouble doing this themselves (for some reason), so I'd give this one to them as an advantage.
**We Add Confidence**  :rolleyes:   no comment here.  I don't know what kind of confidence they could give you that a forum of hundreds of thousands of people couldn't.

Basically, what it boils down to is this:  do you want to pay $50 for someone to install things for you?  If so, give it a go.  If you'd rather install it yourself, don't waste your time or money.  If money's an issue, but you still want CNR or that extra confidence boost, check out [freespire]("http://www.freespire.org/").  I'm pretty sure you can get proprietary stuff with it very easily too.  Just glancing at the feature list, it's basically the same thing as Linspire only cheaper, so you could give it a try and see if it's the easiest or not.

Good luck in your distro hunts...I know mine lasted for a year or two before I finally settled.
:lolflag:

### Post by StooJ on 2007-08-20
As far as I understand it, Linspire logs you in as the superuser by default. This means the experience is more "Windows like" since you have the power to do/change/break anything without being challenged for the root password.

I would personally say that this is a negative aspect, since I'd imagine it would greatly reduce security and completely bypasses one of my most comforting aspects of a *nix distribution of any flavour, the need to prove you have the administrative rights to make administrative changes.

Even if I'm the only one using an Ubuntu box, sudo means I think a lot more carefully about what I'm doing and lets me feel safer messing around with the system and exploring, since there's no way I can mess it up as a normal user.

Sudo is great for a lot more as well, but that's one of my day-to-day reasons for loving it, especially since I am a linux beginner.
