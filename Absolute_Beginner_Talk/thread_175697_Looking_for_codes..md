### Post by skippy81 on 2006-05-13
The most imporatant command of all is 

> apropos word where word is the keyword you are searching for.  

Apropos is a command keyword search.  So if you want to know commands which tell you what your running processes are then just type 

> apropos process and it will list you all the relevant commands for dealing with processes.

Second most important command is man.  If your apropos keyword search listed "pstree" as a possible command for looking at your running processes, you could type > man pstree to learn how to use the command properly.

So baiscally use apropos to find commands and then man to work out how to use the commands effectively.
