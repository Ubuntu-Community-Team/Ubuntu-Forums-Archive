### Post by Six_Digits on 2008-05-09
From what I've read (having the same problem) it looks like when you try to input a search in the text boxes, the text is actually being put in the "Chat" area. To fix it, click over to the "Community Chat" tab and click anywhere - deactivating the text box field. The cursor will stop blinking in the chat, and you can then type in the search fields.
