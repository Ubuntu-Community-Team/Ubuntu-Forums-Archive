### Post by Malta paul on 2007-01-01
Sound Juicer has always worked fine for me.
I would suggest that you first check edit>preferences then:

Check CD Drive  - is set corect
Music Folder - is OK for you
Track Names - Album artist, Album title, is set 
                                                                - Number Title, is set
Output Format - set to say: Normally cd quality, lossy (ogg multimedia )
>Close
 
Tick the track you want to record >extract

Should now work :) Hope this helps
