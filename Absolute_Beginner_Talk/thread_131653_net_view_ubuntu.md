### Post by cjm5229 on 2006-02-17
Ubuntu can be operated from the gui just as completely as windows, however the CLI makes things easier and safer to manage your system once you get used to it. Some programs in Windows do not install shortcuts when the program is installed, then you have to go hunting for the program in Program Files and make a shortcut and then find a way to get it to where you need it. In Ubuntu "Applications Menu Editor" will create a shortcut for you and let you put it where you need it. In other words Windows is as easy as Redmond wants to make it. Ubuntu is as easy as you want to make it.\\:D/
