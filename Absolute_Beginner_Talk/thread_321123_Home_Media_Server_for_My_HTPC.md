### Post by Vidar on 2006-12-23
The desktop edition will do the trick nicely. Just set up a couple of samba-shares and you should be able to get it working just as well as I does now for you.

And yes, you can use the Terminal Service Client, this will act as windows "remote desktop".

Copying files in lunix is as easy, or easier, as in windows.

Dunno about the newsreader, but Im sure there are a bunch of them.. just don't use them myself.
