### Post by bodhi.zazen on 2008-02-08
I think the security of the two OS is independent unless you share files / partitions.

As long as you are running windows you are vulnerable, this is the nature of windows.

Many browser attacks are OS independent, such is the nature of java / flash.

By running one os Virtual you keep them separate, and I do not see security different then if you were running two physical computers side by side.

Unless you run Windows within Ubuntu and restrict (ie remove) internet access you can not really protect windows with Ubuntu. If you do this it is like running windows on a box with your network card unplugged.

So again, not much of a security boost.
