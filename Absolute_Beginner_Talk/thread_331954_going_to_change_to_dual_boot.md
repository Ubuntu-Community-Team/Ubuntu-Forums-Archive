### Post by jvc26 on 2007-01-05
You should be able to access Z (not that it will be called Z any more as the drive letters thing is a windows jobbie) What filesystem is it? If NTFS it may cause you problems, mine never caused me any when I dualbooted I just installed Ubuntu and my drive was there but there is a post knocking around at the minute about it causing probs)

The menu System->Administration->Disks is no longer there - there is also a post (sorry have not got the links to hand which is in the Newbie talk section at the moment) about what you can use instead.

If NTFS and you need read/write access you should use the NTFS drivers: [http://ubuntuforums.org/showthread.php?&t=217009](http://ubuntuforums.org/showthread.php?&t=217009)

Finally, it should be fine wipe windows (make sure all your stuff is backed up it could be an awful pain otherwise ;)) Then install 'buntu and it should automount the other HDD which you can then access and take a look at whats on it. Make sure you dont accidentally format it while installing Ubuntu - shouldnt happen as long as you make sure you dont rush and tell it to wipe the wrong drive :) And enjoy!
