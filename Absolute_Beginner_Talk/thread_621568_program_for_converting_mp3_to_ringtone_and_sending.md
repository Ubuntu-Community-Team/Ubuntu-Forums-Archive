### Post by qamelian on 2007-11-23
You can't convert mp3 to midi because mp3 contains actual audio information and midi doesn't. All a midi file contains is a set of instructions that tell a midi-compliant instrument how and what to play. The midi file itself contains no audio data. There are a few programs that attempt to create a midi file of a song by analyzing various characteristics of the audio data, but I haven't found one yet that does anything remotely close to an acceptable job.
