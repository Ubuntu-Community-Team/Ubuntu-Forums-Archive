### Post by HermanAB on 2007-11-10
Firestarter, Guarddog, Shorewall - these things are all the same really.  They are  scripts that configure iptables, that configures netfilter.

See the iptables man page for details - you would have to read the man page about 5 times before it will start to make some sense though, but it is really quite regular and not complicated once you get past the funny syntax.
