### Post by PMO6022 on 2005-12-02
If you are using e16, try middle click > maintenance > regenerate menus

This should also be done after you make any changes to the menu config files in ~/.enlightenment

### Post by PMO6022 on 2005-12-09
Sorry I didn't see this sooner!

If you middle click on the desktop and go to "help" you can get alot of info, including how to add/remove/change what is displayed on each menu.  Basically you edit text files in ~/.enlightenment.  You can create submenus, or however you want to organize your files.
