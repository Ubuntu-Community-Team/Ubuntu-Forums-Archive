### Post by chrisdugdale on 2008-04-15
VMware from [www.vmware.com](www.vmware.com) will let you install Windows *inside* Ubuntu! You just install the VMware into Ubuntu, then it lets you install Windows. It will make Windows run faster and more reliably! Takes a while to load the Windows disks though. You may find a quicker answer by using the Synaptic Package Manager to install Wine which runs a lot of Windows stuff.
