### Post by RomeReactor on 2007-10-03
Hi. That probably means you tried to download Xubuntu using Firefox's download manager, and it hasn't finished downloading, so you have an **.ISO** file measuring 0 kb, and a **.PART** file that's almost complete. Try opening the download dialog in Firefox again (press CTRL+Y) and see if you can resume the download. If you can't, I suggest you try the [DownThemAll]("https://addons.mozilla.org/en-US/firefox/addon/201") add-on for Firefox, or download the image using a BitTorrent client.

Did you get Xubuntu from [here]("http://www.xubuntu.org/get")?
