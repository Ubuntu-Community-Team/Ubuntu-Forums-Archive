### Post by kevinlyfellow on 2007-03-16
THere are a few reasons to run bit torrent.  First is that it saves the bandwidth of the website your downloading from.  This is helpful for open source projects since we often don't have a lot of money.  Second, it CAN be faster.  The more people that want a particular file, the faster it will go (opposite of how ftp will work!)  So for instance, when feisty is released, their servers may go down (since its going to be so popular ;-)), but torrenting an iso will be fast and is unlikely to stop.

To get bittorrent in your application menu, right click on applications and edit menus.  Then just click the checkbox next to bittorent (under internet).  But I usually just "open with" bittorent from firefox
