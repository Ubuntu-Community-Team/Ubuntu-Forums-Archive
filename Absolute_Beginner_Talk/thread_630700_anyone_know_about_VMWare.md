### Post by Paqman on 2007-12-03
I'm not sure that OS X will be easy to set up in VMWare, but Windows works just fine.

Fire up VMWare, connect to "local host" and follow the wizard for creating a new VM. Then start up the VM and install from your CD exactly like you're installing on an actual machine. Dead simple. After you've got the new OS installed on your VM you might want to install "VMWare Tools" on it, it'll improve performance.
