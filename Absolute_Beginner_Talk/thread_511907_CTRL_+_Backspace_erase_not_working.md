### Post by Pragmatist on 2007-07-28
Works for me in firefox and doesn't work in gnome-terminal.  Each application has its own set of keyboard shortcuts.  Sometimes certain shortcuts are used by many applications (like CTRL-z  to undo) and other times the  shortcut isn't as popular and some applications have it and some don't.  Many applications will let you set your own keyboard shortcuts, so you might want to research that.
