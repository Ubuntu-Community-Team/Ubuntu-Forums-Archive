### Post by H3g3m0n on 2007-04-02
You can do it to each individual file type by right clicking on a file of that type, clicking properties going into the "Open With" tab and choosing a media player.

For things like DVD insert check in System>Removable Drives and Media Preferences (This might be Feisty only since that area changed, but its probably in the control panel thing.

I don't know if there is a way to globally replace the media player, but after you have done avi and mpg you have probably done most of what you want.
