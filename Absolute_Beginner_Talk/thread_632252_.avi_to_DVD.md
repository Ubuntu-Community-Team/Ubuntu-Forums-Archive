### Post by Hospadar on 2007-12-05
Hey,
If you want to put it on a video dvd (like to watch in a DVD player) I would use "devede" (simpler, doesn't make menus) or "qdvdauthor" (more complex, makes menus, good if you have multiple smaller files)
The final filesize will depend on the bitrate you select for the target video, if you use the defaults you'll probably get a ~3 gig video with your 700 MB .avi depending on size/length/quality

Note that you can change the bitrate to whatever you want, and the default bitrate that devede and qdvdauthor will give you is waay more than the bitrate on your .avi, so it's using up unnecesary space, but if it's the only video on the dvd, who cares.

Note: If you use qdvdauthor, you should also install "videotrans" so it can actually transcode your videos

Also:If you decide you want much tighter control over the encoding process (Probably you won't) I would look into avidemux, it has some nice options for transcoding to dvd format video, then you can take those transcoded videos and author them to a dvd with qdvdauthor.  google "avidemux" for their website and more on how to use it.

*Also*: when using any of these programs, you need to be mindful of your tv/dvd player's format, either PAL or NTSC.  if you arn't sure which it is, you can always just try both and see which works, but in general US=NTSC  everywhere else=PAL.  in devede go to the properties for the video to change formats, in qdvdauthor, go to the transcoding option (double click the video)

### Post by sleepingdragon on 2008-01-08
DeVeDe is great if you just want to convert quick and have no fancy menus, it's the fastest converter I've used so far. You'll find it in the repos

DeVeDe conversions go straight to the movie, no menu, no chapter selection, etc. You'll also get the opton of a 60-second (or more) preview, so you can see the final result *before* starting a conversion. This can save a lot of time if the results are disappointing - there's nothing worse than spending ages doing a conversion only to find the screen ratio is wrong! 

There are a range of video and audio options too. Nothing fancy, but the ones given all seem nice and safe.

Output is via an .ISO file, so use a disk burner like K3B or Brasero that can burn disk images. Nero is OK for this too, but you gotta pay for that one.

Hope some of that helps.
