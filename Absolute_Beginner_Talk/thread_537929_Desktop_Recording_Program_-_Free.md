### Post by timcredible on 2007-08-29
you can convert the ogg file to any other format with ffmpeg.  it's command line, but pretty easy to work with - just type "ffmpeg -i <input file> -o <output file> -<tell it what format you want>".  ffmpeg is available via synaptic.
