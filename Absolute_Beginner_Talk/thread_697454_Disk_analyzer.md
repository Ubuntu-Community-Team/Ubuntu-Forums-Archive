Ther is a program called fsck (file system check) which runs every time you boot your computer. It is not necessary to run disk checks like you need to on windows. Junk files would probably accumulate in your home directory. You could use Nautilus to browse for unneeded files. You can remove temporary firefox files using the firefox Advanced tab in preferences.
