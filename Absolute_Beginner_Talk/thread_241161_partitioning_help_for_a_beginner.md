### Post by seshomaru samma on 2006-08-21
If you have several partitions in Windows (C,D,E...)you can just format one (not C though...)and install Linux on it, the Ubuntu installer will do it for you when you chose 'edit partitions manualy'.
If you have just one Windows partition ,you will need to resize your Windows partition.
You can do that Partition Magic on Windows or with Linux' GPparted
Read [this ]("http://psychocats.net/ubuntu/installing.html")for more info
