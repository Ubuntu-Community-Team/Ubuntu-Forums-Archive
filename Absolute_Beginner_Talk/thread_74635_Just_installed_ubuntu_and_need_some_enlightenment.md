Short answer: no.  Long answer: maybe.  There are some tricky things that you can do to adjust that using **modelines** but it is difficult and tedius.  You might want to save that for later.

> I have to just plug in the IP address and it will connect? I'm sure I don't run the disc I got from SBC(correct?). 

Give it a try.  That's difficult to answer without also being an SBC customer in your area--all of the phone companies do things a little differently.

You mean that it wouldn't power all of the way down?  If you are running the beta of Breezy then that's a known issue that seems to have been fixed yesterday (for me anyway).  Once you're connected to the Internet you could run an update.

A lot of them.  I'm afraid you'll need to learn the command line to get some things done in linux.  [Here's a tutorial]("http://linuxcommand.org") that I like.
