### Post by davidgypsy on 2005-07-31
[QUOTE=cadaver][font=Courier New]Which is better suited for an older computer? [/font][/QUOTE]

I installed Ubuntu onto an old AMD K2 with a 6gig HD and 64mg's RAM a while ago for the kids to plonk around on, and although it won't win any speed races, I was very surprised by how well it ran! This is a machine that refused to run XP, and was VERY slow with 98.

I'm not saying that I would recommend this setup ;-) but it just goes to show how little you really need to get a standard install of Ubuntu to run. KDE needs a little more. Do yourself a favour and get some more RAM, then go for it!

I tried out Damn Small Linux two weeks ago to see how it would run, and the live CD booted in about 60 seconds on my P4. I was expecting to wait for some time, and so it caught me by surprise. The live CD ran almost as fast as I would expect a low spec computer to run any other version of Linux. And it certainly is not lacking in features when you consider that it is only 48mb total size! Simply amazing, I can't wait to install it to some ancient machine just because I can!
