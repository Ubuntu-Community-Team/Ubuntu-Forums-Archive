### Post by lollikerwin on 2007-04-05
Hi RBird,

I just discovered Jarnal and am a noob myself, but here goes...  If you are still having this problem, check if you downloaded only jarnal.jar from the website or [jarnal-install.zip]("http://http://www.dklevine.com/general/software/tc1000/jarnal-install.zip").  The zip file is the one you need for pdf support.  Unzip it and run jarnal.sh.  From within Jarnal, go to File > Open Background.  Choose the pdf that you want to annotate as the background.

Cheers,
LolliK :)

PS I didn't need to install xpdf.
PPS As far as I can tell, the only differences b/t jarnal.sh and jarnalannotate.sh are formatting settings (ink color, lined paper vs. plain, etc.).
