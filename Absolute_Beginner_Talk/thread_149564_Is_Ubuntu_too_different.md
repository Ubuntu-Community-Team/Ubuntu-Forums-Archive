### Post by AndyCooll on 2006-03-24
The beauty of Linux is that it is flexible enough to be useful for both beginner and the more experienced folk alike. And Ubuntu is a prime example. 

For the beginner there is the handholding in Ubuntu of Automatix and many gui applications if that is what they wish. I've now set up a number of Ubuntu boxes and it has become quite straightforward and doesn't take long.

However that doesn't mean my learning experience is over, far from it. Linux can be almost endlessly configurable, and aspects I originally accepted the defaults of I now like to tweek. I've also now advanced to setting up a server using only the command line, still with Ubuntu. And then of course there are always more advanced projects that can be tackled.

And of course you can always dual-boot and perhaps try a more "difficult" Linux distro such as Slackware or Gentoo as mentioned. Those skills learnt with Ubuntu are still relevent. Sometimes you just have to modify them a bit for different distros, but essentially they are the same. I've also installed Fedora and Suse and although some things are a little different, much of it is familiar. This is because at the end of the day we are still running the same operating system, just a different flavour. A bit like once you learnt how to use Win98 or whatever, the leap to XP wasn't too daunting.

Ubuntu, and in turn Linux, can be as easy and familiar or as challenging as you wish to make it.
