### Post by adam234 on 2006-07-11
To install Blender simply open up your package manager (adept is Kubuntu's and I believe that synaptic is Ubuntus) type blender into the search box and when it pops up click the request install button.  After you have done this click apply (apply changes in adept) and boom your done.  Oh, you might need to enable the universal repositories. ### Post by DirtDawg on 2006-07-11
Actually, one of the coolest things about Blender is you don't need to install it in the traditional sense. The most recent release (from [http://www.blender3d.org](http://www.blender3d.org)) comes as a ".tar.bz2" file. This is essentially like a zipped file in Windows. Once you "unzip" the file (with the archive manager), navigate to the newly created folder, and inside is a file you can click (probably called "Blender", or something similar)to start. Yay!

Of course, try Synaptic first. I think it's in "Applications>Graphics>get new programs" or something similar. It just may not be the most recent release (Or maybe it is. I dunno).
