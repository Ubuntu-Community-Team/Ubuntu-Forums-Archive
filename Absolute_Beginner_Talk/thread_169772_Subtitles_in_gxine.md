### Post by raul_ on 2007-03-16
use xine-ui instead of gxine (it sucks)

```

sudo aptitude install xine-ui

```

then, everytime you type "xine" the gui will open

NOTE: xine uses 2 separate windows, one for the video, and one for the "remote" . To toggle the remote on and off press "g"

PS: I stumbled in this post, so if someone is interested just do this :)
