### Post by WW on 2007-02-26
Why not install it from the Ubuntu repository? There are several ways to do this:

-- Use Applications->Add/Remove...: You will find Blender in the Graphics section

OR

-- Use Synaptic.  Start Synaptic with System->Administration->Synaptic Package Manager.  In Synaptic, search for blender, right-click on the package and select "Mark for installation", then click on the "Apply" button.

OR

-- From the command line:
> $ sudo apt-get install blender

I'm not sure what version is in the repos but Blender have just released ver 2.43.
Blender is selfcontained so if your downloading from the Blender site just extract it to whereever you want and look for a file called blender in the newly created folder.
