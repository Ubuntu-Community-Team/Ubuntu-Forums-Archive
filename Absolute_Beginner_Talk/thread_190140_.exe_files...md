### Post by xXx 0wn3d xXx on 2006-06-05
[QUOTE=Niranjan81]how do i open/execute .exe files??[/QUOTE]
You can use wine. 
> sudo apt-get install wine

and add this line to your sources.list

> deb [http://wine.sourceforge.net/apt/](http://wine.sourceforge.net/apt/) binary/

Installed applications can be found in /home/USERNAME/.wine/drive_c/Program Files. If you wish to acess this folder you will need "Show Hidden Folders" enabled in Nautilus. (View > Show Hidden Files)
