### Post by tcrroadie on 2007-05-21
I know 100% for shure that your video card will work, because the first video card that I purchased for my desktop machine was a Nvidia Geforce FX5700, and it worked perfect.  Currently using a Nvidia Geforce 6800GT.

It's 64-bit :) Dualcore is great, actually, and they don't run much slower than their single-cored siblings.

You can run 32-bit ubuntu on it, which is easier because a lot of drivers (flash, etc) are not 64-bit yet, neither in ubuntu nor windows. If you are experienced with Ubuntu go 64 but if you are new like me go 32 :)

EDIT: I'll bore you with some more tech stuff. The AMD A64 cpu's have names like "3800+" because they can be compared with an intel 3.8 ghz in real-life performance, despite only running at 2.0 ghz. This is of course not entirely accurate, but close enough to make the pick easier for us non-techies:)
