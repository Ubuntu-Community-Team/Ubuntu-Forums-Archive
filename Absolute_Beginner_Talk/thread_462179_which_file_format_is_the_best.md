### Post by blue_shift on 2007-06-02
It's best to re-rip from CDs etc if possible, but I assume you can't do that in this situtuation. If you go from WMA to another format directly you'll be taking an unnecessary quality loss.

Technically, OGG is superior I think, in terms of quality versus file size. Plus, it's a free codec. However, mp3 is ubiquitous, so if you've an mp3 player with no ogg support you might want to use mp3.

### Post by reacocard on 2007-06-02
Well, converting from one lossy format to another isn't a terribly good idea, as you'll lose quality. Better to re-rip from CDs if you can.

As for format, OGG is supported very well in Linux/Ubuntu, and is a 'free' format. On the other hand, MP3s are trivial to add support for, and work with just about everything, although they take up slightly more space than OGGs of the same quality.

EDIT: A good program for converting from one format to another is soundconverter.
