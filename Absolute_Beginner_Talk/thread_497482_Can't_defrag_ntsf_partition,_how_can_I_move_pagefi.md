### Post by chrome saint on 2007-07-10
I just went into the settings and turned off system restore and the virtual memory functions and all the "unmovable green bars' disappeared.  After you defrag you can just go and turn the restore and virtual memory back on.  Turning off the restore function **WILL REMOVE ALL YOUR EXISTING SAVEPOINTS** so only do it if you don't mind.
