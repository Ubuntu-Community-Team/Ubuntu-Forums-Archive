### Post by thelatinist on 2008-01-18
If you are trying to burn an Ubuntu cd image, you do NOT want to choose the option to create a bootable CD.  You just want to choose the option to burn an image.  The image file you downloaded is bootable all by itself.

### Post by blueridgedog on 2008-01-19
You do not want/need to extract the files, you need to burn the ISO to disk.  Making a bootable disk with those file will not result in the running/installation of Ubuntu.

Open EZCD and tell it to create a disk from an image, burn the disk and you should be good to go.
