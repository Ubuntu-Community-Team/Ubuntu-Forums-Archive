Hello and welcome.
The best part about Ubuntu is that you get to try it out using the LiveCD before you "buy" so to speak. I suggest you burn the ISO, restart your PC with it in the drive and launch into Live mode. From there, if your belkin (which i have a belkin wifi dongle that works a treat) works off the bat, then it should work when you make a complete install.

Try the live cd. Use it over and over and be happy that you will be able to do all that you need before commiting to an install. In addition, remeber that Ubuntu LiveCD is smart enough so that if you want to keep Windows on your system and have Ubuntu, it will allow you to easily dual boot. If you want to know about dual booting come back after you have used the LiveCD and we can go through it.

### Post by darkknight045 on 2008-02-19
Here's the skinny as I have done a similar installation decision.

I purchased a Laptop that was a P3 1GHz, 256 RAM 20GB HD, Video ???

I tried Ubuntu Feisty at first, the performance was unbearable.  Quite literally I was waiting 5-10 minutes for single actions to load.  So I searched around and found that Xubuntu suggested far less resources so I burned an image, and tried it out.  

Words cannot adequately express how much better the performance was on this machine.  The Xfce Desktop is not as flashy as GNOME, which helps performance, but is also very utilitarian.  One of the handiest features it sports over Ubuntu is to be able to access a folder, right-click, and open a terminal already set to the directory you are in.  

Drawbacks: It locks in at 4 desktops, I haven't tried to get rid of them, or increase them, but I am fairly certain it cannot be done.  There's no Beryl, but with the specs you are running, Beryl wouldn't be an option.  There are some features, such as taking screenshots, that don't come with Xubuntu.  Worry not!  They can be easily added!

With both Feisty and Gutsy releases you can have a full set of icons on your desktop just like Ubuntu.  All said and done it has a layout very similar to GNOME.  

Hope this helps!
