### Post by kariopto on 2007-01-08
Right click on an existing icon -> Desktop -> create launcher.
Make sure you have "File/launcher icons" in the Behavior tab in the desktop preferences.
If you don't have an existing icon on the desktop then open /home/yourname/Desktop and create and empty file.
