### Post by testube_babies on 2006-07-25
The place to start would be to install msttcorefonts.

Make sure you have extra repositories enabled:
[http://www.psychocats.net/ubuntu/sources.php](http://www.psychocats.net/ubuntu/sources.php)

Then in a terminal type:```
sudo apt-get install msttcorefonts
```
