### Post by hyper_ch on 2007-03-25
Well, Ubuntu comes (as all Linuxes I think) with iptables as a firewall... basically all ports are closed except the ones that will be opened by installed software...

You don't neet an antivirus program (except if you are going to share files with windows computers or host an email server... then you can filter out windows virii from your files but they won't harm your installation... it's just a nice to do it for windows users)

Spyware: all written for windows... don't worry about it

Well, have a read here:   [http://www.psychocats.net/ubuntu/security](http://www.psychocats.net/ubuntu/security)

### Post by aysiu on 2007-03-25
Your best security tools are:

1. Not installing applications from outside the repositories
2. Using a strong password
3. Not telling anyone your password
4. Using encryption when necessary
