At this time Nvidia graphics are a little more linux friendly, however there is little difference.  Both are run via proprietary drivers that are fairly simple to install via the Ubuntu repositories (think cnet/download.com but with official packages no risk of malware, plus indexed on your computer ready to install with a few clicks).  Building your system is fairly straightforward.  I recommend [www.newegg.com](www.newegg.com) for parts, however [www.tigerdirect.com](www.tigerdirect.com) is also good.  The trickiest part is installing the CPU to the mobo, but just take it slow and make sure you line everything up properly.  If it doesn't want to go, don't force it.  When I installed my cpu it just dropped right in, so if your's does not, take it out and make sure its lined up properly.  Finally, what games do you want to play?  WINE is alright, but if you really want to game try Cedega.  You need to buy a subscription to get it (5$/mo i think), but if everything is supported you dont need to continue past the initial 3 mo.  If you want upgrades, or just to support the program, it is of course preferable that you stay a member.  Hope this helps, and good luck!

To be perfectly honest, that system will run linux really well (your current one).  Gaming and beryl might not be top notch however.  If I were you I would skimp on the vid card in exchange for more RAM.  You wont really tax your vid card much as in linux, (I barely tap the potential of my x800, granted its 16 pix pipes, but still), and the jump from 512 MB to 1GB (or more) in RAM is very noticeable.  Also, the x1600 doesn't work w/ the open-source driver if thats an issue to you.  Something like an NVIDIA FX-6800 would probably be your best bet.  The mobo and processor both look like solid choices.

One final thought:  For the cost of this PC, you could easily buy some more RAM and a new game console (ps3 or what not) and play far more up to date games (linux gaming on wine is usually several years old, if it ever gets there... wine is not really meant for games, so while it is free, don't expect a lot of game support).  Just a thought.
