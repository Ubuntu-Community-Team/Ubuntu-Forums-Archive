### Post by skymt on 2006-09-18
If you're a bit more adventurous, you could try giftd with a GUI. Once you set it up, it connects transparently to four different networks, including Gnutella (Limewire) and FastTrack (KazAA, or however it's capitalized).

Bittorrent isn't like the others. You need to go to a web site, download a .torrent file, and open that file with your client. For popular files, it's very fast. However, it can be hard to get less popular ones.

If you download a lot of less-popular files, you might want to look into the ed2k network, for which you would use aMule (or maybe xMule). It's always, always slow, but you always, always get the file. It has the best file availability of any of the networks.
