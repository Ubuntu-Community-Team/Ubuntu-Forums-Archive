### Post by grizzly on 2006-07-22
I own an ancient Palm IIIxe and  it works wonderfully well with ubuntu.
Infact it was the palm command line tools that showed me the power of commandline. :)
The tools are : 
pilot-link - command line stuff
kpilot( I hardly use it, coz pilot-link works great)
txt2pdbdoc - Ebooks! - 
plucker - this one has a gui and commandline version. Trust me there is nothing better than command line plucker for putting websites on your palm. 

Conclusion: go to pilot-link page, and search for supported palms. Feel free to PM me if you have any issues.
