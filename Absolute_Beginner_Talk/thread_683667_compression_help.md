### Post by pytheas22 on 2008-01-31
Ubuntu ships with a program called Archive Manager to do this.  Press f2, and type "file-roller" in the box that comes up.  I've never really used it to do anything extensive but it looks like a good utility.

You can also create archives of files anywhere (from the Desktop, file browser or wherever) simply by highlighting the icons of the ones you want to archive (use control to select multiple icons) and then right click, and select Create Archive.

I don't think that Archive Manager supports rar, however.  I know that you can install a utility called unrar to unpack rar archives using the command line, but I don't know about creating them, and I don't know if there's a GUI for dealing with rar.  There probably is, but you'd have to do some research.
