> OK we have moved to a new dedicated host and the new DNS has started propagating too. so in a matter of hours or days (depending on how fast your service provider's DNS cache gets updated), the automatix site (which includes wiki, forum and repositories) will be up again.

> Moving to a new host seemed to be a bad idea for Automatix as the new "shared host" refused to allow the kind of traffic Automatix demands and have throttled us to prevent their servers from crashing. ### Post by John.Michael.Kane on 2007-03-07
This should supply most of the codecs needed. The only one missing would be w32codecs. which is explained here [https://help.ubuntu.com/community/RestrictedFormats/WindowsCodecs](https://help.ubuntu.com/community/RestrictedFormats/WindowsCodecs)

```
sudo gedit /etc/apt/sources.list 
``` you should see a few lines with # next to it remove that symbol.

save the file,and run 

```
sudo aptitude update
```


Next up installing the files you want.

Run the command listed:
```
sudo aptitude install dvdrip brasero libdvdread3 gstreamer0.10-ffmpeg gstreamer0.10-plugins-bad gstreamer0.10-plugins-bad-multiverse gstreamer0.10-plugins-ugly gstreamer0.10-plugins-ugly-multiverse gxine libxine-main1 libxine-extracodecs ogle ogle-gui flashplugin-nonfree sun-java5-bin

```

After that.

Run this:
```
sudo /usr/share/doc/libdvdread3/install-css.sh
```

### Post by finferflu on 2007-03-22
Actually all Automatix does is to add extra repositories to your sources.list, so you can still install stuff, I think:

```
#AUTOMATIX REPOS START

deb http://archive.canonical.com/ubuntu edgy-commercial main

deb http://archive.ubuntu.com/ubuntu edgy-updates universe multiverse

deb http://archive.ubuntu.com/ubuntu edgy-security main restricted universe multiverse

deb http://archive.ubuntu.com/ubuntu edgy multiverse
#AUTOMATIX REPOS END
``` 

The only useful thing about Automatix for me is having those repositories added, and also I like the description of the software there, so I am informed about useful apps. It's just quicker, and I don't think it does any harm...
