### Post by steve.horsley on 2006-06-21
dustywb: Do you have a GUI with a mouse cursor? Or are you still at a text boot: prompt? If you have the boot: prompt, just hit enter and go for a cuppa. If you have a brown GUI with a mouse, you should have the installer icon on the left. Have patience - it's very slow off CD. If you get the GUI but no icon, reboot to the boot: prompt and press F1 for more options. You might find a safe mode that's more conservative driving the graphics card. As a last resort, burn the "alternative" text-mode installer CD instead, and install from there.
