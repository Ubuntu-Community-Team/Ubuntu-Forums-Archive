### Post by gatorbrit on 2006-04-10
I have 2 disks, several partitions, and a couple of external USB drives.  I would like to view graphically where all these are mounted and how much space they all have. ---

### Post by ubernoob on 2006-04-10
```
df -h
```
or
```
gnome-system-monitor
```
(the tab that says "Devices")
