### Post by autocrosser on 2006-11-09
OK--your problem is in the metamodes section--

And the reason you can't change them is because you need to be "root" to do so--it's a way to make sure when files are changed, the person doing it has the "permission" to do so....

In any case your metamodes should look like:
"1280x1024,1280x1024;1280x1024,NULL;1024x768,1024x768;800x600,800x600"

The reason for the "null' is so you can play games8)

OK--let's get our hands dirty--

Open a terminal---Menu>Accessories>Terminal

Type in:  sudo cp /etc/X11/xorg.conf /etc/C11/xorgbackup.conf

This makes a copy of your current file if you have problems

Then:  sudo gedit /etc/X11/xorg.conf
this opens text editor with root permissions--you can now edit the file

Goto BOTH lines that start with "Option" & paste this:
  Option         "MetaModes" "1280x1024,1280x1024;1280x1024,NULL;1024x768,1024x768;800x600,800x600"


Save the file after you have done this to BOTH lines.

Normally you can just restart X--but it is just as simple to reboot

When you as back at your desktop, Open Synaptic & download Display Geometry Switcher--it's a panel applet you can place in any panel--allows switch on the fly....
