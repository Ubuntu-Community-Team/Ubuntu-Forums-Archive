### Post by Perfect Storm on 2006-01-07
Download flashplayer here: [http://www.macromedia.com/shockwave/download/download.cgi?P1_Prod_Version=ShockwaveFlash](http://www.macromedia.com/shockwave/download/download.cgi?P1_Prod_Version=ShockwaveFlash)

extract the tar.gz file

Now open the terminal

```
 cd /where/you/extracted/the/flash/application
sudo sh flashplayer-installer

```

When it ask for installation path write:
/usr/lib/mozilla-firefox
