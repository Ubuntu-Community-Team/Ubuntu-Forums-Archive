### Post by luisromangz on 2007-12-02

### Post by Dr Small on 2007-12-02
The Server comes with no GUI by default, but that can be fixed (if you need one). But you can basically do everything with a desktop as compared to what you can do with a server. The difference is, the desktop comes with a GUI preinstalled and is a livecd whereas the server has no GUI isn't a livecd and can have server applications installed by default.
