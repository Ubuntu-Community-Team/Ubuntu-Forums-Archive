### Post by dstew on 2008-01-04
This seems to be a bug in Gutsy. The workaround is the command **killall nautilus**. This restarts the file display software. See [http://ubuntuforums.org/archive/index.php/t-592098.html](http://ubuntuforums.org/archive/index.php/t-592098.html)
