### Post by SphereX253 on 2007-09-20
<SOLVED>

I think.

What I did to fix my problem, was go into the compiz confg settings manager, window decoration, and type   "emerald"   (without " " ) in the command field.  When I typed emerald --replace in my terminal it did the same thing. My window tops disappeared. However, when I then typed compiz --replace the tops came back, along with the themes.

Hope that helps someone.  I was getting pissed that all the other suggestions that were working for others weren't wokring for me. Finally.. I think its fixed.
