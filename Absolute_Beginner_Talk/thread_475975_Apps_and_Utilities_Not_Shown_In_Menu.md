I’m running the Feisty Ubuntu desktop.  I get the impression from what I’ve read that Ubuntu installs a number of GUI apps and utilities that aren’t added to the Menu Bar.  Is that true and if so, what’s a good way to find those apps and utilities to see which ones are useful enough to be on my Menu Bar?

BTW... I've edited the Menu Bar and do realize that apps must be checked in the *Show* column in the editor to be seen in a menu drop-down list.

### Post by Sequent on 2007-06-16
Most apps are to be found in /usr/bin. As for which ones are likely to be useful, start up Synaptic and look for installed programs, Synaptic will give brief description. To put them into the main menu go to the System menu > Preferences > Main menu > Click on submenu you wish to add stuff to, click on new item and browse to /usr/bin (or wherever stuff is located) click on app name then open.
