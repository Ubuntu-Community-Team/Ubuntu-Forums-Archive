### Post by TheBoat on 2008-02-08
You need to boot your system from your CD/DVD drive instead of your hard drive.  At startup, I hit F12 to enter the boot menu and select the CD/DVD drive and Ubuntu will load instead of windows. Obviously, the disc needs to be in the drive when you select the drive in the boot menu.

### Post by Presto123 on 2008-02-08
It's pretty simple to do. Burn that lovely Ubuntu ISO file as an "Image" file at a slow speed to avoid errors.

You can then stick that baby in and reboot your computer and when it comes up with the menu, select "Start or Install Ubuntu" and you will be in live CD mode.
