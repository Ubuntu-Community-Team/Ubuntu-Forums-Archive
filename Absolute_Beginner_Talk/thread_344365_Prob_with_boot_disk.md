### Post by mikewhatever on 2007-01-22
A text installer of Ubuntu is similar to Windows installer. It does not have the fancy graphics of the LiveCD, so usually runs better on low end PCs. You can get it from Ubuntu download page, look for Alternate install CD.
I am not sure, but there may be an option of running a LiveCD in the text mode, also, the system requirements for Ubuntu only specify memory and HD free space size, so your P2 may just be fine. Still, it doesn't hurt to try.
Check this site for userguide [http://users.bigpond.net.au/hermanzone/](http://users.bigpond.net.au/hermanzone/)
