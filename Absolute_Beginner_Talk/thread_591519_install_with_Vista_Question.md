### Post by ezak on 2007-10-25
In any case, I would continue on with the installation and do a complete install without the migration, whether it detects or not. I have only had odd luck, in the "Migration Assistant" detecting my XP installation on another partition on the same drive. Haven't ever got it to fully work with Vista or any other OS for that matter.

Just my recommendation ;)

But, with that said. You can still be able to browse and also write to your Vista installation. Gutsy is configured to mount it by default. 

Hope that helps you, as I do the same on quite a few of the systems I dualboot myself.
