### Post by cowlip on 2007-03-30
It's a hidden file

Anything with a "." dot in front is hidden from file managers like Nautilus and Konqeuror, to keep it clean looking.  If you "show hidden files", you will probably find it.

Try also 'ls -a ~/Desktop' in a terminal to see if it lists that folder.
