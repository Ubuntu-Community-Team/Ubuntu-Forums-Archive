I have used Ubuntu 6.06 and then 6.10 and now 7.04 and I have found that the 7.04 install with fix a lot of the harrowing networking issues automatically, from the time you start up.  I would strongly suggest installing 7.04 as it offers much more anyway. 

However:
1)  Before installing 7.04 onto your laptop ...
2)  Create your network on your main Windows desktop
3)  If you have Windows installed on the laptop (dual-boot) then use Windows
     to establish a connection with the network on your desktop
4)  (if you dual-boot) then use Windows desktop + Windows laptop to check and satisfy that everything is working as expected in that new network
5)  Install 7.04 on the laptop while it is connected to the network by *wired* connection
6)  and 7.04 will do the rest for you, hands free

I did this today in fact, installed 7.04 onto a laptop with the network already configured, and newly-installed 7.04 created all connections with this Windows network.
