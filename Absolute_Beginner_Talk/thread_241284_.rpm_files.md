### Post by Perfect Storm on 2006-08-22
```
sudo apt-get install alien
cd /to/where/the/limewire.rpm/is
sudo alien -i limewire.rpm
```

### Post by Major_Stitch on 2006-08-22
Hi!

Use sypatic to get the alien package. Then in terminal write:
```
alien -d /path/to/the/rpm/package.rpm
```
I'm not sure if you need sudo. If you do just add it!

EDIT: -d option is just to convert it to .deb and the -i option (above) is for a dircet install
