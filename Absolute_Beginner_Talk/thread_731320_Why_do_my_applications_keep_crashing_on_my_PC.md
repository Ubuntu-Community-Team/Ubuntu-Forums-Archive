### Post by Tatty on 2008-03-21
> **3puppies said:**
> wat is terminal and how do you use it

Applications->Accessories->Terminal

That will load a terminal... try running your software from there (for example, to run firefox simply type "firefox" and press enter), then when the software crashes, look in the terminal to see if there are any error messages.  That  might give a clue.

To find out what to type to run other software, right click on the ubuntu menu bar (where it says applications, places, system) and then choose "edit menus", then you can browse the shortcuts in the menus and see what the commands are to run each application.  Though be very careful not to accidentally change something.
