### Post by jcrochford on 2006-11-12
kornoholio,

Where are you downloading '.dat' movie files?  I'm not aware of such an extension for movies.  What happens when you simply change the extension?  If you change the extension from an unregistered file type to one that is registered, you might solve your problem.  I've done this sometimes in Windows when I've tried to send a file that gets screened (i.e., kicked out) by M$ Outlook.  An example is a file with an '.xls [Excel]' extension.  If you change the Excel extension to, let's say, '.jpg', and then attach it to a message, the recipient using Outlook can simply change the extension back to its original and use the file as normal.  I'm thinking that this might work for you?  Just a thought...  Good luck!
