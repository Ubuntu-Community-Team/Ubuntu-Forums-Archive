### Post by Jucato on 2006-07-27
I'm presuming you are using Ubuntu right? (since you had to install Amarok).

AFAIK, even in Ubuntu, Amarok uses the xine engine. And it needs a package called *libxine-extracodecs* from the multiverse repository in order to play MP3s. Install that, and you're good to go.
