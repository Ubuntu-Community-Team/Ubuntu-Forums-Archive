### Post by jken146 on 2007-11-24
You probably downloaded a Ubuntu iso, an image of a (bootable) CD.  (The file name should end in .iso - if not go to [www.ubuntu.com/getubuntu](www.ubuntu.com/getubuntu)).  You need to use a CD-writing program to burn the iso to a CD and then boot your computer from the CD to start Ubuntu.  A guide: [http://www.psychocats.net/ubuntu/iso](http://www.psychocats.net/ubuntu/iso)

Good luck!

### Post by Taboo Mirage on 2007-11-24
What you (should have) downloaded is an ISO file.  An ISO is a CD image; you will need to burn the ISO onto a CD, then boot from that CD to install Ubuntu.  

I suggest taking a look at [this guide](https://help.ubuntu.com/community/BurningIsoHowto) for assistance with burning the ISO image that you've downloaded.  After doing that, I would then suggest taking a look [here](https://help.ubuntu.com/community/GraphicalInstall) for assistance with the installation procedure itself.

Take care.
