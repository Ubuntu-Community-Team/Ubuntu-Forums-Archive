### Post by land0 on 2006-08-29
First you need to select the **folder** where the icons are located that you want to use. You do that by clicking on the browse button on the top right hand side. Once you have found the folder that has the icons you want to use select it by double clicking it. Do you notice how all of the files in the folder do not show up? That is because when you click browse it is only looking for the **folder** that holds the images not the images themselves. Once you have selected the folder that contains the icons you want to use click on the open button. The original dialog box that had the default launcher icons will now be filled with the icons from the folder you selected. Now you can select the right icon for the job by double clicking it. ;) 

As for finding an icon repository go to the top left hand menu and select System->Administration->Synaptic Package Manager. After you enter your password and Synaptic opens click on the "Search" button. When the search dialog box pops up enter gnome-art into the text field then click the "Search" button. After a couple of seconds it should return the results for gnome-art and gnome-splashscreen-manger. Select both for install by clicking on the white box to the left of the application name. Now click on the Apply button located in between the Properties and Mark all upgrades buttons. After the applications install you can open it by going to System->Preferences->Art Manager. Click on the _A_rt in the top menu and select what type of art you would like to install. Any of the icon sets you install using the Art Manager are placed inside of your home folder in a hidden file named "***.icons***". ( hidden file names always begin with a dot "." ) 

Now in order to select one of the folders in the "***.icons***" folder for use with your custom launcher you will need to click on the browse button go to your home folder right mouse click in the area that lists all of the visible folders then select Show _H_idden Files. All you need to do now is browse through the "***.icons***" folder for the icon set you want to use and then select the folder that best fits what you want. If the icon set folder contains a "scalable" folder I would encourage you select one of the folders in that directory because they scale very well. 

I hope that helps if any part of this reply is unclear or confusing please let me know.
