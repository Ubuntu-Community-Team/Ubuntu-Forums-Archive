### Post by Bartender on 2006-12-06
Rudolf -
In Firefox I just go to File>Save Page As> and give it a destination, like Desktop or "Linux stuff" or whatever.  When you're done you'll see two files.  The one with the Firefox icon will open the page, the other is some sort of data file that accompanies the Firefox icon.  So you need them both.

Another really handy tool for use on a Windows machine is PDFCreator.  It's a free download.  It installs to your printer folder, so when you have a webpage up that you like...well, here
[http://www.ubuntuforums.org/showthread.php?t=278392](http://www.ubuntuforums.org/showthread.php?t=278392)
I wrote this down once before.  Try it, I think you'll like it
