### Post by qamelian on 2007-06-28
/usr/share/backgrounds is where you'll find any wallpapers in Gnome installed via package manager. Any personal wallpapers that you add through desktop preferences are cataloged in an xml file at ~/.gnome2/backgrounds.xml.
