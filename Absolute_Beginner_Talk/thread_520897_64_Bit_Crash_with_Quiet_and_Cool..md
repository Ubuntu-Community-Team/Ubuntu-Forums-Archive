### Post by n0npr0phet on 2007-08-08
I made a triple boot machine with XP, Vista 64, and Ubuntu on an AMD Athlon 64 4000+ and using a Biostar 6100-M9 board and using on board RAID 0.  It was fun to make that all work but what wasn't fun were crashes (lockup with repeating video signal) that happened only in ubuntu and Vista 64.   In XP (32bit) the problem never happened but on 7.04 ubuntu and vista 64 it happened every 20 minutes.  Well it took me several days but anyway disabling Quiet and Cool in the bios fixed the problem for both ubuntu and vista 64.

I hope this helps others find the problem faster.
