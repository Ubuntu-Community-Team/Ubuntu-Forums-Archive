### Post by ssam on 2005-12-10
i think you can sometimes get in a muddle when the bios, grub and linux disagree about the numbering of disks.

for example my bios calls the first hard drive it find zero, no matter whether it is master or slave.

you'll have an easier time if you put this disk as master, then everthing will agree that it is the first (or zeroth) disk.
