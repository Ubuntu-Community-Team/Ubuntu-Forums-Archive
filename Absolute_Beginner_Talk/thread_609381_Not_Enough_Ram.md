I just installed Ubuntu on an older IBM ThinkPad X40 (hard drive died, which I replaced) - computer had 256 MB RAM - installed Ubuntu 7.1 using the 'alternate' ISO - ran OK but the computer was still slow - bought an additional 256 MB RAM for $24 from Kingston - all programs run faster - cheap solution for me, but worked fine!
