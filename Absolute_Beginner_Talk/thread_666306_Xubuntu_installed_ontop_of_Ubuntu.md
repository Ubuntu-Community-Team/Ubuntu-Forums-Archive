### Post by xpod on 2008-01-13
If you are just installing the Xubuntu desktop from Synaptic then nothing at all will change as far as your settings & programs are concerned.You will just have the choice of *which* desktop environment you wish at logon......via the sessions menu.

It can be pretty messy having both lots of apps installed though and i much prefer to keep my desktops apart.Xubuntu aint just great on older pc`s btw.....it makes a nice fast new one even faster:)

HI and you can install xfce as the desktop environment and then choose between gnome or xfce on the login screen under sessions. You can use this command ```
sudo apt-get install xubuntu-desktop
``` Yes you may have to configure some of your settings again but it should not interfere with your gnome settings.

### Post by mali2297 on 2008-01-13

does this mean that Xbunutu wiil be a 3rd OS to choose from the boot menu?

Neither. If you install the package xubuntu-desktop, you will be able to use another desktop environment (Xfce) besides the default (which is Gnome). Both run on the same Ubuntu  OS. You can choose which desktop environment to use through the sessions menu at login.
