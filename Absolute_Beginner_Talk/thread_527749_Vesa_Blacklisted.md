### Post by mysticmatrix on 2007-08-18
You have an Intel i845 Integrated Graphics. 
Although its odd that that Intel Drivers are not installed by default, you can easily enable them. Check if you have the below package installed.
```
xserver-xorg-driver-i810 
```

Tip: This can be done by open Synaptic Package Manager and searching for package.

If package is installed, Simply replace the following line in your xorg.conf

```
Section "Device"
	Identifier	"Generic Video Card"
	Driver		"vesa"
	BusID		"PCI:0:2:0"
EndSection
```

by this

```
Section "Device"
	Identifier	"**Intel i845 IGP**"
	Driver		"**i810**"
	BusID		"PCI:0:2:0"
EndSection
```

and then restart X by pressing Ctrl-Alt-Backspace.

If the package is not installed, install it and do the same.

**NOTE: Do remember to backup your Xorg.conf first!!!!**
