### Post by ubuntu27 on 2007-10-14
GNASH is Flash PLAYER, not flash maker. 
So far, there is no Flash maker that is [Free (Libre) Software]("http://www.gnu.org/philosophy/free-sw.html") or [Open Source]("http://www.gnu.org/philosophy/free-software-for-freedom.html") Software. Your best bet is to install it using Wine, or simply use a virtualization. 

I cannot help you in those two, since I never used them.

**EDIT**
 I just did a search with Google and found the following:

**QFlash**
[QFlash]("http://sourceforge.net/projects/qflash/") is a c++ open source IDE to create flash (swf) animation files. It uses ming and qt libraries

[http://sourceforge.net/projects/qflash/](http://sourceforge.net/projects/qflash/)

**OSFlash**
[OSFlash]("http://osflash.org/") is a collection of tools and resources for open source developers who love the Flash Platform and want to expand its horizons! 

[http://osflash.org/ofla_program](http://osflash.org/ofla_program)

OSFlash seems to be for Code developers interested in building applications for Flash.


**2nd EDIT**

[Open Source Flash RIA development]("http://ae.hugeobject.com/archive/a-practical-example-of-how-os-flash-beats-macromediaadobe-ide/")


[Towards Open Source Flash Development]("http://www.actionscript.com/Article/tabid/54/ArticleID/towards-open-source-flash-development/Default.aspx")
