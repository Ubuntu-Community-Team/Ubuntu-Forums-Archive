### Post by kernel tom on 2007-12-17
For a firewall, not that you really need one, I'd go with Firestarter...
sudo apt-get install firestarter

you can then run it under Applications>System>Firestarter

Music: Amarok
Video: VLC/xine/totem/mplayer (pretty much all good)
Word: OpenOffice Writer (there is a package out there for reading docx files)
Excel: Gnumeric Spreadsheet Editor
PowerPoint: OpenOffice Presenter
Internet: Firefox
Messenger: Pidgin/GAIM
Ftp: gftp
Torrents: KTorrent
PDF: Xpdf
Photoshop: The Gimp
DVD creating: DeVeDe/DVD Author
CD/DVD buring: K3B

Probably leaving out few, but that should be enough to start

### Post by steveneddy on 2007-12-17
Firestarter for a new user is not necessary. Linux is by it's very nature, locked down from the first install, unlike windows, which is wide open, necessitating firewalls for any windows machines.

Go for the media codecs, a good music player (I like VLC) and some cool wallpapers.

I also like to go to gnome-look.org and add a couple of mouse icons and maybe a few nice window borders.

Other than that, go crazy! What do you want to do? The world is wide open.

### Post by atomkarinca on 2007-12-19
> **Erik. said:**
> Is there also a program for wachting tv via the internet?

If you're referring to p2p programs, for sopcast you can use [gsopcast]("http://code.google.com/p/gsopcast/") or [qsopcast]("http://code.google.com/p/qsopcast/"). I know that tvants player works perfectly fine via Wine. But with TVU Player, no luck.

If you're referring to watching streaming video, first install the [restricted codecs]("https://help.ubuntu.com/community/RestrictedFormats/WindowsCodecs") then install the firefox plugin for totem:

```
sudo apt-get install totem-gstreamer-firefox-plugin
```
