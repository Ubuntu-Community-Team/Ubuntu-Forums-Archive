### Post by Medieval_Creations on 2006-10-03
This should be all you need to do:

1.) Open a gnome-terminal session.

2.) Click on Edit then Current Profile.

3.) One of the check boxes there is “Show menu bar by default in new terminals”.

Once you uncheck that and restart gnome-terminal the menu bar shouldn't be there.
