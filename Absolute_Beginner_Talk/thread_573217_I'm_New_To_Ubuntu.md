Just doubleclick on the MP3 file and it will ask to install the codec.

What players are you used to on Windows? I like
Xmms and Rythembox

To install, go to Synaptic Package manager in:
System => Administration => Synaptic Package Manager
And search for them

Or simply type:
```
sudo apt-get install xmms
```

In terminal!

### Post by dptxp on 2007-10-11
Open add/remove.
Top right - select all software.
Choose Audio section (on left)
Select gstreamer files from right menu, at least the top two.
Add them.
Totem and Rhythmbox shall play mp3.
