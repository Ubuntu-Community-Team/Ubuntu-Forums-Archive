### Post by Cypher on 2007-04-23
First of all, the noise should NOT be physical. If it's something Beryl is generating, then as long as you find a way of disabling that you should be OK. Beryl does tax the graphics card and CPU, so if doing various things with it is causing the CPU/GPU fans to rev up for some reason, that might be OK as long as it doesn't do it constantly.

As far as turning off any sound, you can always mute the speakers to ensure that it does first go away. Then look around System->Preferences and see where these sounds are generated from.
