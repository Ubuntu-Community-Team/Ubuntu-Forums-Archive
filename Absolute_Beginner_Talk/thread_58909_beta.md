### Post by rpgcyco on 2005-08-22
> is there a apt-get install for breezy? i dont wanna install another iso couse im outta cds right now lol.

You can open your repos list

```
sudo gedit /etc/apt/sources.list
```

and change hoary to breezy, save and then run

```
sudo apt-get update
sudo apt-get dist-upgrade
```

Please note though, Breezy often breaks to an unusable state without manually fixing.

> and whats better in breezy? just wonderin

There are many topics in the Breezy Development forum concerning this. :)

> is there any snap shots? the link u gave me just had iso s lol.

... that's what a snapshot is. An installable ISO (once burnt) of the current state of Breezy.
