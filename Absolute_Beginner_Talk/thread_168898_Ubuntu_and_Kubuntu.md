### Post by gingermark on 2006-05-01
[QUOTE=mandrakethepenguin]most software off the repos work on gnome rather than KDE.[/QUOTE]
That IS misleading! All the software in the repositories will work in both GNOME and KDE.
The difference is the toolkit that is used, and some people like to have all their applications from a single toolkit. That said, many people don't care, including me! :)

In [this thread](http://ubuntuforums.org/showthread.php?t=161779) I wrote
[QUOTE=me]If I was to generalise, which is never a good idea, I'd say Gnome is good for beginners and experts, whilst KDE is probably better for intermediate users. The reason I say this, is that you can configure more from the GUI in KDE, which is why I use it. But there are lots of options.

Gnome has less config options via the GUI, which might suit newer users better, whilst I'm sure experts can just edit configuration text files if they want to change something.

Obviously this is big generalisation though, and the only real way to figure out which one is suitable for you is to try them both.[/quote]
And I stand by that. Please keep in mind that Ubuntu and Kubuntu share the same base, so you can have both GNOME and KDE installed quite easy, and chop and change as the mood takes you.
