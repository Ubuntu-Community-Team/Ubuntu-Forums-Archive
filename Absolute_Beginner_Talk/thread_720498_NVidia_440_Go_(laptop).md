### Post by firestormx37 on 2008-03-11
Well your xorg.conf is not configured to use the nvidia drivers.

Try running:

sudo nvidia-xconfig

This will backup your old xorg.conf and put a new one in place that is configured for the nvidia drivers.  The major changes it makes is changing the driver from "nv" to "nvidia" and loading a module.

### Post by firestormx37 on 2008-03-11
It looks like it is saying you don't have enough video memory to run compiz.  From what I could find about your card it looks like it is a 64 mb card, which is just under the requirement.  Compiz is graphics intensive, so I guess you will have to live without it on the older laptop.  Unless someone has a way around it.

EDIT: Correction: 64 MB = 65536 KB.  I don't know if you need to have more than 64 MB for Compiz or if 64 MB is enough.  If it is enough then there may be some other problem.
