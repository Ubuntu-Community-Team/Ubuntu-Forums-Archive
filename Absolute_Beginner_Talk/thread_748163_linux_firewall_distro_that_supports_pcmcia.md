### Post by jkeyes0 on 2008-04-07
I did a quick google search on IPCop, and some people have said they've used PCMCIA NICs with it.
[http://ipcop.org/](http://ipcop.org/)

If any of the versions of Ubuntu would run on that hardware, they'd have to be the really stripped-down versions. Perhaps Fluxbuntu (if it's still available) or Xubuntu (though that would be a stretch with 64Mb of RAM). Any of the server versions should run OK, but you don't get a GUI with that.
