### Post by Voxxi on 2006-11-25
Try running XP in safe mode. When you put up, keep hitting F8 until a boot menu pops up. Select "Safe Mode" from the list and hit enter. When it loads up it should ask you if you want to use System Restore to revert your system to a previous state. Try that, it's saved my butt a few times. ;)
