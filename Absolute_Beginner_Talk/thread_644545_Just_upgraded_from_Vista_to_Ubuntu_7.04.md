### Post by macogw on 2007-12-19
To install rpms (which are meant for RedHat Package Manager), first install a program called alien.  Then in the terminal, cd to wherever the rpm is (or just put the rpm in your home drive so you don't have to cd at all), then run "sudo alien <whatever>.rpm"  (replace <whatever> with the name of the file) and it'll make a deb.  Then you can double click on the deb and install it.

There are manpages for command line stuff that you can reach by typing "man <command>" (exit the manpage by typing the letter q).  Anything else's help page should open when you press F1, but if not it'll be in the Help menu.
