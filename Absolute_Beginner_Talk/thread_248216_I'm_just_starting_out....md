### Post by meng on 2006-08-31
That computer should be fine. If you're looking to connect to the router via a wireless card, then most PCI wireless cards should do the trick, so long as there's a Windows driver available in case you need to use ndiswrapper. I would avoid a USB-type wireless interface, as my impression is they can be more difficult to get recognized.

You are fortunate to have a separate machine without Windows preinstalled, and to go in with the attitude that this is going to be a fun experiment. In some ways, you may actually be disappointed if there are no glitches with the installation (which in my opinion, is pretty common these days with the current level of hardware recognition). If you run into trouble, be sure to post here and make sure your topic title is informative.
