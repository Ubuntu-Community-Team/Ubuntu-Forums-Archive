### Post by Golyadkin on 2007-06-19
The easy way of finding and installing software is:
    * Click at the Programs menu. Choose "Add/Remove".
    * Search for the sort of program you want to add. Example: type MP3 to see a list of mp3 software.
    * Check the box next to the software.
    * Click the "Apply" button.
    * It will show you a list of the software packages that it will install. Click OK.

### Post by Golyadkin on 2007-06-19
Well, what I did was simply Google for "installling nvu on Ubuntu" and it gave me a website that said: 
> On the Downloads page for Nvu, there is a special .deb file for Ubuntu installations. 
So I went to that [downloads page ]("http://www.nvu.com/download.php")on [www.nvu.com](www.nvu.com) and scrolled down to where it said > nvu-1.0.ubuntu.5.04.deb - Ubuntu 5.04 and Debian 3.1r0 and newer. (It might work with older versions of Debian and Ubuntu). Contributed by Louis Tim Larsen. so I clicked on that link to download the .deb file for NVU version 1.0.

These files that end in ".deb" can be compared to setup.exe files in Windows. They contain the program in a handy installer package.


I hope this helps you out a little.
