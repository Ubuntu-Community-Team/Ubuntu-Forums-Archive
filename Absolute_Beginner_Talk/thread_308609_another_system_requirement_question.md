You can always try using Xubuntu (runs XFCE) instead of Ubuntu (which runs GNOME) as well.  [You can download an ISO, burn it to disc, and then try it out just as did with the Ubuntu disc you have]("https://wiki.ubuntu.com/Xubuntu/Releases?action=show&redirect=XubuntuReleases").  Once you have Xubuntu installed, from a terminal or Synaptic (or whichever whay you like to download apps), you can download the metapackage for Ubuntu and see if that works out for you (sudo aptitude install ubuntu-desktop, that will pull all necessary apps for the Ubuntu install -- then on the login screen, you click sessions, and choose GNOME as your session, and then click "only this time" or something to that effect, login with your username and pass and you'll have GNOME (if all goes well).
