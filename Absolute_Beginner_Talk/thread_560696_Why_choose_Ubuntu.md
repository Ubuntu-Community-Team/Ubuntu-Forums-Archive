### Post by mcduck on 2007-09-26
No, you don't need much knowledge. Only will to learn new things. And of course computer hardware that is supported in Linux.

And yes, business people can use Ubuntu just fine. If your main concern is compatibility with Microsoft Office, that's no problem. OpenOffice can also save into .doc and other Microsoft formats, and also supports exporting your files to .pdf which will work on almost any computer. Actually OpenOffice supports far more different file types than Microsoft Office does ;)

Anyway, if your computer hardware is supported by Ubuntu you should be able to run the Live CD to try Ubuntu without doing any changes to your computer. It's a lot slower than installed system would be, but it gives you a nice opportunity to test if Ubuntu works for you.


I can't really tell you why you would want to use Ubuntu. But I can mention couple of reasons why I'm using it: I can actually spend my time with computer more efficiently with Ubuntu than with Windows as Ubuntu needs practically no maintenance, I don't need to take care of viruses, spyware, defragging the hard drive and cleaning the registry. I can just concentrate on doing the actual work. I also love being able to install thousand s of different programs without buying any cd's  or searching around the web and downloading the installers and the installing them one-by-one with the next-next-agree-next-reboot way. Instead I just open the package manager, select what I want to install and click OK and everything is handled automatically for me. Also keeping my computer and all software updated is easier as the same package manager handles every single piece of software  on my computer.

And if that's not enough I can fully customize the way my computer works and how my desktop looks like, to fit the way I want to work instead of adapting my working to fit the way somebody else has decided. And I also have a powerful tool, command line, that allows me to easily do tasks that might take hours with graphical programs. I'm doing lots of web pages and I often need to create couple of different sized versions of hundreds of images for web galleries. With command line I can type a command to do all that for me in about 10 seconds and then go and have a cup of coffee while my computer is doing the job for me.

And, last but not least, if I ever have any problems I can just come to these forums and I know that I either find the answer with the search or somebody helps me to solve it. For free, just because they like to help people. This is something you really can't get with Windows or Mac. :)

### Post by P235 on 2007-09-26
I did not have a strong knowledge of computers when I began to use Ubuntu.  I was a security conscious Windows user who decided to try Linux because some of my friends were also doing the same.  Only after I became accustomed to Linux did I learn more of computers in general and the features Linux offered that gave me more control over my system.  

For a smooth(er) introduction to Linux I suggest reading a few books about the basics of Linux.  Of course, if you do not like the library or books in general, there are plenty of online materials that will have the same educational effect of a book.  In a few days, I'm sure you will become familiar with Linux and learn that the Linux OS is not as complex to use as it seems to be.

As for word processing, Open Office does allow users to save in Microsoft Word format and many other formats as well.  If Open Office is not enough, one can purchase CodeWeavers' CrossOver Office program that enables Linux PCs to run Windows programs like Microsoft Word.  Wine is also a good, free program that runs Windows programs on Linux, but I have not tried running Microsoft Word with Wine myself.

If you want to try Linux without installing it you can try the Live CD or run Linux on Windows with VMWare.
