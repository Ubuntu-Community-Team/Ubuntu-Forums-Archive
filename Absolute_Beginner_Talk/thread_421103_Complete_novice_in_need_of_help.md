### Post by ajgreeny on 2007-04-24
If you have a wired network card, that should find your internet without problems, particularly if you use the dynamic IP address, rather than a fixed IP address.  If you have a wireless card setup, it may be more difficult, but probably not impossible.  Do a search with the name of yopur wireless card on this forum and you will probably find some answers.

To install programs, once connected to the internet, open synaptic and search for anything you want.  Once you have found it click to install.  Dead simple really, better than microsoft as everything is totally free.  Once you've got used to the differences there's no going back.
