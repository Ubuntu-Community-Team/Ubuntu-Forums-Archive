### Post by 3rdalbum on 2008-02-18
Your media card reader is a wildcard. If it's connected through USB (even internal USB) it should work brilliantly. If it's connected any other way, it might work, or it might not.

Try the Desktop CD.

ATI's drivers are getting better, so I hear; but it might not perform as well as it does under Windows.

The TV tuner card is another wildcard, but I hear that these things generally work. The Firewire is also very likely to work, but if you want to use it you will need to run a short command that enables unrestricted data travel through Firewire. Unfortunately the Firewire specification itself has quite bad security weaknesses, so you don't want it turned on all the time.
