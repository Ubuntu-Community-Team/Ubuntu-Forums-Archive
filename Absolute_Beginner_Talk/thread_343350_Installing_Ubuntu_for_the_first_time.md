### Post by jtibau on 2007-01-22
The dapper "desktop" cd suggests around 3 hundred something MBs of RAM if you want to install it on your system. I've seen it install on 256MB though so it should work.
I'll explain a bit how the desktop cd works. The one you ordered is this one btw.
GNU/Linux distros (ubuntu is a distribution/distro) recently have taken of offering Live cds. One of the uses for live cds is showcasing the Operating System and testing that your hardware works with it. About a year ago (I think) not all live cds where installable, ubuntu 5.10 wasn't, shipit would send you two discs (a live cd, and an installation disc). Even more recently (in ubuntu since dapper) they where able to manage installation procedures from the a live cd (ubuntu called it a desktop cd then). The idea is that it can do pretty much the same a live cd did (showcase, test, etc) but you wouldn't need a separate disc for installation.
So you are getting a desktop cd. When you stick it in. It will load a gnome session which will look just like the installed system (a looot slower though, considering you don't have much RAM). You can play as much as you like with this system, browse the web, try openoffice, etc. When you are ready to install (I would recommend closing everything else, RAM issue again) double click on the INSTALL icon on the desktop. Follow installation procedures and restart the computer.
If you aren't a power user in windows probably you don't know anything about partioning. Just make sure to backup your important data as you might end up erasing all the contents of your hard drive accidentaly hehe. As a matter of fact, it would be a lot easier if you didn't mind erasing everything in the installation procedure since you wouldn't have to worry about partitioning.

Well I hope this explanation helps. Good luck and have fun :)
