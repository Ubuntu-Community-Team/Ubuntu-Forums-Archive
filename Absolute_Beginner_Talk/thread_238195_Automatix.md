Question about automatix -- if I use it to install something that's already installed, will it uninstall old versions/other versions of the program. For example, my azureus install from apt-get is not working correctly, and I was thinking of using automatix to install azureus from there as well as java. But will it uninstall the old versions of azureus and java? Additionally, I would like to upgrade to gaim 2.0, which automatix will install. Will this uninstall my prior version of gaim? Thanks for any information you may have.

### Post by bulldog on 2006-08-17
Automatix can update,not uninstall.

If Azureus won't work uninstall with aptitude or synaptic and reinstall with Automatix.

Gaim will be updated to 2.0 as far as I know and you won't need to uninstall first.
