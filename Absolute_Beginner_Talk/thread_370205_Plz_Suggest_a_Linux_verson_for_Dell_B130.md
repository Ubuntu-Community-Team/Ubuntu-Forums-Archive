### Post by Bachstelze on 2007-02-25
KDE is not very buggy in itself but the KDE Kubuntu runs is very modified from the "standard" KDE, so it introduces new bugs - and I don't like the Kubuntu artworks either.

If you decide do go with Debian, don't download the "stable" version (aka Sarge) because it is very old (inless you enjoy KDE 3.1, Firefox 1.0 and a 2.4 kernel :p), instead une the "testing" version (aka Etch) which has everything you need - don't let the word "testing" frighten you, it is as stable - if not more - than anything else.

[http://www.debian.org/devel/debian-installer/](http://www.debian.org/devel/debian-installer/)

Go with the "weekly build" images. There is a graphical installer but it uses a framebuffer, not a X on a Live CD. About your modem and wireless, if you managed to get it working in Ubuntu, they should also work with Debian, what are their specs exactly ?
