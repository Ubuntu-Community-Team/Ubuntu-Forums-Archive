### Post by catlett on 2006-07-17
as you can see from this excerpt 
```
#
# If you have edited this file but would like it to be automatically updated
# again, run the following command:
#   sudo dpkg-reconfigure -phigh xserver-xorg
```
You can run this command to update your configuration. 
```
 sudo dpkg-reconfigure -phigh xserver-xorg
```If that doesn't work, you can go through the whole configuration again with  ```
 sudo dpkg-reconfigure xserver-xorg     
```This will take through a configuration of your monitor, video card and keyboard etc. The more info about your monitor and video card the better. You might want to look up your monitors  refresh rates and resolutions before you run it. You can usually go to the web site of the company that made your monitor or video card and get a pdf or html with all the specifications on it.
Also if you have a nvidia or ati card there are seperate drivers for those video cards, installing them will give better resolution
