### Post by rykel on 2006-09-17
> **irw said:**
> I've not used this system before; just to clarify, do I need to remove the current version I have installed, before installing the new one?

You can choose to remove the current version, of course, although Ubuntu has made it "compulsory" for you to keep it... without the current version, you cannot display the Ubuntu "About Ubuntu" file, which uses Yelp, which depends on the built-in Firefox.

HOWEVER, if you install the latest version using the official Linux installer from Taj Morton's website, you do NOT have to remove the current version. Simply install the latest Firefox as User, instead of Root.

Please note that if you have SCIM running in the background, the new version may not work. In fact, Adobe Reader and Gizmo Project will NOT work as well. The solution is to remove SCIM.

Hope that helps! If you need further clarification, please write.   =D>
