### Post by glennric on 2008-03-10
If you are using compiz (Desktop Effects) install compizconfig-settings-manager, and go to System-Preferences->Advanced Desktop Effects Settings.  Select General Options and then the Desktop Size tab.  You should be able to figure it out from there.  

If you are using metacity or another window manager you will have to wait until someone else helps you.

---

### Post by forrestcupp on 2008-03-10
If you are using Compiz, go into your settings for Compiz and choose General Options.  Then click the 'Desktop Size' tab and change the 'Horizontal Virtual Size' to however many workspaces you want.
