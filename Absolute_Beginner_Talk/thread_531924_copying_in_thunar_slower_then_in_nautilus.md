### Post by Dave54 on 2007-08-22
Personally I have found that using nautilus for large file transfers is both slow and not dependable. If I tell it to copy a large folder (like from one hd to another) it will take a long time and copy 10% of it. (I have no idea why)

Therefore, when I copy large folders, I always use terminal, and a command like:```
cp -vr /source/directory /target/directory
```
