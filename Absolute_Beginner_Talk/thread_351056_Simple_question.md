### Post by annda on 2007-02-01
right-click on the keyboard applet, select keyboard preferences, go to layout options. under "Group/Shift Lock Behaviour" check the key combination you want - "Alt+Shift changes group"

also, Alt+F9 minimizes windows and Alt+F10 maximizes them (you can see the default keyboard shortcuts when you click on the icon in the top-left corner of any window)
