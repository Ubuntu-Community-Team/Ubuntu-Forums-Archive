On your new PC run Thunderbird for the first time (this is so it creates a profile, so don't worry about giving the right answers) and then close Thunderbird. The profile should have been created in a hidden folder (i.e. .mozilla-thunderbird). Once the profile is created you can just copy the contents of your old profile into the corresponding new folder and restart Thunderbird.
