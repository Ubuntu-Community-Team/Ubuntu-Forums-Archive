### Post by a9bejo on 2008-02-24
> **kriukov said:**
> I use mc and nautilus, but I'd like mc to be somewhat like Far ([http://www.farmanager.com](http://www.farmanager.com)) so that you could extend the functionality with plugins, use keyboard shortcuts, etc.

I used to work for a company where Far was used extensively. It was pretty awesome to see how efficient people can work with that tool after years of using it and learning all the tricks.

However, there is a reason why these [orthodox file managers](http://en.wikipedia.org/wiki/Orthodox_file_manager#Orthodox_file_managers) are not that popular on unix and linux systems:  The unix shells like bash (or better: [zsh](http://zsh.sourceforge.net/)) are so powerful that an orthodox  file manager is a lot less interesting than it was on windows.  

I personally use Emacs to work with files.  Emacs "dired mode" is even a lot more powerful than Far.  Like Far, one needs to invest a lot of time to become really productive with Emacs.  But then it becomes an really awesome killer-application.
