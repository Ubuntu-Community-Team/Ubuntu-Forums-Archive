### Post by taurus on 2007-12-31
Edit /etc/fstab

Applications -> Accessories -> Terminal
```
gksudo gedit /etc/fstab
```
and change the last two lines

```

/dev/sdc1 /media/Misc ntfs umask=222,utf8 0 0
/dev/sdb1 /media/Programs ntfs umask=222,utf8 0 0

```
to look like these.

```

/dev/sdc1 /media/Misc ntfs-3g defaults 0 0
/dev/sdb1 /media/Programs ntfs-3g defaults 0 0

```
Save it and then install ntfs-3g driver for your machine.

```
sudo apt-get update
sudo apt-get install ntfs-3g
```
Now, reboot.

### Post by taurus on 2008-01-01
You can also include the force option in /etc/fstab for that partition so it would mount automatically each time you boot.  Again, edit /etc/fstab

```
gksudo gedit /etc/fstab
```
and make the line for /dev/sdb1 to look like this now.

```
/dev/sdb1   /media/Programs   ntfs-3g   defaults**[COLOR="Blue"],force[/COLOR]**   0   0
```
Save it and reboot.  Now, see if /dev/sdb1 is mounted to /media/Programs.

```
df -h
```

p.s.  Replace the space with a , between defaults and force--defaults,force.
