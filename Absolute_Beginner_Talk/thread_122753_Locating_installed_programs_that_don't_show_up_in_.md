### Post by DonQuixote on 2006-01-28
I'm just curious if there is a quick and dirty way of locating an installed program that does not show up in the menu after installation?

I have noticed that there have been a few programs that I have installed through Synaptic that do not show on the menus.

Thanks!

John

---

### Post by timetunnel on 2006-01-28
Open Synaptic again, goto "Settings->general" and check "Show package properties in main window". After applying this settings,  choose the package you installed, then select the tab "installed files"  that should have appeared below the list of packages
