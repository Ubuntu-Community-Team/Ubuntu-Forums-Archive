### Post by maxamillion on 2007-02-17
If it has successfully installed from the cd that you have before and won't now, you either have a horrible scratch on the cd or it sounds almost like you might have a hardware problem ... and since it is happening during the live session it would more likely than not be an issue with your RAM.

I would recommend running the memtest86+ from the live cd on the computer and letting it run over night (it takes a while to complete)
