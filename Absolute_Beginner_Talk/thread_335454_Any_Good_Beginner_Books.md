### Post by Sunflower1970 on 2007-01-10
Hi there, I'm a newbie, too, and I ordered two books on Ubuntu. The first one is *Ubuntu Linux for Non-Geeks: A Pain-Free, Project-Based, Get-Things-Done Guidebook*
[http://www.amazon.com/Ubuntu-Linux-Non-Geeks-Project-Based-Get-Things-Done/dp/1593271182/sr=8-1/qid=1168450756/ref=pd_bbs_sr_1/104-1362861-4830358?ie=UTF8&s=books](http://www.amazon.com/Ubuntu-Linux-Non-Geeks-Project-Based-Get-Things-Done/dp/1593271182/sr=8-1/qid=1168450756/ref=pd_bbs_sr_1/104-1362861-4830358?ie=UTF8&s=books)

and the second one is *Ubuntu Unleashed*
[http://www.amazon.com/Ubuntu-Unleashed-Andrew-Hudson/dp/0672329093/sr=1-1/qid=1168450841/ref=pd_bbs_sr_1/104-1362861-4830358?ie=UTF8&s=books](http://www.amazon.com/Ubuntu-Unleashed-Andrew-Hudson/dp/0672329093/sr=1-1/qid=1168450841/ref=pd_bbs_sr_1/104-1362861-4830358?ie=UTF8&s=books)

Bought both through Amazon. The first book arrived Monday and has been an enjoyable read, and I'm waiting on the second one to arrive now. 

I've also printed out all the documentation from ubuntu's help files and gotten some good stuff there, but really, the best place for info has been these forums. I've been able to find (almost) all my answers by doing a search on these forums for whatever I've needed to find. :)
