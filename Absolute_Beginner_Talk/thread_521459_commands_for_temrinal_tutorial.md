### Post by proalan on 2007-08-09
this is a popular tutorial on bash (Bourne Again SHell)

[http://www.linuxcommand.org/](http://www.linuxcommand.org/)

some tips

press tab whilst typing to display possible command matches,
press tab twice will offer to list all commands,
type 'history' to list all the commands you've used in a session,
'history 10'  will list the last 10 commands,
up and down cursors will scroll through your recently used commands,

theres far more in the tutorials
