### Post by ComplexNumber on 2006-05-05
i forgot to say, to install the actual clock, go on the command line and type in something like 'gdesklets open clock.display'. i alwasy do that on the command line, anyway, but thats personal preference. you can actually install the desklets by dragging the 'clock.display; file onto the gdesklet GUI which should appear in your menu after you've installed gdesklets.
after you've installed it, it will say something like "starting the gdesklets daemon.....blah blah" and the clock will appear where you cursor is. you can then position it where you like. to move it, press both buttons down on your mouse and move it that way. to configure it, right click on the gdesklet and it will give you an option to change the dimensions and colour etc.
