### Post by bodhi.zazen on 2007-10-06
What files and how many ?

The problem is windows wants a .exe or .pdf or .txt on the end ...

You can ignore that informational message if you are just making a backup and would be restoring the files to Linux.

If you are wanting to actually transfer those files to windows, it would be nice, but not absolutely necessary, to add the appropriate information. You will just get a dialog box in windows asking you what program to open the file in question with is all ...

### Post by FrozenFlame22 on 2007-10-06
You're going to need to go through the files you intend to burn and check for characters that the Windows operating system doesn't allow.

Here's the characters you'll need to remove or change: \ / : * ? " < > |

And here's where I got this info: [http://en.wikipedia.org/wiki/Long_filename](http://en.wikipedia.org/wiki/Long_filename)
