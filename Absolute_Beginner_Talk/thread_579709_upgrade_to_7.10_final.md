Hi, don't worry I was using RC 1 as well. Ubuntu updates installed automatically are all u probably need. Just do these steps:
sudo apt-get update
then
sudo apt-get upgrade
then
sudo apt-get dist-upgrade
If all this is done and nothing new comes up then thats it.
