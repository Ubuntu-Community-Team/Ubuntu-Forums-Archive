### Post by Haegin on 2007-10-02
Hi,

I use brasero which is in the repositories. This gives the option of being able to add files later (so multisession by a more user friendly name) when you burn each disc. I think it is on by default.

Gnomebaker can also do multisession but in a rather weird convoluted way. This post should help:
[http://ubuntuforums.org/showpost.php?p=923765&postcount=6](http://ubuntuforums.org/showpost.php?p=923765&postcount=6)

If you are a Kubuntu user or just want a KDE program then K3B is a great burner with support for pretty much anything so I am sure that can do multisession.
