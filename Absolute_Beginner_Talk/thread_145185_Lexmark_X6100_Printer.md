### Post by TLE on 2006-03-15
If you have the possibility to choose some other lexmark drivers in the installer, then try that and see if it works. If you aren't allowed to try that automatically then you can get some drivers yourself [here]("http://www.linuxprinting.org/printer_list.cgi?make=Lexmark") and then choose to supply the driver yourself in the installer to test them. I also had to try a couple of drivers because my exact version number was not there either.
