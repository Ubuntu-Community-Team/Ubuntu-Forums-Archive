### Post by meborc on 2006-03-27
if you have some version of ubuntu:
go to terminal and type
**sudo gedit /etc/apt/sources.list**
change all **breezy/hoary/warthy **words to **dapper**

then save/close
[B]
sudo apt-get update
sudo apt-get dist-upgrade[/B]
-------------------------------------
for info on dapper flight5 [http://www.ubuntu.com/testing/flight5](http://www.ubuntu.com/testing/flight5)

there are links to isos as well, if you don't have ubuntu installed yet

have fun ;)

oh, and dapper still has some problems, but it is better than breezy in many ways... speed for one...
for more info on dapper problems refer to [dapper forums]("http://www.ubuntuforums.org/forumdisplay.php?f=111")
