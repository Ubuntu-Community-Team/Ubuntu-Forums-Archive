### Post by Nythain on 2007-07-05
one of the options on that main installation screen should be "install in text mode"... thats the option you want... you should never see the ubuntu/kubuntu splash screen with the loading bar, that means you selected teh first option, wich should say something like "Run livecd/install" or something to that effect.

if you dont have and option that says "install in text mode" you might need to download and burn the alternate install iso... wich for most advanced ubuntu users is the preferred iso to install from
