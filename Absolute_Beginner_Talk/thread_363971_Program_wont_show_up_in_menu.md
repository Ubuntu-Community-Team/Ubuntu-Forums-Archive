### Post by tbroderick on 2007-02-17
gnuchess is a terminal/console program. You can add it to the menu by creating a entry by using this;
```
gnome-terminal -e gnuchess
```

You might want to install xboard. That way you will have a nice mouse driven GUI for X. It will use gnuchess as it's engine.
