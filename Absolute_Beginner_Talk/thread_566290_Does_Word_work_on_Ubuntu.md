### Post by multifaceted on 2007-10-03
> **Five_Iron_OBrien said:**
> I'm computer illiterate how do I do a dual boot?  Also I've heard of virtual computers what is that?

Definitely read up a little on disk partitioning. What you actually do is allocate a portion of your hard drive for the new system. For example: You can set aside 30GB from your 80GB hard drive to use for Ubuntu. It actually looks intimidating at first but if you read a little it will start to make sense.

If you partition and install Ubuntu in it's said partition, you will be prompted by the boot loader or **GRUB **upon startup. You can choose to either boot Ubuntu (which is default) or, you can scroll down and select Windows and boot into your old operating system will all files and applications intact.

Like I said, read up a little on it before you attempt to do it :)
