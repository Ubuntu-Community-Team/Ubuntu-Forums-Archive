### Post by mikeyphi on 2008-01-08
If all is indeed OK - you should find your network connections listed under the Network Manager (Systems-Administration-Network) 
Make sure 'Wireless' is enabled and enter any info like DNS etc.

All you should have done with Madwifi is install a driver for your specific wireless card - there won't be any reference to be seen in the normal user system.
