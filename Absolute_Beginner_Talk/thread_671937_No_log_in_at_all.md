### Post by GGLucas on 2008-01-19
I have the same problem, log in starts, the panels appear for a split second, then it restarts X. I would presume it is either a compiz or an nvidia driver issue, since I'm currently using the open nv driver instead of the proprietary nvidia driver, and it lets me log in (albeit at 800x600), I'm currently trying to reinstall the latest drivers from nvidia.com, I'll let you know if I make any progress.

Edit:
I fixed it by doing Ctrl+Alt+F2 at the login prompt to drop to a virtual terminal, then executing the latest driver installer from the nvidia site.

```

sudo /etc/init.d/gdm stop
wget http://us.download.nvidia.com/XFree86/Linux-x86_64/169.07/NVIDIA-Linux-x86_64-169.07-pkg2.run
sudo sh NVIDIA-Linux-x86_64-169.07-pkg2.run

```

That fixed it for me, it's for 64-bit, though, I imagine you need another driver for 32-bit machines.
