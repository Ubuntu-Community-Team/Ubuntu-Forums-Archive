Hooo boy :P

1) Ubuntu will run perfectly well on your machine. I have a similar setup, although my processor is a little faster than yours. Regardless, Linux beats Windows hands down for me, it runs like lightning.

2) Yes.

3) No - what's more, norton won't even run on Linux. .exe files are for Windows, our executable files don't have extensions. You won't need anti-virus protection, but there are anti-virus programs available if you want peace of mind. The chances of you getting infected are pretty much nil.

4) So long as you back them up beforehand, yes. OpenOffice can handle MS Office files.

5) Which one? There are hundreds available. I personally like Amarok, but you should just try them out and see whichever is best for you.


So, on the other stuff. To install programs in Ubuntu, you just click Applications, then Add/Remove. You will be presented with a list of available applications (they are stored in the 'repositories', similar to an ftp directory. Select which ones you want, then click 'apply'. To remove applications, just de-select them and click apply. You don't need to worry about manually updating your software - you will be notified when updates are available, regardless of the program.

You can, of course, install programs which aren't available in the repositories. You need to look for files ending in .deb. These are similar to Setup.exe in Windows. Simply download your program.deb, double click it, and then click 'install'. Occasionally you may need to compile software yourself, but don't worry, it's not nearly as difficult as it sounds. If you ever encounter such a thing, just ask here and we'll help you out. The best course of action is to just stick to the Add/Remove method, there's thousands of programs there for you to try out.

### Post by KCormier on 2007-05-01
First off, how do you connect to the internet?  Dial up can be a super pain to get running on linux.  Broadband is a much nicer experience.  If you're connecting with a wired network then i wouldn't expect any problems at all!  If you use wireless, then it depends on your network adapter.  Some are support no problem, others are a struggle.

As for ubuntu not needing an antivirus, this is simply not true.  There are security vulnerabilities in linux.  Just very few virus writers bother writing viruses for linux as they're all preoccupied writing for windows!  Also, not all software that runs on linux is either free or open source!  AVG, while it is free, is closed source code as it's only free for personal use.  If you're running a server, yes even a linux server, be prepared to pay!!

I've used linux off and on over the last 2 years and after taking a 6 month break or so decided to try ubuntu out again.  I'm thoroughly impressed with it's stability and speed.  It seems lightyears ahead of other distros i've tried and I strongly recommend giving it a shot.

One of the beautiful things about ubuntu is that you can boot from the cd and it gives you a completely working setup with absolutely zero install!  That means it doesn't touch your windows install.  It can be quite slow because it's running off of cd but lets face it.....cd access is muuccchhhhhhhh slower than hard drive access.  If you like how it looks and you like the sound of a faster, more stable os with at the moment very few security problems don't worry about it feeling slow from the cd.  It is SOOO much faster when installed from cd!

### Post by knightnet on 2007-05-01
I don't recommend Norton/Symantec on Windows, I've seen too many problems caused.
Under Linux, I'd recommend AVAST or Panda. Both good AFAIK. You only really need them though if you are working on documents from other sources and want to ensure that you are not passing on viruses.

Excel/Word support is quite good now. If you are using Gnome (e.g. std Ubuntu) then OpenOffice is your only choice. If you are using KDE (e.g. KUBUNTU) Koffice also has some support. You do need to be careful though as support is not 100% and this can catch you out. You can also use WINE with OfficeXP or, for full compatability when nothing else works, VMware/VirtualBox - though you may struggle with a slower MC with only 512MB of RAM.

Media playing is excellent! I used VLC a lot under Windows and that is still good under Linux but there is lots of choice. 

You might find KUbuntu more to your liking if you are a die-hard Windows user like me. It's a bit more Windows like I think.

I've just swapped over both an old laptop and my desktop from Windows and it's fun!
