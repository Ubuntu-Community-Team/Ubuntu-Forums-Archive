### Post by taurus on 2006-06-15
You need to remove the # in front of all the lines with universe (and multiverse) in /etc/apt/sources.list.  You can edit it by running this from a terminal, Applications -> Accessories -> Terminal.
```

sudo gedit /etc/apt/sources.list

```
Then, you would know exactly what others mean when they say universe...  ;)
