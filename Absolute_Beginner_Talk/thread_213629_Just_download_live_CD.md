### Post by T700 on 2006-07-11
1.  Download the Live CD file (the ISO).

2.  Use your Windows CD burner to burn the ISO to a disk.  Be sure you "burn the ISO" and don't just copy it.  There will be a setting to burn it on your Windows software.

3.  Place the CD in your CD drive and reboot.  If your PC is set to boot from the CD (many/most are), it will take you to Ubuntu.  If not, you must go in your PC's BIOS and change the setting to allow this.  Post your make and model of PC if you need help with that.

4.  When done, remove the CD and reboot and you will have Windows again.

Paul
