### Post by Tomosaur on 2007-01-08
You won't be able to do anything permenant using the LiveCD (unless you've very clever and start mounting hard drives), except for installing Ubuntu permenantly. The general rule of thumb is 'if it's working in the LiveCD, it'll work when you install Ubuntu'. Some things may require some extra configuring post-install to work properly.

### Post by harmeet on 2007-01-08
I don't think you can use winmodem in linux. If you need to use modem you will have to find a real modem. I don't know if they manufacture any, because I can't find one in local stores. For your printer, Tom is right, if it works in live CD boot, it will work. Otherwise, check the manufacturer's website if they have any drivers for linux. If not, try to find opensource driver.
