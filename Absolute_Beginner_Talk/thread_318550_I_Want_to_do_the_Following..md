### Post by mykalreborn on 2006-12-14
for the torrent thing you can use the bitorrent ubuntu comes installed with, but it's a little to simple for my taste. [azureus]("www.azureus.sourceforge.net/") has a linux versions, and it's faster to set up than the windows version. if you have some friends who download using torrent clients you can use [tribler]("http://www.tribler.org/"). it has a really nice option to use a friends bandwidth to download faster.
there are more torrent clients out there, but these are the ones i use.

the ntfs can be used for read-only purposes very easy. but for write it can get messy. i suggest you stick to fat32.
