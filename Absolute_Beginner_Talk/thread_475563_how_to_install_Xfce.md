### Post by bapoumba on 2007-06-16
When the install is done, you will be able to choose to start a session on GNOME or on Xfce from the GDM login (click "sessions" or something similar, I do not remember). You can also keep GNOME as the default session (choose Xfce manually) or the other way around.
