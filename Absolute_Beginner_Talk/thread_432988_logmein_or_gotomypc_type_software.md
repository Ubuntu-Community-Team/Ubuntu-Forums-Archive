### Post by lazyart on 2007-05-04
It's still VNC based.  Since I do have a windows machine in my house (actually, bunches of them) I use logmein to get to the windows box, then VNC from there.

Another option is Hamachi, which puts all of the machines (Windows, Linux, OSX) you tell it to on a virtual private network, then you VNC to them directly.
