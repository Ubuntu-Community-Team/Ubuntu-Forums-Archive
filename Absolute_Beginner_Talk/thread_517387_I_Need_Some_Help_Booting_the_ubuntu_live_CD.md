### Post by taurus on 2007-08-04
How did you burn it to a CD?  Did you burn it as an image file, not a data file?  In other words, you are not supposed to just copy that one big file from your harddrive to a CD since it's not going to work.  If you are not sure, look at the content of the CD while you are in Windows and you're supposed to see a bunch of files and directories, not just one large file that you've downloaded.

[https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28burn%29%7C%28iso%29](https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28burn%29%7C%28iso%29)

Then, you need to get into the BIOS and have the system to boot from the CD-ROM first instead of from the harddrive.
