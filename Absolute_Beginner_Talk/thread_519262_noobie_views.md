### Post by Hobo Joe on 2007-08-06
Alternate install cd is for cases where you can't run the live CD becuase of lack of video support. 
Alternate install CD is the text-based installer, the terminal is something different.

GDM and KDM are, to put it simply, window managers. But unless you want to do something crazy, you don't need to know about that.
