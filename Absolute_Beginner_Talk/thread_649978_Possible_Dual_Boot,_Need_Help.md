### Post by Moop on 2007-12-25
Installaiion is the same for a different hard drive or a partition. You will need to set your Ubuntu hdd to boot first in the bios and then you will have the choice to boot Vista or Ubuntu. 

During installation when it asks how you want to partition the disk choose the correct hard drive and use entire disk and you'll be set. 

Take a look at Envy for installing your video card drivers. It should install the latest linux driver from nvidia. 

You can check if your internet works by booting from the live cd. It should all work out of the box. 

I don't "spin the cube" but it's not hard to configure once Ubuntu is installed. 

Good luck! :guitar:
