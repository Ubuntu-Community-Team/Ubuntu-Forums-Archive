### Post by ckaya on 2007-02-25
It is my favorite P2P application in Windows platform too. Gtk-Gnutella and Frostwire seem to be the best choices as substitution for Ares, although they use a different network. 

Also, here ([http://ubuntuforums.org/showthread.php?t=198945](http://ubuntuforums.org/showthread.php?t=198945)), I've found something you might be really interested in.

You can try to google "gift" and "gift ares" for further information.

(When it comes to P2P, never forget about Bittorrent. There are some Linux native clients, but utorrent is reported to work well under WINE too, if you are more used to Windows P2P programs.)
