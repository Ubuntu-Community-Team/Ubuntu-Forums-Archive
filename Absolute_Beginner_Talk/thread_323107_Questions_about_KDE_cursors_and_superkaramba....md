### Post by Jucato on 2006-12-21
1. (Dapper) System Settings -> Mouse -> Cursor Theme tab
(Edgy) System Settings -> Keyboard & Mouse -> Mouse -> Cursor Theme tab
You need to restart X (Ctrl+Alt+Backspace) for the changes to take effect.

2. Most Superkaramba themes are locked in their places. To be able to move them around, you need to unlock them by right-clicking on the theme and choosing the Unlock option.

Hope that helps.
