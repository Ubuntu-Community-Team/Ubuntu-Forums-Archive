### Post by Ba|der on 2006-10-20
Thanks for the link. I seached that forum and found the answer here:
[http://www.oooforum.org/forum/viewtopic.phtml?p=177967](http://www.oooforum.org/forum/viewtopic.phtml?p=177967)
Copied from above link:
"Change the Default template that Calc uses for new (blanc) pages.

Open a new blanc Calc.
Select all and change the font to your likings.
*****
Don't use hard formatting to change the default font for the whole document.

Use F11. Right-click "Default" and select "Modify". Use the font tab. 
*****

Then File > Templates > Save and give it a name (like MyOwnTemplate)

Again File > Templates but now choose Organize

Search for your Template (MyOwnTemplate) and select it.
Click the small black triangle on the Commands button
Most lower option is Set as default template. That's all

Open a new Calc document and your font should show as you installed it in MyOwnTemplate"

The same approch worked for OpenOffice Writer, so it's consistent :)
