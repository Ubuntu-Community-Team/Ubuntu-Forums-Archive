### Post by Najand on 2007-05-05
Open a terminal and type:
```
alsamixer
```
Then push Enter key and then change the volumes you want to change.
For microphone push TAB to go to capture section and then push "m" for the ones you want to mute.

### Post by Najand on 2007-05-05
