### Post by vaximily on 2007-02-13
All the CD images are stored in a file ending in .iso.   All a .iso file is, is an EXACT image (copy) of a CD that was created, including boot sector information and everything.

You can go:
[http://ubuntu-releases.cs.umn.edu//6.06/](http://ubuntu-releases.cs.umn.edu//6.06/)

For all of the CD images.  For a non-Live version you'll want the appropriate Server Install CD for your system (in my case AMD64, you'll probably use the Intel x86 installer unless you know you havea 64bit Processor).

To burn that .iso file image back to it's original form on a cd, follow this guide:
[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

### Post by vaximily on 2007-02-13
Assuming you burn the .iso image to a cd and don't just burn the .iso file to a cd (the .iso contains a whole image of a cd, you have to use a burning program to turn that image back into a bootable cd), then yes, the CD will be bootable.

And yes, the alternate cd would be best.

Again, the instructions at:
[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)
work fantastic.  I've used them on Windows and Mac OS X.
