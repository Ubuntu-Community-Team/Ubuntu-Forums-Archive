### Post by drummer on 2005-07-06
The 2.9gig partition is enough for the root partition ( / ), I think Ubuntu uses about 1.2 gigs with the default install so 2.9 should give you enough space to expand/install new software later on. The 10.7 should be then used for the /home partition, where all your personal files will be stored.
