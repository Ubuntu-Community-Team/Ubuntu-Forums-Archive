### Post by igknighted on 2007-03-24
Basically, its all about how they look.  The core is the same, they just wear different clothes so to speak.  Try looking at the screenshots to see the differences.  Basically Xubuntu is a little more lightweight... less features than Kubuntu and Ubuntu, but runs quicker (especially on older hardware).  Ubuntu uses the Gnome desktop, Kubuntu uses the KDE desktop.  Each desktop has their own set of applications, although KDE apps can be installed on gnome and vice versa.  In fact, if you want you can install any one of them and then install the packages for the other two very easily and try out all 3 at the same time to see which you like best.

Keep in mind that the default brown of Ubuntu is very easily changed, as is the default look of the other two.  Gnome is by far the most popular desktop (mainly because it is the flagship product of the line).  KDE has a bit more features and customizability, and looks a little more windows-esque by default (although gnome can look very windows-esque with a bit of work too).  Basically it is all personal choice.  There are many other desktop environments you could try once installed too if none of these tickle your fancy.  Check out gnome-look.org and kde-look.org for an immense amount of themes and other tweaks for the desktops.

In short, there is no best.  Try them and see what you like.  Luckily, there is no wrong choice here :)
