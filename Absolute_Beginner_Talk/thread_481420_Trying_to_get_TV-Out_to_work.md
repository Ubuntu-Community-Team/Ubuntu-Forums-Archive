its /etc   not /ect

/etc/X11/    with a capital X

if you backed up your xorg.conf, simply find it in the /etc/X11 directory and rename it xorg.conf and reboot (sudo shutdown -r now)

to rename a file, current name is "xorg.conf.BACKUP" and the desired name is "xorg.conf" type:
```

mv xorg.conf.BACKUP xorg.conf

```

assuming you're in the /etc/X11/ directory

oh also, you cant run gedit in the console because its an X-window progam, you'll have to edit the file using a command-line program.

the easiest one is called "nano"

```
sudo nano /etc/X11/xorg.conf
```

some useful commands are: 
CTRL+O -> "overwrite" , same as "save"
CTRL+X -> exit



And if all this nonsense still wont get your display back, try reconfiguring X by running this command:

```
sudo dpkg-reconfigure xserver-xorg
```

it will prompt you a bunch of questions and rebuild the /etc/X11/xorg.conf file for you
