### Post by Rocket2DMn on 2007-07-06
Your live cd can act as a repository, but you will be using the versions of apps that were available at the time the disc was compiled and setup for download, so they might be a little outdated.  I have not heard of anybody creating a customized repo disc, but the live cd is an option.
