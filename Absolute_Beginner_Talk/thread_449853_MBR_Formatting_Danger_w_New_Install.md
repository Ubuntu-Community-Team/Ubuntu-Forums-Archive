### Post by drs305 on 2007-05-20
New dual boot install on old computer. Alternate CD says Windows won't like the FAT16 formatting of the boot partition (about 30MB). It's been on the system that way for years. The message says it's formatted with 2K clusters and Windows expects 0K. I am very reluctant to allow any formatting of this partition and on other installs have always chosen to ignore the warning (to no ill affect).

Is it safe to allow the Ubuntu install to mess with the formatting of the boot partition?

---

### Post by freebird54 on 2007-05-20
There should be no need to allow Ubuntu to format the partition.  If you actually want to run with FAT 16. and your version of Windows will do that - go ahead and let Windows set that up.  It is better to let Windows set itself up before you add Ubuntu anyway.  It *IS* best to leave some space for Ubuntu, however, as it saves steps/time etc later in the dual-boot setup process...

Hope this helps...
