### Post by Donshyoku on 2006-04-12
If you want to use custom themes you have found off of a website (such as gnome-look.org), you can install them easily too.

Open the themes window that was mentioned before.  Choose "Theme Details" off to the , then go to the Icons tab.  

Drag and drop the file you downloaded to this.  It will install instantly and you can choose to use them within another window border and control theme.

Hope that helps!
