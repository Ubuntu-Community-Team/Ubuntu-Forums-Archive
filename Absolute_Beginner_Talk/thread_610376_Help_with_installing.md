### Post by rsambuca on 2007-11-11
One of the benefits of using Ubuntu is its repository of software which contains over 20,000 packages.  To install anything, go to your top menu panel and select "System -> Administration -> Synaptic Package Manager".  Enter your password when prompted, and then you can search for any package you want.  If you search for devede, you will find it in there.  Just select it for installation and click "Apply".  The package, along with any required dependencies will be installed.

As an alternative, any package in the repositories can be installed via the command line.  In this case, you would enter a terminal (Applications -> Accessories -> Terminal).  At the prompt you would enter ```
sudo apt-get install devede
```
