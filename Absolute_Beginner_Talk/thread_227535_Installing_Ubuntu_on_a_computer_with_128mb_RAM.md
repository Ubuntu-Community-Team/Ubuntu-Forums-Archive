### Post by GreenHawkIA on 2006-08-02
Just another thought...you think the problem is the lack of RAM, which is a good possibility.  But I have found that improper burning of a CD is much more common for stalls like that.  Make sure you are burning the ISO at a low speed and run either the MD5 sum for the CD or the integrity check from the initial install menu before you use the CD.  Doing that always saves me a lot of time in the long run, no matter what I'm installing.  Good luck - and I have Ubuntu on a low-system too, and it runs great.
