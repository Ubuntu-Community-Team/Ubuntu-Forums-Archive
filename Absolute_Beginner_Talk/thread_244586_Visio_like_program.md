### Post by Marsolin on 2006-08-26
[ajaxSketch]("http://linuxappfinder.com/package/ajaxsketch") claims to be able to create drawings that can be imported into Visio. It's not a Linux app, but it is web based so it can be run from Linux.

They also claim that the imports are possible because they are SVG files.

As for native Linux apps, in addition to [Dia]("http://linuxappfinder.com/package/dia") that gumbald mentioned, [Kivio]("http://linuxappfinder.com/package/kivio"), [KChart]("http://linuxappfinder.com/package/kchart"), and [XFig]("http://linuxappfinder.com/package/xfig") all are worth a look.
