### Post by aysiu on 2006-02-07
You're asked for a password for the installation CD because you have to set up a new user--you're **creating** a new password, not using an existing password.

The live CD is a different story--it logs you in automatically without a password. If you want to log out and log back in (or hibernate a laptop and then take it out of hibernation), it's a good idea to change the password to something else before doing so.
