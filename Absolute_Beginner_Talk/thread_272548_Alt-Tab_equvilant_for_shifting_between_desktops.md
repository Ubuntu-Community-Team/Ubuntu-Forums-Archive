### Post by Kateikyoushi on 2006-10-06
CTRL+ALT+Left/Right arrow key.
If you want to take the window to the next workspace hold the Shift as well.

### Post by Marsolin on 2006-10-06
I'm not sure about in Gnome, but this can be set in KDE through the System Settings program under Regional & Accessibility. There is a Keyboard Shortcuts section for "Walk Through Desktop List". It doesn't havea default in Kubuntu, but it can be set to any combination you want. I always change it to Win+Tab.
