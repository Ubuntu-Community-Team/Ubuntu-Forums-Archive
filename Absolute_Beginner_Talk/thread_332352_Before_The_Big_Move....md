### Post by meng on 2007-01-05
You need to download/install codecs on Ubuntu in order to play those formats. It isn't hard, but it's not included by default for legal/financial reasons. Burning software is included by default, and if you don't like the default applications there are free alternatives.

If you want mp3, etc. to be installed by default, you should look at other distributions like PCLinuxOS or Linux Mint that have these installed by default.
