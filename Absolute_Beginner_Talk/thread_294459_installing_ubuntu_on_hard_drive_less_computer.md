### Post by K.Mandla on 2006-11-06
> **bruenig said:**
> Without a hard drive you can run the live cd but you can't install it. So anytime you reboot, you will lose all of the changes and such that you have made. You probably ought to get a hard drive.
This is true. If the [live CD]("http://mirror.cc.columbia.edu/pub/linux/ubuntu/releases/edgy/ubuntu-6.10-desktop-i386.iso") works for you, you could, conceivably, turn it on once and let it run indefinitely. If you turned the machine off, you'd have to start over again from scratch. Some people like that, actually.

It will depend on the guts of the computer, but it's even possible to [run a server this way]("http://www.ubuntuforums.org/showthread.php?t=268121").
