To compile the program you will need to be in the correct folder to use 

```
./configure
```

In your case it would be the 3ddesktop folder, once you have un-rar'd the file, you will then need to 

```
cd
```

(Change Directory) Into the newly created folder.


A example of this could be (Assuming the folder is on your desktop)

```
cd Desktop
```

then

```
cd 3ddesktop
``` <Probaly named different

then

```
 ./configure 
```
