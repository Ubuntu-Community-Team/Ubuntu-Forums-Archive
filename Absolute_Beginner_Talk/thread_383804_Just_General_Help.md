### Post by mikewhatever on 2007-03-13
Ubuntu Live CD has a partition manager called GParted. The installation process, including partitioning is described here [http://psychocats.net/ubuntu/installing](http://psychocats.net/ubuntu/installing)

To change the default OS to boot, you'll need to edit the menu.lst file, which is GRUB boot loader menu you see at boot [http://users.bigpond.net.au/hermanzone/p15.htm](http://users.bigpond.net.au/hermanzone/p15.htm)
