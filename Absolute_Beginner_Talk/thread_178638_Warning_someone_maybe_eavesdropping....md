### Post by Borniet on 2006-05-18
Not necessarily, I sometimes have it when starting up a few programs at the same time, and starting to work with them immediately (while others still booting up). This can cause a newly started program to suddenly catch the input meant for another program, and give the impression to your system that someone else is taking over, while it is not necessarily the case.
Try keeping an eye on when you get this message.
