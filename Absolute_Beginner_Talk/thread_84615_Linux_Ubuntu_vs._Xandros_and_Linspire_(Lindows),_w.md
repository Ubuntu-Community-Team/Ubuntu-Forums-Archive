### Post by klug2 on 2005-12-11
Linspire is by far the best Linux distribution in my opinion- if you don't mind personally contributing to open source without writing a line of code.  Here is why I say this.  By paying for Linspire & Linspire services (CNR) your putting money into open source.  Almost all the software they write is open source.  In addition they sponsor a ton of open source projects like FireFox, both Kopete & Gaim (default in Linspire is Gaim), NVU, Lsongs, Lphoto, Wine, IRMA (Language translation), Reiser4 (filesystem), and many others.  They host the Desktop Linux Summit and wineCONF (windows compatibility developers conference).  

I don't discount Xandros simply because they don't contribute back to the community but because Xandros networks doesn't have enough software available whereas Linspire repository (CNR) does, and is continually growing with both commercial & open source software.  On the other hand Xandros does have an advantage at the moment in certain areas- they have CrossOver Office integrated very well within Xandros OS & it integrates well with MS Windows network in a business environment (LDAP).  For the home user Linspire can match and beat what Xandros offers in terms of compatibility with MS Windows (if needed) because they offer Win4Lin (home edition $29) which can boot in about 10 seconds and run ANY (almost) MS Windows application.  Downside to this is it requires an MS Windows license, but chances are you already have this.

My view of Ubuntu is this.  Ubuntu hands down is a great distribution but it lacks integration.  How anybody is suppose to get Java, Macromedia Flash, & Streaming video support is beyond me.  It took me a half dozen complex steps to get it all working and I've been using Linux since 1995.  As far as installing software goes Synaptic is difficult to impossible to use for new users coming from MS Windows XP or Mac OS X.  In addition it's missing essential peaces of software for a modern desktop operating system.

As you can see I really think Linspire is the answer for everyone on the desktop.  It's not that people shouldn't use Ubuntu.  I think it's one of the top 3 distributions but unless your really into Linux, programming, or have someone else maintaining your PC it is not a good distribution to use.  No user should have to open up a terminal window to install software and without a fairly complete repository of both Open Source and commercial software it doesn't fit the bill.  Even Linspire doesn't maintain a complete repository of all commercial software, however it's the only distribution that is continuing to improve at a rapid pace in this regard.  Some very good commercial applications such as ThinkFree Office, Pixel Image Editor, and StarOffice have all been released within the last month alone in CNR.  I have been told by CrossOver Office that they are planning to get it in CNR as well.  Cedega(Run's MS Windows games) is in CNR, and I've contacted Mainconcept about getting Mainactor into CNR.

As far as speed goes- I think Linspire 6.0 will have many of the small kinks worked out (nothing major exists in my opinion) and have even noticed improvements with CNR v6 in terms of speed.  For those who have not used Linspire for any length of time you probably don't know what kinks I'm referring to.  Examples:  CNR rearranges (by default anyway) icons on the desktop which annoys most users (can be turned off),  floppy disks don't auto mount like CD-ROM's do (also fixable with a slight modification to configuration files),  and so forth.

I Have been using Ubuntu by the way for some time now.  My experience with Xandros is limited to a few months with 2.0 business edition.  Linspire I've used for several months.  I have heavily used and relied upon each distribution and based my opinion on the number of changes I needed to make to each to get them running for different users, including myself.  I do not run MS Windows or it's applications, so anything I've done in that regard has been testing for other users who I have then received feedback from.  I am also the president of a Linux User Group which helps new users so I have a good idea of the difficulties users face in tranisitioning.

Chris

### Post by aysiu on 2005-12-11
[QUOTE=gabhla]
As for which could become a Vista killer.....neither.  Just don't see that happening in the near term.[/QUOTE] I agree. I don't even know what it means for a distro to *be* a "Vista killer."

No one I know will be lining up around the block to get a Vista CD or DVD when it comes out, but they'll all eventually end up with Vista. Why? Because after a while they'll get tired of their old XP and 2000 computers and want a new Dell or HP. What will be on that new Dell or HP? Windows Vista.

Most people don't choose what operating system ends up on their computer, and the reason people don't use Xandros, Linspire, Ubuntu, or whatever usually has *nothing* to do with how easy, compatible, stable, or whatever it is. It's because:

1. They've never heard of it.
2. They don't want to install something when Windows is already on their new computer
3. They don't want to use something they've never heard of when "everyone else" is using Windows.

Until Linux distros are preloaded on desktops by a major distributor, no Linux distro will ever "kill" or in any way seriously compete with Windows as a home desktop operating system.

That said, in order of ease of installation *and* use for a beginner, I'd rank the three distros in question: Linspire, Xandros, and Ubuntu. I don't agree with the person who revived this old thread about Synaptic (it doesn't get much easier than that), but I also would not recommend Ubuntu to someone who's never installed a Linux distro before--too much command line needed to set it up.

Once installed, though, I don't see how any one of those is any more difficult than the others. I use Ubuntu because it's free, free, and free, and the support is good, great, and excellent--not because it's easy. Though, having Synaptic does make at least software installation easy.
