### Post by akiro.yamamoto on 2006-03-27
The default setting for FF is to save downloaded files to your Desktop (in my case I created a Downloads folder in my Home Directory, see my pic), so check there for the file..... if it is then at a terminal.....
```

cd Desktop

``` Now you can follow the rest of the instructions from
[ ** HERE !!! **]("https://wiki.ubuntu.com/RestrictedFormats#head-e2ebd70ede0e3eb2117ffbd618d2295dd1540dca")
To complete the java install....

If you need to search for the file then use:
Places >> Search For Files and do a search of your Home folder (see pic)

Hope this info helps ;)
