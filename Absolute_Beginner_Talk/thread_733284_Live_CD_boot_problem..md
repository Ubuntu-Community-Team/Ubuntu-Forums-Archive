### Post by MattBD on 2008-03-23
If you just want to try Ubuntu for the moment, I find the easiest way to do so is to run it in [VirtualBox]("http://www.virtualbox.org/") on top of an existing Windows install. I do this on my Vista machine and it's a great way to try out a distro. It also means you don't get hardware problems, and you can run it off the ISO image. Only downsides are that sound doesn't generally work, nor do fancy graphics like the cube effect in Compiz-Fusion. That, and you take a performance hit because you're running the host OS, the virtualization app and the guest OS.
