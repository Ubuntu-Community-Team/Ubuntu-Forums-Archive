### Post by Cruz on 2006-10-01
Of course there is, way better too! Check out System -> Administration -> System Monitor. Try all the tabs and see what you can do with a rightclick on a process. 

If you can find your way into a terminal, try the top, ps and kill commands (read man pages to both).
