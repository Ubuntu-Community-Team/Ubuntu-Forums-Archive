The best application installers for your Distros programs themselves are the built in package managers.Add/Remove,Synaptic,Adept,Yast etc.All the major distros have some form of package manager and theres no running around the internet looking for software,although you can do that too, just never as safely apparently:).

### Post by AgentZ86 on 2008-01-12
> **Stu Ruckus said:**
> ... what would be the best application installer, my brother uses wine, but on a different linux, fedora I believe. Is there anything anyone can tell me?

Depending on your Linux distro or Linux OS you plan to use, I assume you want Ubuntu ?

But can say the easiest and best for a new user would have to be the installer that is included in the Ubuntu installation.

Simply click applications/add/remove programs and select your programs you want to install.

Also when on the net and installing a deb. package which is what you typically want for linux packages you just click the deb. package as you would with windows and the installer will do the rest.

I use synaptic when deb. installs don't work right and I can uninstall them better from there, because they may not appear at times in the add/remove section of the ubuntu package manager.

But anyhow I have used Ubuntu 7.10 Gutsy as well as on my wifes computer and have kept it up to date with all the current updates on a daily basis or when prompted and it's been working very smooth and problem free.

I can't complain about the ease of installation with the ubuntu installer.

I've installed anything I want even from the un-supported applications and without any problems with package installation.

Hope this helps.
