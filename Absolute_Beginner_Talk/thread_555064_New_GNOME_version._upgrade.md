### Post by qamelian on 2007-09-19
> **alecwh said:**
> I don't see where I can upgrade GNOME in the synaptic package manager....

Unfortunately, you won't find it in the repositories for Feisty. If you want the latest Gnome, you'll either need to compile it from the source code or else upgrade from Feisty to Gutsy when it is released. To maintain maximum stability, once new release is finalized all you'll get is security updates but no major version upgrades unless a package is back-ported.
