> **manfredell said:**
> How can I somehow revert the whole menu structure to defaults. Which file is the menu stored in??

Under Dapper, it's ~/.config/menus/applications.menu. I don't know for Edgy. Delete that file and you should have the default. You may need to log out and log back in for the change to take effect.
