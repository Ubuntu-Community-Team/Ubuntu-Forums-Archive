### Post by Kobalt on 2007-06-25
What I would recommend you first is to [make yourself a separate /home partition]("http://psychocats.net/ubuntu/separatehome"), so that you will keep your personnal data and settings available from your other distro. 
Then, the obvious for you would be to try out Debian unstable : Ubuntu is built from debian so you'll keep the same package management system, but you'll have much more 'manual' things you can do for fun if you'd like...
