### Post by CatKiller on 2006-11-22
As far as I know (which isn't far :) ) the programs get **expanded** into RAM when they're loaded. So they take up much more memory in RAM than they do on the cd.  So it might theoretically be possible if you had, like, 6 GiBs of RAM. And if they'd thought that someone might **have** 6 GiBs of RAM and put in the possibility of unmounting the cd.

I believe that it might be possible to run the live environment from a USB stick, if that helps.
