### Post by urukrama on 2007-04-20
The right-click menu of Xubuntu (Xfce) is a feature, and you'll find it very useful once you get used to it. If it stopped working, make sure you check your options in your settings menus. Does Xfce still manage the desktop? etc.

Xubuntu, which uses the Xfce Desktop environment, is great, but don't expect a copy of Ubuntu, which uses Gnome. I don't know what other 'issues' you have had, but it is quite possible that these are actually not bugs or flaws, but just the way Xubuntu works. Give it some time, and explore its functionality and customisability. When I first used Xfce, I totally disliked it (the default settings of Xubuntu don't really help), but now love it and prefer it over Ubuntu/Gnome. Right click on everything you can, and you'll see this is a great desktop environment. If you want speed, functionality, customisability, and a good look, Xubuntu is for you.
