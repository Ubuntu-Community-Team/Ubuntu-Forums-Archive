### Post by nestor5 on 2007-11-22
I recently got Broadband at home, and I decided to  finally exercise my option to obtain a completely Microsoft free, modern user friendly operating system, that imposes none of the restrictions of Windows.

 I thought Ubuntu would be that system, so I downloaded a copy of the iso (Ubuntu 7.10 desktop i386), and made an Ubuntu boot CD. I had intended to install it on a spare 8GB hard drive that I have lying around, and I had no intention of booting it as a live CD (I read that one requires nearly 400 MB ram for Ubuntu to boot live CD mode, and I only have 256MB of which 64MB is given to  the internal graphics card. my motherboard is an MSI k8MMV). I therefore assumed that one would get the option of being able to directly install the operating system from the disc without booting it up; however, no matter what I did, Ubuntu insisted on trying to boot into live CD mode, without even checking first to determine whether the system could handle that type of install.
 The CD Rom would keep blinking interminably, and after some twentyfive minutes, after the brown screen and a few icons had installed, the system would freeze up, with a blank white box with a tab labelled error would appear in the middle of the screen. I personally have never believed in running an operating system as a live install off a CD rom (a CD rom drive just is not fast enough, and it places unreasonable demands on ram), but I at least thought that I could do a straight install on the hard drive, and boot from there; however, this simple commonsense option is apparently not available! Do the promoters of Ubuntu expect people to go out and purchase extra ram, just for the purpose of viewing their operating system? I think not!
  Any how, my first encounter with this OS has left me bitterly dissappointed. Does any one out there have any suggestions, perhaps I missed something obvious that would have enabled me to avoid the problems I am having?

### Post by steve.horsley on 2007-11-22
As dpar says, use the "alternate" installer CD. This doesn't launch a full GUI and desktop with a graphical installer app on top of that. Instead, it's a text-mode installer (although the questions are the same). It has much less memory requirements. 

Good move using a spare 8G hard drive. That will give you a good intro and let you know how you get on with the OS before you mess with your windows drive.
