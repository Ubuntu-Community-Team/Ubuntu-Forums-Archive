### Post by southernman on 2007-08-09
Welcome to the forum kidstar64... and Ubuntu. 

With a little reading, you can use a default install option of "cdrecord" to burn any kind of cd on the command line. There are gui apps to choose from also. And yes, you can buy a copy of Nero for linux... think it's about 20 bucks or so.

As for installing tarballs, if you need to use a tarball then there is usually a howto written for that specific app on the site you d/l from. If all else fails drop by the forums, someone will be glad to help out.

Yes, Azureus supports UPNP if you need it. It's what I use, but choose to disable that option as I do my port forwarding in my router and use a static IP for my boxen.

You can grab the altermate cd [here]("http://www.ubuntu.com/getubuntu/download"). Careful though, if you do not tick the box below the "Start Download" button, you'll get the Live CD. It's kind of hidden and it seems done so by design, not just an after thought.

Anyway.... Hope that helps with everything but the video problem. It should easily be workable though.

Enjoy your Ubuntu experience... it's a pleasant ride for sure!
