### Post by dustynus on 2008-03-27
This thunderbird extension:

[http://moztraybiff.mozdev.org/releases.html](http://moztraybiff.mozdev.org/releases.html)


Download the xpi and then install with thunderbird, go to it's preferences and put ''show always'' then it'll put a small thunderbird icon in your gnome tray and you can click and hide show, just like amarok.

### Post by forrestcupp on 2008-03-27
Also, if anyone wants to know how to use alltray correctly, you have to enter the program name with it, like
```
alltray thunderbird
```
To run Thunderbird within alltray.  If you don't do that, it will wonder what you want to run it with.
