### Post by mikewhatever on 2008-01-19
Software versions are never upgraded in one Ubuntu release. All you get is security updates. Once Hardy Heron is released in April, the latest VLC version would be there. If you really want it, there is an option to build it from source. Have a look [http://www.jbkempf.com/blog/post/2007/05/06/Build-VLC-media-player-under-Ubuntu-Feisty-704](http://www.jbkempf.com/blog/post/2007/05/06/Build-VLC-media-player-under-Ubuntu-Feisty-704)
