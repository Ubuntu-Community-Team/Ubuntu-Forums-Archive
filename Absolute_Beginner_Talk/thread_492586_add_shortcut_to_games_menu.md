I have downloaded a game, moved the file to my /home/apps directory, now i want to add a shortcut to my Applications>>Games  menu.  Is this possible and if so, how?

---

### Post by Vajra Vrtti on 2007-07-04
Open *System -> Preferences -> Main Menu*, select the *Games* folder and click the *New Item* button.
