### Post by jd65pl on 2006-09-27
There is a firewall installed by default called "iptables" Firestarter is a graphical front end for iptables

**To get firestarter goto**

APPS > ADD/REMOVE

search for firestarter follow instructions on screen for getting the application. 

Read [here]("https://help.ubuntu.com/community/InstallingSoftware") for more info on installing
