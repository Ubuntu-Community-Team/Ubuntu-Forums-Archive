### Post by mikewhatever on 2007-02-21
When resizing, you should be able to choose which partition to resize, so the slider bar will apply to it. 50 GB is more then enough for Ubuntu and if there are space limits, I think it is quite safe to reduce it to 15-20 GB. My installation (default) is less then 3 GB, for example.
