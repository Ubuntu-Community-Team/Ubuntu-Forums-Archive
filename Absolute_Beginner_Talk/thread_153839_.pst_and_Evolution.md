### Post by yager on 2006-04-01
[QUOTE=micrologix]Is there a way to get me MS Outlook data (.pst) file across to Evolution??[/QUOTE]

See this thread for some help.
[http://www.ubuntuforums.org/showthread.php?t=73978&highlight=pst+evolution](http://www.ubuntuforums.org/showthread.php?t=73978&highlight=pst+evolution)

It may get you there.
I have been researching this for the past week and in every case, you have to use an intermediate program since Evolution can not bring the information in directly.  Novell, who maintains Evolution can not access Outlook directly because it uses MAPI.dll which is a proprietary file on Windows.  The link below offers their official answer.
