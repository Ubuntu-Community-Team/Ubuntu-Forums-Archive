### Post by aysiu on 2006-06-09
As far as the software, not a whole lot makes Ubuntu "so great." 

It doesn't have better hardware detection than other distros (I think they're all about the same--they just detect different things, and when one detects someone's hardware, she immediately thinks, "This one has *better* hardware detection").

They all pretty much have the same programs and desktop environments available.

Every Debian-based distribution uses the same packaging system.

Ubuntu is actually missing graphical user interfaces for a few things other distros have GUIs for (configuring X, reinstalling Grub, configuring Grub, etc.).

Ubuntu is missing a lot of proprietary codecs that new users take for granted--MP3, Flash, etc.

Why do I use it? What makes it "so great"? Its users. That's it. The support on these forums is excellent--anything from basic setup to serious tweaking. People here have the answers to almost everything, and these people are all volunteers. And, better yet, they're civil.

Even the veteran users treat new users with respect. There's no "read the f'in manual" remarks.
