### Post by az on 2005-08-15
[QUOTE=red_Marvin]
Now what I need to know is what I need to get/do in order to make the computer
safe for roaming around the net with.
[/QUOTE]


Ubuntu is already safe for roaming the net and such.  If you want to run a webserver or an ftp server, you should consider more protection (like a firewall), but otherwise, you are safe.  You do not need to run a virus program either, since  it is really really hard to make a virus for linux.

Of course there are other kinds of vulnerabilities out there, but so long as you stick with the packages software, you should be fine.

By packaged software, I mean anything you can install using synaptic.  And yes, synaptic is a GUI version of apt-get.
