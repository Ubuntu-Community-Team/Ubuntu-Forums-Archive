### Post by MrHorus on 2006-07-09
Free software is free as in free beer - open source software is free as in free speech.

With free software you get exactly that - software for which you do not have to pay and can do what you want with it.

With open source software you are free to look at the source but you may not be free to modify it and use it as you wish - this depends on the particular liscence.

This is the meaning of the acronym 'FOSS' or Free (and) Open Source Software i.e. software that is free as in free beer, yet the source code is open for you to modify and adapt as you see fit.

### Post by Jucato on 2006-07-09
The difference between free software and open source software is, like what userundefine said, more philosphical or idealogical. There's not real distinction between the two, when it comes to categorizing software. As long as a software is using GPL (or LGPL), it's considered FOSS or Free/Open Sourse Software. So there are really no categorizations like this software is Free Software but not Open Source Software, or this software is Open Source software but not Free software.

But by definition, any free software is open source, by the fact that GPL mandates that source code be "open" or available. The Open source software, is more broad, in that AFAIK, it also accepts other GPL-compatible licenses like the BSD license. I might have some things mixed up, though. :D

### Post by crystal on 2006-07-09
> So there are really no categorizations like this software is Free Software but not Open Source Software, or this software is Open Source software but not Free software.
Compare the list of [Open Source Initiative certified licenses](http://opensource.org/licenses/) and Free Software Foundation's list of [Various Licenses and Comments about Them](http://www.fsf.org/licensing/licenses/index_html). At a glance, the [Reciprocal Public License](http://opensource.org/licenses/rpl.php) is OSI certified (and hence definitely an Open Source Definition compliant license) but classified by the FSF as non-Free.
