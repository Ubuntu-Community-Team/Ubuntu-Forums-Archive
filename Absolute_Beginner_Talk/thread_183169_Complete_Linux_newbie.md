### Post by JSchwage on 2006-05-27
Synaptic Package Manager is the answer you're looking for. I'm not sure about 5.10 Breezy, but in Dapper Drake, Synaptic can be accessed by going to "System > Administration > Synaptic Package Manager". From there you can check off any new programs you'd like to install. And if you're looking for community-maintained programs, you'll need to enable those repositories from Synaptic. I can't give you a detailed instruction on the repositories since I'm actually on a Windows machine at the moment.
