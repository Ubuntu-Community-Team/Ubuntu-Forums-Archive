### Post by nalmeth on 2006-07-13
You may like apollon

By default it only comes with OpenFT and Gnutella, but you can add Fasttrack by downloading the libgift-fasttrack plugin.

sudo apt-get install gift apollon

I think the fasttrack plugin has been removed from the ubuntu repositories, so google for it. There should be a debian package somewhere. 

It's sweet. I usually get about 3 million users on that network (I think its kazaa BTW).

BTW, when you open apollon for the first time, it will ask were the gift plugins are. They are in /usr/share/gift

Maybe that will be an improvement for you.
