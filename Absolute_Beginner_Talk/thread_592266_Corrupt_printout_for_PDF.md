### Post by genbuntu on 2007-10-26
Some possible options :

1) check out a sample pdf file (maybe from internet)

2) try printing from firefox etc

3) Is it only pdf files? try printing a test page from printer options . if it does not print out well then there is some driver mismatch problem

4) try using a different pdf software to print

For all of the above you should print only if print preview is fine; else no need to actually print; just check in print preview
