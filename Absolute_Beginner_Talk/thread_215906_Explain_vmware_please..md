### Post by T700 on 2006-07-14
> **jincast90 said:**
> what exeacly is this? Is it a program, so you can try an os, without installing it?

It allows you to run another OS "on top" of your current one.  Sort of like a virtual world.  People use it to run Windows apps in Linux, multiple servers off one physical box, and to test new OS's and apps.  It requires a pretty powerful physical server to run it well.
