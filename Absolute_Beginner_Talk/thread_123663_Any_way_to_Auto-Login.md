### Post by Heretic Monkey on 2006-01-30
Yeah, i have mine set up so it automatically goes to my user account.  I read "ubuntuguide.org" and got the info from there:

Go to: **System -> Administration -> Login Screen Setup**
Then:
> General Tab -> Automatic Login ->
Login a user automatically on first bootup (Checked)
Automatic login username: Select "system_username"
