### Post by BrendanM on 2007-09-07
I use a firefox extension called [MediaPlayerConnectivity]("https://addons.mozilla.org/en-US/firefox/addon/446") which allows you to extract media links embedded in web pages and play them in any player you like. I play them with VLC because VLC can handle almost any kind of stream. This solution works pretty well for me, maybe it will for you too.
