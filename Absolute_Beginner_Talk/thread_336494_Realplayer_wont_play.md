### Post by arochester on 2007-01-11
The BBC has a help page "Advice for Unix/Linux users" at [http://www.bbc.co.uk/radio/audiohelp_nix.shtml](http://www.bbc.co.uk/radio/audiohelp_nix.shtml)

It mentions audio but it work for video as well. The key points are:
# Many users have found that they were able to use the BBC Radio Player after installing Real Player 10.
# The files nphelix.so and nphelix.xpt must be in the browser's plugins directory for Opera, Firefox and Mozilla
# You may need to log out and back in again before the changes work.

You need to Go to the Realplayer site at [http://www.real.com/linux/](http://www.real.com/linux/) and download Realplayer10 for linux. The installation instructions are there. Make sure the two files are in the browser's plugins directory.

It works for me
