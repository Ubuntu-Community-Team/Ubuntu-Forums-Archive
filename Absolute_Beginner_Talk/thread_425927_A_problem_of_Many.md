### Post by tgalati4 on 2007-04-28
Did you try booting the "Live CD" only?  You normally don't want to install blind without booting safely into the Live CD.  If the Live CD hangs then you know that you may have some issues to deal with before doing a full install.

If the Live CD fails to boot, then try passing a kernel option as listed in one of the function keys.  Try the "normal" video mode.
