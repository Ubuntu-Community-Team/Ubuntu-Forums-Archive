I downloaded and installed the windows version of pidgin on my PC at work and it lets me connect to Sametime, which we use at work.  That version is Pidgin 2.2.1

I was using 2.0.2 on my Ubuntu box at home and it didn't show Sametime as an option for account types, so I upgraded it to 2.20, but it still doesn't have the Sametime option.

Is there a way to get Sametime to connect with Pidgin on Ubuntu?

### Post by Anolis on 2007-10-15
In my version of Ubuntu 7.10, running Pidgin 2.2.1 i go to Accounts>Add/Edit;click add, under protocol i click the drop down menu, select sametime, fill out information, connect, done.
