### Post by wataboutbob on 2007-03-29
The Core 2 Duo (Merom) is a 64 bit dual core chip. So yes, you can install the x64 version of ubuntu in it. However, I think there aren't that many applications under the x64 environment - the same case as under Windows also.

In your case, even though you've installed the x86 version, after system updates, both cores are enabled. My advise is to stick to x86 (the 32 bit) as most support from this forum will be with that OS.
