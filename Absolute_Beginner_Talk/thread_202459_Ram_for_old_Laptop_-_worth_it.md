### Post by emperor on 2006-06-23
[quote=WADemosthenes]I have a Pentium II 366 mhz with 128 mb of ram, would upgrading to 256 mb of ram really make that much of a difference?  Would proformance still be held back by the proccessing speed?[/quote]

Yes, adding more ram will help. Xubuntu will run faster on your older laptop too. Install Boot Up Manager and turn off un-needed services like Bluetooth, ppp, ... Ubuntu and it's relatives default with a lot of needless stuff running.

Other Distro's are also an option. On a PIII 500 Mhz I have FC5 with Gnome running and using only 75-85MB or RAM with 256MB installed. However, I turned off a lot of services including cups and portmap as I do not need all this for a MythTV frontend.
