### Post by dfreer on 2007-07-16
Can you tell us what program you are trying to run, maybe someone has gotten it to work.

Namely, the first thing to realize is that wine is a work in progress (technically it's still in the 0.9 phases, not even a 1.0 release yet). So it just may not work, and it isn't ubuntu/linux/wine's fault.
That being said, a lot of windows programs can run in wine. The best source of information to see whether a particular program will work would be to check the appdb at winehq.org, namely here:
[http://appdb.winehq.org/](http://appdb.winehq.org/)
If a program needs a little hack or DLL override in order to work correctly, often someone will post an howto on the appdb as well. 

Other options to consider are Cedega (games only really) and Crossover Office (may provide better support), although both of those cost $$$.
