### Post by Tomosaur on 2006-08-26
Well, I would use the LiveCD to check your wireless stuff works before you install Ubuntu. If you're just using the machine to surf the net and fool around, then by all means just scrap Windows - you would probably have no need for it.

With the Dapper Drake install CD, you don't get asked whether you want to install GRUB (maybe on the alternate CD, not sure), but I recommend you keep it on anyway and just mess with the settings if it bothers you so much. It can be handy to switch to a recovery mode if anything goes wrong.
