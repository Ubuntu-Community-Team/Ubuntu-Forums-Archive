Short answer, it does not work that way. 

Go ahead and load the iso with daemon tools, that is just like putting the CD into a CD bay it will not harm your system ...

You will be able to read data off the CD (iso)  and interesting I think you can run firefox ...

But loading an OS is something different. You can only run 1 kernel at a time ... The kernel runs the CPU, RAM, HD, etc and a very low level.

Now you can run an OS within an OS with VMWare, qemu, virtual box, etc but that is different from daemon tools. Daemon tools just emulates a CD ROM drive ...
