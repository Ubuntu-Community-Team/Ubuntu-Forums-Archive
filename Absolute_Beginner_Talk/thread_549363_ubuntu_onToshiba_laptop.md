Toshes can be a bit nasty as they are firmly 'Windows' machines. Do you get some icons under the Tosh logo as soon as the machine is powered on? If so, these represent the boot devices and you can use the cursor keys to highlight the CDROM and hit return to boot from CD.

### Post by w4ett on 2007-09-12
To enter the toshiba bios:

Toshiba Notebook [Newer models]
	

1. Turn on computer by Holding down power button while pressing the ESC key.
The machine will beep, then display:
Check System, then press [F1] key.
2. Release ESC key
3. Press F1 key

Hope this helps.
