You can actually go and get  UbuntuStudio through Synaptic...

If you choose to download the whole thing, be advised that it does come with the RealTime Kernel, and that can cause some compatibility issues, but can be remedied through changing the boot order in Grub ;)

When I need to use the studio portion (Audio tools, video editing), I just boot into the RealTime Kernel.
