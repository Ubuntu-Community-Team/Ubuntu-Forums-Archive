### Post by jken146 on 2008-03-07
Are you talking about msn messenger settings for Pidgin?  If you are, choose MSN as the protocol for a new account and type in your screen name (hotmail address) and password.  Everything in the Advanced tab should already be filled in, but if it isn't, the Server should be messenger.hotmail.com -- hope this helps.
