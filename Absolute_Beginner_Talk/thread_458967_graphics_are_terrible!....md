### Post by EndPerform on 2007-05-31
When your X server doesn't start, what does the log file (/var/log/Xorg.0.log) have in it?  It should tell you what error you're getting.  If you can tell us what error Xorg is giving you when you try the nvidia drivers, we might have a better idea as to what's happening.

To see what's in the log file after the Xserver doesn't start, from the command line type:

```
nano -w /var/log/Xorg.0.log
```

Then scroll to the bottom of the file and it should tell you what the problem is.
