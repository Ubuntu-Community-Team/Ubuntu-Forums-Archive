The following is from the Hardy FAQ:

QUESTION. *If I continue updating my alpha or beta installation, will I get the final version of Hardy when it comes out, or do I need to reinstall it?*

ANSWER. [I]You don't have to reinstall it. You'll be able to upgrade to the final version with the update manager once it's ready.

It can, however, be a good idea to do a clean install if you're affected by certain corner cases where you can't access specific features or fixes without doing a clean install, or if you've modified or partially broken your installation with software not provided by the official Ubuntu repositories.[/I]
