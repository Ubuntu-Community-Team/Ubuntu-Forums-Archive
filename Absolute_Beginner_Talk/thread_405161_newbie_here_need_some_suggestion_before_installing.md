### Post by daller on 2007-04-09
If you really wanna use both Windows and Ubuntu, you should install Windows first, then Ubuntu. (Installing Windows after Ubuntu will make the ubuntu-partition unbootable)

Since you are going to use you Ubuntu partition for most of your work, I would split it 10/70 if I were you!

You can run WoW from ubuntu quite easily!
[https://help.ubuntu.com/community/WorldofWarcraft](https://help.ubuntu.com/community/WorldofWarcraft)
