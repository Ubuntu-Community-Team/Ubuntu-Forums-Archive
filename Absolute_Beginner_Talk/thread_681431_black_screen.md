### Post by Amstell on 2008-01-29
you need to get into your xorg.conf

/etc/X11/xorg.conf

type this command

cp /etc/X11/xorg.conf.backup /etc/X11/xorg.conf

that would switch everything back.  Hopefully it backed up.
