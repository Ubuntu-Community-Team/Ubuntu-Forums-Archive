### Post by victorbrca on 2007-03-18
Hey there,


1- Opera as default browser
- System => Preferences => Preferred Applications

2- Import Bookmarks
- Firefox => Bookmarks => Organize Bookmarks => Import

3- Close button on tabs
- Firefox 2 has this feature
- [Psychocats]("http://www.psychocats.net/ubuntu/firefox")

### Post by mrmonday on 2007-03-18
Go to system>Preferences>Preferred applications. You can change your default browser there. 

If you want to do your second option, Firefox 2 has buttons on each tab and to import bookmarks go to (in firefox) Bookmarks> Organise bookmarks> File> Import then choose opera
