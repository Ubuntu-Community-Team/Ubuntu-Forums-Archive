### Post by Zimmer on 2005-12-21
AIM ? do you mean the Internet Messaging application?
If so you need only click on 
Applications>Internet>GAIM Internet Messenger
The application you need for ICQ, etc. is already installed.

As for the command line...
If you have loaded Ubuntu 5.10, Breezy, then you can open a terminal by going to
Applications>Accessories>Terminal

**Before you do**, I would suggest some further reading....
first how to install packages (software) using Synaptic

[http://help.ubuntu.com/starterguide/C/faqguide-all.html#synaptic-whatis](http://help.ubuntu.com/starterguide/C/faqguide-all.html#synaptic-whatis)
and then 
[https://wiki.ubuntu.com/BasicCommands](https://wiki.ubuntu.com/BasicCommands)

If you are lucky you will get by without **having** to open the terminal..:)
for a while at least....
