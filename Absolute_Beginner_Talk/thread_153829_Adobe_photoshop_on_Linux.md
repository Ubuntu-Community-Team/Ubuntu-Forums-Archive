### Post by rcmiv on 2006-04-01
[QUOTE=Darkness3477]Hi, I was wondering If it's possible to get Photoshop to run on my ubuntu 5.10 using WINE or something similar? 
If it's possible, would it just be easier to boot up my laptop to use photoshop?[/QUOTE]

Assuming you have a copy of the photoshop disk and a windows install disk, I would use vmware player.  It is fairly easy to install windows into it, and it's free, though, of course, windows is not.

I use a vm'd windows install for office apps for my work, and it works fine and performs speedily enough to be usable, even on a 500MHz machine with 384meg of ram. Plus it's super geeky and fun to show off to your friends when you boot Win2K in a window on your ubuntu box.

Here's some linkage for you:

vmware player:
[http://www.vmware.com/products/player/](http://www.vmware.com/products/player/)

how to install stuff into the player:
[http://www.virtualization.info/2005/12/guide-to-create-vmx-files-for-vmware.html](http://www.virtualization.info/2005/12/guide-to-create-vmx-files-for-vmware.html)
