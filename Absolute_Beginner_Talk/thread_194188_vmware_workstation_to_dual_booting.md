### Post by rubinstein on 2006-06-11
You can boot your computer with your Ubuntu CD, it will start a live system where you can test if everything is OK; if you want to install Ubuntu on your harddisk to dual boot start the installer in the Ubuntu live session.

The Installer has a partition program where you can shrink your Windows Partition. (But make a backup of your data, everything should be OK, but you never know :-) )
