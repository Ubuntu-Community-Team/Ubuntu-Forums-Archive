### Post by glennric on 2008-03-19
The cube doesn't matter.  Open System->Preferences->Advanced Desktop Effects Settings (if that is not in your menu install compizconfig-settings-manager) and select General Options.
Go to the Commands tab and add the command to start konsole to one of the command lines under commmands.  Then set the same numbered run command under key bindings to the key combination you want.
