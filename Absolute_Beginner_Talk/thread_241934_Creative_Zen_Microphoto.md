### Post by ntetreau on 2006-09-04
Hi all, there are a lot of threads in the forum on using libmtp ([http://sourceforge.net/project/showfiles.php?group_id=158745](http://sourceforge.net/project/showfiles.php?group_id=158745)) and gnomad2 ([http://gnomad2.sourceforge.net/](http://gnomad2.sourceforge.net/)) to read/write to your creative mtp device.  Also, the latest Amarok 1.4.2 has basic transfer functionality built in although I am not sure if you need to compile from source to use it (for sure you need to install libmtp first!).  If after installing libmtp and the amarok 1.4.2 you still cannot connect to your device, then you will need to install amarok from source.  I followed this thread to install the latest amarok ([http://ubuntuforums.org/showthread.php?t=80492&highlight=amarok+svn)](http://ubuntuforums.org/showthread.php?t=80492&highlight=amarok+svn)).  My Creative Zen micro-photo works flawlessly.
