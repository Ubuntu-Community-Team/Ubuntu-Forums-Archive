### Post by Seti on 2005-06-14
Since you're new to linux, I would suggest going with the regular Hoary installation for x86, not the AMD64 version. From what I gather there isn't a lot of support yet for 64 bit software. 
Don't be afraid of the command-line, if you're serious about getting into using linux for your computing needs, you'll soon learn to appreciate the power of the terminal.
Connectivity shouldn't be any problem for you if you're on DSL.
As far as games are concerned, I think Cedega is more what you want. I'm not a gamer myself though, so I can't really tell you how well it works or how hard it is to set up.
