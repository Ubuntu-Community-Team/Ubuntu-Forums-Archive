### Post by PmDematagoda on 2007-11-20
Ubuntu Ultimate has a lot of software pre-loaded, because of this the chances of certain packages not working after you upgrade it is very much higher than what you would get with average Ubuntu installations. So if you wish to upgrade the OS properly I believe it would be better if you waited until TheeMahn2003 released the Ubuntu Ultimate version which is built on Ubuntu 7.10, as I have not tried this myself I cannot assure you that the upgrade process will go seamlessly as even upgrading an average Ubuntu installation using the official methods can also cause problems.
