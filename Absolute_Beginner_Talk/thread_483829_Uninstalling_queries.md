Hi you can remove them via the terminal like *sudo apt-get remover beryl* or you can use synaptic manager. Just open synaptic manager and search for enlightenment and then mark for removal. As for the kernel upgrade I would advise to leave in place for the reason if you have more updates and something goes wrong you can possibly boot into that kernel and save you data.
