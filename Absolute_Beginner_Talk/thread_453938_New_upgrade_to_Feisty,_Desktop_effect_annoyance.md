### Post by REDISISTone.nl on 2007-05-25
Yes it's a common problem, wat I did is right click on the "desktop switcher" (srry for the bad enlish) and open up the desktop effects settings diolog. Enable the 3d cube and then in your desktop switcher make 4 screens. Hope this works!!

And you might want to take a look at the extra desktop effects settings. You dont have to install beryl for these effects (and might help solve your problems)

```
sudo apt-get install gnome-compiz-manager
```

More options for your desktop effects:
System->Preferences->GL Desktop

Good luck!!
