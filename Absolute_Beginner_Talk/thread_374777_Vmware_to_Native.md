There is a trick to install Mac OS X (patched) into a PC via VMware, in case you don't have cd drive or you can't burn. You simply need to create a virtual machine, and use a real partition instead of a .vmdk file. Then you install Mac OS in the virtual machine, but on a real partition. Thus when you reboot, if everything went fine and you set up the bootloader, etc, you will boot into OS X, if your hardware is compatible (which is the reason why it works for some people, but not for others)
