### Post by bruce89 on 2006-07-14
Do you mean Gnome/KDE?  If so, this will help - [http://psychocats.net/ubuntu/index.php](http://psychocats.net/ubuntu/index.php) (see the links near the bottom), it boils down to:
```
sudo aptitude update && sudo aptitude install ubuntu-desktop
``` for Gnome,
```
sudo aptitude update && sudo aptitude install kubuntu-desktop
``` for KDE and
```
sudo aptitude update && sudo aptitude install xubuntu-desktop
``` for Xfce.
