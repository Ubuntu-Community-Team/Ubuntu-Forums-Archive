> **LJFIII said:**
> I want to gain general/programming experience.  My future profession one day will be Intellectual Property law and IT/Executive management (CIO/CEO kind of thing). If you're hoping to become an "IP" lawyer, and you're also very interested in IT, I strongly recommend you study the Groklaw website: [www.groklaw.net](www.groklaw.net)

As for learning more about Ubuntu and GNU/Linux, I can suggest a few topics to maybe get you started:

Find a software package that you like (which may or may not be in the Ubuntu repositories), and try to figure out how to download, compile and install it from source code.

Learn how to use the command-line.  Think of a task (maybe trivial) you'd normally do from the Graphical User Interface, and see if you can work out how to do it from the command-line.

Do you have a USB pen drive?  Try to make a "Live USB", which is like a Live CD, only it runs from the pen drive instead of from the CD.  There is a guide here: [http://www.debuntu.org/how-to-install-ubuntu-linux-on-usb-bar](http://www.debuntu.org/how-to-install-ubuntu-linux-on-usb-bar)
This is great for demonstrating Ubuntu on your friends' PCs.  The pen-drive is much more portable than a CD (it fits easily in your pocket), and the operating system loads much faster off a pen-drive than it does off a Live CD (in my experience).  I got Ubuntu running on a Dell PC with the fancy 3D "wobbly windows and cube" effects.  My friends who witnessed this were quite surprised to learn that it was all freely-available.

Read up on the history of GNU/Linux, and learn about the difference between "GNU" and "Linux".  You will encounter some issues about copyrights and patents in software that you might find interesting.

Lambros
