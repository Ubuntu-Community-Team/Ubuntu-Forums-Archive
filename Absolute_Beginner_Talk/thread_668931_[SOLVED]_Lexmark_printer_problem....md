### Post by Sef on 2008-01-15
In general, Lexmark inkjets, don't work on GNU/Linux.

If you want to get a new printer, read [Vendors Linux Support]("http://www.linux-foundation.org/en/OpenPrinting/Database/LinuxSupportByPrinterVendors") and [Suggested Printers]("http://www.linux-foundation.org/en/OpenPrinting/Database/SuggestedPrinters").
