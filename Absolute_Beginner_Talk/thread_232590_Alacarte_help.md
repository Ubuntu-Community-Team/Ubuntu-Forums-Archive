### Post by catlett on 2006-08-08
You don't delete entries in Alacarte, you uncheck them.
On the right side select a category. Then on the left side a list will appear with the entries for that category. Any entries with a checkmark are visible in the menu. If you uncheck the entry it disappears from the menu.
If you uncheck all the entries for a category it should disappear as well.
