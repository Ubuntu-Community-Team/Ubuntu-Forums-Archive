Hi, I'm wondering if there is a way so that whenever I extract files using a   archiver, I can directly extract the files into a directory other than my username home. It always says I do not have the permissions to do so. So is there a way I can do sudo and then copy and paste stuff via GUI?  Thanks

---

### Post by catlett on 2006-09-04
```
gksudo nautilus
```Will launch the file browser as root. Then you can drag and drop the files to any folder you want.
