### Post by kelnage on 2007-09-18
[COLOR=Black]Go to http[/COLOR][COLOR=Black][COLOR=Red]s[/COLOR]://mail.google.com/ - that will ensure all your communication between Gmail and your computer is encrypted - including the cookie (very important!). However, you must make sure you always use the https address whilst in the unsecured network.

For other sites that don't offer a secure server: all you can do is not give them any details which might be used to hack into an account - so don't login unless you see the lock symbol!

### Post by p_quarles on 2007-09-18
I agree with the above. The best option is to use webmail that goes over an SSL connection. 

If you want to get a bit more involved, and have a computer at home that you can leave on, you can also set up a VPN and log into that whenever you connect from an insecure wireless access point.
