Not an odd request at all.

Check out [http://www.edubuntu.org/]("http://www.edubuntu.org/").

Dansguardian is what you will probably end up using to control access. You can blacklist words, phrases, sites, parts of sites, etc. Alternatively, you can whitelist sites so that he can only go where you want him to go. It's one of my favorite applications!
