### Post by LaRoza on 2007-05-14
Linux views everything as a directory, so there is no "c" drive, but /dev/sda where / is root, dev is a device and sda is the actual drive (a SATA master drive in this case)

This site give a good explanation: [http://www.pendrivelinux.com/2007/03/05/understanding-the-linux-file-system/](http://www.pendrivelinux.com/2007/03/05/understanding-the-linux-file-system/)

[http://www.reallylinux.com/docs/basic.shtml](http://www.reallylinux.com/docs/basic.shtml) has basic commands, for any command type "man commandName" where commandName is the command you want to know more about.

Doing a search for "linux commands" will give you many good sites.
