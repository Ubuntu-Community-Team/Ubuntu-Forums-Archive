### Post by reacocard on 2006-09-07
For security programs, linux doesn't really need them since its much more secure than windows is, but are a few firewall programs available if you want them. firestarter and guarddog are both good options.

Exaile is a really nice music player. It's not in the repos, but its available as a .deb package which is just as easy to install (download, double-click). [http://www.exaile.org](http://www.exaile.org)

Liferea is an excellent RSS/Atom feed reader.
