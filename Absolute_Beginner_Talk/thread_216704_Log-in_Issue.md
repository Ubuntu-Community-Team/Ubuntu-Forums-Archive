### Post by catlett on 2006-07-15
If you ran a regular install the section for a user name runs like this 
```
enter new users full name 
```
John Doe
```
enter user name. suggested name for user...
John
```
Then you would enter what you want. I only mention this because if you just pressed enter for some reason it would have used the default name. In this case John. So you could try using the first name of the full name you entered just in case you hit enter and the computer used the default name it offered,
