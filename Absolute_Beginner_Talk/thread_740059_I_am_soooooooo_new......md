### Post by freebeer on 2008-03-30
There isn't a GUI with server edition as default.  But you can install one.

At the command line type:

```

sudo apt-get install ubuntu-desktop

```

### Post by freebeer on 2008-03-30
True, but in my mind someone so new really wants it as simple as possible.  They can optimize later.  No disrespect intended to the OP, but he installed the Server Edition without realizing that it didn't come with a GUI.  My reply will get him there without any fuss.


[side topic]
I see that soooo often... they want to "serve" something, so in thier mind they must need to install the Server Edition (blame Microsoft for that), but what they really wanted was a GUI-enabled OS that can provide server-like services.  I can relate as that was the situation I was in when I first tried Linux.  But I wanted the training wheels that a GUI gave me and was lucky enough to pick up on the difference when I did my initial research.  So I installed the desktop version and added the server services that I wanted.  The learning curve was a lot easier because I had the comfort of a GUI to help me orient myself around the file system.

I know now that either method is perfectly fine (thanks to the freedom that is Ubuntu/Linux), but the non-GUI method would have been a lot more intimidating (and I **like** the command line and was looking forward to empowering myself with it after far too many years in the Windows Wasteland.) :D
[/side topic]
