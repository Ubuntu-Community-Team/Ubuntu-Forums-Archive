### Post by Regord on 2006-09-19
So I've been playing around with Linux distros for many years now but I've never really had much success.  I mean I'd get them up and running but I don't like command line stuff and I'd very frequently run into problems that I was unable to solve.  Consequently I'd get really frustrated and just give up.

So, I've been hearing soo much about Ubuntu that I decided to try it.  It's been installed for around a week or so and I'm simply blown away.  After I learned about Synaptic and the Ubuntu website, I've been abled to get almost everything I want up and running.  I say almost because I spent A LOT of time d/l gnome desktops and themes and just playing around.

I've run into 2 problems that I thought would require a reinstall but because of Ubuntu's easy to use design and you guys/gals, I've been able to fix them w/o having to reinstall. 

I'm a SR CS student and I need Java 1.5 and C++ compilers as well as some other programs.  heh...no sweat just took a little time getting them set up but no biggie.

I can't believe how much fun I'm having.  I have Ubuntu installed on an older machine (AMD 1.4Ghz) and I don't even care.  It's just sooo fun to play with - it makes my old machine seem like a brand new computer.

I've now decided that everytime I think of going to do something on my Windows box, I'm going to see if I can accomplish the same thing on Ubuntu.  So far so good.

Anways, just wanted to share my excitement and joy.  I mean I've never had a Linux experience like this before and I'm quite frankly blown away by Ubuntu!  I'm telling all my friends about it and they are really excited also....looks like there's a whole new crop of Ubuntu users on the way. :)
