### Post by willskills on 2007-01-03
I would have a good read of [www.gnome-look.org](www.gnome-look.org) - lots of nice tips & themes.

One thing you might try, to get things to look more like windows is resize the task bars!

Right click on the task bar you want to resize and choose "properties" - this will let you slim down the really FAT task bars - this is also dependant on the font and size of font, so set them before you play with your bars.

So, things to check;

Admin > Prefs > Fonts

Admin > Prefs > Themes

Admin > Prefs > Windows

Hope this helps! :)

### Post by Efwis on 2007-01-03
> **manutdfan2850 said:**
> i have the same issue. My desktop feels really cramped compared to when I was using Windows. Isn't there a way to shrink the default icon/font/window sizes?

to change your icon size you can do one of two things.

right click on each icon on your desktop and choose stretch Icon, this will put the icon in a four cornered box that lets you change the size of the icon smaller or larger by left clicking on one of the boxes in the corners and dragging at the same time.

otherwise you can find the icon images in the filesystem, adn then make them smaller using the Gimp.

these are the only two options I know of. although there might be more
