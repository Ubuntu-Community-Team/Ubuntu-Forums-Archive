### Post by dwblas on 2006-12-23
> I've tried getting tahomma.ttf You have to have xfs or another similiar program to use ttf fonts.  See if Synaptic has it installed or not.  There are many howtos on the web for xfs, although I don't have any addresses so you'll have to Google.  Also, you may have to run mkfontdir and ttmkfdir to get it to work, so make sure the tutorial that you find also includes those two programs.  If you do find that all of this is necessary, post back with instructions on how you did it so others will know how to solve this problem.
