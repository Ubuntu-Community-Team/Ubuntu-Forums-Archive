### Post by CatKiller on 2007-11-29
Use ```
gksudo gedit /etc/X11/xorg.conf
``` to edit the file, and in the Monitor section you can put
```
        HorizSync       30-54
        VertRefresh     50-120
```This will tell X to use only frequencies that your monitor can handle. You might still get that message before X has started (ie, before the graphical login screen) but that shouldn't matter too much.
