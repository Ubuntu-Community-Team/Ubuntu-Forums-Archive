### Post by talsemgeest on 2008-04-01
First of all, Welcome to Ubuntu!!!

BSOD (Blue Screen Of Death) is a very deep problem. It means there is something VERY wrong with your windows partition, and windows won't start up until it is fixed. 

I know you don't want to reinstall, but that seems like the best thing to do right now.

The only other thing I can think of is to run chkdsk from the xp installation cd.

Hope this helps :)
