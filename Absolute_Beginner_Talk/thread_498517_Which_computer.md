### Post by Nervouswreck on 2007-07-11
yeah tiger direct is good but building yourself is still a lot cheaper. Either way i recommend them for parts.

It's like a jigsaw only easier, because you can tell easily which peaces go where.
For your needs it sounds like you don't need anything very specific, just slightly high end power. Try a processor 2.4 Ghz and up, 1 gig RAM, and as large a harddrive as you can get. You might want to just put your old HD in to save time, but a second drive is a good idea anyway. Either you partition it for Ubuntu or use the whole thing as 1 volume. Get as many ports as you need etc. You could actually get a $700 desktop for about 350

I know this sounds pretty general but its the best advice i can give. 
HINT: When buying the motherboard read the fine print carefully. The ptocessor compatibility is usually tucked away somewhere. (Moot if you buy a kit that comes w/ processor

let me know by pm how it works out or if you need detailed help.

Regarding the OS while I use Ubuntu as my main OS I need XP and have installed it in Ubuntu using VMware.

Anyhow, if you currently have a licence for XP on your machine you can get installation discs off eBay very cheap, I got one for £9, $18.  You need to make sure that it is the installation disc not a restore disc, mine was a Dell version as I believe they ship the full installation discs.
