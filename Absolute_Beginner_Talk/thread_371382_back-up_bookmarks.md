### Post by 23meg on 2007-02-26
Assuming you're using Firefox: 

Bookmarks / Organize Bookmarks / File / Export

### Post by qamelian on 2007-02-26
If you are using Firefox, go to the Bookmarks menu and select Organize Bookmarks. When the bookmark organizer opens, go to the file menu and use the Export option.
