### Post by Therion on 2008-03-14
Here&#8217;s how to fix the booting to a black screen issue.  

You are going to have to edit a file called menu.lst a few times to make the fix permanent.  

Boot from the LiveCD.
At the Menu you get hit &#8220;F6&#8221;.
Using the arrow keys, find and delete the the words **splash** and **quiet**.
Hit &#8220;Enter&#8221; twice.  This will get you to the desktop so you can do a full install.

If you DON&#8217;T want to do a full install you can quit reading now, but you&#8217;ll need to repeat these steps each time you boot using the LiveCD.  To install Ubuntu and fix this black screen issue permanently continue reading; you are going to need to edit menu.lst a couple more times before you are done.

Once Ubuntu is installed, reboot your machine and get ready to hit the ESCape key.
When you see GRUB (it&#8217;s the countdown timer) hit the ESCape key.
Keep your selection on the first line of the menu and hit "E" to enter Edit mode.
Using the arrow keys, go to the second line of text and and hit "E" again to edit this line.
Delete the words **splash** and **quiet**.
Hit &#8220;Enter&#8221;, then "B" to continue booting.
Once you're in Ubuntu, open up the Terminal and enter:

```
gksudo gedit /boot/grub/menu.lst
```

Find and delete the words **quiet** and **splash** from the line that starts with Kernel  one more time.
Save the file (overwrite), Exit&#8230; And you&#8217;re done.

Now do all your updates and THEN installl the Restricted Driver for your video card if needed. 
Let me repeat that: Updates first, THEN your video driver.  NOT the other way around.

### Post by incen on 2008-03-14
Haha, I'm not sure if it's true...
Well, did you 'just' install Ubuntu?
How about this?

Put in your LiveCD and boot with it. Once you get to its environment. Similarly, check out its xorg.conf. I will suggest you to copy it into a memory stick or something. Then once when you boot into the system you installed, overwrite it.

1. Copy xorg.conf in the virtual environment by:
```
cp /etc/X11/xorg.conf /media/xxxx <- whatever your disk is called
```

2. Boot into the system you installed. First, backup your current xorg.conf by
```
cp /ect/X11/xorg.conf /ect/X11/xorg.conf.bk
```

3. Then, overwrite your xorg.conf with the one you copied from the LiveCD virtual environment.
```
cp /media/xxxx/xorg.conf /ect/X11/xorg.conf
```
