### Post by maddog39 on 2007-10-06
Its archite**c**ture by the way. And you are getting that error because you have the 64 bit of Ubuntu installed. What you need to do is run the 32 bit or i386 version so that 32 bit applications can run. There is a way to run 32 bit applications in 64 bit linux, however its not a very easy task. You will also run into this problem when trying to use flash or any other proprietary software which does not support the 64 bit version of linux. I suggest you download the i386 version of Ubuntu and install that instead.
