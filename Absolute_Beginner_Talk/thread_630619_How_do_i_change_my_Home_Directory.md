### Post by compmodder26 on 2007-12-03
Not necessarily.  If they are coded properly, they will use ~ as your home directory.  You can change the home directory in System->Administration->User and Groups.  Select the account you wish to edit and then click the "Properties" button.  The setting for the home directory is under the "Advanced" tab.

*Note:  I have never done this before.  I'm pretty sure you'll have to copy all relevant program configs from the old home directory to the new one, otherwise some programs may croak.
