### Post by kpkeerthi on 2008-04-08
To check if your package database is corrupted, open Synaptic and
      Choose Edit > Fix Broken Packages from the menu.
      Choose Apply Marked Changes from the Edit menu or press Ctrl + P.
      Confirm the summary of changes and click Apply.

Search for **ubuntu-desktop** in synaptic and choose for reinstallation. This will reinstall packages that were removed by accident.
