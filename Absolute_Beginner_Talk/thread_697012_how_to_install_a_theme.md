### Post by mkoehler on 2008-02-14
Hi,

    To install a theme, go to System>Preferences>Appearances.  On the theme tab, click the "install" button and find the *.tar.gz theme file.  One other option is to put the theme file that you downloaded in your /home/<user>/.themes directory and select it in the Appearances dialog (System>Preferences>Appearances).
