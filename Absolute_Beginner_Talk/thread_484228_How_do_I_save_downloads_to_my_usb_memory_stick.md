### Post by kd_pease on 2007-06-25
When you plug your USB memory stick in you should get a new icon on your desktop (I assume you're running gnome? Not sure about KDE.) If you double click on that icon you'll see the files already on your usb stick. 

You should be able to drag the downloaded file into that window to copy the file to your memory stick. Once the copy has done, right-click on the usb stick icon (The one that appeared when you plugged it in) and choose "Eject." The system will tell you when you can remove your usb stick. Job done, proceed to print shop...!

Hope that makes sense...
