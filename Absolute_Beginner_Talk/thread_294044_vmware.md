well ... what exactly are you trying to do?  do you need to create a virtual machine?  have you already created one an just need to start it?

### Post by Karma_Police on 2006-11-06
You can create a virtual machine here: [http://www.easyvmx.com/](http://www.easyvmx.com/)
After you choose the options to be used in your virtual machine it will provide you with a file to download. It will contain several files, including a *.vmx file that you can open in vmware.

If you put a windows xp cd in your drive, vmware should pick it up when booting (depending on the options you chose in that site), and you can then install it.

### Post by dmizer on 2006-11-06
with vmware server, your host computer is set up as a server to your guest operating system so that when you start your virtual machine (any os will do), it will have internet access.  that's why it's called server (among other things).

again, if you do not already have a virtual machine created, you will have to create one.  there are several ways you can do this, but your best bet is to install vmware server so that you can install your xp just as though you were doing it on a new computer.

just follow the link that boban gave, and you will have an operational windows xp virtual machine with internet access and all the handy features.
