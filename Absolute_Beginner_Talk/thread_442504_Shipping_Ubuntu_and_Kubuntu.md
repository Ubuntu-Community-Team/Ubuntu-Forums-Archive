### Post by MiCovran on 2007-05-13

### Post by Happy_Man on 2007-05-13
if you request from shipit.ubuntu.com, you get ubuntu. from shipit.kubuntu.org, you get kubuntu. If you want to switch afterwards, run either
```
sudo aptitude install kubuntu-desktop
```
or
```
sudo aptitude install ubuntu-desktop
```

---

### Post by Bachstelze on 2007-05-13
If you request from one website and then another using the same Launchpad account, both requests will be "merged" and you will get all the CDs together.

---

### Post by ep2011 on 2007-05-13
Yes, I have done this twice. Just request from both sites, and they will put them in as one request and you will recieve an envelope containing both. (Though, you could just do as Happy_man says and nistall the other one to save Ubuntu money)
