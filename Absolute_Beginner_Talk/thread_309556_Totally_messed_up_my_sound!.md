### Post by lhtown on 2006-11-29
The best solution is to get a sound card that supports multiple channels. Otherwise, you will have to rely on software mixing,

Software mixing, while it is improving, is still less than perfect in Linux. OK, the truth is that it is still fairly bad.

Did you forget what the original settings were? Why can't you just set it back like it was?

Under sound preferences, I have everything set to automatically detect with sound capture set to use ALSA (It isn't working properly right now).

On the sounds tab, you can select the box for hardware mixing and deselect system sounds at least until you get it working.
