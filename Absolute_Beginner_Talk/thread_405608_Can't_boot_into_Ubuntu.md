### Post by kfrance on 2007-04-10
Are you sure that you installed ubuntu?  Did you get through the entire install? One stage of the installation should install grub (a boot loader) for you.  Grub let's you select what operating system you want to boot.  If you can I would see if anything has been written to the hard drive where you indicated that you wanted it installed, and if not go through the installation again.  You should only have to click on the installation icon and then indicate how you want to install it.
