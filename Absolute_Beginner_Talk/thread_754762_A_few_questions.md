### Post by sunstriker on 2008-04-14
Gnome uses less recources while KDE use more. But KDE is overall better looking than Gnome. Ofcourse you can tweak gnome yourself untill it looks even better than a standard KDE.
I'm not sure but I think 8.04 uses KDE4, which I think is worth giving a try.

If you're not sure yet you can install both of them: just install ubuntu and install KDE from the repository (use synaptic and search for kde desktop), or install kubuntu and search for gnome desktop).
64 offers better performance, but you'll have to install a 32bit firefox, because 64 bit firefox's support for flash and java applets sucks. Also some software is not available in 64 bit...
have fun!

edit: you can choose between a KDE or gnome before logging by clicking on 'session' and selecting either gnome or kde
