### Post by taurus on 2007-04-20
Edit /etc/fstab

```
kdesu kate /etc/fstab
```
and replace this line

```
UUID=04d96f79-ea61-47d0-ba90-06d32c932ef3 none swap sw 0 0
```
with this one.

```
/dev/sda5   none   swap   sw   0   0
```
Save it and reboot.  Now, see if swap is mounted.

```
free
```

p.s.  And I would strongly recommend you remove the last line, **/dev/sda5 /media/hda5 swap sw,user,owner 0 0**, in /etc/fstab.  Shouldn't let Automatix modified your /etc/fstab at all.
