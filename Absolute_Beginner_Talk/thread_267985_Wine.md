### Post by pseudonym on 2006-09-29
Hi,

to complete wine setup you need to run 'winecfg' from a terminal. You will be asked to answer a few questions about your system.

For further help, however, where you really need to go is the wine website [www.winehq.com](www.winehq.com) . There is loads of documentation there, a wiki, and an app database with specific info on running a whole range of windows apps.
