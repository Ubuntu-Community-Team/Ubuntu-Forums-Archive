### Post by nanotube on 2006-04-09
if you are using gnome (most likely), then yes, you can assign keyboard shortcuts - go to system>preferences>keyboard shortcuts, and assign away. :)

also, by default, the shortcut to access the main menu is <alt-f1>. you can see a lot of the other default shortcuts in that same keyboard shortcuts preference panel.

### Post by GaFFy on 2006-04-09
I got a question, is there a way to edit the keyboard shortcuts to use the windows key <super> and another key.

On windows I was used to hitting Win+M to minimize all windows but the keyboard shortcuts in gnome only recognize <super> by itself.

---

### Post by htinn on 2006-04-09
Go to System/Preferences/Keyboard, click the "Layout Options" and change the "Alt/Win Key behavior". I have mine set to Super = Win keys (default).
