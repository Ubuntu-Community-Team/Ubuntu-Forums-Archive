### Post by Medieval_Creations on 2007-01-26
You should have no problems running 32bit.  That's how my rig is setup too.
The 64bit would run without a problem system wise.  The catch about 64bit is that not all of the programs have been recompiled for 64bit.  So some applications that are available in the 32bit repositories may not be in 64bit.  There are ways around that to, like installing the 32bit libraries and recompiling the program from source yourself.

I haven't had any problems running 32bit on my AMD64 2200.
