### Post by StefanoCole on 2006-11-30
Well, the fact that you are using an USB 56k modem could be a problem, since an USB modem probably needs a specific driver to work. The best thing would be if you are able to get a serial modem. With such an external serial modem you do not need any specific drivers. If you are not able to get a serial modem then check if the manufacturer of your USB modem provides a driver to work with Linux OS.

Stefano
