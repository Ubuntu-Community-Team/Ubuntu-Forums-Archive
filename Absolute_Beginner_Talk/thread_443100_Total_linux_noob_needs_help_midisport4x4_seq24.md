### Post by nightskywind83 on 2007-05-15
Ehhh...

The Ubuntu install CD is great because it lets you try things out before you commit to installation, but it sounds like for your purposes it might be better to just allocate a few gigs of space and install it... All live CD's run slower simply because they have to go back and access the CD any time the OS would normally access the hard drive (which is MUCH faster).  Also, when you install Ubuntu on its own partition, it will install GRUB on your Windows boot record so that whenever you boot your PC, you have the option of which OS to start.  Just make sure you don't choose to "erase such and such partition and use entire disk" when you install Ubuntu ;)

If you're looking for a Live CD with a lot of options you might want to try Knoppix, available from [www.knopper.net](www.knopper.net).  Still, it's a live CD and its speed is limited.  I'm thinking if you're trying to create music on a live-CD OS you'll just get frustrated because it runs so slowly.

If you want to try Xubuntu, go to this page to download the ISO and burn it to a CD:

[http://www.xubuntu.org/get](http://www.xubuntu.org/get)

Hope this helps.
