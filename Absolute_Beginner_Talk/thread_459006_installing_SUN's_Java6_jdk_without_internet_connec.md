### Post by seshomaru samma on 2007-05-30
use another computer 
go [here ]("http://java.sun.com/javase/downloads/index.jsp")
download java (choose  'Linux self-extracting file')
transfer it with a usb or cd or something 

open a terminal
cd to the directory you downloaded it to 
(if you downloaded to the desktop go 'cd ~\Desktop')
to install it you probably just need to type the name of ther file
so if it's called sun_java_linux6.bin then just run:
```
 sun_java_linux6.bin
```
