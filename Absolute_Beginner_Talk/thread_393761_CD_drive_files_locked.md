### Post by chili555 on 2007-03-26
I would create a directory in your /home/miggols directory (or whatever your username is) called Studio8. Drag-n-drop the file from the CD to /home/miggols/Studio8. Right-click the files and select Properties -> Permissions. Change them to Read and Write, check Executable and you should then be able to run them with wine from there.
