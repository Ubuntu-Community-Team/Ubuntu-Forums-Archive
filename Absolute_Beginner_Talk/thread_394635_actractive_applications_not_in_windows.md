### Post by Gina on 2007-03-27
The quick and easy download and installation of software is certainly something that leaves Windows standing :)  And all dependencies found for you.  Furthermore, all the software has regular updates all in one place - no hand checking and doing it piecemeal as in Windows.  And for the most part you don't have to reboot almost every time you install or update.

Then there's the speed of Linux in general - so much faster than Windows.

I think Open Office beats M$ Office, Firefox beats IE, Gimp beats anything I have found in Windows, I like NVU for webpage design and update - but all these are available in Windows versions so people can try them before changing OS, like I did.  I don't think I've found anything only in Ubuntu as yet - except maybe EtherApe - a network traffic viewer.  Oh and I did find Evolution so nice and simple to set up and get working.

May be a minor point to some people but one big advantage to me is that it's all free!  You don't have to buy software and then find you don't like it or it clashes with somethimng else as I've often found in Windows.

It may be my personal opinion but I find Ubuntu easier to use than Win XP even with default settings and it's certainly more configurable if you want to change the appearence - indeed if anyone prefers the Windows look, they can make Ubuntu look the same.  Or a MAC look etc.

Having said all that, you can still have Windows accessible on the same machine, as Ubuntu automatically installs dual-boot if it finds a Windows system.  Best of both worlds!

I have only been using Ubuntu for just over a month but I'm WELL convinced of it's merits and use Windows as little as possible.  I have it installed on three PCs, one very old (AMD 450MHz ,128MB RAM, 10GB HD), a laptop and my main desktop - all wirelessly connected to ADSL router.

I tried Linux briefly in the past and found it tedious to set up and get working, - though once going much more stable than Windows.  Now with the latest Ubuntu distros it's well caught up with Windows and I think it's brilliant.  Can't say more than that! :lol:
