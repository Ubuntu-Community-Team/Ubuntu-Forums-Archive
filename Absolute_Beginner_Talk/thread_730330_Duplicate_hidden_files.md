### Post by ruy_lopez on 2008-03-20
> **DaveGiant said:**

What is this second file for? Can it be safely removed? Is there a way to disable this? If not is there a way to batch delete these files?

Files ending with ~ are backup files. It is safe to delete them. There is usually an option to disable the creation of backup files. (You didn't say which editor you are using).

---

### Post by The Cog on 2008-03-20
In gedit it's configured in Edit->Preferences, Editor tab, File saving: Create a backup before savning. The tilde(~) is a common way to mark backup copies, so other editors may have similar behaviour.
