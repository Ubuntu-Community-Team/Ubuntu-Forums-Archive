### Post by frodon on 2006-12-18
The fastest way to do it is to open a terminal and type :
```
sudo dpkg-reconfigure xserver-xorg
```If this doesn't work then post here your xorg.conf file, open a terminal and type :
```
sudo gedit /etc/X11/xorg.conf
```
