### Post by ryanVickers on 2007-10-06
use "k9copy" to rip **any** ;) DVD to an ISO, then burn the ISO with K3B

Works every time! :p Just sometimes the DVD's are really big, like over the 4.4Gb mark and you'll rather have to cut out parts of the movie (extra languages, special features), or just buy dual layer DVD's (expensive!) :(

Note: I'm on 64 bit, and I've heard stuff like k9copy and burning a single 4Gb+ file doesn't work on 32 bit... (just to warn you!  Don't know if it's true though...)
