### Post by sillyxone on 2008-03-12
just Googled for you ...
[http://openoffice.blogs.com/openoffice/2007/03/openoffice_impr.html](http://openoffice.blogs.com/openoffice/2007/03/openoffice_impr.html)


- click on menu Format -> Page...
- in the Page dialog appear, click tab Background
- in the Fill listbox, select bitmap, then select the picture you want


if you want to import your own image:
- go to menu Format -> Area ..., select tab Bitmap
- click Import button, choose your image
- once it's imported, go back to Format Page
