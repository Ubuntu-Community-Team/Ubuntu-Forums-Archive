NO need for anti-virus, yes use a firewall in Linux. I use Firestarter and it works just fine.

What do you mean it's always updating?

If you want only ubuntu (thats how my 2 pc's are set up) then insert the ubuntu install cd and reboot. When you come to the ubuntu screen select install (runs live cd) then select the install icon on the desktop.

Once installing I would use custom partitioning... I have all my pc's set as follows:

partition 1 / (5GB) (ubuntu operating system goes here)
partition 2 linux-swap (1024mb) (emergency memory dump)
partition 3 /home (the rest of the hard drive) (your free file space)

Make sure to back up your important doccuments from windoze and ubuntu before crashing the pc
