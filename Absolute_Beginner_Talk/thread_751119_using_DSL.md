### Post by tgalati4 on 2008-04-10
Well, not exactly.  DSL is an integrated operating system in that the repositories have special, modified versions of popular programs.  These are slimmer and use less resources so that they run OK on older systems.

You can install apt on a DSL machine and use aptitude but you may be disappointed with the integration between gnome and kde with DSL.

There are other lightweight distros out there such as Linux Mint XFCE which is based on Ubuntu with a ligher-weight XFCE desktop.

What are your system specs?  500 MHz PIII and 256 MB of RAM is a reasonable minimum to run gnome and kde.  Anything smaller is DSL territory.
