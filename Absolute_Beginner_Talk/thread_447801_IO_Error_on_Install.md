Hey guys,
I've decided to make a partial switch to Ubuntu so I'm dual booting Vista and Ubuntu at the same time. I'm on an AMD 64 3500+ and downloaded 7.04 for AMD64 last night.

"I/O Error
Could not read boot CD
[Reboot]"

Anyway, I'm not sure what's up with it. Here's my system spec:
Athlon 64 3500+ (2.2Ghz)
ATI Radeon x1050 128mb
2048MB PC2-4200 RAM
Lightscribe DVD-RW Drive

Seagate Barracuda IDE 40GB as a slave to the DVD drive.

### Post by netwerx on 2007-05-18
This is usually cause by a bad cd, or cd drive.
you may want to do a media check by booting off the cd, and pick the options to check cd for errors.  I can't remember the exact wording.

Also, this can happen from a bad iso download.  you may want to try to download from another source, and burn again.

Good Luck.
