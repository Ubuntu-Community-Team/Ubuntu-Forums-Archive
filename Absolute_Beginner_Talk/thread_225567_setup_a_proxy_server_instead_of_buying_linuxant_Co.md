### Post by lance bermudez on 2006-07-29
will need two computer one that has the modem driver already working and installed. then download FreeProxy from [http://www.handcraftedsoftware.org/index.php?page=4&action=category&cat_id=2](http://www.handcraftedsoftware.org/index.php?page=4&action=category&cat_id=2)
don't worry FreeProxy is free and set it up if you want username and your own ports other wise just leave it alone but turn it on with the start/stop icon. on the linux box on the menu bar go to system >> preferences >> network proxy and set the manual address for the other computer and the port you are using the default is 8080 unless you changed it. Still on the linux box set the same address for firefox and synaptic. just a side note i use a windows computer with modem and gave it an network address i made up 192.168.3.1 subnet mask 255.255.255.0 no gateway. then i gave the linux box the network address 192.168.3.5 subnet mask 255.255.255.0 no gateway. i have not tried it but in theory you could use FreeProxy with wine on a linux box but i did not have a linux box that had a modem driver or modem that work faster then 14.4 and i wanted it faster so i set this up becuse the reason i use linux is becuse it is free.
