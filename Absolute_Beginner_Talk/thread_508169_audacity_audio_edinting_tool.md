### Post by EXCiD3 on 2007-07-23
You need to install the LAME encoder. To view the preferences go to Edit->Preferences, and the File Formats tab contains that information.

Choose File-> Export as MP3 to save as an MP3. You will need the lame encoder for this.

This link might help you: [http://en.wikibooks.org/wiki/Linux_Guide/Audacity_to_MP3]("http://en.wikibooks.org/wiki/Linux_Guide/Audacity_to_MP3")
