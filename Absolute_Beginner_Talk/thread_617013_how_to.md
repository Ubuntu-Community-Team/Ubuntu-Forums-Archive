### Post by jasay on 2007-11-18
Make sure your video card driver is installed:
  Intel should be ok out of the box.
  Nvidia or ATI go to System Menu -> Administration -> Restricted Drivers Manager and check the checkbox

Then go to System -> Preferences -> Appearance -> Visual Effects Tab and set it to Normal or Extra.  For extra control install [ccsm]("apt:compizconfig-settings-manager") which will then be under the menu as Settings -> Preferences -> Advanced Desktop Effects Settings.
