The small boost in speed you'll get from 64-bit might not be worth the [work around]("http://ubuntuforums.org/showthread.php?t=24575") (that guide is actually for hoary, you would replace hoary with dapper in the procedure) you have to make to get 32-bit apps like wine, flash, and codecs, which only have 32-bit versions. It's not difficult, but just using the generic kernel will save you some time and energy. 
You can make Windows FAT32, but like I said, you'll have to create that partition first (with your own tool, or gparted on ubuntu CD), and in the installer tell windows to use it rather than creating a new NTFS partition. If you so desire, this will leave you with only two partitions. It's a good setup, I just prefer all my media on one medium, so I know where it all is at all times. It's still perfectly doable.
