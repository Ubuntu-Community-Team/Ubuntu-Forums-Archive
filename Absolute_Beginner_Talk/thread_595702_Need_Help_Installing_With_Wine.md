### Post by Henaro on 2007-10-29
Plop the CD in.  When it mounts open it up (double click on it the Desktop icon).  Navigate to where the install .exe is.  In Thunar you can right click in there and go to 'Open terminal here', but I'm not sure how it works in Nautilus.  If you can't do that then open up a terminal change to that directory using the cd command and then run it with "wine <install_name>.exe".

Hope that helps~
