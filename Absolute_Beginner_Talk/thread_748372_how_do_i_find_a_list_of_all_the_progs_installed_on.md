### Post by drs305 on 2008-04-07
If you want a list of all the packages installed on your machine:
```
sudo dpkg --get-selections >/home/<USERNAME>/Desktop/mypackages
```
Or to scroll through them in terminal:
```
sudo dpkg --get-selections | more
```
Or searching for a particular program, such as nautilus:
```
sudo dpkg --get-selections | grep nautilus
```
