### Post by Blutack on 2007-01-28
Click the applications button and in the accessories menu is an option for a terminal, that is where you type.
There is a nice easy way for wireless though, its called network manager but it isn't install by default.
If you go into system, administration, synaptic package manager.  First click settings, repositorys and tick all the boxes.  Synaptic should now update its software list.
Then search for 'network-manager-gnome'.  Tick the white box next to it and choose install.  You get a list of other software that will be installed, hit yes and click apply.  Once the install is complete you should have a picture of two computers in your system tray.  Click on it and a nice list of wireless networks will pop up.  Click one to connect!
