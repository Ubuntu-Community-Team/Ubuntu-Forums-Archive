### Post by Golyadkin on 2007-08-18
Anjuta is a good C/C++ programming IDE, and KDevelop is great if you use KDE.

Also, to create games for a console, you don't just need to program, you also need 2D texture painting software (The Gimp, or Photoshop in Wine) and 3D modelling software (Blender), software to edit and create audio (Audacity) and an engine that works with that specific console (Unreal Engine 3 for instance, which costs you thousands of dollars) or a free engine like Ogre 3D that you can maybe change to make it work on the PS3, which is unlikely. Then you need a development kit for the console as well, and loads of plugins to export to proprietary formats, and the list goes on.

But don't let this demotivate you, if you want to do some bad enough, you can find a way :) even for something as ambitious as this. Good luck!
