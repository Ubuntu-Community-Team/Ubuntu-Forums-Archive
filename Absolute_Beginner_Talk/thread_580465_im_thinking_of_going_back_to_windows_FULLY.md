### Post by American_Outcast on 2007-10-18
Do what you want to do. Stay with Ubuntu or go back to Windows. Thats what is called having a choice.

You could also order the 7.10 CD for free from the Ubuntu website. You could wait a couple  of days for things to ease up.  (Remember these servers are a FREE service for all of us.)

If you don't want to duel boot but don't want to give up Windows and want to keep Ubuntu, then you are a bit stuck. Weigh the pros and cons on each side then make up your mind. If you use Windows for non 3D card programs then you can probably run all you need to in a vmware server. Installing windows on that is almost like installing it on your computer, though there are a few things that you will probably need help with getting them going.

If you have 3D programs/games or non 3D Windows software that you want to run then Wine/CrossOver might be the thing for you. It  covers many programs but of course not all programs.

So again it is up to you. What you feel comfortable with and what you want. From my personal opinion it would be great if you stayed with Ubuntu or Linux, but if you go back to Windows only then that is fine as well.

Keywords here are choices. Windows alone doesn't give real choices. Linux gives many choices and options. Most of them are free too.
