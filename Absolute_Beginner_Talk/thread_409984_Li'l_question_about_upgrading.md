### Post by 3rdalbum on 2007-04-15
Use the Update Manager program to upgrade to Feisty, once it has been released. Synaptic can do the job, but apparently Update Manager will be able to compensate for unusual circumstances.

If you want to do the upgrade from a CD, download the Feisty Alternate CD. When you put it in, you will be asked if you want to add it as a CD source. Click Yes. Now you can use the Update Manager, and it will pull all the new packages from the CD.
