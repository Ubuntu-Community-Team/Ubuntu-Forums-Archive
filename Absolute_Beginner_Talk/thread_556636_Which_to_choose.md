### Post by cmnorton on 2007-09-21
If this is for work, make a list of what you can run on Windows and make sure you can run an equivalent on Linux or run your Windows application under Wine.

I am using a laptop -- installed with Ubuntu 6.06 -- as a portable workstation, and can pretty much do my work. I use a Windows workstation to do VB/Access work.

You can also re-size your hard-drive and try a dual boot for a while, so you have both systems installed. 

If you want an interface the reminds you of Windows, install Kubuntu.
If you want an interface that is a little more like a MAC, install Ubuntu.

Personally, I installed Ubuntu and am happy with it.

### Post by vikram on 2007-09-21
The main difference between the different *buntus is the desktop environments. Generally Xfce is recommended for olderr machines with less RAM CPU. 

[http://en.wikipedia.org/wiki/Comparison_of_X_Window_System_desktop_environments](http://en.wikipedia.org/wiki/Comparison_of_X_Window_System_desktop_environments)

I'd suggest you either with 
a) install Ubuntu(GNOME installed by default) and KDE desktop or
b) install Kubuntu (KDE installed by default) and install GNOME desktop. 

There is no reason why you cant install both. Actually you get several others like Xfce. However each time you log in you can be logged into one or the other. Try both and decide which is best for you. 

Linux is all about choice, you can GNOME apps in KDE and vice versa.
