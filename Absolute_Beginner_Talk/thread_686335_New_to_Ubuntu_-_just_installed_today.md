### Post by fepple on 2008-03-10
I have a VM of ubuntu running on the same machine as I was getting the error on.  So in an attempt to get the gnome-terminal back on my machine I tried the following on my host machine:

$ xhost +
$ ssh my-ubuntu-vm
$ export DISPLAY=my-host-machine:0.0
$ gnome-terminal

So that should have loaded the terminal from my guest-machine on my host-machine.  So I could then SSH back to my host-machine and have the lovely gnome-terminal back.  Complicated and silly way of doing it I know, but thats my kind of thing :)

However, when I ran "gnome-terminal", I got the same error again :O

So I guess that goes someway to confirm that its gnome-terminal not being able to work with my host machines X
