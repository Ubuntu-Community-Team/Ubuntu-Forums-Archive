### Post by Ecthelion on 2006-11-11
> I am going to setting up a new computer and would like to build a dual-processor system. All of the Tyan boards, for example, use AMD 64-bit chips. So, can I us 64-bit chips and install a 32-bit operating systems?

Yes you can

>  (I concerned about the lack of applications in 64-bit, and so am not that interested in a 64-bit OS.) 

In ubuntu, all 32bit programs have a 64 bit version.

> Is there any negatives I should know about?

Some programs like firefox have problems with 64bit because they use plugins (like java & flash) that aren't completely compatible with 64bit. You can install 32bit plugins to compensate.
