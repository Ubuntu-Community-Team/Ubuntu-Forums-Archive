### Post by forrestcupp on 2008-03-12
If you're using a router, the best way to do it is in the router's settings.  That way it's not dependent on one computer being turned on, and you don't have to connect every time you start your computer.  And it always works the same, no matter what OS you're using.

You'll have to check your router's documentation on how to configure it.  Usually, you open a web browser and type the default gateway address into the address bar, and that gets you to your router's settings.

I've never tried it without a router, so I don't know how to set that within Ubuntu.
