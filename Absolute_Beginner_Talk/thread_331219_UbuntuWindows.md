### Post by gitano1 on 2007-01-04
The only way to truly make you computer secure is with a firewall several of which are available. Windows is full of holes which they continually attempt to plug. Linux being based on Unix is an inherently more stable and secure system. The structure of Unix based systems, like Linux and OSX is such that no one but a Superuser has access to the inner workings of the OS. This structure makes it far more difficult for a malicious party to access any of your data or in any way infect your system. Windows, on the other hand, is wide open to attack at any level. 
I find that I can leave my Linux (Ubuntu) and OSX machines running constantly without a problem. My WinXP requires rebooting at least once every couple of days or it develops behavioral anomalies. 
The major advantage of Windows is the software that is available for it. Some things are simply easier to do in a Windows environment because of available software. That is its only advantage.

### Post by Judicata on 2007-01-04
As long as users can install programs without permission from some kind of watchdog, there will be computer malware of some sort. I mean, it would be easy to write a program that deleted everything on a Linux HD if it just asked someone for the admin password to do it. So if a third party had to approve of all software installations, and computers were limited in their functions (i.e., appliances with web browsing and word processing features), then we'd all be more secure. But who wants that (besides MS)?

Linux has fewer security problems because 1) it accounts for far less than 1% of the OS market for desktops (hard for a virus to spread if no one is around) and 2) it has better inherent security features.

Next, open source software is, at least arguably, safer. You will NOT find a program like KaZaA that installs all kinds of malware (spyware, adware, etc...) unbeknown to the user  on a Linux machine that is open source. Why? Well, because it can't do anything hidden - the source is right there (unless everyone who can read the code conspires). Then, not every Linux program is open source...
