### Post by gcos7 on 2007-06-23
I have another longer thread about this, but I wanted to let people know that yes it is possible to use Linux as your main operating system and run your EXISTING Windows installation WITHIN Linux without needing to dual-boot!!  In other words, you can have Windows running in a virtual machine in Linux without needing to create a virtual drive and reload Windows and everything you want with it (and as a result use a lot more disk space).  You can just use you existing Windows installation and run your existing Windows applications!!

For more info, please see this thread:

[http://ubuntuforums.org/showthread.php?p=2878814#post2878814](http://ubuntuforums.org/showthread.php?p=2878814#post2878814)
