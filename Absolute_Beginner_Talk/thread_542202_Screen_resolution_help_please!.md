### Post by david_e on 2007-09-03
Accoarding to your xorg.conf file:

```

Section "Device"
Identifier "Trident Microsystems Cyber 9525"
Driver "trident"
BusID "PCI:0:4:0"
EndSection

```

You have a "Trident Microsystems Cyber 9525" which uses the "trident" driver. So your driver is already ok: it seems that ubuntu is using the specific driver for your video card and not the generic low-res VESA driver. I really don't know the capabilities of the trident driver, so I don't know if it supports every resolutions that your video card supports.

So the problem seems to be in the monitor section here. I don't really know which resolution does your monitor work at. If it's widescreen probably 1280x800 is fine, if it's 4:3 1280x1024 is a conservative solution, but should work. Here I assume 1280x800.

Close gedit if it's still opened.

First we make a backup copy of the file.

```

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup

```

then we open it with gedit:

```

sudo gedit /etc/X11/xorg.conf

```

then find this section:
```

Section "Screen"
Identifier "Default Screen"
Device "Trident Microsystems Cyber 9525"
Monitor "Generic Monitor"
DefaultDepth 24
SubSection "Display"
Depth 1
Modes "1024x768"
EndSubSection
SubSection "Display"
Depth 4
Modes "1024x768"
EndSubSection
SubSection "Display"
Depth 8
Modes "1024x768"
EndSubSection
SubSection "Display"
Depth 15
Modes "1024x768"
EndSubSection
SubSection "Display"
Depth 16
Modes "1024x768"
EndSubSection
SubSection "Display"
Depth 24
Modes "1024x768"
EndSubSection
EndSection

```

And change it like this (change 1280x800 with an apropriate resolution):
```

Section "Screen"
Identifier "Default Screen"
Device "Trident Microsystems Cyber 9525"
Monitor "Generic Monitor"
DefaultDepth 24
SubSection "Display"
Depth 1
Modes "1024x768"
EndSubSection
SubSection "Display"
Depth 4
Modes "1024x768"
EndSubSection
SubSection "Display"
Depth 8
Modes "1024x768"
EndSubSection
SubSection "Display"
Depth 15
Modes "1024x768"
EndSubSection
SubSection "Display"
Depth 16
Modes "1280x800" "1024x768"
EndSubSection
SubSection "Display"
Depth 24
Modes "1280x800" "1024x768"
EndSubSection
EndSection

```

Save and exit gedit. Now you can restart your Xserver by pressing CTRL+ALT+DELETE (all the opened applications will be closed without saving anything so save your work before doing this.). The resolution should be fixed now.

Before trying other resolutions instead of "1280x1024" (that will work on your monitor) remember that a wrong config will make your system unable to go graphical and you will need to fix your xorg.conf, by pressing CTRL+ALT+F1, loggin in and use nano to fix the file:

```

sudo nano /etc/X11/xorg.conf

```
