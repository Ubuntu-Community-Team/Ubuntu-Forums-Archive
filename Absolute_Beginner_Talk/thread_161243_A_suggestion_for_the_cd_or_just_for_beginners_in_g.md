### Post by rorschach on 2006-04-16
I would like to make a suggestion to the Ubuntu developers and a general 'best practice' suggestion for new Ubuntu converts.

Set your /home directory on its own partition, separate from the / partition.

I recently added this to my partitioning setup and it has made me much happier.
I have been having a lot of grief trying to get ubuntu to do what I want, and as a result have been reloading my system over and over. Having a separate /home partition has significantly lowered my cycle time - now, when I reload, my settings are still there.

This would, I think, make the initial install/troubleshooting/debugging and 'getting used to' time for linux converters a lot less painful. Do you think it would be possible to add this to the recommended partitioning setup on the install CD?
