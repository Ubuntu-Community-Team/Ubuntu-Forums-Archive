### Post by Knome_fan on 2005-07-15
[QUOTE=some kid with a powerbook]So I just burn everything on the ubuntu disk onto a CD and boot from it?[/QUOTE]

Essentially yes.

However as there are some issues with burning the iso in OSX, so you might want to take a look here:
[https://wiki.ubuntu.com//BurningIsoHowto](https://wiki.ubuntu.com//BurningIsoHowto)

(Following the firestarter howto you should be on the safe side)

You should also be aware that you need some free space on your harddisk to install Ubuntu on. I think there are tools to do this within OSX, but as I'm not really using it all that often I don't know, but I think the Ubuntu Install CD should also be able to resize your current OSX partition (although I may be wrong here).

Finally, be sure to make a backup of all your important stuff. Not that it is very likely that something will go wrong, but it's of course possible, so a backup is a must.
