### Post by neurostu on 2008-04-08
The Partition Editor that comes with Ubuntu is the same as what you get with the GParted live cd, but you can only work on NON-MOUNTED drives. So if you want to work on the partition that has ubuntu installed on it you will need a live cd. Otherwise you can use the Partition Editor that ships with ubuntu.
