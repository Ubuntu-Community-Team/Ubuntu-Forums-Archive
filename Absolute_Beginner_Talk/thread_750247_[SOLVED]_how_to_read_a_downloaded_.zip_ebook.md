### Post by aeiah on 2008-04-09
.zip is the archive that all the files are contained within. you can extract the files and discard this.

the .nfo file is the information file that the piracy scene group included. it usually includes release group information and some file information. you can get rid of this and all the other bits too. your ebook is the .chm file. you'll have to find software to read it
