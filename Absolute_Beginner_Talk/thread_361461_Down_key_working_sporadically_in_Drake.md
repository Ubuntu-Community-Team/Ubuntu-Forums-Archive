### Post by frostyservant on 2007-02-14
I'm running 6.06 LTS on a Dell Inspiron 1100 laptop.  Recently (possibly when I upgraded to Drake), the down directional key has been working only under certain conditions or in certain programs.  It works in Gnome Panel menus, for example, but not in the system applications launched via those menus.  Most annoyingly, it is also nonfunctional in Firefox and OpenOffice.org.

In those programs where it does not work, pressing the key causes the selected control to behave as if it has lost focus for the duration of the key-press, which suggests that it is a software, rather than hardware, problem.  Supporting that belief is the key's complete functionality when used in tandem with the Fn key to reduce screen brightness.

Thoughts?

EDIT: Problem solved; I had accidentally assigned the down key a shortcut.
