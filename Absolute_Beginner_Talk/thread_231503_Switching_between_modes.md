### Post by billyfoxtrot on 2006-08-07
I've got an easy question:

Sometimes I like to switch to the terminal mode by pressing Ctrl-Alt-F1.  However, I don't know how to switch back into my GNOME desktop.  Is there a keystroke or a UNIX command to do this?

---

### Post by moma on 2006-08-07
cntr + alt + F1...F6  (text mode consoles)
cntr + alt + F7 (graphical GUI on display :0)
cntr + alt + F8 (graphical GUI on display :1 if you have one)
