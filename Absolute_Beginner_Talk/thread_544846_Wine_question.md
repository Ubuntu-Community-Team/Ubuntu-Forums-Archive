### Post by greybirds on 2007-09-06
There's quite a few. You might check out wine's web site ([http://www.winehq.org]("http://www.winehq.org")) and look in the application database. If Halo works in wine, it will be there along with info on any issues. You almost might check out Cedega (commercial), Crossover (also commercial), or Wine-Doors (free), which make configuring Wine easier.
