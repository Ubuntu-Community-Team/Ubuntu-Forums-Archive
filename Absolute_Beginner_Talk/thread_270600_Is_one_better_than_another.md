### Post by kidders on 2006-10-03
Hi there,

The underlying technology is identical in all cases, really. The only major difference is the user interface. Of course, if it turns out you've made the wrong choice and want to try KDE or Gnome (instead of Xfce, say), all you need to do is apt-get it ... you won't need to reinstall the whole OS or anything like that.

My advice is to play around with the version you like the best. As soon as you hit a brick wall, or bump into something you think is ugly, take a look to see if maybe KDE does things better. I'm sure someone on here with experience in more than one graphical environment will be happy to advise.

Hope that helps.

### Post by mojoman on 2006-10-03
I'd say it's both a matter of taste and a matter of needs. Xubuntu is a slimmer and lighter version than Ubuntu but they also come with different default programs, although you can use just about any program with either distro. If you want good multimedia I'd recommend you to go for Ubuntu. If you have a somewhat older computer, I'd recommend Xubuntu. Other than that, I'd say it's a matter of taste. However, If you do like the Xubuntu look then you could use that on Ubuntu. Install Ubuntu, install Xfce (which is the standard desktop for Xubuntu) and if you need the HD-space, uninstall Gnome  (which is the standard desktop for Ubuntu).

Also, take a look at this guy's webpage, he's a regular at this forum. I had good use of it myself and it contains some useful stuff on desktops.

[http://www.psychocats.net/ubuntu/index.php](http://www.psychocats.net/ubuntu/index.php) 

All the best
Mojoman

ps: I'd say wireless support is quite good. I have two computers with wireless network, one desktop and one laptop. Both were identified by ubuntu/xubuntu and run as smooth as silk. ds.

### Post by aysiu on 2006-10-03
Ubuntu runs more services and applets.

It has a trash can. It has drag-and-drop theme installation. It has a configurable external device manager.

You can configure Xubuntu to have a lot of Gnome-like functionality, but XFCE is generally known for being less fully featured and a lot faster than Gnome.
