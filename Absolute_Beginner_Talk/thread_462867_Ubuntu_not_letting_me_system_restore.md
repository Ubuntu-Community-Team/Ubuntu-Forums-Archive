### Post by vanadium on 2007-06-03
You need to go into your BIOS setup program and change the boot sequence there in order to have the system boot first from a CD. How you get into the boot program is different between computers. Usually, you have to hold a specific key during the boot proces for the system to go into the bios. Most computers briefly display what key you must press during startup. Mostly, it is either the F2 or the Del key.
