### Post by schorsch on 2007-08-30
As far as the layout is concerned:

-Right click on one panel
- Choose "Add to panel"
- Scroll down to utilities and add "Keyboard Indicator"
- On the panel you will get a new symbol with the actual layout
- Right click on this symbol
- Choose "Keyboard Preferences/Layouts" and add the ones that you want to .....

If this also changes the language you type in ... I don't know...
