### Post by nalmeth on 2006-08-10
Come now that is windows software. Do you use that software in ubuntu?

There are also native linux tools to do the same thing.

For VCD's I just use avidemux to convert to bin/cue (VCD format), then use K3B to burn the VCD

For a DVD, I think you just have to convert to MPEG-2, and use something like K3B's Video DVD burning, and throw the files in the VIDEO-TS folder. 

Tovid works, if you want to use that guide.
