### Post by aysiu on 2006-08-28
```
sudo aptitude update
sudo aptitude install xubuntu-desktop
``` Then if you don't want Ubuntu any more: [http://www.psychocats.net/ubuntu/purexfce](http://www.psychocats.net/ubuntu/purexfce)

### Post by aysiu on 2006-08-28
Actually, I just realized those *were* the instructions for going from Xubuntu to Ubuntu! I've since switched them.

Here's what you want (the original instructions before I edited them): ```
sudo aptitude update
sudo aptitude install ubuntu-desktop
``` If you don't want Xubuntu any more afterwards: [http://www.psychocats.net/ubuntu/puregnome](http://www.psychocats.net/ubuntu/puregnome)
