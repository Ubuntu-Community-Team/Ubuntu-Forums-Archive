### Post by AmboyGuy on 2006-02-01
Use Firefox's **safe-mode**
on a command line type ** firefox -safe-mode** This disables all installed extensions & themes. 
Now click Tools > Themes and uninstall the suspect theme (also works with extensions).
Restart firefox and you should be ok.
