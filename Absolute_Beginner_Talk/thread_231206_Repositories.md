### Post by Sutekh on 2006-08-07
I don't know of any repositories that have Frostwire in them.

You can always get the Ubuntu specific package straight from the [Frostwire front page]("http://www.frostwire.com/")

This is the link

[http://www.frostwire.com/download.php?file=http://mirror1.peercommons.net/frostwire/4.10.9/FrostWire-4.10.9-2.i586.deb](http://www.frostwire.com/download.php?file=http://mirror1.peercommons.net/frostwire/4.10.9/FrostWire-4.10.9-2.i586.deb)

After you download the package you can install it by opening a Terminal window (Applications -> Accessories menu) and using the command
```
sudo dpkg -i FrostWire-4.10.9-2.i586.deb
```
