### Post by popch on 2007-12-16
OpenOffice and Firefox are available for both Windows and Linux. Evolution is an application which only runs in Linux.

So you are out of luck. You can use Evolution as soon as you have installed Linux.

If using Microsoft applications is an issue for you, there are very nice email clients available for Windows, too.

### Post by Paqman on 2007-12-16
Mozilla Thunderbird is a good alternative to Evolution. I prefer it in fact, I find it's junk mail filtering much better. The only thing it lacks is a built-in calendar.  It's available for Windows, Mac and Linux.
