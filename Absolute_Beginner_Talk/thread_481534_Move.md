### Post by Alex Spencer on 2007-06-22
Try 'Cut' and paste.

Or try dragging them.

If this doesn't work, it's because you have an NTFS partition on the drive, and you need to run ntfs configuration tool (found in 'Applications > ADd/Remove' then search for NTFS to install it.
