### Post by gschoper on 2007-02-25
For audio CD's you can change the default with the following:

```
From the Main Menu select: System-->Preferences-->Removable Drives and Media

Click on the multimedia tab and change the command for Audio CD to your preferred player.
```

HTH,

gschoper
