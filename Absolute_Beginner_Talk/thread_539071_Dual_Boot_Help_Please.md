### Post by Eagle 101 on 2007-08-30
I don't think this is possible, at least as far as I have seen all the guidelines for dual booting assume that you have windows installed and you want to put another linux OS on the computer. Windows does not like being told to install into a particular partition I don't think. I hope I'm wrong, but I think you will have to remove ubuntu, install windows, then use the dual boot tutorials from there. (generally defrag windows, go into LiveCD, make a new partition, tell linux to install into that partition).

What you could do though is use a [virtual machine]("http://en.wikipedia.org/wiki/Virtual_Machine") and boot windows inside of your current OS, but to do this it requires a fast computer, as you are basically running two operating systems. 

Cheers, Eagle
