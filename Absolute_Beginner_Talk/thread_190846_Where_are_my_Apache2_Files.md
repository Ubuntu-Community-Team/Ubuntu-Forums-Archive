### Post by indytim on 2006-06-06
If you did the LAMP install per the WIKI, your Apache root should be at
/etc/var/www/

As far as files, you will need to insert files into your root 

for example

[http://localhost/test.php](http://localhost/test.php)

would be a /etc/var/www/test.php

Hope this is what you're looking for.
