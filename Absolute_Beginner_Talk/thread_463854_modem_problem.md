### Post by orange2k on 2007-06-04
There seems to be a problem with USB ADSL modems in Ubuntu...
You can look it up in HELP documentation, there you will find the information that an ethernet ADSL modem is a preferred connection in Ubuntu. If your modem has the option to connect it via RJ-45 cable, then thats the way to go. If not, then you`re probably in trouble. I know because I used to connect via USB on my modem, and it used to work fine in Ubuntu 6.06. However, it didn`t work in 7.04, but luckily my modem HAS an ethernet connection, and I had a network-ethernet card that I could install, so I changed the way I connect to the internet and it works perfectly ever since...

If you don`t have an ethernet network card - they are very cheap to buy. But as I said you got to have an ethernet connection on your modem for this to work...
