### Post by freebird54 on 2007-04-23
While they are there, you can choose to boot with them, old or not.  If you remove them from the list (/boot/grub/menu.lst) they will still be on the machine - but you won't see them anymore.  If you use Synaptic to remove them from your system, they will be gone - and if they are they will no longer work from the boot-up list - even if you haven't tidied that up.

I suggested GrubEd because just tidying things up is a good idea until you are sure that all is well with the new version - and because there are other neat thing you can easily do with (like a picture behind the boot list).  It is quite easy either way though - and if you don't kill off the newest one, no problems will result. :)

BTW - if you edit the file, make sure to do a backup of it first:

```
sudo cp /boot/grub/menu.lst /boot/grub/menu.lst.backup
```
