### Post by aysiu on 2007-05-31
> **jonward0690 said:**
> I am using the same exact aplications in Gnome and KDE
Well then, that might explain the difference.

If you're using Gnome applications in Gnome, they'll use less RAM because they'll be loading only Gnome libraries into memory.

If, however, you're using Gnome applications in KDE, they'll use more RAM because the KDE libraries will already be in memory, and then you're adding in Gnome libraries as well.
