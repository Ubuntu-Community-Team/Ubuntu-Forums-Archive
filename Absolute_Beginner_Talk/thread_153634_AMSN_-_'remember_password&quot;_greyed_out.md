I posted an issue yesterday about the "remember password" option being greyed out when logging on to AMSN - Linus's nifty (does anyone say nifty anymore ? :o)  )    answer to MSN iInstant Messenger.
I couldn't find anything readily available - but now have the answer:
 - If you go to the  **"Create Profile"** scrren you can then enter yourusername  password and save it so you automatically log on my invoking the application.
You of course have to set up an msn or hot mail accaount beforehand.

Hopes this helps anyone having the same quandry.
