### Post by trak87 on 2007-07-02
There are three different Ubuntu install CD's:

1. "Desktop" - This is the typical version of Ubuntu. You can run Ubuntu directly from the CD (no installation necessary) or it gives you an option to install to hard disk.

2. "Server" - For Ubuntu Linux server installations

3. "Alternate" - Text based install of Ubuntu. 


Do you have bittorrent installed? That's the most accurate way to get Ubuntu. Here's a walkthrough of how to get the latest Ubuntu "Alternate" install disk:

1. Surf to [http://www.ubuntu.com/](http://www.ubuntu.com/)

2. Click on "Get Ubuntu". See the different options to get a disk?

3. Let's download using bittorrent (hopefully you have a bittorrent client installed). Click on "Download Now"

4. Click on "complete list of download locations"

5. Click on a location in your country. If for any reason you don't connect, choose another link.

6. Click on "7.04", that's the latest version and the one we want.

7. Scroll to the bottom and click on "ubuntu-7.04-alternate-i386.iso.torrent" and if you have the required bittorrent client, you will start download the Ubuntu Alternate CD image. After you get the entire file it's also a good idea to verify the download is accurate by running md5sum and comparing the result to the value in the file "MD5SUMS."

8. Now you are ready to burn the iso file to a blank CD ROM disk. Make sure you burn the iso file in ISO or IMAGE mode, NOT data mode.

9. Once that's done you're ready to insert the disk, reboot your computer and start installing.
