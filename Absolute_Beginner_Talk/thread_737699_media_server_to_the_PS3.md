### Post by ayenack on 2008-03-28
Haven't got PS3 so not sure this will work on it. If PS3 has a web browser try GNUMP3D (It's in the repo's) dead easy to set up and very good for MP3/OGG music streaming. No good for video though.

To install GNUMP3D in terminal.
**sudo apt-get install gnump3d**

Visit the home page uses and to see if it's right for you.
[http://www.gnu.org/software/gnump3d/](http://www.gnu.org/software/gnump3d/)
