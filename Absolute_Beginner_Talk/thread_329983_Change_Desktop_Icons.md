### Post by Efwis on 2007-01-03
to change the icons on files you already made launchers too, just right click on the icon, choose preferences, and then click on the icon itself, if will open a file manager program so you can navigate to whatever file you want that has icons in it. usually this is /usr/share/pixmaps.

If you make custom icons with the Gimp, you can either save them in /usr/share/pixmaps or you can save them in a file of their own and navigate to it to get them.

hope this helps.
