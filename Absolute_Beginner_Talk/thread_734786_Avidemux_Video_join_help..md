### Post by Squish on 2008-03-25
To add a video, file onto another
(2 videos to make 1 long one sort of thing)
Click 
File > Append


To save them as one file, hit save~


did this answer your question?

---

### Post by alenis on 2008-03-25
Open the fist file, then from the "file" menu choose "append" and select the second file. Repeat for all the files you want to join into a single file, then check and save the resulting video.
