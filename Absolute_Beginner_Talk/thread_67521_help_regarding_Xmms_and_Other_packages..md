### Post by Wolki on 2005-09-20
[QUOTE=psylvester]
1. i want to install Xmms, i have done it, but it doesnt play any mp3

#sudo apt-get install xmms[/quote]

Start xmms from the menu, set output plugin (in preferences) to esd and it should work. 

> 2. I want to install other packages which i have downloaded. What i have to do.
when i download to my home/ direcotry. What next. In windows i can install by double clicking, here there is no double click..
i read somewhere ./configure is there.... make ...etc...!!!!

it depends on what kind of package you have.

If you can, install from the ubuntu repositories. Synaptic is currently the easiest way; you can find it in the System menu. Enable more software in the options (Repositories), you should add (at least) Universe.

Installing other formats can be a little harder, if you have to install one ask again with a little more detail (esp. what kind of package it is).
