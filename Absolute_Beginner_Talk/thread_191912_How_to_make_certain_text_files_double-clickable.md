### Post by Van_Gogh on 2006-06-08
Hi,

Whenever you double-click a text file in GNOME, you get a "Run Run in terminal Display Cancel" dialog. In most cases, this is annoying, for example for htm and html file I almost always want to open them in a web browser. Also, running a htm file in the terminal makes no sense, really.

So, how do you set the default double-click behaviour for a file with a given file extension(htm, py, etc?).

---

### Post by bruce89 on 2006-06-08
Edit>Preferences>Behaviour in nautilus and then select the option in the Executable Text Files section that you want.  I am guessing that this is a FAT32 partition, as html files shouldn't have an executable flag on them.
