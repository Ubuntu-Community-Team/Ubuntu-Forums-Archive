### Post by Maestro23 on 2007-09-13
> **Tootles said:**
> I am trying to install Virtual Box and its a deb file, i open it, it sais this:
error dependecy is not satisfiable: libasound2
anyway i can install it in terminal or another way?

It's telling you it needs libasound2 before you can install it.  Open up Synaptic, search for libasound2.  You will want the dev package of it.  Mark for Installation and install.  Then try to install the Virtual Box .deb

Dependency issues must be resolved before you can install a program.
