### Post by shaft350x on 2008-04-10
I figured out how I did it.

I have "advanced desktop effects settings" (ccsm) to control and tinker with my compiz settings.

From there I hit the "General" settings button, then hit the "Actions" Tab, then under the "General" option I went to "[COLOR="Blue"]Open a Terminal[/COLOR] and then set it for Ctrl + t.  So there was some setup, but if you're already running with Advanced Desktop Effects Settings then it isn't anything else to work with.
