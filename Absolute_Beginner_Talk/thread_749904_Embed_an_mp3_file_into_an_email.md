### Post by xarquid on 2008-04-09
> **Drakkor said:**
> Anyone know how to put an mp3 file into a email,so that when the email is opened the mp3 file will play ? Thanks :)

You could simply do it with HTML and Scripts. Make the e-mail into a coded web page and send it as a webpage / HTML file.

You are, however, facing the issue that the other e-mail clients support HTML and/or the scripting language that you are using which is inadvisable...and bad practices for the script. Straight HTML is okay, but an on:load JavaScript is pretty bad for e-mails ;-) It would probably be thought of as a Virus by most e-mail clients.

What I recommend is just serving them a web link to the file (which in your case should most likely be a Flash player with the embedded mp3 file/audio file) OR just attach the mp3 file.

This is just complete honesty.
