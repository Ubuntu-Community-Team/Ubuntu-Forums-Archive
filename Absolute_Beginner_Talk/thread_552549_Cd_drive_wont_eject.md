### Post by zbot on 2007-09-16
This is nit a solution, but something to consider.

You will not be able to umount the disk if the installer is requesting information from it (hence the device busy message).  I do not know if there is an easy solution for this.

What I would do is generate ISO images of the disks onto the hard drive and install from there.  But I am sure there has to be an easier solution than this.
