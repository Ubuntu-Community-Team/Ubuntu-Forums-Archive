### Post by Mr Frosti on 2006-10-09
If you are experiencing issues with the installation CD, make sure that you have the PowerPC version of the installation CDs. This is an Apple-specific architecture. 

Also, I would not recommend completely reformatting your hard drive until you are comfortable with your Ubuntu installation. This will provide you an "easy out" if you get in a jam where Ubuntu will not boot.
