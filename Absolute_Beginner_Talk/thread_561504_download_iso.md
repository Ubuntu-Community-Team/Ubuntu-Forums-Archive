You first download the ISO file for the DVD (or CD) to your hard drive.

When the download is finished, you then use a CD/DVD writing program to "burn" the image onto the CD or DVD media. If you have a CD/DVD-Writer, you should have some software preinstalled to do that (for example, Nero). In the software, look for an option called "burn ISO image" and then select the file you downloaded.

You download the ISO to your hard disk, then you use a good CD burning program (you DO NOT drag and drop it to the DVD icon in my computer it will not burn right) You can use Nero (shiped with many newer computers) or Sonic, and many others such as Toshiba CD burning, then you choose to burn from an image, and then you press burn and it should burn right. Remember you MUST burn it from an image NOT just like a file because that won't work.
