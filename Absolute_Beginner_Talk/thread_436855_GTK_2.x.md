### Post by mcduck on 2007-05-08
Those are not GTK themes. GDM themes are for your login screen. Just download one you like, then go to System/Administration/Login Window, and on the Local-tab click Add, find the package you downloaded & click Install. Then mark the theme you wish to use.

If you wanted themes that change how your applications look, use GTK2-themes. Usually you can just drag&drop the theme package to the Theme window (System/Preferences/Theme) to install it. But if the package contains more than 1 theme this doesn't install them all, and you need to extract the themes manually to ~/.themes.

### Post by blackened on 2007-05-08
Download the tar.gz file that you want, then go to System -> Administration -> Login Window and then drag and drop the tar.gz file into the subsequent dialog. Press install, then select it from the list to make it your active gdm theme.

### Post by starcraft.man on 2007-05-08
[Here]("http://www.gnome-look.org/") you go. Gnome look is my favorite place to get themes.

GTK themes will modify how all your menus/panels, icons, fonts (in general, some leave default). I use smoked glass, very cool black theme.

You can select them from the left.

Then if you want to further modify the way your windows look you have the choice of Metacity (default), Beryl (emerald themes) or Compiz (cgwd themes).

You can select them all from the left, in addition to other minor things like wallpaper, mouse themes, and other things like such. 

Enjoy the choice, and make your windows friends jealous :p
