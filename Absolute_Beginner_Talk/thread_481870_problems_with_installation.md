### Post by bgissal on 2007-06-23
You may have to manipulate the boot command before you install. On the screen where it says start or install ubuntu at the top, there should be a help menu if you press 'F1'. I highly recommend taking a look at the help that is there. I believe to access the boot options, you have to press 'F6' in the same screen where you pressed  'F1' for help. I have found that laptops require the command vga=771. Sometimes I have also used the command gdth=disable:y for other installs. 

Your system requirements are sufficient. I have successfully installed feisty on a 256 mb dell with a 866 mHz processor.
