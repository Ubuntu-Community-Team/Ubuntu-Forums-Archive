### Post by Kilz on 2008-02-02
That looks like some very bad fraim rates for the ati driver. what do you get when you type ```
fglrxinfo
``` into the terminal? 
To help get compiz working you might have to enable the composit in the xorg.conf
type ```
sudo gedit /etc/X11/xorg.conf
```
The composite should be enabled , it should look like this.
```
Section "Extensions"
	Option	    "Composite" "enable"
EndSection
```
