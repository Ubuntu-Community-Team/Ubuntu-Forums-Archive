### Post by d3v1ant_0n3 on 2006-10-22
Have a read of [ this ](http://ubuntuforums.org/showthread.php?t=232059) thread.

There's lots of useful information, including how to burn the iso (the ubuntu file you've downloaded) onto a cd, and how to install it, or just try it out if you like.

I hope you have fun with Ubuntu- It will be scary at first if you're used to windows, but remember, it's not better (well, in my opinion it is), or worse, just different! You might have some struggles at first if you have a windows mindset- I got very frustrated at times, because I was used to windows, But I've overcome all of my troubles over time. These forums are full of very useful information and fantasically helpful people.

If you need help with a particular thing, there is a 'search' feature in the top right of the page- just type in some keywords for the problem you'r ehaving, and more than likely, you'll find something helpful. If you don't find anything, ask away!
