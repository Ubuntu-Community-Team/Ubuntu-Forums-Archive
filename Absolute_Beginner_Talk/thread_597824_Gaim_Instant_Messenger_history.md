Does GAIM instant messenger save past conversations in a folder somewhere? There was a conversation that I really needed to save but somehow forgot to do so.

### Post by rudeboyskunk on 2007-10-30
You have to enable logging under Preferences>Logging.  Enabling History under Plugins is also helpful.  Unfortunately, this will only work for future conversations, not ones you've already had.
