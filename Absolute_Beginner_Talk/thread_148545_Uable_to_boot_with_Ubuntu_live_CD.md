### Post by pbaehr on 2006-03-22
Two common mistakes when dealing with ISO files.

1) Decompressing it and burning the files
2) Copying the ISO file itself onto a data CD.

An ISO file is a "CD image." This means that it is basically a snapshot of the CD. In order to be properly burned a program has to read through the image and recreate it on the new CD.

Many CD burning applications have an option that will do that.

Which application are you using to burn the CD?
