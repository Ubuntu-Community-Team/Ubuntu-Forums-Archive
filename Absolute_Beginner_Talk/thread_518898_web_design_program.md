### Post by Ek0nomik on 2007-08-06
-Nvu

Although I believe Nvu is no longer supported, you can still find it out there.  I believe Bluefish is in the repository, so a simple:

```
sudo apt-get install bluefish
```

should do the trick.

You could also use Eclipse, which is mostly aimed at programmers but there is an HTML plug in as well.  I use Eclipse for my PHP programming.
