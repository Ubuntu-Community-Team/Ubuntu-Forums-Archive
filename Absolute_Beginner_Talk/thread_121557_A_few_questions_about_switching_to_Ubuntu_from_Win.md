### Post by m.musashi on 2006-01-25
[QUOTE=dueY]1.  Ubuntu has a graphical installer.  It is very easy to install.  Sometimes easier than Windows.[/QUOTE]

I'm not sure I'd call the installer "graphical" as it's mostly just text and a color progress bar. Of course, it's graphical in the sense that you don't have to use the command line.

It's easy to install if your hardware is compatible. In my short experience it installs quickly on production (i.e. Dell) computers but not so well on custom computers unless you make sure not to use something too new or exotic.

So far I've installed it on several new Dell laptops at work and some older desktops and a server. All were painless and installed in about 20 or so minutes.

### Post by Zahrber on 2006-01-25
Well lets just set it straight for all new to Ubuntu. There is a graphical installer. But this graphical installer can't be used with the mouse. You must read the information and make a selection with the keyboard and not a mouse. Since we all can read and count this is very easy.

If you want to try Ubuntu you need the LiveCD. Just remember it will be slower because your loading from a CD instead of installing it onto a HD. But if you keep that in mind you will be able to make a fair judgement.

If you have certain files you need to use like word documents or powerpoint presentation, mp3's and such then that will be no problem as long as you have the proper program installed to handle the file. 
Know all the files you need to bring over from Windows and learn which programs in Linux handle those file types. In Linux you may have several different choices to chose from and the great thing is most all are free or open source. If you are not sure which program handles certain file types the Forum and Google are your friends.

Also do some baasic reading on Linux about file types and directory structures and you will be a head above most other beginners. Also learn as much basics of Linux as possible and read teh forum post and replies this is good for learning even if you don't have that problem, knowledge is freedom.
