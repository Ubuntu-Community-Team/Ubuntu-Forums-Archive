### Post by ReaderRat on 2007-08-02
Go to the Terminal = MENUS >>Applications>>Accessories>> Click on Terminal. The Terminal will open. Type in: "avastgui" (without the quotes). If it asks for a Password, type in your Password (It will NOT show on the screen.) and press ENTER.

This should start avast with a gui interface. Type in avastgui in lower case letters.
