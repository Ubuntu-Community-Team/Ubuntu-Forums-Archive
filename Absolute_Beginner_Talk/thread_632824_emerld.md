### Post by Hospadar on 2007-12-05
yeah I'm using emerald with compiz fusion in 7.10 right now.  you just need to manually replace the gnome window decorator with emerald after you install emerald with the command ```
nohup emerald --replace
```Note that you'll need to have some level of desktop settings enabled above the minimum in System->Preferences->Appearence->Visual Effects

Also if you want more detailed control of compiz, install "compizconfig-settings-manager" from synaptic or off the command line with ```
sudo apt-get install compizconfig-settings-manager
``` then System->Preferences->Advanced Desktop Setings

*Also* if you want the old emerald themes that came with 7.4, you can grab the .deb off the ubuntu package website and install it by hand.
