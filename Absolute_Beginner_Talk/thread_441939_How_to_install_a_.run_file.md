### Post by jfinkels on 2007-05-13
> **doriard said:**
> I want to install the Return to Castle Wolfenstein (wolfspdemo-linux-1.1b.x86.run) file in Fiesty Fawn.  What is the proceed for doing this?

Make it executable by typing the following in the terminal:
```
chmod +x wolfspdemo-linux-1.1b.x86.run
```
Then run it by typing this:
```
./wolfspdemo-linux-1.1b.x86.run
```
