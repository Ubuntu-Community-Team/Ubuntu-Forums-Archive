### Post by Jucato on 2007-04-11
Strangely enough, those sounds aren't turned on by default on Kubuntu. 

Anyway, run System Settings and look for System Notifications. Then from the dropdown list at the top, you can select the KDE Window Manager as the even source and turn off the sounds from there.
