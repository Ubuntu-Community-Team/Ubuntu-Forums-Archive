### Post by marx2k on 2006-12-19
> **xabbott said:**
> utorrent+wine!

You will notice the topic states "Which is the best BitTorrent Client for _Linux_?"

I would say for Linux, best available is Azureus - It has every option you could ever need in a Bittorrent client.  My only complaint with it is that it has a large footprint.  However, it runs/loads twice as fast as it does on Windows.  Its' plugin system is amazing and its built-in autospeed as well as the plugin version will keep your router quite happy and not pinging out in most cases (if set correctly).  If the question was which torrent client was the best for WINDOWS, I would say it'd be a tie between uTorrent and Azureus.  uTorrent is definitely not as feature-rich as Azureus.  Also, Azureus has the ability to auto-update itself with bleeding edge CVS versions which come out usually once a day, sometimes even more than that!

### Post by user007usa on 2006-12-19
I have heard that ktorrent is much like utorrent.

But personally I use utorrent over WINE and am quite satisfied.

You may wish to consider there might be factors causing the speed difference.  For instance the amount of connections you allow both globally and per torrent can make a big difference.  As can the number of uploads (bt is "tit-for-tat").  I'd try to increase the connections a bit to see what it does if you have yet to try this already.  Of course if one torrent has two seeders and the other has 2,000 seeders, you are probably going to get much better performance on the one with 2,000 seeders too.  I only say this in case someone reading this is new to bittorrent.
