### Post by Ek0nomik on 2007-07-26
If you installed Ubuntu, which uses the Gnome desktop environment, you can still use any KDE application you'd like.  However, you will have to install the KDE dependencies in order to run those applications, which does start to take up a little bit of hard drive space.

You can also do the same vice versa.

The major difference between KDE and Gnome is really the interface.  KDE is, for lack of a better word, a little more spiffy.  I personally prefer Gnome.  It has a more minimalistic feel to it.

### Post by rillip on 2007-07-27
There is one other downside to running kde apps in gnome, or vice versa; more libraries have to be loaded to main memory, as kde apps are built on a gui suite called Qt, while Gnome apps are built on GTK,  So it has to load all the Qt libraries into memory to run them, which a) takes up memory b) may make the applications and the system overall perform slower c) takes up disk space

However, if you're not running a super low-resource build, you shouldn't see much of a difference, except maybe launch time.
