### Post by siciliancasanova on 2007-12-01

### Post by LinuxIsInnovation on 2007-12-27
Ofcourse you can switch away from Microsoft completely.. I was using Vista Ultimate and now I am 100% ubuntu. I backed up my data to an external HDD of mine (you can shift it to other partitions)
Then I formatted my HDD and created an ext3 partition on the primary partition which had my Windows. To install Linux, you have to completely remove all data on one partition. To completely remove Windows, you have to format the primary drive (C -drv). Otherwise, you can also dual-boot Linux with Windows.
