### Post by db9s on 2007-02-01
your last line of xorg.conf should be
```

Section "Extensions"
   Option       "Composite" "Disable"
EndSection

```

### Post by tyler_roach on 2007-02-01
If your video card is ATI, you should also install the &#8220;x11proto-xf86dri-dev&#8221; package (in addition to disabling "Composite"). You can search for and install it from synaptic.
