### Post by zvacet on 2007-04-15
```
whereis file_name ls
```

This command will tell you in wich directories and folders your games are.Most of them goes to the usr/bin but can be other places when some program is installed.To add them to the app menu go to the system>settings>menu layer>games>new item
1.name =name of the game
2.command =path to the game(as I sad commn id usr/bin/Your_game but can be something else)
3.icon = usr/share/pixmaps
