Not sure how much help this will be, but I believe that if you have samba up and running correctly, you can add a shared folder by.

Clicking on SYSTEM > ADMINISTRATION > SHARED FOLDERS

It will ask for your password, and then allow you to add directories to be shared, there is also a help button on that screen for a more in depth explanation. 

Hope this helped.
