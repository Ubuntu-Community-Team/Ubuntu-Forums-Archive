### Post by Inxsible on 2007-05-18
Yes, Live CD is going to give you problems at RAM below 512 MB. If you do want to install Ubuntu on your machine, you can go in for an Alternate CD which runs better on older machines with lower configurations.
 
Although, I dont think you will be able to do much on it, if you are going to keep the XP and dual boot. 3.3 GB is not a lot of space these days !
