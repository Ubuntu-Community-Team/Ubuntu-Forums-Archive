### Post by benfindlay on 2008-01-31
If vista and XP are already installed on hard drives and they are plugged in, installing ubuntu after should allow grub (the boot manager) te determine that vista and XP are already there, so you shouldn't have any problems! What you will get is a boot prompt asking you which OS you want to use when you turn the PC on.

hope this helps!
