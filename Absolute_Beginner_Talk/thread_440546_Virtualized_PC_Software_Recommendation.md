A vmware virtual machine is just defined by a text file.  You can edit it yourself.  You can copy an old machine, hard disk and all, and tweak the settings, and you have a new machine. Install any OS you want into it (just boot from the CD or DVD and install like on any real machine.)  

There are also 3rd party programs, and even web pages, to design machines.

Any OS installed in a VM can print, just as it would from a real machine, either to a printer plugged into a USB port, or to a network printer, etc.

I tried out virtual box, but only on XP at work.  It was not as stable as vmware.  It seemed to have troubles.  It used to use MS's virtual PC too, but found vmware to be better all round.

Tastes vary of course :)
