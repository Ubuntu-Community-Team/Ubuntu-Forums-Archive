### Post by lisati on 2007-08-02
You might find "DeVeDe" useful. You can install it through "Add/Remove programms" on the Applications menu. Type "devede" (without the quotes) in the search box, and set "Show" to "All available applications".

The catch is that it doesn't write directly to disk, but you can right-click on the ISO file it produces and select "burn to disk"
