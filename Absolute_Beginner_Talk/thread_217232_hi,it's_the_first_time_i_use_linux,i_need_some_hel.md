### Post by drtvasudevan on 2006-07-17
i know perhaps much less but...
;) 
one invokes what is called synaptic manager.system>>admin>> and synaptic manager.
give the password.
hit the search button.
type in the application you want.
need not be the name of the application. like search for "browser"
will give you all the browsers available.
select the one you want and mark it for installation.
having completed the list you click onthe apply button.
the s/w is downloaded and installed.
make sure there are no proxy issues as the update and the s/w installation may not work if there is proxy issue. but you can browse the net still.

if you have some .deb you just double click on it and it gets installed but it may fail as it can not resolve dependency issues. ergo better from synaptic where that will be automatically taken care of.
