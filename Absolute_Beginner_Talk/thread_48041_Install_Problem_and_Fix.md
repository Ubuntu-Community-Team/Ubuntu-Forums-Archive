### Post by fussyoldfart on 2005-07-10
The live version of Ubuntu worked so well that I wanted to do an install. I already had another Linux version dual booting with Windows and there was even an empty partition formatted for linux so I figured this would be a cinch. Wrong!

I struggled for 2 days (off and on) but could never get past the partition setup. I always ended in a loop where I would click on "Partition Setup Complete" (or whatever it says) and be taken back to an earlier step.

Finally, following a hint I saw on a forum that Google turned up, I scrubbed that last partition completely and let the Installation go to the largest unallocated part of the disk.

That worked perfectly. I guess I was being a little too clever. Obviously I know just enough to be dangerous.
