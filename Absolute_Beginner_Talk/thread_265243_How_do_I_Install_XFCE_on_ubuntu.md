### Post by waldorf on 2006-09-25
I think the best method I know of is to

sudo aptitude update
sudo aptitude install xubuntu-desktop

if you don't like it you can always uninstall.

Also, you will end up installing all sorts of other apps in the process so be prepared for your menus to get bigger
