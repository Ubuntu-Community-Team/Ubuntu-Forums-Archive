### Post by kpkeerthi on 2008-02-06
1. Enable "nvidia" driver
2. Restart X
3. Stop GDM ```
sudo /etc/init.d/gdm stop
```
4. Reset resolution ```
sudo dpkg-reconfigure -phigh xserver-xorg 
```

Check if you are able to choose the resolution(s) you prefer.
