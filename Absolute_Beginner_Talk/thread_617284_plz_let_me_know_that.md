### Post by kleo skywalker on 2007-11-19
Do you mean Ubuntu 7.04?
Totem (as Movie Player in the Applications-->Sound and Video menu) is a media player installed by default.
If you want a music player, Rhythmbox is the default one - in the same menu.
The first time you try to play restricted file formats such as mp3, the player will prompt you to install some codecs - that's stuff needed to play restricted formats. Say yes to the prompt.
In case you don't like these players, there are other options such as Exaile for music, and VLC for other media.
