### Post by alwiap on 2007-10-30
the compiz settings manager is at 'System > Preferences > Advanced Desktop Effects Settings', or you could press alt+F2 and type 'ccsm' and it will bring the same thing up.

a great site that helped me set up compiz without ANY problems is [http://forlong.blogage.de/article/2007/8/29/How-to-set-up-Compiz-Fusion]("http://forlong.blogage.de/article/2007/8/29/How-to-set-up-Compiz-Fusion")

this link will show you how to easily set up and customize desktop cube, expo, etc.

enjoy!
