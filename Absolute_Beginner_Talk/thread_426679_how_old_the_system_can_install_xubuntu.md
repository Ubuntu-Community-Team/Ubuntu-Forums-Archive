### Post by stoneheart on 2007-05-31
I got Xubuntu 6.06 to install using the alternate text version on an old Dell Optiplex GX1 (Pentium II 550 mHz, 128 megs ram, 6 gig hard drive).  I loaded Automatix2 and the system is very usable, although Firefox can be slow if you have too many windows and tabs open.  The computer seems to be using all the ram all the time and it's using another 128 megs of swap space consistently, so I'll grab another 256 megs ram off Ebay and see how it goes.

I have a much more powerful PC that I run Windows XP on, but I'm enjoying Xubuntu very much.  If I can get the pptp VPN configuration set correctly, so I can connect to my office network, I won't ever have a reason to run Windows again other than games, and I do little of that these days.
