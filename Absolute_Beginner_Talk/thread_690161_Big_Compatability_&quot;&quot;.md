### Post by Tatty on 2008-02-07
Which applications do you use?  There are lots of options depending on your needs.

You might want to install WINE - it is available in the repositiories.  It is a compatibility layer which attempts to allow windows software to run in linux systems.

WINE is far from perfect, however, so dont expect it to always work.  To to [http://www.winehq.org]("http://www.winehq.org") to read more information on it.  It also has a user-maintained application database which can give you some clues as to how to get certain bits of software working in wine.

Alternatively you could install Windows as a virtual machine  on your computer, essentially meaning that you can run it "under" linux.  Check out this documentation page to install it [https://help.ubuntu.com/community/WindowsXPUnderQemuHowTo?highlight=%28QEMU%29]("https://help.ubuntu.com/community/WindowsXPUnderQemuHowTo?highlight=%28QEMU%29")

Of course, dual booting with a windows partition is the simplest solution to the problem.  There is nothing wrong with keeping two OS's on one machine, as long as it works for you.

### Post by Rhubarb on 2008-02-07
That really depends what software you'll be using, and if that software will work with ubuntu.
If not, there may be alternatives to the software you want to use.

A good example is Open Office, that can open up and save most of what ms office can open and save.  Hence Open Office would be a good alternative to ms office.

What software do you intend on using?
What file formats will you be working with?

You may not have to dual boot at all, you can run windows in a virtual machine.
In this case, you have to check whether you're legally allowed to run your copy of windows in a virtual machine, and you'd have to make sure you have plenty of RAM in your computer to support running two operating systems at once.
