### Post by ali4949 on 2007-04-21
I have a P4 -M thinkpad  T30 1.6GHZ laptop that I have been running edgy for the last few months. This was one of the earlier if not the first T30 model. With 768 MB of ram on the system I have rarely if ever used the swap.. I also experimented with all the applications that I would use running at the same time and still it didnt use the swap.. this included GIMP, Firefox with 8 tabs, movie player, gaim, bluefish html editor open office, and a terminal running conky.

Seems like CPU  speed is not as big a factor in performance in ubuntu as much as RAM is. Linux really uses the system ram to take full advantage to make the system responsive. My experience so far has led me to believe that if one is running one of the newer versions of ubuntu, anything over 900 Mhz would do just fine, and to really speed the system responsiveness, max out the ram if possible.. I am in the process of upgrading to 1.5GB of ram on my thinkpad and then its really going to be blazing fast.
