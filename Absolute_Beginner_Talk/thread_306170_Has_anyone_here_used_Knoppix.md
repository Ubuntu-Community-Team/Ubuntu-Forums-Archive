Knoppix is a nice distro, mainly intented to Run as a LiveCD, though there is an option to install it.

If you would like trying other Distro's I would suggest SuSE, Mepis and the one that tickles the last year, but never was determined enough, Gentoo. These pretty much cover a wide aspect of approaching a Linux system.

I 've used SuSE for about 8 months, the only thing that I still recall, and somewhat found distinctive was Yast.

Mepis I hear is to Debian what Madriva is to Red Hat, an easy to use distro based on "it".

Gentoo now, is an absolutely amazing idea, everything comiled from source, and instead of packages there are scripts, that "tell" emerge how to compile, it's all about freedom of choice, I really like the concept, never  got deterined to try it though.
