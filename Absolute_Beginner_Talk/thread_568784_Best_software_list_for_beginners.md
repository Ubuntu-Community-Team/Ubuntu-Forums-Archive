### Post by nick_h on 2007-10-06
On the menu Applications->Add/Remove will list applications by type and popularity.

The [Linux App Finder](http://linuxappfinder.com/) site is also worth a look.

When you find an application you can always use:
```
aptitude search <name>
``` to try to find the application within the repositories.
