Three use different desktop environments.  Ubuntu uses [GNOME]("http://www.gnome.org/"), Kubuntu uses [KDE]("http://www.kde.org/"), Xubuntu uses [Xfce]("http://www.xfce.org/").  It's really a matter of preference.  Edubuntu is, as the name suggests, tailored for use in education.

### Post by atria on 2007-07-07
> **dhani99 said:**
> what should i install ubuntu,kubuntu,edubunt,xubuntu??
why is there 4 different types??

Ubuntu uses the Gnome environment
Kubuntu uses the KDE environment
Xubuntu uses the XFCE environment

Screenshots of the different environments can be found anywhere on the web (search google.com) 
There is no "best" environment per se, since it all boils down to personal preference.

If you're using a slow computer, you might want to check out xubuntu since xfce is a lightweight environment.

However if you need general usability and compatibility, perhaps try Ubuntu.

Hope that helps

//edit
What is Edubuntu?

Edubuntu is a free/libre/ open source GNU/Linux operating system. And if that didn't make any sense to you, we hope that it soon will. In simpler terms, Edubuntu is a free software system designed for school environments, and should be equally suitable for kids to use at home.

[http://www.edubuntu.org/FAQ](http://www.edubuntu.org/FAQ)
