well a big part of it has been used and can be re used by a different program if needed
you might have noticed that if you open a program for the second time it goes a lot faster, thats because it is saved in cache.
if you want to know precisley what you are using you can try the following command

free -m

below is mine
landan@ubuntu:~/Desktop$ free -m
             total       used       free     shared    buffers     cached
Mem:          3364       3251        113          0         43       2779
-/+ buffers/cache:        428       2936
Swap:         1937         34       1903

the fist line says that a total of 3251MB of RAM is filled
but the 428MB on the second line is actually in use at the moment, the difference is saved up in buffers and cache in case the same or any other program likes to use it
