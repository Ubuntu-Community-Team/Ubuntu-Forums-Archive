### Post by hollaburoo on 2006-07-01
put the flock folder somewhere and then make a link to it using alacarte (applications->accessories->alacarte manu editor) just click on internet on the left and then go to file and select new entry, just make the command a link to flock and set the icon.

### Post by bruenig on 2006-08-19
the simplest way to do it is to go to applications>accessories>alacarte clicking on internet then going to file new entry. Type in all the appropriate stuff, for the command just use the path to the flock executable and for the icon, just navigate to the icons folder inside the flock directory and select it.
