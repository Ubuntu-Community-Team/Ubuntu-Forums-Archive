### Post by bludhound on 2006-07-15
Simply enable 'GConf' under 'System Tools' in your Alacarte Menu Editor. Now run GConf, and navigate to 'desktop/gnome/volume-manager'. The appropriate setting should be displayed in the right-hand pane.
