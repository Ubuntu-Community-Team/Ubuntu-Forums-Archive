### Post by oliviacond on 2008-03-01
Information from devede [website]("http://www.rastersoft.com/programas/devede.html")
this is because of the buggy version of Mplayer.
Solve this by enabling the backport repo to install better version of Mplayer
To install it, just go to System->Admin->Software repositories and activate the Backports source.
