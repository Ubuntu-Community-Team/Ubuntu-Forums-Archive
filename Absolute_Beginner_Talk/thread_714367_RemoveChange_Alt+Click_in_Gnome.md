### Post by glennric on 2008-03-03
Are you using compiz (Desktop Effects) or not?  If you are using compiz open System->Preferences->Advanced Desktop Effects Settings
and change the Inititiate Window Move option under the Move Window plugin.  If you don't see the Advanced Desktop Effects Settings in the menu you will have to install compizconfig-settings-manager to get it.
