### Post by SkylineGT-R on 2007-12-21
I tried to install Ubuntu on my external USB hard drive (WD Passport 120GB).  After I installed it, I unplug the hard drive to boot Windows Vista.  It doesn't work though, an error comes up and says "Grub loading stage 1.5  .....  loading grub please wait  .....  Error 21".  I reboot and plug in the external HD and in the boot selection menu I select "boot from USB device".  That doesn't work either it says "Missing Operating System".   So then with the external HD still plugged in I go to the boot selection menu again and select "internal hdd".  This time I can actually go somewhere!  It gives me a choice of  plain ubuntu, ububtu recovery, and Windows Vista.  After I select Ubuntu or Vista everything boots fine.  
So basically to be able to boot anything I need the external hdd plugged in and then I need to select "boot internal HDD"
How can I uninstall Ubuntu and have vista to boot normal again?  I dont care about my ubuntu installation, I just need my Vista working again.

### Post by Duck2006 on 2007-12-21
Use your vista cd to repair thembr of your vista drive and you will be able to boot that one. With the external USB hard drive you will have to install grub on it's mbr for it to boot when at boot time you plug it in to boot from.
