### Post by PurposeOfReason on 2007-08-07
I'd guess that it just gave you a bad resolution to start with. When I used the live CD first I could barely see either panel. Just change the resolution and you should be good to go.

System - Preferences - Screen Resolution

### Post by mcduck on 2007-08-07
If you hold the Alt key you can move windows around from any point in the window, not just the title bar. So if you can't get higher resolution during the install you can at least move the window aroud the screen to access all butons etc.
