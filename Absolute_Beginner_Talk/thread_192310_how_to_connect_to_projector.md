### Post by sitedesign on 2006-07-06
You should have on the laptop a function key (the F1 to F12) at the top of your keyboard with an empty square box or some representation of a screen. There should also be an Fn key on your keyboard (normally bottom left).

Plug the VGA cable into your laptop going to the projector.

Then when in Ubuntu desktop (graphical windows screen) hold down the Fn key and press the Function key with the screen on it.

Repeating this key sequence should allow you to swap between:
laptop screen
projector
laptop and projector
back to laptop.

Hope that helps.
