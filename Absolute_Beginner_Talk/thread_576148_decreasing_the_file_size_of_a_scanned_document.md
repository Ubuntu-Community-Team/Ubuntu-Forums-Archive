### Post by wormser on 2007-10-14
You can edit the .jpg with Gimp.  Applications> Graphics> Gimp.  Open the file> Image> Scale Image.  Then decrease the size.  You will have to save the file and then check the file size.

### Post by entwisi on 2007-10-15
It sounds like you've scanned it at some silly high dpi to start with.

most scanning only needs to be at 100dpi, only go higher if you really have a good reason.

Then save it as JPEG setting quality to about 80% and you should be OK
