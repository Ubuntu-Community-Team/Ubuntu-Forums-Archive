Yes that is Fluxbox window manager.

[gkrellm]("http://members.dslextreme.com/users/billw/gkrellm/gkrellm.html") is the system monitor.

For a transparent terminal: several terminals offer that. Examples: eterm, aterm, xfce4-terminal, gnome-terminal. Some terminals also allow you to remove window decorations (such as titlebar, scrollbars, borders). xfce4-terminal definitely allows this; I don't know about the others because I haven't played with them too much and these days I use a plain xterm.
