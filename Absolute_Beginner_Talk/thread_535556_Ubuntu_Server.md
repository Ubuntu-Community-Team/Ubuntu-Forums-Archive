### Post by bdoe on 2007-08-26
> Is Ubuntu Server the normal version of Ubuntu with features that would be good for servers? Because if it is the normal version with more features, I want to use it as my main OS.
Not exactly.  For starters, the kernels are different (Server Edition's kernel is tuned for multi-client environments) - or, at least, they're supposed to be.  So you may find that multitasking on a server isn't as snappy as it is on a desktop.  However, with that said, I haven't really noticed any difference between the two.

Server Edition lacks a GUI such as Gnome or KDE.  Once it's done installing, you will be presented with nothing more than a CLI.  You can, however, install Gnome Desktop on it (as I've done on mine).  The Server Edition distro is tuned for headless server deployment.

You would be better off installing the desktop edition.  If you decide you need server functionality on it, you can always install the specific apps you need.

So, the short answer is, the two editions differ by what's packaged in each distro.  The server edition would not make a good desktop OS without modification.
