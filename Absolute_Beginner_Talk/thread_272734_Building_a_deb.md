### Post by JayTee on 2006-10-06
I wouldn't go to all the trouble of making a deb. I'd just get Automatix which will install the latest version of JRE and the Firefox plugin for it "automagically". It will also allow you to install many other packages that normally require alot of tweaking and do all the work for you instead. I've used it for JRE and w32codecs as well as several other programs. It's a great graphical install tool that covers most of "essential programs" people want that are harder to install and configure or aren't available in the default repositories. It'll temporarily modify your sources list file and then set it back to the default when it's done. Give it a try, it's well worth it.
