### Post by anaconda on 2006-08-10
You can install ubuntu to an USB-hard-drive, and it works from there if your BIOS supports booting from USB-drives. (most, if not all new PC:s support this)

But I don't know if (USB-) ubuntu would work in a different machine than where it was installed.. because of the differences in hadware. I dont think it would work. 

Some linux distributions would work, eg. DamnSmallLinux and Knoppix, which both can be installed so that each time they boot they go through the harware detection..

Just a heads up (FYI) in case you decide to ditch the OS on the external drive at some point - you'll need to write zeros to the drive before reformatting, otherwise your disk may fail (this happened to me when I did installed/removed Linux from my external drive).
