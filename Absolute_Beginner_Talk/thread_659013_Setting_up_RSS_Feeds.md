### Post by kestrel1 on 2008-01-05
If you want to add this to you Bookmarks Toolbar, try this:
Browse to: [http://ubuntuforums.org/external.php](http://ubuntuforums.org/external.php)
When the page loads you have the option to subscribe, click the button & you get the option to add this to your Bookmarks Toolbar.
