### Post by Najand on 2007-05-10
OK... First of all you need to open a terminal:
Applications -> Accessories -> Terminal
Then you need to type commands:
First of all: I have already written the way how to backup your xorg.conf
```

sudo cp /etc/X11/xorg.conf /etc/X11/xorg_before_trackball_mouse.conf

```
Then: editing xorg.conf:
```

gksu gedit /etc/X11/xorg.conf

```
Then edit it as I wrote above, after you are done, save it.
Then Restart X using Ctrl+Alt+BackSpace
Then login and try to see if it works.
