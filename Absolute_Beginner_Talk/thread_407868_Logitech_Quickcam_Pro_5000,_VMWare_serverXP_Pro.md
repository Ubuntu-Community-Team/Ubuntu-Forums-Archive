### Post by cubbs80 on 2007-04-19
I figured out how to correct the no usb devices error shortly after I posted.  You have to make sure VMWare has a USB device turned on for the virtual machine, otherwise it won't see the USB controller.

Once I did that, I was able to get the camera installed.  The problem was that it'd only work for about a minute or less and then crash the XP virtual machine.  I haven't figured that one out yet.
