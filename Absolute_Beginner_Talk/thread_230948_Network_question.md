### Post by HereInOz on 2006-08-06
You will need to have the same username and password on both machines.  You probably have, but I just thought I would mention it.  In other words, if you have a username of fred, password of jones, as the permitted user on the XP machine, you will need to be logged on as fred, with a password of jones on the ubuntu machine.

Also, is this XP Pro?  If so, it may be worth turning off "Simple File Sharing", and setting up the security.   You find that by opening a folder, selecting View from the menu, then Options.  It is right down the bottom.
