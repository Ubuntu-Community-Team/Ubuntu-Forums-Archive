well, there are several ways of going about this depending on the situation.

most programs you will need reside in the ubuntu repositories (ie: you don't have to download them individually).  there are a few exceptions to this, but i'd suggest waiting until you're a bit more familiar with the way things work before you try that kind of setup.

if you want to use terminal, usually all you have to do is:
```
sudo aptitude install programname
```
