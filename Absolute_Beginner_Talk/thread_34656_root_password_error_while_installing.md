### Post by dave9191 on 2005-05-15
By default the ubuntu install does not set a root password. It only creates one account with a password and you can execute root command using the sudo command and your own password. 

[http://www.ubuntulinux.org/wiki/RootSudo/view?searchterm=sudo](http://www.ubuntulinux.org/wiki/RootSudo/view?searchterm=sudo)
