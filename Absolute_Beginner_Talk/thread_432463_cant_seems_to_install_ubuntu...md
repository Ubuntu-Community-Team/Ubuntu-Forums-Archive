### Post by seshomaru samma on 2007-05-04
look[ here]("http://www.ubuntu.com/getubuntu/downloadmirrors")
it will ask you to choose a place near you
I chose Asia , but you might choose something else
next you will see a screen like [this]("http://mirror.rootguide.org/ubuntu-releases/")
choose the 3rd option (7.04 -Feisty) and you will get a screen like [this]("http://mirror.rootguide.org/ubuntu-releases/feisty/")
you can then choose between a live Cd and an alternate Cd
if you scroll down you get download the MD5Sum
[how to check md5sum]("http://www.psychocats.net/ubuntu/iso#checksum")
you should burn the CD at the lowest speed possible
you can check the CD for errors when it boot
you should set your BIOS to boot from CD/DVD
[how to install from LiveCD]("http://www.psychocats.net/ubuntu/installing")
[how to install from alternate CD]("http://users.bigpond.net.au/hermanzone/") (useful if you want to dual boot)
