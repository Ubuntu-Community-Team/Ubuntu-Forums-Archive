### Post by n00bWillingToLearn on 2006-06-04
Basically automatix does more but in ways that are considered 'bad' and has been known to seriously mess up peoples Ubuntu installations to the point that they have to wipe it and reinstall. But those problems are supposedly only from running automatix on a machine that you have already set up / changed and automatix 'should' be fine if you use it on a clean install.

Easyubuntu is completely safe. It installs everything in standard ways, like apt-get, but it does not have nearly as many features as automatix.

I use Easyubuntu and I may be a little biased against Automatix as it hosed my install when I tried to use it so you might want to also listen to someone more neutral.
