### Post by jvc26 on 2007-02-19
Install it from the repositories.
[Beryl install]("http://http://ubuntuforums.org/showthread.php?t=263851")
Once there it will update itself to the latest available (not the svn version though)

Please note, Beryl is Offtopic in this forum. See [here]("http://ubuntuforums.org/showthread.php?t=349732")

To find out your version type into terminal:
```

beryl -version 

```
That will output the version number: The latest is beryl-core 0.1.9999.2
