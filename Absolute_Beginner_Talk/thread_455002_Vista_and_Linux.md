### Post by ugm6hr on 2007-05-26
Any combination will work - Vista / XP / Linux/ even all 3.

If you've already got Vista on your primary drive, use EasyBCD to edit the bootloader to access the other OS.
You should be able to just install the XP drive into your new computer - I think it will merely go back to default drivers, so you might have to install drivers for your new hardware afterwards.
As for Linux - just put the HD in, format it, and then install (**without GRUB**).

Easy BCD will then allow you to access the non-Vista OS.

This is not difficult, but it depends on your level of experience as to whether it's reasonable for you to just experiment without potentially losing all your data.

### Post by Alterax on 2007-05-27
My own experiences with dealing with MS on moving XP installations has been pretty straightforward (about 3 a month at work).  Do the install, give it the key, it'll install and give you 30 days if it doesn't automatically activate to get it together (unless you're adding it to a domain in which case you have to immediately).

Just call the 800 number that pops up if it can't automatically do it.  Call them and you'll get someone in India that asks you for the number on the screen.  There'll be a brief bit of small talk where they ask you how many machines it is on (you say one and you are re-installing it) and they say okay and give you back a number to type into the boxes.  No names, no obscure instructions.

Pretty easy; don't let the phone intimidate you.
