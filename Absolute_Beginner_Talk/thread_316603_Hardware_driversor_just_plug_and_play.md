### Post by Delkster on 2006-12-11
Most of the time hardware is just plug-and-play in Linux. If there are free/open-source drivers, they're generally included with the system.

If you have to install drivers, things are probably a lot worse anyway. Installing third-party drivers is a hassle and at least things tend to go through some kind of a quality check before being accepted to the official Linux source code tree, unlike some random third-party ones.

Personally, I avoid hardware that doesn't have free/open-source drivers for it.
