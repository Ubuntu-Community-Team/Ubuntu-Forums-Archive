### Post by yatt on 2006-04-26

Debian is a very old Linux distro. Sid is their "unstable" version, which is quite stable compared to other distros, but Debian has impossibly high stability standards. Sid is named after the sociopathic kid from Toy Story. 

Ubuntu is not as stable as Debian is. Ubuntu has a short release cycle and uses alot of new programs. Debian has a long release cycle (8 release over the last 10 years, 3 in the last 5) and doesn't release new software until they are sure it  is as stable as it is going to get.
