### Post by PmDematagoda on 2007-11-20
The ability to upgrade a previous Ubuntu installation is only with the Alternate CD, for now, but that ability may be given to the Live CD as well in the future. You cannot upgrade a previous Ubuntu installation using the Live CD or using the packages in it.
