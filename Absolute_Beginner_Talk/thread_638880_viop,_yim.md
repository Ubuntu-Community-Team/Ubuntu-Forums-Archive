### Post by jken146 on 2007-12-12
Pidgin handles tons of IM protocols.  As for skype, it's not in the usual repos, but it is in medibuntu.  [www.medibuntu.org](www.medibuntu.org)  Follow the instructions for enabling their repository, then you can install skype. ```
sudo apt-get install skype
```
