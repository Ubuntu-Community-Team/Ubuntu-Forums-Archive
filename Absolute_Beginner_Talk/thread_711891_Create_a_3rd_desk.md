### Post by Samhain13 on 2008-03-01
If you're using Gnome, right click on the Workspace Switcher applet (it should be in one of your panels). Choose Preferences and from there, add and remove workspaces via Columns and/or Rows.

I believe it works pretty much the same when you're in KDE.
