### Post by Ek0nomik on 2007-06-07
Download Ubuntu and burn it to a disc.

Put in the CD-ROM, and boot to CD.  You can reformat your other hard drive into 2 partitions.  One being EXT3, and the other being Swap (make Swap about x1.5-x2 the amount of RAM you have).

Then, let the disc do it's thing (install), and you should have an option on every boot to allow you to either boot into Windows or Ubuntu.

### Post by w4ett on 2007-06-07
After you boot the Live CD and select install from the desktop, you can select the option to install Ubuntu on your second drive.  The Grub Loader will do the rest.  This is a very common configuration for Dual Boot
