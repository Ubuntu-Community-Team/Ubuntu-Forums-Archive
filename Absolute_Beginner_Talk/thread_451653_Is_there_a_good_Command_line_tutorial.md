### Post by Sparkster185 on 2007-05-22
I don't think you can install codecs with the live CD, but you might be able to.  If you do, however, they will be lost when you reboot the machine.

I used Automatix to install the codecs and I'm not sure if you can install/run that while using the LiveCD.

To learn more about the command line, do some research for an introduction to the Bash Shell.  This is a good beginner reference:  [http://tldp.org/HOWTO/Bash-Prog-Intro-HOWTO.html](http://tldp.org/HOWTO/Bash-Prog-Intro-HOWTO.html)
