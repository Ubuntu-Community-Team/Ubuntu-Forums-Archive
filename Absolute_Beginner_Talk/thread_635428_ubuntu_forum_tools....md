### Post by jordanmthomas on 2007-12-08
You can subscribe to the thread.
Thread Tools --> Subscribe to This Thread

Then, new posts will show up in your user control panel (and be e-mailed to you if you wish)

The only extension I could find is this one, but it doesn't offer the feature you're looking for I don't think.
