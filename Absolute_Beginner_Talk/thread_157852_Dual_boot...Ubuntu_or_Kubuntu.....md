The core is the exact same. The desktop enviroment is the only thing that is different. You can actually install both and just choose the one you want to log into at the log in screen. You do not need to dual boot at all.

To do this just install Ubuntu and then type this in the terminal.

```
sudo apt-get install kubuntu-desktop
```
