### Post by ellgor on 2008-03-23
Hi,

Just came across your post as I was lookuing for a similar fix for my mic, I see you have the alsamixer (albeit a bit dodgy) so I would suggest you uninstall it and then reinstall it. When you have redone it open a terminal and type in alsamixer (and press enter) a rather basic window will open with vertical bars representing the differerent functions should appear, use the arrow keys to move across the different functions until you see the Mic highlighted, if says off press m to toggle it off and on, also if the actual voume is very low you can adjust this with arrow keys (up and down) and hopefully thats it, hope this helps.

Regards, Ellgor
