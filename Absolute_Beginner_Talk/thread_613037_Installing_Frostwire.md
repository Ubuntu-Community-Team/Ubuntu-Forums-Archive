You can install frostwire, but it will only be in the memory (RAM) not on your hard disk. The Live CD does NOT write anything to disc until you install.

Also there is no concept of D:/ drive in Linux. and you cannot have linux programs installed on windows partitions anyway.

Read up more about partitioning [here]("http://ubuntuforums.org/www.psychocats.net/ubuntu/partitioning").
