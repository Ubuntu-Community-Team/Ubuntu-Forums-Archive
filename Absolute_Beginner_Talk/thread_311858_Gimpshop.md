### Post by Dale61 on 2007-04-03
I have GIMP set-up so as all the different windows open as one.

You drag each tab of every dock into your main tool window, and everything you need is now in a single window.

However, if some windows are rarely needed, you can just open each one as you need it, and thus, reducing the size of the main tool window.
