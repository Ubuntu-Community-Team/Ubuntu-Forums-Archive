### Post by confused57 on 2006-05-24
First of all, I recommend that you "test-run" a LiveCD on your computer before attempting an install, any problems you have running the LiveCD, you'd more than likely have the same problems when installing.  In fact, It's probably a good idea to use the LiveCD to familiarize yourself with the OS.

When you're comfortable with the LiveCD, then you can use the link provied earlier for setting up a dual boot(Herman's site) with WinXP.

Before starting your Ubuntu install, be sure to defrag your WinXP at least once, maybe twice.  The Ubuntu installer will offer to resize  Windows, you just select what size you want Windows, Ubuntu will pretty much install itself without having to use any other partitioning tool.  Be sure to check out Herman's guide, it covers dualbooting Breezy & Windows & has some pretty good screenshots.

You may want to install Dapper(after running the Dapper LiveCD, which has the option to install also), final release version is due out June 1.
Check out this link for installing Dapper:

             [http://psychocats.net/ubuntu/windowstoubuntu.html](http://psychocats.net/ubuntu/windowstoubuntu.html)
