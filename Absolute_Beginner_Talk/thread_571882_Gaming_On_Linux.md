### Post by Hospadar on 2007-10-09
You can always try running them through wine, a lot of new games work well with it (A lot of half-life 2 based steam games for example) and if that doesn't work you can run a virtual windows machine and run the games there, though this option will degrade game performance.

Also, there are quite a few good games written natively for linux.  Have a look at nexiuz or vegastrike.

In general though, extremely few commercial games are released natively for linux.  but between wine and a VM you can generally get almost any windows game working, albeit with a lot more work than it would take to run them on windows. You'll probably find yourself wanting to dual boot if you have a lot of windows games you like to play (especially brand new ones)

### Post by barkej on 2007-10-27
