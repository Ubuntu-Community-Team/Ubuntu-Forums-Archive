### Post by Murrquan on 2007-08-14
[Here]("http://linux-sxs.org/hardware/clie.html") is a link that might help!

I know you want cut-and-paste instructions, so let's make this simple for you. The following is basically what they say there:

**Step One**

First, download the latest build of Pilot-Link. It should be [right here,](http://downloads.pilot-link.org/pilot-link-0.12.2.tar.gz) or you can get it from pilot-link.org. Extract it.

Now, go into terminal, and get into the directory you extracted it to. (If you don't know, check Firefox's settings and see where it's putting your downloads.) Use the command "cd (directory)" to switch to a directory, and "dir" to see the directory's contents if you get lost. You can use the file manager to see where you're going.

When you get there, type "./configure --prefix=/usr". You'll see a lot of text scrolling.

Then type "make". More text will scroll. Wait for it to stop scrolling.

Then type "sudo make install". That should do it. Don't worry if you get errors, I got a few and it still seemed to work.

**Step Two**

You need the C compilers to get this thing to run, I think. Fortunately, that's not too tough. In terminal, type the following two commands:

"sudo apt-get install linux-kernel-headers", and
"sudo apt-get install build-essential".

Remember to type all these commands without the quotation marks!

**Step Three**

Go to Add/Remove Programs, and do a search for JPilot. That is the program you will use to manage your Palm files. It should work just like Palm Desktop. Install it.

**Step Four**

Follow the instructions given [here](https://help.ubuntu.com/community/PalmDeviceSetup). Ignore what they say about Breezy Badger, but pay attention to step 5 about Feisty Faun -- that's the version of Ubuntu that you are using.

**Step Five**

Go into Preferences -> PalmOS Devices, and set up your Clie. You want to tell it to look for a connection via USB, not Serial, and to use /usb/ttyUSB1.

**Step Six**

Plug in your Clie. When it tells you to do a test sync, run Hotsync on your Clie!

**Step Seven**

Almost there! Now go into Applications -> Office -> JPilot. Turn on your Clie (it's still plugged in, right?), and press the Hotsync button on it. Then press the Hotsync button on JPilot.

Congrats! You've gotten your Clie to work! ^.^ I just followed these steps myself. If it doesn't work, try restarting your computer. That worked for me. *nod*

Good luck!
