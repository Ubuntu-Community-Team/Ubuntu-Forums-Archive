### Post by Hospadar on 2007-11-12
If you play a lot of windows games and don't have an internet connection, I would highly suggest that you at least dual boot windows.  While there are many good options for running windows games (vmware, wine) without an internet connection to be reffering to help and downloading software and whatnot, these would be quite a pain to set up and I think you would be better served by a windows installation.

That said, I use linux all the time for almost everything I do, and there are quite a few games that I do play on it, but many that just run better (or run at all) in windows.
