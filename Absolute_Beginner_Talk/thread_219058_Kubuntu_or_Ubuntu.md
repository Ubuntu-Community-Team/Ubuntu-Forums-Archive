### Post by johnkennethhughes on 2006-07-20
I completely agree with installing all three desktops and playing around with them to see which one you like best. However, further to earlier posts, I would recommend using Aptitude to install the desktop packages as it makes it easier to remove them again later if you don't want them:

sudo aptitude install ubuntu-desktop
sudo aptitude install kubuntu-desktop
sudo aptitude install xubuntu-desktop

As far as I understand, apt-get doesn't remember all the packages that make up each desktop package, so you would have to remove them individually if you wanted to remove the desktop (and there are a LOT of them to remove). Aptitude remembers what they all are and so you can remove them with:

sudo aptitude remove ubuntu-desktop
sudo aptitude remove kubuntu-desktop
sudo aptitude remove xubuntu-desktop

Hope that helps. Enjoy!
