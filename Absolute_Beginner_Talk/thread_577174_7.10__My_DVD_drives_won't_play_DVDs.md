### Post by skompier on 2007-10-16
> **MaxGunnar said:**
> how do I do that ?

Sorry...I assumed since you had to install VLC...you already knew.

System>Administration>Synaptic Package Manager

Click Search button, put libdvdcss2 in search box and click "Search"

It will list libdvdcss2. Click the square box. Choose "Mark for Installation"

Click "Apply". Synaptic will install the package for you. Do the same for libdvdread3.

BTW..if they have a green box next to them, then they're already installed.

### Post by RomeReactor on 2007-10-16
Libdvdcss2 is not available through the official repositories; you must add the [Medibuntu]("http://www.medibuntu.org/") repository, or download the package [from their site]("http://packages.medibuntu.org/pool/free/libd/libdvdcss/libdvdcss2_1.2.9-2medibuntu4_i386.deb"). You install .deb packages by double-clicking on them once they finish downloading. Perhaps the solution in my previous post can help you. To enter the commands, open a terminal (Applications->Accessories->Terminal).
