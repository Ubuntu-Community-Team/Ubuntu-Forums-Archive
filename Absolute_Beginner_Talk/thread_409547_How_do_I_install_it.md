### Post by NiklasV on 2007-04-14
You probably need to change your BIOS settings to make booting from a cd a higher priority than the hard drive. This depends on your computer, but I believe on Toshiba laptops, you should press either F1 or F2 when the system boots (before windows boots) and look for an option called boot priority or something similar. Change it so it will look for a bootable CD before it looks for an operating system on the hard-drive.
