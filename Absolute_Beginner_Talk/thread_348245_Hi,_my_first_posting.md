### Post by bernied on 2007-01-28
Hello and welcome.

You can have your ubuntu and keep your windows too. [Here](http://users.bigpond.net.au/hermanzone/) is a site dedicated to dual-booting - there's a lot to read (and it could do with updating, but the principles are still the same), but you say you want to learn...
Installing without a working CD drive is challenging, but not impossible. How challenging would it be to get one - even if it were a loan of a detachable usb drive?
You will need to be able to boot from a device that is not your hard-drive if you need to repartition your hard drive to make space for a new install. This can be done from a usb memory stick, at a pinch.

### Post by bscbrit on 2007-01-28
Hi aileenmac,

Welcome to the world of linux and, in particular, Ubuntu.  You can download Ubuntu easily but, without a DVD/CD writer, you will make things much more difficult for yourself than it need be.  Do you know why the writer is not working - is it something that someone on this forum can help with?

Secondly, it is fairly easy to set up your computer to run either Windows or Ubuntu - you can select which at boot-up time.  I suspect that a large number of users on this forum use exactly that set-up, although there will be plenty trying to convince you to make the full switch.  I have and I do not regret it but, if you enjoy playing games or have specific software requirements (i.e. usually tied to commercial software) then sticking with a dual boot system is probably the easiest way to begin to learn about *nix.

You appear to have the only essential qualifications to begin - enthusiasm and a desire to learn.  There will be frustrations ahead (linux is NOT windows - it doesn't always behave the way you have come to expect Windows programs to behave) but the learning process is well worth the effort.

Let me know if you think that we can help with the DVD/CD writer problems.
