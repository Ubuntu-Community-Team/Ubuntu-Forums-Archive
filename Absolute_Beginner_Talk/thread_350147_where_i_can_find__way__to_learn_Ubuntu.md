### Post by mrdeeno on 2007-01-31
I got a book called something like "Ubuntu for Beginners" or something like that.  It was slightly outdated because I was running Dapper at the time and the book included a Hoary CD.  

It was helpful...it explained some Linux basics, principles behind Linux and open source, a little history with GNU/Linux, differences between Linux and Windows (doesn't bash windows, it was relatively objective), the difference bewteen Ubuntu and other distros, and it gets into actually setting up and using Ubuntu, both the GUI and terminal.  And it gets into advanced topics like optimizing in the later chapters.

It probably would have taken me twice the time to learn all the things I learned without that book.  Before the book, I was confused going through a lot of the posts on here, but after the book, a lot of things were more clear, so to me it was a good base (you can't learn to read shakespeare without first learning to read).
