### Post by enofman on 2008-01-18
Smoke,

I'm no expert but I had some of the same problems. You have to download the ISO burner at [https://help.ubuntu.com/community/BurningIsoHowton](https://help.ubuntu.com/community/BurningIsoHowton) and then install the burner first and then download the Ubuntu version you need ie:, ubuntu-7.10-desktop-i386.iso.  When you select the fiile to burn make sure you change the speed to the slowest and change to track.  The default is maxium for write speed and session-at-once for the write method change those to 10x for write speed and track-at-once for write method. Everything else is default or at least that worked for me.  I also tried another burner and had problems then downloaded Infra Recorder and used the defaults which only saved the file as an .iso again.  Change the two options and it worked the first time.
