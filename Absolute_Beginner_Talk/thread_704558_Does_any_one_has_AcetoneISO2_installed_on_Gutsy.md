### Post by missiledave on 2008-02-27
it took me forever to figure this out as i am a linux noob it seems like all the sites hosting the .deb file were 404 or some other thing.  here is how i got my acetoneISO2 to run on my gutsy machine


download the .deb file to your desktop (or wherever its easy to find)

[http://sourceforge.net/project/downloading.php?groupname=acetoneiso2&filename=AcetoneISO2-1.0.2r-generic-src.tar.gz&use_mirror=heanet](http://sourceforge.net/project/downloading.php?groupname=acetoneiso2&filename=AcetoneISO2-1.0.2r-generic-src.tar.gz&use_mirror=heanet)

open a root terminal 

applications>system tools>root terminal

then simply drag and drop the .deb file into the terminal and it will start running.

let me know if something doesnt work.
