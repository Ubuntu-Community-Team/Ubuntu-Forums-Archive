### Post by kripkenstein on 2006-07-17
I use gedit for Latex. It has syntax highlighting built-in. Add a few keyboard shortcuts (to compile and to view), and it's a nice compact environment.

Alternatively, an even more compact editor is [amyedit]("http://amyedit.sourceforge.net/"). I would use it, but I have gedit open all the time anyways, and it allows for more shortcut customization.
