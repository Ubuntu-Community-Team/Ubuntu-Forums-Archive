### Post by Jucato on 2006-07-09
I presume you are using Kubuntu? Because if you are using Ubuntu (GNOME), you can't really use it.

KWeather is a Kicker (KDE's Panel application) applet. It's something like GNOME's Add to Panel options (Notification Area, Clock, Task list, etc). So it can only be used if you have Kicker, which means you're running KDE/Kubuntu.

If you are running Kubuntu, you can look for the KWeather applet/plugin from Adept. Once you have installed it, right-click on an empty space on the panel, choose Add Applet to Panel,  look for KWeather, then click on Add to Panel.
