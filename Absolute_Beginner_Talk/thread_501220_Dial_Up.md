### Post by Bartender on 2007-07-15
There are at least 3 hurdles to dial-up
#1: Winmodems, the cheap softmodems that pass some of the processing off on the CPU and rely on drivers - Windows drivers - to function.
#2: Linux dialer software, or more specifically peoples' unfamiliarity with Linux dialer software, which works pretty good once you figure it out.
#3: ISP's (like Netzero) that force the user to install proprietary software.  PeoplePC, Juno, and I think AOL are some examples.

I have no idea where you're at with the first two issues, but the last one is best fixed by finding an ISP that doesn't create artificial barriers.  550access, Fry's.com, copper.net, isp.com are some of the ones that have been mentioned as compatible with Linux.  

I'm using the old Juno 5.  Too complicated to explain why.  I can go online with my Linux PC a few times, then their servers send up an alarm and I get immediately cut off the next time I try.
