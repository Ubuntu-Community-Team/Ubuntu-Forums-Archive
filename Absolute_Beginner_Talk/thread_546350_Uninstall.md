### Post by El Rogueo on 2007-09-08
You don't uninstall an operating system, you delete it. Get a formatting utility, like GPartEd, and burn it to a live CD. If you boot up the computer with that in and format your hard drive(s) then you won't have ubuntu anymore.

You can also just install another operating system over it, and it should automatically format it.

You do realize that if you "uninstall" your only operating system, make sure you have something to replace it, lest you end up with an operating-system-less computer, which is 100% UNUSABLE
