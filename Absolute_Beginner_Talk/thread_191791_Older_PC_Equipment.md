### Post by aysiu on 2006-06-08
It's only the processor that seems a bit slow. The RAM looks good, and you have plenty of hard drive space.

If you want things sped up a bit, try Xubuntu--it's a bit lighter than Ubuntu or Kubuntu.

You can download the Xubuntu CD [here](http://cdimage.ubuntu.com/xubuntu/releases/dapper/release/). You can also just add Xubuntu to an existing Ubuntu or Kubuntu installation by following [these instructions](http://www.psychocats.net/ubuntu/xubuntu).
