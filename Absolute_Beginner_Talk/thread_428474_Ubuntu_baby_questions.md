### Post by starcraft.man on 2007-04-30
6.06 is designed with buissnesses in mind I believe. The long term support is so that they don't need to do a massive upgrade every six (Ubuntu releases on the fourth and tenth months of every year) months and instead will have updates and support for many years (the near equivalent of time between windows upgrades).

If this is just for your desktop, Feisty is advisable, its the latest and while it still has a few problems for the most part it is very stable. Try the live CD first to ensure that you can boot into it and if thats fine I see no reason you can't install it. Feisty has support for all the latest versions of programs, it is easier to get restricted drivers (I still favor envy as an installer for graphics drivers though), and has several other tweaks like gnome control panel, and a few media changes.

If feisty doesn't work, you can drop down to 6.10 its still good.

### Post by theninthdimension on 2007-04-30
Mikerr,

     It is sort of a subjective call whether you install 6.06 or 7.04. Ubuntu is committed to making regular releases of new and updated versions of its OS ([http://www.ubuntu.com/products/whatisubuntu](http://www.ubuntu.com/products/whatisubuntu)) but at the same time, they have to provide a stable platform for businesses and people who do not necessarily want to be on the cutting edge of things. So, if you want something that is stable should be fully supported for a few years, then you want 6.06 which is their Long Term Support (LTS) version. If you want to have the newest and shiniest version, then you want 7.04. Actually, if you have enough disk space, you can install both of them and switch back-and-forth (just like dual-booting with Windows) and see which you like most.

     Personally, I'm a glutton for punishment or, as some say, an early adapter. So, I have upgraded to each new release and am currently running the 7.04 version. One thing to keep in mind, even if you install the 'oldest' version you can find, it is generally easy to update and upgrade as time goes on.
