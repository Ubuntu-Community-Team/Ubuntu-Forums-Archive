### Post by ycason on 2007-12-31
Making the whole windows partition available in a virtual machine is doable but frustrating.  It works very well except that windows thinks you've changed your hardware every time you switch from running it natively to running it in the VM.  This leads to many tech support calls where you have to wait on hold just to reactivate your windows.  

Now, I would never be one to promote piracy but there is a way to sidestep the whole activation thing by installing a pirated XP which has already done away with the activation nonsense.  TinyXP would certainly be a good choice because of it's small memory footprint.  Contemplating the switch to TinyXP on my computer right now for this reason.

Forgot to mention that this can be done with "Virtual Box".  Be sure to download the manual because it will tell you exactly how to set this up after a little reading.
