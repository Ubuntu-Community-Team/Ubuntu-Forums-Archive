### Post by riven0 on 2007-02-03
Can you try running these commands one at a time?:

> sudo apt-get update && sudo apt-get install --reinstall xorg-driver-fglrx

> sudo apt-get install fglrx-control
> 
sudo depmod -a

> sudo aticonfig --initial && sudo aticonfig --overlay-type=Xv


If, for some reason, your xserver doesn't want to start after running these command, reinstall the xserver and do a reconfiguration:

> sudo apt-get install --reinstall xserver-xorg && sudo dpkg-reconfigure xserver-xorg
