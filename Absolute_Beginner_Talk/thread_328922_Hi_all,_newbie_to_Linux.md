### Post by Jeff24K on 2006-12-31
> **Gramayr said:**
> 
What I'm wanting to know (yep, there was an ulterior motive for joining this forum) is what website software is there for Linux. I have Dreamweaver on my windows install - is there anything similar as I'd like to make a website for my photography.


There's more web site authoring software for Linux than one person could learn in a lifetime I think. Editors, tools, scripting environments, image manipulation software galore...

If you're looking for a full tilt WYSIWYG editor start with Kompozer. Formerly known as Nvu, it's a spinoff of Mozilla's editor with a "mission statement" to be the Linux counterpart to Dreamweaver.

[http://kompozer.net/](http://kompozer.net/)

For non-WYSIWYG editors and editors that have clean interfaces to browsers for viewing your creations I'm partial to Bluefish and Geany. Bluefish is geared more towards html authors and Geany is more of a "light weight programming environment". Both are easily locatable in Synaptic I think. There's scores of editors in this category though so do some looking around.

On the other end, I find myself most comfortable using a text edtor for most HTML and CSS, with a terminal window open for file management and running utilities, and an actual browser for preview. And that's after quite a few years of dabbling around with Windows WYSIWYG environments like Dreamweaver and Front Page for ages.  Vim (gvim) is my eitor preference but a lot of people prefer Emacs. Either one has a host of addons and scripts to make your life more enjoyable.

For hands on image manipulation I wouldn't look any further than the Gimp.  No reason to. I also use the command line Imagemagik utilities and a small library of scripts for thing like generating thumbnail images and "gallery" html code. That sort of repetitive task is handled perfectly at the command line in my opinion. :)
