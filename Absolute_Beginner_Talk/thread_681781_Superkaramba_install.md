### Post by PmDematagoda on 2008-01-29
KDE4 has a built-in widget engine called Plasma, because of this SuperKaramba may be no longer necessary. 

Also, if you want to use SuperKaramba on KDE4 then you will have to install the SuperKaramba version that was built for KDE4, if the SuperKaramba you installed was built for KDE3 then you may face some big problems when attempting to use it. You can search the applications and packages that can be installed and used properly on KDE4 by searching Synaptic or Adept for "kde4".
