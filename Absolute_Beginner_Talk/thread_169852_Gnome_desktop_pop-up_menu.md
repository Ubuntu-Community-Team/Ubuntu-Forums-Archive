### Post by ComplexNumber on 2006-05-03
> I was wondering if it is possible to edit the menu items! I wanted to add for example a launcher for my most used applications...
you can use the 'drawers' on the panel for that. add a drawer to the panel, then put all you most frequently used aplications in that. do do this, go to the applications menu, right click on the application that you want to add to the Draw, and select "add to panel". then place it in the Drawer. you can even set it up so that you can have a Drawer for commonly used multimedia applications, a drawer for commonly used graphics apps, a draw for commonly used games, etc.
