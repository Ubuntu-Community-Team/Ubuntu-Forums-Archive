### Post by tubasoldier on 2006-12-20
To install Kubuntu type this in the terminal:
```
sudo apt-get install kubuntu-desktop
```

For a more lightweight way to install KDE and keep your Ubuntu bootsplash
You will have to install any other KDE applications with apt-get or synaptic.
```
sudo apt-get install kde-base
```
