The "fun" would be being able to say "I did it when you all said it wasn't possible."  Being the first is always a thrill, right?  Seriously, "there's so few users, nobody targets it" is a dumb, but common response.  Why do you think such a big deal is made when someone gets remote code execution on OSX?  As long as we say "there's no viruses for us" it'll just tempt the virus-writers to prove us wrong.  Windows is targeted more because it's an easy target.

The truth is, there is 1 virus in the wild which can run on Linux.  Others have been created as "proof of concept" to show where a hole exists in code...and then it gets patched.  Because programs can't execute themselves or install themselves on Linux (need admin approval), it'd be really hard to get one.  A trojan would be possible.  That'd require going out of your way to get a program from elsewhere online instead of doing the easy thing and getting it from the repos, and then installing it.  That's a case of PEBKAC, though, not of OS security flaws.
