### Post by Celegorm on 2007-06-02
Well the best thing to do is to pop in the live cd (same as the install cd unless you get the alternate cd), reboot, and see what hardware works and what doesn't. You'll have to install a driver for the nvidia card for 3d stuff, but those usually work. As for the printer, it depends which model, so you'll either have to try it out, or search for the model number on these forum or linux compatibility lists. Wireless might be tricky, some work easily and some don't.
