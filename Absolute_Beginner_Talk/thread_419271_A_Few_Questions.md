### Post by az on 2007-04-22
When you run the Ubuntu installe ron a system that already has windows on it, if sufficient space is available, the installer will give you the option of shrinking down that windows partition and installing Ubuntu beside it.  You will be offered a menu when you boot and you will be able to either boot into Ubuntu or windows.

If you are not offered that option when you install, try defragmenting your drive or freeing up some space and then try again.
