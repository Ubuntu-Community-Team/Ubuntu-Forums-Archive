### Post by BaffledMollusc on 2007-06-17
Hi there, welcome to Ubuntu!

Yes, by all means post the pictures somewhere where we can have access to them. Before you do, however, you might want to make sure your CD doesn't have errors on it - this is quite a common cause of installation problems.

To do this choose "Check CD for errors" on the menu screen you get just after boot. It takes a while but is definitely worth doing.
