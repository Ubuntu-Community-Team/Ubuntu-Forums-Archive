### Post by macdo on 2006-04-19
MAke sure that the languages match: you may be (are?) using EN-AU for Australia, but only the EN-US dictionary is installed.
Go to Tools->Options->Language Settings for that.

You can install new dictionaries (like en-AU...) with DicOOo, a tool that you can get here: [http://ftp.services.openoffice.org/pub/OpenOffice.org/contrib/dictionaries/dicooo/DicOOo.sxw](http://ftp.services.openoffice.org/pub/OpenOffice.org/contrib/dictionaries/dicooo/DicOOo.sxw)
