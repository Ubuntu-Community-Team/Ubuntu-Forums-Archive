### Post by chuckyp on 2007-03-13
Just install xchat.  sudo apt-get install xchat .   It will be a familiar interface to you.  Then you just select a server to connect to.  I believe it asks you which one you want ot connect to on start up by default.   You can also specify your nick etc... right in the server connect window.  You can also save your settings and have it auto log in when the program launches.

The other option I completely forgot to mention is that gaim does irc.  Click on Accounts and add/edit.   Add irc.freenode.net  have it connect.  Once on there you can click on buddies > Join Chat select the freenode from the drop down and type the name of a channel. i.e. #ubuntu.  You would be off and rolling with out installing any software.

Xchat does have scripting functions, you just have to have the required libraries installed. mIRC uses its own scripting language written by the author, while Xchat has libraries that will allow it to interpret scripts written in C/C++, perl, python, TCL or Ruby. The [documentation]("http://www.xchat.org/docs/") on Xchat has a list of links to the libraries as well as instructions on how to code scripts in the native ones.
