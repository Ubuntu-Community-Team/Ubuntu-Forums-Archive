### Post by Brunellus on 2006-03-01
[QUOTE=himtuna]Just two questions

Will I be able to install winXP applications on ubuntu like dreamweaver, photoshop cs2 ..systemmechanics..power director 5 etc,, I don't want alternatives I have bought these softwares..but planning to change the OS I am fed up from winXP.

1) In a word, no.  You will generally not be able to run Windows software in Linux.  There are ways of making them work (WINE, for instance), but they are presently in development.

2) Installation will be no problem, even as a dual-boot (booting Linux or Windows) at once.

Consider well what you think and feel.  IF you dont' want alternatives, then you dont' want a new OS at all--you will migrate and then complain bitterly about migrating.  

Be happy;  use what works for you.  But do not dismiss alternatives out of hand if you intend to migrate, because in many cases, the alternative is what you will have to use.

### Post by Coelocanth on 2006-03-01
I'll chip in and reinforce what Brunellus said. Having just 'migrated' to Ubuntu only a few weeks ago, I can tell you it's a bit of a system shock if you've been using Windows exclusively before this. Things work differently on Linux and there's a steep learning curve. If you haven't got the patience to learn a new way of doing things or you don't want to spend some time getting things working to your liking, then it's not for you. However, if you really enjoy figuring things out and having total control of your OS (for better or worse... :lol: ), this is a great OS to work with. That being said, there are a lot of programs that are very similar to Windows progs and there are things like Automatix that can do a lot of the work for you.

After 3 weeks or so, I find I like the 'feel' of Ubuntu. It somehow seems cleaner, leaner, and more intimate than Windows. I still boot up Windows when I absolutely must, but I find it feels bloated and awkward now. I know this is just perception, but that's been my experience.

Bottom line: Ubuntu can be similar to Windows in a lot of ways, but don't expect to switch to it and have exactly the same 'feel' as you do with Windows. Many things it does better than Windows, and other things it doesn't.

### Post by Garyu on 2006-03-01
I am setting up Ubuntu on my parents computer soon, and I am choosing both Ubuntu and WinXP, but without dual boot, to make their transition easier.

[https://wiki.ubuntu.com/VMwarePlayerAndQemu](https://wiki.ubuntu.com/VMwarePlayerAndQemu)

Follow that wiki page to install WinXP in Ubuntu and run it in a window. You will have to share system resources, so things will be rather slow when you run XP but as long as you aren't playing games it is IMO a good way of making all of your old applications work while getting used to Ubuntu.

To share files between your virtual XP and Ubuntu you need to get Samba working. You don't need an actual network to do this as VMPlayer will use a virtual network within your computer. Follow this wiki page to make it work:

[https://wiki.ubuntu.com/SettingUpSamba](https://wiki.ubuntu.com/SettingUpSamba)

Enjoy your twin-OS system. :cool:
