### Post by dvarsam on 2006-02-27
Nautilus has an embedded .iso recorder.

Question:
How do I burn an ".iso" File to CD-R?

Answer:
In Ubuntu use "Nautilus".

Note:
In Ubuntu 5.10, the program "nautilus" is already pre-installed.


Perform the following steps:

1. Download the ".iso" File to your hard drive.
    (See Where can I download Ubuntu? for download locations.) 

2. Insert a Blank CD into your CD writer. 

3. To launch "Nautilus",  right click on an ".iso" File you just downloaded & choose 
    "Write to Disc".
    The "Write to Disc" dialog opens. 

4. In the dialog, choose your CD writer & speed, then click on "Write".
    The "Writing Files to Disc" Progress dialog opens & Nautilus starts writing the 
    CD.

Now, you got it?
