### Post by nickpaton on 2006-11-14
Before you actually install the distro, use the check disc fun ction to make sure that it's actually been burnt properly.  
This will also check that you also downloaded it properly.  
If it does not check out OK, go back and get the MD5 checksums from where you downloaded the distro, and run the checksum against your original download.
Also, do you have at least 256Mb of memory?  I actually needed 510 on an older PC to install 6.10.
Get back if any questions about this, also how you get on.
