### Post by Qwerty_ on 2007-10-21
First of all make sure you have all of the repositories selected to download from.

System>Administration>Software Sources.

Then you need to go to System>Administration>Restricted Drivers Manager.

From there you can install the restricted drivers which should fix your problem.
