I use Ubuntu 6.10, I have a networked computer (XP) with an HP Deskjet 710c, and I have no idea where to start to connect to the printer. I don't see it anywhere in Ubuntu and I've been searching the forums and haven't found any step-by-step or where to begin. Anyone?

---

### Post by shanepardue on 2007-03-09
System > Administration > Printing. You can then click "New Printer" and make sure to 
click Windows Printer after you have chosen network printer. Let us know if you need 
more help!
