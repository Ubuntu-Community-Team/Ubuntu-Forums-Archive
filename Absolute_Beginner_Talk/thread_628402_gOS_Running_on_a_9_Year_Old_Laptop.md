### Post by Crumpets and Jam on 2007-12-01
Hello everyone. I just wanted to point out to people looking to install Linux on an old laptop that does not run to well with Ubuntu should consider gOS. It's based on Ubuntu but uses the Enlightenment 17 instead of GNOME. I managed to install it on a 9 year old IBM ThinkPad 600E and it works flawlessly. I've also posted a video on YouTube that can be viewed via the link below. I was just curious if anyone else has used or is using gOS? Cheers all.
