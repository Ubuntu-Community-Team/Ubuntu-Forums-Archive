### Post by tocleora on 2006-11-17
I wrote in on July 28th about my success with Ubuntu compared to other Linux distros I had tried in the past (see "Must... Praise... Ubuntu!" or link [here]("http://www.ubuntuforums.org/showthread.php?t=224865")).  I'm still very pleased with Ubuntu and wanted to write about additional successes I've had with Ubuntu since writing the last post.  I should mention that these are more than likely possible on other distros as well (in regards to me and my difficulty in the past) but since I've *only* used Ubuntu since July I wouldn't know! Hah!  Although I only have a few to list , these are huge for me and I think worth mentioning.

1. Games - I've actually found a few games for Ubuntu that are really fun.  Go to digg.com and search for Top 10 free Linux Games if you aren't familiar with these already. TuxRacer is still working great (I've had problems with past Distros where I installed software and it worked sometimes, didn't sometimes), I've installed Frozen Bubble which is fun, Powermanga is cool (I don't think it's in the list), I got the FPS's in that list working although I should mention they are choppy. I still got them working which I've not had luck doing with in other distros.  But the one I play most is Battle of Wesnoth which is similar to Warcraft.  If I could find a better (prettier) network playable Backgammon I'd be set!

2. Dual Screen.  I work on a laptop that has an external monitor connection.  I was able to get Xinerama to allow me to use both monitors simultaneously.  Unlike XP it makes the screen one big instead of two small which has shown to be interesting (like "centered" windows show up in the middle of the two and sometimes like file open windows show up on the other monitor instead of the main one) but I created a wallpaper that puts a different graphic within each window location, I run a terminal window with top partial transparent on the monitor and work on my laptop lcd.  Works great!

3. Proxy Tunnel.  My primary web server at work is internal and behind the firewall.  So I use a Proxy Tunnel to connect to the server with external access which allows me to get to my internal server.  I tried to get SSH and Squid working but Squid kept giving me problems.  Then I found an article that states you can do it with just SSH.  Sure 'nuff I was able to configure SSH and pointed firefox to the port # and I'm able to see my server from home now!

4. Firefox 2.0.  Although it wasn't available for Ubuntu easily in Synaptic/Apt-get, I was able to find an article and get it installed very easily.  In Fedora it seemed if it wasn't readily available via Yum or an rpm was available for it, it was going to be an event to get it installed.  This was not the case with Firefox 2.0 in Ubuntu.

5. XGL/Compiz. I think I mentioned in another post my success with this, but I wanted to mention it here again.  It did take some configuring but I was able to get XGL/Compiz working in Ubuntu, and a lot of my coworkers were very impressed with that capability and are closer to attempting to use Linux.  I don't run it all the time because it's a little choppy, but if someone is going to be watching me I'll turn it on and act casual about it. :)

There are a few quirks I'm still looking to work out, for one, my scroll wheel when clicked seems to have various purposes, most notably "paste".  This poses a problem as I apparently scroll hard sometimes and end up pasting text throughout my php pages without knowing (a setting in Quanta Plus maybe? Just thought of that).  It would also be nice if one of the web browsers for Linux would work fairly close to the way IE works so I could test web pages in both browsers without having to switch to Windows.  Also pages I've designed for work (which I'm the only one that doesn't use IE in so I've slacked in making my pages multibrowser compatible) I could view without problems.  I also need multitrack software as I am a musician and would love to be able to do that in Linux as well, I haven't researched that at all but will today.

If anyone would like more information on how I got something working let me know and I'll provide as much info as I can.  I'm still for all intensive purposes a linux newbie so I still do a lot of trial and error but I'll help any way I can. :)
