### Post by js_fr on 2007-10-23
I think you don't need an Intel Core Duo, if you want to use your laptop only for irc, listening to music and a few office things. 
I have an Acer Extensa 5220 (Intel Celeron M530 with 1.7GHz) with Debian Lenny running on it (Feisty didn't install, but I didn't try Gutsy until now). Only thing I upgraded was RAM (512MB -> 2GB). Listening music, watching movies (fullscreen on TV, which I connected via external VGA) works fine. Also running a 2nd OS in a virtual machine ([Virtualbox]("http://www.virtualbox.org/")) works without performance problems.
