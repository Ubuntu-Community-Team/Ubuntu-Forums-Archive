### Post by durand on 2008-03-16
[http://wiki.compiz-fusion.org/Plugins/Cube#head-dcbb10c4fb334b8df8319d28379fe3245072c87a](http://wiki.compiz-fusion.org/Plugins/Cube#head-dcbb10c4fb334b8df8319d28379fe3245072c87a)

```
 sudo aptitude install compiz-fusion-plugins-extra compizconfig-settings-manager 
```
The plugin should now be activatable at System > Preferences > Advanced Desktop Effects Settings 
The compiz forum may also help: [http://forum.compiz-fusion.org/](http://forum.compiz-fusion.org/)
