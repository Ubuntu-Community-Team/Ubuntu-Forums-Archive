The DVD has more software, but not all. You can get the repositories on DVD, like I did (There are 5 total) at [www.thelinuxstore.ca](www.thelinuxstore.ca) (also with the installer/live DVD)

Kubuntu is not stripped down, you can get KDE from the repositories with: 

```

sudo apt-get install kubuntu-desktop

```
This will give you everything.

It would be better to get Kubuntu if that is what you want.

### Post by NeoLithium on 2007-05-08
Ubuntu is Gnome; Kubuntu is the distro with KDE installed as the default desktop instead of GNOME; it's not a stripped down version. If you're looking at using KDE right off the bat, you would be best to go with Kubuntu instead, as that way you wouldn't have the extra GNOME files hanging around your installation.
