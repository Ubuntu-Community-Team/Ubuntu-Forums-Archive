### Post by Jucato on 2006-08-07
KDE Wallet Manager is an application that stores and manages the passwords that you use in web pages or in other applications (like Kopete or KMail) in a centralized place, where you can delete or modify them. So instead of each application storing each password separately (like in Firefox or Yahoo Messenger, etc.) you can use KDE Wallet Manger to group/view/edit them in one place.

It's safe because you can set up a password for it, and only you can access it. It also doesn't store the password for your user (the one you use to log into KDE and to do administrative work). So you're safe.
