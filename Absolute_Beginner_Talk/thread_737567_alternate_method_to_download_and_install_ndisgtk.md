### Post by alan34 on 2008-03-27
ndiswrapper and/or ndisgtk should be on the Ubuntu cd.

Go System - Administration - Synaptic Package Manager then Settings - Repositories and tick
cdrom. Note you may have to untick all the others just make a note of which ones are ticked.
Then Reload and search for ndiswrapper.
