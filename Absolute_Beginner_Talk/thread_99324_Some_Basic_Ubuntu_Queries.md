### Post by blueplazma on 2005-12-05
1: If you can download the image and burn it, it makes no difference.
2: If you have a tool that allows you to resize partitions, you can use that. Otherwise, you may have to reformat, but it shouldn't make a big difference.
3: Ubuntu's install is pretty clearcut, but so are many OSes these days. You will learn how to set up a partition table, but if you really want to learn about the guts of installing Linux, you may want to try setting up Gentoo someday. I'd start with something easier though, like Ubuntu.
4: Ubuntu is very well known, and has been ranked number 1 on [www.distrowatch.com]("http://www.distrowatch.com") for some time.
5: I would allot *at least* 10 gigs to your install, although if you're really strapped for space it will probably all fit into 5 gigs. The base install is a bit large.
6: I'm not sure what programs would run both in Windows and Linux.  There's a few, but could you name a specific program?

### Post by TeeAhr1 on 2005-12-05
1. Download that bad boy, no more stalling!  Then, when your CDs arrive, bring them down to the coffee shop and drop them next to the indie newspapers.  Pass the love down.

2. The Ubuntu installer comes with an excellent tool for resizing your partitions.  Reinstalling XP is not necessary.  [Here]("http://users.bigpond.net.au/hermanzone/index.htm") is an excellent walkthrough for you, it helped me immensely.  Save it as a PDF, print it out, and have it at your desk during installation, it will save you all kinds of headaches.

3. I have never installed another distro, so I have no basis of comparison.  Ubuntu was super-easy, though.  I went through the whole thing in under an hour, partitioning and all.

4. Growing every day!

5. I'd set aside at least 10GB if you've got it for the system itself.  How much additional space you set aside for data and user stuff is up to you.  I have a 160GB hard disk, and I just split it down the middle.  It'll work until I get my new drive for Xmas :-)

6. You cannot share applications from Windows to Linux (or vice versa).  If you can figure out [Wine]("http://www.winehq.org/"), then you're one up on me.  Fear not, though.  Some programs (like Azureus and Firefox) have Linux versions, for everything else, there are alternatives.

Good luck!
-pete
