Does anyone know, is there a way to avoid having to login and enter password at powerup?

### Post by hashimoto on 2007-09-02
-> Administration -> Login window -> Security (tab)
Then tick "Enable automatic login" and select the user to be logged in.

---

### Post by wormser on 2007-09-02
System> Administration> Login Window> Security tab> check Enable Automatic Login.

I have never tried it but it looks like that is what you want.
