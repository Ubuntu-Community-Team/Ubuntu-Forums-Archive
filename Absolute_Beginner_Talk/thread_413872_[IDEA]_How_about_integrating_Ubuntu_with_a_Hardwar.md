### Post by papercuts on 2007-04-19
What is the single thing that comes as an issue when people complain about the drawbacks of ubuntu (aside from Games): drivers. And that is unfair because Ubuntu does work with most of the hardware without any problems, but admittedly not all.
[B]
How about Ubuntu evolved into a hardware producing company like Apple and bundled only the hardware that is tested to work perfectly out of the box? [/B]

Ubuntu can design custom case, like standard closed cased (as in laptops and iMac) where the user can change some components. Ship it with Ubuntu pre-installed. 

Ubuntu of course won't be closed source software so charging for the software doesn't make sense. Well then is there any point in this exercise other than creating a headache of international scale? 

I think there is. Ubuntu can target the PC users who want more "cool" in their lives but don't have the bucks to buy Apple. Nothing says more use-friendly than a pre-installed, ready-to-use operating system. By knowing the environment and controlling the hardware selection to a certain extent Ubuntu will be more stable. Bundle with that some great Ubuntu software and create a more sensationalist new user experience (like the Apple "Switch") and Ubuntu can get to more users more rapidly.

Microsoft wins mostly because computers come with Windows. But why do computers come with windows? Because a computer without an OS is useless to a newbie (which by the way is most people). So Ubuntu has to come with the computer. But by making its own computer brand ubuntu can take control of the hardware environment, find another (though small) source of revenue to enable more support.

When people talk about linux it's like a program. They pay for a computer, and they value it. They don't often realize that they pay for an OS too. When you want to put Linux on top of what the computer came with it sounds wrong. Like you're doing something that you're now supposed to do. If we ever want Ubuntu to get out of the technically able elite into the masses we have to start thinking like them (while remembering the google motto: Don't be evil a.k.a Don't be Microsoft)

What do you guys think? Is this completely wrong, useless?
