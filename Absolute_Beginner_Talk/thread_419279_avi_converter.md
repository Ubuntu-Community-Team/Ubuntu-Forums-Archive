### Post by phidia on 2007-04-22
it's very possible. You need to have universe and multiverse repos enabled then use synaptic or apt-get to install mplayer mjpegtools ffmpeg mkisofs dvdauthor normalize-audio transcode & vcdimager. After those are installed you need to download and install tovid [http://tovid.wikia.com/wiki/Main_Page](http://tovid.wikia.com/wiki/Main_Page) see man tovid for useage or ask if use need help using it.
