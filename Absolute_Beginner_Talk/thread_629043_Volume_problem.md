have you tried checking the alsamixer by going into the terminal and typing:

```
sudo alsamixer
```

also, if you're running gnome, try:

```
sudo gnome-volume-control
```

and make sure all volume levels are up... 

if these don't work, is it possible the speakers have gone bad?
