### Post by rajeev1204 on 2006-12-04
hi

i have same onboard graphics nvidia 6100. just download nvidia-glx drivers ( not glx legacy) from repository and install it. u also have to enable it by typing in terminal "sudo nvidia-glx-config enable" . it will probably give u a message about x org has been changed . just read it and manually edit xorg file and change setting from nv to nvidia and save it. then restart pc and it will load with new nvidia drivers


good luck
