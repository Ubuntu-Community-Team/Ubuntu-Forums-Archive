### Post by Sef on 2006-12-22
Have you gone into your BIOS to set the computer to boot from the CD-Rom first?  Usually to get into the BIOS, you push either F2, esc, del, or F12 upon boot up.   There is a message at the start of the boot up that tells you what button to push.
