### Post by MCrittenden on 2008-02-21
Yeah I had the same problem with fluxbuntu. Wifi-radar worked for me.

"sudo apt-get install wifi-radar" and then to run do "sudo wifi-radar" to run and configure wireless networks. However, obviously you'll have to connect to download it, which probably means either hooking up with a LAN cable (which usually works in flux without any trouble) or downloading the .deb on a different computer and moving it to fluxbuntu on a portable hard drive or CD.

More info: 
[http://www.linux.com/articles/55647](http://www.linux.com/articles/55647)
[http://wifi-radar.systemimager.org/](http://wifi-radar.systemimager.org/)

Hope that helps or at least gets you on the right track.
