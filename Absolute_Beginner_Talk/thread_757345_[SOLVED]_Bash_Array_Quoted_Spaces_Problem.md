### Post by unutbu on 2008-04-16
It sounds like you can control the format of the file labelled text. If you change

```
Field1           "Item 1" "Item 2" "Item 3"          Field3
```

to 

```
Field1           Item 1 : Item 2 : Item 3          Field3
```

Then I think 

```
#!/bin/bash
IFS=:
VARIABLE=( $(cut -f 2  < test) )
IFS=" "
```
works.

See [http://www.unix.com/shell-programming-scripting/29202-perl-like-split-function-bash.html](http://www.unix.com/shell-programming-scripting/29202-perl-like-split-function-bash.html)

### Post by unutbu on 2008-04-18
```
#!/usr/bin/perl
use strict;
use warnings;

my $filename=shift @ARGV;
my @item;
open(FILE,"<","$filename") || die;
while (<FILE>) {
    my ($itemstr)=($_=~m/^[^\t]*\t(.*)\t/);
    @item=split(/\s*:\s*/,$itemstr);
    my $cmd="icedax -xHB -D /dev/cdrom " . join(' ',map {"\"$_\""} @item) . "\n";
    print "$cmd";
}
close(FILE);

```

Call the script run-icedax.pl.

Make it executable:

```
chmod 755 run-icedax.pl
```

Then you can use it like this:
```

run-icedax.pl test
```

If you are happy with the output of the script you can make the print statements into system calls:

Change

```
print "$cmd";
``` to
```
system("$cmd");
```
