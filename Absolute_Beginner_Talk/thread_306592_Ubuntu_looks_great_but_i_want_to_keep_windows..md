### Post by xpod on 2006-11-25
Ubuntu will set up the dualboot so you have the choice at boot up of which os you want to use.
It will install the grub bootloader to your windows mbr though which some people would advise against so as not to risk any future probs having to repair it.
I`ve never had issues with suchlike though..not when it was windows i dualbooted with and not now that it`s a different os i have on there(edgy).

There is also a way to do it where you unplug your windows hd while installing ubuntu to the second one so when you bootup you have to hit f8 to choose which hd you want to boot to...NO messed up mbr`s as the grub will be on the ubuntu hd.

If you want more specific help you`ll need to seek the advice of someone else m8 but thats the basics in a nutshell.
