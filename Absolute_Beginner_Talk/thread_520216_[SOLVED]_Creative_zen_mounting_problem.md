### Post by macogw on 2007-08-07
It's not supposed to mount like that.  You know how on Windows you used Windows Media Player to sync it?  Well, WMP uses a protocol called MTP to transfer files.  There are a few Linux programs that do this too.  If you use KDE (Kubuntu), AmaroK can do it.  If you use GNOME (regular Ubuntu), Gnomad2 is the program you need.
