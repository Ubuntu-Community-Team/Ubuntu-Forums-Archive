### Post by paradoc on 2007-06-01
My PCplus mag dropped through the letterbox this a/m, with a Linux collection which includes Ubuntu 7.04:p. (Hope this isn't advertising!) 
         I  had previously held off downloading this (via Torrent), but I was able to burn the .iso to a 700MB CD-R disk (Nero/WinO$) and following this, it installed like a dream.
        I updated and added the VLC  media player which I'm currently using-- I've already got Edgy on another PC for comparison & study purposes.
       Now to an old hand  this might seem like candy for a baby, but for many a newbie this simple account could be the encouragement they need to get a start.............it's easier than you might think!
