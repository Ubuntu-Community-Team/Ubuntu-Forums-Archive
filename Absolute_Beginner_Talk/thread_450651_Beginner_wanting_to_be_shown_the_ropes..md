### Post by Cre-ve on 2007-05-21
I'm also new to Ubuntu. I've only been using it for about a month and a 1/2. Its crazy how much you can learn about linux and about your computer in general in about a month and a 1/2 of using Ubuntu.

Things that have been of great help to is are: GOOGLE, and this FORUM. Anytime you want to do something in Ubuntu, just google search it because I'm sure other people have had problems with it as well and they solved it one way or another.

The primary thing to realize is that getting something just the way you want in linux isn't easy by any means. Things will take time. If you spend the time to learn linux you'll be greatly rewarded. However, if you get frustrated, you will only keep getting frustrated and want to go back to windows. TIME is very important. Your knowledge about linux will mimic your knowledge about windows. Recall how lost you might have been when you first started to learn about windows and how to go about doing things within that environment. The same can be said about Ubuntu and linux. 

The 1st thing I would recommend is that you make a DUAL boot computer with linux and windows. You will undoubtedly have problems doing certain things in Ubuntu at the beginning. As such, if you have something due at a given time, it might be better to use Windows and get it done and then learn to do it in Ubuntu. Installation for me was a pain. If you need help with that let me know, I found a very easy way to install Ubuntu..you will need to buy an additional Hard Drive (which is about $50) and install it.

1) There are several google website that contain information about the terminal and its commands. If you google search Linux Command Line you will find several good links that will help you learn it. Again it will take time and patience to learn about it. 

Here is a good website to start learning (remember you can find more by just googling it)
[http://www.linuxcommand.org/](http://www.linuxcommand.org/)

2) The link below discusses the program comparisons between linux and windows

[http://www.linuxrsp.ru/win-lin-soft/table-eng.html](http://www.linuxrsp.ru/win-lin-soft/table-eng.html)

again, if you need help with installing any programs in linux, i recommend that you just google search what you want. 

Another VERY VERY VERY IMPORTANT link that you should store in your bookmark is:
[http://ubuntuguide.org/wiki/Ubuntu:Feisty](http://ubuntuguide.org/wiki/Ubuntu:Feisty)

this contains TONS of information about Ubuntu. If you just scroll though that, you will basically be able to do anything you want in Ubuntu.

3) The main thing about Linux is that it is very customizable. I've made my own "login" page and my own "splash" page and my desktop looks like Apple's OS-X. You can do pretty much anything you want using Linux.

So how do you go about doing something like that? Well again there are TONS of guides and help out there to modify settings on Ubuntu.

Download and Install Beryl (a very cool program that creates a 3D environment). If you want to see what beryl looks like just go to you tube and type in beryl in search and you will find tons of things.

If you want your computer to look like Apple's OS-X follow the links below
[http://www.taimila.com/ubuntuosx.php](http://www.taimila.com/ubuntuosx.php)
[http://www.taimila.com/osx-guide-2.php](http://www.taimila.com/osx-guide-2.php)

Things like this takes time so be patient when you are working on this.
main things that you will need to know that the above site doesn't explain very well are:
1) Install Beryl (just look for ubuntu beryl install -- It is in the ubuntu guide link above!)
2) to create a docking bar like OSX you will need a program. I used [http://code.google.com/p/avant-window-navigator/](http://code.google.com/p/avant-window-navigator/)
this program is hard to understand at 1st. Remember that the program shows up BELOW your bottom panel so you will need to remove your bottom panel to see it.
3) to get the login  page to work google search: gdmsetup (you will probably need to download it)
4) to get the splash page to work you will need gnome-splashscreen-manager (again just google search it)
5) another program that you might want to look into is: gnome startup manager. 

good luck with everything, if you need anything feel free to send me a pm or e-mail me at [email]forgottentruths9@yahoo.com[/email].

Those will cover how to use Ubuntu. As for B, I must iterate that Ubuntu is a linux distribution, it is not a cheap shell/clone of windows that will let you run all of your windows applications. It is preferable to run the native applications, they are even sometimes better than paid products amazingly.You can find apps for Ubuntu [here,]("http://linuxappfinder.com/") as well as a win alternatives section on the front page. 

If you must have one or two apps from Windows, look into WIne (google WineHQ) and you will find infromation on the application layer. Please don't expect to run every windows app in it though...
