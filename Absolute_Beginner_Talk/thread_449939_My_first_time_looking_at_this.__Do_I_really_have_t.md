### Post by starcraft.man on 2007-05-20
Just did a quick search of repos, jamin is there. No need for super complicated steps. Once you are set up with Ubuntu, you can just run this command in the terminal (Applications > Accessories > Terminal):

```
sudo aptitude install jamin xserver-xorg-input-jamstudio
```

It will install any dependencies. I assume jam studio input is how you get it to receive audio, so I put that in to install too.

Both problems resolved, have a nice day now :D.
