### Post by nudnik on 2006-09-19
If you dont want to bother with the extended instructions on the webpage, I suggest installing the "VLC" media player and opening the VCD with the VLC media player.

Open a terminal by clicking on Applications, then Accessories and choosing the Terminal program from that menu. Once the window has appeared, type the following after the prompt:

sudo apt-get install vlc

HIT ENTER
ENTER PASSWORD

After the install is complete, you should be able to access the player from the Applications menu under "Sound and Video"
