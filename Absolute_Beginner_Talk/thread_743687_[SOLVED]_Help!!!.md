### Post by Bartender on 2008-04-03
So you compared the .iso download against the md5 and it checked out?  As far as you can tell the download is 100%?
If so, I'd try burning the .iso to a CD on someone else's PC. Use high-quality CD-R's, not CD-RW.  Burn at 4X or thereabouts.
Burning Linux CD's is not hard once you get the hang of it.  Keep trying different things until we figure out what's hanging you up.

### Post by ugm6hr on 2008-04-03
Did you follow these instructions when burning?
[http://www.psychocats.net/ubuntu/iso](http://www.psychocats.net/ubuntu/iso)

If the md5 matches, but the CD check doesn't then there is a problem with the burn process.

Possible causes I can think of:

1. Bad CD-R's
2. Bad CD-drive
3. Bad CD-burning software
4. Too fast burn speed (4x should be OK - it is he slowest my computer will allow)
5. Vibration during CD-writing

You do not need internet after download is completed.
