### Post by ncappel1 on 2006-05-07
Sound juicer also uses something called paranoia, which is a setting that affects how fast cds rip. basically, a higher paranoia value will slow down ripping, but will also ensure a cd with no jumps or defects. you can access it via the configuration editor-->apps-->soundjuicer 

when paranoia is set to "0" it is off. this will drastically speed up the cd ripping, but watch out as the cds may skip. I personally have not had any problems with it yet though. I'm sure it also depends on your cd burners quality.
