### Post by Tom Brokaw on 2006-08-17

### Post by x64Jimbo on 2006-08-17
I have been using various processors for several years now, and I'll give you my %0.02
AMD has led the market in performance per $, clock cycle, and watt for quite some time now. Only recently with the release of the Core family of processor family has Intel become competitive again. If you want to know the truth, the Core architecture is actually based on the Pentium III architecture. NetBurst was a massive failure for Intel, and it took them way too long to realize it.
AMD's next offering scheduled for release sometime in Q1 2007 is (according to industry experts) supposed to reclaim the crown from Intel. If you're looking for a good deal on a chip, AMD is still king in performance per $ and clock cycle, so I'd start looking at them first.
