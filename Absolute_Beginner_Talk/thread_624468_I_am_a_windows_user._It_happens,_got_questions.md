If you stay at the cutting edge, you can either use a pay-for conversion service like cedega, or dual boot. Sorry, that is about it unless you are a coder and can help develop wine yourself.

### Post by candtalan on 2007-11-27
> **miscrean said:**
> So I read that you can try it first before doing a hard Install. Knoppix was like this with the ability to just run of the CD. Is there a way to do this with a download (burn to disk or image it to daemon)?
yes, the knoppix technology and experience was spread quickly - the world of open source (!) For ubuntu desktop (which is the live cd) go to an official mirror such as
[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)
When the iso image file is downloaded, simply burn it to cd using a 'burn image' facility (not a make iso facilty!). A simple and very useful check can be used in the CD initial boot up menu - check CD. It will check the image is downloaded, burned and also being read fully ok. There are other ways of checking downloads, images and burning (md5sums).
Note that live CD s take 256MB ram to be useful at all. Lower ram and you can install (not live cd) using an Alternate installer - text only  no GUI.
