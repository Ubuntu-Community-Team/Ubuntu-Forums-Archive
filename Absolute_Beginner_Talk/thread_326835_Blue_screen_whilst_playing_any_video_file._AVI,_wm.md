### Post by mahiyar on 2006-12-31
Does the Mplayer work in command line? In my case also not all players work equally well. If MPlayer works from the command line then maybe there is an error in Video out plugin. Right click GUI of MPlayer >Preferrences>Video you will see a list of available drivers just clicking or selecting any enables the driver, select each one by one and see if your video plays, that is how I got mine to work.
