### Post by bodhi.zazen on 2006-08-22
Fedora core 5 is a significant improvement over Fedora core 4.

Both Ubuntu and FC5 are outstanding distro's.

Pro's and cons are a matter of style and personal preference.

Try both and decide for yourself.

For Fedora consider trying BLAG. BLAG is a single install CD based on FC5:

[http://www.blagblagblag.org/](http://www.blagblagblag.org/)

Also Fedora Frog, a script to set up many "extras" in Fedora:

[http://easylinux.info/wiki/Fedora_frog](http://easylinux.info/wiki/Fedora_frog)

The frog is almost cheating.

Last:

[http://ubuntuguide.org/wiki/Fedora_fc5](http://ubuntuguide.org/wiki/Fedora_fc5)
