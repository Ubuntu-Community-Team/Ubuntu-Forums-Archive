### Post by aeto on 2006-07-24
hahah questions like this are invalid. dual core & single core are 2 different things..u cant say which is better than which. If ur gaming, dual core wont provide any performance gain since multi-threaded games are yet to surface. If ur designing, then dual core is ur best friend. Basically, not only if u multi-task, but mainly if ur apps are multi-threaded (like Photoshop, 3DSM etc), u should opt for a dual core system. So..what would u b doing with ur PC?
