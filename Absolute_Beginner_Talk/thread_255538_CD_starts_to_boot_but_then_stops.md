### Post by Metacarpal on 2006-09-11
How much RAM do you have, Eldar?  Are you trying to use the "desktop CD"?  If so, and if you have at or under 256MB of RAM, it probably won't work.

The "desktop CD" is also known as a LiveCD; that is, it loads the entire operating system into your computer's memory when it boots up.  That way you can test drive Ubuntu before you commit to installing it to your hard drive.

If you have a low amount of RAM, you'll want to download the "alternate install CD", which uses a more text-based installation method that requires far less memory.  Also, if you have an older/slower machine, you might try [Xubuntu]("http://www.xubuntu.org/"), as it requires less memory than the standard Ubuntu install.

If you have RAM running out of your ears, I'm sorry I wasted your time with my post! :)
