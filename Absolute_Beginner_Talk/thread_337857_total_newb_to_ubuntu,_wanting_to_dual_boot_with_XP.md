You can check out the wine website for a relatively complete list of applications that other users have tried to use with Wine and see if they had any success. There is also a good guide for running other window apps with Ubuntu in the support section that also covers 3rd party apps that may do a better job. For what it is worth, I got tired of trying to get wine to support some of the windows apps I needed and I started using VMware Server. It is working great and is free. 

I went with Kubuntu edgy and it was better (IMHO) than Ubuntu or Xubuntu. It seems to be an easier transition for a soon-to-be ex-windows user. Good luck!
