### Post by raja on 2007-03-16
You can make a backup of the files as they are and restore if you want to. You can follow [this]("http://www.ubuntuforums.org/showthread.php?t=35087") for example. The other way is to make an image of the existing setup, which makes it easy to go back to. Partimage is an excellent free software to do this and you can boot from the Sysrescuecd to use it. [This guide]("http://www.ubuntuforums.org/showthread.php?t=287522&highlight=backup") will help you with this.
