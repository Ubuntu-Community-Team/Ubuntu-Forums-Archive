### Post by compmodder26 on 2007-05-04
hmm...  should work out of the box.  Did for me anyway.   Open your xorg.conf file:

```
gksudo gedit /etc/X11/xorg.conf
```

Add the following section (if it doesn't exist already) to the bottom of the file and save it:
```
Section "Extensions"
    Option "Composite" "Enable"
EndSection
```

Restart the xserver: Ctrl+Alt+Backspace

*Note: Please back up your original file:

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak
```
