### Post by davmac on 2005-10-26
You mean Microsoft's SQL Server 2000? If so then I think it unlikely. If you look at the product requirements at [http://www.microsoft.com/sql/evaluation/sysreqs/2000/default.mspx](http://www.microsoft.com/sql/evaluation/sysreqs/2000/default.mspx) you'll notice that all of the OS's it'll run on are prefixed with the phrase "Microsoft Windows..." ;) but then of course they would say that! Mind you there have been rumours of a team in Redmond working on a Linux port, but that's of no use to you just now.

If you need a database that'll run on Ubuntu then have a look at MySQL, or is there a particular reason why it has to be SQL Server 2000?
