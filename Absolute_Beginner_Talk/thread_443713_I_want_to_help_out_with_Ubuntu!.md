Hey Dillon,

First of all, welcome to the world of Linux! I can relate to your post in a number of ways. I've been using Linux for around 2 years, and settled on Ubuntu last year because it was the distro that I felt most comfortable using and the distro which had the ideology which I agreed with the most. :) Also the community is excellent and much more active  and supportive than other distros which I have tried.

I also feel like I want to begin helping out with Ubuntu. I am studying software engineering at university at the moment, but when I have some time on my hands then I want to start participating in some open-source projects. I will probably have to look into Python myself as I have not done much with it and it seems to be the scripting language of choice used by Ubuntu.

Don't forget though there are several other ways that you can contribute to the community, for instance if you are any good at artwork (which I am sadly not!), or by offering advice to new users through the community forums, everyone of us can contribute in that respect.

With regards to a programming language, I use Java at the moment because it is what is taught on my course. Java can seem a bit weird when you start off, but once you get into the language a bit, it does have some very nice features. If Java isn't your thing, then perhaps have a look at some C++, I wouldn't recommend C personally, it has a lot of pitfalls - I didn't enjoy programming in it last year! I think that an Object Orientated language would be advantageous. I am currently looking at C# and other .NET randomness for a job which I have lined up for the summer, mono allows you to write applications in C# under Linux (just search mono in synaptic package manager).

On the other hand, if your looking to get into web application programming, then have a look at PHP, it's not that difficult once you get around the basics and you can create some really interesting projects with it :) Some open source projects (Wordpress is just one example) use PHP, so there might be a possibility for you to develop these skills to put to good use in the future!

I rambled a bit more than you :p

### Post by srt5 on 2007-05-15
> **Dillon said:**
> Hmmmm.....what would be the best way for me to start learning, say, Java?  Maybe some good resources for me to learn by myself?  (Doesn't have to be Java, though, any resources would be great!)

Well, the obvious answer that I would give is to get a book. However, these can cost a fair bit, and I find a lot of books have really long introductions to the language and more often than not I get bored reading them.

If your looking at learning Java then I would recommend a book called "[Big Java]("http://www.horstmann.com/bigjava.html")", it's taught me everything I've needed to learn to build some cool applications, it also covers some GUI stuff (so you can escape from those boring terminal programs!), and (I know this sounds childish) but it's colorful and interesting-looking, which I think is important to keep you interested. I'm sure if you look on Amazon, then you will be able to find it at a reasonable price.

As for books on other languages I'm not really sure, if your interested in PHP, then perhaps take a look at the [w3schools ]("http://www.w3schools.com/")website, it has tutorials on a lot of different web technology stuff! If you have reservations about spending money, then there are probably loads of free tutorials out there on the net, a simple search on Google should get you on your way. :)
