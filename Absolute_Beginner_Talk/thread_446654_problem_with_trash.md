### Post by Compyx on 2007-05-17
How are you deleting these files? If you use a terminal emulator and the 'rm' command, the files won't go to the trash. On the other hand, Nautilus by default only provides 'Move to Trash' in it's context-menu, you'd have to manually add the 'Delete' action for the context-menu.
