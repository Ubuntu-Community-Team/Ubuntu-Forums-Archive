### Post by po0f on 2006-11-30
Mazen,

You can mount ISO images via a loop device:
```
[FONT="Courier New"]$ sudo mkdir /mnt/loop
$ sudo mount -o loop /path/to/file.iso /mnt/loop
$ cd /mnt/loop && ls[/FONT]
```
I'm not sure about other image formats, maybe try [cdemu](http://cdemu.sourceforge.net/) for BIN/CUE images.
