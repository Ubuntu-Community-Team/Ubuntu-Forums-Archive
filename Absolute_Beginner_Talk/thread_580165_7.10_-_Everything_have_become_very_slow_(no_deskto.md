### Post by JoSSte on 2007-10-19
I'm having a similar problem.

after upgrading to 7.10 gutsy gibbon, the system is using far more resources, especially the XGL process - using 80% of the system resources. That was definately not the situation in 7.04...

i solved it now - 
> The only problem I can see is the shipping of compiz enabled by default which leads to a number of problems. No way to turn it off unless I place a file called disable in /home/<user>/.config/xserver-xgl/ . Why would I want to turn it off? Well, this is a laptop with a radeon xpress 200M GPU, and playing back video full screen with XGL and compiz running is somewhat jumpy.

read a post at [linuxquestions]("http://www.linuxquestions.org/questions/ubuntu-63/ubuntu-7.10-590682/") which offered the solution of putting an empty file named disable in */home/<user>/.config/xserver-xgl/* - now XGL no longer eats up 80+% of my processing power
