### Post by Crumpets and Jam on 2007-12-21
You don't need the fancy Live CD desktop environment to install Ubuntu. This takes up a lot of RAM and on some computers, slows them down so much that it is a very long time before the installation occurs.

To get around this, use the Alternate Desktop CD. This CD does not include the Live CD, instead it uses a text based installer. To get the Alternate CD, make sure you tick the small box that says "*Check here if you need the alternate desktop CD. This CD does not include the Live CD, instead it uses a text-based installer*" when on the [download]("http://www.ubuntu.com/getubuntu/download") page. Check out the screenshot below if you need a bit of help.

[[IMG]http://img231.imageshack.us/img231/481/ubuntudownloadoe6.th.png[/IMG]]("http://img231.imageshack.us/img231/481/ubuntudownloadoe6.png")

By the way, make sure you get the right CD for you proccessor type. 32 bit and 64 bit computers need different CDs.

The file that you have download is an ISO. This needs to be burned to a disk using special software. You can't just drag it on to a disk in Windows and expect it to work. If you need help on burning the ISO image, [this page]("https://help.ubuntu.com/community/BurningIsoHowto") describes in detail how to do it on Windows, Ubuntu and Mac.
