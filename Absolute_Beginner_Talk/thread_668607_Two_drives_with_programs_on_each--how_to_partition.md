### Post by logos34 on 2008-01-15
> **gwmerc said:**
> On the one hand I can't just transfer everything from the smaller to the larger and make the smaller an Ubuntu drive. Alternatively I dont know if it is possible to partition the larger drive without erasing all the XP data and programs from that drive while creating a space to install Ubuntu on.

Install ubuntu on the 80 gig.  Perfectly safe.  Choose 'guided' install and shrink windows partition down.  Install in the free space.  At boot you will have a choice of OS's.

[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)
