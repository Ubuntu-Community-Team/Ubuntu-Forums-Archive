### Post by ainsworth on 2007-10-25
How can I get gedit to open a text file in a new window instead of a tab as default? It's getting a little annoying not having windows open where you expect them, other than that gedit is proving to be quite handy so switching to another editor would not be ideal.

---

### Post by nick_h on 2007-10-25
You can use:
```
gedit --new-window filename
```

You can also right-click on a tab and select "Move to New Window".
