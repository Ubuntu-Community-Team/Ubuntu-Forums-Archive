### Post by floppy on 2005-09-10
I'm no expert on this, but I think you have to change the "skin" of xmms to get a different font color.

These instructions come from [http://forum.xmms.org/viewtopic.php?t=1596](http://forum.xmms.org/viewtopic.php?t=1596)
1. choose a suitable skin from [http://www.xmms.org/skins.php](http://www.xmms.org/skins.php) 
2. download the (zip) file to folder ~/.xmms/Skins/ 
3. in XMMS, use the menu "options"->"Skin Browser" to activate the skin
