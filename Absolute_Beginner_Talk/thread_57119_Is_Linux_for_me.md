### Post by az on 2005-08-15
[QUOTE=Khane]
I know, Linux is free. Apart that fact , I would like to know if view what I use my computer for, preparing my PC for installing Linux , learning  and getting use to Linux is something I should bother to do or if Windows although his shortcomings is enough for the kind of user I am ? If Linux then why Ubuntu ?[/QUOTE]

It is free (as in no cost) and it is free as in not property.  Some people find the concept that software is property offensive.  It is like saying that someone can own an idea.  And anyone else who would have had that idea by themselves is out of luck.

Ubuntu is free in that it is not proprietairy.  It is built upon Debian linux which has a strict adherence to the principles of free-libre software.  The Debian Free software Guidelines describe how the software has to be open source and compatible with the GPL (GNU Public licence - a licence which prevents others from closing off derivatives - it is always free-libre) to be considered free.

You do not have to agree with this point of view to find that Ubuntu is an outstanding operating system.  But the fact that it does not restrict the user or developer from fully and freely using the source code is the single most important destinction between Ubuntu and other (proprietary) operating systems.

### Post by aysiu on 2005-08-15
[QUOTE=Khane]
I mostly use my computer for surfing the Internet, downloading files, chatting on discussion's forums, sending /receiving email, downloading with emule and sometimes using Word. I don't work with my computer, do not develop programs etc...[/quote] You can use a Linux computer to accomplish all of those tasks.

Is Linux for you, though? That's a tough question. Problem is not that you can't *use* Linux for all of those tasks. The problem is that you have to *install* and configure Linux to be able to do all of those tasks.

Don't get me wrong, Linux isn't any harder to install than Windows, but if you've ever installed Windows (from scratch--not recovery CDs), you know even Windows itself is hard to install and configure. It doesn't always have to be hard, but there are little things you have to think about:

1. If my screen resolution is off, will I ask how to fix it and figure it out, or will I chicken out?
2. If I'm unable to play encrypted DVDs right away, will ask people how to fix it and type the commands they tell me to type, or will I chicken out?
3. If I can't figure out how to X in Linux that I knew how to do in Windows, will I complain that Linux isn't ready for the desktop and go back to Windows, or will I reassure myself that this is a new system, and it does it in a different way?

Think about it. I'm not saying expect everything to go wrong. In fact, when I installed Ubuntu, the only thing that wasn't working was my screen resolution (since fixed, of course). I'm saying, don't expect everything to be a snap with no configuring.

That said, once it's all configured, Linux is simple to use for the tasks aforementioned.

> 
If Linux then why Ubuntu ? I wouldn't recommend Ubuntu to everyone, even though it's my distro of choice. If you're deathly afraid of the command-line, to the point where you can't even copy and paste commands people give you, then use Mepis--it's all point-and-click configuring. If you don't mind using the command-line a little bit in the beginning, as long as people tell you exactly what to type, Ubuntu is an amazing distro. It's free. It's upgraded every six months. It's stable. The community here is terrific.

That's all I can say.
