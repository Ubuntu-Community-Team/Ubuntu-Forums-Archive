### Post by narendranag on 2008-03-14
1.I need an instant messenger service -- Pidgin is already installed, start it up, it's fairly easy to configure
2. a good video and music player (do i need codecs ? i dont want to convert all my media) - If you go toe Applications > Sound and Video you'll find a bunch of software ... otherwise, in the forums go to multimedia to figure out how to get things to work
3. Mirc client > Go to Applications > Add/Remove .. search for XChat, install and use 
4. Anonymous proxy, does Tor work with open source? yes.
