### Post by ddrichardson on 2007-09-27
There are projects similar to VB such as Pheonix Basic, but frankly they aren't too well supported and quite often are proprietry.

If its to create GUI based applications then try using Glade & Python or even TCL/Tk. Glade creates the screens in a screen painter application, providing the code for Python projects and is the preferred language for Ubuntu desktop development. TCL/Tk is just about the easiest syntax ever for creating displays.

Not too sure what you mean by "GUI programming environment", if you mean IDE then there are lots to choose from, Geany, Anjuta, Eclipse, the list goes on. If you mean screen painters then, like I said - Glade.

### Post by ddrichardson on 2007-09-28
An IDE is an integrated development environment - an application that provides, ypically, an editor, debugger and compiler in one place.

It depends what you mean by simplicity. If you mean being able to draw the program on screen and attach code then Glade/Python is not terribly difficult to learn and there a millions of good tutorials out there. And Python is available for just about every platform out there.

I suspect however that you really want something like Delphi or VB, for which there is no really good multi-platform equivalent. Try Glade.
