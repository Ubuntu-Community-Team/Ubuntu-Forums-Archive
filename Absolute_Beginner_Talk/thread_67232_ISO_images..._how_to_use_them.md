### Post by Xian on 2005-09-19
Download the free version of [DeepBurner](http://www.deepburner.com/?r=download).

To create an installation CD using DeepBurner on Windows:

Place a blank, writable CD in the CD drive.
Launch the DeepBurner application.
Enable the Burn ISO image option, and click Next.
Click the ... button on the right side of the window.
Now select the ISO file.

Select the CD or DVD Drive from the drop-down list.
Set the Write speed to 4x or the lowest possible speed, then click Burn ISO.
The files are burned to the CD, which takes about 20 minutes at 4x.

Now the CD should be bootable.
