### Post by Pragmatist on 2006-03-20
Your kernel does have the **eth1394** module.  You can read more about how to use it at:
[http://www.linux1394.org/eth1394.php](http://www.linux1394.org/eth1394.php)

However, this is not to say that I endorse IP over Firewire in Linux.  It seems like it is still insufficiently developed.
