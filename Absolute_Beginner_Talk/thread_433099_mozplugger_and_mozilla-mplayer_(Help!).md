### Post by LuisGMarine on 2007-05-05
To answer your second question, please go ahead at the the [MediaPlayerConnectivy Add-on for Firefox]("https://addons.mozilla.org/en-US/firefox/addon/446").  This is a great tool to allow you to choose how to open certain media formats with the program you choose.

When you install the add-on and the browser restarts, its going to ask you to choose what program to use on certain media formats.  Hoping that you already have mplayer and its proper packages installed, all you would have to do is navigate to your /usr/bin/ and look for the maplayer program.

If you ever need to change this default settings, for example you want to experiment with VLC, all you have to do is in the Firefox window go to Tools > Add-on's > MediaPlayerConnectivity Preferences.

From there you can check to activate the extension for media type and select the application to launch.  

Give it a try, this make it a lot more easier to view videos on the web.

Good Luck
