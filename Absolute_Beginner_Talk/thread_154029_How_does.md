### Post by catlett on 2006-04-02
First here are the differefnt links to Edubuntu,Kubuntu and Ubuntu
[http://www.edubuntu.org/](http://www.edubuntu.org/)
[http://www.kubuntu.org/](http://www.kubuntu.org/)
[http://www.ubuntu.com/](http://www.ubuntu.com/)
The three version are all based on Debian GNU/Linux. So the basics of the operating system are the same.  What differs is the desktop environment. In the Linux world there are two dominant desktops. KDE and Gnome. They are GUI (graphical user interface) window systems. In a real crude sense you can look at Microsoft and say MS DOS is the base and Windows XP is the desktop. The desktop environment lets you point and click to get things done instead of just typing in commands at the command line. 
Although they are based on the same system Ubuntu and Kubuntu are different in appearance and function. The desktop is set up different. They use different programs. Some of their commands are different. If you were to ask which one is better you would get a thousand replies with a thousand different opinions. I have both on my computer. I triple boot XP, Kubuntu 5.10 and Ubuntu 5.10. I'm more partial to Gnome but I like KDE almost as much.
Edubuntu is based on Ubuntu. Meaning it comes with Gnome. It's difference is it comes with educatuional based programs. It is geared for students to use for school. The best way to try out different styles in Linux is Live CDs. If your curious about a certain desktop or distro, get a Live CD and pop it in. This way you can experiment without changing your computer. (I download a new Live CD every week just to see what's out there and how they set up their system)
