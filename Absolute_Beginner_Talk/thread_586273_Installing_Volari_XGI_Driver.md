### Post by Prefader on 2007-10-23
Are you sure you can't use the sis drivers?  The package info for the driver says:

"Description: X.Org X server -- SiS display driver
 This package provides the driver for all SiS and XGI Volari cards."

Should be as simple as telling xserver to use the sis driver.  Try ```
sudo dpkg-reconfigure xserver-xorg
```

and select the "sis" driver.  If it's not listed there, try ```
sudo apt-get install xserver-xorg-video-sis
``` and try it again.

I couldn't get the driver from their website to compile, which is why I ask if it's neccessary to go that far.
