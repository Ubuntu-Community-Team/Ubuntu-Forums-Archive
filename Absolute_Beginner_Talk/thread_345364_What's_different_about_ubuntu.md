### Post by rabid emu on 2007-01-24
It comes with some drivers installed by default so that it supports a lot of hardware without as much tinkering as other distros.  For example, no other distro supported my intel 3945abg wireless card and installing it was impossible, with ubuntu it worked out of the box, so-to-speak.

Installing apps is easy with huge repos of up to date (mostly) software and the apt system.

Basically it's an easier distro to use, but it's still as powerful as anything built on linux.

### Post by Hendrixski on 2007-01-24
Each distribution is different.  Ubuntu provides a philosophy with it that is appealing to many people.  Read about "what is Ubuntu" in the documentation.  You'll find it very interesting.

The documentation for Ubuntu is very good, a lot of people put a lot of work into it.  You're a software developer so you're used to reading documentation.  You'll notice a difference,

Also This forum is VERY active.  Look at how fast you get technical answers, and how many users you can interact with every second.  You can even get programming help here.  

Plus Ubuntu is easy to use, as someone above me mentioned.  But then again, lately all Linux has been getting pretty easy to use.  Synaptic is an easy way to install programs (or apt-get in a terminal, if you're a nerd like me) and everything just works.

That's the Ubuntu difference!    Welcome!
