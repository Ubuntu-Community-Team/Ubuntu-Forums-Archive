A few that are available in Synaptic:

armagetron <- cyberspace-looking 3D bike racing that requires extreamly quick reflexes.

supertux <- 2D Super Mario clone.
enigma <- Hard to describe but it's very addicting if you like puzzles.
wesnoth <- Good turn based strategy, make sure you get the extra campaigns too.
