### Post by equal on 2007-02-06
Well, there's two possibilities. One is to install soundkonverter:

```
sudo apt-get install soundkonverter
```

Another thing you can do is simply install the missing codecs for the files you can't read. If you install this wonderful tool called [Automatix]("http://www.getautomatix.com"), it has a graphic interface that installs codecs as well as dozens of useful programs with a few mouse clicks. Just choose the ones you need. When you go to the site, click the Installation link at the left, and it will bring you to a page with .deb files to install them. Installing .deb files is easier than installing .exe on Windows! Good luck with everything!
