### Post by geirha on 2007-02-23
The answer is probably yes, but your question is wrong. NTFS is a filesystem, not a type of harddrive. But I assume you mean you have windows on your laptop, and you probably only have one large partition with ntfs? If so, you probably have three choices, 

1. Use some windows program to reduce the size of your windows partition and then install ubuntu next to windows (dualboot)
2. remove the entire windows partition and install ubuntu (the ubuntu installer will do this for you if you want it to)
3. reinstall windows, but using a smaller partition, giving room for ubuntu as well (dualboot)
