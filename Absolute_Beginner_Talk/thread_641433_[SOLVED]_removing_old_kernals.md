### Post by zcal on 2007-12-15
If you're using Kubuntu you're probably going to be using Adept instead of Synaptic.  It's somewhere in your K menu.

Anyway, opening it up and searching for "linux-image" should give you a list of all kernels available in the repositories and those that you have installed.  You can choose to uninstall the unwanted kernels there.  Just be careful, you're dealing with critical system components!
