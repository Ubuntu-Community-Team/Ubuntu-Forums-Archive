### Post by iProcrastinate on 2005-11-08
i'm using a hp dv1000 laptop and i am considering running a dual boot of some distro of linux along with my current OS, WinXP.  ubuntu has caught my eye and  i would like to try it out using the live cd rather than going ahead and installing it.  i cannot get my laptop to boot the cd.  i went into BIOS and switched the boot order to boot to cd first.  when i try booting, the screen goes blank for a few moments with a small gray underscore cursor blinking at the top left, and then it proceeds to load windows.  it does this with the ubuntu live cd as well as knoppix.  any suggestions?

thanks!

### Post by jasay on 2005-11-08
my guess is you burned the cd's as data cd's.  You need to burn them as images.  You might check here to see if there are instructions for your particular burning software. [http://help.ubuntu.com/starterguide/C/ch01.html#id2473365]("http://help.ubuntu.com/starterguide/C/ch01.html#id2473365")

A quick way to see if this is the probelm is to browse the cd.  If there is one ubuntu*.iso file, then you will need to reburn it.  If however there are a bunch of files then something else may be the culprit.

---

### Post by iProcrastinate on 2005-11-08
you were exactly right.  i burned both as data cds rather than images.  i reburned ubuntu as an image and it is working just fine now.  i may say i am very pleased with it.  thanks for your help!
