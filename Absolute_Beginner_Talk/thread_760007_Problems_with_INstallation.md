### Post by quirks on 2008-04-19
You say you burnt the ISO image at two different speeds, so I'm assuming you used two different blank CDs. The CDs themselves should not be an issue, hence.

You did a checksum on the image, so the image should be valid as well.

Apparently your CD drive (or the software driver) is malfunctioning. Windows should never freeze because of the content of a CD. Do you have the possibility to burn the CD on another computer or CD drive?

When you boot, do you have the possibility to enter a boot menu to boot from CD? If not, you might have to change the boot order in the BIOS. If this does not work either, the CD or CD drive are somehow corrupt.
