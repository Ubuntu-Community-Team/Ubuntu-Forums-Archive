### Post by troymcdavis on 2007-04-04
When you get to the log-in screen, there is an menu labeled "Session". When you click that, you see a list of all the Window Managers/Desktop Environments you have installed. You click whichever one you want to use, and then it asks if you want to make that default.

If you REALLY want to know where the setting is stored (rather than how to change it), consider this a bump. Best of luck.
