### Post by blackened on 2007-01-18
"Order" is the order that programs start in.
A program with an order number of 1 will start before one with an order number of 2 and so on.

There are four different styles:
Normal - Program starts automatically when Gnome starts up.
Restart - Starts with Gnome, but will automatically restart if it crashes or is manually killed.
Trash - Will not start when Gnome starts.
Settings - Starts with Gnome. Used to store session configuration settings.
