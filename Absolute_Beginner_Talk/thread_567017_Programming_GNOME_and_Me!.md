### Post by ryno519 on 2007-10-04
C# might be a good bet. Mono comes installed with a fresh ubuntu install so it's not a problem there and the development environment (MonoDevelop) is similar to Visual Studio. The class libraries are licensed under the MIT/X11 license which means your program does not have to distribute its source code with the executable. You would have to add a copy of the original copyright notice to your program, however.

Check it out here. [http://www.mono-project.com/Main_Page](http://www.mono-project.com/Main_Page)
