### Post by rapiscan on 2008-02-24
You can customize your theme and select a different title bar.  
System->Preferences->Appearance
Customize your theme, and select window border.

I personally have this theme installed, and although this isn't my main theme, I use the window border from this theme.
[http://gnome-look.org/content/show.php/arsen?content=45954](http://gnome-look.org/content/show.php/arsen?content=45954)

### Post by rapiscan on 2008-02-24
Just to be sure we're on the same page, we're talking about the height of the title bar right? The title bar being where the minimize, maximize and close buttons are?  This can be adjusted depending on the theme that you use.  For example, I think the theme I pasted has a smaller height than the human theme.

Try installing a different minimalist theme and using the window border from that theme.  There are some themes where the title bar is only a few pixels high.

If you want to delve into the gnome configuration, type 'gconf-editor' at the command line.
