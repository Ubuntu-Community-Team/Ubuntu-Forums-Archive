### Post by qamelian on 2008-04-13
Well, Ubuntu does. The gdebi tool that is installed by default will allow you to install any correctly packaged deb file. While this only works on deb files, quite a lot of software is available in this package format. This won't work on source packages and won't work on non-debian packages, such as rpm, but it is a "double-click installer". 

If you are looking for a universal installer for everything thing though, there ain't no such animal yet.

### Post by aysiu on 2008-04-13
While it's true that Ubuntu allows for double-clicking .deb files for installation, downloading individual .deb files isn't the preferred way to install applications, as those programs will never be updated.

You should use the package manager whenever possible. For more details, read [http://www.monkeyblog.org/ubuntu/installing](http://www.monkeyblog.org/ubuntu/installing)
