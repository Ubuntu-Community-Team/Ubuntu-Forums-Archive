### Post by Qwerty_ on 2007-10-20
Ok Ender429 you will have to install those codecs since they are support by Ubuntu out of the box.

This is very simple to do.
There are a couple of methods you can try one of them requires you to try and play the format in a multimedia player. So go to open those files up in something like movie player . When it detects that you are not able to use those formats it will prompt you to search for a codec. A window will open up and all you have to do is select which codec you wish to install.

The other option is to install it via the terminal

You can read this on how to do that.

[https://help.ubuntu.com/7.04/musicvideophotos/C/codecs.html](https://help.ubuntu.com/7.04/musicvideophotos/C/codecs.html)
