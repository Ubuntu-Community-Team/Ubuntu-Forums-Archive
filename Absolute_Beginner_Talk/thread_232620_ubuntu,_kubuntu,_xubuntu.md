### Post by Sef on 2006-08-09
> The only difference is the desktop environment. KDE for Kubuntu, XFCE for Xubuntu, and GNOME for Ubuntu.


Not quite.  Both Ubuntu and Kubuntu are fully featured desktops.  Xubuntu is a lightweight desktop for those with computers that don't have the ram or cpu necessary to use Ubuntu or Kubuntu.  (Xubuntu can run on machines with lots of ram and fast cpus.)
