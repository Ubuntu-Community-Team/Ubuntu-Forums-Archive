It should have asked you just after you set the timezone I think. However, if you reboot and press esc when it says "press esc to enter menu" and counts down from 3, this will give you the option to boot into recovery mode. Do so and you will then be given a command line as root. Type in "passwd <your username>" Set your password. Type in reboot. Et voilà.
