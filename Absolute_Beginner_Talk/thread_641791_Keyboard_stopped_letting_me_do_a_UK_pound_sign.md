### Post by kazamx on 2007-12-15
HAHA,

Check it out ££££££ I fixed it.

It seems that all I needed to do was change it to United Kingdom and default in keyboard setup (System - preferences - keyboard) 

The problem was that for some reason the change to UK wasn't taking effect without a restart, even though other changes did. 

So problem solved

Leave the model generic
Add a new layout United Kingdom
Leave as default (not dvorak, international eyc. 
Select the new option and restart
