### Post by aysiu on 2006-10-05
Right-click file
Properties
Open With
Add 
Select program
Make sure black dot is next to the program

---

### Post by harlan on 2006-10-05
Right click over the file and select "properties". Then click on the tab "open with" and select the application that you prefer.
