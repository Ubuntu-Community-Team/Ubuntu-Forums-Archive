### Post by asimon on 2006-06-11
[QUOTE=Dr. Nick]
I dont think you will find one for amarok and gnome since amarok is meant for kde. Unless someone made a gdesklet for amarok control I wouldnt expect to see one.[/QUOTE]
I dunno if there is a gdesklet for Amarok, but there are superkaramba desklets, like for example [amraoKed](http://www.kde-look.org/content/show.php?content=40311). Superkaramba is the KDE equivalent of gdesklets. The package is called "superkaramba" and in the Universe repository. More Superkaramba desklets can be found at [kde-look.org](http://www.kde-look.org/index.php?xcontentmode=38). AFAIK superkaramba has also the ability to search for desklets and install them from within the application.
