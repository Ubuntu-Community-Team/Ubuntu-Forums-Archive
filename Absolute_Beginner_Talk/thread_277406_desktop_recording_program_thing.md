### Post by Irony on 2006-10-14
Download xvidcap_1.1.4rc2_i386_fixed.deb from;

[http://prdownloads.sourceforge.net/xvidcap/xvidcap_1.1.4rc2_i386_fixed.deb?download](http://prdownloads.sourceforge.net/xvidcap/xvidcap_1.1.4rc2_i386_fixed.deb?download)

You can find details from [http://sourceforge.net/project/showfiles.php?group_id=81535](http://sourceforge.net/project/showfiles.php?group_id=81535)

Simply double click it to install in Ubuntu. To run it type *xvidcap* in terminal, you can set up a button with *System Tools > Menu Application Editor*
