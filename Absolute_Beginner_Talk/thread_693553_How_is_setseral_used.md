### Post by dstew on 2008-02-11
setserial is only useful with a real hardware modem, like an external modem connected by a serial cable to your computer. For all other modems, you need to do some installing and configuring. [This page]("https://help.ubuntu.com/community/DialupModemHowto") is the standard place to start. You have to be very careful to read and understand it, and follow the directions.

Did you run the [scanmodem script]("http://132.68.73.235/linmodems/index.html#scanModem")? Please post the output on the forum if you can, and we can try to figure out if there is any hope.
