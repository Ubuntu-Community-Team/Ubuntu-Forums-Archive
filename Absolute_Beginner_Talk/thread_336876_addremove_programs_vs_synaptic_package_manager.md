### Post by netadptr0719 on 2007-01-12
Add remove kind of just gives you the basic packages that you would expect to come with a distro and a little bit more. Package manager pretty much gives you a list of all the packages available in all of the repositories. Basically, package manager has a LOT more.

---

### Post by rai4shu2 on 2007-01-12
Add/Remove lets you add things in general, like XMMS. Synaptic lets you add things in particular, like the flac plugin for XMMS.
