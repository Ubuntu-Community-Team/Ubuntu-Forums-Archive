### Post by murmelhunter on 2007-07-10
Well I have installed recently YDL on a PS3 and had to lear a K-Boot is working .-)

I think Kboot will behave similar with Ubuntu.

Boot your PS3 and wait until Kboot is offering you following prompt:

K-boot:

Enter here the following:

K-boot:boot-game-os 

hit enter and the PS3 OS will boot up.

View points to add:

The default OS is now PS3OS again you can boot Ubuntu only by selecting in the system settings  as default OS =>otherOS

To return t after a Ubuntu session o the PS3 OS you need to reboot until you see the K-boot prompt and enterat K-bbot:boot-game-os.
 ( Be aware that the prompt wait only 8 seconds and if no activity it will load Ubuntu)

It works well and no reset of the PS3 is needed.
