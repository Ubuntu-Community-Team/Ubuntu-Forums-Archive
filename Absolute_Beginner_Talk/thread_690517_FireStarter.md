### Post by Steveway on 2008-02-07
Firestarter is not a firewall!!! Keep this in mind!
Firestarte is only a GUI to configure IPTables.
IPTables is the inbuilt firewall that is allready running.
On a  normal Desktop-PC you don't have to configure anything firewall-wise.
If you are running a server then it is advisable but not on a Desktop-PC.
So, why do you want to run Firestarter?
