### Post by Sweet Spot on 2007-05-11
I use DVD Shrink under Wine constantly. Installing via WINE is very easy, and it takes no time at all to back up the ISO file. After the ISO is backed up to my desktop, all I do is right click on it, and select "copy to disc".   Gnome has a built in burner. 

That. Is. It. I've not had ONE problem burning DVD's like this, EVER. And I must have about close to 200 burned DVD's using this method (and growing all the time). The only time I've had a problem using DVD Shrink was when the encryption couldn't be broken. And in that case, DVD Decryptor whipped that problem lickety split. Decryptor does what the title emplies . It decrypts and then allows DVD Shrink to do its job. It's a rare event though. In fact, I've only had to do this one time. 

Use DVD Shrink, you'll not regret it. I too tried a bevy of native Linux apps for this, but they all failed miserably, and I don't have time for that nonsense. I'd love for there to be more advanced DVD editing/authoring software available for Linux, but I don't think they've reached that point, yet.
