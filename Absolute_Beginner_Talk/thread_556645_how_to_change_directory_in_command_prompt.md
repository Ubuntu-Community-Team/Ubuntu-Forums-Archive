### Post by Rui Pais on 2007-09-21
> **papermoon said:**
> okay when i put in this

"dir
Desktop    Examples               Music         Pictures
Documents  Firefox_wallpaper.png  NIGHTSKY.jpg  Power.jpg"

how do i use the cd command to get into desktop? sorry im noob :(

do:
```
cd Desktop
```

or with full path (and a shortcut):
```
cd ~/Desktop
```

another trick, type:
```
cd ~/Des
```
then press TAB key (auto completion should finish your path)

### Post by jviscosi on 2007-09-21
By the way, don't forget that Linux is case-sensitive, so **cd ~/desktop** won't work.  You have to match the case of the directory name.
