### Post by zerhacke on 2007-01-21
Shell scripts are ran in the terminal.  First you make it executable by typing the following into the terminal

```
sudo chmod +x shell.script.name
```

Then you execute the file by typing

```
 ./shell.script.name
```

Note the ./ before the shell script name.
