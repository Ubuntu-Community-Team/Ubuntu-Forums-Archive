### Post by jonnywombat on 2008-01-07
I've used 64bit from the get go with gutsy.. and I have had no real problems..

As for the flash java thing them simply follow this [http://ubuntuforums.org/showthread.php?t=202537&highlight=firefox+3in1+installer](http://ubuntuforums.org/showthread.php?t=202537&highlight=firefox+3in1+installer)

Use it to install swiftweasel and you can get a version which is optimized to run as 32 bit on ubuntu 64bit on your amd 64 bit processor. Best of all all the plugins work fine...

I have had no other issue except for managing my mobile phone.. last time I checked there was not a gutsy 64 bit package for the software I use, but it is always possible to force 32 bit software to run on 64 bit with a bit of terminal wizardry..There is a how to on this too but i can't find it ATM..

I say go for it...I was as you are now and have had no regrets from taking the plunge. I found it was a lot easier than most ppl think, I had no significant issues.. you could always dual boot for a while if you just wanna try it out.. and I feel my system is more responsive and there is a small but noticeable performance gain when compared to running 32bit on my rig..but i have not benchmarked anything to see if that is true.

Good luck



Jonny

EDIT: What was your issue with WINE?.. I have had no prblems with i at all in 64 bit although I did grab the latest version from wine HQ and not the version from the repos....
