### Post by TeoBigusGeekus on 2008-03-14
Then the only thing I can suggest is to remove and then reinstall Evolution.
Go to 
System -> Administration -> Synaptic Package Manager
and do a search for Evolution.
Right click the results and mark them for complete removal.
After you've done so, repeat the search and mark the results for installation.
I hope this helps...
