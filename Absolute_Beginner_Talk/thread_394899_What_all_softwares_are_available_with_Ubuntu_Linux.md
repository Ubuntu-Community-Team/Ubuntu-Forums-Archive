### Post by BrianK on 2007-03-27
There are literally thousands of software titles available to Ubuntu - most of them are online and easily installed via "Synaptic Package Manager" (which you can find by clicking "System > Administration > Synaptic").  To install *anything*, you simply  click the check box next to the app's name in Synaptic.  All dependencies are taken care of, the app will download, unpack, and install automatically for you.

It's a really cool system.

So, to answer your question, look in Synaptic.  Just about anything you can think of is in there, ready to be installed.

### Post by IYY on 2007-03-27
Ubuntu doesn't have *too* many applications installed by default, but allows very easy installation of software from online repositories. You just launch the Add/Remove or Synaptic programs, and browse through lists of software way too long to describe here. Select the software you want, click apply, and it'll be installed in minutes or seconds.

By default Ubuntu has The Gimp (a photo editor and art program like photoshop),  Totem (movie player), Rhythmbox (audio player and library manager like iTunes), Gaim (instant messenger compatible with MSN, Yahoo, AIM and most other popular protocols), a CD ripper application, CD burner application, Evolution (a mail and calendar application like Outlook)... You know, the basics.
