### Post by ruedu on 2006-09-15
No offense but you probably don't want to offer hosting to other people if you don't know the answer to your question.  I only say that because you need to get some experience handling a web server for yourself.  It's one thing for your own server to go down, but other people relying on you?  You need to know what you're doing and be comfortable.

That said, you can Ubuntu as a web server just fine.  You'd need at least the apache2 package.
