### Post by mellowd on 2007-11-28
It depends. There is a lot more going on than just clock speed. The new core 2 duo's are clocked at a lower speed but do a lot more per cycle, therefore they are faster.

It doesn't really matter if anything else is faster though. Are you able to do what you need to do? If so it doesn't matter

### Post by Lexyboy on 2007-11-28
Indeed, clock speed is only relevant comparing the same processor line.  Fortunately nowadays they seem to be settling around the 2GHz mark, so there's less silly comparisons as there were round when Pentiums were new and shiny.

Going by Moore's law you should get about 4x the power from a similar priced system now than you did 3 years ago, but as mellowd says it doesn't matter if it works :)  I was until recently using a 7yo machine (still works fine, but I need windows for my lab work).

(BTW, as an example I remember a lot of people upgrading Amigas from '030 @ 50MHz to '040 at 25MHz, which is ~3x faster!  And yes, still boots faster than Ubuntu on 2GHz machine...)
