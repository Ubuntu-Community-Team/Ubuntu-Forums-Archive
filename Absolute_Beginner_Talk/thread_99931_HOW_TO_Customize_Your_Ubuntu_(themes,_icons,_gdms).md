### Post by adduds on 2005-12-06
Hey all!

Themes for me are a very big deal, how a computer looks really enhances the experience. In Ubuntu changing themes is incredibly easy, but if anyone doesn't know how I'll show you how here. Everything in ubuntu is customizable, GDM (your login screen), icons, themes, and even your splash screen.

Before we get started I'll show you guys what I've done(please feel free to post yours aswell!!!! and share you tips):

Here are some screenshots of my desktop:

[Screenshot 1]("http://img228.imageshack.us/my.php?image=done3uf.png")

What do you guys **think???**

This is **what i used**:

I use the T-ish theme from gnome-look.org:

[T-ish Theme Link]("http://www.gnome-look.org/content/show.php?content=30859")

Etiquan GDM Theme:

[Etiquan GDM]("http://www.gnome-look.org/content/show.php?content=29677")

The OSX icon package:

[OSX Icon Package]("http://www.gnome-look.org/content/show.php?content=31618")

First go to [here]("http://www.gnome-look.org")

Select the themes, icons, and gdms you want, d/l them (I like to make a folder in my home directory just to keep things tidy)

**Themes:** System-->Preferences-->Theme

Click the install new theme find your theme package (you do not need to do anything to the package prior) and your done. Just select it

**Icons:** System-->Prefs-->Theme-->Theme Details and click the icons tab, in nautilus find your icon bundle click and drag it into the icons window and it automatically installs for you :). EASY HUH!

**GDMS:** System-->Administration-->Login Screen Setup and select the Themed Greeter Tab. Hit install new theme, find your GDM package in your directory select it, and VOILA TOUT FINIS.

Hope someone finds this useful!

EDIT: I also use gDesklets for the memory, cpu, and eth1 network activity in top right corner
