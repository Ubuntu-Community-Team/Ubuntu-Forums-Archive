### Post by landotter on 2006-01-28
Install both and see what you like, you can choose either a KDE or Gnome session from the display manager.

After you install Ubuntu, install Kubuntu by running synaptic from the system/admin menu, and install the metapackage kubuntu-desktop.

You can run any program in any desktop environment, but you might prefer the style, widgets, panels, file manager, applets and whatnot of one or the other.
