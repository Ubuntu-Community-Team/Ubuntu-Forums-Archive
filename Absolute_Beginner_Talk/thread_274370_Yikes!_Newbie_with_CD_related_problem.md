### Post by meng on 2006-10-09
This problem arises if you "burn a bootable disk". You need to burn from ISO, if you do it properly you should see a complete filesystem and it will be inherently bootable.
Caldera DR-DOS is produced by your CD burning software. It has nothing to do with Linux or the Live CD.
