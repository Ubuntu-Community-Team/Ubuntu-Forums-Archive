### Post by taurus on 2006-04-10
It depends on what you want to do.  If you want something stable like a rock with great apps support, then go with Breezy.  But if you want to see what the next release, Dapper Drake, would look like, then install Dapper Flight 6.  Remember, not everything is support in Dapper so if something breaks or doesn't work, you now know because it is still an alpha release.  However, it has been running fine on my crappy HP laptop so your mileage could be different...
