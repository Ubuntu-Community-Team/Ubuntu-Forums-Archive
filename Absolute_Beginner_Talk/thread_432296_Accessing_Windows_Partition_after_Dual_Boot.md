### Post by taurus on 2007-05-03
You need to use ntfs-3g if you want to write to ntfs filesystem.

[http://ubuntuforums.org/showthread.php?t=217009](http://ubuntuforums.org/showthread.php?t=217009)

And if you want to access Ubuntu (ext3 filesystem) from Windows, then you need to install fs-driver from [http://www.fs-driver.org](http://www.fs-driver.org).
