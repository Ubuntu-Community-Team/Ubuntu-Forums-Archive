### Post by tomcat1965 on 2007-03-22
Got foresight 1.1 and it's a great distro,fast(uses preload see sourceforge.net) Latest gnome,compiz by default with no need to log out and then back in to enable/disable
Icons and artwork superb,mp3 players and ipods automounted,beagle works perfectly and comes with mp3,wmv libdvdcss so all multimedia works out of the box. Also got OO 2.1 and firestarter.
One really cool feature is bieng able to havethe  root  desktop nested in a window.
Conary works by just upgrading the files needed to upgrade a version rather than downloading the whole binary. The first time it's run it will pull in a lot of updates but on future runs when updating or adding app's it's fast enough. Choice of command line or browser based gui.
Comes with a link to the #foresight channel on freenode if you've got a question,it will more than likely be answered nicely by a Dev. too!:)
