### Post by Tyke91 on 2007-06-14
it's pretty self explanatory.

1. Open up virtualbox
[[IMG]http://images3.pictiger.com/thumbs/41/9afe5016bb788642e7c80dc0b3031d41.th.png[/IMG]](http://server3.pictiger.com/img/1051243/computer-games-and-screenshots/-home-mykola-desktop-screenshot.php)

2. Press the "new" button
[[IMG]http://images3.pictiger.com/thumbs/b9/52b7aeaedb1467963560a5fdcbc966b9.th.png[/IMG]](http://server3.pictiger.com/img/1051244/computer-games-and-screenshots/-home-mykola-desktop-screenshot-1.php)

3. Choose a name for your virtual machine (and operating system) 
[[IMG]http://images3.pictiger.com/thumbs/a5/9963992743d7e60d9779f1b82635a9a5.th.png[/IMG]](http://server3.pictiger.com/img/1051245/computer-games-and-screenshots/-home-mykola-desktop-screenshot-3.php)

4. choose a base memory size (192 will do well for you)
[[IMG]http://images3.pictiger.com/thumbs/98/b543eeaea032d5aad2534f0110f54798.th.png[/IMG]](http://server3.pictiger.com/img/1051246/computer-games-and-screenshots/-home-mykola-desktop-screenshot-4.php)

5. select a boot disk (if you dont have one, which you probably dont, create a new one)
[[IMG]http://images3.pictiger.com/thumbs/75/1ecdc0512c1c5ed4b4f7b91e5205e075.th.png[/IMG]](http://server3.pictiger.com/img/1051247/picture-hosting/-home-mykola-desktop-screenshot-4.php)

==================creating a new disk=================

1. press the create new disk button instead of selecting one
[[IMG]http://images3.pictiger.com/thumbs/8e/3bd8f47d2d96719f93c9bd989d0be08e.th.png[/IMG]](http://server3.pictiger.com/img/1051248/picture-hosting/-home-mykola-desktop-screenshot-5.php)

2. choose between dynamically expanding and fixed sizes (i prefer dynamic because it wont always take up so much room)
[[IMG]http://images3.pictiger.com/thumbs/f2/b56f08ac882b801b0f413ee6533b53f2.th.png[/IMG]](http://server3.pictiger.com/img/1051249/picture-hosting/-home-mykola-desktop-screenshot-6.php)

3. choose the maximum room allowed on the virtual machine (leave enough space for windows!)
[http://images3.pictiger.com/thumbs/32/24569194f5b7d579375afba844303832.th.png](http://images3.pictiger.com/thumbs/32/24569194f5b7d579375afba844303832.th.png)

4. press finish
[http://images3.pictiger.com/thumbs/6e/dfea8c174136b419cdcdd76139bcb46e.th.png](http://images3.pictiger.com/thumbs/6e/dfea8c174136b419cdcdd76139bcb46e.th.png)

========================================================

6.  press next
[http://images3.pictiger.com/thumbs/db/a93eb39656ad838b2c45887693fbb4db.th.png](http://images3.pictiger.com/thumbs/db/a93eb39656ad838b2c45887693fbb4db.th.png)

7. press finish
[http://images3.pictiger.com/thumbs/23/878bc500eff99aab16189cd94c235d23.th.png](http://images3.pictiger.com/thumbs/23/878bc500eff99aab16189cd94c235d23.th.png)

8. select settings (to configure your box)
[http://images3.pictiger.com/thumbs/06/69aa092686673febd4dfb68cdc495106.th.png](http://images3.pictiger.com/thumbs/06/69aa092686673febd4dfb68cdc495106.th.png)

9. make sure anything like your CD drive and audio drivers are turned on/mounted... they are unmounted by default
[http://images3.pictiger.com/thumbs/dc/cfb2ef430b6be99594df4bfc5dcaa1dc.th.png](http://images3.pictiger.com/thumbs/dc/cfb2ef430b6be99594df4bfc5dcaa1dc.th.png)

post any questions here :D
