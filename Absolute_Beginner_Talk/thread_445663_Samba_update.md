### Post by starcraft.man on 2007-05-16
Samba is the networking service that Ubuntu uses. If you wish to fileshare or network any of your PCs together, you will be using it. You can find more info [here.]("https://help.ubuntu.com/community/SettingUpSamba") [wikipedia]("http://en.wikipedia.org/wiki/Samba_(software)") too if ya like :)

As for why? I assume they are security fixes/updates that add new functionality.
