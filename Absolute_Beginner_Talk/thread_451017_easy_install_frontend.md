### Post by starcraft.man on 2007-05-21
Theres also always the add remove thing in the menu, I don't use it much though.

If he's just starting Ubuntu, send him to [Ubuntuguide]("http://ubuntuguide.org/wiki/Ubuntu:Feisty") and [psychocats,]("http://www.psychocats.net/ubuntu/index") good places to learn the basics and get programs installed.

I do hope you weren't meaning automatix2... I really don't recommend that. Sure its convenient, but it has caused lots of breakage and problems with upgrades from what I've seen (not to mention, it sees downtime unlike the repos, and then what do you do?). Point your brother to Ubuntuguide for initial set up, he can find simple steps to install any basic/advanced program he needs and learn a bit about terminal in the process.
