Some games designed for windows will work and some will not. The typical install will leave windows in place so you can still get to those tricky games. WoW does work in ubuntu.

Movies and music are all fine, no problems.

It uses firefox. Firefox is available for windows too so you can try it there. It seems faster to me, and it has better features. The best feature of all is you will get NO spyware from surfing.

Yes, it uses a lot less resources.

Your hardware looks fine. The video card will work, but you may have a hard time fully enabling 3D. MS mouse is fine too.

Check out youtube.com and search for BERYL or compiz. You will see lots of videos of different things you can do. Even without beryl, you can change up the theme a whole lot. Just go to [www.gnome-look.org]("http://www.gnome-look.org").

BTW: if you boot the CD, you can try it without installing it, so at least you can try surfing the internet and more. However stuff like WoW and Beryl will not be possible until you do a real install.
