### Post by mar225 on 2007-02-15
Do a search on ip address conventions.  192.168.x.x  is allocated for lan  etc  not sure the 208.x.x.x is so probably no way to use it

in the browser address bar do a "about:config" without quotes  scroll to "network.dns.disableipv6"  highlight right click and change value to true

worked for me :)
