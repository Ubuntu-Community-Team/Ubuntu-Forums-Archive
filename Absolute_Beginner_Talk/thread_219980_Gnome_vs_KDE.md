### Post by Ptero-4 on 2006-07-20
You can switch to kde by looking for the packga kubuntu-desktop in synaptic (it's in main) and installing it. But be aware that it pulls up a large quantity of packages so if you're on dialup, you should get yourself the kubuntu CD and then you can either install kubuntu or add the kubuntu CD to the repositories through synaptic and then install the kubuntu-desktop package telling synaptic to fetch the required packages off the kubuntu CD instead of the web.
