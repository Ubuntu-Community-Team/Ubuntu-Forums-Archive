### Post by LancerDragoon on 2007-07-01
I've tried [Exaile]("http://www.exaile.org/"), and it's pretty good. Basically it's a project to re-do Amarok, but with GTK+ instead of using Qt. It has iPod support through a plugin, but I haven't tested whether it works since I don't have an iPod. 

Alternatively, you can try [Banshee]("http://www.banshee-project.org/Main_Page"). You can install it from the repos by typing this into the terminal:
```
sudo apt-get install banshee
```

Rhythmbox is the default player in Ubuntu which also has iPod support. If you'd like an external program to manage your iPod, you can use gtkPod. To install, type this into the terminal:

```
sudo apt-get install gtkpod
```

Oh, and do make sure you have installed the codecs necessary for gStreamer applications to handle MP3 files. 

Have fun!
