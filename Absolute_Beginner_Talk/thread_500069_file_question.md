### Post by Amplidude on 2007-07-13
In the file browser (Places>Computer) when you select a file click on Properties tab and select the emblem tab. There are a host of icons to choose from.

Hope this is what you were looking for.

### Post by zanglang on 2007-07-13
All Metacity themes are in /usr/share/icons. 'Human' is the Ubuntu theme, so you'd want to look under that for the default icons. For program images most of them would be in /usr/share/pixmaps.
