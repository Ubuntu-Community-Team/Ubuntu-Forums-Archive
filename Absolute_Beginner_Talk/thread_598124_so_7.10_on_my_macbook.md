### Post by shavital on 2007-10-31
I am a total Linux newbie. Running Macbook 15" 2GHz, 2GB-RAM, still Tiger 10.4.10. Parallels 3.0 build 5160. Couldn't install 7.10 because *apparently* Parallels has a problem with it. Installed 7.04, so far so good on a bumpy road (48 hours since install). As to the wireless connection, what I have done: System/Administration/Network- select  (mark if it is not yet marked) 'Wired Connection'. In Ubuntu's tool bar, there's an icon showing two small "displays". click on it once, you should get a blue spiral that will stop showing 'connected'. In Parallels, you should to to 'Devices' and make sure that 'Share Networking' is enabled. And don't forget, I am a newbie.
Good luck.
