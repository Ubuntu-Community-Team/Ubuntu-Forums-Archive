### Post by sawjew on 2007-10-23
If you install Gnome Art (description here [http://packages.ubuntu.com/cgi-bin/search_packages.pl?keywords=gnome-art&searchon=name&subword=1&version=all&release=all]("http://packages.ubuntu.com/cgi-bin/search_packages.pl?keywords=gnome-art&searchon=name&subword=1&version=all&release=all"))

```
sudo apt-get install gnome-art
```
 you will have a menu entry in System>Preferences to Art Manager which gives you direct access to everything on the gnome art web site.

You can browse by category and then install anything you like (wallpaper, icons, window borders, etc.) just by clicking on the install button.

Have fun!!
