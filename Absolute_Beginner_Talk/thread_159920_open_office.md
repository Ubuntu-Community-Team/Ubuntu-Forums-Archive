### Post by WoodyMahan on 2006-04-13
Are you sure you are saving the document as a Microsoft formatted file?  When you hit Save As you will get the window where you choose the name of the file, but you also need to set the type of file which is the drop down box right underneath it.  Choose .doc for word, .xls for Excel, .ppt for Powerpoint.
