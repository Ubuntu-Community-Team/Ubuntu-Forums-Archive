I just finished a fresh install of Feisty on my thinkpad t60p..(my 3rd time.. screwed some drivers up the first couple of times..).. and i've noticed the backspace when in a browser.. works as the page up key or something.. doesn't take me back or anything.. help?

---

### Post by compmodder26 on 2007-05-14
I'm assuming you are using Firefox.  If so, open the location "about:config" in Firefox.  In the "Filter" box, type "backspace".  You should only see one line that says "browser.backspace_action".  Change its value to "0".  That should sort it out for you.
