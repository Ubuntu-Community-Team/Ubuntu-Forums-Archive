### Post by philidox on 2008-01-31
I know how to add programs to the startup but as far as making them start to seperated desktops you got me on that.  Here is a start, so you can at least get them to startup when ubuntu' loads. 
Click on "System">>>"Preferences">>>"Sessions".  Click the "Add" button and for Name and Comment line it does not matter what you type but in the command like type in firefox and then click ok.  Just click "Add" again to add another program.  To make sure your typing the right name in for the command box just use the alt+f2 function and type in the program your looking for to verify thats the name you need to put in the command box in your session.

### Post by emarkd on 2008-01-31
You can have apps start automatically by adding them to your session in System > Preferences > Sessions.  As for pinning them to a specific desktop, I'm not sure.

If you want a  terminal always availabe to you easily, check out tilda.  It's a terminal that hides itself off of the screen and drops down when you need it.  It's modeled after the command interface in first person shooter games, if you've ever played them.  It's in the repositories.
