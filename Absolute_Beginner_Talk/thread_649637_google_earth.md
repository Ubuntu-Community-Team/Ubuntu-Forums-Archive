### Post by taurus on 2007-12-25
Head over to earth.google.com and download the .bin file, GoogleEarthLinux.bin.  Then, open a terminal and install it from there.

Applications -> Accessories -> Terminal
```
cd ~/Desktop
chmod 755 GoogleEarthLinux.bin
./GoogleEarthLinux.bin
```

p.s.  Before you download and install it, make sure you have installed a driver for your graphic card since you need to have 3D working first.
