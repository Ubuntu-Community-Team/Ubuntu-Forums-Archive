### Post by princemackenzie on 2006-12-04
Yes Kiba is just an application launcher, it does not function like the Mac OS X dock.  (Minimized applications do not end up there, for instance)

The Beryl expose-like feature is also quite good and well done.

As for limiting the size of maximized windows, that might be tough.  A hack (which will get the job done, but not well) is to manually fit the windows over it, as they will remember their positions.
