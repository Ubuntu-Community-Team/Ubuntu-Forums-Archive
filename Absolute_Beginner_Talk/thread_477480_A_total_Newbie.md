### Post by codypumper on 2007-06-18
First up, what hardware do you have? You could search around to see if there are any major problems with any of your hardware. 
Now, about Wine. Wine is not really something to be trusted to run all of your old programs on. There are some programs that run well on it, but other than that your really limited. But, there are a lot of linux alternatives to programs. What programs did you plan on running in Wine?

### Post by amazingtaters on 2007-06-18
First I'd burn a live CD and give Ubuntu a try that way, you'll be able to find out if your hardware works and all that good stuff. Like codypumper said, don't rely on WINE as it can be rather flaky. It's worth a try to get stuff running, but be prepared to have programmes not work as well or not at all, because that is a real possibility. I'd say try to find free software that will do the same thing as your windows gear does.
