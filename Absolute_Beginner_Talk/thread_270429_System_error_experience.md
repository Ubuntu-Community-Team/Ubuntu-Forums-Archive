### Post by mediax on 2006-10-03
This is just to let other newbies know about a bad experience turned good!

On booting Ubuntu, I received an error message to the effect that my operating system contained errors and that I needed to run fsck manally to fix them.

At this point I had that nasty feeling (based on my past Windows experience) that I was heading for a reinstall.  I ran fsck and received literally dozens of error messages.  I ran through all of these, hitting "y" every time I was asked if I wanted to fix the relevant error.

At long last I received the message to restart my system, which I did, fully expecting little to have changed.  I was delighted to be presented with a fully working system!  Nice one, Ubuntu. :D :D 

I think the message for all newbies faced with a similar problem is - don't panic, and give your system a chance to fix itself!
