### Post by mcduck on 2007-06-14
Some programs use that information. For example, OpenOffice takes your name from there so it can automatically insert your name and address etc. into text documents. Also at least X-Chat takes it's default realname from there.

You can also set your user imag there, if you use GDM theme with image chooser you'll see it in login screen.
