Went looking, and here's what I found:

There's Indywiki, a python app that you can install, create a menu-based launcher, and add to Gnome panel next to your dictionary. It works great, accompanying the Dictionary very well, though it's separate from the dictionary. 

Read about it here:

[http://indywiki.sourceforge.net/](http://indywiki.sourceforge.net/)

Download the python app in a tarball here:

[http://indywiki.sourceforge.net/download.html](http://indywiki.sourceforge.net/download.html) (page gives easy instructions for installing and running).
