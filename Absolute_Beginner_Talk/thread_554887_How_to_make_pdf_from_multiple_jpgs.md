### Post by rsambuca on 2007-09-19
Probably the best way would be to use Scribus.  It is a desktop publishing program that will let you lay out your jpegs, and then save them to pdf format.  An alternative is to use OpenOffice as it allows you to save files as pdfs as well.  You can install Scribus from the Synaptic Package Manager as it is in the normal repositories.
