### Post by jeffus_il on 2008-01-24
For a start:
[LIST]
[*]Install Ubuntu
[*]Set up two new users using the command line
[*]Create a file
[*]set permissions  on the file so that the other user[LIST]
[*]can only read the file,
[*]and then can also delete the file,
[*]and then so that the other user cannot see it at all.[/LIST]
[*]Each time check this from the first user.
[*]Post your problem each time you get stuck
[*]Once you finish this, go on to pipes and filters.[/LIST]
