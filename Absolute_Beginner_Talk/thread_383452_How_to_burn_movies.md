Here's what I use

[COLOR="Blue"]acidrip[/COLOR] - to rip DVD's to avi or mpeg file
[COLOR="Blue"]avidemux[/COLOR] - to edit video files
[COLOR="Blue"]devede[/COLOR] - to build a DVD from a set of files (avi, mpg, etc). The resulting DVD movies play fine in set top players.
[COLOR="Blue"]xdvdshrink[/COLOR] - to rip DVD movies to a blank dvd (xdvdshrink will shrink a dual layer DVD movie down to fit on a single layer DVD). xdvdshrink can be found [here]("http://dvdshrink.sourceforge.net/"). xdvdshrink comes with an installer that will tell you which apps you need to install before you can use xdvdshrink. The resulting DVD movies play fine in set top players.
[COLOR="Blue"]mplayer[/COLOR] - to view dvd movies and videos
[COLOR="Blue"]xine[/COLOR] - to view dvd movies and videos

I have had lots of success with the above apps on Ubuntu Dapper :)
