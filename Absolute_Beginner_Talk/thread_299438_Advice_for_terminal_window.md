### Post by CatKiller on 2006-11-14
> **nite owl said:**
> Im totally new to Linux I just installed Ubuntu and I cant run dvd's and a few other problems... and when I ask for help or look at other forum pages they all have the same concept of typing something into the command prompts

I've used [this page]("https://help.ubuntu.com/community/RestrictedFormats#head-38508785e53c611dde1859232189b2e823135eb9") to install the DVD library, and it's worked OK for me. Most of the time, if you're given instructions to type into the Terminal, you can normally just copy-n-paste from the browser window. You can find the Terminal at Applications -> Accessories -> Terminal.

Once you've got the terminal open, **man <*commandname*>** will give you the manual for that command. The man pages aren't very user-friendly, but they should get an idea of what people are asking you to do.
