### Post by jem7v on 2007-02-25
Here are the two sites I go to.

[http://art.gnome.org/](http://art.gnome.org/)
[http://www.gnome-look.org/](http://www.gnome-look.org/)

GTK2 themes are the "Controls" in your theme manager and Metacities themes are the "Window Borders."  To install them, just save the .tar.gz file from the website, open your theme manager, and drag the files onto the theme manager (like 720iD said).


For transparent stuff, you'll probably have to install Compiz or Beryl. If you want to know more about those, search the forum or Google.

### Post by Crooksey on 2007-02-25
[http://gnome-look.org](http://gnome-look.org)

Download "Metacity Themes" for window border themes.

Download "GTK 2.0 Themes" For application themes.

Download "Icon Themes" for icons themes.

You want to save the files to your desktop, dont extract them.

Open the theme manager then drag and drop those archives onto the screen to install them.

### Post by v8YKxgHe on 2007-03-01
Simply open up System->Prefs->Theme and drag/drop the file into there.

If it says invalid format, then extract the archive file and then you should see a folder which was in the archive. Simple copy that folder to ".themes" in you're home folder. You may have to go to View->Show Hidden Files, though.
