### Post by MegaJim on 2007-11-18
In general the only way to discover a hardware revision is to physically look at the model number on the component.

As long as you have access to a few versions of the driver for win2k and xp (you should be able to get these from the belkin site) there shouldn't be a problem with it.

One thing I would say is that if you have trouble with network manager (the default applet included in a ubuntu install) try WICD - it seems to work a lot better for some people, you can search this forum for instructions on how to do that.

Just jump in and have a go, there'll be people here to help if you have trouble.  Good luck.
