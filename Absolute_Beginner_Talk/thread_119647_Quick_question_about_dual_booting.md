### Post by sonny on 2006-01-19
Yes... you will have to install Windows first. When you install Ubuntu, it'll ask you to configure the harddrive (if you were to have only Ubuntu, then the automatic process will do just fine) do it manually to resize the WinXP partition, if you don't like resizing, I sugest you to let the free space when installing WinXP ready for Ubuntu; just make one partition without formatting it. Then Ubuntu will take care of the rest.
