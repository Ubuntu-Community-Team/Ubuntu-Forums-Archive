### Post by robenroute on 2006-12-30
You might want to try an old application (available [here]("http://sourceforge.net/projects/tsemgr/")); people have reported working connections with the K-series SE phones.

In addition, [gnokii]("http://gnokii.org/downloads.shtml") seems to also work with certain SE phones (see also [this]("http://www.yourse.de/index.php/topic,158.msg1414.html#msg1414") forum). I believe, gnokii is also available via the standard Ubuntu repositories.

Let us know how you're faring.
