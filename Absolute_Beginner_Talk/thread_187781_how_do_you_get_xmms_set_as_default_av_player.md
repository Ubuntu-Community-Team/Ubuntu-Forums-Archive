### Post by bluevoodoo1 on 2006-06-03
First check this page about mp3s etc. [https://wiki.ubuntu.com/RestrictedFormats](https://wiki.ubuntu.com/RestrictedFormats)

To get xmms to launch mp3s, try right-clicking on any mp3, choose properties, then 'open with.' Pick Xmms and it will be applied to all mp3s-- meaning all mp3s will be opened with xmms now.
