### Post by Qrk on 2006-02-04
It works great. With Fat32 you can actually fix windows, as you can write to it. With NTFS you can only get the data off, by burning it to a cd. Basically any live cd will work if it includes a cd burner. My personal favorite is SLAX, its fast and a small download. I like it because you can boot it in RAM, so if you only have one cd drive you can still burn things.  Just use the K3B package in the modules. Knoppix and Mepis are both great too.
