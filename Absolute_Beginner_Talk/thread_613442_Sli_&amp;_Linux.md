### Post by fain on 2007-11-15
I am using dual 7600gt's and a ASUS M2N32-sli delux mobo with nforce 590 sli MCP. Running Ubuntu Studio Gutsy in AFR. No problems yet, Knock on wood. I did have a problem in feisty with the repo driver in sli. X would completely lock up when id start a 3d ap in sli mode. It was a bug in the nvidia driver and i needed to upgrade manually with the driver on their site. But im using the repo driver on gutsy and it works fine so far. I have only played WoW a couple times though.

On my server though i have a 6200 with ubuntu gutsy and the repo drivers have a real problem with it and standard video play back. There are many threads on this problem and is a nvidia bug as well last i read.
