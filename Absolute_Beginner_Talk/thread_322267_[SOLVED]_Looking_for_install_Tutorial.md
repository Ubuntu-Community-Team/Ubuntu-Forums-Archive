### Post by ddtrust on 2006-12-20
Apt-get: [http://www.debian.org/doc/manuals/apt-howto/](http://www.debian.org/doc/manuals/apt-howto/)
Aptitude: [http://web.pdx.edu/~hegbloom/Aptitude/tutorial-intro-aptitude.html](http://web.pdx.edu/~hegbloom/Aptitude/tutorial-intro-aptitude.html)
Dpkg: [http://www.knoppix.net/wiki/Dpkg](http://www.knoppix.net/wiki/Dpkg)
Tar: [http://www.washington.edu/computing/unix/tar.html](http://www.washington.edu/computing/unix/tar.html)

Using google helps a lot! :D

or on command line:

For apt-get:

```
man apt-get
```

For aptitude:
```
man aptitude
```

For dpkg:
```
man dpkg
```

For tar:
```
man tar
```

And 'q' quits. ;)
