### Post by Ub1476 on 2007-10-26
On a dual-boot it's a little more to do than that, but if you want to just have Ubuntu on your computer, you can do that. Remember to burn on 4x speed so all data burns successfully. After that, reboot and you should get the live cd menu. Navigate to start Ubuntu and press enter. Ubuntu should start, and you can explore it without installing it. When you ar ready to install, just click on the install launcher on the Desktop. 

Good luck:)

### Post by Pumalite on 2007-10-26
To download your iso, torrents are better than HTTP or FTP
Do md5sum on iso before burn
Burn at 4x
Make sure you burn as 'Image' and not as 'data'
Set CD to 1rst in boot order in BIOS, boot from it and install.
