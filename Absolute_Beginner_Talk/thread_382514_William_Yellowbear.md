Have a look at:
[http://www.psychocats.net/ubuntu/installing](http://www.psychocats.net/ubuntu/installing)

You can have XP and Ubuntu on the same machine but you can only boot into one at a time. If you boot into Ubuntu, you can set it up so that you can access data files from the Windows partition (copy them over, for instance). But in Windows you can't access Ubuntu files (because Microsoft doesn't allow it).

I run my business from home using Ubuntu PCs. It's different to using Windows and you have to be able to take a more lateral approach - you have to learn a bit more to use Ubuntu but it gives you more control.
