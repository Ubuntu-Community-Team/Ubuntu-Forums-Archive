### Post by grsing on 2006-06-10
By trial program, do you mean the Ubuntu Live/install CD?  If so, make you're burned it as a CD image, not just burning the .iso file to the disk as a data file (when you open the CD from Windows, there should be many files on it, not one).  If so, restart your computer, and when it's booting up, it should tell you a key to press to go into the boot menu (often escape, F#, or Del, but it varies).  Choose to boot from CD, and it should be self-explanatory from there.
