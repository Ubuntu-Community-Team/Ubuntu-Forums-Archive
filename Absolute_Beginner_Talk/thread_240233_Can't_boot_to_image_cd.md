### Post by Polygon on 2006-08-20
some things to try:

make sure that the .ISO you downloaded is correct. in the download page, in the huge list, there was a text file named MD5SUMS. open that file, and find the one that pertains to the iso you downloaded, and then use a program like md5summer and make sure that the iso's md5sum matches the one in the list

burn the iso as a regular cd from iso. do not create a "bootable" cd.

make sure that boot from cd is listed first before boot from harddisk in your bios (i dont know where to find this though....)
