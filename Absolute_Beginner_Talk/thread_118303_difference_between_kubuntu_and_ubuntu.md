### Post by Sef on 2006-01-16
> that might be a really dumb question

It's not a dumb question.

>  whats the difference between Gnome and KDE?

Underneath they are the same.   It's how and with what they access what is underneath.  Yet it is possible to run Gnome programs under KDE and vice versa. Gnome is more minimalist and less able to tweak than KDE.   One is not better than the other one.  That is a matter of choice.  ME, I prefer Gnome, but try both and see which one you prefer.  You can run both on your desktop and switch them when you are in the log in menu: click on sessions.

To install the KDE:  sudo apt-get install kubuntu-desktop (if you have the Gnome dekstop.)

To install the Gnome:  sudo apt-get install ubuntu-desktop (if you have the KDE dekstop.)
