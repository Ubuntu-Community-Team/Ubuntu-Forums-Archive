I haven't had a PC with a floppy drive for a very long time. No need for one to try Ubuntu, it boots from CD:
[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

There are options to play Windows games under Linux. Wine, free of cost, and Cedega, requires a monthly fee of about $5, work for some games, but I've never been completely happy with either.

I dual boot with Windows XP so I can play games. I recommend you do that.
