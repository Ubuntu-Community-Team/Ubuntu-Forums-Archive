### Post by akiro.yamamoto on 2006-03-21
Now the following is only my humble opinion, it is not meant to start a flame
war. Nor am I trying to belittle the viability or of alternative desktop systems

The problem is that some of the lighter desktops may not give you that complete desktop feel. Gnome / KDE are complete desktop enviroments (XFCE as well, but to a lesser extent) and desktops like blackbox , fluxbox etc.
although light on system resources can also be light with usability features.and 
may require quite an investment in time and patient before you are comfortable
with them. 
      If this is your first time using linux / Ubuntu I would recommend staying
with the default Gnome Install or using the XFCE system, as these may be
easier for you to master. You can always install fluxbox or Enlightenment or whatever after you have become comfortable with your new Linux system and
start experimentation, until you find one that suits you.

The other thing is that although Gnome is heavier on system resources than 
Fluxbox when you start to use apps like Firefox and OpenOffice they use alot of
system resources as well. Switching to fluxbox may not dramatically improve the
response time for these apps.... so the benefits may very well be minimal....
Right now FF 1.5 is using 79MB of mem on my system. So with 128MB system 
ram you will most likely be using your swap partition on a regular basis, regardless of which Desktop you use.

Do you know/ have you tried  : [http://www.ubuntulite.org/](http://www.ubuntulite.org/) ?
  
  
> About Ubuntu Lite

The idea of Ubuntu Lite is to bring the power of Ubuntu across to the users of Legacy systems. So not just to make a small linux as this has been done before (Puppy Linux, Damn Small linux, the Rule Project). Nor to make a useable linux based desktop Operating system (that has been done with Ubuntu Linux, Beatrix) The idea is to bring the power of Ubuntu to the typical users of Legacy hardware (ie the grandma who gets an old P2 set up for her but does not know how to use it).

So to this end the Operating system needs to be:

Easily used by people with minimal computer skills 
Useable on a resurected system (see system specifications below) 
and is able to satisfy a typical users requirement of Applications and functionality. 
These are the three core aims of the Ubuntu Lite Project

We will also provide development guides to anyone who wants to trim down their versions or help others use linux on reconditioned hardware. I hope that you enjoy Ubuntu Lite and that you find it useful.
