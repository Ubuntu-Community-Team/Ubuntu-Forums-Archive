### Post by tatimmer on 2006-09-02
For anyone who has had difficulty mounting a floppy disc or usb pendrive, I have posted on my website a small program called mmrm = ManualMountRemovableMedia.  Its just a small gtk window with 2 toggle pushbuttons.  Connect your device and push the button.  Nautilus will start up and display the device contents.  See the README for details. The file is mmrm.tar.gz.   Hope it helps.
