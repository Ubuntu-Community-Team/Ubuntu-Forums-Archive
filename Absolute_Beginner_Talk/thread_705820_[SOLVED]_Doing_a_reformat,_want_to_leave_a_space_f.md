### Post by seventhc on 2008-02-23
Yes, if you leave 30 gigs for ubuntu, then you can install ubuntu on that partition.
When you install ubuntu on step 4 of the process you choose *guided* *use largest continuous free space*. This will install ubuntu onto the free partition and do the formating for you.
When you reboot, you should have a choice between windows and ubuntu.
