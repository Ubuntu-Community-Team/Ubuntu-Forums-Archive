### Post by cmnorton on 2007-12-08
Determining the best starting place would depend on your interests and the level/area at which you want to program. I came to Linux four years ago after working on Windows-based database fail-over modules. 

Now, I write code primarily in Informix 4GL, perl, C, with sprinklings of bourne shell, sed and awk. I have not so much changed the level of programming as the tools used. I've though working on user-interface code would be interesting. There is always MySQL and PHP, or Firefox plug-ins.

### Post by Taboo Mirage on 2007-12-08
Hello there.

I recommend that you take a look at the [Python](http://www.python.org/) programming language.  It'll teach you some of the principles of programming and good practices.  It's easy to learn and can be used in many situations, but is still a very powerful interperated object-orientated language.

There are some good, free e-books about Python programming too, including [Dive Into Python](http://www.diveintopython.org/) and [A Byte of Python](http://www.swaroopch.com/byteofpython/).  The Python site itself also has extensive [documentation](http://www.python.org/doc/) which you may wish to check out too.
