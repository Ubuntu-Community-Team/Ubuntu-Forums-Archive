### Post by mssever on 2006-08-24
I'm not positive, but I thik that having all the extr Gnome libs mostly means  more disk space is being used. Of course, the programs that use those libs might be using more resources than you want. Gnome/KDE automatically run a number of processes whenever you're logged in to them, and those processes probably aren't running.

If you're willing to compile your programs from source, you might be able to get them to use fewer resources. But compiling, especially compiling custom-tuned software, is somewhat advanced.
