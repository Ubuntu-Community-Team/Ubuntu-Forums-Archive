### Post by blastus on 2005-11-07
You cannot do everything with a Live CD. When you reboot your changes are lost because the OS is not installed. I can access the Internet over dialup with the Mepis Live CD because Mepis includes the kernel modules for my modem installed and ready-to-use with no configuration necessary. But I'm sure I wouldn't be able to do it with the Ubuntu Live CD, simply because with Ubuntu I need to compile and install kernel modules for my modem.
