### Post by ShodanjoDM on 2008-03-18
This won't matter much if you're the only user of your computer, but if you want to install it so that another users can use it too, then extract the archive (in your desktop for example) and run this command:

```
cd ~/Desktop
sudo mv <the name of the icons folder you just extracted> /usr/share/icons
```
