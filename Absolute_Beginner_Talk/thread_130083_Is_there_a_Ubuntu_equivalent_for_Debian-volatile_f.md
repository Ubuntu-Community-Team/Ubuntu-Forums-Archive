What is debian-volatile?
Some packages aim at fast moving targets, such as spam filtering and virus scanning, and even when using updated data patterns, they do not really work for the full time of a stable release. The main goal of volatile is allowing system administrators to update their systems in a nice, consistent way, without getting the drawbacks of using unstable, even without getting the drawbacks for the selected packages. 

This is useful for packages that require consistent upgrading like spam filtering and virus scanners.

Is there a Ubuntu equivalent for debian-volatile?
