### Post by finer recliner on 2007-11-28
i assume you have visual effects enabled.

install ccsm if you havent already

system > preferences > advanced desktop effects settings

go to "viewport switcher" and edit settings as nessary


(P.S. im not 100% sure he option is in viewport switcher. let me know if its not)
