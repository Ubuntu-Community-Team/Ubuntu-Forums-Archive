### Post by IDipSkoalMint on 2006-03-29
Get a software known as Partition Magic (it's farily inexpensive). When you run it, there's an option to add a new operating system. Select that and set up a Linux partition (I'd say at least 10GB) with the necessary swap file, but do so at the BACK of your drive (if you do it at the front, you could screw up Windows). Hit the "Apply" button and it'll restart and partition your drive for you. Once you're back in Windows, insert the Ubuntu install disc and reboot. Go through the steps until you get to the "partitions" section. Delete the two partitions you made using Partition Magic (this will allocate it simply as "free space"). Install Ubuntu on that, and you'll be golden. It will add a swap file, properly install the GRUB, and you'll be dual-booting in no time. Hope that helped.
