### Post by louis_nichols on 2007-03-14
The way to go about this is to read the user manual of the mobo and see what kind of ethernet chipset is uses. Then, run a search on google for an available driver for that chipset. Chances are the driver is already present in Ubuntu, but is not automatically loaded. In this case, you will be able to load it with the command

```
sudo modprobe *driver-name*
```

It can be configured to also be loaded automatically at boot time.
