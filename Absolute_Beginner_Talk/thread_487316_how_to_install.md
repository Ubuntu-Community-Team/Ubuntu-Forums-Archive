### Post by greg_g on 2007-06-28
For me, I just unpacked the tar.bz2 file in my little games folder under /home

Then, go to that folder and double click on the assualtcube.sh file and select "run in terminal"

if it means anything, I have version 0.93 of AssualtCube.


greg

PS: the command for extracting a tar.bz2 files is this:
tar xjvf filename

so in my case:
tar xjvf AssaultCube_v0.93.tar.bz2

That is done in the command line.  I have the AssaultCube_v0.93.tar.bz2 file saved in my /home/user/games/ folder, but you can put it where ever you want.
