### Post by Sef on 2006-01-11
You can have both Ubuntu and Kubuntu desktops on your computer.  If you install Ubuntu, then from the terminal type this:  sudo apt-get install kubuntu-desktop.  If you install Kubuntu, then from the terminal type this: sudo apt-get install ubuntu-desktop.  You can switch by clicking on the sessions button just below where you log in.  Try both and see which one you prefer.
