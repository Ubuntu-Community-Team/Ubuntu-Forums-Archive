### Post by bayouoldguy on 2008-01-28
So-far-so-good!  I am ready to install Ubuntu 7.10 to my old Compaq desktop (AMD Athlon 1.2 Ghz, 1.25 GB ram, 40 gig HD), running Windows ME !. I have followed the forum guides...
-downloaded & installed "MD5 Sums"
-downloaded "Ubuntu 7.10 Live desktop-i386.iso" to a file
-downloaded & installed "Infra Recorder"
-ran "Md5 Sums" (ok)
-burned C-D with "Infra Recorder" (ok)
-C-D runs in Windows ME
-Ubuntu 7.10 Live loads into running system, sound & LG LCD monitor 
 ok
-I got brave & moved DSL connection & USB Canon printer back to the
 Compaq & rebooted....How about that!, Ubuntu detected & 
 configured the internet connection without having to install any AT&T
 software, & found the printer.

I guess I am ready to install from the C-D w/ a dual boot set-up.
