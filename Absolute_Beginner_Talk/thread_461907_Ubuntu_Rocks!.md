### Post by Drate on 2007-06-02
There will be an option during the install process to do something to the effect of "resize hard drive and use free space"... you will want to choose THAT one.  Your windows will be preserved.  I would like to suggest you do a couple of things before  you jump in though:

1:  If you do not have experience installing operating systems, find a friend that does.

2:  Run the defrag utility in windows BEFORE you install Ubuntu.  It's under Start>All          
     Programs>Accessories>System Tools>Defragmenter

3:  And optionally...  if you are more patient than I... put together a list of your 
     hardware specs, like video card, network card, etc., and if you're smarter
     than me (which doesn't take much) just check for yourself or else ask one of the   
     people here to help make sure there won't be any major conflicts.

P.S. Ubuntu DOES in fact rock.  But you'll feel better about things if you go in more prepared.  I've had some hardware issues I wasn't expecting (mostly because I work on Frankensteined and hand-me-down computers)... it can be a bear to resolve.  But once it IS resolved... it's TOTALLY worth it.
