### Post by aysiu on 2006-05-09
Someone probably does.

I don't buy music from the iTunes music store any more. I don't like the DRM stuff. There's a way around them using a program called SharpMusique, but I haven't found it to be a very stable program.

For listening, I prefer AmaroK or JuK--they have global keyboard shortcuts, so I can change songs, adjust the volume, play/pause... all while using other applications.

For ripping, KAudioCreator is my program of choice at the moment.

For iPod transfers and organization, Gtkpod is ugly, but it's certainly reliable. Some other players like Banshee are supposed to have good iPod integration.

If you absolutely must have iTunes, you can run it in Crossover Office. I don't know about VMWare.
