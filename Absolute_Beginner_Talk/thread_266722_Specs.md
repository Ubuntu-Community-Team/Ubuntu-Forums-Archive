### Post by ewl1217 on 2006-09-27
Just to clarify that, Ubuntu includes a firewall called iptables by default. You can edit it manually, but that's fairly advanced. What you can do, if you feel the need to, is install an editor for iptables, such as Firestarter or Guarddog.

**EDIT:** As an afterthought, if the computer doesn't work well with Ubuntu, you could tru Xubuntu. Xubuntu uses the lighter XFCE desktop environment instead of the larger GNOME desktop environment.
