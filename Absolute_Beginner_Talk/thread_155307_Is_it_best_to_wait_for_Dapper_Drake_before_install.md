### Post by Buffalo Soldier on 2006-04-04
If this is your first step into linux, I would suggest for you to wait. Eventhough Dapper is quite stable already, but it's still in beta phase.
_situation 1_
Upgrading Service Pack in Windows = apt-get update in Ubuntu = software upgrade/security update

_situation 2_
Installing a new Windows (such as Windows XP from Windows 2000) = apt-get dist-upgrade = installing a new version of an OS

In both situation, Ubuntu or any other Debian based distro seems to handle things better than Windows.
