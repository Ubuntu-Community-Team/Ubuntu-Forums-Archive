### Post by mikeyphi on 2008-02-16
Yes you can do all of the above using Ubuntu programs. However, you can't use the same programs you use in windows - if you use Open Office you should be able to show your presentations under windows. The best thing for you is to try it before installing!
Download and burn the LiveCD - boot from the LiveCD, nothing is permanintly changed in your system but you can see what works.
Just an extra - you will need to install 'restricted drivers' for full DVD operation.

Also read the guide if you can: [https://help.ubuntu.com/community/](https://help.ubuntu.com/community/)

### Post by laidback on 2008-02-16
Welcome.

Briefly..
*I browse net [Using Firefox]*
Yes

*I listen to online songs, download them, and play them on my PC*
Yes

*I download movies and stuff from P2P softwares like LimeWire, and uTorrent[in many formats, avi, xvid, divx, mpeg, vob]*
I have used Frostwire although I believe that Limewire is available, never downloaded DVDs although I've messed with them off the DVD discs themselves. 

*I rip DVD's and make back up copies using DvD2One, And Divx Author, and so on.*
There is software for this but I cannot say how it compares with the Windoz versions

*I use Adobe Photoshop for playing with my pics*[/I]
You'll probably need GIMP, it's a standard package that comes with Ubuntu.

*I do some word processing work, and read e-books.*
Open Office is the obvious choice. I have found some problems in moving files from Office to Open Office (OOo), I find I cannot always print them. E-books, OK if they're MP3 or in an open source format, I have a friend that subscribes to a club for his e-books, to listen you need to load a bit of software that only works on Windoz, in this case obviously No.

*I some times need to make presentations which include, Media [Pics, Vids] *
Open Office Presentation should work for you here I think

*I use DVD writing software Nero*
K3b...fantastic and a match to Nero

*I also use loads of pendrives and for data transfer.*
I don't find any problems here, mine automounts, and can be moved to a Windoz PC at will.
[I]
For the presentation and word processing I used Office2003.[/I]
In my view Office has the march on OOo Writer but it just depends on how you need to use it. e.g. I find labels far easier in Office than OOo Writer, maybe I've missed something.

*Can I use plug and play pendrives? Mp3 Players?? Like iPod and stuff??*
Pendrives..Yes
MP3...Yes...I use Amarok, rip with Serpentine
iPod...I have a small one which I use with Amarok, read and write is fine.

There will be a lot to learn so don't expect instant results in every respect. 
If you install a dual booting system you can make the switch as slowly as you like. Whatever you do don't get behind with your college work.

Kindest Regards

Laidback
