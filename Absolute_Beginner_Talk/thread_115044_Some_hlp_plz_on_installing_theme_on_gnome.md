### Post by super on 2006-01-09
yeah icon themes don't work with drag and drop on my system either.

the fastest way to get them working is to extract the icon theme file to the icons folder (.icons) in your home directory. you may need to select 'show hidden files' from the view menu in nautilus to see the ~/.icons folder.

as for the menu addon, search your system for the file 'gnome-main-menu.png' and replace it with the icon from the theme. the file should be in /usr/share/pixmaps or something similar. you will need to be root to do this.

if you get messed up, give me a few hour and i will boot into ubuntu and tell exactly what to do.
