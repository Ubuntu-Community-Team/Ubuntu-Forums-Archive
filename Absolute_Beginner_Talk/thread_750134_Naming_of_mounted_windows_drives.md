### Post by ace007 on 2008-04-09
the easiest way to do it would be to boot to windows, go to my computer and then change the name of the NTFS drive to what you want it to appear in ubuntu.  If not that you should search for "ntfsprogs".  there is a ntfs volume label tool included in the package.  


EDIT:

Try this;

```

sudo atp-get install ntfsprogs
sudo umount /dev/hdb1 
sudo ntfslabel /dev/hdb1 windowsd
sudo umount /dev/hdb2
sudo ntfslabel /dev/hdb2 windowsg
sudo umount /dev/sda1 
sudo ntfslabel /dev/sda1 windowsf
sudo umount /dev/sda2
sudo ntfslabel /dev/sda2 windowsh
sudo mount -a

```

the above is assuming you want the labels to match the mount points.  If not you can change the label to anything by replacing the windowsd, windowsg, windowsf, and windowsh with whatever tickles your ubuntu
