### Post by Ben Sprinkle on 2006-10-04
All new people get this, this is the solution.
Go to system=>user manager.
Select the "show all users" button.
Go to your user add root to the groups, then make a new password for root itself. Right click your user and root and add all the options. Then goto groups and add your user to root and root to your user.
:)
