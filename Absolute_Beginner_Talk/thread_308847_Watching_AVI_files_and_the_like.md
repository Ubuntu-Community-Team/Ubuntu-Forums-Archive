### Post by NiklasV on 2006-11-28
Well there's VLC ;) and MPlayer, Xine, Totem...
Either go to Applications->Add/Remove, select "All available applications" on the top right and search for VLC, or get it with automatix (which can also install all the necessary codecs).
Automatix:
Edgy(x86): [http://www.getautomatix.com/apt/dists/edgy/main/binary-i386/automatix2_1.1-1.11-6.10edgy_i386.deb](http://www.getautomatix.com/apt/dists/edgy/main/binary-i386/automatix2_1.1-1.11-6.10edgy_i386.deb)

Dapper(x86): [http://www.getautomatix.com/apt/dists/dapper/main/binary-i386/automatix2_1.1-1.8-6.06dapper_i386.deb](http://www.getautomatix.com/apt/dists/dapper/main/binary-i386/automatix2_1.1-1.8-6.06dapper_i386.deb)

Just select open and install the package.
