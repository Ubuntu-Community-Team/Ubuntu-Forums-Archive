### Post by Brunellus on 2006-08-22
1)  for mp3 and other media support, please consult the RestrictedFormats wikipage

[http://wiki.ubuntu.com/RestrictedFormats](http://wiki.ubuntu.com/RestrictedFormats)

Ubuntu does not ship with these formats by default because (a) they are patent-encumbered, and offering support for them is legally-problematic in some juristictions; and (b) Ubuntu has a policy of using only Free Software by default.  

2) For adding new launchers to the GNOME application menu, use the Alacarte Menu editor.  It should be in the application menu by default.

As a general note:  don't use "shortcut."  Ever.  Microsoft has used the word ambiguously to refer to:

a) URLs

b) Bookmarked URLs 

c) program launchers on the desktop

d) a quasi-symbolic link in the filesystem.

Personally, I think it's a bad term, and discourage its use wherever possible.
