### Post by htinn on 2006-04-23
In GNOME, the window settings depend on the application itself (so most GTK apps will handle things like remembering your window settings). As for tiling, that isn't really easy to do in GNOME 2.12. In Dapper, however, you will be able to more easily adjust windows with respect to each other.

The behavior you're requesting is better achieved in KDE, which gives you a great number of window options.
