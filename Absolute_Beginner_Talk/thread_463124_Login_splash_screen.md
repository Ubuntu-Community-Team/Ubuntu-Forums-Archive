### Post by badguyanil on 2007-06-03
Well found this in some other thread............

The splash screen needs an external utility, so install gnome-splashscreen-manager
Code:
[CENTER]sudo apt-get install gnome-splashscreen-manager[/CENTER]

then you can see it under System->Preferences->Splash Screen. Thanks for the help.

   System > Administration > Login Window > Local
