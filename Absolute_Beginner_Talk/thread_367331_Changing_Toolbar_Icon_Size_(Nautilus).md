### Post by mcduck on 2007-02-22
It's not possible, but if you haven't already done this, go to System/Preferences/Menus & Toolbars and set toolbar style to 'Icons only'. This at least saves quite some screen space..

I also find the spatial mode to often use less screen space than browser view, so I've set Nautilus to use that by default and then I have a launcher for the browser view Nautilus in my panel in case I need it. (Spatial mode is when each directory has it's own window with it's own size, position and other preferences)
