### Post by adelie on 2007-04-28
I don't know about that specific card, But one thing I try when checking hardware for the first time, is I run 'dmesg' in a terminal right after plugging in the new device, if it is detected, it will give some indication of which driver it is loading, otherwise you will see either nothing, or a generic message like 'unknown device' 

Is it an 802.11 "wifi" card, or a cellular card? I've had good luck with both types. Also I always use a rule of thumb with wifi cards, "Linksys will never let me down". They seem to have excelent linux support and have yet to dissapoint me. 

Hope that helps.
