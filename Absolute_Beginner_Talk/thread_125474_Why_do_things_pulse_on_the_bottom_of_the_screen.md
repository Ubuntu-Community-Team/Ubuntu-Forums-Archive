### Post by bscbrit on 2006-02-04
It is intended to draw your attention to that particular program - perhaps it is waiting for an input from you, or it has just completed a task that you gave it.  Some programs also pulse when you start them up if they are obscured.  After all, why would you start a program if you didn't want to use it? :D
