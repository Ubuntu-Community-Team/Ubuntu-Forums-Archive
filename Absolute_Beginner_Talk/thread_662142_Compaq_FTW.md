### Post by Dngrsone on 2008-01-08
Try going into the BIOS and checking the memory in there.  The firmware sometimes has to be coaxed into realizing there has been a memory upgrade.

Also, use your Ubuntu live disc to run memtest and make sure there aren't any problems with the new stick you put in.

Well, mobile devices have this tendency to adjust the clock values according to the amount of perceived load.  This is an attempt to extend battery life.
