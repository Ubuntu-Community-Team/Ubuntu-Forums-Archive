### Post by Ub1476 on 2007-11-23
You can check if there are drivers for your graphic card in System>Administration>Restricted Drivers Manager. If there are, just install, and when installation finish, go there and check the driver. It will download and install automatically. 

If you don't find any drivers there, then tell us what graphic card you have. You can also post the output of this in the terminal, if you would like:

```
lspci
```
