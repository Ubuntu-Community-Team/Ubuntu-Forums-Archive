### Post by Bartender on 2006-10-15
Your processor is certainly up to snuff.  However, you'll want at least 512 MB of RAM for a standard Ubuntu or Kubuntu install to feel snappy.  Or you can run Xubuntu, which uses a lighter weight GUI.  
If you're not sure about your RAM and your Windows install is still working, d/l and install Belarc.  It's a small program that tells you a lot of neat stuff about what's going on inside.  It'll identify how much RAM you have.  
Also, some of the RAM manufacturers (Corsair, Micron, etc.) have diagnostic utilities on their websites.  They'll probe your PC and tell you how much you've got.  Maybe do both - run Belarc for all the neat info, then visit the RAM manufacturers.  They'll be more than happy to tell you the exact kind of RAM you should buy.  Maybe you can scrounge up some compatible RAM from a broken PC.
As long as nothing else is wearing out, such as the power supply or hard drive, it looks like you have an excellent candidate for Linux.

### Post by RKCole on 2006-10-15
Hello, Lina.

I know how to check a few specs concerning your computer which I will list below.  Hope these will be of some help to you.

NOTE:  You will have to enter these commands in a terminal.

To check your available RAM and Swap partition space:
```
free -m
```

To check your processor specs:
```
cat /proc/cpuinfo
```

To check Input/Output Devices:
```
cat /proc/ioports
```

To check the size of your hard disk drive (HDD) and how much space is free:
```
df -h /dev/<linux-partition (i.e. /dev/hda1, /dev/hdb1, etc)>
```

To learn what your IP Address is:
```
ifconfig
```

There are quite a few more, but I don't want to fill up this thread.  I'm not sure of any graphical utilities to accomplish this, but if any of you know of any please let me know.  This is just the way I am used to figuring out information.
