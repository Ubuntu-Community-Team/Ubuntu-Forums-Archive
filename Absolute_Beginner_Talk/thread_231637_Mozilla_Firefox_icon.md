### Post by Tux4 on 2006-08-07
You need to copy /usr/share/applications/firefox.desktop file to your Desktop directory(/home/yourusername/Desktop).

You can do it with thunar, or in command line.

Write it in a Terminal: 
cp /usr/share/applications/firefox.desktop ~/Desktop
