### Post by alastair on 2005-05-26
That's a hard call - there's a LOT around. Remember you have "man" pages e.g.

man ls

As well as a lot of OS/application docs installed under :

/usr/share/doc/

If you are a new user, it might be easier to start with a proper graphical desktop, and use a graphical web browser. The "desktop" system is just as capable of being a "server" as a non-GUI system. Then start to look through the web e.g.

The Linux Documentation Project 
[http://www.tldp.org/](http://www.tldp.org/)

Debian
[http://www.debian.org/doc/manuals/reference/reference.en.html](http://www.debian.org/doc/manuals/reference/reference.en.html)

etc.
