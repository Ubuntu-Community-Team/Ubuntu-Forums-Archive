### Post by kerry_s on 2007-01-27
Use the "add new item" and grab the top "launcher", now right click on that launcher and select properties. You can put the stuff in manually, but the easy way i found to do it is to open>menu>accessories>appfinder, now drag and drop what you want from appfinder to the program launcher, you can put as many as you want on that launcher or add more launchers. i usually have a launcher for the different things(web,files,etc..). For the things that aren't listed just input them manually, usually it's just the name of the app that is the command. You can test the command by pressing alt+F2 and type the name and see if it launches.Just play with it you'll get the hang of it in no time.

I forgot to tell you for the "add new item" you can drag the stuff straight to the panel where you want it so you don't have to click add and move it later.
