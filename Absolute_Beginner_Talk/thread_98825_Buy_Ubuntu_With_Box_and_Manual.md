### Post by d1337 on 2005-12-04
The partitioning part of the install has quite a few options, one of which will even resize existing partition (BACKUP ANY WANTED DATA FIRST).  The manual partitioning option should allow you to create as many as you like.  Most of the machines that I have dualbooted don't have huge harddrives to begin with, so I've more often than not added a second HD for Ubuntu (or other Linux distro) just to insure I had space to play with.  Some tasks require quite a large partition (around 80GB for decoding a DVD for burning) so make sure that if you want that functionality (or might in the future), you allow the partition for it.  Other than that, make sure you read up and understand where to flag as bootable, etc.

d1337
