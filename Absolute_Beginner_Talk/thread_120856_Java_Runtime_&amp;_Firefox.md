So you are saying that when java is running in firefox, the keyboard doesn't register when trying to enter text?  And the keyboard does work in all other instances excluding Java and Firefox?

If so I have had the same problem.  A couple of times I fixed it by opening up a terminal, tapping away on some keys then going back to firefox and tapping the keys in the text field again.  Sometimes that worked, sometimes it didn't.  In the end I disabled Java in Firefox and used epiphany-browser whenever I need to use Java in a browser.  It's not an elegant solution, but it works and that's all I am really concerned about.  What works. :)

If you work out what is wrong with Java and Firefox, I hope you can pass it on to me, as it seems to be a problem that only some people have.  I put it to someone in IRC that Java and Firefox don't always play nice together and it ended up being thrown back on me that it must have been me who misconfigured Java, because nothing is wrong with Firefox and Java.  I've done several installations of Ubuntu and configured Java using two different methods.  Each time the result is the same.  Java and Firefox don't like each other.  Looking around in google though, I think there are a number of people having issues with this type of thing.  Whether the developers of Firefox are doing anything about it, I just don't know.
