Networking between Ubuntu and Win XP works well with Samba and some testing may be possible with live cd before installing.

Partitioning can be done ahead of time from the Dapper live (desktop) cd or you can set things up during the installation, 3rd party software isn't needed.

There's no limit to the # of machines that a cd can install too.

While waiting for your Shippit arrival and testing you're current cd visit [this site]("http://www.psychocats.net/ubuntu/index.php") for some good advice and tutorials.
