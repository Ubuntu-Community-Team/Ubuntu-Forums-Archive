### Post by pizzutz on 2007-05-12
easiest way is from the terminal

Code:
```
sudo dpkg-reconfigure -phigh xserver-xorg
``` follow the prompts, and it will allow you to select which screen resolutions appear in the drop box.

alternately, you can manualy modify the entries in /etc/X11/xorg.conf

Standard disclaimer: backup your /etc/X11/xorg.conf file before making any modifications.

I hope you mean you are running version either 7.04 or 6.10.  you should NOT be running 7.10 since it is very early beta and not at all stable yet.

```
cd /etc/X11

The reconfigure program should have backed up your xorg.conf file  I don't remember the name that it backs it up to.
