### Post by pommattski on 2005-12-16
I found whole 'burning an image' confusing at first... burnt a few 'coasters' before I 'clicked'...

If you're using "Nero Express" to burn the .iso file you've downloaded, you need to select "Disk Image or Saved Project" on the main page.
In the dialogue box that appears next you need to make sure you select the file type as "Image Files (*.nrg, *.iso *.cue")" or you will not be able to find your Ubuntu CD image file.
Click "open" then go on to burn the disk as usual.
