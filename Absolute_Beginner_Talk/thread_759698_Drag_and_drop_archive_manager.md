### Post by ad_267 on 2008-04-19
Assuming you are using GNOME, right click on your menu and select "Edit Menus." Click on Accessories and tick the box next to "Archive Manager." Click Close. Now you can go to Applications - Accessories - Archive Manager to create an archive by dragging and dropping files.

I'm not sure what you mean by using the path. If you're using Nautilus you can just select all the files you want and right click and select "Create Archive"
