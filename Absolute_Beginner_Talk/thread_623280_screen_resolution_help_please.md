### Post by skyjacker on 2007-11-26
> **wagglez said:**
> I'm new to linux and have a problem with my screen resolution, which wont allow me to set the screen resolution above 800x600. I have an old nvidia geforce2 vid card and have installed the glx legacy drivers for it. Any advice, suggestions, etc?

*note: previously had a windows os installed, i know vid card is fully capable of handling higher screen resolutions

Try this Procedure:

1) Back up the present xorg.conf
Code :sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf-backup

2) Stop gdm or kdm
Code: sudo /etc/init.d/gdm stop (or kdm for kde)

3) Edit xorg.conf
Code: sudo dpkg-reconfigure xserver-xorg

4) Select the resolutions you want available by arrowing down to each desired line and pressing the “space bar”

5) Validate your choices with “enter”

6) restart X with Control+Alt+Backspace

7) Choose the desired resolution under the “Systems/Preferences/Screen Resolutions drop down list

NOTE: if you don't need to add any additional resolution sizes, skip # 4 & 5.  The reason you can't get higher than 800 X600 is because thats all that is in your x-org file. Olny add the ones your monitor can safely handle.

Good Luck
