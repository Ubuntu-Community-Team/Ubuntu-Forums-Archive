### Post by finer recliner on 2008-02-14
not familar with a pps, but this is the process to set a default application for most file types:

1) fine the file in nautilus. 
2) right click > properties > "open with" tab
3) click the add button
4) find openoffice.org presentation (or use the custom command "ooffice -impress")

remember to set openoffice presentation as the default in the properties menu, and you're done :)
