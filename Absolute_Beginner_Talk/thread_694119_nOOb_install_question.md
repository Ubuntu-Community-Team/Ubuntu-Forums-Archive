### Post by Rocket2DMn on 2008-02-11
Firefox comes preinstalled in Ubuntu, you don't have to download it.
As far as installing programs goes, most stuff you will ever want is in the repositories so you can access it from Synaptic: System->Administration->Synaptic Package Manager.  Or you can use apt-get which is the command line driven package manager (synaptic is built off of it).
What you downloaded is probably the source code that needs to be compiled.  For more info on that, see here: [http://monkeyblog.org/ubuntu/installing/](http://monkeyblog.org/ubuntu/installing/)
