### Post by forrestcupp on 2008-02-09
Beryl is a compositor that is an add on to Ubuntu.  Beryl can still be obtained from [their web site](http://www.beryl-project.org/), but it is a dead project.  They merged with Compiz to become Compiz/Fusion.  Compiz Fusion is included by default in Ubuntu 7.10 and it is called Visual Effects in System->Preferences->Appearance.  You may have to set up your video card to be able to use the effects, though.

### Post by billgoldberg on 2008-02-09
Beryl is old, it's new version is "compiz fusion".

It's just a program install on most linux distro's.

In ubuntu 7.10 it is installed by default.

If you want to use it to it's fullest you will need to install an extra piece of software.

Go to system -> administration -> synaptic package and search for 'compizconfig'.

Install the first one you find.

Then go to "system -> preferences -> apperance". On the "visual effects" tab, choose custom".

Then go to 'system -> preferences -> advanced destkop effects settings', there you can enable the effects.

The effects will load up everytime you reboot, so you can use the all the time.

You will need to have a graphic card that supports it (most ones do). 

If you have any questions post them on this forum.
