### Post by lisati on 2007-07-11
I, too, initially had trouble using the "standard" installation disk to install Ubuntu on my laptop (from a different manufacturer). You might want to try using the "alternate" disk. On the download page at [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download), just below where it says "Start Download", there's a little check box where you can choose to get the alternate CD.


p.s. don't be "dissed" by dis-ability - you're probably a lot more en-abled and clued-up than you give yourself credit for.
