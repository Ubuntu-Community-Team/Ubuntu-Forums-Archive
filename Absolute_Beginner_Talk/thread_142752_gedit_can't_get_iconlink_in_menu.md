### Post by takayuki on 2006-03-11
yep, that got it.  thanks.  

i right clicked, selected Create Launcher, then typed "gedit" into the name and command fields.

once i figured that out, i clicked Applications > System Tools > Applications Menu Editor, then clicked New Entry, then again, entered "gedit" in the name and command fields.

cool.
