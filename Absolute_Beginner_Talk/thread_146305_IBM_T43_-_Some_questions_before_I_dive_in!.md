### Post by Lux Perpetua on 2006-03-18
I have a similar computer. 

IBM Access Button - depends what you want it to do. In Ubuntu, you can bind the key to behavior of your choice, so I'm not sure what you have in mind. It won't run the same "Access IBM" program as Windows XP though, if that's what you mean.
R & R - not sure what this abbreviation means.
Fingerprint Reader - As far as I know, it's not currently supported on Linux.
Wireless - Ubuntu configured the on-board wireless card flawlessly on my T43p.

To get a feel for it, if you haven't done so, I recommend trying out the Live CD before jumping into the installation.

I hope this helps you in some small way...
