### Post by u.b.u.n.t.u on 2006-06-08
In theory you can seamlessly upgrade ubuntu by this method. In practise, you get the benefit of a fresh install by doing it manually.

So both methods work, both have advantages and disadvantages.

update install
pro - seamless fast migration to latest version.
con - sometimes something may not quite work (rare).

fresh install
pro - nothing on the box except the latest version.
con - time consuming.

One approach may be to alternate between a fresh install and an update install. So once a year a fresh install is done.

It really is a personal choice.

Yes it is safe.

Yes always back up your important files - remember, it is best to fix a leaking roof when the sun is shining!

Dapper is awesome.
