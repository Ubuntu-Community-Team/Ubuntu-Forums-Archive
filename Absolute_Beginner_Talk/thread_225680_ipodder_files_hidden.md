### Post by DarthMandeep on 2006-07-30
In Unix all files or folders that start with "." are hidden, somewhat like hidden folders in Windows. If you are using Gnome click on Places, go to your Home folder, then click View and select "Show Hidden Files" to see all files beginning with "."
