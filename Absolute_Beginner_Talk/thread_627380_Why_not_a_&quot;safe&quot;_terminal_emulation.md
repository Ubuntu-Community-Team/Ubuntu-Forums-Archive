### Post by Linewbie on 2007-11-30
This may seem like a newbie idea (that's probably already been requested or done years ago) but why not hide the terminal, and make a pseudo terminal that won't allow the most obvious dangerous commands (and any later additions) and at the same time allow plain english commands.
The terminal will still be accessible if you enable it, but the pseudo terminal would be the default terminal. If it was harder to do stupid things, and a little easier to use the terminal I think it would benefit the linux community in the long run.

### Post by CatKiller on 2007-11-30
That's pretty much what **sudo** is for. Without elevating yourself to root status, you can only do things in your own Home folder.

Just don't use sudo unless you know what the command does.
