### Post by Niniel on 2007-11-19
Yes and no.

You can download a new desktop a/o window manager (WM) and install that, and delete whichever one you don't like, but each distribution comes with some software that you won't get that way. Which may not be a big problem; all I'm saying is that they are not identical except for the presentation. 
But once you find a desktop/WM that you like, you can always install the full package from scratch.

Btw, there's also Flu(x)buntu, which uses yet another WM and is - if I understand this correctly - generally more "lightweight" as far as bundled applications are concerned.

Good luck.

### Post by newlinux on 2007-11-19
You can install an alternate desktop with supporting apps:
```

sudo apt-get install kubuntu-desktop
sudo apt-get install ubuntu-desktop
sudo apt-get install xubuntu-desktop

```
choose the ones you want.

You can then even choose your display manager (xdm, kdm, gdm) and which session you log into (you don't have to remove any of them and you can log into whichever you want whenever you want).
