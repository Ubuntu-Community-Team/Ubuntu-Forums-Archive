### Post by Qrk on 2006-03-18
It is important to make sure new users are aware of support offered. But even more important is some kind of demonstration of Linux that provides motivation to stick with it.  

For instance, if I were a new user and started with Breezy today as my first linux distro, I'd go back to Windows. This is simply because Breezy didn't detect my default screen resolution or my internet connection. 

Now its easy to fix those problems, but if I were a new user, I wouldn't even know one could fix them. Not having used Linux before, I wouldn't be aware of the advantages, and I'd have no reason to stick to it. 

Unfortunatly, these problems are the hardest for the community to address. There will always be some people with hardware support problems, (even windows has problems supporting some hardware) and there will be people who won't know how to begin to fix them.

I think that is the key... once someone finds Ubuntu works, they'll be excited and start learning. Thats where we can start to help them, but it is the simple problems with the easy fixes that get users to that point.
