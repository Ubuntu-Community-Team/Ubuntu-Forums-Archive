### Post by dbott67 on 2007-06-10
> **Gmbrdilos said:**
> I am about to build a new system and I am going to set up a dual boot.
Should I install windows first or Ubuntu?

I always recommend installing Windows first.  Linux is quite happy to co-exist with other OSes, but MS is a little predatory of the MBR.  If you install Linux first, Windows will overwrite the MBR, wiping out Grub.

> **Gmbrdilos said:**
>  Also: are too many things going to brake if I decide to go with 64 bit version?

64-bit Windows - Steer clear.  
64-bit Linux - You should be okay, although if you want maximum compatibility and support, go with 32-bit (just my personal opinion).

-Dave
