### Post by mostwanted on 2006-04-11
GNOME is an acronym for GNU Network Object Model Environment - not that that really means anything :)

I think Miguel De Icaza and the others just thought it had to start with a G (so GNU could be embedded into the name) and Gnome sounded good so they created a vague acronym for it.
