### Post by Cypher on 2007-04-26
The Linux OS is safe from viruses in general. So you will not find any Anti-virus software. Most viruses do hacks with Windows registry to cause trouble, and Linux has no such thing.

Also, since you're running everything as a regular user you can't cause harm to system files without going through SUDO, so you're OK.

If you do a quick search on Google, you'll find countless pages describing the many useful Linux commands..

Hi. Open **Add/Remove**, and in the "Show" menu select "All available applications"; then do a search for **virus scanner** and you will be presented with at least two options: **Aegis** and **Clam**. They're in the Universe repositories, so make sure you have them enabled (either through "System-->Administration-->Software Sources" or "System-->Administration-->Synaptic Package Manager"). Also, you can find **F-Prot** in Synaptic. Note that F-Prot is command-line only, as is Clamav if you install it from Synaptic and don't install its frontend, **clamtk**. Installing them from Add/Remove will give you the graphical interfaces.

You can install an AV if you want, but it really depends on whether you want security or speed. In my experience, AV tends to slow down systems.
