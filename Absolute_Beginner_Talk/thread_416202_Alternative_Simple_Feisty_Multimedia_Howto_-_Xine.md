### Post by arron on 2007-04-20
I see there is a Feisty howto for multimedia, I would like to add this to the list (maybe sticky worthy ;-) ) for others to use Xine as the default media player.

Go to a terminal and type:

```
sudo apt-get install totem-xine libxine-extracodecs
```

This will use xine for the default media player.  I like this player and I have had better luck with different media clips that I could never get to work before.  This will give you DVD, DIVX and other formats to watch.

This worked for me out of the box, I hope it helps others.
