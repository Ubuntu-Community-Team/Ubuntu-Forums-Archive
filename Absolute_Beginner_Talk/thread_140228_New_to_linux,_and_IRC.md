Using GAIM, just set up another account for the ubuntu channel (as an example) as follows:
Menu TOOLS then
Menu Accounts then
Option Add
Protocol IRC
Screen Name (enter the nickname you want to use)
Server (fill in the irc server you want to connect to {irc.freenode.net should be already filled in as default})
Leave password blank
Leave Alias blank
Save
Then, in Accounts box, just choose online and you should connect to the IRC server irc.freenode.net. A standard GAIM conversation window will open when you do connect, telling you you are logged in to the freenode network. In the section at the bottom where you ordinarily type your conversation in the window, just use a slash command to join the ubuntu channel (which would be /join #ubuntu). That should get you connected and online.
