### Post by GrueTamer on 2007-05-01
I learned about 3/4 of what I know about the terminal from the [Gentoo Quick Install Guide]("http://www.gentoo.org/doc/en/gentoo-x86-quickinstall.xml").  While it looks intimidating, it's actually a really good way to see how the terminal works.  It has tarring things in it, using ssh, using chroot, timing actions, wget, and much much more.  If you have some spare time, just read it through and think about how it all works...I benefited greatly from that guide, even if the system it installed wasn't what I wanted.
