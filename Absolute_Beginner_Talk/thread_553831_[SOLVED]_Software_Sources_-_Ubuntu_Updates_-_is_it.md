### Post by hyper_ch on 2007-09-18
Basically when Ubuntu does a new release (every 6 month) then the programs will be frozen at their state... this means you will not get newer versions of that program except for security fixes.
The backports however are newer versions from newer Ubuntu releases that were "back ported" to your "old" ubuntu release.
