### Post by shad0w_walker on 2008-04-03
You don't need antivirus. Linux is designed to stop unauthorized programs from running in the first place.
You don't need antispyware, again stopped by proper security design
You have a built in firewall in the form of iptables.

Basically put, what is routine maintenance to keep a windows system running really isn't needed because Linux is designed to stop it before it happens. Defragging also isn't an issue.
