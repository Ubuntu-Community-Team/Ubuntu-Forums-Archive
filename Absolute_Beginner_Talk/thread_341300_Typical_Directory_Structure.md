### Post by punx45 on 2007-01-18
yep.  the /home/<username> directory (often referred to as ~/ ) is approximate to the MyDocuments. that is the path that the GUI elements in Places > Home Folder
 from there on its up to you to customize it how you want. if you want folders for documents, music, photos, etc you will have to create them yourself.   

also, you will find the files saved to your desktop in /home/<username>/Desktop (or ~/Desktop) (since helpful to know since that is where firefox downloads to by default.

and finally, you don't *have* to put files in the home directory.   if you have additional drives you can put them there too.   drives that mount automatically are usually found in /media

i would caution you though, you should probably stick to storing things either in ~/ or /media/<device>/ unless you know for certain it is safe to do so.   of course, permissions should prevent you from doing anything too destructive ;)
