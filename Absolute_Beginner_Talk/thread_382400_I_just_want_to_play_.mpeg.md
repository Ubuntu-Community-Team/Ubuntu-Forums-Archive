### Post by Native Dialect on 2007-03-12
Step by step and as deep as I can get

1-Go to the System tab in the start bar and select "Admministration"
2-Select "Synaptic Package Manager"
3-Go to "Settings" then select "Repositories"
4-Make sure that you check "Universe" and "Multiuniverse" then exit that window
5-Click the search tab in the "Synaptic" menu and type in vlc
6-You should see the VLC Media Player as one of the available options. Right click on the file and select "Mark for installation"
7-Click the apply tab and the file should download
8-Go to the Applications tab in the start bar and select "Sound and Video"
9-VLC Media Player should be there.

This player can play DVD's, AVI, mp3,mp4, ogg vorbis etc etc. It is essentially Winamp on steroids, but without the fancy interface. Enjoy your mpeg movies.
