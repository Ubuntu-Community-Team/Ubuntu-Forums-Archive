> **Gam3Junki3 said:**
> So as soon as I finally work out one issue i start on my next, this is a fun OS :lolflag: I like it though.

System--->Preferences--->Themes

There will be a few themes in there.  If you want you can install a different theme.  Remember that when you download the theme to drag and drop the compressed archive do not decompress it.
