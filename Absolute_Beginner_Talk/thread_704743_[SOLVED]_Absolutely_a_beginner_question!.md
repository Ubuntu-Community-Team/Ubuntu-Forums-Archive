OK, looks like you have the Live CD.

First thing you want to do with that is the third Option when it fires up, Check CD for Defects.

Next: Install.  If you are sure you want nothing else off the Windows Partition except your images, then go for it.  The Install process will come to a point where it asks you if you want to use the "whole" drive ... it will format it properly if you say yes, and Windows will be GONE!  No going back!

Curious as to why you don't want your email or bookmarks?
You can get them and burn them to a CD too if you want them. Let me know I'll tell you how.
