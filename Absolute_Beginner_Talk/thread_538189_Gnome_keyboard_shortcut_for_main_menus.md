Let's say I start up Gnome without a mouse and I want to navigate the main menus (usually on the top, labeled 'Applications', 'Places', and 'System'. ---

### Post by loell on 2007-08-29
```
**alt + F1 **
```

you can then navigate by arrow keys, and tab key
