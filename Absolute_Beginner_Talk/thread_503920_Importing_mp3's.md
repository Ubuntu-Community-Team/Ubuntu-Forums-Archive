### Post by pistcivet on 2007-07-18
You really should consider using ogg vorbis,
better sound, smaller file size, free (open) codec.

However if you're unwilling or unable to use ogg,
this link will tell you all you need to know:
[https://help.ubuntu.com/community/CDRipping](https://help.ubuntu.com/community/CDRipping)

### Post by tgalati4 on 2007-07-18
I like Freespire's LSongs.  It automatically rips to 192 kbps VBR and does a decent job over a SAMBA share creating directories by artist and album.  It has an import mode where it will automatically start ripping a CD that's inserted.  Be mindful that it's a two-step process:  copying the tracks to wav files, then compressing the wave files.  You need plenty of disk space and time.
