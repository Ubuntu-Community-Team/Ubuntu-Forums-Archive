### Post by ibutho on 2008-04-13
> **mar225 said:**
> Thanks but that doesn't tell a poor idiot much.

What does  ~./ mean ?

~/ is a shortcut for your home dirctory. If you were working in a terminal and entered "cd ~" or "cd ~/" you would be returned to your home directory. A file or directory which is prefixed with a dot is a hidden file. To see it, you need to run "ls -a" or enable Nautilus to show hidden files.
