### Post by candtalan on 2007-05-25
It may be that the usb suport for that unit is limited.
I see that the TP-Link TD-8811 unit does not only have a usb connection to the PC but it also has an ethernet  connection which is the best one to use. 

USB has some security issues and I would strongly reccommend to use the ethernet connection to the PC if you possiblty can.

Using ethernet to the PC will make it very easy to configure, if not automatic.
