### Post by hoogland on 2007-12-11
You need to run a wizard that should have come up the first time you run Bibus.

It allows you to open UNO connections to OpenOffice. Make sure that when the Bibus setup wizard opens an OpenOffice document, you enable macros and then click on the accept UNO connections button in that document... If this does not help let me know and I will try to help further.

### Post by shuttleworthwannabe on 2007-12-11
Hi Thanks for the help;

I fired Bibus from the applications menu; and it performs online searches just fine. I have also gone through the Help| First Connection Wizard> and Activate|accept UNO pipe, the close OOo (including Quick Starter). Close Bibus.

NOTE: While in OOo Writer|Tools|Options| OpenOffice.org|Security|Macro Security>set security level to low (for now).

It worked fine. Thanks so much.
