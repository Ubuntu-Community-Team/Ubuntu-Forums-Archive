### Post by dndrich on 2007-10-13
I don't think so.  It is odd that Evolution does not allow for the separate accounts to be separate, but it doesn't.  The work around for this is to use a filter.  Make a new folder for the mail for one of the accounts.  Then go to the filters and make a new filter to apply to the one account only.  Set the filter up to send all incoming mail to the new folder.  That will do it.  I set my brother's Evolution that way because he had two accounts he wanted to keep separate.
