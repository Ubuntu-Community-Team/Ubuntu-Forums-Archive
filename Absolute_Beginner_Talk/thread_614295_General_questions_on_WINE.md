### Post by GavinZac on 2007-11-15
wine treats .exe's in the way that GIMP treats images. it will open and run them when you want to open the exe, and will shut down when you're done. theres no hit.

You do "install" your average program and run it from a specific WINE shortcut, because those have to interact with the WINE registry. however, plain executables will run without that bother.

some guy even tested to see if he could get windows viruses to run via WINE, without much success im afraid :D
