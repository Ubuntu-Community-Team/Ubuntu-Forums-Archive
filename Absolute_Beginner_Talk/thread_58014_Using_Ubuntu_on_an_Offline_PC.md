### Post by Kvark on 2005-08-20
Keep in mind that Ubuntu comes with only one CD. This means you'll get most of the programs that are not installed by default from the net. A distro that comes with loads of CDs with lots of programs on them would be better for an offline PC.

You can download .deb files from Ubuntu's servers with another computer to install on your Ubuntu machine but it is a bit more work to install downloaded files then to install online by marking checkboxes for the programs you want and click apply in synaptic.

This is probably the issue whoever said Ubuntu isn't for offline machines was thinking of. It does work but it would be better to have lots of programs on CDs.
