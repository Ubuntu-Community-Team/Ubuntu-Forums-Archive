### Post by Nano Geek on 2007-12-18
Beryl has been discontinued now that it has been merged with Compiz.
Compiz however is included with Ubuntu and to run it type:```
compiz --replace
```

To configure Compiz, you need to install the Compiz Settings Manager.```
sudo apt-get install compizconfig-settings-manager
```

Enjoy. :)
