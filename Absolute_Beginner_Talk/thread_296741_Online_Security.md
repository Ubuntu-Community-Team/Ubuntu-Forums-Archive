### Post by polly1 on 2006-11-10
Hi

One other thing you could do is configure firestarter so that no one can ping
your computer etc. 

Open Firestarter-click preferences-click ICMP filtering and tick Enable ICMP
Filtering box.

I also start firestarter and leave it open before I open my web browser, because by doing this your computer becomes invisible ie Stealthed. See
Shields Test on Steve Gibson Corporations web site for further info.

Strictly this procedure is not necessary, but having been stung once on XP.
now late of that OS, I am security minded
