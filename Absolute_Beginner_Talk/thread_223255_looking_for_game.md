### Post by moma on 2006-07-26
Is it "Tremulous"?

1) Get it
$ wget [http://www.planetgloom.com/trem/tremulous-1.1.0-installer.x86.run](http://www.planetgloom.com/trem/tremulous-1.1.0-installer.x86.run)

2) Make executable
$ chmod +x tremulous-1.1.0-installer.x86.run

3) Install and run the game
$ ./tremulous-1.1.0-installer.x86.run

4) Run the game
$ $HOME/tremulous/tremulous.x86

Requires a 3D accelerated graphic card w/ OpenGL?
