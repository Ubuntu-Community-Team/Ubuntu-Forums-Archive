### Post by annda on 2007-07-11
i'm not sure there are any online guides like that. gui is pretty self-explanatory. maybe you could get one of the ubuntu books...

one good thing you can do for those other users is set up their desktops - put shortcuts on the desktop or in the panel to firefox, openoffice, a media player, and maybe install all multimedia codecs and flash plugin for ff, even configure evolution mail for them.

from my experience, the only thing that a normal user needs in a gui is to know where to click to get some programs open. most people don't like menus. strange, but it seems true.

update: give them a messenger, skype and 2-3 games too. i noticed that people like those things ;-)
