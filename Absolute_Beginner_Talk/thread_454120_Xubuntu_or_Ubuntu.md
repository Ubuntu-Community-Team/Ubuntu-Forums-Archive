### Post by ugm6hr on 2007-05-27
I realise you've made your decision - but I thought I should let you know what Xubuntu offers.  My experience after about 3 weeks of use are favourable.  It is very configurable, and is certainly quick on 256MB RAM.

Problems are:
No default Network Manager installed - so you have to connect manually to wireless (and then install Network Manager if you want)
No default search for file facility in Thunar - so you have to either add catfish or gnome search.
No working CD-writing software (xfburn is useless) - so have to add e.g. GnomeBaker.

As you can see - there are issues, but they can all be solved with a bit of help (mainly in these forums).  I think it's great, and have it set up perfectly for my laptops.  If you find that Ubuntu is slower than you would like on your system - give it a try.

PS: You will not need 2GB Swap for Xubuntu - about 600MB will be fine.
