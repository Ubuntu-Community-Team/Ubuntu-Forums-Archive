### Post by thane1 on 2007-09-28
As a relative newbie when I have problems getting results finding files of installed programs using commands such as "whereis", etc in the terminal, I usually fall back on the following method:  On the top menu, open System - Administration - Synaptic Package Manager - search for the program package you want - when it shows, highlight it - click properties - click installed files.  You'll get a list of all the installed files for your program and their installed directories.  A somewhat slow, messy solution but it works.
