### Post by ScorpionDi on 2007-10-22
So, I found myself with the same problem, and after searching a bit into System Settings found what you need to do:
- Go to the advanced tab and select Login Manager
- Now under the tab Users (after entering Administrator Mode) you can change the "User image source".
I don't really know much about the difference of those sources but it now works ;)

Hope I've been helpfull in my first post.

Btw, it will only show the new image after a reboot (or at least a X Server restart I guess...).
