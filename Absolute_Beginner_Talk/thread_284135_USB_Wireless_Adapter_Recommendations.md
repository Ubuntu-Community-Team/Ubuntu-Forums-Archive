### Post by Synapse56 on 2006-10-25
Thornomad

I use a Netgear WG111T USB device. To install I had to set it up with NDISWRAPPER and use the drivers that came on the netgear CD.

Took a little while to learn how ndiswrapper worked but once it was done I've never had to go back and change anything.

I use Ubuntu6.06 and KNetworkManager under both Gnome and KDE and the device works like a dream, can't fault either the OS or Ndiswrapper.

HTH
