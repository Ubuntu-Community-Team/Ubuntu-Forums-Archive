### Post by zff4 on 2007-06-18
An expat collegue (we're working overseas) remarked his laptop Windows installation essentially "blew up" and he couldn't re-install because he had left his OS disk back in the States.  I offered to do the needful on his Sony Vaio with Ubuntu.  He hadn't heard of Ubuntu but was desperate.

Installation went well except a freeze during "installing languages".  I canceled that portion, but worried that my cancelation would affect the rest of the install.  It did not.  But if that particular issue is a common occurance, it should be looked at.

Anyway, install went fine.  The box hit the Internet right off.  As my collegue only needed Internet access, I didn't care that I was unable to figure out how to get DVD movies to play.   I tried however, to no avail and was frustrated at my failure.

Anyway, my collegue is quite happy, describes his new Ubuntu OS as "elegant" and vows to not return to Windows.  Good for you guys.

However, I'll delay my own install for a number of reasons, mostly game related.  But you guys really need to make it easier to install DVD support (for commercial DVD movie disks).  Although I admit I'm not Ubuntu educated and perhaps I missed the obvious.

So it was mixed.  I'm still not sold, but my collegue is quite happy.  Happy enough that I'll continue to recommend U for my collegues (as long as they don't want wireless).  Good work.   If you can recommend a good useful guide to Ubuntu, something easy to find in US bookstores, I'll be grateful - my collegue is interested.

### Post by jvc26 on 2007-06-18
Installing DVD playing support is very straightforward - it includes all the stuff about .mp3 as well:
[Restricted Formats Link]("https://help.ubuntu.com/community/RestrictedFormats")

Or more specifically the stuff on DVD Playback:
[DVD Players from Ubuntu Community Documentation]("https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs")
[Similar Guide from Ubuntu Guide]("http://ubuntuguide.org/wiki/Ubuntu:Feisty#How_to_install_DVD_playback_capability")

Hope that helps you :) Also, if it froze I would definitely recommend doing a disk check on you install cd, there may be something wrong there.
