### Post by larryfroot on 2007-10-30
With you wanting your linux box to be a media station then you are lucky to be looking at ubuntu 7.10, the latest release as it has (I'm told) a dedicated download manager that exists to download proprietry codecs and software and get them running on your system in a couple of mouse clicks. So MP3's and .avi and anyother media format you can shake a stick at will run fine with ubuntu. The reason that the codecs need your express action in loading them on rather than being included by default is one of those oooh it might be a legal hassle for the ubuntu team (not the individual user, I hasten to add) somewhere down the road. And one thing about the realplayer download if you do decide you want or need it. Download it to the directory /opt and install it from there. 

But there is absolutely nothing wrong in buying an introductory book on ubuntu. There is enough similarities to windows to fool you into thinking it operates in the same way. The good news is that it doesn't. The bad news - such as it is - is that you are in for a mild learning curve, so a dummies book or equivalent will prove invaluable. 

And I cannot tell you what a joy it is to just do what I want and how i want without malware, viruses or the general limitations of a windows system. Enthusiastic about an OS? Fraid so. Its brilliant once you get used to the differences (and then begin to appreciate them). Good luck.

### Post by larryfroot on 2007-10-31
revamping main box - do it with wine. Wine is a recursive acronym - it stands for **W**ine** I**s **N**ot an **E**mulator. It also stands for larry is a smart **** know it all for throwing terms like 'recursive acronym' around before breakfast. Anyways...it provides linux and you with a virtual windows environment into which you can install windows programmes. The best way to find out what progs are supported and how to get them onto your linux installation is to go to [franks corner]("http://www.frankscorner.org/") for a pretty good list of the main apps (like adobe stuff) that Frank has got to run on a Linux box. 

Possible pitfalls are: the latest version of wine works well fot the applications it was configured to include, but inadvertantly it may have broken a previous ability to run certain software. This is known as regression. I will be making forays into wine in the next couple of weeks, but I am certain that you can create different windows 'pretend' environments which are optimised for each windows application installed. These specific environments in wine are called 'bottles'. Each bottle holds a specific windows application that works best with the environment defined by that particular bottle. 

I have had photoshop 7.0 up and running with a free trial of crossover office (a sort of one click wine), but the standard 'one click and its in'  approach **seems** to rely on the lowest possible denominator that can run as many apps in a single bottle as is possible. Or put it this way, the list of apps that can run on franks corner looks a damn sight more useful to a graphics creative than the apps listed in crossover office. 

And as always there is the Gimp. If you go into synaptic and search for 'grokking the gimp' you can download and e-book and tutorials for the Gimp, or if vector based illustration is your thing then there's inkscape....not as feature rich as the gimp, but good enough for a lot of stuff...publishing, try scribus. 

In fact the bezier tool in inkscape blew me away....a total beginner could use it within minutes of playing with it. Which as anyone who has learnt the bezier knows is high praise indeed.

Anyways, welcome aboard and enjoy the freedom.
