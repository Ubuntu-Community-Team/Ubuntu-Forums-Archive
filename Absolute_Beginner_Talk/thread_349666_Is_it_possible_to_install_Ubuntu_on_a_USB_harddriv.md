### Post by yanqui on 2007-01-30
YES IT IS!!!!!  The instructions are [here](http://www.ubuntuforums.org/showthread.php?t=80811).  I know they work, I've done the install several times, had to make sure it works because I'm giving a presentation on it.

Couple of codicils I had to hunt down on my own:

Hook up the usb drive and turn on teh computer, but don't let it boot; go into bios first and see how your bios recognizes the usb hard drive.  Mine sees it as a hard drive.

My computer will not always maintain the directive to boot from the usb hard drive before the installed hard drive if the usb hard drive has been powered down or disconnected from teh laptop.  I have to go into bios every time to make sure it is set to boot there after the CD.  Not too much of a pain.

I don't know if your bios will necessarily NEED to be able to boot from a usb device if it recognizes the usb hard drive as a hard drive.

READ THROUGH ALL THE INSTRUCTIONS before attempting the install--there are a couple of places where you have to "do this first before you do the previous step, if you don't know the answer to the question it asks."  You'll understand when you see it.

DON'T attempt this with any live version.  Use the "alternate" distribution.

That's all I can think of right now, good luck and let us know how it works out for you.
