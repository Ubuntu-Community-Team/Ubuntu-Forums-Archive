### Post by viciouslime on 2007-08-29
If you're getting the same problems in windows then yes it is likely that you have a problem with the card. However, so long as it is still producing some sort of output, it might be rectifiable, this sort of thing often happens when a graphics card overheats. Could this be the case? Try running windows with the side of your case off so that it can stay cooler and see if the problem still occurs. Also, if it is heat it is more likely to start happening a while after you have turned the computer on, is this the case?

Are you using an LCD or a laptop? I have seen loose connections cause problems like you describe on such screens, particularly laptop screens. I think it is something to do with certain colours that causes it, that is why every textbox would show corruption for example, as they are the same colour across all apps. This happened to my sisters laptop, only hers went pink, not green, but taking it apart and pushing the screen cable connector in a bit more firmly fixed it.
