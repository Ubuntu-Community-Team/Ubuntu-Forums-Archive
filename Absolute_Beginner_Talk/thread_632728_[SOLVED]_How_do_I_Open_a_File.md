At work, you have a windows PC? So you sent you back a url shortcut?

In this case, save the url on the desktop.
Right click on it, select 'open with', select 'text editor'

You should see something like this:
------------------------------------
[InternetShortcut]

URL=http://www.microsoft.com/isapi/redir.dll?prd=ie&pver=6&ar=IStart

Modified=D04545FD6F1DC801AC

------------------------------------

copy the part after URL=
and paste it into the address bar of your browser and hit enter. 

finished.
bookmark the page in your linux browser, and delete the file you copied on desktop.
