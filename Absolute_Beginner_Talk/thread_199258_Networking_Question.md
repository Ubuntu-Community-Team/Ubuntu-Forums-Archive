### Post by LKRaider on 2006-06-18
Let me see if I understand... you have a router that provides internet connection to the XP box, and you want to connect your Ubuntu box to the net via the XP box, is that it?

Ubuntu<---------------->WinXP<------------------>Router<--------------->Internet

Well, if this is the case, you'll have to enable ICS (internet connection sharing) on your XP so you can use the XP's internet connection.
You don't need to install anything on Ubuntu, just configure the network card correctly.
