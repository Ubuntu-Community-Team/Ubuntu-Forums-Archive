### Post by m.musashi on 2006-04-19
Are you referring to the start up screen (the one you see as all the computer boots) or the GDM screen you see after you log in?

I'm not as sure about the first but the GDM is easy. Just run configuration editor (applications -> system tools) and click apps -> gnome-session -> options and change the splash_image value to the location of the image you want to use. I created a /.splash folder in my home folder for these to make it easy to switch.

If you are using KDE go to the configure settings (or similar I'm not in KDE at the moment) and there is an option for splash screen. I think it's under appearance.
