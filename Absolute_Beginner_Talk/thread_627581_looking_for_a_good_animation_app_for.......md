### Post by eye208 on 2007-11-30
You can use GIMP to make animated GIFs. Here's a tutorial: [http://www.gimp.org/tutorials/Simple_Animations/](http://www.gimp.org/tutorials/Simple_Animations/)

You can also use ImageMagick to turn separate images into an animated GIF. This is especially useful if combined with other tools that can convert video files into image sequences, e.g. ffmpeg, Blender etc.

Synfig Studio is an application to make animated 2D cartoons. You can export to a video file, scale it down and then use ImageMagick to make a GIF from it.
