### Post by az on 2005-12-05
The default install will run fine.  You have a big enough hard drive to accommodate installing other, faster desktop environments if you want.  You do not need to remove one to install another.  As well, you can still use the tools installed by gnome if you use, say, icewm or XFCE.

Take the default partitioning scheme.  Everything on one partition plus a swap.  There is no reasn to have any seperate partitoins on a desktop machine.
