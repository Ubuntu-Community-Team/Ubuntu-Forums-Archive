### Post by emarkd on 2008-02-11
Fedora uses Gnome as the desktop, like Ubuntu, so it'll seem familiar to you.  Underneath, though, Fedora is from the Red Hat line of distros where Ubuntu is from the Debian line, so there are some big differences.  Personally, I've been using Debian and Debian-derivatives for years, so I like Ubuntu much better, but check it out and see what works for you.
