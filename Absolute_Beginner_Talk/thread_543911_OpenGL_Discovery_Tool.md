### Post by SOULRiDER on 2007-09-05
Try
```
glxgears
``` those are simple 3D gears spinning.
You can also try
```
glxinfo
``` you can also filter the output by doing:
```
glxinfo | grep OpenGL
```

Tell us how it goes, Good luck!
