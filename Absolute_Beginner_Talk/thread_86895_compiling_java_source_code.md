### Post by Moonbuzz on 2005-11-07
If you have the [Java JDK]("http://java.sun.com/j2se/1.4.2/download.html") installed, all you need to do is ```
javac *name of file*.java
```Follwing that you can execute the file with ```
java *name of file*
```
