### Post by Najand on 2006-09-10
Maybe you should edit  your BIAS Setting. You can enter BIAS setting by  clicking Del or F2 or something similar. (It is usually shown on the screen when BIASing the computer. Then select the boot order and put CD-ROM boot before Hard Disk boot. I think  it will boot the CD-ROM afterwards.
