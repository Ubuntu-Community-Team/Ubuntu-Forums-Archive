### Post by iamstretchypanda on 2007-07-23
All of the default repositories (where you download software from) are secure.  Also the multiverse, universe, and restricted are secure.  They are not enabled by default but do contain lots of useful software.  

You can enable these by System -> Administration -> Software Sources.  Under the 'Ubuntu Software" tab check the multiverse, universe, and restricted boxes to enable them.

Good to see the spirit of ubuntu.  Feel free to pm me if you have any more problems. =]
