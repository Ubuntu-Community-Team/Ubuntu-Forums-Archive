### Post by Jab0rnal on 2008-05-22
I know this is an old thread but I thought my reply may help someone.

I have been looking for a better alternative to VNC. Something secure like Remotely anywhere.

I think by far the best application that I have tried is the NX server (NX free edition) from nomachine.com.

You have to install all 3 .deb files in client, node, server order from here;

[http://www.nomachine.com/download-package.php?Prod_Id=5]("http://www.nomachine.com/download-package.php?Prod_Id=5")

Then a client on your remote machine. Worked perfectly for me and I am by no means a linux guru, very basic knowledge. 

[Windows Client]("http://www.nomachine.com/download-package.php?Prod_Id=65")

I use this to connect to my home Ubuntu box from my work windows machine. (Uses ssh port 22)
