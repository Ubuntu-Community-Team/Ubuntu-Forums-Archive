### Post by frodon on 2007-04-02
> Feisty Fawn 7.04 Development Branch

      Click Applications &#8594; Add/Remove. In the top right, change the setting to "All available applications". Then select Other in the left panel and then select the Ubuntu restricted extras package. Click OK.
Is it that painful to click 3 times ;)

Java is in the repositories as well, if you open synaptic you will find it.

It's the ubuntu choice to not include closed source apps so this is more likely to not change anyway a lot of efforts has been put in feisty fawn release to make the install of all those packages easy so now you can install all that in 3 clicks.
All is perfectly documented there :
[https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)
