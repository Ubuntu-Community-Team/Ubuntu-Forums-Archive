### Post by Bakerconspiracy on 2007-03-15
rtorrent uses the terminal if that helps at all.... I'm pretty sure there is no client with an encryption, that would require someone else with the same client using the same algorithm which would make even having an encryption in the first place insecure in the broad picture. You are better off encrypting files before you seed or download them (That is, if you are actually looking for data protection). I think what you are looking for is a way to hide your ip when you d/l torrents (sounds like you are doing something illegal :wink:). The answer to that question will take you on a long journey, but you might want to start [here]("http://www.ubuntuforums.org/showthread.php?t=10825&highlight=privoxy") and [here.]("http://wiki.noreply.org/noreply/TheOnionRouter/TorifyHOWTO#head-de861b050ee8ceb1e439200313f024f09d698c52")
