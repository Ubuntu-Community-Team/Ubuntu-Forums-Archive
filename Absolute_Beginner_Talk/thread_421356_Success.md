### Post by ukemike on 2007-04-24
After spending hours last weekend trying over and over to install kubuntu 7.04, monday night I found success!  The installation appeared to have completed once but wouldn't let me login.  So I reinstalled, and it got hung up part way through.  That happened two more times.  I spent a long time here using the search function.  One suggestion that I saw was burn the image again at a slower burn speed.  That seemed sort of outlandish, but it was by far the easiest thing to try.  I burned the disk at 8x instead of 48x.  I tried again, and JOY!  

It's an older computer so I've turned off most of the visual effects.  I was, without any struggle at all, able to access a shared folder on my windows computer and copy the backup back to the kubuntu home and desktop.  It's still copying now.  That is huge because with edgy it took me hours to configure samba so that the computers could see each other.  This time it just worked.

It's nicely snappy and looks great.  I already find KDE easier to use than gnome.  I tried to uninstall and install a lot of things all at once and that didn't work and I couldn't figure out why, so I'll try more incrementally later.

Anyway, I'm happy.  Actually I'm excited!
