### Post by Forrest Gumpp on 2008-03-05
You surely can.

The computer I am using to post this is set up in exactly this way.

You do have to do a manual installation.  It is best to use the alternate CD.

For an excellent all-encompassing guide to the use of the alternate CD, go to Herman's Illustrated Dual Boot Site.  See:  [http://users.bigpond.net.au/hermanzone/](http://users.bigpond.net.au/hermanzone/)
There are walkthroughs of various sorts of installations.

It is quite a big site, with all of its sub-pages.  Bookmark it or tab it, and there's not much it wont tell you.

I've described my own hardware set up here:  [http://ubuntuforums.org/showpost.php?p=4302069&postcount=7](http://ubuntuforums.org/showpost.php?p=4302069&postcount=7)
