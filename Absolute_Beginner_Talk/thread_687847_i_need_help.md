### Post by emarkd on 2008-02-04
Yes, install the updates.  Restricted drivers are hardware drivers that aren't considered free (as in freedom).  You don't have to pay for them, but Ubuntu can't fix them if something's going wrong.  Unfortunately, they're often the only way to get some bits of hardware to work.

### Post by PeterJS on 2008-02-04
it's generally a good idea to always install all the updates, it can't hurt if you've got the bandwidth to burn. The restricted drivers are the closed source video drivers use by ATI and Nvidia for their graphics cards. As for your specific message I've never seen it, it might be letting you know that new drivers are available, or that you are already using the restricted driver and they were update in that mass of updates. Check the restricted driver manager in System > Administration > Restricted Drivers.

### Post by jken146 on 2008-02-04
The updates are security updates and some bug fixes for all the software packages that are installed.  The reason there are so many is that the last CD image released was in October last year (Ubuntu has a 6 month release cycle).  I would recommend installing them.

Restricted drivers are non-free (proprietary) drivers for bits of hardware you have.  For some devices, especially newer graphics cards, they will give you better performance than the open source drivers that are used by default.  They are labelled 'restricted' because their licences restrict what you can do with them (for example, you may not be able to reada the source code or alter it as you wish).  Many people object on principle to using proprietary software.
