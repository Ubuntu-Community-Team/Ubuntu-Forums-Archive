### Post by Jagot on 2006-08-08
```
wget http://easyubuntu.freecontrib.org/files/easyubuntu-3.022.tar.gz
tar -zxf easyubuntu-3.022.tar.gz
cd easyubuntu
sudo python easyubuntu.in
```

Just copy and paste each line separately into a terminal and press enter after each one.
