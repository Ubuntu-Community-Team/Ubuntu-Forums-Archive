### Post by jfinkels on 2007-05-01
I think you are misunderstanding something. If you want to view the file hierarchy, open nautilus, the file browser, and go up to "/" directory (the root directory) and dig around for a while. Or you can do this from the terminal using 
```

cd /

```
to change to the root directory and 
```

ls

```
to list all files and directories within the current working directory (which you can see by typing
```

pwd

```
).
