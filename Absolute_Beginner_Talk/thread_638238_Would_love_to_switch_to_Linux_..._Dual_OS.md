### Post by Lvcoyote on 2007-12-11
For simplicity sake, I would add another hard drive and install ubuntu to it, and first dosconnect the hard drive that houses your WIndows install.  Load Ubuntu to the new drive, then hook your old drive back up.  Each OS will see the other hard drive, which you probably wont care about anyway.  Most mother boards offer a keystroke during boot to select which hard drive you boot from.  This is how I set my system up, and I prefer doing it this way.  I like keeping OS's on their own hard drives, just my personal preference.

I've only been at this a little over a month, but as time goes by I find myself using Ubuntu more and more and I plan on doing away with XP/Vista as I tinker with Ubuntu more and find the apps I need to get everything done that I need to do.  Its coming along nicely though!
