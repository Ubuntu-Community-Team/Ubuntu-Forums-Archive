### Post by Ek0nomik on 2007-03-15
Gaim includes MSN, AIM, ICQ, Jabber (which you can use for Google Talk), and IRC (I think).

If you go to System/Synaptic Package Manager, you can install Gaim (it should be preinstalled, under Internet).  You can also install VLC from Synaptic too.

As far as LimeWire goes, I know there are some threads on the forum about it.  I just ran a search for Limewire and this came up:  [http://ubuntuforums.org/showthread.php?t=383920&highlight=limewire](http://ubuntuforums.org/showthread.php?t=383920&highlight=limewire)

I hope that helps.

Cheers!

### Post by Kobalt on 2007-03-15
Ubuntu comes with [Gaim]("http://gaim.sourceforge.net/") installed. It's fully compatible with the MSN protocol, you'll be able to send/receive files and add your picture to your profile. But no webcam or voice tchat by the way...
For Limwire or VLC, and many other apps you might want to install, head over this guide : [http://ubuntuguide.org/wiki/Ubuntu_Edgy](http://ubuntuguide.org/wiki/Ubuntu_Edgy) (assuming you're on Ubuntu 6.10).
