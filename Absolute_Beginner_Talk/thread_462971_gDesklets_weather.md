### Post by Bakerman on 2007-06-03
Hi, As far as I'm aware there is only one working weather gDesklet. It's called GoodWeather display.

If you go to [www.weather.com](www.weather.com) and type in the location of your city into the search field, you will be presented with a list of related cities. If you hover the mouse over the city you need the location code for, in the status bar at the bottom of the window you will see the address of the link to your city. It will contain the code, something like UKXX0085, which is the code that I use for London. Configure the gDesklet to use this code and it should work ok. That's how I got it to work anyway!
