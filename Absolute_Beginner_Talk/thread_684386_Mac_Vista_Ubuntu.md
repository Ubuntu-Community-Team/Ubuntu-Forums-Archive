### Post by angryfirelord on 2008-02-01
> **Mastaan said:**
> Abt installing Ubuntu what kind off format should i do 

Ntfs or there is something else?...

and how abt mac os ... what kinda file allocation system do they have
At this point, I've assumed that you've got the LiveCD up and running & now you're trying to install it.

You DO NOT want to install Ubuntu onto an ntfs partition.You will run into problems in the future. Stick with ext3.

However, you must create at least two partitions. One / and one swap. Therefore, create one / partition formatted ext3 and create another formatted as swap (no mount point for swap).
