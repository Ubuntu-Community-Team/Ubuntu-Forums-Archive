### Post by jonnywombat on 2007-11-23
> **blizzy said:**
> Is there any way i can revert back to my windows??.. or did this download of linux delete my windows??

That depends on what you told the installer to do when you installed ubuntu..

If you chose the option to use the whole disk then yes your windows partition was nuked in the process.. If you opted to reduce the size of your windows partition and then create a new partition for linux it will still be there..(this is known as a dual boot set up)

When your machine boots watch just after the POST data watch for the grub boot loader dialog.. IF your windows is intact it will appear at the bottom of the boot options and you can select it by using the arrow keys.
