### Post by bodhi.zazen on 2006-08-23
Depends on preferences.

I use several locations in my home directory (/home/user_name)

1. SRC (for source). I download a program or source files here. This would be where I downloaded firefox. I then unpack and configure make install from ~/SRC/package.name-x.y.z

2. My Downloads (this is the default location for d4x and I have been too lazy to change). I use this for just about everything else.

3. Direct to location: If I am downloading, for example, a theme for xmms download to ~/.xmms/themes. Background image ~/BGimage, etc.
