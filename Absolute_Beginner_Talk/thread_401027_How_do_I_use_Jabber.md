### Post by quark_77 on 2007-04-05
Hi Angry penguin,

I've got Jabber to work for both my gmail accounts. I'll run you through the setup for gmail... I'm not sure how it goes for other accounts on jabber.
[LIST]
[*] When you got your address, it was either [email]something@gmail.com[/email] (old style) or [email]something@googlemail.com[/email] - later style. You'll know which. Both will work for emails but from my experience not for Jabber.
[*] Set your screen name to the something part before the @.
[*] set your host name to gmail.com if you have an address from way back when, when gmail was invitation only and they didn't have problems with trademarks here in the UK. If it's a later account, use googlemail.com
[*] Alias isn't important.
[*] Resource should be set to [FONT="Courier New"]Gaim[/FONT] exactly as it appears here.
[*] On the advanced tab (linux) set the log in server to talk.google.com. Set use TLS if available. Require TLS works for me, but perhaps you are better off trying to connect first then setting this.
[*] Register/save your account. It should now offer you a password dialog and/or give you a silly red message at the bottom of gaim - clicking that and pressing re-connect gives you the password dialog.
[*] Enter your password as you would in gmail login / google accounts / thunderbird or whatever...
[*] If it connects ok, you're in.
[/LIST]

That's my experience anyway, I've got it working for my jabber account on here and my personal email - hope this helps clarify things, it took me a while to work it out!
