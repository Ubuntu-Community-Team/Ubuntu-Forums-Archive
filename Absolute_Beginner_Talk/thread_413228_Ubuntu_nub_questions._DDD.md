### Post by sloggerkhan on 2007-04-19
With your chipset, you will find that by default you are using the ati driver, which is an open source driver. There is also a driver called fglrx made by ATI the company. There are pros and cons to using each one. My experience is that for some games and wine apps the fglrx driver is better, otherwise the ATI driver is fine, particularly because the open source driver supports AIGLX which makes beryl easy, whereas with the fglrx driver you will have to install xgl to use beryl.
