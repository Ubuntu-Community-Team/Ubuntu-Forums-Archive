### Post by blendmaster on 2006-10-12
Also, if you could tell us how much RAM and the processing speed of your CPU, because some older computers refuse Ubuntu.

You should have 192 MB RAM to run the live CD, or 128 MB to run the alternate. With about 2 GB of space available.

If this is the case, you may have to use the alternate install disc for Ubuntu, or use Xubuntu (Kubuntu does have a slightly higher chance of working than Ubuntu).

If your computer isn't below requirements, you may need to redownload the CD.

Otherwise, I'm not sure of any other possible problem.
