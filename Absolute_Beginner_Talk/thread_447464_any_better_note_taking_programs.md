### Post by katch22 on 2007-05-18
Yeah.. The best note programs in the world are gedit and nano! What I mean is, if you're looking for simple there is nothing better than a plain old text editor. For the last ten years or so that I was using a different OS I always had a file right on my desktop called "notes.txt" that opened with notepad (the simplest, most stripped-down text editor that came with that OS).. now that I use linux, I have the same file that opens with gedit when I click it or nano when I am using the shell (I made a simple alias to make it easy to open the file instead of typing: nano /path/to/file/filename each time).
