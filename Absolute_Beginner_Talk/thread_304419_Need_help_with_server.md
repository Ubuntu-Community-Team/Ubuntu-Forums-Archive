### Post by po0f on 2006-11-21
notcrunk,

Hit escape (a couple of times to make sure) to put yourself in "command" (or whatever they call it) mode.  Then, hit the following:
```
:wq
```
The colon (**:**) lets Vim know that whatever follows is a command.  **w** is write/save the file, and **q** is quit.
