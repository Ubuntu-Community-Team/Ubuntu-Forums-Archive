### Post by Sitix on 2007-01-31
At first, choosing Ubuntu for leightweight stuff is a bit of a weird choice. Ubuntu runs best at 192mb RAM and up. If you ask me, anything under that is too slow. If you really want to use an Ubuntu variant, use Xubuntu instead. It's Gnome sortlike (I actually consider it to be a stripped gnome version) and it's fast.

For browsers, Firefox isn't really tough for a computer. I think that Opera would run a little lighter. Not much though.

For chatting there isn't a much faster thing than Gaim, for what I know.

For word processing, I understand that OpenOffice is really tough for your computer. So I'd suggest AbiWord instead. As you said yourself, it runs way faster than OpenOffice.


As a last thing: Ubuntu is Debian-based. And Debian runs faster on slower computers. While Ubuntu is more for 'the new stuff' to compete with OS's like Windows and Mac, Debian is more oldskool. It works about the same (not exactly though) so I would try that out before starting up a slower system with Ubuntu.

Hope I helped you a bit... it's just that the standard gnome applications are already pretty leightweight for the computers of nowadays.

Edit: oh, sorry, I didn't see you already installed everything as leightweight as possible. Still, though, everything counts as I said it :P

### Post by K.Mandla on 2007-01-31
Ubuntu should run great on that. Dillo is a very lightweight browser, but it has some shortcomings; I don't know if it does Flash or not, but I do know Brunellus uses it to moderate here because it won't render animated gifs, and he can't stand animated smilies. Hi Brunellus! ):P 

So that's a browsing option, but after Dillo you might do well with Iceweasel instead of Firefox. I use it because it seems lighter and will do Firefox extensions and Flash. Some people say it's no speedier, but that's just their opinion. ;)

Abiword is a good word processor; Gnumeric is the Xubuntu choice for a spreadsheet, and it's a pretty good one.

Xchat might be an option for chatting, but I don't know enough about Gaim to recommend one or the other. People seem to like Jabber, but I don't know much about it either. 

P.S.: +1 for Fluxbuntu! :D
