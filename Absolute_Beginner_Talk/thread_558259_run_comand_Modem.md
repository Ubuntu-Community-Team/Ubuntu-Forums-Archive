### Post by Maestro23 on 2007-09-23
You checked the manufacturers website: [http://www.netcomm.com.au/Support/downloads.php?Category=MODEM&Products=IN5699](http://www.netcomm.com.au/Support/downloads.php?Category=MODEM&Products=IN5699)

Edit: Found this on the Netcomm support site: [http://kb.netcomm.com.au/kb/default.asp?id=1293&Lang=1](http://kb.netcomm.com.au/kb/default.asp?id=1293&Lang=1)

> USB Modems

Currently the only NetComm USB modem that will work with Linux is the NetComm Roadster II USB (AM5050 Rev3). It requires a Linux Kernel that supports USB CDC communications devices (Development v2.3.x kernels and 2.4 or later kernels). The modem will be allocated a /dev/ttyACM0 or similar name.
