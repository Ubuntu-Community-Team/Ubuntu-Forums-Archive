### Post by RomeReactor on 2007-12-17
Hi. BitTorrent is already installed; to start it, double-click on a .torrent file and a dialog will show up asking you to select a place to save the file. If you close BitTorrent before the download is complete, you can continue downloading by double-clicking on the same .torrent file again. If you want it to show up in the menus, right-click on the menus in the top panel, select "Edit Menus...", on the left pane select "Internet", and on the right check the box for "BitTorrent".
