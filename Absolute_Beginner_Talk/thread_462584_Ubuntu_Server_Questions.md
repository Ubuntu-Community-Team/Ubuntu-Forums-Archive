### Post by thornomad on 2007-06-02
As I understand it: the server version of Ubuntu is what underlies the regular ubuntu distribution ... without the GUI.  Therefore, you can do all the things on a server the same as on the regular ubuntu installation.  It can run all the same services.  If you want to run a file server, you can set one up in ubuntu.  A LAMP server: will work in ubuntu.  You don't need to install the "server" version for these things to work.  Just install the services you need.

So: you can test it all running the GUI.  I think people usually DON'T use a GUI because servers are often headless ... no monitor or keyboard and accessed remotely.  At least, mine is.
