### Post by mikeyphi on 2007-10-23
A partial reply to first question!
Some packages have a GUI built-in, in which case the icon is installed somewhere in the menus. Other packages only work through the command line.

If you read the package description in Synaptic it is usually possible to see if a GUI is available or if it is designed for command line operation.
