1. Navigate to System>Preferences>[COLOR="Blue"]Control Centre[/COLOR]

2. When the window opens, scroll down to the [COLOR="Blue"]System[/COLOR] Heading

3. Click on [COLOR="Blue"]Language Support[/COLOR] and choose your default Language.

I believe that if Language Support id not installed it will ask you if you want to install it.
