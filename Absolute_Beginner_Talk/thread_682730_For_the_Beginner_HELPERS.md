### Post by h0ax on 2008-01-30
Hello everyone,
I love to post on UbuntuForums and try to help people with their errors so that they may better use Ubuntu, and linux in general.

I have found a Firefox extension that greatly increases my productivity in responding to generic questions.  I thought I'd share that with you all, my fellow first-second-....eh responders. (:

Check this out --> [https://addons.mozilla.org/en-US/firefox/addon/578](https://addons.mozilla.org/en-US/firefox/addon/578)

It's a firefox plugin for "signatures."  I find myself often repeating what I've written in another thread for a different user.  With this tool, I can save these dialogs, with CODE /CODE flags and all as a separate "signature."  Then, the next time I need to do it I just right-click in the box and paste it back in.  

I used to create text files for easy access responding, but that was so inefficient!

I hope this will help some of you in your response rapidness, as it has certainly helped me.  

See you around!
