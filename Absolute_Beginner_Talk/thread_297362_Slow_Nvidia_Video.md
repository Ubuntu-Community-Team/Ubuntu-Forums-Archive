### Post by Perfect Storm on 2006-11-11
aye.

Now open the xorg.conf

```
sudo nano /etc/X11/xorg.conf
```

add

[b]HorizSync 30-97
VertRefresh 50-180[/b]

to Section "Screen"

so it looks like this:
```
Section "Screen"
    Identifier     "Default Screen"
    Device         "NVIDIA Corporation NV34 [GeForce FX 5500]"
    Monitor        "G810-3"
    HorizSync 30-97
    VertRefresh 50-180
    DefaultDepth    24
```

save and exit
and restart X alt+ ctrl+ backspace
Then the refresh rate is solved (hopefully)
