### Post by chick penguin on 2007-01-26
[QUOTE=NewOldTimer;

Under Appplications>System Tools there might be a program GUI called Configuration Editor and under Introduction in the menu is something called GNOME System Administrator's Guide which is a clickable link. Both of these huge areas make for good reading and learning, one small step at a time. I am doing this myself as a newbie. It helps me to understand what I am doing in the GUI so that I can apply that to when I go to the terminal. 

If you do not have the Configuration Editor you can go to Synaptic Manager and find it there. 

I do have a problem with editor that I would like to mention though and that on my old machine the Configuration Editor tends to lock up and be very slow. But if you have a newer machine it might be OK.
