### Post by skymt on 2006-09-14
Ignore the above post. This is an interface issue, not a permissions issue.

Anyway, the usual way to select multiple files in Nautilus list view is with key commands. Control-click to add files to your selection one at a time. To select a group of files, select the first and shift-click the last.
