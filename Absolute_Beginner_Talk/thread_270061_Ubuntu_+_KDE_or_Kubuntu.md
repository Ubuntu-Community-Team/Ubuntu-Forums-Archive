### Post by aysiu on 2006-10-03
KDE and Ubuntu are not the same as Kubuntu

Kubuntu has a very Ubuntu-tailored set of KDE packages.

If you want the general KDE, you should install *kde-core* (a few packages) or *kde* (a lot of packages).
