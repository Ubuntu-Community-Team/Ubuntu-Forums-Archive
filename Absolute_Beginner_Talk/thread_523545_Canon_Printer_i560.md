### Post by freesitebuilder on 2007-08-12
I've used the Turboprint driver [http://turboprint.info/](http://turboprint.info/) for my Canon - I see yours is on the supported printers list. You can try it free, but it's about 30euros to buy. It does do duplex, but you have to make sure that your default printer is set to tp0 rather than the CUPS default (System > admin > printers).
