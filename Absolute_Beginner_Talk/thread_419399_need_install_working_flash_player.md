### Post by trak87 on 2007-04-23
A few versions of Ubuntu ago, I tried the 64 bit version for my AMD computer, but I was disappointed that not all the programs are available for 64bit the same way they are available for 32 bit architecture. This may have changed in recent days, but if you don't find all the things you want for 64bit, don't hesitate to install the i386 version of Ubuntu on your AMD 64 bit system. It works great that way.

Ok, here's basically what I did to install the Flash plugin:

In Synaptic, enable the multiverse repository. Then "Reload" and look for and install  "flashplugin-nonfree".

Then open up Firefox and in the URL location bar thing, enter about&#58;plugins and press enter.

You'll see all the plugins Firefox knows about here. If you see the Flash plugin listed, you're good to go.
