In Windows you can select multiple items in a list in a window by ctrl-clicking on the items you want - this functionality is duplicated in Ubuntu.  

If, in Windows you Ctl-Shift-clicked items in a window it picked multiple items at once (click first item, then ctrl-shift-click on the last item and it would select everything in between)... how do you do this in Ubuntu?

### Post by ajgreeny on 2006-07-24
Having selected the first item in the list just click shift+click lower item, not Ctrl+shift+click.  I think this is the way I used to do it in windows too, but it's such a long time now since I used MS that I'm not totally sure if my brain is working properly.

Have you set nautilus (or konqueror, whichever you use) to open files on a single click?  If not, you don't need to bother about the Ctrl key for your first click to select a file, you can just click on it.  After that Ctrl+click will select another file leaving the first still selected, shift+click will select all files in between the two.  Just like windows, I'm fairly certain.
