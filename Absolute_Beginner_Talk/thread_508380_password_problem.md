### Post by tommcd on 2007-07-24
Passwords are required to install or uninstall programs or edit configuration files. This is a security feature and will protect you. Learn to love it. This is why linux is a secure system.
You should not need a password to launch regular user programs like media players, web browsers, chat programs, etc. Are you having problems opening those types of programs?
