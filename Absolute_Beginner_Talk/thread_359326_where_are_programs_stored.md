### Post by orlox on 2007-02-11
As far as I know, most of the programs installed can be found at /usr/bin

---

### Post by jem7v on 2007-02-11
orlox speaks truth!

And if you know the command to run a program (i.e., swiftfox for Swiftfox), you can go into a terminal and type

which swiftfox

and it will tell you where it is! i.e., it will say

/usr/bin/swiftfox
