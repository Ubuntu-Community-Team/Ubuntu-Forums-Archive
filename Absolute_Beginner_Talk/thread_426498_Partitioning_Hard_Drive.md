### Post by PartisanEntity on 2007-04-28
How you partition your hard drive is up to you. If your hard drive is empty, or if you do not care to keep the data on it then you select 'Partition whole hard drive'.

If you already have an operating system installed on the entire hard drive, such as Windows XP, then you have the option to free up some space and install Ubuntu on a new partition.

If you have created a new and empty partition in order to install Ubuntu then you also have the option to manually select that partition during the installation.

It is really up to you. Hope I understood your question right.
