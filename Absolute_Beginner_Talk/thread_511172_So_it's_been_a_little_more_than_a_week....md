### Post by maoglone on 2007-07-27
... since I installed Ubuntu on my Compaq notebook and I gotta say, it's been a mixed bag for me, in terms of my opinion on the OS.  I am, by the way, using Ubuntu *only*.  No more Windows Vista (it's too freaking BIG!)

I am a "standard" computer user, which in my case means that I'm the kind of user who wants to open the lid and only do a few things, like goof around on the 'net, maybe watch a movie from time to time, use OpenOffice, and record/edit/upload my podcast.  Nothing too difficult, right?  Mostly.  

I LIKE: the fact that the GUI is more or less as easy to understand as any other OS; the availability of software and that there's a central place to find it; that downloads/installs are WAY quicker than in Windows AND MacOS; and that the OS boots amazingly quickly and the applications operate a lot more nimbly than Windows & Mac.  I'm not saying anything about stability because as a standard computer user, I assume that my system, no matter the OS, is not going to fail.  It's not really something I should bother to worry about, and in general, I haven't (and I've got experience with Mac, Windows, and now, Ubuntu).  

I DON'T LIKE: the fact that there isn't a central place for documentation that I've been able to find.  I am able to get lots of answers and support via these forums, which are incredibly helpful, but that shouldn't have to be a user's primary source for how to set up a computer.  Which brings me to the next thing--as a "casual" user, I did not like having to set up each little thing, like the fact that Ubuntu doesn't play lots of different media files out of the box.  Granted, this wasn't a complicated fix, but a casual user in general is going to be way irked by this, and maybe even intimidated to the point of not using it.  At the same time, Ubuntu's install is WAY quicker than other OS's, so after setting everything up the way I want it (I'm using Feisty and haven't tried Beryl yet, should I?) ends up taking as long as installing Windows, provided I've never used Ubuntu before.  I do not like the fact that everything is left up to the user's choice right out of the box--this is NOT what the mass user market wants--I say this because I get the impression that Ubuntu wants to appeal to the masses.  It's close, but not there yet.  Perhaps it'd be a good idea for someone to put together a video tutorial that automatically runs upon booting the live disk--You know, something painfully simple that's clearly for the "Granny"-type user out there who has done a little research and is interested enough to try it out, but will be scared away by having to use the command line every once in a while.  You know, Windows users.  

I also don't like having to use the command line, but I'm not scared of it.  

So, does anyone who's recently switched over--and is a casual user like me--have any other raves/gripes about Ubuntu?

In a Terminal do a:
$man "command"
For Example, if you want to know how to use mplayer then do a:
$man mplayer
And you'll see a more or less detailed Manual for that program.
