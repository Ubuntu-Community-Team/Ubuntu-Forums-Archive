### Post by RomeReactor on 2007-05-11
Hi. Most probably the issue you're having has to do with the speed of your processor and how much ram there is on your system; a minimum of 256 Mb is recommended (I think) since the Ubuntu disc is going into a "Live CD" session when you boot from it, meaning it will run *entirely* in the ram in your PC. So I agree that it would be best to download the Alternate CD ([Direct download]("http://mirrors.xmission.com/ubuntu-cd/feisty/ubuntu-7.04-alternate-i386.iso") or [torrent file]("http://mirrors.xmission.com/ubuntu-cd/feisty/ubuntu-7.04-alternate-i386.iso.torrent")). [Here]("http://www.ubuntu.com/getubuntu/downloadmirrors") is the complete list of mirrors so you can choose the one closest to you.

As for Linux not being user-friendly, I disagree; I think Ubuntu (and most of the main distros) are as user friendly, if not moreso, than windows. However, only the person using it can be the judge of that, so give it try and see for yourself. We'll be here to help you in any way we can.
