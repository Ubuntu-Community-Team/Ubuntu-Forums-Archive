### Post by 3rdalbum on 2006-08-16
A good way to tell if a program is tied to a desktop environment is by trying to install it. If it brings in any dependancies which have "qt", "kde" or "gnome" in their names, then they're not just GTK.

Apart from what you've already installed, the Xubuntu CD doesn't really have much else. The Ubuntu DVD, I think, has all the packages in main and restricted; which still isn't a whole lot, but it's a little more to play with.
