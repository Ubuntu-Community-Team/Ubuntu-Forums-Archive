### Post by kyphi on 2007-11-04
There should not be any complications with transplanting DVD and CD drives.

I would be very reluctant to remove and reinstall the CPU.  You should be able to remove any drive without removing the CPU.

You could install another DVD/CD burner.  Ubuntu comes with a burning programme - there is no need to switch to Windows.  Alternatively you could install Ubuntu on your HP machine.
