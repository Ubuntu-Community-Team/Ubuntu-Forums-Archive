I don't know of any interactive tutorial, but there are plenty of websites that teach commands. [LinuxCommand.org]("http://linuxcommand.org/") is one, and [this]("http://www.ss64.com/bash/") website has all the commands. I would not only learn commands, but learn how to script using bash. [Here]("http://tldp.org/LDP/abs/html/") is the tutorial I use.
