### Post by RomeReactor on 2007-02-26
Hi. Synaptic is a frontend to apt, the program that manages all packages available from the online repositories; you open Synaptic, search for a program, and click "Install": Synaptic will handle all the dependencies for you (it will install any libraries necessary for the program you want to install to run).
