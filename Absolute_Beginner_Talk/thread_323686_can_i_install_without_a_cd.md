### Post by macogw on 2006-12-22
Remember that laptop hard drives and desktop hard drives are different sizes.  It'd be easiest to put it in another laptop.  If that's not possible, get an external hard drive enclosure for your size hard drive (probably 2.5") so you can hook it up to a desktop through the USB port.  The enclosures are only $10 on [http://newegg.com](http://newegg.com) if you can't find one cheap at a computer store
