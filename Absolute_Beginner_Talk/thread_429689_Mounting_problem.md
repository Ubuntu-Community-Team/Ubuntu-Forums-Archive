### Post by taurus on 2007-05-01
Edit /etc/fstab

```
gksudo gedit /etc/fstab
```
and add this line to the end of it.

```
/dev/hda2   /media/hda2   ext3   defaults   0   1
```
Save it.  Then, create a new mount point and mount it.

```
sudo mkdir /media/hda2
sudo mount -a
df -h
```
