### Post by pip_134 on 2008-02-17
It could be that your keyboard is set to the wrong configuration. Does pressing the same key twice give the same character on the screen? 

And if you can log in, go to "System -> Preferences -> Keyboard" then click the layout tabs. If you use a QWERTY keyboard then you should have US English or United Kindom listed there (the difference between the two is the interchanging or @ and " and maybe some others I'm not too sure) if not click "add" then add one of those two.
