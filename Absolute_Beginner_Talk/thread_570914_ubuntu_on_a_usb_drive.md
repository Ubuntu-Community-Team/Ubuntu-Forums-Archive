### Post by C.S.Cameron on 2007-10-09
I am running Ubuntu installed on a Transcend 8GB (V10) Thumb drive, seems to have enough room and is fast. Have Wine, Autocad 2000, Rhino 3D V2, SketchUp 6, Skype and lots of other stuff installed.
I also have Ubuntu installed on a Kingston 4GB Data traveler, Have Wine, Autocad 2000 and Rhino V2 installed, It is close to full. It is faster than the 8GB.
Also have DSL installed on a 2GB Kingston Data traveler. This works ok, but may not work with some newer hardware. DSL will install directly to the thumb drive.
To install Ubuntu I unplugged my hard drives, (Just to be safe), plugged in the thumb drive, inserted the Cd then did an install.
Once the drive gets used to a Nvidia machine it does not like booting from an ATI box, and vise versa.
Keeping a copy of xorg.conf from the CD seems to allow access to most of the computers I have tried.
