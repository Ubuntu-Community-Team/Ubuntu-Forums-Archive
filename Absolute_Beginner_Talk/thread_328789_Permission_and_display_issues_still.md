### Post by taurus on 2006-12-31
1.  After you installed nVidia driver for your graphic card, did you at least look in /etc/X11/xorg.conf to see if it uses "nvidia" driver now instea of the generic "nv" driver?

Applications -> Accessories -> Terminal
```
cat /etc/X11/xorg.conf
```
Also, don't forget to restart X again with <Ctrl><Alt>Backspace.

2.  You can mount it again, giving permission for everybody to write to it, or you can use the sudo or gksudo nautilus...
Applications -> Accessories -> Terminal
```
gksudo nautilus
```
