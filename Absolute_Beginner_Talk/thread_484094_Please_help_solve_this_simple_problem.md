Ok I feel like a fool for not being able to figure this out myself but such is life. I do not know if it is a Mozilla or KDE setting but here is my question.

When in Windoze and I have Mozilla open and I click in the address bar to type a new url, the first click turns the entire current address blue (Highlights it) so I can type in a new address. When using Ubuntu I have to double click which sometimes doesnt work or I have to delete the whole address and then type. I am trying to figure out how to make the address bar highlight or select with one click like Windows does. 

For my particular browsing style this is extremely important to me. Please help, point out the setting I can look to, to make this happen.

### Post by Kobalt on 2007-06-25
Type about:config in your address bar and press Enter. Right click and click : *new* > *boolean*. Type *browser.urlbar.clickSelectsAll* and give it the value *true*.

PS: By default, if you click the favicon, at the very beginning of the address bar, all the text is selected.
