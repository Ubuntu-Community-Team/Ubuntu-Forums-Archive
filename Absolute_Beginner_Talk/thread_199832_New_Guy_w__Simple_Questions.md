### Post by aysiu on 2006-06-19
Edubuntu is for children, I think.

I would advise Ubuntu, as that's what most users here have. It's a cinch to install Kubuntu or Xubuntu (or even Edubuntu) on top of that later if you're curious about those.

Unfortunately, with a PowerPC iBook, you're not going to be able to play those ROMs easily. *zsnes*, like a lot of software in the repositories, is available for only x86 architectures--not 64-bit or PPC.

### Post by IYY on 2006-06-20
> Btw, what is breezy badger, hoary hedgehog and warty warthog?

These are the code-names of the different versions. The current version is called Dapper Drake, the previous was Breezy Badger, and before that there was Hoary Hedgehog, etc. 

It's very common for Linux distros to pick silly codenames like that. For example, Debian uses names of Toy Story characters.
