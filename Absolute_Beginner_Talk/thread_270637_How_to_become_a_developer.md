### Post by kevinlyfellow on 2006-10-03
Here's a good place to start [http://sourceforge.net/](http://sourceforge.net/)

There many developers start open source projects.  Depending if people like your project or not depends if it is adopted!  You don't need to use the GNU public license, there are many different ones available, including LGPL (Lesser GNU public license), Mozilla Public License, and many that I just plain don't know about :-) .  Try exploring [http://www.opensource.org/licenses/](http://www.opensource.org/licenses/) for different licensing options (I know its a lot of reading).  GPL is a great way to go though and with it you can still maintain copyright (from my understanding).  Good luck!!
