### Post by ewl1217 on 2006-08-05
It runs a bunch of tests on your computer's RAM, seeing if there are problems with it or not. If you're curious, you can just give it a run.

---

### Post by Tomosaur on 2006-08-05
It's a fairly useful option if you're experience slow loading, or just general memory issues. It can be disabled if you don't think you'll need it, but I personally leave mine in the menu just in case. If you want to look into disabling it, the link in my sig leads to my script which allows you to do this, or you can edit the menu.lst file yourself and then run update-grub.
