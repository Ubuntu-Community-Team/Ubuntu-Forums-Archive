### Post by timboo on 2005-08-05
Hi,
mp32ogg is quite easy to handle: open a shell, go the directory with your music files and type: mp32ogg FOO.mp3
the script will then generate a .ogg file with the same name as the .mp3 file (different ending of course), i.e. leaves the .mp3 untouched. The sampling rate is the same as for the .mp3 file. In case you want to do an entire directory type mpp32ogg *.mp3 and  the program will produce .ogg files of all the .mp3's in the directory.
cheers!
