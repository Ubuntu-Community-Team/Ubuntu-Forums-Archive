### Post by ziggykg on 2007-03-05
Normally, Java on Linux comes in a .bin file.  If you have such a file, then, first mark it as executable thus:

```
chmod +x java-install-file.bin
```

Then just run the install file thus to kick off the install:

```
./java-install-file.bin
```

Hope this helps
