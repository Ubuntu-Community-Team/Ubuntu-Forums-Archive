### Post by nudnik on 2006-06-29
Windows will not necessarily overwrite Linux partitions. In certain cases, probably dependent on which version of Windows you are using, you will receive a message stating something like: “Installation cannot continue, there is another OS installed on the HD” If this becomes a problem, you must use a utility like Active's Kill Disk, available here: [http://www.killdisk.com/](http://www.killdisk.com/) 

If you use this program, be sure to erase all of the Linux partitions to ensure there is nothing left to confuse Windows. :D  Then, re-insert the Windows CD and try again.
