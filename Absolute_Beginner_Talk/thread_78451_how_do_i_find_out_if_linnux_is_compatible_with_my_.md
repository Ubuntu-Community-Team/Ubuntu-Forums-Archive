### Post by brentoboy on 2005-10-18

Mepis can install from liveCD (once you like it, klick "install me" and it puts it on the harddrive)

There are lots of liveCD distro's knoppix is a good one.

### Post by ChrisTheGeek on 2005-10-18
Download the live CD from [www.ubuntu.com](www.ubuntu.com).  It allows you to test Ubuntu without affecting your Windows install at all

Bare in mind stuff runs slower off the CD than it will once it's installed properly.

Also remember you need to make sure your BIOS is set to boot from CD before hard-drive for any liveCD to work.
