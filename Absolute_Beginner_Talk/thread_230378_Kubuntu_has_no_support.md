### Post by kebes on 2006-08-06
As others have pointed out, most problems/solutions in Kubuntu are the same as for Ubuntu... so really many of the "Ubuntu" posts you see are related to both Ubuntu and Kubuntu. Only in the case of a Gnome or KDE-specific bug would the two not be equivalent.

Keep in mind that Kubuntu-specific forums also exist, if you are having KDE-specific problems:

[http://kubuntuforums.net/](http://kubuntuforums.net/)


Also note that you can "switch" to Kubuntu from your standard Ubuntu install just by installing all the KDE packages. In fact, if you perform this kind of upgrade you'll have access to both Gnome and KDE, so you'll be able to decide which one you prefer.

### Post by Jagot on 2006-08-06
And remember to use aptitude and not apt-get, otherwise if you decide you don't like kubuntu then you can get rid of it easily:

```
sudo aptitude update && sudo aptitude install kubuntu-desktop
```
