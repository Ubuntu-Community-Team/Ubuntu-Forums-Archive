### Post by talsemgeest on 2008-02-28
I have never done it before, so I might not get everything right, but I will give it a go.

First you have to install wine. Go to add/remove programs, and search for wine, and then tick it and click apply.

Next, go onto the WoW cd/dvd and right click on the setup file (Probably setup.exe) and click open with wine.

Go through the installer and install.

After that it should be the same as playing it under windows, i.e. double-clicking the desktop icon should start it.

Hope this is what you are after :)
