### Post by Who on 2006-01-25
It's weird, isn't it!

I needed to a) type a user name and password, b) set up the directories for sharing. There may have been other thigns, look in all the pages of the settings if it still doesn't work

Note that you _can_ just make up this username, but if you choose the same one as someone else then it won't work cos you won't know their password. If the server gets a new username and a new password it lets you in - thus i would suggest one way to test whether your configuration is right is to try a completely randome username - say 1234yhgthhuujh, and a new password. You can change the username once you have got it to connect.
