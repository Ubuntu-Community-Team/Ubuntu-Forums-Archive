### Post by phr0ze on 2007-06-05
If the cube doesn't work (which happens alot) try this.

Turn off the cube effects.
Go down to the lower right and rightclick on the box(es)
Select preferences
Make sure to select show all work spaces in 1 rows.
Make sure to select 4 workspaces.
Finally turn the effects back on.
