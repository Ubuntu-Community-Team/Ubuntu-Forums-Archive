### Post by benerivo on 2007-10-16
It's in the Control Centre > Regional & Accessibility > Keyboard Shotcuts > Shortcut Schemes. In the action column, find 'Panel > Popup Launch Menu'. This is the main menu action, and has a default shortcut of Alt+F1 (try that now, it should bring up the main menu). You can select a custom keyboard shortcut of your choice.
