### Post by Nythain on 2007-04-30
ah, but you forget that xp is worth the money for most personall pc users... its user friendly, familiar, and the most widely supported operating system in the country and or world.

To uninstall ubuntu is real easy...
1. make sure your boot order is cdrom then hard drive in your bios settings
2. insert windows xp cd
3. boot from cd, follow instructions.
4. when you get to the "where to install xp" part, delete all current partitions, and create one big partition, selecting to format it with ntfs (i think its the only option really)
5. continue on with installation of windows
