If I recall correctly you don't set the email path and name within OpenOffice.org; you just choose the email program that you have set up to handle your email. You set up the specifics of your email within the email program.  I think the default email program in Ubuntu is Evolution, so if you are using Thunderbird with the webmail extension you would need to change that in Tools>Options>Internet>Email. I access Hotmail with Evolution and hotway, so the default settings work for me. I don't know if there is a method to directly access web-based email in OpenOffice.org, but I do know that Thunderbird or Evolution can be set up to handle a lot of different types of webmail.
