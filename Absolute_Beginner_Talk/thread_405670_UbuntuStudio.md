### Post by jem7v on 2007-04-10
Chances are that he might find some of it functional, and other parts of it a complete loss.  Linux simply isn't the best OS if you want to be doing multimedia work... it simply doesn't have the high-end software needed in those fields.  It's not an issue of the OS being inferior... it's an issue of the software selection for it *in that particular area* being almost abysmal.

Part of the issue if he's doing lots of recording and the likes is that a lot of the peripherals put out by recording hardware manufacturers are Windows/Mac only and don't have Linux drivers written for them... hence why I still don't have a decent recording system set up yet.

From a desktop user perspective, he'd probably benefit, but from a recording/editing engineer perspective, he'd probably end up frustrated.
