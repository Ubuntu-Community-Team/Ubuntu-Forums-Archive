### Post by LostArt on 2007-06-23
Well a torrent is a very tiny file that you put into a bit torrent client such as Azuerus or Utorent, and it tells the client what to download, and downloads the file, at least that's the best way I can explain it. For music you can really use both but I'd say limewire is better for single songs while bit torrent is better for huge Files, for instance getting an ISO of Windows Vista vs. Fifty cents newest track. Limewire will work on Linux, but I'm not sure if you can find a deb of it and Utorrent will work under wine, and you can find tutorials on how to install it all over the net. to find torrent files all you really need to do is go to a site like Scrapetorrent.com, demonoid.com, or isohunt.com. 

About the slowness issue I would say that's naturally going to happen, because Jerks put malicious software, and spyware in things like torrents and on limewire, especially if the software your getting is not one hundered percent legal to download. Although most malware and viruses, don't target the linux audience, at least I tihnk so, correct me if I'm wrong.
