### Post by Sef on 2006-11-04
> I'm thinking of installing Panda's anti-virus for Linux. The download is either an .RPM or .tgz file. Is it possible to install these files in Ubuntu?

Read [How to Install Anything]("http://monkeyblog.org/ubuntu/installing/") on Ubuntu.  Why do you want an anti-virus?  It is not needed for linux, unless you have a email client, and then only so you don't infect your friends who have Windows.

> The pop-ups were quite persistant. My Google search was for info on two-way radios. I just bought a set of Cobra PR650 WX radios from Wal-mart. I was using the FireFox browser. Thanks.

Get [No Script]("https://addons.mozilla.org/firefox/722/") from the FireFox extentions.  It will block all java scripts.

Also get [Adblock Plus]("https://addons.mozilla.org/firefox/1865/").  

Use though two and you won't ever see an unwanted pop up again.
