### Post by rudeboyskunk on 2007-12-30
Yes, Synaptic shows everything that is installed and everything that is not installed.  Definitely DO NOT install everything.  That's like 20,000 packages or something.  Only install what you need.  Besides, you might install something that could mess your computer up to where you would have to reinstall Ubuntu.

And yes, it's just like Add/Remove.

### Post by Paqman on 2007-12-30
Yep, add/remove and Synaptic are both graphical frontends for Ubuntu's package management system. In the command line it uses the command apt-get.

Every package in every repository you're connected to will show up in Synaptic (think of the repo as a library you are browsing) You can search and browse, and it'll tell you things like what versions of software you have, show the dependencies of a specific package, and it can do other things like forcing the system to use an older version.

Awesome tool. [Lot's more info here](https://help.ubuntu.com/community/SynapticHowto)
