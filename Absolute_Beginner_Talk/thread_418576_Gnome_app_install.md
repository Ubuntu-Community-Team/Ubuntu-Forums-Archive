> **topic58 said:**
> Running 6.10 and 5.10. But missing Gnome app installer in 6.10. An easier way to install software, like a simple Synaptic. Someone who can help me find it in Edgy? I just have run and browse programs, but not install apps...

Try opening a terminal and entering
```
sudo apt-get install gnome-app-install
```

If that doesn't work or it's already installed, open the menu editor and make sure that the 'Add/Remove...' item is checked.
