### Post by pvdg on 2007-02-20
There are many ways to install program packages in Ubuntu. One GUI-based is Synaptic Package Manager, which you'll find in the menu: System>Administration>Synaptic Package Manager (you will be asked for your password). You can also use the command-line apt-get tool. For example, to install emacs:

```
sudo apt-get install emacs
```

You will also be asked for a password. If you're running Ubuntu 6.10 Edgy you can find more about it here:

[https://help.ubuntu.com/6.10/ubuntu/desktopguide/C/add-applications.html](https://help.ubuntu.com/6.10/ubuntu/desktopguide/C/add-applications.html)

or here, if you're running 6.06 Dapper:

[https://help.ubuntu.com/6.06/ubuntu/desktopguide/C/add-applications.html](https://help.ubuntu.com/6.06/ubuntu/desktopguide/C/add-applications.html)

I hope this helps. Just out of curiosity, why do all the words in your post begin with Capital Letters?
