### Post by Trail on 2008-02-13
Huh? That's nowhere close to any OpenGL code or anything. It's probably just a command-line parameter to force *something* to use OpenGL instead.

If it's Wine+game, you could check the [www.winehq.org's](www.winehq.org's) AppDB for information specific to the game you are trying to run. If a command is involved that would increase performace, it's likely that you'll find it there.

### Post by pedro_orange on 2008-02-13
+1 what Trail said.

You can use -opengl as a parameter to force an app to use OpenGL. 

For example you have a wine app that normally runs in directX and you want to force it to use OpenGL you have a launcher with the command:

```
wine ".home/.wine/....." -opengl -nosound
```

etc etc
