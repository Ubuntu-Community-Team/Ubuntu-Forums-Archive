### Post by LadyDoor on 2006-08-18
Are you using GDM (the default)? If so, go to Sessions (or hit F10 and then go to Sessions in the menu that pops up) and select GNOME. Log in. You *should* see a dialog that says something along the lines of "GNOME is not your default window manager." It will ask you whether you want to <cancel>, <log in for this session only> or <make GNOME my default session>.

Hope that helps!
