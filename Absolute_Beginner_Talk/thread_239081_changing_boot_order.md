### Post by Tom Brokaw on 2006-08-18
> **drosophyllum said:**
> nope tried that, is it possible to access the bios while booted?

Generally no, and when it is, it's to flash the bios which is not recommended from a Windows/32-bit environment.  

I'm assuming you're running Windows now; if so, go here:
[http://www.oldversion.com/program.php?n=everesthome](http://www.oldversion.com/program.php?n=everesthome)

and get the last one down.  Install and run, and you should be able to ID your motherboard.  From there go to the mfr's site and get the manual.

Or you could try hitting random keys to get in there.  ESC is also used sometimes.
