### Post by xeth_delta on 2007-12-02
Ok, this is what you have to do, just be VERY careful, as you will edit a line in the grub menu and login as root in command line mode.

When you are shown the grub boot menu (you might need to press ESC to have it shown), choose the kernel line, type "e". Now you will be able to edit the kernel arguments. Go to the far end of the line and append **init 1** with a space preceding it. Press enter to finish editing. Press **b** to boot. After a few seconds you should be presented with a root command prompt. Then type **passwd user_name**, where user_name is your user name. Type the password correctly. It will be updated and you should be able to log in. Reboot the machine typing **reboot**.

The changes you made before in the grub menu are not permanent, so no need to modify anything else, just boot normally

Good luck!
Xeth
