### Post by sethmahoney on 2006-01-04
Lots of Windows games run on a Linux machine with Cedega.  Unfortunately, Cedega isn't free.  Fortunately, its worth the (small amount of) money, because that money helps to ensure that the next game you want to play work with future versions of Cedega (they even let subscribers vote on which games they should try to get working).  To check and see if a specific game is supported by Cedega, look here:

[http://transgaming.org/gamesdb/](http://transgaming.org/gamesdb/)

Some other games, like Neverwinter Nights, have native Linux versions.  Others, especially games that use OpenGL rather than DirectX, run well under Wine.

The graphics are generally the same as under Windows, and depend on your system and setup.  The performance is sometimes a little laggy, sometimes a little better than windows, but usually comparable.
