### Post by Nythain on 2007-04-25
if you have already installed wine, and ran winecfg then it should be as simple as opening a terminal and typing:

wine /path/to/file/to/run.exe

you could also just cd over to where the exe is stored, like

cd /home/blah/Desktop
then
wine whatever.exe
