### Post by sigge on 2006-11-05
In nautilus rightclick on the file in question, and select "Properties" in the dropdown menu. In the resulting dialog box, select the tab "Open With" and then select between the installed software in a list below. Proceed with clicking close.
The selected application should now run when you doubleclick a file in nautilus.:cool:
