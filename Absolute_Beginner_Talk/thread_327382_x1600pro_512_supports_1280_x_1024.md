### Post by meborc on 2006-12-29
the easiest way to quickly fix your resolution:

```
sudo dpkg-reconfigure -phigh xserver-xorg
```

if this doesn't help try:
```
sudo dpkg-reconfigure xserver-xorg
```

now you need to answer more questions... if you don't know answers, leave the default ones... but at one point it will allow you to select the resolutions you want... ;)
