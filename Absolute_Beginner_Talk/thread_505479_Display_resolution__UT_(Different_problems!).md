### Post by ravenwere on 2007-07-20
Hi Colin,

When you said you used sudo, did you start nvidia-settings using the sudo command from the shell console? The changes don't take effect until you run "nvidia-settings" as root. Use the command - sudo nvidia-settings - in the console.

This worked for me.
