### Post by 23meg on 2006-12-13
Short answer: Ubuntu is based on Debian, one of the most mature, flexible and capable distros in existence, so there isn't much you can't do with it.

Longer answer: It really depends on your definition of "advanced". > by advanced usage I.e penetration testing, programming, server etc....
Two examples: [nUbuntu]("http://www.nubuntu.org/") is an Ubuntu based distro geared towards penetration testing, and a separate server install version of Ubuntu exists.

There may be distros that are geared more towards each advanced task, but this doesn't mean that you can't utilize Ubuntu efficiently in them; chances are when you've "advanced" enough into those areas, you'll have the skills to stretch Ubuntu to accomodate them. If you're worried about exhausting the possibilities of Ubuntu quickly, don't be.

### Post by Dusty545 on 2006-12-13
When all is said and done there is not really anything that cannot be done with any distro.

If you're really that advanced a user and feel the urge to use nothing but command prompts and get right to the very nub of the kernel then you can still do that with Ubuntu.  The advantage to Ubuntu is that if you find you can't quite manage something on your own then the sheer number of users of Ubuntu makes it easy to get help and there's quite possibly a shiny GUI option to accomplish your task.

Slackware/Gentoo/etc are wonderful products but I'd despair to try to get a newbie to get the faintest idea what was going on...  particularly if they're used to W*ndows.

My recommendation?  Stick with Ubuntu and get used to using command prompts to do things wherever possible and go from there.  If you feel the need to try something different then the source is just a download away.
