I'm using compiz fusion and when I run VLC media player its transparent. Does anyone know how to fix this? Thanks.

### Post by Happy_Man on 2007-07-13
Go to the Compiz Fusion Settings manager, General Options, Opacity Settings, and scroll to the end of that list there. Change the number from 89 to 100, and you should be good.
