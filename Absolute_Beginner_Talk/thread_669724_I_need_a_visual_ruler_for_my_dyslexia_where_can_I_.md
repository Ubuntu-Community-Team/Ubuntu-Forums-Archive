### Post by ~Maxx on 2008-01-16
Hi jingo.  I'm on Ubuntu 7.04, so I'm not sure if there's a difference in the version you're running but...  Go to System > Preferences > Windows.  You'll see a section labeled "**Title Bar Action**".  I believe the default is set to maximize, but if you change it to roll up you'll have your movable title bar by double clicking any open window.  

There may not be a solution for you as far as the text goes.  If I remember correctly the xfce (Xubuntu) desktop allows you to reorient the title bar text in the windows options, but I wasn't able to find a way to accomplish this in Gnome (short of changing the font size to zero, which is global and would get rid of *all* title bar text).

Hope this helps!
