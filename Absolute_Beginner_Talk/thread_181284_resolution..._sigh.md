### Post by aysiu on 2006-05-23
According to [this PDF from the Dell website](http://www.dell.com/downloads/emea/products/snp/MON-2005FPW-SS-16-02-05-UK.pdf), you should add these two lines to the monitor section of your /etc/X11/xorg.conf file: ```
HorizSync 30-83
VertRefresh 56-75
``` Can you go to that link and double-check that that's the same model you have?

P.S. Always back up any configuration file before you modify it--just in case.
