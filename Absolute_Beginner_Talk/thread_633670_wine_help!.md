### Post by popch on 2007-12-07
You don't actually 'use' Wine as you would 'paint' or 'solitaire'.

Once Wine is installed (the posts above this one explain that) you install the Windows applications you want to use in Linux. You then use those applications.

Wine remains discreetly and invisibly in the background if all goes well. 

If not, come back here and yell.
