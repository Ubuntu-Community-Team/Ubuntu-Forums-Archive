### Post by Brino on 2006-06-12
Hi,

A while back I decided to try Linux, but being a bit cautious, decided to have a bet each way and dual boot XP with Xandros v3, which has Crossover included.

Crossover worked fine, but I found myself using Word under Xandros (because it only has OOo 1.0) and OOo 2.0 when under Windows (because I prefer OOo 2.0)!

I then installed Ubuntu on its own and was glad I did.

The lessons?

1. Get Ubuntu 6.06, which works a treat and has OOo 2.0.
2. Practise and get used to OOo 2.0 (or any other of the fine alternatives).
3. If you need to share your work, export it and send as a PDF.
4. Forget about Windows and MS Office.

You won't look back :).

Regards
Brino

### Post by u.b.u.n.t.u on 2006-06-13
Abiword is an excellent word processor. There are other quality office products for Linux under GNU GPL.

The only percieved benefit of crossover office is familiarity and compatibility. I say percieved as both are myths.

Abiword is very easy to use for anyone who has only ever used M$ Word. Also between M$ office Word versions there isn't a lot of compatibiltiy. All the fancy bells and whistles of the latest version aren't much good if someone is using an older version and it looks different.
