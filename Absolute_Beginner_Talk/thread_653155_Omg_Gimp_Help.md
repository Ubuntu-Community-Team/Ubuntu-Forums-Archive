### Post by Samhain13 on 2007-12-30
I know this is comes late as OP has already done the job. But for what it's worth, here's something that I just knew in the last half-hour, which hopefully, someone else will find useful:

1. Draw a line using the Paths Tool. It can be a straight line, a curved line or a shape.
2. Pick colours and brush styles for the preferred tool for stroking (refer to #4).
3. Hit stroke path in the Paths Tool options box, which brings up the Choose Stroke Style dialogue box.
4. Select "Stroke with a paint tool" and pick a Paint tool (in my case, I picked Paintbrush).
5. Hit "Stroke".

(Please see also the attached images.)

The Paths are reusable, by the way. And they may come in handy for drawing-- well, hair?

Cheers! :)
