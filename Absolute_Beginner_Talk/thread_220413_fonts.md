### Post by coffeecat on 2006-07-21
All you need do is install .ttf fonts to your system and they'll be available in OpenOffice. See [this thread.](http://www.ubuntuforums.org/showthread.php?t=215955) There's about half-a-dozen ways to install .ttf fonts described there including installing the MS corefonts package. Or Automatix can install the MS corefonts, and I believe EasyUbuntu will also. See the subforums for each of these if you are interested.
