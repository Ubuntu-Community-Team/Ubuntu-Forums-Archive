### Post by Thunk on 2005-12-07

### Post by kane77 on 2005-12-07
ok I had the same problem,
right click on the upper bar (or wherever you want) and select "add to panel" choose keyboard indicator. And now you have keyboard indicator on the bar You can change active language by clicking on it... for some key combination try setting it up in "system"->"preferences"->"keyboard shortcuts"

### Post by Thunk on 2005-12-07
To add a keyboard shortcut you can right-click on the keyboard indicator icon->open keyboard preferences->layout options-> Group Shift/Lock behaviour now check any combo you'd like to use.
