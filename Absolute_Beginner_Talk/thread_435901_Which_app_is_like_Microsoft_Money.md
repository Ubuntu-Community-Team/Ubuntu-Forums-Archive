### Post by compmodder26 on 2007-05-07
There are a few financial apps for Linux that are comparable to MS Money.  My preferred application is MoneyDance.  It is not free though.  You have to pay $30 for the full version (trial version only allows 100 entries).  There is also Gnu Cash.  It is freely available in the repositories.  Others are KMyMoney and Grisbi.  

If you are looking to transfer accounts from MS Money to any of them, I believe all can handle the QIF format.
