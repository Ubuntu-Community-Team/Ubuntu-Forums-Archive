### Post by tommcd on 2006-09-19
That site uses flash. If this is the only site where you have sound problems, that is unusual. If it is frequent, see this:
[http://ubuntuforums.org/showthread.php?t=255422&highlight=flash+sound](http://ubuntuforums.org/showthread.php?t=255422&highlight=flash+sound)
This worked for me. I lost sound frequently though.
There are apparently other solutions to flash sound as well:
[http://ubuntuforums.org/showthread.php?t=243276&highlight=flash+sound](http://ubuntuforums.org/showthread.php?t=243276&highlight=flash+sound)
[http://ubuntuforums.org/showthread.php?t=75237&highlight=flash+sound](http://ubuntuforums.org/showthread.php?t=75237&highlight=flash+sound)


Alternative method to fix flash sound from:
[http://doc.gwos.org/index.php/Flash_Sound](http://doc.gwos.org/index.php/Flash_Sound)
I only did the 1st link method. Hope this helps!
