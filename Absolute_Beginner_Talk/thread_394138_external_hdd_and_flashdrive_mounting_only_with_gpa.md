### Post by angryfirelord on 2007-03-26
a) The whole UUID mess is an autoconfiguration thing that allows to switch or move drives to a different connection w/o having to mess with the fstab. For example if you wanted to move the drive to the secondary master port (/dev/hdc), the UUID would automatically configure this for you. I don't like it though.

b) & c) Plug in the drive & run *dmesg* to make sure that it is /dev/sda1.
