Well, what is it you did to create the directory? Where did you try to create it?

The accepted (only?) command for creating a directory is 'mkdir'.
For example: mkdir music

It can be a problem if you try to create a directory someplace you don't have access. If you just created it in your home directory someplace, that will be fine.
