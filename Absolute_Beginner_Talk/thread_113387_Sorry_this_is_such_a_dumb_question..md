### Post by hobbit1983 on 2006-01-06
did you mean slmodem?

To run it go into a terminal and type

cd 
./slmodem

This will execute the slmodem program.  You may need to allow the file to be executed in which case type

chmod +x slmodem - this will add execution permissions to the file.
