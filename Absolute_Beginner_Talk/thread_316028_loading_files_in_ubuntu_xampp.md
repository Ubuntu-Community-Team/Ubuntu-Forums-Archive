### Post by Verminox on 2006-12-10
I believe it is /opt/lampp/htdocs

Any files in there can be accessed via [http://localhost](http://localhost) once XAMPP has been started.

eg.

/opt/lampp/htdocs/hello/world.php can be accessed via:

[http://localhost/hello/world.php](http://localhost/hello/world.php)
