### Post by gyf on 2008-02-23
HTML is pretty easy once you get the tags. This is a nice site to learn how to use basic HTML tags which will help you make your About Me page [http://www.w3schools.com/html/]("http://www.w3schools.com/html/").

Also i included a sample piece of html which has two paragraphs and headings with different colors and different alignments just to give a sample guide.

Im sure someone can make a neater piece of code but that will create a simple html document with a heading About Me with to sub-headings with some misc text.

An overview of the tags used:

<BR> adds a break in the text..a new line...like pressing enter in a word processor
<P></P> is used to specify a paragraph 
<P ALIGN=CENTER STYLE="margin-bottom: 0cm; font-style: normal"><FONT COLOR="#800000"> specifies the paragraph while aligning the text to the centre with font size normal and the color dark red ish.

This is a nice site to get the color codes in hexadecimal: [http://html-color-codes.com/]("http://html-color-codes.com/")

Once you get the code you can just replace #800000 with the number of your choosing...for example #000000 for black


Sample Page


[HTML]<HTML>
<BODY>
	<P ALIGN=CENTER STYLE="margin-bottom: 0cm; font-style: normal"><FONT COLOR="#800000"><U><B>ABOUT ME</B></U></FONT></P>
	<BR>
	<P STYLE="margin-bottom: 0cm; text-decoration: none"><FONT COLOR="#000080"><B>Sub-Heading 1</B></FONT></P>
	<P>Stuff about me lalala
	<BR>lalalala lalaala
	<BR>lallalaalalaaa</P>
	<BR>
	<P><FONT COLOR="#000080"><B>Sub-heading 2</B></FONT></P>
	<BR>
	<P>more stuffs!!
	<BR>I like mountain dew</P>
</BODY>
</HTML>[/HTML]

Hope this helps
