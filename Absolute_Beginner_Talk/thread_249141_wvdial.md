### Post by szf on 2006-09-02
In a terminal do:

sudo vi /etc/wvdial.conf

Or create a "Open As Administrator" nautilus script in ~/.gnome2/nautilus-scripts (i've lost the link to thread on these forums):
[FONT="Fixedsys"]
#/bin/sh
for uri in $NAUTILUS_SCRIPT_SELECTED_URIS; do
gksudo "gnome-open $uri" &
done
[/FONT]

And navigate to the file in nautilus. Give a right click to to /etc/wvdial.conf amd select Scripts -> Open As Administrator
