### Post by Tasu on 2006-11-27
Well it's different from windows, indeed, but it's easier, if you know what to do! Well! You probably have downloaded source packages, but there's non need to use them. In ubuntu (and other Linux distros) you don't have to bother with finding the software on the net, download it, then for each one go through a lot of click, click, click install wizards, forget it, this was the past, in Ubuntu you have just to go to System->Administration->Package Manager (Synaptic), the names could be a little different, I have the Italian version, from there you can Search for the software you need, by description, name, etc. there ou can find program for everything.
Ubuntu is based on repositories, web sites full of programs that are accessed by Synaptic, you can add some of these repositories form the synaptic menu called Settings->Manage Repositories, there you can add, delete, manage or activate them.
In ubuntu the "installer" is a file with the .deb extension (not .exe s in windows), if you ave ubuntu, downloading a program in deb format and double clicking on it will give you an installer for that program, but surely there's a little need for this probaly by searching in internet downloading and doubleclicking you will install nothing more than two softwares, one of these could be Automatix, (look for it in google) it's one of the rare software not already in the repositories, but really usefull, with that you can easy find and install codecs and common customizations.
However, first of all take contact with synaptic, and it's ease of use, enable the universe and multiverse repositories from the Manage repositories link (they are not on by default, but you just have to click on a checkbox to enable them, and after the refresh you will have access to more than 15000 programs. After that relax and start to install a lot of things! ^_^

For any question, the forum is the best place to ask, sorry for the long post.
