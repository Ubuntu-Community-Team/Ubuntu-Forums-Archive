### Post by kapello on 2007-11-21
First, a word of warning. In my experience, there is nothing available on Linux which is anywhere near as simple and easy to use as Garageband. But, there are options. 

I recently installed Unbuntu Studio from [http://ubuntustudio.org/](http://ubuntustudio.org/)

This is a complete distribution which includes the linux 'real-time kernel' - basically, this kernel is superior for running music making apps (it lets them have priority access to your CPU). It also comes pre-bundled with many audio applications, but it's kinda 'hardcore'. 

Garageband (which you can only run on Apple Macs - it comes with the system if you buy one) is nice in that it's really really easy to use. But, it isn't as powerful as things like Cubase (famous Windows music program), or indeed, some of the Linux apps available. But, Cubase and these Linux apps, are also harder for the beginner to use. 

Check out the Ubuntu Studio link above and if you have specific questions about it, post back here or send me a private message.

Installation of Ubuntu Studio involves installing it from scratch as a total operating system, from a Live CD, which you download and burn from the site. If you've installed 'normal' Ubuntu before, it's basically the same process.

PS I should add, it is entirely possible, if you're running Ubuntu already, to just try downloading some of the apps that you get with Ubuntu Studio, to try them out. And, if you're impressed, you can also add the real-time kernel I mentioned, to your existing system. So you don't have to go for a clean install. 

Hydrogen is one of my favourite programs - basically just a drum machine, but pretty cool. Go to Synaptic Package manager, search for 'hydrogen', and install both the main program and the hydrogen-drumkits package (to get lots of drumkit choices).

Audacity is very good for simple 'live' recording (through a mic - if you mainly want to record yourself singing, playing guitar etc). Search and install in the same way as Hydrogen. 

Rosegarden is a promising-looking app that offers both Midi sequencing and audio recording in one place, a little bit like a lite version of Cubase, but to be honest, I haven't really got my head round using it yet, so I can't say much about it.
