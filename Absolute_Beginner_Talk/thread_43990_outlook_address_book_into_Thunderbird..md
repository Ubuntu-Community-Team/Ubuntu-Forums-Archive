I have been unable to get Thunderbird to import my outlook address book file(csv), basically because thunderbird does not allow as many options per address as outlook.  Any ideas?

### Post by aum11 on 2005-06-24
Success.

It was easier to install Thunderbird firstly under Windows, and then import address book and emails from outlook into the windows copy of Thunderbird.
Then it was easy to export the address book to a usb drive and then import the file into Linux thunderbird.
A similar path enabled the thunderbird profile, which contained the emails, to be transferred across.
It feels great to have left outlook behind.
