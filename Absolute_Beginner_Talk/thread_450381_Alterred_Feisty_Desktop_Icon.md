What you're seeing there is just the standard desktop icon. The other directory there, Examples, has two emblems. The curved arrow is telling you that it's a shortcut (symbolic link) to another directory. The padlock means that you, as a user, do not have permissions for that directory. In this case, the Examples directory belongs to Root.
