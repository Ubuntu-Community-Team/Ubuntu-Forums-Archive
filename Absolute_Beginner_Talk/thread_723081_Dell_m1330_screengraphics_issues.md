### Post by kpkeerthi on 2008-03-13
Easy solution:
1. Boot with ubuntu live CD
2. Copy /etc/X11/xorg.conf to a flash drive or somewhere
3. Now boot from hard disk.
4. Restore the xorg.conf from the flash drive
5. Reboot.
6. Enable nvidia by running **nvidia-xconfig** in a terminal.
