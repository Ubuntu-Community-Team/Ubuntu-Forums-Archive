### Post by starcraft.man on 2007-05-12
If there are drivers, [envy]("http://www.albertomilone.com/nvidia_scripts1.html") should do it for you. 

Download the 0.9.2 debian, double click and install it. Applications > System Tools > Envy. Tell it to automatically install the nvidia drivers and answer yes to any prompts. Should work.

Though, in my honest opinion, using the 64 bit version of any OS today only invites problems with incompatible software... its still a 32 bit world, and there isn't any significant improvement with 64 bit Ubuntu.


If I were you, I'd reinstall and go to the 32 bit version, it will give you less headaches. That said it's your choice :)
