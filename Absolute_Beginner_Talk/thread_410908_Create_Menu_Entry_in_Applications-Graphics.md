### Post by freebird54 on 2007-04-16
I assume (no Dapper) that right clicking the Applications Menu will give you the editor.  Select graphic.  new item.  Fill in info, including whether it should run in a terminal.  Choose icon if wanted.  Exit.

Something like that!
