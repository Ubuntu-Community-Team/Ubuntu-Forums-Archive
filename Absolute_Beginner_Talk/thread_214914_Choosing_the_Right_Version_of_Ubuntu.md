### Post by taurus on 2006-07-13
- Ubuntu 64 Edition (Desktop)

     It is for AMD64 (or Intel 64)!!!

- Ubuntu i386 Edition (Desktop)

     This is the one you want to download if you have a 32bit chipset (CPU)...

- Ubuntu 64 Edition (Server)

     This is the server version for 64bit.

- Ubuntu i386 Edition (Server)

     If you only want a server with no window manager, then use this version.  However, you can still install a window manager like Gnome, KDE, XFce4, fluxbox, etc. later on if you want...

### Post by mssever on 2006-07-13
It's very easy to install web development tools in the desktop version. Once you install, go to the System menu > Administration > Synaptic Package Manager. You can click the search button (or press <Ctrl><F>) and search for whatever you need (Apache, PHP, Rails, MySQL, whatever). There are also great editors available. I like Bluefish.
