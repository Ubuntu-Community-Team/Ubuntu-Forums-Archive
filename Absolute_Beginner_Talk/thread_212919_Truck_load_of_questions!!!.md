### Post by Brunellus on 2006-07-10
well, I generally use fewer exclamation points, so I suppose I got accustomed to the new environment faster than most.

Java is a very poor platform.  I actually now prefer gnome-bittorrent to azureus.  For your router/port forwarding issues, please visit [www.portforward.com](www.portforward.com)

If you need help, investigate Automatix or EasyUbuntu.  I recommend also reading [http://wiki.ubuntu.com/RestrictedFormats](http://wiki.ubuntu.com/RestrictedFormats)

There are certain legal issues that govern the distribution and redistribution of things like multimedia codecs, java, and flash.  Ubuntu does not redistribute them for a number of reasons.
