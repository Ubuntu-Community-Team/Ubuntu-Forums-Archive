Have you asked the forum to help with your problems? They will be able to get you on the internet, give you mp3 support, and help you find drivers.

The deal with the passwords is so that you don't mess up your system, which you can do quite easily in windows (you could delete files from your C:\WINDOWS folder fairly easily). *NIX systems were always intended as multi-user systems, so that's why there's a password for the root user (the administrator) to deal with sys-admin type things. However, compared to Windows, on Ubuntu you are MUCH more in control of your system: all the config files, installs, programs, etc. are customizable, you get to see where every file goes and what every file does, and you can write your own scripts to do what you want. I understand that it takes a lot of work (and that Ubuntu is still under construction) but if you're willing to work and learn, the rewards can be invaluable.
