### Post by jkeyes0 on 2007-03-02
Not too difficult, if I remember correctly.

```
sudo apt-get install kubuntu-desktop
```

at some point it will ask you if you want to use kdm or gdm. If you want to keep using gnome as your default, choose gdm. When you go to log in again, it will look normal, but if you hit Options (bottom left corner) ==> Sessions and choose KDE, it will let you log into KDE instead. You can set it as your default if you like, and change it back if you don't.
