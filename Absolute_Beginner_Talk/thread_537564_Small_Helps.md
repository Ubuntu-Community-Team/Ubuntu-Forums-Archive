### Post by s26c.sayan on 2007-08-29
Although Photoshop won't run natively in linux, you can be able to run it using a compatibility layer software like Wine or Crossover.
Here's an howto for getting it done using Wine : [http://luiscosio.com/how-to-adobe-photoshop-cs2-on-ubuntu-10-steps](http://luiscosio.com/how-to-adobe-photoshop-cs2-on-ubuntu-10-steps)

Also, the  commercial product [Crossover]("http://www.codeweavers.com/products/cxoffice/") can be used to run photoshop in Ubuntu.

However, the Gimp is supposed to be a great replacement for Photoshop and other such apps! You just need to get a bit used to it, and it can do almost anything! :)
