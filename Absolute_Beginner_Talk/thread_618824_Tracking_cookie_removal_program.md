### Post by MyR on 2007-11-20
you can have firefox remove all cookies when you close the browser

click edit > preferences > privacy
under cookies change "keep until:" to "i close firefox"

hope this helps

### Post by ice60 on 2007-11-20
i don't think there are scanners like that for linux.

you could try looking for an adblocking hosts file so you don't get tracking cookies in the first place. or, allow cookies for sites you trust and use session cookies for everything else.

if you use firefox you can install cookiesafe, then go to the global tab and select "Allow for originating website only" that should stop tracking cookies.

### Post by gn2 on 2007-11-20
AdBlock Plus Firefox extension is worth having.
[https://addons.mozilla.org/en-US/firefox/addon/1865](https://addons.mozilla.org/en-US/firefox/addon/1865)

That's pretty much all you'll need.

Some people like to use NoScript as well.
[https://addons.mozilla.org/en-US/firefox/addon/722](https://addons.mozilla.org/en-US/firefox/addon/722)

Firefox can be set to delete all private data at close, you can also do it manually at any time by clicking Tools>Clear Private Data

### Post by philinux on 2008-04-08
The simplest way is to whitelist sites.

Edit>Prefs>Privacy

Make sure "Except cookies from sites" is unticked. Then click exceptions and add your whitelist sites.
