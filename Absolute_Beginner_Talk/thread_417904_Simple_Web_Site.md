### Post by AAM on 2007-04-22
There are a variety used and some have been reviewed in LinuxFormat this month ([http://www.linuxformat.co.uk](http://www.linuxformat.co.uk)) and they include Bluefish (6/10), SeaMonkey Composer (5/10), Nvu (7/10), Quanta Plus (8/10) and Quanta Gold (6/10, commercial), Screem (4/10) and OOo Web (2/10). The final verdict was that the best was Quanta. If you install this you will drag some of the KDE packages along by necessity.

None were considered to be up to Dreamweaver standard.

### Post by aysiu on 2007-04-22
If you must use a WYSIWYG (what you see is what you get) web design program, try out Kompozer (the new name for Nvu):
[https://help.ubuntu.com/community/Kompozer](https://help.ubuntu.com/community/Kompozer)

I'd advise learning a little hand-coding, though. It's not that hard to pick up. I don't program or anything, but I found HTML tags quite intuitive. There are a *lot* of programs to help you with that (the ones AAM mentioned, for example).
