### Post by Stirling on 2006-10-14
Swiftfox uses existing Firefox profiles by default.  So your profile is most likely located at ~/.mozilla/firefox/<Profile name>/ and can be backed up by archiving that folder.  More information on profiles can be found at [http://kb.mozillazine.org/Profile]("http://kb.mozillazine.org/Profile")
