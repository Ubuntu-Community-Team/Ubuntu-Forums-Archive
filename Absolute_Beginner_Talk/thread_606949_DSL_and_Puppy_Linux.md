### Post by sleepingdragon on 2007-11-08
You didn't mention if you got your wireless working in DSL, but it does support ndiswrapper to run Windows drivers for wireless devices, but ndiswrapper does have reliability problems. If you can, find decent drivers you can compile yourself (if you're comfortable with doing that).

DSL does run as a LiveCD, but can be installed to a hard drive, but the installation set-up needs some sound knowledge of Linux and the mechanism to allocate drive partitions is *entertaining* - good luck with that. Saying that, I've had DSL installed on a multi-boot PC and installed inside Virtualbox and both seem to be perfectly reliable and do their job well.

I often do a lot of work with older PC's. DSL will certainly help keep your CPU cycles down, especially running on 388Mhz! The lowest RAM usage I can squeeze out of DSL is 9.5M with a running GUI - 16M during web browsing, so your 192M should be absolutely fine! DSL is very light and every credit should be given to the team keeping it that small. They've certainly made my life easier :-)
