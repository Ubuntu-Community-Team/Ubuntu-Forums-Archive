Ok, I've installed JRE 5on my machine and need to install the java plug in for firefox. Can anyone help this old tried and true windows user to get jre on my new linux machine.  I'm loving linux and ubuntu 6 but would really like jre so I can see the java on web pages.

### Post by taurus on 2007-12-14
Applications -> Accessories -> Terminal
```
sudo apt-get update
sudo apt-get install sun-java5-plugin
```

Now, start firefox and see if you have java 1.5 plugin.
