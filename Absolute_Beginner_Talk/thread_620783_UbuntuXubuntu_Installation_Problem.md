### Post by Jumpmasterrt on 2007-11-23
I had the same problem until I put more ram in. The OS uses less ram true, but to INSTALL, it uses ALL of it. Try a minimum of 512 (I went for the full 1gb). Don't forget that your nVidia video card is pulling system memory as well. Try to install it using the onboard video and if it works, THEN use your card.
