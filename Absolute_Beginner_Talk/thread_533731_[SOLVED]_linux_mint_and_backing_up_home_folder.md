### Post by mikeyphi on 2007-08-24
I believe Mint is ubuntu with any non-open licence packages removed.
Thus there will be some features in ubuntu which are not available in Mint.
So - fixes, yes basically they work, with the aforementioned proviso about omitted packages.

Be careful if you just backup your Home folder - there are a lot of hidden files which may conflict under Mint. There would be no problems if you just backup the 'visible folders'!!
