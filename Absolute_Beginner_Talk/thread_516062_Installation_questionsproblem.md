### Post by p_quarles on 2007-08-02
You don't need to order a CD. Just make a note of the MD5 number on the version you downloaded, then run a Windows-based MD5 sumcheck client on the downloaded ISO. 

Or, even easier (this takes a little longer though) you can run the "check disk integrity" option when you first boot from the disk.

As for a non-GUI based installer: yes. Download the "alternate install" CD, and you will get a text installer. It works very well on many systems that have trouble with the live CD. 

Like others have said, you should burn the disk slowly. This drastically reduces the possibility of burning errors.
