### Post by dreamsINdigital on 2006-04-13
You don't need antivirus or antispyware software simply because there are no (or very few) viruses and spyware for Linux.  Firestarter is just a GUI for Ubuntu's built in firewall (which you should keep and enable).  Go ahead and uninstall Firestarter if you want.  The real firewall will still be there.
