### Post by Hospadar on 2007-10-08
Well first things first, you need to get linux booting off that cd.  Fisrt I would check and see if you burned it correctly, make sure you burned the disc as an image, not just burned the image file to the disk (if you open it in explorer, do you see a .iso file or a bunch of files)

Next, you need to make sure your BIOS is set to boot off a cd.  while your machine is booting up you'll need to go into the bios and make the cd drive the first thing in the boot order.  Usually this involves holding down F12 or delete or something during the boot.  If you arn't sure which and your computer doesn't say, just hold down all your F# keys and delete and you'll get into some menu.  Locate the boot list and put the cd drive first.

This should get you into linux.  If not, let us know more about your hardware and computer setup
