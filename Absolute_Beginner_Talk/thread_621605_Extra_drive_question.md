### Post by p_quarles on 2007-11-24
If the 80 GB drive was connected when you installed Ubuntu, it should already be recognized by the system. If you left everything at the default settings, you should be able to find the drive under the /media directory. If it's there, it'll be easy to get it working, so check that out before going any further.
