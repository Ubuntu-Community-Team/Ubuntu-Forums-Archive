### Post by brunetteredhead on 2007-12-17
Running programs that aren't designed for the window manager you are using can generally cause some problems.  Basically, programs that are designed with a specific manager in mind will usually run best on that manager.  This shouldn't keep you from running programs in managers they weren't designed for.  For example, I run gnome and I use amaroK and k3b all the time, and they were both designed for KDE.  Just experiment, and if you have problems uninstall the program and use something else.

### Post by rsambuca on 2007-12-17
> **EmosSuck777 said:**
> When I'm using Synaptec on Xubuntu and I install things that are made for Gnome or KDE am I breaking something? Do they not work as well as they should?

They should work perfectly fine.  The only time you really need to worry is if you have a lower spec machine.  If you are running a 'KDE' app in Gnome, for instance, then both the QT and GTK libraries have to be loaded into memory, which can bog your system down if you have low ram.

With most machines nowadays it is not an issue, though.
