### Post by otrojake on 2008-04-13
AWN (the dock at the bottom of the screen) relies on Compiz to render itself.  Without Compiz working AWN will not work either.  Going off of the hardware specs in your signature, you should already have Compiz running.  

I'd recommend using Reacocard's repository following [this guide]("http://wiki.awn-project.org/DistributionGuides#Reacocard.27s_PPA") if you want to install AWN.
