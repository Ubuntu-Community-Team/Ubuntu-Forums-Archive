### Post by aysiu on 2007-01-07
Well, Fedora Core is a totally different distribution. Distributions differ in terms of funding, development model, release cycles, package management method, installation method(s), user account setups, and hardware detection, among a few other things. But when it comes down to it--after you've tweaked everything the way you want and installed all the applications you want, if someone sits down at your computer and starts working, it'd probably take installing a new program for her to realize you're using Fedora as opposed to Mepis or Gentoo.

Edubuntu, Kubuntu, Xubuntu, and Ubuntu are all the same distribution but with different default packages. Kubuntu uses the KDE desktop environment and associated applicaitons. Xubuntu uses XFCE. Ubuntu uses Gnome. And Edubuntu uses Gnome and some educationally-related programs.
