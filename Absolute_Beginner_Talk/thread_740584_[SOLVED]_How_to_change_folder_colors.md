### Post by stchman on 2008-03-31

How to change the color of the folders in file browser

There are numerous icon themes that can be downloaded from [www.gnome-look.org](www.gnome-look.org) or you can use the Themes Manager in System--->Preferences--->Theme.

The icons will change the look/feel/color of the folders.
