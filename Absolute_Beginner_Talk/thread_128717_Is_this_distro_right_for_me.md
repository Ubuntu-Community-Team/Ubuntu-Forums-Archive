### Post by Jimmey on 2006-02-12
These questions you're asking could be asked of any distribution - You shouldn't be asking "is this distro right for me", if you've concerns over technical issues, like installing software. 

You can install Debian packages on Ubuntu, as Ubuntu itself was based on Debian. Aptitude comes with Ubuntu. And then there's the repositories. As far as software installation goes, it doesn't get much better than Ubuntu.

RPMs can be installed by the use of a program called Alien. I've never had any experience with alien, but, from what I've seen, installing an RPM with alien is no different to opening a text file with Gedit. 

>  alien /dirOfRpm/nameOfRpm.rpm 

Installing from source can often be a pain. But, when it needs to be done, it needs to be done. Once you've done it once, you find it's not that hard - And if it is, these forums will ease the process a hell of a lot. Compiling can actually really rewarding...To see that after typing just three small commands, a hell of a lot of technical stuff is happning...You feel like a real computer god.

I think you need to get better aqainted with Ubuntu before you ask the "is this distro right..." question. You'll soon see what I mean. It doesn't matter what distro you hop to, software installation's always going to be there. So's compiling from source. The Linux distrobutions are different in their look, their feel, and so on. If you don't like working with Ubuntu, or find that to many things don't work as well as they should, then you consider another distrobution.
