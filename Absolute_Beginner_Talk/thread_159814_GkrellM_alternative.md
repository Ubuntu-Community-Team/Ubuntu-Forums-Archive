### Post by taurus on 2006-04-14
[QUOTE=x5452]how did you get conky to run?  I downloaded it and it briefly put a bunc of light colored characters in the bottom left of my screen that i could read, then dissappeared, and i can find it since?  how do you like set it all up?[/QUOTE]
You need a ~/.conkyrc where you can tell it what to do.  If you need an example, head over to [http://conky.sourceforge.net/](http://conky.sourceforge.net/) and download yourself one.  Don't forget to save it as .conkyrc and place it in your home directory, ~.  And if you want to run it with nautilus (Gnome), then you need to read this page, [http://conky.sourceforge.net/gnome.html](http://conky.sourceforge.net/gnome.html)...
