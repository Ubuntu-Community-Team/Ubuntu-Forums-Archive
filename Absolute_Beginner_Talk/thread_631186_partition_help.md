### Post by kvorion on 2007-12-04
Tell them to use some of the space to install Vista, and leave rest of the space as free and unformatted. Later when you install Ubuntu, you can simply tell the installer to use the available free space. This, I think is the easiest option for people who are new to Linux installations. I followed the same approach for my first few installations till I became comfortable with partitioning.
