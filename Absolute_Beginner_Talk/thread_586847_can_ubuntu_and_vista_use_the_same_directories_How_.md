### Post by montres on 2007-10-22
You can easily install ubuntu in a 10 - 15 GB partition so you shouldn' t worry about disk space. Shrink the partition as much (or as little) as you want.
Gutsy has built in support for NTFS partitons so once you mount your Windows partition (which is an easy process) you will have full read/write capability. Moving files between partitons shouldn't be necessary.
