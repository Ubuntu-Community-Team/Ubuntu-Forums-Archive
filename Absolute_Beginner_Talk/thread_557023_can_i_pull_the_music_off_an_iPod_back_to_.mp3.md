### Post by Overbyte on 2007-09-22
If you didn't check any option in iTunes that's basically "cram more songs into the iPod" or any option to convert to some formats like AAC you can plug it like a normal flash drive and copy the songs off the "iPod_Control" folder. Although the filenames are kinda messed up by Apple as an anti-piracy measure (and to prevent stuff like this from being easy to do). It should retain its quality just as on the laptop.

If iTunes converted those songs before going to the iPod, converting them back to mp3 won't bring any quality back because the formats are "lossy" (it throws stuff out to make the file size a lot smaller).
