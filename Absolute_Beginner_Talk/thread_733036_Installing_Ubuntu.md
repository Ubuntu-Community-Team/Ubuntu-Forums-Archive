### Post by mikewhatever on 2008-03-23
> **Catfish_lolipop said:**
> when I try and install ubuntu onto my computer I get this message "Cannot Display this Video Mode" How can I fix this?

The default driver used by Ubuntu graphical installation CD is an open source vesa or similar. That driver has a lot, but not all possible combinations of resolutions and refresh rates, so that it might not work on a large monitor. You should look up your graphics card info + monitor specs and either post them here or google to make sure they are OK for Ubuntu.
