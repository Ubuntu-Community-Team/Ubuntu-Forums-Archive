### Post by darkali on 2006-06-09
There's one called 'Istanbul'.
To install it just type:

sudo apt-get install instalbul

in the terminal.

To start it after it is installed:
Applications > Sound & Video > Istanbul Desktop Session Recorder.

This will place an icon near the 'clock'. Click to start recording, and click it again to stop it.

The default file it creates is: desktop-recording.ogg
on your home folder.
