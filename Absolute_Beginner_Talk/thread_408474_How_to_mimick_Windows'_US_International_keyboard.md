I have other gripes about the Ubuntu keyboard, but this is the first one I'd appreciate getting some clarity over.

### Post by MyR on 2008-03-29
> **gerdemb said:**
> This doesn't work for me. Am I doing something wrong? I tried alt+comma then c alt+comma+c. The other accents all work correctly &#324;, ñ à etc...

What you can do is add a keyboard indicator to one of your panels.  This can be done by right clicking on an open space on the panel, click add to panel and double click keyboard indicator (under utilities).

The indicator lets you know which layout you are using, can be used to quickly open your keyboard preferences by right clicking on it, and can also show you all the keys on your layout by right clicking and click select "show current layout".  From this you can see how to create every character that the layout is capable of, using combinations of alt and shift.

hope this helps.
