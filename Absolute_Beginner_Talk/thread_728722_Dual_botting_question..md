### Post by Captainm on 2008-03-19
Doing it the other way around (i.e. Vista first then Ubuntu) is easier but this is fairly easy too. First you need to create a partition for Vista to install on. Then install Vista. It'll override Grub with it's own bootloader. Follow [this]("https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows") howto to reinstall Grub. That's it.
