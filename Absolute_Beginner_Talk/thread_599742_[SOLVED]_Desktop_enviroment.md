### Post by aysiu on 2007-11-02
Well, it actually depends on what kind of theme you're trying to install and how it's packaged.

If it's a login screen (or "GDM") theme, then you go to System > Administration > Login Window to install it.

If it's a GTK, Metacity, or icon theme, you go to System > Preferences > Themes to install it.

In both of those cases, it's very important that you don't unpack the .tar.gz archive manually. Just install the .tar.gz file *exactly* as you downloaded it. Do not double-click the .tar.gz or try to extract it yourself.

The one exception is theme packs. Theme packs are not individual themes but collections of themes. So one theme pack may contain a GDM theme, a Metacity theme, a GTK theme, and an icon set. In that case, you will have to extract the .tar.gz

It may help if you post a link here to the theme you're trying to install.
