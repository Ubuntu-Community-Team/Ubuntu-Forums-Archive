### Post by IzzyWizzy on 2007-10-27
I installed Ubuntu 7.10 On Monday and being a long time Windows user I wasn't really prepared for the initial pain in the back end that I experienced installing it on to my HP nx6325 laptop. By Tuesday this was going to be an "I hate this OS" post which is what I felt when I tried Ubuntu ages ago on release 5.something_or_other. 

However I decided this time I was not going to be beaten (plus I bloody hate Windows Vista) so I took Wednesday and Thursday off work to concentrate on getting this to work. To make a really long story short I am glad I did. Having had my stroppy moan about Windows (XP) being easier to install things on, drivers being easier to get and all the other stuff I am sure you have already heard I had a beer, put the CD in again and started from scratch to ensure I had a fresh build.

Well By Wednesday afternoon I had a fully functional laptop with all its bits configured and tested then by Thursday I was, and still am, experimenting with loading applications and using the command line to do pretty much anything I want.

Now on Saturday night I am a full convert to UBUNTU, this OS is a piece of easy to use and frankly if it doesn't work then there are plenty of resources on hand visa Google or the forums. So if you haven't tried a windows free system yet then you should.

I love UBUNTU and Bill Gates can kiss my hairy backside if he thinks I am going back to Windows.

Hey,

Not sure if Ubuntu is the best for newbies like me but it's got to be a top contender. Most, if not all, of my original issues were caused by me thinking I knew what I was doing and not doing a bit of research before I started. A few things took a bit of tweaking like getting WIFi to work and the fact I am using a dual core, 64bit, AMD processor so I have installed the 64bit Ubuntu version caused a few installer headaches but nothing that a little surfing the web couldn't fix. I am learning that the command line is my friend but the package installer and updater is a breeze too. 

Like you I am impressed with the range of built in applications like OpenOffice and the CD/DVD burning software which is as good as any I have used on Windows. I also like the more descriptive error messages in Ubuntu as opposed to the 'it didnt work and we dont know why' errors in Windows.
