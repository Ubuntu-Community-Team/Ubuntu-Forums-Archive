### Post by RomeReactor on 2007-12-16
Hi. To have different wallpapers on each desktop, you can use [Wallpapoz]("http://wallpapoz.akbarhome.com/"); you can get a .deb package [here]("http://www.getdeb.net/app.php?name=Wallpapoz").

The changes in wallpaper as you move from one desktop to another is not very smooth, as it takes about a second for the change to take effect, but you can have different sets of wallpapers on each workspace, and can set them to change every minute (or more; the time can be configured).
