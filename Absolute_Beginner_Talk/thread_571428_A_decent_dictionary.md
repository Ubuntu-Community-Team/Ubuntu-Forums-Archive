### Post by nick_h on 2007-10-09
Try stardict - it is in the repositories.

You will need to download a dictionary from [here](http://stardict.sourceforge.net/Dictionaries_dictd-www.dict.org.php).  Try the Longman dictionary.  Extract the tar file into either /share/stardict/dic  or ~/.stardict/dic
