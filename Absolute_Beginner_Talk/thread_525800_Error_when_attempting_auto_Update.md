'E:Type '‘deb' is not known on line 50 in source list /etc/apt/sources.list, E:The list of sources could not be read.'

### Post by felicity on 2007-08-14
You can check and edit your sources.list by typing:
```
sudo gedit /etc/apt/sources.list
```
in a terminal.
