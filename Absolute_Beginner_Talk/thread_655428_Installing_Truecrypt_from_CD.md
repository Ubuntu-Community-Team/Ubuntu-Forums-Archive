### Post by zipperback on 2008-01-01
> **wenasoga said:**
> Thanks, but that doesn't answer my question....how to install from a CD.

System -> Administration -> Synaptic Package Manager

This will start Synaptic for you.

Now go to Settings -> Repositories

Select the Ubuntu Software tab, then check the box at the bottom of the window for the Gutsy CD rom.

Click Close and then click the Reload button on the top left corner of the application.

You should now be able to install from the cd as if it were a standard repository.
