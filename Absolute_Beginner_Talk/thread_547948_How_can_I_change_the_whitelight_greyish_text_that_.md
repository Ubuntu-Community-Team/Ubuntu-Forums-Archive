### Post by mcduck on 2007-09-10
That sounds more like a problem with the GTK theme you are using, many dark themes have problems like that.

You can try to fix the colors in Firefox, in Edit/Preferences go to Content-tab and click the Colors-button & disable 'Use system colors'.

If that doesn't help try using some other GTK theme.
