### Post by PriceChild on 2006-10-14
To start with... i suggest dropping the use of automatix etc.

The advice off users is fine in my opinion, you can't be expected to know or find everything yourself.

Check out the wiki ([http://wiki.ubuntu.com](http://wiki.ubuntu.com)) or USDF ([http://doc.gwos.org](http://doc.gwos.org)) for guides primarily using the command line.

You will learn a lot more about how your system works by actually isntalling packages yourself, or applying fixes, such as the flash sound fix.

Its a start, and its how i've stayed with ubuntu for a year :)

### Post by aysiu on 2006-10-14
Some people feel you won't learn how things really work if you use Automatix. 

Others feel it's a good option to have in case you want things done quickly, but you should still learn to do things "manually." 

Still others believe that "manually" isn't manual at all--it's just using the package manager to install things, anyway, which is what Automatix does, too, for the most part.
