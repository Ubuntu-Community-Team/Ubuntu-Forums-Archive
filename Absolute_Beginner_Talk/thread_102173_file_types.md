### Post by kori.mendocino on 2005-12-11
.rar's are archives, opened by an archive manager of some sort, the default one, for example, .exe's are binary executable programs for dos/win32 os-es (which can be run through dosbox & winex), .rpm's are software packages for rpm-based os-es like fedora - the equivalent of .deb's in debian, ubuntu, and etc. you can convert to .deb and install a .rpm with a tool called alien.

installing software without apt-get is easy, once you get the hang of it. software source comes in .tar.gz or .tar.bz2, which is an archive, which can too be opened via your archive manager (applications > accessories > archive manager). when you extract the desired piece of software, you'll read the README and/or INSTALL files in it with gedit (the notepad of gnome) and you'll find the installation instructions there. good luck
