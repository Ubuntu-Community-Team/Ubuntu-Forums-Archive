### Post by Nevis on 2007-04-15
I've had this problem too, it seems a lot of people have. Check your java runtime version
```
java -version
```
I had version 1.5.0.8 and after upgrading to version 1.6.0 Frostwire runs fine. Use [**these directions**]("http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_install_J2SE_Runtime_Environment_.28JRE.29_v6.0_with_Plug-in_for_Mozilla_Firefox") for upgrading your java.
