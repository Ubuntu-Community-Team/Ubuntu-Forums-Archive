### Post by petermck on 2007-01-06
I had quite a bit of trouble with this initially, but the links the other posters have given you are the way to go.
I've used Easyubuntu on one machine and did without it on the other. You don't have to use easyubuntu, but if you want everything to work make sure you select the Universe and Multiverse repositries in Synaptic (I would select restricted as well) and then install the gstreamer good bad and ugly codecs and a package called pitfdll. Doing this on my laptop worked fine without EasyUbuntu.
