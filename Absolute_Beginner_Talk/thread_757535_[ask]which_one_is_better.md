### Post by daengbo on 2008-04-17
I'm going to try to push you away from the 64-bit version. Some things like Flash still aren't available in 64-bit, and the compatibility layers don't always work for these things. I find that my 64-bit machine is much less stable for applications like Firefox than the 32-bit machine is. 64 bit is definitely second class. The x86 version will run fine on your Core 2 Duo. You probably aren't using 4GB of RAM, anyway.

WRT the version, install the 8.04LTS release candidate. RC is perfectly stable and there will be very few changes to the system in the next few days. Once that's on and set up, you can leave it unchanged for three years.
