Hmm. i think this was an ubuntu related question. Why move it to Debian etc..?

You can launch your gnome-terminal by giving it any size (geometry) you want.
This is what I use to launch terminal:
```
gnome-terminal --geometry=95x35
```

You can make a new launcher to your panel or desktop, or use Alacarte menu editor (or meny layout in edgy) and right-click on the terminal (in the menu) and select properties. Then edit the command to be whatever you want.. I use the one above..
