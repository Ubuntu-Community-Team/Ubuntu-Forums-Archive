### Post by dptxp on 2007-05-01
7.04 does not use GParted, 6.10 did.
7.04 has its own tool to be used during installation.

If the HDD already has a separate partition for Windows, just proceed with manual partition after clicking Install from LIVE CD.

Else defragment your HDD first with Windows and proceed to install with Ubuntu Live CD.

You can choose manual mode here too. Resize the Primary (or the only 1) partition first. Should be larger than the total data in it.

Next Resize the next partition for root (/). Leave a few GB for SWAP and HOME.

Next for SWAP. Use 2 GB.

Ensure that Format for Widow partition is not checked.

BACK UP ALL DATA BEFORE YOU DEFRAGMENT
