### Post by Uncle Spellbinder on 2006-07-13
After many problems trying to install Breezy Badger in the past, I thought I'd give Dapper Drake a go.

I deleted two no-longer-used partitions and added the unallocated space to the C:Windows partition. I then installed Dapper via the "install" icon on the live cd desktop. The necessary partitions were created automatically (with the exception of the overall "new" partition size, I chose 25Gb). All went *very* smoothly. Sound and web connection were detected perfectly. Updated Dapper and I'm ready to go! 

A very pleasant install experience that took about 20 minutes. Thanks Ubuntu devs for a great, easy to instll OS! :D
