### Post by ndefontenay on 2007-07-03
Hi.

There's no disk 3 with Ubuntu.

You should have only one disk and the ISO name you got looks pretty accurate.

I would suggest you to try to download the ISO once again. Maybe it's simply corrupted. It happens frequently with big downloads...

When the Cd boots, you'll have Install Ubuntu.

Select this. It will load ubuntu in memory without installing it on your disk.

You can try to use it and test that your graphical card is detected (that is yes if you get a graphical interface). Test your internet connection while you are at it.

The installation will take 6 steps and the hardest part will be to select the correct Hard Disk (which is really easy).

The total installation time should take about 20 minutes.
