### Post by Anonii on 2007-10-30
Well, vim is not an everyday-usage editor. It's an advanced feature-rich editor, like EMacs, for developers to write code fast. Just notice that you won't use your mouse at all with vim and that all the required button-combos are in a specific area (You won't need the right part of your keyboard at all. For example, the right arrow (to move to the next right space) can be done with the button "j" in vim.).
So, if you are not a developer or a hardcore configurator you can always use nano or gedit instead of vim/emacs. If you want to learn vim, tho, this spreadsheet will really help you:[http://www.gentoo.org/doc/en/vi-guide.xml](http://www.gentoo.org/doc/en/vi-guide.xml)
