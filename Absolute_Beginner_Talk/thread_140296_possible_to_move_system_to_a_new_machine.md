### Post by m.musashi on 2006-03-05
That's probably not too probable but it would depend on how similar the two are. I just installed a new motherboard with the same chipset. It booted okay but I did a fresh install anyway. Windows found lots of new hardware and I ended up doing a repair rather than a reinstall. If you are going from an old to a new system I would suggest just reinstalling. Of course you can always try and see what happens. Just back up anything you don't want to lose.
