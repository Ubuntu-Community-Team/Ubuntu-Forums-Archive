### Post by RomeReactor on 2007-09-04
Hi. You have to boot from the CD, so make sure to set that option in your computer's BIOS (you usually press DEL, or ESC or one of the function buttons to enter BIOS after powering on the computer. [This document]("https://help.ubuntu.com/community/GraphicalInstall") will walk you through the installation procedure. If you get stuck at any point, don't hesitate to post back.

Don't worry about asking questions. It's sometimes difficult to get started, but we're here to help you! :)
