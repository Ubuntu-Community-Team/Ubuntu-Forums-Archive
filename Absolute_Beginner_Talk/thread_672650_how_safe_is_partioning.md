### Post by eolson on 2008-01-19
If that's your drive,  it apperas as if you're booted under linux now.   If your question is how safe is it to remove your Mandrake/Mandriva and install Ubunto in they're place,  it's not really any more dangerous than your original partitioning.  The easiest way is to boot from a live cd,  run gparted, and kill the two linux partions you have and then install ubuntu.   There is always a chance of something going awry,  so back up any valuable data that you have (on windows also) before you start.
