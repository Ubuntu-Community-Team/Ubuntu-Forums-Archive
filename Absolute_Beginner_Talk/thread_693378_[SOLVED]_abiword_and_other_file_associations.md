I think you've picked a wrong program. As far as I know, AbiWord can work neither with OO nor with pdf file formats. Read AbiWord FAQ [http://www.abisource.com/wiki/FaqOtherFormatsSupported](http://www.abisource.com/wiki/FaqOtherFormatsSupported).

Pdf files can't be edited as text ones and require a pdf editor like pdfedit from the repositories.
