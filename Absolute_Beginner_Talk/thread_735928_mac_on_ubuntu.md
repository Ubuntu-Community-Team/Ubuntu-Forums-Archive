### Post by scramasax on 2008-03-26
> **akimatsu123 said:**
> Or do i have to have a mac processor to be able to run it?

Yes, the processor isn't the issue -- they now use Intel processors not PowerPC processors.

As the previous post notes, it's the licensing.

If you just wanted to play with a different environment and perhaps learn some objective-C coding, then I guess you could install Etolie in a VM:

[http://etoileos.com/](http://etoileos.com/)

That is based on [GNUstep](http://en.wikipedia.org/wiki/GNUstep) and, as I say, uses objective-C, which is what Apple's Cocoa frameworks use.  It also follows some Mac interface paradigms.  There's unfortunately little software around for GNUstep, although it's actually been around for a very long time.  But GNUstep has been accepted into Google's Summer of Code project, so it's not exactly moribund.

Of course, if your interest is in OS X as such and in running Mac apps, you're out of luck.
