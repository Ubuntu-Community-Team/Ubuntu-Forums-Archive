### Post by dbd on 2007-01-12
Hmm, I'm not sure what you can do. With normal monitors you can configure the display and resize it so that everything shows correctly using buttons on the monitor, but I've not idea how to do that with a laptop (and it seems like that is the problem).

To get a screen shot press Alt+F2 and then type in gimp This will open the GNU Image Manipulation Program. Then from the gimp File menu choose Aquire->Screen shot. Then take the screen shot and save it to the USB drive (which should be auto detected when you put it in.) But that won't help you take a screen shot of the splash screen, I'm not sure how to do that. Also if it is the screen problem which is suspect then the screenshot will look fine, and will show what is not visible on your screen.

Also, it might be worth trying different colour depths and refresh rates as well as resolutions. What options is it giving you? 

And when you have it properly installed you may be able to update your graphics driver (tough, but possible, when using a live disk) with which you may have more luck, so it may be worth trying to install first and then fixing this problem from there.

By the way, according to the xubuntu web site ( [http://www.xubuntu.org/get](http://www.xubuntu.org/get) ) you need 192Mb to install it using the live CD. You will probably be OK with 160Mb, but when you come to installing if your have any problems then you may want to try the alternative install CD.
