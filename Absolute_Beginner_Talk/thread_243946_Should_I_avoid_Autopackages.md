### Post by qamelian on 2006-08-25
Although many people claim that Autopackage breaks things, I've neve experienced any problems with it. There a a couple of applications that aren't in the Ubuntu repositories that I use and all other non-Autopackage solutions have proven beyond painful for me.

As for uninstalling, Autopackage provides the uninstaller. When you finish installing the package, the command for uninstalling it will be displayed.

### Post by ngch on 2006-12-01
Personally, I don't have any problem with Autopackages so far. I have installed aMSN and Abiword with autopackages, and so far they have not given me any problem. In fact, I can only edit my .odt files with the autopackaged version of abiword!

The only cringe I have though is that the menu entries do not have icons. But I can live with that.

To uninstall it, you have to go to "Applications > System Tools > Manage 3rd Party Software". It's quite simple really,just select the program you want to remove. :)
