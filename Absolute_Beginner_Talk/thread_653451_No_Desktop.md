### Post by JoshuaRL on 2007-12-29
Well xorg.conf is the configuration file for the Xserver desktop application that runs your GUI.  Be careful with it, but it's pretty useful too.  Type this into a terminal

```

sudo gedit /etc/X11/xorg.conf

```

and look at what it says for the section that looks like

```

Section "Device"
	Identifier	"Generic Video Card"
	Driver		"vesa"
EndSection

```

except for your video card name and driver.  I'm not sure what yours should be, but I'll go do some research and see.

```

# if it has not been modified since the last upgrade of the xserver-xorg
# package.

# again, run the following command:
#   sudo dpkg-reconfigure -phigh xserver-xorg

Section "Files"
EndSection

Section "InputDevice"
```
