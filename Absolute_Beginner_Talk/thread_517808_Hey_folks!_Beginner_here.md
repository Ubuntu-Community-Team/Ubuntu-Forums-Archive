### Post by superstylin on 2007-08-05
Hello Sal, welcome to Ubuntu!

1 - Preferred version, I think you would be fine with 7.04 (Fiesty) provided you arent running a business computer, or "mission critical" as far as OS stability goes :)

2 - Leave your live CD in the drive and restart the computer, it should automatically boot from the CD, similar to a windows installation (actually easier :P), if this isnt occuring, I would go to your bios options (press DEL on startup) and make sure your BOOT order is CDROM -> HDD -> etc :)

3 - As far as I know you wont need any specific hardware, however, if you have a wireless card, expect to be patient :)

4 - Add these forums to your Firefox favorites bar, visit them frequently, if you have any questions, I usually google it, and you'll find a ubuntu wiki site... >.> 

5 - ill save you some heartache... 

sudo = means Super User Do, took me forever to find out... -_-

ctrl-alt-backspace = restarts X, think of X as the visual component of Ubuntu, so if your running an application and it freezes, you can restart the visual desktop via ctrl-alt-backspace, but it wont restart the computer, its kinda confusing, you can try it out :P

ctrl+h = Show hidden files, I learned this the hard way :P

compiz and beryl (I believe) are both now unsupported, and if you want fancy desktops, make sure you find Compiz Fusion in the Synaptic Package Manager!


Enjoy!
