Maybe I can help.

Can you access your computer's BIOS? If you can, you should be able to change the boot order so that your computer will look to the CDRom before it looks to your hard drive. 

Here is a link showing examples of how to change the boot order in your BIOS: [http://www.windowsnetworking.com/j_helmig/bootcd.htm](http://www.windowsnetworking.com/j_helmig/bootcd.htm)

Once you've changed the boot order so that your CD is read on boot up, pop it in, restart your PC and the CD will start up for you automatically. Follow the on-screen prompts and wa-la!

Good luck!
