### Post by BrendanM on 2007-09-10
If you're asking about the choice that says "Start or Install Ubuntu" then the answer is YES. All that does is start the Live CD. You'll be able to have the whole operating system running from the CD, without messing up your XP install.

If you decide you want to install it on your hard drive, you can just click the install icon on the desktop.

You might also like to check out Wubi: [http://wubi-installer.org/](http://wubi-installer.org/)  which is another great way to install Ubuntu alongside Windows.
