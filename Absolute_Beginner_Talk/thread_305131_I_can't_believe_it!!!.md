### Post by PurplePenguin on 2006-11-22
You are right, it is all quite easy.  I'm a new convert from OpenSuse. :)  I've never seen a cleaner, more intuitive desktop OS before.

The only recommendation that I'd make to new users is to check out [Automatix]("http://www.getautomatix.com/") in order to painlessly install almost everything else that doesn't come "out of the box" (nvidia driver, mp3/dvd support, acrobat reader, etc).  Between this and the software available in the Add/Remove, you're all set. :)
