### Post by az on 2005-12-31
[QUOTE=MaaSTaaR]Hello :)

i have IBM PC with Intel PIII (about 500 Hz)

my question is will ubuntu work fine with this old PC ?[/QUOTE]
Yes.  The minimum ram required is 128 megs, but you will be much happier with 256 megs.

I run the default install (not a lightweight desktop like XFCE or Icewm) on such systems.  I would only consider a lighter system if I were running on something like a 300 MHz Pentium.  

I have not found any issues whatsoever running Ubuntu on a 500 MHz penitum.  I think once you go below that, you start to hit the threshhold where the desktop starts to become less responsive and annoying.

Unless you are looking for it, there is not much difference running a 2.4Gig processor or running a 500 MHz proessor for everyday desktop tasks.
