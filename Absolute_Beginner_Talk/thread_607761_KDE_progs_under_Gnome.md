### Post by rsambuca on 2007-11-09
The one case where you would definitely want to avoid using mixed libraries is if you have low system specs.  If you have low Ram, then running the KDE and Gnome libraries at the same time can really slow your system down substantially.  It really isn't an issue with today's modern rigs, though.
