### Post by psi36 on 2007-01-21
I've added Skype and Gaim to my list of startup programs. They are started when I boot Ubuntu, but I'd like them to start in the background. So just have the icon in the bar on the top right, but not in an opened window as it is now. How can I do this?

### Post by gavintlgold on 2007-01-21
This should work (did for me):

1. Open gaim as usual.
2. Click on the gaim icon on the top right, to close the gaim window (unless it was already closed).
3. Make sure the window is closed when you log out.

The next time you log out, it should be closed already. Just every time you log out, make sure it's closed. I think this will work.

### Post by casaschi on 2007-01-21
> **psi36 said:**
> I've added Skype and Gaim to my list of startup programs. They are started when I boot Ubuntu, but I'd like them to start in the background. So just have the icon in the bar on the top right, but not in an opened window as it is now. How can I do this?

Install the "extended preferences" plugin of gaim.
This gives you the option to start gaim minimized to the tray (check the "hide buddy list at signon" in the configuration of that plugin).

Similarly, there is a checkbox in the main option window "dont show main window at startup" to start skype minimized.
