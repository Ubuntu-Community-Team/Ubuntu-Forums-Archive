### Post by Stephen Howard on 2007-11-26
You can run gnome programs under kde, and the other way round.  All you're doing is running a program that calls a gnome library or a kde one, and so long as the library is installed the desktops will work happily.

The first time you install an application, apt-get will install all of the dependent libraries.  So the first install might take a bit longer than usual.

No idea about the keybinding - are you saying that the keyboard map is wrong?  Are there dead keys?
