### Post by energiya on 2007-01-05
I use 2 programs:
**dosbox 
- easy to configure
- runs only in X
- eats CPU
- works nice with games

**dosemu 
- works in X or terminal (no gui)
- faster than dosbox
- a pain in the * to configure

to get them, just type in a terminal window
```
sudo aptitude install dosbox
```
or
```
sudo aptitude install dosemu
```
