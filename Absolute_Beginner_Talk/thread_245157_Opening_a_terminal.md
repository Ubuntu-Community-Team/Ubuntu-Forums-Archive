### Post by Anonii on 2006-08-27
> **Andrew07 said:**
> Okay, this is prolly a complete newbie question. How do you open a terminal in ubuntu? I've looked everywhere, but I can't find any icon/button or anything that will let you open one.


If you have GNOME:

Applications -> Accesories -> Terminal

Also you can use: <Alt + F2> and then type: "gnome-terminal" (without quotes) inside that box, and obviously then, ok.

If you have KDE:

You can use: <Alt + F2> and then type: "konsole" (without quotes) inside that box, and obviously then, ok.
