### Post by ZeroWing on 2007-05-09
Firefox used to be known for being a RAM hog, slowly taking up as much RAM as it can, making it slow, along with any other application that is currently open. In Firefox 2.0x, this was supposedly solved. Some people, however, still complain about it taking too Much RAM. You can limit this by going to About:config in your browser and looking for browser.cache.memory.capacity. Set this to what you want it to be limited to(in KB).
