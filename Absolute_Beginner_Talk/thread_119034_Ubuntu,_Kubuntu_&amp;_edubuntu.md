### Post by Sef on 2006-01-18
>  whats the difference between the three Ubuntu Builds (Ubuntu, Kubuntu & Edubuntu), 

Ubuntu: Ubuntu with Gnome

Kubuntu: Ubuntu with KDE

Edubuntu: Ubuntu with Gnome with packages for an educational environment

> and which would you consider the most user friendly?

Since you are using Gnome, I would have them use it as well.  Would be easier for you when they had a question about how something works or how to do something.

### Post by jonathanm on 2006-01-18
Not wanting to confuse but i think there is actually 4.

Ubuntu, Kubuntu, Edubuntu and Xubuntu.

Ubuntu uses Gnome as its desktop manager,
Kubuntu uses KDE
and Xubuntu uses XFCE

Edubuntu uses Gnome like Ubuntu but comes preloaded with educational programs, with regard to you're question i personally prefer ubuntu because it is cleaner and my computer has problems with kde.  I also have xfce loaded which i like and is fast but really it depends what the person using it wants.

If you can, try a simple install, with no automatix or anything and load on as many desktop managers as you can find, have a mess about with them and see which one they prefer.  Then you can do a clean install and just load the one(s) that they want
