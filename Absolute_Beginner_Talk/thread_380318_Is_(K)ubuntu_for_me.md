### Post by maniacmusician on 2007-03-10
> **miggols99 said:**
> I really like KDE because I think the applications are better. I heard that if you run KDE programs in Gnome, it make it slower because it has to get to the KDE and Gnome libraries at the same time. Is that right?
You can use KDE with your setup. You just need to disable the obnoxious effects that come with it. The first step is making sure you have access to a control center. In KDE, right click on the bottom panel > Configure Panel > "Menu" Tab > Tick off the "Settings" Menu. 

Then, open the KMenu (the menu on the bottom left) and there will be a submenu called Settings, with lots of options in it. Click on "Control Center". It'll give you access to pretty much everything that can be customized in KDE. You can go through the options and choose which ones you want, which ones you don't want, etc. If I were you, I would turn off the bouncing cursor and disable the "mouseover effects" for the panels.
