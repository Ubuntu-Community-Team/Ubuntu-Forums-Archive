### Post by Enverex on 2007-05-09
No, Gentoo isn't based on Debian. It's also a source based distro, not binary.

"Technically" Gentoo is faster simply because you start with nothing and build from there and any apps you install only have the parts of those apps that you've told it to build, not everything which is how Debian packages are built. Although the tradeoff of "runtime" speed is paid for in that you have to compile every single package on your machine, which, if going from scratch to a full desktop system even on a high-end machine will take days.

Gentoo's speed doesn't come from "-O2 -march=awseome -funroll-loops -omgsofast" compile flags, it comes from the fact that your system only has the exact bits installed that you specify. 

Think of it like this:
Ubuntu and Gentoo are both houses (house being distro). They both have many rooms (packages). In Ubuntu when you add a room it adds everything to the room, chairs, tables, wallpaper, decorations, etc. In Gentoo you select every little piece that goes into that room and only have what you've selected.

Don't take the size of a community as a guage for a distro. Ubuntu may have a bigger more active forum than other distros, but it also carries a much larger volume of "noise".

Anyway, all that being said, Gentoo is only for people that are experienced with Linux. If you don't know your way extensively around a terminal and terminal apps then you'll just not get anywhere.

### Post by opus on 2007-05-09
I used Gentoo a while back.  It is a fantastic operating system.  It is a GREAT way to learn Linux.  You have to essentially build and install everything from scratch.  Sound intimidating?  Not so much - there is fantastic documentation on the Gentoo website.  The install doc walks you through the process step by step.  Not only does it tell you what to do, it tells you WHY you are doing it.  It's a great way to learn what is going on under the hood.

It's very easy to keep updated, the same way Ubuntu is.  The difference is that Gentoo has to compile everything every time it gets updated.   Which can take a while sometimes (like when KDE, Gnome, or x.org get updated).  

One major difference is that there are no versions of Gentoo.  You are always running the latest version.  This is compared to Ubuntu where there is a new release every 6 months, and older versions don't get newer software.

Gentoo also has the latest version of all packages.  You are never a version behind on Gentoo.  Some think that's a plus, some think it's a minus.  

I like Ubuntu because of the quick updates, and not having to deal with etc config file conflicts (which happens in Gentoo).  

Hope that helps!
