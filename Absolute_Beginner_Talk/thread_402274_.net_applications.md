### Post by LaurelLynn on 2007-04-05
Dear BillGod,

What your looking for is Mono,  Novell´s open source implementation of .Net. It includes a C# compiler, runtime library and other tools like MonoDevelop(IDE).

It does not implement all of .Net yet, so not all applications developed under windows will run as is.  All Mono developed apps can however be run under Windows .Net (read their licensing for restrictions).
