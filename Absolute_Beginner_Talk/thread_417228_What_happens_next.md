### Post by annda on 2007-04-21
if you have only this one partition on your disk, you can
1) back up any important data
2) defragment the disk under windows
3) tell the installer to use free space on the disk. 

don't worry, you can resize the partitions later if you don't like how ubuntu handled the space.

next steps will be fairly obvious. only partitioning requires important decisions.

if you still want manual partitioning, post back.
