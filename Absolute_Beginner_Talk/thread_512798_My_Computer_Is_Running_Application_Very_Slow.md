### Post by bigfox on 2007-07-29
Go to System -> Administration -> System Monitor.

Go to the Processes tab.

It will display a list of all processes.  It will also show you the % of CPU power each process is using.

Find the one that is hogging it all and tell us what it is, or just end the process.
