### Post by asmoore82 on 2007-08-25
mplayer

```
~ $ mplayer -vo null -vc null -ao pcm:fast *<anyAVfile>*
```

this will create a file 'audiodump.wav' with the audio uncompressed
(decompressing mp4 and then re-encoding to mp3 is a big potential loss in quality)
((the burning software has to decompress the file to burn a CD_DA anyway))

just make sure you delete the WAVs after burning.
