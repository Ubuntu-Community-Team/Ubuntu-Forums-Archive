### Post by DarthWHO on 2007-04-19
You want to burn these as DVD Video. I personalize use GnomeBaker, but there are various pieces of software that can do this. Burning as a data DVD will not work I don't believe.

Install GnomeBaker (or other software), click Make DVD Video and copy the files created into the video_ts folder.
