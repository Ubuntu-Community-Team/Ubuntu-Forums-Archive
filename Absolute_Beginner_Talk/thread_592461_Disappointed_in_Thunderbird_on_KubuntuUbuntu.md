### Post by kellemes on 2007-10-26
You need to put these lines in /etc/apt/sources.list.. there not commands.. just lines in this file.
```
deb http://ubuntu.iuculano.it feisty thunderbird
deb-src http://ubuntu.iuculano.it feisty thunderbird
```

Depending on the editor you have installed on your system.. assuming you're on KDE surely you have Kate.

```
kdesu kate /etc/apt/sources.list
```
**or** (if you know how to handle vim)
```
sudo vim /etc/apt/sources.list
```

This opens sources.list in your editor with root-privileges, now you can add these lines to the file and do "sudo apt-get update" or push the update/reload-button in synaptic (or something like that, I forgot)
Now you have these repositories enabled!

Feel free to ask.
