### Post by Kateikyoushi on 2006-09-27
You use sudo in terminal and gksudo while using the GUI.

Apt is the debian package manager and aptitude is a frontend for it which adds some extra functionality.

### Post by 3rdalbum on 2006-09-28
Instead of "sudo nautilus", use "gksudo nautilus". This way, you can put a launcher on your panel, or an entry in your Applications menu, which runs the command "gksudo nautilus".
