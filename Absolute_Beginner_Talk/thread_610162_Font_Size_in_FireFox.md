### Post by NilsE on 2008-01-02
FF and Ubuntu render fonts differently.  go into about:config and search on DPI.  Change the value of DPI from -1 to 86, **restart FF** and they should be about equal to the desktop font. You can always try different values or put it back to -1.

### Post by jryprt on 2008-01-02
In Firefox, Edit-->Preferences-->Content, you can change the font and size. Go to Advanced, and uncheck the "Allow pages to choose their own fonts" box and choose a font and minimum font size you like.
