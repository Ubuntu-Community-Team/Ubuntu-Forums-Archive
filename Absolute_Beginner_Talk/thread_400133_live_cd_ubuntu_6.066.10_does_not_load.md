### Post by jem7v on 2007-04-03
Or! You could just install Kubuntu, and then go into Synaptic (or whatever package manager it uses, but I assume it's the same one) and install the ubuntu-desktop package, which will install Gnome for you and you can try them both out. (You just decide which one to use whenever you log in.) Once you decide which you prefer you can remove the other.

[http://www.psychocats.net/ubuntu/gnome](http://www.psychocats.net/ubuntu/gnome) is about adding Gnome.
[http://www.psychocats.net/ubuntu/puregnome](http://www.psychocats.net/ubuntu/puregnome) is about removing KDE so you just use Gnome.
[http://www.psychocats.net/ubuntu/purekde](http://www.psychocats.net/ubuntu/purekde) is about removing Gnome so you just use KDE.
