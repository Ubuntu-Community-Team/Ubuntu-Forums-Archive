### Post by linbetwin on 2005-11-14
[quote=woogyboo]but how do i use the install disk?[/quote]
 
Insert it into your CD-ROM drive, reboot your computer, enter your BIOS settings to make sure your first boot option is CD-ROM, otherwise your computer will boot from the HDD. Save your BIOS settings and exit it. The computer will boot into the Ubuntu installer. Type "server" and press ENTER (if you want to install Ubuntu as server) or simply press ENTER (if you want to install the desktop version. The rest should be pretty simple. It was for me, and it was the first Linux distro I ever installed.

Does the computer specifically ask for a ubuntu disk or is it just looking for a "boot disk" in general.  You may have told your bios not to boot from the harddrive anymore before you installed ubuntu.  Try going into the bios and selecting your first boot device to be the harddrive. If your computer turned off booting from the harddrive things might get confused.
