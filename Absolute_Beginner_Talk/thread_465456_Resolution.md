```
sudo dpkg-reconfigure xserver-xorg
```
This will take you through a setup process.  Pretty much just press Enter on your way through, but make sure you read first.  TAB can move you between options that may appear, Spacebar can select if appropriate.

### Post by Rocket2DMn on 2007-06-05
xorg should be making its own backups.
look for files in the /etc/X11/ directory labeled like: xorg.conf.[date_goes_here]
you can restore a backup like so:
```
sudo cp /etc/X11/xorg.conf.backup /etc/X11/xorg.conf
```
This will overwrite your xorg.conf with the backup.

### Post by Rocket2DMn on 2007-06-05
Sorry for the delay, I just got home from work.

For example: "xorg.conf.20070529200722"
that is a backup file made 5/29/2007
to list the files in the X11 directory type in terminal
```
ls /etc/X11/
```
You should see different xorg.conf.* files where * represents numbers and/or names

Restoring a backup like I stated above should at least get you back to where you were.  
I don't remember the steps off the top of my head that follow giving the reconfigure command, but it's pretty self-explanatory.  The primary thing you need to change is the resolution options when they pop up.  This is when TAB and SPACEBAR come in handy - you need to have an "X" in the correct resolution boxes - you will know it when you see it.

I've checked around briefly for a good guide on doing the reconfigure, but I suggest you just go ahead and do it.  I'll post if I find a good one.
Good luck.
