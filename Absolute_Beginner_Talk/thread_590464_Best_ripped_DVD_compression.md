### Post by ramjet_1953 on 2007-10-25
I've had good success with [COLOR="Blue"]k9copy[/COLOR] to compress a DVD9 to a DVD5. The compressed disk is virtually identical to the original.

In a terminal type this in:

[COLOR="Red"]sudo apt-get install k9copy[/COLOR]

For burning the iso to disk, I would recommend [COLOR="Blue"]k3b[/COLOR]

In a terminal type this in:

[COLOR="Red"]sudo apt-get install k3b[/COLOR]

Both of these applications are KDE based, but work fine under GNOME.

K9Copy has an option to burn directly to disk, but many have had problems with it. I have found it is better just to have K9Copy make the iso and burn it with K3b.
