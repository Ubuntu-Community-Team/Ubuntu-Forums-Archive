### Post by Nekiruhs on 2007-05-22
Whats, the difference between the commands apt-get and aptitude? I've been using apt-get to install things but alot of people on the forum use aptitude in their posts. Is there a difference?

---

### Post by Kobalt on 2007-05-22
Yes, there is a difference in the way they handle dependencies for example. You can search the forum to find technical explanations if you'd like, but my simple advise would be to use *apt-get* to install regular packages and *aptitude* to install meta-packages (such as ubuntu-desktop, kubuntu-desktop, etc.).
