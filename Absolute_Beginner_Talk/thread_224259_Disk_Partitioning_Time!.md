### Post by molly_001 on 2006-07-27
> **dolbinau said:**
> How long does it usually take from this point? Several hours :x ; Is this normal?

About two hours if you are on a super slow PC.  I'm talking a Pentium II or Pentium III of a speed of a clock speed of 500MHz or less.

> **dolbinau said:**
> Will it be easy for me to increase the size of this partition using more free space I will accumulate in the future :)?

I don't advise changing the partition size after you have created a partition for installing Ubuntu.

> **dolbinau said:**
> I'm currently surfing the web using the 'live' CD kind of thing, is this safe to do while partitioning?

I definitely would not do that!  Also, be aware that using the install option from the Live-CD has a pretty high failure rate.  You'll do much better to use the Alternate Install CD for Ubuntu 6.06.  It also gives you much more control over the installation.
