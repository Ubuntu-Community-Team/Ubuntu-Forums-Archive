### Post by skymt on 2006-09-20
> **geofrey said:**
> im a oneday old user of ubuntu OS...can anyone from this forum who can help me how can i install some of my program that i use in windows OS, like microsoft office, adobe products, and many more programs that i want to install...thanks for your help...

There are free Linux equivalents for almost everything, that would run much better on Linux than Windows versions would.

Microsoft Office = OpenOffice, which comes with Ubuntu, so you already have it.

Adobe Photoshop = The GIMP (Gnu Image Manipulation Program)

Adobe Illustrator = Inkscape

Adobe Reader = Adobe Reader (they have a Linux version)

You can install many of these, and many more, by going to your Applications menu and choosing "Add/Remove Applications". Browse through there and see what you like.
