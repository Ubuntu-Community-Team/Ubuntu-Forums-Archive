### Post by jeffus_il on 2008-02-25
Type ```
man man
``` in a terminal to get the man page of the man command itself. There is a tabular explanation of the meaning of the different numbers.
If apropos gives man(2) the command will be ```
man 2 man
``` (excuse the non politically correct command) should be ```
person 2 person 
```
