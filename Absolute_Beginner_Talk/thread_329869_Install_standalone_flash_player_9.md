### Post by Canis familiaris on 2007-01-02
[Follow this link and the instructions]("http://www.adobe.com/shockwave/download/download.cgi?P1_Prod_Version=ShockwaveFlash") to install flash player and copy the tarball to home directory.
When it asks you to enter the directory for browser, for firefox 1.5 (default with ubuntu), ```
enter: /usr/lib/firefox
```
and you can run flash!
