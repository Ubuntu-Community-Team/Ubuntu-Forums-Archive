### Post by amazingtaters on 2007-08-13
When you download ubuntu you are gonna be dl'in a single .iso file. This file can be anywhere on your computer, it really doesn't matter. You will use infrarecorder to burn that image to a cd (do not burn a data cd). Then you will pop the cd you just burned into the cd drive of the computer you want to install ubuntu on. Your computer will boot from this disc into liveCD mode and from there you can run the install process, there is a launcher on the desktop. Hope that helps ya.
