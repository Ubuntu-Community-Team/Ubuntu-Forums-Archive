### Post by oldos2er on 2008-02-01
System, Preferences, Sessions. Click 'Add,' type in 'firefox.'

---

### Post by jaytek13 on 2008-02-01
Go to system->preferences->session, then click add. Name it whatever you want (probably firefox) then under command type ./firefox
