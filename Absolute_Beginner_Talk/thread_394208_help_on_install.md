### Post by Arby on 2007-03-26
When you boot the liveCD there should be an icon on the desktop labelled 'install'. If that isn't there then try looking through the menus for an install option. You should definitely be able to launch the installer from a running liveCD. If not something strange is happening. In Kubuntu (the kde version Ubuntu) 'Install' is found under Kmenu -> System -> Install. I'm not sure where it is in gnome, I would guess System but I don't actually know.

### Post by arron on 2007-03-26
You need to boot of the CD.  You may have to change your BIOS boot order to do so, put the CD drive in front of the hard drive.  Then it will boot up of the cd, and give you a ubuntu gnome desktop.  If you are looking at your E: drive from windows, you missed a step, as well you dont need to partition from windows, the live cd can do it all.

Here is a good howto once the cd is booted:
