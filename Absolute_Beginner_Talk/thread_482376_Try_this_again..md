### Post by molly_001 on 2007-06-23
It's a graphical problem.  Get the June, 2006 version of Gparted, ver. 2.5-2 .
It has same functionality but Xorg seems to be much less of a bother with some graphics chips.
I have had the issue you describe with the current version of Gparted.
Yet when I pull out my trusty June 2006 version, she fires up and I can see the whole shootin' match.
Also try Xvesa if Xorg does not work your first run of June 2006 model.
