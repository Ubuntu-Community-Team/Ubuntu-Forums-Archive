### Post by Skidpad on 2007-05-14
You can also disable IPV6.  This has been discussed before on here, and has worked for me.

Type about:config in the address bar, and search for IPV6.  It should come up under **network.dns.disableIPv6**.  It will read "false" by default.  Double-click it to read "true".  Restart FireFox.
