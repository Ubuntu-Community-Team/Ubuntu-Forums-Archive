### Post by onioneater36 on 2006-09-28
Correct.  Just select the option in your burning software called BURN ISO IMAGE TO CD.  Do not extract anything...  Do not make anything "bootable".  That happens automatically when an ISO image is burned.  Think of an ISO image as a "snapshot" of a CD that is restored when BURN ISO IMAGE TO CD is selected.
