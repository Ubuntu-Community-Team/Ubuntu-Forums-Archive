### Post by LaRoza on 2007-05-22
Having two hard drives is easy. 

Ubuntu will format and partition during the install, so you do not need to concern yourself with the actual setup. 

CDs and DVDs have their own format, and can be used anywhere that they will normally be played.

If you want to make sharing easier, having a separate partition for the shared files is a good idea.

If you are concerned with sharing, use FAT32, makes it easy if you're willing to accept the file systems.
