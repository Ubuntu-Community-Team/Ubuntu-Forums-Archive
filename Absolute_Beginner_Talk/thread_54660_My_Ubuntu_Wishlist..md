### Post by Ampersand on 2005-08-05
Breezy now has a reasonably easy to use hard drive mounting program, it has a list of your drives, and you can set mount points for them.

I'd like to see some beep media player plugins availablethrough apt. I currently have the choice of xmms and its various plugins, or BMP and its japanese language support. I've started using "sleep 8h && beep-media-player -p" instead of xmms alarm.

It would also be useful if synaptic was slightly easier to navigate. It's currently not much use if you don't know what package you're looking for, having slightly more specific sections (web browsers, email clients, video players &c) like Yast does might make it slightly easier for new users.

Unfortunately, the features freeze for Breezy is a few days away, so this will probably have to wait until October/November...
