### Post by Wolki on 2005-10-09
Tis is a bug in Gnome 2.10/Hoary. FileRoller, the Gnome program that archives/extracts files does not work with unrar in any form or shape. You need the rar package installed (in multiverse, use synaptic/apt-get) to extract them graphically. This package is shareware, so remember to remove it after 30 days unless you register.

After installing it, extracting a rar should be as easy as right-clicking it and selecting extract here.
