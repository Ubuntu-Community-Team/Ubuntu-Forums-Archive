### Post by Helgiks on 2008-03-26
1. Video Games (mostly of the Steam variety) - Try wine or cedega
2. AIM - Kopete

I actually sent you a pm about it. But as some people might be reading this and want to know why they should use KDE applications on kUbuntu and GNOME appliactions on Ubuntu.

Part of the reason is because they are built with different perspective. Not same Look and Feel.
 [Here]("http://library.gnome.org/devel/hig-book/stable/") is a link to how GNOME applications are supposed to be made.
[ Here]("http://wiki.openusability.org/guidelines/index.php/Main_Page") are unfinished guidelines on how KDE applications are supposed to be made.


KDE applications need KDE libraries that do not come with Ubuntu, so you need to install them (more disk space)

There have been made some posts about it on this forum made by much more linux savvy people than me... like people which you can trust :D:)
