### Post by richardhwinter on 2008-01-19
Kind of found the answer although I do not like it.

Looks like it is not possible to save the module in Excel format. OOo does open and execute VBA written in Excel but does not save any changes or new modules back to the .xls format.

As I still need to work with MS products @ work and need some files to work in both environments I still prefer to save as .xls.
Solution is: only change macros in Excel.
