### Post by Woody AZ on 2008-03-07
Hi Benbow,

If you're looking to get dohickey into the applications menu, right click applications and select 'edit menus.'  From there you can click the new item button, which will prompt you for type and name of dohickey.  You'll have to browse to the dohickey binary, which depending on what it does it could be in a number of places.  You may want to read up on the find, updatedb, and locate linux commands to help you out with that.  The 'whereis' command is pretty  helpful as well.

After adding the new dohickey item, you should be able to launch it from your applications menu.  I hope this info helps you out.

Take care and enjoy Ubuntu!
