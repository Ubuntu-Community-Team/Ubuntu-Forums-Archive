### Post by lamego on 2006-09-23
If you mean MD5 they are not software files, they are just validation files that you can use to check a file was properly downloaded, it is common to be used with big files like .ISOs.
To get the md5 sum of a file open the terminal and type: md5sum file
Then just check against the value on the .md5 file.
