### Post by forrestcupp on 2008-02-09
Nautilus is the default file browser in Gnome.  If you are wanting to open a terminal from a directory in nautilus, you can install an add on for that:
```
sudo apt-get install nautilus-open-terminal
```Then you can right click in nautilus and choose to open a terminal.

Edit:
You have to restart your computer or at least Gnome for it to appear in your right-click menu.
