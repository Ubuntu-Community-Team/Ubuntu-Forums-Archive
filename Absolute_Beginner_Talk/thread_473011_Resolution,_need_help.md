### Post by phr0ze on 2007-06-19
Sorry, I assumed more due to detail you had in earlier posts. Open a terminal and type
```
gksudo gedit /etc/X11/xorg.conf
```

Then find the monitor section and change it to look more like this. This should involve just adjusting the numbers that are there.

```
Section "Monitor"
	Identifier	"Generic Monitor"
	Option		"DPMS"
	HorizSync	30-64
	VertRefresh	50-100
EndSection

```

Finally in the next section you will see all kinds of available resolutions. I'd remove resolution values other than 1024x768 and 800x600. But you can try the first change above and see if it works before removing the extra resolutions.

To test your changes, make sure you have saved the file and press ctrl-alt-backspace.

Again, I'm sorry for not explaining this better.
