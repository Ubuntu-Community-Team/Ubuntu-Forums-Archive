### Post by Carlos Santiago on 2006-12-04
You wrong! Not in choosing Ubuntu but in coping the files into a directory an try to install them from there.

A few notes:
1) Ubuntu Installer CD (the *.ISO file) is bootable, and you should boot your computer from CD.

2) Get a 700MB CD and burn it.
     i) dont copy the ISO file into de CD
    ii) open the ISO file with your CD burner program
   iii) try double click on the ISO file
    iv) is not impossible to install directly from the ISO file, but it is really much easier to install from CD. And c'mon, a 700MB CD is not so expensive!!!

3) Start (or reboot) you PC with Ubuntu CD into the CD drive

4) It will start Ubuntu Live system, that means you can try Ubuntu without touching your hard disk.

5) Then, if you want to install Ubuntu into your hard drive, you just have to double click on icon 'Install'

6) VERY IMPORTANT: Before installing Ubuntu, make sure you know in which partition you want to install it. Note that the selected partition will be erased!

7) After the installation, when you start you PC you just need to select which OS you want to use, Ubuntu or other (like Windows)! Ubuntu will be the default.

I didn't see this mentioned and thought it might be an option if getting a larger CD proves too difficult: download and install [VMWare Server]("http://www.vmware.com/products/server/"). It's free, it's fun, and you can "install" Ubuntu without making drastic changes to your system. It's available for Windows and Linux (once you make it to this side of the fence ;) )
Once you have VMWare Server installed, create a new virtual machine and point your 'CD Drive' to the .ISO and start the machine. It will go through either the Live CD or the installation (depending on which .ISO you downloaded) and you'll be set.
Mind you, you will need a bit more RAM and processing power than if you just installed the Operating System, but it's another idea to get you started.
Good luck and check back with us if you need anything else.
