### Post by monchichi on 2005-12-14
I prefer Gnome. It's simple, sleek, elegant, and organized.

The problem with KDE isn't that there's too much eyecandy, it's that the desktop and interface is so cluttered. There's just too much crap on the desktop by default-- I'm not nearly as productive in KDE because there are so many distractions. KDE does have some great things about it-- it is extremely configurable, it has built-in compositing (hardware accelerated transparency/shadows/etc), and some great programs like k3b and amarok. Personally I don't like Amarok because it is too cluttered, but I can respect it's full-featuredness.

It's just a matter of preference. If you want all the bells and whistles, go with KDE, but if you want a more productive, simple, elegant desktop go with Gnome.

E16, E17, xfce are all great too. I use e17 when I'm trying to impress somebody.. ;)

### Post by endersshadow on 2005-12-14
[QUOTE=thezerogroup]Linus Torvalds recommends KDE

[http://mail.gnome.org/archives/usability/2005-December/msg00022.html]("http://mail.gnome.org/archives/usability/2005-December/msg00022.html")[/QUOTE]

Linus's rant has been beat to death by people.  Let it go.

Just try both, and see what you like best.  If you have Ubuntu installed, you've got Gnome.  To try out KDE, run this command:

```
sudo apt-get install kubuntu-desktop
```

Then, log out, and click "Session" in the GDM and select "KDE" and then log in as normal and play around.  Also available as an option is XFCE, or xubuntu, which you can get by:

```
sudo apt-get install xubuntu-desktop
```

I personally use Gnome with xfwm4 as my window manager (instead of Metacity).  But that's my preference, and that's what suits my needs best.  And that's that.
