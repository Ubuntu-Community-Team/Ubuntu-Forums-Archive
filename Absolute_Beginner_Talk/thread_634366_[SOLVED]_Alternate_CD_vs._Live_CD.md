### Post by Duck2006 on 2007-12-07
The alternative install does have less installed than the full install. It is for people that have problems installing from the full install. After you have loaded the install from the alternative CD go to synaptic package manager and load the other packs you need.

### Post by xpod on 2007-12-07
> It's not really a very big deal, but I would like to have that System Tools menu.

Try going into your Menu Editor via the Preferences menus(or by right clicking on your menu) and enabling some of the "system tools".The menu entry for system tools never shows up in any fresh install if i remember correctly and will only appear once a "system tool" is enabled,or installed.
