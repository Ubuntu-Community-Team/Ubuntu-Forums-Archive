### Post by LaRoza on 2007-04-05

### Post by mezcla on 2007-04-05
There is a dedicated forum for 64-bit Ubuntu here that will probably give you a good idea of what you're getting into with 64-bit, as well as info about processing power.  The performance really all depends on what you're doing on the computer (video editing vs. browsing the internet).  I have used 64-bit from the beginning (Dapper for a couple months and now Feisty) and have enjoyed every bit of it.  You will have to get used to some "workarounds" (i.e. installing a 32-bit browser or using  nspluginwrapper so that you can use Flash on the internet) but that is the exception and not the rule.
