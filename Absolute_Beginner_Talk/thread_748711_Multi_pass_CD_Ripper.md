### Post by warbread on 2008-04-08
Yes.  I've gotten perfect .flac encoded rips with [RubyRipper]("http://code.google.com/p/rubyripper/downloads/list").  The dependencies are listed in the readme file, and you'll want to delete the .cddb folder in your home directory or RR will have trouble.
