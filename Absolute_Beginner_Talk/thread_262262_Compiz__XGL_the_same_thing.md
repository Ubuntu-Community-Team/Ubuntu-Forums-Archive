### Post by philipgm on 2006-09-21
No they are not the same thing and there are automated scripts that will install them. The Automatix Bleeding edge install script would be my suggested starting point. This will install everything you need and get you running. 

Before you do that though, check that your graphics card is supported. Your card doesn't have to be the latest thing but it can't be too old. Check out xgl/compiz on the gentoo forums for mmore info on support.

HTH
