### Post by Inxsible on 2007-06-07
You should make sure your video graphics is from NVidia. ATI cards pose a lot of problems as their linux drivers are below par.

Once you get your laptop, you should run the LiveCD on it for a few weeks and check whether all your hardware is compatible with Linux  / Ubuntu and then go from there.


Good luck !

### Post by Rocket2DMn on 2007-06-07
FYI: Ubuntu "Feisty Fawn" is the most recent version of the Ubuntu operating system (a distribution of Linux), somewhat similar to how windows released 95, 98, ME, 2000, XP.  However, Ubuntu releases updates MUCH more frequently

If durability is an issue, you should get a laptop with a warranty for accidental damage.  I really like these since laptops just get banged up.  
For spreadsheets there is OpenOffice Spreadsheet which has a program similar to Excel.  Maybe you could "test drive" before you make your final decision.
Happy hunting!
