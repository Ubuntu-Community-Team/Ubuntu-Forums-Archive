### Post by endersshadow on 2006-12-07
I believe that this game can only be played on Windows.  You'll need to use Windows to run it.

If you don't care why, you can stop reading.  Else, I'm going to throw this information up, anyway:

Game developers pretty much only develop for Windows for obvious reasons of popularity.  One may think that it's not really all that different and shouldn't be only more than a recompile to make it available on Linux, but then one would be wrong.  The problem is that games for Windows *generally* make use of DirectX, Microsoft's proprietary 3D rendering engine.  Linux uses OpenGL for that task, and it is also available for Windows.  However, DirectX tends to be faster than OpenGL on Windows...at least that's what my friend who's into making 3D stuff tells me.  There are quite a few games out for Linux, and I implore you to check out the [native games available](http://doc.gwos.org/index.php/Native_Games).  Do know, however, that unless you'd like to go through the hassle of Wine and/or Cedega (more info [here](http://doc.gwos.org/index.php/Non_Native_Game)), it's just easier to dual boot with Windows to play games.  Disclaimer about Wine and/or Cedega: Both are hit and miss when it comes to games...beware.
