### Post by gn2 on 2007-12-16
Xubuntu uses a different desktop environment, XFCE.

It is lighter on system resources and better suited to older hardware.

You can use it by simply adding the package "xubuntu-desktop" in Synaptic Package Manager, then at the log-in screen, click Sessions and choose XFCE.

You can also install Xubuntu by opening a terminal and using this command:

> sudo apt-get install xubuntu-desktop
