### Post by skwishybug on 2007-08-07
You would need to install the proper codec for DVDs, they don't come preinstalled for copyright issues.
 
If you go to Add/Remove programs in your menu and do a search for restricted, it should give you the option to install the restricted libraries which will include libdvdcss which is needed for DVDs.
 
Or look over: [https://help.ubuntu.com/7.04/musicvideophotos/C/video.html#video-dvd](https://help.ubuntu.com/7.04/musicvideophotos/C/video.html#video-dvd)
