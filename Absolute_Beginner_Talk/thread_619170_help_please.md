### Post by daimaru on 2007-11-21
install your restricted drivers (graphics card) 
install  compizconfig-settings-manager

```
sudo apt-get install compizconfig-settings-manager
```
change your workspaces to 4 instead of 2

go to system->pref-> advanced desktop settings 
check "desktop cube" and "rotate cube"

use ctrl+alt+mouse1 or middle mouse on dektop click or ctrl+alt+cursor keys left right to  rotate
