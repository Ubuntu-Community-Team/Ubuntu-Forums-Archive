### Post by barrack on 2006-07-13
I'm not sure if you're looking to have an email account that you can access from anywhere. If so, then gmail is good, but yahoo and others also allow POP mail to be forwarded to their email sites.

If you're looking for a program that will gather the messages from multiple email addresses into an application that is only on your computer, then I would suggest Mozilla Thunderbird. There are others out there. But you can log in using only one password and it'll remember all the other passwords and you can manage the emails from all accounts through one program.

It's similar to Outlook Express in Windows. Ubuntu also comes with a mail/scheduling program called "Evolution." This is comparable to MS Office's Outlook (does emails and calendars and stuff.) Though I haven't used Evolution yet, I think it has the capability, like Thunderbird, to manage multiple email accounts.

Keep in mind, though, that some webmail programs don't allow that their emails be read anywhere other than their website. Yahoo is an example of this. You have to sign up for a premier account before you're allowed to access Yahoo mail from gmail, thunderbird or evolution.

### Post by HereInOz on 2006-07-15
If you install Mozilla Thunderbird, which is available in Synaptic, and then install the Webmail extension - [http://webmail.mozdev.org/](http://webmail.mozdev.org/) - I think this might do what you want.  Not sure from your post as to what you are really looking for, but this may be the one.

When you set up the accounts in Thunderbird, make sure that when you are presented with the option in the set up process to use the Global Inbox, uncheck that, so that each email mailbox is kept separate from the others.

Hope this helps.
