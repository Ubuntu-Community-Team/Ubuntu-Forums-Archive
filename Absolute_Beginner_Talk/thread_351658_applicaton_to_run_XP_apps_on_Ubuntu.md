### Post by h2gofast on 2007-02-02
1)First check out linux applications for tasks you did in windows.  The linux application often, but not always, does it better, and more simply with less bloat.  Exceptions I've noted are "Streets and Trips," an interface with Microsoft Exchange, and certain applications for professionals like Photoshop, or CAD/CAM work.  Oh, and certain video games.  Wine is available but IMHO (in my humble opinion) more work than it's worth. So unless you're a graphic designer you should be fine. 

2)Fo Sho Mo User Friendly than it used to be.    Windows installation works with most all hardware while linux installation works with about 85% of hardware, because most vendors don't play as nicely with linux developers as they do with windows.  The 15% is stuff you don't find much anyway, but ubuntu is shrinking that margin because that margin is 70% of the headaches for linux users.  
      Once Installed, I think ubuntu is more user friendly than windows, but it is different so you will need to adjust.  If you are non-techie, I would pass on xubuntu (unless you're running hardware older than a pentium2) or kubuntu for now and learn to use gnome.  Kde vs gnome is strictly personal preference, but because gnome is the focus of ubuntu's efforts start with gnome.  
Also the ubuntu forums are the most user friendly linux forums.  

3)Like aktiwers said, this is Linux and the workhorse underneath the hood is the linux kernel and it's the same kernel that runs most servers on the internet (give or take a version number).  In other words a lot of things that Windows users need to optimize is stuff that should have been done right the first time.  The kernel and file system is simply more robust.  

Having said all that, linux is still an computer operating system, and there will be moments when you have to configure things properly before they will perform the way you want them to.  The nice thing about Ubuntu is that they have taken a lot of that configuration off the commandline and given it an interface new people can use.  

Cheers, 
h2.
