### Post by Mornedhel on 2007-07-27
Apparently it's a Debian liveCD (so basically, it's - by inheritance - a "really big distro"). What you'll miss is probably what you'd miss if you switched to vanilla Debian : it's a bit less noob-friendly (Debian users make fun of us for being so popular, you know - there's a recurring joke that Ubuntu in fact means "I can't configure Debian").

It also doesn't seem to be very much maintained as a separate project : the last news on their website dates back to December 2006.

Oh, forget about all that. It's KDE-friendly, which obviously means it's the spawn of the Devil. (just kidding, for the irony-impaired)

If you feel really comfortable with Ubuntu now, you can switch to Debian (which *does* have a *huge* community by the way).

### Post by Mornedhel on 2007-07-27
You need to be more curious. If you had read the FAQ on Kanotix's site (I had never heard of Kanotix before, so it's kinda sad I'm doing your research for you), you would have noticed it's based on apt-get (no mention of any GUI, but I'm sure there's one). So it uses the exact same mechanism as Ubuntu, which is, by the way, another flavour of Debian, but one which Debian fans do not approve of since we're not entirely open-source (there are not entirely free packages in the Multiverse section, and the Restricted drivers are proprietary stuff).

If you are already comfortable (as in, you can and do configure stuff with the terminal only and know your bash syntax by heart) with Ubuntu, you should be fine with Debian.

### Post by psterrett on 2007-08-05
I used to use Kanotix long before I ever heard of Ubuntu. It was even better than Knoppix, on which it is based, at detecting my hardware. Until I installed Dapper Drake, this was the easiest Gnu/Linux distro to get up and running known to me. Also, the actual developer, Kano, answers questions on their message board. This helped me tremendously. I could be mistaken, but I believe that this was the first distro that would install a LiveCD on the hard drive... in any event, Kano wrote the install scripts for Knoppix as well.

If you've read this far, you're probably wondering why I switched to Ubuntu.
