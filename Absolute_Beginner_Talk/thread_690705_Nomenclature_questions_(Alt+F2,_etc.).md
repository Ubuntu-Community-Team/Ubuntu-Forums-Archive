### Post by jordanmthomas on 2008-02-07
I think the "official" name for Alt + F2 is the run-dialog.
I think this because you can run it (assuming your panels are running) with
```
gnome-panel-control --run-dialog
```

Usually, I put commands in ```
 brackets
[code]gksudo whatever
```

I suppose it's personal style, some people like to bold them.

Hope this helps.
