### Post by Bachstelze on 2007-06-26
The modem might or might not work. It's kind of a hit and miss and there's not really a way to tell before you try. If the modem works, ICS is definitely possible.

USB cameras are basically USB drives so no problems with that, the SD card reader should work too.

Networking... Depends what you want to to with it ;) Setting up file sharing and the like is not difficult at all.

Games I know close to nothing about, sorry.

Nvidia drivers are very easy to install.

AIM alternatives : LOTS ! Kopete is my favourite since I'm a KDE fan but there are others.

Quake or CS:S server, definitely possible but I don't know how to do that.

The older ATi card should be supported by the Open SOurce drivers so light 3D apps shouldn't be a problem. The PCMCIA wireless won't be a problem either, use ndiswrapper if there are no Linux drivers.

If your objective is to play the latest proprietary  games,  Linux is not for you.
