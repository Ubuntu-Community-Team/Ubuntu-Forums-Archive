### Post by Mimsy on 2006-10-18
If you can get sound, good screen resolution, and connect to the internet and all the other things you want to do, when running from the live CD, then you can do it from an install as well.

If there's just one or two things you *can't* do from the live CD, post here, it may be possible to fix it with an update, command line tweak, or something similar. I.e. my laptop has a built in wlan, W200, that refuses to work without several minutes spent copy-pasting stuff from a how-to into a terminal command line. So I have the how-to saved as a file on the laptop, so it is always available. A situation that would have been an absolute deal breaker is now nothing more than an occasional nuisnce, thanks to the Ubuntu community.

Bottom line: ask for help. If there is a solution, someone here knows it. Just be willing to give it a few days.

/Mimsy
