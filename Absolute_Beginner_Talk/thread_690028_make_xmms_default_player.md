### Post by talsemgeest on 2008-02-07
[LIST=1]
[*]Right-click on a file of the type you wish to set a default application to open it with
[*]Select **Properties**
[*]Select the **Open With** tab
[*]Select the application you want associated with that file-type
[*]Click **OK**[/LIST]I took this from [http://linuxfud.wordpress.com/2006/09/03/ubuntu-linux-file-associations/](http://linuxfud.wordpress.com/2006/09/03/ubuntu-linux-file-associations/)

Hope this is what you needed :)

### Post by daengbo on 2008-02-07
The above is a true and general method for changing the default program and will work for any file type, but there is an easy way to associate all media files with XMMS: go to System -> Preferences -> Preferred applications and choose the "Multimedia" tab. There will be a pull-down to choose your media player.

Best of luck.

### Post by tgalati4 on 2008-02-07
You can also change Firefox's preferences to use XMMS for type mp3.  Edit-->Preferences-->File Types-->Manage-->Search MP3, change the MPGA filetype to XMMS.  I think it uses mplayer by default, and you don't get any controls with audio media.
