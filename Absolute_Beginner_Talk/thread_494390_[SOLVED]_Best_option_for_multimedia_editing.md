### Post by por100pre1 on 2007-07-06
I use Ubuntu Studio. ubuntustudio-desktop actually changes the human theme and wallpaper for a dark theme and gray wallpaper. It will continue to use gnome, nautilus, beryl, etc. The only real big change here is that it add a low latency kernel, that is an under the hood change that you get to select everytime you boot. Your GNOME configuration won't be broken.
