### Post by mostwanted on 2006-05-24
Reinstalling Windows will overwrite Linux. I feel sorry for you that you erased everything, that's the default option when installing Ubuntu and to someone who is not experienced with installing operating systems. When you reinstall Windows, if you want to try Ubuntu as well, you can leave some space unformatted. Then when you run the Ubuntu installer, you'll have the option to install to the free space - that way you can run both OSes on one computer.
