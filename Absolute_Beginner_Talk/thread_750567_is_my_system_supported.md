### Post by R6V2 on 2008-04-09
Just use some cd burner, and burn the .iso file to the cd, boot from the cd, and choose: The Check disks for errors or something, and then if everything's good, then boot from the cd again, and choose: Install or run Ubuntu or something. And then it will load the kernal, and load the desktop (running from cd) then, check it out and everything (Cant save anything while running from cd) Then, if your ready for the change, double click the 'Install' icon on your desktop! (This will install Ubuntu) Also, ubuntu is SLOWER on the disk. So be aware that Ubuntu is much faster than your first impression :)

You can do basically anything on the CD except for saving anything. The CD IS the OS basically :) So, enjoy Ubuntu like I do! PS: Get WINE if you need to run any Windows-based Apps :)
