### Post by cogadh on 2007-06-26
Don't know about the YouTube thing, but Wine I can answer.

Wine is not specifically for playing games, it as an open source implementation of the Windows API that can allow you to run any type of Windows software on Linux. It does allow you to play many Windows games, some will work just as well or even better than on Windows. Before you try playing games with Wine, you should look at the AppDB website for games that work: [http://appdb.winehq.org/](http://appdb.winehq.org/)

As for alternatives to Wine, there are really only two: 
Cedega - a branch off of the Wine tree, it is a subscription software ($5 US a month, three month minimum). It is basically closed source and barely contributes back into the Wine source tree it is derived from. However, through a series of hacks, it is able to make some games function well and it does include a nice GUI interface. Unlike Wine, it also has support for copy protection software that is on most games (Wine is in the process of adding that now)

CrossOver - Wine's major financial backer. It is also a branch of the Wine tree, but it is mainly designed for running office applications and only recently added some support for games. It is based on older, stable versions of Wine, so many newer and all of the newest games don't work with it. It costs $39.95.
