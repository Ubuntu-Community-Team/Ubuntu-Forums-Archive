### Post by 3rdalbum on 2008-04-01
Python is easy and flexible. A reasonable amount of Ubuntu is written in Python! And some dudes wrote an operating system in Python once.

C or C++ are also popular choices, but they take longer to learn and program in than Python. Execution speed and flexibility are much better than Python.

Fortran and BASIC are not a lot of use these days.

Assembly is incredibly difficult.

I think you mean "Ruby", not "Rails". Ruby is the language, Rails is a framework for Ruby that makes it easier to create good web applications.

I'd suggest that you start with either Python or C#.
