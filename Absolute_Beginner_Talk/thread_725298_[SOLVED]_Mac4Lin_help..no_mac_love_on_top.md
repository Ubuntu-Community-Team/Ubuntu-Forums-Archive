### Post by benerivo on 2008-03-15
joshrobinson is right. when you go to System>Preferences>Appearance it opens a window which can install gtk themes and icons sets by way of 'drag and drop'. This only takes .tar.gz files, so don't extract the icon set you have [downloaded]("http://kiddo.forthfreak.net/OSX3.3.tar.gz"). Instead, just drag the OSX3.3.tar.gz file in to this window and it will add it to your themes.
