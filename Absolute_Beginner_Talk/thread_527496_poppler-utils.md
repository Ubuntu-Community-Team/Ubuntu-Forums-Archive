### Post by Elegorod on 2007-08-17
There are commandline utilities, so they have no GUI.
To see program names, select package in Synaptic, right-click, select Properties, tab Installed files, and find files, that starts from "/usr/bin/".
For poppler-utils:
> 
/usr/bin/pdffonts
/usr/bin/pdfimages
/usr/bin/pdfinfo
/usr/bin/pdftohtml
/usr/bin/pdftoppm
/usr/bin/pdftops
/usr/bin/pdftotext
