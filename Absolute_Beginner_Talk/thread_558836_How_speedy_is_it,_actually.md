RAM is much faster than Hard drive access, and Optical drive access.

Your understanding is incomplete: yes, the OS runs in RAM, but it does it with HD installations and Live CDs. Hard drive access is faster than cd access, so it will load into RAM faster, thus, it will run faster.

On the live cd, don't use it for a couple of seconds, then try to open something. You'll notice the cd is accessed, it is loading into RAM.

If the entire system is in RAM, as it is possible to do. It will be fast. Most live cds don't load the entire disk into RAM.

It will be faster installed.
