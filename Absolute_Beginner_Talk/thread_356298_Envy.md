### Post by renzokuken on 2007-02-08
ok, weird, you seem to be using two different drivers at the same time. try this

backup your current xorg.conf

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup
```

open it in a text editor

```
sudo gedit /etc/X11/xorg.conf
```

delete the whole of the first Section "Device" part , i.e. 

```
Section "Device"
	Identifier  "Radeon X550"
	Driver      "ati"
	Option	    "XAANoOffscreenPixmaps"
	Option      "PCIMode"       "8"
	Option      "AccelMethod"   "EXA"
	Option      "ColorTiling"   "on"
	BusID       "PCI:1:0:0"
EndSection
```
