### Post by kerry_s on 2007-09-01
epiphany uses alot of cache for speed. from my testing it seemed to pull the disk cache saved to memory for faster access. you can lower the disk cache so it doesn't pull that much to memory. i think the default was 50mb, i recommend 5mb, that's what i use on all mozilla browsers.
