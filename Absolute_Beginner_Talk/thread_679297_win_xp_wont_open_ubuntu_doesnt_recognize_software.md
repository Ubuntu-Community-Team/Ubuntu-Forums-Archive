### Post by jamoo1980 on 2008-01-26
9 times out of 10 you can just download Ubuntu Desktop edition, burn it to CD, then boot your WinXP machine from the CD-ROM drive, install Ubuntu using all the default options and it will pretty much sort itself out with a pre-boot menu (Grub) to choose Ubuntu or WinXP... The only bit to look out for is when the installer asks how you'd like to partition the disk. I can't remember exactly what the options are, but you want the one that automatically resizes and installs Linux. You definitely DON'T want to 'use the entire disk' as this will erase WinXP! Hope this helps :)
