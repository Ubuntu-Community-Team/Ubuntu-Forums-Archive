### Post by bodhi.zazen on 2007-10-24
One PDF viewer - [evance](http://www.gnome.org/projects/evince/)

```
sudo apt-get install evince
```

Open office will open Microsoft documents, and I expect greater compatibility now the M$ has lost in Europe ...

and Linux is built on C ...

Install [build-essential ](http://packages.ubuntu.com/cgi-bin/search_packages.pl?keywords=build-essential&searchon=name&subword=1&version=all&release=all)

```
sudo apt-get install build-essential
```
