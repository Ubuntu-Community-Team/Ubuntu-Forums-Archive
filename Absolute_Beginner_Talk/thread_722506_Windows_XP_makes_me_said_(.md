### Post by Calash on 2008-03-12
This probably should be over in the Other OS/Windows forum to get the best response.

In that same screen, if you click on Hardware, then the Device Manager button you should get a list of all the devices.  Do any of them have yellow or red around them with a "!" or "x"?  They will be expanded by default so you do not have to search far.

If not, then the driver is in and it is a matter of getting the wireless setup correctly.

### Post by Golem XIV on 2008-03-12
Heh, this should be moved to the Windows part of the forum :)

I don't have my XP desktop at hand, so I can't give you more detailed instructions, but check either the connection icon on the taskbar to access the configuration or (if there is no wireless icon) go to Control Panel -> Networks, right-click on the wireless connection you want to configure and select "Properties".  You should be able to take it from there.

Note:  If there is no wireless icon AND no wireless on Control Panel -> Network, your driver is probably not correctly installed/corrupted/not working.  Try reinstalling it.

Good luck!
