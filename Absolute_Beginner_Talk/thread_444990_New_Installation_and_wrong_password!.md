### Post by heimo on 2007-05-15
On the grub / boot menu, choose recovery mode. If you can't see this menu, hit esc just at the beginning of the boot (there will be message for a short time). Then use passwd to change password.

```
passwd *username*
```where *username *naturally is the actual username
