### Post by skullmunky on 2008-02-01
if you want other WM's just install them from synaptic ... blackbox, fluxbox, enlightenment, fvwm, mwm, xfce, windowmaker, ratpoison, etc, etc, ... 

of course some of those are not exactly noob-proof :)

if what you mean is that you want to write your OWN window manager, yes, you can certainly do that too.  for example, say you want to make a Legend of Zelda windowmanager, where all of your files are either hidden in shrubs that you cut down, or can only be accessed by defeating an octorok with your boomerang.  definitely possible.  though probably a lot of work.  

or if you mean you want to heavily customize an existing one, like add a new feature to Gnome where all the windows on the screen like to clump together like a swarm of bees, also totally possible.  that's what's so great about open source.

### Post by khensucat on 2008-02-14
Why don't you try the gnome art manager? You can install it via Synaptic, and it will show you previews of wallpapers, themes, etc... If you decide you like something, simply click on "install", and it will automatically download and install it for you. When you're done installing everything that looks interesting, you can go into "Appearances" from the system menu and try all of them out until you have something you like. 

It's a pretty easy way to try different looks :)
