### Post by TheMono on 2007-02-22
Just to elaborate on the difference between Ubuntu and Ubuntu alternate discs, the standard one is a bootable live CD which takes you to the Ubuntu desktop direct from the CD without installing. This uses a lot of RAM though, which is why the alternate CD does exactly the same install, but without booting to a desktop first - for computers with less than 192MB RAM.

Having said that, there are other uses for alternate as well, but that's another story.

But still, I agree that Ubuntu server install would be fine - though the strength of Ubuntu is really in the desktop side, you might be better looking into fedora or gentoo maybe for server?
