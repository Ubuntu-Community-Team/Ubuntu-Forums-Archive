### Post by seshomaru samma on 2006-11-07
Welcome1
Linux is a great OS
It can do most things that XP can do plus many other things that XP can't.
For web browsing there is Firefox and other options
For Office there is Open Office and other options
For picture editing there is The Gimp (similar to Photoshop) and other options...
etc... etc...
There are many many options in Linux. You can control almost every aspect of your OS.

One thing that it doesn't do well is running Windows programmes. For most Windows programmes there is a Linux equivilent ,but for games there often isn't . So be warned , if you like playing games , you might be disappointed with it. 
Yes it can play mp3 ,but you will have to install mp3 codecs that don't come by default with Ubuntu due to legal issues. Installing codecs and other programmes is very easy (much easier than in Windows) with an application called Automatix.
You can try the Ubuntu live CD first to get a feel (though the live CD is quite slow compared with an installed system)
Good luck

### Post by 3rdalbum on 2006-11-07
Yes, there's a great CD/DVD burning program for Linux called K3B. It isn't installed by default on Ubuntu, but it's very easily obtainable.

One thing Nero does which K3B can't do is transcode video to put onto DVD. However, there are other programs out there which can transcode your video and create a disc image that K3B can burn, so it's not a problem.
