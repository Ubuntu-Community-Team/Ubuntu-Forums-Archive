### Post by Inxsible on 2007-11-21
you probably do not need java 1.4, 1.5 and 1.6 altogether. Simply installing 1.6 will be enough.

Java versions are always backward compatible, and so whatever runs in 1.4 will also run in 1.6 (unless mentioned on sun's website - which is very rare)

Go into synaptic and search for "sun java" without quotes. Remove 1.4 and 1.5 versions
