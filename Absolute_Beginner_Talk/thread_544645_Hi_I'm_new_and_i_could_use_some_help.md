### Post by annda on 2007-09-06
hi,

it's not a very good idea if you plan to install/try out a lot of new software. linux programs are very modular, so usually if you want to install something new, you will need that file and 0-20 other files that are not included in your program. if you used the linux software management, it would automatically search for those files (called dependencies, since the programs depend on them), download and install them for you.

if you are offline, you might have problems installing and need to check the dependencies on
[http://packages.ubuntu.com](http://packages.ubuntu.com)
download the missing ones, and install them manually. it is a bit of a hassle but if you are not into experimenting with software, you will be fine.
