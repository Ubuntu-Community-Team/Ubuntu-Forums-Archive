### Post by CybaCowboy on 2006-05-10
Once you've downloaded those themes to a directory (say, the Home folder), how does one actually *install* them?

Never mind, I worked it out...


For others wanting to install a theme, save the file somewhere and then install it using "Theme Greeter" (System-->Administration-->Login Screen Setup-->Theme Greeter) - all you need to do is click "Install new theme", find where you saved the theme and double-click it!
