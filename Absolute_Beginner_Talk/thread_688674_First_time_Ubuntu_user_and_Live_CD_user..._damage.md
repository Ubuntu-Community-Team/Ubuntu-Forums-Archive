### Post by pytheas22 on 2008-02-05
The software to play mp3's (meaning the stuff that needs to run in the background, not Rhythmbox) is not free, so for legal and philosophical reasons it's not included in Ubuntu by default.  However it's very easy to install; check out [https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu) for instance or search the forums.

You should be able to run your CD for quite a long time without problems.  Just remember that every time you reboot, you will lose all your settings, including any software you installed during your last session.  If you can leave your computer on indefinitely, this shouldn't matter.   Nonetheless, you might also want to consider doing a hard disk install as soon as you're comfortable because the system will be a lot faster and smoother than it is running off of a CD.
