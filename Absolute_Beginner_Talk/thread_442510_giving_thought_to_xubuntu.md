### Post by benfindlay on 2007-05-13
the 3 different major flavours of ubuntu all do pretty much the same thing, they just have different applications for various functions. For example, ubuntu uses gedit, kubuntu uses kate and xubuntu uses mousepad (for text editing). You can install programs that you like from the other flavours also!

And theres no need to just jump off a bridge to make the switch! You can type ```
sudo aptitude install xubuntu-desktop
``` in a command line and it will install the xubuntu interface alongside gnome. Then, at the main bootscreen (where you log in) you can click on "Session" and choose xfce instead of gnome to try xubuntu out!

Hope this helps!
