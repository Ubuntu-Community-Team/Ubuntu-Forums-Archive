### Post by nudnik on 2006-09-17
RealPlayer is a standalone program which upon first activation usually configures itself with the default browser so that it launches correctly when needed. You may have noticed the message stating that it was configuring its Mozilla plugins on first usage.

I use only Gnome and what little experience I have outside of that is with KDE, so I am completely unfamiliar with your desktop. My recommendation would be to remove Swiftfox and stick with the default browser. I saw no improvement in speed when using Swiftfox on my machine, a Pentium D 820 rig with an Intel 945PSN motherboard and SATA II drive.

If you want faster performance from Firefox, I suggest modifying its settings for maximum efficiency using the "about:config" dialogue as described in this forum and Firefox's own site. Find a direct link below to an alternate site.
