Desktop Environments is what forms the way in which your applications look, and also draw the background image. For example, ubuntu comes with Gnome standard but can also get other desktop environments such as KDE and XFCE. On the ubuntu home page (ubuntu.com) you can download Ubuntu, Kubuntu, Edubuntu, Xubuntu. If you want the homepages of the Desktop Environments, try googling the various names of the projects, like XFCE, KDE, GNOME, Fluxbox.... also in Synaptic Package Manager (Start Menu >> system >> administration >> Synaptic Package Manager) try using the search tool for desktop environments, there are a lot of projects there for download that you can select to use at login time.

### Post by BobCFC on 2007-06-24
Gnome tries to be friendly.. it usually has two bars running across the top and the bottom.   I like it because the menu is split into Applications, Places and System.. so I can jump to something with one less click.

KDE usually has a single thick bar and one monolithic start menu like windows.. with everything coming from the single start button in the corner.   Fans of KDE like it because it has lots of advanced options that let you customise everything for the 'Power User'.  It also has a more consistent look with icons etc..  Some people think it is slow and more bloated.

XFCE is for people who want a fast desktop.  There is less wow but it runs really well on old hardware.  Has a single row of icons at bottom like a mac

Fluxbox is for people who want purespeed and do lots of work typing at the command line.  It runs like lightning even on a ten year old PC.  All you get is a right click menu.. not even desktop icons

Enlightenment is a pretty graphics idea but it doesn't work with Beryl/Compiz so no 3D effects.

IceWM... oh dear back to 1996 no please no. :shock:
