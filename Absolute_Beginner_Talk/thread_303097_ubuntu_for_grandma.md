### Post by IYY on 2006-11-19
- Make sure to have only the software she'll be using in the menus, or maybe have no menus at all and just have shortcuts to the programs she'd be using on the panel. Another option is to remove the top panel altogether and just stick the shortcuts on the desktop.

- Make her a regular user and yourself an admin.

- Install an ssh server, so that you can connect to her machine remotely and fix/install things if needed.

- The simpler the programs are, the better. So, AbiWord and GNUmeric might be a better choice than Open Office! 

- Maybe install a few small games, like Frozen Bubble.
