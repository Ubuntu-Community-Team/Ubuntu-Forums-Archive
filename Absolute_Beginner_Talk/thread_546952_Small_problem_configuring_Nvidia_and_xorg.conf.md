Hi,

I'm having a small problem configuring my screen resolution.

I successfully installed the Envy/Nvidia tool, which installed the Nvidia driver, the tool also changed the xorg.conf.

Problem; when I go to System > Preferences > Screen resolution, the highest screen res. is 1024 x 768.

If I use the Envy/Nvidia configuration tool, I am able to put this to for example 1280 x 1020. But... the problem arises when I let the Envy tool also save this change to the xorg.conf file.

Next time I boot, I only get the command line tool ?

Could it be that I need to edit the xorg.conf and put in the extra screen resolutions?

Thanks, Robin

### Post by ghantoos on 2007-09-09
in the screen section, try to change:
Modes      "1024x768" "800x600" "640x480"

to :
Modes      "1280x1020" "1024x768" "800x600" "640x480"

save xorg.conf, and restart X (ctrl-alt-backspace)

Cheers,

ghantoos

Hi I would say yes and you can use the command in the terminal
```
gksudo gedit /etc/X11/xorg.conf
```
and add the resolution that you want to **each** line But be sure to Back up you xorg first. 
EndSubSection
    SubSection     "Display"
        Depth       24
        Modes      "[COLOR="Red"]1280x1020"[/COLOR] "1024x768" "800x600" "640x480"
    EndSubSection
EndSection
