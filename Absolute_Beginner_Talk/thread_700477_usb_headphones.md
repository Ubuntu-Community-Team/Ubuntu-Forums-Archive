### Post by eggdeng on 2008-02-19
Try running
```
asoundconf list
```
to see how alsa identifies the available sound cards. Then run
```
asoundconf set-default-card xyz
```
where xyz is the name the card you want to set according to the output of the 1st command.
Reboot for changes to take effect.
