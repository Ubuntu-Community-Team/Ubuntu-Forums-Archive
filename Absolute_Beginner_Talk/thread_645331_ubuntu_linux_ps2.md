### Post by adzik on 2008-02-05
No, you can't dump Ubuntu on a Playstation 2, with or without the official linux kit. 
Since I've seen these questions hovering around here and other forums, I'll go into a bit more detail.
The PS2 is a different breed of hardware with next to no RAM by today's standards.
There is only one other flavor of linux available for the PS2, and that is BlackRhino based on debian, and even that still requires the Linux kit.
You need to own a chubby (old) PS2, not the newer slim one, as it absolutely requires the network adapter and an IDE hard drive. Any standard USB keyboard / mouse should work. You also need a dedicated memory card for it.
You might be able to boot up a backup of the Linux kit DVD with an Action Replay, but I don't know about that as I haven't tried.
And at the end of the day, your linux experience would be primitive at best. 
This is more a developer / hobbyist item more than anything. Don't waste your time unless this is just a fun project to pursue.
Get a PS3 instead and load the sucker with whatever linux release suits you.
