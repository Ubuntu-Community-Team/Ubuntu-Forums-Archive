### Post by nickdngr on 2007-08-15
i use a usb keyboard (at work right now so i don't know the type) and it worked right out of the box. the extra functions (volume up, down, mute) worked too. I just got another keyboard in (belkin mediapilot) today that i'm going to try when i get off, so hopefully that will be as simple.

as far as external hdds go, i dual boot xp and feisty, and the hardest part of the installation was partitioning the hard drive. I used partition magic before installing and it took forever. Once you get it installed, if you are running a separate boot partition on the external instead of the internal drive, you'll have to hunt around for grub so you can set up a boot options menu (it's really easy and there are plenty of guides/how-tos here that talk you right through it) and set up your bios for the external hd to boot before the native hd.
