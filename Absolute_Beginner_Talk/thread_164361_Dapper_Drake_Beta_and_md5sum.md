### Post by Qrk on 2006-04-22
Try using bittorrent or a download manager that supports resuming a download. 

If you download something so large via firefox or most web browsers, it is likely that it will be interrupted. Bittorrent is able to recover if you loose a connection, as is (I believe) opera.

### Post by jazzmuzik on 2006-04-22
You could try downloading via bittorrent since it does some crc checking. Perhaps you could put your existing ubuntu iso file in the bittorrent download directory, then start bittorrent, bittorrent will recognize the file as being partially downloaded and will "download" whatever is missing.

Bad md5sums are not a good thing.
