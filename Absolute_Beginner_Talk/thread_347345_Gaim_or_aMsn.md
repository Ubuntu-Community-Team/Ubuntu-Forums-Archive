### Post by AirRaven on 2007-01-27
If it's MSN Compatibility that you want, I suggest that you get "Kopete" from the repos. There's nothing that even begins to touch it. aMSN is about as counter-intuitive as it's possible to be, and Gaim, while lovely in itself, doesn't have any Webcam support whatsoever.  

It's the KDE default Messenger program, so it won't work seamlessly in GNOME without a bit of twiddling- it has an annoying tendancy to actually download the contents of links first, and then open them in Konqueror. Easily sorted, though. 

You'll need to run this in the terminal:

```
sudo aptitude install kopete kcontrol kde-systemsettings qca-tls
```
Alternatively, just grab "kopete", "kcontrol", "qca-tls" and "kde-systemsettings" via Synaptic. 

After the download's done, run the program "kcontrol" from the terminal. 

Go to KDE Components->Default Applications->Web Browser, and set it to look like this:

[img]http://img266.imageshack.us/img266/2226/screenshotdefaultapplicyw1.png[/img]

After that, just run Kopete and you're away. Take a look around at the skins and emoticons available- the "msn 7 transparent" emoticon set is what you'll be looking for if you're migrating straight from MSN Proper. Take a gander at the chat window styles available as well if you don't like any of the default looks. It even supports [Adium](http://adiumxtras.com/) chatwindow skins if you find any you like. 

Excellent Webcam support in a messenger that doesn't make your eyes bleed. Give it a try- you might even like it.
