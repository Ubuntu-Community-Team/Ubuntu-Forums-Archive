### Post by ugm6hr on 2008-03-02
1. Synaptic needs to be told where to look for those 3rd party apps.  You need the repository address, which is normally given by the software you are trying to install.
e.g. Wicd [http://wicd.sourceforge.net/download.php](http://wicd.sourceforge.net/download.php)

2. Use GParted (Partition Editor) to sort out partitions.  Not sure if you are looking to reformat, but that can be done in GParted too.
