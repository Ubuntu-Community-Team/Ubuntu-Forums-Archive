### Post by Endwin on 2006-08-25
Pretty much all you need to do is install the multimedia codecs [http://ubuntuguide.org/wiki/Dapper#How_to_install_Multimedia_Codecs]("http://ubuntuguide.org/wiki/Dapper#How_to_install_Multimedia_Codecs")

and then just point the play.pls file to a application that handles streaming media (Totem sometimes works) In my experience XMMS (The Linux equivalent of winamp) works the best (for me) for streaming audio. You can install that by searching for xmms synaptic or sudo aptitude install xmms.
