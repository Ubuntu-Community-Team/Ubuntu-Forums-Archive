General tips. 
If the modem has a ethernet connection use that instead of USB. It's much easier and work much better. 
USB in most cases require the load of some firmware and extra packages for configuration (pppoe/pppoa stuff)

Use the search function of the forum or google for it with Linux as search word.
You will probably need to get the exact model.
