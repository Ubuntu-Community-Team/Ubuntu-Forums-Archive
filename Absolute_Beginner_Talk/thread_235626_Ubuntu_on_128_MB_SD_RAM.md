
### Post by daou on 2006-08-13
You can try getting the alternate-install CD for Ubuntu and use that to install if you run out of memory during the Live CD session. But 128MB is probably not enough for Ubuntu to run well. I suggest you install Xubuntu or some other lightweight desktop for the Ubuntu core.
