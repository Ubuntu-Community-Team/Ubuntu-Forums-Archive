### Post by bender5788 on 2007-02-17
OK as a general rule you install from oldest to newest so by my memory it would be OS2>>Windows>>Ubuntu in that order you would need to partition it so you have at least 4 partitions  although if you can pull it off 7 is good, a system and data partition for all plus the necessary swap partition linux loves so much (although with 2 GB of RAM I've never seen it used) be warned though by default Ubuntu sets it self as the default (as do most systems) so you will need to be there at boot in order to make sure you select the preferred OS with any given luck you wont need to do to much stuffing around for OS2.
Oh and as for the partitioner if you install off the ubuntu live cd or for that matter boot to it i believe there is one on due to the need to be able to partition your drive.
