### Post by Jammy4041 on 2007-12-15
in a terminal, type 

[COLOR="Red"]sudo nautilus [/COLOR]

to get proper access to modify documents which can only be done as root.

For everything else, its in a terminal.

[COLOR="Red"]sudo (what ever you want to do)[/COLOR]

When you do this, it will ask you for a sudo password, but this is just the password you log in with.
