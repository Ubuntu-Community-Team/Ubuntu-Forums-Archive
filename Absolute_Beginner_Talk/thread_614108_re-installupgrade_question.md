### Post by Paul820 on 2007-11-15
tracker is used to search for any file on your computer, unfortunately when it is indexing your files it can slow down the system considerably. To check what is using the most memory go to System->Administration->System Monitor and click the processes tab. It might not be slowing your system down but it is worth checking just in case it is. When i had it running it was using 936Mb when indexing so i turned it off. My system slowed down to a crawl and typing was taking a long time so that's why i suggest checking it out.
