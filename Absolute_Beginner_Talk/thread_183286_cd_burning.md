### Post by mynimal on 2006-05-27
Ubuntu comes with CD writing capabilities out-of-the-box. Dapper does anyway, not sure about Breezy. You insert a blank CD and it'll ask if you want to write to it, and I think you can just drag and drop. For a more "pro" interface and features, there's Nero for Linux and GnomeBaker, both of which are excellent software. I'm guessing both of these let you erase data from CD-RW's, I'm not sure about the built-in functionality. It may be as simple as navigating the CD and pressing delete.
