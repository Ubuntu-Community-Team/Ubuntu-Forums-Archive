### Post by mike_m on 2006-06-20
Update:
I found that by clicking the title bar menu (left hand icon) > Advanced > Special Window Settings, I get the "Edit Window-Specific Settings"
Where I forced Kmix to Desktop 2 so I don't have to close it everytime I boot.
You can do a lot with windows like minimize & hide from task bar, etc.

So this was an OK work around, Just wanted to post to help others that find this posting :)

### Post by Miles_C on 2008-03-27
If you're using the KDE desktop environment, you can go to the control center, then Window Specific Settings (under Desktop).  Create a new behavior.  Select detect, then click on the kmix window to detect it.  Then under Geometry, hit minimized, apply initially.
