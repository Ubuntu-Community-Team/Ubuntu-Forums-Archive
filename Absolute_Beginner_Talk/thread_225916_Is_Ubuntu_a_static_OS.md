### Post by 23meg on 2006-07-30
After any stable release, only security updates are made available. 
> 
Is there no way to make Ubuntu have the lastest released stable version of software updated within a reasonable time of its release?This is what backports and third party repositories are for. If you're still unable to find a latest version, try Klik and Autopackage, and if it's still unavailable, you'll have to compile from source, or get someone to do it for you. 

> If so I think that that's a little disappointing, as if the main developer of the program thinks that it is stable then Ubuntu should presme it is as well (of course users could disable non-security updates).No, because once the new version of any software ships with a popular distro, it gets tested by millions of people, a lot of whom report bugs and send patches that amend bugs, and that's how the software actually becomes stable. The upstream author isn't the one and only authortity on the software that decides it's stable and "ready".
