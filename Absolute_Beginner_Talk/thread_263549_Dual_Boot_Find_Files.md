### Post by benuski on 2006-09-23
Usually, when you install a dual boot system, Ubuntu loads up your other partition as hda1 and gives a icon for it right on the desktop.  If you click on that, you should be able to open up any file that Ubuntu can read, and then save it onto your new Ubuntu partition if you wish, or you could just copy and past them too.
