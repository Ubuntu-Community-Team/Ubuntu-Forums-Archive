### Post by croak77 on 2006-08-03
I think you are okay. As far as ksysguard, go by the blue in the Physical Memory graph, the blue is memory used by applications, red is for buffered and yellow is cached. When is says 'Memory 412,716 used' that counts cached. As long as you don't have a crazy amount of Swap being used, you should be okay.

I find Firefox to be sluggish anyways. I would suggest using konqueror.
