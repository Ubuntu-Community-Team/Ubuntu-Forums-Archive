### Post by Mike_N on 2006-07-26
If it's asking you if you want ISO 1 or 2 or Juliet, you're making a data disk. You need to "burn image to disk" witch you should see an option for...

Otherwise, i like to use Img Burn for all my ISO burning as that's all it does so it does it very well and very fast...

If you burn a ISO as data, you will not be able to boot from it...
