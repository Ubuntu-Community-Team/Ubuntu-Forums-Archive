### Post by Tod Merley on 2008-07-03
Hi Are Be!

Your question interested me for two reasons.  First, I was a Broadcast Technician so new gadgets interest me.  Second, your question had few answers and I am here to help if I can.

After looking at your recorders manual ( see: [http://www.olympusamerica.com/files/DS2300ManualEN.pdf](http://www.olympusamerica.com/files/DS2300ManualEN.pdf) ) and doing a search for the word "comment" I think that the comments are added by the DSS Player Software ( see: [http://www.olympus-europa.com/consumer/images/DS-2300_system_chart_EN.pdf](http://www.olympus-europa.com/consumer/images/DS-2300_system_chart_EN.pdf) ).  

I believe your recorder uses  the .dss file standard ( see: [http://www.olympus-europa.com/consumer/2590_6730.htm](http://www.olympus-europa.com/consumer/2590_6730.htm) ).  It may be possible to edit a file using a "hex editor" ( e.g.: [http://www.rogoyski.com/adam/programs/hexedit/onlinedoc/hexedit.html](http://www.rogoyski.com/adam/programs/hexedit/onlinedoc/hexedit.html) ) to see the file format and guess at where you can insert the comment data or perhaps there is someone out there who has made SW to do this already for Linux (I would Google .dss voice file format Ubuntu or Linux).

Have a lot of fun!!

Tod
