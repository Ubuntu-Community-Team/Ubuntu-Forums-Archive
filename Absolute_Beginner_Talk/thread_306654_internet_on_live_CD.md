### Post by drr422 on 2006-11-25
If you have a wireless connection, and you have the livecd, then go to system-> Administration-> Networking.  Then you should be able to see your Wireless connection there.  You have to select it, then enter your ssid and password if you have one, and then press ok and it will get it up and running for you.  Setting up WPA is a little more tricky, but still pretty easy though, although I havn't got it working yet( partially because i have wep at home, and seldomly go to wpa locations).  For wpa, just search for wpa and you should find something that works for you.  
I hope this helps
