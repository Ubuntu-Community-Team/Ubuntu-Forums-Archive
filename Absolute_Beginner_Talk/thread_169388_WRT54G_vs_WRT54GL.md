### Post by bryanl on 2006-05-02
The choice of router between these two should not make too much difference unless you intend to get into the router firmware. The difference between the two models you noted is that the WRT54GL has a bit more memory in it and runs a version of Linux that has been subject to a lot of customization by serious hackers. The WRT54G uses a proprietary firmware system that doesn't need as much memory and does not have code available for folks to customize.
