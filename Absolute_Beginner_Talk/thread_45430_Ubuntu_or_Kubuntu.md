### Post by frodon on 2005-06-30
ok,
make a partition around 10Go for linux (windows will not see this partition), try to convert the rest in FAT32 because you will not be able to write on NTFS (except with captive project but it's not a real solution). Then your FAT32 partition will be see by both.
Your NTFS partition will be only readable with linux, that's why the best solution is always to convert it to FAT32.
You can make partitions during the install of ubuntu.

### Post by crashtest on 2005-06-30
There's really no reason not to have both KDE and Gnome.  I installed Ubuntu, and then did 'sudo apt-get install kubuntu-desktop', which worked perfectly.

Having both KDE and Gnome installed, you can then play with both and see which one you like best.  I've been a long time KDE user, but since installing Ubuntu I've come to really like gnome.
