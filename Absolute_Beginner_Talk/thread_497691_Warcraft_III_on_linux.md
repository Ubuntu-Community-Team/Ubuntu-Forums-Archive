### Post by geirha on 2007-07-10
There's a database of how well windows programs work with wine at [http://appdb.winehq.org](http://appdb.winehq.org) .
Searching for Warcraft III there, you'll find that it is rated as gold, which means it is playable, but some features don't work correctly.
