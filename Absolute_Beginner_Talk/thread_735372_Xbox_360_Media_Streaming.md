### Post by jellygoeswobble on 2008-03-26
Hi, I realise you posted this thread a couple of months back, but I thought I'd give you a reply anyway just in case you hadn't found out how yet/ had given up. So I use Ubuntu 7.10 currently, and use a program called TwonkyMedia to stream music, video and pictures to xbox 360. It can be downloaded from [http://www.twonkymedia.com/Download/TwonkyMedia/index.html]("http://www.twonkymedia.com/Download/TwonkyMedia/index.html")

Which has 2 download options for linux, one is a 'setup package', which for some reason doesnt compile on my system, and so I opted for the other download, the 'linuxX86' which you just download, and extract wherever you want. Now, once youve extracted that, there'll be a file called 'twonkymedia', click on this and it will start up the twonkymedia server. the first time you do this it will also create a bunch of stuff in the directory you extracted everything to. Once you have the server running, click on the file in the same directory called 'twonkymedia-config.html' which should bring up a page in your web browser to onfigure the server setting. From here its relatively simple to point it to the directories containing your media. 

Hope that helped! Feel free to message me if you need help/ have any questions, but I'll be going away for a week on saturday so dont panic if I dont immediately reply!
