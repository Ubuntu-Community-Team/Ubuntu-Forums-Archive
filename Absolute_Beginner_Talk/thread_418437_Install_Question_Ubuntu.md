### Post by raja on 2007-04-22
The partitioning can be done during the installation process or it can be done beforehand. You will also have the option of allowing the parititioning to be done automatically or manually. About 5 GB would be enough for the base system ('/'). The swap partition should be 1-1.5 times the size of your RAM. The remaining can be your /home partition. I also keep a separate partition for storing data, but you can have that in /home itself.
