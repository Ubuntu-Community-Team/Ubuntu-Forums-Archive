### Post by Rhubarb on 2008-02-07
To enable more effects or play with more compiz settings and effects, get the compiz settings manager:
```
sudo aptitude install compizconfig-settings-manager
```
Once installed, go to System --> Preferences --> Advanced Desktop Effects Settings
