### Post by northern lights on 2008-02-29
If you feel comfy with command-line apps, there's nothing more simple than "mencoder"

Check out [this]("http://web.njit.edu/all_topics/Prog_Lang_Docs/html/mplayer/encoding.html") and [that]("http://gentoo-wiki.com/HOWTO_Mencoder_Introduction_Guide") link.
It might seem a bit overwhelming at first, but it's fast, reliable and does precisely what you want. If you have a specific task in mind, I'll walk you through it - post back.

mencoder can be installed via ```
sudo apt-get install mencoder
```

### Post by bumanie on 2008-02-29
Personally I use devede for most things. It seems to convert any input format into one of four output formats avi, raw  mpeg2, mpeg2 dvd player compatible etc Older versions are available in the repo's, later  up to date versions from here [http://www.rastersoft.com/programas/devede.html](http://www.rastersoft.com/programas/devede.html) or precompiled from getdeb. I think you need to have medibuntu enabled and to fix a sound issue with mencoder you have to enable backports (if using gutsy gibbon).
