Under System > Administration, if I click on Network a process named network-admin runs.

---

### Post by Nano Geek on 2007-12-14
network-admin is the program that lets you configure your netowork settings.
network-manager (which can be run by using the command **nm-applet**) is the two computer icons that sit near the top of your screen and tell you information about your network. 

[See here for more information.]("http://en.wikipedia.org/wiki/NetworkManager")
