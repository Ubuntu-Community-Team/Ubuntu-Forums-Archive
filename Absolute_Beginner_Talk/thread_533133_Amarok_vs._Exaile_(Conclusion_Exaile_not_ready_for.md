Ok, mocp is a command line music player, let's install it using a "terminal window":

```
sudo apt-get install moc
```

To use it:

```
mocp
```

TAB : switch between playlist and file browser

R: switch repeat

<,>., PERIOD and COMMA : these are for volume

q: quit mocp, but let the music play

Q: stop music and quit mocp

To stop music after using q use  mocp command again or use this:

```
mocp -x
```

Give it a try, maybe you'll find mocp useful. :)
