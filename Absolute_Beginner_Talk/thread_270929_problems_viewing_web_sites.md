### Post by K.Mandla on 2006-10-04
That sounds like everything. If you have flashplugin-nonfree, you should be able to see most Flash 7 movies, and if you have sun-java5-plugin, you'll be able to play [Bookworm]("http://popcap.com/gamepopup.php?theGame=bookworm").

If you want to watch videos through Firefox, you'll need an embedded video player and the w32 codecs. I usually just *sudo aptitude install mozilla-mplayer*, then visit the [Restricted Formats]("https://help.ubuntu.com/community/RestrictedFormats#head-6c942d1939d97331f96e42b63774003fde7daed5") page for information on the w32 codecs.
