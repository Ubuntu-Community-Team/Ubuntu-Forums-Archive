### Post by CatKiller on 2007-12-29
> **theRightNee said:**
> if  i have drivers intended for a machine running windows, i would have to use WINE to open it right?
thanks! :popcorn:

Chances are, no you wouldn't. Windows drivers in a Wine environment would only work for the pretend Windows computer that Wine makes. If you want your device to work with Linux, you'd use Linux drivers, and they are probably built in already.

Notable exceptions are for wireless network cards that need firmware to be loaded, or printer definition files that would work, except the installer has them packaged in a Windows-only way. Otherwise, your Windows driver cd should probably be recycled.
