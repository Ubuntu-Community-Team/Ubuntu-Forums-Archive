### Post by spiderbatdad on 2007-11-22
7.10 is Gutsy Gibbon. Unfortunately beryl was fused with compiz because development of Beryl stopped. I'm not sure you can get Beryl working in Gutsy due to changes in dependencies and libs...it would definitely take a lot of work. Anyway, Compiz is now the 3D desktop effect composting manager. There are a number of how-tos in the desktop effects and customizations sub-forum. It's not too hard to get working if you find the right guide. It will take a little reading to familiarize yourself with what you are trying to do, and determining if your video card with work with tweaking.

### Post by forestpixie on 2007-11-22

to actually be able to configure it you'll need the compiz config manager
 
```
sudo apt-get install compizconfig-settings-manager
```
