### Post by rich.bradshaw on 2007-04-20
Basically the answer is no. You can go to a great deal of trouble to try, but in the end its not worth it.

You can run Windows programs using wine (sudo apt-get install wine, wine LOCATIONOFWINDOWSPROGRAM), I have not heard of mac programs being run though.

Most Windows things work, but smaller programs are better as you say.
