### Post by mcmillan on 2006-01-05
I'd say either one would work fine, so go with whichever you think you would like better, you can install both if you want to see for yourself. If you start with the ubuntu install then add the kubuntu-desktop package, or use the ubuntu-desktop if you start the other way around. When you login you'll you can click on sessions to choose which desktop you want to use. I'd also say it would be worth checking out xubuntu even if the other's work fine.  I have a computer that's only about 6 months old and I run xfce just because I like it better.  The only downside of having all of them installed is you'll have a bit of redundency in some of the applications, but you can just choose whichever ones you like best and remove the ones you don't want if this bothers you.
