### Post by which_chick on 2008-03-23
Right click the menus.
Get the dropdown.
Go to Preferences (where the Control Center is SUPPOSED TO BE)

Then, on the right, pick the + thing to add a new item.  On mine, this window does not come up on top, so check your bar at the bottom of the screen for the thing that says "Create Launcher" if you don't see it.

Click on "Create Launcher"

For "name" put in ```
Control Center

```

for "command" put in ```
gnome-control-center
```

Say OK if needed.
