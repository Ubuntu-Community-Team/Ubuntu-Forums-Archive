### Post by UBuster on 2006-06-02
If it's a file thats meant to be executed I'd assume it's executed to produce the pair of files needed for my win app. I've emailed the place where I downloaded the data but so far no reply. There are some other files included on the download site that look like they might be scripts of some sort. Here is one of them called Create_All:

#!/bin/csh
set USNOBdir = /n/3/usnob		# Input File
set log = create.log

rm $log; touch $log
@ f = 0
while ($f < 1800)
    set file = `echo $f | gawk '{printf("%03d/b%04d.cat\n",$1/10,$1)}'`
    src/usnob_make $USNOBdir/$file >>& $log
    @ f += 1
end
