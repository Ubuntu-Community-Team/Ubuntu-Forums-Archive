### Post by compwiz18 on 2006-07-13
In the package manager, right click the file, and then select the Installed Files tab.  Find the file in /usr/bin and remember the name.  Open a terminal and type it in.

This may be kinda complicated for you, if you need more help, just ask.

### Post by compwiz18 on 2006-07-13
Sorry if I was unclear.

The Terminal is under Applications (in the top right of the screen) - Accessories - Terminal.  Just copy and paste (or retype) the commands.

/usr/sbin also works, and I believe /usr/share may as well.  Just type the name in the Terminal.  If it does what you want, then all is good.  If not, try another one.
