### Post by matthewv on 2005-12-23
packages.ubuntu.com contains many of the packages required for the tasks you listed. You can check what packages in the Starter Guide, System --> Help --> Ubuntu 5.10 Starter Guide . You might want to check the dependancies on the package lists, just to make sure you won't run into dependancy problems. Once you have a copy of the packages you want to install, install them by typing in a terminal ```
sudo dpkg -i <packagename.deb>
```
