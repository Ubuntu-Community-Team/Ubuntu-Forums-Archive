### Post by chewearn on 2007-12-27
1. If only the Firefox window is stucked, click on the close icon [x] at the top right corner.  After a few seconds, a dialog box should pop-up, asking you whether you want to force quit the app.

2. If the whole system lock up, try pressing CTRL+ALT+Backspace.  This will restart gdm (i.e. the entire GUI).

3. By default, pressing the power switch should bring up the log-out screen

4. If none of the above works; if the PC is networked to another, you can use ssh to remote log-in into the machine and fix it (tell me if you need help to work this out)
