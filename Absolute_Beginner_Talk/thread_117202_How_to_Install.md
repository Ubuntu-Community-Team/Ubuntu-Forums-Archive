### Post by Mathias-K on 2006-01-14
I believe there are 6 CD versions of Ubuntu 5.10:

AMD64 Live = Live CD for Athlon64 and Pentium EM64T processors.
i386 Live = Live CD for normal PC's
PPC Live = Live CD for Mac's

AMD64 Live = Install CD for Athlon64 and Pentium EM64T processors.
i386 Live = Install CD for normal PC's
PPC Live = Install CD for Mac's

As you are most likely using a normal 32bit PC, the i386 install CD should be the one for you. Be sure to burn it as an ISO image. This can be done very easily with Cheetah CD Burner --> [http://www.download.com/3000-2646-10172045.html](http://www.download.com/3000-2646-10172045.html)

Also, burning the image at low speeds like 1-2x can save you from a lot of trouble and installation errors.

Happy getting to know Ubuntu :)
