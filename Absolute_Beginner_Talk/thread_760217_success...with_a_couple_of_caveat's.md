Well, understanding that I have only just now heard of this Joomia and have only 10 minutes worth of education in the matter, I think it would be best to have Joomia set up on the server PC and access it remotely. As for the GUI problems, I am speechless.  I have managed to bork my system in some pretty fantastic ways, but this beats all of them.  How fresh is that install?  If you've done a lot of experimenting on it, I would suggest a reinstall.  It's technically possible to undo everything you've done, but I find it's more economical to just do a fresh install.  If you haven't done a whole lot, and the install is still pretty fresh, you should look into heat issues.  You may need to dust your CPU heatsink or something.
