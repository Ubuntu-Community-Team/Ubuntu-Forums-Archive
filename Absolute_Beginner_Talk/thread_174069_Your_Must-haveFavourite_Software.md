### Post by 5-HT on 2006-05-11
md5deep (enhanced message digest calculator that provides support for MD5, SHA-1, SHA-256, Tiger, & Whirlpool digests)

It's very similar to md5sum (part of coreutils), but can recursively examine directory trees.

This comes in very handy when you want to verify the integrity of backups, or just nested directories, but there are many more applications in which md5deep can be really handy.

It also has many different options for displaying results when comparing message digests to a list (listing only those that do/don't match plus much more).

md5deep has made its way into dapper universe :p

### Post by AusIV4 on 2007-11-08
It's expected that any desktop operating system will have certain basic applications, and while there are some great applications available for Ubuntu / Linux, I'm going to list my favorite features that set it apart from other operating systems.

**MythTV** - it's changed the way I watch television. I used to have a few shows that I worked my schedule around, and watched maybe  3 hours of TV a week. Now I have a bunch of shows that I watch whenever I have time. Sometimes I'll go a week without watching a show, other times I'll watch several hours a day, but now I do it on my schedule. Of course, auto-skipping commercials means watching 3 hours of TV now takes 2 hours.

**RAID and LVM** - A fact of recording a bunch of TV shows is that you fill a lot of space. RAID leaves me confident that I won't lose my data due to a hardware failure, and LVM makes it easy for me to organize my data.

**Amarok** - I've never found a music player quite like it. It lets me queue up my music so I can listen to it how I want to listen to it. It tracks my listening habits and has interesting insight into what my favorite songs are. It lets me manage my iPod or any other music device I may wish to manage. All around slick music player.

**The command line** - One thing I hate when I have to use Windows for whatever reason is that I have to navigate through a bunch of GUIs to do a relatively simple task. It has a steep learning curve, but I definitely enjoy the simplicity of the CLI.

**Repositories** - I never knew how elegant installing programs could be. Numerous programs can share common libraries, subsequently reducing memory usage, and dependencies are all handled by one application. Almost all of the applications I need are in one place, and compiled in such a way to work well together.

**Firefox** - My first real forray into the world of open source was Firefox. Aside from just browsing the web, it's extension system makes it easy to tailor to the needs and interests of its users.

**Composite window managing** - Compiz fusion provides an awesome balance of eye candy and useful features. Expo makes organizing multiple desktops much easier than traditional means. Scale saves time searching for the right window. The widget layer along with Screenlets makes useful information available with a stroke of the mouse.

**Tomboy** - I'm a student and spend a lot of time in class taking notes, and I want to be able to read them and easily navigate through them at a later time. Tomboy makes this simple. Before I found tomboy, I was always trying to figure out how to organize my notes - messing with file names and folders, trying to figure out how to reference one note from another, etc. While there are a few features I'd still like to see added to tomboy, it's current implementation makes note taking a breeze.

Those are the software and features I find that set free software apart from the proprietary alternatives. OpenOffice is great, but I can't say it's much better than the commercial alternatives. Pidgin is pretty cool, but it doesn't seem like its living up to its full potential. Nautilus and Konqueror both have their strengths, but they don't seem to significantly surpass implementations on other systems. The media engines for Linux are adequate, but generally it seems that they're just trying to keep up with the commercial codecs. I use Gnucash for my finances, and it's good, but there's nothing particularly spectacular about it.

Overall, I'm constantly impressed with the quality of free software that is available with Ubuntu.

### Post by ShodanjoDM on 2008-04-08
My list:

**gps**
> Description: Graphical Process Statistics using GTK+
gPS is a graphical application to watch system processes, like ps and top. It shows lots of information on each process and allows users to kill, send signals to, renice and  filter processes. And also with the rgpsp package can watch processes on other machines across a network.

**debfoster**
> Description: Install only wanted Debian packages
debfoster is a wrapper program for apt and dpkg.  When first run, it will ask you which of the installed packages you want to keep installed. 
 
After that, it maintains a list of packages that you want to have installed on your system.  It uses this list to detect packages that have been installed only because other packages depended on them.  If one of these dependencies changes, debfoster will take notice, and ask if you want to remove the old package. 
 
This helps you to maintain a clean Debian install, without old (mainly library) packages lying around that aren't used any more.

**deborphan**
> Description: Find orphaned libraries
deborphan finds "orphaned" packages on your system. It determines which packages have no other packages depending on their installation, and shows you a list of these packages.
It is most useful when finding libraries, but it can be used on packages in all sections.

**localepurge**
> Description: Automagically remove unnecessary locale data
This is just a simple script to recover diskspace wasted for unneeded locale files and localized man pages. It will automagically be invoked upon completion of any apt installation run. 
 
Please note, that this tool is a hack which is *not* integrated with Debian's package management system and therefore is not for the faint of heart. This program interferes with the Debian package management and does provoke strange, but usually harmless, behaviour of programs related with apt/dpkg like dpkg-repack, debsums, reportbug, etc.
Responsibility for its usage and possible breakage of your system therefore lies in the sysadmin's (your) hands. 
 
Please definitely do abstain from reporting any such bugs blaming localepurge if you break your system by using it. If you don't know what you are doing and can't handle any resulting breakage on your own then please simply don't use this package.


**dpkg-repack**
> Description: puts an unpacked .deb file back together
dpkg-repack creates a .deb file out of a debian package that has already been installed. If any changes have been made to the package while it was unpacked (ie, files in /etc were modified), the new package will inherit the changes. 
 
This utility can make it easy to copy packages from one computer to another, or to recreate packages that are installed on your system, but no longer available elsewhere, or to store the current state of a package before you upgrade it.

**sysv-rc-conf**
> Description: SysV init runlevel configuration tool for the terminal
sysv-rc-conf provides a terminal GUI for managing "/etc/rc{runlevel}.d/" symlinks.  The interface comes in two different flavors, one that simply allows turning services on or off and another that allows for more fine tuned management of the symlinks.  Unlike most runlevel config programs, you can edit startup scripts for any runlevel, not just your current one.
