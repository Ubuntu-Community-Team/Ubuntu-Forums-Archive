### Post by cymen on 2007-05-27
I've no idea if it's "the best" but I personally use Avidemux. It lets you move from frame to frame and save them as BMP or JPEG images. It was certainly what I was looking for, and does a decent enough job. 

```
sudo apt-get install avidemux
```
