### Post by userundefine on 2006-06-11
When you install Ubuntu, it installs not only the basic operating system but many programs along with it.  It happens automatically and transparently to you.

Automatrix is a script to install lots of basic plugins and drivers and programs (mp3 support, Azureus, etc.) that you would otherwise have to install by reading the UbuntuWiki.  You don't have to use it if you don't want to, and you can pick and choose what Automatrix installs.

### Post by johnc4510 on 2006-06-11
Dapper comes ready to use with Open Office, and many other apps. Download the iso and burn it. Then boot your computer from the live cd. All the programs in the menus are ones that will be already installed if you decide to install the system. Automatix is an automated install of many different multimedia programs, drivers, etc. that are a little tricky for new users to know what and how to install.

### Post by arochester on 2006-06-11
The installation CD will come with some basic programs on it. You will then download others if you want them.

Normally distributions do not come with some programs like Acrobat, Flash or Java. These are not "free" (as in liberty) they are "propiatary" (as in commercial- but free as in no cost). The biggest job of Automatix is to install these propiatary programs.

You won't need to download anything else with the install CD

### Post by matrooswolf on 2006-06-11
openoffice and a lot of other programs are on the cd, and are installed by default.

To download new programs (or packages) you can use synaptic (you can find it under system/administration/synaptic).  You can choose which program and it will be automatically installed in the menus.

(If you are new, do try Automatix, it will help you install a lot of stuff in a smooth way.)

For more documentation read this:

[http://help.ubuntu.com/ubuntu/desktopguide/C/index.html](http://help.ubuntu.com/ubuntu/desktopguide/C/index.html)

it will help you on your way for a lot of things.
