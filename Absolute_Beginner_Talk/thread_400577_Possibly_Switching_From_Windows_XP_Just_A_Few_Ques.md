### Post by Tux Aubrey on 2007-04-03
Welcome!

The short answer is "yes".

Itunes itself is not available for Linux but several music programs are compatible (Amarok and Exaile are two that I know of). I'm no expert and I'm sure you'll get better advice on this.

Firefox 2 is native on Ubuntu and comes as part of the package.  Lots of other browsers too.

There are several DVD burning programs - but DRM copy protection can still be a problem.

There are also several Limewire-type torrent programs - Limewire, Frostwire, Azureus - plus a built in torrent downloader.  A lot of people use the Opera browser which has a built in torrent tracker.

USB and external drives all work fine for me - all of mine are autodetected and plug 'n play.

And your machine should run Ubuntu just fine.

Use the Live CD to check wireless and graphics cards - and your printer (if you have one).  If things don't work out of the box, there is usually a work-around.  Wireless is likely to be your biggest headache (depends on the chip).

Good luck.

### Post by Roelski on 2007-04-04
One advice: during install, make sure you make 3 partitions:

1. root
2. /home
3. swap

If you mess up your root-partition, this way you can wipe it out, and re-install without too much hassle.

Off course, you'll also keep the Windows-partition but you resize it... Remember: NTFS is read-only under linux.

Setting it up this way allows you to dual boot - which is, IMHO, one of the best ways to start since you have something to fall back on.

Good luck!


Roelski
