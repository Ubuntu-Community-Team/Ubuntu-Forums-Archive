### Post by Hospadar on 2007-10-24
You'll need to use the ntfs-3g drivers.  I'm not sure about 6.06 but I think they are available for that.  Open up synaptic and locate and install "ntfs-3g" and "ntfs-config" then run "ntfs-config" from a terminal.
If they arn't available as packages, you may need to google the ntfs-3g website and build and install them manually.
