### Post by Joeb on 2006-06-30
For the program that opens when you  insert an audio CD, go to the System menu, then Preferences then Removable Drives and Media.  On the Multimedia tab you can set the program that plays audio CDs.

If, while in nautilus, you right click on one of the icons of the file type you want to change, and select the properties menu choice, there should be an "Open with" tab on the properties window which is displayed.  From there you should be able to set the default application for that file type.

There may be under KDE, but under gnome, I don't believe there is an editor to set file associations (or mime-types as they are called under Gnome).

Gnome is following the Freedesktop.org standard.  However, it is easy enough to manually add your own mime-types.  The following link has an example: [http://www.ces.clemson.edu/linux/fc4_desktop.shtml#gmime](http://www.ces.clemson.edu/linux/fc4_desktop.shtml#gmime)  (just ignore the fact that it is from a Fedora Core web site!)

Future versions of Gnome should have an editor, it's just not there yet.
