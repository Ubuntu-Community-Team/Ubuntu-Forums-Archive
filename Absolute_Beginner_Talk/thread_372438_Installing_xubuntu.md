### Post by steven8 on 2007-02-28
If you have ubuntu install, and want xubuntu just open a terminal and enter:

> sudo aptitude install xubuntu-desktop

If you decide you don't like it, use the terminal to:

> sudo aptitude remove xubuntu-desktop
