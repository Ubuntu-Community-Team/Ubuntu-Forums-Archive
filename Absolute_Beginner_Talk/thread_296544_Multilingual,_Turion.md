### Post by Timokl on 2006-11-13
> **PartisanEntity said:**
> Did I download the wrong version? This is the one I downloaded:
[ftp://ftp.fu-berlin.de/linux/ubuntu/releases/edgy/ubuntu-6.10-alternate-i386.iso.torrent](ftp://ftp.fu-berlin.de/linux/ubuntu/releases/edgy/ubuntu-6.10-alternate-i386.iso.torrent)

Is this the wrong version for testing off of the CD?

The Ubuntu CD comes in two flavours - there is a live cd, and the alternate cd that you downloaded. With the alternate cd you can only install Ubuntu.

So, if you want to boot from the cd before testing, you have to download the live-cd which is name desktop cd officially. ;) 

You can download it via Bittorrent: [http://ftp.uni-bayreuth.de/linux/ubuntu/releases/edgy/ubuntu-6.10-desktop-i386.iso.torrent](http://ftp.uni-bayreuth.de/linux/ubuntu/releases/edgy/ubuntu-6.10-desktop-i386.iso.torrent)

You can boot up Ubuntu with this CD, and also install it directly from there. 

Be careful: Loading applications from the live cd is sloooooow (saulahm :mrgreen:)

### Post by Timokl on 2006-11-13
And also: With a Turion you can choose between the 32-Bit-version or the 64-Bit-version. You can run both on your machine.

However, if you are new to Linux (like me :rolleyes:), I would definitely recommend you to go for the 32-Bit version (i386) because some applications don't work that well (or not at all) in a 64 Bit environment. This is not a problem of Linux but of some applications. :neutral:
