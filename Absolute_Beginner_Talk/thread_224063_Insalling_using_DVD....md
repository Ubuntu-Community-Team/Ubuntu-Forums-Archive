### Post by taurus on 2006-07-27
You cannot just copy a bunch of .iso files to a DVD and expect it to boot!!!  You have to burn those .iso files as an image file, using whatever burning software in whatever OS you are using.  Therefore, if you download three .iso files, you need to burn them on three different CDs.

### Post by Paerez on 2006-07-27
The ISO has some special boot information on it, so when you put them all on a dvd you probably lost the boot info. In ubuntu, just right-click the iso and click "burn to cd". That's it.

### Post by x64Jimbo on 2006-07-27
