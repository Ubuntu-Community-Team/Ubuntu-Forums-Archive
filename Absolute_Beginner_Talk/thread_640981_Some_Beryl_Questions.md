### Post by toddward on 2007-12-14
If you have a chance, try to post some information about which version of Ubuntu you're running.  

For me, though, using Ubuntu 7.10, I was able to go to:
**System -> Preferences -> Advanced Desktop Effect Settings**

Inside there I checked out the 'Expo Plugin'.  You're then able to edit actions.  In this instance, I was able to set the upper right portion of my screen to activate the mouse action.
