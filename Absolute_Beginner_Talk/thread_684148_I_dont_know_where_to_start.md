### Post by mwacky on 2008-01-31
To run a Live CD, put the cd in your tray and reboot, make sure your cpu will boot to the cd rom (if it doesn't you'll have to change a setting in your bios, hit F2 as soon as the computer boots).  This will allow you to run linux on your PC without having to install it and is a great way to try it out.

As far as Microsoft Virtual PC, I've never had luck running linux. See this wikipedia article under "Emulating Linux Based Environments":

[http://en.wikipedia.org/wiki/Virtual_PC]("http://en.wikipedia.org/wiki/Virtual_PC")

It basically says you have to install in text-based mode, which means you need to select the install in graphics safe mode from the startup options.  I've never tried this.

You should give VmWare Server a shot, it is free you just need to register for key codes:

[http://vmware.com/products/server]("http://vmware.com/products/server/")

You could also install VMware Player and download a virtual appliance.  I would try the Live CD to save you some time if you just want to take a peak at Linux.
