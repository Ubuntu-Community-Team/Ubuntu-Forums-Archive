### Post by arpan on 2007-07-30
Hi i guess this is what you're looking for..

Right click on desktop...
-> Create New ---> Link to Location(URL)

A dialog will appear asking for File Name and URL.

Give File Name = My Computer
and URL = media:/

Click OK.

On my Kubuntu 7.04, it took the default icon but in your case if it does not take any sensible icon then change it in following manner.

Right Click on Icon(URL) that you created for your computer as above, select properties and there will be the icon displayed on desktop.  Click on that icon and you will be presented with many icons to select one for your shortcut.

Hope this will help you.
