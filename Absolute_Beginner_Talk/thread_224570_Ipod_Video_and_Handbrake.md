dvpod does exactly this.

dvpod is a simple wrapper script for HandBrake that simplifies and automates ripping one or more tracks from a DVD to a file which can be viewed on a video iPod.

It can be downloaded from: [http://sourceforge.net/projects/dvpod](http://sourceforge.net/projects/dvpod)

You will obviously need to have HandBrake installed in order to run this.
