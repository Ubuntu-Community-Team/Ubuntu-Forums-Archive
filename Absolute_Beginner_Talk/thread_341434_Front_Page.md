### Post by tonyr1988 on 2007-01-18
[Nvu]("http://www.nvu.com/index.php") (sudo aptitude install nvu)
[Kompozer]("http://kompozer.net/") (I think you'll need to download the tarball)
[Quanta]("http://quanta.kdewebdev.org/") (sudo aptitude install quanta)
[Amaya]("http://www.w3.org/Amaya/") (sudo aptitude install amaya)
[Bluefish]("http://bluefish.openoffice.nl/index.html") (sudo aptitude install bluefish)

There's probably more - you may want to check them all out and see which ones you like the best. Which one's best for you will depend a lot on your skill level and how many features you need in a WYSIWYG editor.

It also appears that Frontpage may work under Wine (via their [appdb]("http://appdb.winehq.org/appview.php?iVersionId=529"))

And, there's always vi and emacs. :D
