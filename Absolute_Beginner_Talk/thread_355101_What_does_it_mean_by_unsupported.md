### Post by igknighted on 2007-02-06
> **-Ghost9- said:**
> "Beryl is not supported on Ubuntu Dapper. Please consider upgrading to a newer version of Ubuntu instead."

[http://wiki.beryl-project.org/wiki/Install_Beryl_on_Ubuntu](http://wiki.beryl-project.org/wiki/Install_Beryl_on_Ubuntu)

It means dapper has old versions of software (like X.org and others) that beryl is not tested on.  Beryl (especially on AIGLX) makes use of new features that are not on dapper.  It is possible to run beryl on dapper, but as beryl develops there may be issues which require you to run an old version of beryl.
