### Post by mlentink on 2007-09-04
Personally, I would advise you to use the LiveCD first. It will run a lot slower that a full install, but it will leave your current Windows installation as is, and will allow you to test if your gear is at all compatible. 
Download the appropriate ISO [here]("http://www.ubuntu.com/getubuntu/download") and burn it (slowly!) as an image. Boot up the cd and try it out. 
When you decide you like it, go for either Wubi or for a dual boot. Youll find more info on help.ubuntu.com
