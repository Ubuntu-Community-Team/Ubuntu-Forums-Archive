### Post by em3raldxiii on 2008-04-12
I have seen that problem once before, quite a long time ago.  I think it was when I was installing Breezy.  Unfortunately, I can't remember what I did to fix it, but I think I might have defragged my windows partition(s), and then everything worked.  It seems that if something is very fragmented, the partitioner has a problem; so I would say give that a try.

As far as being confused goes ... you're not alone.  Many people have expressed much the same sentiment.  The net result is that apparently the new Hardy will come with both a "try the live CD" option, and an "install" option.  I think one of the install options will include the non-graphical installation (not 100% sure about that one).

Anyhoo ... try defragging everything, and then try again.  If it still doesn't work, please post here and we will try something else :D
