### Post by d1337 on 2006-02-13
If you are attempting this on the Compaq Presario 5630 that you were having install issues on about a month ago, I'm still going to say that you do not have enough hardware to make Gnome move any faster.  A PII with 128megs of ram, regardless of how much swap you have, is not going to make for a speedy Gnome environment.  The only thing I could see making your system any faster would be to get rid of all of the Gnome specific packages using [this link]("http://www.ubuntuforums.org/showthread.php?t=96046") as a reference (Thanks Aysiu) which will basically put you back to what you would have had from a server install (while maintaining **most** /home stuff [backup whatever you don't want to potentially lose] and bypassing install issues from before)...and install xubuntu-desktop from there (with xfce).  If you have specific issues with xfce breaking, perhaps addressing those problems would be less frustrating for you.  Gnome is a great environment, but it needs some resources to be enjoyable.

I know that this is likely not the answer you were looking for, but I really think you'd be happier.
