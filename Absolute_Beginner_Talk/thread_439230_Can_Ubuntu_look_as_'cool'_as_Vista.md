### Post by Tomosaur on 2007-05-10
> **tchase said:**
> Ubuntu seems to have a cool enough clean look to it, but is there a way to customize it to look as visually attractive as Vista?

Depends what you mean? There are many themes which are available to make Ubuntu look like Vista, sure - just take a look at [http://www.gnome-look.org](http://www.gnome-look.org). There are quite a few Aero (the Vista interface) copies there, and a few which aren't copies of Aero but which still look 'glassy', take your pick!

If you're talking about the desktop effects which Aero uses - Linux has had all of that for a while now. Do a youtube search for 'Beryl' or 'Compiz', to see what I'm talking about. Compiz is installed on the latest version of Ubuntu by default, but you need to enable it yourself (System > Preferences > Desktop Effects), because some people either simply don't want it, while others have really old systems without 3D acceleration.

Welcome to the forums, good luck with Ubuntu :)
