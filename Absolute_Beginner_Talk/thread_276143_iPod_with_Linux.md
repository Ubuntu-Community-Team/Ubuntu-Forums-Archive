### Post by Marsolin on 2006-10-12
There are quite a few that should work. [Here]("http://linuxappfinder.com/multimedia/portableaudiomanagers?search_text=ipod&search_type=desc") is a list of some potentials. You can either use a full music manager like [amaroK]("http://linuxappfinder.com/package/amarok"), [Banshee]("http://linuxappfinder.com/package/banshee"), or [Listen]("http://linuxappfinder.com/package/listen"); or you can use a dedicated program like [Gtkpod]("http://linuxappfinder.com/package/gtkpod").
