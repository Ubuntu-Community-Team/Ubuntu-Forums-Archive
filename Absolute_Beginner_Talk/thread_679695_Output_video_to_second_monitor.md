### Post by ~LoKe on 2008-01-27
I know you can do something similar with VLC.  In VLC, you're able to choose which display to have it run its fullscreen videos on.  If this is not what you're trying to do, I apologize.

In VLC...
Settings **->** Preferences.
|__Video **->** Output modules

Tick the "Advanced Options" box at the bottom right corner.  Choose the output module you'd like to use (I suggest X11) then enter the screen number in "Screen for fullscreen mode"
