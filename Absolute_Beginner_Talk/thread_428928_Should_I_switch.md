### Post by whee on 2007-04-30
> **Jakk55 said:**
> Well I have been a windows since windows 3.0, I have heard from a variety of sources that Linux is far better and possibly made by god himself, and that ubuntu is the best home distribution out there.

Now my questions are:

1) One of my main reasons for switching was that i felt more freedom when working in Linux/Ubuntu than working with Windows. After a while of using Linux i realized that on certain aspects Windows seems to work against the user, rather then flow naturally with the user. I know this sounds kind of weird maybe, but you'll probably understand it after a while of using Ubuntu.
This doesn't apply to everything in Windows though, just some things, i don't resent Windows, it has served me well for years and has it's strong points, but i feel Linux is the next step for me. 

2) Yes. You can install a program called Wine which is basically a reversed engineered Windows library, which allows you to run some Windows applications. Not all Windows applications work flawlessly with it though. If you would want that you can run Windows in a virtual machine inside Linux/Ubuntu and so run a particular Windows application, but this comes with a performance penalty. Or you can decide to dual-boot.



1) Linux is more secure out of the box, especially Ubuntu. After it is installed all ports are stealth, this is not so in Windows. So it basically has a built-in firewall which is quite good.
Also in Ubuntu you generally only work with a normal user account instead of a root account, this means your system files cannot be changed/compromised so easily. This is also something that is not so in Windows XP, though Vista also started with this strategy, though from what i've heard this is not implemented as gracefully in Vista as it is in Linux/Ubuntu.

2) Some are compatible. Azureus is compatible, it uses the Java Runtime Environment, which is available for Windows aswell as Linux. Nero is also available for Linux, but i recommend K3b, which is free and open source and works pretty much just as well as Nero.
I don't know if Star Wars battlefront is available for Linux(maybe, i don't know), but some games run in Wine, though there is an application called Cedega, which is designed to run games which normally run only in Windows.(You could check out Cedega's compatibility list)


Apart from that, well this is a Linux/Ubuntu forum, people will probably advise you to switch, but i think you have to be more cautious with it. I know Ubuntu is pushed quite hard right now on the internet and i myself find it to be a joy to work in, but i also know it's not yet flawless, it has some bugs here and there and i have run into some problems in the past. But it's getting better, i have found Feisty Fawn to be the most polished release yet.


PS: about the question "1.Why should i switch?", you could try to answer this question yourself, because nobody can find(or not find) a better reason for you than you yourself. Also you could try the liveCD, but i have found i understood better what Ubuntu was about after i installed it.
It's up to you. If you don't want to burn all your bridges you could install it on a spare(non-critical) machine if you have one, or a spare disk or maybe even a spare partition.(dual-booting)
But be careful with messing with partitions, it could potentially mess up your current Windows installation.

### Post by ceo51378 on 2007-04-30
I'm in the same boat as you.  I'm an IT Manager for my company so I've been using Windows since day 1.  I'm rather new to Linux myself.  I played with Suse, Fedora, and Redhat in the past.  So far, Ubuntu is by far the easiest and most user-friendly Distro I have come across.  I have it running on my HP NC6400, Dell D620, and a custom built PC that I threw together at couple years ago (I actually tested it on my PC first).  

Why should you switch?  
- You don't have to worry about viruses.  I suppose it's possible but 1000x less likely on a linux box.  That is actually the #1 reason for me switching.
-Windows gets bloated over time with the applications you install writing startup scripts, crappy memory allocation, etc.  I end up flattening my machines every couple of months due to performance decreases.  With Linux, it only does what you tell it to do - and with Ubuntu, it is the easiest Distro to get you started (at least from what I have tried).
-It's FREE!!!  And so is 99% of the software that goes with it.  So far, I have found an excellent equivalent for all of the Windows apps I use.  
Will you be able to use your Windows apps?
Yes and No.  Check out [http://www.winehq.com/]("http://www.winehq.com/") to see what apps/games you can run.  There is also an app called Crossover Office which allows you to run apps like MS Office and Photoshop.  There is even a Nero4Linux which will set you back $20.  

I'm not going to say that it's been a piece of cake making the switch - especially with my laptops.  I still have a windows box running my file/print/ftp server (FOR NOW ;) )  It will be even harder for you since you have become accustomed to the windows file structure; however, you would still run into this if you switched to a Mac.  On the plus side, Ubuntu makes it VERY easy to install a dual boot partition so you can choose which OS to use until you are comfortable making the switch.  Feisty even gives you an option to pull your windows home directory over during the installation process.  Most off my difficulties have been with hardware detection with my laptops, however, I found all of my answers online rather quickly.

One other thing to take into consideration.  The next inevitable step in Windows is Vista.  I have tested it on a Core2 Duo laptop with 2 gis of ram and a beefed up video card.  Vista dogged it; it is a huge memory hog. If you plan to stick with Windows and upgrade to Vista, you will need to buy a very very fast machine with at least 2 gigs of RAM; actually, I would recommend 4.
