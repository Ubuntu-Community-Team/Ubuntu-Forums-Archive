### Post by ruscoe on 2007-06-04
I got a standard USB gamepad to work on Ubuntu with a utility called QJoyPad.
[http://qjoypad.sourceforge.net/](http://qjoypad.sourceforge.net/)

I found the deb package available on the site above was broken so I used alien to convert one of the rpms. You can install from source if you want.

It's lots of fun to configure the controls to work with Beryl.
