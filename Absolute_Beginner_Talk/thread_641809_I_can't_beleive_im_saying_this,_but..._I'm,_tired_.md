### Post by jfinkels on 2007-12-15
Ubuntu 7.10 Gutsy Gibbon has Compiz enabled by default. If you install Compiz Config Settings Manager by typing:```
sudo aptitude install compizconfig-settings-manager
``` and then run it by typing:```
ccsm
``` in the terminal, you will be able to change the settings for Compiz at your will.

Themes, backgrounds, and much much more are here: [http://www.gnome-look.org/](http://www.gnome-look.org/) and here [http://art.gnome.org/](http://art.gnome.org/)

Ubuntu is completely customizable; it will only be as exciting as you make it!
