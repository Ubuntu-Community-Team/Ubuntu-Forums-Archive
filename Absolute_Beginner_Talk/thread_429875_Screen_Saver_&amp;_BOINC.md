### Post by Paqman on 2008-03-18
In effect, BOINC on Linux works the same way it does on Windows. It runs as a daemon (a process which runs silently in the background) and starts crunching work units when your keyboard and mouse have been idle for a bit.

So you don't get the pretty graphics, but you still crunch data.
