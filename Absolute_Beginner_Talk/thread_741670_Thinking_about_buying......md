### Post by mgranet on 2008-04-01
I can't offer any advice about which system to purchace, but I can offer warning. Take the news of the Ubuntu box at CanSecWest with a grain of salt. Just because it wasn't hacked there, does not mean that Ubuntu is a godlike unhackable system. Many hackers/crackers/whatever found vulnerabilities in the system there, but in order to qualify for the prizes, they had to have working exploit code. I've been to many a security convention, and most folks are there to relax and learn, not spend the whole weekend writing exploit code.

### Post by riven0 on 2008-04-01
I've only heard great things about System76, so I don't think you'll go wrong no matter the system you go with. However, you're probably better off posting this question in the System76 section of the UbuntuForums. You'll find it on the front page under the Main Support Categories section. But as Mgranet posted, don't just switch to Linux because you think it is hacker proof. I do think it is more secure than Windows and OSX and security holes get patched amazingly fast, but before buying a whole new system, first try out Ubuntu and make sure you like it. Burn the 7.10 live cd and test it out and then make your decision.
