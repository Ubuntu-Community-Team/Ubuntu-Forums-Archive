I'm new to Ubuntu and I'm building a computer for use in an "insecure" location. Newegg sold me an ITX case, powersupply, motherboard, CPU and 1 GB of RAM for $125. 

I'm donating it to a church, school, or prison, whichever is the most interested in it. 

It's designed from the ground up to be in a location where it might be stolen, borrowed, walk off, etc.

To prevent this I took the fololowing precautions:

1. made sure the case was small enough to fit in an office desk drawer, and after it was tested used "one way" screws to attach the cover to the case..

2. Glued it to the bottom of the desk drawer with PC 7 epoxy adhesive in such a way that it can be removed only by prying the box loose with a crowbar or melting the adhesive with a torch. Either one of these steps willl either smash the motherboard or melt it before the box comes loose from the drawer.

3. The computer will use a USB mouse and keyboard that go for $9.95 each in local stores, and will make use of a VGA monitor which is  available from almost anybody who buys a new computer and wants to give away the monitor they used on the old one.

4. Loading Open Office on it provides the user with a computer that can be used to teach computer basics and basic word processing and sptreadsheet skills.

What I need to learn to further this project is:

1. Find out how to password-protect both the BIOS and the operating system of Ubuntu, if it's possible to do this.

2. Findo out how to install Ubuntu in computers that people donate after getting a new desktop or laptop system.

 3. Find ourt if there are any manuals on how Ubuntu or Open Office works that are available for free distribution.

If anybody can give me any advice on how to do this, it would be really appreciated.

> 1. Find out how to password-protect both the BIOS and the operating system of Ubuntu, if it's possible to do this. 
When the PC starts up. Hit F2 usually and that will take you into the bios. From there depending on the BIOS there will be a security menu. In there will be a way to password protect the BIOS. In Ubuntu, create another user
