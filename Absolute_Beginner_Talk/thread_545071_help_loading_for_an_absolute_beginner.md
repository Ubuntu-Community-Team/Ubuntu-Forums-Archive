Sometimes hardware is incorrectly detected and the video goes into a mode that your monitor cannot handle, so you get a blank screen.

There are versions of Ubuntu that initially install without using graphics, you may want to go here and select the "alternate desktop CD" download option:

[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

After you get a text Ubuntu system up and running you should be able to get the full graphics working.
