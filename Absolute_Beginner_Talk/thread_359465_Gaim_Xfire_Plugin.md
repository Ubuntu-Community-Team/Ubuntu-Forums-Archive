### Post by Cappy on 2007-02-22
There are multiple versions of GAIM beta and they all have incompatible plugins. If you have GAIM 2.0.0 beta 3 or 3.1 you can just download the deb at [http://gfire.sourceforge.net/](http://gfire.sourceforge.net/)
If you are running Edgy you have to compile the source. If you're running 3.0/3.1 don't use a snapshot, it doesn't work [well].

If you're running beta4, 5, or 6 you can find the downloads at [http://gfire.sourceforge.net/snapshots/](http://gfire.sourceforge.net/snapshots/)
