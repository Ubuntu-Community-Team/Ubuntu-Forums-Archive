### Post by PurplePenguin on 2006-12-12
GnomeBaker burns bin/cue files (as will k3b).  You just need to open up the program, select Tools, then Burn CD Image, select the .cue file and you're all set. :)

The cd that you burn should play with no problem in Mplayer or VLC (provided that you've installed all non-free codecs via Automatix or one of the how-to files floating around out there), as well as in any set-top DVD player.
