### Post by ihristov on 2007-07-19
> **chicoicho said:**
> Thell my how install drivers and intel vga on board not disabled.Yor card how install in ubuntu?

English
--------

You have to disable your Intel VGA from the BIOS. You have practically 2 different video adapters in your system right now. 

After that you need to install the proprietary NVidia driver (nvidia). Most likely at the moment you are using the open-source nvidia driver (nv)

Installing the proprietary Nvidia driver is done either manually by following the instructions at their web site or by using the helper application [Envy]("http://www.albertomilone.com/nvidia_scripts1.html") as suggested by 'alienexplorers' above

Here is how to install it
```
cd /tmp
wget http://albertomilone.com/ubuntu/nvidia/scripts/envy_0.9.6-0ubuntu2_all.deb
sudo dpkg -i envy_0.9.6-0ubuntu2_all.deb
```

Then launch it and follow the instructions.

Once you install the proprietary NVidia drivers there is a utility called "nvidia-settings" that should allows you to manage all the properties of the tvout.
```
sudo nvidia-settings
```

BTW: I am using 7.04 and in there the nvidia drivers were installed automatically at initial install. I am not sure if 6.10 has the proprietary NVidia driver in the restricted repository.
