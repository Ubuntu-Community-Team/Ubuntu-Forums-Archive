### Post by lamalex on 2007-04-18
hey and nice choice moving to ubuntu :) I recommend it 100%. You can totally still share files between your computers. Samba isn't really that hard to set up and yes there are some decent tutorials on these forums, the wiki, and all over the web. [http://ubuntuforums.org/showthread.php?t=202605&highlight=samba+peer](http://ubuntuforums.org/showthread.php?t=202605&highlight=samba+peer) is one I've used a few times.

For your printer, check out here [http://openprinting.org/show_printer.cgi?recnum=HP-PSC_1300](http://openprinting.org/show_printer.cgi?recnum=HP-PSC_1300) you've got a green light so congrats! I'm pretty sure that printer works out of box with ubuntu, a lot of the HPs do, but if it doesn't you can get the driver from there. Samba again will be how those printers connect to the Ubuntu printer. GL, if you run into any snags just post here. We love to help! Good luck!
