### Post by furball_1185 on 2008-01-22
I don't remember the exact name of the package, but if you open the Synapitc Package Manager and search for "screensaver", you will see packages named "extras". These are the screensavers that were not included with out-of-the-box Ubuntu and, once installed, can be selected from gnome screensaver. If all you want is your old selection of screensavers back, this is the easiest way.
