### Post by igknighted on 2007-02-18
The 64 bit Ubuntu really isn't ready for prime time yet, IMHO.  The integration between 64bit and 32bit programs isnt there yet, there is no 64 bit flash (which means if you want flash, you need a 32 bit browser), and overall there are issues in lots of areas.  Mind you, there are work arounds and a 64bit system can be fully functional, its just a challenge.  For most users who don't want this challenge, and don't use super high memory/processor intensive tasks (mainly large compiles and video editing type work) then the 32bit OS is preferable.  There is nothing adverse happening to your system, except that at the very top end of your processing power you might be shortchanging yourself a little bit.

All of this said, 64 bit is the way of the future.  If you want to get on board, try Ubuntu 64 bit again, but also compare it to distro's like Suse and Sabayon.  Ubuntu (and debian in general) are behind these distros in 64bit integration ATM, so you might have better luck here.
