### Post by Daveth on 2007-08-30
I've just started sbackup running as I type. It allows you to specify which files and folders you wish to backup and to where. You can even automate the process if you so wish. I'm pointing mine at an external hard drive down a firewire link.The only problem is that is a little coy about how much it has done and when it ends. But it does the business well enough it seems. I have never had recourse to recover data from the archive though, so run it on "good faith" !
