### Post by starcraft.man on 2007-06-10
Adding to johnny, I think your confused.

Emerald is a window decorator. It alters how the windows **look.** Beryl is a window manager its function is to describe how the windows **function** and any other effects used.

Neither Beryl nor Emerald change the desktop background/wallpaper. That is handled by System > Preferences > Desktop Background.

I hope that clears it up :).
