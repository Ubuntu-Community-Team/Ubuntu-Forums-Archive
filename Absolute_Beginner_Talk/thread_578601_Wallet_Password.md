### Post by linuxwizard on 2007-10-17
You need to install Kwallet Manager and you can install it through Synaptic. Than you should be able to disconnect that applications from using Kwallet.

[http://docs.kde.org/stable/en/kdeutils/kwallet/kwalletmanager.html#kwalletmanager-context-menu](http://docs.kde.org/stable/en/kdeutils/kwallet/kwalletmanager.html#kwalletmanager-context-menu)
