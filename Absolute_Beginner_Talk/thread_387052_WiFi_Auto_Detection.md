### Post by SactoBob on 2007-03-17
You can install network-manager-gnome from the repository and see if you like it.
It will auto-connect to a known (previous) connection.  You can click on its icon at the top of the screen to show and select other networks in range.

### Post by krieger78 on 2007-06-13
If you are using wifi-radar you can select the network you wish to purge then click "Delete". this will remove it from your list of preferred networks. Wifi-radar will still see the purged network, but it will not auto connect.
