### Post by newmo on 2006-06-06
I used to have this problem with Firefox and continue to have it with my other browsers (with the exception of Konqueror)

I fixed the Firefox problem with the following advice from these forums so if you haven't already done it it's worth checking it out.

Write  about:config in the address bar of Firefox.

A field appears called Filter.

Write ipv6 in this field then enter

Highlight the details that appear (network.dns.disableIPv6) and change the Value Column from False to True.
