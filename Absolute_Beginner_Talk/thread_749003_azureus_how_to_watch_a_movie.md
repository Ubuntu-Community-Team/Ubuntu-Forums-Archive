### Post by DexterLB on 2008-04-08
Use VLC media player. It opens EVERYTHING
To install, type in the terminal:

      sudo apt-get install vlc

and when you are ready, it appears in the "sound and video" menu. If it doesn't, type in the terminal:
      vlc
