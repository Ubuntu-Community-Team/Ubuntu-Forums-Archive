### Post by Solver on 2006-02-27
In the prefs.js file, there is:

```

user_pref("browser.startup.homepage", "http://www.google.com");

```

At least that is the case for 1.5.

### Post by Solver on 2006-02-27
I don't think you want /etc/mozilla-firefox/profile . The right file for you, I think, is in the user's home directory: /home/user/.mozilla . There's a profile directory that has a name of digits and letters and .default, like 76wdyg.default . Now, the prefs.js in that directory is probably what you want.
