### Post by bluevoodoo1 on 2006-02-16
Download the icon set, let's say to your home folder, then open...

System > Preferences > Theme

now, you can simply drag the file from your home folder to the Theme dialog window.  Then you can select it from the icons tab.

I use the Mac OSX icons too and they're awesome!
