Following the instructions in my previous post doesn't get it to boot from the external drive?  I'm not sure how the BIOS works (as far as being able to configure it, edit boot options) for Mac, but it can't be drastically different than a regular PC.  It's possible the BIOS doesn't support booting from an external drive.  Very old computers, such as the one you have, don't support such a thing.  This isn't uncommon.  I don't even believe you can flash the BIOS of a laptop, I've never tried.  

I'm not Mr. Morals, but I bet this OSX DVD wasn't originally meant for your laptop.  You may want to try and call Apple to see if you can get install discs, but that likely won't happen.  I think you're stuck (in the nicest way possible) with Ubuntu.
