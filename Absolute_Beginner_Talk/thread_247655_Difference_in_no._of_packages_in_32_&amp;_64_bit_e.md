### Post by Kilz on 2006-08-30
> **sankarv said:**
> Is there difference in number of packages in 32 and 64 bit ubuntu isos?

I don't know specifically about the iso's. But there is a 2% difference in the number of packages listed as available on launchpad. There are 400 more for i386. But packages doesn't mean applications. Some of those may be libraries, development, document, or configuration packages. 
I have both versions running on my home network, I cant tell the difference in avaialable applications. But I do see more people complaining about development packages missing from 64bit. As far as common packages most people use, there isnt a lot of difference. What there is, there are work arounds for since 64bit can run 32bit applications.
