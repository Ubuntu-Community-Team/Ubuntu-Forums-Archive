### Post by troseph on 2007-07-03
Firstly, your fstab doesn't show any ext3 partitions... But all you would need to do is remove the ntfs ones from fstab.

Then make your directory in /mnt. I will name the example "storage"
```

sudo mkdir /mnt/storage

```

 and set your ext3 partition's entry to: 

```

/dev/xdbx    /mnt/storage    ext3   defaults    0    0

```

### Post by mysticrider92 on 2007-07-03
He has an ext3 listing on lines 5-6 of the fstab.

One easy option instead of changing all mountpoints to /mnt would be to use gconf-editior and tell it not to display volume icons on the desktop. Type gconf-editor in a terminal, then go to apps>nautilus>desktop and uncheck the "volumes visible" box at the bottom. This way, they will show up in Nautilus and on your "Places" menu, but not on the desktop.

### Post by ajmorris on 2007-07-03
also, dont use automatix to install software :) there are easier ways to install the software you want, like through Menu>> System >> Administration >> Synaptic Package Manager, and Automatix is a script that tries to install some software, and often
fails and breaks systems. We don't provide support for it, and we strongly discourage its use. Problems caused by Automatix are often hard to track and solve, and it might sometimes be easier to !install a fresh copy of
Ubuntu.
