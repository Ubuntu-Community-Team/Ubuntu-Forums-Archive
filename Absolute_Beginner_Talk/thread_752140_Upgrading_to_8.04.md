### Post by qamelian on 2008-04-11
When the final release of Hardy occurs, the update manager will notify you that an upgrade is available and will display a button that allows you to upgrade. At that point, you have the option of continuing to update your current Ubuntu version or upgrading to Hardy.

### Post by barbedsaber on 2008-04-11
as far as I know, when it comes out offically, you can run

```
sudo apt-get upgrade
```

to get it, but if you want to get the beta now, you can hit alt-f2 and type
```
update-manager --devel-release
```

full instructions avalible [here]("https://help.ubuntu.com/community/HardyUpgrades")

hope that helped
