This is you problem (most likely) you don't have enough RAM. You need 256 minimum to get the live CD going. I have ran the disk with less, but it is hit and miss. You may want to try:

[http://www.xubuntu.org/](http://www.xubuntu.org/)

The live disk may not work, but you can install it, as it is a lighter desk top. You can add Gnome latter, and you shouldn't loose speed. You can also try:

[http://fluxbuntu.org/](http://fluxbuntu.org/)

The Fluxbox desktop is a bit odd, but I like it. The live disk runs on about anything it seems.
