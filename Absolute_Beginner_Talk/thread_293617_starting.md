### Post by LLRNR on 2006-11-05
Ok, plo--koon : for the first issue : no, WINE does not modify the way your Ubuntu looks, because it's just an independent implementation of the Windows API (Application Programming Interface). So, what does this mean ? It means that WINE altogether is an application that you can use to run programs that would normally run only under Windows. And how's that ? Well, the guys that made WINE wrote their own stuff so by using WINE you can run Windows-based software under Linux without even having Windows. The implementation WINE uses is independent of any Microsoft code, but it can "understand" it and "interpret" it. I hope I expressed myself clearly enough, but that's what WINE mainly does - it's a utility, it's meant to make your life easier and it doesn't affect the looks of your Linux.

On the second issue: Gnome and KDE are "desktop environments" and this is what actually determines the way that your desktop looks like. Apart from the looks, each desktop environment (there are other DEs except Gnome and KDE) comes with its own applications and yes, you're right when you say that applications made for Gnome will not work under KDE, but, hey, it's a good thing to know that, by having multiple desktop environments, you actually have multiple tools (i.e. multiple choices) for doing the same thing. One example is that in Gnome, to write some text you can use the application called Gedit, for instance, and in KDE you have a similar one, called KWrite. You see, it's that cool, they do the same thing but in their own DE ! 

Hope this helps. By the way, Ubuntu distro uses Gnome as the default desktop environment and Kubuntu uses KDE. As far as I know there are no limitations, you can later install other desktop environments apart from the default one, and if you want to take a look check out osdir.com, mainly [this](http://shots.osdir.com/slideshows/slideshow.php?release=752&slide=4&title=kubuntu+6.10+screenshots) for Kubuntu and [this](http://shots.osdir.com/slideshows/slideshow.php?release=751&slide=4&title=ubuntu+6.10+screenshots) for Ubuntu (be sure to check out the slideshows on the right). Have fun !
