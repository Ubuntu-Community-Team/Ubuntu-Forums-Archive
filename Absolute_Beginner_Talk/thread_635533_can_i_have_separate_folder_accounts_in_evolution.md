### Post by rsambuca on 2007-12-08
OK, I figured it out.  First create a separate inbox for each of your accounts.  Then just go to Edit - Message Filters.  For Incoming, click "Add". Give the filter a name.  Then just set it up so that "Recipient"  "contains" then put your email address into the last box.

Then at the bottom, select "Move to Folder", and select the one you want for that particular email addy.
