### Post by Lord_Dicranius on 2007-12-21
It looks like both the plugin pack and guifications packages are in the repositories.  Install using this command:
```
sudo apt-get install pidgin-plugin-pack pidgin-guifications
```
You can find more Pidgin related packages [here](http://packages.ubuntu.com/cgi-bin/search_packages.pl?keywords=pidgin&searchon=names&subword=1&version=gutsy&release=all) :)
