### Post by qpwoeiruty on 2007-03-02
How about trying this in the terminal: sudo apt-get install kde
After you log out, you can choose KDE by going to "Options" on the bottom left corner of your screen, choosing "Change Session" and choosing KDE.

For adv/dis, it's a matter of opinion but...
Adv: More configurable
Dis: More configurable

### Post by poohbear1616 on 2007-03-02
If you haven't already done anything do this:
```
sudo apt-get update
```

Then:
```
sudo apt-get install kubuntu desktop
```
