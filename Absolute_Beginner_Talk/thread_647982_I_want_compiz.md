### Post by schauerlich on 2007-12-23
Compiz Fusion comes preinstalled in Ubuntu. To enable it, go to System>Preferences>Appearance>Visual Effects and select extra. To get even more eye candy, run ```
sudo apt-get install compizconfig-settings-manager
``` and instead of selecting extra, select custom. Hit preferences, and tweak to your liking.
