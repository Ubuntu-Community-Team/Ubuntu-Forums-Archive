### Post by The Cog on 2008-01-14
Telnet is a network protocol fol logging into remote computers. It only does text, not graphics. SSH is generally replacing telnet in newer applications because it is more secure - encrypted, but there are lots of devices out there that only accept telnet.

There is also a program called telnet (and one called ssh) that you run on your computer that makes a network connection for you and displays the text on your screen.
