### Post by quinalt on 2006-08-26
I was experiencing the exact same issue...went through all of the forum posts and tried all of the suggestions to no avail.  I'm using a netgear WGR614 wireless router to provide DHCP and network address translation (private to public address translation).  How I fixed my issue is that I configured a static IP address on my Ubuntu machine, went into my router configuration and checked the 'trusted IP address' box and voila!  It now works.  Apparently the Netgear was filtering the URL or just plain mangling the URLs...as soon as I check that box, everything went through just fine.

Hope that helps!
