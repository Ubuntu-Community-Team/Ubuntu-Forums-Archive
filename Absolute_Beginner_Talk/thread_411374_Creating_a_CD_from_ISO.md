### Post by aysiu on 2007-04-16
*Write to disc* will make it bootable. Nautilus in Gnome (unlike Explorer in  Windows) is smart enough to know it's an ISO.

What *write to disc* won't do is allow you to select what speed to write the ISO at. It'll probably write at the maximum speed, but I've found it's best to write ISOs at slower speeds (4x, for example). Any CD burning program will give you that flexibility: Gnomebaker, Graveman, K3B...
