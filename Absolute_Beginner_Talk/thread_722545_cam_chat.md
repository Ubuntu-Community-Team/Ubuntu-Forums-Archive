### Post by gobbles414 on 2008-03-12
chris200x9,

I'm honestly not sure about iChat protocols. I do know that support for AV chat on the Windows Live network is still very experimental in Linux. I think that [Mercury Messenger]("http://www.mercury.im/") is making some progress in that area. Other chat programs claim that support for AV chat is "under development" for future releases.

You may still help someone more knowledgeable than me to help you if you'll verify for us whether or not your webcam and audio are working in Ubuntu. You can test your webcam with an application called Camorama. You can install it using *Applications => Add/Remove* or *System => Administration => Synaptic Package Manager*. To test your audio go *Applications => Sound & Video => Sound Recorder* and try to record yourself speaking for a few moments.
