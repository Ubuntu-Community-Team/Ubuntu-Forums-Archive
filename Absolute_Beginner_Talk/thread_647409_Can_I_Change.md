### Post by jken146 on 2007-12-22
I don't know about changing the username of a user.  You could create a new user, add them to the admin group and then copy everything in your home directory (including the hidden files) into the new user's home directory.

The comand for changing a password is **passwd**.  Use it like this: ```
passwd joebloggs
```
