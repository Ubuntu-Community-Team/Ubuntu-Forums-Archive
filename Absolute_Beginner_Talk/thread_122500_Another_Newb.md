Hi all,
Been lurking here off & on for the past few weeks & finally decided to register & start posting some questions.  I'm new to Linux & have tried out several different flavors over the past month before settling on Ubuntu for its slick interface & easy ability to add new applications.  Not to mention, of course, the great forum you guys have here :)

I'm running XP on my main computer & will probably continue to run Windows indefinitely as my 'main' OS.  But I recently transferred my XP license I had on my secondary computer over to a PC I built for my mom.  Rather than spend the $80 on a new XP license, I figured I'd take the opportunity to learn Linux, which is something I had been wanting to do for a long time.  I currently have Ubuntu installed on the secondary PC, which just for reference has a 2.8 GHz Celeron D processor, a pretty basic PC Chips mobo, a 60 GB hard drive and 512MB of DDR400 RAM.  Ubuntu seems to run very, very quickly on it.

I'll post my questions in separate threads, but just as a sort of intro, here is what I am looking to do with Ubuntu on that secondary PC:
-Web Browsing (got that figured out already :) )
-Email (haven't decided which program to use yet)
-Share files back & forth with my main PC running XP (sort of got this set up but hit a snag)
-Run pretty much every type of media on it: DVDs, mp3s, video files of just about every flavor, including Divx & Xvid.  And play them from my main (XP) PC
-Listen to Shoutcast/other internet radio (fumbled my way through doing this but I'm sure there is probably a better way)
-At least figure out how to set up a wifi connection with the card I have installed (not a priority since I ended up running cat5e cable back to that computer in desperation)
-Watch TV using the Leadtek card I have installed (utterly unsuccessful thus far-- I'm guessing it's a driver issue)

If I can do all of that, I'll forget about getting XP for that secondary computer & will probably be a Linux user of some sort for life.  Also if all goes well, further on down the line I'd like to:
-Set up the secondary PC as a media server to host all my music & video files
-Use it to back up files from my main PC
-Maybe, if I gather up enough courage, turn it into a MythTV or Freevo box.


I have to admit that my first experience using Linux was a rude shock.  I wasn't really prepared the command line stuff, and was spoiled by how easy it is to install/use hardware on XP (yes, I know it's not Linux's fault that hardware manufacturers aren't putting out drivers for Linux, but the fact remains that XP is currently more hardware-friendly).
However, I'm up to the challenge-- I gained some familiarity with UNIX commands in college, and although I can be a bit thick at times when exposed to something new, am able to catch up quickly once the shock wears off.

Like I said, I'll post my questions in separate threads.  If anyone has any thoughts so far, though, I'd welcome any input.

Regards,
