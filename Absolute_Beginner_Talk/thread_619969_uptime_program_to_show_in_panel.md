Check out the Cafe for an ongoing thread of conky screen shots and config files.

### Post by FuturePilot on 2007-11-22
First you'll need to set up a .conkyrc file so that you can tell it what you want it to display. Check this thread for some ideas
[http://ubuntuforums.org/showthread.php?t=281865]("http://ubuntuforums.org/showthread.php?t=281865")
Then to run it, press Alt+F2 and enter
```
conky
```
