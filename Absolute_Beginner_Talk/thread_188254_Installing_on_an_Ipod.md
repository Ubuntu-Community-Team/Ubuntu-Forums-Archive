### Post by CompShrink on 2006-06-03
Installing Ubuntu onto it wouldn't let it work as an ipod with itunes songs. Sorry I don't know how to fix that issue, probably have better luck on an ipod/mac forum.

As for being able to install onto the ipod, it is possible, if the bios in your computer is able to boot off USB, but I wouldn't suggest it, as the Nano doesn't have a hard drive, but flash memory. Flash memory can be read from a whole lot of times, but it can't be written to all that much. A few million times only. That sounds like a lot, but you'll be writting files to your ubuntu partition several times a minute... you get to a million pretty fast that way, and then you have a completely dead usesless nano.

What are you using to partition? It sounds like 44gb is the windows partition, because if you only have 13gb free space on a 60 gb partition, that means windows takes up about 44gb of space... so you ahve the numbers backwards. You can make the windows partition 52gb, which should leave 4gb for ubuntu, and 1gb to make a partition to share with windows.

If you want help going through with it that way, post back here.

### Post by nalmeth on 2006-06-03
Yeah, I can't make any kind of claim to knowledge with your ipod idea. It actually sounds cool though.

Do you have a DVD-burner? If so you could back up the personal data from windows, and assuming you have a windows install CD, you could install ubuntu on the HD. Usually the cause of data loss is a rushed install, accidentally picking the wrong drive to format, or shrinking a *heavily* fragmented window's partition.

If you are considering the HD install, make sure you defrag window's a few times, and back up the personal data.

Another alternative would be to pick up an extra harddrive. They are getting pretty cheap nowaday's.
