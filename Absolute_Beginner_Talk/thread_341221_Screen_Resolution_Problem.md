### Post by crooked on 2007-01-18
Hi there. I am new to Ubuntu and have a problem with my resolution. The strange thing is that this problem only arose after I reinstalled Ubuntu, after a few days of experimenting with my first install. The first time I installed it, my resolution was automatically selected.

I am running Edgy 6.10.


I have a 3D PROPHET 7000 (powered by [COLOR="Red"]ATI RADEON 7000[/COLOR]) and my TV/monitor has a resolution of **[COLOR="Red"]1280x768[/COLOR]**.

The only options in Preferences > Screen Resolution are 1024x768, 800x600, 640x480 and 832x624.


To clarify, i have an [COLOR="Red"]**ATI RADEON 7000**[/COLOR] and i want a resolution of **[COLOR="Red"]1280x768[/COLOR]**.


Thanks in advance for any help you might offer.

### Post by yoopernate on 2007-01-18
You will probably have to manualy enter the resolution that you want into your xorg.conf file.  Just pull up a terminal, then, sudo gedit /etc/X11/xorg.conf  Read through that file and you should find a spot where it lists all the resolutions you can get, and manually add in the one you want. Just make sure that the one you put in is formated the same as the ones that are automatically in.

### Post by floke on 2007-01-18
You could try 'sudo dpkg-reconfigure -phigh xserver-xorg'

This will allow you to redo the display settings - use the space bar to select the highest resolution you can have. See if that works.

If not, please post your xorg.conf file output here so we can see what it looks like - maybe, eg, it is not picking up your monitor for some reason, and giving it 'generic monitor' status.

Good luck
