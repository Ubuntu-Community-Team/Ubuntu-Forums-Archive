### Post by SlappyPappy on 2008-04-13
Definitely download and burn a LiveCD and see if you can get that to work by booting from the CD. That alone will give you plenty to do for a bit. 

If that goes well you can then consider installing but before you go about doing something like that you should read about all the install issues other people are having. To do this you can search in this forum using keywords about your system such as motherboard, sound card, video card, etc. If you don't see a lot of problems then more then likely you'll be able to install with little hassle.

YMMV of course but it's a good way to start! Best of luck mate.  :)
