Hi there and welcome!

You can't just copy the iso to a USB stick - however you can set up a stick drive with Ubuntu so that it can be installed on a computer, the only snag is that the computer must be able to boot from the stick drive.

This is the guide I used: [http://www.pendrivelinux.com/2007/09/28/usb-ubuntu-710-gutsy-gibbon-install/](http://www.pendrivelinux.com/2007/09/28/usb-ubuntu-710-gutsy-gibbon-install/)

You should check the min specs for installing Ubuntu - if you have less RAM say, then perhaps try xubuntu.
[http://www.ubuntu.com/products/WhatIsUbuntu/desktopedition](http://www.ubuntu.com/products/WhatIsUbuntu/desktopedition)
