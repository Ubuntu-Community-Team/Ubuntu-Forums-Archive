### Post by bodycoach2 on 2007-08-21
OpenOffice.org Impress can save presentations in the .ppt format so that Microsoft Office Powerpoint can show them. Alternatively, you can purchase a USB Tumbdrive/JumpDrive, and install the [OpenOffice.org portable apps version]("http://portableapps.com/apps/office/openoffice_portable"), and present .odp files directly from that.
[http://portableapps.com/]("http://portableapps.com")

There is more information on OpenOffice.org tutorial sites:
[http://www.tutorialsforopenoffice.org/category_index/presentation.html](http://www.tutorialsforopenoffice.org/category_index/presentation.html)
[http://www.learnopenoffice.org/tutorials.htm](http://www.learnopenoffice.org/tutorials.htm)

Good choice to stick with OpenOffice.org

You might try doing a presentation for a class on how to use OpenOffice.org, and convert files to MS formats.

### Post by p_quarles on 2007-08-21
You can get the fonts this way:
```
sudo apt-get install install msttcorefonts
```
You can also get the free MS PowerPoint Viewer for Linux. It's in add/remove programs, or:
```
sudo apt-get install pptview
```
That will let you test compatability before giving the presentation.
