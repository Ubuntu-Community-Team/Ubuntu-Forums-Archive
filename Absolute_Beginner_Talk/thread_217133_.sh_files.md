### Post by rdd on 2006-07-16
.sh are shell scripts. To be able to run them they have to have the execute-permission set. In nautilus you can right-click the file and enable the permissions. 

For details on the relevant shell command see [https://help.ubuntu.com/community/FilePermissions](https://help.ubuntu.com/community/FilePermissions)

Regards
