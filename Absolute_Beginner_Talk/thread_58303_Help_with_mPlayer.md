### Post by EnderTheThird on 2005-08-19
Just use the Synaptic Package Installer.  You can use the search button along the top and just search for "mplayer" and you should find it realitively easily.  You will need to install the codecs (searching for "w32 codec" or something should work, I think) as well, but both jobs are easy using Synaptic.  When you mark those packages for installation, it might tell you that it needs to install others as well, so just let it and you'll be fine.

If any of this is wrong, then someone please correct me.  I'm new to Ubuntu as well, but I figured I'd tell you what I did and hopefully that would help you out or at least give you a good start.
