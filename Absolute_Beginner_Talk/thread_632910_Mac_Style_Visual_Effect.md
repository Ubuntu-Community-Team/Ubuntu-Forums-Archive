### Post by pointone on 2007-12-06
It's called the Scale plugin for Compiz Fusion, and it should enabled by default in Gutsy. Traditionally, it was activated by moving the mouse to the top right corner of the screen, but mouse gestures aren't setup in Gutsy. The key combination still exists, though: try Shift+Alt+Up, IIRC.
