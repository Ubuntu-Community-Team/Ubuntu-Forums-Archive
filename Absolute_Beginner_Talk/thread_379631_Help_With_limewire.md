### Post by Blutack on 2007-03-08
Heya there, the reason limewire ain't working is that ubuntu by default comes with a GPL'd version of Java rather than the proper sun one.  To get it working make sure you have the universe and multiverse repositories enabled and then using synaptic search for java.  You want sun-java6-jre.  Just one caveat; the installation will appear to freeze.  What you need to do is click the more details tab, and in the window a license will be visible.  Use tab and enter to accept the license and all should be good :-)
