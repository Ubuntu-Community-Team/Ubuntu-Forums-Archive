### Post by candtalan on 2007-07-17
I have a number of similar machines running ubuntu and unless you want blazing speed, ubuntu will do fine - for a  simple life anyway. Ideally a bit more ram memory would be an advantage - it is cheap now if you can find it.

With your ram you would do best  to install using the text installer - the alternate cd.
Will you want to keep the win98 running or delete it all?

An easy option is to add a spare HD as slave and install onto that, or to get the installer to resize the win98 partition for you (back up data first any of course).

If you want faster response then use xubuntu which has a lighter windows manager and environment.

When installing, connect to the internet first, so it all gets configured automatically (ideally wired lan?)

I even put the heavier kubuntu onto your type machine, but I am ok with the slower response. I like the facilities and convenience kde offers. but more ram would be useful for you.

Apps like openoffice will be slow to start with small ram. If you later install xubuntu-desktop package then you can log into an xfce sessiion and it will be a faster lighter choice.


good luck.
