### Post by tgalati4 on 2008-03-25
Try evince or acroread and install a printer that accepts n-up formatting (like an HP officejet).  Then go through the print motions and select Print-to-PDF and save the file with a new name.

Bring the new file up in evince or acroread to see the n-up document.
