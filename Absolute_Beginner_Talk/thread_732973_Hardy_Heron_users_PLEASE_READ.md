### Post by PmDematagoda on 2008-03-23
_**Why was this sticky made?**_
There are many threads being started in the normal support sections that concern problems with Ubuntu Hardy Heron, while this is understandable with Ubuntu Hardy Heron currently in beta, this is not really acceptable since the normal support sections are meant for people who use current stable releases such as Ubuntu 7.10 and less.

_**Where can we start our threads on Ubuntu Hardy Heron troubles?**_
If you wish to start a thread concerning a problem that is _**directly related**_ to Ubuntu Hardy Heron then please do so in the [Hardy Heron Development section]("http://ubuntuforums.org/forumdisplay.php?f=305"). 

If you want to start a thread concerning a problem that is not directly related to Ubuntu Hardy Heron such as how to install it or how to install programs on it then you may do so in the appropriate section.

_**When will we able to resume starting threads on Ubuntu Hardy Heron troubles in the normal support sections?**_
After Ubuntu Hardy Heron is officially released on the 24th of April this year the Ubuntu Hardy Heron Development section will be closed and you can once again resume starting threads concerning problems with Ubuntu Hardy Heron in the normal support sections in the usual manner.
