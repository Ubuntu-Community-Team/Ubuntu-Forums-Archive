### Post by LowSky on 2008-01-31
Minimal specs that I got it to run on

Pentium 3 at 700 mhz
256 RAM
4GB free HD space

If you want compiz then you will probably want a Nvidia graphics card, but I hear that onboard graphics can do some compiz stuff.

Compiz does com with alot pre installed you will need to download the cssm to control those setting, but its very easy using synaptic to get what you need.
