### Post by zubrug on 2007-03-09
sorry if I did not understand you correctly,
Turn off your computer, start it with the live cd in your drive, you will get a window asking if you want to run/install, click yes, (it will just run off the cd and load onto your hard disk) 
You should then arrive to the ubuntu desktop that will have an install icon, double click and follow the instructions.
Make sure that you defragment your hard drive using windows first, (helps negate partitioning problems), that might take some time but is an important step.

### Post by HereInOz on 2007-03-09
So are you looking to install a 2nd drive in your machine so that you can put Ubuntu on it, dual booting with Windows on the first drive, or what is it that you want to do.

If you are installing Ubuntu on to a second drive in a machine that is already running Windows on the first one, just pay attention through the installation process, and you will be given the options as to where you want to install Ubuntu, and the second HDD should be one of those options, provided it is physically installed properly.
