### Post by Xian on 2006-03-18
You can just include both. When you upgrade your system the binary pkgs will be installed by default. The only time you'd need the source security pkgs would be if you were going to build them for yourself locally. Most people have no interest in such a thing. But even if you have the sources enabled in preferences, they would never be downloaded in a typical upgrade session.
