I take it you are referring to the iso image download for installing ubuntu?

If so, redownloading the cd image will not get around the issue of updating your software once you have installed ubuntu. The cd images are not constantly being updated with latest drivers and software as far as I am aware, otherwise we would have versions-within-versions being released (for example a 6.10b or 6.10c version of edgy)
