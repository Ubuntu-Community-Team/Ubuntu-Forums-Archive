### Post by Her Ghost on 2007-08-29
like davidjmayo said, above:

Kmenu==>System Settings==>Keyboard&mouse

choose Keyboard Shortcuts from the left
on the top tab, choose Command Shortcuts

search for the applications that you want to run on command
select the application you want a shortcut to
at the foot of the window, click Custom
in the dialog box, perform the shortcut (i.e. Win+E)

I have got Konsole set to Win+R and Konqueror set to Win+E to mimic some of what I'm used to on windows.

have a look around in there, there are loads of options.

(couldn't find the show desktop command though - but I'm sure it's possible)
