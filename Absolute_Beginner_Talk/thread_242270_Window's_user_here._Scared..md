### Post by Tomosaur on 2006-08-23
By the way - Windows games don't always work on Linux. To get anything resembling functionality from them, you will need to use a program like Wine or Cedega. I reccommend you just keep your Windows installation alongside Ubuntu so you can run games how you normally would. You will almost definately not enjoy them so much under Linux, since they're just not made for it. There are however, plenty of free games available for Linux, and there seems to be growing Linux support from developers, so it may not be long before commercial games become natively supported. For now though, just keep Windows if you're bothered about gaming, and try out the Ubuntu Live CD.
