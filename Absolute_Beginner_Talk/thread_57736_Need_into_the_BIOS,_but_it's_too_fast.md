### Post by crispingatiesa on 2005-08-17
try <DEL> firts if it doesn't work try <f2> if it doesnt work try <Crtl><alt><Esc> if it doesn't work you have a very weird BIOS indeed.  Get the info of the BIOS (mainboard manual should tell you what combinations of letters to hit). If you don't have the manual find out what is your Mobo and go to the manufacturer web site a seek for the manual
