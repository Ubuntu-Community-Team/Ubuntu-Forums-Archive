### Post by avik on 2007-07-04
ubuntu-desktop is indeed a metapackage, which means that installing it will install all the applications associated with it.  While it will have almost no effect, it's best to keep this package around in case you want to get rid of all those applications (for example if you installed kubuntu-desktop and no longer want GNOME et all).  There's no harm in keeping GAIM.
