### Post by damianubuntu on 2006-08-13
As far as I understand it, and I'm no expert, linux is the operating system, the bit that make the computer work.

in order for you to use linux with pretty windows and a mouse instead of typing everything onto a black and white screen you use a graphical manager.  This acts as a bridge between you and the operating system.

The default graphical manager is Gnome in Ubuntu.  Kubuntu uses KDE (K desktop Environment) Xubuntu uses Xfce (I think), other ones are also available.

In a post here somewhere someone described them as different types of trim on the same car.  I thought that was a good analogy.  As to which one you use its just a question of personal choice.  I use Xubuntu because its better on lower spec older machines.  If you're starting out I would suggest you stick with Gnome, the standard Ubuntu setup, it just makes things easier whilst you're trying to decipher instructions and help files!

Good Luck
