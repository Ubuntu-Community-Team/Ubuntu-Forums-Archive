### Post by starcraft.man on 2007-06-13
If you want to learn command line in general, [linux command]("http://www.linuxcommand.org/") is a good beginning place. Has scripting lessons too. I don't think you'll find that many dedicated server tutorials, they will likely be labelled as simply CLI/Terminal guides/tutorials. Start with learning the shell and the shell script page. Then when you want to learn all the commands go to the super man section it is more complete.
