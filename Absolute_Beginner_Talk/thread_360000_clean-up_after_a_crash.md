### Post by nickburns on 2007-02-12
I have had similar problems, I think that it is all related to wireless network setup.

If you go to System >> Administration >> Networking

Set up you wireless options and then save the settings.  
ie Type in ESSID, password... 
then click the save button and save it as 'HOME' or something.

Then when you come up you may see a difference.  I have two setup - one for work, one for home.  If I leave it on the wrong one, I have noticed it takes a while to to change from one to another, it seems to hang, sometime for several minutes.
