### Post by torgrot on 2007-11-20
Would you copy the output of sudo fdisk -l thats a lower case L not a one.  This just to make sure that Windows is on hd0,1.  Look for the fat32 partition in the listing.  That is the one you want to use in menu.lst.  If it is on the first partition use hd0,0 the second hd0,1 or the third hd0,2,  I suspect you created a Linux swap partition and that it is at hd0,1.  You could just try the hd0,2 like this root (hd0,2)
