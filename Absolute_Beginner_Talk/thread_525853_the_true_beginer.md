### Post by sumguy231 on 2007-08-14
If you want to run a web site, I'd strongly suggest getting professional hosting just because it's very likely against the TOS of your ISP to run a web server on a standard home connection, and they might get suspicious if you get actual traffic. I'm not saying it's not ok to set up a web server, just don't encourage people to visit it or expect it to hold a lot of traffic.

Also, if your computer matches the specs listed on that page, it is pretty low though I have installed Apache+Friends on an old PII box, which gets next to no server load put on it and does ok. But the keyword there is that it gets next to no load put on it.
