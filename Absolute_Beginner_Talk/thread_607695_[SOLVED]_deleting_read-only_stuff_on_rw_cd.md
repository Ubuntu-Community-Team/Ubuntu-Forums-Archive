### Post by amingv on 2007-11-09
Hi,

The exact process pretty much depends on the burning software you use, but you will  find some option labeled as "Erase Disc", that is what you must use.
You can not delete a file from a CD-RW using the file browser, because the cd actually needs to be "burnt" in order to remove the files. Keep in mind that you won't be able to remove just one file, but the whole content of the disk.

[More info on this.]("http://en.wikipedia.org/wiki/CD-RW")
