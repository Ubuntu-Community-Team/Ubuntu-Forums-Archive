### Post by forrestcupp on 2008-03-26
Go to System->Preferences->Advanced Desktop Effects and click on the General Options.  In the 'Desktop Size' tab, make sure the 'Horizontal virtual size' is set to 4, then click Back in the lower left.

Make sure the 'Desktop Cube' and 'Rotate Cube' effects are checked.  Then the way you use it is by pressing ctrl+alt and dragging your mouse with the left button.

If you can't find Advanced Desktop Effects, type in a terminal
```
sudo apt-get install compizconfig-settings-manager
```
