### Post by starcraft.man on 2007-09-11
A Microsoft router? I never heard of them making routers. Do you mind taking a second look please... Assuming your network is ethernet based, plug a line from the router to your Ubuntu machine and if your using DHCP it should automatically configure. If it's wireless get the maker and model and have a look round the forums, it's a bit different for each and you may need to use NDISrapper.

As for sharing, go System > Administration > Shared Folders, then install the Samba parts needed. After that you'll see Network in the Places menu, you can  browse to any Windows networked machine.
