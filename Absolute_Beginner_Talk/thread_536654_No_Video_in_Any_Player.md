### Post by Jimmyfj on 2007-08-28
You'll have to install codecs for the videos to play correctly. You'll find the codecs you need in either Add/Remove or Synaptic. Try installing VLC player and see if that plays your video files after installing the missing extras. From a terminal you could also try this:

```
sudo apt-get install ubuntu-restricted-extras
```
