### Post by dreblen on 2008-01-21
You need to install XP first because I'm pretty sure that windows won't recognize your ubuntu installation and it would overwrite it.
As for the size of the partitions, that depends on what you want :) If you plan on using ubuntu more than windows, then you might want it to be bigger. On a 200 gb hard drive, I'd split it at about 100 gb for each, possibly more for ubuntu because of the swap partition.
Hope this helps...

### Post by mikewhatever on 2008-01-21
Install Windows first, then Ubuntu. I'd suggest about 20 - 50 GB for Windows, about the same for Ubuntu + 1 GB for swap. The rest would be a storage partition. To facilitate the process, please check out these links:
