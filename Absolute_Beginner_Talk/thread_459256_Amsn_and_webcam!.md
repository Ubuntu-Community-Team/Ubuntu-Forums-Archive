### Post by flossgeek on 2007-05-30
To view anothers webcam in amsn in the chat dialogue window you should see a webcam icon in the top right hand corner (providing the person you are speaking to has a webcam connected, the icon only appears if other user has webcam connected). Click this and you will be able to view another users webcam.

So people can see you i.e you invite a user, in the dialogue chat box click "Actions>Send Webcam". 

Hope this helps
