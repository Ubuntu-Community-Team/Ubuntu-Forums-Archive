### Post by viciouslime on 2008-04-02
DVD:Rip does do this, but it's a little bit more complicated to use. If you just want to rip to video files, you might like to try thoggen. It is in the repos. This will rip to the free ogg theora format. If you want something that will rip to other formats, k9copy is very simple to use. It is a KDE app, but it'll run fine under gnome. k9copy will also shrink dvds for you so that you can opy dual layer discs to single layer dvdrs for backup purposes :D
