### Post by taurus on 2006-02-22
You don't have to use the export mode to install Ubuntu.  Just hit return key at the prompt and when it gets to the partition stage, pick the export and tell it where to install it, leaving your Windows alone.  Then, make sure you install GRUB on MBR so you can boot both Ubuntu and Windows later...
