### Post by 5-HT on 2007-08-11
Live/Install CD installations can be really finicky with the partition phase. I'd suggest partitioning the drive prior to doing the install. I don't use Mint so I'm not sure if they have an alternate install CD, but the Ubuntu alternate cd always comes through for me where the live install one failed. As with any install--please be extra cautious about what paritions you're formating. You do have the option to create a mount point for a partition during the install without formatting (for an existing data parition, etc..)
