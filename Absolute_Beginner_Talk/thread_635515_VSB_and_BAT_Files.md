### Post by Espreon on 2007-12-08
UM no you cannot run batch files on Linux.
On Linux we have shell scripts. You can write them like you write batch files, except shell scripts have the file extension ".sh" (w/o quotes).

You will have to study Linux command line tools for more complex scripts but Linux has basics like cd, echo and so on.
