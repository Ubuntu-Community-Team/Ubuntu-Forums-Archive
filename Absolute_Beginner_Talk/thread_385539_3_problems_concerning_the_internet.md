### Post by in_flu_ence on 2007-03-15
While Firefox2 can be installed in dapper, there are people who said that it is not that stable. Therefore, my advice would be to upgrade it to 6.10 edgy. It is highly stable and I have been running it with firefox2 and etc in both 32bit and 64bit environment with no problem.

For java and flash, I use automatix ([www.getautomatix.com)](www.getautomatix.com)). I think there are people who prefer easyubuntu since automatix was said to install stuff more forcifully.

With this two tools, you can install alot more stuff like codec and etc.

Hope it helps
