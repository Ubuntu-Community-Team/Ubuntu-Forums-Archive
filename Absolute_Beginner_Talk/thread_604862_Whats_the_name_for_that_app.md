It sounds like you're thinking of [Automatix]("http://www.getautomatix.com/") (or maybe [Easy Ubuntu]("http://easyubuntu.freecontrib.org/")).

Note that although Automatix was quite popular a few years ago, it's use in Ubuntu is now discouraged because it makes non-standard changes (which sometimes cause things to 'break' if you ever upgrade). In the modern versions of Ubuntu, you get a convenient popup the first time you try to play an MP3, for instance, and it will very conveniently install the codecs you need.

So, you can use Automatix if you want, but it's now quite easy to add all the important things in the "proper way."

I hope that helps.
