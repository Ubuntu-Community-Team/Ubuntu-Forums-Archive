### Post by vicky.irobot on 2007-11-26
I can answer one section of your question

>whether ATI / Nvidia?

I guess the question should be reframed as to which chipset in ATI or Nvidia since you don't have stable drivers for all the chips.

Initially the thing was that you had linux drivers only available from ATI...(anyone plz correct me if I am wrong here) thatz atleast when I had a ATI AGP card and nvidia onboard graphics. The ATI Linux driver was very easy to install and worked without any issues.

Recently which might be a year or close to that, Nvidia has started efforts of pushing their drivers for Linux and they even have a neat nvidia forum setup where you can ask questions for any issues that you are seeing after installing their drivers. One sad part of Nvidia drivers is that you need to do some fixing work after installing their drivers, atleast I never had a single shot working driver. I had to install and fix a few things in their config file for the driver to start working. This was when I was using openSuSE 10.3

With Ubuntu after installation it asked me if I wanted to use Nvidia driver and it downloaded and installed the driver and everything was working neat after reboot.

But if you ask me I would suggest you to go by ATI for Linux box.
