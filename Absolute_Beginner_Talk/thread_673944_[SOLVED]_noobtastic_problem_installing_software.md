### Post by MikeEvans on 2008-01-21
Welcome to Linux bjameswilliams.  You have allot to learn but its going to be fun an beneficial.  First I would agree with hyper_ch.  Ubuntu probably has a free program that will do what you want in Add/Remove programs, so first look there.  If you can't find a suitable solution there you can search online for a linux program thats not on that list, but you will need to read up on installing programs in Linux to take that route.

Next, if you must run your windows App in linux you need to use a program called wine.  Wine works perfectly on many windows programs but for some there are bugs.  You can see if others got your program to work in wine by checking the [database.]("http://appdb.winehq.org/"). To use wine right click on your program (the .exe file) and choose "Open with wine". You can also use wine from the terminal:
```
wine *application.exe*
```

Dual booting and virtualization are also options as hyper_ch suggested.
