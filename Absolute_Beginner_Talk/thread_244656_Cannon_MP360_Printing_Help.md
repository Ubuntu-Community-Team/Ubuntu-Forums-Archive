### Post by coolclassic on 2006-08-26
As i said in my post the printer heads may need cleaning this is normally done through the software supplied by cannon but as cannon do not supply software for linux you have to use third party software such as turboprint. The task of cleaning and aligning print heads will sometimes have to be done if the printer has not been used for a long time or left switched on.
