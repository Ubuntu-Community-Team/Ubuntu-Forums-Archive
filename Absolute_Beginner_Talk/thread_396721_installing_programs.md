### Post by bodhi.zazen on 2007-03-29
Plug 'em in, turn up volume, enjoy.

If you do not have surround sound I advise alsa mixer.

In a terminal type : ```
alsamixer
```

If that is not the correct sound card, you may need ```
alsamixer -c 1
```Or -c 2, ...

Use the arrow keys to go to front/side speakers

use up arrow keys to turn up volume

Exc to exit alsamixer

### Post by tonyr1988 on 2007-03-29
Debian will show up as a menu option in your Applications menu. Right-click Applications and click Edit Menus. In the middle, do you see "Debian"? Check the box next to it to show it in your menu. After you Close it, you may have to reboot again for it to show up. The Debian menu simply shows more of the programs (not all of them show up in the menu by default).
