### Post by ilya123 on 2007-01-18
You should install mysql in the Synaptic package manager. Once installed, run "sudo mysql" to connect to the server. The rest (creating DB, tables, etc) - check docs at mysql.com
Perl process is probably some utility running in background (like file indexing). Nevermind. If you want to play with the language you start the Perl in the terminal. Another language available by default is Python, which I would recommend. For both, just go to Google to look for tutorials.
