### Post by Hospadar on 2007-08-07
Try going to System->Administration->Users&Groups you can change passwords and usernames from here.

If you don't want them to be able to have root priviladges (aka use the "sudo" command, install software, etc) you'll need to also remove them from the "admin" group by going to "manage groups" in that same window.
