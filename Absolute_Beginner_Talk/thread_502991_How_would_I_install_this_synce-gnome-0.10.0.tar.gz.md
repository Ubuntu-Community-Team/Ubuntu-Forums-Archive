### Post by Outrunner on 2007-07-17
Extract it with

```
tar xzvf synce-gnome-0.10.0.tar.gz
```

then navigate to the extracted direcory :

```
cd synce-gnome-0.10.0.
```

run the configure script

```
./configure
```
```

make
```

and finally

```
sudo make install
```

EDIT: By the way, if you get any errors, post them here.
