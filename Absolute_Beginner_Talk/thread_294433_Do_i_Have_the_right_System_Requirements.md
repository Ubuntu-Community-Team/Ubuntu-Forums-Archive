### Post by Fully216 on 2006-11-06
I think your system will be just fine.  You can check out the system requirements at sites like

[http://ftp.ubuntulinux.org/ubuntu/dists/warty/main/installer-i386/current/doc/manual/en/ch02.html](http://ftp.ubuntulinux.org/ubuntu/dists/warty/main/installer-i386/current/doc/manual/en/ch02.html)

I would say 700mhz processor with about 128mb ram will guarentee a system that runs well.  Wikipedia states that Ubuntu requires 256 megabytes of RAM, and, when installed to the hard disk, needs three gigabytes of hard-disk space (as also stated in the 6.06 release notes).  

You can get it to work on a lot lower machines like yours, but for that I suggest lighter environments like XFCE or IceWM or Fluxbox.  If not, there is always minimalist linux, athough I am a big ubuntu fan and would try it first.

Strange that the CDs did not work.  Did you apply a md5 sum check?  They should be okay when you run for the live CD.
