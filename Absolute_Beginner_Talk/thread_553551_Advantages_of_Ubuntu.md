### Post by aysiu on 2007-09-17
Think of a *package* as a software application. A *package manager*, then, is an application that manages the installation and removal of software applications.

Unlike Windows' Add/Remove Programs, which is really (for all practical purposes) only for removing applications, Ubuntu's package manager is used to add applications, remove applications, and update applications.

Read more here:
[http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)
