### Post by mramsey on 2007-10-06
Cable connection should just plug in assuming you are using a router.  If you have a DSL modem you may need some additional config. I did one today for a friend, it was a DSL modem and a PPPoe internet connection. for this I used - ```
pppoeconf
``` In the terminal.  That is where to set the username ans password for the account. Good Luck!:)
