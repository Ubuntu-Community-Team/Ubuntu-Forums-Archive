### Post by medley on 2008-04-15
> **AeThEr777 said:**
> I have compiz running and i only have a 2 sided 2d virtual desktop. How do i get the full cube!

In compiz settings manager, go to the general settings, select the desktop size, and set virtual desktops to 1, horizontal size to 4 and vertical size to 1.

That'll do it.
