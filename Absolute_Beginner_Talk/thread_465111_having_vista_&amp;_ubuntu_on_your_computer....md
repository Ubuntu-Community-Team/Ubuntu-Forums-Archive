### Post by goatflyer on 2007-06-05
You can install ubuntu as a second operating system, and you will have what is called a dual-boot system.  I would recommend this as the best way to learn to love ubuntu, while if you encounter any programs you'd like to run in windows, you still can by booting into windows.

You switch between the two by restarting the computer, a little text menu (called 'grub') offers you a choice of which OS to boot.  If you pick nothing, then after a brief time, it will boot to its configured default (ubuntu)
