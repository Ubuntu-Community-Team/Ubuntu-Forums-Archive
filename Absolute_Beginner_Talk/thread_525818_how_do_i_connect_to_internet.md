### Post by Rul on 2007-08-15
If you have high speed internet with broadband cable then it's an ethernet connection. In that case after plugging in the cable it should just work.

I use Time Warner cable to connect to the internet and it works out of the box in windows and in linux, however a couple of days ago my sisters came to visit me and they were unable to connect using that cable, I don't know why. So if you can't connect with ubuntu using that cable you may want to try with another computer and see if it works (or with windows if you are in a dual boot)

Finally if you want to connect to internet using the phone line (the narrow cable) then you will need to create an account in a similar fashion as if you were in windows. I have never done that but in kubuntu (KDE) there is a program under internet called KPPP and it says it's a tool to connect to internet using your phone, you should fine something similar in Ubuntu (GNOME)
