Well, since no one else has answered you, I'll have a crack...

First, vi?! That's pretty hard core if you're not used to it! You know you can use a GUI text editor instead (Applications->Accessories->Text Editor).

Next, if you're using the live CD you won't be able to mess your computer up by just running things, but I believe you can save and delete files on the hard drive if you choose (I'm not sure about this, though -  I'd don't know if the live CD gives you permissions to write to the Windows partition by default). So just don't delete anything, and running the Live CD shouldn't affect your computer in any way.

Which raises an interesting point - how can you install drivers when you're using a live CD? Surely each time you reboot you'd lose any changes you'd made?

As for things to do and read up on... I'm not really sure. Browsing these forums will let you see what other people are doing and may give you some ideas. What I did was occasionally find I needed some application, or some way of doing things, and tried to find out if it was possible. It invariably was, and by researching how to do it I learned.
