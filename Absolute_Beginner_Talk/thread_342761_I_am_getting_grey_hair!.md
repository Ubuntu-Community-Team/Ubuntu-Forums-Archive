### Post by christhemonkey on 2007-01-20
You can still install with the alternate install cd
(such as this: [http://ubuntu-releases.cs.umn.edu//6.06/ubuntu-6.06.1-alternate-i386.iso](http://ubuntu-releases.cs.umn.edu//6.06/ubuntu-6.06.1-alternate-i386.iso) )

Alternatively you can install with the alternate cd and then install minimal window manager such as fluxbox, but they can be less user friendly than the default environments (gnome, kde and xfce as opposed to fluxbox openbox fvwm etc) ((and that is just my personal opinion, i have and do use all of them))

I would install xubuntu using the alternate cd if i was you.

Im using default ubuntu on a couple of 800mhz machines with 128mb RAM (bit on the slow side and usually 2 apps max)
