### Post by Zack McCool on 2008-04-05
Linux is plenty good with games.  The problem is that the games you are running are written for Windows, not linux.  You can't expect an emulator to run as well as the native OS.

If you want to run a lot of Windows games, the best bet is to split the drive and dual boot.  A virtual machine is going to run worse than Wine/Cedega.  It won't have any 3d capabilites...
