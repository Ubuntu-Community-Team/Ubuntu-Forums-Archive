### Post by ramjet_1953 on 2007-06-09
Just use Synaptic Package Manager to install [COLOR="Blue"]gnome-art[/COLOR]

It is a front-end for the art.gnome.org website

There are heaps of options for login managers there. under [COLOR="Blue"]Other Themes>Login Manage[/COLOR]r.

When you first run this app in any category, it can take a while to download the database, but subsequently it just does an update check, which is quite fast.

I have noticed a bit of a bug with the program, though and sometimes it won't download a theme and you have to use your browser to go to the site and download it.
