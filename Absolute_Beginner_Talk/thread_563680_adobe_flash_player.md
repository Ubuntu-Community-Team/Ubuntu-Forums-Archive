### Post by ORi0N on 2007-09-30
To extract a tar.gz file, open a terminal and enter the following command:

```
$ tar -xzvvf filename.tar.gz
```

Note: The $ is the system prompt. You do not type it.

This will extract the contents in the current directory. Then you follow the instructions in the README file and you're ready...
