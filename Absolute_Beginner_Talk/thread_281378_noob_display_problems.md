### Post by CatKiller on 2006-10-21
It's possible that ```
sudo dpkg-reconfigure -phigh xserver-xorg
``` will successfully configure your X server setting in a live environment - I haven't tried it, personally - although you'll have to restart X (Ctrl-Alt-Backspace) for the settings to take effect, and you'll have to do it every time you reboot.

If it doesn't work, you'll have to wait till you've got it installed to configure it properly. Then you'll have the option of either this method or the one above.

Welcome to the community.

EDIT: When you're configuring X, it's probably the refresh rates that you'll be most interested in, rather than the resolution directly - look up your monitor's specs for that. The usual cause of only getting 640x480 is because the graphics card isn't reporting the monitor's capabilities properly, and so every mode counts as a fail.

### Post by FriscoSoxFan on 2006-10-21
The monitor was the problem, not the video driver. I had to add the following lines to the /etc/X11/xorg.conf file:

Section "Monitor"
	Identifier	"DELL P780"
	Option		"DPMS"
[B]	HorizSync	30-107
	VertRefresh	48-120
[/B]EndSection

Now I can actually browse the internet comfortably enough to look up help for the next challenge I face!
