### Post by etank on 2007-04-27
If you are running on a Windows you can use something like [InfraRecorder]("http://infrarecorder.sourceforge.net/") to burn the iso image to a CD. Burning the iso to the disk itself will not work. An iso is kind of like a zip file. The burning software will extract the contents of the file and create the disk properly.

You should also take a look at [http://doc.ubuntu.com/screencasts/Downloading_and_Burning_an_Ubuntu_ISO](http://doc.ubuntu.com/screencasts/Downloading_and_Burning_an_Ubuntu_ISO). That will show you how to do it.
