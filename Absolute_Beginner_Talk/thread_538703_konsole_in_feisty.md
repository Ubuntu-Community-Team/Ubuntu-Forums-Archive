### Post by sstucke on 2007-08-30
By default it is not installed.
You should install it by opening a terminal window (Applications -> Accesories -> Terminal).
Then run this command:
sudo apt-get install konsole
It will prompt for your password.
After that, you'll find the program in Applications -> System tools
Good luck!
