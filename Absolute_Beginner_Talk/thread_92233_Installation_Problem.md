### Post by moffa on 2005-11-19
You don't need a bootable floppy if the CD image was burned correctly.  If the cd is detected properly and the BIOS is set to boot from the CDROM before the HDD, a line should show up that says something to the effect of "Press any key to boot from CDROM."  If not, either you didn't setup the bios, or the cd wasn't burned properly.
