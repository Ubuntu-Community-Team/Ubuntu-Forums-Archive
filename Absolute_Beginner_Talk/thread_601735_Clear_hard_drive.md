### Post by 1337455 10534 on 2007-11-03
Hi stringer, 
Installing any Ubuntu has these loose steps:
1) pop in the CD
2) reboot, and boot to the CD
3) install
3a) Choose Lanuage & Prefs
3b) Formating and Partitioning options
4) import data
5) Install
 The install CD has an in built partitioner / formatter.
You can clean your disk using the Ubuntu CD itself.
