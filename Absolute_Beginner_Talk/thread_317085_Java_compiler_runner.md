### Post by nickburns on 2006-12-12
Lots of folks like netbeans, written by Sun.

[http://java.sun.com]("http://java.sun.com")

You can get it bundled with the JDK.

Download it, and go to a terminal and:

sudo chmod 777 theInstallFile.bin

then:

sudo ./theInstallFile.bin

and it will create a shortcut in Applications >> Development.
