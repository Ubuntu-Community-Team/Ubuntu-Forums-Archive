### Post by girard on 2007-03-30
I've only just recently used linux on my machine and ubuntu was my first choice. I have only tried ubuntu and opensuse actually, but i was quick to dismiss opensuse. so now i am back to ubuntu - after about 10 reinstallations.

Knowing about how linux OS's allow users a great amount of flexibility, i decided to use both the KDE and Gnome DE's. The problem is that I am torn between the two.

For me, Ubuntu has lived up to its advocacy. It's easy to use, simple and functional, but it lacks certain features in Kubuntu that causes me (at least) to want to have both DE's installed, which is aesthetics.

The problem with having both DE's installed is that you double up on the applications that you have, which basically has the same function (i.e. Konqueror and Foxfire, Adept and Synaptics, including the media applications. 

Personally, I've found this to be a bit... um... confusing.

A thing regarding window behavior also... I like how ubuntu enables users to hold on to a maximized window, drag it so it becomes smaller, and then maximizes again once you bring it back up. This really allows me to view the other applications running behind my web browser especially when downloading something. This feature is not available (i think) when I'm on Kubuntu. On the other hand, I like how KDE allows you to drag a window to another desktop. This makes it easier for you to compare and analyze data or whatever from different windows that you placed on different desktops to avoid cluttering. Again this is not available on Gnome.

You guys see why I'm torn? 

I'd sure like to have the simplicity of how Ubuntu functions and operates, but I'd also like to have the cool side of Kubuntu. And the needs and wants don't get satisfied at the same time because you can only log on to one at any given time.

Nevermind the applications, as you can basically use any on either... it just hit me... FORM and FUNCTION. Why can't we have both?
