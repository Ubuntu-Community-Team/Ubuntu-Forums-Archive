It's as hard as implementing the rest of protocol 14 for MSN. You can actually patch pidgin to send custom smileys, if you compile it from source, and receive offline messages if you enable msnp14 in the configure options. However that is still highly experimental, and hasn't passed to the main build because of a lot of bug chasing that still needs to be done. Lurk in developer.pidgin.im to take a peek at all this ;)
