### Post by Herman on 2006-04-04
I usually just leave that one alone and just press 'enter' and let it find the best resolution automatically. 
I think you are supposed to mark the resolutions you _do not want_ Ubuntu to try to use, but if you mark them all, Ubuntu will just pick the best one anyway.
If you are daul dooting, look in Windows in Control Panel, Display somewhere (I forget exactly), and see what resolution XP is set at if that's any help.
In Ubuntu you can go 'System', 'Preferences', 'Screen Resolution' and you should be able to change the settings without re-installing.
I hope this is helpful, :D (Herman).
