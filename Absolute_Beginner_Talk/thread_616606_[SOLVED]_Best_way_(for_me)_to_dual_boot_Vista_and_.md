Just finished my vista - ubuntu dual boot and figured I should mention that the shrink tool allows you to select in MB how big of  an empty space you would like to create (for the ubuntu install), in my case I choose 30720MB. The rest will remain on your current windows partition. So if you would like more or less space left for your vista partition you can easily select and customize how much space you would like allocated. I am sure this is the best method so it should definitely be the approach for you :)
