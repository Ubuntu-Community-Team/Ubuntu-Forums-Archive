### Post by esoterica on 2007-05-11
Over the years I've played with Linux just because you can't consider yourself a true computer user unless you have. Unfortunately I've never been able to take it seriously though and for getting actual work done I've always had to go back to Windows just because I could install it and be done with it. Linux instead always seemed to require endless hours devoted to just trying to get it to work out of the box, then when it came time to update you could give it up, errors about this library missing or being the wrong version, then the new version causing other once working apps to, blah blah blah.

Ubuntu seems to be on to something here though, I like the way you can run the OS right off the install CD to see it all works, then install it.

Linux has come along ways, the install is now a no brainer, and better still actually works, of the thousands of Windows and Linux installs I've done in my life, this Ubuntu install is without question the absolute easiest OS install I've ever seen. I thought the new Vista install was pretty easy, but this blew it away for simplicity and instantly being usable.

My only complaint is it still has that home made look about it. This seems to be the one area for some reason no Linux developers have seemed to figure out that people are still using Windows because it looks so much cleaner and professional. The "look" of Linux hasn't changed or been updated in at least 10 years now that I know of. If someone could now just update the overall look of the Operating System you'd have an absolute hand down winner that the masses would actually want to use.

I was very impressed with the installation ease finally, and the system actually working out of the box for a change, but disappointed in how it still looks with icons and graphics that just don't match up with each other at all and have that 4th grader or little kid look about all of them.

Hasn't anyone developed a single theme to address these problems with the low grade look of the OS to try and give it all a more professional look that people can be impressed by enough to take it seriously?

I have this Ubuntu laptop setting next to a Vista laptop and I'm trying to put my finger on just what it is about Vista that makes it look so much better and professional? All I can think is it's because all of the icons and everything else about Windows seems to go together, like wearing matching socks. Linux still has that look like I have a dress sock on one foot with a tube sock on the other.

Now that all the installation issues of Linux days gone by, I think the single distro that would take things over the top would be the one that could somehow come out with a complete look and feel about it, or at least a full packaged theme that updated how things looked and got everything fitting together better. I mean I'm running a brand new 2007 version of Linux right now, and by just setting back and looking at it I wouldn't be able to tel this wasn't the same Linux I may have installed and ran 10 years ago, the look and feel hasn't seemed to go anywhere over all these years.

I really wish I was a graphic artist, or at least had some degree of artistic talent, I'd try to come up with some kind of package that covered the entire look and feel issue, with the window look, some color coordination, and matching icons that looked good and followed a single theme or type all rolled up into a single package that seems modern and allows us to use the full potential of modern computers on the market we're buying right now. We'd then finally have a Linux people in the masses would want to use, right now it's like the money I paid for a high end graphics card and high resolution display seems wasted when it should be getting maxed out and used for it's full potential.

This out of the box working installation finally and the easy way to run updates and install software has certainly made it a keeper for me, a great big thanks to everyone who helped accomplish this with Ubuntu!
