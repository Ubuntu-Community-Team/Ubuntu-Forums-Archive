### Post by jleaker01z on 2008-01-30
Double click the file and extract the contents.  Open a terminal and change to the directory you extracted to.  (cd /home/uname/pathtofile)

Type in:

./configure

Then type

make

Finally,

sudo make install

### Post by Niniel on 2008-01-30
It's really easy - go to System/Preferences/Appearances and drag/drop the icon archive onto the window.
It should ask you if you want to use the pointers...
You can also pick an installed pointer scheme by opening that window and then clicking on Customize. Then you just have to go the the Pointer tab.
