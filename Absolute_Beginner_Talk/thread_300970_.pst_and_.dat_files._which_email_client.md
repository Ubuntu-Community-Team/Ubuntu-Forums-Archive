### Post by Scunizi on 2006-11-28
I've been playing around trying to get my outlook.pst file to import correctly into Thunderbird and everything works with one exception.  Catagories doesn't transfer.  What I've found that does work is to have a free Yahoo account and upgrade it to the latest web2.0 version.  Import the pst file into Yahoo's address book then export it from there in a single vCard file to your HD and then import it to Evolution.  Everything transfered correctly and in the right locations!  :D

Edit: One thing I just noticed is I can't edit any contacts. After trying the "OK/Save" button doesn't highlight.  I'll try rebooting and see what happens..... No need for a reboot.  I've found an irratating "feature" of Evolution.  If your imported contact only has a business name that is filed under "Company" in Evolution, to make changes you have to add the business name to the "Full Name" field before the program will allow any changes to the data on that contact. The import feature does place the company name in the "File under:" field.
