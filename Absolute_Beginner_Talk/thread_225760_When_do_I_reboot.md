### Post by taurus on 2006-07-30
If you upgrade your kernel, you need to reboot your machine to use the lastest one.  Otherwise, you will be running whatever the kernel you are using even if you upgrade to the latest one.  It goes the same for all the programs you are currently using.  For instance, if you have firefox open right now and you just upgrade it to the latest version, you are technically still using the old version until you close it and start it again...
