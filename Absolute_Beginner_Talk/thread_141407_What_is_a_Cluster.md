### Post by Zeroangel on 2006-03-08
A cluster is a setup of multiple computers all tasked to do one operation. Cluster usually is comprised of a master computer which directs the operations of other computers while they crunch away at code that the main computer tells them to crunch away at.

See [Beowulf Cluster](http://en.wikipedia.org/wiki/Beowulf_cluster)
