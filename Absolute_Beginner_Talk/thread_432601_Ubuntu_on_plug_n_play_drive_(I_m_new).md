### Post by PriceChild on 2007-05-04
I think you greatly underestimate what it is you are trying to do.

Ubuntu is NOT a windows program. It does not run inside windows. When you boot your computer... you will not load windows and then load ubuntu. Ubuntu is completely seperate.

It is "possible" to install on an external drive but if I were you I would defrag my main internal drive and then resize the partitions to make room. Booting linux off of a usb1.1 connection isn't going to be fun.

help on installing can be found here: [https://help.ubuntu.com/community/Installation](https://help.ubuntu.com/community/Installation)
