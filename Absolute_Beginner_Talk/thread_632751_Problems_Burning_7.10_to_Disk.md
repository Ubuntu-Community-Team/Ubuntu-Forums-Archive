### Post by Threbus5 on 2007-12-05
Hi,

Is the problem that the install fails or that burning ISO images stops at 90%?

If you are certain that you have a good ISO image (per the previous response to your post) then -

I recall that the Ubuntu download site has a link to a free ISO burner.  Try installing that.

While burning the image make sure to select to verify the burn and burn at the slowest rate.

Plan "B" order an Ubuntu install CD - but that supposedly takes a while to arrive. Plan "C" try a bookstore for a CD (I bought an Ubuntu book at Barnes & Noble that included an install CD for Dapper - not the newest release but seemed to work ok and the book seemed full of useful information.)

Take care.
