### Post by qamelian on 2006-09-15
Go to System > Administration > Login Window. Click on the Security tab, then select 'Enable Automatic Login' and identify the user to login automatically. I know some distros set this by default during intallation, but it's really a poor idea security-wise, even on a single user system.
