Hi and with the amount of memory of 192mb I would recommend the alternate cd. Of Xubuntu the system requirements are as follow

To run the Desktop CD (LiveCD + Install CD), you need 128 MB RAM to run or 192 MB RAM to install. The Alternate Install CD only requires you to have 64 MB RAM.
To install Xubuntu, you need 1.5 GB of free space on your hard disk.

And if I remember correctly that xp uses about 2.6gb to install so on the 6gb hard you may fit both OS on the system.
