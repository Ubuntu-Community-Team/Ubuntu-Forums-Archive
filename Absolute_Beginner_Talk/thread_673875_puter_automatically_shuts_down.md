### Post by Golem XIV on 2008-01-21
I had this problem a while ago on a computer that had a problem with the CPU fan.  The fan was choked with dust and turning slowly, the CPU would overheat and the BIOS would shut down the system to prevent damage to the CPU.

Since it is happening to all your OSs, I'd say you are having a hardware problem.  If possible, check the fans and the power supply.  Unplug everything that is not essential (or built into the motherboard) - LAN, wireless, floppies, CD drives, modems, sound cards, etc. - and try again.  If it&#347; a desktop try running with the cover off to improve cooling.

Good luck!
