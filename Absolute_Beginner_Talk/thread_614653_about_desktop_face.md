### Post by atomkarinca on 2007-11-17
if you're using feisty fawn (7.04) then it may not be started on boot. what you should do is system > preferences > sessions > add
name: whatever you like
command: compiz --replace

that should do the trick.

EDIT: i forgot, if you're using gutsy gibbon (7.10) then system > preferences > appearance > visual effects and select whichever you want.
