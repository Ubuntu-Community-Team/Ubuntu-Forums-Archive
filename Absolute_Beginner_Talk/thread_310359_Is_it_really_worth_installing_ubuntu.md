### Post by CatKiller on 2006-12-01
> **wersdaluv said:**
> Should I wait for the time when linux gets much better or should I really exert effort in trying to install linux now?

Currently you've only exerted effort in not using Windows properly ;)

Once you've got the cd booted up, you can try it out straight from the cd without affecting your Windows install - it will be slower because it's running from the cd, but it's fully functional. If you like it, you can choose to install it. If you don't, no one will mind, and you might find that you like Windows more because you've tried an alternative.

If you decide that you like it, you can always dual-boot between Windows and Ubuntu. The installer includes an option to make your Windows partition smaller, installing itself in the free space you create. It will give you a boot-up menu to choose which OS you want to use.
