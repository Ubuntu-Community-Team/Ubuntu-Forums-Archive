### Post by blackened on 2007-01-12
For me this was always the best way:

Click on the torrent link, it opens the "Open With/Save As" dialog. 
Choose "Open With", then in the drop-down list select "Other".  
This brings up the "Choose Helper Application" dialog.  From here, navigate to the Azureus startup script (if you installed from Automatix I think this defaults to /opt/azureus/azureus, otherwise you'll have to search for it.), hit "Open" and you're done.
From now on it will default to that when clicking a torrent link. So now all you should have to do is hit "Ok" in the Firefox dialog.

P.S. This seems long and drawn-out, but it is the most convenient solution I'll wager.
