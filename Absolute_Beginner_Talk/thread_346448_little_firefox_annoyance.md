### Post by jblebrun on 2007-01-25
> **drascus said:**
> When web surfing, when I press the backspace button instead of going back a page it does the same thing as if I hit pg up. I don't really understand why but I really don't like relying on the mouse to hit the back button. any ideas on whats causing this?

You can use Alt-LeftArrow to go back.

You can also re-map the Backspace key in the about:config dialog.

I think Backspace was removed as the back button because of issues with forms. I don't know how many times I'd start typing a long post, and then hit backspace only to find that the cursor had moved outside of the edit box, so the browser navigated back a page, causing me to lose everything I typed.
