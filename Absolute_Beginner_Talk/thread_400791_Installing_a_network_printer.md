### Post by Tilex on 2007-04-04
If the XP machine is connected to the same network as your Linux machine (I assume you're trying to access a printer that is connected to a XP machine, from a Linux machine), then try to go in Systems Settings -> Printers -> Add Printer/Class, and choose the "SMB Shared Printer (Windows)" option.  Follow the steps, and it should work!
