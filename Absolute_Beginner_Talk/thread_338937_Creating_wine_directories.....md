### Post by Kieranties on 2007-01-15
in linux directories can be hidden from view by adding "." to the start of the name.

If you open your home directory and then select the option to "show hidden files" (view menu i think) in nautilus, you can then see all the hidden directories.

the one you are looking for is ".wine"

Inside you will find a structure much like your normal windows machine.
