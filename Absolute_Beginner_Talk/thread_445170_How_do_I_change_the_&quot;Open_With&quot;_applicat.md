### Post by Metacarpal on 2007-05-15
Open Nautilus (Places menu > Home).  Browse to a folder containing the file type you wish to open with the new application.

Right-click the file's icon, select Properties.  There will be an "Open With" tab.  Under that tab, click Add.  You can then select the application, and it will be added as an "Open With" option for all files of that type.  In that same tab, you can also select the default Open application.
