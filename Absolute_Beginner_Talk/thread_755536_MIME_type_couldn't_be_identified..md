### Post by civillian on 2008-05-21
Dies it gove you the option to download the codecs for the unsupported mimetypes at all, because it could be an issue that is related to that, since ubuntu doesnt support some multimedia codecs out of the box to avoid copyright issues, which can be fixed with the addittion of a few extra packages. I have the following installed and all my audio files and video files play fine:

GStreamer ffmpeg video plugin
GStreamer extra plugins
GStreamer plugins for aac, xvid, mpeg2, faad
GStreamer plugins for mms, wavpack, quicktime, musepack
GStreamer Dirac video plugin

which can all be downloaded from the add/remove programs application.
