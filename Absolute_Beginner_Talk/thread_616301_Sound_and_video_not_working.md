### Post by banewman on 2007-11-18
You need to enable all the repositories in the synaptic package manager. Go to applications - system - admin - synaptic package manager - click it and enter your password - then click settings from the top menu - select repositories and select all boxes except source code. When you try to play a media file the prog will ask you to download the codec needed - that'll get the gstreamer plugins. For youtube you need the flash plugin for firefox .
