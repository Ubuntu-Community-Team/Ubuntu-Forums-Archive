### Post by muhkayoh on 2007-02-16
This is just a quick, drive-by response so you don't go completely unanswered.

For the time being at least, Stopmotion is effectively broken under Edgy as far as I can tell.  Luckily, there are a couple of alternatives.  One is a java program that does the same things that stopmotion does.  The program is called ImageJ and I was able to get it working for me, though I haven't used it in a while.  You could google for that and try it.  Alternatively, you could learn to use Mencoder via the command line.  It seems scary at first if you're new to command line (terminal) use, but it's actually quite easy and will do what you need.  You may even prefer this approach once you get used to it.  Google mencoder and read the manual for specific howto info.

Hope this helps.
