### Post by nanotube on 2007-06-12
just to make it clear for everyone
you can run kde apps under gnome with absolutely no problem. so if you like an app, just go ahead and install it, and don't worry about gnome vs kde stuff.

the first time you install a kde-based app, it will install some kde libraries, which may take a few tens of megs. but subsequent kde apps you may wish install (if any) will be using those same libraries, so you won't be needing to install them again, just the app. 

so don't be afraid - use kde apps on gnome! :)
