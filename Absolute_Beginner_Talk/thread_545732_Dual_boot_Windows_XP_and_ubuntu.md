### Post by ORBiTrus on 2007-09-08
A) BACK UP XP
B) Defrag XP [not sure if still necessary]
C) Set XP 12GB
D) Set Ubuntu "/" to 4.5GB (6GB if you expect to compile anything)
E) Set SWAP as needed
F) Set remaining to /home
-or-
F) Set remaining to /mnt/data [FAT32 if you want to talk to XP with ease]
