### Post by jordanmthomas on 2008-01-16
It doesn't matter what OS you burn it in.
Just make sure that after you burn it, it's not just the iso on a cd.  There should be many files.

If you've done it wrong next time make sure you burn it as an iso, not just an iso on a cd (wow, didn't explain that one very well).

EDIT
ignore the stuff about the BIOS if you saw it, didn't see that you already had it set up.
Anyway, to burn your disc right, you can use this free program for Windows:
[http://infrarecorder.sourceforge.net/](http://infrarecorder.sourceforge.net/)
It has an option to burn an iso image.

### Post by Rotaj on 2008-01-16
Once you have the ISO file downloaded to your hard drive, it needs to be burned to a disc _**as **_ an image file. I use ImgBurn. It is a simple windows program and does an excellent job.

When burning the disc image, _never_ burn it at the fastest speed! 

The link below is for ImgBurn.
