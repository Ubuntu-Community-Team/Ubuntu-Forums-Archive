### Post by Tatty on 2008-04-02
You shouldnt lose anything, the upgrade process should take care of updating all software to the newest versions.

However:

a) Remember 8.04 is still in beta, so it is not released as a stable version yet, meaning that it may still be buggy.  So you shoudlnt trust it if you have any data on it you cant risk losing, or if you want a stable system.

b) You should always back up any personal files before a distro upgrade, as this is a heavy change to your system, and there is always a small chance things might go wrong.
