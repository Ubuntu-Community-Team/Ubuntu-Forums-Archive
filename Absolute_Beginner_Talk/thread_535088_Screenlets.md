### Post by cdiem on 2007-08-26
In some newer versions the tray-command may be another command. If you have the screenlets installed, hit Alt + F2 and start typing "s-c-r-e-e-n-l-e-t-s". At some time, about the letter "l" I think, you should be able to see the screenlets configuration editor as an option to start. The just choose it and start it.
