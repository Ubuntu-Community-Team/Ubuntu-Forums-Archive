### Post by Tatty on 2008-04-01
Typically version numbers of software are frozen during each ubuntu release to prevent breakages.  Usually updates are purely security updates, so it wont appear in the repos.

If you want to install the latest version of some software, you have several methods - not all of them may be available.

1. check if it is available in the ubuntu-backports repository
2. see if there is a .deb on getdeb.net 
3. see if there is a .deb on the official website 
4. download and compile the source code.

I have just had a quick look at it appears that the only thing that is currently available for conky 1.51 is the source code.  So you can either have a go at compiling the software yoruself, or wait around to see if someone will package it for ubuntu.
