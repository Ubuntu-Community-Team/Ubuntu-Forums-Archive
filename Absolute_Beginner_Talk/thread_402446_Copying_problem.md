### Post by Happy_Man on 2007-04-05
Is the drive formatted as NTFS? Ubuntu can't write to NTFS-formatted drives. If that is the case, I strongly recommend ntfs-3g. It's a driver pack that allows Ubuntu to read and write to NTFS drives. Here's a link: [http://ubuntuforums.org/showthread.php?t=217009&highlight=ntfs-3g](http://ubuntuforums.org/showthread.php?t=217009&highlight=ntfs-3g)
Hope this helps!
