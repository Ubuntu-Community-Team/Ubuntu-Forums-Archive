The differences between Xubuntu, Kubuntu, Edubuntu, and plain Ubuntu

[LIST]
[*]Ubuntu- Uses GNOME.
[*]Xubuntu- Uses the XFCE desktop enviroment. XFCE uses much of the framework of GNOME. Only *ubuntu that you cannot order pre-pressed CDs for.
[*]Kubuntu- Uses the KDE desktop enviroment. Some programs are patched to work better with KDE.
[*]Edubuntu- Also uses GNOME, but comes with educational software.[/LIST]You can install the software and desktop enviroment for each (with a pre-existing installation) using terminal and typing "sudo apt-get install *name*-desktop" or "sudo aptitude install *name*-desktop". For example, to install XFCE, use "sudo apt-get install xubuntu-desktop" or "sudo aptitude install xubuntu-desktop"
