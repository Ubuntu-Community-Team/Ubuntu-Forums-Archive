### Post by Najand on 2006-09-11
Maybe you need to change your keyboard layout.
Click on System -> Prefrences -> Keyboard
Then Click on "layouts" tab.
Change keyboard model to: Japanese 106-key
Also, add "japan" to the box below (Selected layouts), by clickinf "add" and selecting "Japan".
Remove other selected layouts, if you have any like "US", because there is a bug there which will change your keyboard Form Romaji Mode to Kana Mode.
Then select "Japan" as default and click "Close".
