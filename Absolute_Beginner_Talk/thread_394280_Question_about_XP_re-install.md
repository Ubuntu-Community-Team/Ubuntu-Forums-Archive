### Post by joneil1000 on 2007-03-26
Last recovery disk for windows I used (was an older IBM laptop), it did nuke the entire hard drive, it that helps (or hinders).  When they say they the recovery CD will wipe the hard drive, I think the assumption is your system is totally infected with some horrible virus or worm, so it's something of a scorched earth policy.

    One other problem - beware  of long, long downloads for Windows updates.  How old is your software - for example is it Service Pack One instead of SP2?  

   The reason I say this is two fold.  First,when I did a re-install on an older machine, and the original XP disk had only Service Pack one,   it took an entire evening to download, install, reboot, download, install, reboot, etc, etc, all the darned updates.

   Also, I just bought a new desktop with XP, and even brand spanking new, it still took a good ten minutes of downloading and installing "critical updates".  *sigh*    

good luck.

### Post by oilchangeguy on 2007-03-26
the previous post is totally correct. your system restore disc will overwrite everything on the hard drive. and install a factory fresh copy of the operating system. and the first place you will want to go to is [www.microsoft.com](www.microsoft.com) and get all the available updates, click on custom, as there will be optional hardware and software updates available. first however is to download and install sp2, if your recovery disc does not contain that. and also download and install windows defender anti-spyware software. and then go to [http://free.grisoft.com](http://free.grisoft.com) and download and install the free version of avg anti-virus software. and to the previous poster, it doesn't matter that your computer was new, it was probably built 3-6 months ago. so yes there will be updates.
