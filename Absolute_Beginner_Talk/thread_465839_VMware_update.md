### Post by dannystaple on 2007-06-09
It should be that easy, but beware that the virtual hardware that VMWare provides might be slightly different, hopefully not, or hopefully if it is, it is nothing that the Ubuntu guest cannot cope with. I would say to make a copy of the disk image files if you are really worried about it before sarting the new vmware, then go ahead with the upgrade.
