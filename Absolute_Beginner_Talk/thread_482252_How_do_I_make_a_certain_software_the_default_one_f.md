### Post by iceportal on 2007-06-23
System -> Preferences -> Removable Drives and Media

Click the Multimedia tab and you can define software for various media you add...

As for media files, I know that with lots of them, you can just right-click the file, select "Properties," select "Open With," and check-in the radio button for the one you'd like to use as the default.

This should automatically use that program to open all files of that type from then on.
