### Post by crispy_420 on 2007-03-07
My option adds a virtual printer that will work with any app that can print (openoffice, firefox, abiword, etc.). Your OS will see a "printer" that is not really there, all the PDF's go to a file called PDF in your home directory. I use it all the time to say guides that I find on these forums. 

Give it a try and you won't regret it...
