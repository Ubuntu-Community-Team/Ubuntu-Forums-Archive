### Post by aJayRoo on 2007-08-02
You need root permissions to copy files there. This can be done by prefixing your command (at the command line obviously) with 'sudo'. This will prompt you for your password, and if you have the necessary privileges (you probably will) will allow you to complete your task. Try it out and let us know how you get on. I haven't installed ddclient myself but the documentation on the dyndns site and the supplied documentation should get you through the process. It may seem a little complex at first so don't be afraid to ask more questions! Hope that helps.
