### Post by CroEragon on 2007-01-17
You can use gparted from Sys Rescue CD. It is live cd with all main apps for sys rescuing.It is free and you can download it from [here.]("http://www.sysresccd.org/Download") With it you can erase all partition. If you are not used to command line dont be scared whrn it boots. Just read what it says and type "qtparted" whnen you get command and you will get GUI program with graphical interface. Also, you may want to consider running 98 in VMWare. It will be bit slower, bit i dont think it will be too much, with fairly modern computer it will work good enough.
