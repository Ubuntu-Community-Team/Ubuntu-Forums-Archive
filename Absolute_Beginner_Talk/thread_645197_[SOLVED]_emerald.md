### Post by NateMan on 2007-12-19
sudo apt-get install emerald
emerald --replace
go to ccsm and go to window decorations and in commands add -emerald
open up emerald theme manager to change themes. before you can change themes press ctrl + alt + backspace to restart x so that you can select more themes. go google more themes. installation is self explanatory.

hope that helps
