### Post by MrFSL on 2007-07-23
For others looking for help with Audacity.... I have found Audacity for both Edgy and Feisty to be rather buggy. Under Feisty I would suggest using Ardour instead since it seems much more robust and feature filled. I good DEB file for it can be found here:
[http://www.getdeb.net/release.php?id=1193](http://www.getdeb.net/release.php?id=1193)

If this is simply WAY more then you need I suggest looking into a more recent release of Audacity such as:
[http://www.getdeb.net/app.php?name=Audacity](http://www.getdeb.net/app.php?name=Audacity)

Hope this helps.
