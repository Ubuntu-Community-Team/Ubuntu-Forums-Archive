### Post by Forrest Gumpp on 2008-02-14
supertails,

You say you intend to dual boot Ubuntu with Windows soon, as opposed to being already committed to it.

You can make your changeover smoother and less frustrating if you read not only the APC Guide that jan quark posted a link for, but also the sticky thread by ubuntu_demon at the top of the index pages in Absolute Beginner Talk (in which you have posted).

ubuntu_demon gives a link in that thread to Herman's Illustrated Dual Boot Site home page  [http://users.bigpond.net.au/hermanzone/](http://users.bigpond.net.au/hermanzone/)  It is a big site, so allow a fair bit of time to surf it and read and bookmark the relevant bits, especially the GRUB page  [http://users.bigpond.net.au/hermanzone/p15.htm](http://users.bigpond.net.au/hermanzone/p15.htm)  and the SuperGRUBdisk page  [http://users.bigpond.net.au/hermanzone/SuperGrubDiskPage.html](http://users.bigpond.net.au/hermanzone/SuperGrubDiskPage.html)

Make yourself a SuperGRUBdisk floppy before you even start to set up dual booting.  That last link tells you how, and where to get it.

If you are thinking about dual booting using two HDDs, as opposed to partitioning just one HDD so that it can have both OS on it, study confused57's how-to thread  [http://ubuntuforums.org/showthread.php?t=179902&page=6](http://ubuntuforums.org/showthread.php?t=179902&page=6)

This thread in the Ubuntu Community Cafe forum may give you some ideas as to what things you can do in Ubuntu that you cannot in Windows.  Its a fairly long thread, but you can learn a lot from it.  [http://ubuntuforums.org/showthread.php?t=399997](http://ubuntuforums.org/showthread.php?t=399997)
