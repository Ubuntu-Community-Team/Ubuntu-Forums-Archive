### Post by Toadinator on 2008-04-16
What exactly is on the flash drive? The .iso file or what's inside of the .iso file, because if it's just the .iso file then you need to extract it into the USB drive. If the latter is the case then it should work just fine.

What kind of install do you want to do? If you want it installed over windows and everything else then just follow the included instructions. If you want Ubuntu and Windows on there at the same time look at how much space Windows already takes up compared to it's maximum and reduce it's maximum to a little over the minimum, like 2-20 gigs more and use the rest for Ubuntu and a SWAP partition in case you run out of RAM (you probably don't need to do this if you have like 1 or 2 gigs of RAM). If you want to install Ubuntu inside of Windows then I'd suggest that you wait until Hardy Heron on the 24th.

I hope I helped.
