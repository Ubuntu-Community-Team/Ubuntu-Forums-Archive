### Post by tubasoldier on 2008-03-26
The .uif file format is a PROPRIETY format used by MagicISO. More than likely it is a form of the standard ISO format but not a standard itself. Therefore there isn't much out there in the Linux world to help you with it. You could try running MagicISO under wine.

You may also look at poweriso: [http://www.poweriso.com/download.htm](http://www.poweriso.com/download.htm)

My advise is to just stop using proprietary formats when there isn't much difference between them and the standard ISO files.
