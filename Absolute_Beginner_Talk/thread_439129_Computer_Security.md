### Post by reckless2k2 on 2007-05-10
Linux doesn't work like Windows. There's no need to create separate accounts unless you want multiple users. If and when you need to escalate privileges, you will do so using sudo. Believe me when i say that there should be absolutely no fear of getting online with Ubuntu. No viruses, spyware, or adaware. Ubuntu has all ports closed by default although some will argue to install a firewall. It's not "really" necessary.
