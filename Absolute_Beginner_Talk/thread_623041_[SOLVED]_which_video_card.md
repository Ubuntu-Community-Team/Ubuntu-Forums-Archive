### Post by overdrank on 2007-11-25
Ok if you willing to try then I would reconfigure my xorg. The xorg frankly looks bad. SO use this command ```
sudo dpkg-reconfigure -phigh xserver-xorg
``` in the terminal and then restart x with ctrl, alt, backspace  and I hope this helps and you may  want t add this to you xorg
  Option         "AddARGBVisuals" "True"
    Option         "AddARGBGLXVisuals" "True"
    Option         "NoLogo" "True"
In the device section using the command ```
gksudo gedit /etc/X11/org.conf
```but back up you xorg first
To copy Xorg
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf-backup
To restore backup
sudo cp /etc/X11/xorg.conf.backup /etc/X11/xorg.conf
