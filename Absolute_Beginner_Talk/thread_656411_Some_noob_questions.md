### Post by p_quarles on 2008-01-02
You can *install* IDLE on Ubuntu, so there's little reason to use the Windows version. And, yes, you can access any Python scripts that you might have on the Windows partition.

Compiz is installed by default in the latest version of Ubuntu. You'll need to get the package called compizconfig-settings-manager to get the transparent windows.

Yes. The live disk runs entirely on your system's RAM. Installing it on the hard drive will make it run considerably faster.
