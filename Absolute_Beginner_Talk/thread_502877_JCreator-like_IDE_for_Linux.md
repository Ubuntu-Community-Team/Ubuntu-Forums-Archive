### Post by Majorix on 2008-01-28
> **djbon2112 said:**
> I can't get it working. I get an "External Exception 6BA Abort/Retry/Ignore" for EVERY file when it tries to install (the latest version from the JCreator website). Any advice? Or, how does one get Gedit to do syntax highlighting and easily compile/run Java apps?

Install the gedit-plugins and then enable the "Shell" plugin through GEdit's prefs. Then you will have a shell at the bottom which you can use to compile/run your code. And GEdit already has syntax highlighting by default. I also suggest the auto-indent and auto-comment plugins, which will be available to you after you install the said package.
