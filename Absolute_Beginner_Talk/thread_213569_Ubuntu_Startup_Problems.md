Hi. I'm new to the Linux world and I currently use Ubuntu 5.10

Recently, I've been having a folder in my home directory open up on start up. I didn't include this as a start up option...and as a matter of fact, i've checked my start up list and it's empty.

I tried renaming the folder it always open so that it doesn't see it but then when i start up and log on, it brings up a dialog box telling me that it can't find the specified folder...it however still has my home folder opened.

Please can anyone help me out here. I need it to stop attempting to open any folder(s).

### Post by invalid on 2006-07-11
It sounds like you may have saved a session while that window was open.
Go to System -> Prefrences -> Sessions (make sure you have that window closed now) and check/uncheck Automatically save sessions.
This fixed a similar problem for me a few months back.
