### Post by MasterXandrex on 2008-02-05
First, I would believe that there is something you need to check. I assume in your assertion that you are using RAID 0, you are referring to a hardware RAID. If you are using a software raid, you may run into some problems when it comes to formatting a partition.

### Post by chewit on 2008-02-05
Ypu shouldn't have a problem. When you try the Live CD, Ubuntu will identify two HDD, it will know one has XP on it. When you go to install Ubuntu, there will be an option to select where to put Ubuntu. It will clearly show that you have two HDDs and you select the one you want. This was the same with my internal HDD & ExHDD.
