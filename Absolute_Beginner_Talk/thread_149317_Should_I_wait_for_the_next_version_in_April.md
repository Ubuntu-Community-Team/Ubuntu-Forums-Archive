### Post by compuguy1088 on 2006-03-23
Because they delayed Dapper, it is best to load Breezy, mainly because of its stablity. The only reason you would install flight 5 of Dapper is if: 1. there is some feature in the devel version that you really need 2. you like using the bleading edge (and backup files alot) and 3. because being up-to-date is your thing. You should be fine by just installing Breezy, and also make you /home/ directory on another partition, so if you have to reeinstall to load Dapper, you won't loose your files. There is a good HOW-TO [at this link]("http://ubuntu.wordpress.com/2006/01/29/move-home-to-its-own-partition/").
