### Post by nebhead on 2007-03-20
There hasn't been a simple instruction guide of cinelerra recently, so i thought i would cater for the noobs and make a very simple guide on how to install cinelerra 1.2.0
i cannot guarantee this will not do some harm to your system, do this at your own risk, it all worked for me so not really a problem.

so, the guide:

its as simple as adding this repository, you can do this by opening synaptic package manager and going "system> repositories" then go on third party and click "add" , copy and paste this into the box "deb [http://www.kiberpipa.org/~gandalf/ubuntu/dapper/cinelerra/i686/](http://www.kiberpipa.org/~gandalf/ubuntu/dapper/cinelerra/i686/) ./ "  its meant to be for dapper but it worked for me in edgy, then click "refresh" and search for cinelerra and install it, simple as that. :D
If there are some problems please let them be known.
