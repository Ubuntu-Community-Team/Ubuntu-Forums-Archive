### Post by Bakerman on 2007-05-15
Hi, If you're referring to a CD track listing, Sound Juicer CD Extractor will search for a track listing for you when you rip a CD. Works like a charm for me anyway. You can find it in the add/remove programmes. I found another programme there called EasyTAG which you can use to edit the ID3 tags of the audio files once you've ripped 'em. I'm sure there must be better programmes in the repos, but being new to Linux myself, I just haven't found them yet.

### Post by wataboutbob on 2007-05-15
Sound Juicer will fill-in tags when you rip from your CD. For mp3 already ripped, you can use Amarok or Musicbrainz to fill in tag info. For tag editing, I've found Audio Tag Tools way better than EasyTag.
