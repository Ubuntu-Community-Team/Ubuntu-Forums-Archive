### Post by Requiem on 2006-09-11
There is are many, if you are just using the default one that comes with ubuntu do this:

Click on Edit>Current profile...>Effects>Background Image

Down there is a transparent background option, it doesn't actualy make the background transparent, it just loads your wallpaper picture into your terminal background, it way cooler than just displaying a background picture.

Thesee are other terminal programs:
Gnome Terminal (for Gnome, ubuntu's default)
Konsole (for KDE like in Kubuntu)
XFCE Terminal (for XCFE like Xubuntu)
eterm (for enlightment)
xterm (old favorite, also installed by default in ubuntu, won't load background pictures tho)
rxvt (for veeeeery old machines, no pictures again...)
aterm (never used it, probably similar to eterm)
