Strongly raccomended to uninstall any previous version.

Here the changelog from Pidgin´s webpage:

[I]2.2.1 is out. It fixes a crash that can be triggered remotely on MSN in 2.2.0. If you are using 2.2.0, you should upgrade immediately. See our security advisory for details.

    * libpurple:
          o A few build issues on Solaris were fixed.
          o Cancelling the password prompt for an account will no longer leave it in an ambiguous state. (It will be disabled.)
          o Fixed an erroneous size display for MSN file transfers. (galt)
          o Fixed multiple memory leaks, particularly in XMPP and MySpace protocols
          o Fixed remembering proxy preferences and status scores
          o Gmail notifications are better tracked 

    * Pidgin:
          o Fixed keyboard tab reordering to move tabs one step instead of two.
          o You should no longer lose proxy settings when Pidgin is restarted.
          o Fixed detection of X11 when compiling 

    * Finch:
          o Pressing 'Insert' in the buddylist will bring up the 'Add Buddy' dialog. [/I]
