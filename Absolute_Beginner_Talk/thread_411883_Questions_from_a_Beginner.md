I am sick and tired of XP. I have used it for so long, and it has really started to get under my skin. So, I've decided to move over onto Ubuntu, but I have a few questions first.

### Post by Bekker on 2007-04-17
First off, you'd better wait until thursday, then version 7.04 "feisty fawn" will be released.

1: Ubuntu will install "grub" on your computer, that's a bootloader. What it will do in effect is give you a menu at startup where you can choose which OS to start.

2: Yup, that'll work. Somewhere during the install you'll have some options for partitioningen, manual would be the choice for you, then make sure you select the right partition for your ubuntu install.

### Post by PartisanEntity on 2007-04-17
1. Ubuntu will install a boot loader and place itself as the first option with your other operating systems at the end of the list, however you can later edit the order of the operating systems, so for example you could move Win XP to the top of the list.

2. The easiest method for a beginner would be to install Ubuntu **without** making any partitions for it before. Ubuntu will ask you if you want to use up some free space from the existing partition to make a new partition **during** the installation process.
