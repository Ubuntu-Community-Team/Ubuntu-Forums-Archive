### Post by theninthdimension on 2007-04-06
Dragon665 and JustBrowsing,

     Here's a long response that may help.

     I'm running here at work (work being a medium-sized college) a Dell Dimension 9200 with 4 gigs of ram, a dual core 2.66 mhz 4M Pentium processor. The system has two 500 gig hard drives (I'm running them separate, one has WinXPPro and the other has my Ubuntu 6.10 install), dual combo DVD/CD R+W drives, dual 24inch monitors on an NVIDIA card. Basically, the system is a little less than a year old and when I picked it out I was able to get basically the top-of-line that Dell had to offer.

     I have been a Windows user for about 15 years now and consider myself fairly savvy when it comes to making WinXP Pro as fast and responsive as possible. So far, I have found my Ubuntu install to be noticeably faster and smoother. I think I'm generally comparing apples to apples as I usually use OpenOffice under Windows as well as Firefox under both.

     It has been more of a challenge getting various peripheral devices to work, but nothing that patience, the Ubuntu Forums, and Google haven't been able to get me through. My computer sits on a basically Windows-only network, but despite that, I've had no trouble moving, accessing, or using files on Windows servers, accessing shared HP printers on the network, or accessing my e-mail through the school's MS Exchange system. In fact, in the 12 days that I have been running Ubuntu 6.10, there is nothing that I have tried to solve that I haven't been able to solve (as they say, fortune favors the dumb).

     Don't get me wrong, I don't think Ubuntu is for everyone and, probably, neither is GNU/Linux in general. But, for me, it's working great. I wish the various names and numbering systems made more sense to me. For that matter, I wish the terminal commands made more sense, but I assume that will work itself out in time. I doubt that I will be completely sans Windows any time soon, if ever, considering the nature of the software world, but so far I haven't had to use Windows for anything in 4 days.

     So, yes, I do think Ubuntu is worth it (and by extension GNU/Linux generally). I honestly haven't had this much fun playing with a computer in years.

Jeff.
