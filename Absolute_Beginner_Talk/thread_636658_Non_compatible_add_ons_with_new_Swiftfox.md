### Post by thelatinist on 2007-12-22
The problem is usually that extensions have set a maximum version in their configuration files.  The idea is that you won't be able to run an extension until the author has tested it to make sure it won't break the new version.  Firefox 3.0 has not yet been released, so very few extension authors have bothered to update their extensions yet, especially since changes between alpha and final release might just break them again.  When you're using beta software you can't expect all features to work perfectly.
