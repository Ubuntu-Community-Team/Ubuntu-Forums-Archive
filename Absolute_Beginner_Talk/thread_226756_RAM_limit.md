### Post by catlett on 2006-07-31
It's not a stupid question. Things have changed since the last release. 
The big change is the Live CD installer. You need a good amount of ram to perform this kind of install. Why? Because you first have to boot into a live  session of Ubuntu. That alone will take up your 128mb of ram (maybe more) and then the installer has to run while the live session is going on.
It is a nice "user friendly" way to have new people get a dual booting linux system but it limits the people who can use it to those with 256mb or more of ram.
As already mentioned, the Alternate Install cd is the installer that you have already used. It is the "old" Ubuntu installer. Text based and you boot right into the installer.
I actually prefer the alternate install cd because it gives you more options and more control of your configuration.
So don't put yourself down. As they say "The only dumb question is the one you do not ask."
