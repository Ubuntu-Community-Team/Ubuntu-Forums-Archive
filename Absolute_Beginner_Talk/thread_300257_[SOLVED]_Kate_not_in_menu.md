### Post by jordanmthomas on 2006-11-15
Right click on your Applications menu and go to Edit Menus
This will bring up Alacarte, and you can add Kate from here.

BTW, Kate is my favorite text editor, good choice.

### Post by jordanmthomas on 2006-11-15
If it isn't there, you need to add it manually.
In Alacarte, add a new Item
and make sure the command line reads
```
kate
```
