### Post by annda on 2007-03-05
i don't know the card, but running the cd can sometimes be more demanding of hardware than an installed OS.

before you try the alternate cd, it would be great if you could try ubuntu out first and see if you like it and if it works with all your hardware.

so maybe try again with the cd and when you boot from it, choose from the first ubuntu menu

1) check disk 

if you get errors try burning the cd again (probably not, since it was OK with your friend's computer).if it's OK

2) choose safe graphics mode

if that does not work, hit F6 for extra boot options and type
vga=791 (this is standard resolution 1024x768)

hope it helps. if not, the graphic card is not the problem, but something else.
