ok... for the compatibility part, i would not worry... gnu/linux has a variety of desktop environments and window managers to make your desktop look exactly as you wish (gnome, kde, xfce, fluxbox, e17 etc etc etc) with some tweaking you can even make it look just like windows, although i don't really know who would like to do that

the main problem you will encounter is the compatibility of the programs you use... there are a lot of programs for gnu/linux that let you do the stuff you do on windows programs... the problem arises when you need to use specific program that has no linux port... then you can probably use wine to run windows programs in linux

the games that are for windows only can be played using wine or cadega (if you wish to pay a little)... for more info go to the gaming subforums here and search for thread titles with the games you play now, to see if they run on ubuntu or not

### Post by allstar1971 on 2008-04-17
I believe you would be better off. 

I made the switch about a year ago and went through some frustrating times. It was a tough learning curve but I am excited about the future of Linux. I have tried just about every distro and each have their plusses and minusses.

It really comes down to the desktop. Each distro uses either Gnome or KDE. 

I currently use Ubuntu because I like the daily updates and Gnome is easier to customize and appears to have more choices in customizing it. The Ubuntu community is also huge and their forum is quick to answer nearly any problem you have.

KDE gives you more of a Win-Doze look, but has some serious performance flaws especially since they are moving to KDE4, which is beautiful to look at, but will not allow you to use some of the earlier "skinning or theming" choices that KDE 3x did. (Personally, I liked Baghira which REALLY isn't available with KDE4).

As for some of the Windows apps and games, there are several open source options available. IMHO, OpenOffice does everything that Word, etc. does and more!

However, the choice is yours. Nearly every distro has a live CD which allows you to try it without messing up your current system. It simply runs off RAM.

I would try Ubuntu, OpenSUSE, PC Linux, Mint. Also understand that you can uninstall almost any program you do not need if you find a distro that fits your fancy. You can also change the boot screen, login screen, etc with almost no troubles. Plus -- you have to love Compiz!!!

I hope this helps.:)

### Post by TUOggy on 2008-04-17
I've been working with Linux casually for a few years.  I have found that Ubuntu is easily the most user friendly distribution that I have come across.  I have tried fedora core (Red Hat offshoot), Suse, and a couple others.

If you are more familiar with Windows, you may want to give Kubuntu a try, because it uses the KDE (which is more similar to Windows) gui rather than Gnome (which is more similar to OSX).

One thing that I have found is that no matter what distribution you use, if you want to do things other than the basic browsing and office apps, you are going to have to learn some stuff that you never had to with Windows, and some things that hasn't been commonly done since the days of dos.  Don't let this scare you off though.  I find it very rewarding when I figure out how to do something.  What makes it even better is that it gives me an opportunity to come back here and help others who may be having similar problems.

You will have to learn how to use terminal for some tasks (especially if you want to install applications that are not available through a repository).

As far as cross compatibility, while the Linux community is making great strides with programs like wine and cedega, it is far from perfect.  You will run into problems that require troubleshooting, and some programs which simply won't work.  There are, however, usually alternatives which were designed to be run in Linux available for download.

Once again, I'm not trying to scare you.  I think the benefits you get from Linux far outweigh the downfalls of not having a widely utilized OS.  I just want to let you know what you would be getting into.  The nice thing is, as it becomes more widely accepted, companies and programmers begin to develop more utilities, tools, applets, and applications which make it easier to use.
