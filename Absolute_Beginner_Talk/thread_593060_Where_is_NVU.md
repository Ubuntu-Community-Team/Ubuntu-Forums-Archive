### Post by RKCole on 2007-10-26
KompoZer is essentially a bugfix version of Nvu.  Nvu's original developer stopped working on Nvu to pursue development of a different Web authoring tool currently codenamed Composer.  I have not heard anything about this since February or so, so I don't know its status.

Anyhow, someone took the code for Nvu and worked out a lot of bugs.  KompoZer now has a built-in CSS editor among other things.  It works very well.  I'm not sure about Feisty, but KompoZer is in the Gutsy repositories.  It is still being developed, so it hasn't been ditched.

Take care.
