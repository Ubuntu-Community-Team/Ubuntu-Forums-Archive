### Post by Copter on 2007-06-23
Hi!

Ive used this kind of software on Windows for my LG 15" lcd but it didnt seem to improve anything actually. IMO this kind of addons are worth nothing so you should not worry if there is no linux equivalent for it. The main pupose of it is probably to adjust colors to the light conditions in your workspace. In my case when I used this soft to do it for example at 10am with sunlight it looked crappy at 10pm with artifical light. After some time I used one user setting for everything and one bulit-in setting for watching movies after sundown.

Anyway, one can live without it :)
