### Post by Solver on 2006-06-08
As of now, gamers are definitely better off sticking with dual boot. For people who maybe play one game once in a while, the emulation options are suitable, but for gamers who play new, graphically intensive games, dual boot is the better option. Assuming you can get the games running on Cedega, the performance is usually worse and there may be some glitches. Basically, there is no reason not to dual boot. Just use Windows for gaming, Linux for everything else.

Of course, it's great that more game developers start to care about Linux now. iD released a version of Doom3 that works under Linux. Better yet, you just need to use the regular retail version and download a few files for Linux. Howevre, so far, games with native Linux ports are an exception and not the rule.
