**Yes, pretty much all needed stuff is in add/remove. If something is missing, getdeb.net is a good place - usually things that will go in, but aren't in yet, are there. I think all of the (k/x/edu)ubuntu's have the add/remove, no idea about others.**

4.Am i right in saying you can only install programmes if you have the source code for it?

**No. Compiling source code is for the tech people. You either need the program to be in add/remove, or it provide a .deb file - ubuntu can install those easily. Of course, later, if you want look into compiling source code (it's pretty easy once you figure it out - all you have to do is type the "make" command and ubuntu will compile the stuff)**
