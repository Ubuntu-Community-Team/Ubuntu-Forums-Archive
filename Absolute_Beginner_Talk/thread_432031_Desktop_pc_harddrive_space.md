### Post by mcduck on 2007-05-03
> **linuxhippy said:**
> What is the advantage of Ubuntu over Fedora Core?  Fedora is a much heavier install (1 DVD) where Ubuntu is only 1 cd.  Is Ubuntu trimmed down or is it just as powerful as any other Linux distro?

It's just a question of what you like best. Ubuntu is in no way 'trimmed down' (in the meaning of being less powerful than some other distro), it's just that it only comes with one tool for one task by default. You can always install more from Ubuntu repositories if you need.

Distros with more than 1 CD usually come with huge selection of programs on the install disk, which might be nice if you don't have a working internet connection, but most users won't ever need all those programs.
