### Post by omockler on 2007-03-14
The first option when you boot from CD should be "start or install ubuntu." The CD boots to a live system and you can install from there.  It will always warn you if it is going to alter your hard drives. If it tells you that it is going to erase data you can always just switch off your computer.

### Post by dstew on 2007-03-14
The Start/Install menu item only starts the Ubuntu live CD, and gives you the Ubuntu desktop. After you get to the desktop, there is a separate Install icon to select if you want to start the installatiion process. If you just want to explore Ubuntu, you can use the desktop, without installing.

I think a lot of new users are intimidated by the Start/Install selection. I was too.
