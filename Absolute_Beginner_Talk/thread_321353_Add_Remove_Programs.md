### Post by BarfBag on 2006-12-18
By "Add/Remove Programs," I assume you mean Synaptic?

Yes.  Kubuntu is basically the same as Ubuntu, except with a different desktop environment.  It uses KDE, while Ubuntu uses GNOME.  If you install the program with Synaptic or apt-get, it will automatically grab all the KDE dependencies it requires to run.  Most of the time, it works flawlessly.
