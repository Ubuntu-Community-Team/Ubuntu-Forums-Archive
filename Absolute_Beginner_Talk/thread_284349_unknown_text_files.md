### Post by x64Jimbo on 2006-10-25
One way you could verify that they're installed by Automatix would be to run that program, then check the last modified date on the files. If it's today, I'd say you're good to delete them. You could also create a directory in your home folder called .keys, and store them in there. If something breaks, you know where they are.
