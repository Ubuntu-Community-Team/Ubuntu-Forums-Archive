### Post by xequence on 2005-09-09
You can use a different desktop environment...


this is a really really fast one:
sudo apt-get install fluxbox

so is this:
sudo apt-get install icewm

this is fast but looks good too:
sudo apt-get install xfce4

After isntalling, log out, then choose a different one in "session" I think. Then log back in. You can set your favorite to default.
