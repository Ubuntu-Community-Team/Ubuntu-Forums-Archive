### Post by Bachstelze on 2007-04-14
Yes, whether you use KDE or Gnome doesn't matter, Ubuntu is still Ubuntu so all Ubuntu packages will work fine if you use kde-core.

You are limited in the way that kde-core only installs the "core" of KDE, plus some important KDE apps (Konsole, Konqueror and a few others). Of course, you're perfectly free to install extra KDE or non-KDE apps if you wish. If you want to install all of kde, you can always install *kde* instead of *kde-core*.
