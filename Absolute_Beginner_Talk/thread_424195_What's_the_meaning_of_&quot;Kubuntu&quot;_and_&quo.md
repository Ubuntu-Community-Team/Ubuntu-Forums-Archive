### Post by unnes on 2007-04-26
Ubuntu is a *distribution* of Linux. Like all linux distributions, it uses the linux kernel, but is packaged with different operating system components and applications.

The shell you speak of is the desktop environment. Standard Ubuntu uses GNOME as its desktop environment, but some people prefer KDE or XFCE. When packaged with these other desktop environments, Ubuntu is referred to as Kubuntu or Xubuntu, respectively.

While your install probably uses GNOME, you can easily install an alternate environment, and choose which one you wish to use at logon.

### Post by mikedpirone on 2007-04-26
OK well first off it is spelled Ubuntu and Kubuntu. Ubuntu uses the GNOME graphical user interface. Kubuntu on the other hand uses the KDE graphical user interface. At the core of them there is no true difference. The difference between them is the graphical user interface, which is a choice of preference. Some people prefer GNOME and others prefer KDE. There is also Xubuntu which uses the Xfce graphical user interface which is a little more light weight (uses less resources) than GNOME or KDE. 

Fiesty Fawn is the release name for Ubuntu 7.04. Each release has a different release name, such as 6.10 has the name Dapper Drake, and the release before that was named Edgy Eft.

As for the difference from other linux distrobutions I can not completely give an answer for because I am a newb to linux.
