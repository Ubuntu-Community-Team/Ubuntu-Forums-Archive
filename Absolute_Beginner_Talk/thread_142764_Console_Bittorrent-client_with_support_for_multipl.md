### Post by Svictor on 2006-03-11
I use mldonkey, (it's in the repos). It is commandline but has also some third-party guis availible. You can also pilot it through http, from your browser ([http://127.0.0.1:4080/)](http://127.0.0.1:4080/)). It handles various networks, including bittorent. It can manage several torrents at once. It wasn't very easy to understand for me at the begining but it's definetly efficient (check their wiki on [http://mldonkey.org/)](http://mldonkey.org/)).

I use the browser interface and for torrents, i type "dllink path/to/torrent/nameoftorrent", in the interface (there is a field for passing commands to mlnet ; don't know the syntax for "real" commanline).
