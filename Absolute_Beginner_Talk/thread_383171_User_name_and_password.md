### Post by Kateikyoushi on 2007-03-12
Go to System > Administration > login window, then on the security tab you can enable automatic login.

---

### Post by machoo02 on 2007-03-12
Although I would not recommend doing this, this seems to be the way:

System->Administration->Login Window

Under the Security tab, click the box for automatic login and select your username.
