### Post by pHreaksYcle on 2007-11-29
Hey man you came to the right place! To make a command boot on startup every time go to:
System->Preferences->Sessions
Hit "Add" on the right hand side
Type what you want the entry to be called in name, doesn't matter what.
Type the command you want started every time in the command box.
Type a description if you desire.
Hit "OK" and your rockin' a new boot command! :guitar:
