### Post by aTEK on 2008-06-05
Thats not entirely true... I e-mail Pete Goodall a little while back about where to get an installable copy for UME or when it was due to be released, and this is the reply I got followed by my original e-mail.

"Hi Jack,

Thank you very much for your inquiry and for your interest in Ubuntu Mobile (UME).

Thank you also for forwarding on the link.  The Ubuntu Mobile project is currently working on support for devices using the Intel Menlow (now called Intel Atom and Centrino Atom) and McCaslin processors.  Though we do not test UME on this particular device, I see no reason why it would not run on this device.

The UME project has been in constant development since we announced the project last year, and all the development is public.  If you would like to download an early version of UME you can find pre-built images here:

[http://cdimage.ubuntu.com/moblin/](http://cdimage.ubuntu.com/moblin/)

We will have a release available along with the release of Ubuntu 8.04 Desktop and Server, but it is still a very young product.  I think you make a good point that this type of device could be a good fit for Ubuntu Mobile, and there are already a number of tools included (such as an on-screen keyboard) that would make that support possible.  We could probably even support the fingerprint reader. :-)

We are actively working with device manufacturers, so keep a lookout for Mobile Internet Devices running UME later this year.  Thanks again for your inquiry!

Regards,
Pete
