### Post by erginemr on 2008-01-08
You can find out where ktechlab executable has been installed with the command:
```
which ktechlab
```
and the command:
```
whereis ktechlab
```
will find all ktechlab related files in your system.

and use the main menu, as LaRoza indicated above, to add the launcher (shortcut, if you will) to a suitable section of your main menu. Its icon should be located somewhere in "/usr/share/icons" or "/usr/share/pixmaps".

Also check out the following tutorial on how to add shortcuts:
[http://monkeyblog.org/ubuntu/installing/#launcher](http://monkeyblog.org/ubuntu/installing/#launcher)
