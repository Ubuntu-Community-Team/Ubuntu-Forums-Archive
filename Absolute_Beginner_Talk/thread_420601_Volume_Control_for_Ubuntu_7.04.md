You should be able to go into alsamixer via terminal and turn up the volume on speakers.  When I first booted Edgy all my speakers were muted except the center.  I think you'll need a terminal command to set the volume on every boot as well.  The following link will provide more information:

[http://doc.gwos.org/index.php/Comprehensive_Sound_Problems_Solutions_Guide](http://doc.gwos.org/index.php/Comprehensive_Sound_Problems_Solutions_Guide)
