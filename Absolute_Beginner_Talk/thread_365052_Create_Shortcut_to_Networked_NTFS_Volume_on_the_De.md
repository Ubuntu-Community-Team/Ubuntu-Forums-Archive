I would like a shortcut to /winfiles on my desktop in Ubuntu Edgy. What should I do? Thank you.

---

### Post by nhandler on 2007-02-19
Right click on the desktop, and select 'Create Launcher'
For type, select 'File'
Enter something for the name (i.e. Winefiles)
For command, enter /winefiles
You can also enter a comment and select an icon, but they are both optional

This assumes that the networked ntfs volume is already mounted at /winefiles.
