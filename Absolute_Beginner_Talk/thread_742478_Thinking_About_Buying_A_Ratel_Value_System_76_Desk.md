### Post by DGortze380 on 2008-04-01
Core 2 Duo is a recent Intel dual core processor.  The simplest way to explain a dual core processor (although not entirely accurate) is that it is similar to having two processors in one. E6420 is an Intel model number. 2.13 GHz is the "speed" that each core is running at. For what you're doing that should be plenty. FSB stands for Front Side Bus, which is the "connection" from the CPU to the RAM, 1066 MHz is the "speed" of the front side bus.  L2 references the L2 cache which is a portion of memory that is actually on-board the CPU. Again, I wouldn't worry too much about this for what you're doing with the computer, 1066 should be fine.

Again, a VERY shortened and VERY simple explanation, but that should give you more than enough terms to search if you like.

As for whether or not it's a good computer.  It should be fine for what you want to use it for.  2 gigs should be fine for Ubuntu, but my advie is always buy as much memory (RAM) as you can afford. DDR2 667 is fine, but DDR2 800 is common as well.

You may want a DVD writer, comes in handy for backups.

For wireless internet 802.11b/g is fine. For quicker file transfers within your home network, look into the new 802.11n.

I believe Ubuntu comes pre-installed, so you shouldn't have any driver problems, unless you have other peripherals you're connecting. Then I would research the availability of drivers for those devices.
