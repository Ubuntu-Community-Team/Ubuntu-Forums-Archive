### Post by kevinlyfellow on 2007-07-24
Also, sometimes you can find the command your looking for with the command apropos.  For instance

```
$ apropos edit
alacarte (1)         - edit freedesktop.org menus
atobm (1x)           - bitmap editor and converter utilities for the X Window System
bins-edit-gui (1)    - Set fields in XML picture description files for BINS (using a GNOME-based GUI)
...
```

It's not the best, but it has helped me in the past.

### Post by expatCM on 2007-07-25
man is the manual but it is so big that only what you need is displayed.  So you enter man and then the subject of interest. For example try 

man format 

to find out about the syntax and options available when using the format command
