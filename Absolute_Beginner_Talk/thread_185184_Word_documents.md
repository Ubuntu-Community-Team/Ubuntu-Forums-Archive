Tools > Options > Load/Save > General > Always save as > choose the Microsoft format to save automatically.

### Post by damianubuntu on 2006-05-31
The only way to dump data back and forth with Access is to export to e.g. CSV file, and even then you can have difficulties with line feed/ carriage return diffferences.

with Word, be aware that many features will misrepresent even when you use the save as .doc option outlined above.  The true is in reverse, if your boss does something like add page art to a document (you know, that useful business application that lets you put ice-creams all round the outside of the page!) or anything else like that, it will mess up the document to a greater or lesser extent.

I haven't bothered investigating sufficiently to be able to catalogue a set of incompatible features, just noticed it often enough to add the warning to keep it simple!
