### Post by The Cog on 2008-01-09
The decryption module is not included in the default install because of legal restrictions in some countries like the US. Because of this, you have to make the consious effort to install the module yourself. It is in the libraries, so it's eaasy enough to install though, if you live in a country where it's legal to do so.
