### Post by deadgobby on 2007-01-19
The other post was kinda of right. So lets try this. With your cable modem on and hooked up the USB port. Reboot the system and see if it Auto Detects the signal. If not after you reboot. Go into System>Admin> Networking.  See if the Wire Connection settings is DHCP. Cable internet mostly uses DCHP. Next go  into the same System> Admin> Networking tools. See if detects the devices.
