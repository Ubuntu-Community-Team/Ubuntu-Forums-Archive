### Post by kpkeerthi on 2008-02-21
1. Grab [envy]("http://albertomilone.com/nvidia_scripts1.html") and install the drivers using it. 
2. Restart and when you get to the desktop, open a terminal and type
```

gksudo nvidia-settings

```
3. Click **Detect Displays** and set the twin view mode (clone/right-of/left-of)
4. To make the changes permanent, click on **Save to Xorg.conf**
