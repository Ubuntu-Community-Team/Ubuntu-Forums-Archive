### Post by lespaul_rentals on 2007-11-26
Linux is a *kernel*, a sort of language your computer uses to communicate with devices and applications and the end user.  It was created by Linus Torvalds and furthered by thousands of hackers worldwide.

Ubuntu is a *distro*, a set of applications, files, interfaces, and enviroments delivered with the Linux kernel.  Ubuntu is a Linux distro, meaning that while it is Linux, it is just a user-friendly way of spreading it to the masses for practical purposes.

Some of the hackers and programmers who work on the Linux kernel also work on the Ubuntu distro.

Ubuntu is a form of Linux.
Linux is not Ubuntu.

### Post by jfinkels on 2007-11-26
"Linux" is the name of the kernel (the code that allows software programs to interface with hardware devices) used by each distribution of the operating system called "GNU/Linux".  Most of the time, we shorten the name of the operating system "GNU/Linux" to just plain "Linux" which is somewhat improper, but much more convenient to say/type. Ubuntu is one distribution, Fedora is another, SuSE is another, Debian is another, Slackware is another.... The list goes on and on, take a look at [http://www.distrowatch.com/](http://www.distrowatch.com/) for some info on some other distributions.

and here [https://help.ubuntu.com/7.10/about-ubuntu/C/about-gnu.html](https://help.ubuntu.com/7.10/about-ubuntu/C/about-gnu.html)
