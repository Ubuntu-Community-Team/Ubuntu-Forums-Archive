### Post by beesthorpe on 2007-10-26
When you unpacked the theme the folders "GTK 2.0",  "medacity-1" and the file "index.theme" should all be in a folder named "Deep jonny". Simply move this folder to the .themes folder in your home directory (you'll need to enable viewing hidden folders in Thunar" and I think the theme should simply show up in the theme manager.
