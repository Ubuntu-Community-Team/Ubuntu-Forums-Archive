### Post by PaulFXH on 2007-02-01
I use a UK English 105-key keyboard and type in both English and Portuguese. The keyboard indicator makes it very simple to switch from one keyboard layout to another.

However, although all of the different letter characters (mostly accented letters) are available I am unable to locate the "question mark" in the Portuguese layout.
In Windows, this symbol comes up with Ctrl-Alt-w but this does nothing in Ubuntu.

I believe I have tried nearly every conceivable combination of keys but without success.

I have also tried to use the On-Screen Keyboard (from Applications>Accessibility) believing that it might show what each key or key combiantion represents, but I can't get this keyboard to display itself on the desktop although its icon appears.
I downloaded the Gnome OnScreen Keyboard (GOK) which I believe is the same thing but when I attempted to install this I got an error message saying " Error: Dependency is not satisfiable: libatk1.0-0"

Just maybe somebody out there knows the answer to the missing question mark dilemma (other than switching back to English to type this symbol).
