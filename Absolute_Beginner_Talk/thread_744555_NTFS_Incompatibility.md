### Post by medley on 2008-04-03
I've used a utility called ntfs-config to enable writing to external HDs. Both of mine are NTFS using USB2.0 interface.

Just search for 'ntfs' in your package manager.

### Post by Tteddo on 2008-04-03
In Synaptic, install ntfs-3g and then you can read/write to ntfs volumes.
