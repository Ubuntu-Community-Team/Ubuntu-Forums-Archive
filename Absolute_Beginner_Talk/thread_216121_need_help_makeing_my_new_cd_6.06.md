### Post by ciscosurfer on 2006-07-15
In Nautilus, under the Go menu, there is an option called "CD/DVD Creator". Simply drag the .iso into the window, follow the prompts, et voila!  It should create your disc no problem.  Then boot from it.  You can try to set the write speed lower as well.

### Post by Daniel15 on 2006-07-15
Make sure you burnt the ISO image to the CD, and didn't just put the ISO *file* onto the CD (ie, so the CD only has one file). In Nero, for example, you'd open the ISO image, and choose to burn it.
