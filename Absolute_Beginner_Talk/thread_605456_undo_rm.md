### Post by yabbadabbadont on 2007-11-07
If you used the rm command in a terminal window or at the text console, then the answer is, "not really."  Don't do it unless you are sure.  ;)

If you happened to shutdown the system immediately, and then boot using a specialized CD with recovery software, then you *might* be able to recover something.  See the following:
