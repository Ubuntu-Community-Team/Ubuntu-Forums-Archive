### Post by dacool25 on 2007-04-15
The entire virtual machine gets saved as one big file on your hard drive (the file varies from what kind of vitrulization that you're using).  There are a couple ways of transferring the data through operating systems, one would be a usb stick that both os's support, or if you're using VMware then you can install VMtools and it will set up a wireless connection so that you can see your ubuntu box as a part of your network.

Hope that helps.
