### Post by rich.bradshaw on 2007-05-12
If you boot from the CD it doesn't touch your harddrive, unless you install.

If you don't install (which you can't really do my accident!) then it is very simple, just put the CD in, maybe change the BIOS to boot from CD first then it will work.

Yes, you can boot it on any PC, as long as they are set to boot from CD first. College PCs might have a BIOS password which you will need to allow them to boot from CD.

If you install then you get a menu on boot to choose Windows or Ubuntu.

when you first start your computer you should see a prompt to enter setup by pressing a certain key. for example delete, F1 etc. if you don't see any options hold down the esc key and reboot the computer while holding down the key. and you'll get a keyboard error and should be prompted to press (usually) the F1 key to enter set up. once in the bios look for a boot tab. then you can change the boot order.
