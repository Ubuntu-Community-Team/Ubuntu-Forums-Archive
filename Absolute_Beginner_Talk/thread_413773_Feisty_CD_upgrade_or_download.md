### Post by PartisanEntity on 2007-04-19
In order to do a clean install, download the cd image.

In order to do an upgrade, use the Update Manager which is to be found under System > Administrator.

Make a backup of all your personal files before you try either method.

As far as I know, you cannot use the cd image to upgrade.

p.s. If you go for the upgrade method then be aware than thousands (soon to include me too) are doing so right now and you might get some errors about not being able to fetch files due to server overload, this should abate (second time I use this word today) over the coming days.
