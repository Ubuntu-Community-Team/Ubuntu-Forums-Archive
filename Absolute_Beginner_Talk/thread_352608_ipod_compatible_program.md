### Post by kebes on 2007-02-03
For linux you can use gtkpod or Amarok:
[http://www.gtkpod.org/about.html](http://www.gtkpod.org/about.html)
[http://amarok.kde.org/](http://amarok.kde.org/)

There are probably other programs, too, but those are the ones I know about. I don't have an iPod myself, but I've been told that getting an iPod working on Linux is not a problem. Amarok is a fantastic music player/organizer. More information on Amarok's iPod compatibility can be found here:
[http://amarok.kde.org/wiki/Media_Device:IPod](http://amarok.kde.org/wiki/Media_Device:IPod)


I hope that helps. I hope you enjoy playing with Ubuntu! Remember that you can boot a LiveCD of Ubuntu on your computer and see if things work, and if you like the way things run. The "kubuntu" variant of Ubuntu should have Amarok installed by default.

Good luck.
