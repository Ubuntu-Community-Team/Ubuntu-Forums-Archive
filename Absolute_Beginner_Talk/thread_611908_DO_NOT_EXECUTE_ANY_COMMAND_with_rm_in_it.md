### Post by aysiu on 2007-11-13
Not that anyone reads stickies, but for the few who do, this advice would be beneficial:

The command ```
rm
``` stands for *remove*. The command ```
sudo rm
``` means *remove with administrative privileges*. The command ```
sudo rm -rf
``` means *remove recursively (files within folders and folders within folders) with administrative privileges*. The command ```
sudo rm -rf /
``` means *Delete my entire hard drive*. The command ```
sudo rm -rf /home
``` means *Delete all user data, including personal files*.

**Do not run these commands.**

More importantly, if you ever doubt whether a command should be run or not, wait for at least one or two people to respond validating the command **or** Google the command to see what it does.
