### Post by Kal9001 on 2007-05-13
What hes suggesting you so is edit your [xorg.conf] file. This file is the configuration file for the X-server which is the brains behind the beauty of the Gnome GUI

To do this you need to enter terminal, once you have you should see:-

```
[username@yourpc]$_
```

Type 

```
sudo gedit /etc/X11/xorg.conf [return]
```

It will then prompt for **your** password, enter this and hit return again, now the "gedit" program should run and open the said document.

Once you have finished editing then you just save it and restart the X-server. if you dont know how to do that just restart the whole PC. 

Of course its advisable to create a backup of the unedited xorg file. to do this once you open it, before you alter anything just goto "save as" and call it "xorg_backup" or something, this can then be restored if something goes wrong during the next boot.

I'm nowhere near good enough to know WHAT you need to edit so ether look through the other post and see what looks relevant or hopefully someone else or tgalati4 will reply and highlight the important parts that need to be changed.

[I]/* Its also worth noting that this is how you open most if not all system config files so get used to it :D 
The commands you use there are 
**sudo** (superuser do) this lets you become the system administrator to preform the next instructions, you must do this as the file your will be editing in this and most other duties of this type are owned by the root user
next is **gedit** this is the part that says what you want the superuser to do, Gedit is simmaler to wordpad.exe in windows.
edit needs to know what file you want to open in this case it is **/etc/X11/xorg.conf** and is case sensitive. */[/I]
Sorry if that last bit was a bit OTT but you did say you wanted to know all the little bits. XD Oh yea you will need to turn the PC on 1st and log in. Did I mention you have to buy a PC and install Linux 1st...ok maybe too far :lolflag:
