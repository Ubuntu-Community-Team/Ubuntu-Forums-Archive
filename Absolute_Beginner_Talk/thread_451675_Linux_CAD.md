### Post by Peter76 on 2007-05-23
I'd suggest you make a dual boot system as suggested above and see how it suits your needs. Linux gives you a lot of freedom. but you should try it to see if the software provided suffices your needs... Also count on a bit of learning curve as things behave differently under Linux and also you need to adapt to new programs.

As for the programs you need; I think Bricscad is the only good Cad alternative, though not open source at the moment for Linux. As for the other programs you mention ( if you haven't found out already ) I dug up the following ( but don't know if it comes close to what you need ):

Photoshop   Gimp
Rhinoceros   Take a look at k3d and blender 
Maxwell       There are ( I think ) quite some high end renderers available for Linux. k3d advises aqsis

If you can get by with these programs and you take some time to make the switch, you'll probably end up with a very stable, virus free and cheap os...

Good luck
