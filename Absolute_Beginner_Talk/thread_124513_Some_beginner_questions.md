### Post by sprok8 on 2006-02-01
If you manage to set up a dual-boot machine, transferring files from Windows to Linux is straightforward. You would simply mount the Windows partition under Linux, and copy them over.

A word of caution if you plan to transfer files from Linux to Windows... Depending on your Windows filesystem, it may be problematic. FAT32 works best, although NTFS support is improving all the time.
