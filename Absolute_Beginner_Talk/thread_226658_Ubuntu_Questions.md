### Post by az on 2006-07-31
Ubuntu takes only a small subset of debian Sid packages and officially supports them with biug fixes and security updates.  It's a lot different than running Sid, since it will not break.

The rest of the packages are supported by the community and ususally work really well.

Ubuntu also distributes some binary-only drivers (nvidia, ati, some wireless drivers) which are not GPLed.  These are removed easily by getting rid of linux-restricted-modules* packages.

Ubuntu's community also has a different personality than Debian's.
