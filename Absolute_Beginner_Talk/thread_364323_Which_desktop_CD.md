### Post by xopher on 2007-02-18
Id still recommend the x86 one, for new users. The 64bit one is getting along quite nicely, but there are still things that you really need to make workarounds for. Eg. Flash, Java, Wine etc.

So until you feel 'as one' with linux, Id say go with the x86 version. The performance improvement is only marginal, nothing you'd notice.
