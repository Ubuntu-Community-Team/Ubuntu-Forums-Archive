### Post by seshomaru samma on 2007-05-19
to install it - go to the sourceforge site
make sure you download the tar.gz file
if you downloaded it to the desktop go
```
cd Desktop
```
then
```
 sudo cp audio-convert-0.3.1.1.tar.gz /usr/bin
```
then cd to the usr/bin directory like this:
```
cd /usr/bin
```
then untar it 
```
tar xvfz  audio-convert-0.3.1.1.tar.gz
```
(or you can use nautilus to browse there and just click on it)
then > 
cd  audio-convert-0.3.1.1
then
```
 audio-convert-install 
```

### Post by mystictim on 2007-12-19
**This post could be related to an Ubuntu bug filed at**: [https://bugs.launchpad.net/ubuntu/+source/audio-convert/+bug/130055](https://bugs.launchpad.net/ubuntu/+source/audio-convert/+bug/130055) [br].[/br] ---------------------------- [br].[/br] [br].[/br]
				To get the audio-convert script working first make sure you have all the convention software and tagging tools you might need installed (you can always add others later):

** lame ** to encoding mp3 files
**libid3 (libid3-3.8.3 and lib3-3.8.3-dev)** to tag mp3 files
**vorbis-tools** to encode and tag vorbis music files
**flac** to encode and tag flac files
**faac** to handle aac and mp4 files
**faad** 


Then install the audio-convert script. Which is now available via Synaptic (or your preferred package management solution) with the Universe repository enabled. The package is called 

nautilus-script-audio-convert.

Unfortunately as of version 0.3.1.1 the script isn't activated by default.  You will need to take the following steps to get it working. Open up a terminal and type in

     nautilus-script-manager enable ConvertAudioFile

To convert files in Nautilus select them and then either  from the File or right click menu select Scripts and click the ConvertAudioFile option and follow the instructions.
