### Post by scxtt on 2007-09-08
you don't have any swap defined, looks like sda5 was supposed to be, but its Id is 83 (Linux) instead of 82 (Linux swap) ... not to hard to fix, you can probably use Gparted, i'd use **sudo fdisk /dev/sda** myself, but that's just me :p ...

then once sda5 is swap, you can do **sudo swapon /dev/sda5** and add an entry to fstab ...

### Post by scxtt on 2007-09-09
if you made /dev/sda5 a swap partition, you need to add it to fstab.
```

/dev/sda5   none            swap    sw              0       0
```
