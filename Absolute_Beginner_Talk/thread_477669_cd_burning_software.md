### Post by lambros on 2007-06-18
There are lots of these, but the one I use is called GnomeBaker.  You need to enable the Universe repository if you haven't done that already.  Then you can install it using Synaptic, or from the command-line:
```
sudo aptitude install gnomebaker

```
