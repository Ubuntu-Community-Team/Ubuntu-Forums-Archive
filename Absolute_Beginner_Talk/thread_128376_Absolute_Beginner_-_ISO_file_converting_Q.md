### Post by Estariel on 2006-02-11
if you see only one file on the CD, you probably burned the ISO file on your CD.
This is not entirely correct. The ISO is an archive, that contains instructions on how the CD is to be burned in addition to all the files that will go on the CD. 
However, you are not supposed to "unzip" it yourself, Nero will do that for you.
If you go through the instructions again, you will see that at some point, you are supposed to select an Option called "Use ISO to create CD" (or similar, it has been ages that I have used a windows computer...). If you select that option, everything will be ok.
