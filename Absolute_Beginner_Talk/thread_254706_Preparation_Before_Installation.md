### Post by bodhi.zazen on 2006-09-10
General advice:

1. Back up any data from Windows. It is unlikely you will trash your windows install, but all things are possible.

2. Remove any unnecessary software/data from your windows install.

3. Defragment your hard drive.

4. Boot Ubuntu (live CD).

5. Install Ubuntu. Go for the defaults.

6. Boot Ubuntu (HD).

7. Learn Ubuntu.

8. Delete Windows partition (He he).

If you have a specific question or problem, post. Overall it is smooth sailing for most. 8)
