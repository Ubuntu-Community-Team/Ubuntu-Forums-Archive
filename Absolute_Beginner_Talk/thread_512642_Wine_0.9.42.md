### Post by SirGrant on 2007-07-29
I say just wait for the repositories unless you absolutely need to update to the latest and greatest version for some reason.  I've been using the wine repository for some time now and I usually see the day a new version of wine is released (on digg.com) and then like 2-4 days later it automatically updates in the repository.  So there is a slight delay between the updates but I mean really unless you need to update I just find it easier to wait a few days and have it update automatically for me.  

The delay is usually because when they announce the release they have released the source code but haven't released the binaries for it so it takes a while for them to actually get out there in the public.
