### Post by Bartender on 2006-06-17
Don't feel too bad, this is a never-ending source of confusion.  The d/l is an .iso.  What you want to do is convert that one big huge .iso file into a bootable CD that has several folders and files on it.  Toss the CD you made into your Windows PC and explore it.  If you see one huge file you did it wrong.  If you see 7 or 8 folders and some files you did it right, barring any questions about loss of data because of high burn speeds, cheap media, etc.  
There are so many different kinds of burn software out there it's impossible to give directions.  The NTI software that came with my Sony DVD drive recognized the .iso and did the conversion without even asking.  
Lots of folks have recommended a free d/l that will burn a bootable disc, but I don't remember the name off the top of my head.  A little searching of the last few days or so oughta turn something up.
