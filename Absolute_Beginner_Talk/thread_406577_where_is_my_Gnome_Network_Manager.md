### Post by RomeReactor on 2007-04-11
Hi. If this is not **network-manager**, but the panel applet, then right-click on the top panel and select "Add to panel" (or something similar); now scroll down the list of applets offered to find gnome network manager. If it *is* network-manager, then I suggest you reinstall it.
