### Post by FreewareFan on 2008-03-17
Well, the only ones that I've learned of so far would be the Super Grub cd, which gets you into Windows or Linux if something happens to your Grub, and the other is called Linux System Rescue 0.4.3, which has a lot of features that you might be looking for..  There's also one called SystemRescueCD, which has many tools on it for use with both windows and linux.
