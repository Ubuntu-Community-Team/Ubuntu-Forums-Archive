### Post by K.Mandla on 2006-12-20
You should have a hidden file in your home directory called .xinitrc or .xsession. Either one should work. If you edit that file, like this:

```
nano -w .xinitrc
```
you should be able to add this line

```
conky &
```
somewhere in the middle. If there's nothing else in that file, that's okay. Just add it like that.

That should trigger conky when X starts up, and keep it running while you're using the desktop.
