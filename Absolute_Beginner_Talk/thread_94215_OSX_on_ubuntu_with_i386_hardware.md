### Post by endy on 2005-11-23
The only way I know of getting OS X to work in Ubuntu is with PearPC which is fabulous.

It's a virtual machine emulator like VMWare but it emulates the PPC architecture instead of the x86 architecture. It's free, the website is here:

[http://pearpc.sourceforge.net/]("http://pearpc.sourceforge.net/")

You can install it through Synaptic or run the following command line:

```
sudo apt-get install pearpc
```
