### Post by IYY on 2006-04-11
The most important command you need to learn is "man". It is used to get information about other commands. For example, you said that you don't know how to use and format apt-get. No problem! Just type "man apt-get" and you'll get the answer.

### Post by therunnyman on 2006-04-11
You're in good shape, no worries (and you may have every command except apt-get already in that book).  "apt-get" is the get command for the Aptitude package manager; if you need a program from the repositories, you'd type "sudo [unless you're root] apt-get [the name of what you want]."  I think there's a search too, "apt-search."

In any case, Synaptic is the GUI for that.  System-->Administration-->Synaptic.
