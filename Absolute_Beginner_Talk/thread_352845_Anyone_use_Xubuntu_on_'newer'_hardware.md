### Post by jml on 2007-02-03
If speed is most important to you, check out this site:

[http://fluxbuntu.org/](http://fluxbuntu.org/)

This distribution is based on Ubuntu, but uses Fluxbox as its desktop environment.  It is even less resource intensive than XFCE (the desktop for Xubuntu.) So it should run even faster than Xubuntu.  Two warnings,  first, this distro is not supported by Cannonical/Ubuntu so if you choose this distro, you have fewer resources to help you.  Second, the web site warns that this distro is not for regular production environments, ie not ready for prime time.  So use it at your own risk. 

Personally, if speed is a major concern for you, and you want to stick with the Ubuntu family of distros, then use Xubuntu.
