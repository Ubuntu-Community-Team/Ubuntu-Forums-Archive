### Post by Kvark on 2005-11-27
To search on websites or p2p networks is an old fashioned way to get programs. Try this instead:

1. from the system menu in the panel open: System -> Administration -> Synaptic Package Manager.
2. Look in the sections or search for keywords to get a list of programs that might do what you're looking for.
3. Click on a program in the list to get more information about it.
4. Mark checkboxes next to the programs you want.
5. Click Apply.
6. Run your new programs from the menu and start using them.

I bet you will agree after trying this way a couple times that it is much easier then to look for programs on websites.


But first follow the instructions [here](http://www.psychocats.net/linux/sources.php) to configure Synaptic to be able to find many more programs then it does by default.
