### Post by podunk on 2006-10-03
If you need to set up your network go here:
[http://ubuntuguide.org/wiki/Dapper](http://ubuntuguide.org/wiki/Dapper)

Once the network is up all you should have to do is on the CD computer
Open Nautilus
Navigate to /media
Right click the CDROM and chose Share Folder
Select SMB and read only
