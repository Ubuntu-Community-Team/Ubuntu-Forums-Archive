### Post by jckdnk111 on 2006-09-28
Although not exactly native, I finally found a good one.
MyEclipse has excellent html editing (among many other things).

It costs $30 for the standard edition and $50 for the professional and you get a great wysiwyg editor that understands jsp tags, javascript, and css and has probably the best property editor of any ide I've ever used. It has nice code snippets, split screen view (half code - half design view), inline preview using whatever browser you want, and a whole host of other very nice features.

It also comes with a database explorer, j2EE support, web services support, and more ... and it is, afterall, built on top of eclipse (3.2 compatible) so you can extend it 'till you're blue in the face using the countless eclipse plugins.

The pro version comes with additional oracle and SQL server support, an image editor, a nice UML app., javascript debugger, and a visual swing designer.

This is one of the best tools I've ever seen especially considering it is only $50 a year (yeah, it sucks that it's subscription based, but $50 isn't that much considering everything you get). 

I hate paying for software when there is so much excellent OSS out there, but I was at the end of my rope. Until a nice wysiwyg editor is released I'll stick with MyEclipse.
