### Post by v8YKxgHe on 2007-03-17
The thing with a release based distro like Ubuntu (not really sure what to call it) is that things like that, major updates of programs, can not (officially) be updated. For example users of Dapper still have Firefox 1.5 even though Firefox 2 is out, because that was the latest at the time. There are of course ways to get Firefox 2 on Dapper, but it will never (afaik) come from the official Ubuntu Repositories. 

The same goes for Gnome 2.18, you wont be able to update that through the official repos, so unless you can find an un-official repository to add and download from, the only time you'll be seeing/using Gnome 2.18 is by compiling it you're self (not recommended) or waiting untill April 19th when Feisty is released, which comes by default with Gnome 2.18.

Yes Firefox is just an application, just like Gnome is. There will most likely be very easy ways to get Firefox 3 on Ubuntu Edgy once released, well you could even just go to Firefox's website and download it from there, or wait for someone to create their own repository which you can download from. 

Ubuntu doesn't update release's with major application updates, because it means they would have to do a lot more testing to make sure everything is stable etc, so instead they include what was the latest software into the release and then will only do security/bug updates, but nothing major like Firefox 2 to Firefox 3.

But like I said if you wait for Feisty which is released April 19th, you will have Gnome 2.18 and every other application of the latest (stable) version.
