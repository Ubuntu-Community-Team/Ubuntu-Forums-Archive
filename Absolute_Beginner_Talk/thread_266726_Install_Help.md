### Post by b_martinez on 2006-09-27
First thing, load the cd and have it do the self check. If it passes, you (and we) will know that it isn't the CD. Then we can start to diagnose the system to try to get it working for you. When you re-boot the box, check to make sure the hd is not write protected. You need to get into the BIOS to do that. Look for a small message that will read something similar to "press F2 to enter setup" or "press delete to enter bios".
It may be password protected (either/both the BIOS/hd write protection so you may need to go to the manufacturer's site to learn how to re-set the BIOS.
 Post back the results.
