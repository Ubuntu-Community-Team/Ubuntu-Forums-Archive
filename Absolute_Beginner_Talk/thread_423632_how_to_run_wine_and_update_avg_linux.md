### Post by dubrict on 2007-04-26
if you want to run wine, just double click on an .exe file and it will open it.  If it doesn't, then right-click on it and hit "open with" and type in "wine."  You can also run the command "winecfg" to set some options for it.

I wouldn't even worry about the antivirus program.
