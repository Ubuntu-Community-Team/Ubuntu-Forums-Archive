### Post by Jagot on 2006-06-30
Programs that come as .bin's often have uninstallers with them.  If you navigate to the netbeans directory there should be a subdirectory called something along the lines of '_uninst'.  Then in terminal you should be able to run the uninstaller by typing something along the lines of './uninstaller'.
