### Post by duckville on 2007-10-21
Alot depends on whtat you are actually trying to do.... more info needed...

As mentioned prior, port forwarding is normally an application specific option, not OS specific.
e.g. changing ports for ssh access on a SSH Server...
to do that you'll normally edit the "sshd_config" file in "/etc/ssh/".

You can do port forwarding on a router, so that incomming traffic from one port
goes to another port on a defined PC on the network; but that's not Ubuntu.
