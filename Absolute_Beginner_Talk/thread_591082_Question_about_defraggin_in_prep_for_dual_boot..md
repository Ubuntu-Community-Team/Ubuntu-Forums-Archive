### Post by af20001 on 2007-10-25
What you want when defragging in preperation for dual boot is as much free space as possible - this means you want to see all your current files displayed in the defragger as far to the left as possible. The Linux partition will be created after the last windows file, so if there are substantial gaps between current files, it essentially means your Linux partition is going to be smaller than it could be.

For what it's worth, it took about a dozen or so goes with the windows defragger to properly defragment my drive. The windows defragger doesn't seem to be particularly aggressive.
