### Post by GreyFox503 on 2006-02-06
I run America's Army and UT2004 on my Ubuntu installation.  They both have native linux versions (no WINE required).

To run AA, you just download their linux version of the game and install.

If you already own UT2004, that's all you need to run it in Ubuntu!  I bought Unreal Tournament before I switched to Ubuntu (I used Windows only).  After I installed Ubuntu, I took the exact same UT discs and installed it in Ubuntu.  I don't remember exactly how, but just insert the first UT disc, and in the root directory of the CD there is an installer shell script to run.  Just make sure you have 3D acceleration working before you install and you should be good.


I tried True Combat: Elite, but I have to say I was disappointed.  It *might* compete with CS 1.6, but CS: Source blows it away.  Still no Counter-Strike or replacement on linux.

### Post by GreyFox503 on 2006-02-07
Having 3D acceleration means having proper drivers installed for your graphics card.  I would follow this guide:

[https://wiki.ubuntu.com/BinaryDriverHowto](https://wiki.ubuntu.com/BinaryDriverHowto)

An easy way to test it is to look at a 3d screensaver or run 'glxgears'.

If that method doesn't work, you can still try downloading the drivers directly from ATI/Nvidia.  Nvidia actually has a good installer for their drivers.  Never used ATI before.
