### Post by Kulgan on 2006-11-25
in firefox, the image should be saved in your home folder as "Firefox_IMAGENAME". Have a look there, and if it isn't, save it there and put it in the list in "System > Preferences > Desktop Background".

The only disadvantage is that you can't delete it from your home folder, then, so you might want to save it as a hidden file (just put a "." in front of the filename.) and import it into the list again.
