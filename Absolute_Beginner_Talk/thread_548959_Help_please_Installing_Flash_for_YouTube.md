### Post by Maestro23 on 2007-09-11
Two ways:

From Gui (Synaptic):

> System-->Administration-->Synaptic Package Manager-->Search-->flashplugin-nonfree-->Mark for installation

Terminal (Command Line):

> sudo apt-get install flashplugin-nonfree

Welcome to Ubuntu.  By the way, there is a link in my sig to the Feisty 7.04 Starter Guide.  Valuable source of info.
