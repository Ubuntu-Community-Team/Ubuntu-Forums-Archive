### Post by overcast on 2005-07-02
Hi there,.I would like to discuss many distros with packages they offer,their target audience etc.just overview for most of the new users.I actually have 8 Linux Distros /& /FreeBSD/Darwin under my VMware software.i will list few of them.Well i m not ubuntu user(Secret: CD is under SHip-it list).

Let's start with distros:-

1)  Knoppix:-This is the first Live CD i heard over open source communities.Offers more packages than any other distro.Have software for developers,gamers,internet users.Default window manager is KDE by default.
You can install Knoppix on HD.Good start for newbies,developers as Kdeveloper,QT is included.

Audience>>For Demos,Presentations,Developers

2) Mepis:- Live CD is installable.You can first boot with the CD. then after getting desktop you can have install option.under mepis OS center.You can use Kernel 2.4.2 or 2.6.It is debian based distro.KDE Window manager is there by default.Very few packages for developers only python,tcl-tk is there.DTP users can have SCribus,Gimp,Openoffice packages.Most of the application are suitable for internet users.

Audience>>For Demo,New users,Internet Cafes

Before moving on to any other Distro,you can ask me any questions if you have.I will post more info under this topic as soon as  compile the information into short Paragraph.
