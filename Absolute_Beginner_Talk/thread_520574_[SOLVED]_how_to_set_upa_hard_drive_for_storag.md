you have to replace the Xd*? with the partition name something like hda1 or sdb2 or whichever partition that is you are trying to mount.

X is always h (for ide drives) or s (for scsi drives) 
a,b,c are the drives that you have on your machine. if you have just one it will be a, a second one will be b and so on.
1,2,3..is the partition number on that particular drive that you are trying to mount. for eg. drive a might have 4 partitions on it  so you will see hdb1 hdb2 hdb3 and hdb4

i hope that makes it more clear.

but the Read only file system error tha tyou get, try to connect the drive to windows and do the safely remove hardware thingy. i think windows still has a lock on the drive which might also be a cause for you to not be able to mount it properly.
