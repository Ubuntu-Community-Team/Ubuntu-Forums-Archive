---

### Post by Nano Geek on 2007-11-20
To do that it looks like you would have to edit the gtkrc file for that theme. You can find that file by going to** /usr/share/gdm/themes/*ThemeName***.

Be warned however that you could potentially mess up your GDM and it's always best to make a backup copy before you start.
