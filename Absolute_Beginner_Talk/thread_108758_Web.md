In Linux, you have a software called NVU to design webpages, and you can FTP them to your webspace directly. To install NVU, open Applications--> Add applications from the menu, and search for nvu. Then check the box in front of nvu's name and click APPLY. Nvu will appear in the Applications--> Programming--> Nvu menu
