### Post by DizzyTech on 2007-08-02
You said it's a U3 drive, right?  You can't use U3 itself in Linux (I'm assuming the password system is part of U3), however, if you backup your files and use the U3 removal tool, you can use it in Ubuntu.

You see, here's how U3 works:  The main partition of your U3 drive is represented as a CD drive (read-only), which in Windows launches the U3 system and mounts the second partition where all your files are.

If you're planning on a dual boot, I have some tips for you when (or if) you install Ubuntu:
Install ntfs-config via the Synaptic Package Manager, and then use it.  You can access your Windows partition in Ubuntu.  I symbolically linked my Documents, Music, Pictures, Videos directories in my home directory to my Windows stuff.  Smooth.  Also, use the EXT3 driver for windows:  Visit [http://www.fs-driver.org/](http://www.fs-driver.org/)
