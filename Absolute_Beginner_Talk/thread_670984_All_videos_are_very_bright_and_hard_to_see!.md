### Post by anaconda on 2008-01-18
I had this problem in ubuntu 6.06.. And I noticed that there were differences between movieplayers.. Try different players.. Totem, vlc, mplayer, kaffeine. Some of them had better default settings than others..

And you could also adjust the brightness of totem (the default player).
go to Edit>Preferences and select "display" tab. You can adjust brightness etc. from there. 

The drawback is that you will have to adjust the settings every time you restart totem. If I remember correctly mplayer had the best default settings..
