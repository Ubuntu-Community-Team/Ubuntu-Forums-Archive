### Post by TenPlus1 on 2008-03-30
VLC is a media player, not Vince which controls desktops over a network...

Their is a repo address in the VLC developer's page that points to the latest version of VLC, but I dont really know if it works 100% with Gutsy since it's aimed more at Hardy...

For Ubuntu Hardy I386 add the following line to your sources.list:

deb [http://nightlies.videolan.org/build/hardy-i386/arch](http://nightlies.videolan.org/build/hardy-i386/arch) ./
