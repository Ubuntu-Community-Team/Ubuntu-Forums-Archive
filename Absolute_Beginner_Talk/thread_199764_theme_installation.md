### Post by someusernoob on 2006-06-19
Copy the extracted folder into /home/your_name/.themes (its invisible, make it visible by pressing CTRL+H in nautilus file manager) That should just work fine, but it only works for your account then (if you use multiple accounts). Otherwise you have to copy the folder to /usr/share/themes .
