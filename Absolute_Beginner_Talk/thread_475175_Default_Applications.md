how do we make an application the default one for a particular file?
For example torrent files are opened by "bit torrent" by default, but I have "Azureus" installed and using that. How do I make torrent files open in Azureus by default? (is Azureus the best option for a GUI program in Ubuntu?)

### Post by machoo02 on 2007-06-15
Right-click on the .torrent file, and select 'Properties'.  Click on the "Open With" tab, and if Azereus is not on the list, click on the 'Add' button and select it from there.
