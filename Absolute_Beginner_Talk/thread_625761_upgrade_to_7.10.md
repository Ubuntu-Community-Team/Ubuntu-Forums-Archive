If you backup all contents of your home directory (including all the hidden files) that will take care of your personal settings for programs. But there is no easy way to backup the programs.

If you really can't use the internet upgrade you should get the **Alternate CD** for Ubuntu, you can use that one to do the upgrade.

But I'd rather try to get the Internet upgrade working as the Alternate CD of course doesn't have the latest versions for all the software you have installed in addition to the default Ubuntu desktop..
