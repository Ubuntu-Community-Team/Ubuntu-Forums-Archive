### Post by RomeReactor on 2008-01-19
Hi. By default, the BitTorrent client included in Ubuntu does not show up in the menus since it handles only one download; to start it, double click on a **.torrent** file and it will pop up asking you for a location to save the download. If you close the program before the file is complete, you can resume the download by double clicking the same .torrent file again.

You can open multiple clients if you want to download more than one file at a time, but for that it's better to use another client such as [Deluge]("http://deluge-torrent.org/").

### Post by Thund3rstruck on 2008-01-20
> **jwkungfu said:**
> .
I used to use Limewire with Windows which I guess is like BitTorent?.

No, not at all. Limewire is a file-sharing application based on the gnutella standard used primarily for trading movies, music, and applications (but mostly for propogating virus infested files across the Internet). 

BitTorrent is a file-sharing protocol which defines communication paths among distributed swarms of computers. There is no front-end user interface for BitTorrent for finding files. You must locate the torrent tracker file and load that file into your client. Popular client include Azuereus, kTorrent, and Deluge.
