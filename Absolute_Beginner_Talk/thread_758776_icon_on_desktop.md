### Post by CJ56 on 2008-04-18
Sounds like you might have added a launcher by mistake by right-clicking on a bit of empty desktop..

If you right-click, you'll see a command Add Launcher. If you don't want it, just delete it, it won't do any harm -

(Launchers are quite handy things, though, and you'll probably be using one in earnest before too long...;))
