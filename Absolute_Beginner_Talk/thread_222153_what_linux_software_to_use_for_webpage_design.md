### Post by SilverBear on 2006-07-24
Quanta Plus is what I seem to prefer using. The interface is fairly straightforward and customizable. But there is no WYSIWYG feature like in NVU.

I used to use "HomeSite" in Windows, and Quanta Plus has an interface not too different from what I was used to.

I know you said you don't want to waste time learning different progs, but there is no better way than to try them out on a few simple pages and see if one clicks. Or the opposite -- if you find one that is irritating after a few minutes, you can scratch it off your list.

I used Screem for a while last year and thought it was OK also. It has a built-in CSS editing feature that is good if you know what you're doing.  I'd recommend trying it along with Quanta Plus, and see which you are in sync with.

all the best,
SilverBear
