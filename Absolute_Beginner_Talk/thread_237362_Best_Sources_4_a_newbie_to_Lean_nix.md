### Post by andb on 2006-08-16
Everyone well knows that the beginning is the hardest... It all falls into place once you've begun to learn whats going on under the hood. The forums are great for specific questions, but docs are better for building general knowledge. 

For these reasons, Id like to ask, which resources where key in getting you started, and what would you recommend to a newbie? 

Ill update this first post with your contributions so as to build a comprehensive list, then later hold a poll for the best newbie learning source.

I'll start of with a few:

**Comprehensive Newbie Learning Resources**

**Learn Linux Project**
A project that aims to develop Free/Open Source GNU/Linux Training Materials. The project initiative is supported by The Shuttleworth Foundation (founded by the same founder of Ubuntu) under the direction of the Open Source Team. The project also uses Moodle, which is an interactive training environment that provides students with the opportunity to access the Learn Linux training materials in a virtual classroom environment.
[http://learnlinux.tsf.org.za/](http://learnlinux.tsf.org.za/)

**The Linux Documentation Project**
LDP is a loosely knit team of volunteers who provide documentation for many aspects of Linux. There are several forms of documentation: Guides, HOWTOs, man pages, and FAQs. The number of documents can be overwhelming. Be careful, some documents are out of date.
[http://tldp.org/](http://tldp.org/)

**Rute User's Tutorial and Exposition**
This book covers GNU/LINUX system administration, for popular distributions like RedHat and Debian, as a tutorial for new users and a reference for advanced administrators. 
[http://rute.2038bug.com/index.html.gz](http://rute.2038bug.com/index.html.gz)


Please list comprehensive sources, not focused how-tos about installing, shells, X, etc...
