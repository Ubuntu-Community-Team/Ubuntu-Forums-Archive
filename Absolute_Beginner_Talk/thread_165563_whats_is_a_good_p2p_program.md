### Post by Bloch on 2006-04-24
Hi proboller, 
Frostwire is really good, but demands a lot of resources. On my friends 256MB ram machine I had to give up using it.
I am using gtk-gnutella at the moment. The interface is totally geeky, there's no proper guide to it, the FAQ is laughable, but it works straight out of the box.
This is one exception where you are better off getting the latest version - it has a built-in list of up-to-date servers. The repository version might not work.
Do a google and find the file below from the gtk-gnutella site.

GTK1_gtk-gnutella_0.96.1-0_i386.deb

Put it in your home folder. Open a terminal and type ls (means list) to check that the file is there in that directopry. Then enter
dpkg -i GTK1_gtk-gnutella_0.96.1-0_i386.deb

and you're off.
