### Post by Tyche on 2008-01-19
> **Elf Wizard said:**
> [COLOR=Navy][B]Hi! :)

[/B]Is it possible, to delete some files (or entire directories) directly, instead of sending them to the Thrash?[/COLOR]

If you are running Gnome, open Nautilus (the file manager) and go to Edit -> Preferences.  A window will pop up with the File Manager Preferences.  On the second tab, "Behavior", go to the bottom and select "Include a delete command that bypasses trash".  Then, when you click on a file or folder to remove them, you can select "Delete".  A warning box will come up advising you that this will permanently remove the file/folder.  Hit "Delete", and the file/folder will be gone.

Hope this helps.
