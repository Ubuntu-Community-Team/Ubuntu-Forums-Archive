### Post by trak87 on 2007-05-22
Pretty much any cd burning program you use will burn in a format that will be viewable in MS Windows. Genreally Rock Ridge and Joliet extensions are used and Windows reads that fine. If you insert a blank disk in Ubuntu, you'll get a pop up asking if you want to burn a disk. Choose it and insert the files you want to burn. Then burn it.

### Post by eldepeche on 2007-05-22
