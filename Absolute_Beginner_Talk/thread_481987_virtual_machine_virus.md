### Post by maddot on 2007-06-23
Is it possible for a virtual machine to get a virus? I'm curious, as since linux is technically "virus proof", so should i take the effort to install some fancy anti-virus into a virtual machine and update the windows?

---

### Post by Golyadkin on 2007-06-23
Yes, a Windows installation in a virtual machine can get infected just as well as a real "physical" Windows installation. However, the virus will be contained within the virtual machine and won't infect your Ubuntu installation or it's files. I suggest therefore that you update your Windows and run a lightweight antivirus like NOD32 to keep it clean. You wouldn't want to have a virus in your virtual machine :)

### Post by hoges on 2007-06-23
As Golyadkin said, it's just as suceptible to malware as a normal windows installation BUT it really depends on what you're using it for. If you use linux for all the online/network stuff and the virtual machine stays offline, than there's little chance of it catching anything... I'd put one on, but get a free one like avg if you don't have access to corporate editions (eg free employee licences). It's not worth paying for if it's not for regular use.
