### Post by alienexplorers on 2007-09-17
The version of Ubuntu you download would be in an iso file that needs to be burnt into a bootable disk.  Never heard of just having it download and install all at once.

The disk should be burnt at no more than 4x and you should verify the MD5 checksum.

### Post by Maestro23 on 2007-09-17
> **ES_EF_EX said:**
> I'm currently running Puppy Linux.  Is it possible for me to download Ubuntu straight to my hard drive?

You can download the ISO and then burn it to disk to create the LIVE CD.  You can then run the LIVE CD from your drive to see how you like Ubuntu and if it likes all your hardware.  You then can install to the hard drive from the CD as well when you get ready.
