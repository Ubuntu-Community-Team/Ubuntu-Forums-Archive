$ cd <rom dir>

### Post by aysiu on 2007-03-05
glisteringwaters is right, but there's a step missing.

First, you have to [enable extra repositories](http://monkeyblog.org/ubuntu/installing/#enabling_extra_repositories) (the Universe ones in particular for this program).

Then you can [use Synaptic Package Manager to install *mednafen*](http://monkeyblog.org/ubuntu/installing/#installing_with_synaptic).
