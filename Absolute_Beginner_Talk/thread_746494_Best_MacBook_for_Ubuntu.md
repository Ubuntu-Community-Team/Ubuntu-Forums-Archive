### Post by ElTimo on 2008-04-05
All the latest MacBooks use Intel GMA X3100 or something involving "G" "M" "A" and "X3100" If you have the spare cash, I would go for the pro, as that uses NVidia graphics, although i hear that the X3100 is about on par with the NVidia Geforce 7300, which I have and which works beautifully. You COULD try the macbook air, but that would take a bit of extra work, seeing as it has no optical drive and it uses a solid-state disk.
