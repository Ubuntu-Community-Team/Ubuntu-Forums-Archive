### Post by Tatty on 2008-04-16
**1. is it eazy enough to change to linux [ as im a super dooper noobie in terms of linux]**
Yes and no.  Installing is usually simple enough as long as there are no problems with hardware compatibility.  There are usually ways to get round any problem though, just post on here if you are stuck.

Learning how to use linux is not as easy as most windows users expect it to be.  Linux is a very straight-forward and logical thing to use, however *emphasis* linux is not windows!  In many ways, learning to use linux is kind of like going back to square one, as you ahve to learn how to do most things on your computer again.  But the plus side of this is that you really do learn a lot in doing it, and after just a couple of months you will start to get comfortable with it.


**2. shuld i go 4 a 32 bit version or 64 bit version[ as i have heard 32 ubuntu is super cool ,but its 64 bit version is a nightmare like XP or vista 64 bit version]**
Use 64bit Ubuntu if you have a 64bit machine.  I started using 64-bit ubuntu when i re-installed my laptop a couple of months ago, and I have not run into anything yet which has caused me any problems.  It has all worked just as the 32-bit version would.

64-bit linux is more mature than 64-bit windows, as linux has had a 64-bit kernel for over a decade now.


**3. are ample software available for ubuntu for both 32 and 64 bit**
3.  Tonnes of software is available.  There are thousands of apps in the repositories and plenty more available from other sources - although it is rare that you will find you need to go outside the repos.  What do you need?


**4. i have a Pinnacle pc tv stereo tv tuner card can it be used on ubuntu [ my frnd has the same and told me he couldnt figure out on how to use it said it was black listed] so can it be modded or hacked to work on ubuntu**
 I am not sure about your particular card.  You will probably have to do some research.  I struggled to get my nova-t 500 card working at first, but after a bit of googling i found a nice wiki page detailing how to compile the driver for it easily.


**5. did some research and found....there are no converters to rip my dvd [ no im not talking bout sumthing like dvd shrink or dvd decrypter]**
I have never done this myself, but i very much doubt it is hard, and i also doubt that tehre isnt a gui to do it.  Im pretty sure MythTV can rip DVDs,  so i imagine there is a standalone DVD ripper.  Not on my ubuntu machine now so cant check the repos, but i bet there is one.


**6. will my GFX card work out of the box or it needs some geeking up [STUPID NVIDIA Y CANT THE BE LIKE ATI]**
Nvidia is totally the best card to have in a linux machine.  They support their linux drivers much better than ATI.  We would need more details on the card to give you a precise answer, but it should be fine.

Ubuntu will be default install an open source driver for your nvidia card, although the proprietry drivers will work much better, and are easy to install if you want them.  all you will need to do is System->Administration->Restriced Drivers, and then they should give you an option to install them from there.


[B]7. did some more google and find out two things called KDE and GENOME.........
and people say KDE IS BETTER[/B]
Wether you prever Gnome or KDE is truthfully just a matter of choice.  All they are *in simple terms* is basically a way of deciding what your desktop will look like - although both of them can be easily modified to look like whatever you like.

You arent forced to choose one or the other either.  You can ahve both installed on your system, and then when each time you log in you will get an option to choose which one you wish to use for that session.

Personally i like Gnome, because it is very visually compact.  But the choice is simply down to your preference.



**8.also i would like to use office 2007 or 2003 in linux is it possible??**
Wine is software which allows you to run windows applications in linux.  Bear in mind that this is far from perfect, and it can be a hard thing to get working.  if we look on winehq.org though we can see that runnign office in wine is possible, but not perfect [http://appdb.winehq.org/objectManager.php?sClass=version&iId=4992]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=4992")

However, why do you specifically want to run MS office 2007 at all?  Ubuntu comes with Open Office pre-installed, which in my honest opinion is much better than MS office.  You can  get Open Office for windows if you want.


Final thoughts, if you alredy have a working windows install on your machine then i strongly reccomend dual boothing both windows and ubuntu.  It will take you time to learn how to use Ubuntu properly.  Linux is easy eanough to learn, but it is not windows, and so you will find yourself learning a lot of things from scratch.  It wont take too long to get up to speed though.

> guyz .....i google and found something called VMWARE.......can somebody shed some light on it and will i have to worry bout BSOD on vmware

should i go 4 VMWARE or dual boot ...........????????

VMware is an OK way to try out Ubuntu while still running windows in the background.  It's slower because you're running both at once, and it's not as pretty (graphical effects won't work as well if at all).

Virtual machines are awesome though because you can run several "computers" on the same physical computer, and have them network with each other as if there were several... but you've only spent money on one of them.  What I recommend to people is to just delete Windows and install Ubuntu, then to install Windows inside of a virtual machine if they need it because that way when you get a virus you just click "revert to last snapshot" and it's unbroken.

Go 64-bit.
