### Post by JayTee on 2006-10-15
The main goal of the folks at Canonical was to make Linux on the Desktop as easy as possible for the non-technical user. The long term goal is to have an OS that is easier than Microsoft Windows, better supported through the community and most of all free. The technical reasons for having "so many distros" of Ubuntu is so that people that want to try it can burn one iso image or order 1 CD for the Ubuntu that best meets their needs or desires. All the different desktop managers use lots of space so all three wouldn't fit on 1 CD especially one that is created to be a LiveCD. The LiveCD format is an ideal mechanism for distribution. It allows a user to test drive the OS _before they install it_. Some people are fortunate that they have a fast computer but not everyone can afford to upgrade every couple years. Xubuntu is designed to be more compatible for older hardware and still provide solid performance. Kubuntu uses KDE which many people prefer to Gnome and has a very large collection of applications written for the KDE desktop. 
Edubuntu bundled several apps and was customized to appeal to educators using the computers in schools. These plus Ubuntu server which has no GUI built in were targeted for specific uses to suit an individuals needs. I think the distro model they use offers the most flexibility. Less download time and few CD's to burn. It would be nice however if they could come up with a LiveDVD version that had the Gnome, KDE and XFCE desktops included and a menu to choose installation options. Maybe they'll come out with one sometime in the near future. For now I think they've come up with an excellent solution and whether you're running Ubuntu or Xubuntu they both run great in my opinion.
