### Post by chio on 2007-06-14
Ubuntu Studio is only worth getting if you've an interest in professional-level multimedia creation such as audio recording or video editing. Individual Studio programmes can be obtained via normal Ubuntu Feisty if you feel that you need them after installing. There are details on how to do that here: [http://www.ubustu.com/globe/2007/05/23/add-ubuntu-studio-to-an-existing-ubuntu-install/](http://www.ubustu.com/globe/2007/05/23/add-ubuntu-studio-to-an-existing-ubuntu-install/)

Installing is reasonably straightforward, the thing you're most likely to slip up on is partitioning; without knowing how much free space you've got on your HD etc I can't help much with that, but I'd advise you defrag your HD through Windows and BACK UP your important files before touching Ubuntu. When you install, it'll detect that you have Windows and add it to the GRUB boot menu to be selected.

### Post by Happy_Man on 2007-06-14
1. UbuntuStudio is geared towards multimedia production. If you just want the basics like web, music, etc. stick with regular Ubuntu Feisty.

2. Yes. Photo Editing is handled by the GIMP, comes with the install fresh. Music recording I'm not so familiar with, perhaps someone else could recommend something?

3. It's very easy. Here's a link to a basic guide (you will need to have downloaded the .iso from the site [http://ubuntu.com/download](http://ubuntu.com/download) first): [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

4. Yes, GRUB comes standard, will install with the rest of Ubuntu, no sweat.

Now, Ubuntu comes with stuff like word processing, spreadsheets, and presentation software, photo editing stuff, a music player and organizer, web browser, all standard. You just install the CD and voila! there it is! Whatever you want afterwards, it's easy to install using Synaptic Package Manager. Don't worry, you'll get the hang of it. Ubuntu is immune to viruses and malware, and spyware, and all that other crap you get on Windows 30 seconds after hooking it to your modem. No Windows virus will get a chance to work. And, you don't have to go hunting all over the Internet whenever you want an application. Most of the time, it will be available through the repos, which are giant servers chock-full of software for every conceivable purpose. Search through those first. Also, Ubuntu has automatic security updates, like Windows Update, but here's the thing: it will upgrade ANY and ALL installed programs you put on it too! How great is that? And there's so much more, but I'm not sure how long I could keep going, so it's best to stop here. Best of luck!
