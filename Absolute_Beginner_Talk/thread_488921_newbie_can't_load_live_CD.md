### Post by DishBreak on 2007-06-30
For most computers, when you power it up, it will say, "press F2 to enter BIOS setup" or something like that. When that goes through, go to boot order and make sure that your CD drive boots first. Then continue on, and select "Start or Install Ubuntu". 

Remember, Live CD will run slower than an install on the hard disk, because your computer is esentially running off your CD drive as opposed to your hard disk.

Good luck!

### Post by Qrk on 2007-06-30
The live cd can have starting on older systems, particularly if they have slow cdrom drives. Why don't you try this link, to get the alternate cd, which is better for older systems.

[http://www.ubuntu.com/getubuntu/downloading?release=desktop-newest&arch=i386&mirror=http%3A%2F%2Fubuntu.cs.utah.edu%2Freleases%2F&debug=%5B%27country_US%27%2C+%27country_UK%27%2C+%27continent_NA%27%5D&download-button=&alternatecd=alternate](http://www.ubuntu.com/getubuntu/downloading?release=desktop-newest&arch=i386&mirror=http%3A%2F%2Fubuntu.cs.utah.edu%2Freleases%2F&debug=%5B%27country_US%27%2C+%27country_UK%27%2C+%27continent_NA%27%5D&download-button=&alternatecd=alternate)

Keep in mind that unlike the liveCD, this one does not give you a way to make sure everything will work before installing. But it also allows you to install when not everything works.
