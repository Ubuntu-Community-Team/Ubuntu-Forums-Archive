### Post by NilsE on 2007-08-02
If you can select boot from USB device on your BIOS and or boot screen then remove the Windows drive and boot from the LiveCD and install to the USB drive. Then put your internal drive back.  This way there is no chance of damaging your windows MBR.

Then at boot it will default to the internal drive and you can select to boot from the USB drive.  You could also change the boot order in the BIOS setup to boot from the USB if it is present just like it would for a CD drive.
