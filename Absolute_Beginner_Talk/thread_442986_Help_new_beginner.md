### Post by Jeff24K on 2007-05-14
Welcome to Ubuntu, Teresa, from a fellow Floridian. :)

Programs aren't "labeled" with any particular extension under linux, in fact any file can be made exexutable. Shell scripts are plain old text files, much like DOS batch files for example, sometimes given the extension .sh. Most compiled programs don't have a file extension at all, it's all done with "permissions". 

Software is generally installed from repositories, maintained by your Linux version's development teams, as well as third parties. These repositories are accessed numerous ways, but the most "Windows-like" is going to be Synaptic. In the menus it's System > Administration > Synaptic Package Manager. The password it asks for is your normal user password. Along the left side are categories you can browse, or there's a "Search" button toward the top. 

Here's the official (?) [Starter Guide]("http://ubuntuguide.org/wiki/Ubuntu:Feisty") for Feisty also. In case you want to do some reading. And don't neglect the "Help And Support" selection off the System menu, either. Much information there. :)
