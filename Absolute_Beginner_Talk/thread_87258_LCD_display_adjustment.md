### Post by Original Brownster on 2005-11-07
Hi,
Glad you like Ubuntu, I've used Linux for years now and Ubuntu is probably as good as it gets.

You monitor problem I'm guessing is to do with the default resolution that ubuntu is using, I would suggest using the same resolution you use in windows as the default in linux. 

Find out what that is if you dont know already, then you can set this in (system > preferences > screen resolution)
Incidently this little program edits the /etc/X11/xorg.conf file which si where all the settings relating to your display are held.

Once both windows and linux are using the same resolution you should find it works without adjustment.
