### Post by Richard Kut on 2007-02-23
Hello jprb85! I have never had much success with AcidRip myself, just because I found the interface too confusing.

If you are willing to try something new, then I can sincerely recommend k9copy. Look here for details:

[http://k9copy.sourceforge.net/]("http://k9copy.sourceforge.net/")

I am not sure, but it might be in the repositories. If so then you could install it with:

> sudo aptitude install k9copy

Give it a try. It has saved the day for me many times when DVDshrink in WINE failed miserably. Good luck!
