### Post by glederfein on 2007-03-23
Hi all!
I'm using the Epiphany Browser and I want to hear a ".pls" file from shoutcast. The problem is that the file opens with "Rhythmbox Music Player" while I want it to open with "XMMS Music Player". I understand Epiphany uses the default GNOME applications for opening files, but how do I change these defaults so that Epiphany will automatically open .pls files with XMMS?

---

### Post by docshawnc on 2007-03-23
Hi -
    Save a .pls file to your desktop.  Right click on the file and go to properties.  Choose the tab "open with" and pick xmms (you may need to find in on your drive)
