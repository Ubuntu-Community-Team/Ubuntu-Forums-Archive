### Post by Jerry N on 2007-08-30
People keep telling this fellow to "burn the .ISO file to a CD" but no one has explained how yet.  With Nero, select "Nero Burning ROM".  A "New Compilation" menu should pop up - cancel that and select "Recorder" on the tool bar.  One of the items on the resulting menu is "Burn Image".  Select that and then on the resulting file browser navigate to the location of the correct .ISO file, ie "ubuntu-7.04-desktop-386.iso".  Select that (double click or "Open") and then select the "Burn" button.  At that point, Nero will start creating a bootable CD. Some folks here claim that you need to burn the ISO image at a very slow speed but I have never seen the need for that - maybe low quality CDRs or a bad CD writer?

Jerry
