### Post by Najand on 2006-09-11
If using KDE, amaroK with using the Xine library or the gstreamer plugins can do playback of AAC/MP4 and other audio formats.
If you find mplayer's right codecs it will work too...
VLC player has its own aac/m4a codec so can playback them without installing any more plugins.
