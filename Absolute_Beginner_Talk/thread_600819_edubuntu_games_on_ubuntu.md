### Post by gate on 2007-11-02
The most fundamental differences that I have seen are that it comes with different default programs, like a paint program for kids, etc. a more kid-friendly theme, parental controls (I think).

  yes, you can get all the games for Edubuntu in Ubuntu, they are all in the package managers.

  I personally reccomend the childsplay packages, my nephews love them.

---

### Post by gate on 2007-11-02
another note, if you have an Ubuntu/Kubuntu system and want Edubuntu, don't reinstall, run 

sudo apt-get install edubuntu-desktop

That will get you everything edubuntu has.
