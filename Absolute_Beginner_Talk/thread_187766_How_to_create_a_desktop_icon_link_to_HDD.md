### Post by PhilOSparta on 2006-06-03
Right click on your Desktop and select "Create Launcher.."

Fill out the data that you want to appear on the Desktop and select the Type to be Link.  You may have to browse to the /storage directory etc.

This has worked for me to place directorys and links on the Desktop.

### Post by rdd on 2006-06-03
RightClick on the desktop and select 'create Launcher' 

Select 'Link' as type and enter the '/storage' at 'URL', the rest accordingly.

### Post by catlett on 2006-06-03
FYI...If you mount things in the /media folder they will automatically have an icon on the desktop. So if you hdd was mounted at /media/storage instead of /storage, it would have an icon on your desktop at startup without your doing anything.
