### Post by Lord C on 2007-11-12
Try

```
Section "Screen"

# Removed Option "metamodes" "DFP: nvidia-auto-select +0+0, CRT: 1280x800 +1440+0"
# Removed Option "metamodes" DFP: nvidia-auto-select +1280+0, CRT: 1280x800 +0+0"
    Identifier     "Screen0"
    Device         "Videocard0"
    Monitor        "Monitor0"
    DefaultDepth    24
    Option         "TwinView" "1"
    Option         "metamodes" "DFP: nvidia-auto-select +0+0, CRT: 1280x800 +1440+0"
    SubSection     "Display"
        Depth       24
    EndSubSection
EndSection
```

That's assuming both monitors can handle the same resolutions.
