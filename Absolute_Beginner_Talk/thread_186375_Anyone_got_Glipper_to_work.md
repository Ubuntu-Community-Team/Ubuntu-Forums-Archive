### Post by %hMa@?b&lt;C on 2006-06-02
to run glipper, go to terminal and type ```
glipper
```.  if you see it as the same thing in my screencap, then go to 
SYSTEM>PREFERENCES>SESSIONS
and go to the startup programs tab and click add
after that paste ```
glipper
``` in.
reboot and glipper should stay in the tray.

glipper is in the breezy repos? well, i never knew that.
Well, I will give you a screenshot of what a running glipper looks like on my machine, its running dapper, but it is the same way that glipper runs.  Glipper is just a small applet next to the volume applet.  If glipper doesnt work for you, i suggest using klipper, it uses kde libraries, but it works just the same.
