### Post by christhemonkey on 2006-04-01
According to this wiki file:
[https://wiki.ubuntu.com/IntellimouseMousemanBackForwardButtons](https://wiki.ubuntu.com/IntellimouseMousemanBackForwardButtons)
Did you remember to add the xmodmap file?
```
sudo nano /etc/X11/Xsession.d/57xmodmap
```
and write:
```
#/bin/bash
  xmodmap -e "pointer = 1 2 3 6 7 4 5"

```
then type
```
sudo chmod 755 /etc/X11/Xsession.d/57xmodmap
```
Then restart X server with Ctrl+Alt+Bkspc.
