Where are desktop backgrounds stored by default? Or where should I put new ones when downloaded?

---

### Post by mikewhatever on 2007-10-02
The default ones are in /usr/share/backgrounds/. You can use that same directory for the new ones, or create another to facilitate the issue. I've simply created a folder /home/wallpapers, put all the new ones there and that add them.

---

### Post by Caffeine_Junky on 2007-10-03
yeah exactly!

I just put all my own Wallpapers into a "Graphics folder" that I made in PLACES > HOME, and you can navigate there from the System > Preferences> Desktop Backgrounds OR "Right Click"  the Desktop and select "Change Desktop Background" > [Add Wallpaper]

If you want to put your Wallpapers into the "File System" > /usr/share/backgrounds folder, you will have to use the Terminal" & "sudo"  , blaahh, to much hassle when you have 50+ Wallpapers :p
