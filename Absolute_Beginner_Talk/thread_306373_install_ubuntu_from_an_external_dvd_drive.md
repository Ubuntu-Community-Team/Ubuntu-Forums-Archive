### Post by drphilngood on 2006-11-25
I would think that either you have a bad installation CD or you didn´t correctly set the bios to boot from the external drive(did you save & exit properly after making the change?).  Try pressing F2 at boot to change the boot order and try another bootable cd.
You might also check if there are any known bugs that prevent your machine´s booting from usb; a bios update may be required.
You can always install from the HDD.  Although, it is more involved, [guides like this]("http://marc.herbert.free.fr/linux/win2linstall.html") are available to walk you through it.

Good luck!
