### Post by nicedude on 2008-04-21
The thing you need is called video 4 linux, its software for all kinds of video input and capture. I am not sure which exact packages you need to install though so you will have to wait for someone else to chime in or search the forums for that. Also you can try searching in the Synaptic Package Manager ( its like windows add & remove on steroids ) you can get to it by clicking on the top toolbar menu names System -> Administration -> synaptic package manager. after you type your password and it opens you can click the search button and then select to search by description & name - then search for webcam and see what you find in the way of utilities. its the best place I think for someone to get a good idea of all that is available from the ubuntu repositories.
