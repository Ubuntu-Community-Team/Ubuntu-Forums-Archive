### Post by stormblue on 2006-08-04
You can read about Ubuntu Servers [here]("https://help.ubuntu.com/community/Servers") to get started

Download Ubuntu Server and install the LAMP stack.  That takes care of the webserver and can be found [URL="https://help.ubuntu.com/community/ApacheMySQLPHP"]here

[/URL]

Next is your mail server, which you can read more about [here]("https://help.ubuntu.com/community/MailServer")

I don't know about a fax server, I'm sure someone will come along with an answer.

You may also want to look into Hive for your server.  It can come in pretty hand, I would imagine.  You can check it out [here]("https://help.ubuntu.com/community/Hive")
