### Post by chlorinekid on 2007-11-08
presumably the 'folder' field in the 'connect to server' options is either set to blank and so defaults to the root directory on connect or it has a forward slash in it '/' which i guess will connect you to the root directory..

can you not enter the address of your home directory in the 'folder' field?

something like '/home/david' maybe?
