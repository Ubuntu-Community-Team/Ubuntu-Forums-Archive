I have the same printer as you and it works in Linux for me so drivers should not be an issue. I did some searching in these forums and found a common solution is to setup Samba to allow communication between Unbuntu and XP.

This is actually something I plan on doing once I return home from this deployment, but for now I haven't the slightest clue how to do it.
