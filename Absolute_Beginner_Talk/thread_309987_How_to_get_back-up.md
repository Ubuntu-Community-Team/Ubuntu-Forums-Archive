### Post by podunk on 2006-11-30
There are many backup tools in Linux. As a previous Windows user I found Partimage to be the simplest. Not as fast as some, but very intuitive.

It works particularly well if you have an external USB or a second internal harddrive. I backed up over the network once just to try it out and it was very slow.

[www.partimage.com](www.partimage.com)

For daily backups that run unattended there is Simple Backup which is in the repositories. To be honest I don't really like this one because it's backups take up a lot of space. I just run partimage once a week. Only takes 20 minutes or so for my Linux drive.
