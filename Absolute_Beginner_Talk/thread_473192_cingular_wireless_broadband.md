### Post by shearn89 on 2007-06-15
first off, try not to start a post so aggressively- the beginner's forum is exactly that, and virtually everyone here is a n00b. Its fine.

As to the card, check here:
[https://help.ubuntu.com/community/WifiDocs/WirelessCardsSupported](https://help.ubuntu.com/community/WifiDocs/WirelessCardsSupported)
or here:
[http://doc.gwos.org/index.php/Networkwifi](http://doc.gwos.org/index.php/Networkwifi)

to find out if it is supported. You may have to use ndiswrapper to run it properly (check the comments), for which a howto can be found here:
[http://ubuntuforums.org/showthread.php?t=31926](http://ubuntuforums.org/showthread.php?t=31926)
and the official site here:
[http://ndiswrapper.sourceforge.net/joomla/index.php?/component/option,com_openwiki/Itemid,33/](http://ndiswrapper.sourceforge.net/joomla/index.php?/component/option,com_openwiki/Itemid,33/)
