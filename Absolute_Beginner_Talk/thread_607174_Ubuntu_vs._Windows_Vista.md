### Post by Mondoman on 2007-11-08
Download the Ubuntu 7.10 .iso and burn it as a CD.  Boot your laptop from that CD and choose the "live" option.  That will run Ubuntu off the CD, not touching your hard disk.  It'll obviously be slower than "normal", and likely will just give you a generic 1024x768 screen, rather than the native resolution of your laptop, but it will allow you to try out Ubuntu basics.  If you think it's OK, then move on to setting up a dual-boot on your laptop.  I just shrunk the extended partition on my VIsta laptop by 30GB or so, and then used that free space to install Ubuntu 7.10.  The system now boots to the Ubuntu GRUB loader, from which I can choose to boot Ubuntu or Vista.

### Post by HermanAB on 2007-11-08
The Ubuntu Live CD is a kind of a test and recovery CD.  If it works, then you can be pretty sure that when you install it on the HDD, that it will still work.  You cannot add programs to the live CD, but you can mount a partition on the HDD or a memory stick and save stuff there.  So it is possible to do real work with a live CD.

An example of a blazingly fast live CD (or live USB stick) is Puppy Linux [http://www.puppylinux.com](http://www.puppylinux.com)  I use that one regularly, running off a 1GB USB stick, since my laptop is an encrypted company thing (I do sensitive work), so Puppy allows me to use it at home.

Linux is rather different from Windows, so it is a good idea to fluff around with one or two Live distributions first, before taking the leap.  Various things can go wrong during an install, so it helps tremendously if you are not completely new to the problem anymore.

Cheers,

Herman
