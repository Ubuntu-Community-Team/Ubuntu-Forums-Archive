### Post by RobsterUK on 2006-09-11
Providing you have free space on the existing partition the Ubuntu installer will resize the partition for you, and create the new partition for Ubuntu, you just select how much of the free space you want to use for the Linux partition.

I did this recently and it worked with no problems.
