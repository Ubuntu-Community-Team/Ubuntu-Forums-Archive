### Post by Repabil on 2007-03-28
Gnome 2.16 is in Ubuntu 6.10 (Edgy Eft). Gnome 2.18 is in Ubuntu 7.04 (Fiesty Fawn). It may be possible to install Gnome 2.18 in Ubuntu 6.10 but there's no officially supported packages for it and I guess it'd cause more trouble than good. I'd recommend sticking with Ubuntu 6.10 and Gnome 2.16 until Ubuntu 7.04 is released next month in which you'll get Gnome 2.18.
