### Post by DirtDawg on 2006-10-09
> **djsroknrol said:**
> Try automatix...it has numlock and alot more...
*Don't* use automatix unless you're absolutely sure it's something you want to use. 

Install **numlockx**. Go to 'Prefrences>Sessions'. Go to the startup programs tab, press 'add' and enter: ```
 numlockx on 
```
