### Post by Duras on 2006-02-21
Hey, I got the problem sorted... Actually not me, but a friend.
There was this option I had to at the boot start:
live vga=771

So I just got my first lesson in Linux: when something is not working check the advanced options and you will find something that will make it work. :)
