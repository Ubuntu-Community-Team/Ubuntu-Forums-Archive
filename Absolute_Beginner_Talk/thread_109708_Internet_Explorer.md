### Post by deNoobius on 2005-12-29
I think that just means it's going to be slower navigating than using a native browser, which of course it will.  You can certainly use it to get into sites that only work with IE, and you can go from site to site.  Not sure what else you need it to do.

If you must use IE, IES 4 Linux will work.  I use it when I need to.  But of course I prefer one of the other choices because they're faster, more secure, and better in every way IMESHO).
