### Post by PriceChild on 2006-10-11
I doubt you'll ever receive a virus. Without root permissions all a malicious script can do is trash your home dir.

You may install something like ClamAV or AVG if you like.

Personally i use AVG because i am on a uni network sharing files between pcs, and i would rather not inadvertantly pass on an infected file.

### Post by Not the man I once was on 2006-10-11
Greetings,

I agree with PriceChild, although there is a minimal issue of Malware within GNU/Linux, with the only In The Wild Malware containing an UnInstallation script, you can, if you are within a networked enviroment, take common courtesy for Microsoft Windows systems and install AntiVirus software on your GNU/Linux system.
Although you are not going to be affected by any Malware that may be transferred to your system via an infected file or perhaps an Internet Worm, you will minimalise the threat to Microsoft Windows PCs recieving infected data. I would fully recommend GriSoft AVG AntiVirus if you are researching an AntiVirus package to use. I hope this helps your query.
