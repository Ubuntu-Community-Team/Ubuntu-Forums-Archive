### Post by Tux Aubrey on 2007-07-28
What do you want windows for?

Really.  If you want it for high power 3D games, you will want it installed "properly" (ie. not in a VM) so it gets direct access to your graphics card.  Much easier to install it first - but a pain if you later want that disk space for your Ubuntu install.

If it is just for regular apps (Photoshop, Office etc.), go with XP in a VM on Ubuntu 7.04 (provided you have the RAM to run two OSes)  XP runs like a charm in 512Mb (but that's absolute minimum for Vista).  

The great thing about running Windows in a VM is that you can eventually delete it and recover the disk space without going through a difficult repartitioning.
