### Post by jasay on 2006-04-07
[QUOTE=x5452]does anyone know how to change the color of the background of gDesklets?  I have some up and running, and they are cool, but they are all in a black background which wont match my desktop?  Is there any way to change the color or make it go away?[/QUOTE]Try right-clicking on the gdesklets icon (puzzle pieces) in the notification area (usually the top right) and selecting configuration.  Unclick translucency.  That should make the desklets transparent so you can see the background through them.

If that doesn't work, some of the desklets have a place to enter a background picture (or picture of a solid color) in their configuration.

Hope it helps.
