### Post by CroEragon on 2007-01-05
Every computer (i think mac also have boot sequence). That is order where irt will check for operational system. My computer first check floppy, then hard drive, then cd drive, then network. If your order is set same as mine you need to change so it looks first on cd, before hard drive. Then it will find Ubuntu that will boot up from cd, merging with RAM and not using Hard disc for writing, just reading. So when you turn computer off all RAM memory will be erased, with it everything about Ubuntu. After your system will look like you never had ubuntu on it. As long as you do not click install, Ubuntu wont try to make permanent changes to your system. When you turn off and take CD out it will be same as before.
