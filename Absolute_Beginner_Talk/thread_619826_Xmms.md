### Post by FM1 on 2007-11-21
"Practice safe text" means that if you read something that tells you to type (whatever command) into a terminal, and you don't have a clue what it does, you shouldn't do it unless you *really* trust the source. If the command begins with "sudo", then be doubly sure because if you have root privileges (that's what sudo does), you can do *anything* to the system, including deleting critical files. If you used DOS, then you remember the old "format C:" joke. Same idea.
