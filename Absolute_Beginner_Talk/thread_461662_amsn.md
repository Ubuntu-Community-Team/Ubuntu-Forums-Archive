### Post by Inxsible on 2007-06-01
Use any one of the methods below to install aMSN

1) ```
sudo aptitude install amsn
```

2) Go to Applications --> Add/Remove. Select All Available Applications -- > Search for amsn --> Select and install

3) Go to System --> Administration --> Synaptic Package Manager -->Search for amsn --> Select and Mark for installation
