### Post by noMScraig on 2007-05-23
My solution was embarassingly easy...I have both a sound card and the usb speakers.  The system sounds will work without any tweaks if I route the system sound throught the sound card and not the usb port.  I just used

System  --> Preferences --> Sound

Selected the sound card for Sound Events and viola.

I did check out the following post that seems close to your problem
