### Post by Frak on 2007-10-15
> **teh'p3nsi0n3r said:**
> simple question, is there a way to make Firefox open Ktorrent as the default application for torrents when i click a torrent link so it opens the client, i know i can choose "other" but i don't know exactly what I'm looking for and i cant find anything in the preferences in Firefox to set this up? some help would be apreciated.
When trying to save something, use the first option, then other in the dropdown box.

Then navigate to Filesystem->usr->bin->ktorrent

Double click that and it will set it as default.
