### Post by Rocket2DMn on 2008-02-08
sdc IS your external drive (look at your "sudo fdisk -l" output), that's what we've been fiddling with the whole time.
Unmount (sudo umount /dev/sdc1), then unplug the external drive, then comment out the sdc1 line in /etc/fstab
```
gksudo gedit /etc/fstab
```
make the line
```
# /dev/sdc1 /media/sdc1 ntfs-3g defaults,locale=en_US.utf8 0 0
```
Save and close, then plug your external drive back in.  It should be able to automount now.
