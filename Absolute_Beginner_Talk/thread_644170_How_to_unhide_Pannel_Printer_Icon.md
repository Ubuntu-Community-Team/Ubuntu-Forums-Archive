### Post by galfor on 2007-12-18
I stumbled onto this how to un-hide the icon. 
1. Goto [http://localhost:631/jobs/](http://localhost:631/jobs/)
2. Click on Administration
3. Click on Edit configuration files
4. Click on use default config files
And it will un-hide the panel icon.
