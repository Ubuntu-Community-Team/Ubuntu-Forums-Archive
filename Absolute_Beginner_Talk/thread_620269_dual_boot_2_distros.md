### Post by bongobonga on 2007-11-22
You just have to add additional lines to your grub menu file. 

The important lines are 

title fedora (2.6.23.1-42.fc
root (hd0,0)
kernel /boot/vmlinuz-2.6.23.1-42.fc8 ro root=LABEL=/ rhgb quiet
initrd /boot/initrd-2.6.23.1-42.fc8.img

You should add something like the following:
title Ubuntu
root (hd1,0)
kernel /boot/vmlinuz-ubuntu ro root=LABEL=/ rhgb quiet
initrd /boot/initrd-ubuntu.img

look in the /boot directory for the correct names of the vmlinuz-?? and initrd-?? files for ubuntu. What ever you do, don't change the existing lines in the conf file unless you know what you are doing. If you leave them as they are, you will always be able to boot into F8 even if you make mistakes in the lines that you added.
