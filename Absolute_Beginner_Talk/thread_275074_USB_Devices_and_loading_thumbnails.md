### Post by Aikon- on 2006-10-10
Is there a way to disable a certain USB device (such as a digital camera) from loading thumbnails when I open the folder for viewing?

Sometimes I want to plug in my camera and grab, say, the latest pictures or whatever, but when I open the folder, it sits there eating resources while it goes through all the files and generates thumbnails for them, which is made worse by the fact that its not a particularly fast access speed (SD through the camera through a USB cable, i.e. the camera is having to do some file processing).

### Post by dmizer on 2006-10-10
if it's nautilus, just click on view > view as list.  there should be something similar in konqueror.

this is a global change however.  not sure how to make it happen for specific folders or devices.
