### Post by rouge568 on 2007-12-09
It depends. If it is out of a flash-based outlet (think youtube), then no. But if it is just embedded, I would use the Firefox extension "Down Them All", and put /.mp3 (or /.wav, etc.) into the filter option and download it. 
If itunes doesn't have it, try projectplaylist.com, dogpile.com audio search, or [this mp3 search engine]("http://www.jimmyr.com/mp3.php").
