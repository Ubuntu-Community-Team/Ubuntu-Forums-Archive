### Post by HermanAB on 2007-11-30
Just install standard Ubuntu/Xubuntu and add Samba and CUPS features to the machine as you learn and figure things out.

For internet sharing, you could add a second ethernet card and configure Ubuntu to do the routing and firewalling, but it would be less hassle to spend $50 on a Linksys/Dlink/Netgear  'Home Router', which will typically also provide WiFi for a laptop at the same time.
