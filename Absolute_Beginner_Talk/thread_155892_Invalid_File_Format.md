### Post by aysiu on 2006-04-05
Sometimes a .tar.gz is just a theme--say, window decorations, for example.

Sometimes it's a .tar.gz full of a bunch of theme components--window decorations, controls, icons, wallpapers, login theme...

If it's the latter, you have to unzip it first and then manually "install" (i.e., copy to the appropriate folder) each theme component.

If it's the former, you can just drag and drop the .tar.gz to the theme manager window.

Can you provide a link to the theme you're trying to install?
