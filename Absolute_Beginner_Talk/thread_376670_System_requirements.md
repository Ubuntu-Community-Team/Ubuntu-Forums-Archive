### Post by george29 on 2007-03-05
The requirements for a xubuntu install are below... you'll probably need to use the 'alternate install' CD and _NOT_ the 'Desktop' CD.


To run the Desktop CD (LiveCD + Install CD), you need 128 MB RAM to run or 192 MB RAM to install. The Alternate Install CD only requires you to have 64 MB RAM.

To install Xubuntu, you need 1.5 GB of free space on your hard disk.

Once installed, Xubuntu can run with 64 MB RAM, but it is strongly recommended to use at least 128 MB RAM.

taken from [http://www.xubuntu.org/get](http://www.xubuntu.org/get)


the Alternate install is text based - rather than graphical and doesn't boot into a working liveCD desktop.

you could always investigate the use of lighter distros such as Damn Small Linux (DSL), puppy linux or feather linux
