### Post by firefly2442 on 2007-02-28
You might want to try runing the "alternative" Ubuntu ISO.  This has better support for older hardware and has a less "graphical" installation process.  Or you could try the XFCE version of Ubuntu which is good for older computer systems as it is less "system intensive".  Also, make sure you are setting up a swap partition.  This is kinda like "extra ram" on the hard drive that the computer can use if your RAM is full.  I bet getting some more RAM in the machine will help a lot.  Good luck. :)
