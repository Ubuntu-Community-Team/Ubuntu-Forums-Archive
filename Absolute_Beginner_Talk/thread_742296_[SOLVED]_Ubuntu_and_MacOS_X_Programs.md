### Post by Calash on 2008-04-01
Ubuntu is Linux, not Unix

It's a common confusion point that they are the same, or so closely related that porting is easy

[http://wiki.linuxquestions.org/wiki/Linux-UNIX](http://wiki.linuxquestions.org/wiki/Linux-UNIX)


Applications can work between them, but would require source code to recompile them for the other OS.
