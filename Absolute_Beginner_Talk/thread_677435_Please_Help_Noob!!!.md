If you don't install Ubuntu with a CAT5 cable plugged in, I think your wireless stuff probably won't work. On my system, the install always goes to the Internet and downloads drivers for my wireless. If I don't have cable connected, I have no wireless when it's done.

I think you need to do some searching in the forum to find some answers. What you need is some hand-holding to get you up and running, but typing all that you need to do will take a lot of time. Telling it to you would take a very short time.

Here's a start:

Click network icon in pannel (two computer monitors) and see if you are getting a signal from your router. If so, click on your router name and put in password if you have one. If not connecting and you don't know protocols, you might want to try and remove wireless password from router to begin with and try and connect that way. I believe, in order to do admin for your router, you have to type 192.168.1.1 into the URL address area of your browser. Then put in a password. You can leave USER blank; just need to put in a password. Default password for router is 'admin' if you never changed it.

It's all actually pretty easy when you know what you're doing, but unfortunately - you're not quite there yet.

Good luck...
