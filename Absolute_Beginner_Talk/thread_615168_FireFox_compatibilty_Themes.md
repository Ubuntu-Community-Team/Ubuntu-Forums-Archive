### Post by jordanmthomas on 2007-11-17
There should be a way that you can force themes to work for you.

Here's an example, using the first theme you said you try:
1.  go and download the theme onto your desktop (right click on its link and go to save target as...
2.  On your desktop, rename the file to whatever.zip instead of .jar and extract it.
3.  delete the zip file.
4.  In the newly created folder, edit the install.rdf folder.
5.  Edit the lines highlighted in my screenshot.
6.  Now, rezip the theme and rename it to whatever.jar
7.  Drag the newly created .jar file onto your themes window in firefox.
