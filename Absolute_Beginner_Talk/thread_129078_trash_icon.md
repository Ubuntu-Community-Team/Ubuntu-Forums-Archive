### Post by Nunsta on 2006-02-12
gTweakUI is a collection of simple gui's for gconf.

apt-get install gtweakui or do a search in Synaptic for gtweakui and install it.  After you will have some options in you System > Preferences menu that include gTweakUI - Nautilus, which will let you select a few icons to display directly on your desktop, including the Trash Can.

For what it's worth, the same dialog will let you turn off the annoying (for me anyway) 'view everything in browser mode'.
