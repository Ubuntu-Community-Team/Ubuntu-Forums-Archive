### Post by JBAlaska on 2007-09-11
I used mandrake (Mandriva) for a while in a virtual machine but just lost interest after a time. I also started using Ubuntu primarily for Compiz Fusion and liked it so much that now I'm 100% on Ubuntu and haven't even booted to windoze for any task for quite a while.

If there's something that simply needs XP (Gordian Knot or VirtualDubMod), I just remote into my MediaServer and run it there.

I have a hunch after you use Ubuntu for a while, you'll use your OSX partition less and less.

Since your new to Linux, you should go with a proven release (Feisty) IMHO and stay away from Gusty until it goes public, that way you won't have the old "Is it a bug or is it me" thingy coming up all the time.

As for Ubuntu Ultimate 1.4+, it's a great package and I'm using it on my main machine, while I have the standard release of feisty on my backup box.

If you setup Ubuntu to use a /home partition, it will be allot easier to upgrade later or reinstall without loosing everything.

Good Guide for partition planning: [http://www.psychocats.net/ubuntu/partitioning](http://www.psychocats.net/ubuntu/partitioning)
