### Post by bdalzell on 2006-07-29
Thank you, that was the solution! It was under Applications>Accessories.

It used to be under Applications>System Tools according to all the older help things I found on the net. 

One uses Applications>System Tools>Application Menu Editor to find the Filebrowser under Accessories in the Application Menu Editor and turn it on with the check box.
