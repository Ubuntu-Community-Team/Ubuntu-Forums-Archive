### Post by Herman on 2005-12-31
Thanks, jimrz :D

Irish_Mob,
'GRUB', (short for 'GRand Unified Bootloader'), is our bootloader that comes with Ubuntu, and most modern Linux operating systems. 
It replaces Lilo, which was less convenient for new users, because Lilo always needed to be manually configured. So do most other boot loaders like Windows NT loader and the like. You need to do some reading and learning before you will be able to configure any boot loader manually.
With GRUB, anyone can install it without needing any special knowledge. Once in a while it needs a little manual correcting to get it to find and boot Windows for which you can get all kinds of help and advice in these forums. You will still be a lot further head than if you chose another bootloader that needs to be manually configured from scratch. For those you might not be able to get any help either.
Most of the time if you use Grub, you won't need to know or do anything. 
GRUB is the most advanced, and is actually more than just a boot loader. You can learn later on when you are ready, how to do more tricks with GRUB, and customise GRUB to your own liking, which is something special that I don't think you can do with any other bootloader.
 I love GRUB!
At the moment I have just installed it in a computer I have booting Windows 98, Mandriva, and Ubuntu Warty, Kubuntu, Dapper and Breezy. I'm not sure what the limit is for the number of operating systems is that you can boot with GRUB, but I am sure it is quite a large number. 
You will find lots more info in the link that jimrz gave you as well. 
:D (Herman)
