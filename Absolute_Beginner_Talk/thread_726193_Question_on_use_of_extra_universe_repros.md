### Post by JKeefe on 2008-03-16
From what I understand, all four of the Ubuntu repositories can be considered to be sources of "safe" software.  The definition of "safe" varies by repository.  

[http://www.ubuntu.com/community/ubuntustory/components](http://www.ubuntu.com/community/ubuntustory/components)

This link does a good jobs of describing the difference between the four different repositories.  

Non-free doesn't necessary mean that you have to pay...  Just that the software doesn't satisfy Ubuntu's licensing requirements.  For instance, the source code might not be available.  It is hard to audit a closed-source component for security purposes, but many in this community must rely on such components to use our hardware effectively.  

I favor software in the main repository when I have a choice, but it is often necessary to install packages from the other repositories.
