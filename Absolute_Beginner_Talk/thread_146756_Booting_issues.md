### Post by IYY on 2006-03-19
The key used in order to modify the BIOS differs from one motherboard to the other (it's often F2 or Delete), but it should usually tell you which key it is when your computer boots. On the very first screen, it should say which key to press in order to change the BIOS, and when you press it, you should be taken to a configuration menu. One of these menus will allow you to change the boot order. Place CD-ROM as the first.
