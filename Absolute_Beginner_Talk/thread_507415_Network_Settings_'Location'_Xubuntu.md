### Post by e_james on 2007-07-23
I believe that 'location' is a reference id for your current situation e.g. 'home', 'office', 'Fred's broadband router', 'My dial-up settings' etc. I think it's supposed to remember appropriate network settings. I believe there is another step which can make one of these choices the startup default, but I don't yet know how to do this.
