### Post by Kateikyoushi on 2006-09-30
GTK is a gimp toolkit to create graphical user interfaces, those are just different versions of it.
Icons are what you expect them to be. ^^;;
GMD is an acronym for Gnome Display Manager.
Compiz is a window manager with openGl acceleration.
Metacity is another window manager uses GTK+2.

### Post by IYY on 2006-09-30
> GTK1.x

This is a toolkit (a way to draw the interface) created a long time ago for the Gimp. It's very fast, but doesn't look too good and can only be themed to change the colour, not look. It's no longer used for new programs, but can still be seen in older programs like XMMS.

> GTK2.x

This is the newest version of GTK. Apps written for the Gnome desktop use GTK2 nowdays (whereas ones for KDE use QT). GTK themes can make your interface look very different: it can change the sizes, shapes, colour of the components inside the application.

> ICON

The little pictures on your desktop, in the file manager, and some applications.

> GDM

The look of the login screen.

> COMPIZ

Compiz is a fancy window manager that uses the video card's 3D features to draw things on the desktop. A copiz theme is just the titlebar of the window when Compiz is used.

> METACITY

Gnome's default window manager. Once again, the title bar of the window.
