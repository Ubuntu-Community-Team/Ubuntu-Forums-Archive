### Post by Kilz on 2006-07-23
There is an extension firefox that works with hotmail. It gives a little more functionality like color of text, font, table, ect.
[https://addons.mozilla.org/firefox/389/](https://addons.mozilla.org/firefox/389/) Once you install the extension and restart firefox go to hotmail. Click on the New Message to create a new email. There will be a check box above the body area, check it for the toolbar.
