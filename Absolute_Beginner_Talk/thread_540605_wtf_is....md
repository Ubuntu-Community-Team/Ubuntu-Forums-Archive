### Post by foxx on 2008-01-10
When you want to use the command "wtf" you need the package "bsdgames" like suggested (The program 'wtf' is currently not installed.  You can install it by typing...).

do
```
$ sudo apt-get install bsdgames
```

and than type:

```
$ wtf omg
$ wtf afk
$wtf lol
$ wtf rofl
$ wtf wtf
```

and so on. You can also specify another "dictionary" database file.
