### Post by cotcot on 2006-02-11
PowerPC is referring to Apple computer architecture. If you have an Apple computer you should download the powerpc iso. If you have for instance a Pentium processor you should download the i386 iso.

---

### Post by heimo on 2006-02-11
powerpc / i386 / amd64 is the hardware architecture the install / live CD is designed for. If you have Apple's computer, use powerpc, if you have PC (ie. Intel or AMD processor), you're safe using the i386 version and if you have 64-bit PC, you could use amd64 (but you don't need to, i386 could be easier at this point).
