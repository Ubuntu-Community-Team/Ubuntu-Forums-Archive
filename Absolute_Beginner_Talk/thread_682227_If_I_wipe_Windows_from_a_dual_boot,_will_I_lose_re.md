Hello,

On my Acer Aspire 3680 laptop, Ubuntu uses restricted driver(s) to use my wireless card. 

I am currently dual-booting Ubuntu 7.10 and Windows Vista.

---

### Post by emarkd on 2008-01-29
Ubuntu doesn't use that Windows data for anything.  The restricted drivers are included in Ubuntu but are restricted because of licensing or other issues, so they don't really conform to the Ubuntu policy of all-free (as in freedom) software.  Removing Windows won't effect Ubuntu.
