### Post by b1g4L on 2007-05-04
It's pretty simple and painless to use change directories using 'cd'.

Start typing where you want to go and use the Tab key to autocomplete directories / files.....

Also, to go up a folder, you don't have to re-type the full location -- just use 'cd ..'
