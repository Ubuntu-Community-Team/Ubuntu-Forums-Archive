GEEZ!!! Why is it everyones composite extension is getting disabled by default? I never had this when I installed mine.

Ok, easy fix. Open up your xorg by getting any terminal open and pasting this in:

```
gksudo gedit /etc/X11/xorg.conf
```

There should be a section at the bottom I think that says composite extension and either a 0 or disabled. You need to enable it, if your not sure just paste the whole xorg here (Ctrl A then copy it and paste it here with code tags pls, the number sign) and we'll tell you what to edit to make it right.
