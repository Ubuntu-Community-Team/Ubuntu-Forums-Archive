### Post by ajgreeny on 2006-07-21
Breezy Badger will work fine for another few months (April 2007 I think) as updates will be available till then.  It will continue to work after that as well, but will not be updated through the repos.  I've never yet done an update from one version to another but I know many have without problems.  It does take a long time so if you do decide to give it a go, be prepared for a long wait while it all happens.

You're missing a lot of great stuff in dapper if you do not give it a go, however, so consider a clean install if not an update.

### Post by avtolle on 2006-07-21
You did a dist-upgrade without a clean install. I did the same on a Mac G4 Cube, without disaster at the end of June; all is working well to date. You are properly upgraded. What you will need to do at some point is DL and burn a Dapper CD, for use in rescue mode (if all goes to crap), or if you want to resize the partitions on your HD, etc. Until you do such, you should keep your Breezy CD around. You do not need to enable it as an additional repo; probably shouldn't, as a matter of fact. 

BTW, since you upgraded this way, you have fewer to no updates to DL, as compared to what you would have had w/a clean install from the CD.:rolleyes:
