This will seem elementary to many but I searched the problem without success.
There are bugs posted on it but for me this was the problem and fix.

I had Wne installed and configed.

Tried to run it but  got a warning that it might be dangerous
Had to select the program (wine)to open it with every time
Did read in the bug that others were experiencing the same trouble
Did read in bug reports that changing .exe to .EXE would solve the problem

Then execute the install program; It was done in seconds

The trick was to download the installer for the program and then install.

If your facing this problem try it both ways and see if it works for you
