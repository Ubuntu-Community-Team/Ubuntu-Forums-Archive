### Post by prodigalson666 on 2007-11-17
Hidden files are prefixed with a period so can hide or unhide any file by applying a period before the file name or by removing. Example,  .filename will hide it, to unhide delete the period.

### Post by dpar on 2007-11-17

On the right pane, check 'show hidden files'
