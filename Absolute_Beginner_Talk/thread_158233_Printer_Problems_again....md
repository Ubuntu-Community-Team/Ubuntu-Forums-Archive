### Post by Chuck_W on 2006-04-10
I went straight to the install version of Ubuntu rather than the Live but it shouldn't be too different. You have to set up a printer first. I'm not on that machine right now but after going to the Add Printer wizard it will walk you through the process. The first step is local or network, then it asks the printer make (giving a list of manufacturers), then you see a list of models. Selecting the right model uses the correct ppd file. It should work then. One bit of advice. My network is a little different and printing gave me some trouble. I found that if I set up a printer that didn't work it did no good to try to change the settings. The only thing that worked for me was to delete the nonworking printer and start over. Hope this helps.
