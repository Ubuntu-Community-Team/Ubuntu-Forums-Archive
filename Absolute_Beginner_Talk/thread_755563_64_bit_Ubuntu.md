### Post by blithen on 2008-04-15
Been sporting the 64bit Ubuntu for awhile now. But the biggest things I find that don't have 64bit builds are games, and not very popular programs.(Even fewer still though) Don't be afraid though, VERY few programs don't have a 64bit build. Remember flash is a bit buggy on 64bit. So if you play flash games, or are a heavy flash user be prepared. Though youtube and most flash games work.
