### Post by houseisland on 2006-09-23
> **halitech said:**
> I have Xubuntu running on an AMD K6/2 350 with 64 Meg RAM but since the serial port is busted (old socket 7 board so no PS/2 mouse) I just run it CLI.

If you want to run you K6-2 system with a GUI.....

If your Socket 7 board will accept a K6-2 processor, it will almost certainly have a PS/2 header on board.  Adapters brackets are cheap.  You just need to find out what the pin outs for the header are so that modify the cable end if needed.  You should also be able find a used ISA serial/parallel card for less than $5.
