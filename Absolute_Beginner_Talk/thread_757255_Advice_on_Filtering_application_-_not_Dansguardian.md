### Post by tnl2k7 on 2008-04-17
I'm not sure it's really worth doing this, as I expect you'll be upgrading to the 8.04 release when it comes out later this month, it's a Long Term Support (LTS) release too.

This might not be ideal for keeping all the blocking lists up to date, but the Firefox extension AdBlock Plus seems to offer what you need, it works for Firefox only, but is easy to use. Problem is, it's also easy to disable, just right click an icon.

If you want more control, DansGuardian (I know you said you didn't want to use it, but it's the best application for the purpose!) or Squid, I know we use this system at school to prevent us going off focus (lol).

I can't help with Squid or DansGuardian, but you can find AdBlock Plus [here](https://addons.mozilla.org/en-US/firefox/addon/1865).
