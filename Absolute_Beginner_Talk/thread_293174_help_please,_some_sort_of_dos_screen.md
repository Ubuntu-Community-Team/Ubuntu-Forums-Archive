### Post by ReaderRat on 2006-11-04
The iso file you downloaded must be burned on a disk as an image file, not a data file. You need to burn it at a low speed (4X) to get a good copy. Then put the CD in the drive, set in your BIOS to boot from CD first, and off you go.

### Post by 3rdalbum on 2006-11-04
Don't worry, lots of people have made this mistake before.

When you burn the CD, you must burn it using the Disc Image option of your burning software. In Nero it's called "Burn Image to Disc".

What you seem to have done is used a "Burn Bootable CD" setting of your software, which is incorrect. The disc image already contains everything needed to boot from the disc.

This page is probably the best tutorial for burning the CD correctly: [http://www.psychocats.net/ubuntu/iso]("http://www.psychocats.net/ubuntu/iso")
