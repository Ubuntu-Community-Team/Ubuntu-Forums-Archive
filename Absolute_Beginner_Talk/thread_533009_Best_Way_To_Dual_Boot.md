### Post by Steve1961 on 2007-08-23
OK, here's the easiest method,  When you reinstall XP delete the existing ntfs partition and create a brand new one for the new installation, but make sure you specify teh size you want for the partition.  This should leave free space for Ubuntu, and the installer will give you the option of creating the partitions it needs in that free space.

### Post by rexy on 2007-08-23
while resizing works it's easier and quicker to leave some free space for ubuntu when you install windows, ie if you have an 100gb disk you create an 80gb disk for windows and leave 20gb unpartitioned. The ubuntu installer will then allocate the remaining space for a partition for ubuntu.
