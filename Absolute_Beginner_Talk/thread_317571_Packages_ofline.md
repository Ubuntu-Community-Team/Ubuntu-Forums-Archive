### Post by Kobalt on 2006-12-12
You can find all the packages you want at [http://packages.ubuntu.com](http://packages.ubuntu.com)
And from the command line, to install a package you must : 
```
sudo aptitude install *name of the package*
```

And if you're looking for some more help (because all of the answers I gave you are there) you can look at [http://help.ubuntu.com](http://help.ubuntu.com)
