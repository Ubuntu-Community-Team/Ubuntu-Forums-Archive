Q: I was wondering if Ubuntu has the ability to modify the UI in such a way as to have the MS styled start button (not the actual button but an Ubuntu version) rather than having the menu's drop down from the top of the screen. Kinda like Mandriva.

### Post by sloggerkhan on 2007-01-19
yes.
All you do is right click on your lower panel and select "add to panel."

Then add a "main menu" panel widget/applet thing by dragging onto wherever you want on the lower bar.

Some of the panel objects may be locked. So you might have to right click on them and uncheck "lock to panel" in order to move it to where you want.

### Post by sloggerkhan on 2007-01-19
I don't think K is pretty, but if you want to try it, you can indeed replace gnome with it. All you have to do is install it from the repos (look for KDE desktop environment). 
Get the base packages installed.

Then log out, click the sessions buttons, choose KDE, and log back in.

You will now be in KDE.
