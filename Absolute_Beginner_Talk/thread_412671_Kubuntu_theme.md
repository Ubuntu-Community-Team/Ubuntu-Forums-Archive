### Post by kinson on 2007-04-18
Edubuntu is obviously very much geared towards the education side of things. But that doesn't mean the others have that kinda target either.

Its really like this:

Ubuntu = [GNOME]("http://en.wikipedia.org/wiki/Gnome_desktop") Desktop
Xubuntu = [XFCE]("http://en.wikipedia.org/wiki/Xfce") Desktop
Kubuntu = [KDE]("http://en.wikipedia.org/wiki/KDE") Desktop

I'm not going to go into the specifics of each Desktop Environment here, but I've provided links to each of them above, and if you search around the forums, you'll find plenty of topics related to them :)

I don't really think Xubuntu is *just for office use.

Btw, the reason some people might say that XFCE is better for the office use, is that it uses less resources, so it probably runs slightly faster on older machines(stuff you find in the office :p ).

Aside from that, you can install KDE apps in GNOME, GNOME apps in XFCE and whatever. The Desktop Environment doesn't limit which programs you can install.

Hope it helps.
