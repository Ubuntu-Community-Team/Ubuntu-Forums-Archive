### Post by Soarer on 2007-09-30
I guess it depends on the price and your pocket :)

I have a P4 (now used as a server) which had Feisty (Gnome) on it. It had 1.25 GB and I didn't think it was enough. I don't game either. It will run perfectly fine, of course, but Linux will use any memory you have available, so it is always snappier the more you have.

I now run Gutsy on a Core 2 Duo (1.8MHz). Started with 1 GB but have now added another GB and it feels much better. It sometimes claims to be using 50% of memory for programs, but who knows how accurate that is.

Firefox also seems to have a memory leak which means it's usage goes up over time. When I had 512MB on the P4 this would sometimes lead to a very slow system, until rebooted when it was fine again.

So, if you buy the extra you probably will feel a difference. But adding the 1GB will make a big difference on its own, and you may find it enough.

That probably doesn't help very much, i know. Sorry!
