### Post by Tux Aubrey on 2007-06-01
I second smoker's advice.  Using the Live CD, you will be able to answer all the questions you have (as well as all the ones you haven't thought of yet).  I haven't used WUBI.

The Live CD is risk free - it does not alter your set-up in anyway until you decide to install.

Ubuntu is a full "Distribution" - the apps (Open Office, Firefox ++++) are part of the distro.

Provided you have a few Gb of spare harddisk space on your machine, you will be able to install Ubuntu as an _alternative_ OS to your current Windows set up (This is called "dual boot").  This lets you settle in with a safety net.  As you become more familiar with Ubuntu, you may find that you don't actually need windows anymore.

You will still have access to all your files on the windows partition but will use Ubuntu's apps (like the Linux version of OpenOffice) to work with them.

If, like me, you find that Ubuntu serves your needs better than any version of windows, you will start to wonder why you are wasting that disk space.  That's when you start to plan a full reinstall without windows. Muhahahaha (evil laugh).  Give it three months and see how you go.

Oh - welcome to the community!

Welcome to ubuntu and ubuntu forums! And there are no silly questions! :D

Unfortunately, you will not* be able to run your Windows programs in Ubuntu, even though Openoffice is available to both Operating Systems. The important thing to note here is that Ubuntu is not Windows, it is a completely different OS. This means that your windows programs (exe, msi, etc) will not work in Ubuntu natively.
That being said, a LOT of software exists for Ubuntu/Linux!! Just about any program you will need, with a few exceptions, will have at least one open-source alternative available, if not two or three. Openoffice is a great example of an alternative to Microsoft Office, as an example.
The best thing is to simply ask some questions. "I have XYZ program in windows, it does this, is there any similiar programs available to me in Ubuntu?"


* = You may be able to get some windows programs to actually install and run in Ubuntu, check out Wine [http://www.winehq.org](http://www.winehq.org) You can also run Windows IN ubuntu using a virtual machine (not that hard to setup), with a slight performance hit.
