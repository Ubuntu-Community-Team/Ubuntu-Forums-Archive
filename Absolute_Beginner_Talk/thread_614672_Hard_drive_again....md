### Post by runemaste644 on 2007-11-16
Try making it a raw partition first. Then the xp install will format the partition as ntfs. Though you might consider making it a fat partition as it requires less defragmenting. Then after windows is installed you might consider installing ext2fsd in xp so you can read and write to your linux volume.
