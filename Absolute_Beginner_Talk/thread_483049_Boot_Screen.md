### Post by falciron on 2007-06-24
First, uninstall Gaim using Add/Remove, or Synaptic, if you installed it some other way. Then, to install Pidgin, just download/run the .deb files provided on [http://www.getdeb.net/app.php?name=pidgin](http://www.getdeb.net/app.php?name=pidgin). Make sure to pick the right version at the bottom, and then on the next page that appears, in the middle should be the download links that you need. Get pidgin-data first, then pidgin. It won't work otherwise. After all this, it should appear in your Internet folder on the program menu. If you get this done, you might also try to get the plugin pack, as there are some fun/useful ones in there.

### Post by falciron on 2007-06-24
As to the other problem, I'd suggest installing Art Manager using Add/Remove. That'll allow you to both download and change boot screen pictures. After installation, it'll appear in System>Administration, not in your regular programs section of the applications menu.

### Post by BobCFC on 2007-06-24
Click on System->Administration->Synaptic Package Manager it will ask you for your password.

Click on the Search button at the top and enter Gaim


It will bring back a list of about 20 things, most of them have white boxes meaning they are not installed but a few will be dark.

Right click on gaim and choose Mark for Complete Removal.. if this is connected to any other package it will say that also need to be removed.  Click OK

You should now see an X next to gaim.   Click apply at the top to start.  Then OK


EDIT:  You can also use this way to add new software.   Have a look around
