### Post by KenSentMe on 2007-01-07
The package abcde is available in the ubuntu repositories. So just install it using synaptic or apt (see the wiki link you got) and then you probably will be able to run it from the menu.

If you want to install a program, first look if it's available in the Ubuntu repositories. Those packages are optimized for running on Ubuntu. Most of the time it's better to use them instead of compiling yourself. It's always a lot easier.
