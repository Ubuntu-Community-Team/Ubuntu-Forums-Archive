### Post by conjur3r on 2007-09-04
If you are serious about getting into server administration, I would suggest you do everything manually.  That way you expose yourself to the process and have a better understanding of what is happening behind the scenes.

If you will be installing apache/php/mysql on your main machine, then I would suggest you use the normal desktop ubuntu version and install the needed packages on top.  Otherwise, if you have a dedicated server, give the server version a try.
