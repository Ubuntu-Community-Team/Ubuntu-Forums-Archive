### Post by DaMan on 2007-01-22
Had same problem... Just follow the instructions:

Go to the Beryl Settings Manager -> General Options -> Level of Focus Stealing Prevention
Select **Normal** (Seems Extreme was selected as default for me)

Let me know if it helps...
