### Post by eeGhoong0ais on 2006-10-02
> **lorre said:**
> I am not at my ubuntu pc right now but it is de image viewer that looks very much like the windows image viewer (with limited functionality like next/previous F11 for full screen). It was the default application for viewing pictures when opening a jpeg.

It's most likely Eye of Gnome, then. That program will allow you to save changes after you flip or rotate an image - use File >> Save or [CTRL]+S - but it **isn't** lossless and will degrade JPEGs.

Gthumb transforms JPEGs without degrading them, and saves the change automatically, so it's usually a better bet. Right-click an image or folder in Nautilus and choose "Open with ..." to use it.
