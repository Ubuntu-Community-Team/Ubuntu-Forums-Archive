### Post by lambros on 2007-06-02
This also confused me when I first started using GNU/Linux.  The actual firewall functionality is built right into the kernel, Linux, and therefore is "running" all the time.  All these "firewall" applications, such as guarddog and firestarter, are merely tools that allow you to configure the rules that the kernel is using (called "iptables").

Sorry I can't help you with your question about how to "monitor things", hopefully someone more knowledgeable can help.
