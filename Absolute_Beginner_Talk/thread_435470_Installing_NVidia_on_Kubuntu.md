### Post by Nythain on 2007-05-06
if you have access to another computer with the internet you can download the pakcages from [http://packages.ubuntu.com/feisty/x11/](http://packages.ubuntu.com/feisty/x11/) then burn them to a cd, load up into kubuntu and in a terminal sudo dpkg -i nvidia-glx.deb nvidia-kernel-common.deb
that should do the trick hopefully... not sure if there's any other dependencies, but i dont think those packages have deps

edit, without a blank cd, you are gonna need a usb storage device, floppy, or as suggested, mp3 player
