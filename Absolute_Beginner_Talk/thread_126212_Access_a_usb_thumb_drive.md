### Post by 5-HT on 2006-02-06
Hi, if you are using the Gnome verision of Ubuntu (regular Ubuntu), it should automatically mount you're flash drive when inserted, create a desktop shortcut, and even open nautilus (the filebrowser) to let you see what's on the drive.

So basically, you shouldn't notice a difference between using under windows or Ubuntu.

If you're using Kubuntu (Kde version of Ubuntu), I wouldn't know for sure as I don't use it myself, but I believe it should do something similar.

For any other windows managers, you'd have to do some tweaking to get it to automatically mount, but I'm sure someone here can guide you through that.

Apart from this automagic stuff, it is possible to mount it via the CLI as well.
If you'd like to know how, post again for further information (we'd need to know which device the drive was recognized as and modify a config file to save some typing and make it easier to mount everything manually).

HTH
