### Post by mlentink on 2007-06-22
1 Make sure your PC can boot from cd. If not, change the BIOS settings (at boot time) to enable this
2. Download the Live-cd ISO [here]("http://www.ubuntu.com/getubuntu/download")
3. Burn this on a cd (make sure to use the 'burn as an image' option
4. Insert siad cd in the cd drive
5. reboot

On the desktop that will appear after several minutes, there will be an icon to install ubuntu on the disk. Follow the instructions and let Ubuntu use the largest free space (most likely your free partition). Let it rip
