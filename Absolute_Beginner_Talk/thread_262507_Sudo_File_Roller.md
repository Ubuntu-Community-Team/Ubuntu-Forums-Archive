How do I sudo File Roller so I can extract a tar.gz to /usr/share/themes ?

---

### Post by croak77 on 2006-09-21
Try running gksudo file-roller from a terminal or a run dialog (Alt-F2).

---

### Post by aysiu on 2006-09-21
Alternatively, you can just do a regular file-roller and then Alt-F2 and ```
gksudo nautilus
``` and drag the folder over to /usr/share/themes
