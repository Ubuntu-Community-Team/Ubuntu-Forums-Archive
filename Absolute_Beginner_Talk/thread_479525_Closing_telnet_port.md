### Post by atria on 2007-06-20
By default all ports are closed by firestarter. Do you have a router? The telnet port might belong to the router instead of your computer. Also verify that the IP that is scanned is the same as your WAN IP.

Anyway i don't think ubuntu comes with a telnet server by default. Same for Windows. So the port should be closed even if you don't have firestarter around.

Firestarter merely "stealths" the ports.
