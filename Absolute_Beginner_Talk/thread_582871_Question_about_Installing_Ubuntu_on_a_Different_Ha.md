### Post by ukposter on 2007-10-20
Yes it is possible to do that. I've got Windows on one h/d and 2 Linux distros on a 2nd h/d. You'll probably be asked to name and /or partition your new h/d when you install Ubuntu. When you boot up your computer you will have to tell it which OS to boot from, there are plenty of references on this forum as to how to do that and to answer your original question.

### Post by pcteacher on 2007-10-20
Quite easy to do on an internal hard drive (installed as a slave if IDE or to Channel2 if SATA).  Both the graphical (desktop) installer and the Alternate installer give suggested options for partitioning etc that work just fine.  
External (USB or Firewire) seem to give problems with GRUB as described in the previous reply.  Should still be possible though.  Perhaps someone else can help??
