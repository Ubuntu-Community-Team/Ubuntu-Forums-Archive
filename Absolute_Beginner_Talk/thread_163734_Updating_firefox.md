### Post by mostwanted on 2006-04-21
Your Firefox version is as updated as it can be. The breezy repositories do not get updated with major new versions, that would cause instabilities with other software that is dependant on the old versions (like Epiphany which uses Gecko, the renderer in Firefox).

You can install Firefox 1.5 with the tool Automatix in a way that will work around your package manager. Try going to the Automatix forum (one of the Ubuntu projects on the front page of this site).
