### Post by TeoBigusGeekus on 2008-04-19
A swap partition is an auxiliary partition in Ubuntu. It is being used as a temporary ram, so to say, when your hardware resources start to scream...
Create a partition no more than 1.5 to 2 gigs, set it to be used as swap and you'll be fine.
