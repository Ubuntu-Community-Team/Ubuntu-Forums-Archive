### Post by tgalati4 on 2007-07-05
Ah, yea.

Unfortunately.

However, hardware detection is getting better.  But there are new architectures for sound, for wireless, for video, for compositing.  So you will need to account for these changes and how they run on your hardware.

It might be prudent to keep your working distro and put a fresh, new install of the latest on a separate partition.

This way, you will always have a predictable, working system, and you can boot into the new distro to check out the new features.

After a while, when you get everything working on the new distro, you can wipe the old distro and put an even newer distro on it.

And the cycle continues.

That's the Ubuntu Genuine Advantage.

### Post by macogw on 2007-07-07
Because new hardware comes out all the time.  The new one every 6 months is to catch up to the new hardware.  Dapper (6.06) doesn't support any hardware that happened in the last year.  It is stable.  Stable means there are not new features and versions added.   That would add bugs.  Bugs are bad.  Stable releases only get security updates to squash the security bugs.  If they just stuck with 1, then everyone with a new computer would be s*** outta luck.
