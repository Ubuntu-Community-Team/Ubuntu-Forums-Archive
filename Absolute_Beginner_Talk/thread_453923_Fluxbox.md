### Post by kerry_s on 2007-05-24
Open up your terminal and run-> sudo apt-get install fluxbox
then in terminal run-> sudo update-menus
now log out and select fluxbox from the session menu and log back in.
 you will be in a bare bones fluxbox setup, nothing fancy yet, just enough for you to start playing with it, to get the feel.

### Post by NeoLithium on 2007-05-24
Fluxbox takes a bit of practice to set it up just how you  might like it. However, it is well worth the effort;I use it off and on as I switch through various window managers and desktop environments; and always found the set up and tinkering to be more fun than work.

Another major plus, is that it has a noticable speed advantage, which is nice when you need nothing but a simple, functional desktop.  Fluxbuntu isn't bad, but it's just as good right now to simply install through the regular ubuntu repositories.
