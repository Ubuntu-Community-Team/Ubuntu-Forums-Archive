### Post by Razorback on 2007-04-17
Download the iso install image, burn iso to create install disk, setup bios to boot from CD, install ubuntu on drive D, setup dual boot in the GRUB bootloader loaded when ubuntu was installed, enjoy!

Setting up the dual boot will keep you from having to change your bios each time.
