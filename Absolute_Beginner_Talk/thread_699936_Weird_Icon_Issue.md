### Post by Anduu on 2008-02-18
Well there are a number of things that could be happening but my first question to you is can you still select the icon theme and have it apply normally?If it does I wouldn't worry about it.
I have a number of them with red x's that still work fine.I am assuming it is because the theme wasn't created properly.

One thing you really need to look out for is poorly created icon themes.They can mess up other themes or even,as I experienced one time, bring Gnome down.

Also in my experience the Appearance Manager can get a little flakey sometimes especially when you have a lot of themes as I do.

In short if everything is still working don't worry about it.If strange things start happening ditch that theme.
