### Post by justsomedude on 2007-10-16
You need a composite manager like Compiz or Beryl to run AWN.

Good news is, Compiz (Feisty) or Compiz-Fusion (Gutsy) are already installed by default.


1. Make sure 3D acceleration works. You can test 3D acceleration by typing 

```
glxgears
```

in a terminal window.

If you see the gears moving, 3D acceleration works.


2. On Feisty, you can start Compiz under *System --> Preferences --> Desktop Effects*. Just enable the effects.

If you're runnin' Gutsy, Compiz Fusion should be enabled by default.
