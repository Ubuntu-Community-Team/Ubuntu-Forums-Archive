### Post by spd106 on 2007-08-28
I suggest that you delete the .mplayer directory from your home directory and then restart mplayer. As long as you haven't modified any system level files such as the /etc/mplayer* files, you should be ok. 
It's also a good idea to install the mozilla-mplayer plugin from the Ubuntu repo.
