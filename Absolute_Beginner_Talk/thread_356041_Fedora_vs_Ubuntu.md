### Post by jordanmthomas on 2007-02-08
You won't get any better hardware support in Fedora than you will in Ubuntu.
Fedora simply includes a lot of software (games, graphics programs, administration tools, etc) that the Ubuntu CD does not.  Ubuntu does have a DVD if you'd be interested in lots of programs on a disc.

Fedora is actually a little worse (through my experience) about hardware...for example, Ubuntu includes firmware for my wireless card while in Fedora I had to install the ipw2200-firmware package.

Both Ubuntu and Fedora are great though.

### Post by igknighted on 2007-02-08
> **jurianbotha said:**
> Why is the Fedora Core installation package so much bigger that the Ubuntu installation package?

My worries are that Ubuntu will not have all the necessary drivers to get my laptop working smoothly. (Acer Travelmate 8005LMi)

The drivers are mostly contained within the kernel, so there shouldn't be much difference.  Also, consider what is on the Fedora DVD: Desktop apps, Development apps, and server apps.  On top of that, Ubuntu has a stated goal of one program per task and no more.  This is the reason Fedora is so much bigger, they include many many many more apps. (Note: this can either be a good thing or a bad thing, depending on your internet connection... if you have dialup you may prefer the Fedora DVD so you dont have to download programs, you can install from the DVD, while if you have high-speed internet you may prefer Ubuntu because you can always just download the software).  Also, keep in mind that Fedora and Ubuntu both share the free-software philosophy, so commercial drivers (such as video drivers and most wifi drivers) wont be included in either distro.

Your personal hardware will be supported differently and recognized differently by various distro's, however.  Therefor, before you settle on one distro you might consider a few, maybe install each (or at least run a live CD) to see what supports your hardware the best.
