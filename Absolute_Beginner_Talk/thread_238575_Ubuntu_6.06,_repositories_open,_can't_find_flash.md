### Post by Malakia on 2006-08-18
Go here [http://ubuntuguide.org/wiki/Dapper](http://ubuntuguide.org/wiki/Dapper) scroll down a little until you find the contents listed. Its number 4 that you click on and follow those directions exactly including running sudo apt-get update. If you don't run sudo-apt-get update it will still say the same thing, that it can't find the flash player. The instruction for installing the flash player are also on this page too. This guide is very good resource to use incase you want to install other things such as java runtime or adobe reader.
