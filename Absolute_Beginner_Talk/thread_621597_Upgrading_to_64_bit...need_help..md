### Post by rsambuca on 2007-11-23
It depends on the applications that you are using.  If you just surf the net, send a few emails, and use a work processor, then no, you probably won't see a difference.

If you do any tasks that are cpu intensive such as video/audio transcoding (like backing up DVD's, and CD's, or editing home movies), 3D rendering, etc, then you can have upwards of 30% performance gain by using 64bit OS and app's.

### Post by PmDematagoda on 2007-11-23
Since you have 4Gb of RAM, then it would mean a good performance boost if you use 64 bit since you can then use all 4Gb of RAM, which would mean a 1Gb addition to the 3Gb that is usually detected under 32 bit. If you wish to stay with 32 bit though, you can recompile a kernel with support for 4 Gb of RAM, otherwise you can simply install and use Ubuntu x64 which is better and more easy to use, especially as the installation of Java and Flash plugins for Firefox has been made more easier in Ubuntu 7.10.
