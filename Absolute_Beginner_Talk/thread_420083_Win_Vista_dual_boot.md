### Post by silverglade00 on 2007-04-23
ONLY use the Vista partition manager to shrink your Vista partition. Right-click Computer -> Manage -> Disk Management -> Right-Click Vista Partition -> Shrink...
Type in the TOTAL number of MB you want for all of your Linux partitions. Click OK and go watch a movie. Then do everything else with the Ubuntu disk. This will eliminate the corruption problem.
