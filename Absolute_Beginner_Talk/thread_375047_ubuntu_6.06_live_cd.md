### Post by Carpe.Diem on 2007-03-03
> **prrao3 said:**
> ...what i exactly want is LIVE CD link(so that os can be booted from cd whenever i want so that i could use ubuntu without installing it on my pc)

The current Live-CD version of Ubuntu (version 6) may not do what you think it does; however, some other Live-CD versions of Linux do actually accomplish what you want.

Here's some of what I assume you may want:
[LIST=1]
[*]To be able to boot your computer without using your hard disk.
[*]To be able to browse the web and send and receive email in a basic manner.
[*]To be able to play music and videos.
[*]To be able to play games or entertainment you might want to add that are not part of the Ubuntu distribution.
[*]To be able to add other programs or features that are not part of the Ubuntu distribution.
[/LIST]
_Unfortunately, Ubuntu's Live-CD seems to only give you 1 and 2 above_.  This is a VERY unfortunate oversight when the term Live-CD is used.  The term "Live-CD" is **not** supposed to mean stripped-down or almost non-functional in terms of the 5 items I listed above.

I have been unable to play music or videos with the Live-CD Ubuntu, even with the help of nifty sounding Easy Ubuntu.  Adding anything to the Live-CD version of Ubuntu will be completely lost when you shut down your computer, as nothing whatsoever is or can be saved it seems.

This is NOT true of all Live-CD versions of Linux.  Some, if not most, versions of Live-CD's allow you to do items 1, 2 _AND_ 3, and others allow you to do all 5.  Some versions of Linux Live-CD's allow you to either save your modifications to a single file on your hard disk and some even save your modifications to the CD itself (this is called multi-session Live-CD, like [Puppy Linux]("http://www.puppylinux.org/"), for example) -- in which case, if you have enough basic memory, you don't even need to have a hard disk on your computer.

My single BIGGEST frustration in working with wonderful Ubuntu is the lame Live-CD workings that seem to be implemented.  I certainly wish it were otherwise!  Since I cannot play music or videos totally turns me off and makes me feel the Live-CD label is misused!  If I want to write a letter, I can neither print nor save it to disk without jumping through arcane command-line gymnastics, which I'll have to do again and again every time I start the Ubuntu Live-CD.  I really feel the purpose of the Live-CD is to give you a *little look* at how Ubuntu appears (not works!) so you'll install it, and THEN maybe you'll be able to do everything if you can just figure out how (and these forums are GREAT in that regard)!

**If Ubuntu is meant for the common and ordinary person, this is a BIG mistake and Ubuntu will never get anyone other than tech-types (and this makes me sorry).  Ubuntu's philosophy is TOO good to be handicapped by the current implementation this way!  It should be approachable and useful to non-computer types in order to reach its awesome and inclusive philosophy which embraces everyone!  Right now, Ubuntu is NOT inclusive, it's "exclusive" (you've got to be a "certain type of adept" in order to deal with ALL that needs to be done to get it to work at all) -- and this makes it inapplicable to the common man.  How counter to the expressed philosophy!**

Your goal of having a Live-CD do everything which can reasonably be done with Linux cannot be accomplished with Ubuntu's current offering, but you CAN indeed do most if not all of these things with other Live-CD versions of Linux (but certainly not all of them).  Try this [link]("http://www.frozentech.com/content/livecd.php") to see what else is available (but you've got to be prudent and selective as the number of options is amazingly large) -- and good luck!

I say most of the above with sadness.  Ubuntu is TOO good to be SO unapproachable!
