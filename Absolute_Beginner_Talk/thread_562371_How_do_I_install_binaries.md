### Post by Technophobia on 2007-09-28
Right Click the file and goto properties

next goto the permissions tab

select "allow executing file as program" check box.

hit close.


I think you can down double click on it to run. If not open a terminal and goto the folder where the files is located and type "sudo ./<YOUR FILENAME HERE>". No <> around the file name
