### Post by Azoix on 2007-07-29
[B][FONT="Comic Sans MS"][SIZE="3"][COLOR="Blue"]Start by turning the machine off, then remove the battery, and power cord, leave it 10 minutes, without power , use this time to remove the cover and remove and re-insert the RAM, just in case it is the culprit (it can be) i have an old IBM A20m that does the same thing, in my case the RAM slots are worn and don't maintain GOOD contact, the removal and re-installation seems to fix it each time.
Now reconnect the power cord only, no battery, boot the machine and see if you can access the BIOS settings.If you can, reset these to default, then save and exit, re-enter the BIOS when the machine reboots, and set it for NON plug and Play O/S ( if this is available in the options) save and exit, the other defaults should all be OK.
Reboot and see if you get it to the desktop again.
If this doesn't work, you may need to either replace the RAM or HDD, or possible flash the BIOS.
[/COLOR][/SIZE][/FONT][/B]
