### Post by aysiu on 2005-10-09
There should be a little notification button that lets you know of new updates. If you click on that, it'll prompt you to install updates.

Other than that, if you want to manually upgrade things as you go, type this into a terminal every now and then ```
sudo apt-get update
sudo apt-get upgrade
```

If you're switching from Hoary to Breezy... well, that's a whole different issue.
