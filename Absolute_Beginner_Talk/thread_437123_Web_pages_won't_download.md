### Post by annda on 2007-05-08
or it COULD be an IPv6 issue with firefox (some routers don't support it). type 'about:config' in the address bar and hit enter.
use 'ipv' as filter in the box on top of the page and make sure that 'network.dns.disableIPv6' is set to 'true' - if it's 'false', double-click the line and restart FF.
