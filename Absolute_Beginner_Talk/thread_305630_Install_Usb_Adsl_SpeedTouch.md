### Post by fishpie on 2006-11-23
It depends which guide you mean. There are two speedtouch drivers for linux a usermode driver and a kernel driver. Both drivers will work with 6.06 but only the kernel driver will work with 6.10. This guide [http://www.linux-usb.org/SpeedTouch/ubuntu/index.html]("http://www.linux-usb.org/SpeedTouch/ubuntu/index.html")
Explains how to use the kernel driver, the instructions don't mention 6.10 but following th instructions for dapper will work
