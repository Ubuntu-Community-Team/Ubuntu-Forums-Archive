### Post by kbsuperstar on 2006-09-23
I've got the normal Ubuntu 6.06 with Gnome right now, and I was wondering if there was a way to change it to Xubuntu or Kubuntu without reinstalling Ubuntu. Also, will I have to reinstall everything, like drivers, programs, etc?

---

### Post by pay on 2006-09-23
No you don't have to reinstall drivers and programs. Just click Applications>Accesssories>Terminal and type into the terminal```
sudo aptitude install xubuntu-desktop
```log out and click Options>Sessions and XFCE.
Goodluck

### Post by aysiu on 2006-09-24
Probably wouldn't hurt to update first: ```
sudo aptitude update
sudo aptitude install xubuntu-desktop
```
