### Post by bscbrit on 2006-02-04
Ok, you have to enter each of the commands on a command line or terminal window.  To open the window Applications -> Accessories -> Terminal.

Many of the commands will require you to have root privileges, so you will have to use the 'sudo' command in front of each of them, or change user to become root.  I recommend (very strongly!) that you do the former.  This link explains the whys and wherefors : [https://wiki.ubuntu.com/RootSudo](https://wiki.ubuntu.com/RootSudo)

Start the troubleshooting guide at the beginning.  Run each command and ensure that you get the response that you expected.  When you hit a problem (either with the troubleshooting guide or with your hardware) come back for help.
