### Post by Rocket2DMn on 2008-03-05
In FF, go to Bookmarks->Organize Bookmarks.  In the new window go File->Export and you can save the favorites as an html file which you can then import into FF on the new machine using the same method (select Import instead of Export).
If you use both machines, you may also consider the FF addon called Foxmarks so you can share your bookmarks.
