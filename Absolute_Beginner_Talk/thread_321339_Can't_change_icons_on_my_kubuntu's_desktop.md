### Post by timboe73 on 2006-12-19
The easiest way i have found is the create you own shortcut. If you right click on an icon already on you desktop and select properties then click the launcher tab, it will give you the command you need to start that program.

Now if you right click the desktop again and select "create launcher" you can give it anyname & icon you like, just copy the original command into the new command & you should be right.
