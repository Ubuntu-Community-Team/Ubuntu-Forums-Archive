### Post by radagast2 on 2006-07-08
You might try NIS ([http://g-scripts.sourceforge.net/nautilus-scripts/Multimedia/Image/NIS)](http://g-scripts.sourceforge.net/nautilus-scripts/Multimedia/Image/NIS)), a Nautilus script that uses ImageMagick to resize images.  Once you install it (by making the script executable and copying it to ~/.gnome2/nautilus-scripts), all you have to do is select (in Nautilus) the images you want to resize, right click on them, and select NIS from the scripts menu.  

If you want the images renamed as they're resized, I've modified the script to do that.  You can find my modifications here: [http://rhosgobel.blogspot.com/2006/07/bulk-resizing-and-renaming-images-in.html](http://rhosgobel.blogspot.com/2006/07/bulk-resizing-and-renaming-images-in.html).
