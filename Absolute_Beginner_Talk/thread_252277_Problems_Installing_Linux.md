### Post by doobit on 2006-09-06
> **ConnorHB said:**
> Um, I just burned a disc with it on it and it just showed the ISO file, that's it.

You don't have to redownload it. If you have the .iso file then you open your CD burning software and find the part of the menu that says something like "burn CD from disk image" or "make CD from .iso" or something similar. Then you select that and hit enter or OK. Another window will pop up and you need to navigate to where you have the .iso file and select it. then burn a new CD from that window.
