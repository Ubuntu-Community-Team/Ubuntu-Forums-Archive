### Post by thelatinist on 2008-01-21
The terminal is a basic text interface that allows you access to all of the functions of the Linux system.  It's very useful for configuring and troubleshooting your system and for easily offering instructions to others.  You can find it at Applications > Accessories > Terminal.  To run a command, you just type it at the prompt and hit return.
