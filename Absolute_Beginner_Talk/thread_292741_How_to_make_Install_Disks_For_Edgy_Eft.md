### Post by catlett on 2006-11-04
If you don't mind waiting a few weeks, Canonical (the parent company of Ubuntu) will send you installation cd's for free. [https://shipit.ubuntu.com/](https://shipit.ubuntu.com/)
Updates are different. The cd's only contain the initial install. Updates come via internet.
If you have access to a computer with broadband internet access, you can make   your own repository cd's.[http://cargol.net/~ramon/ubuntu-dvd-en](http://cargol.net/~ramon/ubuntu-dvd-en)  That way you can add packages to your other systems without internet access.
