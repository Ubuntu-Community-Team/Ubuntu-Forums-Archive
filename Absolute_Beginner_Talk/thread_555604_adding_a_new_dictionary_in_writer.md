### Post by Maestro23 on 2007-09-20
Taken from the following thread over at Openoffice.org: [http://www.oooforum.org/forum/viewtopic.phtml?t=62371&highlight=dictionaries](http://www.oooforum.org/forum/viewtopic.phtml?t=62371&highlight=dictionaries)

> You must make sure that you have the dictionaries you need. Go
Tools>Options>Languages Settings > Languages and see if the language you want has a check mark beside it. See this for an example [http://ca.geocities.com/jrkrideau/images/Checkmark.jpg](http://ca.geocities.com/jrkrideau/images/Checkmark.jpg)) Only languages with a checkmark are actually on your computer. If it is not there go to
File>Wizards>Install new dictionaries and follow the instructions from there.
