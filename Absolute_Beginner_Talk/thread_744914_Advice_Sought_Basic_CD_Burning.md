### Post by amingv on 2008-04-04
If you close session on a CD-R or a CD-RW, the disc will always appear as full to a CD reader/burner; if you wish to reuse a CD-RW you have to erase all the content and burn it over.

On most burning software you can create a "multisession disc", what this does is leave the session open so you can write several tracks of data into one CD. (Although for a regular CD reader, the disc will always appear as full.

If you're burning from an ISO, the session will most likely always be closed.
