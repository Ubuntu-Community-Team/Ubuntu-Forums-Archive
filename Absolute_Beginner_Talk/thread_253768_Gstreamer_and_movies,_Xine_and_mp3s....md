### Post by croak77 on 2006-09-09
For xine you need the libxine-extracodecs package. For DVD's you need libdvdcss2 but I would not recommend using totem-gstreamer to play DVD's. It only plays automatically when you insert the dvd. You can't navigate to it or select it from the menu. I would use xine and either gxine or totem-xine. I use kaffeine which is a KDE frontend to xine.
