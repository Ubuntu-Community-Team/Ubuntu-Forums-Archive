### Post by Qrk on 2005-09-06
There could be a few problems.
You may not have the whole iso image... make sure it is the right size. Often the download cuts out half way done. 
Also use the burn iso image to CD, not just copy paste, but you probably did that right. Make sure you check the 'bootable tag' if you need to. I have never used that burner. I used the trial version of Nero, which may work better. 

One other thing... there are no .exe in Linux. It'll help later. Linux executables are shell scripts with execute permissions.
