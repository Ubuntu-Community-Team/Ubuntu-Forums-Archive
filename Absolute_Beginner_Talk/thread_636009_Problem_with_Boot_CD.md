### Post by parhelia on 2007-12-09
It is a known "problem" that WinRAR hijacks the .iso file format. You should not open with WinRAR. Instead, burn the .iso file as an CD image (see the appropriate instructions of your burning program). After burning, you should in Windows see many files, and not one large.

Let this help you: [http://screencasts.ubuntu.com/Downloading_and_Burning_an_Ubuntu_ISO]("http://screencasts.ubuntu.com/Downloading_and_Burning_an_Ubuntu_ISO")
