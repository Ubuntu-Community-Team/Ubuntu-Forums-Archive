### Post by Hinko on 2005-11-03
No, you can safely remove all gnome programs (the ones you still want to use will work with KDE as well (synaptic installs everything needed for running these programs.)

KDE programs are independend from gnome programs, which is also true the other way around. Usually you run either gnome or KDE, but you still use some programs from the other desktop anyway. For instance I use Kile on gnome because I think there's no good alternative. Most programs however are neither KDE nor Gnome but just independend.

You can just go to synaptic and uninstall everything you don't want anymore. If there are any dependencies that will conflict with programs you want to keep synaptic will warn you about that.
