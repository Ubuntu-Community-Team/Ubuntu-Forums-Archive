### Post by LaRoza on 2007-12-24
Dependancies == Libraries and other programs required by programs. For example, Opera requires QT libraries. (Kind of like DLL's in Windows)

Universe and multiverse repositories == Collection of software for Ubuntu. They are where the updates and programs available to install are located. You can add other repositories.

Wget commands == wget is a program to fetch content from the network (internet). If you know what you want, it is faster and easier than trying to download it is a browser, and works in the terminal

Repository == See above

Grub == Boot Loader. A small program that is run when you turn your computer on (not technically, but close) It is what boots Ubuntu. If you have other Operating Systems installed, it can boot them too. I use it to boot Ubuntu and Windows and have used it to boot 14 different Operating Systems on one computer

Hope it helps
