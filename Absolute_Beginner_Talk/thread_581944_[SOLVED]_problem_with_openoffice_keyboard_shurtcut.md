edit: Solved it. Damn, you only start seeing it when you see it... If you have this issue:
make sure that in the customize field you select under Category **insert** and then footnote and then assign you're favorite keyboard combination.

There is also a footnote option under Edit and this will not work. The categories are corresponding with the ones  above the toolbar in Openoffice. Seems logical... There is no working Footnote option under edit so that is why a shortcut will not work.  There is a working footnote option under insert, so if you want to make a shortcut: select insert in category footnote in Function  and favorite keyboard shortcut in keys. This will work. Simple if you know it ;)
