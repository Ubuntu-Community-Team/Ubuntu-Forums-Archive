### Post by crossedout on 2007-01-24
Well, I am trying to figure what might be preventing you from viewing the video.  To help me understand a few things:
- You can see the images on the 'Full Episode Player' page?
- Firefox 2.0?
- Any other extensions dealing with JS or Ads?
- Did you remove the old version of flash?

The last one is important because the sites that are running properly may have also run properly with flash7 and so you wouldn't know that FF was using the old version until you tried a flash7+ only site.
Open a new tab and type, about : plugins (no spaces) in the address bar.  Look for 2 entries of libflashplayer.so.  One of them will be the new v9, the other v7.  Obviously if there is an entry for v7 then you will need to remove that so that FF defaults to the v9.

As far as AdBlock goes, if you're showing 0 blocked items then AdBlock isn't the problem.

-Xout
