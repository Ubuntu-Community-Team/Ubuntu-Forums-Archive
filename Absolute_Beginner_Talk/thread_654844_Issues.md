### Post by JoshuaRL on 2007-12-31
I'm not sure, but I think that OSX is locked down to Apple hardware.  It would work on other hardware, but normal PCs don't have a special chip that tells the OS that it's official.  So probably not, sorry.  That is, unless this was a Mac machine to begin with.  Then you probably could if the specs met the requirements.

And my desktop looks like OSX kinda.  I run Xubuntu because it just runs flat faster.  As far as anything else, try gDesklets in the Add/Remove app.  gDesklets has a launcher widget that looks and acts like the launcher in OSX.  Then go to Menu>Settings>Desktop and pick the Compositing tab.  There you will find options to run transparency.  That makes it look like a OSX/Vista clone, but so much better underneath.
