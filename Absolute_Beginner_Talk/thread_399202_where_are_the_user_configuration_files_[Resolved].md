### Post by NeoLithium on 2007-04-02
You're not a dummy, don't worry.  any files that are listed with a . before them; are hidden from the general windows; so you kind of have to work to find them.  If you have a window open to your /home directory, type CTRL+H; and the hidden files will appear for you, and you can re-hide them with the same command.

(~/) is a nice shortcut that means your home directory :)
