### Post by parmdeep on 2008-02-21
Sound and notifications can be switched on via the plugins menu of emesene(options,plugins) when you are logged in.
the file transfer is a known issue there may be a work around for it.

If live messenger is the onli thing that is holding you back from making the "switch" i would strongly recommend installing live messenger via WINE.

There are many tutorials on how to install wine. [www.winehq.org](www.winehq.org) is their homepage where you can find the lastest release of it, once wine has been installed download the live messenger client and open up a terminal

type the following: cd Desktop (assuming the desktop is your default download location)
then type wine followed by the exact name of the live messenger installer file with the .exe

please dont let something as trival as a messenger client affect your operating system ;o) enjoy the power of freedom
