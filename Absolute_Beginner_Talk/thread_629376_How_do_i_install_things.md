### Post by corney91 on 2007-12-02
> **mattfatrat said:**
> I downloaded Beryl and when i opened it came up in the archive what do i do next?

You shouldn't need to download anything. You can install things from Add/Remove Applications, Synaptic Package Manager, or the terminal.

Are you running Ubuntu 7.10 (Gutsy Gibbon) or Ubuntu 7.04 (Feisty Fawn)?

If you are running 7.10 , then you have Compiz Fusion installed which is the latest version of Beryl (Compiz and Beryl merged) and all you have to do is install the settings manager.

### Post by jken146 on 2007-12-02
If you want to get the cube etc. then you don't need to get Beyl.  You need to install the package compizconfig-settings-manager and then use System > Preferences > Advanced Desktop Effects Settings to set up what you want.

If you're unsure about how installing programs works in Ubuntu, I suggest this guide: [http://monkeyblog.org/ubuntu/installing/](http://monkeyblog.org/ubuntu/installing/)
