Midnight Commander will rule forever.

sudo apt-get install mc

mc

Hit F9, then cursor right to Configurations, pick Lynx-like motion and F10 get out.  You may need to tell the Gnome terminal to ignore the F10 key so that mc handles it.


It is the fastest file manager out there.
