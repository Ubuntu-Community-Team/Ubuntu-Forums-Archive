### Post by crimesaucer on 2007-05-15
Besides the necessary Fasterfox, Adblock Plus, Adblock Filterset. G Updater, Tab Mix Plus, and 
Ubuntu Forums Menu...

I couldn't keep it 5, so I put 10:

1.) Media Player Connectivity

2.) Gmail Manager, and also (a.) Gmail Space (b.) Gmail Skins (c.) Stylish to make it look better

3.) Mr Tech Local Install - for installing add-ons with no delay, theme switching in the menu, and a nice restart button, Plus other features.

4.) Measureit - for measuring any thing in exact pixels which helps design things like web pages.

5.) BetterSearch for thumbnails for Google, Answers.com, del.icio.us, and other search pages.

6.) Forecastfox Enhanced

7.) CuteMenus - Crystal SVG

8.) ScrapBook - for saving whole pages with pictures that can be view off line.

9.) FireFTP

10.) DownThemAll
