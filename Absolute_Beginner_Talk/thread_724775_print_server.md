### Post by spiderbatdad on 2008-03-15
This should be fairly easy to set up. Is your print server connected to a hub or switch?
Just go System>>Administration>>Printing and select App socket/HP direct jet..  and input the ip address of the print server...next...make and model.  If you don't know how to find the  ip address of the print server, try right clicking on the printer in windows select properties>>ports.
