### Post by Najand on 2006-09-11
Ubuntu is usign Gnome as default Desktop Manager... They made another versions with other Desktop Environment Like KUbuntu which is UBUNTU with KDE, XUBUNTU which is UBUNTU on XFCE. They are some other unofficial ones like, nUBUNTU (For security purposes), and fluxUbuntu (Ubuntu with fluxbox)... 
They made it because you generally need one Desktop Environment... People who work with gnome, rarely use with KDE and reverse. So they made different editions.... It is very easy to convert one to another by:
```

$ sudo apt-get install EDITIONNAME-desktop

```
which EDITIONNAME can be kubuntu, xubuntu or edubuntu.
