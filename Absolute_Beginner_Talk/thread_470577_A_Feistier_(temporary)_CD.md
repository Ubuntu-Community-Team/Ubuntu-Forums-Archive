### Post by paradoc on 2007-06-11
When using a feisty live CD to play a youTube video,it tried to download the Flash player , but came up with the message 'cannot open unsupported .rpa file'

This rang a bell with me, as I had recently read an article about a blanket install, which included Flash player and many other apps in one convenient download [www.pcw.co.uk](www.pcw.co.uk), search for 'hands on ubuntu' and the article is 'Hands on : Ubuntu 7.04' by Barry Shilliday.On page 2 he details how to download and install the several packages automatically, which I did from the Feisty CD
.
Applications>Add/Remove>restricted + all applications, 
This conveniently downloads 30 files, and on 'Apply' a further 20, which are duly installed into RAM for the live Feisty, which can now open the videos on youTube.

I soon realised that this has important ramifications for the installed Feisty, and can save hours of introspection and head-scratching when looking for many of those missing applications which turns Feisty into a more pleasurable experience at one fell swoop!

### Post by Tux Aubrey on 2007-06-11
I haven't read the article, but it is good that someone is enhancing the live CD experience (although five minutes was enough to convince me to install ubuntu and get on with the Real Thing!).

Proprietary codecs, flash and a whole heap of goodies are readily available from the repositories if you enable multiverse and universe. Codecs can be installed via add/remove and there are very straight-forward instructions for everything [right here]("http://ubuntuguide.org/wiki/Ubuntu:Feisty").  It seems a lot of people make a great deal of trouble out of installing these things, but it really is very easy if you follow the guide.
