You can download the mint-menu *.deb(Debian) package from mint's APT Repositories and install it on ubuntu.
However it depends on 3 or 4 other mint packages. such as mint-desktop, mint-disk, and mint-config

mint-menu is very nice but it is scripted in Python so it has a tendency to get laggy
when the system is under a load;
I installed it under 7.04 but haven't bothered on Gutsy.
