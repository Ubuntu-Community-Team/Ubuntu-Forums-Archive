### Post by jmtjet on 2006-12-26
The beeps you're hearing are beep codes. You'll need to find what BIOS version your computer is using(e.g; Award, Phoenix etc.)Then count the code beeps(e.g;3 short/1 long etc.) Then post back with the code and someone will help you diagnose the problem. It's usually a hardware problem when the BIOS starts giving you beep codes.
