### Post by linear on 2006-08-20
Some possible solutions:

1. Compare the checksum of the iso file with that posted at the download site. If it doesn't match, you'll need to re-do the download.
2. Try burning the CD at a slow speed (4x or less).
3. Try the alternate install CD.
4. Try a Breezy CD and then do a dist-upgrade.

This problem comes up from time to time, but there doesn't appear to be one sure solution for everyone...

(Oh, and try booting from the CD by holding down the "C" key after the startup chime, or hold down the option key to get a menu of bootable disks.)
