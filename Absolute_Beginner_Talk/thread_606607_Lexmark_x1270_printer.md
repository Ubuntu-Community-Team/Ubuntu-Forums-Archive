### Post by P4man on 2007-11-11
Go to system / hardware / printing. Is your printer listed there ?
If so,  did you try printing a test page ?
If not, click "new printer" and see if it is discovered.

If it is listed, but not working properly,  select the printer and look for "make and model" in the settings tab. Click the "change " button next to it, and see if there are any other drivers that work with it


edit: I just checked, it seems like drivers for your printer are not installed by default. That a bit of a bummer. If the above doesnt work, have a look here:

[http://ubuntuforums.org/showthread.php?t=49714]("http://ubuntuforums.org/showthread.php?t=49714")

If you need help with that (pretty likely) let us know.
