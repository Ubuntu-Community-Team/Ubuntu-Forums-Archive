### Post by editrix on 2006-09-07
If you're dual booting, you don't have to transfer them. Just create a fonts directory in your home folder (or wherever you want them) and then a symlink--at least, that worked for me with Kubuntu.

Or you can install the msttcorefonts package with Synaptic.
