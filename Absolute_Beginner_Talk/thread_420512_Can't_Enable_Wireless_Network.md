### Post by jfinkels on 2007-04-23
> **jk001 said:**
> I just installed version 7.04 and the installation went flawlessly but I am unable to activate my wireless network card via System->Administration->Network.  I looked in the device manager and my network card shows up on the list. Any thoughts?

Select your wireless card in the Network Settings manager, press Properties, select "Enable this device", type in the ESSID (the name of the wireless network), any necessary passwords, then press Okay.

If you have any problems with any of these steps, post back here :)

Thanks!  I was confused when I first looked at the properties because there was a box labeled "Enable Roaming Mode"  that was checked.  I unselected the box and then I was able to enter my network information.  I am now able to access the internet!
