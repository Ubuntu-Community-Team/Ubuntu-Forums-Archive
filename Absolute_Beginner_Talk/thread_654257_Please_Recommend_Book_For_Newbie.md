### Post by pjones0404 on 2007-12-31
i can not recommend specific books but i would get 2 books each one focusing on the following subjects.

linux/terminal - one book that goes over the fundamentals of how Linux works and how to manipulate the OS via the terminal. i would stay away from specific distributions for this book and get one made for linux in general. the nutshell books are always nice. 

Ubuntu/Hacks - this book should focus more on using the graphical interfaces that are available. subjects should include desktop environments and the usual applications that are used in each. it could even go into shell scripting if needed. 

you would not have to read these in any specific order but i think that it is easier to break down into subjects.
