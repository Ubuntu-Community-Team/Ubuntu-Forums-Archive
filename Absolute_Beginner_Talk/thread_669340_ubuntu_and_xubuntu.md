### Post by sub2007 on 2008-01-16
They are the same base system but use different desktop environments and so have a different look and feel. Ubuntu uses [GNOME](http://www.gnome.org/) and xubuntu uses [XFCE](www.xfce.org). They also have different programs included, for example Ubuntu includes OpenOffice.org and XFCE includes AbiWord and Gnumeric for your office needs (it's easily possible to install OpenOffice.org is you want), It's possible to change between Ubuntu, Xubuntu and Kubuntu at any time (and even have all three installed at the same time).

I find XFCE much more customisable than GNOME although some things are more manual in XFCE (I would recommend GNOME to beginners). The base XFCE is lighter than GNOME on system resources and so may be a better choice for older hardware. It can still run Compiz and all those pretty effects though.
