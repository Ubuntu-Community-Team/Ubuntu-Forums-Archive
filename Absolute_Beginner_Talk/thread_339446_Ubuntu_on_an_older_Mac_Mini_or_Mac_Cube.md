### Post by irish_flu on 2007-01-15
I ran "Breezy Badger"  (two releases back) on an old "Blue & White" G3/300MHz with 512MB RAM.

It wasn't the most awesome thing ever, but it ran much better than OSX did on the same system; I only say that because the "old" Minis run OSX just great (much better than my poor old G3), so I have to assume that Ubuntu would as well.

Just make sure that you pick "Mac (PowerPC) desktop CD" when downloading.  I enjoyed using Ubuntu on that platform, and I hope you do too.
