### Post by aysiu on 2005-12-06
[QUOTE=Patrissimo]Nope, nothing else times out.
Could it be a Firefox issue? Is Firefox somehow different for windows and ubuntu/linux?[/QUOTE] I haven't experienced an issues myself, but that doesn't mean they don't exist.

You can do some other things, though, to see what the problem is:

1. install and use other browsers: Internet Explorer (using Wine), Opera, or Epiphany, to see if you have the same problem.

2. Use an email client like Evolution or Thunderbird to see if you have the same problem.

If another internet browser works with your email, you know it's a Firefox issue. If the other browsers don't work, but an email client does, you know it's a browser issue. If neither works, you may have something with your internet configuration stopping connections to email servers for some strange reason.
