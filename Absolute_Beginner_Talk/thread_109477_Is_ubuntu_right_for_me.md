### Post by piedamaro on 2005-12-28
It is right, of course. Any modern distro is reasonably compatible when it comes to command line use so you could be fine with other distro too.
Your work on java depends on the sun sdk, or on gcj the open source java compiler even capable to produce native java binaries but has less API than jse-1.5. So given that sun sdk or gcj are installed /installable every distro shold be the same.
That said, stick with ubuntu!! ;)
