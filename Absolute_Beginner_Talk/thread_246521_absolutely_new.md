### Post by davmac on 2006-08-29
I've not done this for a while but, from memory you have to make the bin file executable and then run it. To do this start a terminal window (applications/accessories/terminal). Navigate to the directory where the bin file is located.

Type "chmod +x nameofthefile.bin" (without the quotes)

Type "./nameofthefile.bin" and it should take you through the install process

But rather than doing it this way, I've found it much easier to use a script called Automatix to get everything like this set up after a fresh install.

There's a guide to installing Automatix at [http://getautomatix.com/wiki/index.php?title=Installation&Itemid=38#Installing_on_Ubuntu_6.06_Dapper_Drake](http://getautomatix.com/wiki/index.php?title=Installation&Itemid=38#Installing_on_Ubuntu_6.06_Dapper_Drake)

If you follow the steps here, then run Automatix (applications/system tools/Automatix) it takes you through all the steps of getting realplayer, mp3 and video play set up.

Hope this helps,

Dave Mac
