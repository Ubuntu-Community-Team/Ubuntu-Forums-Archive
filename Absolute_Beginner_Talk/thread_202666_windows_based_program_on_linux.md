### Post by IYY on 2006-06-23
Generally speaking, no. You can't run a program compiled for one system on another (even if the systems are as similar as Ubuntu and Redhat). However, there is a program called Wine: it stands for Wine Is Not an Emulator, but it acts as one. It allows you to run many Windows programs inside Linux, but don't count on it to make your programs work perfectly. Some work well, some work so-so, others don't work at all.
