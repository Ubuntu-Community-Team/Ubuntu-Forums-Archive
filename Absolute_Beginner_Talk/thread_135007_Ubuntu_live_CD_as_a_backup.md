### Post by aysiu on 2006-02-23
You *can* use Ubuntu live as a recovery, but Knoppix is really built to be a recovery CD.

For example, it's a lot more difficult to mount a partition with the Ubuntu CD than it is in Knoppix.

### Post by GreyFox503 on 2006-02-23
Yeah, Knoppix seems to be better at hardware detection, and adds some nice touches like displaying all the partitions it finds on the desktop.

If you get the DVD version, there is a crazy amount of software on there (the K-menu is HUGE), but the CD version is probably all you need.
