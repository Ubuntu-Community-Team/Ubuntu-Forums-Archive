First, welcome to Ubuntu. Yes, a new OS is somewhat hostile, I understand the sentiment.

In terms of installing, there is an install icon on the desktop that will guide you through the installation including partitioning if needed. If you want to manage your partitions without installing run the program Gpared.

Here is an installation guide : [https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall)

And a guide on Gparted : [Gparted Documentation](http://gparted.sourceforge.net/documentation.php)

In terms of hardware/ internet connection, things should basically work out of the box. It is a real pain in the neck if you hardware is not recognized, as it sounds like in your case. In order to help I suggest you start a thread in ABT asking for assistance. Be sure to describe any hardware you are having a problem with.

Worse case scenario (as with all OS), you may to replace some hardware or add ram. This can be done hopefully with minimal expense, but hardware compatibility exists with ALL OS, and Linux has the widest range of compatibility. However, as with any OS, there will always be incompatible hardware.
