### Post by rdd on 2006-11-03
I don't think actual syncing will be possible. Thunderbird does however, use the same profiles in both the windows and the linux version. So you could share your profile folder on the windows-box and write a script that gets your profile from there before you start tb on linux and one that copies it there after you close tb to save the changes. 

It is the dirty way, but I couldn't think of anything else.
