### Post by mivo on 2007-11-07
The cube is already built-in, and the easiest to do. :) Go to *System -> Administration -> Synaptic*, and search for "Compiz configuration settings manager". Install this package, then go to *System -> Preferences -> Advanced Desktop Effects Settings*. Turn on the 3D Desktop Cube and Rotate Cube. If it does not work (i.e. you only have a 2D pane instead of a 3D cube), click on the workspace switcher (should be in the right lower corner) and change the number to 4. You use the cube by holding down the mousewheel button over an empty spot on your desktop and moving the mouse.

Note that you might need to install the right drivers for your video card if you haven't done this yet.

Try out some other plugins, too, and look through the options. The rest is hunting down wallpapers, skydome graphics, artwork for the cube, etc. It's a lot of fun to create a beautiful desktop :) (very addictive too!).The taskbar in the video is Kiba Dock, but Avant Window Navigator looks similar and is easier to install (takes 5 minutes, just read the how-to).
