### Post by esodin on 2007-04-10
"A complete installation" may be a bit ambiguous. I think there is, overall, more than 20.000 packages that can be installed under Ubuntu. There are separate packages for Ubuntu and Kubuntu that contains the basics for Gnome or KDE respectively; but these packages also contain the most common used software (Openoffice, Firefox, various media players, etc).

There is no one package that has it all. All you need for a complete installation of the OS is a LiveCD (DVD) for either Ubuntu 6.10, Ubuntu 6.06, Kubuntu 6.10 or Kubuntu 6.06. :)
