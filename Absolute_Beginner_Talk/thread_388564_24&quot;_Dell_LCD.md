The problem sure is not the display. Computers don't care what display you have, you just need to use some resolution your display can handle.. What is graphics card do you have and what is the native resolution for your display? Try using the safe graphics mode, or if that doesn't help then install using the Alternate CD and then you can configure Ubuntu to use correct resolution (or install drivers for your graphics card if it's Ati or nVidia one).
