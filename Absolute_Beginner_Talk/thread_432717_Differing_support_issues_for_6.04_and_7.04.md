### Post by Sef on 2007-05-04
> I notice that the 6.04 version has Long Term Support while the latest update is supported to 2008.

This has been puzzling me for the last day or so but could somebody please explain why the two Desktop versions of Ubuntu have differing support dates?

Long Term Support will be available from time to time on Ubuntu.   Every six months there is an updated Ubuntu version released.   If it is not LTS, then the support will last for 18 months on the server and the desktop.   If it is LTS, then support will last for 3 years on the desktop and 5 years on the server.   LTS releases tend to be very stable and problems are expected to be minimal.   NonLTS releases will be stable, but not necessarily as an LTS.
