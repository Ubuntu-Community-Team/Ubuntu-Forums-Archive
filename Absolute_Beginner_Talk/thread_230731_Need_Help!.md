### Post by Stirling on 2006-08-06
Any extensions that work with Firefox 1.5.0.6 will work with Swiftfox 1.5.0.6.  I would say it is likely the Firefox plugins just need to be copied to the swiftfox/plugins folder or alternatively copied to /home/<username>/.mozilla/plugins.  Automatix will install the plugins for you or if you use the [installer]("http://www.getswiftfox.com/installer.htm") it should copy over your plugins to Swiftfox.
