### Post by vikram on 2008-03-27
Kaddressbook, part of Kontact automatically stores backups of your addressbook if you chose the vcard format. Try replacing the corrupt addressbook file with a backup version. Also opening the app in the command line will provide an error message which can hopefully help in trouble shooting.
