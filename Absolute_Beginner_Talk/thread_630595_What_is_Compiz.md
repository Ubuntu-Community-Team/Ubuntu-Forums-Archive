### Post by rich.bradshaw on 2007-12-03
It's a compositing window manager.

It runs on top of gnome.

In 7.10 it's running by default - if you have drivers for your graphics card installed.

Run restricted drivers manager in one of the admin menus.

Install graphics drivers if there are any available.

Restart.

Go to preferences/apperance and the desktop effects tab - are desktop effects enabled? If not, enable them.

For more control over setttings, click [apt://[SIZE=-1]compizconfig-setting-manager[/SIZE]]("apt://compizconfig-setting-manager") to install [SIZE=-1]compizconfig-setting-manager. Then go back to desktop effects and click the advanced button that has now appeared.[/SIZE]
