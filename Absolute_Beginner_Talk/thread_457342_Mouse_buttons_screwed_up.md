### Post by NT4usB on 2007-05-28
Are you familiar with the command line and editing stuff using a text editor and sudo?

Anyway, looks like:
Option		"Protocol"		"ImPS/2"
should be:
Option      "Protocol" "ExplorerPS/2"
and:
Option		"Emulate3Buttons"	"true"
should be false
