### Post by LaRoza on 2007-05-17
Before installing, see how it handles with the live disk, I assuming you are using Feisty, so

   1. onboard sound, video and LAN ---Will work, but research the actual devices to make sure there are no problems
   2. a USB 2.0 memory card reader---Try it on the live disk.
   3. an HP printer--- Will work out of the box, I use an HP Deskjet F380
   4. a Canon LIDE scanner---research this, I never used it

If something doesn't work with the live disk, you can probably get it to work easily.

Generally speaking, most things will work, very few things do not work at all.
