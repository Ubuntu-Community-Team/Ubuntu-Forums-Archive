### Post by mivo on 2008-03-30
It is low maintenance once it works on your computer and you "get" some of the basics/differences to Linux. It works flawlessly and out of the box on most computers, but if it does not, it takes some tweaking. Your best bet is to try the Live CD and see if your hardware is properly found. :) (Just keep in mind that the real thing is snappier, since the Live CD has to load everything from the CD.)

Unless you are a hardcore gamer or require very special interest software, Linux is a wonderful alternative to the commercial and often bloated operating systems. Even if you are a gamer, ther are some Linux games, and there are even top-end games, like Quake Wars, that offer a Linux client.

In short, it is worth trying out! :) Ubuntu is a good starter distro -- you get the best Linux community as a package deal with it!

### Post by JoshuaRL on 2008-03-31
A couple things you want to remember:

If you didn't have the internet connected at install you'll want to make sure the software sources are enabled.  Go to System->Preferences->Software Sources and make sure everything except the CD and source code is enabled.

Then go to System->Administration->Update Manager and download/install everything in there.

After that, go to Applications->Add/Remove Applications and serch for "restricted".  The package you want is called "Ubuntu Restricted Extras".  That should give you MP3 playback.  If you want to record MP3s you'll need the Lame engine, found the same way.

Hope that helps.
