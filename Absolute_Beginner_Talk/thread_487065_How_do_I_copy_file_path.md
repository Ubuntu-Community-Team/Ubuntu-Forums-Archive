### Post by PKraszewski on 2007-07-01
[LIST=1]
[*]Mark the icon of the file you want the path of
[*]Copy it (hit ctrl-c)
[*]Move to wherever you want to paste the path (editor, text field)
[*]Paste it (ctrl+v)
[/LIST]
Both gnome and KDE even allow drag-and-drop of files into pure-text editors (eg. kate or mcedit) or text fields. As long as the destination doesn't accept FILE (OpenOffice for example allows file drop), full FILE PATH will be dropped instead.
