### Post by David_T on 2006-11-12
It's a gzipped tar file (tarball)
From the terminal, you can type 
tar -xvzf z-lock_0.1.tar.gz
Then you cd into the folder it creates, and then type ./z-lock.
There's probably a way to do that with a right-click option in gnome, but I don't run gnome so I can't tell you offhand.
