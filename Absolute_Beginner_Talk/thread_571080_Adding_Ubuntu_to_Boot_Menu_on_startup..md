### Post by Eemerica2s on 2007-10-08
How do I add an option to boot Ubuntu on the startup menu?  I installed Ubuntu onto another Sata HDD, and the only way to get to it is if I disconnect the HDD with Windows on it.  So I just want to know how to get Ubuntu onto the startup menu while both HDDs are connected...

### Post by cameronw on 2007-10-08
Change the BIOS to start on the other drive then a quick google search for "add windows GRUB" and you'll find some howtos on how to add windows to the bootloader. You might want to set Ubuntu not as default otherwise it'll start it instead of Windows on boot up.
