### Post by Rolcol on 2008-03-11
Download the firefox add-on called "user agent switcher".  It can cloak you as using vista or any other version of windows.

[https://addons.mozilla.org/en-US/firefox/addon/59](https://addons.mozilla.org/en-US/firefox/addon/59)

This actually lets me bypass my school's ban on browsers other than IE.  I like firefox portable :D
