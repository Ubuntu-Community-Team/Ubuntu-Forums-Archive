### Post by kaivalagi on 2008-04-20
xmms is no longer supported in the repo's (hardy onwards), audacious is it's equivalent I guess. Although people would argue there are plenty of music players equally as good if not better than this. Exaile, Rhythmbox, Listen etc etc

xmms2 has no UI, it is a sound daemon. You can use front ends against it such as Esperanza though

I use xmms2, I like the music daemon thing. xmms2 is very lightweight and supports network streaming etc etc. Not everyones cup of tea though :)

### Post by fenian on 2008-04-20
> do i need to install any additional plugins for audacious. Plus can I directly add Winamp skins to it?

There are additional plugins for audacious,to install it with all the plugins open a terminal and enter...
> 
sudo apt-get install audacious  audacious-plugins audacious-plugins-extra

Audacious supports winampskins to use them put the skins in the directory....
> 
~/.local/share/audacious/Skins

You can read more about Audacious [here]("http://en.wikipedia.org/wiki/Audacious_Media_Player")
