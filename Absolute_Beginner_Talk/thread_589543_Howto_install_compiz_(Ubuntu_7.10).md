`ccsm' stands for `Compiz Config Settings Manager'. Although Compiz is included in Gutsy, ccsm is not. Install it using Synaptic as rolnics explains and use it from the System > Preferences menu. ccsm helps you control how windows are rendered on your system. Once you are done with this, also try the Emerald Themes Manager and Screenlets. :-)
