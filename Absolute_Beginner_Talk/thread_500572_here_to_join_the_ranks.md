### Post by BobCFC on 2007-07-14
Welcome! 

Most libraries such as python and sdl etc are called dependencies - when you install a program using synaptic or apt-get that depends on them it will install them automatically as well.

Also hardware openGL really needs proper ndivia or ati drivers. Ndivia web site is really great for linux but ATI is patchy.

When you click on a video that uses codecs it will usually ask you if it want to look for them itself these days, much easier.


Good luck with C++! ):P  Too many java kids we are a dying bread.  For a heavyweight IDE like visual studio try Eclipse.  It's java centric and a bit slow but it's got loads of plugins and a big community.  I think they call c/c++ plugins CBT (c-based tools)

For a medium IDE try Anjuta.  Leaner and smaller it also has native GTK interface building forms etc.

Lightweight superfast IDE try Geany.  Just a text editor with syntax highlighting/code hiding and tabs etc.  Also compiler output at bottom window.

For easy source code control don't worry about CVS or SVN bulky crap use Bazaar or BZR from makers of Ubuntu... which is distributed like GIT but friendlier and works on windows too.

Hi, I don't know about games, unless they are DOS games, which  work fine in Linux, (DOSbox).

If you want to be programming in Ubuntu run this command:


### Post by Hendrixski on 2007-07-14
WELCOME TO THE COMMUNITY!
We're all here to help new Ubuntu users because we love ubuntu, and we want to share our joy with others. And you'll be one of us in no time, coming here to help out others, and contributing code, documentation, artwork, and bug reports.

If you prefer chatrooms then you should totally check out the channel #ubuntu on IRC.  Installing IRC is easy and fun.  Just go to applications-->Add/Remove and pick XChat from the list, click OK and you're done.

You can meet other Ubuntu users in your area too!  Just check out the Ubuntu LoCo teams for your city or state.

There is plenty of GREAT documentation written about Ubuntu, just ask google and you'll find it.  A lot of it is written by the same people who answer questions on these forums.  If dry manuals aren't your thing then relax and watch some videos.  [http://screencasts.ubuntu.com/](http://screencasts.ubuntu.com/) is a great website to get started, how to install, how to customize, how to set up your mail. etc.  Also check out [www.ubuntuvideo.com](www.ubuntuvideo.com) and [www.ubuntuclips.org](www.ubuntuclips.org).

I hope I'll see you around some more.  :-)
