### Post by tashmooclam on 2007-10-23
I think either using the synaptic package manager (system>administration>synaptic) or Add/Remove (Applications>Add/Remove) you can get it off.
Maybe in Synaptic package manager click the Search button, and write Thunderbird in the field. Then, Thunderbird will show up in the window. 
You can use the Package menu after you select the thing you want to add or remove.
I am not clear about the difference between "complete removal" and "removal".
