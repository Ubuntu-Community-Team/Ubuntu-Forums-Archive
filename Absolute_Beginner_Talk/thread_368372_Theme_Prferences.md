Go to System -> Preferences -> Theme to change GTK2, Metacity and icon themes.

More themes can be found at [Gnome Art]("http://art.gnome.org") and [gnome-look]("http://www.gnome-look.org").

If you want to create your own themes, have a look at [these tutorials]("http://live.gnome.org/GnomeArt/Tutorials").
