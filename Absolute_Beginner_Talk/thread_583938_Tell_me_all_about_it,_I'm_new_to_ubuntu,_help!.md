### Post by mike555 on 2007-10-20
When you get the disk start your computer from the CD and try it out before installing , if you decide to install you will need to partition your harddrive (read up on how) and then pick "Manual" install so you won't wipe out your Windows install , set the partition to use as Ubuntu as " / " and also set a smaller partition (about 1GB ) as " swap" ......
   == another words DON"T just click through default install or you will lose your Windows ==
