### Post by skymt on 2006-10-13
1. To install the m4a plugin for Beep Media Player, open Synaptic (System > Administration > Synaptic), search for bmp-mp4, and install it. For XMMS it's xmms-mp4.

2. This is a common problem that will be solved when Adobe finally releases Flash 9 for Linux. In the mean time, try [this workaround](http://www.ubuntuforums.org/showthread.php?t=255422&highlight=flash+sound).
