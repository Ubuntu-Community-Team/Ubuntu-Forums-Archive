### Post by Nekiruhs on 2007-07-03
When I use a root permission app, like adept, it has the default theme, not my theme. In Gnome I solved this with a few symlinks. How do I fix this in KDE?

---

### Post by Happy_Man on 2007-07-03
Run ```
sudo kcontrol
``` and edit the themes as you wish in there, and it will apply to all root applications.
