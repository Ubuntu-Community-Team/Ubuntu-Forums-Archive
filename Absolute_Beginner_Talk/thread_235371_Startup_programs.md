How do you make programs load on startup? For example: I would like Firestarter to load as soon as I log into Ubuntu.

---

### Post by 23meg on 2006-08-13
Firestarter already runs at startup with the set of rules you last specified in its GUI so you don't need to set up anything. To add other programs to GNOME startup, use System / Preferences / Sessions / Startup Programs .

### Post by bhoughton on 2006-08-13
If Firestarter doesn't start for you, in the sessions/startup add "gksudo firestarter".  You will be prompted for your password upon logging in.
