### Post by Cypher on 2008-02-15
The [VLC Wiki]("http://wiki.videolan.org/VLC_command-line_help") suggets
```

vlc --play-and-exit <file>

```
FYI, in the future if you have a question about a specific application, check out the man page for that command or online for the documentation, that's always going to be the authority..searching is good..:)
