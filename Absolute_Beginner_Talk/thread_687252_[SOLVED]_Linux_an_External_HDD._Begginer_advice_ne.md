### Post by hyper_ch on 2008-02-04
> **SuperT said:**
> 
1. Will i need to partition it and will this involve formating my EXTHD and loosing all my data? Also what file system will i need to use.
You need to have unpartitioned space on it which you can then use for linux. Basically defrag your drive a couple of time and then resize it so that you create new unpartitioned space.

> **SuperT said:**
> 
2. I know ubuntu is the most popular but am i better off using a lighter version of linux?
You are best of using a version that supports all your hardware out of the box...

> **SuperT said:**
> 
3. Do i download the iso file then burn it and run it on my pc with my External Hard drive pluged in and then specify it as its install destination?
You have to burn the .iso as image file. See [http://www.psychocats.net](http://www.psychocats.net) on how to do that. Then you need to set your bios to boot from the cd-rom as first boot device. After that you should be able to run the cd and install it on the external harddrive.

> **SuperT said:**
> 
4. How big will my partition need to be, will 20gig be ample?
Just for testing 20 GB are more then required.... I'd say for purely testing 5 GB are the minimum.

> **SuperT said:**
> 
5. Is there anything more i need to know?
