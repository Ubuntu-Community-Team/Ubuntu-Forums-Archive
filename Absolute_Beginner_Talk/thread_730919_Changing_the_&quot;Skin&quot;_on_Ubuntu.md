### Post by Therion on 2008-03-21
Navigate to: System/Preferences/Appearance.  This opens the Appearance Manager.  Keep the Appearance Manager open on your taskbar and open your broswer and go to [Gnome-Look](http://gnome-look.org/index.php?xsortmode=high&page=0&xcontentmode=100) and find a theme you like.  Now, click on the theme name, find the "download" button/link and click and draaaaag that download button right ONTO the button for the Appearance Manager.  You should get a message saying the Theme was installed.  Now just use the Appearance Manager to select the newly installed theme and it will apply immediately.  

Lather. Rinse.  Repeat.

### Post by Therion on 2008-03-25
> **nrcooled said:**
> I am pulling the .tar.gz file and tried dropping it in Theme Manager and when that didn't work I tried emerald.  Still nothing.  I can open the file and pull individual things and install them but I would like to have the entire theme to work.
Well the problem is kind of two-fold... 1. You have to make sure that what you're pulling down onto the Theme Manager is a compatible theme (meaning it has to be a GTK theme) and 2. Not everyone who puts a theme on Gnome-look, or whatever site, packages the theme in such a way that it allows you to use the Drag-and-Drop method -- it's rare, but it happens.  If you're sure that the theme you are trying to install is GTK and you're getting the error about the file format being incompatible, best thing to do would be to post about it specifically and ask for help.  I've seen some pretty oddball stuff, like a compressed folder inside a compressed folder... INSIDE A COMPRESSED FOLDER.  <shakes head>
