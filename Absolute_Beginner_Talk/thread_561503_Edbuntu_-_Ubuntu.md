Edubuntu has some extra education software. They both use GNOME.

If using Ubuntu you can see the differences using this command:

```
aptitude -s install edubuntu-desktop
```

---

### Post by Linux_Man on 2007-09-27
Ubuntu is mostly for thick clients (Installing the OS on a hard disk and accessing it from there using only your hardware), and has more general-use software and a different default theme. Edbuntu is more for thin clients (Installing the OS on a server or hard disk and loading it from there using the server's hardware) And has more school-appropriate software
