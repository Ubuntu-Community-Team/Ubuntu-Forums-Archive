### Post by Rever75 on 2007-06-27
So you have downloaded the gdesklet goodweather. Now just start up gdesklets and in the Gdesklet Shell go to File => Install Package. Browser to were you have the tar.gz and done. You can also choose File => Install Remote Package and place this url .... [http://www.gdesklets.org/files/desklets/GoodWeather/GoodWeather-0.3.tar.gz](http://www.gdesklets.org/files/desklets/GoodWeather/GoodWeather-0.3.tar.gz)
