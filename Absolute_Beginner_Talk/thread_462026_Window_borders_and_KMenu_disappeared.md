### Post by Clay_Banger on 2007-06-04
are you using beryl?

if not, it could be to do with kwin, the window manager for kde.

if it ever happens again, you maybe able to start kwin again using:
Alt+F2
typing 'kwin' (without the quotes) into the box
clicking run

if it keeps on happening, something is causing kwin to crash, and you will have a more serious problem on your hands.
