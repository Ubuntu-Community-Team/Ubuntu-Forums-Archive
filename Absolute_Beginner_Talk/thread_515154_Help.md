### Post by weresheep on 2007-08-01
Hello,

The way Ubuntu works is that all the software (and all the drivers) that is available is stored on Ubuntu's online repositories. There is a built in graphical program that you can use to browse the repositories for new software and click to have it automatically installed.

However, during installation Ubuntu will (most of the time) detect what hardware you are using and install the correct drivers without you having to do anything at all.

There are some exceptions to that rule. But before things get too complicated I suggest trying the Ubuntu install CD. The way the Ubuntu installer works is to boot up the computer into Ubuntu itself, off of the CD. This way you get to see in advance how well Ubuntu will work on your computer. If your computer has a lot of unusual parts that Ubuntu doesn't recognise then maybe it will be really difficult to get it working properly. However, it may be Ubuntu detects everything straight away and you wont have any worries.

The only way to know is to boot up the CD and find out :)

Hope that helps.

-weresheep
