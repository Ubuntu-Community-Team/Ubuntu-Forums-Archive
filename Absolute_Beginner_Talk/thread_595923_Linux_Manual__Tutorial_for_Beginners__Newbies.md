### Post by erginemr on 2007-10-29
Linux Manual / Tutorial for Beginners / Newbies:

If you are new to the Linux world and are after the basic Linux commands, not only after learning Ubuntu only, then for a hands-on quick start guide, you may search "linux for newbies" through Google.

Apart from that, I would suggest reading the following two sites, whence I also benefited a lot from:

1. UNIX Tutorial for Beginners:
[http://www.ee.surrey.ac.uk/Teaching/Unix/](http://www.ee.surrey.ac.uk/Teaching/Unix/)

2. Tux Files:
[http://www.tuxfiles.org/](http://www.tuxfiles.org/)

Although both sites start teaching from ground zero, the first site has a more academic approach, while the second is pretty downright hilarious. The articles are really easy to follow (i.e. not for geeks) and are full with very funny remarks, and I enjoyed every bit of it while reading.

I would suggest the newcomers not to be afraid from the console (terminal) commands and start learning the widely used commands right away. They are much more powerful than MS-DOS commands, and are also very fun to learn.

Console commands are incredibly powerful in Linux, indeed. Although graphical environments have evolved a lot over the last years, I still believe the one who masters the console also governs the whole Linux system.

Happy Linux'ing!..
