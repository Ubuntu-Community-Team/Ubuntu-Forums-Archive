### Post by Hospadar on 2007-10-12
I'm sure if you google em you can find some really good basic guides.  Also wikipedia is a great place to start.

I can define all the terms you've listed here, however to save you a little time.

Distro: (or Distrobution)
The basic backbone of any version of linux is the linux kernel.  It's written by this guy Linus Torvalds (among others) and it does all of the basic low-level functions of your machine.  All linux distro's use this kernel, but they may pile different applications on top of it, different gui's, different starting packages, etc, etc.  So the basic system parts will all be the same, and any software that runs on one distro can run on any other.  so beyond that it's just preference as to how you like the way different distro's are set up.

KDE and Gnome are both desktop managers.  when you are in the desktop, all the panels and start menu and little widgets and buttons and everything there are all thanks to the desktop manager.  KDE and Gnome are really very similar in terms of the user, and any application that runs on one will run fine on the other.  It's just a preference issue of which one you like better, which for most people boils down to the looks.

GNU is actually a different operating system, but is the precurser to linux.  Many (almost all in fact) of the applications you use on linux are based on GNU, but since the linux kernel is much more complete and active, generally people just run GNU programs on linux.

Qt and it's gnome cousin GTK are the libraries used to draw (almost) all of the graphical applications.  when you program an application, instead of having to manually draw out buttons (or whatever) over and over again,  you can use one of these libraries and just say you want a button with this label that does this function when I click it.  Qt is the standard for KDE and GTK is generally used by gnome.  these don't in any way conflict with each other however so you can use both at the same time for different applications built on them.

Hope that helps you get started!
