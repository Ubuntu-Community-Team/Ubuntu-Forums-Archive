### Post by Phurious on 2007-11-02
I can say without hesitation, SABnzbd!  I discovered this program a couple of months ago, and if you pull a lot of binaries, you NEED this program.  It will auto-import your NZB files from a directory you specify, download the binaries, PAR check and repair IF necessary, then extract automatically.  Set it and forget it.  It does not natively support SSL, but used in conjunction with Stunnel4, it's probably the best solution I have seen on either Windows or Linux.  BTW,  JMHO!
