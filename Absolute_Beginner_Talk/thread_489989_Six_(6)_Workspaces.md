### Post by njknight on 2007-07-02
Go into Beryl Manager
Select Beryl Settings manager
Select General options (the first part)
Scroll down to where it says "horizontal virtual size"
Change value from 6 to 4 (or whatever you want)
exit the Beryl settings manager
