### Post by pja on 2005-11-17
[QUOTE=simple1]Dumb question: boot one or the other?  Or boot one and the other = nice on dual monitors?[/QUOTE]
I think you will find "dual boot" means that your PC can run two or more operating systems (say Ubuntu and Micro$oft Windows) but, unfortunately, not at the same time (geeks: please don't tell Simple1 and myself about elaborate switching software that most users don't have or understand).

In my case I have two disk drives (C: and D: in windows terms) and run WinXP Home on C: and Ubuntu 5.10 on D:.  When the PC starts up it runs a "menu" program called Grub which allows me to select whether Windows of Ubuntu will be booted.

You can dual boot using a single hard drive but I feel its safer with two.

If you are very new to Linux, try using a "Live CD" version of Linux which lets Linux boot from a CD and has no impact (absolutely no impact) on your hard drive.  You can get the feel of Linux without making the hardware commitment.

Hope that helps,
