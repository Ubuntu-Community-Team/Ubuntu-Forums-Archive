### Post by TrendyDark on 2006-02-14
Installing Windows on a computer running only Ubuntu will remove GRUB, thus making it impossible to get into your Ubuntu partition. It's not at all hard to fix this problem though, here's the link to the wiki entry about it: [https://wiki.ubuntu.com/RecoveringUbuntuAfterInstallingWindows?highlight=%28Windows%29](https://wiki.ubuntu.com/RecoveringUbuntuAfterInstallingWindows?highlight=%28Windows%29)

Other than that, just pop in the windows CD and let it go.
