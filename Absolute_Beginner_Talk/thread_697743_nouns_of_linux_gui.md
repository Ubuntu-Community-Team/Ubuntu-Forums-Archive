### Post by rbprogrammer on 2008-02-15
**Desktop Environment**: the program that controls all the gui operations.  eg GNOME vs KDE, GNOME has the system menus and preferences at the top, while KDE has the K-Menu that is at the bottom..  KDE: [http://www.howtogeek.com/wp-content/uploads/2006/11/WindowsLiveWriter/InstallKDEKubuntuonUbuntu_C102/kdedesktop.png](http://www.howtogeek.com/wp-content/uploads/2006/11/WindowsLiveWriter/InstallKDEKubuntuonUbuntu_C102/kdedesktop.png) GNOME: [http://www.osnews.com/img/8286/ubuntu1.png](http://www.osnews.com/img/8286/ubuntu1.png)

**window manager**: the program that manages the windows :) ...

**themes, style, window decorations**: all more or less the same, but they allow you to change the look and feel of your guis.  i use GNOME, so i use the site: [http://www.gnome-look.org/](http://www.gnome-look.org/) for different themes and styles,... i'm sure KDE has a site liek the same

hope i was general but basic enough to clear up your confusion.. :lolflag:
