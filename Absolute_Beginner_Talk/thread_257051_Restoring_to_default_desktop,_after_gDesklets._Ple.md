### Post by CatKiller on 2006-09-14
I'm sure that there is a default configuration for the panels, since when you create a new user it makes a default desktop for them. However, I don't know where that configuration would be, nor how to use it to restore a default desktop.

Instead, you could right-click on the top panel and select "New Panel", move the new panel to the bottom with a middle-click and drag, and then add what you wanted to it by right-clicking on it and selecting "Add to Panel".
