### Post by kebes on 2007-02-06
The only thing Linux can't do that Windows can is run Windows programs and games! (And, in fact, things like Wine let you run many, but not all, Windows programs and games.)

For every application I've ever used in Windows, there is a highly equivalent Linux application. Sometimes the Linux one is better, and sometimes it is worse. (But there are often many to choose from, until you find what you want!)

The things you mentioned (burn DVDs, archiving, etc.) can all be done in Ubuntu.

### Post by BarfBag on 2007-02-06
In terms of productivity, you can get a lot more done in Ubuntu.  Things are easy to configure, the system is low maintenance, the GNOME desktop environment does an excellent job of staying out of the way, etc.

As kebes mentioned, the only thing Ubuntu can't do is run Windows games and programs natively.  [Wine]("http://www.winehq.com/") can run a decent amount of programs (most at almost native speeds) and even some games (World of Warcraft, Call of Duty, etc.).  [Crossover]("http://www.codeweavers.com") is another option for programs.  If you're a hardcore gamer, [Cedega]("http://www.transgaming.com/") can run most of the popular games with decent performance.

The things you mentioned work better in Ubuntu.  K3B burns CD's and DVD's (and is the best burning program I've ever used), and the OS does a great job of archiving out of the box.
