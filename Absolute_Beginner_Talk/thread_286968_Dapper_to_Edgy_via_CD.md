### Post by stalker145 on 2006-10-28
I'm going to take a stab here with this one...

Right now, I'm actually upgrading from Dapper to Edgy via the interweb. If you simply want to do an upgrade, open the terminal and type
> gksu "update-manager -c"
This will bring up your Update Manager which will give you the option of upgrade.

You can also, if I'm thinking correctly, insert the CD-ROM and it will ask you if you wish to start the package manager. You can upgrade from there as well.

Since I haven't done it, and to try to answer your question, you should be able to install Edgy directly from CD the same as any previous distribution. Have fun with it... I'm hoping to enjoy the upgrade.:)
