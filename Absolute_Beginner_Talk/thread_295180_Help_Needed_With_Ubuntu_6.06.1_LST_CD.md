### Post by Sef on 2006-11-07
> Ubuntu just sent me the 6.06 CD in the mail. When I run it in in Windows it offers only demo packages of Firefox etc. The instructions with the CD say in part "to install Ubuntu, run the Live CD and double click on the install ikon."

You need to set your BIOS to have it boot from your CD-ROM/RW first.  Then it will boot up from the CD, and there will be an install icon on it.  To boot from the CD-ROM/RW first, push either (usually) F2, esc, F12, delete.  Your computer will tell you what to push to get into the BIOS upon boot up.  

Note: Before installing, **back up** all your data first.  Usually the install goes fine, but on occassion, it has not.

Also read, Pychocat's [Installing Ubuntu]("http://www.psychocats.net/ubuntu/installing").
