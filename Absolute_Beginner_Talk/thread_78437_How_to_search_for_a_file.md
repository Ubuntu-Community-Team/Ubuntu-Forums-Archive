### Post by ChrisTheGeek on 2005-10-18
The search isn't integrated into Nautilus, but it is very powerful - much more so than the Windows search.  You can leave it searching for just filenames or choose to add a variety of extra filters such as file size, date modified, contains the text etc.

You can also set up a shortcut key to bring up the seach dialog whenever you want it (there may be a shortcut key already, I forget :))

Beagle is also extremely good and worth installing.  Do bare in mind that it is still beta software at the moment - although it's not crashed at all on me.  Here is the main site to find out about beagle: [http://www.gnome.org/projects/beagle](http://www.gnome.org/projects/beagle)

Happy searching :)
