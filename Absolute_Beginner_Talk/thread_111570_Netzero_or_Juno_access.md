NetZero/Juno does work on Ubuntu, but you have to use apt-get or Synaptic Package Manager to download and install the Java 2 Runtime Environment (j2re). The NetZero/Juno dialer depends on j2re. Without j2re, the NetZero/Juno dialer will NOT work.
