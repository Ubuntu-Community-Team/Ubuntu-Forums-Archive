### Post by i8bugs on 2007-02-04
I solved this problem in a much easier way.  Goto Synaptic and uninstall the flashplayer non-free.  Then goto the nbc website  website and try the play the streaming video.  You will get the "missing plugin"  button in firefox.  Hit the button, hit install plugin, and it will install.  This is the FIRST time I have seen that button work in firefox.   If is doesn't work for you, you must have a different setup than me, but I was quite surprised (pleasantly) to see that button work for the first time using Linux firefox.
