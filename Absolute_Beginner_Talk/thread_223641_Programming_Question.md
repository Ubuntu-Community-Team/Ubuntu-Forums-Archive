### Post by Cath0de on 2006-07-26
C, C++ (of course).

Perl or Python are useful scripting languages and Apache is useful if you want to set up a server.

But if you want to program on a UNIX based system, learn C and then C++. Basically everything is programmed in one or the other.

### Post by IYY on 2006-07-26
It's always a good idea to learn the popular scripting languages: bash, perl, python... These aren't really "programming" languages, but are very easy to learn (a day or two) and can be quite useful for small things (note that this doesn't really apply to python, which is actually a powerful programming language).

Then I'd suggest learning C and C++.
