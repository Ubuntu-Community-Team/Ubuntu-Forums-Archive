### Post by Paulmd on 2008-01-16
> **fedex1993 said:**
> i am using a mix of flac mp3 and ogg also

Now we're getting somewhere.

In a nutshell, you need to install different libraries for different filetypes.

```
sudo apt-get install libmad libogg libvorbis
```

This will install support for mp3 and ogg files. If this gives you any errors, post them, and we can work those out.

[http://mpd.wikia.com/wiki/Dependencies#Audio_Decoding_.2F_Encoding](http://mpd.wikia.com/wiki/Dependencies#Audio_Decoding_.2F_Encoding)
