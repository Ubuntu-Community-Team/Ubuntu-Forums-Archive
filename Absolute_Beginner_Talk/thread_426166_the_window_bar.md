### Post by Metacarpal on 2007-04-29
If you're using Gnome (ubuntu-desktop), either open a terminal (Applications menu > Accessories > Terminal) or hit [Alt]+[F2], then type in:

```
metacity
```

That should restart the metacity window manager.  Then, when you log out and back in, your windows should be back to normal.

Ooh - if you added "beryl-manager" to your startup programs (under System menu > Preferences), removing it is a good idea if Beryl is no longer installed.
