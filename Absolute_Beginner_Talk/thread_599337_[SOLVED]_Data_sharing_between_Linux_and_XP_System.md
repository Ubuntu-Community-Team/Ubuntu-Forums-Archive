### Post by AndyCooll on 2007-11-01
Yes, depending on how you save them. So a JPEG (for instance) is saved in JPEG format and is a JPEG format whether saved on a ext3 hard-drive or an NTFS one. What format the hard-drive uses for storage is actually irrelevent.
