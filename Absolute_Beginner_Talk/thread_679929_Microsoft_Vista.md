### Post by mortsahl on 2008-01-27
Usually during intitial boot you'll get a black screen with some quick scrolling on it .. one of the things displayed tells you to hit DEL (or some other key, depending on the vendor) to get into the BIOS.  Once you get there you should be able to find an option that will allow you set your CD rom as the first boot device.
