I have received some negative comments on this forum for even asking questions about Crossover (it is not open source), but I was desperate to take leave of Windows and being able to run Adobe PhotoShop and my trusty old Adobe PageMill made the Big Difference. (I have tons of existing Web site built on PageMill and it works fine for me so I don't want to "upgrade.")

I downloaded and installed Crossover 6.2.0 and it accepted PhotoShop and PageMill as if I were installing them on a Windows machine.  So far I have not encountered a single glitch in their operation. To me this is a far better option than making my computer into a dual-boot machine and still having to deal with Windows just so I can still run PhotoShop. And trust me, regardless of what anyone says, GIMP is NOT a substitute for PhotoShop, especially for anyone who has been using it professionally for years.

So as much as I support Linux, Ubuntu and the whole open source community, I wanted to make the switch to Linux and now I have done it.  If and when an open source app evolves to the point where it can satisfactorily replace Crossover, I'll go for it in a heartbeat. In the meanwhile, this works.
