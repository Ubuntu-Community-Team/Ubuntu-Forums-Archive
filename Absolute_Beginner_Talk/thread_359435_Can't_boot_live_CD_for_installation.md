### Post by seshomaru samma on 2007-02-12
I think a LiveCd cannot run on your computer
Download an alternate version of Xubuntu (a command line installer CD, found[ here]("http://neacm.fe.up.pt/pub/xubuntu/releases/6.06.1/release.1/") )
If it still doesn't work it could be that Xubuntu is too heavy to run on your computer.

You then have a few options :
try a different distro like :
[puppy linux]("http://www.puppylinux.org/user/viewpage.php?page_id=1") - super fast ,easy to set up but with very few programmes , if you only want to do internet , mail ,chatting ,listen to MP3 and edit some pictures Puppy is ideal
[DSL]("http://www.damnsmalllinux.org/")- super fast, not as easy to set , more programmes than Puppy 
[Fluxbuntu ]("http://fluxbuntu.org/")- ubuntu with Fluxbox graphic interface , this will be very fast and will enjoy the variety of programmes the Ubuntu distro has to offer and a good support at the forum but it will require some tinkering and a lot of command line work.

since both puppy and DSL are live CD (very small in size) you can give them a try first , in case the xubuntu alternate Cd doesn't work.
