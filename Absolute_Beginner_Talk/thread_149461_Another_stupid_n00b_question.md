### Post by xhie on 2006-03-23
to run xmms open a terminal and type:

*xmms*

and there she goes...

In generial if you dont know where something is open a terminal and type:

*whereis NameOfSomething*

it will give you a nice little list of all the places it can be found. 
If something is in /bin or in /home/yourname/bin you can run it by just typing in the name of the executable. By default that it, messing with the path can change that.
