### Post by aysiu on 2005-11-27
I'll give the pros of each, as far as I know:

Ubuntu:
Only one CD
Simplicity (one application per task)
Amazing community support
Debian (apt-based) package management for installing software

Fedora:
Anaconda installer
Choice of Gnome or KDE during installation
Sometimes, depending on the obscurity of the program, RPMs can be easier to find than DEBs

Have you tried the [Linux Distribution Chooser](http://www.zegeniestudios.net/ldc/)?

### Post by ssam on 2005-11-28
they both have similar goals

fedora has some exciting stuff going on with new linux technologies in security for example.

ubuntu has a faster release cycle, but is stricter with allowing anything into the stable release.

they tend to leap frog each other in terms new software.

anaconda is friendlier in some ways (but i asks far more questions).

synaptic (plus the new add application and the update notifier) beat yum (unless they have go a good gui yum interface sorted). command line package managers are only useful if you know the name of what you want, and you can spell.

no fedora live cd. (red had based live cds seem very rare. is this just because everything is derived from knopix? are debain based systems easier to make into live cds? does anyone know?)

i strongly recomend that you try both.
