### Post by Rocket2DMn on 2008-01-10
Overall you will get better performance out of the faster drive, so if you plan on using Ubuntu on it, you should install to that drive.  You will not have to physically move any drives around inside your computer or deal with cords.  When you install Ubuntu, tell it to install on that drive (like hdb or sdb as opposed to hda or sda).
If you want the drive space, there is no problem keeping the old drive around so long as you trust it with data (keep good backups!).
