### Post by bryan4134 on 2006-09-23
I have WINE version 0.95 working more or less.  When I allow the system to update WINE nothing works, so....  I want to keep my earlier version.  How do I permanently stop the UPDATE feature from constantly trying to update the older version.  I know that I can manually search through pending updates and uncheck the WINE update but that is a real pain - is there anyway of forcing Ubuntu to accept that I am very happy with version 0.95.  Appreciate any suggestions.

Thanks, Bryan

---

### Post by benuski on 2006-09-23
If you highlight the wine package, then go to the "Package" menu up at the top, there should be an option called "Lock Version." That will prevent Synaptic from automatically marking it for an upgrade anymore.
