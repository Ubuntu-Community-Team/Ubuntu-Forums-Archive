### Post by michux on 2006-10-12
If you used automatix, no need to use EasyUbuntu additionally. They are exchangeable. The multimedia players that provide the longest list of codecs and supported formats are Xine and MPlayer. Xine is a bit better for DVD since it supports the DVD menus. For GTK environment you can install Totem with xine support (totem-xine package) or just any other xine frontend like gxine, etc.

Considering the iRiver support, there is a webpage listing all the apps you can use for sync: [http://ifp-driver.sourceforge.net/](http://ifp-driver.sourceforge.net/)
