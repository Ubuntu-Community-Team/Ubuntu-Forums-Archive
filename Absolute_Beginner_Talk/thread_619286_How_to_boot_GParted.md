### Post by Dr Small on 2007-11-21
Set your BIOS up to boot CD ROM before hard drive.

### Post by Dr Small on 2007-11-21
> **RobotoWorks said:**
> How do I set my BIOS?
When the system boots, at the POST screen, it should tell you what key to press to enter the BIOS (or Setup). If it doesn't, try F1, F2, Del, Esc and such.
