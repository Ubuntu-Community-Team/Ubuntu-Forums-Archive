### Post by cymen on 2007-05-24
Conky doesn't create a .conkyrc, you create one yourself in your home directory. [Here's some example files to work with]("http://conky.sourceforge.net/screenshots.html") or you can use a sample file which should be bundled in with the install - I found mine at /usr/share/doc/conky/examples/conkyrc.sample.gz
