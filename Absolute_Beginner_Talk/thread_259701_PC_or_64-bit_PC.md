Its a 64 bit processor. I wrote this earlier, maybe it will fit here.

You have a choice. You can run either the 32bit or the 64bit. They both have there stengths and weaknesses.
64bit
Advantages - It will be faster in a lot of cases, in some it wil have 30% gains, some it will be barly noticable. But you will have the ability to run 64bit applications.
Disadvantages - Some packages will be harder to install. You will have to run some auto setup scripts or follow a few howto's to get some proprietary codec's working. Most are cut and paste, and it will take you an hour or 2 more in setup. If you are doing development of 32bit software you will need to setup a chroot, but if you are doing development you should want to set one up regardeless of the version so you dont ruin your install.
32bit
Advatages - Everything will be click install, you can run a few more packages, 2% more.
Disadvantages - Everything will run as slow as a 32bit computer, you will have wasted money on hardware because you are not using it to its full potential. You may cioncider buying a high end 32bit processor instead of a low end 64bit one if you chose to go the 32bit rout. But that means you will be using yesterdays technology.

Since you are building it yourself , it must mean you are interested in learning the most about your computer. You might be more interested in building it than just having it up and running fast. If thats the case go with the 64bit version.
