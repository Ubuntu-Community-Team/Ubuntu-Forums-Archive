Its as easy as typing
```

```
But its going to use a ton of resources. You might want to concider the lighter xfce desktop from Xubuntu in wich case the commands would be
```
sudo apitude install xubuntu-desktop
```
There are even lighter desktops avilable like [icewm]("https://help.ubuntu.com/community/IceWM") and [fluxbox]("https://help.ubuntu.com/community/Fluxbox?highlight=%28fluxbox%29") in the repositories.
I have icewm on my server.
