### Post by ugm6hr on 2007-08-11
> **jingo811 said:**
> By the way is there some aptitude update procedure I can do with this install?  

No.  It is a standalone .deb, it is not from a repo, so there is no way to upgrade it without knowing where the updated .deb is.

Medibuntu doesn't appear to keep realplayer in its repos any more either - so that's not an option...

This is not too hard to fo:
