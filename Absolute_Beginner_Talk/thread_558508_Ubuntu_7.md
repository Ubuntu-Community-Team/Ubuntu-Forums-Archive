### Post by Paulmd on 2007-09-24
> **eriza said:**
> i hope somebody can help me...im using ubuntu 7 with mozilla firefox. When i started to open it, the mozilla will suddenly closed. Sometimes i cant do my work...I hope somebody can help me to solve this problem...


Is it just Firefox that crashes, or so other programs crash?

If it's just Firefox, you can try an alternate browser, such as Opera. ([www.opera.com](www.opera.com))
Also, If it's just firefox, you can see if uninstalling plugins, such as Flash help any.  To see a list of installed plugins, type ```
 about:plugins
``` in the address bar. (feel free to paste it into your reply)



If a lot of programs crash, too, it is possible you have a hardware problem.  The top 3 on the list are bad hard disk, bad video card, and bad capacitors. (there's about a dozen or so more :) )
