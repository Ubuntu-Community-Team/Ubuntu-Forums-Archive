### Post by 3rdalbum on 2007-05-23
> **KnowNothing said:**
> I know one can add more screen resolutions but it is disappointing that there aren't more options to begin with.  It isn't obvious that it is possible to add more and the process itself is daunting.

Why aren't there more to begin with?  Why isn't it easier to add new ones? 

True and true, in a way. Other distributions have utilities which can add new resolutions to the Xorg.conf. However, the amount of resolutions listed depends on what sort of graphics card and monitor you have.

> Why doesn't the OS detect the native one for the connected monitor and default to it or at least automatically make it one of the options?

It does. But nothing is perfect.

Recently, a customer of mine bought a widescreen TV to use as a computer monitor. Windows just stretched the 4:3 image across the entire screen. The Ubuntu Desktop CD recognised it as a widescreen monitor and detected the LCD screen's native resolution. There were also a number of other resolutions listed in Gnome.

YMMV depending on what graphics card you have.
