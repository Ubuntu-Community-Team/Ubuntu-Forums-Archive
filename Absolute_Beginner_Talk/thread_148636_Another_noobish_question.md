### Post by pbaehr on 2006-03-22
The Install CD downloads as a .iso file. Your computer is probably showing it as a zip file because it has a program on it that can decompress it. You don't want to do that, though. You need to open up a program to burn it on CD and select the "Burn a CD Image" function. Then just select the file you downloaded and let it do it's thing.

Don't make a "Data CD" and just copy the file to it. It needs to be burned specially.

Good luck!

### Post by Princey on 2006-03-22
[QUOTE=Ob1]Ok, Windows just showed it as a zip file.[/QUOTE]

Do you have a zipping utility like WinRaR or WinZip or something close to that? Normally, when you install these applications they default to be the file handler for *.iso files too and will show *.iso files using their respective icons. One way to be certain and I'm assuming here you're using Windows, is to 
1. open Windows Explorer
2. go to the folder where the file is located
3. click on Tools, Folder Options. 
4. Click on the view tab with the window menu that opens
5. Scroll down to where it says "hide extentions of known file types"
6. Uncheck that box and click apply

Look at the last three letters of the file now. Does it still say *.zip? or does it say *.iso?

Remember to repeat the steps but reverse step #6 just in case you have to rename files and actually forget to add the extention at the end.
