### Post by speed145a on 2007-12-02
an alternate installation cd is extremely similar to the graphical install process.  if you are familiar with the graphical method you shouldn't have trouble with the alternate.

the only place you may run into trouble is the partitioning.  if you usually select the automatic usage of free space option then you will be fine because that is an option on the alternate install.  I would give the alternate a go and if you feel too intimidated by the partitioning you could try throwing in a different version live cd (maybe feisty or openSUSE)

hope that helps you
