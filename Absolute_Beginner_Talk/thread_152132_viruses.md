### Post by Stealth on 2006-03-29
As of right now, Linux is virus/spy/ad/malware-free.

### Post by aysiu on 2006-03-29
[QUOTE=meborc]
but since you need a root access to do any damage and that access those viruses don't have... well... lets just say that linux is virusresistant...[/QUOTE] I have a couple of objections to this.

While I don't relish the idea of some program taking over my computer and changing system-wide settings, I'm also not too fond of it erasing my user settings and files either (which do not require root access).

Also, Windows does not need to be run in administrator mode all the time. True, it defaults to administrator mode. True, a lot of programs designed for Windows practically require you to run in administrator mode. But you can be relatively safe in Windows if you have good habits.

The truth is that you can generally rest easy using Linux, but you should always be aware that every system connected to the internet is vulnerable to attack. Just because something is *unlikely* to occur doesn't mean it's *impossible* to happen.

Whether it's the default configuration that's more secure or the low market share (smaller--less desirable--target)... or some combination. You're relatively safe for now.

Get strong passwords.
Do not run as root.
Back up your files regularly.
