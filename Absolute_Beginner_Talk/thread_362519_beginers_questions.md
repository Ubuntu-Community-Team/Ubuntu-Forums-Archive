### Post by meng on 2007-02-15
If you're talking about running it only from the Live CD (not installing), then the "option" is to have the Live CD in the drive or not in the drive.

If you're talking about "dual-booting", i.e. two operating systems - Windows and Linux - both installed to the hard-drive, with the option to choose which one to boot from when you start the computer up, that happens automatically after installing Ubuntu:

[www.psychocats.net/ubuntu/installing](www.psychocats.net/ubuntu/installing).
