### Post by 5-HT on 2006-06-18
Are you using GNOME (Ubuntu)?

If so:

[LIST=1]
[*]Open Nautilus
[*]Right-click on a file of type you want to change the default program to open
[*]Select properties
[*]Select 'Open With' tab
[*]Add the wanted program to open that type of file
[*]Either select the little radio button for the new program or remove the old default[/LIST]HTH
