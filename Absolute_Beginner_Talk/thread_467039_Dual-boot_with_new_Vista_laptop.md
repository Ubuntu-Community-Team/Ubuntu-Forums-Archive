### Post by p_quarles on 2007-06-07
Funny enough, these forums seem to have the most information about dual-booting with Vista. 

First, make sure you have your recovery disks for Vista. If the computer didn't come packaged with them, you should be able to burn them yourself.

Second, for Vista it's (apparently) best to use its built-in partition manager to shrink the main volume and then create a new partition (10 GB at the very minimum) for Ubuntu. 

Then, run the install CD and select the new partition for the installation. It's pretty straightforward, but you do want to make sure you have everything backed up and recoverable before you start.
