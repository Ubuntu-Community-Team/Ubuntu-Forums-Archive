### Post by annda on 2007-06-13
avidemux is good for extracting audio from avi files. you can cut the video file first to make sure only the fragment gets extracted.

i often use the included sound recorder (applications>sound&video>sound recorder) to just record some sound from various sources - just select Mix as input and later use the save as... option (simple 'save' prints an error). if it freezes, check the configuration.
