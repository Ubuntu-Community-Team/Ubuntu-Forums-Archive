### Post by dajhorn on 2007-11-15
Outlook Web Access is the webmail client for Microsoft Exchange.  (OWA isn't a protocol.)

The Firefox web browser should be compatible with OWA, but it will detect Firefox and hide many features like the calendar.

To get all OWA functionality with Ubuntu, you need the "User Agent Switcher" for Firefox, or you need to run Internet Explorer in Wine or Crossover.
