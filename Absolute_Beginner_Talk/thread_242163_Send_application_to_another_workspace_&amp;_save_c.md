### Post by mlind on 2006-08-23
> **raggit said:**
> Someone told me you can use a keyboard shortcut to send an application to another workspace, is this true? If so how?  
They also said that I can save the configuration to where when i turn the computer off then back on those applications will remain in there respective workspace?  Is that true, if so how?

For Ubuntu, see System > Preferences > Keyboard Shortcuts > Window Management
Actions you're looking for are unassigned by default.


Second one is probably possible using System > Preferences > Sessions > "Automatically save changes to session"
