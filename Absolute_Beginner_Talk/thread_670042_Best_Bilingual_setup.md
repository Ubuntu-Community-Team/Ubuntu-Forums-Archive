### Post by charleseddy on 2008-01-18
I apologize for the late response from our community.

Whether you prefer to install in hebrew first or english, it will work perfectly.  The language packs are designed by native speakers, so you shouldn't have problems with them.

You speak English, though, so I recommend starting in that language and adding later.  You can change a user's default language and keyboard layout, so that should be taken care of.

If you have any problems, e-mail me and I'll answer by 1200CST.  Others in the forum might also answer.
