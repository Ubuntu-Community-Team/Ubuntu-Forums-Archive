### Post by skippy81 on 2006-05-13
If the home directory is on the same partiton as root then I should think you would have to format over it.  Back it up, and when you do a reinstall I recommend you put / and /home on different partitions.  Leave 7-10gig for the root system, 1.5-2gig for swap, and use the rest of the space for home.
