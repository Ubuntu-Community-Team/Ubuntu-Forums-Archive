### Post by conscious on 2008-04-09
You can change your home dir in System -> Administration -> Users and Groups. Select a user, then go Properties -> Advanced -> Home directory.

If you want to do it manually, you should not only move the folder with mv, but also inform everyone you've moved by editing /etc/passwd (be careful enough not to mess it up).
