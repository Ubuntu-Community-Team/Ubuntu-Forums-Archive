### Post by theredcross on 2007-09-06
Download the .deb, right click and choose install (sorry, I don't use Gnome so I don't remember if you have that option), so alternatavely you can use ```
sudo dpkg -i ~/Desktop/devede_0.01-1~getdeb1_all.deb
``` assuming you downloaded it to the Desktop. should update everything all right for you.
