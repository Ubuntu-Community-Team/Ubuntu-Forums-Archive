### Post by Orochi on 2007-07-25
If you have a CD burner and a blank CD, it's extremely easy to burn the disc yourself. However if you only have a 12 gig drive, that might not be enough room. You'll be cutting it pretty close IMO. You can probably fit both OSes on there, but machines run more slowly if their drive is near full, so keep that in mind (Windows will also get fragmented a lot more if the drive is full). It might be better to get another drive to install Ubuntu onto.

Also from what I've heard, the 64bit version, while running faster, has some issues because it can't run 32 bit apps correctly and not all apps have 64bit versions out yet. So you may want to install the 32bit Ubuntu anyway (I may be wrong about this, I don't have 64bit myself).
