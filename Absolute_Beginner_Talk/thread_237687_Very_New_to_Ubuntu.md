### Post by bodhi.zazen on 2006-08-16
antivirus software is up and coming. There are Linux virii out there. Expect more as Linux becomes more popular. I have looked at most Linux anti-virus applications and they not yet as sophisticated as windows.

There are several GUI's available for Clam, try klam.

In general, install antivirus and run as soon as possible on a fresh Linux (Ubuntu) install and make note of any "infected" files. With a fresh install they are not likely to be infected, but rather false positives. Do not remove them but make note of them.

More important then viruses are root kits. There are 2 root kit scaners I am aware of. They both work well and I run them both from time to time.

jrjr- 
1. You should not need to scan outside or your home.
2. Don't fear the CLI, it is your friend.
3. See above for GUI for clam.

Klaidas is correct regarding the need to "protect windows boxes". I can't stand the thought of my shiny Linux box as a repository for windows virii even if the do not infect me.
