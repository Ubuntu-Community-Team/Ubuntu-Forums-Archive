### Post by xpod on 2007-05-20
Do you mean you want to install Ubuntu and beryl and then also run Windows(xp) from within your Ubuntu?
IF it is then the best thing you can do is install ubuntu first and make sure everything works with that before you go worrying about beryl or any Virtual Machine software....patience ;)
For beryl you`ll need to install the relevant drivers for whichever Video card you have then find a beryl tutorial relevant to your card and distro.

Nowadays with feisty, it`s as simple as using the restricted drivers manager in many cases then just installing beryl & co.....search for "beryl howto" on the forum main page and you`ll surely find one here that can help.

For the "other Windows" you might have a look at Virtualbox which is probably one of the simplest ways to set up a virtual machine.(if thats what you meant?)
You need to have sufficient ram to run both your Ubuntu and the virtual XP obviously but if you get it all set up it`s quite cool just flipping that cube back & forth between  Ubuntu and whichever VM you happen to have running....sure beats dualbooting:p

[http://wiki.beryl-project.org/wiki/Install_Beryl_on_Ubuntu](http://wiki.beryl-project.org/wiki/Install_Beryl_on_Ubuntu)
[http://www.virtualbox.org/](http://www.virtualbox.org/)

Worry about your Ubuntu first though.You got all the time in the world and will enjoy it much more if you take it one step at a time instead of possibly suffering "information overload" ....as i`ve often done myself

Good luck,,,and goodnight:p
