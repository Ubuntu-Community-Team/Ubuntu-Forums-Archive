### Post by Tomatz on 2008-03-26
Shared graphics memory is when the graphics processor uses some system memory this is usually done by "on board" (on the motherboard) graphics processors. Classically this slowed your system down but with the pcie buss you can do this without degrading noticeable performance (aslong as you have 2gb or over memory). This is called Turbo caching but its basically the same.

### Post by tylerspaska on 2008-03-26
On my laptop I only have 32 MB of video memory, and I can run every Compiz plugin with ease. I can also play all the 3D games that are available in the Ubuntu repos. When I had windows on the laptop I played Max Payne 2 and GTA: Vice City. They both ran very well. I even used to use the laptop for video editing with Adobe Premeire and After Effects - it was slow but worked.

On my desktop I have 256 MB of video memory and all the Compiz features still work well. I've been able to play every game I threw at it, even Crysis although I had to turn the resolution way down.

So, my advice is that if you are going to use Linux exclusively on the laptop I would go with the cheapest available option. If you are going to use Windows and play games, spend the money and get a nicer card.
