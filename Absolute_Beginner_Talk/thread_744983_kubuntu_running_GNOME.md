### Post by luisromangz on 2008-04-04
You don't need to install the whole Gnome desktop environment if you plan just to use a Gnome based app. You just need to install the Gnome, libraries needed by that program but not the whole desktop environment.

The app wouldn't look if your system look exactly like a Ubuntu/Gnome one, it will just try to load the libraries required for it to use, and if they are installed, it will work.
