### Post by mysticrider92 on 2008-02-06
Is your new hard drive recognized by Nautilus? It might set up the mount points on its own. If not, you will need to edit the /etc/fstab file by hand. Open it with this command:> gksudo gedit /etc/fstab (be very careful what you change in this file, it will make your system unbootable!!!)

You will need to decide on mount points for each partition. The line for your main hard drive can be copied and edited for settings. You only need to edit the <mount point> and <file system> spaces. If you need help with how to edit this, just post your current /etc/fstab, and the partition setup you used for the new hard drive.

This guide should help you with moving the /home partition: [http://www.psychocats.net/ubuntu/separatehome]("http://www.psychocats.net/ubuntu/separatehome")
