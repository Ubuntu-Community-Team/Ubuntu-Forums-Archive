### Post by EvilMarshmallow on 2007-02-01
Actually if you know the hostnames of the other boxen, you can connect to them with the "Network Servers" option on the main ubuntu menu (System? I think).  You'll have to log in with a valid windows account when you try to access the folders but it worked for me.  On mine it showed me exactly what the windows box would have shown if I'd gone through "My Network Places".

Sharing files FROM the Linux box TO the others (afaik, I'm a noob too) requires Samba to be set up.  Install it from the repos and then search the forums for "Samba Setup" or some such.
