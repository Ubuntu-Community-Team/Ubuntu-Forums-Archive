### Post by LuisGMarine on 2007-05-03
> **Kerrisis said:**
> I have a related question. I've managed to change my splash screen OK, but the background outside the splash screen is still that dusty orange. Is there any way to change that to, say, black?
**EDIT: Found the answer to this one here:  [http://ubuntuforums.org/showthread.php?t=430569&highlight=splash+screen](http://ubuntuforums.org/showthread.php?t=430569&highlight=splash+screen)**


Also, is it possible to alter the main Ubuntu loading screen (The one with the progress bar)?


Go to System > Administration > Login Window

Once the Login Window is open go to the ' Local ' tab, then under the list of available login themes, there is an option for 'Background Color' 

That is what you want to change, pick w/e color you choose and good luck :) :guitar:
