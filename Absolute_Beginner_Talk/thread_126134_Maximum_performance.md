### Post by GreyFox503 on 2006-02-06
If you're new to linux, don't do a so-called 'server' install.  It will not install any graphical interface, leaving you with just a command line.

I would try installing regular Ubuntu (or Ubuntu lite... I've never used it before) and get your half-life server running.  Once it works, then you can work on making it faster.  You could, for example, change it afterwards so that you boot directly to a terminal w/o a GUI.  That would save some memory.
