### Post by Kateikyoushi on 2007-03-12
If you plan to try both totally separately you could use ony / with home in it for edgy, if it works out you can swap to the /home partition and remove dapper. In case you really need to access something from dapper home can still copy it over.

I think this is a really safe way to try it.
