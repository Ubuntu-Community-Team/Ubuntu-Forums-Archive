### Post by fornix on 2006-05-16
Yes u can very much use windows and ubuntu together. Just make some free space(unformatted) available on ur windows drive. You can do this using tools like partition magic or u can do it during install time. Install ubuntu on this new unformatted space (linux will make the partitions for you), ubuntu will install the grub loader in mbr and you'll get a prompt at boot time to choose ur operating system. ubuntu or windows (which it will auto detect)
