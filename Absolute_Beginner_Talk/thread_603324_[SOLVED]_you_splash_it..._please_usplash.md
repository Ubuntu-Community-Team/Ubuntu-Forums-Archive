### Post by Perpetual on 2007-11-06
Check gnome-look.org for Usplash screens.  Download the Usplash and unpack to a folder of your choice.  Use SUM and the Appearance tab.  From there you can choose Bootloader themes (Grub background) and Usplash (login) theme. 

```
sudo apt-get install startupmanager
```
