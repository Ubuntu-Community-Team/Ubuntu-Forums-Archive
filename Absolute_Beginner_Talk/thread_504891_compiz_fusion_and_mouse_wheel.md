### Post by amadeus266 on 2007-07-19
In compiz-config-manager, under rotate cube, set the initialize option to "button 3". Yes, Emerald is th same as the one for Beryl. The easiest way to use emrald is to install beryl-manager and emerald from synaptic and create a start-up in sessions for beryl-manager. right click the ruby icon and then emerald to change your theme.
