### Post by Sef on 2006-07-27
> sorry, i'm pretty new at this, where can I find this "security tab"

No apologies needed.  We all started where you are at one time or another.

Security Tab > System > Administration > Log in Window > Security Tab (Fourth from the left) > click on Enable Automatic Login (Top one) > make sure that the name you want to automatically log in is set to the right of user.
