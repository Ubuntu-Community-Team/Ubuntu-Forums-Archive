### Post by Paqman on 2008-01-14
256MB of RAM is a little on the small side for a LiveCD install (although it should still work)

If you want something that moves a little quicker you could try donwloading and burning the Alternate Install CD. It's a text-only installer (no LiveCD desktop, no pretty windows) which can seem a little daunting. However, it's actually just the same process as the graphical install, and since you're going for a nice simple setup (no dual-booting) you shouldn't have any trouble. Once installed it'll behave exactly like Ubuntu installed the usual way.
