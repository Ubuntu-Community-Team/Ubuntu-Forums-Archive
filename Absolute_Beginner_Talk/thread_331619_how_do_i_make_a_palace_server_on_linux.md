### Post by 3rdalbum on 2007-01-04
First: You don't need 2 bumps in the same hour. We're unpaid volunteers, and so it may take a while before your question gets answered (if at all).

Okay. I had a quick look around on the Internet for "Palace Server Linux". Luckily, there is one:

[http://practice.chatserve.com/extras/pserver-4.5.1.i686-unknown-linux.tar.gz](http://practice.chatserve.com/extras/pserver-4.5.1.i686-unknown-linux.tar.gz)

Download that to your home directory, double-click on it and extract it directly into your home directory. From there, there should be a text file included in the folder that tells you how to install it.

There is a full instruction manual here:

[http://www.thepalace.com/assets/userguides/unixserver.pdf](http://www.thepalace.com/assets/userguides/unixserver.pdf)

I know it says it's for Unix, but it seems like it will still be relevent for Linux.

If there's something about it the instructions that you don't understand, look it up on Google, and if that doesn't help come back and ask us. I can't guarantee that anyone here has even run the Linux Palace client, much less installed the server.
