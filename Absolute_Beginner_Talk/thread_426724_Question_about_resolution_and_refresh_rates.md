### Post by LuisGMarine on 2007-04-28
First lets start off with what you got.  To resize the overly huge icons that gnome display simply Right-Click the icon, and from the menu select ' Stretch Icon ' option.  This will allow you to resize the icon(s) to the size the you prefer, hint giving you some extra room.

The next thing that I would do is shrink the size of the gnome-panels.  The gnome-panels are obviously the bars that stretch across the bottom and top of your screen.  To do so, Right-Click on any open space of the panel and select the ' Properties ' option.  This will bring up the ' Panel Properties ' window.  Under the " General ' tab, go to where it says ' Size ' and bring down the size in pixels.  By default I believe is somewhere around the mid 20's, Go down to a size that you are comfortable with.

The last thing to do, is the font.  For me, when I wanted more space on the desktop, the fonts had the biggest affect.  To edit the size of the systems fonts, go to System > Preferences > Font, and edit the sizes, pick other fonts and so forth.
