### Post by cstudent on 2005-12-27
You should have a file under your /home/username directory called automatix.log. It will tell you what you chose to have installed. If the power went out during the install, then not everything in the log probably got installed. You can run the install again. Whatever it has already installed should quickly get passed over and it should pick up where it left off.
