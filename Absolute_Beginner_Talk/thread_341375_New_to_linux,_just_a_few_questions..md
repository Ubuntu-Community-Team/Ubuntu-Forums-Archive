### Post by davebgimp on 2007-01-18
6.10 isn't necessarily newbie unfriendly, it's just the latest. 6.06 has been around longer and is therefore considered by some to be a safer bet. Personally, i don't see a problem starting out with 6.10.

You can always download sources, but take care of where you get them. Ubuntu, fully updated should be running the latest of it's build for Firefox. Personally, I rely as much as possible on the Ubuntu repositories as I feel comfortable trusting them.

### Post by Hendrixski on 2007-01-18
Ubuntu is one of the easier Linux distributions.  You may also be interested in Suse Linux, it's made to be ready for the enterprise, so brain-dead computer users can use it at work to perform mind-numbing tasks (like edit spreadsheets, or write memo's for their boss).  If it really is super-easy to use or not is up for debate.  Also Red Hat Enterprise Linux is a solid option.  And it only costs about $50, a hell of a lot less than Windows.

The Ubuntu repositories are not always the most cutting edge, but you don't want them to be, otherwise your system would be in a permanent Beta, and may keep crashing.  In other words it becomes a nerd toy and not a business tool.   Which may be what you're looking for, but make sure you know the difference.

If you really want a challenge try Gentoo Linux.  You have to compile everything from source, and the install alone takes a few days of dedicated work.  But you have 100% complete control over anything that happens in your computer, and learn more than you care to know about how Linux works.

If you want to try other *nix operating systems try Solaris 10.  They're giving out free DVD's now.  I'm not sure what they cost normally, but it's pretty cool.  Especially the Java Desktop Service.  Very easy to use by people with a Windows background.  Also comes equiped with CDE, which is still (after all these years) complete crap.

Hope that helps

### Post by jbus on 2007-01-18
Edgy is fine for new users... I wouldn't recommend editing your sources list manually though. Use System>Administration>Software Sources to add additional repositories if you must. Use Automatix [http://getautomatix.com/](http://getautomatix.com/) if you want to add more software/codecs than is available in the ubuntu repositories.

### Post by mikewhatever on 2007-01-18
Ubuntu 6.10 is great for everyone. The difference between 6.10 and 6.06 is that 6.06 is Long Term Support version. Both should work ok. 
If you are new to Ubuntu, save yourself the trouble and use 32 bit version. It works on both 32 and 64 bits processors.
About Firefox updates, yes it gets updated through Ubuntu repositories, as well as Open Office and the rest of software.
