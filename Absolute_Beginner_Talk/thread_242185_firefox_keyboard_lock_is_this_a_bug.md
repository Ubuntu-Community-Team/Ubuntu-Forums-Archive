Update, it's the libflash plugin. I was reading [www.carzorthade.com](www.carzorthade.com) and some of the more recent pages break the plugin.


So I have two firefox windows open, one to this forum, and the other to a webcomic I'm reading. Anyway I want to type a URL and although I can position the cursor the keys have no effect. I switch windows and in the other window the cursor is in the search box and I can type normally. If I cancel the cursor by clicking somewhere else then switch windows it may work as normal I think.
