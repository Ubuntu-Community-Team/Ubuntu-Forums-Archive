### Post by ajgreeny on 2007-06-03
If you will keep mint on the machine there is no problem; you can have as many distros installed as you want, as long as the disk partitioning maxima allow it, and even then you can install many virtual partitions in an extended partition, so no real problem.

You could even try using the same home partition, or at least copying all the folders from mint home to ubuntu home.  Mint is based on ubuntu 7.04 so most of the packages should be the same, I think, so worth a try, and if so, I imagine most of your prog settings and configs will be carried over.
