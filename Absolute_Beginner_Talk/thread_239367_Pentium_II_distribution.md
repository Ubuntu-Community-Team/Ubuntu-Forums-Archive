### Post by louieb on 2006-08-20
anaconda has it right - get some more memory. See if you can find your motherboard doc and find out what type of memory it uses. 
My guess is for a machine of P2 vintage it is using [COLOR="Red"]PC100 168 pin SODIMM[/COLOR] memory. PC133 memory will also work. 
Open the case and see how many memory slots you have my guess would 2 or 3.
One note: a lot of motherboards that use this type of memory will only use 128mb per stick, you can put in a stick with more memory but only 128mb will be used. 
Do a search on E-Bay for PC100 or PC133 you will find plenty of sticks in the $10 to $20 dollar range.
I am running Ubuntu on a P2 384 meg. Works just fine.
