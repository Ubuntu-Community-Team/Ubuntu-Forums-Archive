You should seriously consider upgrading to 6.06.1 as 5.10 is very outdated.

The best way to download and install software using this version of Linux is to access the Ubuntu repositories either by using the Synaptic Package Manager or opening the Terminal and using the "apt-get install" command.

For example, if you wanted to install the Bittornado bittorrent client, you could open a terminal and type:

```
sudo apt-get install bittornado-gui
```
