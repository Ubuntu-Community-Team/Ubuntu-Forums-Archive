### Post by drowner on 2007-05-31
What are the specs of your computer?

The LiveCD runs slowly, obviously as it runs from the cd and ram only. When you install to your HD the efficiency at reading and writing, and the usage of the swap file makes it run much more efficiently, don't worry.

I, for instance, can't run the latest liveCD on my computer very well, but the install flies, much faster than XP.

There are several ways to kill a misbehaving app in linux. The most aggressive is to open a terminal and type 'xkill' then click on the window that is misbehaving

But normally you won't need to do this, trust me.
