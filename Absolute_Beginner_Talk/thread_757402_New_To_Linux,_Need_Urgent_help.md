### Post by Bachstelze on 2008-04-16
Completely erasing Windows is not a very wise thing to do. It is possible that you will we able to do in Linux everything you used to in Windows, but even so, there will be a learning curve, and it might take a while before you are comfortable enough with Linux. So in the meantime, it is better to keep Windows as a "failover" solution, so you can get back to it if you need to get a job done quickly that you don't know how to do in Linux yet.

To answer your question, if you *really* want to wipe Windows away completely, all you need to do is to start your Linux installer and, when you're at the partitioning step, delete the Windows partition. You will then have an empty drive and be able to create partitions to install Linux on.

### Post by Tatty on 2008-04-16
I would agree with the reccomendation to dual-boot.

If you have had no linux experience at first, you will probably struggle at first to do certain things - so you dont ever want to find yourself stuck when you are in a rush.  Over time you will get more and more confident with linux and will eventually get to the point where you may not boot into windows for months...

So i would reccomend keeping a windows partition just in case.  Well, if you have paid for it anyway then why not?
