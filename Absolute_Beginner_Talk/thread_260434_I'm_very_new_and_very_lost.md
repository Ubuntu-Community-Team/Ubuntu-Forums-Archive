### Post by aktiwers on 2006-09-18
Hi and Welcome!

You can start by getting Automatix. Thats a program that will help you get the basics to work.

This is there website:
[http://getautomatix.com/wiki/index.php?title=Installation&Itemid=38](http://getautomatix.com/wiki/index.php?title=Installation&Itemid=38)

This is how to install it.
Go to the terminal and copy paste this. (the Terminal is in Applications=>Accessories=> Terminal.)

```

wget [http://www.getautomatix.com/files/automatix-installer](http://www.getautomatix.com/files/automatix-installer)
chmod 755 ~/automatix-installer
./automatix-installer

```

Then go to Applications=>System Tools and open Automatix. Pick all you like (incluting Java packages) and hit install. Now it will install all you need.
