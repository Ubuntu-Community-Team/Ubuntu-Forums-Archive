### Post by eljalill on 2007-06-25
You execute executable files through navigating  into that directory (cd /and so on), and then type
./[executable file]
A script is essentially an executable file.
You might have to make it executable with the chmod command.
[http://linuxcommand.org/wss0010.php](http://linuxcommand.org/wss0010.php) is a good explanation on scripts for beginners, I think.
