### Post by Williams477 on 2005-07-10
In Windows, ctrl-alt-delete brings up the task manager and you can kill a program from that. In Ubuntu, you can set a key combination to bring up the system monitor and you can kill a program from that. Should be under System>Preferences>Keyboard Shortcuts
