### Post by spiderbatdad on 2008-04-11
I use camorama and gtk-recordmydesktop in combination.
gtk-recordmydesktop depends on recordmydesktop...all are in synaptic.

I start camorama..select view large. Then start gtk-recordmydesktop.
I get video and audio in an .ogg file.
Set audio and video to 50% for best results
Select only the portion of the desktop you want to record, ie. the webcam screen.

ffmpeg allows me to convert to other formats like avi.
