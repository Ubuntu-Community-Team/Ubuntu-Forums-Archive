### Post by bkildow on 2006-09-23
This is kind of a silly question, but in Quanta Plus 3.5 is there any way to turn off the dots that appear at the beginning of a line when you hit tab? Also I like the code completion hints quanta has, but when I start typing a function and hit ctrl-shift-space to bring it up, and I continue to type, it disappears. If it do this before I start typing anything and then type it stays up. Is there anyway to keep it up the first way? Thanks for your help!

### Post by bkildow on 2006-09-23
I figured out the dot issue, a little persistance pays off! To turn them off go to settings -> configure editor -> editing and uncheck show tabulators. Now if only I could get the code completion hints to work a little better...
