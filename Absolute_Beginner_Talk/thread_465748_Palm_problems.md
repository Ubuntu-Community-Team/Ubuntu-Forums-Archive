### Post by lazyart on 2007-06-06
In j-pilot (feisty)

File>Preferences>Settings

Set serial port to usb:

Before Feisty I had to do a network sync because I couldnt get usb to work.  In that case set the serial port to "net: any"

Then configure the Palm to sync across the network and specify the ip address of your ubuntu box.
