### Post by Valok on 2007-12-12
In order to use screenlets you first have to DL the screenlet manager which I assume you already have.

Once that is done, all you have to do is DL the screenlet that you want, exract it, and then move that extracted file to the following location /usr/local/share/screenlets/

Once the newly DLed and extracted folder is in this location it should show up on the screenlet manager window.  If it does not, try closing out of and restarting it.
