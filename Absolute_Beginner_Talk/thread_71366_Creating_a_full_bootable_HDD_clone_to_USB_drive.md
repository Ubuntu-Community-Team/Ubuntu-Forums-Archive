I know there is a detailed HOWTO in these forums on making a system backup using TAR, but from what I can tell, this is just a zipped copy of the files on a hard drive and requires a separate new system to be set up to allow the copy to be un-tar'd back to a new hard disk in order to recover a system.

As far as I know this 'TAR' method doesn't include all boot loader gubbins that go into making a complete bootable system - please correct me if I am incorrect.

I want to know if there is a similar package for Ubuntu, or a way of simply achieving the same results, so that in the event of a castatrosphic failure all I need do is change my BIOS to boot from the USB HDD to get up and running in as short a time as possible.

This is a travel plus work issue - sometimes I don't have the time or facilities to do a regular recovery, but I desperately need to use my system.
