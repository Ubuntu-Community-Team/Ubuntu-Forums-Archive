Hi. Can anyone tell me where to find the open office equivalent of the command in the Word window menu that allows you to divide a file into two windows or to show two or more files on the screen horizontally or vertically, etc.? Thanks.

---

### Post by SteveHoffmanUK on 2007-10-16
Window > New Window

This opens a new window for the current document, which you can then resize so that you can view different parts of the same document at once, either side-by-side or one below the other.
