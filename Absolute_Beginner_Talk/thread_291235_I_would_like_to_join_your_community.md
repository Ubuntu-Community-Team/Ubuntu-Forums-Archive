### Post by bluenova on 2006-11-06
It will be better once installed. When you are running from the cd it has to load each program you use from the cd, so would take much longer than from the hard-drive. Especially OpenOffice as it is a big program.

Games is still a big problem with Linux, If you use the computer mostly for games you would need to sadly stick with MS Windows, or dual boot.

Ubuntu uses the [Gnome desktop](http://images.google.com/images?svnum=10&hl=en&lr=&client=firefox-a&rls=org.mozilla%3Aen-GB%3Aofficial&q=gnome+desktop&btnG=Search)
Kubuntu uses the [KDE desktop](http://images.google.com/images?svnum=10&hl=en&lr=&client=firefox-a&rls=org.mozilla%3Aen-GB%3Aofficial&q=kde+desktop&btnG=Search)
Xubuntu uses the [XFCE desktop](http://images.google.com/images?svnum=10&hl=en&client=firefox-a&rls=org.mozilla:en-GB:official&sa=X&oi=spell&resnum=0&ct=result&cd=1&q=XFCE+desktop&spell=1)
