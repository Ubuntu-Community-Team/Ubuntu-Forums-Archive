### Post by kb3hkg on 2006-06-30
that would depend on the dvd, if it is an encrypted one then no you cannot just copy from one to another. If it is just one you made yourself you should be able to use a program like K3B or GnomeBaker. To copy an encrypted dvd would require possibly illegal actions depending on where you live. To do this use a program to rip it like DVD Rip-O-Matic, and then burn out the video file it creates to a new dvd. Obviously if it involves breaking the law I do not recommend you do it.

### Post by felipead on 2008-02-24
Just install the "libdvdcss2" package and use k3b to copy the DVD.

You can copy the DVD to another media or to a ISO image.

By the way, K3b is the best media-writter software I've ever used. It's more usable, attractive and definitely better than many programs for Windows, including Nero and Roxio.

---

### Post by felipead on 2008-02-24
And if you want to copy a double-layer DVD (9GB) to a single-layer DVD (5GB), you can use other tools, like Thoggen and K9copy. However, these tools only work if "libdvdcss2" is installed.
