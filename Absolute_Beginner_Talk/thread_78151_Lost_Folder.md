### Post by erikpiper on 2005-10-17
You know the trash in windows? Ubuntu has an equivelent. Normally in the bottom left corner. I however never use that. see below.

----------------------------------------------------------------------------------------------
Go to the directory you deleted the second directory in, and show hidden files. (If you are using gnome or nautalis, this is by clicking view, then checking show hidden files. I think ctrl-H does the same.)

Once hidden files are shown, look for .Trash (- The . in front hides it. All files with a . in front are hidden.) Look thru the .Trash. It should be in there.
