### Post by slymi2005 on 2008-04-21
hitting alt+F2 and then typing in " update-manager --devel-release " will allow you to update to the 8.04 release candidate. Not sure if it would be 8.04 or 7.10 but I would assume it would go to the latest.

You need to follow the instruction on upgrade from the respective release.
From 7.04 to 7.10: [https://help.ubuntu.com/community/UpgradeNotes](https://help.ubuntu.com/community/UpgradeNotes)
To Hardy: [https://help.ubuntu.com/community/HardyUpgrades](https://help.ubuntu.com/community/HardyUpgrades)
