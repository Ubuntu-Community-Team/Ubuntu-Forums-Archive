### Post by darkmatter on 2005-10-22
KDE is a more 'integrated' environment for your desktop, with many more configuration options.

GNOME is more about simplicity. It has a more consistant UI and fewer GUI 'wizards'.

That is a very basic comparison. I could go into more detail, but the result would resemble an essay.

My honest recommendation: Try them both, and see which is better suited to your needs.

### Post by duffman25 on 2005-10-24
[QUOTE=arsbadmojo]I just installed ubuntu yesterday; I did a 'standard' CD based installation. I don't remember being prompted to pick a desktop environment. Am I using Gnome or KDE? How do I tell, and more importantly - how do I switch between them to tell which one suits my needs?[/QUOTE]

Ubuntu is delivered as a one-cd-distro. This means that:
ubuntu is based on gnome
kubuntu is based on kde

The core of the distro is the same in both cd's, but the desktop eviroment (which is large in size of packages) changes. You can have both installed (what's more, you can have every d.eviroment you want installed). If you have already installed ubuntu, you can install the kde desktop with synaptics/apt, check:
[http://www.kubuntu.org/faq.php#installfromubuntu](http://www.kubuntu.org/faq.php#installfromubuntu)
