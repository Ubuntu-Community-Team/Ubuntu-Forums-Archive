### Post by RelativelyQuantum on 2007-05-02
The type of problem you described might be solved by switching to roaming mode, available in feisty. You may have already tried this, but you can open the network configuration window at System>Administration>Network. At this point, select your network card, and the "Properties" button should become active. Click this, and you'll get a window in which there is a check box at the top which enables roaming. I found this useful when I had a similar problem.

-Good luck!
