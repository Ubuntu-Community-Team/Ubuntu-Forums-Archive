### Post by Rocket2DMn on 2008-01-05
Sounds like you may need to reconfigure your X server (and Xorg) by running this following little program that will ask your questions about your hardware.  Do your best to answer, and when you get to the part about screen resolutions, choose the TV's highest resolution and everything less - this will give you the option of using this resolution.  We will start with a manual backup of xorg.conf.
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
sudo dpkg-reconfigure xserver-xorg
```
I think you will want to use the "nvidia" drivers, not "nv" (the open source drivers).  Then restart the X server with CTRL+ALT+BACKSPACE.  If for some reason the X server fails to load, you can reinstate your backed up xorg.conf with
```
sudo cp /etc/X11/xorg.conf.backup /etc/X11/xorg.conf
``` and restarting X again.

### Post by Rocket2DMn on 2008-01-06
The resolutions that are looked at are under Screen, subsection Display.
Under the Modes part, there should be an entry for "1920x1080" or even "1920x1080@60".  You can try manually adding it and restarting X with CTRL+ALT+BACKSPACE.  If it fails to load, just restore your backup.  If this doesn't work, I don't really know what else to tell you other than check and see that you have the latest nvidia proprietary drivers.
