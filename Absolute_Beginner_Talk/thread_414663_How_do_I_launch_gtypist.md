### Post by Jussi Kukkonen on 2007-04-20
Reading The Fine Manual helps (*man gtypist* is the command for it). It should start just fine without any options. If you want to use the error percentage option try something like this:
```
gtypist -e 5
```
If it still doesn't work, please include the actual error messages.

EDIT: by the way, if you're still evaluating different programs, take a look at ktouch -- it's a nice GUI touch typing program.
