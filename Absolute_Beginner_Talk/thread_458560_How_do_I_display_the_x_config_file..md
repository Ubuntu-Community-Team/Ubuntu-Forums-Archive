Take your pick:

Open a browser and go into /etc/X11/xorg.conf

OR

Whatever you want to do.  :)

Put "sudo" in front of all those commands if you need to edit the file.

### Post by RomeReactor on 2007-05-29
Ok, seems the file *is* there, only empty. Let's see if this one has the configuration:
```
gedit /etc/X11/xorg.conf~
```
If it does, then you do this:
```
sudo cp /etc/X11/xorg.conf~ /etc/X11/xorg.conf
```
to restore xorg.conf.
Fortunately there are a lot of backup files, so you can use the ones that are more recent.
