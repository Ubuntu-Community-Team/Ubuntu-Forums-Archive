When Ubuntu starts it asks me the user name and the password in the session login, and the same when I want to shut down (first log off, and only then I can shut down!) . Is there a way (I am sure it has to be!) to avoid this boring step?

### Post by KIAaze on 2007-07-29
1)System -> Administration -> Login Screen Setup
2)General Tab -> Automatic Login ->
Login a user automatically on first bootup (Checked)
Automatic login username: Select "system_username"

[http://www.zolved.com/synapse/view_content/28020/How_to_login_automatically_in_Ubuntu]("http://www.zolved.com/synapse/view_content/28020/How_to_login_automatically_in_Ubuntu")

However, it's not normal that you have to log out before shutting down.
And the password shouldn't be necessary to shut down.
