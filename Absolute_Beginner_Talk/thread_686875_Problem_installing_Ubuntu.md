### Post by spiderbatdad on 2008-02-03
I have installed Ubuntu on two T-40's. Generally, a simple install with all hardware (even wireless) working out of the box. It is pretty simple to replace the hard drive. You should get an installation error like..."bad sector on disk" or something if the hard disk is bad.

Did you buy a refurbished T-40, or was it new? If the mini-pci wireless card is incorrect, yet "hacked" to work, it will begin to cause os crashes, then bios lock ups, finally motherboard ruination. This is not an uncommon patch-job replacement, for some reason. If you google the FCC ID# printed on the wireless card, it should bring you to lenovo, where you can verify that # with the 2 or 3 available for the T40.
Otherwise...the hard disk comes out with one screw, and is certainly worth replacing for a computer as reliable as a thinkpad.
