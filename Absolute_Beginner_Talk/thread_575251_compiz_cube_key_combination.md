### Post by ajmorris on 2007-10-13
if its enabled in ccsm, the key combination to rotate the cube is ctrl+alt+1st mouse button and drag, or ctrl+alt+left/right arrow keys. There are too many key combos for all the different effects to put here, if you go to run a command (usually alt+f2) and type ccsm, you can go through all the different effects enabling/disabling them and also going to the key combination tab within them, to see the key sequence to start them.
