### Post by LancerDragoon on 2007-06-27
I like Rhythmbox myself, but Banshee is also a good media player. Both mimic the iTunes interface. 

To install Banshee:
```
sudo apt-get install banshee
```

For Rhythmbox, it's installed by default, so look for it under Applications -> Sound & Video.
