### Post by hanexar on 2006-07-18
> **K-J said:**
> how do i save changes to the Sources.lit file?

You need to have root privilege to save change to the sources.list file.

Open the file with the terminal, with this code:

```
 sudo gedit /etc/apt/sources.list 
```

You can easily save with the save button in the tool bar.

good luck.
