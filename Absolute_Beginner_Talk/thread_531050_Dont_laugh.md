> **lululin said:**

Why wait until you can build a computer? Ubuntu really doesn't require any special skills to install, and out-of-the-box it is a very easy operating system to work with.

The transition from being a Windows Power User to being an Ubuntu power user does require learning, but this is the best place to start learning, and not something you need to get into if you don't want to.
You don't need to know how to manipulate the registry in Windows to use Windows. Likewise you don't need to understand the terminal to use Ubuntu.

Everyone here has suggested you use a Live CD. If you follow the download now link on [ubuntu.com]("http://www.ubuntu.com/") you will be asked what kind of computer you have and what Ubuntu you need.

Just like Windows there is a 32-bit version & a 64-bit version. The 32-bit version is the one you want (and will work even if you have a 64-bit processor).

Once you have downloaded and burned the iso (see [http://psychocats.net/ubuntu/iso]("http://psychocats.net/ubuntu/iso") for an easy guide to follow). You now have a working Ubuntu Live CD.

Live CDs are wonderful things. If you switch your computer off and then switch it on again with the Live CD in your disk drive, you can boot from the CD. Some computers do this automatically, some need you to press a key when starting up... however you do it your computer will load Ubuntu instead of Windows. Without making ANY changes to your computer.
You can use Ubuntu without touching any information on your hard drive, since it runs off the CD.

Then you switch off the computer, take the disk out, turn back on and Windows will load up as normal.

This is a great way to have a play with Ubuntu, see if your hardware/printers etc will work straight away or need a bit of work to get going and generally get an idea of the OS.

Just don't run the Installer until you are ready to install for real.

You have two options for installing: you can have a dual boot or install ubuntu over Windows.
If you have a dual boot you can choose if you want the computer to load windows or Ubuntu.
If you install Ubuntu over Windows you'll just have Ubuntu.

Both are very easy to do, it's up to you.
