I'll assume that you are on windows then..

Open the connection manager that came with the card, insert the card when it asks for it, what happens?

Does it still say please insert card? go to the device manager, make sure you have no unknown devices (yellow question mark). IF you do, right click, and say update driver. If not, check under Modems, that's where the card is going to show. 

If the driver is cool, and it still says please insert card, uninstall and reinstall the software WITH THE CARD OUT, start it up, and insert the card when it asks. Still doesn't work, 

Try it in a different computer

Make sure you have no other internet connection

still says please insert card....then you got a bad one. Get it replaced.
