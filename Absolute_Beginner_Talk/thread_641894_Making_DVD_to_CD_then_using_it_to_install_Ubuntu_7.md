### Post by Arthur Archnix on 2007-12-16
If you have an iso of ubuntu 7.1 then it is going to fit on a cd. Using your favorite burning software just tell it to burn to a cd and it will write the iso to a cd. If it says it can't do it because the iso is too large, you'll just have to download a cd version from the website.

I'm not sure what you mean by best, but if you tell ubuntu to use the entire disk during install that wouldn't be very different from using something like gparted to remove the drive. On the other hand, using something like dban would achieve something very different than just letting ubuntu use the entire disc.
