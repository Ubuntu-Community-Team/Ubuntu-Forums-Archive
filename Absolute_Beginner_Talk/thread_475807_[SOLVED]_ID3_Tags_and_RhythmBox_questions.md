### Post by starcraft.man on 2007-06-16
Ok, I know this might not be completely intuitive with easy tag. Open it up and find the songs you want to batch edit. Select the first song, then shift click on the last song in the batch. Now you selected. Now right into the artist/album fields whatever you like and any other fields you want to apply to everything. For each of the fields once you get the right name, right click on it and select the first option "Tag selected files with this field". The only field it doesn't seem to work for is the year, I'm not sure. Thats not so important though. Once your done modifying the files, push the save button with them all still selected.

Fix all the tags with easy tag, thats my solution. It won't take long with batch modifying, I assume all the names are right...
