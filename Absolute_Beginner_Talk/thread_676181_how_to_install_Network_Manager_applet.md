### Post by dasunst3r on 2008-01-23
Try this:
[list=1]
[*]Right-click on your newly-created panel, select "Add to Panel."
[*]Under the "Utilities" selection, select "Notification Area."
[*]Click "Add."
[/list]
This should give you your NetworkManager applet and other things back.
