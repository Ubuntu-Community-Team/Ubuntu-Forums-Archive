### Post by thelatinist on 2007-12-16
If an application stops responding there are a number of ways to close it:

1.  System > Administration > System Monitor.  Click on the process you want to close an choose "kill."

2.  In the terminal, type **killall *process-name***

3. There is a button you can add to your panel called Force Quit which will allow you easily to close a window.
