### Post by GreyFox503 on 2006-02-06
Some options for Gnome applications can only be changed through the "Configuration Editor".  I didn't look too hard, but at least one app (nautilus) has a place to set its startup size & location.

Open Applications > System Tools > Configuration Editor, then browse to /apps/nautilus/preferences/navigation_window_saved_geometry

You can edit that string to modify the program's behavior.  Changes take effect instantly.

Hopefully some of the other apps listed have a similar option.
