### Post by tchoklat on 2007-01-08
Burn the CD Image you have downloaded to CD. Set your PC bios to boot from CD first and reboot. Ubuntu will load from the CD giving you the option to install it. Ubuntu exists quite well with XP as a dual boot. It will create it's own partitions for you.

### Post by confused57 on 2007-01-08
Here's how to burn an iso to cd:
[http://www.psychocats.net/ubuntu/iso](http://www.psychocats.net/ubuntu/iso)

The key is to select "burn as an image", not as a data cd, burn at a low speed(8x).  As tchoklat mentioned, set your bios to boot first from your cd drive.

Since you have 2 hard drives, you might want to consider this option:
