### Post by zorkerz on 2007-11-11
Im pretty sure ive been able to do that. Athough I have not spent much time in the live cd environment. I change mine with the gnome keyboard indicator in 7.10.

You can also go to system->prefrences->keyboard->[tab]layout and then clicking Add. In the "layout" dropdown pull down to U.S. Eglish (this would be for an english dvorak layout). Lower in the "variants" dropdown select dvorak or the preferred layout. Click the Add button again. Select the desired layout and close.

Hope you found a solution sooner than this.
