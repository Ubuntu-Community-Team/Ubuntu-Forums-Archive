### Post by LaRoza on 2007-05-29
Notepad is not the best for reading text files created in other programs. It uses crlf whereas most others do not, so if something is written with a different editor, it won't render properly in notepad.

notepad++ is good, as is Notepad2, and NotepadSX. Each one has different features, but they are very good notepad replacements.

For the ultimate text editor, try Crimson Editor, it has everything you could possible want.

If you name files with a .txt file extension, Windows will open that file with whatever program is associated with that extension, by default, notepad, but you can change that.
