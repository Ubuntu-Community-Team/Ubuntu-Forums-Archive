### Post by souneedalink on 2008-03-13
It is a 800mhz PIII not a 233mhz PII or anything. I suggest tweaking gnome to lighten it a bit and use lighter apps rather than the heavier ones. Gnome is not that heavy by itself, only if you load a ton of crap at startup  and use heavier apps and so forth.

You should simply be able to use the alternate CD to install or you could use it to upgrade your system without reinstalling.

You should be able to pick up some extra RAM on ebay for a few bucks and that will improve performance considerably. I should have some 64mb and 128mb sticks laying around and would gladly send you one...

Before I got my new thinkpad R61e I used my omnibook 6000 700mhz PIII for everything and with 256mb it was just fine, and 384mb was even better.
