### Post by deepwave on 2006-10-24
Beryl should run under any window manager.  It works definitely with Gnome and KDE.  Beryl is graphics intensive, an ATi or nVidia card is recommended.  It should run decently on a Intel Integrated graphics chip though.

Only one warning, Beryl is in the alpha stage and can be a bit tricky to setup.  Tricky but not extremely difficult, if you feel comfortable with the console you should have no real problems.
