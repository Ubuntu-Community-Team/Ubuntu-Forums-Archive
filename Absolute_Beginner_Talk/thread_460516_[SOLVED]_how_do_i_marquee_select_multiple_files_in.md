### Post by wormser on 2007-05-31
I am not sure what you mean by marquee select.  It sounds like you want to highlight multiple files and move them.

There are 2 methods for selecting multiple files in nautilus.  Click on one file then hold the control key and click on others.  The other method is to select multiple files and all the ones in between.  Click the file at the top then hold shift and select the last file in the list.  Every file in between will also be selected.

Hopefully that is what you were looking for.  If not could you explain it better.

### Post by starcraft.man on 2007-05-31
I just checked this out quickly, and this is what I come up with.

1) Thats a limitation of the way nautilus browser displays list I assume. There is a very easy way to fix it. Either A) Click the first folder you want to select and then hold down shift key and click the last folder, that will select all folders in between. Or B) Hold down the ctrl key and click files individually to select the batch you want (use this method if their separated by space i.e. not all in a row.

2) I believe this is also a limitation of the browser itself, I don't see the problem with using menu/keybinding, how many folders do you make daily?
