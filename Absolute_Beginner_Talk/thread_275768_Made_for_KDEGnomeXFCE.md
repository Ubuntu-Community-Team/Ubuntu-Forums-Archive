### Post by taurus on 2006-10-11
Basically, you can run KDE apps in Gnome, XFce4, fluxbox, or whatever window manager you are using.  It will load all the necessary libraries first before it loads the program.  For instance, I always use k3b to do my burning no matter if I use Gnome, XFce4, or fluxbox.  Works every single time.  ;) 

p.s.  You can also run Gnome apps in KDE too...

### Post by tonyr1988 on 2006-10-11
(someone correct me if I'm wrong)

All of your programs will work no matter which window manager you use. However, it just may look "different" from programs designed for your window manager. For example: if you're using Gnome (default for Ubuntu), load Epiphany (the Gnome browser) and Konqueror (the KDE browser). They'll both be functional, but you'll be able to tell a huge difference.

I try to stick with Gnome apps, just because they integrate better. But there are some KDE apps that are (IMO) just plain superior. I use K3B for burning and Amarok for playing music.

In the end, it's all your preference.
