### Post by JoshuaRL on 2008-03-02
Definitely use Compiz-Fusion.  As Crooksey says its the merged product of Compiz Core and Beryl.  If you're running Gutsy (7.10) it should come pre-installed.  You will want to install the Settings Manager though.

Just go to Applications->Add/Remove Applications.  Then search for "compiz".  It'll be the Compiz Advanced Settings Manager.  That will allow you to configure compiz how you want.

Hope that helps.
