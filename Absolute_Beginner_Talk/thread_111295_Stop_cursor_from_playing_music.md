I am using Ubuntu 5.04 (Hoary).  When the cursor is placed over a mp3 file in File Manager the song starts to play.  Unfortunately, File Manager often locks up and I can't figure out any way to stop the music other than going to a terminal and shuting down the system.  Also, this is quite annoying when all I want to do is copy a file.  How do I stop music playing when the cursor is placed over the file?

### Post by Madpilot on 2006-01-02
In the file manager, go Edit menu --> Preferences; the last tab is Preview and you can turn off the music preview stuff there with Sound Files.
