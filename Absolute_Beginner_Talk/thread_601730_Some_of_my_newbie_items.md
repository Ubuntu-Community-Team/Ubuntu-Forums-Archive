Hi,

I wanted to put a few newbie items i have done to make my Ubuntu experience better.  Plus if I do a reinstall, I can track what I have been doing.

1.  Change login screen settings (so I dont need a password).
2.  Install Virtualbox  -- sudo apt-get install virtualbox-ose
     a.  mainly needed to run outlook + exchange
3.  Install Amarok & Compiz - sudo aptitude install amarok compizconfig-settings-manager
4.  Configure better desktop effects
5. Install Crossover, Outlook, change to XP mode, install SP2, configure - no domain name needed
6.  Install IE (for OWA)
7.  Install Outlook in Panel


Its a small list for now, but that will change over time!
