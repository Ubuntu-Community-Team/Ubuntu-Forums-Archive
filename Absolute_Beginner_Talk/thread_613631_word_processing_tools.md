### Post by deserthowler on 2007-11-15
> **kleo skywalker said:**
> What is latex?
What about software used by publishers for books etc?

LaTeX is essentially a type setting program.  It produces output that totally blows away any word processor output you have ever seen or probably imagined.  Just AWESOME.  Your output will look like it is done by a professional printer.

OK, there is a problem with LaTeX.  It's complex with long learning curve ... real pain for the casual user.  It's learning a programming language.  A good text editor (I use Emacs) is recommended and control codes are entered for different styles, etc.  There's a plug-in for Emacs that helps with the LaTeX formatting.

LyX is a GUI front-end to LaTeX.  It's more word processor like.  It saves learning a lot of the codes but you can still work on the control code level if you would like.  It's still some work to learn but might be worth the extra effort.  I don't know about the graphics handling abilities because I haven't used them.  Once you become familiar with LyX, you can concentrate on your writing and struggle less with styles and formatting.

LyX is in the repositories for 7.04.  I'm sure the other versions have it too.

Earl
