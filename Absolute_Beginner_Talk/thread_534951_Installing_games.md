### Post by rkrug on 2007-08-25
Install wine, then in the terminal run ```
winecfg
``` 
You don't really need to do much, just click OK.

Then put in the CD for your game and do ```
wine /media/cdrom0/setup.exe
``` (or where ever the setup file for your game is)

And install the game as you normall would.
