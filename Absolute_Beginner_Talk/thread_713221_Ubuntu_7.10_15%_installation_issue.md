### Post by jken146 on 2008-03-03
The alternate CD is a text based install CD (nothing scary, it just asks you the same questions as the installer on the live CD).  It doesn't run a live Ubuntu session.  You can get it from [www.ubuntu.com/getubuntu/download](www.ubuntu.com/getubuntu/download) and ticking the box labelled "Check here if you need the alternate desktop CD. This CD does not include the Live CD, instead it uses a text-based installer."  Some people get better results with the alternate CD, especially on older machines (although your specs seemed fine).

You could give the live CD another go, leaving it lots of time to do its thing (it normally doesn't take very long but I have seen it appear to hang when it isn't), then if you don't get any luck, download the alternate CD.

[www.psychocats.net/ubuntu](www.psychocats.net/ubuntu) has a section on using the alternate CD, if you need it.
