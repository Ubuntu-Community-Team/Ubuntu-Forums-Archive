### Post by Placenta Juan on 2007-06-13
This section is for the default drivers:
Section "Device"
Identifier "ATI Technologies Inc RV350 NJ [Radeon 9800 XT]"
Driver "ati"
BusID "PCI:1:0:0"
EndSection

And this section is for the proprietary drivers:
Section "Device"
Identifier "aticonfig-Device[0]"
Driver "fglrx"
Option "VideoOverlay" "on"
Option "OpenGLOverlay" "off"
EndSection

You have 2 "Section "Screen"" entries, the first calling the default driver, the second the proprietary fglrx driver.
I'm not an expert on xorg configurations, but comparing your config to my own I notice that I only have one Section "Device" and one Section "Screen", both the fglrx version.
I would try deleting or commenting out the Section "Device" with the ATI identifier and the Section "Screen" that refers to it and then rebooting, but make sure you make a backup before making any changes! I'm not sure if that will fix it, but it would make your xorg.conf look like mine (which is succesfully using ATI's proprietary fglrx drivers)
