### Post by timzak on 2007-06-21
Do you know about Add/Remove and Synaptic?  They're both graphical, so they're more intuitive to noobs than the command line.  Things that are installed via these two methods automatically install...completely, including launchers in the appropriate category.  Each method has a Search function where you could type 'p2p' or 'java' and you'll find easily installable programs that fit under those categories.  Java is available in both Add/Remove and Synaptic Package Manager.

Add/Remove is under your Applications and Synaptic is under System->Admin.

Good luck.
