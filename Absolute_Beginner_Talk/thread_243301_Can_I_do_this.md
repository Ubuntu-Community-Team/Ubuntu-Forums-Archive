### Post by ciscosurfer on 2006-08-24
The LiveCD doesn't install anything...it's run from RAM and temp file directories...you can then choose to install if you like...but to get an idea of what Ubuntu is like, the LiveCD is the best way to go.  Again, the LiveCD doesn't install anything.  Just hit Enter on the first option on the LiveCD boot menu (don't worry, it won't "Install" anything to your drive...after it boots to your desktop, you can then choose to install if you like)
