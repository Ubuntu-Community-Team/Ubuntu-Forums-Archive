### Post by getut on 2007-09-03
As security conscious as the linux community is, why isn't Macromedia/Adobe Flash bombarded with issues of security concerns?

The fact that the ability to control the security and privacy settings isn't provided with the base install on the local PC should have everyone at arms, but yet you never hear about it. You have to go to a really buried web site on their web page to change the security settings and dump the tracking info (flashes equivalent of cookies) that you already have before that point.

Flash is everywhere now and knowledge is so low for everyday people that don't realize that  flashes tracking capability far exceed cookies. It just really shouldn't be ignored because it is a popular add-on. It actually seems very nefarious and underhanded that that functionality isn't provided to end users with the base install.
