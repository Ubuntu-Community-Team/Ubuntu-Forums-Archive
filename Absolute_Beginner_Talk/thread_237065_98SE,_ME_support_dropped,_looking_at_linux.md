Hello A_608,

WoW!! First of all, I would like to say that you are the first person I've seen going to Linux, because of the end of support for Windows 98SE, ME.

Now for your questions:

The only problem I can see is the memory. Maybe, instead of Ubuntu, Xubuntu would be better for your case. [Xubuntu]("http://www.xubuntu.org/") is a lighter version of Ubuntu especially for older machines without much memory. It uses the XFCE desktop environment and I'm sure you are going to like it. It's basically Ubuntu in the core but with a different desktop; that's all.

Ubuntu, on the other hand, it will run at a decent speed, but with this amount of memory you cannot install it with the *Live CD*. You will be able to do the install with the *Alternate Install CD*.

You will be able to open your Office 97 documents with OpenOffice and then save them to OpenDocument format which is an open standard and you will be sure that you will be able to open your files in the distant future. Just for info, [Xubuntu]("OpenDocument") come with AbiWord as a documents editor. I don't know the capabilities of AbiWord but you can install OpenOffice easily if you want.

You will easily be able to dual boot with XP but you can try if this program, Sibelius, or any other Windows application you need, can run with Wine on Linux. This, way, maybe you will not need Windows at all!

You will not have problems with viruses in Linux; they are very rare. If you will feel unsecure, you can easily use an firewall in Ubuntu.

For the printer, I don't know but I can see it was answered already.

I wish you good luck!

Greg:  for a system like yours, I would actually recommend that you try a much lighter distribution first, like [Damn Small Linux]("http://www.damnsmalllinux.org/").  It's very very cut-down, but that'll give you an idea of what sort of Linux install you'll need to build to be fast & responsive on a machine like yours.

Even Xubuntu might be a little too heavy--I'd try it out first.

Also, as the time comes for you to install, you might want to consider reading the [Installation on Low Memory Systems]("https://help.ubuntu.com/community/Installation/LowMemorySystems") wikipage.  

A general caveat:  "ultralight" installs tend to assume that you know what you want and you know what you're doing.  If you're in doubt, well, that's what the community is here for.  But the ultralight route has a much steeper learning curve than 'regular' ubuntu.  The leaner you go, the farther away from your Windows experience you will travel, so don't expect *anything*---and I mean *anything*--that you think you know from Windows to mean much.
