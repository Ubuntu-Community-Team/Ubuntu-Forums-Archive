### Post by forrestcupp on 2008-03-11
But honestly, one of the Gimp's downsides is it's text capabilities.  You may still need Inkscape.  You'll just have to make sure you're using file formats that each program recognizes.

I don't know how to export a chart from OpenOffice, but if all else fails, you can always take a screen shot with your Print Scrn button and paste it into the Gimp.

Edit:
Actually, I just tried it, and you can select the chart, right click it and choose copy, then paste the chart directly into a Gimp file.  Then you can go from there with Inkscape and Impress.  Just make sure you're using compatible file formats.
