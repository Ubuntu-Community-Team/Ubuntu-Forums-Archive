### Post by Shay Stephens on 2006-09-08
Gimp lacks a simple batching/action manager.  Instead you have to program what you want to do in gimp-fu, a language based on scheme that is not easy to master.  You can also use python-fu which should be easier to learn.  However these are really poor ways to perform a simple action that need to be repeated on multiple images.

As of right now, my best suggestion would be the one above and try to learn imagemagik to do batch processing.

As an alternative, if you have it, you can install and run Photoshop 7 in wine and perform your batching operations there.  That is what I currently have to do as I have yet to be able to get anything done with gimp/python-fu.  And I have not tried imagemagik yet, though I have installed it.
