### Post by hyperair on 2008-02-15
If you want to highlight and copy lines from PDF files, Ubuntu's default PDF viewer should do the job for you. 

If you want to edit PDF files, you should try pdfedit. Open Synaptic Package Manager (System->Administration->Synaptic Package Manager) and then install pdfedit from there. Or use this command to do the same thing: ```
sudo apt-get install pdfedit
```
