You shouldn't have any problems installing a distro on a virtual box, I've installed debian, kubuntu(was while I tested a potentialy dangerous script, forgive me) and gentoo on a vbox, none of which had problems.

As for which is best:

Ubuntu is a good choice if you want to learn things like Python(which most ubuntu apps are written in).

As far as which distro to run as a guest, that depends on what your virtual machine software will support:
