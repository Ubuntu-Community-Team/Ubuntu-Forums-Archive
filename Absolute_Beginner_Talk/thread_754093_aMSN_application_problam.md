### Post by lswest on 2008-04-13
the problem is amsn by default uses mozilla to open urls, well, guess what?  Ubuntu doesn't have mozilla.  From the main amsn window go to account-->preferences and then under the "Other" tab change the line that reads "mozilla $url" to "firefox $url" and the next line you can change to "nautilus $location" so the actual filemanager opens.
