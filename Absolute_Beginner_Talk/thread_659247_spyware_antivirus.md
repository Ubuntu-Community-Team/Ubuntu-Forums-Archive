### Post by p_quarles on 2008-01-05
> **aktiwers said:**
> But most likely if there were any treats they would be made for Windows, and could not run on Ubuntu.
That's really not true. Most browser exploits use client-side scripting, which is OS-independent. Where Linux gets its greater security is in the fact that a non-root user can only mess up his own home directory -- thus its a good idea to ensure that any potentially dangerous commands are run without any possibility of root access.

### Post by thelatinist on 2008-01-05
I would never recommend frequenting known infected sites.  You should definitely make him use a separate user account and disable sudo. Still, your computer should be safe...certainly much safer than a Windows PC accessing the same websites.
