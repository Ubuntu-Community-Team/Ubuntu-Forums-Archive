### Post by aysiu on 2006-11-14
> **alepliotu said:**
> The Run command line and the console is the same thing?
Not exactly.

If you run one command to launch a graphical program, then, yet--they are the same.

But if you run in the Run dialogue a command that has terminal output, you won't see that output, and most terminal applications cannot run from the Run dialogue properly.

---

### Post by GStubbs43 on 2006-11-14
> **alepliotu said:**
> The Run command line and the console is the same thing?

The *command line* and *terminal* and *console* are all the (basically) same thing
But if you have one command (for a program to run such as "firefox") then pressing alt+f2 will bring up a window for you to type in a program.
