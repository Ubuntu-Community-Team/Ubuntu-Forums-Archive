### Post by mejy on 2006-09-01
> **Jareth said:**
> any suggestions on what to do with it now?

I map it to my gnome menu - go into System -> Preferenes -> Keyboard Shortcuts.  Scroll down to show the panel menu, select it and press your windows key.  Makes keyboard navigation much easier.  Alternatively, you might want to map it to 'shade window'.  I don't use this feature much since my display's pretty high-res (1600x1200).
