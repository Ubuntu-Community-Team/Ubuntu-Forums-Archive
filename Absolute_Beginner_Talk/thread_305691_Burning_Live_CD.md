### Post by Derek Tomes on 2006-11-23
Once you have a live CD you start it as follows:
1. Stick it in the CD drive

2. Restart the computer

3. As the computer restarts - jump into the BIOS (this is before Windows starts) this usually involves pressing F2 or F8. Your computer will normally display something like F2 = Setup as soon as it begins starting.

4. The BIOS is usually character based. Use arrow keys to move around and read the instructions at the bottom of the screen.

5. Look for "boot sequence" and make sure that the CD option is first. Save and exit

The computer should boot into Ubuntu...

Good luck ;)
