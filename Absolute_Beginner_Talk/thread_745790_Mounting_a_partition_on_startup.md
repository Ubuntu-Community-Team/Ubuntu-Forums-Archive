### Post by joshrobinson on 2008-04-04
first backup your fstab file
```
sudo cp /etc/fstab /etc/fstab.backup
```

unmount your drive
```
sudo umount -a
```

open fstab
```
sudo gedit /etc/fstab
```

next add this line to the bottom
```
/dev/sda3   /media/disk-1    fuseblk  defaults,umask=0000  0  0
```
save and close fstab

make the mount directory
```
sudo mkdir /media/disk-1
```

mount your drives
```
sudo mount -a
```

### Post by joshrobinson on 2008-04-04
this is my last idea, then someone else will have to help you, i really have no experience with fuseblk filesystems

```
sudo apt-get install ntfs-3g
```

then try this fstab line
```

```

then 
```
sudo mount -a
```
