### Post by annda on 2007-02-24
when you install ubuntu, you will be able to access the windows partitions and files in any file manager.

when you install the fs-driver in windows, it will ask you under what drive letter it should access linux and you will see another disk in "My Computer".

of course this is for installed systems. if you want to see your windows files using ubuntu's liveCD, you will have to mount the windows drive in linux. there are plenty of threads decribing how to do that.

### Post by Dylnuge on 2007-02-24
> **jamil_1 said:**
> i have all my software pakages made  for windows such as that of maya and photoshop 
do i need to purchase new copies of these packages made specifically for linux

Unfortunatly, with Windows and Linux being different Operating Systems, not all software is made for Linux. Maya does have a linux version, you may be able to contact the company and "trade in" your windows version for a linux version.

Adobe does not make software for Linux, neither does Microsoft. This is usually the most common request, equvilent wise. However, there is a free program for Linux called Wine. Wine allows you to run Windows software under Linux, with some tweaking.

LiveCD is the Installation CD for Ubuntu. It lets you use Ubuntu without installing it, before installing it. Since you cannot install applications or save files in the LiveCD (you can save files to a flash drive), it is essentialy to let you see if you like Linux before commiting.

You can get a free copy of Dapper from Canonical (Ubuntu's host). However, it will take a while. Since Linux is free, they don't mind. Its much faster to burn an ISO.

Li-nux is the correct pronunciation of Linux. Thats Li as in Lint, and not Li as in Liability.
Oo-BOON-Too is the correct pronunciation of Ubuntu.

Enjoy Linux.
