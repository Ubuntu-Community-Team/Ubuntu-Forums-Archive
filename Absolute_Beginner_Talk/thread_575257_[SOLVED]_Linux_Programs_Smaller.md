### Post by HermanAB on 2007-10-13
Linux systems are extremely modular.  Code re-use is strongly encouraged either through libraries, or by spawning utilities in /bin or /sbin to do some of the heavy lifting.

Also, since the code is open, big balls of spaghetti code will not live long and will either be dispatched to the big bit bucket in the sky, or improved by someone else.
