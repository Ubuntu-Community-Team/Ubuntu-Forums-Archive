### Post by Shatrat on 2006-12-03
I'm not terribly familiar with edubuntu but from [http://www.edubuntu.org/GettingStarted](http://www.edubuntu.org/GettingStarted) it doesn't seem like quite what you are looking for.
I think it would simplify things to just install regular vanilla ubuntu, 6.06 or 6.10, and then install whatever educational programs you wanted.

As for turning the computer off, if edubuntu has gnome and a gnome-panel you can right click the panel and use + Add to Panel to add a Shut Down button anywhere you like.

### Post by steve.horsley on 2006-12-03
It's not clear from the home page - you have to get halfway down the Getting Started guide, but Edubuntu is intended as a server install for schools, and to which diskless clients will connect. This is **not** what you want for home use. Like Shatrat, I suggest that you install the standard Ubuntu. There are plenty of educational things you can install. 

Once you get 3d acceleration going, you **must** look at celestia. It's an awesome space/planetarium thing. It's a little hard to get into, so I attach a small tutorial that will give a feel of what it can do.
