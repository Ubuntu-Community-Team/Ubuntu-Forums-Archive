### Post by atlien on 2007-02-23
Are you 
right-clicking the file ->
 selecting 'Properties' -> 
selecting the 'Open With' tab -> 
selecting 'XMMS Music Player' (MAKE SURE THE BUTTON TO THE LEFT OF THE PLAYER NAME IS NOW LIT) -> 
then 'Close'

That does it for me in Ubuntu 6.10
