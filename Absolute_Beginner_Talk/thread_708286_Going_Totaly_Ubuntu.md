### Post by pedro_orange on 2008-02-26
See [www.winehq.org](www.winehq.org)

There are alot of games out there for linux. There are alot out there for Windows. Some Windows ones you can run through WINE/Cedega. But be prepared to fiddle.

You can always dual-boot your games machine if you want to run Linux & still be able to game on Windows. It's not the end of the world ^^

### Post by aeiah on 2008-02-26
it depends on the games you're playing. you may prefer using dualboot. games from ID software tend to be well supported (doom3, quake4, quake4 enemey territory) and popular geek games like eve and WoW work well. others work ok through wine with minimal fiddling, like half life 2 and some call of duty games etc, although a lot of games will still have you bashing your head against the wall to get them running. a lot of the time you'll find a slight performance hit if you're running them through wine, although this isnt always the case.

its usually more convenient to dual boot but it depends on your gaming habits. if you have long gaming sessions then id just dual boot. if you like to play a game for 20 minutes every now and again dual booting could become very annoying though
