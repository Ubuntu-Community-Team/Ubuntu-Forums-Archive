On my computer when there where no drivers for my card it went black, so it isn't the same problem. But you should try the same thing as I did (which worked).

1. Safe graphics mode
2. Change resolution to either bigger or smaller screen, with less or more colour range.

When you are in the cd's start menu you should click... [forgot] something which makes you able to edit the boot up.
There you add vga=791 or something.    (click [here]("http://www.knoppix.net/forum/viewtopic.php?p=36777") and find the resolution table).
