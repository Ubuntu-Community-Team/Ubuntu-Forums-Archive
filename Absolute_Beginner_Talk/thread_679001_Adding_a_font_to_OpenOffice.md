### Post by twin_57103 on 2008-02-07
Here's a method that worked for me:
In your home folder, create a folder names .fonts Ii.e. a hidden folder).  You can do this from nautilus or from the command line (mkdir .fonts).  Copy the .ttf file into that folder (subfolders are OK, too).  In order to view hidden files from nautilus, you will need to turn on the display hidden fonts option (view...show hidden files).  That was enough to add fonts for me (had to close & re-open Open Office).
