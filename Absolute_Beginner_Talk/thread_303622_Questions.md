### Post by NetworkGuy on 2006-11-20
I personally would order the Dapper CD, or download the ISO and make your own CD if you don't want to wait 4-6 weeks for Shipit.

Dapper, by default will install a generic 32 bit kernel which you can then work on installing the 64 bit kernel that should work better for your Athlon.

I'm not sure that GameSpy will work with Linux.  Most commercial games in the world are Windows based, while some of them will work when run on top of other programs like Wine, its not guaranteed.  You will probably need to do some research on your own for that.

Surfing the Web - Not an issue, FireFox works fine for the sites I hit.

### Post by jaboua on 2006-11-20
You probably want the PC version, and may then install some 64bit-apps (kernel and mplayer, for example) later on as mentioned. The 64bit-versions can give slightly better performance, but from my own experiences you will have lots of trouble as well...

If you want to play lots of games, dual-booting with windows is probably the best alternative as well. Windows-games work best under windows, and in my experience *some* of them will be emulated OK in linux too, but then they will suffer from worse performance and more bugs.

Linux is great for browsing the net though, as you don't have to worry about viruses and such. You might have some problems with plugins though - there has a long time been a problem with flash as the most recent version was flash 7 while the windows world moved on to flash 8, but now there is a beta version of flashplayer 9 too for linux (and it's generally worked fine for me, but it hangs from time to time... Reload and it works again :P). Flashplayer is currently only released for 32-bit OSs (read: the PC version), thus you need to install a 32-bit browser in order to use them in 64-bit ubuntu. There is no version of the shockwave player or Active X for linux afaik, but I've read of people running browsers in wine for emergency cases, and a quick google revealed that CrossWeavers has released an ActiveX plugin for linux now (but I'd guess that it's commercial)
