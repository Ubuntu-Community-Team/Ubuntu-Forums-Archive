### Post by jamvegan on 2007-09-23
Each user has a ~/.mozilla/firefox/<random>.default/bookmarks.html file, where <random> is a random personal identifier.  You can copy any user's bookmarks.html file to any other user's directory (noting that doing so will overwrite any bookmarks they may already have).

### Post by R.Bucky on 2007-09-23
Try installing Google Browser Sync. [http://www.google.com/tools/firefox/browsersync/](http://www.google.com/tools/firefox/browsersync/)
It syncs your bookmark settings over the network. I have the same settings at work and at home. It is beautiful!
