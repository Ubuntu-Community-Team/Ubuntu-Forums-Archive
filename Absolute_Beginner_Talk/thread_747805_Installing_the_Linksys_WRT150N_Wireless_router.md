### Post by tommcd on 2008-04-06
So you connect to the internet via a DSL pppoe modem, and you want to hook up a wireless router? In other words it would go like this: phone jack > DSL modem > wireless router > computer. Is that right?
Check the router's documentation. You probably need to change some settings in the router's firmwear in order for it to work with pppoe modems.
See this: [http://kcnet.org/dsl/linksys_pppoe.htm](http://kcnet.org/dsl/linksys_pppoe.htm)
