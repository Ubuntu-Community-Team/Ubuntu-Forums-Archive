### Post by inksmithy on 2008-04-16
Thats one of the beautiful things about Ubuntu. It is based on Debian and as a result has possibly the largest installable base of software in the Linux world. If you are really desperate to add repositories, add the debian testing and unstable repositories, but in all honesty, there won't be much in them which you don't have access to already.

Medibuntu is useful for codecs and so on, but apart from that, just go with whats already in the repositories - more than 22000 binaries last time I looked I think.
