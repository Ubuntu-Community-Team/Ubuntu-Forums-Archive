### Post by php_penguin on 2007-06-24
Windows strives to improve as well, but the difference is that Windows is a business.

It therefore wants the maximum return on any investement, and so will do the bare minimum to convince users that it's latest iteration is a vast improvement (and so people will buy it).

Linux however, is run by a community of people who want the maximum praise on their investement, and the only way to get large amounts of praise is to produce a program or OS which raises the bar.

Microsoft rules because of the relative obscurity, or abnoxious fanboys, of other OSs. Unfortunately this means that Linux will always be playing catchup to Windows and Mac, because only by improving on the key features of the competitor OSs will Linux draw users away from their comfort zone. You don't advertise a bed to a hammock owner by saying it has springs, you advertise it by saying they won't fall out as often.
