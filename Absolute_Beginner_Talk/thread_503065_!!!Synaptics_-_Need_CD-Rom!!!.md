### Post by Sandlst on 2007-07-17
I believe this is happening because some packages are on the install cd, so instead of connecting to the internet to download them, getting them from the cd would be faster.  I usually disable this first thing after an install, to do this open synaptic, goto Settings/Repositories (in the top menus) and uncheck all cdroms in the bottom of the window that opens up.  After reloading the packages this should solve the problem for you.  Good luck, post back if you have any problems.
