how to i make a shortcut to a folder in my hard drive for my desktop/toolbars?

---

### Post by BobCFC on 2007-06-24
If you right click on the desktop and choose Create Launcher... 

in the type choose File

in Location write the directory path such as  /home/fawn/Downloads

You can also choose an Icon.   It is the same process for making an application shortcut but choose a different type instead of file.


To add a shortcut to a panel... right click on free space in the panel choose Add to Panel then Custom Application Launcher lets you do the same thing.
