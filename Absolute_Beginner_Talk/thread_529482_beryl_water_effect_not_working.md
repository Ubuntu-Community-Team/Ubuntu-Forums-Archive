### Post by schorsch on 2007-08-19
Right click on the beryl manager (the red gem in the panel) and choose:

Beryl Settings Manager --> Extras

Check if the item "Water Effect" on left is checked.
You can also check there which keyboard shortcut you have to use or change it as you like.
