### Post by tyggna1 on 2007-11-17
Check out your 
System->preferences->Removable Drives and Media

Have a look at those options.  Your fix might be as simple as un-checking them, rebooting, and then checking them again.

### Post by daengbo on 2007-11-18
It sounds like the Gnome Volume Manager isn't running. Check System -> Preferences -> Seesions and make sure that Volume Manager is set to run on session start-up.
