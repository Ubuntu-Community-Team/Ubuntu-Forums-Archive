[QUOTE=Szary]Sorry for asking, but i looked everywhere, Ubuntu Forums Gallery, DeviantART, Google... Can't find this wallpaper. Does anyone know where to find it?
[/QUOTE]

```
sudo apt-get install ubuntu-calendar ubuntu-calendar-december ubuntu-calendar-february ubuntu-calendar-january ubuntu-calendar-march ubuntu-calendar-november ubuntu-calendar-october
```

Then look in /usr/share/backgrounds
