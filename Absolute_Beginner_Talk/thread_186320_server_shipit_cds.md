### Post by catlett on 2006-06-01
From Ubuntu web page [http://www.ubuntu.com/support/faq?highlight=%28server%29%7C%28install%29](http://www.ubuntu.com/support/faq?highlight=%28server%29%7C%28install%29)
> Server

There are two server install versions.

First we deal with the version that you can do with a "Ubuntu Install CD". Then we deal with the "Server-install CD".

During the install there is a pause. You are presented with the word 'Boot:' at this stage if you type in the word "server" at this time the machine will do the minimal install. This does not install any packages other than those in the base Ubuntu install.

After this you may choose to install a desktop or you may choose server packages. Such as Apache, Zope, postfix or some other package. Some people asked why this was not done by default, the reason is that some installations a box does a simple job and the choice of server software is quite vast.

The server-install disk which is referenced on the download site above, has more server like packages (web servers ntp servers and the like) on the server-ubuntu disk, with nothing more than you could find with an internet connection. [http://cdimage.ubuntu.com/ubuntu-server/](http://cdimage.ubuntu.com/ubuntu-server/)
