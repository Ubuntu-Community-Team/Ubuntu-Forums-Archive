That's a good start. :-) 

> 
how do i know if ubuntu is right for me? will i still have all of my programs once i make the switch?

If you're specificly wanting to know if Ubuntu is right for you, a Ubuntu live CD is a great way to find out. If you're checking on Linux in general, Knoppix is a good choice. 

About if you'll have all your programs...depends what you use, and how flexable you are. Most Windows programs have Open Source alternatives which work very well on Linux. Some of the programs that don't can be run via "wine" a program that lets you run some Windows programs in Linux. 

The flexable part could come in, for example, with the photography. The Gimp is a great image manipulating/editing program. It's generally considered an alternative to Photoshop. Some people are very used to Photoshop's workflow though, and the switch is too much to handle. I personally prefer the Gimp's setup and use it even when Photoshop is available (such as in the school's computer labs.). 

To sum it up: Ubuntu is a very nice version of Linux, and can probably do all the things you need it to. Try it on a live CD to be sure. Make a list of the least-likely-to-have-a-free-counterpart software you use and check what your options are in Linux for that software. You can have high expectations, but realize that Linux is not Windows so there will be differences.

### Post by pbaehr on 2006-02-21
The only hesitation I have in recommending Ubuntu as a primary operating system is that you note you're using your computer for "very amateur photography."

As long as it stays "very amateur" you'll be fine in Ubuntu but if you find yourself crossing the line from "very amateur" to "professionally interested" you're going to find yourself quite locked to Adobe's fine line of products.

As a professional graphic designer I'm keeping a close eye on Gimp, Scribus and Inkscape but I'm afraid it will be a long time before they can compete with Adobe's Creative Suite.

I use Ubuntu on my home machine but my work machine needs to run Windows to operate in my professional environment. I pray for the day that Adobe decides to port their applications to Linux.

But, again, I stress: On a "very amateur" level and even an "amateur" level you'll be quite happy with the freely available software that does what you're looking for in Ubuntu. It's a real cost saver for people who design as a hobby.

### Post by nalmeth on 2006-02-21
If you'd like a version of GIMP that is supposed to be similar to Photoshop in interface, there is the [GimpSHOP]("http://www.gimpshop.net/"). It is not included on the liveCD, but if you do decide to dual-boot, you might want to look into it. I'm sure you will see alot of differences still, and some things missing from photoshop, but you might see some things on gimp that photoshop doesn't have.

Some people have suggested wine if you need some of your specific windows programs going, but don't be persuaded that it will run all or even most of your windows apps. Of course it depends on what you need. It could run all of them, or none of them. Personally, I would only guarantee that it will run some of your apps, at the least. And they may require some tweaking. Crossover office is not free, but they guarantee that microsoft office will work, and photoshop too. OpenOffice is the open-source alternative, and is compatible with microsoft's .doc format, though it has been known to layout some work differently. 
If you're into gaming, Cedega is a company that runs "most", not just some windows games, but not all. It has a 14-day trial, and subscription is $5 a month, for at least 3 months. After that, you can stop paying and keep playing, but you won't get updates for new games. Some games have native ports to linux, and have linux installers for the install CD's, and there are some open sources games always in development.

hopefully you can get linux to work for you, but expect a different experience. If you run into troubles, forums like this one will help trememdously
