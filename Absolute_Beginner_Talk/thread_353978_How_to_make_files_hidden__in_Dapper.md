### Post by mcduck on 2007-02-05
You can create a file called .hidden and type names of files and directories you want to hide. Nautilus will not display those files. Also all file and directory names starting with a dot are hidden.

You can show hidden files any time in nautilus with Ctrl-H.

But the hidden files that Windows puts on USB sticks are not usually anything important and you can safely remove them.
