### Post by starcraft.man on 2007-06-13
> **luckyd said:**
> Do wine and crossover run applications faster or the Virtual Machine emulators like virtual box?

WINE and CrossOver do run near native, they form an application layer and work very well if the game is supported.

You can't game in Virtual Box. It doesn't emulate a Video Card of any quality. If you want to game in a VM, only one that works that I know of is the latest 3.0 parallels version and that is Mac only for Windows host. If you don't have a mac, you need WINE/Cedega/CrossOver to game or a dual boot to Windows. VMware is trying to catch up to this but its not ready yet I don't think, Virtual Box hasn't announced anything about this either...

Edit: Wuups, read bad. You said application, second half is general answer, first half is for gaming.

WINE and CrossOver are still better for applications in general cuz the Application Layer is lighter and runs better with less problems. Thats only if the app is supported though.

A VM creates a full implementation of a Virtual Computer, emulating everything and thus takes a lot more resources and is in general slower. They will of course support any application but require more resources to do so well.
