### Post by Trollax on 2007-02-26
Assuming you're not running Beryl (dunno anything about it); there's no additional software required. Just  go to "System > Preferences > themes" and drag/drop the tar.gz file into the window that opens up. it should ask you if you want to apply the theme or not. Afterwards you can customise the finer details by selecting a theme and clicking on "theme details."

You can install Icon themes the same way.

To install GDM(Login screen) themes you need to open "System > Administration > Login Window" and drag/drop the file there.
