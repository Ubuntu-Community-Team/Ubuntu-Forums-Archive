### Post by Kilz on 2006-07-11
> **riothaus said:**
> How would I go about making RealPlayer the default mediaplayer on my system?
Using Drapper by the way.

I dont think there is a place to assoicate all media in one place. Find a media file you want to assoicate with Real player. Right click on the file, chose Properties, Click on the Opens With tab. Select Realplayer, if it isnt there click add, scroll down to real player in the window that pops up. Then click add. From then on all media of that type will open with Realplayer by default.
