### Post by Perfect Storm on 2008-02-26
It's

```
chmod +x <bin file>
sudo ./<bin file>
```

In my case it is

```
cd ~/Desktop
chmod +x PlaneShift_CBV0.3.020-x64.bin
sudo ./PlaneShift_CBV0.3.020-x64.bin
```
