### Post by mjm115 on 2006-05-17
If you are familiar with KDE, you should give [Kubuntu]("http://www.kubuntu.org") a try.  It is Ubuntu with KDE as default instead of Gnome.  If you already have Gnome, then you can:

> 
sudo apt-get install kubuntu-desktop


This will install KDE for you.
