### Post by Warren Watts on 2007-08-02
I'm running Ubuntu on a 450 MHz PIII with 384 MB  and 10 GB, and it runs just fine.  
Does everything I need to do.  I haven't tried running Compiz or anything fancy like that on it, but it is just fine as a basic apache and ftp server, as well as running Firefox.

When I first installed Ubuntu on the machine, I only had 192MB of RAM, and the lack of memory did slow it down a little, tho.  I would suggest that you at least up the memory to 256MB, if not a little more.  It will make a noticeable difference in performance.
