### Post by NicoleM on 2007-03-29
can you be more specific?  what sites are you trying to view but can't.

if the site says it is not compatible with firefox and recommends IE you can use a firefox extension called IEtab to view these sites.  see more info here: [https://addons.mozilla.org/en-US/firefox/addon/1419](https://addons.mozilla.org/en-US/firefox/addon/1419)

if you cannot view the site because of flash, shockwave, or other plugin issues then you need to install the required plugins.  see more info here: [https://addons.mozilla.org/en-US/firefox/browse/type:7](https://addons.mozilla.org/en-US/firefox/browse/type:7)

if you outline your specific problem i'm sure someone will be able to give a more detailed solution.  good luck!
