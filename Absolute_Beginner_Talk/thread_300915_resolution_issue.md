### Post by TorenC on 2006-11-16
if your using regular ubuntu, you can go to system -> preferences -> fonts. then there's a button that (i think) is called advanced (or something like that), and in there you can set your font DPI. Try 120. Back on 6.06, you could do the same thing in Firefox, but if you have 6.10 with Firefox 2, you'll have to just tell it to use bigger font sizes.
