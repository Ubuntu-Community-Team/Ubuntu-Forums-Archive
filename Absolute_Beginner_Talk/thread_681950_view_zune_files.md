### Post by MasterJS on 2008-01-30
If you dont have a Windows Partition, either use VMWare or use Virtualbox to set up a virtual machine. Just give it a 6 GB limit and then share your home folder between the virtual machine and linux. I think that's what an earlier poster was trying to tell you. I personally use VirtualBox for windows because of the interface and the possibility of seamless mode. It helps with those programs that you absolutely need and you can set the Virtual hard disk to be dynamically expanding so it will only take up space once you install things in it.
