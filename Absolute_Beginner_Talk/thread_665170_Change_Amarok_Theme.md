### Post by PeterJS on 2008-01-12
I assume you downloaded the amarok theme file? To install it go to Setting > Configure Amarok > Apperance and down at the bottom is a button that says install new theme. After the theme is installed select it from the drop down box and hit apply. Just a heads up, this only changes the context browser theme. I couldn't find a QT theme on that page that would change the overall and widget look of amarok if that's what you're going for.
