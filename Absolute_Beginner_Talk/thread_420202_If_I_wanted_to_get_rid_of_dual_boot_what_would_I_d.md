Right now I dual boot my ubuntu with windows 98. I need some of the windows office function for work, but if I can get WINE working, then I am thinking about getting rid of windows entirely. Is there a way to get rid of my windows partition without having to reinstall Ubuntu and losing all my Ubuntu data and programs? Thanks.

---

### Post by LaRoza on 2007-04-23
You can get rid of Windows without affecting Ubuntu.

You have a number of options, I would use GParted to delete the Windows partition and then resize the Ubuntu partition to fill the area.

I used GParted and did not lose any data, but you should back things up, just in case...

### Post by LaRoza on 2007-04-23
GParted is very easy to use.

It is a live cd.

It will give you a graphical representation of your hard disk and any other disk.

It is very graphical and intuitive.

You can perform more than one task at once, like delete a partition and reformat it, or delete it and resize another partition to fill the area.
