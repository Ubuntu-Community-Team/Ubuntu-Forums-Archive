### Post by t0p on 2007-12-21
> **Joeb454 said:**
> 
The Live CD needs 384 Mb of RAM to run. Hope this helps :)

The System Requirements printed on the Live CD sleeve certainly *claims* that you need 384 MB of RAM, but this isn't true.  My PC has 256 MB of RAM, and I was able to use the Live CD.  Really, all I wanted to do was click the Install icon; but before doing that, I had a little play with the Live CD OS.  It was a little sluggish, but it was usable.

If you just want to Install to your hard drive, you definitely don't need 384 MB of RAM.  I think you only need the Alternate Install CD if you've got *very* restricted system resources, or unusual requirements (eg. you want to install the server version).
