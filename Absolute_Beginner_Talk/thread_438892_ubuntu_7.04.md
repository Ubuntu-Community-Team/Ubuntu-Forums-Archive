### Post by tkjacobsen on 2007-05-10
You should probably post this under "Absolute Beginner Talk"..

You can find an Ubuntu starter guide here:
[http://ubuntuguide.org/wiki/Ubuntu:Feisty](http://ubuntuguide.org/wiki/Ubuntu:Feisty)


/root/ has to be root. The root user in unix systems is the ultimate system user who can do everything with the system. (However this user is disabled in Ubuntu. Use sudo instead. Learn about that in the following)

You can go to system->administration->users to add/remove users. 

You can probably find a place in system->preferences where you can change password. I think there is some user preferences or something like that. (I'm not using gnome, so a can't see for myself)

Good luck and welcome to the community !!
