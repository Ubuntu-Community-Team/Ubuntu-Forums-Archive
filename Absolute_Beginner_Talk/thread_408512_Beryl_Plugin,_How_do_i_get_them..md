### Post by jvc26 on 2007-04-13
Once you've installed beryl, then you have all the plugins (except for some of the unsupported ones which are in the unsupported plugins package) such as window grouper and some other funky closing and opening windows effects.
```

sudo apt-get install beryl-plugins-unsupported

```
Presuming you're using 0.2.1 then the flame thing is just under the Beryl settings manager -> Visual Effects -> Animations

> 
Like a way to like scroll through open windows, i seen one that like connected with all the windows and was able to like be moved all at once.

Alt-Tab does that, and its already available in the standard beryl installed package, Super+Tab does the ring switcher

> 
The Snow Effect, but instead of snow could i get like Rain.

I think you can pop to Settings Manager -> Extras and edit it in there.

Finally, all of this stuff is pretty much available in the Beryl Settings Manager screen you just need to look through, Beryl's forums will be a lot more helpful to you than here really as its beryl plugins you're asking about, and as said [here]("http://ubuntuforums.org/showthread.php?t=349732") beryl is off topic in this forum, take a peek into the desktop environments section instead for beryl questions.
