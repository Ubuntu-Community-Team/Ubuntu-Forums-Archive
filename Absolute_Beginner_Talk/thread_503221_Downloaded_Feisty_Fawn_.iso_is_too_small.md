### Post by Barrius on 2007-07-17
The whole point is to make the distribution (live CD and install), with the standard packages necessary to fit onto a single CD - faster and easier to download (or cheaper to ship), and faster to install.

One CD to try out (liveCd), if everything works install it.   You should be up and running in a half hour, an hour at the most.    The only advantage that an Ubuntu DVD has over a the CD version is that it has more applications/packages - it's still the same base OS.    

The lack of an internet connection will prevent downloading updates, but you can download the DVD version from an internet connected computer to install new packages.
