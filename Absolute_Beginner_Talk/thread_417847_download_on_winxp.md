### Post by Spr0k3t on 2007-04-21
Not sure I understand... you downloaded a zipped ISO file and you are trying to write the ISO file to a disc?  Not sure where you are getting your zip file from.  What I would do is either redownload the ISO image file direct from ubuntu.com or preferably find a suitable torrent shared of the same location.  If you know your ISO file is good, then you can read up on how to create the LiveCD here:

[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

Here's a great guide which will help you on your installation:

[https://help.ubuntu.com/community/Installation](https://help.ubuntu.com/community/Installation)

Here's another guide which will cover the details of dual booting

[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)

When you go to install 7.04, you will need to select the partition you want to use such as /dev/sd1b for device one and second partition.  Let us know if you have any other questions.
