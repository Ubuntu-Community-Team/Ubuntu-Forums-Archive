### Post by punx45 on 2006-10-26
Abstract: you can get by on 8-10GB for an install to just play around with. 

The longer version:
im running a dual boot with a dual hard drive setup, i think hda is windows and hdb is Ubuntu, could be the other way round.   the ubuntu disc is ~20GB with about 4GB (filled to about 75% capacity at the moment) partitioned for system files, and the rest for the /home directories.   you could easily get by with about 10GB partitioned for Ubuntu if you just want to try it out.   then if you decide later on you like it and want more space, USB drives mount easily and you can store on the external.   make sure its FAT and not NTFS though to make things easier on yourself.

p.s. if you are really concerned about bricking your Windows partition (ive done it) you can get a cheap small HDD on ebay for $10-20.
