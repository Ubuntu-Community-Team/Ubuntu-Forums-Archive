### Post by pbb on 2006-01-01
[QUOTE=macneacail]Can one run windows programs on UBUNTU?[/QUOTE]

There is a famous Windows-emulator for Linux, Wine, which is used by many people. There are also several PC-emulators, where you would run a new virtual PC in a window on your Linux desktop, in which you can install anything you can install on a new PC. A famous one is VMWare, but I understand there is also a free one.

Short answer: Yes, you can run Windows and Windows programs under Ubuntu, but they may not be able to access all your hardware and they may not (or may, if you're lucky) co-operate well with your Linux programs.
