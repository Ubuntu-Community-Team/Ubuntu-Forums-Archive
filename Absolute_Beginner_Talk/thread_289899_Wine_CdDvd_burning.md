### Post by steve.horsley on 2006-10-31
Wine is an application aimed ar allowing you to run windows executables on Linux. Of course, they think they're running on windows. Many windows programs work and the list is getting linger as wine improves, but the compatibility is not perfect and some apps can't run. I don't think wine is good enough to be able to get proper windows viruses yet, but I supppose it could happen as wine gets better. It may be able to host things like Word macro viruses already.

The easiest CD/DVD burning s/w is probably Nautilus, Ubuntu's file manager. It has a Go->CD/DVD Creator menu option if you look for it. Open a second explorer window and drag/drop the files you want to write into the creator window. Then click Burn.

Other packages I know are are Gnomebaker and k3b. 

K3b is a KDE app and will pull in a whole raft of libraries if you install that, but is in my opinion the best one if you are having difficulties - it works when all others fail. But I haven't had a failure with nautilus for the last 3 versions of Ubuntu. k3b is probably the most similar to Nero, but they are all ver similar in user interface really.
