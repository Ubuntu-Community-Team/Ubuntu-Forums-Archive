### Post by tomxyz on 2007-03-23
If you want to reinstall ubuntu then format it in fat32 or ext3. Not ntfs (that windows is always using) because you won't have write permissions afterwards. If you really want ntfs (like I did because I couldn't reformat my drive because of the data on it) there are ways around it. Let me advise you to take a look at a perfect thread on this forum by Givré. Just type ntfs in the search field and you should find it.
