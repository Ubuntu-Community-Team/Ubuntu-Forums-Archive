### Post by Dynamo_Joe on 2008-05-15
Hi All, 

What I did was install additional langauge support from the mneus (note: must have an active internet connection - for Hardy anyway). 

I then selected Chinese Traditional and Japanese, when I did it in Fiesty, I had an error message about SCIM not installed properly and was asked if I wanted it fixed - clicked on "Yes" and Ubuntu "worked its magic" and the filenames were displayed properly. 

Hardy was simpler - language options in the menus - add additional langauge support - select languages - apply - done. 

Then again, I haven't tweaked the system - just the base install and dvd playback support.
