### Post by felicity on 2008-04-21
I can only answer your third question. To embed links in the Ubuntu forums text you surround the words you want to be linked with [ URL="http://this is the web address"]this is the text you want linked [/URL ] 

(notice the spaces between the first bracket and the last bracket.. I had to use those to display it to you.. but when you use it you won't need those spaces)

Note that you can always select the text you want to be linked, then click the Insert Link button (the one that's green and blue and looks like a little world) to add a link.
