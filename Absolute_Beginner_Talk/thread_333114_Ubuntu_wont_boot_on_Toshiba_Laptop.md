### Post by quickwitt on 2007-01-25
If you are still wondering about this ...

When you boot the computer, the Toshiba splash screen comes up. Under the Toshiba logo there are several icons. Use the arrow keys to highlight the icon that looks like a CD/DVD and press enter. The computer will then boot from the CD. You can use the other icons to boot from a USB port and a network.
