### Post by catlett on 2006-07-23
I do not know of a command line tool but alacarte menu editor can do the job. Go to Applications<Accessories<Alacarte Menu Editor. A window will open with a list of the menu categories on the left. When you select a category on the left, a list of possible entries appear on the right. You can then select the ones you want by checking off the box or un-select entries by un-checking the box.
