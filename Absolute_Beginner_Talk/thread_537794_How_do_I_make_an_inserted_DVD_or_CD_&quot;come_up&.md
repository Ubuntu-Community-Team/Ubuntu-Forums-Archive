### Post by Blofeld on 2007-08-29
OK, this is probably a really basic question.
When I insert an optical disc, DVD or CD, into the drive an icon usually pops up on my desktop.
How do I make this icon appear if it doesn't come up automatically?
I'm using Xubuntu 7.04, thanks.

---

### Post by marco123 on 2007-08-29
hit Alt-F2 and type in "gconf-editor".  Then navigate to apps>nautilus>desktop  and check/uncheck "volumes visible" .
