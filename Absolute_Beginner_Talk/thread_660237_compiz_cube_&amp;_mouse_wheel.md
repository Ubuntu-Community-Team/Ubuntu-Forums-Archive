### Post by adam.tropics on 2008-01-06
Do you have 'compizconfig-settings-manager' installed? If not, install it. Then when you run that , you can change the settings for cube rotate in there. Funny it was a feature that went off the defaults for a while, everyone screamed to have it back!

For the double click close, not sure, but if you are using emerald for a window decorator, then if you find the emerald settings manager, you can certainly play around with what buttons do what on the titlebar from there.
