I maintain that, following community input on what's useful.

It's main purpose is for people with no internet or dialup, but it is also useful for people who don't know what to install to get things done. For one thing, it gives you more multimedia support, more internet applications (RSS newsreader, commonly need plugins, etc).

And Dapper is the nickname for 6.06 LTS.

There is also a deb of transmission I packaged for Ubuntu in it. It's a basic, lightweight, one-window bittorrent program similar to utorrent. This is the only deb I know of. The version of MPlayer included also has more codecs than the official one, while still not using the win32codecs which is of questionable legality.
