### Post by RomeReactor on 2007-09-03
Hi. That file belongs to root, so you need administrator privileges to edit:
```
gksudo gedit /etc/apt/sources.list
```
It will ask you for your password; enter it and you should be able to modify the file. What are you trying to add or delete from it?
