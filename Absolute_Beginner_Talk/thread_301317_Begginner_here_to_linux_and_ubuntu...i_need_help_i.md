Kopete is nice, but if you're running regular Ubuntu (not Kubuntu), it will require a lot of KDE libraries, which might slow down your system.  If you're running Kubuntu, you should already have kopete installed (it is installed by default, isn't it?).  If you do decide to install it, just get it from the repositories by using Synaptic or, in a terminal, typing

sudo apt-get install kopete
