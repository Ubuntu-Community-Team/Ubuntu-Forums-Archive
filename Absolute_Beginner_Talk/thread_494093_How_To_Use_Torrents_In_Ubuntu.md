### Post by ugm6hr on 2007-07-07
> **questpro said:**
> I would like to use ktorrent in my ubuntu 7.04 . So i downloaded it. I wonder that I can not find any option to set it as default.
To auto-download from Firefox:
Menu: Edit->Preferences
->Content
*Manage...* (File Types)
Select TORRENT, click *Change Action...*
Select *Open them with this application* - and browse to */usr/bin/ktorrent*
OK, then close all preferences windows.

Then, every time you click on a torrent file, it will automatically open KTorrent, and start downloading.

To default to KTorrent from a previously downloaded .torrent file - in Thunar (Xubuntu):
Right-click any .torrent file, select *Open with other application.*
Select *KTorrent* in list, and tick "*Use as default for this kind of file*", and then OK.

Presumably, Nautilus (in Ubuntu) has something similar.

### Post by ugm6hr on 2007-07-07
> **questpro said:**
> But I don't have the torrent file option in the file types ( Edit/preferences/Content/manage/File types)...
Try this in Preferences:
In *Main*, select *Always ask me where to save files* (in Downloads), then Close

Then click on any .torrent link on a website, e.g. [http://releases.ubuntu.com/7.04/ubuntu-7.04-server-i386.iso.torrent](http://releases.ubuntu.com/7.04/ubuntu-7.04-server-i386.iso.torrent)

This should bring up an option box.
Select *Open with*, and select *Other...* from the dropdown menu
Browse to */usr/bin/ktorrent*
Then tick the *Do this automatically for files like this from now on* and click *OK*

This should now bring up KTorrent, with a request for download location.  You should be able to just cancel (unless you want to download the torrent).

Now it should work automatically - but you can check the previous Preferences page I mentioned earlier to confirm.

PS: Don't forget to try loading the UPnP plugin for KTorrent - it works, unlike Azureus UPnP.
