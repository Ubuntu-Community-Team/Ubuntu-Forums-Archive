### Post by zanglang on 2007-07-24
I don't use SDJukebox, but I do have foobar2000 on Wine for tagging, managing my mp3 library and occasionally playing, although I use native Linux players way more often. You don't really need a Windows install just to use wine; that's if you want to setup a *virtual machine* like VMware. Wine only automatically initializes a "fake" Windows drive with most of the necessary system files when run.
