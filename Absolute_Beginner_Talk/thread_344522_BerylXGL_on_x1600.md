### Post by bwhite82 on 2007-01-23
Your card isn't that new and it is supported by both the fglrx driver and the ATI open source driver. Check out this thread for setting up fglrx:

[ Howto: ATI Mobility Beryl/XGL FGLRX in Edgy]("http://www.ubuntuforums.org/showthread.php?t=291464")

Additionally, the ATI driver page is here:

[ATI Open Source drivers]("http://ati.amd.com/support/drivers/linux/linux-firegl.html")

If you go to a console and type:

> $fglrxinfo

You should have something that looks like this:
> 
soldierboy@ubuntuLinux:~$ fglrxinfo
Xlib:  extension "XFree86-DRI" missing on display ":1.0".
display: :0.0  screen: 0
OpenGL vendor string: ATI Technologies Inc.
OpenGL renderer string: ATI Mobility Radeon X1600 Generic
OpenGL version string: 2.0.6011 (8.28.8)

If not, then it probably says, "Mesa", and you probably don't have Rendering, hence no 3D. You would then need to follow instructions on various threads on this forum to get it working. Do a search for "fglrx" in the "Howto, tips, tricks" sub-forum. Cheers.
