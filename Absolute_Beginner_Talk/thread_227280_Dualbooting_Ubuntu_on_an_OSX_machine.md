### Post by Lew on 2006-08-02
A couple of things you could try:

1: Start the Mac up while holding down the "C" key.  This *should* force it to boot from the CD.

2: Start the Mac up while holding down the "Option" key.  This should bring up a graphical list of all bootable volumes.
