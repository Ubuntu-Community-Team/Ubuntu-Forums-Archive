### Post by oodledoodle on 2008-01-21
system > preferences > session

you can add commands to run when you login. for example to run pidgin at startup click on "add" and then type "pidgin" as the name and "pidgin" as the command. it is important that you type the correct command line name of the program otherwise the system wont run i. what programs did you want to run?
