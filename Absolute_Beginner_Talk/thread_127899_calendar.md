### Post by Klejs on 2006-03-09
It is true that gdesklets have applets for calendars to put on your desktop, but I dunno if they will look like rainlendar's ones.

to install gdesklets:

**> sudo apt-get install gdesklets gdesklets-data**

if you are promted for an password just enter the one you have to your
user account. Now you can find gDesklets in the menu under

**Applications -> Accessories -> gDesklets**

Have fun and stay cool \\:D/
