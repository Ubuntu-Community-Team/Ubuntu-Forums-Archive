### Post by forestpixie on 2007-06-05
I had the same problem a while ago - I edited  my fstab to comment out extra drive and all was well. 

```
# /etc/fstab: static file system information.
#
#  -- This file has been automaticly generated by ntfs-config -- 
#
# <file system> <mount point>   <type>  <options>       <dump>  <pass>

proc /proc proc defaults 0 0
# Entry for /dev/sdb1 :
UUID=c9d650b6-eb0e-4d1f-8d83-d270a3e8af75 / ext3 defaults,errors=remount-ro 0 1
# Entry for /dev/sdb5 :
UUID=38b6b578-45be-45f5-b978-b6464bcd28df none swap sw 0 0
[COLOR="Red"]#[/COLOR]/dev/hdc /media/cdrom0 udf,iso9660 user,noauto 0 0
/dev/sda1 /media/System ntfs-3g defaults,locale=en_GB.UTF-8 0 0
/dev/sda5 /media/Media ntfs-3g defaults,locale=en_GB.UTF-8 0 0
```

There are quite a few threads lately with the same problem so I would try and search for them as well

Not sure whether I did the right thing or not but it hasn't all come crashing down on me.
