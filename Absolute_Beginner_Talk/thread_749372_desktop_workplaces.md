### Post by qamelian on 2008-04-08
Go to [http://www.microsoft.com/windowsxp/downloads/powertoys/xppowertoys.mspx](http://www.microsoft.com/windowsxp/downloads/powertoys/xppowertoys.mspx) and download the Virtual Desktop Manager. That gives you virtual workspaces in Windows XP. Its part of Microsoft's Powertoys for Windows XP. There are several other useful tools for Windows users there as well.
