Character map seems ok to me, not much different than my limited use of the Windows one. If you don't like the default grouping then click View > by Unicode Block. This will separate them out a bit more clearly for some, the first 4 are the latin based language packs. 

I just use two keyboard set ups for my different character languages (System > Preferences > Keyboard). Add two (or more different ones, and select a default on the Layouts tab) different ones and the default switcher is pushing both alt keys at once.

Language support seems fine to me. If there's a serious issue mind being specific with screenshots if needed for better illustration.
