### Post by Bartender on 2006-12-26
nick - 
Don't do like I did (on a friend's PC at that!) and start following the directions for Dapper.  If you've got Edgy make sure you follow arnieboy's directions for the the Edgy version!

Automatix does nothing that can't be done by hand.

Quick story.  Same friend's house.  He likes Opera.  We went to Opera's site, downloaded the right version for Ubuntu, installed it, it wouldn't work.  We let Automatix install Opera and it worked.  I don't know what I did wrong, but Automatix did it right.

### Post by meng on 2006-12-26
I don't use automatix, I did once but I didn't much care for it.
For many users, automatix is an "easy" way to install java, flash, acrobat, and multimedia codecs and other multimedia players (besides totem).
On the other hand, these things are relatively easy to do manually, by installing from repositories. Each item takes up to 5 commands, perhaps 5 minutes if you have a broadband internet connection, most items take 1-2 commands to enact.
