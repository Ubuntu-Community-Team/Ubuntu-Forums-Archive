### Post by MetalMusicAddict on 2007-01-08
Where different lossless formats really shine against each other is at lower bit-rates. .OGG-Vorbis is great but generally more CPU intensive. So it will drain you battery faster than say .mp3s.

If you have the space, FLAC is awesome and cant be touched as far as I'm concerned.

I still run .mp3s on my iRiver H340 with Rockbox because of the battery issue. I have the same battery as the 4th-gen ipods.
