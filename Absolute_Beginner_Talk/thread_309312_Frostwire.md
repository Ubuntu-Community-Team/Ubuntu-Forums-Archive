### Post by nixclusive on 2006-11-29
This one's specifically packages for Ubuntu/Debian:

[http://www.frostwire.com/download.php?file=http://mirror1.peercommons.net/frostwire/4.10.9/FrostWire-4.10.9-2.i586.deb](http://www.frostwire.com/download.php?file=http://mirror1.peercommons.net/frostwire/4.10.9/FrostWire-4.10.9-2.i586.deb)

```
wget -Sv http://www.frostwire.com/download.php?file=http://mirror1.peercommons.net/frostwire/4.10.9/FrostWire-4.10.9-2.i586.deb
dpkg -i FrostWire-4.10.9-2.i586.deb
```
