### Post by Qrk on 2005-12-08
The 32 bit version will work just fine on a 64 bit machine. It might not be quite as fast; but thats not the end of the world. 

So I would recommend trying both; if wine won't work on the 64 bit version, move to 32. 

But, really, you should consider dual booting. Wine is a great windows substitute, but its nowhere near the real thing. Besides, disk space is cheap.
