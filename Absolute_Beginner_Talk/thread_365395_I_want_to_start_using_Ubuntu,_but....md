### Post by crimesaucer on 2007-02-19
Just go for it, I also am a beginner with xubuntu/linux and I am really happy I switched.

My first time using it for real was this last October 2006. I also only have 55.89 GiB for hard drive space. I left 40 GiB for Windows Xp, and used the rest for a 14.00 GiB xubuntu, 2.00 GiB for FAT32, and a SWAP file of 1.00 GiB. 

You can learn the OS from just spending time reading other people's posts and asking questions when you run in to things you don't know how to do.

Wifi might take some effort, but it's a good way to learn about doing other things in Linux, so if you really want to learn a new OS, then consider the things that seem like an extra effort, like installing Flash/Java or the correct driver for your wireless card, a learning lesson on getting to know your new OS and how to preform tasks using Terminal.

This forum is very helpful for beginners. Most things only take a few lines of Terminal code, and most of the effort is asking the correct questions so someone can just give you a simple answer that works for you.

I have been on xubuntu now for about 4 months now, and I don't even use my Windows partition for anything other than a few things that mostly can be done in wine. I keep my Windows Xp as a backup partition just in case I ever do something to my xubuntu and need a working OS to get into a forum to figure out how to fix it. When the Feisty xubuntu comes out as a 7.04RC, I will probably do a clean install and resize my Xp partition to about 30 GiB to use as my backup. Plus, you can access anything you have in your Windows folders through your /media/hda1 folder.

### Post by floke on 2007-02-19
Sounds like you've already hit the worst case scenario - a complete wipe of your hard drive. Did it kill you? No. So there you go, you survived the worst. Now what can go wrong with installing and using ubuntu that's worse than that? 

Get the live CD and have a look at it. Be warned, though, that they don't work on all PC's (a friend of mine has a PC that won't run the ubuntu CD but will run xubuntu), so don't be discouraged if this happens to you. Ask around on the forum for help if you need it. Make sure you select the 'check disc' option from the menu first to make sure you burned it ok.

It took me a while to suss out how to get my wireless set up and my card is supported fully. Ask around here if you need assistance and we'll see what we can do.

If you don't try you'll never know. The fact that you're here shows that you want to find out.
