### Post by Inxsible on 2008-04-15
Ubuntu has a minimum memory requirement of 256 MB, but I would say 512 is generally preferred, if you want things to get done fast.

KDE is just a touch higher. xfce as mentioned earlier is lighter. Xubuntu uses xfce. There are other distros as well which cater to older machines with less memory and hard drive space. DSL, Puppy and Zenwalk are a few of them.

Bottom line, you really need to know which distro will run best in your environment and for that you need to know some basic details about your computer, like memory and hard drive space.
