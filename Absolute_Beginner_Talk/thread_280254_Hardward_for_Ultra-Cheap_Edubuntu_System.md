### Post by Hairyred on 2006-10-19
What about a Sempron 2800 AM2 64 bit cpu in an all in one mobo like an Asus M2V-TVM, add 512m RAM, cheap case and PSU (not too cheap), 80 gig hd and a dvd burner, keyboard and mouse and you're away! I can't help with shops, I'm in AUS. Personally I would go with a slightly more expensive nVidia chipset board because I know they are well supported. I have no idea of the compatability with the Via chisets, you will have to check.

Going with the AM2 socket will improve later upgrade possibilities, and give the system a longer life. There are always used pc's, I had an AMD xp 2500 which worked very well, but it is not possible to upgrade such a machine with new parts.

If I were doing it? Sempron 2800 AM2, Asus M2NPV-MX all in one mainboard, 2x256m generic RAM, WD or Seagate 80g HD (SATA), Pioneer DVR 111 Burner, Aopen ES50F 350W or similar case and psu, generic keyboard and optical mouse.

Just my 2 penneth worth.
