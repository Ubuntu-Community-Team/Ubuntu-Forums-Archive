### Post by jken146 on 2007-12-01
That depends.  See if there are any instructions from the people who provided the download.

Basically you'll have to extract the contents of the archive, then either run an install script (having made it executable first) or, if there is no install script, compile it from source and install it (more steps).  Or the tarball may include a .rpm or .deb package if you're lucky - these will be installed for you following a double click.

Post more details if you want a better answer.
