### Post by p_quarles on 2008-01-03
The source code is the same (mostly), but they're compiled very differently. Linux Firefox versions are compiled for broad sets of architectures (i386, x86-64, Sparc, etc.) whereas Swiftweasel is compiled for more specific sets of CPUs. 

That, plus some of the source code for Iceweasel/Swiftweasel has been optimized specifically for Linux, whereas the Firefox source was initially written with Windows in mind.
