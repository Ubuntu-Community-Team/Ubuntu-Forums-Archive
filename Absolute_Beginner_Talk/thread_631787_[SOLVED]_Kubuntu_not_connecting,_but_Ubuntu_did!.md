### Post by Xzallion on 2007-12-04
Okay doke, since you did a offline install it didn't configure and activate your network interface (I think this is why you can't connect to the internet).

Assuming this is, click K Menu>System settings>Network Settings and log into administrator mode.

Set you interface up and make sure to click the checkbox that says to activate it on startup and that should fix it.  If it doesn't, reply back and hopefully someone can get this sorted out for you.
