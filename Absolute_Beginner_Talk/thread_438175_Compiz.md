### Post by kebes on 2007-05-09
Check out this thread for more information on Beryl and Compiz:
[http://ubuntuforums.org/showthread.php?t=349732](http://ubuntuforums.org/showthread.php?t=349732)

Yes, you can easily install it at a later date. Both Compiz and Beryl are "in the repositories" for the recent versions of Ubuntu, so installing them is as simple as picking them from the list and clicking "install." There is no advantage to installing it now if you are not going to use it right away.

As you say, it's a good idea to get used to Linux a bit before trying something like Compiz/Beryl. If you ever decide to try it, there are lots of community documents explaining how to install it.

Good luck!

Assuming everything went fine with your Ubuntu install, a barebones version of compiz is installed (System > Preferences > Desktop Effects). It is deactivated by default. Unless you actively turn it on, it will continue to be off. So yes, you can turn it on at any time. You would need [nvidia]("http://www.albertomilone.com/nvidia_scripts1.html") drivers before you turn it on, you can install them with envy. Don't turn it on until you have drivers in, else... you'll have problems >.>

Just download and doubleclick on the 0.9.2 debian. After you install, go to Applications > System Tools > envy. Click automatic install of Nvidia driver and then answer yes to any prompt, then when you reboot you will have drivers.
