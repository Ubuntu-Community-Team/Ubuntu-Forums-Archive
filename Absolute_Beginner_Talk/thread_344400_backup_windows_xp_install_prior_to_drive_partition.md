### Post by freebeer on 2007-01-22
I know people who use Ghost all the time with great success.  I've only used it once, and it went very well.

After you do your backup, it's recommended that you defrag fully, as well - smooshes all the data to one end, if you catch my drift, and makes for a better resizing of the partitions.  

Good luck!
