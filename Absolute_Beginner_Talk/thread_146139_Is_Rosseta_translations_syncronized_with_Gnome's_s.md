### Post by TLE on 2006-03-17
Well we don't, prevent double work, not with the current system anyway. First of all not many who want to translate GNOME do it through Rosetta because only Ubuntu users will benefit from it, whereas if you translate on the GNOME cvs server all destributions benefit. There has been a lot of talk on this subject on the Rosetta(Ubuntu) mailing list. I do not know what the solution will be, but I hope that it will be something like locking the GNOME translations in Rosetta until the local GNOME cvs translation teams have had a chance to translate them and the import the in Rosetta and let the Rosetta commuty translate the rest if there is something the local teams haven't finished. But that's just wishfull  thinking, no such arrangement has been made. So as of now I would encurage translating in GNOME cvs and then we will have to persuade the Rosetta folks to import them in their completeness.
