### Post by HermanAB on 2007-10-05
Well, as usual there is more than one way to do it.  You can use MS Windows networking with Linux, known as Samba.  It works very well but it suffers from optionitis and is easy to get wrong.   Samba wizards do exist, but generally do not cut it, since the problem is too complicated for a wizard system.  You typically have to read a book even to use the wizard.

An alternative is NFS. NFS is extremely simple and easy to install.  You only need about one line to configure it.  NFS wizards do exist but generally make it more difficult since how do you simplify one line?.  Some googling will get you on your way and there are NFS drivers for Windows too.

I suggest that you go and see what you can learn about NFS and in the rather unlikely event that you have a problem, post another question.

H.
