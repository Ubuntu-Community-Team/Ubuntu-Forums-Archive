Thanks.

I would suggest that he try fixing that windows install first.  If this is an XP installation, he could use his install CD to perform a "repair" installation.  Press "R" when instructed during the initial install steps, and run the FIXMBR command.

That, hopefully, will restore his access to the drive.  Then you can proceed to back up his files and move on with Ubuntu from there.

If he has any time/money invested on the Windows side, he probably is going to need to regain access at some point.  Might as well do it now so you reduce the danger of losing his files.

Good luck.
