### Post by coffeecat on 2007-12-16
You can't get quicktime for Linux, but the easiest way to play .mov files is to install mplayer. Have a look for mplayer and the mplayer plugin in Synaptic. The plugin is for Firefox. I can't remember the exact name of the package - I'm using my Mac atm so I can't check - but it's some variation on mplayerplug-in or mplayer-plugin.

I'm fairly sure you don't need the medibuntu repository for this, but I could be wrong.

**Edit:** I've also had .mov files playing in totem before now. You just need the right codecs - have a look in Synaptic.
