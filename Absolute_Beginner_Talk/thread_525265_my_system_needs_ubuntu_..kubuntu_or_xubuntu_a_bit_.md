### Post by chuckyp on 2007-08-14
All of them will work with your specifications; however, if you are looking for the best performance you may want to check out Xubuntu.  Keep in mind the only difference is just the GUI.  And you can always switch or install all three.  Like if you downloaded xubuntu cd and installed it and you wanted to check out gnome (ubuntu).  You can just install the ubuntu-desktop package and your off.  Or install the kubuntu-desktop package and you'll have KDE.

The better route may even be to create a seperate /home partition so that you can reinstall and not lose any of your data.

### Post by ugm6hr on 2007-08-14
I personally prefer Xubuntu, but if you are just starting out, I would actually suggest Ubuntu.

The reason for this is that finding solutions for problems tends to be more straighforward, since it is the most popular of the 3 official versions.  So all the advice in these forums apply.

While the advice can be *adapted* for Xubuntu / Kubuntu, you obviously have to know what the differences are in order to do this.  Hence - easiest to start with Ubuntu.

After you get used to Ubuntu, it is easy to try the other "flavours":
