### Post by sigma2805 on 2005-11-26
Hello,
Try this,
go to System->Preferences->Sessions and then in the session options tab delete any sessions that are in there. That will delete your GDM setup and then you can start anew. Next time you shut down, make sure the save current setup box isn't checked.
