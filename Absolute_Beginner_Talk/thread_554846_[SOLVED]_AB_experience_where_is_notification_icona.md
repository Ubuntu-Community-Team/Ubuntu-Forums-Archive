Lets work out backwards. 

1) AMD64 = Things to install for 64 bit computing. Includes all AMD 64 bit processors and Intel 64 bit processors like P4 HT, COre 2 duo's ,etc.
    Intel x86 = Things to install on all availible computers. Uses 32 bit. Includes all processors after 486.
If your hardware supports 64 bit computing, go for AMD464. x86 will also work anyway, but 64 bit computing is faster.

2) Slackware is basically made that way itself. If you needed GUI, you should have tried Debian, REd Hat, etc.

3) You should see a Orange colored icon. Notification area is similar to NOtification area of Windows. It docks icon of applications like Amarok, utorrent, etc when they are minimised.
If for some strange reason that Orange square icon is not visible near the clock, use System --> Administration --> Update Manager.
