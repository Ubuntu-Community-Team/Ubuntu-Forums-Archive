### Post by SavageBeginner on 2006-02-25
To have it start when you login:
Click System -> Preferences -> Sessions 
Then click the 'Startup Programs' tab and the 'Add' button

For the Desktop:
Right-click on the desktop and select 'Create Launcher'
Type in the info.
