### Post by CraymelCage on 2008-05-17
You can edit mednafen through it's config file. You can find mednafens config file in:
```
/home/"username"/.mednafen
``` 
This folder is hidden though so press control H when you're in: 
```
/home/"username"
```.
In this link you can find all the arguments you can use to edit mednafen.
```
file:///usr/share/doc/mednafen/mednafen.html (copy and paste this in your address bar)
```
Look under the subtitle ***Command line***. I hope this helps.
