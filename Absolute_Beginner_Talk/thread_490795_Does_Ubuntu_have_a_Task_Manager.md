### Post by Rocket2DMn on 2007-07-02
the System Monitor can be reached from the Administration menu or by running the command ```
gnome-system-monitor
```  You can even make a launcher for that if you want.

---

### Post by Krosis on 2007-07-02
You can also right click on one of your panels (top or bottom), select "Add to Panel" the scroll down and select System Monitor.  This way, it will always be there for you to view.
