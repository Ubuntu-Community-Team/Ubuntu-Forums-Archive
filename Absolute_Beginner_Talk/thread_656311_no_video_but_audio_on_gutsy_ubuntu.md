### Post by Ub1476 on 2008-01-02
Try installing ubuntu-restricted-extras

```
sudo apt-get install ubuntu-restricted-extras
```

eventuelly reinstalling:

```
sudo apt-get install ubuntu-restricted-extras --reinstall

sudo apt-get install totem --reinstall
```

VLC is also a very nice media player:

```
sudo apt-get install vlc
```
