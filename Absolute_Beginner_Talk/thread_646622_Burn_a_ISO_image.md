### Post by coffeecat on 2007-12-21
> **X40nick said:**
> Or do I need to use a program? So far the one I like is Brasero.

Yes you can use the terminal, but Brasero is an excellent GUI program, and easy to use. Another popular one is K3b (have a look in Synaptic), but if you're using Gnome as your desktop and you install this, it will drag a load of KDE dependencies in with it.

**Edit**, the previous poster has given you a link for doing this in Windows, but I've assumed you want to use Ubuntu. But that link reminds me that you should do an md5sum check before burning the ISO. [Here's a useful link](https://help.ubuntu.com/community/HowToMD5SUM).
