### Post by Garyu on 2007-06-15
Your microsoft license is good for one computer. As long as you've only got it installed on one computer, I can't imagine that it would be a problem to activate it. If you are installing windows on a second computer without paying a license fee you are a software pirate. It would be wiser to use Wine for your windows applications in that case.
