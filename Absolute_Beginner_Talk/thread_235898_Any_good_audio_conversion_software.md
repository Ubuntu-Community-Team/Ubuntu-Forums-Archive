### Post by anasofiapaixao on 2006-08-15
The package is named "soundkonverter", just in case you don't find it because of the c/k thing. PLEASE NOTE that for some reason you have to specify all your conversion options and THEN add the files to convert; for some reason if you add files and change the definitions later, it will use the early ones!

And FLAC is like WAV; as it is lossless, it creates HUGE files... so beware!
