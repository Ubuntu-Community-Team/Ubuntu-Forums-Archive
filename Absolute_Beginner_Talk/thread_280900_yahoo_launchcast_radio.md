### Post by podunk on 2006-10-20
You will need to install something like wine which will run windows programs to get netscape 6 to work.  Wine can be sort of hit and miss, I can't help you with that - I've never been sure what I do to make things work there, or why one thing works and another doesn't. :-)

If it's Internet radio you want – you might try streamtuner to connect to staions and xmms to play them. Both are available in the package manager.
System
Administration
Synaptic Package Manager
click search, enter the program name.
Right click the program, chose “Mark for installation
Click the “Apply” button at the top.
