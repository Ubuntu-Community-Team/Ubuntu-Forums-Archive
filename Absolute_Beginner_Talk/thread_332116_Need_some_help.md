> **GILBERTO15 said:**
> every cd I try I have to do some sudo root thing which is complicated for me, i need to find a cd of linux that is easy to install such as windows

I haven't yet found a Linux distribution that does not require the user to enter 'sudo', 'su', or something of the sort to gain administrative priviledges. This is one of the security measures in place to keep web sites, hackers, or simple programs from installing things or altering your system without your express permission. In the long run, is not typing a few extra letters a good tradeoff in regards to security?
