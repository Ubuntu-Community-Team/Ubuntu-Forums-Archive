### Post by Raineer on 2007-07-01
Sounds like a Gnome bug I've had strike me a few times.  When you get to your login prompt, select options and it will say something like "Choose Session".  This is how you select between Gnome, KDE, and others.

If you have KDE also installed, try that and then you can logout and switch back to Gnome. If not then try the "fail-safe" selection one time.
