### Post by zekonology on 2006-11-22
Rar (.rar)

Rar is a non-free archive format created by Rarsoft. There are several versions of rar and the newest version, 3.0, is not supported by any free tools.

To add support for it, install the package from the multiverse repository ([AddingRepositoriesHowto]("https://help.ubuntu.com/community/AddingRepositoriesHowto")).
Ubuntu 6.06 (Dapper Drake)

    *

        sudo apt-get install unrar
