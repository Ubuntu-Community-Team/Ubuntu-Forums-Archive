### Post by stump138 on 2008-01-25
your openoffice.org user profile is a hidden folder in your home directory.  You can see it if you look at hidden folders you will see it.  location is /home/user_name/.openoffice.org2/user

It has a similar file structure to the one located at c:\documents and settings\user name\application data\openoffice.org2\user

you could certainly try to copy over the stuff from your windows install, but I don't know how it would react :) gl!
