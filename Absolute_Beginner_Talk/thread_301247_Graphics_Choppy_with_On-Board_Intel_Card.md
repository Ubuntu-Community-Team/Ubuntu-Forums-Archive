### Post by 56phil on 2006-11-16
I had the same problem. You need to install a proprietary video driver. If your graphics card is from ATI try this:
```

sudo apt-get install xorg-driver-fglrx
sudo aticonfig --initial
sudo aticonfig --overlay-type=Xv

```

The proprietary drivers are available on Intel's website: [http://www.intel.com/](http://www.intel.com/) Just navigate to support and downloads > download center > graphics > desktop graphic controllers > [your graphics card].
