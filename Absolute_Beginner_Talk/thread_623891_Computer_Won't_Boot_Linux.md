### Post by candtalan on 2007-11-26
On older (low ram and slow too) machines things can take a looong time. There is also a chance that the CD drive is not reading the CD perfectly, either from wear or some dirt or something. Using the CD boot menu item 'check CD' will help to verify that even if the CD is perfect, then the drive will read it fully - might be worth considering?

And - using the text installer (non-live CD) will give th ebest chance of success
