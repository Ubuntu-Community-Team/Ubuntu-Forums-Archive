### Post by frodon on 2005-07-03
you can install icon themes from gnome-look.org : [http://www.gnome-look.org/index.php?xcontentmode=120&PHPSESSID=fdc68144b301c3e77a7b6784a2e13279](http://www.gnome-look.org/index.php?xcontentmode=120&PHPSESSID=fdc68144b301c3e77a7b6784a2e13279) and install it using drap and drop in the theme window. If you want to get the icons of the theme go to  /home/your_user_name/.icons/ they are installed here.

### Post by ceti on 2005-07-03
[QUOTE=frodon]If you want to get the icons of the theme go to  /home/your_user_name/.icons/ they are installed here.[/QUOTE]

...or /home/your_user_name/.local/share/apps. Please note the dot "." before *local *and* icons*. That's because they're hidden folders. To access them, first you have to click on View/Show Hidden Files in Nautilus.

Cheers (again)
