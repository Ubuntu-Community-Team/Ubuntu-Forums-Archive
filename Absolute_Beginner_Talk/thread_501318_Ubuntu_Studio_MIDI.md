### Post by luctor on 2007-07-15
Midi / Soundcard drivers are available through alsa, which is installed by default on ubuntu (or any other linux as far as I know)
It seems not all your hardware is listed on [http://bugtrack.alsa-project.org/main/index.php/Matrix:Vendor-MAudio](http://bugtrack.alsa-project.org/main/index.php/Matrix:Vendor-MAudio)
The midisport is listed, but not the firewire 410 , however there seem to be firewire drivers for audio, freebob, now called fffado
[http://freebob.sourceforge.net/index.php/List_of_Supported_Devices#Devices_with_a_DM1000_but_no_BeBoB_firmware](http://freebob.sourceforge.net/index.php/List_of_Supported_Devices#Devices_with_a_DM1000_but_no_BeBoB_firmware)

I've never used or installed firewire stuff, so I can't help you with that.
