### Post by wormser on 2007-07-31
Did you read this [post]("http://ubuntuforums.org/showpost.php?p=3109000&postcount=2")?

Applications> Accessories> Terminal

```
df -l
```

Look for the drive with the **[COLOR="Red"]/[/COLOR]** (slash) in the results.  It will look something like this:

/dev/sda3 28834744 4973344 22396676 19% [COLOR="Red"]**/**[/COLOR]

In this case /dev/sda3 is the root device.
