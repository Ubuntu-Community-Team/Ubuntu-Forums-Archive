### Post by ironfistchamp on 2006-08-13
Hey I have come to offer my opinion and that is all it is.

I have a 64bit Chip but run 32bit dapper. Every so often I try to use 64bit Ubuntu dapper. It is possible to run either on a 64bit chip although some people say the 32bit on a 64bit chip is unstable. Personally I have seen no evidence of this in my case. The things you mentioned can be run on 64bit dapper but it can take a bit of doing. 

Personally I would buy the 64bit chip and have the 32bit dapper. If you have your home dir on a different partition (there are some guides/info around the place) then you can easily change to 64bit dapper when it becomes easier. 

We may well get people on here saying that 64bit is just as easy as 32bit but I have found that it takes more time to get things working. There are little work arounds needed in some areas that I just don't have the time for. 

To sum it up I say get the 64bit chip but stick with 32bit dapper for the time being. 

Hope this was some help.

Ironfistchamp

### Post by KoRnholio on 2006-08-13
I too run 32-bit Dapper on a 64-bit processor, and I have had no problem.

The 32-bit OS is more stable than the 64-bit OS right now, but eventually support for 64-bit will come up to speed.  So for a new computer I'd recommend getting the 64-bit processor.
