### Post by Henry Rayker on 2006-12-06
Invariably, someone will come in and say that you do need to worry about viruses and the like, as a code of conduct.

Personally, I would only use an antivirus software if you are connected to windows machines (you might be a carrier, but stay uninfected).

Ubuntu (and Linux in general) is very security minded in the first place. There are viruses for Linux in existence, but a lot of them are proof of concept and very few are in the wild.
