### Post by aysiu on 2006-05-20
A few things:

1. CDs you download need to be burnt as disk images, not data.
2. Your BIOS has to be set to boot from CD first, not hard drive.
3. Live CDs run off the CDs themselves and your computer's RAM--they don't affect your hard drive, so they're great for test-driving Linux distributions.

For help on #1 and #2, go here: [http://www.psychocats.net/ubuntu/windowstoubuntu.html](http://www.psychocats.net/ubuntu/windowstoubuntu.html)
