### Post by LLRNR on 2006-11-14
One of many: [http://www.linuxcommand.org/learning_the_shell.php](http://www.linuxcommand.org/learning_the_shell.php)

Also, while in a Terminal (Applications -> Accessories -> Terminal in Gnome), you can try 
```
command --help | more
```for most commands, for seeing a brief list of options and other stuff, or the long explanation by typing
```
man command
```

Note: you can exit either of these displays by pressing '**q**' on your keyboard.

HTH,

LLRNR
