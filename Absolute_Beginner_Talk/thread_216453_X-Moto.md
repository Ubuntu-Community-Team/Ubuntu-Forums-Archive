### Post by 3rdalbum on 2006-07-16
Just a note: Deleting and reinstalling a program doesn't generally fix problems with the settings, as the settings are usually kept in a hidden folder inside your home folder.

I messed up my resolution settings for that game too, so I deleted the directory ~/.xmoto. When I ran the game again, it generated a new folder containing the default settings. So that's the way to start afresh if you get stuck.
