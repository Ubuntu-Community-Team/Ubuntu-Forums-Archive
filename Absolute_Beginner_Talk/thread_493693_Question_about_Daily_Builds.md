### Post by NeoLithium on 2007-07-06
A daily build is the most up to date build that there is; essentially, you're getting the new release with the updates already added on.  Keeping it on, you'll get the subsequent updates that come out as time goes along :)  If you install 7.10 now, and keep with the updates, by the time the full release is out; you'll basically have the officail release as you've fully updated up to that point.
