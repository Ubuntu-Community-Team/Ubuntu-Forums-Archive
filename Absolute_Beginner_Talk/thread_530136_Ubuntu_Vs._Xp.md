### Post by lyceum on 2007-08-20
The good:
You can make Ubuntu look how ever you want. There is gdesklets, which has a Mac like doc station. I just make the bottom bar larger and put my icons on it. Compiz Fusion has really cool 3D effects, if your graphics card can handle it. Check out 

[http://gnome-look.org/](http://gnome-look.org/)

for some of the things you can do. There is a Mac icon set and windows so you can look just like like Mac. You will also have thousands of free programs to replace almost anything you use in Windows. 

The bad:
If you are looking for Windows, stick with windows. You can make Ubuntu LOOK like Windows, but it never will be. You can use a program called Wine to get some Windows programs running, but not all of them.

The ugly:
Just think of Ubuntu as what it is, a new OS. If you went out and bought a Mac would you expect everything you had for Windows to work? It is the same for Ubuntu. As for why the switch, everyone has their reasons. I just like Ubuntu better. Not must more to say than that.

Good luck!
