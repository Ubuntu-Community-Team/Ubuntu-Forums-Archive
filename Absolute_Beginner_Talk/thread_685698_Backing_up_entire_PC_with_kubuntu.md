### Post by little_penguin on 2008-02-02
I'm wondering whether it's a permissions problem? Are you logged in as root (i.e. sudo before the command) when you issue the tar command (not as a normal user)?

Another way to backup your whole Kubuntu partition is to use partimage, but booting from a CD like SystemRescueCD or Gparted CD. There are some great tutorials on how to use it. It creates an image of your partition and is pretty easy to use.
