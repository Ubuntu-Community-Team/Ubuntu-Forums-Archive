### Post by rsambuca on 2007-10-04
Yup.   Just make sure you **back-up your data first**, and then either download and install the Gutsy beta over your existing ubuntu partition, or just upgrade via the command line:

sudo update-manager -c -d

### Post by drsmaw on 2007-10-04
Please, unless you don't care about your current data, don't install a beta over a stable distro.  You never know what you're going to encounter. Remember Beta versions are for those who want to test and mess around with a distro that is not stable yet, these people can usually recover their mistakes a lot easier.  

At lease create a partition and install it there, you'll be a lot safer.
