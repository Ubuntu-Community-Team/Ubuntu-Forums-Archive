### Post by ajgreeny on 2007-09-30
Hidden files or folders are those that begin with a . (period, full stop, whatever you want to call it) and will be visible to you if you set the view in whichever file manager you use to show them.  With nautilus it's Ctrl+h, in konqueror it's in the view menu, but in xfce I haven't a clue as I've never seen it in action or used it.  It makes no difference which desktop you use, all hidden files etc will be shown if you set the system to show them.
