### Post by Chris S. on 2007-07-28
You'll need a bit more than just forwarding ports through your router (assuming you are behind a router).  Ubuntu server is all you'll need to set up your own webserver.  You can do the same thing with Ubuntu desktop after making some additions (they're just different configurations of the same OS as I understand it).  You can also look up Ubuntu LAMP Server for more information.
