### Post by Sef on 2007-01-08
It should work perfectly says [Linuxprinting]("http://linuxprinting.org/show_printer.cgi?recnum=HP-LaserJet_6P").

System > Administration > Printing.   Click on foward and not install.  That is not well set up.  Foward will install the drivers, and it should work afterwards.
