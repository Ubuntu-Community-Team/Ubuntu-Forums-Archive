### Post by Raineer on 2007-07-25
You want to extract the .rar file on your HDD.  It should contain a single .ISO file.  When you load Nero, tell it to "Burn Disk Image" or "Burn ISO Image".  After you burn it, just browse to it with Windows, if you see a single .ISO file on the CD then it didn't burn correctly, you should see a bunch of directories and files on your CD as an .ISO is like a "picture" of a CD.

And yes, that ISO will create an entire CD which is an entire *bootable* operating system. The Ubuntu authors created it that way so you can test-drive without putting a single byte on your hard drive (I have used it occasionally on systems with no hard drive or storage at all, for diagnostic reasons or web browsing)
