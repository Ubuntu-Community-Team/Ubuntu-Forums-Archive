### Post by indytim on 2007-03-02
The preferred method for editing your partition is to use GParted.  There is a Live verson (you download the iso file and burn to a CD which will be "bootable"), or you can access the program from the Ubuntu (or Kubuntu) Live CD.

GParted can be found at:

[http://gparted.sourceforge.net/index.php]("http://gparted.sourceforge.net/index.php")

Hope this helps,
