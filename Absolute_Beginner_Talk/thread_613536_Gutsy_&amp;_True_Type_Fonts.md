### Post by pissedoffdude on 2007-11-15

> How do I go about installing a .ttf (True-Type Font) file on Gutsy? I know it's a very simple proceedure on Windows, please don't break my heart and tell me it's not on Ubuntu.

System>Preferences>Appearance>Fonts>Details>Go to font folder then copy and paste any fonts that you would like to install into that directory.
