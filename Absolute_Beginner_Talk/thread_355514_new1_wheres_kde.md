### Post by kebes on 2007-02-07
The package is called "kubuntu-desktop." You can install by by using the Synaptic package manager, by going:
Applications > Add/Remove

Or you can do this on the commandline, by typing:
sudo aptitude install kubuntu-desktop



After installation, you will be offered a choice, at login time, between Gnome and KDE. Try them out and see which you prefer.

Good luck, and welcome to Ubuntu!

### Post by kebes on 2007-02-07
"kubuntu" is just Ubuntu with KDE added on top. When you install "kubuntu-desktop" you are installing the KDE desktop, plus the "most important" KDE-based applications (Amarok for music, Kate for opening text files, etc.).

It's best to install kubuntu-desktop (instead of trying to install the sub-packages yourself), unless you really know what you're doing. You can always uninstall it, or pieces of it, if you don't like it.


Btw, see my reply to your other post if programs do not appear in your package list.
