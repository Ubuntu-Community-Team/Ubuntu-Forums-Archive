### Post by phossal on 2007-03-11
Strangely, I noticed a similar problem when I set certain options during the configuration screens. For example, if I chose a different locale *before* the install (ie, the US), the installer would hang on my AMD64. For all *non-essential* configuration options, leave them as defaults until it's installed.
