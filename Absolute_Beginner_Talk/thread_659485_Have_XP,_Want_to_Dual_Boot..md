### Post by eolson on 2008-01-05
first off back up anything really important just in case.  Then degrag XP.

XP and Ubuntu dual boot,  one way to do it from the live disk.

Set bios to boot from cd drive
insert live cd
boot the system
Select "start or install
   This is going to take awhile to get to the next step a lot of cd drive action going on
Double clike the install icon
Choose your language
Select your time zone
Pick your keyboard layout
Select guided resize (if unsure of size, take the default)
Select continue from the information block that pops up
"migrating docutments" take the default
Set your user name and pass word, etc.  (don't forget them)
If everything looks o.k. on the ready to install screen select "install"
Reboot now
Remove CD from tray and hit enter
long on and install restricted drivers and updates (icons in upper right corner)
your done.

It's going to take awhile depending on your system, maybe half hour or so.


There are other ways,  but this one has always worked for me.
