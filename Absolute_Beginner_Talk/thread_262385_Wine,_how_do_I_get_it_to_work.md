### Post by JNowka on 2006-09-21
Depends on the program.  There are some programs that just won't work, then there are others that will work.  If you want a guide to tell you what will work, you can goto [http://appdb.winehq.com/appbrowse.php](http://appdb.winehq.com/appbrowse.php) and that will tell you what works and doesn't work.  If it is not listed then you can only hope it works.  Sometimes you have to do other installs or a change in the configuration using the command 'winecfg' in the terminal in order to get it to work.  There are a few Howto guides on the web describing how to get something to work.  There are even Howto guides here on the forums under the Howto section.  And if there is a game you just want to play that is native to windows I would suggest searching/asking in the gaming forum they will usually be able to tell you if it can be ran, and how to do it.
