### Post by Henry Rayker on 2006-12-11
First, welcome. That being said, let's get to business.

> **Shinzou said:**
> 
In what way is ubuntu better than windows, how can it help me out and make my life a cake walk? I know this is a silly question that can be answered by looking at wikipedia or user reviews, but I'd like to get your input on it.

Ubuntu (and Linux in general) is better, primarily for the following. 1) security. By default, a Linux machine is far more secure than the average windows computer. In fact, it is more secure than most Windows computers (unless you're a network admin or your computer is COMPLETELY locked down, it will probably be more secure than your PC). 2) There are very few viruses. I don't recall where I saw the numbers, but there are apparently something like 40,000 (or 60,000 or some god-awfully high number) of viruses that target PCs. There were about 40 at the same time that target Linux OSs. Almost every single one of these must be installed by the user through a series of commands that you'd be a fool to run on unknown code.

That being said, it won't exactly be a cake-walk. There will be CLI (command line interface) and configuration text files. Not everything is GUIed. Once set up, you probably won't have to deal with these much at all. 

> Because I want to play the latest computer games, such as F.E.A.R, Prey, and Neverwinter Nights, will I be able to play them via WINE, or is it possible to some way be able to play them via Ubuntu but through 3 partitions, an NTFS, A Shared Folder (Fat32) and a Ubuntu Partition?

To play the latest games, WINE will be hit or miss. It would be best to keep around a Windows partition and dual boot, if you need these games. It will be similar to what you said (NTFS, a Fat32 and an Ext3 [for Ubuntu]) but you will have 2 OSes and a screen at the start of the computer to decide between OSes.

> I'm going to get an amd fx2 4600+ dual core processor (2.2ghz I think), 2 gb of ram, probably around 240-340 gb. Now, if I'm not mistaken that an fx2 is dual core (or 2 processors, the jargon is confusing <.<), will it be supported by ubuntu, as well as the rest of my hardware in a way that I can get good performance, if not the same as Windows, then better?

This will sort of depend on the device manufacturer. The processor should be just fine. I haven't heard of any real problems with processors(outside of processor scaling and the like, which is usually solved quickly...for the record, the AMD dual-cores are true dual cores and not just 2 single cores hacked together). However, if you have a wireless connection, some mp3 players have trouble and things like that. The more devices you connect to the machine, the more trouble you may encounter. If there are open source drivers for the devices, though, you should be just fine.

> And the final question. I am a pretty fast learner, but overall, how easy is it to take up the controls of the OS. I imagined Linux, at first, it would be like ms-dos <.< Though I was mistaken. GNOME looks stunning, sometimes the "desktop themes" look even better than Windows...the interface has that kind of new age vibe. So, will there be weeks of studying under a waterfall? For the average tasks (after the OS is installed and configured) Ubuntu is VERY simple to use. My girlfriend needed to get some work done while I was away and just used my laptop (running Ubuntu) and had no trouble...she did everything she needed to (instant messaging, office software, browsing) without any "training" or instruction. The tweaking and all that is where the difficulty may come in.
