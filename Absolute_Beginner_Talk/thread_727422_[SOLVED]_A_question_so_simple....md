### Post by Tatty on 2008-03-17
It sounds like you are reffering to updates which the manufacturer of your machine is sending you via some third-party system.

If you are dual booting and still have windows on your machine then it will continue to automatically update whenever you are logged in to windows.  As far as windows is concerned all that has happened is that the hard disk appears to have got smaller.

If you are not using windows then there is no need to worry as you will not need the updates your manufacturer is sending to you - as these are updates for windows.

Ubuntu will let you know when there are updates available... and will give you the option of installing them.
