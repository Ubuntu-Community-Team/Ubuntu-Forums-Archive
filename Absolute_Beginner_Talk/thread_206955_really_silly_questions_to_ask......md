### Post by Arisna on 2006-06-30
[QUOTE=leebrendalee]One of my questions was what was Dapper and I saw mention of some other names in regards to Ubuntu?  Are there different verisions of Ubuntu or something? Like What's Dapper compared to????[/QUOTE]

"Dapper Drake" is the code name for version 6.06 of Ubuntu.  Previous releases were code named "Breezy Badger," "Hoary Hedgehog," and other alliterative adjective-animal combinations.

There are some different flavors of Ubuntu, though.  There is Kubuntu, which uses KDE rather than GNOME for a desktop, Xubuntu, which uses XFCE, and some others.  Further, the Dapper release has different CD's for Desktop and Server installations.  The "Dapper" name just refers to the release number, though.

[QUOTE=leebrendalee]How does basic installation of usb hardware work?  When I plug in a usb device is it suppose to say.."found new hardware"? (I've yet to get my ipaq pocket pc us device installed), I haven't tried my usb printer yet......I'm just curious how the hardware usb works...??[/QUOTE]

You probably won't receive a popup message about the hardware being detected.  However, the hardware may be mounted automatically and be accessible through an icon on your desktop.  If not, you may still be able to view it by navigating to the "Computer" entry under the "Places" menu at the top of the screen.  The printer, by default, will not cause any automatic installation screens to appear.  You will be able to add it through virtually any "Print" dialog or through "Printing" under the "Administration" portion of the "System" menu.

Ultimately, the sure-fire way to determine how your system has reacted to a new piece of hardware is to open a Terminal (Applications > Accessories > Terminal) and enter the command "dmesg" (without the quotes).  There will be some sort of message printed there about the new hardware.  This can simplify troubleshooting greatly.

[QUOTE=leebrendalee]What's the difference between Firefox and Ephipany browsers???????[/QUOTE]

Firefox has a larger feature set because there are many extensions available for it that add functionality.  Epiphany has some such add-ons available, but not as many because Firefox is much more popular.  Epiphany is generally considered more light-weight.

However, as both Firefox and Epiphany use essentially the same rendering engine, these differences are trivial.

[QUOTE=leebrendalee]Is GNOME a desktop program?[/QUOTE]

GNOME is a Desktop Environment.  You seem to have the right idea about its function, which is to provide all of the nice eye-candy and many of the graphical applications on your system.

[QUOTE=leebrendalee]When folks talk about bash etc..what is that??? For command line stuff I'm just using terminal....no clue what I'm doing but amazinly reading stuff I"ve actually figured out stuff and used them to find specific information..kinda cool...
ok..so if someone could help answer these questions I'll be able to direct some questions to the correct place after that..thanks[/QUOTE]

Bash **is** the program you are using when you enter commands in a terminal window.  Your desktop simply provides a nice interface to it via the Terminal.
