First, do not expect full functionality with dragon naturally speaking and wine/crossover office. You will get voice recognition, but additional features of DNS are not there yet.

I have been using DNS 7 on Ubuntu + wine for some time now and basic voice recognition works, but you need to be careful which version of wine you use.

I do not know anyone who uses DNS 8 + Linux, wine or crossover office....

DNS 9, as far as I know, is almost completely untested.

IMO, if you are going to dual boot, I would go with DNS 9 + windows.

I run DNS 7 and do not dual boot.

Ask yourself if it is worth the hassle .... If so, see here :
