### Post by Sef on 2006-07-25
How much ram do you have?  If you have 1 gb or more, you don't need a swap partition, unless you are heavily into gaming, graphics, video editing, etc.

### Post by Sef on 2006-07-25
You need to set up a swap partition and not a swap file. Since you are going to do video editing, I would make your swap double your ram, especially if you only have a 1gb of ram.  Read this faq about adding more swap.  It should work fine with you.  Unless you want to reinstall Ubuntu.

[Add more swap]("https://help.ubuntu.com/community/SwapFaq")
