### Post by AtrejuT on 2007-05-03
basically it is a file used by the operating system to extend your RAM. so if you are using a lot of ram, and at some point you run out of 'real' RAM then your operating system (Ubuntu, Windows, OSX, whatever) will use parts of your harddrive as (very slow) replacement for ram...
