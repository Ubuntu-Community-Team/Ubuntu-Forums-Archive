### Post by gunksta on 2006-12-14
> **civic_si said:**
> It sounds like the CD is bad. Try burning it again with another program.


I agree. Sounds like you have made a kaleidoscopic coaster. Also, make sure you burn the iso to the disk as an iso, not a file.

Here's the difference. After you burn the .iso onto the CD, check the CD from Windows. If the directory of the CD is name_of_file.iso, it won't work. You should see a complex file structure on the CD.

An iso is a picture of the disk, not a traditional file, like an mp3.
