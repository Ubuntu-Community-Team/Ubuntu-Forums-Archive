### Post by welsh_spud on 2006-01-09
You can get lots of cool themes and wallpapers from gnome-look.org

To install the themes, go to System > Preferences > Themes (at the bottom) Then select the downloaded tar.gz file and click install.

Easy as pie!

### Post by Delkster on 2006-01-09
[QUOTE=welsh_spud]For themes like the _ [] X on the top of a window, you want GTK 2.0[/QUOTE]
Actually, to change the look of the window borders you'd want Metacity themes. Metacity is the window manager that performs operations on the windows and draws their borders.

The GTK (2.0) theme determines the appearance of user interface components such as buttons, menus and scrollbars, to name a few.
