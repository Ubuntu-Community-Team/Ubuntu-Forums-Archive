### Post by Bachstelze on 2007-04-28
That CPU is pretty good, go for it.

However, unless you have *very* specific needs, install the 32 bit version of Ubuntu on it. The 64 bit one will give you no significant performance gain but could bring you much more problems for a desktop use.
