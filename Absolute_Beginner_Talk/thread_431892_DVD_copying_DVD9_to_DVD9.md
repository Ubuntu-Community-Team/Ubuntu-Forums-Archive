### Post by starcraft.man on 2007-05-03
Just install k3b, that should do it for you. Type this in terminal:

```
sudo aptitude install k3b
```

Once its installed, open it up from the menu, go to Tools > Copy DVD. That should do it :)

Most burner programs support DVD9 they just don't say it explicitly, its not anything really special >.>
