### Post by tagra123 on 2006-08-31
I use VMPlayer for Windows only apps. Crossover Office works but programs meant for windows run better in VmPlayer.

You can get a free trail of the VM Workstation and create your Virtual Machine and Virtual Disk. Install Windows and Vmware tools. Don't forget to install the tools it makes everything work better.  Once its working you can make a backup copy of the Virtual Disk in case something installed screws up windows. To restore a messed up Windows just copy the backed up windows virtual disk to the directory containing your working virtual machine directory.

I store all of my files on the Ubuntu host (laptop) and access them from the Virtual Machine using samba. It's like two laptops in one and is easier than dual booting.
