### Post by Sef on 2006-07-26
> Most errors occur because the image file was not burnt correctly.


Most errors occur because no md5sum was done on the download, the image file was not burned correctly, or the cd is bad.  For the first one, if you download via a torrent, it will automatically md5sum.  With experience, the first two can be minimized, but bad cds will always be around. ### Post by A2A on 2006-07-27
Hi, you should be able to test the CD in about 20 minutes tops.  It's about 450MB only.  If you are planning to install this on an older computer, you may pop the CD in to your primary (newer) computer and check the CD there.  Once the CD passes, you should be fine.  If possible do not use any USB devices (mice or keyboard) during installation to reduce the chance of hang up's.  Another tip would be to install on a computer with no extra peripherals connected (Printer, Webcam, etc)  All these devices can be connected later.  
Keep us posted on the progress,

A2A
