### Post by nir78 on 2006-04-14
I had the same issue two days ago but now everything works great \\:D/ 
even dragging windows!

What I did is installing the ATI drivers, than change the xorg.conf file permission to write and not just read (otherwise the ATI changes did not worked).
apart from it I manually added to the file this options in the Device part:
[PHP]Section "Device"
	Identifier  "ATI Graphics Adapter 0"
	Driver      "fglrx"
	Option	    "(null)"
	Option	    "VideoOverlay" "on"
	Option	    "OpenGLOverlay" "off"
    Option "DesktopSetup"               "Horizontal,Reverse"
	Option	    "UseInternalAGPGART" "off"
	Option 	    "MonitorLayout" "TMDS, CRT"
	BusID       "PCI:5:0:0"
EndSection[/PHP]

Remember to insert only the option part and not the others!
After it I was able to drag windows and use wide desktop.

Nir
