### Post by fatdadkev on 2008-02-16
You could check if you have compiz running, I turned off the Alt-Tab function as I use a different screen switch tool in there.

The compiz-settings-manager will allow you to change the shortcut if you still want the same tab switcher ?

Its just a thought but it might help you remove the conflict by either re-assigning the shortcut keys or using an alternative from the options.

There are plenty of guides to installing and setting up compiz, I don't use Beryl as the new version of compiz effectively incorporates the features of Beryl so you don't seem to need both so check the guides first so you just install compiz.

One of the simpler guides is - [http://ubuntu-tutorials.com/2007/10/04/compiz-fusion-on-ubuntu-710-gutsy-gibbin/](http://ubuntu-tutorials.com/2007/10/04/compiz-fusion-on-ubuntu-710-gutsy-gibbin/)

Bear in mind though that the command to install the settings manager is all one line although it shows it on two.
