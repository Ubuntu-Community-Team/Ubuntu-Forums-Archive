### Post by megabyte405 on 2006-07-08
Just a note about the AbiWord autopackage - The most recent autopackage will be available at [http://abisource.com/download](http://abisource.com/download) when it is built after each release.  There was a delay in the last (2.4.4) release's autopackage, which is probably what you noticed.  However, AbiWord is included in Ubuntu's apt repository, which is the recommended location.  If you need to upgrade to 2.4.5 or later when they are released, either try an ubuntu-backport, urge the Ubuntu folks to upgrade their copy, or get the autopackage from the above download address.  If you need an older version (for instance, if the autopackage is lagging a version or something) you can check [http://abisource.com/downloads/abiword/](http://abisource.com/downloads/abiword/) under the desired version .  Good luck!  If you have any questions, please feel free to ask either me personally or the AbiWord user list!

-- Ryan, AbiWord dev and win32 maintainer, AbiWord Community Outreach Project
