### Post by 3rdalbum on 2006-11-04
My advice is: Forget about Beryl (the 3D desktop thing) for now. Forget about getting the proprietry 3D drivers, at least for a little while.

Instead, learn about the repositories, how to install the codecs and other programs that might be useful to you. Get some command-line experience - just the basic commands like cd, ls and sudo; as well as how to run programs from the command-line.

I think most of us here assumed that because we were experts with Windows that we would be experts at Linux. Most of us tried jumping into the deep end by attempting to get the latest unstable stuff working and configuring absolutely everything. It's akin to a computer-illiterate person trying to install Windows - it will lead to a lot of frustration.

When you become more confident in Ubuntu (which should come before too long), you can give the Nvidia and Beryl stuff a go. Tip: Make sure you use Ubuntu-specific instructions for the Nvidia driver installation, and make sure the Beryl install instructions are the newest. Beryl is so difficult to install because it's still in heavy development.

Just so you know, I only started using Windows *after* becoming confident with Ubuntu... I used Windows preinstalled and I still had a lot of troubles with it to start off with. I suppose I'm still not that confident with Win XP. We're in the same boat really.
