### Post by reiki on 2007-09-02
If all he had before was XP, and trying to install Vista screwed up his computer, chances are that adding linux into the mess isn't going to help.  If his system is set up and working correctly, AND if all of his hardware plays nice with both XP and Vista, tehn adding a Vista installation into a new partition on his hard drive (or on another hard drive entirely) should be tolerated and handled by Windows itself. 

#1 he needs to know what he's doing as far as setting up ANY dual boot situation

#2 It is my OPINION... that it is often safest to have separate operating systems on their own hard drives. It's not NECESSARY and in fact I've had as many as 6 operating systems in various locations among 2 hard drives.

But #1 is most important. Windows can handle booting multiple versions of windows. If he messed that up, don't throw linux at him as well. You'll only confuse him totally.
