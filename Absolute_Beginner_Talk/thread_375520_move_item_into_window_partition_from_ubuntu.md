### Post by taurus on 2007-03-03
I bet your Windows partition uses ntfs filesystem which means you can only read from it.  But if you want to write to it, then you need to install ntfs-3g driver first before you can write to it.

[http://ubuntuforums.org/showthread.php?t=217009](http://ubuntuforums.org/showthread.php?t=217009)
