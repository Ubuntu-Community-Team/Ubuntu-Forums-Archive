### Post by rainwalker on 2007-12-02
Those "alternate" iso's are text-based installers, rather than a graphical one.
The "desktop" ones are the normal graphical live cd iso's.
The "powerpc", "i386", etc have to do with what type of hardware or architecture you need to run it on

### Post by jken146 on 2007-12-02
"Desktop" discs are live CDs.  They will let you boot a live session of Ubuntu which loads into your RAM.  You can try out the OS before you install it and it won't touch your hard disk(s).

The alternate CDs are not live CDs and they have a text based installer.  This is useful if for some reason the live CD doesn't work on your computer (e.g. not enough RAM, some graphics card issues).

Both kinds of CD will install the same thing.

You need to choose the right CD for your kind of computer.  i386 is for an ordinary 32-bit PC (most people have one of these), powerpc is for a PowerPC, sparcs is for Sun Sparc systems.  There is a 64 bit version of Ubuntu too.

I didn't know there was a DVD but I guess it just has more packages on it that you can install later from the DVD.  If you have a fast internet connection just download a CD.

### Post by irish_flu on 2007-12-02
> **kaber said:**
> Hi, could someone explain the difference between these iso,s?


> 
 Ubuntu 7.10 alternate-powerpc iso

That's the old Macintosh architecture, used from 1996 or so up until 2005 (or so).  "PowerPC" is the name of the processor type.  "Alternate" means "no graphical installer or 'Live CD'"..
> 
Ubuntu 7.10 alternate-spars iso

"Sparc" is an architecture made by Sun Microsystems.  "Alternate" is "no graphical installer or 'Live CD'"..
> 
Ubuntu 7.10 alternate-i386 iso

"i386" is short for "Intel 386", and includes Intel and AMD chips (as well as other companies whose names I've forgotten) common to most processors over the last fifteen years or so; 386, 486, pentium, all the AMD "K" chips, Athlons, etc.  Almost all desktops and laptops will run this, except for some Macs.  The new 64-bit "Athlon64" and the new Intel "Core" line of processors can also run this just fine.  As always, the "alternate" in the name means "no graphical installer or 'Live CD'".
> 
 Ubuntu 7.10 Desktop-powerpc+ps3
 Ubuntu 7.10 desktop-powerpc iso

PowerPC arch mentioned earlier, with graphical installer.
> 
and what has Ubuntu 7.10 dvd iso got in it to make it so large?
kaber
Lots and lots of extra stuff from the repositories, so that you don't have to download packages you might want from the internet.
