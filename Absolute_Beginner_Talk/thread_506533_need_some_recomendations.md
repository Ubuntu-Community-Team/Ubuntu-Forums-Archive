### Post by Tux Aubrey on 2007-07-21
Of course, I have to say that depends what you want to do and why you are asking!

There are good Ubuntu derivatives for general purpose work:

[LIST]
[*]Linux Mint is a rebuild that takes away the codecs and binaries issues pretty well, but it is still Ubuntu.

[*]Mepis is nice, but it uses the the Ubuntu repositories.
[/LIST]

To get away from Ubuntu completely, you could go to:

[LIST]
[*]Both Fedora and Suse are great desktop distros (I'd lean towards Fedora for a newish user)

[*]PCLinuxOS - very clean and neat and easy

[*]Mandriva - (from which PCLOS forked) - also a good general purpose distro.
[/LIST]

[LIST]
[*]Dreamlinux would be high on my list too - it is Debian-based, like Ubuntu.
[/LIST]

Of course, there are about 300 others you could try!

I see you use Xubuntu, so if resources is an issue, have a look at the new Mepis-based Antix.  I happen to really like Damn Small Linux and Puppy Linux too for older machines and the next version of Fluxbuntu (to be based on Gutsy) will be worth looking at.
