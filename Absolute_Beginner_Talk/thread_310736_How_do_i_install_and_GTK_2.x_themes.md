### Post by SyvanX on 2006-12-01
You probably downloaded a theme file that ends in ".tar.gz" or ".gz"?

You should be able to open up your theme manager and drag the file into the themes, and it should handle unpacking for you.  You could also select the "Import" button and select your downloaded file.  Then you can get rid of the original file.
