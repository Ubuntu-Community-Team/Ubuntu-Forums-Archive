### Post by Ziox on 2007-05-16
I know that when installing from the alternative disc, you can switch to a virtual console to see the action behind the blue screen, so when you are reinstalling Ubuntu again with the live cd, try hitting ctrl+alt+(f1, f2, f3, or f4).  If there is any error, it is likely to show up on one of those virtual consoles.  

Sometimes, the installer might be doing something behind the scene that you can''t see any progress of (such as downloading files from the internet), so perhaps wait for bit to see if it'll continue.
