They're version numbers for Ubuntu. Higher version numbers are newer version of Ubuntu. People usually just use the codenames instead of the version numbers because they reflect the identity of the version better.

Feisty Fawn = Ubuntu 7.04 (to be released)
Edgy Eft = Ubuntu 6.10
Dapper Drake = Ubuntu 6.06
