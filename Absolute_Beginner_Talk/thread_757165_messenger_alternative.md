### Post by crjackson on 2008-04-16
aMSN is a clone of MSN Messenger.  It works great but I've never checked to see if it supports offline message retrieval.  It probably does.  It DOES support webcams and it works well.

Pidgin supports pretty much all messenger servises (yahoo, msn, icq, etc...)

It also supports offline messenging but you need to go into it's settings and install the pounc plug-in (and any of the other special feature plug-ins you want).

I use both of these and have no complaints other than pidgin doesn't support webcams.
