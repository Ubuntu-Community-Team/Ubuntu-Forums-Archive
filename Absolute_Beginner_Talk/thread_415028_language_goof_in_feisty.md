### Post by tbg58 on 2007-04-20
Try this in your terminal:
set $LANG=en_US.UTF-8

This will work for the current terminal session, I think.
You may have to reconfigure the .xconfig file.  If you want to bring up OpenOffice write in english, 
~ $ LANGUAGE=en_US oowriter
This should get you started.  I'll look around and see if I can find some more info for you.
