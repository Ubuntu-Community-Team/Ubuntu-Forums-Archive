### Post by PARG on 2007-08-03
Hi,

After you download the .deb package ([http://www.last.fm/download/):](http://www.last.fm/download/):)

1) Open the terminal: Applications > Accesories > Terminal.

2) Go to the folder where you save the .deb file:

cd packages

In my case, it is under my user folder in packages, a.k.a /home/myuser/packages.

3) Install the packages using dpkg:

dpkg -i lastfm_1.3.1.0_i386.deb 

I hope it worked!!
