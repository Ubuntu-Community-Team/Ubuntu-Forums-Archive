### Post by jem7v on 2007-04-14
For Xubuntu to run smoothly, here's what I'd Suggest:
- 500mhz processor or higher (not necessary, but definitely more pleasant)
- at LEAST 128mb ram, but I'd strongly suggest 256 or higher
- Intel or NVidia graphics, 32mb or higher (Intel and NVidia have the best driver support for their cards; NVidia is generally a bit more powerful)
- Search the forums/wiki to find out what wireless network cards work best with Ubuntu BEFORE you buy one that might not work

I'd also suggest trying Seamonkey for your web browser instead of Firefox because it loads up a bit quicker - especially on slower hardware.  I've also heard that Opera is pretty speedy.

RE getting your mp3s to play: have you read this page yet? [https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)
You might want to check Xubuntu's volume control (to see if maybe the sound is muted) and your Sound control panel (to see if your sound card is selected at all).  I'm not sure where those are in the XFCE menus (haven't used it in a while), so you'll have to look.


I have an old bruiser of a computer with 450mhz processor and 64mb ram running Xubuntu 6.06.  It runs pretty slowly with XFCE, but it's a bit better if I use Fluxbox instead and make sure I stick to lightweight apps.  Still, it's the lack of ram that's killing it.

### Post by jem7v on 2007-04-14
You can download packages (.deb files) from the online archive: [http://packages.ubuntu.com/edgy/](http://packages.ubuntu.com/edgy/)

Then you can burn them to a CD, take them to your Ubuntu computer, and install them.  All you have to do to install a .deb file is double click on it and then press the Install button.  (A .deb is a software package/installer, *sort of* like a setup.exe would be in Windows.)  Just try to make sure you have any dependcies satisfied before you install!  (Dependencies are extra packages that are needed so your new program can function.)
