> Also, I need info on WINE. What it is, what it does, where to download it from, etc.

**W**ine **i**s **n**ot an **E**mulator.  ;-) What it does is allow you to run a lot of Window programs under Linux. You can probably get it from Synaptic or in console with a "sudo apt-get install wine". If not, just enable the universe repositories. Their official site is [here](http://www.winehq.com/)
