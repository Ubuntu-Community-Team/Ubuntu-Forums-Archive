### Post by schauerlich on 2007-11-19
Compiz-Fusion (what Beryl has become now) is already built in to Gutsy Gibbon. However, you do need to download a program to access all the features. Run this in a terminal:

```
sudo apt-get install compizconfig-settings-manager
```

There should be a new entry in System>Preferences to set up advanced desktop effects.
