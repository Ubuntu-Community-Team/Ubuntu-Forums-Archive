### Post by anewguy on 2007-08-01
Okay, so you know Unix is an operating system, as is Windows 98, Windows XP, Ubuntu, etc..  At the heart of every operating system is a kernel - the basic functions that handle the low-level activities of the operating system.  Many other utilities, etc., are added on top of the kernel so that you can actually use it.  The package of utilities, desktop manager, extra programs, kernel, etc., make up a distribution of Linux - Ubuntu is the name of a distribution of Linux with a specific set of utilities, programs, kernel, desktop manager and adaptations.  There are varying releases of the distribution - Feisty Fawn is the "name"  of a release for which others have already explained the naming nomenclature.

As far as Ubuntu and gnome go:  gnome, kde, etc., are desktop managers that run on Linux.  Ubuntu by default uses the gnome desktop, Kubuntu (sp?) uses kde by default.  Each of these desktop managers normally has it's own set of utilities specifically written for that desktop.  If you look at synaptics package manager, for example, you will see packages that have "k" or "K" in front of them.  Most of time the "K" or "k" indicates it is a kde application.  You can run kde stuff in gnome as long as the supporting libraries, etc., are installed (synaptics package manager takes care of these dependencies for you).  I don't run kde in Ubuntu, but I would assume you could probably also run gnome things in kde with the proper libraries.  Note that gnome and kde are not "Ubuntu" things - they are Linux/Unix things.

Maybe that helps you a little more?:)
