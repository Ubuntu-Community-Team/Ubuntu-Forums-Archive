### Post by switchcode on 2007-08-22
doubleclick the tar file and it will open archive manager. Click "Extract", then browse to the extracted folder and find a file called "Readme". doubleclick it and choose "display" if the option comes up. Do what it says in the readme..
  You may not have to use Alien; Alien is downloadable through Synaptic (System>Admin>Synaptic Package Manager), and all it does is convert .rpm install files to .deb install files (Ubuntu likes .deb files, but it thinks .rpm files taste funny so it spits them out..)
