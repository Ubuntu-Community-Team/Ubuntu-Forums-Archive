### Post by AnRa on 2007-05-19
I think it could be the CD drive, if it's connected as slave (i.e. the jumpers are set to the slave position) then the connector of the IDE cable should be grey, if it's connected as master then the connector should be black. The blue connector has to be connected to the motherboard.

Maybe you can try looking in the internet for a more graphical explanation.

I hope that helps :)
