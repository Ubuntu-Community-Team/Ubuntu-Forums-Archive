### Post by Sandlst on 2007-10-26
You can set your own under System/Preferences/Keyboard Shortcuts 
(then go to Desktop, run a terminal).
Good luck, Post back if you have any problems.

### Post by nick_h on 2007-10-26
You can add an item to the panel.  Right-click on the panel and select "Add to Panel" then click "Application Launcher" then you can select any program in the menu.
