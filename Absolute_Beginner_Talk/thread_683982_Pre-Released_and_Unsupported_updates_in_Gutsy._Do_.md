### Post by misfitpierce on 2008-01-31
I use them and have been using them since forever. Never a problem with Ubuntu. You can use them they are quite safe. Keep in mind I may not install all software that you do but I have never had a problem. They are safe enough to use.

I recommend them yes as they may provide a newer version of software than Ubuntu does.There may be bugs in some software you install from it giving you newer version etc. If you get a problem you can go through synaptic and roll down the version of program to Ubuntu main version if thats the case. Should be fine using them.
