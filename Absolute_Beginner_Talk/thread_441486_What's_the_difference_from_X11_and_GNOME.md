### Post by Happy_Man on 2007-05-12
X11 is software that things like GNOME and KDE use to draw the desktop. You can run more than just those two on X11, like Fluxbox, E17 and the like, but that's a different story. Without X11, GNOME and KDE would be dead before the were born, and Linux would be a terminal-based type commands like affair, which nobody likes. Did that help?

### Post by Cypher on 2007-05-12
X11 is also known as the X-Window Manager. It is the core of the windowing interface. GNOME, KDE, Enlightenment, Fluxbox, XFCE and others are programs that run on top of X11 to give you all the window placement, movement and so on..

### Post by Bachstelze on 2007-05-12
X is a Windows System, i.e. the program that lets other program draw things like windows on the screen. KDE and Gnome are desktop environments, i.e. a collection of programs designed to look and feel alike and which, put together, make a fully functional desktop system.
