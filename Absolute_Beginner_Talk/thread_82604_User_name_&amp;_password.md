### Post by dbott67 on 2005-10-26
Try this:
[http://www.ubuntuguide.org/#gainrootwithoutlogin](http://www.ubuntuguide.org/#gainrootwithoutlogin)

After logging in as root, you can change the user password by typing:
```
passwd username
```
If you can't remember your username, just type:
```
cd /home
ls
```
to get a listing of all of the user's home directories.

-Dave
