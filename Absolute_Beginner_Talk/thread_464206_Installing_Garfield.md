### Post by ramjet_1953 on 2007-06-04
I know that you want this software, but might I suggest that you learn to crawl, before you walk?

This software will require one of two things:

Either convert the rpm package to deb, or compile the sourcecode yourself.

Both will require you to resolve dependency issues.

Also, you will need to install the [COLOR="Blue"]build-essentia[/COLOR]l package for compiling, or the [COLOR="Blue"]alien[/COLOR] package to convert rpm to deb.

Also, you will need to do quite a bit of reading, to familiarise yourself with the above.

Like most things, it is not hard, but you need to do the groundwork first.

Search these forums for advice on compling and converting rpm's to deb and make sure that you understand it all before diving in.
