### Post by candtalan on 2008-01-19
This is not a direct answer, sorry, however, I am a frequent Kubuntu user, and I enjoy compiz - but in gnome not kde (!).
I found it very easy to get it working in gnome (ubuntu that is, not kubuntu) while I have not yet been able to find what to do to use it with kubuntu.

If you would like to try it in gnome as I did, then you do not need to do a separate install of ubuntu, all you do in kubuntu is to install the package ubuntu-desktop using Adept package manager, it will bring a number of packages and libraries down. Then when you log in, you will have a choice of which 'Session' you want to use - kde or gnome. When you install ubuntu-desktop and it installs in the final stages, you will need to answer the question about which session manager you want (knome or kde login etc), you might want to continue with kdm as you use this currently.

good luck
