### Post by encompass on 2006-08-27
I don't play with windows things... but for your sound I have a few answers.
If you wants something like winamp go wtih xmms.
But I recommend something like banshee, or rhythmbox.
I have a lower end system so I like texbased everything and linux even has that.  I use a program called mp3blaster.  But I doubt you would need anything like that.
There are many option even beyond that... but if you are using standard ubuntu I recommends those three to chose from.  You already have rhythmbox installed.
And you can install the others by searching with synaptic...
[https://help.ubuntu.com/ubuntu/desktopguide/C/music.html#music-files](https://help.ubuntu.com/ubuntu/desktopguide/C/music.html#music-files)
is another helpful link...
[http://www.beginningubuntu.com/dapper_tips.html#Install_XMMS_the_old-school_MP3_player](http://www.beginningubuntu.com/dapper_tips.html#Install_XMMS_the_old-school_MP3_player)
