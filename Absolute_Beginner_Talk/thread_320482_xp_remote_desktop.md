### Post by dannystaple on 2006-12-17

not sure exactly which program to use but I know I need a remote desktop program. Any 

Suggestions?

This one is pretty easy. You can enable remote desktop access on Ubuntu via System->Preferences->Remote Desktop, and you should probably set a password on it too. This enables VNC access using a tool called Vino.

Now for Windows, you will need to install a VNC Client, I recommend getting TightVNC - a more advanced and recent client. It is completely free. You can then just install it, start it up, type in the IP address for the linux box, pop in the password and there you go.

Do not try anything 3D or with fancy graphics - or you will be in for a rough slow ride.
