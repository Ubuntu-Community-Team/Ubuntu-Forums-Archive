### Post by candtalan on 2007-12-11
It is aimed at kids. It has attractive theme items, it has more games out of the box, the non live cd (text install) comes with server and workstation all ready for classroom implementation, it has its own specialist interest groups and support groups.

All these are of course easily got also after installing basic ubuntu first anyway. One way is to install ubuntu then using synaptic package manager, install edubuntu-desktop. This will allow edubuntu (workstation) in a menu choice at login time.

You are correct to see edubuntu as a close relative to ubuntu - it is - however it benefits from its own identity, as one of the official variants.
