### Post by cstudent on 2005-12-24
The desktop environment is one of things that come down to personal preference. You could list pros and cons all day and still not decide which is better. You can have both environments installed at the same time. When you log on you will get a menu where you can chose which environment to log in to. I would suggest you do this and experiment with each. You'll soon start to form and opinion on which one suits you best. If you're new to Linux you might as well be informed now that the first several months you use it will sometimes be a frustrating experience with many rewards along the way. But in my opion it's very much worth it.

I would also suggest you install Ubuntu first and get a feel for it. Then install the KDE environment later. You'll just open Synaptic and find kubuntu-desktop and install it. It will install everything it needs for the KDE environment.

Good luck and welcome aboard!
