### Post by SyCo123 on 2008-01-28
First I'd suggest you read this so you know a little more of what to expect.
[Linux is not Windows]("http://linux.oneandoneis2.org/LNW.htm")
This might be useful post install
[beginners guide]("http://www.simplehelp.net/2007/08/13/linux-for-parents-a-beginners-guide-to-ubuntu/")


I suppose most people go the route of 
[LIST]
[*]Boot from Live CD to play around
[*]Dual boot or Wubi install linux
[*]Use windows less and less.
[*]Wipe drive and fresh install Linux without Windows
[*]Install a virtual computer and Windows again (*fills buckets of water to put out flames)
[/LIST]

One of the issues I found with Ubuntu is things like flash and other proprietary stuff. I've heard nothing but good things about Linux Mint which apparently installs Ubuntu with all the extra stuff Ubuntu avoids (so as to provide a completely non proprietary system option).
[URL="http://linuxmint.com/"]
http://linuxmint.com/[/URL]

Mint is basically the same thing as Ubuntu with the extra bits you need already installed so is probably a better choice for a total noob. Certainly less post install hassles.

Also you might want to make a copy of your windows drivers folder to DVD as sometimes getting wireless to work can be a pain and other things can be made to work if you have the windows driver. Sure you can get them on the net but a backup will be a time saver. It gets easier all the time though and for me the last time I just had to run a simple script that another user of  this forum wrote. Totally painless!

**[SIZE="3"]Recover your Windows product key and back up your data before doing anything.[/SIZE]**
It might be on a sticker but if it isn't you'll need a key recovery tool but get all the keys you can off your system before trying to install any other OS. It possible something will go wrong and you'll have to fresh install. It's not happened to me (except with HP windows recovery CDs) but it can happen so take precautions. You might not like Ubuntu and want to fresh install windows too, again that didn't happen to me :)

You can get most things working in Linux that work in windows using Wine or some other solution but sometimes it's just easier to use a virtual PC. The VPC route is easier than I thought it would be and now I have my old XP key working inside virtual box (a free program) so I can use Photoshop and office 2007. I'm going to play with CS2 and Wine tonight but I *can* use XP within the security of Linux, albeit with basic graphics power.

It's possible you'll have problems, but remember you have problems with windows too so try not to evaluate the entire system in in the same time you evaluate a single issue in windows.

The other advice I'd give is to actively participate here in the forums, as you can see they're a helpful bunch!

Good luck and if you experience is half as positive as mine you'll be a happy PC user again!

### Post by JoshuaRL on 2008-01-28
Good ideas.  I'm currently testing out a Linux Mint install to see what it's like.  Here's what I've found so far:

Pretty, right off the bat.  Very nice.
Codecs and all that work right away.
Great selection of pre-installed apps.  I really think that Ubuntu should include AptOnCD.
Easy to install new ones through MintInstall and .mint files.  (.mint files?  But at least debs work too.)
Unfortunately I could only find a couple dozen apps on their install site and their frontend.
And try as I may, I haven't found the way to install the Add/Remove GUI that I know and love.
So you really need to get used to Synaptic if you want to install anything but "base" apps.

But overall a nice distro to try for the real Absolute Beginner that just wants it to work no questions asked.
