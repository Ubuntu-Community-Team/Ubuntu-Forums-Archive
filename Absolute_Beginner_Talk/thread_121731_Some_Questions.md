### Post by Sutekh on 2006-01-25
[QUOTE=MasterChief1234]1. I used automatix to install it but it doesn't show up. Is it supposed to or does it run sliently ?[/QUOTE]
Type
```
killall gnome-panel
```
Then check in Applications -> System Tools -> Automatix.
Or type Alt + F2, and type in automatix.

[QUOTE=MasterChief1234]
2. The control alt del thing won't work. I am running gnome. Any ideas ?[/QUOTE]
Have you tried after a restart?

[QUOTE=MasterChief1234]
3. Does linux need to be defragmented ?[/QUOTE]
Short answer, no.

[QUOTE=MasterChief1234]
4. Is there a linux optimization tool ? like system mechanic for windows ?[/QUOTE]
Optimisation for what?  I don't know system mechanic.  
There may be, best to check Synaptic to see whats available.

[QUOTE=MasterChief1234]
5. Does linux use a registry ?[/QUOTE]
Not in the sense that windows did.  There are things like configuration files, such as the folder /home/user/.gconf/ that contains the .xml configuration keys for options in GNOME.

[QUOTE=MasterChief1234]
6. Do I need a spyware scanner ? (Tracking Cookies)[/QUOTE]
I don't think so....

[QUOTE=MasterChief1234]
7. .....that's all I have for now. Thanx[/QUOTE]
Cool! :p 

[QUOTE=MasterChief1234]One more thing: wtf is wine ?[/QUOTE]
wine - **w**ine **i**s **n**ot an **e**mulator.  

wine provides a abstraction(?) layer between linux and programs that desire windows to run.  You can use it to run a large number of applications for windows.
