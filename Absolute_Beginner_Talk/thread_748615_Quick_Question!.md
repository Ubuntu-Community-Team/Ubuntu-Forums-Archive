### Post by lackofcreativity on 2008-04-07
It is definitely a good idea to wait the 17 days because it will take all 17 for bugs to be worked out. When it is released, you just go to System -> Admin -> Update Manager. Wait for it to load, then click on the Upgrade button. Its a huge download, though, so if you have dialup it's probably better to go to the library or something, download the alternate CD ISO onto a flash drive, then burn it. It should ask to upgrade to 8.04. If you really can't wait, though, do Alt+F2 and type in:
```
update-manager --devel-release
```
Hit run and it will download the **BETA** version of Hardy. I strongly emphasize that you don't do this, though, because of all the bugs that have yet to be worked out. Just 17 more days...
