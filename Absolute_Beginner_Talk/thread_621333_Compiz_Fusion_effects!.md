### Post by RomeReactor on 2007-11-23
Hi. I didn't use that effect before, so I don't know if it's still around. You could try installing **compizconfig-settings-manager**; look for it in Synaptic, or to install it from the terminal:
```
sudo apt-get install compizconfig-settings-manager
```

After it installs, you can find it in "System->Preferences->Advanced Desktop Effects Settings", or in "System->Preferences->Appearance->Visual Effects tab" and click "Preferences".
