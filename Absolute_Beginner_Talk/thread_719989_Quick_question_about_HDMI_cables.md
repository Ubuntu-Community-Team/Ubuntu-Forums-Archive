### Post by dcstar on 2008-03-10
> **xhhux said:**
> nope, but nvidia-settings (in repository) will help adjust all your settings

The nvidia-settings package is ONLY required for legacy Nvidia cards, installing it in a system that uses the nvidia-glx-new or nvidia-glx driver **WILL break that system**.

nvidia-setting is already installed with the new drivers.
