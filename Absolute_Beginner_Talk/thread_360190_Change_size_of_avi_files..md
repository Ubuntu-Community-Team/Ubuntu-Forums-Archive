### Post by 454redhawk on 2007-02-12
> **ron999 said:**
> Hi
I checked out Avidemux but couldn't see how to change the file size.
Can you explain please.

Load your .avi and select the Video codec (xvid?).  Select configure under the video codec. Under main select 2 pass encoding and input your file size (233).  After that select the save button. It will take quite awhile. Maybe do it before you hit the sack and check it in the morning.

This is not an exact science but the file size should be very close. Trial and error may be in order.
