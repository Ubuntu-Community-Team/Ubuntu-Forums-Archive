### Post by Hospadar on 2007-08-08
I would assume you are trying to use the livecd, did you just copy the iso to a cd, or did you actually burn an image?  You can check by exploring the cd in windows or whatever, if you just see an iso on it, than you'll need to re-burn the cd, if you see a bunch of files, you burned it correctly, but are having some other probelm.

I know in infarecorder there is a "burn image" option, i believe nero has something similar.  this is what you want to use.

Also, make sure you are using the right version, mainly, be sure you arn't attempting to load the x64 (64 bit) version onto a (x86) (32 bit) machine.  If you are in doubt, load the x86 (32 bit) edition, it will work on any intel or amd processor.
