### Post by overlord.gaurav on 2007-10-11
Well, the manager works only when AWN is already started.
Go to Applications >> Accessories >> Avant Window Navigator
Now you will be able to run the Manger.

For cube rotation:
Press CTRL+ALT+Left mouse (hold the left mouse key) and roate.
Press CTRL+ALT+Arrow keys

Edit: These features may not work is Desktop Effects are not enabled.
To enable it: Go to System >> Preferences >> Desktop Effects >> Enable Desktop Effects

Note: You don't need Compiz-fusion for cube rotation!
