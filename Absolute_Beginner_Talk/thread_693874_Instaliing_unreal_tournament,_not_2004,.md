### Post by ajm8127 on 2008-02-11
just install wine from the synaptic package manager, and goto the command line and type 

# wine [path]

where [path] is the path to the installer on the cd. Once you get it installed, type

# wine [path]

where [path] is the path to the game executable on your hard dirve.

I just installed it today, with the hi res textures and it works great. This is for UT GOTY on Gutsy.
