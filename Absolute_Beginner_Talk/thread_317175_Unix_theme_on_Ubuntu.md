### Post by Qrk on 2006-12-12
If I remember correctly, that option is only cosmetic. Basically, if you are migrating from windows, it defaults to double clicks in Konqueror, and so one. It only configures things that vary slightly between different operating systems so that you can get used to KDE easily. 

Unix doesn't have a single operating system, let alone a single GUI/Desktop Environment. Your KDE already looks like Unix, as Unix runs KDE just like Linux does. 

Of course, when most of us think of Unix desktops, we think CDE (common desktop environment), but I have no idea why anyone would want to use that.  In fact, I don't think even Sun still uses it.
