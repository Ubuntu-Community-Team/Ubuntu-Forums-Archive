### Post by PmDematagoda on 2007-12-15
Amarok for KDE(Kubuntu) as a music player:-

```
sudo apt-get install amarok
```

Exaile for GNOME(Ubuntu) as a music player:-

```
sudo apt-get install exaile
```

Brasero as a CD/DVD burning software for GNOME:-

```
sudo apt-get install brasero
```

K3B as a CD/DVD burning software for KDE:-

```
sudo apt-get install k3b
```

One thing to remember, you can use KDE applications on GNOME and vice-versa, so for example if you do not like Exaile on GNOME then you can use Amarok instead without any problems.

EDIT:-

Kaffeine as a media player for KDE:-

```
sudo apt-get install kaffeine
```

Totem as a media player for GNOME:-

```
sudo apt-get install totem
```
