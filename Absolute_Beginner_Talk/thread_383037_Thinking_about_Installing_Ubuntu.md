### Post by Lanky Juggler on 2007-03-12
Not sure i can answer everything, but I'll give a few of those a shot.

You can look here to get a taste of what linux programs are like the ones you now use on windows: [http://www.ubuntuforums.org/showthread.php?t=33183]("http://www.ubuntuforums.org/showthread.php?t=33183")


There are several FTP editors out there, as seen here: [http://linuxappfinder.com/internetandnetworking/ftpclients]("http://linuxappfinder.com/internetandnetworking/ftpclients")

Also theres a supposedly pretty good add on to firefox, which you can try out on windows, here: [https://addons.mozilla.org/firefox/684/]("https://addons.mozilla.org/firefox/684/")

I'm not familiar with what WYSIWYG is, but a quick search of the forums (Try it, you'll be surprised what you'll come across!) came up with NVU and bluefish as two more popular ones.

As for a non-gimp graphical editor, what you want will depend on what you need it for.  There are plenty of small programs that'll do some cropping and such (gpaint) or ones that are a bit more complicated (Pixel, which you have to pay for as the demo will leave watermarks).  If you dont like Gimp because it was too complicated/cluttered, some people reccomend Gimpshop, which is a little simpler.  Theres a pretty good guide here: [http://linux.suramya.com/tutorials/Install_GIMPShop/]("http://linux.suramya.com/tutorials/Install_GIMPShop/")

Last, there are lots and lots of music programs (Amarok is one of the more popular, along with Rhythmbox.  There are maybe half a dozen "major" ones to choose from, i'd say, and if I tried to list them here i'd probably leave an important one out any annoy someone).  Because of the variety, though, you get to pick one that best suits your tastes (some are a lot like iTunes, others are a bit like WinAmp, even more are like Windows Media Player, and then theres all sorts that'll look new.  Shop around, is all i can say; you'll figure out what you like best only by trying it out.


I hope I was helpful to you, and happy hunting!

### Post by jkeyes0 on 2007-03-13
Alternately, if you don't want to dual-boot, you could virtualize your current XP install using VMware Converter, and run it in VMware Player or VMware Server (all free apps from [http://www.vmware.com)](http://www.vmware.com)). I'm running Ubuntu at home, and I've got my XP machine in VMware there just in case I need it. That way, if you find a program you need that you can't find in Ubuntu, you can load up your VMware session and plug away at it.
