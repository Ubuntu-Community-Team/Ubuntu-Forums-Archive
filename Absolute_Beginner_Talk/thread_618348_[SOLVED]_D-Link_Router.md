### Post by MegaJim on 2007-11-20
Routers are inherently operating system independent (although out of interest most routers of this type will be running a compact linux of some description).

Configuration is done through a web interface after connecting through a wired or wireless connection, so the more pertinent question would be: is your LAN/WLAN device compatible with Linux.
