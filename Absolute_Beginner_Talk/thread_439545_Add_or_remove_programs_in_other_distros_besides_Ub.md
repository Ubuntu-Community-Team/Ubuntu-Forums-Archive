### Post by jordanmthomas on 2007-05-10
Linux Mint has the Add / Remove option because it **is** Ubuntu with a bunch of preinstalled packages.

The Add / Remove program is (I believe) specific to Ubuntu.  You could probably install it in any deb-based system (debian, knoppix, etc) though.  In short, you can only have it with Ubuntu.  The thing that distinguishes most distributions from one another is the way they install / handle packages.   

Long answer:  I think Mandriva may have something as simple as Ubuntu's Add / Remove.  It uses rpms though so it would be a little slower.  What don't you like about the more full-featured package managers?  You can still search for what you're looking for just like in Add / Remove.
