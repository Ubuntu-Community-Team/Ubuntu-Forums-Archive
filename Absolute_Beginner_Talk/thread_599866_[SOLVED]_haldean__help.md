### Post by jordanmthomas on 2007-11-02
For the most part, you can delete .whatever directories.  All that will happen is you'll lose your configuration for that program.

So, if you see a folder and you know what it is, feel free to delete it.  If you're not sure, it's probably best to leave it.  Deleting .ICEauthority, for example, will mess with your current session so you shouldn't delete it.

However, these are not "installed by the operating system" like mikewhatever suggests.  He is, however, correct in saying that some programs will not work right without their .whatever files.  Some programs will refuse to start if they don't have a configuration file (pypanel comes to mind) but most will just generate a new default config if it's missing.
