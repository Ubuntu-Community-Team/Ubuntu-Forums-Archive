### Post by Therion on 2008-01-27
**Method 1.**

Navigate to System/Preferences/Appearance Manager.
Use the "Install" button and navigate to the download file for the theme.
Click on the "Customize" button and select the theme you want.

**Method 2.**

Navigate to System/Preferences/Appearance Manager.
Drag and Drop the .tar.gz file you downloaded onto the Appearance Manager.
Click on the Customize button and select the theme you want.
