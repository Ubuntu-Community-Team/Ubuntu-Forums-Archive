### Post by benfindlay on 2006-09-21
In ubuntu in System>>Preferences there is a keyboard shortcuts panel, where you can set your chosen shortcuts. I use the "Windows" key for the menu (may as well get some use out of it! ;))

There should be a similar option in xubuntu for this. Once you've found it you just double click the option you want to change and press the new key you want to map

Hope this helps
