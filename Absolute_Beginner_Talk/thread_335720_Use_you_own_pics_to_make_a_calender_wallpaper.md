### Post by jem7v on 2007-01-10
Because you just want to make wallpapers for your computer, you might just want to try adding text in The GIMP, which should be in Applications -> Graphics.  It's the most readily available image editor that works well.  Making the layout of the text might not be very simple, but I can't think of any programs with "calendar wallpaper" wizards.  Scribus has a calendar wizard if you want to print them, but nothing for just plain image editing.  You could mess around with it a bit though, to see if you could export the work to an image file.  Not sure if you can or not.
