### Post by some_random_noob on 2007-11-16
If you open your home folder and create a new folder (Say, "music" for example) then you can drag that folder onto the sidebar on the left (The sidebar with all the locations). Now you should see "music" (Or whatever) on the sidebar along with all the other commonly used folders. This will also show up on the list of where you want to save a file.
