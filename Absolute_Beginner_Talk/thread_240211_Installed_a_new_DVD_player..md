### Post by catlett on 2006-08-20
You have to get codecs and playback capability. These things have patents in the US so Ubuntu does not load them because it would be illegal in the US to do so.
Following 2 guides will get you there. The Dapper Guide (just make sure you change the repositories like it says at the beginning of the guide)[http://doc.gwos.org/index.php/DapperGuide](http://doc.gwos.org/index.php/DapperGuide)
And the restricted formats page for win32codecs [https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)

Or you can take the easy route and install Automatix. It is a script that does it all for you [http://ubuntuforums.org/showthread.php?t=223087](http://ubuntuforums.org/showthread.php?t=223087)
