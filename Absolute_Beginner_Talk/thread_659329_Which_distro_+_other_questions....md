### Post by SOULRiDER on 2008-01-05
Thats because it isnt installed. An OS running from a CD is sluggish as hell. Once you install it will be much much faster and it wont freeze (its freezing because it has no usable hard drive so RAM fills up).

I suggest you use Xubuntu since XFCE is much lighter than GNOME but fluxbuntu will be even faster although I dont know how newbie friendly it is.

If you have further questions or get stuck during the installation just go ahead and ask for help :)

Good luck.
