### Post by sYs^ on 2006-04-11
You should install Ubuntu to another partition and yes, you can share those programs because Ubuntu has read/write support for FAT32, but if you want to run them you will have to use a Windows emulator like wine. You won't have to reinstall anything because Ubuntu is fully independent from Windows.

### Post by dabear on 2006-04-11
Ubuntu can resize your partition, so that you can choose at startup which operating system you want to start. That way you won't loose any data(but remember to backup anyway before you do this). 

Most windowsprograms won't work in linux, but apps such as firefox, openoffice, amsn/gaim, gimp and inkscape is available in what we call the «repositories».

 In ubuntu, you don't install programs by downloading them manually from some website, but rather you press system ad then go to administration followed by the program «synaptic package manager». There you'll search for what program you want to install, tick it off and press «apply».
