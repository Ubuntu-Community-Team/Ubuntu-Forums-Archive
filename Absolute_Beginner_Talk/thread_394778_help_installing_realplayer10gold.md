### Post by camz on 2007-03-27
I did it as well but I don't remember exactly. I'll give it a shot. Change root to whatever it is, home or whatever.. In the terminal, type:

sudo install /root/desktop/RealPlayer10GOLD.bin

Now follow the prompts inside the terminal window. When it asked if you want to use system wide links or whatever it is called, make sure to click yes, and leave it as the default option, or it will be hard to get it back to normal. After it's ready, you can delete the .bin file.
