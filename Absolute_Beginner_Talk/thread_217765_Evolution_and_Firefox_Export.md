### Post by jordilin on 2006-07-17
If you do a fresh Dapper installation, I would replace the whole .evolution directory. As for firefox, I would replace the whole .mozilla directory. If you only wanna save your bookmarks then save the file .mozilla/firefox/yourprofile/bookmarks.html.
