### Post by FuzZy2006 on 2006-07-10
How can I change file associations in KDE and in Gnome?

---

### Post by aysiu on 2006-07-10
In KDE, right-click the file, select Open With, select the application and then check the box to always open it with that application.

In Gnome, right-click on the file, go to Properties, Open With, and select the default application.
