### Post by FuturePilot on 2008-01-03
Well there's a couple of way. You could go Applications>Add/Remove and search for ubuntu-restricted-extras which will install Java along with many other things like codecs Flash and extra fonts. Or you could go more direct and enter this in a Terminal. Applications>Accessories>Terminal
```
sudo apt-get install sun-java6-jre sun-java6-bin sun-java6-plugin sun-java6-fonts
```
