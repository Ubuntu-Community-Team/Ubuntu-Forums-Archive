### Post by kostkon on 2007-07-30
It is possible, if you use *ffmpeg*.

It is recommended to install the version of *ffmpeg* from the [medibuntu repository]("http://medibuntu.org/").

Thus, first of all, [add this repository to your software sources]("http://medibuntu.org/repository.php"). Don't worry, the repository is safe.

Then, go to *Synaptic* and search for *ffmpeg*, select it and install it.

Afterwards, you will have to find a GUI frontend for *ffmpeg*. A simple one that you can use and it has a preset for *wmv* videos is [WinFF]("http://biggmatt.com/winff/"). Thus, go [here]("http://biggmatt.com/winff/downloads/") and download the *deb*, double click on it, and install it.

The program will appear in the *Sound & Video* menu. The rest is yours...
