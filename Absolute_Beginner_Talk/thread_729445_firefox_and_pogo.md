### Post by sowelie on 2008-03-19
There aren't any settings that I'm aware of in Firefox / Java for this, but if you're using Compiz you could use the zoom feature.  If it's enabled hitting Super (the windows key) + 2 will zoom in quite nicely.  Hit Super + 1 to return to normal.

### Post by Sisyphus48 on 2008-03-23
I use Pogo all the time and it is simple to make it big.  As soon as the game loads go to the System (on the Ubuntu bar), Preferences, and Screen resolution and change it to a lower rating (I use 800x600) and that makes it a lot bigger!  When your finished playing change it back.  I do it daily  and it's quick and easy.
