### Post by por100pre1 on 2007-09-21
It's takes time (and some hacks) to get a desktop like the one shown in that web page. Let's start with the easy part:

```
systemsettings
```

Once there go to Appearance, there are many settings for colors, desktop, windows, fonts and more. Right click a panel to unlock it and then right click it again and select to configure panel. You'll get to configure many Kicker (panels) settings then.

Hope this helps. Sorry for the delay.

### Post by por100pre1 on 2007-09-21
If you right click the panel to unlock it, do it again and select to configure panel. In the new window select appearance and select a background image. If the default ones don't do the trick; download [these ones]("http://www.kde-look.org/content/show.php/Kicker+backrounds?content=61444"), save them wherever you want, and browse to select one of them from the panel settings.
