### Post by taurus on 2007-02-24
/dev/hda1 is Linux, not Windows, ntfs.  Therefore, you need to edit /etc/fstab and make the change.  See if you can boot into recovery mode from GRUB menu and at the prompt, run

```
nano -Bw /etc/fstab
```
and remove this line

```
/dev/hda1 /media/hda1 ntfs-3g defaults,locale=en_AU.UTF-8 0 1
```
and replace it with 

```
/dev/hda1  /media/hda1   ext3   defaults   0   1
```
Save it.  Now, reboot.

```
shutdown -r now
```

### Post by taurus on 2007-02-24


### Post by taurus on 2007-02-24
Edit /mnt/ubuntu/etc/fstab

```
gksudo gedit /mnt/ubuntu/etc/fstab
```
and put a # in front of these two lines.

```
/dev/hda1 /media/hda1 ext3 defaults 0 1
/dev/sda1 /media/hda5 vfat defaults,umask=0000 0 0
```
Save it and eboot (and don't forget to remove the LiveCD).
