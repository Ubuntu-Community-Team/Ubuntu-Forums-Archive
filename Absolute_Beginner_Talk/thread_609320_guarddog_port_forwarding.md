### Post by jARLAXL on 2007-11-11
-in guarddog go to tab advanced.
-click new protocol
-write a name for the protocol
-below specify a port range and protocol type
-go to protocol tab then for both zones internet and local tick this protocol to enable it
-click apply

i hope this helps...
