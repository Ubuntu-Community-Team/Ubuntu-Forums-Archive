### Post by davmac on 2005-10-30
Included in the original install is cron. It is something you'd tend to use from the command line. There may be gui front ends available for it but I'm not aware of any.

There's a basic tutorial at [http://www.clockwatchers.com/cron_main.html](http://www.clockwatchers.com/cron_main.html) and something a bit more detailed at [http://www.unixgeeks.org/security/newbie/unix/cron-1.html](http://www.unixgeeks.org/security/newbie/unix/cron-1.html)

Hope this helps,
