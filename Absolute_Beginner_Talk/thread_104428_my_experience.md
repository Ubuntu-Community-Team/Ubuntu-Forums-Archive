### Post by davebradford on 2005-12-16
I have had a funny year when it comes to computers i though i mite just post it here for people maybe in the same situation as me to read (also partly because i have some free time at work!).

It all started in Jan this year when i found a small comp box very small very low spec.. at that time i was hosting apache on my windows xp machine with ftp services etc etc. I looked at the box and though about dumping all my web sites and ftp server onto this one box to take the load of my machine, at first i ran windows 98! doh! no system services for running apache :o( tired windows 2000 and it wasnt having any of.. too slow and hardware problems, i came across fedora core because a few of my friends where using it at the time, totally new to linux i got fedora core 3 and install it using the the full packages, it was fine worked ok and did everything i needed it to do.

Later on this year i built a new xp machine for myself and used my old main machine as a server.. dumping the low spec machine. At first i was going to use windows 2000 server as the spec of my machine was now mightyer than the pentium I i was using before :p but instead i just installed fedora again. In the summer i was looking for screen shots of linux desktops and noticed a distro called ubuntu, never heard of it didnt know if it was any good etc etc. When on their website and it looked really interesting, so i order a couple of free cd's and did some live cd'ing on my notebook. Nice UI but lacks applications and tools, and then i though! I was looking for too many applications becuase i was used to a LOADED distro like fedora!!! i canned fedora and started using ubuntu amazing! i though it was going to be very different as i was used to a rpm based distro!!! hardly any difference wot so ever! and apt-get works a whole lot better than yum did! 

What i'm trying to say is i was blinded by all the hyper these "popular" distro's cause, but a little n00bie research goes a long way you can pull up things that are far better than what you've already got! i just wish i found ubuntu so i could have run it on my low spec machine for ease! Also another tip! If your a n00bie and are running as a server.. you will learn so much more by using ubuntu server without a UI! Though youself in at the deep end, it's the better way to learn.. there's nothing great about being able to use a GUI! it's whats underneith that counts!

Hope that wasnt too long just though poeple might get inspired by wot i did!

Dave

I've got to agree with you.  My mail server is a PII at 233Mhz.  It does full spam and virus checking and serves emails to 5 user machines.  OK, perhaps most people simply use a single computer for everything but this is a good example of reusing 'old' hardware that cannot do anything useful under modern versions of windows but can do a good if limited job in one specific area.  True, I cannot run any decent GUI but, there again, I do not want to.  I have learnt so much by implementing my own server that it has helped me in many other areas.
