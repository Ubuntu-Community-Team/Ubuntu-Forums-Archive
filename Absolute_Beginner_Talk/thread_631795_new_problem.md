### Post by davidgarcin on 2007-12-04
I'm assuming you have a thread running about this already?  You really should post in that thread so that people trying to help have some background on your question.

For partitioning, use the program installed on the Ubuntu Live CD.  Put the CD in the drive, reboot, go to System -> Administration -> Partition Editor (or something similar... I can't remember the exact name off hand.

Here's a pretty good install guide: [http://news.softpedia.com/news/How-to-Install-Ubuntu-7-04-Windows-User-P-O-V-52973.shtml](http://news.softpedia.com/news/How-to-Install-Ubuntu-7-04-Windows-User-P-O-V-52973.shtml)

You can do the partitioning while you're installing, you don't have to do it beforehand.

Hope that helps.
-David
