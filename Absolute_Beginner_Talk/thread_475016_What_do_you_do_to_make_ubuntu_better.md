### Post by 3rdalbum on 2007-09-25
The reason why the desktop effects don't work is because ATI doesn't support a particular technology called AIGLX which is necessary to get the desktop effects working.

However, there is a program called XGL which you can install, which will allow the desktop effects regardless. Try looking for a HOWTO on the Compiz Fusion website for installing Compiz Fusion on XGL.

As for making Ubuntu better, I thought outside the box and got rid of my taskbar - rearranged my desktop layout to maximise the use of space. I'm also making an effort to make Ubuntu better for everyone, by learning programming.
