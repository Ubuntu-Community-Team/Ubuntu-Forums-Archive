### Post by Paqman on 2008-04-16
You won't get a lot of desktop effects from a VM. That's because VMs don't emulate 3D graphics acceleration. Having said that, some people report getting good performance from Compiz on quite modest hardware.

In general though, I would suggest using Xubuntu for a VM. Even then performance won't be very smooth.

You can always try out different desktops, regardless of which install image you use. Simply install the packages:

ubuntu-desktop
kubuntu-desktop
xubuntu-desktop

Log out > Options > Change session and pick what flavour of desktop you want to use.

A VM is great to try things out before committing to installation, but if you want all the desktop effects, you'll have to set up a dual-boot. Which really isn't that hard, by the way.
