### Post by steveneddy on 2007-06-24
In KDE and Gnome, if in terminal you can type the name of the application and it will open.

I think gnome-open is used to open a URL, or an address on the internet.

When I type 

```
gnome-open
```

in terminal, it comes up with

```
Usage: gnome-open <url>

```

meaning it is looking for an address on the internet.

If you want to open Gaim for instance, just type "gaim" in the terminal and it should open.
