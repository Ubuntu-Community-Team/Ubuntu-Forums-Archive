### Post by Kateikyoushi on 2006-10-17
1. You can leave XP on your machine and ubuntu next to it, when you start the computer can select which OS to boot.
2. Yes you can remove XP from ubuntu in a matter of minutes.
3. Ubuntu recognizes and automatically configures most hardware depending on your setup you might have to install some drivers manually.
4. I doubt you will find it lacking and as a beginner you can get lot of help on this forum.

### Post by caravel on 2006-10-17
1/2. When I first tired Linux back in 2004 I nuked windows completely, but ended up going back to it, because often there is no alternative if you want to play games.  It depends on you individually.  I wouldn't be too hasty to take it off just yet.  The problem with having it there, is that it offers and easy way out...
