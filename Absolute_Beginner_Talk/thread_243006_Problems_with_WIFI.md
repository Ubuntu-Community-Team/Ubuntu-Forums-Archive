### Post by Carbonfish on 2006-08-24
Hi eg-maverick,

Download nad install network-manager. You can search for it in Synaptic. It will solve those problems pretty much automatically (WEP stuff anyway, WPA is another issue). After you have it installed it will display available networks, you click on the one you want to join, and that's it. If it (the network) requires a WEP log-in it should just ask you for your user name and password info. That's how it's worked for me anyway. :D
