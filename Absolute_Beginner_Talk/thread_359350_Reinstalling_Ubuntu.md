### Post by nickoli_02 on 2007-02-11
Sure, grub isn't limited to linux. It's just a boot loader that sits in the master boot record of the hard drive. You should be able to find out how to install grub easily enough under windows, do a search of the forum/google.

When you install ubuntu the only changes you need to make to the partitions is to reformat the partition that your broken ubuntu is currently on, then repartition it. Everything else is done as you did the first time around.
