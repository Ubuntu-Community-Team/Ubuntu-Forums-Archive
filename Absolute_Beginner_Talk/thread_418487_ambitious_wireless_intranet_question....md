### Post by mabelrxu on 2007-04-22
the kind of network you're describing is called (in network admin circles) a full-mesh - each computer connects to each other computer in the network "ad-hoc" ... they're nightmares both to setup and to maintain ... basically, the number of cards necessary for _each_ computer goes up _exponentially_ as you add more and more computers to the network ... you do not want to have to suffer through that

nevertheless, it is possible to do what you're describing ... if only one of your computers (the client) is accessing those other computers (all acting as file servers) and if you only wanted to get data from one "server" at a time

if that's the case ... think hard, cause you'll be up for a lot of work and hassle and trial and error ... it's much easier to just get a router that all cards can talk to, then configure each machine to talk to that router ...
