### Post by Cypher on 2008-02-05
Edgy Eft was at 2.6.17, you'd have to go back to Hoary Hedgehog (5.04) or Warty Warthog (4.10) from late 2004 to early 2005 to get a distribution with 2.4.

You can download the LiveCD's for these versions, boot it up and check the version. Attempting to run the older Kernel versions with the newest version of Ubuntu might not be the best way, so starting with the older Ubuntu versions will give you the best chance of getting the 2.4 Kernel up to begin with.
