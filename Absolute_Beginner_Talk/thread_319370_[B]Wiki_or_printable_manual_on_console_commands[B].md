### Post by kebes on 2006-12-15
The commandline can be complicated at first. You are right: there are thousands of commands available. The best thing is to just learn slowly, one command at a time. Each time you run into something you want to do (on the commandline), do a quick web search and you'll probably find a command that can help you out.

When you're wondering how to use a command, and what the options are, you can easily consult the "manual page" or "man page" for that command by typing:
man command
So if you want more information about the "ls" command type "man ls" and it will tell you all about the options you can use. Another useful command is "apropos", which will tell you about commands related to the given topic. So if you want to know what commands are related to mp3, just type:
apropos mp3

There also man Linux Reference Cards (sometimes called "cheat sheets") that summarize the most frequently used commands. A long list can be found here:
[http://www.digilife.be/quickreferences/quickrefs.htm](http://www.digilife.be/quickreferences/quickrefs.htm)

For linux commandline commands, check out these in particular:
[http://www.pixelbeat.org/cmdline.html](http://www.pixelbeat.org/cmdline.html)
[http://www.tuxfiles.org/linuxhelp/linuxcommands.html](http://www.tuxfiles.org/linuxhelp/linuxcommands.html)
[http://www.digilife.be/quickreferences/QRC/UNIX%20commands%20reference%20card.pdf](http://www.digilife.be/quickreferences/QRC/UNIX%20commands%20reference%20card.pdf)
[http://www.digilife.be/quickreferences/QRC/LINUX%20Admin%20Quick%20Reference.pdf](http://www.digilife.be/quickreferences/QRC/LINUX%20Admin%20Quick%20Reference.pdf)
[http://www.digilife.be/quickreferences/QRC/The%20One%20Page%20Linux%20Manual.pdf](http://www.digilife.be/quickreferences/QRC/The%20One%20Page%20Linux%20Manual.pdf)


Hope that helps. Remeber that there is always more than one way to accomplish a task (in general, there are many GUI ways and many CLI ways), so don't feel like you need to memorize every single command. Learn the ones you need and you can always look up more information when you need to.
