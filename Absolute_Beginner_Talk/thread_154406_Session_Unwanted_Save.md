### Post by sony102600 on 2006-04-03
Hey guys... I do NOT have save current session checked when I shut down, however, upon start up, all the programs I was using about 10 sessions ago start up.. amarok, open office, it trys to find my ipod, and gaim all start automatically upon startup, any idea why?

---

### Post by nanotube on 2006-04-03
you must have saved a session about 10 sessions ago, so it's still saved.
what you want to do is enable save session, then close all your apps and log out. that will save a new "empty" session on top of that. now you can disable save session, and live with the empty saved session.
