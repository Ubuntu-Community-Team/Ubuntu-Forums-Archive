### Post by xXx 0wn3d xXx on 2006-10-13
1. Grab a clean cd

2. Keep the folder compressed. Leave it as it is.

3. Burn the image to Disc with Nero (at 4x select burn image to disc)

4. Set your bios to boot from disk and reboot with the disc in the computer.
