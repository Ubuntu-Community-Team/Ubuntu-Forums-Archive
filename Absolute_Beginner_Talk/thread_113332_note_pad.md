### Post by Chipmunk on 2006-01-06
The equivalent of notepad (at least in Gnome) would be Gedit.

One thing I have noticed is if you close the donor application, you lose the ability to paste. Such that if you open this thread, select and copy some text, then close the browser **before** pasting, it won't paste. Same applies for the next paste into yahoo (I assume you're editing the text in gedit or something)

Simple answer is to keep the donor application open until done pasting :)
