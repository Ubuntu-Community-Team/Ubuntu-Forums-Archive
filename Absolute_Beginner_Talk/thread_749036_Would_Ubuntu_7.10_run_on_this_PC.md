### Post by warp99 on 2008-04-08
The gOs is Ubuntu with the enlightenment (E17) windows manager. It's not "Google's OS" it just has some Google apps included with the default install. Ubuntu 7.10 would run because that's what already on the computer.

See what the manufacture did is drop E17 onto a Ubuntu base install, included some wallpaper/icons, and called it "gOs". It just marketing hype since you can install E17 on Ubuntu if you wanted:

[http://www.enlightenment.org/](http://www.enlightenment.org/)

So to answer you question of course you can install Ubuntu, Kubuntu, or Xubuntu no problem. I would since E17 is not a very stable desktop compared to the other flavors of Ubuntu.
