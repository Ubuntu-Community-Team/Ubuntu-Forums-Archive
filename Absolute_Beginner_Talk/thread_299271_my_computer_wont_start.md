### Post by halitech on 2006-11-14
are  you using the onboard video or an addon card?

if using an onboard card, try taking it out and using the onboard video. also try reseating the power connectors on the motherboard.

make sure the RAM is PC100 as the board doesn't support anything else and you don't have anymore then 256 installed

far as the CPU, looks like it is a slotket like the old celerons. it should pull straight up and out.
