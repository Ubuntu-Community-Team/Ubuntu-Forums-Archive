### Post by kostkon on 2007-11-03
No worry, there is such a feature in *Pidgin*. Just go to *Synaptic*, search for the package *pidgin-guification*s and install it. Then start *Pidgin*, go to *Tools -> Plugins*, find the *Guifications* plugin and set it up to inform you for any event you like.

You can change the theme of the pop-up. You can find many guification themes at [gnome-look.org]("http://gnome-look.org/").
