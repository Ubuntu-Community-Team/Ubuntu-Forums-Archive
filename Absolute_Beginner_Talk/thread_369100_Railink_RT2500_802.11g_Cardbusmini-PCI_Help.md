### Post by crispy_420 on 2007-02-24
If I remember correctly, it's been a bit since the install, my rt2500 card was supported right away. No driver install or ndiswrapper here. The only reason I don't use it is due to bad support for network-manager. But you want links to drivers, so here they are:

[Main RaLink Support Page (all OS)]("http://www.ralinktech.com/ralink/Home/Support.html")

[Windows Driver (direct download)]("http://www.ralinktech.com/ralink/data/IS_AP_STA_6x_D-1.2.0.0_2500_D-3.2.0.0_RU-1.2.8.0_AU_1.1.5.0_122106_0.0.7.0.zip")
[Documentation]("http://www.ralinktech.com/ralink/data/Software%20Release%20Note%20RT6X_STA_122206.pdf")

[Linux Driver as source code (direct download)]("http://www.ralinktech.com/ralink/data/RT2500-Linux-STA-1.4.6.6.tar.gz")
[Documentation]("http://www.ralinktech.com/ralink/data/ReleaseNote-1466.txt")

Like I said earlier, mine worked right away. Does the card show up under System -> Admin -> Networking?
Also this was something that I have overlooked in the past. Did you turn it on? On my laptop it was a combo key (Alt + F2).

Good Luck!
