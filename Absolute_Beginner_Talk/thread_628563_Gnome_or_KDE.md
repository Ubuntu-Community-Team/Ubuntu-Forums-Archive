### Post by Whiffle on 2007-12-01
> **steelmole said:**
> What difference does it make?  I hear people say "So and so is a great kde app" does that mean I can't run it if I use gnome(and vice versa).


The difference between KDE and gnome apps is that they use different libraries (similar to dll's in windows).  The effect for the user is that they may not match in appearence much (although they can with the right themes applied), and if you're going to say run a KDE app under gnome it will have to load KDE libraries instead of just the gnome libraries.  I for one don't really care about the difference, I run KDE or Openbox mostly with a pretty good mix of gnome and kde apps.
