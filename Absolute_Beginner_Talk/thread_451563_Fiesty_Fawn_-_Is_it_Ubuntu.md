### Post by Herman on 2007-05-22
The very first version of Ubuntu that came out was code named 'Warty Warthog 4.10'.
         Then Ubuntu 'Hoary Hedgehog 5.04', and the next version was, 'Breezy Badger 5.10 '. 
           'Dapper Drake 6.06  ' is the Ubuntu release that has long term support, it will be supported for three years from it's release date. If you want peace and quiet you can use that.
Adventurous Ubuntu users usually go for the next releases as soon when each one is officially released so we can be right up to date with the cutting edge software. The conservative types wait a week or two or just stick with Dapper Drake.
'Edgy Eft 6.10' followed 'Dapper Drake', and now 'Feisty Fawn 7.04' has been released. The next Ubuntu version has been named already and will be called 'Gutsy Gibbon'.
The numbers after the names indicated the release dates, for example '6.10' means October 2006 was the date that version was released.

One tip is to remember that since there are no patent restrictions, and it's free, we are free to install more than one copy of Ubuntu per computer. In fact we can have as many as we like! I installed two copies of Ubuntu for my wife, Edgy Ultimate Edition and Feisty Fawn. That way if she has any trouble with anything in one installation while I'm at work she can just reboot into the other and keep doing what she likes until I come home and help her.
For me, I like to preview the next release ahead of time, so I have the current version plus the future version when think it might be mature enough for me. I'm a little lazy about getting rid of past versions too.
In my personal opinion that way of doing things is more convenient than making an installation with a separate /home partition. I just make all mine one-piece installs +swap and transfer data from one to another when I'm ready. It only takes a few GB of extra hard drive space to do things this way, Ubuntu can be installed on as little as 3.0 GB, or less.
My wife has an 80.0 GB hard disk with 20 GB Windows XP which she never uses anymore and two Ubuntu installs on 30 GB each. She has lots of room because she keeps most data on an external drive.
Whatever you do, enjoy Ubuntu.

Regards, Herman :D
