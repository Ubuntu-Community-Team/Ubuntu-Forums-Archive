I would think your onboard video would be ok. Only one question - often onboard video shares system memory. Is that the case?
If it was me, I'd just run the install and see if it worked. If there was any problem with the memory size, you can often allocate more system memory to be shared through your BIOS.
If that didn't work and you need a dedicated video card, you can pick up a cheap nVidia 6200 for about $45 US. Plus, it is not bad even for gaming.
