### Post by cmnorton on 2007-11-13
Suggestion:

Install Ubuntu on the new drive, or browse through these forums to see how you're going to set up dual-boot and make sure that Windows will boot with the boot sector on the first drive and C:\ on the second (if that's possible). 

Offhand, the Ubuntu install is very quiet, but your question has peaked my curiosity as to how you would perform a more interactive disk configuration with Ubuntu.

My message is poke around for a while and get all your ducks in a row before you jump off and try this.

Hi there,
Ubuntu will let you choose whatever drive you wish to install it on or partition a drive to share it with Windoze.

When it installs it will install a file called a Grub file on the boot sector of the first hard drive, from this you can choose to boot into Ubuntu, as default if nothing is selected in the given time, or Windoze.

Hope this helps
