### Post by davmac on 2005-10-28
1. I used fedora briefly after red hat, but quickly went on to Mandrake and Suse. Happily using Ubuntu now to the point where it has become my primary OS. I still use WinXP for a couple of apps I need but with any previous distro it was never more than a irregular alternative to a microsoft OS.

2. Overall fantastic. My wireless card worked out of the box as did pretty well everything else I needed. Would second the recommendation regarding the Automatix script. Fedora Core 3 was the first distribution that I really used extensively, and I liked it a lot. I had a few issues with it, though, and have found Ubuntu to be more to my liking. I think that apt is a better package management tool than yum. In my case, there have been fewer hardware compatibility issues with Ubuntu than with Fedora. Last, but far from least, I found Ubuntu's included documentation, the Unofficial ubuntu Guide, and the Ubuntu Wiki to be very concise and easy to use--while there are great guides to FC out there, IMO none of them are as good as those for Ubuntu.

>

Great! Breezy worked very well with a minimum of tweaking.

I'm not a very advanced Linux user, so I'm not as aware of missing features as you probably will be. I will say that Ubuntu is a smaller download and is simpler to install than FC3, and that the software I needed to install was very easy to get (thanks to the aforementioned documentation). Like FC, Ubuntu doesn't come with proprietary software (Java, some codecs, etc.). My experience with installing these in Ubuntu was a lot better than it was with FC3, which is part of the reason I've had an easier time using Ubuntu. 

>

Very much so. The update manager is just a GUI for apt, and the command line updates aren't much different. Instead of switching to root and running "yum update" I type "sudo apt-get update" and enter my password. Other than that, it's essentially the same from an end-user standpoint.

That I can't tell you, but I know that the default installation of Ubuntu is pretty lean.

I recently switched from Fedora Core 4 to Breezy, and have tried Hoary before FC4.

Fedora is a very fantastic distro, with a much nicer installer than Ubuntu's.

But installing you do only once (if all goes well) and after that you can work, play and love your installation :)

As mentioned before in this topic up2date in Fedora is not nice to work with and falls way back when compared to synaptic or apt (which in fact are the same).

Also I think that there are many more programs in the Debian- or Ubuntu repositories than in those of Fedora.

As an example I found a progam kst in both Debian and Ubuntu (Hoary and Breezy), which I did not in Fedora, though maybe I did not search for it enough there ...

This is what I think of both great distro's :)

{I have one issue with Breezy at this moment: I can't type accented letters: no dead keys, it seems to me that xkeyboard-configure is broken. But I am sure this will be fixed, I now have a work around for this:
Type in a terminal

xmodmap -e "keycode 117 = Multi_key"

After that you can use your menu key to type accented letters by typing first the menu key, and then the accent key and the letter which is to be accented. This works until you shut down your computer}
