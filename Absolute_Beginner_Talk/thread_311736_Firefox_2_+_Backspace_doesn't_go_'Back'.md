### Post by stupidkid on 2006-12-03
Go to about:config and find the browser.backspace_action. Change the value from 1 to 0.

### Post by 505 on 2006-12-03
Type in the address bar: 'about:config'. In the filter type 'backspace', double-click the entry browser.backspace_action and change the value to 0.

See [here](http://kb.mozillazine.org/Browser.backspace_action) for more information.
