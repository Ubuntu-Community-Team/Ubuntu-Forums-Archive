### Post by Jussi Kukkonen on 2007-03-26
eds and some other evolution services run pretty much all the time so you have two sensible options:
1. Install all updates (otherwise you might miss security updates to services that run on your machine).
2. uninstall evolution altogether (this of course uninstalls ubuntu-desktop, which is a bad idea if you want to dist-upgrade to Feisty at some point).
