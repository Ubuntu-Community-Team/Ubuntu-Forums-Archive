### Post by Najand on 2006-09-14
You can find Dapper Drake Live+Installation CD at [http://www.ubuntu.com](http://www.ubuntu.com)
To install it you need first to backup your important information from
Red Hat and then reboot. Use the cd to boot to Ubuntu. When gnome starts
you can click on "Install" icon on your desktop to start installation. 
During installation you can select Red Hat partition to format it again 
and install ubuntu on it. 
If you are looking for other desktop environment like KDE or XCFE, download KUBUNTU or XUBUNTU instead of UBUNTU. You can also change them after installation of any of them.
