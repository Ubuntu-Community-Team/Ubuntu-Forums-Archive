### Post by Ribs on 2007-10-22
Don't panic. It's a built in program which isn't finished yet.

It should tie into the mozilla activex control (if installed) to view websites. The gui isn't finished yet.

The idea is to make applications which just blindly call iexplore.exe work.

There is an interesting thread about it here:
[http://www.winehq.org/pipermail/wine-users/2006-June/021970.html](http://www.winehq.org/pipermail/wine-users/2006-June/021970.html)
