### Post by whizbaby on 2006-10-23
@ terminal
You have to have a reference like
man bash  (typed in terminal. jump to the end to find builtin commands descriptions. BTW *man command* gives the manual of a command)
and
[http://tldp.org/LDP/abs/html/](http://tldp.org/LDP/abs/html/)

try to do some little tasks (e.g. get process ids, kill process, get formatted information from /var/log, restart xserver (or other deamons), tracking tasks to an own logfile and so on). Some useful commands are grep, cut, dmesg, ps aux (command ps with options xua), apt-get and apt-cache (or aptitude) for installing packages, nano (small simple text editor), emacs (not preinstalled, big-mean-supertank editor), sudo (give a command root privileges)

There are many more useful commands, a list would be too long. Finally you'll get the hang only by using it and trying to understand...
(and by googling up some stuff)
