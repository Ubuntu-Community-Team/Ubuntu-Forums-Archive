### Post by IYY on 2006-08-28
The system monitor is great, but if you're stuck in the CLI (or just want to do things the old fashioned way), there is the 'top' command to show the processes that take most CPU (or memory, depending on the arguments). Also, the command 'ps' can be used to kill processes, and 'killall <processname>' is commonly used to force-quit processes by name. 'kill <pid>' is used to force-quit by ID.
