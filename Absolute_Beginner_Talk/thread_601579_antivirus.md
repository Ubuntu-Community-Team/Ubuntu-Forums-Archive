### Post by Hendrixski on 2007-11-03
If your operating system is completely transparent, how is someone going to hide a virus in it?  If a program is going to do something immoral (like adware, spyware, or viruses) then it's going to do it hiding behind walls and in shadowy corners where nobody can see it.

Those spaces exist in Windows, not in LInux.  Thus, no viruses.

Don't believe me, google it.  There has not been a single **successful** Linux virus, ever, though many have been written, none have propagated.
