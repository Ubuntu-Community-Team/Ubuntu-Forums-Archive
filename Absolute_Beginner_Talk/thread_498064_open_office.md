### Post by davidsrsb on 2007-07-10
The older .ttf fonts work fine. Recent "opentype" or .otf fonts don't work in Linux OpenOffice, although they can be converted to .ttf and then work, using fontforge

This is a frequently reported OO bug that is causing a lot of problems now as Adobe only sell .otf these days.
