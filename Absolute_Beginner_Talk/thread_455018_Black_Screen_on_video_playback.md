### Post by whistlerspa on 2007-05-26
I had this problem using Beryl and effects desktop and solved it by doing the following : - 

Press                                    Alt + F2

In the input box type        gstreamer-properties

In the Video tab select    X Window System (No XV)

Save this setting and then try playing video
