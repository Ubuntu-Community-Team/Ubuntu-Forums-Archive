### Post by LuisGMarine on 2008-03-03
Extract them into your */home/username/.icons* folder.

Then select them by going to **System > Preferences > Appearance** and clicking the _custom_ button, then going over to "_Pointers_ "  and select it from there.

Or you can open up the appearance program and drop the tar.gz's in there and it will install them for ya.

### Post by zerhacke on 2008-03-03
> **cpu_medic said:**
> um i dont have a .icons folder, should i make one?

You have it, it's just that folders and files that start with a . are hidden from view normally.  If you enable hidden files in your nautilus file browser you will see them.  Alternatively, you can enable text input on the address bar and type /.icons at the end of your path (you'll start in ~) it will pop up.
