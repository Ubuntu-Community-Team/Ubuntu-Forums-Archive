### Post by Happy_Man on 2007-01-23
Well, the major differences are:

Ubuntu uses the Gnome desktop environment, Kubuntu uses KDE, and Xubuntu uses XFCE. Kubuntu is more "Windows-like", and makes for an easier transition. Ubuntu is easy to pick up, and Xubuntu is geared more for lower-end systems. In the end, though, the decision is yours: simply use the one you like the best! Try out all three if you want-just remember to uninstall the ones you don't need later.
