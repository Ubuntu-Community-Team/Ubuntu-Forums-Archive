### Post by yabbadabbadont on 2007-06-16
Looks like video card driver corruption to me.  Try using the open source driver and see if it still happens.  If so, then it is probably hardware, either the video card or the monitor.  If not, then it was the restricted driver.  In that case, see if you can find a different version or different settings that might help.
