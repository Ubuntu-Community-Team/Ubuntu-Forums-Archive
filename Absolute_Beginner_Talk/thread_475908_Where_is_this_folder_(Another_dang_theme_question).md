### Post by testube_babies on 2007-06-16
you can extract them into the .themes folder in your home folder (~/.themes NOT ~./themes).  It's a hidden folder, so in Nautilus hit CTRL+H to see it.

You have to make sure that the individual theme folders themselves go into ~/.themes, not one folder that contains the theme folders.
