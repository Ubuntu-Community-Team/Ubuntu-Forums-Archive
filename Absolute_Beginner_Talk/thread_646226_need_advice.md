### Post by 3rdalbum on 2007-12-21
I suggest you install the Ubuntu Desktop version. Once installed, go into Synaptic Package Manager, then the Edit menu and "Mark Packages By Task". Click "LAMP Server" and press Ok.

Once the packages have finished installing, you have a ready-made LAMP server platform. I think Ubuntu even starts Apache immediately for you. Have fun!
