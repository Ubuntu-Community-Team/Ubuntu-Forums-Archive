### Post by RomeReactor on 2007-12-10
Hi. As far as I can recall, there's no control panel in Dapper, but practically every administrative tool can be found in the **System** menu. To enable a second monitor, you would need to edit a file called **xorg.conf**; In Gutsy, there's already a utility for that purpose ("Screens and Graphics"), but I don't think it's available for Dapper, so maybe you could try an upgrade. As for editing the file to add a second monitor, I've never done that myself, but there are some threads around here discussing it. Sorry I can't be of more help.
