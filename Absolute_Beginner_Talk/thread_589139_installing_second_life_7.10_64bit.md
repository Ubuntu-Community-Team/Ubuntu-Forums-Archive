Have not tried this program myself, but there are instructions:

From the README.txt doc inside the extracted folder:

> 3. INSTALLING & RUNNING
-=-=-=-=-=-=-=-=-=-=-=-

The Second Life Linux client entirely runs out of the directory you have
unpacked it into - no installation step is required.

Run ./secondlife from the installation directory to start Second Life.

For in-world MOVIE PLAYBACK, you will need GStreamer 0.10 installed on your
system.  This is optional - it is not required for general client
functionality.  If you have GStreamer 0.10 installed, the selection of
in-world movies you can successfully play will depend on the GStreamer
plugins you have; if you cannot play a certain in-world movie then you are
probably missing the appropriate GStreamer plugin on your system - you may
be able to install it (see TROUBLESHOOTING).

User data is stored in the hidden directory ~/.secondlife by default; you may
override this location with the SECONDLIFE_USER_DIR environment variable if
you wish.

### Post by khabarkhan on 2007-11-08
I hope it helps you:

1-Extract your SL download folder on your desktop.
2-Open extract folder.
3-Double click on "second life" file.
4-New window appears and asks: "Do you want to run "second life", or display its contents?" , just click on "Run" . That's it. (I hope it works for you because you use ubuntu 64 bit!)

I run second life in Ubuntu 7.10 x86 - clean install on my notebook Dell Inspiron 6000.
My graphic card is Intel 915 , I run compiz without any problem,but when compiz is run and I run second life, in SL my screen  starts to jump for each click and motion!!! So I went to preferences (Ctrl+p) then went to "Graphics" tab and disable "Run in a window" in "Display Resolution" option. Then I selected "1280x800" resulotion . So graphic works good for me when compiz is run in Ubuntu in background.
