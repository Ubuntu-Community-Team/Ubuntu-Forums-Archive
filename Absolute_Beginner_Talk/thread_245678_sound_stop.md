### Post by janbockaert on 2006-08-28
because flash is in bad shape on Linux. it uses oss as soundarchitectures, where every other program (like amarok) needs alsa. Installing aoss from synaptic will solve some flash problems, but we have to wait untill flash 9 get released (real soon now) for linux before we will be able to get past this problem.

search for flash in this forum, and you will find a lot of threads, complaining about the same issue. Well, flash isn't open source, so what can you expect. :(
