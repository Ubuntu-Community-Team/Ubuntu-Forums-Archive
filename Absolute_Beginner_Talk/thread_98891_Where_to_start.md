### Post by deNoobius on 2005-12-04
As far as I've been able to tell, most of the user guide for 5.04 still applies:

[http://www.ubuntuguide.org/](http://www.ubuntuguide.org/)

For a good site to get you started on the command line and what it can do:

[http://www.linuxcommand.org/index.php](http://www.linuxcommand.org/index.php)

In addition to what Aysiu said on your specific question, in the Gnome file browser, click on the "File" menu at the top.  The option to create a folder will be there.  You can also go to the command line and use the "mkdir" command to make a new directory.  Type "man mkdir" for a description of how it works.
