### Post by rich.bradshaw on 2007-05-15
Welcome aboard!

I don't really know about your procmail problems, as I have not used it.

You won't be able to run Trillian in Linux, becuase that is a Windows program. Some programs make a version for Linux and Windows, but I don't think Trillian does.

You can try using Gaim, I prefer it to Trillian personally anyway. What would you like to do with the messenging? You can use MSN, AIM, Yahoo, GTalk, Jabber and loads of others in Gaim.

If you want videos on youtube to work in Firefox, you need Flash, like Windows, this doesn't come ready installed.

What version are you using?

If you click the K-menu, then click Help, this has help on every application. If you click the top entry "Kubuntu Documents" in help, then "Kubuntu System Documents" There will be a number at the top, mine says Kubuntu 7.04. What is yours?

If people mention the terminal, in KDE it is called a Konsole. To open a Konsole, (Click on the K menu, the System, then Konsole).

If you click the start menu, then Add/Remove programs, then you can find software to install. If you search for flash, you should be able to find it somewhere and install it.

If you want any software, this is where you can go to find it. On Linux it is all kept together so it is easy to find, not like on Windows where you have to look all over the internet for it. Add/Remove programs will help you to find what you need.
