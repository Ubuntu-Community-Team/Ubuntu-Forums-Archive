### Post by confused57 on 2006-11-06
See this website, especially the sections on "Planning Partitions"
and installing with the Desktop cd:
[http://www.psychocats.net/ubuntu/index.php](http://www.psychocats.net/ubuntu/index.php)

Here's another website for dualbooting:
[http://users.bigpond.net.au/hermanzone/](http://users.bigpond.net.au/hermanzone/)

You basically want to defragment your Windows partition a couple of times, before installing Ubuntu...then you can choose to resize the Windows partition, and install Ubuntu to the remaining free space.
   You definitely want to backup any critical data that you want to lose, before you install.
If you have a second hard drive, you could check out this link:
[http://www.ubuntuforums.org/showthread.php?t=179902](http://www.ubuntuforums.org/showthread.php?t=179902)
