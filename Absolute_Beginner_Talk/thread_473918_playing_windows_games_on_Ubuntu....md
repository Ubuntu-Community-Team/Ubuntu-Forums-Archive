### Post by Calash on 2007-06-14
You have a few options to look at.

Wine is a great program, but you will want to check with it's database to make sure it is compatible with your games.

Cedega is what I run, it seems to have a bit more compatibility that Wine, but it cost money.  It is still best to check the database of supported games first.

Unless I am mistaken there is no Virtualization product out there that support 3D acceleration, so that will bump a lot of games off the list.

Dual boot is not that hard, but you do have to be careful.  You do not want to accidentally delete the Ubuntu partition while installing XP.  You should be able to fix the GRUB boot loader once XP is installed....not sure the exact instructions but it is possible.
