### Post by renzokuken on 2007-05-16
ok so you are using the ati driver. it may be worth trying the radeon driver instead.

open your xorg.conf in gedit 

```
gksudo gedit /etc/X11/xorg.conf
```

and change the Device section from this

```
Section "Device"
Identifier "ATI Technologies Inc RV350 AP [Radeon 9600]"
Driver "[color=blue]ati[/color]"
BusID "PCI:1:0:0"
EndSection
```

to this
```
Section "Device"
Identifier "ATI Technologies Inc RV350 AP [Radeon 9600]"
Driver "[color=red]radeon[/color]"
BusID "PCI:1:0:0"
EndSection

```

save and close the file

press Ctrl+Alt+Backspace to restart X and see if google earth works any better with it.

if not i'm afraid your only option is to probably try and get fglrx working properly. how did you try and install fglrx?
