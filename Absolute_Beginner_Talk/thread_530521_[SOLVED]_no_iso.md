### Post by schorsch on 2007-08-20
The fact that the downloaded iso file is shown as a winrar file is caused by the extensions in windows. Somewhere in the past you have configured that iso files should be opened with winrar but an iso file is an iso file. Dot.
You have to burn an image, not a data disc. If you burned it as a data disc, there would be only one file on the cd, the iso file.
If you burned it as an image, the iso image will be extracted and the extracted files are on the cd. Nearly every cd burner application is capable of burning an image.
