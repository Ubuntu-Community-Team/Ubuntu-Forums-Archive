### Post by scrooge_74 on 2007-02-25
to restore your backup you can type:

cd /etc/X11
sudo cp xorg.conf_backup xorg.conf

also when trying to reconfigure the Xserver check which driver you are using, maybe you got the wrong one, you can try using a VESA driver as last resort to then start all over again.
