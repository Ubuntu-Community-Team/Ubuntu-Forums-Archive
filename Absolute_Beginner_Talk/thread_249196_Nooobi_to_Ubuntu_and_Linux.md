### Post by tribaal on 2006-09-02
What I do every time I install a new linux distro is to "build it up" to the sate of my previous OS, to learn a maximum about my new operating system.

My "challenges" are somthing like the following list:

- Get the resolution of the screen right (1440x990 or something).
- Get online (including with the wireless WPA card)
- Get the sound working if it isn't
- Burn a CD
- Code an audio CD to mp3
- Playback an mp3
- get 3d accelleration working
- play quake 3 on the net with the sound working :)

Once you're able to play Quake3 on your system I guess you've come to know pretty much all you need to be using linux on a daily basis... And it's not about the result, it's about learning how linux works in the process, too :)

Hope this sort of answers your questions...
