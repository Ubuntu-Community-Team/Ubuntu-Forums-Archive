### Post by sethmahoney on 2006-07-22
Ubuntu can run some Windows software if you install wine.  However, you shouldn't need most of that.  Anti-virus and anti-spyware software isn't really necessary, Ubuntu is already fairly well protected, but you can install firestarter if you feel you need a firewall.  Click on the system menu, then on Administration, then on Synaptic Package Manager.  Search around in there to get an idea of what is available to install on Ubuntu.  Generally, to install software, that's where you want to go.
