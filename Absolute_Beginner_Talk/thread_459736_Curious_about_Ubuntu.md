### Post by steve.horsley on 2007-05-31
Should run nicely. I take it you are prepared to remove windows and give the whole drive over to Linux? This would be the easiest way to install - just tell the installer to  rease and use the whole hard disk. I suggest you use the "alternate" installer CD - it tends to be less problematic than the "desktop" installer CD. You **will** want internet connectivity though, so you can download other software bits and pieces. Sharing files with windows isn't too hard, but will require a little research. 

Your biggest hurdle is that some things are done a different way than with windows, and it can be hard to unlearn some of the things that you thought were fundamental truths about how computers work. It should prove very interesting for you, but don't expect to pick it all up in a few days.
