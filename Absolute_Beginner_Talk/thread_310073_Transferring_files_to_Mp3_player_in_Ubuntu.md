### Post by JayTee on 2006-11-30
if your Lyra is recognized as a removable storage device and you can drag n drop in Windows then it should work fine in Ubuntu. I have a SanDisk 250 and it automounts the 2GB player as a removalble drive and I can drag mp3 files to it and from it with ease. I've never used a Lyra so first I'd check to make sure your storage system on it is formatted as FAT32. You can check this by right clicking on My Computer | Manage and then selecting Disk Managment from the treeview in the left pane. That will show all the recognized partitions and the partition type. If it's not showing up there I'd be wary of using it on Ubuntu. You could always boot from the LiveCD and then plug it in to see if it is recognized but you'd need to mount another drive to test copying to and from with drag n drop.
