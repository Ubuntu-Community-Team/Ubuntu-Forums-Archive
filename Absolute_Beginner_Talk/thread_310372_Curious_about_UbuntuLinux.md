### Post by lloyd_b on 2006-12-01
Two choices:

1.  Run the "Live CD" version of Ubuntu for a while, and see what you think.  With the "Live CD", it runs exclusively off of the CD-ROM, without having to mess with the hard drive.

2.  "Dual-boot".  With Dual-Boot, you can split the hard drive between Ubuntu and Windows, and each time you start the machine up pick which of the two you wish to run.  That way, if you really need Windows to run something, then you can.

It's possible to run many Windows programs under Linux, by using special utilties (Cedega for games, Wine for many programs, VMWare for just about anything).  Not all Windows programs will run well in such an environment, though, so it'll take some testing on your part to see if Linux will handle the programs that you need.  So long as you have the option to boot Windows if needed, this shouldn't be too much of a problem.

Lloyd B

### Post by CatKiller on 2006-12-01
Welcome to the community!

Windows programs will not work directly in Linux. You'd either need to find Linux equivalents or use a translation layer, like [WINE]("http://www.winehq.com/"). Some Windows programs work, and some don't. The standard Linux image programs are [the GIMP]("http://www.gimp.org/index.html") and [Inkscape]("http://www.inkscape.org/index.php"). Some people like them and some people don't.

There are some native Linux games, and some Windows games work through Wine. I don't know if Guild Wars is one of them.

Download and boot the Ubuntu ISO. You'll have a live environment to play with. It'll be slower from the cd, but it will give you an idea. If you like it, you can always dual boot - using your Windows install for the image things that can only be done in your Windows applications and playing games that only work in Windows, and using Ubuntu for Internet things, the image things that can only be done in your Ubuntu applications and playing the games that work better in Linux.

### Post by lumpki on 2006-12-01
Check the Wine app database to see about compatibility with your Windows apps and games. I don't recommend Cedega, Wine has surpassed it.
[http://appdb.winehq.org/](http://appdb.winehq.org/)

You'll probably want to keep Windows and dual-boot while you get used to Linux. If you find out you can do everything in Linux, either with equivalent apps or getting things set up with Wine, then later you can get rid of Windows altogether.

--
WINE: Wine Is Not an Emulator - it's more like a "compatibility layer". emulators are slow. windows apps on wine run almost as fast as natively.
