### Post by jerrylamos on 2007-06-06
Well, CD Live can get them started so they get the feel of Linux Ubuntu.  Next step might be dual boot, however that does involve XP disk cleanup, set paging to zero, and tedious defrag.  If there's some space left at the top, estimate it carefully, then with CD Live's sudo gparted they can resize XP down, then turn XP paging back on.  You can get a Ubuntu demo system on 4 GB, a pretty usable one with 10 GB.
