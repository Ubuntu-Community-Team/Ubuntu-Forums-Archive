### Post by bodhi.zazen on 2007-10-07
Unless you use qemu/vmware/virtualbox you can run either linux or windows, but not both at the same time.

Here is an install guide : [https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall)

During the install you will need to resize and partition your hard drive. This is done as a part of the installation with a program gparted which is included on the Ubuntu live CD.

So you can :

1. Run linux from a live CD. This will be slower but will not harm your hard drive. You might want to look at a lighter live CD like Wolvix

2. Install Ubuntu and dual boot.
