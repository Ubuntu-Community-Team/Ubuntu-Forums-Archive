### Post by jbrown96 on 2008-03-22
You can run it in a terminal with
```
nvclock
```

I've never used it, so I'm not sure about how you would set overclocking. I would reccomend using nvclock-gtk or nvclock-qt (both in Synaptic). The GTK version is native for Gnome (regular Ubuntu) and QT is native for KDE (Kubuntu).
Those programs seem to be easier. The only program I ever used was the nvidia-settings manager, which has an option for overclocking if you enable "cool bits" in your xorg config file. Google it and there should be directions. This worked great for me, except you have to agree to a license (about overclocking) and choose the values everytime you boot. I think there are ways to make the changes permanent but I never bothered. 

Hope this helps
