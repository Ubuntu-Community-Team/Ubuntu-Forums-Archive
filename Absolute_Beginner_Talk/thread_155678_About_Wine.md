### Post by pbaehr on 2006-04-05
False.

Wine can run many programs without much effort, but many applications rely on a complex set of dependencies. Some of these can be tweaked to run with a little effort. Others cannot be run at this time.

For example, Adobe Creative Suite 2 is currently on the list of software that won't work. If you can settle for Photoshop 7 (2 versions behind), however, that one will work.

Edit: If you're looking to see if something specific runs under WINE or just curious about what apps are currently supported check out their application database here: [http://appdb.winehq.org/](http://appdb.winehq.org/)
