### Post by w1zard on 2007-04-19
How do I change the default viewer for JPG files to gThumb permanently?

If I use the 'Open with' function it only works for that one file and defaults backs to the Image Viewer for the next file.

---

### Post by kyphi on 2007-04-19
Right click on the image -> goto Preferences -> select "Open With" and put a check mark next to your preferred application.
