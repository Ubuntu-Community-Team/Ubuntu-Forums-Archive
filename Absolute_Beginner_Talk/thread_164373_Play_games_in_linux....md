### Post by Imexius on 2006-04-22
If you want to play games like Guild Wars in linux you would have to purchase [Cedega]("http://www.transgaming.com/"). To install wine open up the terminal and type in:

> sudo apt-get install wine

To get limewire download the limewire [other] package. 
To install it (first make sure the limewire binary is in your home folder) open up a terminal and type in:

> sh /home/yourusername/LimeWire/runLime.sh
