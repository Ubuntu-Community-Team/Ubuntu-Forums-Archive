### Post by Daremo on 2006-05-24
Be sure to use the PPC version of Ubuntu, this is for Macs' with the G3, G4, G5 CPU. To boot from the CD: insert disc, restart the machine and hold down the "c" key on the keyboard while it powers up. This tells the Mac to boot from the CD drive...
