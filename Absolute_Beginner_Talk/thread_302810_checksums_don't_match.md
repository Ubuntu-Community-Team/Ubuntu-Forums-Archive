### Post by nickpaton on 2006-11-19
It seems unlikely that you have 7 downloads which all appear not to check.

It may be worth sacrificing one CD and burning a copy, and to then use the verify CD option when the CD first starts loading Live CD.  

If that checks out then your original md5 checksums should also be OK.  If not then there is a problem somewhere.

When you burn the CD, make sure you use a good quality CD-R (NOT CD-RW) and burn really slowly, say x10 or slower.
