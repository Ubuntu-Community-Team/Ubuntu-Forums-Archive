### Post by Rocket2DMn on 2008-01-31
No anti-virus is needed since there no are no viruses in the wild that affect linux.  Even if one were to come around, a fix would be released so quickly the virus would disappear.  Most AV clients for linux just scan for windows viruses to prevent you from passing on viruses to windows users, but most of us believe it is up to windows users to protect their own systems - don't waste your system resources on it!
iptables is a built in firewall for linux, and you can get a GUI to fiddle with it called "firestarter" which is available in the Ubuntu repositories, so it's easy to install.  Normally you don't need it, but it's there if you want it.
