### Post by erginemr on 2008-02-12
1- Have you got a wireless modem? If so, did you try using the technology "ndiswrapper", which allows installation of wireless modems using their windows "*.inf" drivers. "ndiswrapper-gtk" is the name of the package:
[https://help.ubuntu.com/community/WifiDocs/WirelessTroubleShootingGuide](https://help.ubuntu.com/community/WifiDocs/WirelessTroubleShootingGuide)

These are a bit more advanced, you should refer to them as a last resort:

2- Are you dual booting with Windows (which is connected to Internet) on the same machine? If that is the case, you can try "nonetdeb":
[http://ubuntuforums.org/showthread.php?t=572819&highlight=dependencies+offline+download+packages&page=2](http://ubuntuforums.org/showthread.php?t=572819&highlight=dependencies+offline+download+packages&page=2)

3- Or better, you can install VirtualBox ([http://www.virtualbox.org/](http://www.virtualbox.org/)) under Windows, then install Ubuntu onto a virtual harddisk, update & upgrade your fake Ubuntu installation, install packages / themes to your taste. When you are done, take all debian (*.deb) packages from "/var/cache/apt/archives/" to your USB stick, boot into Ubuntu, copy the debian packages to a folder on your hard drive and install them all with:
```
cd ~/your_new_folder
sudo dpkg -i *.deb
```

Again, these are cumbersome so are prone to errors, so finding a way for your internet (say,via ndiswrapper) would be much easier.
