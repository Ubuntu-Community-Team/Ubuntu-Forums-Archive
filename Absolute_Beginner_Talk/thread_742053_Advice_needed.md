### Post by Zralou on 2008-04-01
Linux usually has an 'alternative' to most windows software if you search around.

Linux has a learning curve, just like any OS when you first try it out.  The hardest part of learning linux is forgetting everything you know about windows, its different, not nesessarily harder.
I installed Kubuntu last week as a hardened windows user, and straight off the bat I had hardware installation problems, but over the last week I have learned more about operating systems than I ever learned with 10 yrs of windows.

Linux can be scary to a windows user, but it can be mastered given the time and patience, and the main difference between windows and linux: Nowhere in the windows community will you find anyone prepared to spend 4 days walking a total novice through their problems, but on linux you will, I know that to be true, because that's just happened with me.
