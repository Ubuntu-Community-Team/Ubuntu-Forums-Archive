### Post by Albalabs on 2006-09-01
I also have the same problem you have with shutting down.  I have an SB86i Shuttle box, with the i915g chipset.  I have heard that with these motherboards there is a Bios setting regarding NX settings in the CPU that needs to be enabled.  I enabled this, but still no shutdown.

Check to see if your motherboard uses the 915 Chipset with the Integrated Graphics.
