### Post by CybaCowboy on 2006-05-11
Okay,

Before I start, I don't intend to start a war and this is stricly only for everyone's individual opinions!


Now that that is out of the way...

Which do you prefer and why - the [GNU Network Object Model Environment (GNOME)](http://www.linuxnewbieguide.org/chap6.php) or the [K Desktop Evironment (KDE)](http://www.linuxnewbieguide.org/chap7.php)?

I would imagine that each has its advantages and disadvantages, but what are they?


For those that don't already know, GNOME and KDE are different *interfaces* to the *same* operating system, in other words, each changes the way that the operating system looks and behaves (Ubuntu uses the GNOME interface by default whilst Kubuntu uses the KDE interface by default, although you can easily change between interfaces regardless of the version you start with)...

Read more [here (GNOME)](http://www.linuxnewbieguide.org/chap6.php) and [here (KDE)](http://www.linuxnewbieguide.org/chap7.php).

### Post by ComplexNumber on 2006-05-11
i prefer gnome for a multitude of reasons. i used to use KDE all the time in my first years with linux because it was easier to use for me at that time. KDE always felt like a windows carbon copy in its way of going about things, the functionality, and even the wording used in many plaes throughout. i was fully familiar with windows in those days, so using KDE felt much more comfortable than gnome. although, even then,  i always thought gnome to have more character. i've always felt that gnome is more unixy(as someone has already mentioned) and has more individuality and inventiveness. in many ways, i think KDE has gone worse while gnome has got much better, but still kept its character and individuality. there are many other reasons that i've discovered over the years. some major, some minor. but because of the above and other reasons, i've found that KDE just isn't me. 

KDE has a great many strong points and i can see why people like it, but the more i used it the more i grew tired of it.
