### Post by KIAaze on 2007-11-15
How did you install it?
The .zip file from the official site sems to be only for Windows.
Try installing wine and running transfusion.exe with it.
```
sudo apt-get install wine
cd <extracted zip file>
wine transfusion.exe

```

or
```
chmod 755  transfusion.exe
./transfusion.exe

```

It seems to use a lot of RAM however.
