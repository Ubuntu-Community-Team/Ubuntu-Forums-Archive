### Post by marufaberlin on 2008-02-07
Do you want to manage databases or MySQL itself?

If you want to manage the databases and users, etc... then you should have a look at phpmyadmin ([http://sourceforge.net/projects/phpmyadmin/](http://sourceforge.net/projects/phpmyadmin/)). If you want to change the server configuration, any configuration file editor should do the job.
