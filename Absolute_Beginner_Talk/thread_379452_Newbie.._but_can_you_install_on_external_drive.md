### Post by dstew on 2007-03-08
What you heard is not quite true. You can certainly partition a drive that has an operating system on it. What you do is defragment the drive (using the Windows defragmentation program under Programs->Accessories->System.) This shrinks the windows part down to the smallest size possible, and puts it at the beginning of the drive. After you defragment the drive, you have all the unused space at the end of the drive. You can make a new partition out of this unused space without affecting your Windows operating system.

You can also get an external drive, like a USB drive, and install linux on that.

### Post by Pikestaff on 2007-03-08
The Ubuntu CD can automatically partition your drive for you when you install, even if you already have an OS on your computer... it will get to a part during the instillation where it asks how you want to partition your drive and you can just tell it how much room you want to give it and then it does it all by itself.

That said, I wouldn't recommend doing it without backing stuff up first, and finding another way to install Windows if you have to since you don't have the CD... because although it usually works fine it can indeed go wrong, it's gone wrong for me before.
