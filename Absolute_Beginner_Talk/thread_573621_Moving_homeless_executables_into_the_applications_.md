### Post by pollywog on 2007-10-11
Is it possible to manually place any program into the gnome *Applications* menu? I have accumulated a bunch of programs in user/bin that I don't want to browse for. I also don't want to crowd my desktop with launchers. *Debian Menu* is much better, but still doesn't show some programs. I also know how to use the ala carte menu configuration tool, but that only shows certain applications as well. Does anyone out there really know how to edit the Applications menu?-:)

---

### Post by conehead77 on 2007-10-11
1. right click at the menu symbol, choose "edit menu"
2. choose the category you want the new entry in
3. select "new entry"
4. type the command in the command field (xterm if you want to start a terminal for example)
5. select a symbol
6. enjoy :)
