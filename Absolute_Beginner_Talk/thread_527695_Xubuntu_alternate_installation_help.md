---

### Post by aysiu on 2007-08-16
Sounds like an internet connection issue. When you get to the part of the installation where it's trying to set up online repositories, tell it to not bother to look (i.e., look for packages on the installation CD instead of the internet).

Once it's installed, you can worry about setting up the proper internet repository servers.
