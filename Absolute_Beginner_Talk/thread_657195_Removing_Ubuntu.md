### Post by Niniel on 2008-01-03
You will need to change the boot sequence in the BIOS so that CD is the first option. Or actually, in the case of Win98, you probably need to boot from a floppy, so make sure that's first, then boot from the floppy and continue from there. 
If you don't have boot disks, you can find images on the web easily, just search for the "Win 98 boot disks".

Alternatively, you could just tell this "young person" to not look a gift horse in the mouth and that he should be using Ubuntu instead since that's alot better than Win98.
