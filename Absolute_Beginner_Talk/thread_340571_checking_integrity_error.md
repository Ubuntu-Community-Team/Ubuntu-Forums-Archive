### Post by irish_flu on 2007-01-17
Yeah, it sounds like there was an error either during the downloading process or the burning process.

Try to burn another CD at a slower speed (heck, maybe you just got a bum disk) and if you still get the error you might consider downloading the file again.

Before you re-download, check out this page:

[http://www.psychocats.net/ubuntu/iso](http://www.psychocats.net/ubuntu/iso)

 it tells you how to validate the checksum of the ISO you downloaded, so that you know where the problem is at (if the checksum is OK, it's the CD; if the checksum is not OK, you need to download again).
