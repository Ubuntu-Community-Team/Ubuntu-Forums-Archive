### Post by unutbu on 2008-03-11
Are you trying to set up a dual boot (Windows+Linux) system, or a single boot (Linux) system? If you want dual boot, you must install Windows first.

If you want a single boot system, I think you can just let Ubuntu do a 'guided partition' which makes all the decisions for you. If you don't have a specific reason why you want to design your own partitions, perhaps you should just let Ubuntu decide.
