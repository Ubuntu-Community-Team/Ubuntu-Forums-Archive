### Post by mrboojive on 2008-04-06
Your other systems should run great, even with 3D effects, as long as they've got some kind of video card. The performance hit from upgrading Ubuntu is nothing like going from XP to Vista. Even the 900 MHz should be good but you should probably stick to Xubuntu there.

### Post by Paqman on 2008-04-06
Yeah, i'd stick with Xubuntu for the Pentium. I'm running the desktop version of Xubuntu Hardy at home, and it seems very stable.

The only difference between the versions is the desktop environment (and associated services). Since you're running it as a file server you aren't (i'm guessing) interacting with the desktop much, so it doesn't matter much what desktop you use. It's perfectly possible to have all three DEs installed anyway, and just pick whichever one you feel like at login.
