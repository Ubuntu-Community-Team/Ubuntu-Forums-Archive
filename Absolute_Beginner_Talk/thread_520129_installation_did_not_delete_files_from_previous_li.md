### Post by Rocket2DMn on 2007-08-07
Do you have a second hard drive in the computer?  You can format that with gparted if you do.  Otherwise, when you selected the location to install Ubuntu during the installation process, did you tell it to use the whole hard drive or just a particular partition?  If you selected to use the WHOLE drive, there will be nothing left from your previous system.
