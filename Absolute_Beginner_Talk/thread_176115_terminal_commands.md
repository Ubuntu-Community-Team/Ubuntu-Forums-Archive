### Post by skippy81 on 2006-05-14
and use apropos and man a lot.  If you want to know a list of commands which deal with dolphins use 

> apropos dolphins

if you find a command called 'echo' and want to know more about it, then use:

> man echo

CONTROL+Z is a useful key combo, if you get stuck on a man page control-z will take you out of it.  Its the 'break' key.
