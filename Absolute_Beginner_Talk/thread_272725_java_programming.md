### Post by szf on 2006-10-06
You can download BlueJ to get a start on OOP. It was developed expressly for teaching that, so you'll have a user community for leverage on the things that don't stick easily.

RE: what is java mainly used for? 

A lot of web infrastructure (i.e. "Enterprisey" applications, Web-based Portals/Forums, Web Services) frequently select the technology. You will also find it in x-platform applications (i.e. Eclipse, Open Office/Star Office(?)) and the slower interfaces in a cell phone...
