### Post by felicity on 2007-05-07
If you have a Windows disc you can run Virtualbox for free and those programs work great through that.

You can also use Wine and they should work OK, though I haven't tested them personally in Wine.

But I recommend you check out the free alternative programs to Photoshop and Dreamweaver instead. There are quite a few good ones that, with a little practice, will allow you to forget you ever needed Photoshop and Dreamweaver.

### Post by TURNER on 2007-05-07
some handy alternatives

Photoshop = [The Gimp]("http://www.gimp.org/")

Dreamweaver = [Bluefish]("http://bluefish.openoffice.nl/index.html")

The Gimp also has an addon/plugin which mocks the gui of photoshop, perhaps handy for you if you are not so good with change :) 

As an html editor I have recommended Bluefish, although this doesn't allow WYSIWYG facilities, it is great if you hand code.

If you would like to look at other options (and plenty exist!) take a look at [Linuxsoft.cz]("http://www.linuxsoft.cz/en/")
