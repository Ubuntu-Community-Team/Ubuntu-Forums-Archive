AeThEr777

If it's a CD ISO you should burn it to a CD, and if it's a DVD ISO you should burn it to a DVD. Also, note that an ISO is an image file, so you need to burn the *image* not just copy the file onto a CD.

If you did that correctly, the next thing you should do is check the CD for errors - it's one of the options you should get when you boot up with the CD in the drive.
