### Post by igknighted on 2006-10-16
First of all, calm down.  There are tons of people here to help, large bold fonts are not necessary.  Second, and most importantly, there are millions of software packages and drivers available for linux, so you might want to narrow your request a little.

The most important tool to learn right now is "add/remove".  If you click on the applications menu, the very last item in the menu is "add/remove".  Click it and you will find a vast array of software that you can download and use.  It is sorted into nice categories so if you just know the type of software you need, you can browse through several programs.  This is why installing software is sooooo much easier in Linux than windows.  Almost all software you will ever need is maintained in repositories, basically online storage warehouses which are set up so only legitimate software can get in.  No need to worry about whether that hot new program you downloaded is full of spyware.

As for drivers, if it works don't try to install any new drivers, but if it's not working post what isn't working and someone will help you out, drivers usually have their own specific fix.
