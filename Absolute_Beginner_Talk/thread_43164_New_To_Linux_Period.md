### Post by TristanMike on 2005-06-21
I'm still a week new, just got my cd's a few weeks ago (I ordered them in March and got them early June and I live in the armpit of Canada :) ) and after some preperation I installed Ubuntu with a little confusion and some help from the wonderful people here.  ;-)  and I love it, since installation, I haven't really looked back at Windon'ts,  Ubuntu does everything that the *other* OS does, save a few things I have to learn.  My advice....

1) Read up on what the Ubuntu versions of the Windows software are.  That way when "zilpha435" talks about gxine or XMMS, you're not left scratching your head wondering what these programs are.  A good list can be found as a "sticky" at the top of this forum.

2) Read up on the command line.  I like here [http://www.tuxfiles.org/](http://www.tuxfiles.org/) and [http://www.linuxcommand.org/index.php](http://www.linuxcommand.org/index.php) which I believe someone already mentioned.  That way you'll understand, well, the command line a little at first and so you'll understand what sudo, mv. cp, and others mean.  Plus this place [http://ubuntuguide.org/](http://ubuntuguide.org/) wont be as head-banging.

3) Try some keywords in the search, it will be a vaild companion.  Something should give a direction to go and if you're still stuck, I'm sure someone can help or direct you to where you should go.

4)Don't forget to bookmark pages.  You'll be using them for reference.  Dont forget this forum, it's invaluable.

If you know nothing about code, prepare for a little time, a little learning. and a whole lotta fun.  I've found that Linux can be somewhat intimidating, but it's not so bad with a little homework and some patience. I hope you have as much fun as I've been having this past week.  Ubuntu runs awesome!!!!!!
