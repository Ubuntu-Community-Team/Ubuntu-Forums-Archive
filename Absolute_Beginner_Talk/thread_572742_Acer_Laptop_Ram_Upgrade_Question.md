### Post by LowSky on 2007-10-10
it should work, just make sure you use the correct type of RAM. The only issue is you cannot use the dual channel capaiblities of your motherboard, but you weren't alreay so it is rather a moot issue.

Acer says you need  200pin PC2-4200 @ 533 Mhz, should run you under $50.00
