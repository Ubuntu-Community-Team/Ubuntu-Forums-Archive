### Post by 3rdalbum on 2006-11-25
AV: If that stands for Anti-Virus, then you don't need one.

Ventrilo: Apparantly, they're working on a 32-bit Linux version now. I recommend running 32-bit Ubuntu anyway, unless you need to access more than 4 gigs of RAM.

Winamp: XMMS. Or you could go up a few notches to a proper music management program, of which there are loads for Linux: Amarok is the most popular, but there's also Exaile, Quod Libet, Rhythmbox, Banshee...

MMOs... does that stand for Massively Multiplayer Online (games)? World Of Warcraft can be run on Linux through Wine or Cedega, and now Blizzard has stopped banning Linux users ;-)  There are a number of others with Linux clients, but the only one whose name I can remember is Eternal Lands.

To answer your real question, there are ways to run *some* Windows programs. Wine is a program that emulates the Windows DLLs, and there are a couple of programs I've got which work well. Crossover Linux is an improved,  paid version of Wine. Cedega is a version specifically for gaming.

If worst comes to worst, you could install VMWare (a kind of emulator, but very fast) and run Windows XP in it. Not sure how you'd go running XP 64-bit in VMWare on a 32-bit Ubuntu, but from what I hear certain processors can achieve this.
