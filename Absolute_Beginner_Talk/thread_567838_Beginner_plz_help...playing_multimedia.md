### Post by Jimmyfj on 2007-10-05
You need to install som extra stuff in order to do that.

```
sudo apt-get install ubuntu-restricted-extra
```

Or go to Programs/Add/Remove Programs/Music & Videos and install the codecs from there.

Ubuntu should offer you to download the needed codecs but does not always do that.

In Programs/Add/Roemove Programs you'll find lots og media players too
