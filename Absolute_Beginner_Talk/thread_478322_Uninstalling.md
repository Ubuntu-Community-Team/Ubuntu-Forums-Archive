### Post by Golyadkin on 2007-06-19
the short answer is to use "Add / Remove Programs" in the menu on the top left, or go to System > Synaptic Package Manager" to mark packages for uninstallation.

If you want an extensive detailed guide on installing and uninstalling, [click here]("http://monkeyblog.org/ubuntu/installing/").

If you want to automatically remove unneeded packages you can run this command in the terminal / console:
> 
sudo apt-get autoremove
