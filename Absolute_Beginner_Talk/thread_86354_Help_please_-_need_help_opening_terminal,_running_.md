### Post by TDonaghe on 2005-11-04
Oh my goodness, I figured it out!  You have to do ctl+alt+spacebar, then release the spacebar while holding onto ctl+alt and then typing F2 and then releasing all three.  Only, it's not just any ctl+alt - only one of the alt keys does the trick.
