### Post by fourchannel on 2006-08-07
Well, I would go about copying my windows files like this: Open up the file browser and go to 'Computer' I would find my windows directories that I want to copy and "Ctrl-C" copy them.

I would then go to my Ubuntu home folder and make a new folder called windows or something.

In my experience home folders can get pretty crazy unless you try and impose some directory order upon them

For example, I would make folders such as *download, [music or audio], images, files, [programs or the ehaulted 'bin'], documents* and then start moving the stuff in the windows clump into those directories. That way it won't be nearly as hard to find stuff, explore the crap that you already have, and keep order to your home folder as you add more files without having to panic about what new folder to make and how should i label this problem.
