### Post by aysiu on 2007-02-15
Well, if you want to give Firefox 2.0, give this a shot:
[http://www.psychocats.net/ubuntu/firefox](http://www.psychocats.net/ubuntu/firefox)

It also has an uninstall script, too, in case you decide 1.5 is better.

Or, actually, you don't have to uninstall it. If you use the script, the command ```
firefox
``` will launch 2.0 and the command ```
firefox.ubuntu
``` will launch 1.5.
