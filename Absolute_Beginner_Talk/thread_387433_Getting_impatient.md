### Post by eljalill on 2007-03-18
Generally you run executable file with
```
./.run
```
For packages and application to be listen in add/remove or synaptics, you have to have the repositories enabled, either in "settings" or through editing your "sources.list" . For Vega Strike you need universe enabled.
I am not quite sure about you archive manager, mine runs fine. What kind of file have you downloaded? for .zip or .rar you need to install extra packages.
