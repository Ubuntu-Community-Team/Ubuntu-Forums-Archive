### Post by Weka on 2007-01-20
In Kubuntu I found them here:

/home/<username>/.kde/share/apps/kmail/mail/

When looking for them I had 'show hidden files' turned on.

---

### Post by rhb on 2007-01-20
an easy way to find the files using the command line is:

```
locate kmail
```

That should find most files related to kmail, whether they are hidden or not.  One of the directories within your home directory will likely to be the one you want.
