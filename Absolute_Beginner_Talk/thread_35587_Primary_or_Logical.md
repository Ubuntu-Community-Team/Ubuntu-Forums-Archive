### Post by Amarack on 2005-05-19
You can have only four primary partitions on a hard drive. However, in place of one of these four primary you can have an extended partition. In the extended partition you can have more partitions (logical partitions) which allows you to have more than just 4 partitions total on a single hard drive.

Using logical partitions can be useful when you need more than 4 partitions. For instance if you had a large hard drive, and wanted 2 partitions for windows and then needed 3 for linux. You could use logical partitions to allow this.
