### Post by kscott121 on 2007-08-31
I thought Dansguardian is configurable on a user by user basis.  Not sure how that is impacted by the nice GUI that ubuntu CE has.  Have you created a new user and set the filtering to a different level to see if that works?
Also the Dansguardian docs on their page tell you how to replace the current "forbidden" page with you of you own.  I did it on a standard dansguardian implementation a few years ago and it is quite easy.  Just use NVU or the Mozilla web composer (i think it has a new name now)  to modify the forbidden page how ever you want to.
I used phrases like 
"You have attempted to access an unauthorized page.  Your transgression is noted and the web police are currently enroute to your location to poundify your head, seize your lunch money and tell your mother"!!  
Hope that helps.
