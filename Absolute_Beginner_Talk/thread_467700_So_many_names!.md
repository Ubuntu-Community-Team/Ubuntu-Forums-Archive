### Post by diatribe on 2007-06-08
nautilus draws the desktop in gnome, so removing it would be bad; but you could install another file manager (thunar is nice) and it would be ok

also gtk-themes change the actual window controls; ie everything that isn't window borders.  metacity themes are what control the window borders.  even is you use beryl/compiz, you will still need a gtk theme
