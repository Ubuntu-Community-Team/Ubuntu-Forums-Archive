### Post by dhdallas on 2006-11-07

---

### Post by 23meg on 2006-11-07
You may be trying to run more than one of the following:

- an *apt-get* or *aptitude* command
- the update manager
- Synaptic or Adept
- Add/Remove Applications

You can only run one at a time.
