### Post by rccharles on 2007-03-07
> **strivehosting said:**
> I select use the largest continuous free space? Will windows get deleted? Or will i still be able to choose which one i want from the boot menu

There are two contexts for the question "largest continuous free space."

1) Within a file system.  This would be space in a partition that is not used by data and programs.
     You will recall that a harddrive is divided into partitions. Each partition contains a file system.

2) On the harddisk.  This would be space on the harddrive that has not been assigned to a partition.

When you are installing Ubuntu, the Ubuntu installation program ask you where you want to put Ubuntu.  One of the options is to use space on your harddrive that has not been assigned to a partition.  The Ubuntu installer uses the words "use the largest continuous free space."

Picking this option will not effect your other partitions such as the Windows partition.

I'd used this option on install and found it easy.

------------------------

You need to backup all your important data before partitioning.
