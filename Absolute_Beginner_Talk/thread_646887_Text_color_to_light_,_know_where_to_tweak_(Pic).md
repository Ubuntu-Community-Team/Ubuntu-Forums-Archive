### Post by JillSwift on 2007-12-21
The GTK theme you're using asks for light text to use against the dark background. This combined with the fact web page designers tend to assume too much about folk's settings gives you that result.

[SIZE=3][SIZE=2]To fix this: [/SIZE]In Firefox, Edit menu --> Preferences.
"Content" icon.
Under "Fonts and Colors" click the "Colors..." button.
In the Colors dialog, un-check "Use system colors".
Click OK on the colors dialog and Close in the preferences dialog, then re-start Firefox.[/SIZE]

Things should improve then =^_^=
