### Post by Tomosaur on 2007-01-29
The standard Ubuntu installation uses Gnome, a desktop environment which is supposed to be clean, efficient, and usable rather than flashy. If it's eye-candy you're going to judge it on, you should use Kubuntu as your standard install, or get Beryl up and running. You may also want to look into e17, yet another desktop environment which is focused on a mergence between outstanding eye-candy and utility. It's really very good, but is still in development and is quite unstable.

You can install any number of desktop environments and choose which one you want to use when you log in. Ubuntu and Linux in general are far, far more customisable than Windows and Mac.
