### Post by eph1973 on 2007-09-23
Yes, it's possible.  Unless you have a spare HD or some unused partition space you will have to make some room for Linux.  [This site]("http://www.linuxdevcenter.com/pub/a/linux/2006/05/08/dual-boot-laptop.html?page=1") has information you will probably want to know about how to go about making a new partition to prepare for the install.  If you are still unclear about how exactly to install Ubuntu once you have made the partition for it to run, you can check out the [apcmag site.]("http://apcmag.com/6101/dualboot_windows_xp_and_ubuntu")
Hope this helps, good luck!
