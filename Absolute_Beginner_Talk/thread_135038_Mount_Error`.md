### Post by heimo on 2006-02-23
No problems unless you also got some ugly warnings and errors. Just some automatic maintenance done periodically to make sure your system works as expected - checking the filesystem for any potential problems. This happens after a set number of days or "mounts", which translates roughly to number of reboots. Takes a little bit of your time every ~30 reboots (can be changed per filesystem if needed), but you don't need to do anything.
