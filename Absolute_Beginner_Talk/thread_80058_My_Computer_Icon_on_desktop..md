Is their any way I can get the Gnome version of the my computer icon on the desktop?

---

### Post by davmac on 2005-10-21
Yes there is.

Applications -> System Tools -> Configuration Editor

Expand "apps" then "nautilus" and then select "desktop". Make sure the computer_icon_visible is checked and that should be it.

Hope this helps,

Dave Mac
