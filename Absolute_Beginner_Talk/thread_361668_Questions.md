> **abc006 said:**
> What's the difference between ubuntu, and say, xubuntu?

Unlike Windows, the fact that you have access to the components that make up the OS means you can change it, and that's what's happened.  In essence it's all the same, but some people like their Desktops to behave one way, using software called Gnome, which is what Ubuntu runs on, and other like XFCE, which is what Xubuntu runs on.   If you're not sure what the differences are, rather stick with Ubuntu, and if you start asking the right questions someone will direct you to start using something else, e.g. I'm running Ubuntu on my Pentium III/750mb laptop and it works ok, but is REALLY slow ... Try Xubuntu, which is "lighter" ... or How can I make Ubuntu look more like Mac OS ... try Kubuntu ... 

> **abc006 said:**
> Also, will my files my transfered over to ubuntu in the same format as windows xp? Meaning, all in the same folders, "my documents" for different users, and the same drives, etc.?


Err No, and Yes - Ubuntu will create a "home" folder for each user which will have their documents etc. in there.  However, you should be able to (with little effort, pending on if your windows is on an ntfs or fat32 partition) also open your Windows files.  

Ubuntu also lables your drives differently, do "c:\documents and settings" will most likely be called "\winnt\documents and settings" - Different, yet still the same?

Some games work great (even better) under Linux, depending on your graphics hardware etc.  Some work pretty much the same, and can be easy or hard to get going.  Some just never do... However, you may be surprised to fine a whole new bunch of games with Ubuntu that aren't available for Windows - Best to snoop around the Ubuntu Gamers forum for more details on that ...
