### Post by _sAm_ on 2007-04-24
My program makes no sounds when opens/closes, only if there is a bug(like program crash). That sound is controlled from "System settings > Notifications. 

But this stupid dancing/jumping graphic is terrible, hope someone can guide me to where I can shut it off.

Edit; Found the solution
Alt-F2 (run command) and type kcontrol. Go to  appearence & themes>launch feedback, there is an option here to change 'bouncing cursor' to 'no busy cursor'.
