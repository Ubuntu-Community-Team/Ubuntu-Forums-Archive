### Post by Footissimo on 2006-07-26
NP, just wanted to make sure I was answering correctly....as PCMCIA is a laptop thing, I wouldn't be terribly concerned that it fails running the service ([wikip about PCMCIA](http://en.wikipedia.org/wiki/Pcmcia)).  To stop it from trying to run, you could try BUM (Boot-up manager) - it's available via synaptic and there is a forum section for it on here.  There should be an entry for PCMCIA in there which you can safely de-activate and disable if you're not using a laptop. :)
