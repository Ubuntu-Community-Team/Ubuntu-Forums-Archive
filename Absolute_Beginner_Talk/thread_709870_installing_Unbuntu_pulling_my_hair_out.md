### Post by oakgrove on 2008-02-27
In my experience, some distros just don't want to work with some hardware.  My suggestion would be to download the Ubuntu "alternate install" CD which features a non-graphical installer so instead of booting up all the way to a desktop like with the regular CD, you can just boot up and go directly to the install process.  Another suggestion would be to try a different distro altogether.  Puppy Linux works really well on older hardware and I have an old Toshiba laptop that while refusing to boot Ubuntu has a working install of Puppy Linux on it right now.
