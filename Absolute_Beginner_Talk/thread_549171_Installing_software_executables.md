### Post by hyper_ch on 2007-09-12
those were made for Windows and not for Linux so you can't just use them. There are options like WINE that lets you run windows programs on linux or as alternative you can use a virtual machine e.g. VmWare or VBox to run a virtual Windows where you can install them.

And for many programs that you have on windows you have in the repositories (and additional repositories that are not activated by default) tons of software and for most things you'll find an equivalent...
