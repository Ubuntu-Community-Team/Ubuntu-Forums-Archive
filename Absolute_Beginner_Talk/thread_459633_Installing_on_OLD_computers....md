### Post by NICU on 2007-05-30
I would suggest using Xubuntu on old PCs.  I've run it on quite a few old PCs and on virtual machines with very little RAM.  If Xubuntu runs too slowly you can always try [PuppyLinux ]("http://www.puppylinux.org/user/viewpage.php?page_id=1")or [DamnSmallLinux]("http://www.damnsmalllinux.org/").  Keep checking back on the forums if you come across any problems.
