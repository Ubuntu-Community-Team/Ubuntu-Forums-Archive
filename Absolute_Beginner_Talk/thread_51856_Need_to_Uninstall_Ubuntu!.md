### Post by doobit on 2005-07-27
Sometimes you can't do it in BIOS, but the computer will allow you to do it if you hit the Esc key right after the RAM check. Then you should get a list of bootable devices. Pick CDROM, and make sure the WinXP CD is in the tray. It should then boot from the CD and tell you that the Hard Disk is not formatted in a way it understands. Then you get choices to delete partitions. If I were you I would leave one spare partition besides the one you set up and format for NTFS. That way you can make a dual boot computer later when you feel more comfortable with Linux.
