### Post by southernman on 2007-05-31
Welcome to Ubuntu, Pilchard.

You are correct the app to run windows apps in linux is called [Wine]("http://appdb.winehq.org/"). This link will take you to the apps database of which I do see some mention of a starcraft.

Now, about the "console" thing... It's a big black hole on your screen. Yeah, it's scary for us new comers but relax! You won't have to use the console unless you really botch things up with your ubuntu installation. What you'll wind up having to use from time to time, is called the "terminal." It's basically a gui version of the console. It's not so bad.

Just do your homework and NEVER do rm -fr if anyone tells you to do so!!! EVER! ;)

### Post by Outrunner on 2007-05-31
There's a different way of mounting an .iso file in Linux

```
sudo mkdir /iso
```

```
sudo mount -o loop -t iso9660 /path/to/iso/file /iso
```
