### Post by MONODA on 2007-12-18
What happens when you delete a file is the space that it is taking up is marked as free so when a file needs to be created, it is created over it so if you have bee using your computer for a while after this and have been putting files on it then your deleted files may have been overwritten. Just in case this happens again, I suggest you download the program flyback, it backs up th directories you specify every so often so in case you delete a fiel on accident you can recover it.(flyback is like apple time machine) I also use it and it has saved my life many times.
