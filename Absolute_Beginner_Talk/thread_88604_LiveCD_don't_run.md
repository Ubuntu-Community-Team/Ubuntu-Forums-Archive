### Post by michael_salcher on 2005-11-11
see if you can boot from any other device besides a hard drive. e.g. try a floppy drive. once this works. try the cd rom drive again. if you find that the cd rom drive really does not boot ubuntu, see if you can boot anything from cd rom (e.g. windows). if windows works, check md5 checksum for the downloaded iso.
