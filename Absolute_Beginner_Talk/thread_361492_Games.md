### Post by justin whitaker on 2007-02-14
Some combination of Cedega, Crossover Office, and WINE can run many titles. Personally, I find WoW to run great on Cedega, but I prefer other titles on WINE. Some titles won't run no matter what you do. 

There are several AAA tier games that run on Linux natively: Neverwinter Nights, Doom III, Quake IV, Unreal Tournament 2004.

The only one of those that I ever had an issue with is Q4, and that was with sound. Ran beautifully, just had to disable the sound. 

YMMV.
