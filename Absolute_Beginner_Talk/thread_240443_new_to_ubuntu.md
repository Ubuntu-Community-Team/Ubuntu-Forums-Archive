You have a few options:
Install ubuntu (gnome) then apt-get kubuntu-desktop which will give you both desktop environments.

Install kubuntu from the iso directly for only the KDE desktop.

OR:

Install SimplyMEPIS 6.0 which is based on ubuntu, but with KDE as its default desktop environment, plus several other nice tweaks.  

I'm a gnome user, but I have tried Kubuntu and Mepis, and have found Mepis to be a more refined KDE distro compared to Kubuntu.  So my personal feeling is that if you like gnome, there is no better distro than ubuntu.  However, if you really like KDE better, Mepis is a better choice than Kubuntu.  (I'm sure many on these forums will disagree.)  However, K-X-Ubuntu forum support is far and away better than any other community support in my opinion.
