### Post by ErusGuleilmus on 2008-04-07
By "3d Windows" do you meant the Cube that displays your desktops? If so, install the configuration menu by running ```
sudo aptitude install compizconfig-settings-manager
```

Or, are you talking about Desktop Effects in general? If so, do you have the correct video drivers installed?
