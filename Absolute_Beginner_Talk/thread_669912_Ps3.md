### Post by EXCiD3 on 2008-01-16
You can install Ubuntu on the PS3. I did :) Basically it just lets you use your ps3 as a computer. You cant use all the hardware of the ps3 but it runs fairly fast. I'd still recommend using a PC for gaming...not sure if they can even use the graphics card on the ps3 yet, so probably you will have a tough time getting the graphics configured correctly, not to mention playing games. Havent touched Ubuntu on the ps3 in a while as ive been too busy playing UT3 and the like, but id recommend trying it out!
