### Post by oskar2000 on 2007-11-02
Totem, the default player is good enough for me. Mplayer and VLC are great also. For mp3's I use listen-gnome.
You can always do a search in Applications-Add-Remove. For cd burning gnomebaker works for me. Many people like K3b which is a kde application, which means it will take a bit longer to load in gnome since it needs to load the kde libs too, but its worth a try if gnomebaker doesn't work for you.

If you have Gutsy Gibbon, you just need to install the compizconfig-settings-manager to enable the desktop cube. Start it with 'ccsm' or System - Advanced Desktop Effects  and enable the cube and rotate plugin.

-- Just to clarify things - you should always install applications by using Applications - Add/Remove, Synaptic or apt-get. If you are missing programs you should try to enable the extra repositories through Synaptic (start with "gksu synaptic") rather than downloading debs from websites or compiling them yourself!
