### Post by aaronhall555 on 2008-04-21
> **Mazza558 said:**
> 
-I can't find any option to turn off the system beep. 

To get rid of the system beep you just have to edit your /etc/inputrc
```
set bell-style **none**
```
Sorry can't help you with the rest, it's been awhile since I've dug in to KDE, but I know how annoying that system bell can be.
