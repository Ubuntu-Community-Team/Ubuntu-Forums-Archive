### Post by go2dell on 2007-02-16
I think your problem may be that you're hiring a chef when all you need is a paring knife. :-D

If you only need to do screen shots to illustrate error messages, your desktop setup, etc, then Gnome already has that function built in.

[LIST]
[*]Press the **Print Screen** key (to the right of the F12 key, sometimes shortened to something like "PrntScr") you will take a screen shot of the entire visible screen.
[*]Press **Alt** with **Print Screen** to take a shot of *only* the active window (your web browser, OpenOffice, Nautilus, but not the entire screen).
[/LIST]

These screenshots are typically saved in [PNG]("http://en.wikipedia.org/wiki/Png") format, which should already be an acceptable size to post in the forums.  When the dialog box pops up asking for a filename, it will typically name it something reasonably sensible and put the "png" extension on for you.

HTH

EDIT:  There's a topic on this in the Ubuntu Help Centre that can be accessed [here]("file:///usr/share/gnome/help/user-guide/C/user-guide.xml#tools-screenshot").  You can navigate to it by opening the Help Centre (the blue question mark icon in the top panel or under *System > Help > System Documentation*).  Click "Desktop," then "Desktop User Guide," then "Tools and Utilities," then "Taking Screenshots" (section 7.2).
