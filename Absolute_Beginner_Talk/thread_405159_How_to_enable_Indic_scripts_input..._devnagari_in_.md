### Post by krsnendu on 2007-07-22
When using gnome choose the keyboard layout called india.
keyboard preferences/layout/add
 It is the root of the indian languages menu. ie click India. Don't click the triangle that opens the list of other indian langauges.  It is considered a design flaw in gnome by many.

I found this keyboard was inadequate for my needs. I found using scim was the solution to inputing indic font is the most logical way. I use itrans encoding.
