### Post by robophred on 2007-02-24
You will probably need a LiveCd.  Get an Ubuntu live cd if you dont have one already, and run 'gparted' from the terminal.

Hmm, just checked my Ubuntu that was installed through the live cd, and it doesn't have gparted (only has parted, the non-gui version).  Does this mean the live cd wont have it?

At any rate, the only time I used a partitioner for that purpose was running qtparted from the knoppix live cd.  You can use that if the ubuntu live cd doesn't have gparted or qtparted or any variant thereof.
