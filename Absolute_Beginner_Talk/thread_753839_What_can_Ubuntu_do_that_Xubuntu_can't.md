### Post by roachk71 on 2008-04-13
If you used the Synaptic installation path and set XFCE to launch GNOME services on startup, you won't miss a thing, since you'd be able to run the standard Ubuntu GNOME applications.

You'd gain not only a speedy desktop environment, but perhaps smaller and faster extra apps. And I believe XFCE also has its own compositor.

Unfortunately, User Environment-wise, you wouldn't be able to use Metacity window decorations, and a lot of the GNOME icon themes wouldn't work, either.

All this aside, I see no reason not to switch (just might do that myself after work.) :)

### Post by roachk71 on 2008-04-13
It would seem that nothing has to be done to enable GNOME services loading at XFCE startup, since I found the option already enabled after installation.

This may, however, not be the case for all users. So, to be sure, open Applications-->Settings-->Settings Manager-->Sessions and Startup. In the settings manager you can set these options, along with your screensaver, window manager decorations, etc.
