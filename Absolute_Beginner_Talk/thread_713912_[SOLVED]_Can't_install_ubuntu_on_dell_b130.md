### Post by Papa-san on 2008-03-03
Welcome to the community!

Your bottleneck is your memory.
256 isn't usually enough to run the live cd.

I have 7.10 installed on a Dell C-610. 1 gh pentium processor, 256mb of ram, and a 20GB HD.
 It can be done, but you will need to get the alternative install CD. It uses a graphic install process that works well with less memory.

The links in my signature line will probably help with some of the other issues you may run into! (I found that the native bcm43xx wireless drivers didn't work for my card...)
