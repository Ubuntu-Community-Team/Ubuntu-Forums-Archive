### Post by FrOzeN89 on 2007-11-16
How do I setup default file association? I want it so whenever I open an audio or video file it will always use VLC Player.

---

### Post by rahimveron on 2007-11-16
Right click on the audio or video file Properties and navigate to"Open With" Tab and select your player in your case VLC.
Another way is to Start->Systems->Preference-> Preferred Applications and select your media player of your choice
