### Post by Happy_Man on 2007-07-28
Desktop Effects is a stripped down version of Compiz. Both Beryl and Compiz are being merged into Compiz Fusion. If you are satisfied with Desktop Effects, then install gnome-compiz-manager for a better settings dialog. If you want more, use one of the many guides available to install Fusion. Beryl is in the repos...... so is wine. Hope this helps.

EDIT: Previous poster had some typos, use this code:

```
sudo apt-get install beryl emerald-themes
```
