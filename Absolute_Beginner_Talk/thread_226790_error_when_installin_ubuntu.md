### Post by avtolle on 2006-07-31
Common issues: 1) Did you burn it as an iso image? 2) At what speed did you do the burn, 8x or slower is recommended; 3) which iso did you download, e.g., "Desktop", "Alternate Install"; the reason for #3 is that in some cases, the Desktop's graphical installer seems to present problems, many feel that the Alternate Install method (text based) is the better way to proceed. Did you copy the iso to a cd? If so, then it will not work. Burn the iso using Nero or CD Burner XP Pro. In CD Burner XP Pro (free download), just do File -> New Disk from Iso

And burn it at low speeds, < 8x.
