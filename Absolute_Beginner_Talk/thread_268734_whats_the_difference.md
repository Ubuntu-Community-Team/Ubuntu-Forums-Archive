### Post by aysiu on 2006-09-30
Different default programs, different default desktop environments.

If you're new to Linux, the term *desktop environment* might not mean much to you, since Windows and Mac each have only one desktop environment.

As an example, Kubuntu uses KDE and Ubuntu uses Gnome. You can see a comparison of a few differences between Gnome and KDE here:
[http://www.psychocats.net/ubuntu/kdegnome](http://www.psychocats.net/ubuntu/kdegnome)

So:
Kubuntu - KDE
Ubuntu - Gnome
Xubuntu - XFCE

Edubuntu is special, as it's not a different desktop environment (it uses Gnome, like Ubuntu), but it's a special distribution tailored toward education, so its default programs are more educationally related.
