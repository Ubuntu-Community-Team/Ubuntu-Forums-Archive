### Post by Martin on 2007-05-29
In one respect some of the advice you have been given is correct. WINE while being brilliant for what it has acheived is not _The Solution_ for running Windows apps on Linux. It will run a lot of them well and even more not so well but don't be surprised if it crashes from time to time. I spent a lot of time fiddling with getting it working and while I was very impressed with what it achieved, bu tin the end  I either found a Linux equivalent or I just used a Windows machine when I needed to run AutoCAD. WINE is not a Windows substitute yet.
