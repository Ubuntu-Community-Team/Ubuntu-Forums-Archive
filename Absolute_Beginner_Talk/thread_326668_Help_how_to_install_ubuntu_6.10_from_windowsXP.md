### Post by rccharles on 2006-12-29
> **perfectblack9 said:**
> I don't know how to boot the Ubuntu installation from a FAT or NTFS partition.

Big warning:  You need to backup all important data on your windows system.  

While the Ubuntu software does give you warnings about anything destructive it will do, you can make a mistake and destroy your data.



First, you need to create a Ubuntu boot cd.  See this link:
[http://www.psychocats.net/ubuntu/iso](http://www.psychocats.net/ubuntu/iso)

Or, you can request a free copy of the cd to be mailed to you.

Second, you restart your computer with the cd in the cd drive.

This should bring up a menu asking you what you want to do.  I suggest you pick the option "Start or Install Ubuntu". This should bring up the live Ubuntu installatin.

In the case where windows comes up, you need to get into your bias and set the cd drive to boot before the harddrive.  To get into bios, you typically press f2, del, esc, or some other key.  You do this just after you power up the machine.

See this link:
[http://www.psychocats.net/ubuntu/installing](http://www.psychocats.net/ubuntu/installing)
