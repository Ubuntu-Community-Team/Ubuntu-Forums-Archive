### Post by Hospadar on 2007-08-02
You are probably safe to just install any updates that come around, they are almost always bug fixes, security updates, etc.  Whenever aptitude (the program that handles the packages installed on your system) finds a newer version in the repository than the one installed on your machine, it will prompt you to update, and if something makes it into the repository, the newer version is pretty much for sure going to be very stable and better than the previous version.
