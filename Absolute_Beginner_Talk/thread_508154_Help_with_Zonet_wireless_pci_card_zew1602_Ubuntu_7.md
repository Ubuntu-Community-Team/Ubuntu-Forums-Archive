### Post by JesterDev on 2007-07-23
Quick google search reveals that it works under ubuntu 6.10 with the XP drivers from the vendor CD. However according to this: [http://www.linuxcompatible.org/ZEW1602_-_Wireless_Network_Interface_Card_c13361.html](http://www.linuxcompatible.org/ZEW1602_-_Wireless_Network_Interface_Card_c13361.html)

"Partial support in Ubuntu 7.04 - WPA functions of this product will not work." 

So if you have not yet done so try to vendor supplied drivers from the CD. If you don't have them however... Well try asking around. Sorry if that's not much help, but it's a start.
