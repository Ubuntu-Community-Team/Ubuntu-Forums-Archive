### Post by wickstopher on 2007-08-30
Try Audacity, for starters.  Very simple, very functional.  If you want some more complexity, check out Ardour.  Fully featured a la Cubase/ProTools, although admittedly a bit less intuitive.  Both programs are in the repos.  If you're really interested in audio production in Ubuntu, check out the Ubuntu Studio project at [http://www.ubuntustudio.org]("http://www.ubuntustudio.org").  You can either download their modified distribution of Feisty, or there are very simple instructions there as to updating your current Feisty install with the necessary packages.

### Post by splintercellguy on 2007-08-30
GUI way would be to launch Synaptic, search for the audacity package, then mark for install. Command line would be sudo apt-get install audacity. For ardour it's ardour-gtk or ardour-gtk-i686 if you have such a processor.
