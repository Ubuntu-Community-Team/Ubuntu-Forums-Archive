### Post by dfreer on 2007-06-27
Yeah, basically shut down the machine, take the CD out and try and see if your original data is still there.

Basically a live CD has a basic linux file system it loads into RAM. It shouldn't actually even mount your local disks, other than USB hard drives, until it is actually installing the OS itself. This lets you install the OS from a GUI, and since it is basically the exact same desktop that will be installed you can use it to test whether various hardware is detected by ubuntu before you install a single thing!

The best part, of course, is playing games or surfing the internet while it installs. No more watching the progress bar for me!  You can do pretty much anything from a live CD, so if you just want to see how ubuntu works I'd recommend it.

EDIT: it could be a hacked up ubuntu installer designed to be an unintended install, if your data is not there can you post a link to where you ordered the CD?
