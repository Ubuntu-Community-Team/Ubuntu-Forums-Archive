### Post by mustang on 2005-12-09
1) gDesklets: 
```
sudo apt-get install gDesklets
```

Screen: [http://gdesklets.gnomedesktop.org/shot/starterbar.png](http://gdesklets.gnomedesktop.org/shot/starterbar.png)
Download: [http://gdesklets.gnomedesktop.org/files.php?func=gd_downloadfile&gd_filename=starterbar-desklet-0.31.3.tar.gz&gd_fileid=742](http://gdesklets.gnomedesktop.org/files.php?func=gd_downloadfile&gd_filename=starterbar-desklet-0.31.3.tar.gz&gd_fileid=742)

2) aDesklets:

```
sudo apt-get install aDesklets
```

Screen: [http://adesklets.sourceforge.net/images/yab_screen.jpg](http://adesklets.sourceforge.net/images/yab_screen.jpg)
Download: [http://prdownloads.sourceforge.net/adesklets/yab-0.0.2.tar.bz2?download](http://prdownloads.sourceforge.net/adesklets/yab-0.0.2.tar.bz2?download)

3) engage

```
sudo apt-get install engage
```

Can't find a screen

Notes: The gDesklets option is probably your best option. It is certainly the easiest one to get up and running. Engage, on the other hand, is the docker that most closely resembles the original macosx docker. However, it is difficult to set up. I got it up and running after a couple hours in XFCE. It freezes when I use it in GNOME but your mileage may vary...
