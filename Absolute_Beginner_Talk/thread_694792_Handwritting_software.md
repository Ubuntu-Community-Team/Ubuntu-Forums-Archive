### Post by dstew on 2008-02-12
> Second, before under Windows XP Tablet PC edition they had a handwritting utility that would translate from script to text, does anyone know if Ubuntu has something similar?There is [cellwriter.]("http://risujin.org/cellwriter/") It is not a full-blown handwriting recognition program, but it does allow you to input characters one at a time into "cells" on the tablet. You can install cellwriter using the synaptic package manager. See [this thread]("http://ubuntuforums.org/showthread.php?t=563736") for enabling login with cellwriter. You set it up in the System --> Preferences --> Accessibility --> Assistive Technology Preferences menu.
