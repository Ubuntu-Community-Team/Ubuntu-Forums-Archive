### Post by c0nv1ct on 2007-02-28
You are kinda spoiled :P  

Installing Suse/Redhat/Ubuntu and many other popular "user-friendly" Linux distros gets easier with every release.  Installing Windows is a joke, a monkey straight off the tree could do it. 

If you really want to get some experience for being a SysAdmin (you in college? or a young buck trying to get ahead?) try messing around with Solaris or one of the BSDs, they require a bit more learning.  You could also xperiment with Windows servers (2000/2003) by setting up common services like IIS/SQL/Active Directory etc, which is pretty easy to do.  Or if you are really liking Linux, try setting up a Linux server with Novell's eDirectory.

IMO, one of the best things to do as a SysAdmin, is to have a spare box laying around to install Linux, Windows, or whatever on, and then constantly mess with it, break stuff, then learn to fix it.
