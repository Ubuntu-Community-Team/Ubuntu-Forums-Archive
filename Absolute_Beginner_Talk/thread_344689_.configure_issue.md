### Post by pizzutz on 2007-01-23
Looks like you are trying to compile a program, as opposed to simply installing a package.
check this guide.

[https://help.ubuntu.com/community/CompilingSoftware](https://help.ubuntu.com/community/CompilingSoftware)

My guess is you are missing the build essential package.

```
sudo apt-get install build-essential
```

should help.


The build-essential package contains the essential libraries for compiling programs.  It may or may not contain everything you need for your build.  once you run ./configure, look for error messages at the end that tell you to install other libraries.  You may need to install more to get your program to work.

### Post by taurus on 2007-01-23
Make sure you have enable both universe and multiverse repos in /etc/apt/sources.list.  Remove the # in front of all the lines with **deb** at the beginning (and the word universe or mulitverse near the end).

```
gksudo gedit /etc/apt/sources.list
```
Then, try to install it again.

```
sudo apt-get update
sudo apt-get install netatalk
```
