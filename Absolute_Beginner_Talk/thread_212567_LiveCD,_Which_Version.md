### Post by dmizer on 2006-07-10
there are benefits to using the 64 bit version, and there are benefits to using the x86 version.

if you're only going to be using ubuntu as a live version, then use the 64 bit version.  if you're planning to install, you will have to do some serious consideration.

it will be more difficult for you to find applications which will work in your 64 bit install.  knowledge base is not as expansive (because less people have the hardware), so support may not be as forthcoming.  and you'll have to jump through significantly more hoops to make things work sometimes.

biggest setback for using the x86 version would be a performance hit.  but there would be nothing physically harmful about running the x86 version on your 64 bit processor.  though you should upgrade to the 686 kernel instead of the default i386 kernel.  and you'll probably get ridiculed by die-hard 64 bit users ;)
