### Post by MasterOfDisaster on 2007-02-23
mkisofs is a tool to create .iso files.  Type 'man mkisofs' for more information.
For .rar files, type 'sudo apt-get install rar unrar' in the terminal.  This should allow you to use file roller to extract them.

Cheers.
