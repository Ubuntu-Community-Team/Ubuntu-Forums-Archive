### Post by kebes on 2007-12-14
On the command-line, you can probably use "mencoder" and/or "ffmpeg" to re-transcode a video file to a different size.

If you want a GUI program, [Avidemux]("http://fixounet.free.fr/avidemux/") can probably do it ([kdenlive]("http://www.kdenlive.org/"), [kino]("http://kinodv.org/"), and [Cinelerra]("http://cvs.cinelerra.org/") can probably also do it, but may be more complicated than you need).

### Post by shirilover on 2007-12-14
To resize(make file size smaller) in avidemux, you need to set the video codec to something other than copy. Then press the filters button, and choose the resize option from the list. I usually use the MPlayer resize option.
