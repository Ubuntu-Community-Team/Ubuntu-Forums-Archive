### Post by David Corrales on 2006-08-15
The easiest way to do it goes like this:
[LIST]
[*]Open the themes application from Gnome
[*]Download the .tar.gz from [www.gnome-look.org](www.gnome-look.org)
[*]Drag'n'drop the .tar.gz into the theme application
[*]It'll ask if you want to install the theme, say yes
[*]You'll get a confirmation box if everything went alright or there were problems
[/LIST]
