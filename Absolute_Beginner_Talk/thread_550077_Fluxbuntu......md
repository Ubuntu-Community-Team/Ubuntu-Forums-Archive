I never got into Fluxbuntu - but there are plenty of differences.

Fewer Office applications as default etc
Fluxbox / Rox-filer instead of GNOME / Nautilus
No auto-mounting of USB / CDs etc
No GUI for wifi


Just some of the things I recall that pushed me towards Xubuntu instead.

Importantly, I think it is still Ubuntu 6.06-based.
