### Post by catlett on 2006-04-02
When you get a chance go to this thread. It has links to many places with help for new Ubuntu users. [http://www.ubuntuforums.org/showthread.php?t=142716](http://www.ubuntuforums.org/showthread.php?t=142716)
This is the link for the program Automatix(which has already been mentioned)[http://www.ubuntuforums.org/showthread.php?t=138405](http://www.ubuntuforums.org/showthread.php?t=138405)  This should be the first thing you do. Automatix will set your computer up with the basics for web searching, dvd playing and mp3 playing.
Also if you are reading instructions other than from Ubuntu, youneed to know a few things.Ubuntu is a Debian based system. You can install files that are for Debian(deb). Alsp if you see instructions that say "log in as root". There is no such thing in Ubuntu. Ubuntu uses the sudo command for root. Simply put when you need to do a command as the root user, you type the word sudo before the command. You will be asked for a password. Use your password. You don't need a different password. Follow this link to learn more [https://wiki.ubuntu.com/RootSudo](https://wiki.ubuntu.com/RootSudo)
The best asset of Ubuntu is these forums. If you have a problem,any problem, post it. You will get answers. There is no other Linux forum like this. So stick with it and post if you have any problems, peopl here are always willing to help.
