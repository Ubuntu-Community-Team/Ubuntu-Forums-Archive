### Post by mcduck on 2007-03-15
There's a shortcut in Metacity to full-screen applications, there's just no keys set to it by default. It can make pretty much anything full-screen. (I usually set it to Alt-Enter).

Take a look at System/Preferences/Keyboard Shortcuts.

Edit: if you are using Beryl the default shortcut for this is Ctrl-Alt-Enter..
