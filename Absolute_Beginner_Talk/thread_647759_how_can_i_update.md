### Post by cmnorton on 2007-12-22
You have a long way to go to accomplish this. Back up all important stuff, and let the upgrade installation overwrite files when it prompts you.

You should be able to update by running System->Update Manager, and you will see your ability to upgrade to 7.04. Do that upgrade; make sure everything runs fine; and then Update Manager should display that you can upgrade to 7.10.

I'd do a clean installation from a CD after backing everything up.

### Post by seventhc on 2007-12-22
As far as I know each version can only be upgraded to the next higher version, so you would really want the newest version on cd.
You having 6.06 then there are a few more versions that have been released since then. 6.06>>6.10>>7.04>>7.10
