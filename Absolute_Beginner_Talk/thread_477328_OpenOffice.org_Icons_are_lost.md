You can go to System --> Preferences --> Menus & Toolbars

And make sure 'Show Icons in menus' is checked

### Post by g14 on 2007-06-18
> **wersdaluv said:**
> Only oo.o's icons are lost. The 'Show Icons in Menus' is checked.
This same thing happened to me after installing feisty and changing to the "Tango" icon theme because the icons are so much better than the default human theme. Here is how you fix it:

sudo apt-get install openoffice.org-style-tango openoffice.org-style-crystal openoffice.org-style-industrial

Edit:
After installing those packages, close and restart OpenOffice. Then you will have the nice new themed icons!
