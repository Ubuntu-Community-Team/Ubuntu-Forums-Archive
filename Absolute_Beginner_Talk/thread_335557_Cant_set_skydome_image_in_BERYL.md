### Post by d3v1ant_0n3 on 2007-01-10
The skydome sizing has to be power of 2- 1024*512, 2048*1024, etc,etc. You can either find images that fit those dimensions, or crop/resize pictures in GIMP or whereever. I go for the resize option.

Oh, and make sure 'Skydome' is checked :p
