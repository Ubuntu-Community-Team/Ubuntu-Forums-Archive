### Post by mjwood0 on 2006-03-16
This topic has been discussed a lot in the 64bit forums.  A quick search will produce many threads -- I'll try to summarize them here:

**Cons of 64bit**
[LIST]
[*]Many common applications aren't compiled to use the 64 bit architecture
[*]There are some compatibility issues with 64bit Firefox and Flash plugins.
[*]Many of the non-opensourece codecs aren't compiled for 64bit
[/LIST]

**Pros of 64bit**
[LIST]
[*]Slightly faster
[*]You're utilizing the full width of the processor which makes some feel better
[/LIST]

I'm in the same boat.  However, I decided to go with the 32bit version simply since more of what I wanted to use was compiled for it already.  Furthermore, a 64bit processor runs 32bit applications VERY quickly.

Dual booting is a viable options and one I think I will embrace here in the future.

Hope this helps!
