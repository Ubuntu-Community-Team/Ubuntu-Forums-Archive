I can only help you with one of your problems; how to make problems auto-start.

You can do it two ways from what I've learned.

1: Click on SYSTEM -> PREFERENCES -> SESSIONS

     In the Start-Up tab click ADD and put the executable file name there

2. Open the program from the desktop and minimize it.

     Click on SYSTEM -> PREFERENCES -> SESSIONS
     Click on the SESSION OPTIONS tab at the top
     Click on the "Remember Currently Running Applications" button
     Close

I think the 2nd option places the program file name in the Start-up tab for you without you having to do it.

### Post by lpevey on 2007-10-27
There is a file at the following location that determines the screen resolution when X (the graphics manager) starts, when you boot your computer:

/etc/X11/xorg.conf

I don't know if there is a simple way to change this setting in that config file within Gnome without manually editing the file.  It is a setting that is created when Ubuntu first installs.

Before I ask you to do anything with that file, do the following to back it up to a safe place:

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.works

You will need to enter your password.  This will create a copy of your current configuration in the same directory in case you need it later.

Before changing anything, cut and paste the entire xorg.conf to this forum.  You can do that by typing:

gedit /etc/X11/xorg.conf

and then cutting and pasting.  Note that if you wanted to make a change to the file, you would need to use gksudo gedit /etc/X11/xorg.conf, but we're not there yet.

Finally, when you paste the current config file, let us know what you want the default resolution for the X server to be.

You will have to enter your password.  Go to 'X Server Display Configuration' and in the drop-down box next to Resolution make sure it says 1280x1024, not Auto.  Then press 'Save to X Configuration File.'
