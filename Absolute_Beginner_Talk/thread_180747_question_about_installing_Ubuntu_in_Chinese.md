### Post by catlett on 2006-05-22
[https://wiki.ubuntu.com/LanguagePackVsSupport](https://wiki.ubuntu.com/LanguagePackVsSupport)
It appears (I haven't seen anything about Dapper yet, this pertains to the current version, Breezy) Because of Ubuntu's "one cd" policy they can't fit all the language packs they want on the one cd. If I'm understanding it right, there (I don't know about Mandarin specifically) will be a language pack for the desktop but their won't be language support for applications like Firefox and open office. If you wanted to have all applications be viewed in that language you would have to download another package and install it.
Ubuntu is big on keeping the install size small and then have everyone rely on their internet connection to get them the packages they need. This is not a good thing I would think in China, but I may be wrong. How is there internet capabilities over there? Broadband is really what you need to download whole applications realistically. Dial up can do it but of course it is time consuming. 
Why don't you download the live cd and see if it sets up in Mandarin? If you get it running in Manndarin see what happens when you launch firefox or email. Firefox has it's own language extensions but I don't know about the other applications in Ubuntu. I would run the live cd and then open up all the apps you think they will use and see what happens.
