### Post by Bachstelze on 2007-02-20
The differences between them is the fact that Ubuntu uses Gnome as it's desktop environment by default and Kubuntu uses KDE so the look'n'feel is defferent, as well as some of the applications they install (for example Ubuntu installs Gaim as it's instant messaging client while Kubuntu installs Kopete, Ubuntu installs Firefox as it's web browser while Kubuntu installs Konqueror, ad so on).

Keep in mind though that this is only *by default*. Once the installation is complete, you can install whatever software you want - installing Firefox in Kubuntu for example it perfectly possible, as well as having both Gnome and KDE (and possibly other DE's) installed at the same time.


Aysiu :

*"Gnome favors brown and orange"*

This is not perfeclty true. *Ubuntu* favors brown and orange. In other distros, Gnome is blue by default, too.

### Post by skyhopper88 on 2007-02-20
Ubuntu uses gnome as the desktop environment (less frills, more usability). Kubuntu uses KDE, more eyecandy and graphic driven, Xubuntu uses XFCE, which is meant for a very light approach for usability or older systems. You can install any of the others from one varient, or even switch between all three.
See
[http://www.psychocats.net/ubuntu/index.php](http://www.psychocats.net/ubuntu/index.php)
Under Playing Around. Look around there a bit too, its a great resource.

I prefer gnome myself, and I'm a newbie. It just seems cleaner to me. Also, since you don't mind beta stuff you might want to look into Beryl or Compiz for a cool 3D window manager (can be used in on top of the evironments). And for a foobar replacement, the best next thing i found is Amarok.
