### Post by louieb on 2007-04-18
> **RealG187 said:**
> I could just get some old RAM chips. 
What am I lacking CPU or RAM?

**Both!** Most of those P1 machines used 72pin edo memory. and had 4 slots.  The memory has to be installed in pairs. That is it either has 2 sticks or it had 4 sticks  installed.   If you can find the sticks you could install up to 256 MB ram. (4 X 64). But the 64 MB simms are rare, did not see any on eBay just now (32 MB was the largest). And if you did find them expect to pay $20 a pair + shipping.    
For my 40 bucks I've seen P3's for about that out there. And there plenty of used memory for them out there. 

[quote=RealG187]How slow does it go? Back when I ran XP on this machine, I could even play MP3s while typing in word, will it be faster than windows?[/quote]
Ubuntu Breezy was a slow dog on mine.  Puppy, DSL and Slackware all seemed to run faster. But all are pretty slow. Can't remember but I think WIN95 and later WIN98 ran about the same as Linux. but at the time It seem fast.
