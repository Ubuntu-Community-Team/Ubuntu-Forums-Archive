### Post by motin on 2008-05-21
Wondering the exact same thing here... Is the disk recognized at boot time ie is it possible to install Ubuntu on to it?

If it does, then it seems like a valid approach to use the SSD for the boot and root partitions, but put /var, swap and /home on the internal laptop harddrive. That should theoretically speed up boot-time and general OS performance.

If you are not too concerned with burning the SSD after some time (unsure of how long...), then I guess we can put at least /var, but maybe also swap onto the SSD. This way the internal disk will but occasionally be used and could be put into idle mode for the most time, increasing battery time and noise levels of the laptop. 

Any more experienced user to second these thoughts?

Cheers!
