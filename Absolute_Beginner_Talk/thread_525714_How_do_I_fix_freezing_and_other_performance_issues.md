### Post by ajgreeny on 2007-08-14
At some point when the hard disk is active open a terminal and enter top which will list the things in order of their activity.  This should give some idea what is using all the hardware at that moment.  Perhaps if this is a new install it is just the files in /home being indexed by beagle.  When they have been fully indexed, most of this activity will stop.  If it turns out to be something else, get back to the forum and maybe we can try to help further, and hopefully put things right.
