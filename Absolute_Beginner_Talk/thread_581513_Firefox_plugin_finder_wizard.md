### Post by andrewsomething on 2007-10-19
Adblock is in the "Get Ubuntu Addons." I don't think Noscript is yet. You'll have to get that one from Mozilla.

The terms are a little confusing since they are so similar, but there is a difference between add-ons and plugins. It goes some thing like this:

Firefox extensions like Adblock or Noscript are add-ons. 

Plugins are what allows other programs to interact with Firefox (i.e. you need a Flash plugin to watch flash movies or a Mplayer plugin to watch mpg movies).
