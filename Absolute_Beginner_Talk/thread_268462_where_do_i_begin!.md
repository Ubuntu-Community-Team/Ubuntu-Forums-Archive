### Post by Jussi Kukkonen on 2006-09-30
> I am on 98se for surfing an old compaq presario 5150 with 192mb ram which surprising enough even burns stuff ok, but as memory is tight I dont think i can run urbuntu on this as its a 256mb minimum.


192MB is enough, although Gnome might be slow -- Xubuntu should run ok ([https://wiki.ubuntu.com/Xubuntu)](https://wiki.ubuntu.com/Xubuntu)). btw, it's Ubuntu, not Urbuntu.

> 
My main concerns are how to get it up and online as i know nowt about connections ie servers and other such gobblegook I can get windows online and installed under 2hrs without patching but i fear linux will be tougher...
If all goes well, that's 1,5 hours too much for Ubuntu install... But it's good to be prepared for some problems.

>  how do i get online all my isp settings email etc configured?

Normal internet connections should just work (if it's DHCP based), and e-mail is easy to setup, if you have the information at hand (server names , user name, password,...).
If you have a wireless network, be prepared for more problems.

> i glean from here you dont need a fireall or anti virus is this correct ? 
Well, pretty much. I suggest using a hardware firewall (like a cheap router) with any operating system, though. There are easy to install software firewalls available, if you want one.
