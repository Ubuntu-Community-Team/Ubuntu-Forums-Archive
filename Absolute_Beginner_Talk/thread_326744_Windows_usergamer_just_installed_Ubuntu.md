### Post by SurferJoe on 2006-12-28
If you need to run games on your pc , it is best to run VMWare Workstation and set up virtual pc of WinXP and install your games to this.
Currently I do this on my pc - have one virtual pc for XP ( only for Visual Studio at present until I find a Linux alternative ) and one for Win98 as there are a LOT of really old games that I liked that will only run on this and not XP even in compatability mode.
Speed shouldn't be too much of a difference as long as your pc is fairly good.

The really mind blowing part of this is I have my pc doing downloads with Ubuntu in the background , and then run WinXP and play a game at the same time.

I have found little compatability with gaming as far as Wine goes - as it is I only use Wine for uTorrent for my downloads.

Hope this is of some help - we are all learning too!
