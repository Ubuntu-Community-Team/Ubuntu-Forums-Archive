### Post by jaytek13 on 2008-02-18
The easiest way, assuming you have a middle mouse button/wheel, is to use that and double click on the folder in question which will open it up in a new window. Much easier and faster than right or shift clicking.

As far as changing the configuration to make Nautilus operate by default in what is called "spatial mode," I'm not sure that's possible in Ubuntu.
