### Post by LLRNR on 2006-11-21
You got this right, there are many Linux flavors. Eventually you'll find out that you can't describe a distro as "the best one out there" simply because it might suit some people's needs, while for others it might not be a good choice. 

The best thing to do is to go "wandering" about and have a look at other distros too, to read something about them, until you feel that you are no longer so confused and that some alternative choices emerge. I'd recommend you [DistroWatch](http://distrowatch.com). ;) I did this myself.

Eventually the choice is yours. I also read in a few places that new Linux users are advised to try out a couple (2, 3, ...) distros before making up their mind. A good idea seems to be to try some LiveCDs, I don't know what to say, since this has to be your decision.

Whatever it will be, i wish you good luck !

### Post by IYY on 2006-11-21
This is a difficult question to answer, because different ``beginners'' have different needs. For example, some are just starting out, but want to learn the ``proper'' way of doing things. Some want everything to work automatically without any configuration at all.

Generally speaking, I think Ubuntu is an excellent choice as a first distro. It's easy to use, easy to install, quite proper (things are often done through the command line when it's the best way), and most importantly: has the best community. 

Another option I would have recommended a few months ago is SUSE, but after their recent deal with Microsoft it no longer is a very attractive choice.

### Post by aysiu on 2006-11-21
While I'm not surprised (on a Ubuntu forum) that people are recommending Ubuntu, I disagree with it as a recommendation for someone who has no Linux experience.

When I first started using Linux, one of the first distros I tried was Ubuntu, but the command-line configuration of files turned me off because I was used to Windows. I used Mepis for a month and got comfortable with it (doing everything point-and-click). Then, I was ready to move on to Ubuntu, and I've been using it for the past year and a half.

For new Linux users, I'd recommend Mepis, PCLinuxOS, or Linspire.

There's a great distribution chooser quiz you can take online, too, and that will give you a sense of what may be a good distro for you to start with:
[http://www.zegeniestudios.net/ldc/](http://www.zegeniestudios.net/ldc/)
