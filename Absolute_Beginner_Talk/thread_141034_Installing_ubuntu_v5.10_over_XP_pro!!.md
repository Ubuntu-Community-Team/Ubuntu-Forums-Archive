### Post by evilgold on 2006-03-07
You will need to tell the ubuntu installer to resize your current partitions and install to the free space if you want to keep windows. 

As for sharing the folder, yes you can. You will be given options about mounting them during the install, you can also manually do so afterwards. If your using NTFS you will only have read access to them.

### Post by cotcot on 2006-03-07
Before you shrink the XP it is good to defragment it. 
For the shared partition format it as FAT or ext2 or ext3. Install in XP the driver to read and write ext2/3 (available on [www.fs-driver.org)](www.fs-driver.org)).
