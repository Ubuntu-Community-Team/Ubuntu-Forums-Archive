### Post by mlentink on 2007-06-22
For help installing software, you might want to read [this guide]("http://monkeyblog.org/ubuntu/installing/").
You can install TrueType and OpenType fonts in a .fonts folder in your hiome folder. If it does not already exist, you can create one:
```
mkdir .fonts
```
