### Post by Hallavej on 2005-11-04
[QUOTE=Tadhg]Is there a command to have programs startup with Ubunutu, namely gdesklets and GAIM?
[/QUOTE]

Go to:
System->Preferences->Sessions
Click "Startup Programs" and write the command that starts your programs eg. gaim
