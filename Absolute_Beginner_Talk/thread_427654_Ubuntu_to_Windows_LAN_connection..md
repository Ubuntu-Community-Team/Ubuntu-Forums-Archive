### Post by agurk on 2007-04-29
Sure, you can grab files off Windows computers. Click on Places -> Network, to browse to the Windows computer that has the files.
If you want to share files on your Ubuntu computer, go to System -> Administration -> Shared Folders. You'll need to install SMB (Samba).

This is assuming you're running Ubuntu 7.04 (Feisty Fawn).
