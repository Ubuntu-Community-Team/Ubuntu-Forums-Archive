### Post by IYY on 2007-01-17
>    1. Is Ubuntu Linux suitable for C++ programming?

Yes. The  compiler can be easily installed with the command

```
sudo apt-get install build-essential
```

and there are also many editors for C++ and other languages, all available from the repositories.

>    2. Are there suitable music creation programs for Ubuntu?

Yes, Audacity. 

>    3. Will I be able to run Photoshop, The GIMP or any similar programs on Ubuntu?

The Gimp is installed by default, some versions of photoshop can be emulated with Wine, but it's better to just use Gimp. There are also vector editing programs like Inkscape and Xara Xtreme.

>    4. Will Ubuntu Linux run well on just 256MB of RAM?

It will run fairly well, but you may notice some sluggishness, in which case you can change the desktop enviornment to XFCE, Fluxbox or IceWM (no need to re-install the OS for this).
