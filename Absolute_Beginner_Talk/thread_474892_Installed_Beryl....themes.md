LOL! Beryl does much more than spin the cube :p. Expose (called scale) is one of my favourite features. [Here is a full plugin list/explanation.]("http://wiki.beryl-project.org/wiki/Main_Page")

Themes are installed via the window decorator. By default it is Emerald, you open the theme manager by right clicking on the Beryl gem and clicking emerald manager. Then push import and direct it to the emerald file. [gnomelook is a good place to get themes from.]("http://www.gnome-look.org/index.php?xcontentmode=103") Emerald only controls how the "windows themselves" look though, thus the name window decorator. To modify icons and other things, you'll need a gtk 2 theme from gnome look.

Have fun, Beryl 10000X > Aero :D.

> So I download an emerald file, I open the Beryl Settings Manager and I hit Import,
browse to the file.... nothing :(

You have to actually select it in the main window to bottom left of the import window. Else you stay on the same theme as the default. It's usually added to the bottom. Use the edit tab if you wanna change the colour/settings.

Edit: Yes, if you want to change the Icon theme, panels theme or anything else thats not a "Window" you need to go to gnome look and download and install them separately to the theme manager built into Ubuntu (System > Preferences > Themes as you said). You can mix and match themes with icon sets until you find one suited to your taste. Emerald only deals with how windows "LOOK" and Beryl controls how they "Function".

### Post by meuge on 2007-06-16
You need to have beryl-manager installed. If you have the red diamond icon, right click on it, then click on Emerald Theme Manager. If you've already imported, then just click on the theme that you want, then right click on the red diamond again, and select "reload window decorator" and the theme will change to the one you selected.
