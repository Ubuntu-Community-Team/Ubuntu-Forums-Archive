That problem has nothing to do with Ubuntu. However, you can try this for more information:

When XP starts up again, keep ALTERNATELY TAPPING Ctrl, then F8, then Ctrl and so on. At some point you will be presented with a menu. In that menu, choose the option "Disable Automatic Restart on Failure", then continue booting windows.

After a few seconds, you will be presented with a blue screen, that will contain an error message: probably something like DRIVER_IRQ_LESS_THAN_OR_EQUAL or something else. Post the error message and as much details as you can; a screenshot with a digital camera will be nice.

Once we know the error message, we can attempt to solve the problem.
