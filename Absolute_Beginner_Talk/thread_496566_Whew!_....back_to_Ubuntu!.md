More like, Ubuntu uses a different command structure, as far as command line root access is concerned. **su** has been the main way to get root access in most distros, but Ubuntu (afaik) replaced it with **sudo** which offers you root access for just a command, for various security reasons. You can still use **su** in Ubuntu, with the command ** sudo su** (This will offer you root privileges for the rest of your terminal session, except if you switch back to your user with **sudo su <user>**, which is not safe if you don't know what you are doing! (Your home directory changes to root's home directory, and some more changes also.))
