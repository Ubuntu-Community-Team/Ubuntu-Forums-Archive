### Post by heimo on 2007-04-19
Do you mean dual boot? Yes. Basically Ubuntu desktop is Ubuntu server plus graphical user interface and bunch of programs. So you can even run server programs (like apache) on desktop version. Or you can install Xen virtualization and run several instances of servers on your Ubuntu desktop (of course in this case you'll need enough memory and cpu power for all the instances).

Maybe if you tell us what you're trying to achieve, someone on these forums will be able to help you to choose which scenario is best for your purpose.
