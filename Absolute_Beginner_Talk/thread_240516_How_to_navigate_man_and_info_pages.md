### Post by BoneKracker on 2006-08-21
Xman is neat.
  - at the command line, type xman
  - if you like it, make a menu item for it

Also useful: instead of typing 'man', make your initial search using 'apropos' or 'man -k'.  They list all man pages dealing with the key word in question.  Then you can zero in on the most appropriate page.

There are also man pages online.  In addition to being in html format which may suit you these may be more complete.  One example is the set provided by The Linux Documentation Project (TLDP):
[http://www.tldp.org/](http://www.tldp.org/)


edit - and don't forget to see man xman
