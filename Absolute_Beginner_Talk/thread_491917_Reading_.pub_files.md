### Post by HotShotDJ on 2007-07-04
> **Bothered said:**
> Is this an example of [this]("http://news.bbc.co.uk/1/hi/technology/6265976.stm")?Exactly.  That is the problem with proprietary file formats.  Vendors effectively lock you in to their products, and when (NOT if!) they drop support for those formats, you are very often stuck with your data locked away from you.

If you still have access to a Windows box with Publisher, you could always install [PDF Creator]("http://sourceforge.net/projects/pdfcreator") on the Windows box, which you will use like a printer to migrate your .pub files over to .pdf files, which you can then view on any operating system.
