### Post by Threbus5 on 2007-01-13
Hmmm,

Exactly what do you want to experience?

If you want to run Ubuntu from a windows machine download and install the free version of NX server [available from [http://www.nomachine.com]](http://www.nomachine.com]) (I recall that it takes three separate files installed in the proper sequence - described in the server docs) on the Ubuntu machine.

Then download and install NX client for windows on the windows computer, and follow the client to server connection procedures (as described in the client docs) to connect to the Ubuntu machine.
