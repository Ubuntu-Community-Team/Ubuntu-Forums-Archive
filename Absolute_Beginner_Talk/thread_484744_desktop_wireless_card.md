### Post by shearn89 on 2007-06-26
"sudo" is a command put in before a terminal command to run it as the admin.
As regards the wireless:
Use [this]("https://help.ubuntu.com/community/WifiDocs/Driver/Ndiswrapper") howto, which is generally a good one. Sometimes of courese, it just doesn't work - I was using Dapper until a few days ago, and only since i've upgraded to feisty can i get my wireless to work. Linksys is notoriously unsupported by linux, as they refuse to release the code to enable people to write drivers for the cards, however you should be able to get it to work.
