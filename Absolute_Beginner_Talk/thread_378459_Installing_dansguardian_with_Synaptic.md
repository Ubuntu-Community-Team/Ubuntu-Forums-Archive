### Post by tonhou on 2007-03-07
Dansguardian is a reasonably complex program to set up and get functional. I assume you are wanting to use it for filtering your internet use on a single computer.

If this is the case it could be helpful to read this [howto]("http://www.ubuntuforums.org/showthread.php?t=207008") that I put up some time ago. It does not use squid but tinyproxy which doesn't use as much of your system resources.

This howto has been picked up by Ubuntu CE (Christian Edition) and Jereme the developer of that has made a nice graphical front end to it. It is available [here]("http://www.whatwouldjesusdownload.com/christianubuntu/2007/02/ubuntu-ce-v2x-edgy-downloads.html").

Note that at the bottom of the page are some options. The easy way would be to download the Web Content Filtering only package (and probably uninstall what you have already installed - especially squid!) Note also that the options on the page are for Ubuntu 6.10 (Edgy). Elsewhere he has the same available for Dapper.

Hope this is of some help.

--Tony
