### Post by pbaehr on 2006-04-20
If you know your old password (which it sounds like you do) just type
```
passwd
```
and follow the prompts.

Edit: Oh, do that in the terminal. If you're using gnome it's under applications.

---

### Post by cgjones on 2006-04-20
To change the password for your user, type:
```
passwd
```
This will prompt you for your current password, and then let you enter a new password.
