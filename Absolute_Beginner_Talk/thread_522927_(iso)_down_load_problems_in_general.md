### Post by armandh on 2007-08-11
got problems downloading and burning an iso.
thousands have done it successfully BUT
if you keep trying and failing don't expect different results
WITHOUT changing some aspect of what you are doing.

an  iso burn failure can be due to connection or hardware/software problems... 
best try a different computer for the download.

Ubuntu install  is not very forgiving of hardware problems [IMHO] 

old computers can be in the reuse or junk pile. 
some will work, some can be fixed, some not. know the difference.
cd drives wear out. list this as another problem Ubuntu will not cure.

the memory test is on the live cd for a reason.
If you do not understand that Ubuntu will not fix everything... 
Ubuntu may not be for you.
