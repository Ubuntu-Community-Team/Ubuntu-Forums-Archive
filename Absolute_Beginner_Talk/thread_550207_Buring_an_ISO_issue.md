### Post by orange2k on 2007-09-13
I dont quite understand. Do you have an existing ISO file, or do you want to create one from your files?
If you have an existing ISO file, then you dont actually need Gnome baker, you can burn it directly by right-clicking on it and selecting burn-to-CD/DVD...

If you are about to create an ISO file from your selected files, then in Gnome baker prepare your compilation of files first, then select burn, then in the dialog-box that appears select "create image only".
