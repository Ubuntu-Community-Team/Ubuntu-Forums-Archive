### Post by Kralizec on 2007-08-06
Sorry to be the bearer of bad news, but Project64 only works on Windows. However, Mupen64 does have a linux version. Here's the [tarball]("http://mupen64.emulation64.com/files/0.5/mupen64-0.5.tar.bz2") to download. Once its downloaded, extract the archive (either right click on the icon and select 'Extract Here' or in a terminal type ```
tar -xvf mupen64-0.5.tar.bz2
``` while in the directory with the tarball; this should create a folder called "mupen64-0.5"). Then either in the file browser, double click on 'mupen64' (the one with no extension) or in terminal, enter the mupen64-0.5 directory and type ```
./mupen64
``` to run it.

You'll have to get the ROMS For it somewhere else though, although they're not hard to find. Heres one site: [http://www.coolrom.com/roms/n64/]("http://www.coolrom.com/roms/n64/"), also you should Google 'n64 roms' and see what comes up.
