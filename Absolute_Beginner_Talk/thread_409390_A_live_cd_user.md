### Post by KIAaze on 2007-04-14
No, it's not through synaptic.
It's done through an update manager:
[http://www.howtogeek.com/howto/ubuntu/upgrade-ubuntu-edgy-to-feisty/]("http://www.howtogeek.com/howto/ubuntu/upgrade-ubuntu-edgy-to-feisty/")

However, I don't know if this is the only method or the best method.
I recently read something about a way to upgrade which was not recommended, but I don't remember which it was.

The only thing I'm sure of is that upgrading to Feisty doesn't require to download a new CD and reinstall. :)

edit:
Found it:
[B]Offical Upgrade method (Recommended):
To upgrade to the next distribution, bring up a shell. (Applications --> Accessories --> Terminal) Type 'sudo update-manager -c' and follow the prompts.
[/B]
Just as info:
The NOT recommended method:
-Update your sources.lst file (official mirrors will be made available upon release).
-sudo apt-get update
-sudo apt-get dist-upgrade
-When that completes, reboot, and you're running the latest version!
