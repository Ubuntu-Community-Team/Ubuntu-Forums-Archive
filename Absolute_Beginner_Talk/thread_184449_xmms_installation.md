### Post by Perfect Storm on 2006-05-29
You can use synaptic package mannager to install application(software) with. (System tab ---> Adminstration ---> Synaptic package Manager)

or open the terminal (Application tab --- Accessories ---> terminal) and write:
```
sudo apt-get install xmms
```

When done you can write xmms in the terminal or go to applications tab ---> Sound and video ---> xmms.
