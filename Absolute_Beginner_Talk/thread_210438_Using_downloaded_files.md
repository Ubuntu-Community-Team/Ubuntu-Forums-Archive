### Post by Iarwain ben-adar on 2006-07-06
Hi,

Just a little question though: why 'd you want to download something NOT via Adept (or Synaptic) ? It's waaaaay easier than any other way (or so i believe :D) 

The only setback might be that you'd have to enable the [extra repo's]()

Now, for your question: normally you'd make it so that you can execute it
```
chmod +x yourbin.bin
```

And then execute it
```
./yourbin.bin
```

But i seriously recommend using Adept (or Synaptic)
