There are ways to get some games to play using linux, however none of the windows games seem to be as responsive, if you can get them to work at all, 

Cedega is OK but is not that great, Open Office allows some of them to install and work.

Windows games are made to be run in windows -- best things for games is to dual boot. Dual booting is very easy to do when installing ubuntu you will be given the option.
