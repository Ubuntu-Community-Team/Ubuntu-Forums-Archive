### Post by seshomaru samma on 2006-11-16

My personal opinion is that even Xubuntu will be super slow.
If you have time and want to do some work and research you can install a server version of Ubuntu (with no graphical interface) and then add a very light desktop manager like fluxbox. Some may dispute this but I think that if you dont have much experience with Linux and would like something that would work out of the box on such low memory,  forget Ubuntu.
The perfect solution is Puppy Linux - it runs entirely on RAM ,is super fast and very easy to configure. Check out Puppy Linux !
