### Post by nfinitone on 2007-09-03
Hey, if this is a dedicated box, try out the [LinuxMCE Project.]("http://www.linuxmce.com/") You should check out the [[COLOR="DarkOrange"]GOOGLE VIDEO[/COLOR]]("http://video.google.com/videoplay?docid=2176025602905109829&hl=en") of the software first. I think you will be highly impressed.

Make ABSOLUTE SURE that the drive you're installing LinuxMCE on is not currently in use. I didn't see any options to install to a particular partition, and it wiped my entire drive. I should have read the tutorials first. But if you're installing this on a dedicated media box, then you will not be disappointed.
