### Post by luke411 on 2007-03-01
The quickest way to change your associations is to right click on the file type in question, go to properties and then select the "Open With" tab. There you can select whatever program you want as the default for that file type. To the best of my knowledge, there is no 'list' of associations like in Windows. Hope this helps.
