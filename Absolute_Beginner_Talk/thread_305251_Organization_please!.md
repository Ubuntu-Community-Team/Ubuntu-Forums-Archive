### Post by Circus-Killer on 2006-11-23
the reason for this is pretty much customization. some people like to run kde, but still have a couple of gnome based applications running, and vice versa.

this ability allows people to run a certain desktop environment whilst still being able to run applications from a different desktop environment.

it all boils down to choice over simplicity. if they did what you suggest, they would take out a lot of choice. in order to keep choices, some simplicity gets lost along the way.
