### Post by PmDematagoda on 2007-12-27

Amarok can be installed using:-

```
sudo apt-get install amarok
```

Exaile can be installed using:-

```
sudo apt-get install exaile
```

### Post by PmDematagoda on 2007-12-27
LowSky's post also made me remember something else(Thank you LowSky:)), if you want perfect integration then use a player that has been made for the Desktop Environement you use, which in your case is GNOME, so this points to Exaile. 

A good reason in addition to the extra files as stated by LowSky, is the fact that you can easily configure shortcuts for the music player through the main GNOME settings without resorting to other methods. Also Exaile looks much better on GNOME than it does on KDE and Amarok looks better on KDE than it does on GNOME. So in your case I would recommend Exaile the most, though the other music players suggested by LowSky are also very good.
