### Post by 5-HT on 2007-04-20
Ubuntu has the root account locked by default for security purposes and utilizes 'sudo' for privilege escalations. You can always unlock the root account if you'd like. Please refer to the following for more information: [https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)
