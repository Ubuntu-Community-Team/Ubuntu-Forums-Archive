### Post by Kobalt on 2007-08-28
> **tictactoe said:**
> i know that most of the packages available in medibuntu,are .tar files ,or simply zipped files.how do i install them after extracting them???
You need to download the debian packages from the repositories, not the sources.
You can install a debian package simply on double-clicking it. But you must satisfy its dependencies, such as libraries etc. See [this]("https://help.ubuntu.com/7.04/add-applications/C/install-file.html") for more info.
If you really cannot have an internet connection with Ubuntu you can consider using [AptOnCD]("http://aptoncd.sourceforge.net/screenshots.html").
