### Post by glennric on 2008-03-15
Go to System->Preferences->File Management and select the Behavior tab.  Then select "Single Click to open items"

### Post by glennric on 2008-03-15
You can also open a nautilus window and select Edit->Preferences to get to the same place.  I am assuming you are in gnome and not xubuntu, kde, or some other desktop manager.

---

### Post by glennric on 2008-03-15
You can also type "nautilus-file-management-properties" in a terminal to get to the same place.  There is nothing you need to install (unless you don't have nautilus installed).  The File Management menu item is not activated by default.  You can edit the menus and activate it.
