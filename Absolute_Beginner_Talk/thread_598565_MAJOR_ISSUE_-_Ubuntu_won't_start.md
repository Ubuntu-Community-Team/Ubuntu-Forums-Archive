### Post by taurus on 2007-10-31
Press <Ctrl><Alt>F2 to get to another console.  Log in with your username and password.  Then, edit /etc/X11/xorg.conf with

```
sudo nano -Bw /etc/X11/xorg.conf
```
and scroll down until you get to the Device section.  In the Driver, change it to "**vesa**".  Save it with <Ctrl>o and exit with <Ctrl>x.  

Reboot.

```
sudo shutdown -r now
```
