### Post by WHICKS on 2007-05-30
You are probably better to use open office applications (it takes only a couple of minutes to get used to the differences.

Just remember to use the safe file format as Windows, so you can use Word or Excell at the office, and OO at home.

There&#8217;s a trick to using OpenOffice.org, and that is to use the proper default file format. If you use OO.o&#8217;s native file format, whether it&#8217;s for a text document, a spreadsheet, or anything else, Microsoft Office users will be unable to open it. You need to go into the options for each OO.o program you use and under Load/Save, in the General section, set the default file format to the most recent Microsoft option (for the word processor, it&#8217;s Microsoft Word 97/2000/XP; for the spreadsheet it&#8217;s Microsoft Excel 97/2000/XP).
