### Post by shavenlunatic on 2007-07-06
Openoffice.org is great for your standard user and has pretty much all the features of MS Office.. I only use MS Office now due to VBA, I write a LOT of applications which run in VBA under Excel and this, I can't do without.  As far as I can see, if you aren't bothered about writing applications using VBA you can't go far wrong using openoffice.org (and it does have a limited macro recording functionality which I haven't looked into much)

If you specifiy what exactly you feel is missing from OO.O, people may be able to help/point you to software which provides these functions.
