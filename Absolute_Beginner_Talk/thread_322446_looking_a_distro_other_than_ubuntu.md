You might want to take a closer look at your machine. A 2.2Ghz should be plenty fast to run Ubuntu. If it's sluggish, it's probably something else that's wrong.

I am using an Apple PowerBook G4 TI 12-inch which has a 1GHz PPC processor with 512MB RAM and its very very fast and extremely responsive as well. But also keep in mind I compile most of my programs from source and I run Xfce4.2 which was compiled from source as well. Gentoo is probably the most responsive of them all, because you compile every package from its source code so everything is optimized for your hardware. But i've heard that Gentoo is for hardcore linux people only because it's so difficult to understand and not user friendly in the slightest bit.
