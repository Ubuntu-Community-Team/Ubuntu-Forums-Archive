### Post by Paloseco on 2005-11-22
You need at least 3 partition, one for swap (the same size of the RAM), another for linux and the last for windows. You can create aside a partition for data, but in a 8GB hard disk, depends on you.

The windows partition must be primary and must be at the beggining of the hard disk, the rest of the space for the two linux partitions and aditional partition(s) for data if you want.

Depends on the programs you are going to use in each operting system in order to decide the space you are decide to dedicate for each operating system, or you will get out of space.
