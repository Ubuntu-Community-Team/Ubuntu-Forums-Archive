### Post by Scunizi on 2007-06-26
If you're in the terminal the prompt will change at the end from "$" to "#".  However since Ubuntu is designed to use sudo instead of su, you typically are not in root very long. Root access is granted by using sudo for each command needing root access.  If that's cumbersome, there are methods of attaining a static root terminal window or even loading Nautilus as root.
