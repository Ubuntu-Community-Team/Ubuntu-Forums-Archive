### Post by jayson.rowe on 2008-04-18
I think (GNOME) Ubuntu would load (from the alt. cd only) but would barely be usable. Xubuntu would be better, but from what I've seen Xubuntu's XFCE implementation seems a little heavier than some other XFCE distro's I've tried on other hardware. 

I second the reccomendation of Fluxbuntu (although it may seem very different than anything you've ever used before). 

I also highly reccomend Openbox (here is some documentation for you: [https://help.ubuntu.com/community/Openbox](https://help.ubuntu.com/community/Openbox)) 

If you were to go the Openbox route I'd suggest installing Ubuntu Server which will give you a basic CLi only install, and then you can apt-get install openbox obconf xserver-xorg and be up and running - you'd have to also install Firefox, VLC or what other apps you needed, but you'd wind up with a very lean fast system that way.

If you are interested in stepping outside the Ubuntu family, and want something with a more "familiar" interface both Wolvix and Vector are Slackware based distros that use XFCE and are VERY fast. Wolvix has always felt a little faster to me, but Vector has a "nicer" look to it.

Personal opinion though - have fun trying out some Ubuntu variants, as well as possibly some other distros on that guy, and I'm sure you'll find something you are happy with.
