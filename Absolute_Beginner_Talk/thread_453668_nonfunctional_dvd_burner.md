> **DrTaylor said:**

You need to be a bit more specific with "won't work". Does the tray still open and close to put CDs in? It is still powered right? If you know the hardware is correct, then I guess maybe a small glitch?

You should be able to manually mount it following this section of [Ubuntuguide.org]("http://ubuntuguide.org/wiki/Ubuntu:Feisty#Optical_Drives"). If it doesn't mount... maybe its met its unexpected end as an optical drive?

In future, always best to provide more info than less. Oh and if you do have a dual boot, boot windows and try burning something to make sure its still functional. Or if you have old drive around that works, swap it in.

Ya always need a thorough process of elimination when fixing comps... hope that helps.
