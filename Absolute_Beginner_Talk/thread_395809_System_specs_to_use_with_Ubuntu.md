### Post by DreadPirateRoby on 2007-03-28
I have an AMD64 machine so I'm mainly answering you're question as to how the two compare.

Well you may not need to pick between one or the other. I've been running a dual boot system with XP Media Center /Ubuntu 6.10 64 then XP MC /Xubuntu i386 without the dual boot itself causing major problems.

You'll most likely have to set up your vid card and media support, but there are guides to that in the community documentation.

Ubuntu is likely to be more stable once you do get it running and obviously, won't have all that junk the manufacture dumped in your windows install.

You should be able to do most of what you can do in windows, short of shockwave support and most commercial games.

There is Wine for windows emulation, but I haven't tried it yet.

It's been hard for me to get everything set up in ubuntu, but that's mostly because I'm on dial up. Things should be much easier if you can use add/remove and apt get.
