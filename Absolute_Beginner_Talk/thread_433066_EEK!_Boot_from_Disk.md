### Post by Metacarpal on 2007-05-04
When you say you burnt the iso to CD, but not as an image, do you mean you just burnt the .iso file directly to the CD?  If so, and you insert that CD in a drive on a running computer, you should be able to save the .iso file that's sitting on the CD, then burn it properly to a different CD.

(This will only work if browsing the CD shows a single file, of course.  If you extracted the iso to the CD, you'll have to re-download it.)
