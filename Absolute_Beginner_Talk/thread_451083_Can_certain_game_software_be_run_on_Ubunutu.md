### Post by jrocket on 2007-05-22
What type of games are you talking about?  You could run a full installation of windows through vmware or another virtualization application, but I doubt this would work well for games other than freecell, it would be too slow for a full screen game like WoW.  You could also run a game through wine or cedega.  I have wine running a few apps and games such as windows' freecell, and cedega running WoW, and starcraft by the time the weekend's here.
Here's a list of a few of the games that wine supports:
[http://appdb.winehq.org/](http://appdb.winehq.org/)
