### Post by brim4brim on 2006-09-01
So from your post history your problem is in partitioning.

It is unclear what your partitions are from readin it though.  It seems like your saying you have two drives and that windows is using both of them for NTFS and using them as one drive.

I'll just say I find the easiest way to get Ubuntu duel booting which is what I think you want is to setup some free space that isn't partitioned to anything (I use partition magic for this in windows so I don't know if you can use gparted for it) then I just run the Ubuntu installer and it gives an option for installing in the unpartitioned space.

Basically you need to resize an existing partition if you have no unpartitioned space on the drive.  Windows by default will have setup everything for windows so you have to drag the windows partition smaller.

I don't know if gparted will loose you information.  Partition Magic can do this.  Wait for confirmation from someone who uses gparted if it can do this.
