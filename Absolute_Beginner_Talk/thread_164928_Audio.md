### Post by xXx 0wn3d xXx on 2006-04-23
Use Automatix:

In terminal copy and paste the following. Terminal can be found under Applications > Acessories > Terminal. It is sometimes called the command line.

> wget [http://beerorkid.com/automatix/automatix_5.8-1_i386.deb](http://beerorkid.com/automatix/automatix_5.8-1_i386.deb)
sudo dpkg -i automatix_5.8-1_i386.deb

Then select it from Applications > System Tools > Automatix then you should see some messages. Then select the Non-Free AUD/Video codecs a cnd the libdvd codecs. Then hit ok and it will install them.
