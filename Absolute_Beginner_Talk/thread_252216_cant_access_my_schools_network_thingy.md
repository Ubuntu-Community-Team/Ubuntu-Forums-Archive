### Post by Jussi Kukkonen on 2006-09-06
You should ask the network administrator about the services they're running (or try their web pages)... 

I guess they are running SMB-shares (which you should be able to access from *Places-->Connect to Server*), if you know the details. Try looking at the U:\-share properties in Windows, maybe that's enough info. But they could be doing some funky Windows-only-authorization.
