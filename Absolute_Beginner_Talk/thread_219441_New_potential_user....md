### Post by nalmeth on 2006-07-20
Have you used linux before?

Your system sounds perfect for xubuntu, if you have no linux experience, I would recommend that. Download the "ALTERNATE" install, rather than the desktop install, because it doesn't work properly on systems with less than 256 megs of RAM. 

You should try that out, and if you want something even faster, you have a few more options, but things will get more complicated.

Xubuntu is great for low-speed systems, and novice users.
