### Post by Alterax on 2007-04-20
You sure can!  In fact, the boxes most people think of when they hear "servers" are basically just high-powered PCs built for performance.  Great if you have lots of users, if you only have a handful of users, that kind of box can be overkill and you can get by with something smaller.

In my office, we tend to use the word "server" to mean a computer that serves content of some type.  Only a couple of these are server-type boxes, the rest are PCs that we have set up to run server programs on.

So, to finish up a long answer to a short question, yeah, there's nothing to it.  You can even use a standard Ubuntu distribution for your server, but I recommend the server version instead because it doesn't install as many other programs the server won't need.  More room for what you DO want it to do.

--Alterax
