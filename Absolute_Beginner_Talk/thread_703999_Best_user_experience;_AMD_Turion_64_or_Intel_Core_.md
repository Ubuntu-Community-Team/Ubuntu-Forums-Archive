### Post by InlawBiker on 2008-02-22
> **OZFive said:**
> What is the difference between the two?  I have seen them but sometimes some retailers on their websites claim it is a Core 2 Duo on the same machine I saw at another site that was a Core Duo.  Hard to tell.


You should scoot over to [www.notebookreview.com](www.notebookreview.com) there's a lot of hardware info there on notebook computers.

The main difference right now is the Intel chips are slightly faster and use less power.  To compensate for that AMD has priced their chips lower.  So you can save a few bucks buying an AMD, knowing it'll run hotter and the battery won't last quite as long.  How long exactly, who knows.  15 minutes maybe?

Core Duo is not 64 bit, Core 2 Duo is.  if you want to run 64 bit get Core 2.  Core 2 is also more efficient with power and runs cooler.

To save yourself a lot of headaches make sure you get a laptop with Intel graphics (or nvidia if you must), and Intel wifi.

### Post by Presto123 on 2008-02-22
OZFive:

The reason that DDR is more expensive is because they can make DDR2 which is considered to be faster and is used more. I know you're still thinking why? This is because AMD decided to not upgrade their boards because they thought it was only a nominal amount of difference, so now that DDR is an older version, it is a bit harder to come by. BUT now the new boards are coming out with DDR2 capable mobos, but are still making DDR capable mobos at the same time.

So...now that the circle of confusion is complete...let's simplify that: in DDR RAM, you are paying more for harder-to-find equipment while DDR2 mobos are more expensive due to the idea that you are getting more up-to-date system in a sense.

Confused? I am.
