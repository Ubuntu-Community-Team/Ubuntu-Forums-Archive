### Post by Dual Cortex on 2006-10-29
It's in the specifications...

Memory Supported
Memory slot	2x 240Pin
Memory Type Supported	DDR2 667/800
Max Memory Supported	4GB

Storage
IDE ATA	1x ATA 100
Serial ATA	2x SATA 150


You need DDR2 (240 'pin') RAM. In other words, your older DDR RAM is not compatible.

And as you also see, it supports both SATA + IDE.

If you're going to buy a new peocessor, RAM, etc. I'd recommend you to go with a Core 2 Duo processor.
In fact, there are C2D  motherboards that support DDR and AGP so the minimum you'd have to buy to upgrade would be the processor and well, obviously, the motherboard..
