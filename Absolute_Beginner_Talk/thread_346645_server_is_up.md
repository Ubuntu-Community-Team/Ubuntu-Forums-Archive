### Post by lamego on 2007-01-26
insanitywetrust,
before use a command try "man command" to learn about it.

If you don't know how to use a text mode editor, look for a tutorial.
Is far more easy to learn how to use vi by searching for a "vi tutorial" on google.

I do recommend you to search for a linux tutorial before logging on to your system without any idea on what to do.

If you prefer to learn the easier way, start by installing a desktop system, with all those handy graphical tools so that you can browse for the tutorials and learn how to use the terminal.
