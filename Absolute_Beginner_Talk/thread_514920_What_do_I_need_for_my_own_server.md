### Post by Ek0nomik on 2007-08-01
[http://www.strdoc.net/ubuntu-apache-php-mysql-server/](http://www.strdoc.net/ubuntu-apache-php-mysql-server/)

[https://help.ubuntu.com/community/ApacheMySQLPHP](https://help.ubuntu.com/community/ApacheMySQLPHP)

If you have LAMP, than you have Linux, Apache, MySQL, and PHP.  That's a good setup to get you started.

[http://localhost/](http://localhost/)

This should show you your default webpage.  If you know your external IP address, you can also type that into your web browser and that same page should appear.

Your webpage files are stored in /var/www - so you can edit away if you wish.
