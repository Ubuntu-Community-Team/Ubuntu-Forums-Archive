### Post by jan quark on 2008-01-13
~ is the sign for home
and the others like e/e/bla bla something are the directories and subdirectories

for instance I open the terminal and type

sudo cp kon.png /home/jan

sudo sets the root permission
cp is the copny command
kon.png is an image of a horse
/home/jan is the directory I want the horse to be copied
be sure you are in the folder where the file actually is you want to copy

### Post by nikoPSK on 2008-01-13
It means to use the mv command, example, say I wanted to move myfile in the home folder to my desktop:

mv ~/myfile ~/Desktop

Simple as that :), you put the mv parameter then the target of the move to where you want to move it
