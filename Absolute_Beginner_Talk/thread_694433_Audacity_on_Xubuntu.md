### Post by defenestratos on 2008-02-14
It is easier to install software in Ubuntu than in Vista.
Go to system-Synaptics Package manager. It will ask you for your root password. Once open go to search on the toolbar and enter Audacity. It will find the package along with a whole lot of related packages. click on the little box next to the Audacity package and mark install package. There you are.
While you are in Synaptics, have a look around for some of the other cool audio software we have access to for free. Personal favourite sequencer is Ardour, a protools copy.  A good drum machine is Hydrogen.
Hope this helps
