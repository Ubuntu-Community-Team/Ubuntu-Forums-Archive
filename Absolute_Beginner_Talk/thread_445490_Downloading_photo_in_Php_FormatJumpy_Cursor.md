### Post by nhandler on 2007-05-16
Chances are you either
A) Downloaded the wrong file
B) Downloaded the correct file, but a php extension got put on the end.

If it is the later, simply right click on the php file and select properties. Under mime type, it should say image/XXX where XXX is an image format. Now, replace the php extension with the extension that goes with XXX.

I have no clue about the cursor problem.
