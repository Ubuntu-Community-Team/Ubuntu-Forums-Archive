### Post by Sef on 2006-04-28
I  have set up a second language on mine.  I did the first two steps and am able to choose which language I want to boot in.

You can set it up, so that when you start you have choice of what language you want to use at log in.

Step one: System > Administration > Language (click on Greek and set it as the default)

Step two: System > Preferences > Keyboard (set your keyboard as Greek.)

Step three (optional): System > Preferences > SCIM (set your keyboard as Greek there too.)

Step four: Set Greek as the log in language when you log in.
