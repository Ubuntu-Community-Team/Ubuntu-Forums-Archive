### Post by Inxsible on 2007-06-03
Where are you looking ?

System -- > Networks ? You cannot create new networks there

Try this:
Make sure you are on wireless and not on a wired network. Click on the network icon near your system clock. Select Create New Wireless Network. You can select the Wireless Security as WPA Personal then.


Hope that helps !
