### Post by skyhopper88 on 2007-02-19
Ubuntu uses gnome as the window manager (less frils, more usability). Kubuntu Uses KDE, more eyecandy, graphic driven, Xubuntu uses XFCE, which is meant for a very light approach for usability or older systems. You can install any of the others from one varient, or even switch between all three.
See
[http://www.psychocats.net/ubuntu/index.php](http://www.psychocats.net/ubuntu/index.php)
Under Playing Around. Look around there a bit too, its a great resource.
