### Post by diego898 on 2006-05-15
Hello everyone, first and foremost, I want to thank the Ubuntu community for its patience, and amazing interaction and willingness to assist new people. I have used a Windows box my whole life and want to make the transition to a linux machine before I get to college, hopefully gaining some experience. My installation of Ubuntu was and interesting undertaking (had some problems with the partitions and properly setting up the network) but those in the IRC channel really helped me out. 

Its this amazing community that is something incredibly special that I want to become a part of. I am going to catalog my journey from Windows to Ubuntu, by something like my customized manual. Any questions I have, I will post them on this website. Once I find a suitable answer, I will add that to each question. Hopefully, by the time I become a self-sustaining Ubuntu/Linux machine, my journey will help others to make the migration from Windows as painless as possible.

I know that there has to be something like this already created, a kind of personal manual. However, this is going to be *my* personal manual. 

I will be updating very soon with a link.

EDIT - Quite similar to the [Unofficial Ubuntu Guide]("http://ubuntuguide.org/") Except from a more personal perspective, coming from a Windows perspective. I'm very excited about this undertaking, and hopefully, will have some kind of value ;]
