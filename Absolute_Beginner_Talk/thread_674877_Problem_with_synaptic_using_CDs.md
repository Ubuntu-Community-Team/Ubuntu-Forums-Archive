### Post by Rhubarb on 2008-01-22
If you have broadband connection to the internet, I recommend telling ubuntu that you don't want to have to put your ubuntu cd in the drive so you can install software.

System --> Administration --> Software Sources
Uncheck the cd-rom checkbox.

Next time you go into synaptic, press the reload button, then you may install software / updates freely without having to insert the ubuntu cd-rom disc.
