```

wparker@geko:/lib$ nslookup homx
Server:         127.0.0.1
Address:        127.0.0.1#53

** server can't find homx: NXDOMAIN

wparker@geko:/lib$ nslookup homx01
Server:         127.0.0.1
Address:        127.0.0.1#53

** server can't find homx01: NXDOMAIN

wparker@geko:/lib$ nslookup sydnotes01

Address:        127.0.0.1#53

wparker@geko:/lib$ nslookup homx.my.domain.com.au

wparker@geko:/lib$ cat /etc/resolv.conf

; Use a local caching nameserver controlled by NetworkManager

```

If I ask for the host to resolve with its full domain name it is able to resolve the IP...

This is wierd because all my browsing is working find (probably due to the network proxy setup).
