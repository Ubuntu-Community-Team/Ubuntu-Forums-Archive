### Post by aysiu on 2006-03-26
Firefox 1.5 is not in the Breezy repositories.

There are three ways to install it, and they basically all use the same method:

You can follow these instructions (just copy and paste commands):
[http://wiki.ubuntu.com/FirefoxNewVersion](http://wiki.ubuntu.com/FirefoxNewVersion)

You can use [this script](http://www.ubuntuforums.org/showpost.php?p=856576&postcount=15), which automates that tutorial

You can use [the Automatix script](http://www.ubuntuforums.org/showthread.php?t=138405), which install the latest Firefox and lets you install a bunch of other popular packages.
