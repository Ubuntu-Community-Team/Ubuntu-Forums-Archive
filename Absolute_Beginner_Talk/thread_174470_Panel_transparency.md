### Post by jasay on 2006-05-11
Assuming you are using gnome (as in ubuntu and not kubuntu, xubuntu, etc.) some transparency of your panels is dependant on the theme you are using.  Go to System > Preferences > Theme and play around a little.  If you go to theme details, I think it's the "controls" specifically that you want to change, but I can't remember for sure.  Not all themes allow for transparency so it's frustrating when you find an otherwise nice theme that doesn't.  You might try [www.gnome-look.org](www.gnome-look.org) and [art.gnome.org](art.gnome.org) for more themes.

Also be aware that some icons in the notifications area (e.g. gaim) do not have transparent backgrounds and look a little out of place when you do get the rest of the panel clear.:(

### Post by Kvark on 2006-05-12
[QUOTE=georgetoon]I agree.:)  How do I enable it?  I'm new to ubuntu.:)[/QUOTE]
Short answer: Install it and then you'll find it in the menus "System -> Settings -> Art Manager".

Long answer:
1. Follow [this guide]("http://ubuntuforums.org/showthread.php?t=171919") if you haven't added the extra repositories yet.
2. Search for "gnome-art" in Synaptic (which is at: System -> Administration -> Synaptic Package Manager).
3. Mark the checkbox next to the search result named "gnome-art".
4. Click apply.
5. Now it's in the menus at "System -> Settings -> Art Manager".
