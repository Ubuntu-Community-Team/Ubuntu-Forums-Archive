### Post by cmnorton on 2007-12-16
> **k.l.l said:**
> Sorry for a basic question but a friend has recommended Ubuntu and today is my first day. How does it compare to XP.

A lot depends on what you want to do. If you want a day-to-day computer for email and other stuff, you will find that with less powerful hardware, you can do quite a bit of work. You get regular updates, not only involving security, but applications as well. Ubuntu comes with tons of applications for which you would pay a lot on Windows. Some open source apps do run on Windows, like Open Office.  Oh, and by the way, I really like XP; let's not mention VISTA.

I use a low-powered, older laptop running Ubuntu 7.10 as a primary, portable workstation. Most of my work is Linux command-line and database work on remote servers, so I use terminal emulation and Epsilon's (emacs') ftp file ability to do my development remotely. My goal is to purchase a newer laptop, with more memory, a faster processor, and more disk space. Then, I will do my development locally.

If you are doing Windows development (C, C++, and so on), you probably need an XP system. If you are doing Microsoft Access development -- I have not tried this -- for a $38.00 license, you can run MS Office on Ubuntu. I've offhand forgotten the name of the vendor from whom you would purchase this software, but it runs in conjunction with WINE.

The disadvantages are, Linux takes patience, and things may not always work right. This is truer on laptops. Any non-laptop system on which I've installed Ubuntu has just installed easily and runs like a charm.

Finally, I have installed Red Hat Enterprise, Fedora, and Ubuntu. By far, Ubuntu has just installed without any major problems. During version upgrades, I have found up backing up configuration info, and just let the upgrade write its version of config files. The upgrade installation seems to go better that way. However, many Windows upgrades I've done are far worse.

I am probably missing something, but that's my take on things.

### Post by dasunst3r on 2007-12-16
There are many reasons why I like Linux:
[list]
[*]Get the latest and greatest with lower-spec computers.  This means you have a longer lease on life with your machine.
[*]No need to fight your computer for control. (with great control comes great responsibility -- think before you type).
[*]Update all the programs you have on your installation in one go.
[*]No DRM to bog down your system and treat you like a criminal.
[/list]

But don't take my word for it -- download a LiveCD and try it out for yourself!  Before making the jump, though, you MUST backup your data and you should plug in all the devices you own to make sure they work correctly.  Do your homework before going for a full-blown install so you won't be faced with unpleasant surprises later.

### Post by MyR on 2007-12-16
I use linux because...
linux is generally more stable (no blue screen of death, less freezing, less file corruption).
software for linux is (almost always) free, easy to install, and easy to find.
linux is much more customizable.
i don't have to restart linux as often, and it never forces me.
linux runs faster, especially on old hardware.
linux is more secure (no viruses, adware, spyware.)
linux is faster and easier to install.
linux has cool desktop effects; it just looks cooler.
linux is low maintenance (no need to defragment, virus scan, clean registry, etc.)
i can test out my hardware before installing, and i can use linux without even installing it.
the linux community is very friendly and willing to help ;]

things i dont like:
the (windows) pc games i play are too difficult to install on linux with wine
my printer is windows-only (i use a virtual machine to print while on linux)
not all hardware is as linux-friendly as mine is, though it can usually be made to work.
