### Post by 23meg on 2007-03-14
Upgrades can be tricky if you make heavy use of third party repositories (especially if you don't revert to older versions of libraries before the upgrade). Also if you use proprietary drivers and firmware, and have poorly supported hardware.

If you don't want to take the risk of having upgrade headaches, install 6.10 with a separate home partition, and when 7.04 is out, make a clean install. That way you'll be able to retain your personal data and application settings.
