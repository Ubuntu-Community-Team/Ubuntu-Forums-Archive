### Post by Rocket2DMn on 2007-06-28
Should looks something like

Subsection "Display"
Depth       1
Modes       "1280x1024" "1024x768" "800x600" "640x480"
EndSubSection

for each subsection of depth.  
Then hit CTRL+ALT+BACKSPACE to restart the xserver (close any open files first).

PLEASE make a backup of your xorg.conf file first
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```
you can recover the backup in the case that the xserver doesn't come back up by running
```
sudo cp /etc/X11/xorg.conf.backup /etc/X11/xorg.conf
```
