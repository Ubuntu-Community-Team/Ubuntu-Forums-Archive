2. Take a deep breath. It will likely all go smoothly, but sometimes problems come up. Most are solved pretty quickly by posting a thread here.

### Post by Thund3rstruck on 2006-07-24
If there's one thing I always recommend for Windows users is to follow this [Fantastic VMWare server Guide]("http://www.ubuntuforums.org/showthread.php?t=183209&highlight=howto+vmware+server") and get windows installed in a virtual terminal in Linux. This way when all seems lost and you're ready to give up, you can load up the console and have a fully functional windows xp session running in linux. As time progresses you'll probably use it less and less until finally you can give up windows all together. Using a dual boot is ok, but in all reality you end up booting into windows and then rarelt find the time to boot into linux
