### Post by b_martinez on 2006-08-31
I've been using Linux since 2003. Some of the first ones I used were the little known distros. Some were well known. All were a 'trial' to install the first time, as all I knew was Windows. I was a win 'power user' and all it took was patience and the desire to learn. I am learning Ubuntu and derivatives to help family/friends to use Linux. This distro is easy to use, the learning curve isn't one that will have you pulling your hair out, installation of OS and supplemental programs is fairly easy. All-in-all, you have chosen one of the best, if not THE best distro to learn on. You may however, prefer another desktop, more similar to the way win looks. If so, d/l the KDE files. Try XFCE also.
 One last thing. There are a LOT of programs to do what you want. Don't let it put you off that there are 3 to 50 different programs to do anything you want. Office work to listening to your cd's. It's all what you learn suits you best.

### Post by telegramsam on 2006-08-31
IF you like to explore and learn, you'll be SO GLAD you did this.  I had so much fun doing this.  I also wiped Windows off the drive and decided I'd sink or swim.  After about a month, I'm swimming nicely.

The only real struggle I had was my own fault, and that was to make myself quit trying to follow windows procedures and protocols.  The terminology is somewhat different, but you'll find that if you're a more advanced user of Windows, that Ubuntu has everything in it that you wished Windows had--like virtually every software package you would ever need (and 19,000 or so more that you would never need) in one list.  It's THAT easy.

There are some things that you will find yourself wishing were different, but if you didn't experience that with Windows, you weren't using much of Windows.

Trust me, if you have any kind of adventurous spirit, this is the place for you.  

Look at the replies you've had to this post.  When you have a problem, the help comes flying at you--within minutes.  I have never seen anyone treated as if he/she were stupid, and I've always been treated with the utmost respect and kindness by these people.  THIS forum is what makes Ubuntu the best OS you can have.

### Post by tht00 on 2006-08-31
I started using Linux about a year ago, and, for a while, switched back and forth.  I liked Linux, but as I had a few hardware problems that just would not resolve themselves, I would go back to Windows for a while.  Eventually, Linux stuck, and it's basically all I use anymore, and I've learned a great deal.

I could've been called a Windows 'power user'.  I knew the system pretty well, where stuff was at, how things worked, and where to start troubleshooting.  You might notice that I used past tense in the above sentence, as I pretty much don't run Windows anymore.  I've still got it installed to run games, but that's really it.  Really, though, I'm not sure that knowing Windows well really helped me with Linux.  Naturally, being familiar with computers helps, but when you use Windows, you learn how to use Windows, and not how to 'use a computer'.  The processes of switching was kind of difficult.  As far as on the surface, many things are the same, but the techincal side of me wanted to know how everything was setup, and this is where I had trouble wrapping my head around concepts... especially the idea of having a root partition ('/'), and not having different drives mounted as C:\ , D:\ ,etc.  It is certainly different in places, but different is not bad.

I'm very comforable with installing/setting up/running/modifying Linux now.  Just spend time with it, try not to get discouraged, and it usually isn't a bad transistion.  Once you've used it a while, you won't want to go back. ;)
