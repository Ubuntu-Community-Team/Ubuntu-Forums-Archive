### Post by Najand on 2006-09-11
Click on 
System -> Administration -> Language Support
Find your language and then check the support box beside it. 
It will then automatically install all neccessory files for 
supporting your favorite language.

To enable keyboard layout for your favorite language, Click 
on System -> Preferences -> Keyboard 
Select "Layouts" and slect the language (It is in country 
name order) so you might find yours in India. 
And add it to the list. 

If you add an applet in gnome's panels it will be much easier to use multiple keyboard layouts.
You can do it by right clicking on the panel, Clicking "add to panel"
and then selecting "keyboard indicator"
