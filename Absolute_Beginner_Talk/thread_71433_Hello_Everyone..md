### Post by Emerzen on 2005-10-03
Hey Animus,

Welcome!  Glad to have you and try to help as bumps in the road crop up...they become fewer and sometimes even fun.

1.) Regarding 32bit vs 64bit: install them both.  I have NO experience w/ 64 bit but you could make a large partition for 32bit Ubuntu and a small partition for 64 bit, then boot into either one as your heart desires.  That's the beauty of open source.

2.) Java is not installed by default--licensing issues don't allow it.  But, here's a link to tell tell you how to install the latest-> [http://ubuntuforums.org/showthread.p...d=1#post383058](http://ubuntuforums.org/showthread.p...d=1#post383058)

3.) XMMS - type 

sudo apt-get -i xmms-cdread 

to read from cd's (or go to Synaptic and search 'xmms' and install the package xmms-cdread)

3.) In general, multimedia requires some work but it's simple once you gain a little familiarity.  The best place for all newcomer's to start, in my book, is w/ the Unofficial Ubuntuguide [www.ubuntuguide.org](www.ubuntuguide.org) .  Read it from top to bottom and install everything that looks useful.  Keep in mind that it's current for Hoary (not Breezy yet) and not everything is compatible/necessary.  For example, I haven't needed to configure sound since Breezy.  In any case, it's a nice crash course in Ubuntu/Linux and gives you a hint at the fun/versatility.

-Take care
