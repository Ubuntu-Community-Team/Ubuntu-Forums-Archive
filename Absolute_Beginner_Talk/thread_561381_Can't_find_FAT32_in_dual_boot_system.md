### Post by nb123 on 2007-09-27
Oh, ok, it seems to be mounted at /windows, I presume that you've checked that this folder exists under your root folder? It should be mounted there, but if it's not, first make a backup of your fstab file, then try editing your fstab. The following will do this for you -

First copy this in your terminal (note: `whoami` uses backticks, not apostrophes. It's below the squiggly ~ next to the 1 key on your keyboard) :

```

cp /etc/fstab /home/`whoami`/fstabbackup
sudo mkdir /media/sharefat
sudo gedit /etc/fstab
``` 

Change this line in your fstab: --- 
```
UUID=DBA1-1E2C /windows vfat defaults,utf8,umask=007,gid=root 0 1
```

to this: ---

```
UUID=DBA1-1E2C /media/sharefat vfat defaults,utf8,umask=000  0  0 
```

After this, type:

```
mount -a
```

so that you don't have to restart your computer to see the changes in fstab mounting.

The above will ensure you have full permissions. There shouldn't be any more problems, let me know if it goes ok! If it doesn't, please post the output of 

```
df -h
```
