### Post by jonathan.lees on 2006-12-31
It's the candybar from [gdesklets]("http://www.gdesklets.org/?mod=project/uview&pid=32"). You can install gdesklets with:

```

sudo aptitude update
sudo aptitude install gdesklets
```

and then you can load the candybar and many other desklets
