### Post by Alterax on 2007-06-07
If you don't like it, get your Windows installation CD and key out.  Boot from the Windows CD-ROM and follow the steps.  You will lose your documents and files, so if there's anything important you will need to back them up to something like a memory stick or external hard drive.  Same goes for switching Windows to Linux.

I suggest trying it out as a LiveCD for a while and not installing.  Then if you don't like it, you have lost nothing.
