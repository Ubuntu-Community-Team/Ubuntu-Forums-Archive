### Post by jeffathehutt on 2007-03-23
Most logs are indeed stored in /var/log but certain programs store the logs elsewhere.

On Dapper, there is a link in the System->Administration menu called System Log.  If there is such a link in Edgy it should help you to make more sense of the logs.
