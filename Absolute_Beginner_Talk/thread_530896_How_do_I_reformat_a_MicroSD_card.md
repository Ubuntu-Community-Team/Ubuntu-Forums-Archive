### Post by George3 on 2007-08-21
Not exactly sure, but once Ubuntu mounts it, if you run the command "df -T" it will tell you the file system type used by the card (I'm guessing it will be FAT16) and then *I assume* you could format it using any method that supports that file system type?

(An easier way may be offered by the device it's used in - for example, digital cameras have a menu option to reformat their own media.)
