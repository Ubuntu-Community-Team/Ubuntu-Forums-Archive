### Post by jon.reeve on 2007-08-22
I had several problems with Wubi when I had it installed. For one, all the Wubi documentation says that it's a real installation and is not that much slower than an installation into a partition, but I didn't find this was true at all. Even on a fast desktop of mine I found it to be 3x as slow as a normal installation. When I finally installed Ubuntu the real way, ie making a partition, installing it there, my Ubuntu system was much faster and well worth the effort I put into it. In fact it was a lot easier than I thought.  

My advice is, unless you're just trying out Ubuntu for a few days, install a real installation and you'll probably be very surprised at how fast it is compared to Wubi.
