It's the Scale plugin being activated by Screen Edges function. 

To configure Screen Edges, it's Beryl Settings Manager -> General Options -> Shortcuts -> Screen Edges

To configure the Scale Plugin, go to Beryl Settings Manager -> Window Management -> Scale
