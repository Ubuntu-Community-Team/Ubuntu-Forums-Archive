### Post by tommcd on 2008-03-16
You could install Windows XP on as little as 10GB. I have done it and it is plenty. It depends what you want to use it for. Keep in mind you will need extra space on the XP partition for your programs, any data files, virtual memory. etc. You also need at least 15% of the disk free for proper defragmenting of Windows. So, at least 10GB for XP, plus whatever else you want to do with it. 
After XP is installed, boot up the Ubuntu live CD and make 3 partitions:
10GB for ubuntu root /. (can be pimary)
1GB swap. (can also be primary)
the rest for /home. (logical, so you have the option to re-partition it later if you ever need to).
