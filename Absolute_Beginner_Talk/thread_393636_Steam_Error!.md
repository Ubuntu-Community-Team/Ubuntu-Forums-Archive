### Post by carlosfocker on 2007-06-05
I would suggest installing envy to install your ATI drivers.  Its real easy to use.  Open up a terminal.  Then type

```
wget http://albertomilone.com/ubuntu/nvidia/scripts/envy_0.9.3-0ubuntu6_all.deb
```

Install envy

```
sudo dpkg -i envy_0.9.3-0ubuntu6_all.deb
```

Run envy from command line 

```
envy
```

or use the GUI tool located under Applications --> System Tools --> Envy. You might need to use the uninstall option before installing the new drivers.
