### Post by DaveBorealis on 2006-12-09
> **Phil Smith said:**
> However, when I deliver the machine to their house, I'm not sure how I'll modify things to suit their internet connection.

Hello,

This is a very nice sentiment, however for every Ubuntu machine you give to a new Linux user, you'll be committing yourself to possibly hours of tech support.  My experience is that the typical user has a hard time switching from one OS to another, especially if they've only used Windows their whole life.  In which case you might do more harm than good because you could be potentially setting them up for a frustrating experience.

If you really want to do this, then maybe try the following.  Perhaps have a Saturday afternoon workshop for everyone you set up with Ubuntu, and walk them through the various configurations and how to get things like OpenOffice to open and save documents, etc..  It might be fun, and I bet you'd personally learn lots in the process!

Best regards,
Dave
