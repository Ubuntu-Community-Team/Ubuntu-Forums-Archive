### Post by jesterthejedi on 2007-07-09
you need to edit that last file, /etc/X11/xorg.conf

try:
sudo gedit /etc/X11/xorg.conf

when the file opens up, change all the "800x600" values to the mode supported by your monitor. I have a Dell also, so I use "1024x768"

save the file and reboot your system.
