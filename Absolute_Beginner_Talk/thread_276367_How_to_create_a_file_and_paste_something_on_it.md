### Post by Kateikyoushi on 2006-10-12
For example with nano which is a small editor. 

```
nano mkdev.sh
```

Then paste into it and CTRL+X at the end.

### Post by fritz_monroe on 2006-10-12
Just launch a text editor.  Then paste into the file.  When you save it, just give it the name mkdev.sh  That's all there is to it.

If you are at the command line, do a vi mkdev.sh  Type in your commands and to save and exit, do a :wq  Saves out as the name you gave it.
