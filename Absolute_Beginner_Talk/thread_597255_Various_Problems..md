### Post by kebes on 2007-10-30
> **tarchy said:**
> 1. What is that client that I've heard about, it's a Firefox based client, apparently is faster on Linux/AMD processors. 
You might be thinking of [SwitfFox]("http://getswiftfox.com/"), it's a Firefox-based web-browser optimized (for Linux) to be leaner and faster.

> 2. How can I speed up my boot up/shut down with or without programs?
3. How can I speed up my application loading?
If your computer seems too slow with Ubuntu (possibly because it's older hardware?) you could consider running Xfce instead of Gnome as the desktop manager. That is, use Xubuntu instead of Ubuntu. I believe you can install Xfce by installing the package "xubuntu-desktop."

[Xfce]("http://www.xfce.org/") is a lightweight desktop environment designed to run smoothly on older hardware.

Hope that helps.
