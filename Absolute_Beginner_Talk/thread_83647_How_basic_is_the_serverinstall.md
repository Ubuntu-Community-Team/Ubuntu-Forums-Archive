### Post by davmac on 2005-10-29
The server install should be fine for what you describe. If your pcmcia network card works under the full install then it should be ok under the bare install though if you have to do any configuration you'll obviously have to do it without the aid of any of the gui tools.

Have a look at [http://www.ubuntuforums.org/showthread.php?t=75971](http://www.ubuntuforums.org/showthread.php?t=75971) for a good guide to starting with a server install and then adding xfce etc.

HTH,

Dave Mac
