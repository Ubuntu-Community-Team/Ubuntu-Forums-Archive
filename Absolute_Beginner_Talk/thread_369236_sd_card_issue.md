### Post by xmastree on 2007-02-24
There may well be a hidden trash folder on the card, If you open it with nautilus (the default file manager in gnome) and press Ctrl-H to view hidden files, there may be a folder called **.trash-username** containing deleted files.
Assuming you really wish to delete what's in it, you can safely delete the folder.
