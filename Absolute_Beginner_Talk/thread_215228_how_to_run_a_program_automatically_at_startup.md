### Post by maniacmusician on 2006-07-13
it would depend on what distro you're using. If you have gdesklets installed, then its probably either Ubuntu or Xubuntu. The method by shanepardue will work for gnome. for xubuntu, go to Applications (top left hand corner of the screen, unless you've moved it), Settings, Autostarted Applications. Enter the name (Gdesktlets), a description if you want, and I think the command to start the daemon is just gdesklets.
