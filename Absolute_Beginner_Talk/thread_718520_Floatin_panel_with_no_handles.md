### Post by Hospadar on 2008-03-08
I don't think you can, actually.  the handles don't change in appearance, so unless you go edit the panel code you might not be able to change the look.  Also, panel transparency only applies to the desktop at the moment, not windows.

If you want a nice floating panel like that, you might want to look into Avant Window Navigator (AWN).  it gives an osx-like panel.

Installation: [http://wiki.awn-project.org/DistributionGuides#Gutsy_Gibbon_.287.10.29](http://wiki.awn-project.org/DistributionGuides#Gutsy_Gibbon_.287.10.29)
