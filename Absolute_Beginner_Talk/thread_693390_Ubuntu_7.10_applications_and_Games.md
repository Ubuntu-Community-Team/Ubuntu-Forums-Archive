### Post by aquavitae on 2008-02-11
> **kraddark said:**
> emulators for different consoles..Not entirely sure what you mean by that?

The media players I like are totem for movies (the default) and amarok for music... but there are plenty of threads arguing over which players are best.  The only problem with amarok is its dependant on a lot of kde libraries so starting it up in gnome is quite slow.  VLC is quite popular too, although I've never used it myself.

Stylers: Ubuntu comes with compiz installed by default, which is similar to Beryl.  Right click on the desktop and turn on desktop effects to activate it.  There is also a package to customize it - search for 'compiz' in the synaptic package manager and you should find it.

### Post by JoshuaRL on 2008-02-11
I agree, just try out the games and if you don't like 'em, just uninstall.  The cool thing with Linux is that it's all free so it doesn't cost anything to try.

But for Beryl, don't bother.  It got merged into Compiz and now they're called Compiz Fusion.  It's installed already, but get the Advanced Desktop Settings Manager in Add/Remove.

I would also suggest Avant Window Navigator.  It will put a Mac OS X-like Dock on your desktop.  A lot of people really like that.  And I believe it will be in the repos, and maybe even installed automatically, in Hardy Heron.  That's the next version that is going to be released in April.
