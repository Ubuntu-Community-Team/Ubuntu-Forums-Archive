### Post by Daniel15 on 2006-09-27
> as far as security etc
Linux is great for security. On a normal system, you don't really need to do much (it's quite secure out of the box).

>  For some reason I am having a dificult time installing applications such as realplayer, macromedia flash player, and other desired applications
A lot of programs are installable via the in-built package manager. To access it, click System --> Administration --> Synaptic Package Manager. From there, you can choose what to install. Make sure you into the repository config, and enable all the repositories. Once that's done, every available package will be shown in the list.
Some programs (like Flash Player) aren't listed there (Flash player is non-free [Source code is not available], so it can't go in the Ubuntu repository). However, there is an easy way to install them... It's using a program called 'Automatix'. Automatix can install lots of programs that aren't available in the Ubuntu repositories, including Flash Player, multimedia codecs, and more. See [http://getautomatix.com/](http://getautomatix.com/) for more details, and instructions.
