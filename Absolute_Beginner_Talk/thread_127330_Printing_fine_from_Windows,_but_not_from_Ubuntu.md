### Post by facefur on 2006-02-08
I've been through something similar, but I now have both my Epson R200 and HP Designjet 100 working, if not perfectly, then very well.

I'd start with the printer installation process from the System->Administration menu.  If your printer is listed, as a local printer, I think, then Ubunutu has at least found it.  The next step is to insure that you have the proper driver and printcap files.  If you find your printer, you need to follow the menu steps to install the printer from the list.  Breezy (5.10) has a much longer list than Hoary (5.04) has.

If that doens'nt help, try the Linux Printing website at [http://www.linuxprinting.com](http://www.linuxprinting.com).

Good luck.
