Hi all

I am a 55 year ole noobie and just migrated from windoze2000. Installed Ubuntu6.10 in a dual boot mode on my generic 1GHz Celeron/640 Mb PC. The install went without a hitch, though I did have some initial issues with the partition.

The  HDD was earlier partitioned with Partition Magic 8 and for some reason, I could not make Ubuntu  see the root partition, kept coming up with an error - "cannot find root file system". The fix was to delete the partition with gparted and configure it again, Sweet !!!

Updates -- I only have a slow internet and finally figured out  through these forums, about downloading the updates at work and etc etc. The Easter weekend comes by and I just did the down loads over a "few" hours on the slow dial-up.

I must have been blind all these years not to switch away from windoze!!!!

In general, no. If you are sharing downloaded files with windows users, like on a file server in your house for example, you can install an anti virus to help out your windows using friends. Clam antivirus is the one I've used. But virii themselves won't hurt your linux system.
