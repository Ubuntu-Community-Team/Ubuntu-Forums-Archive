In case somebody else wants to do it, more detail:
->open terminal
->type ```

->user window opens
->find and select item to be added
->go to 'Bookmarks' and 'Add Bookmarks

### Post by CatKiller on 2006-11-04
You don't need to open the terminal to run Nautilus. Just pick some place already in the Places menu. Or, if you've got launchers on the desktop for your Home folder or Computer or whatever, click on them. Or, if you really want to type the command, Alt-F2 will open the Run Application box that you can type into.

Just some more options for you.
