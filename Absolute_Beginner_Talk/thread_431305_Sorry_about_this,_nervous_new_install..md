### Post by Cypher on 2007-05-02
I don't envision you facing any problems with the hardware you have. However, the first thing I would suggest is to grab the latest 7.04 Feisty Fawn Desktop CD which is a LiveCD. Put that in and boot into it, you will now have a full-fledged Linux desktop that you can play with and get comfortable with.

Once you're ready to take the plunge, you'll need to create some free space (unpartitioned) for Ubuntu. For this, you'll have to resize one of your existing partition to make some room. Once Ubuntu is installed and it installs the GRUB bootloader, you will automatically have the dual-boot option set up for you..

### Post by Drooling Iguana on 2007-05-03
Wine can theoretically run a variety of Windows apps, but I've never been able to actually get it to work with anything (then again, I've mostly been trying it with games, which aren't Wine's strong suit.) Nearly every major Windows program has a Linux equivalent, though, so you shouldn't have to worry too much. I'd still reccomend you keep a Windows partition around for when you absolutely, positively need to run a Windows-only program, though.

As for your Wacom, if it worked in the LiveCD, it works, period. The LiveCD doesn't touch your Windows installation at all, so anything that works there will work when you do the full install, too. I've never tried running Linux from an external USB drive, but I wouldn't be surprised to find out it can be done. You can stick Linux into all sorts of places that you wouldn't normally assume an OS could be installed.

### Post by teddybairs1 on 2007-05-03
I would recommend installing Ubuntu to your drive E:\ partition, it looks to be the one with the most free space and the least amount to lose. The biggest problem I can see you having is your video card. You shouldn't have any problems using it except for the OpenGL rendering and using the Desktop Effects, but I could be wrong on that. In general the two chipsets I've seen people having the most problems with are NVidia and ATI.

The safest thing to do where installing it to a partition is concerned is to put it on a partition on the internal drive.

Be prepared after installation to hunt down all the codecs and libraries which will make running multimedia possible on Ubuntu (i.e. DVD, MP3, WMV, etc.) It's not available out of the box but can be downloaded from the medibuntu site by adding it to your Synaptic repository list. Same thing for Flash 9, and certain other pieces of software which just make life on a computer that much more comfortable.

Read everything you can on this forum site. Do searches for things like "medibuntu" and "DVD playback". If you have a question or run into a snag, don't hesitate to come running for help. Everyone else does; that's how we learn, grow, and are able to help others out from the experience.

One more thing, you will most likely lose your PIXMA printer where Ubuntu is concerned. I have a printer from the PIXMA line, and still can't find a working driver for it. I have to dual boot between Windows and Ubuntu to print anything.

### Post by kolinab on 2007-05-03
Ken,

Glad everything is working out! That's super. I never dual booted, that's great that you got it working. At least this way you can 'run home to windows' if you need to in a hurry for some little project and don't want to figure out the linux equivalent in 2 minutes (which is usually possible anyway, for what it's worth). 

One thing I would recommend to you right off the bat, this will save you much heartache and frustration, since you seem inclined to partition your drives up in an organised fashion anyway, I would do everything you can to create your /home on a separate partition (if you haven't already).
