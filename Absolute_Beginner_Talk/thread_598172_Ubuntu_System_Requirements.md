### Post by macogw on 2007-10-31
I've done it on a 5GB hard drive.  Ubuntu itself takes like...1.5GB probably, then whatever you install adds to that.  If you have a full gig of RAM (assuming you won't be making Apache go berzerk, that is) and won't be hibernating, you can get away with not making a swap partition too.

If you do a server install (no GUI), it'll be smaller, then you can just install whatever pieces you want.  Fluxbuntu might also be a good choice since it's even more lightweight than Xubuntu (or at least it looks like it).
