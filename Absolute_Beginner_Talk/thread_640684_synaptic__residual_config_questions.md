### Post by disturbedite on 2007-12-14
do you know what residual config is?

its usually the type of stuff in your home folder. for instance (this is just a hypothetical example):
say you install wine and install some windows programs.  then you decide uninstall wine, uninstalling it will obviously remove wine, but it leaves the user modified/installed stuff.  (like the windows programs you installed).

but, doing a Complete Removal of wine will also uninstall wine along with all user modified/installed stuff (in your home folder).  (as well as config stuff).

and to clarify, if something is in the residual config section, it *isn't* installed, its config files are still installed.  (so that in the case you want to reinstall it, you don't have reset all your settings).
