### Post by Ub1476 on 2008-02-04
Open add/remove and install "startup-manager". With this tool you can modify the resolution and depth (this is what causes the "no logo" bug). I suggest you set a** safe resolution and depth** though. Now I currently have 1280x800 with 24 depth, on boot I've set it to 1024x768 with 16 depth.
