### Post by erginemr on 2007-12-11
Please copy your xorg.conf file first, to restore it in case things go wrong, with:
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.bak
```

Then, open your xorg.conf file for editing with the root permissions:
```
gksudo gedit /etc/X11/xorg.conf
```

[Note: As a rule, you should use sudo to get root privileges for console programs, and use gksudo to get root privileges for graphical programs.]

And change the line:
```
	SubSection "Display"
		Depth	24
		Modes		"1024x768"	"800x600"	"640x480"
	EndSubSection
```

as

```
	SubSection "Display"
		Depth	24
		Modes		"1280x1024"     "1024x768"  	"800x600"	"640x480"
	EndSubSection
```

Restart your X server with "Ctrl+Alt+BackSpace" key combination, to see if your resolution has switched to 1280x1024. 

If not, or worse, if you cannot start X server after this change, switch to another virtual terminal with "Ctrl+Alt+F1 to F6" (Your X session is under "Ctrl+Alt+F7" by the way), login if necessary, and issue the following command to restore the backup you have just made:

sudo cp /etc/X11/xorg.bak /etc/X11/xorg.conf

Good Luck!
