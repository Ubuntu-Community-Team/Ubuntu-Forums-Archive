### Post by Marsolin on 2006-10-31
Kubuntu will restore your session on start up. Controls to change this are under System Settings | KDE Components | Session Manager. I'm assuming that you tried closing them before logging out, so I suspect that the On Login setting you have is "Restore manually saved session". I'd suggest changing it to "Restore previous session".
