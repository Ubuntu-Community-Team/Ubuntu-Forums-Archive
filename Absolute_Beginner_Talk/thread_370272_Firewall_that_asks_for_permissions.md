### Post by mcduck on 2007-02-25
Firestarter _is_ the 'lazy mans firewall'. Just use the default settings, which allow all outbound connections and block all incoming ones. This is the best & easiest solution for normal desktop use, as long as you only run apps from repositories (and other trusted sources) there is no need to limit any outbound connections from your machine :)
