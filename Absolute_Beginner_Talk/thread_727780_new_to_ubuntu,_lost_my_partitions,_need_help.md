### Post by Sef on 2008-03-18
> /dev/sda3 7650 14592 55769616 7 HPFS/NTFS

I imagine that the files you want are saved on this partition.   What is needed is ntfs-3g; it will allow you to read and write to NTFS.  

To get it:  

Applications > Add/Remove > Change the top right box to 'All Available Applications' > Search: type in ntfs-3g > click on NTFS Configuration Tool > click apply changes > click apply > that tool will be downloaded to your system.  It will be under Applications > System Tools
