### Post by Bartender on 2006-03-15
There's gotta be some sort of identification silk-screened onto the motherboard.  I've been able to find .pdf manuals online, even for weird boards like the Jabil's that were in Gateway's a few years back.
Very handy to find out whatever info you can about your mb.  
Also, I've read that you should have the biggest stick in the #1 RAM slot.  If you don't replace the 64MB with a bigger one, at least make sure the 256 is in the #1 slot.  If you have questions about the stick of RAM you've been offered, pull out all existing and put the new piece in the #1 slot.  If it won't boot, you'll know that it won't work w/ your PC.  If you do some experimentation, remember - it's gotta be in the #1 slot if you only have one piece installed!.
