Download the icons, it should be a tar.gz file. Then click on System > Preferences > Theme . The Theme Preferences will open, Drag the file onto the themes, the install box should pop up. You could also click on Install theme and browse to the file. To select the icons, click on Theme Details and then the icons tab, select the icons and click close. :D
