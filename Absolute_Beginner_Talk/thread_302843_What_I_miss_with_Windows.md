### Post by jaboua on 2006-11-19
It is also possible to make gnome display your home folder (/home/username) as the desktop - if you press Alt-F2, enter "gconf-editor", press enter, and then browse to /apps/nautilus/preferences and check "desktop_is_home_dir".

If you only want an icon to your home directory, go here instead: /apps/nautilus/desktop and then check "home_icon_visible"

Hope this helps.

EDIT: If you create the "Documents" folder mentioned above, you should be able to go to /apps/nautilus/desktop and check "documents_icon_visible" in order to link it to the desktop. You might offcourse also manually create links to wherever you want...
