### Post by bollix47 on 2006-04-27
Type about:config in the location bar.

Type middlemouse.contentLoadURL in the Filter field.

Double click value (true) will change it to false.

Re-start Firefox.

See the following for details:

[http://www.mozilla.org/support/firefox/mouse#n1](http://www.mozilla.org/support/firefox/mouse#n1)
