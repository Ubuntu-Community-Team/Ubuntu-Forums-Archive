### Post by cjparad on 2007-06-02
Hello, UBUNTU Users

I am new to UBUNTU I am familer with other Linux and Solaris builds. I have discovered that magical and wonderful thing called WINE, which stands for Wine is not a windows emulator.

If you install and configure WINE you can install windows based software to run on UBUNTU.

I have done this with little complications. Wine does come with the Feisty build, but needs installed and configured. Then start setting up your favorite windows based software.

For help with this check out:

Setup Inter Explorer with UBUNTU: [http://www.howtoforge.com/ubuntu_internet_explorer](http://www.howtoforge.com/ubuntu_internet_explorer)

This should get you on your way!

P.S.

I have installed the following:
IE 5.5
BeJewled
Adaware
WinAVI Video Converter

And what ever else my Hard Drive can hold.
