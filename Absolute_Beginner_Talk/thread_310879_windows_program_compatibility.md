### Post by aysiu on 2006-12-02
Depends on the program.

Your best bet is always to find a native Linux program.

There are some helper applications, though, like Wine, Crossover Office, and Cedega, that allow you to run certain Windows programs within Linux.

### Post by nixternal on 2006-12-02
Yes and no. They will not run natively, but you can use programs such as Wine, Cedega, or CrossoverOffice in order to do so. Cedega is for gaming, CrossoverOffice allows you to run applications like MS Office, Media Player, and such. Cedega and CrossoverOffice both cost money, whereas Wine can be installed from the repositories.

Wine's website: [http://www.winehq.com](http://www.winehq.com) (noted when I replied here the site was currently down)

Cedega's website: [http://www.transgaming.com](http://www.transgaming.com)

Crossover Office's website: [http://www.codeweavers.com](http://www.codeweavers.com)

More information about Wine can be found at [https://help.ubuntu.com/community/Wine](https://help.ubuntu.com/community/Wine)

I don't use any of these applications, and haven't used any in a long time. I have finally freed myself years ago from any Windows dependencies. From what I have heard from others, all 3 programs rock.
