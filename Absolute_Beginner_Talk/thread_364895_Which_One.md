### Post by jml on 2007-02-18
I've played with Linux since 1999, but I do not consider myself an expert.  I've used various distros over the years and have definite opinions.  ;) 

There are several ways to parse the various distros but the way I look at them is by how they handle package (software application,) management.  SUSE, Fedora Core, Mandriva use something called RPM packages.  My personal experience has not been very favorable  with these distros.  They do not support many of the functions users want, ( playing MP3's, Commercial DVD's, Flash, Real Player, WMA, or WMV files.)  These distros also make it very difficult to add these unsupported capabilities.

Another set of distributions are based on a different package management system utilizing .deb packages, and an application called apt-get.  This system is much more flexible and lends itself to customization more easily than the other distros.  Some of these distros are Debian, Ubuntu/Kubuntu/Xubuntu, Mepis, Linux Mint, Knoppix, Damn Small Linux.  For a good overview of Linux distros go to this site.

[http://distrowatch.com/](http://distrowatch.com/)

This site tracks the majority of Linux distros and is interesting reading in my opinion.

As for my suggestions,  Ubuntu is a very good first choice for a new user.  The install process is easy, the live cd option gives you a flaver of how everything works.  While it does not support proprietary multimedia formats, (MP3's, Commercial DVD's, Flash, Real Player, WMA, or WMV files,) out of the box.  Instructions on how to enable them are posted on this web site.  If you want to have these formats supported out of the box, then consider Linux Mint:

[http://linuxmint.com/](http://linuxmint.com/)

Its based on Ubuntu and include support for all of the proprietary formats out of the box.

If you need wireless networking support above all else, then I would definitely go with Ubuntu.  Getting wireless to work can be a challenge and this forum has a lot of smart people who can help you with that.  Just my two cents worth.  Good luck, and welcome to Linux.  :)
