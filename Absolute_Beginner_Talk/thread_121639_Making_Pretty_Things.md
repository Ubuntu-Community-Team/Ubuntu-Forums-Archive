### Post by JAwuku on 2006-01-25
Hello,

Installing skins in Mplayer involves a few, but only small steps.

Skins are available from [http://www.mplayerhq.hu/homepage/design7/dload.html](http://www.mplayerhq.hu/homepage/design7/dload.html)

Just scroll down to the Skins section and you can preview or download skins.

Also listed is the skin seciton in Freshmeat.net, at [http://themes.freshmeat.net/search/?q=mplayer&section=projects](http://themes.freshmeat.net/search/?q=mplayer&section=projects)

To install the skin, you can untar the skins and put their directories in the **~/.mplayer/Skin** directory. Create the Skin directory (**mkdir ~/.mplayer/Skin** if it does not already exist, before moving the skins there.

Make sure you set a default directory, by renaming one of the skin's directory to **default**

otherwise Mplayer does not know where to look. Mplayer is great, but can be unintuitive on this part.

I have no experience of Totem skins, or Rhythmbox as I use KDE in Kubuntu instead.

Hope the MPlayer skins tutorial helps.

If you get stuck, refer to the docs at [http://www.mplayerhq.hu/DOCS/HTML/en/index.html](http://www.mplayerhq.hu/DOCS/HTML/en/index.html)

specifically the skin section at [http://www.mplayerhq.hu/DOCS/HTML/en/gui.html](http://www.mplayerhq.hu/DOCS/HTML/en/gui.html)
