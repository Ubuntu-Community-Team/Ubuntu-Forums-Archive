### Post by Slushdoot on 2008-04-07
What sort of compatibility problems are you foreseeing?  Linux has better hardware compatibility that Windows does out of the box.  Most of the drivers you need are already installed in the kernel.  Rarely will you have to futz with anything except for proprietary video drivers.  Everything else, USB, chipset, LAN, audio, etc, are usually already ready to go.

Yes, it is.  If that disk is attached when you install Ubuntu it will be automatically set up as readable and writable.  It won't have any trouble accessing those files to play in a player like XMMS, amaroK, VLC, or Rhythmbox.

If you want to try it out a bit before installing, play around with the live CD a bit.

### Post by mivo on 2008-04-07
The Live CD (there is one for Hardy, too) should let you safely explore any compatibility issues before installing it. Not all of them, but certainly most. It's not so much that Linux isn't compatible with some hardware, but it is that some hardware is not compatible with Linux. :)

I completely understand your worries, though. When I switched to Linux, I was worried a bit too, because not only was everything quite different, but I also feared I may run into troubles. I didn't keep a Windows partition, and jumped right in. There were some issues, though no show-stoppers and I learned quite a bit while trying to optimize the situation.

By now I'm on my second Linux-only computer, both for home and work use. and I haven't regretted the switch-over. There are sometimes games that I'd like to play, but I also found that I enjoy the few commercial games that are available for Linux more deeply. The small selection doesn't allow for gaming ADD. ;) Productivity-wise, my computing time has greatly improved and I'm a much happier user. Linux, in many ways, has put the fun back into computing for me.
