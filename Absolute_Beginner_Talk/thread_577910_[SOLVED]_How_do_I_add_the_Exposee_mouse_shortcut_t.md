### Post by hotweiss on 2007-10-16
Hi, how do I add that Mac OS X Exposee top-left hand corner mouse shortcut to Ubuntu? In Mac OS X when you move the cursor to the top left-hand corner all your windows appear in front of you. Now, Ubuntu has that same feature, but you have to press CTRL-ALT-up to get this to happen. Is there a way to replicate the Exposee mouse shortcut to have all you windows appear?

---

### Post by Whiffle on 2007-10-16
Open up the compiz config settings manager, should be under System>Preferences.  Click on the Scale plugin, go to Actions tab, double click the line that says "Initiate window picker" , and tick the box next to TopLeft.  Hit ok and you should be done.
