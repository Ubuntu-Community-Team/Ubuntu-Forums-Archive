### Post by stmiller on 2007-12-11
[HandBrake]("http://handbrake.m0k.org/")! There is a [Linux interface]("http://handbrake.m0k.org/forum/viewtopic.php?t=3302") available on sourceforge now, too. 

FWIW you *can* make an exact iso copy of an encrypted DVD, but the only application that will play those is perhaps VLC. And the iso file will be of course the size of the DVD, anywhere from 4-8GB. This is the only true backup of the entire original disc, if that is what you are after.

All of the DVD rippers like HandBrake or k9copy take the main feature (or other parts of the disc if you wish) and encode them to an MPEG-4 (or other) video file. So no menus, no special features, just the video. So if that is what you are after, try HandBrake or others.

### Post by PhoenixP3K on 2007-12-11
I really take care of my DVDs. Sure some of them have minor dust scratch for some reason, but I don't see the day where I'll be careless enought to see them become unplayabled. But the other day I authored a DVD on my Vista machine (compilation of videos for a contest) and I needed to make extra copies of the DVD on site, so I used my Ubuntu laptop and k9copy to create an ISO image of the DVD and burn it with the built-in burner in Ubuntu. 
So if there is no copy protection on a DVD, like mine, the DMCA is not broken (outlaws circumvention of copy-protection), but if you live in different countries the laws are hopefully different.
