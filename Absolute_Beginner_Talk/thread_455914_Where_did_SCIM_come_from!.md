### Post by teasum on 2007-06-17
I had the same problem, and I had to turn SCIM off and uninstall it.  A small correction to the previous post: You don't need SCIM to input Thai and English; you only need the keyboard switching applet and the proper layouts installed.  I find this configuration (via System -> Preferences -> Keyboard) much simpler, at least for Thai.  For more complex input schemes, such as for Japanese or Chinese, I think SCIM might be required.
