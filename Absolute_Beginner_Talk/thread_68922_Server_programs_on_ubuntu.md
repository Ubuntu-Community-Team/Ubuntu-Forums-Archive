First question: it doesn't come installed with the install cd, but official packages for apache2 and other server software are available through Synaptic (apt-get) for you to easily install.

Second question:  Ubuntu uses the debian package manager and DEB files natively.  If you want to use RPM packages, you'll most likely want to install an application called Alien to help manage RPMs in Ubuntu.
