### Post by janrinok on 2005-12-11
First of all, make sure that the file is executable ie. chmod +x path/to/file/jre-1_5_0_06-linux-i586.bin    

Then a simple sudo path/to/file/jre-1_5_0_06-linux-i586.bin should execute it and the rest will be automagic.
