### Post by kellemes on 2007-11-12
Kubuntu-desktop just installs a bunch of KDE-packages and a hole lot more.. cannot imagine kdevelop being one of them.. you can simply install kdevelop separately.
If you open synaptic from the administration menu you can install all the packages you wish.. any conflicts with already installed packages will be handled so don't worry.
Just search for "kdevelop" and you get a list of stuff related to it.
Likewise for OpenOffice.
