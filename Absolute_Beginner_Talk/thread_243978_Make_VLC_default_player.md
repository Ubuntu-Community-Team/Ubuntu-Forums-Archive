### Post by benfindlay on 2006-08-25
If you right click on an media file, and go into properties, under "open with" you can specify the default player to use.

Unfortunately doing it this way means you have to do mp3s, wmas, avis separately, but once they're set you won't need to do it again for that particular file type

Seriously though, VLC is ace! ;)

### Post by muep on 2006-08-26
You first need to right click on a video file. Then, click on properties. In the Properties window, select the open with tab, where you can select the application you want for the file. If VLC isn't in the list, it is easy to add.
