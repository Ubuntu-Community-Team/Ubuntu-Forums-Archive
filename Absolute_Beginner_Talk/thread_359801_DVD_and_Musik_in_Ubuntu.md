### Post by wickstopher on 2007-02-12
Windows software will not work in Ubuntu, unless you are using something like wine (and even then, there is no guarantee that they will work).  There are, however, thousands of great applications for linux, and there are programs that will easily do what your listed windows software will do and more.  Just make sure that you have all of your repositories enabled under System > Administration > Software Sources and start looking around.

I would recommend Automatix to you.  It is an easy way to get a lot of things (most of the things that you mentioned) working very quickly.  Go to [http://www.getautomatix.com]("http://www.getautomatix.com") and download the easy direct installation for your particular distribution from their Installation page.
