This is prolly pretty simple but...I installed the new ATI drivers and have my resolution set to 1024x768 in Ubuntu, but the login screen resolution changed to 1280x1024.  I was wondering, how can I change the login screen back to 1024x768?

---

### Post by akiro.yamamoto on 2006-02-03
Try using 
System >> Preferences >> Screen Resolution
and set it to 1024x786 then check Use as Deault...
That worked on my box.
You can come back later and change your set-up after.
