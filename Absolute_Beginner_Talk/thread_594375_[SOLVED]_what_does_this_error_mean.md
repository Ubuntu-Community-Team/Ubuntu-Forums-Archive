### Post by Cato2 on 2007-10-28
Not sure if this is really a problem - it's probably related to GNOME, which uses DBUS internally.  If your system is working fine I would ignore it.

You need to provide more info as to what's happening in order to get a useful reply here...  What were you doing when you got that message, are you upgrading, which Ubuntu version, etc?

If this blocks your upgrade from Feisty to Gutsy, have a look at [http://ohioloco.ubuntuforums.org/showthread.php?mode=hybrid&t=554921](http://ohioloco.ubuntuforums.org/showthread.php?mode=hybrid&t=554921) and also google for "sudo dbus-launch".  I had something similar on upgrading Xubuntu via update-manager, and ended up having to do a more manual upgrade using 'apt-get dist-upgrade' etc.
