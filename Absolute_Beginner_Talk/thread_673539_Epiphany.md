### Post by FuturePilot on 2008-01-20
1. They should. Epiphany uses the Gecko engine so the same tweaks should still apply.
2. You can simply use the Import Bookmarks feature to import your bookmarks. You can't import Firefox extensions though. The good news is, in the epiphany-extensions package, there is an Ad Blocker extension that works just as good as AdBlock Plus. 
3. The only major difference is the interface. Epiphany integrates into the Gnome environment better.

### Post by CupofDice on 2008-01-20
I couldn't use epiphany at first because there was no tab mix plus, which meant there was no new tab button, but if you right click on the top area, you can add a new tab button (just drag n drop). Also to use greasemonkey scripts, go to userscripts.org, and click to install the script. It will take you to a page with the code. Copy that code into gedit, and save as whatevername.user.js in ~/.gnome2/epiphany/extensions/data/greasemonkey.
