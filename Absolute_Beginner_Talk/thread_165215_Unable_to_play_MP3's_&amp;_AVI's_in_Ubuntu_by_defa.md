### Post by tribaal on 2006-04-24
I agree this is pretty annoying to most of us, on a fresh install.

However this is not a bug, unfortunately: legal restrictions makes it impossible for Ubuntu to ship with the appropriate codecs. You can easily (very very easily) download and install them yourself, or with [Automatix]("http://www.ubuntuforums.org/showthread.php?t=138405"). 

I can't really get into details on why or how it is that way (legal stuff is a programming language I could never learn ](*,) ), but I'm pretty sure there is a darn good reason to it.

But hey, don't worry, you can still use [automatix]("http://www.ubuntuforums.org/showthread.php?t=138405") :)

Don't worry, there is an easy solution for almost every problem you may have in theses forums :)

- trib'

**Edit:** see post above if you need instructions / directions on how to install the codecs yourself (automatix will do a bunch of other goodnesses you might not necesserly need or want).
