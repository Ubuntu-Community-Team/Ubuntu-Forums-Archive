### Post by LaRoza on 2007-08-22
> **pgar23 said:**
> which language does Ubuntu use, Python?

The Linux kernel is written in C and ASM, many apps are written in C/C++. Python, and Perl are often used, the Ubuntu installer is written in Python.

The Python interpreter is already installed on Ubuntu, open a terminal and type "python", you will get a prompt. This feature of Python is very handy, being able to work in interactive mode, instead of having to write an entire script.

At the prompt, type : 
```

print "Hello, world"

```

This will be your first Python program!

To make an actual script, use an editor to put the code in the file, and run it with: 
```

$python <scriptName>

```
The $ is the prompt, don't type that!
