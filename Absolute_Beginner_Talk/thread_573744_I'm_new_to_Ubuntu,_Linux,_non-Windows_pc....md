### Post by R.Bucky on 2007-10-11
The 700+MB download is the LiveCD version, which means that you can use Ubuntu in RAM only without a hard install. But, you will install from that. The 4000MB+ or 4GB download is the desktop version. You will need a DVD-R for that. Stick with the LiveCD and install.

Other than that, check out tutorials online for burning .iso images from Roxio to make sure that it is done properly. It has been a while since I have used Roxio. That is about all that I can help you with on my end. 

Another option would be to order an ubuntu cd from [http://www.ubuntu.com/getubuntu/purchase](http://www.ubuntu.com/getubuntu/purchase)

Ask again if you have alternate additional questions.
