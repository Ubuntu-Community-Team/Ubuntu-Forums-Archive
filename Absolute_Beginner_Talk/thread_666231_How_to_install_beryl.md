### Post by overdrank on 2008-01-13
> **Obinoobie said:**
> I dont have the "Advanced desktop effects settings" option in the preferences menu :(

Hi you can use this command to install the manager
```
sudo apt-get install compizconfig-settings-manager
```
Then as stated before you can use Advance desktop manager it will be located under system,preference. Choose the general options , desktop size set the horizontal virtual size to 4. Also make sure the rotate cube box is checked.
