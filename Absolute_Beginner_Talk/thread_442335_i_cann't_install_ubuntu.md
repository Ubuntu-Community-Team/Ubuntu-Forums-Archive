### Post by Sef on 2007-05-13

The alternate cd is a text-based installer.   It is easy to install.    To get it go to [Ubuntu download]("http://www.ubuntu.com/getubuntu/download") and click on "Check if you need the alternate desktop cd."   That is right below the download button.   If you want to check an excellent tutorial, then the [Illustrated Dual Boot]("http://users.bigpond.net.au/hermanzone/") is for you.  If you are not dual booting, then just skip the parts about installing the NTFS or FAT32 partitioning.
