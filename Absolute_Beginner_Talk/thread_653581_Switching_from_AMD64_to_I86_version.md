The ubuntu install CD will install ubuntu no matter what OS is currently on the machine. Check the md5sums of the disk you burned, and make sure your PC is set to boot from the CDROM in the BIOS.
[https://help.ubuntu.com/community/HowToMD5SUM?highlight=%28md5sum%29](https://help.ubuntu.com/community/HowToMD5SUM?highlight=%28md5sum%29)
EDIT: I think a fresh reinstall of Gutsy would be the best way to upgrade and switch to i386.
