### Post by Atreus12 on 2007-02-11
I've read almost the entire thing, but I'm not sure if I can say if it would be helpful for you or not. I read through Ubuntu Unleashed 6.06 as I was installing and learning Ubuntu 6.06, my first experience with linux. I thought it was great, because it walked me though all the basic stuff that would have taken me forever to figure out. 

The book starts out very beginner oriented. As you move through the book, it gets more and more advanced. However, it still seems to be mainly an overview book. It touches on a lot of subjects, but nothing too in depth.

I read though most of the book, skipping stuff like 'setting up an apache server' and other sections which I knew wouldn't apply to me, and I'm certainly glad I read it. I learned a lot about linux that I'm not sure I would have learned otherwise.

I guess it depends on what you are looking for. If you could be more specific on what kind of book you want, I might be able to help more.

### Post by AndyCooll on 2007-02-11
I have the book and have been very disappointed. Essentially it is a rewrite of the authors "Fedora Core Unleashed" book with Fedora amended to Ubuntu. Unfortunately there are some glaring errors where he couldn't even get this right (such as referring to Yum for installing software). The author writes for the UK magazine Linux Format which I get every month and really like, but I have been very disappointed with this.

The Ubuntu Bible has been recently released, and if it's anything like the Debian Bible I'd highly recommend you purchase that instead.
