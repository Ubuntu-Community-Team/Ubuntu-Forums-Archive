### Post by spd106 on 2005-05-14
Hi, first you will need to check that your modems are not winmodems. If they are do a search to see if anyone else has got them to work with linux. They can be very hit and miss. 

Sometimes they have full support in the kernel, so you are ok.

To configure the modems look at this wiki page for help
[http://www.ubuntulinux.org/wiki/DialupModemHowto](http://www.ubuntulinux.org/wiki/DialupModemHowto)

I would suggest that you start your journey to linux by dual-boot. This means you can always switch back to windows in an emergency and not lose valuble time reinstalling. To do this you will need min 8Gb disk. It is possible on less, but it's kind of restrictive.

Install windows first on a primary partition, probably 5Gb+, you can set up the partition with the windows install wizard. Then install Ubuntu on the unused space. Just follow the wizard and it will sort out most of the difficult parts.

You got an error before because Linux and windows use different file systems, ext3 and ntfs respectively.

For more information check out this wonderful guide
[http://ubuntuguide.org/](http://ubuntuguide.org/) or the wiki.

Hope this helps.
