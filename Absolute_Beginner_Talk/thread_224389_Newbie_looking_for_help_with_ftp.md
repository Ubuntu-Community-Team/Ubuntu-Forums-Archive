I figured it out, before I got a responce.  Thanks though!
For those who are looking here is how I did it:
1. goto "Places"
2. "connect to server"
3. Service Type: I needed FTP (with login)
4. Server Name: ftp.providername
5. Port: usually "21"
6. Folder: didn't apply
7. Username: the user name you login with to connect to server
8. Name to use with connection: name it whatever you want your connection name to be.

I hope that this bit of info will help someone in the future.
