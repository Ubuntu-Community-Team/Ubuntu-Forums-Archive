### Post by umattu on 2007-12-07
This depends on alot,

ubuntu software raid, won't work for dual boot

You will have to use a Hardware RAID controller for dual boot

What raid level are you wanting to use?
RAID 0 - striping
RAID 1 - mirroring

Dual Booting on a RAID array can be done, HOW you implement the configuration will determine how easy/difficult it will be.
