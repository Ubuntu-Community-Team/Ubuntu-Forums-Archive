### Post by jken146 on 2007-12-09
Better?  How so?  It depends what you want.  Flash drives are good for quick read-write access, but the memory can be more volatile than some other methods.  Hard disks are generally very good, plus you get more storage, but they're more expensive (not per MB).  Hard drives can fail when they get old.  CD/DVDs should last ages and you won't get corrupted data if you burn them right.

The best way to ensure you don't lose precious files is to keep multiple up-to-date copies.

I'm not sure what you're looking for, here.   It's not like we know of some hitherto unknown storage medium.  There are basically three data storage methods in use today: solid-state (flash) memory like that in a USB key, optical discs like DVDs and CDs, and magnetic storage like hard disks.
