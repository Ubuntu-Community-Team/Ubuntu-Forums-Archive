### Post by Wobblybob on 2008-04-09
Hi I've used this app to create Gutsy live CD's and yes they perform just like the normal live CD, you can add apps to the iso along with start-up options but I've not tried to create user accounts. In some cases you will find that you need to use DVD if you add too many apps.

There is a user guide in pdf format [here]("http://files.filefront.com/Reconstructor+guidepdf/;8845802;/fileinfo.html/") which helped me.
