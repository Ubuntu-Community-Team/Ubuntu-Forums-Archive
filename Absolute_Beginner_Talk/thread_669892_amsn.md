### Post by crjackson on 2008-01-17
Save the images in your home folder.  You should have a sub-folder there called pictures.  If not, the right click on an empty space inside your home folder and use the menu option to create a new folder.  Do that and name the new folder pictures.  Save you pictures there.  Then when you want to load one of the images for an application, browse for it in your pictures folder.
