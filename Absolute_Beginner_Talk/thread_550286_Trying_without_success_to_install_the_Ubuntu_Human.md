### Post by cubeist on 2007-09-14
Changing themes is really easy.  Go to System/preferences/theme  Once that is open there is a button on the top right called "install theme" ... click that and select the theme you downloaded... if the file you downloaded refuses to work it is probably not a proper theme... where did you download it from?  Have you tried the themes on gnome-look.org... do any of those work?
