### Post by sapo on 2005-05-25
Yes.. decompress means to extract...

cd is a command do enter a folder...

Try reading the README or INSTALL file in the folder  you have extracted the files  :razz: 

Usualy to install linux appz you will need to run a terminal.. then type it:

tar xzvf app.tar.gz -> decompress the file
cd app -> where app is a folder created when you extracted the file
./configure
make
sudo make install

Btw this is for source codes... Some programs have a installer.. so you have to type:

./install

But it can change.. the best way to install things is using synaptic and searching for pre-compiled softwares for ubuntu...
