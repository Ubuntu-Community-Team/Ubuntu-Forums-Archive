### Post by bodhi.zazen on 2006-11-16
gregmcavity: If you already have Ubuntu installed, and are going to install Fedora, **there is nothing you have to do other then run the fedora install disk**. Install Fedora into the Ubuntu partition and re-cycle swap. Done.

There is no need to remove Ubuntu. The Fedora install will reformat your Ubuntu partition and re-install GRUB. You will be able to boot windows or fedora.
