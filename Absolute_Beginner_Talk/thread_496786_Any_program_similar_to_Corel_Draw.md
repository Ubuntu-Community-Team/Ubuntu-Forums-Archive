### Post by Kumeelyun on 2007-07-09
I've been using [Inkscape]("http://www.inkscape.org/index.php?lang=en") for my vector work and [my cartoon]("http://www.hoodyhoo.com/"). It works well for me, even when I'm using the latest build.

There's another program called [Xara Xtreme]("http://www.xaraxtreme.org/") which is a port of the Xara program for Windows. It works OK, but it isn't completely ported and no steady news of it's development has been posted for over a year due to flux in the company.

Both of these are easily installed under "Add/Remove Programs" under the Applications menu.
