### Post by Hospadar on 2008-01-16
You might look into gpsd, I know it has some web interface, but I think maps can be downloaded.  That said, I don't know that it does any route finding, I think it's really just for using with a gps.  You could look into getting yourself a copy of garmin's mapsource software and running it under wine.  

I'm running mapsource 2008 on wine just fine (although it's a tad slower than on windows, that's probably due to the ridiculous 3d fish swimming behind my translucent desktop compiz nonsense) the only problems I have with it are that it doesn't transfer to my serial connected gps, although I think a more modern usb gps connection would work fine.

If you had less than savory ideals about software copyrights, I bet you could find it on a certain swedish website.
