### Post by jujoje on 2006-09-14
Hi,

I'm trying to do the same at the moment. I don't think that the DWL-122 dongle is detected automatically, rather you'll need to install a driver to get it working.

The most useful link I've found so far is [http://linux-wless.passys.nl/query_part.php?brandname=D-Link]("http://linux-wless.passys.nl/query_part.php?brandname=D-Link")
 which links to the manufacturers driver, although I haven't tried installing it yet.

Below are some links I've found on the Ubuntu forums relating to getting this working. I haven't tried them myself yet but could be worth a try:
[http://www.ubuntuforums.org/showthread.php?t=106846]("http://www.ubuntuforums.org/showthread.php?t=106846")
[http://ubuntuforums.org/showpost.php?p=618022&postcount=3]("http://ubuntuforums.org/showpost.php?p=618022&postcount=3")
[http://www.ubuntuforums.org/showthread.php?t=69575&highlight=dwl+122]("http://www.ubuntuforums.org/showthread.php?t=69575&highlight=dwl+122")

Also it might be worth posting which version of the DWL-122 you've got (i.e. DWL-G 122 A1) because I think that each of the different revisions uses a different driver. If you have the DWL-G122 A1 then the wireless-ng driver might get it working for you. I've got a DWL-G122 C1.

I'll probably give it a go tomorrow and I'll let you know if I have any success.
