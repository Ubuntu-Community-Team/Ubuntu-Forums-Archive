### Post by Crooksey on 2007-02-13
Sources are where you get your programs from.

Add and Remove software used one main Ubuntu server.

Synaptic and apt use the /etc/apt/sources.list file for sources, and can be edited at your wish.

### Post by pay on 2007-02-13
