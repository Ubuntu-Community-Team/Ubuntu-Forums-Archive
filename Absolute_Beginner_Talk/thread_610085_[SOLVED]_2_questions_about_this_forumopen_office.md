2) Should be able to do so from the Thread Tools menu at the top of the post.

### Post by carlosjuero on 2007-11-11
There is an issue with OpenOffice and certain Themes/Styles :/

What I did was remove the gnome theme support from Open Office and haven't had it crash again yet. Try uninstalling the openoffice.org-gtk & openoffice.org-gnome packages (uninstalling the -gtk package will uninstall both).

To do this use Synaptic Package Manager for ease (System -> Administration -> Synaptic Package Manager - enter your root password - click on Search and then search for openoffice.org-gtk). Check the package(s) for complete removal, apply and then try Office again.
