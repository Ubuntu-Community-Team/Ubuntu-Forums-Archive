### Post by lbyrd33 on 2007-04-13
For your first question, most distros should work fine with your pc. With the 8 Gb hard drive, os's like ubuntu and fedora will require about half of that. You have plenty of room to play around with the other hard drives so I would give that a shot. Also, keep in mind that many linux distros have switched completely to live cd's which will enable you to get the feel of the operating system before using it. I will say, do not judge the ubuntu live cd as it is pretty bad compared to others that I have used. Ubuntu is a very good system as a whole.

### Post by wormser on 2007-04-13
I would just remove that small 8 gig drive and keep the other 2.  Split them up how ever you like but make them extended partitions so you can have more of them. Remember to have a swap partition that is double your ram.

Ubuntu is a great distro with tons of support.  You might want to try CentOS which is just a free version of Red Hat.  Gentoo, Suse and Slackware are others you should try.

You might run into some boot issues when you get over 2 distros.  Try googling for a guide, it might save you some head aches.
