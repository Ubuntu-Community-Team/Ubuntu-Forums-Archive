### Post by Qrk on 2006-04-03
The DVD is 2.9 GB.

On an ADSL modem, it shouldn't take too much longer than a few hours.

Note that the DVD isn't necessary. You can get the same effect by using a CD and installing packages from the "main" repository. However, if you want both KDE and Gnome (along with other software not found on the CD) on a computer without internet access, the DVD is the way to go.
