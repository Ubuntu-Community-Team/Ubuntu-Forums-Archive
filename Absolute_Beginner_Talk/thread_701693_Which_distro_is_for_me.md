### Post by bobbocanfly on 2008-02-19
Standard Ubuntu would probably suit you fine. It is stable whilst staying up with most newer technologies. The main difference between the different Ubuntu Derivatives is the desktop environment which is mostly just user preference.

If you are looking for a really stable environment (stability is your main worry) then Ubuntu LTS (Long Term Support) will be the way to go as you are supported for a lot longer than the standard releases. The new LTS release of Ubuntu (standard, Kubuntu 8.04 isnt going to be LTS) in April will be stable and supported until 2011.
