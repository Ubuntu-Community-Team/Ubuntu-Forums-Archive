### Post by strabes on 2007-10-17
Please search the forums and the internet before asking a question. This solution can be found in the second result of a google search for "mp3 amarok."

You should simply install ubuntu-restricted-extras and libxine-extracodecs. See [this page](http://strabes.wordpress.com/2007/04/01/get-mp3-support-for-amarok-in-kubuntu-linux/) for more information. If you want to watch encrypted dvds and other proprietary formats you should also install the w32codecs and libdvdcss2 packages, located in the [medibuntu repository](http://medibuntu.sos-sts.com/).
