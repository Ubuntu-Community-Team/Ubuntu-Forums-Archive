### Post by aysiu on 2007-02-12
I'd say since Mepis is based on Ubuntu, they're equally stable.

They are different in some ways, though, that have little to do with stability.

Mepis is KDE. Ubuntu has KDE, Gnome, and XFCE flavors (Kubuntu, Ubuntu, and Xubuntu, respectively).

Mepis includes a lot of proprietary software and codecs (Flash, Nvidia drivers, Java, Skype, Adobe Reader). Ubuntu does not include them.

Mepis has a lot more point-and-click options than Ubuntu (reinstallation of Grub, reliable mounting of any partitions or drives, installation of Nvidia drivers during OS installation, default availability of "browse as root" menu item).

Mepis has an Earthlink icon on the desktop and a whole bunch of random other things (aquarium applet in the panel, weather applet). Ubuntu strives for simplicity--one application per task, not a lot of clutter in the panels.

Ubuntu has six-month releases and is the base for Mepis. Mepis isn't the base for Ubuntu.

Ubuntu has more knowledgeable users than Mepis.

For new users, I'd recommend Mepis as a first distro, but I'd definitely also recommend Ubuntu as a *second* distro.
