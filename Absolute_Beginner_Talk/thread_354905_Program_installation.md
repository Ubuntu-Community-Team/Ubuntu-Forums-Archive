### Post by Maestro23 on 2007-02-06
Open a terminal and type:

#whereis opera

Will show you the path to the the bin file.

Can then type opera from terminal or create a gui launcher on your desktop.

### Post by Ptero-4 on 2007-02-06
type opera in the run box (press alt-f2 to bring it up) to ensure that opera is actually installed. You may try to make a menu entry with the menu editor or install the menu-xdg package to get the ¨Debian¨ menu.

### Post by aseneca on 2007-02-06
Also try right click on your applications menu, edit menus which should open alacarte.  Highlight the internet category of your menu and check the checkbox next to opera in the right panel of the  menu editor window.  Not all the time does ubuntu make an installed program visible in your applications menu.
