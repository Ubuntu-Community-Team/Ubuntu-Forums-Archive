### Post by Sef on 2008-01-08
1) Did you [md5sum]("https://help.ubuntu.com/community/HowToMD5SUM") the download? (Verifies a good download.)

2) Did you burn the iso as an iso? (Not as a data disk.)

3) Did you burn the iso at 4x or less? (Faster may corrupt the burn.)

4) Did you 'Check Disk for Errors'.  (At start/install menu, go down toward the bottom.)

### Post by Eddie Wilson on 2008-01-08
Good Day,
It sounds like you may need to try the alt. version. I've had a few issues with the desktop version on some computers but the alt. version seems to always work for me. If you have a new drive and good media then you do not need to burn at a slow speed. You will only have problems with a fast burn if you have defective hardware or poor media. It sounds like you are doing everything correct. The only thing that I can suggest is that you try the alt. version.
Good Luck,
