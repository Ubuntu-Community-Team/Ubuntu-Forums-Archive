### Post by jobezone on 2006-06-06
When you click on the Icon it a file's property window, comes up a new window which acts a little strange. When you click browse, don't select the icon in the file dialog but instead the directory containin git. Then all png's and xpm's(I think) inside that folder will show in the previous window. Then you choose which one you want.
