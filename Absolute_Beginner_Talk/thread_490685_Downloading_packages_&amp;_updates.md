### Post by sebast on 2007-07-02
Go have a look at [APTonCD]("http://aptoncd.sourceforge.net/")

I neved used it, but from what I heard, it permits you to easily get the packages you would normally get on the Internet on a CD.

Just in case you don't know, the APT part from APTonCD is the package management system of Ubuntu (meaning it manages the way you install software, but also do updates, remove things. In other word it gets sure you have all the files to run a software and that things don't get mixed up)
