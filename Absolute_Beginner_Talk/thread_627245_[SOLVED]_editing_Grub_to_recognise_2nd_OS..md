### Post by PmDematagoda on 2007-11-30
Forgive me for taking time to reply hogwartsnigel. But as it is, Sabayon's menu.lst file states:-
```

root (hd2,4)
```

Yet what you put into Ubuntu's menu.lst file is:-
```

root (hd0,1)
```

Do you realise the error? The root entry must be the same, otherwise GRUB will read the wrong drive and partition and make a mess of booting Sabayon.
