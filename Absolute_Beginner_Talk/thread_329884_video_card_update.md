### Post by kinematic on 2007-01-03
he gave you a link ;) 
go to albertomilone's site and select the "new legacy" repository.
there you'll see a repository fro dapper drake and edgy eft.
open gedit as root with this command
> sudo gedit /etc/apt/sources.list
and add either the repo for dapper drake or edgy eft to it.....close it once you're done and select to save.
then open synaptic,refresh the list,search for nvidia and select the driver.
