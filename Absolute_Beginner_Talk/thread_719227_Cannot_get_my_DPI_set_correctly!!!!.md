### Post by mikeyphi on 2008-03-09
I'm using standard Ubuntu - so not sure if you have these programs but guess you must have something similar!
To change screen resolution I would open System/Administration/Screens and graphics
after inputting password you get a GUI
select whichever resolution is best from those provided (from your post there are many!)

For screen text - open System/Preferences/Appearance and under the fonts tab - they should (normally) be set at 10

If that's not the problem - and I've missed the real question, sorry ...but the final suggestion is to do as suggested in your xorg.conf file:
open a terminal and enter:

sudo dpkg-reconfigure -phigh xserver-xorg

After asking for your password you will get a lot of text; after each section there will be a question and a default answer: you should accept the default answer unless you know better!

When finished reboot. you should then be in a better position to set a different screen resolution, if required, using the screen GUI.

### Post by chewearn on 2008-03-09
I have essentially the same problem, and there is no need to do anything complicated to solve; simply take a ruler, measure the horizontal and vertical dimensions of your monitor (in millimeters).  Then, add one line in xorg.conf under Section "Monitor":

```
DisplaySize x y
```Example, this the relevant section in my xorg.conf:

```

Section "Monitor"
  Identifier "Monitor0"
   VendorName "Unknown"
   ModelName "CRT-0"
   HorzSync 30.0-70.0
   VertRefresh 50.0 - 160.0
   DisplaySize 430 270
EndSection
```where my monitor is 430mm by 270mm.

Restart Xorg by CTRL+ALT+Backspace and the fonts should now be ok.

Hi,

I managed to solve this tiny font problem on my system by cheating in the xorg.conf file.

Instead of entering the actual DisplaySize value (which should be 850 495) under the Monitor section, I entered the smallest resolution values of my system (320 200), figuring the aspect ratio would at least be correct.

It worked. After a complete system restart the system font size is now legible.
