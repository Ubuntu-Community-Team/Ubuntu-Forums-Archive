### Post by Mustard on 2006-01-14
It would be good if the psychocats link had instructions on setting up checkinstall instead of using the standard ./configure, make, make install instructions. :)

If you want to try it out you can install checkinstall via synaptic package manager and then when you are installing from source, you could substitute the checkinstall command for the 'make install' part of the process.  This creates an entry in synaptic for the install and allows you to uninstall via synaptic.
