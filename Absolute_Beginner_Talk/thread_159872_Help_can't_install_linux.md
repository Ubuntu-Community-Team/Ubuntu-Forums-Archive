### Post by taurus on 2006-04-13
Before you burned the ISO image, did you check the integrity of the file with either md5sum or sha1sum?  Also, don't forget to burn it at a slower speed like 4x even though your media can handle a high speed burning...  Yes, it would take a little longer to burn (~20 minutes) but it can save you a lot of headaches when you try to install from it.
