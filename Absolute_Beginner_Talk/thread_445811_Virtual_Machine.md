### Post by tompickles on 2007-05-16
Wine is available through the Add/Remove Applications screen. just search for it ensureing that you show 'All Open Source applications'.

VMWare PLayer is also available this way, but this only allows you to run virtual machines, and not create them - not the most usefull i know. VMWare Server - jut download it from their website (seach Google) - is the 'free' version, as in you don't have to pay. And the server support means you can access machines over servers, but it still works fine on a standalone machine.

hope this was useful?!
