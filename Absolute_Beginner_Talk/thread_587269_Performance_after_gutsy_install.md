### Post by mivo on 2007-10-22
The first time using Gutsy/7.10 can also seem slow because Tracker is indexing your files. This can take some time, too, which may cause the system to run noticeably slower. If after, say, two hours the problem still persists, kill Tracker in the process list and see if there is an improvement. Turning off desktop effects is also a good test, if you still have them on. Tracker is often what causes an intitially poor impression.
