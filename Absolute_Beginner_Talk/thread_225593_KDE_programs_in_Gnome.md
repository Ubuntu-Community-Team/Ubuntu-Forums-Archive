### Post by T700 on 2006-07-30
You can easily use most KDE programs in Gnome and vice versa.  If you install a program from another desktop using a package handler (aptitude, apt-get, Adept, Synaptic), it will automatically take care of installing just enough additional files to let it run.

### Post by Clay85 on 2006-07-30
You can use KDE programs in gnome. The only problem is the KDE programs use KDE libraries, so you may notice a little lag when you start up a program for the first time in a session (like amaroK in gnome). But once it starts up it's fine. And you can open and close it without lag as long as you don't log off or reboot.

 I haven't heard of anyone having problems running KDE programs in gnome, though. I installed KDE and didn't like it, but it came with some nice programs, so I use them in gnome all the time.
