### Post by hyper_ch on 2007-04-16
Well, the install from the LiveCD is sometimes a bit of a problem. You can try again (booting up again from the LiveCD) and see whether it's better now OR you can download the alternate install cd (that one works better for installation).

Before you install, make sure that most of your hardware run. Especailly networking is good to have :)

And then make sure - if you want to dual boot - that you have defragmented the windows partition at least 2x and also be sure to have backups of all your important docs.


[However for Feisty Fawn the alternate CD has been buggy for me (the partitioner didn't work correctly on Xubuntu Feisty Alternate Herd 5).]
