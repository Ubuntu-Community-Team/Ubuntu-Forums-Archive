### Post by candtalan on 2007-11-04
> **Tlingit said:**
> Can i just put the cd in and have it dual boot vista and Ubuntu? when i get to the partition part it tells me this can not be undone . In the end will i be able to just put the recovery disk in it?

If you use the live CD and boot the machine with it, then you will be running a live full version of ubuntu without changing anything in your machine. When you stop the machine and remove the CD then the ubuntu session is lost and you can continue with your machine as it was before.

If you go into the install procedure and continue with it then you certainly will change your machine, and if things go wrong you will loose data which cannot be recovered. It is a serious thing to change partitions and if you are not sure that you have a good backup of your information, or if you are uncertain about partitions, then it is good that you ask such questions. 

Yes, using the live CD install process, you can install so that ubuntu and vista will dual boot. It will be useful to look at some of the useful information before you go into the process fully.

Recovery disk: if you need to go back  then things are not just to do with a recovery disk. For a beginner, it may be a bit difficult at first. You may be able to remove ubuntu and re size the vista partition, and also refresh the vista boot loading process.
