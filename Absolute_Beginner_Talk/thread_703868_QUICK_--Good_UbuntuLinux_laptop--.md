### Post by GSF1200S on 2008-02-21
Im not sure about Java on a LiveCD, but im sure that you can get it to work once you install Ubuntu on the hard drive after purchase. The computers hardware wont have anything to do with the working of Java. It will be super easy if you install 32bit buntu, as just installing the ubuntu restricted extras will take care of it. On 64bit youll either need to go with blackdown java on a 64 bit browser, or install a 32 bit browser and everything will go smootly.

Take that live CD and check compatability with the LAN card and the wireless. If all that works, the bios is respectable, and youre using an Nvidia card, you should be fine.

I know alot of ATI users who will disagree, but just as an example, running CNC the first decade under cedega with an ATI card creates strange effects such as the ground not being rendered etc. On nvidia cards it works perfect. Just keep that in mind. Both will work fine if your not looking for gaming under wine/cedega or native games; compiz will work on an ati card with the proper packages installed. 

Make sure to check the bios version on the net and find out how acpi linux friendly it is. I was lucky in the fact that fans work correctly and that I can use lm-sensors to get the temps from my CPU, GPU and hard drive. Also, with an ATI card or a crappy bios, you will have problems getting suspend/hibernate to work.

For reference, my computer has the following hardware and it works 100% perfect with Linux including suspend/hibernate, video, wireless, and even cpu and hard drive sensors. Im not sure if my computer even has fan sensors or voltage sensors, as lm-sensors hangs in the beginning for some reason, so I have to skip to the CPU portion of the test.

2gb ram
Nvidia GeForce 7600 Go (pci express, 256MB)
Atheros Communications chipset (Gigabyte wireless card)
Phoenix Bios (some are crappier than others; I got lucky)
Hitachi 200GB 7200 RPM hard drive- sensors for temp on the drive work.
The computer model is generic- HEL 80, which alot of budget gaming laptop makers build on.

Good Luck ;)

**EDIT** Do some research on the newer 8000 series of Nvidia cards if your prospective computer has one. Some people have had issues with them not booting up or the drivers not working, but only on certain models.
