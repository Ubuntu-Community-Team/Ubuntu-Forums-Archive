### Post by PriceChild on 2008-04-13
There are no viruses out in the wild for Linux, always a good thing.

Ubuntu also ships with no open ports by default. This means that a random person can't come along the network and hack you unless you install server applications which opens & listens on ports. (and even if you do there has to be a vulnerability in the application that hasn't been fixed)

The standard users in Ubuntu also have write access to only their home partition. This means it is hard for them to break the system, or install malicious software for another user to run unwittingly. The first user gets access to "sudo" which lets them do anything though. With physical access to the machine, a user can gain full access though.

Along with that, Ubuntu has about 16,000 different programs available in "software repositories". These are all open to public scrutiny which can help keep them secure. Its reccomended you install software only from these repositories to ensure security and stability.

The two main things which will cause you problems, is if you are phished and give your secret details away to an imposter after spam email etc. Or if you install/run software from outside the repositories which you cannot trust.

In short, you're pretty safe unless you, as an admin do something exceptionally silly.

### Post by LeoSolaris on 2008-04-13
Another good point to add to the pile of security reassurances, Ubuntu won the PWN to OWN contest from CanSecWest. Mac fell the 2nd day and Vista the 3rd, though Vista's fall was from Flash, not Vista itself. Ubuntu never fell.

When you add that to the already mentioned lack of viruses, malware, and the difficulty hackers face in planting rootkits in the system... Ubuntu and Linux in general, are pretty secure.

Nothing is ever perfect, and the only way to have a perfectly secure system is to chop up all the wires running out of it with an ax, turn the computer off, then cover it in cement and drop it into the ocean. Failing any of those steps and your system's information is still vulnerable in some way.
