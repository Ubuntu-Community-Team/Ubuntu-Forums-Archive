### Post by logos34 on 2007-12-09
> **GeorgiaBoot said:**
> Okay so even if I unplug my internal HD's and put the install disc in while I have my new external hard drive plugged in it will work the same?

yep, the same.  It's text-mode, (think alternate install) though (and slightly over 800MB, so you'll have to burn it to dvd).

you'll get the real-time kernel by doing a full install, vs. just installing the studio desktop and apps through synaptic.

Test out suspend and hibernate first thing and post back if any problems.  I never did get mine to do either on the low-latency kernel, so I'm running basically x64 gutsy -generic kernel with all the studio apps and desktop theme, which is why I refer to my setup as 'ubuntu pseudio'!
