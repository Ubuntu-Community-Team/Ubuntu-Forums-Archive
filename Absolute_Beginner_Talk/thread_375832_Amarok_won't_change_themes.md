### Post by nike984 on 2007-05-11
those theme that you guys installed are a theme for context menu,
so even if you change the theme, only the context appearance will change.

To change the theme for KDE application,
press alt+f2 and print 
```
kcontrol
```
(under the assumption that you have kcontrol installed already)

go to "appearance & theme " tab ===> change the color & style.
It will change the look of kde apps.
