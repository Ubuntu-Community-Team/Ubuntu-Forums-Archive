### Post by oss_monkey on 2006-09-21
Usually you can run the programs through the command-line interface (CLI) by typing the name of the program listed in Synaptic.

For example, I installed the game "[FONT="Courier New"]scorched3d[/FONT]". To run it, I open the terminal and type:

```
scorched3d
```

Hope this helps.
