### Post by wilberfan on 2006-11-19
When first starting Kubuntu, you're given the opportunity to select some 'performance' options--in terms of those little eye-candy details.

Where can I go to adjust those options after that first signin?

---

### Post by aysiu on 2006-11-19
```
sudo aptitude update
sudo aptitude install kpersonalizer
kpersonalizer &
```
