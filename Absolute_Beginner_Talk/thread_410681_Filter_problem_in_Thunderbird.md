### Post by daynah on 2007-04-16
About the filters: You will have to have the filters on "local folders" rather than a specific email account. The default is for it to be on a specific email account, but if you normally operate on a "local folders" set up, your filter needs to be applied to local folders.

This is a pain because you can't just drag and paste the filters or anything like that you have to actually put them in by hand again. (I had this same problem).

If, per chance, it is on local folders and you have your emails set up per email, maybe try switching it the other way? I haven't had this problem myself, but logically it should seem to go the other way.
