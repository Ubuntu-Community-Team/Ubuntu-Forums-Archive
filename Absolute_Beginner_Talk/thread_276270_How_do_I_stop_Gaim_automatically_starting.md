### Post by taurus on 2006-10-12
Did you happen to load it in System -> Preferences -> Sessions -> Start up Programs?

### Post by taurus on 2006-10-12
In the Sessions Options (first tab), click the "Automatically save changes to session" and log out.  Don't forget to remove Gaim from your "Start up Programs" before you log out.  Log back in and see if Gain is still starting up.
