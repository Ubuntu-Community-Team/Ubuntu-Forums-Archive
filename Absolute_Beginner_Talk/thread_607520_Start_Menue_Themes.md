### Post by jordanmthomas on 2007-11-09
Ah, that is a per gtk-theme thing.  Unfortunately, unless you are familiar with creating themes yourself it's not too easy to change.

The menu is changed by whatever theme you choose in the appearance manager.

My suggestion:  find another dark theme you can use that makes the menu dark as well.

However, if you are feeling adventurous, give this a try:
1.  press Alt + F2
2.  type ~/.themes
3.  go into the folder of the theme you're using.
4.  open up the .gtkrc file within 
5.  edit it...you'll be looking for things with names like menu and options for their colors.

I don't want to sound mean or anything, but you are not likely to have success editing the file yourself.  Here's a link that might help you some if you do try to edit your theme.
[http://orford.org/gtk/](http://orford.org/gtk/)

Good luck.

If you still mean the Gnome panel (not just the menu), in the same dialog where you can select the color and transparency you can set the panel to use a background image. Just use any picture you want, or alternatively create one that is exactly the size your panel is to get the best look.

If you use a .png or .svg image you can even have transparency in your panel.
