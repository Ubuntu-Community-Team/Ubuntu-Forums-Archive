### Post by sdennie on 2008-04-18
Do you have multiple keyboard layouts enabled?  In Ubuntu 8.04 (and maybe earlier versions), under System->Preferences->Keyboard there is an option to allow each window to use a different keyboard layout.  It's possible you accidentally hit the keyboard layout switching combo (by default both Alt keys) while in Firefox and so it's using a different keyboard layout than the rest of the system.  If you aren't using multiple keyboard layouts, I'm not sure what the problem could be.
