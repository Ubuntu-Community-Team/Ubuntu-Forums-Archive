I don't think there are any guarantees.  For what it is worth, I have installed feisty 7.04 on two separate systems - AMD 64 bit desktop - 64 bit version, and i386 version on a Dell Inspiron 6000  (Intel) laptop.  With the desktop I dual boot with Windows Xp as I thought I may have problems learning a new system, and needed to have working system in case things didn't go well or it broke.  I have booted into Windows about twice in the last month, and I will be going to Linux exclusively.  My laptop is Ubuntu only.  I have not had any major problems - BUT - it depends on your hardware setup.  The advantage of the Live CD is generally it will let you know if there  are going to be hardware issues - that is - if it works with the live CD it should work with the full install. 

My experience has been that the problem areas (and this is mainly from reading these forums) are - installing media and proprietary codecs,  printing (occasionally), wireless networking (with some wireless cards which are not supported well), and certain video cards.  All of these problems have solutions (this forum is a great source of help and advice) but time, patience and a willingness to learn is essential.

My advice is to try running it from a Live CD before you decide whether to install.  It will run far slower than after a full installation, but at least you can check it will work with your system, and if this OS is for you.

Good luck and if you choose to stay - welcome
