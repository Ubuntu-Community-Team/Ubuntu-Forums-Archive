### Post by jordanmthomas on 2007-01-13
Chances are you were supposed to "Enter Description" at some point and accidentally renamed it.
Perhaps that is what the drive's volume name has been changed to.  Just click on it and hit F2.  You should be able to rename it (though I am not at a Linux box right now so I can't be 100% sure)

Ubuntu itself is already pretty safe from viruses.  Someone probably could make one but it would be patched and in the security updates before you knew it.  If you want antivirus for Linux there is AVG for linux and there is also ClamAV.  Generally, these are just used to help stop the spread of virus as 99.9999% of viruses are written for Windows and cannot mess with Linux.
