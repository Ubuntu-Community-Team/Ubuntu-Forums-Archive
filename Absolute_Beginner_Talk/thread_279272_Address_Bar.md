### Post by stalker145 on 2006-10-17
> **MrKlean said:**
> Hi Guys,  I was wondering if there is a way to put an address bar in the panel at top ??Thanks

You could always use the Deskbar. To get that, **right-click **on the panel that you wish to install it on, choose **"Add To Panel"** and select the **Deskbar Accessory **from the box. It has all manner of different things that you can do from the applet to include send e-mail, launch programs, open a web page, yadda-yadda just from typing something into it.

*Disclaimer*: I haven't used this efore and I can't use it now to test as I'm at work and am not allowed to put my computer on the network. It did open up firefox when I typed a web address, though.
