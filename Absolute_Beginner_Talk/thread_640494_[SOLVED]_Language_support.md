### Post by Don Gatto on 2007-12-18
Hello,

I think you should download the Tamil language packages: language-pack-ta, and language-pack-ta-base. After this you shall have the Tamil language on your list.

(In case you need more information: If you have Internet connection at home, you can do that easily via Synaptic. If not, you have to get the packs from an other machine - visit packages.ubuntu.com, then use the terminal at home: jump to the directory you have the files, then use this command: dkpg -i language-pack-ta.deb language-pack-ta-base.deb. Feel free to ask if you still experience problems.)
