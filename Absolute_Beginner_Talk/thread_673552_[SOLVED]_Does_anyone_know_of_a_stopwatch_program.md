### Post by pijits on 2008-01-20
theres one that you might be looking for but you'll need to download a program called gDesklets. 

I think you can download it from the Add/Remove Progams menu if your running Ubuntu or type sudo apt-get install gdesklets. 

The app in gDesklets is called Countdown2 which is under the misc/Utilities menu which you double click on then mouse over a location on your desktop and then click again. Right click then to configure or move. It'll prompt you on what to do.

### Post by mjpatey on 2008-01-20
I have one on my Gutsy box at home, and it works great.  I think it's the one here:

[http://packages.ubuntu.com/feisty/utils/stopwatch](http://packages.ubuntu.com/feisty/utils/stopwatch)

Ugly default icon for it, but it does the job well!  You have to have the Universe repository enabled, then install it simply with:

```
sudo apt-get install stopwatch
```
