### Post by elcasey on 2006-12-26
GIMP is comparable (arguably) to Photoshop and free Nikon software bundled with a camera is not going to be superior. 

I've had bad luck with WINE myself. I can only run a very old version and even then it doesn't want to install anything (tells me I'm out of disk space when I'm not). So I've written off WINE until it becomes *much* more stable. Besides, I only needed WINE to run a piece of Windows software I was using for a math class, which is now over.

You won't be able to run Mac software under Linux unless you're running Linux on an actual Mac, in which case you can install MacOnLinux. I don't have any experience with it since my iBook G4 doesn't seem to "play well" with Kubuntu 6.10 (and I don't want to reformat my iBook, anyway).

It's pretty much guaranteed that you're not the only one running this camera under Linux and if you've got the GIMP working with your camera, what more do you need, honestly?
