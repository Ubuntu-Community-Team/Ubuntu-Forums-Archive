### Post by nickburns on 2007-02-12
You need to see if you burned the image to the CD or if you burned the ISO file to the cd.  You cannot just burn the ISO to a cd (must burn image).  If you have Roxio or Nero or some other downloaded burning software, you can right click on the iso and do burn to cd.

Once it is burned, reboot, make sure your cd is set to bootable (in you bios) and you'll be good.
