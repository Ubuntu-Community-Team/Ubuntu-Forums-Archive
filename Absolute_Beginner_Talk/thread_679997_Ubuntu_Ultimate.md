### Post by Therion on 2008-01-27
Well I've been running UUE (version 1.6) for some time now.  I'm not entirely sure what exactly you're asking but feel free to ask more questions.

In an attempt to explain better, UUE (1.6) is, basically, a Gutsy-based Ubuntu distro that comes with a lot of preinstalled software, a customized theme and so forth.  I'm using it because I was having no end of trouble getting some display driver issues sorted out using "plain vanilla" Ubuntu installs.  UUE installed flawlessly for me and everything (and I do mean 'everything') ran smooth as silk for me, right out of the box.

Again, feel free to ask more specific questions if you want.  I'll do my best to explain.

Edit:  Just thought I'd point out there is a LiveDVD version of UUE v1.6 if you want to give it a test drive.  UUE is DVD only (no CD) because of all the additional software that comes with it.
