### Post by ddrichardson on 2007-11-15
There is if you are prepared for a lot of work. You would need to rebuild the Live CD in such a way as to load the OS into a ramdisk, which would require some careful package selection due to Ubuntu's larger footprint. I suspect however what you would really like would be easier to implement as an Ubuntu install on a [USB disk]("http://ubuntuforums.org/showpost.php?p=1062799&postcount=100").
