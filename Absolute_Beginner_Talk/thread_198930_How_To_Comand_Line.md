### Post by cidica on 2006-06-20
[QUOTE=msixp_k7]When I trie to right a comand it do not work. For instace I want to download flashplugin so what do I type?????????????????????[/QUOTE]

You should google the topic bash commands. What I did when I first started to learn command line is I would type:

```
a [TAB][TAB] 
```

and it will ask to display all 100+ commands and hit y for yes. It will show you tons of commands that start with the letter a. Then I did b etc etc etc. I know that seems a little um odd? But it showed ones I could use. Then I would google them or do:

```
command --help
            command --info
```

where command is the command you see or what you want to know about. Also you can do
 
```
man command
```
that will bring up manual pages for just about any command listed.

Hope this helps 8) 
PS to get flash working in linux get it from macromedia's website and untar/gzip it and type

```
sh ./flashplayer-installer
```
inside the directory where you untared it. Assuming you dont have 64bit liinux running.
