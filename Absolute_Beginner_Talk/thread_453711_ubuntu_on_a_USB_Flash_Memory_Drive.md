### Post by threadmetal on 2007-05-24
Short answer? Not easily for a noob. However, since we all start as noobs:

I believe you can use the regular installation procedure (just select the USB drive as the target) if you're installing to a USB drive of at least 2GB. (size apparently does matter) Make sure to partition the USB stick manually *before* beginning the installation, as ubuntu currently remounts removable drives when partitions change, causing trouble with the installation scripts.

For the capacity-deprived, there's ways to make it work with just 1gb, but none of them (that I have found) are even as simple as above. Search the forums (fora?) for "Ubuntu USB Flash Drive" and you'll see what I mean.

If anyone knows a really nice and easy way to do this, please correct me! :-)
