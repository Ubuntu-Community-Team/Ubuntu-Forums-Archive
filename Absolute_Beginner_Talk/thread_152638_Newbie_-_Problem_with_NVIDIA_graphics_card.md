### Post by Mustard on 2006-03-30
I'm curious if you were to try adding this option to the Section "Device" part of your xorg.conf whether things might improve, Option "RenderAccel" "false" 

```
Section "Device"
	Identifier	"NVIDIA Corporation NV35 [GeForce FX 5900XT]"
	Driver		"nvidia"
	BusID		"PCI:1:0:0"
        Option          "RenderAccel" "false"
```

There are a number of different options you can fiddle with.  You could even try setting RenderAccel to "true".  Its possible that you may experience lockups though if you have it turned on.  I've been doing some googling trying to think of what might be a good option.

There are a couple of options you can try that are listed at the bottom of the first post in this thread.

[http://ubuntuforums.org/showthread.php?t=57368](http://ubuntuforums.org/showthread.php?t=57368)
