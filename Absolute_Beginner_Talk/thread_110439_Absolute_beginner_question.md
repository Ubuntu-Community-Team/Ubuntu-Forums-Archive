### Post by Gimbo on 2005-12-30
You're obviously trying the methodical approach of building up a base of knowledge by reading books/manuals/faqs etc. For this, I'm not sure the included Starter Guide is that great (System>Help>Ubuntu 5.10 Starter Guide). The most well-known guide on the internet is (as far as I know) [http://www.ubuntuguide.org/]("http://www.ubuntuguide.org/"). This was written for 5.04, but I've not encountered anything there that doesn't apply to 5.10.

This may not be the way you like to do things, but there is another way of learning Ubuntu, and it's the one I'm using - muddle through by searching for answers to specific problems you come across. For this, these forums are absolutely brilliant, but Google will help as well.

There are few very inportant commands and maybe a list of them may help.

 cd =change directory which allows terminal navigation or the file system 
 you can also type ./ then just use the name of a file within your current directory.
 eg   cd /home/Your_username will take you to your home directory.
 ls   =list of files in the directory you are currently in.
 chmod = changes the permitions of a file
 chown = changes the owner of a file
 . =.exacutable_file_name =runs a exacutable file
