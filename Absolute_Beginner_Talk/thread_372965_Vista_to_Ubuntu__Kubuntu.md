### Post by Arisna on 2007-02-28
A quick note about software availability--I'm quite certain there is actually more Free/Open Source Software available for GNU/Linux than for Windows.  Many Free software projects do not support Windows at all, such as virtually all of KDE for now, to give a big example, as well as a variety of small projects that don't want the hassle of porting to Windows.  This should seem logical--a Free/Open Source system will have more Free/Open Source Software available for it than a closed, commercial system will have.
