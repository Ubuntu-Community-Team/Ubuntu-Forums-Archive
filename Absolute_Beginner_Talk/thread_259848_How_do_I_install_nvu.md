### Post by Najand on 2006-09-17
Add universe/multiverse by:
System -> Administration -> Software Properties
Then Check all unchecked options.

Then Use synaptic at:
System -> Administration -> Synaptic Package Manager

Click Reload...
Find Nvu and right click on it.
Select it for installation,

Then Click Apply on top of the page.
