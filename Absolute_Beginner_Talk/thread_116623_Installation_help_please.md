### Post by mvaniersel on 2006-01-13
What you download is an .iso file, which is a CD image that you have to burn to a CD.

When you restart and boot from that CD, ubuntu will install.

During installation there are several options for partitioning. You can do it manually or you can let Ubuntu do it for you.
