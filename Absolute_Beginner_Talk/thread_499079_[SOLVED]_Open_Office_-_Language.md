### Post by luisromangz on 2007-07-12
Its very easy, follow these steps:
[LIST=1]
[*]Go to the Tools/Options dialog.
[*]In that dialog, expand the Langage Settings node, and select the Languages node.
[*]There, in the "Default language for documents" section, select the lenguage you want to use, and check the "For the current document only" checkbutton if you don't want to set that language as default for all new documents.
[/LIST]
