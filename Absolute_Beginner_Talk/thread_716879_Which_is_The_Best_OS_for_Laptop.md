### Post by Barlam on 2008-03-06
I would not recomend formatting your Laptop because it can void warranties. Also if you want it fixed, most major repair places will require windows on it. Instead I suggest creating a dual boot system with two operating systems on it. Just create a new partition on your HD and install Ubuntu or something similar onto the new partition.

### Post by dwanders on 2008-03-06
Most of the current distro's have the Live CD. Download a few and try them out. Make sure they run good on your computer and all the stuff works (like the NIC, Video etc.) Once you have decided on the one you like better, there is probably an Install ICON you just click to and follow the steps.

Personally I am no fan of DUal booting, if its an older Laptop without support and you have some skills with PC's, just jump in :lolflag: If you still want support from the Manufacture (I thought Packard Bells were gone in the 90's :confused: ) then I would not advise removing Windows. 

Linux Mint is good but you need good hardware and resources
Ubuntu - same but with more restricted rives etc (Gnome interface)
KUbuntu - same but with a different interface (KDE)
XUbuntu - uses XFC - very quick and runs on older stuff (needs less hardware)
Pro+ech - Like XUbunut - but more geared to network troubleshooting, runs very nicely on my PIII with Linksys WiFi card - you would need to add Office though. 

Try the Live CD's first, if you have 1/2 gig ram any of them should run OK, but expect that there will some lagging because your running off CD. 
There is also a nice Google OS that I downloaded some time ago and cannot seem to find a link for any more - anyway there is a ton to pick from .
