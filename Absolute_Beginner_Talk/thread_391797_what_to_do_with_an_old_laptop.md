### Post by joethenoob on 2007-03-23
> **windfer said:**
> i also have an old dell with 256 ram and 10 gig hardrive thats running windows but i figure that since this is a linux forum i wanted to keep it on topic but interesting. how?

Actually, it was very easy. It's pretty well documented on mame.net. After loading a minimum install of XP, I downloaded the Windows command line version of MAME and extracted it at the root of the C: drive (C:\mame). The extractor creates the directory structure for you to store roms, samples (sound files), and artwork (overlays like for asteroids 2 or space invaders). Once you have dropped your roms into the roms folder, all you have to do is open a command prompt and do the following:

C:\Documents and Settings\joe>**cd\**
C:\>**cd mame**
C:\mame>**mame <rom name>**

And it runs the game. I use a MS Sidewinder USB control pad with it. XP recognized it without any other drivers. If you want to run a game with a game pad you would type:

C:\mame>**mame <rom name> -joystick**

All of this is documented pretty well in the FAQ at mame.net. You can also download one of a few GUI front-ends that will prevent you from having to use the cmd prompt every time you wanna play.

The tricky part is finding the roms. You can google "mame rom" and come up with a few sources.

Let me know if you need help.
