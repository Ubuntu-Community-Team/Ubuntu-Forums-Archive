### Post by Megaqwerty on 2007-07-07
It was a little hard for me reading your question, but I think you were asking how to use WINE to run Windows programs. Basically, you need to open a terminal **Applications>Accessories>Terminal** go to wherever you have the file by doing: ```
cd /path/to/folder
``` and then type: ```
wine program-name.exe
```
Wait a few seconds, and the program should launch.
