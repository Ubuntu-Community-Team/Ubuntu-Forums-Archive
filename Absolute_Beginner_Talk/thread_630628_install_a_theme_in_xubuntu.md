### Post by urukrama on 2007-12-03
Installing a theme for Xfce is identical to installing a theme in Gnome. The directories you mentioned are correct.

Use the following command to bring up the settings manager of xfce:

> xfce-setting-show

Then click on User Interface (bottom right) and change to whatever icon and theme you like.
