Exactly what did you download?
Most of us download the Live CD - burn the file to a CD - then Boot the computer from the Live CD......the rest comes automatically (almost).

The Live CD puts ubuntu onto your computer so that you can try it before you install. When happy  - you just click on the 'Install' icon.

If you have problems:
1. Make sure the Live CD has no errors
2. Make sure you have the correct version for your CPU (32/64)
3. Use the Alternate (text) CD if you have problems with the (graphics) Live CD
