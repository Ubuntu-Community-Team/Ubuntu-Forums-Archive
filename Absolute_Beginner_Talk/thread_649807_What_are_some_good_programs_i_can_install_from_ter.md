### Post by SOULRiDER on 2007-12-25
Flock doesn't seem to be on the repositories, that means you wont be able to isntall using Synaptic/aptitude/apt-get. You will have to find a Deb package for Ubuntu and install flock using that package. If you can't find one, youll have to compile from source and create the package yourself.

### Post by SOULRiDER on 2007-12-25
Theres a deb package in [http://www.getdeb.net](http://www.getdeb.net) for you to download and install.

[http://www.getdeb.net/app.php?name=Flock](http://www.getdeb.net/app.php?name=Flock)
