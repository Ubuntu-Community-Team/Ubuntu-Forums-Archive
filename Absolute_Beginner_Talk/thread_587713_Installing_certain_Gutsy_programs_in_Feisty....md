### Post by Inxsible on 2007-10-23
The software that you specify are not necessarily "only Gutsy" software.

Open Office is surely not. You can install it by getting the rpm from the openoffice.org website and then using alien to convert it to .deb and install it that way.

Compiz 0.6.0, I am not sure if this is the Gutsy version. but there are a million posts about how to install Compiz Fusion on Feisty. Use Amaranth's repos or Trevino's repos to do it.
