### Post by Ob1 on 2006-04-15
Does the Update manager upgrade the kernel itslef, or do I have to instal the newest version of it?

---

### Post by Qrk on 2006-04-15
The update manager will update the kernel just like any other package. This won't be the latest version of the kernel, however. If you want the absolute newest version of the kernel, you can compile it yourself.
