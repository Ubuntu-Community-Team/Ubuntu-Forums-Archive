### Post by Spike-X on 2007-02-14
> **UWhat said:**
> 
I wanted to download Ubuntu, I managed to download infra recorder as suggested, I clicked on the "Actions" menu, I selected "Burn Image" and then ... well that is where I come unstuck. I get a box coming up but the "Image File" which I am supposed to open is empty. 



It sounds like you haven't yet downloaded the actual image file that needs to be burned to the CD. 

You want to download the 'Desktop' version (A desktop computer is just your standard home computer what sits on a desk). From the Ubuntu main page, click on 'Download' under where it says 'Desktop'.

From here, You'll have to choose whether you want to get the latest version (6.10, also referred to as 'Edgy'), or a slightly older version (6.06, also referred to as 'Dapper'). I don't think I'm qualified to advise you on this - I'm only new to this myself, and I don't know much about the differences between the two. 

Click on whichever one you've decided to download. You will then be asked to choose a download site. Pick a site that's closest to where you are. From there, you'll be asked to Select an image. You want the PC (Intelx86) desktop CD (I'm assuming you're not running an AMD 64-bit processor - if you are, choose that one).

Click on the image you want, then save it. REMEMBER WHERE YOU ARE SAVING THE IMAGE FILE TO!! You'll need to know where to find it when you go to burn it to CD.

I've tried make this is simple as I possibly could. Remember, though, it's a computer, not a toaster. Some things can only be simplified so far. If you're not sure about anything, though, don't be afraid to ask - this forum is filled with helpful, knowledgable people. I knew nothing about Linux three weeks ago, now I'm compiling programs from source code!
