### Post by th3james on 2006-06-30
Sounds like your still configuring the basics of your system? Easiest way to do this is using either easy ubuntu, automatix, or BUMPS
My recommendation is [EasyUbuntu]("http://easyubuntu.freecontrib.org/")

These are all scripts where you can just click a couple of boxes, and have everything set up for you!

Easiest way to install most software after this is to use Synaptic Package Manager, found in System -> Administration -> synaptic package manager
