### Post by Haluci on 2008-02-09
Try looking under Applications --> System Tools.

Also, to add yourself to the vboxusers group, go to System --> Administration --> Users and Groups, click Manage Groups, scroll down to vboxusers, click on it then click properties, then click the checkmark next to your user name.  Click OK, close the User window, then run Virtualbox.
