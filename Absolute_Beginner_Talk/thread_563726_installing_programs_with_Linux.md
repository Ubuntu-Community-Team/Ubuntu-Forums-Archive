### Post by Nehvrook on 2007-10-03
If you want to run a fsck you should use the live CD and run it from there. Just open a terminal in the live CD and type

```
fsck /dev/xxx
```

xxx being the location of the hard drive.

If you don't know the location of your hard drive type 

```
sudo fdisk -l
```

(The l is a lower case L if that doesn't show up well to you, fonts can be funny some times)

and that will tell you the location, it will be something like /dev/sda1


Also have you tried making a new installer CD? Which version of Ubuntu are you installing?
