### Post by apunc1 on 2007-06-01
no, upgrading in the way you describe should not overwrite your data.

do you have you data on a separate partition than your / Ubuntu install, such as a /home partition or a /shared partition? this makes it possible to do a clean install if you ever needed or wanted to. for example, i use dapper and do not want to use edgy. if i want to upgrade to feisty i need to do a clean install of feisty. then, i would install without touching my /home partition, so my data would be untouched. i would overwrite my / dapper install with edgy.
