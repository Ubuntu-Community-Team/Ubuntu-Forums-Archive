### Post by herbster on 2008-03-21
As for games, it depends on which ones of course. You can use the Wine solution to run games, many work well but of course will likely require some fiddling/tweaking to set them up correctly. You can see appdb.winehq.org and search for games you'd like to play and see what the results are that others have had, and there are HowTos there as well.

Certain games have native installers (like Quake 3, Enemy Territory, Quake Wars, etc.) and they work perfectly native in linux.
