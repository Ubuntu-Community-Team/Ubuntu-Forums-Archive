### Post by leveret on 2008-04-18
Excuse followup to 6 month old thread, just adding the answer for benefit of others searching here.

The sgzcache directory with the two .sgz files is created by Cisco's Security Device Manager (hence the "sdm" subdirectory inside it).  At some point you've probably configured a Cisco router from your machine.
