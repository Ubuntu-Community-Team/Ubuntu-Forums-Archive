### Post by johndollison on 2008-04-23
OK, I figured it out... the Update Manager can't get 7.04 until all the other patches and updates for 6.10 have been installed.  So...

1. Install all the 6.10 patches and updates (took several hours on my laptop);
2. Run Update Manager again, and upgrade to 7.04 (Farting Fawn) (also took several hours);
3. Run Update Manager again, and upgrade to 7.10 (Gaseous Gibbon) (took even more hours).

Tomorrow 8.04 comes out, so I guess I'll load updates for a third day in a row!

Hope this helps someone... if it does, post a reply here and let me know.
