### Post by johnc4510 on 2007-03-29
In addition to scrooge_74's suggestion, almost every application has a man page. These are command options that work with each individual app. To access them, in a terminal, Applications>Accessories>Terminal, type man and then the name of the app.

Example:  johnc@Feisty:~$ man firefox

### Post by billmeek on 2007-03-29
If you want to look at a list of all the (many) possibilities, open a terminal and tap the TAB key twice and then answer "Y".  This is using the autocompletion to list all entries.  You can type just the first few characters of a command (or path) and a single tab of the TAB key will complete it if there's only one entry or provide a list if there are multiples.
