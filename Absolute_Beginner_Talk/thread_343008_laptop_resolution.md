### Post by ramjet_1953 on 2007-01-21
1. In a terminal type in 915resolution -l
This will give you a list of available options.

2. Choose a listing that you will never need ie 640X480 8 bits/pixel and note its Mode number down. (for this exercise let's say it's Mode 30 and you want 1280X800 24 Bits/Pixel)

3. In a terminal type in 915resolution 30 1280 800 24

4. In a terminal, edit the /etc/default/915resolution file to the following

MODE=30

XRESO=1280
YRESO=800

BIT=24

5> Save and re-start X

You should now be able to change to widescreen, in fact it should re-start automatically into the correct mode.
