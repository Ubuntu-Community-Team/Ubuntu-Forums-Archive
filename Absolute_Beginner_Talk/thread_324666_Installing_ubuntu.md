### Post by Crakie on 2006-12-24
Nope, just moving it across won't work. Normally, you need burn it on and then install it from a CD. It is possible, I think, to install it from an external drive. You would either need to boot from a Linux CD, mount the installationdisk and run it from there OR be able to boot from the external disk (which some BIOSs support). Search on google or this forums on how to do it :)

### Post by Mushie on 2006-12-24
As crakie said, moving it across will just copy the file, not act as a cd. If you have a larger enough jump drive, I know a lot of BIOS setting to allow booting from an external device. Just as you would change you boot settings normally, just check to see if usb boot is an option. We use a pocketTech at work alot to boot an image onto a machine.
