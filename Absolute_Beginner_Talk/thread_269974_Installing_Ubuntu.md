### Post by bulldog on 2006-10-02
You boot to the desktop where the install button is.

When you get installing it you have to answer some questions.

Then you get to gparted to patition your hdd,at this moment you have to choose which hdd you want to install.

Choose the correct hdd and your XP won't be harmed.

There's a little catch though,wich is your primary boot hdd?

If this is the hdd with windows it is advisible to change the boot sequence so,that the ubuntu hdd is your first boot device if possible ofcourse.

So will GRUB be installed on your ubuntu install disk and leave your windows install on is own.

It's not a big issue,it's reasonable easy to change it afterwards,but if you can do it before the install is the most easiest way.:D
