### Post by doodah52 on 2007-07-29
When I was a WinXP user, I used Thunderbird.  I am now a confirmed Ubuntu user (wish there was a flash player for 64 bit Firefox) and since the Ubuntu install program uses Evolution, I was wondering how I can make Thunderbird the default email program.  When I want to send a link, Ubuntu launches Evolution, not Thunderbird.   Many thanks in advance for any and all help.

---

### Post by 505 on 2007-07-29
Go to the menu System, Preferences, Preferred Applications and set Thunderbird as your default mail client.
