The difference between gksudo and sudo is that gksudo gives a GTK frontend to sudo. It has also been documented that it is better to use gksudo when running graphical applications. This article gives a pretty good explanation on why.
[http://www.psychocats.net/ubuntu/graphicalsudo](http://www.psychocats.net/ubuntu/graphicalsudo)
