### Post by HereInOz on 2007-10-04
The only thing that I can throw into the discussion is that I believe that the nVidia drivers are better than the ATI drivers.  

Apart from that, you may run in to trouble with things like dial up modem functionality, as well as problems with the keyboard and/or mouse pad.  These last two, though, have become much better since Feisty but you really need to load a live CD on the units to find out if there are any holes in the system at all.
