### Post by gr0kzer0 on 2006-04-24
> I wanted to evaluate Ubuntu as if I'd not used Linux before, but was familiar with Windows (I'm guessing this is where most new Ubuntu users will come from).

I'm not so sure if Ubuntu is the O.S that does it. The idealism that causes Ubuntu to not include proprietary codecs by default means it's  not immediately user-friendly to the newcomer straight from Windows. Sure, the necessary codecs are only a simple download away, but it isn't necessarily "simple" for that migrant from Windows who has possible never used a command line before in her life.

BTW, this is _not_ a criticism of Ubuntu or its values. I love the distro,and I also believe strongly in Free/Open Source Software. I'm just trying to point out something that I think is true, but maybe not so obvious, and often overlooked when people are looking for the latest distro to bury Microsoft.
