### Post by y-lee on 2008-02-07
*If you are sure that is the driver you need* then better instructions for Ubuntu are found at [UVC Linux Driver]("https://help.ubuntu.com/community/UVC").

your next step is making sure you have the kernel header files. I always do but anyway try the below to make sure:

```
sudo apt-get install linux-headers-`uname -r`
```

and follow the rest of the instructions at the link above, the instructions are fairly  clear,  Any questions post back.
