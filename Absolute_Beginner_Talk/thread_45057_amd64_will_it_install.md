Well....I advise the 32 bit version, even on an AMD 64. The only advantage of the 64 bit version (I believe) is that you can use more RAM (if you have more than 4 gigs worth). But it has MANY MANY disadvantages, such as no java or media codecs.

As far as hardware working.....well you don't know till you try. My SATA works great, In fact, all my harware does. But I don't know if your's will. 

I would try the Live CD. It runs slower than the installed version (don't judge it) but it will tell you if you hardware works.
