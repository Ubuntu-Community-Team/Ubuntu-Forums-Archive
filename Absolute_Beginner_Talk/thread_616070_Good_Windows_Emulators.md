### Post by Taboo Mirage on 2007-11-17
Firstly, WINE isn't a Windows emulator.  It's a compatibility layer.  It will not flawlessly support all native Windows applications.  You should refer to the [WINE App. DB](http://appdb.winehq.org/) for information about specific applications and whether or not they're supported.

If you're looking for better results for applications, you should look into virtualization software which will allow you to install a Windows OS and run it from within Ubuntu.  Such packages include VMWare or VirtualBox; the latter of which is available in the repositories.
