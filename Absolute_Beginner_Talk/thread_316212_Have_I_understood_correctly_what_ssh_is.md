### Post by steve.horsley on 2006-12-10
SSH provides secure remote access to other computers. It provides:
* Remote login rather like telnet
* Remote command execution
* File transfer rather like FTP
* Tunneling of X display protocol
* Tunneling of other TCP protocols
* VPN-like tunneling
