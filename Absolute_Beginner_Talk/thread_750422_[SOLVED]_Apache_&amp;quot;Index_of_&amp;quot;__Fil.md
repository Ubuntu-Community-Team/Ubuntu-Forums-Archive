### Post by very_japi on 2008-04-09
If i undesrtood your question right (probably i didnt):

Theres and Option in the apache configuration file that indicates wether "listing" is enabled. When Listing is enabled apache tries to look for an index page (index.php, index.html, default.htm and so on)and if none is found it display a list of folders and files.

If you do not want to have apache list your server's contents, you need to disable that option.

### Post by kebes on 2008-04-09
> **saj0577 said:**
> What about editing the file which formats how the files are listed?

That index listing is generated by Apache's "mod_autoindex" module. The exact options for it depend on your Apache version. Here's the information for mod_autoindex in Apache 2.2:
[http://httpd.apache.org/docs/2.2/mod/mod_autoindex.html](http://httpd.apache.org/docs/2.2/mod/mod_autoindex.html)

There are many options for controlling what the output of that page looks like. Take note of the "IndexStyleSheet" directive, which lets you specify a stylesheet (styles.css) to control the look of the generated index page.

Hope that helps.
