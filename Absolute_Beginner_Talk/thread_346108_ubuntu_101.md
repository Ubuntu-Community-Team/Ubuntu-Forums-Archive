### Post by Aberrix on 2007-01-25
I've found it easiest to install everything from the command line.

if you can find a repo, its just a simple 'sudo apt-get install package'

if you can find a .deb, its just 'sudo dpkg -i package.deb'

if you're *specifically' talking about nvidia stuff here, id highly recommend [automatix]("http://www.getautomatix.com/"), as it installs and configures a lot of basic but important things for you.
