### Post by Ub1476 on 2007-10-20
Take a look at [Gnome-Look.org]("http://www.gnome-look.org") for icon sets, GTK2 
themes, window border themes and so on..

Most of it can be installed by opening Appearance (System>Preferences),  and drag it to the window, or locate it by clicking on "install".

For those who won't install automatically extract it and paste it in (say you want to install a new icon set) .icons in your home folder. Folders with a . in front of them are hidden so press CTRL+h to show them. 

Yes, you can install KDE on Ubuntu, but I wouldn't recommend it, can be confusing.. It's "cleaner" to set up a dual-boot with it if you want to try it out.

Anyway here's a howto: [KDE in Ubuntu]("http://www.psychocats.net/ubuntu/kde")
