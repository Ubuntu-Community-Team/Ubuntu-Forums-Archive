### Post by Bachstelze on 2007-02-24
There is absolutely no problems in having the same login on two different systems.

For your question about the ~ sign, it is an alias to the current user's home directory, which can be whatever but usually is /home/username. Typing ~ or the compte path to the dir is exactly the same.
