### Post by rbprogrammer on 2007-12-06
for me, the easiest way to install fonts was to make a folder in my home directory called ".fonts" and just move the fonts into that folder.  then everything worked

ie:
```

cd ~
mkdir .fonts
mv path/to/fonts/* ~/.fonts/*

```
