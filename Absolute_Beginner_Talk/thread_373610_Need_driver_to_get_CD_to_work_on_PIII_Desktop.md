### Post by luke411 on 2007-03-01
annda probably has something there. You have to make sure you specifically burn the cd as an image, as opposed to a file otherwise it won't boot. Not all cd burning programs will give you that option, though most of the newer ones will.

While Linux does not use anything from your bios except date and time it should have no effect on the live cd. This is the case only after you install it.
