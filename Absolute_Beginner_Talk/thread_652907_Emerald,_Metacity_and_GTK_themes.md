### Post by mattgaunt on 2007-12-29
Heya

Right well this is how i look at it, i could be wrong but ill tell you anyway :-D

Metacity - This is the window manager that comes default with gnome - it is a nice clean little window manager and metacity themes are what u use when just using a normal install of linux (Themes are applied under System->Preferences and themes (I think)

GTK - Not totally sure on this - but what i think your talking about is the parts of gnome that can be customised - there is some program in synaptic i came across once for editing all these settings and colours for menu bar and so but cant remember

Emerald - This is a window manager (Like metacity) but emerald enables you to do more (i.e. transparency ) to the window borders.

Compiz - Compiz is the core that is used in compiz-fusion which gives you all the cube and so on, emerald is run through compiz-fusion.

So hope that helps a little bit - sorry if it doesnt
