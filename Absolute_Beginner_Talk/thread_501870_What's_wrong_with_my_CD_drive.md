### Post by tgalati4 on 2007-07-15
In Dapper, my fstab looks like:

/dev/hdc        /media/cdrom0   udf,iso9660 user,noauto     0       0
/dev/hdd        /media/cdrom1   udf,iso9660 user,noauto     0       0

Where cdrom0 is actually a CD-RW and cdrom1 is a CD-R.  So Dapper doesn't make a distinction in its file system table.

Could it be something as simple as a jumper setting on the drive confusing the operating system?  Linux handles hardware differently than windows.

One nasty possibility: the drive has OEM firmware such that it will only work in Windows.  You might search the web for a firmware update or confirm this possibility.
