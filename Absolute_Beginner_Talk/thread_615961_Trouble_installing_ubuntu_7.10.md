### Post by x64Jimbo on 2007-11-17
You may need to use the Alternate install CD. There are some systems that don't play nicely with Ubuntu's graphical loader. I just installed Ubuntu on a friend's computer last night, and had the same problem. Try the Alternate CD. MAKE SURE that you use a tutorial for setting up dual boot because the default option for partitioning the HDD in the Alternate CD is to erase everything and start fresh.
