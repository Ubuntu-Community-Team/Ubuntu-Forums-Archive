### Post by cdhotfire on 2005-05-16
in ubuntu partitioning screen, select the first fc3 partition and press enter, then select delete, and press enter.  Then select the next partition and do the same.  Then select guided partitioning and select remaining free space, then select to finish and select yes. Thats about it.:)
