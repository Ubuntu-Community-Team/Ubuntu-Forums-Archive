### Post by benuski on 2006-07-23
You could try downloading the windows version and running it through Wine, the windows emulation layer, but you would be much better served running a different, linux compatible program that can perform the same functions as iTunes.  The two most popular are Rhythmbox for Ubuntu and AmaroK for Kubuntu, although either one can be run on either version.  And if you just want a Winamp clone, XMMS is a very good program as well.
