### Post by kostkon on 2007-09-17
I think the official source for the font is [here]("http://www.volvoguy.net/ubuntu/"). Nevertheless, you don't have to download it. Just install it, right away! Do a
```
sudo apt-get install ttf-ubuntu-title
```
and that's it, you will install the font to your system.
