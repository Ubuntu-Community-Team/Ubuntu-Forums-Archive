### Post by indigoshift on 2006-09-09
You should be able to right-click on the "Applications" menu, then select "Edit Menus".  That'll open the Alacarte Menu Editor.

From there, click the "Internet" entry in the leftmost pane, and swiftfox should show up on the right.  Put a check in the checkbox, then click "Close".

Note I keep saying "should"...I haven't installed swiftfox myself, so I don't know for sure if it'll show up in Alacarte.  But it should.  ;)
