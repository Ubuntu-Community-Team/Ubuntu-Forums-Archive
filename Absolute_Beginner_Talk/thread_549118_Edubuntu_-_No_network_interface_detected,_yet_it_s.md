### Post by str8line on 2007-09-13
That message comes from the network discovery.  It is just telling you that the network it found is not part of a "domain".  As long as it gets an IP address from the network it will still function as it does.  Not to worry it comes from the design of Edubuntu to be deployed in schools on a large network of classrooms.
