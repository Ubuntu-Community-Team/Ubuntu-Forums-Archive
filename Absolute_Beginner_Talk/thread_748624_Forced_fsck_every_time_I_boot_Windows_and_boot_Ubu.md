### Post by philinux on 2008-04-07
UUID=0E1CA2C61CA2A863 /media/hda2 ntfs-3g defaults,locale=en_IE.UTF-8 0 **[COLOR="Red"]1[/COLOR]**


Looks like thats the windows partition.

You need to 

gksudo gedit /etc/fstab

Change the **[COLOR="Red"]1[/COLOR]** I've highlighted to a zero then save the file. That will stop ubuntu from trying to fsck your windows partition.
