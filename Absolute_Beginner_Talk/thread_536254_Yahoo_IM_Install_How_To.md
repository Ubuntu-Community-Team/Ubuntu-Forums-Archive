### Post by Serenitude on 2007-08-27
Hi there.

In a nutshell, you can't install the official Yahoo IM without likely breaking your system. It relies on heavily outdated libraries, and hasn't been supported in years.

Were you to actually get it running, you would find most of the features and protocols unusable on the current Yahoo servers.

There are, however, several open source alternatives that perform just as well, if not better. GAIM/Pidgin is one, which should be in Applications/Internet.
