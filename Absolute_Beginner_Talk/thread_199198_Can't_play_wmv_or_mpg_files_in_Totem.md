### Post by flargen on 2006-06-18
The correct codecs are probably installed now, but Totem doesn't know how to use them. You should install the gstreamer0.10-pitfdll package: ```
sudo apt-get install gstreamer0.10-pitfdll
```
I hope this helps you.
