### Post by yabbadabbadont on 2007-03-08
As long as it is *not* a 3d game, one that uses DirectX, then it should run in vmware.  Currently there isn't any DirectX support in vmware.  (or at most very experimental support)  You might want to check out the application database for WINE to see if her games will run using it.

[http://appdb.winehq.org/](http://appdb.winehq.org/)
