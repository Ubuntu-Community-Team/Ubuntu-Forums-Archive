### Post by bsussman on 2006-10-20
Each starts with exactly the same basis - in fact, the same repositories and much of the code you run is identical in each.

Each uses a different window manager and programs that, while performing the same functions, tend to look a little different and work differently as well, in the same fashion as Wordperfect, MSWord  and OpenOffice are different.

ubuntu (Gnome), kubuntu(KDE) and Xubuntu(Xfce) each have their fans.  Provided all the dependencies are met, most if not all programs can run on any of these variants, but if you really like gnome, chances are that you will, for instance, like nautilus better than konqueror.
