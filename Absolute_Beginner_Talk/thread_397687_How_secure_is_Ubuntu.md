### Post by jem7v on 2007-03-31
Linux, because of the very architecture of the system, keeps** your computer** more secure than Windows. Linux was originally written with the intention of running several processes independently of each other, of being able to have multiple users logged in at the same time without having permission to affect each others' processes.  Thus, access and permissions are very strict in Linux.

Windows was originally envisioned as an operating system for stand-alone systems - it didn't begin its life as an overly fancy program to dial up a university's unix Server as Linux did.  (Source for that information - Linux Torvalds' autobiography.) As such, there wasn't any need for control over file/process permissions or the likes, because they didn't consider the possibility of an outside source doing anything to the system... hence, there's basically nothing to stop a virus from blowing things up (except maybe a virus scanner, but their reliability is... sometimes suspect).

Today, Linux is still built on this secure framework. Windows (Vista included) continues to be, at its core, not as carefully segmented and controlled as Linux.  Thus, Linux remains the more secure OS.


As for doing online banking, though? It might be more secure, maybe? I think that depends in part on the stuff your bank uses and your web browser.

### Post by alien21010 on 2007-03-31
As the others have said, a default install of Ubuntu is many times more secure than a default install of Windows XP or Vista for that matter.  However, once you start doing things to any machine, with any OS the opportunity that the system will become compromised increases exponentially.  However you have a tremendous advantage with Linux in that as others have mentioned, access and permissions are much more strictly controlled.  Linux also is much less prone to viruses and other malware.  Online banking though mainly has to do with the encryption that your bank uses and the browser that you use.  Firefox is my default browser of choice, and I guarantee you that I feel much more comfortable online banking on Ubuntu than windows.  

For a while I felt weird not running an antivirus program in the background, but you simply do not need one for Ubuntu.
