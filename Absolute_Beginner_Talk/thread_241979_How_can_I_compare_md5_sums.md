Hi there,

Quite often, people provide .md5 files with their downloads, specifically designed to cooperate with md5sum. Check out **man md5sum** for more information, especially the **--check** option.

Save the md5sum text file as, say, md5sum.txt (which is the list of md5sums and corresponding filenames provided to you) in the directory with the files you have downloaded. Then run this command:
```
md5sum --check md5sum.txt
```

The output should list all the files that it verifies, with an 'OK' to let you know if it checked out correctly.
