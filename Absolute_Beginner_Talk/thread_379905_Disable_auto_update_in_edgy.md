### Post by bobpaul on 2007-03-14
> **Ric95 said:**
> I knew how to disable auto update in dapper, the documentation still shows that gui.

I'm using Feisty, so it /might/ be a little different, but I think it was the same in Edgy...

What do you mean? Disable update notification? Or do you have automatic installation of security updates set and want to turn that off?

Applications-> Add/Remove -> Preferences -> Updates Tab

From here you can turn off notification (untick the box next to "Check for updates ever...") or change the type of notification. Choose between Automatic Security updates, automatic downloading, notification only--download/install when you say so. 

You can also choose which types of updates (backports, security, etc) you will be notified about. I don't remember this from Edgy, so that might be new...
