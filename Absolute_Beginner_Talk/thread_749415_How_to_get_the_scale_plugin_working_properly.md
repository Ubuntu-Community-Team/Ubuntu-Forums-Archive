### Post by alan_daniel on 2008-04-13
First of all, I assume you're talking about the Scale plugin for Compiz, available through the Advanced Desktop Effects Settings application.

If that's the case, I'm not sure if there's an alternative to the problem you're facing. You could consider setting it to activate whenever the mouse simply moves to the corner (i.e. if there's a corner of the screen you never mouse to).

You could also consider using a key combination. I'm on a laptop, so it's really quick for me to switch from keys to the mouse, but it might not be for you. Regardless, it's nice to make it a combination that doesn't require two hands (like Control + Alt + Z).

I'm sorry I didn't answer your question, but I just wanted to make sure you knew of the other options.
