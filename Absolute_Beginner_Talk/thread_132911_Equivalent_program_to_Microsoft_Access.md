### Post by damianubuntu on 2006-02-19
I used access in my business environment, self taught, for about 5 years, and then wanted to switch.  I tried open office, but found it really poor, and heavy on the system resources.  So I got the PHP, MYsql, Apache setup XAMPP.  Its an easy install on windows or linux, and it is so much more stable and reliable than access.  The only problem is that you have none of the wizards that Access provides for building forms and so on, and either have to do it through phpmyadmin (included) or learn some php code, some mysql code and some html, and build all the forms for delivery in a browser.  Its a bit of work, but then Access can take hours of your time as well, with this set up, I found it took me about a week to get things up and running, and another week to fine tune, since then things have run by themselves in a way that was inconceivable under Access, so it was a big time save in the end.  As regards data compatability, export the Access files to (e.g.) text CSV files and you should be able to import them into anything.  There's lots of forum support for php,mysql and the online manuals arer pretty exhaustive too.  PLus its platform independent, and if you want you can put it on someones webserver, they look after all the backups, etc. for you, and you can acccess your data from any web browser anywhere.
I've now set up databases for a few other companies as well, clients of mine, and I'm really glad I toook the step away from Access, though the first couple of days were a bit of a headache!

Good Luck with it.
