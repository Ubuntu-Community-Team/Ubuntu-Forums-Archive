### Post by TheMono on 2007-03-02
Back to the original question: 

Go in to the terminal and paste in this:

glxinfo | grep 'direct'

It should show up "Direct Rendering: Yes", or something to that effect. If so, you'll be fine.

I've never heard of someone with an intel chip having them not installed by default though, so I imagine it should be there.
