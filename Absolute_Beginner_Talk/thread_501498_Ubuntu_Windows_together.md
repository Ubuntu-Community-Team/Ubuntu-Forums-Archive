### Post by Skidpad on 2007-07-15
> **Cozza said:**
> 
Are you saying create the partition with the live cd before using the installation cd?

To clarify:

In Ubuntu, the installation cd ***is*** the LiveCd.  If you download the 7.04 (Feisty Fawn) .iso file, and burn it to disk (as an image, not a data file), you have everything you need.

With this LiveCd, you insert it into your computer, reboot, and you will eventually be taken to to the Ubuntu LiveCd desktop.  On this desktop is an "install" icon.  Double-click that, and the installation & partitioning program will run.

You don't need anything else to install Ubuntu.  (Make sure your computer is set to boot FIRST from the cd, not the hard drive).
