### Post by Buck2348 on 2007-01-23
As I understand it, the system doesn't always write data to a device immediately (it writes asynchronously), so just pulling it out risks leaving some of its data behind.  When you "eject" it that tells the system to hurry up and do all the writes.  I should think that would take only a few seconds, but don't really know.  I believe I've read something about a way to make the writing to be always synchronous, but can't find anything about it right now.
