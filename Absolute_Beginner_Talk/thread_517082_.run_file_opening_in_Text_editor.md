### Post by RomeReactor on 2007-08-04
Hi. **.run** installers have to be installed from the command line; open a terminal (Applications->Accessories->Terminal). If you downloaded it to your desktop , enter the following:
```
cd Desktop
```
and to install:
```
./FILENAME.run
```
make sure you change FILENAME for actual name of the file.
