There is a "MSN" client for Ubuntu called aMSN, and you can install this by going in terminal and typing:
```
sudo apt-get install aMSN
```
That has all the features that MSN have. 

As far as burning a DVD goes, download a program called "AutoMatix" it configures everything for you and there are DVD burning programs and codecs on that script. 
[http://getautomatix.com]("http://getautomatix.com")
