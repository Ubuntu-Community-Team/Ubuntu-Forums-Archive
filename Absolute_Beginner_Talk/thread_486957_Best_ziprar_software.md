### Post by stchman on 2007-06-28
> **lepz said:**
> I have some files I need to upload to Rapidshare, which would be the best zip or rar software to use?

The Archive Manager that comes with Ubuntu called File Roller works well for .zip, .tar, .gz, .tar.gz, .bz., etc.  It however needs the unrar package to be able to unpack .rar files.

sudo apt-get install unrar

You will then be able to unpack .rar archives.
