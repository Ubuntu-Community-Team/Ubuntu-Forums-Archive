### Post by cmittle on 2008-01-15
To delete these unused/incorrect profiles you want to navigate to /home/username/.unison.  This is where the profiles are stored, simply delete the ones you don't want.

I discovered unison a little while ago and just built a file/music server last week that will be used between my gilflriend and my computers.  I thought this would be a nice useable interface for the both of us to back up important information.



I just stumbled across [sbackup]("http://www.debianadmin.com/backup-and-restore-your-ubuntu-system-using-sbackup.html").  This appears to have a much nicer frontend than unison, maybe you could try this one as well?

### Post by cmittle on 2008-01-15
You will see the learning curve flatten.  I'll admit it's hard to get used to all of the differences after being a windows user since the third grade.  Just think about it we, or at least I personally, have been using Windows for 15 years.  The thing is in the relatively short time using Ubuntu, I know so much more about it than I ever knew about windows.  

I believe you are doing the right thing, get out there and try programs, read tutorials, ask questions, eventually you will find that most of these things that seem so unorganized in the beginning will begin to make more sense than 2nd grade math.  

A little while ago I started a personal wiki on my laptop.  This is a place where I can store solutions to problems I've had that are similar to what you are asking.  I also use it for research of new projects.  It is called Zim.  If you'd like to try it out just open a terminal and type sudo apt-get install zim.  [TiddlyWiki]("http://tiddlywiki.com/") is another option.  I have not tried it so I can't speak for it's usability, but it sounds like you can store it on your flash drive and use the program on any computer as long as you have a webbrowser.
