### Post by varunus on 2005-08-12
Yes, nautilus is GNOME's file manager; responsible for the file browser windows explorer-like interface, the spatial file viewer, and the GNOME desktop.

You can switch from nautilus to an alternative if you want; the only one I really know of is called ROX-filer.  Its very fast, but is much harder to use...and the icon theme doesn't correspond with your GNOME theme. (Or at least it didn't use to, I heard they were working on this...)
