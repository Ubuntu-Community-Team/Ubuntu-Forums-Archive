### Post by Donshyoku on 2005-12-30
You could try amaroK... it is by far the prettiest player and has way more useful features than WMP ever had!  There is even an OSD that notifies you of song changes!

You can download in it Breezy using the Add Programs menu.  This program is designed for KDE, so it will install some KDE software along with it, but should run just fine in Gnome.

Good luck!
