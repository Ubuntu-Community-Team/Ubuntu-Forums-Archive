### Post by JShadow on 2007-01-26
Gnome's login program is actually a totally seperate app from KDE's, so you would need to install that as well.

The program is called gdm. To configure it you should need gnome-system-tools.

Keep in mind GDM doesn't really integrate with KDE, so if you switch your shutdown and logoff options will be limited.
