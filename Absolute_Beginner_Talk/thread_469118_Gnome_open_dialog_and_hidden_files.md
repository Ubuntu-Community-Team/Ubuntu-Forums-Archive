---
Happy Ubuntu Feisty user.

If you only want to hide it one time, push Ctrl + h.

If you wanna hide them by default go to Edit > Preferences and Uncheck "Show hidden files and backups". Then push ok and reopen. They disappeared :).

Push ctrl + h again to get them back if you want them just for a short while.
