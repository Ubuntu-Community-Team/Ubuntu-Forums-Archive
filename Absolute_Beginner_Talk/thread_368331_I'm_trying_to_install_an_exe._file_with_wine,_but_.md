### Post by Tomosaur on 2007-02-23
Thanks for waiting a total of 12 minutes ¬_¬

Some .exe files just wont work. Check [here](http://appdb.winehq.org) to see if your program works or not. If it's been submitted, you should be able to find some information. If you find no record of it, then submit it and tell them what errors (if any) you get, otherwise it'll never improve.

.exe is for Windows - don't be surprised when they don't work. It's a miracle wine works as well as it does for the number of apps it does work for.

On the command line, type 'man wine' for usage instructions. Hit q to exit the man page.
