### Post by rsambuca on 2007-05-21
> **teknosexual said:**
> How does that work? I have wanted to do the same thing. I'm only dual-booting Ubuntu and XP here. Any tips?

I have a small FAT32 partition that I use for the thunderbird and firefox shared folders.  In thunderbird, after I have set up the account profile on one OS (on the FAT32 partition), I just point all the other thunderbird accounts to the same folder.  Just go to:  Tools -> Account Settings.  Under the Local Folders, just point the Local Directory to the same folder as the first one you have set up.  Do this for all of your OS's and you have your mail, settings, addy books shared across the board!
