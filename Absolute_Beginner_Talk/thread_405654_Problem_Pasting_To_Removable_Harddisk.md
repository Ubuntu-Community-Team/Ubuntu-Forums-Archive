### Post by LaurelLynn on 2007-04-10
Dear Saini Kapil,

If the drive has windows NTFS partitions they will mount as read only.

To be able to write them you need to down load drivers with NFTS write capability.

You can find the ntfs-3g drivers at Sourceforge. com (just put ntfs-3g in their search box).
Also there is a link that says goto ¨Project Website¨, they will have Ubuntu (or Debian) instructions there.

Those drivers have write capability

Laurel Lynn
