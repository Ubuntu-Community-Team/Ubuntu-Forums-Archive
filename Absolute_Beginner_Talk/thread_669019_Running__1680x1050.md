### Post by kpkeerthi on 2008-01-16
Applications -> System Tools -> Nvidia Setttings and see if you are able to switch to desired resolution. If you are able to, then open terminal and type **gksudo nvidia-settings**, change the resolution and click **Save to Xorg.conf**. Reboot.

### Post by kpkeerthi on 2008-01-16
If the above method did not work, boot into recovery mode. Edit /etc/X11/xorg.conf. Find the section that looks like
```

Section "Device" 
        Identifier      "NVIDIA Corporation NV10DDR [GeForce 256 DDR]"
        Driver           **"nvidia"**
        BusID           "PCI:1:0:0"
EndSection

```
and change it to
```

Section "Device" 
        Identifier      "NVIDIA Corporation"
        Driver          **"nv"**
        BusID           "PCI:1:0:0"
EndSection

```
Save and Exit. Reboot normally.

Reinstall the binary driver as suggested in the wiki [here]("https://wiki.ubuntu.com/BinaryDriverHowto/Nvidia")
