### Post by logos34 on 2007-06-17
Download and install all the updates and reboot.  Applications>add/remove.  System tools>nividia binary xorg driver (8776) [or the 'legacy' driver, but this seems to conflict with the info that is listed on nvidia website for the geforce2, which uses a relatively new 9755 driver].  After it's installed make sure to run 'sudo nvidia-glx-config enable', then restart X - alt-ctrl-backspace.
