Since your using you have k3b the burning suite by default (I believe, if not install). Excellent app. If it is a video DVD you can open k3b (in menus under multimedia) and go Tools > Rip Video DVD. There should be an option to save as an image.

If you want to make a data project (i.e. just files) click on Create New Data DVD Project. Then select all the files on the DVD into the project window and say burn, then choose the "Create Image only" option and I believe it asks you where to save (or does it go in the default directory set in properties). That should do it.

I hope that's right, I don't make DVD images...
