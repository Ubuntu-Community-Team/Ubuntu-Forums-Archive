### Post by scruff on 2007-10-14
> **limac said:**
> well blender was just an example. So i am supposed to download WINE in order to run apps downloaded from internet? And what kind of files am I supposed to download if not exe.

Thnx in advance for the upcoming help.

No. You are supposed to run software that was written for the operating system you are using at the time. In this case, Linux based software. Those .EXE's are Windows applications. Some Windows apps can be ran through WINE (an emulator - see here: [http://www.winehq.org/site/about](http://www.winehq.org/site/about)). 

Personally, I've never used WINE. I've ran Linux exclusively for 6 years and I find the alternatives to windows based apps to be excellent quality.

Software for Linux, posted on the internet, generally has one of the following extentions: .tar, .tar.gz, .tgz. In any case, you should be able to get damn near anything you might want through Ubuntu's package management system without bothering to manually download and install.

Here is a good beginner guide: [http://simplyubuntu.wordpress.com/2006/06/27/a-beginners-guide-to-installing-programs-in-ubuntu/](http://simplyubuntu.wordpress.com/2006/06/27/a-beginners-guide-to-installing-programs-in-ubuntu/)

Remember - .EXE = Windows, *not* Linux.
