### Post by DogsOfWar on 2007-07-17
Never used an iso before? ;-)

An iso file is a cd image, to use it you need a program like _[Infra Recorder]("http://infrarecorder.sourceforge.net/")_ or _[Iso Recorder]("http://isorecorder.alexfeinman.com/isorecorder.htm")_ which will let you burn the image onto a cd.  Then you can use the cd to boot and/or install ubuntu.
