### Post by jbonll05 on 2006-11-05
This machine I built myself, It has a Biostar N7MCC400 mobo(2yrs old). This is an old board that still is quick I also have a later one with an Elitegroup K8M800-M2 mobo. It has an AMD 64 processor. I built a Celeron D 530 machine for a friend that works well with Linux. Most Linux, including Ubuntu, work well with Nvidia and Via chipset on the mobo, pci, etc. If you wifi, generally stay away from BroadCom lan. This is not an issue if you do not use wifi. Read the posts in Ubuntu forms this if necessary. I have a laptop with a Broadcom wifi lan card, after 3 weeks it is speaking to me, showing active, no connect. Getting closer.
   I would think the question would be 32 or 64 processor and the mobo to go with it. Most of the mobos that are 2years old or less do not need a pci video card, somedo not need a TVcard.
   Connecting your monitor - old VGA or new DVI for late model flat panel monitors. Some thoughts, hope they help. JB
