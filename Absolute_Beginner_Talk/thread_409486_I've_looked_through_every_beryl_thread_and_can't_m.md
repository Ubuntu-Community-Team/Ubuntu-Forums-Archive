### Post by chalex on 2007-04-14
The text for the Identifier is entirely arbitrary.  However, the text gets referenced in the Screen section further down in the conf file.

So if you change your Identifier to "a" in the Device section, make sure you change the Device to "a" in the Screen section.

You have two Screen sections and two Device sections and two Monitor sections.  I'm not sure if X will start with this config.  How does it know which Screen to use?
