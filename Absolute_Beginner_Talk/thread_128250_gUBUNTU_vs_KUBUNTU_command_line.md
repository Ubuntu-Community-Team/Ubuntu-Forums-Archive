### Post by heimo on 2006-02-11
There's no difference when using command line (also called shell*), but there's difference in default set of programs installed. gedit is graphical text editor for Gnome (Ubuntu's desktop environment) and kate (as far as I know) is similar program for KDE (Kubuntus desktop environment). These are both GUI programs. To edit text files you could use text mode editor like [I]pico.


*) [/I]default shell for (K)Ubuntu (and most Linux distributions) is bash.
