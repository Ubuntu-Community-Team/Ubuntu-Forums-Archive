### Post by freebird54 on 2007-06-03
What you may have is the 'Recovery Console' optional boot showing up as an icon.  In a default Ubuntu install you would have that choice automatically.  Essentially you get to the same installation, but as a repairman without a GUI to help!

Have you tried the 'second' one yet?  It might just be an anomaly - but I think my first guess is more likely.  Very unlikely that you actually have 2 installs on the go (if you don't see any partitions for the purpose).

Hope this helps!
