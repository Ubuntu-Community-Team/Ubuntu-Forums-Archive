### Post by sbkcu on 2005-12-27
I finally found the answer to my own question with Wikipedia.  Hold CTRL and SHIFT, and type the character code.  For instance, a unicode guide (see Applications->Accessories->Character Map) will tell you that the code for é is U+00E9.  To do this, hold CTRL and SHIFT and type 00E9.  I've found that in openoffice, you don't have to type the leading zeroes  (e.g. to type é hold CTRL SHIFT and type E9.  It goes into effect when you type the spacebar).:smile:
