### Post by Inxsible on 2007-10-18
If you are using Gutsy, reading and writing to a NTFS partition comes standard.

If you are using Feisty, reading from NTFS comes standard, but you have to install ntfs-3g to be able to write to NTFS drives.

Of course, in both cases it means that you can access the Windows files thru Ubuntu :D
