### Post by Calash on 2008-04-01
OpenDNS has the advantage of working on a different level than something like Dansguardian.  The down side, if you see it this way, is that OpenDNS applies it's rules to the public IP, so if you have a router all systems attached will have the same restrictions.

There are ways around that...but that gets a bit more complex.
