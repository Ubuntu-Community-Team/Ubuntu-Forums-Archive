### Post by PeP on 2005-08-13
It depends on the hardware router and who shall be allowed to see your web server.

If you want your web server to be seen from the internet, you'll have to set up your router for that (it is the port redirection or port mapping functionality).
If your router can't  do that then you will have no choice but put your linux box first.

Else just use the router.

PS if you are a geek and want to learn, hide the router, do it all the "hard" way. It is quite interesting the first time.
