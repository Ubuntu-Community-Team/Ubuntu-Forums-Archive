### Post by ayenack on 2008-01-07
OK this is doable. This is how I did it. First set the screen size the same as your desktop. Mine was 1024x768 so in terminal server I set screen size to 1024x768. Then with **ctrl+alt left or right key** I was able to switch between desktops. Tried this with compiz effects on and off and it works with both settings. I'm using xnest as my XDMCP client, using a different client may make it not work.

Would just like to say that cmnorton's post made thing along these lines, so thanks to cmnorton for that.
