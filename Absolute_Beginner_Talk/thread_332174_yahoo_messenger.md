### Post by energiya on 2007-01-05
The Yahoo Messenger offered by the yahoo guys is very old, Gaim works better. If you still want to try, use the debian files. Download and write in the terminal
```
sudo dpkg -i file.deb
```
where "file" is the name of the package.
