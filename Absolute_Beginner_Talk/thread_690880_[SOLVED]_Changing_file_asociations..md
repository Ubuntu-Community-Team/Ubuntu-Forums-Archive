### Post by linuxisfree on 2008-02-07
Right-click the file then go to properties... there should be a tab which says something like open with (or an equivalent) then if you don't see vlc in the options, then press the add button... then look for vlc (pathway is **usr -->bin**) then vlc should be somewhere there.
