### Post by alienexplorers on 2007-05-09
If you got the program from the mozilla website then you just need to un tar the file.  Then go to your desktop and right click and select create launcher.  

In the menu Add the name Thunderbird 2 and on the line command enter /home/yourname/thunderbird(name of directory where thunderbird is loaded).  Pick and icon and click OK

Now each time you click the icon it will bring up thunderbird 2.
