PS

Same goes for a **firewall**, Ubuntu has one built in, *iptables* with a default setup.  If you're not running the Server Edition it is very safe as long as you stay with Ubuntu's Programs, or only get programs from sites you trust.
If you have a need to configure *iptables* there is a GUI program, *Firestarter*, that you can use to change the configuration.  You only need to run *Firestarter* when you want to "change" the configuration, as *iptables* is always running
