### Post by tubasoldier on 2006-10-21
Literatka,

 First of all welcome to linux. I hope. Anyways to the icons...

Any icons you get from kde-look.org can be installed very simply. 
Kmenu ---> System ---> System Settings ---> Appearance---> Icons --->
Under the icons dialog you will see a large button near the bottom that says "Install New Theme" click on it and select the .tar.gz file you downloaded from kde-look.

An easier way is to open synaptic and search for "icons". select the ones you want and press the apply button. Then you can change your icon theme under the icons dialog in the system settings.
