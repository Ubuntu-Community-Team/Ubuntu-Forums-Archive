### Post by NeoLithium on 2006-10-15
For gnome, there is gdesklets, which is available in the repository ```
sudo apt-get install gdesklets
```
They have a list of desklets available for download from their homepage [http://www.gdesklets.org]("http://www.gdesklets.org")

Adesklets was another option as well; but I haven't personally used that; so I'm not quite sure about it. There is a forum thread [here]("http://www.ubuntuforums.org/showthread.php?t=94782&highlight=adesklets") which compares the two programs, and has links to their downloads as well :)
