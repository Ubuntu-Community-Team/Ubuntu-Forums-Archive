### Post by gh0st on 2007-01-24
Hi, welcome aboard :D

You might wanna try using a tool called Automatix to install your media players, nvidia driver and so on. It's great for getting a machine set up quickly for newbs, I like it anyway. It will take a lot of the pain out of the set up process for you.

This is their site: [www.getautomatix.com](www.getautomatix.com)

You might also wanna have a look at this guide on installing popular programs with Automatix, I followed it and it worked for me. Fixed my nvidia driver problems as well, I have a 19" Widescreen LCD and it didn't look very pretty in 1024x768. Looks sweet now though.

[http://www.howtoforge.com/automatix_ubuntu](http://www.howtoforge.com/automatix_ubuntu)

The article is a month or so old but it's still relevant I think. Can't help with the SATA problem sorry dunno much about that, I'm sure someone else will sort you out though.

As for 3D desktop stuff, there are 2 main choices really Compiz and Beryl they both do 3D desktop. I use Compiz personally so I can't really comment on Beryl. I used this guide to get my Compiz working [http://lunapark6.com/?p=2501](http://lunapark6.com/?p=2501)

That Lunapark article is also a nice guide on setting up a new Ubuntu install.

If you need anything else let me know, welcome to Ubuntu :D
