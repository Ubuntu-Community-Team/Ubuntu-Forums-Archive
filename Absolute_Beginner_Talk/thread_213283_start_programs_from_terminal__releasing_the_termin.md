### Post by bluppfisk on 2006-07-11
The terminal is probably the quickest way to start a program. But what I get when I start a program from the terminal (f.ex. I type gaim into the terminal window), is that I need to reclaim access to the terminal, because it then contains output from the program I've just started. Pressing ctrl-c works, but also terminates the program.

So my question is, how can I start a program from the terminal so that it immediately detaches itself from the terminal?

---

### Post by mogelhead on 2006-07-11
Try adding the & sign after the command, for example:
```
gaim&
```
