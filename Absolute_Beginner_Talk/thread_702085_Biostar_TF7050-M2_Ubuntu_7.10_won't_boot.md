### Post by Rocket2DMn on 2008-02-20
The LiveCD does not work for everybody, so your other option is to get the alternate install cd which doesn't have a live session, just a text based installer (which actually works very well, and is intuitive to use) - you just can't take Ubuntu for a trial run with it.  You can download it here by checking the box at the bottom for Alternate CD - [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)
