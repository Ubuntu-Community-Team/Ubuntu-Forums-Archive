### Post by Klaidas on 2006-09-03
Well, you could first look up some php tutorials on the Internet and start coding! :)

Then, you should put your .php files to your /var/www which is the default Apache's web directory. (You have installed Apache, right? :))

Then, just go to [http://localhost](http://localhost) and there you go!

Here's an example:
```
<?php
  echo("Ubuntuforums!");
?>
```
