### Post by rsambuca on 2007-02-12
That is a rather strange setup for your xorg.conf.

Try commenting out this section as I don't think you need it for your current setup.

#Section "Device"
#Driver "nvidia"
#Identifier "Device[1]"
#Screen 1
#Option "TVOutFormat" "SVIDEO" #or SVIDEO etc
#Option "TVStandard" "NTSC-M" #or NTSC etc
#Option "ConnectedMonitor" "Monitor[1]"
#Option "TwinView" "True"
#Option "RenderAccel" "true"
#Option "AllowGLXWithComposite" "true"
#BusID "PCI:1:0:0" #adjust using 'lspci' or cat /proc/pci
#EndSection

### Post by mr.v. on 2007-02-12
Also...use the nvidia-settings program to set things up. See if it detects both monitors etc. It's quite useful and makes things pretty easy. Just be sure to back up your previous xorg.conf in case.

You can run in either from a console with nvidia-settings
or
it generally installs a launcher in Applications --> System Tools --> Nvidia X server settings
