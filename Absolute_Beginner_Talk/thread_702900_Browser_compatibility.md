### Post by spiderbatdad on 2008-02-20
Obviously IE does not run on Linux. There is a program called ie4linux, but It has some issues and limitations. Another option is to install vmware-server from the package manager in Ubuntu and install a virtual windows operating system into a folder of your Ubuntu file system. From the server you can run windows and IE.
