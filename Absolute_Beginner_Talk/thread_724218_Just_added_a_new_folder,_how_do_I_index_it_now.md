### Post by BluntBox on 2008-03-14
Assuming Gnome. System -> Preferences -> Indexing Preferences.

You can add directories on the Files tab.

I've never played with indexing options. But I think you only need to add directories such as /etc, /var, /usr etc. Devices that are auto mounted and anything inside Home should be indexed by default. I could be completely wrong though :D
