### Post by Famicommie on 2007-04-10
I keep additional background images saved in /home/fami/images/desktops
I believe that you can simply drag and drop images into the "Desktop Background" window and it will automatically save apply them/save them in the list. Also, firefox supports the right click -> set a background feature.

The default file browser in linux is called Nautilus. You can open it by clicking Places->Home and change to tree view with View->list

To install adobe acroread, open up a terminal and type in the command

```
sudo aptitude install acroread
```
