### Post by HereInOz on 2007-12-29
If you are judging it by the length of time to open a browser or a word processor, do not forget that with Windows, a great deal of Internet Explorer is already loaded at boot up, as is Office, so the apparent "time to load" will be shorter.

This may seem like a good idea, but it is this interactivity which gives the Windows programmers such headaches when trying to make their system secure.  Because Windows uses IE to display folder listings, and such like, if you compromise IE, you have a free ride into the system.

Linux is not built like this.  It is all individual components and yes, gnome does seem to take longer to do things than XP, because of the need to load everything from disk (or cache) each time.  This is something which I am happy to put up with (should that be "that is something up with which I am happy to put"?) for the added security of the system.

Try running Open Office on your XP machine, and see how that performs.  That is a fair comparison.
