### Post by dbd on 2006-09-16
If it runs OK with the CD then it should run fine when installed. But if your WLAN card doesn't work in live then it won't work straight away when Ubuntu is properly installed, and it may not be easy to get it to work (or it may be easy, I just don't know).

As for dual booting, yes, you can install Ubuntu on a different hard drive from your windows hard drive if you have 2 different hard drives in your computer. Then at boot the GRUB bootloader will ask you what operating system you want to boot when you start up your PC .
