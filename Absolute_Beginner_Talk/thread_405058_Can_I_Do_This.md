### Post by Hendrixski on 2007-04-09
Yes, of course.

You can install it so that it's the only operating system on your computer, or one of many.  The installer asks you those questions so it's easy to set it up.  If you erase windows you'll lose all of the data, so e-mail yourself any important documents that you have saved on there.  If you do "dual boot" then when you turn on the computer it asks you which operating system you want to boot to, Windows of Linux.

There is a TON of documentation about this on the web.  Just google "install Ubuntu" and you'll get hundreds of blogs, HOWTO manuals, documentation, troubleshooting guides etc. etc.  If you can't find the answer there then come back here and we'll help you.  There's also a help channel on IRC called #ubuntu, for live help.

Most importantly.  Welcome to the Community.  :-) We love the software, and what it stands for (it stands for freedom, where you use your software and not your software using you)... and we're sure that you'll love it too!
