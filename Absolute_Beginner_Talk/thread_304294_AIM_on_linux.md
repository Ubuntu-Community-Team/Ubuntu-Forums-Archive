### Post by nonpareilpearl on 2006-11-21
Not to beat a dead horse, but gaim is really the best way to go. I used it on Windows for MSN, Y!, ICQ, and AIM and all had the same functionality as if using a native client. I recently installed Ubuntu on my computer and so far the same has been true. In order to set up MSN on gaim do the following:
1. Open gaim
2. Go to "Accounts" by right clicking the icon that appears by your clock to the upper right (if there are no accounts, I believe this is default)
3. Click "Add"
4. Under protocol select MSN
5. Enter your username (including @hotmail.com or @msn.com) and password as though you were logging into your email account
6. Alias: this is what your name will appear as to you (for instance all of my aliases for my screennames are set to display "me" so instead of seeing "whatever@hotmail.com" on my buddy list I see "me")
7. Select remember password, Email notifications, or add a buddy as you please. When you are finished select 'Save'
