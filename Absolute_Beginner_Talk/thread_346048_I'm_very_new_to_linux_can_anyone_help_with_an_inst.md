### Post by indytim on 2007-01-25
A couple of things on a quick reply (other members of the collective will chime in shortly).

1.  If the H/W you are attempting to install on is <192M RAM, you will need to use the alternative installer (text-based).
2.  When you boot to the install CD, there is an option to check the install CD.  In addition to the md5 that you referenced, check this option before your next attempt.
3.  More witchcraft that tech, but I always burn my iso cd's on the slow side.

Should see more responses momentarily.

Good Luck,
