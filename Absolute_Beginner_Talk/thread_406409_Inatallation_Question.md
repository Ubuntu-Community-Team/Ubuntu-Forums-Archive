### Post by lamalex on 2007-04-10
if you're new to Ubuntu/linux go with x86. You're not going to see a big performance boost with 64bit unless you want >4gb of ram. 32bit will run pretty much all the programs you want whereas the 64bit version you might have more trouble getting things to work.

### Post by baeckel on 2007-04-10
If your machine's heart is an AMD Athalon 64 then you're going to want to use the 64-bit version of Ubuntu.

I would suggest downloading the .iso via bittorrent for a much happier file transfer experience.


(If you're still running windows you can double check your chip by keystroke: windowsKey + pause/break and look down towards the bottom of the window that opens, if it doesn't say 64 anywhere you probably want the i386 .iso)

### Post by jem7v on 2007-04-10
The 64 bit version will probably give you slightly better performance, but you'll almost certainly have an easier time getting everything to work in the i386 version (such as Flash, various codecs, some programs, some printer drivers, etc).

You don't actually HAVE to use the 64 bit version with a 64 bit processor. The i386 will run fine... it just won't quite take full advantage of all the chip's processing power.
