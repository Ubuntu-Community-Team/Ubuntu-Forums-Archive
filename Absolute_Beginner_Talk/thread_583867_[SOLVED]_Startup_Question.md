### Post by nrfool on 2007-10-20
If you are in Gnome, you can find the command to launch the program by right clicking on the program icon in the "Application" menu. Then you can try to run that command in commandline,  if the program you want to launch does launch successfully, then you have to right command.
If you are in xfce, you can try "menu"->"Accessaries"->Appfinder, type in the name of the program, hit enter. When the result comes up, right click in the search result and choose "information".
