### Post by Skefflock on 2007-11-17
The best way to save all the information during any changes like distros, reinstallation and so on is to create a separate partition and mount /home folder on it, so you'll be able to mount the whole partition on different system as well.

### Post by daimaru on 2007-11-17
you should backup all your files before install. i never had problems, but you never know what might go wrong.
a different partition might be sufficient, but that all depends on what you do during partitioning @ install. if you choose wrong partition accidentally everything will be lost. so again backup first then install. :)
EDIT: 
i would suggest using partiton magic to create 
one root directory / - 10GB
one /home directory - as much as you can spare
and one swap space of about double your RAM size.
during install select manual partition and define your partitions as / , /home , and swap and your good to go
