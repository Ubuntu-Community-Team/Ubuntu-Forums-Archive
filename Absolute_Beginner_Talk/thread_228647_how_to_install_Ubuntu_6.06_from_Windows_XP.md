### Post by MartinJ on 2006-08-03
Instlux ( [http://sourceforge.net/projects/instlux](http://sourceforge.net/projects/instlux) ) is a windows application which loads part of the ubuntu installer onto your computer. Then all you need to do is restart windows and select 'Install Ubuntu' before windows starts up.

You will need a reliable broadband connection as all the packages are downloaded during the install - took about an hour for mine to finish downloading.

Hope that's what you were looking for...
