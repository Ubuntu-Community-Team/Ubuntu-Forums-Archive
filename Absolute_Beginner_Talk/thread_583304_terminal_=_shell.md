### Post by Linux_Man on 2007-10-20
Not really, the terminal is a program that opens up a command line program (the shell) the shell is most likely bash (The Borne Again SHell) but can be used with other shells like sh, zsh (A shell for power users) or fish (The friendly interactive shell for newbies) for example most likely you open up Gnome terminal for your terminal and bash for your shell, for commands try either picking up an older Linux/UNIX book those help a ton (I learned the most about linux by reading an old UNIX sysadmin guide) or a pocket reference to Linux commands (If they say Fedora or SuSe or other distros it still should work for Ubuntu for the command part) or try googleing linux commands
