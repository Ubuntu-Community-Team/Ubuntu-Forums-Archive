### Post by eljalill on 2007-03-24
Open synaptics package manager, 
search for your program,
right click on it and chose "mark for complete removal".

But I am not sure which program you are talking about...
If synaptics says, there are other programs, which it will also uninstall, if you do that, look at them. 
Do NOT uninstall, if there are packages like ubuntu-desktop or linux restricted headers or anything like that which will also be uninstalled.
If in doubt, please make a backup!
