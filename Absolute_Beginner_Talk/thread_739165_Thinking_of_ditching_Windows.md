### Post by dsauxier on 2008-03-29
The liveCD is a great way to see if it will work for you, but because it is running from a CD, many things are much slower than they are when running from a hard drive.  

To answer the original question in the post "how do I get rid of Windows"

The most important question here is "what do you use your computer for?"
When I made the switch, I went through my start menu and listed all of the programs, scratched off some that I didn't care about and went looking for linux-equivalents for the rest.

Running dual-boot is a good way to get started.  If you want to just jump in, that's fine too.  But first, use the liveCD to make sure that things like printers/scanners/copiers, etc that you may have are functioning.

If you come up with a list of applications that you want to find linux-equivalents for, post it here.
