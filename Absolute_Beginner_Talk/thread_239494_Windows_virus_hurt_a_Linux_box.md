### Post by bodhi.zazen on 2006-08-19
In the Linux box: Nothing happens. The virus will try to run each time you open the file, but it can not. Some people have tried to run virii under wine  and worse case scenario has been the wine session will freeze.

All virii and worms exploit flawed code, which Windows has in abundance. 

However, any files infected will remain infected and if you transfer or E-mail the file to a Windows box the windows box will become infected.

At this time the only reason to run antiviral software on Linux is to protect you Windows using friends & family. I dabble with clam antivirus because I hate the thought my box could act as a repository for virii, but that is me.
