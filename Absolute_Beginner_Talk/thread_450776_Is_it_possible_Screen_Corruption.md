### Post by steve.horsley on 2007-05-21
I can't offer much help, just a few comments.

I very much doubt that your nvidia drivers have been corrupted - if they were, they would probably just crash. And I don't see any harm in leaving applications installed - if you don't run them, they just use up a little disc space. And removing the default applications then tries to remove package Ubuntu Desktop which is harmless but very scary sounding.

Sudden corruption is not normal - these forums would be full of it if it was. Did installing the DVD player involve installing hardware? I might be inclined to remove it again for a while, as an easy-to-do test, and see if that makes the problem go away.
