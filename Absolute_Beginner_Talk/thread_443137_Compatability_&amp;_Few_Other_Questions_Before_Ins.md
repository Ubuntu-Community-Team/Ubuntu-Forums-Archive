### Post by ugm6hr on 2007-05-14
Unlike Windows, you can try Ubuntu out on your hardware by using a Live CD boot.  It will load within a couple of minutes, and you'll have your answer!  If it works from the CD, it's generally straightforward to install to hard drive, and obviously loads and runs a lot quicker.

I installed Xubuntu7.04 yesterday, because it is supposed to be quicker than Ubuntu or Kubuntu.  But the point is, I haven't had to use the Terminal command prompt yet (although I did use text entry to install the full Synaptics Touchpad driver on my Acer Aspire5051 laptop, but the Touchpad worked without it).  As for things that I have achieved without text entry - wireless networking, firewall, GAIM internet messaging, Azureus torrents, and obviously internet access.  Everything is fairly easy to install from the GUI.  Xubuntu even notified me that my Graphics card (ATI Radeon 1100) and Atheros wireless (5005) card drivers are "Restricted", and allowed me to decide whether to use them or not (which obviously I ticked yes). 

And now I'm giving advice to you!  How's that for easy progress?
