### Post by mcpish on 2005-09-29
Killer apps for me:

VLC - Definatly the best media player, fully supports the latest x264 codec
K3B - A KDE CD/DVD burning app but it works/looks fine in Gnome
Nvu - good frontpage style web development suite
The Gimp - Good Photoshop substitute 
Pan - Great newsreader, similar to Forte Agent on Windows
Klibido - Mass Binary newsgroup downloader, similar to Newsbin on Windows
Audacity - Good Multitrack audio recorder/editor, similar to Sound Forge on Windows
KAudioCreator - Good MP3 Ripper/encoder
NVTV - Unity to enable and customize the TV-OUT feature on Nvidia video cards (ie enable overscan, etc), great for watching video files on a TV
