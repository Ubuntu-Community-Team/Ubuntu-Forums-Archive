### Post by erginemr on 2008-03-31
I can also suggest you to download GParted Live CD and partition your drive with it:
[http://gparted-livecd.tuxfamily.org/](http://gparted-livecd.tuxfamily.org/)

Then, once you partition your hard disk as you like, you can install Ubuntu form Ubuntu Live / Alternate CD, but pass the partition stage and indicate the corresponding mount points ( / and swap) to match with the partitions you have already made with the GParted Live CD.
