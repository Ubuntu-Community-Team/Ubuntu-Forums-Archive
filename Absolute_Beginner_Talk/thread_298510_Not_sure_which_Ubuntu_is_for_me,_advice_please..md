### Post by PilotJLR on 2006-11-12
> **Steve Pullman said:**
> 
I have an AMD64 based system and I see that there are Ubuntu's for this, as well as for i386. Is there any difference? 64's are supposed to be *much* faster than old CPU's.

I would recommend using the i386 version... most users do not need the special AMD64 binaries (some important applications are not available for AMD64).
It's a common misconception that 64 bit processors like AMD64 or ANY new Intel are automatically faster... rather, the 64 bit capability only means the processor can directly address much more RAM.
You will not have a performance hit using the i386 binaries, provided you have less than 4GB of system RAM.

I have an AMD Athlon 3700, and I use the i386 binaries as well...

### Post by elcasey on 2006-11-12
My first Ubuntu book was (and I'm still using it for reference) *Ubuntu Linux for Non-Geeks*. Don't let the title fool you, it's chocked full of excellent information, but without getting too bogged down in the minutiae. It does not, however, cover much on the Terminal front (but that's what O'Reilly's *Learning the Bash Shell* is for, eh? 8)).

I've been using Ubuntu for about 2.5 weeks now and I'm in love with it. I will not go back to Windows for a daily OS ever again (although I said that over a year ago when I bought my first Mac). If/when I build myself another desktop PC I'll dual-boot it with XP or Vista so I can get back into gaming, but for productivity, media and everything development- and internet-related there's absolute no reason to ever use Windows again, especially with Wine (although Wine is giving me some serious trouble).

As soon as Apple releases iTunes for Linux it's unlikely I'll use my iBook much after that!

After trying both I prefer GNOME over KDE, but that's mainly because I don't care for the "XP-ness" of KDE, even if Linus Torvalds does (:p). Even if you install Ubuntu you can install the KDE packages via Synaptic and choose to log in to KDE (or Fluxbox, or Enlightenment, etc). I probably won't use KDE that much, but it's nice to have it available.

No matter what, if you just have some patience, aren't afraid to ask questions here (which you obviously aren't) and have an open mind I think you'll really like Ubuntu. I certainly do!
