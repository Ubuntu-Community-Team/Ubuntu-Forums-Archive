### Post by JayTheHun on 2007-04-20
Right-click My Computer, Properties  (or Start-Control Panel-System), Advanced tab, the Settings button in the Performance area, Advanced, Change button, click the No Paging File radio button, click on the Set button.  OK and Apply and OK your way out, then reboot.
