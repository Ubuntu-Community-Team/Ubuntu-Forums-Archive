### Post by andrewsawyer on 2006-03-08
You can use Open Office in either Ubuntu or Kubuntu - they are installed as standard in both (as far as I know - I've never used kubuntu).

Kubuntu uses KDE as its GUI where as Ubuntu uses Gnome.  If you like to mess around with different windows borders and stuff then Ubuntu is easier to do this in, however I *think* that Kubuntu has more eye-candy as standard.  Someone correct me if I'm wrong on this.

For any server purposes there is absolutaly no difference.  For games I'm not sure, but I guess if you'll be using Cedega for Windows games then there prob wouldn't be much difference between the two either.

I guess it just comes down to which you like the look of most.

### Post by ShanghaiTeej on 2006-03-08
Starting out, I would recommend Ubuntu, just because it has the least amount of bugs and problems.  Also, I've find that hardware works better out of the box on Ubuntu than Kubuntu.  Mind you, nobody's computer is the same, but I think people have more trouble with a working installation Kubuntu than Ubuntu.  The reason is that Kubuntu is a newer distribution and is still working out the kinks.  Once you get yourself familiar with Ubuntu, install Kubuntu and see which one you like better!
