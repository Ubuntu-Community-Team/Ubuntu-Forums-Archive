### Post by KentS on 2007-02-08
'uname' prints system information. The option '-r' tells it to print kernel information. So 'uname -r' gives the kernel version your using.

'Sudo apt-get install' takes care of the installation process for you. I didn't have to do anything manually when I installed the kernel-headers.

You may have to do
```
cd alsa*
(perhaps as root) make distclean
```
before trying to build the program again, to make everything work properly.
