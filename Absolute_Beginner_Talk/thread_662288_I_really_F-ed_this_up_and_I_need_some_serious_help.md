### Post by blueridgedog on 2008-01-08
You have a choice of two window managers now that you have installed compiz.  

compiz --replace 
activates compiz window manager

metacity --replace 
activates the metacity window manager

See:

[http://ubuntuforums.org/showthread.php?t=662149](http://ubuntuforums.org/showthread.php?t=662149) for a glimps of my experimenting.  I am still figuring window managers out.
