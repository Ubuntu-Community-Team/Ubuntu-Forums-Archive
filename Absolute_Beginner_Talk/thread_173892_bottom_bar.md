### Post by joshrobinson on 2006-05-11
[QUOTE=fake123]tell me something else i dont know (just about anything in this case)[/QUOTE]

um you can make your top and bottom bars transparent by right clicking it going to properties on them, clicking the background tab then click solid colour and move the transparency slider

you can get themes from gnome-look.org for new icons and window themes

alot of nice programs to download in synaptic package manager.. you can click applications and go to add remove programs for some free games

when using terminal you can type sudo before your commands which stands for super user do, so it runs your commands as root (administrator)
if you type su in a terminal you can log that terminal in as root so you dont have to type sudo in front of all of your commands

mplayer in the repositories is a nice media player for videos

you can find and download an application called frostwire to download some mp3's
xmms is a winamp clone for playing music

you can sync up your ipod with gtkpod (if you have an ipod)
