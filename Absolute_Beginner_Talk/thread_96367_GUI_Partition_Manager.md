### Post by matthewv on 2005-11-28
GParted is not installed by default, it is only on the live cd. You'll have to install it: ```
sudo apt-get install gparted
```

### Post by joseftu on 2005-11-28
Just below "system tools" on the Applications menu you'll see "Add Applications".  You'll have to use that to add GParted...but it's pretty easy.  You'll see it right there in the System Tools list after you open the add applications window.

(Or if you're more skilled, you can use the command line! :))
