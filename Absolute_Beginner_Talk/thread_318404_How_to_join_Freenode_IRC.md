### Post by synacktion on 2006-12-13
Sounds like you just want to connect.  For starters, you will need a client.  I recommend XChat because it seems to be pretty intuitive for newbies.  The built in server list should have freenode in it.  Just select it, create a nick name, and connect.  If you need help with something join the #ubuntu channel or if you just need some socializing join #ubuntuforums (we're very welcome to new people :)).

### Post by seshomaru samma on 2006-12-13
Download Xchat
```
sudo apt-get install xchat
```
then on xchat's main windows you will see many IRC servers , choose Freenode and press 'connect' , wait a few seconds and a new window will appear asking you which channel to join , you can then choose your channel . Ubuntu's support is 'ubuntu'...

### Post by md5 on 2006-12-13
