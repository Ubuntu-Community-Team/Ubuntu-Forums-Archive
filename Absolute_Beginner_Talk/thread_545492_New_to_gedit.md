### Post by avik on 2007-09-07
To install a plugin, either install the appropriate package, or download the files and put them in your ~/.gnome2/gedit/plugins/ folder.  You can find more information at [http://live.gnome.org/Gedit/Plugins]("http://live.gnome.org/Gedit/Plugins").  Hope that helps.
