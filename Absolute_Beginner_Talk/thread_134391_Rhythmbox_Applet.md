### Post by nalmeth on 2006-02-21
You have to install gdesklets, then install the particular gdesklet you're after -- rhythmlet

You might want to use automatix (ubuntuforums.org/showthread.php?t=66563) to install gdesklets. then launch gdesklets, and go here: [http://gdesklets.gnomedesktop.org/categories.php?func=gd_show_app&gd_app_id=162](http://gdesklets.gnomedesktop.org/categories.php?func=gd_show_app&gd_app_id=162)

drag the .gz package of your choice into the gdesklets window, and launch the gdesklet from there.

Check out the others while you're there too
