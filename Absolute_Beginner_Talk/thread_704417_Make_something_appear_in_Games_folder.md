### Post by natirips on 2008-02-22
As for it appearing under games, if you use Ubuntu (with Gnome) just right-click on "Programs" next to that orange ubuntu icon and select to edit the menu and under Programs - Games add a new icon. As for desktop starter you can do it pretty much the same way (right click on your desktop and select to make new starter (I'm not sure how exactly the labels are written in english, because I use localized language settings, but you should be able to manage your way through :D. And generally a hint about using graphical interfaces on computers: whenever you're trying to do something that seems like it could be simple, always try right-clicking around. USUALLY just right-clicking around won't do any harm.

Edit: As for the terminal, I know there is this "alias" command (try "help alias" for syntax), but it seems that is forgets aliases when I restart my computer so.... I'm afraid I can' really help about that part :(
