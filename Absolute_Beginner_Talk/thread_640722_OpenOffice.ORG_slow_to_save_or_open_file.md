### Post by ajmorris on 2007-12-14
Hi,
That is a window focusing error, which i was under the impression had been fixed. And the option turned on by default, however, if you wish to make that dialog box appear on top of windows, open up CCSM (/usr/bin/ccsm), click the 'general options' button then go to the focus and raise tab (along the top), then enable the Auto-Raise option, there are also other things there such as auto-raise delay, etc.
