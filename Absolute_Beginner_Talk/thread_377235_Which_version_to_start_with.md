### Post by 23meg on 2007-03-05
> Should I wait for Feisty Fawn on 19 April or is it best to start with the last version Edgy Eft?

Start with Edgy, and upgrade when Feisty is out. If you have severe problems with Edgy, try Dapper, which is generally known to be more stable. 

> Do the new versions have bugs that need to be fixed?

Every version does; development versions obviously have many critical ones. For full time use, stay away from Feisty for now.

> Is support to new versions not as good as for older versions?

No.

> Can you easily upgrade to a newer version once it is published?

Yes.

### Post by Arisna on 2007-03-05
I'll give you the impression of someone who has been following subjective and technical commentary to some extent about both Edgy and Feisty.

For a lot of people, Edgy has been a great release, but others had problems with it that they did not have in the previous release, Dapper.  I personally was fairly successful, but I ended up downgrading part of my system to a Dapper component due to hardware problems.

On the other hand, I've heard primarily good things about Feisty development.  I hope and believe that most of the major issues seen in Edgy will be corrected in Feisty.  My understanding is that Edgy was meant to be a very "cutting edge" release, hence its name and nature.

Ultimately, my advice would be to wait for Feisty unless you are really raring to go with Ubuntu.  This will save you from working through any difficulties you might have had with Edgy and allow you to start fresh and cleanly with the most current release when it rolls out in April.
