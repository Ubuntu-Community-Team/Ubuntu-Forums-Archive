### Post by mkoehler on 2008-02-25
Hey,

In order to install programs from source, you need to follow the following guide: [https://help.ubuntu.com/community/CompilingSoftware](https://help.ubuntu.com/community/CompilingSoftware).  However, Ardour is available in Synaptic, the ubuntu package manager (this will make the process of installing the software 100x easier).  Just go to System>Administration>Synaptic Package Manager, find "ardour", mark the package for installation, and click apply.
