### Post by brash on 2006-10-24
According to the Cedega website ["we're not quite there just yet"]("http://www.transgaming.com/latest_news.php") for Oblivion. 

The closest they have managed so far is using the "Oldblivion" fan-made custom shaders that allow the game to run on extremely old videocards, that run the game on severely restricted graphic settings. You might be able to do that also on Wine without the $60/year subscription cost of Cedega, I am not sure, but it won't be as "pretty" as the windows version.
