### Post by Zombie process on 2007-02-20
EDIT: See the post below this one for the official method. Disregard this one ;)  


The easiest way I can think of would be to use [Automatix]("http://www.getautomatix.com/wiki/index.php?title=Installation#Easy_Direct_Installation"): 

And there select Multimedia->Multimedia Codecs (And if it is legal in your country, also Multimedia->AUD-DVD Codecs) 

If for some reason you prefer NOT to use automatix, you can use the steps described in the [Unofficial Ubuntu Starter Guide]("http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_install_Multimedia_Codecs") for installing the multimedia codecs

(I am assuming you are using Ubuntu 6.10 "Edgy Eft", however a similar guide exists for 6.06 LTS "Dapper Drake" and older versions of ubuntu)


PS: I don't believe that any current ubuntu release includes support for mp3 audio out of the box. But I might be wrong.
