### Post by AscendedDaniel on 2007-04-30
I am a new user as well, but have had some sucess getting wifi working. I recommend using ndiswrapper. Ndiswrapper uses windows drivers (specifically the .dll files) and is able to let linux use them. I found a howto here: [http://ubuntuforums.org/showthread.php?t=31926]("http://ubuntuforums.org/showthread.php?t=31926")

I hope this helps. By the way, you should post what wifi card you are using, as certain ones are easier to get working than others. If you need more help, feel free to post back.
