### Post by Cousindaddy on 2006-01-27
I was given the following info when I had the same problem:

There are two ways:
1. Easy way: Install Firefox (and a whole set of other programs) easily by using Automatix. The thread for that is stickied here.
2. Longer way: [https://wiki.ubuntu.com/FirefoxNewVersion](https://wiki.ubuntu.com/FirefoxNewVersion)

If you're an absolute beginner, I recommend the first one. If you have a bit of experience, or want to experiment, or want to gain more knowledge about how things are done, the second link is good.

I used the second approach. It worked fine for me.
