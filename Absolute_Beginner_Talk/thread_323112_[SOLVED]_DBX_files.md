### Post by angkor on 2006-12-21
From [here]("http://www.linuxquestions.org/questions/showthread.php?p=2508864"):

kmailcvt can do it. To install:

```
sudo aptitude install kmailcvt
```

Or use Synaptic to install the package.

```
 apt-cache show kmailcvt
Package: kmailcvt
Priority: optional
Section: mail
Installed-Size: 380
[.....]
Description: KDE KMail mail folder converter
 [B]Converts mail folders to KMail format.  Formats supported for import
 include Outlook Express, Evolution, and plain mbox.[/B]
```

I have no experience with it, so I couldn't tell you how it works.
