### Post by intrepidus on 2006-12-14
My friend is in the process of installing Ubuntu, but we've hit a snag. He's got 2 IDE drives, one for the OS, one for stuff. He also has a new 160gb SATA for Ubuntu. So we have XP on one drive, and Ubuntu on the other. But his bios only shows the cd drive and hard drive for the boot sequence, so he can't set the 160 as the master...even though it's set as the SATA master. Is there a way to fix this? Computer is a Dell Dimension 8300.

### Post by nickburns on 2006-12-14

### Post by wpshooter on 2006-12-14
Surely if this motherboard has the capacity of using a SATA drive, there has to be a way to boot to it !!!  Check your BIOS again and make sure there is not some kind of sub-menu for the hard drives that specifies which order they are in the boot sequence.  See if there is a BIOS update for your motherboard.  Or perhaps, if you want to use the SATA, then you must sacrifice by NOT using the IDE drives. If all else fails, call DELL.

P.S. - Have you read your DELL reference manual ?
