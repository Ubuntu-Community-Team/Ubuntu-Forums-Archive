### Post by Cre-ve on 2007-05-21
Just buy another Hard Drive (best if you get an IDE hard drive) and make it a slave. 
They are less than $50 for the most part. Installing them is not that difficult either since I've only done that twice in my life!

Newegg.com has some good/Cheap Hard Drives. I'm using one of their Hatachi drives. 

Then have your windows in the SATA hard drive (which is probably what it is in already). Put in the Ubuntu CD and install Ubuntu into the Entire IDE Hard Drive...takes all of 15 mins to install..LITERALLY! its crazy how well it worked. now i have Windows on the SATA drive and Ubuntu on the IDE drive and the Loader at startup of the computer asks me which one i want to go to..and BOTH WORK!

VERY EASY!

### Post by Cre-ve on 2007-05-21
> **colikath said:**
> [FONT="Comic Sans MS"][SIZE="3"][SIZE="4"]Hello. Not sure what having two hard drives gains? I just dual boot (with Windows XP). But then I don't keep much stuff permanently on the pc.[/SIZE][/SIZE][/FONT]

basically you don't have to worry about all the partitioning problems with having 2 OS on the same hard drive. 

Windows is set up so that it takes up the ENTIRE HD if that is what your computer came with. If you try to partition it, you will be installing Ubuntu over some Windows critical files which will basically cause windows to not start up and you are left with a black screen at start up..causing u to freak out (which is what happened to me!)

Basically you are going to have to reinstall windows (more often than not) if you are trying to share the HD which takes TONS of time and effort.

If you get another HD you don't have to mess with all that and you can take comfort in the fact that anything you saved on your windows HD has not been messed with.
