### Post by Inxsible on 2008-03-30
free is a command to check RAM usage. To check hard disk space(I am assuming you mean hard disk space by saying memory because you said 40Gigs and I still haven't seen a commercial machine with 40 Gigs of RAM) you need to use ```
df -h
```
