What you're missing: You didn't tell us how much memory these machines have. You must have 256 megabytes of RAM to run Ubuntu from the CD; it's unlikely that you have that much in a 400 megahertz G3.

If all your iMacs stop at the same point, it does sound like a faulty CD. You must burn the disc at 8x or slower. Don't burn it faster than 8x - discs cannot be reliably burnt at faster speeds.

If that doesn't help, try the Alternate CD. This is a CD image that just installs Ubuntu when you start it up. You have the "Desktop CD", which runs Ubuntu from the CD. That's right, the progress bar you saw was for starting up the operating system, not installing it. I suggest reading up a bit about the Desktop CD and what it does, so you have a better idea of what to expect.

### Post by irish_flu on 2007-03-17
Hey friend, for starters, when the CD starts up it's not actually beginning to install anything yet.  What it's actually trying to do is run the OS from the CD, at which point you'll have the chance to install if you desire (just like the OS9 CD).

I'm with some other folks that you might try the Ubuntu "alternate CD", especially on computers with less than 256 MB of RAM.  This one just installs; it doesn't try to load up the OS from the CD first.
