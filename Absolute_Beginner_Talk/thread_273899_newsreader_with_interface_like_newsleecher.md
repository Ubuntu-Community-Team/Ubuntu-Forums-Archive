### Post by harry555 on 2006-10-09
Try Pan newsreader.   It is a Gnome program.   Its interface is similar to Forte Agent.  Here is a post I made yesterday.
"hi re. Pan,
Pan is a fabulous news program!
To set it up you need to enter the name of your news server, for example
news.madeupisp.com for your own isp, or news.payserver.com if you subscribe to a pay news server such as tera news, easynews, giganews etc. If you go to [www.easynews.com](www.easynews.com) then 'news server info' there are screen shots of how to set up Forte Agent. The procedure for Pan is very similar. There are free news servers around but they are crap if you want to look for linux.iso
distros and other 'free' stuff.
Here is a link to Pan page. [http://pan.rebelbase.com/faq/](http://pan.rebelbase.com/faq/) There are screenshot and really good tips.

If you use your own isp news server then you won't have to enter a user name and password, but you will need a user and pw if you use a pay server. FWIW most isp news servers are absolute crap with articles expiring in days. If you want to look for newsgroups such as linux.iso etc you need a pay server. Isp news servers are o.k. for just the text discussion groups. Some ISPs don't even have a news server these days.

Usenet is basically about two things. Discussion text groups and binary groups. Binary groups carry stuff such as linux iso's, video content,
pictures, mp3s etc. For example alt.binaries.linux alt.binaries.cd.image.linux alt.binaries.pictures.vehicles
There's over 100000 newsgroups in total.

I'd suggest trying out your own isp news server (if they have one) or going
a google search for a free news server so you can try it out. Don't get put off by all the features. Learning the basics is easy peasy.
Hope that helps.
harry
P.S. A quick google search on Pan setup or usenet faq will give you heaps more info. But let me know how you get on!"

### Post by MrBlond83 on 2006-10-19
Use Pan's beta version.

You can download packages here: [http://pan.rebelbase.com/download/]("http://pan.rebelbase.com/download/")

It has full NZB support and is great and simple client to use.

I have been using Newsleecher in my Windows days - One thing Newsleecher has that many Windows clients don't have, and so far none of the linux clients I tried have, is the search server (called super search there or something). It's a great ability, but you can just use nzb search websites instead, and use Pan to import the NZB and download the files.

I'm sure you'll find Pan simple and easy :) And don't download the one in the repositories no matter what, it's an old old version that stinks. Just download the Beta deb's from their website.
