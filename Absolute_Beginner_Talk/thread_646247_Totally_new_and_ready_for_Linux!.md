### Post by 3rdalbum on 2007-12-21
I'm not entirely surprised that it messed up your hard drive the first time around, if you were installing onto a disk with the Mac OS. The Mac OS seems to be quite intolerant of moving around partitions; I know from experience.

I would suggest installing Ubuntu onto an x86 PC. Most proprietary Linux software (Skype, Realplayer, Windows programs in Wine, VMWare, Gizmo, Flash Player etc) is only compiled for x86 Linux, and won't work on your PowerPC-based Macintosh. And there's a lot less love for Linux on PowerPC platforms than there is for x86. It's a real shame, but you'll have a much better Linux experience on an Intel or AMD box.

Installing software mostly isn't a command-line affair. The Synaptic package manager has enough software for most people, installable through a few mouse clicks. Sites like [www.getdeb.net](www.getdeb.net) also have easily-installable software specifically-built for Ubuntu (once again, x86 only). The latest Firefox 3 beta is also only a double-click away, too, on x86.

It's only when you get to the newest, bleeding edge software that you need to use the command-line for compiling or configuring. And let's face it, there's plenty to do on Linux until you feel confident enough to compile, and it's almost impossible to stuff up your system through compiling. And, once again, x86 users can go for longer without compiling than PowerPC users, as more programs are packaged for x86.

So, there you have it. I think you'd do swell with one of the Dells (hey, that rhymes!). You'll be able to get a 3D desktop cube out-of-the-box too.
