### Post by arvevans on 2007-06-30
In your home folder there should be a hidden directory (hidden filenames start with ".") called .mozilla.  Use CTRL-h in the file browser to see hidden files.  Follow the directory structure into that path and you should find an html file that contains your bookmarks.  You could set up a symbolic link from this to your desktop if you want that to always be readily available.
