### Post by Sunflower1970 on 2007-05-15
When I started with Ubuntu, this forum was the biggest help, but I also picked up a few books: 
Ubuntu for Non-Geeks by Rickford Grant.
Also Ubuntu Unleashed by Paul Hudson and Andrew Hudson

Both of these books are getting second editions by this summer.

Another good book is Ubuntu Hacks by Jonathan Oxer, Kyle Rankin, and Bill Childers
