### Post by meng on 2006-10-23
A full desktop install will not fit on 1 GB. You can, however, copy the live CD to the USB drive and then make some minor modifications to make the whole thing bootable. The procedure is documented on pendrivelinux.com, although Ubuntu is not specifically mentioned, the same procedure should work.
