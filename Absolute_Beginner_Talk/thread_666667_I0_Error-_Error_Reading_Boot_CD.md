### Post by TBerben on 2008-01-13
Lets start off with the way in which you burn cds: you should burn them at the lowest speed possible, to minimize the risk of burning errors. Most burning tools have a 'verify written data' option, use it. Then before you even think about installing Ubuntu from the disk use the "check installation disk" (or whatever it is called) option from the boot menu. That should rule out write errors.
