### Post by YokoZar on 2007-04-22
Wine isn't yet configured to do this automatically upon doubleclicking with steam's new installer file (an .msi file rather than a normal windows executable).  A future version will have a script to fix this (it's on my todo page for the Wine package [here](https://wiki.ubuntu.com/ScottRitchie)).

Anyway, open a terminal, navigate to wherever you downloaded the steam installer, and run:
```
msiexec /i SteamInstall.msi
```
