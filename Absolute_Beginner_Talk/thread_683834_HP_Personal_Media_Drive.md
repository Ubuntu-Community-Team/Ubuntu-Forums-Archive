### Post by LowSky on 2008-01-31
It should work, with the help of google I found out that the PMD is USB powered. SO as long as your computer allows you to boot from USB (check your BIOS) you should be able to. I have an external USB hard drive that has a copy of Xubuntu That I can use if my regular hard drive fails, or I need to fish info off someone elses computer without starting Windows.

First check you BIOS for a setting called Primary Boot drive..and see if USB is an option, you may need the drive installed to pick it

Then insert liveCD and tell it to install to USB drive.

If you can boot to USB then you might need to make a BootDisk from a CD check google for that.
