### Post by Ferri on 2006-04-13
If you use KDE, Konqueror should be in Kmenu -> Internet (I don't know about Endeveaour).
If you want to add a program to Kmenu, right-click on Kmenu -> Menu editor.
One of the icons (second on the left in my case) will let you add a new item wherever you want. Just put the name, and afterwards, on the right, the command (konqueror, for example). Click on the icon on top right to change the icon you'll see in Kmenu (if it's a "normal" KDE program, you'll be able to find an icon with the program's name).
If you want it on the desktop, with a right-click on it you'll have the option to add a link to an app. I think it's quite intuitive, but post again if you get into trouble.
