### Post by Sef on 2008-03-15
> one is that when i use ctrl+alt+delete and when the screen pops up there is no restart or shutdown option. is there a way i can fix this?

ctrl + alt + backspace takes you to a terminal.

Then type this code to shutdown:  ```
sudo shutdown -h now
```

or type this code to reboot: ```
sudo shutdown -r now
```
