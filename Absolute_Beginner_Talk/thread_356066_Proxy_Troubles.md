### Post by argie on 2007-02-09
I had to do this on Dapper; 
First open /etc/apt/apt.conf:
```
gksudo gedit apt.conf
```
It may be empty. Add the following lines to /etc/apt/apt.conf (replacing relevant parts)

```
ACQUIRE {
http::proxy "http://username:password@127.0.0.1:1080/"
}
```
