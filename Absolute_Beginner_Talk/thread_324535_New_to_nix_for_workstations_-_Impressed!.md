### Post by 3rods on 2006-12-23
I'd just like to say that the latest release of ubuntu is very good. 
My opinions were always:

Best Server OS: CentOS/RHEL 
Best Workstation OS: WinXP Pro
Best OS for your mom: OS X (or any Apple flavor)

I just installed ubuntu on a Compaq N610c I found in the trash and it is working very well!

My only issues are minor:

Could use better ACPI support (I understand this is in the works for April release)
Could use Laptop LCD functions support (brightness, etc. Unless I just haven't found it yet)
Could recognize more CPU stepping FRQ's (This CPU has 7, I think it is only using 3 or 4 in Ubuntu and maybe only 2 - 1.2/1.8ghz -Max 1.8) 

That's about it. Everything else is working swimmingly. :KS  - even the W200 WLAN module.

Ubuntu flavored linux is really a very good product, with GREAT documentation. I tried a few other flavors of linux on the Compaq first, but Ubuntu ran the best and had the most h/w support. 

I don't think it beats WinXP for overall compatibility (i.e. IE plug-ins, MP3 out the box, etc), I know, I know, that is the nature of the beast and opens the system up to vulnerabilities and/or costs $$$ to implement, but it still keeps Grandma from buying a computer with and putting Ubuntu on it. 

Over all - very good work! Kudos to everyone who worked/is working on this project.

### Post by IYY on 2006-12-23
Hi, and welcome to the forums! This thread really belongs under the ``Testimonials'' subforum, I think. Anyway, here's my comment:

> compatibility 

This is a funny term. Really, when we say compatibility, we mean `similarity to the standard', which in our case means `similarity to Windows XP'. Of course, Windows XP will *always* be better at being Windows XP than Ubuntu, or any other OS. When it comes to being able to ``communicate'' with other computers, Windows obviously wins, because everyone is running it. However, when judged on the basis of quality, ease of use, security, etc, I believe Ubuntu wins.

> Best OS for your mom: OS X (or any Apple flavor)  

This is true, but you may find that OS X actually is a very powerful system perfectly suited for many professionals.
