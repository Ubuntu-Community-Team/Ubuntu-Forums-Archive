### Post by toobaz on 2008-02-02
Add the following lines to /etc/fstab:

/dev/hda4 /media/nameyouchoose auto defaults 0 1
/dev/hda6 /media/othernameyouchoose auto defaults 0 1

(if /media/nameyouchoose and /media/othernameyouchoose don't  already exist, you must create them)
