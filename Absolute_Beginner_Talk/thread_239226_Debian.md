### Post by Imexius on 2006-08-18
[I]Debian is a free operating system (OS) for your computer. Debian uses the Linux kernel (the core of an operating system), but most of the basic OS tools come from the GNU project; hence the name GNU/Linux.

Debian GNU/Linux provides more than a pure OS: it comes with over 15490 packages, precompiled software bundled up in a nice format for easy installation on your machine. [/I]

For more information visit [http://en.wikipedia.org/wiki/Debian](http://en.wikipedia.org/wiki/Debian)
