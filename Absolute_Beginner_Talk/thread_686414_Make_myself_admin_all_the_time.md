### Post by 3rdalbum on 2008-02-03
> **DaveGiant said:**
> Is it possible to set my account as admin all the time?

Yes. It's also possible to run Windows viruses in Wine, and it's possible to purposely delete the entire contents of /bin; but these three things are seriously bad ideas. If you run your account as administrator, one of the following will happen:

1. A cracker will compromise your computer
2. You will run a command that you wish you hadn't (that borks your system).

Seriously, utterly seriously, reconsider. Typing your password occasionally is so much easier and quicker than reinstalling your operating system.
