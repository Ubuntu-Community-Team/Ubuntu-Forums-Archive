### Post by Luigi239 on 2007-11-24
First of all, KDE should be able to detect and access an external HDD just as well as a Gnome desktop should be able to. If you want to install the KDE desktop as well as the Ubuntu one, run the command:

sudo apt-get kubuntu-desktop

Your applications menu will become cluttered due to all the Kubuntu apps also being installed, but it is an option.

As far as media players, I have had good luck with Banshee, but there is not an equalizer. You may also want to check out Armarok.
