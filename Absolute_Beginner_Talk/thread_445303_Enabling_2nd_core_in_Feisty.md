### Post by Bachstelze on 2007-05-16
The Nvidia driver contains three parts : a kernel module, which is in the linux-restricted-modules package, a X11 module, which is in the nvidia-glx* package as well as some other graphic libraries. The kernel module is the same for all the versions of the driver, only the X11 module and the libs differ, so if you have nvidia-glx-new, you have a 7xx driver, the linux-restrcted-module package has nothing to do with it. If you want to get back to a 96xx driver, you'll have to remove nvidia-glx-new and install nvidia-glx.
