### Post by ubuntu27 on 2008-02-14
> **geebob said:**
> IIs there a Linux program that will allow me to use at least the windows program?

First, there might be (and often there is) an application software for GNU/Linux that better or same quality than the Windows application you were used to use. Search for it. For example, if you are use Photoshop in Windows, try [GIMP]("http://www.gimp.org/"). If you use MSN Messenger or Windows Live Messenger then try [aMSN]("http://www.amsn-project.net/") or [emesene]("http://emesene.org/")

Here is a site that lets you find Linux applications: [Linux App Finder]("http://linuxappfinder.com/")

That's nto the only website you can use to find an alternative to your windows applications.
[http://linuxappfinder.com/](http://linuxappfinder.com/)


If you cannot find an alternative that is native to Linux. Then you may want to try Wine (Wine Is Not an Emulator), it an program that fools windows apps to think that they are in a Windows directory.

Check out Wine's website: [http://www.winehq.org/](http://www.winehq.org/)

Also, check out what programs are "supported": [http://appdb.winehq.org/](http://appdb.winehq.org/)

You maye also want to check [CrossOver Linux]("http://www.codeweavers.com/")
