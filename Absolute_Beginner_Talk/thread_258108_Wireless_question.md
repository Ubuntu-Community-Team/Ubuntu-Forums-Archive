### Post by bswilson on 2006-09-15
Wireless connectivity relies on standard protocols that are fully supported through Ubuntu.  The only thing I can think of that **might** cause you a problem is the Web application that most hotels use to authenticate you (and thus take your money).  

I haven't experienced any problems at all, but there could be some hotel chain that uses some ActiveX control or something...

### Post by Metacarpal on 2006-09-15
You should be able to use any wireless access point that your hardware is compatible with.  For instance, if the hotel as wireless-g (802.11g) you'll need a card compatible with that.

Otherwise, as long as it's an open network or you have the WEP/WPA key, you'll be able to connect.
