### Post by emarkd on 2008-02-14
1.  Did you check restricted drivers at System > Administration > Restricted Drivers

2.  .tar.gz (normally called a tarball) is a compressed file like zip, but you do lots of different things with them.  For instance, to install your theme tarball, just open System > Preferences > Appearance and drag-n-drop that tarball onto it to install it.
