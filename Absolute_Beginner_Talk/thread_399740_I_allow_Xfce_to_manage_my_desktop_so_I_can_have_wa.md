### Post by garybrlow on 2007-04-02
That's exactly it. :p It preserves the current settings, used programs, icon positions etc. on a per logged-in user basis during the duration of his/her session, in other words until the users log off on the default session file. You can always set xfce to save sessions automatically via settings>sessions and startup settings. I usually want to have the option to save it or not depending on the situation, I use the Prompt on Logout Option. It shows a check box to save the session for future logins. Check to save and remove to ignore current session changes/settings.
