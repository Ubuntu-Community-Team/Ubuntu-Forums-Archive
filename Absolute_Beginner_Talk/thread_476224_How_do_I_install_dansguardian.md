### Post by nxMarc on 2007-06-17
Run your terminal
Applications > Accessories > Terminal
Type in
sudo apt-get install dansguardian

On Ubuntu, you can also install your applications via Add/Remove/Synaptic package manager
Applications > Add/Remove
System > Administration > Synaptic Package Manager
Just type in the name of program you want, check it and install it.

Sometimes, it can go a little bit complicated, so, if you never installed anything on Ubuntu, I suggest you reading some tutorial about it.
