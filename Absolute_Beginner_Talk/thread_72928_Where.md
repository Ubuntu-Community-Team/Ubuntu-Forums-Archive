### Post by SilentCacophony on 2005-10-08
Hi. I can field a couple of these.

> Where is the default place to store programs when they are automagically apt-get'ed onto my computer?
The program executable itself will usually go to /usr/bin (or /usr/games for games.) Docs will usually go into /usr/share/doc.

> XMMS skin needs to be put into the 'XMMS skin folder', which is easy enough if I knew where the folder was. >.o
As for customizing apps, you'd usually want to look in your ~/ (home) directory for a folder named after that app. I haven't used XMMS myself, though.

EDIT - Also, those folders in your home directory are usually hidden (i.e. ~/.gnome). You can use **ls -a** to show hidden files in the console, or set the option in preferences>views in nautilus.
