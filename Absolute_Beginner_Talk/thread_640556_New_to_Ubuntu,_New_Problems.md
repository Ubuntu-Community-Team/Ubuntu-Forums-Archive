### Post by siciliancasanova on 2007-12-15
Looks like you need to install the screenlet manager

If you want some eyecandy I might start by doing a

sudo apt-get update

followed by a

sudo apt-get install compizconfig-settings-manager

then go to system > preferences > advanced desktop effects settings
