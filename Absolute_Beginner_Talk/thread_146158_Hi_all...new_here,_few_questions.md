### Post by FizDev on 2006-03-17
[QUOTE=æþeling]
-What programs do I use instead of excel, access and winamp? Also, where can I get them and how do I install them?

-Are there any autocad-like programs for Ubuntu?

-How well does Ubuntu support headphones and USB flash drives? Should they work?[/QUOTE]

For excel, access, word, power point... There is OpenOffice, it comes with Ubuntu. For Winamp, you should try xmms. To install xmms just do :
```
sudo apt-get install xmms
```

As for autocad, try QCAD (haven't tried it, but should do the trick), to install, same thing as xmms
```
sudo apt-get install qcad
```

USB flash drives and headphones should work out of the box.

To replace Excel, you'll see a program called Calc already installed on your system. (Applications->Office menu) It's a part of OpenOffice.

To replace Winamp, I use XMMS. Others use BMP.  THey're both great, and a cinch if you're used to Winamp.  I use XMMS precisely because it's just like my good ol trusty Winamp friend.. right down to the look. And it can use Winamp skins you download!  Classic only, not Modern.

I'm not sure about Access, because I don't know what it is.  Nor am I sure about CAD apps.

[QUOTE=æþeling]-How well does Ubuntu support headphones and USB flash drives? Should they work?[/QUOTE]

Like a charm!  Plug in a USB key, and moments later it's on your desktop, ready for use.  I'm not sure what you mean by headphones.  Headphones and desktop speakers are treated the same by Windows and Ubuntu both, as far as I know. Just plug em in and blast some music!
