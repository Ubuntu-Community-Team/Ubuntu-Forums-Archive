### Post by Dragonbite on 2006-01-31
*( anybody help me if I'm off here)*

KDE uses QT programming language while Gnome uses GTK+. Some programs base based off of one or the other or they can be based off of neither.

> -is there a point to having 2 apps that do the exact same thing but look a little different as well as named differently?Technically you can run almost any program in either desktop environment, providing the correct libraries are installed, even though they may be a bit slower than they would in their base DE. This is one example of the "free"(dom) of Linux... choice.

I've mixed and matched programs though I do prefer using the ones made for my DE, if there is a compelling enough reason then I use what I want.
