### Post by scorp123 on 2007-12-26
> **sovereign_soul said:**
> Is there a utility in linux to extract files from self-extracting archives (SFX )made with Winrar on windows ? If you know what utility was used to create the "SFX" you can unpack it with the appropriate unpacker tool on Linux as if it were a standard archive. e.g. you'd use "unrar" to unpack the file in your example. This should work, I've also done it myself many times e.g. with TrueType Fonts packed into self-extracting *.EXE files ... you just unpack them, voila, you're done. :)
