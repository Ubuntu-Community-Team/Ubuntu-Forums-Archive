### Post by stalkingwolf on 2007-05-26
Yep.  If you use multiple virus scanning tools one will often read the files of the other as virii.  the solution
is simple, turn one off , run your scan, then turn it back on.  Or watch the location of the file.  If you can
identify the file as one from another scanner ignore the warning, unless the extention looks hinky.  I have run across utilities that had malware imbeded in them.
