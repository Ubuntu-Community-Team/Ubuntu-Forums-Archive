### Post by pseudonym on 2007-02-03
What kind of drive is it and is it being detected properly in the BIOS? If it's an IDE drive is the jumper still set correctly and the right part of the cable connected (Master)?

Also, if you want you can dual-boot Windows and Ubuntu on two disks - one OS per disk (it's what I do). A good howto on that can be found [here]("http://www.ubuntuforums.org/showthread.php?t=179902&highlight=dual+boot").
