### Post by Hendrixski on 2007-02-22
This is an important topic for people to understand.  Thanks for bringing it up Steve.  If you go to Wikipedia there is a list of viruses for Linux.  So it is possible for there to be a Linux virus.  However none have successfuly propagated "in the wild"... yet.  There are fewer than windows because the quality of the code is significantly better.  It's "linuses law" that "given enough eyeballs all bugs are shallow", and shallow bugs are easy to fix.  viruses usually work by exploiting bugs, so fewer bugs means fewer viruses.

Also on Wikipedia there is a "Comparison of Windows and Linux" page that has a pretty large security section, it could help you learn more.  After months of wikipedia flame wars the article is pretty neutral, I am beginning to recommend it to people.

Enjoy your stable new operating system  :)
