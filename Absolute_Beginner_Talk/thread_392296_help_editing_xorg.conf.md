### Post by RudolfMDLT on 2007-03-24
Hi!

open the terminal and:

> sudo su

sudo dpkg-reconfigure -phigh xserver-xorg


Make sure to select the fglrx driver and then in the following screen please select the resolutions that you want - note that Xorg will ALWAYS take the highest res.

See if that works...  if it doesn't then please post the output of "fglrxinfo".


Goodluck,

Rudolf
