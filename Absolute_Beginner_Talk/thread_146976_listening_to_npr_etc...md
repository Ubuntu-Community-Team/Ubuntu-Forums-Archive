### Post by Edward The Bonobo on 2006-04-19
I installed RealPlayer and the codecs from Automatix, and it works for me.  I've noticed that the NPR file format is SMIL.  You can read this as a text file and copy the url out from there, if that's any use.  I've also played it via Streamtuner...but you have to configure it to launch RealPlayer for .rm streams anyway.

Presumably there's a similar issue for BBC streams, which are .ram?

btw - I've managed to rip NPR streams and convert to .mp3, using a combination of Vsound/Sox and Lame.  As far as I'm aware, this is perfectly legal, provided it's solely for personal use/ not for commercial gain.
