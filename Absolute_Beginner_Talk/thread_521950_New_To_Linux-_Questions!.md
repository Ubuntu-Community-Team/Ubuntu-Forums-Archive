### Post by p_quarles on 2007-08-09
The Celeron processor might slow you down a bit, but you have enough memory that you should get very decent performance. Your video card will work, but it might take some manual configuration. The only thing I don't know about is your printer. Everything else will work just fine. 

To get MP3s, WMA and other proprietary media formats working, you will need to install the proper codecs. This is easy to do, and you can find the information by going to the online documentation or by asking here. 

Firefox is installed by default. I don't know what Ares is. MS Office doesn't work with Linux, but OpenOffice is a complete replacement and is also installed by default. Limewire is available, but the replacement called Frostwire works better. 

The best thing to do is run the live CD and see how you like it.
