### Post by talsemgeest on 2008-03-14
Ok I just looked up about F@H. It is actually simpler than I thought.

Simply right-click the .exe file, go to the permissions tab, and select "allow executing file as a program".

Double click the file and a whole lot of files will appear in the same folder as the .exe . Then your done!

But I do find it strange that F@H would package the program as an exe, but anyway it works
