### Post by qamelian on 2007-11-08
Give it time. Apps don't just magically appear in the repos. Someone needs to package them. Since minor releases like this tend to be bug fixes or security patches, the speed at which it makes it into the repos will very likely depend on just how important the changes are. If they are not very important, then bumping up the version of Firefox might take a lower priority in favour of patches to other apps.

If you really, truly have need of the latest release, you can always grab it directly from Mozilla and install their vanilla version. You won't be able to upgrade it from the repos afterward though.
