### Post by doobit on 2006-10-05
> **leona said:**
> Thought you where running Win 2000??  One of my laptops have both windows 2000 and Ubuntu, grub will boot both with no trouble.

Grub will dual boot with any Linux and Windows 2000 with no problem. 

One bug that has been persistant in Dapper Drake, all versions, has been in the install sequence. When you get to the point of making partitions there can be all kinds of issues that can even crash the install. The recommendation that worked for me there was to use GParted from the liveCD menu to make my partitions before I did the full install. The alternate install CD would probably have worked better yet because it is lighter on the RAM. The more RAM you have available to run programs, the better. Also, make sure you have an active swap partition that's twice your RAM size. Once I got it installed, though, it's worked great. I was totally happy with XFce, but decided to try Gnome as an experiment.
