### Post by Pragmatist on 2006-03-18
This is from the gnome mail archives:

> On Sat, 7 May 2005, Esben Stien wrote:

Every time I start beast, it has forgotten it's window
position. Talking to the metacity developers, remembering window
position is not a job for the window manager, but for the application. 

So, apparently, the window manager for GNOME (now it is metacity) believes that it is each programs responsibility to handle remembering its window position.

This does not necessarily mean that this is GNOME's policy (even though it uses metacity).  However, my guess is that this is GNOME's policy too.  You can search the archives some more, however.
[http://mail.gnome.org/archives/](http://mail.gnome.org/archives/)

Personally, I use **fluxbox** where it is very easy to control everything such as window position.  You can install fluxbox and use either GNOME or fluxbox interchangeably.  If your interseted in fluxbox, read this:
[https://wiki.ubuntu.com/Fluxbox?highlight=%28fluxbox%29](https://wiki.ubuntu.com/Fluxbox?highlight=%28fluxbox%29)
