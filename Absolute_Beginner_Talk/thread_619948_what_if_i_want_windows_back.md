### Post by aysiu on 2007-11-22
If you [set up a proper dual boot](http://www.psychocats.net/ubuntu/installing) or [installed Ubuntu inside a virtual machine within Windows](http://www.psychocats.net/ubuntu/virtualbox), you could go back to Windows.

But if you choose to do the former, you should know exactly what you're doing *and* have everything backed up. If you're doing the latter, you should have at least 512 MB of RAM, if not 1 GB.

I would start with the live CD. If you have no idea what that means, check out the first few links on this site: [http://www.psychocats.net/ubuntu](http://www.psychocats.net/ubuntu)
