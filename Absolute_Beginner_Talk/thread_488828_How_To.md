### Post by casperlingus on 2007-06-30
Download the CD image to your harddrive.  It will be an ISO image.

Open the software you use for burning CDs.  You want to burn the ISO file as an image.
Let me reiterate:  **Look for the option that says "Burn CD Image"**  or something like that.  **Copying the raw iso file onto the CD will not do what you want.**

Now, when you reboot with the CD in the drive it will load "Live Ubuntu" which is just Ubuntu running from the CD (it won't touch your harddrive, yet).  At any time you can put the Ubuntu CD and boot off it to get to the Live version of Ubuntu, but you won't be able to save any settings or files.

From the live boot, you can click "Install" and then you can go through the process of putting Ubuntu permanently on your computer.  It should walk you through the process for installing it next to windows so you can select which one you want to run when you boot.  Once you have installed it, you will no longer need the CD.

If you make it that far.  Be sure to bookmark  [www.ubuntuguide.com](www.ubuntuguide.com) -- that has a LOT of useful information for new users.  Also, be sure to visit [www.getautomatix.com](www.getautomatix.com) and go through the directions to get and run Automatix, extremely useful programming.

Enjoy!
-Casper
