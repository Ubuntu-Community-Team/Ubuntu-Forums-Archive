### Post by qamelian on 2007-10-15
OEM mode is not for typical end-users. OEM mode is for use by computer resellers who want to pre-install Ubuntu but will not know who the end-user will be and, therrefore, cannot set up an appropriate user account. If you are build 20 PCs to sell with Ubuntu installed, use OEM mode. Otherwise, don't.
