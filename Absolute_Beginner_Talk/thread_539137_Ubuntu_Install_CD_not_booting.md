### Post by Jimmyfj on 2007-08-30
It seems that something went wrong either during the burn process or you have a faulty download. If you are not sure how to read the MD5SUMf, you might want to compare the size of the file you downloaded with the amount of bytes/megabytes listed on the site from which you got the file.

Best thing is to go to [http://www.ubuntu.com](http://www.ubuntu.com) and get the file from there. Here you can compare the file size with the size shown for download size. The iso is about 694 MB big. Nothing assuring compared to MD5SUM but still an indicator that'll tell you if your downloaded file differs from the official file size. Comparing those two sizes have never proved wrong for me and all my iso files from the official site works fine.
