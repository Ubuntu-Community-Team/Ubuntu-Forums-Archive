### Post by khgiese on 2007-12-26
[http://www.ubuntu.com/products/whatisubuntu/desktopedition](http://www.ubuntu.com/products/whatisubuntu/desktopedition)
At the bottom of the page there is a brief system requirement section.

System Requirements

Ubuntu is available for PC, 64-Bit and Mac architectures. At least 256 MB of RAM is required to run the desktop install CD. Install requires at least 4 GB of disk space.

That is for Ubuntu, I would guess the others are similar since they are built on the same kernel.

### Post by linuxlizard on 2008-07-11
Try pcfluxboxos. It flies on my p3 700mhz 192 mb ram laptop. And you can use synaptic to install all the latest software, not old and crippled software like dsl or puppy. 

If you find fluxbox too confusing at first, try zenwalk linux instead. It is also very fast (not quite as fast as pcfluxboxos but is still very comfortable on older hardware), and it uses xfce rather than fluxbox, which is a little more full featured and intuitive. The downside is that it has far less software available in an easily installed way than pcfluxboxos.

Puppy can be fun in spite of it's weaknesses (I'm actually fooling around in puppy right now), DSL IMO isn't worth the effort. To get puppy to run on your hardware you will likely need to create a swap partition on your hard drive (256 mb should be about right) and it would probably be best if you do a true hard drive install rather than trying to run it live or frugal. Plenty of people would probably be happy to help you on puppy's forums. The downside of puppy is the lack of latest software and lack of variety of software (dozens compared to thousands of packages in pcfluxboxos or ubuntu).,

Anything ubuntu or ubuntu spinoff is likely to be too heavy. Which is a pity.
