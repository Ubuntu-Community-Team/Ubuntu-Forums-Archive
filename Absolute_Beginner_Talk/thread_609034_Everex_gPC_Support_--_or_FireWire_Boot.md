### Post by mikeserv on 2007-11-10
According to [this]("http://www.desktoplinux.com/news/NS7114714037.html"), the gPC comes with Ubuntu preinstalled.

And there shouldn't be any reason for Ubuntu not to be able to boot from anything - so long as your BIOS can recognize the device as a bootable drive.

-Mike

Edit: Actually the link says the machine runs "gOS" which is their own distro based on Ubuntu.  And yeah, if you want to wipe a harddrive in the Ubuntu installer, you shouldn't have much difficulty.

### Post by mikeserv on 2007-11-11
Well, memory is easily upgradeable.  But did you know that the gPC has a Via motherboard and chipset?  The processor isn't very powerful, but they say that Via puts out more mHz per watt than any other.  Also, from what I understand, the gPC is really a mini-atx motherboard, but they put it in a large case because "Walmart customers will equate size with quality."

When I first read about the gPC I considered buying it and, with a few mods, making it into a tidy little HDTV recorder/media center.  It would make a good base.  I've also read that Everex is encouraging modifications, they want to know what you can do with it, so maybe they can do the same thing later on down the line.

-Mike
