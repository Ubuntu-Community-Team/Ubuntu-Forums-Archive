### Post by petersjm on 2006-10-21
Open a Terminal window (Applications > Accessories > Terminal) and type **locate parallels** or whatever the program is called. This lists all the places "parallels" is found on your system. Many apps don't automatically add an icon to your menu list, but if you have "Debian Menu" installed, from Automatix, you might have better luck finding it.
