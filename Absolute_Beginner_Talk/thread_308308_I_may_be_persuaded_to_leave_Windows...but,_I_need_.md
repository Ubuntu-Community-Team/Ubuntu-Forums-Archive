OK I'll bite...

First, I would advise you download and run the live CD's and see for yourself...

[64 bit](http://ftp.ussg.iu.edu/linux/ubuntu-releases/6.06/ubuntu-6.06.1-desktop-amd64.iso)

[32 bit](ttp://ftp.ussg.iu.edu/linux/ubuntu-releases/6.06/ubuntu-6.06.1-desktop-i386.iso)

Second, IMO, based on your post, I would advise you start with 6.06 32 bit. This will minimize "tinkering".  As you gain experience you can migrate to edgy or 64 bit.....

Third I see no reason you will not be able to use your video card and monitor, although your video card is a little cutting edge....

You may need to install the nvidia drivers. I did as the open source drivers did not work well with my nvidia 7600...

### Post by Cynical on 2006-11-27
1. Going 64-bit is very easy nowadays. There is a 32-bit binary in the repos you can use for firefox and the latest ffmpeg has native support for wmv files. (the 64-bit forum has info on getting everything working) Edgy had this ugly gray usplash when it first released I think thats been fixed now but I haven't checked.

2. Upgrading to edgy eft worked flawlessly for me. Some users had issues upgrading with automatix installed, I don't use it so I'm not sure if its been fixed yet. 

3. lol yes there is support for that resolution.

4. Yes, I'd suggest getting the latest drivers from nvidia's website though.
