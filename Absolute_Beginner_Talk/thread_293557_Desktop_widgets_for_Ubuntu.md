### Post by Super King on 2006-11-05
The most popular set of desktop widgets is called 'gdesklets', which had various weird clocks and weather widgets, and some other cool things. You can install it in Synaptic or type "sudo apt-get install gdesklets" in the terminal. Then I believe it shows up in your Accessory menu under Applications.
