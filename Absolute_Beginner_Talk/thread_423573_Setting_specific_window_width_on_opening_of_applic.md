### Post by guardsman85 on 2007-04-26
When I open a new OO.o document it just opens to the exact same size and position as the last document I had open.  Then when I open a second document of the same type it creates it in the same size window, but snaps it to the upper left corner.  So I suppose the  you could open a document and size the window to half the width (or height) of the screen and snap it to the right (or bottom) edge of the screen.  Then when you open a second document it should take up the other half of the screen.  Sorry if that's not quite the answer you're seeking. :)
