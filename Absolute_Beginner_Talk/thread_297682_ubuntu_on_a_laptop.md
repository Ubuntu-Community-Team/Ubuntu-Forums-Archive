### Post by elcasey on 2006-11-11
My Toshiba Satellite M45-S269 runs Ubuntu without a single problem. I would think most Toshibas would be able to run Ubuntu well, IBM Thinkpads come pre-loaded with Linux from Lenovo and several other firms specialize in Linux laptops as well. 

It's highly likely that *some* distro of Linux will run on your new laptop, although it may not be Ubuntu. You shouldn't have any problems. If you're really apprehensive, take a LiveCD with you to some computer stores (CompUSA, Best Buy, etc) and ask them to let you boot it.

I couldn't get my wi-fi to work when I ran the LiveCD on my iBook G4, but it runs without a hitch on my Toshiba. I imagine it'll run just fine on the new Macbooks as well.

Anyway...it's not like the old days where you need a very particular hardware setup to run Linux.

### Post by Alveric on 2006-11-11
I'm using an Acer Travelmate 4101 (P-M 1.6GHz) and it worked fine with Dapper out of the box (even the wi-fi).

I added fglrx (?) to run the ATI graphics card (X600).

There are a couple of small glitches, related to special hardware bits'n'bobs:

The big button on the front to turn on/off wi-fi lit up under Windows XP (flashing when finding network, solid when connected) - it now doesn't light up at all, but still turns the wi-fi on/off;

The little hardware buttons (which I never used anyway) to fire up browser/email etc (top of the keyboard, for those Acer fans out there) don't work any more, but I think they were tied into the proprietary stuff Acer had bundled with Windows XP.

The laptop boots up faster under Dapper than under Windows XP, and is more responsive (loads up apps much faster).

I probably won't move up to Edgy, though, as my skill-set needs upgrading before I upgrade my OS ;) 

Cheers!
