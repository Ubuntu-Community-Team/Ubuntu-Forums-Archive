> If I use ClamAV (don't know any other Linux AV) does that update definition files as in say F-Secure.
yes you can update defs in ClamAV as you do in windows. And for your information you can also use F-Prot antivirus as shown [here ]("http://www.ubuntuforums.org/showthread.php?t=88357")

I really do not know if there are any applications that do both.. but if you really want to have a firewall.I suggest Firestarter. You can install it by using the following command in the terminal:
sudo apt-get install firestarter
