Good afternoon everyone!
    As an FYI for anyone that has this problem in the future, after installing Win2k Professional and can't get a true fullscreen, you'll have to change to resolution within the virtual machine to at least 800 x 600. When i had it set to 640 X 480 and went to fullscreen, the screen was split between the vm screen and the lower half of my ubuntu screen. 
    After changing the resolution up to 800 X 600, it is now in full screen. One step closer to going Ubuntu fulltime.
