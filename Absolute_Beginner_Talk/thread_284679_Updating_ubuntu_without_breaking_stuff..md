### Post by nickburns on 2006-10-26
It is possible to uncheck the kernel updates when you run the updates. When you get the form to run the updates, you'll just need to find them in the list of updates your downloading. I don't think it is recommended, it usually is the best practice to stay current.  I would suggest downloading the latest iso file so you don't need to run too many updates for your next install.
