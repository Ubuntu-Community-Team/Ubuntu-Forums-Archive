### Post by -MoonDoggie- on 2007-12-14
if ubuntu was the only thing on your slave drive, then i guess its ok to just: 

1. load up the live cd you installed ubuntu with,

2. load up gparted from the administrations menu

3. look for your slave drive that contains the ubuntu distro that you want to delete.

4. wipe that sucker!

important note: does this ONLY if ubuntu is the only thing (or OS) on that slave drive. if you're dual-booting in that slave drive then dont do the above as it will brake your grub.
