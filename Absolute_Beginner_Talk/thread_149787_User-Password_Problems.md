### Post by mcmillan on 2006-03-24
You should have had a user created when you installed. If that's not working and you haven't done much configuration yet the easiest solution would be to just reinstall. I'm not sure about with VMware, but with the normal install it has an option to boot into a rescue mode. If you can get into this then you can add a user from there.
