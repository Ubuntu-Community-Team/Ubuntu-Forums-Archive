### Post by Paerez on 2007-01-19
find this line in your /etc/X11/xorg.conf:
```
Option "HorizScrollDelta" "0"
```
and change the 0 to a 1.

To edit the file, press ALT+F2 to open the run dialog and type in:
```
gksudo "gedit /etc/X11/xorg.conf"
```
