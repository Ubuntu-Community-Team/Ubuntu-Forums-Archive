### Post by Rocket2DMn on 2008-02-07
Are you running compiz?  That is a metacity theme you downloaded so it probably won't work with compiz.
```
metacity --replace
``` will turn off all the fancy graphics stuff with the cube and whatnot.  To get back into compiz you would run
```
compiz --replace
```
