### Post by nhandler on 2008-01-12
That's usually caused automatically. For example, if you type a colon : followed by a parentheses ), you get a smiley. When making a post, you can scroll down to Additional Options->Miscellaneous Options->Disable Smilies in Text and check the box. That will stop the smilies from appearing.
