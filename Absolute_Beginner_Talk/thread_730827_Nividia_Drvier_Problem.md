### Post by arochester on 2008-03-21
An easy way to install either an Nvidia or ATI driver is to use Albert Milone's script called "Envy" which you can get  from [http://www.albertomilone.com/nvidia_scripts1.html](http://www.albertomilone.com/nvidia_scripts1.html)

It will try to check out your video card, get the correct driver and install it.

It works well for MANY people but not ALL.
