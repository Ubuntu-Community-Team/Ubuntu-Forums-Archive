Hi there

You need to install openssh-server (so you have for sure access with CLI).
After that I recommend using NX (from nomachines) which is in my opinion a superior way to access a linux box with GUI.

NX is for free (as long as only one or two people are actually using it*). Just check their webpage.

* it means...NX, not the linux box itself
