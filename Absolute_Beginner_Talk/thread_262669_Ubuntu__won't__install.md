### Post by GoneWithTheWind on 2006-09-22
The regular install kept freezing and didn't work for me.

Then I used the gparted and alternate install live CD's, and everything went quickly and smoothly after that.

Try the (1) gparted and (2) alternate install live CD's.

Also I'd recommend running "defrag" on windows about 20 times in a row before you start the partitioner and installation of Ubuntu.
