### Post by Najand on 2007-06-08
Ok, then copy all gnome setting folders to another place and login again. It will change all settings back to default... For that, push Ctrl+alt+f1 and use the following commands:
```

$ cd ~
$ mkdir gnome-backup
$ mv ~/gnome* gnome-backup
$ mv .gconf gnome-backup

```
Then go back to X by pushing Ctrl+Alt+F7 and then restart it by Ctrl+Alt+BackSpace and then login again.
