### Post by Sparkster185 on 2007-05-22
The community here is quite helpful, don't hesitate to ask questions.

I installed my graphics card drivers with Envy.  If you have an ATI or NVidia driver, I would recommend using that.  It set up my dual-monitors perfectly fine.

As for Adobe products, only Flash is officially supported for Linux (AFAIK).  There is a pretty great alternative to Photoshop called GIMP.  There's even a plugin that emulates the interface of Photoshop.  For the more advanced users, and a bunch of the Adobe plugins, GIMP will not suffice.

You could try installing/running Photoshop via WINE.  I read on Crossover that Photoshop got a Silver medal (meaning it works for the most part).
