Do you mean as a LiveCD or as a Hard Drive install?

Knoppix is mainly used as a LiveCD, especially to rescue systems.  Ubuntu comes on a LiveCD (at least, that's one option), but is mainly designed to exist as an install - which sounds like what you want.

As for your more detailed questions re viruses, spyware etc, I'm no expert on such things - but they're not crucial for Linux as they are for Windows.

> **kieranmullen said:**

which has been out longer? what are the main differences between each?
Both are based on Debian and use the more recent packages from Debian.  Knoppix has been around for four years and Ubuntu, two.

The main difference is that Knoppix is a live cd only.  You can install in onto a hard drive, but that is mostly experimental.

Ubuntu is a full-fledged distro with proper support.
