### Post by pbaehr on 2006-03-10
I am new to linux and initially installed the 64 bit version of Ubuntu and had some problems. Granted, now that I've learned a bit more I could probably have solved them, but being brand new to linux I decided to downgrade to the 32 bit version which seems better supported.

I didn't notice any difference in performance and everything I've read seems to agree it's a minor difference. I'm sure there's a difference, but it doesn't seem especially noticeable to the naked eye...or naked user (ew)...or whatever expression makes the most sense here.

After changing I did find I had an easier time installing compatible software, though, it's always bugged me that I'm not taking full advantage of my processor's capabilities so I may try the 64 bit version when I update to Dapper in a few weeks. But I've been happy with 32 bit...so we'll see.
