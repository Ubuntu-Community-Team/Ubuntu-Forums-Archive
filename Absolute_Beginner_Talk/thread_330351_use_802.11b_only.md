I just want to update everyone that I figured it out thanks to bernt.

For my specific wireless driver it has different commands than on that link that bernt gave but I figured it out by doing some research.

First type in
```
sudo iwpriv
```

Locate your wireless connection (eth1 for me)
Note how to change the mode ( for me it was set_mode )

Then type in:
```
sudo iwpriv eth1 set_mode 2 
```

Where the mode is a number in the range 1-7:
[[[ these numbers may be different for your driver ]]]

	1	802.11a
	2	802.11b
	3	802.11ab
	4	802.11g         
	5	802.11ag
	6	802.11bg
	7	802.11abg

You can check it afterward by typing
```
sudo iwpriv eth1 get_mode
``` or your appropriate command
