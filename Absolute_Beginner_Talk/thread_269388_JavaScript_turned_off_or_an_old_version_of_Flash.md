### Post by meng on 2006-10-01
You shouldn't need to install Javascript. Are you sure that you aren't getting confused with Java, which is completely separate, although the names are similar?

Whichever browser you are using should have an option for enabling/disabling Javascript. For example, in Firefox: Edit > Preferences > Content.

However, I wonder if this is not your problem, but that Flash wasn't installed in such a way that your browser is aware it is there. Can you explain how you installed Flash?
