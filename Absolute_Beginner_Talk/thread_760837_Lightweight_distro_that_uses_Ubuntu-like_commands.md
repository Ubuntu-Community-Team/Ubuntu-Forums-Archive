### Post by y-lee on 2008-04-20
Damn Small Linux or Puppy Linux are both highly regraded light weight linux Distros.

> Also, in order to train myself to be able to use ubuntu (I've never used terminal or command prompt before) I wanna train myself in VMware... lol

So what lightweight distro will use same commands as ubuntu in terminal? Like sudo stuff...

But if all ya want to do is play with the terminal it is possible to have a Linux like terminal in Windows, that way you can learn  and use Linux/Unix commands from within windows much simpler than installing Linux in a Virtual machine. See [Cygwin]("http://en.wikipedia.org/wiki/Cygwin").

The sudo command would not be available but many if not most of the traditional Linux/unix stuff is there or can be installed. :)
