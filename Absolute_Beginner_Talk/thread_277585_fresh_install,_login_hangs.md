### Post by phossal on 2006-10-14
Although the root cause of the problem may be possible to diagnose, it's unlikely that you'll find a solution in the time it will take you to reinstall Ubuntu. A lot of us have AMD64 boxes that, after minor tweaking, run very well. My first install led to a situation where the OS "restart" option would shut down, and then hang on the restart. If I simply powered down, and then restarted manually I would have the same problem. In short, my computer would only start every other time. I just reinstalled it. It has been running (I'm sure I'll regret this) problem-free for months now.
