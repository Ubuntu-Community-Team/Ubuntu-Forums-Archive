### Post by jordanmthomas on 2006-10-29
When I run fluxbox, I either
a) run gnome-settings-daemon at startup with fluxbox (makes gtk look nice and if you have a key set to raise volume, sets it as well
or
b)  use alsamixer in an xterm

There are also probably apps (like kmix) that you could run when you want to change the volume.
