### Post by kylor on 2007-06-03
Okay, here's what you do.

Right click a music file, select properties. Go to the "Open With" tab. Since you tried opening it with Amarok before, Amarok should be one of the programs listed there. Select the radio button next to Amarok, and try again. If Amarok isn't there, just add it and then select the radio button for it.
