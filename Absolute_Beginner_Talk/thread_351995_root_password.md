> **rannasjem said:**
> Is there a way to recover my root password.  I am 99% sure I am typing the right password in the terminal but it keeps coming back as failed.

How did you set the password for root? Ubuntu uses sudo:

[https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)

If you are using sudo you should use your log in password.

To set a password for root:

```
sudo passwd
```
