### Post by Amenemhet on 2007-04-25
Seisen is correct. You first must edit your /etc/apt/sources.list file and change all 'dapper' to 'edgy'
Then command line >
#apt-get update
#apt-get dist-upgrade
#apt-get upgrade
#reboot
