### Post by ubuntu27 on 2008-02-07
> **blasteryui said:**
> Okay two more questions haha, what does it mean by drag left mouse?
Second Question I tried installed AMSN and I get to the last part where it's installing it and it says... 
Checking for required C library versions ... OK
Checking for X ... OK
**Checking for TCL Scripting Language ... failed**

So, it looks like you are trying to install amsn from an executable obtained in some website, probably Official ALvaro's Messenger's site.

AMSN, is in the repository, meaning you are able to download and install it automatically by using the following command in the terminal [Terminal can be found at Application menu/Accessories/Terminal]

```
sudo apt-get install amsn
```

"sudo" means SuperUser, when you conjure this command, it tells the terminal that you are about to work as an "Adminsitrator". 
"apt" is an program that download and install programs
"install" is a command that tells apt to install something


The above command will install Amsn that it is in the repository. 
There must be a reason why you want to install it manually, perhaps there is a new version that it is not in the repository. 

The intaller for amsn needs a tcl script language. You have to install it. 

```
sudo apt-get install tcl8.4 tcltls
```

And try to install again.


Oh! Check if the package you want to install is the same version that it is in the repositories. If it is, then it is better to install from the repositories.
