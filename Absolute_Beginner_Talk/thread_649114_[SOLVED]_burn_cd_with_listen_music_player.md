### Post by Paulmd on 2007-12-24
> **<<joe>> said:**
> how do i burn a cd with listen music player

You mean copy a cd and listen to the same cd while you're copying?

The tools for doing this don't exist under linux yet. Later versions of Windows media player can do this, with a penalty in ripping speed.

If you mean burn a cd and listen to other music while it's burning, that is doable. For example you can run k3b and xmms smultaneously. Provided you have a system with enough horsepower to spare. (High p3/low p4 at least)
