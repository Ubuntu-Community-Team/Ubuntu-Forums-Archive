### Post by DoorGunner on 2006-11-23
IM not sure what your skill level is but forgive me if i seem to over simplify this> :) 

You can divide your hard drive into sections. This is refered to as a Partition. So instead of having just a C drive, you can have any number of letters along with C.

In order to be able to read any of these drives. You need to mount it. This is a software hack similar to if you were to actually put a physical hard drive on your machine. This is so you can read what is on each partition.

I have two physical hard drives and which contain 4 partitions. Ubuntu is on one of these partiions. So i need to be able to mount the other 3. One has windows, one has a share drive and my last one i use to test out other linux variants. 

Has this been help full? 

for more info try to look up info on fstab. That is were you tell Linux what you want mounted.
