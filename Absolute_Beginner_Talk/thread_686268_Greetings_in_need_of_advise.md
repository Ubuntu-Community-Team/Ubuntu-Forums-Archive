### Post by finer recliner on 2008-02-03
ubuntu's install disc is a live disc.

this means you can boot from it, and will run ubuntu from memory, without touching your current hard drive set up. 

this gives users who arent sure how ubuntu will work on their computer a chance to try it out before installing.

so boot from the disk, play around with it, see if your networking, sound, video, etc works well. and if you want to install it, just click the link on the desktop.

please note that running an operating system from a CD/DVD is slow. this is normal. it will be faster after a real installation to your hard drive.

enjoy
