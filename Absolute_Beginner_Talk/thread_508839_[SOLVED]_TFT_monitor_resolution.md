### Post by Jellyffs on 2007-07-24
ok. w4ett tip is a good one. Try what he suggests _first_.

If it doesn't work then replace _this part_:

```
     SubSection "Display"
        Depth        24
        Modes        "1024x768" "800x600" "640x480"
    EndSubSection
EndSection

```_By this_:

```
     SubSection "Display"
        Depth        24
        Modes        "1440x900" "1024x768" "800x600" "640x480"
    EndSubSection
EndSection
```Here, as you might know, you can had any other resolution that your screen support. But be careful, if you put a resolution that your screen doesn't natively support, you might damage it.

EDIT: you'll need to be root in order to modify this file: sudo /etc/X11/xorg.conf  // Then reboot your PC.

A good way to edit xorg.conf

Alt+F2

then add this ( An edit box will appear )
[HTML]
gksudo gedit /etc/X11/xorg.conf[/HTML]

in top text box 

You'll be able to edit xorg.conf with root privileges


These are my settings to get 1440x900 @ 60Hz


[HTML]
Section "Monitor"
	Identifier	"Philips 190C"
	Horizsync	33.0	-	83.0
	Vertrefresh	55.0	-	75.0
	Option		"DPMS"
EndSection

Section "Monitor"
	Identifier	"aticonfig-Monitor[0]"
	Option		"VendorName"	"ATI Proprietary Driver"
	Option		"ModelName"	"Generic Autodetecting Monitor"
	Option		"DPMS"	"true"
EndSection

Section "Device"
	Identifier	"Generic Video Card"
	Driver		"fglrx"
	Videoram	524288
	Busid		"PCI:1:0:0"
EndSection

Section "Device"
	Identifier	"aticonfig-Device[0]"
	Driver		"fglrx"
	Option		"VideoOverlay"	"on"
	Option		"OpenGLOverlay"	"off"
EndSection

Section "Screen"
	Identifier	"Default Screen"
