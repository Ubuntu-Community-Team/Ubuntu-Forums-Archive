### Post by EXCiD3 on 2008-01-20
The package you are trying to install requires other packages to be installed first. You can easily fix this problem by opening up terminal and typing ```
sudo apt-get install -f
```

What this does is it installs all those dependencies that you do not have installed. Then after that you can try installing the .deb package again using GDebi, KPackage Manager, or use "dpkg -i <.deb>" from terminal.

Awesome is (quote from synaptic)

dynamic and tiling window manager 
A tiling window manager initially based on dwm code.
It's extremely fast, small, dynamic and awesome.

Windows can be managed in several layouts: tiled and floating. Each layout
can be applied on the fly,  optimizing the environment for the application
in use and the task performed.

Managing windows in tiled mode assures that no space will be waste
on your screen.  No gaps, no overlap.

awesome has a good support for multi-screen (Xinerama) support.
