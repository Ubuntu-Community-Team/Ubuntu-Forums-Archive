### Post by lswest on 2008-02-12
first, make sure the extra repositories are enabled under system-->administration-->software sources.  Check to make sure everything except CD-ROM and sources are checked, then open synaptic package manager (system-->administration-->synaptic package manager) and go to "search" and type in cheese, hit enter, and it should be fine.  otherwise just run ```
sudo apt-get install cheese
``` in a terminal
