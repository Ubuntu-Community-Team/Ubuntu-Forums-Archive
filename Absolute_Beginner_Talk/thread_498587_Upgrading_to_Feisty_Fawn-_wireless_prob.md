### Post by Keen101 on 2007-07-15
When I upgraded, I lost wireless too. I found that Network Manager was the problem for me.

I just went to >System >Preferences >Sessions and unchecked network manager then logged out hen back in, and then set it manually with network admin on >System >Administration >Network.


  It worked fine after that. And after upgrades and stuff I have tried network manager again and it seems to work fine now. don't know if it was a bug or what. Just telling you what worked for me. 

There have been a lot of people who have had problems with wireless after upgrading, and it usually turns out to be some problem with network manager.

hope that helps,
