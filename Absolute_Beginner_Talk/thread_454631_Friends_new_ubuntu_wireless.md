### Post by owise1 on 2007-05-25
Have you tried the Ubuntu live CD?  Did it find your network card.  I did a quick goole of that caard and it is reported as working.  You can also have a look at [http://www.hpl.hp.com/personal/Jean_Tourrilhes/Linux/](http://www.hpl.hp.com/personal/Jean_Tourrilhes/Linux/)

If the card is not detected you can try using ndiswrapper - a program that lets you use the windoze drivers for the card.  See [https://help.ubuntu.com/community/WifiDocs/Driver/Ndiswrapper](https://help.ubuntu.com/community/WifiDocs/Driver/Ndiswrapper)
