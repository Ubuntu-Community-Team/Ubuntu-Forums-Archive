### Post by Rocket2DMn on 2007-08-06
You probably need to change your BIOS boot order.  When you restart the computer, you will have a second or two (but no more) to boot into the BIOS by pressing the correct key.  Once inside you need to change your boot order so that the cd drive is ahead of the hard drive.
It should tell you which key you need to press to access the BIOS (or Setup) - common keys are F2, F8, F12, and DEL.
Once you get the boot order right, save and exit the BIOS and restart the computer.  You should then have the option to boot from the CD (or it will boot from it automatically).
