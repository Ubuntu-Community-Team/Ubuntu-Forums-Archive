### Post by zephyrcat on 2008-02-28
1 - Go to System -> Prefrences -> Screensaver. Scroll down the list at the left of the window and choose "Pictures folder."

2 - Assuming this is just a problem of the system using the speakers instead of the headphones and not a problem of the headphone not working for some reason, go to System -> Prefrences -> Sound and try changing the setting for sound playback.
