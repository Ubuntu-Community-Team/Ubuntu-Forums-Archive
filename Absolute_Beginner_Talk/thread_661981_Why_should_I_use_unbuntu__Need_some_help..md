### Post by sub2007 on 2008-01-08
Somehow I think if you are only wanting to try Ubuntu because you don't like WIndows then you might be dissapointed although it is what pushed many people to try Linux. You just have to be aware that it's not Windows, it works differently to Windows and there is a degree of relearning that you'll have to do (just like you had to learn Windows once). There is different software (on the whole) for Linux. If you are willing to embrace the change then the people at the forums will be more than willing to help you learn. But you DO have to bear in mind that it will be different from Windows. Once you do learn it a bit better then you'll find that you have a stable OS which works how you want it to.

Gaming is probably the only area that won't satisfy your needs, everything else works fine on Ubuntu. I would definitely second the dual boot suggestion so that you haven't lost Windows if you decide that Ubuntu isn't for you. But it's definitely worth giving it a try, you might end up loving it.
