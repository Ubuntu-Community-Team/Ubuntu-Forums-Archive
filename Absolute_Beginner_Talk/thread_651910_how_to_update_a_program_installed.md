### Post by stijngysemans on 2007-12-28
Ubuntu keeps your system up to date with the lasted security patches but ubuntu will not upgrade to the last version of an application. 

What i did for F-spot for example: I searched a deb file on [www.getdeb.net](www.getdeb.net) and I installed the last version of this application because i liked it very much.

I though find it strange that aMSN alerts about an update, it shouldn't do this I think: I like my applications not alerting every time they start up about a new update like in windows!
