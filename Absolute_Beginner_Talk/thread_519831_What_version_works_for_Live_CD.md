### Post by Rocket2DMn on 2007-08-07
7.04 has a LiveCD.  You don't need anything special, just a cd burner and a cd-r.
What you SHOULD do is check the md5sum before you burn:
[HowTo]("https://help.ubuntu.com/community/HowToMD5SUM")
[Hashes]("https://help.ubuntu.com/community/UbuntuHashes") to check against
Then when that checks out, burn at a slow speed, like 4x, to help prevent write errors.  Use quality media if you can.
