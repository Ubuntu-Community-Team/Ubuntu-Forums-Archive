### Post by esoterica on 2007-05-16
I had to take (suffer through) an entire semester of Java programing to get my web development degree, I can tell you first hand Java sucks. It's extremely resource intensive on your local computer or on a server, the latest release is also extremely buggy in Windows Vista and in Linux. I tried using the JDK 6 and had so many problems with it that I removed it and resorted back to an older version 5 of their JDK, you may want to consider doing the same thing with the JRE.

Edit:
I didn't have any problems with the new compiler in the JDK, the problems were with the JRE included in the JDK.
