### Post by xmastree on 2005-10-28
Seemingly random?
Sounds like you're downloading source code, which needs to be compiled and installed. Not the easiest thing in the world for a new user.
Best to stick to what's in the repositories for now. Enable the extras and use synaptic, unless there's something specific you need which isn't available there. In which case, look for a .deb file. Failing that you can use a .rpm and convert it with alien.
see [this]("http://www.ubuntuguide.org/#extrarepositories") for more information about repositories.
