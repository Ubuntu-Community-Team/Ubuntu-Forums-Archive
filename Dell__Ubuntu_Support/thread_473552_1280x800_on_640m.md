### Post by angryfirelord on 2007-06-14
You're using another driver that doesn't support widescreen resolutions. Here's how to (possibly) fix it:

First, open up Konsole and type:
```
sudo apt-get install xserver-xorg-video-intel
```
Now, run:
```
sudo dpkg-reconfigure xserver-xorg
```
Select **intel** as your driver. Accept all defaults until you get to the resolutions (near the end). Check off the ones that you want (1280x800 should be there. Don't select any that are not supported by your monitor. I'd say for suggestion 1280x800, 1024x768, 800x600, and 640x480). Next, select Medium under the best resolutions and pick the one you want to always use (that would be 1280x800 at some frequency). Once you're done, log out and log back in.

If you notice that your desktop is stretched out, then you picked the wrong resolution. If you don't know, search for the monitor's native resolution and you should be fine.
