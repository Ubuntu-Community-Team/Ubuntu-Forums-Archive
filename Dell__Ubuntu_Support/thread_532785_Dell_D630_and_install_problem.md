### Post by gashcrumb on 2007-08-23
My friend at work ran into this kind of problem if I recall.  Try using the "alternate" CD, which uses a text-mode installer.  Once you've gone through the installation you'll then need to log in and do "sudo apt-get install ubuntu-desktop" to get everything else installed.

Although, the guy at work that had run into this problem had just gotten a really new Dell to replace his old one and I think he ran into a rake of problems with hardware compatability and 7.04 and he said that he did some research and found that Gutsy pretty much had support for his hardware, but when he tried it (last month I think) it apparently didn't work out for him.  He's counting the days until Gutsy comes out so he can stop running Windows...
