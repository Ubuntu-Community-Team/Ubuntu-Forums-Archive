### Post by leehach on 2008-04-07
The Windows model probably lets you configure it to have an Intel wireless card.  This is probably a good idea.  I have a 4-year old Inspiron 9100 (still running Windows) with Dell wireless and have found it dodgy, i.e. doesn't connect to some wireless networks that Intel cards can connect to.

I recently installed Ubuntu on a desktop Inspiron 530s that I bought with windows.  Main problem with the install was getting onboard graphics working (I had to back down to generic VESA). Since then, everything seems to be working fine.  You might consider buying a machine with a separate graphics card instead of onboard graphics.

If you're buying a Dell I would highly recommend that you check out Dell factory refurbished.  I saved ~$200 on a low-end desktop compared to what I would have paid new.  If you're going for a high-end machine, you could save more.  You won't find any Ubuntu machines in the factory refurbished section, but since you're planning on buying a Windows machine anyway…

### Post by ugm6hr on 2008-04-07
Indeed - the Ubuntu machines are merely respec-ed Windows machines.

Just get the same specs (generally available as "upgrades") on a Windows machine.

Don't get a "Dell Wireless" - it is a rebranded Broadcom card (which is a bit rubbish on Ubuntu).  Intel is much better.

Or you can buy a separate Intel or Atheros mini-PCI wifi card for about £15 ($30) after the purchase.
