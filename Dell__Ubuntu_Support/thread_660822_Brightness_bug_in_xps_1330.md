There is a bug with brightness controll with Fn+up/down in Ubuntu 7.10 on Dell XPS 1330. The brightness goes back to the values specified in BIOS spontaneously each 5 min or so (only if the computer is idle). 
This is not a big deal but is quite annoing when you a watching a movie for example. 
Anybody knows how to fix this?

### Post by Fascination on 2008-01-07
If you go to 'Screensaver' then 'Power Management' there is an option on the 'On Battery Power' tab called 'Dim When Idle'.
If you uncheck this, it will stop dimming when you aren't using it. Hope this helps. :)
