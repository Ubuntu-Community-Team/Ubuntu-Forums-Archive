### Post by gbrainin on 2007-12-19
I am currently downloading the new Dell image, but I have some questions that perhaps someone here knows the answer to.  Note that I have an Inspiron 530n (two actually; one with the 7300 and one with the 8300 video card).

I have been using the standard version of Gutsy on both for over a month now, without any obvious issues.  So basically I'm trying to figure out if it's worth switching.

3. Anyone know why the image is so freakin' huge?  The 7.04 images were about 750MB; this is like 4.4GB.  What on earth are they doing with all that extra space?

4. Anything else I should know that would make me want to switch?  If I had a laptop, the difference is obvious (the Dell version addresses most or all of the known issues), but with the 530, I'm just not sure it's worth the extra work, since I'm already using 7.10.

### Post by notwen on 2007-12-19
I see no need to switch if your current setup is working w/o any issues.  Other than the working DVD playback(which you mentioned) I don't see any advantages over your current setup and their Gutsy image.  As for the size, I would guess they stuck w/ adding in the majority of the Gutsy packages in there as well so you could use the image as a repo should you need to.  I personally own a Inspiron 1420n and am waiting to see other's experiences w/ this new image on the 1420n before proceeding w/ a install.  Again, if you have a perfectly working setup already installed I see no reason to risk losing your 'working' setup for a few bonus packages they may have included.  Hope this helps. =]

### Post by fragos on 2007-12-19
Installed the Dell 7.10 on my 1420n and all is well.  I first installed in live CD mode and things worked well but on some installs it asked for the DVD.  which worked.  Since all was well I decided to do the factory install of 7.10.  It hasn't needed to ask for the DVD so now we know why the 4GB DVD.  I'm doing a few installs and system custimizations and everything is working out of the box.  Dell has made a number of tweeks.  For example the mouse preferences has a tab for the touchpad.  The 7.10 from the Ubuntu repositories doesn't.  I also think the the "Ubuntu-restricted-extras" package has more included that the Dell 7.04 did.  So far I'm a happy camper and think I was wise to wait for the Dell 7.10.
