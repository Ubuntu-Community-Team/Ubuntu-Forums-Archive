### Post by gbrainin on 2008-02-05
I have not done a dual-boot configuration, but I believe when you install Ubuntu there is an option for keeping your existing windows partition (which is made smaller).  You should defrag your windows partition before starting.

As for media direct, I only know what I've read: It is on a "hidden" partition which is difficult to delete, and not ordinarily touched by regular partitioning software.  However, if you actually press the button, it'll mess up your configuration big time.  Search for threads on "media direct" for much more information.
