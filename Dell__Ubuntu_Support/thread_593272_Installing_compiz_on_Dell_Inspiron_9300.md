### Post by jsnelli2 on 2007-11-09
If you have opened up all repositories so that you are able to download updates, type
sudo aptitude install compizconfig-settings-manager into the terminal.  After you have installed that, go to System>Preferences>Advanced Desktop Settings.  You can then enable the 3d cube. Ctrl+Alt+ Drag or Arrows will do it.  Good luck
