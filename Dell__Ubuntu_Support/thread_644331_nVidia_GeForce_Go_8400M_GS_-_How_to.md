Hi Tigerplug

Here are some tips for you : -

Install Ubuntu-7.10 32bit

Use [***_[COLOR="Blue"]Envy[/COLOR]_***]("http://albertomilone.com/nvidia_scripts1.html") to automate the install of your Nvidia card drivers.  Ubuntu-7.10 already has Nvidia drivers, but you might want the latest that Envy provides.  Also note that you will need to use 24 bit colour (the default is 16 bit colour, I think) 

Use this [***_[COLOR="Blue"]Sound card solutions[/COLOR]_***]("http://ubuntuforums.org/showthread.php?t=530374&highlight=dell+1520") link to get the sound card to work correctly.

And if yo have any other questions about the software setup or installation on the Dell 1520, then read this [***_[COLOR="blue"]thread[/COLOR]_***]("http://ubuntuforums.org/showthread.php?t=577469&highlight=dell+1520").
