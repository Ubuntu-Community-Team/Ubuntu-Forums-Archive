### Post by JawsThemeSwimming428 on 2008-02-11
Are you planning on installing Ubuntu to the same hard drive as Vista? If so, do you already have a partition for Ubuntu to be installed on? If you do not have a partition you will need to resize the Vista partition which has been the cause of many issues. It is advised that you install Ubuntu first and leave a partition to reinstall Vista on. If you can't I am not sure the success rate of resizing Vista partitions.
