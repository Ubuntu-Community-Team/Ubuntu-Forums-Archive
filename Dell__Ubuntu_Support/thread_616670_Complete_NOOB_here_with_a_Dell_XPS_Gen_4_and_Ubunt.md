### Post by Linicks on 2007-11-18
First (but backwards) when you install a deb package it cleverly installs the dependencies needed also... they are only small library files, so don't worry about that.

Secondly, as to startup services (re msconfig or mmc.exe in Windows XP) you don't need to worry about that either.  Nothing is hidden in GNU/Linux.

Windows runs more services that are *unknown* and you can't see (using any tool available a lot are invisible and undocumented so you can't stop tham anyway), I wouldn't worry about moving to GNU/Linux.  Just use it  as is.

You will be enlightened.
