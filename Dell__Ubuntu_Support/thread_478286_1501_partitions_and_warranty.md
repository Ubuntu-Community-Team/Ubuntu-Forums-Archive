### Post by Austin_KW on 2007-06-19
I dont know that model, but Dells usually ship with a few partions; a diagnostic partitions, a recovery partition and the windows OS (xp) partition.

If you want to reinstall you need not wipe the entire disk. Either resize the XP partition or delete the XP partion and recreate it as say 1/2 the size leaving the other 1/2 free for ubuntu to use. If you are using the ubuntu to re-partition the disk you then need to shutdown and install windows first. Then install ubuntu.
