### Post by opivy22 on 2007-09-30
Well, I've solved part of my own problem. If anyone else is having trouble with this I fixed it by changing /etc/acpi/lid.sh

if [ `CheckPolicy` == 0 ]; then exit; fi

to

#if [ `CheckPolicy` == 0 ]; then exit; fi

maybe not the most elegant solution since the policy is the issue, but it works.
