### Post by gbrainin on 2007-11-27
JMO, of course:

Extra system memory is always a good thing, but you can probably get the extra GB yourself for cheaper.  Personally, I got mine equipped with 1GB and haven't felt the urge to upgrade yet.

Similarly, more video memory is considered a good thing.  For most of the stuff you're running, I wouldn't think it would make a difference, but if you're doing video you might notice it.

All in all, I suspect you'll be happy with the system whichever way you go.  Hmm, could I be any less helpful? : - )
