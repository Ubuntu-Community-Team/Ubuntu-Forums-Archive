### Post by Sandlst on 2008-02-27
Hello, I'm running an E1505 here, my bluetooth led is working in 7,10, but I have not had a chance to actually test it yet.  I ordered the bluetooth module after I ordered my laptop, and had to install it myself.  Because of this, I had to go into the bios (F2 when starting up) and enable it under 'wireless settings' before ubuntu would recognize it.  I would start there if I were you, even if you had bluetooth pre-installed.  

Also, does your bluetooth led come on at any point during bootup before ubuntu is started?  If it does not, and you have checked the bios, the problem is likely hardware-related.

*EDIT* to get bluetooth working in ubuntu, for Dell's at least, you shouldn't have to do any extra special steps.  If it works, you should see a constantly lit up led as well as a bluetooth icon in your system tray.
