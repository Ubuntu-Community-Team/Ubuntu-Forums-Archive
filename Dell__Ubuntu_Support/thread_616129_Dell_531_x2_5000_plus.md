I just thought I would write a quick note to say that ubuntu 7.10 32 bit installed pefectly. 

specs 

Dell 531
amd x2 5000 +
2 gb 800 mhz ram
22 inch wide screen monitor
and I installed a nvidia 8600 gt 512 meg video. 
320 gb hd for windows vista
and I installed a 320 gb wd drive from newegg (85$) for linux

it was by far the most simple install I have ever done.

after install  just clicked to install the video drivers.. and clicked to install flash.. way to easy

one thing I did different is that I unpluged the windows drive while I was installing 7.10.. and installed grub and everything as though it was the only drive on the system. on dells during the start up, you have a chance to get into the bias and a boot menu. I am just using dells built in boot loader rather then over write my MBR on the windows drive. 

easy peasy :guitar:
