### Post by ridgeland on 2007-09-30
I've had several installs with sound problems.
Most were solved by changing settings in Volume Control.
My wife's Gateway laptop had the same issue of headset but no speakers.
My first suggestion is double click on the speaker icon in the panel and under switches try what's there.
If that doesn't solve it try Edit -> Preferences while still in Volume Control.  You can find more switches to try.
If that doesn't work experiment with 
System -> Preferences -> Sound and the settings there.  There be sure you have a device detected.
