### Post by sciurus on 2007-11-24
Chen Jun, your configuration is for the i810 driver, which isn't recommended anymore AFAIK. If you switch to the intel driver it won't work because the configuration syntax has changed; see [http://bugs.debian.org/cgi-bin/bugreport.cgi?bug=420419](http://bugs.debian.org/cgi-bin/bugreport.cgi?bug=420419)

### Post by bwakkie on 2008-02-06
I have 945GM and use Hardy. I have dual screen with xrandr with compiz and all works more or less ok... (sometimes the external screen blanks and I have to restart the computer, even restarting X doesn' t help... a hardware problem?)

Did you try upgrading to Hardy? It comes with a better xorg (7.3) which will automatically detect most xorg settings. In Gutsy I needed to play a lot with the xorg.conf but now in Hardy I'm barely touching it.

Attached a nice script I use these days (found it somewhere in these forums I think) to manipulate the monitors with the following commands:

to get the external screen on:
external-monitor on above 1280x1024

to get the external screen off
external-monitor off
