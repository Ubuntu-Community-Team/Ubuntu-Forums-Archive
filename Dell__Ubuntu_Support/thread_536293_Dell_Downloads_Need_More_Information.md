### Post by aikiwolfie on 2007-08-27
There's a simple rule I follow with things like updates. Don't install it unless you need it. So if your system is running fine you probably don't need it. BIOS updates either provide access to previously undocumented features or in most cases fix bugs.

If you need more detail the only people that can tell you are Dells support people. They have a support forum on their web site. Have a poke around and see what you can find.

Okay, if it's a minor 0.0.1 upgrade, and your bios is fine, don't upgrade it.  It may cause more problems then you are ready for.
