### Post by luisito on 2007-11-12
I am about to buy a Dell Latitude D630. I don't have much choice on this unless I want to spend my own money (it's either this, or a Lenovo T60 or a Mac). The laptop comes with either the Intel® Integrated Graphics Media Accelerator X3100 or the 128MB NVIDIA® Quadro NVS 135M&#8482;.

Which of the two graphic cards would be better to be used in Ubuntu? I don't have any moral issues in using a restricted (closed source) Nvidia module. I believe the Nvidia card is supposed to be better. But it is not among the video cards that Dell supports for Linux ([http://linux.dell.com/wiki/index.php/Tech/Video/nVidia](http://linux.dell.com/wiki/index.php/Tech/Video/nVidia)) and I wonder if there is any issues with it. Moreover, do you think that in the long run the open source Intel driver will be better supported? Would I be able to run compiz in either card?

### Post by nutz on 2007-11-13
If you want graphics performance then get the nvidia card. If you want battery life then get the intel...

The Intel card will run compiz just fine with the latest driver and your battery life will be better. But if battery life is no issue then I would go with the nvidia card.
