### Post by gbrainin on 2008-03-27
When I installed WINE in 7.10, it created a new sub-menu in the Applications menu: Applications -> Wine.  It also created new entries in Applications -> Other when I installed Windows programs under WINE.

I don't remember if I used the Package Manager or Add/Remove Programs to install Wine.  I thought they should have the same result, but maybe you should try uninstalling the packages and reinstalling using Add/Remove Programs.
