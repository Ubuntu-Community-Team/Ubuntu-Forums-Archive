### Post by ddrichardson on 2007-08-26
Either you haven't got CD set as your first boot device in BIOS; you built the CD rather than burned an ISO, loosing the bootable sector; or you have a bad burn - check it against the MD5SUM on the Get Ubuntu page - if it is a bad burn try it at a lower speed.
