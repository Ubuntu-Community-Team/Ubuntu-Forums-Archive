### Post by HermanAB on 2007-07-16
Download a Knoppix CDROM and see if sound works.  If it does, run 'lspci' to see what the name of the sound chips are.  Then run 'lsmod' to see what drivers are loaded to make it work.

Knoppix is also a Debian distribution, so you can learn a lot from it and the information applies directly to Ubuntu.

'Hope that helps!
