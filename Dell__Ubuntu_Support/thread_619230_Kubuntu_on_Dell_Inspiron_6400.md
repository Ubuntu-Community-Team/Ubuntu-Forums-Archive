### Post by dacap06 on 2007-11-21
Perhaps you may have gotten a bad burn on the disk?  I'd suggest reinstalling Ubuntu 7.10.  Once you are sure it is stable, add all the KDE stuff from the respository.   There's a KDE metapackage called simply "KDE" that you can select and in stall in your favorite package manager. So select KDE and KDM (the KDE X display manager), then apply your changes.  Doing so will install all the official KDE modules and the KDE display manager.  Following that, search for "KDE" in your package manager and then install anything else that looks good to you.
