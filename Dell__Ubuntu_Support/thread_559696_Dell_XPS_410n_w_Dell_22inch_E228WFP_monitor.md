### Post by Louis Cypher on 2007-09-25
Just wondering fi anyone else has heard of this.  I just purchased a Dell XPS 410n with the 22in monitor.  a great linux pc so far.  I've had it for 2 weeks.  I did have a couple of issues:

1:  My Nvidia card wasn't listed under Restricted Drivers Managers like the Dell wiki says.  I used Envy script to install and get it working.   I have World Of  Warcraft and EvE Online running on it so all is good.

2:  Still can't get it to start up with the 1680x1050 resolution.  I have to run nvidia-settings and select it from there.  It won't list that resolution in System->Preferences->Screen Resolution until I chnage it in nvidia-settings.  Not a huge issue because I don't reboot very often.

The last issue is more specific to my LCD.  I have a pixel that is out.  I wouldn't have noticed it but it shows in my Google search box, which is white, so it stands out.  It looks like dust, not too annoying but I called Dell anyway.  According to "John" at the XPS support center you need 6 total pixels out for the warranty to kick in.  Kinda lame, but I have a year left on the warranty so I'm sure 5 more will go out by then :twisted:

All in all a great Linux experience.  Linux has come a long way since Slackware 0.1.   With this pc I have moved to Linux as my main OS.
