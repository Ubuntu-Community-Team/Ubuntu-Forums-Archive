### Post by SmallFish on 2008-02-25
Ubuntu has been working nicely since 6.06 on Dell. You don't need specific Dell driver for running Ubuntu on Dell. Just go to Synaptic Package Manager and choose the stuffs you need. One good thing of running linux is that you can do anything you want instead of depending on windows.  

Though, I would say Linux has been lagging behind in configuring special keyboard and mice features. you do need to configure a new config files before enjoying the full features.

I suggest you get Desklet as well. You will like it.
