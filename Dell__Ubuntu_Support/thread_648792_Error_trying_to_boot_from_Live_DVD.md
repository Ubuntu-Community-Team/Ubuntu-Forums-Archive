### Post by gbrainin on 2007-12-24
Sounds like there's an error with the DVD itself.  Your system is not recognizing it as a bootable device.  This could happen if there was some physical error with the disc or with the burn, or if you burned a copy of the file rather than burning it as an ISO image.

It's also possible that there was an error when downloading the ISO image, but I would try re-burning first.
