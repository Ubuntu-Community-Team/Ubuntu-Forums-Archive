### Post by Eldar11 on 2007-12-09
Automatix2 is your friend.

Install Automtix2 by going [[COLOR="Blue"]here[/COLOR]]("http://www.getautomatix.com/wiki/index.php?title=Installation#Ubuntu_7.04_.28Feisty_i386.29") and choose your distro, use the package installer to install.

Once installed, it will be under Applications > System Tools

open it up and look under virtualization and check the box next to Wine and click start.  It will run all the scrips needed and once installed, there will be a separate menu item for Wine.  Hope this was of some help to you

### Post by gbrainin on 2007-12-09
If you're using 7.04 or 7.10, you should be able to install Wine via the Add/Remove dialog in the Applications menu.  It won't be the very latest version of Wine, but it should work.  Now, whether the particular program you're using will work under Wine, and to what extent, is entirely another matter.
