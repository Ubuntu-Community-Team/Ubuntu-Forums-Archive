### Post by Dellfan on 2007-11-05
Q10 does not run under Wine... at least not without mucking about. Just tried it. Does not play well with Wine.

If you want a nice, lightweight fast editor that plays well with Wine (so is available in Linux) that is Windows based so you can keep the same editor across both platforms I would suggest Notepad++. If you are looking for specific features and are willing to learn a new editor then are lots of great editors to choose from.  What functionality in particular are you looking for?

### Post by Dellfan on 2007-11-06
Fonts are easy.... Find a decent editor you like and then set it to use the fonts that make you happy. Any vaguely modern editor will do WYSIWYG and you can install any TT font out there.

For keyclick noise, have you just tried turning it on via "xset -c on" or "xset -c <value>", where <value> is a number between 0 and 100 indicating the percentage of the maximum volume that you desire?
