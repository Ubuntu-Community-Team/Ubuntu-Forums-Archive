You should have the option to upgrade to 7.1 in software update. If it is not there you can always download an image of the alternate install cd and use that to upgrade. Instructions are available at the bottom of the page: [http://www.ubuntu.com/getubuntu/upgrading](http://www.ubuntu.com/getubuntu/upgrading)
