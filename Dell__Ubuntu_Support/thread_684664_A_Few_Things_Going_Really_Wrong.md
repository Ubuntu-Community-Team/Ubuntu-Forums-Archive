### Post by monkey56657 on 2008-02-01
1. Its not normal but I have no idea how to fix (windows I can help on but still not expert on linux). 

2....

3. Simply place the Ubuntu Live CD into the tray and restart the machine. Once Ubuntu CD has booted select the install option. You will be prompted to choose partition size on install. You can just select the entire disk and continue. If you choose to do the manual partition choice then you need to select to format the drive when you get to that stage. 

4. The recovery partition may be gone. Again use your Ubuntu Live CD and once logged in go to System > Administration > Partition.. From here you can check whether there is a recovery partition still available. If you can see one then reply here that you can. Alternatively you can use a recovery CD for windows (if you have one).
