### Post by khedron on 2007-07-01
For moving your documents, apparently there is some utility that can do this for you.

Check out [https://help.ubuntu.com/community/SwitchingToUbuntu/FromWindows](https://help.ubuntu.com/community/SwitchingToUbuntu/FromWindows)
The relevant part is section 6.

As for your hardware, the easiest way to find out whether they will work is to boot the Ubuntu Live CD and try to use them. Does your flash drive connect via USB? If so, it should work.

In any case, you can type the model number and "linux" into Google to see if there are any howtos or problem pages posted on the internet about it.

### Post by Steven_B on 2007-07-01
> Looks like it should be a reasonably easy transition.
Hopefully it will go well for you.  Just remember that Linux is very different from Windows, and takes some getting used to.  If you're patient and willing to learn, I think you'll find Ubuntu is a great OS.

Transferring your documents should be pretty easy if you have a network, large flash drive, or can burn CDs from your old computer.  This wiki page has a lot of helpful info about transferring settings: [https://help.ubuntu.com/community/SwitchingToUbuntu/FromWindows/TransferringFilesAndSettings]("https://help.ubuntu.com/community/SwitchingToUbuntu/FromWindows/TransferringFilesAndSettings")
You may also find this helpful: [https://help.ubuntu.com/7.04/windows/C/index.html]("https://help.ubuntu.com/7.04/windows/C/index.html")

Your printer may take a little bit of work, but other Ubuntu users have gotten it working.  Search around on Google and these forums, and you'll find a wealth of information.  If you run into a problem and can't find an answer, just ask - someone else has probably had the same problem.

Welcome to the Ubuntu community!
