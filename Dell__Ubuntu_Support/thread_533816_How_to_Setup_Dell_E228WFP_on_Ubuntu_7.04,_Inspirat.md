### Post by ntom-taylor on 2007-08-24
After spending a few hours researching, and trying various things, i finally found a solution on ubuntu forums which sorted out the monitor, and finally the E228WFP displayed a resolution of 1680x1050 which looks absoloutly gorgous !

**Configuring the E228WFP for ubuntu**

Firstly, backup your existing xorg.conf, we do this so that if somthing goes wrong on your next reboot and it puts you on the terminal (should be able to understand what happened from the error log) you can restore from this file if nessesary

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```

Next you want to edit the file and put in the following details.. starting at "Section Monitor"..

**The xorg.conf from the original dell installation (Straight from factory box) _This is unmodified and will not fix the monitor_**

### Post by toddward on 2008-03-16
Well the first step is to familiarize yourself with the terminal commands so that you know what you're doing.  After you've done that, it's time to change your xorg.conf file.  Additionally, read about how to switch between virtual terminals.

It should be noted that the xorg.conf file can only be finally changed when logged in as root.  So you'll need to login as root:

```
toddwardzinski@miniServer:~$ su -
Password:
```

Then you want to change directory to where the xorg.conf file is located:
```
cd /etc/X11
```

Make a backup of your xorg.conf file:
```
cp xorg.conf xorg.conf.backup
```

Finally, edit your xorg.conf file:
```
gedit xorg.conf
```

When you're done save the file in gedit, close the window, and close the terminal window.  To restart X you're going to have to press Ctrl+Alt+Backspace.

Note, if you're changes do not work you'll probably have to change to another virtual terminal.  Log in as root again, change to your X11 directory, and:
```
cp xorg.conf.backup xorg.conf
```

Switch back to your original virtual terminal and restart X.
