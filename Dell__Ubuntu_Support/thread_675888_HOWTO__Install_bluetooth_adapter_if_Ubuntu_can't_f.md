### Post by tr333 on 2008-01-23
Tested on: Dell Inspiron 1520 with Ubuntu Gutsy Gibbon 7.10

If you have laptop or desktop with a built-in bluetooth adapter, you might find that Ubuntu can't locate the hardware for the bluetooth adapter.  It will appear to be nonexistent to the system and all bluetooth utilities.  The solution for this was to install Windows XP and install the bluetooth driver in Windows.  This seems to enable the bluetooth adapter in the hardware,  thus enabling Ubuntu to find it when you restart the computer.  This might also work with other versions of Windows/Ubuntu but I haven't tested them.
