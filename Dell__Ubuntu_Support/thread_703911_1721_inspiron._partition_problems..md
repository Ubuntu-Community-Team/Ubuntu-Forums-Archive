### Post by evolving_monkey on 2008-05-03
I have the same laptop. The only way I have ever really had a successful dual boot with Vista and Ubuntu is with a clean install of Windows. You can set up the partitions from the beginning correctly. After that, if you have to, reinstalling Ubuntu is easy because the space for the install is already there. If you have the room I recommend setting up just a storage drive between Windows and Ubuntu. That way if either or both of your installed operating systems crash hard you won't necessarily have to move any data around. You can just reinstall the operating systems and the data is still in the partition you set up for storage. Unless something really serious has happened you usually won't lose the data.

It has been my experience that Vista does not like having it's partition adjusted once its installed. I have gotten it to work by resizing but Vista was really flaky.
