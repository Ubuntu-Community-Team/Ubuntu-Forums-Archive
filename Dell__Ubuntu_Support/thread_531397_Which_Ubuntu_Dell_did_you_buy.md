The Dell Utility partition is a bunch of system utilities (mostly hardware tests) provided by Dell.  The FAT32 partition is where the system restore is stored.  The other partitions are /boot, swap, and /, respectively.  See [http://linux.dell.com/wiki/index.php/Ubuntu_7.04/Default_Partitions]("http://linux.dell.com/wiki/index.php/Ubuntu_7.04/Default_Partitions")
