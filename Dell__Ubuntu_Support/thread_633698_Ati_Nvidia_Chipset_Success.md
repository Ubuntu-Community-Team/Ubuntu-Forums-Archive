I had an ATI X1400 and replaced it with NVidia 7300 because the driver support is so bad in ATI.  But, a year from now, that could change, because ATI is going open source.  However, just because they release documentation and some code does not mean their driver will be better than NVidia's closed source.  For example, the existing open source drivers for older ATI cards are not as good as NVidia's closed source driver.  Why? Because NVidia has a lot of people working on it, and the open source programmers are few in number and time.  But, ATI has now hired at lease 2 programmers full time, so they can catch up.  But, it'll probably be 6-12 months before all the features work as good as NVidia is now.  And, NVidia is constantly improving.

If you want to get a good comparison, check out phoronix.com.  There you can find forum threads with complaints about each.  It seems to me that ATI is behind right now .. for example, their driver fglrx will not permit suspend support on gutsy, so if you are buying a laptop and want to suspend, you have to stay with feisty.  Also, ATI seems to lack support for TV out right now.  Here's a thread discussing what's lacking (and wished for) in the current driver:

[http://www.phoronix.com/forums/showthread.php?t=6800](http://www.phoronix.com/forums/showthread.php?t=6800)
