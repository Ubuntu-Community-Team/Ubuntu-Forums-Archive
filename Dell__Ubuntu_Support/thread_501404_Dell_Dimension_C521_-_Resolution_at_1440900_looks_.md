### Post by strabes on 2007-07-15
First of all, you need to know what kind of video card you have. Go [here](https://help.ubuntu.com/community/BinaryDriverHowto/) to find out what kind of card you have and how to install drivers for it.

It sounds like you have ATI card, in which case you need to install fglrx, ATI's proprietary 3d linux driver. It's linked to on the above page.

### Post by angryfirelord on 2007-07-18
LCDs don't really have a refresh rate like CRTs do. LCDs are refrshed by a response time measured in milliseconds rather than CRTs which are measured in Hertz. Usually, the recommend refresh rate on an LCD is 60Hz, but if 50 works, then you should be fine.
