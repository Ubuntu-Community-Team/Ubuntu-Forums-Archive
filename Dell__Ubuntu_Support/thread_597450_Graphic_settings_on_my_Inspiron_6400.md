### Post by dacap06 on 2007-10-30
This is a tough one.  I've got three guesses.

1) Try setting the monitor selection in the display configuration GUI  to "Plug and Play" if it isn't already.  This works the best for me on my Dell Inspiron 5100.

2) Try setting the monitor selection in the display configuration GUI  to the exact make and model of your monitor.

3) Try setting the monitor selection in the display configuration GUI  to "LCD Panel 1680x1050".

Hope that helps.
