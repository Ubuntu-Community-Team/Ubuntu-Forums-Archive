### Post by rufius on 2008-01-09
To be honest you won't gain much from using the x86_64 version of any operating system unless you're doing some heavy scientific work that needs the extra address space for accuracy. 

For the most part it should all work fine out of the box, but problems can arise and to avoid them I would **HIGHLY RECOMMEND** just installing the 32 bit version of Ubuntu. I have two computers that are technically x86_64 and I don't even bother running 64 bit operating systems on them, they're both plain 32 bit systems. 

At work though I do have a few servers doing number crunching so they're running 64 bit but in reality, desktop users can't gain much from it, with the exception of gaming.

Hope that helps :)
