### Post by rcdegges on 2007-09-03
I've had problems with the current Ubuntu installer on older machines as well. I would recommend (before anything else), that you try to install it in "safe graphics mode". To do this, when your computer boots to the cd, simply hit the down arrow to select that option, then press enter. You will only have 30 seconds to do this before it auto boots the default installer. I've had a much higher rate of success with the "safe graphics mode" because it uses less system resources and therefore, would be more suitable for an old machine. I'm not sure why you are getting the error that you are, but hopefully this will help!
