### Post by Malcy on 2007-11-10
Not by just downloading it.

If you burn Ubuntu to CD and run it as a live CD then your Vista install will also remain untouched.

If you install Ubuntu from the CD to your hard disk, you have the choice of wiping out Vista (\\:D/) and installing Ubuntu as the only OS or installing Ubuntu along with Vista in a dual boot setup.
