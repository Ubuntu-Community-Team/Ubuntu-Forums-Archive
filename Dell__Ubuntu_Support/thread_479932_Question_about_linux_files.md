### Post by qamelian on 2007-06-20
> **cbranha1 said:**
> I can see windows files in linux as read-only, but I can't see linux files while I'm in windows. How do I fix this?

You need to install a driver in Windows that lets Windows read the Linux Ext2/3 filesystems. You can find the driver at [http://www.fs-driver.org/](http://www.fs-driver.org/) .
