### Post by ridgeland on 2007-08-10
I bought a Dell E520N.
I don't follow much of the nuts and bolts but I had expected to use one of my old hard drives.  I didn't know about the SATA and ATA difference.  So I cannot use the old ATA hard-drive. Maybe I can somehow but I haven't bothered to find out.   I was also thinking of using my old DVD-RW drive, but luckily I bought a new DVD-RW with this PC.
My on board sound card was fine except for one issue.  I wanted to be able to record from streaming audio (Radio-Canada, eh?) but the on board sound system would not allow it.  I solved it by putting in the Sound-Blaster Audigy card from my old PC.  Now I can record what I'm listening to.
I would expect issues from MS if you try to install Windows on a PC that was not blessed at the factory.  I don't have Windows on this PC. :)  I have re-installed WindowsXP on PCs for others and had to call MS (in India) to get a magic code to enable the installation, even when entering the product key from the sticker on the PC.  This has not happened when reinstalling WinXP on my Gateway with the original Gateway CDs, even on a new hard drive.
