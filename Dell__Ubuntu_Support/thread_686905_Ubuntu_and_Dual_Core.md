### Post by willybk3 on 2008-02-03
I ordered a basic 530s with no options - 299 +tax & shipping.  It came with Vista basic.  I reduced the Vista partition by 50% (250 GB -> 125 HDD) and installed Ubuntu x86 (64 bit version).  The only problems so far is that grub sometime fails to correctly boot Vista when selected other than that, I added 2GB memory to existing 1GB and machine seems very quick in both Ubuntu and windows, I do not think I could have assembled by own with a legit copy of windows for less.  When I run system processes, both cores seem to be working in both operation systems.  The main drawback with using the x86 64 bit version is that the majority of the Ubuntu software has not yet caught up and is still i386 (32 bit version), example - Skype video.  Same issue when the industry went from 16 -> 32 bit processors / operating environments a few years back.
