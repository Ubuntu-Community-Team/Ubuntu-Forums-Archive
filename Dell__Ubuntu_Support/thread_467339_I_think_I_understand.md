### Post by reckless2k2 on 2007-06-07
hahaha..very good one. "everything" won't work out of the box with a stock XP install on that Dell. If you did a stock install with a pure clean XP CD, you'd still have to go to the manufacturer site to get a lot of drivers to support a lot of the hardware. Just check Dell's site and you'll see. The issue is the a lot of hardware manufacturers write drivers for windows not linux. the linux community has to develop their own drivers and many of which get built into the kernel eventually. this means that a clean linux install will support more devices out of box than a clean xp install. the linux community is full of ninjas that write a lot of these drivers for linux support from scratch and give them to the community for free. it's just getting around the difference in thinking.
