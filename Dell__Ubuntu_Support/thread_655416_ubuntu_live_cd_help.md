### Post by gbrainin on 2008-01-01
If you boot from the live CD, you can try out the operating system without making any changes to your hard drive at all.  (Of course, you could manually delete files, but the operating system won't make changes by itself.)  It will be a lot slower than running from the hard drive, but you can see if everything works before you try installing it.

I think the "run with windows" refers to some of the software that comes with Ubuntu, in particular Open Office, which has windows versions that you can try by putting the CD in while windows is running.

Correct.

Live CD gives you two options:

Run directly from the CD (will be slow) so you can test everything. NOTHING will be saved to your HD by default.  Very safe.

Use the "Install" feature to ... well ... install.
You'll have the options of a "total" install, destroying everything else on your HD, or changing partition information to keep whatever other OS's you have.  NOTE:  Backup important data and files first if you choose this.  One wrong keystroke, and you could loose it all.  But if you go slow, and check what you are doing, there is a good chance that you won't and you'll have duel boot capability.
