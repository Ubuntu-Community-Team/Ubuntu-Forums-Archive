### Post by sdennie on 2008-04-11
You can also install virtualization software like virtualbox or vmware and literally install Windows into a window.  That should allow you to run any Windows program but, games aren't likely to be very playable because it emulates the video card.  If you google for virtualbox, you should find more information about how to do this (it's fairly trivial).
