### Post by amiga_os on 2008-03-09
_Firstly, in answer to your question:_
Vista will be an absolute nightmare to install... this is based on my knowledge of installing 95, 98 and XP on numerous occasions.

Many of the device drivers you'll probably have to source separately, burn them to CD first, and have them ready before you install Vista.  Vista will make things extremely difficult for you to dual boot... and you may lose all your data in the process of installing.  So you'll have to back up your Ubuntu data first.

Compare that to installing Ubuntu, can be done in 7 clicks... will import all of your Vista personal data, and will allow you to look at the partition system in an easy to understand way.

Frankly... you would have to buy Vista pre-installed, ironically because it's just not as good a product as Ubuntu.

_Now a shameless attempt to get you to change your mind:_
However... Ubuntu + your faster processors + wine (or even cross-over office if you're dead lazy) + any Windows dependent software you want will work most of the time.  I use a specialist piece of Windows dependent software called BibleWorks, last year it didn't work in Wine, now it does - perfectly - because I got involved and pestered the Wine people, and they listened and delivered.

Oh, and don't forget the software stack:
Remember, if you buy Vista pre-installed, you then have to either buy hundreds of pounds worth of software to get the same functionality as Ubuntu... or you have to trawl the web and find the free software alternatives, download and install them all individually.

_In summary:_
Because Vista is a poor product when it comes to end-user installation, if you must have Vista, it will be extremely painful installing it later.

But, in saving packets of money, supporting the operating system of the future, having a functional software stack immediately, having thousands of packages of software available at the click of a button, being more secure, and being able to run Windows programs through Wine... my advice is buy Ubuntu pre-installed.

But the choice is yours...:)
