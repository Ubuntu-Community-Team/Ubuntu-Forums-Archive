### Post by dougfractal on 2007-08-16
You're doing alright .

leave the the mouse0  , you can have multiple input devices.

I wish I had a laptop here so I could confirm this for you.

add in Section "Module"
>  Load "synaptics" 
add in Section "InputDevice"-"Synaptics Touchpad"
 > option          "SHMConfig"             "on"


I've just found this post with your unsolved prob
[http://ubuntuforums.org/showthread.php?t=526459](http://ubuntuforums.org/showthread.php?t=526459)

If you are using an Alps-Touchpad, which you can find out by typing
> 
cat /proc/bus/input/devices 
or
> grep ALPS  /var/log/Xorg.0.log  post

if its an ALPS try this InputDevice instead
> Section "InputDevice"
        Driver          "synaptics"
        Identifier      "TouchPad"
        Option          "SendCoreEvents"        "true"
        Option          "Device"                "/dev/input/event2"
        Option          "Protocol"              "event"
        Option          "SHMConfig"             "on"
EndSection

### Post by strabes on 2007-08-18
I have the E1705 as well. Here's my synaptics touchpad section of my /etc/X11/xorg.conf. It's configured exactly how I like it.

```
Section "InputDevice"
	Identifier  "Synaptics Touchpad"
	Driver      "synaptics"
	Option	    "SendCoreEvents" "true"
	Option	    "Device" "/dev/psaux"
	Option	    "Protocol" "auto-dev"
	Option	    "SHMConfig" "true"
	Option	    "MinSpeed" ".1"
	Option	    "MaxSpeed" ".3"
	Option	    "AccelFactor" ".0017"
	Option	    "MaxTapTime" "150"
	Option	    "MaxDoubleTapTime" "150"
	Option	    "VertEdgeScroll" "1"
	Option	    "HorizEdgeScroll" "1"
	Option	    "VertScrollDelta" "300"
	Option	    "HorizScrollDelta" "250"
	Option	    "BottomEdge" "4300"
	Option	    "RightEdge" "5250"
	Option	    "TapButton2" "2"
	Option	    "TapButton3" "3"
	Option	    "RTCornerButton" "0"
	Option	    "LTCornerButton" "0"
#right click
	Option	    "RBCornerButton" "3"
#middle click
	Option	    "LBCornerButton" "2"
EndSection
```
