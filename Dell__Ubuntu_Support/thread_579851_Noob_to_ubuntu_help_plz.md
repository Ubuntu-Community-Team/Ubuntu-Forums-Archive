### Post by Mordac85 on 2007-10-18
The [Intel Core 2 Duo]("http://en.wikipedia.org/wiki/Intel_Core_2_Duo") is  64 bit, but you can run 32 bit (Std PC ver) just the same.

Since you're just starting out, I'd recommend trying the 32 bit and get comfortable with it.  If it seems a bit slow to you at that point, you could then rebuild w/the 64 bit.  I'm not sure if you could upgrade or how well supported the 64 bit version of various apps are, so that's why I'd say start w/the 32 bit.
