### Post by ctuk on 2007-07-21
OK - worked it out.  For anyone else facing the same problem, the issue is the monitor refresh rate.  Find out the refresh rate for the monitor, get instructions on how to fix from the ubuntu documentation site (I haven't done this yet, but remember seeing them), and start ubuntu in graphics safe mode so you can get in there and fix it.
