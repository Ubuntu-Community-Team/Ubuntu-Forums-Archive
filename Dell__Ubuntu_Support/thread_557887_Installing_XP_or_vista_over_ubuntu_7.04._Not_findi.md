### Post by neptho on 2007-09-23
Go into your BIOS and reset ALL settings to default.

Then, when it comes time to do this 'last' install, hit F8 for the pre-boot menu.  Tell it to boot from the CD; you can now (re)install Windows.

After your computer reboots, do not press any keys.  Let it sit a full 10 seconds.  Then it'll probably work.  If you mash the keys with the CD in the drive (and the system is set to boot from it), Windows sees this as 'I want to run setup again.'
