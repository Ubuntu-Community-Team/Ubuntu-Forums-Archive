### Post by Linicks on 2007-09-27
To be honest, 'suspend' isn't intented to be used overnight (I don't care what Vista does!).

If you need to save current state on a laptop for say > 30 minutes, use 'hiberate'.

Hiberate saves a snapshot of current machine status to disk and shuts down... then powering back up restores the state.

Literally 'suspend' is to be suspended for a few minutes... 'hibernate' means go to sleep.
