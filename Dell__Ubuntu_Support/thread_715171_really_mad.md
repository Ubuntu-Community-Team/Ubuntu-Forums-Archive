I don't know if you have been able to resolve that particular issue yet? If not, this is the solution that worked for me...

When typing, if ALL your words are underlined in red, even if they are spelt correctly, right click on a word which is underlined in red and hover over 'Languages'. Check which language is set. Mine had set "de_ AT" as the default language! I selected English and all is now fine!
