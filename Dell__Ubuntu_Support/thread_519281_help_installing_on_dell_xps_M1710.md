### Post by ridgeland on 2007-08-08
Sounds like a screen resolution problem.
Like displaying 800x600 on a screen of 640x400.
When you boot the Ubuntu 7.04 Live CD at the bottom is an option:
F4 VGA
Set this to 800 x 600 x 16 ( or x 32 if you have plenty of RAM) and try again.
