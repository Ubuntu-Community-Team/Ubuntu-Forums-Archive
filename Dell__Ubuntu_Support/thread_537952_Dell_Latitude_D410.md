Have you burned the ubuntu iso as an image or just copied it as a file.  You can tell by booting from your hard drive and then looking to see what files are on the CD.  There should be a couple of folders and several files, I cant remember exactly how many and don't have a CD to check.  If you have just the iso file showing you burned a useless coaster and will need to start again.  Make sure your cd burning software burns a disk image; it will be there somewhere in the menus, I'm sure.  Burn it slowly and don't do anything else with the computer whilst it's burning.  That should do it!

PS.  Don't forget to make sure your computer is set to boot from CD first in the bios.
