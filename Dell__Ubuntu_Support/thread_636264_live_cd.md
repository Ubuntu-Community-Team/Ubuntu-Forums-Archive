### Post by elliotjreed on 2007-12-09
From the live cd you can:-

- Run all the preinstalled software (firefox, the GIMP, etc.)
- Save files to hard drive (another partition, eg. your windows system)
- Connect to the internet

However, you probably won't be able to install more software until actually installed

### Post by gpilkay on 2007-12-09
I'll be honest, my Dell Vostro 1500 does NOT like Ubuntu, an attempted install crashed it badly.  So nowadays, when I want Ubuntu on that machine, I just boot from the Live-CD.  This works fine.  I have, from it, OpenOffice, GIMP, Pidgin, etc, with no issues.  I save my work to a USB drive, so my XP drive is never touched.  Works great, especially if you, like me, have a computer that you are happy with and don't want to change.

However, I do have to admit that a Live-CD is slightly slower then a full install.  I have two hard drives on my desktop, one of which is Ubuntu, the other XP.  On this old Celeron D system, Ubuntu goes faster then the Live-CD on the duel-core laptop.

If you just want occasional ubuntu use, or better yet, Ubuntu from any computer you happen to be near on the go, the Live-CD has many more uses then just installing linux.  It's also very handy to recover files from an unbootable or virus-muddled system.  I've done this many times.
