I have the same laptop. So far:
-Wifi led: I ignore this. I recall trying a hack but it was erratic.
-3D games and driver: try to install the fglrx driver from the ubuntu repositories.
-Disable the wifi: use fn+f2
-Multimedia keys: the volume ones work without trouble. The playback ones can be mapped in System/Preferences/Keyboard Shortcuts. Keep in that that your software will need to be aware of them.
