### Post by grenet on 2008-03-05
when you installed Ubuntu, did you use the Dell installation CD provided at [http://linux.dell.com/wiki/index.php/Ubuntu_7.10?](http://linux.dell.com/wiki/index.php/Ubuntu_7.10?)

Some of the drivers for the wireless cards are restricted, and Dell installation CD has the patches built into the install.

I would recommend installing with this, as my 1420, purchased in December, had the same issues when I tried to do a vanilla install of 7.10.

The Dell CD fixed all that stuff..
