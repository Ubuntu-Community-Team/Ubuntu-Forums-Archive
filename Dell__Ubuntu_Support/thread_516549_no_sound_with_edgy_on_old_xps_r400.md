### Post by ridgeland on 2007-08-08
To trouble shoot sound I start with the volume icon (a speaker) in the panel.  Double click on it and explore all the settings and options.  This should open "Volume Control".  Be sure to explore the menu option Edit->Preferences.  

I once had a problem where there was no sound because the default was a digital output plug, but my speakers were analog.  A simple click on the option box solved the problem.

The next thing I explore is main menu -> System -> Preference -> Sound.  Is there a driver installed?  choice of drivers?  Experiment with all the test options to see if you can find sound anywhere.
