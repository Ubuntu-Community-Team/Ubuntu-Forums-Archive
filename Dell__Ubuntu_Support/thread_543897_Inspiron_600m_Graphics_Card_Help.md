### Post by tito2502 on 2007-09-05
Hi.

What version of Ubuntu are you running? If you are running the latest verion (7.04, Feisty Fawn) you can install the drivers by:

Click the system menu
Choose the administration submenu
Click "Restricted drivers manager"

You graphics card should be listed, you will be able to check the box next to it. Ubuntu will download and install the drivers, all you will need to do is reboot.

I would just like to point out that the radeon mobility cards are one of the few that don't work with Compiz and Beryl "Desktop effects".

Hope this helps.
