### Post by khedron on 2007-07-28
The Live CD is slow, since it has to read from the CD drive rather than the Hard Drive, which is about 10 times faster.

If you can use all your other hardware, then you have all the needed drivers. This includes playing music, having the right screen resolution and being able to print. (You might need to add the printer to Ubuntu's printer list first.) Installing onto the Hard Drive should fix the slowness.
