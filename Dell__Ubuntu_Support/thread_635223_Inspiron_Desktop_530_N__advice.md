### Post by JangMunho on 2007-12-09
Recommend you to install 7.10 rather than 7.04, since kernel 2.6.20 (used by 7.04) has a big bug which can destroy SATA drive.

Use safe graphic mode to load live CD, because the open source driver cannot support GeForce 8 series. After installation, you must install nvidia's official driver manually.

### Post by zesty on 2007-12-11

It comes with Open Office which is as easy as it gets.
