### Post by strabes on 2007-07-25
I have the Inspiron E1705 (last year's model which has since been discontinued) with the ATI Mobility Radeon X1400 (slight upgrade from the basic card) and everything works perfectly out of the box on feisty. You do have to use the alternate install CD to install ubuntu and then install the fglrx driver in a tty using the steps located [here](https://help.ubuntu.com/community/BinaryDriverHowto/ATI#head-d8c6fd05bce340dfc3ad483abf0e18997868540b)  due to a bug in feisty. I would recommend getting an nvidia card because then everything will work, including the live CD.

Things I have not tested: Card reader, i394 port, modem, the TV out and other white rectangle video out plug that I can't remember the name of.
