### Post by jazzcat on 2007-06-16
Hey all,

I probably ought to let my fiancee do a review since it's her laptop, but here are my initial thoughts.  I'm a CentOS user and I've been using Linux wince 1996.  When my fiancee needed a laptop for school, it was a choice between a Mac or this Dell.  The Dell won out.

As I've heard other places, Dell underpromised and overdelivered.  They promised a two-week lead time but got it here in a week.

As far as I know, the first bootup was plug and play.  I was trying to tell her how to install yahoo IM via text message (I was at work), but in the end I couldn't find libssl 0.9.6 so we went with Kopete instead.  She's still up in the air as to how much she likes it though.

The first thing I noticed was the goofy resolution and the fact that things were stretched out.  I tried to download the 915resolution package, but it took me a while to figure out that I need to use Synaptic for this.  The Add/Remove option under Applications didn't find the 915resolution package.  So we got that working after using Synaptic.

Flash installed without a hitch - I was impressed.

Java, on the other hand, was a bit feisty.  At first I tried to install the binary packages from Sun, but that didn't work so well.  So, I tried to install the version 6 JRE via Synaptic.  It installed OK, but none of the java applets from popular pages worked.  So I rolled that back and installed Java 5, and everything works very well now.

All in all, I think this is a good machine.  I think it would have been better if Dell had installed the 915resolution packages from the start.  My fiancee actually did not mind that images were stretched, so it's questionable if that aspect of the system is actually a problem.

Cheers,
-J

(Posting from Feisty Fawn on an ancient Sager NP9820 - 450 MHz, 256MB Ram)
