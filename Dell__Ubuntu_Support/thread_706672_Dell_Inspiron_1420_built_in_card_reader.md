### Post by spartan_87 on 2008-02-24
I have a Dell Inspiron 1420 running Ubuntu 7.10 with a built in card reader that reads SD/MMC and MS/Pro. When I insert a SD card Ubuntu reads it fine and I can write to it, but when I insert a Memory Stick Pro Ubuntu doesn't read it and no mount icon comes up on the desktop. Anyone have any ideas to why it won't read the MS Pro card?

---

### Post by jdelaros1 on 2008-02-25
Memory stick is only supported on the flashmedia chipset from TI (the
7xx1 parts). Both Memory Stick and Memory Stick Pro should be supported now. There is no support for Memory Stick from other vendors right now.
