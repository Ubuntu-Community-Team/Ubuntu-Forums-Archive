### Post by notwen on 2007-09-04
There should be a Ubuntu Feisty LiveCD included, but by default it is not functional w/ the 1420n's disc drive.  You would need to use an [alternate install disc iso]("http://releases.ubuntu.com/feisty/") to re-install Ubuntu from CD.  As for the additional packages, they are listed [here]("http://linux.dell.com/wiki/index.php/Ubuntu_7.04/Add-on_Packages").

Here is a [thread]("http://ubuntuforums.org/showthread.php?t=509408") detailing the difficulty of installing Ubuntu from cd onto the 1420n, hopefully Gutsy will remedy the majority of this and 'just work'.
