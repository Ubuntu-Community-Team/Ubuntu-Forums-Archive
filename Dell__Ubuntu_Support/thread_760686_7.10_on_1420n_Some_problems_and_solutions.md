### Post by kdevil on 2008-04-20
I got a 1420n with Gutsy a few days ago, and found that it had some issues that needed to be taken care of. So, for anyone else having trouble, here's a list of some problems I've had that weren't in the [Dell wiki]("http://linux.dell.com/wiki/index.php/Ubuntu_7.10"), and how to fix them:

[LIST]
[*]**Wireless completely disappears after resuming from suspend**: sudo rm /etc/udev/rules.d/70-persistent-net.rules, then reboot.  That file will get regenerated and hopefully things should work after that.
[*]**Firefox thinks every word is misspelled**: Right-click the text field, go to "languages", and select the correct language.
[*]**Webcam not found after resume**:  sudo rmmod uvcvideo, then sudo modprobe uvcvideo.
[*]**Built-in mic doesn't work**: I tried a lot of stuff, but I think what fixed it was going into alsamixer and setting "Digital Input Source" to "Digital Mic 1".
[/LIST]
