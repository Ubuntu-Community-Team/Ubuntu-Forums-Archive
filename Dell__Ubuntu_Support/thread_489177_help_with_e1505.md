### Post by PaulFr on 2007-07-01
Two things come to mind: you can try out Ubuntu before installing by booting from the CD (this will load Ubuntu into memory and use files from the CD, but not install it), and you can also take a look at **[https://help.ubuntu.com/community/SwitchingToUbuntu/FromWindows](https://help.ubuntu.com/community/SwitchingToUbuntu/FromWindows)**. 

Once you decide to go ahead with Ubuntu installation to the hard disk, Ubuntu's installer will try to reduce the Windows partition size, to make some room for the Ubuntu installation. It will also add Windows to the boot options, so at any time you can boot either Ubuntu or Windows.

For lots of details regarding testing with your laptop model, head over to **[https://wiki.ubuntu.com/LaptopTestingTeam/DellInspironE1505](https://wiki.ubuntu.com/LaptopTestingTeam/DellInspironE1505)** You need to look at the Feisty column for the results for the latest version of Ubuntu.

Hope this helps.
