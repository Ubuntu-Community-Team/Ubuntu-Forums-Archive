### Post by sebald on 2007-08-24
Well, there are ways to get Feisty working, just search the forums, but after all that they're saying that just skipping to Gutsy alphas solves most of it with less fiddling. I have a dual boot working reasonably well with Gutsy Tribe 5, alternate install used  (Kubuntu version), finds 1440x900 video and both network interfaces just fine. But somehow it didn't find my Vista partition to add to the grub boot list, that only took some editing to add back (fortunately I've done this before :mad: ...

One review said that the out of box experience with Ubuntu preinstalled (1420n) was imperfect anyway, taking just as much fiddling and installing of packages as your own install. Looks like Gutsy is the one that will strive for true "just works" functionality for the new Dell machines.:popcorn:
