### Post by camarojones on 2007-06-24
10 GB is fine for a Linux install.


Throw in the Ubuntu disk, and when you click install, follow the prompts. It will ask you about teh disk. Choose Resize and install using free space, and move the bar to where you have 10GB less than the maximum. The install automatically creates 2 linux partitions, the regular one and the swap one, in that new 10gb space.

Once the install is done, reboot and now you can choose what you want to run.

Enjoy
