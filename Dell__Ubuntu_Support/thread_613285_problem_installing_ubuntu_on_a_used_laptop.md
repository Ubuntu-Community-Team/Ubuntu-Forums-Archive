### Post by Dellfan on 2007-11-14
Can you post the specs of your notebook. Older processors are not an issue but lack of ram may be. I have Linux running happily on old 400 MHz Celeron based desktops.

If it is a base unit it likely only has 128 MB of ram which really is too little to run a current Linux with Gnome, which is a full featured and fair large footprint window manager. There certainly are "lighter" versions that will run great or you could upgrade to 512 MB (the max it supports, I think) and Ubuntu would run fine.
