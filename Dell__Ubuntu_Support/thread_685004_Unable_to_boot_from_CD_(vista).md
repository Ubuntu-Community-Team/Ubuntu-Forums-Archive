### Post by skierkegaard on 2008-02-01
I'm not sure exactly how your BIOS is laid out, but there are sometimes multiple screens that you may have to navigate to.  It may not go by the exact name "Boot Priority" either.  If it shows any kind of list for booting, hard drive, cd drive, that would be your screen.

### Post by CloudFX on 2008-02-01
I don't see one... You press ESC when the system is booting up to enter BIOS right? From there, the only two options are boot Windows Vista, or run System Memory Test. If you press F8 on top of booting Vista, a few more options appear, but none are to do with a CD boot.

### Post by skierkegaard on 2008-02-01
Usually you have to press a Function key (mine is F2) before the operating system boots.  If Vista gets going, you are too late.
