### Post by jnorthr on 2007-08-16
If your BIOS does not have the ability to boot from usb, then many of your choices are non-starters. If you can get ubuntu on a cd then that may be your best bet, as you can run it from the cd until you decide if you like it, then install it later. It runs slower from the cd but all the same functionality is there. Then if there are ny hardware issues you can find fixes for them before install or give the idea a miss.

If you have a floppy drive as your main source of external media it may be possible to boot from there once ubuntu is installed. You may need to look over these forum pages more to see how it's done. Think i have seen it in a previous post but cannot provide you with a link just yet.

Just a thought, if you can get ms dos to boot then you may be able to use a ubuntu cd in your external drive by starting it in dos.
