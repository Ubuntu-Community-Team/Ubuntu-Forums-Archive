### Post by mtbsoft on 2008-04-22
In theory, I belive, it is possible to make VMWare map down onto physical drives but I don't think you could do exactly what you want even then.

An alternative might be to install a bare bones XP then run your "real" XP in a VM which would therefore be accessible from both Linux and Windows.

That said, what is your reasoning for using the XP install anyway?  Since you are suggesting you would be using the same software in the real and the VM worlds, this suggests to me that you would be OK with just a Windows VM running under Linux.

Just a thought.
