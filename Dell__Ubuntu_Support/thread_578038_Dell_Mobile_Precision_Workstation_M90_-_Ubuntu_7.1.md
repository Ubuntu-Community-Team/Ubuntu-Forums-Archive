### Post by JG53_Jaguar on 2007-10-16
I decided to get Ubuntu 7.10 RC on my M90 laptop. The first thing I did was to buy 200GB SATA Hitachi 7200RPM drive with 16MB of cache so that I could have two partitions: XP and Ubuntu.Then I installed XP and dedicated about 160GB to it. I then installed 7.10 RC from CD image. Everything went very well...I was very impressed. Once I finished the installation I have downloaded the proprietary nVidia drivers and about 90 ubuntu updates :) Enabled the extra fancy gnome stuff;  it's nice to see my machine running at its full potential! I don't use hibernation so I didn't test that (I never use it). I also don't have wireless router so I didn't test to see if my M90 wireless network card to see if it is working; I also down know if my memory card reader is working fine....but so far so good!

Good job guys !!!

Next step is to get a decent C++ setup including IDE and start working on some gnome applications.

I'm assuming that once 7.10 final is out I will be able to updated my RC to final build...correct ?

btw my laptop specs:

M90 Precision Mobile Workstation, Cure 2 Duo 2Ghz, 2GB of RAM, FX2005M 512MB video card, 200GB Hitachi 7200RPM drive
