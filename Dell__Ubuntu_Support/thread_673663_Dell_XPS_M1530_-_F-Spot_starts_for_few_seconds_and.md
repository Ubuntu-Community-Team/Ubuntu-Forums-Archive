### Post by x001 on 2008-01-20
Hi all, I installed Ubuntu 7.10 (used the dell factory reinstall dvd) on my new xps1530 and everything seems to work very well. The only problem I experienced is with f-spot. I initially imported my photos into f-spot (about 3gb worth) then I attempted to remove couple tags that were attached to the photos. F-spot crashed. Now when I attempt to start it, f-spot seems to start for about a sec then just disappears.

Anyone know how I can resolve this?

### Post by x001 on 2008-01-21
Fixed: removed the photos.db from ~/.gnome2/f-spot/ folder then renamed the Photo folder to Photo_bak. Started f-spot and reimported all the photos. Seems to work now.
