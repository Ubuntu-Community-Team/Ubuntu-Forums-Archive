### Post by kevinatkins on 2007-12-04
> I advise using the latest version of Ubuntu, 7.10. An all-Intel laptop should work with no problem at all.

Um, not necessarily - I've got two laptops, both with Intel chipsets, and one of them is a Dell Inspiron 2500. On the Dell, the Live CD didn't work, exhibiting exactly what you describe, and the other laptop didn't work at all.

The problem is incorrect screen resolution, causing a blank screen. Once the system has booted and you've heard the Ubuntu startup jingle, try pressing <Ctrl>-<Alt>-<+> or <Ctrl>-<Alt>-<-> (ie, plus or minus) which *might* switch to a resolution your laptop can display.... If that doesn't work, you will need to edit a file.. post back..
