### Post by tgm4883 on 2008-01-19
> **natehall said:**
> Why would Dell choose to preload a 32 bit system on an Intel 64 bit chip ? (The Inspiron 1420N I'm using right now for example). Or for that matter not even have the preloaded Linux option for the AMD chip?

Ignorance.  Use the AMD64 build if you have a Core 2 Duo or an AMD X2.  If you have a 32-bit processor, then you are stuck with 32-bit Ubuntu.

I'd also point out that anyone that says 32-bit is easier than 64-bit most likely doesn't A) Own any 64-bit hardware and therefor are talking purely on speculation, or B) runs 32-bit Ubuntu on their 64-bit hardware and are therefor talking purely on speculation.  

The few people that have run 64-Bit Ubuntu on 64-bit hardware and still do not recommend it usually have not tried to do it since Hoary.  As long as you use Gutsy or later it should be just as easy to run 64-bit as it is to run 32-bit.  On a side note, 32-bit Ubuntu isn't always a cakewalk either.
