### Post by yojimbosteel on 2008-07-18
Is this after you've installed Ubuntu or during? I also just got an XT and I've heard that the installation only works with the alternate CD so I'm assuming you are using that plus the video driver from ATI's website. This aside, Dell's cards are Broadcom wireless cards which have proprietary drivers only, this means that Ubuntu legally can't include them on the installation disc itself so you'll need to find a wired connection somewhere to connect to the repository and install the Broadcom packages. I don't know what the packages are called but I believe there is a restricted drivers program in Ubuntu that will guide you through the process of getting your wireless card working. Hope that helps a little.
