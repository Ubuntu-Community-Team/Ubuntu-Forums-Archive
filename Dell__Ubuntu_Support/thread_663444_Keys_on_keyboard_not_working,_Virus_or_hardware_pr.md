### Post by telenorx on 2008-06-28
I finally got it fixed. It was really a horrible virus. This is what i did:

1. Remove the CMOS battery and wait
2. Format the hard disk
3. Flash the BIOS using a floppy (downloaded from dell site)
4. Remove the RAM
5. Reboot
6. Set a password for BIOS
Apparently it seems that the virus stays in the BIOS as well as on the hard disk.
I think setting the password for BIOS pervents viruses from messing with your BIOS.
