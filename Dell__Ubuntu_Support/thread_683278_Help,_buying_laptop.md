### Post by jsonder on 2008-01-30
I was doing fine until you added WOW.  The 1420N is a fine workhorse if you aren't demanding high end graphics.

The XPS M1330 is a road warrior notebook.  My middle son likes his, but he got it when Dell bought the company he works for.  They sweetened the pot a bit for the employees.  The UK markup is hideous!

The inspiron 6400 is similar to the inspiron 1505 that I've been running Ubuntu on since it arrived 21 months ago.  It was an $800 special code item in the sunday paper.  All that I've done to it is double the memory 2 months ago when memory got cheap.  It travels in a backpack, especially when I'm on the motorcycle.

If you can ignore the gaming, just get a basic notebook but be sure that it has the Intel 3945 wireless.  That is well supported.

good luck
