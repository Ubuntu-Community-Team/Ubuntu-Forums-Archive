### Post by Rocket2DMn on 2007-07-20
I think this forum was originally created because Dell was starting to sell computers with Ubuntu which meant there could be an influx of dell-specific questions.  Of course anybody with a dell computer can post here, though.
I have a dell laptop that is a few years old which I run Ubuntu on, and there are hundreds or thousands of other dell pc owners running Ubuntu on their machines.
I have heard of Dell's tech support guys searching these forums for solutions to problems they don't have immediate answers to.  However, I don't believe they run the forum themselves.

The differences between OEM Ubuntu and a user install should be minimal - they will just sell PCs with parts they know are supported in Ubuntu and can troubleshoot.  It's the same operating system.

### Post by rscott5 on 2007-07-20
The difference is that Dell includes a recovery partition for easy reinstallation of Ubuntu. Also, on two of their Ubuntu machines, the XPS410n and one of the laptops, there is a bug where the computer hangs on reboot, but that it fixed by passing a simple command to the Linux kernel on boot up. Those Dells with Ubuntu preloaded have the fix already put in, while I bought a regular XPS410 and had to put it in myself. Other than that, they are just regular installs of Ubuntu Feisty.
