### Post by Ek0nomik on 2007-08-10
It is probably related to your video card.  Do you know what kind of card it is?

You can also install via the Alternate CD instead of the Desktop (Live) CD.  It's a text based install so it won't rely on your video card nearly as much (so long as it's plugged in you should have a display).

[http://www.ubuntu.com/getubuntu](http://www.ubuntu.com/getubuntu)

Below the green, 'Start Download' button there is a checkbox to get the Alternate CD instead.  Then you just need to burn it to a CD and run it like you were before.
