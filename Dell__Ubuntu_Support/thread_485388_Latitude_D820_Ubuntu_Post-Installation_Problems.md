### Post by dugh on 2007-06-27
I also have a D820 (with nvidia card).  I haven't tried an external/second monitor with it yet, but apparently you need to enable twinview in your /etc/xorg.conf file, see this tutorial:
[http://www.ublug.org/ubuntu/twinview/twinview-howto-breezy.html](http://www.ublug.org/ubuntu/twinview/twinview-howto-breezy.html)

About the wireless card, yeah it didn't work for me either at first.  Dell's drivers for the 1490 wireless card in your laptop are windows-only and proprietary, so you need to use ndiswrapper to make it work (although it still poops out on me every once in a while).  Use this tutorial to get ndiswrapper working with your wireless card:
[http://ubuntuforums.org/showthread.php?t=297092](http://ubuntuforums.org/showthread.php?t=297092)

You need a wired/ethernet connection to download the required software mentioned in that tutorial.  But, those instructions will work even though they are for another Dell laptop (including the part about downloading the R151517.EXE file from dell's site), but they do require use of the terminal / command line.

The only minor change you need to make to those instructions is download ndiswrapper 1.47 (which was just recently released) instead of 1.46:
[http://sourceforge.net/project/showfiles.php?group_id=93482](http://sourceforge.net/project/showfiles.php?group_id=93482)


.....

Let's see, the only other issue I've had is with the obnoxious loud beep coming from the little speaker on the motherboard.  You can disable that by going to the menu System->Preferences->Sound->System Beep and uncheck 'enable system beep'.  

Also in the terminal / command line application, go to Edit -> Current Profile... and uncheck 'Terminal Bell'.  I also added this line to the end of the /etc/bash.bashrc file but I don't know if it's necessary:
setterm -bfreq 0
