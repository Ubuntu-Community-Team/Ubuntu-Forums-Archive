### Post by smithno on 2007-08-15
I have a 1420n. I used the Gnome package manager to download and install KDE. Then reboot. At the login screen, you can click on the Options icon and select the window manager to run. I think it was "Options", it's been a while and once I switched to KDE, I've never run Gnome again. The advantage of doing this instead of loading Kubuntu is you keep the base Dell install with the proper drivers, etc.
