### Post by Yoooder on 2007-06-21
just a random tidbit regarding LightScribe:  If you find the labels to be too faint for your liking, you can actually burn the same label to a single disc multiple times, and it will result in a higher contrast.  I suppose there is the possibility that it may affect the data-life of the CD if it burns the images darker than the manufacturors intended--but that's just speculation.  I've done it for a couple Ubuntu CD's I've handed out to friends (tested them first, they worked )
