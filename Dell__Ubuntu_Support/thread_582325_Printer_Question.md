### Post by tturrisi on 2007-10-19
It's not that printers are "unfriendly", it's that for a printer to be able to work it needs a driver.  Any op sys can detect and identify any hardware connected to the comp IF the hardware has an ID in a database somewhere.  Your printer is detected but has no driver installed for it, thus you can't run the printer.
