### Post by Rumplesmigskin on 2007-08-14
Be careful with bcm43xx-fwcutter: the link from which it downloads wl_apsta.o doesn't seem to be working any more. This will cause it not to install properly. ([http://boredklink.googlepages.com/wl_apsta.o](http://boredklink.googlepages.com/wl_apsta.o) is the link in question, 404's abound...)

What you might do (which is what I myself did) is to google wl_apsta.o - look for anything that looks like a link, for example: [http://svit.epfl.ch/stuff/wl_apsta.o](http://svit.epfl.ch/stuff/wl_apsta.o). 

Once you have one, download and save it. 

Then do: 
```
foo@bar:~$ sudo bcm43xx-fwcutter wl_apsta.o -w /lib/firmware/`uname -r`
```

Then you should probably do ```

foo@bar:~$ sudo aptitude -f install 
```
bcm43xx-fwcutter should then install without problems. 

However, I have been hearing things about 4318's and bcm43xx-fwcutter - so if this doesn't work for you, maybe consider using ndiswrapper instead.

Hope this helps!

### Post by jackcy on 2007-12-09
Kubuntu 7.10 gutsy gibbon

Activating wireless lan on an ASUS A6000 with bcm4318 chip was really easy.
I simply installed the firmware with 

sudo aptitude install bcm43xx-fwcutter

reloaded the module with 

sudo modprobe -r bcm43xx && sudo modprobe bcm43xx

and wireless lan was activated.
