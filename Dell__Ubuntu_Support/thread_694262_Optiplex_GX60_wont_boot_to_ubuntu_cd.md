### Post by gonesolo on 2008-02-29
MIMO most likely hit the nail on the head.

Windows CD burning software will not burn an ISO as anything other than the ISO file. 

To create a bootable CD you need to use another program. Nero should handle it with no issues.

The PC itself may need a BIOS update so check support.dell.com to see if any updates are available (you'll be surprised how many issues this fixes)
