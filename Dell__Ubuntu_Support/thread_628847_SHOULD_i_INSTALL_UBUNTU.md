### Post by Skuzniak on 2007-12-01
There is no reason not to install Ubuntu whatsoever, unless you have limited hard drive space. Pop in the live CD and see if you like it. If you do, go ahead and install it. Just be careful not to use the MediaDirect button if you install GRUB, chances are it will destroy your partitions and cause you endless amounts of grief.

As a person that recently switched from XP, I found the switch to be incredibly easy. Most people would call me computer savvy. The main thing that surprised me about the switch was that Ubuntu actually integrated better with the other Windows computers on the network than Windows ever could.

### Post by LinuxLuver29 on 2007-12-02
I have Ubuntu on the same exact Dell system that you have.  All I had to do was delete my MediaDirect Partition and my Recovery Partition (I don't need it since I have the install media.)  I had to install my wireless and that's pretty much it.  I dual boot.  I have a 20GB Windows Partition and a 20GB Ubuntu partition and the rest of the space I made a FAT32 partition(Windows and Linux can access it so I made that my data partition where I put all my music and movies and things like that.)
I would definately install it!
