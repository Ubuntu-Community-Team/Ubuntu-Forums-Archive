### Post by spartan777 on 2007-08-19
the only way you can possibly change the boot sequence is in the BIOS. there is no other way around it. it sounds like you are working on a corporate/business laptop. the techs probably changed the computer to only boot from the hard drive and have locked the BIOS with a password (they do this so unwitting people don't put some cd/floppy that has a virus in- I'm assuming). The only thing you can do is to ask your help-desk people to help you out.

Did you install ubuntu over windows? this might be a problem- you aren't supposed to do that. everything on windows would be lost.
