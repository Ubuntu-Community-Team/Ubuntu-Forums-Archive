### Post by lazymints on 2007-09-19
run check disk on your drive, google fsck for syntax

ndiswrapper does work on my dell 1350 wireless card in a 9100 inspiron.  i have been using it since fedora core 2 or 3 when i got the laptop.  you need to get the latest dell driver for windows xp and install it then copy the two files bcmwl5.ini and bcmwl5.sys to a location where you can access them in linux then use these for ndiswrapper.

amarok is a kde program and doesn't seem to play nicely with gnome, try exaile which is gtk+ so fits in to gnome better.

can't help with wine or games, don't use them

i know this may be contentious but linux mint 3.0, which is based on ubuntu, found all the hardware in my inspiron 9100 and configured it properly.  you might be better off scrapping your current install and putting mint on if ubuntu is causing too many headaches
