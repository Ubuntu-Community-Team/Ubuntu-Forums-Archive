### Post by AlexThomson_NZ on 2008-04-01
I got mine yesterday, blew Vista off it straight away, and tried Gutsy.

There were some graphics glitches (I got a 24" Dell and a 8800GT) booting Gutsy, and gdm wouldn't start at all (both in default and safe mode).  Feisty did work however using the vesa driver, so have installed that for now, and will install the official nvidia drivers when I get home tonight.

I noticed the onboard ethernet device is an aetheros (spelling?), so requires dodgy non-free drivers, and didn't work for me.  Luckily my d-link wireless when I plugged it in worked out of the box.

Interestingly, I tried to reinstall Vista and XP (out of curiosity)- Vista stopped at the step asking where to install to, and XP blue-screened early on (both problems are probably caused by the RAID setup).

Overall my first impressions were- Vista looks quite nice (to be fair), and everything seemed to work ok with it (although, because it was the 32-bit version, it couldn't see all 4Gigs RAM).  Linux on this box is let down somewhat by: Ethernet, NVidia card, and (I suspect) soft-RAID.  The supplied modem wouldn't surprise me if it was a winmodem either (can't confirm).  I think when I sort out these issues though I'll have a killer box!
