### Post by paddy1978 on 2008-02-13
Hi,

I've just got an XPS m1330 - Windows Vista version as the Ubuntu one wasn't announced until after I had ordered!

It was easy to install Ubuntu on it. I didn't use the Dell Ubuntu DVD, I just used a standard Ubuntu CD and then applied the 'Add-on' packages and fixes listed in 'Known Issues' on this page: [http://linux.dell.com/wiki/index.php/Ubuntu_7.10]("http://linux.dell.com/wiki/index.php/Ubuntu_7.10")  I think these are the only differences between the Dell install and the standard Ubuntu CD. 

I actually kept the Vista partition and the Dell 'Media Direct' partition  (just in case...haven't used them yet though...) and used the standard installer to put Ubuntu on a different partition.  Instructions for this are here: [http://www.daryl.mu/2007/12/29/howto-triple-boot-dell-xps-m1330-with-vistaubuntu-and-media-direct/]("http://www.daryl.mu/2007/12/29/howto-triple-boot-dell-xps-m1330-with-vistaubuntu-and-media-direct/") (N.B. I didn't delete all the partitions as it suggested, I kept one, the Dell Utility one which has system diagnostics, etc.)

I've not had chance to use the laptop much yet but it's really nice and everything seems to work OK on Ubuntu (though I haven't tried HDMI output or fingerprint reader).

As to the SSD drives, I'm afraid I don't know anything about those... 

Hope this helps
