### Post by Archangelos on 2011-11-10
If you want a Dansguardian GUI that actually works and makes Dansguardian do what its supposed to do, there is only one that I have ever found that will consistantly do just that -- Webcontentcontrol.  So if you are running an 64 bit arch then   1) Install Tinyproxy and Fifehol from repositories.  2) Download and install Dansguardian amd64.deb here  [http://launchpadlibrarian.net/31288865/dansguardian_2.9.9.7-2.1ubuntu1%7Eintrepid2_amd64.deb](http://launchpadlibrarian.net/31288865/dansguardian_2.9.9.7-2.1ubuntu1%7Eintrepid2_amd64.deb)  3) Download and install Webcontentcontrol amd64.deb package here  [https://launchpad.net/~webcontentcontrol/+archive/webcontentcontrol/+build/1890267/+files/webcontentcontrol_1.3.7-0ubuntu8_amd64.deb](https://launchpad.net/~webcontentcontrol/+archive/webcontentcontrol/+build/1890267/+files/webcontentcontrol_1.3.7-0ubuntu8_amd64.deb)  Install that and press start. You're done.  The only thing left to do is change filtering levels and banned extensions in the GUI.  And no, I have never found any other setup from manual to GUI that works. This is the only setup that works at a GUI content filter app using Dansguardian. Period.  Cheers.
