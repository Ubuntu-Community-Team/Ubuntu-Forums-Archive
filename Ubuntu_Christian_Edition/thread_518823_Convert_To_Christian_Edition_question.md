### Post by mysticrider92 on 2007-08-06
The convert_me script will simply add some different theme parts (bootplash, login screen, wallpaper, panels, firefox theme, and a few other little things), add a few programs, including Dansuardian with a GUI (and the related programs), a few Bible-related programs, and gdesklets with a Bible verse RSS feed. So all of the changes can be reversed, with a bit of work.

Also, just as a note, Ubuntu CE is not really a different version of Ubuntu in a technical sense, it is just some add-ons. CE v3.* is still Ubuntu Feisty.
