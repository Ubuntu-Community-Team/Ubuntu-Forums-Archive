### Post by henjj2 on 2016-01-11
Just wanted to start a discussion / post a little info on accountable2you (kind of a continuation of this thread: [http://ubuntuforums.org/showthread.php?t=1989331](http://ubuntuforums.org/showthread.php?t=1989331)).  

Some people are interested in being able to implement some type of filter on their computers/devices to block certain content for a variety of reasons (young kids in the house who are starting to use computers, kiosk computers, work computers, etc.).  Others don't necessarily want a filter, they would rather have some type of accountability/monitoring software that logs / alerts what is going on with a computer/device.  

It seems there are a couple of different solutions out there for accountability / monitoring software (Covenant Eyes, net responsibility, any others?).  Covenant eyes charges a monthly fee and only works with windows, mac, and android. Net responsibility is free and works with linux but I'm not sure if it will be continuing as a project (see [http://forum.netresponsibility.com/discussion/171/no-activity-since-2014-is-this-still-viable-software#latest](http://forum.netresponsibility.com/discussion/171/no-activity-since-2014-is-this-still-viable-software#latest)).

Accountable2you is a paid accountability service like Covenant Eyes but the nice thing about Accountable2you is that it now works with Ubuntu (it is also cheaper...$4.99/month as opposed to $9.99/month). Accountable2you is just an accountability / monitoring service (no filtering).  

I have been using it for a few months in our house on multiple devices (Android and Chromebook, now testing it out on Ubuntu).  You can set keywords for instant text alerts.  It seems to effectively monitor and alert on user defined activity.  

I had emailed the developer when I initially learned of the service asking if they planned on supporting Linux.  He emailed back saying that they were working on a version for Ubuntu and that it would be coming out towards the end of 2015 / beginning of 2016.  I was surprised when I got an email after the new year saying that they were rolling out support for Ubuntu 14 and 15.  Hats off to the developer for bringing support to Ubuntu.  

I remember reading about requests for Covenant Eyes to support linux to no avail.  If you are searching / looking for something like Covenant Eyes that works with linux, take a look at Accountable2you.  

Has anyone else been using / used Accountable2you?  Any thoughts?  Am I missing any other monitoring services that work on Linux / across multiple platforms?

Looking forward to a discussion.  Thanks!
