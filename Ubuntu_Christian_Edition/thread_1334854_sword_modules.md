### Post by jonathonblake on 2009-11-24
> **rotata said:**
> Where can I find alternative translations and modules for Xiphos? 

A list of repositories for _The Sword Project_ can be found at ">Edit >Module Manager >Sword >Sources"  (Alternatively, hit <F4>, and then select "Sources").

There are more officially released resources for _The Sword Project_, than for e-Sword. On the flipside, there are far more user created resources for e-Sword, than _The Sword Project_.
