As for getting rid of the virtual rosary (I'm with you on that. :))  I just delete it from the script so it never gets installed.  Open up **System -> Synaptic** and then search virtual rosary and then remove.  If it doesn't show up in there (I'm not sure how it is registered in the system)  You could just right click on the start button, and select edit menus. Then click on Accessories, then uncheck the virtual rosary, that will at least remove it from your menu listings.  Let me know if it is there in the synaptic listings.
