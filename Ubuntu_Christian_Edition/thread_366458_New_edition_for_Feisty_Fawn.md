Yes, once Feisty has been released all development will be focused on what will be Ubuntu CE v3.x (Feisty). This version will replace the current Ubuntu CE v2.x. (Edgy). The Ubuntu CE v1.5.x (Dapper) version will continue to be maintained as long as Dapper is still the LTS release of Ubuntu.
