### Post by HareBall on 2007-01-13

---

### Post by aysiu on 2007-01-13
An email attachment's associated program is usually defined by the email client (Evolution, Thunderbird, KMail).

In general, though, you can right-click the .wmv file (in the file browser, not the email client) and select Properties > Open With and make sure the black dot is next to the program you want (VLC, for example).
