[QUOTE=nitep]
At the end you can download the power point version [/quote]

If you are going to advertize here, you could at least do one of the following:
* Provide data in a file format that meets ISO 26300 standards;
* Provide data in a format that can be opened/run as anticipated by the data creator on Linux;

Powerpoint does not natively run on Linux.
Powerpoint does not meet any part of the ISO 26300 specifications.

Linux apps that can open Powerpoint files do **not** render them the same way as they are rendered on Windows using Powerpoint.
