### Post by david_kt on 2009-10-19
After a few months of testing and received many feed backs from you, I have made some improvements and I think it is time to launch a production release.

Major improvements has been done on dansguardian gui such as not-so-restrictive default settings, full indication of status, and more options on filter settings. I have also removed firehol and replace it with custom made firewall.

I have also added wine christian repository, which contain selected windows program to run it through wine. As wine itself is still progressing, this repository also still need continual improvement. But as I have maintain e-Sword installer for more than 2 years, I believe I could support more windows programs through wine christian repository.

My approach on wine christian repository would implement multiple wine version as different windows program sometime require different wine version due to regression.

On the server side, I have fixed some bugs related to Christian Website in a Box, and now should be fully functional out of the box.
