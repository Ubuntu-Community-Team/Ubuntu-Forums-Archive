### Post by mysticrider92 on 2006-12-21
There currenty isn't anything in the repos, but there is a script that you can use to convert to Christian Edition. You can download it [ here]("http://www.whatwouldjesusdownload.com/christianubuntu/2006/11/ubuntu-ce-v20-edgy-downloads.html") for Edgy or [ here]("http://www.whatwouldjesusdownload.com/christianubuntu/2006/11/ubuntu-ce-v151-dapper-downloads.html") for Dapper. Just look for the line that says Convert, download it, untar it and open the file "Start" and tell it run in terminal.

As for all of the packages, they are found in various repository catagories, but I don't know the exact ones.
