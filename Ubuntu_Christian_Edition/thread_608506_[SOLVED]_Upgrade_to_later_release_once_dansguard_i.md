### Post by IanB2 on 2007-11-12
I believe that I've found an answer to this.

It would appear that if you use the CE version of Dansguardian, then you are locked in to the CE update release cycle.

However, the Dansguardian program is available through the main Ubuntu repositories see [URL="http://https://help.ubuntu.com/community/Servers/DansGuardian"] 

If you use this source, then you can use the normal Ubuntu upgrade path through Update Manager. The only disadvantage is that you don't get the nice GUI that the CE version provides, however the .conf file is well commented and easy to follow.

Hope this is helpful.
