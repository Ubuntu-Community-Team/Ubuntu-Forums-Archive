### Post by mysticrider92 on 2007-01-02
There aren't any official cd's available right now for CE, but if you have some internet access, you could order cd's for regular Ubuntu and use the convert_me script. Depending on where you are, someone may be able to burn and mail you a cd though. There are no update cd's for Ubuntu, you either need to do a clean install or use the internet to update.
