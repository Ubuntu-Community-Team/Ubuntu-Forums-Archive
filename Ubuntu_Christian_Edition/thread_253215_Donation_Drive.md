### Post by mhancoc7 on 2006-09-08
**Edit: See next post for update on bandwidth issue.**


Let me first say thank you to everyone that is supporting this project. Without the support of so many wonderful individuals this project would not be possible. Let me also say that I really do not like to ask for financial support. I don't want to come across as trying make some financial gains off others support.

With that said, we are facing the need for a new server. When I started this project I never expected the huge reaction. It has been amazing how fast this project has grown. Ubuntu CE is currently ranked #99 on DistroWatch for the last 6 months and it has only been available for about a month and half. It is ranked #7 for the last 7 days. Since I really didn't expect this I initially hosted the project site on my WhatWouldJesusDownload.com server and accepted donated server space and bandwith for the iso downloads. I figured that with out the load of the isos it would be ok. Boy was I wrong! My sites traffic has jumped from an average of 100 unique people a day to an average of 7000 unique individuals a day.

Ubuntu CE along with all my other projects are funded out of pocket. Of course there are advertisements, but these have never really generated a significant amount of income. This is changing slowly due to the traffic increase, however it is not keeping up with the cost of bandwith needed to keep Ubuntu CE online.

I am humbly asking for any donations. They would be most appreciated. I hope you all understand that I would never ask for money if it was not absolutely necessary. The current need is $109.23 to get Ubuntu CE moved to a new server that should have plenty of bandwith. If you have any questions or want to know more about how the money will be spent please let me know. I plan to create a page that I can add to the project site to disclose all donations and expenses so everyone knows how the donations are being used.

If you want to donate, [click here]("http://www.whatwouldjesusdownload.com/christianubuntu/2006/07/donations.html").

Thanks so much and God Bless, Jereme
