### Post by Eladon on 2007-04-22
I figured it out.  You have to click on **Edit**, then **Advanced Search**.  Then under **Search Type** you have to choose "Attributes".  Then under the **Attribute Search** tab, make sure "Strong's Numbers" is selected.

As with certain other Bible programs, you have to be sure to type the search text in correctly.  In other words, to search for a greek number, you need to type in "G932" for Strong's # 932, whereas to search for a hebrew number you need to type in "H0932" (note the 0 in front of the strongs number, only for hebrew searches).

Sadly, GnomeSword is a poor study tool for this sort of thing, as these kinds of searches are very slow.  Plus the results don't even show which of the words in the verse is the Strong's number you were searching for (The "Show Strong's Numbers" option under Attribute Search doesn't seem to do anything for me).  And to make things worse, there is no simple way to get the context of any of your results (no context button, can't double click on a result to open it in the main program), so you're forced to manually look up each result in the main program.

So looks like back to e-Sword for me!  :roll:
