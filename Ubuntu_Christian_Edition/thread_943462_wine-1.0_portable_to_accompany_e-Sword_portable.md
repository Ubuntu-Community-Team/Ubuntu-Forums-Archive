### Post by david_kt on 2008-10-10
I have just created a script to compile wine-1.0 portable to be used together with e-Sword portable.

But compile wine-1.0 portable, you have to run:

```
sudo apt-get build-dep wine
```

and then you could launch the script and choose to install wine.

This is additional chance for you to launch e-Sword portable, if the host computer do not have wine installed. Anyhow, if wine dependencies are not satisfied, it would still fail to run.

Please check below thread:

[http://ubuntuforums.org/showthread.php?t=404042](http://ubuntuforums.org/showthread.php?t=404042)
