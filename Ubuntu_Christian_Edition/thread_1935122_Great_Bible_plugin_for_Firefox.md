### Post by forrestcupp on 2012-03-03
I know this isn't specifically about Ubuntu CE, but I found a pretty nice extension for Firefox.  It's called [Bible Refalizer](https://addons.mozilla.org/en-US/firefox/addon/bible-refalizer/?src=search). It scans the web page you have opened for Bible references, and turns them into links where you can read the passages. Pretty handy.
