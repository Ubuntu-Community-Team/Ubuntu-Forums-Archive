Ubuntu Christian Edition can be installed straight over the top (there is no such thing as "uninstalling" an operating system).

When you boot the Ubuntu CE disc and choose Install, you will be given the option of erasing Ubuntu and installing Ubuntu CE.

However, remember that Ubuntu CE is just regular Ubuntu with a few added apps. You can simply install those apps from Ubuntu Software Center. That may in fact be better, because the copy of Ubuntu you have may be slightly modified to run better on that computer. Ubuntu CE would not have those modifications.
