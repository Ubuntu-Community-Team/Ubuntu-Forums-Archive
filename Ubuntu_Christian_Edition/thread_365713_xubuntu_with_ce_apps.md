So you are asking what application an configuration is what makes Ubuntu CE different from a regular Ubuntu.

Let's see...

[DansGuardian]("http://dansguardian.org/"). A web content filter

install by:

```
sudo aptitude install dansguardian
```

[BibleMemorizer]("http://biblememorizer.sourceforge.net/")

[GnomeSword]("http://gnomesword.sourceforge.net/") Bible Program for GNOME (could be used for other DE as well)

```
sudo aptitude install gnomesword
```


[Bible Time]("http://www.bibletime.info/")
 Bible software for KDE

install by
```
sudo aptitude install bibletime
```
