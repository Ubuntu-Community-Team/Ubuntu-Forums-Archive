If you plan to use the convert_me script to install Ubuntu CE to your current Ubuntu installation then you should have no problems with the items that you mentioned.

Now let me caution you that you should read all of the instructions and warnings before using the script. 

You will be installing parental filtering which may require some post installation tweaking for sites that you frequent that may be blocked. The filtering is quite effective and can be easily configured with the provided GUI.

The rest of the changes are mostly the installation of packages from the repos and some cosmetic changes. 

If you have any questions please feel free to ask.
