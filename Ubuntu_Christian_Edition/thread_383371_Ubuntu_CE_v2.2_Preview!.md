Thank you as well.

WIth the release of Ubuntu CE 2.2 you will have two options of changing the dansguardian settings to a state that will allow mp3 downloads. You can use the Basic tab to set the filtering to "Moderate". You can also use the Advanced tab to edit the "Banned File Extensions" conf file. Just put a # in front of the entry for mp3s and save the file. Then click "Save & Exit". That should do it.
