### Post by beta.tester on 2009-07-17
Hi Guys

Just another little problem, this time with Opensong. When run for the 1st time it fails and aborts. I have discovered it is looking for:

/home/OpenSong which it fails to create or find.

Solution: Open Places => Home Folder and create a new folder called: OpenSong

Now go into OpenSong program and open new :) bingo :)
