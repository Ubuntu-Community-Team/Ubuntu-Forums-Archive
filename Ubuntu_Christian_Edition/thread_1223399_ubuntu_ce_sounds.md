I have added ubuntu-ce-sounds.
After installing, you have to go to:

System > Preferences > Sound

Click sounds tab and on Sound Themes drop down box choose ubuntu ce.

Please note that I have moved the repository to:

For 64 bit:
deb [http://ubuntuce.com/repos/Ubuntu_CE/](http://ubuntuce.com/repos/Ubuntu_CE/) jaunty_amd/

For 32 bit:
deb [http://ubuntuce.com/repos/Ubuntu_CE/](http://ubuntuce.com/repos/Ubuntu_CE/) jaunty_i386/
