### Post by refdoc on 2010-01-06
Someone on another thread asked about the ability to quote the Bible directly in OOo e.g. 

The same thing came up in CrossWire-s sword-devel list and a simple rough and ready solution was posted [[http://sites.google.com/site/adyeths/theswordproject](http://sites.google.com/site/adyeths/theswordproject) here]. It uses Autokey (you need the 0.6 version form the autokey ppa) and seems to do the job reasonably well. I have myself only started testing it.

The Bible texts used are your CrossWire modules installed e.g. via Xiphos or Bibletime.

I am sure (and so was the author) that the programme can be made much better. For starters it uses the diatheke instead of using directly libsword python bindings, but there is ample scope for anyone interested to drive the matter forward.

### Post by cdekter on 2010-07-06
