### Post by jonathonblake on 2009-04-22
> **refdoc said:**
> Ubuntu CE could have been a great resource if you had made some serious effort to get the actually existing Christian Linux software 

I don't think that Jeremy is to blame for that. He set up a few specific goals for Ubuntu Christian Edition,and stuck to them.

The majority of Christian software for Linux is so lacking in functionality, that packaging for Ubuntu Christian Edition requires a full time commitment to writing code that will provide roughly equivalent functionality as the Windows program.

For Bible Study, there is nothing in the FLOSS world akin to either Accordance, or _Libronix Gold Scholar_. There isn't even anything in the FLOSS world that corresponds to BibleWorks 7.0, which is an entry level Bible Study Program.

Where is the FLOSS equivalent to _Sermon Builder_?  
Or of _Ministry Notebook_?

_Church Info_ doesn't have the features, or functionality that _Membership Plus_. offers.   Furthermore, how many non-techies can install, and then use ChurchInfo? (The sad thing is that _Church Info_ is probably the best of the FLOSS church management programs.)

Functional FLOSS worship support packages are also missing.

Had a coalition of developers around Ubuntu Christian Edition congealed, then maybe some of those programs/packages would have been written/updated, and included in the Ubuntu repository.

jonathon
