### Post by shane2peru on 2006-11-16
Jim,

   I'm not really familiar with either of these programs.  What kind of an error are you getting?  Can you boot from the CD?  My guess is that you are burning the iso file as a data onto the disk.  An iso file is a picture of what needs to be burned onto the disk.  There is a special way to burn iso's.  Here are some instructions I found on burning iso's with Nero 6:  [http://www.ultimatebootcd.com/nero6.html](http://www.ultimatebootcd.com/nero6.html)  You can check these out.  If this is not the problem, you probably need to re-download the file, and then burn it again.  If you have any other questions or info, just ask.

### Post by mysticrider92 on 2006-12-12
When you use Nero 6, what happens when you boot the cd? One thing I had a hard time figuring out was why it was loading a version of DOS on the disk and booting that instead of the image I burnt. If you have this problem, try using Nero Smart Start. Go to the copy/backup section and select "Burn image to disk" and it should work properly.
