Thank you,

1) I do not know about privoxy, I have to take a look into it before I could answer you.
2) The ubuntu_ce_firewall has redirection to make transparent proxy (redirect port 80 to 8080). What you could do is edit it manually:

```
gksudo gedit /etc/init.d/ubuntu_ce_firewall
```

Copy all codes under "stop section" to "start section".

3) To turn off UCE splash, you could try below
```

sudo rm /etc/alternatives/usplash-artwork.so
```

Edit for privoxy:

If you use squid proxy, you could combine viproxy with dansguardian using below method:
[http://www.nyetwork.org/wiki/Privoxy](http://www.nyetwork.org/wiki/Privoxy)

```
Privoxy is configured to only accept requests from Squid, and Squid is forcely configured to use Privoxy as a parent cache. Edit /etc/squid/squid.conf to read:

    cache_peer 127.0.0.1 parent 8118 7 no-query
    # ....
    acl all src 0.0.0.0/0.0.0.0
    # ....
    always_direct deny all

```

If you use tinyproxy, may be you could try editing below:

```
gksudo gedit /etc/tinyproxy/tinyproxy.conf
```

Find below:
```
#  upstream firewall.internal.example.com:80
```

and change it to:

```
upstream 127.0.0.1:8118
```

Please let me know how it goes.
