### Post by Bodizzle on 2007-07-05
Here is a screenshot of the orange

This was done by using a screenlet.  Basically, a .png file similar to the one you created for the button is used with the top missing just like yours.  Then, to create the missing top part, a basic screenlet file was created to put an floating image above the menu bar that was then locked to the desktop.  

From what I have read you can also use gdesklets to do this in a similar way to your bible verse.  Instead of an RSS feed, you would load an image that would sit on top of the start button image.  

So, it creates the look of the vista style button but it is actually two separate images, on that sits in the menu bar, and one that sits above the menu bar.
