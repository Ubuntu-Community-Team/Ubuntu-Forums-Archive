Ubuntu CE 12.04 has been released!

Thanks to our new Lead Developer, Matthew Byers,(stlsaint) we have finally released a new version of Ubuntu CE based on Ubuntu 12.04!

Here is the latest release info:

What's new in Ubuntu Christian Edition.

- Built of the LTS Ubuntu 12.04.1
- Includes many applications for:

-System management
- Dconf-tools
- Synaptic package manager
- BleachBit
- gAlternatives
- gnome-tweak-tool "Advanced Settings"
- ubuntu-tweak
- bum
-Daily usage
- Wine and WineTricks
- Google Chrome
- VLC
- Y-ppa-manager
- MyUnity
- terminator
- nautilus-scripts for better file management
- Unity scops and lens to include:
- wikipedia lens, cities lens, and more
-Bible software
- Quelea
- bibledit
- verse
- xiphos
- openlp
-widemargin

Plus many more

Customizations ranging from Artwork, themes and icons along with the ppa's added to enable more artwork installation.

Along with all these new additions, UCE comes with the award winning DansGuardian Web Content filter configured out the box to block pornography and other questionable content.

This is Ubuntu Christian Edition like never before seen. Our most up to date and advanced release yet.

Download here: [http://ubuntuce.com/](http://ubuntuce.com/)

God Bless, Jereme
