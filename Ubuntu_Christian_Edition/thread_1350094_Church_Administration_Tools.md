### Post by funkydan2 on 2009-12-13
It probably won't do everything you want (particularly not donations) but possibly for you, and definitely for others who search this thread I reckon Jethro [http://jethro-pmm.sourceforge.net/](http://jethro-pmm.sourceforge.net/) is a great web based tool for organising your church contact database.

It's Australian, so it's not as focussed on donations as American programs (probably because donations to churches are not tax deductable in Australia, so we don't need to issue 'receipts') and thus it's more focussed on personal contact details and tools that aid in Christian Ministry.

I think the developers are working towards another program called 'Levi', which will integrate with Jethro to make organising services/rosters more streamlined.

### Post by funkydan2 on 2009-12-20
Hi David,

I have installed it before, but it was a while ago. My recolection is that I didn't have any difficulty.

The 'instructions' are to be found [http://jethro-pmm.sourceforge.net/about.php](http://jethro-pmm.sourceforge.net/about.php). Once you've created a MySQL database (which when I did it on a webserver I was able to easily do through cPanel), you just need to edit the config.php file and set about five variables (system name, url, and the database name, username, and password) then it should be easy from there.

If you have any problems, maybe you could ask in the Jethro forum or make a new thread here.
