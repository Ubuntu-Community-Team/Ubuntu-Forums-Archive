### Post by jonathonblake on 2011-08-15
> **not found said:**
> looking to supplement my Bible Study with some software

[http://www.bibletechconference.com/blog/selecting-bible-study-software/](http://www.bibletechconference.com/blog/selecting-bible-study-software/) i a blog I wrote about selecting Biblical software.

> would really like is to have a number of different Bible translations 

Both *BibleTime* and *Xiphos* run as native applications.  However, they may not have the versions of the Bible that you are wanting.

Both *e-Sword* and *TheWord* require WINE, in order to run.  They both offer Bible translations that *The Sword Project* has been unable to obtain permission to distribute. (I'll grant that in some instances, permission to distribute the *e-Sword* resource was not obtained.)

> and any other tools that may be useful (which I may not be aware of).

What tools you may find useful depends upon how you study the Bible.

If your typical Bible Study resembles *The 2009 SBL Bible Software Shootout*, then your options are:
[LIST]
[*]Use text files and grep;
[*]Write your own software;
[/LIST]
There is nothing that runs on Linux that comes close to doing an adequate job in fullfilling the criteria outlined in that shootout.
FWIW, only two Bible Software programs did all five criteria successfully.  *Olive Tree* could almost do them, but was hampered by the lack of applicable resources. None of those programs run on Linux, either directly, or using emulation software.

> and how have you enhanced it's functionality if applicable.

Write code that provides the data I'm looking for.
