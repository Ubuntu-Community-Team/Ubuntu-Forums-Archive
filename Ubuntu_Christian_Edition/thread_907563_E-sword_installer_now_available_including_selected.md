### Post by david_kt on 2008-09-01
As I see people miss E-sword from latest ubuntu CE but unsure to install it manually, I have created E-sword installer to make it easier to install E-sword.

The installer would do the following:

1. Download and install E-sword 798
2. Copy riched20 to correct location
3. Download msls31.dll
4. Set dlloverrides for riched20 and oleaut32
5. Install additional bibles and classical bible map, of your choice

All are done in special bottle (.wine_Esword).
If your favourite addon is not included in the installer, please let me know so that I could include it in the installer.

Please see below link:
