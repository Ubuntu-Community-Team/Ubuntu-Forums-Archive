### Post by david_kt on 2009-09-23
> **eldews said:**
> I have installed Ubuntu 9.04 and converted to Ubuntu CE pasting 'wget....' into a terminal and answered 'yes'. How do I start DansGuardian and get it working in a web browser?

1. Go to System > Administration > Dansguardian GUI
2. Type your password when requested.
3. Click autoconfigure/Reset Dansguardian and Tinyproxy

Please see the status on the top bar of the menu. Those three status should on. If one of the three indication inform it is off, then you should click 

Stop Dansguardian

 and then 

Start Dansguardian

The three indication should be on now.

DK
