Do you want to call these Bible apps?:
[LIST]

[/LIST]

Easy for users to install, once WINE is correctly installed and configured:
[LIST]
[*]There is thread ( *[COLOR="#000080"]http://ubuntuforums.org/showthread.php?t=404042[/COLOR]* ) that includes a script for installing *e-Sword*;
[*]Adding "[COLOR="#000080"]ppa:shane2peru/bibleanalyzer-stable[/COLOR]" to your software repository gives you *BibleAnalyzer*. (Yes, I do point people to it, despite my abhorrence of its licence.);
[*]*[COLOR="#000080"]http://ubuntuforums.org/showthread.php?t=1923376[/COLOR]* describes installing* TheWord*;
[/LIST]

FWIW, the e-Sword Installation Script from mid-2008 would correctly configure WINE for most of the low-end Biblical Software that was available for Windows back then. (Low end being the sub US$300 price point. The current [2012] version does not include that functionality.)
