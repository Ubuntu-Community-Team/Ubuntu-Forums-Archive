### Post by scragar on 2009-10-14
The ubuntu based distro's(ubuntu, xubuntu, kubuntu, edubuntu and ubuntu CE) all draw from the same ubuntu repositories, but distro's with different bases will all have different repositories(or in some cases none at all).

Ubuntu and ubuntu based distro draw majority of the softwares from the same repository, except for special packages that are not available in ubuntu yet. For Ubuntu CE, almost all the additional packages comes from additional repository.

For other distro, they have their own package management, so it is different repository. Some do not even have package management/limited pre-packaged software, they would compile it from source.
