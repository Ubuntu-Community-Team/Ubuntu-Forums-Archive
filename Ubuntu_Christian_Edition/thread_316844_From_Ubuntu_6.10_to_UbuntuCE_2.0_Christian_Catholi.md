### Post by happy-and-lost on 2006-12-11
I think it *is* just Edgy in terms of functionality and software, it just comes with some extra software, new artwork and some minor rebranding. This is from the FAQ...

[I]Q: Is Ubuntu Christian Edition a new distribution of Ubuntu or a derivative?

A: Well it is really neither. Ubuntu Christian Edition is based directly from the standard Ubuntu distribution and only contains extra Christian software available from the Ubuntu software repositories as well as a few additional tools to make the transition to Ubuntu easier for a Linux newcomer. The graphical changes are minor and are only intended to tailor the project to Christians.[/I]

EDIT: Oh, and...

[I]Q: Is Ubuntu Christian Edition designed for Protestants or Catholics?

A: Both! Ubuntu Christian Edition will always try to cater to the needs of all Christians.[/I]

ANOTHER EDIT: It'll use all the same language packs as Ubuntu, as it uses the Ubuntu repos for most of its software.

Hi Ederico,
Thanks for your interest in the Ubuntu CE project.

I will try to answer your questions one at a time.

> 1. I'm using Ubuntu 6.10, and I would like to know whether I would lose my stuff if I moved to UbuntuCE 2.0. I mean, would a fresh install be needed or is there some way to migrate without losing data? What would I risk losing if there is some sort of way to migrate without a fresh install?
Answer: You do not have to do a fresh install. You can use the [convert_me]("http://www.christianubuntuukmirror.co.uk/ubuntuce/scripts/convert_to_ubuntu_christian_edition_edgy.tar.gz") script. Be sure to read all of the instructions first of course.

> 2. This is a Christian edition, but I need clarification. I'm a Roman Catholic Christian, is this "Christian" edition a Protestant Christian edition or is it some sort of ecumenical edition?

Answer: One of the goals of the Ubuntu CE project is to cater to both Protestants and Catholics. This can be a daunting task, but hopefully we are showing a good faith effort ot be all inclusive. I am personally working towards Catholic conversion. The next release of Ubuntu CE (Edgy) will include the [Virtual Rosary]("http://www.virtualrosary.org/") program which I think will be an excellent addition. One of the major factors concerning what apps get included is the availablity.

> 3. In what languages is it available? Is Italian available? If it wouldn't be hard and/or terribly time-consuming, I might consider working on a Maltese translation with the potential recruitment of other interested people. The latter thing depends on the answer to question 2 though.
Answer: Ubuntu CE (Edgy) is basically just Edgy with the added programs and customizations. So all of the languages available for Edgy will be available for Ubuntu CE (Edgy). Now with that said the next release may be missing a few of the more obscure languages.  This may be necessary in order to make space for some new features.

> 4. Does it come with all the stuff in Ubuntu 6.10?
Answer: Yes and no. There have been a few apps removed for the purposes of making space for the extra apps.  However, you can use the Ubuntu CE Installer to add all of the removed apps back as well as install some other great educational and Christian programs.

If you have any questions please let me know.

God Bless, Jereme
