### Post by stlsaint on 2010-02-04
Anastasis has been doing some awesome work in the advancement of UCE with his testing of UCE-UNR. Now he has proposed and addition to our great distro UCE with the adding of the application [Hebrew Date]("http://ubuntuforums.org/showthread.php?t=1398508"). This is a calendar that goes thru timelines from the bible and can be a invaluable tool to anyone who is studying the bible or wants to learn more about the timelines in the bible. So now the next step is for us to test it out and vote on whether or not we would like to see it in the next release of UCE. hdate-applet is already in the ubuntu repository as it stands so compatibility is not an issue. I see no problem with it as long as it doesnt add too much to the total size of the UCE iso and with hdate-applet being at 618KB i doubt that will be an issue. 
Please vote so we can give DK plenty of time to implement if all is well.
