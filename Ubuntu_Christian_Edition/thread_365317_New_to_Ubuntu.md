### Post by Resurrection on 2007-02-19
I'm not an Ubuntu expert, so I can't tell you if that number of updates is proper, but I will say as a former Windows user, that yes, Ubuntu does update much more often than you are probably used to, especially with regard to security updates which are constant (i.e. not a "hotfix" like in windows or a "service pack").

I would say go for it.

### Post by chggr on 2007-02-19
Welcome to the wonderful world of linux!  :) 

You can install and uninstall applications by clicking "Applications" -> "Add/Remove..."
My favorite ones are:

K3b -> Writing CDs, DVDs
Amarok -> Really cool mp3 player with MANY capabilities
VLC -> Plays almost any multimedia file
Audacity -> Audio Process and mixing
aMule -> For sharing files
GIMP -> The Image Manipulation Program is used to alter pictures like photoshop 
... and many more!

You can find a list of equivalent of windows software in linux here:
[http://www.linuxrsp.ru/win-lin-soft/table-eng.html](http://www.linuxrsp.ru/win-lin-soft/table-eng.html)

You can also use the Synaptic Package Manager (System -> Administration) to install specific packets. You need to install these packets, so that you can play mp3/avi/mpg etc files:

gstreamer0.10-plugins-ugly
gstreamer0.10-plugins-ugly-multiverse
gstreamer0.10-plugins-bad
gstreamer0.10-plugins-bad-multiverse
gstreamer0.10-ffmpeg

About the updates: I think that generally it is necessary to install them, in order to be up to date. You should definitely install the "Important Security Updates".

Well, if you considered that you installed Ubuntu from a CD which is some months old, it is pretty logical that a lot of patches and updates have been published since then.

The fact that there are so many updates in linux is actually a good point. When a security issue comes up, the linux community responds within hours and publishes an update. You can then download it, install it and be safe. On the contrary, when a vulnerability in Windows is discovered it takes many weeks or even months to issue an update (Service Pack) and protect your system.

### Post by mysticrider92 on 2007-02-20
I assume you are coming from Windows and are used to updates that more or less take away your control of the computer. Updates in Linux actually help many things and keep your computer running nicely. As an added bonus, some don't even require a reboot. 

About chggr's first post, some of those programs are for KDE. I would suggest trying things like GnomeBaker instead of K3b, and Rythembox instead of Amarok, unless you have installed KDE or the Ichtux desktop (which you may want to install anyway). Ichtux is availible from the Ubuntu CE Installer (System>Administration>Ubuntu Chrstian Edition Installer) and will be good if youy want a different look and feel.

Also, the GStreamer plugins will probably need different repositories enabled. Go to System>Admin>Software Sources check all of the boxes under "Internet".
