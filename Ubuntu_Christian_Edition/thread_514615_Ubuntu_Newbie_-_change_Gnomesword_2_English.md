### Post by dmohn28 on 2008-09-22
MODULE MANAGER INSTRUCTIONS

NOTE:  According to Gnomesword, if you live in a persecuted country, USE MODULE MAKER WITH CARE

Open Gnomesword

Click Edit in the menu bar
Click Module Manager

Under the SWORD list, choose configure
make sure that the remote option is chosen and it says Crosswire
(if not skip down to next set of instructions)
Click the Refresh button on the bottom of the module manager 

Now go to expand the modules list and click on install
this should open a list of all downloadable files from the crosswire repository.  Just check the file(s) that you want, and click install.


IF THE REMOTE OPTION DOES NOT SAY CROSSWIRE

you will need to go to Sources in the sword list
in the bottom box that says current remote sources, click the add button and add the following

Caption:     Crosswire
Type:        FTP
URL:         ftp.crosswire.org
Directory:   /pub/sword/raw 

click OK.  that will put the crosswire repository on the list in the REMOTE option of Configure for you.

There are also a few other repositories.  None quite as extensive as crosswire's, but if you do a google search for Sword Project Module Repository or something similar to that, you should find a list.

Best of luck,

Dave
