### Post by jonathonblake on 2009-09-13
> **Viva said:**
> it'd be poor marketing to change the name too.

Whilst changing the name of a product, or company requires careful consideration, it need not be "poor marketing" to do so.

Were an organization is to provide commercial support for "Ubuntu Christian Edition", then a name change would be mandatory.

Basic issues for a potential foundation are:
* Host for source code;
* Host for the web forum;
* Host for mailing list;
* Host for documentation: Wiki;
* Host for Internet based localization and translation: Pootle;
* Host for disk images;

It should  also be noted that Ichthux migrated from being a distro in its own right, to a metapackage for Ubuntu. One major reason for that change was logistics.

jonathon
