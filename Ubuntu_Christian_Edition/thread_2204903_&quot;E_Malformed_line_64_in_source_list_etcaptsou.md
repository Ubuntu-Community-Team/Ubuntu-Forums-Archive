Ok--I was trying to add a ppa for the new version of  Libre Office to my update manager.  I must have typed it in wrong because all of a sudden my update manager crashed and my software center keeps crashing when I try to open it.  I get this error message that says:

   E: Malformed line 64 in source list /etc/apt/sources.list (dist parse)

Becky

### Post by Steve_and_Becky on 2014-02-11
Ok I found an answer already and fixed my problem.  I typed into my terminal sudo gedit /etc/apt/sources.list and I was able to get in and fix the problem in the command lines.  I missed a space in 2 of the command lines.  Where there was supposed to be a space, there was no space at all.
