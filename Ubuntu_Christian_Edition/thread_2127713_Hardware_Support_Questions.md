### Post by tgalati4 on 2013-03-20
Either Mac USB or wireless keyboard work with Ubuntu.  I don't know about the Mac trackpad.  Most Logitech USB products work with Linux.

There are lots of Okidata printers in the CUPS database, but I could not find a specific driver for the Okidata MC361MFP.  You will have to do some research.  Send an email to Okidata and ask them which CUPS driver to use with that printer.

iSight cameras work, most Logitech webcams work.

The Okidata website link shows drivers for Kernel 2.6, which is on older versions of Ubuntu.  For Kernel 3.2 or 3.5, you would need to ask Okidata directly.
