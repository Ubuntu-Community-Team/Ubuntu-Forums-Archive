### Post by mhancoc7 on 2006-10-08
Hi all,

Ichthux was actually one of the inspirations for Ubuntu CE. I watched the progress of Ichthux from a distance for several years. It had seemed that it had stalled out. I had always wanted to see a Christian distro so I just decided to do it and here we are. 

I noticed after I released Ubuntu CE that the Ichthux project had really started picking up some steam. I am not sure if it was a coincedence or if the success of Ubuntu CE helped spur things on. Either way I think it is great to have them both.

Ichthux is a similiar project with some very distinct differences. The Ichthux project is built by Ubuntu developers and they stick very closely to the Ubuntu philosophy and only include Open-Source applications. The Ubuntu CE project is built and maintained by a team of Ubuntu users. Ubuntu CE is really a "pre-customized" version of Ubuntu. We will always choose an Open-Source app over a Closed-Source one. However, Ubuntu CE does include some wallpapers, a firefox theme, and a firefox extension that are not Open-Source. We are also open to the idea of including any app that will benefit the end user whether it be Open or Closed source. 

These differences have caused quite a bit of tension between the two developement Teams. I really regret that it has even been an issue. To me both projects are a great resource. I really respect the Ichthux Team and I think what they are doing is great. The Ubuntu CE project simply has a more pragmatic approach.

Anyway, I hope that helps explain a little more about the two projects.

God Bless, Jereme
