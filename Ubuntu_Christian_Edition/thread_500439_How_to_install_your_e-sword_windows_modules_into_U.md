### Post by brodanny on 2007-07-13
Simple instructions to install your e-sword windows modules into Ubuntu  

1.Copy all of the .bbl .cmt and .dct files to a USB drive or burn to a CD
2.In Ubuntu look in the menu for places, then open the home folder. This opens the natulis file manager
3.Locate on the left pane your USB drive or CD. highlight and cut or copy the files you wish to move.
4.Double click on the file system icon in the left pane.  Hold down the ctrl h keys to unhide the files. In the right pane click on the files in this order usr/local/apps/esword4linux/.esword/drive_c/Program Files. You will then see the esword file, right click one the file and select Paste Into Folder.
5.When you open E-sword all of your files should be there.
