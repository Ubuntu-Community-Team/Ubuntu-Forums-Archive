### Post by mhancoc7 on 2007-05-07
I have just uploaded an updated version of the convert_me and dansguardian scripts. I have added a few new features to make things a little better for those who do not want to lose their personal settings (firefox, desktop, dansguardian).  The new scripts will create backups of these settings in your home directory.  The new scripts will also only run on Feisty. If you try to run the scripts on anything Edgy or another Ubuntu release a message will let you know that you need to upgrade to Feisty first and give instructions on how to do that.  I hope these changes will make the scripts a little more user friendly.  Thanks for all you input.  God Bless, Jereme

*** Edit:** Added an error message to alert the user if one or more packages do not get installed and instructs the user to re-run the script.*
