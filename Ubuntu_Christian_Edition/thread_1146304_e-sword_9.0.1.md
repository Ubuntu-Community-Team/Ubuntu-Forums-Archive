All;

e-Sword 9.0.1 was released on 29 April.

I haven't yet installed it on my Linux box.  (That box currently doesn't have Internet access.)

For those that do upgrade to 9.x, do the following:

* Put _all_ of your e-Sword resources in the e-Sword user directory;
* Then install e-Sword 9.x;
* Download all of the resources from e-Sword.net;
* Log into eStudySource.com, and download the resources that you have previously purchased;

As soon as time permits, all user created resources that can be legally distributed, will be available from e-sword-users.org;

After I've hooked my Linux box up to Internet, I'll do  a couple of experiments, to see if both e-Sword 9.0.6, and e-Sword 9.0.1 can be on the same Linux box, at the same time.
