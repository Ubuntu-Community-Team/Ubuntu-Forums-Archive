### Post by tak1150 on 2007-08-28
I have never tried using the Thai language in Ubuntu, but Japanese works perfectly.
If you didn't know already, you can go to (click on):

System --> Administration --> Language Support

After you install the initial language package, you can just check "thai" in the list to install the thai package.

Then go to:

System --> Preference --> SCIM method setup

to set up SCIM which lets you read / write Thai and other languages.

Hope this helps.
