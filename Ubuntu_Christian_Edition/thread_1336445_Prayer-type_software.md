### Post by whitefort on 2009-11-24
Hi, and apologies if this isn't quite the right place to post this - but I was wondering if a certain type of software exists.  Here's what I have in mind.

I often find it helpful to pray by 'emailing' God.  I know it sounds crazy, but if I sit at the keyboard and type, it keeps my mind focused and helps prevent me drifting off into daydreams.

I've just been using a word-processor for this, but recently I got to thinking how cool it would be to have a app that allowed space for typing, but also called up a random 'verse of the day' from one of the sites that provide such things.  Even better if it also had a little window which would randomly display pictures from a folder, so that there could be a stream of inspirational or thought-provoking images: landscape, holy land views, religious art, or possibly even scenes of natural or man-made disasters.  The ability to add 'tags' to each prayer-mail would also be nice.

Does anyone know of something that might partly fit the bill, even if it's something not originally intended with prayer in mind - maybe some sort of motivational software or something?

I'm even wondering...  I'm no programmer - I last programmed in Basic, a long time ago!  But I'm wondering if it would be feasible to learn programming just to try to make such a program?  If (BIG 'if) I did try something of the sort would the rules of free software mean that I would be able to find some free OS app that *nearly* does what I want, then make the necessary changes?

I'm also wondering if I'm alone in my crazy quest for this sort of software, or if it's something that other people might find useful?

### Post by samden on 2009-11-24
It sounds a cool idea, I just can't get the picture out of my head of the "Yah-weh!" prayer mail software off Bruce Almighty... ("You've got prayers!")

If you do make it, you might like to include encryption in the prayers you write, so you can save them in an encrypted form and not worry about anything private being stumbled across by another user of your computer. Or set it to just delete them upon completion.

But I don't really see the need for a standalone piece of software. You can already type in a text editor, an email program (sent to a bogus address or saved to drafts), a word processor, or anything else. And you can get a 'verse for the day' from any number of places already. 

It might therefore be a fun application to program if you have the time and want a project, but that might be all.

I completely understand where you're coming from, I used to have a little notebook for the same thing.
