### Post by James Willis on 2009-04-02
Assuming that you've followed Ferb's suggestion to set the download location.  Then, using the Bookshelf Manager (Bibletime) or Module Manager (Gnomesword) click on remove and try removing the current selection.  You should be able to select the language, bibles, and other resources that you want to download by clicking on Install/Update. Be sure that you have the correct Install Path with read/write privileges, otherwise it will not work.

### Post by mbspringer1 on 2009-06-01
> **Orlsend said:**
> Hi I just downloaded gnomesword and Bibletime, I think of my current location it downloaded the bibles in Arabic,I can read Arabic but its painfully slow for me. is their anyway where I can download and add a English bible to my Bible apps?
I use BibleTime 1.6.5.1

Go to settings
Goto Bookshelf manger (F4)
In the left hand panel click Install/Update works.

In the lower right corner click 'connect to library'
A small download window appears for a short time
the windows says 'downloading library information'
when the download is finished the small window will close.
the right hand side of the main window will show a list of Bibles.
there is a check box to the left of each bible.
check the boxes of the bibles that you want to install.
After choosing the bibles you want to install on your computer.
Click on the button in the lower right corner that says 'Install Works'
