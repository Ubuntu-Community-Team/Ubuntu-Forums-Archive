### Post by lykwydchykyn on 2014-01-23
I have recently released an open-source project which some of you may be interested in called Omega Hymnal; it's a lyrics & chords display program for use in informal worship settings.  

You can read more about it here:  [http://www.alandmoore.com/blog/2014/01/12/introducing-omega-hymnal/](http://www.alandmoore.com/blog/2014/01/12/introducing-omega-hymnal/)

And download the code here:  [http://github.com/alandmoore/omega-hymnal](http://github.com/alandmoore/omega-hymnal)

Omega Hymnal is written as a web application in Python and requires python-flask.  I'm looking for testers and possibly contributors, but the software is pretty usable at this point, so if you just want to be a user that's ok too :).
