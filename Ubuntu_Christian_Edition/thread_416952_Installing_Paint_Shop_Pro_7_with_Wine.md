### Post by urukrama on 2007-04-22
Yes, it should work. Have a look [here]("http://appdb.winehq.org/appview.php?iVersionId=1405") (That site is useful for any Windows application you might want to install with Wine).

To install it, all you should do is 1) have wine installed (there is a good howto [here]("http://ubuntuforums.org/showthread.php?t=149585&highlight=wine+howto")); 2) double click on the .exe install file, and all will (or should) be well. 

Applications installed with Wine can be found in the hidden .wine folder in /home (so /home/username/.wine).
