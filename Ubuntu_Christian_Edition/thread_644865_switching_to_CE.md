[http://www.whatwouldjesusdownload.com/christianubuntu/2007/05/features.html](http://www.whatwouldjesusdownload.com/christianubuntu/2007/05/features.html)

above is the feature list.

IMO there are 2 ways to benefit from the CE project
1) install Ubuntu CE (either fresh install or covert-me script)
2) install Ubuntu CE individual packages on (x/k)ubuntu

I do #2. I've seen some incompatibility with some KDE apps when I was using Ubuntu CE 3.x (Feisty) and I don't like to have the Dansguardian filter on my laptop (slows things down; I put Dansguardian on the proxy server for my home instead).

So I guess as always, it's up to your needs/wants.
U-CE is most definitely Ubuntu, so yes it does get all the updates that Ubuntu gets. I expect U-CE 4.0 (gutsy) will also have all the compiz-fusion stuff. If you do the #2 method, you will definitely have it.

Hope this helps.
