If you are now running Feisty then yes, you should be able to download the newest convert_me script and run it to get the latest Ubuntu CE. Keep in mind that you will lose some of your configuration settings such as dansguardian, firefox, and desktop. These will be replaced with the newest version of CE so you will have to do your customizations again.
