### Post by mysticrider92 on 2007-01-07
There is a script to automatically convert regular Ubuntu to CE availible [here]("http://www.whatwouldjesusdownload.com/christianubuntu/2006/07/download.html").

If you want to install it all yourself, the pakcages are Dansguardian (with a GUI that you can get from one of these threads), gVerse, GnomeSword Bible Study, and Automatix2, plus some custom artwork.
