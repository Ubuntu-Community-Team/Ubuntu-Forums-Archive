That depends upon which file manager you are using.  Some of them include FTP functionality, and some don't.  For various reasons, I prefer using stand alone ftp-clients.

> 2. How do I go about installing custom alert sounds? I have an alert sound from my other machine I want to move over.

> 3. Is there a way to remove the Wacom Tablet app?

> 4. Is there an extension or something I can tweak in LibreOffice so that the menus appear in the menubar?

How did you install LibreOffice?
[LIST]
[*]If you used Synaptic, then the menus should have been automatically included;
[*]If you installed the version from [https://www.libreoffice.org/download/](https://www.libreoffice.org/download/), then you need to go into the desktop-integration subfolder of the unzipped file, and run "*sudo dpkg -i *.deb*";
[*]
[/LIST]
