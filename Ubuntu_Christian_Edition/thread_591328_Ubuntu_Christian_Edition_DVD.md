### Post by jimburt372 on 2007-11-08
I am certainly not an expert, but I think I can suggest at least a work-around for your concern.

If you install Ubuntu natively, then use the convert_me script, you don't omit anything.

I know that this is not the same as just installing directly from an UbuntuCE disc, but it would avoid any package concerns that you might have.  

I am a noob deluxe (meaning that even though I have been running linux for an extended amount of time, I have truly learned little about tweaking and working 'under the hood'), but since CE always follows the official release of Ubuntu, I usually start over with a clean install of the latest native Ubuntu.  Then when CE is updated through Jereme and the rest of the contributors hard work, I run the convert_me script.

Just a thought, and I hope that it is helpful.

I don't have any idea how much work would be involved in creating a DVD version of CE, but I do like the idea if the workload is worthwhile.  In the mean time, I will most likely just 'convert' to CE.
