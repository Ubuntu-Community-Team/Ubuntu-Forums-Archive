### Post by WorkingOnWise on 2007-10-23
Well, I've never been accused of being overly patient! I have been running Gutsy since July, and am very happy with it. The only thing to make it better is, of course, if it were Ubuntu CE Gutsy. Well, it is now....um, mostly...I think.
Last night I ran the 'Convert" script against machine,after some minor mods. I know I should have taken more time with this before I did it, but, remember the first sentence in this post?
In the convert_me script, I replaced all instances or feisty with gutsy, and all instances of 7.04 with 7.10, and then ran it. (I told you I was impatient!)

Well, my system it still alive. 
I know dansguardian is up, because it stopped me from downloading an iso last night. 

The ff themes are in, as is the gnome customizations and themes. 

e-Sword is in, but I had installed prior to this, and it needs the tweaks to get the fonts to display.

I can't install esword or ies4linux with the links provided because the gui for dansguardian is broke. It asks for the admin password, but then nothing shows up. I know, I can disable it manually, but I want to see that gui, so I'm using that as motivation to fix it!

The splash screen no longer works on boot. I now get a text mode screen detailing the boot process instead of the pretty logo and status bar.

After I rebooted, firefox was 'reset" to defaults. I have to go get al my addons (60 or so) again. I may have been able to avoid this with a profile. I'm not sure.

When I logged in to gnome, all my panel mods were gone, and compiz was downright persnickity! The wiindow decorator cashed several times in a row, and all my compiz customizations were gone as well. 

I restarted x, and things looked good. I was able to reconfigure compiz to my liking again, and only the setting manager was a bit unstable. 

I have all my panel mods back in now, and compiz is massaged to my liking, and everything is stable again. I just ran the latest update (via Gutsy's Update Manager) for 10/23/2007 at 7:00am EST. Included in those updates were a few libs that the dansguardian gui uses I think, and an update to dansguardian, so I'm hoping afer a reboot the gui will work.

If anything improves after the reboot, I'll post again.

The main point of this post was to help get some testing done for Gutsy. The latest release of Ubuntu is so great that I am finally off Windows totally, and still am able to get all my daily tasks done. Finally....after 10 years.....Linux just works!

While my system made it though this foolish and ill-planed upgrade with just a few bumps and bruises, You may not be so lucky! YMMV!

BTW, I'm on Ubuntu 7.10 AMD64, so I'm all the more surprised that it was as smooth as it was!

Thanks
Keith
