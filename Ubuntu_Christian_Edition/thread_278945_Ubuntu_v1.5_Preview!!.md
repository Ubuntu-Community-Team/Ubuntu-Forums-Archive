### Post by mhancoc7 on 2006-10-17
I am hard at work putting together Ubuntu CE v1.5. This release started out as a simply change to include the new [Automatix2]("http://www.getautomatix.com/"). It has grown to include some nice new features and some bug fixes as well. Below is a list of the changes.

New Packages:
Automatix2 replaces the original Automatix

Dansguardian GUI Enhancements:
added Naughtyness Limit to dansguardian GUI options
added "Visit Dansguardian Website" button
added "Lock/Unlock Firefox Proxy Settings" buttons

Ubuntu Christian Edition Installer Enhancements:
added Ichthux Desktop to the UCE Installer
upgrade Ubuntu CE button added to UCE Installer

Bug Fixes:
fixed sources.list not surviving install
fixed Install menu item not getting removed after install
I am really excited about the addition of the Ichthux Desktop to the Ubuntu CE installer. This will allow those who want to benefit from both Ubuntu CE and Ichthux to easily install Ichthux to there Ubuntu CE installation with just a few clicks of the mouse.

I am also really happy about the new "Upgrade Ubuntu CE" button in the Ubuntu CE Installer. This will make upgrading Ubuntu CE much easier for users. It basically downloads the "Upgrade_Me" script and then checks to see if your system needs to be upgraded or not and acts accordingly. Again just a few mouse clicks and you are upgraded!

Anyway, I just wanted to let you all know what was coming. Thank you all for your support.

God Bless, Jereme
