### Post by oeb on 2007-01-10
With the help of folks posting to the forums I finally got e-Sword to work in Wine.  Unfortunately, I don't know of a gospel parallel module for e-Sword, so; thinking, "I'm now on a 'roll'," I made a directory on my Desktop, pasted a copy of an old version of Gospel Parallels (v. 1.3 for Windows) there, navigated to the new directory in terminal, typed 'wine pgospel.exe' and; then, sat back to watch Gospel Parallels install without a hitch.  I only had to provide input for folder name and icon location (wine; Gospel Parallels -- Desktop) and the Windows installer took it from there.  It even provided an icon on the Desktop that works perfectly and it ran a program tutorial.  I have had a couple of riched20.dll errors but nothing fatal and these may have been my operating technique.  This will be a handy reference tool from which to copy and paste parallel passages.  Hope it helps others,  Juzme oeb
