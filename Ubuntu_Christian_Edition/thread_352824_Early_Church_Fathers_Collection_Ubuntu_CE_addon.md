### Post by mhancoc7 on 2007-02-03
This is a deb package that I built by user request.

This is a simple GUI to launch .hlp files with winhlp32.exe using wine. The 'Early Church Fathers Series in WinHelp Format' is a 37-volume electronic collection of writings from the first 800 years of the Church. This collection is divided into three series, Ante-Nicene, Nicene and Post-Nicene Series I, and Nicene and Post-Nicene Series II. It can be found at [www.zeitun-eg.org]("http://www.zeitun-eg.org/").

You can download it here:
[http://www.whatwouldjesusdownload.com/christianubuntu/bin/early_church_fathers_ubuntu_1.3_all.deb](http://www.whatwouldjesusdownload.com/christianubuntu/bin/early_church_fathers_ubuntu_1.3_all.deb)

Hope you enjoy.

God Bless, Jereme
