### Post by david_kt on 2008-10-06
Finally I have time to make script to backup and restore e-Sword.

Backup script will open a browser for you to locate the directory you want to keep your backup file. Backup file will have date and the name to make it easy for you to identify which backup is the latest.

Restore will also open a browser for you to locate the backup file you want to restore from. It could also restore even if you have deleted your current e-Sword directory.

WARNING: restoring from backup file will delete the current e-Sword (send it to trash, you could find it there) and replace it with the backup file. It could also restore even if you have deleted your e-Sword directory.

Please check the installer on this thread, it include backup and restore now:

[http://ubuntuforums.org/showthread.php?t=404042](http://ubuntuforums.org/showthread.php?t=404042)
