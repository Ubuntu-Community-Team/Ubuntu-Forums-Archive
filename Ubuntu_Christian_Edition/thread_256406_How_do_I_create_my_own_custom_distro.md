Hi,
There are several ways to create your own LiveCD. I am using the [Ubuntu Customization Kit (UCK)]("http://uck.sourceforge.net/") to build Ubuntu CE. There is also [Reconstructor]("http://reconstructor.aperantis.com/index.php?option=com_frontpage&Itemid=1") which looks nice, but I have yet to get it to work. The only problem that I really have with UCK is the lack of documentation, but once you get the hang of it it is awesome.

Hope that helps, Jereme
