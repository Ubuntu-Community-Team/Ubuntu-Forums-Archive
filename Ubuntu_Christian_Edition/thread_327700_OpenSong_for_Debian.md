### Post by Coburn on 2006-12-29
I'd like to ask a favor from the UCE developers.  OpenSong is useful presentation software for projecting lyrics and printing leadsheets.  Currently it's only supported under Windows, Mac, and Novell Linux.

My skimming the OpenSong forums gives me the idea the developers are having problems compiling the code with REALbasic for Linux such that it will run on a Debian (Ubuntu) OS.

Who knows--this could be a mismatch issue between REALbasic (generic .tgz) and Debian, rather than anything wrong with the OpenSong code.

What seems to be needed is for somebody who is an experienced Debian developer to tackle the project.  They are saying compiling generates a lot of frustrating errors.  The current developers are balking at the (apparently daunting) task.

--But for someone familiar with adapting *nix apps to Debian, the quirks might be familiar ones, and ironing them out, something that is not too hard to do.

This would add a very nice song presentation editor to the UCE repertoire.

Plus I went to Acquire the Fire with the developer a few years ago :D 

Thanks
