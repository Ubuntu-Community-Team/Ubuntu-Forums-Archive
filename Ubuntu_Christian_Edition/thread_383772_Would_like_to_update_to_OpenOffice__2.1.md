### Post by mysticrider92 on 2007-03-16
I wouldn't download it from OpenOffice.org because the Ubuntu package management system is different from the generic .tar.gz packages they have. You should probably try Synaptic, if the latest version isn't there yet then it will probably be soon. Make sure to click the reload button in Synaptic before searching for it so you will know if it is current.
