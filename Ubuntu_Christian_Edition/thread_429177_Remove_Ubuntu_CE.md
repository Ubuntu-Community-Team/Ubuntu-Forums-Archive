### Post by mhancoc7 on 2007-05-02
> **mrphantastic said:**
> **  Regarding the 'compatibility' I mentioned, I am just wondering if I can run the CE version that is designed for Edgy Eft, if I have currently done the upgrade process to Feisty Fawn?**

I would suggest that you download and run the latest convert_me script for Feisty. This should adjust all of the Ubuntu CE stuff for Feisty. Keep in mind that you will lose some of your configuration settings such as your desktop setup and your firefox bookmarks and such. These are stored in directories in your Home directory. They are .gconf and .mozilla . These are hidden files so you will need to select Show Hidden Files in the Nautilus. Then you can save yours before running the script. Then you can put your original settings back if desired.

Just let me know if you have any questions.
