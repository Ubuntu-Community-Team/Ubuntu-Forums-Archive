### Post by slayerboy on 2007-02-21
How fast are the Pentium II's you have?

From the [Xubuntu site]("http://www.xubuntu.org/get"):
> 
Minimum system requirements

To run the Desktop CD (LiveCD + Install CD), you need 128 MB RAM to run or 192 MB RAM to install. The Alternate Install CD only required you to have 64 MB RAM.

To install Xubuntu, you need 1.5 GB of free space on your hard disk.

Once installed, Xubuntu can run with 64 MB RAM, but it is strongly recommended to use at least 128 MB RAM.


To convert this to the CE edition, see [http://www.ubuntuforums.org/showthread.php?p=2186633](http://www.ubuntuforums.org/showthread.php?p=2186633)

I think this should run fine on these computers if they are on the upper end of the Pentium II scale.

The program that the CE edition comes with for content blocking is Dansguardian.  Take a look at [http://dansguardian.org/?page=introduction](http://dansguardian.org/?page=introduction) for more info on the methods.

I have not used the CE edition, and if all you're looking for is the filtering, you don't need the CE edition, as you should be able to use dansguardian from synaptic.
