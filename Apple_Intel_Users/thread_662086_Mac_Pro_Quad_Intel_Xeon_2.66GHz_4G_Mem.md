I'd just like to report that I've installed ubuntu 7.10 AMD-64 on this system and it works great for me (yes there are problems with flash but I don't care about that, I just write software).

This is not a bootcamp install -- its a complete install from a boot of the installation CD
to the whole of the 500G /dev/sda1 disk (I've also formatted the second drive):-

I'm looking at a 30inch Apple Cinema 2560 x 1600 display with a standard:-

nVidia Corporation G70 [GeForce 7300 GT] (rev a1) card

I have three displays but only one 30inch display and I'm running ubuntu on that.
