### Post by cyberdork33 on 2007-10-11
You can do things that way, but it may be beneficial to have a small OSX partition (and the EFI partition) for firmware updates. You can use rEFIt to boot the linux partition by default.

Otherwise, yes you can just pop in the cd, boot, and tell it to use the entire hard drive, then install pretty much any howto for configuring hardware. If you have a Santa Rosa Machine, there may be some other issues as well. there is a wiki page, and a very long thread in this forum covered the eccentricities... You might also try Gutsy (due to be released in a week) as there are some changes that help with compatibility.
