### Post by CowzRule on 2007-08-31
Compiz is included with the Feisty 7.04 Live CD. Look for Desktop Effects under System/Preferences. If the Live CD detects that your hardware will support Compiz, it'll run.
But to answer your question, Yes you can install Beryl and/or Compiz Fusion while running the live cd. However, since the live cd is basically using your memory as the hard drive, installing it will just eat your available memory and if/when you reboot, it'll all be lost.
