### Post by jrc03c on 2008-07-16
I got it to work.  I have Leopard (10.5.4) and Ubuntu "Hardy Heron" with all updates installed.  I turned on remote desktop in the "Remote Desktop" preferences of Ubuntu, and then I can connect to it (via IP address...I can't confirm whether or not the computer name works) using Leopard's built-in Screen Sharing application.  It does warn me, however, that this type of screen sharing doesn't support keyboard encryption, but that's the only pitfall I encountered.  To my knowledge, I had no special other software or settings installed.  I can also connect to a friend's computer, which is on a different network.  I hope this helps!
