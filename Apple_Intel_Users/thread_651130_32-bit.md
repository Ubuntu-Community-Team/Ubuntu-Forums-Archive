### Post by Rog-Mahal on 2007-12-27
The best way (so I've heard) is to use rEFIt for OSX, use it to free up some harddisk space and install Ubuntu on that. I singleboot Ubuntu myself, but I may attempt to create a small OSX partition for firmware updates etc. It should be interesting to try the rEFIt method in reverse. 

I would recommend searching the forums and doing some research, it looks like there are some pretty common pitfalls. Good luck!
