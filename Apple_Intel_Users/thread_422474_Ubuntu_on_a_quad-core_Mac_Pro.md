### Post by tribaal on 2007-04-25
Ubuntu installs pretty easily provided you install bootcamp (I had to boot in "safe graphics mode" for some reason).
The only thing lacking for the time being is sound - it's a little messy right now: you need to recompile the alsa driver with a patch for some of the outputs to work, and the front headphones output doesn't work.
This should be fixed pretty soon hopefully.

Apart from sound everything works nice.

- trib'

### Post by jdemesse on 2007-04-25
I have installed feisty 64 bit on a Mac Pro recently. You can read more about it at [http://ubuntuforums.org/showthread.php?t=421311](http://ubuntuforums.org/showthread.php?t=421311).
