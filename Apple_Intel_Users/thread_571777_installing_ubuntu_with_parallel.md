Parallels is a virtual machine. You are not really installing to your real hard drive, it is an image file. There are many limitations to using it that way, but if you are just wanting try it out, that is a good choice. Parallels will walk you through the steps needed to create a image file to install Ubuntu on, then just install Ubuntu by using the option 'use entire disk'
