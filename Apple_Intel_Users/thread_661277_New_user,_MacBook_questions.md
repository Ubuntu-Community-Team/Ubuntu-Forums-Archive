### Post by Rog-Mahal on 2008-01-07
You need to install a package called 915resolution in order to get the correct screen settings. You can get it using Synaptic.

You can also use either 32 or 64 bit. Both will run fine. I chose the 64 bit version, and so far I have had very few issues and had some noticeably faster boot up/application launch times.
