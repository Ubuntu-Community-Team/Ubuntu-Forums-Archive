### Post by ronocdh on 2008-03-09
I think this is great! From the [Brightness page]("http://packages.ubuntu.com/dapper/gnome/brightside"):> Brightside provides "edge flipping" to allow you to switch to the adjacent workspace simply by pressing your mouse against the edge of the screen.

Brightside also allows you to assign configurable actions to occur while you rest the mouse in a corner of the screen. Currently available actions comprise:

   * Fade out volume
   * Prevent screensaver starting
   * Start screensaver and lock screen
   * Enter DPMS standby mode
   * Enter DPMS suspend mode
   * Enter DPMS off mode
   * Dim laptop backlight
   * Custom action

When you said "hot corners," I thought about Expose and other such functionalities which can be set to be triggered by mousing to different corners in OS X. Compiz already offers this, but I see that this Brightness package is a bit more.

Looking forward to taking it for a spin!
