Speaking with experience of Ubuntu (instead of Kubuntu, but I really don't believe the desktop environment makes a difference), VMFB3 is definitely more reliable. There are, however, threads here that indicate one can game Parallels into installing Ubuntu comfortably by using the Solaris option.

Performance is adequate, I suppose, but for me it was too far short of running natively; I'd much rather triple boot than run other OSes in emulation. In fact, one of my favorite thing about Linux is generally that it feels so slick, runs so quickly! Oh well.

Wireless will be a chore, but it's certainly doable. You may want to check out the Kubuntu forums for anything specific to the desktop environment. Good luck!
