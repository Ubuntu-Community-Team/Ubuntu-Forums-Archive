### Post by Chadversary on 2007-12-07
For anyone who is viewing this thread, there is now a wiki page on the [MacBook Santa Rosa.]("https://help.ubuntu.com/community/MacBook_Santa_Rosa")

Smutronic, if it was purchased in December then it is most likely a 3rd Generation, aka Santa Rosa. Only revisions of pommed greater than 412 work with the Santa Rosa. The particular version I've installed through Aptitude is **1.12~dfsg-svn412cwi1**. If you have a different version, try changing the configuration in Synaptic Package Manager > Settings > Repositories. The relevant settings I've checked are **gutsy-updates** and **gutsy-proposed** under the Updates tab and **universe** under the Ubuntu Software tab.
