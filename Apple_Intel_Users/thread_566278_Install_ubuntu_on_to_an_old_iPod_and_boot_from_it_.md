### Post by benweston on 2007-10-04
I'm not very Mac-savvy, unfortunately, but it is perfectly possible to run Ubuntu from an external device. I've run it from a 4GB flash drive before.

As long as the BIOS is set to boot from external devices (USB, etc) before the hard drive. This is standard configuration on most newer PC systems but I'm not so sure about Mac.

So as long as the BIOS has support for USB hard drives (essentially a flash drive in the eyes of the BIOS, I think), and you can set it to boot from that first, you'll be good to go.
