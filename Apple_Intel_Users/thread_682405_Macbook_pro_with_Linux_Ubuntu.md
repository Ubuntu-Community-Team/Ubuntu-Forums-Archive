### Post by cyberdork33 on 2008-01-30
> **ydmaster said:**
> I have a Macbook Pro 2.4GHZ and I want to know if I can install Linux by using Bootcamp. Do I need any CD or other accessory to install Ubuntu? You can use Bootcamp to partition the hard drive, that is all it is really useful for anyway. There are a few other ways to partition also.

> **Rog-Mahal said:**
> You don't use Bootcamp, you need to download another program called rEFIt. 
Bootcamp and rEFIt are not equivalents. They have nothing to do with each other. One partitions a hard drive, the other displays a boot menu. rEFIt is not required, although suggested.
> **ydmaster said:**
> Does Linux being on a Macbook Pro have problems rather than Linux on PC?
There are a few issues because of the unique hardware of the Mac. There are several pages in the wiki that describe the specifics for your machine. I am not sure which Macbook Pro you have, but here are the links to the two wiki pages.

[https://wiki.ubuntu.com/MacBookPro](https://wiki.ubuntu.com/MacBookPro)
[https://wiki.ubuntu.com/MacBookPro/SantaRosa](https://wiki.ubuntu.com/MacBookPro/SantaRosa)
