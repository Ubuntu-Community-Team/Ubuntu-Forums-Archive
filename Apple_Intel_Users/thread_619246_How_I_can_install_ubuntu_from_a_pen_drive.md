### Post by volanin on 2007-11-21
I have already tried to install it from a pen drive with NO SUCCESS.

The most common method of converting a LiveCD to boot from a pendrive uses syslinux.
Syslinux is a linux boot loader for these cases, but unfortunatelly, it depends on BIOS calls.
And since Macs do not have BIOSes, it does not work.
