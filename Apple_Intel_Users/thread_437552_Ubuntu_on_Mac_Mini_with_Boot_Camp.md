### Post by ronocdh on 2007-05-08
> **milind4 said:**
> I was wondering how I could install ubuntu on mac mini w/ boot camp. is using boot camp to do this ok? if there a walkthrough or a faq or something somewhere? any help is appreciated!
Using Boot Camp is no problem at all. Just use the graphical partitioning utility to create your space, then boot up with the Ubuntu live CD and install on the newly created partition.

You can also partition using the terminal in OS X if you like, and perhaps triple boot, if that interests you. I'd say the most helpful guide is [this one]("https://help.ubuntu.com/community/MacBook"). 

Good luck, and please post back with any trouble you have, or at least search through the forums for similar issues. =)
