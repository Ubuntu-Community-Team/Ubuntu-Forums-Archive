### Post by black_raven2525 on 2007-07-27
Unfortunately, when they say parallels 3.0 comes with 3d acceleration, they mean for windows users.  It is not yet possible to use 3d accelerated apps under any of the currently available VM software.  You wont be able to get the nvidia drivers to function properly because the emulated graphics card used by parallels doesn't support 3d accel.  If you want the extra eye candy of desktop effects, yo will have to install ubuntu natively on your mac book.  For information on how to do this, you can search this thread. [http://ubuntuforums.org/showthread.php?t=474144](http://ubuntuforums.org/showthread.php?t=474144)
