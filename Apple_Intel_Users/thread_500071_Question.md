### Post by thully on 2007-07-13
No, you can't eject devices by dragging them to the Trash.  However, you can eject them by either 1) right-clicking them and choosing Eject 2) the eject command (pass the device as the argument)  3) the Eject keyboard button (for CDs/DVDs).

Regarding the command line - both OS X and Ubuntu use the same shell (bash) and use UNIX-type tools, so you shouldn't have a problem there.  Some things may be slightly different due to the use of GNU command-line tools in Ubuntu vs. BSD command-line tools in OS X, but for the most part it's the same.
