### Post by ivesjd on 2007-08-05
Install OS X on the mini, Then partition it so that you have about 30GBs for OSX. Install windows on the other 50GBs. Now you need to boot from the live cd and resize the windows partition to about 25GBs and Install Ubuntu BEFORE the windows partition. Then once you have everything running create folders on the external for you documents and such. Then make links to those folders in all of your OS's so that all of your documents, movies and music are saved to the external. 
That is a fairly simple way to do it.
