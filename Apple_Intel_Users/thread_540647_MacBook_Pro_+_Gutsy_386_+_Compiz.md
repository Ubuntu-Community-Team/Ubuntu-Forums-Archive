The HOWTOs I've read say to run the generic kernel on a MacBook Pro but after upgrading to Gutsy only the 386 kernel runs, is this correct?  If so do I need to switch my entire system over from the generic version to the 386 version?

How I Got Here:
I originally installed Feisty on my MacBook Pro Intel Core Duo (NOT 2) with Beryl and it worked great except for the power management.  I upgraded the kernel to 2.6.22-10-generic but after I couldn't get around a boot time freeze tried the 386 version which works great.  That didn't solve my power management issues though so I decided to upgrade to Gutsy.
