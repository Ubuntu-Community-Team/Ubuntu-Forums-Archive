### Post by cyberdork33 on 2008-02-13
> **tubasoldier said:**
> 1 - Can I switch users? Fast user switching? I want to stay logged in so the computer will still keep doing my work while my wife logs in on her account.Yes. System Preferences > Accounts > Login Options. Check the "Enable Fast User Switching"

> **tubasoldier said:**
>  2 - Can we share documents between users? I have lots of pictures, videos, and music that my wife and I both want access to. I don't see a reason to have these duplicated on a hard drive for each user. Can we share them in some way? Even one user having permission for the documents but accessible by another user would work. permissions are very similar to Linux (It is Unix afterall under all the graphics). You can set permissions of folders, individual files, etc just like you can on a linux system.

> **tubasoldier said:**
> 3 - Are there developer tools on OS X already? Can I compile wget or must I find dev tools before I can compile it?THe OS X developer tools are on the OS X install DVD. This includes gcc and such. 

curl is installed by default in OS X and does the same thing as wget. If you really want to get some other common *nix tools that you are used to, check out one of the several 'ports' projects such as macports, darwinports, or fink.
