### Post by altonbr on 2008-06-11
> **anuj.pathania said:**
> Hardcoding HTML on Gedit produce the optimium result.

It may be hard, but it is always more efficient and flexible.

Plus, gEdit has a plugin called 'snippets' that allows you to save and re-use code over and over again.

For example, I always put the GPLv2 at the top of my scripts. I'm not going to type out the GPLv2 or go hunt it down every time, so I added 'gpltwo' in snippets along with the actual content.

To activate it, I type 'gpltwo' then hit 'tab' and gEdit autofills the GPL content for me!

That means its great for setting up basic (X)HTML skeletons:

Create a basic HTML page, save it as 'htmlskel' in the plugin 'snippets' and then every time you make a new page, type 'htmlskel' plus 'tab' and you'll have a new HTML skeleton every time!
