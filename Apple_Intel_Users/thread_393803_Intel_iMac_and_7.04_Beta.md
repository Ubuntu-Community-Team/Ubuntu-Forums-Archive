### Post by pikz on 2007-03-26
I installed Ubuntu 7.04 Beta on my iMac Core duo. Here are some of my experiences with it.

-Nothing unexpected happened during the installation. I didn't need to use rEFIt, so that is a nice improvement. 

-I still haven't been able to set up my wireless internet connection, though. This is still too complicated in my view. I've managed to make it work with a previous version, but I've always had to struggle with it before I've managed to get it working. 

-Suspend to ram has worked during the times I've tried it. However, since I don't have the internet connection working, I'm not sure if the connection would work after resuming from sleep. Sometimes that's a problem with Linux.

-Sound doesn't work. I don't know how to fix this, but I guess I'll have to do some research on it. If somebody knows how to make it work, I'd appreciate it if you told me.

-Installing the ATI drivers didn't work when I tried it using the new application included in 7.04. I don't know, but maybe it needs an internet connection. It didn't inform me of anything.

-When I tried selecting "Enable Desktop Effects", something happened on the screen, but after a short while it turned back to normal, but nothing had changed. It probably didn't work because the ATI drivers weren't installed, but it would have been nice if it would have told me that.

-I also wonder where the new Control Center is?

Well, I haven't spent much time using the beta yet, but these are my experiences so far.
