### Post by hesee on 2007-10-23
Hi,
I've been struggling to get my macbook work together with Benq FP241VW lcd monitor. With OS X there was no problems, and actually this was the last thing why I was using OS X anymore. I could set some smaller resolutions which were working, but that was not an option, I have to have the whole 1920x1200 :) 

Anyway, because I had so much difficulties to get this combination working, here are the instructions, in case someone has same problems:

[http://www.thinkwiki.org/wiki/Installing_Ubuntu_6.06.1_on_a_ThinkPad_R60e#Ubuntu_7.10_with_Intel_Graphics_Media_Accelerator_950](http://www.thinkwiki.org/wiki/Installing_Ubuntu_6.06.1_on_a_ThinkPad_R60e#Ubuntu_7.10_with_Intel_Graphics_Media_Accelerator_950)

It's actually for Thinkpad, but it works for macbook too. Also what is great is that with xrandr commands you are able to change which screens you are using without xorg.conf modifications (and x restart). I have one minor problem still: when I try to change video player etc fullscreen in Benq, it uses only part of the screen (1280x800 pixels) instead of fullscreen.

 If you have questions, i'll try to help.
