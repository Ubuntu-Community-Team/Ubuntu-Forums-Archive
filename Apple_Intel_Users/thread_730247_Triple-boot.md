### Post by blznazn on 2008-03-22
You can resize your OS X partition as long as the data size isn't bigger than the size (I'd recomend like 50[OS X], 50[Ubuntu], 60[Windows]).  Use the diskutil either the GUI or the terminal version.  But install ubuntu before installing windows, otherwise windows will break with a hal.dll error.
