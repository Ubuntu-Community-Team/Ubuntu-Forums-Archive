If you are trying to get the Bootcamp driver cd, then you do not have to 'run' bootcamp. Although I don't know if the rivers included with work on Win2003 anyway...

In OSX, go to /Applications/Utilities and right-click on Boot Camp Assistant.  Select Show Package Contents and then open the Contents/Resources folders.

There's a file named Diskimage.dmg. You can burn it to CD with DiskUtility.
