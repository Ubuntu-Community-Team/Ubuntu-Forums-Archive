### Post by cyberdork33 on 2007-10-04
I haven't done this before myself, but you might want to go ahead and install gutsy as the dual screen setup is completely different. No xinerama anymore.

For your wifi, you are probably going to want ndiswrapper, which requires a windows driver. You could also try bcm43xx, but I don't think that it works with these newer broadcom cards.

This should work:
[https://help.ubuntu.com/community/WifiDocs/Device/Broadcom_BCM4311_rev_01_%28ndiswrapper%29](https://help.ubuntu.com/community/WifiDocs/Device/Broadcom_BCM4311_rev_01_%28ndiswrapper%29)
