### Post by {{corona}} on 2007-08-01
Hi i cant seem to set Super_L key in combination with any other key using the "keyboard shortcuts" manager under System>Preferences. It does not let me record anything else that i press while holding Super_L key. It ends up capturing just the Super_L key. Any ideas?

Thanks!!

---

### Post by tomaspollak on 2007-08-02
You need to go to the Layout Options (in the Keyboard configuration), and make sure that in Alt/Win key behavior, the selected checkbox is "Super is mapped to the key (Default)". That's it.
