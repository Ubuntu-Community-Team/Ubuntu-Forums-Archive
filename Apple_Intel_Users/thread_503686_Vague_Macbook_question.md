### Post by cyberdork33 on 2007-07-18
you are not going to get native speed in a VM. It does run fairly well though. Running in a VM does have limitations though, such as lack of 3D acceleration (although this is changing).

Parallels is really more focused on running windows, you might want to look into vmware fusion which has much better linux support IMO. It is still in beta (well it is a realease candidate now), but runs well.
