### Post by oskarloko on 2007-04-27
No, it's not  a virtual machine... 

Wubi installs Ubuntu more like a standart installation than a VM.

The differences with a normal install are:
- It install it to an ISO file, rather than a ext3 partition
- Uses the winxp bootloader
- a swap file, no swap partition

Well, I managed a triple boot on my machine... but yes, this can be an alternartive to not mess newbies with  harddddd triple boot methods...
