### Post by Ghola28 on 2007-11-19
There's a tutorial on how to install Ubuntu on a MacBook: [https://help.ubuntu.com/community/MacBook]("https://help.ubuntu.com/community/MacBook")

I used it to install the 64-bit version of Ubuntu Gutsy on my Core 2 Duo Macbook.  It works great, though I had to fiddle with madwifi to get wireless internet working.  

Remember you can burn the Ubuntu install disk to a CD and boot from that to try out Ubuntu before installing it on your system.  

As far as being "production-ready," I've wiped OS X from my harddrive and only use Ubuntu.  With [Wine]("http://www.winehq.org"), I can actually do more than I could in OS X (namely buy Windows games that never got ported to the Mac and play them on my Macbook).
