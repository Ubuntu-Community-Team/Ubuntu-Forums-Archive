### Post by BrettW on 2007-04-07
Just to note: I'm not a MacBook Pro user (not yet, at least..)

The cool thing that I've noticed though, since the change from PPC to Intel Core Duo/Core 2 Duo's, is that Mac OS now has Unix built in, per se. Not exactly built in.... but the Unix technology is used as a backbone of the Mac OS for the architechture. So, obviously, there's now a console! yay! I played with a desktop Mac on Intel CPU in a retail store recently, and was able to run stock-standard unix/linux commands. So really, I don't think you'd need to actually install Linux on the computer at all, you *should* just be able to use it as you like, at the same time as all the other mac stuff.

Hope this helped

-Brett

### Post by davidsiegel on 2007-04-07
So, I was in the same position a few months ago. I owned a PPC Powerbook, on which I was running Ubuntu and Mac OS X. I decided to buy a new Intel-based notebook because I wanted more support in Ubuntu, for software, 3D graphics, etc. I was deciding between a Macbook, Macbook Pro, and a ThinkPad. For $1,500 I configured an AMAZING ThinkPad - Lenovo is basically giving away computers. I had all of the accessories - extra batteries, 2gb RAM, huge HD, ATI graphics, etc. But then I saw the ship date for my order was three months out so I canceled it and bought a Macbook Pro. Running Edgy was a bit painful, but under Feisty everything works really well. I'm happy with the DVI output, bluetooth, wireless, and nice trackpad. Also, I wanted the security of knowing I could boot OS X if I need to. Basically, I would say it boils down to this:
[LIST=1]
[*]Do you need your machine to work flawlessly on a fresh Ubuntu stall? If so, go for the ThinkPad, although the playing field may be leveled at the next Ubuntu release.
[*]Is price an issue? If so, get a Macbook - that seems to be the highest quality your dollar.
[*]Do you want a great system that can still run OS X? If so, don't get the ThinkPad![/LIST]

### Post by ronocdh on 2007-04-07
David has said it well. I'm typing this on a triple boot C2D MBP, and I couldn't recommend it more highly. Final Feisty will ship with built-in support for most of this hardware, so you won't have to spend extra setup time configuring bits and pieces here and there.

Yes, right now on Edgy you'll have to install extra things to utilize the advanced functionality of your hardware: adjusting screen brightness at a keystroke, adjusting keyboard backlight, configuring a second mouse button. There are excellent keyboard remappers for both Linux and Win XP... my personal favorite solution is to use the Enter key--Macs have both Enter and Return, remember--because that's conveniently placed, but Function-click works really well too!

All in all, I couldn't recommend it more highly. Right now there's a bit of extra config time... by the time Feisty ships, or even if you're running the Feisty beta, you'll be just fine. You can still run Windows AND Linux and even OS X--that's something you can't do with a Thinkpad. My endorsement is based solely on the hardware, really. Sudden motion sensor, backlit keyboard, Firewire 400 and 800 in addition to USB2, optical audio out... it owns. =)
