### Post by omax on 2007-09-21
Solution:
```

1. Run an apple update from Mac OSX
2. Download "Psst" from http://www.satsumac.com/Psst.php
3. Mount dmg and install, set volume to none in Psst,

```

On reboot, there is no chime.
