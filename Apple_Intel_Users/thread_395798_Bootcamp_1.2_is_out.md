### Post by ronocdh on 2007-03-31
Which problems are you referring to? The Boot Camp update provides better driver support for XP and includes support for Vista. This is from Apple's release page:

[INDENT]Boot Camp 1.2 beta contains several updates and is intended for all new and previous Boot Camp beta users.

Boot Camp 1.2 beta includes:

    * Support for Windows Vista (32-bit)
    * Updated drivers, including but not limited to trackpad, AppleTime (synch), audio, graphics, modem, iSight camera
    * Support the Apple Remote (works with iTunes and Windows Media Player)
    * A Windows system tray icon for easy access to Boot Camp information and actions
    * Improved keyboard support  for Korean, Chinese, Swedish, Danish, Norwegian, Finnish, Russian, and French Canadian
    * Improved Windows driver installation experience
    * Updated documentation and Boot Camp on-line help in Windows
    * Apple Software Update (for Windows XP and Vista)[/INDENT]


Although this is good news for a lot of triple booters, I don't see it easing issues regarding Ubuntu, especially with the impending release of Feisty. Nonetheless, I give it a thumbs up. =)
