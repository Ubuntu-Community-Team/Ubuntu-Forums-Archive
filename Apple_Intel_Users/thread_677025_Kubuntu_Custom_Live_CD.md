### Post by david_edmundson on 2008-01-24
It's certainly do-able. There are plenty of tools out there for creating LiveCDs. The most common probably being remastersys. Which creates a custom version of the normal LiveCD/Installer which installs to exactly how your system is set up.

Here is a how-to:
[http://www.howtoforge.com/ubuntu-linux-mint-livecd-with-remastersys](http://www.howtoforge.com/ubuntu-linux-mint-livecd-with-remastersys)

However, in terms of drivers I think the Live version and Installed version use a different Kernel, swapping out the LiveCD kernel may be tricky.

In terms of getting everything going I made a Ubuntu package with working suspend and wifi on the latest Kernel [http://ubuntuforums.org/showthread.php?p=4030061that](http://ubuntuforums.org/showthread.php?p=4030061that) might help get most of it going.
