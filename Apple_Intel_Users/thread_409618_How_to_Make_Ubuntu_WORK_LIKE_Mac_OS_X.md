### Post by jordanmthomas on 2007-04-14
Spotlight:  Try beagle + deskbar (there is also a project called tracker that aims to replace beagle)
[img]http://img365.imageshack.us/img365/6899/beagledeskbarintervalnf3.png[/img]

There is a way to make the panel have the menu of the application, as well:
[http://ubuntuforums.org/showthread.php?t=241868](http://ubuntuforums.org/showthread.php?t=241868)

If you're interested in beryl, there is a widget plugin so you can bring your widgets up like in Dashboard rather than having them visible on your desktop.

I know you know about replacement docks, but you should check out avant-window-navigator if you haven't already:
[http://code.google.com/p/avant-window-navigator/](http://code.google.com/p/avant-window-navigator/)

That's all I really have to add for the time being.
