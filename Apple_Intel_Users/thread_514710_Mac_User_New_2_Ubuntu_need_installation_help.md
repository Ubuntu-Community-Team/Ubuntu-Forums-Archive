### Post by cyberdork33 on 2007-08-01
The "latest's" support is running out sooner because there will be a newer release in october, which, will come with a new support date. The LTS versions of ubuntu are mainly for business use and is only really beneficial to use if you also buy a support contract. 6.06 is lacking in the Mac support anyway, so use the latest.

you may want to get a trial of one of the VMs just to play around with ubuntu and see if you like it, then move on to the native install. Just a suggestion.

You can use the 'standard' version or the 64-bit version. I suggest the i386 version because you will likely encounter less problems with software later.

To dual boot with native Ubuntu, you have to resize your OSX partition. This can be accomplished with Boot camp as well as a few other tools. Here is a guide for version 6.06, but the basics are the same for any version. [https://help.ubuntu.com/community/Mac_mini](https://help.ubuntu.com/community/Mac_mini)

You might also want to look into getting rEFIt installed. This will present you with a menu to choose the OS your want at bootup. It defaults to using OSX if you do not select anything.

Lastly, be sure to read through the various threads here, as you will inevitably run into some issue, and there is a lot of information about getting everything working properly in the forum.
