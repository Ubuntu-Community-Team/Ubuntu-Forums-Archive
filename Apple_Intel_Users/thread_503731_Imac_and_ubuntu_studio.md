### Post by apotaktikos on 2007-07-19
[LEFT]Hi,

I did this a few days ago and it went very well. I cheated somewhat and used the scripts I found here 

[http://ubuntuforums.org/showthread.php?t=463791](http://ubuntuforums.org/showthread.php?t=463791)

I recommend you do the same. I've never had an installation run so smoothly. No endless searching forums and googling (and then working out which advice to follow) to get everything configured. I followed all the steps to the letter and  now have ati-fglrx-xgl with compiz-fusion and isight (and frankly getting that to work was a bonus I didn't expect!). I haven't checked madwifi yet so can't vouch for that. The only problem I had was preparing the partition. I use Refit as the bootloader which works very well, but using Bootcamp to partition the drive was a pain as it wouldn't let me specify the size I wanted because it ran into files it couldn't move. Eventually I moved as much data as possible to an external drive and got it to work. The install itself was one of the fastest I've ever done, and I have to say that Ubuntu Studio just looks amazing on an imac!!

I hope it goes well for you.
