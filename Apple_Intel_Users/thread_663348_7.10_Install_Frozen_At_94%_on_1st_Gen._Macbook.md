### Post by cyberdork33 on 2008-01-10
> **Apt403 said:**
> I'm trying to install Gutsy on a first gen Macbook following [this guide]("https://help.ubuntu.com/community/MacBook"). It says Boot Camp will cause problems if I create more than 2 partitions, so how do I allocate space for the swap partition?

#1. You do not have to use Bootcamp to partition, but it is a method you can use.

#2. Create a partition with Bootcamp (so you will have an OSX partition and a Windows partition). Then Boot the Ubuntu Live CD. Start the Partition Editor from the menu and select the last partition on the disk (probably a FAT32 partition), and delete it.

#3. Start the Ubuntu Installer and, when prompted, choose to "use the Free space"... The installer will create a swap and root partition in the place where the FAT32 partition used to be. (You can also choose to manually partition and create the partitions you want if you know what you are doing.)
