### Post by cyberdork33 on 2008-03-18
you are not saving it in plain text. If it was plain text you would not be able to bold, etc. When you open a new file, set it to plain text by going to Format > Make Plain Text (or Shift+Cmd+T).

It would be to your benefit to get a better text editor such as [smultron]("http://smultron.sourceforge.net/") that can do syntax highlighting and such.

Finally, this is a linux forum, so the best advice I can give you is to use Ubuntu !
