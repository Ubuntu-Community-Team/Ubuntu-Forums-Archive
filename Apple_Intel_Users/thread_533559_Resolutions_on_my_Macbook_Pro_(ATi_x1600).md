You said the icon size was the problem...

> **pabloiran said:**
> But my mother says that icons seem very small...

No matter, font size can be adjusted even easier. In the main menu in the top panel:
System > Preferences > Font

Adjusting the resolution for readability is not correcting the problem. It only lowers image quality. The native resolution for you Mac's screen is 1680x1050, which is where it should be, and should stay. If icons need adjusting, adjust them. If fonts are too small, make them bigger.
