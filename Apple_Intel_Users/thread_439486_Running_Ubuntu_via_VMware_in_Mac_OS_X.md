### Post by Lt_Data on 2007-05-10
To the Mac people that are having trouble with their Ubuntu installation: give VMware Fusion a try! I had a lot of trouble having my WiFi work (<shakes fist at sky> Damn you Atheros! </shakes fist at sky>) and I tried ndiswrapper and MADWiFi - I honestly didn't like the way everything was going and personally couldn't get it working properly, or working properly for very long. I tried VMware fusion (which didn't recognize my partiotion made with boot camp at first, then boot camp kept crashing when I tried opening it) so I got rid of that and went through the process of installing Ubuntu 7.04 via VMware. It's pretty sweet, I personally like the fact that I don't have to reboot to access my main OS and can run Ubuntu in a window in Mac OS X. And with two key strokes I can alternate between the two operating systems flawlessly, as well as make Ubuntu run full screen without having to boot into it with just two keystrokes. I'm sure it isn't for everyone, but I implore those that are willing to try some new things to give it a try. It recognizes all the mac controls and peripherals as well as the Wireless (and you don't have to sacrifice your connection strength by using an open source alternative).

I've got nothing more to say, I'm just excited it's working well. Discuss amongst yourselves!

### Post by arsh89 on 2007-06-06
> **Y2joey6 said:**
> Do you know where i can find a walk through to install Ubuntu on Macbook via VMWare? If someone could help a newbie it would be greatly appreciated. Thanks


i couldnt tell you a walkthrough but what i did was download ubuntu(make sure its the 32-bit one. if you are a newbie, that all you should need) as it should come, as an iso. then on the vmware startup, it should give you the option of creating new virtual machines. rather than picking a pre-existing, create the new vm. select the linux os and the debian linux type. create a file where you want all your information stored. lastly, and most importantly, it will tell you to insert the cd from which to boot the os. if you have the downloaded version, click the "more options" link, select that you want to boot from an iso, and specify where the location of the iso is. that should be all you need to do. the computer should take care of th rest.

as for further configurations, i couldn't help you. i used fedora with vm, not ubuntu. good luck.
