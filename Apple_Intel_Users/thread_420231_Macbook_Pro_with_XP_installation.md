### Post by ronocdh on 2007-04-23

> Hey As the topic says I have a Macbook pro with a Xp installation ONLY (No Mac Os X) installed. What is the easiest way to create a partition in windows and install ubuntu onto the partition thus multi booting between the two o/s`. thank you.
The [GParted CD]("http://gparted.sf.net") should treat you quite well, but please wait for confirmation from others here in the forum; I'm not sure whether it'll work nondestructively on partitions in Windows, though I've used it elsewhere and had great success.

And I know it's a bit solicitous of me, but I recommend you consider triple booting. OS X will be needed for any firmware upgrades to your hardware (I think upgrading your wireless to N speed, for instance). It's worked well for me so far, and using the OS X installer DVD you can strip down the installation to take up very little room.
