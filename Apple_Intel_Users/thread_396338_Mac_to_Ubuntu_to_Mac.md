### Post by fredronn on 2007-03-29
Yes, you're going to be able to reinstall OSX just fine. Keep in mind though, that if you go with the easy install and devote your entire drive to Ubuntu (and don't go via refit), you need to use disk utility available on the install disc (before choosing installation disk) and erase the entire disk so you can get back an HFS partition.
