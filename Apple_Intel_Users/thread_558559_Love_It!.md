### Post by freakalad on 2007-09-24
Howdy!

I've been running Ubuntu Feisty Fawn desktop inside Mac OS/X via VMWare Fusion for close to a week now, and I have to admit that I love it to bits!

Speed & stability's great! Better on my new MacBook Pro than on my old laptop.
I can switch between Linux, Mac & Windows instantly without the need to reboot; been told I can port my VMWare images to my other systems (win32 to my Ubuntu media server).

Couple of issues that are still a bit lacking, such a as Beryl in Mac (which should be resolved with the next OS/X release), and the ability to use linux apps 'natively' in mac, the way win32 apps run via Unity.

Considered going the route of Boot Camp & rEFIt, but too many ifs & buts. Not an elegant enough solution for me. 
Besides, OS/X is SO DAMN SLICK! Has much of the spit-&-polish that I feel many of the Linux desktop distro's lack. Besides, with a bit of tweaking, I can get many of my Linux tools to run just fine in Mac's X11, so I don't feel I'm really cheating (and I get to dodge an application layer)

I'd be interested to know what your take is on Linux on Mac, as well as any tips, advise & help you might have to offer.

### Post by freakalad on 2007-10-02
you use a combination of modifier keys with the button to achieve additional-clicks, such as the apple, ctrl, alt and/or shift. You can also set up advanced touchpad behaviour in you preferences.
I'm personally not very big on the idea of a single mouse button (though I can see the value), nor am I too comfortable using a touchpad, so I make use of a standard USB pointing device such as (my personal choice) Logitech's wireless laser laptop mouse or the trackball (use both with no hassles).
Look out for updates in the new OS/X release; standard features might include multiple desktops & 'slapbook' features.
Proprietary's OK in some cases, as long as there's delivery on promises. Besides, I only make use of FOSS, for both my OS/X & Ubuntu platforms, so it's six of the one, half-a-dozen of the other. Give credit where it's due...
