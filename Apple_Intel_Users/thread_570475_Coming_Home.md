### Post by wdo_will on 2007-10-08
I would just like to report how happy I am - after getting a MacBook Pro (Santa Rosa) back in July and living with OS X since then (OS X is an okay OS in and of itself, but nothing beats Ubuntu), I finally decided to work on setting up a triple-boot configuration with OS X, Ubuntu (my primary OS), and XP (for the one game that I play).  For whatever reason, Feisty would never load off of the Live CD, so I just decided to go with the Gutsy beta (I also though about trying Feisty x64, but I'll be doing a decent amount of programming on Ubuntu, and wanted to stay with x86).

If anyone wants a comprehensive guide to setting up Gutsy on a Mac, I might give it to you.  It actually was, for the most part, simple - I really just combined the procedures from guides across the interwebs and used a bit of my own guesswork.  For installation itself, actually, I believe there was only one instance where I *had* to use the command line (in OS X, actually), although I much prefer the command line to a GUI when doing more simple tasks, so I used it liberally.  To the best of my knowledge, Gutsy really increased compatibility with Intel Macs, so bravo to the Ubuntu team for another excellent release.
