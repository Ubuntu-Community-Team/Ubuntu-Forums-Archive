### Post by cyberdork33 on 2008-02-14
Apple hardware is not the best to run Ubuntu on, and I wouldn't buy one only for that... That being said...

There is a wiki page describing how to get most if not all the hardware working decently. You will probably want to click through to the Santa Rosa version of the page.
[https://wiki.ubuntu.com/MacBookPro](https://wiki.ubuntu.com/MacBookPro)

WiFi is Atheros which works with madwifi (or Broadcom on other Apple hardware which works only with ndiswrapper at the moment.) Bluetooth works without any tinkering.

The nvidia video card in the Macbook Pro did have issues at one point, I don't know if they have all been ironed out, but once you get the proprietary drivers installed, Compiz should have no issue.

There are some touchpad issues as the Apple touchpads seem to be a bit squirrelly, but there is infomation here about getting that working for the most part. There are even some patches to add functionality that is similar to OS X in one thread.
