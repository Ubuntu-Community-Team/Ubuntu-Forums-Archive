### Post by benanzo on 2007-08-22
I don't believe Apple has any incentive to support a Linux/MacOS dual boot option with Bootcamp.  The whole reason Apple offers Bootcamp is to appeal to Windows users who want to try a Mac but don't want to give up their Windows programs.  Apple's hopes, in that case, are that those users will see that OS X is better than Windows and that they will eventually become "switchers" and not use Windows at all.  There are no (or very few) "switchers" from Linux to OS X.  It takes a conscious effort to install and use Linux.  Most Linux users aren't going to switch.  Besides, Linux is a breeze to install in a dual/triple boot system as it is.  I'm not sure what benefit would be gained by having Apple "support" it.
