### Post by ronocdh on 2007-05-20
> **andrewbossie said:**
> im running Ubuntu Feisty on my 1st gen mac book as the only operating system on it. i then realized that there are some things that i need osx for. so is there a way that i can install osx and keep linux on my mac book?
I think the best thing to do would be to wipe the drive completely, install OS X, then install Feisty on a separate partition. You can keep OS X to a small partition if you don't think you'll use it often. Just back up your /home directory, that's what I recommend.

That said, you can always try iPartition or even GParted, maybe you'll have some luck there. But as I understand it, the OS X Installer disc isn't friendly with installing onto just a partition. I haven't tried, though, so please experiment and post back with results! (Back up first, though.) ;)
