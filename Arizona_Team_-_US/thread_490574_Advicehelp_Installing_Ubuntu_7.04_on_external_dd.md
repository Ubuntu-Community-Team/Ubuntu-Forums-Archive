### Post by bribri124 on 2007-07-03
I am in Tucson, but I think I can help you.  Yes, it is possible to install Ubuntu onto an external drive.  The first step is to make sure your BIOS can boot a USB device.  If this is possible, you should have no problems installing Ubuntu.  Assuming your system has only 1 HD installed, when you load the disk, change the installation drive to (hd1).  You will be able to install GRUB, and Ubuntu will set up 2 partitions: swap and root.
