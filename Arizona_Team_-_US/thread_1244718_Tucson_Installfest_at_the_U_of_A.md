### Post by johnc4510 on 2009-08-19
We've got everything set up now for our Ubuntu Installfest at the University of Arizona in Tucson. This is a co-hosted event with the Computer Sciences Dept. Details below:

Location: University of Arizona
          Gould-Simpson Building 
          1040 East 4th Street
          9th Floor Atrium

Map showing building and parking: [http://tinyurl.com/pr75md](http://tinyurl.com/pr75md)
Anyone coming from Phoenix needs to exit the St. Mary's exit, then turn East(Left) under the Interstate. St. Mary's turns into 6th Ave. so don't be alarmed. The University is about 2 miles from where you exit the Interstate.

Date: Saturday
      August 29th, 2009

Time: 11:00 AM - 5:00 PM

We can have access to the building at 10:00 to start setting up for the event. We have plenty of Ubuntu pre-burned CDs to give out. We'll also have some Ubuntu Stickers available. Free WiFi will be available along with tables and chairs. We also think we're going to be able to have access to one room off the atrium for demos (projector included with the room).

Downloadable flyer for the event is here: [http://www.flickr.com/photos/22090195@N03/3834675045/](http://www.flickr.com/photos/22090195@N03/3834675045/)

This is going to be a great event for our team, so don't miss out on the fun. 

johnc4510
