### Post by mtod1 on 2009-02-25
Hi all.

I'd like to suggest the development of a single piece of software that could be used for the downloading, editing, submitting and viewing of Ubuntu documentation.

Something along the lines of an app that will allow documentation contributors to download, review, edit, save and upload help and other Ubuntu documentation. 

I'm imagining an app with a gui frontend with menu options as described above and others as neccesary. Help documentation could be opened and viewed as standard help files, however, with the ability to edit, save, upload if they would like to contribute

Perhaps it might offer the ability to search documentation topics for those who wish to contribute to particular areas.

Edited documentation could then be directly upload to the documentation team for review, approval etc.

I am no programmer or developer, but I would like to contribute and I think an app such as the one I have outlined would make it a much simpler process than that presently used.

Let me know what you think and make some suggestions.
p.s I have also posted a similar idea on brainstorm. idea #18104

Cheers
Mark
