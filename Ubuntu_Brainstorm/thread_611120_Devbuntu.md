### Post by ZacDavis on 2007-11-12
I read about this in the idea pool, and that it was a pretty good idea, as I would've found it extremely useful, and I think others would find it useful too.  I was wondering how much interest there would be in using Devbuntu.

In case you were wondering Devbuntu would be a spin-off of Ubuntu, geared toward developers and programmers.  The repositories would contain all the latest updates to all the development/programming apps possible.

I would also like to find some people to help out with this.  There wouldn't be a real need for programmers, but there would be a need for packagers, package maintainers, and artwork.  I don't believe I missed anything there.  An and all ideas would certainly be considered as well.

If this peaks some interest, I could start working on it pretty soon.  I don't see any sense in doing it, though, if not a lot of people are interested.
