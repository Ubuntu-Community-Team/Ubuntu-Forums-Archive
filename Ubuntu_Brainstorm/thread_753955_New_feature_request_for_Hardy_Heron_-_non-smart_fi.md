### Post by natirips on 2008-04-13
I'd like SO VERY MUCH if Nautilus could sort files in a non-intelligent way, for example:

0
01  <----
1

Not like it does:

0
1
01 <-----

So I'd like Nautilus to be able to sort the filenames the simple way, like "ls" in console does. So that two files starting with '0' (zero character) ("03" and "08" for example) are placed together, not "03" with files starting with '3' and "08" to be grouped with files starting with '8'.

P.S.: There is an older thread concernig this issue: [http://ubuntuforums.org/showthread.php?p=4690231](http://ubuntuforums.org/showthread.php?p=4690231)
