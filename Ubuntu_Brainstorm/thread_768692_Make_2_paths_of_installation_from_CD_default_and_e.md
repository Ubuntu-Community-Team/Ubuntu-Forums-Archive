### Post by Gina on 2008-05-05
The Minimal Install does this.  The [mini.iso]("http://archive.ubuntu.com/ubuntu/dists/hardy/main/installer-i386/current/images/netboot/mini.iso") of just 9.5 MB installs only what is required to download data from the repositories.  You can select what software you want or even a CLI install from which you can select a very minimal installation by installing what you want from the command line.

How about that for an Experienced User Installer?
