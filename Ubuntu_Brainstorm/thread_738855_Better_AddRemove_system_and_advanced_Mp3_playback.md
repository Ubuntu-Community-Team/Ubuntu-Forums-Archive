-If Ubuntu wants to become more mainstream, there has to be an easier way to uninstall programs. FOR INSTANCE.. Google Desktop cannot be uninstalled in the conventional sense. It should be much easier than what I had to go through to get it off my computer, especially for a program created by Google.

-I have a 500 gb external HD that I store most of my music on. It would be great if a native ubuntu prog like Rhythmbox could recognize the difference between songs stored on my external HD and my internal HD. So, if my external is unplugged, it would only display the songs on my internal HD-- as it currently stands, it will display ALL songs, but (like iTunes), there will be thousands of broken links. I just want a more intuitive mp3 playback prog that knows the difference between internally/externally stored music, and shows me ONLY the music I currently have available on my computer so I don't have to get frustrated by being tempted to play music that really isn't there.
