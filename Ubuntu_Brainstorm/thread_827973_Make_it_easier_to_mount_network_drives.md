### Post by Johnsie on 2008-06-13
On Windows its pretty simple... just click "map network drive", select the folder you want to map and choose a drive letter. You can decide if you want it mapped at boot time just by checking a box. It's all easy to do and has a pretty intuitive and simple gui.

I find that if I want to do this in Linux  I have to follow long tutorials and edit config files (fstab). The average user should be able to map/mount networks drives in an easy, quick and intuitive way. So I think this is one way Ubuntu can be improved.
