### Post by 23meg on 2007-11-21
The [GNOME Human Interface Guidelines]("http://library.gnome.org/devel/hig-book/stable/") recommend applying settings immediately in property and preference windows, and using "explicit apply" windows (those with a separate "Apply" and "Cancel" buttons) only when "most controls in the window are not suitable for instant apply".
