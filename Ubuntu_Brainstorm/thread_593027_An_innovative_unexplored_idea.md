### Post by joshier on 2007-10-26
Hello,

Being a bit of a lurker here on the forums, I don't often post topics, but I'm heavily interested in the future of ubuntu, and linux as a whole. 

Recently I have come across a really special - buggy - but great -working concept - idea.

The idea is simple. Ubuntu is installed onto a USB stick, and it goes onto a cw write partition, allowing the default live CD to think that it's using the live cd still, but is in actual fact, running off of the memory stick.

The tutorial over at [http://www.pendrivelinux.com/2007/09/28/usb-ubuntu-710-gutsy-gibbon-install/]("http://www.pendrivelinux.com/2007/09/28/usb-ubuntu-710-gutsy-gibbon-install/") tells you how.. but I've not had much luck. It will install onto a USB stick fine, and I will be able to run off of the USB stick with recent machines and use the live cd (non write) mode and it works great - but upon loading the save mode on option, it does not seem to work.

When I say 'it does not seem to work' it will only work on the original machine.. which is a great shame, but I think if we work together we could get this going.



My hope and aim for this whole idea is being able to buy 2/4/8/16GB USB sticks officially from ubuntu's website, then being able to pop it into *any* computer, having it run without any bugs... 

I have lots of other ideas too.. for example, different users (say, you could have - Office - Home - Mikes house ) .. so that, you are able to load up the drivers and/or system preferences to get those particular hardware set-ups running as you would like them.

I'm going a bit far now, but you can see where this is heading and I really think it's a fantastic idea.. one of which microsoft would never be able to combat.
