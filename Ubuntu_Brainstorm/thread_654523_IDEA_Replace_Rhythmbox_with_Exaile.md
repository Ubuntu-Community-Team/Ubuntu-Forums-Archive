### Post by mihai.ile on 2008-01-03
Ok before gusty the battle was between banshee and rhythmbox.
Banshee loosed that time and sincerely I found Exaile worse than banshee. I don't like amarok, and exaile, and Windows media player.
All of these have something in common. Are very confusing.
Rhythmbox is very stable, and to speak in terms of features...
- crossfading support
- automatically transcodes music in mp3 for ipod*
- magnatune plugin
- music recomendations plugin (not official yet, I am helping on that)
- Play queue
- brose genere/artist/album
- visualisations
- from bugzilla I saw that there is work for an equalizer
- generic portable players support, and Sony PSP
- python plugins support, easy to learn, easy to add new features
- a very intuitive and simple interface
- watches library for new files, keeps history of missing files and when last seen on hdd.
- radio, podcast support
- FM radio tuners support.
- very fast searches in big music libraries

Ok many of these still don't work perfectly or missing little things but there is no player that does all this perfectly.
There are still features that are missing from Rhythmbox but there is NO reason to replace it with other players.
If you like other players install them, and use them, no problem. I personally use more Banshee than Rhythmbox because of lack of features but searching a little in Bugzilla you'll see as I saw that all those features that are lacking actually are being coded.

Instead of asking for another player i decided in helping with the Music recommendations plugin, if you like Rhythmbox help the team make it better.If not, you are free to install and use another player.

I have no python skills, never saw code written in python but in two days I almost made a complete refactoring of the plugin.

I think is good to have a gnome application that has some of the Amarok look&feel but don't think that is a good ideea to come as the default music player. Definitely not.

And one again.. this is my personal opinion.

*transcoding to ipod I think requires gstreamer ugly or mad installed and under preferences->music set the preffered format to mp3
