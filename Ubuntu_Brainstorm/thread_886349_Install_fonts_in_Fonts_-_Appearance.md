### Post by Ub1476 on 2008-08-11
I think there should be an option next to "Details" to "Install new fonts". Nothing to fancy, simply click, choose "local" (installs to ~/.fonts) or "system" (/usr/share/fonts). 

Both single .ttf files and folders/archives which contains fonts (.ttf) should be installable.

I don't think it's a very important thing, but it's a matter of user friendliness and provides a more complete interface.
