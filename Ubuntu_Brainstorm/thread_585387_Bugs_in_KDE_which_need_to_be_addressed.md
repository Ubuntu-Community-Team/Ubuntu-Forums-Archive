KDE 4 may not make it to the LTS release of 8.04.  So a few bugs present in KDE 3 itself, need to be fixed before hardy heron is released.

Here are a few bugs:

[http://bugs.kde.org/show_bug.cgi?id=146692](http://bugs.kde.org/show_bug.cgi?id=146692)
[http://bugs.kde.org/show_bug.cgi?id=150222](http://bugs.kde.org/show_bug.cgi?id=150222)

Some of these, I have reported at launchpad and mostly got a response that they must be fixed upstream.  If Kubuntu community can fix it upstream early enough for the fixes to make their way into Hardy, it will be good.

There is also a kubuntu specific problem (magnified due to use of Dolphin as the file manager), which is available at [https://bugs.launchpad.net/ubuntu/+source/kde-systemsettings/+bug/50320](https://bugs.launchpad.net/ubuntu/+source/kde-systemsettings/+bug/50320)

If these issues are solved before the release of Hardy, KDE in Kubuntu would be really good.  I request the programmers in the community to work out a solution for these bugs.

If any of you have been facing other bugs in KDE available in Kubuntu you are welcome to list them too.
