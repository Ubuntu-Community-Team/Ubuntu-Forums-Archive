### Post by ericesque on 2007-11-22
Extreme caution?  I think that statement is a bit extreme.

I used an alpha version of Wubi to install 7.10  and it worked beautifully.  Of course, any time you mess with filesystems, make sure you back up critical data first.  That said, it is rather unlikely that Wubi will destroy any data.   Wubi is purposed similarly to a live cd--without the performance hit.  After I decided Ubuntu was here to stay, I uninstalled Wubi (which removes the Ubuntu install) repartitioned, and installed Ubuntu in standard fashion.
