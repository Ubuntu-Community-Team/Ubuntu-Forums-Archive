### Post by Krydahl on 2008-02-07
Not a Mac user, so not entirely sure this is what you're after but...

You might like to check out the Control Centre. It's under system -> preferences, but I think it's hidden by default. Right click your menus, select edit menus, find the control centre under preferences and tick the box then you ought to be able to see it.

If it does what you want then you can add a launcher to the desktop or a panel for easier access.
