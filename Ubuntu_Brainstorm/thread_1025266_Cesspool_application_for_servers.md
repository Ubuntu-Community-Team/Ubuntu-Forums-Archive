### Post by arvevans on 2008-12-29
There are programs floating around on the internet which only serve special interests, in a manner which is detrimental to the rest of us.[LIST=1]
[*]email harvesting programs which scan any open file/server for email addresses.
[*]URL discovery programs which probably are the basis of some attacks on our servers.
[*]personal data discovery and harvesting programs.
[*]"login" locators which are then used as the focus of password cracking programs.
[*]...and many more obnoxious applications.
[/LIST]

These obnoxious and potentially disruptive applications use up bandwidth on local links and on the main Internet backbone.  Use of the data which these programs discover adds even more load to the Internet bandwidth.

Maybe what we need is a "cesspool" program which looks attractive and potentially usefull to these information harvesting bots and applications, but in reality is a sticky-place to draw them in and then support the application of various plug-in reverse-hacking tools (satan, etc.), and which looks like a repository for data but in fact feeds them megabytes of bogus data and info.

Such a black-hole application would require that it banner a warning to anyone who might stumble onto it by accident, but if this were a .gif image it would be difficult for a bot or automated info harvestor to detetect it.

The intent here would be to provide the info-harvesting application with so much fake information that it becomes useless to those who rely on such things.  If done properly, this could possibly significantly reduce the amount of unwanted advertisements that reach your email in-box, and could possibly put some of the less ethical Internet based advertisers out of business.
