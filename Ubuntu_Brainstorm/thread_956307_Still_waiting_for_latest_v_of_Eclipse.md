### Post by gnomeuser on 2008-10-23
If you want the latest version of Eclipse and that is a priority item for you that Ubuntu can't fulfill then look at Fedora - as Red Hat has a lot going on the Java front they generally keep Eclipse updated at all times.

Aside that, you can look at Eric3 (a KDE environment but specially designed for python) or Monodevelop (my personal favorite IDE, it should do python magic or at least have a plugin to do so for IronPython, alternatively look at boo, that's python like as well). Finally the OLPC people are doing a very neat little python IDE for their Sugar interface, maybe you can make that run.
