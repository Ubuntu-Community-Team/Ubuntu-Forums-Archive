### Post by Spr0k3t on 2007-10-21
There needs to be some sort of tool which allows extended configuration of mice, trackballs, and touchpads.

For mice/trackballs, give the option of configuring multiple buttons past three buttons.  In this configuration tool, include the hand orientation, speed/acceleration, double click speed and pointer decorations/animations.

For touchpads and stick points, give the option to disable the touchpad/stickpoint when a mouse or trackball is plugged in.  Another option would be to hotkey toggle the touchpad/stickpoint if such a key does not already exist on the laptop.  Don't include the touchpad/stickpoint tools if none are detected in the hardware.

These ideas have been brought up in the initial development of Gutsy and even mentioned during Feisty.  So this idea is not something new.
