### Post by raynevandunem on 2007-11-18
I know that it would take a bit of hackery for this to work, but would it be possible to give Nautilus "shelves"?

Instead of the plain white background that we're used to in all file managers (Nautilus, Explorer, Finder, etc.), what about shelves that hold the files and folders?

It would look similar[ to this]("http://www.delicious-monster.com/images/librarypage/screenshots/Delicious-Library.png").

However, the shelves would have a 3D look like the current AWN/Leopard Dock, and you would scroll down to see more shelves.

It's not much more than an aesthetic change, but I think it would define "depth" in the window of the file manager, turning it more into a cabinet. plus, it would integrate well with AWN and the current overall "pseudo-3d" trend that's being followed in window management.

It will also be easily themable as well.
