### Post by retnug on 2008-01-06
Dear Ubuntu Community,
I was very fed up with Billy Gate's Approach (BGA) to be kept in constant tutelage so I thought Ubuntu would be the thing to install on my computer &#8211; everything easy, all programmes accessible, I can install my own desktop as I want it, free, finally ....

OH, WHAT GREAT ERROR! Little by little I had to discover that Ubuntu is as authoritative as BGA &#8211; and in some respects even more. Bye bye freedom!

It starts with the required obligatory password &#8211; fastidious! I am the only one to handle my laptop &#8211; and if somehow it gets lost/stolen, than it's MY problem, and I have to deal with that (and not some anonymous ubuntu community, which does not help anyway ...)! So why can't I get rid of this obligation? I hate to type my password when I want to access the data disk also, or when I want to access to Synaptic. And what is that gimmick: the window that appears after having entered the password: YOU CAN CANCEL THIS OPERATION ... - I just WANTED TO OPEN the folder! Completely superfluous.

And there are many &#8220;can't-do's&#8221; like that; here a short selection:

can't make ubuntu silent with startup (I don't want to hear these sounds! With BGA, I CAN delete them, at least ... because I can locate them &#8211; which I cannot with ubuntu, because the file sytem is not clear and some folders can be found only after extensive research; and if you find what you look for, you can't do anything in the folder, because the programme does not give you permission to do what you wanna do ...)
can't rename disks, not even external HD's
can't make Opera my default browser; I am stuck with firefox, which I don't appreciate
can't make self-made shortcuts work
can't create launchers of applications (... and I tried &#8211; I'm not THAT stupid!)
can't delete these dummy fonts where I don't perceive any significant change from one to the other &#8211; even BGA allows to remove all but some 4 or 6 fonts!
can't change menus &#8211; although there is a function, which should allow this. I discovered (also after backup) that there were new launchers in the &#8220;Places&#8221;-menu: Documents, Music, Pictures, Videos. I DON'T WANT THEM! I want to handle my files like I consider is best for me!
can't remove programmes I don't want (often they are in packages, and then I'd have to delete also the ones I would like to have ...); so now I have three vid-players accessible! And each one direct the audio volume of my laptop, especially MPlayer is fastidious, because it does not retain the last order to keep the volume on max; in consequence, when I wanna listen to some music, i don't hear a sound. Further, removed programmes come back: I realised that launchers reappear after backup: Universal Access/Keyboard Management/Palms OS [I don't have a palm, so why should I want to have this icon on this ever longer menu?]
can't change menus &#8211; although there is a function, which should allow this. I discovered (also after backup) that there were new launchers in the &#8220;Places&#8221;-menu: Documents, Music, Pictures, Videos. I DON'T WANT THEM! I want to handle my files like I consider is best for me!
can't remove programmes I don't want (often they are in packages, and then I'd have to delete also the ones I would like to have ...); so now I have three vid-players accessible!
Once removed programmes (with SYNAPTIC!) come back; I realised that launchers reappear after backup, for ex.: Universal Access/Keyboard Management/Palms OS [I don't have a palm, so why should I want to have this icon on this ever longer menu?]

These things are fastidious, OK, and I am convinced that you know about these things. My question: when are you going to do s.th. about it? I think it did not happen before to me this often that I had to FORCE QUIT (i.e. switch off) my laptop because the programme (OpenOffice and Opera especially, sometimes also GIMP) does not respond anymore (caused a crash).

When it comes to make work my hardware, then the problem is more serious: I know how to make my Epson 660 perfection scanner work now &#8211; although the community and the eggheads of Ubuntu did not reply at all, or not conveniently, respectively, on my question in the forum. BUT I CAN'T DO IT! I am denied to save the changed config file &#8211; either directly (found after a long search) or via terminal. The programme simply denies in either circumstance my right to change the file ...
Printer problem: whenever I print out a picture-file, my parameters are not respected (origin x, y; size in cm ...). Here, the same problem: I could change the config file with the correct driver, but I can't save the changes. What is more: even when I changed from inches to cm or mm the print-window, the change is not saved; what a waste of time and money! I happened quite often that I first print the file, realise that it was still in inches, I change that, thry to have the correct parameters, but then it is still not printed according to them.

I thought that with Ubuntu, I could free myself from restrictions another OS inflicted on me, but I see that there are as many &#8211; and often weirder ones &#8211; with this OS (Ubuntu). Sure, there are nice things, which come with Ubuntu, but do they really outweigh the advantages of a clearly presented and therefore easily understandable file system? Where you can make many changes (not all you want, surely, but at least the ones, which convene YOU!) to your agreement?

What's the choice? - especially when considering that most of my games cannot be played with Ubuntu ...?

Greetings from gugu
