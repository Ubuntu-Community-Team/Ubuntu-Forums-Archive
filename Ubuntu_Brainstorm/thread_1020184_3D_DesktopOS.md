### Post by Anonymous111 on 2008-12-23
Ok, heres my idea: 
3D Operating System based on a 3D FPS game

It would be basically like a 3D FPS game, just without the shooting part.  The mouse would replace the crosshair, no guns, etc. You would be able to walk around your desktop using either the arrow keys or WASD or whatever -- look around with the mouse.

So the desktop would the map, any desktop icons/shortcuts would be represented as buildings/signs/any 3d model you want.  Folders would be represented by buildings that you would be able to walk into and view the files inside.  Files could be represented from anything from a small picture to a billboard.  You could even customize you own 3D object.

There would also be 2D windows, of course, not everything can be in 3D.  For example: web pages, text documents, movies, etc,

There would probably have to be some sort of toolbar at the top of the screen to switch between the maps that represent different folders. There would need to be a map/folder that provided easy access to the entire filesystem.  This could be accessed by holding down the alt key, for example, and clicking on the toolbar with the mouse.

GAMING - 
If this or a similar idea was implemented in a *buntu distro, because of the entire OS being 3D, there would have to be a lot of support for 3D applications i.e. GAMES.  As well as the 3D part of this, there could also be a minimalist part of the OS that would run ONLY required processes, creating an ideal gaming environment.  

This idea is far from complete, and any suggestions would be greatly appreciated.

### Post by Progressive on 2008-12-24
Something like this will only happen when we have more elegant input devices. There are already some devices such as those special VR gloves

[http://www.vrealities.com/P5.html](http://www.vrealities.com/P5.html)

Also, those Emotiv headsets which track facial expressions and brainwaves is a step in the right direction. However, we would need something more robust and more open source.

We also need true 3D displays. Sure holographic displays would be nice, but at least good VR head mounted displays would be necessary. Some of this stuff works on linux, as a recent slashdot post pointed out.

[http://linux.slashdot.org/article.pl?sid=08/12/19/1424219&from=rss](http://linux.slashdot.org/article.pl?sid=08/12/19/1424219&from=rss)

I was impressed that some of these things use OLED. I bet the image quality would be very impressive.

Finally, better open source voice recognition technology would likely be necessary. Without multiple modes of input, 3D would be too clunky with just a keyboard and mouse. Compiz is about as 3D as a keyboard and mouse interface can get.

All of this would take quite a bit of re-training to move people away from the standard WIMP model. I'm not saying it shouldn't be done. In fact, I'll be first in line to beta test. 

Once it is proven to improve productivity, it will become mandatory in the workplace, and *that* will be the entry way into the desktop market. That likely won't happen until most of the above criteria are met.

### Post by james09173 on 2008-12-27
Back when I was playing with Windows desktops I ran across this thing called Tactile 3D. I think this program comes closest to a true 3D desktop environment.
If your desktop played like the old-school game DESCENT, that's pretty much what it's like. I'm pretty sure the WASD keys and mouse were how you navigated it. 

Here's a link of it in action . . .

[http://www.youtube.com/watch?v=u1otw1nVqzg&feature=related](http://www.youtube.com/watch?v=u1otw1nVqzg&feature=related)

As you can see, you start off inside this huge dome, your directories consist of giant spheres and then you can "fly" inside these spheres to see individual files or "fly" into yet another directory.

You can see how easily you can get disorientated with a desktop like that. I tried it for about half and hour and got a headache from it. Pretty neat idea, but not practical--at all. 

There was another post I saw recently on here that had a link to a video with a rabbit avatar on a 3D Desktop/webpage and it shot lasers out of its eyes or something to highlight something (at least that's what it looked like to me.)

The main problem I see with 3D desktops is that, while they look great, they're anti-productive. It is so much easier to just left-click on a file or folder, so why on earth would you want to walk to it? Unless 3D desktops implemented shortcuts to reduce or nullify the travel time to your destination I don't see them ever becoming popular.
