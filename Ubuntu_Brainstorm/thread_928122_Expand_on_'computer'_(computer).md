### Post by musther on 2008-09-23
The computer location is good for managing drives, but it lacks a couple of key features:

- The ability to format an external drive (a USB stick, floppy disk etc)
- The ability to run fsck, to check an external drive (USB stick again)

Nautilus has done wonders for making writable and re-writable CDs easy to use form the file manager, but this ease of use needs to be extended, specifically to USB drives.
