### Post by khurrum1990 on 2007-10-26
I was thinking of these things in the next release:
1.excellent stability though Linux is very stable by default.
2.excellent security and a built in firewall, no need for firestarter.
3.excellent hardware support meaning Linux should be able to run anything in the world.
4.ubuntu should work with companies of lets say vga card drivers such as nvidia and ati to provide suspend/hibernate support out of the box.
5.improve windows compatibility and I know it can be done cause just think of this, we started running world of warcraft on ubuntu and that means we can run anything we want.
6.windows programs can be installed with wine without a problem.
7.better support for games.
8.make it more power efficient.
9.fix all the remaining bugs as there are still some bugs with the nvidia driver from 7.04.
10.improve or add some new eye candy even though we r still better of than Windows or Mac OS X.
11.With gnome they should also supply their own dock bar and make it even better than Mac OS X's dock bar, and make kde look amazing as well which shouldn't be too difficult as kde 4 will be out soon.
12.make all devices like wireless, digital camera, and stuff work without any configuration.
13.better documentation cause personally I think OpenSuse has the best documentation of all Linux distributions.
14.Make Linux the best Unix like operating system ever made.
