### Post by smartboyathome on 2008-05-14
The reason they don't is because 1) it will add more code to the installer, meaning there is a higher likelyhood of bugs appearing in the installer, and 2) because it can't be done with the current installer, which just installs an image of the CD to your hard drive.

VLC isn't included because of licensing (I think). I am pretty sure it uses a license which isn't compatible with the GPL, and if it isn't compatible, then Ubuntu can't ship it.
