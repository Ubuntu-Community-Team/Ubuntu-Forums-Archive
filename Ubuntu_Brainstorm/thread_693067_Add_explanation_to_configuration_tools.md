### Post by steve196 on 2008-02-10
The various configuration tools (especially those for gnome) all have a lot less options than the configuration files they are writing to, so it would be great, if their documentation included the following:
* Is there a more detailed configuration tool for the same settings available?
* What configuration files  ( or sections thereof) is this program reading?
* What configuration files ( or sections thereof) is this program writing to?
* Where is the documentation of those configuration files?

That would also make troubleshooting a lot easier, if something goes wrong, that the tool just assumes is right.
