### Post by orvils on 2007-10-23
We have restricted drivers manager for drivers and we have "install restricted codecs" feature when some movie doesn't play. It would be good to have some way to let user install msttcorefonts package as well.

If you don't have this package then some websites look strange and there are problems with some MS Office documents. So if on first encounter with some mizrosoft font you could get a dialog and msttcorefonts would be installed it could help a lot.

**Suggestion:**Expand restricted drivers manager to "Restricted stuff manager" and let it be the tool to install restricted drivers, restricted codecs and restricted MS fonts.
