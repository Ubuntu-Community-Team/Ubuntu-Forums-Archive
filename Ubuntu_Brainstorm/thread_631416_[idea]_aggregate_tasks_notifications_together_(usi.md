### Post by 23meg on 2007-12-04
It can be a part of the default Ubuntu desktop once GNOME decides to include it, and GNOME apps start to utilize its API. There's not much to be done on the Ubuntu side until that happens; once it happens, Ubuntu desktop apps that aren't part of GNOME, such as Firefox, would need to be modified to utilize it as well.
