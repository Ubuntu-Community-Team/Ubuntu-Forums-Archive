### Post by clubsoda on 2007-11-20
A fair amount of time seems to be wasted opening up 50 or so language packs at the beginning of an installation and then discarding 49 or so at the end.  As far as I can tell the only purpose served by this is to provide a pretty list of languages in different fonts for the language chooser.

One might think that if someone couldn't recognise the name of their language in English, then they're going to find linux quite a challenge...

However, there could be a win-win alternative approach.  How about displaying the list using images of the different fonts instead of text?  That way, it would only be necessary to install one or two languages, which should make for a quicker installation.
