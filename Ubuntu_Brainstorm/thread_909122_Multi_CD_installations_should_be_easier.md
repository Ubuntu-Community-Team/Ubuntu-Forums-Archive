### Post by smartboyathome on 2008-09-03
Multi-CD is a third party project, not a concern of Ubuntu's. As for inserting and removing CDs during installations, this isn't possible unless you boot to RAM or from a hard drive, both of which have limtations. Booting from RAM with Ubuntu requires you to have enough RAM, which is usually 1GB for smaller CDs (500-600mbs), while booting it from the hard drive renders that drive useless from things like partitioning, since the CD is using it.
