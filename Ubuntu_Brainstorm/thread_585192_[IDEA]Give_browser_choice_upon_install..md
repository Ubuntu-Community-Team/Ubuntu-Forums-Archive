### Post by Frak on 2007-10-21
I've installed (X,K)Ubuntu multiple times on many machines, and what was always my pet peave, was the lack of browser choice upon install.

IDEA
Upon installing (X,K)Ubuntu using the Ubiquity installer, during installation processes, somwhere around user profile importer, give the user a choice between using the default browser (Firefox on (X)Ubuntu, Konqueror on Kubuntu), or a different choice giving a short description of each, such as Opera, Konqueror, Epiphany, and Firefox.

Opera - Proprietary
Cannot be modified to fix bugs, add features, etc. and is not supported by Ubuntu., yet is widely hailed as one of the fastest, most secure browsers on OS X, Windows, and Linux.

Konqueor - Open Source
If there are any bugs with this program, the source is available for repairing the program or adding more features. Konqueror is the default browser of Kubuntu, and is stable, lightweight, and uses the KHTML web engine that is used by companies as their default Web Engine for its fast browsing capabilities.

Firefox - Open Source
If there are any bugs with this program, the source is available for repairing the program or adding more features. Firefox is the most widely used Open Source browsers on Earth. It's built using XUL, which makes it perfectly integrated into your system, no matter what Desktop Environment you run (KDE, Gnome, XFCE, Aqua, Explorer, etc.)

Epiphany - Open Source
If there are any bugs with this program, the source is available for repairing the program or adding more features. Based on the same Web Engine as Firefox, the Gecko Web engine, this makes Epiphany stable and reliable. Epiphany is built using GTK instead of XUL, which makes Epiphany much lighter and faster than its Firefox counterpart.
