### Post by gobuntu on 2007-10-29
At boot time, Ubuntu presents:

One screen for User Name and
as soon as username is entered, screen is replaced by

One screen for PW.

It seems to me that just ONE screen should be more user-friendly due to the following:

1. If one mistypes the user-name, one has no chance to correct it and yet one is forced to put-in a password, in vain.

2. The two screens are so SIMILAR that often I may type my PW instead of the User-Name, specially when I mis-type one of the two, which is bad if someone were watching.

I don't see any real advantage to having TWO screens, specially since the user-name is not meant to be 'secret', for if it were, it would show dots or stars.

And, by the way, it would be nice if a message: "Caps Lock is ON"  would show on the screen when such is the case, a very-useful feature, which might not be too difficult to implement.

Thanks for considering this, if users like the request, let's hope it is implemented, for the sake of improvements that may be small but quite helpful.
