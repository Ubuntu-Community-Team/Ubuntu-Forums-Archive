### Post by vishnumrao on 2008-01-26
Hello,

Here is a few features I would like to see in Ubuntu. 

1. Synaptic: Make is more simpler for newer Ubuntuers. It would be nice to have to settings for using synaptic. A beginner mode and an advanced mode. Have the beginner mode as default where you one does not see the dependencies e.g. libcairo or agrep etc. In this mode the user should only see only the end program or utility that he needs to install and use. All dependencies get checked for install, by default. This makes it less confusing for the newer folks. I still remember feeling confused, the first time I tried synaptic, the first time I tried Ubuntu, when I had to scroll through a list of packages just to find Thunderbird.

For seasoned folks they can have an option to show all the packages, i.e advanced mode. One should be able to switch to advanced mode through the Settings --> Preferences option.

2. Update manager: Have an option to set a custom update intervel. Currently there is only 4 options with the maximum of 2 weeks. I have noticed that my colleagues do not like it, when in windows XP, a baloon pops up displaying an annoying "Updates are ready for your computer" message every so often. So a custom interval should be included. Gives the user the freedom to update at an interval of his/her choice.

3. Have Thunderbird as the default email client. I know a lot of folks are going to disagree with this. But I believe more newer users know about Mozilla Thunderbird  as "an email client from the makers of Firefox".

I do not know how many people would think my ideas are worth implementing. Please do post your inputs.
