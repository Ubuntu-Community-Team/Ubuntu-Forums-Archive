Would it be possible to add a field to the Launcher Properties dialog box, so the user could if s/he wished, define a "hot key" combination (e.g., <CTRL> + D, or <ALT> + K), which would trigger the launcher from the keyboard? Just a suggestion, but I think it'd be a nice feature.  It should (of course) default to 'no hot key defined'. :)
