### Post by Gaunt on 2007-10-27
I would like to see a simple function for backing up important files like Xorg.conf. A simple GUI so that you with just the press of a button can confirm "My computer works fine now" and then have all critical files backed up locally or to a USB-drive/CD. If later on you fiddle too much with the wrong settings you should just be able to run a simle restoreprogram to get everything back in place and most likely up and running. If this should be really advanced it should also detect broken/missing files and replace these.

I would also like to see a "New to Linux" mode where people who has just transferred from M/S can get additional tooltips and help. Stuff like "Writing sudo before a command in the console, makes that command run in administrator mode". I think stuff like that would make many new users feel much more secure with their new OS. Also if this is marketed heavily I think we should see a lot of new U/KU/XUbuntu users.
