### Post by MRiGnS on 2007-11-17
The main advantage of GNU IceWeasel is that it is entirely free software.[3]

Differences from the official Firefox release:

    * Contains only free software
          o Replacement of the proprietary artwork with free artwork
          o Removal of the proprietary Talkback crash reporting system (due to its non-free "binary only distribution" licensing)
          o Use of a plugin finder service that offers only free plugins
    * Added privacy features — protection from:
          o 3rd party zero-length image files resulting in 3rd party cookie (web bugs).[3] Note that in Firefox 1.0 and 1.5 3rd party cookies can be disabled completely via the UI. However the UI option was removed in 2.0.
          o URL redirection (warning only)[3]

The MPL prohibits Debian from shipping firefox with patches to fix things that are broken. Debian is required to upstream these patches, get them approved by Mozilla and wait for downstream in order to maintain the firefox branding. ### Post by CarpKing on 2007-11-20
> **tehet said:**
> You are missing the point by miles and so is thegnuguru.

The MPL prohibits Debian from shipping firefox with patches to fix things that are broken. Debian is required to upstream these patches, get them approved by Mozilla and wait for downstream in order to maintain the firefox branding. Some of these patches will not be approved because Mozilla prefers cross platform consistency over desktop integration.

For some reason Ubuntu and a few other mayor distros have an understanding with Mozilla that allows them to ship virtually the same patch set that Debian does, without having to change the firefox branding.

Debian can't make a similar agreement because their software guidelines prohibit them from using parts that are subject to trademark.  This includes the Firefox icon (which is why Ubuntu used to ship Firefox with a different icon).  Mozilla eventually told them that if they wanted to use the branding, they had to use the whole branding, and IceWeasel was born.
