### Post by oasmar1 on 2008-03-21
I think that a good idea for ubuntu to make it really user friendly and kind to new users is to make a simple post installation sequence which would go through some of the simple things for example:
It could start off by covering the entire screen, the first option would ask which background the user would like (possibly connect to an on-line background picture database to provide large choice.)
Then it could ask whether the user would like small, medium or large fonts to be used in the system (i always change all the theme fonts to size 8 which looks nicer to me on my LCD, while my dad needs my help changing everything to size 14).

There are many options which could be put into this introduction set up.
Also, there should be a "skip" button which would just set all the default settings.
