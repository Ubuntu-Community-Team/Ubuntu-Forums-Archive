### Post by erginemr on 2008-03-10
I cannot tell anything about the random pop-up notifier, but I can suggest you to try kvoctrain:
[http://kvoctrain.sourceforge.net/](http://kvoctrain.sourceforge.net/)

which is a foreign language vocabulary trainer. It is in the repos. So, you can install it with:
```
sudo aptitude install kvoctrain
```
