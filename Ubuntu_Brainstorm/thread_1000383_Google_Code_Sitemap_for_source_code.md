The source code for all packages in the Ubuntu repository is available at [http://archive.ubuntu.com/ubuntu/pool/](http://archive.ubuntu.com/ubuntu/pool/)

Create a [Google Code Sitemap]("http://www.google.com/support/webmasters/bin/topic.py?topic=12640") so that [Google Code Search]("http://www.google.com/codesearch") can index all the source code and make it searchable.

This would allow you to conveniently search through the source code using Google Code Search.

Imagine a search query like, "site:archive.ubuntu.com package:gimp radial blur"

Pretty neat, eh?
Would be good for our developers.
