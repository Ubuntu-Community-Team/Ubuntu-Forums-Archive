### Post by AlexThomson_NZ on 2007-10-30
A few things I would like to see implemented in future versions (these are things I always have to do manually after a clean install, and to me are must-fixes)
   * Enable color prompts
   * Install vim
   * Alias vim to 'vi'
   * Enable things like tabspaces to 4, syntax highting, etc.
   * Fix the ugly widgets in Firefox (really, why has this been allowed to continue so long!)  At least FF3 uses native GTK widgets (yay!)
   * Install VLC and Thunderbird

Other minor things that would be nice to see fixed
   * Tidy up the default .bashrc (it seems poorly laid out and overly confusing)
   * Tidy up and simplify the default grub menu.lst
   * Stop using the old widgets in Synaptic (thankfully there aren't too many native Gnome apps using them any more, and the UI is almost fully consistant now)

On the whole though, my gripes list is getting shorter and shorter with each release, so I am very happy with the general direction things are going :)

That's all I can think of off the top of my head, thanks for the chance to rant! Apologies if any of these are brought up constantly- I didn't bother going through too many old posts, etc.
