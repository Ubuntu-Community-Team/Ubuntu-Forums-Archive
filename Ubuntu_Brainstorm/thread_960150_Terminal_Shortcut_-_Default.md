### Post by Jim! on 2008-10-27
I haven't done a clean Install of Ubuntu in quite sometime now, but with each release there has always been one thing that I've done each time, and that is setting the terminal to a keyboard shortcut "Alt + Enter". 

These keys (Alt + Enter) are not assigned to anything in the default Ubuntu install and since the terminal is a widely used application I think It would be great to have this keyboard shortcut set as default, It would be simple and easy to implement (I imagine) and would make Ubuntu just that 'little bit more' complete.

What do you think?
