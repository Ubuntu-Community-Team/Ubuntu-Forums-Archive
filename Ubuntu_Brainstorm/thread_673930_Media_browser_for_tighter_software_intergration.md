### Post by fordplay on 2008-01-21
I would like to suggest a light weight media browser similar to that in ilife and karelia's imedia. 

The software would enable users to easily open a small window containing tabs for pictures, audio, and movies. For each tabs all files from that category are displayed as thumb nails. This allows a user to very quickly find an image / movie / audio file and drag it to other existing app for use there.

The software could be built using a filter applied to a tracker with results displayed in Nautilus. I imagine this app being accessed from the top window next to the internet browser, email client default buttons.
