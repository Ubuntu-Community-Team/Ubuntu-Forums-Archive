### Post by andrewc6l on 2008-04-02
For Firefox, the behaviour of an app is defined by the application. You can submit that idea to the Firefox programmers if you want to have it changed.
[https://bugzilla.mozilla.org/](https://bugzilla.mozilla.org/)

If you want to see which programs start with a in a terminal, try entering a then hitting the TAB key a few times - the autocomplete will bring up a list.
