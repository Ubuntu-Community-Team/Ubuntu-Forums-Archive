### Post by steve196 on 2008-02-20
I know, there is a wiki list with supported hardware, but that is a "it worked for me" kind of thing and often only valid for a particular version of the hardware, which is useless, because there is no way, to determine the version before purchase. 
There is also a list of officially supported hardware but it has only preassembled systems and is therefore almost useless.
There should be a list of hardware, that is officially supported in Ubuntu. The list should come from an official source and only contain hardware, that has mature and complete drivers and is unlikely to give surprises. If some hardware comes under the exactly same name with different chipsets, it should only go into the list if all of them qualify.
