### Post by suchawato on 2007-10-27
So far, Ubuntu's stated purpose as in Bug number one, has been to go after Microsoft Windows.
But what about OS X?

OS X is a really good OS.
IT is able to run effects like dashboard and Widgets even on really old Macs.
I've seen dashboard and widgets run on an old clamshell iBook.
The desktop effects are really practical too.
Compiz-Fuzion has equivalant effects, but they require a 3D card to run.
If Mac can get these usefull desktop effects to run on an eight-year-old computer, Why can't Ubuntu have some usefull, effects that are well writen enough to run on an older laptop? Even XP service pack 2 has a middle click task switcher effect.

Which brings it back to the original question, Why not a Mac competitive version?
One made with putting the user's art, music, and pictures at the forefront, and office things in the background?
UbuntuStudio is sort of a nice attempt, but is just Ubuntu with some additional packages. I mean like a whole different Desktop environment.
Somthing other than KDE, Gnome, or XFCE. one with smooth running effects as the primary concern, not an afterthought.
