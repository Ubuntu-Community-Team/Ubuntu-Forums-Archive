### Post by piege on 2007-12-12
I am currently looking for a bulletproof todo program. 

So far I have tried everything i could get my hands on ( thunderbird+lightning, evolution, tomboy, simple todo.txt file with conky, tomboy, remind). But they always seemed to lack of something ... usability. I almost always found myself organising the piece of software or lacking of something. 

I was thinking something along the lines of the GTD system. The main thing about this 'virtual assistant' is that you can add reminders and notes quickly and new events. I'm thinking of 1 or no click from any other window ( something like deskbar would be perfect.Also,  I think hot-keys are essential). 

I believe there should be about 3 aspect to the program. 

- Listener/ Alarm : Discrete yet always there when needed. ( alarms should disrupt the user depending on its importance ) 

- Calendar/ Stuff happening day / soon :  I would think this thing should be able to be viewed easily. I really like the idea of having this king of thing in the background ( just like something conky would do ).  Personnally, I don't think that major change should be done from this interface ( although basic stuff such have tic off something that is done would be cool ). This aspect should give a quick overview of what's to come in the near futur and what has to be done right now. .

often, several tasks and events are linked to a project/subproject/etc these should be made linkable easily. Or they should simply be linked to notes taken earlier. So I figure having something like a pseudo-Tomboy would be a good idea. 

Something i think would be useful is maybe having addons to popular email manager that would detect events in new mail ( a bit like what gmail does ). And that 'forces' the user to process any new email ( the user would have to choose what to do with the new email that just came either it's to file it, link it to an event or an action, etc )

Most people need to have access to their stuff from more than one computer.So a assume that it would have to be compatible with standard calendars and online stuff without having to constantly having to export to etc etc ... ( this is really annoying in thunderbird/lightning).

Maybe this kind of program isn't for the usual user but I would think it would be a great tool to developp for people who need a something to organise their busy life. 

If you can tell me of any program that actually does most of this please tell me !!
 This idea isn't perfect but I think it's a good start for a great easy virtual assistant. Personnally, i beleive it would make my life much more manageable and give me easier access to the stuff I need. 

I hope this might inspire somebody out there.

P-L
