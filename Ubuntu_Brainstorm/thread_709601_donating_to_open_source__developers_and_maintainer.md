### Post by nemroot on 2008-02-27
Recently I was read at [http://www.zdnetasia.com/news/software/0,39044164,62037500,00.htm](http://www.zdnetasia.com/news/software/0,39044164,62037500,00.htm) that in Ubuntu it wil be possible to buy commercial apps right from the desktop.
Then I would like to suggest to add a feature to donate open source developers of this or that software right from the desktop
If it easy, with one or two clicks, then a lot of people will start to donate, thus supporting open source.
That means developers will be able to spend more time and resources for their own projects, and be able to compete with proprietary software producers who will get money from regular users more easily
We need the same for OOS.
