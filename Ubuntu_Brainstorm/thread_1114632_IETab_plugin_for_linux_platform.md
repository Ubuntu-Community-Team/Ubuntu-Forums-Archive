### Post by smartboyathome on 2009-04-03
No way to do it. Internet Explorer runs under wine in Linux, which can't be embedded into Firefox native. Only way to do this is to run both Firefox and Internet Explorer under Wine, which requires no special plugin (the IETab plugin works). You may want to check out User Agent Switcher, though.
