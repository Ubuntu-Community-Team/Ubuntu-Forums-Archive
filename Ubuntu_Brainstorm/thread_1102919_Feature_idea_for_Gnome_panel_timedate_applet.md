I've got a feature idea, but do not know where should I post it so I'll start here.

Application: Gnome panel "date/time" applet.
Feature idea: It would be cool if user could set alarm quickly by just clicking the date/time bar and setting alarm in some easy way. It would be quite useful in many cases. You're boiling rice and want to be alarmed when 25 minutes has passed, you're making tea and do not want it to steep for too long, your favorite TV show starts in 17 minutes etc.
