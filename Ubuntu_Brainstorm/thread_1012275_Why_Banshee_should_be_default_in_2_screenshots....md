### Post by directhex on 2008-12-15
Firstly, please note that the OP's Banshee screenshot is not the default UI.

This is:

[img]http://download.banshee-project.org/shots/1.4.1/android-purchased-music.png[/img]

Banshee's UI is highly customizable - for some this is a plus.

My understanding is that the desktop team will not consider replacing one app with another app unless it has at least the same functionality. There are two issues with Banshee - one minor, one major.

Minor issue: Banshee does not support serving music over a network. Rhythmbox supports serving music via DAAP (iTunes sharing server)

Major issue: Banshee uses a large number of custom widgets for its interface which cannot be enhanced by accessibility features such as high-contrast themes, screen readers, etc.

I'd encourage the desktop team to frequently reassess the default application selection - e.g. Rhythmbox versus Banshee versus Quod Libet etc etc etc. But for now, I think Rhythmbox is the right "good enough" choice as a default - even though I use Banshee myself
