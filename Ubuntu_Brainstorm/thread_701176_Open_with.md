### Post by erginemr on 2008-02-19
The first one is easy: 

I have created a text file, right-clicked on it and selected properties, passed on to the "open with" tab, used the add button to add "gvim" to the list of  applications to open a text file. I can also select the  default program by selecting the radio button beside its  name. 

Now that I have selected at least two programs, whenever  I right-click a text file, I will be presented with a list of applications with which I can launch the file.

### Post by erginemr on 2008-02-19
This function is already available in Gutsy. You don't need to download and install anything. You can follow my example by just right-clicking on any file type (say an html file) and add extra programs (say web browsers such as Firefox, Opera, Epiphany, etc.) to open it with.

Or you can right-click on a music file (ogg, mp3, etc.), select properties -> open with -> add -> totem, vlc, mplayer, etc.
