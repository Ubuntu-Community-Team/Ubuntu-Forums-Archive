### Post by coolen on 2007-10-31
Windows power users will be familiar with Daemon Tools.
Linux power users will be aware that this functionality is available out of the box.
What's lacking is integration with the GUI.

In Ubuntu (and most modern Linux distros), removable media can be mounted and unmounted by the user. I understand this is due to a wrapper around the mount command.
Could this not be applied to disc images, too? I know this has been done with Nautilus extensions, but I think this is a useful enough feature to be included out of the box.

The main issue for usability I see is naming the mount point. Ubuntu can mount removable devices, such as media players, by the volume name. Would it be possible to determine the CD title, and mount it to that?
