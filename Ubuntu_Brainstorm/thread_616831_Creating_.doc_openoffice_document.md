### Post by qamelian on 2007-11-20
You can set this up yourself very easily. In your home folder, create a new folder named Templates. Then open OO Writer and save a blank document to the new Templates folder with the name ```
New Writer Document.odt
```

Now right-click on a blank piece of desktop and you'll see a Templates entry on the right-click menu, with a sub-entry  for you newly created blank Writer template.
