### Post by Ubuntese on 2007-12-11
As emerge from lots of discussions in ubuntu national and international forums, more and more ubuntu users need preinstalled GUI for the configuration and the administration of the operating system. These tools wouldn't change Ubuntu Unix and Linux philosophy, but simply be a possible user's choice. We think that it's right as well configuring the system through shell, editing by hand .conf files  or helped by a specific GUI.
We appreciate the great work made by  developpers who give back Ubuntu as the best desktop linux ditribution in the world and we realize that you need time to make other miracles; however we would propose some ideas that could help the spreading and developping of our preferred operating system.
Actually the market is dominated by close source operating system which aren't superior to GNU/linux, but are imposed by few software manufacturer. The simplification of GNU/Linux migration could be an important step for GNU/Linux diffusion.A system which is at the same time user-friendly, stable and open source could be efficiently used by everybody with no difference of cultural level and social class and at the same time without limiting expert users who could continue using the shell.
While we are waiting that time become a friend of developpers, we would point out some feature that we believe as very important for a modern operating system:

1) Include by default and improve a GUI for FSTAB
2) Develop a unic applet to configure adsl connection, LAN and Wireless network with full support to WEP and WPA. This could be included in an improve of network manager
3) Improve of displayconfig-gtk applet with more options and the possibily to check the xorg configuration. Also could be included an help which explain the meaning of each driver.
4) Include by default the conf-compiz-settings-manager
5) Include by default gnome-color-chooser which grant a better personalization of gnome desktop
6) Gnome control center: group similar applet to reduce the menu entry obtaining clean preference and administration menu

Ubuntese , Kidrock , lanternacieca, deviltigre , Guiodic and other users . [http://forum.ubuntu-it.org/index.php/topic,139840.0.html](http://forum.ubuntu-it.org/index.php/topic,139840.0.html)
