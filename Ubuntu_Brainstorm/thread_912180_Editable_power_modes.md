### Post by rossjman1 on 2008-09-06
Like Windows XP and Vista, I would like to have different power modes for laptops that I can easily switch to. Right now there are only options for what to do when Ubuntu is plugged in or running of battery. I would like to create a few power modes like Maximum performance, Maximum battery life, Presentation mode, etc.
Some options that can be changed per mode are:
* Screen brightness
* Power going through components (less power = less performence = better bat life)
* When to dim screen
* When to suspend/hibernate/shutdown
[IMG]http://web.mit.edu/ist/initiatives/it-energy/images/02xp.gif[/IMG]
They are called power schemes in Windows XP. It would be great if there was something like this in Ubuntu already. I'm tired of getting poor battery life in Ubuntu (Ubuntu always bumps up the screen brightness automatically for some reason).
