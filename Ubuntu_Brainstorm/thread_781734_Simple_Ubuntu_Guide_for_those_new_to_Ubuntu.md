### Post by eyecreate on 2008-05-04
After reading this wordpress article([http://contentconsumer.wordpress.com/2008/04/27/is-ubuntu-useable-enough-for-my-girlfriend/](http://contentconsumer.wordpress.com/2008/04/27/is-ubuntu-useable-enough-for-my-girlfriend/)), I thought that there should be a better way for those who want to find out how to do something who have no idea where to start. The result of this is the following program. This program should start up with a fresh Ubuntu installation and will be waiting for the user to say if they wants it's help(so as to not be in their face).
[[IMG]http://img240.imageshack.us/img240/7296/screenshotkx0.th.png[/IMG]](http://img240.imageshack.us/my.php?image=screenshotkx0.png)

[[IMG]http://img396.imageshack.us/img396/5901/screenshot1wr3.th.png[/IMG]](http://img396.imageshack.us/my.php?image=screenshot1wr3.png)

Please test this and see what you think of this idea. The data is all contained in an XML file so that more guides can be created easily.

[http://eyecreate.dre.am/ubuntu-guide.tar.gz]("http://eyecreate.dre.am/ubuntu-guide.tar.gz")

EDIT:i forgot to say you will need to have the python library Amara installed. This is not in the repos right now.
