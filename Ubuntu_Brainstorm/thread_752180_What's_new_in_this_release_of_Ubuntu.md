### Post by wpshooter on 2008-04-11
Our tax software (CCH - Prosystem Tax) here at work, has a feature that upon every update to the software by CCH, that when the user opens the program for the first time after the software has been updated, that a popup comes up giving the user the option to review (either now or at a later time) all of the new features/items that have been added to the software in the latest round of programming changes/enhancements to the software.

It would be really nice to have a similar feature built into every new release/version of the Ubuntu O/S releases.  I know that this information is probably available either somewhere hidden within the O/S or perphaps on some Ubuntu website but it would be much better, IMO, if the new enhancements were brought to be users attention when they boot into the O/S.

It is great that the developers are going to all the efforts of making this a better and better O/S with every new release, but I would be willing to bet you good money, that the average user of Ubuntu has no idea as to ALL of the improvements/enhancements have been made to each new version of the O/S when the final release is available for installation.

It is sort of a shame for the developers to spend all of this effort in making all kind of improvements and then have the users of this great O/S not be aware as to what tools they have available to them.  I know this is true because some of the new features in Ubuntu have been discover quite by chance by me !!!

If someone might be interested in talking to CCH Prosystem about how they go about incorporating this feature into their software, they can call their support number at 1-800-739-9998.
