### Post by Omnios on 2008-01-15
K I have been playing around with this idea for a few days now as I just got a new 15" laptop that has a max res of 1280x800 which is just ok for the size of the screen. 

 Anyways being limited to lower res options I got thinking that it might be interesting to have a few tweek function combined into packages for users with new and older laptops where the res is limited.

 So far there are fond and icon options but I am thinking more on the lines where a lot more things can be tweeked. Also from firefox there are slim or small packages for laptop users where the bars and other screen components are a smaller size than normal. 

 This gets me thinking about Gnome and possibly KDE in that it may be possible to develop and implement laptop setting pertaining to the max res and screen size for limited res laptops.

 This can be accomplished by adding these options to have all the desktop features and file managers features tweekable pertaining to these res restrictions. The main parts of these would allow for smaller borders and user areas.

 Also it may be possible to have a icon generator that  resizes the icons etc to a custom size.

 Now the end result should be that one package can reconfigure and generate a desktop package perfect for a user of a limited res laptop. 

 Also this would be a new generic feature for Ubuntu and Linux to help linux stand out from others. 

 Also it could also be a feature that can attract limited res laptop users to Linux and Ubuntu.

 Also as listed in the title I will to invite limited res and regular laptop users to help hammer out a formula to develop a new desktop package for laptop users.
