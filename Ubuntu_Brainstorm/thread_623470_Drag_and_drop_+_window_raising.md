### Post by googleaseerch on 2007-11-25
Picture the following: you have a video folder open in Nautilus, and then you start up VLC, which places itself in front of the nautilus window.  When you try to drag and drop the file from the nautilus window into VLC, nautilus gets sent to the front, so you can't complete the operation. 

Is it plausible to add a feature to raise a clicked-on window only after mouse button up, and/or to cancel the impending window raise if the click becomes a drag?  In my opinion, this would be quite useful at least as an option.
