How about bundling a list of the best websites and the most useful ubuntu information in a Special Ubuntu Guide that is supplied out in the open for new users by default? 


**Documentation is Linux's biggest weakness**


Why not take a step to make this a thing of the past?

For every cool thing there is to do with linux, like customizations, themes, etc. There should be a little write up and links to sites to point them in the right direction.

**Also include up to date help documentation**

Why not vote on the best websites to include in this bundle so the new users don't have to go searching the web?

(Makes it easier since a lot of linux websites had odd names that don't pertain to their materials)

**Plus this easily spread awareness to important linux topics and information!!!!**

**Also this is an important one**

- Why not have pidgin or another irc client  have the ubuntu irc channel preloaded?  
- Or even make a little icon that says need some free online help from our Ubuntu irc channel?
- Then when they click it, pidgin  will auto open to the ubuntu channel on irc?

This would be for new users who have never used irc before, making it very easy for them to find some help ect.  

This would be a great step towards bridging the documentation gap.

### Post by LaserJock on 2007-10-25
I would like to point out that there **is** an Ubuntu Documentation Team. It takes people who are willing to learn, write and proofread to make Ubuntu's documentation better. It is an all-volunteer effort. Canonical does not employ people to work on the Documentation Team. So get involved!

**IRC Channel:** #ubuntu-doc on irc.ubuntu.com
**Mailing list: **ubuntu-doc on [http://lists.ubuntu.com](http://lists.ubuntu.com)
**Wiki page with lots of info:** [https://wiki.ubuntu.com/DocumentationTeam](https://wiki.ubuntu.com/DocumentationTeam)
**Mentoring Program:** [https://wiki.ubuntu.com/DocumentationTeam/Mentoring](https://wiki.ubuntu.com/DocumentationTeam/Mentoring)
