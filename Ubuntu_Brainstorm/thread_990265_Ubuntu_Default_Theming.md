### Post by smartboyathome on 2008-11-23
> **Merk42 said:**
> Would it be possible to have the theme tab of appearance list recommended themes (just links to them, not actually installed) much like Firefox 3's recommended addons?

Possible, yes, but since there are several sites for GNOME themes, it isn't very practical. Opendesktop.org, which includes gnome-look.org and ubuntu-art.org, is down a lot, though is one of the bigger repositories of GNOME themes. There is also gnome-art.org, which contains a lot more dated themes, but is up a lot more. There is even Freshmeat, which is in a similar situation to gnome-art.org.
