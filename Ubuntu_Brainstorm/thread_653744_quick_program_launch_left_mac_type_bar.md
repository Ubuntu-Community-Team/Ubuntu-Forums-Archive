 I think that a side quick launch bar on the left or right side of the screen may prove a new and useful feature for Ubuntu. Now the basis of this is that normally a left or right bar is kind of useless because it is oriented vertically and to see what is in it can be difficult. 

 Though I mentioned a mac bar this is way different. Now as for the side bar  there are a few possibilities.

 First there is the normal part where the bar is hidden till the mouse is put to the side of the screen. 
 
 This is the big part. When the mouse is put over a icon in the side bar it will be enlarged similar to a desktop icon with a file name. This is loosely based on a small icon that pops up to a desktop icon like launch feature. 
 
 Another feature may also allow for such things as say a menu icon to for access for the main menu feature etc. Also if done correctly places may also be done nicely.

 Now I am not sure if this can be done by Ubuntu or rather with the gnome developers.
