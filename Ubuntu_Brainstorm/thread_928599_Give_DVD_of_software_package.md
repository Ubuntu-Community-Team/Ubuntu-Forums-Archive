### Post by signifer123 on 2008-09-24
I'm pretty certain that the packages in the 4 main repositories wouldn't even fit on a bluray(almost ten times the capacity of a dvd).

 But with Synaptic after you select the apps you want, go to "File>Generate package download script", which you then run on another computer with faster internet. From then on you can use "File>Add downloaded packages" to install those downloaded packages.
Another alternative is to download them on one pc with a fast connection, then use reprepro to setup a repository. [http://www.jejik.com/articles/2006/09/setting_up_and_managing_an_apt_repository_with_reprepro/](http://www.jejik.com/articles/2006/09/setting_up_and_managing_an_apt_repository_with_reprepro/). Which then can just be inserted to act as a repository.
