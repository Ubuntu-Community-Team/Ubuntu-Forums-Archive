### Post by kingleer on 2009-01-10
Something I would like to see in 9.04 would be a program built into ubuntu that performs the same function as aptoncd (if not aptoncd itself). 

[http://aptoncd.sourceforge.net/](http://aptoncd.sourceforge.net/)

From the Link: 

What is aptoncd: 

APTonCD is a tool with a graphical interface which allows you to create one or more CDs or DVDs (you choose the type of media) with all of the packages you've downloaded via APT-GET or APTITUDE, creating a removable repository that you can use on other computers.
APTonCD will also allow you to automatically create media with all of your .deb packages located in one especific repository, so that you can install them into your computers without the need for an internet conection.


I feel this is a great backup feature. It would especially be useful if you were ever without an internet connection and needed to install something/reinstall something. 

One feature I'd also like to have that aptoncd isn't capable off is the ability to make one big .deb package instead of just on .iso with all the packages inside.

Well, what do you think?
