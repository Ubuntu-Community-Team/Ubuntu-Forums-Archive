### Post by Afoot on 2008-03-30
It is pretty annoying, and for newbies confusing, when clicking several times on for instance the tracker icon in the top left of the screen opens several tracker windows. You should only be able to open one window, in my opinion. There are other similar cases where it isn't necessary to have several windows open, and it should be limited.

### Post by nhandler on 2008-03-30
I disagree. For certain programs, having multiple instances of it running doesn't make any sense. For others, it can be very handy. Take the nautilus file manager. Since it sadly does not support tabs, I often have multiple windows open at the same time. If I were limited to only having one open at a time, I would have to waste a lot more time.

I think this should be implemented by the various developers instead of becoming a feature in ubuntu. That way, it would only exist in programs that it could benefit.
