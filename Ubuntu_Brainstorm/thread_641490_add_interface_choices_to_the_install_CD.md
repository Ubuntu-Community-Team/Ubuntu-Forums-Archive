### Post by coolen on 2008-01-01
I've seen this suggested a lot recently, and each time, the answer is the same: no.

I fully support someone going and making a DVD containing all three distros. I'd even download it for my own personal use, since I'd love to have the choice there. It'd be a great tool, and I'm sure many would enjoy it.

This is not, however, what Ubuntu is about. It could perhaps be Canonical supported, but an extra (like the alternative installer). The main product, however, should keep the current model. Ubuntu didn't rise to be (arguably) the most popular distro by doing the same thing everyone else has been doing for the past decade.

Ubuntu is a distro that should be usable by someone who's never used Linux. They need to be able to type, move their mouse, know their name, and that's it. They should not be confused with talk of desktop environments. Sure, we all know what they are, the differences, and we all have our preferences. The new user, however, does not. Don't try to make them learn too fast.
