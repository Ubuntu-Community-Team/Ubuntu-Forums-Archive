### Post by LaserJock on 2008-01-21
> **23meg said:**
> The nearest server is chosen by default, according to the user's location that they indicate during installation.

I don't think it's the nearest server. But the server pool of that country. For instance us.archive.ubuntu.com is not one particular server but a pool of servers. If you check for the fastest specific server the problem is that quite often the fastest server is no the same from day to day. A pool allows more uniform distribution of load and handles mirrors dropping in-and-out.
