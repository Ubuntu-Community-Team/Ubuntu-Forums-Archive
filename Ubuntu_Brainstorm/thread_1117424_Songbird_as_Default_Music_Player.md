### Post by zjagannatha on 2009-04-05
I think that Ubuntu should use [songbird]("http://www.getsongbird.com/") as it's default music player. Songbird has several great features that most linux music players lack.

**First: Songbird can easily handle iPods.** I have had far too much trouble with banshee and amarok with regard to iPods. I was never able to use either to sync my iPod on either of them.

**Second: Songbird is easy to use.** I've been using songbird for about a month now and I will never go back to amarok or banshee. It automatically scans your music library (which you can set to be any directory you choose).

**Third: It's got developer tools built in to it.** I started using songbird because it was more stable on my computer than Amarok or Banshee, and because Rhythmbox is horrible. After I used it for a few days I decided to browse the addons on the built-in web browser and discovered development kits that add on to your existing Songbird setup and help in creating plugins or 'feathers.'

What does everyone else think?
