### Post by BrendanM on 2007-10-29
I think it would be a good idea to have some sort of "Welcome" tour that pops up when booting Ubuntu for the first time (maybe only on the live CD, or possibly also on the first boot of an HDD install). I'm thinking of something along the lines of the Windows XP welcome tour. It could walk new users through some really basic things, like opening the file manager, showing them how to launch programs, maybe mounting/unmounting drives. The idea would be to quickly show them how to do some things they were used to knowing how to do on their Windows system, but might not know how to do on Ubuntu.

Also, to prevent it from being really irritating, the first thing it does should be ask, "Would you like to take a tour of Ubuntu?" and if you click "No" it should go away and never pop up again.

Thoughts?
