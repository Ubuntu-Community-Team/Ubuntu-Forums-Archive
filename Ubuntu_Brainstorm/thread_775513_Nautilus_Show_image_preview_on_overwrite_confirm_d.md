### Post by The Grum on 2008-04-30
KDE does this and its quite a useful feature. If you are copying image* files from one folder to another, the overwrite confirmation dialog shows both images side by side so you can clearly see if the file you are overwriting is the same image or if its another image with the same name.

This would be useful for people like me who don't rename pictures when copying from a digital camera (so I get a lot of pic0001.jpg, for example) and if i get a confirm dialog, I have to select "skip", then manually check each image to make sure Im not overwriting a different picture.

* I think its just images. It might do other files as well, I cant remember.
