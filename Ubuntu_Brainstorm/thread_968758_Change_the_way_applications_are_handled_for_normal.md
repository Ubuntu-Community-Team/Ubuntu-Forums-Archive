### Post by yaknowwat on 2008-11-02
My idea is to integrate all the neat technologies of linux into how it is handled.

Such as linux has the ability to stop and continue processes pretty much creating a pause application feature.  Why not integrate this into the window manager and allow people to pause their applications so that they dont waste processing power on them when they don't want to or to assist in killing an application.  Also assigning a hotkey to this function such at  Alt+(Pause/Break) for the window manager to use.

Step it up with pulse audio and make a true gnome like applet that has the functionality to match the PulseAudio Device Chooser 's volume control manager.  Maybe a similar interface to the network manager except for controlling sound outputs.
Also a very much appreciated visual sound piping interface that is accessible from the applet.  This application would allow simple visual piping (by say drawing a line) from one audio stream to its output or input device. ( Yes this would be made to work with pulseaudio. )  Maybe the ability to save preferences for common arrangements.

Also Integrate the functionality of something like AllTray into the window manager allowing any application in any state to be sent to the system tray.

Possibly new tactics for window tiling for more advanced users.
