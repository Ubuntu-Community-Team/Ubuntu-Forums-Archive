### Post by Afkpuz on 2007-11-08
**_There should be some sort of hardware compatibility tester provided by Ubuntu._**

I've surfed the forums a lot and have noticed that all the hardware problems people have.  On my personal system, I was able to have a flawless install with gutsy and I think that would save a lot of frustration involved with converting.  If more "converts" were able to check **beforehand** if their hardware could run linux, I think life would be way better for everyone involved.  

So, I propose a compatibility checker similar to the one provided by microsoft for vista.  Here are some of the specs I came up with for the checker.  There's room for more, but these are the barebones.
         1-It should run on windows, mac, and as many other linux distros as possible
         2-It should be small, so that you could put it on a flash drive and use it to test multiple computers
         3-It should be easy to understand
         4-It should tell the user whether or not it would be a good idea to install ubuntu
         5-It should be presented on the main ubuntu site and advertised
