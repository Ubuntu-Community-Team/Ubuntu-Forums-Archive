### Post by Rabindranath on 2008-01-31
I like ubuntu very much and I would like a few new features which I think are essential in the days to come.

1. There should some sort of 'offline' installer for programs without the use of internet. Many people dont have access to internet and ubuntu (linux in general ) should have an alternative mode for installing programs. Linux is pretty useless without a network connection. 
2. Some sort of 'system restore' like program for ubuntu included in the default install.
3. Few more themes included in the default install prompting the user to choose one during the first booting.
4. A universal program for installing device drivers (like wireless cards, modems -win), extra codecs like mp3s, etc (Similar to the restricted driver management). VLC can be included in the default install.
5. I haven't heard of a single portable application in ubuntu and ways for installing the same on devices such as usb drives would be a welcome feature. 
6. Creation of bootable dvd from ubuntu so that booting and installing ubuntu from the DVD would restore all programs and settings which are present from the ubuntu install from which it is created. 
7. Ability to install both KDE and GNOME on the same install without affecting the stability of installed programs. 
8. Automatically configuring the best download server during first network connectivity.
9. Some sort of data backup tool in the default install.
10. Above all these, stability and hardware detection should be awesome. (Ubuntu already has this but it would be nice if it is still higher)


           Sorry for my high expectations but if the these were included, it would be my dream machine. :lolflag:

### Post by smartboyathome on 2008-01-31
1. Like how? This would (in my mind) turn the CD into a DVD, as you would have to provide packages for people. I do agree that the package.ubuntu.com website needs improvements to make getting packages from another location easier, though.
2. Like the "system restore" programs included by manufacturers on computers, or system restore like in Time Vault?
3. I do not think this should happen because it would be annoying for someone who wants to get another theme from ie gnome-look.org to be presented with this. I think it is already easy enough to change a theme. The only difference is that there needs to be newer themes included, which I think will start being solved in Hardy+1 and beyond.
4. VLC can't be included with the default install due to licensing issues. Also, we already have the restricted driver manager, I don't see a need for this.
5. I heard of a way to do this before, I think it was included with ROX Desktop. It was a program that allowed you to install programs onto removable devices and run them off of it. Can't find it right now, though.
6. You can already do this with Remastersys.
7. This isn't Ubuntu's problem, its KDEs and GNOMEs. They don't play nice already, so I don't see this happening anytime soon. ;)
8. This would be nice, though it can cause problems when servers have become screwy (this happened to me before with Ubuntu, the server was chosen because of how fast it was, and it didn't have all the files needed by Ubuntu).
9. Like NSBackup?
10. Hardware can only come when more users come and manufacturers see it fit to make drivers for Linux. Stability is the top priority for Hardy.
