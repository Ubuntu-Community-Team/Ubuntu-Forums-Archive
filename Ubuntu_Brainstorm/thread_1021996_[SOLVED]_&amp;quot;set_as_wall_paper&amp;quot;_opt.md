### Post by steeleyuk on 2008-12-26
You'll probably need to restart Nautilus. For anybody who wants to do all this I might as well write it here:

1. Run *sudo apt-get install nautilus-wallpaper*
2. Once it has finished installing run, *killall nautilus*.
3. When you right click on any image, it will have an option "Set as Wallpaper..."
