### Post by gnomeuser on 2008-02-16
You could push for Tor to be shipped on the default install with a nice little gui to enable it optionally. This would also require a setting to allow easy sharing of your connection as the added pressure on the existing tor nodes otherwise would cripple the network.

That would solve the problem at least to an acceptable degree, all TOR needs to be truly useful for everyone is more nodes and easier tools to enable it.. Ubuntu can prove both at a fairly reasonable investment in development.

### Post by coolen on 2008-03-13
Creating a new IP protocol is not Ubuntu's business. That has to be a very open, collaborative effort.

Otherwise, I think this would be a bad move for Ubuntu. It's not our place to question censorship laws. If you want to do that, I suggest you get involved with Amnesty, or some other human rights organisation.

If users wish to do this, then they may do so at their own risk, but Canonical cannot run the risk of being responsible for any action which may be regarded as illegal.
