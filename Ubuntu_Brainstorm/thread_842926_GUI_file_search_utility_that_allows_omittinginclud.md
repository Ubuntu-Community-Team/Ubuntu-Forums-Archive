### Post by caljohnsmith on 2008-06-27
I noticed in the gnome-search-tool (Places > Search for Files), it has an option to "include other filesystems". So a relative newbie will probably ask, does that mean by default that any mounted partitions in my /media folder are not included if I don't use that option? And if I do use that option, then are *all* of my other partitions included in the search?

I think there should be a special place where the search tool listed all of the mounted partitions, and the user could merely check/uncheck them to include or omit them in the search. In addition, I think the search tool should allow us to simply input a list of directories to either omit or include. That way we could also do a search through a specific list of directories that may have no relation to each other. Wouldn't this be much easier to use? Seems like it would be easy to implement since I would imagine that the search tool is not much more than a GUI for the "find" command. 

Anybody else have any opinions about this?
