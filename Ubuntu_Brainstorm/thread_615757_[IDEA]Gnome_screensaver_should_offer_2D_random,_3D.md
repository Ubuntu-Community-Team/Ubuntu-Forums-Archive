### Post by feisty_hot_lover on 2007-11-17
The Gnome screensaver should offer the options 2D random, 3D random, Random all.  I use the nv driver and having the screensaver pop up a 3D screensaver is most unfortunate.  Me and other nv users would like the option of 2D random screensavers.  For those with 3D drivers they should have the option to pick only 3D screensavers.  I think this is a good setup as it allows 2D drivers users to get a random assortment of screensavers and lets 3D users get their eye candy.  Of course if someone wants all screensavers there should be an option for Random all.
