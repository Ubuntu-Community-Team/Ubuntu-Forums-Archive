### Post by Zeroangel on 2008-01-29
I remember posting a few months ago in the ["CTRL-ALT-DEL brings up system monitor"]("http://ubuntuforums.org/showthread.php?t=420170") thread in the Gutsy Idea pool, but am sad to see this not yet implemented by default since the idea got such overwhelming support from the ubuntu forums community. Therefore I propose here to post ideas and points on why this is a valid idea that should be implemented.

**1) Despite the fact that a vocal part of the Ubuntu and Linux communities dislike Windows -- its prevalence on computer desktops is a strong reality.**

a) It is because of this  that some attempts to unify the interface and usage of the operating systems takes place

b) Not everyone knows how to set keybinds when confronted with a new OS

c) Most people who install Ubuntu come from a Windows environment and many of them even dual boot to and from that environment when Ubuntu does not all of their needs, or out of undesired necessity

**2) Constantly setting and resetting options is time consuming. **

a) While there is a community of hardcore linux users who expect linux conventions to be used (ctrl alt del as logout), there is a growing influx of linux newbies who should also be considered. As an example, I sometimes install Ubuntu in order to introduce people to Linux when they cannot acquire a legal version of Windows.

b) The instances of the many should outweigh that of the few

c) Those who are used to long-standing linux/GNOME conventions typically have enough knowledge/inclination to set the keybinds themselves to the older style.

**3) While heurestically it makes sense that ctrl+alt+end brings up the option to ****end task, it also makes sense heurestically that this brings up the option to ****end session.**

a) Think about it, remapping this preserves the convenience of having the switch-user/logout/shutdown menu available and requires only a minor mental change to make it.

b) Saving users frustration even if it defies or cements long standing conventions can be worth it.

c) Nobody should ever have to visit a forum to find out how to do such a trivial thing as end a task.

Add your own arguments if you see fit on why this is a good or bad idea to do. :KS

### Post by chrisccoulson on 2008-04-15
I'm sure this was discussed in another thread a while ago. On Windows, CTRL+ALT+DEL is a multi-function combination that will do different things depending on the current state of the machine and its configuration (ie, sometimes it brings up Task Manager, sometimes it unlocks the screen, sometimes it brings up a menu to open task manager, change password or lock screen etc). I thought that there was another key combination in Windows which always opened the Task Manager.

CTRL+ALT+DEL in Unix/Linux has always traditionally rebooted the machine I think, so it is the wrong key combination to allocate to launching the System Monitor.
