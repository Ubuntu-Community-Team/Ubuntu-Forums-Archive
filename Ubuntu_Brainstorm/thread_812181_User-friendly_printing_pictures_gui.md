### Post by xens on 2008-05-29
Hi, I just added my first Ubuntu brainstorm idea  :)

[http://brainstorm.ubuntu.com/idea/9236/](http://brainstorm.ubuntu.com/idea/9236/)

_the idea_:

With a basic ubuntu installation you can choose the orientation and the size of the picture (with a resize-bar) you want to print, that's all.

If you want to print two or four pictures on a A4 page you have to open open-office and import the pictures then print them, or use gimp to do the same.

If you take windows XP for example (my family / neighborhood / ... came for there) you can chose the arrangement of your pictures on the final A4 / A5 page and print them easily.

There should be a way in ubuntu to select the desired pictures from nautilus and then do a right click "print selected pictures", then a wizard with multiple printing options:

- Number of pictures per page
- preview
- Pictures orientations
- Arrangements
- Possibility to add a text under the pictures
- ... 

I've made a mockup (in french, sorry :/)
