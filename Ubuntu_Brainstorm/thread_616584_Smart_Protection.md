### Post by smartboyathome on 2007-11-18
I think Ubuntu should try to make something like "smart protection". Basically, it warns you when a script is run from the applications/system menu (and no GUI is started). It could say something like "This program is trying to run a script instead of an application, which could potentially harm your system. Click Run to run it, click run in terminal to run it in a terminal, click do not run to halt the execution of this script (recommended)." This would allow for protection of Ubuntu users from unwanted virii (when they do come).

There are some problems with this, the biggest of which is that it would have to detect when something is running in the background (a terminal) and when something is run as a backend to an application. These would be big hurdles to climb, but I think there could be a way to differentiate between the two. Anyway, discuss.
