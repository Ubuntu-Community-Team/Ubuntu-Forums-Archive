### Post by doriard on 2008-10-24
In order for Ubuntu to make it as a widely used OS, several things have to happen.
1) Upgrading the OS must occur without it repeatedly breaking the OS.  For the past year, about half of the time I install upgrades, the OS breaks and I have to completely re-install Ubuntu.  This must be made more bullet proof.
2) CD/DVD drives must mount and unmount automatically.  Currently installing software via CD (doing this in wine is a common example) is WAY to much trouble, especially if it's a multi-CD installation.  Adding and removing a CD/DVD should be as seemless as in Windows.
These are the two biggest headaches I run across, but installing third party apps should also be easier.

Thanks for reading this.
