### Post by MONODA on 2008-03-07
This is a featue that I have wanted to have for quite sometime. What I mean by filesharing is filesharing over an ad-hoc network. It would be great it we could just create an ad-hoc network (or any other type of network), share some folders, and then others who are connected could access these files. Currently you need to install samba to share files and it is not exactly easy or quick to set up so I think something similar to what Fedora 8 has would be fine for now.  Also something similar to the giver project by OpenSuSE which lets two people who are connected to a network send files to each other. Do you guys think this idea is worth implementing?
