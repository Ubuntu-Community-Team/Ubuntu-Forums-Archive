### Post by last1 on 2008-06-01
Hi, is it possible for there to be greater collaboration between all versions of ubuntu so that they all have feature parity? There's gotten to be a number of things like overall functions of the OS that exist in one version but are not implemented in another. Being the most popular, you Ubuntu users have a more stable codebase and it certainly seems as if the developers contribute more code toward Ubuntu than other official ubuntu flavors such as Kubuntu. I'm not asking devs to come take over the work of KDE guys but we need a good dialogue going back and forth between the developers of all the official versions of ubuntu so that all the cool little things that "just work" in this version are also in the other versions. At the moment, this isn't the case. And that's a shame because it's missing out on an opportunity to increase the quality of all variants. Not only that but anything that has the word buntu in it ought to be known for the same high quality software, stability, and features as in the Gnome version.
