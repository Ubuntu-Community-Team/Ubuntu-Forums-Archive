### Post by Jan-Nik on 2008-01-28
The newest version of the Liberation Fonts can be downloaded here: [https://bugzilla.redhat.com/show_bug.cgi?id=427791](https://bugzilla.redhat.com/show_bug.cgi?id=427791)

I think the Liberation Sans would make a better sans-serif font because its hinting is better and IMHO it also looks nicer.
Also Liberation Sans has the same metrics as Arial. There are some web pages which aren't displayed correctly with DejaVu Sans.

Here's a screenshot which shows that DejaVu's hinting is worse: [http://watteimdocht.de/jan-nik/liberationdejavu.png](http://watteimdocht.de/jan-nik/liberationdejavu.png) (AA turned off)
