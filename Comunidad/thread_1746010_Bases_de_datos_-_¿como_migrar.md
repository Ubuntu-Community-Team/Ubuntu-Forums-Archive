### Post by juancarlospaco on 2011-05-08
Python viene instalado por defecto en Ubuntu, 
Ubuntu tiene aplicaciones hechas con el y ademas es parte del Standard de Linux.

La base ya esta instalada, lo que aparece para instalar son Librerias y herramientas de desarrollo,
Por las Librerias, 
es para extender y facilitar funcionalidades, depende que queres hacer son las que debes instalar,
para algo mas que un HolaMundo, pa no escribir todo desde cero vienen estas librerias,
Que necesitas hacer puntualmente con Python?, asi se puede mas o menos nombrar algun paquete...

Por las Herramientas de desarrollo,
es para realizar proyectos mas comodamente, si bien escribir codigo se puede con papel y lapiz o con Gedit,
las IDE agregan funcionalidades para el confort y mejora de productividad del que lo escribe,
un poco depende en gustos personales, Vim, Gedit, etc etc
podes probar Ninja-IDE: [http://ninja-ide.org/downloads.html](http://ninja-ide.org/downloads.html)   ;)
