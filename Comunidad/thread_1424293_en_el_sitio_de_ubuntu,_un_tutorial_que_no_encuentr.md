### Post by alfplayer on 2010-03-08
Para usar el minimal cd mini.iso no hace falta un tutorial. Para actualizarlo o instalar paquetes hay que ejecutar "apt-get update", después se puede actualizar con "apt-get upgrade" (ejecutarlo si se quiere mantener Ubuntu actualizado), y después se instalan los paquetes que uno quiere con "apt-get install *paquete1* *paquete2* ...".
