### Post by Mauro22 on 2010-01-24
Buenas!

Primero que nada, quiero avisarles que esto no es por andar haciendo propaganda, pero queria compartirlos con aquellos que usen emesene...


Este plugin esta desarrollado por Gustau "karasu" Castells y yo.


Dejo una descripcion corta, y si les interesa, pueden vistar la web oficial.


YAYoutube permite ver videos de youtube dentro de la misma conversacion o en una ventana aparte. YAYoutube detecta automaticamente cualquier link de youtube que se envie/reciba durante cualquier conversacion y ofrece las opciones correspondientes.

Como bonus, YAYoutube permite guardar los videos de youtube en el disco rigido.


[IMG]http://img31.imageshack.us/img31/6389/mainy.png[/IMG]


Importante: 

Son Necesarios los siguientes paquetes para funcionamiento del plugin.

python-gdata
python-gtkmozembed
python-webkit

```
sudo apt-get install python-gdata python-gtkmozembed python-webkit
```

-----------------------------

Para tener el emesene al dia:
```
svn co https://emesene.svn.sourceforge.net/svnroot/emesene/trunk/emesene emesene
```

-----------------------------

Para obtener YAYoutube al dia:
```
svn checkout http://yayoutube.googlecode.com/svn/trunk/ yayoutube-read-only
```

o simplemente guardar este archivo:

```
http://yayoutube.googlecode.com/svn/trunk/YAYoutube/YAYoutube.py
```


Copiar YAYoutbe.py en 

```
/home/usuario/.config/emesene1.0/pluginsEmesene
```




Gracias, y espero que lo disfruten :D
