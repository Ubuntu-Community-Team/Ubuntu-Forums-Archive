### Post by exegeses on 2009-05-09
hola, he vuelto y con el nuevo JJ (9.04), desktop con chirulitos y todo eso. :popcorn:

bueno, respecto del código postal, y porque a veces lo uso para programar, les comento cómo está compuesto:

ejemplo:
```
A1234BCD
```
la primera letra [ A ] es el código de provincia y se usa el mismo código que se usaba hace muuuuchos añños para las patentes de los autos: 
a=salta 
b=buenos aires
c=capital
etc, etc

luego vienen 4 números [ 1234 ] que es el viejo código postal, el de siempre

y finalmente 3 letras [ BCD ] sirven para identificar, zona, manzana y lado de la manzana. esto les sirve, cómo han dicho a los carteros para ubicar mejor al destinatario del correo.
este código es diferente sólo en localidades de más de 500 habitantes (localidades con menos de 500 habitantes poseen un único CPA)

finalmente, si necesitan buscar su CPA (código postal argentino) pueden ir [aquí]("http://www.correoargentino.com.ar/consulta_cpa/cons_.php") y buscarlo.

bueno, espero mi explicación haya sido clara y haya sido de ayuda.
saludos
