    Estimado Lauro, gracias por seguirnos. Por favor aclaranos cual es el tipo de radio ¿sintonizada con una capturadora TV/FM? Directamente de internet (streaming)?.

    Asumiremos que escuchás radios que hacen streaming (en vivo).

    Perón usa el peronista TOTEM con un Códec que te instala el Ubuntu cuando abres por primera vez un MP3. Se encuentra entre el paquete de extras restingidos (artículo ya analizado).

    Para instalar los extras restringidos, prueba usar en la consola Terminal y peronizarás tu PC:
    sudo apt-get install ubuntu-restricted-extras

    Con ese te cargará, entre otras cosas, los códecs para ver peliculas, unos para escuchar radio similar a Windows Media Player, etc.

    Si usas radios con formato realmedia (se abrian en Windows con realplayer) puedes usar amarok o rythmbox. Seguramente otros peronistas que nos sigan podrán aportar más opciones.

    Si no puedes solucionarlo, por favor indica qué radio no puedes cargar e intentaremos fijarnos a ver si podemos reproducir el mismo problema que comentas.

    Un saludo y gracias por contribuir al blog del general Juan Domingo Perón en su variante más Ubuntera.
