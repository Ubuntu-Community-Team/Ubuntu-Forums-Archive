### Post by tinoq3 on 2009-08-27
Está disponible Quake Live para sistemas Linux mediante el plugin de Firefox. Desde ayer existen servers argentinos online con un ping excelente.

No necesitan tener el juego instalado, sólo registrarse en la web [COLOR="Blue"]http://www.quakelive.com/[/COLOR] y descargar el plugin necesario para Firefox 2+.


Para más información visiten:
[COLOR="blue"]http://www.quakelive.com.ar/index.html[/COLOR]

Y el foro de Quake en Argentina:
[COLOR="blue"]http://foros.gxzone.com/quake_series-32.html[/COLOR]

Saludos!
