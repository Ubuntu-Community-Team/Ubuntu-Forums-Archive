### Post by guillermolisi on 2010-05-31
Este tipo de acontecimientos se lleva a cabo usando IRC (chat).
Para el desarrollo ordenado de las presentaciones se habilitan dos canales: uno para la presentacion de contenidos y respuestas a las preguntas que la gente haga y otro exclusivamente para formular las preguntas.

Si se usa Lernid, el aplicativo permite tener a la vista en sectores separados ambos canales y en un tercer sector poder visualizar diapositivas de apoyo a la disertacion (en caso que el presentador las haya preparado para el caso).

Si no se sabe como usar y comportarse en este tipo de ambientes, lo mejor es utilizar el navegador web siguiendo las instrucciones que figuran en la [wiki de "Dia del Usuario Ubuntu"]("https://wiki.ubuntu.com/DiaDelUsuarioUbuntu").
