### Post by sdennie on 2009-06-08
Capaz que solo tienen problemas con el DNS.  Fijate con:

```

dig ubuntuforums.org

```

Vas a tener un numero que dice cuanto falta para cambiar el nombre a un IP.  Si el numero es grande y sigue siendo grande despues de hacer el dig un par de veces, instalas dnsmasq para cachear el DNS.
