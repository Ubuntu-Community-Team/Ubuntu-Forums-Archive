### Post by Carlos C on 2009-12-23
Recuerden que deben haber firmado el código de conducta. Para saber cómo firmarlo pueden ver las instrucciones aquí:

[https://wiki.ubuntu.com/ChileanTeam/General/CodigoDeConducta/FirmandoCodigoDeConducta](https://wiki.ubuntu.com/ChileanTeam/General/CodigoDeConducta/FirmandoCodigoDeConducta)

El código de Conducta es fundamental para ser un Ubuntero. La idea es comprometerse de verdad con lo que dice ahí si quieres tener derecho a participar de las decisiones que tomamos como comunidad. Es por eso que incluso este tipo de elecciones requieren de haberlo firmado.

Justamente debido a lo anterior les recuerdo:
> Para identificar quienes votan, todas las personas deben tener un resumen en su perfil de launchpad.


Espero que con eso se solucionen las dudas. Tratamos de responder lo más rápido posible pero a veces el tiempo no nos alcanza como quisiéramos.

Saludos!
