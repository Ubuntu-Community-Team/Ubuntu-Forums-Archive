### Post by Lars Noodén on 2011-05-19
In addition to jitsi, there are other SIP clients.  Remember, that any SIP phone can call any other SIP phone.  Both the caller and the called can be using different software, it is the protocol that counts. SIP is like the web for VoIP.

[list]
[*] [Blink](http://icanblink.com/)
[*] [Ekiga](http://www.ekiga.org/)
[*] [Empathy](http://live.gnome.org/Empathy)
[*] [Google Talk](http://www.google.com/talk/)
[*] [Jitsi](http://www.jitsi.org/)
[*] [Linphone](http://www.linphone.org/)
[*] [Mumble](http://mumble.sourceforge.net/)
[*] [QuteCom](http://trac.qutecom.org/) (formerly Wengo)
[/list]
