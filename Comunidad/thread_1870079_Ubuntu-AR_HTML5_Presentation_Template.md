### Post by juancarlospaco on 2011-10-26
**Una Plantilla, para hacer una Presentacion, pero en HTML5, GPL, lista para usar y facil de usar.**

**presentacion.html**________________HTML5 Presentation Template
**presentacion-intro.html**___________HTML5 Presentation Intro/Block Page *(Opcional)*
**musica.ogg**______________________Musica de fondo para la Presentacion *(Opcional, .OGG)*
**click.ogg**_______________________Efecto para presentacion-intro.html *(Opcional, .OGG)*
**manifest.appcache**_______________App Cache para  presentacion.html *(Opcional)*
**style.css**________________________Estilos CSS para la Presentacion
**devicenzo.py**_____________________Mini-Browser Webkit+Python+Qt *(Opcional, compatible, de Auxilio)*
**LEER.txt**_________________________Eso mismo

**Funcionalidades:**
[LIST]
[*]Usarlo desde el Teclado
[*]Retroceder a Diapositiva anterior
[*]Recargar Diapositiva Actual
[*]Rebobinar Diapositivas al Inicio
[*]Editar una Diapositiva al vuelo *(permite guardar)*
[*]Imprimir Diapositivas
[*]Avanzar a Diapositiva posterior
[*]Avance Automatizado a 30 Segundos X Diapositiva
[*]Avance Automatizado a 1 Minuto X Diapositiva
[*]Avance Automatizado a 3 Minutos X Diapositiva
[*]Avance Automatizado a 5 Minutos X Diapositiva
[*]Reproducir, Pausar, Detener Musica de Fondo con Volumen Variable, Opcional
[*]Notas Rapidas para el Disertante, Opcional
[*]Informacion del Browser *(Debug)*, Opcional
[/LIST]

Se puede editar *"presentacion.html"* con Kompozer, Gedit, Vim, Ninja-IDE, ed, etc
No se necesita saber HTML, si se escribe en texto plano tambien funciona.
Trae Lorem Ipsum de ejemplo de contenido.
Lista para meter en un Apache o usar suelta.
Esta pensado para que sea facil de usar, cada Diapositiva esta delimitada por etiquetas:

```
<section>
 Todo lo que se escribe aca es 1 Diapositiva, el Javascript que tiene incrustado hace el resto.
</section>

<section>
 Otra Diapositiva :)
</section>
```

