### Post by sergiom99 on 2008-10-08
creo que seria algo así como :
> "Libre como libre expresión, no como cerveza gratis"

Es un juego de palabras entre Free (libre) y free (gratis), dos significados para la misma palabra en el idioma inglés.

### Post by Hei Ku on 2008-10-09
Se refiere a que el codigo GPL es libre, no gratis. El objetivo es que el software se mantenga libre, para que el usuario final pueda hacer con el software lo que quiera, aunque tenga que pagar dinero por ese software.
