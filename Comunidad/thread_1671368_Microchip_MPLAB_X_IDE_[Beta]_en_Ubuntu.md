### Post by madbyte on 2011-01-19
Una buena noticia para los amantes de la electrónica y la programación de microcontroladores PIC. Microchip lanzó hace unos meses MPLAB X IDE. Un entorno de desarrollo IDE completamente nuevo en Java bajo plataforma NetBeans, soportando no solo Windows sino también Linux, y MacOS. Aunque todavía se encuentre en fase Beta y las herramientas de debug no están todas soportadas, se presenta como un mundo completamente nuevo frente al viejo y desactualizado MPLAB para Windows.
Un acierto, que se hizo esperar, pero al fin, ya es una realidad.
