### Post by cfoch on 2009-01-27
Buenas noches, usuarios de esta comunidad, con un amigo hemos publicado una pagina web para difundir el software libre, especialmente Ubuntu. en todo el mundo, pero principalmente en Sudamérica.
Con esta web, pretendemos ir formando una gran comunidad, que cada vez se nos una mas y cada vez mas gente con la idea de apoyar en la difusion del software libre por medio de charlas o conferencias por chat.
En ubuntu-sud.tk damos herramientas para compartir conocimiento y/o solucionar problemas, tales como el chat y el foro.
