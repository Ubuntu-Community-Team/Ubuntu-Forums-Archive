¡Llegaste al lugar indicado! La página de Doral Net tiene una función que no es soportada por Mozilla para Linux. Esta es la solución:

1. Instala Wine:

```
sudo aptitude install wine
```

2. Bájate SeaMonkey para **Windows**:

[http://www.mozilla.org/projects/seamonkey/](http://www.mozilla.org/projects/seamonkey/)

3. Ejecuta SeaMonkey para Windows y entra a Doral Net desde él.

Espero que te sea útil. Saludos desde el Área Metro. :)
