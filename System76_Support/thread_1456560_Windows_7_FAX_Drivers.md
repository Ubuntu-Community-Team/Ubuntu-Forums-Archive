### Post by vgrisham on 2010-04-28
This has been resolved.

For anyone subsequently installing Windows 7 (dual boot of course), you need to make sure to install the chipset driver first. Everything else works pretty will work pretty well after that. (Thanks thomasaaron for that tip.)

As an aside, I wanted to avoid wrestling with grub and partitioning, so I installed Ubuntu via WUBI in Windows 7. So far, it's working very well, and since it's all NTFS, it's easier to access user data from either OS.
