### Post by thomasaaron on 2009-05-16
The primary problems would be:

1. It comes with pretty low performance graphics, as it is designed to run Ubuntu server edition (which is a command-line OS). You *could* install a nice graphics card in it, maybe an nVidia or an ATI card. You can also install Ubuntu Desktop on it.

2. It's too loud to sit under your desk. This machine is designed to sit in a server closet.
