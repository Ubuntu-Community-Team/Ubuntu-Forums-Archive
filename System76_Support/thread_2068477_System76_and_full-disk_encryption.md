### Post by Welly Wu on 2012-10-09
I re-installed Ubuntu 12.04.1 64 bit Long Term Service on my System76 Lemur Ultra Thin (lemu4) notebook PC using the Alternative Install ISO CD-R with full-disk encryption using LVM and then I proceeded to upgrade to Ubuntu 12.10 64 bit Beta 2. It went by smoothly without one single glitch. I am still testing Ubuntu 12.10 64 bit Beta 2, but so far nothing catastrophic has happened. I just wanted to let other System76 users know about this fact.
