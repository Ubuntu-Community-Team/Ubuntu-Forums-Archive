### Post by thomasaaron on 2009-04-26
Get recording working on Sound Recorder. Then go into the audio settings of skype.

Audio Input should be set to HDA-Intel.
Everything else should be set to Skype.

If your cam works in Ekiga or Cheese, it should work no problem in Skype. Make sure you have pressed Fn-F10 to activate the camera.

Also, if you have another cam-using app open, you should probably close it before opening Skype.
