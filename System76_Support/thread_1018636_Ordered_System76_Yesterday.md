### Post by ProfDecoy on 2008-12-22
By default, Alt-F2 will bring up the run dialog in Ubuntu.  The run terminal option appears to be disabled by default in 8.10.  However, it is configurable in the System\Preferences\Keyboard Shortucts option.  When the run dialog comes up, you can type in xterm or gnome-terminal to bring up a terminal window with out needing a mouse.

I've personally reconfigured the run dialog to Super-R (Windows Throwback there) and the terminal to Super-T.
