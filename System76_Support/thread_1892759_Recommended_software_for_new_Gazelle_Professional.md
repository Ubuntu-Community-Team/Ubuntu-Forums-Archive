### Post by ubuntu27 on 2011-12-08
BY VLC, do you mean [VLC Player]("http://www.videolan.org/vlc/")? If so, it is available in Ubuntu's repository. You can install it using *Ubuntu Software Centre*, the terminal, or *Synaptic Package Manager*.

For firewall, you could use [UFW (Uncomplicated Firewall)]("https://help.ubuntu.com/community/UFW")

[https://wiki.ubuntu.com/UncomplicatedFirewall](https://wiki.ubuntu.com/UncomplicatedFirewall)

[https://help.ubuntu.com/community/UFW](https://help.ubuntu.com/community/UFW)


You can still Firestarter if you want to.
