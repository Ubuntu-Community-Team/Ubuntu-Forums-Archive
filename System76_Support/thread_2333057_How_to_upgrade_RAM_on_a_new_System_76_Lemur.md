### Post by jlh68 on 2017-01-18
Another thought on this.  Try taking the original 4GB RAM out and put your new RAM in its place and try to boot.  If it does not boot then your new RAM is suspect, if it does boot then install the original RAM in in the second slot and see if the original is recognized along with the new RAM.  If the new RAM is recognized and the original RAM is not recognized the system has a problem.  Get in touch with System76 and get them to help.
