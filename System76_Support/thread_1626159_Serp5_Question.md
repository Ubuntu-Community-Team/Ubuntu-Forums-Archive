### Post by zitch on 2010-12-09
I'm pretty sure you do.  I have a Serp5 and I know I do.  If you want to do a quick test (assuming you're running a recent Ubuntu, I'm running Ubuntu 10.04), See if it activates with Fn-F12 (It's not marked on the function key, but that's how Bluetooth is activated on my Serp5).

When I press Fn-F12 on my Serp5, the "Wireless" light (third light from the left on the front-right of the laptop) will turn yellow and the bluetooth symbol shows up in the indicator applet. That should pretty much show if you have bluetooth if the same happens for you.

Unfortunately, there's no way to have Bluetooth activated on boot; you have to press Fn-F12 each boot if you need it up.
