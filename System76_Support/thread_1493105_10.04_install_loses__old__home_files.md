### Post by isantop on 2010-07-12
Okay, ensure the filesystem mounts properly:
```
sudo mount -t ext4 /dev/sdb1 /mnt
```
Check to make sure your old home folder is named exactly the same as your current one.
```
ls /home /mnt
```
Both folders above should have the same contents. If you get no errors there, go ahead and open up your fstab file:
```
gksudo gedit /etc/fstab
```
and add the following to the end of it:
```
# Custom option for old /home directory
UUID=6ab8e9e1-a6ea-4d4d-943a-2af1f0559d06 /home ext4 errors=remount-ro 0 1
```
Reboot to check.
