### Post by PoachedSalmon on 2009-12-30
For some reason, my Starling is now failing to detect my wireless network. I can connect to the Internet using a wired connection and the Starling detects other wireless networks in my immediate area.

Two days ago I had no problem connecting via my wireless network. This morning, I got nothing. I hadn't done any updates so it couldn't be that. 

However, this evening I installed some updates that were waiting for me and then reinstalled the System76 driver thinking that might be the cause. Nothing.

I've also restarted several times, I even tried connecting by using "Connect to Hidden Wireless Network" even though I haven't been hiding my network. My network then showed up in the list of networks, but indicated no signal (no blue bar), it kept asking for my encryption password, and tried to connect, but never did. Now, again, my network fails to show up in the list of available networks (in the top toolbar).

I also deleted another wireless network that I had used this past weekend, leaving only my network listed in Network Connections (Preferences -->Network Connections). Still nothing.

Any ideas would be greatly appreciated. Thanks!

---

### Post by PoachedSalmon on 2009-12-30
Never mind. I tried turning off my modem/router for a couple of minutes and then turning it back on. After doing that, the Starling detected my network.

I'll mark this as solved. :P
