### Post by jml on 2011-01-23
If System 76 is selling them, then the answer is yes.  I am not trying to be smart, or start a flame war.  But one of System 76's strengths is to build, configure and sell computers that run Ubuntu well out of the box with minimal tweeking by the owner.  I will let the S76 representatives answer the graphisc question for themselves.  But my experience has been that if System 76 sells a computer with hardware that is not fully supported by Ubuntu, they will write custom drivers and patches to make sure the hardware will work up to spec.  A good example is the wireless card in the first generation Starling.  It shipped with a wireless card that got sub-optimal performance with straight Ubuntu.  System 76 wrote a custum driver/patch to allow the card to work correctly.
