### Post by Geoffrey_Arndt on 2015-07-01
I own a Sys76 Galago Ultra Pro Laptop.    Have used it daily (4-12hrs) since Dec 2014 . . no issues at all to-date.   Sys76 does not use UEFI BIOS in this laptop, and I believe it uses a stripped down standard BIOS on all their machines.   Ubuntu is installed using legacy MBR.   So, the answer to 2-5 is no, but I would call Sys76 and actually ask (or send tech support an email) to verify if my "assumptions" are correct.   They've been quite responsive and decent to work with (in my experience).   

I would be surprised if they have any intent to add or support UEFI/EFI anytime in the near future.   Their BIOS seems simple, fast and easy to work with, and System76 provides updates via their PPA.   

If UEFI is needed or preferred (as in running a dual-boot with MS),  then it may be useful to check out the line of Dell laptops that have Ubuntu pre-installed (also, their desktops allow for easy install of Ubuntu on certified models (see Ubuntu documentation re hardware certification).
