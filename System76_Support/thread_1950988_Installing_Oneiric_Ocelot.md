Actually, we recommend waiting in the case of upgrades. Clean installations are clear to update as soon as the ISO is available. 

It's true that both have risks associated with corrupt downloads. However, with upgrades, a corrupt package is installed soon after it is downloaded without user intervention. If a bad .ISO is downloaded, it usually won't boot or install, meaning you can simply boot back into the installed copy of Ubuntu to correct the issue.
