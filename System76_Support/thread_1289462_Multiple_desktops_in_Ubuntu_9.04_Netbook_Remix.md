### Post by macabrem on 2009-10-13
I think I saw on a previous forum to do this:

Switch to "regular" Ubuntu mode, enable multiple desktops, then switch back to Netbook remix.  Now there should be multiple desktops in the remix.  I'm not sure if you will actually see a small visual indicating you have multiple desktops, or if you will only be able to access them by hitting control+alt+left or right arrow...

For some reason, the remix doesn't work well for me on my Starling - very slow.  I think I like the standard display more anyway.

### Post by fjrichman on 2009-10-19
Another way you could do this is installing Compiz Settings Manager then go to:
General Options > Desktop Size
Change the numbers accordingly then you can use the various features of Compiz to change desktops Like Expo if you have more than one row of desktops.

Atleast this is how I setup multiple desktops on UNR using 9.10 I'm assuming it'd work in 9.04 and it works without switching to Regular mode.
