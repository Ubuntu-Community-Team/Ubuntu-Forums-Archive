### Post by jml on 2011-09-21
The previous poster is correct.  The Apple EULA for OSX specifically states that the OS is licensed only for installation on Apple hardware.  So getting support for the OS on other hardware would be a major issue.

As for Windows, another poster is correct, one will need to purchase a full version of Windows 7, and then find and install the drivers for your particular hardware.  System 76 does point you to the appropriate location for the drivers as previously mentioned.

One other thing about dual booting Windows and Linux.  The Windows installer does not play well with other boot loaders.  It will usually over write the boot loader, so to successfully dual boot, one needs to do a clean install of Windows on the entire hard drive.  Install your needed drivers.  Once you have a functioning Windows system, install the Linux distro of your choice and repartition the hard drive from within the installer program.  This is why I always keep a very cheep used laptop loaded with Windows for those very rare times I need that OS.  That way, I do not have to mess with dual booting.
