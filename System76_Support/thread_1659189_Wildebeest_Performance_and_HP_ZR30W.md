### Post by isantop on 2011-01-05
It is mostly assumption, but the GTS 210 (the default) supports 2560x1600, and that 9400M does as well, so i assume that the large, more powerful cards do as well. If you're really doubtful, then I can check, but you're probably good to go.

The Dual-Link cable will be required to run at that resolution, as a single-link cable doesn't have enough bandwidth. If the cable it has runs the native resolution, it is a dual-link cable.
