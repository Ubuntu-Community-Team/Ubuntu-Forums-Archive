### Post by damedpunk on 2014-07-02
I bought a panp7 about 4 years ago, and it worked fine for a while. I installed crunchbang last year after failing to control the CPU/GPU temperature and the fan speed. 

Yesterday, the screen started displaying funny squares and colors. It had happened before when it was running Ubuntu and overheating, so I didn't think much of it and shut it down. Today when I tried to boot it up again, it barely made it past the system 76 splash before the screen went black. Now, it generally does not make it to the system 76 splash, although I can sometimes log in and use my computer for a few minutes. Then one of the following happens: the background will turn green with horizontal lines and text showing, the entire display will start misbehaving and showing random patterns of colors, the screen will be on, but black, or the screen will shut off altogether. The following steps have been taken:
- motherboard beeps:none
- external monitor: doesn't display anything, but seems to be recognized.
- trying to reboot on live CD with/without HDD: no effect
- rebooting on the battery, without the battery: no effect
- reseating everything: no effect
- blowing the motherboard thoroughly: no effect
- reapplying thermal paste to the CPU: no effect (other than apparently cooling down faster than it used to. I should have done that before... Like a year after I bought it *cough-cough*)

Is there anything I haven't thought of? What then?

Thanks

### Post by RedRat on 2014-07-08
> **eightbits2010 said:**
>  I am now thinking it is better to just build my next PC and then install Ubuntu and save a few bucks.

I did this way back when WinXP came out. Got a pretty hot machine, but it quickly got surpassed by the latest and greatest. Eventually converted it to Ubuntu 7.04LTS. Not a bad idea but it didn't save me all that much--I wanted the latest and greatest at the time of the build. It is fun. However, I wouldn't recommend doing that now. It can be a hassle especially trying to match up hardware with the motherboard. Why not just go down to your local Big Box store, e.g., Costco, Best Buy, and pick up a Windows 7 machine and then install Ubuntu on a dual partition. You can then use Windows when you want or need to use for specialized Win programs. Reboot into Ubuntu and use that when you want. There are some pretty good machines that fully integrated in terms of USB (2 & 3), various SD cards, good audio and well made. I am now running an much older HP Pavilion that can handle TV and PC stuff. I dumped the old WinXP that was on it and now it dual boots Ubuntu 12.04 and 14.04.
