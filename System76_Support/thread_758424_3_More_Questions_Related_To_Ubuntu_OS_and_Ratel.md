> I hear about a Ubuntu OS program called 'Wine' and I am wondering since I always see it referred to in posts about Windows machines, if it in any way is a Microsoft program? If it is a Microsoft program, is there a way to delete it from the Ubuntu OS? 

WIne is a linux program that fools Windows programs into thinking they are running on MS Windows. It doesn't work with all Windows programs, but it is getting better all the time. It does not come pre-installed with Ubuntu. You have to add it yourself. Here is a wikipedia article on it: [http://en.wikipedia.org/wiki/Wine_(software](http://en.wikipedia.org/wiki/Wine_(software))

### Post by jeamer on 2008-04-21
I've been using live365 (free) for a couple years on Ubuntu now, I found that I avoided a lot of headaches by downloading beep media player--I initially installed it because of it's similarity to WinAmp, which I miss dearly :). If you set it up so the station plays through that player, you can force quit (i.e. via system monitor) the website and beep continues to play the station! This is awesome if you don't want the annoying and RAM eating advertisements that come with having the website player window open, also if you keep the streaming station on your beep player playlist, you don't have to go to the website every time, just load beep and hit play!
