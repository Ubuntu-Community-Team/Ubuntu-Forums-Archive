CPU
AMD Athlon 64FX/Athlon 64 up to 3800+
Socket 939
800 MHz FSB

Chipset
North Bridge: VIA K8M800
South Bridge: VIA VT8251

Memory
2 x DIMM
DDR 400/333
Support max. 2GB

Expansion Slots
2 x PCI
1 x PCI-e
1 x AGP 8X

Graphics
VIA Graphics Integrated

IDE
2 x UltraDMA 133

S-ATA
4 x Serial ATA

LAN
10/100 Mbps

Audio
Realtek ALC 653
6 Channel Codec

Dimension (mm) W x H x D
168 x 380 x 354

Drive Bays
2 x 5.25"
3 x 3.5" (2 external)

Front Panel
2 x USB 2.0
1 x Microphone
1 x Headphone

Rear Panel
4 x USB 2.0
1 x Line-in/Line-out/Mic-in
1 x PS/2 Keyboard
1 x PS/2 Mouse
1 x Parallel Port
1 x Serial Port
1 x D-sub (VGA Out)
1 x RJ45 LAN

Power Supply
300 W (PFC)
