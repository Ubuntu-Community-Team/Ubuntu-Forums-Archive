### Post by shingalated on 2007-07-28
Yes, it looks good.  I have never heard of pioneer preinstalling linux, but I guess its just becoming more common!  I doubt you will have any hardware compatibility problems.  The only thing is I have never used a  brother printer with ubuntu, so I don't know how well that will work.  HP and Epson are both very well supported, and I have used them both.  You may have to go online and download the driver.  As far as installing the OS yourself...It couldn't be easier.  I am higly confident that you should have no problem installing ubuntu on a new computer.  The install cd is great, it will boot into a live version of the OS where you can try it out running of the disc, its kinda slow....but its running of the CD.  All you have to do is double click the Install icon and it will take you through a quick installation wizard.  It is easier that installing windows.  If you are choosing between 32 bit ubuntu and 64 bit ubuntu,  I would definitely pick the 32bit.  The 64 bit one may have trrouble running a lot of 32 bit applications, (which most are)  and is just way to hard for a beginner.  You will end up having to install 32 bit library files and ...yeah just dont.  It's really not that noticable of a performance gain anyway.  Hope this helps. :)
