### Post by thomasaaron on 2008-04-10
The System76 driver will run on Kubuntu. I know we have some customers running Kubuntu on the Serval, and I've never heard any complaints. You might run into some key-mapping issues, but since we do no testing with Kubuntu, I'm not really sure.
