### Post by zenlunatic on 2012-08-28
There is no need to match up the card with the router.  They're both wireless devices which communicate using standard protocols.

Hardware is only as efficient as it's lowest bottleneck.  Therefore, if you're not setting up a serious home network with heavy traffic, your internet connection will be in this instance your bottleneck.  Therefore wireless N isn't necessary.

I would recommend a WRT54GL and install DD-WRT on it.  It isn't that hard and it offers a lot of features and is quite easy to use once you get the firmware flashed.  I have my folks on that and they're going along quite nicely.

For simple setups a budget G router will be quite enough.
