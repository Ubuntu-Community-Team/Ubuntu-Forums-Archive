### Post by cj.surrusco on 2010-07-10
In Firefox, it can be done this way.

> 1.
Open a new tab in Firefox, and type about:config, then click I’ll be careful, I promise!

2.
In Filter area, type "browser.urlbar.clickSelectsAll". Double click the result line and set its value to "True".

Finally, restart Firefox. Left click in the address bar and see the effect!
source: ubuntuguide.net

### Post by mikewhatever on 2010-07-10
Let's translate the question into intelligible English.:p

Question:
How to highlight the URL in Firefox with one click?
Answer: 
Open Firefox and type 'about:config' in the address bar.
Locate the following entry - 'browser.urlbar.clickSelectsAll'.
Double click to change its value to 'true'.
