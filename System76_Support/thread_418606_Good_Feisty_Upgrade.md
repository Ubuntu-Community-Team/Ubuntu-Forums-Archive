### Post by AusIV4 on 2007-04-22
Man, I wish it'd gone that smoothly for me. I set it to start the upgrade when I woke up on the morning of the 19th. It wasn't done when I went to bed. It paused to ask me a question in the middle of the night, so it wasn't done in the morning. When it finally finished, it would always boot up giving me a command line that said:

Command apt-get not found. Type apt-get install apt to install.

Which of course didn't work, but if I typed exit, it would continue to the desktop, where everything worked somewhat normally. Sound didn't work, my swap partition had been disabled, so I couldn't hibernate, suspend didn't work, and Kvpnc wouldn't connect me to my campus network. I reinstalled from CD (my home partition is separate, so it wasn't a big deal), reinstalled the needed programs. It now booted properly and had a swap partition mounted, but the rest of my problems continued. I've got sound, hibernate and suspend working, but Kvpnc and the fact that it freezes up for 30 seconds at a time are keeping me on edgy until I hear some more fixes have come through.

Feisty looks like it will be pretty cool, but it's not there quite yet.
