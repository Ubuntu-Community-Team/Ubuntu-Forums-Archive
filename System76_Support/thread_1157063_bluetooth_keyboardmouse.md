### Post by Lee_Machine on 2009-05-12
S76 does a good job making sure that what hardware they put into their computers will work in Ubuntu.

Ubuntu comes with a BT manager out of the box. I have found that installing other BT mangers such as Blueman tend to create more problems. 

The only time you need to install extra BT apps is when dealing with transfering files via bluetooth. 

I know on the Pangolin Performance you have to press Fn-F12 to turn on Bluetooth...Im not sure about the Gazelle.

### Post by miniyak on 2009-05-25
for a mouse bluetooth might make sense, however for a keyboard only to be used at home you may want to consider options that come with dongles. I say this because normally bluetooth keyboards are over priced and hard to find. I was shopping for one last fall and found (through forums) that on some models that the dongles included were necessary for functionality.(which makes very little sense IMHO) That was the denovo edge that i was looking at, if you want big keys instead small laptop ones obviously you would be looking at something else though. 

in my experience bluetooth has always been disappointing. particularly when using odd devices like stereo headphones. getting the odd balls to automatically connect is probably going to be a pain. 

i should note that i had more problems in windows then linux however
