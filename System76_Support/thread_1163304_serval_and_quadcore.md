### Post by zitch on 2009-05-18
At least as a reference, I'm running the Core 2 Duo P8700 2.53 GHz 1066 MHz FSB 3 MB L2 (dual core, 25 watt) processor on my Serval and it lasts just under 2 hours on battery.  I'm also running the compiz desktop effects, bluetooth, and wireless internet, too.  Turning those off might affect battery time, but I've never tested that.  I wouldn't imagine the quad core would last any longer than that.

Was tempted to get the quad core myself, so I do have a curiosity about this too.

It also seems to take the heat pretty well.  The keyboard doesn't get nearly as warm as my older laptops would, even when I'm doing something like playing Left 4 Dead on Cedega.  Haven't pulled the temps at load through.  I'd imagine the quad core might be warmer, but I barely hear the cooling system as it is.  Seems like it have much more cooling capabilities to go.

For one reference, I'm running a light load right now (Windows XP Pro idling in a Virtualbox right now, several apps like Firefox and Zend Studio open) and the CPU is reporting 46 C.
