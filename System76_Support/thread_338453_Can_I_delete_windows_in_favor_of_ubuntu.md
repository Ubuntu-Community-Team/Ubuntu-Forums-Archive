### Post by cilynx on 2007-01-14
You boot the Ubuntu install CD.  You don't run it from inside Windows.  Once booted into the Ubuntu install system, you have the option of installing over (and effectively deleting) your Windows setup or installing along side such that you can choose on boot of the machine which operating system you want to run.
