### Post by Flyers2391 on 2010-01-28
> **eddietours said:**
> that would be cool Flyers2391 l like to know the resolution and if it plays nice with avi files :D

Alright, well, I set it up but unfortunately I cannot give you as complete of a review as I would like to (I was only on it an hour or so and it's not mine so that's my excuse :)), and I'm sure not as much information as you would want.  However, I'm looking at the lemur and like to read anything people have to say about.  

2 GB RAM and 160 GB hard drive were the only upgrades

As I mentioned, it looks nice and I like the metal body instead of plastic like most other desktops.  This Meerkat is not the Ion so I can't be sure how that would run but I hooked it up to a new 22" monitor and it defaulted to the highest resolution listed at 1680 x 1024.  
I had some network card issues and while I was trying to figure out the problem (solved by the way) I saw that the atom in there shows as 4 processors (2 cores, each HT I imagine).  The entire time using it I had no response problems.  
Desktop effects were set at normal and all ran with no noticeable slow-downs (desktop slide, ring switcher, minimize effects).  
Unfortunately I didn't have time to play with it some more but I would definitely purchase it for myself, if I needed something like it.  

I hope some of that was at least useful to someone :D

### Post by thomasaaron on 2010-02-01
```
What is the difference between the Meerkat NetTop and Meerkat Ion NetTop? 
```

The primary difference is that the Ion has nVidia graphics, which gives it better graphics performance, particularly if someone wants to use it as a media center.

> It will be used mostly for running an application in WINE - Finale PrintMusic. Other uses include GoogleEarth, and the usual email and net surfing/videos, no gaming.

The wine application shouldn't be a problem.

Anyone out there running GoogleEarth on a standard Meerkat? How about an Ion? If memory servers, it's significantly better on the Ion.

> Also, any thoughts on 32 vs 64-bit?

64-bit may be a little faster. Flash works a tad better on 32-bit.

> If we choose the 64-bit, should we get more than 2 gigs of memory (RAM)? 

Never hurts to have more RAM, especially if you're going to have multiple windows open at once or do anything graphically intensive.

### Post by edlyk on 2010-04-12
I bought a Meerkat Ion last month.  I am using at as a media center running XBMC.  I can get 1080p via HDMI to a Vizio TV, although I keep it set on 720p to make it better for occasional web browsing from across the room.  My only upgrade was 4GB of ram.  I have two 1TB USB drives on it.  Getting sound over the HDMI took an additional setting, same with getting nVidia to remember I wanted 720p (it kept defaulting back to 1080). Overall I am very happy with the result.  I am now working on getting a Streamzap remote to work with XBMC.
