### Post by isantop on 2010-05-26
That's a pretty accurate picture.

The large panel to the left is the main panel, which is removed with three screws. It provides access to the CPU, GPU, Heat sink, CPU fan, memory, Bluetooth, Wifi, Etc.

The smaller panel towards the thop right is the HDD access panel. It provides access to the Hard Drive. It comes off with two screws.

So, I'd say it's pretty serviceable. Definitely one of the easiest laptops that I've ever worked on.

Also, the access is about the same going back at least to the the Panp4. I haven't used anything older than that.
