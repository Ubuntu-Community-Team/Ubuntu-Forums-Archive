> One last thing here. I've read that 'System 76' closes ALL the ports except the essential ones, that you need for the machine to operate, is this true?

Let me take a step back here for a moment -- for anybody new to Ubuntu/System76. Ubuntu is a Linux Operating System, that System76 puts on hardware (in this case, the Ratel). 

So when you ask if the Ratel will burn music CD's or if System76 closes ports, it seems to reveal an underlying misunderstanding about what we do. Ubuntu contains software for many different types of jobs. We make sure that Ubuntu works on our computers and provide patches where necessary so that our machines are configured to utilize the Ubuntu Linux OS. We do not fundamentally change Ubuntu. If Ubuntu closes ports, then they will be closed on the Ratel. That's not the kind of thing that we mess with. Does that make sense? (I'm not trying to be condescending, I just want to make sure I'm clearly explaining things for everyone.)

That said, I believe you are asking about security.Ubuntu does ship with all public ports closed. So, a firewall in Ubuntu isn't really all that valuable IMHO except to protect you from accidentally installing a server that might open a public port. Ubuntu is a very secure operating system. There are a number of differences between it and Windows, and there are a number of similarities between Ubuntu and Mac -- as it pertains to security.

If you're asking if Ubuntu can be hacked, nothing is impossible. However, it is vastly more difficult than hacking Windows. However, if someone can hack a mac, they might be able to do the same to Ubuntu.

Here is an article that lays out some good security practices for Ubuntu.
[http://www.psychocats.net/ubuntu/security](http://www.psychocats.net/ubuntu/security)
