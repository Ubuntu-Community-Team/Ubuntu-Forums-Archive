### Post by thomasaaron on 2010-07-09
We don't really characterize it as x amount of cells. It is a Li-Polymer battery pack (3800mAh 42.18Wh).

It gets about an hour of battery life. The Serval is really designed to be a desktop replacement system. As such battery life is sacrificed for system power.

### Post by thomasaaron on 2010-07-12
cprofitt, I think you may have a previous model Serval, the one in which the battery attaches to the back of the machine, allowing for multiple battery sizes.

In the current model, it is molded into the *bottom* of the machine, so it isn't possible to add a larger battery. There is only one size, an 11.1V 3800mAh 41.18WH Li-Polymer Pack. It doesn't have any designation on it as to how many cells it has. I've attached a pic of the battery for everyone's examination.
