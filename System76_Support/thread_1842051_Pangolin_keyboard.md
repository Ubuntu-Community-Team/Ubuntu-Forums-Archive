### Post by jml on 2011-09-10
As Dm stated the first place to go with a System 76 hardware question would be the System76 Knowledge Base.  Here is a link to the Pangolin Performance section:

[http://knowledge76.com/index.php/PanP8](http://knowledge76.com/index.php/PanP8)

And the home page to the Knowledge base:

[http://knowledge76.com/index.php/Main_Page](http://knowledge76.com/index.php/Main_Page)

But to give a partial answer.  The top row of keys labeled F1 through F12 are function keys.  They operate by first depressing and keeping depressed the key labeled Fn (bottom row, second key from the left.  Pressing F3 will mute your speakers.  Pressing it again will un-mute them.F5 gradually reduces volume.  F6 increases volume.  F8 reduces screen brightness. F9 increases screen brightness.  F11 will both turn the wireless wifi off and then on.

If you are totally new to Ubuntu you may want to review some on-line references.  A good place to start would be Ubuntu's support page.  I have included the link to that site below:

[http://www.ubuntu.com/support](http://www.ubuntu.com/support)

Hope that this helps and welcome to Linux and this forum.
