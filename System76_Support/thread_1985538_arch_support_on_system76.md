### Post by jaylittle on 2012-05-29
Arch works fine on my panp9... however I had some trouble finding a working installation ISO image that would provide wireless support out of the box.  The current install image from August of last year doesn't work with the wireless hardware in my panp9.  The latest nightly builds of the Arch installation ISO have a nasty bug in the installer that keeps you from setting up your disks correctly, thus preventing you from being able to install Arch at all.  The work around I used to get Arch on my panp9 was to grab ArchBang (which is a great way of getting a quick GUI based Arch install up in a hurry) which simplifies the installation process, didn't have this bug but correctly supported all of my hardware out of the box.
