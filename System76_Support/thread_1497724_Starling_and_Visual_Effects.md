### Post by dv3500ea on 2010-05-31
I don't think you can run any visual effects because you have a netbook with only a low powered GPU. However, I don't think visual effects need to be enabled to get a transparent terminal. In gnome-terminal you can go to Edit -> Profile Preferences, click on the background tab and there should be an option for transparency.
