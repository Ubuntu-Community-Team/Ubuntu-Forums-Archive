### Post by thomasaaron on 2007-07-26
> 1) Can the line of Sytem76 Laptops run WindowsXP? (This is a bad q but i'm forced to for some things i love Linux dearly)

Yes. You can dual boot it or run it with VMware.

> 2) Can they run KDE variant of Ubuntu?

Yes. However, we officially only support Ubuntu. Our System 76 driver, however, will run on Kubuntu. If you run into KDE-specific problems, we may not be able to help you through them.
> 
3) Do they run "Out of the box" Meaning no drivers? If so do they come with drivers or a specially modified vers of Ubuntu?

Comes with Vanilla Ubuntu plus a few apps our target audience likes (GnuCash, Beagle, etc...)
Also comes with our System76 driver, which patches areas Ubuntu doesn't natively cover.

> 
4) Do they all come with gfx cards..are they "capable" of running games eg WoW etc. and running apps that require 3d accel like Google earth..

You can see the card options for each individual computer on our website.
They all run Beryl and GoogleEarth fine. I'm not a gamer, but some of our customers are, and they've reported good results. The Serval has nVidia and 3D acceleration. The new Darter and Pangolin have Intel Integrated graphics with full 3-D acceleration.


> 5) How is Bluetooth work on the laptops? - Ive had problems with my USB adapter on my desktop is it easy to get that working?

It takes a little work. It is supported, though. Again, see the website for the computers with supported bluetooth.

> 6) WiFi, External Antenna port? If not can it run a card with a wifi antenna port, Does it support Wireless N? (stupid q..)

It has built in wifi (no eternal antenna ports). It does not support wireless N. I've heard (knock on wood) that will be coming with Gutsy (7.10)

> 7) For people who have these laptops, How is the keyboard is it well spaced easy to use? anything would be helpful - thx

I work all day long on a Darter, and I've never had issues with the keyboard. But I guess folks with big thumbs might notice.

> Does the WiFi work to connect? Back a while i had auditor/PCLOS and Ubuntu on a laptop and tried to access an encrypted wifi network (my own) but it would not accept the key, has this been fixed does wifi work easily?

Network Manager works fine with all common network set-ups. DHCP, Static IP, WEP and WPA codes, Shared and Open Networks. My guess would be that it will work with just about anything else too, but it might take some configuration know-how.
