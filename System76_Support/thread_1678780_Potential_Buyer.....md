### Post by isantop on 2011-01-31
Thanks for your questions! My answers are below:

1. Does the smaller screen size hamper these task for long periods? 

IS: It will depend on your frame of reference. Coming from an 11.6" screen, you'll probably be fine, especially since Ubuntu Netbook is better optimized to fit on smaller screens then Mac OS X is.

2. Does System76 regularly support newer versions of Ubuntu on older generation systems?

IS: Yes. When a new version of ubuntu is released, we update our driver application and release a new version with support for the new version of Ubuntu. Once we do, simply upgrade and run the new driver.

Does that include the ability to upgrade to a newer Ubuntu version or am I only supported using the version that initially ships with the laptop?

IS: No, I'd actually encourage you to run the latest version of Ubuntu available, as it's easier for us to support it. 

3. Does the Starling NetBook have separate left and right touchpad buttons? It's hard to tell from the photos available on the website.

4. If any System76 representatives stumble across this, is there an updated date in which the Starling NetBook will be back in stock?

IS: Unfortunately, early February is still our best estimate.
