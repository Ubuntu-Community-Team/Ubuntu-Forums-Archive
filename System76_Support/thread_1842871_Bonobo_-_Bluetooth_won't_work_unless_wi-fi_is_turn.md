### Post by isantop on 2011-09-14
On the current Bonobo, the WiFi and Bluetooth modules are on the same card. If you need bluetooth, but don't want to be connected to WiFi, I suggest using the network menu to disconnect from the WiFi access point, then start using bluetooth.
