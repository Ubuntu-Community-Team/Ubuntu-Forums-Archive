### Post by lukeiamyourfather on 2015-03-04
That's for a 2.5" SATA drive which is very common. There are two heights of 2.5" SATA drives (9.5mm and 12.5mm). Most drives are 9.5mm height and some of the larger capacity drives are 12.5mm height. The documentation for the machine will tell you for sure if it can accommodate a 12.5mm height drive. If you're not sure or don't want to look it up get a drive that's 9.5mm height to be on the safe side (the 9.5mm height drives will work in either size drive bay).
