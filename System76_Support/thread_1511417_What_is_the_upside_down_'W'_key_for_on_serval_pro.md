### Post by isantop on 2010-06-17
It is indeed an "M". With a special font. The M stands for Mode, or silent mode, as others have guessed.

A good number of our laptops support silent mode. When pressed, the CPU will slow down a bit and the fan will scale back, so that it is not as loud. You can test this by going to a quiet room and pressing it. Of course, if the CPU is cool and the fan is already as low as it will go, it won't scale it back further, but it will keep it from getting faster.

I use is if I'm in class or a meeting, and I don't want to be a disruption or distraction because my CPU fan gets loud.
