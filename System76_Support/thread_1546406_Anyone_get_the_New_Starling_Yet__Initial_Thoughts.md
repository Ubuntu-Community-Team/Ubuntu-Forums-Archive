### Post by isantop on 2010-08-05
Well, they just started shipping today. I don't think we offer teleportation shipping yet. We'll definitely look into it! :p

We've done testing in house. The wireless performance is definitely better than the previous Starling. We've tested out to a range of 120 feet, the the signal doesn't drop off to much. That is, by the way, with walls in between the computer and router.

We don't have conclusive numbers on battery life, as we haven't yet run the battery down, but we appear to be looking at about four hours with the 3-cell battery. The 6-cell battery should let this system run a full, 8-hour day on one charge. We'll have more conclusive number available next week.
