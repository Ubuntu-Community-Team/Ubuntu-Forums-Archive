### Post by HotShotDJ on 2009-11-07
The best thing for you to do is download the installation CD and then run it in LiveCD mode.  Now you can test most things to see if there are any issues -- if you are happy with the way your hardware is working in a live session, go ahead and install Karmic (I recommend a clean install), but if you run into problems, then wait.

My experience has been that a majority of problems have been occuring when people do blind updates -- no testing, not reading the forums, not checking for known issues with the new version.  But even with testing, nothing is ever guaranteed.  I always make a full backup (using rsync for my /home partition and partimage (from a liveCD) for my root (/) partition) -- that way I can always fully restore my system if something goes wrong.
