### Post by perce on 2007-09-21
I have a white Darter too. At first I was disappointed, because it was shipped with several software problems. However Tom was very helpful and efficient in helping me to fix them, and now it works perfectly (hibernate, suspend, hotkeys, everything). Beryl works very smoothly on he intel card. For the hardware consider that System76 uses the same barebones as Asus, which is a respected company. At a first sight it doesn't look as solid as my office mate Thinkpad, but it's lighter, and it survived a fall of more than one meter open and turned on with only a minor damage at the case. I alse always carry it in a bag together with books and other stuff and it didn't suffer from that. The battery was supposed to last 4.5 hours, but I got slightly more that 3.5. I don't think they cheated in he website, so if you're interested in a long time you should ask Tom what configuration they used to test it. I don't like the keybord , but other find it fine. So I wouldn't worry about the hardware, unless you have some very specific needs.

### Post by octathlon on 2007-09-22
I'm very happy with my Pangolin Value.  It's fast and quiet. The display is very clear and sharp, much easier on my eyes than my old notebook is.  I've only had it about a month so I haven't even tested everything out yet; for example I haven't tried to burn a  dvd yet.  The 3D "desktop effects" seem to work fine, but I have only turned them on a few times to show them to people then turned them right back off :). 

The only negative I have noticed is that I think the wireless card isn't as good as the one I have in the old notebook.  Don't know if it's the card or the driver or what, but it does not report as strong a signal, and it drops the connection several times a day, taking 5-15 seconds to reconnect itself, whereas on the old notebook this never happens no matter if I'm running windows or any of the various Linux live CD's I run on it.

I'm not crazy about the / partition being set up as a LVM because I had originally planned to shrink it and create a separate partition for /home, and I think the only way I can do that in this case is to reformat and reinstall. 

Still like I say I am more than satisfied with the laptop and System 76.  I think it's one of the best deals out there and you won't be disappointed.

### Post by Zxaos on 2007-09-23
Just got my pan value about a month ago, only a few issues so far.

The negatives:
When I originally ordered the laptop, I was told it would ship in 7 business days if I recall correctly.  I had heard nothing after the 9th day, so I contacted them, and was told there had been a delay and that it should ship shortly. I received it a few days after that.

When I received the laptop and powered it on, there was not the initial setup screens as indicated in the provided documentation - in fact, I couldn't do anything or log on at all. I had to resort to rebooting in recovery mode to change the password for the oem user and then log on using that to create my main user. Not major, but would have been an issue if I wasn't familiar with how to do it.

The screen on the pangolin value is a little chunky, there's some banding on gradients. However, it's my understanding this is a driver issue and should be fixed for Gutsy in October.


The positives:
Once I actually got into the machine, everything worked right off the bat. Wireless worked perfectly, printers easy to install, desktop effects/compiz working as expected.

The battery life is quite good. I have the 9 cell battery, and it lasts for at least four hours through all my classes.

Suspend worked perfectly out of the box, although I'm still working through a list of possible solutions to deal with wireless not working after hibernate.

Discounting the banding, the screen looks quite good. Nice and bright and clear.

The hotkeys above the keyboard worked correctly with no configuration


Comments:

The chasis of the machine *feels* cheap, but I don't think it actually is - I think it's a matter of the machine being a little lighter at the front combined with the texture of the plastic.


Overall, I'm very happy with the laptop.  The experience wasn't perfect, but it was pretty good, and I'm quite happy with the performance of the machine.
