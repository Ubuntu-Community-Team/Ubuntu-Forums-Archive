### Post by CrimsonS on 2009-11-09
1. The keyboard is 92% of a normal keyboard size. Being a student myself, I use it all the time for my homework an this is one of the main reasons I picked one of the larger netbooks.

2. As for the processor, it's an atom n270, 1.6 Ghz. But it feels much faster probably due to HyperThreading (the OS 'sees' 2 cores instead of one, so it can share the cpu load on 2 virtual processors). This is probably standard for netbooks, and I feel it's more than enough. I get 30 second boot time with ubuntu 9.10 (I think the boot time was actually better with 9.04 though maybe 22-25 seconds, but this is supposed to be fixed with later releases)

3. I don't play games much, but I haven't had much luck running WoG with integrated graphic cards on my desktop. You could probably play 2D games without any problems though. I can enable compiz effects on my netbook without slowing it down too much.
