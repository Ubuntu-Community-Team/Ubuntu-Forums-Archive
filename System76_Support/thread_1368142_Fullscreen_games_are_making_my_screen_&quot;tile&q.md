### Post by UberElvis on 2009-12-30
I got my new Pangolin Performance laptop almost a week ago.  I love pretty much everything about it.  However, I'm running into a problem with fullscreen games that has not happened on any other computer I've used Ubuntu on (this is my first System76 computer).  I've verified that fullscreen flash works fine (such as youtube), but when I tried World of Warcraft on wine the game loads but the screen "tiles" basically copies itself and appears in two columns and five rows, if you know what I mean.  I verified that this is not just wine, because I tried SuperTux, and the same thing happened.  I also tried SuperTuxKart, and since it loaded in a window instead of fullscreen, this did not happen.

The model number is panp6 and I'm using the preinstalled Ubuntu 9.10.

I really would like to be able to run a few games, and I think this has something to do with the particular configuration of this laptop, so I could use some help.

Thanks,
UberElvis

### Post by UberElvis on 2009-12-30
I figured it out, it turns out that it does that when the resolution is changed, I just had to set the WoW resolution to the regular screen resolution.  Hope this helps people in the future.
