
### Post by crichell on 2007-01-24
Hi Craig - you must have Intel graphics instead of nVidia.  This is an easy fix.  Boot the machine and press Escape when you see the GRUB boot loader.  Choose "Recover Mode" - this will take you to a terminal.

```
sudo nano /etc/X11/xorg.conf
```

Change the following:

```
Section "Device"
	Identifier	"nVidia Graphics Controller"
	Driver		"nvidia"
	Option		"NoLogo"
EndSection
```

To

```
Section "Device"
	Identifier	"nVidia Graphics Controller"
	Driver		"i810"
EndSection
```
