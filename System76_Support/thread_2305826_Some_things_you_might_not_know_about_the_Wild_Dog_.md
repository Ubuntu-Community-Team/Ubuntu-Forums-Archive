### Post by MechaMechanism on 2015-12-09
So I have been asking System76 a bunch of questions about the Wild Dog Pro.  Naturally this is about my system.  I encourage you to add to it.

1.  If you ordered the wifi option then the connectors will be SMA and there are 2 of them.

2.  The blue usb ports are 3.0 type A and the red usb ports are 3.1 type A and the black usb ports are 2.0 type A and the black mini usb port is 3.1 type c.

3.  The M.2 SSD hard drive option for the 120 GB drive for November early December is Samsung EVO 850 M.2 SATA SSD - MZ-N5E120BW

4.  Auto trim is enabled for Ubuntu 15.10.

5.  Bios clock is set to UTC time by default.

6.  TPM module is turned off as it is not compatible with Ubuntu 15.10.

7. Intel virtual machine tech is turned on.

8.  The Intel HD 530 graphics option has 1 DVI-D dual link output and 1 VGA output and 1 HDMI 1.4 output.

9.  The optional memory card reader is connected to motherboard via USB 3.0.  The reader has 1 external blue usb 3.0 type A port.  The reader is black and clashes with the silver color of my WDP.

10.  There is only 1 ps2 port.

11.  One headphone port and one microphone port on the front.

12.  The WDP is very light for it's bulk.

That's all for now.  If I get more information I'll update this list.

Please add your own!
