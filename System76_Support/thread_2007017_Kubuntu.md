### Post by joe4ska on 2012-06-20
The warranty will not be voided if you install another GNU/Linux Distribution.

[One of their employees answered this question in an old thread]("http://ubuntuforums.org/showpost.php?p=11603940&postcount=4")

however if you ask them directly at [email]sales@system76.com[/email] or register on their website and ask the question there they will confirm your question directly.

As mentioned in the thread you can install any distribution but they won't provide support if something doesn't work. However, it's very likely you can find the answers in these forums if you do install Kubuntu and run into some kind of glitch. I'm willing to bet someone else has already solved the same issue previously.
