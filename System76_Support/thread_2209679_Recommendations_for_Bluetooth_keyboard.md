Price difference is negligible, if there is any.

### Post by jaylittle on 2014-03-09
I'd recommend you buy a Logitech wireless keyboard that doesn't use bluetooth.  Why?  Because wireless devices using Logitech's dongles and associated protocol work at a much lower level than bluetooth does.  This means that your host machine will treat the wireless keyboard as if it was a USB keyboard even before the machine has booted Linux (or any other OS).
