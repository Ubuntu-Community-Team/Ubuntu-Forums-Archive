### Post by sgtron on 2007-06-07
SOLVED!!! 

I was lacking the symbolic link from my java plugin to the firefox plugins as described in this page:
[http://www.mozilla.org/support/firefox/faq.html#q2.2](http://www.mozilla.org/support/firefox/faq.html#q2.2)

Not exactly the same as this page but it was enough to get me the answer I needed.
