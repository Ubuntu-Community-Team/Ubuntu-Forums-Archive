### Post by isantop on 2010-06-30
This could be any number of things.

First off is to ask if the computer's original memory still works in the computer.

Next, can you post what the labels on the original modules say?

Also, when working with memory, always amke sure to remove the battery and unplug the computer. As an added safety measure (The computer's, not mine), I press and hold the power button for a few moments to drain any residual charge from the capacitors. It may also help to touch an external metallic object touching the floor and a piece of the computer frame.. All of this may seem redundant, but it helps to ensure you have no static electricity buildup, which can fry DIMMs.
