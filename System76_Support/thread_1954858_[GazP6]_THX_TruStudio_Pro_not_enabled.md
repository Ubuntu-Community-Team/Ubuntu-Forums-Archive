### Post by isantop on 2012-04-10
In order for the System76 Driver to take effect, you have to run it from the System > Administration menu. However, the driver will only run on an OS that identifies as Ubuntu. 

That said, support for the Audio in the Gazelle is built-in to Ubuntu, and we don't make any changes in the driver for that to work.
