### Post by greg_g on 2007-06-20
Well, it looks like 2048x1536.

I got there by this process which I am providing just so you can double check that it makes sense:

System76's product description says "Graphics: Intel 945 GM Integrated Graphics", so I googled "Intel 945 GM Integrated Graphics"

Got this page:
[http://www.intel.com/products/chipsets/945pm/index.htm](http://www.intel.com/products/chipsets/945pm/index.htm)

Got this pdf:
[http://download.intel.com/design/mobile/datashts/30921904.pdf](http://download.intel.com/design/mobile/datashts/30921904.pdf)  (3+ megs)

Page 26 says:
        LVDS Interface
         &#8226; Dual-channels LVDS interface support: 2 x 18 bpp panel support up to QXGA
           (2048 x 1536)
        Analog CRT
         &#8226; Analog Monitor Support up to QXGA

And for more information on what QXGA means:
[http://en.wikipedia.org/wiki/QXGA](http://en.wikipedia.org/wiki/QXGA)

There ya go.  

greg

EDIT: ok, so the first link google produced, the one I went to and posted here is for the 945PM, not GM.  So I went to Intel's 945GM page.
