### Post by Newbunto on 2014-04-25
I've done every upgrade from 11.10 -> 12.04 -> 12.10 -> 13.04 -> 13.10 -> 14.04 without any hardware support issues and without any showstopper breakages. I wouldn't get too stressed about upgrading unless you have particularly old hardware.

As others have said, if you are concerned:

* always do a backup first

* test using a live USB/DVD

Edit: The only significant breakage that I have had, which is somewhere between a minor annoyance and a showstopper, is Google Earth, which broke somewhere in that sequence of upgrades and remains broken.
