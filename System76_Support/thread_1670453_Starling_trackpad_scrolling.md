### Post by isantop on 2011-01-21
Hi.

The scrolling on the Starling is unusual, but it is there. To scroll up, just touch the upper-right corner of the trackpad. To scroll down, touch the lower-right corner. 

There is a utility you can use to set up the pad. I'm not sure if it supports two-finger scrolling, but you can set other options. Instructions are here:

[http://knowledge76.com/index.php/Star3](http://knowledge76.com/index.php/Star3)
