### Post by sudodus on 2014-05-12
Ubuntu is portable between computers unless you have installed or need proprietary drivers (typically for graphics and wifi).

Windows is *not* portable, so do not expect to be able to run it, if you move the drive to another computer. Furthermore, Windows is made to refuse booting from USB.
