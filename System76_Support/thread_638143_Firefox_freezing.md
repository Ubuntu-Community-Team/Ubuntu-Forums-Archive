### Post by thomasaaron on 2007-12-12
There was a cd drive firmware problem that caused freezing. A temporary fix was to keep a cd in the drive. Later a permanent solution was issued -- which was to flash the firmware in the drive. Did you ever do this.

Not EVERYBODY had this problem, though. If Only firefox is freezing, that would not be indicative of this problem. If everything is freezing periodically, that WOULD be indicative of this problem.  Are you sure it is only firefox freezing?

Here is a good thread on installing flash. If the problem is not the firmware, then this should get your flash working correctly:
