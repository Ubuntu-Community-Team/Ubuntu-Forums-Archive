### Post by bill516 on 2009-04-19
1. I'm assuming that all the hardware on the Pangolin is fully supported by Ubuntu through a special system76 driver set. Is there any circumstance where that is not the case?

The hardware on this machine, which I have had for a bit more than a week, is fully supported.  Swap out various parts and it makes sense that you might be on your own, depending on what you do.  But with a laptop I assume that is unlikely.  In any event System 76 can answer specific questions.

&#65279;2. When I upgrade Ubuntu releases (say, from 8.10 to 9.04), will there always be an upgraded driver set for the specific version quickly available, or does it work with all versions? In other words, will my Pangolin still be able to work fully if I upgrade Ubuntu releases? If yes, how long before updated drivers become available?

I asked this same question before I purchased.  The answer was that S-76 releases a new set of drivers when a new Ubuntu release is finalized.  If we upgrade the new drivers will be available to us.  Note we are talking about final releases here, not betas or release candidates.  Note also that we are talking about the 64 bit version of Ubuntu and not 32 bits.  Seems fair enough.

3.  &#65279;The fingerprint reader is in (beta). Is that regarding the hardware, or the software required to run it?

Can't answer, have not tried it.

&#65279; 4.  How long will the 6-cell lithium ion battery last with a full charge, with Pangolin's specific hardware?

Depends somewhat on what peripherals you order and how you use your machine.  I run right at 2 hours.  Frankly this is not the best machine if you want to do most of your work for extended time away from a line power source.  No larger battery appears to be available despite mention of such a thing in the user's manual.

&#65279;5.  What's the difference between a core duo and a core 2 duo processor (besides the price)? Faster? More efficient? 

Depends in part on which cpu we compare but Core 2 duo's are newer and they may consume less wattage.  Copious information about all of these is available on the Intel site.  In my experience trust that before anything else.  My previous machine ran a T7200, this Pangolin runs a T6400.  I can discern no operational difference at all, frankly.

&#65279;6. Keeping my options open, if I choose to dual-boot with Windows XP (for school reasons =/) will all the hardware still work (i.e. are there generic drivers available)?

The Pangolin manual makes it clear that this machine frequently ships from its various vendors with Microsoft Vista.  That being the case I would not expect problems.  System 76 has a very nice HowTo on adding Windows should you choose to do that.  This is a pretty straight-forward, plain vanilla machine, really.  I would be most surprised if windows lacked drivers for something.

If you like, travel over to the Extreme Tech website and you will see a "first impressions" report I posted there on the Pangolin.  User name there is julian516.  Look in the section dealing with Mac and Linux questions.  There is also at least one quite good review on this site.
