### Post by isantop on 2012-01-03
In 11.10, you will only need to download the latest System76 Driver package available here: [http://planet76.com](http://planet76.com). Simply download and install the package, then run the installed "System76 Driver" application, and use the restore function to bring your system up to factory spec.

10.04 is not supported on the BonP5 though. The BonP5 requires software support for it's Intel Sandy Bridge processor, and this is not available for 10.04. You need to run 10.10 or later.
