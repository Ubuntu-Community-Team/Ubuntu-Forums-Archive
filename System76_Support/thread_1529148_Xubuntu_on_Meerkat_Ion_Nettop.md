### Post by thomasaaron on 2010-07-12
> 1) Will I have to worry about compatibility of regular Ubuntu applications? They should work the same on Xubuntu correct? From what I gather, Xubuntu is just a different interface?

There are some library differences, I'm sure. But, basically, it's about the same under the hood. Not sure what office software Xubuntu uses, but if it's not Open Office, you might run into some compatibility issues. Hopefully some Xubunut experts will chime in.

> 2) Will I have to worry about the System76 drivers? What will I need to do as far as drivers go? Do I just add the System76 PPA to the Software Sources? Or should I not use those since it's not Ubuntu?

I wouldn't worry about it. The ION is completely supported by Ubuntu without the System76 Driver. The only problem I can think of with Xubuntu might be enabling the nVidia driver for accelerated graphics. Not sure how to do that in Xubuntu. In Ubuntu, you go to System > Admin > Hardware Drivers and enable it. I'm sure Xubuntu has an equivalent.
