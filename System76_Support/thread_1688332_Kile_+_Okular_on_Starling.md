I've never used a Starling (I hadn't heard of it until 5 seconds ago :-) but I did run LaTeX on my Nokia N800 PDA and it could typeset 300pp of thesis in 15 sec compared with about 3 sec on my Dell Latitude D610. That was editing with Emacs (yes, really Emacs on an N800), not Kile, and displaying with the default Maemo PDF reader. A modern netbook should be many times faster than the N800; probably faster than this old laptop here.

(Okular sucks little black toads, and it's missing some key critical features that were in KPDF and KDVI, like Print Current Page. Plus it crashes, blacks out, and freezes. Really not ready for distribution and should never have been let loose in its current state. Check if you can retro-install KPDF and/or KDVI.)

But I can't see any reason why LaTeX shouldn't work happily at full speed with Kile on any netbook, unless the netbook has been seriously crippled.

If it's running Ubuntu, you should be able to install texlive-full from the application manager.
