### Post by EuclideanCoffee on 2020-03-23
Essentially this works below your operating system. Without programming or flashing the drive, you won't need to learn anything new.[B]

coreboot[/B], formerly known as **LinuxBIOS**,[SUP][[4]]("https://en.wikipedia.org/wiki/Coreboot#cite_note-4")[/SUP] is a software project aimed at replacing proprietary [firmware]("https://en.wikipedia.org/wiki/Firmware") ([BIOS]("https://en.wikipedia.org/wiki/BIOS") or [UEFI]("https://en.wikipedia.org/wiki/UEFI"))  found in most computers with a lightweight firmware designed to perform  only the minimum number of tasks necessary to load and run a modern [32-bit]("https://en.wikipedia.org/wiki/32-bit") or [64-bit]("https://en.wikipedia.org/wiki/64-bit") [operating system]("https://en.wikipedia.org/wiki/Operating_system"). [https://en.wikipedia.org/wiki/Coreboot](https://en.wikipedia.org/wiki/Coreboot)

Coreboot is an extended firmware platform that delivers a lightning fast  and secure boot experience on modern computers and embedded systems. As  an Open Source project it provides auditability and maximum control  over technology.
