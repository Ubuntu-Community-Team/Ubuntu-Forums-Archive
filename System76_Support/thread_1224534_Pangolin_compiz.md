### Post by 3Miro on 2009-07-27
You have everything installed and compiz is running, you just need the manager that would let you set up things your way.

1. System -> Administration -> Synaptic Package Manager
2. Enter Password
3. Search for compiz and look for compizconfig-setting-manager then install it.

Then you go to:
System -> Preferences -> CompizConfig Setting Manager

Enjoy the tons of options!
