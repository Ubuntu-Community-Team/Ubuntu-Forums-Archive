### Post by gpstar on 2009-05-21
> The "amd64" distros are for any 64-bit processor (Intel included). AMD was first to the table with the 64-bit instruction set extension for x86, and thus most 64-bit linux distros are called "amd64". Intel's em64t instruction set is amd64 compatible, so an amd64 distro works with all 64-bit x86 processors.
