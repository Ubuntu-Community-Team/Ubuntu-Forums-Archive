### Post by isantop on 2010-12-22
As long as we can still track down the charger, we can replace it for you. If they stop making the charger, then you're probably out of luck, but the ones we use now are pretty standard, so they should be good for a long time.

Furthermore, we don't use proprietary connectors, so if you can find one with the same output and plug, it should work.
