### Post by isantop on 2012-04-12
> **Rastlinux said:**
> So the questions are how does the laptop behave when hooking it up to a projector for presentations?
Also, how about using it for a multi-display? 

They work very well. The Intel graphics in the Lemur are very flexible and well supported. Both of these will be handled pretty much identically.

I should note that the system supports up to one external monitor. You can use it in conjunction with the built-in display for multi-monitor setups, or on its own, but you can't set the computer up with two external monitors.

> **Rastlinux said:**
> Is this something I have to plugin and boot and configure or is it more automated and will recognize the displays when plugged in and adjust accordingly?

You'll need to set it up (Resolution, etc.) But Ubuntu makes this extremely easy. You won't need to reboot, log out and back in, or anything else.

> **Rastlinux said:**
> Is there additional hardware like a docking station that can be purchased that is not shown on the site?

No; we can only offer the options and accessories listed on our website.
