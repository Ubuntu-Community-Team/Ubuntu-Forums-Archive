### Post by thomasaaron on 2008-02-08
Hi, Squish.

It is 4GB of Ram. The problem is that 32 bit Ubuntu can support a maximum of 4GB of Ram. However, that includes other mapped memory installed in the machine. So, after the other devices are deducted from the 4GB total, Ubuntu can only use the 3.* GB of installed RAM.

If you install 64-bit Ubuntu, you'll have all of it.
