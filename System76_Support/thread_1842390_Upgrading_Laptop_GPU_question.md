### Post by 2F4U on 2011-09-11
I don't know these partitcular laptop models but in general, RAM and hdd upgrades are relatively easy. What is important with respect to RAM is how may RAM sticks can be installed and how many are currently used. Most laptops provide two slots, but some only provide one. If there is no slot free, you have to replace all the existing RAM if you want to upgrade.
