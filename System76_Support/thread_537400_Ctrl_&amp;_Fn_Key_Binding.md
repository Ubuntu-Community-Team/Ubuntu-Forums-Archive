### Post by thomasaaron on 2007-08-29
I'm sure it can, somehow. But I've messed around a bit with keybindings before and found them very confusing.

You would think that it would be as simple as switching a couple of keycodes in a file somewhere, but my experience is that every key combination that is used either by the OS or software applications is designated individually. When you multiply that by the fact that there are several combinations that include the Fn key, many more that include the Ctrl key, and then a bunch that include Fn+Ctrl plus some other key, the whole idea of reversing the keys begins to sound a bit nightmarish.

So, while I hate to be the nay-sayer, I would rather just get the hang of the reversed key positions. But that's just me. :lolflag: 

But, again, that's the beauty of linux. Worse case scenario, if you hose it, you can always fix it somehow for free.
