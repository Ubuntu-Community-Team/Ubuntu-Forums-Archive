### Post by darkapec on 2012-08-18
If you have ubuntu installed run 

```
sudo lshw
```

and post the results here, if your on windows run CPU-Z, I will need to know your motherboard in order to tell you what kind of ram and maximum capacity. I would assume either DDR or SDRAM if DDR the maximum is either 1gb-4gb and if SDRAM the maximum is 512mb-2gb (this is generally speaking of course) I will need motherboard model number to tell you exactly.
