### Post by joe4ska on 2012-06-11
I have a few questions and suggestions.

Try logging into a Guest or as another user to see if maybe the issue could be related to your user account's  customized settings?

If the problem still persists

Try installing Xubuntu or Lubuntu from the software center. Log into either environment and see if the problem continues.

If the problem persists try reinstalling Ubuntu (after backing up your music, videos, documents, picture etc of course)

If a clean install doesn't fix the problem It could be hardware related. Perhaps your FN key is sticking?

Hope this helps.
