Prior to ordering the system, the only use for a caddy case is to have a hard drive in it, and since we can only sell a complete, fully functional system, we only offer it pre-populated with hard drives. Afterwards, someone may want to add a hard drive instead of the optical drive; hence, the caddy case alone.
