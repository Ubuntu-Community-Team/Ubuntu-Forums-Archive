### Post by jml on 2007-11-14
I would suggest that you download the CD version of Fedora 8.  Its a combination Live CD and install CD.  That way, you can test to see what works and what does not work.  If it works well, you can either install that version and add additional aps with the add remove program utility, or you can then download the DVD version and do the install with that.  I tried Fedora *.  Its a reasonably well built distro.  Wireless can be a bit of a pain.  You might want to check out the review on this week's Distrowatch Weekly News, here:

[http://distrowatch.com/weekly.php?issue=20071112#review](http://distrowatch.com/weekly.php?issue=20071112#review)
