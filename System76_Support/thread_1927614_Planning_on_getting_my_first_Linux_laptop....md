I can answer everything except for the backlighting on the keyboard. The only backlit keyboard I have, is my old trusty i-rocks keyboard for my System76 desktop PC. I own two System76 PCs. One is running Ubuntu (laptop), the other is running Linux Mint 11 (desktop). 

Out of the box, it's going to be an Intel system. Desktops will come with either NVIDIA or ATI video cards. I think they're back with NVIDIA now days.  Laptops will present Intel setups down to the wireless chipset. In the end, the components are selected due to them being Linux compatible. You'll do fine with Mint

A PC purchased from System76 will come with Ubuntu 64bit. To the common user, this means you can add a LOT more RAM and with some select applications, some performance is a "little better" than it would with 32bit Ubuntu. These days, Adobe Flash has a 64bit Linux compatible release and most apps will work flawlessly.

Should you opt to go Mint 12 or something else, you should have no issues. The famed [custom BIOS]("http://ubuntuforums.org/showpost.php?p=11696793&postcount=6") are more for show based on my experience. Yes, there are some custom tweaks I've heard. But it shouldn't present you with any issues using other distros specifically. Think of it as OEM branding, with a dash of secret sauce. ;) The cool thing about Mint is that you can use the same Debian packages you used with Ubuntu. There are some differences, but Mint is based heavily on Ubuntu. And any firmware tweaks made by System76 are likely going to flow over nicely to Mint (speculating based on experiences thus far).

As for the touchpad, yes, it should work fine out of the box as described. But if you're running Ubuntu or Mint, get to know the[ Jupiter Applet]("http://www.jupiterapplet.org/downloads.html"). While the new 3.x kernel has improved power consumption, this applet will be your best friend on a new laptop. Disable/enable your touchpad easily, same with wifi, bluetooth, video display handling, and so on. It's really handy.
