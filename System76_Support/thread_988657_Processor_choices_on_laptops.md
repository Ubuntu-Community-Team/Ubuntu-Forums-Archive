### Post by ResumeMan on 2008-11-20
I'm not a computer expert, but from my recent research while planning my Pangolin purchase I have a few things to contribute.


> **bakelitedoorbell said:**
> I understand that faster is better, but there are other factors.  I think there might be a large performance jump when going from FSB_3MB to FSB_6MB (is that a ram buffer size?).

Yes, that's correct. FSB stands for "front side bus" which is essentially the interface between the CPU and the RAM. The higher FSB speed helps eliminate bottlenecks and speeds up overall performance.

> **bakelitedoorbell said:**
> What is the difference between the 35 Watt and 25 Watt choices?  Does the 25 Watt use less energy and therefore give the computer more battery life?

Precisely. And -- this is just a guess -- I would suspect that it'd run cooler too.

### Post by drewbenn on 2008-11-21
One post I read when getting ready to buy my current laptop (just over a year ago) recommended getting a slower processor, because it will run cooler, and because a top-of-the-line processor isn't as necessary in the Linux world as it is for running Windows.  I don't know if those statements are true or not, but I went ahead and got one of the slowest available processors (1.8GHz), but got the most RAM that was available and a 7200 RPM hard drive.  I don't run a lot of really demanding apps (I typically only have Firefox, Tellico, Moneydance, a couple terminals, maybe emacs, and a lightweight game running, and I'll compile a program once or twice a month), though.  But I am happy, my computer runs reasonably fast (I'd say its more responsive than my >2 GHz work PC running XP, although that computer has a lot more running on it), and stays pretty cool.  This is my first laptop, though, so I don't really have anything to compare those statements against.
Just my two cents.
