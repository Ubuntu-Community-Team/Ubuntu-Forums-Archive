### Post by crichell on 2007-01-09
> what is the speed difference between the celeron and the cheapest core 2 duo chip?

That's a good question but kind of hard to answer.  The machine is snappier with a faster processor.  Applications open faster - processing is faster and the core 2 duo has a much large cache so it has fewer data bottlenecks.

> Is the graphics performance (compiz, beryl maybe some 3d games) with the intel integrated graphics relatively good?

Beryl and Compiz run well for me (I have Intel graphics on my machine).  3D games depend on the game specifications.  Some simply require nVidia or ATI graphics to run properly.  Google Earth is pretty slow on the Intel graphics chip.  For the things I do it's all that I need (Probably the case for lots of folks.)
