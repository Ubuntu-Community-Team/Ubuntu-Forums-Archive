### Post by jml on 2007-02-12
I'm sure Carl can answer for himself, but System 76 does not officially support Windows on their products.  And what another poster said is correct, if you install Windows after a Linux install, Windows overwrites the boot loader and you will not have access to your Linux partition.  When I set up dual boot systems, I usually install Windows first, and then install Linux.  The only potential problem I see is that System 76 installs custom drivers so all of the laptop's hardware works out of the box.  With a standard Ubuntu install, you would not have those drivers.  Maybe Carl will have some other thoughts.
