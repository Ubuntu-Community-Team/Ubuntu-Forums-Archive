*Do not choose to auto-detect hardare/monitor
*Choose all available resolutions
*Select "intel" as the driver
*Choose "Advanced" when you get to it.
Everything else should be default

### Post by thomasaaron on 2007-10-02
The reason it won't work is because the 965 card is blacklisted by compiz:
[http://www.realistanew.com/2007/09/23/compiz-in-ubuntu-update/](http://www.realistanew.com/2007/09/23/compiz-in-ubuntu-update/)

The fix is to comment out the line that blacklists it:

sudo gedit /usr/bin/compiz

Then put a comment (#) in front of this line:

T="$T 8086:2982 8086:2992 8086:29a2 8086:2a02 8086:2a12"  # intel 965

Best,
Tom

BTW: I'm not sure why they blacklisted the card. But I've been running for a while now with it enabled.... No problems.
