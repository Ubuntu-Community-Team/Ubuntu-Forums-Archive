### Post by crichell on 2006-12-29
Hello Everyone -

Today we are releasing a new driver that adds support for the Gazelle Performance built in camera.  The new driver version is 1.9.7.  If you have a Gazelle Performance make sure to run the new System76 Driver (after it has been released of course).   The new camera driver applies to both the old silver model and the current black model.

System > Administration > System76 Driver Installer

We have also added a brief wiki article to our knowledge base.  It includes information about using your camera.

[http://knowledge76.com/index.php/Cameras](http://knowledge76.com/index.php/Cameras)
