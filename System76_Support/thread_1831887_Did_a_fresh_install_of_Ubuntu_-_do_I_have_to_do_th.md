### Post by isantop on 2011-08-25
Officially speaking, the only thing it does that "Install" doesn't is set up our repository so that the driver will be updated automatically in the future. I'd recommend it so that you don't have to manually download and install any new driver versions that get released, but it's not strictly necessary.
