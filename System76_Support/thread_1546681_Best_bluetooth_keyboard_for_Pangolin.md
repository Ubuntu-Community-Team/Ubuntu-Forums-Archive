### Post by HotShotDJ on 2010-08-06
Hello, William --

I use a Rocketfish multimedia keyboard/mouse ([http://hubpages.com/hub/Rocketfish_Bluetooth_Wireless_Keyboard_and_Mouse_Multimedia_Pack](http://hubpages.com/hub/Rocketfish_Bluetooth_Wireless_Keyboard_and_Mouse_Multimedia_Pack)). Everything, including the multimedia buttons, just worked out-of-the-box.  It comes with a blue-tooth dongle, but you won't need it with your Pangolin, since it has a built-in blue-tooth transceiver.  Both the keyboard and the mouse go into a battery saving mode after about 5 minutes of inactivity.  It can sometimes be a pain when you start typing and nothing happens.  The mouse reconnects very quickly.  The keyboard takes a bit longer.

Quite honestly, I don't think Linux-Compatibility is much of an issue any more... I have not yet found a blue-tooth device that doesn't work -- including a head set and 3 cell phones.  I think if you shop for a good price, any bluetooth keyboard will work for you.  And if you run into a VERY rare one that doesn't, you can always return it.
