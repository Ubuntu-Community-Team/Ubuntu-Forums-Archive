### Post by Noah0504 on 2010-09-03
As far as I know, this is a feature at the BIOS level of the computer.  There is no way to change it.  It is a little inconvenient, but I've learned to deal with it.  If you search the System76 section of the forums, you'll find a few other threads about this.

### Post by isantop on 2010-09-03
It's actually an effort to save on the battery. Bluetooth will drain your battery fast, so we disable it on boot up by default.
