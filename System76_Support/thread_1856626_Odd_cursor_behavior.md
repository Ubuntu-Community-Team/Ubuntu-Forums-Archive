### Post by isantop on 2011-10-10
Can you try pressing Fn+F1 to turn the touchpad off, then press it again to turn it on? That will power cycle the touchpad and could clear any odd issues.

Other than that, make sure that only one finger is touching the pad when you click. If there is anything else capacitive on the pad, it will not function correctly. This would include water, any electrical cords, etc.
