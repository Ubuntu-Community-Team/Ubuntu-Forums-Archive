### Post by Eldera on 2009-08-29
If it is just the S76 driver you are asking about, it is backwards compatible. If you are re-installing it, just use the latest version which is the second line from the bottom on this site:

[http://planet76.com/repositories/](http://planet76.com/repositories/)

More detail instructions here:[http://knowledge76.com/index.php/Restoring_Your_System](http://knowledge76.com/index.php/Restoring_Your_System)

>     *  Download the latest System76 Driver from this link 

[http://planet76.com/repositories/](http://planet76.com/repositories/)

    * Double click on the downloaded package to install
    * Go to System > Administration > System76 Driver
    * Choose the Restore tab and click Restore. Restore will take a few minutes. 

That's it - your system is fully restored!
