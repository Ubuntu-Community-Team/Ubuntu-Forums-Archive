### Post by HamletDRC on 2009-08-21
I'm running WIndows XP in VirtualBox. 
I'd like to run it full screen, which for my Pangolin is 1280 by 800. 
IN the WIndows VM display settings, there are only options for more common resolutions, meaning full screen isn't really full screen. 

Are there WIndows Drivers for the Pangolin that I can install to get to 1280 by 800 resolution?

---

### Post by HamletDRC on 2009-08-21
I figured this one out myself. 

WIthin the Virtual Box Window, with the VM running, go to the menu Devices -> Install Guest additions. Once this installs into the running windows VM, you reboot the Windows VM and the Ctrl+F makes full screen /real/ full screen.
