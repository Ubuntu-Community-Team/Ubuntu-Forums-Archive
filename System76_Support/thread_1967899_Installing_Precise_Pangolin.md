### Post by hkarl629 on 2012-04-28
I try to follow System 76's advice to wait awhile before upgrading to new versions of Ubuntu due to heavy server traffic.

Well last night my wife (78) was using our 9+ year old IBM Thinkpad when a window popped up asking if we wanted to upgrade to 12.04. My wife thought this was just an update so she clicked OK. It took a while but we ended up with 12.04 installed and the old IBM working flawlessly. All our files were intact. Didn't skip a beat.

This morning as I (82+) was working on my System 76 Serval 2 that same window popped up asking the same question. I said OK and away we went. It took some time, but the results were the same. Everything that was there when it started was still there at the ended and functioning correectly. I went through the drill to redo the System 76 Drivers. Even before I did that eveerthing I did worked; so I asked, "What did those drivers do?" "Was it necessary?"

Anyway, we are happy that the unexpected upgrade to 12.04 worked out so well.  I would say it was easier than my upgrading my HP laptop from Vista to Windows 7.

Love Ubuntu.

hkarl629
