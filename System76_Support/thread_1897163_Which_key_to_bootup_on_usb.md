### Post by isantop on 2011-12-19
You can access the boot menu on a Starling by pressing F1 during the System76 splash screen. You may need to press it repeatedly, but once you do, you will see a menu where you can select which device to boot from. This is a one-time override, so it will switch back to default once it reboots.
