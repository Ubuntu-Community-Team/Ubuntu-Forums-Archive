### Post by Dave_L on 2010-10-13
I currently have a "wired" router that I use with a desktop computer (not System76). The incoming connection is DSL.

I'm awaiting delivery of a Starling Netbook, whose features include 802.11 b/g/n wireless support.

I'm shopping for a wireless router to use with the netbook, but I have no experience with wireless routers.

Are there any specific router features that I should look for, or avoid?

---

### Post by isantop on 2010-10-13
As long as the router supports both g and n, you should be good. We've had cases where one or the other was causing problems, though g-only is probably a safe bet. I'd recommend getting one that supports g/n mixed mode.
