### Post by abhiroopb on 2008-06-08
a .exe file is a windows installation file and it cannot be run in ubuntu (without special software). Go to Applications>Add/remove programs and search for whatever program you want. Check it and click apply. 

For example if you want to get a music playing software, just search for "music" and you can read about the various applications.

For more information about installing files: [http://monkeyblog.org/ubuntu/installing/#installer](http://monkeyblog.org/ubuntu/installing/#installer)
