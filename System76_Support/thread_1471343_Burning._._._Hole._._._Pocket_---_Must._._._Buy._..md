### Post by isantop on 2010-05-04
There is one bug you might notice. Due to an interaction between Plymouth (The program lucid uses for the boot screen) and the proprietary drivers, the boot screen is displayed at a very low resolution, meaning it looks really bad. Unfortunately, this is not a bug that is consolidated to System76 computers, and is rather a bug in plymouth, so we will have to wait for a fix from Canonincal.

In the meantime, the performance of the computer itself is not affected, so you can continue to use it normally, and ignore the boot screen. Many users simply use suspend all of the time anyway.
