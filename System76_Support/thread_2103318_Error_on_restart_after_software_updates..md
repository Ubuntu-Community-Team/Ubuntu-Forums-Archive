### Post by ibjsb4 on 2013-01-10
Doesn't sound like cause for concern, as it has repaired whatever package(s).

You can run an update in terminal just to see if any errors come up, but Im betting your system is clean.

[Open a terminal]("https://help.ubuntu.com/community/UsingTheTerminal#Starting_a_Terminal") and enter:

```
sudo apt-get update
```
