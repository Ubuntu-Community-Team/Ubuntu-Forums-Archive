I would recommend the  4MB of cache...it really comes in handy, especially if you can't overclock to make up for the cache difference (which you don't want to try on a laptop). 

I should think that the Sys76 laptops would be able to utilize the VT technology...you can buy as strong a processor as you want, but it won't make up for not having VT. It lets you run virtual OS's at near native speed. I would wait for an official rep to answer this question, but I don't think you'll have any worries.
