### Post by isantop on 2010-09-30
The Edubook's hard drive is located beneath the main service panel. It is held in place with a bracket, which is useful for removal, and has rubber sleeves on both sides for shock protection. I'm not sure how they attach, but it may be adhesive, meaning any removal and reapplication will not be as strong as the original installation, but it should work.

The Standard Starling is a little more unusual. The hard drive is below the keyboard, which is easy to remove. Simply release the clips holding the keyboard in place, carefully unplug it, then replace the hard drive.

Both of these models use a standard 2.5" laptop hard drive.
