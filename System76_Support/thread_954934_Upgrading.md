### Post by thomasaaron on 2008-10-22
Using Update Manager is fine.

Personally, I like to do a clean install (but, then, I also like the smell of a clean sheet of paper).

Here's the thing: There is *ALWAYS* a huge rush for the newest version on the week it is released, and there are *ALWAYS* several hosed upgrades because of server timeouts, etc... If you like living on the edge, rush right into it.

The more cautious folks like to wait a week or two. This way they avoid server timeouts, and if any serious bugs were missed they are usually fixed within a week.

But, it's a free (software) country, right?
