### Post by thomasaaron on 2008-03-26
> I heard that many thing do not work under 64bits system

Actually, there are some things that did NOT work well under 32 bit that work great under 64-bit.

The biggest problem I'm seeing out there with 64-bit is that IcedTea (which provides the jvm for java applets) seems a bit haphazard. It will work with some applets but not with others.

You can always install 32-bit firefox on a 64-bit system, though. No biggie.
