### Post by Lee_Machine on 2009-02-18
Did you boot up with the dongle plugged in? I know that for some reason  ubuntu has issues with that?....at least in the past. 

Try plugging it in after you boot up. Does that work?

I found [this ]("http://ubuntuforums.org/showthread.php?p=2104915") for the V450. Kind of old, so not sure if it's relevant.

You cannot install the driver that comes with it, it's for windows.

For the most part peripherals like this usually don't have full functionality, and require the driver. I wish there was a wrapper for windows mouse and keyboard drivers. That said, they do work. Just not all buttons, all the time.

Advice for future purchases is to make sure it's 100% out of the box compatible before you buy. Some are even coming with Tux on the box :) 

Great list of Mice that 100% work.

[http://hardware4linux.info/module/usbhid/](http://hardware4linux.info/module/usbhid/)
