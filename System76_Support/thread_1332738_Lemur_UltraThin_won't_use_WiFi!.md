### Post by grikdog on 2009-12-11
This is solved.  Lemur Ultrathin requires Wireless-N, which means I had to update my ancient Qwest Actiontec modem to the latest model PK5000.  Works fine after that.  YMMV, but I also had to call my ISP to enter username/password for the ISP connection.  This was a good move, because the old modem was set up for WEP, while the new modem defaults to WPA2-Personal.  Thanks to System76 support for the help!
