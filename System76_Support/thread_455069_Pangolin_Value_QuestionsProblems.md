### Post by thomasaaron on 2007-05-29
The sound menu should have a control for this.

Double-click the volume Icon on the top right of the desktop.
Under the switches tab, there is an option to enable or disable this switch.

Make sure you are set to the Alsa Mixer. To do this, Double click the icon in the upper right. Then click File > Change Devices. Select the Alsa Mixer option.


You can also try muting the main speakers.
