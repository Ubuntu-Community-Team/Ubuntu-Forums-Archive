### Post by jml on 2011-09-21
The LTS release for the desktop is supported for 3 years.  That is roughly the interval at which new ones are realeased.  The last LTS version was 10.04, released in April of 2010.  the next scheduled release of an LTS version would be in 2013.  Here is a link to a Wiki page that explains it.  

[https://wiki.ubuntu.com/LTS](https://wiki.ubuntu.com/LTS)

So version 11.10 will not be an LTS release if Cannonical maintains the previous pattern.

System 76 tracks the release cycle pretty closely.  they usually ship with the most current version within days of it being released.  One exception, however.  If there is a serious issue with the new release and System 76 Hardware, they may hold back the upgrade on select computers.  they did that once with the Starling when an upgrade broke some functionality.

Joe
