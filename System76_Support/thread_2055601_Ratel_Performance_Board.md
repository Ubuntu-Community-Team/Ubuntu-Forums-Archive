### Post by Carborundum on 2012-09-10
> **isantop said:**
> Actually, it's both. Micro-ATX and Mini-ITX are the same.
Not at all. Micro-ATX is 244 by 244 mm, mini-ITX is 170 by 170. Mini-ITX boards typically have a single PCI-express slot, while micro-ATX can have as many as four.

### Post by mevun on 2012-09-10
I think the computer case can hold either micro-ATX or mini-ITX motherboards.  At one point someone mentioned the exact Lian-Li case (but I can't find that now), but the specs for that do allow either form factor for the motherboard.

I think the actual motherboard sold by System76  is mini-ITX.  In fact, I think it is the Intel DH77DF which has firewire.  Even if it is not that specific motherboard, the system76 configuration options for memory only allows 2 RAM sticks which is usually the limit for mini-ITX motherboards.  Micro-ATX motherboards have the space to go up to 4 RAM sticks.
