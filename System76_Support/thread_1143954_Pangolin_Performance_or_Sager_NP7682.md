### Post by thomasaaron on 2009-04-30
> Is there a reason to buy a Pangolin over the Sager.

Support. The benefit of the version-to-version driver support provided by the System76 Driver. 

> Does the Pangolin not work nativly with Ubuntu so they install custom drivers?

TA: The Pangolin *mostly* works out of the box with Ubuntu. We do have to install drivers for the fingerprint reader still. There may be something else...

> Would I not be able to install a clean version of Jaunty on a Pangolin? on a Sager?

TA: On the Pangolin, you would do a fresh Jaunty install and then install and run the System76 Driver. This would restore your machine to factory specs. The driver won't run on a Sager machine.
