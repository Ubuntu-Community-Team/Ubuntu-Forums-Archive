### Post by miniyak on 2009-12-13
can you still use the track pad on the gnome start-up screen? I think there is an option to change layouts there.

if all else fails try starting in recovery mode

from the grub boot screen key down before it loads your default install of karmic. One of the options says recovery mode

try to start normally (you will be prompted with this option)

in 9.10 you will have to enter your password

then we can see if things work from there 

if your keyboard is unresponsive when trying to mess with you bios or select something from the grub menu, then that may indicate a hardware issue.
