### Post by Bucky Ball on 2011-07-03
I would go the fresh install of 10.04 LTS, a long-term release until 2013. The two release you mention are not supported anymore. You might be able to do it the from 8.10 but it would involve a lot of faffing about. And I say 'might'.

I would clean install and be done with it. Make a backup, naturally, of valuable data, and perhaps include a separate /home partition in the new install so that valuable data is separate from the / OS partition in future (you may already be using a separate /home, of course).

(PS: Even if you did progressive upgrades from 8.10 you would need plenty of time on your hands or a super-fast internet connection. It is not quick. Clean install would be waaaay quicker.)
