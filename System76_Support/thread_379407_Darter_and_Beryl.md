### Post by PriceChild on 2007-03-08
It has intel graphics and "should" work out of the box by adding ```
deb http://ubuntu.beryl-project.org edgy main
```to your /etc/apt/sources.list and then apt-get installing beryl.

[http://wiki.ubuntu.com/BerylOnEdgy](http://wiki.ubuntu.com/BerylOnEdgy) contains some xorg.conf changes that will give speed increases.#

### Post by thomasaaron on 2007-05-04
I run Beryl on my Darter (which uses Fiesty).

The easiest way to load Beryl is:

Applications > Add/Remove Software

Search for Beryl.
Select all Beryl items and click apply.

It's flawless.
