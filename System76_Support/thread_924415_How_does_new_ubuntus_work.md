### Post by wrender on 2008-09-19
When the new version of Ubuntu comes out how does that relate to system76?

---

### Post by thomasaaron on 2008-09-19
When the new version of Ubuntu comes out, System76 will test it on all of our computers and send out bug fixes via the System76 driver (if necessary).

---

### Post by wrender on 2008-09-19
So I would just download ubuntu the normal way then I guess there is some sort of repo for sys76 that I add  and all is well?

---

### Post by thomasaaron on 2008-09-19
Yes, that is what you would do if you were doing a fresh install: Install Ubuntu, install the System76 driver, run the System76 driver and reboot.

If you just upgrade from one version of Ubuntu to another, you would just upgrade to the new version, run the System76 driver and reboot.

---

### Post by wrender on 2008-09-19
Would it be the same with kubuntu?

---

### Post by thomasaaron on 2008-09-19
Yes. While we do not officially support Kubuntu and Xubuntu, the System76 driver will run on them. The fixes are *usually* the same.
