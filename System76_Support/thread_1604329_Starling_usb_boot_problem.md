### Post by Dave_L on 2010-10-23
I wasn't sure what the "F12" meant.

I created a 10.04 Live USB and booted from it this way:

With the USB drive inserted, restart, and during boot, press F2 to enter the BIOS utility. (Supposedly there's a message telling you this while it's booting, but it flashes by so quickly I've never seen it.)

Then in the list of boot devices, move USB up to the top. Then save the BIOS settings and reboot.
