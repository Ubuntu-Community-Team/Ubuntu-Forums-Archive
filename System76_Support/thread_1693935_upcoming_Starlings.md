### Post by isantop on 2011-02-23
Hah! That's an amazing coincidence! Unfortunately, that's about as far as it can go. But that is neat!

1:
[LIST=1]
[*]Star1 = "Old" Starling, with rounded, flat black case
[*]Star2 = Starling Edubook
[*]Star3 = First Generation "New Starling", with Atom N450
[*]Star4 = Second Generation "New Starling", with Atom N550. No external change from Star3.
[/LIST]

If the Starling we ship then is a new model, with updated hardware, then it will most likely be the Star5. If we don't change the hardware at all, then it will be Star4.

2: The current Starlings (From Star2 and onward) support 64-bit. We don't install it because there is no 64-bit version of Netbook Edition. In April, with 11.04, the Netbook and Desktop editions will be merged into one image, with one for 32-bit, and one for 64-bit. We will test with both, but if it works, then most likely we will ship 64-bit after 11.04 comes out.
