### Post by DRM_free on 2007-09-25
[LinuxBIOS]("http://linuxbios.org") has announced [support]("http://linuxbios.org/index.php/MSI_MS-7260_Build_Tutorial") for the affordable yet state-of-the-art desktop motherboard [MSI K9N Neo (MS-7260)]("http://global.msi.com.tw/index.php?func=proddesc&prod_no=255&maincat_no=1&cat2_no=171"). This is the second desktop motherboard to run a completely Free/Open Source BIOS, and at only [$72.99]("http://www.newegg.com/Product/Product.aspx?Item=N82E16813130123") it is significantly less expensive than the previously supported motherboard, the [GIGABYTE M57SLI-S4]("http://www.newegg.com/Product/Product.asp?Item=N82E16813128014").

Motherboard features:
AMD Athlon 64 X2 / Athlon 64 FX / Athlon 64 processors 
AM2 CPU Socket
8 GB max memory
1X PCI Express x16 slots
2X PCI Express x1 slots
3X PCI
SATA RAID
4 X SATA 3Gb/s slots.
1 X ATA100 (2 devices)

I would highly recommend that System76 sell desktop computers with motherboards that support LinuxBIOS. Combined with an recently [open-sourced]("http://www.phoronix.com/scan.php?page=news_item&px=NjA0Ng") AMD/ATI graphics card from the Radeon X1000 and HD 2000 series, this desktop would make a killer Free Software computer. Just imagine what a shiny review it would get on Slashdot and how many units you'll be able to sell!
