### Post by isantop on 2013-11-27
It's probably going to be fastest to just install Ubuntu on the new drive and copy your Home Folder over. You won't need the System76 driver if you install a recent version of Ubuntu.

You'll probably want to consider upgrading Ubuntu to a newer version. I'd recommend 13.10, but 12.04 would be a good minimum. 10.04 will no longer receive security updates for the desktop components, which can leave your system open to outside vulnerabilities.
