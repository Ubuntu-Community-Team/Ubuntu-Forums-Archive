### Post by neophyte_of_linux on 2010-08-18
Hey, my new Ratel Ultra arrived yesterday and WOW!  what a great looking machine.  I am really impressed with the craftmanship-like quality of the aluminum case!  It really feels and looks like quality. I like the machined system76 logo and the onyx color.  I bet the red looks really nice.

I suppose it sounds strange to be so keyed up on the "looks" but it was unexpected.  It is even better than the pictures.  

As for performance, I haven't really been able to "push" it any since I've only been using it for the internet.  The DVI looks great on my viewsonic monitor!  It definitely feels "snappy".  I upgraded to the base i3 processor.  I really like the looks too of the lucid lynx or whatever 10.04 release.

So far I'm really happy with the purchase. :popcorn::D:D

### Post by neophyte_of_linux on 2010-08-19
The box is 193 x 209 x 291 mm.

Yeah, I'm not too sure about what is inside for space.  I know I asked if a second hard drive would fit, in addition to the DVD drive, but it can't.  So my old 3.5" 80 GB drive I will put in a case and use as USB drive for back up.

I counted about 23 seconds for it to boot up to log in from pressing the power on button.  

My only little quibble has been it didn't come with any documentation, so I am assuming the Firewire and S/pidf, etc, are acutally usable features.  

Otherwise, I was impressed that it detected and identified my officejet 6000 printer simply by plugging in the USB.  A year ago,  it was me, a stack of books on linux, the web forum and a few hours of reading how to download and install the driver package from the HP website.  So Ubuntu has seemingly improved greatly!

At my age and interest level in computers, I would like to be more proficient in Ubuntu linux and all that, but I am greatly relieved that it so far "just works" right away.

But again, I really like the quality of the build of hardware.  I like italian motorcycles, and other "crafted" mass production type things, so I'm happy with the construction so far.  It is also a very quiet machine!  I like that too.

I'm looking forward to getting a chance to play a DVD movie for entertainment.  As soon as I get back "Live at Pompeii" from on loan, I'll report back on DVD playback quality and sound. It is supposed to have good sound too.
