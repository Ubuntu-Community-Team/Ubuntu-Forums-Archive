> **WinterMadness said:**
> The touchpad is about as cheap as it gets. Youll be trying to use it and it will randomly go in different spots all over the screen. This makes it very very difficult to actually click on something.
Adjusting the "Jumpy Cursor Threshold" with the command below may solve this.
```
xinput set-int-prop "SynPS/2 Synaptics TouchPad" "Synaptics Jumpy Cursor Threshold" 32 110
```
Try different values where the 110 is until you get something that works to your preferences.
If you want this to be permanent you'll have to make it into a script in the startup applications.
