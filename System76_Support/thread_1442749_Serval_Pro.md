### Post by bakelitedoorbell on 2010-04-01
I have the previous version of the Serval (serp5).  I chose this model because it had the option of a high resolution screen.  This one has a 1920x1200 pixel matte screen, which is beautiful.

I have no regrets over the purchase.

### Post by PabloH on 2010-06-22
I wrote a review a couple days ago.

Mine is setup like this:

CPU core 7 I-820
NVidia Ge-Force GTX285M
8GB 1333
160GB Intel SSD
led lcd display

It is fast. Very fast. Lightning fast.

I don't like touchpads, and this one is no different. So I use a usb rat. The battery lasts around an hour. The screen is nice. The keyboard is good, not quite as good as an old IBM thinkpad but close.  The fit and finish is very nice. 

It is not hot. You notice some fan noise, but no overheating at all. It is much cooler to the touch than the Dell 620 I use at work, although the ACPI cpu reading is a tad warmer.

The only thing that really annoys me about it is that either it or Ubuntu Lucid has some bug where the x server restarts when you hit the enter key twice in a row (really fast) after it has been running for several hours and sitting idle. I have not lost any data, but it is starting to get annoying. Any ideas would be welcome.
