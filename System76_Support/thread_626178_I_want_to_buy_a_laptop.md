### Post by ijason on 2007-11-28
hey there.

i've been running ubuntu on my dell inspiron6000 for a few months now, and other than the little smart-card reader not working out of the box, it's been a really easy transition for me.

the only advice i CAN give, is that there will always be a program or 3 that simply won't ever run right (or, at all) on linux. i haven't bought it yet, but i've heard really good things about win4lin, which lets you run windows applications on ubuntu. there are several programs that do this - virtualization programs they're called - some are free. i like win4lin because it's only $50 to buy it, and they seem to have a really active support system. 

so, if you're wanting to run a laptop on ubuntu i would say 1) try to get one with an nvidia chipset video card, which generally have better linux support in drivers. and 2) don't expect finger-print readers to work any time soon. otherwise it seems like most machines run pretty well out of the box. 

i would just plan on learning how to use a virtualization program. there are certain programs which are too explicitly dependent on windows to ever be ported over to linux, and to run them you'll have to use a program like win4lin.

### Post by jml on 2007-12-01
weverjames,
I have experience with two different set ups.  Opening PowerPoint presentations goes "ok"  But you are correct, some of the more advanced features like annimations, and sound can be problematic.  If I am just reviewing the document for myself, that is not really a problem because all of the real data is there.  The only formatting problem I noticed is the shape of bullet points.  Seems that M$ uses some type of proprietary format to draw tha actual bullet points.

For presentations that I make in Open Office.org, I find that if I keep it  simple, standard backgrounds, no animations or sound, the run in PowerPoint just fine.  On those rare occaisions that I need annimations, non-standard back grounds or sound files, I break down and use PowerPoint from within a Windows XP partition.  

Another option I have used at my institution has been to bring my own laptop and connect it to the LCD projector directly and bypass logging into our network entirely.  That way, I know how my presentation will look and minimize suprises.

The time it takes to get your computer may vary depending on many factore, the popularity of the model you order, the availability of parts, etc.  I would suggest contacting System 76 via their customer support e-mail address from their web site and ask once you have decided on a specific model.

As for quality, I own a first generation Darter (Daru1,) and even though its was, at the time, their smallest model, its built very solidly.  Hinges are still tight, case feels solid, key board also has a good fee.  

Size, I agree with palintheus, once you carry around a smaller laptop, like the Darter, or a ThinkPad X series, you will never go back to a 14 or 15 inch laptop.  Now if you are going to use it on a desk most of the time, and only occaisionally carry it around, you might want a unit with a larger screen, especially if you are going to be watching movies on it.

Routers, again palintheus is right.  Most routers and wireless access points are configured with a "web" interface.  The only problem I encountered in the past was when I wanted to download and install new firmware for the router.  It was supplied as a zip file that required Windows to expand and instaa as an .exec file.  The only time I had a problem was when I set up an internet account with Charter Communications, (my local cable company.)  I needed a Windows computer to actually set up and configure my account.  After set up, I can connect with any computer I have.

If you do not want to dual boot with Windows, and I can sympathize with you an that point, and if your finances permit, you might want to buy a new System 76 unit for most of your work, and buy a refurbished laptop running Windows for those very rare times you need to use Windows.  Here is a link to check out.

[http://www.usanotebook.com/index.php](http://www.usanotebook.com/index.php)
