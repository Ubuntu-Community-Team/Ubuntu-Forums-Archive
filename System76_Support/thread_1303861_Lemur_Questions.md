Yes, it is somewhere in between, but it is leaning more toward the Core 2 Duos in standard laptops. It runs virtual machines like nobody's business. The Lemur definitely isn't lacking in the power, but it's not as high-performance a machine as the Pangolin.

> How long will the Lemur run on a charge? Will you be offering an additional/replacement battery for the Lemur? Is the battery even user-swappable?

The battery is user swappable. It is a flat panel on the bottom of the computer. It is getting approximately 4 hrs on a charge (...cpu is a 10-watt). Not sure yet when/if replacements will be available, but I'm *fairly* sure they will be

> Also, I was initially surprised that an upgrade from 2GB of RAM to 4GB of RAM cost $230.00 until I saw that the 4GB was still using a single DIMM. I assume this is because the Lemur’s mainboard only has a single slot for RAM? Can you confirm? 

One slot. DDR3.

> Is there a reason for this limitation other than "that’s what the manufacturer offered."

No. Remember, this isn't meant to be an uber-powerful machine. It is a highly portable, ultra-light, ultra-thin, low power consumption machine. In order to accomplish that, you have to make a lot of tough choices as to what hardware options are included.

> If there is any way to get a Lemur with 2 DIMM slots so that the RAM upgrade would be "4 GB - DDR3 1066 MHZ - 2 DIMMs ( +$60.00 )", please consider it.

Not possible at this time.
