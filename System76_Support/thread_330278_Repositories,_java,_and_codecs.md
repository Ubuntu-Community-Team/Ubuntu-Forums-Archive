### Post by crichell on 2007-01-03
universe and multiverse are enabled by default.  codecs aren't installed nor is Sun's Java (it currently requires accepance of a license before installing.)

For codecs this is probably the best source:

[https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)

We also have a restricted formats installation How To here;

[http://knowledge76.com/index.php/Installing_Restricted_Formats](http://knowledge76.com/index.php/Installing_Restricted_Formats)

The best source for Java install is again the ubuntu wiki:

[https://help.ubuntu.com/community/Java](https://help.ubuntu.com/community/Java)

With Java going GPL we'll be able to load it by default soon (if feisty isn't already going that route).  We're also working on providing codecs out of the box.  That's quite a beast to tackle though.
