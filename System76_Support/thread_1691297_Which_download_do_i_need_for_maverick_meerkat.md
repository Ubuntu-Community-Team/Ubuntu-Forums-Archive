### Post by JOHNNYG713 on 2011-02-19
Yes, that's correct. The architecture is called after AMD64 (AMD was  first to create such processors). Intel came later and called it EM64T  but it is compatible to AMD64 so you can use the AMD64 iso to install  64-bit Ubuntu. On an Intel rig !

### Post by jml on 2011-02-19
J697, if you try to "leapfrog" versions when you update, more likely than not, you will get serious breakage.  Both Ubuntu, and the techs from System 76 recommend updating sequentially.  In my opinion, a simpler solution would be to back up all your data and do a clean install.  Then you do not have to update sequentially.  This is one of the reasons I prefer Debian as my main distro.  The updates do not come out as often and you can set things up to do a rolling update.
