### Post by cprofitt on 2012-11-03
Need to know which Gazelle Pro -- this years model with the third gen core i7 series processor should take PC3-12800.

I bought the following ram for my new laptop - [Corsair Vengence]("http://www.amazon.com/gp/product/B0076W9Q5A/ref=as_li_ss_tl?ie=UTF8&camp=1789&creative=390957&creativeASIN=B0076W9Q5A&linkCode=as2&tag=fretrabeo-20")

If you already have 8gb in a single dim you would not need a pair to make 16gb, just one stick. If you have 4GB you just need a single stick to get 8GB.

Ram timings are an indicator of performance -- the lower latency the faster the response of the ram is... though the 1600Mhz ram is also an indicator of that. Here is a good [wikipedia article]("http://en.wikipedia.org/wiki/CAS_latency") on the topic.
