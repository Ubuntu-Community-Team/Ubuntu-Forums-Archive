### Post by thomasaaron on 2008-10-30
The GazV5 is not completely functional without the System76 Driver. The internal mic requires a model designation in the /etc/modprobe.d/alsa-base file. However, the System76 driver will run on 32-bit and install that.

*Chances are* everything would run fine in 32-bit. We no longer test 32-bit though. There has been some talk about including it as an option, but I'm not sure what the status of that idea is.

If you don't mind, why do you not want 64-bit?
