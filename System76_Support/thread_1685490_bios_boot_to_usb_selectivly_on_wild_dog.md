### Post by isantop on 2011-02-11
Actually, this is possible. Set the Hard Drive to boot first (Or at least above the USB drive). Then, on boot up, press the F10 key. This will take you to the boot menu, where you can select any device to boot from (CD, HDD, network, etc.) Any USB inserted should show up there.
