### Post by Dennis N on 2012-05-10
I would check Add-Ons Manager > Extensions. See if the Global Menu Bar Integration extension is disabled.

---

### Post by shochatd on 2012-05-10
Thanks. That was it. I must have disabled all extensions (not realizing that was one of them) when I was trying to stop firefox from maximizing the window every time I started it (a problem that disappeared on its own once I upgraded to 12.04). I hope I haven't now brought that back!
