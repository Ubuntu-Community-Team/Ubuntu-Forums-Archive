### Post by crichell on 2006-08-14
System76, Inc. launched the first available line of Ubuntu based servers featuring Ubuntu 6.06 LTS Server Edition August 14th.  Ubuntu on System76 represents a new choice in quality server platforms including standard 3 year overnight parts replacement warranties and 5 years of Ubuntu Linux operating system security updates.

[http://system76.com/index.php/cPath/43](http://system76.com/index.php/cPath/43)

The release includes the 'Eland Series' General Purpose Tower Servers and 'Jackal Series' High Density Rackmount Servers. System76 servers feature:

* Ubuntu 6.06 LTS Server Edition Installed (32 and 64 bit Available)
* Up to 2 AMD Dual Core Opteron Processors
* Up to 2.4 TB of Storage
* Up to 16 GB of Memory
* High Performance RAID Available
* SATA Hard Drives
* 3 Year Overnight Parts Replacement Warranty and Standard Technical Support

The Eland and Jackal Server Series fulfill numerous data center needs including:

* Application Server
* Internet Server (Web and Email)
* Database Server
* NAS (Storage Server)
* File and Print Server

Open Source Applications Available with Ubuntu 6.06 LTS Server Edition include:

* DHCP and DNS Server Servers
* Apache2 Web Server
* Full LAMP Software Stack
* MySql Database Server
* SAMBA
* NIS (Network Information Services)
* Open LDAP
* Tomcat Application Server
* Jabber Instant Messaging Server
* And much more...

System76, Inc. is a Denver, Colorado US based company providing Ubuntu linux laptops, desktops, and servers.
