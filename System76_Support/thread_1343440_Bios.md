This might be a dumb question but hoe do I change the boot order of devices on Meerket Nettop? I wanted to do a clean install of Ubuntu 9.10 on it. Note that I'm using an external drive (USB) and not an internal.

---

### Post by thomasaaron on 2009-12-02
Try plugging in the USB drive and rebooting.
When you see the Bios Mfg splash page, press F2.
Arrow over to the boot tab.

You should be able to set your boot order from there.
