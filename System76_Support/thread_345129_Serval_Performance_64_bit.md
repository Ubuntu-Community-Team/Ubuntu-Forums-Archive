### Post by crichell on 2007-01-24
We only pre-install 32 bit Ubuntu.  There is some performance gain by running the 64 bit version; however, most applications and libraries are compiled for 32 bit Ubuntu so you're required to chroot those applications to use them.  Probably not worth the modest gains in performance.
