### Post by isantop on 2011-01-13
By default, Ubuntu will only show an Icon for the battery when the battery isn't full. I also prefer to have it available all the time, so I change it too. You can access the setting by going to System > Preferences > Power Management. There, you can go to the General tab and select how verbose you want the indicator to be. I personally like "Only display an Icon when a battery is present".
