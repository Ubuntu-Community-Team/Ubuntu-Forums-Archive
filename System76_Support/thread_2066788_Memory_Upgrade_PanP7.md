### Post by rsmeineke on 2012-10-06
My trusty PanP7 came with Apacer memory ... 4GB (2 x 2GB) labeled 2GB SODIMM PC3-10600 CL9. Basically, DDR3-1333 ([http://http://www.tomshardware.com/forum/274587-31-10600-10666-what-difference](http://http://www.tomshardware.com/forum/274587-31-10600-10666-what-difference)) 

The laptop is based on this old Clevo model: [http://http://www.avadirect.com/gaming-laptop-configurator.asp?PRID=15899](http://http://www.avadirect.com/gaming-laptop-configurator.asp?PRID=15899)
(scroll down for the memory specs.)

It looks like the max memory is 8GB which seems to ring a bell with me as one of the ordering options from way back when.

I'm also considering a memory upgrade and perhaps an SSD just for fun -- but on balance I'd have to report that this machine (core i7 version) is still snappy as heck.
