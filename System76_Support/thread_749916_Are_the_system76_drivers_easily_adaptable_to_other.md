### Post by thomasaaron on 2008-04-09
We use a Python application called the System76 Driver to install patches in holes left by Ubuntu. It is an open source application, so you can crack it open to see how we fix various things.

We would also provide you any information we can about hardware, etc... to assist you in getting your comptuer up and running with another distro.

That said, we only install and support Ubuntu. We don't test with anything else, so we can't guarantee that all the hardware will work with other distros -- though it probably would.
