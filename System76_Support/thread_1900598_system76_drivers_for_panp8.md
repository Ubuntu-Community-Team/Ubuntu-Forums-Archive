### Post by isantop on 2011-12-27
Lucid is not supported on the PanP8. The oldest release that it will run is 11.04, and optimum performance isn't had until 11.10.

10.04 is simply too old to run on it, and it doesn't have support for the newer hardware present in the PanP8.
