If noise is a big issue (i.e. you don't want it sounding like a turbine), look into liquid cooling.  It is usually whisper quiet and very efficient.  Does take a bit of effort for up keep though.

### Post by thomasaaron on 2009-07-27
Sorry, I went camping/fishing/canoeing over the weekend, so I didn't check the forums.

> 1. what is th case built of? (aluminum, steel, plastic?) What is the brand of the case? (so that I can research about its quality/specifications).

Aluminum. The case is made by Lian.

> 2. Is the Wild Dog QUIET? This is very important to me, so I will need honest answers. I don't want disturbing noise in my office.

Having a front and rear exhaust fan, a cpu fan, and a fan on the graphics card, it is not *whisper* quiet. But, for a tech support perspective, I've never had a complaint about the Wild Dog's noise level, and we've sold a *LOT* of them. They are incredibly quiet, in my opinion. But you can hear them.

> 3. Build quality. Does is feel like a solid system that would last for years?

There's NOTHING cheap about this box. Period.
> 
4. Did ubuntu worked out of the box? Any issues with flash player or fire fox running with the factory loaded 64 bit version? I ask cause I was having issues with firefox freezing when opening heavy web pages with flash in the 64 bit version of ubuntu 9.04. The issues resolved after installing the 32 bit version.

By default, the 32-bit flash plugin is wrapped with nsplugin wrapper. While it works pretty good, it still has it's occasional bugs. I've started sending out directions for people to install Adobe's 64-bit flash plugin, and that seems to be working great. Hulu, etc... works beautifully.

> 5. Any problems with large monitors. I will be using it with a 26 inch wide screen. I am concerned that they use ATI cards in the wild dog and I have read that nvidia offer better drivers and better support to linux. Any flickering or blank screens on /off?

We actually test with 26" monitors. It works just fine.

> 6. Sound quality. Is the sound good enough to use high def speakers and use the desktop like an enterteinment center?

It is set up for up to 5.1 channel surround sound. We crank it up in the office every once in a while and have the whole building rocking with The Eagles.

> 7. Are you satisfied with your purchase?

76-ers?
