### Post by williumbillium on 2009-05-17
For the past couple of weeks that I've been running Jaunty on my Daru2, I've been experiencing poor graphics performance.  Firefox and Xorg are typically eating up huge chunks of cpu power.

These symptoms are apparently typical of intel integrated graphics setups running Jaunty and are outlined here:

[http://ubuntuforums.org/showthread.php?t=1130582](http://ubuntuforums.org/showthread.php?t=1130582)

Also in the above post, are some instructions to upgrade to the latest Xorg drivers as well as some other fixes/workarounds to improve the situation.

I only made the changes a few minutes ago but it already seems a world apart from the sluggishness I was previously experiencing.  I chose the sage configuration btw.

Hope this is of some use to other Daru2 (and integrated intel graphics) users.
