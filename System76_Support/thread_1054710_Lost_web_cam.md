### Post by thomasaaron on 2009-01-30
The kernel update probably hosed the web cam. If so, you will need to go back to the cam installation instructions that you used to activate it and re-run them.

Or, it probably still works in the previous kernel. You could boot into that kernel by pressing "Esc" when you see the Grub countdown. Then select the previous, non-Recovery-mode kernel.
