I have been using it for some embedded systems development. I connect to my network (and the Internet) via the wireless, and use the wired for my development network. The Meerkat hosts DNS, NFS, and TFTP for the development network. The serial port is nice for connecting to an external modem. I also use the port for remote debugging.

As far as web browsers, streaming or playing media files, etc., it does as well as my 64-bit Serval (P3). I have not tried running VirtualBox or any other virtual machine (except java and mono) on it since it does not have as much processor support. I have read a post somewhere in here where an author was running a virtual machine and was satisfied.

I have not experienced any problems with it so far. I have read several reports of corrupted file systems with Jaunty, but have not had this issue.

The wireless adapter that came with it is a DLink WUA-1340 which works just fine. This is my first experience with one of these and it 'just works'. I must use manual configuration of my network for my use.
