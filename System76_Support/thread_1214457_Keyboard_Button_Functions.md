### Post by thomasaaron on 2009-07-16
It throttles down your fan. I believe it also decreases CPU performance as well in order to reduce heat production so that your CPU doesn't overheat.

You would notice it audibly if you're doing CPU intensive work. Or you would notice it visually if you had the CPU scaling applets installed.

### Post by thomasaaron on 2009-07-17
> Or you would notice it visually if you had the CPU scaling applets installed.

Right-click your upper panel. Select "Add to Panel." Select "CPU Frequency Scaling Monitor." Click the Add Button. Add one for each CPU core. 

It won't exactly tell you that you're in silent mode. But it will scale back the CPU's when you press the button -- if you're doing CPU intensive tasks. That's probably the best there is right now for a visual.
