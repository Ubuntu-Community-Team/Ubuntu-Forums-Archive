### Post by thomasaaron on 2008-02-25
> (1) How many hard drive bays does it have? The specs/config page does not mention anything in particular. I just want to shove a couple 1TB drives in there.

The motherboard has 4 sata ports, but the chassis only has one mounting frame (which is on the "floor" of the chassis. So you'd have to be a little creative in coming up with a mounting solution.

> (2) Can I have the system delivered with Ubuntu Server installed?

No, we don't install the Server edition on Ratels.

> (3) If no to #2, can I download the System76 driver separately, as that seems to be the only real difference between the regular install and the System76 install? I don't see it anywhere on the website or in the repos.

Yes, it is available through our repos. But it doesn't add any support for the server edition. Also, we've not tested the server edition on the Ratel. Should work fine, but 
I just can't confirm it.
