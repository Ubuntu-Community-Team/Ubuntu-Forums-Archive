### Post by isantop on 2012-02-07
Our systems are configured using the standard OEM installation method, meaning that after installation, there is a temporary user account that we can use for QA testing and final setup. Before shipping the system out, a script is run that causes the Ubuntu installer to run through the initial user and system setup again.
