### Post by thomasaaron on 2008-02-26
You will basically run updates as usual (using Update Manager). If an update hoses something, running the System76 driver fixes it. If it doesn't fix it, an System76 driver update will come down shortly (usually within a day or two), and it will include the fix.

When a new version of Ubuntu is ready, we have already done our testing on it and updated the System76 driver to accomodate any regressions caused by the new kernel.

So you would just update to the new version of Ubuntu, and then run your System76 driver.
