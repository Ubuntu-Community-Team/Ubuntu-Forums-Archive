### Post by isantop on 2011-09-15
It's still a D-Link to my knowledge. I can't remember the model number, but I'd be happy to look it up if you still need it. 

Pretty much, just avoid anything based on a broadcom chipset. Realtek would be pretty shaky too, if you're not careful. Intel would be a great bet, and I've heard good things about Atheros as well. Note that this is the chipset, not the brand of the actual card. You should be able to find the chipset used on the manufacturer's web page, and if not, then someone in their tech support center should be able to help you out.
