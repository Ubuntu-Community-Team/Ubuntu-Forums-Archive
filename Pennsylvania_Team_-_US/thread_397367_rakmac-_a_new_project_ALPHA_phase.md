rakmac: random acts of kindness macs

This project is meant to make use of old iMacs or any other G3+ Macintosh computers that might otherwise be useless, loading them with xubuntu or edubuntu (or a hybrid) and then donating them to community organizations who are in need of computers.  So far I've been given 2 machines and bought 2 for an undisclosed price.  I have offers from various sources for another 8-10 machines and the response has been very positive.  for the first installment of the story of how this all came about, please go to the [website]("http://rakmac.org") and if you're interested, register and comment, etc...

Thanks for checking it out!

Aloha,
Garrison Gunter
PS- the site is only a day old so it's not finished and generally in a rough state.  If anyone would like to help with this project I'd love the help.  The next stage will be to customize the install and to get artists to custom paint the shells of these machines.
