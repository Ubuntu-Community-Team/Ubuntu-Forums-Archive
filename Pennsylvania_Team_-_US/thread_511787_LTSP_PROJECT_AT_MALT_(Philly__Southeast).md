### Post by jedijf on 2007-07-28
[https://wiki.ubuntu.com/PennsylvaniaTeam/CommunityOutreachTeam/LTSPMALT](https://wiki.ubuntu.com/PennsylvaniaTeam/CommunityOutreachTeam/LTSPMALT)

The project is NOW getting started.  See the above link for info, and to signup for Phase 1 (Cleanup and Setup)

Briefly, LTSP is the Linux Terminal Server Project.  We are setting up and instructing the Mt Airy Learning Tree Computer Volunteer Group on how to implement and administer this in a 'Classroom' style environment.

This should prove to be an interesting and exciting project, so I hope that you can get involved.

No knowledge of LTSP is required.  This can and will be a 'learn as we go' project.  

If you are uncomfortable with editing the wiki page, just leave a post here to the forum and we will add you to the volunteer list for PHASE 1.

This will be an ongoing project, so if you can't make PHASE 1, keep checking this forum thread for info on upcoming Phases of this project.

Please get involved, that's what this is all about.

Whenever possible, we will try to make major 'Phases' of the project on Saturdays, so that other REGIONS can roadtrip and get involved.
