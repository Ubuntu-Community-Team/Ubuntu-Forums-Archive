Wine is 32bit, even in 64bit ubuntu. The reason is that its a 32bit application layer. It has to be 32bit. The package for Ubuntu 64 just makes it easier to install. 
As far as being able to install games without any setup, I personally have never seen that. Wine is far from perfect. Looking at the [Wine Application Database,]("http://appdb.winehq.org/appview.php?iVersionId=6482") there is a whole wiki linked about getting WoW to work with any version of wine. [Guildwars has a page there also ]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=9194")
I know that Neverwinter Nites has a linux binary and its possible to install it, there are instructions on the Neverwinter Nites site.
[Diablo 2 runs great but requires some setup ( everything in wine does) ]("http://appdb.winehq.org/appview.php?versionId=49")

But since you are having trouble getting any game to run,, lets ask the most obvious questions. Do you have the accelerated graphics drivers for your video card installed? Are you running Desktop effects?
