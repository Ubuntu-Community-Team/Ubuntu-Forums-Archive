### Post by jamesford on 2007-06-02
if u want the windows fonts u can install the msttcorefonts package. i dont think it includes tahoma though, but u can copy tahoma from a windows machine or search google for it and download then make a folder called .fonts in your homedir and put the tahome files there

u can also go to ubuntuguide.org and search for 'font' and you will find some font optimizing tips.

you select fonts in system > preferences >font. you may want to play around with the settings there to get the look ur looking for

personally i prefer the bitsream vera fonts to the windows one (size 8 suits me). when i was new to linux i thought linux fonts sucked, now im used to them and think windows fonts suck. its all a matter of getting used to

as for the desktop effects, search this forum of ubuntuguide.org for beryl or compiz
beryl and compiz are 2 things that do the same, altho beryl is better (imo)
