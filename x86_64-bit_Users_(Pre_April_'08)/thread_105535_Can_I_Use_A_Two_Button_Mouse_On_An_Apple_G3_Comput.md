### Post by MonolithTMA on 2005-12-18
[QUOTE=gconn77]Just curious if I can use a regular ps2 mouse on this Apple G3 B&W computer running Ubuntu 5.10?[/QUOTE]

Not without a PS2 to USB converter. Those round plugs; the one to your keyboard, and the one from the keyboard to the mouse, aren't PS2 plugs, they are ADB or Apple Desktop Bus. I think the B&W G3s were the last to use the ADB ports. Fortunately, you do have USB 1 ports as well, so you can use any USB mouse or a PS2 mouse with a PS2 to USB converter. a lot of mice and keyboards come with them, and you can usually buy them as well.

Edit: Here are some useful links about the various interfaces.

[ADB]("http://en.wikipedia.org/wiki/Apple_Desktop_Bus")
[USB]("http://en.wikipedia.org/wiki/USB")
[PS2]("http://en.wikipedia.org/wiki/IBM_Personal_System/2#Keyboard.2Fmouse_interface")
