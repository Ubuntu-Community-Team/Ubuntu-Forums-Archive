### Post by ssam on 2006-02-20
you probably can't. their linux version appears to be compiled for x86 processors. to make it work on PowerPC linux it would need to be recompiled from the source code. as it is not open source they wont let you have the source code, so you can't do this.

you could contact them and ask if they could make a powerpc linux version. or you may have to look for an alternative product.

possibly gnomemeeting would work for you. its a voice over IP program. it has recently changed its name to Ekiga, and got lots of new features. it will be in the next release of ubuntu in april. (you can try gnomemeeting now)
