### Post by Steve1961 on 2007-10-07
Use a live cd to delete your partitions, then install XP - but rather than letting it take the whole disk choose the advanced option and set the partition size yourself.  After that install Ubuntu in the free space.  That said, if you have an OEM recovery disk rather than a standard oem xp disk you might not have the option of setting the partition size.  In that case you would just need to install xp and then shrink the partition as needed.
