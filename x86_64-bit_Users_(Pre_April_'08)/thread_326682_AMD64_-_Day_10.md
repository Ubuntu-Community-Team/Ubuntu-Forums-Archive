### Post by Brian Boyko on 2006-12-28
Here's a quick update on my progress.  

The Good:

* I was able to install RealVideo using the download on the Real Web site; it worked.  
* I have finally been able to install 32 bit Firefox and with it, Flash 7. 
* I'm really digging the Gnome interface - I haven't had to touch it much to have everything where I need it.  I'd probably move to something different if I was using a smaller desktop resolution though.  Transparent panels, however, rock.  
* I've been able to play DVDs in Totem - most of my problems with video have been solved when I switched from Gstreamer to Xine based media codecs.  

The Bad:

* My printer, an HP Lazerjet 1020, worked before - now it doesn't, and I don't know why. Reinstalling doesn't help.  
* Although Kino's stability problems are gone, AND I've been able to encode into h.264 and Xvid with no problem, I'm still having trouble with encoding to Video_TS DVD MPEG2 in Kino, which is a must-have.  
* The computer refuses to boot if either my SanDisk Cruzer Titanium 2.0gB flash drive or 80 Gig USB external hard drive is connected.
