### Post by xizo on 2006-02-01
Are you using mozilla or firefox?  From the config description listed above it looks like you put the gplflash plugin into your mozilla plugin directory.  Check to see if there is a **gplflash.so** (not sure if it is exact name) file in /usr/lib/mozilla-firefox/plugins directory and also check the /usr/lib/mozilla/plugins directory.  If it is in /usr/lib/mozilla/plugins and you are using firefox you may need to soft link the file into your firefox plugin directory.

ln -s /usr/lib/mozilla/plugins/gplflash.so /usr/lib/mozilla-firefox/plugins/gplflash.so

Xizo
