### Post by nwillis on 2005-07-25
It may use an open protocol, but that doesn't make it open source.

In fact, as [this Slashdot comment](http://hardware.slashdot.org/comments.pl?sid=155055&cid=12998916) points out, the license on their web page is pretty explicitly NOT open source.

Likewise, offering a client for an open source OS doesn't make the software open source either.

But even if it weren't for the license, and if it weren't for their mention of clients for open source OSes, there is one simple way to see whether it's open source or not: 

Where is the source?

If there's no source available, it's not open source.  That's by definition.
