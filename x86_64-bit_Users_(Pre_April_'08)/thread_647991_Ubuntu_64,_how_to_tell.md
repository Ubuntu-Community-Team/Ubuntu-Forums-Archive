### Post by toupeiro on 2007-12-23
universally, you can run uname -m at the console  of any linux system to tell you the platform version installed.

i386-* = 32-bit 
x86_64 = 64-bit 

Unfortunately UNIX tends to vary a little bit.  I would type: 
uname -a (for the whole string, since the hardware is typically not x86 hardware)
