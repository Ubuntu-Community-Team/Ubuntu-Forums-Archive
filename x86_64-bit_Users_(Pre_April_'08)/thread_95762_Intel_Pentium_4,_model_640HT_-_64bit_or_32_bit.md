### Post by xelink on 2005-11-27
EMT is an marketing plot by intel. Consider it the same as AMD64 technology except slightly less coded for(imagine intel aying: Pentium 4, NOW WITH AMD64 TECHNOLOGY)

yes the CPU is 64bit. any chip in the 6XX series is. I'd suggest going 32 bit at this time though if you are new to linux, it is a pain to install things on the 64 bit version at this time due to lack of 3rd party work.

I have a REAL Athlon and I opted for the 32-bit version after installing the 64. Speed for the most part is the same, so dont' worry. In a year or two, 64 will be more supported, now that Intel finally joined the game.
