### Post by zedd_D1abl0 on 2008-03-22
Not a whole lot changes between 32bit and 64bit for the end user. I had my laptop running 32bit and my desktop running 64bit, and the configuring was the same, the install process was the same and there wasn't really a whole lot of changes between the two. As long as you can build from source (it's not difficult) you're all set because any package you need that you can't apt-get or aptitude, you can build from source and it'll be fully 64bit capable.
