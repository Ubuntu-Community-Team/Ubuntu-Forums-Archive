### Post by steveneddy on 2007-10-03
I'm a new user, also. New to 64 bit world, not Ubuntu. 

I installed ndiswrapper through a tutorial forgetting Flash for Fireox, and I've been able to compile Pidgen and RealPlayer and get all of my codecs installed.

I read about 64 bit for months, some of the stories nightmares, but most of them very good. I finally ordered the CD from ShipIt and it was here for 6 weeks before I had the time to install it. I started to install it as a dual boot, but I knew it was going to be good, so I backed up my stuff to a four gig flash drive and installed away.

Pidgen, Realplayer, firefox with Flash, Compiz-Fision from Trevinio's repos, Emerald from the Ubuntu repos. I installed the System76 drivers ( I have a Serval Performance less than one year old), but I don't know what good they are doing. Everything seems to be working, so far. USB, DVD, movies, sound (sounds better, BTW) firewire, SD Card reader, Suspend (finally), wireless and nVidia driver.

So far I am very happy with the "new" machine, and I will report back any hiccups I have.

Will Gutsy be going 64 bit at the update?
