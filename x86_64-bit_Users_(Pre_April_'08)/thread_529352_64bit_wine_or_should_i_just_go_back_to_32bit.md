### Post by Kilz on 2007-08-19
In reality there is no 64bit wine. 64bit wine would emulate (yes I know what wine stands for) 64bit Windows. Wine as is commonly installed is a 32bit windows application layer. For 32bit Windows Applications.
There are 64bit packages of 32bit wine. But it just makes installing the 32bit application easier. So if you have installed wine some other way, just leave it installed. You will gain nothing and waste your time. Why because the one you will remove is the exact same application as what you will install.
