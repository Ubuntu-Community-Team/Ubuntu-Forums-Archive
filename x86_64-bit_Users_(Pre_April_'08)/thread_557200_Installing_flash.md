There are a few problems with Swiftfox.
1. Its a proprietary application with a restrictive license. If you like Free as in Freedom software (as a lot of people running Linux do) you will want to stay away from it.
2. Its old, The last version was for 2.0.0.6.
3. The Swiftfox version designed for 64bit processors is a 32bit application. Most 64bit users like to use 64bit applications.
