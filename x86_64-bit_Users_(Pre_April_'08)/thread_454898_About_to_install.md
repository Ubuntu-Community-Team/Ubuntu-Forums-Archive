### Post by Cappy on 2007-05-26
1) You do not need to remove windows but if you use the Ubuntu CD you will need to resize your Windows partition and then let ubuntu install itself in the free space.
2) You don't install it in fat32 or NTFS, it creates its own EXT3 partition for only ubuntu in the free space.
3) Ubuntu will automatically install drivers, you don't need to. If you goto (System-->Administration-->Restricted Drivers) it will install them with a click.
4) Yes.

"Desktop Live" won't affect Ubuntu.

If you don't feel comfortable resizing your Windows partition you can try Wubi. 
[www.cutlersoftware.com/ubuntusetup/wubi/en-US/index.html](www.cutlersoftware.com/ubuntusetup/wubi/en-US/index.html)
