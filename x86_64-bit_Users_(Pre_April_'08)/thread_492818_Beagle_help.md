### Post by vladu on 2007-07-07
to force beagle into indexing, open up a terminal window (Applications->Accessories->Terminal) and type

```
$ beagle-shutdown
$ export BEAGLE_EXERCISE_THE_DOG=1
$ beagled
```

and if you need more info, check out [http://beagle-project.org/FAQ](http://beagle-project.org/FAQ)
