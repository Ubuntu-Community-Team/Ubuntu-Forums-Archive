1. I agree

2. New users should understand that Just Works does not mean that they wont have to do any setup or that the 32bit version is problem free for 64bit hardware owners. At this point in time the 64bit and 32bit versions are pretty close and the difference in setup isnt something to wory about.

3. That quote is from [a post over 1 year old.]("http://ubuntuforums.org/showpost.php?p=2086318&postcount=9") New users should not think that proprietary applications are impossible to run on the 64bit version. Sometimes the fix is so minor it isnt even worth the worry. An example of this is Flash.[ A 60 second script gets it working.]("http://ubuntuforums.org/showthread.php?t=476924")

4. Agreed a clean install is best. If you have room on your hard drive setup both the 32 and 64 bit. That way you can take your time setting up and if any issues make the 64bit version impossible to get working you still have the 32bit install.
