### Post by skylark on 2006-02-18
Hmm... I haven't tried the live boot cd, so I'm not sure why it's not working for you. 

Here are a few things you could try...
1. Turn off acpi in your BIOS and/or use the boot options: "live apci=off noacpi"
2. Try the 32-bit live CD - I've heard some have more luck with this even on AMD64 machines.

As for burning DVDs... there are lots of ways... GnomeBaker is quite popular, but I have personally had far more luck with a program called "Graveman"... when you do eventually boot up you have to install these apps to use them though (add applications should work I think).
