### Post by fireturd on 2008-01-19
Hello:

Thought that I'd post a quickie to help other members.  Installed 7.10 AMD 64 version.  Lots of fun getting the video card stable with the non-free drivers, but got that done (All hardware on the laptop works).  Usual hard time with java and flash, but the forums have lot of info and got both working.

Updated system to 4Gb.  Bios recognizes, but only 3 Gb show up.  More forum searching for fixes, most fixes relate to 32bit enable PAE or 64Gb support.  (not an option when compiling kernel for AMD64)  For fun, installed XP-64bit on a spare hard drive and same problem.  Believe it to be a BIOS issue with gateway.  They're no help.  "We only support the software sold with the laptop and don't sell 64Bit OS"

So, after all... Ubuntu is rock solid.  Compiz/fusion some cool eye-candy.  

Only drawback is the 3Gb limit (but thats a firmware/bios issue)

Cheers.

FT.
