### Post by cmost on 2007-01-22
I use x86-64 bit version of Edgy and I find that all the programs I'm used to "just work".  I have had to perform a few minor tweaks here and there for certain things such as Java but nothing outside the realm of a newbie.  Google and these forums are your friends.  The best approach, I found, was to do a fresh install of 6.10 x86-64.  Then, run the 'Envy' script to get your accelerated driver for nVidia or ATI.  Next, run Automatix2 to get all your essential, day-to-day goodies like a 32bit implementation of Swiftfox complete with plugins (flash, Java, etc.,) file sharing tools, and all available codecs, etc.  Finally, setup AIGLX and Beryl for the best possible visual experience.  This creates a fast, stable 64 bit OS environment that you will grow to love.  There's nothing significant missing from the x86-64 implementation of Ubuntu.  Enjoy!
