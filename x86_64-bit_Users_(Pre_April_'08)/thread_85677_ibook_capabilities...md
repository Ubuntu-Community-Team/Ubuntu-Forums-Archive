### Post by Rxke on 2005-11-05
I think 128 is workable... Not great, but workable. With a rather decked-out Firefox, Gaim, a tabbed terminal window and process-manager running simultaneously, I use 118Mb, and about 97 Mb swap memory. So that's *just* within your limits. The reason the live cd is so unuseable is probably because it won't use your HD as a swap memory, I think. An installed version of Ubuntu runs much faster. I recall testing the Live CD on my 192 Mb first gen iBook, and it was glacial, compared to the installed version, I thought it 'hanged' constantly, but it was just... glacial. 
But it confirmed what I wanted to find out: Ubuntu works, sound works, network works, etc... So i switched, and am a happy chappy!
