### Post by cmost on 2007-02-13
Welcome aboard!  Updating to the x86-64 version of Ubuntu has several benefits, but most of these are aimed at enterprise uses.  For the casual workstation user, you'll only notice a slight boost in performance.  On the other hand, you'll have bragging rights to your friends who are stuck on lesser, 32 bit operating systems.  If you do endeavor to upgrade to 64 bits, be advised that there are a few caveats to consider:  The main thing is that you'll likely need to run a 32 bit web browser (I recommend using Automatix2 to install Swiftfox and all its plugins) to really take advantage of flash and java (though there is a 64 bit implementation of JRE.)  If you need any help, these forums are your first stop.  Most answers and workarounds can be found here.  Good luck!
