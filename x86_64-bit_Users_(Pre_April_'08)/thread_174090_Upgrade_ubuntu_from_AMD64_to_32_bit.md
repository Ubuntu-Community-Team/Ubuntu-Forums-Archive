### Post by zakusage on 2006-05-14
Yes, you'd need to do a fresh install. However, as an AMD64 user of Dapper Drake, let me tell you that it is a much better experience then Breezy was. As well, with a few useful tutorials, I've been able to run any outstanding 32-bit program in the compatibility layer just fine, and completely stable. Now I'm running Firefox32, with Flash, Java, and MPlayer (with all codecs possible) plugins, along with Neverwinter Nights, Doom 3, Mupen64 (a Nintendo 64 emulator), RealPlayer, LimeWire, and Skype in 32-bit mode.
