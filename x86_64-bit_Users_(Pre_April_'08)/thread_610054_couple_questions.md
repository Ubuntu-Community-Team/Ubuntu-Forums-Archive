### Post by david_2001 on 2007-11-11
64-bit doesn't use significantly more memory than 32-bit.  For general desktop use, I've found 1Gb RAM to be an elegant sufficiency for either a 32-bit or a 64-bit system.  I haven't noticed any difference in stability with 64-bit but the cpu in my system runs 1 to 2C cooler (= quieter) for any given load.
