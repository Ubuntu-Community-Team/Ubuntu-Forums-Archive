### Post by linear on 2006-04-01
OK, what you have done doesn't give you a bootable CD. You need to burn the iso as an image, not as a data CD. There are instructions here: [BurningIsoHowto]("https://wiki.ubuntu.com/BurningIsoHowto"). If Disk Utility doesn't work for you, try [FireStarter FX]("http://www.projectomega.org/subcat.php?lg=en&php=products_firestarter").

### Post by jdp on 2006-04-02
Reboot and hold the 'c' key down at the chimes.  It should boot the CD if it was burned correctly.  It is highly varied on what quirks happen when booting the CD on various models of Mac.  G3 iMacs, eMacs, and all G5s seem to have certain quirks that need workarounds.

What model are you trying to work with?
