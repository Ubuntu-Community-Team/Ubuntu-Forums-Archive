### Post by eentonig on 2007-08-12
I would recommend against installing Beryl.
1.  Beryl is dead (development wise) and has merged with Compiz in 'Compiz Fusion'
The new project is more stable and has some nicer effects.
2. Beryl and/ord Compiz Fusion is very young software and under heavy development. So you should expect crashes when using it.

Ok, to answer your questions. There are some nice howto's on this forum. But since you're still looking for '.exe' files. I advice you to first read up on how to install programs in linux. terms as 'sources.list', 'apt-get', 'synaptic', ... should become familiar.
