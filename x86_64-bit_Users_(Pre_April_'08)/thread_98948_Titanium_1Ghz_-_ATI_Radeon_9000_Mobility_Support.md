### Post by ssam on 2005-12-04
i am affraid neither ATI nor Nvidia release drivers their linux drivers for powerpc.

you are in luck though, this card is well supoorted by the open source ati driver in xorg.

you should not need to do anything to make it work.

if you want to test it install planetpenguin-racer (see [http://help.ubuntu.com/starterguide/C/ch02.html](http://help.ubuntu.com/starterguide/C/ch02.html) for how to install software in linux), and run it from the games menu.
