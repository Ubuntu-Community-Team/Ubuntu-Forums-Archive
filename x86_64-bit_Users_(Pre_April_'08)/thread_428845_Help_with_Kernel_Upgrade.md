### Post by ronacc on 2007-04-30
if something doesn't showup or is greyed out in update manager it is because it is not yet ready for a safe upgrade ( not all depenecies or modules are available) best to wait for it to come down when it is ready. If it is a kernal update and not all parts are ready you could find youself in a terminal window at boot with no way to start X.
