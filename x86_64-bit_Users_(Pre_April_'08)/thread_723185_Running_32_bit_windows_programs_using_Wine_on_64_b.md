### Post by Cappy on 2008-03-13
Wine is 32-bit program in a 64-bit package (so that it sets up correctly on a 64-bit system). It cannot run 64-bit Windows programs.

Wine can ONLY run 32-bit Windows programs. Wine on 64-bit Ubuntu runs the same as it does on 32-bit Ubuntu.
