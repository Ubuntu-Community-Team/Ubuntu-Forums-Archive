### Post by dbenhur on 2006-06-21
I recently built a new system with X2 3800, MSI GeForce 6150 mb, and a pair of barracuda 300GB SATA2 drives.

I had issues with the graphical install: GParted would lock up hard.  The alternative text-mode install worked fine. I don't know what made gparted fail, but it could be some subtle interaction with the chipset and sata-2 drives.

I've been running stable for a couple weeks now with the two drives in a software mirror raid.
