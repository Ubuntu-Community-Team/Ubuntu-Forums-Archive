### Post by Shay Stephens on 2006-09-10
Crossover will run PS7 a lot better than wine currently, I am using crossover at the moment.  vmware or qemu would allow you (presumably) to install and run PSCS2, but I have not tried that yet.

### Post by FluffyElmo on 2006-09-10
I've had better luck running Photoshop (and Fireworks/Dreamweaver) in VMware than WINE. There were always small glitches with WINE, Dreamweaver especially, that you don't see in VMWare.

Make sure you have lots of RAM though, VMWare is memory hungry even before you run programs in it. WINE uses no more than the program needs.
