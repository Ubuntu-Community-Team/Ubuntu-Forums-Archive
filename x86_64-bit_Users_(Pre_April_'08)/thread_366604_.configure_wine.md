No, you dont need to install the 32bit version. Wine is a 32bit compatibility layer for linux. It runs 32bit windows applications. But you are always better off running native applications. So if there is a linux build of that application give it a try. Even if you needed a 32bit linux application (there are a few you might need) you can install it to the 64bit system with a little setup.
