### Post by mad_max0204 on 2008-02-14
Since this is a major problem on almost every 64bit distribution I wanna share a thing I managed to do last nite.
I moved my system to a new HDD that arived just yesterday and I didnt want to migrate but installed a fresh 7.10 64bit install.
Since nspluginwrapper gave me nothing but trouble after update I did the foloving:
Installed fresh, added repositories, updated (didnt install anything before that).
After update of about 280mb I went on installing flash on 64bit Firefox 2.0.0.12.
(Mozilla/5.0 (X11; U; Linux x86_64; en-US; rv:1.8.1.12) Gecko/20080207 Ubuntu/7.10 (gutsy) Firefox/2.0.0.12)
Then I just typed

**sudo apt-get install flashplugin-nonfree**

and it worked like a charm. I had some bugs with nspluginwrapper before but now everything works like it should.
I dont know if this will work 100% for everyone but if anyone tries should leave a msg here to confirm.

Cheers
