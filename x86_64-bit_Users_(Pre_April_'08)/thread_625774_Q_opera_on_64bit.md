### Post by fineas on 2007-11-28
The simpliest way to deal with installation of opera on 64bit, in my opinion, is this:
1.Unistall all previous versions of opera
2. Go to [ftp://ftp.opera.com/pub/opera/linux/950b/final/en/x86_64/](ftp://ftp.opera.com/pub/opera/linux/950b/final/en/x86_64/) and sdownload the .deb package (it's opera 9.5beta 64bit edition)
3. Install it with gdebi package installer (just left click it twice)
4.Enjoy
