### Post by Kilz on 2006-08-13
> **topic58 said:**
> Is it possible to install 32-bits Dapper on a 64-bits Intel P4? And then run w32codec, flash in Firefox, Skype with no problems? OR should I buy a 32-bits Intel P4 instead to avoid problems with w32codec, flash in Firefox, Skype?

If you buy the 64bit chip you can run either the i386(32bit) or amd64(64bit) operating system or OS.
W32codec, flash in Firefox, Skype are all avialable in the 64bit OS, they just require some manual setup. Basicly following a cut/paste howto or using a automated setup script. My signature has one for Firefox/flash. 
Alternatly you could install the 32bit os, but its like buying a corvette and putting skinny 13 inch tires on them because they are easy to get. They will maybe fit, but your not going to get all the performance thats possible.
If you buy the 32bit chip you can only run the 32bit or i386 OS. If there isnt a big price difference, get the 64bit chip, imho they perform better.
