sadly, 64bit is pretty much based upon AMD systems, so optimizing for an Intel based processor is hard.

none of the optimizations will help the system, so don't bother with them. with Intel based systems, pretty much a fresh install gives you the most you can get out of them. Intels run 64bit pretty well, so don't worry about it.
