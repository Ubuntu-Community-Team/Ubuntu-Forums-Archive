### Post by ssam on 2006-02-06
this is quite a problem if you need to use flash on PowerPC linux.

it would probably be quite easy for macromedia (adobe) to recompile the flash plug in for PowerPC, but after many requests and petitions they have not. as it is not open source no one else can do it for them. you could try convincing them that PowerPC linux is a worthwhile platform to port to, but you wouldn't be the first. maybe things will change post merger.

the alternative is to use an open source implementation of the flash plug in. unfortunatly these do not play all flash files perfectly. i did a search in synaptic (system -> admin -> synaptic)and libflash-mozplug looks like it could be the right thing, but someone may know better. gnash has been in the news recently, but i dont think its in ubuntu yet.

my local solution is to not bother with flash. it reduces the amount webpages flash at me, and stops them making noises. not everyone likes this solution.
