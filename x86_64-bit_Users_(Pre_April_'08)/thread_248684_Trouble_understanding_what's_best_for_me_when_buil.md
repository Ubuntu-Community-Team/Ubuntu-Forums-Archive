64bit Ubuntu isnt limited. The only limits is the ability of the user to setup the system. [I made a post the other day explaining this.]("http://www.ubuntuforums.org/showpost.php?p=1447132&postcount=2") There is a lot of FUD and misinformation floating around. Before believing anything make sure the person is actualy running a 64bit system and didnt fail, and then blame the software.
SMP is needed for multi processor systems to use more than one processor. But all 64bit kernels are smp. They are not labled as smp and non smp because there is no non smp 64bit kernel in Ubuntu.
