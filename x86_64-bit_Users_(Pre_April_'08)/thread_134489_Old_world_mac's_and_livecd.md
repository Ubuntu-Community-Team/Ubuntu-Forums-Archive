### Post by jdp on 2006-02-22
Old World are Macs that have all boot code stored in a ROM chip.  New World are Macs that have basic boot code in ROM, but lod the rest to RAM from the HDD at startup.  The easiest way to tell the difference is if it has built-in USB or not.  No Old World Mac shipped with USB from the factory.  All New World Macs do.  Every Mac from the B&W G3, iBook, iMac, and PowerBook Lombard on up are New World.  New World machines can boot the Ubuntu CDs directly by holding **c** at boot up.  Old World needs the BootX program and an install of Mac OS.

I dunno about your issue, but the eMac is certainly NewWorld.
