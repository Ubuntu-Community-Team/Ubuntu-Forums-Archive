### Post by jdp on 2006-04-08
If the tech is setting up a modem or router, ask him to make sure it has DHCP enabled.  Then, when you plug your iBook in go to the Networking program in the Admin menu.  From there you can select eth0 or eth1, depending on if you have an AirPort card installed or not.  Then instead of a static IP turn the option to DHCP and it'll get all the setting for you when you close the panel.  If you've ever done it in Windows or Mac OS, it'll basically the same idea with differnt software to do it.

If it's wireless it's a tad different to setup, but it's still pretty easy.  If you ask the tech nicely they might even like to get the learning experience of helping with it.

Setting it up in Linux isn't hard, just different.  That's the main idea to keep in mind.

Will it be hardwired or wireless?
