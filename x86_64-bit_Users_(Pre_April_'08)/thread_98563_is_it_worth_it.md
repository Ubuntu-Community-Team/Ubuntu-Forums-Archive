### Post by RAOF on 2005-12-04
Well, the 64bit chips are **also** faster at running 32bit code, so if you don't want the (slightly) added hassles of a 64bit kernel, the AMD chips are still a fine option.

### Post by grsing on 2005-12-04
I'd go for the AMD, they're almost always cheaper for the same performance, and, as RAOF mentioned, if you don't want to hassle with Ubuntu 64, you can just install Ubuntu 32 and it will work just fine (that's what almost everyone who is running Windows on a 64 bit CPU does, since Windows XP 64 isn't particularly great at running much of anything).
