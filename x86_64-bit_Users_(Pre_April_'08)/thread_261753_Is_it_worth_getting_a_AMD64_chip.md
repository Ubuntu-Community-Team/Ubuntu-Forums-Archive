Whatever new processor you buy, it's (almost certainly) going to be capable of running the AMD64 (or x86-64) version of Ubuntu.  Both new Intel & new AMD processors support x86-64.  Whether or not you want to install the 64bit version of Ubuntu is a different matter; the chips will happily run **either** the AMD64 version **or** the i386 version.

Which one you want to install depends on what you want to do.  If you want everything to Just Work(tm) (as far as it is possible with linux), you probably want the i386 version.  If you don't mind doing a minimal amount of work, the x86-64 version should run faster.
