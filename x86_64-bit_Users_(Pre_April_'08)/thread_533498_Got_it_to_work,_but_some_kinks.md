### Post by bdoe on 2007-08-24
Beeps?  The only beep you should be hearing is the BIOS POST complete beep, unless you've got hardware problems (and if that were the case, your computer wouldn't even boot).

Once you've run the install icon and everything's done installing, your computer should boot straight to the GRUB bootloader, and all should be well.  In fact, if all went well, you should see a bootmenu, and you should see an entry for Windows Vista.
