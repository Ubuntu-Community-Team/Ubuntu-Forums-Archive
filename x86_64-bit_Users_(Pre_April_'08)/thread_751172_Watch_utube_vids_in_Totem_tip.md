Hi, 

Rather than go to utube and have to install 32bit flash, here's a simple solution. Install totem-xine and remove totem-gstreamer.

Now open up totem and activate the utube plugin in the Edit--->Plugins menu, now in the menu on the left side you will see a option for utube which you can search for you favorite vids. It's a real kick **** feature and great for 64bit users if your having issues with flash.
