### Post by mrscintilla on 2005-05-18
It's easy to switch if you have a dedicated /home partition.  You save all important data there, and when you run ubuntu 32-bit installation CD, you will have an option to mount that partition as whatever ( in this case /home), and "not format" the partition.

I tried it when I switched from 64-bit to 32-bit. It only formatted the partition I wanted it to format.  It kept all my data, session settings, bookmarks, and etc.

Great installation routine!!
