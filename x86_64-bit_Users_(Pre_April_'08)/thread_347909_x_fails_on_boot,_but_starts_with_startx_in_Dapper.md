### Post by bender5788 on 2007-01-28
Ok your best bet is to post the contents of your xorg config file which should be located at /etc/X11/xorg.conf regardless of the distro your using normally.
I may not be the best but ive had issues as-well so maybe if you post it and a much more detailed set of system specs i can help you.
Also if you want the command to make it easier just in case 
Just use
```
sudo gedit /etc/X11/xorg.conf
```

### Post by Anaximander Thales on 2007-01-28
Doh -- ID 10 T error.

I forgot to undo changes I had made to gdm.conf-custom.

Problem solved.
