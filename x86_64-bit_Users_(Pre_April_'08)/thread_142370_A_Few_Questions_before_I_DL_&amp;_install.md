### Post by ssam on 2006-03-10
did you try holding the 'C' key as the computer started?

some older cd drivers arn't so good with cdr. you could try to get hold of one of the [ship it cds]("https://shipit.ubuntu.com/"). they take a while to arive, but you might find someone who'll post you one on [https://wiki.ubuntu.com/forum/GiveAway](https://wiki.ubuntu.com/forum/GiveAway) (i can post you one if you are in the UK).

the trouble with a network install is that you still need something to boot off, and the ibook only has the cd drive. it might be possible to take out the hard disk and plug it into another computer to install, but thats a bit extreme.

have a look at [https://wiki.ubuntu.com/HardwareSupportComponentsWirelessNetworkCards](https://wiki.ubuntu.com/HardwareSupportComponentsWirelessNetworkCards) for wireless cards known to work (or not work as the case may be)
