I however recommend that when you go to install your Ubuntu AMD64, that you set a different partiton for / , /home, I heard that it makes upgrading from version to version a lot more simpler.  

Anyhow, if you need any help once you get your system up and running, just ask here.  This side of the community is very tight, and everyone is helping each other all the time.
