2. You have 2 options for running windows games.
 A. Wine = Its free, but getting some games to run is a real pain. You will have to do a lot of configuration and reading.
 B. Cedega = It costs $15 for a 3 month subscription, during that time you get support and are able to get updates that add new games. After the 3 months you can let the subscription lapse and the Cedega program will still work. Just make sure to have backup copies in case you have to reinstall. Installing games is click, click install , play, with minimal configuration.
