### Post by juicemansam on 2006-07-26
I don't know if it really show's off 64-bit hardware, but I'm currently hooked on [Nexuiz](http://www.nexuiz.com). I also played UT2004 alot when I had my 32-bit system running, but now it's there, but I don't play it much, at least not like Nexuiz. It's probably due to a crackly sound problem, which only happens in UT2004, or that my teammates aren't around much.

Just so you know, Nexuiz is a multi-mode, multi-player, FPS game that runs on a modified and enhanced Quake1 engine. It's really fun game :D. Although, it might run a little slower than one might expect. One of my teammates has a Dual Opteron 165 (I think) set-up with 2GB RAM and an Nvidia 7950, and said that with all settings maxed it was glitchy. I was playing it with everything on the lowest setting and getting pretty good performance until heavy action and guts filled the screen, but that was the case right up until I upgraded to my current system; Pentium D 805, 1GB RAM, SLI'd 6600 LE's. I'm currently using default settings, which look pretty nice :D.
