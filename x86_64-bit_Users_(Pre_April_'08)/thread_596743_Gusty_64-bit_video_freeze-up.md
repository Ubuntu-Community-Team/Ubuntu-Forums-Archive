### Post by Jhongy on 2007-10-30
downgrade the nVidia drivers. Look for a tool called 'Envy' by Alberto Milone. It makes uninstalling nVidia drivers easy (if you try manually installing different versions, it rarely works neatly). 

Using Envy, uninstall the nVidia drivers, then install the oldest version Envy allows you.

The newest nVidia drivers... to put it mildly... suck on Geforce 6100s & GeForce Go 7x00s
