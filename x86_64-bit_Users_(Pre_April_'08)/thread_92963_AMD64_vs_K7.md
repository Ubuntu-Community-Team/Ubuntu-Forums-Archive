### Post by RAOF on 2005-11-20
AMD64:
Pros: faster - the amd processors seem to run 64bit code 10~20ish percent faster than 32bit code.  How much you notice this depends on what you run.
Geek cred - add one Red Geek Star card to your hand.
Cons: slightly less supported apps - flash, java browser plugin are more difficult to get working.  Wine, & particularly Cedega, just flat doesn't compile.  At least for me.


K7:
Pros: More things will Just Work(tm).  Cedega should compile.
Cons: Marginally slower.

If you're in any way interested in Windows gaming on Linux.  K7 is the way to go.
