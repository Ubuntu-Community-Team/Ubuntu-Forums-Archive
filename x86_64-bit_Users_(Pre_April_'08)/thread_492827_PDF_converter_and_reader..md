### Post by incubus on 2007-07-05
I'm not sure about Gnome, but I assume it's the same thing.  On KDE I can go to print and select "print to PDF", which converts whatever document that's open to PDF.  Also, KPDF is not a bad reader.  It can convert pretty well from PDF to text, as long as the document was generated from text, not image.

To extract/insert/delete/move pages, there's this tool:
[http://www.kde-apps.org/content/show.php/KPDFTool?content=33194](http://www.kde-apps.org/content/show.php/KPDFTool?content=33194)

which I myself never tried, but it has good reviews.  You can find other tools on that website.

I'm sure someone will come and point you to the Gnome alternatives.
