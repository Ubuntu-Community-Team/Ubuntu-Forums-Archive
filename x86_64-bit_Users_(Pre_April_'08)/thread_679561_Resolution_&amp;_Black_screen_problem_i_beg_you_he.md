### Post by Strike_105x on 2008-01-27
Managed to resolve my problem :D i'm so happy seems that by using commands like sudo aticonfig --initial or sudo aticonfig --overlay-type=Xv some things get deleted in the xorg.conf so decided to manually edit it in an editor (i used nano first because i coudnt get pass the black screen) good thing i had a backup of the original config before i configure it so what i did was to delete all resolutions until 1600X1200 in the monitor section and add the options:

	Option	    "VideoOverlay" "on"
	Option	    "OpenGLOverlay" "off"
