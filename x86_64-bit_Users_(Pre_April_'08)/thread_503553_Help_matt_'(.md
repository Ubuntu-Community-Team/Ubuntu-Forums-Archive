### Post by Kilz on 2007-07-18
There are some things the fine post above mine forgot to touch upon. Since you posted in the 64bit section I will assume you are running the AMD64 version of Ubuntu/Kubuntu/Xubuntu. If not, and you just have 64bit hardware , but installed the i386 version of one of the buntus it does not apply.
To start off each version of Ubuntu has its own quirks and things you need to setup. None is automatic, even if it were IMHO the knowledge you would gain is vastly more important. Being that you need to do some setup specific to the version you installed. If you installed the AMD64 version you are posting questions in the correct place.
The post above links to some great general info that should work, but when it doesn't. Please post to this section. If you installed (or will install) the i386 version dont post to this section.
Lets tackle the First thing on your list , First. Flash and some of the plugins for Firefox. The browser in the 64bit version is 64bit, the commercial , proprietary plugins like Flash are 32bit because the company that owns them has not released 64bit versions and will not give linux the code for them.
There are ways to make flash work. But you now have a choice to make.
1. Run a 32bit Firefox and all the plugins. (link to a install script in my signature)
2. Install the nspluginwrapper to allow the 32bit plugin to work with your 64bit browser. (link to install script in my signature)
If you just need flash and the other plugins are not needed, install nspluginwrapper (easy install script, click on it , wait 30sec, your done). If you need java, install the 32bit Firefox. The script will walk you through asking yes/no questions.
