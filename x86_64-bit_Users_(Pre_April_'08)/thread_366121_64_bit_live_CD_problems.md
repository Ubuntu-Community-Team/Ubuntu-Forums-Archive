The live CDs will always be very slow. The reason is because everything has to load from the CD drive. Your hard disk is many times faster than the CD drive, so if you do a real install to the hard disk it should run normally. 

Regarding the ugly boot up, there are updates that are not on the live CD image that will fix those problems. 

If your live CD is Edgy you might want to try Dapper instead. Dapper doesn't have quite all the cutting edge features that are available in Edgy, but is far more reliable.
