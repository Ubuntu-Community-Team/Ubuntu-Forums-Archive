### Post by ajgreeny on 2007-03-12
What do you mean by  OEM user account?  Do you mean the user name and password you set up when you installed ubuntu?  If so you will need to start up in recovery mode and make a new user with sudoers rights, ie, the ability to do things as root using your user password.

Alternatively, if you have only just installed and have few, or no files to backup, it may be easier and simpler to just start again and reinstall, and put this down to experience; it's something you won't do again, I'm sure.
