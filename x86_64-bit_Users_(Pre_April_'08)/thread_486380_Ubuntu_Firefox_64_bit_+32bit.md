As long as you have any version of Firefox open, be it 32bit or 64bit. If you click on a firefox launcher or it is opened in some other fashion, it will open another windows for the already open version.
Example - I have 64bit Firefox open, I click on the Firefox32 launcher in the menu. Firefox32 will not open , 64bit Firefox will.

If you would like a local file to open with any application. Right click on the file and choose opens with tab. Select the application. It becomes the default for that file type. But if you have 64bit firefox open at the time, it still wont open an instance of 32bit Firefox.
