### Post by FrzzMan on 2005-06-26
[QUOTE=DarkManX4lf]i see that ubuntu installs the generic amd64 kernel how do I install the k8 kernel? is there a guide somewhere? i tried to search this forum but i only found posts of ppl asking if the k8 was good or not....[/QUOTE]
 1. Go to menu System -> Administration -> Synaptic Package Manager
2. Use the search button for string "amd64-k8"
3. Mark for install this package "linux-amd64-k8" and choose to accept all dependencies
4. Click Apply on the menu
5. Reboot as needed
