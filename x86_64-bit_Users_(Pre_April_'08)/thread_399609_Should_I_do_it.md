First, for threads that discuss your main question, go here:

[http://ubuntuforums.org/showthread.php?t=368607](http://ubuntuforums.org/showthread.php?t=368607)

As for Wine and games, a lot depends on exactly which games you want to play. Some work better than others. Installing Wine is certainly possible on 64-bit Ubuntu, although you have to do some finagling. 

Ditto for flash on Firefox-64, although many people prefer the simplicity of installing 32-bit Firefox. Personally, I got it installed in 64-bit Firefox, which made it also work in 32-bit Opera. Later I decided I didn't like it, so I removed it from the plugins in 64-bit Firefox. And after doing so, I discovered that it still worked in Opera. So now I have a situation that suits me fine -- 64-bit Firefox without flash, and 32-bit Opera with flash. If I want to see something on youtube, I just do it with Opera.

I should also add that installing 32-bit apps in 64-bit Ubuntu is not as easy as it could be. I've been messing with Fedora 7 test2 on a new machine lately, and it is a snap to install 32-bit apps on it. But there is a downside, because in order to support 32-bit apps Fedora ends up installing a lot of 32-bit stuff even for some 64-bit apps -- sometimes doubling the amount of installed code. I prefer Ubuntu's approach, but there is plenty of room for disagreement.

Since you have a new machine to play with, I would suggest you try several different distros. You can always wipe one out and try another one until you find one you like best. Choice is all what Linux is about.
