### Post by Ptero-4 on 2006-05-17
Install Apple X11 under your OSX, then run it once and then copy the XF86Config-4 file to your home folder (you don't need to keep X11, you only need the config file) and then from within ubuntu copy the XF86Config-4 file and copy the needed parts of it over to your ubuntu's /etc/X11/xorg.conf.

P.D: What you did was to have OSX create a functional X11 config file for you, since ubuntu uses X11 the config file you got works nice on ubuntu.

### Post by RIVE on 2006-05-18
The problem is that the VertRefresh and HorizSync lines aren't in the xorg config file, you have to add those lines like this:

Section "Monitor"
Identifier "Color LCD"
Option "DPMS"
HorizSync 28-51
VertRefresh 43-60
EndSection

If you check your configuration those lines are missing.

Hope this will help you
