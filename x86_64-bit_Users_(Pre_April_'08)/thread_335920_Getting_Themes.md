### Post by kuja on 2007-01-11
There don't seem to be too many themes in the repos, I'll list off which ones I find below, aside from that, I'm not sure where else than kde-look to go. 

Window decoration styles
kwin-style-crystal - semi transparant window decoration for KDE
kde-style-polyester - The Polyester widget style for KDE
kwin-baghira - KDE theme for Apple junkies :)
kwin-style-alphacube - Alphacube window decoration for KDE
kwin-style-blended - a window decoration theme for KDE
kwin-style-powder - Powder plasmaoid window decoration for kde
kwin-style-suse2 - KDE window decoration from SUSE 9.3

Widget styles
kde-style-comix - Comix flat style for KDE
kde-style-klearlook - The Klearlook widget style for KDE
kde-style-lipstik - Lipstik style for KDE3
kde-style-polyester - The Polyester widget style for KDE

Artwork
kde-wallpapers-lineartreworked - Some reworks of the lineart background that comes with KDE 3.4
kdeartwork-emoticons - emoticon collections for KDE chat clients
kdeartwork-misc - various multimedia goodies released with KDE
kdeartwork-style - widget styles released with KDE
kdeartwork-theme-icon - icon themes released with KDE
kdeartwork-theme-window - window decoration themes released with KDE
kdeartwork - themes, styles and more from the official KDE release
kdewallpapers - wallpapers released with KDE
kscreensaver-xsavers - KDE hooks for standard xscreensavers
kscreensaver - additional screen savers released with KDE
kde-icons-mono - a monochromatic icons theme for KDE
kde-icons-crystal - Crystal icon theme for KDE
kde-icons-crystalclear - Everaldo's "Crystal Clear" icon theme for KDE
kde-icons-gorilla - Yellowish gorilla icons for kde
kde-icons-korilla - Blue version of the gorilla icons for kde
kde-icons-noia - Noia icon theme for KDE 3
kde-icons-nuovext - nuoveXT icon theme for KDE
kde-icons-nuvola - Nuvola icon theme for KDE

I wouldn't doubt if there are more in there, but that's what I could find just skimming through.
