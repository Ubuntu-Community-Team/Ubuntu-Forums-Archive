### Post by Leang on 2006-02-09
I recently installed Ubuntu on my 500 mhz G3 iBook 12" and it runs pretty nicely after tweaking, but there are a few things you should know. After my new install, sound was kind of buggy. I fixed it using this [thread]("http://ubuntuforums.org/showthread.php?t=79597"). Also, sleep didn't function correctly. iBook was able to enter sleep, but would freeze on trying to wake up. I found a solution in this [thread]("http://ubuntuforums.org/showthread.php?t=2368"). I followed the instructions in the second post, but changed all instances of dbus-01 to dbus. Though these answers fixed my problems, it seems as if they didn't work for everyone though, so be forewarned. I have yet to try Flash, but I have gotten mp3 files and avi files to play. Overall, Ubuntu runs about 1.5X faster then Mac OS X. Avi files were extremely choppy in Mac OS X, but playback is actually decent under Ubuntu. Battery life remains about the same too.  Good luck :).
