[http://www.linux-gamers.net/modules/wiwimod/index.php?page=HOWTO+Steam&back=HOWTO+INDEX+Wine+Games](http://www.linux-gamers.net/modules/wiwimod/index.php?page=HOWTO+Steam&back=HOWTO+INDEX+Wine+Games)
> 2.3. Downloading Microsoft core fonts

Steam requires the tahoma.ttf font. It is included in the Microsoft core fonts package. The package name depends on the distribution.
Alternatively google for "filetype:ttf inurl:tahoma", download it and put it into your ~/.wine/drive_c/windows/fonts directory.
