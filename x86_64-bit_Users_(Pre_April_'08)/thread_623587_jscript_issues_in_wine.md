### Post by michaelzap on 2007-11-26
Office 2007 currently cannot be installed via Wine:
[http://appdb.winehq.org/objectManager.php?sClass=version&iId=7797](http://appdb.winehq.org/objectManager.php?sClass=version&iId=7797)

Search the applications database on that site for the specific Adobe programs you're trying to install  (and possible solutions).

If you really need to run Windows software that won't work in Wine (or Crossover), then try running Windows in VMWare or VirtualBox (or just dual boot).
