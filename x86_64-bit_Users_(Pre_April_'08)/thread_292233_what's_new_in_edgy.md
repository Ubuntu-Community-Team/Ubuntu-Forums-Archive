You might want to be aware of what's not in edgy; if you ever used the disks program (accessories>administration>disks is the path I think) then you will not find it in edgy. If this is something you find useful, then get pysdm from synaptic, and install it. (it's a little more useful, in my book, because you can mount a partition/drive using it)
