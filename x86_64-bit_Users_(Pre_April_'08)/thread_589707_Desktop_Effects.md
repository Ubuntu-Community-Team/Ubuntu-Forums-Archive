### Post by MighMoS on 2007-10-24
Yes.  Pressing alt+F2 will bring up the run dialog, and from there you can type in "compiz --replace" (that's compiz a space, two dashes then replace, don't type in the quotes) which will just go ahead and turn on desktop effects.  If you haven't already, I would submit a bug on launchpad.net about the latest kernel not working.
