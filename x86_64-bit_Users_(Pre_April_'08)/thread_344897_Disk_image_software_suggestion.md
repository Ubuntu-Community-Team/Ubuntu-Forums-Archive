### Post by FluffyElmo on 2007-01-23
In the past I've just done it from the command line using dd and other commands to zero out and compress the image...I forget the exact commands since I just use Ghost for Linux now. You boot from the CD and it will image your disk and store it on another drive or elsewhere on the network. [http://www.feyrer.de/g4u/]("http://www.feyrer.de/g4u/")

System Rescue CD also has imaging software (Partimage) included so that might be another option. [http://www.sysresccd.org/]("http://www.sysresccd.org/")
