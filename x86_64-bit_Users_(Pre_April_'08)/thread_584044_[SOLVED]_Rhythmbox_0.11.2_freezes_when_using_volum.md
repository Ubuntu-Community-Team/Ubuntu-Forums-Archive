If I wait a few moments (1-2 minutes) it will unfreeze but I can't make good use of the volume controls regardless.  I had the SAME issue under 7.04 as well.

The strange thing is that it all works perfectly after the fresh install until I import my rather large music folder located on a different drive (NTFS).

So I removed Rhythmbox 0.11.2 and installed Banshee which always works perfectly.  But then when I hit the media button on the keyboard, it tries to launch Rhythmbox 0.11.2 which isn't there any more.

I reinstalled Rhythmbox 0.11.2 thinking this might clear the issue, but it's still borked.  How can I either fix it or kill it all the way?  I don't mind using banshee all the time, but I don't want my keyboard calling up a program that is no longer present.
