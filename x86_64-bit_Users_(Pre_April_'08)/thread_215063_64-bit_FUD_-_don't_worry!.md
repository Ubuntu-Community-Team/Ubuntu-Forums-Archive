### Post by professor_b on 2006-07-13
I just recently built my own budget linux box with an asus A8N5x motherboard, 1g ram, 160g sata hd, geforce 6200, and a single-core AMD64 3500+. I was contemplating putting the 32-bit version of Ubuntu on the box to avoid the troubles that supposedly come with the 64-bit system, but I figured I'd throw caution to the wind, and run with the 64-bit. Plus, until Ubuntu came along, Gentoo used to be my favorite distro, so I'm not shy about mucking around under the hood and breaking/fixing stuff.

Anyway, I'm posting here to say that the occasional FUD -- fear, uncertainty, and doubt -- I've been seeing about 64-bit Ubuntu just isn't justified. I installed the 64-bit ubuntu, used 64-bit automatix, and now I have just about everything I need. Granted, I don't run wine or anything else exotic, but so far, everything *just works*, including flash and java in Swiftfox. The entire system is very snappy and responsive and exteremely stable, and I've spent perhaps a total of a few hours getting everything just to where I like it -- about the same amount of time that I would have spent getting everything setup on the 32-bit version too.

In sum, if you're thinking about going 64-bit but are scared, let me reassure you: it isn't as bad as some make it out to be. Again, this is my experience with a box that I built for maximum compatibility with Linux, and YMMV, but from my experience, 64-bit Ubuntu is mature enough to use without much trouble.

Finally, thanks to everyone on this forum -- the tips and posts here have been invaluable.
