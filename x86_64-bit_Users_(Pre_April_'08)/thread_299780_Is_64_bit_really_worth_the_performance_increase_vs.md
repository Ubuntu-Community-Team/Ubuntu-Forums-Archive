No, not really.  Just for desktop work (email, web, office stuff) computers haven't really got any faster in the last copule of years.  That sort of work doesn't really require any more processor power than we had 5 years ago.

That said, audio/video stuff, especially encoding, **always** wants more processor power :).  But if you're not doing video work, eeh.  Normal desktop performance is much more dependent upon available RAM, disc access speed, video performance, etc than raw CPU power.
