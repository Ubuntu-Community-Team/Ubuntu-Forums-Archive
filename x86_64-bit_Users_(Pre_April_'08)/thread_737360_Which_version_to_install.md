### Post by jespdj on 2008-03-28
> **arbeck said:**
> I'm curious as to which version I should install.  Should I go with 8.04, even though it's in beta?  It seems silly to install 7.10 and then just upgrade the OS in a few weeks.  If I go with 8.04, is the 64bit version preferable?
8.04 is not yet out so you should not expect it to be 100% stable and working. I am using 64-bit 8.04 beta on one of my computers and it works fine, but there are still some bugs on which people are working. If it's a priority for you to have a very stable system, then wait until 8.04 is finally released.

Whether you should go for the 32-bit or 64-bit version: In my opinion, if you have a 64-bit capable processor, there's really no reason to use the 32-bit version.

See the sticky topics in this forum for more information about 32-bit vs. 64-bit.
