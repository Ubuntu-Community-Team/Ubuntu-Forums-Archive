### Post by Kilz on 2007-09-06
I truly wish you all the luck in the world getting any features into any version of Ubuntu. But from experience I have learned a few things
1. The developers never read wish lists.
2. The developers want little if anything to do with users. (They dont use the forum much and have made their mail list impossible to join)
3. Features for the 64bit version are low on the developers list.
4. Eye candy for the 32bit version always tops the list.
5. If you dont spell out in exact detail exactly how to do each step to bring some feature to life, it will be ignored. (If users could figure that out we wouldn't need developers)
6. Functionality for 64bit will be ignored over Eye candy.
7. How you use an application can be used for shooting it down. (Even if its a FOSS application)
