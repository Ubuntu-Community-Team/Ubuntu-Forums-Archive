### Post by jeslek on 2006-08-03
I installed the 64-bit Dapper on an A8N-VM CSM with no problems. It booted into X using the free nv driver (limited at the time to 1024x768 ), but installing the proprietary drivers let me raise it to 1280x1024.

I'm using the DVI output from the onboard gf6150 - yet to try the CRT, but I assume it'd work just as well. I've also yet to update the BIOS at all so it's whatever revision it shipped with back in December.

The only issue I've had, which was true for my old gf2 as well years ago, is sometimes when I switch to a console from X the screen is black. No clue exactly how to fix that one, but it's something I do so rarely I've also not bothered researching it.
