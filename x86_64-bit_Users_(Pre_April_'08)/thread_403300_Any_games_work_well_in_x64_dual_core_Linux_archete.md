### Post by russell.h on 2007-04-12
You should be able to run most anything on 64 bit, just that if it isn't open source (which rules out the option of just compiling it for 64 bit) and a 64 bit build isn't available then you need to get 32 bit versions of all its dependencies (or at least that is my understanding).

That is really only a problem for a few media related things, mainly flash and a few win32 codecs and whatnot. Plus some proprietary drivers. Most everything else you can just compile for 64 bit, or if its in the repositories then you don't even have to do that.
