### Post by in_rainbow on 2007-11-05
I had the same problem.

1. Install KGet

2. Install Konqueror

3. Install the FlashGot Firefox extension.

4. Log into your rapidshare premium account in Konqueror. (KGet shares cookies with Konqueror.)

5. Configure FlashGot to use KGet as download manager.

6. Right click on the list of rapidshare links in Firefox and got to 'FlashGot Selection'.

All the links should be added to Kget and automatically start downloading.

### Post by malusx on 2008-01-07
Hi all,

i normally don't  post anything, but I've found a very EASY and comfortable solution, so I want to share it:

Quick instruction:

"Down them all" plus enabled direct downloads in rapidshare plus web-based email account to created real links out of text.

In Detail:

1. Get rid of all download managers (they didn't work for me)
2. install "Down them all" (dTa) (extension for firefox)
3. make sure you have activated direct downloads in rapidshare (RS).
How do you know you have? Paste a RS a link in your firefox adress-bar and hit enter. If the download starts, then you've done it right. Otherwise check in RS options.
4. copy the links you want to download (normally they are no real links but text, so you have to make links out of them).
[with this step you create links that are displayed in firefox instead of plain text, so that dTa can recognise them. if you have already real links, i.e. you can click on them and the DL starts, you can skip this]
log in your web-based email account and create a new email.
paste all the text-links (e.g. [http://rapidshare.com/files/myfile.rar](http://rapidshare.com/files/myfile.rar) etc.) into the mail.
if you've done so, your email-acc will have created "real" links. if not, save it as draft and open it again or send the email to yourself, so that the text is displayed as link. (btw, i've used gmail and had to save my mail as draft and open it again so that it will display the text as clickable links)
5. right-click and select "download with dTa"
6. type in the filter field "rapidshare" or select the links manually.
7. start download

I think that's pretty easy and fast, if you know how it works.

have fun
