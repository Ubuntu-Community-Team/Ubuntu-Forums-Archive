### Post by John.Michael.Kane on 2007-11-13
If I'm not mistaken usually when updates are grayed out this mean all the dependencies for that particular file have not yet made it to the repos. so to keep users from half updating a file they make it non selectable.

The file should have the requirements meet soon, and installable eventually.
