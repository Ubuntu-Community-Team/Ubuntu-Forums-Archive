### Post by dabl on 2008-04-16
There may be a defect on the CD -- did you make it from a downloaded image file?

- verify md5sum of the downloaded ISO image
- burn the CD at 4X speed
- use the "Check CD for Defect" menu item on Live CD (but it is not perfect)
- possible defect in blank media
- maybe try a different brand of blank media

Also, since you already know you want to install it, you might use the Alternate Install CD for AMD64.  I like that one better.  :)
