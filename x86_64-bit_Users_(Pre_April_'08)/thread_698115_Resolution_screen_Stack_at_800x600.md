### Post by dayanandasaraswati on 2008-02-16
Your xorg.conf file does not have any resolutions set. You have not configured your xserver properly. Try adding these lines of code in the xorg.conf file in the place mentioned below:
> Section "Screen"
Identifier "Default Screen"
Device "nVidia Corporation NVIDIA Default Card"
Monitor "Generic Monitor"
DefaultDepth 16
############Add the code given below here########
EndSection


Code:
> SubSection "Display"
                Depth   24
                Modes           "1280x1024"  "1152x864"  "1024x768"  "800x600"  "640x480"
        EndSubSection
