### Post by Cappy on 2007-08-05
Most 3rd-party packages, such as games, are usually packaged for 32-bit. It's honestly not harder but it is an extra step sometimes with a few packages that you can't get in the ubuntu repos. I think it is completely worth it for the speed boost, personally (ranging from 5-60 percent boost depending on the application).

### Post by osaeris on 2007-08-07
I've been using 64studio on an Athlon 64 system with 1Gb RAM for a good while now (I was encouraged by Kilz' posts to use the whole 64 bit CPU!). I use Cinelerra, Gimp and Ardour. Cinelerra and Ardour mainly (and to a lesser extent GIMP) are clearly superior when running in the 64bit Debian environment than if run in 32bit Ubuntu Feisty. If you try to edit some video using Cinelerra on 32bit you're in for a big dissapointment because it performs very badly.

I also happen keep a 32bit Ubuntu Feisty partition for surfing the net, typing letters, playing music etc. yes I could do those things in 64Studio but I like to keep that for work. 

Depends what you use your computer for I suppose.
