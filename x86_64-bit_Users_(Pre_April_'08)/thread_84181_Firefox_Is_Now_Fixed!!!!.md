### Post by xlulux on 2005-10-30
ok so i was having problems with firefox and the bookmark system. The problem was that the bookmarks were not being saved when i added new ones. I looked on  the properties of the folder and found that it had not been modified in 2 weeks which was odd because i had added bookmarks just today, then i check to see if i had write permissions to the bookmarks.html file and voila the mistake was corrected, if anyone else had this problem: just add write permissions to the owner of the file and it will be fixed
