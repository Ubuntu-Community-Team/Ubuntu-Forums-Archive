### Post by Kuolio on 2005-09-20
Just install new kernel via Synaptic, no need to delete the old one if it is working. It's good to have a working "backup" at hand, if you brake something with the new one :) In any case, dont uninstall your kernel before you install and boot into the new one.

btw. When you install new kernel, and dont delete the old one, you can select to boot the older kernel if you press <esc> when GRUB is booting.
