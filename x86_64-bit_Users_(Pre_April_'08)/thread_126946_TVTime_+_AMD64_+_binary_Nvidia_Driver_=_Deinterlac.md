### Post by Kazriko on 2006-02-07
I have a Geforce 6100 built in video chipset on a Sempron 64 2800+ system. The system requires the nvidia built in drivers so I'm using them. I also have a BT878 based tv card and have used TVTime with it before on another system.

TV Time works, but leaves horrible motion artifacts every time there is motion on the screen. I have found one solution that gets rid of the artifacts. There are 3 deinterlacer settings where the artifacts go away. Full screen, half screen, and one called a vertical something or other. These only fix the problem if accompanied by setting it to half framerate on the deinterlacer. 

Has anyone experienced this problem before with TVTime and an amd64 system? I'd like to go back to using the slightly better quality deinterlacers if I can find a way. There was another Gnome based TV viewer that eliminates the problem as well, but it sucks. ;)
