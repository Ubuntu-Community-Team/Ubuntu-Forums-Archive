### Post by 3rdalbum on 2006-05-01
Try running the latest OS X on a tray-load iMac, THEN try telling me that Ubuntu isn't special.

**1.** Free software for Linux is also much easier to find than free software for Mac.
**2.** Many pieces of hardware work on Linux where there are no Mac drivers.
**3.** Ubuntu Server works faster than an OS X server. In OS X you'd be wasting processing power on the GUI.
**4.** Ubuntu is free to obtain and upgrade, and you never need the latest version to run new programs.
**5.** Ubuntu is more exciting than OS X - 18 months is a long time to wait for an update.
**6.** Being familiar with a form of Linux can be an asset to getting a job, even if your familiarity is with PPC Linux ('cos it's basically the same as x86 Linux!)
**7.** Ubuntu is more secure - the last time I checked, there were no unpatched security holes in Ubuntu, but there were a couple in OS X. Also, the GNU Interpreter for Java (which comes with Ubuntu) has probably never had a security hole, whereas OS X's Java implementation is holier than a priest on Sunday.
**8. **Ubuntu's default user interface agrees with me more than OS (ne)X(t)'s - God, I can't believe people actually use OS X without getting annoyed.
