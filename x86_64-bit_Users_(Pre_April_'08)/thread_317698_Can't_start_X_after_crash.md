### Post by catlett on 2006-12-12
boot to the recovery mode and when you get to the command line enter ```
sudo dpkg-reconfigure xserver-xorg
```That runs you through the xserver setup. It will ask you about your monitor, keyboard, mouse etc. This should fix the issue by giving you a new xorg file. At least I would start with that and see if it solves the issue before going into other stuff.
