### Post by FluffyElmo on 2006-10-26
A word to the wise, if you're upgrading today grab the alternate install CD via bitorrent and do the CD upgrade. (*gksu "sh /cdrom/cdromupgrade"*)

It was less than hour to grab the image this morning, while a network install is *much* longer with the current server load.

I'm a little annoyed. I started the network upgrade and when I realized how long it would take killed it and tried the CD upgrade. However the CD update script seems absolutely bent on completing the prior network install first.

Over 5 hours in it's telling me I have between 3 and 19 hours to completion. Your mileage may vary but I'd definitely go the CD upgrade route first.
