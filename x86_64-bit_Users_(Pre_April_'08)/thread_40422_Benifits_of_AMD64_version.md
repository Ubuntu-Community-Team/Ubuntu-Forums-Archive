### Post by RedCokeBottle on 2005-06-09
[QUOTE=Jerrac]Ok, I have built a new computer based on AMD64+NFORCE4. I have been reading all kinds of posts on here about how the 64bit version of Ubuntu causes problems. So why should I install the 64bit version? What are the benifits?[/QUOTE]

Well by running the 64bit version you are running the processor in 64bit mode that can also run 32bit code. If you choose to use regular Ubuntu for 32bit processors it will not be possible to run anything other than 32bit programs.

For certain mathematically intense operations such as encoding/decoding music/films or generating public keys using 64bit mode would make a bit difference. 
If you do not use those types of programs then it is probably better just to use 32bit Ubuntu and save yourself the hassle of getting things to work in 64bit mode.
