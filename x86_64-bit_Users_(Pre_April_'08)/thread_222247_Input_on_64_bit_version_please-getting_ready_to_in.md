The 64bit version has its benefits and its problems. Just like any OS. One of the biggest is that some things will have to be setup with a howto/script to get the most functionality. [URL="http://www.ubuntuforums.org/showthread.php?t=191205"]You may want to look over the 64bit howto list.
[/URL] 
I also suggest looking in my signature for 32bit firefox w/pulgins and Wine. The 64bit version comes with the 64bit firefox, unfortunately some of the common plugins don't work with 64bit firefox. Wine is another thing that needs a howto to get running. Mainly because it is a 32bit emulator on a 64bit system. If you cant follow the howto's there are also auto setup scripts.
