### Post by Highlag on 2006-05-06
If you have windows installed: 
Simplest way to install Linux is to first make some space on any of your disks.  This space must be shown in windows diskmanager as unused space. 8GB should be enough for start. When you enter Ubuntu setup, you can then chose unused disk space for instalation. Instaler is smart enough, and makes necessary changes in Grub startup list so you have Ubuntu and Windows option to chose for booting.
