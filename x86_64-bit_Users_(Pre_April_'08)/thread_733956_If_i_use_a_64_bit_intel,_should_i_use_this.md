Works fine on my E6400. I'm only using 64 bits because I have 4 gigs of RAM and don't want to use the PAE server kernel; the performance increase is negligible- there's performance loss in some areas, some performance increases in others, so it's a wash. Unless you do rendering, encoding, or other CPU intensive tasks there's not much difference in performance gains (and potential performance loss), and you may have compatibility issues.
