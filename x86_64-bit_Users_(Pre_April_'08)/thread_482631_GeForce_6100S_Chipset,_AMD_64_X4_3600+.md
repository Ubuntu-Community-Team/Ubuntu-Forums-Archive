### Post by jpkotta on 2007-06-23
You should be fine.  NVidia makes 64-bit video drivers, so you can be Blender-ing like no tomorrow.  I have noticed a *massive* speed increase since I got my E6600 Core 2 Duo (vs. P4 Northwood).  This is both because the chip is much better, and the OS is built for modern CPUs, rather than i386 (sort of, Ubuntu apparently has some post-386 features compiled in).  Of course, AMD chips look pretty much the same as Intel ones as far as the software is concerned.  The the default Ubuntu kernel is built for SMP (i.e. multiple CPU) systems these days (it is the "-generic" kernel).  It will handle more than 4 GB of RAM.  You should be fine.
