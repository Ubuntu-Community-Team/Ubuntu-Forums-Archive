### Post by Kilz on 2006-09-07
> **ttallos said:**
> When I loaded in the OEM install I didnt get to set up a user name. I usually use TEXT install so I can set up raid. Does anyone know the difference??

The difference is just what you pointed out. The oem is for equipment manufacturers to install Ubuntu on lots of machines. It will not add a user during install. The text install is for one machine and will setup the user and sudo for that user. Always use the text install unless you are a manufacturer.
