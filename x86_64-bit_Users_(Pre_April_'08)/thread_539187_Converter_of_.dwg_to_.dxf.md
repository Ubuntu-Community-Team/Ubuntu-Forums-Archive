### Post by Lonthong on 2007-08-31
to view & edit dwg files, you can try varicad. However, varicad is not a freeware but they have trial version that you can try. 
Moreover they only have 32bit deb package version. Anyway, you can just force install onto 64bit.

You must also install some 32bit library:
sudo apt-get install ia32-libs ia32-libs-gtk ia32-libs-sdl dpkg-dev
