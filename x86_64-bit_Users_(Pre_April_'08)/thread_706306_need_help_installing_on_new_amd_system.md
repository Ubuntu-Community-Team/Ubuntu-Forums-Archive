### Post by Lutin on 2008-02-24
Some people, including myself, reported the same problem. The resolution to this, for most people, is to change the resolution on the install screen using one of the function keys - F3 or F4.

On my Acer Aspire 7003 laptop, I had to select 1024 x 800 and then everything went fine - the install even ended up with the correct native resolution of 1440x900.

Hope that this is some help.
