### Post by jespdj on 2008-03-31
For Flash: There is no 64-bit Flash, but you can install it using nspluginwrapper. As far as I know this is done automatically on Gutsy and Hardy (although the Gutsy Flash package might still be broken?).

For Java: There is no 64-bit Sun Java browser plug-in. You could use IcedTea or OpenJDK Java.
