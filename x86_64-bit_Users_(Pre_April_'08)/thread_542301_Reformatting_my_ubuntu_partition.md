### Post by John.Michael.Kane on 2007-09-03
What did you do to corrupt your install is a better question. If this is simply a grub issue when you boot normally it should return a error code.  Which may help narrow it down, and allow for a possable fix.

In any case you can reformat using the ubuntu-live cd, As it has gparted.  Let the live cd load up, and then click System--->Administration--->GNOME Partition Editor, after which you can delete the partitions, and reinstall. As well as booting from the alternate cd, and doing a fresh install that way.
