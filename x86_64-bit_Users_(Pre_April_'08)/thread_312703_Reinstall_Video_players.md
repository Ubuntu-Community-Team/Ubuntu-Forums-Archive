### Post by RAOF on 2006-12-05
Your new monitor is bigger, right?  "1680x1050" is a nice high resolution.

Were you using the nvidia drivers before?  The Option "NoLogo" there suggests that :).  You're now using the nv open source drivers.  It's possible that they don't handle high resolutions as well (particularly for advanced stuff, like XVideo).  Try installing the **nvidia-glx** package, and running **sudo nvidia-xconfig** to (re-)enable the proprietry, binary, and better working nVidia drivers :)

Well, that error confirms what I thought.  Some points:

1) You're cards are not not running SLI'd.  You're using the open source "nv" drivers, which don't actually support any 3d acceleration at all, and **certainly** don't support SLI.  And also, obviously, don't handle XVideo at that high a resolution, also.

2) You should try installing the binary nvidia drivers (as I suggested above.  See, I'm always right :p).  That will get you 3d acceleration, and probably fix this problem for you.
```
sudo aptitude install nvidia-glx
sudo nvidia-xconfig
```
will install and configure the nVidia proprietary drivers.

Then you just need to restart X, and everything should work.
