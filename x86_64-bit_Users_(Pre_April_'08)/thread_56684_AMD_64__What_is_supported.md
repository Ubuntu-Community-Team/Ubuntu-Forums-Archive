### Post by wmcbrine on 2005-08-13
IIRC, everything in 64-bit Hoary except OpenOffice is true 64-bit. However, not quite all the same packages are available as in the 32-bit Hoary (e.g., Flash and Java plugins). So, for those, you can use chroot.

As for hardware, try this:

[http://www.ubuntulinux.org/wiki/HardwareSupport](http://www.ubuntulinux.org/wiki/HardwareSupport)
