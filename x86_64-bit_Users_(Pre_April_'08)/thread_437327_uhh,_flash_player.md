### Post by Kilz on 2007-05-08
There are simple solutions to the flash problem that dont require a lot of work. Here are 2 solutions that require little work.

1. [Nspluginwrapper script]("http://ubuntuforums.org/showthread.php?t=425672") - This script will install flash and the nspluginwrapper so that your 64bit installed browser can use the 32bit Flash that is avilable. The drawbacks is that there is no java support with the wrapper. There is an older 1.4 Blackdown java plugin, but it causes the browser to crash for some people.

2. [Install Firefox 32 script]("http://ubuntuforums.org/showthread.php?t=202537") - This script will install a 32bit Firefox (version 2.0.0.3) or iceweasel, or flock. Also the Flash , java, and mplayer plugins. The downside is that you must go to the web page when a new version comes out, download a deb file , and click on that deb file to upgrade.
