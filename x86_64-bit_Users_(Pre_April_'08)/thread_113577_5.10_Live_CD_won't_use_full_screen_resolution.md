### Post by imacQ on 2006-01-07
i also had a problem with resolution using the live cd (imac g3 rev d), however i did NOT have the problem with ubuntu installed on hard drive. just other problems.

this might work? 
open the terminal window
you're going to change the HorizSync and VertRefresh numbers.

and try code below, originally posted by twongkee to help someone out of blank screen.

** type this in exactly as you see it (vi is a terminal editor)**
sudo vi /etc/X11/xorg.conf

change the frequencies in monitor section as follows:
** find this part in the file you are now in (xorg.conf) and make these changes **

Section "Monitor"
Identifier "Generic Monitor"
Option "DPMS"
HorizSync 60-60
VertRefresh 43-117   
EndSection

** after the changes then hit the 'esc' or 'escape' key, then type ':wq'
** no 'quotes' -- but yes the ':' is part of the command
** that will write the changes to the file and quit from vi **

### Post by aeg on 2006-01-18
Thanks imacQ!

I have successfully removed (unsuccessfully retained) Mac OS X on my iBook G3 700Mhz (600+ MB RAM) and replaced it with Ubuntu. Until reading your post, I had only troubles with the the screen resolution (on both Live CD and the install).

In addition to the HorizSync and VertRefresh changes, one must ensure the proper depth and modes. I've played with "1600x1200" "1280x960" "1152x864" "1024x768" and "800x600", although only that last two have worked properly with 24, 16, and 8 bit depths (higher resolutions displayed only the top left of the screen, 1600x1200 and 1152x864 were ignored, and GDM barfed on 32 bit depth).

Section "Device"
	Identifier	"ATI Technologies, Inc. Radeon Mobility 9000 (M6 LY)"
	Driver		"ati"
	BusID		"PCI:0:16:0"
	Option		"UseFBDev"		"true"
EndSection

Section "Monitor"
	Identifier	"Color LCD"
	Option		"DPMS"
	HorizSync	60-60 # previously 28-33
	VertRefresh	43-117 # previously 43-72
EndSection

Section "Screen"
	Identifier	"Default Screen"
	Device		"ATI Technologies, Inc. Radeon Mobility 9000 (M6 LY)"
	Monitor		"Color LCD"
	DefaultDepth	24
	SubSection "Display"
		Depth		1
		Modes		"1024x768" "800x600"
	EndSubSection
	SubSection "Display"
		Depth		4
		Modes		"1024x768" "800x600"
	EndSubSection
	SubSection "Display"
		Depth		8
		Modes		"1024x768" "800x600"
	EndSubSection
	SubSection "Display"
		Depth		15
		Modes		"1024x768" "800x600"
	EndSubSection
	SubSection "Display"
		Depth		16
		Modes		"1024x768" "800x600"
	EndSubSection
	SubSection "Display"
		Depth		24
		Modes		"1024x768" "800x600"
	EndSubSection
EndSection
