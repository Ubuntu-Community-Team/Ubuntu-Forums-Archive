### Post by RAOF on 2006-02-05
It depends on what filesystem your Windows partition is.  If it's NTFS, you're out of luck; you can't write to an NTFS filesystem from linux*.

If it's a FAT32 drive, check out the "Accessing Windows Drives" link in my sig.

*Not entirely true - there are some projects to write to NTFS from linux, but it's (in general) slow and/or experimental.  Captive NTFS is one such project.
