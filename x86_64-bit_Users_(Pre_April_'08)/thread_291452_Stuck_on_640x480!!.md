### Post by pony-tail on 2006-11-02
Lets start at the beginning .
What hardware do yo have ?
Did you use the fglrx drivers from ATI or the ones from the repo ?
Did you have your kernel sources installed ? (the driver install , if you used the one from ATI's website , needs them to  build a module .
If you did not install the kernel source try this.
In a console :-
sudo uname -r
sudo aptitude install linux-source-(without the brackets put the kernel version from uname -r here)
sudo sh ati(whatever the driver is-just hit Tab and it should autocomplete)
sudo aticonfig --initial
If I am correct in what is wrong that should get you started
edit-: This assumes that the ATI drivers are in you home folder - if not you will have to cd to the directory they are in .
