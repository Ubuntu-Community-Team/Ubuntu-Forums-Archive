The server version dose not have a desktop. Servers usualy dont need a desktop, the desktop uses a lot of resources. Gnome and kde use a lot. xfce isnt bad for a server icewm is even better.
You can install a desktop if you want, I suggest xfce
```
sudo apptitude install xubuntu-desktop
```
But gnome is installable
```
sudo apptitude install ubuntu-desktop
```
