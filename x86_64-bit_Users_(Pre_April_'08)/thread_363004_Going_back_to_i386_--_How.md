### Post by Kilz on 2007-02-16
> **dhanjel said:**
> Might be a stupid question, but is there any convenient way of converting a amd64 system to i386 without having to reinstall everything?

In short, No. You will have to reinstall everything. If you are having problems with the 634bit version it may be easier to fix the issues than reinstall. If you are thinking that if you install the 32bit version you will avoid some setup or problem you are most likely only fooling yourself. No version of Linux can be installed without some setup and work. A lot of people that switch versions think they are going to make it easier on themselves in some way, most of the time this isnt the case.
You could install the 32bit version on a different partition if you have the room to see if this is the case. That way if you find you have the same problem you can still use the 64bit version.
