### Post by soonindallas on 2006-02-17
I have a widescreen Acer that I am running at 1280x800 with no problem, though I am using the fglrx driver.

The monitor section of my xorg.conf looks like this:

```
Section "Monitor"
	Identifier   "Generic Monitor"
	HorizSync    30.0 - 70.0
	VertRefresh  50.0 - 160.0
	ModeLine     "1280x800@60" 83.9 1280 1312 1624 1656 800 816 824 841
	Option	    "DPMS"
EndSection
```

my xorg.conf was generated by running 

```
sudo dpkg-reconfigure xserver-xorg
```

and I answered Y to the question about probing for monitor specs.
