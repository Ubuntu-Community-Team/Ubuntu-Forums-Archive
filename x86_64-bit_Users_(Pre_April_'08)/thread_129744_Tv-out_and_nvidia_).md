### Post by FluffyElmo on 2006-02-14
The official Nvidia driver supports TV-Out, you just have to configure x.org to use it. I run TV-Out on a 6600GT with good quality. The driver readme contains instructions or  Google search will probably turn up a number of how-to's. 

TV-Capture is a different situation and is, as far as I know, not supported for Nvidia cards under Linux.
