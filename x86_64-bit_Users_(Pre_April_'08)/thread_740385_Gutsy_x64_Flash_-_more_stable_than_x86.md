### Post by Tom Mann on 2008-03-30
I realize this is a statement rather than a question - but testing the Gutsy beta of Ubuntu (x64) has been a delight for me...

The majority of my problems with Ubuntu have always been down to Macromedia's Flash plugin - which I have always used over Gnash and SWFdec for compatibilities sake. It would occasionally crash Firefox (x86) when moving from one youtube video to another whilst it was still playing.

It still crashes in x64 Gutsy, but now it only kills nspluginwrapper - Firefox lives! A quick refresh of the web page and I'm back in control.

I guess this is thanks to whoever made this work more stable - Ubuntu devs? Firefox devs? I suspect the former.

Cheers :)
