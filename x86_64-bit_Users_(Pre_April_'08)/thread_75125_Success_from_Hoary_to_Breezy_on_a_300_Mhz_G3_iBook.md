### Post by graynotgrey on 2005-10-13
About a month ago I successfully installed Hoary on a 1999 clam shell tangerine Apple 300 Mhz G3 iBook with 128 MB RAM. When the Breezy Preview came out I upgraded to that and have been updating on a daily basis since then. 

Everything on the notebook is recognized and runs great.:D  Breezy is an improvment over Mac OS 9. Hibernation/sleep feature works when I close and open the lid; all function keys function including dimming the screen, adjusting the volume and using F12 for the right mouse click. Sound, USB port (with a flash drive anyway) and network adapter all work. I was able to use Soundjuicer to rip tracks off of a CD. Battery meter works, too.

The notebook is a little on the slow side. I tried XFCE, but did not notice much of an improvement in speed and I liked the GNOME desktop better. More RAM would probably help, but otherwise the notebook is very suitable for web browsing, email, word processsing and listening to music.  

My next venture is to try a USB wireless network adapter. The Airport cards for the first generation iBooks are no longer made and cost over $100.

The only program I cannot get to work is BUM, it generates some perl error messages, but that is minor.
