### Post by jhnphm on 2008-03-25
1. Yes
2. WIth some work, either by forcing architecture and using getlibs to fetch library dependencies, or using a chroot
3. Probably won't have problems w/ that laptop if it's already 32-bit Linux compatible 
4. Haven't noticed many differences
5. Hardy works for me, although glitches running 32 bit firefox causing me to set up a full-blown chroot install since I'm OCD and want my working icons :) . Of course, having set one up, I recommend having a 32 bit chroot anyway for running all 32 bit stuff :) (Some say it's hard to setup though, I seem to be in a minority).
