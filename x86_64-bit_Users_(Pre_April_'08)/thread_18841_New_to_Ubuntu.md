### Post by Pse on 2005-03-09
There are TONS of docs out there you can try reading to get an idea of how Linux works. If you fell like studying how ubuntu in particular works, you can check its documentation in [http://www.ubuntulinux.org/support/documentation/](http://www.ubuntulinux.org/support/documentation/).
You might as well want to try reading: [The Linux Documentation Project](http://www.tldp.org/).
For starters, you may refer to [http://linux-newbie.sunsite.dk/](http://linux-newbie.sunsite.dk/). (For beginners, but extensive).

Hope it helps =).

If you get stuck in any particular part of the installation process of Ubuntu, try to gather as much information as you can about that and then post here, so we can help you =).
