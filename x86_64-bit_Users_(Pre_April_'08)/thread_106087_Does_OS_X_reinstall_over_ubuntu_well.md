### Post by Ptero-4 on 2005-12-20
OS X doesn't have trouble reinstalling over ubuntu as long as there's a dos or hfs formatted partition the OS X installer will work and after installing OS X you can boot back to ubuntu with the open firmware bootloader (the one that comes up when you press the "option" key) and fix yaboot. As for ubuntu itself, no OS X doesn't affect it.
