### Post by dabl on 2008-01-31
Yes, a lot of "settings" live in your /home directory.  I run Kubuntu so I don't remember all the Gnome terminology, but it's the desktop settings, panel/taskbar settings, keyboard layout, PIM contacts if you're using the Gnome one, e-mail (!) maybe, etc. etc.

Probably your cleanest and quickest path to success would be to just back up your data to DVDs or another hard drive, and start with a clean formatted partition(s) for the new OS.  I keep all my long-term data on a separate partition from the OS and /home, for this reason.
