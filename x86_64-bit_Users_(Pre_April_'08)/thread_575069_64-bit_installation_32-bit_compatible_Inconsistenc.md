You have the right idea, but flawed information. 32bit applications do work in the 64bit version. The only thing that would limit that is required files. Sometimes there are a lot of them. But even that isnt a worry any more thanks to GetLibs (search its easy to find)
But there are not a lot of applications you need 32bit versions of. There is a less than 1% difference in the available packages. What isnt there are proprietary applications like flash. Simply because we dont have the code. Flash has been addressed in Gutsy. So you have maybe 2 or 3 applications , depending on what you use, all of which have install scripts or howto's.

All this is spelled out in the sticky posts in this section. You might want to take the time and read them.
