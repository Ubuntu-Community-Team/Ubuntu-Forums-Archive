### Post by MButterman on 2007-02-23
After some looking around, nvidia-settings had insufficient privledges to write to xorg.conf. The quick fix was to run "sudo nvidia -settings" from the terminal, set up monitors as requires then select save configuation to file. i figured a warning would be displayed if privledges were insufficient, however none were given. I had to check xorg.conf right after a save attempt to establish that the changes were never saved. Not sure who offered the information but thanks for it.
