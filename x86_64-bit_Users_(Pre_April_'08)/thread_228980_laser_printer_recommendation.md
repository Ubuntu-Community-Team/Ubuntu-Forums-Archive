### Post by juicemansam on 2006-08-04
This is just my opinion, but if you're willing to pay about $380, I'd recommend an OKIDATA B4350N. It's an awesome printer. It spits out the first page within the stated 6 seconds, except when it's in powersave mode, in which it would be about 10 seconds. The best thing about it is that you can network it and just TCP/IP the print jobs. It also handles Postscript, which is native to Unix/Linux environments. There is a PPD file available that can be used with CUPS. The reason I'm recommending this printer is because I've recommended it to a law office that I provide tech support for, and by far it's the best printer in the whole building. I currently have Minolta PagePro 1350 for my own use, but I'd drop that printer for the B4350N in a second. Another nice thing about this printer it's normal yield toner cartridge costs about $25 online, or a little over $30+ at your local office supply store. If you need a large paper capacity tray, you can order it from Okidata for about $180 + sh, that would allow you to have 1,000 sheets (250+750) in available for printing on. The only thing that's a possible con is that sometimes you have to clear ficticious paper jams; but that's a simple button press, which doesn't happen often. Just my opinion :D.

### Post by Hendrik van den Boogaard on 2006-08-04
I have a ML-1210, which is directly supported by Ubuntu (in 32bit and 64bit flavours). It does not matter if you run 32 or 64 bit for these driver as they are built-in in CUPS.

AFAIK you can use newer printers as well, I remember connecting a ML-15xx on Ubuntu telling Ubuntu that it was a 1200 series, the printer worked fine.
