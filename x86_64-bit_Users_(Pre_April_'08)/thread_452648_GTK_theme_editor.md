### Post by boteeka on 2007-05-29
It's a shame that there aren't (or just I'm not aware of) any GTK2 theme editors out there. I mean a GUI editor in which one could tweak the different parameters of a widget's style and have them previewed in a window or something.

I know it's possible to get a text editor and write it yourself, but it's hard to save it and have it applied over and over again just to see what you need to modify.

From where I stand it looks to me that GTK theming  resembles in some manner CSS used for HTML and XML (there are differences though), only it doesn't uses a single rendering engine, but it can use more, and the theme defines which one.

So, if someone out there is able and willing to write such an application, I think the whole community would be VERY thankful.
