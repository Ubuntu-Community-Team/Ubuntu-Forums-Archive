### Post by vigleik on 2006-07-16
The first thing is to check if you have a bad CD. There are two ways. You can either calculate the MD5 sum (search the forum for how), or you can choose something like "check CD for errors" at the first menu when you boot from the CD. I don't remember exactly what the option is called, but you'll find it.

If the CD checks out, you could try another version. I assume you tried the AMD64 version, but you could try the regular 386 version instead, that one is also supposed to work.

Vigleik
