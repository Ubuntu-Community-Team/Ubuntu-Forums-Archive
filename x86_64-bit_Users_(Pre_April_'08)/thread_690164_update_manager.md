### Post by Yellow Pasque on 2008-02-07
The Update Manager displays an icon in the 'Notifications' panel only when there are updates available. By default, it's configured to automatically check once a day and notify you when updates are available. If you'd like it to auto-check less/more or apply updates automatically, you can configure this behavior in System -> Administration -> Software Sources, under the 'Updates' tab.
