If you look specifically for a PowerPC application, you won't find much at all.

If you can get by with what's available from the repositories, then that's great. Otherwise, most things written for x86 Linux will compile and run on PPC. Unfortunately, there are some things which are badly written, and won't. It'll take some trial and error.
