### Post by Rhubarb on 2008-01-28
**imagemagick** is already in the ubuntu repositories.

If it isn't already installed, just grab it by issuing this command:
```
sudo aptitude install imagemagick
```
That is unless you want the most recent version of imagemagick.

With more practice, compiling from source becomes easier and easier.
