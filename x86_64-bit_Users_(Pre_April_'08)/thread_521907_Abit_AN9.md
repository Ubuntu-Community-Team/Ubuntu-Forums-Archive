Abit AN9 590 chip set
AMD64 x2 5200+
7300GT
SATA Hard Drive
SATA LiteOn DVD
Ubuntu 7.04 64bit

What worked on this pile of parts is as many others have said on this forum.
At boot menu press F6 then after the -- add nosplash noapic nolapic

 ```
 -- nosplash noapic nolapic
```

I'm sorry if this seems redundant but I thought it important to know exactly what system this worked on.
Thanks to all of the people that took the time to figure it out.
