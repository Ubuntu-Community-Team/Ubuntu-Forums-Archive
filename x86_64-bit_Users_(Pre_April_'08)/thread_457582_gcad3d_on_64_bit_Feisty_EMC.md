### Post by Cappy on 2007-05-28
```

wget http://www.wienermaler.at/_CAD/gCAD3D-1.08-Linux-x86
chmod +x gCAD3D-1.08-Linux-x86
./gCAD3D-1.08-Linux-x86

```

Post back any errors, if there are any, and we'll help.
