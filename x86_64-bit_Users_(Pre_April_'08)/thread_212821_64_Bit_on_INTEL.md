### Post by Kilz on 2006-07-10
[Go here]("http://www.ubuntu.com/download") if you want Ubuntu. Pick a mirror that is close to you. The next page will have several versions.
Desktop CD - This is a liveCD and will let you test out Ubuntu before you install it. This way you can test your hardware(for the most part) to see if Ubuntu supports it. Some times you may need to install a driver and the liveCD doesn't work 100%.
Server - You don't want this CD image unless you are going to use the machine as a server. It will not install a desktop.
Alternate CD - This is a pure install CD. If the Desktop CD gives you trouble loading 99% of the time this one will work without problems. It doesn't have fancy graphics. But a lot of 64bit users use this one. Make sure to read the partition section and don't write to disk until it is exactly as you want it.

Download the iso image and burn it to a cd. Alternatly you can use [Shipit]("https://shipit.ubuntu.com") for free CD's. But they take 4-6 weeks to arive.
