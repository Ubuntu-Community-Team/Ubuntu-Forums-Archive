### Post by PaganHippie on 2005-12-15
Possibly, though it would depend mostly on what kind of support hardware Apple chose to include in the new system.  From what I've seen thus far online, Win XP seems to work OK on the pre-production Intel-Mac architecture (indicating that Apple hasn't strayed too far from 'standard' Intel-based hardware), so one might conjecture that if Apple doesn't stray too far from what they're currently supplying to developers, Ubuntu would probably work.  Might even work better than on PPC Macs.

OTOH, even if it didn't work perfectly 'out of the box,' it would probably need only minor tweaks, and in any case I can't imagine the Ubuntu developers letting that situation fester for very long.  My guess would be that OOTB support would almost certainly be implemented by the time Dapper Drake is official (next April).

But then again, this is but an educated guess.
