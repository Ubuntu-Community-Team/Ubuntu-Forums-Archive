I'm currently running this kernel on my Dell 1850:

It has two Intel Xeon processors that support em64t ( 64 bit extensions )...

Does this kernel support SMP? ( dual CPUs )? 

and if so, how can I tell that the kernel has recognized and is using both CPUs?

Also, do you know if it supports HyperThreading?

### Post by RAOF on 2006-10-03
All amd64 kernels support SMP.  You can check the output of **cat /proc/cpuinfo** - if there are two (or more) processors listed there, it's done.

It does support hyperthreading, since that just appears as two processors to the kernel, etc.
