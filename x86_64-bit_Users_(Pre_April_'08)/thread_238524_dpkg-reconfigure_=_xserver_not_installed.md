After tooling around and looking for a solution I came across a HOWTO that I used earlier yesterday. It involved setting up xgl and modifying the gdm.conf-custom file. For whatever reason, this code worked before, but after doing something else, it broke something. So I just commented it out, rebooted and everything is fine again, and all my goodies still work. I'd be interested if there is a guru who wants to explain what might have gone wrong. 

Here's the code I commented out in my gdm.conf-custom file: 

```
#0=Xgl
#
#[server-Xgl]
#name=Xgl server
#command=/usr/bin/Xgl :0 -fullscreen -ac -accel glx:pbuffer -accel xv:fbo -kb
#flexible=true[CODE]
```[/CODE]
