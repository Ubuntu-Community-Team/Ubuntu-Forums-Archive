### Post by Ken_Lewis81 on 2007-05-16
Sadly, the [Sourceforge page for PonyProg](http://sourceforge.net/projects/ponyprog/) states that it's 32-bit only.  The two ways round this with Ubuntu AMD64 are to:
[LIST=1]
[*]Use 32-bit libraries within your 64-bit system
[*]Use a 'chroot' containing a 32-bit kernel
[/LIST]
I've not done either but am sure that there are useful articles at [https://wiki.ubuntu.com](https://wiki.ubuntu.com) and elsewhere on the internet.

Take care.
