### Post by Jouke74 on 2008-01-20
([SIZE="1"]Getting a bit tired of these posts[/SIZE]) What exactly is not working? We can't look into our glass ball and figure out what's wrong without any information on that. Most likely, if it is hardware related like videocard, wireless etc, the 32 bit version will give you similar problems. If you look for 64 bit drivers, then please state the hardware. For the most well known problems like the non 64bit verion is available only, there are four basic things:

1. Read the forums here, there must be someone with the same problem here.
2. Look in google for the things you need and add 64 bit to your search.
3. Install the ia32* bit libraries.
4. Learn how to compile from source to program and / or Deb file, since that will almost always work.

Good luck, JJ.
