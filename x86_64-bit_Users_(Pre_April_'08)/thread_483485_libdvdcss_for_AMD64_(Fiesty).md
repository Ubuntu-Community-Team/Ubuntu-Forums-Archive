### Post by yabbadabbadont on 2007-06-24
I'm not sure about x86_64, but when you install libdvdread3 in x86, it includeds a shell script in the /usr/share/doc/libdvdread3 directory that you can use to install libdvdcss2.  Assuming that libdvdread3 exists for x86_64, you might see if it has that same script.  (and if the script installs a 64 bit version of libdvdcss2)
