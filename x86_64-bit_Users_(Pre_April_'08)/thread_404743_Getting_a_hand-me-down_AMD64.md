### Post by Medieval_Creations on 2007-04-08
Ubuntu does a good job with amd64 support; everything that runs in 32bit can typically befound in the repositories for 64.  I'm running 64 and I haven't had any problem with the codecs.  I did use simple64 to install 32bit Firefox w/ plugins and followed Kilz HowTo for java.  Aside from those few things everything else runs in 64bit.

Helpful Links I used.

Simple64 (firefox & plugins):  [http://www.xnowherex.net/simple64/index.php](http://www.xnowherex.net/simple64/index.php)
Kilz's HowTo (java):  [http://ubuntuforums.org/showthread.php?t=202537](http://ubuntuforums.org/showthread.php?t=202537)
Mediabuntu (codecs):  [http://medibuntu.sos-sts.com/repository-old.php](http://medibuntu.sos-sts.com/repository-old.php)

The only thing that I know of that has problems in 64bit is wine & some virtualization apps like VMWare & VirtualBox.  Most anything else you might need that isn't in the repositories can always be compiled from source.
