### Post by jdp on 2006-03-27
What Mac is it on?  Newer ones will let you hold the Option key at boot to see all bootable partitions and drives.

Also, during the install, did you wipe the drive?  If you did so you may have overwritten Mac OS.  If your computer was old enough, you might need Mac OS to boot into Ubuntu.

There are many points in the install that could cause problems, and many problems vary by the particular Mac you are using.  It helps much more to be specific about what you are using and what you did so that we can know more and give you a better answer.  Right now it's all guess work and all we know about the system is that it's not booting.
