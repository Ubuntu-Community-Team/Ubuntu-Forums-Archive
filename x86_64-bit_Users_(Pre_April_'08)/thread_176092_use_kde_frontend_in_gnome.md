### Post by Jussi Kukkonen on 2006-05-14
The only problems with using KDE apps on Gnome are increased disk space and memory consumption (because you need *kdelibs* in addition to gnome libraries) and differences in visual appearance. The last one can be fixed somewhat with themes (search the forums -- can't remember how it's done).

Just install kstreamripper and try it. take note of the packages that get installed so you can easily get rid of them later.
