### Post by Kilz on 2007-10-18
That depends on the amount of customization imho. If you are running a mostly stock install with applications from the ubuntu repositories it should be safe to upgrade. If on the other hand you have added hundreds of things from sites all over the internet a clean install may be better for you.
In either case backup everything you need that cant be replaced. In either method something could go wrong. Better to be safe than sorry.
