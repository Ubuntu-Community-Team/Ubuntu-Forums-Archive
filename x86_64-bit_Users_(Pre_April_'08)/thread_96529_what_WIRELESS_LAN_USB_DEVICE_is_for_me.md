### Post by ssam on 2005-12-04
if its on [https://wiki.ubuntu.com/HardwareSupportComponentsWirelessNetworkCards](https://wiki.ubuntu.com/HardwareSupportComponentsWirelessNetworkCards) and it says that it works then it is almost certain to work on powerpc. if it mentions that it needs ndiswrapper then it will only work on x86. one that is said to work 'out of the box' is best. one that links to a howto may be difficult to set up.
