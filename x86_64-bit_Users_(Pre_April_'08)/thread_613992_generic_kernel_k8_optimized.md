### Post by John.Michael.Kane on 2007-11-15
Starting with the release of edgy 6.10 Ubuntu's previous kernels that where "k7 k8 etc base" are obsoleted, and are no longer used. Ubuntu uses a standard generic kernel which includes all the features of the kernel in question, and smp support built in as well.
