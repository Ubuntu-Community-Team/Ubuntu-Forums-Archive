### Post by jbennett on 2006-03-30
[QUOTE=yashton]I just installed ubuntu on my laptop, but can't really use it since the only internet access I have is wifi. How do I reformat the Ubuntu partition so I can use the partition in OS X, and how do I reset the boot sequence?[/QUOTE]

Wireless internet can be used in Ubuntu, it just may take a little work to get it up and running (I use wireless internet from Ubuntu everyday).  Try searching the forums for wifi or wireless internet or maybe even the name of your wireless NIC, I'm sure someone has had a similar issue.

As far as reformatting, you should be able to insert a LiveCD and boot from that and use a partition manager like gparted or qtparted to reformat the partition that Ubuntu is installed on.

Hope that helps.
