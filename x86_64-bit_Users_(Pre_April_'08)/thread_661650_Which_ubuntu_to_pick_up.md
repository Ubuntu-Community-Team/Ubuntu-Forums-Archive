### Post by Kilz on 2008-01-08
> **kvonb said:**
> lol

Yes, Ubuntu is a great beginners distro, but I wold suggest sticking with the 32 bit version (called i386 desktop), unless you aren't intending to play any (3d type) games or install non-supported non-64 bit applications, as it can be a bit of a steep learning curve (as in I've been using Ubuntu for quite a while now and I couldn't live with it - and I'm not THAT unskilled!).

You might run into problems with wireless (if you use wireless), but search this forum for answers, I'm sure you will be OK.

I would not recommend anyone with 64bit hardware installing the 32bit version. Sadly some uninformed people still spread misinformation. There is no reason that 3d games will not play, and its possible to install 32bit applications if by some reason you want to.
