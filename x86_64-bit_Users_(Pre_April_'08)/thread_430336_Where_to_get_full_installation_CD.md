### Post by Scheater5 on 2007-05-02
You have, in fact, recieved the full installation cd.  Ubuntu comes on one CD.  You most likely recieved the desktop cd, which is a live cd, but also an installation cd with a GUI installer.  There are thousands of additionall packages, which other distributions such as Fedora include on cds, which you can download via the internet.  In your case, you may have to find someone to create a cd of downloaded packages for you, i.e. via APTonCD.  However, you may find that you do not need any additional software.  In contrast to Windows, where a fresh install leaves you with poor hardware detection and little more than a calculator, Ubuntu comes with all the software your average user would ever need, minus a few codecs kept out for legal reasons.
