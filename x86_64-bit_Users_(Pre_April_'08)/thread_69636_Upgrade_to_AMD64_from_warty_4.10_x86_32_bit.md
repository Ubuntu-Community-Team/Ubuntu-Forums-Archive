### Post by DancingSun on 2005-09-27
Nope, no easy way at least.  Everything needs to be recompiled for the AMD64 architecture for any code to take advantage of the 64-bit CPU.  Though your data files *should* be usable without any changes.  So it's best to backup all your data files and do a clean install of Ubuntu 64.
