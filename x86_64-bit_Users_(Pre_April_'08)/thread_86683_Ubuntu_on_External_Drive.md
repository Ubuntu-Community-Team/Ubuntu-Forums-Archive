### Post by farruinn on 2005-11-06
You could try installing the iso to the external drive then to boot from that drive hold the option key after the boot chime.  If the external drive comes up on the screen select that and it should boot you into Ubuntu.  I've never done this so I don't know if it will work, but that's how I'd go about doing it...

If you ever run into problem booting, you can reset the PRAM by hold the open apple-option-P-R keys down until you hear a third boot chime.

Good luck!
