### Post by DancingSun on 2005-10-22
[QUOTE=Chris Cromer]Well I do programming(C, php, perl, etc), I do alot of web related things like building websites which means alot of ftp and http connections and things of that sort, I also like to do a bit of gaming as well(not really advanced games, but stuff like Age of Empires III), I also play around with databases like mysql, sqlite, mssql, etc.

Take a look at this article then:
[http://www.anandtech.com/showdoc.aspx?i=2249&p=1](http://www.anandtech.com/showdoc.aspx?i=2249&p=1)

It has many benchmarks, of your interest, is the content creation and workstation benchmarks.  This [page]("http://www.anandtech.com/showdoc.aspx?i=2249&p=14") gives a comparison between an Intel and an AMD CPU in terms of performance for around $200 last year around this time.  I think both CPUs are well suited for your development needs, although if you do a lot of code compiling, it looks like AMD will be your choice of CPU.  In my experience, tasks like building websites, making FTP and HTTP connections for development purposes don't really tax the CPU that much, and the performance of the tasks are well within acceptable parameters.  On the other hand, compiling source code is something where performance is still not fast enough that it will incur a period of wait time.  Now that I've given more thought to this, I think AMD is a better choice, especially if you're using Linux.  Many applications only work well in Linux if you compile them from source (many programs also only come in the form of source code), in this case, the AMD chip can save you more time.

In the end, it comes down to how much performance you can buy with your budget.  If you can buy more performance with AMD (usually the case), then go with AMD.  If it's a tie, consider that the new Pentiums draw considerably more power than the AMD CPUs, so in the long run the AMD chip may provide you more savings by using less electricity.  This also means a cooler running chip, which is always good.

Also, consider the platform costs as well.  Currently, AMD motherboards are cheaper than the Intel ones.

BTW, Age of Empires 3 is just freshly released, I'd consider it an advanced game.  In game performance, AMD wins hands down.

Also, Intel's EM64T does not perform as well as AMD's AMD64 in 64-bit applications.
[http://www.linuxhardware.org/article.pl?sid=05/02/24/1747228&mode=nocomment](http://www.linuxhardware.org/article.pl?sid=05/02/24/1747228&mode=nocomment)
