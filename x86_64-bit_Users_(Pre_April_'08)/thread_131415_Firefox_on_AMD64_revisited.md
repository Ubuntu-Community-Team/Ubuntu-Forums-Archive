### Post by Daggett on 2006-02-16
Someone had already made a .deb package, but it didn't work for me with Kubuntu AMD64.  I got that big gray area below the status bar that some people were talking about.  

So I compiled my own version of Firefox under AMD64.  It is in tar.bz2 format so that non-Kubuntu/Ubuntu users might be able to benefit from it.  You can download it from here:

[http://www.varlinux.org/vl/html/modules/mydownloads/visit.php?cid=3&lid=6]("http://www.varlinux.org/vl/html/modules/mydownloads/visit.php?cid=3&lid=6")

Untar in from the / directory.  It will go in /usr/local, just like the .deb version. 

I hope it works for you.  I have not tried using any plugins with it, but I have two extensions (add bookmark here, bookmark synchronizer) and one extra theme that work fine.
