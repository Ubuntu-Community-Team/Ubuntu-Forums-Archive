### Post by followme on 2006-03-07
instead of a regular install, do a server install, you can use a lightweight desktop like XFCE, fluxbox ...etc.  There is a xubuntu-desktop in the repositories.  or you can read up more on Xubuntu.  Uninstall many programs you don't need.  

Dillo is a great lightweight and fast browser, but I find myself always going back to firefox due to the features.

You will have to install the w32codecs to play your videos, which you'll have to get it directly from a Hungarian site I think.  But the instructions are easy.

When I did this, I noticed a huge increase in performance over the full gnome install.

### Post by ssam on 2006-03-07
install as much RAM as you can afford.

maybe switch to a light weight window manager, xfce is quite friendly and a bit lighter, something like openbox is very light (but does not even have a panel built in).

install dapper. this is the development branch of ubuntu and will be the next version in april. it has many improvements in speed. it might be wise to wait until april if this is your main computer.

for video playback you could try enabling DMA, or trying a few different video players, xine, vlc, mplayer etc

update:
the above post mentions w32codecs, these are only useable on x86 computers, but you should be able to manage without.

### Post by jdp on 2006-03-07
I've got an iBook G3 600 (16VRAM) model.  It runs Breezy fine, but I've not tested DVD movies on it as it's only got a CD-ROM drive.  Otherwise, the AP card works great and browsing is fine.  Battery life isn't as good as OS X, but otherwise it's quite useable.  You can install a 512MB PC-100 SODIMM for under $100 to get a total of 640MB RAM.  That'll make it run fairly well.  If it's got just the stock 128MB, the RAM will make the most difference, even under OS X.  If you feel like an involved and complicated disassembly, you can swap in a faster HDD, but it might not be worth the effort for the gains until you see how Breezy performs as-is or with more RAM.
