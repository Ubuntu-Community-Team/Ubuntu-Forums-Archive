### Post by GTvulse on 2006-01-28
You will need to add the correct VertRefresh and HorizSync ranges for your monitor to /etc/X11/xorg.conf. The easiest way is to run
```
sudo dpkg-reconfigure xserver-xorg
```
and edit the values for your monitor in advanced mode (check the small flyer that came with your monitor, the values should be in the technical specification list).

### Post by Askari-za on 2006-01-28
I had the same problem. What solved it for me:

sudo dpkg-reconfigure xserver-xorg

When you get to the part where it asked you what screen you have or something you get 3 options: simple, medium? advanced? Try simple and just tell it what size monitor you have. Dont try manually telling it the hsync etc etc. Somewhere else it also asks for your desired resolutions, chose like from 1600x1200 all the way down.

### Post by ravi.csin on 2007-12-21
Hi Rahulcool,
I had similar problem with Ubuntu 7.10 and Samsung SyncMaster 753s. The problem is solved by adding following lines in the monitor section of xorg.conf :
HorizSync	30-70
VertRefresh	50-160
# V-freq: 69.00 Hz  // h-freq: 69.10 KHz
Modeline "1280x960" 122.24  1280 1344 1496 1768   960  960  963 1001
-Ravi
