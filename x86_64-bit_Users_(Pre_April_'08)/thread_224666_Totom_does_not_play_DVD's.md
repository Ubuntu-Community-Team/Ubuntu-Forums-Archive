Doubble click the file anyplace and gdebi will start to install the file. But since you downloaded the file you may need [to read this page.]("http://www.monkeyblog.org/ubuntu/installing/") Most of the time in Ubuntu we use the Synaptic package manager to download and install packages/applications. You should also enable the [mutiverse and universe repositories so you have all the applications available.]("http://www.monkeyblog.org/ubuntu/installing/#enabling_extra_repositories") 

But getting back to the fact that you cant play dvd's. You may need to get libdvdcss2 setup. [Make sure libdvdread3 is installed with Synaptic before following this instuctions in this link.]("https://help.ubuntu.com/community/RestrictedFormats#head-38508785e53c611dde1859232189b2e823135eb9")
