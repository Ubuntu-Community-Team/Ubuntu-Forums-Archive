### Post by veloce on 2007-06-03
In your server layout section, you need a line for the second screen, e.g.:

Screen 1 "default screen" left of "ati screen"

Also, if you are going to number the screens (0 and 1) then you need to have that line in the screen section.  Might be better to remove the numbers.

### Post by veloce on 2007-06-04
> **ghpike said:**
> When I try and save the file after making changes it says I do not have permission.  Also can you be more specific as to what lines of code to add for my specific application.  I am a little slow when it comes to linux code.  Php was stretching it for me haha.

To edit your xorg.conf, use ALT+F2 to bring up the run command window and enter:

```
gksudo gedit /etc/X11/xorg.conf
```

Translation:
gksudo - run a graphics program in super user mode
gedit - gnome editor
/etc/X11/xorg.conf - the file you want to edit.

In that file, change the server layout section from:

```
Section "ServerLayout"
Identifier "Default Layout"
Screen 0 "aticonfig-Screen[0]" 0 0
InputDevice "Generic Keyboard"
InputDevice "Configured Mouse"
InputDevice "stylus" "SendCoreEvents"
InputDevice "cursor" "SendCoreEvents"
InputDevice "eraser" "SendCoreEvents"
EndSection
```

To:

```
Section "ServerLayout"
Identifier "Default Layout"
# Only the following two lines have been changed.
Screen "Default Screen" 0 0 
Screen "aticonfig-Screen[0]" leftof "Default Screen"
InputDevice "Generic Keyboard"
InputDevice "Configured Mouse"
InputDevice "stylus" "SendCoreEvents"
InputDevice "cursor" "SendCoreEvents"
InputDevice "eraser" "SendCoreEvents"
EndSection
```
