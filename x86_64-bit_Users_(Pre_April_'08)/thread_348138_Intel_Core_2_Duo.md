### Post by bward1 on 2007-01-28
On an intel core 2 duo you can choose whether you want to run the 64 bit ubuntu (amd64) or the 32 bit one.  I personally run 32 bit because my ubuntu system is my main workstation so I like for it to be stable, and a few of the packages aren't stable for 64 bit, or at least so I have heard.  The nature of 64 bit also takes more ram because it processes data in chucks of 64 bits instead of 32, so that is another reason I chose to run 32 bit.  I will say though that I am running 64 bit gentoo (a different linux distro) on my server and laptop, and it works very nicely.  A 64 bit operating system should increase your performance in a lot of tasks, so really either one is fine, just a matter of choice.  You are right though, it is deceiving that it is called amd64 when it can be installed on either amd or intel hardware.
