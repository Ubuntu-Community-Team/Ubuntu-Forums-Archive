### Post by jdfreshwater on 2007-10-15
Synaptic automatically installs nspluginwrapper whenever you install flash-nonfree.  It works without a problem much easier then having to install it myself.  Another thing I noticed about gusty, is the including of the Cwiid driver to use my wiimote for mythtv.  Now I did not have to compile the package on my own.  I am fairly happy with my xubuntu 64 bit install.  Not having to do as much manual installing of packages gave me more time to get running.  My only complaint so far is with some mythtv video freezing and display issues.  This may be fixed with the current updates that I am downloading.  

Thanks to the team who has made this release even easier to use then Fiesty, and way better then the sometimes buggy Edgy.
