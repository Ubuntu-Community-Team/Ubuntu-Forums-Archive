Swiftfox isnt distributed under the gpl. But a proprietary license that limits your [freedom to redistribute]("http://www.fsf.org/licensing/essays/free-sw.html"). A freedom the program (Firefox) and code its based on give you. Even if it were the owners of the code would have to sue , not a user.
To some people the question isnt if other applications are free, but if the application you are thinking about using is free. IMHO to many people have worked to hard to give us a free operating system (linux) to just give away a freedom for no reason. 
The reason I say no reason, is because there are free alternatives. Firefox is a little better. Iceweasel is even better. They do the exact same thing, in fact are the same program, but do not limit any of the [4 freedoms of free software.]("http://www.fsf.org/licensing/essays/free-sw.html") The only thing about Firefox that isnt free, is if you make changes to it you cant call it Firefox. If I wanted to redistribute the Firefox appication from the Firefox website, I can. Not so with Swiftfox.
Perhaps it doesnt matter to you. In that case, go ahead, use it. But to some people it does matter. The thing is, Swiftfox doesnt tell you its not freee, it hides the fact in a file. People think, wow its like Firefox, so its free like Firefox.  But it isnt. So I sometimes bring to their attention it isnt to people.
True some people use non free software and codecs. But if there is a choice to use free they use it. Some applications do not have a free alternative. Swiftfox has an alternative, in fact it has a few.
I just think we should support and use free software. Because if we dont we send the wrong message to developers. That its ok to limit a freedom. Like I said above, to many people have worked to hard to make sure we have free software for me personally to not make the free software choice where its possible. You have a chioice. But at least now you have chosen to give up the freedom the parent application gives you for the sake of ........ well Im not sure what you gave it up for. But at least you have made the desision knowing the application isnt free.
