### Post by mrsym0r on 2007-04-07
Hello all,

I am having difficulty getting wine to run. It is a fresh install, on
a fresh machine; specs as follows:

AM2 3500+ 64 bit AMD cpu
Gigabyte GA-M61VME-S2
Asus EN7600GS 'Silent' graphics card (nvidia 7600GS chipset)
Kubuntu 6.06 Dapper

cat /proc/driver/nvidia/cards/0
Model:           GeForce 7600 GS
IRQ:             217
Video BIOS:      ??.??.??.??.??
Card Type:       PCI-E
DMA Size:        32 bits
DMA Mask:        0xffffffff

when installing, I followed the instructions from winehq.org regarding
64 bit installations -

sudo dpkg --force-architecture -i wine_*_i386.deb


Now, when I attempt to run anything with wine, it gives me the following error:

==================================

$ wine mirc621.exe
wine: creating configuration directory '/home/simon/.wine'...
X Error of failed request:  BadAlloc (insufficient resources for operation)
 Major opcode of failed request:  142 (GLX)
 Minor opcode of failed request:  3 (X_GLXCreateContext)
 Serial number of failed request:  14
 Current serial number in output stream:  15
wine: wineprefixcreate failed while creating '/home/simon/.wine'.
simon@kubuntu:~$ wineserver: could not save registry branch to
/home/simon/.wine-i5Uetg/system.reg : No such file or directory
wineserver: could not save registry branch to
/home/simon/.wine-i5Uetg/user.reg : No such file or directory


=================================

I get the same error when attempting to load winecfg. I have added my
account to xhost localhost.

There's an existing hardware issue where ubuntu does not currently
support the Realtek 8201 chipset that controls the onboard ethernet
port (easily resolved by slapping in a $15 NIC), however I can't see
how this would effect WINE.

I've hammered google over this, and tried the usual recommendation of
updating the nvidia driver, but without any luck. The only other issue
I found with this error was a documented bug, which was fixed several
updates ago around mid 2006.

Has anyone else stumbled across this before? Any input is greatly appreciated.
