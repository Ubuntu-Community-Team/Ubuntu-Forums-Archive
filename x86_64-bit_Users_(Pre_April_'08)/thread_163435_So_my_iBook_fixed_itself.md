### Post by dailyplanet on 2006-04-20
I had orginally successfully installed Breezy on my iBook G4 1.33 Ghz. Then one day, while running OSX, I hear this awful spinning noise. Thinking it might be a hard drive failure, I braced for the worst. One day, the noise didn't stop and my Mac froze. I rebooted, only to discover that I couldn't boot OS X. I thought my hard drive had gone kaput. I tried booting Ubuntu. IT WORKED. So it wasn't a hard drive problem. I then reinstalled OS X and everything worked. 

I'm now hesitant to install Ubuntu onto my machine again as I suspect something probably went wrong during the whole re-partioning process. Oh well. Until there's support for the Mac keyboard layout, flash, and realplayer, I'm taking a break from Ubuntu. :-k
