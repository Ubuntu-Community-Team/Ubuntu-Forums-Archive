### Post by x64Jimbo on 2006-10-19
Do you want to run a 64 bit OS? You don't have to. Unless you're doing a lot of video rendering or encoding, you probably won't even notice the difference. If you want a 64 bit OS, you will want everything to be 64 bit (binaries, libraries, etc). Just re-download the 64 bit version and re-install.

You could have installed the i386 version. But don't think that if you did that you would have had no problems. Its not a cure all, easy, sure fire solution some people make it out to be. There are reports of problems when people install that version on a 64bit system. Slowness, crashing, unresponsive systems, and hardware not working. My cd rom drive refuses to work once the 32bit version is installed as an example. There are always headaches regardless. Linux is never a do nothing install.
