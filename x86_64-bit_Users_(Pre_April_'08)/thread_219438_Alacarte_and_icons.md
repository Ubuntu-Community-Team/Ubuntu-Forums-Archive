In the process I also figured out why Alacarte was not displaying the icons sometimes. You type in a path but it doesn't show the icons in the window below the path entry box. You have to click on Open and then it will display the icons for that path, in a separate window. Yet sometimes it does automatically display the icons for a path in the window below the path entry box. Confusing.
