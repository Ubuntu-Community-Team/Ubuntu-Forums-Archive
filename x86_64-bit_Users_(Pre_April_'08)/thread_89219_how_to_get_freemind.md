### Post by stefanborremans on 2005-11-12
Try the apt-get installer at the terminal:

sudo -H -s (for root access)
apt-get install freemind

If that doesn't work, try to find more info on their site:

[http://freemind.sourceforge.net/wiki/index.php/Main_Page](http://freemind.sourceforge.net/wiki/index.php/Main_Page)
