### Post by bwallum on 2007-08-09
Hi, thanks for the response.

I have run stellarium from the terminal and received:-
Warning: Couldn't set 1280x1024 video mode (No video mode large enough for 1280x1024), retrying with stencil size 0
Error: Couldn't set 1280x1024 video mode: No video mode large enough for 1280x1024!

I reset the resolution to 1280x1024 using nVidia settings in Envy and bingo! Stellarium works! 

Once in Stellarium the default screen resolution can be set using the configuration>video tab (look for the small buttons at the bottom left of the screen)
