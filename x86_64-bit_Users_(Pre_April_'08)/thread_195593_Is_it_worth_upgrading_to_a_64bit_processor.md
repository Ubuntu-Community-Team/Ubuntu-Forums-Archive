### Post by czhu on 2006-06-13
Firstly, congrats on quitting!
Currently, i think it's not worth spending extra to get 64bit processors when you're just going to use it for desktop needs. If you're going to run your linux machine as a server or do 3d modelling and such activities then you might notice a performance boost from 64bit (you can see lots of threads about this in 64bit forum)

But currently, the software and especially media codec support for 64bit dapper isn't as good as the 32bit. Which means you'll have to read hundreds of threads and bend over backwords to get simple things to work...

### Post by Kilz on 2006-06-13
Its not that hard to get a 64 bit system running with everything that is needed. Granted you cant run an automated setup script like you can with the 32 bit version. But its not impossible to do. 
Its not a lot of hunting down or installing complicated chroot setups. For the most part its downloading the package and using 
sudo dpkg --force-architecture -i <packagename>
Or following a few howto's. There is eaven a sticky now to point to them.
