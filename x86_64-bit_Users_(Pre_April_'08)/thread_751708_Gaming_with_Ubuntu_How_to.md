OK so I have seen some of you want to game with Ubuntu. well the thing is that almost all games are made for Microsoft  boxes! But there are some very good games out there for linux to.
here are some with a installation howto.

Next up are a Nintendo shell with Linux characters.
open terminal and type
```
sudo apt-get install supertux
``` 

Next up are a Flight sim, named Flight Gear

```

```

I found that gaming with ubuntu can be slow. The thing is that compiz and other  programs take a lot of power. disable visual effects and you are good to go! Heres how.
Go "menu>system>preference>appearances>visual effects" and choose none.
