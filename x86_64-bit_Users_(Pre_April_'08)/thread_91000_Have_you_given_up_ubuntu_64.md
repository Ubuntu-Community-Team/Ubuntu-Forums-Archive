### Post by theBuggane on 2005-12-14
I have a triple-boot machine: breezy32, breezy64, XP. My Breezy64 issues were: DVD playback, Flash, Java plug-in for browser. The DVD issue is a general one and solved thanks to the experts on these forums.

Since solving the flash issue ([solution]("http://www.ubuntuforums.org/showthread.php?t=90106&highlight=flash+chroot")) - thanks RAOF!- I hardly use the 32 install.

For Java, I have the 64-bit JDK. To allow me to use the java plug-in with my fixed browser (Firefox 1.5 by the way), I installed the 32-bit version of the JRE. The required plug-in is included. create a link in the Firefox plugin directory:

> ln -s /usr/java/jre1.5.0_06/plugin/i386/ns7/libjavaplugin_oji.so ./libjavaplugin_oji.so

Works a treat. The only thing I use this JRE for is Firefox.

Now I need Shockwave (not on Linux at all yet I think?)...

Windows is now for photo printing (Epson drivers) and video editing (Premiere 2). Ubuntu64 for everything else!
