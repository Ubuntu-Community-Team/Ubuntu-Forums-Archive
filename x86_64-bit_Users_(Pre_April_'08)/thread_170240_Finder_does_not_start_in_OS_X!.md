### Post by William the Conquerer on 2006-05-04
If finder does not work, it's not a problem caused by ubuntu.  First of all they're on completely different partitions which means the data that ubuntu uses is seperated from the data that OS X uses.  Second these different partitions have different filesystems that as I believe are not compatable so it would not be easy hypothetically disrupt or corrupt files on the OS X partition from the ubuntu partition...  I would say it's a problem inside of OS X and I would look into how to configure Finder correctly in that...  If you still want to remove ubuntu just delete the partition from within OS X...
