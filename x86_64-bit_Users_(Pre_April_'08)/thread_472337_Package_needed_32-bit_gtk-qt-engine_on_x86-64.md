### Post by Nosedog on 2007-06-12
The 32-bit libraries for gtk-qt-engine should be available for x86-64 Ubuntu. The reason is that I'm testing the Firefox 3 beta, which seems to be only available for  i386. Firefox looks ugly because it can't use the gtk-qt-engine theme manager.

I did download the i386 gtk-qt-engine package from packages.ubuntu.com, then installed the .so files to the lib32 folder. It makes Firefox look a little different, but still blocky and ugly.
