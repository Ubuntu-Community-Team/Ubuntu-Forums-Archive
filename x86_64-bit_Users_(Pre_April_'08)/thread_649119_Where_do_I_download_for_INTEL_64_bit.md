This link is wrong (pointing to Feisty) the correct link is: [Gutsy Gibbon 7.10 Desktop Live CD](http://releases.ubuntu.com/7.10/ubuntu-7.10-desktop-amd64.iso)

I always urge people to download images via [Bittorrent](https://help.ubuntu.com/community/BitTorrent) instead; because by the time the download completes, the image is already checked to be error free. So:

[**Desktop CD for 64-bit PC** (AMD64) computers (BitTorrent download)](http://releases.ubuntu.com/7.10/ubuntu-7.10-desktop-amd64.iso.torrent) <- Graphical install
[**Alternate install CD for 64-bit PC** (AMD64) computers (BitTorrent download)](http://releases.ubuntu.com/7.10/ubuntu-7.10-alternate-amd64.iso.torrent) <- Text install

The reason it says AMD, is because Intel used the same instruction set extended by AMD for their 64 Bit platform, called [AMD64](http://en.wikipedia.org/wiki/X86-64).
