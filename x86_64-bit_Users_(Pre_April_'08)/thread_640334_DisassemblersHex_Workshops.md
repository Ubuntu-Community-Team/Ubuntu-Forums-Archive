### Post by nxt on 2007-12-14
under ubuntu (gnome) you can try GHex, which is in the ubuntu repository
[http://www.abclinuxu.cz/images/screenshots/1/5/68951-ghex-8454.png](http://www.abclinuxu.cz/images/screenshots/1/5/68951-ghex-8454.png)
```

sudo apt-get install ghex

```

under kubuntu (kde) try KHexEditor
[http://freshmeat.net/projects/khexedit/](http://freshmeat.net/projects/khexedit/)
```

sudo apt-get install khexedit

```


for gnome there is also Bless
[http://www.gnomefiles.org/app.php/Bless](http://www.gnomefiles.org/app.php/Bless)
