### Post by mainalisuyog on 2007-06-06
Run killall python. Also, remove autostarting of gdesklets and screenlets from system->preferences->session->autostart. Log and and log in. Is the problem still there?
Try uninstalling screenlets and gdesklets. Also, you could create a new user and log into gnome and see if the problem persists when u log in as the new user as well. Hope this helps.
