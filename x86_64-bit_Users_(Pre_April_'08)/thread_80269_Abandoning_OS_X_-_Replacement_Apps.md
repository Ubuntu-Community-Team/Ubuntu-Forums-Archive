### Post by TDave on 2005-10-21
All

Having finally taken / been able to take the time out to work on downloading and installing / uninstalling most of the apps I need to make Ubuntu useable for nearly all tasks (Internet access is a gift I rarely have whilst at work), I thought I'd put it to the rest of you to see what apps people have chosen as 'replacements' for the apps bundled with OS X (the 'iSuite' series more specifically).

So far, I've found the changeover surprisingly easy:

**[SIZE="4"]Office[/SIZE]**
My powerpoint slides that I need for work that before sometimes dropped links in OpenOffice.org 1 or NeoOffice/J work pretty much flawlessly by being able to use OpenOffice.org2 (the last time I checked, this wasn't yet available for OS X.

**[SIZE="4"]Image Editing[/SIZE]**
It goes without saying, is done with the GIMP (as I did in OS X)

[SIZE="4"]**Image Album Software**[/SIZE]
I have yet to find a suitable replacement for this at the present time - what are people's reccomendations?

[SIZE="4"]**Music Application**[/SIZE]
The best equivalent I have found for (and actually with 1.3 I personally think is better than) iTunes is amaroK.
A little bit frustrating that I need to install so many extra KDE libraries without running KDE in order to use it though.
Anybody know of any Gnome equivalents (for me, Rhythmbox, while nice, was just too minimal)?

**[SIZE="4"]Video Editing[/SIZE]**
Although its still bulky in places, I've found Kino to do the job well enough (admittedly I don't edit all that much video) to replace iMovie.

**[SIZE="4"]Movie Playing[/SIZE="4"]**
Not being a fan of Totem, that was one of the first things to go (it also seemed to render my *.avi and *.mov files very roughly (pixelated, occassionally running from the start at double speed, sound out-of-step with picture [these are not file errors by the way - they ran fine within vlc on OS X]) and has just recently been replaced with VLC which seems to be rendering things much better.
It's still a shame not to have a DVD app 'out of the box' like OS X's DVD Player but then tweaking is half the fun, right?

[SIZE="4"]**Others**[/SIZE]
Of course, the majority of apps stay pretty much the same - I still use Firefox and Thunderbird for Web Browsing and Email respectively and can use GAIM as my Instant Messenger (didn't find a GAIM port for OS X so was using Fire).

For Web Editing, again, I haven't been able to find anything that quite matches the ease of use and included functions of, say, Dreamweaver, available for the platform, but then I was quite happy with my time spent using Quanta when I was using YellowDogLinux and other KDE-based distros so I have that running. Again, it means more unneccessary KDE libraries but, until I try and see how Bluefish works out for me (downloading now) then I haven't really heard about much else that is in that league. Any thoughts?

Wireless - I have heard all the latest excitement regarding what the Gentoo team got working through mol and admit I haven't been able to try it yet, but, for now, I have a Netgear WG511T PCMCIA Card working just fine for the times I am able to get Wireless access. As everybody has said before though, it would still be nice for Broadcom to release drivers, or source code.


So, what extras can you think of? What applications do you use in OS X that you just can't do without or can't find a suitable replacement for? Which of my choices suck?

I still have an OS X partition on my hard drive, simply because there is one aspect of my job (manuals) that require online updating with the downloads online available as dmg or exe - I am currently trying to arrange for a third line of downloadable updates - if I manage to get that done then I will be able to abandon the mac partition completely, but for now I'll just have to keep using mol for the times I need it.

Any and all feedback hints/tips/tricks are appreciated.

Dave



P.S. As an after thought, I am currently running Ubuntu on my PowerBook G4 15" without any major problems.

### Post by KDPRoss on 2006-05-14
The biggest "killer apps" for me are as follows; I would *very* much appreciate any advice / suggestions regarding GNU/Linux replacements for these!

==Backup==
I have had *many* hard drive failures, and, as a result, I am obsessed with taking the task of backing up my data seriously. I use Retrospect to backup my OS X machine, and have had excellent results with it -- the last time I had a hardware failure, I was up and running with all my data within a couple hours. The features I like in a backup software are: incremental backups, ability to perform specific-file or -subdirectory restores, and one-click (or some small number of clicks) backup.

==DVD Ripping==
HandBrake is awesome: I remember back in the day that ripping DVD's was *at least* 10 steps, and HandBrake has a GUI that hides all the details that I'm not willing to trouble with any more. It takes about 15 seconds to select the settings I want (including output-file size, colour / B&W, bitrate for audio, subtitles language, audio track, etc.) and on the order of 5 hours (1.5gHz G4) to rip / transcode (to Xvid / MP4) a 2-hour movie.

==CD / DVD Burning==
I use Toast on OS X, and I've always hated it, mainly because it uses some bizarre file format for images instead of ISO's. But -- and I have to admit I've not looked into this seriously -- I've not found anything as easy for GNU/Linux ... I've used mkisofs and cdrecord, but I'd really rather some made-for-your-mom point-and-click process: It's just too much hassle otherwise. I know that there are a large number of GUI's for this, but what have people had good experiences with?

==Image Editing==
I use and love Gimp ... but I've never had any luck with getting it to play nicely with my Wacom's pressure sensitivity, which is almost the entire advantage of using a tablet. So, I'm somewhat "stuck with" PhotoShop.

==External Display==
Has anyone been able to make an external display work (with Ubuntu and a PPC machine)? I remember screwing around with it for a while, but the best I could do was single-head (i.e., mirroring) at some ridiculous resolution (400 x 300, I think). I would be happy with mirrored displays, although dual-head would be great, too.

-and- In reply to several earlier posts:

> can use GAIM as my Instant Messenger (didn't find a GAIM port for OS X so was using Fire).

Well, you can install Gaim on OS X with Fink and X11. OS X is hardly useful without Fink / X11, anyhow.

> web editing

That's why there's Emacs! Html isn't *that* hard, and it ends up being *much* faster than using a WYSIWYG editor ... but I'd say the same about using LaTeX v. Word or OpenOffice.

> Movie Playing

Much though I love Ubuntu, I've been using [GeexBox]("ttp://geexbox.org/en/index.html") to play DVD's, etc. recently, and it's really nice. It's MPlayer-based and really streamlined.

-KDPR
