### Post by coolowlBrisbane on 2006-05-24
[QUOTE=pindar]What about network? If you have a router or switch, you could just plug both machines into it, or you could simply connect them with an ethernet cord and assign some fictitious IP address to both.[/QUOTE]
It may be as simple as using either straight Ethernet or crossover Ethernet between both machines.

Make sure 'sharing' is turned on via System Preferences - I turn it on for both machines, but that mightn't be necessary. Hit Cmd K or select 'connect to server' in the GO menu. You get a window 'connect to server' and you can browse for a machine connected.

In earlier OSes you had to assign an IP address - it's a while since I've set this up so I'm relying on memory - but I don't remember typing in any IP addresses at least in Tiger. Both machines need to be awake, whether straight or crossover Ethernet is necessary depends on the hardware, if one doesn't work the other probably will. I needed crossover for the Lombards but the PBG4 only works with straight.
