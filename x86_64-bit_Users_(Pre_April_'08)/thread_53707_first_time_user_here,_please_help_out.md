### Post by negatory on 2005-08-02
Hi there!
First you've downloaded the wrong file,the right file is "jre-1_5_0_04-linux-amd64.bin".Second,in a terminal, you must "chmod +x jre-1_5_0_04-linux-amd64.bin" and then "sudo ./jre-1_5_0_04-linux-amd64.bin" and proceed with the instalation.Third "that bar/graph thingy on the side of the screen" you are talking about must be gkrellm,you can get it with apt, just "sudo apt-get install gkrellm".
