Also the issue is not will it run on the cpu you have,The issue will come down to your hardware as whole, and without knowing the specs it will hard for anyone to give you a complete answer.

In any case.The 64bit iso is brand nutral, and will run on your 64bit machine be it amd or intel. 

To use with Amd your system/cpu should use AMD64 also known as x86-64 as the instruction set architecture.

This would include Athlon64,X2,Opteron,and some Sempron processors,As well as all curent am2 processors

To use with Intel your system/cpu should use Intel 64 also known as EM64T as the instruction set architecture.

This would include current Xeons,Core2 duo,quad,and some PentiumD processors
