### Post by IYY on 2006-03-29
Well, if your burn the ISO file to a CD properly, you should be able to just reboot your computer with the CD in the drive, and follow the instructions on your screen. If instead you still boot into Windows, you either didn't burn the CD correctly (just copying the file into a data CD doesn't cut it), or don't have the CD drive as your top boot priority. If this is the case, you should change it in the BIOS.
