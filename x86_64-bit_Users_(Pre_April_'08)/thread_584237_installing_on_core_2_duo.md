Intel 64 also known as EM64T is Intel's implementation of AMD's x86-64. 

Support comes in newer versions of Pentium 4, Pentium D, Pentium Extreme Edition, Celeron D, Xeon, and Pentium Dual-Core processors, and in all versions of the Core 2 processors.

64bit iso links are clearly given in the thread below. Download the version you want.
[Please Read First Advantages and Disadvantages of 64bit. (Plus install Guides)]("http://ubuntuforums.org/showthread.php?t=368607")

The above thread also clearly states (The iso's are also for intel users with EM64T ie: Pentium D/Celeron D/Core2Duo/Core2Quad/Xeon processors/and certain Pentium4's)
