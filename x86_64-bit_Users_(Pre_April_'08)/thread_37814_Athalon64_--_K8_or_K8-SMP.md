### Post by ::DoGG:: on 2005-05-28
SMP is only for multiprocessor machines. Windows is seeing HT as 2 processors cause without it xp wouldn`t be able to cerrectly drive the Hyper Thread future. So smp is only for opteron.

### Post by Optimal Aurora on 2005-05-29
Yes smp is for multiprocessor or multi-core machines... if your system came with something saying just AMD athlon64 then only use the K8... If it came saying AMD athlon64 X2 then use the SMP, or if its an Opteron and there is more than one Opteron processor then use SMP otherwise you will be better off to use the K8 only... and not the smp...
