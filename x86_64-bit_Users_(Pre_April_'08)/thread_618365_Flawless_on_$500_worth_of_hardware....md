### Post by nutz on 2007-11-20
Went to the local retailer yesterday to build out a new Ubuntu PC. I walked out with $518.32 worth of new goodies and quickly rushed home. Here is the list...

Athlon 64 X2 5600+ Processor
ECS GeForce 6100SM-M Mainboard
4GB Patriot PC6400 800mhz Memory
Maxtor SATA2 500GB Hard Disk
Antec NSK1300 mATX Case

All items were retail versions with the extra goodies like cables and such.

I installed the 32bit version of Gutsy first but quickly realised I couldn't use all 4GB of system memory in 32bit mode. Then I grabbed the AMD64 build of Gutsy and burned it on a CD...


This machine has been running a clean install of 64bit Gutsy for about 24 hours now and its flawless. Everything works perfect and despite the graphics only being a GeForce6100 they are liquid smooth and all the effects work perfectly. I couldn't possibly be more pleased with the way the AMD64 version of Ubuntu runs on this PC...
