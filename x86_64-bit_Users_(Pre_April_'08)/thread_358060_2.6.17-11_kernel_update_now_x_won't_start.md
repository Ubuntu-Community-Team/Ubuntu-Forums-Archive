### Post by kenden on 2007-02-18
I have found an easier solution than uninstalling/reinstalling the nvidia driver.

Just boot in the old kernel, start synaptic, search for "nvidia", you will see two packages:
linux-restricted-modules-2.6.17-10-generic , that is installed
and
linux-restricted-modules-2.6.17-11-generic, that is not installed

So install:
linux-restricted-modules-2.6.17-11-generic 
and reboot in the new kernel.
That's it!:) 

You can also use apt-get, from the console:
sudo apt-get install linux-restricted-modules-2.6.17-11-generic

Hope this helps,
kenden
