### Post by Yellow Pasque on 2007-12-11
After you install xorg-driver-fglrx, run:
```
sudo aticonfig --initial --overlay-type=Xv
``` 
See where that gets you.

BTW, what kind of video card do you have? If you don't know:
```
sudo update-pciids
lspci | grep "VGA"
```

### Post by vanadium on 2007-12-11
What might help to allow your system to boot in graphics mode again is to copy an old version of your /etc/X11/xorg.conf file over the current one. Usually, sole old versions are also laying around in the /etx/X11 directory. You can view them (including date) with the command "ls -l /etc/X11/*
