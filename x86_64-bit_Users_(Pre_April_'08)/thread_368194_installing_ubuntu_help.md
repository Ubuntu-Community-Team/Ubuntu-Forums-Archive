A little more information would help diagnose the problem.

Why are you installing the alternate version? Did you try the regular version and it failed? If so, why? Error messages?

Have you successfully installed any version of Linux on your computer before? If so, which one. If not, did you try but fail, in which case, what happened? 

Assuming that the only problem is a bad CD, where did you get the alternate version? How did you create the CD? if it was from ubuntu.org, did you check the md5sums when you burned the image?

Sometimes error messages are misleading. Programmers are lazy and make one error message do for various problems. Maybe the files are not corrupt and the real problem is something else. For example, is there enough room to install Ubuntu?
