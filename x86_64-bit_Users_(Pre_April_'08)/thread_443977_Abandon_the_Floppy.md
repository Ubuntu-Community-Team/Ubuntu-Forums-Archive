### Post by strumluff on 2007-05-15
The day of the floppy is gone, the last few years it was only possibly required if you wanted to flash your bios or boot up a rescue prompt. Now we don't even need it for that as we have bootable cd's with nice pretty GUI's and a suite of features with them. 

But then again just to add an argument for the other side :P, if you don't want to use cd's for booting the tiniest of programs/utilities, a floppy drive nowadays costs £3, absolutely nothing. Then again modern motherboards now allow for booting off usb-hdd's so if you have a flash pen drive then you may aswell use that.

Floppy's are a thing of the past, oh but I will never forget the memories like 28 floppy disks to install an office writer program. Please Insert disk 16 of 28 \\:D/  

I will never forget how it doesn't matter how carefully you looked after a floppy disk with important data on, the smallest amount of static electricity made the disk unreadable.

Floppy disks, I salute you.
