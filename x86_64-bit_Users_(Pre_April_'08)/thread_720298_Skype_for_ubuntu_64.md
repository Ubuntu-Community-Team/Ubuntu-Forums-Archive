### Post by jespdj on 2008-03-10
> **kdarkentity said:**
> can ekiga make calls over to a Skype user?
No it can't, and there is no program that can talk to Skype clients, because Skype uses a proprietary, encrypted protocol which has not (yet) been reverse engineered by open source developers.

There is no 64-bit version of Skype, but installing the 32-bit version on your 64-bit system is easy. Have a look at [HOWTO: Install Skype on 64-bit Ubuntu](http://ubuntuforums.org/showthread.php?t=432295).
