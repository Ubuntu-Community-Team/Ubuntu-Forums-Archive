No, its not possible to choose one or the other without installing to a different partition. But you could run the 32bit version as a virtual machine with vmware player. [Go here to get the virtual machine]("http://www.vmware.com/vmtn/appliances/directory/843"). Then install vmplayer thats in the repos. That way you wouldnt even have to reboot, just start the player.
