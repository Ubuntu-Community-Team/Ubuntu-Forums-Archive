### Post by MariusSilverwolf on 2007-10-29
I tested out all 4 available MS browsers on the work laptop.  Yeah, Opera is a little different, but here's what I've learned that I like:

* Speed Dial:  Having 9 paned bookmarks at startup makes hitting my most oft visited sites easier.
* Skins:  Not as many as FF but still enough to alter the basic look of things
* BitTorrent built-in: Helps sometimes, especially if I don't want to bother loading up a 3rd party client
* Image support:  PNG and RAW images look better for some reason
* Widgets: Built in support for widgets.  Useful if you like 'em.

Yeah, it takes some getting used to, but so did the switch from Mac to MS back in the day, and the switch from IE to FireFox for MS users, and the switch from MS to FF.

Opera is almost as customizable as FF, though.  It's a nice feature.  Either way, still far superior to the default options on an MS OS.
