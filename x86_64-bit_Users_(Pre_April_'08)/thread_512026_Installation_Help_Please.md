### Post by qneill on 2007-07-30
There can be hardware problems with the installation software itself.   Post as much info as you can about the PC: PC brand and model, motherboard brand and model, hard drive brand and model, video card or chip, monitor. 

Then I would start googling for someone else who's had similar problems:
  'ubuntu "install problem" [PC model]'

If it is a video card problem, try installing using in text mode.  IIRC, there is a choice at install time to do this.
