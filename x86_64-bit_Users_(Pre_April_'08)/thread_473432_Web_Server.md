### Post by simonn on 2007-06-14
You can always compile from source.

Is there a reason that you need the latest release?

Basically, unless there is a real resason to upgrade to the latest release, e.g. security issues or a new inispensible feature, then the devs will probably not upgrade the versions in the repositories for the version of Ubuntu you are using. This is a Good Thing (tm) because it actually makes the system more stable and means that the devs spend their time doing infinitely more useful things than pointlessly worrying about having the latest point release.

If you want to upgrade to the latest and greatest for no other reason than it is the latest an greatest, then Ubuntu is probably not the distro for you. Sorry, but it kinda-sorta has to be said. However, this is one of the reasons I moved to Ubuntu from Fedora.
