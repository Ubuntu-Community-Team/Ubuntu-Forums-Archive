### Post by kuja on 2006-10-04
All current 64-bit processors, with the exception of maybe a few server processors, are capable of either running 32-bit or 64-bit code. Thus, you can install 32-bit programs in your 64-bit setup, and unless the program was trying to do something tricky (and those are far and few between), the program will run without a hitch.

As per things that may give people trouble, check the sticky in this subforum, it has information on several common things.
