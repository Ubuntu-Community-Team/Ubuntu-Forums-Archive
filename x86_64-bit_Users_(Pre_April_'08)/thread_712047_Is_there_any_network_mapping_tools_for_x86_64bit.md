### Post by gmcastil on 2008-03-01
Nmap can be found in the standard Ubuntu repository.  Try something like:

sudo apt-get install nmap

It's pretty much the best tool around for network and port mapping.  There might be a GUI frontend for it, but I don't know what it would be offhand, since I do the majority of my work from a command line.
