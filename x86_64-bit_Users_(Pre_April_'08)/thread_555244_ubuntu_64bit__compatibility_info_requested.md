### Post by rsambuca on 2007-09-19
If you are running the 64bit version of Ubuntu, then any programs that you install through the "Add/Remove" menu or by using the "Synaptic Package Manager", or via the command line "apt-get install packagename" will be 64bit compiled versions, so don't worry.

As for the java and flash, there is a [how-to located here]("http://ubuntuforums.org/showthread.php?t=202537").  Just a couple of clicks and you are done.
