### Post by lH)4~mK0e on 2008-03-22
If you go into System --> Administration --> Users and Groups, there will be the list of the available users.  If you select the user and click "Properties" you will see a tab that says "User Privileges."  Make sure there is a check mark in the box "Administer the system."  That way if there are changes made to compiz it will save those changes.

Hope that helps.
