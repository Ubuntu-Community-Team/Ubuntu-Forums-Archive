Thanks.

While it is possible in theory to run a 32bit operating system. Some owners of 64bit hardware report problems (errors, lockups, things not recognized) when running a 32bit operating system. It is not guaranteed that you will have an error free install.
As for performance, you will get the best performance running a 64bit operating system on 64bit hardware. But the improvement may differ depending on what applications you are running. Some applications are written to take advantage of the hardware (encoders, rendering, 3d, number crunching applications) and some is just recompiled 32bit code (firefox).

### Post by mancel on 2007-03-27
I started with the 64 bit version because I figured I'd get all these performance gains with my X2 as well.
The thing is, most software is still 32 bit so there's really no increase in performance unless you're running 64 bit software.  I encountered a lot of compatibility issues as well.  As mentioned, Flash won't work with the 64bit version of Firefox so in order to make it work you have to install the 32 bit version.
If you're new to Linux I suggest you go with the 32 bit version.  You won't notice any difference in performance and it runs just as stable on a 64bit cpu as it does a 32.
