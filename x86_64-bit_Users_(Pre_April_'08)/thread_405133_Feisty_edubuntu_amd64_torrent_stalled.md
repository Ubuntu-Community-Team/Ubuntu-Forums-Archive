Stalled in torrents can mean a few things, I used to run a tracker a long time ago.
1. There are no more people seeding the torrent. You have all the available parts and there is no one with a complete file.
2. The tracker server is offline.
3. The torrent was removed from the tracker.
4. Your client cant connect to the tracker.
