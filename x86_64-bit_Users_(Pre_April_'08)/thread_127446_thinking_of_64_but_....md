### Post by dickohead on 2006-02-08
I know that there is no 64bit flash player yet. But a good idea will be to do this:

Install 32 and 64 bit versions, use the same /home partition on each (make sure /home is on a seperate partition of course) and make them also use the same swap space. That way you will have 4 partitions:
64 /
32 /
/home
swap

Or... if you can figure out how to get chroot jails working, you can install the 32 base system within 64 and install 32 bit programs that you need, there's a  really good tutorial on this forum... I think it's a sticky, or just search for "64 bit synaptic"
