### Post by RAOF on 2007-09-04
The 32 bit version will be using the processor's (legacy) 32bit mode.  While there are conditions where the 64bit mode will give better performance (it is basically easier for compilers to optimise for) unless you are doing computationally intensive tasks it is unlikely that the performance difference will be noticable.
