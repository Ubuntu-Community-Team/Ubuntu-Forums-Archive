### Post by cmost on 2007-06-22
Firstly, Compiz-Fusion is in early alpha.  I would hesitate to run it on my system.  While it's true there are some cool new effects (reflections, dodge, fire annotate...) you will lose some features already in Beryl.  If you insist, however, I do know they're in Gutsy's Universe repository.  Did you try adding Gutsy repos and grabbing them from there?  I'm not sure what kind of dependencies it will try to install along with it (could really mess up your system.)  Alternately, you could always compile your own debs.  Daunting task yes but highly educational.  Good luck!
