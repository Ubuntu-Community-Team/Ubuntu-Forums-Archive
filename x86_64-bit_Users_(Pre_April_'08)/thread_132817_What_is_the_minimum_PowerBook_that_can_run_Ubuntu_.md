### Post by jdp on 2006-02-19
The Lombard is NewWorld as well.  The main thing to look for is factory built-in USB. The G3s earlier than the Lombard (actually it's the 101... ;) ) don't have USB, and all iBooks do.  So, any iBook or any PowerBook from the 101/Bronze Keyboard on up should boot Ubuntu without BootX.  The same should apply to desktops as well.  The Beige G3 won't boot Ubuntu without BootX, but the B&W can, as can the iMac.
