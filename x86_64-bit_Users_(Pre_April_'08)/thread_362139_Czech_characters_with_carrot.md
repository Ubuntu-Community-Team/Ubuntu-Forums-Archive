### Post by John Jason Jordan on 2007-02-15
You need to change the keyboard mapping to Czech. Go to System > Preferences > Keyboard and then the Layouts tab. You must also use a font that has the characters that you need. You should also be sure you have the dictionaries and other language packs for Czech installed, otherwise your word processor and other applications will not be able to do spell checking.

If you only need to type these characters occasionally, then you can type Ctrl+Shift+u followed by the hex number for the Unicode character in the font. You can find the hex numbers for the characters that you need in Character Map.
