### Post by JoshLukas on 2008-03-29
Yes, yes. If you are a gamer type, then linux is definitely not the OS you want to use. ... But... If you're going into real production, whether it's some office work, creating music, developing software or whatever, linux could be the better native OS, you want to use.
For a long time I used Win2k, then XP, because of Photoshop and some other Adobe programms, which will not run under linux. Today, I run Photoshop on a VM (using virtualbox) on a XP system. It's probably the best way on working with my needed applications hardly loosing performance.
Just think what your needs are. For the reason of stability, I suggest that a clean installed ubuntu, or any other gnu/linux is more stable then any windows available on market.
