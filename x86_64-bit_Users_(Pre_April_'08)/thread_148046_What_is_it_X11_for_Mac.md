### Post by fraterf93 on 2006-03-21
The Macintosh GIU is called Aqua.  You do not need X11 installed to use Mac OS X, and the Aqua user interface functions the same whether you have X11 installed or not.  What X11 IS is the X window environment commonly found in Unix based systems (Ubuntu included) for Mac OS X.  What this means is that you can run all of your fav free Unix, and Linux apps under X11 (GIMP, Openoffice, etc.) natively in Mac OS X.  Though some may need to be recompiled.  If your not fimilliar with the command line this can be a bit intimidating, but Apple has plenty of documentation on the subject, do a search on their web site.  Kinda makes one wonder what then is the point of installing and running Linux on a Mac huh? :-k

### Post by jdp on 2006-03-21
As was pointed out, the OS X GUI is called Aqua.  That's not just a brand name for their own X11, as Aqua is actually a fully custom Apple GUI solution.  There is the X11.app software available that provides an interface for programs made to run under X11 to also be able to display on Aqua.  They won't get the pretty Aqua interface, but you can go so far as to install your fav WM, like Gnome or KDE, if you want to and run that along side Aqua.
