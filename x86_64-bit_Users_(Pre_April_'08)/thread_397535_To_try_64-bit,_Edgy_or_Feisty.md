There are 2 kinds of Ubuntu install cd's 
Desktop or live cd= This is a live CD. You can test out the distro. You can run the operating system from the cd and install from it to.
Alternate install cd = This is a pure install cd. It doesnt have the live operating system to try out. The graphics are not as pretty. But it will install on machines that may be having problems running a live cd.
