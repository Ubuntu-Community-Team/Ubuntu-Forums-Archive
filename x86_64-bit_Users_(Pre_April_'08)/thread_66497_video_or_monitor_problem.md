### Post by DancingSun on 2005-09-19
What resolution did you select when installing Ubuntu?  I once tried to start the Breezy Preview LiveCD with 1280x1024 as max resolution and got something similar.  It all worked fine when I bootup again and selected 1024x768 as the highest supported resolution.

I forgot the command (do a search for it), but try reconfiguring X with a lower max resolution.  800x600 should be relatively fail safe.  After that, install the nVidia drivers via Synaptic and see if that lets you raise the resolutions.

Oops, just saw your monitor is LCD, in that case try 1280 x 1024 as the highest resolution, since that's your LCD's native resolution.

Also, you might want to comment out the refresh rate lines in xorg.conf, so that X can attempt to automatically detect the proper refresh rates.
