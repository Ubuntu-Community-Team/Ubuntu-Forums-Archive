For evolution 2.0 and later:

Select **Edit->Preferences** to get the required popup. Select **Mail Accounts** on the left, though this should be the default. Select the account for which you wish to set the server details then click the **Edit** button on the right. The tabs which you probably need to fix are Receicving Mail, Receiving Options, & Send Mail.

Ealier versions of Evolution have **Tools->Settings** instead of Edit->Preferences in the main menu.

Don't forget to enable the account before closing the preferences dialogue. :)
