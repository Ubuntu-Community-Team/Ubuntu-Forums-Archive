### Post by Ptero-4 on 2006-04-19
Sound/Speakers: Yes
Airport: Yes(Through experimental driver or through Mac-on-Linux using OSX)
Suspend: Yes (probably needs kernel patch for breezy, fully functional on Dapper)
Modem/Ethernet: Partial (Modem only with beta drivers, only up to 14.4k, Ethernet funcitonal)

And other thing: Flash isn't available on Linux for PPC, w32codecs aren't available for PPC either.
