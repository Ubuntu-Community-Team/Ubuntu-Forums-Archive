### Post by ts2000 on 2007-11-20
After installing gMFSK I noticed the audio quality was crackling or ticking every half a second.  I fixed this by increasing the default sample rate on the soundcard from 8000 to 48000 in the settings.

I've also installed wine and with that installed ham radio deluxe, G4FON cw trainer, G4ILO voaprop, and ionoprobe.  All run brilliantly under wine.  I run ionoproble by accessing my windows installation of ionoproble and running the exe program by selecting 'run with'=wine.
