I have made great strides with Feisty 64 in permanently retiring MS as my OS.  Like many 64 bit users, multimedia (both sound and video)  has been one of the biggest challenges.  I now have four versions of Firefox on this system: Firefox 32 bit, Firefox 64 bit, Swiftfox 32 bit and Swiftweasel 64 bit installed.  I notice that all of them recognize certain common configurations (like Favorites bookmarks and preferences) but otherwise are differentially configured.  Swiftweasel calls Totem and can play .mvs files.  Firefox 32 will play flash content.   Swiftweasel dies looking for Flash plugins and Firefox 32 stops when trying to play .avi files  In total, the four browsers will do everything I need but none is configured correctly by itself.  I want to use Swiftweasel or another 64 bit variant for everything if I can.  Is there a way to commonly configure any of them? I know that this is a very common goal and if there is a link to a FAQ on this I would welcome it.  If I have to use multiple instances of Firefox for now, that's ok.
