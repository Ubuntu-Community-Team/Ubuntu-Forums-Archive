### Post by Kilz on 2007-04-20
The easiest way to fix this is to at the command prompt type
```
sudo dpkg-reconfigure xserver-xorg
```

Then more or less agree with everything. At least that will get you back to a gui. Otherwise just reinstall if its a new install with not much done.

### Post by rivera151 on 2007-04-20
Also, I noticed you have an ATI PCI Express card and you are using fglrx.  On my system, this works better when you set your memory on BIOS to SIdeport Only (e.g. use dedicated vram only).  I'm not sure if this nuance is only on integrated graphics (like on my laptop) or with all ATI PCIe products, but it's worth looking into.  My $0.02.
