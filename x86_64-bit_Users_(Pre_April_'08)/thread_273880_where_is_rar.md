### Post by H.E. Pennypacker on 2006-10-08
Rar is a file extension, not something you would install.  You would, however, install the capability to open a rar file.  

Follow this link for how to install the capability to open one:

[https://help.ubuntu.com/community/FileCompression#head-32ba956d13d49934f65bf67dd40646653a7a6140](https://help.ubuntu.com/community/FileCompression#head-32ba956d13d49934f65bf67dd40646653a7a6140)
