### Post by bonzodog on 2005-11-14
well, if you've read the forums you will know that flash on 64 bit is a problem as Macromedia refuse to support it(windows as well). So most of us end up running gplflash or similar in an attempt just have it display on web pages. A more drastic alternative is to run firefox32 in a chroot environment. As for web developing please remember that w3c standards are important, so those of us with a dedication to linux and open source tend to do it natively on quanta or Bluefish or similar.
