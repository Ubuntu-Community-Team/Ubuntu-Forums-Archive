### Post by stuporglue on 2005-12-27
I don't have a G5 iMac, but I ran Linux on my iBook for three years. It was a G4 but I think most of it should still apply.


You won't have Flash, good wireless support, or Java 1.5. You will have support for just about everything else. 

As for ATI, you're actually not so bad off. The in kernel drivers do a pretty good job for most cards. Good enough for BZflag anyways :-D 

It *is* now possible to get wireless working under Linux. It's a dirty hack, but it works. You have to run Mac-on-Linux. The wireless is tunneled through Linux to OSX then to Linux and back again. 

If Ubuntu isn't working for you, you've got other options.
[http://stuporglue.com/content/view/14/32/](http://stuporglue.com/content/view/14/32/)
