### Post by cmost on 2007-02-18
Do you have something plugged into one of the USB ports (e.g., external hard disk or CD/DVD-ROM, memory key, dongle, etc.?)  If so, unplug everything except the bare essentials and start the installation again.  Another possibility is that your install disk is a little bit corrupt.  If all else fails, burn a new disk from your ISO image but at the lowest speed possible.  Post back your status.
