That lib isnt in Ubuntu,[ I did a search]("http://packages.ubuntu.com/"). My best guess is that its a Debian .deb file and not a Ubuntu .deb file. A lot of the time packages from Debian will work in Ubuntu, but there are always exceptions.
You have 3 options, you can install the version [in Synaptic]("http://www.monkeyblog.org/ubuntu/installing/#installing_with_synaptic"), try and [compile it yourself from source]("http://www.monkeyblog.org/ubuntu/installing/#source"), or lastly (not recommended as it can make your system unstable) install some [Debian libraries]("http://www.debian.org/distrib/packages").
