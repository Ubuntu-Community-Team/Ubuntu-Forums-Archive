### Post by picopir8 on 2007-09-20
Over the past few months I have installed 32-bit feisty on:
64bit AMD
Core 2 Duo (64 bit)
Core Duo (32 bit)
and Pentium 4 with hyper threading (32 bit)

I also put  the 64 bit version on the above mentioned AMD after realizing I accidentally installed the 32 bit version.  

All of the installs went the same with a few minor problems here and there.  Wireless card not detected on AMD, video card not detected in Core 2Duo or P4, wrong (earlier generation) video card detected in Core Duo.  Other than those, no problems.  No issues running 64 bit os or running 32 bit os on 64 bit machine.  The only issue with the 64 bit os is no flash support but there is a work around (and IIRC, flash works in 64 bit gutsy).
