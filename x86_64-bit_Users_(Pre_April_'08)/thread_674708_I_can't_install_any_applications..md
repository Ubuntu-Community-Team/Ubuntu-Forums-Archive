### Post by misfitpierce on 2008-01-22
sudo apt-get install wine in the terminal is easiest route.

Its the stock version of wine but it should work fine. Also note that sometimes switching discs wont work on installing WOW. Best idea is to copy file from each cd into a folder then click setup in folder and itll auto install with all files there.
