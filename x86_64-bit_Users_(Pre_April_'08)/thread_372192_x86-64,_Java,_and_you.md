You can install several of them. In some apps (e.g., OpenOffice.org), you have to specify which one the app should use in the app's preferences or options dialog box. Some apps just use whatever you have designated as the default. I have Blackdown, Sun 5 and Sun 6 installed. On my computer everybody likes Sun 5 except Firefox. Nobody likes Sun 6. I set Sun 5 as the default and installed the Blackdown plugin for Firefox.
