### Post by strumluff on 2007-05-27
Last I checked there was only a deb for feisty.

I know it works on Dapper because I compiled it from source on a friends pc without firefox 2 installed (never heard that it depended on ff2's libraries).

Just download the source from the pidgin home page, extract the folder to a directory and navigate to the directory in the terminal and do a 

```
./configure
make
sudo make install
```

or use checkinstall to make uninstallation easier.

But still if you don't have a particular reason for using Dapper,then Feisty 64 is just great, you'll probably find life a lot easier using it \\:D/
