### Post by rh1zome on 2007-09-08
Hi all,

Is anyone out there running Ubuntu 7.04 (or Gutsy) on an Intel Pentium E2140 or E2160? If so, I'd be interested in hearing your experiences (i.e. has anyone encountered any quirks or stability issues?) 

Just some back story. I'm looking for a budget CPU for a new (Feisty) system and have noticed that the E2140 and E2160 seem to be getting a reputation as the new 'Celeron' of the Intel dual-core world. I'm more interested in flexibility and multi-tasking capabilities than straight-line speed. I'm not bothered by the relatively small L2 cache (1MB shared between cores) as I've already got one Feisty system running on a Celeron 2.6GHz with 256K of L2 cache (and 1GB RAM) that's running very sweetly (responsive and stable). Given that each core of the E2140 should theoretically be able to outperform the Celeron, I'm looking at the E2140 as a (so to speak) dual-core Celeron with superior internals, rather than viewing it as a crippled Core Duo. 

Anyway, would be interested to hear of people's experiences with this CPU. I'm not really anticpating any significant issues (being Intel 'n' all), but it's a relatively new CPU and it's not clear to me exactly what differences there are between this and the mainstream Core Duos (apart from the L2 cache size).

For those who haven't encountered this processor, there's a brief but encouraging review here (especially if you're an over-clocker!):

### Post by rh1zome on 2007-09-18
OK. I've been running Feisty on this CPU (these CPUs?) for at least ten hours per day for the last five days or so and so far, all is fine. The only glitch I've encountered has been my mouse pointer's propensity to suddenly leap to the edge of my display about once every fifteen minutes or so, for no apparent reason. This is a little irritating, but tolerable and it's presumably down to my graphics setup (dual-head, TwinView on an nvidia 7600GS) rather than my CPU.

For the record, the relevant details of my setup are:

- Pentium dual-core E2140 (1.6GHz, 1MB shared L2 cache)
- Asus P5L-VM 1394 M-ATX motherboard
- Nvidia 7600GS (passively cooled) running in TwinView mode using proprietary Nvidia driver (1280x1024 NEC 175VXM+ and 1680x1050 Philips 200W panels)
- 2x1GB 240pin DIMMs, PC2-5300, Non-ECC RAM
- Xilence 420W PSU

The above minor glitch apart, this configuration is working fine and runs 64-bit 7.04 sweetly. Despite the reduced CPU L2 cache and 800MHz FSB, the 2GB RAM and dual-cores mean that I'm seeing few slow downs. If you are building a budget systsem, or else just don't need the extra cache, save yourself around (UK) £30.00 over the E4400 and go for one of these cheap and cheerful little beasties.
