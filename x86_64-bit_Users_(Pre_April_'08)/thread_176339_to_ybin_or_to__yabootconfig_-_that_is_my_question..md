### Post by ssam on 2006-05-14
you might find some usefull info on [https://wiki.ubuntu.com/YabootConfigurationForMacintoshPowerPCsDualBoot](https://wiki.ubuntu.com/YabootConfigurationForMacintoshPowerPCsDualBoot)

if it helps here is my /etc/yaboot.conf
```
## yaboot.conf generated by the Ubuntu installer
##
## run: "man yaboot.conf" for details. Do not make changes until you have!!
## see also: /usr/share/doc/yaboot/examples for example configurations.
##
## For a dual-boot menu, add one or more of:
## bsd=/dev/hdaX, macos=/dev/hdaY, macosx=/dev/hdaZ

boot=/dev/hda9
device=/pci@f2000000/mac-io@17/ata-4@1f000/disk@0:
partition=12
root=/dev/hda12
timeout=100
install=/usr/lib/yaboot/yaboot
magicboot=/usr/lib/yaboot/ofboot
enablecdboot
macosx=/dev/hda11

image=/boot/vmlinux
        label=Linux
        read-only
        initrd=/boot/initrd.img
        append="quiet splash"

image=/boot/vmlinux.old
        label=old
        read-only
        initrd=/boot/initrd.img.old
        append="quiet splash"

# This entry automatically added by the Ubuntu installer for an existing
# Linux installation on /dev/hda13.
image=/pci@f2000000/mac-io@17/ata-4@1f000/disk@0:13,/boot/vmlinux
    label=hda13-Linux
    root=/pci@f2000000/mac-io@17/ata-4@1f000/disk@0:13
    append="root=/dev/hda13 ro quiet splash"
    initrd=/pci@f2000000/mac-io@17/ata-4@1f000/disk@0:13,/boot/initrd.img

# This entry automatically added by the Ubuntu installer for an existing
# Linux installation on /dev/hda13.
image=/pci@f2000000/mac-io@17/ata-4@1f000/disk@0:13,/boot/vmlinux.old
    label=hda13-old
    root=/pci@f2000000/mac-io@17/ata-4@1f000/disk@0:13
    append="root=/dev/hda13 ro quiet splash"
    initrd=/pci@f2000000/mac-io@17/ata-4@1f000/disk@0:13,/boot/initrd.img.old

```

### Post by davygravy on 2006-05-15
My results & summary:

No need to use yaboot config...

I just:

1.  did a "sudo nano /etc/yaboot.conf"

and  added the following lines


macosx=/dev/hda10
defaultos=macosx


2.  I did a  "sudo ybin -v"

and it did it's thing....

All is well now.

davygravy
