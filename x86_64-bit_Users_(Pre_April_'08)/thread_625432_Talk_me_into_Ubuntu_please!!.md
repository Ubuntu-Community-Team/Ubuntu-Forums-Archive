> H - and why do the other distro's have 4gb distros but U is only 1 CD? Not that I like useless fluff, just wunderin....

Well, the ubuntu is a liveCD designed to work on as much hardware as feasible (not every computer has a DVD reader), but assumes you have an internet connection, basically. You might want a DVD with all your packages on it if you were, say, installing an OS on hundreds of computers, etc. But even in that case an organized person will have a local network repository set up with the packages they need.

So, that's kind of why, I guess.

I'd have to say the KDE hotkey setup is probably the best n00b-friendly/easy to manage setup going in Linux these days.  The fact is that you can set up great hotkey schemes in other desktop environments/window managers, but it may not be so nice and easy as in KDE.
