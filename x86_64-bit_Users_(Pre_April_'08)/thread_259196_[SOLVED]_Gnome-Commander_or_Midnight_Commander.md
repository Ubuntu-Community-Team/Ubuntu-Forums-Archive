### Post by kuja on 2006-09-17
As for the two of them, GNU Midnight Commander is a text-mode file manager that has been around for ages. Gnome-commander is a GTK+ file manager based on midnight commander. If you don't mind text-mode interfaces then MC might be a good candidate for you. Then again, if you want something that blends in better, and you don't mind having what is likely to be fewer features then gnome commander might work. It's all up-to-you really.
