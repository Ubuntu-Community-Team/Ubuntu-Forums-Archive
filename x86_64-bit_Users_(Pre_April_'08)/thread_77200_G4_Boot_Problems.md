### Post by Brian McConnell on 2005-10-17
it may not solve your problem but

```
sudo gdm
```

will start gnome's graphical login if your /etc/X11/xorg.conf file is setup properly. If X won't start, then xorg.conf is probably not set-up correctly.

To do that, you can try editing the text by hand:

```
sudo nano /etc/X11/xorg.conf
```

or, if you know your hardware well enough:

```
sudo dpkg-reconfigure xserver-xorg
```

which will walk you through xorg.conf reconfiguration by asking questions like what's your graphics card name and select a driver from the list and what's your monitor's refresh rate, etc.


of course it's possible that everything is fine and that Gnome is defaulting to non-graphical login which I'm not sure how to help you resolve. It'd probably be something like runlevel init2 or something similar. I'm a newbie.
