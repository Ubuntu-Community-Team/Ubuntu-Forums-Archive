### Post by RAOF on 2005-10-18
It will almost certainly not be added to the Breezy repositories - once an Ubuntu version is released it doesn't get new versions of programs ('cause they won't have been tested, it makes it more difficult to support, etc)

That's one of the reasons for the rapid release cycles.  Once Dapper gets well underway, you can expect an (unofficial) backports repository to open up.  That will contain some of the new stuff from Dapper, repackaged for Breezy.  That's where you'll find MonoDevelop 0.8
