### Post by gingerbread_man on 2006-05-02
People install linux for a great many reasons, but I think we can narrow down those reasons to a few in this forum.
Over on the pc world, people cite lack of security, viruses and unreliability of Windows to moving to Linux (desktop users). However, you can't apply those same reasons to the mac.
What really intrigues me is that installing Linux on a mac is painful compared to the breeze it is on a pc. Linux is still not ready for macs. It is almost impossible to non-destructively install linux on a mac. You need to take great care in how your partitions are setup and stuff like that.
and still the end result isn't that great. most people report sound not working, airport doesn't work and in some cases sleep/hibernate doesn't work either!

As I see it

-You already have a copy of OS X, a UNIX based OS that is arguably a far better desktop operating system. (sucks as a server OS)
-almost all linux based software runs fine on OS X, in addition to the OS X only software that there are plenty of.

-it's just as secure, virus free and reliable as linux is.

So this is my question. If not for the thrill of trying out something new. Why do you choose to run Linux? It it your OS of choice?
