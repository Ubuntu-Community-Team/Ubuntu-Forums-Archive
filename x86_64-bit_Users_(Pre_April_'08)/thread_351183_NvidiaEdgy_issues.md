### Post by John.Michael.Kane on 2007-02-01
@alex10819 what guide did you use to install your dirvers? 

If you have already installed the nvidia binary drivers. 

**Try this:**
```
This backsup your xorg:
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup

This command enables the the driver in your xorg:
sudo nvidia-xconfig

This creates a link to the “Nvidia-Settings” Panel:
sudo gedit /usr/share/applications/NVIDIA-Settings.desktop

Copy paste this info into the new file:

[Desktop Entry]
Name=NVIDIA Settings
Comment=NVIDIA X Server Settings
Exec=nvidia-settings
Icon=
StartupNotify=true
Terminal=false
Type=Application
Categories=Application;System;

save,and ctrl-alt-backspace
```

Theres more info here should you need it. [http://albertomilone.com/latest_nvidia_udsf_edgy.html#METHOD_1](http://albertomilone.com/latest_nvidia_udsf_edgy.html#METHOD_1)
