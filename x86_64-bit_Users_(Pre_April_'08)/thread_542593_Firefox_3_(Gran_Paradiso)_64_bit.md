> **shafin said:**
> And one last question,if I can run it,can i configure it before install so that it'll use a new profile folder.The last time I installed Gran Paradiso,it messed my FF2 bookmarks.Yep, you can.  First, run Firefox 2.0's profile manager and create a new profile (call it whateveryoulike).  Be sure to UNCHECK the option to not prompt for which profile to load when Firefox starts.  Then, when you start Gran Paradiso choose the profile you created and it won't mess with your Firefox 2.0 settings.
