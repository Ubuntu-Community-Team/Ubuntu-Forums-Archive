### Post by AndrewShugg on 2006-07-22
Ubuntu's pretty good.  I've got to say that first!

My first disappointment when trying out the PPC 5.10 LiveCD was that the wireless network didn't work ... I've since found that 6.06 (Dapper) has Airport working so that's good.

So I've gone ahead and installed Ubuntu 5.10, and it's working well.

The two things that are frustrating me though are:

1) I've turned up the volume control as high as it will go, and MP3s still play very softly. It's audible but I know from using Mac OS X on this machine that it can play a lot louder.  I tried nudging the PCM mixer control higher but it caused a lot of distortion.  I'm using the default GNOME desktop environment and haven't tried any funny stuff with ALSA or OSS at the command line, only what I could access through the GNOME menus.

2) With a 2-button USB mouse plugged in, I have full use of right clicks etc. But without it, with just the touchpad and the single button in front of it, I'm kind of hampered in Ubuntu. (Or maybe that should be "in GNOME".) In Mac OS X keyboard modifiers can be used with the single button mouse to make it do different things - command-click, control-click and option-click. I'd hoped these would work in Ubuntu/GNOME as well, but they don't, and that makes a lot of things hard, as many applications don't provide other ways of accessing the "right-click" functionality. I guess this is arguably an accessibility issue in GNOME.

So, has anyone else encountered and/or worked around either of these problems?  If so I'd be interested to hear about it.

Some time in the near future I will upgrade to Dapper but for the moment at least I want to keep using Breezy.

Cheers,

Andrew S.
