### Post by loumsmith on 2007-04-17
This is my first post to the site.  I apologize if this lacks sufficient detail.  I am running an HP desktop Pavillion a1700n.  I have a usb keyboard and mouse.  I have drivers instaled for an HP photosmart 1315.  No problems.  Today I added the R2400 printer.  Shortly thereafter the screen locked up.  On reboot I could not move the mouse or type.  I tried another reboot.  This time I was able to log on.  Soon I had the same problem.  This time the mouse would move only.  Not able to select an icon or type.  I removed the Epson R2400 driver and now I am fine.  I don't think this constitutes a bug, but I wanted to share.
