That all depends on what question you are asking. 
If you are asking if a celeron runs as well as all AMD chips, the answer is no because a celeron is a low end processor and amd has some good high end chips.
If you are asking if the AMD version of Ubuntu runs in intel chips just as well as it does AMD chips the answer is yes. The Ubuntu version name is historical because AMD was the first to make popular the 64bit platform for most users. Though the version is designed to run on both intel and ADM chips.
