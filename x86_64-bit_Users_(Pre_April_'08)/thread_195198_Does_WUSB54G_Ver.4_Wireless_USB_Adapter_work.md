### Post by vvlist on 2006-06-12
Yes, you can get it to work. You must compile the latest NDISwrapper though. I used version 1.16. Make sure you read the README in the ndiswrapper tar.gz so you know how to use it. And just a fair warning: if you use azureus, it makes the adapter stop working. Its a known bug. Other than that, good luck.
