### Post by stmiller on 2007-10-28
Ubuntu 64bit can run any and all 32bit code. There are a set of 32bit libraries (the ia32* packages) so that 32bit apps will just work. Backward compatible, roughly speaking.

Google Earth works. I just type in 'googlearth' in the terminal as normal. I believe it's in medibuntu if you want to apt-get and install it.
