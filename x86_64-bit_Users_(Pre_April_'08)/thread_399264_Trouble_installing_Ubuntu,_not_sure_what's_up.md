### Post by Majorix on 2007-04-02
Make sure your xorg.conf has a line like this:

> 	Option		"Device"		"/dev/input/mice"

To edit xorg.conf type this in Applications > Accessories > Terminal:

```
sudo gedit /etc/X11/xorg.conf
```

If the problem persists come back and we will help further :)
