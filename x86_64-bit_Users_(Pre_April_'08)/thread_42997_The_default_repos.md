### Post by FluffyElmo on 2005-06-20
The amd64 version is all 64bit packages (except for the odd 32bit package), and all the repositories are 64bit only as well. Most typical software you would want/need is available in 64bit format though, with more available as time goes on.

For packages that aren't available in the 64bit repositories (most notably Flash and  Win32 Codecs) more work is required. You can install a 32bit chroot (essentially a seperate 32bit Ubuntu install that runs under the 64bit OS), and then install and run 32bit applications from it.

Here is a how-to on installing a chroot to give you an idea of what is required:
[http://www.ubuntuforums.org/showthread.php?t=24575](http://www.ubuntuforums.org/showthread.php?t=24575) 

Myself, I've found 64bit to be fine for my needs. Typical user applications work fast and reliably and I don't mind spending some extra time to get the occasional app running.
