### Post by Mr_bleu on 2007-04-30
I tried kde and have tried switching back to gnome environment.  Upon booting into linux, I get an error message stating (not verbatim) "no exec file found for KDE. starting Gnome in failsafe mode."  I don't know how to correct this.  I've gone to system>preferences>session.  Gnome OSD is checked.  I am  still a linux newbie.  Any help would be appreciated.  In System>administration>login Gnome is the default session.

### Post by Mr_bleu on 2007-05-01
I think I fixed it.  I selected log out and hit the options button.  Then I selected session menu and put gnome as the default session there.  Rebooted, no error message.
