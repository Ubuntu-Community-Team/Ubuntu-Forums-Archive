### Post by grazie on 2006-03-11
In a terminal

$ sudo apt-get install fluxbox

If fluxbox isn't found, you'll need to update you repositories. On installation complete, logout, hit the Session button and select Fluxbox and then Login. On my machine the GUI slows right down while Fluxbox is loading, but after that it goes really well. Enjoy!

I'm assuming you're using the default Gnome install at present.
