### Post by blahquaker on 2007-10-19
to delete the old plugin: go to the firefox plugins directory and delete anything that says flash. you can find the exact name of the file by looking at about:plugins page in firefox and then searching for the file it names (mine was something like nswrapper.libflash.so).

then, to install the new one: go into system>administration>synaptic and install the flash plugin (adobe or gnash)

I tried both very briefly and the adobe seemed to be behaving better. gnash worked fine for some sites (youtube) but not others (espn, myspace) in my very limited tests. adobe seems to work more often, but not always the way you'd expect...
