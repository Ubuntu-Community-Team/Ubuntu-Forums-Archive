### Post by amohanty on 2006-03-11
> Ubuntu show me that the cd ubuntu has not the integrity and that I needed check the cd because i can not continue with the instalation or someting like that.


Some good links:
[https://wiki.ubuntu.com/GettingUbuntu](https://wiki.ubuntu.com/GettingUbuntu)
[https://wiki.ubuntu.com/BurningIsoHowto](https://wiki.ubuntu.com/BurningIsoHowto)

They have spanish version links there too. Now to your problem.

This means thae either:
1. The ISO you downloaded was corrupted during download.
2. Your CD burner did not write the ISO properly

What you need to do is to follow the instructions on the links above. Make sure that you use the md5 sums to verify the ISO before burning. Then verify the CD using the CD writing software itself. And if nothing else works plug in a new CD drive (if possible) and try again.
