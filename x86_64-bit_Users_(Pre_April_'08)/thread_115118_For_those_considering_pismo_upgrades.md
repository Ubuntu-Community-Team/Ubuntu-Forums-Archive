### Post by shawn12345 on 2006-01-09
I have done two major upgrades recently. I was using my old pismo under fcwm or fluxbox. I upgraded the old stock 6GB to a 40gb 7200 rpm and wow....i would say over all gained about 15-20% in performance. Noticable difference was obviously file access and general performance I wont say the word *alot* but will say it was a noticable improvement. Still felt little sluggish in gnome, i upgraded the 400 cpu to 500 hesistantly...but have to say it added another 15-20% boost over my stock 400.  

I'm running gnome quite nicely now on 384 ram...very nicely i must say. I dont run any eye candy applets or mail notifiers which suck plenty of memory and cpu cycles, i use simple yet attractive themes. 

cost...probably somewhere around 230's for those 2 upgrades. I need to upgrade the memory to more than 384mb next.

I have a powerbook 17 1ghz...but its big, it gets very hot, and the battery life sucks. I hate carrying power adaptors and my pismo with 2 batteries last freakin a long time. So i use my pismo 90% of the time and the 17 is basically a desktop.

so for anyone contemplating upgrading and wish to get more years out of thier pismo (this only applies to ubuntu...i haven't run OSX on it) i can say this was a very worthwhile upgrade for me. However if you like to run alot of applets, lots of eye candy, heavy video and graphics,etc obviously you might need more horses.

One other thing...webbrowser...i found firefox too slow, galeon seems noticably more efficient not sure why.

### Post by chascon on 2006-01-12
I'm sure you'd get more horse power out of your system if you run a minimalistic window manager, as opposed to a desktop.  Even xfce4 is slow on my old 400mhz imac these days.  Try fluxbox on top of a server install of ubuntu.  If you're feeling adventerous, try a debian server install, just don't install a desktop environment when prompted, and install fluxbox over that post install; Use apt-get install.  Debian installs smaller than ubuntu.  You'll have to do a dpkg-reconfigure xserver-xorg.  I wrote an article on it here:

[http://www3.telus.net/public/lusseau/gnu-linux-ppc_docs/minimal_Ubuntu_Install.html](http://www3.telus.net/public/lusseau/gnu-linux-ppc_docs/minimal_Ubuntu_Install.html)

Chascon

PS: Read the whole thing before attempting an install.
I haven't tried it but finnix at [http://www.finnix.org/](http://www.finnix.org/)
is a very small distro and supposed to be fully compatable with debian.
It can now install to the hd and upgrade to the latest debian release.
You'd have to install X and other related material, as with the ubuntu server install, to get a work station with a gui.
