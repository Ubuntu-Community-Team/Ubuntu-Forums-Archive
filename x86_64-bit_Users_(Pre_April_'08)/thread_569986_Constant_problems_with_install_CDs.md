### Post by picopir8 on 2007-10-08
On the download page, select the checkbox for the Alternate Install CD.  Odds are you are using hardware that was not around when the LiveCD was created so it gets confused.  If you use the alternate install CD, everything should install (though you will be in a somewhat crippled state) and then you will be prompted to download updates which should get you all the latest drivers you need.  From my experience the biggest issues are with video and wifi drivers.
