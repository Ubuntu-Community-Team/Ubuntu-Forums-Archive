### Post by fancyydk on 2006-01-12
It doesn't really matter which version of NDISwrapper you use. They all work well (at least for me). But the latest version has the fewest bugs, so it's probably a good idea to use the latest (which I think is 1.7 as of now). However, what's more important is what driver you have. If you have 64-bit Ubuntu installed, you should use 64-bit version driver. Mine didn't work because I kept on using a 32-bit driver. And it finally worked when I used a 64-bit one.
