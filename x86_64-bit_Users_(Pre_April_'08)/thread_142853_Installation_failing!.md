### Post by grazie on 2006-03-11
I've seen this before and it's most likely due to difficulty reading the CD. Did you check the MD5 hash on the ISO image? Did you burn the CD slowly and did it verify OK? Did you burn the disk on the same machine as you're trying to install Ubuntu? If you can say yes to everything you're more likely to get a good CD burn.
