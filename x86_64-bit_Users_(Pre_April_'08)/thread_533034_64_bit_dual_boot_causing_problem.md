Here are a few steps you can take to fix this problem.

1. Make sure you have a 64 bit processor. Dual Core basically.
2. Partition the drive when you are installing Windows, I left a 82 gig partition when i first installed windows. 80 for ubuntu and 2 for the swap.
3. If you don't want to reinstall it -- use a bootable GPARTED Disc to partition your drive. -- 

other than that you should be fine.
