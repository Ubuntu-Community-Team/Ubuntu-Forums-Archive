### Post by Jad on 2005-01-29
to install .bin file
Applications -> system tools -> Terminal 
browse to he directory where that .bin file is 
using "cd" without the quotes
type 
> pwd  to know where you are not
then cd to that directory where .bin is 
then 
> ./file.bin
