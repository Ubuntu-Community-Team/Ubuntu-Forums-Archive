Full support really is going to depend on what was, and not included in the Ubuntu server kernel. 

The kernel included with Ubuntu server 64 should support the cpu's in question,however. the issue in most case's is not the cpu itself it will be the systems hardware as a whole.

Also looking over the specs of the machine (SAS/RAID/Gigabit Ethernet) "might" be an issue. The only way for you to know for sure is to install the OS.
