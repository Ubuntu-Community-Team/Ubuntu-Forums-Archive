As you can see from this extracted section, you only have 1024x768, 800x600 and 640x480 listed.  You can either use ```
sudo dpkg-reconfigure -phigh xserver-xorg
```
to try to rectify this, else you can just add the new resolution in by hand.

Example:

```
	SubSection "Display"
		Depth		24
		Modes		"1280x1024" "1024x768" "800x600" "640x480"
	EndSubSection
```

Your system is set to use 24 bit colour as default, so if you like you can delete out all the other subsections, just leaving the one in the example.

Please make sure to make a backup copy of your xorg first though.
