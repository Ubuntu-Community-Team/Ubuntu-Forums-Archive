### Post by Kilz on 2007-10-16
> **Timothy Benton said:**
> cool thanks, but can you back that up at all?

64bit windows is a driver nightmare, it has few 64bit applications this is because the source code is secret limiting what can be done. Due to the open source nature of Ubuntu most everything in 32bit is already ported to 64bit. The only things that are not are a few proprietary bits that are not required. Those all have work arounds to use their 32bit versions. In fact looking at launchpad, the number of packages avilable to i386 and amd64 is identical.

### Post by picopir8 on 2007-10-16
I got a free update from XP Pro to XP 64 and eventually had to dump the 64 bit version because lack of hardware support.  I could not get drivers for the touch pad or network adapter.

Conversely, I have put both the 32 bit and 64 bit versions of Ubuntu on 6 machines recently (2 desktops, 4 laptops) and have not encountered a single piece of hardware that lacks a driver.  Granted, in a couple cases the correct driver was not included on the install CD, but as soon as I booted up, I was able to download the correct driver.
