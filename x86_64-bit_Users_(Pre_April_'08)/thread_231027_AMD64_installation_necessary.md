### Post by bingus on 2006-08-07
I originally was running the 64 bit version of Kubuntu, and had the usual issues that crop up with 64 bit software. In hindsight it really wasn't worth the trouble. 

After running the x86 version of Ubuntu on the same machine, I'm hard pressed to see the difference :)

I was able to get Flash etc working with a bit of mucking around, but unless you're doing some hardcore CPU intensive stuff, I don't think its really worth it.
