### Post by ThaRabbit on 2006-10-07
The normal Ubuntu CD provides a graphical installation, much like windows XP and such. The alternate provides the text based installation method. 

The graphical installation cd is failing on quite a lot of systems due to driver issues. The alternate cd gives you the same thing (fullblown ubuntu desktop), just via another way :)

I have a Turion laptop, it worked fine for me :)
