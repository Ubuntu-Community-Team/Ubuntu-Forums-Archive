### Post by SectionThree on 2006-01-20
Well, I'm no expert, but this sort of thing happened to me once (not the LiveCD part but the part where it wouldn't boot).  Just run the installer again (use the install disk, not LiveCD;)), and when you get to the part where you have to set up partitions, do it manually, and be sure to reformat the root partition while you're in that stage, then continue with the reinstallation...

Also, if I may go off on a bit of a tangent, I suggest you invest in an external HD for backup purposes (I got mine on eBay for about $60 -- 60 GB).  With something like that, you'll feel WAY braver about wiping everything out when  things aren't working quite right...

OKAY, here's another bit of advice. If you have a broadband connection, I suggest the net-install ISO. The initial download is considerably smaller, and you get the latest of everything.

Hope this helps (bear with me, I'm running a 1999 iMac G3)...
