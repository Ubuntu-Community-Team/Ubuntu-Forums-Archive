### Post by H.E. Pennypacker on 2006-06-07
I've figured out that, in order to return from suspend mode, pressing CTRL-ALT-BACKSPACE ends suspension.  Then, the screen will turn black and show a prompt (as you would seen in the Terminal).  Let that sit for a few moments, and it will return you to the log-in screen.  

All should be well.
