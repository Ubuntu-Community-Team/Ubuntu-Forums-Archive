### Post by mytuba on 2008-01-21
This should fix your flash problem:

[http://ubuntuforums.org/showthread.php?t=476924](http://ubuntuforums.org/showthread.php?t=476924)

Personally, Sun Java seems to work best for me. Most programs seem to install the Iced Tea version by default. That doesn't seem to work on my setup (also hp). Try installing this

```
sudo apt-get install sun-java6-bin sun-java6-jre
```
