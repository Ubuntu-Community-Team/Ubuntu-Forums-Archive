### Post by 3rdalbum on 2014-03-08
Themes only affect the look of the window borders, title bars and the look of buttons, scrollbars, checkboxes etc. Anything else you see in a theme like docks, desktop backgrounds, information displayed on the desktop, etc are other things you would need to install. The dock along the bottom of the screen is definitely an extra thing the person has installed.

To fully apply a theme you do need to log out and log back in again. Until you log out and log in some of the colors will be wrong.
