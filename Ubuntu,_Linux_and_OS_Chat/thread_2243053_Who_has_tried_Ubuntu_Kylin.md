### Post by monkeybrain20122 on 2014-09-07
I have checked out Kylin since I was setting up some kind of *buntu for a friend who uses Chinese quite a bit, but decided to go Ubuntu instead using fctix for Chinese input  and removed IBUS (the same software for Chinese input in Kylin) Unless you want to have a completely Chinese system*  I would go for Ubuntu and install whatever Chinese software if needed (whatever you can install in Kylin you can install in Ubuntu)

You can always try a live usb first.

* the Chinese integration is not complete yet so there is still a lot of English in menus and so on, so it is kind of hybrid.

P.S. I don't think there is any QQ app preinstalled, as far as I know there is no stand alone QQ client that works natively on Linux. there is one based on WINE, it installs and starts without problem but not sure if it actually works (I don't use QQ, some guy on this forum said that it didn't work) But QQ has web login which should work on all OSes.
