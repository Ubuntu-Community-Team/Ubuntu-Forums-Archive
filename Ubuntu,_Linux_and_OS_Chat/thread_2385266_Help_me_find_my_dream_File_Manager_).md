### Post by ubname on 2018-02-23
> **Hasimir said:**
> I'm running Ubuntu 17.10 and I've been using Pantheon Files. And I love it.
But because it's a hacked install of a program meant for ElementaryOS, which is based on Ubuntu 16.04, there are enough problems here and there that I now want to find a better File Manager.
What I look for in a FM is:
- MacOs-like columns browsing ... no, Nautilus "expand folders" option is not the same ;)
- an address bar that can show the full path of a file or folder, and allows for user input ... the opposite of how Nautilus works
- good keyboard shortcuts, allowing me to easily jump from the left-hand bookmarks to the actual columns (usually with TAB)
- a setting to switch file selection/execution between one-click and two-clicks

Thanks in advance to all that will offer suggestions :)

The standard file file manager has most of your dream features:

CTRL + l -> an address bar that can show the full path of a file or folder, and allows for user input ... the opposite of how Nautilus works

TAB and arrows -> good keyboard shortcuts, allowing me to easily jump from the left-hand bookmarks to the actual columns (usually with TAB) 

file -> preference -> a setting to switch file selection/execution between one-click and two-clicks

HTH
