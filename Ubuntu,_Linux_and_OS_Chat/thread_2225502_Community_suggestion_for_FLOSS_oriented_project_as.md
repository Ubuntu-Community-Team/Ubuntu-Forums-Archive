### Post by ssam on 2014-05-22
Beware that some projects can be slightly hostile people submitting new features out of the blue. Sometimes they like to have a discussion first about whether the feature is appropriate for their project, how it should be implemented and who will maintain and fix issue with the feature in the future. It will be worth check if they have a road map or todo list, and how compatible that is with your project.

Quite a few FOSS projects have a web pages with info for new developers. Most have a mailing list or IRC channel where you can ask the developers questions.

You could look for GSOC project ideas that did not get filled. So given that you are interested in AI maybe OpenCV is interesting to you:
[https://www.google-melange.com/gsoc/org2/google/gsoc2014/opencv](https://www.google-melange.com/gsoc/org2/google/gsoc2014/opencv)
[http://code.opencv.org/projects/opencv/wiki/GSoC_2014](http://code.opencv.org/projects/opencv/wiki/GSoC_2014)
So they had quite a few projects accepted, but there are still a few more.

Have a look at [https://www.google-melange.com/gsoc/projects/list/google/gsoc2014](https://www.google-melange.com/gsoc/projects/list/google/gsoc2014) for more projects.
