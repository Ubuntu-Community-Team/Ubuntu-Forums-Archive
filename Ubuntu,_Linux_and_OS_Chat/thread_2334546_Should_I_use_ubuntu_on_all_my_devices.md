### Post by sudodus on 2016-08-20
It depends on the hardware. One computer might run better with one linux distro, version and flavour, another computer might work better with another one. A simple example is that old computers need a light-weight desktop environment, for example Lubuntu, Xubuntu or Ubuntu MATE (instead of standard Ubuntu or Kubuntu).

It also depends on you, how many linux distros you want to manage - many things are similar, but there are some important differences.

### Post by Olav on 2016-08-20
Personally I would just install the same operating system on both machines. Less hassle and makes it easier to copy configurations from one to the other. In my house we are running Xubuntu on everything - except our server which is running Debian without any GUI.

If you like to tinker with different distros you could always try them in VirtualBox or something.

Oh, just to add a bit of opinion here in reply to Sudodus: we hear that often, that lightweight desktop environments are best for older computers. It is a common idea but I respectfully disagree. I think lightweight desktops are even better on fast, modern computers! Why buy good hardware if you slow it down with an overwrought GUI.
