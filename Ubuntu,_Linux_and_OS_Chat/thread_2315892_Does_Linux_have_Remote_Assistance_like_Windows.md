### Post by deadflowr on 2016-03-03
Ubuntu comes with a vnc server package called vino.
It should be off by default.
But if you want to remove it, simply uninstall that package.

But that is not really a remote assistance package. It simply allows you or someone else to connect and control the desktop of the machine remotely.

The closest thing I can think of that Ubuntu comes with regarding remote assistance is landscape.
As far I can tell the landscape package installed is only a client.
It, too, can be easily removed, simply search the software center for anything related to landscape that is marked as installed and uninstall it.

### Post by grahammechanical on 2016-03-03
There are 4 landscape packages in the repositories & only one is installed by default. It is landscape-client-ui-install. Which is described as an automatic installer for landscape-client-ui.

So, the landscape client is not default installed. Just the installer for it. And it can be removed. Landscape is a paid for service.

[http://www.ubuntu.com/management/working-with-landscape](http://www.ubuntu.com/management/working-with-landscape)
