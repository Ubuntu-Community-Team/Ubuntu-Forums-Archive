### Post by 3rdalbum on 2014-08-12
I've done a little bit of amateur Python on Mac OS 9 and Ubuntu. There seem to be more libraries for Unix-like operating systems. I especially like the command-line; instead of needing to find/write a Python binding to libav (for instance) I can simply call FFMPEG on the command-line from within my Python script.

As an amateur it's very useful to me, but I can also see it being useful for prototyping or rapid development as it takes less time to do that than to learn how to use the actual library.

### Post by coldraven on 2014-08-12
I just found this link today. It's primarily about teaching kids to code but there is a couple of things that may be relevant to your question.
[http://blog.codestarter.org/post/93985346780/how-we-turn-199-chromebooks-into-ubuntu-based-code](http://blog.codestarter.org/post/93985346780/how-we-turn-199-chromebooks-into-ubuntu-based-code)

> **It must be as easy as possible to try out a large variety of programming languages.**
 A lot of time is wasted at learn-to-code programs just getting a  development environment set up. An hour can easily go by as kids try to  get Ruby or Python installed. We decided to pre-install Python, Ruby,  Node.js (plus npm), and Java. Both Python 2.7 and 3.0 are available, and  while we only ship with the latest 2.x Ruby, new versions can easily be  installed with `ruby-install` and selected with `chruby`. If another  language is needed, the Ubuntu package manager makes hundreds of others  available with a few simple commands.



> **It must have a great, local code editor.**
 Coders spend a lot of time in their code editor, and as kids get into  web technologies and text-based programming languages, they’ll need  something that’s easy to use. We install Sublime Text Editor, which has  an awesome Linux version and provides a simple but powerful editor.
For completeness, we also install Vim and Emacs. If we get enough  donations from proponents of one side or the other, we’ll remove the  losing editor from our laptops. Just kidding! Maybe.
