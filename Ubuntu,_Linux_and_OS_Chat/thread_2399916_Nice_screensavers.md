### Post by Dennis N on 2018-08-31
From the Ubuntu repository, the package **rss-glx** has some nice screensavers - see Helios, Euphoria, for example. There are settings to play with on these. As for fish, there is gl-school (from **xscreensavers-gl** collection), which simulates schools of stylized, monochrome fish rapidly swimming about, I think applying some mathematical algorithm to make the behavior realistic.
