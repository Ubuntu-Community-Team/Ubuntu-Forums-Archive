### Post by pauljw on 2016-05-31
So the OP is saying, I think, that he currently has 14.04LTS that he claims "thankfully" is stable.  What is he, or anyone else on 14.04LTS doing upgrading to 16.04 at this point when it's recommended, for good reason, to wait for the .1 version release in July?

14.04LTS is supported until 2019, so unless there is a pressing reason to upgrade your os, it's best to stay with what works.  That's the whole point of having a long-term support version.

I could be wrong, it has happened, but I was under the impression that there have been some major system changes implemented in the new release.  So even though it's going to be an LTS version, there are going to be some hurdles to get over before it's truly stable.  We all need to be patient and when we encounter problems, file bug reports to alert the devs so these issues can be corrected.  Whining and complaining aren't going to help anyone.  Just my opinion of course...
