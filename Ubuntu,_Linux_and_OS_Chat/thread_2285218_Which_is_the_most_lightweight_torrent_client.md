### Post by Bucky Ball on 2015-07-04
*Thread moved to **Ubuntu, Linux and OS Chat**.*

Best to do some research and try a few. You will get a lot of opinions as there is no best and a million ideas of which is the lightest. I use Transmission myself. 

Search for 'lightweight torrent client ubuntu' and see what comes up. Go to Synaptic Package Manager and search for 'torrent' and try a few. If you don't like, mark to 'completely remove' and try another.

PS: If you want to go lighter than Lubuntu-core, as it seems you are going for the minimalist approach, why not try the [mini.iso]("https://help.ubuntu.com/community/Installation/MinimalCD")? That will only install the Ubuntu base kernel, you install the rest, for instance, install lxde and synaptics and take it from there. You'll learn plenty and end up with the lightest hybrid install, if you're careful and research what you're adding.

Good luck.
