### Post by grahammechanical on 2016-08-28
I have never had a problem installing Ubuntu since I first installed it in 2007 and I have installed every version of Ubuntu since then. But then again I have more sense than to buy a machine with Windows 10 pre-installed and try to install Linux on it. I have read too many posts on this forum to assume that it would be easy.

The only way you are going to avoid jumping through hoops and jumping over hurdles is to buy a machine with Ubuntu pre-installed. But then again, you would learn very little about Linux.

Microsoft and the manufacturers have no financial incentive to make it easy to install Linux and several good reasons from their point of view to make it hard to do so. This is the situation for anyone wanting to install Linux on Windows & Apple machines. It is not limited to would be Ubuntu users. And there is little that the developers of Linux distributions can do about it except set up the hoops and hurdles.
