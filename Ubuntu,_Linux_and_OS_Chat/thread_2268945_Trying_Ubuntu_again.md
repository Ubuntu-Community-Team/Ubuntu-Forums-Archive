### Post by Mark Phelps on 2015-03-12
Zorin is popular with folks that want a Linux distro that looks and behaves a lot like Windows.  And, that is a good place to start out without having to learn a lot just to use a distro.

Another popular choice is Linux Mint. It has kept the legacy menu system, installs add-ons by default, and has even managed to keep the Windows controls in the right upper-corner of the Window -- something that Unity changed a while back.

There is also Ubuntu-Mate and Ubuntu-Gnome, for folks who want to use Ubuntu but want an entirely different desktop environment from Unity.

### Post by Bucky Ball on 2015-03-13
*Thread moved to **Ubuntu, Linux and OS Chat**.*

Not a support request. Welcome back. ;)

I'm a big xfce4 fan. You can install xfce4 from the Software Centre, logout and choose it from the Sessions menu at the login screen, then login. Or you can install Xubuntu which uses it as its default desktop environment, or you can do what I do, which is install the Ubuntu [minimal install]("https://help.ubuntu.com/community/Installation/MinimalCD"), then xfce4 and the apps you need/use/want and have a sleek machine with no bloat. 

Good luck. ;)

PS: Be aware that official Ubuntu flavours are supported here, which Zorin is not. Check the activity on their forums to see how much support you're likely to get. Distro hopping is fine, but if you're 'out there on your own' with an 'exotic' spin, it might be a struggle if things break. It's a learning curve.

### Post by mikodo on 2015-03-15
Hi,

I think also, that sticking with a well supported and familiar distro to many is important also. It allows as you mentioned for many tutorials and documentation, to help one along, and usually come with good security support. Ubuntu is of course excellent for that. Please don't forget the [official flavors of Ubuntu]("http://www.ubuntu.com/about/about-ubuntu/flavours"). They share the same security support as Ubuntu, though some have less time support. What's important also, is that they all share with Ubuntu, much of the same underpinnings as Ubuntu, so most of the on-line guides for Ubuntu work well with the flavors. Keep in mind though, that each desktop is different. All the flavors have knowledgeable support in these forums. Some, more than others.

I like [Xubuntu]("http://xubuntu.org/"). There is a large knowledge base for it's desktop [(Xfce desktop environment)]("http://www.xfce.org/") here and across all the landscape of linux. It is a mature desktop that doesn't change rapidly. That is a plus in my mind.
