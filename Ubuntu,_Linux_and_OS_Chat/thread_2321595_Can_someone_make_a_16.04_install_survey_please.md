### Post by grahammechanical on 2016-04-23
That is not a survey. It is a census.

As far as I know all Flavours of Ubuntu use the same installer. It is called Ubiquity. All Flavours of the same version (14.04, 15.10, 16.04) will have the same Linux kernels and the same drivers. All flavours have the same boot loader. So, as far as I can tell, general issues relating to installer, kernel, drivers & boot loader would apply to all flavours.

Regards.
