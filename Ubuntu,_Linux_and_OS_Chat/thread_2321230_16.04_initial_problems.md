### Post by Iliyancho on 2016-04-22
> **ilko2 said:**
> I managed to install VirtualBox AND Unity Tweak Tool trough the Software center (did not hang this time). No luck with Google Chrome so far, but I am sure the issue will be fixed soon.

For Google Chrome and any other app you're trying to install and it is hanging in Software Center, just download the .deb file, open a Terminal and type:
```
sudo dpkg -i packagename.deb
```
If it gives you error run:
```
sudo apt-get install -f 
```
It will install all missing packages and Google Chrome.

Hope I helped.
