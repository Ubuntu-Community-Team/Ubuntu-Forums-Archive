### Post by night_sky2 on 2016-02-09
Short answer: No.

The Surface Pro is a device designed by and manifactured for Microsoft. The hardware is specifically choosen so as to make the most out of Windows OS. That doesn't mean that* you can't* install Linux on it but it isn't part of their marketing strategy. Think of this device as the Microsoft equivalent of the iPad Pro by Apple.
