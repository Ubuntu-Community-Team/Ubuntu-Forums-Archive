### Post by guiverc on 2020-10-24
As I remember it, Mark Shuttleworth used to come up with the names (eg. [https://www.markshuttleworth.com/archives/1521](https://www.markshuttleworth.com/archives/1521) & 18.10/*cosmic cuttlefish*), announcing them in a well written & usually education/amusing blog post.

The 20.10 (groovy gorilla) name wasn't from Mark, it was suggested by Martin Wimpress to Mark (*as far as I recall*) and Mark said yep. Martin spoke about that on a podcast (*likely Ubuntu Podcast (UK)*).
