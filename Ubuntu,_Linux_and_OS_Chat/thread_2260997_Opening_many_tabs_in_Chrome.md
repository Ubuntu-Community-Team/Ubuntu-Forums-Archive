### Post by monkeybrain20122 on 2015-01-15
I typically keep many tabs in my browser. Chrome is kind of bad for this because when started, it tries to load all tabs and this often result in temporary lock up. This is an important advantage of Firefox for my use case. But I have found [http://sourceforge.net/projects/footab/](http://sourceforge.net/projects/footab/)  With it Chrome opens without loading the tabs until focus and Chrome uses a lot less ram as a result.

To install just download it and drag and drop it to Chrome's Settings > Extensions.

Just want to share this. Hopefully will be useful for Chrome users. :)
