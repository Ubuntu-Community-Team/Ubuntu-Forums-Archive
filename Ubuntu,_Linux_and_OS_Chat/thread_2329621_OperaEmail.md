### Post by CantankRus on 2016-07-03
Opera is now based on chrome and no longer includes a mail client.
I have the latest opera but I still use the old 12.xx version just for mail.

The old version is still in the opera repo but you can't have **opera** (old version) and **opera-stable** (new version) installed at the same time.
You have to install opera-developer or opera-beta.
I have been using opera-developer for a while now with no problems.
