### Post by Geoffrey_Arndt on 2016-05-16
Would not recommend Kubuntu - - probably run slow with the antiquated turion dual cpu.   But Xubuntu has a reasonable chance to run well.    

If you're OK with getting most of your support from these forums, AskUbuntu, youtube linux channels - - then go for it.    The install should be easy as you select the option to wipe the hard drive and install Xubuntu on the whole system.   That's usually the fastest and most reliable type of install (e.g., replacing XP versus running alongside it).   While the install should be pretty straightforward, it never hurts to watch a few videos of how to install, etc.  (and how to use the installed system).

Get it from:   [http://xubuntu.org/](http://xubuntu.org/)
