### Post by fantab on 2015-01-31
Dual-boot Windows and Ubuntu until such time you don't need the former anymore.
Generally when people switch to ubuntu only or any other linux distro, they get frustrated when their favourite games and software don't work on linux. And dump the move.
Having Windows as backup won't frustrate you and you can gradually adapt to the Linux world.
[Linux is not Windows]("http://linux.oneandoneis2.org/LNW.htm")

### Post by grahammechanical on 2015-01-31
Install Windows and then install Ubuntu. When we install Windows after installing Ubuntu we get a Windows boot loader that cannot recognise any OS except a Microsoft OS. So, we lose the ability to load Ubuntu. But it is usully possible to load either Ubuntu or Windows if we install Ubuntu second and this is because the Linux boot loader (Grub) can detect MS operating systems.

You might want to look at developer.ubuntu.com. We now have a Ubuntu Software Development Kit (SDK) based on Qtcreator and for those who want to develop for Android an easy way to set up an Android development environment in Ubuntu. So, you might want to check out ubuntu-make. A lot of effort is being made to turn Ubuntu into the platform of choice for programmers. 

Regards.
