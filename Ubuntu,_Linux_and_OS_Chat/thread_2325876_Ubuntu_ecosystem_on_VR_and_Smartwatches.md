### Post by grahammechanical on 2016-05-26
As far as I know, Canonical has not intention of entering these markets.

Snappy Ubuntu core is available. It is minimal size Ubuntu code. It has transactional updates both of the OS and applications. It is secure. There is nothing to stop some OEM from using Snappy Ubuntu core in any kind of device. And if Canonical was in discussions with an OEM we would not hear about it until the manufacturer was ready to release the news.

These things will follow the same pattern of the Ubuntu phone. People kept asking about it. But no information was released until the OEMs were ready to make their own announcements.
