### Post by mooreted on 2017-04-13
There are some Gnome extensions I use that make Gnome much easier to use:

Applications Menu - Gives you a nice category/favorites menu in the upper left of the top bar
Dash to Dock - Give you a configurable dock on the left. You can have it auto-hide when a window border touches it or you can make it permanent like Unity
Topicons Plus - Puts the system tray/notification icons in the upper-right of the top bar so you can always see them

Some people use a lot more than that, but you can tweak Gnome to be more usable.
