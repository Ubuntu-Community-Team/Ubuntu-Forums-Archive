### Post by mikodo on 2016-04-29
> **vasa1 said:**
> Isn't the update notifier common across official Ubuntu flavors (other than Kubuntu)?

Yes, for Xfce but, the indicator in the panel has to be written for each distro. See [Nick's]("https://bugzilla.xfce.org/show_bug.cgi?id=10732") response here, (the main Xfce dev). It was available in Xubuntu until 13:10 I believe. 

My Xubuntu 14.04 displayed a bubble on the desktop for important security updates. Explained in [post #9 and #24]("https://bugs.launchpad.net/ubuntu/+source/update-notifier/+bug/1246364") of this bug. I don't use it. 

Addendum. I meant to mention, if one wants to check with a gui for other updates other than the security ones, one can click on the Whisker Menu and search for Software Updater and run that to find out if any other updates are available. Also, one could install a launcher for the panel, and set it to launch Software Updater and manually click on it to run. Another option is, one could set a key combo for Software Updater in Settings Manager > Keyboard > Application Shortcuts.

Or, go all out like Toz [here]("https://forum.xfce.org/viewtopic.php?id=10118") and have a bubble that displays for updates.

I hope Mint aren't using [update-manager]("http://debian.2.n7.nabble.com/Advice-needed-update-manager-in-wheezy-considered-dangerous-td2888924.html") to display all updates.
