### Post by grahammechanical on 2014-02-27
Work your way through this

[https://help.ubuntu.com/community/Applications](https://help.ubuntu.com/community/Applications)

I doubt though if it includes 14.04 as it does not officially exist.

I have been running Trusty Tahr (14.04 under development) and there are no big changes to the default installed applications. What particular applications do you think will be dropped or replaced? Firefox is still there. So, too Libreoffice.

The utilities are much the same. The way of changing settings is much simplified. There are On/Off sliders for many of the settings. The Launcher size can be altered from the Appearance utility. It does away with the need to install an additional settings editor for that task. There are not so many visual differences between 13.10 and 14.04 but big improvements in Unity over 12.04.

I suggest that you download a Trusty Tahr ISO image and run a live session. 

[http://cdimage.ubuntu.com/daily-live/20140223/](http://cdimage.ubuntu.com/daily-live/20140223/)

Do not forget that many of the applications that we install through the Software Centre are described as may be receiving updates from the Ubuntu Community.

Regards.
