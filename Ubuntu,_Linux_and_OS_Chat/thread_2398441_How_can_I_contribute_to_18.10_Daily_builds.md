### Post by PaulW2U on 2018-08-12
Welcome to the forums eden123,

You can contribute to the next release of Ubuntu by:

[LIST]
[*] Reporting bugs on Launchpad - [http://bugs.launchpad.net/](http://bugs.launchpad.net/)
[*] Testing daily ISOs - [http://iso.qa.ubuntu.com/qatracker](http://iso.qa.ubuntu.com/qatracker)
[*] Feeding back any problems in the development forum - [https://ubuntuforums.org/forumdisplay.php?f=427](https://ubuntuforums.org/forumdisplay.php?f=427)
[/LIST]
Be prepared to downgrade an update that doesn't work properly. It doesn't happen very often but it **does** happen. :)
