### Post by night_sky2 on 2015-05-23
> **monkeybrain20122 said:**
> Is there any reason to use Opera instead of Chrome? It is closed source and built off Chromium just like Chrome, so they are similar but Opera lacks things such as pepper-flash and Chrome's pdf reader.

Opera's UI is quite different with the squared tabs and a feature like speed dial can be interesting. But it's a chromium browser, so it is expected to behave like Google Chrome. I would say that some people might prefer to trust Opera - a Norwegian compagny - with their browsing data instead of Google. They have a small but loyal base of users which has been consistant over the years.

- Chrome's pdf reader is no longer closed-source, so it's more Chromium's Pdf reader. By default it's also featured in Opera.

- Pepperflash + media codecs can easily be made to work with Opera on Ubuntu, just like Chromium. On a side note, the devs announced last summer that they are working with Adobe to add a built-in pepperflash plugin in Opera. So expect that to reach the Linux build at some point in time. See: [http://blogs.opera.com/desktop/2014/06/opera-developer-23-pepper-flash-coming-opera/](http://blogs.opera.com/desktop/2014/06/opera-developer-23-pepper-flash-coming-opera/)
