Hello Parisk89, I recently updated the BIOS of my Dell XPS 14 (L421X) using one of Gandalf's Windows images, mentioned here: [http://en.community.dell.com/support-forums/software-os/f/3525/t/19289214](http://en.community.dell.com/support-forums/software-os/f/3525/t/19289214).
