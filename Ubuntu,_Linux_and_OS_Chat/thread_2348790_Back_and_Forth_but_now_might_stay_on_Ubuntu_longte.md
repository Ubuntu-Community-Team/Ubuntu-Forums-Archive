### Post by Eggnog on 2017-01-07
I was somewhat the same in that I used Ubuntu early on several years ago, then moved to Mint, and now back to Ubuntu, probably for good.  What did it for me was Ubuntu MATE 16.04.  I haven't found one thing yet I don't like about it or even one thing that has given me any trouble.  I've got Plank set up on the left side with the Numix theme so it doesn't ever overlap the top panel (it just resizes with the number of things on it), and I've been happily off to the races.  I find it works very well on the desktop and on my laptops.

Not that it's necessary for many people, but I like the Welcome screen that pops up on new installations, and the Software Boutique.  That has to be very helpful for many new users.  Ubuntu MATE is pretty nice.  I understand 16.10 has all sorts of new bells and whistles but I'll just hang out until the next LTS.
