### Post by buzzingrobot on 2015-07-16
I've been running a machine with Win10 pre-release for some weeks now. Updated yesterday to Build 10240, which may be the RTM, or at least the basis for the RTM. This is my first long-term exposure to Windows in quite some time.  I have to say that the pre-release has been pretty good, actually. This build and the last few builds have been solid and stable (more so than any of my experiences with KDE5 distros).  Desktop responsiveness is good. The new browser, Edge, currently lacks things like extensions (will get Chrome extensions later, I hear) and can be slow to load ad-heavy pages, but it's OK for casual use at this point. I expect Edge will see frequent updates throughout the year.

Frankly, a Windows user who wants to move from any previous version, is wondering about Linux, and wants to stay with the panel-and-menu approach, ought to look at Win10.
