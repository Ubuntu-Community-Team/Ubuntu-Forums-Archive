Caveat:  I'm a bit demanding about font rendering. And this is very much in the weeds.

I've noticed a subtle difference in the appearance of fonts between 14.04 and 14.10.  The word I'd use for their appearance in 14.10 is "shinier".  I find that less than pleasing.

Using the same monitor (Dell 2410) and the same video card (either Radeon or Intel Haswell) and the same drivers, Unity 14.10's fonts look shiny to me, while the same fonts in Fedora 21 Gnome 3.14,  using fontconfig and freetype patched with the Ubuntu font rendering fixes, are not shiny. (Kernel's differ:  Fedora is using a 3.17 kernel.)

I notice the same thing on an Ubuntu Mate 14.10 installation *after* installing Compiz, but not before. 

I don't notice it when using compositors other than Compiz.  I.e., in Xubuntu, in Kubuntu, or on Fedora with Gnome 3.14 or Fedora 21 with KDE.

Compiz went from 9.11 in 14.04 to 9.12 in 14.10. Does that account for it?
