### Post by Bucky Ball on 2016-05-01
Go a step further and get a bit more from the machine with [Xubuntu-core 16.04]("https://unit193.net/xubuntu/core/") LTS if you're liking Xubuntu and its running well on the old beasts. I use it on all my machines, even my i7, 16Gb desktop with a 4Gb GPU! Racehorse. 

You get the Xubuntu desktop and a few bits and pieces, but no 'apps' as such. You install them. So, if you just use your Xubuntu for web surfing and email, install the web browser and email client of your choice, whatever else you feel like, and you're done. Apart from a few standard Xubuntu accessories, that's all you'll have. You can install further apps via a terminal or install Synaptic Package Manager and take it from there. Fun and lightweight.

### Post by David_Wright on 2016-05-03
> **The_Woodpecker said:**
> I would use Puppy Linux if its really old [COLOR=#000000]if its not so old Xubuntu[/COLOR]
That matches my linux adoption path almost exactly.

Tried **Puppy Linux** on an old Pentium 4 desktop, which had become virtually unusable with Windows XP and it was a dramatic improvement.  Then moved to a full install of **Lubuntu** on the Pentium 4, and although it was miles better than XP it still struggled a little with modern browser / video etc, so "upgraded" to a Core2Duo and put **Lubuntu** on that.  When Chrome recently stopped supporting 32-bit systems I decided to switch to **Xubuntu **and am very happy with it!  Also run **Xubuntu **on my i5/8Gb laptop.

I would describe (perhaps uncharitably) **Puppy Linux** as a "rescue" OS for ancient software, or for complete portability on a USB stick.  I liked **Lubuntu**, but **Xubuntu **seems to have more polish for very little additional overhead.
