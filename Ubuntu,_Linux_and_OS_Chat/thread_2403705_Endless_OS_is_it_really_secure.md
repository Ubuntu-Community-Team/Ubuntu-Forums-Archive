I bought Acer Aspire 3 A315-21-43WS which came pre-installed with [Endless OS]("https://endlessos.com/"). It is based on Debian, but not really so because it uses [OSTree]("https://ostree.readthedocs.io/en/latest/manual/introduction/") to build the operating system. For applications it uses [flatpak]("https://flatpak.org/").

There is one more problem with this Laptop. I can't install any other Linux Distribution in dual boot, however, I think I can reformat the disk and install Ubuntu forgetting the Endless OS.

But I may be falling in love with this modern Linux Distribution called Endless OS.

The point I would like to discuss in this thread is "Endless OS is it really secure?" since it is on read only file system.

Another point: is it pointing towards future of Linux Distributions?

More points are welcome.
