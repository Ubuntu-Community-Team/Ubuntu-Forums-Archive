### Post by TheFu on 2019-02-03
> **ferfykins said:**
> I'm using google chrome on windows.... Not sure what i wanna use on Ubuntu... Chrome or Firefox....

I use **firefox** from inside my LAN for general use. Use adblocking, noscript, ghostery and a 300K /etc/hosts file. Never for anything with money involved. It runs in a standard **firejail** which limits access to much of the OS and allows write to selected locations under $HOME.

I use **chromium** inside a **private firejail** on a tiny VM for online banking and with travel sites, restarted fresh each time I need to switch sites. The private option ensures nothing it saved, ever.  The browser cannot write to disk.

I use **dillo** in a private firejail when I don't want to be bothered with javacript or ads. Mostly ad-overun news pages.

I use **lynx** when I just want the text - lynx works inside a terminal, so it is good for server setup how-to guides.

I use **wallabag** from inside firefox to snag multi-part articles into a single page that can be shared across devices, while disconnected. Reading long articles on a tablet, sitting in a waiting room is nice, especially when it doesn't get any ads - just the images and text.

I use **pandoc** to grab just the important part of a page for later long-term reference.  Pandoc is a filter/translation tool to convert between different formats.  html--> text --> markdown --> epub --> pdf .... or any mix of those AND many others.  Sometimes I just want the text, not the HTML and almost never PDF.

I will never use chrome. Why give a huge internet advertising company direct access to your total browsing tool, locations, dns, basically everything done inside the browser?  Why?

Opera was a nice browser, before it was bought by a company [https://techcrunch.com/2016/02/25/opera-ceo-sale-to-chinese-consortium-wasnt-our-decision/](https://techcrunch.com/2016/02/25/opera-ceo-sale-to-chinese-consortium-wasnt-our-decision/) in a country with a checkered past/present on privacy. Be careful using the built-in VPN. Please.
