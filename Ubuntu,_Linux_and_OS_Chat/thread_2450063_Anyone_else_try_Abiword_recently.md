I always loved abiword for providing absolute simplicity and after getting frustrated with Libre Office today I tried installing it from the repos - yep it's there in 20.04 :D

It seems to be working OK and easily reads recent .odt docs. I just LOVE simplicity - "just works" has to be among the greatest compliments a project can earn :-D

Quite honestly if I had any money to spare Abiword is a project I'd invest in. If you need to just type a very simple letter it can't be beat.

I need to install [BunsenLabs Lithium OS]("https://www.bunsenlabs.org/") on some older netbooks and laptops soon so hopefully abiword will be available from the Debian repos it's tied to.
