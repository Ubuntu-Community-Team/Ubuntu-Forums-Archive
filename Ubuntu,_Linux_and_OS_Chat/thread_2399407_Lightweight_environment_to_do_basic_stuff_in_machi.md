### Post by ajgreeny on 2018-08-24
You may find that you can best get exactly what you want by installing the [Minimal CD]("https://help.ubuntu.com/community/Installation/MinimalCD") version which has no GUI at all and after installation add the packages you want.

You will need a window manager at the very least, and xorg to get any GUI but everything else is up to you.

### Post by TheFu on 2018-08-25
TinyCore is the smallest distro with a GUI. It is NOT Debian-based so no APT.

If you install using the Alternate Minimal installer, as ajgreeny suggests, you might find that some drivers aren't automatically installed, but you can definitely have control.  I'm not certain if someone new to linux will be able to navigate any difficulties, like a missing network driver.  

Small seems like a great idea, until you have to deal with all those missing things manually.  For most people, Lubuntu is about as light as you can safely get, but still have things work.

You'll never know until you try.
