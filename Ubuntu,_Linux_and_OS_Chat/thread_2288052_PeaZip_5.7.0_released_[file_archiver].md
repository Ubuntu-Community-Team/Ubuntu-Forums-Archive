### Post by giorgiotani on 2015-07-24
PeaZip 5.7.0 for Linux was released Jul 22.
The new release is compiled with new Lazarus IDE [http://www.lazarus-ide.org/](http://www.lazarus-ide.org/) and comes with updated Delphi/FreePascal math/cryptography libraries from [http://www.wolfgang-ehrhardt.de/](http://www.wolfgang-ehrhardt.de/)
5.7 version updates to p7zip 9.38.1 backend, and provides UNRAR5 plugin to extract new .rar files created with WinRar 5 or newer using "RAR5" archive format option (available as separate plugin due the non-OSI compatible unrar licensing restriction).
The official website of the project migrated to [http://www.peazip.org](http://www.peazip.org) and the repository was migrated on GitHub [https://github.com/giorgiotani/PeaZip](https://github.com/giorgiotani/PeaZip) which also hosts the new directory for translations [https://github.com/giorgiotani/PeaZip-Translations](https://github.com/giorgiotani/PeaZip-Translations)
