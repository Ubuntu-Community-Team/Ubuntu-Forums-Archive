### Post by grahammechanical on 2015-06-19
I am not a developer. I do not clearly understand what is being requested.

All my favourite applications are in the Launcher. The Dash also gives us recently used applications and files. We can filter the Dash search results as well as use the 5 default scopes. If we have a mind to we can write our own scopes.

I have been using Unity from the beginning. I am happy with Unity the way it is.

And there is another thing to consider - convergence. At present Ubuntu runs on the X.Org server and uses Unity 7 and the Debian package format. And there will be a version of Ubuntu like that at least until Ubuntu 16.04 reaches end of life. But by the release of Ubuntu 16.04 there will also be a Ubuntu desktop version that is running on the Mir server, using Unity 8 and with Snap packaging. A lot of development work is being put into this version of Ubuntu. I do not expect the developers to invest a lot of time and effort into Unity 7 innovation. Not when their declared course is with Unity 8.

Just in case you are wondering. Unity 7 works with the X.org server but Unity 8 works with the Mir server. And it is Mir + Unity 8 + Snap packing that is the future direction of Ubuntu desktop.

Regards.
