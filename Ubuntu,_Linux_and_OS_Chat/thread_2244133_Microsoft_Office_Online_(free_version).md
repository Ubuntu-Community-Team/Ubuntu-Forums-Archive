Thanks for checking it out. That subtotals are available is great.

I didn't explain about "visible cells" because I don't have Excel at hand but I'll try. Suppose you have several rows of data and then filter by some criterion to get a subset, what you'll see on the screen will be just the rows passing that filter. If you select the first and last row and everything in between of what you see on the screen, you'll be selecting even the rows that were excluded. However, Excel has a neat feature under Edit > "Go to" > "visible cells only" that lets you really select just what you see on the screen. LibreOffice doesn't have this feature. There was an extension but, IIRC, I had some problem using that. AFAIK, Google Spreadsheets lacks subtotals and "visible cells only".
