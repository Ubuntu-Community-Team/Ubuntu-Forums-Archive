### Post by TheFu on 2022-07-19
There are many things to like about both.  Which is better is more about 'taste'.

For example, Wayland isn't mandated in Ubuntu. Xorg is still there and works. In fact, it is the default for some specific Ubuntu flavors in 22.04.
Mint doesn't support snap packages - that can be a plus or a minus, depending on your taste. Of course, Mind users can add snap package support, if they like.

Does Mint have a Gnome3/4 release?  If you are a Gnome lover, maybe Mint isn't a good choice.

It is about taste.

OTOH, if you want to build a resume, Mint won't impress most companies seeking Linux skills.  Honestly, Ubuntu Desktop shouldn't either, but since Mint doesn't have a "server" release, Ubuntu Server, Debian Server or RHEL would be the main 3 server distros to pad a CV.

Just depends on your goal ... and taste.

Is there a way to pay for Mint support to a real company, if, say, I have 500 desktops?  I know I can pay a company to support 500 Ubuntu desktops.
