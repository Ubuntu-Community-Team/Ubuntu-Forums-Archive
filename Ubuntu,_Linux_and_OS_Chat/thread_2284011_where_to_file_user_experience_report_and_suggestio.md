### Post by grahammechanical on 2015-06-26
There is something you need to understand. Any Linux distribution is a collection of software from many software development projects. Ubuntu developers call these projects by the name "upstream." Ubuntu itself is an "upstream" to distributions based on Ubuntu.

Suggestions for improvements to upstream projects are best sent to the developers of Ubuntu upstream projects. There is also another expression that we need to think about: "practical to do but non-trivial to do." What seems to us as an "easy fix" does not always look that way to a programmer.

I do not see any connection between making a user interface suitable for using 2 hands with the 2 recommendations for improvement that you then make. A mouse that can be used with either hand would still only be used by one hand. And what is that got to do with the user interface being suitable for 2 handed use? 

A re-design of the keyboard would improve things but it has been tried before and these attempts failed because hardware manufacturers would not produce these different designed keyboards and they took that decision because computer manufacturers would not buy the newly designed keyboards and they in turn took those decisions because they thought that users would not like the new keyboards.
