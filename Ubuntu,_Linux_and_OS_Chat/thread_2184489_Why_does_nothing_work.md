### Post by sudodus on 2013-10-29
If you want a simple and clean desktop, use the window manager ***Openbox*** without any panels. Just right-click on the grey background and start a terminal emulator and start your applications from there. If you want panels with smaller foot-print, try ***Xubuntu*** as suggested by *hattpa* or ***Lubuntu*** with even smaller foot-print.

In Lubuntu you can select a plain Openbox session at the log in screen. The standard Lubuntu session uses the desktop environment LXDE which is turn uses Openbox plus a panel at the bottom of the screen (similar to Windows Classic).
