### Post by amiga500 on 2018-03-01
The problem of putting windows in vmware is that you miss out on speed efficiency as well as miss out on the native 3D video card that is suited for heavy intensive gaming ability. But if you only want to use windows for applications and not gaming, I guess sure you can go that route. Personally I would have a PC for windows and another PC for Ubuntu and any other Linux distro suite. This is how I would do it.

### Post by QIII on 2018-03-01
Windows in KVM with a dedicated GPU (one for the host, one for the guest) to pass through using Spice is very close in performance to bare metal for gaming.  There are any number of tests and reviews demonstrating how well it works.

Using something like Synergy improves the integration.

But don't use the VM for Steam, as it may be detected by their cheat detection and get you banned.
