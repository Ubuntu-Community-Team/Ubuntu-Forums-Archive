### Post by TheFu on 2018-10-19
If you are planning to enter the data recovery business, visit MyHardDriveDied.com . Scott teaches data recovery classes. He's world class and you'll learn how to deal with any sort of data that can be recovered and the required specialized hardware and software needed for legal, forensics.  

Data recovery professionals NEVER write to the problem storage. They use hardware and cables that physically CANNOT write.

If you are just playing, then a google for "*data recovery ubuntu*" will take you to the Ubuntu Data Recovery How-To. It is pretty detailed.

### Post by SagaciousKJB on 2018-10-22
You could check out CAINE.  It's a live CD that's suited towards forensics.

It might be a good starting point since it has hundreds of tools you may have never heard of.
[https://www.caine-live.net/page11/page11.html](https://www.caine-live.net/page11/page11.html)

A lot of it is imaging software, since as TheFu pointed out, you pretty much never write to the media you're investigating.  In fact most of the time you're supposed to make a single image of it on a totally different drive and search the image file rather than the actual disk itself.
