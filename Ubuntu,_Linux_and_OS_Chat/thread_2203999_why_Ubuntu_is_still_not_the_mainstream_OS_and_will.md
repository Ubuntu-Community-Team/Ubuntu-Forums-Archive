### Post by melinux on 2014-02-06
Hello everyone,

What I am going to write is not about bashing ubuntu of Linux in general, I am a server administrator who manages around 500 centos/debian servers and I LOVE linux in server environments, but about my frustration about having Ubuntu or any other Linux based operatingsystem on my laptop and why I think ubuntu/any other linux based operating system for laptops/desktops will never reach mainstream adoptation:

1: Basic functionality does not work - I do not really care that the laptop is hot and using continuesly the main graka, nvidia is a bitch on providing drivers for linux, but even on a fresh installed system it is not possible to use HDMI. People, we are 2014 and HDMI came out in 2003!! 11 Years after introduction of HDMI, it is not possible to use it on ubuntu 13.10. (optimus configuration)
2: Things which worked before, does not work afterwards. Ex: backlight of keyboard worked in 13.04 out of the box, but needed to be fixed again by using a script which binds a command to the key in 13.10. Wtf is this? Why does it seem to be impossible to stick with something that works?
3: Fixed release dates. Why does canonical has to follow EA in their foodsteps? Releasing something which is not even worthy to be called beta? I tried 13.10 when it just came out last year and it is was unworkable, more errors then stuff that worked. Installing gave me already 5 unrecoverable errors. Do not release something for the mainstream when it is not capable of providing the productivity which is required for the mainstream market. Even MS (not even windows vista) is that crappy when it came out. I reinstalled 13.10 yesterday and now it is much better. 

In general, ubuntu is still not on a level where I would say: Hey, I am going to invest a line of computers with state of the art hardware and start selling them as the ultimate consumer pc. There is just still too much basic functionality which does not work.

Free is not an unlimited value proposition. 

I expect constructive input and no bashing.
