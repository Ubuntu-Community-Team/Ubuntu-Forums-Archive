### Post by puffcorn on 2017-08-01
My choice is Xfce because it's light on resources (my current desktop is rather old), it's very stable, and I like the Whisker menu--useful and looks nice.

Can't comment on GNOME since I haven't tried GNOME 3 yet.

### Post by TheFu on 2017-08-01
Drop the DEs. You don't need them at all.  Go with a lighter, pure, window manager-only setup.

### Post by jpberes on 2018-04-03
I can say that I used almost all or at least the main Desktop Environments available yet (also on different distros of course)...
Every DE has its strong and weak points,
If you want to go for Resource usage (the less amount of Resources used) then you should definitively go for LXDE, XFCE or Mate ...
Where LXDE (Lubuntu) has less customizing options, XFCE can be customized quiet a lot, Mate (especially in Ubuntu 17.10) with Mate Tweak = easy to change layout of panels
Gnome3 for those liking it, but a little bit more heavy on ressources,
Budgie, new DE, currently based Gtk (Gnome) .. quiet modern, nice and not too resource demanding ... currently Budgie 10.4 is the latest version and probably the last one based Gtk, this since the Budgie 11 (Work in Progress) will be Qt5 based (more like KDE apps then)
And you have KDE, very customizable ... So quiet a choice ... on other distros you can also try Pantheon, LxQt, OpenBox ...
Of course when you're eager to experiment other DE's can also be tried on Ubuntu ;-)
