I was trying to find a video player which supports multiple video simultaneously. My needs are "Live Video Support", "Many(over hundreds) simultaneous videos support with regularly updated still screen shot", "Volume Up/Down", "Video Focus Change". What I need is not very complicated. But it was not easy to find such multi video player in the internet. It could be because my googling skill is very poor. :(
Anyway, I couldn't find what I need, so I just make one using clutter-gst-2.0.

[ATTACH=CONFIG]266927[/ATTACH]

I name it "Oh! Multi Video Player". The homepage is [http://ohhara.sarang.net/omvp](http://ohhara.sarang.net/omvp) .

I have also created "Oh! Multicast Video Scanner". The ISP I am using provides over 300 free IP-Multicast IPTV channels but it is not easy to find the multicast ip addresses. So I created "Oh! Multicast Video Scanner" to find IP-Multicast IPTV ip addresses. It scans multicast ip addresses the user specifies. If it finds something in a multicast ip address, it tries to decode it with gstreamer and saves it into a png file.
"Oh! Multicast Video Scanner" homepage is [http://ohhara.sarang.net/omvs](http://ohhara.sarang.net/omvs) .

I didn't spend a lot of time on them. So, there are still some stability issues, but not very terrible. And they are enough for my needs.
I made them for my own needs but I think they could be helpful for others too. If there is anyone who want to use "Oh! Multi Video Player" or "Oh! Multicast Video Scanner". Use them freely. They are available under GPL.

I only tested them on Ubuntu 14.04 64bit and Windows 7 64bit. But I think they will work on other platforms also by recompiling.
