### Post by Lars Noodén on 2015-06-23
The shell is the same whether the machine you are connecting to is attached to your keyboard or located on the other site of the planet or, these days, in orbit around the planet.  The graphical interfaces are more restrictive and often watered down.  The web interfaces like webmin are ok once everything is set up and as long as everything is running ok, but once things go wrong or need special changes, you have to move to the shell anyway.  Graphical interfaces change a lot and the shell is pretty consistent across versions and even, within limits, distros.  

Also the shell is fully scriptable, anything you can type can later be pulled into a script and automated.
