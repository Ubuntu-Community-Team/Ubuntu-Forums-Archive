### Post by CaptSaltyJack on 2015-05-15
This is a little bit ranty, though my hope is that people (especially people working on Ubuntu) will give it some consideration.

I'm sitting here on a fresh install of 14.04 LTS, on a Lenovo laptop, struggling and cursing over a few basic things:

* slow wifi (I should point out that it works perfectly fine when Windows 8 is running on this laptop)
* nearly unusable trackpad (also works perfectly in Windows)
* no drag-lock option for trackpad, *still*, after other operating systems have had this option for years

It's 2015. These things shouldn't be issues anymore. Ubuntu *still* can't get a handle on Synaptic touchpad, I try to move the cursor and half the time it doesn't respond until I make vigorous movements. And how is it there's no drag lock option in the touchpad settings? Meanwhile, Unity keeps getting plenty of attention while very basic things like the aforementioned go ignored. Simple, but important factors that create a more friendly and usable experience.

I can't take Ubuntu Desktop seriously anymore and am abandoning it as a platform. I still like Ubuntu Server and will continue using it, but Desktop IMO is ignoring too many important issues while focusing on shiny objects like Convergence. If basic components such as wireless and touchpads are still an issue after all these years, I don't have much hope.

Best of luck.
