### Post by buzzingrobot on 2015-11-29
Jessie is, more or less, the same vintage as 14.04.  The OP might easily run into similar problems running testing or unstable.

Talk of "corrupting" is a bit over the top, though. In Linux, if you need or want new, you have to make compromises with stability (both kinds).. If you want or need stability at all costs, then you have to make compromises with access to the latest packages. 

That's not any single distribution's fault.  It's inherent in the way Linux software is developed and packaged by distributions. 

(Users usually mean "doesn't break and no bugs" when they say "stable".  The IT industry means "doesn't change" when they say "stable".)
