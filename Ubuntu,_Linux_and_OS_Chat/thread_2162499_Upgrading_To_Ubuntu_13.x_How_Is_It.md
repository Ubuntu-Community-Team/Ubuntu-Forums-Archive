### Post by Copper Bezel on 2013-07-14
There aren't any major differences in the UI or customization options. Some third-party customization tools are broken in 13.04 and have not been updated to work with the new system. That's about the extent of it. Compiz and dconf settings are unchanged, while a few Unity tweaks are slightly more readily accessible within the GUI. The changes to Unity are small tweaks and iterative improvements, really. There are some things that I like (better visual consistency, slight tweaks to Alt+Tab behavior to be more intuitive) but that doesn't really connect with tweakability.
