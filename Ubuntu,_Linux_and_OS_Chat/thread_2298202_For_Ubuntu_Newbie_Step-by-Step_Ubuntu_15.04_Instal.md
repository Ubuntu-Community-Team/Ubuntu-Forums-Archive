### Post by TheFu on 2015-10-09
Any performance tuning settings included?
[http://blog.jdpfu.com/slowVbox](http://blog.jdpfu.com/slowVbox) is my tuning guide for virtual machines. 

It also has a PDF with a similar how-to for Virtualbox - slightly older, but the instructions are still valid.

Virtualbox tends to be more popular because it doesn't cost $100+ to own and there is a 100% F/LOSS version. OTOH, for people who want to run servers, KVM would be a better choice than any of the "desktop-on-desktop" solutions like Virtualbox, VMPlayer,  or  VMworkstation.  Plus with KVM, we don't need screenshots (the Windows way) to explain things. ;)

IMHO.

Update:  BTW - New ubuntu users probably should run an LTS release - like 14.04 which provides 5 yrs of support, not the non-LTS releases which get 6-9 months of support. Support for 15.04 ends in January. [https://wiki.ubuntu.com/Releases](https://wiki.ubuntu.com/Releases)   15.10 support will end in June. 14.04 and 16.04 (when released) will be better choices for people new to ubuntu.
