### Post by Bucky Ball on 2015-07-22
Here are the links for Ubuntu, Xubuntu and Lubuntu:

[http://www.ubuntu.com/download/desktop](http://www.ubuntu.com/download/desktop)

[http://xubuntu.org/](http://xubuntu.org/)

[https://help.ubuntu.com/community/Lubuntu/GetLubuntu](https://help.ubuntu.com/community/Lubuntu/GetLubuntu)

Download the .iso file and create a bootable [disk]("http://www.ubuntu.com/download/desktop/burn-a-dvd-on-windows") or [USB]("https://help.ubuntu.com/community/Installation/FromUSBStick") from it. I recommend 14.04 LTS. Boot from the live install media you've created and 'Try Ubuntu'. Let us know if you get stuck, have questions or get that far. :)

To learn about Ubuntu it is best to install it and start using it. That is a learning curve in itself if you are unfamiliar with Ubuntu. Good luck and let us know how you go. :)

### Post by MGrowl on 2015-07-23
Try doing what you're used to doing on Windows, or whatever OS was installed before Ubuntu. I'm sure certain things that you're used to doing on the old OS would command something different in Linux (Ubuntu). Try and browse some applications that you'd like to use through the Ubuntu Software Center, read up on automating certain tasks if you'd like or the numerous features that Ubuntu offers.

In general, don't worry about it like the others said. Jump in! Have fun, and once in a while get stumped. Once you encounter a problem or question, which I'm sure everyone goes through... Post it on the forum. 

Enjoy the learning.
