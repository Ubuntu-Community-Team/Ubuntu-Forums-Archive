### Post by grahammechanical on 2017-04-14
The Ubuntu Live session already gives options to Check disc for defects & Test memory. 

[https://help.ubuntu.com/community/BootOptions](https://help.ubuntu.com/community/BootOptions)

Also, the Grub boot menu will offer Advanced Options for Ubuntu and that will allow us to load the Linux kernel in recovery mode. Among the options:

clean - Try to make free space
dpkg - repair broken packages
fsck - check all file systems
grub - update grub boot loader
network - enable networking
root - drop to root shell prompt

From the live session desktop we can use the terminal and the file manager to access the Ubuntu partition. Recently I used the live session to upgrade the kernel on an install of Ubuntu that had a broken internet connection. I downloaded the three kernel files in the live session and used file manager to copy them onto the Ubuntu partition. Then when I loaded into Ubuntu I was able to run to appropriate commands to install the new kernel.
