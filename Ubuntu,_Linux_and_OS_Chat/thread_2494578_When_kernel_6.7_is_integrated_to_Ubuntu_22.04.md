### Post by deadflowr on 2024-01-20
Most likely not,
I think the plan is to set the initial general available kernel for 24.04 as 6.6.
Which would in turn make the 22.04 HWE version the same.
The version used in 24.04 would be the last HWE kernel upgrade for 22.04.

Looking back at the last 2 LTS releases they did the same thing with both,
as all 3 would use the upstream LTS kernel versions.

But if we want we can speculate that they do move up to 6.7 for 24.04, then we would see that kernel in 22.04 around July.
