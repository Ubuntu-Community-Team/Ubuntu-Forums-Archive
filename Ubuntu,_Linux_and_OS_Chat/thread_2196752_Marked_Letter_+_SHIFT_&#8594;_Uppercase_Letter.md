### Post by grahammechanical on 2013-12-31
What you want would not be a function of the operating system but of the application that you are typing characters in. For example, in Libreoffice we can mark or select text and then go to the Format menu and select Change Case and that will give us another menu with the options of: Sentence case; lowercase; UPPERCASE Capitalize Every Word; tOGGLE cASE.

I do not know if there are keyboard short cuts for these functions. There comes a point when an application has some many functions that is becomes very complicated, if not impossible, to assign a keyboard short cut to every function.

In Ubuntu+Unity we can press Alt+F10 and that will bring up the File, Edit, View, etc., menus of the application. We can then use the arrow keys to move back and forth across the menus as well as up and down each menu.

Regards.
