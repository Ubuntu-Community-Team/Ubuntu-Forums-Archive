### Post by davposs on 2016-08-06
I recently purchased a FujiXerox DocuPrint P265dw laser printer. My research indicated it was attractively priced but with the only downside being all the posts I found stated there were no linux drivers available. As my sons who use Windows were going to be the main users I purchased it anyway - with no expectation of successfully printing from Linux other than Cloud printing.

When I picked it up at the shop,  the shop assistant indicated it was essentially identical to another Brother model - the HL2365DW. I therefore tried the Brother provided driver for that model and have managed to successfully install and print to this printer from Linux - I'm using a Brother Driver HLL2360D.ppd which should be readily found on the internet. I've only done some limited testing but so far it seems to work inclusive control of duplex priniting.
It's possible some features won't work but so far so good.

Not sure if I posted in the right place but hopefully this helps someone.
