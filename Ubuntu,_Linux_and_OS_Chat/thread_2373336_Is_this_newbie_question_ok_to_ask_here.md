### Post by mastablasta on 2017-10-05
Gnome is a desktop environment ([https://en.wikipedia.org/wiki/Desktop_environment](https://en.wikipedia.org/wiki/Desktop_environment)). 

Ubuntu uses Unity that kind of runs on top of Gnome. In newer version it will be dropped and Ubuntu will use Gnome.

Apps are not desktop dependent. for example KDE apps will work fine in Gnome, but if you install them they will install plenty of KDE dependencies (an issue if you are low on disk space or have limited internet connection). they might also look more like KDE than other apps found in Gnome (Gnome apps).

This is a good question.  A little, quick, background reading will help to answer these sorts of questions.  Start here:  [https://www.dedoimedo.com/computers/ultimate-linux-guide-for-windows-users.html](https://www.dedoimedo.com/computers/ultimate-linux-guide-for-windows-users.html)

Then read this:  [http://linux.oneandoneis2.org/LNW.htm](http://linux.oneandoneis2.org/LNW.htm)

And if you get really ambitious and want to really understand things: [http://linuxcommand.org/tlcl.php](http://linuxcommand.org/tlcl.php) is a no-hassle, free, professionally written, edited, PDF book (you can buy the dead tree version at any bookstore).

The short answer about voice control on from your PC is that you can set this up, if you are willing to give up privacy to get it.  Both Google and Amazon have provided systems that you can deploy. These aren't really ready for end-users to deploy at this point, especially someone really new to Linux.  But if you really want it, then it is possible.
