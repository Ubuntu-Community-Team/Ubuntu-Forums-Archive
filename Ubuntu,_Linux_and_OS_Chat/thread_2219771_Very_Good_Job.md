### Post by SLaCK3r on 2014-04-25
Hi all. 

I bought an HP Laptop that came preinstalled with Windows 8.1. I've been so disappointed with 8,1 ever since I purchased it. It looks like an absolute trainwreck. The Metro UI is disgusting and I just couldn't stand how it looked. Sure, everything worked great on it, but it was far from eye-candy. 

That's when I decided to theme it. I installed a few different themes on it, but nothing seemed to satisfy me. I first started experimenting with Ubuntu back in 2008. That's when I made my first account here, Back then, I thought Vista was a trainwreck! I never stuck with Ubuntu, just because I was very young and couldn't deal with terminal. 

But now, I've made the switch, I dabbled around with Ubuntu 12.04, then upgraded to Ubuntu 14.04 and I was so amazing by the OS. The native hardware detection is amazing, and the UI is gorgeous. I love how I have thousands of theme options and can do SO much with the OS. I no longer have to worry about an anti-virus or paying for the subscription, or paying for software, or get blue screens, or worry about not having a bootable USB with Windows 8.1 for when things get borked. And I think that's just the start of the benefits. I'm sure I'll discover more as I gain more experience. 

I have so much freedom with Ubuntu and a new enjoyment for relearning an operating system. I really appreciate all the hardwork that goes into developing a free operating system that, in my opinion, is more amazing than OSX or Windows.

I will be a long-time Linux user if things are always this smooth. 

Thank you.
