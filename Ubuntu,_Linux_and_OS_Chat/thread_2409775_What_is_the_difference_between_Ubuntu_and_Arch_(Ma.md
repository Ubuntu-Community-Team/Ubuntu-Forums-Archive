### Post by jdeca57 on 2019-01-07
As a first distribution Arch must be one of the worst choices one can make. You will encounter glitches, and they suppose the user to be capable of solving them. That is because it's a rolling release, injecting novelties and new programs when the are released. Problems are a certainty.
Manjaro is better, because the developers hold back these releases during a short testing period and let them pass if there are no obvious problems. Still, one can encounter problems that aren't easy to solve for a newcomer.
So the only good solution (comparing only with Arch/Manjaro) is Ubuntu , simply because it has a fixed release and if you use the LTS version like 18.04 most of the bugs are already solved. There is only the question of a desktop. For a Windows user Kubuntu (based on KDE) may be the best solution.
Of course all these distributions have user forums and documentation so you're not exactly alone.
By the way, I've used Manjaro as a first system during more than 6 months. It's very interesting to have the latest software but you have to be willing to invest more time in maintenance. That's why I abandoned it.
