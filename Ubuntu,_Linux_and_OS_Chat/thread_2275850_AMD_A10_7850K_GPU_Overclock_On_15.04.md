### Post by brian62 on 2015-04-28
I recently upgraded to an [AMD A10 7850K]("https://www.youtube.com/watch?v=_qLi3u-z9yw") processor and eschewed my discrete graphics card (a Radeon R7 260x) for the integrated R7 graphics on the 7850K. I've since been overclocking the heck out of the integrated GPU and I've pushed the gpu core clock from 720MHz **stock** to 1200MHz **overclocked.** So far, it's been stable and I haven't had any crashes or seen any artifacts.

I've been reading about folks overclocking their 7850K and I haven't read a single Windows user that has made it above 1200 and kept it stable. I don't know if Ubuntu or the Linux kernel plays a big role in keeping things stable but, it's pretty awesome I've been able to overclock it this high!
