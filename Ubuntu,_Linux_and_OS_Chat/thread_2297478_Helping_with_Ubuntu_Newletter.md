### Post by PaulW2U on 2015-10-20
Thanks coffeecat although I never did hear from Linda_Q. :(

Unfortunately we won't be publishing issue #439 of the newsletter this week. Despite many hours of preparation no-one actually turned up to write the thing. When publication time came around there was simply too much for [José]("http://ubuntuforums.org/member.php?u=1235250") to do and [Elizabeth]("http://ubuntuforums.org/member.php?u=10746") wasn't available this weekend. José and I agreed to issue a bumper edition next week which will of course take in the release of Ubuntu 15.10. 

So I'm taking this thread slightly off-topic by making an unofficial appeal for help with the newsletter. We get many offering to help, some do for a while but then drift away never to be seen again. Others do what they can on an occasional basis while others move on to more important Ubuntu projects which takes up so much of their valuable time.

All I'm appealing for is some help with summary writing, those two or three lines that summarise each news story. If there are enough of us it means a commitment of no more than 30 minutes or so each weekend. If you're not a native English speaker then that doesn't matter. Each issue is proof read for spelling, grammar and conformance with the style guidelines.

Here are some links that will give you some information about what is required:

The Team's home page - [https://wiki.ubuntu.com/UbuntuWeeklyNewsletter/](https://wiki.ubuntu.com/UbuntuWeeklyNewsletter/)
Joining the team - [https://wiki.ubuntu.com/UbuntuWeeklyNewsletter/Join](https://wiki.ubuntu.com/UbuntuWeeklyNewsletter/Join)
Guidelines to summary writing - [https://wiki.ubuntu.com/UbuntuWeeklyNewsletter/StyleGuidelines](https://wiki.ubuntu.com/UbuntuWeeklyNewsletter/StyleGuidelines)
Amber Graner's video "Ubuntu Weekly Newsletter: How to Contribute" - [https://youtu.be/TpaYsbRnteQ](https://youtu.be/TpaYsbRnteQ)

Please PM me if you're interested. Thanks for your time if you've read this far. :D

Edit: I can also be contacted on IRC - PaulW2U on #ubuntu-news on the Freenode network.
