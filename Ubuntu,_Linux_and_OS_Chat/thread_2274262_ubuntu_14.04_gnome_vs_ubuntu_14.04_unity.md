### Post by monkeybrain20122 on 2015-04-22
I haven't used gnome shell on Ubuntu but I would prefer Unity based on using it in Fedora (dual boot Ubuntu 14.04 and Fedora 21) 

I find work flow in unity a lot smoother if you set up hot corners and edges. Gnome feels kind of constricted. I can't duplicate the workflow even with addons.  The way the activity corner is located is kind of out of the way, workspaces are confined to the vertical. Edge flipping (use the mouse to switch workspace by moving to the edge) used to work (only vertically) with an addon but it is now unmaintained and no longer installable. Unity's dash is a lot more powerful than Gnome's and as far as I know you can't search recent files in gnome's dash. You can put many more items on Unity's dock than in gnome's (in GS dock the icon get smaller and smaller if you pin more than a few things on it) and there is hardly any visual cue of which apps are running by looking at the icons on the dock. The way menus and button work on gnome apps are not consistent with other things.

On the aesthetic side animation is a lot smoother in Unity than in gnome shell (alt-tab, flow cover, switching application windows etc) There is an addon that tries to do wobbly windows in GS but it is so bad that I have to remove it. I do like the dash to dock addon for GS. This is basically dodge windows in Unity. I think it was a bad decision that they remove the feature.

I think Fedora's implementation of gnome is as faithful to upstream as  possible, Ubuntu's version may be heavily patched so the experience may  be a little different.
