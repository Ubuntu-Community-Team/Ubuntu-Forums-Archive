### Post by PaulW2U on 2019-06-20
It sounds like you are suggesting an improvement to Ubuntu's installation package, ubiquity. These forums are run by users to support fellow users so you will not reach those that can consider or even implement your suggestion by posting here.

You should create a bug report on Launchpad: [https://bugs.launchpad.net/ubuntu/+source/ubiquity/+filebug](https://bugs.launchpad.net/ubuntu/+source/ubiquity/+filebug). However, you will need to create a Launchpad account if you do not already have one.

Take a look at the [ReportingBugs]("https://help.ubuntu.com/community/ReportingBugs") wiki page if you're not familiar with either process.
