### Post by Kale_Freemon on 2016-01-16
> **t0p said:**
> Is [www.how-to-remove.com]("http://www.how-to-remove.com") good, and real, or is it the kind of dodgy site it accuses others of being?

While this is by no means a particularly good way to tell whether or not the site is dodgy, I did a search for one of the programs that they recommended via one of their [articles]("https://www.how-to-remove.com/uppersearch/#zhpcleaner") called ZHPCleaner. The Google search brought up a [forum thread]("http://forum.piriform.com/?showtopic=43644") where people seem to say that it is an ok program.

If you're intending to use any of the programs that the site recommends on a Windows install, I'd recommend scanning the software for viruses before running them.

In the end just be cautious and use common sense and you'll probably be fine.

I personally cannot vouch for any of the software since I am no longer a Windows user (my cell phone notwithstanding).
