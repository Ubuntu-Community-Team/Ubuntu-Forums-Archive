### Post by Bucky Ball on 2015-07-22
*Thread moved to **Ubuntu, Linux and OS Chat**.*

This question is asked continually. Without the specs of the machine, hard to know. Really, only you can answer your question. If the machine has not much RAM, try Lubuntu or Xubuntu, but Mate is also pretty lightweight so should work fine. If it has plenty, try whatever you like. See which suits. 

If you are wanting to install another flavour because mate is slow and you couldn't be bothered working out why, then you probably should post a new thread in a support area and clearly state what the support issue is and try to figure out why mate is running slowly. Swapping the OS rarely works if the problem lies deeper. Good luck.

MATE is neither the "lightest" or the "heaviest" of environments.

Remember that MATE is the visual desktop interface.  The primary components of the various Ubuntu flavors are essentially the same.

Some info about how it was installed might be useful.  Were any error messages displayed?  How were the disks partitioned? How much space on the drive(s) was allotted to Ubuntu?

What, specifically, is "slow"? Window painting?  Opening apps?  Closing apps? Moving Windows?  Saving files?
