### Post by QIII on 2017-10-31
Why would Ubuntu's fate be any different than any other Linux distro on the desktop?

The "lack of software" is not unique to Ubuntu.  Any Linux distro of any sort - including mobile - is limited to the availability of Linux compatible software.

The Cloud is a fertile field for Linux.  Collectively, Linux tops other installations on Microsoft's Azure.  Seeing that trend, MS partnered with Canonical on the Linux Subsystem for Windows.  Ubuntu accounts gor roughly 57% of installations on Amazon EC2.  Canonical is heavily invested in Juju, Kubernetes and LXD.

I run Ubuntu MATE (ARM) on my R Pi.  My home server and my websites run Ubuntu Server.

Together, Ubuntu and Debian account for roughly 65% of web servers, with Ubuntu narrowly beating Debian in that share.

Ubuntu is most certainly not descending into irrelevence.  The devices people touch to play games, edit text and watvh cat videos are just the surface.  Canonical has long been moving towards the churning magma driving tectonic plates.  That is the future.
