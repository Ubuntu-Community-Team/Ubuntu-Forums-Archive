### Post by SurfaceUnits on 2014-06-27
**It’s that time of year again: the first alpha of the Utopic Unicorn has been released.** As is the case for all of the alphas, mainline Ubuntu does not  participate, but Lubuntu, Kubuntu, Ubuntu GNOME, Ubuntu Kylin, and  Ubuntu Cloud are all taking part. Xubuntu does *not* appear to be participating this time around, though [daily live images]("http://cdimage.ubuntu.com/xubuntu/daily-live/current/") are always available.
 Utopic Unicorn Alpha 1 includes the 3.15.0.6 Ubuntu Linux kernel.
