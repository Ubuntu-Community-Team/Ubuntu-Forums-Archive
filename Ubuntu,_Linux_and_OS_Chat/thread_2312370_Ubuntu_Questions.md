### Post by ajgreeny on 2016-02-05
Probably the best way to end up with a custom OS is to start with the [minimal-install-CD]("https://help.ubuntu.com/community/Installation/MinimalCD") which will give you a command line OS and then lets you add exactly what you want without the applications etc etc that you don't want or need.

You will need a a bit of Linux knowledge to do this but it can be done relatively easily if you follow the guides that are available.  I can not help with your specific queries about "cloud" or applications to manage payment for time, but if they are available it will not matter how you started; anything that can be installed in the default Ubuntu OS can also be installed into any of the other *ubutnu OSs.

### Post by grahammechanical on 2016-02-05
As I understand it what you are calling Ubuntu Cloud is actually a means of setting up Open Stack on Ubuntu with an Open Stack that is optimised for Ubuntu.

[https://www.openstack.org/](https://www.openstack.org/)

The approach of Canonical developers seems to be to make it easy for people to do what they want regarding cloud services but do it by using Ubuntu. Having an easy way to do things when using Ubuntu is a method of encouraging more & more people to use Ubuntu. And it seems to be a method that is working.
