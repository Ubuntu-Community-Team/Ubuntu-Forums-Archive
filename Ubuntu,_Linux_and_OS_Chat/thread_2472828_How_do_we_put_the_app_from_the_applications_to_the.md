I believe you are using the default gnome desktop, but please confirm that in replies.

The gnome developers do not wish users to have launcher icons or anything else on the desktop, so it is not possible without making use of an extension of some sort.

I do not use the default gnome version of Ubuntu so can't give you more personal experience but have a look at this recent thread which gives a lot more detail of how to deal with this.
