### Post by tgalati4 on 2013-10-13
The first rule of Fight Club:  Don't talk about Fight Club.

The first rule of Ask Ubuntu:  Don't ask a question.

Ask Ubuntu is a useful resource for cookbook questions and answers.  For edge case stuff (after all linux is an edge case to begin with) these forums work better.  We all have unique hardware, different distro versions, different desktop environments and window managers, and different configurations.  Put all of that in a box and shake it up.  How could a cookbook answer possibly cover all of those use cases?  If your question is not precisely defined (with all of the relevant information) and the answer is not equally definitive, then you have a problem--downvotes.

"I have Windows 8 running in a VM and I can't access my encrypted ZFS shares on another VM running gentoo.  Do you think it is related to my sound card not working?"
