### Post by 3rdalbum on 2013-12-05
XFCE has not stated any plans to support Mir. Mir will not run X11 window managers, they must be rewritten. (the window manager itself, not the whole desktop environment).

XMir will run XFCE, and you can install Wayland or Xorg on any Ubuntu distro that ships with Mir or XMir.
