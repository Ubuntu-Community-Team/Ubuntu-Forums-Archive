### Post by grahammechanical on 2017-08-03
Perhaps a little explanation will help.

Every two years towards the end of April Ubuntu developers will release a Long Term Support (LTS) version = 5 years support. In between, Ubuntu developers will release versions with only 9 months support. These come out every six months at the end of April and the end of October.

These non-LTS version are useful for those who are buying the latest hardware as they contain the latest Linux kernels and firmware (hardware drivers). The LTS version gets updated with these Linux kernels and firmware in time. This is to keep the LTS version up to date.
