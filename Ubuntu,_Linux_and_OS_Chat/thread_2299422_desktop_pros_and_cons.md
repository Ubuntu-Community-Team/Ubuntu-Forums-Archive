### Post by marcus_s on 2015-10-19
> **Hakan__Venderlof said:**
> Hello
I wonder what are the differences between ubuntu desktops?
Which is easiest to use?

Well... the answer to that first question could be a very long one. Each desktop environment has its own approach to solve the computer usage paradigm. [GNOME Shell (GNOME 3)]("https://www.gnome.org/gnome-3/") for example is fundamentally different from, let's say, [Cinnamon]("http://cinnamon.linuxmint.com"). [XFCE]("http://www.xfce.org") is different than Enlightenment. To learn all these differences, I guess the only way to find out is to try each one of these.

Which is easiest to use? This is not a question that can be answered with the ultimate DE. I personally find GNOME Shell and Ubuntu's Unity to be pretty good approaches to the desktop experience. [Enlightenment]("http://www.enlightenment.org") is pretty powerful all right... extremely configurable in almost every thinkable aspect, yet fairly easy to use and can be customized in itself (panel positions, number of panels, and so on).

Ultimately the easiness of a desktop environment is something only you can determine for yourself - but the differences of each one can be learned.

### Post by mystics on 2015-10-20
Try making a live USB of each and trying them out. That's the best way to know which one is best for you. You may even fall in love with one that's completely different than what you're used to.

But if you're looking for good places to start based on ease-of-use, that depends. If you've been a long-time Mac user, go with Unity, which is the default desktop on Ubuntu. If you've been a long-time Windows user, go with Xfce, which is the desktop on Xubuntu. I think that those two would offer the easiest, most familiar experience based on where you are coming from.
