### Post by Shibblet on 2020-05-01
The first Ubuntu I ever used was 8.04 Hardy Heron, which came out in 2008.  So, if we watch what happened from then to now, we can see some major improvements.

Most of the improvements have to do with architecture.  The first Core i7 came out in 2008, and was shortly followed up by the AMD Phenom II in 2009.  A lot of this is kernel level stuff, and not specifically geared toward Ubuntu.

However from 8.04 to 10.04, Ubuntu's interface was Gnome 2.  Then in 2009 - 2010  we saw the rise of the "Netbook" Computer.  Little 10" computers with lower resolutions (1024x600) that ran great with Linux based OS's.  So, this led Canonical to develop the "Netbook" and eventually the "Unity" interfaces.  The Ubuntu Netbook Remix was available for download with 8.10 Intrepid Ibex.

Gnome 2 stuck around until 11.04 Natty Narwhal, and gave way to the full featured Unity interface.  Unity was the contender until 17.10 Artful Aardvark, when Gnome 3 was finally usable.  But Ubuntu never took Gnome 3 at it's basics, they created a "Unity Like" interface with Gnome 3, and that is the features it uses until this day.

During this time Canonical created PPA (Personal Package Archives) so that newer updates or independent programs could be added to the APT library, and function as they were supposed to.  This PPA idea, I believe, eventually evolved into Snaps and FlatPaks, both are great package management for the entire Linux Community.  

Canonical also worked with Steam and Wine to create the Proton project, so that Windows Games and other software could run in Linux without too much tweaking and messing around.

Obviously, this is a quick bullet-point, highlight reel of things they have already done.  But to guess what is coming depends on the needs of the users.  

With one release of Ubuntu (I cannot remember which one.  I think 12.04) they moved the minimize, maximize, and close buttons to the upper left side of the window, and did not include a way to move them back to the upper right side in the system settings.  They quickly realized that many users were not ready for that kind of change, and released a patch to give the option back.

So... to answer your question... (here come the crazy predictions)

In 2 years:  Minor changes.  Probably more focused on fractional scaling as many people will move from 1080p to 4k displays in the next 2 years.

In 5 years:  The Linux interface with Windows 10 will be fully feature complete, and most software released for Windows (like Edge and Office) will be able to run on Linux based OS's.  There will also probably be a new flavor of Ubuntu, called "Ubuntu-Windows-Remix" or "Ubuntu-Redmond" made to look just like Windows 10.

In 10 Years:  Ubuntu will release a 10 year anniversary edition called Ubuntu-Covid-19, which will incorporate a lot of Windows viruses trying to run on a Linux kernel.  It will be feature complete, but barely functional.
But the actual 30.04 Zany Zebu, Release will be when Ubuntu finally decides to be a rolling release.  Gnome, KDE, and XFCE have all got together to make a new desktop environment called GnuKiX, and the creator of Compiz finally brings back his Window Manager, now called Completely-Iz, but forgot to add "wobbly windows."
