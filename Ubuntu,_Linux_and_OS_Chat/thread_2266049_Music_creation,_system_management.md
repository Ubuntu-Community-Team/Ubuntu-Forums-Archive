### Post by hamnstar on 2015-02-19

### Post by tgalati4 on 2015-02-19
Yes, that is the dilemma for the creative professional.  I have a machine set up for recording that I don't connect to the network.  It's running an old distro, but it works.  I transfer files using sneakernet (usb sticks).  

Because linux is free and roadside computers are free, the combination allows one to set up a machine for a specific purpose and then not have to change it.

Trying to set up a studio machine and use it for web browsing (with all of the required updates) means you need to allow for downtime to fix breakage.

So, find some old machines and repurpose them for recording only.  When you have a setup that works, don't mess with it.  Ever.

### Post by tgalati4 on 2015-02-19
There's nothing worse to stifle creativity than to have to fix broken packages after an update.  So set your system up and jam away.  I put a handle on my system (a generic, metal whitebox PC) so I can carry it to gigs.  I bolted the preamp inputs to the top of it.  

You want a low-latency system, without a lot of extra cruft for music recording.  Good luck with your system and include some SoundCloud links of your creations.
