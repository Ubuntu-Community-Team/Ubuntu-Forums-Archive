### Post by poorguy on 2018-05-11
If you've just installed Ubuntu 18.04 than you should update it.

Lower left hand of desktop square dotted box open it and look for an icon*** Software Updater ***click to open and check for updates to run.

Lower left hand of desktop*** square dotted box ***open it and look for an icon ***Terminal ***click to open ***copy and paste*** this command ***sudo ufw enable ***to turn on firewall.

To check ***ufw firewall status *** open terminal and ***copy and paste ***this command ***sudo ufw status.***

This will get you able to use the internet.
