### Post by grahammechanical on 2015-09-21
Every Libreoffice document loads from default template. It makes it easier for the user. We can stay with the defaults without needing to understand about styles, headers, tabs or margins. Or we can save the document giving it an name and then change things and the changes will stay for that document. We have not changed the default template. Libreoffice Help gives advice on how to edit the default template or create and use Custom templates. I have never done this myself.

Most applications have keyboard shortcuts. It dates back to the days before the WIMP (Windows, Icons, Mouse, Pointer) interface. Using the keyboard was the only way to do anything in any application. Ctrl+key will still do things. So, mis-hitting a key when the Ctrl key is pressed will cause unintended actions. For example, Ctrl+B = Bold characters. Mis-hit the letter B and hit N instead and a new document will open. I have been known to do that a few times. Also by mis-hitting keys I sometimes trigger the screenshot utility. You may have done something similar.
