### Post by Keith_Helms on 2016-12-27
If all the hardware seems to be working, probably playing around with the preinstalled apps and going to the software option to see what is available in categories you're interested in - multimedia, games, office tools, etc.  What you do is determined by what you want to use the computer for in the first place.  There's not really any checklist of stuff you have to do before you can use the system.

### Post by Geoffrey_Arndt on 2016-12-27
A typical install of Ubuntu works just fine "out of the box" so to speak - - but many do a series of "tweaks" to improve the overall experience (such as enabling DRM'd DVD playback, which is not enabled by default).

So, I would suggest to start bookmarking key web sites that contain the info to do this (besides the main Ubuntu site and this forum).

Here are some common tweaks for a post install of 16.04:   [http://www.omgubuntu.co.uk/2016/04/10-things-to-do-after-installing-ubuntu-16-04-lts](http://www.omgubuntu.co.uk/2016/04/10-things-to-do-after-installing-ubuntu-16-04-lts)
