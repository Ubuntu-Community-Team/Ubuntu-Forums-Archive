### Post by deadflowr on 2017-05-04
Ubuntu has always been a gnome desktop, even with unity.
(albeit, with unity it has been a sort of frankendesktop with mixed and mashed versions of gnome packages and a few unity spices builtin, like unity scopes and the like.)
But the base of Ubuntu is gnome, through and through.

The change will try to streamline Ubuntu better with gnome as a whole, which includes using gnome's desktop interface --gnome shell, in place of unity.

### Post by mastablasta on 2017-05-04
a better question is why not KDE (they could have gone with KDE instead of Unity and they would get additional dev support there for the mobile platfrom that was floating arroudn at the time).

but the reasons are explained - Ubutnu was a gnome desktop before and it simply continued to be Gnome oriented. Unity was a shell for it. Gnome philosophy is to keep it minimalistic, simple. which is good for new users. they will not get overwhelmed with options like those available in KDE. 

then again when things get tough you might need to drop to command line with Gnome. KDE on the other hand has GUI for so many things that it is not even funny anymore. They have very good applications as well (Skrooge, Krusader, Ktorrent, Amarok, Koffice suite etc.). but again these applications are aimed at those that like to tweak, that like to have plenty of various options displayed to them. Gnome applications are usually simpler to use. which again goes hand in hand with Ubuntu's goal as good entry level linux OS.

### Post by TheFu on 2017-05-04
I think Ubuntu (X/L/K/Gnome/Mate) are bloated.  I wish they'd put a minimal window-manager in place and let the users choose if they want more.  Something like fvwm-crystal would be wonderful and take us back to when men use Linux.
Take a look at the desktop images here: [http://fvwm-crystal.sourceforge.net/screenshots.html](http://fvwm-crystal.sourceforge.net/screenshots.html)  It provides nearly complete user control over anything displayed on the screen.  
Want thinner or wider scrollbars? No problem.  
Want thinner or wider titlebars? No problem.
Want certain icons in certain workspaces?  No problem.  
Want Firefox to always launch to ws:2x3?  No problem.
Don't want to screw with any settings, just want to apply a theme?  No problem. Must be 50 different themes.
A menu is available by right-clicking anywhere on the desktop. No need to move the mouse all the way to the top/bottom/left/right of the current workspace.

Of course, if you don't want to screw around with any of the available settings, you don't have to. There are reasonable defaults. Applications run fine.  

Not trying to take away anything from the choices of other people. I just find them all too bloated.
