### Post by kemtnbkr on 2012-09-16
+1 for Xubuntu.  Or Lubuntu is also another option.  I used Lubuntu for a while, but now I think I am leaning towards Xubuntu- it has a bit more elegant of an environment imo.  But, see for yourself what you like!  Both are very lightweight- even with only 512MB of RAM, you should be fine most of the time unless you're trying to do all sorts of stuff at once.

For example, I'm running xubuntu right now on my laptop, and my current memory usage is ~505MB.  I have a 4-page PDF open, and Firefox w/ a gmail tab, and 3 others (including this one).  So, Firefox is using ~275MB.  I think you should be fine for the time being with either Lubuntu or Xubuntu 12.04
