When I was in school and couldn't afford a decent computer I would run Linux on a lot of cheap/free computers.  I found that used cheap or free computers almost always required some work to get them decent for streaming video.  Usually a full cleaning and ram upgrades.  I also tested a lot of lightweight Linux OS's and found that I prefer Mint with Xfce or Mate desktop environments on older computers, however Xubuntu is also good and highly configurable (I just prefer the default look of Xfce on Mint than on Ubuntu).  Any of those distros you listed in your post will do the things you wanted them to do very well.   

In the past as a "project" I installed Puppy Linux on a "computer" with 250MB Ram and a 8 Gig compact flash memory card for a hard drive.  It was able to surf the web ok, but streaming video was not so great.  I expect nearly any modern computer, made within the past 10 years and still working, can run some version of Linux and do modern web activities.  

My current laptop is Compaq that I bought at a big box retail store for about $300 several years ago and it runs very well with the current full LTS versions of Ubuntu without any issues.

Honestly, for traveling and with what you want to do, a tablet may fit your needs and budget better.

### Post by coldraven on 2015-08-26
I just bought a HP Compaq 6715b on eBay for $90. It runs Ubuntu 14.04 with no problems although I have not bothered to try and get the fingerprint reader to work. The screen is not as bright as a new laptop but that is to be expected for something this old.
A word of advice, if the machine has Windows on it then update the BIOS before removing it. For this laptop the HPQflash utility worked perfectly on MS Vista and the BIOS is now version F. 20. Without updating the Bluetooth might not work.
To recap, everything except the fingerprint reader will work after the install without having to tweak anything.

It meet all your requirements, it will not be any good for 3D games.

### Post by kurt18947 on 2015-08-26
That machine should work fine. I recently sold a Thinkpad core2duo 1.5 ghz. w/ 2 GB. RAM. It'd run Mint, Ubuntu-Gnome no problem.  I sold it because Core2Duo doesn't do Virtual machines well.  If there were a challenge, I'd guess it'll be WiFi. If it has Intel WiFi you'll probably have no issue. If Broadcom or other chip supplier you may need to spend some time in the networking and wireless section here. Personally I might install a 32 bit O.S. rather than 64 bit. It seems like 32 bit O.S.s run better on older hardware and for what you're doing I doubt you'll be giving up much running 32 bit. Unlike Windows, Ubuntu has PAE enabled by default so should use more than the 3.2 GB. RAM  Windows limit. I seldom see more than 1.5 GB RAM use unless I have VirtualBox running a Windows 10 guest O.S. Ubuntu-Gnome host Win10 guest uses about 3.5 GB. RAM.
