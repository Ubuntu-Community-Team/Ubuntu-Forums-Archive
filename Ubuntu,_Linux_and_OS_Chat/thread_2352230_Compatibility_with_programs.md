### Post by SeijiSensei on 2017-02-11
Do you own a legal copy of Windows with an authorization code?  If your computer has enough memory, you can run a program called [VirtualBox]("http://www.virtualbox.org/") that lets you create "virtual machines" and install Windows into one of them.

That said, I'm quite suspicious of the claim that a laptop cannot run Windows since nearly every computer comes with Windows installed from the factory.  Perhaps he meant that it was underpowered for something like Windows 10?  What about Windows 7, which is still available from places like [NewEgg]("https://www.newegg.com/Product/ProductList.aspx?Submit=ENE&DEPA=0&Order=BESTMATCH&Description=windows+7&ignorear=0&N=-1&isNodeId=1")?
