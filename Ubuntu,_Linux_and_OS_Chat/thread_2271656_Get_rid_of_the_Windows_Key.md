### Post by bouwerik on 2015-03-31
Hi All,

Have you all noticed the windows key on every keyboard out there? If it is a standalone keyboard or integrated on a laptop, a windows key is always present.
I would like to do something about that because there is a huge community that does not use Windows. Like me.
Instead, why not create a universal start button. So every OS can use that button.

If you like to join me in the struggle against the Windows Key, please feel free to sign the petition at [http://www.nowindowskey.com/](http://www.nowindowskey.com/)

With the petitions i will get in contact with hardware suppliers to discuss any options.
Thanks in advance for your support. Hopefully we will get rid of the windows key.
