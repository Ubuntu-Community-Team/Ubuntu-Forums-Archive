On Ubuntu we do not add or remove panels. But this can be done in some of the other Linux distributions (flavours) built on Ubuntu but with different desktop environments. And I am now starting to suspect that you are using one of the flavours.

In Ubuntu we find our installed applications in the Dash. In the flavours the applications usually appear in some menu list under various sub categories. And Ubuntu Software is indeed an application store.

In Ubuntu hold down the Super (flying windows) key to see a overlay with keyboard shortcuts. Super+A opens the Dash at the applications lens. And there are all the applications under Installed See xxx more .
