Hey all,
I have been using ext2fsd [http://www.ext2fsd.com/](http://www.ext2fsd.com/) to mount and read my ext 2, 3 and 4 partitions in Windows. The program is great and works flawlessly for read-only access. If you are looking for a read/write ext 2, 3 and 4 fs driver for Windows then y'all need to check out this link.

[http://www.paragon-drivers.com/extfs-windows/](http://www.paragon-drivers.com/extfs-windows/)

it's free for personal use.
