### Post by mbott on 2019-02-10
Over the last twelve years that I've been an Ubuntu user, I've always kept my eye on Linux Mint.  Ubuntu ran om my primary i7 laptop and also on an older A6 laptop I keep in the basement.  The A6 had been running 14.04LTS mainly because it did not want to upgrade to anything past that.  I didn't want to mess with it so I left well enough alone.  Now, with support for 14.04 rapidly approaching, I thought I'd have to pick up another laptop for downstairs.  On a whim, I installed Mint 19.1 Cinnamon and it is fantastic on that old machine.  Not one single issue I've encountered.  

I was so impressed with it on the A6, I installed it on my primary laptop replacing 18.04. So, for a while anyway, I'll have to keep my eye on Ubuntu like I used to with Mint.
