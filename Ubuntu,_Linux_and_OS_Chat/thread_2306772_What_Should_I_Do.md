### Post by mikodo on 2015-12-18
Hi danny.

+1 to all the advice so far. I especially think you need to get a hold of some .iso's of distro's she might like and use the live versions to:

1. Check each to be sure everything works. And possibly has enough resources on her computer to run the more heavier distros like Ubuntu, but if she is already using Windows 7, Windows 8.1 or Windows 10, Ubuntu would need less resources than those.

2. Let her play with the slower live cd's on her computer to get a feel for which she likes. Keep in mind not all desktops are given the same amount off attention to the aesthetics. Ie. Xubuntu. very plain on .iso but is wonderfully able to be customizable to anything you want. On the other hand Ubuntu is very polished right off the hop. A couple lighter official Ubuntu derivatives that are also well aesthetically set on the .iso's are Ubuntu Mate and Lubuntu. 

So, it depends on what her computer and peripherals will run well with and which of those she likes.

I would stick to Ubuntu or Ubuntu official derivatives if, you want the best support for problem solving.

[Here is a link to the official Derivatives of Ubuntu]("https://wiki.ubuntu.com/DerivativeTeam/Derivatives"). All have Ubuntu workings underneath. See the section headed, "Recognized Flavors".

Addendum. Keep in mind the support time lines of each distro. [Here is Ubuntu's]("http://www.ubuntu.com/info/release-end-of-life"). Each flavor has their own time lines. So check those on their pages.

I'm sure given you doing the groundwork for her and letting her decide which she likes, there will be "options" she will find herself happy with.

### Post by buzzingrobot on 2015-12-18
> **danny_w2 said:**
> I have a family member who wants me to install Linux
She is new to this whole thing so what should I install? Ubuntu or Linux Mint

Unlike Windows, you can boot installation images of Linux distributions and play with them to see what they're like. They run entirely off the DVD or USB stick you use without touching the hard drive.

Mint uses a different intrerface -- the GUI you interact with -- than Ubuntu but, with very few exceptions, the applications you use are the same and come from the same place.

Go here to download an Ubuntu install image and for instructions on burning it to a DVD or USB stick: [http://www.ubuntu.com/download/desktop](http://www.ubuntu.com/download/desktop)

Go here for Linux Mint:  [http://linuxmint.com/](http://linuxmint.com/)  There's a link to their download servers as the top of that page. The instructions for burning the install image provided at the link above work just as well with Mint.

Live mode off a DVD is rather slow, but off a USB things are fast enough to get a reasonable impression.

Don't be in a hurry.  If you decide to install one of them, be sure you are comfortable with the instructions. If you intend to dual-boot with Windows, then be sure you understand how to set that up correctly.

If you are not going to dual boot and will wipe out Windows completely, then count the first couple of install attempts as a learning experience if something doesn't work correctly. Come back here and ask questions at that point.

### Post by night_sky2 on 2015-12-20
> **danny_w2 said:**
> I have a family member who wants me to install Linux
She is new to this whole thing so what should I install? Ubuntu or Linux Mint
I usually recommend Linux Mint to beginners for the following reasons:

- Built on Ubuntu LTS base.
- Ship with all media codecs, Java and Compiz out of the box.
- Skype and Steam are available in the Software Manager.
- Ready to use with apps such as VLC Media player, Banshee, LibreOffice, Gimp, Mint USB stick formatter ect.
- Xfce Edition is supported for 5 years instead of 3.

This just makes it easier overall. Once they install the distro, they can start enjoying it right away.

That being said, you can't go wrong with Ubuntu and it's flavors either. But there are extra steps the beginner needs to go through in order to get that fully productive system.
