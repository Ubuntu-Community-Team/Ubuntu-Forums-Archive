The Ubuntu installer will offer you an option to select additional packages for installation.  Those include the Flash plugin.  If you do not select that option, Flash will not be installed as you install Ubuntu, but can always installed from Ubuntu's repos. The package name is "flashplugin-installer". Welcome to the Ubuntu Forums, OctaVive! =)

As a programmer, I'd just like to add that no software is "bug free" - including Windows.  Yes. Ubuntu 16.04 has some bugs.  Are they going to affect you?  Possibly, but the situation is no better or worse than using Windows 7 (which is outdated, and will cease support completely in 2020) or Windows 10 (with its constant patch cycle that can leave things in an unstable state).   

My advice to you is to decide what you want to do with your laptop first, and then worry about what operating system you use after that.  Windows can become slow for a variety of reasons, most often because of fragmentation or profile problems, both of which can be fixed with a reinstall.  I'm not trying to dissuade you from trying Linux - quite the reverse actually.  What I am trying to do is give you a realistic view rather than a fanboy/fangirl response.

Linux is different than Windows, because it was designed for a different purpose.  Contrary to popular opinion, programmers have no interest in making Linux take over from Windows.  Linux was designed to be customizable. Everyone has a different idea of the way things are best for them. (Android is little more than a custom version of Linux.) Linux lets you change Linux to fit your needs, rather than the way Windows makes you conform to the way Microsoft wants things done.  That means that if you are going to commit to using Linux long term, you may want to get your hands dirty and tweak a few things to suit yourself.  You don't have to be a programmer. It can be as simple as choosing what kind of "point and click" style you like. Once you get used to the idea that you are in charge of what you want, you may find that you don't like Windows anymore.

Linux can do everything that Windows can do, but in some rare areas, Windows is easier for non-techies.  For example, if you are an avid gamer, the majority of games are written for Windows, not Linux. If so, then Ubuntu 16.04 is probably not for you - mostly because AMD's drivers are "blah" and do not work well with games.  A more technically minded person can get around these limitations.  For example, with my setup I can run any Windows game flawlessly using a technique called KVM (where I can start a copy of Windows inside of Linux). 

As a first time user, you may feel overwhelmed with the freedom Linux offers to customize.  If you feel up to the challenge of overcoming a few obstacles on your journey, we are all here to help you.  If you prefer that things remain at a "no brainer" level, perhaps reinstalling Windows might be best.


Take care!
T.J.
