### Post by straxus on 2013-10-19
I've been running Mythbuntu on my HTPC since 7.10.  In all that time, I've only performed one clean install of the OS, and that was at 9.04 because I had all new hardware. Since then, I've performed nine OS upgrades, each time resolving whatever issues came up.  Sometimes input from the IR remote stopped working.  Other times HDMI sound would break.  Sometimes the Nvidia driver would end up borked, and I'd be dumped to the console.  Sometimes certain applications wouldn't run as expected.  Every time this happened, I worked out the issues and got everything working properly again.  With 13.10, HDMI sound broke (again), Apache wouldn't run, and the N64 emulator wouldn't display anything when launched.  I've sorted through most of these problems.  It's a process that's frustrating, but rewarding.  And I'm going to miss it.

I decided about a month ago that I want to switch to a 64-bit environment for the performance gains in certain operations that I think will be beneficial to me.  While there are guides for switching from a 32-bit system to a 64-bit one, and the idea of such a dangerous and pitfall laden task appeals to me, ultimately it would probably take too long for me to sort through all the problems it would cause, which would severely inconvenience my family who rely on it far more than I do.  So the only sane course is to do a clean install of Mythbuntu 14.04 64-bit when it releases. (Mythbuntu only releases ISOs for LTS releases these days)  So this is it for this particular install.  It's had a good run, and I'm a little sad to see it go.

I've also decided to stick with the LTS release for two year rather than upgrade.  It has been a while since upgrading the OS brought any noticeable benefit to my system.  So while I enjoy the process, it makes sense to just upgrade MythTV going forward, and spare my family the extra "futzing" (as my wife calls it) with the HTPC every six months.

Does anyone else keep upgrading a very old installation of Ubuntu?  If so, why do you choose to upgrade rather than perform the recommended clean installations?
