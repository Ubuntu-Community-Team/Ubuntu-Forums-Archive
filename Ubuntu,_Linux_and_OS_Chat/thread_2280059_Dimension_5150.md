### Post by mattharris4 on 2015-05-28
Lubuntu should work for most sites and tasks.  However in my experience using Lubuntu with Firefox and Chrome browsers there are some websites that require 1.25GB of RAM to load and run smoothly.  Can you afford to add another 512MB or 1GB to this computer (with most computers of this era that is easy and the sticks usually cost less than $30 on Amazon)?  The Passmark on your Pentium D 3.20GHz CPU is 707, it should run but some web pages will scroll jerkily (not that big of a deal of that on my laptop scoring a 620, close enough for comparison but it is there and you will notice it), especially if you are also using the computer to play music or have several tabs loading a CPU and RAM intensive website at the same time.  I think it will be plenty usable with a RAM upgrade, though.  

If you don't mind not being able to watch many videos (if not all) and not being able to view picture galleries in the usual manner using the web browser Midori might be an option and certainly would work on 1GB of RAM for any website's text.  You can install Lubuntu, then install Midori using the Software Center to try -- if you don't like it and prefer Firefox or Chrome you can install new RAM at that time.  You could also use Firefox or Chrome to load the websites you use and attempt to use them in your usual manner, using the system monitor to keep an eye on RAM use.  If you start going over your installed RAM your computer will go into swap and might slow to a crawl (swap is meant to handle a RAM overload issue so you can save whatever you need to and lighten the load or reboot), then you know you need to take my advice and up the RAM in your computer.  If your computer goes into swap the most you need to do is reboot into Lubuntu to rectify that (and exiting out and reloading your web browser may rectify it), you won't wreck your computer going into swap.

Also if you wish to use XP for anything you should load Service Pack 3 before installing any other OS (you can dual-boot with XP as an option if you wish).  In general you should not use XP for any internet but I think if all you do with it online is download the service pack you should be OK.  You might need XP for some program using Windows that either do not have a Linux version or WINE cannot run in Linux, you don't say what programs you are currently using.  Certainly you should install some Linux OS that your computer can handle for internet purposes.
