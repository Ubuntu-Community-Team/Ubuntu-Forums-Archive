No issues [here]("http://askubuntu.com/questions/551320/ubuntu-on-asus-x205ta"). Also gives instructions for install. There are quite a few examples of people installing with few issues and if this is a newer model machine you have an even better chance of getting 15.10 onto the machine.

As there are no links I could find regarding that ASUS machine and 15.10 it is hard to know how they get along. As there is an absence of reported problems, perhaps blissfully, but 15.10 has only been out a week. I'd better say something here.. as the starter of the "megapost" :)

x205TA is a good machine for working and surfing on. Mine has Windows 8.1 on it and I run Ubuntu with the Mate desktop in a VM under OpenBox. I use it for developing small programs, and it is perfect. In the VM, there is full access to all the hardware and it's flawless.
