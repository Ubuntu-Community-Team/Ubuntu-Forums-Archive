### Post by TheFu on 2019-08-30
I use my own photos. I KNOW they are safe.
```
#!/bin/bash
env DISPLAY=:0 feh -qd --draw-tinted --randomize --bg-max -f ~/Pictures/pict-list.2019
exit;

```
Run that to change the background, if you use normal X/Windows.  You could put a sleep-loop to change it periodically and run it as part of your session startup.
