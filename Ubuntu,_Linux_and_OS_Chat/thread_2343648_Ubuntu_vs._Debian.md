### Post by QIII on 2016-11-17
Hello!

It's hard to say if your performance would be similar with Ubuntu Mate.  However, Unity is certainly a very resource intensive DE and using another DE might result in a substantial performance increase.  

If you already have Debian Mate installed and you like how it's working, it may not be worth the trouble of reinstalling a new OS again.  Remember that Ubuntu is based on Debian.  If what you have right now works for you, that's what matters.
