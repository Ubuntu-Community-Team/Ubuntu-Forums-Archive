### Post by a.frings on 2014-04-23
I like to give some feedback about Unity. 

Whenever german computer magazines wrote something about Unity, people commented this with a lot of negative feedback. Surere there had been technical issues, but it seemed that most people did not like the style of Unity at all. However we should keep in mind that people usually write more about things they dont like than about things they like.

So I want to report that I like Unity. 

For years I preferred KDE. I loved the many configuration options - they were more a toy than something to work with. And that's what I wanted.

However, I had lots of problems with KDE (under Ubuntu as well as Debian) since the NetworkManager had been Introduced. Switching to other Linux distrbutions did not help. The Network-Manager was source of big stress. You can imagine that a computer without proper Internet connection is not funny. So this was the first time I really thought about switching over to another desktop.

I installed Ubuntu (i think it was version 11.04) with Unity desktop. It ran quite unstable and I had difficulties to find out how to do some simple things like starting several instances of the same application. I continued to use several KDE programs (e.g. it's file manager) because I missed lots of features in the default tools. After only some few weeks, I gave Gnome 3 a try. It ran much better than Unity, so that remained my choice for almost 2 years.

But time has changed things. 

In the meantime I lost my interest in playing with my computers operatin system. I now "play" with programming electronics and mobile phones and I bought a japanese game console to play electronic games. My computer is now a tool to work, play music and surf in the internet. And my new computer has a quite small 12" display - so KDE is not applicable anymore.

This month, I installed Ubuntu 14.04. And this time, I keep the default desktop as well as all default Tools. They work fine, they simply do what they should do. Neither more nor less. Unity became much better during the last years. 

I prefer Unity because it minimizes useless windows decorations. The idea to merge menus with window frames and remove frames of maximized windows was a great idea. I also like that Unity does not try to copy the style of Microsoft, Apple or Google. Ubuntu found its own unique style and thats good. Why else should Ubuntu exist, if it would look like another Linux distribution?

You did a great job. Thanks for that.

### Post by Copper Bezel on 2014-04-24
I feel compelled to sound off as another big fan of Unity. Once I tried it, I was hooked - nothing has felt quite as cohesive and complete since. I did also like Gnome Shell early on, but it's just not as featureful and feels limited now by comparison. Unity keeps implementing new tweaks and polish, and although there a few features I can't abide out of the box, like the poor use of multitouch gestures, there are some truly unique features in Unity that no other system has an answer for. The search is simply the best and most powerful, and once I found out I could modify the categories displayed through dconf to remove anything related to suggestions and add my Tomboy notes as an ever-present section in the home lens, it quickly became indispensable.
