### Post by sudodus on 2016-05-19
0. You can download standard Lubuntu iso files via [community/Lubuntu/GetLubuntu](http://help.ubuntu.com/community/Lubuntu/GetLubuntu)


1. But if you want Lubuntu Core, you should start from an Ubuntu **mini.iso** or Ubuntu Server iso file.

You can download the Ubuntu mini.iso from [http://cdimages.ubuntu.com/netboot/](http://cdimages.ubuntu.com/netboot/)

and install Lubuntu Core according to the following link,

[Lubuntu/Documentation/MinimalInstall](http://help.ubuntu.com/community/Lubuntu/Documentation/MinimalInstall)


2. You can also take a short-cut and get a tarball or compressed image file of an already installed [Lubuntu or] mini system, and from the mini system install Lubuntu Core, Xubuntu Core or a full desktop environment or maybe only a simple Fluxbox window manager to get a light-weight system,

[One Button Installer, 'OBI'](http://ubuntuforums.org/showthread.php?t=2172971)
