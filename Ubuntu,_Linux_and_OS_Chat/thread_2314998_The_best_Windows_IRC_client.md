### Post by qyot27 on 2016-03-03
> **t0p said:**
> What IRC client for Windows would be good, considering that I like XChat?  I've heard tell of HexChat, which is supposed to be similar to XChat, but I tend to find that things supposedly like other things are often not at all like those things.
> **verymadpip said:**
> HexChat is pretty much an XChat clone.
For the record, HexChat is not 'supposed to be similar' or a 'clone' of XChat.  [When XChat shafted its Windows users by making official Windows builds shareware](https://hexchat.github.io/shareware.html), the still-GPLed source code that had to be published for the still-free *nix builds was forked and resulted in XChat-WDK.  Then when XChat development slowed down/died (the last release was back in 2010), XChat-WDK moved to re-support non-Windows OSes and was rebranded as HexChat.  So calling it a 'clone' or the idea that it's trying to imitate XChat are incorrect, because it's descended from XChat directly (but with the requisite post-fork evolution and active development).

I'd been using XChat or HexChat on *nix for a couple years, so when I bought a Quantum Byte I ended up just going with it on Windows 10 as well.

> **t0p said:**
> Also, pile in with suggestions of Win IRC clients that are absolutely nothing like XChat.
Prior to switching to HexChat everywhere, I used to use Nettalk.  It has a couple features I'd actually like to see elsewhere (the quick connect list or XDCC transfers having a dedicated tab rather than a floating dialog box), but its development is pretty much stopped too.

If I want to go way, way back, the first IRC client I'd ever used was IceChat, but it didn't have support for XDCC RESUME at the time (maybe it still doesn't, I don't know), which is why I moved to Nettalk and eventually HexChat.
