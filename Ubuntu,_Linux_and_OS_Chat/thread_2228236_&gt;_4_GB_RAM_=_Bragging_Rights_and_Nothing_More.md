### Post by pqwoerituytrueiwoq on 2014-06-08
i just mount it in /etc/fstab
```
# /media was moved to RAM after installation
tmpfs                                     /media          tmpfs   defaults,size=1M,mode=755 0       0
# /tmp was moved to RAM after installation (50% of ram is used for this)
tmpfs                                     /tmp            tmpfs   defaults,noatime,mode=1777 0       0
```
/media just contains folders that contain mount points, a ram disk makes these folders temporary

my main ram disk is /tmp i mount other folders within /tmp using bind like this
from /etc/rc.local
```
if [ ! -d "/tmp/Desktop" ];then
    mkdir  /tmp/Desktop
    chown me:me /tmp/Desktop
    chmod 1750 /tmp/Desktop
    mount --bind /tmp/Desktop /home/me/Desktop
fi
```
i use this to store temporary files files, basically anything i don't plan on keeping very long and stuff i don't want saved (don't ask) to my hdd
