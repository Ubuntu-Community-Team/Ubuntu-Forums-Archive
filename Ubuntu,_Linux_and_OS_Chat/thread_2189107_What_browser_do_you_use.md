### Post by shantiq on 2013-11-22
[pepper on chromium anyone?]("https://launchpad.net/~skunk/+archive/pepper-flash")

    
then it is all a lot smoother

1. install PPA

```
sudo add-apt-repository  ppa:skunk/pepper-flash
``` and click enter
```
sudo apt-get-update ; sudo apt-get install pepper-flash
```

2. flush cache ```
rm -rf  ~/.cache/chromium
```

3. > MAKING CHROMIUM USE PEPPER FLASH

 Once the pepflashplugin-installer package is installed, you will need to configure Chromium to use the plugin. 

Open /etc/chromium-browser/default in a text editor

```
sudo gedit /etc/chromium-browser/default
```

 and add the following line to the end of the file:

```
. /usr/lib/pepflashplugin-installer/pepflashplayer.sh
```

 (I.e. a period, then a space, then the filepath.)
