### Post by grahammechanical on 2017-11-26
What we are seeing is the difference between Linux distributions and operating systems by monolithic organizations such as Apple and Microsoft. A Linux distributions is made up of many separate software projects. And so there are several Linux file managers available. If all these developers of file managers were to call their project simply "file manager" we would really be confused.

So, the default file manager for the Gnome desktop environment is Nautilus. That for KDE is Dolphin. The one for XFCE is Thunar. And they are not the only Linux file managers around. 

Each Linux distribution will have an icon and a menu listing for a file manager and the default file manager for that distribution will load. But when it comes to loading the file manager through the terminal, then we have to use program's actual name because that references all the files for the program.
