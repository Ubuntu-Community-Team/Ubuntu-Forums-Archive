### Post by grahammechanical on 2014-06-22
I have found that the default installation of Ubuntu usually takes care of most things. We have to be unlucky for our wireless adapter to not be recognised by Linux. Proprietary video drivers are installed during the installation process by our ticking the box "install third party software." We do not need to set the monitor screen resolution because the video server reads the EDID (Extended Display Identification Data) stored in the monitor. Oh, yes, things could be much, much harder and with some distributions of Linux it is a lot harder for the new user.

With Ubuntu we have a simple choice. Do we accept Ubuntu as it comes to us after it is installed? If so, just read the Ubuntu Desktop Guide (Help). Or do we want to modify the OS beyond what Ubuntu offers? If we want that then we need to understand that there is no risk free method of learning how to do things. There is no easy way to learn the power of the Born Again Shell (Bash - terminal).

If we want to experiment with the powers of the Bash terminal, then my advice would be to dual/triple boot into another installation and do the experimenting there. If we break the OS beyond our understanding to repair we can then simply re-install and continue with our experiments. And we will not have broken our working Ubuntu.

Regards.
