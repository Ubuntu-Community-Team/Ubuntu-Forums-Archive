### Post by giorgiotani on 2020-10-06
PeaZip 7.4.2 was released yesterday.
p7zip 16.02 is now superseded by szcnick/p7zip fork 17.02, introducing support for FLZMA2, LZ4, and Zstandard compression algorithms for 7z archives.
Also, Brotli and Zstandard are available as standalone single files compressors, multiple input is consolidated into a single TAR archive before compression step.


PeaZip is available for Linux x86-64 as DEB package, and also as RPM and portable package not requiring installation: [https://peazip.github.io/peazip-linux.html](https://peazip.github.io/peazip-linux.html)
