It only uses a small amount of system resources(they say it has requirements of win 98) - I installed debian with LXDE on 10 years old laptop which wasn't very powerful when bought (1.2 GHz Pentuim 3, 512 MB RAM) and it runs surprisingly well. Some time ago, I've read a nice comparison, so if you want to read further, here is the link:
[http://pclosmag.com/html/issues/201109/page08.html](http://pclosmag.com/html/issues/201109/page08.html)

As far as 32/64 bit distro is concerned, I don't think it matters too much. Main advantage of 64 bit OS is that it works with more than 2 (or was it 3?) GB of RAM. I think, I could be wrong.

### Post by mattharris4 on 2015-04-19
> **gordintoronto said:**
> That configuration should run Xubuntu well.

That configuration should run either Lubuntu or Xubuntu well.  I am commenting under the assumption that by FOREX you mean the financial trading market professional financial traders use and an affiliated web-based trading portal.  If that is the case I don't think you will have any issues with those OSes using a well-recognized web browser such as Firefox (which comes with Canonical OSes) or Chrome.  Light web browsers such as Midori may not work for your intended use but with your hardware you don't need to go there anyway.  Be sure to use the LTS version 14.04 of a Canonical OS so you are not forced to change versions every 6-9 months to keep security support (very necessary when using a computer for FOREX transactions).

Also, you want the 64 bit version since that is what your computer's processor is designed to accept.  32 bit OSes may work if you have a BIOS and not a UEFI-based system but if you do add RAM to the computer it may not all be usable (32 bit OSes are limited to about 3.2GB of RAM).

If you were using the computer for non-specialized uses any of the OSes you list would probably work.  It is likely FOREX that could cause trouble with less-developed OSes such as SilTaz as most financial websites require the OS and browser support 128 bit security (which Lubuntu and Xubuntu do when paired with either Firefox or Chrome/Chromium).  I cannot speak knowledgeably to the other OSes you list although Elementary OS is supposed to be well-developed enough where I would be shocked if they did not take secured websites into account in its design.  You will likely need to install Pepper Flash (it should be in the Software Center of either OS) to play videos from a website such as Bloomberg.

Also, good luck with your career in finance.  I wish you well and hope you make a mint for both you and your clients.
