I'm pleased to annouce a new Nautilus extension.

nautilus-mediainfo - View media information from the properties tab
Uses mediainfo lib ([https://mediaarea.net/en/MediaInfo](https://mediaarea.net/en/MediaInfo))

Installation
ppa:caldas-lopes/ppa
or
[https://github.com/linux-man/nautilus-mediainfo/releases](https://github.com/linux-man/nautilus-mediainfo/releases)
