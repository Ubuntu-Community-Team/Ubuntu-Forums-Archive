### Post by farrinux on 2016-04-08
I left the junk strewn MS road around the time Ubuntu 8.04 came out. My brother in-law had told me about Ubuntu and had it running on his laptop. I quickly put together a rig from old pieces and put 8.04 on it. For the next year or so I played around with it trying to get use to it.  Yes it has a learning curve, but so did Windows 95 when I bought my first PC.  Finally I took the plunge and installed I think 10.4 on my main machine in dual boot mode. I found myself using Ubuntu more and more. In fact my wife started to use it because she did not want to reboot into windows. As of today all of my machines have dual boot, however they are always in Ubuntu.  Most of them have WinXp on them and it's there just because it was when I installed Ubuntu. The newest version of windows I own and my last is Win7 on a computer that I built to game with and just can't seem to game any more. But it too has Ubuntu and it is set by default to boot first.  Yes I love Ubuntu and Linux in general.  I think that it is amazing considering the fact that it is FOSS. 
I highly doubt that I will ever use anything else from now on.  Since I build all of my machines thats  good cause I am not paying for a windows.  I may stop building them when I can walk into a retailer and get a linux loaded machine or one that is completely blank.
