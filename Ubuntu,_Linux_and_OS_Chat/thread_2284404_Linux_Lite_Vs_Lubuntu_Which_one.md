### Post by sudodus on 2015-06-29
Are you using the free linux graphics driver, that is used automatically, or have you installed an nvidia proprietary driver (for linux, you can't use drivers made for Windows)? I think that most nvidia cards need a proprietary driver to work at full capacity (sometimes even to work at all).

As I wrote before, please try different distros live before you decide what to use as installed. And after installation you can install an nvidia proprietary driver.
