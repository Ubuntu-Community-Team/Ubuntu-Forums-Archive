### Post by The Cog on 2014-06-14
Welcome to the forums, and to Linux.

Programming on Linux is much the same as programming on windows or mac. I guess the first thing is to decide which language might best help you solve your problem. That will help you decide whether you need a plain text editor or a fancy development environment. The sticky threads at the top of the programming sub-forum here could be a good start, especially this one: [http://ubuntuforums.org/showthread.php?t=1766253](http://ubuntuforums.org/showthread.php?t=1766253)

The sticky threads at the top of the Absolute Beginners sub-forum are a good place to start looking for general how-it-works info. 

Also, just hanging around these forums and reading threads that catch your interest when they appear is a good way to get a feel of things.

I'll suggest the OP dabble in a variety of languages before settling down with one.  Languages vary widely in style, syntax and expressive power (do you need 100 lines or 10 lines to implement your idea?). Different people will have affinities for different languages.

A simple text editor is fine while you are working with a few files per effort. An IDE is intended to help organize complex projects with many files.

### Post by cariboo on 2014-06-17
I'm not sure why you think you need to learn programming in order to use a Linux distribution, 99% of what you need to do can be done with a GUI interface. For that 1% you may need to use a terminal, but there is no programming needed.

My suggestion would be to set your self a goal, and work towards it. For instance if you want to set up file sharing, and you want to do it the hard way, install Samba, and manually set up shares, by the time you're done, you should have learned several different commands and used a few programs you've never used.
