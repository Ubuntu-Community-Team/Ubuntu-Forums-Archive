### Post by chopra on 2014-12-16
> **haplorrhine said:**
> I hate to type in a complicated password just to realize that I typed it into the wrong user.  Password entries should stick as you navigate the users.  Perhaps the best setup wouldn't even require navigation, but would instead determine which user you're accessing just by the password you enter.

You can set things up so that you have to type in the username first, then the password.  

If someone is hacking into the system, and they can type a password in only once and apply it to multiple users, that would greatly decrease the time it took to guess passwords for all users.

Besides, what if two users had the same password?  Not likely if they choose good ones, but the machine would still need to have a way of dealing with it.
