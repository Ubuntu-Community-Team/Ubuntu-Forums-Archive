### Post by Tar_Ni on 2014-12-18
I don't believe Linux will work on it at this point, at least not in a friendly manner. The reason for this seems to be that no Linux distributions (including Ubuntu) support EFI for their 32 bit spins.

Here's a very helpful comment on Amazon:

*''FINAL UPDATE:  I found the problem.  The Intel® Bay Trail-T Quad Core  Z3735 1.33 GHz Processor requires a 32-bit EFI bootloader.  No version  of Linux currently supports EFI for their 32-bit spins.  Ubuntu has  stated a refusal to address the issue, as they see systems with these  chips historically being a small percentage of the market.  Now that a  $199 EeeBook is hitting the market, perhaps they will change their  stance.  But, for the time being, the only way to get Linux on these  machines is to use work-arounds involving dropping in a special file in  the EFI folder of the bootable media, and results have been mixed.  I  have seen a post where one guy got his Zotac Z-Box with the same CPU to  take Ubuntu successfully.  So, it is possible, just not easy.  I'm going  to keep my unit and when I get time I'll perform similar tests.  If  anyone is already meeting with success, please post your steps so I  don't have to piece this together from a number of sites speaking to  different models carrying the same chip.*''

Link: [http://www.amazon.com/ASUS-X205TA-DH01-11-6-inch-Laptop-Dark/product-reviews/B00OBA5AZU](http://www.amazon.com/ASUS-X205TA-DH01-11-6-inch-Laptop-Dark/product-reviews/B00OBA5AZU)

So unless you are a tech-savvy guy who can manage to install Linux in a very unconventional way, I think you have 2 options here:

 A. Looking for something else.
B. Buying the laptop and using Windows for the time being. [URL="http://www.amazon.com/ASUS-X205TA-DH01-11-6-inch-Laptop-Dark/product-reviews/B00OBA5AZU"]
