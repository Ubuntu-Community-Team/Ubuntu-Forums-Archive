### Post by verymadpip on 2015-02-06
Hi there.
i don't think you need them all.  Only the first DVD or CD for a working basic install. 
*If I recall correctly* a bunch of the software you can install yourself afterwards, if you have an internet connection, comes on some of the DVDs.
If you look at the CD images there are up to a 12 of them in a "complete" set.

From the Debian installation guide:
```
As CDs have a rather limited capacity by today's standards, 
not all graphical desktop environments are installable with only the first CD; 
for some desktop environments a CD installation requires either
network connectivity during the installation to download the remaining files 
or additional CDs
```
(I would imagine that this is not the case with a DVD).

I believe there are also live versions of "spins" available with various desktops.

### Post by buzzingrobot on 2015-02-07
The 4-DVD set contians most, if not all, the packages in the Debian repositories.  You certainly do not need it to do an install.

The most common way to install Debian is likely with the netinstall image, as others have mentioned. 

Debian has a number of other installation images, so be sure to read what the Debian site says about them.

Also, be sure to read what's said about firmware.
