### Post by TheFu on 2019-07-04
I have an Amazon Fire 8 gen7 tablet. It was $50.  Many things about it suck, but after you use adb to stop the ads and remove a few programs, then side-load some application using F-Droid store, it is fine.

For reading epub books, I use Aldiko.  Obviously, the device is vastly connected to all of Amazon, but that can be minimized if you are willing to hack a little.

If heard that people load up the google play store and load apps that way.  I'm a google -hater, so not having google-anything on the tablet is something I see as a plus.  I self-host most things, so besides Calibre, there's Nextcloud with about 10 addons, wallabag, library access tools like Overdrive/RBDigital, a few games and local media players for DLNA conections and local streaming using Clementine.  Also have openvpn client loaded to access everything at home when I'm away.  I'm in the process of switching over to Wireguard for the VPN - we'll see how that goes. 

The Gen7 Fire8 just became software-root-able this year, so I've done that and put in a huge ad-blocking & pro-privacy /etc/hosts file.  Haven't done anything else that needs root.

I've owned a 10in tablet previously. Too large for my needs and it was $350+.  A friend had a 7in tablet that I borrowed. Too small but my eyesight is getting worse the last few years.  The diff between 7in and 8in makes all the different to me.

I've used the camera and video recording for trivial stuff. Not great, but good enough for accident photos.
It doesn't have a GPS, so no navigation.  I really miss that. If it had a GPS chip, I wouldn't need to have a cell phone for overseas travel.
It plays h.264 videos well enough. It doesn't handle hidef mpeg2 or h.265 videos. Perhaps newer generations do/will?  A plex server that can transcode effectively solves that issue.  For non-hidef video, VLC can play almost anything.

If mine breaks tomorrow, I'd buy another.  At $50, it is a good value to me.
