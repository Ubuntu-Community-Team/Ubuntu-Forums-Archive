### Post by Roasted on 2013-07-12
Lately I've found myself growing increasingly bored of the mobile space. This is largely due to the fact that you don't really have any degree of flexibility with crossing other operating systems or environments with existing hardware. Sure, you can flash and install Cyanogenmod on your Android, but Cyanogenmod is still Android, so you're just getting a different Android. You're still largely stuck with Android-related OS's on Android devices, Windows on Windows devices, etc.

I'm a huge fan of many desktop environments. I love what's happening these days. While I'm not a KDE fan, Plasma Active is an interesting concept. Gnome Shell is downright gorgeous and Unity is something we're all aware of in regard to Ubuntu Touch. As a result, this got me wondering. I can easily install KDE on Ubuntu, Gnome Shell on Ubuntu, etc etc etc as it stands now. Of course, I can also install the DE-specific spins (Kubuntu, Ubuntu GNOME), but that's something we'll ignore for right now. What I think would be downright awesome is the ability to install some sort of alternative desktop environment to the Ubuntu Touch base. I would love to see Gnome Shell light up on my tablet, as an example. I'm sure KDE fans out there would love an easy way to fire up Plasma Active in the same regard as well.

Has anybody read anything about this? Everything I hear is all about Ubuntu and the Unity-based convergence, which is all well and good, but my prime interest is not Unity. If something like this would happen, it would truly shake up the mobile market in a way we Linux users are currently spoiled with in regard to our computers. We can install any distro we want with any desktop environment we want. Having THAT capability on a mobile device would be tremendous.

I understand a lot of this is trying to pry into the depths of the future, but in the event I missed some sort of news announcement or some user report that "hey this is possible", I figured I'd post about it to find out, or, if nothing else, generate some conversation.

Thoughts?

### Post by grahammechanical on 2013-07-12
The devs of Kubuntu and Lubuntu have already decided not to use MIR on their desktop versions. So, forget about those two because Ubuntu touch device will be running on MIR. Any distribution that does not support MIR will not work. Canonical is just not working to get Xserver as is presently known working on those devices.

[http://www.markshuttleworth.com/](http://www.markshuttleworth.com/)

There has been a lot of hateful words spat at Ubuntu and Canonical since Unity appeared on the the desktop and it continues with the news of MIR. I really do not see the developers of other distributions swallowing their pride and using something that Canoncial has developed.

Nor, do I think that the DEs that you describe will work very well on mobile touch devices. 

Regards.
