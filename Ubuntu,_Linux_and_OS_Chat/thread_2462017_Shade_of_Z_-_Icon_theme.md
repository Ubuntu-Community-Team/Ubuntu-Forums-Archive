### Post by Perfect Storm on 2021-05-11
Shade of Z is an icon set for Zorin OS (16). It aims to streamline the icons in the Zorin envoriment, so application icons are in match with the panel icons (menu launcher, systray etc.). Shade of Z is based on Papirus icon theme.

[IMG]https://cdn.pling.com/img/f/2/1/f/bf7680269424f44d62379aefa7bfd824f6eac870d799a08aa2b5f577f5087c00b992.png[/IMG]

[video=youtube;8FM0g3Nt8HU]https://www.youtube.com/watch?v=8FM0g3Nt8HU[/video]



I'll take requests for which app icons I should put in next. If I get no requests I take (almost) randomly from the long list of apps and put it in as I go. So don't hesitate with the list of apps you think I should put in. 

Shade of Z comes in different colors which match the color settings of Zorin OS.

[IMG]https://cdn.pling.com/img/f/d/a/4/783681855f995d1526c408d66046fdd592cd49b473e0114e21  0db67b5e245e97b412.gif[/IMG]

Download: [https://www.gnome-look.org/s/Gnome/p/1516492/](https://www.gnome-look.org/s/Gnome/p/1516492/)
