### Post by brian62 on 2016-12-12
A month or so ago I discovered I couldn't play PlanetSide 2 on Wine anymore because DayBreak changed their terms of use. Around the same time, Paladins did the same thing. Two games that formally ran pretty OK on Wine suddenly didn't work at all.

I've been working with virtual machines and Vagrant at work alot lately so I decided to install Windows 10 inside VMware Workstation Player. Turns out, it works. VMware supports up to DirectX 10, too.

Not every game works well but both PlanetSide 2 and Paladins run pretty well and are definitely playable.

I did a few videos on my channel talking about if you are interested in how it works.
