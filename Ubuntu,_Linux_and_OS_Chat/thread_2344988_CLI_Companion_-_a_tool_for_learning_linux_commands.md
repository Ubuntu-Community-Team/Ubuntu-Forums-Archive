CLI Companion is a tool to store and run Terminal commands from a GUI. 
People unfamiliar with the Terminal will find CLI Companion a useful way to become acquainted with the Terminal and unlock its potential.
 Experienced users can use CLI Companion to store their extensive list of commands in a searchable list.


It have been downloaded by more than 5000 people so far. We have recently released a new version - 1.3.
