### Post by Shibblet on 2022-02-17
I have been a Kubuntu user since 9.04...  Back when Plasma was still in the 4's.

To be honest, I don't really know what attracted me to KDE over Gnome 2 at the time.  I watched Gnome 2 get phased out for Unity, and then Gnome 3 was on the rise... and I never really enjoyed those interfaces over KDE.

I haven't really given Gnome 40 a try, but it looks a lot like Gnome 3, so I don't think I am really missing anything.

KDE has always felt to me like a UI that stays out of the way when I have things to do, and is quickly there when I have things I want to change.

That being said... if you're not used to KDE, it takes some time to get used to.  I feel the same way with Gnome and other UI's as well.
But, if you don't like it... then you don't like it, and you can choose another UI.  That's the beauty of Linux.  Don't like Gnome, use KDE.  Don't like KDE, use XFCE.  Don't like XFCE, use Enlightenment... etc. etc. etc.
