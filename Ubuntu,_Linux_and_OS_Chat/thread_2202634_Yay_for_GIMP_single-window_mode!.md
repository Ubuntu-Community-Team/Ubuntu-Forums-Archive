### Post by Don_Stahl on 2014-01-30
In Ubuntu 12.04 I've had quirky problems activating the various windows in the GIMP -- sometimes one of the menus keeps popping up whenever I try to go to the main menu bar, for instance. (No, I don't want to apply hue-saturation-color again, I'm done with that!) I think part of the problem is Cairo-Dock taking focus instead of the GIMP. The problem is not a major one, it's always resolvable with a little fiddling, but it was a nag.

Well, I just installed GIMP 2.8 and I think single-window mode will take care of it. So -- kudos to the GIMP team for a major interface rebuild.
