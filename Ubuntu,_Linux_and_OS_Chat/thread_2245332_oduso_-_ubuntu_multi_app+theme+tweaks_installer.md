Hi everyone and I present you lorenz and Mash's latest project - oduso! (Tadaaaah) oduso is a tool to quickly install multiple apps, themes and tweaks effortlessly. Just select the apps that you would like to install and paste the generated command into the terminal and oduso will take care of the rest.[/COLOR]
[COLOR=#666666]
[SIZE=5][B]Why use oduso?
[/B][/SIZE][/COLOR]
[COLOR=#666666]Ever wished that you could install multiple apps with a single command &#8211; batch install apps? Or that you didn&#8217;t have to look up ppa for that random app? Happens to me a lot and I am prepared to bet you are bothered by that too! This is where oduso comes in. With oduso, you can batch install apps, even the one from third party ppa&#8217;s without looking them one by one. We also have icon themes, gtk themes, tweaks, top fixes &#8211; all in one places &#8211; everything ready to be installed on your tap.[/COLOR]
[SIZE=5][B][COLOR=#666666]
We are just getting started
[/COLOR][/B][/SIZE]
[COLOR=#666666]During the period of beta 1, oduso will support Ubuntu 12.04, Ubuntu 14.04, elementary OS Luna, elementary OS Freya. As we progress, we plan to support a wide spectrum of distros. Also the number of apps and themes are limited &#8211; for now. We wanted to build a solid and self sustaining platform before pouring down all these apps and themes. Again, with time &#8211; we hope we&#8217;d have a library with a seizable numbers of apps.
[/COLOR]
