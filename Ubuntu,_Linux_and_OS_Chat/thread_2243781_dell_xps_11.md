### Post by Tar_Ni on 2014-09-11
Since Dell is one of Canonical's partners, there is a lot of collobaration to ensure that a wide range of Dell's hardware works with Ubuntu.

Live boot mode let's you try Ubuntu without installing it on the machine. So when you boot the USB drive on the XPS 11, you will be prompted either to ''try Ubuntu'' or ''install now''. Trying Ubuntu will allow you to test if everything works as it should. Wifi, touchscreen, touchpad ect.

For more information on this: [http://www.computerworld.com/article/2519158/linux/5-ways-to-use-bootable-linux-live-discs.html](http://www.computerworld.com/article/2519158/linux/5-ways-to-use-bootable-linux-live-discs.html)

There is really no reason why it shouldn't work well with the latest Ubuntu 14.04. The XPS 11 comes with high end Intel hardware. Intel has put a lot of efforts to open-source it's drivers so it's a no brainer to use on Linux. See: [http://www.tuxradar.com/content/interview-kaveh-nasri](http://www.tuxradar.com/content/interview-kaveh-nasri)
