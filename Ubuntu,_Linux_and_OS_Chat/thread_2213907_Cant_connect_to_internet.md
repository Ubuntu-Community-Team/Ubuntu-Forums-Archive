### Post by PondPuppy on 2014-03-30
For myself, Joel, I'm uncertain how the network you connect to is set up. 

As I recall, when using a ethernet cable connection, I had no setup required in Ubuntu or in Windows 7. The connection to the router was auto-detected. However, I don't know how your network is configured.

In Ubuntu, press the "super" key -- the "windows" key. You'll see the Dash system search box open. In the search field, enter the word "network". That should display some graphical tools for configuring and troubleshooting network connections.

If you're connecting to a corporate network, perhaps your system administrator can provide information. Or your internet service provider might be able to help.
