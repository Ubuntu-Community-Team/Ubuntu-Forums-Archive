### Post by slooksterpsv on 2013-10-22
Every time I've had a computer I've always used Windows and Ubuntu side by side. My last favorite version of Ubuntu was 10.10 when Gnome 2 was still Gnome 2. I was always on AMD hardware - which I love AMD still don't get me wrong. Well I recently got the following computer (for very cheap too):

Asus X502CA
Intel Core i3 3217U
4GB RAM
15.6" Screen
No DVD drive (Ultra-book I believe)
500GB HDD
Hot Pink/Chrome Red like color
All for $265.xx from BestBuy

Well I used Windows and found a lot of my games don't work or run too slowly due to the Intel HD 4000 GPU. My Samsung does better with the AMD Radeon 7520G. I migrated all my info to this new computer, used Windows until 8.1 was released - upgraded to 8.1; and said I miss Linux. I miss Ubuntu. I tried Ubuntu to make sure my hardware all worked (which it did flawlessly) and then download Ubuntu Gnome - as I love Gnome 3.

Well I'm using Ubuntu Gnome 13.10 - I have all my data on it, the games I do play or played in Windows work better in UG than Windows. I feel more productive,  I feel I have the power of my computer back and I even use my secondary monitor to help organize my stuff.

I don't think I've ever been as pleased with Ubuntu as I am having Ubuntu Gnome. It feels right, it's so fluid, and where I'm not on Windows anymore, I feel more compelled to help the community even more.

What are your guys' takes on Ubuntu Gnome 13.10?
