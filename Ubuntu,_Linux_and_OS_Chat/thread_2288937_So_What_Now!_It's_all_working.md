### Post by grahammechanical on 2015-07-31
What now?

Now you mess around so much that you break the OS! Or you just get on with using Ubuntu for what it is - a computer operating system. My advice would be to:

1) Learn how to partition
2) Move all your important data and documents on to its own partition
3) Dual boot with a second install of Ubuntu
4) Do all your experiments in the second installation
5) And in the process learn how to re-install without losing all your important data.

And may be give something back by testing the Ubuntu family of Linux distributions and giving support on this forum.

[https://wiki.ubuntu.com/QATeam](https://wiki.ubuntu.com/QATeam)

Regards

### Post by Mike_Walsh on 2015-08-17
I'll go along with everything that's been said so far.....especially the bit about learning backups, and just getting the hang of re-installing, etc., when you break it (which you **will**, trust me.....we've **all** been there!)

If you're ready for a bit more excitement after that, have a look at some of the other mainstream distros; Debian itself, Elementary, PCLinuxOS, Fedora, CrunchBang..... Stick a few extra partitions on your drive, and play around a bit.

If, after that, you're still not sated, have a look at some of the more esoteric stuff out there. Explore the world of 'micro-distros'; DSL, Tiny Core, SliTaz....even Kolibri, if you want a real laugh (it's all written, 100%, in machine code.....with a big pair of silly eyes on the desktop, that follow your cursor everywhere it goes; great fun!) Experiment with running these tiny OS's from a flash drive, so you can take them with you, everywhere you go.

In between these, you have the lightweight distros; Bodhi Linux & Puppy Linux spring to mind, here.

And **then**, if you **still** want more (**and **you're feeling masochistic, into the bargain), why not take a look at Gentoo? The whole thing has to be built from scratch (it's all right there, though, in the repos); from what I hear, if you ever get it up & running, exactly the way you want it, you'll have have a system you can make sit up and beg.....plus you'll have the satisfaction of knowing that you did it all yourself.

You haven't even scratched the surface, yet..! :lol:


Regards,

Mike.

### Post by coldraven on 2015-08-18
1) If your system has the resources install VirtualBox and run Ubuntu server in it. Keep a text file record of the changes you make. You can learn how SSH  in to the virtual server and install Apache etc. When you break it you can roll back to a previous snapshot.
Ready made VirtualBox images here:
[http://virtualboxes.org/images/](http://virtualboxes.org/images/)

2) Buy a Raspberry Pi 2, then play with the various Debian based operating systems. Raspbian or perhaps OpenElec with the Kodi media centre. It is so easy to burn the OS image to a micro SD card. Just right click the .img file and select Image Writer.
[https://www.raspberrypi.org/downloads/](https://www.raspberrypi.org/downloads/)

Have fun :)
