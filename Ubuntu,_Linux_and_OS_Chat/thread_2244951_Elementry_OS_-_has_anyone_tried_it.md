### Post by buzzingrobot on 2014-09-20
Elementary is a bit more than the usual Ubuntu fork, which typically changes little except the desktop interface.

Elementary uses a desktop interface of its own design and creation, as well as its own window manager, file manager, panel, etc. The distribution also includes a number of applications created by the Elementary team.

Elementary does not really look much like OS X beyond use of a panel and a dock, shared by any number of other interfaces. It does, like Apple, have a commitment to the value of design and adherence to design standards.  This will please users who like the results and annoy those who do not. (Elementary is not one of those grabbag distros with near-infinite configurations possible.)

A new release based on 14.04 is currently in beta. The Elementary team isn't huge, so releases happen when they happen.

It's a solid distro built on a solid Ubuntu base.
