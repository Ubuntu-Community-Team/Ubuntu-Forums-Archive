### Post by PaulW2U on 2016-11-12
While looking for news items for the next issue of the [Ubuntu Weekly Newsletter]("https://wiki.ubuntu.com/UbuntuWeeklyNewsletter") I came across [Ubuntu Budgie Is Now an Official Ubuntu Flavor]("http://www.omgubuntu.co.uk/2016/11/ubuntu-budgie-now-official-ubuntu-flavor").

I'm very surprised that there was no official announcement from either Canonical or the Ubuntu community but I seem to remember that was the case with Ubuntu MATE in 2015 too.

The Ubuntu family continues to grow. So who's next?

Edit: Re-reading the IRC conversation it seems that the recognition of Ubuntu Budgie depends on their developers gaining upload rights to the official archives. There's still some work to be done rebranding from budge-remix to Ubuntu Budgie and daily ISOs will no doubt come in time.

The **ubuntu-budgie-desktop** meta package is now in the 17.04 archive along with several other budgie specific packages that are required for the flavour to be recognised as such. The package description for the meta package reads:
> Ubuntu Budgie is an Ubuntu flavor that offers a desktop
 experience using the Budgie-Desktop. This package depends on all of the
 core Ubuntu Budgie packages.

Daily ISOs, for 17.04, available for testing later this month according to the articles I linked to.
