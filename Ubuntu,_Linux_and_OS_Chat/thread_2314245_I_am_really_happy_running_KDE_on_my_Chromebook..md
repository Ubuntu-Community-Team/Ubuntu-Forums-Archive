### Post by irv on 2016-02-19
I was amazed at what you can do with a Chromebook. First if you are looking for a cheap way to get a nice light weight, long lasting battery that will run both Chrome OS and another Linux based OS and run them very fast the Chromebook will do all of this.
I took a $100 Chromebook and turn it into an all around machine that is now running Kubuntu with the KDE desktop and I am able to switch between the Chrome OS and KDE with a hotkey. Both run very fast for sharing the memory. I installed a 64 gig SD card so I have more than enough storage. I installed all the programs I use on my laptop and they all run great. Here is a screen shot running installs from the terminal.
[ATTACH=CONFIG]267382[/ATTACH]
 I am able to edit graphics, do word processing surf the web, send emails, and do anything you can do on a $1000 laptop running an expensive OS.
This is going to be my travel with me unit from now on. The only thing this thing won't do is run Android apps, but I can do that on my phone.
I did try running unity desktop on it, and it runs okay, but for some reason I like the KDE desktop on this smaller screen. (11.6") And by the way, this is not a touch screen so i don't really need Unity. I find I have to turn off the touch screen on my laptop because it interferes with my typing. The focus changes once in awhile. I tune it off using the command:  > xinput set-prop 12 'Device Enabled' 0

You can get these Chromebook if you look around. I found this one at CowBoom. [http://www.cowboom.com/product/1581125/]("http://www.cowboom.com/product/1581125/")
I had to buy a $15 power supply for it, and a $20 SD Card, but over all that was a cheap price for everything this thing will do.

I used Crouton to do the Linux install. Here is the Howto: [https://www.linux.com/learn/tutorials/795730-how-to-easily-install-ubuntu-on-chromebook-with-crouton]("https://www.linux.com/learn/tutorials/795730-how-to-easily-install-ubuntu-on-chromebook-with-crouton")

There are ways to install the full Ubuntu and remove Chrome OS but I like to use Chrome for all my Internet stuff and use Ubuntu for all my on computer stuff I do. I think I have the best of both worlds.

Here is hoping others will pickup on the idea. It would be great to see a bunch of these being used in a coffee shop near you.
Have a great Ubuntu day.
