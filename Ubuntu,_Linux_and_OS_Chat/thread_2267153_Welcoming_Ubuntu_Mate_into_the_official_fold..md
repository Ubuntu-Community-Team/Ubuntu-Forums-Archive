Ubuntu Mate is, as many will know, now an official flavour. As such we've added it to the main prefix choice and added it to the Main Support areas.

Well done to all concerned I think - [https://ubuntu-mate.org/blog/ubuntu-mate-vivid-beta1/](https://ubuntu-mate.org/blog/ubuntu-mate-vivid-beta1/)

On a purely personal note it was good fun helping Martin Wimpress getting Mate reporting on the image tracker for the recent Beta. 

I'm amazed that they managed to go from not being on the tracker when I started the ball rolling on Tuesday with Iain Lane, to being in a position to mark their images ready for release 2 days later.
