### Post by sudodus on 2014-11-27
I think that you are referring to the next version of Ubuntu, the version being developed, but not yet released.

The current long time support versions are 12.04.5 LTS and 14.04.1 LTS. The version number consists of two digits for the year, dot, two digits for the month, dot, and for LTS releases also one digit 'the point release number'. Standard Ubuntu and I think also Kubuntu have 5 years support, Xubuntu has 3 years support and (only) Lubuntu 14.04.1 has 3 years support. I'm not sure about the other flavours. The other versions are supported for only 9 months. See this link

[https://wiki.ubuntu.com/Releases](https://wiki.ubuntu.com/Releases)

If you want to participate in testing and discussing, you are very welcome :-) and you can check these links

### Post by grahammechanical on 2014-11-27
Ubuntu for developers or Ubuntu loves developers, as one Canonical engineer calls it, is an attempt to make Ubuntu the favourite OS for developers to use.

[http://blog.didrocks.fr/](http://blog.didrocks.fr/)

The basic idea is to provide, or at least make it easier to install, the software that application developers need. Ubuntu already has a Software Development Kit (SDK) to help developers to write apps for Ubuntu on the phone/tablet and to make it easy to package those apps and get them approved for inclusion in the Ubuntu phone/tablet app store.

Now, there is the Ubuntu Developer Tools Centre (expect the name to be changed to something else). This will easily install everything an Android developer needs to write Android apps on Ubuntu. That is the intention, anyway.

Another idea is to have a component store where developers can put snippets of code that other (would be) developers could use. This is still an idea that is being discussed. It seems there will be two categories in the component store, certified code that can be trusted not to do nasty things and use at your own risk code that has not yet been certified but maybe so.

[http://summit.ubuntu.com/uos-1411/meeting/22331/ubuntu-developer-tools-community-input/](http://summit.ubuntu.com/uos-1411/meeting/22331/ubuntu-developer-tools-community-input/)

[http://summit.ubuntu.com/uos-1411/meeting/22356/sdk-tools-updates-planning-and-feedback/](http://summit.ubuntu.com/uos-1411/meeting/22356/sdk-tools-updates-planning-and-feedback/)

[http://summit.ubuntu.com/uos-1411/meeting/22338/ubuntu-touch-component-store/](http://summit.ubuntu.com/uos-1411/meeting/22338/ubuntu-touch-component-store/)
