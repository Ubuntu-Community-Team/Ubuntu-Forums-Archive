I'd consider this to be one of the most turbulent, experimental, and progressive periods for Linux in years. The Unity/GNOME 3 controversy has just settled down, Steam/mainstream gaming is breaking into Linux, open source graphics drivers are evolving rapidly (on par with binary blobs/Windows in some cases), SteamOS has been announced, and soon we'll be switching to Wayland/Mir.

I don't see how this is a dull and boring period. Look around some non-Ubuntu-specific Linux news sites to see what I mean.
