### Post by grahammechanical on 2013-11-20
Old news!

It was never intended for MIr to be on Ubuntu 14.04. The goal was for MIr on 14.10. There is confusion between what is Mir and what is Xmir. It was intended for Xmir to replace Xserver by default in 13.10 and so on into 14.04. But that seems to have been dropped. The big issue was with proprietary drivers not being compatible with Xmir. In my experience Nouveau worked well on Xmir.

The goal is convergence. Mir+Unity 8 is running on version 1,0 of the Ubuntu phone platform which needs to be optimised for the tablet and this is the focus for the present development cycle. By the 14.04 release date it is intended that the Ubuntu phone/tablet platform be at version 1.5. Then attention will be moved to converging the code base so that there is one code base for the desktop, phone, tablet and TV.

So, by 14.10 or 15.04 expect the Ubuntu desktop to be running Mir (full MIr) + Unity 8 (which is a different code base to Unity 7) and expect the phone/tablet core apps to also be available on the desktop.

Ubuntu is open source. It is free to download. This cannot be changed. How can you liken Canonical to Microsoft?
