### Post by DuckHook on 2017-02-21
According to the following non-authoritative source, Whatsie has been abandoned and will not be developed further unless someone else picks up the ball. It is still hosted on Github but will have no further updates. This is the final version: [https://github.com/gsantner/whatsie](https://github.com/gsantner/whatsie)

Security-wise it is not a good idea to run abandoned apps, but you are probably already aware of that.
