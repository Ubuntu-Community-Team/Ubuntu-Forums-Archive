### Post by vtpoet2 on 2014-12-20
I finally got around to finding a [very inexpensive Windows system]("http://www.staples.com/Acer-AXC-603-UR10-Intel-Pentium-4-GB-DDR3-1TB-SATA-Hard/product_1268144?externalize=certona#desc_content"), needed it for IPOD connectivity, Taxes, and my wife's occasional MS Word compatibility issues (Libre still isn't there). It's the first time I've used Windows 8.x. I've been using Linux, exclusively, for almost 2 years now.

What's so interesting is to come back to Windows, comparing it to when I originally *left *Windows. Now, I'm mildly concerned by what *_Windows_* can't do, as compared to Linux (the reverse of my earlier experience). Despite all the negative reviews of Window 8.x, I was nevertheless expecting greater polish that that of a Linux DE (KDE in my case, Unity & XFCE). Interestingly, while there have been little refinements added (like the graphical copying feature), if Windows were a Linux DE, I would actually be more apt to compare it to XFCE, LXDE or even DEEPIN. What really struck me was that when I changed the colors of the titlebar, the font color remained black. That is, if one makes the titlebar black, any text becomes unreadable. I can't imagine any Linux DE would tolerate this _Bug_ for long. 8.1 feels like it's a decade behind KDE. There's no comparison. None. I understand that Windows 10 may introduce desktop activities and become much more KDE-like, but as for now I'm really surprised by how limited and chintzy Windows feels!

I wonder what using an Apple would feel like? I quit using Apples around 2000 -- when it was obvious that deliberate, hardware obsolescence was part of their empire-building strategy.
