### Post by grahammechanical on 2015-08-17
As a speculation, consider this.

Ubuntu 16.04 may be the last Ubuntu desktop version using deb packaging. It does have a support life of 5 years. There may not be a need to go beyond Z code names. The letters X, Y, Z can be reserved for LTS versions which takes us beyond 20.04, In the mean time there should be coming to the desktop Ubuntu Personal or Snappy Ubuntu Desktop Next as it is also known. With transactional updates it never needs a version release. It could be like the Ubuntu phone OS getting numbered Over the Air (OTA) updates. It is already being built on Wily Werewolf (15.10) code.

Anything can happen in the next 2 years. And before you say: "Wot about the flavours?" They do not need a released Ubuntu version. All they need is the Ubuntu archives containing deb packages. And those archives are not going away. They are the basis for building every version of Ubuntu including the Snappy versions.
