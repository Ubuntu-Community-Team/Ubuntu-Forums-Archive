### Post by TheFu on 2015-04-25
I think this probably belongs in either reoccurring questions or the Cafe ...

There are things that W7 is better at than any Linux - but that depends on your definitely of "better" and which programs you need/want to run.  For example, there are some stock market analysis programs that only run under Windows. They will not run under WINE.  Don't forget video editing and many video capture hardware only has drivers for Windows. What about gaming?  If your favorite game only runs on Windows, then you'll probably run Windows.

If you don't have any software that absolutely requires W7/W8/W10/W.... then certainly I'd wipe that disk and let Linux have a chance.

Ubuntu is not all the options in the Linux world either. There are 50 other flavors - each has a place based on what you think is "best."  For example, I won't run Unity or Gnome3. I dislike the bloat and dislike the interfaces.  There are "better" options as far as I'm concerned. OTOH, I like Ubuntu Server though it is a little bloated too. The Ubuntu-lite server distro is just missing too many things that I want pre-loaded. Sure, I could load those packages manually (or via an ansible script), but disk is fairly inexpensive and the hassle-factor just isn't there for me yet.  

There are other distros for different purposes - Mint, Debian, Fedora, TinyCore, Kubuntu, Lubuntu, Xubuntu, Mate-buntu, CentOS, and many others.  For online banking, I use TinyCore Plus. It has a purpose - get me on a network and have a web browser that all my financial institutions work with.

Is normal Ubuntu Desktop lighter than W7?  I don't think it is.  Is xubuntu/lubuntu lighter than either of those - DEFINITELY!  Of course, those distros don't look as "pretty" and usually don't include the fancy GUI effects that wow other people.

I think Lubuntu is bloated, so I take Ubuntu Server and add openbox, then the 20 programs I want. Some would disagree that my technique doesn't provide enough GUI (it doesn't have any menus) to be useful.  Only you can decide the best solution for your needs.  At least with Linux, you have many, many, options.

Lastly, the OS is NOT the GUI.  You can load almost any Linux distro and then elect to swap a different GUI in - takes about 3 minutes for most broadband connections. There are caveats, but it works.
