You can easily install unity on 18.04 if you love it.

```
sudo apt install ubuntu-unity-desktop
```

choose lightdm as default.

if need to use touchpad

```
sudo apt install xserver-xorg-input-synaptics
```

Reboot, that's it, you can uninstall gdm and gnome shell if desired

see also [https://community.ubuntu.com/c/desktop/ubuntu-unity-dev](https://community.ubuntu.com/c/desktop/ubuntu-unity-dev)

I have been using unity on 18.04 since release, it has been great.
