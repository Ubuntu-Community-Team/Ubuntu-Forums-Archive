### Post by brian-mccumber on 2015-09-03
I am fairly new to Ubuntu but I have found that it is fairly easy to change the style of Unity. After a little trial and error I found an easy way to customize the desktop appearance by using the Unity tweak tool that is available from the software center and a website, [Noobslab]("http://www.noobslab.com/p/themes-icons.html"), that uses ppa's to distribute the theme files and they are easy to copy and paste into the terminal to install. I am interested in seeing what other people have come up with in the way of modded desktop appearances. I am currently using Numix theme with Buff-remix icon set. I would love to see some cool desktop styles.
