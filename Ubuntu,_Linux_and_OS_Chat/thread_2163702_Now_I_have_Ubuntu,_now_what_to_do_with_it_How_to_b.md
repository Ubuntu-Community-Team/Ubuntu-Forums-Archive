### Post by montag dp on 2013-07-19
There's no need to do something drastic to prove to your friends that Ubuntu is better.  In fact, there's no reason to do anything at all.  If your friends prefer Windows, let them use Windows.  If you prefer Ubuntu, use Ubuntu.  Remember, it's an operating system, not a religion.

I'm sure you have your reasons for using Linux instead of Windows, but some good ones to bring up if they are mocking you (which doesn't sound like something friends should do, by the way) include:

- No need to worry about viruses, and no need for annoying resource-hogging anti-virus software.
- Less bloated.  On my computer after booting up, Windows uses double or more the RAM that even the heftier Linux DEs use, hurting performance.
- More choice.  On Linux, I can make my desktop behave any way I want.  Don't like how your desktop environment works?  Install a different one.
- Open source.  If you know what you're doing, you can change your operating system (or programs) directly by modifying the source code if you want.
- BASH instead of DOS command line interface.  Some people might not care about that, but for those that do there's no question that BASH is 100x better.
