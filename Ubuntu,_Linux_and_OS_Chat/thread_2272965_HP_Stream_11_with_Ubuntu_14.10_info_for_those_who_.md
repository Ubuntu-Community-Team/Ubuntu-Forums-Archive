### Post by jetstormnation on 2015-04-09
This is going to be a mirror of my [reddit]("https://www.reddit.com/r/Ubuntu/comments/31yy34/hp_stream_11_with_ubuntu_1410/") post on my experience with installing Ubuntu 14.10 on the little HP Stream 11-D010WM.

I didn't have a lot of money but I needed a computer so I did some research and decided to get the HP Stream 11. Since I don't really like using windows if I don't have to, I installed Ubuntu 14.10 on it and so far it's been great. Everything worked immediately upon installation.

First thing I did was boot into windows and create the system restore USB, for which I needed an 8GB USB drive. Also (and this is important!) you MUST update to the latest BIOS or else some things might not work. I then downloaded Ubuntu 14.10 image and used LinuxLive USB creator to put it on a USB drive. As far I as have researched Ubuntu 14.10 is the only distro that will work without extra tweaking. I also tried Lubuntu 14.10, but the trackpad bugged out to the point it was unusable.

Once you restart the computer with the USB drive in place you have to hit the esc key on boot to get into BIOS to change the boot order and restart and install Ubuntu per usual.

Once everything was installed I tried a lot of commonly used software in Ubuntu just to test it and see if anything undesirable would happen and so far so good.

Although, I will point out, if you "push-click" either of the mouse buttons down it will fix the mouse clicking in that location. It's a strange thing to try to explain, but if it happens just restart or log out and you'll be fine. Even though that doesn't work you can still use the "tap-to-click" on the track pad with all of the usual gestures like "two-finger-tap" to right-click, "two-finger-scroll", etc.

After getting everything setup I installed "tlp" and it extended my battery life by about 2 hours. So as I'm writing this, with the screen dimmed to about 25% and battery charge at 83% I'm showing 9 hours and 3 minutes. I've done a few minor tweaks other than that, but I don't think they contribute that much so I won't go into those.

On the whole this little laptop has been a great computer. The processor and RAM are enough that it takes care of all of my general computing needs like email, web browsing, and word processing. When you are watching youtube videos or related media types it will hiccup every once in a while, but never bad enough to make a difference. And while it doesn't have a lot of disk space (about 26GB free after installing Ubuntu by itself) the SD card slot is completely recessed, so it's easily "up-gradable".

I've been running it every day for the past week and I love it so far. It's small, light, and does what I expect it to.

Anyway, I didn't really find a whole lot on the subject when I looked so I figured I'd post some info for people. If you have any questions please feel free to ask.

tldr; Ubuntu 14.10 works on the HP Stream 11 if you have the latest bios and don't dual-boot with windows. Install "tlp" to increase battery life by 2 hours. Don't expect the thing to be a supercomputer...
