About this:

> Maybe I should wait until 16.04.3 LTS

I personally always try the 1st point release first because of [LTS HWE]("https://wiki.ubuntu.com/Kernel/LTSEnablementStack"), if you jump forward to the [14.04 kernel support graph]("https://wiki.ubuntu.com/Kernel/LTSEnablementStack#Kernel.2BAC8-Support.A14.04.x_Ubuntu_Kernel_Support") you'll see that only the OEM Trusty kernel and X-stack are supported for the entire life of Trusty, whereas those who installed using 14.04.2, 14.04.3, or 14.04.4 media will have to perform an HWE upgrade about 4 months after release of the next LTS (16.04).

I use the later point release images for installation only if needed for hardware that was not yet supported when the OEM kernel and X-stack were released. Sometimes it takes a bit of searching to find the archived images. For Ubuntu itself if the images aren't located at [releases.ubuntu.com]("http://releases.ubuntu.com/") they can typically be found at [old-releases.ubuntu.com]("http://old-releases.ubuntu.com/releases/"), and images for official flavors can usually be located from [cdimage.ubuntu.com]("http://cdimage.ubuntu.com/").

AFAIK release upgrades always result in the OEM kernel and X-stack regardless of when during the support cycle the upgrade is performed.
