### Post by monkeybrain20122 on 2014-06-18
I have tested it on Fedora (which I have not installed any flash other than Chrome), it works unbelievably well even at this  early stage! All the sites I have tested work: Youtube, dailymotion,  wimp, crackle,  guardian videos, ted, cnn,  some local News sites that don't  work with flash 11.2, a few foreign sites. Even the audio in google  translate works. Can't find my webcam though. The response to clicks may  be a bit slow at some sites, on crackle the buttons don't work  correctly and it doesn't have hardware acceleration as in Chrome, but  othetrwise it is very smooth (smoother than pipelight flash). It is just pre aplpha according to the  developer but it is already almost fully functional for common use  cases. 

In Fedora one needs to compile the flash wrapper (though a very easy compile) and install Chrome to get pepper flash .But in Ubuntu you can install  the flash wrapper with ppa to get updates and get  pepper-flash from repo without even installing Chrome (see webupd8 link above)
