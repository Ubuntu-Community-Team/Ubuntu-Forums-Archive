Some of us like to keep up with the latest open source software  available. Often the best way to get a steady flow of new software is by  running a rolling release distribution. ### Post by grahammechanical on 2015-06-01
What is meant by "conservative?" In the context of the quote a Ubuntu LTS (long Term Support) release would be conservative. Ubuntu LTS releases are supported for 5 years they do not automatically upgrade to the next six monthly Ubuntu release.

I my opinion Ubuntu is a conservative distribution because the Ubuntu maintainers put stability first. This is the case even with the next version of Ubuntu during its development period. I am running Ubuntu Wily Werewolf (15.10). It is updated every day and like Vivid Vervet before it this system is so stable as to be boring. And yet it is, for Ubuntu, the bleeding edge.

Keep in mind that with Ubuntu it makes no difference if we are a single user or a commercial organisation we all get the same Ubuntu. We do not have a very stable version for businesses and a less stable version for the home user. If the Ubuntu developers were not conservative when it comes to stability then Ubuntu would not be a distribution that businesses could use. And ordinary users would not want it either.
