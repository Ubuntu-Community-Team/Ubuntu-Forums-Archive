### Post by grahammechanical on 2016-08-08
No idea. Besides, Ubuntu already has a "click to install" method of installing applications.

First, we need snap packaged applications that are useful to desktop users. Second, we need the 16.04 Ubuntu Software store to access the snap store because snap apps are not in the usual Ubuntu repositories. We will also need some way to identify snap apps from deb applications. An installed snap will appear in Ubuntu Software but apart from its icon there is nothing to indicate that it is a snap package app.

Up until recently it was possible to see a list of snaps by using the command: snap find. But that command now gives "error: cannot list snaps; empty query." Whatever that means. I think that we now have to use a command to login to the snap store.

The snap man page does provide a command to buy an application for the snap store. That is progress and may be it is in preparation for the move over to Unity 8 but as for Unity 7 there is still no GUI method of installing snap packaged applications.
