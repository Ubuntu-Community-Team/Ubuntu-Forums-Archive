### Post by EmpireITtech on 2014-04-17
First, welcome to the Linux World!

Second, to piggy-back from what pfeiffep said - Ubuntu comes with Firefox builtin, but you'll need to make sure you install the Flash Player (from the Software Center is the best option) to be able to watch any videos on YouTube, ESPN, CNN, whatever. Or you can install Google Chrome which has it's Flash Player built into it (best way is to go to the Google Chrome website and download/run the installer package from there). I wouldn't mess with Chromium (an open-source version of Chrome) bc for me it's been harder to configure and get working right. 
I just know internet surfing is the number one thing people need nowadays =]
