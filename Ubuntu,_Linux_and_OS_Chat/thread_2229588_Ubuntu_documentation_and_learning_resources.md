### Post by LastDino on 2014-06-14
You mean; using documents in Ubuntu home folder with windows or accessing documents in home folder with Ubuntu itself? Windows can't read ext4, so you're unlikely to be able to access Ubuntu home from windows, while latter is quite possible since that is how it is intended to work.

You don't need to know any programming to learn using Ubuntu. If you want to, you can read Wiki and learn commands. You can also read about scripting and using bash.

### Post by coffeecat on 2014-06-14
Welcome to the forum!

You've asked two very different questions and a good rule of thumb for a forum is one thread per problem, but only one thread each. So....

For your first question, I suggest you start another thread in either [General Help]("http://ubuntuforums.org/forumdisplay.php?f=331") or [Absolute Beginners]("http://ubuntuforums.org/forumdisplay.php?f=326"), whichever you feel more comfortable with, asking about sharing folders and files between the two operating systems in a Windows/Ubuntu dual boot. Give it a meaningful title - &#8220;Sharing files in a Windows/Ubuntu dual boot&#8221; would work. To get you started, my preference is to have a separate data partition formatted NTFS for shared files in a dual-boot, but there is a complication with Windows 8 which by default has a hybrid hibernation/shutdown and it erases anything in a shared partition when these were saved from Ubuntu. There is a workaround, but that's best for your new thread. And other people may have alternative suggestions.

But if you have anything to ask about sharing files in a dual boot, please ask in a new thread - this is a chat, not a technical support, area.

Second question - +1 to what LastDino said. I've been using Ubuntu since 2005 and I'm not a programmer. I've renamed your thread to make it more meaningful and to concentrate on getting you suggestions for documentation and learning resources for Ubuntu.

Good luck!
