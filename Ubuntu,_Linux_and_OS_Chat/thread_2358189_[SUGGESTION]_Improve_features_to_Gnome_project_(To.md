### Post by bcbuch on 2017-04-13
As has been stated the app categories in the dash missing is a Gnome team issue. The easiest way to deal with that issue for me is to enable the Applications Menu with Tweak Tool under extensions. Not a huge fan of hot corners and the dash, I install the Activities Configurator extension. I use it to disable the hot corner, remove the rounded corners from the top bar, remove Activities, and make the top bar transparent to my taste. I enable the Applications Menus extension which has a Activities Overview link at the bottom. This all gives me a more traditional experience, but still have access to Activities Overview to set favorites and such. For organizing the Applications menu I install Alacarte, and edit to my taste. I realize that this isn't the optimal solution for those wanting to use minimal extensions, but once set it works the best for my computing needs.
