### Post by Dry Lips on 2013-06-02
Opera isn't in the repos either, but I see no reason to complain about it. Both Chrome and Opera is very easy to install on Ubuntu if you want to use them. Now, regarding the tracking and privacy concerns... As a person who is very concerned about privacy, there certainly are some features in Chrome that worries me. [http://en.wikipedia.org/wiki/Google_Chrome#User_tracking](http://en.wikipedia.org/wiki/Google_Chrome#User_tracking)  For instance, when using the omnibar you are talking to Google's servers, although the data apparently is anomynized. If you do, then Chrome isn't worse than Google, and if you don't, then Chromium certainly is a brilliant alternative.
