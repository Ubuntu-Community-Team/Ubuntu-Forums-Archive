### Post by TheFu on 2020-08-25
A few ideas:
[LIST]
[*]Teach other people. Consider starting a LUG - local, University, or virtually, online.
[*]Create How-Tos. Post to different location depending on the skill level needed.  Giving a Bible to people in the church already isn't as useful as giving one to someone not near the church.
[*]Post blog articles about stuff you do with Linux.
[*]Create a python replacement for all the most popular php webapps. Publish it.
[*]Create a python tool for something you wish existed. Publish it.
[*]Give presentations to Linux and non-Linux groups.  Imagine how many businesses only use MSFT Windows because they don't know any better or think Linux is too hard.
[*]Submit documentation fixes or translated documentation to projects that could use it.
[/LIST]

Even 1 hour a month can make a huge difference. 1 hour a week can't achieve lots of things.
Or just sit back and enjoy using the OS. Sometimes we all need a break.
