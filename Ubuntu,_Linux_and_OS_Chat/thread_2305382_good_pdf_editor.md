### Post by lindsay3 on 2015-12-05
I checked the forums and there is a closed thread about pdf editors. 

I tried a couple of the suggestions, with not much luck. 

Today I stumbled across Masterpdf editor from code-industry.net

It does exactly what I want it to do: I can add a note or comment to a pdf. So far, it seems to do exactly what the latest adobe acrobat reader does. 

It is likely that others want more functionality, but I am very pleased with this so far.
