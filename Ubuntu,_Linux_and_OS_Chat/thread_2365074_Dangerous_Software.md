### Post by oldos2er on 2017-07-02
Any software that makes system-level changes can potentially be dangerous; this is nothing new. GUI apps like you mentioned are meant to make these kinds of changes simpler to implement, but that doesn't absolve the user's responsibility to understand what they're doing. 

When I ran openSuse, if you opened a root shell you'd see the warning ```
We trust you have received the usual lecture from the local System Administrator. It usually boils down to these three things:

    #1) Respect the privacy of others.
    #2) Think before you type.
    #3) With great power comes great responsibility.
``` Maybe all these GUI system changers should come with a similar warning.
