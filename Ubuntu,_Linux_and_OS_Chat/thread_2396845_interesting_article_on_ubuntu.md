### Post by coffeefiend on 2018-07-24
Great article! As with most Windows babies, I took the plunge for many of the same reasons. Getting a new laptop with Win10 installed seemed like a great idea at the time. And I still have it and use it almost daily. But having Win10 force me to run updates (loosely translated, that means giving me no real choice as Win10 ran the updates for me without any prompting) was very much a nuisance. So I started out by downloading many different Linux releases to try as bootable USBs. I narrowed my preferences down to Mint and Ubuntu. So I bought another cheap HP laptop and first tried to load Mint in a dual boot with Win10. Got a kernel panic every single time I tried to load it. I then tried Ubuntu 17.10. Flawless! After using it in dual boot for a couple of months I went all in and completely ditched Win10 off of that laptop for good. I upgraded to 18.04 and have been very happy with it so far.
