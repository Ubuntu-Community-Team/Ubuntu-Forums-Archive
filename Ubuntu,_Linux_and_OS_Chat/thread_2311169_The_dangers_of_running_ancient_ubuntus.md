### Post by sudodus on 2016-01-25
There are ***many*** examples of linux-servers that have been attacked via the internet.

See this link for more details (the link is old, but the general conclusions are valid)

[https://wiki.ubuntu.com/BasicSecurity](https://wiki.ubuntu.com/BasicSecurity)

A linux system that is not a server, and that hides behind a router with a firewall is less vulnerable to attacks, but I would not recommend running old systems without current security updates when connected to the internet. Of course, you might be lucky, and it might work well for a long time, but you never know when you would be attacked. Furthermore, I think it depends very much on your internet browsing habits.
