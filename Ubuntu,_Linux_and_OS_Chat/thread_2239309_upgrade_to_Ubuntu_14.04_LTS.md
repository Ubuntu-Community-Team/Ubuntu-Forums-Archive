### Post by kc1di on 2014-08-13
Hi Christon74 :)
Stay claim and cool off.  you may need to re-install some propritory codecs to get mp3 to play again.  

go to a terminal and type the following:

```
sudo apt-get update
sudo apt-get install ubuntu-restricted-extras ubuntu-restricted-addons
```

see what happens. 

This is one reason I always tell folks to do a fresh install rather than an upgrade. too many chances for problems with an upgrade IMHO. 
But now that you've done it you'll just have to spend some time getting things to work again. 

Enjoy the learning experience.
