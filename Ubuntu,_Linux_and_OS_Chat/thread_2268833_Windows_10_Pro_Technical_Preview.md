### Post by buzzingrobot on 2015-03-12
I tried it very early on. Like the OP, I'm pretty unfamiliar with Windows at this point, so getting around was a bit interesting. I'm not enthusiastic about the aesthetics of the thing.

I'm sure the preview is bulked up with debugging and testing code.  The terms of service explicitly detail that Microsoft's motive in offering the previews is for testing purposes, and that each preview installation is pretty much sending as much data back to Microsoft as it would if it was a machine in a Redmond lab. 

My take is Win10 will turn out to be a pretty good version of Windows for Microsoft's essential base of enterprise and institutional users. Those folks need to have a way to migrate off Win7, sooner or later, and it's clear Win8 didn't entice them.  (These users will be motivated to stay with Windows as long as it's around and usable rather than endure the cost and upheaval of moving to an entirely different platform.) Win10 won't replace iOS and Android in the small device market, for all sorts of reasons. Microsoft, though, would probably be very happy if those products just managed to return a profit rather than conquer the universe.

Personally, some apps I use have no Windows equivalents, and the font rendering is still worse than Ubuntu's and worse than Apple's unless you buy a HiDPI device with a display that's small enough to make it difficult to resolve individual pixels (same thing applies to any OS, of course.) Plus, Windows seems to use several different font rendering engines/techniques, which leaves the user out in the cold trying to tweak things for the better.

### Post by gordintoronto on 2015-03-12
This week, I attended a presentation labeled. "Windows 10, ask me anything". The presenter was a local Microsoft "evangelist".

A few of the key points:

For one year after Windows 10 is released, anyone with a valid Windows 7 or Windows 8 license can do a free "in place" upgrade to Windows 10, which will be valid for the life of the computer, and probably even for the life of replacement computers. Taking a full image backup before doing the upgrade would be prudent. (See Macrium Reflect.)

The free upgrade is especially encouraged for enterprise clients, so everyone will be on the same page when it comes to support.

The specs for Windows 10 are not yet finalized, and our questions/comments were recorded for forwarding to Redmond.

There will not be a Windows 11. There could be major (free) enhancements to Windows as time passes, as was done with the Windows 8 to Windows 8.1 upgrade. Microsoft doesn't get a lot of revenue from upgrades, so free upgrades are the way of the future. (Aside: we have discovered that a lot of computers which used to run Windows XP are totally viable with Windows 8.1. We are addicted to the pro version, which costs $155 at Canada Computers.)

Anyone who upgraded from Windows 8 to Windows 8.1 got a new license key, which IT support people need to record. (I did at least one such upgrade at the office.) A program called "intune" might help.

Microsoft believes that it has a 17% market share for "client software," what we used to call non-server operating systems. The most popular client OS is Android, followed by IOS, Windows, OSX and Linux.

Windows 10 will be released this year.
