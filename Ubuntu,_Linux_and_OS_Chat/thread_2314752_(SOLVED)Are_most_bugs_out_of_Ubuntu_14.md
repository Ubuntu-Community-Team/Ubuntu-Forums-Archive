### Post by grahammechanical on 2016-02-23
It all depends on what you define as a bug. Feature requests are bugs. Fixing bugs in applications or utilities are the responsibility of the maintainers of the utilities. Archive Manager or File roller is a Gnome utility. So, any bugs would need to be fixed by the Gnome developers.

One sure way to get updated versions of applications & utilities is to install the latest version of Ubuntu. At the End of April Ubuntu 16.04 LTS will be released. It will have 5 years support. Bugs from 14.04 will have been fixed in 16.04 but there will be new bugs. Software development is ongoing.

Cinnamon on Ubuntu is only installed through a PPA which is described as "unofficial". This blog is old and the writer promised an update but I haven't found it.

> For Ubuntu, the two Cinnamon unofficial stable PPAs I mentioned [a while back]("http://www.webupd8.org/2014/06/new-cinnamon-stable-ubuntu-ppas-ubuntu.html")  weren't updated with the latest Cinnamon 2.6 yet, but as soon as  they'll be updated, I'll publish a new article. Ubuntu users have  another option for installing the latest Cinnamon: the official Cinnamon  [Nightly Builds]("https://launchpad.net/%7Egwendal-lebihan-dev/+archive/ubuntu/cinnamon-nightly")  PPA, but the packages provided by this PPA are unstable and things can  break so I do not recommend it (the PPA should only be used for testing  purposes).

[http://www.webupd8.org/2015/06/cinnamon-26-officially-released-video.html](http://www.webupd8.org/2015/06/cinnamon-26-officially-released-video.html)

Don not expect Cinnamon on Ubuntu to be without bugs.

Regards.

Yes, 14.04 LTS is the most stable release available, as LTS releases are intended to be for machines that need stability in working/production environments and situations where downtime is not an option. 

I have 14.04 on three machines, they are on mostly 24/7 and I haven't had issues in an age. I installed 15.10 on my new desktop (very new hardware) and nothing but trouble. 16.04 LTS, which isn't released yet, actually works almost faultlessly. I intend to install 14.04 LTS on it today and see if it copes ok with the new hardware. If so, 14.04 LTS will be my main squeeze there, too.

For stability and longevity, stick to LTS releases. The interim releases can sometimes be flaky, sometimes rock-solid. Luck of the draw there, but never had major issues with an LTS release from 8.04 LTS on.

14.04 LTS has been around for nearly two years, so yes, it is pretty much bug free (at least for myself and the majority of users and as evidence of this, check the support threads on this forum: the 14.04 issues are generally user generated or incompatible hardware, not breakage of the OS). Sure there's bugs there, but they're under the carpet and out of sight. In other words, ineffectual here.
