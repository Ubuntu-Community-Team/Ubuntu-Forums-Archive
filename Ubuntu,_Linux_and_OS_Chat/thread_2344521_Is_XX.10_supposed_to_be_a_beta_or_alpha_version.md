### Post by deadflowr on 2016-11-26
Any version of Ubuntu that you can download from [ubuntu.com]("https://www.ubuntu.com/download") is an officially released version, being neither alpha nor beta.
If it feels alpha/beta it might be because a large majority of users are long term support users, so the amount of user testing of releases in between the LTS releases is lower.

As to whether or not an average user would knowingly use 16.04 over 16.10. The download page clearly states (and has  for as long as I remember) that the LTS release(currently )16.04 is supported for 5 years
(5 years is an always debatable topic, it seems here...) 
and the standard release (currently 16.10) is supported for 9 months.

I would think that in most people's minds longer is better.
And from a practical, anecdotal, observation, that thought would be right.
