### Post by TheFu on 2014-10-25
I have a C720.  There is much to like and much to hate for Ubuntu use. I only use it for Ubuntu. 
You can read my blog about the C720 [http://blog.jdpfu.com/2014/03/12/ubuntu-on-acer-c720-chromebooks](http://blog.jdpfu.com/2014/03/12/ubuntu-on-acer-c720-chromebooks) is one article, there are a few others - likes/dislikes and the other hassles. Do you want to rebuild kernel drivers to have the touchpad work every time the kernel is modified?

2G is NOT enough RAM for firefox and thunderbird - definitely NOT enough for running Unity too. I don't run any DE, just a pure WM.
16G is NOT enough storage. You'll need to upgrade that - which will break the warranty.
I needed an hdmi to VGA adapter and a USB3-->GigE Network adapter - really need a delete-key addon too.

All these upgrades were over $150 more - wish I'd gotten the Dell 11inch laptop. I completely serious. Plus that laptop can hold more RAM, a standard 2.5" HDD, ethernet, 

oh .... and don't get me started about the Chromebook keyboards - THEY SUCK!  No DEL key!!!!!!!  I'm completely serious - they don't frackin' have a DELETE KEY!!!!!!   No F11 or F12 either.

Dude - get the Dell non-chromebook 11inch netbook.  You'll thank me. Dell Inspiron 11 i3137-3751sLV is the model - it is $330 on Amazon. [http://www.dell.com/us/p/inspiron-11-3137/pd](http://www.dell.com/us/p/inspiron-11-3137/pd)   Delete key included. ;)  

Get the Dell laptop dude, NOT the chromebook.
BTW, I'm using the C720 right now.  I've been using it for about 4 hrs non-stop on battery:```

Battery 0: Discharging, 52%, 03:05:19 remaining
Battery 0: design capacity 4030 mAh, last full capacity 3865 mAh = 95%
``` I really think this is all due to the CPU being extremely low-power, but still about as fast a 1st Gen Core i5.

If you can't tell - I really, really, really, really, really miss the DELETE key.

Chromebooks have a terrible keyboard layout and are very difficult, if not impossible, to upgrade.  That is NOT an ultrabook - it is a very thin, light, travel, netbook.  Ultrabooks come with current-gen Core i5/i7 CPUs, not a low power Celeron.

I'm not suggesting this as a desktop replacement, but not having a fracking DELETE KEY makes "lite-word processing or email" next to impossible.

No body is suggesting this box for Java/Android development - heck - I doubt Eclipse would even start.  However, for python, perl, ruby and the web frameworks it is fine.  The only real issues are:
* delete key missing
* 2G soldered-in RAM
* M2 SSD
* No ethernet port
* No VGA out
* 1 USB2
* 1 USB3
* 768p resolution (but on an 11", that isn't so bad)
* touchpad drivers are not part of core kernel distro ... 

I am NOT complaining about the CPU or GPU or battery or power brick or ... chicklet keyboard or rebuilding touchpad drivers every few weeks.

Ok - so I like that Dell 11" model in port #2.  Heck, buy it and we can trade my C720 with the upgrades I've already done. You'll save a little. I had an Asus Eee netbook with 2G of RAM and never noticed any RAM issues, but it was slow and didn't handle any video playback over DVD (480p) resolution. I traveled with that Asus for 4 yrs - 5 continents.  I have a Dell 1558 laptop w/ a 1st Gen Core i5 - it is 1080p, 2.5 hrs of battery and easily runs multiple VMs, but it is heavy and doesn't get onto airplanes.  This C720 feels just as fast for most workloads and easily gets on airplanes.  There are also many desktops here ... that get used for anything where just a little power is needed.  A $400 desktop will beat any $1500 laptop in performance.

I'm very aware of what a C720 is used for - that is how I use it.  BTW, it run x2go client nicely to securely connect to a more powerful machine from anywhere in the world. Since getting the C720, I've only traveled in the USA and it worked great, but expect it will work great (better than the Asus Eee) from anywhere.
