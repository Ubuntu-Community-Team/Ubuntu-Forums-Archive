### Post by PhilGil on 2015-09-23
It has nothing to do with Canonical. Yahoo pays a fee to Mozilla in exchange for being the default search engine on Firefox (Google used to be the default and had a similar financial arrangement). Yahoo is the default for new installs, but (as VMC stated) you can change it with a few mouse clicks.

The default you choose isn't supposed to change when Firefox is updated. I didn't have a problem with today's update, but I have had my default search switch to Yahoo a couple of times in the past.
