### Post by ssam on 2013-10-03
Its only a few projects that actually announce release dates in advance. Distrowatch weekly news has a list for distros [http://distrowatch.com/weekly.php?issue=20130930#upcoming](http://distrowatch.com/weekly.php?issue=20130930#upcoming) . [http://freecode.com/](http://freecode.com/) lets you subscribe to get updates for software when it is released (not in advance).
