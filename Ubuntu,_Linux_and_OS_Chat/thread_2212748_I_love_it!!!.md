### Post by coldraven on 2014-03-23
Install Synaptic Package Manager and browse through the "Sections" to discover things you never knew about.
Also check "Status", is the program already installed or not? Right click an application and  select "Properties" for info on it's installed files and maybe discover some docs that are of interest.
You can think of Synaptic as the Pro version of the Software Centre. You may end up breaking your system but that's the way to learn and a re-install is easy and quick.
```
sudo apt-get install synaptic
```
