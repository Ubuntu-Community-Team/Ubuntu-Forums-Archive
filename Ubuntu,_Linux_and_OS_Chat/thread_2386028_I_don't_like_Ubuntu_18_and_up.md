I tried that as an exercise back in last April when they announced that Ubuntu was switching back to gnome, at that time the future of Unity was uncertain (now it is pretty sure that it will be installable on 18.04 and beyond thanks to community effort and there are talks of making an official Unity flavour)

Yes, you can customize plasma to look kind of like Unity even with some duplicated functionalities, but plasma has its own issues like the weird behaviour of the widgets which always seem to be changing position even if pinned, the menu to log out is too close to the button to show additional options in the top panel, docked software launchers behave inconsistently depending on whether you pin the software or create a widget (both methods have there own issues, e.g if just pinned then everytime you open something its position changes, if widget then for something already running instead of minimizing when click again you create a second instance ..), inconsistent font size in home run's dash.--there are some css hacks IIRC but this just reduces the font size uniformly, so it remains inconsistent, just scaled down) etc etc

Overall it feels very flaky and fragile as if it would fall apart any time. It is ok if you just want to look at it or make a Youtube video, but the actual user experience is not good ( I would say the same about efforts to make gnome shell look like Unity, cosmetic changes != functionalities)
