[http://www.omgubuntu.co.uk/2017/04/ubuntu-gnome](http://www.omgubuntu.co.uk/2017/04/ubuntu-gnome)

I think installing the Gnome desktop now, whether you are running 16.04, 17.04 or 17.10, will be the better way to transition. But honestly, I'm not worried; I will still use Unity and see how that goes. Probably Unity will be kept untouched, and Gnome will be installed next to it. Then upon first boot you will be given the option to choose Gnome.

I personally don't get why people get all excited about the desktop environment. People make it seem like Lubuntu, Xubuntu, Kubuntu, Ubuntu Gnome, etc. are completely separate operating systems. For me, it's just a skin. The exterior is a bit different while the internals are the same. So, I don't really care about the skin, I will use the default as long as it's reasonably usable, which all major desktop environments are.
