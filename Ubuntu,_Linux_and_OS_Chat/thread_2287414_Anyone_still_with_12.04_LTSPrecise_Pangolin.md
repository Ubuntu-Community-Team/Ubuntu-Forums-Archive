### Post by night_sky2 on 2015-07-19
You ask this question as if it was somehow unnatural, but it isn't since Ubuntu 12.04 LTS will be supported until 2017. There's a reason for that.

I am a firm believer of stability over new features when it comes to operating systems. So LTS releases are the only thing that I use, unless I should happen to be confronted with bleeding edge hardware not currently supported by the LTS kernel (e.g. waiting for the next hardware enablement stack to come out) in which case I would simply use the latest release in the meantime.

I am  running Ubuntu MATE 14.04 LTS and it's rock solid for me. I don't intend to upgrade this machine until the hardware starts to malfunctions and it's time to get another computer.

Man, if it ain't broke, don't fix it. ;)
