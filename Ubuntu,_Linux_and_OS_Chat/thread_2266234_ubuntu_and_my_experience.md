### Post by mooreted on 2015-02-21
I have had the same issue installing Windows 7. Boot up to the DVD, format the hard drive, install Windows, put in the license key. Everything goes smoothly until I get to the desktop and realize the Ethernet driver wasn't installed. Now I can't download drivers and I don't have a driver disk for this machine.

Most of the time Ubuntu is easier to install than Windows because it automatically installs all the drivers for you. Windows 8 is much better at it than Windows 7, but still not as good as Ubuntu.

When there is a system update on my machine there is an icon that pops up, I click it and the system updates, so it does have one button to update the system.

With any operating system, if you don't have the drivers for your hardware available, you're going to have problems.

### Post by tgalati4 on 2015-02-22
"Easy-to-Use" and "Install Operating System" are not normally used in the same sentence.  Find someone in a local, Linux User's Group (LUG) to help you out.  Once you have Linux installed and the bugs fixed--many times a simple fix; the system operates as you would expect and "It Just Works."  Getting to that point requires some patience, but everything is fixable.

Some hardware (like the Samsung Nopebooks) are difficult or impossible to install Linux on.  Others are just slightly difficult.  None are super easy, one-button, install operations.  If you bought a laptop with Linux installed, then the work is done for you.

Your concerns are shared by others and have been expressed many, many times on these forums.

### Post by PondPuppy on 2015-02-23
Toshiba -- yes, I have an old 32-bit Toshiba laptop that is my go-to machine for distro-hopping. So far everything I load on it works pretty much out of the gate. 

But with regard to the OP -- I suppose that, for myself, I've installed Windows and Linux enough that getting things to work is no longer a major thing. Which is quite nice, actually. If you get to the point where missing an ethernet driver in Win 7 (that happened to me too) or having to install a program using apt-get in Ubuntu is not a big deal, then a lot of angst is ameliorated. 

I might think it's useful in this vale of cyber-tears to have a few different OSes installed, and becoming somewhat competent in installing, using, and troubleshooting them. To my mind, it's not about running ONLY Windows, or ONLY Ubuntu, or ONLY Mac. If you have the luxury of having several OSes, then if one system becomes corrupted or tanks, you have other options. But that's just my feeling.
