The lack of a 'close' button (or a 'save' button or an 'expand' button) is deliberate.
See [https://wiki.ubuntu.com/NotificationDevelopmentGuidelines](https://wiki.ubuntu.com/NotificationDevelopmentGuidelines)

Please file a bug against that software package.
Software shouldn't deliberately make itself hard to use, nor should software be publishing notifications of the obvious.
Perhaps the developer hasn't tested the software against your Desktop Environment.
