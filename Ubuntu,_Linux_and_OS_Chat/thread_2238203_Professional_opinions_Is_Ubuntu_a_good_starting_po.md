### Post by Old_Grey_Wolf on 2014-08-06
Ubuntu is as good as any for a noob to use to learn about Linux; however, if you what a profession with Linux then it is only a starting place. I worked in IT and was responsible for several Linux systems. Some were DEB based; such as, Debian and Ubuntu. Others were RPM based; such as, CentOS, Red Hat, and SUSE. There aren't a lot of  difference between DEB and RPM based systems; but, for a Linux profession you will need to be comfortable with both.

### Post by Nistegmos on 2014-08-08
> **Austin_Powers said:**
> Hello,

    I have been wanting to learn linux for some time now,and I recently installed ubuntu v14.04.The reason I chose to go with ubuntu as my first linux distro is because a friend of mine has extensive I.T. experience and he recommended ubuntu as a good place to start,and I would appreciate other peoples opinion as to why ubuntu in particular is a good starting point for linux noobs?

I think that either Ubuntu or maybe even a Puppy Linux based upon Ubuntu, such as Carolina (X-Precise) Puppy Linux would be a good intro.  That's how I got into this stuff.  
Actually, the very first thing that I learned was basic Linux commands from a book about the history of Unix that a friend insisted I read back in 1992.  He also helped get me a college 
Unix account to play around on for learning.  I didn't think much of it at the time, but what he taught me really changed my life, and I am grateful to that guy, (nickname Loki).  

The main commands that I learned at first were:

cd
ls
pwd
man
mv
rm
rmdir
and probably cp

I may have learned a few process management commands, but I can't totally remember them anymore.  But if you learn those commands, then it helps a lot.  

My experiences with Puppy Linux helped me to learn how to use gParted, how to see how Linux labels and sorts drives and hardwares, and how to configure desktop settings and use window functions.  

After I got used to a few versions of Puppy Linux, I tried a few other distros such as WattOS, CentOS, and a few others not worth mentioning.  The advantage of trying out LiveCD's like that is that you see what the different versions all have in common, and it reinforces the learning process.  

These days I use Ubuntu Studio, which is pretty much regular Ubuntu, as far as I know, but it features programs for creative work and is designed to support those features.  I have it two old computers; a laptop and a desktop.  Only the laptop has internet access.  I find that it has been an extremely user-friendly distribution OS to learn on.  

Yesterday I tried to install Linux Mint Debian, and unfortunately, I would say that is NOT user-friendly in terms of installation, although it's actual interface is extremely similar to Windows--so much that I find it annoying.  For example, did they really need to implement several autorun behaviors?  That's one of the most annoying things to me about Windows.  

Anyways, good luck and be sure to read and search the entire internet for support.  And it's also worth reading the Wikipedia pages about Linux distributions, Comparison of Linux distributions, and things like that.  The history is amazing.
