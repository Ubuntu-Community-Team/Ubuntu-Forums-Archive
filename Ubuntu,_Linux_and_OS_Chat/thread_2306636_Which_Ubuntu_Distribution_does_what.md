### Post by tdawgf on 2015-12-17
Welcome to Ubuntu,

While not being a Linux Guru by any stretch of the imagination, I can tell you that judging from your experience, the only main difference you will notice is what the names themselves represent. Each of the different distros you mention replace the GUI for different alternatives (this sometimes will change some of the default applications with the system as well). For example. Ubuntu includes the Unity Desktop, Xubuntu includes XFCE, Kubuntu includes KDE, and Lubuntu includes LXDE. The most noticeable change is in the look and feel area. With Ubuntu and KDE, I have found that, in large part, are targeting more modern platforms, XFCE and Lubuntu seem to be able to run better on older systems (I had an older version of Lubuntu running on a pentium 3 with 384mb of ram really well). It is entirely possible to run Xubuntu and Lubuntu on new hardware (I have done this many times myself because I prefer the look and feel over Unity under certain use cases). I am sure some more OS oriented specialists can give you even more in depth info between them.

### Post by grahammechanical on 2015-12-17
By having a different Desktop Environment (DE) the distribution gets more than a different User Interface (UI). It also gets different utilities and different default installed applications.

Ubuntu has the Unity UI or shell but it is built on the Gnome 3 DE. So, it has utilities from Gnome.org. In comparison Ubuntu Gnome also uses the Gnome 3 DE but the user interface is the Gnome 3 shell. And the utilities are the same Gnome utilities.

The great variety of Linux distributions is a result of the holding to the philosophy of Free & Open Source Software (FOSS). It means that programmers can do things differently if that is their desire and at the same time benefit from the work of other programmers.

Regards.

P.S. A great thing about Linux is that it costs us nothing to download an ISO image and run a live session or even dual boot and try for ourselves. It is also less boring than reading list after list of applications in this or that distribution.

### Post by buzzingrobot on 2015-12-17
> **oskar10 said:**
> ...what do they all do different?



They all *do* the same things.

They all draw from the same pool of applications and other software on Canonical's servers.

Each is based on a different desktop environment, the interface with which a user interacts with the system. Each desktop environment -- Unity, KDE, Gnome, XFCE, Mate, LXDE -- also includes its own set of core and common applications, like a file manager, a terminal application, a photo organizer, a music player, etc. Those are the defaults and can be altered, at will, be the user.

### Post by deadflowr on 2015-12-17
While most of the flavors are general purpose systems, which vary in how they look and act.
Two that are truly purpose specific are Ubuntu Studio and Mythbuntu.
One is designed for media creation and the other is designed for media consumption.
But at the same time, those two can be utilized just the same as the others for general purpose computing, if you wish.
And by the same token the other systems can be re-imagined into either of those systems by installing the packages they require and setting them up as such.

### Post by mystics on 2015-12-17
For the most part, all you're getting is a different desktop environment. For people on low-end hardware, this can make a difference, as certain desktop environments (e.g. Xfce, LXDE) will run better on old hardware than other environments (e.g. Unity). If you aren't on really old or really cheap hardware and don't plan on doing really high-end stuff (e.g. gaming), then you probably won't need to worry about how lightweight a particular environment is and can choose based on which interface you prefer. If you want to figure out which one is best for you, just try each of them out.

Some versions of Ubuntu are made for specific tasks, but I'm reasonably certain you can set up another version of Ubuntu to do almost everything a specialized one does. It will just take more time and effort on your part.
