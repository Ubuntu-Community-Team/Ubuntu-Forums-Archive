### Post by hako1 on 2014-06-09
There are a couple of items in 14.04 which are different from earlier versions.
To name a few:
- Nautilus (the file manager) is not as powerful as before. If you need to work with your directory trees, you would have to install Nemo (not a very easy task).
- Caffeine is not working to inhibit screensaver/blanking. Its conditions are set to full screen only. So far, no solution available.
- Bluetooth does not work without manual loading a certain module.
- Upgrading is not always easy, I have done 2 PCs, one had to re-install from CD, the other had to re-install essential applications (e.g., libreoffice, callibre) which were lost in the process.
Most of these "improvements" are the outcome of evangelist ideology, without care for users who have a job to do.
If I would have started with 14.04, I'd stick to Windows. This way, Ubuntu missed the chance to replace XP.
Also, 14.04 is not much faster than 12.04. Especially, if you use the online results in unity, it can be even slower.
So, if you have used 12.04 before, just stick to it until the main problems have been solved, or easy circumventions are available.
