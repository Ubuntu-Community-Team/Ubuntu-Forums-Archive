### Post by buzzingrobot on 2013-09-14
OK.  SeaMonkey was last in the Ubuntu repositories in the Lucid Lynx release.  It is not in any repo for any supported release.

Per the SeaMonkey site, the current stable release is 2.20.  I found no Ubuntu-compatible deb packages there.

Here is a link to some info about Seamonkey on Ubuntu:  [https://help.ubuntu.com/community/SeaMonkey](https://help.ubuntu.com/community/SeaMonkey)    Note that some of the info is outdated, particularly the directions to install via apt-get.  However, there are links at the bottom to PPA's that provided Seamonkey packages for Ubuntu.  One of them appears still active, with the latest version at 2.19.

As for *updating* via Software Center:  I have found that if I have installed an Ubuntu package from a non-Ubuntu location, and later download a deb containing an upgrade to that package, Software Center *may* offer an update option (activate it by clicking on the deb package, perhaps specifically selecting Software Center if you have altered the default preferences). I don't use that approach because Software Center does not show you any dependencies it will also download.)
