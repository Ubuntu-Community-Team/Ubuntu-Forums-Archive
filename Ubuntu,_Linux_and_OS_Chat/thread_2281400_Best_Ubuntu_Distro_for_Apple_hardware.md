### Post by Bucky Ball on 2015-06-06
All good. I can only speak for Ubuntu and its flavours (do you mean 'distro' as a flavour of Ubuntu, as in Kubuntu, Xubuntu, etc., or as distro; Redhat, Opensuse, Arch?). All the Ubuntu flavours use the same base kernel so if one runs, you can be fairly certain the whole family will (and probably many other Ubuntu based 'spins' and distros which are not part of the official family but sit on the Ubuntu kernel). As Ubuntu is based on Debian, Debian will also run on a Ubuntu compatible machine I'd say.

Ubuntu certainly runs fine on Macs. The only real difference that I've heard of is the installation process with Mac as opposed to that of 'PC's.

Hope that is some help and hoping others can add more.
