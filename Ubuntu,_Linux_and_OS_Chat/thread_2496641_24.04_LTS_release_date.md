### Post by guiverc on 2024-04-06
You can view the Ubuntu release schedule for *noble* or 24.04 anytime, it's a document that is updated where there are changes (*you can also view the history of these changes if required*).

[https://discourse.ubuntu.com/t/noble-numbat-release-schedule/35649](https://discourse.ubuntu.com/t/noble-numbat-release-schedule/35649)

Release date for Ubuntu 24.04 LTS is 25 April 2024.

Only the *beta* release date is been recently changed (*delayed a week*).

FYI:  All systems are slightly different, yes Ubuntu defaults to having a created user in the *sudoers* file, where as Debian doesn't, but a simple edit should fix that.
