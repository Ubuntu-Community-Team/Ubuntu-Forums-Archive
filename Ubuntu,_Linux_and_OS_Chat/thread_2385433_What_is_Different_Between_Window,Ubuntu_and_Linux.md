### Post by mg2003 on 2018-02-21
As you probably already know, all three are operating systems.  They sit between the hardware and your applications making the computer easy to use.

Windows - Developed my Microsoft, it is a paid OS.  Most people use the desktop versions, but there are server versions as well.
Ubuntu - Ubuntu is a specific kind of Linux, tracing its roots back to the Debian project.  Most people use it for desktops, but it is perfectly suitable for servers as well.
Linux - Linux is a broad term that includes multiple OSs, such as Ubuntu, CentOS, Debian, Mint, etc.  Ultimately all of these trace their code back to the same kernel, but each one takes the project in a very different direction, making it specialize in a specific purpose.  For example, CentOS is a very good server flavor of Linux, but not so good as a desktop.
