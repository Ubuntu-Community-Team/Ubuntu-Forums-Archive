Just found this [https://addons.mozilla.org/en-US/firefox/addon/yesscript/](https://addons.mozilla.org/en-US/firefox/addon/yesscript/) 
I always find noscripts kind of an overkill since many websites don't work with scripts disabled. This is especially confusing for inexperienced users (who would benefited from this kind of protection). I find this takes a more reasonable approach by blacklisting sites instead of banning scripts by default.

Edited: actually the names should be the other way around, since scripts are opted in for noscript it should be called yesscript;  for yesscript js scripts are opted out it should really be called noscript.
