### Post by Bucky Ball on 2016-03-25
Welcome to the forums. 16.04 is not released for another month. Of course it's got bugs, to be expected and expect more. You shouldn't be using 16.04 as a stable, day to day install at this point. Make a comparison when 16.04 is released late April because any comparison between it and anything else is redundant at this time.

If you were to do an update/upgrade to 16.04 in 24 hours then do your comparison again it would possibly have a different conclusion. If you waited a week it definitely would. Open a terminal and 

> sudo apt-get update
sudo apt-get dist-upgrade

Reboot. Repeat daily.

Incidentally, if you fancy being proactive, go to where people talk about 16.04 and its bugs and post about your 16.04 issues in the ***Ubuntu Development Version*** sub-forum here, which is the place for anything to do with 16.04.

If you want to make a valid comparison, do it with a supported, released version; 14.04 LTS or 15.10. 14.04 LTS is rock solid (and upgradeable to 16.04 LTS via the net). Compare that. If you are after a reliable, day to day desktop computing experience, I advise you use that anyway. 

If you are ready for the learning curve and have some time to tinker, keep using the unreleased 16.04 and help the community stamp on bugs when you find one by posting and checking for it on Launchpad. No doubt others have exactly the same issues as you're having and you'll find the community already talking about them and working on it. ;)
