### Post by Shibblet on 2019-01-24
I am a great fan of KDE.  So I use Kubuntu.

My first delve into the Linux world was Ubuntu 8.04 (Hardy Heron if I remember correctly.)  And I distro-hopped for almost 8 years.  Never more than 6 months with an OS.  I wanted to try the new desktop environments as they came out, and I also wanted to see the differences between the distros.  I also wanted to see them in the "natural" environment, and not within a VM.

I chose Kubuntu for a number of reasons.  First off, KDE is my favorite DE, and Plasma 5 quick, snappy, and doesn't bog down the system.  There are still KDE desktop effects (which is fun), as well as an intuitive interface.  If you are a gamer, one of the best things about KDE, is that it suspends the desktop compositing during game rendering for better performance.

Ultimately, around 16.04, I stuck with Kubutnu as my primary distro, but have a separate partition that I use for checking out new distros.

However, all of the other desktop environments are good in their own right.  But my favorite "flavor" is KDE.
