### Post by buzzingrobot on 2013-06-28
13.04 here has been as solid and smooth as 12.04 LTS after bumping up the default kernel, which took about 2 minutes. I have not noticed "hundreds of MB" of updates.  

If updates seem to cause problems, it is likely that they are being installed on a system that has been altered. Updates to any OS assume they are being installed on a baseline system.  If you aren't running a baseline system, you can have problems. In Ubuntu, for example, if you've installed PPA's or other unofficial repos, you aren't running a baseline system.

[EDIT:  LTS' 5-year lifetime is fine and all, but how many home users *really* keep any version of Linux on one machine that long?]
