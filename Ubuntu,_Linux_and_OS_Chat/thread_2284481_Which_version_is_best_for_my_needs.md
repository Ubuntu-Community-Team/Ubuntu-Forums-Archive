### Post by sudodus on 2015-06-29
Ubuntu 14.04 LTS is a long time support version (5 years of support). Some of the community flavours, at least Lubuntu and Xubuntu have only 3 years support. The next LTS version will be 16.04 LTS, to be released in April 2016. There are short-lived versions in between, 14.10, 15.04, and the currently developing version to be released as 15.10 in October. Those versions are supported for only 9 months.

So as suggested by sammiev, [try different flavours of Ubuntu, and try them live](http://ubuntuforums.org/showthread.php?t=2230389) in the new computer, and then install the one you like best.

And I suggest that you stay with the LTS releases. Start with 14.04.1 LTS, skip to 16.04.1 LTS (the first point release, which will be much better debugged than the original LTS release).

### Post by Bucky Ball on 2015-06-30
*Thread moved to **Ubuntu, Linux and OS Chat**.*

There are not a million versions of Ubuntu out there. There is a small official family of flavours supported by Canonical:

[http://www.ubuntu.com/about/about-ubuntu/flavours](http://www.ubuntu.com/about/about-ubuntu/flavours)

... and a million spins that are based on Ubuntu, but may have little else to do with it apart from the kernel. One thing to consider is if the distro you're intending to install has a good and active support community. The more exotic you go, the less likely you are to find help with it if you need some. These forums support:

Ubuntu, Kubuntu, Xubuntu, Edubuntu, Lubuntu, UbuntuGnome, Ubuntu Studio, Mythbuntu and Ubuntu Mate.

Anything else, look for a vibrant support community before you dive in. 

Gimp is not an operating system. It is a program, similar, and an alternative to, Photoshop.

Questions like the one you pose are almost impossible to answer. Any of the official flavours would be fine. It depends entirely on what suits your workflow and that is about the desktop environment it uses. As far as web dev and anything else, that is about apps and any Ubuntu app can be run in any Ubuntu flavour (and many derivative spins), so by the by on that one. Doesn't matter which flavour you use when it comes to the apps you want to use.

My suggestion is prepare yourself with install media of all the flavours/distros you're interested in, boot from them, choose the 'Try *buntu' option and try it from the install media. See what you like. The only concern would be the specs of your machine, but from memory, that won't be a concern for you. You would be able to run any flavour comfortably. Start with Lubuntu, lightest and fastest, then Xubuntu, Ubuntu, Kubuntu, etc. 

When you find something you really like, install. Good luck.

PS: If you're feeling like a learning curve, why not try the minimal install? You install the base kernel then the rest is up to you. Create your own spin! Install your favourite desktop environment and apps and that's it. You're own hybrid OS. ;)

### Post by d-cosner on 2015-06-30
Since you seem relatively new to Linux my suggestion would be to stay with Ubuntu 14.04 LTS. Ubuntu is very user friendly and the length of support for LTS releases is very good. Linux Mint 17.2 Cinnamon might also be worth your time too because it is also an LTS release and new user friendly. You really can not go wrong with either of these choices. Both these distributions are very popular and there is a great deal of information available if you would need help with anything. Linux Mint also has the Gimp installed by default along with all of the multimedia codecs.

It's very simple to install codecs and the Gimp in Ubuntu as well using the software center. The only real difference between these two is the user interface. Mint uses a traditional layout not so different from Windows. Ubuntu is more Mac like for lack of a better description. It really just comes down to personal preference though because both are very easy and pleasant to work with.
