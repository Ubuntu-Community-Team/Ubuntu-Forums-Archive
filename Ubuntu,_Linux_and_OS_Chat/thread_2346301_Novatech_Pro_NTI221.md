### Post by ajgreeny on 2016-12-13
As it does not mention a graphic card I suspect itwill be using the integrated Intel graphics which almost always work very well for general use, perhaps not so well for gaming if you want to use Steam, for example.

Is wifi important to you as there does not appear to be a wifi card, only an ethernet port whcih will also usually work out of the box.
If you really need wifi you can find low cost USB adapters pretty easily that will also work out of the box. Come back for help with that as I have never needed wifi on my desktop machine, so others will need to help you with personal recommendations.

As a general comment, I have used Novatech several times here in UK and found them to be very good; I still use one of their X10 Netbooks from 2009 that came with no OS, but with Xubuntu 16.04 on it now; it's no speed merchant but it is still very usable.
