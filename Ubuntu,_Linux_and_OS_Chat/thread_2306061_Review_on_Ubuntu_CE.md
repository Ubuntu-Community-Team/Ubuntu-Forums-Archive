### Post by incognito4 on 2015-12-11
Hello; In my efforts to ascertain the viability of Ubuntu CE, I have the following results. After installing UCE 12.04, I found that the dansguardian web filter in the default mode does not allow access to my google account, hotmail account nor any website that is among my favorites. The only website that is accessible is the ubuntu forums( of those that I tried) Ironically even the dansguardian site is not accessible.

            This means that without configuring dansguardian, this OS would not be usable for even basic computer use such as email, web browsing, online shopping and more.



 Using the man command you can bring up the dansguardian manual, which shows that a configuration file exists, to access it though, I was unable, the manual gives links to help sites at dansguardian, which of course you would have to use another OS to access.
             I did try pasting commands from another thread regarding dansguardian configuration, however these unfortunately did not work.

 I can understand the merit of  the dansguardian web filter for institutions such as church groups where there are open, unsecured and unsupervised computers available for users. Or in a home setting that has young children that are allowed access to a computer for a variety of uses such as games. In these cases dansguardian acts as a very effective supervisor, restricting internet activity to the very minimum.  

            As far as being an effective OS for everyday use by average computer users, It doesn't appear to have even been designed for that function. If it is usable and configurable for online use, it would have to be done by people with computer skills beyond mine. 

As for using the other features and programs within this OS, I did not explore these. Why bother when it is not useful for the most basic functions? This OS could be a supplement to another system, for the reasons stated above, but as a stand alone system it is restrictive to the point of being ineffective.
