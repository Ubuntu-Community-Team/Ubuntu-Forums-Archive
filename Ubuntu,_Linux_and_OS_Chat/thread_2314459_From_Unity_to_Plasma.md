### Post by irv on 2016-02-21
Years and years ago I used SUSE with the KDE desktop but once I switch to Ubuntu I went to the Gnome desktop and then to the Unity desktop. I really gave it my all. I can't remember when it came out, but I been using it until this past week when some thing happen to change my main desktop. So there is no one how here in Ubuntu land that has not given Unity a real good and long try as I have. Like I said, I have been using it since it came out.
Now this is what happen this past week. I picked up a cheap Chromebook and installed Ubuntu on it with the Unity desktop but the screen was only 11.6 and I wasn't real happy with how it looked so I switch the desktop to KDE. I installed the new KDE Plasma Desktop and I really liked it. In fact, I liked it so much I installed it on my big laptop that I use most of the time.
I have been using Windows 10 on and off since it came out and I will say it is the best windows that MS has come out with yet, but it is still Windows. All us Linux users know there is a much more out here then just Windows.
The bottom line is I am going to leave unity for KDE for my everyday use and see what happens. Just so you know, I did try Mint with its different desktops and I liked it but it was just not my cup of tea; Green tea that is. Not that there was anything wrong with it, it was that it didn't fit me like KDE is doing.
