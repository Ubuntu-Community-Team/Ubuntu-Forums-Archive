### Post by oldfred on 2017-07-13
Do you really need a laptop?

Laptops have smaller screens, are not ergonomic on desktop as keyboard is too high and screen too low. And any laptop that says it has the same performance as desktop is really comparing to 3 year old desktop. Laptop has limited upgrade abilities, usually just new hard drive.

One user did post that he buys a bit lower end laptop with hard drive. Removes hard drive before even booting into Windows and installs SSD and Ubuntu. Then later sells laptop with unused Windows and does not have to worry about any of his data on system.
