### Post by PaulW2U on 2018-04-16
> **phanhuyen said:**
> Does anyone know if there is any development going on for a pure Gnome edition of 12.10 ?
If you install either Ubuntu 17.10 or 18.04 (due for release later this month) and add the **vanilla-gnome-desktop** package you can, by removing many of Ubuntu's customisations and themes, experience what you might call 'pure GNOME'. There's no longer a need for a separate flavour or remix.

It's what I have done as I don't like many of the choices that the Ubuntu developers have made in recent years regarding Ubuntu's themes and colours. I now see applications pretty much as the GNOME developers intended although I have chosen to keep the Ubuntu dock which resembles Unity's Launcher.

[How do I get a vanilla GNOME after Ubuntu 17.04?]("https://askubuntu.com/questions/964598/how-do-i-get-a-vanilla-gnome-after-ubuntu-17-04")
