### Post by buzzingrobot on 2015-08-26
> **SomeDudeOnline said:**
> 
\
So is Ubuntu always buggy garbage...


No.

Bugs exist in every complex system.  So there are bugs in Linux, in Windows, in all software.  Some of them will be fixed. 

The ability of Linux users to post bugs very likely increases the total bug count.  Few projects have the resources to prune and edit bug lists to remove duplicates, redundant posts, non-bugs, etc. Users often post bogus bugs that are really due to something they've done. I'm sure if we looked at Microsoft's or Apple's bug lists we'd see thousands of bugs, too.  Given that we're talking about hundreds of millions of lines of code when we look at an entire OS plus the thousands of software packages that run on it, that's to be expected.

The only realistic way to fix all those bugs is to freeze development and focus only on bug patching.  Of course, user demand for new features makes that impossible, even on Linux distributions like Red Hat and Ubuntu LTS..

Your specific problems can't be addressed here without specific info from you.
