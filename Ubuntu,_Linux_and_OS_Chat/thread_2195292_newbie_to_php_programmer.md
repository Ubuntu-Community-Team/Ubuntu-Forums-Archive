### Post by Anonimista on 2013-12-23
I recently switched from Windows to Ubuntu, but it was because my old XP box died and I wanted to try Ubuntu, so I bought the new one without Windows. I've since tried Ubuntu, Xubuntu, Kubuntu and Fedora. I'm on Ubuntu with Gnome classic right now and I've tweaked it look more or less like Windows, so there is definitely a learning curve if you don't have any experience with Linux. I had no issues with using Apache/PHP/MySQL on Windows, NetBeans is nice with project management, PHP auto-completion,GIT and unit test support, although can be slow on older machines (like my old one). Right now I have LAMP and NetBeans on my new box, no issues once I managed to install all of it properly.
