### Post by zebra2 on 2021-08-20
If you are using Ubuntu 21.04 Gnome is 3.38. If you are using Ubuntu 21.10 Gnome 40.2 is part of the Ubuntu Development series and GTK is not yet fully supporting Gnome 40. 
Bottom line is Gnome 40 is a development version. Also libhandy and adwaita are independently developed by Gnome and then shipped as part of GTK. It doesn't matter if the GTK is 2,3, or 4 the Gnome display support is whatever Gnome has deployed. This in itself is a greater oddity than the fact that the dock is messed up, but it is Gnome not Ubuntu that is in charge of the matter. Ubuntu 21.10 is still in development and I use it every day as my only installed OS on three computers and everyone here is OK happy with what they have to use. But there are problems.

PS. Since Gnome 40 is a part of 21.10 I think the thread was properly started in the Development section since they are both in development and we don't really know who the problem belongs, probably both. It seems to me that when I was using the PPA version of Gnome 40 with 21.04 the bottom dock was centered.
