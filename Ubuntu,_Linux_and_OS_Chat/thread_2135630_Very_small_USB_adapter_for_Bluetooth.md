### Post by Yoshamano on 2013-04-15
I'm currently using the ASUS USB-BT211 [http://usa.asus.com/Networks/Accessories/USBBT211_Mini_Bluetooth_Dongle/](http://usa.asus.com/Networks/Accessories/USBBT211_Mini_Bluetooth_Dongle/)

It was plug and play on 12.04 (aside from installing bluetooth audio support, took a few to track that little nugget down). The only issues I've run into with it is you need to get the generic Atheros 3011 drivers for it if you're going to use it in Windows, and I broke the black plastic casing (which could have been from my carelessness). So aside from some stratigic clear take it works like a champ.
