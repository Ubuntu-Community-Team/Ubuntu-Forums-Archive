That is a screen shot of an icon theme . The noobslab ppa is pretty simple for a beginner,  select the name of the theme you want to look at and follow the terminal commands on bottom of the page . Example : [http://www.noobslab.com/2014/02/install-e17-gtk-theme-in-ubuntulinux.html](http://www.noobslab.com/2014/02/install-e17-gtk-theme-in-ubuntulinux.html)
