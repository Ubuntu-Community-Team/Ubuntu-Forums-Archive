### Post by ajgreeny on 2018-02-16
If you install the Skype .deb file from [https://go.skype.com/skypeforlinux-64.deb](https://go.skype.com/skypeforlinux-64.deb) (it is 64 bit only, no 32 bit available) it will add the repos to your software sources so it will then stay updated to the latest version.

I hear what other people say about using alternatives to Skype, eg, Telegram, but if all your contacts use Skype, as many of mine do, you have no alternative but to use Skype itself; nothing else will connect to another users Skype application.
