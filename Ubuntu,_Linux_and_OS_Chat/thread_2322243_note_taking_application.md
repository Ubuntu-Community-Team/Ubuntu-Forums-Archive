### Post by Bucky Ball on 2016-04-27
*Thread moved to **Ubuntu, Linux and OS Chat**.*

Zim. Light and easy. Available in the repos.

Zotero is absolutely brilliant for referencing. You can also input notes, but not the main aim. Flexible and powerful. Available as a Firefox add-on or standalone from their website. Also has a Libreoffice plugin for inserting references directly into documents from Zotero.

If you are studying and need something for referencing, Zotero is it for Ubuntu IMHO.
