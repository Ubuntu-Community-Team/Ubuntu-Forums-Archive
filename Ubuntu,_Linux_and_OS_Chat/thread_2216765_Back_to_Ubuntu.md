### Post by Grez on 2014-04-13
Just come back to Ubuntu 12.04 after using Mint for about 18 months.

Was having a few problems with Unity (too clunky for me in the end), and a couple of issues with drivers and stuff. However, I began getting a regular set of filesystem errors with Mint shutting down, and I think I isolated it to the interaction of my SATA HDDs along with the nVidia driver I was using. Nouveau was overscanning on my monitor, so I ended up having to use the nVidia proprietary driver. That's when the problems started.

Nouveau works fine with Ubuntu Precise, but I still don't like Unity all that much, so I've added Cinnamon 2.0 and it seems to work great. Here's some screenshots if you're interested...
