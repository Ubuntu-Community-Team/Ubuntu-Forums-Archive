### Post by ajgreeny on 2015-09-04
As grahammechanical says, you are asking this on the Ubuntu forums so you can expect more to recommend Ubuntu or one of its family than Mint, but it is very much personal choice.

Mint saves a bit of installation time according to some users, as codecs etc etc, can be installed from the DVD or USB you use to install, but these days you can add them during the installation of Ubuntu if you so choose.  I prefer to install quickly and not update as I install, nor do I add codecs and third party applications, as I prefer to get the OS installed and then have full control over what else is added.

With regard to the overall look of the OS, if you like unity desktop you don't really have a choice, as it is not available in Mint as far as I'm aware, and for the other desktops I think there is parity as far as their availability is concerned.  Wallpapers and other things such as themes are of no importance so far as the choice between the two OSs is concerned as they can be easily changed to suit yourself.

### Post by mystics on 2015-09-04
Try them both. Actually, try everything in both families. Linux Mint has Cinnamon, Mate, Xfce, and KDE versions. Ubuntu also has more versions than the base one that uses Unity. This includes Xubuntu (my favorite!), Ubuntu Mate, Lubuntu, Ubuntu GNOME, and Kubuntu.

All of these are relatively easy to use, and I'd imagine a casual user won't recognize much of a difference between them beyond desktop layout and design. So give them all a try using a live USB and see which one works for you.
