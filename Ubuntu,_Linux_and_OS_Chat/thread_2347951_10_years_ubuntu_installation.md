### Post by grahammechanical on 2016-12-31
I started with 7.04 and am now using 17.04 (development version) but it is not the same installation. I do clean installs from time to time.

As an experiment I once tried re-installing 7.04 and then doing end of life upgrades to 14.04. Ubuntu 7.04 would not install although the motherboard and CPU had not changed. I had changed the graphic adapter. It was newer than 2007 and so the open source video driver was not specified for it.

I did get 8.04 or 8.10 installed and I managed to get as far as 10.10 but the jump from 10.10 to 11.04 (and unity) messed up the graphics. But it was an interesting experiment. I got practical experience with End Of Life upgrades.

So, I am running Ubuntu 17.04 on an Intel 2.40Ghz Core2 Duo with 1GB of Ram and a graphic adapter with 1GB of video memory and things work fine. Well done Ubuntu developers as well as the Linux kernel developers.
