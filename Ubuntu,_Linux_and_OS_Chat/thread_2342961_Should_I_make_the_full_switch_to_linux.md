### Post by ian-weisser on 2016-11-11
Run Windows because you want to run Windows.
Run Linux because you want to run Linux.
Linux is not designed to be a free substitute for Windows. Treating one as a substitute for the other will lead you down a path of great frustration.

Generally, when considering a switch, start with applications instead of OS. Change to cross-platform applications (LibreOffice, FireFox/Chrome, Evolution) in your Windows workflow first. It helps you to identify the roadblocks and makes the later change of OS much easier.
