### Post by tgalati4 on 2015-09-27
Welcome to the forums.  The Desktop version of Ubuntu gives you a graphical interface similar to Mac or Windows.  The server version is console or text-mode only, so you need to be an advanced user.  It is used for creating websites or running other services.  If you are new to linux, start with the Desktop version.
