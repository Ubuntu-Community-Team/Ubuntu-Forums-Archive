### Post by deadflowr on 2018-08-30
It's not amd version.
It's 64-bit version.

AMD64 = 64-bit 
I386 = 32-bit.

It's all interchangeable.
amd64 will run on Intel and i386 will run on AMD.

### Post by QIII on 2018-08-30
It is an industry naming convention arising from the fact that AMD brought the first 64 bit hardware and instruction sets to the consumer market.  It was called AMD64 at the time and the name stuck.  You will also see it referred to as x86-64 and x86_64.

There is no attempt by Canonical to cause confusion.
