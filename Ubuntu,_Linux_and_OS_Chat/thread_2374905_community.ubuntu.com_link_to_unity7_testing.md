### Post by cariboo on 2017-10-25
> **Chanath said:**
> Do you see "an appropriate support sub-forum" for Unity 7 on 17.10 technical support questions?

EDIT: Not a problem any more. I just read Mark's words, dated 24.10.17, the 5th para.

Unity will still be supported until 2021 in the LTS versions, so any questions and bugs can be asked/reported in [General Help]("https://ubuntuforums.org/forumdisplay.php?f=331"). Starting with Bionic, unity is no longer in the main repositories, as it has been moved to the universe, where it can be maintained by the Ubuntu community. Bug reports and question can still be posted in the U+1 development forum, but any other discussions will be moved to the Chat Forums.
