### Post by Bucky Ball on 2015-09-17

It seems like you answered most of your own question by trying them all out. 

> But how do Gnome, Mate and Unity behave? 

Give them a go as well and find out. The less resource hungry the OS, the more battery life, naturally. I've heard Mate and Gnome are lightweight but believe Unity to be around Kubuntu. I don't use any of them so couldn't say with any certainty. :)

You could do a vanilla install of each (don't install anything after install) and check I guess. Once you start installing stuff, things are going to change a bit from user to user.
