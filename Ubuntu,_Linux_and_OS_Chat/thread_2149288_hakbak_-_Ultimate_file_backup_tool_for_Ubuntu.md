[FONT=Georgia]But worry not! 'hakbak' can help you! What on earth is hakbak i hear you say?!?

[FONT=Georgia]Well hakbak is a simple bash script i have pieced together that automatically create backup of a file you want to open for editing! It can be a text file, an audio file, an image file, any file! hakbak supports them all as long as your system supports them.
[/FONT]
[FONT=Georgia]hakbak can also be described as your personal 'Version control system' as it creates backup of file with opening date and time stamped as an extension. So you will have multiple backups of your frequently edited file with precise time and date of modifications at your disposal.
