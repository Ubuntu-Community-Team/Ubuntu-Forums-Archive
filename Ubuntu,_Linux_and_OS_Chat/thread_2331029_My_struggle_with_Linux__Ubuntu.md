### Post by QIII on 2016-07-24
Nobody should be made to feel sullied or somehow unfaithful for using whatever OS works best for the task at hand.

Linux as a kernel and an ecosystem.  Ubuntu is an OS.  It's not a religion.  It's not your spouse.  It does not demand your full devotion.

I think the problem with getting people to "switch" to Linux is because we seem to see it as a "switch" and to sell it as a "switch".  It isn't.  It's another OS to use when you don't happen to be using Windows.  

If we talked to people that way instead of presenting a binary choice, I think adoption would be higher.

### Post by grahammechanical on 2016-07-24
Computers long ago reached the point where they were powerful enough to do most everyday tasks. But hardware development was driven by the ever increasing specifications of Microsoft Windows & competition between games companies to produce ever more exciting and realistic games.

Hardware that cannot run the latest Windows or the latest games can still be made useful by installing a Linux distribution. I dare say that if people were willing to buy the same hardware and pay the same price for the OS and games as Windows gamers do, then the gaming experience on Linux would improve somewhat. But if a person is prepared to pay those prices then they are already paying out for Windows and Windows games. They have no need to switch. 

The most demanding of customers are those who expect something for nothing. They forget that they are not actually customers and have not purchased the right to make demands.
