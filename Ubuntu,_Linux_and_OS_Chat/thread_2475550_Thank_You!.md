### Post by agentl074 on 2022-05-31
Thank you to all of the developers who worked hard on Ubuntu 22.04 LTS! I came from Windows 10 and Linux Mint 20.3, and I was about to abandon hope for using Linux on my newer machine (Asus M712U AMD 5500U laptop) until I took a chance and loaded Ubuntu 22. I am so glad that I did! Everything works and is tuned very nicely right out of the box. One thing that I had to do was sudo apt update && sudo apt upgrade -y to populate the app store categories and ensure functionality for installation and removal.

 I also installed Brave browser via OEM repository -- as well as the -- in my opinion -- critical package Ttfmscorefonts via command. Lastly, I installed Gnome tweaks utility and added other nice apps like weather and other essential apps, and did some customization. 

Once all of these adjustments were made, I fell in love with the elegance and functionality of this OS. Thank you so much!
