### Post by TheFu on 2022-10-06
All DEs are too bloated for me.  fvwm, a pure window manager works for me.  [https://www.fvwm.org/Archive/Screenshots/](https://www.fvwm.org/Archive/Screenshots/) has screenshots. 

It is amazing how a single WM can be customized using 1 text file to control vastly different looks.  It has supported transparency for decades. No fancy GPU required.  Works well over any remote desktop and inside virtual machines too.

Plus, every new Ubuntu release with some new DE "feature" doesn't matter. My desktop doesn't change, so I don't lose productivity trying to figure out some new GUI and where they've hidden stuff this time.  FVWM has been around and stable since the mid-1990s. It isn't going anywhere. It has been 
Here's one with a 3x3 workspace step, conky system performance and a few windows: [https://www.fvwm.org/Archive/Screenshots/2003-08-28_Ploum-desk-1152x864/screenshot.jpg](https://www.fvwm.org/Archive/Screenshots/2003-08-28_Ploum-desk-1152x864/screenshot.jpg)
If you like a dock, you can have that. I'm not a fan of docks or menus stuck in specific locations. Guess it is whatever you are used to.

I loved KDE4 but some of that magic got lost as it moved to KDE5, but I still have **KDE** installed and use it on occasion, and I can *feel* why I loved KDE4 now and again, but then realize it's not the same and I find myself logging into another DE/WM the next day usually.

(*I'll stop mentioning/listing DEs due to length*)

I have many WM's installed (*including **FVWM** used by TheFu*), but I mainly use them when using older boxes with limited RAM (1GB for example). I do find myself using **ICEWM **somewhat often, as well as other WMs, but they're hardly a 'favorite', just nice for a change*.*
