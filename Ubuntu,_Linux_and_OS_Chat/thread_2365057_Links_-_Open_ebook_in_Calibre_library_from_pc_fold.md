### Post by monkeybrain20122 on 2017-07-01
Calibre works natively on Linux. Not sure why you need to use wine. [https://calibre-ebook.com/download_linux](https://calibre-ebook.com/download_linux)

I don't think what you suggest is possible. But you can create calibbre's library anywhere, you can create it in your book folder and delete the original if you don't like duplication. I use calibre only for books in epub, mobi etc and there are only a few of them so I don't mind the duplicates. For pdf or djvu books I just use a document viewer (qpdfview in my case)
