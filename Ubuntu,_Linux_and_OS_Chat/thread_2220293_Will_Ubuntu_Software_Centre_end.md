Ubuntu one cloud storage and ubuntu software centre are completely different things. Ubuntu software centre or some form of it will not end untill ubuntu does and the backend will exist as long as debian or some form of it does.

### Post by grahammechanical on 2014-04-27
I also prefer to use the software centre for installing and removing applications. You may not be aware of it but there is a lot of encouragement for people to write applications for Ubuntu. There is a Ubuntu Software Development Kit (SDK) under development which makes it easy to package applications for use in Ubuntu and the SDK also makes it easy for a developer to get their application accepted in the software centre.

The main focus of this development is towards Ubuntu phones/tablets with a special phone/tablet software centre. But the intention is to merge the Ubuntu code base for phones/tablets into the Ubuntu desktop code base. And this means that phone/tablet applications will also run on the Ubuntu desktop. It may be that Ubuntu Software Centre will be replaced by Ubuntu Software Store. But one thing we can be sure of is that there will always be a utility that makes it easy to install applications in Ubuntu.

[http://developer.ubuntu.com/publish/](http://developer.ubuntu.com/publish/)
