### Post by su:bhatta on 2014-03-22
Well, there are 2 paths:

1) Install lubuntu-desktop/ xubuntu-desktop from Software Center. Once it is installed Log-out and when you log in choose the LXDE/XFCE from the Log-in screen, that will give you the LXDE/XFCE desktop.

2) Download a iso of Xubuntu/Lubuntu from the respective sites mentioned in my earlier post and have a go :)

In case you install from Software Center, there will be some application that you will see twice over, Like 'Settings' 'Online Accounts' etc, when you do, you post a fresh thread to sort it out, else like me you can keep them all.
