### Post by PaulW2U on 2016-11-09
> **simonsaysthis said:**
> I noticed that my Kernel still shows at 4.8.0.26 Generic. However I believe there is already 4.8.6 and all 4.8 users are urged to upgrade to this version. Is this a bug in my system that the Kernel doesn't update
Ubuntu uses a different versioning system to the mainline kernels.

[http://kernel.ubuntu.com/reports/versions.html](http://kernel.ubuntu.com/reports/versions.html) shows that 4.8.0.26 is the latest version that has been released for Ubuntu 16.10 and that 4.8.0.27 is currently in the proposed repository awaiting testing and eventual release. I can't find it now but somewhere on the Ubuntu wiki is a page which maps to two versioning methods. So no bug. You have the latest kernel version installed.

Edit: [http://people.canonical.com/~kernel/info/kernel-version-map.html](http://people.canonical.com/~kernel/info/kernel-version-map.html) doesn't show 4.8.6 but does show the two versioning methods. Hope that helps.
