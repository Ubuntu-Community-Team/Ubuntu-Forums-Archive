### Post by jpberes on 2017-10-30
Ubuntu is a Linux Distro (call it a kind of Linux variant) based upon Debian, one of the oldest Linux versions.
The (K) (X) or (L) in front of Ubuntu are referring to the Desktop Environments.. These are all Ubuntu, but with a different Desktop Environment or simply a different User Interface...
It's a bit more difficult to get it if you come from Windows, since windows has it's incorporated Desktop Environment ... but in Linux and so within Ubuntu you have the freedom to choose how your Ubuntu version would look like.
The (K) in front of Ubuntu, so Kubuntu is the Ubuntu with the KDE Desktop Environment, the (X) .. Xubuntu is Ubuntu with the XFCE Desktop Environment and the (L) ... Lubuntu is Ubuntu with the LXDE Desktop Environment. All different layouts of the same Ubuntu, so of course all for free. However, a difference is also the resources used by the different versions, KDE is a lot heavier on resources (you need a more powerful PC) then for instance XFCE, which can even run on older machines. LXDE is less customizable and like this very useful for machines with a little RAM.
