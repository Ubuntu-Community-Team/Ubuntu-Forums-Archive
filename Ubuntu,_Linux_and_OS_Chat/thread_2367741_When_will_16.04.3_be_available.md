### Post by exhile on 2017-08-02
There has to be scheduled releases. For desktop Linux users it's understandable that releases be pushed back since Linux on the desktop represents a very small portion of the market. However, operating systems like Windows have to be released on time because OEM manufacturers receive copies of the O/S before the masses. Not to mention the developers get paid to write Windows code.

I bet 16.04.3 will be pushed back... there are so many Thursdays that it could be released... being the 10th, 17th, 24th and 31st of August.
