### Post by newbie-user on 2014-12-10
I just open a file in vi, go into edit mode and then type # as part of the text, then save and quit.

For example:
user@R16d:~$ **touch test.txt**
user@R16d:~$ **vi test.txt**

Then once vi opens:
type '**i**' to get into edit mode
type your text: **The sign # is made using shift-3 on the keyboard ######**
press **esc** to exit edit mode
type **:wq** to save and quit vi
