### Post by grahammechanical on 2016-04-07
> Certainly I am trying to install Ubuntu 12.04 in a new PC and I can't do it.

The latest version of 12.04 is version 12.04.5 and that was released 7th August 2014. If that hardware is newer than August 2014 then Ubuntu 12.04.5 might not have drivers for that hardware. This is especially true in regards to video adapters. You really should be using Ubuntu 14.04.4 if you want compatibility with newer hardware.
