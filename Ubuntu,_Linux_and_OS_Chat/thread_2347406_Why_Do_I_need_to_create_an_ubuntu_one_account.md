I am guessing that you are trying to install the snap version of VLC which will require a login to the Ubuntu One to download.

If you are trying to install the "traditional" version then install the non snap version. If you click on the VLC icon in Ubuntu Software and note the version details - "daily" is the snap version requiring login and the traditional deb package install one which doesn't need a login to the store is 2.2.2-5.
