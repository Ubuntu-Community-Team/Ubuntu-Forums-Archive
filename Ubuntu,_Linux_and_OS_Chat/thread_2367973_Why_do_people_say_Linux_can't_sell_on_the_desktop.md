### Post by ardouronerous on 2017-08-05
I've been using Xubuntu on my Gateway Desktop for five years, since 12.04, now I'm using 16.04, I tend to stick to LTS releases.  That being said, my experience on Xubuntu on Desktop has been pleasant as of late, here's my experience:

1) Xubuntu is highly customizable, unlike Windows, where you have to install 3rd party software, Xubuntu already comes with features out of the bat for customizing.
2) Finding and updating software is easier, unlike Windows, where you have to Google for software and sometimes you have to download updates yourself, in Xubuntu, the Software Center and the update manager does that all for you.
3) No system slowdowns, in my 5 years of using Xubuntu, I've experience no significant slowdowns, unlike in Windows, after a few weeks of usage it slows down and needs to be defragged, in Xubuntu, no defrag is needed.
4) Software cannot run without your permission. In Xubuntu, software cannot run without your consent, unlike in Windows, software can run in the background without you ever knowing it.
5) Faster startups and shutdowns, in my experience using Windows, starting up Windows takes about a minute while shutting down takes about 1 to 2 minutes, while in Xubuntu, starting up and shutting down takes only less than a minute, which tells me Windows has lots of background software running, while Xubuntu had less background processes.
6) Less viruses. Linux does have viruses, please do not spread the myth that no virus exists for Linux because there are, but not as widespread as Windows, in my 5 years of using Xubuntu, I do have an antivirus installed, I've never really experienced any viruses, unlike in Windows, every week I experience a virus, luckily my antivirus catches and quarantines them.
7) There is finally gaming in Linux. Although I'm not a gamer, Steam for Linux is available.

So, why do people say Linux can't sell on the desktop? Am I missing something? Because my experience on Xubuntu on my desktop has been very satisfactory, and after switching from Windows to Linux, I find Windows very limited and restrictive.

### Post by QIII on 2017-08-05
By pure accident of History, a confluence of several conditions launched Windows to dominance.  There is simply too much inertia in the Windows desktop for a sea change to occur now.

Most people associate "Windows" with "computer", their only experience being with the combination.

I don't think the Linux desktop is worth getting wound around the axle over.  Linux, the kernel at least, is installed on more devices by an order of magnitude.  Although the typical Windows user is blissfully unaware, the world now runs on Linux.  That makes me smile enough.
