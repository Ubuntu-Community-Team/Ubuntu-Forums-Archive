### Post by monkeybrain20122 on 2013-11-29
It is an entirely free (in the sense of Stallman) distro based on Ubuntu. I think it is among the very few distros that RMS actually endorses (could be wrong there, need double checking) I admire their efforts but I have the feeling that it is not going to be quite usable unless you don't need any proprietary firmware or driver for your hardware and can live without things like flash. I remember reading that they compile mplayer with some functions disabled to stay totally free (I use a flash replacement script developed by a Trisquel user so I read the bug tracker from time to time). So I suppose you may have to compile a lot of things yourself if you are used to the "Ubuntu lifestyle"
