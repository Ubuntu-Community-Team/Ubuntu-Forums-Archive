Common question: [http://blog.jdpfu.com/2014/12/28/learning-linux](http://blog.jdpfu.com/2014/12/28/learning-linux) has my answer.

Security is not a checkbox. It is a process that requires constant learning - forever. For webapps, be certain to read the OWASP top 10 lists for each of the technologies you deploy.
