### Post by newbie13 on 2015-01-14
I have almost zero knowledge of tweaking ubuntu to fit my needs so I have to use any os as it was made. And that is the reason why I still need to use windows once in 3-4 weeks.

First reason when I need windows is, when I have to copy photos from my phone to laptop. I am not saying I can't copy files in ubuntu. But windows has a tiny bar at bottom which displays number of files and folders in current directory at a glance which is quite helpful for me because I can check if I have copied all files correctly to the laptop or phone quickly. Again, there is definitely a way to do it in ubuntu but I am not aware of it yet. Will find it soon.

Second, when I have to use softwares like autocad, photoshop, etc and games. I am not sure wine can give high performance for these softwares as I get in windows.

Third, when I have to use bluetooth. Though I don't waste 4-5 min starting windows (yes it literally takes 4-5 mins to start since I have disabled fast start up, when it is active, it takes 12 seconds) just to use bluetooth, but windows is the only os where I can use my laptops bluetooth. It is rt3290, which is not supported on ubuntu. After trying for days, I could just make ubuntu detect bluetooth adapter but it cannot do anything further.

That's it. These are the only three reasons why I would log into windows. I know many people might find these reasons lame, which they really are, but I can't do much to it. Except for the first reason for which I will find a solution soon. And as stated before, I hardly use windows once in a month so it's not that problematic. But deep down inside I really want to use ubuntu as sole os without any need to use any other os.

Why I am posting this?
Just see if anyone else needs windows for some small purposes too. And to get advice/tips from experienced people who used have such problems.

### Post by SeijiSensei on 2015-01-15
I like having a [VirtualBox]("https://www.virtualbox.org/") virtual machine lying around with a copy of Windows 7 so I can run Microsoft Access from time to time.  I don't use any other commercial software like Photoshop or AutoCAD, so I don't need Windows as my primary operating system.  I wish virtualization was up to the demands of PC games, though.  I tried installing Final Fantasy XIV, but the virtualized video adapter could not meet the challenge.

If you decide to give VirtualBox a try, install the appropriate binary from [https://www.virtualbox.org/wiki/Linux_Downloads](https://www.virtualbox.org/wiki/Linux_Downloads).  For most anyone using a recent computer and Ubuntu 13.04 or later, you probably want to install [http://download.virtualbox.org/virtualbox/4.3.20/virtualbox-4.3_4.3.20-96996~Ubuntu~raring_amd64.deb](http://download.virtualbox.org/virtualbox/4.3.20/virtualbox-4.3_4.3.20-96996~Ubuntu~raring_amd64.deb).  Once VirtualBox is installed, install the [Extension Pack]("http://download.virtualbox.org/virtualbox/4.3.20/Oracle_VM_VirtualBox_Extension_Pack-4.3.20-96996.vbox-extpack"), too.

In the long-run you might want to add the Oracle repositories to your copy of Ubuntu using the instructions here: [https://www.virtualbox.org/wiki/Linux_Downloads#Debian-basedLinuxdistributions](https://www.virtualbox.org/wiki/Linux_Downloads#Debian-basedLinuxdistributions). That method will automatically keep VirtualBox up-to-date with the most recent releases from Oracle.
