### Post by EnkiduinNZ on 2022-09-16
Ubuntu 22.04 uses Wayland by default. Unfortunately, a utility to configure Wacom tablets, xsetwacom, no longer works (for valid reasons), and there is no alternative being developed under Wayland that I know of. It is possible to return xsetwacom functionality by starting Ubuntu 22.04 with XOrg instead.

I've reached out to a couple of the developers who develop the drivers and tools for use with Wacom tablets, and it appears that they don't see it as falling within their purview to provide a tool to configure Wacom tablets.

One of them suggested that I raise the issue with Ubuntu, but I feel that it would result in me being referred back to the developers, or being reminded that I could switch to using XOrg.

This is an unfortunate situation, and I can see the viewpoint of all those involved. It's a pity that a new to Ubuntu user will find that he/she can't configure their tablet under the defaults that the Operating System comes with.

But, at the end of the day, there would be very few Ubuntu and Wacom users affected by this issue.
