The current version of Mate is 1.8. Components of it have been updated a number of times, as were components of 1.6.

### Post by buzzingrobot on 2014-11-20
The *Ubuntu Mate* distribution's install images are very recent, and the 14.04 image is newer than the 14.10 image. The packages in them are the files current in the Ubuntu repos at that time, plus the files maintained and hosted on the Ubuntu Mate PPA's. Post-install updates, then, will be mimimal. I installed the 14.04 image within a few hours of its availability and I don't recall any updates being applied immediately post-install. That makes sense. More, obviously, for the earlier 14.10 image, but not a great deal.

Likewise, the current Mint Mate 17 release is some months old, so a post-install update would pull quite a few packages, but the 17.1 RC image released last week would pull in few, if any. ("Few', in my case.)

The current Mate release is 1.8.  Like KDE, Gnome, XFCE, etc., once a release is made it goes into effective maintenance mode, so any updates will be fixes, not new features. But, yes, when a *Mate* package has been updated on the distribution I happen to be using, I've received it.  They are not frequent.
