### Post by sudodus on 2015-07-25
Many people use Knoppix for repair tasks. But I think you can find many dedicated repair systems via the internet (iso files, that work from either CD/DVD disks or USB pendrives. Multiboot is one alternative, but it is quite fast to flash an iso file to a fast USB 3 pendrive (and use the pendrive as a temporary device).

Try a several of them to find out which systems you need (or combination of systems you need).
