### Post by Bucky Ball on 2016-03-17
It is much else than xfce4. If you do a vanilla minimal install (don't choose any machine profile at tasksel and install everything from the command prompt at first boot) you find you need to install xfce4 to start with. Then you have nothing but xfce4. 

I've just changed to Xubuntu-core after being a pure minimal install user for some years and there are a lot of differences. For a start, when you install the mini you end up at a command prompt. All you have is the Ubuntu kernel. The rest is up to you. There is stuff in the XB-core that I delete on install as not needed and not there with just xfce4. :)
