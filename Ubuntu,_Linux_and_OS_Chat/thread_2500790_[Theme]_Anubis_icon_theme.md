### Post by Perfect Storm on 2024-09-12

Anubis icon theme is a Black and Golden icon theme for Gnome, XFCE, Pantheon and Budgie DE with a touch of Ancient Egypt mixed with modern style. Bastet icon theme is White and Golden.
Cinnamon and Mate DE support will be added soon(tm)...

The Anubis icon theme is in heavy development, so if there's missing some icons don't hesistate to request which icons I should put in.
For request/issues/wishes: [https://github.com/SethStormR/Anubis/issues](https://github.com/SethStormR/Anubis/issues)

Download: [https://www.gnome-look.org/p/2202459/](https://www.gnome-look.org/p/2202459/)

Tested on:
Solus (Gnome/Budgie/XFCE)
Fedora (Gnome/Budgie)
Zorin OS 17 (Gnome)
elementary 8 (Pantheon)
Mint (Cinnamon/Mate)
Manjaro (Gnome)
Ubuntu (Budgie)
Pop OS! (Gnome)
