Hi everybody,              

I compiled an emulator bundle for Linux called 'Marley'. It is comprised of Mednafen, Mupen64plus, PCSX2, PPSSPP, and Dolphin, plus a launcher interface.
Emulation targets are: PS1, PS2, PSP, GameCube, Wii, SNES, Sega Genisis, Sega Saturn, N64, GBA
The project aims to resemble a gaming console. It is entirely controlled by a gamepad, everything runs out-of-the-box, and the best way to play it is from a sofa :popcorn: :popcorn: :popcorn:.

The first release is scheduled for Christmas. Now I'm looking for testers ):P. You can leave feedback and or open pull requests here:

[github.com/beaumanvienna/marley]("https://github.com/beaumanvienna/marley#readme")

Version 0.1.6 is available for Bionic and Focal in my PPA:
[launchpad.net/~beauman/+archive/ubuntu/marley]("https://launchpad.net/~beauman/+archive/ubuntu/marley")

Demo video: [streamable.com/gsjezg]("https://streamable.com/gsjezg")

Marley requires OpenGL-accelerated graphics, a gamepad, and bios files for the PS1, PS2, and Sega Saturn.

Do check it out and meet the project on GitHub. Thanks!
