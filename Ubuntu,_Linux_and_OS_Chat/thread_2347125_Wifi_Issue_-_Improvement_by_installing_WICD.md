Had trouble in getting a stable WiFi connection with Ubuntu 16.04.1 LTS on Asus Laptop X5550 (AMD A10) purchased quite recently. Applied various remedies indicated on the Web but no luck i.e. still unstable connection. However, removing the "NetworkManager" and installing WICD appears to give a more stable connection (So far so good) . I am still monitoring the WiFi connection traffic using "System Monitor". 
Information on WICD can be found in Official Documentation - Community Help Wiki - Contribute discussing WICD. I applied the instructions on that Web Page and so far it gives me a stable connection.

I would appreciate any feedback from anyone who has done the same.

PS Moved the laptop away from WiFi router (from basement to fisrt floor) and still stable connection.
