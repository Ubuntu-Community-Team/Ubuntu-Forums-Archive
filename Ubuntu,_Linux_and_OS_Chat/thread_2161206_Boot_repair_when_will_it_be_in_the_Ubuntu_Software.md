### Post by grahammechanical on 2013-07-09
It all depends on whether or not the developer has submitted an application for inclusion in the software centre. Then the code has to be checked to see if it can do something nasty to the OS. And that takes time.

And when you think about it Boot Repair and OS-Remover are powerful tools that are useful when there is a serious issue but not necessarily the sort of tools to mess around with when there is no need.

And what if you cannot boot into Ubuntu to run Boot Repair. Perhaps it is better to have them as a downloadable ISO image to be run from a live session.
