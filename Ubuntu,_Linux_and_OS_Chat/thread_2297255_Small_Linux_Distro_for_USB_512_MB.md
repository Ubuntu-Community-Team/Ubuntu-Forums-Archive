### Post by sudodus on 2015-10-02
There are different Puppy flavours, for example Wary Puppy and TahrPup. Try the other one (if you haven't tried both already). Other alternatives with iso files smaller than 512 Mibibytes are Tiny Core, SliTaz and Android for PC.

Most full-size linux distros today come with iso files bigger than 'CD size', 700 Mibibytes.

You can flash Ubuntu mini.iso onto the USB flash drive, and use it for installing for example Lubuntu into an internal drive, but it does not run a live session for you.

If you get a USB flash drive with at least 2 GB, it can host most live linux systems available, and with 4 GB I think it can host all live linux systems. If you want persistence, it is a good idea to have a fast pendrive. To get a fast pendrive, you should search among USB 3 pendrives with at least 16 GB. See this link and links from it for more details

[Installation/FromUSBStick#Prerequisites](https://help.ubuntu.com/community/Installation/FromUSBStick#Prerequisites)
