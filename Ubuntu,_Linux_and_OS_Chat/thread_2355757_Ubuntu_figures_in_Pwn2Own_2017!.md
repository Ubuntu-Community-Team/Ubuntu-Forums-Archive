### Post by vasa1 on 2017-03-16
[Pwn2Own 2017: Experts Hack Edge, Safari, Ubuntu]("http://www.securityweek.com/pwn2own-2017-experts-hack-edge-safari-ubuntu")> The Beijing-based Chaitin Security Research Lab earned $35,000 for gaining root access to a Mac through Safari. The team exploited six flaws, including one information disclosure, four type confusions and a use-after-free.

The same team also successfully hacked Ubuntu Desktop via a heap out-of-bounds access in the Linux kernel, which earned them $15,000. It&#8217;s worth noting that this is the first Pwn2Own where participants get rewarded for finding local privilege escalation vulnerabilities.[s]I guess that accounts for the kernel update today.[/s]

More here: [http://blog.trendmicro.com/results-pwn2own-2017-day-one/](http://blog.trendmicro.com/results-pwn2own-2017-day-one/)

henke54 in [this thread]("https://ubuntuforums.org/showthread.php?t=2355952") posted a link to an [eweek.com piece]("http://www.eweek.com/security/ubuntu-linux-falls-on-day-1-of-pwn2own-hacking-competition") which has this:> ... the flaw found was in the Linux 4.8 kernel used by the Ubuntu 16.10 distribution that was hacked on stage at Pwn2Own. The flaw itself was triggered by a researcher who only had basic user access but was able to elevate privileges with the vulnerability to become the root administrative account user.
