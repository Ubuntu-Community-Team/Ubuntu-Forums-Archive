### Post by Bucky Ball on 2015-06-05

Welcome. Not a support request so moved here. (Don't be shocked if another staff member moves this to ***Recurring Discussions*** as we get this question all the time.)

With some solid research you should be able to come to some conclusions. It is really about wireless and graphics card. Find out which each of the computers you mention uses then research how well they work on Ubuntu. You could have a look [here]("https://help.ubuntu.com/community/MacBookPro") and [here]("http://www.ubuntu.com/index_dell") is a write up of the XPS with Ubuntu pre-installed. 

Dell is known to be Linux friendly, generally, but some of their machines can be problematic. They've just released a whole series, though, with Ubuntu pre-installed, no Windows. They are obviously very compatible with Ubuntu.

Mac can sometimes require jumping through a few hoops and is not the same as installing on a 'PC', regardless of the model. As for the hardware, varies from model to model from what I've seen here.

You are only worried about Ubuntu running well on the machine and that is about hardware. Dual-booting is a non-issue (just takes a little more effort with Mac, think you need to use Boot Camp or rEFIND, but easy enough from what I've gathered from the users here).
