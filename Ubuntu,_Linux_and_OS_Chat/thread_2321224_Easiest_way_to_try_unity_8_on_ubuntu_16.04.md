### Post by grahammechanical on 2016-04-21
Now that 16.04 has been released there is no longer a development version. The next development version will be 16.10. First, we need a code name which is usually chosen by Mark Shuttleworth & at the same time repositories for the next version will need to be set up and other stuff. All this should happen within the next few days.

As for Unity 8, well you can install unity8-desktop-session-mir or unity8 in lxc. Both will give a login screen option. At the moment either will only run on open source video drivers. And it is a technological preview only. Do not expect it to be anything like Unity 7 desktop experience. It is a little bit like the Ubuntu phone experience.

I tried installing both just the other week and I could not get beyond the log in screen. Others have had more success. But I shall not try again.
