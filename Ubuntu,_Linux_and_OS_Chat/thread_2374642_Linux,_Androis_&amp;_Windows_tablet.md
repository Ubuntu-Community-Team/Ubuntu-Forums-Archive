### Post by Geoffrey_Arndt on 2017-10-18
Seems your best option is to have a smaller Windows laptop for the banking need.   Then, install a Virtual Machine program like "Virtual Box" and install Xubuntu on it.    That would work well provided you have a laptop with 8 GiB of ram and a fast processor (core I3, I5 etc.)   No gaming, but everything else should work.
