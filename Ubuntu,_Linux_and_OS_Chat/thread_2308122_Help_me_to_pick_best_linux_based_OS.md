### Post by grahammechanical on 2015-12-31
The Intel Celeron 560 is a 64 bit CPU. Both 32 bit & 64 bit Linux will run on it because the 64 bit architecture was designed to be backwards compatible with 32 bit operating systems and the 32 bit applications that ran on them. The 64 bit CPU became mass market before 64 bit operating systems did.

Just for the record. A 64 bit OS will not run on 32 bit CPU.

### Post by 1clue on 2016-01-01
I would recommend that, if your laptop supports it, you expand the RAM to whatever the max is.  It will likely only cost a few dollars but will show a noticeable improvement in performance.  RAM for older hardware is usually very cheap.

If, when you get it running, your system regularly hits swap, then you either need more memory or you need to trim something down, or you'll get a premature drive failure.  It's OK if it hits swap once in awhile (although I don't allow this on my hardware) but if your drive light is always or usually on then it's a real problem.
