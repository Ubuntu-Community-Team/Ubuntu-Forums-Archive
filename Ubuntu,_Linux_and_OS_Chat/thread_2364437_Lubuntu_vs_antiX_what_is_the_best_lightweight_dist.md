### Post by poorguy on 2017-06-23
Hey erotavlas,

Well I use both and like both.

For really low powered hardware Antix is the way to go.
Antix has a lot of cool useful software that is installed OOTB.

There is a little bit of a learning curve with Antix but no big deal.
Antix has a small footprint and uses little resources and on my old computers it flies.

I haven't ran Antix with less than 2.0gb of memory so uncertain how it will run only having 512mb or 1.0gb of memory.

[http://antix.mepis.org/index.php?title=Main_Page](http://antix.mepis.org/index.php?title=Main_Page)

With 512 MB or 1GB of RAM, I would go for Puppy Linux. It is lighter on ressources than both Lubuntu or AntiX and does not require any installation (can be runned on Live CD or USB flash drive).
