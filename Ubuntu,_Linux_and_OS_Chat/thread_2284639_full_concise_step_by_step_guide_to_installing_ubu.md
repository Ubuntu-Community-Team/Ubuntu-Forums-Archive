3).   Yes, this is done everyday by several million Ubuntu users.   It's called a "dual-boot" install. It is considerably easier in Linux (e.g., Ubuntu) to return your PC to it's customized condition than on Windows . . . between 1 to 2 hours in many cases.

5).  (or 2nd number 2):   Re Learning Linux - start by viewing many of the intro videos on Youtube, and also, read the Ubuntu Community wiki and documentation.   Most of it is pretty good.    Also, just ask specific questions (best is one per message) and you'll good solid advice.

### Post by Bucky Ball on 2015-07-01
Unsure what you're going on about. You have been given numerous guides and none of them, apparently, are simple, concise or clear. Perhaps you can find something [here]("https://duckduckgo.com/?q=how+to+install+ubuntu"). If you hit a brickwall, ask questions. Have you downloaded the .iso, created install media (disk or USB), booted from it and selected 'Try Ubuntu' to see how Ubuntu runs on your machine? That's easy and somewhere to start. You won't learn much unless you're willing to give it a try and that is as easy as downloading an .iso file, creating a bootable disk or USB, and booting from it.

You could try the [OBI (One Button Installer)]("https://help.ubuntu.com/community/OBI") but I have no idea about it myself. 

I suggest downloading the 14.04 LTS .iso from [here]("http://www.ubuntu.com/download/desktop"), creating install media and getting into it. If you get stuck, ask questions.

Before installing, if you get that far, you are going to need to boot Windows and create some free space to install Ubuntu on, if you haven't got any. You are also going to need to know whether your Windows is installed in UEFI or legacy mode. You need to boot into the BIOS to do this. Good luck.

(Having said all this, Linux is not for everyone. Perhaps it's not for you. Never know until you actually boot it. Once you've tried it you may lose all interest.)
