### Post by tfrue on 2014-04-29
Well, here's my two cents.

I have always avoided using IE, except from the early days, but I used Chrome mostly for its features and usability. I then wanted to be able to things that Chrome wasn't able to handle. I remember Firefox from 2005 and I hated it, but they have improved a lot since then and I mainly use Firefox for it's massive selection of add-ons that Chrome just doesn't seem to have. Another reason is Firefox seems much more configurable as you can type about:config in the URL box and you have control of Firefox.

I would say that both are much better at security and privacy than IE, but I would argue that where security and privacy fail, is at the end user level. There are plenty of add-ons for both Chrome and Firefox, such as: Ghostery and Ad Block Plus, that will help you stay even more secure and private as you browse. Both have native support for disabling tracking, so it comes down to preference and which can do what you want better and faster.  

Some of my favourite features of Firefox are: the add-ons, the awesome bar, grouping tabs, and I LOVE being able to use Firefox as a file browser, shockwave player, media player (without having to install any of those things), and anything that I need to do on a local level I can usually complete with Firefox.

Also, Firefox doesn't seem to have as much trouble with sites and it also comes pre-installed on Ubuntu . 

So I prefer Firefox much more than Chrome for its ability to do what I need it to do. There are my my two cents...

### Post by PondPuppy on 2014-04-30
The reviews I've read in the past have been pretty ambivalent about whether Firefox, Chrome, or IE are "most secure" out of the box. 

Firefox with NoScript, AdBlock, and Disconnect add-ons will be more secure than FF. Many attacks use Javascript, so blocking it whenever feasible is a good security measure. Making Flash play only with user permission is better than indiscriminate auto-launch. There are similar add-ons for Chrome, I believe, though I've not used them.

My two cents would be to configure a more-secure browser, and use it for general Web messing-about. Use another for sites which you believe are safe and which require JavaScript etc.

### Post by Tar_Ni on 2014-04-30
The obvious choice for privacy is Mozilla Firefox. Mozilla is a non-for-profit foundation dedicated to promote an open web. It was granted several awards in the last few years and none other than *The Most Trusted Compagny For Privacy* in 2013. There: [http://blog.mozilla.org/theden/2013/02/06/mozilla-is-most-trusted-internet-company-in-privacy/](http://blog.mozilla.org/theden/2013/02/06/mozilla-is-most-trusted-internet-company-in-privacy/)

As an open-source browser, it has a lot of flexibility. Since the browser belongs to anyone who want to contribute to the project, it is harder for spying agencies to snoop in it and gather your data.

[U]*''Because Firefox is open source, outsiders can not only audit the code, they patch holes in the software and distribute such changes independently of Mozilla. In other words, if there's a problem with Mozilla or Firefox, someone else can fix it and publish a new version online.''*
[/U]
 [http://www.wired.co.uk/news/archive/2014-01/15/mozilla](http://www.wired.co.uk/news/archive/2014-01/15/mozilla)

Which is not the case for Google, Microsoft, Opera Softwares as closed-source, who can all be coerced to hand over requested data.


As for security, Google, Opera, Firefox and IE all have very high standards. It becomes a matter of trust and preference at that point. Recurent security bugs left me skeptical of Internet Explorer however. Also it has become a likely target, as the leading browser among Windows users.
