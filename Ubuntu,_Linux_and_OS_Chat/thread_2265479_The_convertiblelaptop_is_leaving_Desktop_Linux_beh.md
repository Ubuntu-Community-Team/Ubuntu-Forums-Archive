### Post by grahammechanical on 2015-02-15
Unity 8 is meant to address Ubuntu phones and tablets. And from watching a few Ubuntu phone videos I am convinced that Unity 8 will work well with touch screens. The issue for laptops with touch screens (removable or not) is that Ubuntu desktop is not yet running on Mir and Unity 8. This will come out of the work being done on convergence but do not expect convergence to be complete any time soon. Not for another 12 to 18 months.

There is an x86 ISO image called Ubuntu Desktop Next and that does run on Mir and Unity 8 but at the moment all we get is the phone user interface and the phone apps. Work is being done to bring the UI more in line with what we see with Unity 7 But at the moment it is not possible to run the default desktop applications such as Libreoffice. Not without installing a special branch of the code. So, I do not see any point of installing Ubuntu Desktop Next on one of those convertible laptops. Not yet.

If you want to see how things are progressing, here are some videos:

[http://discourse.ubuntu.com/t/unity-8-desktop-mode-windows-on-mir/1986](http://discourse.ubuntu.com/t/unity-8-desktop-mode-windows-on-mir/1986)

Watch for posts by dinamic6661 on discourse.ubuntu.com. Or do what I have done and install Ubuntu Desktop Next as a dual boot.

[http://cdimage.ubuntu.com/ubuntu-desktop-next/daily-live/current/](http://cdimage.ubuntu.com/ubuntu-desktop-next/daily-live/current/)

Regards.
