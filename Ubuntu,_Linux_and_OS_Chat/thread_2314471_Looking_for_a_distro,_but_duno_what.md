### Post by user1397 on 2016-02-22
I've gotta say, I've found the latest kubuntu with the kde 5 plasma desktop to be amazingly stable.  I'd try that first and see if you like it.  It sounds like you wanna stick with something fast and easy to setup, and stable.  This rules out a lot of those DIY distros like arch, gentoo, slackware, and their derivatives.  Also, going with something like Debian might work, but it is a little rougher around the edges compared to ubuntu so I would honestly just recommend sticking with ubuntu and its derivatives given your requirements.  Ubuntu is simply the most supported desktop distro right now, and if you use an LTS version it is rock-solid stable.

Let us know which one you pick in the end!
