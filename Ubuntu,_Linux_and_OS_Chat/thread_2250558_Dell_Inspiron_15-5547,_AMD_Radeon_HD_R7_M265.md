### Post by rob107 on 2014-11-01
So I have installed ubuntu on the laptop and it all went better then I expected.
To install the AMD drivers, download the beta drivers from the AMD website and follow this guide: 
[https://help.ubuntu.com/community/BinaryDriverHowto/AMD](https://help.ubuntu.com/community/BinaryDriverHowto/AMD)
Follow part 3
Before you start update your whole system. 
And you will encounter some errors of missing packages, don't worry if you just do an apt-get install package you will be fine. 
It works great!
