### Post by buzzingrobot on 2014-03-20
Instructions for using OS X to burn a bootable image on a USB stick are here: [http://www.ubuntu.com/download/desktop](http://www.ubuntu.com/download/desktop). (right side, half-way down the page or so.)

That's for installing on Apple hardware via a USB stick.

If what you want to do is *install* Ubuntu to the USB stick and then boot and run it on the iBook, that's a different matter.  You'd need to research it, but, basically, you would burn an install image on a DVD or USB stick, attach the USB stick you wish to run Ubuntu from, boot the install image and install Ubuntu on the target USB stick.

Installing and using any Linux on Apple laptops is not always as straightforward as installing it on standard Windows hardware. Googling for the experience of others, especially  on your *specific* hardware, is likely to be useful.  Googd luck.
