### Post by grahammechanical on 2016-03-31
[https://sourceforge.net/projects/loop-aes/](https://sourceforge.net/projects/loop-aes/)

Software like this will need someone to package it for Ubuntu/Debian and then maintain the package as Ubuntu moves from release to release. To get software into a Ubuntu repository the developers would have to apply to have it included and then Ubuntu maintainers would have to audit the code. Perhaps the developers of this software do not have the time or see a need to do more than they are already doing.
