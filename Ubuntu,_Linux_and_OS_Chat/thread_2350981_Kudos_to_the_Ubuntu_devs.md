### Post by kurt18947 on 2017-01-29
I've started a position where I'm working in a Windows/Mac office.  I don't require MS Office functions - Outlook web app works fine for email and documents can be saved as .pdf. I'm not involved with core IT functions but still need to be able to print. The primary networked printer is a Xerox 42xx. I'm able to connect to their network without jumping through any unusual hoops and clicked on 'printers' just to see what came up. Hey look, there's a Xerox listed under network printers! Clicked the 'install' button, driver found and installed and what do you know, printer worked as expected.

SWMBO had a Brother inkjet whose print quality has been declining and various cleaning methods have not helped. We bought a new similar machine (MFC-J485DW), connected the power and usb cables and installed it via Brother's installer script.  Everything works as expected on the local machine. The unexpected surprise is that on another networked PC the new Brother printer installed without any action on my part (It did have another Brother inkjet MFD installed). I did not expect that on a USB connected machine without jumping through printer sharing hoops. Printing, including duplexing works as expected. What pleasant surprises!
