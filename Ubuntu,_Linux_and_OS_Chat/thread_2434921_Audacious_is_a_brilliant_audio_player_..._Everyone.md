### Post by wolftrax on 2020-01-13
I been using audacity for many years.  due to its lightweight nature as a part of a recording studio set up.  Ardour is more powerful but if one just needs to edit a few files , like samples audacity does a good job.  I never liked to use it for audio recording however.  some do use it to record multi tracks because its easy to work with but I prefer in all honesty as good digital hard disc recorder and from that to transfer files to the computer to edit and mix. 

I used sonar cubase logic pro 8 and Ardour as DAW but audacity I used a lot to create samples and edit files with before loading them into Ardour (when on ubuntu studio).  

one neat thing with audacity as I remember playing around with is the option to create your own plugins using the nyquist language.  its based on lisp and actually the first attempts to learn programming I ever did because I wanted to create my own plug ins and sounds. 

I did not use audacity for a couple of years because I don't have my recording studio set up anymore.  I sold 64 channel mixer and most of my instruments and only kept a bare minimum just in case I ever get into recording again.  

My old powermac which I used to record on using logic is also dead and I really don't want to bother setting up new studio because I will never get anywhere with it anyway other than its fun to record stuff.  not so fun when you have to buy drums several guitars and mics and bass guitar and learn to play all instruments yourself which was what I did,. 

a simple set up would be lap-top zoom h1n for recording and sound card and Audacity + Ardour and perhaps use Hydrogen as drums ,   but as said I really don't want to record anymore.  

but yeah Audacity is a good program.  many people uses it because its not too overly complicated and it works.
