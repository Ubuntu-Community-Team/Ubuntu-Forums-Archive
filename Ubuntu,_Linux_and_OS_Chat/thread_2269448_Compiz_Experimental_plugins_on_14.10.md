### Post by grahammechanical on 2015-03-16
Many Compiz plugins are no longer viewed as official and have been dropped from Compiz Configuration Settings Manager. We can get some of them back by installing compiz-plugins from the Software Centre.

When we install CompizConfig Settings Manager (CCSM) we get a default set of plugins but we may also install the compiz-plugins package to get some more plugins.
