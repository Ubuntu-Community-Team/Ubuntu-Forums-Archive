### Post by buzzingrobot on 2016-04-01
> **Jatin_kaushal said:**
> I was so happy when they announced that the crap spyware (no offence to those who support it) would be disabled by default in 16.04, but why are the teaming up with Microsoft?! Can anyone explain this to me? Perhaps you Mark if you're reading this!

Man, just so disappointed.

What this *really*  amounts to is another in a recent series of indications that Microsoft has accepted that Linux is not going away and that it is a legitimate inhabitant of the IT universe.  Companies that are Microsoft customers also use Linux and employ developers who run Windows *and* develop Linux code on those Windows machines. I.e., they need to use both Windows tools and Linux tools, often at the same time. This targets those companies and those developers. It is an effort to keep them in the Microsoft club.

Microsoft making it easier to create and integrate Linux, rather than engaging in slash-and-burn warfare against, seems to me a Good Thing.

Someone motivated by ideological concerns to leverage Linux as a tool to destroy and supplant Microsoft is on an entirely different, and, I think, futile, endeavor. Linux just needs to focus on creating new things Linux software can do and getting that into the hands of users.
