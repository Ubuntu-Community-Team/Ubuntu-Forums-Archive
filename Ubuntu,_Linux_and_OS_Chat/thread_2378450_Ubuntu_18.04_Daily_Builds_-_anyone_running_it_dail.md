### Post by ajgreeny on 2017-11-23
There is no simple answer to this question.

However, if 18.04 is the only OS you have access to it is not a good idea to install it yet.

At present it is working fine as a lot of it is still basically 17.10, but it is very early in the development and things could quite easily go completely wrong between now and April next year, and it is likely that bugs will arise on that journey some of which might make it almost unusable for many users.

Stick with 17.10 for now or even 16.04, and then update to 18.04 when it is released, or even a month or two after release when some of its bugs should have been squashed.
