### Post by buzzingrobot on 2014-12-06
I've used it quite a bit.  It looks to be a very nice release. Fedora supports a release for longer than Canonical supports non-LTS releases, but has no LTS release itself (Red Hat and CentOS target those users). Fedora releases are generally pretty smooth, but you will see more updates than in most distributions. I.e., they seldom update something like Gnome 3.14 to 3.16 during a release cycle, but libraries, kernels and such stay reasonably current.

> **Ko_Char said:**

And thick title borders and *not-pretty fonts are a big no-no for me to use Fedora regularly. To be honest, desktop environments or distros don't really matter. Everything is almost the same for me.

If by 'multi-task' you mean switch between apps on multiple workspaces you created manually, or doing the Windows thing -- minimizing windows to a panel so you can see what else is on screen -- neither of these are the way Gnome works.  Workspaces are created and deleted automatically. One empty workspace is always available; a workspace is removed when it no longer contains a window. Windows can be minimized, much the same as in Unity, if you wish. If you don't like the reliance on the hot corner, extensions can deal with that in a variety of ways.
