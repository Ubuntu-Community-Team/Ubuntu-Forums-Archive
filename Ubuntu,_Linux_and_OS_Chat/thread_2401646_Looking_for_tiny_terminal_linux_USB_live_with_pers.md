### Post by TheFu on 2018-09-20
If all you need is something to run a single 100% bash script, then any Ubuntu is overkill.  Check out tinycore.  It it 11MB [http://distro.ibiblio.org/tinycorelinux/downloads.html](http://distro.ibiblio.org/tinycorelinux/downloads.html) and boots in 1-2 sec on any BIOS-boot computer.

USB storage tends to get slow as the storage chips lifetime comes to an end.  If you don't want to change distros, try a new flash drive.
