Ok, I'll bite.

> Ubuntu Core is a tiny, transactional version of Ubuntu for IoT devices and large container deployments. It runs a new breed of super-secure, remotely upgradeable Linux app packages known as snaps &#8208; and it&#8217;s trusted by leading IoT players, from chipset vendors to device makers and system integrators.

Features of Ubuntu Desktop
> 
Enjoy the simplicity of Ubuntu&#8217;s intuitive interface. Fast, secure and with thousands of apps to choose from &#8212; for everything you want to do, Ubuntu has what you need.
One is a desktop distro. The other is for servers, usually containers or virtual servers. That would be the primary difference. They have very different purposes. Advantages for one is a disadvantage for the other for most users.  If you want a desktop, Ubuntu Core isn't likely what you want.  OTOH, if you want a lite, minimal ,server, then Ubuntu Desktop isn't likely what you want.
