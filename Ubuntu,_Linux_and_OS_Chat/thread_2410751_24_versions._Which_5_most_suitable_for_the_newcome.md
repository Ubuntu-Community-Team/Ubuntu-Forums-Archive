> **kibiras said:**

Ah, well I was mistaken about Ubuntu then, but I've only heard good things about Mint having never used it myself.  If you have Mint installed and running, then I think you should dive in head first and get acquainted with it.  Under whatever pretty face a distro is wearing lives the same basic OS.  Linux is linux, they say. :)

  I'm not offering 5 choices simply because "best" is an opinion.  I honestly feel that you'll be better served to learn the basics at which point you'll be better equipped to choose for yourself.  When I started a while ago, I couldn't get enough of learning all about this great new discovery of mine.  Now, I've settled in and I'm just an end-user who hasn't looked back in several years.

Good luck and more than anything, enjoy the ride.  There is so much to learn.  Oh, while you're on your learning curve, do yourself a favor and stick to the distro's repositories for software.  It's been extensively tested to assure good results.  It may not be the latest version of software available, but it's best to not worry about "latest and greatest."
