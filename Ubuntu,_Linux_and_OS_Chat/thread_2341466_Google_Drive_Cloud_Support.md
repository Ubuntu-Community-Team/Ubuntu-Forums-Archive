i don't think that one is syncing. it's mounting the Google drive folder i believe. and it is not a good option when storage fills up.

if you don't mind closed source and if you are willing to pay a little then look at OverGrive.

another option that seems to work for users is setting up grive2 and then syncing a folder every couple of minutes.

alternative "cloud" storage providers with linux clients available are the aforementioned dropbox, then there are the encrypted ones: spideroak and megasync (more storage than Google drive)

if you have an old PC or at least a RaspberryPi that can act as server for your cloud, then you can relatively easilly set up Owncloud, Nextcloud or SeaFile server.
