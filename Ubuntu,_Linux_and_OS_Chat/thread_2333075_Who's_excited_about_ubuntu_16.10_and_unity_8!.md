### Post by grahammechanical on 2016-08-06
It all depends on how you define success. Ubuntu 16.10 will only have 9 months support. So, 16.10 will have many, many fewer users than 16.04.

We can install a Unity 8 session on 16.04 and we can install snap packages on Unity 7. What is so special about 16.10? It may have an ISO image that installs by default the Unity 8 session as an alternative to the Unity 7 session. All that means is that we will not have to install the unity8-desktop-session through a terminal. That is no big deal.

We have been told that Unity 8 will not become the default user interface until users say it is ready to be the default. So, the developers have until the release of Ubuntu 18.04 to get it right. I am hoping that by the release of 18.04 Ubuntu will be Ubuntu Snappy Desktop Next based on Snappy Ubuntu Core + Mir + Unity 8 + transactional updates + rollback if an OS update fails + snap package applications. But will such a Ubuntu be successful? Time will tell.
