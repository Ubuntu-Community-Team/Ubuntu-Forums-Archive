How to install gwenview and gwenview-dbg 

$ sudo add-apt-repository ppa:kubuntu-ppa/backports

$ sudo apt-get update

$ sudo apt-get install gwenview

For Gwenview to work properly you must install both

sudo apt-get install gwenview-dbg

Gwenview - Image Viewer

[http://www.kde.org/applications/graphics/gwenview/](http://www.kde.org/applications/graphics/gwenview/)

Gwenview is a fast and easy to use image viewer for KDE.

Features

Supports simple image manipulations: rotate, mirror, flip, and resize

Supports basic file management actions such as copy, move, delete, and others

Functions both as a standalone application and an embedded viewer in the Konqueror web browser

Can be extended using KIPI plugins.

Gwenview Wiki

[http://en.wikipedia.org/wiki/Gwenview](http://en.wikipedia.org/wiki/Gwenview)

Gwenview is an image viewer for the KDE Software Compilation desktop environment. 

The current maintainer is Aurélien Gâteau. 

The word "Gwen" means "white" in the Breton language and is commonly used as a first name. 

While it was available in K Desktop Environment 3, the KDE SC 4 version has a simplified user interface, making it more suitable for quickly browsing through collection of images. 

Gwenview is also used to display images. It provides a full-screen interface that can be used to display images as a slide-show.
