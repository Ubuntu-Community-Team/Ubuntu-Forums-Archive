### Post by SagaciousKJB on 2016-02-18
I officially began using Ubuntu more than ten years ago, but I'm counting from when I officially converted 100%.  I started out on Ubuntu 6.06 and found the 8.04 LTS version to be perfect up until about 3-4 years ago.  That's a long time to use a version considering they released tons of uprades since then, but I always hated upgrading.  Some things would still work, other things would have newer software which wasn't compatible, the idea of a "simple" upgrade has never existed in my use of Ubuntu.  When I upgraded to 12.04 I was really happy though, so much more was available and worked right off the bat.  I have a Haugepauge tuner card for example, in 8.04 I had to compile the module myself and everything and the chances of it working correctly were largely random--sometimes it would just be a colored screeen fill, other times it would work.  Once I did get it working and started running MythTV, I had a nice little media computer and I didn't want to have to update ever again but alas, the repositories eventually stopped offering updated software and I was forced to upgrade to 12.04.  Given the amount of time I resisted, it was probably the most painful upgrade I've encountered.  Lucky for me hardware support for my card had been built into the kernel and Ubuntu had really polished up a lot of its more user-friendly configuration tools and I soon got all my stuff reconfigured.

A few days ago I finally upgraded from 12.04 to 14.04.  Mostly because I finally got a cable Internet connection and wanted to get it done before LTS support for 12 stopped and I would have to go back to compiling new software.  That isn't such a big issue but in 8.04 I got all the way to the point where there was a new libc and so compiling with the old one just wouldn't work with newer software anyway.  If there is one piece of advice I can give to people who do not like upgrading ubuntu, is just bite the bullet and do it before it gets so far down the lifeline of the distro that it's that much harder to upgrade.

14.04 is going pretty well for me, sure there's been a few little things I had to reconfigure and honestly if it weren't for my Xorg configuration file I saved I probably wouldn't have got Xorg configured corectly this time--I like to run with two separate xscreens.  Plus then other things, like ffmpeg being removed from the repositories so some of my scripts not working.  Pretty annoying, but honestly it's easier to get this stuff sorted out soon and be able to use the apt system than it is just compiling software as I need it.

Anyway I'm rambling but I just wanted to share my experience as a long time Ubuntu user. As for using Linux itself, I mean what can I say? I've not been able to use a Windows PC without it feeling like a crippled experience for the last half a decade at least.  Whenever I think of Windows now I am reminded of that Dilbert cartoon, "Here's a nickle, buy yourself a real OS." Haha okay sorry not trying to bash Windows, but seriously Windows doesn't even have bash.  Using a Windows based PC feels like I'm on a locked down grocery store kiosk, and GUIs now feel sluggish and inconvenient. Most users when first beginning to use Linux, gripe with how much you have to use the terminal; after a while you will find you're more irritated by when you cannot use the terminal.

It's amazing how far Linux has come for a desktop computing experience, and I can only reflect from the Red Hat 7 days. Package managers were new and fancy, and I downloaded manual pages on dial-up.  The OS came with a user manual 3/4" thick and showed you how to compile and install software... Boy it was weird the first time seeing a screen full of CC linkings and what not. Then you get logged in and still have to type "startx"... I'm not sure how many Ubuntu users in general know about "startx" lol

I am using my Android tablets way more often now days than I use my desktop, but it is still a reliable media machine. I am excited for the new Ubuntu tablet being released and really want to get in on the ground floor of using Ubuntu on tablets.
