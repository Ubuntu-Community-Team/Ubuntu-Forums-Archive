> **arieleoar said:**

*Should I wipe the entire disk and start from 0 with everything?

A completely fresh install will give you a clean system. It is often easier to add what you want than to remove what you do not want.
> 
*Should I wipe the Kubuntu partition to install an stable *buntu?

If there is space enough in your hard disk drive, you can dual boot or multi boot with systems side by side.
> 
*What *buntu distro is the most stable actually?

I think there are different opinions about that. Different people have different favourites. With 1.2 GB I would not install standard Ubuntu and Kubuntu. I would install a flavour with a medium light desktop environment.
> 
*Should i install Ubuntu Studio? (with low latency kernel)

You can try it :-)
> 
*Should I install a x64 *buntu? (it's recommendable with 1.2Gb Ram?)

No, the 32-bit versions use less RAM for the corresponding tasks, and with less than 2 GB RAM, that makes a difference.
