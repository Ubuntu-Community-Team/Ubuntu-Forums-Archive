### Post by QIII on 2015-09-15
This might be fun:  An Ubuntu minimal install.

Don't install anything the easy way.  Download the source code, make any adjustments you want, and compile them to roll your own.

That's pretty much what we all used to do 20 years ago or so -- only without much online support.  That's how Arch users do it now -- but with a lot of online support.

Arch users will sniff at Ubuntu saying "It's too easy."  Well, don't start out with all the bells and whistles that make it easy.  Do it just like Arch users do.

### Post by malspa on 2015-09-16
1. Install Debian Stable. It's a bit more of a challenge than installing Ubuntu or Mint, but you'll probably learn some good things. Once you get it all set up the way you want it, maybe you'll be bored again, so switch to Debian Testing or Debian Unstable. One of those ought to keep you busy.

2. Try antiX. The latest release, antiX-15, ships with various window managers and a lot of other cool stuff to check out. Good distro to run live on flash drives, too. antiX might not be one of the more popular distros, but you'll still be able to get excellent help at their forums.

3. Get some experience with a .rpm distro -- openSUSE, Fedora, Mageia, CentOS, or something else.

If all that fails to keep you from being bored:
