### Post by TheFu on 2016-12-22
I think your choice of Mate will be fine.  

I'm running it on a low-power Core i3 with 4G of RAM and it is fine - actually faster than my 1st Gen Core i5 laptop.  I wouldn't use normal Ubuntu/Unity for a number of reasons, but it should handle that well-enough too.

But really ... I'd lean non-power users towards [Chromium-OS]("http://www.howtogeek.com/217659/how-to-get-a-chrome-os-like-operating-system-on-any-pc/"). This is the base-OS used by Chromebooks, just without all the Google-stuff added.  It will be extremely fast on that box and simple is better for many people.

"passably computer savvy" can mean many things.  If the box has 30G, install all 3 versions, 10G each - chromiumOS, Mate, Unity and let them choose which they like best. Is that an option?  Then they can remove the 1 or 2 they don't prefer and move their /home/ over to the empty space.  If you use LVM, it will be easier.
