### Post by dbass81 on 2013-11-23
You guys may have heard of the Ubuntu package called Netflix-Desktop that lets you watch Netflix under Linux.  The way it works is it installs a WINE version of Silverlight and a Windows version of Firefox to watch Netflix under Linux.  I was looking for a way to watch Netflix on my Linux laptop and I have tried Netflix-Desktop, but the performance was choppy and poor on my older laptop.

I was reading on Slashdot there is a new package called Pipelight.  It works a lot like Netflix-Desktop, it uses WINE to install Silverlight on Linux, but instead of requiring you to install Windows Firefox, it works with your existing Linux-native web browser, such as Google Chrome or Firefox, and it features hardware GPU acceleration.  It also works on distros other than Ubuntu.  Check it out:

[http://fds-team.de/cms/pipelight-installation.html](http://fds-team.de/cms/pipelight-installation.html)
