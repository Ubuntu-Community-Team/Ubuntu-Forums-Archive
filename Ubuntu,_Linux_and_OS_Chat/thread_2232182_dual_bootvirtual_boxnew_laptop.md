Whether 1 or 2 is viable depends on the current specs for the system. Please post those. CPU, RAM, storage. A low-end system will struggle, but a Core i5 with plenty of RAM will handle it easily (provided Java isn't needed).

Also, there are other options besides virtualbox - but for free virtualization of a desktop OS on another desktop OS, it is really the best answer.
