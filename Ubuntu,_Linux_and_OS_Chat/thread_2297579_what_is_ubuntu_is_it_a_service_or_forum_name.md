### Post by grahammechanical on 2015-10-05
There is a code of conduct associated with the Ubuntu distribution of Linux. We are not required to live by the code to use Ubuntu. But if we want to get involved in the Ubuntu Community by joining a team we will need to sign the code of conduct to express our intention of applying the code in our association with others in the community.

[http://www.ubuntu.com/about/about-ubuntu/conduct](http://www.ubuntu.com/about/about-ubuntu/conduct)

There is a code of conduct for users of this forum that we are expected to comply with when posting on the forum.

[http://ubuntuforums.org/misc.php?do=showrules](http://ubuntuforums.org/misc.php?do=showrules) 

[http://community.ubuntu.com/](http://community.ubuntu.com/)
