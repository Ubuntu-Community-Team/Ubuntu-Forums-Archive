### Post by buzzingrobot on 2013-10-26
We often make the mistake of projecting our problems onto the entire userbase. We forget that if everyone else was experiencing the same problems, this and other forums, as well as bugzillas, would be packed with complaints about it.

If many people are, in fact, experiencing the same problem, a little research is likely to uncover those reports and, possibly, a workaround.

If it turns out that you seem to be the only person with the issue, then you really should file a bug report to ensure it's brought to the attention of the developers.  Otherwise, they won't know.

Rhythmbox is widely used as a default app in many Linux distributions, for example.  Are you certain that you are providing it with URL's that point to actual audio feeds, since many stations disguise and obscure those links?  Also, if it's only streaming radio you're after, check out the RadioTray application, which does only that, and quite well.
