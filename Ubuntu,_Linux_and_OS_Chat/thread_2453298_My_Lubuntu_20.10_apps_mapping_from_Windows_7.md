Only MS Office and VMware Workstation Pro are installed to Windows 7, the rest was portable programs,

For PDF needs, I find that Okular can not edit/add/delete "contents" where the other (Windows) apps called it "bookmarks", this is disturbing, 
I have not resolved this need yet, may be some comments? thanks!

GTK based are only: Firefox, Asbru CM, FileZilla,
The rest are QT based,

With Windows 7 I use windows tile snap often, in Lubuntu I use this solution wihout using any software: [https://itectec.com/ubuntu/ubuntu-lubuntu-19-10-how-to-tile-windows-the-window-tiling-section-is-missing-in-lxqt-rc-xml](https://itectec.com/ubuntu/ubuntu-lubuntu-19-10-how-to-tile-windows-the-window-tiling-section-is-missing-in-lxqt-rc-xml)
Virtual desktop in Lubuntu I use them right away, very useful and fast,

I did consider Fedora based, but I found out I need large community support, distro that is desktop oriented from ground up, massive software repo for desktop works, and just found out actually Lubuntu also use systemD, just like CentOS which Im familiar with, so I guess not much different to use Lubuntu,

To make the windows theme goes dark, I use kvantum manager, [https://www.youtube.com/watch?v=oxG54VF8XPo](https://www.youtube.com/watch?v=oxG54VF8XPo)
