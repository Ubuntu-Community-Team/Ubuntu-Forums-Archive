### Post by Dan Wood on 2014-11-02
Fresh install is mostly fine, execpt that my swap never got activated and my trackpad is a bit annoying now.

A swap partition was created by the installer, but not set up for some reason.
I thought it might be because I'd used full disk encryption, but I tried the install again without that, and it still didn't activate my swap space.

My trackpad used to allow me to do click and drag with a double-tap-and-hold (to click and hold) and then drag with a second finger. Now the dragging only works with the first finger. Other than that, everything else is great. I've been able to do away with using the 'legacy ROM' option altogther on my laptop now, which is nice.
