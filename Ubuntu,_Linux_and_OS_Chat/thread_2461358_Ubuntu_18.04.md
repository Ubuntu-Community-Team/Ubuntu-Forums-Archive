### Post by poorguy on 2021-04-29
I don't know anything about changing the software center from one to another however just open the terminal and copy and paste this command.

```
sudo apt install synaptic
```

Press enter and then enter your password and press enter.

When finished restart your computer.

You can use Synaptic Package Manager to download whatever software you need to install or uninstall.

I don't even bother with the software center and install and remove any software using Synaptic Package Manager.
