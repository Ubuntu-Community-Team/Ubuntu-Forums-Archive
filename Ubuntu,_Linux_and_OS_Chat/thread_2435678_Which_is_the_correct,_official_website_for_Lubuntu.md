### Post by guiverc on 2020-01-24
There are more than just the two you mention if you use a search engine and don't limit your language to english.

If unsure, I'd suggest use ubuntu.com instead of a search engine, you'll end up at [https://ubuntu.com/download/flavours](https://ubuntu.com/download/flavours) which contains the official web site for each flavor.

fyi:  [http://lubuntu.me/](http://lubuntu.me/)  is the official site, the official manual is found at [https://manual.lubuntu.me/](https://manual.lubuntu.me/), the Lubuntu forum at [https://discourse.lubuntu.me/](https://discourse.lubuntu.me/) etc
