### Post by kansasnoob on 2015-12-08
This is a snip from something I'm working on:

> Ubuntu offers the Unity desktop environment running on top of the Compiz window manager. No alternative desktop environment is available out-of-box but the [GNOME Flashback DE]("http://ubuntuforums.org/showthread.php?t=2220264"), which is used for Edubuntu's LTSP deployemnts, can be installed easily after installation with a light footprint. After installing GNOME Flashback in Ubuntu you'll then have 3 login options; Unity, flashback w/metacity, and flashback w/compiz. Of those flashback w/metacity is the least resource hungry, even less so once compositing is turned off. Since the LTS versions of both Ubuntu and Edubuntu are supported for 5 years I find that to be a good alternative lightwieght DE.

Ubuntu GNOME offers the GNOME Shell DE by default, but after installation a GNOME Classic DE is also available from the login screen. But that "classic" session is totally different than the aforementioned "flashback" session. The panels can not easily be modified, it's really just a highly modified GNOME 3 session based on some extensions but it does offer a somewhat "retro" user experience. Both GNOME Shell and GNOME Classic run on top of the Mutter window manager. Just as with Ubuntu "flashback" can be installed in Ubuntu GNOME but only flashback w/metacity will be available at login unless the user specifically installs 'compiz'.

*DO NOT install the metapackage 'gnome'! This is a common mistake people make thinking it will give them the old GNOME 2 style DE. It does NOT! It attempts to give you the basic Debian GNOME experience and often results in various levels of borkage.

There are others, some of which have unofficial re-spins:

[https://en.wikipedia.org/wiki/List_of_Linux_distributions#Third-party_distributions](https://en.wikipedia.org/wiki/List_of_Linux_distributions#Third-party_distributions)

Bohdi Linux and Peppermint are nicely done, I haven't tried any others lately.
