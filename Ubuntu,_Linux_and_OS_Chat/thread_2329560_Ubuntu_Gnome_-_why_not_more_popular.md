### Post by buzzingrobot on 2016-07-03
IMO, Ubuntu Gnome is a great implementation of Gnome.  However, Canonical's release cadence -- the six-month releases -- and Gnome's release cadence -- also six-month releases -- are about 5 months out of synch. As a result, a new Gnome release appears at the end of Canonical's release cycle, very near feature freeze. There's no time to roll that new Gnome release into the almost-ready Ubuntu Gnome release. The Ubuntu Gnome team does customarily maintain a PPA hosting packages from the new Gnome release.

For myself, I don't find the differences in the last several Gnome releases to be that significant.  It's a mature effort.
