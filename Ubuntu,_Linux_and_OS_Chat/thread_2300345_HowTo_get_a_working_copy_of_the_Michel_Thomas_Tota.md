### Post by kdavies on 2015-10-25
I wanted to learn Spanish and got the Total Spanish by Michel Thomas.

I was able to use KD3 to convert the audio files to play on my mp3 player.

Used Thunar file manager to bulk file re name. 

The last disk is a review with tests, using html and flash.

This installed ok with wine but did not run, so copied the contents from the fake c: drive to my home directory and used the web browser Mozilla.

Then used Opera, then gchrome.

I then wanted to have a desktop entry to launch a web browser in the working directory.

After several hours searching I found Arronax, all works as intended but opera issues a java error at the end of the tests.

Then I used gchrome, with java script turned off all ok and then found a useful option in the file menu...

Google Chrome web browser File > Create Application Shortcuts

This then gave me a desktop icon to start the Total Spanish review.

I managed to get this working ok but in doing so has highlighted a 'use ability' question that could be easily solved by the devs.
