What is "OPS?"

OPS is:

**(1)** (**O**pen **P**ublication **S**tructure)
**(2)** (**O**pen **P**rofiling **S**tandard)   A specification for sharing data and ensuring privacy of a user's  data.  The OPS provides a standard way of identifying which data can be  transferred from the user to a website and how it can and cannot be  used.
