### Post by ardouronerous on 2017-06-24
I'm not going to lie, my reasons for choosing Linux is swallow, it's because of economic reasons.

Around 2012, my desktop was originally running Windows Vista, the original HDD finally died after five years, yeah, Seagate sucks. I couldn't reset my PC to factory settings because that feature was tied to my HDD and my Vista DVD which came with the desktop had so many scratches on it, it was unbootable, and the warranty had already expired, so the only option left to me was to purchase a new Windows DVD, either 7 or 8, however, I couldn't afford that, so I searched Google on my dad's laptop which was running Windows XP and discovered Xubuntu.  So I bought a new WD HDD and installed Xubuntu 12.04 on my desktop, and after 6 years of using Xubuntu, sticking to the LTS releases, I like it better than Windows.

Recently, my dad's laptop which was running Windows XP, the HDD died and I replaced it with a SSD and installed Lubuntu 16.04. So my family is officially off Windows.
