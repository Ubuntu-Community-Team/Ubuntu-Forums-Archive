### Post by Shibblet on 2018-08-12
Got a new computer the other day, and like all good computer users, I decided to clean-install the operating system in order to get rid of all of the bloatware that comes pre-installed.  So, I grabbed the drivers off of the web-site, and loaded up a copy of Windows 10.

The installer didn't take too long, only about 20 minutes to get through the basic portion of it.  Then it started to ask me questions about the things I wanted to have activated.  Privacy questions mostly, and I opted to turn them all off.  Then it asked it I wanted to connect my phone, to which I replied "I'll set it up later."  The same answer applied to my use of Microsoft's Cloud Storage service called "One Drive."  Again, I'll deal with it later.

Finally the computer rebooted, and started up for the first time.  I know this because it told me in a sleek looking font "Getting your computer ready for the first time."  After about 5 minutes of waiting while the screen pulsed blue to black, I was finally greeted with a giant blue emblazoned and electrified Windows 10 Logo background, and the start screen.  This being Windows, I knew just what to do.  I opened up the options panel, and went straight to applications, to start removing the ones I wouldn't use.  I had to remove these first, otherwise the Windows Update process would upgrade them to the next version.

This took about 15 minutes to do, and some of the programs I would NEVER use, are unfortunately uninstall resistant.  I went into the Control Panel options and started removing Windows 10 Components that I would never use, such as XPS and Internet Explorer.  This took around 15 minutes, and required another reboot.

After the reboot, I ran the Windows Update to get the latest security options downloaded.  Now, I have pretty fast internet.  It's not Google Fiber, but I average around 900Mbps Download and 10Mbps Upload.  Let's just say that Alaska is quite a distance away.  Anyway, It took a good 45 minutes to get all of these updates downloaded.  It took another 30 minutes to install all of the updates, along with two reboots in the process.

Finally Windows is set up.  Now to download my graphic drivers, my utility programs, and the applications that I use.  Firefox Browser, Nvidia Drivers, CCleaner, VLC Media Player, SSD Utilities, Keyboard Utilities, Virus Scanner, etc.  This process took around 30 minutes, only because I know what I am after.

Now that everything is set up and running, I do a complete virus scan of the HD.  This takes a good 60 minutes.

Now, I make a backup image of the HD.  Fortunately, this only takes about 10 minutes.

Done.  Fin.  Usable Windows Computer.  20 Minute Install + 5 Minute Wait (While we set things up) + 15 Minute App Removal + 15 Windows Component Removal + 45 Minute Update Download + 30 Minute Update Install + 30 Minute Application Install + 10 Minute Backup = 170 Minutes - 2 Hours 50 Minutes

Now, many of you may think I am exaggerating, and it's a bit true that the times are rounded to the closest 5 minutes.  So, some are rounded down, others rounded up.  Etc.  It still averages out just shy of 3 hours, start to finish.

The only reason I have Windows 10 on this computer, is, of course, I use Adobe and I play Games.  Some of my Steam games are available for Linux, but many others are not.  I did however get a Beta key for GeForce Now, and have sent word to Nvidia (through the application) that they need to make a Linux Client for this service.  +1 Linux Community

Anyway, I partitioned my HD for a Kubuntu install, and loaded up a USB Stick to install the OS that I really wanted.

Booted right into the stick, and started up the install.  The entire install process took about 10 minutes.  I tend to choose the "Minimal Installation," but I also choose "3rd Party Drivers, and Install Updates."  Rebooted, and picked Kubuntu from the Grub Menu, and in about 10 seconds, I was at the Login Screen.

First thing Kubuntu did, was tell me I had updates.  So, I hit the Download All button, and in about 5 mninutes they were all downloaded and installed.  I utilized this 5 minute gap to set up KDE the way I like it.

I then opened up the Hardware Driver Manager, it took about 30 seconds to recognize my Nvidia Graphics Card, and I checked the new 390 Drivers.    The download took approximately 1 minute.  I then blacklisted the nouveau driver, and rebooted the computer.

After a reboot, I saved my xorg.conf file for future boots, and opened up Discover Software Center to download Chromium Browser, Kmail, and Steam.  Each of these programs take about 3-5 minutes to set up to my specifications.  So, let's just say 15 minutes all together.

So, then I did a drive image backup of the partition, so that I wouldn't have to do all of the setup again, and because it uses so much less space, it only took about 5 minutes to do.

I then hook up my phone with KDE Connect (half a second,) and I am up and running with a full OS install of 30 minutes.  10 Minute Install + 5 Minute Updates + 15 Minute App Install and Configure + 5 Minute Backup = 35 Minutes.

The process of setting up Kubuntu 18.04 vs. the Process of Setting up Windows 10 is almost 5 times faster.  It took 4.8 times longer to set-up Windows 10, than it did for Kubuntu.  I understand that a lot of this process is getting the computer to your personal preference, and this can be done over time.  However, I think we all like to set things up the way WE like them, instead of just doing a standard "slap-n-go" install of the OS.  In which case, 10 minutes for Kubuntu, and 25 Minutes for Windows 10.

Well, I hope you all liked my little story, I know it's a LOAD of garbage!  

Thanks!
