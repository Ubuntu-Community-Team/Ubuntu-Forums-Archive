### Post by ilikecolors on 2013-12-15
I recently made the migration to Linux. Although I still have some problems, I have found myself quite comfortable here with Ubuntu. Anyways, although this might take you a minute to do, I pondered over this for a while and finally got it to work, as simple as can be, and it made me smile. I finally installed "Mari0" on my computer and would like to share the game and how to install it with you all. :D
Mari0 is a cross between Super Mario and Portal, or Portal and Super Mario. You can either play through Super Mario with a portal gun, or some 2D Portal with Mario. Its so very customizeable and free DLC and a simple level maker, and its completely free.
So if you want to install it, first you have to go into the Ubuntu Software Center. Look up "Love engine". It should look like a bouncy blue ball. Install that.
Next, go on [http://stabyourself.net/mari0/#download](http://stabyourself.net/mari0/#download) and scroll down. Download it for Linux.
Wait for it to download. Once it does, extract it. Open the new folder and open mari0_1.6.love
It will look like a cardboard box if you don't have Love installed and won't run at all, so make sure to do that!
K, bai! \\:D/
