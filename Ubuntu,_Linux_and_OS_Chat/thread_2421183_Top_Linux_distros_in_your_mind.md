### Post by Shibblet on 2019-06-18
Legitimately, Kubuntu.

I am a big fan of the Ubuntu Flavor Family, because it is Debian Based, and the PPA system.  Even Snaps work without problem.
I also prefer KDE, and the new "Minimal" install option available.  But all of the other Ubuntu's (Ubuntu-Budgie, Ubuntu-MATE, Xubuntu, Lubuntu, etc.) work fantastically as well.

I have heard great things about Arch, and Manjaro, but they seem to be more focused on system building than I personally prefer.  If that's your thing, more power to you.

### Post by Tadaen_Sylvermane on 2019-06-18
I personally use Ubuntu for everything. But best distro for me is Debian. I've seen it said that Debian is so stable it hardly breathes. While I haven't had much issue with Ubuntu updates, Debian package updates are so minor they don't change anything. The system never changes if you don't want it to. Their security team seems to be a bit ahead as well, far fewer security patches meaning the software was gold when it became stable.
