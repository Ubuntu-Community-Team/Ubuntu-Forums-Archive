### Post by oldrocker99 on 2014-12-09
I have't used Fedora and its derivatives, but I've used several Ubuntu derivatives, and I always come back to Ubuntu; I liked Mint for a while, but found some annoyances with it, although major props go to Mint for being the first to introduce and help develop my favorite DE, MATE. I do not like the default Unity interface (among a boatload of other people), but (1) it is liked very much by new Linux users, and (2) it is a very advanced DE, with features **no** other desktop, including OSX and certainly Windows, has.

So welcome to the wide world of Linux, which I've been using for 6 1/2 years. It's only gotten better as long as I've used it, and zero malware all that time. I did dual-boot so I could play Windows games (and nothing else with WIN7, which I found ten times more annoying than Mint ever was), but, after Desura, and Steam, why bother with an OS that "cannot install updates while Windows is running," while Linux, when installing updates which are running, **stops** the task, installs the update, and restarts that task. What a concept. There's no fragmentation of hard drives like Windows always has done, either. Being introduced to what many have discovered is simply the best OS can be a real joy and once again, welcome. 

And, if you have questions, these forums are **the best** source of getting them answered. This is a community, living up to Ubuntu's name ("I am who I am because of the people around me" is my interpretation of the word) and philosophy.

WELCOME!

### Post by Bucky Ball on 2014-12-15
Got it, but you're still not quite there. Application Shortcuts tab is what you want. Then 'Add' to add a shortcut. Stick in the command to launch the program. An example for a shortcut for Firefox:

Add;
Type in 'firefox';
You will then be asked for a key combination;
Hold down the Superkey and hit 'f'.

Now, when you hold down the superkey and 'f', firefox will launch.
