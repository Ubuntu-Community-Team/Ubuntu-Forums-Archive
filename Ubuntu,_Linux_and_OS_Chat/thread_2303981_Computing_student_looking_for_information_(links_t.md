### Post by TheFu on 2015-11-23
No, Ubuntu is not a "time sharing" system, though it can be used in that way.

All GNU/Linux systems are basically the same, until you get to the package manager and GUI used.

The types of questions you are asking are not Ubuntu specific. Best to go with answers from kernel.org on those. Expect to read C source code for many of the answers.

If you want to learn Linux, this is my best recommendation: 
      [http://blog.jdpfu.com/search/?q=learning+linux](http://blog.jdpfu.com/search/?q=learning+linux)
It is not Ubuntu specific, but there is some ubuntu links for basic stuff.

help.ubuntu.com
wiki.ubuntu.com

Google found these reputable sources:
[http://www.tldp.org/LDP/tlk/mm/memory.html](http://www.tldp.org/LDP/tlk/mm/memory.html)
[http://www.tldp.org/HOWTO/KernelAnalysis-HOWTO-7.html](http://www.tldp.org/HOWTO/KernelAnalysis-HOWTO-7.html)

Those sites are reputable.
