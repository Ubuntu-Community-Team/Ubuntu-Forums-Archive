### Post by grahammechanical on 2015-10-14
I am running an Android app in Ubuntu. So, it can be done. We need three things 1) Google Chrome web browser; 2) A Google app installed from the Chrome web store called ARK Welder; 3) the apk file of the application or a zip file containing the Android app.

[http://www.howtogeek.com/214734/how-to-use-googles-arc-welder-to-run-android-apps-in-chrome/](http://www.howtogeek.com/214734/how-to-use-googles-arc-welder-to-run-android-apps-in-chrome/)

There is one thing I have learned from using ARK Welder. Every time I "test" the app ARK Welder offers to remove the app. It is best to accpt this offer. When we click "Cancel" the app will load but each time we run it a copy is stored on the hard disk and we will soon start getting out of disk space messages. That is the situation I am in now. Clicking "Remove" will, I guess, remove that last tested and stored app. It will also still load the present app to be tested. Also running the app maximised will work most time whereas running it tablet size sometimes fails to load. ARK Welder is beta software after all. And it is meant to be run on Chrome OS.
