The USB/IP project which enables virtual local/remote USB devices, is in the linux kernel, and looks like it could be quite useful


You can read about the USB/IP project and how it enables virtual USB devices remotely at the USB/IP project: [http://usbip.sourceforge.net/](http://usbip.sourceforge.net/)


I found a tutorial for using USB/IP with Ubuntu here: [http://www.linux-magazine.com/Issues/2018/208/Tutorial-USB-IP](http://www.linux-magazine.com/Issues/2018/208/Tutorial-USB-IP)

Wanted to share the info as I had been unaware of it...
