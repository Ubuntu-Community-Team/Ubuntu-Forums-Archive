### Post by mooreted on 2014-07-03
Also Xubuntu would work well. The problem with Unity is that it uses hardware 3D acceleration for effects, animation, transparency which need a fairly decent video card. Since you have an older AMD graphics adapter, you won't be able to install newer drivers for it. But with Lubuntu or Xubuntu you won't need 3D acceleration to run the desktop so it should work fine.
