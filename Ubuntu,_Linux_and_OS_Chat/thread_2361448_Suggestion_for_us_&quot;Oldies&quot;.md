Hi.

I've always had vision problems. I had eye-surgery in Dec/17, (not for cataract-lense replacement, like you are going through). I had a bleed in my right-eye that caused it to lose much of its vision. I'm old too!

A couple of things that help me, that haven't been mentioned by others are:

- My 'new' optometrist has me trying different prism lenses that correct problems with discerning proper "vertical de-placement" (and horizontal). You may want to ask about that at your next Optometry visit. (My previous optometrist didn't even suggest them after my vision loss. I was lucky I went with 'another opinion' to another optometrist and she said, 'let's try prisms', and it has helped, (a lot), 

- I find using the inverse display makes things clearer for me. (It can be turned on and off a number of ways. If you want to try it, a simple way is using xcalib. I use it on startup and if I want to view without it, I reverse it by command key-press settings. You can try it like this if you wish:
```
sudo apt-get install xcalib

and

sudo apt-get update

```
And then to inverse or return to normal in terminal:
```
xcalib -i -a 
```
Full command.[Ubuntu Manpage: xcalib]("http://manpages.ubuntu.com/manpages/xenial/man1/xcalib.1.html")
```
xcalib -inverse -alter
```
HINT:

If you leave the terminal window open in your panel or wherever, you can go back into that terminal and hit the arrow key 'up' to show the command again. If you hit enter, it will reverse what you last did for display.

I hope this may help.
