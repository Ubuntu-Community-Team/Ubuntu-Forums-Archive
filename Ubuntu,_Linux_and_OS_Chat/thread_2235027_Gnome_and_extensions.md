### Post by d-cosner on 2014-07-18
Some of you know that I have been looking at various desktop environments trying to find the ideal set up for the way I want to work. First I have to say they all have their good points and I like many of them for all different reasons. Gnome Shell, Unity and Xfce seemed to capture my attention more than the others though. I tried a variety of distros with all of these desktop environments.

I kept going back to Gnome Shell, something about the mix of the old familiar style with a modern twist just seemed appealing to me. I had read about the simple dock extension and a friend had also mentioned it to me. I decided to go ahead and install the simple dock extension and give it a try. I ended up liking it almost instantly!

After installing simple dock I realized that I needed to fine tune things a little to make the dock more usable. The message tray interfered with the dock so I installed insensitive message tray. Messages themselves were covering the dock so I installed panel osd. After installing these extensions the dock was very usable and it was saving me time by not having as many mouse clicks to do things.

While looking through the extensions I came across one that was called Flash fullscreen fix. My system has Intel graphics and has had some problems with full screen flash and I thought, why not give it a try? It does seem to help even though I have no idea how it fixes anything. I installed the recent items extension too to keep things cleaned up.

When I was all done installing these extensions I realized just how customizable Gnome Shell is! I had completely changed much of it's default behaviour and it was even more pleasant to use than before. I ended up having all of the features I really wanted without the things I did not want. Something else I got to thinking about was by having these features in Ubuntu Gnome 14.04 I would not have to worry about the extensions not working because of an update to Gnome Shell.

So now I have all of the features I want and I have a decent length of support! I still have a rolling release on my extra drive to play with but my main drive and my laptop are all set for the foreseeable future. When the Gnome developers said they were going to make Gnome more modular they weren't kidding! I knew about Gnome extensions but had never taken them too seriously, now I understand the value of them and where Gnome Shell is headed.

Because of the extensions and the way Gnome Shell is built, I was able to take it and make it my own so to speak. I do not feel much of a need to have the most current version of Gnome now, not that it is a bad thing but staying with a version will keep the installed extensions working. It was fun experimenting for a while but it is nice to just sit down and work on my music collection and other things and have my system set up the way I want it.

There has been at least one issue with extensions in the past - compatibility.  For instance, an extension that worked perfectly in gnome 3.8 may not work at all in 3.10.  Sometimes a simple file mod fixed it sometimes not.  There is a default package included in gnome-shell but many of my favorites are not in that group.  I and others rely in the extension's developer(s) to keep it compatible with the latest gnome versions.  I did install gnome's 3.12 PPA for 14.04 gnome and my extensions were all present and functional so I guess progress is being made.
