### Post by deadflowr on 2016-04-29
The package can be installed any which way all caps and lower case, or a mix of both, but apt-get and sudo need to be lower case.

I don't think most user would even consider capitalizing the Package name, unless they deemed it necessary.
Hitting shift is too much extra work, imo.
