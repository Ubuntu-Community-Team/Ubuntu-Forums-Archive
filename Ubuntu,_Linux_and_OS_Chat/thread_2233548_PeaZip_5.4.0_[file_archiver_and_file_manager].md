### Post by giorgiotani on 2014-07-09
I'm glad to announce that PeaZip 5.4.0 was released yesterday, with a major GUI update, new option to queue deletion of archives after extraction or deletion of files after archiving (quick and secure deletion), and ZPAQ backend updated to 6.54; full changelog: [http://peazip.sourceforge.net/changelog.html](http://peazip.sourceforge.net/changelog.html)

Generic DEB packages are featured on the website for download, help is welcome for creating distribution/version-specific packages.
Portable packages can be used to test the application without modifying the host system - or to be used from removable devices or shared network units without installation.
Linux x86 [http://peazip.sourceforge.net/peazip-linux.html](http://peazip.sourceforge.net/peazip-linux.html)
Linux x86_64 (portable and no-architecture DEB package) [http://peazip.sourceforge.net/peazip-linux-x86_64.html](http://peazip.sourceforge.net/peazip-linux-x86_64.html)
Linux ARM (experimental, user contributed) [http://peazip.sourceforge.net/peazip-linux-arm.html](http://peazip.sourceforge.net/peazip-linux-arm.html)

Help is also appreciated for translations in order to make the application available for non-English speaking users, [https://sourceforge.net/projects/peazip/files/Translations/](https://sourceforge.net/projects/peazip/files/Translations/)
