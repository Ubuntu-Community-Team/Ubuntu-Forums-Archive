### Post by anon_private on 2014-08-01
Out of Interest:

If Ubuntu distributions are more or less the same 'under the bonnet'. If someone installed a lighter version, and then installed the more sophisticated applications, would they be any better off than installing Ubuntu in the first place?

---

### Post by Impavidus on 2014-08-01
The largest difference is the desktop environment. If you start with a light flavour and add some sophisticated applications but keep the light desktop environment, your system will still be lighter than a default Ubuntu. If you also install the heavier desktop environment the result will be like you installed the heavier flavour at once.
