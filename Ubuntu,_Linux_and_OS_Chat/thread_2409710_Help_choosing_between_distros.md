### Post by oldrocker99 on 2019-01-05
I personally find Ubuntu MATE to be ideal for new users, and experienced users alike. It's lightweight, very configurable, and easy to use. Ubuntu MATE has its own curated software installer, and installation of drivers and firmware is easy.

MATE is available for other distros, but MATE development is handled by the Ubuntu MATE team.

AFAIC, it is the very best distro I have ever used.

You should **never** have to pay for Linux. Go to the distribution's website and download it for free, as a torrent (which is quick) or as a a standard download.

The .ISO you download has to be burned as a bootable USB. Copying the file to the USB won't work. Go here:[https://rufus.ie/en_IE.html](https://rufus.ie/en_IE.html). Rufus is pretty much the best USB burner for Windows, and it's, BTW, available for Linux as well.

Go here to download the Ubuntu MATE ISO: [https://ubuntu-mate.org/](https://ubuntu-mate.org/). Burn it, boot with it, and you'll have the whole system to examine. It doesn't touch your hard drive unless you select "Install."

Have fun!

### Post by yetimon_64 on 2019-01-11
As a regular user of xfce/xubuntu, ubuntu unity (on Trusty 14.04) and ubuntu mate (on Bionic 18.04), for a new user I'd agree Ubuntu Mate is a good starting point. I have many other DEs installed but wouldn't recommend any of them, or distros using them, to new users as they could not be regarded as new user friendly environments.

As mentioned by oldrocker99, Ubuntu Mate is very good either for new starters or experienced users alike. I originally started on Ubuntu in late 2007 on the old gnome2 interface; the Ubuntu Mate release is virtually identical in function to the old gnome2 interface but uses newer libraries etc. This all makes for a very functional system which is easy to learn on, well worth a tryout for you IMO.

### Post by heroquestelf on 2019-01-16
I use Ubuntu MATE on both my laptop and my desktop, and both are older machines. MATE does not require a lot of memory and I like it way better than the vanilla Ubuntu. 

My daughter has a high end laptop, and it's running Kubuntu (KDE). Kubuntu LOOKS stunning, but it takes a lot longer to load and it eats a lot more memory. Still, if you want a cutting edge desktop environment and you got the power to push it, Kubuntu is a good build. 

I've run all the major flavors of Ubuntu, and of them my least favorite is Xubuntu. However, if you have an older machine, it is a great distro to run. It is clunky as heck and does not look real shnazzy, but it can run on lower end systems and has really **super** memory allocation.
