### Post by Lars Noodén on 2015-02-17
No Starch has print copies of [The Linux Command Line](http://www.nostarch.com/tlcl.htm) available.  There is also a CC PDF for download.  PDF is ok as an accompanyment to the print, the print is easier to learn from IMHO.  Part 1 goes through some basics and then part 4 goes into the details of making shell scripts.

### Post by tgalati4 on 2015-02-17
I find old Linux and Unix books at used book stores.  Since the operating systems have been around for so long, even the old books are useful.  When an old command fails, it's a useful learning exercise to research what went wrong and update the book in a different colored pen.  Sometimes the switches have changed, sometimes the directories change where configuration files are kept change, but 80% has not changed.

I also have several No Starch Press and OReilly books.  If you register your OReilly books on their website, you can often get PDF updates of the older titles, which is a nice service from a technical publisher.

[http://www.nostarch.com/catalog/programming](http://www.nostarch.com/catalog/programming)

[http://shop.oreilly.com/category/browse-subjects/programming.do](http://shop.oreilly.com/category/browse-subjects/programming.do)
