[*]**America** &#8211; Starting July 2001, The White House started migrating the computers to the Linux-based [Red Hat]("http://www.linuxfederation.com/linux-everywhere/www.redhat.com")Linux and [Apache]("http://www.linuxfederation.com/linux-everywhere/www.apache.org") HTTP Server and the migration was completed in February 2009.
