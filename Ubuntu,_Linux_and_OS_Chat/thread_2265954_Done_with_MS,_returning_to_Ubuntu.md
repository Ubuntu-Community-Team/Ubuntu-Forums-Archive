### Post by fwrun2011 on 2015-02-18
I had a dual-boot system with Ubuntu 12.x (I think) and Windows 7 Ultimate. After a while, I found that I wasn't using the Ubuntu much. That was because most of my apps run in Windows. Stuff that I spent a lot of money on, and didn't want to give up.
It was too much of a hassle to switch between Ubuntu and Windows when I wanted to do different thing.
So I removed the Ubuntu partition and returned to a Windows only system. Bad mistake!

Now, I find that yet another piece of malware has somehow slipped past the almighty Norton 360 and gotten onto my Windows system. While this little bug hasn't had a chance to manifest into anything destructive - because I must have caught it soon enough - I have decided that enough is enough. Having to deal with Microsoft's vulnerabilities, almost daily updates to fix these vulnerabilities, only to have more pop up the next day.
So I will re-create my dual-boot system, installing Ubuntu 14.04.1 64-bit. I will use the Ubuntu for everything except what I must run on Windows. I still need MS Access 2010, and maybe Excel, but I believe that Open Office has a spreadsheet that I can import .xlsx files into. I don't know about Access, but if I can find an open-source database that will allow me to create relational databases like I do in Access, then I can give up Access as well.
I also have some games for Windows that haven't been coded for Linux yet. So there will still be the necessity to keep that dastardly naughty Microsoft Windows for a while longer.
I can say one thing though: I will never purchase another Microsoft Windows product - or any other Microsoft product for that matter.

And one more thing. Being that I have just moved to an Android smartphone, I will have lots more options for music players in Ubuntu. I plan to become familiar with Java so I can create apps for that Android phone. In the mean time, I will be happy to say goodbye to iTunes - another thing that has caused me pain over the past 10 years!

Sure, there is a lot to learn with Ubuntu Linux. Maybe some things will take a bit more work than they did in Windows. But hey - I've got a brain, and I know that it is more than capable of learning anything I need and want to learn with Ubuntu!

I'll be coming back here often with questions, and hopefully will be able to help newer users with their first steps into Ubuntu!

Happy Ubuntu-ing

FW

### Post by PhilGil on 2015-02-19
If you have at least 4GB of RAM you might want to look into virtualizing your Windows install using VirtualBox or VM Ware player. This allows you to run Windows inside Ubuntu without rebooting. Office programs should run fine in a virtual machine, and Win 7 will be happy with as little as 1GB of RAM (although 1 1/2 or 2 is better).

LibreOffice Calc should serve most of your spreadsheet needs (unless your Excel workbooks use VBA code or have complex formatting). There really isn't a Linux database program that's comparable in functionality to Access, but you can give Kexi or LibreOffice Base a try and see if they meet your needs.
