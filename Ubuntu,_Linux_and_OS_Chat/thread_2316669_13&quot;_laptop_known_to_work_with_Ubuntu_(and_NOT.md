### Post by kevdog on 2016-03-14
You need to know the components within the machine -- like chipsets.  My recommendation is to stay away from NVDIA GPU, in fact if possible I'd go with Intel graphics if possible. Broadcom chipsets for networking work, but many are difficult to configure.  Almost anything can work if there is a driver available. Go for ones in the mainline kernel if possible.
