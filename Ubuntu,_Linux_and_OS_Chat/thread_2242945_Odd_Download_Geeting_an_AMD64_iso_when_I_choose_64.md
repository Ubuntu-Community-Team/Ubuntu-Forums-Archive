### Post by ssam on 2014-09-05
Most Linux distros use the term 'amd64', to mean the 64bit version of the x86 architecture developed by AMD, but now used by AMD, Intel and others. Every 64bit consumer Intel chip (and most of their server chips) use the amd64 architecture. Intel did create a distinct 64bit version of the x86 architecture for its Itanium server chips, but it did not catch on. That is usually known as IA64. See [https://en.wikipedia.org/wiki/AMD64](https://en.wikipedia.org/wiki/AMD64) for more.

### Post by oldrocker99 on 2014-09-05
32-bit programs are called "i386" in honor of Intel's 386 CPU, the first 32-bit processor offered to the public. 64-bit programs are called "AMD64," in honor of the first consumer 64-bit processor. They can be installed on **any** 32- or 64-bit CPU, regardless of manufacturer.
