It's already handled like this - LTS Hardware Enablement Stack ([https://wiki.ubuntu.com/PrecisePangolin/ReleaseNotes/UbuntuDesktop#LTS_Hardware_Enablement_Stack](https://wiki.ubuntu.com/PrecisePangolin/ReleaseNotes/UbuntuDesktop#LTS_Hardware_Enablement_Stack)) 12.04.2 which got released with kernel 3.5.XX (12.04.1 come with with 3.2.XX) and next LTS (12.04.3) will come with a 3.8.XX kernel and if there are requests for newer hardware drivers they can get backported and rolled out.
