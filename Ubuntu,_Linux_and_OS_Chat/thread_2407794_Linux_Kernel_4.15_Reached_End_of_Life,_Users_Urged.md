The 4.15 kernel that went EOL is upstream, the 4.15 kernels used by Ubuntu are maintained by Ubuntu until 2023.  The upstream kernel support and Ubuntu kernel support are two different things

The Linux 3.13 kernel went EOL in 2014, however Ubuntu's 3.13 kernel is still supported until April 2019
