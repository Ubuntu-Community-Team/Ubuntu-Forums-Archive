### Post by drfox on 2016-05-12
For most users, a PDF viewer is all we need, and there are several good FOSS products available, but...
AFAIK, there is no good freeware version of software that can do what Adobe Acrobat does: create and edit PDFs, add comments, use a typewriter tool, add and delete pages, etc.
Qoppa has just released PDF Studio 11. It's not FOSS--it is a commercial product, but it supports Linux (as well as Mac and Windows).
I've been using PDF Studio for years and highly recommend it. The price is reasonable, and a license allows you to install on 2 separate machines.
You can check it out at [https://www.qoppa.com/pdfstudio/]("https://www.qoppa.com/pdfstudio/")
Hope this helps.
Larry

### Post by David_Wright on 2016-05-12
**Foxit Reader** for Linux (relatively recently released) has a very good feature set compared to other Linux pdf viewers, although not as comprehensive as its Windows version.

For more complex editing you could try **Master PDF editor**, which I have installed but only briefly played with.  It offers much more comprehensive pdf editing.

Both are **free**.  Unfortunately neither has a repository yet (as far as I know) but can be installed via the deb or other download from their respective websites.

### Post by Bucky Ball on 2016-05-13
I'll throw this in here, although perhaps not exactly what you're looking for, but I use PDFShuffler for arranging PDFs. At uni, I sometimes need to fill out PDF forms which then need further documents attached or a page signed an put back in. That can all get awfully messy.

With PDFShuffler, you can simply drag any page from any PDF file and plonk it where you like. So you can mix pages from multiple PDF documents and arrange them in whatever order you want (for instance, remove the original, blank signing page and replace with the signed, scanned PDF page). Neat! And saves me a heap of time when I need to use it. :D

Just thought I'd mention. Should be in the repos.
