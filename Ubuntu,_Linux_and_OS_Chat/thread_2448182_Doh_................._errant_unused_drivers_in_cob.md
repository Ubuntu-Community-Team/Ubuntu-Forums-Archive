I just thought I'd share a funny story. I pulled out an unused PC, I think it's been about 2 years since it got stuffed into the corner where all old stuff gets stacked, just long enough no memory resides as to what exact circumstances I last used it under. It had a dual boot of Win 10 and Ubuntu 18.04 so I updated 18.04 and all seemed well until I tried opening gnome-control-center which was just a no-go.

Turns out I'd apparently used an nvidia graphics card in it when last used but that had been removed and I was now using the on-board Intel graphics. Just purging all nvidia packages solved the issue, but not before I tried a dozen other things - DOH #-o

I always tell myself I should throw a post-it note on such things before they get hidden but never do, what fun would there be in that?
