### Post by darkon-turas on 2018-04-02
It occurred to me that the [Brave]("https://www.brave.com/index/") web browser, if ported over to Linux, would make for a natural partner for Ubuntu. Both properties have an African identity, use a similar color language for their logos, and are open source. Linux could use a privacy-centric browser offering. Canonical might even be able to assist in the effort by sending an expert over there and help to fast track a Linux port.

```
snap find brave
