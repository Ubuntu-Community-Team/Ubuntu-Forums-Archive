### Post by vasa1 on 2014-11-16
The DOM Inspector offered by default from [here]("https://addons.mozilla.org/en-US/firefox/addon/dom-inspector-6622/?src=search") doesn't allow elements to blink with a red border when selected in Firefox 33. To get back that feature one currently can install the .xpi linked to in [here]("https://bugzilla.mozilla.org/show_bug.cgi?id=957149#c6").

I got this tip by reading [Dom inspector element blink not appearing in Fx 33]("http://forums.mozillazine.org/viewtopic.php?f=7&t=2881061").
