### Post by TheFu on 2016-09-03
The GUI isn't the OS. It is just a GUI and you can try 50 different GUIs on the same OS installation, if you like. Takes about 5 min to install a new one. HowToGeek has a how-to for this.  The only important thing to know is that configuration settings are stored in your HOME and sometimes these conflict between the different GUIs.  That means the easiest way to try a new GUI is with either the guest account (dynamically created at login) or using different userids/accounts for each.  After all, Linux **is** a multi-user OS.
