### Post by Cliff_Simonds on 2014-08-11
Something I've really noticed lately---
I'm using an ASUS X54C laptop (dual boot: win 7 & ubuntu 14.10 dev) with INTEL Sandybridge mobile (HD2000 intergrated) graphics, and when I'm in the internet especially using Bing, I haven't noticed any compression artifacts when using ubuntu, where as windows 7 is always full of them. I've updated my microcode from revision 0x23 to revision 0x29 on saturday, and now even sites that in windows, for example: windows 7 help forums, majorgeeks, Bing weather...etc. that are fast unreadable in windows are showing up crystal clear in ubuntu. I know my screen is only 1366x768 (16:9) but with ubuntu 14.10 dev, the letters, video and pictures show up HD quality.
**UBUNTU...GOOD FOR THE EYES!!!:shock:**
