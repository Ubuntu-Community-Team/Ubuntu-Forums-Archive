### Post by guiverc on 2020-03-13
I'll add this.

If you're unsure which site is official, it's best not to ask search engines, but seek answers from ubuntu.com itself as it's under Canonical/Ubuntu control.  Searching there you'll eventually end up at 

[https://ubuntu.com/download/flavours](https://ubuntu.com/download/flavours)

which will send you to the official web site for each flavor.  

You'll find other Lubuntu resources there too, such as [http://manual.lubuntu.me/](http://manual.lubuntu.me/) (Lubuntu manual), [https://discourse.lubuntu.me/](https://discourse.lubuntu.me/) (forum), [https://lubuntu.me/blog/](https://lubuntu.me/blog/) (blog) to development (phabricator) and other resources...
