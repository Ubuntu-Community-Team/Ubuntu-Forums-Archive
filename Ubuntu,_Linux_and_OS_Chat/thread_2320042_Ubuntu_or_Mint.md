### Post by Bucky Ball on 2016-04-10
Wrong. As previously, if you want Windows, use Windows. Some desktop environments can be made to 'look' a bit like Windows, but that is not something developers are aiming for and not something users are wanting or asking for. Regardless of how much it looks like Windows, it isn't, and the basic operations are going to be 'Ubuntu-centric' and have nothing to do with Windows. 

Windows programs will not run in Linux (.exe files) without workarounds via a kludge called Wine, and even with that, many programs will not work at all. You will need to find alternatives to programs you use which are specifically Windows software.

[Read this]("http://linux.oneandoneis2.org/LNW.htm"). If you are looking for drop-in Windows replacement, Linux is not it, Windows is.

Bottom line: You may get something that looks like Windows but it won't work like Windows. It is Linux.

### Post by kurt18947 on 2016-04-10
Two benefits to Mint for the newbie IMO. Bear in mind that Mint is built on Ubuntu.

1) The Cinnamon UI is pretty familiar to Windows XP or 7 users. Not identical by any means but similar enough to figure out pretty easily.

2) Mint is said to take liberties with certain software licensing. They include a lot of drivers and codecs by default that may not be legal in all countries. The benefit is that most media will play 'out of the box'. 

I started with Ubuntu when it was using Gnome 2. I've stayed with Gnome 3 but was familiar enough with terminology that I was able to work around some of gnome 3 idiosyncrasies and shortcomings.  A few gnome shell extensions makes Ubuntu Gnome work quite well for me. 

The latest bleeding edge hardware is not necessarily the best for a smooth Ubuntu /linux experience. A bog standard 3 year old business class machine would be likely to yield a smooth trouble free experience. Some of the newest ultrabook-type machines can have new chipsets or UEFI BIOSs that are problematic. AMD motherboards using 970 chipsets have problems with 64 bit Ubuntu. They may well be supported in a year or two once devs get experience with them.

### Post by sgian on 2016-04-11
I switched to Mint for a while when I got frustrated with Ubuntu security updates (like kernel updates with the fglrx drivers for those who will deny that Ubuntu ever has a problem) crashing my system.  The general impression I got is that Linux Mint has a stronger focus on stability at the slight expense of some security, while Ubuntu has a stronger focus on security at the expense of some stability.  That said, they are both good distributions with better security than Windows, and excellent support forums (although the people at the Mint forum have been nicer to me than some of the people here).  As mentioned, Linux Mint comes with more drivers and codecs that are considered copyright infringements in some countries, while you have to install them manually in Ubuntu.  If you want to play all your music and videos legally, check out the fluendo player.  The license is about $35 a computer last I checked.  Otherwise you can easily get the necessary codecs online for free.  Apparently Windows owns some codec copyrights or pays the royalties already for the rest or something like that, that is why they work out of the box for Windows as I understand it.

For a person migrating from Windows, I would recommend a dual partition boot.  That allows you to choose between Windows and Linux when you start up the computer.  I would also suggest Linux Mint for a beginner because it is more stable than Ubuntu while being still being much more secure than Windows.  However, either linux distribution is fine for a beginner.

The nice thing about linux is that if you don't like one distribution, you can always try a new one without paying anything.  Depending on the speed of your computer, it may take 30-60 minutes to install Ubuntu or Linux Mint and for me another 30-60 minutes to update and install my preferred software.  Windows on the other hand is an all day experience for me to install, update, and customize.  I would recommend sticking with Ubuntu or Linux Mint for a beginner because Ubuntu tries to make sure everything works out of the box for beginners, and Linux Mint is based on Ubuntu so it works well out of the box on most computers too with a little more stability.

As for desktop environments, I tell my kids that it is like the interface or gui that you see when interacting with the installed operating system.  That of course is an oversimplification.  Linux Mint has Cinnamon (based on new gnome but tweaked more like Windows), KDE (lots of choices), Mate (older more stable gnome), and XFCE (a faster, lightweight environment better for older, slow computers).  Ubuntu has all of that and more except Cinnamon.
