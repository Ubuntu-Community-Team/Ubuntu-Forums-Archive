### Post by jeff127 on 2015-10-17
I'm using a Radeon card on this new PC of mine, I've had the system for a few months. The fan is always going mental, then going quiet, then going really loud again. I installed a driver (not Catalyst but the open-source one) and instantly the problem is gone. I was surprised! This entire time I'd been wasting extra power because I didn't have the driver installed. I didn't realise how bad it was. Videos no longer have that tearing problem :) 

Another thing that bugs me is the dictionaries. I have to use Google to find the exact package names... until now. If I were to recommend a fix to new users, I'd say use **apt search EN-GB** or whatever your language is. I'm in New Zealand so I go for British English.

I used that command and found the right dictionary for Libre Office Writer. I didn't realise that apt was a standalone command, I thought I had to do **apt-get search** but it seems that apt-get has no such feature. I've used Linux for years and I'm only figuring out these things now.
