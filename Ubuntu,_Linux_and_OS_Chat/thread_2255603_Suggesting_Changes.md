### Post by PondPuppy on 2014-12-06
+1 grahammechanical. 

The free, open-source software community does a number of things very well, and one of them is provide much wider ranges of choice and customization than proprietary software systems. The main distributions I use -- Salix, Manjaro, Ubuntu, and Mint -- all have conveniences that Windows doesn't have. 

Some of the things on the list are already available. Use the Krusader file manager and you can copy file paths, from either pane -- you can copy from one pane and paste to the other, if you have a need. Mint and other distros have a taskbar with application icons, as well as a menuing system reminiscent of Windows (better than Windows, actually, due to hierarchical sorting). Those might be more congenial to the OP. 

But the main thing is, to my tiny mind: Ubuntu is not Windows, Mac OSX is not OpenSUSE, and none of them are FreeBSD. Enjoy the differences. I haven't tinkered with BSD, but I like the other three systems just fine. They're great.

As far as charging for Ubuntu: free, open-source software is, crucially, *free*. When code is available to anyone, when it's free, cooperation and innovation take on a particular kind of power. (My opinion, unqualified and unfounded as it may be.) Charging individual users for Ubuntu is not a good idea. (Canonical does charge for a range of enterprise services, such as large-scale Ubuntu installation design and consulting, support, and management.)

And changing Ubuntu as suggested will not make users flock to it. Users buy devices. Nearly all PCs are sold with Windows pre-installed. Therefore about 90% of the PC market uses Windows, and most of the other 10% use OSX -- which is of course pre-installed on Macs.
