### Post by pissedoffdude on 2017-11-17
I'd say try out some live USB's for yourself to give them a quick spin.  Personally, I've always just fell back to Ubuntu when I wanted a workable linux system without so many hassles (heh, I like arch, but sometimes I just want to be up and running quickly and Ubuntu excels there).  That said, from distros i've tried, give elementary, mint, pclinuxos, and opensuse a shot.  I recommended elementary and mint first since they're Ubuntu based and try add some neat GUIs on top of already existing functionality, but pclinuxos does offer an awesome mandriva based config center (gui based) that lets you configure basically everything in your system without having fire up the command line.  OpenSuse's yast GUI also lets you do something similar, and it has a command line interface as well, so I'd say try them out and see what you think.
