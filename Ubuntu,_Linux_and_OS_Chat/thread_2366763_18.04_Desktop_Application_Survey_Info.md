### Post by TheFu on 2017-07-21
Web Browser: Firefox on secure networks; firejail Chromium for general surfing; lynx for quick non-GUI surfing; dillo to avoid javascript.

    Email Client: Thunderbird

    Terminal: xterm (yes, the pure form)
    IDE: Geany - as heavy as I'll go.
    File manager: xterm; GUI is too slow.
    Basic Text Editor: vim (not vim-tiny)
    IRC/Messaging Client: Thunderbird
    PDF Reader: evince for printing/reading; xournal to modify.
    Office Suite: libreoffice - seldom used
    Calendar: Lightning + Thunderbird + Zimbra
    Video Player: mpv / vlc / mplayer
    Music Player: mpv
    Photo Viewer: geeqie
    Screen recording: simplescreenrecorder or ffmpeg
    Video Transcoder:  handbrake, unless it crashes, then ffmpeg.
    Alarm Clock: alarm-clock-applet
    Password Manager: KeepassX
    Calculator: galculator
    Media Center: Kodi + Plex Media Server
