### Post by buzzingrobot on 2014-11-17
> **Ubi_one_2014 said:**
> Hi

It looks like the same as ages ago, without any reasonable features.

KDE 4 looks different than KDE 3, and the new KDE Plasma 5 desktop differs from both.

Open source efforts like KDE do not have a surplus of resources. An interface redesign is a major effort.  Once done, it is not going to be overhauled just for the sake of provoking interest. 

On the Windows side, XP looked the same for its lengthy lifetime. Win7 hasn't changed. And Win10 looks very much like Win8.

The look of OS X has remained remarkably consistent over the years and its many revisions.

>  If i compare it with windows, there desktop looks slick, and contains with every new version new features
I do not see that in Linux. 

'Slick' is subjective and a matter of personal taste.  E.g., I don't like shiny graphics and gradients, so most Windows desktops don't appeal to me.

> Most packages are stuck in an old version, an only rebuilded against latest library's. It's hard to get latest software in ubuntu

KDE 4 has been in effective maintenance mode for some months now, as KDE developers made way for KDE Plasma 5. Maintenance mode means no new features, only bug fixes.  This is the common, typical path for software to follow.  It's the same path XP followed for years, and the same path Win7 is following until its end of life.

Also, the individual programs in KDE are typically contributed by individual developers. It's down to them whether or not they put new features in their packages. 

Ubuntu doesn't try to be a distribution offering the very latest software.  Given the nature of FOSS, the very latest releases often haven't seen enough real-world testing by real-world users to allow the new bugs to be discovered and eliminated.  There's a tension between running the very latest software and running problem-free software.

IMO, unless you are following a new effort, most Linux popular Linux software is mature enough that moving from a .2 to a .21 release isn't going to get you much at all in terms of new functionalty.

For help on your NAS and the HDMI-TV issue, you'll need to outline the specific issues and provide details of your hardware.  Might be best to start two separate threads for those subjects.
