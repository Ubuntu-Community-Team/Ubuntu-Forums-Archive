### Post by deadflowr on 2015-02-14
Even before I ever used unity, I always set all my launchers on the peripherals.
That way when I space out, which is often, I can stare at an uncorrupted image/wallpaper. 

Though I do understand people putting their documents on the desktop, since most taskbars and menus don't have easy to access features/capabilites for those.
You typically have to open a program first to get to those files. Putting them directly on the desktop makes it so you just need to click on it to open.
No need to fiddle around with file managers or some arcane program buried deep in a menu hierarchy. Just see the file and open. Simple.
I don't use this methodology, but I can easily relate.
