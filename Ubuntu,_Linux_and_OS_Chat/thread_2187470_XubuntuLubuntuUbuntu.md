### Post by lykwydchykyn on 2013-11-12
> **joyousjake said:**
> So which do you recommend, Xubuntu or Lubuntu?

I recommend you take time to try them both; but if you want an off-the-cuff assessment:

- Lubuntu uses LXDE for its desktop environment.  It's a very light desktop, but a relatively new project (been around like 5 or 6 years, I think).  Might be missing some features here and there.  Lubuntu's default layout is a lot like Windows XP, which some people really like.


- Xubuntu uses XFCE, which has been around as long as KDE and Gnome (started sometime in the 1990's).  It's not quite as lightweight, but very mature and functional.  Xubuntu's layout is a lot like the old Ubuntu before Unity, which some people really like.
