### Post by ssam on 2013-11-29
Quite a few opensource projects have a page of google summer of code ideas, some of those ideas did not end up being sponsored by google. They are a projects that ought to take a student a few months to do, so they might match what you need.

For example here is the future ideas page for GIMP [http://wiki.gimp.org/index.php/Hacking:GSoC/Future/Ideas](http://wiki.gimp.org/index.php/Hacking:GSoC/Future/Ideas) and libreoffice [https://wiki.documentfoundation.org/Development/Gsoc/Ideas](https://wiki.documentfoundation.org/Development/Gsoc/Ideas)

If you see one you like the look of, be sure to check that it has not actually already been done. Then maybe check it past your supervisor. Then maybe send a message to the project mailing list, saying that you are interested in working on it.
