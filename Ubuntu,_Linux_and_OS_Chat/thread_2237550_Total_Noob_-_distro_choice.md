### Post by buzzingrobot on 2014-08-03
Zorin is a Linux distribution based on Ubuntu that uses interfaces intended to resemble Windows and that targets new users migrating from Windows.  Because it is based on Ubuntu, Zorin users have access to the same software repositories. It comes with things like Flash, multimedia codecs, etc., installed by default, which is not always the case elsewhere.

A variety of so-called lightweight interfaces are available for older and less powerful hardware. Without knowing the technical specs of your laptop, though, it's difficult to recommend one or another.  The video card and the amount of installed RAM are typically the critical factors.

Linux distributions typically provide install inmage (ISO's) that can be run in live mode off the DVD/USB stick, without any impact on the machine's existing drives, and that is a good way to determine if a distribution can work with your hardware, and if you like it or not. (If you want to check out how fast or slow one distribution is compared to another, running off a USB stick is probably a better bet.  Running off a DVD is inherently much slower and performance differences will be harder to spot.)

### Post by Rob Sayer on 2014-08-03
There's someone on a media forum I use who installed Zorin thinking it'd work like windows 7.  It won't.  You can make linux look like windows but it does not work the same.

Plus, I looked at the tech support forum on the Zorin site.  Pitiful.  Just *pitiful*.  I wouldn't touch it with a 10 foot pole.  Linux is very reliable but it tends to need some configuring, so tech support is very important for noobs.

There is simply no other linux distro I'd recommend for a linux novice other than ubuntu, unless the computer has less than 512 Mb of RAM.  In those cases I'd recommend a really light linux distro like puppy and be prepared for more of a learning curve because they don't automate installation and configuration like ubuntu does.

It's hard to say which ubuntu DE flavor would be right for you without knowing hardware details.  I had Unity based ubuntu on my 4Gb i3 laptop and found it useable but slow.  It's running Kde based kubuntu 12.04 now.  KDE is also big but you can config it fpr speed, unlike Unity, but I probably wouldn't use it with less than 2Gb RAM.  My 1Gb netbook runs xubuntu 14.04 just fine.

Agreed.  People ought not to confuse visual similarity with actual similarity, but I'm sure they do.

In my experience, the applications I use place more demands on memory than the desktop environment in use.  I typically have, at a minimum, a browser, an email client, and an RSS reader open, each in its own workspace. Memory use for that, regardless of Linux distribution or desktop environment, runs at about 1.2 gigs. It goes up 100 or so megs in something like KDE, and down by the same amount in something life XFCE.
