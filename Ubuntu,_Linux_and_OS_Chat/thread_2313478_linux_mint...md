### Post by buzzingrobot on 2016-02-14
There is also the "Gnome Classic" approach, which is the default in Red Hat, which is now using Gnome 3.14.  It's a small collection of Gnome extensions that layer a panel, menu, etc., over the standard Gnome Shell interface, which remains accessible. I believe the Gnome project has decided it will maintain those extensions, which they don't otherwise do.

I actually think that, with judicious choices, extensions could provide a better emulation of the old Gnome 2 design than what's offered in RHEL. The apps remain the same, of course, along with client side decorations, etc., as they do in Gnome Classic and Gnome Flashback.
