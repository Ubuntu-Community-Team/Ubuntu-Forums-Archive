### Post by hornetster on 2013-11-11
Just a general comment, please don't take the wrong way, but I am not overly impressed with the overall reliability/stability of Ubuntu.
Have been using Linux for quite a few years now, and have used several different distros as my main PC over this period. Generally love Linux, and would never go back to other o/s' fulltime, but obviously work with them.
My "main" distro for the last several years has been openSuse, having first started using it back in the V8/9 days, I believe, but with every new version of Ubuntu, I generally try to install, and use, with the idea that if all is good, may even begin using as my main system, particularly these days, as, unlike some, I quite like Unity....
I have 12.10 installed on another PC on my network, it sits in a cupboard and runs a number of services, generally fairly reliably, and is on old hardware.
But...
My experience for the last 3 versions as a desktop has not been wonderful. 
Installed 13.10 x64 about 2 weeks ago, and at first quite impressed. But, I have found with ongoing use, that there seems to be a general instability, that unfortunately again is going to send me back to openSuse. Don't believe it is hardware related, as openSuse on the same system ticks along without missing a beat. Running x64 in both cases, always keep fully patched, etc. I always install a clean system ie no upgrades.
I find that a lot(?) of applications are generally unstable, or at least unreliable.
eg Ubuntu Software Centre often (15-25% of the time) either crashes when I open it, or regularly while running, the window goes grey for an extended period, as it tries to catch up with itself. 
Have given up running Virtual Box. Very rarely it seems to work OK, mostly it just sucks all the resources from the system and makes everything else impossible to use, but still runs slowly itself.
Thunderbird often "goes grey" while processing.
Ditto Amarok, VLC, Chrome etc
System takes "several minutes" to load apps after logging on, even though the only 'extra' apps I generally load are Thunderbird/Skype.
Often get system messages re: Sorry, the system has experienced an internal error... and generally it is always different - ie not one problem with the system.
As I say, I think I have given it a fairly good go, was intending to use at least until the next release of openSuse (still a week or 2 away, but don't think I can be that patient.... :(
This has been my experience with the last 3 versions used as a desktop.
Any suggestions considered... :P (Please be nice, though!)
Thanks.
