### Post by Bucky Ball on 2016-04-16
Welcome. You might like to try some of the other flavours while you're at it. Ubuntu, Kubuntu, Xubuntu, Edubuntu, Lubuntu, UbuntuGnome, Ubuntu Studio, Mythbuntu and Ubuntu Mate are supported here and there are others also based on the Ubuntu kernel. They can all be booted and tried in a 'live' session from install media.

I know nothing about Arch, but I believe it is a 'ground up' thing from a user perspective. You might be interested in the Ubuntu [minimal install]("https://help.ubuntu.com/community/Installation/MinimalCD"). It installs the Ubuntu kernel, you install the rest. In other words, once installed, install a package manager, if you need one, and a desktop environment, ditto, and you're away. 

Have fun.
