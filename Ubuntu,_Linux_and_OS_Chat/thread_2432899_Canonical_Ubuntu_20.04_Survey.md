### Post by yetimon_64 on 2019-12-17
> **bernard010 said:**
> Ubuntu has it at the snap store you can install it easy. It has always been there.

It is also available as a deb package for those who don't use snaps. Yes "It has always been there", the thing is it used to be included in new installs by default and is, in my opinion, probably the most useful package management tool ever made.

It is always nice to find a flavor or distro that includes it by default as used to be the case with Ubuntu. I still miss seeing it installed by default as well even though it is quite simple to add after an initial installation.

Personally I agree with the sentiment expressed earlier by Artim "bring back Synaptic Package Manager" and I for one always add it on new installs.

My comments on the survey were a suggestion of an arm64 alternate installer with full raspberry pi4 support. I would love to be able to do a base Ubuntu install with a net installer and then build up to a Xfce or mate desktop. Currently there only seems to be an unofficial installer based on the server installer which has too much stuff not needed for a desktop only install. I'd like a much cleaner base install to build on rather than a server base on my raspberry pi4. I believe the raspberry pi 4 support work is in progress I just hope that is included into an alternate installer (netinst image) as well.

Cheers, yeti.
