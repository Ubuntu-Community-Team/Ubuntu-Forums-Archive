### Post by grahammechanical on 2014-12-10
Amusing, if a little inaccurate. Ubuntu does not have release names but labels taken from the month of release and the year of release. The name chosen by Mark Shuttleworth is the code name for the release while it is under development. Of the two parts of the chosen code name it is the adjective that is most often used to refer to the release to come and not the noun.
