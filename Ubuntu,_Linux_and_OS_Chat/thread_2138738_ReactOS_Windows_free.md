### Post by forrestcupp on 2013-04-25
A lot of us have had our eye on ReactOS since they first began. The problem is that they are always going to be way behind everything Microsoft puts out. There's no way they could ever possibly catch up.

And philosophically, it doesn't make sense. Windows comes pre-installed on almost every computer. The only purpose to use ReactOS would be because of a personal philosophy of FOSS. But if you felt like you needed to run a FOSS version of Windows, then it wouldn't make sense to run proprietary software on that FOSS version of Windows. And all of the FOSS software is already available for Linux, so why not just either use Linux or Windows?

The only other legitimate purpose for ReactOS is just messing with it as a hobby. In that case, I think it's very cool. But it will never have a real purpose for people who just want a good operating system.
