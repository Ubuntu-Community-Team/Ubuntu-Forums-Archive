### Post by 3rdalbum on 2013-06-03
1. Install Windows 7 first.
2. Put the Ubuntu CD into the drive and boot from it.
3. Click "Install Ubuntu" or "Try Ubuntu" and then double-click the Install icon on the desktop
4. Follow the on-screen instructions. When you reach the screen that asks how you want to install Ubuntu, choose the "Side by Side" option and drag the dividing bar between Windows and Ubuntu, to allocate space for each operating system.
5. Continue to follow the on-screen instructions and it will install.

It's really that easy.

The main way is to download a *disk image* from [http://www.ubuntu.com/download](http://www.ubuntu.com/download)

You can then create an installation disk - see [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto) - or make a bootable usb-stick - see [https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)
