### Post by TheFu on 2018-04-07
I think you should install win10, since wiping it will feel great in a few days. If you care about privacy, win10 isn't really an option at all. Just read the EULA.

The CPU performance for an N2830 is really poor.  Stay with a distro designed for low-end systems.  I'd think that Lubuntu would be the best option of the Ubuntus.  Definitely go with the 32-bit version for that amount of RAM.  Wasting 2x the memory for integers and pointers throughout the OS is a waste on a 2G RAM system. 
The built-in GPU should handle mpeg2 and h.264 video playback fine.  Newer vcodecs will be an issue, since the CPU will have to decode during playback.  Video decoding in the GPU is important.  Most handle mpeg2 and h.264 in hardware. When h.265 becomes the standard, this CPU will struggle.

A few tips:
* verify that zswap is used (google will help)
* make certain that swap is 4GB in size; that's important on lower memory systems.

Lubuntu should be fine, though not speedy.  Mostly that is due to the N2830 (only 956 passmarks).
