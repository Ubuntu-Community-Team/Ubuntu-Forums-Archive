### Post by grumblebum2 on 2014-06-24
After a bit of testing found that when you install opera-developer it picks up your bookmarks from
your **~/.opera** folder when the previous version of opera 12.xx is already installed.

If you don't have a **~/.opera** folder you won't see a **Bookmark Importer** menu item.
When i first started opera-developer it had imported my bookmarks bar and history from opera 12.16.

The bookmarks importer gives you the choice of **convert to speed dial** or **add to bookmarks bar**.
Then just shows your bookmarks from opera 12.xx.
[ATTACH=CONFIG]254229[/ATTACH] [ATTACH=CONFIG]254230[/ATTACH]
There is nothing to import manually from a .adr or html bookmarks file.
I don't even have a bookmarks menu entry.

### Post by grumblebum2 on 2014-06-26
I found the tab preview annoying because I use keyboard shortcuts to switch tabs.
You effectively have to hit the shortcut twice to switch a tab because the first switch shows a preview.

To disable tab previews, in the address field enter...
```
opera://flags
```

Then in the search box enter...
```
tab preview
```
and disable
