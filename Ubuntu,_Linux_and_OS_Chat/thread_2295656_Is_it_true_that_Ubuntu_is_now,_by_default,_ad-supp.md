### Post by buzzingrobot on 2015-09-20
> **Nixarter said:**
> 
I don't use the new "gnome" interface...  I was originally perplexed as to why Canonical would abandon an excellent system for this inferior new "gnome" system.

The standard interface for Ubuntu is not Gnome.  It is Unity.

For a number of years, Ubuntu's standard interface was Gnome 2, as it was on many other distributions.  Gnome 2 was not a Canonical project.  In 2011, the Gnome project decided to end all work, including maintenance, on Gnome 2 and launched an entirely new and different Gnome 3 interface.  

With the demise of Gnome 2 in 2011, then, distributions had to choose a replacement interface. Canonical chose to move forward with its own in-house Unity effort.

The Gnome 3 interface is available in the Ubuntu Gnome spin.

While Ubuntu uses a number of tools and applications from the Gnome project, Unity is not based on Gnome, although you still see misinformed people making that assertion. Unity is implemented as a Compiz plugin.

Others have responsibly and accurately address the bogus claims about malware, etc. When people make claims without evidence, when they expect us to rely on their reputation as proof of honesty and infallibility, we should ask what their own agenda is. From my own point of view, the hardcore approach advocated by RMS, the Free Software Foundations and others is completely at odds with how real people behave.
