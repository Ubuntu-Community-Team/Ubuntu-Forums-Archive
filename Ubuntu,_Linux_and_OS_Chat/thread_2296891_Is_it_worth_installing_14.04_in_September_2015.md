### Post by deadflowr on 2015-09-29
Instead of completely ditching Windows, try to setup a dual-boot.
(Unless Windows is XP, in which case, yeah nuke it; my opine)

If you setup a dual-boot correctly, you'll have the benefit of at least having a Windows install.
(You never know if or when you might need to do something that only Windows can do)

And if it fails, well, you were already prepared to lose it anyway.
Think of it as a no harm no foul.


Also, you might not want to install 16.04 until a month or so after it's released.
So probably tack on a couple of extra months to that 6 month timeline figure.

(I'd even go as far as to say wait until August 2016, when the first point release hits and comes with a boatload of bug fixes that usually hit when the first release hits in April)
