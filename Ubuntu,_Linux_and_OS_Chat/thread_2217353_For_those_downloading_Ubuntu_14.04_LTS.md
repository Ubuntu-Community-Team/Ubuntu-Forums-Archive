Yes, you are missing something. Ubuntu 14.04 comes out today. Every day for the last six months, Canonical has published a "daily image" of the latest state of Ubuntu 14.04 during its development. Every day, **including today**.

[http://cdimage.ubuntu.com/daily-live/current/]("http://cdimage.ubuntu.com/daily-live/current/")

Technically, it's possible that one or two packages will change in the twelve hours or so between the daily live image and the official release version. If that happens, those packages will appear in Update Manager as ordinary updates, and then you'll be running the official release version of Ubuntu 14.04.

People get hung up on an actual release date and release time; I understand it's exciting to see 14.04 actually "finished", but isn't it more exciting to realise you can download it early?
