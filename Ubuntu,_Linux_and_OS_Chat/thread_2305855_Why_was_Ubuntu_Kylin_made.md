### Post by grahammechanical on 2015-12-09
Anyone can take the Ubuntu code and make their own distribution. And this is what some Chinese developers did with Ubuntu. In recent years both Ubuntu Gnome and Ubuntu Mate have gone from private initiatives to official Ubuntu derivatives. And that has been the same path that Ubuntu Kylin has gone through.

Yes, there may be commercial advantage for Canonical in the Chinese market but it is also the mission of Canonical to make open source available to people everywhere. Canonical has also been encouraging Chinese community developers to get involved in developing for the Ubuntu phone OS.

[https://insights.ubuntu.com/2015/08/26/winners-crowned-in-the-china-mobile-ubuntu-developer-contest/](https://insights.ubuntu.com/2015/08/26/winners-crowned-in-the-china-mobile-ubuntu-developer-contest/)

Linux is open source and with access to the source code a person, if they know what they are doing, can examine the code to make sure that the OS & applications are not going to anything that you do not want them to do. If that is not in high demand with Companies and governments, then it should be.

### Post by Bucky Ball on 2015-12-10
> **grahammechanical said:**
> Anyone can take the Ubuntu code and make their own distribution. And this is what some Chinese developers did with Ubuntu.

As far as I know Kylin had nothing to do with 'some Chinese developers'. It goes a little deeper than that. 

> In 2013, Canonical reached an agreement with the Ministry of Industry and Information Technology of the People's Republic of China to release an Ubuntu-based OS with features targeted at the Chinese market.

And some more info:

> Kylin is an operating system developed by academics at the National University of Defense Technology in the People's Republic of China since 2001. It is named after the mythical beast qilin. The first versions were based on FreeBSD and were intended for use by the Chinese military and other government organizations. With version 3.0 Kylin became Linux-based, and there is a version called NeoKylin which was announced in 2010. 

In 2013, it was announced that a new Linux-based operating system with the same name would be released using Ubuntu. The first version, Ubuntu Kylin 13.04, was released on 25 April 2013.

From Wikipedia. Read all about it there. Kylin was never an organic meeting of the minds of a few happy-go-lucky developers having some jollies (not that that's what you were alluding to, grahammechanical). 

Tons of info about Kylin if the OP has a quick look online.
