### Post by Bucky Ball on 2015-04-24
The distinction should be made, here, that Ubuntu is Ubuntu and Xubuntu is Xubuntu, etc. They all run on the same base kernel, it is only the default apps and desktop environments, basically, that make them one flavour or the other. Consequently, install Ubuntu and if you don't like Unity, change the desktop environment. xfce4, gnome-2, lxde, RazorQT, take your pick. You are not stuck with Unity just because you want to use Ubuntu. To install xfce4 for instance, install from SCentre, log out, choose 'XFCE session' from the sessions menu, log in. You are using xfce4 DE as used by Xubuntu (among others).

The only advantage Xubuntu has over Mate is whether you prefer the xfce4 desktop environment to whatever DE Mate happens to use and the default apps that come with Xubuntu to the ones that come with Mate. That is not a show-stopper, either, as I think Mate uses the Ubuntu repos so you can install whatever programs you like into whatever flavour you like (as they all use the same base kernels and repositories). 

I personally stick to [minimal installs]("https://help.ubuntu.com/community/Installation/MinimalCD") and only install the desktop environment and apps I want/need. Super slick, little bloat and fast. 

Bottom line: use Ubuntu with whatever DE you want. You are not stuck with Unity. The difference between Ubuntu and its flavours is the desktop environment and the default apps, but they can be interchanged between flavours and you can Frankenstein yourself a proper hybrid (I have xfce4 DE, pcmanfm and lxterminal from Lubuntu (lxde) and bits and bobs from all over installed).

Good luck and hope that helps clear things up a little. ;)

PS: You will notice Linux Mint and some other derivatives come in a variety of their own 'flavours'. You can have your choice of desktop environments when you download the ISO.
