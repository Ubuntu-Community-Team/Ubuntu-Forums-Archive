### Post by d-cosner on 2014-04-20
I worked on an older Sony Vaio this weekend for a friend. The computer has a P4 2.66 GHz, Intel on-board graphics, 1 GB DDR RAM, 30 GB Maxtor IDE hard drive, on-board sound and on-board graphics. Unfortunately, this is not a powerful enough system to run Ubuntu with Unity. Xubuntu seemed pretty nice though and it has plenty of support so I downloaded the 32 bit iso and gave it a shot. All of the hardware worked perfectly out of the box.

The Xubuntu Team did an amazing job with the Plymouth screen, default theme and artwork, it really looked nice! The default applications were light for the most part and they all looked like they belonged together because the theme was so consistent. The menu was also nice and since the computer originally had XP on it the user will most likely like it. The system was quick and resource friendly! I was not sure about Xfce at first and almost went with Lubuntu but Xfce seemed like the most attractive choice.

My goal was to make the system as nice as humanly possible and still keep things easy enough for someone that was new to Linux. Length of support was a factor too so that the user could get a few more good years of use out of the computer. Multimedia worked well except full screen flash being choppy, I expected this but flash worked perfectly in a smaller window and that really is not bad considering the specs of the computer. 

Everything looked nice, it was reasonably fast, it was resource friendly and I thought it really turned out nice! :D I of course cleaned the computer so it was spotless inside and out because I think it is good for another 3 years! It has been a long time since I had looked at Xubuntu, they did an amazing job with the 14.04 release!

### Post by sudodus on 2014-04-26
Lubuntu 13.10 uses zram live and installed. Lubuntu 14.04 LTS uses zram live, but *not* installed.

Using zram makes it possible to run the installer with lower RAM than otherwise (the necessary RAM is reduced to about half). This is not a problem in a computer with 1 GB RAM, where you might want Xubuntu. See this link for more details

[https://wiki.ubuntu.com/Lubuntu/AdvancedMethods#Test_results](https://wiki.ubuntu.com/Lubuntu/AdvancedMethods#Test_results)
