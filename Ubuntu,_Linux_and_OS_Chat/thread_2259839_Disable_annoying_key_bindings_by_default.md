### Post by Vormeph on 2015-01-07
As a gamer, I find it annoying and impractical that whenever I alt+right click the game window minimises in order to bring up a dialog regarding the current window. I lose focus, and because I usually use the said combination in action I usually have a near-death experience. This isn't a rage post if that's what you think; and there are among other annoyances with Ubuntu's unity, especially with full-screen games. My proposal is that Unity keybindings be disabled if there's a full-screen application running. My other proposal would be to, by default, unbind keys which can otherwise cause annoyances. I know not everyone is a gamer, but it's reasonable to say that no one wants the dash to pop up while gaming. 

Has anyone encountered any other annoying key combinations they had to disable while gaming? We could raise this thread to the attention of the devs so that incoming gamers don't have a recurring problem. My biggest problem so far is alt+right click. On TF2 for example, I use ALT to call a medic, and I have the right mouse button clicked to shoot - that minimises the window and am dead! Go figure.

Btw, you can use Compiz Settings Manager to disable the annoying key bindings. 

Discuss!
