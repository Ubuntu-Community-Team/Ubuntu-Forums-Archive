### Post by kpatz on 2015-02-03
My very first distro was some CD in the back of a "creating a web site on Linux" book in the mid-late 90s.  I don't remember what distro it was now, some predecessor to CentOS I think, very primitive, probably a 2.0.x kernel.  I installed it on a box and played with it a bit, but didn't get very far with it.

My first distro that saw regular use was some old version of Mandrake (circa 2000) I had picked up on CD at a computer show.  I installed it on a box and used it as a firewall/router for my broadband internet connection, before consumer routers became popular.  This was a 2.2 kernel so iptables didn't even exist yet, it was the more primitive ipchains.  I ran that box from 2001 until 2005, when we had a house fire.  The box survived and still booted, but it smelled bad, so after we rebuilt, I installed my first Ubuntu (probably 6.04) on an old Dell Pentium III box and used that as my firewall/router.  The box went through some upgrades (6.10, then 8.04 after having to replace the hard drive).  That box ran for years, and I finally retired it last month and built an Atom system with 14.04 server, which is my current firewall/router.

In the meantime I've had a few other *buntus on other boxes.  My MythTV system which I originally built in 2007 had 7.xx I think, then 8.04, then after a PSU failure took out a few pieces, I rebuilt it and put Mythbuntu 10.04 on it.  I just recently updated that box to Mythbuntu 14.04.  In 2009 I built a file server with Intrepid (9.04, or 8.10, I forget now), then upgraded it to 10.04.  It's still running 10.04 but I plan to upgrade that to 14.04 as well.

So, between some old bundled-in-a-book distro CD and an old Mandrake, those were my first distros and then I jumped on the Ubuntu bandwagon and stayed there ever since.
