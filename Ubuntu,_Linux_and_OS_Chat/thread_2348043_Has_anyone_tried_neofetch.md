### Post by vasa1 on 2016-12-31
[NeoFetch — See System Information from the Command Line on Linux]("http://www.omgubuntu.co.uk/2016/11/neofetch-terminal-system-info-app")
> you can “fetch” all sorts of information about your system, from what kernel you’re running to what icon pack you’re currently using

...

Written in Bash, Neofetch displays information about your system next to an ASCII operating system logo ...

It seems to be available in the repos [for 17.04]("https://launchpad.net/ubuntu/+source/neofetch") and as a [ppa]("https://launchpad.net/~dawidd0811/+archive/ubuntu/neofetch") for currently supported versions or from [its GitHub page]("https://github.com/dylanaraps/neofetch").
