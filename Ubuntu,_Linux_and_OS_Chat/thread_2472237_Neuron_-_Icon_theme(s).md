The idea with this icon theme is to save space when using Plank (dock) as your app launcher etc.
It will be available for Gnome/XFCE and Budgie DE. Even though it will be available for XFCE which uses small icons for the most, it's advisable to use Plank with this theme.
A slim plank theme will also be available.
