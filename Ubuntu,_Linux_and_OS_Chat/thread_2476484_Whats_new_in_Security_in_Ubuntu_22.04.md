I found this comment interesting.

> Intel’s Software Guard eXtensions (SGX) provides hardware-backed secure  enclaves which applications can use to either store sensitive data or  perform sensitive computation without the risk of interference from  untrusted components. Ubuntu 22.04 LTS enables support for this feature,  which has been [present in Intel processors for a number of years]("https://www.intel.com/content/www/us/en/support/articles/000028173/processors.html"). 

Let us not see every innovation to make computers more secure as part of some conspiracy to lock us into one particular operating system. Also. I would find it interesting to know which Linux applications are taking advantage of this security feature.
