### Post by vidtek on 2015-08-01
I have now had this for a month or so and am getting to grips with it.  The TBS linux drivers revert to an earlier v4l tree and so conflict with my Hauppauge HR2210 which is now fully incorporated into the linux kernel.  The team at TBS is now working on a solution to this.
The people at TBS have been extremely helpful and have responded to every concern I have with this card both with the Windows software and the linux drivers.  It is very refreshing to be able to report on a company which is actively engaged in developing linux solutions for their products so it would be good for the linux community to reciprocate and promote their tuners to the whole community.
This TBS6205 tuner is very stable, excellent value and performs well in my Mythtv system.  For quick and easy viewing it also works well with Kaffeine.  I can heartily recommend this tuner to the linux community.

Tony.
