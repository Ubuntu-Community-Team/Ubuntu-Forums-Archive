### Post by PhoHammer on 2013-09-20
You might be able to get by with something from with the free option from 000webhost: [http://www.000webhost.com/features](http://www.000webhost.com/features). It's not a full server, but you can run php scripts and create cron jobs.

Otherwise, take a look at lowendbox.com. I wrote about my experience here: [Alternatives to the Big VPS Companies]("http://magnatecha.com/alternatives-to-the-big-vps-companies/")

I'm using a company called called [BlueVM]("https://www.bluevm.com/aff.php?aff=397") (disclaimer: that's an affiliate link). I have a 4-core, 1GB RAM, 50GB storage VPS with them for $25/year - it was a sale I found on lowendbox.com.
