### Post by david362 on 2015-03-10
I Googled comparison between Windows and Ubuntu/Linux system resources, but did not find any recent articles. I wonder if my experiences with system resource usage are common?? 
 
Using Ubuntu 12.04 LTS with 2 GB memory and dual core Athlon 2 GHz processor uses 19-24% memory and flat-lined 4-6% processor usage after startup. With Windows 7 Home, the stats were 45-50% memory usage and 15%+ processor usage, although continued boot of services often maxed out the processor.  I got memory usage down to about 40-45% after the latest of many rounds of combing through system start services. 
 
Using a computer with 1.6 GHz i3 quad processor and 4 GB memory with Ubuntu 14.04, resource usage is even better! The computer responds so quickly, I used to think something had gone wrong haha eg files were copied before a progress bar had time to appear.  
 
With both computers using Ubuntu, I get much better performance than with Windows for everyday use. The difference when moving files on disk or between disk/USB memory is truly shocking. I can move a 1.2 GB file to USB with Ubuntu in less time than it takes to transfer 200MB with a Windows 7 Professional machine (2.7GHz i5 with 8GB memory).  
 
Starting programs: On the Athlon computer, whited out windows and "program not responding" messages are common running Windows 7. 
Doesn't happen on Ubuntu. 
 
Going online: 500kb-1MB handshake with server running Windows (if some auto-update program doesn't decide to go online and download 3-400 MB without my knowledge, freezing my Internet usage in the process). 
With Ubuntu, the handshake with the server is usually 6.7kb! 
 
Startup and shutdown: 2-3 times as long with Windows. 
 
Years ago, I wished for an OS that would allow me to have control over my computer, one that did not devour resources, and one that was reliable and responsive to allow me to be productive. I thought then that Linux sounded too difficult to use, but I am so glad that I gave it a try a couple of years ago. (Soon Windows free). 
 
Using a terminal and open source programs, I get things done so quickly, I am left scatching my head sometimes wondering if I have forgotten something. 
 
To all of the members of the open source/Linux/Ubuntu community out there who have played a part in creating my dream OS, THANK YOU!  
 
You people are amazing! I actually look forward to turning on a computer again and now I can really get some work done! THANK YOU!
