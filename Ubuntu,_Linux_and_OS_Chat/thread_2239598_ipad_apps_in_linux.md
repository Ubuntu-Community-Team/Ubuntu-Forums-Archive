### Post by Tar_Ni on 2014-08-14
Why not using both? Though I am not a big fan of Apple, I suppose you could use the Ipad for web browsing and running a few useful apps. With your laptop, you can install Ubuntu 14.04 and do some productive work if need be, LibreOffice is provided by default and for free. GIMP maybe can help you with customization too. Theres a lot of free app in the Ubuntu Software Center and you might be suprised.

 The best thing to do is to try it out, if it works for you than that's good news! A laptop with Ubuntu can cover most productive needs, unless you are absolutely required to use a specific program not available on the Linux platform such as Adobe Photoshop CS6. LibreOffice or web-based alternatives such as Google Doc can replace Microsoft Office in my opinion.
