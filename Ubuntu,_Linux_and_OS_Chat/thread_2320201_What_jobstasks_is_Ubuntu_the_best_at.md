### Post by grahammechanical on 2016-04-11
Ubuntu is very good at being a Linux distribution.

You do mean Ubuntu desktop? You want to exclude all other the things that Ubuntu is also very good at? The cloud, the Internet of Things, Snappy Ubuntu Core with transactional updates & apps that the developer can easily get into an app store & then update as soon as an upgrade of the app is ready. Rather than waiting for the next release of Ubuntu to push it out. Snappy Ubuntu core embedded into Network switches and that sort of stuff. You do want to exclude all those kind of things?

Ubuntu is very good at being a development platform. There is an application that will make it easy to install and keep up to date different IDEs for different operating systems. Want to work with an Android IDE or Visual Studio for Linux or some other IDE? Ubuntu Make is what you need.
