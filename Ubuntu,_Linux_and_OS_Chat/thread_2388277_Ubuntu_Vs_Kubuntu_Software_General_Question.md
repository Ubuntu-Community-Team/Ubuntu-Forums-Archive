### Post by deadflowr on 2018-03-31
Ubuntu and Kubuntu utilize the exact same software repositories.
All packages are built to be compatible with all flavors regardless of desktop.

If something does not work cross-desktop-flavor, then the developers would most like to know it so they can try to make it work cross-desktop-flavor.
In which case you can file a bug for it to get the devs attention.

And you'll also fine most (or at least quite a few) packages are fairly desktop agnostic, such as libreoffice and firefox (and chrome/chromium)

Same applies to [all official flavors]("https://www.ubuntu.com/download/flavours")
