### Post by grahammechanical on 2016-03-01
Why is loading an OS from an SSD faster than loading an OS from a hard disk? That is easy to answer. The data transfer rates of SSDs are greater than the data transfer rates of hard drives. And then there is variation between the data transfer rates of different hard drives.

What does all this have to do with the question?

Different desktop environments and user interfaces have different amounts of data that have to be retrieved from the hard disk and put into memory and processed by the CPU & GPU.

So, we take into account the type of CPU, GPU, storage medium, speed of RAM and the amount of data that has to be retrieved and processed and we end up non-the-wiser. :)

Life is too short to worry about these things. If we have the hardware to give us a good user experience with a user interface that requires hardware accelerated 3D rendering, then we are happy.

If our hardware is not up to handling a user interface that requires hardware accelerated 3D rendering, then we can still be happy by using another Linux distribution that is better suited to our hardware.
