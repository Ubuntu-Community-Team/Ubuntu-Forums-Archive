### Post by grahammechanical on 2020-07-01
Ubuntu 18.04 reaches End of Life April 2023. So, there is no rush. To upgrade or not upgrade is a personal decision. If you are happy with what you have now, then stick with it until nearer the End of Life date and then prepare well. I think this is especially true for the person who is a user of a single machine. 

The owner of a business with many machines will have different considerations. A purchase of new hardware might require the latest version of Ubuntu which would have the latest Linux hardware drivers. It might not be desirable to have machines running different versions of Ubuntu. So, a situation might develop where the organization has newer machines with the latest Long Term Support version of Ubuntu and older machines that need to be upgraded from the last Long Term support version of Ubuntu in order for all employees to be using the same version of the software and for the operating system of all machines to have the same length of life support. Forgetting to upgrade a machine or two when its OS reaches End of Life is not good security management  in my opinion.

As a single user I am using 18.04 but I have have 20.04 on my machine so I can test it. I will be upgrading 18.04 to 20.04 in a few months time. In time I will upgrade the 20.04 version to 20.10 Ubuntu. I like to follow the development of Ubuntu.
