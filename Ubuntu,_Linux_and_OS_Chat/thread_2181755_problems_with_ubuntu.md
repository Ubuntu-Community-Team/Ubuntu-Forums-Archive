### Post by Bucky Ball on 2013-10-19
All good. Having thought more about this, you could try the 'nomodset' option using the current install (using different instructions to what I have given), then if that works, try installing different desktop environments to see which you like (and I believe 12.04 has a 'gnome fallback' DE already but don't use so not sure). Personally, I would just download the ISO for Xubuntu, Lubuntu, and whatever else you find that takes your fancy, burn some CDs and try them all with a Live desktop (running directly from the CD) to see which you prefer.

Once you've found and researched your OS of choice, install. Xubuntu and Lubuntu are more lightweight (and faster) than a regular Ubuntu install so this, among other things, is the reasoning there. 

It is a bit strange to get the head around at first but, let me put it like this, you can install Ubuntu then install whatever desktop environment you like (including Gnome) and use that. You don't have to use Unity. But there is a Ubuntu 'flavour' based around most DEs. So if you decide to run Ubuntu with xfce4 (DE used by Xubuntu), that is all good, BUT ... if you install xubuntu-desktop to get xfce4 you will also get ALL Xubuntu apps and everything else! You don't want that. So best to just install Xubuntu first up or just xfce4 in Ubuntu (default Unity).  

Hope this makes some sense and is of some help ... ;)
