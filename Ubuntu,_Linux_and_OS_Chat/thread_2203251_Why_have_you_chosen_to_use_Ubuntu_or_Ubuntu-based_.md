### Post by BoogeyOfTheMan on 2014-02-06
My first taste of linux was a Knoppix live cd that I used to troubleshoot a non booting windows laptop. Then I started experimenting with Mandrake on my insanely old hand-me-down pc, and I think I may have even tried *buntu 4 or 5 (I was trying out quite a few to see if there were any that I really liked). But I stuck with windows on my main pc because I was into gaming. Then I ended up having to install windows on my old pc for my fiancee because she wanted to game with me. A few years later I was getting annoyed with having a 32bit operating system that was very inconsistent with the way it showed how much ram I had. I had 4gb, but some apps only showed 3, some showed 3.5, and only windows itself saw all 4. Then one day I was called into my bosses office so he could show off his new netbook, which happened to be running Ubuntu 8.*. After seeing how smooth it ran, I decided to go home and install it on my main pc. My wife was so impressed with it she told me it better be on her computer as well when she gets home from work. I stuck with Ubuntu until 11.10, then tried Mint when I decided I didnt like Unity. I also tried out Xubuntu on my laptop, then switched to Lubuntu on it, and am thinking about installing Kali on it since the main thing I use it for is network troubleshooting. 

Now I'm using Kubuntu because I always liked KDE, but until recently, I couldnt get it to work without the fonts looking like they were 40pt. I'll probably try some other distros at some point, but I'll keep some flavor of Ubuntu on at least one of my pc's. Hell, I may even give Windows another try if they ever get a package manager, support for multiple workspaces, and support for filesystems besides NTFS and FAT. Though never on my main box ;)
