### Post by raphaell2 on 2019-02-09
When it comes to the live installation dvds for the various official Ubuntu flavours, people are generally advised to download them as torrents, to reduce the load on the official servers. But there's no torrent for the network installer, also known as the mini-iso. Sure, it's less than 70 mb, but there might still be some potential for reducing server load. So, should there be torrents for it?

(I really like the network installer. It doesn't take forever to boot, it lets you easily install things like the font selection and the various multimedia selections, and you can try out different flavours without having to download and burn several isos. And I have a soft spot for text-based user interfaces - they remind me of the first computer my family got when I was a kid.)
