### Post by lammert-nijhof on 2017-04-10
Ubuntu is the best and I know it, because I tried a lot of distros in the last 2 weeks for various reasons. Ubuntu works very well as VM Host for Virtualbox (Vbox) unlike some other distros, it is reliable and has a simple, very effective GUI.

Lubuntu is a little bit dull with its Windows 2000 menus. But it runs on P-III hardware with less then 512MB. Don't use it as host for Vbox, the scheduling is mis-tuned for it.

Peppermint is the sexy sister of Lubuntu, it is derived from Lubuntu and has a beautiful GUI. I like its facility to use and install web applications.  Don't use it as host for Vbox, the scheduling is mis-tuned for it. A disadvantage is that you have to re-install each release from scratch.

Xubuntu 16.04 often crashes during install, skip it. That error has not been corrected in version 16.04.02, so the distro has probably other problems too.

Kubuntu, the distro which always makes my PC look really old. I  always buy my PCs second hand and I never had one that could run  Kubuntu. My current PC has a 3-core AMD Phenom processor at 2.3 GHz and  that is too slow for Kubuntu. If you have a new fast processor, Kubuntu has some very nice looking applications. Kubuntu installed automatically my Vbox client, but 5.1.16 instead of 5.1.18 and it refuses to install the correct one. The Vbox host (Windows or Linux) should determine the Vbox client release, not the App store of the guest Linux distro!!

Linux Mint Cinnamon is basically a Christmas tree and it installs too many applications, many of them you won't use. It has rather classical menus, like Windows XP/Vista/7. I hate the fact that I have to reinstall each release from scratch and can't simply upgrade to the next release as with Ubuntu. For me that kills Mint as a main distro.

Ubuntu Mate is basically Ubuntu 05.04 till 10.04 from the previous decennial. It seems reliable, but it is typically a distro for old or nostalgic people.

Ubuntu Gnome, the distro with the biggest memory utilization. It installs the Vbox client 5.1.18, but the client crashes during system initialization. Despite that, It looks nice and operates good, when "tuned/adapted" a little bit. Ubuntu Gnome 18.04 will be better, if it rejoins with Ubuntu 18.04. It needs a little bit of trimming with respect to memory usage. Its standard Gnome GUI often requires two manual actions, where one would be sufficient, but you can change it.
