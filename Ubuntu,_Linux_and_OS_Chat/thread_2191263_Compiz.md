### Post by monkeybrain20122 on 2013-12-01
Have been triple booting Ubuntu, Fedora and Manjora Linux. Fedora used to be on KDE and Manjaro on Gnome shell. Then I realized the combo kind of didn't make sense and also I never could get myself to like kde, so I switched Fedora to gnome shell and Manjaro to xfce. 

I put Compiz and Cairo dock on Manjaro and made it look like my Ubuntu 10.10 for old time's sake. Xfce is kind of old fashion.Thunar is a very basc and I find it hard to work with the "classic" Windows like menu now, but I am amazed how smooth and slick Compiz was (it is an old 0.8.8 version from AUR)All the exotic eye candies and window management tools work perfectly. There is no black windows bug with Nvidia cards, ccsm doesn't forget configurations, you don't have to resort to obscure workarounds for things like scale or edge flip and 3d windows don't break into an unrecognizable mess. It is perfect.

 I love Unity, but it is kind of sad that in order to make Compiz work with Unity, many bugs and regressions were introduced into the 0.9 branch and many of them never really get fixed (though the performance tweaks in Ubuntu 13.xx are great)  Now Canonical is off to Unity 8 and Mir(with new bugs on the way for sure), it looks like the many bugs reports on Compiz are going to be forgotten and compiz is going to be left in its present state for the next LTS (so I see no point in filing or following such reports,-- but I still have a faint hope that I am wrong. :))

What's next? The Linux world is moving to Wayland or Mir. Whatever it is Compiz will be dead in a few years. Big thanks to the developers for giving the Linux world such a fine piece of software, and if there is a heaven for dead software Compiz certainly deserves a place there.
