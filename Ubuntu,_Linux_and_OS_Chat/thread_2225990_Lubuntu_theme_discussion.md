### Post by QIII on 2014-05-29
What a DE should or should look like is really up to the aesthetic sensibilities of the user.  That's not for anyone else here to decide.

That said...

If an attempt is made to make a Linux distro look like Windows for the purpose of making people feel comfortable switching, I think the mission is doomed to fail unless it is made quite clear to the potential user that the similarity is only skin deep.  The same learning curve will apply.

For that reason, I think that the attempt to make the interface look like Windows may actually be counterproductive.  It will raise an expectation that it will be just like Windows and will run counter to the necessary realization that Linux != Windows.

One man's opinion.
