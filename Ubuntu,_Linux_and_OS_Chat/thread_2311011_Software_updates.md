### Post by grahammechanical on 2016-01-23
What version of Ubuntu are you using? I use the development version of the next version of Ubuntu (16.04). That gets updates every day. Released versions do not get so frequent updates.

Do you really not want security fixes to the OS? Go to System Settings>Software & Updates>Updates tab and change the setting in the panel labelled "Automatically check for updates" and change it from "Daily" to "Never" and then you will get peace of mind.

In Ubuntu we do not get updates to most applications. To get newer versions of most applications we have to upgrade to the newer versions of Ubuntu. There are a few exceptions to this.

Perhaps you are used to a different method of operating system development. A method where you have to wait for a service pack to be released before things get fixed.
