### Post by Bucky Ball on 2017-09-17
People generally only post anywhere when they want to complain or they have a problem. Of course you are going to find lots of threads from people with problems here. It is a Ubuntu help forum! That's what we do! Try and find threads that praise Ubuntu. You won't find many, as should be expected.

There are millions using Ubuntu without issue. All my installs have been running pretty much faultlessly for ages. When something pops up, as it invariably will when using any OS, I research for a fix and if unsuccessful, I post a thread here. Threads like that are mostly what you're looking at here. Threads like 'everything is running fine except ...' some niggly little thing that is usually fixed without a lot of fuss.

You should have a read of threads rather than take the number of them on face value. We have several chat areas here. They are not support threads. You are seeing those threads in any thread list. Many threads are solved and people omit to add the 'Solved' tag to them. Many have the 'Solved' tag. Many with and without a tag are solved, but a discussion may be continuing regarding the original issue or something else. These discussions can be educational; we are here to learn in the process of supporting each other through solving problems. 

Good luck, whatever you do.
