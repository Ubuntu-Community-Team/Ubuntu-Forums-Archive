### Post by timfinley on 2015-02-19
To start this little bit off, I would like to say that I have been using Ubuntu for many, many years. Also, I am kind of rushing this bit of writing since I have other things I need to be doing. That said, let's get into it.

First, let's talk about UbuntuOne. Admittedly, I didn't do any research into the reason why Canonical stopped this service. My assumptions are that many users weren't liking it as it could be seen as intrusive. There might have also been security issues that might have been approaching and the company wanted to shift focus on their mobile platform. Either way, I would like to say that I miss that service. It was extremely convenient to simply ctrl-s save a file and pull up a web browser elsewhere and print or edit it. I have tried a few other drop boxes like this but I have either lost patience trying to install it, was confused about the installation or usage, or simply ran out of time dealing with the installation process. The likelihood of some piece of some kind of dropbox code floating around my computer is there, but since I plan on a fresh install soon, I'm not too worried about it.

I understand that it isn't going to come back. But I do miss it. In retrospect, I regret not paying for the service. I was apprehensive about using "the cloud" but have warmed to it since using UbuntuOne. I will never rely solely upon cloud based computing, but I appreciated UbuntOne for what it was.

### Post by deadflowr on 2015-02-19
TLDR

Ubuntuone cloud storage closure?

Cloud storage is a highly competitive field, and Canonical just could not keep up with the rates others can offer.
Simple economics is all.

Did see this, in passing, about Unity:
> I do wish, however, it was either way faster or I was given the option to turn off the internet from it all together.


In the "Privacy" settings page, there is a toggle to turn off online search results.
You could also dig through the software center and remove any unity-scope not related to files or applications.
(I know 14.04 and 14.10 and up have a whole bunch, related to various online queries, like wikipedia.)
