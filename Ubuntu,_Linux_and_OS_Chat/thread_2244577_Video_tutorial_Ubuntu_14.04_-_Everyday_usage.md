### Post by Scunizi on 2015-02-24
PDF (FDF) fallible forms with calculations ...

My system is my primary production machine and I often have to fill out stand-a-lone pdf forms that I've saved to a directory.  The built in readers work great if the form is fairly static.  However, if the form has any type of scripting behind it, ie numerical calculations or whatever, the built in readers fail miserably.  Adobe reader isn't really an option anymore on linux.  It's 2 years old and only available from Adobe's FTP servers if you can find which one it's on.  There is an easy work-a-round I just discovered.  It was one of those face palm moments when I flashed on it.  Open a new tab in Chrome, drag the file to the address bar and viola! a functioning form.

### Post by melchiaros on 2015-03-17
48. Desktop panle with Cairo-Dock

   In Video 48 (of 56) the extension of your desktop with Cairo-Dock is shown to you. Cairo-Dock is an additional docking panel that can be used right beside the default unity-launcher or instead of this. It extens your desktop with some more smoother access options to your system than the default  launcher in Ubuntu can do. It is one of the component right beside compiz-config-manager to transform your desktop away from a classical look and fell to a more modern form that will allow you to work much faster with the system.

[https://www.youtube.com/watch?v=1b0iLRwH_tE](https://www.youtube.com/watch?v=1b0iLRwH_tE)
