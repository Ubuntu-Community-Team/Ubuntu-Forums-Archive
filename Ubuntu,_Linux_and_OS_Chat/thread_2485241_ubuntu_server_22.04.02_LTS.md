I'm a bit perplexed by your questions.

Most people who install a server don't want "desktop display". The point of running a server is to avoid the whole GUI thing with its large attack surface and greatly increased risk. This does require you to learn the command line and this is admittedly a roadblock for many.

What do you want to do next? We have no idea why you installed server, so cannot even begin to advise you.

I run my servers for many things. Among those many, a fileserver, cloud server, calendar server, contacts server, media server, torrent server, newsfeed server... the list is pretty much endless. Only you can decide what you need, so only you can answer your own question.
