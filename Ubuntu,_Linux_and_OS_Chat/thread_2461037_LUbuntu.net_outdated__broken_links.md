### Post by grahammechanical on 2021-04-23
What do you find confusing about ISO images for "Intelx86" and "AMD64"?

Intel X86 refers to a computer processor architecture that was developed by Intel and progressed from having instruction sets 8 bits long to 16 bits, 32 bit and finally 64 bits long.

As it was Intel that developed the first 32 bit CPU, then Linux that runs on 32 bit CPUs is called Intel i386. So, for a 32 bit CPU we need an i386 ISO image. On the other hand AMD produced the first 64 bit CPU and so the version of Linux that runs on any 64 bit CPU no matter who manufactures it is called AMD64. Provided of course that it is compatible with the Intel X86 architecture.

There are other CPU architectures around so X86 is used to distinguish between CPUs compatible with the Intel X86 architecture and other architectures such those designed by ARM.
