### Post by SeijiSensei on 2018-04-09
Give KDE's Dolphin a try.  You can install it on vanilla Ubuntu with "sudo apt install dolphin".  It will bring in some other files ("dependencies") that it relies on to function.

However you will lose some of KDE's unique features, so you might consider taking a test drive of Kubuntu.  You can boot from an ISO image in the "Try Kubuntu" mode, or install a virtual machine manager like VirtualBox and install Kubuntu into a VM.
