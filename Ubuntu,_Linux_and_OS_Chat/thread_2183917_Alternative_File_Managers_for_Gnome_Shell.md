### Post by buzzingrobot on 2013-10-27
Nemo likely brings in many Cinnamon dependencies.  The desktop icons might be disabled via dconf.

In addition to Nautilus and similar file managers, there are several file managers of a different design that might appeal to you.  They're two-paned tools, with a listing of the current directory in ne panel and the listing of a target directory in the other panel.  Search for "Linux file managers" and you will come across them.
