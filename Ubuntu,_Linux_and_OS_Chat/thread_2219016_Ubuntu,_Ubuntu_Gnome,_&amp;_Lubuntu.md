### Post by pfeiffep on 2014-04-22
I've been using Ubuntu (gnome flaskback Metacity) for a while and decided to try out Ubuntu Gnome & Lubuntu (all 3 14.04). 

My observations:

1)Only Ubuntu LTS has a 5 year support plan - the other 2 are only 3 years

2) For my taste Ubuntu with gnome flaskback is superior to the other 2 flavors I just tried. While I didn't have to install flashback using U-G the overall presentation was considerably less polished. All the Metacity functionality was present. When I booted into the default desktop it wasn't nearly as functional or polished as Unity. The installation was extremely smooth and fast since I mounted my Ubuntu 'home' - settings for Chromium and Thunderbird imported without a hitch. 

3) I installed Lubuntu to a flash drive on my laptop - it took 1.5 hours AFTER reboot to get wifi working (I installed using wired ethernet). Another hour and I don't have a printer driver. The HP Photosmart C-6280 isn't even listed in the HP database.[INDENT]***Note that Ubuntu and U-G found the network printer and installed the correct driver just by 'adding a printer'.   ***
[/INDENT]

4) I will be hard pressed to ask a Win XP user to try Lubuntu based on my experience. I felt like this install was 'plug and play with it'
