I have installed kubuntu in a virtual machine to get more familiar with Linux and to write some programs for my new server. As I don´t use kubuntu for surfing the web, reading e-mails and stuff like that, but playing around with it and trying to program programs that need root permissions, I get constantly spammed by password prompts. As I don´t like easy and short passwords as those are useless, this gets really, really annoying. And no, running only the terminal as root is not enough as the GUI is not only a decoration.

I want my OS to simply follow orders and not to play nanny. If I order my OS to break itself, it should do it. If I install a harmful or faulty program, I am out of luck. If I play around with something I don´t know and something breaks, that´s the price of stupidity.

Restoring kubuntu to it´s pre-damaged state would only coast me a click and a minute of waiting with vmware workstation. And that´s a price not worth playing enter the password for some idealist reasons. Security has its price and that price is not always worth paying.

Trying to convince me otherwise won´t help. As I can get any Windows edition for free, including the server variants, I will simply switch my server and delete the virtual machine if I won´t get the root account running. I want to spend my time learning something and being productive and not entertaining a security feature which constantly throws me out of my workflow.

Sorry for sounding rude, but I am really annoyed after three days of tweaking and exploring kubuntu (after years of not using Linux). I simply want to have full control over my very own PC. Something even a proprietary, monopolist like Microsoft is willing to give me, though not by default which is something I agree on.
