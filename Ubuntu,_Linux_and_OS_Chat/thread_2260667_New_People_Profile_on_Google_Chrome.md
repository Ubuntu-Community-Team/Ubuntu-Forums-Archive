### Post by slickymaster on 2015-01-13
If you're on version 39.0.2171.99-1, then that's expected, it's the user/profile button. I think it was initially introduced with Chrome 37 if I remember it correctly.

You can disable it, if you want to. In the Omnibox enter ```
chrome://flags/
```search for **#enable-new-profile-management**&#8203; and disable it in the option box to the left.

Accordingly to Google there are some benefits that this change brings:> 
[LIST]
[*]Since your profile name is displayed, it's easier to see you are signed into Chrome.
[*]You can use incognito mode without going through the Settings menu.
[*]If someone needs to use Chrome on your computer temporarily, you can set them up in Guest Mode. Guests don't need to create a new profile and will not leave browsing history or cookies behind after closing the Guest Mode window. They also won't be able to see your Chrome settings, such as your bookmarks, history, etc.
[/LIST]
