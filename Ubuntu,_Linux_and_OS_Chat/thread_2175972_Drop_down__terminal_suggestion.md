### Post by zero2xiii on 2013-09-22
I personally use guake.

Some custom keybinds:
Toggle Visibility: "Backslash"
New Tab: "+"
Close tab: "-"
Next Tab: "Ctrl + Right"
Prev Tab: "Ctrl + Left"

I rarely use backslash as an escape, if I need to I just close guake and use the regular terminal.

Also have a custom keyboard shortcut to launch guake: "ctrl + alt + backslash"

Cheers
