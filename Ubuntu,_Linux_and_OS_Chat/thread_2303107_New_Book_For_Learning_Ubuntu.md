### Post by neu5eeCh on 2015-11-16
> **shane_faulkinbury2 said:**
> Does everyone think this book is pretty good for learning Ubuntu?

I once bought a book on Ubuntu with a free DVD. Could have been the same title but a much earlier version?

I found it fairly useless.

The book primarily discussed how to use your GUI apps: browsers, email, file managers, etc... But was useless when dealing with bugs or simply fixing things or, for example, locating software when Thunderbird wants to know what program to use. I had curiously little useful information as regards _Linux_.

Useless.

The most useful book I have, which I still regularly refer to, is [LPI Linux Certification in a NutShell]("http://www.amazon.com/gp/product/0596804873?*Version*=1&*entries*=0"). Way more information than we need, but if you need to know how the linux file structure works, it's there. If you need to know how to create and run a bash script, it's there. It's a lovely book. 

Also [Linux in a NutShell]("http://www.amazon.com/Linux-Nutshell-Ellen-Siever/dp/0596004826/ref=sr_1_4?s=books&ie=UTF8&qid=1447705710&sr=1-4&keywords=linux+in+a+nutshell").
