### Post by raphaell2 on 2019-02-07
I hope this isn't a repeat post, but I couldn't find anything about this through the search function. 

I recently switched from Xubuntu 18.04 to Kubuntu 18.04 when I decided that my computer would probably be able to handle it, and that, with all the KDE packages I had already installed, I might as well get myself the whole KDE package. Everything worked fine. Then, in order to have more choices when it comes to artwork, I tried to install the artwork packages for the other Ubuntu flavours. Most of them installed themselves just fine. But then I got an error report when I tried to install edubuntu-artwork. Then I got another error report when I tried to *deinstall* edubuntu-artwork. And afterwards I got error reports about edubuntu-artwork whenever I tried to install anything else. Eventually, the only way I could get out of that mess was by completely reinstalling the whole system. 

So, be warned: if you're running Kubuntu 18.04, don't try to install edubuntu-artwork.
