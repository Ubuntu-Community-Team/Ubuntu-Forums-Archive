Now as for myself when it comes to computers and OS's I am like the first group - I want the car to go from A to B - push button; I'm not a techy, I'm not a programmer, I'm not in IT - I'M A CASUAL END USER and it seems to me that UBUNTU is for the second group.

Now imagine for a moment I had to do that in order to start my car because it was raining or I wanted to go somewhere different??? I would not be impressed.

The ONLY thing I have managed to get working is chrome - I DO NOT USE FIREFOX - all my stuff is synced across all my devices on chrome . BUT that was not good enough because unlike chrome on windows, I was unable to view Netflix because Ubuntu doesn't have a simple silverlight thingy blah blah (really, i'm not interested - as i said, I'm an end user, A to B ok?) so WHATEVER it would take to get that to work, after all the code you guys have been shoveling......Netflix (and loads of other stuff i would guess) does not work within chrome. SO YES CALM DOWN I GOT THE UBUNTU FIREFOX BASED NETFLIX THINGY BLAH APT GET BLAH BLAH.

BUT - it is so poor in its performance that it is not watchable. (at this point i must make clear that win 7 on same machine did everything just fine, so no, no hardware issues)

I don't look for trouble, honest, I just thought ubuntu was supposed to be so great. Obviously, it isn't. It just isn't worth all these massive headaches every time I think 'ooo, id like to just do this or that' and what happens - hours and hours of endless searching through forums - it's like the quest for the holy grail.
End result is I'm not prepared to do that to myself any more. So as time goes by I will slowly wind up my usage of ubuntu and say goodbye and good riddance.
