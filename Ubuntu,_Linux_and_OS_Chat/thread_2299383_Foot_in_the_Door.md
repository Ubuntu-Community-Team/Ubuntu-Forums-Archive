### Post by Bucky Ball on 2015-10-18
Welcome. You could start by learning how to install Ubuntu. 

[Download the ISO]("http://www.ubuntu.com/download/desktop") (I'd suggest 14.04 LTS as it is a long-term support release and you won't need to upgrade in January)> create install media (USB or disk)> boot from that and 'Try Ubuntu' to see how you like it. This will not install anything to the hard drive or change your current setup in anyway. It is a 'live' session running from the install media (so a bit slower than a hard drive install).

Best to work out what you want to do, then do it and start asking questions when you hit brick walls. What do you want to do with the computer and OS? Get the OS installed is a good start. :)

PS: Depending on your hardware specs, Ubuntu may be too heavyweight for your machine. You can always try a lighter flavour, like Xubuntu or Lubuntu. Good luck and just ask if you get stuck. Post specific questions in the appropriate support areas, one question a thread and give them descriptive titles to improve your chances of help.

Welcome to Ubuntu, we're glad to see you here!

Well, I guess the most important thing you can do to start with is actually *use* Ubuntu for a while and get acquainted with it, like Bucky Ball said. If you run into problems, don't be discouraged, just ask. There's almost always somebody here who can help you out.

Secondly, if you want to get to know your operating system better, start playing around in the commandline (aka the terminal). There's an introduction [here]("https://www.digitalocean.com/community/tutorials/an-introduction-to-the-linux-terminal") that you can read if you're interested.

And lastly, if you want to understand the community a bit better and have some spare time, why not read up on the history of Ubuntu, Linux and Unix (the operating system that Linux is based on)?
