### Post by Version Dependency on 2013-07-02
Debian has more packages in the official repos.  The AUR is awesome....but it is a bit of a crapshoot...kinda like PPAs in Ubuntu.  It is a community maintained repository and thus you can run into problems like multiple versions of the same packages, out of date packages, broken packages, improperly configured packages, and (never heard of it but possible) malicious packages.
