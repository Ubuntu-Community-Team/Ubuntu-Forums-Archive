### Post by grahammechanical on 2014-08-31
> [COLOR=#000000] if the future of Ubuntu is a mobile/tablet style (Windows 8 style) desktop environment,[/COLOR]

It is not. You are wrong about that. Although I cannot get past the login screen I still update using recovery mode and I see all the standard updates that Utopic Unicorn is getting plus those special to Ubuntu for phones and tablets.

The aim is to have one Ubuntu code base that will present a phone type UI when installed on a mobile phone device and a tablet UI when installed on a mobile tablet device and a desktop UI (that is Unity 8) when installed on laptops and desktop machines. But the aim is far higher than just that.

The intention is for Ubuntu to be pre-installed on super phones that can become a tablet when connected to a tablet size screen and a PC when connected to a monitor, mouse, and keyboard.

Imagine a PC that is also a mobile phone even when being used as a PC. Imagine a tablet that is also a mobile phone when it is being used as a tablet and when it is being used as a PC.

The main difference between Unity 8 and Unity 7 is that Unity 8 has been re-written to run on Mir. Unity 8 will look much the same as Unity 7 when Ubuntu 16.04 is released. Unity 8 will come with additional functions that are suitable for mobile devices which Unity 7 does not support.

Besides, Unity 8 is not going to be back ported to Ubuntu 14.04. So, the future of Ubuntu is at the moment 14.04 LTS.
