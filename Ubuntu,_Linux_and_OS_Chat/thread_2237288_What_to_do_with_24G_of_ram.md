### Post by Jonor on 2014-08-01
See [this recent thread]("http://ubuntuforums.org/showthread.php?t=2228236") ibj.
I don't do virtual or gaming but cache memory used to go above 16GB about 100 hours after a reboot (mostly use suspend here).
My switching graphics from onboard to a card seems to have reduced that now though.
As unfettered cache supposedly brings the RAM to do a lot of the work done more slowly by other drives
and since matching multiple sticks of RAM is considered important, if there is a spare RAM slot left (? - but seems likely),
i suggest saving up for another matching stick while their availability is at its best.
