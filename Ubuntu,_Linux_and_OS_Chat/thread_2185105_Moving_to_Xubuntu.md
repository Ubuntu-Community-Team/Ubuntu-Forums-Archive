### Post by Bucky Ball on 2013-11-04
> **Cariboo1938 said:**
> Are there any difficulties to expect? Special precautions to consider? Minimal requirements to check? etc.

Not really. But I will say this: make a plan before you start. Prepare to make one long install code line, which is what you will be installing after the kernel is in. Mine is a bit like this off the top of my head (but not conclusive):

```
sudo apt-get install firefox thunderbird brasero vlc gimp synaptics xfce4 transmission kompozer audacity gcc getdeb pcmanfm
```

... and that is not conclusive. You need much less than this to get a working system. There are others things you are going want/need but this will get you up with a desktop, file manager and synaptics. You can take it from there and install things as you need them. but you will find things you might need to add as you go. 

It can take a little time to tweak and research once you're up and running (this is mostly for things that you didn't realise you needed and will become obvious when you try do something that involves some background software, not a GUI app as such).

PS: In effect, you could probably get away with:

```
sudo apt-get install xfce4 synaptics pcmanfm firefox
``` 

... and do the rest from a desktop once all is happening. Good luck.
