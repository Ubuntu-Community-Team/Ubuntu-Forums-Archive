### Post by Bucky Ball on 2016-10-19
*Thread moved to **Ubuntu, Linux and OS Chat**.*

Welcome. You can boot Ubuntu on a disk by itself, you can dual-boot (or multiple-boot) Ubuntu with other operating systems (Linux, Mac, Windows) all on their own partitions or you can run Ubuntu as a virtual machine in Windows and probably Mac. I'm no expert on Mac.

Ubuntu does not work 'over' Windows, nor is it a drop-in replacement for it. Most Windows programs will not work natively in Linux.
