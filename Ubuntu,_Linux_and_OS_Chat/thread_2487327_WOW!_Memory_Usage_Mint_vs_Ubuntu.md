### Post by exploder on 2023-06-01
I have had both Ubuntu 22.04 and Mint Cinnamon 21.1 installed for a long time now. Using the exact same tweaks, I can get cinnamon down to about 650 MB of RAM at idle, about the same as XFCE. Ubuntu is always about 300 MB higher, not a big deal by any means. The biggest difference I notice is that Ubuntu has more little graphical glitches with both sessions and the mouse pointer does not have the same smooth feel for lack of a better description. 

An example of glitches is, type you password wrong for synaptic in a default Ubuntu session. The password box and radio buttons turn black. Opening my bookmarks in Firefox the first time the browser starts and wayland displays thick black edges. It's like wayland is struggling to refresh the display at times. Some apps work fine under wayland and some work better under xserxer. I have none of these glitches in Mint 21.1.

I understand why wayland is becoming the default session for distro's but it is going to take a lot of time to get the bugs sorted out. These are just my thoughts and experiences so far.
