### Post by TheFu on 2017-01-18
Support for 15.04 ended about a year ago.  You really need to get on an LTS release like 16.04 (5 yrs), not 16.10 (6 months).
Newer isn't always better.  [https://wiki.ubuntu.com/Releases](https://wiki.ubuntu.com/Releases)  Plan the OS you use carefully.

I find all books published to be out of date BEFORE they are available. Learn about the non-GUI parts of Ubuntu and Linux in general. Those don't change all that much over time. In the last 20 yrs, maybe 80% of them are the same. For non-GUI stuff, [http://linuxcommand.org/tlcl.php](http://linuxcommand.org/tlcl.php) is a reputable book.  Almost every bookstore will sell it. The PDF is free online. No hassle to get.

If you want to be a power-user, learn the shell/CLI interface.

If you will depend on books, Ubuntu 14.04 (the Unity type) maybe the only option. Most of the official flavors have rather small groups of developers and probably don't have the wherewithal to publish books even for their LTS versions, let alone the intermediate versions such as 16.10 (and the now unsupported 15.04).
