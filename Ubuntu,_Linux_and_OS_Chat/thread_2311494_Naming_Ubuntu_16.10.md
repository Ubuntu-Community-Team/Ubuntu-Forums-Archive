### Post by grahammechanical on 2016-01-27
I would not be surprised if there is not a 16.10.

Keep listening out for something called Ubuntu Personal. Or Ubuntu Snappy Personal. Or Ubuntu Snappy Desktop Next. It all seems to relate to the same thing.

This will be built on snappy Ubuntu core. It will run on Mir and have Unity 8 as the UI. It will have transactional updates. The OS will be a snap package as will the user environment and all the applications.

Ubuntu Personal will not use apt-get and it will not be updated package by package as at present. Because the whole OS part of Ubuntu will be a snap package it is the OS snap package that gets upgraded. And it will be done in such a way that if the upgrade fails in some way the install will revert or roll back to using the previous, working version OS snap.

If this vision of Ubuntu Personal does become a reality it will completely change the way Ubuntu is released. No more six monthly releases. Just regular Over The Air Updates (OTA) like Ubuntu phones.

[https://developer.ubuntu.com/en/snappy/](https://developer.ubuntu.com/en/snappy/)
