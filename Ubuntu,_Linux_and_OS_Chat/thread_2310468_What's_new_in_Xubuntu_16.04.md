### Post by neu5eeCh on 2016-04-11
From what I can tell --- not a lot.

> "From a development perspective, we&#8217;re targeting bug fixes to make 16.04  super stable. Don&#8217;t expect a lot of new features, so testing done at all  parts of the cycle will be incredibly relevant." From [here]("http://xubuntu.org/news/16-04-package-testing/").

One of XFCE's goals is to keep the DE light, so one can't be adding new features a-la Cinnamon or Plasma5. Over the years, Xubuntu has added some Control Panel features (controlling window decorations and color schemes for example) and added to Thunar's capabilities. They've also added and refined various plugins for the panel but what's really kept their hands full (far as I can tell) is GTK3 integration -- still think that's ongoing. Sexy new features isn't what Xubuntu is about. 

However, if you love Xubuntu and love sexy new features, then check out [Voyager]("http://voyagerlive.org/") -- my fav distro next to Kubuntu. This is XFCE/Xubuntu with the whole "lightweight" fetish thrown out the window.
