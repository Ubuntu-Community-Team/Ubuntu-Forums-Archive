Alrighty, with your system, download VirtualBox or like VMWare Player and run a virtualized instance of Ubuntu. As for the programs here's a few just to throw out:
Video: KDENLive or PiTiVi
Audio: Jokosher Audio Editor
2d Animation: Pencil Synfig or there's others; I've used Synfig its similar to Adobe Flash
Gaming: That's the tough part if you get steam games that have linux counter parts, you're good, but depending on the game that can be a different story. You'll probably find 100s of threads (if not more) just asking about games or games similar to or games like xyz. 
Multimedia Format Changing: WinFF or XFCA

If you like the virtualized instance of it, I would HIGHLY recommend booting off of the Live CD and download the apps I mentioned above and try them out ***YOU DON'T HAVE TO INSTALL UBUNTU TO TRY OUT THOSE APPS AS YOU CAN INSTALL AND RUN THEM WHILE USING THE LIVE CD PORTION***

That way if you find it's not for you you're not out anything, be sure you have backups of your data and OS just in case! =D

### Post by Old_Grey_Wolf on 2014-05-20
If you try it in a virtual machine or a USB, try searching the Ubuntu Software Center. Just enter search terms like animation, audio editing, etc. The software has ratings provided by Ubuntu Software Center users. It also has as links to the developer's site so that you can review it to determine if the software may serve your needs before installing. There is a handy uninstall button if the software doesn't work or meet your needs. There are a lot of programs available.
