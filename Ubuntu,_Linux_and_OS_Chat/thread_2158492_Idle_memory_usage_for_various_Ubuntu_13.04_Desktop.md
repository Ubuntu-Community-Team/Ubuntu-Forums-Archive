### Post by Copper Bezel on 2013-06-29
Unfortunately, total memory usage at login isn't a very good indication of overall speed and responsiveness, particularly when graphics cards get involved. Each desktop also has its own selection of default apps, which can be lighter or heavier weight, and users aren't always going to use the same applications or run the same services at startup. Numbers sound convincing, but it's not more precise than simply talking in broad categories and saying that Xfce and LXDE are lighter-weight, while Unity and KDE are heaver.
