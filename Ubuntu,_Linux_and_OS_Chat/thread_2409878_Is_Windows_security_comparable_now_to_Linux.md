both statements are not true or are only partially true. in 98 you were the super user, but on XP the kernel is NT and multiuser. you can setup administrator and then multiple users, and just use the users. this is only one layer of security. there were multiple holes in the OS on the start and majority of breaches came from users not updating their OS, their browsers and antivirus (there is a lot more malware made for windows due to it's huge user base). browser is nowadays usually an entry point so having some sort of script blocker can help a lot there but is not user friendly. XP also had a few ports opened by default. close that and you solve plenty of issues. i am still using XP (since a week ago in dualboot). properly setup XP was a secure OS.

in linux administrator has no password, though the user can be elevated to admin. this is one security layer. there are many others. can you still hack linux? yes you can. but there is not that much malware on linux. it also has a lot more "out of the box" security features than XP.

now on to windows 10 - they improved their security a lot. virus scanners are now built in as are some other features (such as improved firewall, better user management...). there are also a couple of bad/annoying features on win 10.
1. in home version updates can break your PC or even delete the personal files (see October update issue). basically they let go of the testing team and then let enthusiasts "test" before releasing the OS. while the user reported issues, they 
2. update can require multiple reboots to complete and on the home version it is practically forced upon the user.
3. the PC reports some data to microsoft and to some 3rd parties. we don't know what that data exactly is, since they do not want to reveal it.
4. the OS itself need plenty of disk space, though it does support a lot of hardware.

5. we don't know yet what will happen when hardware starts to get "too old" for MS support.

the version i use is enterprise. our IT team tests releases updates to us. it also manages most things. overall i am ok with the OS in this version.

is windows OS home usable and safe for user that doesn't want to worry too much about their PC. i would say yes, but make sure the files are backed up (same goes for linux).
