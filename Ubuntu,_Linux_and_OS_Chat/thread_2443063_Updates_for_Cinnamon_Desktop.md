### Post by deadflowr on 2020-05-11
Cinnamon from what?
Ubuntu?
Linux Mint?
PPA?

As far as I can tell, Cinnamon on Ubuntu is more or less an unsupported package pulled from upstream Debian, as is.
Any updates have to make there way through Ubuntu SRU system:
[https://wiki.ubuntu.com/StableReleaseUpdates]("https://wiki.ubuntu.com/StableReleaseUpdates")
Since cinnamon on Ubuntu really doesn't seem to have anyone to maintain it, not a lot of SRU is going to happen.

Cinnamon for Linux Mint is constantly updated since they actually maintain it and develop it there.

And PPAs are solely at the whim of the PPA maintain/publisher.
