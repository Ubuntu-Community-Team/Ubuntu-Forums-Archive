### Post by buzzingrobot on 2013-12-11
> **gregory12 said:**
> how does that help me? get your own thread?

Here's my take:

1. No adequate Linux Evernote client exists.
2. Evernote has made it clear they do not intend to create a Linux client.
3. The Evernote API is out there so an adequate Linux client, in theory, could be created.  I expect the odds of that happening are directly proportional to Evernote use by Linux developers.
4. Therefore, an ad hoc DIY approach leveraging existing tools to meet specific needs is the best alternative.

### Post by saurabh5647 on 2014-02-26
If you want to use evernote on Ubuntu, then Nixnote (Formerly Nevernote) is the best alternative. However, if you just want a note taking software for Ubuntu, there are some good ones like Tomboy, RedNotebook and Zim Desktop Wiki. I generally use Tomboy but there is also a lighter version of Tomboy called Gnote.

Btw, I maintain a website for open source alternatives to popular software. You can check the alternatives for Evernote here:
[http://www.opensourcealternative.org/business/open-source-alternative-to-evernote/](http://www.opensourcealternative.org/business/open-source-alternative-to-evernote/)
