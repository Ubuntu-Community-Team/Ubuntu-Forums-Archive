### Post by yancek on 2014-06-28
>  What if I am not happy with Ubuntu?

You can try it without installing it by running from a DVD or flash drive.  You can get information on how to do that at the Ubuntu download site or here or by simply doing an online search.  If you try it and don't like it, don't install it.

> What if the files I saved from windows on external hd won't be accepted by Ubuntu?

Linux/Ubuntu can create and format windows partitions/filesystems.
Linux/Ubuntu can read/write to windows partitions.

A default windows installation can't read, write to or even recognize Linux filesystems and generally show them as unallocated space.

> Will my laptop really run smoother?

Since you haven't posted any information about your hardware it is not possible for anyone to answer that.

What "pro app" are you referring to?  Windows is usually better at games so doing a dual-boot might be better for you.
