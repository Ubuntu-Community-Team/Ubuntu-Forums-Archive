### Post by engineerarun on 2015-11-13
Just wanted to share two utilities I wrote for cmdline fans, both very much feature complete and hopefully makes life easier by bringing the web a bit closer to the terminal.

[MarkIt]("https://github.com/jarun/MarkIt"): a cmdline bookmark manager with support for tagging, auto-fetching of page titles and searching.
[google-cli]("https://github.com/jarun/google-cli"): navigate through Google results from the cmdline. Highly flexible with custom time-range, number of results, starting page.

Both the tools allow you to open any URL directly in the browser.

Hope you enjoy them!
