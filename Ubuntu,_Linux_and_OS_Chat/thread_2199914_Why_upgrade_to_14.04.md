### Post by Bucky Ball on 2014-01-16
*Thread moved to **Ubuntu, Linux and OS Chat**.*

Not a support request.

Short answer: no. 12.04 has five years support so you are set until April 2017.

Longer answer: The newer LTS will have improvements, changes, but most importantly, newer versions of software. This is no biggie if the older versions do what you want. I always have a fully tweaked LTS install as my main install and play around with others on another partition.

Currently testing 14.04 actually. If you have the space to create another partition, install it and see what's different so far. Expect breakages as it is not released until April and strictly testing. You could also try it out as a virtual machine.

### Post by ajgreeny on 2014-01-16
Keep 12.04 if you want to and use ppa repositories to use updated versions of software.  I am using 14.04 (Xubuntu 14.04 64bit) as a virtual machine in Virtualbox, and it seems to be very stable and extremely good.

There are masses of updates coming at the moment, not surprisingly, as it is still only at alpha stage, but nevertheless I think I shall be doing a clean install when it finally arrives, so it will be taking over from my current Xubuntu 12.04 64bit system.

I have tweaked my 12.04 quite a lot with updated versions of xfce4, libreoffice, gimp, and several other packages that I can't remember at the moment, all from ppa repositories, so I already have some of the same package versions as will be in 14.04, I think, so as long as the underlying OS is stable I am pretty sure the application packages will be as well.

It is also worth pointing out that if you are using Xubuntu 12.04, support ends in just over another year, as it only has three years support, not 5 years. 
If using Lubuntu 12.04, it has already lost support as it was not an LTS version.
