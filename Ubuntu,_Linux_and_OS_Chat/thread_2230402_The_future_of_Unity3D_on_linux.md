### Post by monkeybrain20122 on 2014-06-19
It already doesn't work in Chrome. But it will continue to work on Firefox as far as I know. It is just that like other plugins it will be set to 'ask to activate' instead of 'always activate' by default. No big deal really. 
On Chromium they have made a patch that would restore pipelight but it will work only til the end of the year when google remove the codes for supporting plugins for all platforms (right now the codes are there, just disabled)

[https://answers.launchpad.net/pipelight/+question/250336](https://answers.launchpad.net/pipelight/+question/250336)
