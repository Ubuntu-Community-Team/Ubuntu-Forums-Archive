### Post by Lars Noodén on 2015-08-03
It sounds like you are looking for a meta search engine.  There are a few around: metasearch, dogpile, ixquick, etc.  You'll have to investigate to see if they search the engines you want searched.  There are also some add-ons for Firefox if you are using that still : [https://addons.mozilla.org/EN-us/firefox/tag/metasearch](https://addons.mozilla.org/EN-us/firefox/tag/metasearch)
