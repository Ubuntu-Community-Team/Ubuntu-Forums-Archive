### Post by TheFu on 2017-05-01
Welcome to the forums.

WINE doesn't support every possible Windows program you might have.  In fact, it doesn't support **any** of the programs I want to use. Spent years getting WINE to work with a few programs, then at the next OS upgrade and WINE update, everything broke again. I worked for 40+ hrs to get it working again.  Then bought a newer version of the Windows software and could never get that version working under WINE. 
At that point, I loaded Win7 into a virtual machine and loaded any Windows program into it. They almost all work - just the heavy GPU-dependent programs don't work.  Under WINE, about 80% of the program features worked. Using Win7 inside a VM, 100% of the program features work. It has been years and everything is still working. No hassles.

I agree with your desire to never load Win8-Win10. I've been blocking the MSFT updates that added all the spying, they call it "telemetry", and the changes to the EULA over a year ago.

WINE is not the greatest thing for everyone.  You'll need to do some research on the exact programs you want to run under WINE before deciding if that is an option or not.  There is a DB of supported applications at **winehq**, each with detailed setup, install and running steps.  If you aren't interested in doing all the legwork yourself, you could buy a WINE addon which embeds the DB into a GUI and I hear they automatically setup the required configuration for the different programs. Never used any of those tools.

I don't game on Windows/WINE. The programs that didn't work easily/at all were business, financial, and drawing apps.

IMHO.

Please get backup religion ASAP. Bad things can happen to any computer. Backups would have solved your original issue. There will be more issues that having versioned backups would solve too.  Please get backup religion ASAP. Seriously.

### Post by TheFu on 2017-05-03
**Getting Win7 and Patching:**
You can download ISOs from MSFT directly, but I doubt Win7 ISOs are still on their site. They don't want people running it, as we can tell by all the forced upgrades, which happened without clear enough notification for many end-users.  I took extreme steps to prevent it.  
Regardless of what others say, there is no way to disable MSFT "telemetry 100%" using tools running on Windows. Can't be done. Thing about it and use some common sense, don't simply believe anyone else.  With **any** patch, they can change your OS to do anything. People who patched Window 7-10 after Feb 2016 agreed to allow it in the EULA.  Only blocks from outside Windows can stop it and I expect domains unrelated to Windows or Microsoft will be used to make it happen.  They will call it a "service" to end users, even when we've clearly decided NOT to update (it isn't an upgrade).

**Linux Gaming:**
As a business person, I would never agree to fund desktop games on Linux just for the sake of gaming on Linux.  I'd need a much stronger reason - like knowing that a Linux-based gaming platform was coming out. People only running desktop Linux are less than 1% of the market, so it really doesn't make sense. It is a chicken/egg problem.  There are lots of people running Linux with access to any other OS.  The polls seem to assume people have only 1 device or 1 OS.  I have 20+ Linux systems. Does anyone count that compared to the 2 running Windows and 1 running FreeBSD? Not that I've seen.

Remember how the SONY PS2 supported Linux **and** PS2 gaming?  Why did SONY decide to break the Linux support?  Because they weren't making money off anyone who elected to run Linux, but not game.  There were govt labs using thousands of SONY PS2 devices for computing needs. They weren't buying games, so no extra profits to SONY.  Often, selling the hardware is a slight loss-leader for gaming platforms, expecting to earn $500+ on future games sold.

The same applies to Chromebooks.  Google subsidized the early ones to gain more complete access to personal data.  With a stock chromebook, not using google and google docs and google hangouts and google (50 other things) is hard.  I know. I did it.  Google makes MSFT seem like the privacy army in comparison.
