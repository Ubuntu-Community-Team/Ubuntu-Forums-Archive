### Post by whatthefunk on 2014-06-12
The nice thing about having a fast machine is that you can have a pretty desktop AND run intense programs.  My main computer has an Intel i5.  I run Kubuntu on it and have never had a problem with lag or anything else, even when running resource intense programs.

If youre still wanting a simpler DE, try out Lubuntu or Crunchbang.  Theyll minimize your resource usage but will give you a far easier, safer computer than Arch.
