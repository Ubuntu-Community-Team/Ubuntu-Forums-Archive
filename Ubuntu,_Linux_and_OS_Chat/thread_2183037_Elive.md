### Post by ajgreeny on 2013-10-23
I haven't seen elive for a long time but did try it as a live CD several years ago, and it certainly was fast.

Have you also had a look at Bodhi, which uses the same E17 DE, but rather than the debian repos which elive uses, Bodhi uses ubuntu repos along with its own.  I ran it on my netbook for a while and it is screamingly fast, giving you the option to install whatever you want from the ubuntu repos that you already know and use.  It does not have many applications in the default installation, leaving the user to add their own choice of office apps, web-browser (though I think it has midori as default), and graphics package.

Worth a try, I think.
