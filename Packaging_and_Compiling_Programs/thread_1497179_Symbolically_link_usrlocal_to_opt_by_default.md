### Post by zer0pulse on 2010-05-30
Just a thought, since I've been doing this for ages.
Whenever I install a new linux distro whether it be ubuntu or redhat or what have you, I always link /usr/local to /opt
So when I can't find a .deb of a program I'm trying to install and I have to compile something from source, it always defaults to /usr/local.
So later when I have to look for it to add it to Ubuntu's menu I'd rather go to /opt than dig around in /usr
Also I've found, not often though, that sometimes makefiles like to go to /opt by default.

Anyway, that's my suggestion.
