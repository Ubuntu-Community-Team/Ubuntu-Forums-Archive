### Post by bobbocanfly on 2008-07-19
An easy way to start is with a simple bash script with minimal dependencies. Read [http://ubuntuforums.org/showthread.php?t=599473](http://ubuntuforums.org/showthread.php?t=599473) to see where to get started. There is an official packaging guide maintained by the Ubuntu developers that has an example tutorial you can follow to learn the basics. After that, trying to do a bash script is very simple. 

For an example bash script package you should download 'showfsck' which I bugfixed a while ago (it was my first contribution to Ubuntu).
