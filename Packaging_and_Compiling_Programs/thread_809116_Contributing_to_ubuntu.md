### Post by Vadi on 2008-05-29
I guess it depends on which *part* of Ubuntu would you like to contribute to. Contributing to Ubuntu itself to a mortal means either a) translating, b) documenting, c) trianging bugs, from what I grasp.

I'd say it would be best for you to help out some open-source project that you're interested in with coding. For ideas, either look on brainstorm.ubuntu.com, or browse through projects available on launchpad.net.
