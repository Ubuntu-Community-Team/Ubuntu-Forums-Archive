### Post by ian-weisser on 2014-09-04
Okay, do you know Python?
After you know Python, you learn dbus to interface with the rest of the system. Start with the *d-feet* application, provided by the d-feet package.
After your application is complete, you learn how to create a .deb file (a package).
When you are good at creating a .deb file, you submit it to Debian (it it's free) or Ubuntu (if you plan to charge for it).

Then you download the SDK from [http://developer.ubuntu.com](http://developer.ubuntu.com) , and suddenly the system API (dbus) and packaging system (.deb creation) make a lot more sense.
