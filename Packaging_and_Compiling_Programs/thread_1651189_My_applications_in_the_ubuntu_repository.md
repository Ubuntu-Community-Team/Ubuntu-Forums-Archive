### Post by MadCow108 on 2010-12-23
First you should create a package of your program (see links below).

You can offer this on your website or add it to a ppa (personal package archive) from which users can install your software over the package managment system:
[https://help.launchpad.net/Packaging/PPA](https://help.launchpad.net/Packaging/PPA)

When you feel ready and willing to support it for the lifetime of the debian/ubuntu release you can attempt to get it into main repositories.

Some links:
[http://ubuntuforums.org/showpost.php?p=10053129&postcount=2](http://ubuntuforums.org/showpost.php?p=10053129&postcount=2)
[http://wiki.debian.org/DebianMentorsFaq](http://wiki.debian.org/DebianMentorsFaq)
[http://www.debian.org/devel/wnpp/](http://www.debian.org/devel/wnpp/)
