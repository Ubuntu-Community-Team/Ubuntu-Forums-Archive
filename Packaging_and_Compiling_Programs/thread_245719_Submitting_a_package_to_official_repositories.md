### Post by mlind on 2006-08-29
You should probably post your request to #ubuntu-motu on irc.freenode.net or MOTU [mailing list]("https://lists.ubuntu.com/mailman/listinfo/ubuntu-motu").
You need to get someone to package this application, then submit it to [REVU]("https://wiki.ubuntu.com/MOTU/Packages/REVU") or other channel for possible inclusion to Ubuntu.

I'm not sure what's the more official channel for this, Debian has [http://www.debian.org/devel/wnpp/requested](http://www.debian.org/devel/wnpp/requested).
Submitting a bug report to MOTU without asking first sounds bit rude.
