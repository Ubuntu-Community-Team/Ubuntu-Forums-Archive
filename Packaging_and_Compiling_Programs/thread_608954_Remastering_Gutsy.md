### Post by smartboyathome on 2007-11-11
Still not the right forum, but I will help you. :)

You should use Remastersys. It will take a snapshot of a currently running system and turn it into a backup (which keeps users in tact) or a distro (which creates a different user). I find this better because I can tell if the stuff actually works before I make the CD. Anyway, you can get it from this repository (add it to your sources.list using System > Administration > Software Sources):
deb [http://www.remastersys.klikit.org/repository](http://www.remastersys.klikit.org/repository) remastersys/
Then, reload your software list and install remastersys.
