### Post by MrStill on 2010-08-11
The g++ compiler comes as part of GCC which should already be on your box. Simple editors are Gedit (GNOME) and Kate (KDE) which do keyword color coding and bracket highlighting. One of those two editors should be on your box as well based on the desktop manager you have. 

As far as IDEs are concerned:

NetBeans has a C++ plugin; but, I don't know if it comes installed from the software repository. If you download the installer from the NetBeans website you can choose the version with C++ plugin installed. If you use NetBeans you may eventually end up messing with makefiles.

Eclipse also has a C++ plugin; Eclipse, in my opinion, can be tricky. It can be installed through the repository, but you will need to install the C++ plugin.

Code::Blocks is another IDE. It is in the repository and does C++ by default. I've never used it and know very little about it

I use Gedit and g++ for small projects and NetBeans for larger ones.

I hope this information is useful to you.
