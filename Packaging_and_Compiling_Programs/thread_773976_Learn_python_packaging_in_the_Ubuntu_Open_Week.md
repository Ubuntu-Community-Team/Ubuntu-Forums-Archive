If you wrote some python code that you want to package or know a cool python application of which you like to make a deb installer, the python packaging session is all for you! 

This session will be hosted by the respected Emilio (pochu). I've worked closely together with him (and others like POX & ScottK) to make sure these packages landed well in Ubuntu Hardy and Debian Unstable:
