### Post by lykwydchykyn on 2010-04-29
Pretty much any decent text editor on Linux will offer syntax highlighting and a shell.  Gedit, Kate, Emacs, geany, etc.

There are some that are python-specific like stani's python editor, but you might as well use something that works well with things other than python.

It might help to know:
 - what you intend to write in Python 
 - what other kinds of text editing you do
 - what environment you work in
