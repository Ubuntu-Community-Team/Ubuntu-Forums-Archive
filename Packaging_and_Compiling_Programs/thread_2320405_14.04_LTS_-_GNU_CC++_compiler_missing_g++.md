### Post by T.J. on 2016-04-14
No, Xinghao_Chen, it won't hurt anything.  The VAST majority of the source code in UNIX is C rather than C++.  The operating system kernel and utilities are almost entirely in C, while user programs sometimes use C++.  Because of this, C++ is normally not considered essential.

You can easily install the C++ compiler with *sudo apt-get install g++*.

Good luck to you.
