### Post by geraldm on 2007-11-23
(not an expert in 64-bit stuff)
The log shows that it needed to link libc.so or libc.a. 
It searched, and found incompatible lib, and continued to look for a compatible version, which could not be found.  It needs that to 
produce an executable, so it declared that your C compiler cannot
create executables.  The file should come in glibc package.

Check your libs closely to see if you actually have a compatible version
and the error message was wrong? 

Gerald
