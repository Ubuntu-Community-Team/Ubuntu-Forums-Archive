### Post by zombifier25 on 2012-05-12
Type this into the terminal:
```
apt-get source packagename
```
It will download packagename's source code and place it in the current directory. Happy examining.
More info here: [http://askubuntu.com/questions/28372/how-do-i-get-the-source-code-of-packages-installed-through-apt-get#28373](http://askubuntu.com/questions/28372/how-do-i-get-the-source-code-of-packages-installed-through-apt-get#28373)
