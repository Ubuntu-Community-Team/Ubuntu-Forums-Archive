### Post by sonofusion82 on 2009-01-02
first have a look at this: [https://wiki.ubuntu.com/ContributeToUbuntu](https://wiki.ubuntu.com/ContributeToUbuntu)

great to hear you want to contribute back.
there are so many ways you can contribute.
helping in this forum, documentation, wiki, language translation and software/hardware testing are some that doesn't even need detailed knowledge in programming and compiling codes.
and it doesn't even need to be ubuntu specific, for example, if you like gnome, you can help contribute to gnome project which will eventually benefit ubuntu and all linux/gnome community.
for me, i m a software developer myself and i like kubuntu but KDE4 has not been working too well, so i decide to do some bug fixes for KDE.
like many advices, start small and specific. once you get the hang of it, you will gain confidence for yourself and also from the community. from there, you can get to know more and explore further.
