### Post by bear24rw on 2009-09-02
g++ is command line
```
g++ main.cpp -o main
```
will compile main.cpp, to run it
```
./main
```

if you want a GUI i highly recommend codeblocks
```
sudo aptitude install codeblocks
```

its just like devc++ but alot more powerfull and you dont need crossover
