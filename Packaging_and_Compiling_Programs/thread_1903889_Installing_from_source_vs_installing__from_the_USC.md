### Post by mathprof on 2012-01-03
When should one rely on the Ubuntu Software Center and when should one install from tarballs (e.g., from source)?   Often one can find more up-to-date versions than are available in the USC.   Any opinions?

---

### Post by Bachstelze on 2012-01-03
Basically, it all boils down to: do you *really* need the "up-to-date" version? Remember that fixes for critical bugs (including but not limited to security-related ones) *are* applied to the version in the Ubuntu repositories, even if the version number does not reflect it. The only reason why you would want to install from source is when you need a feature that is only present in the newer version (and you can't find a third-party repository that has it).
