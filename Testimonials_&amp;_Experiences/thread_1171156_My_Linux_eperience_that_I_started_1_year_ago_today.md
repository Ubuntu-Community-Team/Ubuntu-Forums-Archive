### Post by Bigtime_Scrub on 2009-05-27
I started using Linux when I was unemployed and broke. I passed the time with a new hobby, to learn Linux and make an older computer just work with it. It's been a long road with a lot of bumps. The only experience I ever had with Linux is when a friend of mine put Red Hat on a computer I had that was full of viruses that ran Windows 2000 Prof. It did not end well and eventually I learned how to a) Pirate Windows and b) become a Windows "power" user.

I did my homework before even downloading the .iso file for a live CD. When I was reading I installed Ubuntu 8.04 Hardy Heron. To be honest I did not have very high expectations. If I could surf the net with it I would have considered that a miracle. My first bump was flash support for you tube. I then realized my cd's and dvd's didn't work either and I learned about ubuntu-restricted-extras. 

I had an I-pod classic that I learned how to "jailbreak", "hack", or whatever you call it when you get rid of all that apple firm ware so you can use gtkpod. 

I did my first dual boot with Ubuntu and Vista. 

I used Knoppix, Puppy, Linpus, DSL, CentOS, and Fedora.
I was a real distro slut.

I can now repartition a hard drive using fdisk, install slackware, and recompile a kernal. 

I feel like I can do it all and yesterday I deleted my Vista partition. I don't need it anymore because I never boot into it. I'd rather have the free space to give to Ubuntu. I tell my friends to try Linux and the only one I could get to do it loved it and bought an Acer net book with Linpus Linux Light on it. He loves using it and even though it is not Ubuntu, it is good for him so I say more power to him. I can't convert my other friend who is an IPhone fan boy but hey you don't win them all. 

I realized that support for Linux is the same as support for Windows only better. You see if you need support in Windows you call Microsoft and you wait so long you get frustrated and google it yourself on the net. I no longer worry about Viruses, malware, BSOD, .dll erors, or whatever. I can't play PC games anymore but thats ok I don't mind the trade off. It is less head aches for me.   

I would pay for Ubuntu. I use 64-bit 9.04 now with no problems (Yes, my NIVIEA graphics card is working great google search a prog. called 'compiz-check'). I can't thank everyone enough on this board enough for all the help I was given. I got help other places too but I came here first and you all guided me when I was extremely frustrated. I thank you all for having that patience.  

I think the biggest competitor/ threat to Linux is not Windows but Apple. A Mac is stable, virus free (realativly speaking of course), effecient, and extremly appleaing. Which is great.

However a Mac is also extremley proprietary and expensive with a fan base that grows larger everyday. IMHO is is the easiest of all OS's to use and so it has mass appeal to the masses who don't care that their DVD's don't work out of the box because DSM is evil. They just want that damn thing to work.

Thank you all for reading. You have a Linux user for life.


/end_rant
