### Post by codeking on 2008-10-27
Our family's main computer is running Vista (which I hate)  During use today it crashed and wouldn't boot up again.  Has some fixing tool, but guess what, it needs the Vista install CD.  (Which we don't have)

No big deal, I figured I'd run Ubuntu LiveCD on it to give my family basic internet until we get this whole thing worked out.  Unfortunately, before I managed to start downloading Ubuntu, the internet went out.  It usually does this, and is easy to fix - if you have a computer to hook it up with.  And guess what, our desktop next to the modem and router won't boot up.  And our aging laptop's Ethernet port decided not to work.

Eventually we managed to borrow a friends laptop and fix the internet and now I'm downloading Ubuntu.

Moral of the story:  Always keep an Ubuntu CD handy :lolflag:

On another note, I have Ubuntu installed on my current computer and am very happy with it.  Except that it freezes often (A problem that many people suffer from) which I'm hoping is fixed in Ibex
