### Post by bryceowen on 2009-04-13
My first real exposure to Linux was at work about 10 months ago. They setup a new telemarketing system using an Asterisk server. All of the workstations are thin clients running CentOS5. As I poked around, I started to really enjoy it. I even got the idea to install CentOS on my box at home... That didn't work so well. I could never get Gnome to come up (my monitor kept throwing an 'out of range' error). The CentOS forums were helpful, but I was never able to resolve the problem, so I lost interest in linux.

Working on the web-based interface of Vicidial also led me to my first exposure with PHP. Until that point, I'd only ever used ASP for scripting. I fell in love. (I've been working on converting all my old web pages to PHP so I can get rid of that damned IIS...

Shortly after, in December, I bought an EEE 900 with Xandros. I've gotta say, I liked it, but it was just trying WAY to hard to look like XP. That, combined with the fact I wanted to try installing Half-Life 2 on it, I slicked it and put an nlite'd XP on it. It's the fastest booting XP machine I've ever owned.

Move forward to a few weeks ago. I've made more progress on my web site conversion and have begun to consider setting up a LAMP to free the XP machine the intranet is currently being served from. I spent a lot of time on Wikipedia, trying to find the pro's and con's of each distribution and decide which would be best for what I have available.

I chose ubuntu because it was billed as the 'most popular' distribution. Fair enough. I burned the iso, installed it on a flash drive and booted it on my EEE to check it out. Gnome was familiar and comfortable and everything loaded up smoothly. No driver issues, no annoying 'tour' balloons, and it installed so damn fast! Being so accustomed to spending an hour or better installing XP, I was confused.

After playing with it for a while, I saw some screen shots of kubuntu and really liked the look of KDE, so I downloaded that iso to give it a whirl. (I have since learned I need only 'sudo apt-get install kubuntu-desktop'.) Kubuntu was nice, but the fact I couldn't set a static IP was really irritating (I've found how to edit the etc/network/interfaces to manually configure it since then). So I figured, I've tried those two, let's try xubuntu!

Needless to say, I've hopped through quite a few different ubuntu incarnations, but classic ubuntu is my favorite. I want to install the x64 edition on my main box so I can try some video editing under the full potential of my hardware. (I tried installing it on an external USB hard drive, but it doesn't seem right. I think my USB adapter might be dying.)

tl;dr edition:
I dig ubuntu! Want to use it for video editing. Not giving up XP completely because I like L4D too much.
