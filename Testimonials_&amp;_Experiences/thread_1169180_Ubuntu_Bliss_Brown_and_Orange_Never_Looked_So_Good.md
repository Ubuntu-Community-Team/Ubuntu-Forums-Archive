### Post by chessnerd on 2009-05-24
I recently decided that I would do a bit of hardware upgrading and combine two old computers into one. So, I ended up giving one of the computers a second hard drive (raising my storage space from 20 GB to 60 GB), a DVD drive, and a second RAM stick (from 256 MB to 384 MB). The RAM was the most important modification and I noticed a HUGE improvement in speed on Xubuntu. So much so, that I felt confident about loading Ubuntu onto the new hard-drive. After wiping off Vector, the distro I had loaded on the drive earlier, I installed Ubuntu using the Alt-CD (for some reason I couldn't get the installer to work on the Live CD) and loaded up true Ubuntu for the first time (as opposed to Xubuntu).

I have to say, I'm very impressed.

Every process that I did under Xubuntu was simplified and a lot of the problems that I had before (from hardware problems to printer problems to network problems) all seemed to go away. It was true bliss...

I've only been using Ubuntu for 4 days now, but already I'm very much satisfied with it and I haven't booted into Xubuntu or Windows 2000 since I loaded it up. All I've done with my old drive was drag info off of it using Nautilus (which is like Thunar, but 10 times better) and that was that.

I have had a few problems with the system freezing, but I had similar problems under Xubuntu and I'm now able to easily use Ctrl+Alt+Backspace to do a force quit if I can't get things to work. This feature was hit and miss under Xubuntu, but has worked perfectly under its brown and orange older brother.

The GNOME desktop is a great improvement over XFCE and is able to run perfectly under the higher RAM. I have to say, I expected some problems because 384 is given as the minimum system requirement for Ubuntu on the Wikipedia page, but I have yet to break 320 on RAM usage.

I like the fact that it is much easier to set up task launchers on the panel than in Xubuntu (where you had to use the command rather than getting a nice list of options) and I enjoy the fact that everything has more GUI customization options. In a typical day of computing with Xubuntu I would usually bring up a terminal at least twice, with Ubuntu, I've only had to bring it up once in the four days I've been using it. It's a whole new kind of Linux experience for me!

Networking is much easier as well. Ubuntu instantly recognized all of the Windows computers on the network and, after installing Samba, the Windows computers recognized it too. Networking under Xubuntu was a mess that I had yet to figure out, but no longer have to. ;)

Xubuntu is nicer and easier to use than most distros, but Ubuntu puts it to shame. It truly is Linux for human beings. I'm still not sure that anyone off the street could pick it up and use it as easily as Windows or OS X, but most probably could.

My final verdict is this: Ubuntu is Linux bliss and a brown and orange desktop never looked so good. :)
