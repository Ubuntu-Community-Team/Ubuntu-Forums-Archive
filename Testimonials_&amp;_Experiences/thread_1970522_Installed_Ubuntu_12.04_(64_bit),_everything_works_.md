### Post by arunb on 2012-05-01
Hi,

I just installed 12.04 (was previously using 10.04) on my Acer Aspire 5745, the installation went perfect, there was absolutely no problem, except for some minor issues.

I used a pen drive instead of a CD to install the OS, the first time I selected the upgrade option, the installation went OK for some time, but then hanged up towards the end, and I had to 'hard reset' the machine. I then found that ubuntu would not run from the USB drive, and I got some video error (insufficient resources). 

So I had to re-install 10.04, formatted the pendrive and then made a fresh strartup disk using the downloaded 12.04 image.

This time  I could boot perfectly from the USB drive, and now I selected the 'fresh' install option (ie: format and install 12.04). The install now worked perfectly without any problems.

Other than this small issue I had no other problem..

I suppose its better to make a clean install...

I think unity is really excellent, it takes time getting used to it, but I am liking the side launcher panel and the HUD...

Thank you canonical for making a wonderful Operating System.
