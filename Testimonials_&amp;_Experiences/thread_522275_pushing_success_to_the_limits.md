I expect it to work. [the hardware is all compatible] more to follow when finished. it proceeds slowly but it does proceed.

### Post by K.Mandla on 2007-08-20
It depends on what you're doing with it, really. If you're talking about the default setup, yes, you'll need a significant amount of memory to work with Ubuntu.

But I've run stripped down (but still graphical) versions of Ubuntu with as little as 36Mb, and they're usable. I've booted to desktops on as little as 17Mb, and had plenty of space leftover for memory gluttons like Firefox.

It depends on what you're doing, and how you do it.
