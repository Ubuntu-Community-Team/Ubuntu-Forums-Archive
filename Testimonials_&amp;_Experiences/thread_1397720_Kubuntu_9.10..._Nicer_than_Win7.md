### Post by JayKay3000 on 2010-02-03
I've had ubuntu on my main PC for a while now as the main OS (it feels AGES). So a friend told me about Kubuntu and got around to trying it tonight. First in virtual box which it was horrid but then as i'll repeat, I learned you can install it as a session.

I have a laptop running windows 7 and decided to go ahead and install Ubuntu inside Windows 7 using the free disk I got through the post (THANKS!) and then Kubuntu as another session on the pc because i'd just learnt that you could do this. Awesome!

So how am I finding Kubuntu vs Ubuntu or 7? I've only used it an evening and the desktop environment crashed on my pc. uh, oh. But it was cool and no probs on the laptop. Windows 7 one has crashed on me also so blah.

I'm blatantly comparing KDE to 7 and as a like for like KDE does look fantastic. But personally the out of the box bombardment of blue was a bit overkill and ugg. But now i've got my own background my eyes are ok and i've been able to take it all in.

I have used an earlier incarnation of this back in... ...2006 maybe on a uni machine and I can recall even then thinking it seemed cool because of its windowsness which it has been designed to look like. I do like that the rhs pane is customisable so you can get rid of a lot of the 'crap' if your not into that which i'm not really a fan of. 

Overall I really like it, it's a bit different to use from the gnome interface, things are obviously not in the same place so initially it's a bit tricky to navigate. But it works brilliantly and yes I did play with the ball for 10 mins... oh i'm so easy to amuse. 

The KDE interface looks fantastic on my 15.4 laptop too. I'd love to go ahead and say i'd use it as my main OS but i'm in the middle of some Java work which is using databases which all run with ms, which I could change, but I want to concentrate on passing. Perhaps cross platform support if I have time.

So i'm sold on KDE, looks fantastic, runs brilliant, has some nice differences from gnome. I have to admit i've not found myself using the standard web browser, I switched back to firefox as soon as I could and have been using Safari (which is brilliant) on windows 7 (please make it for linux!)

If you want the windows style experience without the Windows cost then this might be what your looking for. I belive in trying everything. No OS is perfect and they all have bugs. But i'm really enjoying my time with Linux which has been mostly spent with Ubuntu.

I'd love to be able to switch 100% as our uni runs a lab of Linux machines on 22inch screen (the windows users only get 17inch :P), they allow me to submit my work in native openoffice formats and runs a linux based server.

Technically I don't need windows? What do I really need it for... msn voice comms. 

&Format factory (please make that for linux too)

Hope you enjoyed reading as much as I enjoyed raving.

You may know this already, but if you like KDE, there are some other distros that do a better job at implementing it.  Fedora and Mandriva are both at the top of that list, if you ask me.

I like KDE myself and have a partition in my hard drive dedicated to Mandriva 2010.  

Having said so, there are many things I don't like about KDE.  There are some very simple things that are missing from it and it is frustrating.

For example, I am yet to find an easy way of creating custom keyboard shortcuts.  What takes 10 seconds in Gnome (i.e. Creating a custom keyboard combination to start Chromium) is painfully difficult under KDE.  

Another element that seems to be primitive in comparison to Gnome is the panel launcher.  Adding an application launcher to the panel is not that straight forward, and when you make it happen, the options available (customize icon, application name, etc) are very limited.

KDE is supposed to be about the looks, but I think you can make Gnome look much better if you know what you are doing.  KDE font rendering is plain fuzzy, there are very few Kwin themes that are worth it and even less sets of controls available.  I ended up using the Clearlooks controls theme and some Emerald window themes!

In addition, applying an icon theme is very inconsistent.  It may change some menu icons, but then you find that many applications don't get those icons updated (folder icons displayed by dolphin, for example).  On the contrary, This feature works very well under Gnome.

I also like Gnome applications best.  Most of my favorite Linux applications are developed under GTK, so they look terrible in KDE.  Back when I started using Linux I loved Amarok, K3b, Kopete, and some other KDE applications, so I used to install them under Ubuntu.  I don't anymore, I just found their Gnome counterparts are as good or better.  Brasero is a terrific application, and so is pidgin.  The Gnome system monitor is king, and so is the network manager (as opposed to Knetwork manager).

Anyways, go for KDE if that's what you like, but I do recommend to try a different, more KDE friendly distro.

Good luck!
