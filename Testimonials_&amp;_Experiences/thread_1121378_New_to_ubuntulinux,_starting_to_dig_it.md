3. windows is really slow and requires a lot of maintenance.

4. the last time I used windows it asked me for a product key!!!

5. its boring.

6. it doesnt have a built in spell-checker.

7. when you drag a windows in ubuntu it displays a "grab" hand cursor, but in windows it doesnt.

8. the start menu in windows get too cluttered.

9. if you want compositioning you have to get vista.

10. you have to press "start" to shut down the computer.

11. when you drag an item in ubuntu (text,images,ect.) when you let go it displays an animation of the icon or whatever flying back to its position, but in windows it doesnt do that.

12. in ubuntu, when you drag an item into a folder the folder icon will appear to open, but windows doesnt do that.

13. ubuntu has millions of themes to choose from, windows doesnt.

14. ubuntu has multiple workspaces, windows doesnt.

15. windows takes ages to set up.

16. windows keeps interrupting my work to update itself!!

17. ubuntu doesnt require the user to reboot everytime they install something.
