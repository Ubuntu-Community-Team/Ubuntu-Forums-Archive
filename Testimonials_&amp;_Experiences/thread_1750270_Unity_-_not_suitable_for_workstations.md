### Post by numbercrunch on 2011-05-05
Hi,
I just upgraded 64bit Ubuntu 10.04 => 11.04 on my 17" mackbook Pro.


Hmm, It suffered the "graphics driver activated but not in use" problem, which meant I had no GUI and just hung to start with. Rebooting in low graphics mode worked. Strangely fixing the problem required uninstalling & reinstalling the current Nvidia driver with Synaptic, then removing and reactivating it with the hardware manager, and finally rebooting. So what appeared to have been installed in the upgrade, somehow hadn't been, and when installed with user gui tools, did work and the GUI came to life at the proper screen resolution.


But then .... hmmmm.... Unity :((

Simply put Unity might be right for a PDA or a small tablet.
It is the wrong concept of a GUI for a workstation.

Okay, so it will have its fans. So make it an option on the default login, but don't remove classic Gnome from the default login.

I need a GUI that makes use of nested drop down lists, and fine print, with peasant aesthetics to help me find things on a large, complex system. This is not a smart phone, it is a workstation with a 1920x1080 monitor, and a lot of different software installed.

Frankly Unity reminds me of the amount of information provided by the MS Windows 3.x GUI, and what an improvement Windows 95 was. ..... It is a abusrd for a Linux machine with X-Wndows to be back at that level.

Fortunately reverting to classic Gnome is straight forward, <system settings> <login screen> <select default session> <Ubuntu classic> .

So please lets have Gnome back on the default login, even if Unity is also there as an option.

### Post by akand074 on 2011-05-05
> **numbercrunch said:**
> Okay, so it will have its fans. So make it an option on the default login, but don't remove classic Gnome from the default login..

They didn't really have a choice. Gnome team dropped support for Gnome 2 (gnome classic) and it's only logical to release Unity now so that they can get user experiences early on to be able to improve it and mature it for when gnome 2 is no longer really an option. Give it a chance, it's much better than gnome classic aside from obvious flaws but it's brand new. I don't really understand most people's problem with it (when you learn the keyboard shortcuts). You can do everything gnome classic does (quicker) the only big change is the global menus and the lack of panel for your windows (you have the side panel). Next release won't even have gnome 2, I believe gnome 3 will be integrated with it (not sure about the option for gnome-shell). Get used to it now I'm sure you can make it work and actually like it better, but if you insist against it, you might want to start thinking about a different desktop environment (or revert to 10.04 for the last two years of it's life).
