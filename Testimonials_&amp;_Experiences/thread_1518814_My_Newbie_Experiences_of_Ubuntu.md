### Post by m623d on 2010-06-27
Here's a little background.

Had Windows 7 installed and did everything I needed on my HP Laptop.
But I dont want to pay for an OS when there is a Brilliant free alternative.
So I started with Ubuntu 9 for my first dip of my toes.
I was blown away how good it was, but there were some things I could only do on Windows, so I had a dual boot.
Problem was I hardly used Ubuntu eventhough I wanted too, cause it was too much hassle rebooting everytime I needed a windows programme.
When 10.04 came out I did a fresh install, still with a dual boot win 7, but this time I have set myself a goal of switching over completely.
I had a few problems initially with my wifi and getting flash to work, but eventually got the wifi sorted, flash works, but occasionally crashes.
All the basic stuff works, so I now have a clean slate to work on
So here are my requirements to avoid Windows:

**Email:** currently use yahoo webmail, I would prefer a client, but I cannot get IMAP working, IMAP works fine on my Iphone. Tried alsorts of different settings, but no joy.
**Google calendar:** if I can sort the email, the Calendar plugin will be great.
**Itunes:**or similar, currently using Rhythmbox, seems ok at the minute. **{SORTED}**
**Windows Home Server:**mmm, this is a conundrum, I like the remote login and GUI. I have sussed the remote desktop and can even connect to the WHS connector via Ubuntu, so I can use the server just as I did with win7, however, one util I do need is **Synctoy:** but i have yet to find a good alternative.
**Command & Conquer:** I love this game, but I cannot get it to run under wine, this would be a massive plus
**Printer/Scanner:** I have a Epson CX6600, and it operates great with Xsane. **{SORTED}**
**Office:** All my docs etc are saved in office2000 format, when I open them in openoffice my excel files dont print out correctly, the formating is different, not a major drawback when I finally switch permanently, but whilst in a transition it is a bit of a problem.
**Utorrent:** my server downloads my torrents with utorrent. When on win7 I have a desktop widget that displays downloads constantly. Not a major 'want' but it is a nice feature.
**WHS:** Again on win7 when I bootup my windows server shared files show up as an icon on the desktop. Need the same on Ubuntu, I think its just a case of making Ubuntu mount the folder/s automatically. Anyone with an easy guide?
**Google Earth:** Got it installed and working, but at times it is a bit jerky
**Firestarter:** Got this installed but it doesn't autostart, I have to run it
**Paintshop Pro:** Any good alternative?
**Autoroute:**I need this, or a good alternative, once again it wont run under wine

Thats it for now.
I'll add stuff as I go, cause I know I'll come up with more needs as I go.

One other thing, I would like to move over to a Linux server eventually, but for now my WHS is a must.
I do like a GUI and I believe linux server hasnt got this and also needs alot of command line stuff to make it work.

So there you have it, please feel free to advise me, or correct me when I go wrong!

Thanks
