### Post by davidsiegel on 2007-06-18
I thought I'd share my Ubuntu switcher story, which I believe is of a less common sort because I switched from Mac OS X to Ubuntu:

I bought a crappy old Gateway in 1999 that didn't run Windows 98 so well, so a friend of mine turned me onto the FreeBSD project and I was quick to replace Windows with FreeBSD. When Mac OS X (10.0 beta!) came out with Darwin (FreeBSD) under the hood, I switched to Mac and I have been a happy Mac user ever since. After playing with Ubuntu on and off since the Ubuntu project began, I started dual-booting it on my Mac this year; first I ran it on my G4 PowerBook, and later on my shiny MacBook Pro. After Steve Jobs revealed the new features of Leopard at WWDC, I realized that Apple users don't have the kind of freedom you need to truly "Think Different," but instead are encouraged to "Think Like Steve." As of two weeks ago, I traded my MacBook Pro in for an all-Intel ThinkPad and I've never enjoyed using my computer more.

I'm a Computer Science undergrad, and I've worked for Google and other high-tech companies doing software engineering so I consider myself a fairly advanced user and I'm very comfortable using Ubuntu. On the other hand, I've helped around ten friends who are novice users install Ubuntu on their PCs and although they learn to love Ubuntu, I'm certain that without my help they would have gone running back to Windows in confusion. I think this is less of a technical hurdle and more of a conceptual hurdle that this project needs to overcome; I have to carefully explain to my friends that although they can't use iTunes, there are many other nice music applications, and no, you can't install Microsoft Office but you won't need it. Maybe we can create a "Reasons to switch" list like Apple's - [_reasons to get a Mac_]("http://www.apple.com/getamac/")?
