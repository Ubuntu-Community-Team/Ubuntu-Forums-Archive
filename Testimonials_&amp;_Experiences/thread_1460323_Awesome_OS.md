### Post by thomas144 on 2010-04-22
Ubuntu 10.04 LTS (Lucid Lynx) comes out on Thursday, April 29, 2010. That should answer your question.
If you're new to Ubuntu, what's an LTS release? LTS means term support. An LTS release is aimed at stability and reliability, and it is supported for 3 years (5 years on the server edition). This means, if you wish, you can upgrade to Ubuntu 10.04 and then keep it until the next LTS release comes out in April 2012. This is because you can upgrade from one release to the next one, or from one LTS release to the next one.
