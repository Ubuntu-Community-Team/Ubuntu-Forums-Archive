The Fedora XFCE spin shouldn't be any different than Xubuntu in terms of the desktop. You'll need an application called Easylife to help you out with installing codecs and stuff. You need the infinality patches to smooth out the fonts. You might not like SELinux, but it's not really important for desktop use, so you can easily disable it. Other than that, the graphical package management is not as good as Synaptic or USC, but you can install Synaptic if you want. The CLI package manager is as good as, if not better than apt, but you can install apt-rpm if you want to. 

Carve out a 10 GB partition and set it up. Unlike the Ubuntu graphical installer, the Fedora one doesn't force you to install GRUB, so i case you don't like it, you can easily just wipe the Fedora install clean and update GRUB, and your machine won't even know it was there.
