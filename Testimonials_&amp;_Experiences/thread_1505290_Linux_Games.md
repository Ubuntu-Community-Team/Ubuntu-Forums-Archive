### Post by e4uforums on 2010-06-10
With as much as I talk about this you'd think I get a commission (I don't by the way   :p)

I use PlayOnLinux to manage WINE and it has worked very well for me.  It handles all the installations for you and configures WINE automatically.  If you do need to manually assign WINE versions to programs, it can be accomplished in but a couple of mouse clicks.  I've gotten many games and applications installed using PlayOnLinux.  It makes it all very simple and user friendly.
