Congrats.. It aint that hard really once you get used to it :)
Your updates will be handled for you with the update manager, so little worry there :)

If you want some serious linux security & are willing to learn a little ( maybe alot) more ,find an old redundant machine, any old crap will do & set up a Smoothwall firewall/gateway machine, Out of the box the security is excellent and by using it and customising it the way you like you'll learn alot about the way Linux actually works.
