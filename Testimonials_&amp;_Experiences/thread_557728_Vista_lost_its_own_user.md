### Post by por100pre1 on 2007-09-23
For the benefit of any readers who want to keep Vista, there's a way to install Ubuntu with Vista easily.

1. Shrink Vista's partition with its own Disk Manager.

2. When installing Ubuntu, install GRUB in Ubuntu's own root partition.

3. When finished, login into Vista, and add Ubuntu to Vista's MBR with EasyBCD.
