### Post by SebastianM on 2006-07-03
I've been using linux for near 10 years.  (since Redhat distro Colgate)  I've played around with damn near every distro and flavor of *nix there is.  In the past 5 years it's been primarly Redhat, FreeBSD, OpenBSD, Gentoo and Fedora.  The past year I was stationed abroad so I wasn't on the computer much.  I got back and started downloading Fedora 5.  My DSL is very slow in my area so looking at Distrowatch I saw this Ubuntu thing at the top of the list (it's how I got into Gentoo origionally...later dropped it though..long story).  Anywho...the download was smaller than Fedora so I opted to try it before Fedora (I actually got half of it done at a local coffee shop...I love torrents).  I didn't do any reading yet on the forums and anywhere online simply because I like surprises.  I didn't want to know of any problems, etc.

Right off the bat I was pretty impressed.  All my hardware was detected on my new Inspiron E1505.  Primary concern was my wireless card and it worked right after first boot.  So smiles there.  I used the alternate CD with a text mode install. Didn't get to choose individual packages...I was a little upset about that.  Maybe I just missed the option to choose individual packages during the install...(could someone clarify on that).  I don't like spending too much time reading all the install screens, I generally just try to whiz by.  (mc not being installed by default I didn't like either)

I update everything and start pressing on.  Install fluxbox and get it all setup.  xmms has locked up on me a few times.  Gotta toy around with that and see what's going on.  I didn't like the whole way the runlevels were setup.  No problem though...deleted the symlink to gdm in rc3 and set default as 3 (I don't like booting into a GUI).  Primary focus being to use sudo I don't really like.  Just habit though to type "su -" as apposed to "sudo" I guess.  Quick question on that...is the root password something random with every copy of Ubuntu or is it the same?  I just did a "sudo passwd root"...but is there a normal "default" root password or what?  Just something I'm curious about and I really don't feel like looking up.

The fact that Ubuntu recognized my winXP partition and booted it flawlessly was great.  The last version of linux I ran on a dualboot desktop/laptop was Fedora 2 and it fragged previous windows installs and required some work to...work.  I like it when things just work like they have been with Ubuntu.  Once my Fedora 5 install is complete I will probably install it over this to see which I like better.  New computer and not using Linux for a year...need something to compare it to.  But I must say this is a pretty nice distro.  It would have been nice if things "just worked" 10 years ago with Redhat 4.  Novel Netware 3.12 gave me less problems. (kinda apples to oranges as Netware wasn't an entire OS...but it's what I worked on at the time.

Cheers
