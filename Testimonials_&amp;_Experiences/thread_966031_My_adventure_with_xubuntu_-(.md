### Post by jivabill on 2008-11-01
Well, I installed Xubuntu on the second HD of my older (2001 vintage) win98se machine.  Has two 13gb HD and 384mb RAM AMD 450 CPU.

Used the alternate install, went OK.  BUT found several problems some of which are not fixable, some are: don't see my sound card (ES1688) don't see my Canon 320b scanner (I can use that on the windows side so not a big problem), no way to access the network, no icon, no entry in the System menu, nada.  Samba IS running, just can't get at it. Then there is my Canon IP3000 printer.  Works fine under Ubuntu on my other machines, (it is configured thru Samba).  But on Xubuntu it runs at 1/3 normal speed. 

And the very biggest problem is that the system font is tiny and I can't read any of the menus or application windows without reading glasses.  

So There may be a fix for the network thing.  I am sol on the sound card. Maybe if I get Samba right my printer will work right, I am not holding my breath.   But I tried about 5 different command line capers to fix the system font, none work as they don't apply to 8.04, they are for earlier versions.

So I installed KDE Desktop hoping that would fix the font problem.  It did and had all sorts of neato settings.  I went about setting it up, and soon went to Display and set the resolution.  I saved my new setting 1024 X 768 and did some other minor settings in another area when all at once the system changed to resolution 640 x 480 (which is not usable). I tried to move the resolution slider but it was full right at "higher" and that was 640 x 480 and there was no way to change it bigger, only further smaller.  

So then I tried to log out and change the session back to XFCE but the system stayed with Kubuntu and would not let me change to anything else except a terminal window.  Made many attempts to change the session.  All failed.

So I don't know enough about command line magic to use the terminal window to fix the problem. So as we speak I am re-installing Xubuntu from scratch (same remedy as when Windows goes haywire). Then about two hours of downloading and installing updates, again.

My two cents then is DO NOT try to install KDE Desktop on your Xubuntu.  And second, don't count on ever being able to fix the system font size problem either.  The problem with the network not showing up is fixable but it is a lot of work I gather.

Seems like, for a distribution that is supposed to be especially good with older machines and is supposed to be simple and straightforward to install Xubuntu is lacking.  And the tiny system font is unforgivable.
Bill
