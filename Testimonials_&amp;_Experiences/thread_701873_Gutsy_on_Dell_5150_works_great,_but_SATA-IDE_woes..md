### Post by hughdoar on 2008-02-19
Having installed the configuration manager for compiz fusion, I can see what all the fuss is about. As a fun system to use, this beats anything else I've seen.

I've also let the kids loose on it, and suffered no damage so far, with the added advantage that any exes they accidentally download don't work! The flash games all seem happy enough. Mozilla is fantastic.

Network configuration picked up my usb wireless fine, but wouldn't work until I rebooted, which I felt was a shame. 

I've had a play with Ardour and my MAudio card works well. The Jack connection system with Patchage is very impressive.

There are some downsides - for a start, the first time I installed, the system refused to boot. I had an old IDE drive connected as well, and I think Ubuntu decided to install the boot loader onto that, instead of the primary drive. This drive isn't recognised by the BIOS, so everything broke. Unplugging the drive and re-installing, then plugging it in, fixed everything. Should have read the forums first, I guess! 

Overall, I'm now using this at home more than XP. I don't use Vista at all.

Keep up the good work!
