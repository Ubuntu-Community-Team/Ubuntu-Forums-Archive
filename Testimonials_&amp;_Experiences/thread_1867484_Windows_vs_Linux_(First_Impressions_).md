In our tests (2+ years ago)  Win XP ran vanilla desktop apps fractionally faster when virtualised inside Linux using KVM than on bare metal.

Most of these machines run Win2K with insufficient RAM to add anti-malware and still load a web page.  This version of Linux that I am using is a derivative of Ubuntu called 'Bodhi Linux'.
