### Post by Bandicoot on 2007-12-05
With a grin on my face I've been reading some of the rants and bashes in this section.. It has allways been this way, back in the early 80"s when I owned a C64 (Commodore 64) the rants and bash were between de commies (C64 owners) and de speccies (ZX spectrum owners) and the battle continue some years later between Atari ST owners and Amiga owners each with there own share of true followers . This battle is probably ongoing and will never end.

i never felt the need to rant or bash, just stick to what you're happy to use  or endeavour a trip to the 'other"side once in a while and see if you like it. If you like fine if you don't like it also fine just return to what you like the most, no need for ranting or bashing.

I enjoy using Ubuntu and it gives me much pleasure to try out new things and make the system behave the way I want. I also enjoy using Windows for some specific tasks. So I just use the easiest solution there is: dual boot and even virtualization so I can can enjoy the best of both worlds at the same time.

I don't see the need to convert to one "side" or the other. Especially users new to Ubuntu/Linux should be happy with the dual boot option , It gives you time to ease into the environment (no rush needed, just take it slowly and thorough). Dont try to do everything as fast as possible and take time to eaxmine and enjoy the merits of the new evironment .. Don't try to make the new evironment and except copy of  the old environment or expect it to be an exact copy if that was the case you wouldn't need another evironment.

I think you catch my drift by now .. This ofccourse totally my own opinion and not a "that's the way it should be "opinion.
