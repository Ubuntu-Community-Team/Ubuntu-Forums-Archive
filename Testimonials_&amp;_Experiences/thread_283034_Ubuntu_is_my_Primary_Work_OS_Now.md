### Post by MrDigital1 on 2006-10-23
I've actually been runing Ubuntu 6.06 at work as a primary OS now for a couple weeks.  Here is my story.

Some specs too.
ASUS Intel Based Mobo (don't remember exact model)
4GB DDR RAM
2x 200GB Western Digital HDDs
nVidia GeForce 6200
Using built in Gigabit LAN and Audio.
etc.

I am a (mostly) happy Windows XP user, in fact I help support about 300 Windows workstations here at work.  My computer is a powerhorse that I'm always pushing limits etc.  I am a VMWare VMTN subscriber so I run VMWare Workstation and beat my machine to death with multiple virtual machines running at the same time, everything from workstation OSes to Server OSes etc.

I started getting BSOD screens which absolutely piss me off to no end all the time.  I thought maybe I had a Hardware issue since I keep a pretty clean and tweaked XP system and also verified I had no malware or bad stuff (also a long time Firefox user and advocator).  Problem would remedy itself then crop back up at the most inopportune time.  Plus, my Sony DVDRW drive would utilize almost 100% of my CPU time whenever I would try to burn discs, didn't matter what software I used to burn.

Finally after one of my BSODs I turned the thing off, slapped in two new 200GB Drives so I could go back to Windows if I needed to, pulled out my Ubuntu 6.06 Desktop CD that I had tinkered with (and was impressed with) threw it in the drive and installed Ubuntu.

As any new OS it took some time to get used to, but since I'm also a Mac OS X user (I have two macs at home in addition to Windows XP) it really wasn't that big of a deal, plus the forums here are awesome.  I installed the Linux version of VMWare Workstation because I *am* required to have Windows XP for a proprietary Windows piece of software that I can *not* get to work right under Wine plus I still use Outlook 2003 for email and our department has an Access 2003 database that I have to access all day long.

For those that haven't ever used VMWare Workstation, it has this great little feature called Snapshots that are 5000% better than anything Windows XP can natively do for me.  I can set up a complete working Windows XP workstation that I *know* works and create a snapshot.  That way if I completely hose something (which is what I use Virtuals for anyhow) I can simply go back to that point in time and be back up and running.  It's great! 

So now I have a Ubuntu Edgy virtual that I play with, but it's not ready for *my* main workstation yet.  Last time I tried that my networking started to act wonky and things didn't quite work right.  Problem resolved after going back to 6.06.

So here I am today.  Running Dapper, with a Virtual Windows XP that I know will always work or it's a simple click to go back in time to a working install.  The Ubuntu system has not given me a lick of trouble with reboots on it's own or anything of that type, so now I know that my hardware is OK.

The community beats any other Linux distro I've been involved in hands down.

I now have a couple extra copies of the disc in case I can get somebody to take one home to try it out :)

Thanks for the awesome Distro Ubuntu Team!! :KS
