### Post by EricG1793 on 2012-03-09
Hi all!

My interest in Linux in general sparked when I used a friend's computer with an old version of Ubuntu. It was so much faster than their bogged-down Dell with Windows XP, and I really liked how the interface was set up and how simple it was to do just about anything.

So I researched and came to the conclusion that Ubuntu would be the best for me as I branch out from Windows for the first time. I installed it to the external hard drive with some help from some very helpful people here.

What impresses me most about Ubuntu is how easy it is to install and get going. The setup process was very minimalistic in user input, and upon booting  for the first time, it went into my user account immediately and  peacefully installed a slew of updates. I was also impressed by the fact  that, once it boots, it's done loading; as soon as I see the desktop,  it's ready to go, unlike Windows, which takes ages to "wake up" even  after it boots! To borrow the stereotypical Mac feature, "It just works." All hardware was fine right off the bat for my ThinkPad L412 -- display, wireless card, touch pad (even the scrolling lines), sound, webcam, etc. Even plugging in the external drive to the family desktop after Ubuntu was installed to the drive was flawless; Ubuntu instantly accommodated the new system without missing a beat.

Likewise, I'm very happy with how easy it is to map a network drive, install printers, etc. (far easier and more reliable than Windows). Also, there's no or negligible lag when plugging in a flash drive (no installing drivers), reinitializing wireless connections upon resuming from standby, etc.

I like the launcher on the left, but I think it would be better if it would shrink the icons to accommodate extra ones instead of requiring scrolling to get to the trash and whatnot. Perhaps have a changeable minimum icon size so the user can keep important things directly visible and then scroll for less-needed things.

I'm still getting used to the floating taskbar at the top (the way it changes according to whichever program you're viewing), but it's OK. I'd prefer that the close/minimize/maximize buttons be on the right of the screen because of accidentally showing the launcher when trying to close a program, but I can see that if they were on the right, it would be the same issue, and exiting at the left sets you up to launch a new program with the launcher, so whatever. I think an option to switch the location of the taskbar (or whatever the cluster of icons including settings, calendar, wireless, battery, etc. is called), close icon, etc. around would be great.

The font rendering is taking some getting used to, but it's tolerable.

My biggest gripe with Ubuntu is how difficult it is to access many  programs. Of course the most important ones are on the launcher, but I  hate the whole "dash" thing.  Its graphics have a much different feel  than the rest of the OS. Then to have to dig through "apps," then "more  apps" only to access a whole list with every single program, game,  system settings function, etc. all thrown in one disorganized place is  very bothersome. Even adding some extra categories to the dash home such  as utilities, games, etc. would be helpful, then having one "all apps"  area.

I also find the tiny/hidden scroll bars hard to use. It would also be nice to have more increments with screen brightness and volume control.

The only glitches I've experienced are that the copying process seems to be somewhat unreliable (is Control + C not a Copy shortcut?), the launcher would not un-hide on one occasion, some "ugly text" appears when changing power states, and the shut down dialog box when clicking shut down on the login screen is antiquated. Also, when the update manager icon appears in the launcher, clicking it appears to accomplish nothing, so I'll have right-click on it and tell it to install updates, which it once in a while won't do.

I felt like there was a lot of lag when opening programs, launching the home folder/dash, etc. Same with an internal installation alongside Windows on the family desktop. However, I installed 64-bit internally on my laptop and and I'm VERY pleased with the speed gain; everything is much snappier (practically instantaneous), and I have absolutely no issues between 32 and 64. Boot time is noticeably faster, too, but that might be helped by the 7200 rpm hard drive. I think that having 64-bit default for downloading in the new release is a good thing, but of course cautioning users to ensure that they have a 64-bit processor (perhaps keep a list on the download page) is a must. Anyway, it's much faster than Windows, and my Windows runs quite well.

I like that Ubuntu appears to be very minimalistic on the surface, but it actually comes with just about everything I need to get by. Firefox, LibreOffice (which I find has compatibility issues with Office and isn't as pleasant to use), the media player, several games, a program to view PDFs, etc. It only was missing Flash Player, but Firefox self-installed that flawlessly, although I can't seem to get Shockwave to work. I like the integrated chat/email program.

Overall, I'm very happy with Ubuntu, which I describe in general as being faster and easier than Windows with a clean, simple, aesthetically pleasing Mac-like interface within the comfort of my PC hardware (gotta love ThinkPads ;)). Not to mention the fact that it's free!!!! Count me a happy customer.
