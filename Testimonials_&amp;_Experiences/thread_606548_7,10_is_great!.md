### Post by homerhomer on 2007-11-08
Rant Time!

The Good!

I'm liking the latest and greatest of Linux desktop applications. There has been some nice updates to apps like gaim/pidgin evolutions and rhythmbox and a ton of others. The install was easy as I could imagine it could be and it's nice and snappy OS. I'm finely able to use my 64bit laptop (Laptop is  two years old) in 64bit mode. I have everything working! Freaking love the restricted driver installer. It noticed my sl-modem, ATI card and broadcom wireless. Tracker search is freaking nice! I have to admit that it the best desktop search that I have ever used.  And for that matter the best Linux Desktop too. 


The Bad

Me not verifying that my laptop's hardware would work better before buying it. 

I acknowledge that these issues are not really Ubuntu's fault, but hay here they are 

Ati's graphics sucks. If it's not compiz, its hibernation or it will be something else. BTW the free ATI works great but with no GLX (okay not that bad).

 Network manager is buggy. Its funny that it's still in gnome because from the very start it has had problems. Sometimes that just searches and searches and I have to log out and back in to get it to quit.  I'm sure that it has something to due with the reverse engineered broadcom wireless driver (almost as bad as ATI). I'm still amazed that it works ( maybe this is a good )

No video chat pidgin and bluetooth is working but it could be so much more.
No default ubuntu backup options

Overall I really like 7.10.   I don't see myself going back to Windows any time soon.  Simply because crap like licenses, de-fragging, viruses, spyware, inability to upgrade all applications all at once.  And quite frankly Windows seems really trashy.  I understand the Linux has Viruses and spyware but if an application\architecture has an issue I will just stop using it and or remove it. Not so easy on Windows.
