### Post by artir on 2007-02-24
Its very simple, when i heard about ubuntu, like 1 year ago, i downloaded the Dapper Live CD and tried it. I had to configure my 3CRUSB10075 USB Wifi with WEP by hand, but no other problems. A lot of software, i could make documents, presentations, export to PDF and in a very small space!. My old WIndows XP, in the other hand, worked slowly, the Wifi worked only the first years and then.. nothing. When Edgy was released, i updated and Wow, lot of new software and actualizations. I have used sudo a lot, so my system is a bit dirty. I have my important data into another partition and i will erase the ubuntu partition, the windows partition and clean install Feisty. No hand-driver compilation for ZD1211. No Wifi problems. Auto-codec download. All with LiveCD Herd 4. FInal Feisty will be better. Feisty is better(and cheaper, and more secure) than Vista. Beryl is better than Aero and can run in my a bit old laptop. I can customize my icons and i know what i am using beacuse i can see its code and other people also can.
I can imagine in 10 or less years a huge migration to Ubuntu. If MIcrosoft needed like 4 years to make VIsta and Ubuntu is released 6-monthly, what new things will bring us 8.10 Ubuntu?
Since 7 months. I havent touched Windows. Im copying data do Ubuntu to finally destroy that big disk eater called WIndows.
Thank, Ubuntu community.
