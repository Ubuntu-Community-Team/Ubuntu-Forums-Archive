### Post by Sycron on 2008-09-04
This article has version 0.2. Any newer versions would be announced on this thread.

In this little story i'm speaking about my activity in computers and how i was feeling atracted by the OSS. What I say here is real, except any other people named here. I don't want to create wars, and destroy my friendships so everyone called in this text is NOT a real person, just an interpretation.

First I'm grateful to show chicks what computers I used all my life. This may not be some important information right now, but while reading this text help's you understanding what i'm trying to say.

(1999-2003)My first computer was a 486DX2 66Mhz, 20MB RAM , which is still working today, would be nice to build Linux From Scratch on it...
(2003-2008)The Secondth computer was a AMD Athlon 1700+ , 1.4GB RAM DDR, 128MB ATI RADEON 9550 AGP 8X.
(since March 2008)The latest computer is a eeepc 701 which i'm really happy with it. I don't need anymore my desktop computer so my sister really liked that Athlon :) .

Now I think I'm ready to say it. So let's start.
My hackish stuff started when I was 6 years old, on a 486 DX2 with windows 95. I remember using Office97, winrar, some games, paint, and writing some unimportant FoxPro programs.

I was growing up fast, i've had the time to test applications just by using them and that time I was amazed how I've helped people around me. I was too little that days to see that what I was doing was wrong, very wrong.

In 2000 I've heard about linux for the first time. Since then everytime something was going wrong with windows , in my mind I heard a voice telling me "linux".

As I can remember my first contact with linux was in 2003 or 2004. I tried Fedora Core it was on 6CD's. I ended this when trying to install some rpm's and compiling some tarbals. That time I didn't even know what open-source, GPL, cc, GNU, copyright it is.

In november 2001, my kiddie mind heard about Windows XP then I wished to get the newest computer with Windows XP. Nowadays I see that this idea was the foolish thing that I ever wished. Since 2003's no important achievements I've done. In july 2006 I was for the first time online at home...

It was amazing that period of time. I was surfing on the web, speaking with other people and I sayd : "Wow this is cool". I think I would never forget this. :)

In february 2008 my hax0r life stopped. To much pirated software, BUGS, exploits. My freedom was compromised. Windows XP can be really fast, but it's very hard to maintain it so... Tons of software that I've used that time we're completly uncapable to do what I want and I was felling very very angry. So i've googled "linux". Then I found Ubuntu (Gutsy Gibbon that time). I've downloaded it, burned to a CD. wiped everything and voila my hacker life started. It was something really cool for me, something like "Yesterday I was using Windows, and Today I don't know what windows is". That was a totally full switch from Windows to Linux.

I've google what I can't do with ubuntu, or what is wrong, and always found something that fixes it.

This changed me very much.When I start programming I've heard about GPL, cc , etc , I readed them all. I really wanted to code open-source.

The first day with Ubuntu changed my computer perspective totaly. Since then I have NEVER used pirated software again...

I was recording me saving some sites from SQL Injection. One month ago when I was "rewatching" me i sayd. "Wow, this IS NOT ME...it's a M$ fanboy. It has hundreds of pirated software on his desktop that only haxors&newbs uses it."

Three ot two months ago I've started helping people on ubuntuforums.org, i really like this so here I am...
