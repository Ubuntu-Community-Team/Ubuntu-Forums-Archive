### Post by skoalman88 on 2007-07-03
I have been using Ubuntu for about 7 months now I must admit I have never been happier with an OS. This forum provides awesome support for every issue (most of which I caused) encountered and everyone is friendly and seems willing to help. I have become so comfortable with Ubuntu that I really dread having to use a Win. based machine. 
It is really nice that I am finally USING my computer rather than WORKING on it. For the first month or two of using linux I was lost; no longer did I have to spend 99.9% of my time ensuring my virus/spywar/etc. software was up to date, making sure I had all the arbitrary updates needed to not only prevent computer failure but also to allow me to access future updates. I was actually able to just USE my computer. 
Thanks to those that created and continue to develop Ubuntu and to those in the forums that have helped me fix my errors and those who have posted how-to's (AKA lifesavers). 

I was apprehensive about switching to linux, but support from members of this forum has made the transition a very pleasant experience.
