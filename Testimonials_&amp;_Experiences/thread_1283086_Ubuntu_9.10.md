### Post by murderslastcrow on 2009-10-06
Yeah, I think that it's vastly improved for the most part, if not only for the kernel being used. XD

Let's look at the features for a moment- X-based splash screen, ext4 by default (extremely fast), updated GDM for animated login, empathy with video/voice, Firefox 3.5 with HTML5 support, massive amounts of new drivers in the kernel, as well as basic speed improvements, Software Sources doesn't require you to get the key anymore, and you can have more than one package management process running with the Software Center.

And that's just a few, on the DESKTOP side of things. I hope they add checkbox-view to the software center, though. @_@ Not sure if I can stand going into an install page for every individual program.

^_^ Things can only get better, it seems.
