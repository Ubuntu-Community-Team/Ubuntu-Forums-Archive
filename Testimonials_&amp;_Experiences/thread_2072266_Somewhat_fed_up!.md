### Post by TheFu on 2012-10-17
Is there an issue or do you just want to rant?

You don't need to swap the entire OS over a GUI issue.  Ubuntu will happily let you have 5-50 other GUIs as the main interface. You can try them until you find one that you prefer. After you install the alternate GUIs, at the login screen, you can switch to whichever GUI you want to try. After you've picked one that you like, remove the others.

A few to try:
* xfce
* lxde
* KDE
* fvwm
* enlightenment

These each have a different level of GUI, so some don't work with the menu system you are used to ... from Windows.  Linux is not Windows. If that is what you want, then you should buy that update (from what I've seen, most are not calling it an "upgrade") and pretend you are happy again.  There is a reason you tried Linux, a few weeks back with Windows might help you remember.

Often, Linux stability issues are really hardware issues that have been uncovered.

Regardless of what you finally decide, I hope you discover what works best for you. That isn't always Ubuntu or Linux for everyone.  It is 100% your choice.

### Post by TheFu on 2012-10-17
I should point out that I've been using the same fvwm settings since the mid-1990s.  Clearly, my rsh were all converted to ssh and my ftp commands were converted to sftp years ago, but it is basically the same settings.
FVWM is not going anywhere.
FVWM is easily installed under Ubuntu.
FVWM settings are fairly portable, easily understood and easily maintained.

If you don't like the GUI changes, do something about it. 

However, you don't need to leave Ubuntu.  There is much to like about this OS, package management, more current packages than the upstream, and more resources than other "down stream" distros.  I like Ubuntu, even if I don't "like" the current default GUI.  It is just a GUI, hardly the entire OS.  Lots of people DO like the new GUI. It works for their needs. I think that is fantastic.

[B]There is no need to switch distros between Xubuntu, Lubuntu, Kubuntu or Ubuntu either, unless you really want too.  Just swapping in the other DE, Desktop Environment, is enough for most people.  That's 3-5 minutes of effort, not a complete OS reinstallation.
[/B]

### Post by ratcheer on 2012-10-18
@chris.olive, I understand what you are going through. Personally, I have landed on Siduction Linux with LXDE. This is basically vanilla Debian Sid with some enhancements.

But one major point is, if you need good support for your iPod, you are not going to find it on any Linux. You are forced to rely on either Mac or Windows with iTunes.

Some Linux software supports some iPod needs, but nothing really integrates with it the way it is needed. Of course, the reason is that Apple keeps a tight grip on the iPod, its software, the iTunes store, and the song encoding format. Apple actively prevents open source from integrating with their devices.

So, if you want to use Linux, I recommend a dual boot setup with Windows, even if the only thing you use Windows for is iTunes.
