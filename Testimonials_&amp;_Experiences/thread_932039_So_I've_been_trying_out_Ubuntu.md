### Post by oldsoundguy on 2008-09-28
Just a quick comparison:
You need no Anti-Virus
You need no Spyware protection
You need no "garbage remover" such as CCleaner
You need no registry repair tools
(only thing you should have is a firewall if you go on line direct behind a router, not needed.)
You will never have to worry about clicking on a link or a picture or ANYTHING in an eMail and installing some garbage in your computer.

You will not have to spend 2 - 4 hours each week cleaning, updating and servicing the computer.(De-frag and Chkdisk not required)

You will not see the computer crash (unless there is a hardware issue.) 

Using the repositories you can put all of the programs you want to use ON the system for FREE.

Yes, SOME games are limited, but using WINE you can play some of them.

So you now have a machine running Linux that you can work on.

Welcome!
It will never start slowing down as it gets older and gets the file system gets trashed.

### Post by lian1238 on 2008-09-28
Have you tried compiz?

Oh, forgot to mention. I love the screenshot plugin of compiz. Drag a box while holding down <Super> (a.k.a. Windows) key and a screenshot of that portion will be on your desktop. Pressing PrtScn will bring up the screenshot dialog. In windows, I used to do..
1. Hit PrtScn
2. Open Paint and paste
3. Save (low quality too)

I also love the nautilus scripts, like converting pictures format, resizing, etc.

Another thing is, showing/hiding hidden items is just pressing Ctrl+H. No need to go to Tools.. Folder Options.. uncheck Hide hidden.. Click Ok.. You get what I mean. It's quick! But you might not know how to hide files and folders in the first place. I didn't either. There are two approaches.

1. Rename the file to '.filename' (or 'filename~')
2. Without renaming, create a file '.hidden' in the folder with the files to hide, then put the filenames of the files to hide in that file
[I]note: the .hidden file will be hidden, so hit Ctrl+H to unhide first

Edit:
[/I]Did I mention the great open source finance management software, GnuCash?? It's great! There's a windows version too, but I didn't know it until I used Ubuntu.

Oh, and there are many docks out there, AWN, cairo-dock, Kiba-dock,.. they look and function great!
I love AWN. It's simple and easy to use. It works like a launcher/window selector (like the Mac??). Middle click an already-open launcher to launch a new one.

Sorry, but my list keeps growing. I set keyboard shortcuts to minimize windows and toggle maximized. So, to maximize, there's no need to aim for the maximize button on the title bar, double-click the title bar, or press Alt+Space then x - just one button! There's also a window shading feature, which minimizes the window to the title bar.

And there's the "Shade" plugin for compiz, which will lay out all the open windows. I config my compiz so that my right-bottom edge triggers the windows in the same workspace, left-bottom edge triggers all windows on all workspaces to show. And Middle-click to close that window. Enough playing "click the small X's at the corner of the window" game.
