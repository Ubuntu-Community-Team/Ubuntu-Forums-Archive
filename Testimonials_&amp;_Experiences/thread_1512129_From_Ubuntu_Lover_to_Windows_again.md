### Post by pmla on 2010-06-17
**This is just my story, just another story about Windows & Ubuntu**

**The Story:**
For the past 3 years I turned Ubuntu big time. Back in 7.10 all my computers / servers at home changed from windows xp / vista to Ubuntu 7.10. Desktop and server versions 32&64b.
Except for 1 that I left with win XP Pro, because I you guys know some software's only run in win environments.

**3 Fanatic Years:**
And I turned bad, raging against everything that was not FOSS, specially windows. I saw the linux community grow exponentially... But I think this was not only related to how good linux as getting to be (specially the desktop versions). Now I have realized that this growth was basically based on Windows Vista flop.
XP was dying, the sucessor was a big mistake and it was not FREE = more linux (debian ubuntu users).

**@Work:**
I took the changes even to my work. Changed my laptop to Ubuntu, and a few servers there too.
1 Backup Server
1 Email Server
1 Apache Server
Many other small servers and projects like a punch system
What a fight there... a company with 100 windows users and 1 Ubuntu crazy, spitting at everything that was windows, but considered alien. What a daily fight. Won some lost many.

**3 Years later and Conclusions:**
Well **Windows 7 and Server 2008** came out. I was very reluctant and kept fighting it. Installed the Beta version in my Virtual Box and was not very convinced... Kept on thinking that this was just another Vista with a lot of marketing behind. Installed a few win 2008 @my company, good stuff, easy to work with, nothing bad to say about it except the price.
**The war was lost in my company**
- The email server running Citadel was a big flop... crashing all the time, missing a lot of calendar functionalities for production, really bad web interface. Combined with Thunderbird client, the only crap that could run in win/linux environments and light years away from anything that is professional or production like Outlook for a company. The damage was done the linux empire was about to fall there.
So, going to another FOSS like postfix? Without support like citadel?! or going for some linux licensed software like Zimbra. The company administration made the right call. If they were going to pay... then let's buy win 2008 and exchange 2007.
That solved all the email issues, and 1 of the worse years of my life. Trying to tape and glue citadel every day... and get Thunderbird to do what it was supposed to do. Just one big integrated, easy to use email enterprise solution.
- Second problem, the backup server running BACULA FOSS. lololololol "Bacula, it come at night to mess the IT guys life"... one of the hardest softwares I have ever fiddle with.... shhhh 10 config files in the server, bunch of deamons, bad client with stupid config files... I don't know... just crazy. Well, some of the clients, without any apparent reason would disconnect from the server, some deamons would stop.... lot's of backups lost. The vault box would also disconnect from the server scsi card (because there was no native support for the adaptec scsi card ... lol it was built from rpm...etc. Troubleshooting every day.
So we had to pay for something that works without having to have a guy babysitting the software. Amanda was extremely expensive, so what to do.... let's go windows 2008 :)
- Small projects all started to die, several reasons... no FOSS software would not cover all professional needs in very specific areas, no support, no patience anymore in the company management.
- The surviving server.... the web server. The only one still breathing, running webpage, nagios3 and other web based applications. Good stuff there.
- My laptop still survives with 10.04 64b... but I have win XP installed with Virtual Box, (few comments about virtual box, works OK, far away from anything professional like VmWare. Desktop sharing does not work, usb does not work, sound = bad...etc.)
Since VmWare is expensice, the best option is just to buy win7 Pro. Right now I'm seriously thinking about killing Virtual Box and install Win7 in dual mode.

@Home:
I installed win7 in my main computer at home **(in secret)** to play C&C and because of a video capture card that as no native linux drivers or support available. So I'm running win7 64 and Ub 10.04 64 in dual boot mode.
I noticed that after installing win7 dual boot that I would start to boot in win mode a lot more than ubuntu?!?!
The reasons... good boot/shutdown time, very good resource usage, games, support for any hardware, specially anything related to video or graphics, and a huge amount of software that just works out of the box (and I could still have a lot of FREE software like OpenOffice, crappy Thunderbird, etc.)
So right now I boot in win7 mode 6 days a week. 1 day I boot ubuntu 10.04.
**The Score @ Home:**
1 Desktop with win7 / Ub 10.04
1 Desktop with win xp
1 Web Server with Ub 10.04
1 Email Sever with crappy citadel and Ub 10.04
1 Email server with postfix and Ub 10.04
1 Server with 2008


**CONCLUSION:**
1. I think the Open Source momentum is slower now that win7 & 2008 came out than 3 years ago.
The community is weaker now than 3 years ago... easily visible in this forum... less how-to for 10.04, less help, less posts with something useful and troubleshooting. Just the same old guys and old posts about how to tweak the desktop and their graphic card.
Same old stupid posts about newbies that can't afford win7 but in reality are not FOSS believers and expect things to work like in windows.
Releases every 6 months... come on guys... STUPID... Do you guys remember the 10.04 flop launch and the iso images...shhhh, most of the stuff did not work and needed to be upgraded 1 to 2 weeks later. Lol alpha and betas ...shhhh lol. FLOP.
Just my opinion... What do you guys thing about this?

I really don't know about the future? Everything is easy in win environments except the price.
Everything seems a lot harder in Ubuntu Environments but it's FREE.
When it comes to hardware support... shhhh ... no comments. Even the easiest webcam can be a nightmare.
My determination is a lot weaker, but I truly believe in FOSS!

God Bless
PMLA
