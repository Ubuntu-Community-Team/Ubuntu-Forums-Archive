### Post by mauserrifle on 2006-10-31
Yes, finally I've got to the point to be using Linux as my main OS.
Windows XP just if it really is necessary.(For C# programming and stuff I do for school. I have to work alot with MS products)

I've been trying linux for many years now. It began with Mandrake Linux. Then Ubuntu when the first release came out.
Also been playing with knoppix livecds.
But always got to the point going back to Windows XP.



Why?

1. X crashed every time when I did something wrong.
I didn't know much about the terminal. So getting backups back was bit hard.
I didn't understand the X config either.
It irritated me, going through guides and end up with a broken os.

2. Installing video drivers was a pain. I had an ATI 9600 pro card which was no fun on linux.

3. I just couldn't get used to the environment. I was used to fast clicking in WinXP so it frustrated me I couldn't do that on linux. And all these terminal commands. pfftt (Just a matter of getting used really).

4. It's so frustrating starting a topic every hour about a problem and not understanding what it is all about.
Reading all these commands to typ. But what do they all do? 

5. Drivers weren't so great, I had alot of problems with my on board audio.


Now things changed.

I'm so used to environment. Much better than any OS I've tried.
I'm getting used to the file system. Finding my way.
I love Gnome!

I've got a Nvidia card at the moment :) 6800 GT.
Drivers are so easy to install now. Thanks to these great packages and guides!

I understand X.
I even got dual screen twin view working + beryl-manager!I got a 3d kubus desktop over 2 17" screens! How cool is that ;)
I was just playing with my desktop for 2 hours. Amazing stuff, really.

I'm used to the terminal, can do most stuff with it.

Feel so free using this software. I love it! Best desktop experience I've ever had.
I specially like the community. It's just fun reading other topics, it's a good way to learn.

To all the new people, you got to make mistakes to learn. Keep it up, it will be worth after :)
I still consider myself totally new to the OS but slowly learning.

Thanks Ubuntu for your great OS :)

Cheers,

Maurice
