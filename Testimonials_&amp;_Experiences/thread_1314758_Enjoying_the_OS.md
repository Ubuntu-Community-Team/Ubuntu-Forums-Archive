### Post by Twilight- on 2009-11-04
I've actually had some very good experiences with Ubuntu, and I'd like to share them.

First of all, installing 9.04 when I first started using Ubuntu (and any distro of Linux for that matter), it's been nothing but great for me. Sure, there are a few things that I had to get used to such as different programs rather than the ones I use on Windows, but for the most part, adjusting to things like terminal didn't take that long.

I've been using Ubuntu for 6 months and have recently upgraded to 9.10 on my desktop (yay me!) without any problems, and am currently *attempting* to install it on my laptop as of the typing of this thread. I enjoy the fact that I can use it beside my current Windows XP/7 dual install without any problems. Mind the fact that grub takes longer to load with 9.10 for me, but that's alright... I'm not letting that ruin my happy moment.

Thanks. Great operating system, great software, great user base.
