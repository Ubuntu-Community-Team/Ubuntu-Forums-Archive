### Post by sharkinfested on 2008-05-31
I grew up using Windows and Macs.  I have a laptop running XP and a MacBook running Tiger – both are extremely stable operating systems but due to the overwhelming number of viruses, worms, adware, and spyware aimed at Windows I primarily surf the web on my Mac.  

My sister recently bought an Eee PC with Xandros and after playing around with it I have to say I was really impressed with the OS.  I decided to give to give other distributions a try simply because it’s free.  We have an older Acer laptop in the family that no one really uses anymore and it became the test subject for several different distributions.  In the end I found Ubuntu to be the best choice for me.

I’m really blown away at how easy it was to try out the different distributions and then load one on my laptop!  I even did it as a dual boot with XP and after a few minor glitches I was able to get the wireless working.  (I’ve always been able to use my computers but I’ve never tried to load an OS – other than use a recovery CD to reload an existing OS.) 

My nephew was so impressed that someone with my low level of expertise could download, burn, and then load an OS as a dual boot that he decided to dual boot his laptop with Ubuntu and XP too.  I’ve only been using Ubuntu for about a week but I’m finding it to be a very pleasant experience.  I even find myself reaching for it more often than my other laptops.

Does this mean I’m going to switch over to Linux full-time?  

No – of course not!  There are things I like about each OS – of course there are things I dislike about each one too, but the fact is I already own a lot of software for both of the other systems and it would be silly to throw it all out and start over.  I am keeping Ubuntu though, and I will add it to my current XP laptop (too difficult keeping up with 3 laptops).  Everyone keeps insisting that switching to a different OS will give them freedom.  I think ADDING a different OS to your repertoire gives you more choices – that’s REAL freedom.
