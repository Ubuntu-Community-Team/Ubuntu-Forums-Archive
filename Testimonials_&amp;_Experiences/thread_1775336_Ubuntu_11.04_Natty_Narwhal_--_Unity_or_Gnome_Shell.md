### Post by 5rc on 2011-06-04
Hi,
 
 I would to give feedback to the Ubuntu team for the first time. Although  I have been a Linux user since the early beginnings of Red Hat.

I  had adopted Debian as my favourite distribution for quite some time,  but after its appearance have then since changed to Ubuntu.

I still test major distributions annually so I can say that my opinion whichever it is is certainly solid and reasoned.

I  am also a developer although my time is mostly spent on Web  Development, unfortunately I am not a Ubuntu contributor, but an all  time user.

Okay, so here it goes.

Natty Narwhal is  something of a "starting revolution" on the Linux desktop, and that is  why you will certainly find divided opinions. Here is mine.

I  have installed Gnome Shell on Ubuntu for at least three times now (kept coming back) --  this is my feedback on what are the advantages and disadvantages of  Gnome Shell and Unity, and why have I ended up with Unity.

As I have read several times, people often have a "depreciative" or at least "divided" opinion about Unity.

This  is in my opinion, due to more "traditional" users not  really adapting to "radical" changes on usability interface, which is quite understandable, as for some this might be "unaffordable" time consuming (or for any personal opinion).

Most Ubuntu users came  from Microsoft Windows operative systems, and as we all know from  Windows XP to Windows 7, nothing changed that much for better, but how  everything is "organised or displayed", whether you agree or not -- I strictly only use Windows for cross platform/browser testing  or to occasionally use other major commercial licensed software. So despite of a lot  of Ubuntu users having a not so well reasoned opinion, "Joe User" still  "points the finger" to usability in the line of change of Unity how they  acceptably would do to Windows, although for me both operative systems shouldn't really be compared on their foundation at all.

I am self-confessed to have  done the same, initially. I mean, "who are these guys" to take  "deprecation" over the GNOME Desktop? So I thought.

Only to verify that Unity is actually in the right path, and this is why.

Gnome  Shell is still very "buggy", and so is Unity -- point blank, nothing  new. Although Unity still seems to match with most of the existing  proprietary drivers better than Gnome Shell.

Gnome Shell, in  my opinion, brings new life to a Graphical Desktop Environment, as the  name suggests, you actually will end-up using your keyboard a lot more  on your actual GUI, reducing the mouse pointer usage to the absolute  minimum. Really practical, and very fast to browse, search, run commands etc. So  does Unity (although I found it a bit more "buggy" on these terms).

But one thing is clear, where Gnome Shell has failed Unity seems to have prevailed.

It  is for me the main difference that Unity tries to gather a little more  of "Mac OS" than "Windows", and having actually never been a Mac OS user  myself, I love it. The "fusion" of the "Desktop bar" and the "window  top bar" is absolutely brilliant, the ultimate "coolness".

Gnome  Shell though adds a little more of the "magical spice" of the Mac OS on  its turn, and also has the multiple-desktop visualisation interface  better worked-out, but fails on these little "workspace" details that  actually make all the difference.

Which makes me think, that  Unity is still in its start -- check for example when the "Desktop bar"  does not include the closing/maximising buttons, on hovering, the "ALT  menu" still shows up with left indentation -- not quite matching the title  position, a little aesthetic detail that would actually make a  difference.

In contrast, the Gnome Shell top bar is not that  practical in anyway, for example, its rounded corners seem rather  purposeless or even taking some flexibility of window themes while it  really only takes "unnecessary space".
 
 Although the Desktop "Dock panel" might look more pleasant to the eye (somewhat more aesthetically pleasing) and the  top left corner action give you a bit more advantage of the "Shell  environment" itself -- out-of-the-box, Unity joins the end-user  experience with simplicity to the "stable" navigation experience.

I think Unity is in the right track. And it will certainly not stagnate, but evolve from its stand -- I hope.

Next  Ubuntu versions will eventually also include the Gnome Shell as an  "alternative" desktop environment as they currently produce the "classic  Gnome Desktop" -- I reckon.

I think that a small improvement  would be removing the the dock panel top bar section and pulling  the logo to the left (somehow).

**Note that if you try to install Gnome Shell at this moment in time, it will brake your Unity.**

At this moment I'm loving the Unity concept. And left Gnome Shell for now. Let's see how it goes...

Well done to Canonical and the Ubuntu team!
 
 Thank you for all that you have done to Linux (and Debian) -- truly.
