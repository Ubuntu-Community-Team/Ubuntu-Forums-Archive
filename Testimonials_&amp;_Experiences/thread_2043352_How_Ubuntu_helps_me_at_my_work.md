### Post by slooksterpsv on 2012-08-16
I know sometimes people have questions like how can Linux make someone's life easier in the workplace. Well, I have a real-life example of how Ubuntu has helped me with my line of work.

I work for a Cell Phone accessory company, and while small, there's still a lot that has to be done. We've had to go through spreadsheets of inventory, upload data to websites, etc. There have been a few instance where I've had to use Linux to help make things go more smoothly and easier.

1. We needed a dropbox like server but without all the ramifications and policies that dropbox has, so we setup an owncloud server for use in case we need to share files, upload documents for backup, or even just sync documents to shared computers.

2. We've had to mull through data on spreadsheets and needed to pull that data out or grab items from it that has links to the web. With Ubuntu I use python and bash scripting to make this process a lot easier. Search through the spreadsheet, pull what data I need, use any data in the other columns to help describe said data, and done.

The scripting is probably the most useful at the moment.

3. Using Ubuntu at work I'm able to run multiple things at once without hiccups unlike our Windows machines. I can be updating lists, downloading data, and having a program run to modify said data. Windows, even with tons of RAM, still has hiccups every now and then (lags, freezes, etc.).

There's more info I can give in the examples, but thought I'd share that real quick.
