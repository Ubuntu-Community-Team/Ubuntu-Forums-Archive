### Post by Damaniel on 2007-04-20
I'm not a newbie to the world of Linux -- I've been using various distros off and on for over a decade, and my job involves developing software for an embedded Linux-based product.  Even with all this Linux experience, a personal sore spot has been the continued necessity to run Windows on my PC at home.  

Every time I'd go to install the distro *du jour* on my main PC, I'd find that the system would work great overall, but just enough hardware wouldn't work (or would require ungodly amounts of tweaking) that I'd end up frustrated.  On top of that, I've been an avid PC gamer for as long as I've been using PCs, so even a successful Linux install would be relegated to the secondary partition of a dual boot system.  I didn't want to be hassled with swapping between Windows and Linux, so my Linux partitions usually ended up completely unused.

All of this has changed in the last couple of months.  I put together a PC for a co-worker running Ubuntu Edgy (installed at his request), and I found the installation and configuration to be very easy.  Everything worked right away!  After realizing that I wasn't really doing any PC gaming anymore, and that almost every application that I ran on Windows was open-source anyway, I figured that I'd make the switch and finally abandon the world of Windows forever.  

I went ahead and installed Edgy on my PC as well, after unceremoniously deleting my existing Windows XP partition.  I did have an issue with the 'nv' driver during the install process (apparently a known problem with my GeForce 7800GT card), but I was able to switch to the 'vesa' driver at bootup and do the installation.  

After the install (and the downloading of the latest Nvidia driver), everything just worked!  The graphics tablet, dual monitors, sound card, my printer -- even my USB MIDI interface all worked out of the box.  Getting Beryl running and installing the non-free codecs was very easy.  The amount of tweaking necessary has been very small (really just downloading additional software packages), and everything runs fast and stable.  I've also installed a few non-repository based applications (Scratchbox, Xephyr and the Maemo development kit for a Nokia N800-based game I'm working on), and have had no issues whatsoever in making them work.

Out of all of the supposedly 'user friendly' distros that I've used over the years (Red Hat, Mandrake/Mandriva, Fedora), Ubuntu is the first that truly deserves to be described that way.  It's easier to install than Windows, and far more functional out of the box.  Even family members that have seen my PC drool over the Beryl-powered eye candy and ask me 'how can I get that'?  At the rate that Ubuntu is progressing, I think the number of people asking that question (and taking the plunge themselves) will only continue to increase.  Congrats to everyone who has contributed to Ubuntu and the Ubuntu community!
