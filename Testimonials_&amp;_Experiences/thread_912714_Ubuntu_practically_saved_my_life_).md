### Post by latinoheat69 on 2008-09-07
Well, I'm 16 years old and have been a Windows user, still I'm.  But i have tried many times Ubuntu, though I always remove it from my dual boot after a month since I miss my Windows :$.  So the reason I'm writing this is to share my experience. So I have an iPod classic which it's hard drive recently got bad blocks (or sectors, don't know the proper term) and my music was there. Oh, did I mention that my laptop's hard drive had also got formatted by mistake :D. So my iPod was basically my music backup. I installed Windows on my machine and started to copy my music from my iPod, but Explorer started crashing because it couldn't handle error from the iPod's bad HDD. So I thought I was screwed, my music library which I had been collecting ever since I was born, all those 30GBs of music we're basically lost. That's until I decided to try and see if Ubuntu would be a life saver to me, basically a miracle. After I started copying the files to my computer's HDD, it was transfering extremely fast, faster than Windows, but it stopped at a point, and I got so scared that it was gonna do the same as Windows, so I prayed, and prayed to God that everything turned out OK. It did, Ubuntu displayed the name of the song that was on a bad block so I wrote it down to re-download later, and told Ubuntu to skip it.  After doing this for about 3 days I had all my music back on my computer and was extremely happy.  Ubuntu = Miracle. I would've probably commited suicide if I would lose all my music.  I owe my life to Ubuntu, LOL. So after this I decided that I'm going to buy an external hard drive to store all my media in there and dual boot Ubuntu and Windows on my machine (notice I say Ubuntu first ;). Well, that's my experience.  Thank you Ubuntu Team. :D
