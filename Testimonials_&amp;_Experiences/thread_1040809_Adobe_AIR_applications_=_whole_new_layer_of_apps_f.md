### Post by rmausser on 2009-01-15
I hope this isn't already posted, I did a quick search, but I just wanted to say that if you haven't checked out Adobe AIR yet, do it now!

Basically in the simplest terms Adobe AIR is a free application from Adobe (duh) that allows for programs to be easily created using Flash or HTML, or allows for the porting of web based applications as normal apps on your computer. 

Since, the installer is available for linux, what this means is now we have access to 1,000's of additional programs, the majority free. 

check out the applications here;

[http://www.adobe.com/cfusion/exchange/index.cfm?event=productHome&exc=24&loc=en_us](http://www.adobe.com/cfusion/exchange/index.cfm?event=productHome&exc=24&loc=en_us)

and download adobe AIR from here: 

[http://get.adobe.com/air/](http://get.adobe.com/air/)

After you have downloaded the program, in a terminal go to where the .bin file was downloaded (for this example, the desktop):


> cd Desktop

then to install it type:

> chmod +x AdobeAIRInstaller.bin
./AdobeAIRInstaller.bin

Once it is installed you can navigate to the first site I posted of applications. 

Then there should be a link to download the application. If you don't have the latest 32-bit flash installed (I don't I have the 64-bit for example) then the download button won't show up. 

There is a link to the developers site as well where you can usually find a link to install the application. 

Either once downloaded the installer will automatically start, or make sure you specify "open with" Adobe AIR if it does not.

There you go, there are some really great applications to be downloaded, so enjoy. 

Rob

You are right. Flash can be rubbish. But people can also code using a multitude of other languages with AIR. 

I think this is a great thing for linux because imo cross-platforming is really what is going to get people attracted to open source software and then linux. The average user is not very comfortable completely changing the way they do things, as we saw by that extremely intelligent induvidual who almost dropped out of college because they were unfamiliar with Ubuntu. 

I feel that if people get comfortable with using open software in Windows, then realize that they are the same or better than proprietary applications, it will hopefully spark their interest that maybe an entire open source OS is a better choice as well, especially if these same applications can run in it easily. 

I dunno, its always exciting to me when certain companies take the effort to make programs that run in linux.
