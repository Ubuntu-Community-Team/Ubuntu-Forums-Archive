### Post by chrisjsmith on 2012-06-06
I've quit using Ubuntu and Linux as a whole.

12.04 has blown it for me in less than a week. Now I've been with Debian since 1997, Ubuntu since 2006 (6.06). I've been on these forums since 2008.

I've actually spent the entire last week moving all my crap from Ubuntu to Windows 7 (and moving our hosting ops to Windows Server 2008 R2 which was quite painless).

Windows has improved to the point that it offers me two things I just can't get from Ubuntu any more:

1. Consistency. I don't want change for the sake of change. Canonical and Ubuntu are too easy to jump for change every two minutes on things that don't matter (Unity) yet leave people in the lurch for things that do (buggy MySQL package on 10.04 LTS). For stuff I don't want to upgrade, I can do everything out of band on Windows i.e. I can upgrade mysql independent of the repositories. It allows my software environment to NOT be 100% synced with the distro repositories.

2. Reliability. Seriously it has declined to the point of it being unusable on all my kit (see signature for details). My wireless breaks every two minutes, none of the kit sleeps or hibernates properly and Ubuntu eats my laptop battery in 1h25m compared to 5h15m on Windows - both with 9-cell+ultrabay batteries on a Lenovo T61). I don't have the time or inclination to piddle around getting stuff working every time a new kernel drops either.

I really don't mind paying for stuff as it's a time efficiency thing, so Windows it is. Sorry but it just works properly, and that is saying something!

(I know someone is going to mention Windows 8 being a turd which I'm fully aware of - I've got until 2020 with Windows 7 which is fine by me!).

I will say a big thank you however to anyone who has helped me on the forums. 

Good luck and please take these notes as advice, not a flame.
