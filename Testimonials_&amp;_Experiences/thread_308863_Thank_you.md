I'll (try to) keep it short.  I wanted to get it out of my system and get on to the fun.

Thank you Ubuntu / debian / GNU and others.

I've used *nix off and on over the past 14 years--mostly off, sadly.

I've followed the development of the GNU / Linux systems over the years, but never took the plunge.  GNU/Emacs was a good friend in college.

Ubuntu deserves a big hand in making a distro easy to install with manuals and a forum.

I've been dabbling with Ubuntu for about a month now.  I'm enjoying the journey.  And the crash course in relearning bash commands as I trashed a couple installs--ouch!

Unfortunately, my computer dosen't like burned CD so well, so it takes a long time for it to read them--my burner is external.  Well, I guess I should give my computer specs:

Old sager laptop from 1998!
333 MHz AMD K6 -- screammin'
256 Meg Ram -- Manual claims max is 128 Meg
3 Gig removable hard drive (Ubuntu 6.10)
4 Gig removable hard drive (Window XP --long story)
Netgear 108Mbps Super G Wireless.
Lexmark Z52 Printer.  Hey, it was free!

Yeah, I probably should run the Xubuntu, but I wanted it all!

Ubuntu runs a little sluggish on this system, but that is okay.  I'll get a new computer soon.  Another long story.
