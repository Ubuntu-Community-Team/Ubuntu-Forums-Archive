### Post by sisco311 on 2008-05-26
[HowTo restore the default panel in xubuntu]("http://www.uluga.ubuntuforums.org/showpost.php?p=4846557&postcount=2")

After you restored the panels add a Keyboard Layout Switcher to the panel (right click on the panel: Add New Item -> Keyboard Layout Switcher)

Go to Applications -> Settings -> Setting Manager -> Keyboard -> Layout and add the Turkish layout.

Select the Turk&#305;sh layout from the Keyboard Layout Switcher.

Maybe you need to log out and log in in order the changes take effect.
