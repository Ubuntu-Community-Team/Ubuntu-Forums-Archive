People keep saying this but it isn't true. The purpose of an LTS release is NOT to be more stable than any other release. The purpose is to provide a periodic release with a longer support cycle for those (like corporate users) who may need it. That's it. No additional stability, no extra testing, no super-powers. This used to be spelled out pretty clearly on the Ubuntu website. For what it's worth, three out of the four releases that were least stable for me were LTS releases.

Bottom Line: There is no guarantee that any individual release will be more stable on a given computer than any other release.
