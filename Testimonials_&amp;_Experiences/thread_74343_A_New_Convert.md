### Post by claco on 2005-10-11
I've used windows forever and a Mac user long before that. I've done programming on both sides of the fence...DotNet, VB, Perl, PHP, Python etc. I run XP on my laptop, my desktop (gaming rig) and FreeBSD on my servers.

I've always disliked Linux in favor of FreeBSD. That was mostly due to the file system layout, and ports. After attending the Ohio Linuxfest this month and snagging a few Ubuntu CDs, I'm a complete convert on my laptop. I've been running it for 5 days now, and I only boot XP to run tests on my CPAN modules during development. The only problem I has was with my wireless card. More about that later. :-)

Ubuntu is without a doubt the friendliest 'just works'
Linux distro I've ever tried, especially on a laptop wth wireless. On top of that, I've always been a KDE person, but Gnome on Ubuntu is clean and simple.

I don't see me using any nother distro, or even XP full time on the laptop again. That's saying something coming from a mostly XP user.

<wireless_rant>
I spent a day and a half trying to get my wireless to work using the network config utility. This utility has a few problems. First, it needs example text. It isn't obvious that you need to entry a key in the xxxx-xxxx-xxxx-xxxx-xx format instead of 0xxxxxxxxxxxxxxxxxxx.

Second, ther's not place to set the channel, or the drivers don't appear to autoscan for it. Once I got my key in, I have to manuall set my channel to 1, instead of the default 6. That's not good when I'm not in control of the WAP itself.
</wireless_rant>
