### Post by BigSilly on 2007-05-12
I've been reading these types of post here for a couple of weeks now, but after installing Ubuntu 7.04 I suddenly feel utterly compelled to add my own.

I'm no Linux expert by any stretch, but I've been slowly moving towards it over the last year or so, starting with a couple of versions of Mandriva, then onto Freespire, and now Ubuntu Feisty. I have to confess, I'm not exactly aut-fait with the terminal commands, but I have had some limited succes with what I did attempt, and it was certainly enough to encourage me to continue with Linux usage.

Freespire particularly, was very easy to use and didn't require much command based instructions at all, thanks to the excellent CNR. So with that in mind, I was a little worried about trying Ubuntu. I expected it to be a bit more of a hardline Linux system, with much emphasis on text input. But my curiosity was such that I really wanted to sample what users had been raving about, so I installed 7.04 over my Freespire.

I'm so glad I did. I cannot believe that in such a short space of time, maybe an hour or so, I'd got my new Ubuntu system up and running just how I like it using Ubuntu's package managers. I'd performed a clean install of Windows on the same PC on the same weekend, and that had taken me a whole day to get running properly! Yet here is Linux, that's notoriously awkward to use, up and running in a fraction of the time.

I've been using it for a week now, and it's already like an old friend. The family are all using it like they've never used any other system. It's so approachable and capable, and everything you use is so seamlessly integrated. 

My only regret is not dumping my Windows partition. I'd toyed with the idea, especially as no-one here has used it for ages. I decided to keep it around for my games stuff, but in hindsight I've not really played them for a while, and wouldn't really have missed it. Still, that's certainly an option I look to take in the future after this positive experience.

So, my thanks to the Ubuntu Linux teams, and the software creators. I don't know what else to say but thank you to you all so much.
