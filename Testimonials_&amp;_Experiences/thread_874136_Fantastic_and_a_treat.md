### Post by FranMichaels on 2008-07-29
The experience was pure awesome. The machine is a brand new Dell 530 from costco. Integrated intel graphics, quad core2 2.40 ghz, 500 gig hard drive, 4 gb ram. 

I used the 8.04.1 64-bit release, as the core2 handles it. I wiped Vista immediately, (pressed F2 to change boot order for CD first). 

The install auto detected everything, correct monitor res and driver, keyboard and mouse, the onboard sound and graphics etc. 
Install was speedy. I set my partitions (as I prefer /home on a separate one.)

For internet, after setting up the machine (as it has only an ethernet port) I got a ring at door bell with my Belkin Wireless Plus G MIMO (USB). I hooked it up, rebooted. clicked the network applet and am surfing the web and downloading updates as we speak. The best part was ignoring the label on the Belkin, STOP use driver disc first. Nope. 

Plug and play and on 64-bit, I'm super impressed.

:D
