### Post by elmerfud on 2007-03-01
I've been a RH/Fedora user since RH8 was released.

This week I converted to Kubuntu.  It rocks.  I am very happy with it.  Prior to this I was running FC6.  

FC6 was OK, but it always required a lot of fiddling.  Always something broken or something that didn't work.  I got the impression that the Fedora developers got stuff done and shoved it into release to see what would shake out of it, with no regard to the end user.  

Ubuntu is much better.  I can already tell its way more stable.  Lots of stuff that didn't work in Fedora works properly in Ubuntu.  Things like the Media selector in Konqueror work properly.  Its refreshing.

There are a few things about Ubuntu that took some getting used to.  First the whole pkgs versus rpms.  I think dpkg and apt-get could use more functionality.  I love adept.  Wow, thats the best installer yet.  Synaptic is OK, but its so slow. I'll be using adept more from now on.  

The sudo versus root thing is a change too.  I don't mind it near as much as I thought I would.

I pretty happy with Kubuntu !
