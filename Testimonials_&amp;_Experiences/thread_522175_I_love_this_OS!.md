### Post by BigSilly on 2007-08-11
It is great isn't it? I still consider myself a newbie with Linux, but I have used other distros such as Mandriva and Freespire. There's no doubt that Ubuntu is simply the best Linux there is. It's secure, stable, fast, easy to use, constantly updated and the repos are stuffed with up to date software. Other distros can claim some of these attributes, but not all like Ubuntu can.

I've been using it now for about four months since Fiesty's release and I'm utterly sold. The whole family here use it with no bother at all. When the next release comes out I'm getting rid of Windows to dual boot Ubuntu with another Linux.
