### Post by pete04 on 2012-07-26
I have a few interesting cases, mostly very positive.

At home, my wife -- not technically inclined at all -- really likes it. Never a single complaint.

I've been phasing in Ubuntu at work and interns I've had -- high school and college -- take to it without ever asking a question. They create docs, edit images. No matter what, they have no learning curve. 

Whenever I'm asked about it, I always try to be really clear that going linux is not for anyone married to specific applications. Some realize pretty quickly that they can't live without something they like on Windows. And that's that.

### Post by Irihapeti on 2012-07-27
My elder son, who has a learning disability, uses Firefox at home. He stayed with me for a few days at a time when I was doing ISO testing for one of the 12.04 pre-release milestones. I did a test install late at night and forgot to reset Grub back to the standard installation afterwards.

The next morning, before I got up, my son turned the thing on, was presented with the login screen (the other install does an automatic login) and found the guest account with no trouble. Never mind that it was Unity instead of my standard gnome-panel, he found Firefox and loaded that. He did want to know if I'd need to tweak NoScript so that he could access Facebook, but that was all.

Interesting what you can do when you don't approach it with presuppositions.
