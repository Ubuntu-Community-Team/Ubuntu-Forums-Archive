### Post by TheMono on 2006-07-06
I'm a full blown linux neophyte - the only reason I tried it at all was I saw ShipIt Breezy discs in a local computer store.

I sell computers as well - and I've ordered a bunch of discs for Dapper so I can help spread my experience as well.

I have to admit though, I set up my dual boot using windows software to create the other partitions. I installed Breezy, and was very impressed. Except that my wireless didn't work.... So I never really got using it.

My Ubuntu partition sat dormant for a while. 

Dapper was released, ironically only a day or two before the last day of my broadband data cap, so I was able to download it. Installed Network Manager - My wireless just worked. Well, this was a start. Tried setting up my palmtop - just worked. Easier than the hours I spent with Palm Desktop in Windows. Faster than using the Outlook conduits to use GPilot and sync with Evolution. My external HDD and DVD writer - just worked. Even my IR remote for my music just worked. Where Breezy had problems with me using different Keyboard layouts on my USB keyboard as compared to the laptop keyboard, and I had to manually switch them, no longer. In Dapper, it just works. Following instructions on the forum is so easy using the terminal - I don't even have to understand what I'm being suggested if its something complicated.

For everyday activities, I never see the terminal though. I then bought a new Laser printer. Plugged it in, went to add printer. A ten second web search told me which driver to select, chose it, printed something. Surprise, surprise. Just worked.

I haven't used my XP Partition in weeks except for playing network games against my flatmates. And even then, I'm working on WINE to solve that last ten percent.

It's not the freedom (as in speech) that lures me as much as freedom (as in beer) as well. Although, that's not strictly true. I love the fact I don't have to pay, and that I get upgraded for free. But I also love the speech aspect. I'm not a programmer, never will be (my only achievement was getting my second monitor working, the one thing that did not just work lol). But I love the fact that I am free to download things that other people have written (for free [beer]), which modify any little part of my OS. As long as I can find a programmer who wants it the way I want it, I can do it. Installing via synaptic just works, installing via a .deb just works.

My vision for what Edgy should be is simple. All it needs is: an easy way to edit xorg.conf, the option to install XGL/Compiz (not that I would, my laptop won't handle it), an easy way for true neophytes to find Automatix, and there is the perfect OS for me. Also, I'm unsure as to whether every six months is appropriate for a really good release - I'd be perfectly content with annually, or maybe even biannually with a "service pack" chucked in in the middle.

The developers of Ubuntu have done an incredible job. I can't imagine the amount of work that goes into it, but I wish them all the best for the next release.
