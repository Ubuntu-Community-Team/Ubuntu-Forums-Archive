### Post by MattBD on 2007-08-05
The layout issue isn't necessarily a big problem. If you've come to Linux from Windows, and you want something that looks familiar, then I think KDE is a better choice of desktop than GNOME, because the layout is more like Windows. You could download Kubuntu, burn that to a CD and see whether your family prefer that.
Alternatively, if you want to install KDE in your existing Ubuntu installation, it's easy to do so. Just open a terminal session, and enter the following:

sudo apt-get install kubuntu-desktop

Then input your password when it prompts you. It will also ask you later on whether you want to use KDM or GDM as your desktop manager. I found the only practical difference is that it uses the Ubuntu login screen if you go for GDM or the Kubuntu one if you go for KDM.

The Kubuntu desktop may also satisfy the iPod users in the house, as it includes Amarok. I use Amarok to sync my 2nd generation iPod shuffle and it works fine once the appropriate codecs are installed.

Don't get me wrong, GNOME is a great desktop, and is probably the nicer-looking one, but it's not for everyone. KDE is well worth trying if you have come from Windows.
