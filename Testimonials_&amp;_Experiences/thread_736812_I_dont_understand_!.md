Finally!  Somebody who acknowledges how great Ubuntu is!  About the command line... we call it the terminal.  Before you had a graphical user interface (GUI), everything was text-based.  Ubuntu and Linux stess the importance of using the terminal over some graphical things.  For example, if you wanted to install a program such as gtk-gnutella (a P2P client like Frostwire and Limewire), you could search the Internet for it and download it and hope you got the right one for your system, or you could go into the terminal and type ```
sudo aptitude install gtk-gnutella
``` and press Enter.  Technology evolves, and we should learn how to do things the old way as well as the new way.  That's where Windows lacks.  Windows stresses doing things the new way, and when something breaks, the users don't know how to fix it.  Ubuntu teaches you how to get "down and dirty" and fix your computer, something that most people who switch from Windows to Ubuntu don't want to do and find it really hard to do so anyway.  Give it a chance.  I feel like I've learned so much more about computers now that I've switched to Ubuntu.  Oh yeah, Ubuntu's support for many programs and applications has skyrocketed too.  You wouldn't believe how much support it has now compared to it's first release!  If it doesn't support what you have, just send a request or comment to Launchpad and here and just wait and see!

### Post by lswest on 2008-03-27
I do understand how some people are afraid (or just too lazy) to learn the different ways of how things are done, but I (being pretty good with computers - mac, linux, windows, hardware, software, so far i haven't run into anything i can't fix) can't really judge, since i used the command prompt in windows, and i use the terminal in Linux.  I actually used cygwin (bash shell in windows) for a while.  I just find it so much more quick and efficient than doing it the GUI way.  But that's just my preference (i actually used Debian for a while without an xserver, because it wouldn't recognize my gfx card right xD).

Besides, any of the friends i convert to Linux (i've done it to a few) will ALWAYS be able to expect me to help them where and when i can.  I think it's always easier to switch to a new OS if you've got someone (physically) around to help you with it.

I guess my point is this:  Often, people mistake Linux for Unix, and Unix is often shown as all text-based, which scares people off.  And if people do trust themselves to switch it's always easier to test the waters if you have someone around who can physically help or show you how to set things up (or to play around with Linux on their PC/laptop before switching).
