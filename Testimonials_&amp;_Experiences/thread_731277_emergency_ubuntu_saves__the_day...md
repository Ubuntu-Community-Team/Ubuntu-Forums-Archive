### Post by sneeks on 2008-03-21
well here is the story....have got 2 units in the house with gutsy on for the kids but my main unit still runs xp as i do need it for work for customers.
then a few days ago a few corrupt files appeared on the xp machine and would not boot,even using a boot disk(super grub).
had a spare base unit so thought ok,time for Ubuntu as i do need to get on-line.
install went well as usual and setup all the stuff i needed..
mail,net,music  dvd's(thats got a lot better)and even used wine for some things.
flash installed with no prob's when using the installer,but i still think the terminal is quicker.
sound was cool even with the hda chipset.
well i had it all.
but it was all too smooth,i was trying to find something wrong but it was all going great.
the only problem i got was that trying to rip to mp3 was a no go even with all the codec's.
but hey,ogg is better sounding anyway.
i sat at the pc thinking,"mnnn what can i do" well the answer is anything,but with no hassles.
i now install a lot of pc's with Ubuntu as a lot of people here have not got the money for a new "gates" install.
and when they get the hang of it i dont hear from them.
the teenager's who use Ubuntu cant believe all the software you can get for free,especially the programing tools,it give's them a kick start so it seems,and good for them.
so on the whole,me thinks i will migrating permanently very soon..
