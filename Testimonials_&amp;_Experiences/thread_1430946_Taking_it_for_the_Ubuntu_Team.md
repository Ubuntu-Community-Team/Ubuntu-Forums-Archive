### Post by sween1911 on 2010-03-16
Long story short, my boss and I went at it today for not having Windows on my home computer. I'm a systems programmer, writing programs for Unix boxes. I support from home using a terminal client. I've never had a problem with being clear on support or being able to support. I worked very hard to get the same terminal client running on my Ubuntu box so our keyboard map was the same as our users. 

I use Ubuntu. I love it. It works, it's simple, secure and free. My box is not that new, so it works perfectly fine using Ubuntu rather than any of the current memory hogging OS's. Just recently, our web programmers put some code into production that caused some problems, so the on-call programmer from our group... ME... got slated to test THE WEB APPS on Saturday morning from home. Apps are written for IE. I don't have IE. To be clear, I've never been required to have Windows or IE to support our users. The users I support use our terminal apps. I tell my boss this morning that Firefox, that I have, doesn't work. You'd have thought I wanted to come to work in a bathing suit. Then I have to sit through people looking at me... "You have what? Ubuwhat? Mac? Is it a MAC? You have a MAC. What? Linux? Why don't you have Windows? What's wrong with Windows? I have Windows. Do you have Windows?"

All this happened before 9am, so I spent the rest of the day staring at my screen fuming in disbelief.
 
************************

We were able to come to a resolution eventually. As most arguments are, it evolved from some misunderstanding on both our parts.
