### Post by bdstein on 2008-04-12
I am now a Hardy Heron user and am loving it.  The only issue I had was with the wireless but it was fixed with some great support from this site.  I have been away from linux/unix/solaris since I changed jobs from a small mom and pop company to working for the navy as a DOD civilian.  I went from working in an all unix IT shop to almost all Windows.  The only unix action I get at work is IBM AIX on a really old RSC-6000 that takes about 30 min to fully boot.:(

In my old job we developed informational touch screen kiosks on a stripped down version of mandrake 9.0 that fit on a 256MB flash chip and would run a Java app from a remote server for the gui and used mysql for the backend......you get the idea.

Well let me say that I never knew linux could be so easy!  Actually having an update manager that tells me when updates are ready allows me to ok the install is awesome.  It's like the ease of windows with the reliability and stability of linux...wow.  Man I may just go back to writing sed scripts for fun!  Ok maybe I wont.:)

In any case I love the new OS, hopefully it will eventually completely displace all the windows in my house.  Vista is such a friggin hawggg!  My wifes new dell core2duo with 2GB of friggin ram actrually runs pretty slow until I stripped all the frigin pretty features off of it and made it run more like XP which is also to friggin slow.

Sorry for the rant, I love Ubuntu and this forum and all the awesome developers.

Peace!
