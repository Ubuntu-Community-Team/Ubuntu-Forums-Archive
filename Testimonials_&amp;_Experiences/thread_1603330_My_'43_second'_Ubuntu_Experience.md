From startup to shut-down  my Ubuntu Install on an Acer Aspire 3620, 512MB, 1.5GHz laptop took approximately 43 seconds to blackscreen (with the exception of log-on).  All this from a fully updated, Ubuntu 10.04 LTS running off a 16GB pendrive that thinks it's a harddrive!!*

*note: Pendrive is not loaded as persistent drive , but , rather is a direct install of Ubuntu 10.4 where the Ubuntu Installer  recognizes  the 16GB harddrive as if it were an IDE harddrive and proceeded to install the operating system (bootable) on the pendrive.
