### Post by dorkdork777 on 2008-03-08
So today I decided to install Alpha 6 as well as my existing 7.10 and XP. These were my results.

[LIST]
[*]Ubuntu 8.04 started and installed fine - didn't test the new item on the boot menu "Install Ubuntu", I just went for the "Test Ubuntu" option and installed from there.
[*]One problem with installing though - Ubiquity hung on 100%, with the message "Copying log files". I just quit, and it was all OK.
[*]GRUB showed Hardy differently to Gutsy (as I had installed already).
[*]With earlier versions, installing the restricted driver for my Nvidia GeForce 6200 would cause the screen res to go to 640x480 and stay there, with no way (except the command line) to set it higher. I was hoping this would happen again, for old times sake. It did, but lo and behold, there was an option for 1280x1024 LCD Monitor in the Screens and Graphics section!
[/LIST]
Great work to the whole team, and I can't wait for the betas! :D
