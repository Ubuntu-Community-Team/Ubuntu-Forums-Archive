### Post by Ruuki on 2010-06-30
Hi, this is only my 2nd post on this forum haha :p I just thought I'd share my testimonial here and thoughts on using 10.4 as an everyday OS. I'm a long time Windows XP user (for almost 6-7 years), but I'm no stranger to Linux. However, I want to briefly explain the awesome-sauce that is Lucid Lynx, at least according to my experience. 

The system that's running Ubuntu at this moment is only temporary and I intend to replace it with in a handful of months with a nicer system, i7 build, set for gaming. I was just going to put windows 7 on it but due to the current experience and everything I'm learning from using it, the PC is also going to use Ubuntu as well. Yay. 

So I booted up Lynx a few days ago after a fail format from a 'missing file' error on the same XP disc I've used for years (I know there's no missing files, the problem is obviously a lot deeper then my patience has time for) so I threw Ubuntu on it via a flash drive (pen drive linux really, live -> full install) due to my lack of a CD burner that I didn't have to walk across the house to use (lazyness is epic), pretty flawless setup. Easily formatted then partitioned a new drive, backup drive, and swap through the live and installed the full version. Very impressed with everything. Fast install, fast boot, runs great, and looks great. 

Of course I'm disappointed with the lack of Photoshop (will never part ways with it), and other programs like FL Studio. But I don't blame Linux for that, these programs just weren't developed for it, and even when you DO manage to get them functioning they either run badly or look ugly. Playonlinux makes it a bit easier though, saves me the trouble of manually making prefixes and organizes everything. It even came with a pre-installed list of games, too bad I don't game that much at the moment.  

Personally, I found that everything ran fine straight out of the box. 

To simplify things here's a list of the little things I like:

[LIST]
[*]The ability to switch from my speakers to my stereo headset without restarting the music player.
[*]The boot times are pretty sweet, very clean startup, no program spam or anything.
[*]As usual, everything runs in a single aesthetically pleasant environment. The programs don't look like they're from separate operating systems or something. Everything looks great.
[*]System runs great and I have more available hard drive space then on XP.
[*]Pidgin is extremely useful. Every IM in one :p I wouldn't mind seeing a better file transfer system and a few more integrated features from each IM (like AIM 7's "life stream", yahoo's ability to add hotmail/live accounts, and god knows pidgin is lacking in the xfire department, and many more)
[*]Gwibber/Evolution make a mean social networking combo. Saves a lot of daily firefox browsing.
[*]The media player that came with it works great (Rythymbox)
[*]The screenlets are cool.
[*]The black-box alert thingy for pidgin/media haha
[*]Really customizable OS
[*]Works great for an every day OS - And it's free.
[/LIST]
Not onto the cons: 

[LIST]
[*]I do find a few glitches here and there. (Like today I mounted an iso and even after I unmounted it and deleted the file, it's still showing up under the drives. Even after reboot. Grr. )
[*]I was a bit disappointed that BOTH the photoshop CS3 and CS5 installers crashed (extended versions), and after getting FL Studio to work it ran terribly. Pretty much every game I install runs terribly as well. Either a messed up/buggy GUI or they just don't function well (programs included). Of course they'd run amazingly if they could be ran natively in Ubuntu but that's just not the reality. This is why I am still getting Windows 7 when I build my new system. I'm flat out married to too many programs/games. Hopefully in the future there will be better support for windows-designed programs.
[*]Nothing else really. I'm sure I'll find more when I dive into the more deeper aspects of Ubuntu, like any OS. Let it be said, I still find Ubuntu to be far more proficient then my Windows experiences - although I haven't tried Windows 7 yet which I am looking forward to doing.
[/LIST]

That's really it, Ubuntu rocks my socks. I think it'll be fun learning to write C/Python/VB scripts on here :p

### Post by sparklet on 2010-07-02
After using Lucid for several weeks I think it is GREAT. I used Debian for 10 years and have just switched to Ubuntu to give it a try.

Good switch !   Everything either works straight away or is easy to get going. Installation is the easiest I have ever seen. I like the appearance and it is also fast.

For working with digital photos I use the GIMP. This looks a bit daunting at first but a bit of "playing about" and a quick read of parts of the manual and it soon becomes familiar.

I tried a commercial editing programme under Win XP. Like almost everything Windows I found it way too restrictive, it just didn't give me the fine control that the GIMP does.

For an exercise in frustration - use Linux for several months without ever using Windows. Switch back to a session in Windows and experience the sensation of being in a straight jacket ! I very rarely find any need to use Windows for anything, but I don't play games....
