### Post by sidraba on 2010-05-12
First things first: My native language is not English and feel free to ask or correct.

I wanted to share my expierence with Lucid Lynx, talk about problems and good things I noticed when switched from Windows 7 to Lucid.

**_About my Windows expierence_**: I guess I'm semi-advanced windows user. I know how to set up computer for my needs, how to find programms, how to find drivers, how to make partitions and so on. I'm using Windows for about 8 years and I like Windows 7.

**_About my hardware_**: I'm having Acer Aspire laptop with 2 gb RAM, [s]not sure about other parameters though but I guess it's not so important[/s] **Acer Aspire 5720 (Model name: ICL50)**.

**_Why I switched to Ubuntu?_**
[LIST=1]
[*]I wanted to change something in my life. :D I know that it's not the best reason why, but it works. You need to set new goals when you have reached the old ones and I thought about Windows as reached goal, I felt there very comfortable...and bored. 
[*]New Ubuntu release, it looked so good that I couldn't resist. 
[*]Freeware and Open Source are great terms too and means a lot for me, I like the overall idea about them.
[/LIST]

**_Positive things:_**
In overall I like my new OS. There are many good things about lucid lynx and ubuntu. I think that ubuntu has user-friendly interface and it has changed a lot in two year frame when I used it last time. 

[LIST]
[*]1) Installation process is madly easy, especially if you want to go dual-boot, couse when Ubuntu sees Windows 7, it offers to divide partitions and set up Ubuntu for dual-boot (ubuntu explains it better than I do). I was surprised that ubuntu does not offers to "remove windows and set up ubuntu instead" (what I needed), so I made partitions by myself.
[*]2) Interface default theme. It was such an eye-candy. I find windows default theme too shiny and glamorous, so Ubuntu default theme met my expectations completely and even more.  
[*]3) I'm very glad about ubuntu usability and user-friendliness. After I set up Windows I do not use about half of default programms, because they are unusable (for example sticky notes has no configuration options), strange interface (my little brother uses Windows Media Player as default media player and I find it almost like miracle, never got this program work for me) or slowness (IE). I gladly found out that default ubuntu software is usable, after realising that I got a little bit distracted couse it was the opposite of what I expected. :D   
[*]4) Drivers. Ubuntu found all drivers itself and I had no need to surf for them. And and and Rhytmbox offers to find codecs for media files  - how cool is that?!
[*]I installed only VLC, Pinta (Paint.net analog), Gimp i guess.. Yep, that's it! :)
[*]5) Configuring global hotkeys. It was nice surprise that I can configure hotkeys for music player in OS.
[*]6) It has nothing to do with Ubuntu but I found Google Chrome bookmark sync fabulous! It felt like home having all my bookmarks after few clicks.
[*]7) Ubuntu system files takes only 10 gb of my hard disc space. I used to have 30 gb for windows 7 (not fair comparison couse I had weighty programs in Windows but still)&#8230;
[/LIST]

**_About problems and bad expierence:_**
Firstly, I guess most of my problems are in curve_hands.so and unstable new release. I understand that, so I'm thinking about downgrading to Karmic Koala for a while.

[LIST]
[*]1) One of the biggest problems (not big in general but big for me) is that I cannot insert special symbols into text by pressing key combination, something like Fn+Alt+0000. It hurts me. :D
[*]2) I was very happy with my Classic Media player in Windows, it had feature that I could stop and start video by touch to my touchpad, I'm not keen on pressing 'space' key. I haven't found solution for this feature in ubuntu.
[*]3) I could not disable *login keyring*. I tried it by adding @include common-keyring to the /etc/pam.d/gdm but it did not work. 
[*]4) No I find it funny but in first few hours using Ubuntu I managed to remove notification area and got stressed about that. Not sure that it is a good thing that you can remove it from panel, it's confusing. :D I was so upsed that my tray icons do not appear. :D
[*]5) Switching interface language. I cannot understand Language tool, so I cannot switch interface language (not fan of local translation). I just don't get the way it's being done. 
[*]6) (Not sure how good I'll explain this problem, so ask, if I get you confused) My data disc (partition) is ntfs as I used to be windows user. To make it my /home I need to format it and mount as ext3 (haven't done that yet). Also I have problems with this ntfs disc = torrent client gives me errors that disc is not accesable, music player does not recognise library from this disc and it shows on desktop (like icon) only when I run it after rebooting my computer.
[*]7) I kinda do not understand configuring of MeMenu too. It gives me no notifications about new tweets or e-mails. Not sure is it bug or curve_hands.su. I miss my Windows sidebar gadgets tweetz and Google Calendar Agenda. Tweetz is fantastic, small twitter client with all features you need and without features you do not need, very useable. And Google Calendar Agenda is notifier for Google Calendar and GMail.
[*]8) I miss Aero Snap feature. I tried this [solution]("http://www.omgubuntu.co.uk/2009/11/aero-snap-ubuntu-linux.html") but it was buggy.
[*]9) I was used to foobar2000 music player in Windows and I hadn't found similar in ubuntu (I loved interface and folder-browsing-sidebar-feature). 
[*]10) The most annoying and dangerous problem is that computer is unable to wake up after closing it's cover. And after shutting it down physicaly (it doesn't wake up, what else I could do?) it loses data.
[*]10a) "My Heritage" software doesn't work on .nix. It wouldn't be problem couse there are alternate solutions for this but I, honestly, tried to use Gramps - in my opinion it's unusable and unfriendly. So I decided to run "My heritage" with Wine.
I used to use Wine two/three years ago when I had dual-boot period in my life. I HAD to say that Wine is changed to the good side since that. It's easy to use and very user friendly.
Problems started when I run "My Heritage" and expierenced so many bugs. I used Ctrl+S after all edits I did (I got "My Heritage is sorry but has to shut down" at least 3 times and at least 2 times restarted programm couse it had glitches). Also it crashed every time when minimised.
Despite of that I did my family tree (I was at grandma's house and writed down her memories). There were about 100 persons in this tree. I saved my work, closed "My heritage" and put computer in sleep (by closing it's cover - my mistake, I know, but I saved data before that). When computer was not able to wake up I had to restart it physically and after that I saw that all the latest changes are lost and I have only ~60 people in my family tree now. It really got me upset. I hate to tell about these ubuntu fails to people because I want to tell them about good things to inspire them using open-source and freeware, to let them know - it's possible to use something different but it's imposible if I have to redone my work wick I cannot do without telling others about why I need to redone it. :(
[*]10b) after that I switched preferences and now instead of going to sleep, computer hybernates after closing cover. But now it takes foreeeeeeeeeeeeever to wake up for him.
[/LIST]

**_Conclusion:_** I'm using Ubuntu 12 days now as my OS and I like it but there are things that could push me to remove Ubuntu and use Windows instead. Couse it's not always shiny in paradise.
Have a nice day!

### Post by bjcubsfan on 2010-05-20
>  I miss Aero Snap feature. I tried this solution but it was buggy.

I've never used Aero Snap, but I've seen it in commercials.  The grid feature is very similar, and I have found it invaluable. It is available in the compizconfig-settings-manager.

If you have not installed compizconfig-settings-manager, do so.  I am not sure if you need all the plugins that I have installed, so I say install those as well.: 
```
sudo aptitude install compizconfig-settings-manager compiz-fusion-plugins-extra compiz-fusion-plugins-main
```

After it is installed, navigate the Ubuntu menu to System | Preferences | Compizconfig Settings Manager. Under Window management check the box next to grid.  You can view the settings by clicking the grid button.  Basically, it allows you to place a window somewhere on your desktop.  By default, it uses <CTRL>-<ALT> and then one of the keys on the number pad.  This works perfectly for a full keyboard.  You just hold <CTRL>-<ALT> and then punch the number that corresponds to the area of the screen where you want the window.  Experiment with hitting the same button more than once in a row.

Enjoy!

BJCubsfan
