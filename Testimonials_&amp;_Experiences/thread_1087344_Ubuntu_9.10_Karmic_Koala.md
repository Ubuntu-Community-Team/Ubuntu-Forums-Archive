### Post by K7522 on 2009-03-05
Yes, your computer will be able to run faster. It's not so much an issue of age of the hardware per se. 

Jaunty is (to my knowledge), the first Ubuntu release to openly support ext4 file system. You can read about ext4 here, I haven't read through much of that to confirm the validity of any of the data there though.

[http://en.wikipedia.org/wiki/Ext4](http://en.wikipedia.org/wiki/Ext4)

Supposedly the ext4 file system can run 6x (or more) faster than ext3 because it requires fewer disk reads and it has better layout optimisation. You would probably see a 30-40% speed increase in boot times, however on a day to day use I'm still sceptical of the actual visible difference between the two file systems.
