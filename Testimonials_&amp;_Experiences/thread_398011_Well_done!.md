### Post by drhiii on 2007-03-31
After years with other distros and having installed a slew of server environments in the past, decided to give Ubuntu a go.   All I can say is WELL DONE!  The logic, ease of installation, the forums and community... well done.   Am sure I will run into gotchas, like video card not recognized on this first test machine hence running generic mode, but so what.  I'll go find the answer or will just ask.

But I can see that on a single CD distro, after the system came up so effortlessly... I am impressed.  I did server install first to see what it was like.  Kewl.  The desktop version.  Kewl.  Wanted to throw a LAMP under the desktop.  Solution easily found and working just fine in a few minutes.  And a few other things I wanted to explore, no problem.  

After years of getting the "can you help me with my machine" distress calls and my advancing cranky response of "go away, especially if you want me to look at "THAT" OS", I think I can see myself saying "here, install this CD, take two asprin, and call me in the morning."  

Well done Ubuntu community.  Well done.  I like it.

Agreed. I am very impressed with this Feisty beta. Apt is just marvelous, and Ubuntu has done a great job pulling the latest software together on the solid Debian base. Installation was very easy--absolutely flawless on my desktop. All my laptop buttons work, too, as does suspend. My only gripe was wireless networking--I did not like the NetworkManager so I just got rid of it. Hopefully that will be improved in the next release. I have looked at previous Ubuntu releases and didn't see reason to switch, but now I have converted both my machines.
