### Post by Nalroff on 2007-12-09
Hi all.

First off, I just want to say that I'm thrilled that there is a potential system out there to send Microsoft running. Unfortunately, the time of Ubuntu triumph is not now. I am a complete noob to Linux as a whole. I did some server-side programming in SQL my freshman year in college, and I enjoyed it. I gave Ubuntu a try seeing several YouTube videos, and enjoyed it. A lot. The GUI on Ubuntu is great, and for a Windows veteran, the file system is not too difficult to get a handle on. My biggest issues are user-friendliness and compatibility. Ubuntu is very pretty, and FAR smoother than Windows X could ever dream of being, but everything out there in the mainstream is written for Windows. I tried Wine, and there are just too many things that wind up being "System Specific" problems that I couldn't stick with it. I have to go back to Windows for now, but seeing the progress this project has made, I have a feeling I'll be back in the near future. I will keep up with the Ubuntu times, and if I can stabilize my system enough, I may even start dual-booting again. I know one thing: I'm never paying for MS Office again as long as OpenOffice.org is around.

Cheers and God Bless,
Nalroff
