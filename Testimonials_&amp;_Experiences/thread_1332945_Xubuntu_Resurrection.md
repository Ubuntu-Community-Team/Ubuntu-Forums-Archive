### Post by 1roxtar on 2009-11-20
I was given an old desktop computer by a friend who thought it was just a piece of junk because it "had so many problems".  First, I'll give you the specs:

Gateway ATXSTF FED
Pentium III 733mhz Processor
384mb RAM
40 Gig HDD
Original OS:  Windows ME, but had a bootleg version of Windows XP installed at the time (reason for most of it's problems).

I decided that I would try installing Xubuntu 9.10 and seeing how that would work.  Amazingly everything was working right out-of-the-box.  I added the Restricted-Extras, installed Ubuntu-Tweak (easier for downloading Chromium browser), the Ubuntu Software Center and ran the updates.  

I  couldn't get over the fact that it was running so snappy.  I watched a DVD with no lagging, viewed online flash videos with no problems.  Sound worked flawlessly, too.  Xubuntu (Karmic) resurrected this "useless" old desktop and made it fun and useful again.  As I have said many times, THANK YOU Canonical, Ubuntu and Linux for making computing exciting again.
:guitar:
