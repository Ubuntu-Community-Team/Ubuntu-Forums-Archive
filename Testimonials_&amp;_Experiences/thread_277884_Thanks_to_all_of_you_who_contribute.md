### Post by JayTee on 2006-10-15
Sorry about posting in the wrong place. I'll try to put posts in their proper category from now on. 
PriceChild, hope you have good luck with Vista. I've been using it since the first beta but I had issues installing it as dual-boot with Ubuntu. I had to create two partitions during the Vista install on a clean disk and then install it to the SECOND partition. I then ran the Ubuntu install and deleted the first partition and hit the back button to get to the option to use the available free space. This worked. MS has changed the boot process in Vista. It no longer uses ntloader so the Ubuntu detection gets messed up. I then had to manually add the entry in my /boot/grub/menu.lst file to have a boot choice for Vista. Alot of work for that. I still think for what MS wants for the upgrade it's not really worth it for a little extra eye candy and more security. Lot of compatibility issues so far. As usual there are many new "features" and the added security reminds me alot of linux where if you make a system change you need to approve it from a popup dialog. Only difference is it doesn't ask you for the root password like Linux does so I think that could easily be exploited. They also changed things so you have to hit the Alt key just to see an application's menu. Duh! and moved everything around to make things more confusing to users who were finally getting used to XP.
