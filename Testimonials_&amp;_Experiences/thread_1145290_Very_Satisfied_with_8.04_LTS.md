### Post by 3rdalbum on 2009-05-02
Ubuntu 8.04 has plenty of bugs - probably about as many as 8.10 or 9.04.

For instance, I'd suggest you plug in your Sansa Clip as soon as possible to make sure that it automatically mounts. There was a regression in 8.04 that causes some MTP-capable devices to not mount.

Nvidia's graphics card drivers have improved dramatically since the version that is available for 8.04. The bugs you can expect include kernel panics, green-looking video, Xorg crashes, and strange artifacts appearing in newly-created windows.

There is nothing about Ubuntu 8.04 that is "stable-as-in-fewer-bugs". Ubuntu 8.04 is stable as in "no-unexpected-software-changes".

I'm not knocking Hardy, it's a good release, and it's great if you want to run your machine for years without having to deal with changes. But please don't say that it has fewer bugs than other versions.
