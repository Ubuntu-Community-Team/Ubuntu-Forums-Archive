### Post by some_random_noob on 2007-01-10
Ok, I've already had Ubuntu for a few weeks so I'm not new to Linux, anyway I thought it would be nice to have XP as well... so now I have both :D

As a comparison, I was pissed off at how many drivers I needed to install with Windows and many "I agree" things I needed to click. I knew all along that my monitor runs best at 1280x1024 at 60hertz but Windows thought I was too stupid to set up the monitor by myself. Instead it ran at the bare minimum of 640x480 until I installed the drivers. 

I also needed a ****** load** more drivers for things like sound and lots of it was 3rd party software which kept nagging me to register online, which is pretty stupid since I have no plans to ever go online with XP! I had enough problems offline - the Help and Support center crashed, and bear in mind this was a fresh install and nothing has gone onto it besides genuine driver disks :mad: ... oh yes and it felt like a 30minute advert break installing XP. The things they promote it for are so unrealistic, guess they never suspected we'd install it after it lost support? "secure" - yeah right! Lots of ads from my dvd drive installation package too.

I can only imagine how horrible it must be to pay for Micro$uck office, XP "Professional" and a firewall. Plus the cost of upgrading those things when they become "obsolete". XP loaded very quickly, faster than Ubuntu. But I guess with a firewall and triple-x downloading in the background it would take a lot longer.

**Why I'm not using XP for the internet**
Firstly because its insecure, even when you're paying $100 a year per computer for a firewall you shouldn't need, I can't even afford to waste money like that in the first place. Another reason why I'm not using XP for the internet: because I don't trust the Windows patches and all of that "calling home" stuff. The patches are unstable and make it worse, plus I can't trust them to kill off non-microsoft software.

**To conclude that, Ubuntu is far easier to maintain and install than XP, and anyone promoting Windows as the OS for home users is AN IDIOT.** Linux is the OS for home users, I don't think that anyone should be afraid to recommend it to someone who is compter illiterate. As someone who knows a lot about computers, I always feel guilty when people use XP - most of them never pay for a firewall or get a free one, if they get one at all! Also you can't get the patches for XP on dialup - and what guarantee do you have that a "computer-noob" will patch their os/firewall anyway?

**Spreading Ubuntu**
I hope that I can get someone using Ubuntu in the future, just set it up, fix the refreshrate and add internet and it works - never need to do anything else! Linux is far better than Windows. I'm going on an i.t. course this year, and I hope to get people into it unless people already are.
