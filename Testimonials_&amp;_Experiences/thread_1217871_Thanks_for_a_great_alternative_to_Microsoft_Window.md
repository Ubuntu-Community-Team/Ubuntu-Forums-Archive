### Post by tomasrey88 on 2009-07-20
I have a windows xp pc that died, so was shopping for a new computer and I didn't want to be pressured into buying a new computer right away without doing some research, so I decided to pull an ancient 10 year old computer out of the garage and installed ubuntu on it. To my surprise, this ancient computer runs like new with ubuntu installed as the operating system. Wow! I had tossed it out into the garage in exasperation when it ran Windows 98.

I wanted to experiment with Linux because Windows Vista takes up too much system resources. An operating system should sit in the background doing nothing and using very little system resources. The application you are using should be the only one taking up system resources. Therefore, Vista is CRAP. So, I waited for Windows 7. Windows 7 is not a one-time fixed purchase fee software. It is a subscription service. WTF?! Can you say corporate greed? Windows can't do the job right the first time, so they'll charge you money to fix it with "updates"?! Can you say, "Corporate Greed?" I don't mind paying for software, but after you pay for it, you should have software that works and you hsouldn't have to pay for them to fix it if it doesn't work. 

Desperate for an alternative, I am testing Linux. So far, so good. My 10 year old computer works like new with this linux operating system. It is also very stable with no crashes so far. Everything is available except for;
1.. a shockwave flash editor
2. video editing software that will import video from an analog webcam. I do not have a DV recorder. it should also burn vcd.
3. my cd rw drive doesn't work but I'm sure my fellow brothers on this forum will be able to help me. 
4. i am still trying to figure out how to install my dell 3110cn printer. 

I think all of these problems are easily solvable except for the shockwave flash editor as I think that a sw flash editor doesn't exist in linux due to it being proprietary code, am I right? If I am wrong, then please tell me where I can find a sw flash editor in linux.

Thanks coding geniuses, Mark Shuttlesworth, and all you helpful fellow forum participants for a wonderful product, ubuntu linux,
:P
