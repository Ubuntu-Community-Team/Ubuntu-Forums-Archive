### Post by Landie_UK on 2008-09-24
Hi just thought I would add my story. I used to use windows 2000 it worked just fine for me did everything I wanted it to do ( read mail,watch youtube vids,surf web,word process). Wifey had upgraded to XP a while back I didn't like it as it "argued" instead of doing as it was told. Well I had heard that MS wasn't going to support 2000 anymore after June 2008, so I looked into Linux.

I chose Ubuntu as it was the distro I had heard talked about the most,I D/L the live cd and took a look at Gutsy( I was going to use some version of Linux anyway as I was not going to use XP). I liked it right from the start worked great with my AMD CPU and onboard VIA peripherals. All was good for 1 month then Wifeys M/Board decides to fail (same sys as I had). OK find Socket "A" M/Board discover it doesn't have onboard graphics card (ooppppssss) ok np use old graphics card for now. Took old M/Board etc out put new 1 in. Won't boot XP says not the same so i'm not booting. took my sys out put in Wifeys so now she got XP running again (after making validation phone call).

Put new M/Board in my PC with old graphics card worked ok just had very low res due to old graphic card. Got new graphic card GeForce 5500 Nvidia copy but sposed to work as Nvidia, D/L drivers for Nvidia still only got low res. Did some research (shoulda done first) got ATI card so now everything working great again.

before I bore you all to tears 1 more thing I did notice I had probs when I upgraded to Hardy, after formatting and doing clean install probs disappeared now it's wonderful and I even got Compiz working. Only thing that doesn't work for me is Google Earth but I can live without it.
