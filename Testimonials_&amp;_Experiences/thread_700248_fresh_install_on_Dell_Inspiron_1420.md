### Post by weird_c00kie on 2008-02-18
Today I did a fresh install of Ubuntu 7.10 desktop i386 on my Dell Inspiron 1420. I was expecting a fair amount of trouble, because when I first got the laptop it came infected with Vista which wouldn't let me resize its partition so I could dual-boot with Ubuntu. In the end I resized the partition through Ubuntu's Live CD (High 5 for Ubuntu!) and installed it. Fortunately (yes, fortunately) the resizing corrupted the Vista partition so I could no longer boot into it.
So I thought it was time to wipe the slate clean and reinstall Ubuntu. Last time though I had a fair bit of trouble. See my old thread for more details: [http://ubuntuforums.org/showthread.php?t=520189](http://ubuntuforums.org/showthread.php?t=520189)

This time though... it was a near-orgasmic experience. About half an hour after I booted into the Live CD the installation was finishing (take that you evil Windows installations!). Once I restarted I was in heaven. Ubuntu had already configured my video, sound and network devices. My display was set to a beautifully crisp native 1280 * 800 resolution, the boot-up chime was pouring out of the speakers and the network manager was already asking for the passkey to my wireless home network.

A short while later it kindly asked for about 280mb worth of updates and it was all done. The only thing I had to do manually was to add the extra repositories as described in the [Ubuntu Guide]("http://ubuntuguide.org/wiki/Ubuntu:Gutsy#How_to_add_extra_repositories") website.

So that's it. In less than 1 hour, I had a clean installation of Ubuntu, my graphics, sound and network were configured, I had Firefox running (and Opera downloading) and Open Office was ready to welcome the new uni year.


By that token, I owe a HUGE thanks to Canonical for all their awesome work. I was expecting to spend a whole day setting up the computer and fixing whatever problems arose, but there were no problems whatsoever.
