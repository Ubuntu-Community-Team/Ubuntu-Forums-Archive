### Post by jamie0nz on 2009-11-04
Oh for the days of UMSDOS filesystems, roll your own kernels (0.9), compiling the app's yourself and sorting dependencies manually. Where has Ubuntu gone wrong?

I still treat a new OS with CARE. I have two OS partitions, one for the old one, one for the new one. I have a data partition and a swap partition. I recommend the procedure to anyone. Upgrade ... ha, I used to format C: on a Windows box annually just to keep the thing running well.

So for me, 9.10 was a clean install (as was 9.04). I also made the 32 to 64 bit jump.

Finally Pulseaudio works properly for me. Apps doing sound in Java no longer break the music. The new Skype 64 bit beta works a treat. Bluetooth features that never worked are superb.

I've also gone VMWare 7 Workstation. I am hoping the screen and X crash issue I have experienced in 9.04 is gone ... we will see. However for me ... 9.10 is full of outstanding improvements.

Nice!
