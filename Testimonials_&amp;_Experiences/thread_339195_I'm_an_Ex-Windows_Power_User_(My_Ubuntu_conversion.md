### Post by FuturePilot on 2007-01-15
Yes, you heard me right. I read in another thread here that you shouldn't try and convert a Windows Power User to Linux. I guess I'm an exception to that.  About 2 years ago I started getting an interest in computers. I was curious as to how programs worked, all the stuff the computer did that you couldn't see, how they were executed, how the OS worked and so on. I was eager to learn and learn I did. Most of it I learned on my own through experience and Google-ing. I got to know Windows very well. I've edited to registry on numerous occasions for various reasons, even to allow myself to rename the Recycle Bin. I've even found a way to theme everything without the use of 3rd party programs. I was always testing out Beta programs. I had the first Beta version of IE7 like the day it came out. I got to know Windows so well that at one point I actually liked it. I look back now and think that I must have been on drugs or something:p . In fact I actually preferred IE7 over Firefox. (Those must have been some pretty strong drugs:p ) I tried Firefox but I uninstalled it because I thought it was ugly and I couldn't stand the way it was laid out. (Don't do drugs, they make you think crazy stuff:p ) Little did I know Firefox is theme able. I ended up reinstalling Firefox for some reason, but I didn't use it much. Slowly though, I started using it more and more. I started to embrace Firefox. I installed various themes and extensions. I then realized what a powerful browser it actually was. Eventually I wasn't using IE7 at all. I only use Firefox now.

Every so often I would hear something about Linux. I knew Linux was an OS, but never knew what it was all about and I though you had to be like a genius to use it. I saw a screenshot of a Linux desktop once and it had gray windows and buttons and stuff, and I thought who would want to use such and ugly thing. Maybe it was my eagerness to learn about new stuff or maybe it was the start of my dissatisfaction with Windows. So I started reading stuff about Linux because I was curious. I found myself at linux.org reading stuff about Linux.  I thought it would be cool to try Linux out one day. The chance came when I found an old 30GB hard drive for my laptop. It still worked fine and I already had a 100GB one in my laptop. I checked to make sure there was nothing on there that was important. So then I figured hey, why not try out Linux, I've got a hard drive that I really don't care what happens to it. So I go looking for a distro to download. I found one that I though would be a good one. So I did what any life time Windows user would do, I double clicked on the .iso. After some time searching Google I found out what I was supposed to do with this file. I installed it on my laptop but I had some sound issues with it and I didn't really like the way it was set up. Now the average Windows user would probably give up here and go back to Windows. Not me. I knew there were hundreds to flavors of Linux. One had to be right for me.

So I went back searching and that's when I found Ubuntu. I kept hearing really positive comments about Ubuntu and how good it was. So I though I'd give it a try. It was nice because Dapper hadn't even been out a month yet. So I downloaded Ubuntu Dapper and installed it. Right away I knew there was something special about this distro. I like the way it had a clean look to it and all the programs were nice and categorized. I played around with it for a few days. Yes it did feel weird coming from Windows. But I liked the whole philosophy behind open source and Linux that I was willing to stick with it. I really really wanted to learn Ubuntu, learn all the ins and outs, and get to know it like the back of my hand. I didn't give up. I was able to solve my mp3 playback problem by searching the forums here. I got everything set up and running. I really liked the idea that I had a powerful OS and software that was completely free and that you could edit the source code of programs and even redistribute them. Coming from Windows, that made me feel like I could do what ever I wanted to. No dumb M$ EULAs and restrictions.

Linux is definitely not hard to learn. It just takes some time and patients. And it's definitely harder to mess anything up than in Windows. I am still and running my first install of Ubuntu on my laptop. Now I've also got on my desktop. I am now using Ubuntu 95% of the time and I totally hate Windows now. Too many problems, don't like their views etc, but that's another discussion. Ubuntu is faster, more stable, and more powerful than Windows. I hope one day that I can be completely Windows free.

I would like to give a huge Thank You to all the developers and the people that have contributed to the Ubuntu project and have made it possible for a person like me to discover such a wonderful OS.

Thank You Ubuntu!
