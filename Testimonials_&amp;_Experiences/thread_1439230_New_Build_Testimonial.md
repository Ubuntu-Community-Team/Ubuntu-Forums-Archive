### Post by mullinsn2000 on 2010-03-25
This is for all of the new users saying how convenient and easy Windows Install are compared to Ubuntu.
 
Here are the specs of the build I just did today:
 
Gigabyte Motherboard w/ on-board audio and on-board ATI graphics
AMD Phenom II X4 965 3.4GHz
2x2GB G.Skill DDR3 1333 RAM
WD Caviar Green 1.5TB HDD
2x Samsung DVD Burner (forget model, Maximum PC Best DVD Burner)
SMC Wireless Card
 
Brand new parts, just put together with clean installs of Windows 7 and Ubuntu 9.04, both 64-bit.
 
Windows install took over an hour.  Had to install Open Office, anti-virus, malwarebytes, install printer, install Abobe Flash player.  The wireless card did not automatically install drivers like Microsoft claims happens.  Drivers disc did not contain drivers.  Had to go to website to locate drivers. Had to download updates for Windows, which took another 30 minutes or more.
 
Ubuntu install took about 25 minutes, including the repartioning of the HDD.  Open Office already installed.  Plugged printer in and it just worked.  Nothing needed to be installed.  Wireless card just worked, nothing had to be installed.  On-board ATI graphics worked perfectly out of the box.  No anti virus needed.  No malwarebytes needed.  I still need to install Adobe Flash player but I highly doubt that will be a problem.  OS updates took about 10-15 minutes.
 
 
This is for all the people that are complaining that Ubuntu is not Windows.  It is not and I am glad.  To have the entire system up and running with Ubuntu took about a third of the time as it did with Windows.  People new to Ubuntu need to give it a chance before saying that Microsoft is easier.  It is not easier, just different and more familiar.  People do not remember going from the 5 1/4" floppies using an old Macintosh system to using Windows 3.1 or 95.  That was something that was different and hard to get used to.
 
Guve Ubuntu (and all open source software) a chance before saying that it is too difficult.

### Post by fritzman on 2010-03-28
I keep a virtual installation of WinXP on my Linux box just for the annual pilgrimage to tax-land.  I haven't been able to get any of the tax prep software to work on wine, and I'm completely averse to doing taxes on-line, no matter how much it is claimed to be completely safe.  I'll do it on my own box, thanks.  So, I've kept that installation of WinXP on my box through several versions of Fedora, a couple versions of Mandriva, and now two versions of Ubuntu, currently staying with 9.04.  Odd, but as a virtual machine, XP doesn't crash like it did running native.
This week-end, I downloaded the win7.iso, just for fun, and installed it using Sun VirtualBox.  It was just under 40 minutes to install the basic system, another couple of hours to install Clamav for Windows, and Spybot, and Quicken 2009, and OpenOffice.  So far, Win7 looks to be the least awful I've seen from MS in a long time.  Don't know how it will work running native.  Haven't decided if I really want to give up a partition for it.
One cool thing about win7 installed from the downloaded .iso, it didn't demand a license to complete the install, and I can run it for a month before it will refuse to boot.
Maybe I'll just hang on to the installer DVD I burned, and install it once a year for the tax season, and then let it expire.
Anyway, as my signature might indicate to some, I'm a Linux guy for the duration.
