### Post by indijaal on 2007-11-28
Of late I had been using Windows XP for all my Java Development and found it to be pretty stable and fast. With good support provided by Sun for Java on Windows, I never thought of switching to some other OS. However, with release of Windows Vista, I upgraded and moved all my development to Vista and was really shocked by its overall performance. My productivity decreased by miles. 

Since then, I had been using Ubuntu for all my Java Development. Its fast, easy and stable. Not to say, Sun too has made some major improvements in Java (especially on desktop) in order to support Linux. Also Java is now supported by Ubuntu as well. I have discussed more on my [**[COLOR="Blue"]blog[/COLOR]**]("http://arpitsblog.blogspot.com")  at [**[COLOR="Blue"]http://arpitsblog.blogspot.com[/COLOR]**]("http://arpitsblog.blogspot.com"), if anyone of you is interested. There is also a poll where developers worldwide are sharing details of OS they are using.

Anyways, Which OS are you using for your java developments?
