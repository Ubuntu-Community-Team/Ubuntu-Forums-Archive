### Post by N00b-un-2 on 2008-09-28
I've been trying in vain to get all the little quirks banged out of my Ubuntu installation, and try as I may I still can't seem to get everything just right to where I can justify deleting my windows partition.  I've tried Wine, and as far as I'm concerned Wine is not worth the time or effort.  I've been dual booting since I installed Ubuntu because I've had one thing after another that has affected my ability to do what I'm trying to do on my computers.  Be it the wifi card on my laptop, the runaround I had with my Lexmark printer (still can't use the scanner...), or all the goofy little ins and outs of linux sound in it's myriad forms (I write/edit a lot of music and I've used Adobe Audition along with Fruityloops for years).
I've been able to remedy the problems I've run in to so far pretty successfully.  I was able to get my Atheros wifi card to work, and work WELL on my laptop.  I was able to get my printer to work despite being told flat out that my Lexmark X1150 is not supported by linux.  I installed Envy which fixed the problems I was encountering with my nVidia graphics card.  I was able to figure out Samba well enough to get my computers to network properly.  Sure, I can't play most of my windows games in Linux, (Wine is virtually useless when it comes to emulating games, and Cedega just isn't worth forking out the cash for) but I've foudn that there are plenty of excellent games out there specifically for Linux, like Urban terror for example.
But the ONE thing that is still keeping that other partition on my hard drive is still the fact that you can't WEB CAM WITH SOUND in Linux.  The best I've been able to do so far is to install Windows XP inside Ubuntu with Virtual Box and go through the dog and pony show it takes to get USB to work with Vbox.  And then the sound is SO choppy that it's not even worth using and I'm better off just booting into windows whenever i want to web cam with someone.  I'll tinker with the settings to see if I can get it any better, but I'm not holding my breath.  When will amsn support USB microphones built into USB web cams, and when will it support bidirectional audio/video?
