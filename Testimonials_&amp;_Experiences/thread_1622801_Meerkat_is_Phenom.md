### Post by ventrical on 2010-11-15
Eh,eh...Wow... I had a problem with one of my PCs .. a Sony Vaio. I had taken one of the DDR mem sticks in to see if I could get some more match memory. I ended up getting some memory for my Acer Aspire 3620. The Sony had Ubuntu 10.10 Maveric install on the HDD. The memory stick went belly up and the PC started malfunctioning so I took the 40GB HDD out and slipped it into the Acer Aspire. (I thought the whole thing would go belly up!!) Well it didn't !! In fact it works better on the Acer 1.5GHz than it did on the Sony 2.4GHz!!

But not only this .... I am now able to use some sort of built in assistive technology that allows me to magnify the screen when i am typing by pressing the <Windows-logo>key and this little <up>down>left>right> key between the left clicker and the right clicker on the touch-pad mouse!! And NOT ONLY THIS but as I move the mouse pointer the whole screen navigates (without a glitch) without having to key in or key out (which I could never, never  get working in Windows !!!)

Ubuntu has got it right!

Great work !! and thanks.
