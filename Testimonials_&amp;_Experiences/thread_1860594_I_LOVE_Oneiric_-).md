### Post by Dangertux on 2011-10-14
No seriously, despite the Unity hate going on, I really do like this release a lot. 

Being realistic about the whole thing, Unity was new it has come a long way and looks a lot cleaner than it did in 11.04.

My wife has been using it since beta2 about a week prior to it actually being released, she did an upgrade this morning to the release, and it's working spectacularly. Minus some Atheros driver issues, but that's always been the case and same fix in every version.

She loves the interface, being newer to Linux and Ubuntu in general she found it very trendy and interesting.

I am not a huge critic of the interface one way or the other, other than I really love the dash, I think it looks amazing, fantastic job.  What I really like about 11.10 is more under the hood though. I like the fact of the newer kernel and much needed newer libraries (ruby in particular). I like the fact that native support for Xen has been re-added, and that I no longer have to spend 6 hours recompiling a kernel if I want to use Ubuntu server as my host OS.  In terms of being a guest Oneiric has done well for me in all environments I've used it thus far. Xen DomU , VMware Player/WS and Virtual box. 

Overall I think it was a really good release. Though I can see why it's not for everyone. It's a step in the right direction. To be honest if having to choose between Gnome 3 and Unity, I prefer Unity. Great job to all the developers involved on this thanks for your hard work :-)

If I could have one wish for a feature for the desktop version it would be something for Apparmor similar to YaST in SuSE. Apparmor is a great tool that I feel is made entirely too inaccessible to the average desktop user.

### Post by 3rdalbum on 2011-10-15
I like Oneiric too, but if you blink you'll miss the changes.

Sure, it's running on Gnome 3 now with Gnome 3 utilities. These aren't any better or worse than Gnome 2 utilities. There are small tweaks and a visual enhancement to Unity. The login screen is now powered by LightDM.

There's not ENOUGH change. I want Unity to look stunning and super-modern. I want everything conveniently placed and a couple of clicks away at most. I want the desktop to work smarter to provide flawless search and features that I could never live without.

Oneiric looks nice, but not stunning. A few things are more clicks away than before. There are few new features, and it doesn't seem to provide better search. A little disappointing. They haven't even given Update Manager a "mono" icon, and in fact I think it's still using the Notification Area API rather than the Indicators API.
