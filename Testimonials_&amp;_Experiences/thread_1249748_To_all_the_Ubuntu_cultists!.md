### Post by Methuselah on 2009-08-25
Open source software is not about cost, it is about freedom.
The reason why linux is as successful as it is is because it is a free OS not a no-charge OS.
Very few monolithic companies could have hoped to establish linux in the number of niches it is currently established in while keeping it closed source and proprietary.
From supercomputers to cellphones to being embedded in your bios and from Hollywood movie studios to home offices, and from x86 to MIPS to ARM to PPC, linux is there. 

So you can't really separate linux from the concept of open source because that's why it is where it is today and it influences everything about it including it's technical properties.
The linux desktop and associated programs are based on open formats so there is no proprietary lock-in of your creative efforts.
It took me a while to become unsatisfied with lock-in but experiences eventually taught me to be.

### Post by Thanh-BKK on 2009-08-25
Hi.

Why could you (OP) nit just be happy with using Windows (whatever version) for your business and have private users like myself run Ubuntu (or any other Linux for that matter) without too much headaches?

I have been using Ubuntu 8.04 since the day it was released, hard switch (Vista off, Ubuntu on, never dual-booted) and while i sure don't create art work or do any other form of commercial things on my Ubuntu machine i DO edit videos (and quite a lot, it is a hobby) and have no problems doing so with all-free software that can be downloaded through Ubuntu's repositories and a few other sources. 

Why did i switch? Well i WAS happy with Vista (and XP before it), didn't experience any crashes (seriously, no BSOD on my own machine since like 2002 or so when i first installed XP) however i DID experience the fact that commercial software for Windows did get larger and larger while offering just the same functionality (yet more and more useless eye-candy and extra functions crammed in... talk about an 800 Megabyte printer "driver"!) and the associated need for faster and faster hardware all the time. 

Then also the fact that the absolutely required security software which was running all the time took more processing power than any productive software (anti-virus, anti-spyware, anti-hacker, anti-whatnot) and the permanent need to keep all of these updated, which for the first ten minutes each day made me unable to use the internet as everything started updating itself at once...... 

And one day i got that funny idea to put MY computer back into MY control. Ubuntu allows me to do just that, and it has been running pretty much flawlessly from day one.

Yes, there are people who have problems with Ubuntu installing or running. But there are also people who have identical problems with Windows, believe it or not. 

And no, i am NOT a geek and although i have learned meantime to use at least some of the command line things i still prefer to do things with the mouse, i am just a plain user. 

Do i still use Windows? Yes, all day. At my office, that is. Yet by now my machine there is one of only two that remains on Windows, all others have been changed to Ubuntu (we are talking 18 machines) and every single one employee loves it. Why not mine? Because some mission-critical software is not (yet) available for Linux and here we are back at the beginning - businesses shall run Windows if it is required, Windows is a good OS too. But my company saves money by not needing to keep 20 commercial anti-virus suites up-to-date and buying ever newer versions of MS Office to stay compatible with files sent from clients.

Oh, and on my Ubuntu box, safely tucked away in the virtual world of VMWare, are no less than three full Windows installs.... XP Home (genuine!), Win 95 (genuine!) and, just for fun, WfW 3.11 (downloaded, but i don't think MS would still care if that gets pirated). I hope that some day actual computers would work like virtual machines - you just destroyed your OS with the latest "MegaBlaster" virus or a truly dodgy device driver? One single click and you are back on in seconds....... and it would even be possible, just use Ubuntu and a VMWare install as "BIOS" and an extra HDD for the snapshots.... *dreaming of the ideal computer*

Best regards.....

Thanh
