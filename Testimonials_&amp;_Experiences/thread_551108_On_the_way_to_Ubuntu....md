### Post by k8thegr8 on 2007-09-14
This post is a diary of my journey of going open source all the way. I've been using open source software for a while now, but haven't taken the big leap to Linux yet. My journey begins with a clean machine, not a single preloaded program on it. Eventually, I will completely replace my current OS with Ubuntu.

At the moment, my boyfriend is piecing together a this new machine. There are some new parts, some old parts and some old parts that look like new. This machine is a clean slate. so to speak. Ubuntu will be all it will ever know.

This may end up being a weekend project. I'll keep you posted!
