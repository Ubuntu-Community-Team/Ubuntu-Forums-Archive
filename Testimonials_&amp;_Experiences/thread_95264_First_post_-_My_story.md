### Post by ceeter on 2005-11-26
Hello everyone,

First off let me say this forum is fantastic, a wealth of great information. A month ago my brother gave me a Ubuntu live cd and told me to try it out. I have used Knoppix and Damn Small Linux live cd's in the past and wasn't very impressed with them, Ubuntu blew me away. I played with the live CD for a few days and decided to install it.

I deleted my windows installation and instaled Ubuntu, I had a few problems installing due to corrupted CD's but after that everything went fine. Ubuntu detected all my hardware except for my ancient RAID card. After installation every question that I had was already answered in great detail here, thats why this is my first post. Everything from enabling TV-out, extra mouse buttons, editing fstab so my slave drive is mounted at boot, mounting my ntfs hard drives, etc.

Next up is installing Ubuntu on my file server and getting them networked. Windows free for four weeks and loving it. The only complaint that I haven't found a solution to is a replacement for Zoom Player (OSD, on screen file browser) and getting my ATI Remote Wonder to work properly.

Thanks again to everyone for the great how-to's and detailed posts helping us that are new to Linux. (sorry for the long post)
