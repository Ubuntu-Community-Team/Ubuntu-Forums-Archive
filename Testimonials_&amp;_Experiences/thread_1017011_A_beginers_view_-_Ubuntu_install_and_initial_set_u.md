### Post by SerpicoUK on 2008-12-20
Hi all,

Just thought I'd give my impressions of installing and using Ubuntu 8.10 (first week):

Having no real linux experience I was surprised by how easy everything was. On a new Sony Vaio laptop everything installed and worked out of the box. Including the wifi which was a relief.

Adding and removing programs was very simple. Using flash/removable drives - no problem. In fact, the only thing that doesn't work properly is the headphones which don't mute the front speakers when in use.

DVD play back initially suffered from black screen flickers but a search of these forums resolved that issue by amending the output.

One thing that caught me out at the start and I suspect quite a few beginers:
Ubuntu recognised that I had an ATI card and a pop-up let me know 3rd party drivers were available. When I tried to activate them a few process boxes flashed up but nothing really happened.

The reason was that I hadn't ticked the 3rd party repositories - for Windows(tm) refugees this is a very alien concept. A prompt on the original pop-up would have made things smoother.

Also, compiz should really come pre-packaged with Ubuntu. Until I read around the internet I had no idea what it was. But after reading a guide to installing and setting it up I now have a theme and desktop effects just to my liking. 

I also really like the fact that there are so many how-to's on the web and on this forum, most are easy to follow. It has allowed a novice to set up a fully functioning virtual drive (using virtualbox) with WinXP/office 2007 on it. I hope I won't have to use it but I just wanted to see if I could do it.

I've since installed Ubuntu 8.10 on my old Dell inspiron 6000 for my folks. It wasn't quite as easy due to the desktop being almost unusable until I installed the ati drivers - it suffered from desktop distortion that made reading windows/menus almost impossible. Had I not just set up my laptop I don't think I would have been able to install the ati drivers!

So, in terms of installing and initial set up on new hardware I think Ubuntu is almost there [Sony vaio SR - 9.5/10]. On older systems I suspect it may throw a few more curve balls than you'd get with Windows(tm) [Dell inspiron 6000 - 1/10].

Having scratch the surface, I like what I see. What do I need Windows(tm) for? Just games?...May be Ubuntu on my PC and a PS3...

Serp
