### Post by statikeffeck on 2008-10-06
... you keep wanting more and more until something doesn't work!

My GNU/Linux History:
I first wanted to try linux around 1998 or 1999, when my computer at the time was a blue iMac (300 mhz G3). I was looking into "Yellow dog linux" which ran on G3 processors, and ordered a CD-ROM of it (since I was on dialup at the time, and had no CD burner). When I got the CD I realized that I would have to format my computer and I didn't have any way of backing up my data, so I never got to try it.

Fast forward to 2001, my first semester at college. I met this cool guy in my calculus class who did some linux administration and knew a lot about it. He drove a nice jaguar 2 seater and somehow was allowed to have it on campus. I asked for some assistance and he got Slackware installed on my athlon 600 mhz machine in a dual boot setup (with windows 2000). I had a graphical desktop, but i was finding things didn't work -- my extra mouse buttons for browsing, sound problems, to name a few. So I eventually gave up on it.

Maybe 2-3 years later I wanted it again so I looked into VectorLinux this time. It seem like it was user friendly and had more support, so I installed it. I don't exactly remember the reasons but I wasn't too happy with it and eventually stopped using this, too.
A little later I was looking into setting up an inexpensive server (just for file sharing) for my mom's office, which used all Mac. I figured I could use Linux to do it. So I was researching appletalk support on Linux and attempted to get it to work with a Debian system. I liked the Debian package manager and additional hardware support, but couldn't really work out the appletalk file sharing, so I gave up on it.

Around 2006 I heard guys talking about Ubuntu, saying how great it is and the rate at which it is improving, so I researched it. I loved the community & south african concept of Ubuntu so I burned an ISO (Ubuntu 6.06) and installed (dual boot with XP). I loved it. Sound and graphics support was excellent and with ubuntuforums.org I was able to fix a lot of things. I did some programming in it, too. Still, I eventually found things that just didn't work right, and closed-source hardware or software was always the root of the problem.

Now I am once again using it (I always kept Ubuntu on my hard drive, just didn't use it much). The only things I really want Windows for is Games like Counter Strike Source, Call of Duty 4, and MAME. I'm trying to get more versed in using WINE and Virtualization so I won't need to boot into Windows anymore. 

One thing I realized (and this is where I explain the topic title) is that when I use Ubuntu, I'm happy about things that work but I keep trying to do more and more stuff (greedy!) until something doesn't work. Then I get discouraged... 
I know Linux has great games, like Quake, Unreal, and id software (thank you john carmac), so maybe I should just only play games that support linux.

So, really, what I am trying to say is that maybe my mindset must change, and not the system?
