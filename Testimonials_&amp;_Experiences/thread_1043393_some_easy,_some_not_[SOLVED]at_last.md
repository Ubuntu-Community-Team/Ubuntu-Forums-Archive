### Post by armandh on 2009-01-18
printers ..... OH [expletive deleted] 
some, like my old HP laser jet 5, are easy to do the network setup.
some are not easy at all [still]

one of the last few reasons I still have an XP desktop is an old printer that is cheap to operate, provides great quality, 2 side printing, and is built to out last my life.

while I had succeeded getting it to run from the parallel port under ubuntu I had no luck with the network, and xp did work over the network. [with old 2K beta drivers]

I should note support for this printer [DDS-32] seems to have ended contemporaneously with support for studebaker and packard.

a few hints from the local users group and I thought I would give it another try. after several more failures I tried 

system>administration>printer>new 
and under LPD/LPR Host, the IP address, under Queue, postscript
then the driver I had working with the LP1 [hatachi] WOW it works

now if intuit would port quickbooks to linux I would have little reason to keep an xp desktop.
