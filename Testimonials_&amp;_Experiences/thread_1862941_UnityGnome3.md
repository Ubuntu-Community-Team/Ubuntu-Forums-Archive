### Post by 3rdalbum on 2011-10-18
> **unixcrab said:**
> I have never had a reason to get angry with Ubuntu before but now that 11.10 has removed gnome classic and forcing me to use either unity or gnome shell

Unity is definitely incomplete, but it's not especially buggy.

Ubuntu is not "forcing" you to use either Unity or Gnome Shell.

Unity comes as default. You can install Gnome Shell, or Gnome Fallback Session (which might be your piece of cake), KDE, XFCE, LXDE, Window Maker, Englightenment, etc. All these are available.

It was not possible to retain Gnome 2. It's no longer maintained by its developers and can't co-exist with Gnome 3 on the same system. If Ubuntu had stuck with Gnome 2, the available software for it would have become out-of-date very quickly. A transition was necessary.

Some people find they like/love Unity after using for a week. If you really can't bring yourself to try it for that long, then just install the "gnome-session-fallback" package and choose it at the login screen. Or install XFCE. After all, Unity and Gnome Shell are two additional choices to all the other choices you had before.
