### Post by gobberpooper on 2010-05-04
I've been using Ubuntu for the last few releases and have been very pleased with it. So obviously I've recommended it to a lot of friends, to my parents, and schoolmates etc. Now even my computer teacher is trying to get Ubuntu PC's, but the IT admin guy is too heavily pro-Mac.

But almost all of them have stopped using it for one of two reasons:
1) Lack of games
2) Trouble installing applications

Yes, I know that the games problem is more of a commercial developer issue and a non-official Wine issue. But it's still a huge problem. And of course the only way to really fix it is to get more people to use Ubuntu, all at the same time. Like proper advertising. Hulu, TV, websites, whatever. If a lot of people use at the same time and they stop using it after a few months, it will definitely get the interest of game developers, and then the domino effect comes in to play...on Ubuntu. Sorry bad pun.

People like a good interface, one that looks cool but not annoying, but it's still easy to use. For the most part, Ubuntu has a good interface. I go to the software center. Awesome. Every piece of good, useful, up-to-date software is all here and for free. I click through all the little buttons, easily find what I need to. And then I install. All of a sudden I'm thrown through this mad flashing of code and CLI and nerdy stuff that I don't care about and I don't even get to pick my features. No "Please select the version you'd like." No nice little buttons I can click through or images that give each installation its flavor. And then it's done and if it doesn't work, well then I have to go through the CLI and figure stuff out, and no normal person wants to do that.

The CLI should still be a major part of it, unlike that other random OS which eliminated its actual CLI. But nobody should have to go through the CLI for anything except the advanced.
