### Post by mostwanted on 2006-06-08
I'm glad for you ;)

Auto-detection of removable storage is not bad at all in Ubuntu. That is just my experience (and your's apparently), I know other people are having problems.

When I plug in my camera, I've set Gnome to automatically open F-spot in the import state so I can add my photos to it (I prefer F-spot to the other photo apps). Just go to "System &#8594; Preferences &#8594; Removable drives and media", select the camera tab, make sure the import camera option is ticked and put this command in the text-field:

```
f-spot --import %m
```

That will open up f-spot in the import window ready to import the card/camera you've plugged in if you choose to import the photos like Gnome asks you.
