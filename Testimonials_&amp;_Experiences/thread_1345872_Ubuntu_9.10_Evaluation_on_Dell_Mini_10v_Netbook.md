### Post by dmengo on 2009-12-04
[COLOR=black][FONT=Verdana]I installed Ubuntu 9.10 on a recently purchased Dell Mini 10v netbook. Works great so far.  Had to do some tweaking to get the wireless to work correctly, but other than that all of the hardware including the card reader worked out-of-the-box.

Overall Ubuntu doesn't seem to be as polished or refined as Windows or Mac operating systems, but the OS works well enough. I'm impressed that when idle the system only uses between 170-200 MB of RAM, while Windows 7 on the same hardware was using around 350-400 MB of RAM when idle.  Ubuntu 9.10 seems about as fast and responsive as Windows 7 was on the same netbook.  I only experienced one kernel crash when fixing the wireless driver.  Ubuntu 9.10 seems stable and rock-solid.

I've always been a Windows and Mac support person throughout my IT career and will probably never give up my Windows desktops entirely. Almost all of my support work these days deals almost exclusively with Windows servers, desktops, and virtual applications.  With that said, I'm getting ready to start studying and learning Linux system administration to be more “well-rounded” in my field. I'm also considering studying for the LPI and Linux+ certifications. I certainly don't want to be a "one trick pony."[/FONT][/COLOR]
[COLOR=black][FONT=Verdana] [/FONT][/COLOR]
[COLOR=black][FONT=Verdana]Overall, I'm still not completely sold on recommending Linux for the average person at home or "non-technical" user.  Linux certainly has its place in the server world, but I'm just not convinced that it will ever gain dominance on the desktop.[/FONT][/COLOR]
