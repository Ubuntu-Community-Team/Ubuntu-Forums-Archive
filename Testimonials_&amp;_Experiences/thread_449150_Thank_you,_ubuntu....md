### Post by cbmeeks on 2007-05-19
I'll keep my introduction short.  I've been a linux fan since the early days of Slackware.  I "grew up" and moved to Windows NT, Red Hat, OpenBSD, etc.  I am a huge fan of FreeBSD that I used as a server for years.

This weekend, I decided to format my Windows 2000 server and put ubuntu 7 on.  This Pentium III 450 Mhz machine runs it beautifully (same machine that ran FreeBSD for years).

Anyway, I am a programmer at heart and I've been learning MySQL, PHP and Python (.NET programmer by day).

I installed ubuntu and chose LAMP.  Bam.  Instant web server with PHP, MySQL and Apache.

Python was also already installed.  (I searched for 30 minutes on a QUICK way to install LAMP and Python on FreeBSD to no avail).

But, I ran into a problem.  The basic LAMP did not include mod_python.  Great, here comes tons of research and compiling I thought.  A quick Google search and I found these forums and someone that told me how to install it in about 2-3 lines  (apt-get was involved).

Worked the first time!

Then, I realized that python didn't have a MySQL adapter.  Again, Google points me here and the one line.  Now my python programs work with MySQL.

This is great.  Now, I need to install and configure Samba.  More compiling and config....nope!  Google and this forum to the rescue.  One line and Sambe is running on my server.  10 seconds later, I'm mapping drives.

So, I wanted to thank ubuntu and the people behind it.  In less than an hour, I have LAMP with Python, MySQL, mod_python, SMB and more.

Thank you.  Thank you.

cbmeeks
