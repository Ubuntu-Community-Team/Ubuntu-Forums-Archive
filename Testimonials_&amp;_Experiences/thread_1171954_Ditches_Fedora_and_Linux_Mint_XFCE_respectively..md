### Post by Crunchy the Headcrab on 2009-08-08
There are two distros that I love no matter what.  Fedora and Ubuntu.  I find that I use less ram using Fedora 11.  Load-up times might be a little longer, but then again I'm not sure because I can't tell a big difference.  Leonidas has newer kernel which I need because the one Jaunty is currently using has issues with my machine (don't know why, previous ones didn't and later ones don't appear to).  I prefer Yum to apt-get.  However, I like the broad range of programs available as .deb files.  I find the 2 OS's quite comparable.  For now I give the edge to Fedora simply because it works on my system ootb.  I also like being able to choose program by program what is installed from the standard install dvd/cd's.  I'm going to try Karmic Koala when it comes out.

I absolutely prefer the ubuntu community.  It's full of the nicest, most helpful people I've ever met on a forum.
