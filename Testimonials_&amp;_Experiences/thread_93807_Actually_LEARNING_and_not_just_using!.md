### Post by Gadren on 2005-11-22
I've been enjoying my Ubuntu 5.10 for quite a while, and I've really enjoyed the fact that it doesn't just set everything up for me.  Distros like MEPIS do most of everything, which may sound like a good thing, but I don't really learn anything from using MEPIS.  On the other end of the spectrum, there are distros like Gentoo and Debian that are so difficult to configure and install that a newbie like me is quickly frustrated.

Not so with Ubuntu.  Yes, multimedia support wasn't included, but by working to get it added to my installation (even just copying and pasting the terminal code), I feel like I'm actively learning pieces of the Linux tapestry.

Today was my first true test of this.  I booted up to find my X system not working...I had no idea was what going on, and was about to go onto these forums for help, but then I looked at the actual error message...

"Hmm... /etc/X11/xorg.conf looks familiar..."
"Error in 'DisplaySize'?  Oh, that's right!  I edited that... I must have done it wrong."

So, using the command-line, I was able to login, do a sudo su, and then use nano to edit the xorg.conf file and fix it.  And I rebooted, and voila!  It worked again...

While it's a small, simple thing, it truly increased my faith in my own ability to use Linux.
