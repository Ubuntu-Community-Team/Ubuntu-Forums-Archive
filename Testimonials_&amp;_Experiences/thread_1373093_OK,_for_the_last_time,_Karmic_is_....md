### Post by Methuselah on 2010-01-05
Karmic is the fastest Ubuntu release I have ever installed.
Imagine my shock when I installed it on an old 800Mhz PIII from circa 1998 and have it respond more readily to input than when it was running Hardy Heron.

Karmic had some problems with another machine 
(the source of which I had tracked down and would have added to that thread if it were not locked) 
but what it did on the older machine was impressive.
I was thinking of setting up Xubuntu on it but honestly, there is no need at all because the full featured gnome desktop is completely usable.

There might be certain tricky hardware configurations so it will still be wise to test before upgrading.
However, if Karmic likes your hardware, it is a better experience that previous releases.

Since I have no real use for the machine, I'm planning to use it for Lucid Lynx testing.
It'll be my way of putting my time and effort where my mouth is, so to speak, to contribute to making the upcoming LTS release better.

Keep chugging Ubuntu!
