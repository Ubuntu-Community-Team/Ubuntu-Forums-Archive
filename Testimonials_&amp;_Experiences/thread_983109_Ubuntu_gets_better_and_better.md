### Post by bd@cb8be8510 on 2008-11-15
I started with version 6.10. Major problem : no wireless network. Solution : download the serial-monkey-drivers. Each Ubuntu and linux-kernel-update, I encountered the same problem : no wireless network. So each time I did an update, I first downloaded the latest driver-sources in order to anticipate the network-breakdown.

Due to a disc-failure I had to install a fresh Ubuntu 8.10. After installation, the networkmanager detected the networks around. I selected the right one and everything went smooth! This is the first release which works right out of the box! Well done, guys! :KS:KS :popcorn: 

I installed back all the packages I had learned to work with. Everything installed right out-of-the-repositories!

Dear developpers you have done a wonderful job! Thanks a lot ! :guitar::popcorn:
