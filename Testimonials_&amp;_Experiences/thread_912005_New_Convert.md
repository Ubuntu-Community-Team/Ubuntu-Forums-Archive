### Post by pwaugh on 2008-09-06
When about ready to install Fedora because my remote server runs RHEL, a friend recommended I check out Ubuntu.

I was very impressed that I could run it live from the CD to check it out without having to configure it, and that it was so easy to get running.  I run FreeBSD on a box, and could not believe how easy Ubuntu installation is in comparison!  As they say, "It just works".  I have just grown tired of how hard it has been to get anything working on FreeBSD.

Now that I am no longer developing for Windows, I finally feel like I can make the switch completely as I learn more over time.

### Post by jhenager on 2008-09-06
I also liked FreeBSD, but when I went to reinstall it a few years ago, had too much trouble and went with Red Hat. What I found was a page showing the spinning cube (Beryl running on top of Dapper), and I built a system with a nVidia card, and I was hooked!!!!!
Welcome, and I hope you enjoy Ubuntu as much as I do.
