### Post by 3rdalbum on 2011-04-27
> **Frogs Hair said:**
> I just removed 10.10 to get ready for 11.04 .

Er... why?

11.04's installer can remove 10.10 for you.

Or, it can do an "update install", which is like a clean install except it installs new versions of the repository software you had before, and it saves your /home.

Or, you could do a dist-upgrade.

And you can download the daily build of Ubuntu 11.04, it's going to be nearly identical to the final release, and you can just run Update Manager in 48 hours to be sure.

I'm sure you'll love Ubuntu 11.04, but be aware that there's a new UI called Unity. It's a pretty awesome desktop shell if you give yourself time to get used to it.
