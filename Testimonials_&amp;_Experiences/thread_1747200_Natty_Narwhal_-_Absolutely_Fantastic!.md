### Post by CB13RY on 2011-05-02
Over the past two years I've dabbled with various flavours of Linux, but have always ended up going back to Windows. That changed 3 days ago with Natty Narwhal.

In the past couple of years I'd used Linux Mint for about six months, but gravitated back to Windows XP. I almost did the same a week ago, but thought I'd take a look at Natty. I'm so glad I did. I've now replaced all my Windows apps with Linux ones. There's a couple of Windows apps I do need to use, but just run them inside VirtualBox.

Ubuntu has definitely come a long way in the past couple of years. Today I think Natty offers a much more "Beginners Friendly" Linux platform. So, hats off to the community and Canonical for producing and promoting Ubuntu.

What finally made me want to use Ubuntu? Well, quite a few things, but the thing that stuck in my mind the most was the French Gendarmerie using Ubuntu and Open Source apps for their IT infrastructure.

Well done guys!

### Post by Fraoch on 2011-05-02
At first Unity had me completely frustrated.  However I soon did a reality check.  All my programs were still installed and working perfectly.  All my hardware was still working fine.  I can't say that's been the case every time I upgraded Ubuntu, so I'm thankful for that.

I'm beginning to get used to Unity today.  It's more a case of doing things differently.  It's all still there, just in a different place than before.

There's more screen real estate for most programs since the program title integrates into the top bar.  It's actually pretty slick that the menu disappears until you need it as well.

I'm really liking the new vertical/horizontal scroll bars in dialog boxes.

Really about the only thing I'd like is the ability to remove the workspace switcher in the Launcher and most of the notifications in the top bar.  But if that's my only concern, then Natty is a pretty good upgrade, all in all!

### Post by CB13RY on 2011-05-02
So far, Natty has dealt with everything I required of it. With regard to my laptop, no hardware issues at all.

No issues with the Desktop either. Have found my way around quite easily.

Last time I had a go with Linux Mint, my printer and scanner had major problems integrating with the OS (I gave up with the scanner). But with Natty the printer was supported and within five minutes I'd got the scanner working as well.

So far I'm really happy.

I would have liked an easier way to import Thunderbird folders into Evolution, but in the end dumped all my separate folders into the Inbox in Thunderbird and then imported that as a single file to Evolution. I've only been using Evolution for a day, but I've really begun to like it a lot.

So, as someone who has come to Ubuntu from Windows (with a fairly short stay with Linux Mint) I feel Natty has really made that transition easier. The other thing I like about Ubuntu is the whole open source philosophy.
