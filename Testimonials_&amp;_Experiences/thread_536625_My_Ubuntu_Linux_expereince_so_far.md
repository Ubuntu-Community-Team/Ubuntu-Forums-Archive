### Post by tjksn on 2007-08-27
Greetings.  I want to leave some of my impressions of Ubuntu Feisty Fawn (actually Ubuntu Ultimate version).  I have tried some other distro's before and have always come away disappointed.  On this go-around I have Ubuntu installed on another drive so I'm dual-booting to take learning Linux slowly.

My experience so far has been both positive and disappointing.  Positive things are it was able to use the USRobotics USB wireless adapter without any special setup.  The negative here is it won't use WPA security no matter what I've tried.  At least I had internet and email access.

I appreciate and have used OpenOffice so switching from Microsoft Office would not be a big deal for me.

I have also been pleased with the Synaptic package manager and it's general ease of installing the available applications.

What I'm very much disappointed in is the support for Windows applications that some of us will still want to use and the lack of ease in getting WINE or anything else like it to run successfully and easily.  I've tried, unsuccesfully, to install Quicken (even with CodeWeavers) and although it appeared to install it would not run.

I will post, hopefully where it is appropriate, some of these issues to see if I can get them working reliably.

One last thing I would hope everyone here would carefully consider.  Linux would do much better and have more of a presence if there was a capitalist approach to it.  As it stands, with so many people developing so many different applications for free it seems to me that Linux is not even close to what it has the potential for.  Just my 2 cents worth.

### Post by Hallvor on 2007-08-28
tjksn: First of all, welcome to Ubuntu!

Linux is distributed under a free license (GPL), so you can actually charge as much money as you want, but must at the same time make the source code available. A free license has its pros and cons, but you must either accept it with its pros and cons or use something else.

When it comes to Windows applications, what gave you the idea that Windows applications should run perfectly on Linux? To turn the table: What would happen if I try to install, say, Amarok or K3b in Windows? It is two very different systems, and even though some applications work well in Wine, MANY if not most, work only with glitches or don`t work at all. Fortunately there are free Linux equivalents to almost anything out there, so if you are willing to learn using new applications, Linux should be no problem at all. But if you *must* use this or that Windows application, and if it doesn`t work perfectly in Wine, you are probably better off just using Windows.
