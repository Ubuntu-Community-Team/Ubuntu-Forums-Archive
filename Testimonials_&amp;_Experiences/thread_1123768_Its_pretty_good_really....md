### Post by lordyosch on 2009-04-12
I've been dual booting for what seems like ages now (its actually a handful of months. Perhaps since September last year).

I've barely used XP except when I 'had to'. I've got Ubuntu 8.10 working pretty much as well as I'd like it to -certainly as well as I need it to (except when there are problems without its control)

One small thing i'm working on is getting a sound problem fixed -need to try the proposed solution yet...

I'm enjoying Ubuntu sufficiently that I've just ordered a new desktop and made sure it is Ubuntu compatible -preinstalled even!


Things I still use XP for are... Excel -its just faster than OOO3 and the filters are cleverer.

I've got a few documents that OOO3 just can't open properly, which is annoying as they're official proforma for my work

Finally, printing/scanning. My canon is not properly supported so a lot of the features can't be used.

Well, these are work-roundable so I'm a happy linux bunny (penguin).
