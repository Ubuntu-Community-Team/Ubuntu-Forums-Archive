### Post by askyourpc.com on 2008-10-02
The pros and cons of Ubuntu are very relative. I mean that, in the sense that we judge from different perspectives on what we expect Ubuntu to do for us.

Coming from windows some will say that Ubuntu is more user friendly. I agree to an extent. Ubuntu is very user friendly because you are a few clicks away from doing a lot! To install programs you don't have to search for them you just go to the add and remove program feature of Ubuntu. Ubuntu will download and install the programs you decide to use.

On the other hand people want to see some windows programs running on Ubuntu. Some can be emulated with WINE but others cannot.

One of the features I like is that Ubuntu is constantly evolving but somehow it stays connected to windows trying to understand it. Like a fast developing child trying to understand how its uncle reads the newspaper.

I think the aspect of wine not being able to emulate a lot of games would be a big reason not to move over to Ubuntu. As for me, I use Ubuntu because I feel more secure on it but I keep being driven back to windows xp because of the gaming. I am not a huge gamer but it is nice to play a game once in a while. To see the new graphics of the latest game.

Ubuntu has so much potential. Some people may argue that it is already busting out of its potential into its kinetic phase.

Conclusion:
"People WINE about Ubuntu"
