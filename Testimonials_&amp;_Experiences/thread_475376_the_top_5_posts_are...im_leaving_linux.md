### Post by OzzyFrank on 2007-06-16
Hi. I Multiboot (Ubuntu, Windows XP, Mac OSX on the same PC) but only go into Windows every few days now. I have been using Ubuntu since Edgy 6.10, and it was becoming my desktop of choice right up to when my hard drive died. I finally got a new one just after Feisty came out, so of course had to start all over again. Now I've already gotten to the point it is my main OS, and I can do pretty much everything I used to be able to do in Windows, then some. Some things are much easier in Ubuntu, like being able to right click an iso file and it burns a CD (the hassle I had to go through in Windows to burn isos! Finally got a program after wading through feature-disabled trial versions).

And now that I've started playing with Wine, those few Windows apps I still rely on are making their way to Ubuntu! Was totally amazed when I got Mailwasher Pro going, as the ability to bounce back spam I see as important (to the point I was leaving mail on the server till I went into Windows and ran Mailwasher Pro first).

I'm pretty much what you'd call a Power User, and I can tell you that if your needs are more modest than mine, well Ubuntu should certainly do you well. If you're not all that computer-savvy, well you've got about as much to learn as with Windows; if you use Windows but complain about having to learn Ubuntu, remember that you weren't born with the knowledge of how to run Windows, but had to learn it!
