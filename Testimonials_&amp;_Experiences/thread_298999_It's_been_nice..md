### Post by russofris on 2006-11-13
Good evening,

I just wanted to comment how nice it has been using Ubuntu for the past couple weeks.  I've been a Gentoo user for the past few years and wanted a little extra "free time" while I spent the year in Europe.  I figured I'd try a pre-built distro, and gave this one a shot.

Overall, I'm really impressed with the efforts of the Ubuntu team.  That said, I did encounter some of the same issues that drove me to use Ubuntu in the first place.  I did my best to be a "good citizen" and file bug reports, make constructive comments, and upstream recommendations.  As it stands, I really haven't gained anything (meaning free time) over using Gentoo.

So I'm heading back to my old distro, where it will be the typical find-report-fix a bug, so that packages are all nice and stable by the time they hit Ubuntu.  Hopefully, future users of Ubuntu FF will not have to experience the same configuration issues that I encountered.

My best wishes to the Ubuntu team, and its users.

Thanx,
Frank Russo
