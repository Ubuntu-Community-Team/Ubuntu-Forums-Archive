### Post by besby11 on 2007-12-30
Ok, so back around 2001 I found a install CD "set" (it was like 8 cd's) of Redhat Linux with a case i bought to do a new system build. I never herd of linux before that and the thought of a operating system that was created by nothing more than the users of it sounded amazing, I got the idea that because the people building it were the exact same people using it that if any problems arose, they would be fixed immediately. So after I got done building the computer I dove head-on into something I've never done before. It was hell, It reminded me of being a deer on a 4 lane freeway, Everywhere I turned there was something to do with a command line (something my small Windows 98 user brain had not understood ever) and all this apt-get and umm... well alot of other stuff that after spending about 3 days installing it and then 2 days of using it (if you call wondering what a "/" directory was using") I un installed it and put windows 98 on there. Well, then after a while and I moved form windows 98 to windows XP and saw more and more flaws with the system i was using, First blaster then mydoom then sasser then bagel, I felt like a king in a crumbling castle. I tried again maybe 3 years later with ubuntu warty warthog but still I couldn't get the hang of installing apps and just general using it.So then I bought a second hand Mac G4 from Ebay and used that for a year or so and, I'm not really sure what it was but i could never get into the groove of using it. It worked and it worked great for checking mail and web surfing, my 2 hobbies and I learned alot about the terminal through using it, but then I got a copy of Counter Strike and Half Life 2 for Xmas one year and felt bad that my system couldn't take advantage of these apparently awsome games. So then i went back to XP and my G4 Config for about 3 years lisening in to distrowatch and web sites like that about the great open source community and by this time the ol' G4 was showing its age with not being able to keep up with 10.4 Tiger so I put together a new system and BUADADADADADA!!!! I PUT VISTA ON IT!... terrible mistake. This is a brand new machine and none of the devices were found... I was literally frothing at the mouth cussing my computer out and flaying my arms around in haste to this horrible piece of software and so I went about using this up until about a week ago when Linux sparked my intrest and so i downloaded a copy of Ubuntu 7.10 and burned it and after usuing the live CD for about 30 minutes I realized that the days of having to buy a manual with your operating system were gone, no more having to stay up into the wee hours of the day figuring out how to get devices to work. I had nothing to fear, with the knowledge that I've gathered from using the Command line in OS X and just being generally good at computers now I installed Ubuntu and when it came time to partition the drive I did the "crazy person" laugh and pressed the big ol' "take over the entire drive button" ohh did that feel good but any way, I want to report that the Ubuntu army is once again 1 more strong as I'm hooked on this and i see no reason to go back to windows, No Malware, and games work too "thanks, Wine Team!" so yeah, I guess welcome back me!

### Post by besby11 on 2007-12-30
After trying them all this is my thoughts

KDE/Kubuntu: It has ALOT! of apps and eyecandy that come installed, its suited for newcommers and people that have newer hardware.

XCFE/Xubuntu: This is minimal Apps/eyecandy and is great for use on older hardware but not really for noobs

Gnome/ubuntu: This is what I settled on, Its got the right mixture of apps/eyecandy for everyday use, but hey, if you want to tweak it you can install beryl or compiz fusion and have some really neat effects or if u need alot of programs the add/remove program in the menu is great or if you want to step it up go to synaptec in the system>admin menu for more packages and more extreme than that is to use the apt command in terminal.

for you i suggest using gnome or kde if u want somting with minimum tweaking options
