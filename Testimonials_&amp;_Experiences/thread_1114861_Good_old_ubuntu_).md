### Post by yahoo on 2009-04-04
I was ready to give up on Ubuntu.
My wife hated me 'updating' and fixing' various installs, and I got tired of Intrepid not working properly on my laptop.

But then the laptop died, and I bought an Acer desktop with Windows XP,
Merrily partitioned it, then stupidly deleted the pre-set recovery partition, prior to installing dual boot with Ubuntu (Windows for the wife, Ubuntu for me).
Trying to start Windows after that, the damn thing would start ... log in.... stop... reboot... start... log in....

I have been running off a live Ubuntu 8.10 cd to surf the web looking for a fix that would not terrify Windows, but any easy solutions required a Windows OS.

Then I hooked up my old laptop drive in a usb case this evening, and booted from my desktop.
Great - saw all the OS's, and tried to boot Windows XP on there.
Of course, I got a blue screen and shutdown message, Windows didn't want to play.

So I booted the usb laptop drive again, and booted into the old Ubuntu Jaunty installed there.
It was if I had never been away.
Ubuntu ran, just started and ran every program I had without question.

Don't you just love Ubuntu and Linux.
