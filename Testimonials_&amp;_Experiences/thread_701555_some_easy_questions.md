1. There are no known Linux virus currently in the wild. Ubuntu by default is not susceptible to any that might show up, without the user actually installing it himself.

2. Youtube videos only require the Abobe Flash plugin in Firefox. System 76's site shows a basic Ububtu install, so you will probably need to add the "non-free" common codecs . An internet connection will take care of that. When Totum finds a file it can't play, it will just ask to install the needed codec from the repositories. Pretty painless.

3. Everybody's list of must haves will be different. That's the nice thing about an easily customizable system. Yours will be completely unique.

### Post by Kevanx on 2008-02-19
The answers above are perfect.

2) System76 won't install any software that isn't considered "open" as far as I know, meaning that things like DivX, Adobe Acrobat, and others won't be included by default, but it WILL probably include players and viewers like totem or other programs that will take a lot of the guesswork out of it.

But the beauty about Ubuntu is that you can search these forums (e.g. a search for *install youtube*), and you'll find some pretty painless instructions, and more importantly, become less of a rookie in the process!

3) I think xmms media player, the gdesklets application and launchers, and a game called Enigma are probably the ones that I really couldn't live without, and aren't included by default, but that's for a whole other post.

p.s. Good on you for ordering a system76! It's quite the investment in your own computing use, but one that will pay off tenfold. I just ordered mine and I've been using ubuntu for just under a year. Mine's in the mail as well.
