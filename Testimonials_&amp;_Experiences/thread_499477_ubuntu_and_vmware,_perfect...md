### Post by raves71 on 2007-07-12
Well, recently I wrote a testimonial on these forums - and now I've realized it may not have been fair after using ubuntu as a primary OS for a little longer..

 I work as a techy for primarily windows based solutions - in fact we're a Microsoft and Citrix solution advisor, so at first it was hard adapting to linux and trying to keep all the functionality I needed for work while still using ubuntu.

 Some say that I might be cheating but if you absolutely need some windows applications in order to complete your daily tasks then I woudl strongly recommend installing VMWare server (or if like me you have VMware workstation available to you) as with this valuable tool you will find the switch to Ubuntu so much easier..

 I have all the benifits of an XP and the reliability and speed of a ubuntu workstation all in one.

 One thing I've also noticed is that using VMware workstation on Ubuntu used FAR less resources then it did running workstation on XP

 I've not removed my windows partition and reinstalled ubuntu as the sole OS on my laptop (reinstalled with 10 times less hassle then the installation of windows XP had on the same laptop as I only had to download 1 driver rather then a driver for each different component).

 After seeing the success of my ubuntu conversion another of my colleagues has also converted to ubuntu and so far.. all seems well :)

 I've got to say a big thank you to the ubuntu forums as whenever I've had a problem, a quick search on the ubuntu forums did the trick! 

 The only problem I now have is trying to install the net 6 vpn but I've got a feeling that's never going to happen!
