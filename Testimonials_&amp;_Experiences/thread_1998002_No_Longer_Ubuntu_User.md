### Post by 3rdalbum on 2012-06-06
It seems that you don't really like computers changing, and you don't really like commercialisation of computers.

I'd really question your choice of Windows, considering that it is changing even more than Ubuntu, and widespread support for Windows XP WILL be dropped. Change is inevitable in computing. If you've actually given Unity a good try (a week, generally, is enough) then you can easily install Gnome Fallback Session for a "normal" old boring unhelpful desktop. No terminal commands required. It's a lot easier than installing Windows XP.

Ubuntu has never shipped the Broadcom firmware on the CD as they don't have permission to do it.

Ubuntu has always shipped web browser, instant messenger, e-mail client etc on the CD. If you don't want it, remove it. Easy to do, and doesn't require the terminal.

There aren't really any advertisements inside Ubuntu. There's one area of the Software Center window that is an advertisement space, but guess what - it's only one small area. You can go to almost any website and find a bigger advertising space. If you don't like Ubuntu One, don't use it - I don't.

Canonical pays for the development of Ubuntu and all it wants in return is two hundred pixels by five hundred pixels in a program you can remove. The advertisements are not ugly and they don't flash or claim that you've won an iPad or that single ladies in your area want to meet you.
