### Post by OneSeventeen on 2006-08-11
As an Analyst/Programmer, I don't have to do all the complicated stuff that takes industry standard software (like writing memos and synchronizing iPODs), instead I just run a testing server with apache, postgresql, and PHP 5 that I develop on, then migrate my apps to my regular server via a mounted SSH folder.

My productivity has shot through the roof since switching, and I have been able to come up with more creative solutions, because most of the stuff doesn't involve requesting software purchases to try out.

Managing software licenses on my machine is a snap, and I've actually done just fine without Photoshop or InDesign.

I must admit, there is no equivalent to Photoshop, but as a web designer, being forced to use vector is causing my designs to come out much cleaner! [sample screenshot of page designed entirely in Ubuntu with Open Source Software](http://bin.woventhorns.com/hrsite2.png)

The best part is, I still interact with my office, which includes about 75 users running Windows on a Novell network.  I am mapped to all my network shares, and when MS Access questions come across, I just open up the VMWare server, and get the job done, then close it to go back to windows.  I also have to use VMWare for Microsoft Word documents that have weird formatting and VBA scripts, but there are only one or two left floating around anyway.

By far, the best decision I made as a web developer was to switch to linux, even though my peers use windows!

(The live server is also Ubuntu, and has been working flawlessly for 249 days, and then only because we moved locations!)
