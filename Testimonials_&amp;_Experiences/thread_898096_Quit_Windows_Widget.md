### Post by lovinglinux on 2008-08-22
Hi, I'm looking for some sort of desktop gadget/widget (for XP) that looks like those "Quit Smoking" gadgets, with some info about how many times I log into dual-boot Windows, total uptime and some list of goals towards total liberation from BSOD's and re-formating nightmares. :)

Now seriously, I tried Ubuntu before, mostly to understand it and test it with my hardware. This time, after a week of BSOD's, Firefox 3 crashing 3 times per minute and re-formating the system drive to solve regular Windows issues, I decided to force Ubuntu as my primary OS. ](*,)

I can't wipe Windows from my hard drive yet, because my TV card is not working and my favorite game needs PunkBuster to run on official servers, but I'm one step of doing it. I want to get rid of Windows really bad!

I discovered that the BSOD problem was due to a memory stick and I still have Windows on dual-boot, on Virtualbox and some applications on Wine, but I'm more pleased with Ubuntu each and every day I use it. I'm really impressed with it. 

The most pleasant thing was watching Windows BSOD inside the Virtualbox while Linux was up and running smoothly.\\:D/

On a side note, I had the impression before that Wine was really complicated, but is much easier than I thought. It is not perfect, but is very simple to use with basic applications. Wine HQ could put a banner in their site saying "Just install Wine, click windows executables and you are ready to go".

Another side note: If you ever install any windows game on Wine that requires PunkBuster, be aware that the nasty PunkBuster services are loaded every time you run any application under Wine and they make your Linux run slow like Windows, even if you close all Wine apps. The PB services are designed to run automatically and they stay running unless you kill them. :twisted:
