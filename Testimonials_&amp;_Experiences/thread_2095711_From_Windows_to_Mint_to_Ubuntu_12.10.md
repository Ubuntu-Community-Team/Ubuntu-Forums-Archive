### Post by The Spectre on 2012-12-17
Hi Everyone!

I switched from Windows to Linux about two months ago.

Every few years when Microsoft releases a new Operating System I looked forward to all of the new features and bells and whistles and when I decided to try out the Windows 8 Public Preview I was like Oh No this is Bad.:shock:
I was just going to stick with Windows 7 but I had the New OS itch now so I decided to give Linux a try.

I started with Linux Mint 13 KDE and I was very pleased with it but I became Interested in Ubuntu because Linux Mint is based off of it plus the new Ubuntu 12.10 was just released. 

I installed Ubuntu 12.10 into VirtualBox and started playing around with with it. 
At first I did not like the Unity interface at all, but the more that I used it the more that I liked it. 
I have come to appreciate that it is different and what the Unity interface has to offer. Thou it did take a little time to get used to closing open windows from the Left hand side.

After a couple of weeks of using Ubuntu 12.10 in VirtualBox I decided to switch from Linux Mint 13 to Ubuntu 12.10 and I have been ***Extremely Pleased*** with it.=D>
