### Post by shane_on_u on 2007-11-12
So before I begin, I'd like to state that this is NOT an Ubuntu bashing thread. I've been using Ubuntu since 6.06 and I've loved using it. 

It was the release of Gusty that has been such a huge disappointment for me both functionally and innovation wise. Fiesty worked almost perfectly for me but when I moved to Gutsy I had problems with the system from day one starting with repeated dropped network connections, desktop effects causing stability issues, and the loss of USB entirely after a week of perfect functionality. I tried for weeks on end to resolve these issues through these forums and the web and my own experimentation. I was eventually able to fix my network issues (myself) through a fluke but USB was still not opperational. I had to turn of desktop effects entirely for stability. Furthermore, I wasn't taken back by any new features... nothing special in my opinion.

So I decided to give Suse 10.3 a go and what can I say... I'm impressed. As opposed to Ubuntu, it looks great out of the box. I can use a graphical installer for complex setups including RAID whereas I needed to use the alternate CD to accomplish this on Ubuntu. I love how menu's work in Suse (I'm afraid to say it's innovative because I'm not sure if the Suse devs came up with it) but regardless, it great, easy to find stuff and I don't need a separate hierarchical dropdown menu at the top. Stability of the system so far has been great and I've got it setup running everything I ran in Ubuntu.

I do have to say that Suse isn't as simple to use as Ubuntu but not by far. And I definitely do miss apt-get and Synaptic but the Smart Package Manager is filling that void fairly well for now.

Anyway, I just felt I'd provide some feedback as to why I've left Ubuntu (for now). I hope to return one day but I don't think it'll be anytime soon as I feel Ubuntu still has some time to go to mature a little more.
