### Post by kubug on 2011-10-19
This isn't a post to knock the new Unity interface. In fact, I have not even tried it in Ubuntu 11.10, since once I heard that there is no more Ubuntu Classic setting, I didn't even download the new Ubuntu (I hated the Unity interface in 11.04 - unusable!). And I have been with Ubuntu since 2005.

But this is a 'Thank You' for making Unity a default now, forcing me to try other distributions. I actually used to use KDE in Suse 9.2 and loved it back then. Then comes the switch to the Ubuntu/Kubuntu side of thins. But Kubuntu back then (2005, 2006) seemed to have been the red-headed stepson, that didn't get much love from the Ubuntu side of things, so it was a fairly incomplete distribution (in what I experienced) so I stayed with Ubuntu all these years, never feeling that I needed to change.

I am actually typing this on my brand-spanking-new Kubuntu install now, and I am loving it. Everything is so much more customizable ('no-password-after-suspend-resume option' anyone?), and easier to use. There a couple things that I am having to get used to, but for the most part it's a much better experience than Ubuntu had been for the last couple releases.

Maybe so of you will feel the same about the new Kubuntu 11.10, so I really do urge you to give it a shot and try it. I now feel that it certainly should receive much more attention than it had been getting.

I probably posted this in the wrong part of the forum, so please feel free to move it to where ever you feel it should reside.
