### Post by 3rdalbum on 2006-09-20
Sadly, I must agree with the above poster. "This stupid O/S keeps asking for my password! I'm going back to Windows."

To the original poster: Check to see if there's already a Linux User Group in your city, and if so then arrange it with them or ask them for tips. If there isn't, then just make sure there are enough knowledgable people doing the installing, and that you've allowed plenty of time. Make sure all the installers know eachother's names, so they can ask for help if something doesn't go right.

This probably doesn't need to be said, but do the installing in a place with wired broadband. Make a CD with heaps of packages (restricted formats, popular OSS, all that kind of stuff) so you can install software on all the machines without putting a strain on the broadband connection.
