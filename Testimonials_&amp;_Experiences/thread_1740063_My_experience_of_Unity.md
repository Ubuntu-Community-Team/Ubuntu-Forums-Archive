### Post by benweston on 2011-04-26
Dear Community!

I have been a long-time Ubuntu user, as well as a regular user of OS X and Windows 7. Like many others, I use the best OS for the best task. I thought I'd write some of my thoughts on Unity and the direction Ubuntu is taking. I am not trolling, I am trying to be as objective as possible. Ultimately, these views will end up being subjective by nature.

The real thing for me is whether Unity was worth investing in over Gnome (3). In short, I don't believe it is.

Unity's usability is pretty good when you get used to it but it is nowhere near as intuitive for new users as Gnome. The learning curve is too steep. My family and fiance (average computer users) run Ubuntu and Unity left them completely stumped. Without me pointing out how to use it, they were lost. This was never the case with successive Windows upgrades, even big ones such as Vista and 7. The most important thing in UI design is 'constants', 'hooks' in the design that make a user feel at home. Windows has the start button, 3 buttons in the top right of windows and the File Edit View menu. OS X has the Apple logo and the dock. Their designs were evolutions, rather than complete redesigns every time. I'd wager even the OS 9 to OS X change wasn't as big as the one Canonical has made.

Unity (and, to a large extent, the Ubuntu theme) is clumsy, drab and out-dated. Visually, the system does not appear modern or fresh. Icons are too large, colours are too 'primary', fonts are badly spaced. It lacks a professional gloss that OS X and Windows 7 both have. OS X and Windows 7 seem so clean compared to the experience of Ubuntu & Unity.

I guess the overwhelming feeling is a lack of polish. This isn't a new thing to Ubuntu, I've always felt the UI was a bit second-rate, but it's exacerbated by Unity. Aesthetically, the system feels a bit 'Fisher Price'.

Out of curiosity at Canonical's decision, I downloaded the Gnome 3 live CD and found it felt immediately more professional. Infinitely more intuitive and clean than Unity.

Ubuntu has a lot less draw for me now. My fiance is back to Windows 7 (she demanded I took Ubuntu+Unity off). I'm well aware you can install other desktop environments but the whole point of Ubuntu is that it's for 'human beings'. Installing Gnome 3 is not something your average user (the type of person we're trying to reach out to) can, or wants, to do.

I'm sure many people will disagree with me, and that's fine. Thought I'd put in my $0.02.
