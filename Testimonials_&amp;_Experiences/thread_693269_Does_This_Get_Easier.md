### Post by sharpycore on 2008-02-10
So, I'm looking for some advice from long time linux/ubuntu users. Have you found that over the time you have been using ubuntu for, it has become more stable/usable/HW compatible?

But as much as I'd like to get rid of it I've had to keep my windows installation because I'm unable to do almost anything that I'd like to in ubuntu. But I was wondering if people have noticed ubuntu getting 'better' with each release? I've only been using it for a short time and I'm not about to just drop something because it doesn't work perfectly straight away, especially when the *whole point* of open source software is that if something is wrong you can be part of the solution, but I'd like to be able to look forward to scrapping my windows partition, and doing something fun/useful with the space (like a second linux/ubuntu install that can be a distinct environment for r&d or something). And I'd like to be able to go out with a friend to help them chose a new laptop and format the HD as soon as we get home, installing linux knowing that it will work. Right now that's not really possible and I'm looking for opinions on whether it ever will be like: how often bug reports are followed up on and how often the community fixes incredibly useful but amazingly buggy programs like tilda. Give me encouraging stories of winning.

### Post by sean42 on 2008-02-10
I would have to say that with a few caveats, my experience with Ubuntu has been better with each newer version I have tried.  I have almost exclusively used Ubuntu on two different laptops.  The first would be a Compaq laptop with an 800mhz duron processor that was running 6.06lts.  That was the highest version I was able to run on the laptop due to a shortage of ram. It ran great. It was peppy even with the slow processor and was very stable.  The only Issue I had with it was the lack of native WPA support.  I was able to fix this with the help of the forums, but was still a challenge for a newb like myself.  I would still be running this laptop if the power plug hadn't broken off the motherboard.  

Fast forward to today. I have a spankin new laptop loaded up with 7.10.  I love it.  I find it to be faster and more flexible than the Windows Vista install that came with the machine, but it is not without it's troubles.  My laptop came with the broadcom wifi chipset and I was never able to get it working with Ubuntu. I eventually ended up pulling out the wireless card and replacing it with one more compatible with Linux.  Flash based websites like youtube and other do not work out of the box like they used to because of licensing problems.  My headphone jack didn't work correctly until I dug around and found a solution on the forums.  These problems were easily fixed with a little work, but one has to wonder how many newbs would go through this.  I like breaking my computer just so I can fix it.  Not everybody likes to fix their computer that much.

I would have to say the older your equipment is, the better the support will be and the better your install experience will be.  Newer equipment can be hit or miss and probably will continue to be until more vendors support linux out of the box.  Until then it will fill a smaller, yet more rewarding place in the the computing world.
