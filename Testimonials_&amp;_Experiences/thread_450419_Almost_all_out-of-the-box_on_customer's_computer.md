### Post by 3rdalbum on 2007-05-21
Today I had the misfortune to be asked to troubleshoot someone's Windows machine.

Problems with Windows:

1. Internet Explorer refused to go to any websites typed into the address bar.
2. Outlook only downloaded 16 messages out of 143 before giving up.
3. Picture on widescreen monitor looked stretched. (Windows didn't recognise it was widescreen)
4. No 2D or 3D acceleration, and the ATI driver wouldn't install.
5. Windows kept asking for a driver CD for the HP printer.
6. Programs in the system tray constantly bugged him for input, and Windows occasionally seemed to be running programs without our consent.

I put the Ubuntu CD in to test it out.

1. Firefox went to every website (on Windows, as well as Ubuntu)
2. Evolution downloaded every message.
3. 2D acceleration out-of-the-box.
4. All drivers included.
5. Nothing in the system tray made the slightest peep, and Ubuntu never ran a program it was not asked to.

The only negative experience was that, although the monitor was recognised as widescreen, the picture was half off the screen. That may have had something to do with a broken video card still in the computer; maybe X was trying to split the screen image across the real monitor and the non-existent one. In any case, I got it going okay after a trip to the Screen Resolution program.

I'm just sorry that I didn't install Ubuntu on the computer right then and there. If he has any more troubles, I'll definately suggest installing it.
