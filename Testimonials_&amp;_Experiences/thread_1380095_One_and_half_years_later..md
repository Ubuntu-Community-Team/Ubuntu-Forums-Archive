### Post by kian.tern on 2010-01-13
Well one and a half years passed since I wrote the first [post]("http://ubuntuforums.org/showthread.php?t=869252") and I thought it would be nice to write a follow up.

One thing to mention first is I got my self a mac book, and now I can see how a user friendly *nix can look like.
Every system has it's own problems, but OS X has the most amazing out of the box experience I've seen.
I think that Desktop Linux people should stop trying to best Windows, they already did, they should try to best OS X.

Now with the "apple fan boy-ism" out of the way I want to talk about Kubuntu 9.10.
I didn't use the last 2 releases of Ubuntu (Switched to Arch at 8.10 release) and now I decided to give Ubuntu another try.
The installation was fast and easy, nice work on the QT installer.
The boot/shutdown times are great, I was really impressed how fast it booted.
I had Plasma crashed a couple of times during initial configuration. After I was done with it everything is nice, fast and stable (at least for now)
BTW something needs to be done about Plasma crashes, although it rarely happens but if it does it's very ugly(visually).
Unfortunately not everything is good.
For one audio quality is awful.
I'm using Amarok, with the same configuration I used it in Arch and the difference is staggering, there is more noise then music.
I'm not sure where is it coming from, but it's not hardware or mp3 files quality issue.
I'll try to switch the backend to gstreamer but it's still to much hassle just for properly playing music.
Another thing that strikes me strange is lack of mp3 support AFTER installing kubuntu-restricted-extras.
I had to manually install ffmpeg package to make it work.
Equalizer in the default Amarok installation is also missed.
MAPI (Exchange) support is alpha quality at best.
I almost got beaten up by our sys admin because I locked my AD account 5 times
while trying to configure the thing (using the correct login/password each time).
Why include it in the stable repository?

Ubuntu has still a long way to go to become a real competitor in the Desktop scene.
There is still to much hassle around small things that should work out of the box.
Anyway Karmic Koala is a very nice release and I going to stick with it for now.

It would be really cool if people who wrote earlier write a follow up as well.
