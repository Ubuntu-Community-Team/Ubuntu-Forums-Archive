### Post by hajk on 2005-10-27
Yup, no M$ in my computer past! 

Started as an IBM loyalist, wearing out three generations of IBM PCs: the original PC, the AT, and a 386 model; and using IBM's version of DOS and OS/2 (ending with the excellent WARP.) IBM lost my loyalty only when it stopped supporting OS/2.

After OS/2, M$ Windows 3.1 seemed so ugly and clumsy -- there would have to be something more capable. And there was: Linux, by means of Slackware. It took some doing to install and to tweak, but that was part of the fun. Soon after, I discovered Debian (potato or hamm, I guess), even more fun to install and tweak. 

Needed some drivers available at SuSE's website, then decided to try SuSE itself (then at 6.1) mainly for the printed manuals. Great disappointment, badly translated English, not containing anything much new compared with the man pages; and complicated/opaque services in /etc/rc.d. So, back to Debian, Woody this time.

Stayed with Woody several years, waiting for Sarge... and waiting... and waiting some more. Finally, tired of waiting, decided to try Gentoo 2004.3.
Hey, Gentoo was fun! Did a complete bootstrapping install, compiled everything -- took days and days to finish -- boy was I proud! I made sure the toolchain was always up-to-date (taking hours of compilation everytime something changed in binutils, glibc, gcc...). Wasn't enough, though, still got compile problems on updated packages, grrr. 

So, back to Debian, Sarge had finally been released. Sarge didn't take long to install (although X had some problems) and I found it very usable -- yet still lacking a bit in polish. I had always read through the various Debian forums, and noticed that some knowledgeable Debian developers (Watson, etc) had moved to Ubuntu. 

So, made one last move to Breezy -- nearly flawless install on my desktop computer and on an old IBM ThinkPad. Installed the various "restricted" codecs, etc, as per the Wiki, and everything just works: play a DVD, play an audio CD, play MP3/OGG files, listen to radio streams, watch TV streams, do programming (Squeak, Oberon, Java, Fortran), write a book in LaTeX... Solved a few minor glitches with printer VPNC/firewall setup with help from forums.

Finest Linux install ever, no question about it. 

The only surprising thing: the many noobs in these forums getting so upset when something doesn't work straight out-of-the-box. What will they do when they hit some real rough spots in life, I wonder...
