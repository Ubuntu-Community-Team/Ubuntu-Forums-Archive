### Post by Breakar on 2008-09-12
After using win98, XP, vista, other linux distros(not often as windows) i have found a perfect OS for me which is ubuntu.

For me a perfect OS must do the following,

[I]1. Allow me to do my work
2. Allow me to play games
3. Allow me to edit audio/video
4. Able to use my USB devices perfectly[/I]

Out of the box ubuntu has acceded doing so, even better than windows based systems. With a basic video editor in ubuntu i can edit a 700mb video and it only takes a few seconds to actually load the video into the editor whereas windows id either freeze and have to reboot or it would take 20mins to do so.

With gaming, i went the crossover route because i read they are giving back to wine so i went with them and i'm very satisfied thus far.
The games i play and that are working in crossover are.
[I]
Counter strike: source
Half Life
Never winter nights
Need for speed underground[/I]

I do most of my work using microsoft office, i had my doubts at first i would use linux as i use microsoft office but so far i am using microsoft office less and openoffice more, the more i use it the more it becomes second nature and on pare with microsoft office.

So far i have been using ubuntu for 3 months, without dual booting :) .. i love ubuntu so much i buying a gift from their store to support them.
