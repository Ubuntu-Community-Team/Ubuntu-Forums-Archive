### Post by RichardRT on 2010-04-30
As an end user I just want to say that I'm really happy with Ubuntu. My neighbor installed the Ubuntu Remix version for netbooks on my EM250, which came with Windows 7 starter edition. 
Windows 7 starter edition was using 500mb of my total 1gb RAM, or 50%. Ubuntu is using less than 200mb, or around 20%. 
The desktop environment of the Ubuntu OS is easy to navigate, software is easy to find and install, and I'm really impressed with it. I use the netbook exclusively with a wireless or wifi connection, and it works well. I've been using Ubuntu for a few weeks now.
Several years ago I tried a Debian Linux OS briefly on a home pc and didn't stick with it because it was too hard for me to understand the "language". I'm happy to say that this is not the case now. You guys have come a long way. Way more user-friendly for the average user. 
I still have Windows 7 on my netbook which I use sometimes. But I'm finding more and more that I boot into Linux instead of Windows. Way faster. 
Thanks for all the hard work that everyone puts into this project. :)
