### Post by rjbl on 2011-09-10
> **squenson said:**
> I also like very much 11.04 but comparing it with Windows XP which was on the market 11 years ago is a little bit too easy... Ubuntu should be compared with modern OSes like Windows 7 or Mac!

Done that thing this very week.

1. Ubuntu 11.04 is quite unlike either Windows 7 or Mac OS. Ubuntu is a complete personal computing system; Windows 7 and Mac OS are simply bare operating systems, neither have any useful functionality. Ubuntu's OS is GNU/Linux. Try to learn the difference between 'Operating System' and 'Computing System'

2. The Ubuntu 11.04 installation is trivially easy and runs exceptionally fast when run from the current Live CD. Believe me, the **12 minute** complete install took my breathe away - and I have been a PC systems' pro since 1984/5

3. Hardware recognition and driving by Ubuntu remains in a class of its own. Everything drove perfectly except the GT430 card - the Ubuntu default drivers produced a perfectly acceptable display but couldn't drive 3D acceleration. It took less than a minute to load up the proprietary driver, via 'Administration' -> 'Additional Drivers'. It worked flawlessly, but Ubuntu always does.

         My experience in installing Win 7 on the same box was a very nasty one. Microsoft's default drivers all gave poor results and I had to spent a couple of days scratching around the 'net, and installing, manufacturers' drivers. Two days of my life that I ain't going to get back. Then I found that Microsoft's Flight Simulator X doesn't actually run properly in Win 7 - that was the only reason I built the bl@@dy Windows box in the first place.

4. Ubuntu's on-line configuration management system - 'Update Manager' - worked flawlessly - as it always does. Update of the LiveCD installation whistled through in 20 minutes, keine problem - but there never is with Ubuntu Updates. The afore-mentioned bl@@dy Windows box? Yeah .. right; SP1 blew the installation apart, so I rolled back to XP SP3; and about a week ago dual-booted the box by WUBIing Ubuntu 11.04 onto it. Flawless process.

rjbl
