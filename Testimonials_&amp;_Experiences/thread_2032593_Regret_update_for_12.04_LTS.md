### Post by maruda on 2012-07-24
Hi all.
I have using Linux since 2004, since 2008 I'm using Laptop Dell XPS 1330M and in 2008 I switched from Debian to Ubuntu. 8.04 LTS release I used without any bigger problems until June 2012. I have decided upgrade to newest Ubuntu version because my computer slightly slowed down. So I did it, unfortuantely :( I still regret.

I'd like to give you some information what is wrong with 12.04 LTS and what pushed my out of this distributon (I have decided today change distribution and back to Debian or Mint. 

1. I don't understand why my computer become unsopported in Ubuntu? WiFi doesn't work, video card doesn't properly work with animations (for instance round coube). After 4 days I have finally configured wifi! - wow in almost the same way what I remember from Debian in 2004 with my old noname network adapter :(
2. Laptop hung off every time when I shut it down or restart. Yes, I changed GRUB settings wit force, nolapic and another options - it doesn't work :( I don't keen on around with uninstall Nvidia drivers... I want and expect full operate system without any temporary solutions. I found in ASK Ubuntu website that problem have been registered - OK, maybe one day solution will come, but I don't have any time and patient to wait for it.
3. Compiz doesn't work properly with Unity. Many useful options conflicts with Unity options. Finally I got compromise I used Unity (I really wanted to learn and find it easy in use)... But it doesn't work in my lapop.
4. I don't understand idea of Unity and shortcut left bar - not configurable and not flexible. I don't understand why if I want to run any software user has to (in fact has to) every time finds software in Search option by application name. Very often application doesn't be found :(
5. I don't understand why in can't create my shortcuts in Panel. I understand new idea of using that area in Unity but why there is so big functionally limitation ? Ofcourse I was able to turned off Unity in compiz but - surprise - in this moment many options in compiz is turned off too. Regular user could thought that system lost important part of funcionality like Alt + tab switcher.
5. I don't like Settings option it brings my windows XP Control Panel. As far as I remember Linus said in one conference that Linux is going in wrong way and become to much similar to windows... now we have drivers, Settings, Unity... 
6. And finally the best... Yesterday I installed 63 new updates/fixes. Today I have problem with connection to my WIFi!!! dmesg shows me (I have wandered) that after I connect to WiFi suddenly my card is reseted with communicate that it is according to regulatory in GB - I am in Poland and whole my system is configured to Polish. As you can gues my laptop is lossing connection to WiFi and doesn't work :( - yes, doesn't work because I don't want to configure it any more.

These are the most important mistakes/bugs new Ubuntu - unfortunately there is much more problems than I described. 
I really regret but I have to switch to another distribution I dan't want to fight. I chose Ubuntu because it was user friendly, really functional COMPLETE operating system - I belive still is. But past release better support hardware than new one. Maybe one day I come back.

Regards,
Piotr.
