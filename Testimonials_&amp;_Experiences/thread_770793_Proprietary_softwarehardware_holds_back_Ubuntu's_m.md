### Post by enchantedsky on 2008-04-27
Ubuntu Linux has become my favourite operating system. I switched from Windows Vista to Ubuntu Linux, and I like the community and open-source nature of it all. I love all the innovation it has; Compiz-Fusion, repositories, etc. I just hope that one day, hardware companies drivers could release more and more drivers for Linux so Ubuntu "just works" on any computer you install it in ----- without having to worry about what works, and what doesn't. 

Even if drivers exist for certain things, they don't always work well. For instance,

1) Adobe Flash - CPU speed jumps up to 100%, the video is very choppy because Adobe made a flawed Flash driver for Linux. Even on the fastest computers this is happening.

2) Java - It doesn't always work well according to many on this forum. Sun is starting to embrace Ubuntu though, and I've read that they are starting to make Java more open-source friendly to work out bugs.

3) DVD/Divx/MP3 - All the hurdles of adding Medibuntu repositories, and then even so, the DVD menus didn't work until adding Xine. Too many hurdles to do even though it is technically illegal in my country to install this stuff :)

4) Hardware drivers made for Windows, but not Linux, so you can use your device with Ubuntu. (e.g. ALL Dell printers lack drivers for Linux, even though Dell sells Ubuntu computers)

5) Restricted Graphics Drivers - I'm using Nvidia restricted drivers, but it is not exact driver for my particular graphics board; it is a generalized driver for my chipset. If Nvidia really cared about Linux users, it would create drivers for every single graphics card it sells for Linux to have 100% compatibility with your computer, hence fully eliminating any potential graphics problems you may have.

These issues are NOT the fault of Ubuntu, or Linux in general..... these are all the fault of companies with closed-source software/codecs not making their drivers for Linux. And even if they tried to release something for Linux, it is not of good production quality (e.g., Adobe Flash).

I believe that Ubuntu 8.10 will be a more significant release than 8.04, merely because more people will embrace Linux, hence more drivers and software will be available to it. We are unfortunately in a Windows Vista world until then, where more drivers are available to it merely because more people use Windows than any other OS.
