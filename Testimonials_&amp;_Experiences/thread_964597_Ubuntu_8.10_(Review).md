### Post by Crafty Kisses on 2008-10-31
As an experienced Linux user I'd have to say this is one of the more solid Ubuntu releases  it's like a release from 6.06 to 7.04, it's very nice and I love the features. I really like the stability obviously and I'm liking the new Compiz effects that come pre-installed.

What I've been having troubles with is "Google Maps" street view doesn't want to work correctly but hopefully that will get fixed in following updates and what not, but overall Intrepid is one of the better releases to hit the Ubuntu circuit, so I want to give the Ubuntu team credit.

The hardware support is better than ever, everything was detected, I have a pretty solid machine so I was kind of expecting that, but some things actually run a lot smoother.  The only gripe I really have is the NVIDIA  issue but that really doesn't effect me I use the 177 driver, but I feel bad for other people.

What I like about Intrepid:
[LIST]
[*]Stability
[*]The new Compiz effects
[*]The new features
[/LIST]

What needs to be improved:
[LIST]
[*]Google Maps (Possible Flash Issue)
[*]Sound issues (sound panel)
[*]The NVIDIA issue (I'm aware this is in on NVIDIA's end)
[/LIST]

Overall I give Intrepid: **8.5/10**
