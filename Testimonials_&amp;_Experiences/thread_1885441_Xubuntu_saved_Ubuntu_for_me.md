### Post by Rail1 on 2011-11-23
Gnome 3 and Unity are awful. God awful. I really tried to give them a chance, but was aggravated for 2 weeks straight. Even the Gnome classic shell that comes with Gnome 3 was limited. Work flow completely disrupted. Kubuntu runs like a pig on my computer. Computer just crawls on it. After hearing Torvalds himself and gone to Xfce, I figured I had nothing to lose and tried it out... The only reason I am still using Ubuntu (or Linux at all really) is because I found Xfce. 

So if you are a technical or heavy user if need an actual working desktop environment that promotes workflow and process efficiency, give Xfce a try before you quit Linux. I found the Xfce shell to be buggy if I installed it on top of the Gnome-based Ubuntu... a fresh install of Xubuntu works flawlessly.

Good luck...
