### Post by ironslave on 2008-10-17
So it's been some time since I've used Ubuntu and I am glad to have it back.

I first started using Ubuntu back in the days of 6.04, it was right after I got married and all I had was a couch, a bed, a T.V., and my Laptop. Windows took a dive for the deep end and i had lost m restore CD some tome ago (I managed to keep the same install of XP running for 2 years..) with nowhere to turn i said, Linux! so the first thing i did was grab this Knoppix Live CD i had been looking at and booted her up.

Oh this is horrible i thought. (the installation didn't go over to well) 

but it was good enough to get me some O.S. to work with i then frantically looked around for something to use until i could get a copy of windows. (my wife does not do Linux) I tried Linspire, Gentoo, Mandriva, all with to much trouble. i had not used Linux since Red hat 7 (about 7 years ago) i then found Ubuntu (yay!)

it worked well for the most part. I then traded my laptop for a desktop PC. Immediately installed Ubuntu and then broke it beond understanding and reinstalled it again. I managed to bounce between them for about 6 or 7 months before I got tired of breaking the system mostly from using XGL with Compiz and MythTv. 

I then gave up my passion for Ubuntu and returned to windows.

I now own a new PC and recently stumbled upon the buzz for Intrepid Ibex. with Vista delivering sub par performance, and a spare partition with a failed install of OSX86 on it i though why not... so I Downloaded and Installed the Latest release which was 8.041 and ran her for a spin. but I could not boot :(. Apparenty Easy BCD isn't as easy as I thought it was and i had to Install grub, which I chose not to do.  Ok so lets install grub, cant be that hard. I'm sure i can find the info on the forums right.... NOPE!  all the info i found was for Re-installation... looks like 30 Minutes down the drain :( besides the fact that my keyboard didn't work.(usb legacy) I'm off to a bad start I wonder if i am just wasting my time I thought. well i got Ubuntu Reinstalled and rebooted into Linux and havent seen windows Since..... it's been almost a month since I've installed Ubuntu and I am amazed. the only reason I need to boot to windows is to update my Itouch when the firmware comes out or if i want to add anything to it. but once a method to install Itunes 8 in lunux comes out i am golden.

Ubuntu has made leaps and bounds in the last year and a half since I stopped using it, my 2 biggest issues Ive had with this OS is Fixed  (Compiz And proprietary Video rivers)
I dare not Touch Myth TV.

I'm glad to be back :)
