### Post by projectblu on 2007-05-06
i switched from windows xp to ubuntu about 2 weeks ago. and i have to say that im very happy. i got everything working without any real problems. I found help with any problems i had in about 5 minutes on the fourms. There are basically replacement for all my windows applications in the repositories. I even got a second box running apache / php / and a mysql server with phpmyadmin in about 1 hours total work time. and it Accutally worked the first time. Unlike trying to set up a windows server which refused to recognize the public ip my router tried to give it. I even bought a wireless desktop from Logitech earlier today and It recognized it and work in less time than it did in windows. I have to say 5 stars for ubuntu.
