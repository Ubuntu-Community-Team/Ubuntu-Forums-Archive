### Post by texla on 2011-03-28
I know this post goes against the grain but I installed the Netbook Remix 10.10 last night and I think it's very cool.  The first time I saw it on my live usb session I almost ripped it out of my computer to have the desktop version installed asap.  But I returned to it and installed it on my netbook in the end because it intrigued me how well it conserved space.  My computers are becoming an organizational nightmare and I needed a system that would help take that over.  No, Windows doesn't come close to achieving that for me but this is taking care of it nicely.

The Unity dock as been a really pleasant way to get to both my apps that are open and the ones that that I want to launch.  The new and improved Desktop switcher is my favorite app by far.  It organizes the multiple open files in a very calm and fluid manner.  It is extremely easy to get to my other opened files, folders and programs with it - and the drag and drop into desktops is awesome, too.  I've been using Compiz and cairo-dock for a while now and no matter how  much time I spend customizing, it just seems too busy for me.

Just a note on the install.  I was able to get it to dual install next to XP on a netbook with two partitions with no hiccups.  Actually, when I opened it up in Gparted there were 4 partions - one unknown and small sda4 that I deleted and one that I believe is a recovery partition.  The XP install was all on C so I partitioned out the D drive to have / /home and swap partitions.  It was a tense 15 minutes but when I rebooted (and took out my flash and sd cards!) all was there in Grub.  Both XP and Ubuntu have booted up nicely since. This is the first time I've done a dual install with Windows and it didn't require a recovery disk or at least a reinstall of grub - nice surprise!

Of course my netbook is not my main computer and it is set up primarily for web, office, media playing stuff so I'm not freaking out about all of the lack of customization and how hard it is to find everything!  But it's flowing through all of those programs well and looks so much better than the XP it came with!  Actually I added it from synaptic on my main computer, too in an experimental distro partition just to see. Looks like it might stick with me there, too... Probably in a day or two the crashes will destroy my good will towards this distro but hopefully not.  I hread they're dropping the need for this netbook distro entirely in Natty so I  hope the next release with it's combined unity/netbook/desktop ubuntu finds a way to merge the clean organization and look of this version while bringing back the ease of customizing features of the desktop edition.
