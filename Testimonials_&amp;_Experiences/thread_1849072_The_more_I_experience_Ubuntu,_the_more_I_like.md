### Post by 555byron on 2011-09-26
I have just set up my old (formerly a Compaq xp machine) desktop as a dual boot, dual hdd.  I tried ubuntu on the original desktop configuration which sported a whopping 256 mb of ram... a real power house.  Ubuntu 10.04  was unstable at best (not Ubuntu's fault) I figured it was the DDR ram and lack there of.  So I looked at maxing out the DDR ram to its 2 gig potential.  $50... for this old crap.  So I got the cheapest MoBo/processor combo I could find, a Biostar board with an AMD Sempron 130 with 2 gig of DDR3 ram for $105 shipped, a much better choice. (that included a few brackets/cables too)

Windows 7 is on a SATA 160 gig laptop HDD, this drive was originally in my wife's Compaq laptop that was struck dead by lightning via fax line to printer then through usb cable and into the computer.

Ubuntu 10.04 is on the original compaq desktop 80 gig IDE HDD.

I set up windows 7 first, formatted the 80 gig drive (it was really wiggy due to the 6 partiions on it).  unplugged the Windows 7 drive, then installed ubuntu on the 80 gig drive.  I plugged everything back in and it works great.

I too find that I like ubuntu more and more as I use it.
