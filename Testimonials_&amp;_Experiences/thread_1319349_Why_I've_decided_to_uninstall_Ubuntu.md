### Post by dustgrain on 2009-11-08
Hello All,
I'm a simple home user with a 15.4" generic laptop of decent value (Acer Aspire 5920).
I love open source projects and have been a Linux fan ever since the late 90s (I even got the original Red-Hat Linux 6 CDs to prove it :))
I bought a laptop about over a year ago in a PC World sale in Manchester U.K with a Windows Vista pre installed (obviously) and thus far up to several months ago I was using the Vista for my simple needs (browser, multimedia, email, etc) without any abnormal problem that was unsolved.
Few months ago I felt I needed a change and had some extra time to learn and get adjusted to a new system, so I downloaded Fedora 11 and installed it. Installation went fine and there were no problems. I did, however had some small issues with plugins and alike but that didn't really bother me much.
When after few days I ran into a good friend who mentioned that he's using Ubuntu 9.04 on his HP laptop saying how smooth and great it works I had to check it out for myself, and behold, the next day I hurried to download the live cd to test the magic for myself.. I was very impressed and without further a due decided to install it there and then.

I love Ubuntu. I really do. But there are 2 major issues that won't let me keep using it.
The first and MOST CRITICAL issue is that I got on my laptop a widespread, popular, generic wireless card (Intel Wireless Pro 3945) , and the iwl3945 module that Ubuntu is using as the driver.
The iwl3945 is the problem, it's the only module Ubuntu has for this chipset and it has a serious problem that causes it wireless transfer rate drop significantly after few minutes of use. The bug is already known and was posted on kernel.org, launchpad, Intel wireless developers forum, etc since it was found some months ago (since starting to use the iwl3945 instead of the ipwl3945 some previous kernel versions ago) 
This issue is critical for me as I'm living in a shared house and running a cable is a no go. Besides, I feel entitled to use the bandwidth that I pay for, (I pay for 5MBps when really getting on my wifi about 1.2MBps). That Issue didn't exist for me when using Vista.
The other issue is a problem of slow transfer rate when copying or moving files between partitions and to some USB flash drives (it goes down to 300kbps, wtf?) and sometimes can take days to transfer a 2 GIG file to a flash drive. That bug was also posted in several forums but just like the other bug no one seems to be addressing it. Both bugs have had no solutions for a long time (using NDSI wrapper is a bad idea stop thinking it) and my time of waiting has ran out.
I'm currently downloading Windows 7, which I find awfully shameful, not because I need to pay for it (I don't, the OEM of the original Vista should work), but because I really am an open source fan

So, until Ubuntu solves these 2 issues, or until I get another machine, it's farewell Ubuntu.
