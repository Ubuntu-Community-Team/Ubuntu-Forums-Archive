### Post by zealibib slaughter on 2010-09-21
Hi all.  Just wanted to share my experiences with Ubuntu, hopefully anyone trying it out can find encouragement from this.  The first time I tried Ubuntu I remember thinking its brown, that was gibbon or heron I can't remember which.  It installed easily enough and everything seemed to run ok but I couldn't shake the feeling that I was again sitting in front of a system running windows 95, little did I know at that time just how wrong I was.  After much frustration of not being able to navigate the alien looking desktop as quickly as I did W2k I gave up.  A few weeks went by before I dared to probe the inner workings of Linux again but thankfully my windows install suffered a BSOD and forced me to try once more.  Although I still felt lost I began using Ubuntu as much as possible even though I had fixed the problem with windows (with ubuntu).  Soon though I came across a problem, I had a cheap lexmark 3 in one printer which I wanted to set up and It seemed like it was going to be easy.  Lexmark had a linux driver on their site for this model so I downloaded it but it wouldn't install.  The installer reported wrong cups version need cups 1.x or higher.  Well I figured as a "former" windows power user this should be easy to fix, all I had to do is update cups. This was not the case though as I soon found out, I actually had the right cups version installed and a quick google search returned a work around.  Now this is where it got interesting.  The work around required editing a configuration file in the installer by compiling the file without installing and changing one line.  I was in far over my head, I had never compiled anything and the only programing I've ever done was way back in the 80's with BASIC and some HTML in the 90's.  It seemed every time I asked a question I just got more confused until I ran across someone who had been in my shoes not to long before and was able to explain not just how in a way I grasped but also what was going on with what I was doing.  Looking back it wasn't so much I didn't understand what the others was saying but it was a fear of not knowing exactly what I was doing that was blocking me.  As I compiled the installer and edited the configuration file I realized that it wasn't as hard as I thought and that was the break through I needed.  Soon after I came to understand and then like the Brown Gnome desktop, and with some good exploration learned just how many ways I could manipulate everything about the system.  Not long after I dumped windows and continue to use Ubuntu on my desktop and laptop.  I've also continued to learn about Linux and picked up a bit of pearl along the way (but I'm no programmer by any means unless you want it written in BASIC) and have tried out many different distros (currently running puppy from a flash drive on my laptop). So if you are new to Linux and Ubuntu hang in there, the rewards far outweigh the time and effort you will put forth learning this great os.
