### Post by dormant on 2007-09-25
I'd become a Linux convert. I taught myseld a lot, read from the forums, and followed other's advice to the letter, even when it was clear I was out of my depth. Do you know how scary it is seeing screenfuls of error messages flashing by, knowing that you don't know what any of them mean and fearing it stops at one?

I had three machines running Ubuntu. Now one is broke, the second is showing worrying signs. And the third only cost me 55 Euros from a second hand shop and is only powerful enough to run Ubuntu Server and has now been usurped by a NAS device.

**Asus Pundit**

I bought this from a computer shop in Vienna - they'd built it from a barebones chassis for some project and it was no longer needed. I got 50 Euros off the price because they couldn't find the Windows drivers. It ran Fedora Core 4 for a while, then Ubuntu's 5 and 6. I'd set it up as my Media Centre, playing avi's etc through a 24" Dell monitor and my hi-fi, running Slimserver for my Squeezebox. I had a small wireless keyboard as my remote control. I was so proud and happy.

It is now dead. 

It wouldn't boot two days ago, telling me to run fdisk manually. So I did, and pressed the "y" key a lot when it wanted to fix inode things. After that it wouldn't boot into X-windows, complaining that the X-server was misconfigured. At that point I saved all my files - it only took me two hours to work out how to mount an external USB drive from the command line.

A new install I thought. But then I found that it wouldn't boot from the CD drive. In the many boots, I think there was an error message about cdrom0. But I can't even check that now. Which may be a good thing as it usually takes me an hour or so to remember where to find errors and important things.

Because it won't boot at all now. Power on - one long beep - then another long beep. All that's missing are the wisps of smoke coming out the side where the quiet fan is.

A Mac Mini beckons.

**hp nc2400**

This is less than six months old. I installed Ubuntu 7.0whatever when it was in Beta, no problems at all. It is dual boot with Windows XP, but I have never used Windows since the first week or so.

But I am slowly having more and more hassles, and am worried it is running away from me.

* A while back, the touch button things for sound gave up on me.
* I have lots and lots of problems when I try and do any network things more than DHCP. I use net-manager (I think), but don't seem to be able to set it up for different places and set-ups. When I try, it totally buggers up the DHCP as well.
* I am having more and more problems with Nautilus. It often freezes on me, then refuses to close and I have to force a quit. There is also some wired behaviour when minimising a window leaves lots of thick black lines on the screen.
* I cannot browse the WIndows network at work. A colleague with Suse Linux can, straight out the box.

I almost feel a re-install is needed, but my Pundit experience has really scared me. Also, I invested a lot of time configuring the thing and installing various software.

I want to use this computer. Not spend more time fiddling.

**My Dilemma**

I should be getting a new desktop at work soon, a twin-screen Dell with lashings of memory and disk space and everything. I had planned to install Ubuntu. 

Maybe I'll go Suse. At least then I'd have a savvy-ish colleague at work who can help me in times of need. 

--
Notes

1) Some technical terms above may be incorrect. I'm not geek enough to know, nor patient enough to check.

2) I have lost the will to be an Ubuntu Enthusiast.
