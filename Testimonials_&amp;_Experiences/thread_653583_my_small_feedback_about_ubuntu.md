### Post by ffaeon on 2007-12-30
Hello,

I just began installing ubuntu 7.10 dvd version. It wasn't my first contact with linux : I tried Mandrake which deceived me. Now I know opensource is essential to computer so I want try again with the popular ubuntu. Someone said linux distributions became easy to install. Let's see...

[LIST]First of all, about wifi
[/LIST]
-> channel problem : I can't find in any wifi tutorial how to set the channel. That's why my wifi dongle was blind.
-> lack of documentation : I need to iwconfig manually and... google a lot lot. Things doesn't work naturally
-> too bad I have to gedit some files to add a line here and there (/etc/modules and /boot/.../menu.lst)
-> iwconfig doesn't work sometimes : if I set essid, it doesn't do anything. if I set channel 11, it doesn't do it.
-> instability : after Internet works, it doesn't work after a moment... because my my wifi dongle became once more blind. have to desinstall, reinstall, reconfig...

[LIST]
[*]wine
[/LIST]
-> can't install following winehq.com instructions : I have to google in order to know I have to check a "universe" repository or sth like that...
-> When I emulate nintendo 64 emulator (project64 v1.5), it lags, doesn't always work, then freezes my pc.

[LIST]
[*] Ubuntu
 still need internet for synaptic manager (universe repository, ndiswrapper, wine) althought I downloaded dvd live ubuntu just to avoid installing from internet.[/LIST]

_Conclusion :_
A linux user needs to google lots lots lots (from my point of view, the documentation inside ubuntu about wifi for instance is incomplete and sometimes wrong/old)
A future linux users must geek.
Linux make me lose a half day.

I don't know if linux is mature yet for non geek users but I will try again to make it works fine.
bye !
