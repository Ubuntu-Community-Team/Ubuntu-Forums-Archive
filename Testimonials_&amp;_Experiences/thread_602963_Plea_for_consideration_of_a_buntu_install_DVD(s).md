### Post by 2damncommon on 2007-11-04
I really have not seen the correct place to say this so I am posting it here.
I did not see a distribution feedback address on ubuntu.com or even here on the forums. Perhaps this is the correct place for this.

I am one of those Linux users that will try any distribution I wish. But I am also using a preferred distribution over a longer term. Long term preferences have been Suse, Mandrake/Mandriva, and Debian.

Since I generally prefer KDE I tried Kubuntu 6.06. It was nice but I experienced problems with adding and updating software. I replaced it with Ubuntu 6.04 when it was available and I was very impressed. Even though I do not prefer Gnome, everything worked great. I started using it as my default OS and still do.

What I want to bring up may seem strange to some, perhaps not to others. Since I am an "old" Linux user I am familiar with being able to choose to install multiple desktops and applications during install if I wish. One thing that seems odd to me is that *buntu seems to create the impression that one must install Ubuntu if one wants Gnome, Kubuntu if one wants KDE, Xbuntu if one wants Xfce, besides Christian, Muslin, Satanic, Flux and etc.

I really can see that having a simple live/install CD is a huge plus, but I also wish I could easily install everything I want at install and not be forced to add the pieces later. With my chosen main OS I like to install KDE, Gnome, Fluxbox, multiple editors, web browsers, and generally anything I choose.

The one project I have seen that does this is not even on the Ubuntu homepage and not up-to-date.

I would like to ask that it be considered that some users would like a 1 or 2 DVD install that would allow for multiple user choices at install.

### Post by mivo on 2007-11-04
The DVD downloads are not really advertised much, but as Rinzwind said, they are available for download. I think for most people install CDs are sufficient and they are a little easier on everyone's bandwidth, too.

If you look around a bit, it is mentioned in several places that you can install any desktop environemnt in any distro, so Kubuntu is just Ubuntu minus Gnome plus KDE. I agree that the sheer number of slightly differently named choices may be confusing to someone who is either new to Linux or the distro. Personally, I fell it might have been clearer to just offer CDs that are called "Ubuntu 7.10 + Gnome", "Ubuntu 7.10 + KDE", etc.

The Christian, Muslim, etc. pp. distros are spin-off, they are not official. Official are only Ubuntu, Kubuntu, Xbuntu, Edubuntu and Gobuntu (I hope I didn't forget one :)).
