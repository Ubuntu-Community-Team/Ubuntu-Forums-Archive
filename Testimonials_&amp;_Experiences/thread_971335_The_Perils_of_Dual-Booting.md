### Post by WaeV on 2008-11-04
No, I don't have technical problems, I just get used to features from each operating system that I miss in the other.

**Ubuntu's Annoyances:**
Simultaneous right and left click = middle mouse click (what's up with that?)
Clicking the center mouse button does not activate the scroll-by-moving-mouse feature
Clicking the center mouse button in a text window (or left+right) = paste
Highlighting something and pressing backspace is the equivalent of the cut function (combine with above annoyances and my random-mouse-clicking habit)
Firefox 3  (FF2 was much more stable for me)
Gimp != Photoshop
Video editing capabilities are meh
Lack of easy to find/use freeware apps for uncommon needs - converting .avi to .mpg, etc.  (WINE lets you use Windows' tools though)
Problems with playing .wmv files (Hey Microsoft, want to give us your proprietary video codec for free?  kthxbai!)
DirectX doesn't work so hot in WINE (Specifically Halo PC)

**Windows' Annoyances:**
Lack of terminal (command prompt is so inferior)
Only one bar
Lack of cube
Susceptibility to crap from the internet
Slows down over time
Lack of K3B, the best burning software I have ever used
More freeware for obscure purposes; less solid, well developed freeware (ex: K3B for linux and Winamp for windows are well developed.  "AVI WAV OGG Converter" is not well-developed.  The good windows stuff tends to cost $$.)
The Wings 3D window has to have something called "erlang" open for it to run, which takes up space in the task bar.  Plus the window opens in the lower corner of the screen instead of maximized how I leave it.  (Somewhat specific, I know, but hey!)

> **WaeV said:**

Clicking the center mouse button does not activate the scroll-by-moving-mouse feature
In Firefox, enter about:config, search for "general.autoScroll" and double-click where it says false (should change it to true).
> **WaeV said:**
> 
Clicking the center mouse button in a text window (or left+right) = paste
Highlighting something and pressing backspace is the equivalent of the cut function (combine with above annoyances and my random-mouse-clicking habit)
I find the middle-click to paste obscenely useful, but you clearly have different preferences.  Whatever floats your boat.
> **WaeV said:**
> 
Firefox 3  (FF2 was much more stable for me)

If you're still using Hardy, try "sudo apt-get install firefox-2".  Otherwise, you can try downloading [this file]("http://www.mozilla.com/products/download.html?product=firefox-2.0.0.17&os=linux&lang=en-US") and unpacking it.  Then execute the file called "firefox" inside the folder.  That worked fine on my system, but I don't use Ubuntu, so no guarantees.

By the way, you can run BASH or Zsh in Cygwin on Windows.  There are also alternate desktop shells (think window managers) available, though I don't like any as much as dwm.
