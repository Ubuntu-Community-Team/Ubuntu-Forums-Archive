### Post by Giovannino on 2009-04-28
I installed Ubuntu 8.10 2 months ago after using Windows XP for years. It took a little getting used to, but not much. Everything I did with Windows I can do with Ubuntu, I have just had to learn the subtle difference between applications. I was not able to install my HP Laserjet 1005 printer in 8.10 (it has a non-removable paralell cable and the HP driver for Linux supported only USB), but with 9.04 and about an hour of time I got it up and running perfectly with the paralell, so driver support seems to be improving.
I think the biggest problem with Ubuntu is not Ubuntu, but hardware and software manufacturers' lack of support, which seems to be changing. 
If someone started using computers with Ubuntu, I doubt that they'd gravitate towards Windows, but many people gravitate to Ubuntu from Windows. I think that Ubuntu is more intuitive, and has a wide range of easy to use applications available (mostly free or inexpensive), though you can make it as complicated as you want, if you want (you're less restricted).

### Post by Giovannino on 2009-04-29
I'll add that looking around this forum, many people are angry about their installation. Not unhappy, ANGRY. There are many ways to try Ubuntu without a commitment, and I recommend doing so. Things aren't "plug and play" as much in Ubuntu, and some hardware and software manufacturers intentionally don't provide compatability with "non-Windows". Also you do have to be willing to spend a little time learning if you want to do more than look at websites, though my wife just jumped on the computer after I first installed Ubuntu and opened Firefox to connect to the internet AND used Open Office to do some translation work, and she's got two left feet when it comes to computers. I have found the forums and bug reports super-helpful when something didn't go perfectly, and things never go perfectly, as they never do with any OS.
