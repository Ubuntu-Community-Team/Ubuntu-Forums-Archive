### Post by John L on 2008-04-25
I have to say that I'm very impressed with the final version.  

I had been using hardy alpha for a little while but after alpha5 I had along list of issues - from "hal failed to initialise" to failure to run the live cd.  This lasted up to the RC version!  So I hadn't any great expectations for Hardy final.

Just for something a bit different I installed xubuntu 8.04 and was more than pleased.  So I decided to have one more go at ubuntu 8.04 and I'm very glad I did :guitar:.

My 8600 GS graphics card worked fine after installing the proprietary driver, a version of flash installed no problem and I'm just after installing java and that works fine as well.  Sound works straight off. 

I do know that others are having problems and I've felt the frustration.  Keep the faith!  

John

Not everybody is having problems. 64 bit Ubuntu 8.04 is running fine on my Inspiron 1501. I knew I was going to need restricted wifi drives, so I had the ethernet cable plugged in before I booted the install cd. Feisty and Gutsy were true learning experiences on this laptop and knowing what to expect, made the install easy. 

This was a "works out of the box" install....literally! The box that the install came in was a 450Mhz PII that has been running the beta for three weeks. Transmission and the default cd burner made me a perfect install cd for the 64 bit LTS. 

I do feel for those having problems, but I sure ain't one of them.
