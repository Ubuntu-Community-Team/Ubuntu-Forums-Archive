### Post by OrionXIII on 2012-11-01
Well, Ubuntu just managed to take me from being a 100% Ubuntu cheerleader to abandoning it.  I've been a HUGE Ubuntu fan for the past 4 years and have despised Windows since they created it.

However, The 12+ series has been a nightmare experience for me and I simply cannot be reinstalling my OS two to three times a day. I'm about to wipe my drive and reinstall everything - assuming I can get Ubuntu 12.04 LTS up (and keep it running) long enough to GET my data off it - and go back to Windows 7.

Every day something else disintegrates off of 12.04 LTS. I tried 12.10 LTS but that crashed so frequently that I spent more time clicking on cancel (attempting to send crash reports, ironically, crashed) then I did doing anything with it. Today for instance, my NVIDIA drivers just stopped working. In fact, the video drivers in general stopped working. So after patching THAT together enough to get a lovely 640x480 screen - although Ubuntu still drew the desktop like it was 1280x1024 - A lovely zoom - I managed to reactivate it enough to work. Which I had to do because Ubuntu decided to stop displaying menus. Always fun to try to guess where the right option might display were it drawing menus.

It lost the ability to even FIND my second display.

This is a very clean, basic Ubuntu installation - All I've done is install it from the latest download and then do the latest updates and it explodes. I've added exactly 3 pieces of software - VMware, Truecrypt, and MyUnity. 

This is all AFTER it managed to wipe out my one NTFS drive containing decades of photos and programming code because it failed to copy files from a totally different partition to a VMware Ubuntu machine. Still no idea how THAT happened.

So, after reinstalling the VMware Ubuntu machine and beginning to add software packages to that, 12 dies in the middle of one install, requiring me to cycle the power - which destroyed Python on the VM installation, requiring me to install it AGAIN. I've been using Ubuntu for 4 years without a single crash - but 12.X has generated more crashes in the past week than I've since since Windows ME.

Which I suppose I should've expected since I've been unable to get 12.X to copy files reliably from one drive to another since I first installed it. Usually it gets part way through and then fails by disconnecting the drive automatically, claiming the file doesn't exist, and then automatically reconnecting the drive (and popping up a helpful window showing me the files that are in that drive).

Sorry. I'm done with Ubuntu. I'll keep using 10.04.4 as my main working machine in a VMware setup on Windows 7, but whoever is in charge of the Ubuntu project and design needs to really look at what they've done.

The entire Unity concept should be abandoned. If I wanted to type out the name of every piece of software I wanted to run, I'd run from a command line. If I want Amazon links 'helpfully' suggested to me for every piece of software I want to run, I'll go shop at Amazon on their website. The 'Dash' concept kinda makes a mouse useless. Essentially, we're back to DOS, except with colorful spinning, jumpy graphics. Thanks, I'll pass.

So congratulations, folks. You've taken a loyal follower and made me so angry that I'm giving up and going back to an OS that I despise - but at least it runs. In a stable fashion, which is about my only requirement.

Orion
