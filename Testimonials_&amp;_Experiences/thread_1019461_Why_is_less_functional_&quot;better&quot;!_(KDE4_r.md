### Post by helwitch on 2008-12-23
So I borked X on my 8.04 install, and have a new 500GB hard drive for my laptop that I'd been meaning to install ANYWAY. I figured I'd use the Kubuntu 8.10 live cd to do the backups I need to do, to play with 8.10 again and see if a bit of time has dulled my annoyance at it, made it seem more logical. And no, not at all, in fact I'm finding new things that are just STUPID!

For example, the livecd doesn't have a graphical partition management system installed. So, I fire up adept. First thing I see is that the screen real estate is being wasted with big shiny icons. Well, meh, whatever, pretty is nice. so, I type part into the search box and hit enter. Huh, that's weird, no results. oh, wait, that's because I have to click each software category to see the results FOR THAT CATEGORY. *sigh* Well, I suppose that's a useful feature for people who don't know what they specifically want. Surely there's a way to make it show all results? No, hmm. Maybe there's something in settings that lets you adjust it? There....is no settings? No way to tell this program 'look, I just want to see everything that matches my search, even libraries!'? *sigh*

Why is it that "better" seems to equal more dumbed down, less available functionality? I run linux instead of windows because I don't want an operating system that tries to manage me, tries to keep me from doing what I want because some developer decided that most people trying to do that wouldn't know what they were doing and would break the system. I do know what I'm doing, and if I don't, then I'll bloody well figure it out when I'm fixing my broken system. Make it so I have to actively enable the possibility to break my system, but leave me all functionality and access! Sure, fine, make the initial default setup user friendly and over-protective, but leave the less simplified, more powerful control available without having to resort to the command line! If I can do it by command line, I should be able to do it by GUI!
