### Post by AffiliateX on 2010-10-21
My Ubuntu 10.10 Maverick Meerkat experience thus far has been excellent. As a long time windows user I had always been interested in trying Linux but thought the task would be too daunting. Then I heard about Ubuntu. My first experience with Ubuntu was with 9.04 Jaunty Jackalope which didn't turn out well since the wifi drivers were a pain, so I gave up on it (I was a very average  lazy windows user). Then sometime thereafter curiosity and hate for windows struck and I tried 10.04 Lucid Lynx and I was extremely impressed.

     Since day one of trying Lucid Lynx I knew I wasn't going to use my windows partition. Ubuntu had the speed, reliability, support, and sexy look that windows never had. I am now 100% Ubuntu. I'm just surprised I put up with all of Micro$ofts nonsense for all of these years. Windows never felt safe, reliable, or fun (all which Ubuntu provides).

 The only cons of my experience with Ubuntu is, of course, lack of Windows software support. But that was to be expected.
 
     I am definitely enjoying my cup full of Ubuntu. Also I'd like to thank all of the Ubuntu team, and users of the ubuntuforums community for making Ubuntu my favorite distro.
