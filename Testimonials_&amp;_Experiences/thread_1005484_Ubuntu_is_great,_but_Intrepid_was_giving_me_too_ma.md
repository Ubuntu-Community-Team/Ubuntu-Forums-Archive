### Post by honda882000 on 2008-12-08
I have a new Compaq laptop with an Atheros wifi card.  I've been an Ubuntu user for about 2 years now, and love it, but for my new Compaq, not only did wireless not work "out-of-box" (Madwifi had to be manually compiled), but I was also suffering from Kernel Panics (computer would freeze with blinking caps-lock), which seems to be a very common issue with version 8.04+ for some reason.

I tried OpenSuse, but still preferred Ubuntu, even with the random Kernel Panics, but yesterday I installed Fedora 10 for the first time, and its great!!!  Not only does my AR242x card work out of the box, but so far I have not had any freeze issues.  I will keep this post updated with any issues I have, but since I could not find a solution for my Ubuntu freezes (Kernel Panics), I would recommend anyone with the same issue to give Fedora a try.  I'm very impressed, even though I will try Ubuntu again once these bugs are fixed.

### Post by gypaete09 on 2008-12-10
After various attempts over the last 8 years with various linux flavors, I definitely quit the Windows world and run only Ubuntu (Gutsy) for more than a year now.
Since I have only one decent PC, and because I don't believe that a second "play-PC" under Linux would give me enough everyday experience, I use Ubuntu for my one-man-company's production:
email (Thunderbird) web (Firefox) Office (Openoffice) scanning / printing / fax (HP-CUPS) and invoicing.

Invoicing, for the moment, has to run under Virtualbox and XP until a national tax-office-validated version of a Linux compatible package exists.

Lucky for me, I was a tech-support on Novell and MS NT / W2000 etc. before, so I am used to pulling myself out of troubles, but the total freezes i experienced after an attempted online update to 8.04LTS have made me wary and I continue using 7.10 after reinstall.

A good point: I have experienced that the Ubuntu forums *always* gave me the solution for my problems, quite the contrary on M$ paid support sites.
Also, my long-forgotten (only 3 day) basic Unix training 10 years ago, made a crash-return in my mind when several problems had to be solved all in console mode - fit again, but still a Unix newbie.

My plans: I'm slowly preparing for a clean reinstallation with Intrepid, but still hearing about freezes makes me hesitate.
Again, some of us have to use Ubuntu *every day* for *serious work* if we want to develop some real-life competence, which is what I try to do.

For the moment, I have not switched any of the PC's of my "simple Windows user"-type clients to Ubuntu, just because I will have major difficulies explaining when they might have issues with Ubuntu after updates.. (they already have these under XP and Vista, so where's the Linux "advantage" to the "dumb" user?)

Last not least, the local computer shops all sell PC's with Windows preinstalled, they aren't allowed to do otherwise. this is sad!

Hopefully, the future will see Linux everywhere, but trouble-after-update issues do a lot of damage to it's overall reputation.

Dear Linux developers: keep up the good work! - I'll try to do mine.
