### Post by MeditatingHamster on 2012-07-10
I've now been using Ubuntu 12.04 for 1 month on 6 PC's. I'm a desktop support tech and have spent over ten years supporting MS O/S's in almost all incarnations. Every now and then over the years  I tried a few linux distros because I liked the philosophy behind linux but kept getting stuck with command lines/crashes or lack of flash support etc.) One look at Windows 8 pushed me to have another look. Unbelievable! Within four weeks I have an NFS server/Webserver/DLNA server, an old lappy used as a picture frame and dynamically updated using Ubuntu one. A MythTV server with a freesatHD card in the livingroom. The online community is awesome for tutorials! The only issues I've had so far have been:
a) lack of ability to easily create web shortcuts (now solved with some launcher creator app from the software centre)
b) lack of ability to hide the grey notification bar (burn-in on my plasma worries me!)

Overall it's been an amazing experience so far and an eye opener. If Ubuntu stays this good, I will never return to Windows again (except for my job, :-( )
