### Post by chrismoore on 2006-06-23
I just finished installing Ubuntu Dapper Drake on my PC and I am amazed. I have a PC I built myself with an Asus K8N motherboard, Athlon64 3200 processor and 512MB of memory with SATA and all the goodies that the average man needs. 

Configuring Windows XP you not only have to install the OS, but afterwards you have to install a CD full of drivers which came with the motherboard to get the integrated sound and ethernet working, then on top of that you have to install some kind of office suite to be productive, and also install drivers for the video card. 

With ubuntu the OS took half the time of Windows to install, was almost totally automated, and installed built in drivers for everything but the printer.  OpenOffice was installed along with it, so no worries there. I set up my printer shortly afterwards with little difficulty, though I must confess that Lexmark's support for Linux is not so good. 

I've taken time to learn a handful of commands and am able to do all the same things as I do in Windows XP.  I am amazed - Ubuntu is actually EASIER to set up than Windows is. This could easily replace Windows. I set my computer up as a dual-boot system, which was easy with built in partition resizing btw, but I see little point in it.  How long will windows last on this PC? I simply cannot say.  I don't particularly care about watching DVDs on the computer, or playing really sophisticated games, because I have a playstation 2 and a home theater to meet those needs. From what I've read if I REALLY WANTED to do either it is possible with linux and some elbow grease though.  

On top of all that it seems to perform much faster. I don't have benchmarks to prove it of course, but it just seems to be more responsive. 

My compliments to the developers of this system.  I can hardly wait for the next big release. I hope to really get to know this new OS and really expand my horizons. I hope all of you have experiences that are every bit as good as this and better.
