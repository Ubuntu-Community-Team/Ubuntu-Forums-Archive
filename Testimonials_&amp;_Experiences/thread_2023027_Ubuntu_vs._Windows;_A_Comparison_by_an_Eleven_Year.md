### Post by thatguruguy on 2012-07-11
All of the computers in my house run Ubuntu, or a variant. The one my 11 (soon to be 12) year-old  daughter uses is set to dual-boot Ubuntu and Windows. I've noticed that she rarely boots into Windows, however.

When I asked her why, she pointed out that Ubuntu is easier to use. But she didn't stop there. In her words, "Ubuntu is like bubble letters, and Windows is like text. Bubble letters are more interesting and more creative. And more fun."

Now that Canonical no longer uses "Ubuntu: Linux for Human Beings," they need to come up with another catchy slogan. I propose:

**Ubuntu: It's Like Bubble Letters.**
