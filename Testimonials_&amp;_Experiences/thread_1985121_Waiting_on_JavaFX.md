### Post by Unterseeboot_234 on 2012-05-22
I'm stuck on Ubuntu 11.10. I could build Cinderella, a custom ffmpeg, a custom ImageMagick, perl scripts, support for H.264 and other installs. We tried the upgrade feature to Ubuntu 12.04 on one machine and that wiped out a lot of built software and Wine. The post-upgrade was not fixable at all. We had double-nested folders in the file system and the desktop menus had double menu items. I blame the doubling on the Language Support. In retrospect there probably isn't a script that could upgrade correctly with all of my customization.

I did not lose any data because of that upgrade.

I don't blame Ubuntu -- which I use exclusively and the office computers seem to always to boot into Ubuntu over Win7. I re-formatted the drive with the botched 12.04 and re-installed 11.10 from CD. I will rebuild the software I lost, no problem, it just takes some time.

Oracle has promised support for JavaFX on Ubuntu at the end of this year. Right now for Linux JavaFX is beta and 32-bit with limited functionality. If and when Oracle ever gets their act together, I will install whatever Ubuntu will support JavaFX and try to stick with the Unity interface.

I've had a great past and will have a bright future with Ubuntu.
