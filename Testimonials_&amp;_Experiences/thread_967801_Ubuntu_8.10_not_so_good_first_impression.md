### Post by kian.tern on 2008-11-02
Hi,

I've been using Ubuntu since 7.04 (Used Fedora and Debian before).
The past 3 releases were good, although there were some minor issues, they were good.

Unfortunately 8.10 is not the case for me.
I did clean install on my Laptop and at work, both PCs were running 8.04 perfectly.
The laptop had issue with the installation time. It took about 2 hours to do the install.
The desktop at work for some reason couldn't connect to the network (DHCP) during the install.

The main problem I now have on both of the computers is the Window Manager.
Metacity and Emerald really piss me off. They both flicker, buttons become invisible, general graphical artefacts appear now and then.
Compiz performance is awful.
For some reason the entire desktop seems blury.

It might be an issue with the nVidia video driver ( I still didn't test it),
but still if you have a driver included in a distro it should be stable and tested. Even just some sanity checks on 5-6 different video cards.  

Same problem I had with 8.04 still exists, I can't configure network manager to use pptp vpn properly. 
I still have to configure pptp tunel by hand and keep the password in an ASCII text file.

And general the system feels a lot slower when 8.04 does.

I'll keep 8.04 on my home PC for some time and I'll give 8.10 a chance to improve with time. 
But if it doesn't happen, I guess I'll have to switch to a more stable distro (still need to find it though, LFS anyone :lolflag:).
