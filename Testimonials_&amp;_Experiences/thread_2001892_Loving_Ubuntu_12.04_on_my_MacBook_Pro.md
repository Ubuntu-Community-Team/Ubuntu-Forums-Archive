### Post by virucidal on 2012-06-12
Just to summarize:

If you want to use a USB stick to install Ubuntu 12.04 on a Macbook Pro 6,2 or a Macbook Air 1,1, you have to create the USB stick using Ubuntu 12.04 "Startup Disk Creator". No other option I tried worked. I tried:

UNetBootin
Disk Utility (as per Ubuntu instructions on their website)
Refit
Some crazy partitioning tricks (ugh, this failed and took forever)

This is unfortunate for people that only have a Macbook Air (without the external CD drive) and want to install Ubuntu on it, for they need to have access to Ubuntu 12.04 to create the USB disk. I first installed Ubuntu on my Macbook Pro 6,2 using a DVD, and then used Ubuntu 12.04 on my Macbook Pro to create the installable USB stick for the Macbook Air 1,1.

Regardless, I am delightfully impressed with how well Ubuntu 12.04 runs on our Macbook Air 1,1. Apple was convincing us that we needed to purchase a new laptop as *everything* was so slow on it. I tried many tricks to try and speed up OS X on the laptop (resettings SMC, clearing up cache, etc), with only slight improvements. Flash movies (Hulu, HBOgo, etc) were way too choppy and even copying files was a drain on the system. Now that the Air has Ubuntu 12.04 installed, everything is zippy. Flash is smooth, programs load quickly, and I never see the spinning wheel! Thank you for fixing my wife's laptop!!
