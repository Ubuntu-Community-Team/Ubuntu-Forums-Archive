### Post by cyclist14 on 2007-08-30
So, like most of you I came from a lifetime of using windows, I started out on Windows, I was 5 when Windows 95 came out and my parents had a Win 95 machine back then that I started using. I moved on to 98 and then 2000, seeing plenty of bsod all through the OS's. 

Then, XP came out, I grew very comfortable with that OS and did a lot with it.

Then I got Vista, which I am still using now, Vista is an OS that I have more out of necessity than choice because I need to run several flowcharting programs and Visual Studio for school.

Just today, I got Ubuntu 7.04 Feisty Fawn, I installed it on an older computer that I was able to get from my school for free and it works very, very well (except for the mobo audio outputs not working, I'll probably just buy a dedicated audio card anyway) The OS connected directly to my network as soon as I plugged in the Ethernet cable and it downloaded updates and installed them. 

So far Ubuntu has been awesome, it's really stable and doesn't hog system resource like windows.
