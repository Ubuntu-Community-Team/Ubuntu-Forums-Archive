### Post by jolilly on 2008-05-30
I'm a new user of Ubuntu.  I started with a VMware player install of Ubuntu Gutsy about 3 months ago and was blown away by the refinement and quality of the distribution.  Because I was so impressed with the quality of Gutsy, I decided to completely change my home desktop architecture from a pure Windows XP setup, to Ubuntu Hardy Heron as my primary OS, with a VMware Workstation virtual machine setup for Windows XP (there are a few apps such as Photoshop, Dreamweaver, Quicken and Creative Zencast that I still want access to).

At first I had a few hiccups with Hardy, but I've been able to resolve all of these with the help of the Ubuntu and VMware forums, and I now have an extremely stable setup for my desktop running Hardy as my primary OS.  I love hardy and will never go back to Windows as my primary OS.  Ubuntu has won me over!  I use VWware Workstation 6.0.3 for Windows XP and also for a "test" Hardy VM where I can test new apps and updates before applying to my primary Hardy environment.  The setup is more complicated than what I'm used to, but it is working great for me.  Once again, I'm very happy with Hardy and I appreciate all the effort that has been put into this distribution.

I think that Microsoft should be more than little worried about excellent Linux distributions such as ubuntu as well as Mac OS X Leopard.  These compete equally if not blow away MS Vista!  Thanks again to all the developers who have worked on Hardy.  It is excellent in my opinion and I am very happy that I've switched to Ubuntu!
