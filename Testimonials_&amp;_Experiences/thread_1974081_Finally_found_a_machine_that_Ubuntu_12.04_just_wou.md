### Post by Version Dependency on 2012-05-05
After installing Ubuntu 12.04 on four machines successfully (for family and friends), I had one last machine to update...my dad's computer that was still running 10.10.  Popped the live CD in and loaded it up and...whoa boy!  Screen was a jumbled mess and nothing was working properly.  

So I decided I would come here and start a thread here about how Mark Shuttleworth is forcing Unity on us.

Just kidding.  :lolflag:

I took out the Ubuntu disc and put in the Xubuntu 12.04 disc (I came prepared) and the install went flawlessly.  My dad liked the interface since it was similar to the old Gnome 2 interface on 10.10.  You gotta love Ubuntu and all its derivatives...there's an Ubuntu for every situation and every kind of hardware.  =D>
