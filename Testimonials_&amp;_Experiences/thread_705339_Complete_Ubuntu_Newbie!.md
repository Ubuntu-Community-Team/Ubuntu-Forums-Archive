### Post by madkid on 2008-02-23
Hello everyone! I'm new to Ubuntu and fairly new to Linux. Ubuntu is the second distribution that I have tried so far. And it works. I mean, it just works. I'm in love with it. Of course, I'm using 7.04 instead of 7.10 but I'm on dial up and I happened to have a 7.04 CD at the time of install. It beats Windows XP by far. I've only had trouble connecting to the internet with my linmodem. I had to use my mother's Win98 computer to download the martian driver. Even after that, it took me a while to compile the driver and install it. But I managed. Now I have almost all the codecs I need, Java Runtime, Wine, VLC and everything that came pre-installed. I'm even enjoying OpenOffice.0rg 2.2. The session I'm currently in has been active since yesterday morning meaning that this baby has been running for a little over 24 hours and it's still kicking. My last complaint other than driver installation was the fact that I have to use the terminal to connect using the 'pon and poff' commands. Otherwise, everything works great. I've already played runescape with it this morning after waiting all night for Java to download and install. I hope to be a full time user of Ubuntu from now on.
