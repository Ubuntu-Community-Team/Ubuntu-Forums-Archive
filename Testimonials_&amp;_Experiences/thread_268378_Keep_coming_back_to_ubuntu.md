### Post by NeoRedux on 2006-09-30
I have tried various distros, such as FC5, SlackWare, SuSE, and MEPIS, after all of which I always come back to Ubuntu. Although I have some problems with it (lirc *cough* *cough*), it seems to have the best user support base and a huge repository of files.

After using SuSE from version 8.0 to 9.1, I became aggrivated with the slow and trivial YaST2 system. The first other distro I tried was Debian. Debian introduced me to the .deb format and apt (both of which are fast and convienent) and the gnome UI. However, I re-installed often and the updates list became too long for my liking with few major releases. I then moved on to slackware, where I compiled everything from scratch. This went on for about 6 months, whereafter I found my self in "Dependency Hell" and gave up with no attempts at the .tgz format. Next I used Ubuntu, and it was by far the easiest to set up and use. Although I still try out new distros, Ubuntu is still the top of my list.

### Post by BoneKracker on 2006-09-30
If you really, really want to know, do this (you have to do all of the lines).

```
apt-get moo
```

```

$ aptitude moo
$ aptitude -v moo
$ aptitude -v -v moo
$ aptitude -v -v -v moo
$ aptitude -v -v -v -v moo
$ aptitude -v -v -v -v -v moo
$ aptitude -v -v -v -v -v -v moo

```
