### Post by nickdbliss on 2009-03-14
It has been a year since i first personally started using Linux. And i must say it has been a great experience ever since then. (I tried it back in 2000 as well but it wasn't a good experience at that time. Although i used to use it at University's lab PCs for doing programming in C and other stuff.Red Hat lol) 
I use my hp laptop primarily for doing all work. I was using Ubuntu dual booted with XP, until few days back, when i thought to do a complete installation of Ubuntu only. I was very impressed by Ubuntu that i thought to get rid of XP for good. But i must say it was rather an impulsive decision lol. 

I am in love with this game Counter Strike and we usually play it at university on LAN. I deleted XP and thought i will be able to make it work on Ubuntu with wine (Used wine-doors then playonlinux etc.Didnt do my homework of course!darn). But i couldn't. The game freezes after starting. I normally use open drivers for graphics so thought to try closed one but it didnt work either(ATI). So i was stuck with it and couldnt play CS with my mates. 

Eventually i thought to install XP again with Ubuntu, just for the sake of love for the game. So, i tried and damn my cdrom acts stupid. It is giving Input Output Errors while reading from CDs. Guess the lens is not good anymore (although i freshly installed ubuntu only few days back via same cdrom).Maybe I didnt notice it because i normally use Flash drives and my external USB hard disk for data transfers etc. Or maybe XP is pissed and got mad at me and now taking revenge for deleting it.haha

Anyways whatever it is, after this, i thought to boot XP from Flash drive like we can do it so *easily* with Ubuntu. So i searched for methods but there are not so many easy methods out there for XP like we have for Linux. I haven't tried them yet. Maybe i should try installing via LAN like i once installed Ubuntu on my friends laptop (PXE protocol). But then it was ubuntu haha. 

So bottomline is: I am stuck. I can't play CS. The game i love so much.damn. Seems like it turned out to be a bad decision to remove XP in the end. I should have kept it although i hardly use it apart from playing games. If only gaming wasn't an issue in Linux, it would be a perfect OS for me. Period.

Nick#-o
