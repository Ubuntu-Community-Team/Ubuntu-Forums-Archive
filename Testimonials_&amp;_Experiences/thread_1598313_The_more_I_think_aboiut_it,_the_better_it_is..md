### Post by Aspiring_Failure on 2010-10-16
I'd be the first to say I had some rather nasty first impressions of Ubuntu, -Sky high CPU usage, over half my memory seeming to have to be used for just browsing the web, spending forever looking for .config files that were hidden *without unchecking the show hidden files option*, blash blah blah.

But I think maybe my expectations have been skewered somewhat, by linux's generally being labeled as *super fast* and being compliant with dated hardware.

But labels like that were pawned a long time ago. Times have changed and Ubuntu competes with Windows 7, not XP. That Ubuntu STILL manages to have a smaller memory footprint than XP really is amazing. Especially considering how much sexier Gnome is.

On Ubuntu, I notice ~110 Megs being used without anything running, and ~180 - 190 being used with Firefox and a few tabs open.

XP, on the other hand, uses ~**200** at **idle**. And can go over 300 easy when browsing the web. This is without any services turned off, but there aren't any startup applications, either. If I turn off all excess services, I can trim XP down to about ~120, but Ubuntu still beats it by a bit. (It's important to note the average joe doesn't dig around the XP services menu looking for stuff to turn off, like I do.)

As for CPU load, I'm both disappointed and pleasantly surprised. Ubuntu does, for some reason, have an unacceptably high idle CPU. Versus XP's small 2 - 4% with services, and solid 0% without services, Ubuntu idles at 10 - 12%. But in my opinion, this is worth the trade-off of a sexier environment, 100x native customization, and to my surprise, how **quiet** it is. 

I've noticed on XP that my machine is quite noisy, every action; every click, drag, tooltip, etc makes the machine utter some noise. It's file indexing, I believe. You can turn it off, but launching applications and a few other tasks still makes alot of noise.
Ubuntu, though, is completely quiet. Always.

And e ven if it does suck up a lot of my CPU's time, I never encounter any stuttering or lag with basic tasks. If I wanna play a nice game, I can simply ope n it up in an OpenBox session and be lighter than air on resources. Which is just great.

As for the actual look Ubuntu, I'm also satisfied. The default Gnome theme is great, and all the extra effects offered by Compiz make every little thing *that *much cooler. There are tons of ways I could customize it, lots of themes and applications and such, and the best part is that It's all done natively.

With XP on the hand, I need to patch some files, change the registry, and download very suspicious looking software, to customize it to even 50% the level of Ubuntu's customizability. 


But the coolest part is that for something which is cimparable to windows 7 in terms of features and sexiness and such, Ubuntu runs fine on my old little machine which wasn't even fit to run XP. I simply love it.

It would be nice if I could reduce Ubuntu's CPU load, but one complaint from Ubuntu vs 10's of complaints from XP are very clear numbers. I'm glad I decided to try Linux, definitely one of my more satisfying experiments.
