So far Ubuntu is way easy to mess up and requires one to learn the commands that are for the terminal early, which is bad for newbies because one change of a file and render your video to shutdown - esp. those with External cards such as me. Altho when the screen freezes for some reason - you can ctrl + alt + backspace  to get out which unlike windows has no such feature at least when I used XP and you HAD to no matter what do an hard reboot if it came to that in the end..
 
Linux Ubuntu so far for me in general has been an uphill battle - besides learning a bit and reading this forums I wouldn't have been easily able to get back up to where I was.
 
The Ubuntu team seems to take on more than they can handle and don't really let newbies to linux know that compared to Windows after so long they are on their own and must continue in the path of learning the terminal commands, reading boards, asking linux friends, etc..
 
What I was looking for was more of a stable platform which wouldn't in a week or so suddenly get slow on me, one where leaving the computer with XP on it and it be a zippy operating system to only come back in about 5 minutes having to reboot because it got too slow.
 
So far when leaving Linux Ubuntu for that long to get a snack or another cup of soda and coming back to a nice fast machine without having to reboot. From being an early windows user I did bring along with me some habits such as doing a reboot every so often.
 
Many wonder why Microsoft Got to where they are now, it is not of their cruddy copycat, and such software but because of Marketing, if you don't get your product or name out then it isn't going to get out quickly and many will be left there under Microsoft's unstable software.

Linux Ubuntu when in stable mode and official releases will do a great job. While many gripe at Microsoft for getting the Operating systems they have out in years compared to half a year it is sad really because it doesn't matter how long it takes the only thing that matters is how great it is when it is finished..
 
Compared to XP I feel somewhat safe, when I was running XP I always felt like I was being watched by some hacker.
