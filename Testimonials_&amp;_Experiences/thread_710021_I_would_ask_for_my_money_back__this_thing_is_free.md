I'm not kidding here. Some people have the ability to figure out most of these things on their own. Some need to read manuals. And some fail to do even that.

Installing software on Ubuntu is _much easier_ than it ever was on Windows. The problem is, you know Windows and try to apply your knowledge on Linux. Sorry, that won't work. You have to free your mind first. Then look at [this](https://help.ubuntu.com/community/SynapticHowto).

Synaptic enables you to install any kind of software with just a few clicks. You will never have to search for download locations again. The packages are digitally signed and guaranteed to be free of malware. And as a cherry on top, you can keep your entire system up to date automatically.

Hmmm, I will not pronounce myself in this regard, as I don't have enough information, but you could have a look at post #20.

To the OP. The reason you need permission to install a program is because linux is thought as a multiuser OS. The user wth administrative privileges is called root and in Ubuntu a regular user can acquire such privileges temporarely only and only for administrative purposes.
Having root privileges all the time (as it has been in Windows for a long time) is a great security risk. This is why you normally work as a normal user on your personal directory (/home/user_name).

Software installation. I find the different centralized package management system you see in Ubuntu and other distros much better than the Windows model. Installation and updates can be performed fast, secure (it is from secure repositories, after all), efficient.

I understand your initial frustrations, as changes can be difficult at times, but time is a great teacher. PAtience and don't give up.
