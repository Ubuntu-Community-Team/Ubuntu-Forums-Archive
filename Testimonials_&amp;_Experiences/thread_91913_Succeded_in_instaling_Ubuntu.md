### Post by Mr_J_ on 2005-11-18
I instaled Ubuntu on an :
Asus A8N-E (Nforce 4 Ultra)
XFX Nvidia 6600GT 128MB PCI-E
512MB DDR400
AMD Athlon64 3200+


I am using Ubuntu in 32bit because of the missing parts in 64bit world.
All the problems I heard about programs that don't work and for example missing flash support. I'll just stay in 32bit land until the 64bit world gets to grow up a bit.
Everything worked out of the box, including my MSI PC54G2 which is based on rt2500 (ralink 2500) and my Logitech Cordless keyboard it was a part of the Cordless Desktop now not using that mouse just the keyboard, which worked very well out of the box. Just some keys without shortcuts defined.

So far the only thing I need to get working is a webcam, but I haven't tried too hard to make that work yet.

In a previous install of 5.10 I had to install my wi-fi card following the wiki about these cards which worked perfect. Just needed internet to do it so a cable was required.

Most extra normal tweaks to Ubuntu can be made threw Automatix.:D
