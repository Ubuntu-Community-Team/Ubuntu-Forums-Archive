### Post by uberdonkey5 on 2008-09-28
Well, I started 12 months ago and had this progression:

month:

0. installed gutsy on whole HD, very excited about 'open source' etc
1. realised I needed Excel to do Visual Basic for Applications and reinstalled windows and ubuntu as dual boot (70% windows, 30% ubuntu)
2. after initial excitement, and set up, spent most of my time on windows
3-6. used open office wordprocessor and nothing else, and just for simple documents
7. started to use ubuntu more cos start up time was quicker
8. properly configured ubuntu and found files (esp. video files) copied quicker
9. started to use ubuntu only for DVDs and music and word processing
10. tried to change HD partition and messed my computer completely (deleted grub by mistake). Used ubuntu disk to rescue files and reinstalled with 50/50 partition (most of my files were now being stored in ubuntu and I wanted more space)
11. started configuring ubuntu to work well with keyboard shortcuts, clean and pretty desktop, video editing software. Also realised it was easier to rip DVDs and to play ALL my DVDs. Reinstall of vista seemed bad and drivers not working properly. Also, with this reinstall, didn't have antivirus software so was worried about using internet.
12. realised that over past months, only went into vista when couldn't do something in ubuntu, only to find couldn't do it in vista either and then found a solution in ubuntu.

...having not used vista for several months now reflecting on dedicating my whole computer to ubuntu. Speed of boot and file transfer, ability to set up printers quickly, and download useful software. Starting to feel that windows inhibits me (I use it at work still).

THUS, don't worry about HAVING to use ubuntu. Often it is something small, like needing a bit of software, or being frustrated with anti-virus, that gets you into it. Use it when it is best, when it is not use windows! I think ubuntu (linux) is much more versatile, and when it is set up properly it does what YOU want and gives you more freedom! Maybe get a good book & you'll see what you can do.
