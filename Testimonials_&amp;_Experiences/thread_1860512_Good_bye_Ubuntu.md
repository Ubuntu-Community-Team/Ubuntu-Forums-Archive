### Post by 3rdalbum on 2011-10-15
I'm really tired of this thread, too. Especially when people complain about Unity's inclusion as though it's the ONLY desktop environment you can install.

Listen, people: If you can't take the time to get used to Unity (give it a week), then you DO have options. Install gnome-fallback-session, it's very similar to Gnome 2. These are really easy to do on Ubuntu, and just because the default desktop isn't like what it used to be, doesn't mean that you should change distribution.
