### Post by Newbie2910 on 2011-05-01
Hats off to the Oracle team that developed VirtualBox.

This is exactly what I needed, to run two Windows apps that I need for Ham radio and WPF application development.

After installing the Guest Additions, having an XP box running as a separate window is smooooooth... almost like the entire XP operating system is just another application running on my Ubuntu box (which, in reality, it is). 

I have been in software development since 1978, and have to say the Oracle team has done an amazing job with VirtualBox.  And, the fact that Ubuntu is able to host this without even breaking a sweat says volumes about how strong and robust Ubuntu is.
