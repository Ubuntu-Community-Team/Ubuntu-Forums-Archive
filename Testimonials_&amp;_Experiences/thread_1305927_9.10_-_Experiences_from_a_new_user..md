### Post by JorgeGon on 2009-10-30
New Ubuntu user here. Started using 9.04 a few weeks ago and am now using the Karmic Koala official release.

Below are categorically, my experiences with Ubuntu Karmic Koala. The good, the bad, and those that rest at the line between the two known as undecided.

The good:
* Boot up speed has improved greatly for me. It went from 3 minutes (9.04) to a mere 1 with Karmic.
* Login screen remembers my username. (Yay, less to type)
* Uses less memory. Right now I'm using all the software I used in 9.04 and it's using about 30% less memory.

The bad:
* Ubuntu Software Center. I liked 9.04's software centerish feature better because it had software ratings besides the software's name. This was good for new users because it gave them an impression of what's popular and possibly what's good.
* Software center again! I noticed prices were listed for some software.. where is this going?
* Still no out of the box music playback support! As with Jaunty, I had to let RhythmBox search for and then download files so it could play back MP3 format music. Seriously, why include it if it can't play audio out of the box?
* Still no proper intel onboard graphics support. Yes, I know it's old, and yes I know it's not much but on windows my onboard can handle Morrowind @ 20FPS.. Morrowind -- a huge CRPG. But not Ubuntu visual effects..

The undecided:
* Ubuntu One. 2GB of space for free.. upload and optionally share all you want within that limit.. unless you pay for more space. This is another feature that I wonder where will go.
* Empathy. Not sure why the switch from Pidgin to Empathy was merited, but I will stick with it for a while and see how it is.

> **JorgeGon said:**
> * Ubuntu Software Center. I liked 9.04's software centerish feature better because it had software ratings besides the software's name. This was good for new users because it gave them an impression of what's popular and possibly what's good.
* Software center again! I noticed prices were listed for some software.. where is this going?
The Software Center is still in early days. There are [many plans]("https://wiki.ubuntu.com/SoftwareCenter") to improve it. In future software developers are going to be able to sell software trough the center, which was called Store at first.

> * Still no out of the box music playback support! As with Jaunty, I had to let RhythmBox search for and then download files so it could play back MP3 format music. Seriously, why include it if it can't play audio out of the box?
It can play audio out of the box, but not MP3 Audio, due to legal issues. Install the [restricted extras]("apt:ubuntu-restricted-extras") from the software center and you'll be fine. [According to Wikipedia]("http://en.wikipedia.org/wiki/MP3#Licensing_and_patent_issues"), there are MP3 patents expiring until 2017, so until that year it would be not legal for Ubuntu to ship with those codecs in every country.
