### Post by Johnsie on 2007-11-02
I've been using Ubuntu since Breezy.  One day I was on the Yahoo message boards saying how crap Linux was. I'd tried Mandrake and Phat Linux in the past but never been able to get my wi-fi connection to work. Anyway, this guy said "Ubuntu is the best Linux I've ever seen". I'd never heard of this "Ubuntu" Linux so I went on the site to read about a new Linux for Human Beings. Back then Ubuntu was not anywhere near as famous as it is today.  Mandrake and Red Hat were the big players.  So I decided, WTF, I'm bored let's try this thing out. So I installed Ubuntu for the first time. It installed reasonably easy. I did find the partitioning part of the installation a bit confusing but somehow I managed to get it right. Once Ubuntu was on there I was easily able to get in to the networking options and get my wifi working. That was very important to me because it enabled me to get to help sites. So finally I was in a version of Linux where the internet worked properly.  I love trying out new programs so Add/Remove Programs was like Christmas come early. I was installing programs and trying them out and every new experience was exciting. When I needed to learn how to do something I used the #Ubuntu IRC chat  room, [http://ubuntuguide.org](http://ubuntuguide.org) and of course [http://ubuntuforums.org](http://ubuntuforums.org),. The fact that there were so many people out there willing to help me get used to Linux was great and was definitely a big part of what kept me using Linux.  The more time I spent on Linux the less time I wanted to spend on Windows. Sure, Windows had some great software but I just felt more at home now on Linux. I felt a bit safer, and I liked the idea of using apt-get to install software without having to go to download.com to find software. 

Since then the novelty of using Linux has worn off. I guess what keeps me going is the performance. Ubuntu does what I want it to do and it does it well. I also like the fact that  and the fact that I'm part of a bigger movement. Linux and Open Source is much bigger than Ubuntu but being an Ubuntu user I'm able to contribute to free software as a whole because most flavours of Linux share the same software. I've watched Ubuntu grow, seen great improvements and also seen some of the regressions.   

Where does Ubuntu go next? Who knows but right now things look very promising. I would like to see more Ubuntu machines on the high street. But I dont know if having a multitude of Linux users is the best thing because it would overwhelm at Launchpad, meaning that less issues relationg to me would be dealt with.

I want Ubuntu at work but I doubt that would happen.

### Post by scorpiusmaximus on 2007-11-02
I had been contemplating jumping on the bandwagon for some time. I love the whole premise behind Linux and in particular, Ubuntu; Free, versatile, powerful, easy to use, the community fellowship, contribution to development, incredible choice of software packages, etc. I like everything about it. 
I put Edubuntu on a dual-boot system for the kids (the 17 year old will not do Linux. He is anti-everything. I am not surprised) Everything but the printer(Lexmark PrinTrio Photo 1350 Multi-function) works on that one. The 8 & 9 year old will not use Windows now.
The problems I am having are obviously the part manufacturer's; Brother, ATI & Lexmark.(drivers and support) They need to get their act together. I will remember this when I purchase products from now on. 
If I could get my printers and my video working, I'd leave windows like a bad dream. They are missing out on, and alienating, a growing population of consumers that are leaving Windows' offerings.
These are, unfortunately and obviously, of monumental importance on a computer, and in particular my small business. It would be very costly to replace video cards and fairly expensive multi-function machines to run a free OS when everything does work in Windows. It's hard to justify that expense, in this context. I still have a few good year's of productivity left on my equipment.
However, I do plan on building a new system soon and compatibility with Linux is going to be top priority. So sorry ATI!.  
You know, this is disheartening, too. I run AMD in all of my computers and I was rooting for ATI/AMD after the merger. They have the potential to do great things but I can not support companies that do not support.(Scratch my back and I'll scratch yours)

### Post by _aXXe_ on 2008-05-08
I started with 7.10 Gutsy and was very happy with the support for my printers. I couldn't upgrade to Hardy because of some errors in either the synaptic package manager and/or the normal updates. I did a fresh install of the Hardy and though the change has been relatively painless, I did notice that some of my wine apps no longer work. Neither does my Lexmark E238 printer which gutsy recognized at install. The printer works well using the HP laser jet as the driver and I'm still looking at the 16 and 32 bit windows apps I want to use. I'm sure I will figure out the problem in time. There are no issues with the instant messenger or Skype communication programs, and I'm currently using the Pan newsreader and Xirc which work flawlessly. Envelope Email, which is highly configurable for spam and phishing, works to perfection as does the Firefox web browser. The office programs are excellent. All in all a very good release that shows great promise for things to come. 

I specifically enjoy the lack of virus, Trojan and Adware issues with LINUX. Since I'm behind a router, I no longer need extensive repair and protection programs as LINUX is not as yet a real target for those that wish to cause harm. Having checked my footprint for open ports and integrity at grc.com, I am invisible to the outside world because my system does not respond to probes therefore an instance of a rouge bot being an issue is nil.

Games for Windows continue to be a hassle but that a is wine/cedega/crossover issue which will be worked out in time as more great games will be ported as LINUX use increases. 

Having tried Red Hat, Mandrake and Linux for dummies I can honestly say that Ubuntu is the best LINUX OS I have found.

The support is excellent both through the forums and the IRC channel. I have recommended the OS to many friends and several are in the process of using the Live CD for gutsy as a prelude to an actual install.

I would like to see more carry over from a previous version so we don't lose what already works. I believe that will happen as more people and businesses migrate to LINUX.
