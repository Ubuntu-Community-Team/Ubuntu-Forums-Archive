Well I want to start with one thing. Have a problem? Use Google.  
 

 I am a mac user. I have been using mac since 1995. I don't like Windows ok I know that it is not nice to say. But that is the truth. But What to do with all my friends that use PC and windows. It is not easy to get people to thing outside pc and windows. Telling them that you should by a mac is not the easiest thing.  
 So now I had a plan. I will learn Linux and get my friends to use that. It must be much better than windows. And they do not have to get rid of the machine that they already have.
 

 (I have some one that I want to get to go over to Linux. But he uses Autocad. So now I have to find out how to get Autodesk Autocad to run on Ubuntu. But that I will come back to that)
 

 Well the first thing to do is I have to start using Ubuntu. I don't have a pc. I have a mac with G5 power processors.  So I had to borrow a friends pc. So that I can try it out. So I did. I borrowed my friends laptop and installed Ubuntu on a stick. Well everything went without any problems. It ran and it was very nice. Well I had to give the computer back and I found out that I could not boot the computer without my stick plugged in the computer. What is the problem? What to do now. I can not give the computer back and it will not boot. I started to google. What to do. I found out the problem was grub. Ubuntu had installed grub on my friends computer and now it would not boot. Without my stick. Well the solution to my problem was super grub(my friend did not have a windows disk. That would be solution nr2). Super grub solved the boot problem. Wow I did it. I solved my first problem.  
 

 Well I liked ubuntu. And I wanted to try it out more. So I bought a PC computer. I must be mad. A laptop. But it was cheap and out of date. The reason why it was cheap.  
 

 Well now I have my own pc-laptop. Wow ubuntu 9.10 is out I have to download it an install it on my computer. So I did. Installed it. It worked. But the mouse did not respond like it should. And some other problems too. What to do now.  
 

 I installed ubuntu 9.04 and everything worked. What a bummer 9.10 did not work. But no problem.  
 

 I started using 9.04 and it worked well. Until the system crashed. And when I turned off the computer and started the computer again it did not boot. What it did not boot? I tried to put the install cd and start from that. It did not work. What to do now. I do not know. I tried to put super grub disk and start. And it worked. I could boot again.  
 

 But it was short. The system crashed again. And the computer would not boot. I tried everything again. Nothing. And suddenly it worked again. I do not know why. So I must say this is strange...
 

 Remember. I do not know anything about linux. So I am learning by doing.  
 

 So now the next thing will be using ubuntu and installing autodesk autocad. Hope it will work. And that I find out how to do it...
 

 If I get in to more problems I will post it here.  
 

 But I must say. Using Ubuntu is easy and I like it.

### Post by hashimoto on 2009-11-01
> **tigerøye said:**
> ...So now the next thing will be using ubuntu and installing autodesk autocad. Hope it will work. And that I find out how to do it...


AutoCAD is a Windows program and won't run directly on Ubuntu, but can be installed/run with a program called "Wine". Wine allows you to run Windows programs on linux. Depending on your AutoCAD version it may run perfectly or not at all, see:

[http://appdb.winehq.org/appview.php?appId=86](http://appdb.winehq.org/appview.php?appId=86)

To install wine, open terminal and run:
```
sudo apt-get install wine
```
