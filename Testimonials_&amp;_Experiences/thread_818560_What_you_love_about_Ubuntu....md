### Post by GOfree on 2008-07-04
Wow, there is just so much that I love about Ubuntu.

Some of this applies to Linux in general, and some applies to certain pieces of the Ubuntu pie. Some of these have been mentioned before, but deserve to be said again.

1) Free and Open Source software revolutionizes computing.

2) The Compiz Fusion 3D desktop is not only only super-sexy eye candy, but is also extremely functional for multitasking--flipping between applications and tasks.

3) Apt, Synaptic Package Manager, and Update Manager make installing, uninstalling, and updating programs so incredibly easy that they make going back to Window absolutely inconceivable.

4) The command line, Gnome Terminal, is great for letting a newbie (like me) get to know the computer and feel comfortable working directly with its configuration.

5) There is just a wicked wealth of great free software available. OpenOffice, Gimp, Inkscape, and Rosegarden are all top-notch applications to rival all that super expensive stuff I used to have to steal for Windows.

6) The internet security is amazing. I am not sure yet how it works, but basically, apparently, there is no more need for all that ridiculous anti-virus software (Norton, AVG, etc.).

7) It is so much fun playing with the desktop appearance, and there is so much you can do with Compiz Fusion, Emerald, Gnome, AWN, and others.

8) I rarely have problems with hung applications anymore, and when I do, Force Quit works like a charm.

9) One of my first favs was the Gnome Panel with the Drawer buttons--so cool.

10) I love my computer. For the first time it really feels like my own. It really feels like an extension of myself--as opposed to being an extension of some alien corporation.


Thanks everybody!

Thanks Toshibawarrior for asking the question.
