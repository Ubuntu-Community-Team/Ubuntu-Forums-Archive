### Post by BlazeFire247 on 2009-05-25
I never used to be that much of an Ubuntu fan. Well not until I installed it. 

The past few weeks, I was using Windows XP. I never used, or even heard of Ubuntu. 
A few days later, I download some different stuff and on Wikipedia, I found this article
about OpenOffice.org Writer and I saw something new: Ubuntu. My thoughts were 
"What is this?" I clicked the screenshot and saw new icons and Ubuntu's default 
Human theme. Of course, as a Windows XP user, I thought this was just a theme. Until
I was once again serching for random programs that I don't need.

So it always ends up in a Google search, and a Wikipedia article. There I saw it. A 
program "Running on Ubuntu". I wasn't that interested at first, but I was curious so I 
searched WIkipedia about it. I read about it and I was fully interested. Only it was an 
operating system and I was afraid that I'm going to lose all the files on Windows. So I tried 
to download a theme, like I usually do on every operating system I like. Only, it didn't 
turn out as I expected it. It's still different, and when I messed up with that theme patcher, 
I couldn't skin my taskbar. 

So it's time I do some research on Ubuntu, and found Wubi. I installed Xubuntu, it was fast! 
But I still wanted Ubuntu, so I did that. Un-installed Xubuntu. Boy was it slow (no thanks to 
Windows), but it was worth it! I am now enjoying the new desktop I have, with no problems 
at all.

I actually like the fade effect when making wallpapers, and I enjoy making wallpapers of 
my own. I also like how it's faster than Windows XP. Actually, I love a lot of things about 
Ubuntu: the keyboard shortcuts, the lock screen feature, the amazing log-in screen and more!

For me, Ubuntu is the best operating system that I have used. Thanks to Ubuntu, my new 
favorite colors are brown and orange :biggrin:.
