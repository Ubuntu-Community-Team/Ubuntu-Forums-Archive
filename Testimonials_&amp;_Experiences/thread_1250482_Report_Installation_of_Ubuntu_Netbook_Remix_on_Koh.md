### Post by SuperBaby on 2009-08-26
I bought my Kohjinsha SH6 about 2 years ago in Singapore. It came with genuine Windows Vista pre-installed. With a CPU of 600MHz, pre-installing Vista is definitely the manufacturer's mistake. The SH6 was crawling.

I then decided to remove Vista completely and installed Windows XP on it. It seemed to work faster but there was a bug. Every now and then the CPU would shoot up to 100% halting the whole system. I suspected the bug was hiding in the driver for XP.

I was just about to dig a hole and bury my SH6 before a friend suggested me trying the free Ubuntu Netbook Remix 9.04 (codename Jaunty Jackalope). For those who have not heard it before, Ubuntu is the most popular Linux distribution in the world. I saw some light when seeing "Kohjinsha SH6" was under the list of compatible models at www ubuntu com.

Result? Here it goes.... I have been using my SH6 running Ubuntu for nearly 2 weeks. It is fast and healthy. It is performing much better than what I expected! Most hardware is functioning better than expected too:

So far I have not experienced any system halt. It is definitely running faster and more stable than Windows. Until today I have not installed any driver. Kohjinsha Singapore does not provide any driver for Linux anyway. It is amazing that everything is working fine out of the box without the need to install any driver.

- Sound [COLOR=SeaGreen][OK][/COLOR]

- WiFi  [COLOR=SeaGreen][OK][/COLOR]

> Screen saver

Some Windows lovers would make a step back when hearing Linux. The Ubuntu actually comes with a very user-friendly graphic interface called Gnome. You can, however, switch it to the Windows-like desktop mode.

One more plus point... you can forget about installing any antivirus software. Linux does not require that. Although you may install the free ClamAV, it is mainly used for scanning emails in Linux-based servers.

To install Ubuntu to SH6, you need to burn the UNR image file onto a USB thumbdrive. Change the BIOS of SH6 so that you may boot from the USB drive. After that, the installation is a piece of cake.

I am so excited. I feel like just buying a new netbook. I think my SH6 is on steroid. The only thing I failed to do so far is to change the bootup splash screen to the ones found in www gnome-look org.
