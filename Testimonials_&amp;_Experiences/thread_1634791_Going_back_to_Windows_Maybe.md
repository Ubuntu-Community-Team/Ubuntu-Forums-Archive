### Post by blacktrance on 2010-12-01
Having had some experience with using Ubuntu, then Linux Mint, then Xubuntu, I have reached the conclusion that Linux is simultaneously too paternalistic and arcane. Arcane because so much messing with stuff is necessary, including sometimes fiddling with system files. Paternalistic (and this sometimes applies to the community as well) because it asks for the password when you try to install anything (even updates!) or modify system files (I know there's a way to fix it, but if it was a better OS, it would be easier, or better yet unnecessary). The community is paternalistic too, excessively concerned with security. It is not understood that if I want to do something, I want to do it, and asking me if I'm sure is only annoying.  Then there's my preference to MS Office 2003 over OpenOffice/Abiword and the others, and I play Windows games. Using Wine is even more troublesome, and even so it often doesn't work.

Then again, I do have to say the one positive thing about Ubuntu. It's much faster and less bloated. And that's the main drawback of going back.

### Post by 3rdalbum on 2010-12-01
It's really quite rare that you have to mess with system files; probably rarer than a Windows user would need to mess with the registry. What did you need to do that requires this?

As for your paternalistic claim: This is ridiculous. It's called "computer security" and it's the reason why we're not riddled with viruses. The virus and malware problem costs the world more than a small country's GDP per year!

Sorry we didn't help you break your system, but I personally couldn't sleep at night after telling you how to activate the root account, knowing that it's now so much easier for a remote attacker to steal your internet banking password. If you really still can't bear the thought of having to confirm system-wide actions by typing your password, then you need to install Windows XP or earlier - Windows is moving toward our system of permissions and separation of administrator and user. Vista and 7 have implemented it. Why? Because it's more secure and it DOES protect users.

My honest advice to you would be to get used to the tightening of security in computing, because soon it's going to be everywhere. Computer users can't afford not to practice good security, it costs too much NOT to.

### Post by blacktrance on 2010-12-01
I've never had to edit the registry in Windows. In comparison, I had to fiddle quite a bit with various system files to get Ubuntu to do what I want. 

The reason Linux isn't riddled with viruses is because it's not compatible with them. That is by far the main reason - not "security" measures. And besides, computers don't get viruses. Users do. If you're stupid enough to run something like "porn.jpg.exe", then that's your problem, not the OS's. I don't even use anti-virus in Windows, because I know what I'm doing. The fact is, if you use Firefox with the Noscript add-on (so it doesn't run whatever scripts the webpage wants) and you don't run stuff you don't trust, nothing will happen to you. Ever.

That said, Windows is moving in a bad direction too. Of course, I've completely disabled UAC on my Windows 7, but it's an annoyance that I even had to do that. A bigger problem is just how bloated it is. It takes far too long to launch something. Ubuntu is much better in that regard. And I like how customizable Ubuntu is.
