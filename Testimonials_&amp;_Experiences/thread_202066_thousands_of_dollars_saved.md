### Post by BuffaloX on 2006-06-22
In 1984 or 85 I got my first Amiga 1000. That computer rocked. Way better than any other system available at the time. Unfortunately the Amiga didn't last.
Then the alternative was a PC or Macintosh. I chose the PC. And found myself bombed back to the computer stoneage.
Windows sucked bigtime. Extremely slow, bad multitasking system, freezes all the time. But it did allow multiple DOS sessions simultanioously.
Windows has gotten better over time, XP is the best Windows ever. But that dosn't say much.
I have tried Linux a couple of times over the years, but given up because it lacked on software, and I found it hard to configure.

This time I'm staying, it's amazing what has happened in a very short time.
Several programs are even better with Linux than with windows.
Rosegarden for music, Blender for 3D, Python seems fun, and it just works!
OK Windows has good apps in those areas too, if you are willing to fork out thousands of dollars EACH!

Have I had problems?? YES there has been a hickup here and there.

But I guess part of the problem is, I have gotten dumber over the years using Windows.
Linux feels a bit like being back to the old Amiga days. I can actually do things with my system. I have options, I have access, I have freedom.

With the latest Java update my homebanking also works. This is important for Linux in Denmark, because 80% uses homebanking. 

So now I only need a couple of good games, then it's bye bye Windows.

A big thank you  to everybody contributing to Linux.
