### Post by r-senior on 2012-04-15
> **WindowsToLinux said:**
> I'm obviously still used to executable files and installing applications through a command line interface is completely new to me.
You don't need to use the terminal to install software. Have you found the Software Center icon in the launcher on the left side of the screen?

Terminal commands for installing software are often used in the forums because it's easier to help people that way but the Software Center is great for browsing and installing programs. It uses the same commands behind the scenes that you see quoted for the terminal.

One of the biggest differences between Windows and Linux distributions is that on Windows you download or buy software and run the install program that it provides. On Linux you use a tool, e.g Software Center, to locate, download, install and update your software in a controlled way. 99% of the time you won't need to download third party stuff and install it by hand.
