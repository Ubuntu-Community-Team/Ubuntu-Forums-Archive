### Post by Empath on 2007-08-11
I've tried various flavors of Linux in the past. Having tried Mandrake, Red Hat, Fedora as well as Windows 3.1,9x and XP I've never seen a system which makes it so easy to Install and remove programs. Now you no longer have to waste hours and hours looking for programs, it's just a button away. With the new improvements being made, you can even have a 3d desktop like Vista minus the 100+ dollar cost should you choose. It's definitely recommended if you're tired of dealing with spyware and want more security while still having an excellent desktop system. You can even play some of your Windows games on it.
