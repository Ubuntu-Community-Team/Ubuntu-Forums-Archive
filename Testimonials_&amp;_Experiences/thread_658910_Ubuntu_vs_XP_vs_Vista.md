### Post by getmeoutofhere on 2008-01-05
I&#8217;ve been messing around with Ubuntu for the best part of six months, so I feel that I can make some useful comments.  However, I am not tech-savvy.  I&#8217;m a computer user with above average knowledge, but I have my limits.  Don&#8217;t ask me to get your wireless card working with nsdwrapper!  My statements are, I think, very subjective.  

I started with XP, moved to Ubuntu (7.04), then Vista, and now I am back to Ubuntu (7.10) &#8211; with my second computer running Windows 2000.

In the meantime, I had two Dell laptops &#8211; first an Inspiron 6400, then an Inspiron 1520.  I couldn&#8217;t get 7.04 to load on the 1520, but 7.10 works fine.

The first comparison is between XP Pro and Ubuntu.  It&#8217;s difficult to say which is better.

XP plus points:

Runs Windows programs very well.  That might sound an obvious point, but I use Word a great deal.  

The Control Panel.  I like the Windows control panel.  Ubuntu doesn&#8217;t have an equivalent that I find comprehensible.

XP minus points:

I&#8217;m a sloppy computer user.  I leave loads of things open and I forget to defragment.  Over time, in my hands, the Windows operating system deteriorates.  It stalls, it takes longer and longer to close down.  

Ubuntu plus points:

System stability.  So far, I haven&#8217;t noticed system deterioration.  Starting up and shutting down are prompt. 

SSH.  This is a feature I love &#8211; it makes it so simple to connect computers, provided, I assume, that one of them is Linux.

Lack of extraneous processes.  With Windows there always seems to be something going on in the background, which seems to slow the computer down.  Sometimes drastically!  This doesn&#8217;t seem to happen with Ubuntu.

No need for defragmenting.  And apparently no need for anti-virus software.  Even less extraneous processes!

Ubuntu minus points.

Lack of a Word Processor that I like.  I don&#8217;t like Open Office, and I therefore have to run Word on Crossover.  Crossover is not perfect, and I can&#8217;t use Vertical Select on Crossover Word.  Ie selecting text to the left or right of a point on the x axis.

Lack of support for .vbs files.  I use VBScripting quite a lot.

Sound.  I don&#8217;t really understand sound on Ubuntu.  On Ubuntu it&#8217;s fine for listening, but for recording I would want to use Windows.  Though personally I wouldn&#8217;t recommend either of the earlier mentioned Dell laptops for serious audio work.

Lack of support for other specialist programs, that are only available in Windows and won&#8217;t work on Crossover.  I know, they&#8217;ll almost certainly work on Virtualbox (a magnificence piece of software, by the way), but I haven&#8217;t got a Windows program I can use on it.

Vista vs XP, Ubuntu.

I have Vista Business.  No longer installed on anything.  System deteriorated very quickly, and constantly shut me out of the internet &#8211; I had to reboot.  Vista seems very difficult to work with, and if I had to use Windows, I &#8216;d prefer to use XP - or even Windows 2000 or Windows 98 SE.   

While it's obviously true that Vista has a control panel, and runs Windows programs, it doesn't appear to be as usable as XP.  Although Vista doesn't stall as much as previous Windows versions, and it shuts down better, my overall experience of it wasn't particularly good.

On New Year&#8217;s Eve I got the message &#8216;NTLDS is missing&#8217; on a black screen when I started Vista.  That was it.  I couldn&#8217;t rescue the system, and I couldn&#8217;t find the Recovery Console.  Is there one?  I suspect it was something to do with the master boot record, but where could I type in &#8216;fixmbr&#8217;? I was able to save all my data, thanks to SSH and an Ubuntu Live CD.  Then reformatted and put Ubuntu on the entire disk.

Conclusion.

Will soon buy another computer, with Windows XP on it, and will then run two computers, one with XP, one with Ubuntu 7.10.  It&#8217;s not an either/or situation.  Currently I could do my work without Ubuntu, but I couldn&#8217;t do it without Windows.  However I would want to keep Windows use to a minimum, for very targeted use &#8211; because I don&#8217;t want the problem of system deterioration, which I&#8217;ve experienced on every Windows OS I&#8217;ve ever used. But maybe that&#8217;s just me and my bad computing practice? Though I will not, under any circumstances, use Vista again.

### Post by Niniel on 2008-01-08
> **Hightide said:**
> I defragment my windows partition about 1 every two weeks. It does reduce the risk of hard disk failure.

:)

No, it does not. Fragmentation has nothing to do with the health of a disk. The only thing is does, in theory, is speed up access to files since they can be read in one continuous block vs having to collect bits and pieces from all over the disk. And I very much doubt that that puts enough strain on a drive to speed up its eventual demise.
