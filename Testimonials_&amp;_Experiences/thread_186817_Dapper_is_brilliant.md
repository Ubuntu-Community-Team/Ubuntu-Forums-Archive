### Post by doloctona on 2006-06-02
Well, I guess you guys know that, but..

For a long time I've been feeling like I'd love to switch to Linux again (I used to use it back in the days of Redhat 5.2-7, then I played with Debian a little, Knoppix, Mandrake, SuSE, etc.. but I was never quite content).

Anyway, I tried out Breezy a few months ago, and it just wasn't quite there.. I always feel like things should _just work_, as they do in Windows.. call me lazy, but I just want to get to work (or entertainment) on the PC, and setting up Sound/Video support isn't my idea of fun any more - I spent a LOT of time doing this a few years ago when I first started using Linux, it was fun then, it's painful now.

So anyway, grabbed Dapper Drake with vague expectations that the following would just WORK:
- Wireless (did work out of the box pretty much in Breezy)
- SPDIF Coax audio output
- NVidia drivers (working PROPERLY with acceleration, not installed but still using the nv driver or something)
- Java installed with minimal fuss
- Netbeans installed with minimal fuss
- MP3 playback, movie playback, all using SPDIF output
- Skype
- Auto-mounting of windows drives, and the ability to view them from the desktop (in Breezy they were there, but only root could view their contents.. in dapper I can view them as my non-superuser.. perfect!)
- And finally, no crashing / instability

Guess what! it ALL just worked, perfectly, just like Windows, maybe even easier (for example I have two SATA drives using BOTH my on-board RAID adaptors, a Promise FastTrack 378 and a VIA something-a-rather - they JUST WORK!)..

You guys have done and continue to do a brilliant job.. I don't think I'll bother going back to Windows for a few days

### Post by ajgreeny on 2006-06-05
Couldn't agree more!

I've been waiting for the final dapper to appear, but got too impatient so installed a beta iso from a UK comp magazine, just to look at.

I've been using breezy since it came out without any real problems and was just waiting and waiting for dapper.  Then I saw it was on *Personal Computer World* magazine here in UK so got a copy and tried it.  Brilliant!!

After install there was a huge update (345Mb so if you have no broadband beware), but then I ended up with the full and final version to which I have added the *kubuntu-desktop* package and now there's no looking back.  It is faster than breezy, boots in about half the time it took breezy, and so far seems just top class.

I'm almost speechless with admiration for all you people who put together this whole thing.  It's a real MS beater, hands down.
