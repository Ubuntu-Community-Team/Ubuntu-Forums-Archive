### Post by mostwanted on 2007-09-29

> i installed ubuntu and it's working fine. but i increased my ram from 256 to 512
so now i cant hibernate. is there some way to increase size of swap partion withought having to reinstall the os.

Reboot with your Ubuntu Desktop cd and open the program gparted which is in Adminstration I think. You can use that to resize partitions. The reason you need to do it from the live cd is because you can't edit mounted partitions.
