### Post by coffeecat on 2010-03-16
I think you became an official Ubuntu user the day you installed Ubuntu in Virtual Box. :) In fact, if you managed to install guest additions in Ubuntu as a beginner I take my hat off to you.

Wubi is a good way to explore Ubuntu but there are some issues with it, so once you have some self-confidence, you might want to think about a proper dual-boot, that's with Ubuntu on its own native partition. It runs much better that way.

Whatever you choose, good luck!

By the way. For that one essential app that will only run in Windows, I've experimented with Windows running as a guest OS in VirtualBox with Ubuntu as the host. It worked a treat. It seems more fitting that way - Windows as the subordinate OS. :wink:

I wouldn't attempt Wubi, you'll run into trouble later, and we'll all be trying to solve your file system problems because of the NTFS. Go for a real install. Remove Vista. If you want to run Vista, run it in Virtualbox, or better yet, just buy a copy of Crossover Linux and run all your vista programmes in Ubuntu.
