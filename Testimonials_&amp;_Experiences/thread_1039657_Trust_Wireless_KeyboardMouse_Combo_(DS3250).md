I wondered... "Will a flashy wireless mouse/keyboard combo work with Ubuntu?".


**Yes.**


I nearly peed a little bit. Not only did it work first time, but *every* 'extra' button worked too. Multiple media control buttons (on the mouse, no less!), system keys ('Home', 'Mail', 'Favourites', and more)... Here's what I liked best:

The damn calculator button actually opens up the calculator!
The volume dial on the keyboard not only works, but has a nifty OSD to go with it!
The scroll wheel on the keyboard works like... well, i've never used one before, but it's already indispensable.

Obviously, i've only used this set on my own PC (C2D, 2GB, nVidia, 1.75TB), but at the insane price of £16 (yes, sixteen pounds), how can you go wrong??

I'm not going to go hunting for the person/peoples who set this up, but whoever you are: THANKYOU!
