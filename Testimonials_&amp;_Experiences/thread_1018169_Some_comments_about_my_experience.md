### Post by bgast1 on 2008-12-21
I by no means know much about Linux even though I have been using it now for about a year, maybe more. I've found that Amarok is the best music player for me. Banshee is pretty good but it doesn't do everything that I want it to do. Rythmnbox is boring but it works well. 

I like Compiz a lot, but don't know how to change themes yet, either in Emerald or the default way. Actually that is not entirely true, I change what is already there, but going to Gnome-look and getting them from there and installing is something I haven't learned to do. Changing Wallpaper is no problem.

Having said that about Compiz, I've found that when I want to watch DVD's I have to turn it off. Basically Compiz is just pretty. 

Open Office is a great package, at least to me. I don't miss Microsoft Office one bit. 

I run Linux almost exclusively. The only reason I have Windows on my machine is for games, and I have a Playstation 3 for that. I like to play golf and still play Links 2003, Haven't figured out how to install it with Wine. Hence for that I need windows. Otherwise, I would toss windows in a heartbeat. 

I wrote this mostly for those of you just starting out. If you stick with it you will get Linux to work. This forum is the best out there for Linux by far. I have been a distro junkie and out of every distro out there Ubuntu has always worked. The rest work, but they need more tweaking. Others are prettier, but in long run I always end up choosing Ubuntu because it just works. 

Also, synaptic is great but you really should get comfortable with using a terminal. 

I hope that these comments are helpful to some of you who just installed and took a look at Linux for the first time.
