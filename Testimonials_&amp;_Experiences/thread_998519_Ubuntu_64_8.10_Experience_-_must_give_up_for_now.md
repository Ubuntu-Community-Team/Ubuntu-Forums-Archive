### Post by pablolie on 2008-11-30
I have loved working with Ubuntu for a while now. 

For a variety of reasons I had to grow my system to high levels - I had to support RAID (at least for the Vista Ultimate 64 partition running business software) and 8 GB of memory. The system is based on an ASUS P5Q-EM motherboard with an Intel Core 2 Duo E8600. The motherboard supports RAID (I use mirrored for utter reliability) off the Intel G45 chipset.

Installing Vista Ultimate 64 was initially painful, but I must admit it is rock solid and driver and application support exists for all I need. 

I tried to install Ubuntu, both off the standard and the alternate installation 64 bit distribution disk, to somehow coexist with the Vista installation, but Grub simply could not get Ubuntu to start - after installation the system would never find the Ubuntu partition again and simply start Vista. Strange. In the end, I went a step too far and destroyed the Vista install.

Upon reinstallation (and deciding to momentarily forget about running Vista on a mirrored RAID) I just discovered that Ubuntu 64 unfortunately is too limited in scope. Flash does not work, making cruising the net cumbersome (I am aware one can fight that if determined, but it should be easier to run), and more importantly, I could not get a key application of mine to work (Squeezecenter), whether it is because the application developer or Ubuntu 8.10 I do not know.

But unfortunately for now I will have to run an Ubuntu-free home network, which is a shame. I truly look forward to deploying Ubuntu 64 once it comes more versatile. And i will miss the Ubuntu interface for now - I truly tried hard, but must move on...
