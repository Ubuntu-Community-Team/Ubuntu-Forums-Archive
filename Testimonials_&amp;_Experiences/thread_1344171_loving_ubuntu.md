### Post by JoePits on 2009-12-02
well windows 7 came out and i was still on vista and no $ for upgrading.  didn't want to be using an older OS cuz thats lame lol.

so i put on karmic koala on my laptop and it works great.  one problem with previous versions was shoddy microphone support which isn't a problem now.  i need microphone because i use voice chat and a java web based guitar tuner.

ubuntu looks real nice too.  and its fast.  i admit i still use virtual box to run a few things on XP.  including itunes to sync my ipod (which works flawlessly with the non-free virtualbox btw) and to play netflix on demand.

we have an older computer that had 8.04 on it and i upgraded it to 9.10 but its a little slower now, due to the fact it only has 512MB of ram...so i ordered some more ram for $13 which should fix that.


ubuntu ftw.
