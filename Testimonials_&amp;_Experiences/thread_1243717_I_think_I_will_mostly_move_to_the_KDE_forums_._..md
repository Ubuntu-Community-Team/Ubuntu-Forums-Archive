> I personally use Gnome. i think it nooby friendly. As far as Linux itself,it would fair alot better if updates would update firefox to newer versions and Kernel updates would stop kicking my nvidia drivers out.
It's not Linux that prevents you from updating.  For one thing most distros (including Ubuntu) do update the firefox, etc. versions.  They just don't update to major releases.  For example I'm using Fedora.  When I started using it we were on Firefox 3.5beta.  Now it uses 3.5.2.  However it will never use 3.6.  I can still install firefox 3.6 (or whatever the next major release is) on my own, just not through the repositories.  Repositories are for user convenience and part of that convenience is that they only allow you to install stuff that won't brick your system.

Now there is another way to solve this problem:
Update your entire OS (happening this October)
or
Use a distribution that has rolling releases (always upgradable to the latest versions)

However, none of the aforementioned problems are linux specific.  It is just how the various distros choose to do things.
