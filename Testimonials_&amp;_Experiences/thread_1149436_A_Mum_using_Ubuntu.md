### Post by joey-elijah on 2009-05-05
> **Twitch6000 said:**
> Get aMSN out of add/remove that will work better for MSN :).

aMSN is ugly, badly designed and a resource hog! Even the developers themselves would tell you that!

  The best 'MSN replacement' you can find is the brilliant, gorgeous and simple-to-use 'emesene' (say the name aloud ;) ) You can find it in add/remove, just search for 'emesene'. It's much nicer and if you're used to the official MSN application on Windows then you will feel right at home!

If you have a webcam, you can install a community-maintained version called Emesene-Crazy which has webcam support. 

I recently reviewed lots of different IM applications, and i would wholly suggest Emesene for anyone who only uses the MSN network. 

[http://d0od.blogspot.com/2009/04/msn-linux-im-ubuntu.html](http://d0od.blogspot.com/2009/04/msn-linux-im-ubuntu.html)
