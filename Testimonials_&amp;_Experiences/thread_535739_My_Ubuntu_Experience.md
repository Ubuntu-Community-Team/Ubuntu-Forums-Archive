### Post by -SpaZ on 2007-08-26
I acquired an ancient Gateway PC about 2 months ago and at the time it was running Windows XP.  I had been interested in Ubuntu for quite some time but I hadn't had a computer to experiment on.  After installing Ubuntu on it, my experience was nothing but positive.

I was quite surprised to find out that it runs relatively smoothly on 256gb ram and 80gb of memory.  There was only one drawback, it doesn't run as fast as I would like it to be, so for now it is my secondary desktop.  At the moment I am saving up my money to get an nice system76 Pangolin Value laptop.  I can't wait to be able to get it and finally have my new favorite operating system running as my primary operating system.
