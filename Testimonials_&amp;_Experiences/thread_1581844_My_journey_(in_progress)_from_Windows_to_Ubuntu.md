### Post by xenton on 2010-09-25
Hi Everyone, 

I came across Ubuntu whilst looking for something else and just asked myself "Why haven't I tried this yet?" So - I did.

Installing Windows was a nightmare on my current desktop as it is a 'fakeRAID' system. Windows does not natively support my RAID controller and figuring out how to install it took days.

Ubuntu was very promising it detected my RAID arrays right off the mark, and even offered to install GRUB 2 to my RAID array. However, despite this, it tried to install it to /dev/sda and this failed. A little disappointing and a bug with the installer on the live CD. However I pressed on with the installation and hit the forums for help. I tried a few things and kept coming back for help and after a couple of days got it working. I threw up a thread for other users to give something back, in case anyone should run into a similar problem.

Next problem was my printer was shared on a network and used by a couple of other computers and a laptop. A quick search and 5 minutes later it was all up and running - easy.

Then they were the the shares I had set up in windows which were on NTFS drives that I couldn't get to work in Ubuntu, again help from this wonderful forum and some playing around and they are up and running, bit more experimenting and I'll be ready to throw up a HOWTO soon.

I've got WoW running under wine, not very well (low FPS). I have got the latest  drivers for my graphics card but need to do some more reading on the subject.

Next to try is to sync my HTC HD2 with Ubuntu and setup the mail client, I've only done some light reading and the sources seemed a few years old, but seemed it involved getting 2 or 3 programs to talk to each other (which seemed hard).

Will try and install BURG and find a NZB client (I currently use one that is configured via web browser in Windows, can't remember the name but hopefully that has a linux version).

In summary I've not been using Ubuntu that long and already use it more than Windows 7, and look forward to the day when I can get rid of Windows completely :P
