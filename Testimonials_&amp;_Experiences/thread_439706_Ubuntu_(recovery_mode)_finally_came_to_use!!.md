I guess messed up my xorg config file yesterday and today i chose the "force XGL" option in Beryl for no reason whatsoever. After doing that i couldn't click on anything, so i rebooted and chose the ubuntu (recovery mode) option in grub. At the prompt, I typed "startx" and it logged me into root. I then cut all the Beryl config files from my home partition to my desktop. Rebooted the PC and everything worked (with default beryl options of course). :KS
