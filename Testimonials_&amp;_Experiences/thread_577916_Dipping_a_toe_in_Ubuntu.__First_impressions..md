### Post by Kappity on 2007-10-16
First of all, yes I did read the linux criticism FAQ thread. ;)

I have a Mac Mini and a MacBook, and love them both.  I'm definitely not returning to Windows.  Still, it's nice to have alternatives, and when it comes time to buy a new computer, then something like the Dells with Ubuntu pre-installed would be a lot less expensive than most Macs.

So I put 7.04 on my six or seven year old IBM Thinkpad.  512M RAM, new 40GB hard drive, 500 MHz Celeron processor.  Not a powerhouse, but it can handle the task for the most part.

No need to go into the hardware issues in detail here.  Suffice it to say that I've found many threads and other resources that describe *exactly* what I'm seeing, but the solutions given don't work on my computer.  I haven't given up, but I'm not taking bets on whether all the problems will be solved.  Still, things are working tolerably.  This is being typed on my Ubuntu machine, using a wired Ethernet connection.

I've failed to install a couple of programs not on the Ubuntu package list, but have succeeded with a couple of others.  I may succeed with one more when I've absorbed the information on how to rewrite the makefile.

I certainly don't blame Ubuntu for difficulties in getting an old computer to work perfectly, particularly when I haven't had it installed for that long.  I could have just put Windows 98 back on this one, and at least I'd get the wireless working.  But security would suck, and I still remember the other problems I had with it.  Buying XP  wouldn't be worth it, and I doubt if Vista would even run.  It was toss the computer or put some version of linux on it.

On the other hand, this current installation of Ubuntu won't do everything that I want in a computer, yet.  Maybe it will be closer if I figure out the wireless and the screen resolution issues, but there are no guarantees of that.  There are excellent applications for almost everything I want, but there are a couple that I'm missing out on which are available for Mac or Windows.

Early days yet, but here's the point, such as it is.  On the one hand, I could definitely be tempted by a new computer with Ubuntu pre-installed, and all the hardware working.  My current installation on this Thinkpad is a good chance for me to learn more about linux, while keeping my Macs for the serious work.

On the other hand, I can't yet see going to Ubuntu entirely.  Although I don't mind spending some time learning the nuts and bolts of an OS, I want at least one computer that just works out of the box.  If it has a problem that I need to solve *now*, then I don't want to have to sift through years worth of forum posts and wikis, hoping to find a solution.

Mac is there.  Windows, for all the criticism, is there.  Ubuntu?  Well, I don't know.  Anybody got one of those new Dells?:-k

### Post by Kappity on 2007-10-22
Update.  Wireless works after upgrade to Gutsy.  I didn't do anything clever to make that happen.  I don't promise it would work for anyone else, just giving my own experience.

Confirmed that max screen resolution for this model of Thinkpad is only 800x600 for any OS.  Oh well.  Most of the programs I want to use can be sized adequately for this screen.  All of the ones I *need* to use can be.  Still, It would be a lot better with 1024x768 or greater.

Printing works.

I've found software for everything I want to do, given the limitations of this hardware.  For example, it only has a plain vanilla non-writable CD drive, so I'll still have to use my Macs to burn CDs or play DVDs.

So now I've got a third fully functional computer for home, and a second laptop that's good for travel.   After only a couple of weeks, I've got some confidence in Ubuntu.  No intentions of giving up on Mac OS X, I just like having a good alternative other than, well, you know. ;)
