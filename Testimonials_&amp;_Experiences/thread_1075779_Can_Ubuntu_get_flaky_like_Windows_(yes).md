I started out by installing Kubuntu, however I got tired of that after a few months (KDE 4.1 still really buggy) and so I simply installed ubuntu-desktop. That's when the problems started:

[LIST]
[*]Using update-alternatives to change my usplash back to the Ubuntu one, resulted in only the shutdown screen being updated (startup screen still showed Kubuntu)
[*]The mouse cursor theme refused to update when changed in Appearance Settings, and remained stuck on the default Kubuntu theme
[*]QT-based apps started randomly changing their theme - I had QGTKStyle enabled, however every other time I started a QT app it would appear with Oxygen theme.
[/LIST]

These are fairly minor issues, and they were of course resolved by a quick reinstall (easy with a separate home partition, and vastly simpler than reinstalling XP or Vista!)

### Post by gtrtx on 2009-02-20
The title of your thread is nonsense.

I mean come on, you changed your whole desktop environment with one simple command(sudo apt-get install ubuntu-desktop). Can you even do that in Windows? Last time I checked you couldn't(but that's been quite some time).

With windows, it gets flaky just from general use. Usually the easiest way to cure it's flakiness is to reinstall(I know, this isn't even entirely true). This is simply not the case with Ubuntu. 

Now if your going to start with Kubuntu, then later install Ubuntu, sure some odd things may happen. I've done exactly what you described and a reinstall was never necessary. Usually just a couple tweaks here and there. Also, if I want to try a different Desktop Environment, I normally create another user account to try it out. I've never found the need to reinstall Ubuntu. Back when I was using Hardy I had 4 different window managers/desktop-environments installed. Never once did I have to reinstall.

With my current install of 8.10 I've stuck with the default interface and there is nothing "flaky" about it. I use it everyday and usually have it running 24/7. It's my desktop and server for the rest of my network. and it's running like a champ. I installed from the release candidate and have had the same setup ever since. I don't foresee a reinstall unless my hard drive fails. 

I just don't think it's fair, or accurate to say that Ubuntu can get flaky like windows when what you did to make it flaky would not even be an option in windows.
