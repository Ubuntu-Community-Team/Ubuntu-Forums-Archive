### Post by GoonrGrrl on 2013-01-06
Hi all, Linux noob here with first post.

The last time I tried Linux was SUSE 2.0 I think...  And all I can say is WOW.

Everything has gotten SO much better.  My hardware isn't quite all it should be for the latest and greatest versions of all the distros, and I wasted a lot of time trying to get a few of them to work, but dropped back to Ubuntu Studio Precise Pangolin and everything worked PERFECTLY the first time!! :D  

In particular I'm absolutely blown away by the improvements in MuseScore.  I'm more of a classical composer than a studio person per se (though I look forward to learning about Ardour just as a "personal growth" kind of thing!), so having a truly world-class music notation system for this OS finally is a JOY to me.  The lack of one was the main thing that has stopped me trying Ubuntu Studio before.  It's GREAT now, and so easy to use! I had probs in Windoze with getting Finale to recognize my dirt-cheap, age-old MIDI keyboard, but once I figured out the JACK settings, MuseScore found it right away.  And these Calf plugins are GREAT!!

I'm still having a little trouble getting JACK set up quite right but that's an end-user problem, b/c it has worked before, so I know I'll get it working again. ;)  That's today's project.

So anyway, that's me, a Big Happy Bunny - trouble-free install, great software, NETFLIX nowadays (that's awesome tooooo) - I'm so glad I persevered and really happy I can finally ditch M*cros*ft!!

All you developers who work so hard developing stuff for this OS - THANK YOU ALL SO MUCH - you all rock.  I truly can't belieeeve how great this is.  I salute you!!  ):P
