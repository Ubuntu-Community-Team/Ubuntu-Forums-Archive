### Post by lisati on 2008-07-08
This message posted in the hope of possibly helping someone somewhere sometime

I'm part-way through reconstructing the Window partition on my Desktop - backups duly made and the like.

**The problem:** I go to copy some backups from an external HDD after getting Windows into a good state, and Windows promptly objects, "Permission denied"

**The solution (in progess as I type):** reinstall GRUB (using an Alternate install disk) so I can boot the Ubuntu partition on the machine, fire up Ubuntu, and do the copy from there. Apart from a brief moment of panic (an error of the "Unable to mount partition, NTFS messed up" varitey, easily fixed using advice read some time ago on these forums: namely run Windows CHKDSK) it seems to be going smoothly so far.....
