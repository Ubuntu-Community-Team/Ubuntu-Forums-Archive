### Post by thedoorguy on 2008-04-22
A couple days ago I posted a rant after spending nearly 2 weeks **trying** to get ubuntu 7.10, installed inside w98se using wubi, working after attempting to make a few "must have" additions/changed to the os.  I spent hour_s_ everyday trying to find solutions to the bugs and problems I was repeatedly experiencing ... with no success.

Some responders to my rant were irate over my assessment, some were sympathetic and some were kind enough to offer and give .... assistance.  The most frequent assessment was that the root of my problems was probably using wubi to install and w98.  The most frequent recommendation was uninstall and install using the dual disk approach.

I did.  Wow... oh my God... wow!  I cannot believe that the ubuntu I got by disk is the same OS and version I supposedly got using wubi.  The install was completed... with out a hitch.  The install and reboot of the 200+ updates were completed... with a hitch. (The OS would never boot directly after updating the wubi version. That bug reported to launchpad has not received even one response.)

The few changes/additions that rendered the wubi version completely useless were done to the disk version in an hour or so.... with out a hitch.

I have one little problem... its in a thread... my floppy drive won't mount... it did before... but not this time.

I do not rescind my opinion that the ubuntu I got, installed using wubi... was junk.  But that is not the true ubuntu.  They "pomote" that wubi will install ubuntu in w98se....based on my experience... THAT IS TOTALLY NOT TRUE!

Well... that's the update!  The unbuntu I have now is working great and I am happily on my way to see if I can break it. Not!

Thanks for listening,

thedoorguy aka Richard
