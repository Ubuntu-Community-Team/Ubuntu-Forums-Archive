### Post by darkod on 2012-04-11
Did you try running the cd in live mode first to see if it works with your hardware?

I am not usre about the magazine cd, but if you downloaded the image also, burn a cd at low speed, 8x or 10x, and first try the Try Ubuntu mode to see if it will load correctly.

If the cd can run in live mode, you can try opening Terminal and posting the output of:

### Post by 3rdalbum on 2012-04-15
Ubuntu 10.04 is the Linux equivilant of Windows 98. Generally stable, but far too old for you to be installing it these days.

Ubuntu 12.04 is coming out in about ten days, but you can download a pretty stable Beta 2 right now. If you run the Update Manager after the final release you'll be updated to the final release.

Also, I'd recommend not using the Windows-based installer. Instead, boot the computer from the CD and run the Ubuntu-based installer from the disc.

If you need to download the 12.04 CD image you should use Bittorrent as it will do an MD5 check as it goes. When burning to CD or DVD, use a slower speed like 8x.
