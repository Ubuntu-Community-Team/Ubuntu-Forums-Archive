### Post by solwic on 2009-02-20
> **guywithcable said:**
> How do you break the password on Linux?  I know they're stored in /etc/shadow but aren't they hashed with a salt? Can you break that, cause I've needed to a couple times in the past and didn't know how?

I don't remember the process exactly, but it requires that no root password has been set.  If you hit one of the F keys at startup (F1 or F2, I think) and log in as root on the CLI only, you can view the /home folder to get the user names on the system.  From there it's just a matter of running the passwd command. 

It needs to be an unencrypted drive with no root password, but that's exactly how Ubuntu installs by default (it also requires, obviously, physical access to the PC).

That's all I can remember of it.  I only tried it once on my own system, to see if it worked.  It did.
