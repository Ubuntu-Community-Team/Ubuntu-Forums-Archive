### Post by MARP1961 on 2011-11-06
Unity looks awful. Switched to Gnome Shell and was initially impressed. However, I have come to realise that it looks nice and flashy but just makes simple things take one or two clicks more than previously.

Switching workspaces involves the extra step of going to the Overview Screen first (an extra click). How is this an improvement on Gnome 2? I installed the workspaces extension along with a number of others. It seems you need to install a number of these extensions just to get back to anything like the flexibility of the Gnome 2 panels.

Someone suggested trying Xubuntu, so I did. I was disappointed with the plain default setup. It seemed to be a 'poor man's Gnome 2'. However, I persisted and have discovered it to be highly customisable. Like Gnome 2 you can have one or two panels again. You can alter the appearance of the windows and panels. You can use the desktop fully and alter the colour and appearance of the panels. You have to 'dig a bit deeper' to get it looking the way you want but it can be done.

I've been using Xubuntu for over a week now and I think it is actually the best DE I've tried in Ubuntu. I admit I was wrong to call it a 'Poor Man's Gnome 2'!

### Post by 3Miro on 2011-11-06
I was switching between Gnome and KDE, when I got a crappy laptop that wasn't working too well. Then I decided to try XFCE. Right now, I have Phenom II X6 desktop and i7 laptop, both working with XFCE. Many people think that XFCE is less than Gnome, but I think it is actually more. One or two things may need a bit more work to set (like media keys on the keyboard), but this is something that you do once. Other components (like xfwm4) are actually better than Gnome's (i.e. Metacity).
