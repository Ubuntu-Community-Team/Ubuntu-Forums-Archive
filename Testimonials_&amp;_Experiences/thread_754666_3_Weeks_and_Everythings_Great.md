### Post by RiceMonster on 2008-04-14
So I've been using Ubuntu for 3 weeks and I'm loving it so far. I'm definitely going to stay with it. It's only the second distro I've actually tried, but I find it so comfortable that I don't feel the need to search any more.

Anyway, I first tried Linux because I know some people who use it, and after a lot of talking to them and trying live CDs I decided I wanted to install it. The first distro I installed was Arch Linux, which I still think is great, however. Basically I liked the idea that it was really light, and that I got to put a lot of stuff together myself. It's a great distro, but I just wasn't ready for it. I couldn't get my wireless working all the time with encrypted networks (I'm on a laptop, and it would sometimes connect to my WPA router, then disconnect me, and not work again). Then finally, I tried to get my laptop hibernating, but I messed it up, and it wouldn't come out of hibernate. So out of frustration, I downloaded Ubuntu on my parents computer and installed it, because I had heard that it was one the easier to use Linux distos.

So I installed it, and it set up my wireless perfectly, but it didn't recognize my sound card, which angered me at first, since I was already frustrated it. But it turned out with a quick search, the fix for it was incredibly simple. All I had to do was install the "linux-backports-modules-generic" package, and now absolutly everything works, even my printer which I couldn't get working under Arch. I can still change pretty much everything too, which I like doing; for example, I installed fluxbox and it works great.

Thanks Ubuntu!
