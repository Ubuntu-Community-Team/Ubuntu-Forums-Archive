### Post by Pigeon. on 2008-07-18
General message to Ubuntu developers.

Please ensure that none of the font configuration is changed between releases without providing the user option to accept or reject the changes.

I have just upgraded from gutsy to hardy and the single most noticeable difference is that it has borked the fonts so web pages in Opera now look rubbish. I am now thrashing through changing the font selections in Opera wondering why not one font now looks the same as it did before the upgrade.

The change seems to be most noticeable on Qt apps (of which Opera is one) so I have chosen the "kde" thread prefix. Gnome apps such as GIMP seem OK. Not sure about web page rendering in firefox because I use it too rarely to have a fixed idea of what it should look like.

Font rendering is one of the most important aspects of the look of an application and its configuration should not be changed without asking the user.
