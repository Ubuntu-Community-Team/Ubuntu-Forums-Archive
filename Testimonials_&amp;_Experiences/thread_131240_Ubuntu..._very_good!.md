### Post by Mr.X on 2006-02-16
Windows = Frustrating, annoying can take HOURS to fix a small little problem. Little support for the 100000's of programs microsoft has, and its just plain annoying. Used for 11 years.

Ubuntu Linux = Just works. Enough support on the forums, and everyone is just so.... friendly and helpful!

I tried fedora for a few hours, but nothing was detected (modem, usb drive, ethernet....) - this was mid 2005... no support what so ever.. :rolleyes: 

1 day i was randomly checking ebay, and a auction for a CD with Ubuntu Linux and a whole lot of themes was selling for like $30... But i knew that linux was free.
Then last week, I decided to randomly try out linux, and chose Ubuntu.
Installed...

BOOM! Modem Detected!!
BOOM! ATI Card detected!!
BOOM! Ethernet Detected!!
BOOM! Keyboard, mouse detected!!

By then I was like :D :mrgreen: , all I did was mess around with the modem a little, and the net worked.. Then asked a couple of questions and in less then 5 minutes I had an answer, and now the network works! \\:D/ 

So i've only been using linux for 3 days now, but I have it running PERFECTLY, thanks to everyones help here, the guides etc.

Just one more little thing left, and thats to copy linux to my main harddrive, but I have no idea how to do this :cry:.

So thats my "little" linux story!
