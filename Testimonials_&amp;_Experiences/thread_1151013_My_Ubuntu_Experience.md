### Post by rmconard on 2009-05-06
Hey all,

I'm a first time Ubuntu user and I must say I'm quite impressed with it. I work in the IT Department for HearUSA, the major hearing care provider in the US.

My dad, the network admin, started using Ubuntu about 8 months ago and tried to get everyone to switch. I refused at first because Windows XP was more convenient and I needed it to connect to our center computers across the country.

After the Help Desk Manager made the switch I switched as well because they said vBox would run Windows XP efficiently so I bought into it and made the switch.

Right off the bat I was liking it. The install was seamless and required very little input on my part. Once I had the desktop up and running everything seemed to immediately work.

My printer was automatically detected and installed, my network was automatically set up, even audio worked automatically. I will admit at first I was stumped as to why my graphics were so poor, but then my dad said it was nVidia restricted drivers. So I went to the admin panel, found the driver and enabled the 180 version and it worked instantly. Even my dual monitor set up was a very simple set up.

After I started to play around with X and Compiz I got everything set the way I like and I could already feel my productivity was increased with response time of the processor because Ubuntu doesn't demand that much.

After that I went for the vBox set up. Even this was very simple, I told it to install it and it did. I opened the config for vBox and set up space for my Windows XP install then installed Windows. At first I had a little trouble with the network set up and the Windows XP setup wouldn't accept inbound connections. After a little forum searching I got it to work on our network and now I run Ubuntu and Windows XP seamlessly together on my dual screen.

The speed and productivity is amazing and I would high recommend this to ANY IT Department. Now myself, our Help Desk manager and the Network Admin all use Ubuntu systems. Our Help Desk frontend system also runs on a Ubuntu server. This makes us the first publicly traded, national company in our industry to implement Ubuntu systems in a working environment.
