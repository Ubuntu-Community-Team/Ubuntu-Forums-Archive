### Post by spezticle on 2010-05-19
I started using linux because I was sick of windows crap. The install process took over an hour, memory managment, system stability, and security are rediculous, and boot time took 3 to 10 minutes depending on the state of the windows system.

Ubuntu 9.10 was awesome. My computer would be off, i'd hit the power button and about 30 seconds later I would be ready to go.

My system has 512mb ram and a 2.8ghz dual core p4 processor. it's an old dell dimension e310.
I'm using the gnome 
But since upgrading to 10.04 i'm finding that my system is a lot more like windows now. boot time has trippled, memory usage is almost always maxed, and any time i use the synaptic package manager or the ubuntu software center, i have to stop what i'm doing and wait for the install to finish because the process hogs my systems resources.

This 'Ubuntu One' thing seems like a neat idea, but it's constantly using 25% of my ram.
I don't know what the desktop couch service is at all, and it's using multiple threads hogging another 15% of my ram, so i removed it with apt but htop reports that it's still active in my system after the remove and killing of all active processes.

I'm still learning linux but i'm pretty familiar with it and i'm working on tweaking the active running processes and whatnot, but average users that are new to ubuntu and linux might have a frustrating time with a limited resources system and not want to bother with a linux system when initially it doesn't feel much different than windows.

this is my experience thusfar, i'm 10.04 for about 2 days now and still working on the tweaking.
