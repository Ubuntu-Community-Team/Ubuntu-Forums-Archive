* Researching what proprietary codecs to install to get certain functionality
* Researching how to set up dual boot
* Researching what each Ubuntu disk does
* Researching how to set up my Xorg.conf for the proper screen resolution
* Experimenting with various programs available in Synaptic to see which ones I liked best

Summary:

And that's it for Ubuntu. Notice how the Ubuntu ones are one-time deals. Once I have Ubuntu set up, all the research is done, and I can get to work. Of course, I still "waste" time on Ubuntu for fun, but I don't have to. But if you aren't accustomed to that much power, or know how to tap into that power, then you will think that Ubuntu, and Linux in general, is a loaded POS.

If you are working two jobs and really don't have time to learn a completely new OS, then don't. 

Everyone here that has had success with Ubuntu has taken the time to educate oneself in new ways of doing things. And yes, that meant spending a weekend sometimes figuring out how to make the 24" monitor work with my system.

Once you learn, like riding a bike, it suddenly becomes easy.

To the OP:

**[COLOR="Blue"]Look at the links in my sig. Educate yourself a little[/COLOR]** before diving into something looking for a magic bean to save you.

This is a completely different environment than you are used to. You actually have to work with Linux. There is very little point, click, open a Wizard and install with one click software here.

Don't go to a new country expecting them to act like everyone in your country. Learn the ways of the new country.

And, FYI, it's not our fault that you have two jobs and no time to learn about Linux or Ubuntu. If you put the time into this OS, it will pay you back many times over, if you let it.

Linux is not Windows. Please don't expect it to be. Read up a little about Ubuntu, learn the ways of the force, THEN dive in.

Good Luck.
