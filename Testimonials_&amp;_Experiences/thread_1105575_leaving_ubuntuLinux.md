### Post by 123456789123456789123456 on 2009-03-24
you can dual boot, both windows and ubuntu.

that way you can use both.

I can tell you, and even walk you through a dual boot.
do you remember how you got the programs working for you to begin with?

if you are going to do al dual boot with vista, the best, and most stable way is to get a program for vista called easyBSD.
that program allows quick, and automatic editing of the vista boot loader.
dual boot process as follows
first, you can boot from the vista dvd, remove the ubuntu partitions.
create two partitions, possible halved.  Install Vista on the first partition you make.
you may want not to make two partitons, it would be easier to tell vista to create a partiton that is half the size of your disk.  Leave the other half of the disk unpartitioned space.  After vista is finished installing, install EasyBSD onto vista.
Restart computer with Ubuntu disk.
Under the installer, where it has you select the partitons to install, tell it to guide using unpartitioned space.  Either at that point, or the next point, there is an option called advanced.  click that, under that tell Grub that you don't want it to install under the MBR, but to install grub on the disk partiton that Ubuntu will be installed on.  should be something like hd0,1.  After Ubuntu installs, you will notice that Vista will only boot not ubuntu.  When vista loads, open EasyBSD, follow the following instructons, you are going to add a new entry, linux, Ubuntu, Grub is not installed in MBR.  It will give you a list of partitons, asking wich one ubuntu is installed on, choose the correct partiton, and tell the program to save.  Restart computer, next startup, you will see a list, listing Vista, and Ubuntu, you can now boot in any one.

You can also have Grub boot Vista, and Ubuntu, but there has been some problems associate with that.
