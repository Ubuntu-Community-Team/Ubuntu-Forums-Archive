### Post by aristosv on 2012-04-15
Every few years I install on my main computer the -at the time- best Linux distribution. I try to get off Windows, and since I have some technical knowledge and have used Linux before, I aways tell myself, what the hell, let's see whats up with this operating system. Let's see how it evolved!
 
So I installed, or should I say tried installing Ubuntu, since the problems started during installation.
 
For some reason, when I would tell Ubuntu to take over my whole hard disk, and automatically partition it, it would stop and pop up an error message, promting me to go back. So I manually created a root and a swap partition, and it continued.
 
After the installation, my second monitor wouldn't work. It wouldn't even detect it. So I after a lot of Googling, and after manually editing xorg.conf, I managed to get it working.
 
Then, moving on to my secondary hard disk, were all my data is. After every reboot, Ubuntu wanted me to manually mount this hard disk. It wouldn't do it automatically unless I manually edited the fstab file.
 
Then, whenever I closed firefox, somehow it forgot how I positioned and sized it, and every time I had to do it all over again. So I thought I would ask help on the Ubuntu forums. After clearly specifying my question, a guy come on telling me to specify what my requirement is.
 
As you can see, my whole Ubuntu experience was full of problems, where I had to try hard to find solutions. It's strange, because this operating system is supposed to be one of the easiest Linux distributions today, and it hasn't even figured out how to automate the simplest of tasks.
 
[LIST=1]
[*]Installation should be a breeze. Others might not know how to manually partition.
[*]My secondary monitor should work automatically (It does in Windows)
[*]My secondary hard disk should be mounted automatically (it does in Windows)
[*]Firefox should remember last position and size (IE and all windows do this in Windows)
[/LIST] 
So as you might have already guessed, after last night's easter celebrations, I re-installed Windows and now everything is working as it should again.
 
Well, what can I say. I guess this operating system still needs some growing up to do. I guess I'll find out in a few years, when I try the -at the time- best operating system again.
