### Post by treris on 2006-09-28
I have been running Kubuntu on my desktop pc for about 18 months now and obviously it has been up to my expectations. When Xubuntu came out earlier this year I decided to have some fun and see what it was all about so I cleared a partition on my desktop and installed it to have a look. 
I loved it, but it just didn't give me all the options to play with I'd gotten used to in Kubuntu, now it's still on that partition of my desktop pc, but my main OS is still Kubuntu.
About 3 or 4 months ago I decided I needed to have access to my desktop pc from just about anywhere I go so I installed FreeNX. 
Wow how one app can change your life huh, brilliant, I can now use my desktop from just about anywhere I can get the Nx client installed. 

Now I'm coming to the main part of this testimonial, this semester (yes I'm a student) I need to work on a couple of projects with fellow students. Most of the times that works best if everybody has access to a computer and you can still talk about what you're doing. To do that I have recently gotten hold of a very old laptop with modest specs (Asus L7300 with P2-333mhz, 160Mb and 4,5Gb).
I intended to use it as sort of a thin-client that can work independently but whose primary function would be to connect to my desktop using FreeNX so I can use heavier apps such as openoffice without everything slowing down to a crawl.

AND IT WORKS!!!!!!

it works like a freakin' charm, I have installed the full Xubuntu desktop and it's still only using about 50mb after bootup. Since I'm not going to be using for printing and everything I've uninstalled all that stuff along with some other things I don't use to make it even lighter. It's hard to imagine a fully operational, safe, OS using only about 50Mb.
I can only say that now I can go anywhere, plugin my laptop and enjoy the comfort of working on my desktop pc.

I've got to tell you all it's awesome how much you can do with older hardware this way, now that laptop is fully functional again (with a little help from FreeNX) instead of being a WindowsME reject.

If anyone is still doubting the power linux can give older hardware they probably haven't ever tried it out.
