### Post by mittwit on 2008-02-21
I have been playing with various distributions of Ubuntu and freeBSD for a while but still use Win98 on a couple of home PCs and I work in a Windows XP environment.
Anyway the Win98 hard drive on my old IBM PIII died so I got a couple of old 10GB drives and added some memory so I have 512MB of ram.
The plan was to try do a clean reload of Win98 on one drive and a installation of Xubuntu on the other.
I started with the WIn98. I had a get a dos boot disk, run fdisk, run format and restart. Next launch Win98 CD from a dos prompt, provide the original Win95 CD since the Win98 was an upgrade. Slowly the system was coming together but there was trouble finding device drives and it took 3 hours and 18 restarts to get an up to date, networked, basic Win98 system. It has no office apps, no pdf reader and no zip file reader but it starts up and shuts down quick which is handy since you seem to do that a lot with Windows.
Next Xubuntu. The system boots from the installation CD and it looks after setting up the hard drive partitions and formatting. The system slowly loads but after an hour and 1 restart I have a basic Xubuntu system which includes office apps, gimp, CD burner and more. Obviously Xubuntu had the advantage of being much newer than my hardware so it had the drivers it needed. I did have to do a bit more like getting the flash plugin for Firefox and getting access to the codecs in the restricted updates. Also I had a do a bit of searching to work out how to access my home file server. If I had installed Ubuntu this would have been easy using Nautilus. I guess Nautilus finds and mounts shared network resources itself were as Thunar only displays what is mounted. A program called pyNeighborhood came to my rescue and now I can get to my network drive.

In summary the Win98 set up seems old and close to extinction. I am impressed that, 10 years on, the windows updates are still available and still via the default update link in the start menu. (I did need to install IE6 from a CD for updates to work) But there is no new support for Win98 so I will run Xubuntu when I boot up this PC because the interface is very adaptable, the Add/Remove Applications function is amazing and the information online regarding set up and trouble shooting is great.
