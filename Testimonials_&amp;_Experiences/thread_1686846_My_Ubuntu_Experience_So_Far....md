Howdy everyone...  I discovered Ubuntu about four months ago, shortly  after the release of 10.10, and I have to say that it is an absolutely  amazing product!  I have been nothing but pleased with it so far, and it  has enabled me to get a great deal of further use out of 2 laptops (1  Xubuntu & 1 Ubuntu, both V.10.10) and an old Frankenbox'd PC (Ubuntu  10.10), all of which were languishing around the house with outdated  OS's.

### Post by Max Blyss on 2011-02-13
Aside from the default Apps / Programs in Ubuntu 10.10, I have either added or switched to a few and deleted others...

I added 7Zip and the add - ons to the default Archive Manager (File Roller)
Deleted Evolution Mail and switched to Mozilla Thunderbird
Added the Asunder CD Ripper, Bleach Bit, Sweeper, several games, Ubuntu Tweak, and opted for Firestarter instead of the Firewall configuration tool.
I also added Gimp (which is included in Xubuntu, but not Ubuntu, apparently (GREAT PROGRAM!).

I also deleted several default apps that I had no use for to 'lighten' the OS.

Oh, and I added the ClamTK Virus Scanner, just because I'm a former Windows user, and figured 'Why not?', although it does seem to offer a lot fewer options than AVG...  The research I have done into Linux and the Virus/Malware issue explains why this program can afford to be next to useless.


I really can't say that I've had any problems so far...  Coming from windows, the .deb packages and tar.gz files were slightly confusing, but that hardly rates as a problem if you've been using any kind of a PC for more than a year or two.

So far I have succeeded in getting at least 5 other people to install Ubuntu or dual boot it.  Will work on more, despite most people's skepticism of free software.

### Post by Rubi1200 on 2011-02-13
Hi and welcome to the forums :-)

It's great to read such a positive testimonial about your first experiences with Ubuntu.

I hope you enjoy and learn in this manner going forward. Don't forget there is a warm and helpful community here should you need it.

Just some comments:

Firestarter is deprecated and is no longer maintained. As such it poses a potential security risk.

I suggest you look at gufw, in the repositories, if you want a GUI firewall app.

ClamTK; unless you are running a mail server or sharing files with Windows users, anti-virus is really not needed on Linux.

BleachBit, Sweeper: be careful with these tools as they can, and do, remove more than you want sometimes.

### Post by whatthefunk on 2011-02-13
Instead of having the security programs, Id recommend reading up on file permissions, encryption, basic bash, and user accounts.  This will go a lot further in a Linux system.  You may, at some point, want to install rkhunter or something similar.

Like Rubi1200 said, Id avoid using disk cleaning utilities.  They arent usually necessary in a Linux system and they can erase important things.  The bash command 'aptitude' has built in cleaning options that will serve you much better for now.

I moved to Linux a few months ago and installed all the virus software you mentioned.  After a few weeks, I realized that its kind of pointless if you have a well maintained system, a secure browser, and common sense about what is good and bad on the internet.  I bet youll have the same experience.  

Welcome to the other side:)
