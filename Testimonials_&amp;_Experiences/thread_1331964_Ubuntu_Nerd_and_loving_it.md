### Post by FreakTimmah on 2009-11-19
I have to say I'm really happy with Ubuntu. I first tried it out last fall with 8.04 and upgraded that distro to 8.10 and 9.04. I've learned a lot in my time with it and I feel like its really a breeze to set up a linux desktop thease days. I even managed to get a mythbuntu box up and running and have been using that as an open source DVR since Feburary. This community is really great and the few problems I have run into have been resolved. I just upgraded to 9.10 the other day and it killed my bootloader and messed up my system. It was anoying but I didnit panic. I simply downloaded an ISO of 9.10 onto my macbook pro and reinstalled ubuntu on my desktop. It worked out better because I got to reformat with the new EX4 file system and now I've got a fresh install. The only issue I ran into was getting my Lynksys WTRG54 wifi card working. For some reason the current windows drivers were not working with Ndiswrapper. So I had to dig up some old Rt2500 driver on the internet and once I blacklisted the preinstalled drivers wifi has been up and running smoothly. Even suspend works with this distro. I'm really impressed with all the hard work that goes into this software and I just wanted to say thanks. I think if more people approach ubuntu with a postive attitude and accept that theres a bit of a learning curve they'll come away with a much better experience. For the record I'm running a Dell dimension 4600 2.4ghz P4 wiith 2gigs of ram and a nivida geforce 6200 card and two 250g sata harddrives from 2003. thanks to ubuntu its still a great computer and does everything I need it to. I think I'll easily get a decade out it. :o
