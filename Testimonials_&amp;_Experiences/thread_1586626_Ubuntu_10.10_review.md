Install of 10.10 went without a real hitch for me. Here are the like/dislike things:

Dislikes:

After install I noticed the keys for the repositories didnt come thru. Couldnt install anything. But this was quickly fixed by uninstalling the repos and re-installing them.

The settings for a static ip, dns ect didnt take in the network manager. I had to edit the resolv.conf file by hand to get my dns settings to take. 

Likes:

Installing the restricted codecs and drivers at the time of install is a big plus!

Install went quite well. 

Running on an AMD Dual core 3 gig with 4 gigs of ram. Nvidia 8400 video card.

I found the upgrade reinstalled the iced tea plugin which caused a conflict with Sun Java, so I just removed the iced team plugin via Synaptic and all was well again.
