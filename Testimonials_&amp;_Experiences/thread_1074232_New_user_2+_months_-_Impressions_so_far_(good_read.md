### Post by tgeer43 on 2009-02-19
I felt compelled to relate a very positive experience.
Cut my teeth on Unix in the late 70's/early 80' then went on to early Apple OSs for quite a while before moving to Windows OSs from Win95 to Vista. I guess I was complacent enough to put up with all of the incessant MS irritations all those years. Finally got fed up with Vista's miserable performance and decided to set up a dual boot with Linux. I chose Ubuntu 8.10 and couldn't happier. Here's why:

**Memory Usage:** Vista has no problem hijacking all of my 4GB of RAM. In addition, I've provided it with another 4GB of 'Readyboost' flash memory. Even with 8GB at its disposal, app startup times are not great and general performance is somewhat sluggish. This is on a dual 2.8GHz machine after spending the last nearly 2 years tweaking Vista out and making it as fast as possible. Under Ubuntu idling memory usage is around 200MB. That's just 5% of the installed 4GB. I put the 4GB flash memory into my camera - don't even need it with Ubuntu.

**Applications:** At first I was concerned about giving up my precious Windows apps that I'd become so comfortable with. Well, it took me no time at all to find equivalent or better applications for everything I was doing under Windows. And Ubuntu gives you several easy ways to find and install packages. One exception - eBay Turbolister 2. No Linux equivalent and doesn't run under Wine. If it weren't for that I would have already wiped Vista from my drive and extended the Linux partition. Oh, and the fact that all of my Ubuntu apps are open source freeware doesn't exactly hurt my feelings either.

**Installation:** A full install from the Live CD was quick and easy. Best of all, when I first booted the new installation everything 'just worked'. Wireless internet, bluetooth, printing, scanning, webcam, microphone, etc. all functioned with little or no configuration. I did have to mess with the NVidia drivers to get full functionality but that was a piece of cake. I don't think I've ever had an installation go so smoothly.

**Performance:** Boot time, application startup, and desktop/application speed and responsiveness are all far better than anything I've experienced under Windows. I can run many concurrent apps with negligible performance hit and no memory usage concerns. My small swap partition never gets used. Everything just seems snappier than I had become accustomed to.

**Visuals/GUI:** I love my gnome desktop. The folks at Microsoft only wish they had a desktop with effects half a nice as Ubuntu with Compiz Fusion. Even with tons of eye candy turned on there's no noticeable degradation in performance. Definitely not the case with Vista.

**Security:** Let's face it, when talking about general system security there's really no contest between Windows and a *NIX box. Even with only moderately restrictive settings I am more secure now than ever before. I no longer have to deal with constant internet borne malware threats or have to mount a multi-layered defense against them, tying up system resources. Iptables all the way!

**Choice:** The choice of an easy, comfortable GUI or a fast, powerful CLI. The choice of a truly vast array of freeware apps and a half dozen ways to get and install them. The choice to configure my desktop and other system settings limited only by my imagination.

**Community:** There are zillions of MS support forums (heaven knows they need them) but few, if any seem to have the same sense of community and purpose that I've found here and at other Linux forums. Around here people are more concerned with helping each other than petty infighting.

**Documentation:** There's tons of it. 'MAN' is my favorite shell command. Nuff said.

**It Exercises My Brain:** While Windows may be easier for a complete computer neophyte, it turned me into an apathetic application user. Now I feel more like a computer operator again. I haven't felt that since the early Apple DOS days.

This didn't start out to be a comprehensive treatise but as you can see there are a lot of things that I like about Ubuntu.

I'm never looking back.

Tom

### Post by Mercury_Alpha on 2009-02-19
> **tgeer43 said:**
> And Ubuntu gives you several easy ways to find and install packages. One exception - eBay Turbolister 2. No Linux equivalent and doesn't run under Wine. If it weren't for that I would have already wiped Vista from my drive and extended the Linux partition. You can run Windows in a VM. I use VirtualBox to run XP because the Rhapsody "Jukebox" software is DRM-encumbered and will not cooperate with Wine. And there are some podcasts I like that are exclusive to the iTunes music store.

My XP VM uses about 15BG of space (just to be safe -- I probably could have trimmed it down to 10GB for these purposes). I give it about 1GB of RAM because I have 3GB in my system, and 64MB of video RAM. It runs just fine, and actually seems to boot up faster in a VM than with a dual boot.

With the Sun xVM version of Vbox, I'm running in Seamless Mode. (I don't recommend the Virtualbox OSE that ships with Intrepid.) The Windows taskbar shows up right above my bottom Gnome panel, and I can minimize it to the panel like any other app running on Ubuntu. In Seamless Mode, the Windows desktop disappears, and all apps running within the XP VM "float" within the Ubuntu desktop space as though they were running *within* Ubuntu.

It's much handier than dual booting, not to mention safer.
