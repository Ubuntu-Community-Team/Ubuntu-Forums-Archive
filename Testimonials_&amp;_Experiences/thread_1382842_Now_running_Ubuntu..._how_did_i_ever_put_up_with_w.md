### Post by Dixi1801 on 2010-01-16
Hi guys :)

I felt compelled to post something in here, just to tell people the problems I have been having with Windows and how Ubuntu seems to have none of them!

My pc was crashing, when i was doing anything, mainly playing wow but at other times too, and it was getting really frequent and doing my head in.

I decided to dual boot Ubuntu to see if I could get used to it, considering I had tried it a couple of times in the past!

So basically, after running Ubuntu just about every day for a few weeks, not one crash!

Today I decided to try windows so I could go on wow (got it working on wine but got a problem with the cursor >.<)

Within 5 minutes... a crash.

So i've made my mind up, Ubuntu is my main OS now, once my wow problem is solved, I think I'll get rid of Windows all together!

I would recommend Ubuntu to anyone who has problems on Windows or would just like a change!

Fantastic OS, and a Helpful community!

### Post by XubuRoxMySox on 2010-01-16
It is very liberating, isn't it? I find that I can't be entirely "Windows free" because there are still alot of things that simply work better on my hardware in Windows than in any Linux I've tried so far (OpenSUSE will be my next distro-hop experiment). *If* Linux works for you *and* does *everything* you need a computer to do, then go for it! I'm loving Linux on two of three 'puters. But Windows is still indispensable for some things. Don't rush too quickly to ditch it permanently until you know for sure that you really don't need it. It took me several weeks to discover what Linux *can't* do yet (or can't do *as well as* Windows did) on *my* hardware. Just say'n.

-Robin
