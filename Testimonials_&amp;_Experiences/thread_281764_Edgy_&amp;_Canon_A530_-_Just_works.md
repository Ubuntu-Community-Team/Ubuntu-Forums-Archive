### Post by proctor on 2006-10-21
Hey,

Ubuntu has been fairly stable for me compared to other distros I've used. The only times I have trouble with it is when Im messing with XGL/Beryl configuration or using beta versions of apps.

Just wanted to give the distro praise once agian. I recently received a Canon Powershot A530. Given I am running linux I figured I would run into tons of problems. I read the manual several times over just to get the hang of everything.

I inserted the software disc that is for windows and macosx only, it wasnt recognized. Being a newb, I wasn't sure how to mount the cdrom..and assumed it would be slightly different than doing mounting a hdd. Whatever, so i just powered on the camera regardless of installing any photo apps, plugged the usb cable in & imported the photos in one click. It was far easier than doing so on any windows box. No bloat ware or hassles. I over-thought the situation. 

Piece of cake :D


Edgy is stable as well, proud to running ubuntu linux

```
sudo apt-get install gnome-utils kdict
```

This will install other nice tools like Baobab, a graphical HDD space analyzer. Enjoy!
