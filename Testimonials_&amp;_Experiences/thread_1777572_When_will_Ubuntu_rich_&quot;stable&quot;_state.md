### Post by gecka on 2011-06-07
I use linux as my main OS since about 8 years.

I started to let my 9 years old son use my Ubuntu Laptop. He does some searches for school, and some youtube.

When copy/pasting images and text from Firefox to LibreOffice, LibreOffice crashed about 7 times (total freeze of the app, needed force close)

He then went watching some youtube vids. After him complaining about blinking videos, I had to install the 64bit flash player from a PPA. He could play 2 vids until the laptop totally freezed. My son then said that he doesn't want to use the computer anymore...

Maybe I don't realize how many times I do some hacks, like some ctrl+alt+f1, to solve many problems, but it seems that I do it a lot ...

I still cannot add my 60Gb music library stored on my Zentyal server to Ubuntu default music player, it can't deal with it and fails adding all songs. 

Sometimes SMB shares (gvfs connected) do disconnect by themselves, with no reason. 

LibreOffice asks me my password everytime I generate a PDF file for a file opened from an SMSB share.

My Brother printer still cuts off the top of every page. The scanner can't be run without sudo. Even after adding it to /lib/udev/rules.d/40-libsane.rules and rebooting.

If I copy a big file or a lot of files to a network share the laptop wireless randomly breaks, and only a reboot will get it back.

After some hours of use the Browser uses 20% of my 8Gb of ram... Closing the tabs doesn't help. A restart of it is the only fix.

So my question is, when will Ubuntu get the "stable" state? We are in 2011 and a child can't use it without having to face LibreOffice and OS failures?!? We are in 2011 and Ubuntu still can't deal properly with network shares ?!? 

Unity is funny but I really prefer having a stable usable OS than a  funny one...

The problem is that I can't imagine going back to a closed source OS, after so many years supporting and believing in linux, patiently waiting for it to get stable in basic tasks.

### Post by snowpine on 2011-06-07
You bring up some good points, a few thoughts:

1. The Adobe Flash plugin is proprietary "closed source" software. There is nothing Canonical (makers of Ubuntu) can do to improve your Flash experience.

2. Stick with the "Long Term Support" releases (currently 10.04) if you want the most stable Ubuntu.

3. Debian is the super-stable project that makes Ubuntu possible. Ubuntu is based on the "unstable" branch of Debian. Many Ubuntu users find that Debian is a good option if they're searching for a similar experience to Ubuntu but with more stability. Other distributions well-known for their stability include Slackware and Red Hat/CentOS/Scientific.
