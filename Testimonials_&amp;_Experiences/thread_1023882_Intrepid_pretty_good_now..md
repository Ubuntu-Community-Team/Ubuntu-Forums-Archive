### Post by arubislander on 2008-12-28
> **exploder said:**
> What would happen if the development cycle was changed to 8 months and the applications for the previous release were updated in the main repos? Wouldn't this benefit everyone?

What are your thoughts?

Well, since you asked... :)

I doubt that an 8 month release cycle would result in releases that are more stable.

The amount of bugs that are found are directly proportionate to the size of the user base of the particular software under consideration. This being so, there are always going to be bugs that slip through and only rear their ugly head once the software has been released and the size of the user base has reached critical mass for finding the particular bug.

An 8 month release cycle would only have us waiting longer for new releases of our favorite distribution.
