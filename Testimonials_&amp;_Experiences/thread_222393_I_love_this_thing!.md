### Post by BigRedJake on 2006-07-24
I've been using Windows for as long as I've used computers.  Windows (well, at least XP) never really gave me any trouble, I handled it pretty well, I guess I got lucky.  But over the past couple months circumstances out of my control have forced me to lose all the data on my computer 3 times now.  (I guess I'm too lazy to back-up...oh well)  This last time I figured if I was going to be starting over *again* I may as well try something new so I went looking for a Linux distro to install.

I'm so glad I found Ubuntu!  I didn't have any previous experience with any OSs other than Windows but I've found it very easy to get the hang of ubuntu.  The last few nights I've even been confining myself to the terminalfor most of my commands just to get the hang of it.  But yea, I love it, I don't really play video games anymore so other than that it's had everything I could ask of it.  I don't know, maybe it's just because it's so new, but it seems so much more *fun* to use linux.

Either way, thank you for this.  Thanks for making it so easy and quick to install and thanks for making it user friendly enough for me to ease myself into the more complex usages.
