### Post by jbysmith on 2009-02-25
Ok, it's not even in beta yet, so you need to take everything in a grain of salt.  

Just finished toying with the latest nightly build, and in a word, *wow*.  

Boot time is significantly faster.  Almost on par with my Arch desktop more or less, and that's pretty fast. I'd say a tick slower as it's loading a few extras that I don't have running on Arch; bluetooth and the like.  Lose all that and I'd bet it'll be near identical.

The new login screen is dead sexy.  I'd actually seriously consider keeping it on my own desktop, it's *really* good.  (One of the first things to go along with the default theme.. sorry I do that with every OS.)  Hope the final GTK theme looks as good.

The notification system has a potentially very high cool factor if it gets implemented uniformly throughout the system.  From what I've seen so far, pretty slick.

EXT4.  In some cases, stupidly fast.  'Nuff said.  (Up to an exabyte of storage?  Good lord, thats a lot of pr0n.)

Definitely don't want to try it yet as your primary desktop as you're risking serious breakage obviously, but definitely should take a peek if you haven't yet.  Some bugs to be found, sure, that's to be expected this early in the game.  But from what I've seen so far, looks like it's turning out to be a real winner.
