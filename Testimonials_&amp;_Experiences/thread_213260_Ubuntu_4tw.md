### Post by orgy on 2006-07-11
I've been using ubuntu for about 6 months now, and im really impresed. Starting with such an easy installation, it recognized every single thing on my laptop (toshiba satellite m65) except for my modem of course. The thing is, this laptop has a built in SD card reader, and windows doesnt seem to read them :( so i decided to try it on ubuntu with not a lot of hope, slided it in and:  OMG! it did read it! god i love ubuntu <3. 

When i installed windows from scratch i had to: install video drivers, wireless drivers, modem drivers, touchpad drivers, fn-keys drivers (some of them dont work in ubuntu :( but is ok :D), SD card reader drivers, Java, Firefox, openoffice, gimp, inkscape, then configure windows to make it use less resources, remove unwanted software. This takes about 2-2:30 hours!

Ubuntu installation: boot cd-rom, run graphical installation, wait about 40 minutes the most. Then download EasyUbuntu installed some stuff i like, done. EASY n' nice. You want more software? Synaptic package manager :D EASY again. AEverything free, open source and secure.

thx for ubuntu, it rocks.
