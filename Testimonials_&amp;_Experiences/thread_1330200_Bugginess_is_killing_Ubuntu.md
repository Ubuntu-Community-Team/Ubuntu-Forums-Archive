### Post by stephen.james85@yahoo.com on 2009-11-18
I don't wish to troll but this is something I really feel I need to say.  I have been a user of Ubuntu on and off since 7.04, and have had only a few issues (namely PulseAudio & upgrading) with it up until the release of Jaunty. It seems at this point that the rot really started to set in with the whole Intel fiasco. In fact because of that very issue my Jaunty install has laid dormant for the last 6 months on my spare HDD. After recently reading about the new Karmic release and the vast array of improvements it brought with it I was a little optimistic and ecided to give it a go, but unfortunately my optimism ended about here.  After updating Jaunty fully and upgrading to Karmic  I got the seemingly infamous flickering issue. Sadly even Xorg -configure didn't fix it. So I tried a fresh install and lo and behold..... flickering. Thankfully this time Xorg -config actually worked and I was able to get to the desktop,  albeit at a max 800x600 resolution. This is when some of the other, maybe even worse problems began to emerge. PulseAudio. It doesn't seem to get any better. It's still causes issues that just don't happen in ALSA. Its been this way since 8.04, I was really expecting something better this time around. The USB Startup Disk Creator has totally gone to the dogs, it essentially doesn't work anymore. The new Software Center is currently inferior to Add/Remove, which means if I want to install more than one package quickly it's the good old Terminal or Synaptic. The right click format option doesn't work properly either, which is especially sad because Ubuntu has needed this for a long time.  I wasn't really able to test out much more due to the resolution limit. I find it pretty unacceptable for an OS released in 2009 to have issues that my even my venerable Windows XP 2002 disc doesn't. I also find it very sad that an OS that a couple of years ago looked like a shining star next to the Vista disaster seem to be crumbling into dust. I have now switched to Mandriva as my backup OS and couldn't be happier. Everything works, the art style is beautiful compared to the still ugly default of Ubuntu, and more importantly it feels reliable, something I have notbeen able to say about Ubuntu in 7 months.  So sadly my love affair with Ubuntu has ended, I hope all those that stick with it the best of luck in the future, and who knows, maybe one day I will get homesick and come back. Until then, cheerio!
