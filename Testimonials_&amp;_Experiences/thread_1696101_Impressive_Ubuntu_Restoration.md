### Post by mikex on 2011-02-26
I have two PCs in my Room. I recently built a new primary PC to replace my main machine - an AMD dual-core, Asus MB. The new PC is an AMD hex core Phenom w/Asus MB. The second machine I had was running Ubuntu 10.10 on an older Soyo MB with an Intel processor. I was making regular backup images with Acronis on that Ubuntu machine

After I built the new PC, I moved the machine that was my primary PC (the AMD dual-core) to be the Ubuntu machine, and put the old secondary machine in the closet. I was going to install Ubuntu on the old primary PC (it was running Windoze), but I thought I'd give it a try and restore the image I had of the older Ubuntu machine and see what happened. I really didn't think it would work out since all the hardware was different.

Well, it actually worked, and even better than that, when I booted the restored image I didn't notice a single problem. Remember the Ubuntu image was from an entirely different PC with a different MB, CPU, chipset, etc.

Pretty impressive!
