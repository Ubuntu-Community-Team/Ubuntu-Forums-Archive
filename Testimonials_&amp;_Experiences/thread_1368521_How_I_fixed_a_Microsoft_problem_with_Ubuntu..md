### Post by FreezWay on 2009-12-30
My neighbors computer broke (someone (or some virus)) wiped the system32 folder so I ended up reinstalling. It hung at 63% formatting so I tried again. It hung again. Next I tried reformatting from Gparted in Ubuntu from a LiveUSB and it worked and there computer is almost fixed. (They lost the driver disc.) I find it funny ubuntu is better at formatting to NTFS than microsoft.
