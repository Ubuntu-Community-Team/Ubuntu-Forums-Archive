Those effects you mention can be enabled by installing the compiz config settings manager and activating the various wiz bang plugins. The Unity desktop of Natty is actually a compiz plugin itself. Be careful as unity doesn't play nice with some of the effects and or visa versa. I used to use "The cube" and some people have managed to get it to work with unity but unity is really designed to go with the EXPO plugin so I have stuck with that. The only extra effects I use now are the magic lamp one and burn the window on close. If you mess things up you can get unity back to defaults via ```
unity --reset
```
