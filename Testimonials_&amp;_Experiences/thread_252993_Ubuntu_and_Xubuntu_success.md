### Post by henspace on 2006-09-07
Just like to say how impressed I've been with ubuntu and xubuntu.  I've got three systems at home; one desktop with XP, Ubuntu on another desktop and Xubuntu on an old Tecra 8200.

Both the Linux systems are on wireless networks and I expected the installations were going to be really difficult.  Wrong.  Really straightforward.

Everything has been pretty straightforward.  Printing to my Lexmark via the XP, sharing files, Opera, Netbeans, f_prot antivirus, music ... everything, with a few searches on google, has worked brilliantly.

Excellent work on the distros.

Welcome to Ubuntu. I too use Xubuntu on an old machine - it's great to be able to actually have a practical use for the computer again.
You do realise you don't need anti-virus software on Linux, don't you. I think there are [14 known viruses]("http://en.wikipedia.org/wiki/List_of_Linux_computer_viruses"). Bearing in mind how long Linux has been around, and the fact that they simply can't spread or do any damage because of so much built in security, it's kind of over-kill. Appologies if you knew all this - I don't want to sound patronising, just thought it free up some system resources.
