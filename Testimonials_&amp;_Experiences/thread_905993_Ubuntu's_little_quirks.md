### Post by fd9_ on 2008-08-30
Hey guys,

I've been using Ubuntu for a while now on my T61 laptop and for the most part everything has worked smoothly. The two biggest things that worked out of the box was wifi and suspension, which is great. However, there are a few minor quirks which give me the impression that Ubuntu is not as polished as it should be. For instance,

- My wireless password does not get saved when restarting (the one in System->Administration->Network) and each time I have to manually change it
 
- Trying to 'cancel' when switching users does not work

- Changing the volume via the keyboard brings me out of fullscreen when watching videos

- Middle clicking in firefox sometimes doesn't work the first time, which is strange. Only happens sometimes.

- Sometimes the laptop does not want to suspend when I close the lid, it gets stuck and I have to do a hard reboot (although this is rare).

- Printing something always comes out too small

- Sometimes I accidentally move around the top and bottom panels on my desktop. The little hand icon gets stuck and it thinks I'm holding it down when I'm not. There seems to be no way to lock the panels so that this doesn't happen.

### Post by swoll1980 on 2008-08-31
To save your network pass word: click your network icon, manual configuration, set up you wireless connection then click save as location, and call it home (for example) now you will not have to enter password.

To lock panel: Open a terminal type "gconf-editor" click apps, panel, global, then check the ticker that says locked down

the printer: Bad driver maybe? Check it's compatibility. Can't really help you there. 

middle clicking: are you sure your pushing it down all the way? Maybe the button is sticking(not working properly) 

Suspend seems to be an issue with Ubuntu can't help you there either
