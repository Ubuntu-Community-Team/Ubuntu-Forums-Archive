### Post by alliterationArtiste on 2008-04-04
Been trying various flavours of Linux for years only to be driven back to Windows by some application requirement.. I fell in love with Xenix years ago on an old Altos system but haven't had time to pursue it in a 30 career (so far!) in the Windows IT world.

Went for Ubuntu server version on an old box primarily because I had no previous experience of X and unlike the vast majority (I suspect) of users I have my security blanket in a terminal session rather than the point and click world.

After playing with that I went finally for the big deal on my main laptop and I don't think it's going back. I have to run vmWare for .Net development but my xp virtual machine hosts that and nothing else. I was expecting it to host more, but so far Linux supports everything I need natively.

The only thing I have yet to tackle is mobile phone integration. As a Sony-Ericsson owner I suspect this may be something of a challenge.

If you are considering the move from Windows, my advice would be to break the whole process down first. Whilst a tool like vmWare won't solve your driver problems, it will take you through the whole installation process and allow you to have a play in a virtual world before you do it for real. And you can download a 30 day 'play' license before you have to pay. I believe there are other virtualisation tools out there which are free. Microsoft do one but for some reason it doesn't support Linux :)


So in summary, my experiences have been very positive.

Do it. You know you want to!
