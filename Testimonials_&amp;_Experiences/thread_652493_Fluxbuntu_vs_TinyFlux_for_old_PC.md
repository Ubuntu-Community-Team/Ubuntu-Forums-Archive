### Post by Bungo Pony on 2007-12-28
I wanted to find a nice OS for a PC I've got sitting here. Nothing fancy:

Pentium III
733 MHz
130M RAM
(2) 20G HDs

Although I enjoy Damn Small Linux, I knew this thing could run something a little nicer. The first thing I installed was Puppy. Puppy is okay, but I honestly didn't like the fact that you run everything as root. So, I decided to search for something different. Upon hearing that Fluxbuntu was soon to be released, I waited in anticipation.

Today, I had the opportunity to try out Fluxbuntu. Unfortunately, I ran into some problems. First of all, the CD wouldn't boot. After trying different CD drives and burning another copy, I had to go fetch sbootmgr.dsk and throw it on a floppy. No big deal, this floppy will come in useful for other installs :)

After booting off the floppy, I got the install for Fluxbuntu to run no problem. It took about two hours for it to install, but I figured it would be worth the wait. After it was done, I rebooted. The splash screen came on, the stopwatch counted down, and the screen turned black. Nothing. Everything had stopped.

I figured an install like this, and on this particular PC isn't worth a lot of trouble, so I went searching for an alternative. I eventually decided to give TinyFlux (PCFluxboxOS) a try. So I downloaded it, burned it, and tossed it in the drive. It booted immediately, and the live CD worked wonderfully! Excellent!

So, I decided to install it. It gave me a nice option to repartition the hard drives to my liking, which I did, and the entire install took about 15-20 minutes. I rebooted, and it started up no problem. Obviously, I'm very happy with it :)

I see a lot of PCLOS fanboys touting about how wonderful this distro is compared to Ubuntu. After this experience, I really can't say I blame them. It was simple, easy, and it worked flawlessly. It makes Fluxbuntu look like a bloated, crippled pig. If I actually liked KDE, I would probably end up switching. Nevertheless, my Ubuntu install is working pretty good on my main PC.

But I just figured I'd share this experience with you in case any of you are looking for a nice distro for an older PC.
