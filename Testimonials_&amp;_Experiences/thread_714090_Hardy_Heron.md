### Post by Feivel on 2008-03-03
I wanted to try Linux for quite some time but I really didn't want to be bothered working through the problems (before live CDs were around). Well I am getting ready to move out of state so even though I had a Live CD of Ubuntu 7.10 I didn't think it was a good time to experiment. Well i did and was impressed with everything being so easy and all my hardware working (screen was shifted to the left about 0.25 inch). Since it was so easy to use and I really saw no reason a switch would be hard I installed Wubi and it downloaded Hardy Heron. First thing I noticed is vista's installation took about 45 minutes where Hardy Heron was set up and running in under 10 minutes. I already set the boot to default to Ubuntu. I haven't got rid of Vista yet but I don't see myself booting Vista for a long time.

### Post by anaconda on 2008-03-03

> (screen was shifted to the left about 0.25 inch). 
this can be usually corrected from the displays setting buttons.. eg. with my monitor all I need to do is to press "auto" -button (from my monitor) for a couple of seconds.

by the way.. Hardy Heron is still in development, and it may contain more bugs than a released version.. It is not meant for new users. For example updates might break the system for a while, so be careful with the updates, and check hardys forum before updating. The broblems (if any) are usually corrected fastly, and a new update will usually fix them.

Having said that I must say that I am also using hardy(8.04) and it is working very very well ;) so good luck.
