### Post by OzzyFrank on 2007-07-13
Just my 2c worth:

UBUNTU IS FRICKIN' AWESOME!

I have a fairly knowledgeable Windoze background (in some areas, quite extensive) and I have to say that problems and issues aside, things are usually pretty easy to fix in Ubuntu, especially with the tips and tricks from the strong Ubuntu community. Some of the battles I've had with Windows (my own and clients') shouldn't have even occurred, let alone be so hard (or impossible) to fix. I think many who complain about Ubuntu not "just" working forget some of the hassles with Windows they've had. I can tell you I have seen MANY issues with hardware drivers (made for Windows) not working in, or conflicting with, Windows (an OS you pay for, so should expect absolutely no hassles).

I've had all sorts of things happen to my Ubuntu, from things I've caused through tinkering to buggy updates, and EVERY ONE OF THEM has been fixed. This is without precedent if you compare it with Windows. The only thing Windows has in its favour is the amount of drivers available. I use Ubuntu for most things, but still have to use Windows to use a couple of things, like my TV/FM card. This will obviously improve as more Linux drivers are made, and I reckon the surge in popularity of Ubuntu will help fuel this.

Oh, and some things in Ubuntu just totally kill the Windows counterparts. Some features I now rely on don't even exist in Windows. In Ubuntu, I can do cool things like right-click an .iso image and choose to burn it to a disc; in Windows, I had to go looking for software to do so (then uninstall the ones that ended up being crappy trial versions). Looking for software and installing it is almost too easy. The amount of interface customisation available is staggering, to say the least. The issue of Windows drivers aside, the hardware detection can actually be better in Ubuntu (has picked up some things Windows couldn't recognise without drivers being installed). And lets not forget extremely powerful commands like **dd** (easily backed up my Windows drive to another before it died) and **vobcopy** (with which I can back up my heavily copy-protected DVDs in less than an hour, rather than waiting 15 hours or more for Windows programs to get through the dummy blocks!!).

So, in conclusion, I have to say if you are having some problems with Ubuntu, stick them out, as it will be worth it in the end! Another piece of advice for newbies is to install at least one other desktop environment, so when you mess up Gnome trying to get a bunch of window managers running at once or something, hehe, you can boot into Kubuntu's KDE desktop (for example) and either fix the problem, or be able to get online and search for the answer in the forums. That way you'll realise it is a Gnome (or Compiz or whatever) problem, not a flaw in Ubuntu, so won't be tempted to post a vitriolic thread about how crappy you think Ubuntu is.
