"Microsoft's Windows XP service pack 3 has made Windows XP unusable on my machine!
I decided to install WindowsXP (SP2 streamlined) onto my machine today as I couldn't be arsed with all that WINE config stuff in Linux so that I could play some games.
Installation went fine, the Indian lady was efficient at the authorization phone call, things were going well, but then went pear shaped after "Windows update" :/
Dynamic volumes couldn't be recognised and USB mouse was "turning off" whenever the harddrive was being accessed, great fun! (Latest drivers used etc)
So now, I not only cannot use Vista (The motherboard drivers for my mobo suck, not Vista's fault) but I can't use XP either!
Before all this, I was like...
"Meh, Linux is fun to use, faster, much more configurable and free but damn, more software needs to be written for it, I'll keep Windows installed on a seperate hardrive for the stuff that doesn't work too great in Linux."
Now I'm all like...
"Well damn, I've got no choice, Linux only"
Thanks Microsoft, maybe this is the kick I needed to go entirely Linux :) "

I just hope I can get some games running with WINE :/
I will miss games :'(
I'm kinda excited though :)
I've been fiddling with Linux on and off for the past, umm, 10 years? Could never bring myself to leave Windows entirely, but now, I've got no choice!
