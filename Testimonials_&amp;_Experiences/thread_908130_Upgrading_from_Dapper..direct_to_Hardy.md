### Post by mdsmedia on 2008-09-02
I only opened this particular forum because I was going to write my own testimonial, and I still am, but I found a couple of threads that peaked my interest.

This one was far more polite than the other, so I'm not going to have a go at the OP. I can't agree with his logic but at least he's polite about it.

I've been running Dapper on this notebook since its inception. I haven't upgraded for various reasons and had my doubts about upgrading.

My desktop is far less mission critical and so I've upgraded it from either Edgy or Hoary or maybe Dapper, through to Hardy. Though there are no real problems it's really slow and some things need a workaround.....until I have time to fix them or find that I can't, so upgrading to Hardy on my notebook seemed problematic. I've tried Xubuntu on both computers and quite like it. It still seems slow on the desktop though, so I was concerned about upgrading the notebook.

On Monday I backed everything up and hit that upgrade button. A few hours later my notebook was running Hardy, and I LOVE IT!! It feels like I've clean installed and everything's new again, but I didn't. I ran the upgrade directly from Dapper to Hardy, and it's running like new again.

After Dapper started to feel like it was labouring, I thought maybe a lighter Linux was going to be the only way my 3yo laptop would breathe again. I thought it might not be only Windows which needed updated hardware to run properly. It's restored my faith in Linux. I'm running Ubuntu Hardy....Gnome, not Xfce....and it feels like I'm running a new system again.
