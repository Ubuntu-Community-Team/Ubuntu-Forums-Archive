### Post by Pablo_Escobar on 2005-11-09
This is my little testimonial about Ubuntu
I used to be a strong Micro$oft OS user for years - Dos, Win95, Win98, WinMe, Win 2000, WinNT, WinXP - used'em all. I was never satisfied with the Win, because it was simply lacking in terms of stability (BSOD), security (gotta love spyware). I also didn't feel comfortable not knowing what the system was doing and being treated by an OS as a 3 year old child.
About 6 months ago someone told me that Linux got more user friendly. I visited distrowatch, checked the ratings and my choice went with Ubuntu. One day later, I've started my love affair with Gnome
I was a happy little noob, started to find out more about Linux, the way system handles things, what's a kernel, module, work around in CLI. The more I got into Linux the more it devoured me, the more I learned the less I booted XP.
Recently I wanted a change. I browsed the web and my choice went with Fedora. From about a month ago I switched between Fedora and Ubuntu about 10 times. I just couldn't make up my mind.
I thought - Ubuntu is great but I want more new packages. Yep Fedora has fresher packages, but to compile something which isn't in the repos is a great pain.

So yesterday when I came back from work,
1. Out went the FC disks
2. **Out went the XP partition (which I'm very happy with)** 
3. On went Breezy (3 hour work and my box is setup the way I like it) :)
4. No more distros switching for me.

I love Ubuntu for it's simplicity, fine folder setup. If I want a fresh package -> I'll checkinstall one (that's what I've already done before).

So I'm back with my head down (for ditching Ubuntu for a while). Now my box  is the way I like it :)

Cheers to everyone, every Linux, BSD users.

PS. We don't need to be hatefull to other distros users. We need to have harmony to keep on the development of one of the greatest ideas ever created - "Open Source" :)

### Post by sumanjay on 2005-11-09
Welcome back! I have a similar story. Used to be a Linux user at Uni, got into work developing for the Windows platform and so switched entirely to to Windows for a period of 3-4 years. Got an old Mac laptop to try OS X some time last year (I'm now an Apple fan for life) and installed Ubuntu (PPC) on it just to try it out. Didn't really enjoy my struggles with video support on PPC and dropped Ubuntu for a year entirely. Recently had to help out a friend with setting up a new PC and talked him into installing Breezy as a dual boot. Liked the smoothness with which the x86 version of Ubuntu worked that I ditched WinXP on my own PC and am using Ubuntu solely. It'll never be OS X, but then again, it'll never be Windows either and that's good enough for me ;)
