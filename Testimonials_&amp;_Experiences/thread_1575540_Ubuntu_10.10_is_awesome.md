### Post by kreggz on 2010-09-15
Installed 10.10 a few days ago, I have to say it is really good. I enjoy using this more than Windows 7.

**Good Things**
- It's Stable
- Looks really good (except wallpaper)
- Fast
- Less reliance on command line, you don't really need to use it at all now.

**Bad Things**
- The wallpaper is horrible
- The sounds are annoying and need to be switched off
- Not many applications and games for Linux

**Things that would make Ubuntu better:**
- The release names e.g Maverick Meerkat should just be called Maverick or Meerkat and then:
- Give us a high quality high resolution wallpaper which matches the theme of the release name.
- A native Exchange compatible mail client
- Steam with lots of games to choose
- Optional Enterprise login which is Active Directory (or other authentication services) compatible and is easy to setup and integrates with the Login screen properly. Fedora has a pretty good implementation.
