### Post by KingAlanI on 2008-11-02
Hardware & OS

I installed Ubuntu a few months ago, but hadn't gotten around to using much.

I installed Ubuntu (8.04) via Wubi, so installation was dead-simple; just took a couple hours to fetch the ISO. That I was very pleased with, made it reasonable for me to try Linux (no partition shenanigans and no buying a new machine)

System is a laptop, Toshiba Satellite, Celeron 1.6 GHz, 512MB RAM, that originally came with Windows Sh*t-sta (Home Basic)

Have not used Linux on desktops, so some of my experience is colored by my opinion of laptops in general (don't like the form factor & either trackpads or external mice can be a pain to use)

The interface does take some getting used to, which has been a major stumbling block for me. (Have used Windows since a Win95/Pentium I/2GB HDD box in the mid-90s) - Thus, I wonder if I would have found the Linux interface easier/more intuitive had I always been using **it**...

I have found myself using the Linux boot option a lot more lately.
Since my sophmore year started ( Sept. 2008 ), I've started using my laptop a lot more, mainly for note-taking purposes.

(Previously, I either didn't take notes, used pen & paper, or used a borrowed laptop)

(I have an XP desktop that I tend to use as my main home computer, and there are generally plenty of on-campus computer labs to use before/between/after class)

Linux boots a LOT faster than Vista, somewhat faster than XP. It also wakes out of hibernate/closed-lid quicker than Vista. This I really like, especially for notetaking [running to class just-on-time without really enough time to wait.]

Wireless? No problem (probably have ndiswrapper to thank for that, though)
* both on my home network and my university network

I feel I'm not gonna go Linux-only, but dual-boot I like.

### Post by KingAlanI on 2008-11-02
FireFox 3 for Linux is pretty much the same thing I use over in the Wintel universe, so no big complaints there. Adblock Plus works wonderfully. (That's definitely my favorite addon) Greasemonkey does too, but I don't depend on it as much.

However, the default font in FF3 for Linux is different; making pages look different. I wne tto preferences and switched to Nimbus Roman 9L; close.

How do I get other fonts, BTW?

Open Office I've also used in the Windows world to some extent (2.4)

I mentioned above that I often use my laptop for notetaking; much of this is paid (regular job [not informal cash payments], for the purpose of my university's disability-services offices).

I also mentioned that I often use the Linux laptop during the school day because it boots quickly; this thus means that I'm often taking my notes in OO Writer. As my supervisor's office is on M$ Office 2003, I save my "finished product" as .DOC or .RTF before sending it over; she reports no major issues with the resultant file; this is obviously a Good Thing (TM).  
I've never noted any problems going from OO to MS-Office with the Office 2003 file formats, either

For Windows, I have M$ Office 2007. (I didn't bother to upgrade from 2003, someone else did, but I'll still use it.) While OOXML is admittedly a can of worms, I admit that I like the interface, and that Word 2007 seems to do a better job of autoformatting lists/outlines/bullet-points

The OS definitely boots faster; OO seems to open somewhat faster

Granted, some of those OO comments also show up when using OO for Windows.

The way Wubi set things up, I find my Windows files easily, simply by navigating to /host. Can't hurt. :)

### Post by KingAlanI on 2008-11-02
Ubuntu, as you know, comes with a package manager.

My opinion on those, summarized: it's really easy to install stuff that's available via the package manager, but it's harder-than-Windows to install stuff that's not available via the package manager
"Just click setup.exe" has its advantages.

I noticed the following with Firefox in particular: Updating (perhaps short of major version updates) is not an obnoxious process at all. :)

Also, the package amanger lists a lot of KDE apps, whereas I have the standard GNOME version of Ubuntu.
