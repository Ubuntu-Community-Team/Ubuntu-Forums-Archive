### Post by marcopolo1981 on 2009-11-20
Your problems aren't Ubuntu's - or any Linux distro's fault. Blame the wireless cards manufacturer for not making it easier to create free drivers for Linux. 

Microsoft only make BASIC drivers, but they are licensed to include proprietory drivers with the OS. This gives the illusion that Windows 'Just Works'. Have you ever done a clean install of XP? I don't mean from an OEM disc because they contain all the drivers. I mean from a retail copy. With all the new, modern hardware, XP barely functions! But then I have to install the graphics, sound, modem, wireless, ethernet drivers and possibly more if I include the drivers that come with windows update...

With Ubuntu, I install the OS, connect to web via ethernet, install the Broadcom wifi drivers, configure my graphics and I am finished.
The reason you are having difficulty is because many hardware manufacturers only provide Windows drivers. Then the good folk down at Linux central have to hack them to make them work for Linux - in their own free time, and for free. Yes, they aren't perfect all the time, but they do their best. 

Instead of complaining, why not ask for help?

What wireless card are you using?
