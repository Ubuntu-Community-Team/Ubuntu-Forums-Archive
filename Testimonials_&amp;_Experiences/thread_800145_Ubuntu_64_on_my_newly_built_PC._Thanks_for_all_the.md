### Post by sargonious on 2008-05-19
I have a LE-1200 512kb cache AMD Sempron Processor running at 2.1Ghz on a generic motherboard from MicroCenter which has built-in NVidia 6100 nForce chipset with 2GB of Ram. I have an SATA HDD and an IDE Lightscribe DVD RW. I have a clean black tower on a Sylvania Flatscreen 17 inch with max resolution at 1024x768 which I actually like for internet and games. I'm running everything onboard except for the network card I'm using a USB wireless Netgear dongle. The goal of this computer was to build it cheap and I did from parts my dad had lying around the house. I only purchased the mobo/cpu/mem combo from microcenter for less than 100 dollars. It is running great better than it does on my vista partition thanks to the help on my previous thread. I would have never installed vista if not for it being free with some class my dad took and he already had a few licenses so I'll take it for no cost. At least I have both Windows and Linux which is great for work experience.
