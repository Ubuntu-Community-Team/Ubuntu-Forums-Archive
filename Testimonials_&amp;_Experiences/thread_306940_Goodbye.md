15 months on Ubuntu. I still dual boot though.....
Dapper and Edgy! :)

I have a removeable drive carrier and I do have a spare drive with XP on it that I COULD put in here and boot from. But I haven't. I'm not a HUGE gamer, but I do have one game I play. World of Warcraft. I play it in Wine 0.9.26 and it works beautifully.

Other than that .... I am really enjoying being windows-free. Using my open source machine is far more enjoyable and far less work than using a windows machine. A bit more... trouble free. Less concern about "oh my god, what if I get another virus" even though I have never gotten a virus in many years of computing (yes I'm careful). Tired of the hacks and patches and lousy performance.

When I installed Ubuntu on my previous machine (I've built this current one since then) it was a Pentium 3, 1GHz with 512MB and a 128 meg graphics card. It had been running WinXP and I THOUGHT it was running fine. Using Ubuntu it was like I had a new machine! It was so much faster. 

Now I have a dual core machine with 2 gigs of ram and a 512meg graphics card using SATA drives. Sometimes I think it opens an app before I actually click on it. (*wink*)... ok just kidding about that part. But this sure is fun again.
