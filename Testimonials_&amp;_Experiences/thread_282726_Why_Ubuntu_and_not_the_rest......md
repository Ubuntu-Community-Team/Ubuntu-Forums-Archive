### Post by Circus-Killer on 2006-10-23
I would have to say I am a big supporter of open source software. I have been using linux since 99 but could never find the right distribution. I've used everything including RedHat, Mandrake (now Mandriva), Fedora, Slackware, and Gentoo. Out of the above listed, my two favorite would have to be Gentoo and Ubuntu. Now gentoo is great and all that, its a really nice "concept", unfortunetly it works more in theory than it does in reality. i dont know why this is, but i always find inherent bugs, not minor ones, but big problematic ones that were deeply embedded into the overall system. and when one big bug gets fixed, another two arise. i guess i just got tired of fighting with my gentoo system, spending more time maintaining the system rather than using the system. it did teach me a lot about linux and all, but i need an OS thats usable. Enter ubuntu. With ubuntu, i must say that there is no customizing ability during installation, which would usually bother me (thats one of the reasons i liked gentoo so much). but if you actually look at the default ubuntu install, it is very clean and light, which makes customization not much of an issue. the default packages are all the useful ones, whilst the less necessary being left out. what is also nice about ubuntu is that once installed, it is still very customizable. in fact, i find it more customizable than virtually any other distro. let me explain before you shoot me down. after adding ALL the customizations i like on my system, it is more likely to happen that something will go wrong, even with gentoo. often base gentoo system work fine, but when you try add advanced custom features the system inevitably bombs out. i find this a problem with virtually all distros, but not with ubuntu. no matter how far i seem to push ubuntu, it seems to take the load. i personally run a lot of different stuff on my system, apache, mysql, and even native games like wolfenstein: enemy territory. and as always ubuntu serves me well. ive been searching since 99 for the distro that suits me, finally i found one.
