### Post by decoherence on 2009-05-29
I apologize for the rant like tone this post will take, but...

I really, really frickin HATE Evolution! Now, I don't just go around saying I 'hate' software. There's plenty of software I don't particularly like but I don't say I hate it.

I hate Evolution because it has the potential to be so nice! When it works, it generally works quite well. Plus, it's the only native linux mail client that I know of that supports GroupWise (aside from Groupwise for Linux, which is a Java monstrosity and not even worth hating.) 

That's why it is so frustrating when it freezes up as it has done a few times before. I force quitted it and tried to re-open it. No joy, no error, nothing!

It was only by running killall -9 on evolution, evolution-alarm-notify and evolution-data-server-2.26 that I was able to start it back up again.

Is this supposed to be user friendly?!?! I suppose a Windows user might just tell me to fix it by restarting... well guess what: I hate Windows, too!!

Okay, I feel better now. Sorry again for the rant.

And yes, I WILL look for a bug report and file one if I can't find it :p

### Post by BslBryan on 2009-05-29
That sucks, man.  I'm sorry the features aren't introduced into Thunderbird, which has always proved good software.  If you want a quicker way to kill the process, there is a graphical way (much like ctrl+alt-delete in Windows) by following System --> Administration --> System Monitor --> Processes --> (Process) --> Right-Click --> Kill Process.

I just hope you don't have to use it too often, because it really reminds me of Windows. :p
