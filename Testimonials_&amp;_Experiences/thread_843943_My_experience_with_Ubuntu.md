### Post by h_howee on 2008-06-28
I've been using Ubuntu for about a year now and so far I have run into many problems, most of which are currently solved. Since upgrading to Hardy, I couldn't get sound from more than one program at a time and I only got that solved a few minutes ago. A while ago, I had a problem with Xgl eating up over 50% of the cpu so I went back to windows. During this short time back on windows, I haven't gotten any issues that I couldn't resolve in less than 5 min so I thought I'd settle down on Windows. More problems arose as I started installing new softwares until the point where it was as bad as the thousand pages of poorly written code in the projects that I couldn't finish. I never wanted to set foot on Windows again. When I came back to Ubuntu, Xgl miraculously disappeared and I've been here ever since :).

**What I'm unhappy about...**
[LIST]
[*]One of the main problems I get with Ubuntu is installing from source. It doesn't automatically install dependencies so I have to look all over to find them and then those dependencies have other dependencies and it just keeps going like that.
[*]I'm afraid to uninstall anything because something else might be using it and if I uninstall it, then it might mess up something I need.
[*]It takes me forever to solve the problems I get on Ubuntu but that's probably because I have a habit of googling for at least a week before posting on forums. I've also been using Windows for a long time so I know how to fix most of the problems that come up there.
[/LIST]
Other than that, I'm happy with Ubuntu.

### Post by jjk7288 on 2008-06-28
As far as installing from source is concerned, there isn't anything Ubuntu devs can do about it really, except for provide those dependencies in the repositories, or make the software you want available in the repositories so that things are done automatically.

If you uninstall something in Synaptic, it will inform you if other things that depend on it are also going to be removed.

Googling (usually) solves my problems fairly quickly. If no solution is apparent, don't be afraid to post or ask in the IRC channel.
