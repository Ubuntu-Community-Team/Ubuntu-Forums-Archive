### Post by ventrical on 2012-02-13
I've  been reading some more stuff on interfaces and noticed some veiled slights at Unity. However, keeping on the beta test theme - I have been using Unity now since Natty Edubuntu and all the way up to Precise alpha. So far , any bugs encountered are worth it  and are no way comparable to desktop interface bugs I have encountered with other systems and DEs.  I know some voices are not so affirmative of Unity but I find that it's practicality beats anything on the market. As  a long time end_user programmer , Unity eases my mind.  Being left-justified, it has already done  most of the work for me and if I feel that because there is no bottom panel and I am falling off the edge of  cyberspace then I can always run Unity in Gnome Classic.

  I think that Unity has been the right path to take for Canonical. It is a smart move and will be successful.   Unity with Compiz will be even better !

  Great work so far!!!
