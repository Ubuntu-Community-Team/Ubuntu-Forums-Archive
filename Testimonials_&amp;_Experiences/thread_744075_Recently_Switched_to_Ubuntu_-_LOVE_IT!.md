### Post by Deamos on 2008-04-03
Well, last weekend I took the leap and removed Vista and installed Ubuntu 7.10 and I am completely blown away.  I had dabbled in it before, but with recent issues involving windows products made me want to try a stable system.

After a few days of figuring out different commands and setting up my desktop and such, I dropped my Web server and brought it back online with Ubuntu Server.  Now my web server seems to run tons faster and can stay online longer.

Personal Likes:
Boots faster
Much more Stable - I haven't crashed it yet
Built-in application install
I don't need to use some tweak application like windows to make it run at a normal speed

Cons:
Still would like to see more support in Wine for some of the games that I have


Great job, Ubuntu Team!
Looking forward to the new version!
