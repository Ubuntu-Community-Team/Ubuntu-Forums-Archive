### Post by rokstaEnSweden on 2009-07-31
Hey guys,

First I want to say thanks, because without this forum I would not have had such an easy time getting Ubuntu 9.04 up and running. This was my fourth attempt at installing linux, and the previous three were pretty discouraging. But this time I was motivated by separating myself from vista (which had committed suicide again), and the fourth install worked flawlessly.

So I spent the first couple days or so setting up the desktop, making sure it could do the cool cube effects and reinstalling the linux equivalent of my most used applications. But after a while I got really interested in the efficiency of using the terminal instead of the gui, so I started messing around in that. It was actually really fun learning something new.

Then when I had my system set up and some knowledge of the basic commands, I started getting a little more brave by attempting to install some windows games with Wine. Well, the outcome of that project a mixed bag. Fallout 1 & 2 worked great with minimal fuss, but Quake 3 had me going for three hours trying to get the sound to work. Eventually I did, but I lost the music in the process. Oh well!

So then on the fifth day, I decide to put /home on a separate partition. Yeah, I'm sure everyone knows where this one is going. I'll spare you the details, but I spent all night trying to recover my system and ended up having to reinstall from scratch. It was pretty agonizing having to set it all up again after spending so much time creating it, but what can you do!

And now finally, I'm ending the week with celebrating getting Diablo running (which took another 3-4 hours of craziness). So that pretty much sums up my first week with linux. I say it's like Myst because some applications take a ton of work to get running which can be extremely frustrating and puzzling, but finding the solution gives me a great sense of accomplishment. :popcorn:

### Post by tgalati4 on 2009-07-31
Soon you will learn to create muliple partitions and have different versions of Ubuntu (or other distros) on each.  Keep one stable (such as Hardy) and use it when you need things to work.  Load the others to see the new stuff (such as Jaunty).  When things break in the new distro, you can always boot back into the stable version when you really need to get things done.

One thing you quickly learn--never trash a working system.  Add to it, but don't trash it.  You will find it's cheaper to buy a second hard drive and add a new distro to it than to spend a lot of time reinstalling a working distro that got trashed because you were trying to be clever.

When you break it, you get to keep both pieces.
