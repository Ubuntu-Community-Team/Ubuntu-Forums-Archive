### Post by Mikecore on 2010-11-12
I generally don't post very many positive experiences with Ubuntu. However this time around my thinking has changed. I just got a Dell E5410 business class laptop I bought for personal use, After getting its slightly larger brother the Dell E5510 for work. Both from a hardware standpoint in my opinion are great laptops. At any rate the second it came in the mail I ripped Windozes 7 business off and did a fresh clean install of Ubuntu 10.10 64bit Desktop edition. I usually install the latest 64bit version of Ubuntu, because I don't mind when somethings go wrong. However with this new laptop I wanted to install the LTS 32 bit version. Funny thing is due to some bug it hangs at boot. So for time sake I just installed the latest 64 bit version which booted just fine. Once I figured out which version to install the whole process went along great. There are a few minor bugs, for one I cant use my external 25in monitor due to some buggy driver and two the touch pad isn't recognized correctly and I don't have the corner scroll bars working. Other then that everything else works perfectly. I can't say enough good things about how well this laptop works with Ubuntu. I'm losing my linux skills not having to fix everything all the time.
