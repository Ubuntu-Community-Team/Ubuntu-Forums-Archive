### Post by mattxian on 2009-02-08
No hiccups what so ever to report, Ubuntu is running smoooooth. I am dual booting with Vista and Windows 7 Beta. The only problem I had was when I started messing with Compiz but that's to be expected considering this laptop failed the compiz-check I ran. Oh it works but when you start enabling all effects and 3d it will freeze the system, but that's expected as by the specs below you will see why.

1.6 GHz Intel Pentium T2330 dual-core processor, 160 GB hard drive, 2 GB RAM, Intel GMA X3100 integrated graphics.

I have also tested out openSUSE 11.1 both the gnome and KDE installs, the KDE one failed miserably. Fedora 10 worked fine but I prefer Ubuntu 8.10.

So that's that, not bad for a cheapo year+ old lappy.

### Post by running_rabbit07 on 2009-09-28
> **Sliiice said:**
> Was the Ubuntu 8.10 install working out of the box? Also what would you recommend to do immediately after installation to prevent some hiccups?

```
sudo apt-get install ubuntu-restricted-extras
``` will install java, fonts, and codecs among other things that will make the experience go better. 

```
sudo apt-get install firefox-3.5
``` will get you firefox 3.5.

Going to Synaptic Package Manager you can search for and install many of other programs that will make life easier and better.
