### Post by NCLI on 2010-01-24
You can't test hardware in Virtualbox ;)

The best way to test Ubuntu is to use it as your primary OS, then second-best is to use it on the side on the newest hardware you have, the third-best to use it on the side on older hardware, and the fourth-best, almost useless way, is to run it in Virtualbox, since everyone running it in Virtualbox should encounter the same problems.
