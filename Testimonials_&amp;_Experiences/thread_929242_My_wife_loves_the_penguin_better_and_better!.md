### Post by patrickballeux on 2008-09-28
We are windows free for the last 4 years...  My wife was really annoyed by Windows always having problems, over and over again.  

So one day, I suggested that we could install Ubuntu and get rid of Windows as all she was doing was internet stuff, email and some little games (on internet...).

This convinced her:
1 - It's all icons and mouse, like Windows
2 - If you're not happy, I'll put back Windows ...

Guess what, Windows never came back...


She likes the Gnome interface as it is clean and easy to understand.  No freezing, no rebooting, no waiting.  It just works!  

And my kids are used to it, actually, they don't know much about windows.

It's nothing to be proud of, but it shows that the Linux Desktop (Gnome in my case) is appreciated by non-geek users.
