### Post by Dhi on 2005-10-20
All,

I have recently installed Breezy Badger 5.10 on my Dell 8500, and overall,well.. it has been a breeze! 

This is my first experience with Ubuntu, and I have previously used Mandrake 10.1, but for me Ubuntu was a clear winner right out of the box - it configured my laptop volume keys and my WXVGA (1600x1050) resolution without any dramas - something I always had to do manually after a Mandrake installation. Plus the Ubuntu experience is way more friendly! 

I was initially concerned about the whole wireless issue, but it took me hardly any time to get my wireless network (Netgear WGR614 Router & WG111 USB) up and running at WEP-128 using ndiswrapper. 

While I do have a few minor teething problems, all in all, I am more than impressed with Ubuntu. 

So my sincere thanks to all contributors for a job well done.

Dhi

### Post by anil_robo on 2005-10-22
Hi all

I just installed ubuntu on my brand new Dell Inspiron 6000. Surprisingly, everything was autoconfigured. The volume keys, the widescreen display, the wifi. I was afraid it would not detect my Kodak Easyshare C360 digital camera, but then I thought of giving it a try. And I was shocked it detected that too!

Overall, Ubuntu has been running smoothly on my system. Nautilus crashed 2-3 times, and I have reported the problem to the developers. One of the key elements for open source software to survive is FEEDBACK. Although the feedback system in Ubuntu is slightly more complex than that in Windows, I was determined, and could post my bug report through the web only. The bug report tool supplied with system kept asking for "where is sendmail". "Dear, I installed you just half an hour back, and YOU should know where did YOU install sendmail! How is a Linux newbie supposed to know where is sendmail?" ;)

Some of the word documents have been behaving strangely, maybe because I made them in windows. Otherwise, new document creation is just perfect in Openoffice.org. Similarly, some powerpoints made in Windows do not display animations correctly, but non-animated powerpoints made from window run just perfect.

The wi-fi configuration tool needs to improve in terms of usability - instead of double clicking it, user should be given more options in right click itself.

Sound quality is not as good in XMMS as it was in Winamp, but I guess Winamp comes loaded with a lot of plugins and sound technologies. I'm sure XMMS will learn, just like Winamp did few years ago.

The repository system is a little complex for a typical windows user to understand, and I feel repositories should not need to be copied and pasted in the configuration file. Ubuntu should handle them itself. Same rule applies to permissions. If a user can't access a file, the dialog box should tell some more details, and possibly a button to go to the file permission settings.

I still have no idea how to send these thoughts to the Ubuntu developers...  Someone help me please. I want to contribute to Ubuntu because I love it.
