### Post by inksmithy on 2006-11-24
Hello all.

I'm a complete noob when it comes to ubuntu and almost as much of a noob when it comes to linux. 

The last time I installed linux it was Debian Sarge onto a desktop computer and I was using a USB modem which fairly resolutely refused to work. The entire installation was a bit of a nightmare and I ended up going back to windows in disgust.

This time, about ten minutes ago, I installed Edge onto a Dell Inspiron 2650 with a Belkin RT2500 based wireless card. From everything I had read about wireless on linux I was a tad worried, but I have to say, that part of the install went perfectly, it was detected and connected within the installation process and as you can see, it looks to be working pretty well.

The only place I had any real difficulties was the during the partitioning process, which for some reason wouldn't allow me to create the partitions I wanted. I ended up having to go for the swapfile and one large partition containing all the files.

At the moment I believe the partitioning didn't work because I was doing something wrong, rather than ubuntu doing something odd, so on the whole, it exposed my ignorance fairly effectively, but luckily I managed to pull back some shreds of dignity there.

This is probably not the right place for this post, but I wanted to let any other noobs out there who are considering a ubuntu laptop install know that it IS possible and it CAN work exactly the way its meant to.

Super happy with my install and ubuntu, and am happy to say that all of my computing these days (exceppt for my phone) is linux based - even my handheld gaming console is using linux, being the mighty GP2X from Game Park Holdings.

Looking forward to doing some development and chatting to you guys as I learn what it is I have got myself into.

regards,

Alan
