### Post by Rinzwind on 2005-12-06
Hello,

I've been wanting to use another OS since my 1st BSOD in  Windows 98 and all the troubles I had installing some very insane weird cards. I promissed myself that when an OS came along that had the same easy installation and hardwaredetection as Windows I'd JUMP into it and permanently switch over.

I tried several Suse (from 6.x to 9.x) and Redhat installations on my PC but it always ended up in tragedy. I also tried dual booting for a while and just got annoyed of those Linux installations. 

My laptop is an Dell Insirion 9300 and I decided it was time again to try it out and see if I could install Linux on it as a dual-boot. I searched the web for installation reports and complaints about my laptop with installing Linux. At that time I didn't decide on what flavour to use and used my PC to download Suse live, Ubuntu live, kUbuntu live and Fedora. For some unkown reason I decided to start up with Ubuntu Live as the 1st OS and the only expression comming remotely close to what I saw in the end was WOW!! Everything(!!) got detected from scratch. My Wireless network got installed perfectly, the wide-screen didn't start with a yucky 800x480. Soundcard worked out of the box. 

The only thing I really didn't like is that I did not have an install list during install. I always like to have ALL software installed and the un-install them when I(!!) decided it sucks, doesn't cut it compared to something else or whatever other reason I decide to hate some sort of programm. 

Sooooooooo I decide to take a leap and enter tty1 and I typed:
sudo apt-get install ubuntu-desktop
sudo apt-get install kubuntu-desktop

Well it took a little while but the result was perfect. I never before used apt-get(!!) and it was amazing what software choices I unleashed on my laptop :) After that I started tinkering.

I saw a post about someone wanting to install an iRiver flash-player. Hmmmm I thought. I got one of those too. 1st problem I ran into: the things I needed for a GUI install where NOT inside any of the installers :( But behold after I posted a topic here I found this website and saw someone even made a special package for ubuntu. [http://ifp-gnome.sourceforge.net/](http://ifp-gnome.sourceforge.net/) and this worked perfectly. This should be in the packages tho ;)  

Next was the printer. I have a PC with Windows on it and it has an all-in-one HP SPC 2170. I played around with the printer-installer but didn't get it to work. But it wasn't (k)Ubuntu bugging me, it was Windows :D  This webpage explained it perfectly: [http://raldztech.blogspot.com/2005/12/share-windows-xp-printer-to-linux.html](http://raldztech.blogspot.com/2005/12/share-windows-xp-printer-to-linux.html) and I now have a laptop that can use my printer. 

After alot of tinkering I now can browse, e-mail, stream and rip audio, listen to my mp3's and ogg's that I have on a share with my PC, I can print remotely, have alot of fun with alot of games, use Gimp, OpenOffice.o__rg, And I've learned alot along the line.

Still to do:
ATI X300 driver install (I am still to scared to do this :) ).
Get the (remote) scanner to work (but it's not a priority).
DVD playback. Haven't tried watching a dvd yet.
Logitech USB headset. It worked at some point but I can not seem to get any music from it anymore. And I swear I didn't do anything but take it out of the USB plug and use it again the next day :( This is a must to have working tho otherwise I am gonna get kicked from public transport :D
Maybe I even will atempt to install FF 1.5.

And I bet I'll find alot of other cool things along the line :) 

*does happy dance*
