### Post by KPexEA on 2008-01-25
We had an old windows ME machine and the hard drive failed, so I picked up a new 160GB drive for it and installed Gutsy Gibbon.  It was a painless install and also very easy to have it connect to the other windows machines and printers on my home network. I am mainly using this system for converting an open-source app I am working on to linux. Now my few gripes that I hope would be fixed on the next release.

1) Since I was using this machine for programming I needed to download a bunch of packages using the apt-get command. At first I couldn't get some of the packages that I needed and it turned out that it was because I had not turned on the ability for it to "download" the needed packages from the net. It would have been nice if the error message that I got when I was trying to download packages would have told me that I didn't have the downloading ability enabled.

2) When using Firefox it seems to be constantly doing DNS lookups over and over again for domains that it just looked up minutes ago. It would be nice to have a DNS cache installed by default. I looked online for how to install one but it looked like too much editing of config files was needed.

3) I have been using gedit for editing of my source code, one item that has been a pain is when doing a search, I type "Ctrl-F" and then immediately start typing what I am looking for, The search window pops up right away but until it is ready the input that I typed too quickly is placed into my source code and if I don't notice right away then it will barf once the compiler hits this extra code. It would be nice if they can detect that the pop-up window is in the process of being created and once up send the key-presses to it instead of back into the source.

4) Last but not least is ( gnome ) when adjusting the window sizes I have a hard time getting the mouse in the adjust size area. Typically I get the mouse pointer to change to the proper "adjust size" pointer but as soon as I try to click and adjust the window size I immediately get the window underneath popping to the top since I "missed" the adjust area and now brought it up to the top. It would be nice if the adjust areas were a tad bigger so it would be easier to adjust them.

Overall I have been very impressed with it, keep uo the good work.
Thanks
Kevin
