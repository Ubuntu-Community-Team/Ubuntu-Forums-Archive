### Post by Zaiden on 2008-06-15
I've been using XP for years now, and honestly, I never had a problem with it. However, it eventually became boring, being too simple (as odd as it sounds, this is NOT an insult to Ubuntu/Linux), so I heard about Ubuntu Linux, and the things it offered:

*Freedom from Microsoft
*Most games I like and play work
*An awesome community
*More Customization
*Something I actually want to learn about

So, first time installing Ubuntu, I wasn't familiar with it, and eventually went back to Windows. Something called me back, and some time later I've become used to using Ubuntu, losing my comfort zone within Windows XP.

Onwards to the Good Things: I was able to change my desktop theme around very easily, making it feel like something I want to use, and sometimes show off to friends. With some work, I'm able to get my games working, like The Orange Box, Guild Wars, and even Quake Wars. There's also so many open source and free games to play, it makes me giddy inside. While I have to google for them, I'm slowly learning the basic of the terminal and how things work, even made a Linux Notebook to remember some things. Being able to run Windows XP in VirtualBox is just very awesome also I might add (I'm not against Windows, I believe that every OS is good, and people shouldn't be judged for what they like to use).

Sadly, there's the things that I don't like. Biggest issue is the number of times I've had to reinstall Ubuntu. Seems like everytime I tried to fix something, I'd screw up and completely ruin the installation. Another Big issue is PulseAudio. While it works, it delays the sound for some programs and games. The work part is that your forced to use it in Hardy Heron, which seems kind of stupid to me. Some games I play, Mainly Ragnarok Online and PSO: Blue Burst (Both Private Servers), no matter what I try, won't work, even under XP in VirtualBox. I know not all games work, but getting so close only to fail...well, sucks.

Overall though, I'm VERY happy with Ubuntu. It works, I can play (most) games, still use XP, and so much more to learn about it. I will admit, it can't do everything, but it can do enough to satisfy a person like me. Heres to many more years of Ubuntu Goodness!

(Open to any criticism anyone may have, since I dumped this all from my head at 6AM)

### Post by StooJ on 2008-06-15
Games are always the sticking point, and will probably always be a sticking point unless the games industry in general changes it's attitudes towards linux.
This would also require a change in attitudes of linux users towards the games industry; I think the suits believe that users of a free operating system with free software, free upgrades, free source code and free-dom ( :p ) might not be that willing to fork out not-so-free cash for a non-free game.
I would love it if the linux community proved them wrong, and I'm sure there are plenty that are prepared to pay for games that run under linux (me for one - I've bought all of Introversion's games and love them), but I don't think there will be enough people.

With regards to reinstalling, I was the same. Discover the awesomeness of your /home directory on a seperate partition, and it won't be nearly so much of a hassle. Keeping your notebook filled with the commands you use to set up your system will speed things up immensely, or even better: write a script & keep it in your home directory somewhere. When I reinstall I run the script and drink tea until it's done.
