### Post by aufan19 on 2009-05-05
Hi everyone,

I just wanted to tell everyone about another positive experience I've had with Ubuntu. 

A few days ago, I wanted to scan some documents onto my MacBook. My printer, an HP PSC 1610, has scanning capability, and I expected it to work with Image Capture. When I start Image Capture after connecting the printer, I get the message "No Devices Found". I do some digging on the internet, and find that there is a workaround for this by installing the TWAIN SANE OS X ports. I install all of the necessary SANE things, only to realize that I don't know what backend my printer/scanner uses, and I can't set it up. A little more digging, and I can't find my printer under the list of SANE-supported ones. I try to scan with Image Capture anyway, and it crashes. I try again, same result. My scanner still fails to work under Image Capture. All the SANE files are still buried somewhere in my computer. Hopefully I'll need them one day.

Next, I try going to HP's website and find the software for my printer that is supposed to make everything work, including the scan feature. I download the software, agree to a license, install everything, and set up my printer (successfully, according to the software I downloaded). I go to scan a document, and I get a message that there was an error connecting to the scanner. This software is made for my printer, and scanning still doesn't work. :shock:

Next, I try Ubuntu under VirtualBox, my only remaining option. I enable my printer in VirtualBox and start up my 9.04 VM. I open SANE, press scan, and it works. Beautifully. Nothing goes wrong. I scan something else. It works to. This is amazing. Open source wins.

After some more digging, it sounds like Apple doesn't put a lot of effort into making sure printers and scanners work under Image Capture. So much for out-of-the-box usability on that regard. I have no idea what went wrong with the HP software. But I'm perfectly happy using Ubuntu to scan documents. The next computer I get/build (I've never built a PC before, but I want to some day) Ubuntu is going on. It really does "just work".
