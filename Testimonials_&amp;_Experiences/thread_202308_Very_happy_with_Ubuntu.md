### Post by snek on 2006-06-23
Hey all, I'd just like to say I am very, very happy with Ubuntu Dapper so far. I have been an avid XP freak for years and am known as the biggest nerd amongst all my friends. A few months ago I thought I would dive into the deep end and learn using a unix based system. It started off with FreeBSD on my laptop which I configured from scratch to run XFCE4, which worked out pretty nicely. However, I wasn't happy with the speed of it, so I decided to try out some Linux distro's. Debian was pretty fast, so eventually I stumbled upon Ubuntu. 

I managed to install 5.10 on my laptop (500Mhz, can't boot from USB stick OR cd-rom) and was amazed at the ease of use! After some upgrading to 6.06 I decided it was time to move one of my main pc's to Ubuntu and get rid of XP all together. This is my media-center/download-server. 

So far installed (and used often):
 - Azureus
 - LinuxDC++ (still has some sharing problems though)
 - aMule
 - MythTV
 - VLC
 - TV-Out working on GF2MX400 (TwinView)
 - Swiftfox

A very nice system imho:

AMD Sempron64 3400+
Asus K8V-SE Deluxe
Abit Siluro T400 (GF2MX400 AGP)
1.5GB DDR400
80GB UDMA/133
120GB FireWire external
250GB SATA
300GB SATA
