### Post by xt0rt on 2008-05-16
Long time Windows user here, up until just the other day... \\:D/

I ran different versions of Windows for roughly 13 years. I'm 25, so for me that is quite a long time. Anyway, around 1997 or so I somehow got involved with the newly spawned console (specifically NES) emulation scene, which introduced me to IRC. IRC, in turn, made me aware of alternative operating systems. I remember attempting to noob-install FreeBSD 4 on a poorly equipped Pentium 2 system around 2000... let's not talk about that anymore.

In the years since, I was a Windows user. A few tries here and there with various PCs: BeOS (great, bad support), Corel Linux (rofl @), and Redhat (still had poor networking). Still reverting back to Windows, XP (and subsequently SP1 & SP2) was released. Finally, a decent and at least halfway securable desktop OS for the average user (school, work, casual gaming). Fast forward a few years...

Vista is garbage. About a half year ago I bought a fully equipped HP Pavilion dv9500t laptop with a docking station. This investment cost me about $3000. A small price to pay for a decent "Entertainment" PC - or so I figured.

None of my games worked. I played Quake 2 and 3 for years and was really into Wolfenstein: Enemy Territory. So I stopped playing my favorite games, and tried some of the newer ones. Vista handles most of those pretty well, but without a desktop and a high end graphics card, I felt like I was trying to run Unreal Tournament on a 486.

Trying to suck it up and not feel like an idiot for buying an expensive laptop, I pretty much stopped gaming and just focused on keeping the PC operating in an acceptable manner. I DJ for an Internet radio station, play games from time to time, write way too many papers for school, watch movies, listen to music, the list can go on. This should not be too much to ask from an OS that has been developed for years on end. I wasn't messing around with the Windows registry or trying to run anything out of the ordinary. Vista was stealing file associations for Microsoft (and partner's) software, locking up, and generally irritating me.

Whatever.

I run Ubuntu (Hardy) now. Just about all my hardware was functioning properly immediately, and that which wasn't was easily made available to me through the use of these EXCELLENT forums. The only thing left is getting Bluetooth to work with my phone (check here for more info: [http://ubuntuforums.org/showthread.php?p=4968845]("http://ubuntuforums.org/showthread.php?p=4968845")).

I have a pretty cool looking desktop now, that caught my eye first. I can run ALL my older games, as most of them run natively on Linux (thanks ID Software). Ubuntu enables me to perform all the tasks I needed Windows for, and much faster for that matter. I don't think my swap file has even been accessed yet... incredible.

So, thanks to everyone in the Ubuntu and Linux community. You've all been working at this for years, and certain people like myself have been paying attention. All that work has paid off, as this is the most stable and aesthetically pleasing OS (after a few tweaks) that I have ever used.

So, in fewer words, keep it up!
