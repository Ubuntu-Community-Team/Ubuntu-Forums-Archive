### Post by h2o4444 on 2009-01-18
I just installed Ubuntu 8.10 about 2 weeks ago and I've been playing with it A LOT! I'm running it on an HP Compaq tc4200 tablet laptop with Intel Pentium M processor 1.86GHz, 512 MB RAM, and 60 GB HDD. It is dual boot with 10 GB for Ubuntu and about 45 GB for Windows XP SP3.

This morning I decided to do a speed test on [http://www.speakeasy.net/speedtest/]("http://www.speakeasy.net/speedtest/") to see if the internet speed differs in each system.

The tests were done in Firefox 3.0.5 for both systems, I also did a test in Google Chrome since I have that installed on the XP. My internet connection is a wireless connection that I'm currently mooching. The speed tests were all done with the "New York City" server on the website. I tested 3 times for each web browser using the same NYC server and here are the results:


On Windows XP SP3:

Firefox:
download (kbps) 272,   287,   326
upload (kbps)   466,   341,   493

Google Chrome:
download (kbps) 517,   1176,  598
upload (kbps)   482,   495,   494

On Ubuntu 8.10:

Firefox:
download (kbps) 2569,  3179,  1906
upload (kbps)   350,   245,   279

As it turns out, Firefox runs a lot faster in Ubuntu, but its uploading rate is poor. I really don't care about that since I rarely upload anything. Google Chrome beats out Firefox in Windows XP, thus makes me wonder if Chrome will be faster than Firefox in Linux once Google releases it later this year. 

Now I can watch shows on [hulu]("http://hulu.com") and [casttv]("http://casttv.com") without stuttering!
