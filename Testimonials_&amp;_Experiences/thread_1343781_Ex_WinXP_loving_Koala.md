### Post by Fast_Wyvern on 2009-12-02
Hi, funny title, but true

I had 2 laptops on Vista, switched them back to XP, then tried Karmic Koala

1st laptop Sony Vaio, smooth install everything was working except the webcam in Skype, found the solution in the forums so all good

2nd laptop Alienware M5500i-r3 smooth install, but had problems getting mic to work as it required a specific mdel being set in the alsa-base.conf file. Found the solution in a convoluted way. Identified that this laptop is a rebranded Fujitsu and used the snd-intel-hda database on the forum to track down the solution. All is good and got it working well with an SSD installed as the main drive

So all in all loving the Koala
