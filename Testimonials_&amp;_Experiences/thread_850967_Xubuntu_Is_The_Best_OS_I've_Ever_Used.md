### Post by Vince4Amy on 2008-07-06
I've been distro hopping so much lately. I went back to Ubuntu for a period but I still wanted to try something and carry on using it. Well the whole time I've ignored Xubuntu, never really taken a good look at it, but I have to say it is probably the best OS I've ever used!

The default theme is really stylish.
It's so fast, everything launches so quickly on:
[LIST]
[*]AMD Athlon X2 4500+
[*]4GB DDR2 Memory
[*]ATI Radeon X1950
[/LIST] 
The software selection is more minimalistic so it's easy for me to customise at the way I want.
I never encountered Meta packages when I was remove some default packages.
My monitor was detected with the correct Resolution and Refresh settings.

The whole thing just feels so polished and I would recommend this to anyone over Ubuntu, it just seems that much more stable and polished to me.

No, Fluxbuntu must be installed from scratch. You can't add the fluxbuntu-desktop to an existing ubuntu/xubuntu/kubuntu install.
