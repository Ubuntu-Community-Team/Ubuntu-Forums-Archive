### Post by armandh on 2008-12-16
with the HDD you wish to save disconnected boot the live CD and click install [takes a few moments for the first screen to appear] do the Q&A and for the partition select "use entire drive". this will wipe your old drive and install Ubuntu.

### Post by falcon61102 on 2008-12-16
armandh's idea for leaving the 80gig disconnected is a good one while you are installing so you don't accidently wipe your data.  The install itself is pretty straighforward and depending on your hardware setup, you should be good to go as soon as it's done.

In reference to you installing the new HD and doing a dual-boot, it is recommended that you install Windows first and then Ubuntu.  Windows has issues with being installed second and it's quite easy to get the dual-boot set once both OS's are installed.

Welcome and enjoy.
