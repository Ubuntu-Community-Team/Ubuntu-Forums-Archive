### Post by surfjdh on 2007-03-29
[LEFT]About 3 days ago I had an overclocked box with a chopped and twisted (4th install) xp os.  Lots of registry tweaks, overclocked cpu, (amd2080MHz"athlon/barton"), etc, etc. Anyways, after owning this system for 3 years; same HD; I decided I was going to try Linux again(my initial attempt was flamed on by P.H.L.A.K., into YDL.  and KDE, no fun, **** wouldn't mount, anything usb was fubar, you get the pic.) So I remembered a newer release called ubuntu, decided to check it out, dl'ed the iso for dapper 6.06lts(it had support for the longest), and ran the live cd.  ......HOLY CRAP, this actually looks good.  I really wanna try making this work for me.  BTW at this time i had about 30gb of anime, 12 gb of music, steam&cs1.6, and I used it all.  Without any guarantee that I would ever be able to re-dl my anime, or use steam, or stepmania, or opera, or watch dvd's, etc., I did the unthinkable. 
 Here's how it went
 1) reboot  
2)opened cd drive, and pressed f8(boot menu on my bios) 
3)closed cd drive, chose boot from cd 
4)followed install screen instructions, reformatted, made swap, ext3, and said DO IT! 
5)and it did, told me to take the cd out cause it wanted to reboot, so i did, and it rebooted 
6)...when that logon screen came up for the first time, it really was the beginning of something great, and hearing the ubuntu logon sound however simple it may be gave me the assurance that this was a friendly os, and it wouldn't fu*k me over like xp constantly did.ie:'programs you can't uninstall, virus' you cant delete, etc.'  and I've been plugging along ever since, learning new terminal commands every day, downloading new aps, and doing my cgs1570 homework in openoffice(in fact the same night i installed 6.06, i was using it to edit excel spreadsheets for school)           
       I just want to sincerely thank the devel team on this distro, you have given this nerd new hope for the open source community, and the fastest, most stable, and most ingenious operating system ever, Ubuntu.  Keep up the good work while I learn how teh f**k I watch an .mkv file in vlc without it looking like string cheese. -Devon Hawkes[/LEFT]
