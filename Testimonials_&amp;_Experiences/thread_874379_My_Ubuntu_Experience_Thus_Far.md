### Post by Unanimated on 2008-07-29
I've had Ubuntu for about a month now, and read about it on CNET's spyware horror stories (yes, I'm that nerdy). One night I decided to visit the website and download the ISO. I tried to burn it to a CD, but it rarely worked, so I bought some more. I burned Kubuntu and tried it out through the Live CD. For some reason, it didn't quite click with me. I did the same thing with Ubuntu, and I immediately fell in love with it. I don't know why, just after the desktop loaded I knew I had to have it. So I installed it as a program inside Windows and used it a lot. I really liked it, and I saved everything on my D drive to my C drive, wiped the D, and did a full install of Ubuntu. I was using it for a while, had my music set up, internet, sound, games, etc. I decided that my XP installation was so mucked up after a few years of using it, so I wiped it and reinstalled it. This of course overwrote my GRUB file, so I wiped my D drive again and reinstalled Ubuntu. (Shortly after this, I read a topic on these forums on how to reinstall GRUB. :mad:) I made my sound work again (kinda hard because my sound card is so old, but it works now), ripped all my music from my now-defunct MP3 player (I shipped it to Sandisk to get it replaced), and reinstalled all those games. I'm now enjoying my experience even more, because I have about 20-30 gigs to use up with Ubuntu, and I'm thinking about buying a 500GB external hard drive when I scrape up $100. All I'm really saying is, Ubuntu has really opened my eyes of what the open source community can do, and I think I've learned how to trust people a lot more because of it. I now dual-boot into Ubuntu and XP, and I have yet to install SP2. And also...
[COLOR="Red"]**DOWN WITH THE EVIL EMPIRE!**[/COLOR]
