### Post by adrian-g on 2009-08-04
I use to run Windows XP on an old IBM Aptiva E-Series, but I got fed up with security issues and the extremely slow speed. It use to use 25% CPU and 40% RAM idleing. Apache would crash under more than 10 requests.

I popped in the Ubuntu 8.10 server disc, and began the installation. Setup was a breeze, and the preconfigured options were just what I needed (although I prefer MySQL over PostgreSQL).

A quick apt-get update, apt-get upgrade, and then apt-get dist-upgrade, and the server was up and running.

I then installed Wordpress on it along with ddclient (using my DynDNS credentials) and I am now hosting my blog.

When I first started this journey, I was totally dependent on GUI's. I even tried to install ubuntu-desktop on the server because I hated the CLI... and failed tragically.

After 5 hours, I finally removed ubuntu-desktop and most of the dependencies... but now I am totally comfortable using the Terminal :D

Server specs:
AMD K6 MMX 233.0 MHz 
256MB RAM (21% currently in use)
8GB HDD (27% currently in use)

Requests go through at least twice as fast as when I ran my blog on Windows XP, and now I don't have to worry about intruders in the network.

Thanks Ubuntu!
