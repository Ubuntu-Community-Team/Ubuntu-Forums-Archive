### Post by Suiname on 2009-11-03
Do you want to know if you are running the 32-bit or 64-bit version of the OS, or if your processor is 64-bit capable?  It appears from your posting that your processor is 64-bit capable, but you can see if you are running the 64-bit version of the OS by running "uname -a" from the terminal.  For instance, my output is:

```
Linux S10-desktop 2.6.31-14-generic-pae #48-Ubuntu SMP Fri Oct 16 15:22:42 UTC 2009 i686 GNU/Linux
```This shows me my computer name, the kernel version, whether or not symmetric multiprocessing is enabled (that's the SMP bit), the timie, then the processor type (that's the i686 bit).  If you are running the 64-bit version of ubuntu you will see AMD64 where i686 shows above.  If you are running the 32-bit version you'll either see i686 as I have above or i386.  Hope that helps
