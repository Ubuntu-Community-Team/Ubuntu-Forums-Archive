### Post by aatyler on 2009-09-26
Recently I did an install for a friend. On thuresday I did a Wubi setup for a friend of mine with the intention of re-partitioning after he got used to linux and made up his mind that he definately wanted to use it full time. 
 
I expect to get a call from him that he wanted to do a dual boot setup. Instead he called me and setup a time on sunday for us to meet so that I can not only re-partition his HD, but get rid of windows entirely. In three days his experience with Ubuntu (9.04) has been so much better than with windows, he's decided to switch over entirely. Here are some of the reasons that he switched over:
 
1) He can't believe the "speed" and "smooth experience" of ubuntu as compared to windows.
 
2) The innate security features and built-in firewall (I installed UFW for him) make him feel much more secure than with windows
 
3) He loves the fact that linux is so much more efficient when it comes to resource usage than windows
 
4) He said that he was expecting that installing programs was going to be extremely difficult and was very pleased with the Synaptic Package Manager setup.
 
5) Everything he needs is free
 
6) He loves the Philosophy of open source and feels like a better person for using it, instead of windows (his words, but that is so true).
 
So far I've had a wonderful experience with Ubuntu, and I'm glad that I could help someone else along. Heres a few questions though, my friend is an intelligent person, and he can do most things with windows, but he's never touched a command line and doesn't understand much about drivers, hacks, etc..etc... Any reccomendations of things that I can teach him that will be most helpful? I plan on walking him through the installation of Ubuntu instead of just doing it for him so that he at least starts learning and of course I already turned him on to the forums for support.

### Post by jmszr on 2009-09-26
aataylor,

This is my standard advice when someone new wants to know where to get Ubuntu information:

        I would suggest this sticky by ugm6hr: [http://ubuntuforums.org/showthread.php?t=801404](http://ubuntuforums.org/showthread.php?t=801404) . The links to Psychocats Ubuntu Guide:  [http://www.psychocats.net/ubuntu/](http://www.psychocats.net/ubuntu/)  and Free Ubuntu Pocket Guide by Keir Thomas:  [http://www.ubuntupocketguide.com/index_main.html](http://www.ubuntupocketguide.com/index_main.html) are of particular note but the entire sticky is full of a lot of good information.

Also: [http://ubuntuforums.org/showthread.php?t=790485](http://ubuntuforums.org/showthread.php?t=790485)  (Newbie 101) and: [http://ubuntuforums.org/showthread.php?t=714338](http://ubuntuforums.org/showthread.php?t=714338)  ( Newbie Terminal Intro)

The Free Ubuntu Pocket Guide has a section on the Terminal starting on page 69.

You might want to teach him :

```
sudo apt-get install "application name"
```

Hope that helps.

Edit: Include "site:ubuntuforums.org" when using Google to search Ubuntu Forums or use this: [http://www.ubuntu-search.org/](http://www.ubuntu-search.org/) to add the Ubuntu search engine to Firefox.
