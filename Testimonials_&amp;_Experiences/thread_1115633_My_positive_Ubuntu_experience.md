### Post by agreen416 on 2009-04-04
Apparently the latest Windows update has a new WGA tool that has blacklisted some VPK keys that have worked perfectly for many years.  I am not angry at Microsoft.  They are a business protecting their brand.  But to go in full panic mode as Windows does when it suspects that you've stolen their OS is a very troubling thing because I am sure there are many false positives.

This happened to some family friends that I had built a system for.  Luckily they called me first to save the headache of being labeled as a criminal by MS support.

I knew all they really do with their system is to send the occasional e-mail, read news from back home and listed to some streaming radio from back home as well.  I knew that the first two steps were going to be easy but I was most worried about the streaming radio.

The streaming radio that they listen to uses an aspx format which I wasn't sure that I would be able to work around.  This would be the deal breaker.  If I didn't get that to work then they would be stuck having to use XP.  I did some research and after installing the VLC plugin for Firefox, it worked perfectly!

As a bonus, the wifi card worked perfectly without a hitch and the printer worked perfectly.  It is funny because with XP, I had to search around to find a driver for the D-Link wifi adapter, installed the latest version of the adapter and realized that it didn't work correctly.  I had to then look at the model number of the adapter, make sure I had that particular version of the driver to make it work correctly.  With Ubuntu it just flat out worked.  Same with the printer.

I am very excited about how quickly and easily everything worked.  Microsoft is correct in what they are doing to protect their brand but they have to realize that 90% of casual PC users can get away with using free software as in the case with my friends.

I personally use OSX because at the time when early Vista reviews labeled it as a POS, Ubuntu wasn't as polished as it is now.

This experience has pushed me to be a more active member in the forums and to spread the word even more.  So thanks again to Windows Genuine Advantage for pushing me to never install Windows or troubleshoot a Windows machine ever again.  If someone asks for advice I would suggest Ubuntu first and if not, suggest that they invest more money and purchase a Mac.
