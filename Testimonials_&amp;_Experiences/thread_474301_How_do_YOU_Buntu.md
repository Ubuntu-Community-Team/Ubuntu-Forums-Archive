### Post by Geekkit on 2007-06-15
I've installed 7.04 and I find that many problems that I had with 6.10 have now been resolved (e.g., USB issues, theme issues, Synaptic issues, Samba issues, etc.). I find this version to be incredibly stable and quite peppy too.

I have two media applications that handle all my media needs: RhythmBox and Totem. I'm also using Glade to write GTK+ software right now and I'm getting heavily into GIMP, Cinepaint (I need a tripod now because I really want to take some HDR photos), and Inkscape. I'm also using NetBeans and JDK 1.6 which all seem to work just fine.

I have my desktop networked with my wife's computer running XP (she needs XP unfortunately because WINE simply won't run QuickBooks or SimplyAccounting). File and print sharing work flawlessly and were dirty easy to configure (no more /etc/samba/smb.conf editing ... woohooo!)

The only problems I've had are with my old (ancient really) NVidia GeForce 4 card, or, more like the driver which is the one from NVidia. Sometimes when I run 3D games (e.g., World Of Padman, Tremulous, Warsow), Ubuntu/Linux freezes and crashes the kernel. I don't blame Linux for this (this is a monolithic kernel after all), and I suppose I don't even blame NVidia for this since this is quite an old card. Heck I'm not really too much into gaming anyways so it's not really a loss and even when I was in Windows I didn't game much either so no loss.

Minor things now need to be fixed but I'm sure I'll see those too taken care of in, what, six months? Anyways, happier than a clam now and have my desktop system configured the way I want it.

:D
