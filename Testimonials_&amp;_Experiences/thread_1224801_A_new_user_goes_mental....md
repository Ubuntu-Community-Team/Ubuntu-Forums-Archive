### Post by Markustee on 2009-07-27
Good Day (or night depending on where you are).

I am a damned new user (about 8 months now) to Ubuntu.

I bought myself a netbook at christmas 2008 to see if I was able to learn something about the open source software/operatig systems. Until I realised there was a fantastic little program called Ubuntu I was going crazy (the netbook had linpus linux :x:-& I hated it). 

Then as I finally got a decent working netbook (had installed VLC within 5 minutes of finishing initial install), I decided to get cracking on learning something of it.

Well, I had heard it was possible to use a nifty little program called WINE. Great, now I can use some Windows programs on my netbook. Let's see what it can do...

Well, I had searched and searched around the net, lurking in the deepest darkest corners of it to find little known information about this subject, windows programs on a netbook. Wow is that a little touched on theme...

So, I decided I would try stuff out and see how far I got, not to mention I will post any and all findings here (if allowed) as a project and a collection area for those who simply cannot give up gaming. :mrgreen:

So. It starts.

The netbook;
Acer Aspire One. Model nr. 150 - zg5.
Intel atom 1.6 Ghz cpu with multi threading support.
1024MB ddr2 ram. 
Intel GMA 950 Graphic chip.

Ubuntu Jaunty Jackalope 9.04 Ubuntu netbook version.



First of all I downloaded WINE. Using the packet manager.

Worked first time and runs perfectly.

This is the basis of my games etc.


Secondly: Steam. 

Runs well if a little slow.

Steam games I have tried:
Audiosurf. Started, then it opened into a large tabbed window. The screen was too small to see anything, the game took an age to load then the netbook stopped working altogether. 
Hard powerdown was needed to gain control.

Halflife Deathmatch Classic. Starts fine. Finds servers great. Connects great. Runs well with little slowdown. No sound. Lots of popping even with sound turned off in the game itself.
Stops responding after a short time, needs a hard reset as above to regain control. 


Non-Steam games:

Command and Conquer The First Decade.
Installed via a network DVD reader/writer.
Launcher installed and runs perfectly.
 

Tried games: 
Red alert 2 - Runs well on lowest settings, but when cursor moved it has massive slowdown. Only then. Not sure why, but it is playable if not the best speed. *Note, cd only used for installation, runs without cd and no hacks used. Not even connected to network and still runs. Highly recommended for a bit of old school fun even though slow.*

RA2 Yuri's Revenge: Asked for Cd.

Tiberian Sun/Dawn and expansions. Both did not run. Asked for Cd.

Generals and Zero Hour:
Both asked for cd. 

Red alert 1 not as yet tried.


Eve-Online: 
Installed fine. halted at startup and refused to run because it needs (!!!) shader model 2 provided by the hardware. 
No solution found as yet. 


Teamviewer 4. 
Both installed and live versions run fine if a little slow.


So that concludes the first round of testing games etc.

If anyone needs or wants something testing give me a shout (if I have the game or can get a test version/demo or so) then I will try it. 
Important note though, any requests for games or applications like Crysis will NOT be tested as it is simply too much needed hardware wise and I seriously doubt I would even be able to get it running to see. 
Old school games suggestions are welcome.

Best Regards.
Your friendly n00b netbook tester.

Markustee. :mrgreen:
