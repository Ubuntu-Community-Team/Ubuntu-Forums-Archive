### Post by alexpwnsn00b2 on 2010-06-14
After using Ubuntu for a while now I have to give it up for the people that continue to help push this OS forward it is truly an amazing OS that frees you from windows.

PROS: its free, I no longer have to dish out money for another OS again. Although I might donate a small amount of money like $50 or something to help support. Security is another plus, because 90% of the world doesn't use this OS viruses are few. I installed 10.04 on my netbook (had some trouble with the graphics but all seems better now) and It actually runs compared to when it had Windows Vista Basic. I found Windows OS to just suck all the resources my Netbook had to offer but Ubuntu is Light weight compared to that crap.

CONS: Not for complete computer Noobs. I did a great deal of research before attempting to dual boot. First I used Wubi. Then I dual booted. After finding myself almost never booting into vista I installed Ubuntu on the Entire HDD. At times when trying to install software I have problems with different things that can really frustrate the average computer user. Again Ubuntu is not for complete n00bs to computers and how they work.

Overall I have found my experience with Ubuntu to be great I absolutely love it. It will forever be the OS for future netbooks or notbooks. Thank you Team Ubuntu I really appreciate all the hard work you guys and gals have put into developing this OS keep up the good work!!


P.S. I couldn't find a suggestion forum so I'll but it here. 

I think a rating system for the Ubuntu software center would be very beneficial. I don't have the time to go through each one and see which works the best if there was a rating system in place it would increase my production of searching for desired software.
