### Post by squashpup on 2008-07-03
Puppy is great for what it is, but the packaging system leaves something to be desired AFAIC.  I like being able to install all the packages from the Ubuntu repositories, but also to be able to find pre-compiled packages on the internet specifically for my system.  For example, I can go to the Opera website and get the latest version of it anytime I want.  Same for Gizmo Project, which actually works well on this computer.  

Of course, things could be different now, since I haven't worked with Puppy in a while.  Don't get me wrong...I like Puppy, but it's not Ubuntu.  And, since Xubuntu/LXDE is quite usable on this computer, it seems silly to trade all that versatility in order to gain a little speed.  

Just my opinion.  If Xubuntu/LXDE didn't run so well, I'd be trying out Puppy.
