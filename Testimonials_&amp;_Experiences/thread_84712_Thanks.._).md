Just installed ubuntu on my computer, extremely happy with it as it detected all my hardware and loaded all kernel modules correctly! Even my wireless card was detected and all I had to do was to add details into the nice network configurator to get it up and running.

I've used gentoo a lot in the past, but I am going to stick with debian based distro's from now onwards because I simply prefer them (compiling from source can get tedious on my athlon xp 2000). apt-get is genious, and the way synaptic is bundled in and ready to use is fantastic.

I also have everything I need for my comp. science course too, I installed gcc/g++, hugs, prolog etc. all through apt-get without any problems whatsoever. Bye-bye windows!

Well done with Breezy! It really is great - and a great example of how Linux is getting easier and easier to set up and use.
