### Post by SinrG202 on 2009-05-20
Firstly, I'll introduce myself. I'm a network administrator from the UK. Of course SinrG (Synergy) is not my real name but who cares. I started to write this post before this intro so If you stick with me through it thanks. I've tried to keep it entertaining. So get something to smoke and make a cuppa coffee.. Here goes....[U]

Intro[/U]
I've been an avid fan of linux for years, but haven't been able to make the switch due to workloads and knowing my way around Windows made it so much more efficient. As a net admin it's not easy to put aside the time to learn an operating system and at home I just want to relax, or game online.

_Requirements_
So... For a new project I decided to setup a network monitoring system. I wanted to know when the many servers/switches and printers I managed were up and running or running low on drive space, services not running etc and along with error logs monitoring. So after browsing for many solutions I found Nagios. Possibly the best open-source projects available for this need. Now to find a suitable OS as it's Linux based. So I started with CentOS as it was raved about and reccommended to me. To be fair, It wasn't right for me. Took too much effort tracking down solutions and some of the commands were a little unfamilar to me. So next... Ubuntu.. Now the word I'd like to use I can't, but lets just say Wow... I like it. I love the repository and the fact that almost any issue I have that I Google gives me results. Someone's already been there and fixed it, or if not fixed it, there seems to be a highly active and friendly community that's helped as much as possible. So I'm happily stuck with it now. ):P
[I][U]
Impressed[/U][/I]
Anyway.. So impressed was I, after installing an operating system and Nagios and having it running (without all my data) in under an hour from barely using linux since Redhat & Calderra about 8-10 years ago, I decided that I might just try this at home. So I stuck Ubuntu on a USB stick and took it home. Ran the live distro happily with all hardware found, so I installed it. Sweet. I can't explain (you've probably already been here) how easy to install things. If you read about a piece of software that's open-source/free you can try it out with the good-ol' apt-get install ....... Simple. So on has gone the nVidia extras, compiz fusion, wine, Kompozer, OpenOffice plus a few other windows software replacements. I have barely touched my other machine since this... Until... One problem... I play Project Reality a lot... along with Battlefield 2 in fair amounts. So using Wine, the game works fine.. It's just that blasted punkbuster. The only solutions to getting around pb in wine would render it pointless and open the game up to any hack that's ever been made being made again... Wise move to not release the solution, although I can't help but wish it was me that knew it... So I've decided I can't switch my main (Gaming) machine over as games like that I play will not for the life in me work. But other games do work. The amount of games released on linux seems short, considering it's capabilities and stability I'm surprised, but Wine is a godsend with games and software running sweet-as-hell or in some cases better on Ubuntu. 

_More Tests_
So next test.... I have a Asus 901 EeePC running Windows 7 RC1 (Quite smoothly to be fair) that I'd like to test the Netbook Version of Ubuntu on it. You see we distribute these machines out to mobile staff for light net usage and email checking plus the rest of it. Nothing too fancy just a quick, light reliable system. Happily wiping the 2 SolidStateDrives (which total a whopping 16Gb -- ooOOooo) and chucking on if I remember correctly eeeXubuntu (XFCE Ubuntu for eeePC's) was happy as larry. does everything I need and is user friendly. I even gave it to some problem staff to test and after a few mods and tweaks didn't require much more interaction. Anyway I've since found the official Netbook Remix ad I'm about to trial that (not sure quite on what the differences are just yet).

_The Future_
So the next project I've got lined up is a replacement for a windows machine I built. Basically is duplicates mp3 players & usb sticks. Up to 24 at a time (A: B: & D: - Z: drive restrictions). I'm presuming with Ubuntu/linux I'll be able to do more as it doesn't use A: B: drive conventions but a more expandable /media/drive1 system I'll be able to duplicate more than 24 at a time. Then just need to replace the scripts I have with linux equivilents (which I'm pretty sure I've almost already figured out) and design a GUI then I could see if that is less buggy than windows (which creates ghosts in the registry and forever spawns issues which need fixes to be run every-time).. - Anyone interested in this PM me.

_The conclusion..._
After one week I've setup 3 machines and plan to create one more
:- One Ubuntu server with Apache, MySQL, PHP5 & a light Mail Server with Nagios, NagioSQL, NagioVis and a few other network resources and services.
:- One Laptop (9.04 Desktop) running replacements for all my windows sofware and to test and play with as many features as possible.
:- One EeePC running eeeXubuntu and next Netbook remix 9.04
:- Plus one one the way... USB Duplication.

More than impressed with a free community providing such a great solution to home and business/education use. I will be reccommending it to friends and assisting in finding solutions and replacements for the software they use (basically Office 2003/2007 and the internet). I know.. Easy eh?!

So, If you are still with me (Major first post!) then thanks for reading. And thanks for being part of the community that's already provided several solutions to the problems I've run into. Thanks for the free software and I'll see you around. :D ):P
