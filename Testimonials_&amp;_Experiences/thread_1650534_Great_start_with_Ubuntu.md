### Post by Emir70 on 2010-12-21
Hi all, just wanted to share my install experience and thank the devs for the great work done with 10.10.

While I've tried (and used) Ubuntu before, I kept using Sabayon/Gentoo due to hardware incompatibilities on my main laptop (which required me to use a newish kernel version that was not yet available on Ubuntu at the time I tried it). I've been a Linux user for quite some time now, and switch back to Windows on very rare occasions only, mostly to use work related software such as Visio or MS Project (yes, I know there are OSS alternatives to these, but older Project and Visio files are not readable from Dia or Planner).

Anyway, after an update gone wrong, my X server stopped working on my Gentoo installation. While the problem was solvable with a rebuild, I needed a quick workaround and decided to use an USB live distro until I had the spare time to rebuild X from source. So I went for Ubuntu 10.10 AMD64.

What to say, a few days later I decided to switch to Ubuntu permanently and are happily using it since... everything just worked from the very first boot: WiFi, sound, Suspend to RAM, Bluetooth, Hibernation, Network printers, you name it; even my broadband wireless dongle just worked without fuss! Pretty impressive!

So, thanks for the excellent work! I have tried a fair amount of distros, but so far never had such a painless experience as with Ubuntu 10.10!
