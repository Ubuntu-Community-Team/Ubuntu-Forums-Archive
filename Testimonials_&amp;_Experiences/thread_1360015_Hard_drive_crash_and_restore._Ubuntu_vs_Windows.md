### Post by OnlyTim on 2009-12-20
Hi All,

Had my first hard drive crash since switching to Ubuntu four months ago. Normally, with Windows, I'd have to take a whole day just to reload the OS and install all the programs, apps, and drivers associated with it.

Luckily, a month ago, I read a post here about someone using Deja Dup as a back up program so I backed up to my Toshiba 250G backup drive. I'm so glad I did!

Put the new HD in. Re-installed Karmic then reloaded Deja Dup from the Ubuntu software center and hit restore.

Amazingly, everything was restored as it was prior to the HD crash! Right down to the old wallpaper! All emails, contacts etc. 

The best part was it only took an hour and a half! I'm doing the happy dance!

Cheers!
