### Post by 3rdalbum on 2012-05-15
Unity is not for everyone, but I (along with a lot of fellow Ubuntu users) believe it's a great step forward in desktops.

Computing is one of those things that changes so quickly. It's the nature of the industry. Linux changes even quicker; it's the nature of Linux development. If you want to use a computer, you'll have to accept change. If you want to use Ubuntu, you'll have to accept rapid change. If you don't like too much change, then Ubuntu is not the distro for you - try Debian or CentOS.

If the PC desktop experience never changed, we'd still be typing in commands on an MS-DOS terminal. Big changes are initially painful, but eventually even Blind Freddie can see the benefits and eventually nobody can live with the old-fashioned desktop.

If you still can't get to grips with Unity, there are other desktops. Gnome 3, KDE... there's even a Gnome Classic (Fallback) option that emulates the old Gnome 2 desktop for you, right there on 12.04. Try it.
