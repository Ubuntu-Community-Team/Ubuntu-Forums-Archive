Well, I don't know about that. But as a developer running my own business I've been kind of waiting around for Linux to come along. I've chosen MS as a platform for a while because of the rapid deployment time of the test and development enviroments. And of course the well known frontend apps and compability. But I always loved the way Gnome allows for customization of the workspace.

But after getting deeper and deeper into that mess, Linux and the simplicity in this system makes it more and more attractive. And now that there's a disti out which almost completely automates just about everything so I can focus on getting work done is a major improvement to me. As time = cash I need to keep maintanence low. After trying to run Fedora Core at some occations I usually end up in Windows. But this time it feels different. Gutsy seems much more stable, redundant and reliable. It allows for complete customization presented in a very usable and intuitve fashion and the simple and logical layout of the "x" type OS. Along with the compability it offers, Gutsy with the visual effects on makes Vista look like something the cat dragged in.

I'll have to get a server edition up now, so I can get going with some php and java training.

Fun fact: I went to the MS Partner convention in Denver this summer and listened to some WS2008 speakers.. They have a "Core" edition now. Like a console textmode installation of Windows Server without explorer...to save hw/virtual-hw cost and give priority to services and so forth.... and I'm like.....hmmm....yeah, reminds me of this other operating system where you have been able to choose to install a window manager for years.....
