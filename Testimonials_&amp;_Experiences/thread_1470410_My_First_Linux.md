### Post by jonnny_j22 on 2010-05-02
Hey all,
I just thought I'd post up my experience of moving over to Ubuntu (having never used Linux before) To reassure anyone who's looking around and thinking of switching, and also to thank the users of this forum for their help and to let them know that even though us newbies ask the same questions all the time and drive you nuts it really is very appreciated!

So go back about 6 weeks; I had been using windows xp for 7 months since the last install and it was just 'naff' again - we've all been there - I had always been interested in what I'd heard about linux - not linked to one power hungry company, developed by its users, safer and more stable than windows, but put off by hearing that it's really hard because you have to know programming language.

So I started googling about and found out that everything i thought i knew was wrong - linux is mostly developed by professional programmers creating professional OS, and for most releases you don't need to know anything other than the more complicated parts of using pc - installing things etc.

I was quickly told to check out this forum due to it's community, and having been put off from posting in other forums before I've read around,began searching and reading around the newbies forum and followed some advice i saw about the live CD's.

I decided to use the live CD's for fedora and ubuntu to see which i liked. These are great for newb's because you can try out the operating system without affecting your current files. Idecided I liked ubuntu (partly because I knew about this community) and went for it.

I wont lie, installing it wasn't quite as easy as i had hoped, but i think this was because it was the first time i had done it and was putting windows and linux on the same disc. Thanks to everyone asking the same things here i was able to find out what to do, and strongly urge everyone thinking of switching not to be afraid to ask for a step by step instillation if going for 2 operating systems - nearly everyone on here has ubuntu and windows on their hard drive and they will be happy to help! I only got a bit unstuck because i tryed to gues my way through the 1 bit i hadn't expected and then spent an hour moving partitions about.

Most importantly for newbies like me, don't panic! I havn't had to install one driver since I installed ubuntu - once it loaded up after the install it was just done, ready to use! I put in my wireless password and off I went. The only thing that hasn't worked for me has been a little app called gwibber that connects your desktop to the social network sites, but this seems to be a problem with the software, not the operating system.

And you really don't have to know any commands, sure if you use these forums, then you start to pick bits up as people tell you things, but i went 3 days before i used one. You use it just like you used windows if you want. Don't let the reputation of linux and commands scare you, sure as you use it it more you start to use more and more code because it's more convenient, but you don't have to know anything!

The only thing I would recommend to newb's is to install 'wine' - this will let you run almost any windows program in ubuntu and massively ease your transition by using the programs you are familiar with and then experimenting with what linux has to offer!

The people here have been so helpful and I would really like to thank them for all their help, without their patience towards all the newbie threads i read I probably wouldn't have decided to switch, and it has been the best pc decision i have made - I havn't even used my windows xp since i installed it again.

I hope I have helped inspire someone to give it a go, and remember if you don't know, ask!
