### Post by nerd0795 on 2008-11-01
Well, Ubuntu 8.10 got released so I thought I should go check it out.  I already had some problems with compiz and some other things with ubuntu.  Well... so I go start upgrading.  The download takes 6 hours.  When it finished it told me that it couldn't get 50 files (my isp probably aborted my connection from my high bandwith usage.  Though I have a 20GB limit per month._  Since it was late I was gonna wait till tomorrow to continue.  So after I woke up (today) I started to continue the download again.  I had to make sure to move the mouse every minute because in two it would go to screensaver and screw up the GUI.  So it finally started to install.  Took about an hour.  So I went and watched what it said in Terminal (to occupy my time).  Then the all of a sudden everything looked so much better.  I got better detail, an awesome wallpaper, colorful colors.  So then it finished.  I restarted and everything was good.  Booted up faster.  Then I noticed one of the problems I had in the log-in screen was fixed. (After logging in the log on screen I would be stuck in a black screen for a minute then I would be at my desktop)  everything looked so much better.  But I noticed... my theme was different it was a mac os x like theme.  I was like wtc?  I last enabled Human.  So I found now emerald works.  Since I tried 3 months ago to make a Mac OS X theme.



Now, disabling it drove me crazy.  I finally figured out to go in compiz-manager and edit window's decorator's destination thingy, back to default.  It worked :)

So now I'm using a faster, extremely stable, awesome ubuntu installation :). 

Hi-5 to the programmers who made ubuntu and the upgrade!:guitar:
