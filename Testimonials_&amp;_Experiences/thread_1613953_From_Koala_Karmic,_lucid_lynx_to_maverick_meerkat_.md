Now in my maverick meerkat experience the touchpad is not working properly (not really working) only my mouse( I have tried many solutions). The next problem in meerkat is high temperature (fan not working) but I tried to close the lid then open again. after opening the lid the fan start working (fan running forever). it will save my laptop from burning. so for now, i can use ubuntu. it's been couple of years I wanted to install this ubuntu but just now, I can use it with the help of some tips to prevent high temp (fan to work), also the touchpad enable and disable using terminal.
