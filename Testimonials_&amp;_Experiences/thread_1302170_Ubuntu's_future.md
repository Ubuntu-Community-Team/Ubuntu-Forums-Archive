### Post by 3rdalbum on 2009-10-27
Free and Open-Source software is about choice. A huge number of Ubuntu users are pragmatists - they prefer FLOSS, but they'd also like to be able to buy and install commercial software easily if they so want/need. It also gives ISVs some interesting numbers of how many people are willing to buy software for Linux, which means that more ISVs will port software to Linux. Which is great if your needs are only satisfied by some commercial software that you can't live/work without and you're a free software pragmatist.

Would I like to buy Call Of Duty Modern Warfare 2 for Linux? Oh yes!

Canonical is helping to try and make this sort of thing happen. If you're opposed to commercial software on our platform, or proprietary software in general, then it sounds like you should be using gNewSense instead of Ubuntu.

I don't see commercial software being pushed over FLOSS any time soon on Ubuntu. It doesn't sound like something Canonical would do.
