### Post by Monstroxus on 2007-01-31
I installed Edgy for a few months ago, but actually never "dared" to install Beryl. I saw some demos on youtube and thought, I wished I could install something like this.

I always thought it was too difficult for a noob like me... not! I just followed the instructions I found on the ubuntu forums etc. and within 20 minutes I was running Beryl. I thought this will run very slow. (Lenovo Laptop 1.6GHz with a shitty onboard graphic card, you can not even play decent games with the card, but its a great laptop and cheap!) so... Beryl will stutter, hang for sure...no, it actually runs beautiful, smooth etc.

Later on when searching for some Beryl demos on youtube (to see the Beryl features in action, so I could try to do the same), I came across Ubuntu + Beryl + Kiba Dock... what is that, I thought? Kiba Dock?... after seeing the youtube demo I thought, lets try installing that as well, which I did and it works perfect. Nice funny bounching icons! Windows Vista has a slogan that mentions something like, let the Wow begin, or something like that... well I think it will be pretty difficult, if not impossible to uber wow ubuntu+beryl+kiba dock.

Beryl is not only eye candy, it is very useful too, for example the rotating cube and the option that you can go the right top corner with your cursor and see all the applications running in a birds eye view is very neat too!

Lets see what the future will bring... this evening I will try to install "recordmydesktop", see if I can record some action, together with Kraftwerk's "Home Computer" playing :)

P.S. I showed all of this to my wife's niece and she asked me to install all of this on her pc as well... uh oh... hope it will work on her pc as well.
