### Post by Morot313 on 2008-03-02
**Video playback sucks**
I want to watch video. The video quality is poor, the same video looks much better in Windows. The colors are bad, and it looks pixelated. It does not look smooth. When skipping in the movide, there is a delay.

**Rhythmbox is dumb**
I use Rhythmbox to play music. Everytime I start Rhythmbox, it plays the first song in the library, it never remembers the last song I played. I get sick and tired of hearing the same song. When I listen to radio with Rhythmbox, after a while, it stops, and I cant reconnect, or listen to media in the library, I must restart Rhythmbox.

**Restart after update**
People told me "Windows sucks, it needs restart everytime its update".
In Ubuntu after I use Update Manager, it asks me to restart the computer too.

**It thinks files are executable that are not**
So I mount my NTFS partitions. I double click on document.txt, Ubuntu tells me "Do you want to run "document.txt", or display its contents?", "document.txt" is an executable text file.", and presents me with 4 options. It's a plain text file damnit, just open it in a text editor.
Then it marks .dll and .ini and other files as executables, they're not.

**Stability**
Linux users constantly spread FUD saying Windows is unstable, crashes, etc.
For dumb people who install AOL, BonziBuddy, CometCursor, RealPlayer, Norton Internet Security, and surf porno with Internet Explorer, sure right.
But for the people who know how to use it, its rock solid, very stable, never crashes. I've had no crashes in years with Windows XP.
Applications in Ubuntu crash the whole time, its worse than Windows ME. One app can freeze the whole system making it all unresponsive.

**Mouse button works dumb**
I have a mouse with many buttons. A Microsoft IntelliMouse Exporer, one of the worlds most popular mices. It has two buttons on the side for forward/backward. In Ubuntu, they're mapped all wrong, and used for right-click and paste.

**Trashbin**
There are no secure deletion mode to securely wipe files. They have this in Mac OS X.
When I press the delete button in Nautilus, it deletes the file without confirmation or warning.

**Drag & drop**
You cant drop and drop files into executable binary. Example compile a program called "foo" and then drag a file called "bar" into the foo icon and make foo run with bar as a parameter. :(

**Copy & paste**
Open Firefox, copy a string, close Firefox, and now try to paste it. It don't work. :(
Windows have had this functionality since Windows 3.11, released 15 years ago.
1992 called and they want their copy & paste back.
When you cut a file, there is no visual indication that it is cut. The icon does not get dimmed.

**Calculator**
The calculator (gcalc) in Ubuntu has no colors on the button. On Windows the [calculator]("http://en.wikipedia.org/wiki/Calculator_(Windows)") have the buttons colored by their functions. So numbers have one color, and functions have other colors.

**ISO images**
In Mac OS X you can double-click on ISO images to mount them. In Ubuntu you cant. Surprise! :(

**Keys get stuck**
Ubuntu is buggy. Keys get stuck, and press the same button all the time, when I am trying to play games or something. :(

**Fonts are ugly**
The fonts in Ubuntu are ugly. They look bad. :(

**Dictionary**
The dictionary is dumb. It does not provide with you real-time feedback of word suggestions as you type.
[COLOR="MediumTurquoise"]Windows don't come with any dictionary at all though. But Mac OS X does, I think.[/COLOR]

**System Log Viewer**
The System Log Viewer is dumb, it basically just opens the log files in a text box, I can do the same in a text editor. You cant sort anything. The Event Viewer in Windows is much better, events have icons depending on warning, error, information. You can sort by time, date, severity, event, app, etc.

**System Monitor**
The [Windows Task Manager]("http://en.wikipedia.org/wiki/Windows_Task_Manager") in Vista is much better than the System Monitor in Ubuntu. You have a "Applications", "Processes" and "Services" tabs instead of just "Processes".
[COLOR="MediumTurquoise"]In Ubuntu it does have smooth graphs though.[/COLOR] :)

**X Window System**
In Windows I can use 100 hz refresh rate. In Ubuntu I can only use 50 hz.

**Window management**
The window management is dumb. Its unintuitive. Often when you start apps, it places them in the upper-left corner. They should be more in the center of the screen.
When you have apps like Firefox or Rhythmbox in maximized, then you double-click the titlebar to unmaximize them, it often just gets 1px smaller, or gets unmaximized into the top-left corner.

**Games**
The chess game, Chess Titans in Vista is way cool. Chess (and other games) in Ubuntu sucks.
[COLOR="MediumTurquoise"]Its nice that Ubuntu pauses the game when their window loose focus though, and Ubuntu comes with more games than Windows, including brain games like Soduku.[/COLOR] :)

**Webcam**
Pidgin does not support webcam. Can't chat with girls! :(




**The good**
[LIST]
[*]It comes with a BitTorrent client.
[*]Update Manager updates all packages in the system.
[*]Add/remove lets you easily install lots of software without effort. Repositories are large.
[*]gedit has syntax highlightning.
[*]Compiz (with desktop cube) is pretty damn cool.
[*]The SkyRocket screensaver is awesome, I can just sit and watch it excited like a little kid.
[*]It comes with a virtual PDF printer.
[*]It comes with software to read PDF, DVI, PostScript and DjVu files. (Evince). It's quick too.
[*]Comes with software to read ODF documents (OpenOffice.org)
[*]Comes with powerful graphics editing software (GIMP).
[*]The terminal has colored output and filename auto-completion.
[*]Pidgin supports 15 different IM protocols.
[*]It can read partitions from other operating systems.
[*]The 'iptables' firewall is awesome.
[/LIST]
