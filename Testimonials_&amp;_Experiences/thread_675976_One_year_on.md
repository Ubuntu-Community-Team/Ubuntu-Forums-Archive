### Post by GavinZac on 2008-01-23
Just to begin, I'm very much a geek so my experiences are probably not all that appealing to Average Joe, but I hope somewhere a fellow geek reads this and thinks "why not give it a go?".

I first became tempted by linux when I realised that a lot of the apps I was using in Windows were open source and native to linux - openoffice, firefox, gimp, *AMP, pidgin. This good impression of the software already available persuaded me to install linux, and specifically, Fiesty Fawn given the hype around it at the time. I initially had one or two problems with my graphics card but everything else pretty much worked out of the box. It was around this time too that I got back into console gaming so I didn't miss that much PC gaming which I feel has become saturated with same-same FPS's and MMORPGs.

Eventually I learned and picked things up to the point where I can now help other beginners out, and I'm try to learn Python so that I can develop for the desktop aswell as in PHP for the web. I'm now at the point where I consider running virtualised web servers and setting up file servers as something I am capable of, which I probably never could have done in Windows. I have Ubuntu on this laptop I'm typing with right now since the first day I got it - Fiesty wouldn't work, but Gutsy (then, in alpha!) worked out of the box. Apart from having to keep an eye on the kernel updates and reinstall the graphics driver accordingly, everything was smooth :)

Along the way I've tried, and admittedly, failed to convince my girlfriend to switch - her problem not being with ubuntu, which she is well able to use, but that her last laptop broke so she's paranoid about the new one. However, I ressurrected an ancient (1998, pentium 2 :D ) laptop using Xubuntu which is now a perfectly good web browser/word processor/IM machine for her younger sister. In setting it up, I like Xubuntu so much that I installed it on my father's old office computer - he never really got the hang of Windows anyway so he's able to do as much on Xubuntu as he ever was on Windows, without having to worry about viruses, etc.

Finally, I got very brave and using spare parts around the house, put together a little computer for under my tv. Turns out, the easiest thing I've ever done with Linux was installing Linux MCE + Kubuntu!

The biggest problem I have had was when I bought a new motherboard for my desktop, my original ubuntu install. There was an incompatable SATA/IDE controller, but the problem was that the manufacturer of the motherboard (MSI) has never updated the BIOS of this controller, which apparantly works fine once upgraded, for ASUS motherboards. Oh well, I'm just never buying from MSI again. I got another motherboard, and once again everything worked out of the box - impressive since I've again gone for an alpha release, Hardy Heron.

So my initial curiosity is now an addiction: every computer in my home has *buntu on it, and I use linux almost exclusively. I'm looking forward to the next time I buy a computer so I can confidently specify: No Windows Please!
