### Post by vexorian on 2007-06-15
Hey wanted to share.

Last year I heard about ubuntu after getting an internet connection at home, since I had too much browsing to do I heard about this whole linux for human beings thing, was very interested, I had previous (failed) experiences at trying Linux but they didn't go well, so it was a good slogan since I was a human being trying to use Linux.

The installation was easy but in text mode so I knew not a lot of people would think that was friendly at all. But that Ubuntu version I tried was fine, was impressive at the improvements and loved the package manager, the applications bundled were fine and googling what I wanted to do + "ubuntu" helped me find some solutions for little issues I was having, I could even find some neat games, as in prboom which let me play doom all over again... 

I eventually also tried downloading the KDE packages to try Kubuntu, I liked it a little more because of the default theme and icons. I was impressed by K3b, I was finally able to burn CDs.

The support for ssh and sftp in Linux is so impressive! there are solutions for windows but they aren't implemented to the desktop so it is less effective than handling them as folders for example, my web dev times were more productive in Kubuntu than in windows.

I tried changing the grub configuration but I couldn't find any option in the system configuration sections, I eventually learned that googling what you want plus "ubuntu" would almost always lead you to precanned solutions, I learned how to edit my menu.lst and gave it some work. Since my brother also uses this comp I made windows XP the default operating system and added a delay, I also changed the splash screen, all thanks to this forum actually.

OpenOffice was fine for me, I only use word processor and I liked the fact it shows you how the layout works (shows those table like lines that help you notice what's going on) word was more obscure at this and I didn't like it, I was however impressed at the equation addon/editor something that was much better (and faster to use) than MS' equation editor. It helped me a lot since I have to do plenty of homework that requires math notation.

What held me to windows was that even though I could run warcraft III in WINE, the editor would simply fail, I mod wc3 for a hobby so I have to boot in windows too much.

But I noticed there was always a reason to go back to Kubuntu, whenever I had to burn CDs or edit the web or do homework I booted to Kubuntu since it was simply more effective at that. 

--
This month I decided it was time to update Ubuntu, I heard and was curious about the improvements in Feisty, after upgrading to Dapper I found that upgrades are hard to do, take more times and you end with a more bulky system, I also didn't want to have 2 dekstop variants anymore, in fact wanted more free space for my Ubuntu system since all the previous tests kept it kind of full, so I backuped my home folder and wiped everything.

I decided to download feisty, I would say that to hear that the latest Ubuntu was using tango' like icons and the fact that I was checking those icons in windows and I liked a lot of them and the style, I thought it was a good idea to give Ubuntu a try again and do not download Kubuntu directly. 

**Feisty**
After downloading the CD I booted on it. Feisty is VERY impressive of an improvement. I was shocked at how much things improved in Ubuntu, in fact I actually liked the default theme, some of the games looked much better and icons were a lot more elegant.

I also noticed that I was in gnome again thus I could use a feature I missed during Kubuntu, the emblems! This was a quick disappointment  though since I noticed that all the emblems looked the same in the human theme.

I do not think that's a good thing I certainly wasn't able to recognize folders with different emblems easily, I think that's a huge usability issue. But luckily it was VERY easy to change the theme, tried tangerine and then Tango, stayed with tango.

The whole live-cd installation is such a feature, I have installed windows 3.11,95,98, and XP  plenty of times, and got to say that even installing XP is very painful.

Live-cd also easily detects even my ntfs partitions so it would actually make a great recovery tool unlike the windows XP cd which barely gives me a very limited console.

With the live-cd I did something that is just amazing in therms of installation of OS, I was able to browse the web during the installation! That is such a nice detail!

After installation everything was almost fine, but when I tried openoffice it had no icons! I figured tango icon theme had no icons for open office, found a package though , this is another usability issue that would be good to fix, the Tango theme should either not be included or be included complete.


But all the features in feisty are very impressive, in short it is visually much better, It is also nice that it would tell you "you need codecs" and assist you getting them just after you double click an mp3 file.  It is also fine that if you type wine the first time on terminal it would tell you "you have to do sudo apt-get wine" the same with many other tools I tried to use.  I also appreciate I could install Java stuff from package manager that was VERY cool.

An important thing is that I am actually staying with human+tangerine (after I found out tangerine got emblems that are not the same) with the previous version I tried I could not stand the default theme.

 If something can improve this much in a year I can not help but wonder what I can expect for 2008:)
