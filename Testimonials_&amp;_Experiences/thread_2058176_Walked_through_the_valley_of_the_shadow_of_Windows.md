### Post by 3rdalbum on 2012-09-15
I posted a bit of a rant a few weeks ago, about Ubuntu, and how I was going to try Windows 8.

Well, I still have Windows 8... but I'm currently only using it for a couple of games I have lying around. Everything else, I use Ubuntu for.

About half the time when I start up Windows 8, it recognises my keyboard and mouse, but any other USB devices (like wifi, flash memory or Android phone) it claims they have "malfunctioned" and so Windows can't use them. This is pretty useless for me, as my only means of Internet is through phone tethering.

When Windows decided to actually work with phone tethering and I tried to post a couple of messages to the Cnet forums, I found that the text fields in Internet Explorer (the normal one, not the Metro one) weren't working properly; there was also some bug that prevented the reply field from appearing.

Yeah I'd probably have more success with Windows 7, especially since I don't touch any of the Metro stuff. But currently everything I want to do on my computer, apart from those games, still works fine on Ubuntu.

I might run into something that doesn't, where I may need Windows. But until then, and until Microsoft can actually get basic functionality to work (USB, text fields), then I don't think Windows is ready for the desktop... no, just kidding, I don't think I'll be using Windows much at all.

Ubuntu is far, far superior to Windows 8 and XP (I'd rather use 8 than XP). It's also quite superior to Windows 7 or Vista. I just wish there was something better than Ubuntu.
