### Post by Ginsu543 on 2010-07-25
Or if you have a 1GB USB thumb drive, you can use [_UNetBootin_]("http://unetbootin.sourceforge.net/") to create a LiveUSB stick and run Ubuntu off that. It would be exactly like running off the LiveCD but only faster (assuming your computer can boot off USB).

But to really get the most out of Ubuntu, you need to install the full OS to a hard drive partition. I too made the switch from Windows to Ubuntu as my primary OS and have never regretted it. For the few programs I need on the Windows side, I run Windows XP as a virtual machine inside Ubuntu using VirtualBox. I used to dual-boot, but virtualization is more convenient because I don't have to shut down and reboot to get into Windows.

The only reason I can think of to run Windows natively is to play games available only in Windows.
