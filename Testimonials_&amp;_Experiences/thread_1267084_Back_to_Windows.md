EDIT:

Came across this post and it mirrored what I was thinking.

[https://answers.launchpad.net/ubuntu/+faq/312](https://answers.launchpad.net/ubuntu/+faq/312)

**Why don't you just buy an inexpensive external card reader for use with your Sony memory sticks?**

I own and use an external memory card reader (10 different kinds I believe) even though my laptop will read SD and something else.

Of course, I only own SD memory cards but on occasion I have to read other card types and the external reader comes in handy.

I also made the observation that you have only one post here in the forums.

Why didn't you ask for help?

I also wonder what version of Ubuntu you were using?

We see lots of folks like this here where their only post to the forums is the last time they ever come here and think that it's all Ubuntu's fault.

I used the newest version of Ubuntu and it sucks because it's so buggy, they say.

My whatsit doohickee doesn't work in Ubuntu. LINUX SUKS!

My answers:

Do more research. Google is your friend.

Not all hardware works in Linux. There are many good alternatives or work arounds that you will discover if you do the smallest amount of research yourself. The external card reader is one of those "work arounds".

Linux in general, or Ubuntu in particular, are not based on Windows or any Windows or Microsoft code at all. Ubuntu is NOT Windows and will not act like Windows performing many tasks. One must learn to accommodate the operating system. Linux is not Windows.

Hardware compatibility is a big issue. If the hardware isn't compatible, Linux will fail.

Using the stable versions of Ubuntu (8.04 or 8.10) as opposed to the latest, greatest versions will probably bring great joy to those who suffered from Jaunty issues or the "bleeding edge" syndrome, where one believes that the latest version available will work great, when the opposite may be true depending what the end user is trying to accomplish.

And I always say that if Windows suits your needs better, please use Windows. Use whatever it takes for you to get work done. do not define yourself personally or professionally by the operating system that you use.
