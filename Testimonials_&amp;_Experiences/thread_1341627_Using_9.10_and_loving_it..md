### Post by aaronchall on 2009-11-29
OK, what follows is a list of thoughts and ideas about Ubuntu, as well as my experience, but the bottom line is that I view Ubuntu as one free AND premium OS distribution, with free AND top-level software accessible directly from the organization that puts out the OS.  I don't get obtuse messages saying "* is not here, insert disk" or "* is only available in the premium package."  While I understand profit maximization and the profit motive, and I believe it has provided impetus for the creation of Linux and Ubuntu, I also find it quite infuriating to be constantly asked for more and more money for something that was written long ago and that has little or no duplication costs.  Money, for something that gets continually bloated, broken, retarded, infected, hijacked, spied upon.  Money for a black box that doesn't work very well, comes with features broken which can be fixed for an extra fee, that doesn't work well with the majority of software made specifically for it, that won't explain to developers how to get software to work well with it, who's own software constantly messes up on it, that requires a team of IT people to keep it working or lock it down so no one can disturb its fragile state.  It doesn't needlessly suggest software that breaks it (desktop search, anyone?).  Ok, Rant over.  On to Ubuntu...

Here's my experience thus far:

I've had similar problems installing 9.10 as I had with 9.04, sound and video issues.  The issues were minimal, and I've worked through them, however, and this distribution seems to be running Youtube and other video more smoothly than before.  

I've done a bit of customization, since I don't care for the brown theme (I went with one of the popular blue ones, selected the Bay wallpaper, a dark one, and made the Terminal have the same faded background.  In 9.04 I went with the cloud picture I found in the examples folder.  Both cool.  Dark, yet not evil.

[The problem with changing graphic themes is that there can be a real clash when visiting contrasting web pages or using contrasting programs.  Makes me think web page designers should go a little more neutral.  Thank God for Superkey-N (makes the browser invert the colors, compiz I think is responsible...).]

I would like to see a program that would let me get to all the temperature sensor outputs on my laptop, which I had in Windows (but of course it was from a free source, not the manufacturer or MS...)

Anyways, I have completely over-written my Windows partition with 9.10.  I am happy I did it.  As a web power user, I like to open tab after tab after tab for each link I want to follow.  Ubuntu lets me do this all day long, while Windows slows to a crawl at about 20 to 30.  Even if Firefox messes up, it doesn't slow down the rest of the system, and it reopens with the tabs I had open.

I like knowing my /home partition is secure, if my computer gets swiped.

I really enjoy the less than 30 second boot up time.

Zotero, an open source citation software, works better than Office 2007's citation managment.

Folders/directory organization seem to make more sense.  I can find stuff faster.  

The command line is fun for me to use, since I used to spend hours playing with DOS even though it completely lacked documentation.  It's like DOS with documentation.  It's so cool.

Photoshop type software (GIMP) comes standard.  Office software comes standard. Instant messaging comes standard (still haven't figured it out, though). Notepad (Tomboy) automatically saves what you have in it.  Games don't come packaged with bloatware and spyware.  Audio and video editing packages are either standard or easily downloaded from the repositories.  CD ripping software comes standard.  Multimedia packages and codecs (some of which don't come on Windows either) are easy to find and install by going to the multimedia thread.  Crap doesn't automatically load every time you boot.  My OS uses less of my computer's resources leaving more for things I want it to do.

OK, I think that's enough.  If you think I'm being unfair to anyone, feel free to correct me.  But it's nice to have a laptop that instead of me coaxing it along and hoping against hope that it doesn't crash, it flies as fast as I want it to...
