### Post by HistoryGuy on 2007-05-20
I started using Ubuntu about a month and a half ago. I wasn't sure that Linux would be right for me, so I made sure that the dual boot would not delete XP from my system in case it didn't work out. I've only booted up XP twice since the Ubuntu install and each time reminded me why I wanted something new. 

I know that Ubuntu doesn't have much in the way of game support *yet*, but I think I can convince all but the hardcore gamers in my life to make it their default OS. A stable, legal, fast, and a virtually virus free OS isn't a tough sell. :D Even my technophobic mother watched me install programs on Ubuntu and said, "I want that!"

Thank you to all of the Ubuntu community members that provide programming, art, technical support, financial support, and all of the other things that it took to make and keep such a great distro running! I hope I can find a way to be helpful to the Ubuntu community as well.
