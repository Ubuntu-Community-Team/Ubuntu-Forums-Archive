### Post by richard.e.morton on 2008-08-02
Hi There,

I have been playing with Ubuntu, Kubuntu and recently Xubuntu. Although I have played with Linux before (10 years ago, and multiple tries with it since) I have finally got rid of Windows from my laptop and now have Kubuntu on it... after trying Ubuntu first.

I thought I'd make a few observations about my experiences which I hope will make it's way back to the dev teams. I am running an IBM Thinkpad T60 which for those not familiar with Thinkpad hardware has some nice touches and many standard ones.

Here are a few things about the T60 and the out of the box experience... All these features work in Windoze out of the box. I have made the change to Linux, but many many people will not change to a harder to use OS and that means we need to make the out of the box experience easy for all common hardware platforms.

1. Thinkpads generally have dual pointing devices (stick and pad, with buttons for each. 
Ubuntu stick doesn't work/stick buttons dont work
Kubuntu/Xubuntu everything works

2. Thinklight is a small light to gently illumintate the keyboard in low-lighting conditions, it is switchd on via Fn+PgUp.
works fine for all

3. There are dedicated volume up, down and mute. 
works fine for all

4. Fn+arrow-keys provide media controls play/pause, stop, next previous. 
Don't work in any of the distributions

5. The keys above left and right arrow keys are dedicated back & forward buttons for web and file browsing.
Ubuntu - doesn't work
Kubuntu - works fine

6. There is also a wifi and bluetooth activity light. 
doesn't work in any of the distributions, been told need to recompile the kernel to resolve this... you're kidding right!?!?

7. Fn+F7 swaps between lcd panel and external VGA connector
doesn't work in any of the distributions

8. Fn+F5 switches on and off WiFi/Bluetooth
works in Kubuntu, sorry didn't test in other distributions

9. Fn+F3 is show battery state
works in Kubuntu, sorry didn't test in other distributions

10.Fn+F2 is lock system
doesnt work in any of the distribution

11.Fn+F12 suspend to disk
doesnt work in any of the distribution

12.Fn+Home/End brightness Up/Down
works fine in all distributions

13.Fn+Space - Zoom
doesn't work in any of the distributions

14.Special Key (Windows Key)
does nothing in all distributions

15.menu key (right click key)
does nothing in all distributions

16. ThinkVantage Button
doesn't do anything in any of the distributions

17. Suspend/ACPI
doesn't work in Kubuntu

18. System temperature
much much hotter than under XP, even with all the glitz turned off

19. Personal preference, prefer a single bar on screen and it to be at the top of the screen with minimal space used for menus (single button, yep... like Vista, and the way you can configure all desktops, but can't we have that as the default - it is what most users expect!)

20. There is no differentiation between an icon which is a menu, system tray icon or a launcher... makes the UI confusing to newbies... you get used to it.

21. icon top left of windows
Kubuntu/Xubuntu - double click = close
Ubuntu - double click = nothing

22. Sound
Ubuntu - pretty unreliable, sometimes it works, sometimes I have to change the settings to ALSA or something, sometimes it just wont work.
Kubuntu - absolutely works, everytime, although mute sometimes get confused between applications.

23. Network Files Access
I have a small LAN with Windows and Linux systems in it. All the distributions seem to not allow you to directly use a file from a network location, so you have to copy the file to your local system to use it (upload it via firefox or whatever). Very inconvenient - I think this is a deal breaker for me... all my files are on a file server, if I have to keep copying them locally, alter them and copying them back... I know I am going to forget then I am going to have a version control issue.

24. DVD playback...
I understand the legal issues... it's such a shame that we can't have DVD/MPEG support as standard, especially for the territories that allow zero cost binary distribution. Could we not have a simple guide on first start on how to do common setup tasks... such as dvd/mpeg playback, wifi.

25. support
Ubuntu, well supported, forums get responses...
Kubuntu - 90%+ of the posts are for Ubuntu... much lower support for Kubuntu even if the issue (seems to me at least,) to be in the core.

26. Installing non-package-manager applications
I would agree with several posts that installing applications should be easier than it is currently, downloading a package from a website and installing it should be simpler and have a more unified interface for completing the process (even if there are still multiple packaging methods - rpm, deb, etc). 

27. Virtual Box
I have virtual box installed and when some updates are installed virtual box breaks, and then the next day another update comes out and fixes it again... this is a shame.. you never know if an update will break virtualbox.

28. Interface Tweaks
I would like to see a single area where you can setup the system so that it is more like windows... so a themeing and button setup area which can re-configure many settings in one place to make ubuntu appear and work more like windows (one panel at bottom, one menu button, quick tray with browser, show desktop, system tray), make windows key open the "start" menu.

29. Performance Tweaks
A single area which will enable a change to turn off all the stuff which saps CPU and memory, translucency, wobbly windows... a simple option which optimises for max performance, default, max glitz, optimise for system.

30. Export Desktop Layout
Export the settings of your desktop, layout of panels/toolbars, graphics options (transucency etc). so that you can import them to another user/system. 




ok, so that is a long list... and I don't mean to moan. This is a load of feedback that I hope can be considered for incorporation... this is just a bunch of my views on my experience with Linux... Many of the things I thought wouldn't work, work brilliantly (CD Burning and DVD playback - once you install restricted extras... for instance)


I don't mean to moan, I am using Linux cause I like it and like the concept of free software and I am willing to help out, one of my past roles was UI design for speech systems, not that that helps with GUIs! To get people to be prepared to move to Linux there are a few things we need to sort and I hope this sort of feedback helps. 


Please don't flame me, I mean this feedback in a positive way, Linux is soooo close that it is frustrating. Just a little bit more feedback and development and it'll be there.

I hope this will help... but who knows!

Rich
