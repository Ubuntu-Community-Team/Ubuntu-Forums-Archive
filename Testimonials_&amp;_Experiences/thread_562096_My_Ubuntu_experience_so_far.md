### Post by BSK on 2007-09-28
Hi,

I work as a support tech/network admin in a Windows domain so I use Windows a lot at work and at home but I have been interested in Linux since 1999.

I always tried distro on virtual machines or my main system with a dual-boot setup.  Before finding Ubuntu most Linux distro would not survive too long on my systems since a lot of things were hard to install or configure due in part by my little linux knowledge but each time I installed a distro I could see Linux getting better and better pretty fast so I always had faith.

About 4 months ago I tried the Ubuntu 7.04 live disc, cleaned up my second hdd, that I used in Windows as a backup drive, and installed 7.04 to it.

Installation was very easy, it even detected my Windows drive and configured GRUB to dual boot the two OSs.  Wow!

At first I was a bit lost about playing DVDs and Flash support in Firefox but the support from the community is awesome and I learned fast to correct my newbie problems.  Then I learned about 'apt-get' and now I couldn't live without it.  Being able to do about anything in the console is awesome too, as a network admin I do a lot of work with the Windows console even if it's nothing compared to the linux one so I'm pleased with this.

Even if I have a dual boot I spent about 80% of my time in linux, my girlfriend is learning quite fast to do her things in linux too so she almost never reboot in Windows.

I was also able to put 2 usb keys in a software raid0 array, how great is that!

I can't wait for 7.10 and I plan to get a new faster drive for Ubuntu.

I have to keep Windows for work reasons and for some gaming too because I'm pretty bad at making things work with wine but I can say that Ubuntu has taken priority in my system.

Thanks to everyone related to this awesome distro!
