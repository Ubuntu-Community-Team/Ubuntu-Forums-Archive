### Post by Proffa on 2006-06-28
Greetings to you all Ubuntu forum members,

I've been a Linux user since 1996. I work as a Unix System Administrator and part of my job is to test new Linux/Unix distributions and releases to see how well they perform.

Last february I began testing several Linux distributions. The main purpose of this test was to select a new distro for my workstation. I needed a stable  distribution with resent software. And when I work, I don't want to spend any more time than it is absolutely necessary to get everything in their right places.

After testing six distributions, I decided to start using Ubuntu (Breezy Badger at the time). And I have to say that I've been very pleased with it. A very big thank you to all Ubuntu developers out there!

Just last week I did a dist-upgrade to Dapper, and everything went smoothly. Later this week, I'm going to upgrade my personal mail and development server from Debian to Dapper. Wish me luck :D 

-Proffa
