### Post by 3rdalbum on 2009-10-09
One of the cool things with 64-bit Ubuntu is that, because all 64-bit i386 processors are fairly new, the system is compiled to use modern CPU features that aren't available on old 32-bit-only CPUs.

Some of those newer CPU features increase speed, and some increase security "for free".

And, of course, I can have 6 GiB of RAM addressed which helps keep my system responsive even after I've been ripping DVDs.

Yes, anything that is NOT compiled 32-bit software will work on 64-bit. Themes are generally just text and image files, so they will work unmodified. Your filesystem contains no architecture-specific data, so it will work without needing to be modified.
