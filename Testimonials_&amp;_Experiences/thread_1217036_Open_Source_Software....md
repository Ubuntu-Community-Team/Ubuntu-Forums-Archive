### Post by Crafty Kisses on 2009-07-19
Now I usually don't make threads in this section, and in fact I don't think I have made a thread in this section ever, but anyway to my point. I always see threads about how Open Source Software can be improved and what not, but I'm just not seeing the issue. I've been using Linux since early 2005, and I must say it's came so far in such a short time. Now I'm working on a website for my friend, and I'm going to tell you about a couple of programs I've used and my experiences with them.

I think Bluefish in someways is even better than Dreamweaver, I can get things done so quick in Bluefish when I'm coding in HTML, it's ridiculous, and in Dreamweaver there's something about it that just feels sluggish, I'm not sure what it is, but I know for a fact in Bluefish I can get work done faster.

Then as far as the FTP client goes, it was perfect. Instantly connected no issues and I was able to upload the files I've done in a matter of seconds, it's crazy. I think Open Source Software in someways better than proprietary software.

Now I've also seen a lot of people complaining about iTunes not working in Linux, but that's not our fault is it? People we have alternatives. Use gtkPod, I mean that works with so many iPods, and in fact I had an iPod Touch but I sold it for Cowon S9, which works perfectly with Linux, instantly recognized and I was able to put all my music on it, what a piece of work. 

Now I'm not starting this thread to start some flame between OSS and Proprietary software, I'm just saying I think OSS is highly underrated. If you actually take time to use the software, you would probably know this. I've never been able to do work so fast.

Thanks to everyone that posts on this forum, developers, and the founding father of Ubuntu Mark Shuttleworth. Thank you to the whole Linux community, and personally I don't think there's a better computer collaboration going on in this world than the Linux community. Thanks for reading.
