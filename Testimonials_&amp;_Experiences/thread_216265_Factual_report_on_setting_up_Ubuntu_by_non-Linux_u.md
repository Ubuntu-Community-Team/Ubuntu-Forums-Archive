### Post by guopai on 2006-07-15
I have used Windows for 10+ years and never used other operating system, until I found Ubuntu 2 weeks ago. Now Ubuntu is my default system, along with XP as a dual-boot for IE-only website (internet banking) and scanner, which I couldn't find the way to make it work in Ubuntu yet.

My reasons for switching are many. I am a Wikipedia fan, and was very impressed with the idea of open content/open source/free software. Second reason is for stability and speed, as I realize I need to reformat and reinstall Windows every few months. Third reason - on XP I recently found that I just could't download big file in one piece as the package would currupt very frequently, so I wanted to diagnose whether it's my network, hardware, or Windows' fault. It turns out that Ubuntu never broke my downloaded file, no matter how big it is...

After an hour on Live CD, I decided to install Ubuntu. I use installation wizard to automatically separate half of my HD for Ubuntu, and the whole process was done in half an hour (AthlonXP 2000+, 384mb ram, 80gb HDD, Nvidia Geforce FX5200). System boot up flawlessly. Everything just work out of the box. I was very impressed, unlike XP that I had to install driver for mainboard, sound, graphics, ethernet port, and so on. As mentioned, only problem is scanner, which is in All-in-one Brother DCP-115C.

As Thai is my native language. It took me many hours to setup Thai to work correctly in Ubuntu and Firefox. The rest of the setup are to get multimedia work. I just installed VLC, and later XMMS, and gtkpod for my iPod shuffle. All of those work perfectly. I figure I need win32codecs to play WMV in Firefox, but the repository server seems to be currently down.

Setup process, I admitted to be painful, but I learn a whole lot of Linux system during the google search from countless number of forums and website. It's like learning computer from ground up, but it's fun and worthwhile. I even learn how to compile a program (Lame encoder) and get necessary pre-requisite. Also how permission system work and how to change it. I'm now very happy with command line and the way Linux work. Once you get to know it, it's quite comfortable, as you can really take control of anything, in the most logical way.

So far, no thought about going back to Windows. However, I realize Linux is not for everyone. But at least it's for me!
