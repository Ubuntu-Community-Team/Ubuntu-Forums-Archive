Ever heard of "Screens and Graphics" ? It solved exactly the same problem for me: most games didn't work in fullscreen because refresh rate was set too low, monitor gave an "input not supported" error. It turned out that after installing the nvidia 3d driver, the OS thought didn't detect my monitor make and model anymore and instead loaded the settings for "generic monitor". Solving the problem and selecting the correct refresh rate took a whole 3 mouseclicks, and solved the problem forever. Graphically. And it should also solve the problem for the original poster.

To get to that screen, open a terminal and type:

sudo displayconfig-gtk

Don't worry, that is the first and last terminal command you'll ever have to type in if you use ubuntu for normal things (gaming, internet, watching videos, music, photo's, etc.) :wink:
