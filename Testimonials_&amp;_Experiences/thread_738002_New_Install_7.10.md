I managed to pick up for free, an "OLD" HP 530n w/ 512mb RAM and a 160gb hard drive. I also got a "OLD" dell 19 CRT. Anyhow, I installed Ubuntu 7.10 and I have to say it screams. I'm very, very impressed with the installation. The only drawback was my Lexmark x3350 printer, which is now a paperweight, but I can pick up a cheap HP multi-printer for under $100. 

I hadn't used Linux for about 7 years; had a Slackware box running with X, so you know what that was like to setup. Ubuntu was very easy to install, and runs like a champ. Great documentation site too.
