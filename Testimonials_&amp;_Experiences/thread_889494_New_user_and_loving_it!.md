### Post by 73ckn797 on 2008-08-14
Greetings to all.

I have recently switched over to the Ubuntu system and have been very pleased with the results. I installed to a Toshiba Satellite notebook and to a desktop system. The notebook was a completely clean install as a single boot system. For the work I do Ubuntu has available everything I need. I was up and running within moments after the install. I added several packages that have matched the tools I was using under Winders.

My desktop system is a dual boot with Winders XP. I am rarely accessing the Windows system and will probably wipe the drive soon. Installing as a dual boot had its glitches but I was able to fix those fairly easy after searching this forum. I am learning much and look forward to gaining more knowledge of the Linux OS. I am recommending it to all that I talk to.

There are several Windows programs I would like to have installed and have tried those using Wine. Not all will install but I am sure that it can be worked out. The use of those programs is not enough to make me stay with "those other guys".

The latest Winders service pack and the multitude of updates drove me to the edge in my decision to switch systems. The Ubuntu platform has been very smooth and recovers nicely from my tinkering.

There are a few minor issues regarding reading Nero/InCD disks which will eventually be worked out. I will probably have to re-burn under Ubuntu the disks that have anything of value for use with Ubuntu. That would probably be the easier method to follow.

I ramble.
73ckn797
