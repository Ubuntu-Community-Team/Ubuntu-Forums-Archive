### Post by sicofante on 2006-06-12
I've been trying Linux every six months for some years now. Every time I've done so, I've found the experience frustrating. This last version of Ubuntu is the first that has changed that. Seems like the promise of Linux "for the rest of us" is about to become true. (It's about time, BTW... :-))

I've tried it only for a few hours so I can't be very thorough. These are the things I like most:

- Installation. Yes: text installation is ugly and poor. The idea of a live CD that will install on HD if you like it is great. I know others have done it, but that doesn't make my point any weaker.

- Synaptic. Finally installation of software and updates is what it should be. The user is not required to type cryptic commands in a terminal (I know how to do that, but I simply don't like it.) The system will inform promptly of the available updates and install them for you right away. I haven't had the time to check if the system will update drivers in the same way (it should), but this is anyway a huge step in the right direction.

- Deskbar. Although I think Beagle should be there by default, the Deskbar makes a true difference.

- Look and feel. Human and its variants finally look like some designers have stepped in (at last!!!). For the first time Linux looks professionally designed. While I still see a few usability issues, I would say the improvement is fenomenal.

I can't say much more after playing with Ubuntu for a few hours.These few hours have been enough to dismiss whatever distro I had installed before. As I said, for the first time, this has changed. I still can't use Linux all the time because I need Adobe tools and a few other Windows-only apps to be there for me, but I believe this is finally heading in the right direction, so I guess I'll be around longer this time.

Usually, being used to Windows and Macs made Linux feel like there was something missing. Most Linux fans would say "it's just you're not used to it", but I think most people know the difference between "just not being used to it" and "this thing is not finished", and the latter has been my feeling over the years about desktop Linux. As I've repeated a few times now, this has changed, for me, with the current release of Ubuntu.

Thanks for the great job!

CM

PS: Not everything is great, but I will go into details about the bad things in the proper forums, once I've tried the new distro a little bit deeper.
