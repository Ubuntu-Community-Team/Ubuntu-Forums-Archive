### Post by ukripper on 2008-11-05
Just to thank the developers for such a smooth upgrade without breaking a single thing I've successfully made the upgrade to 8.10 from 8.04.1 on Sony Vaio laptop within 2 hours. Even the key mapping stood there untouched.

Compiz didn't work properly in 8.04 for me but now it is flying on 8.10 out of the box. My GTK, metacity icon themes including shortcuts added to panel are all there as if I am still using the same system as before only wallpaper has changed for me in lookwise; which is indeed a lovely one. 

Since Dapper/Edgy this is the only fully successful upgrade i've ever conducted where nothing has been damaged, kudos to ubuntu developers: sehr schoen..:guitar:

Hope this post will encourage others to think about upgrading as well.
