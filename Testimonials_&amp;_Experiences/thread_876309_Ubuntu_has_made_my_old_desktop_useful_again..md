### Post by Syndacate on 2008-07-31
It's funny that you say that, but that's how I got started with Ubuntu also.

EDIT:
I used RedHat way back in the day with a 300Mhz processor and 10gb hard disk, with 256mb ram on a Toshiba Tectra 8000 series ;).  Way back when.

I always had a place in my heart for linux, but I was always a windows user primarily - until maybe 8 or so months ago.

Now I'm known for my CD books full of linux CD's ;).  I love trying new linux operating systems, and Ubuntu is all Linux has, made mad easy, gotta love it.
/EDIT

I first tried Ubuntu when Feisty just came out, and I was like "ahh, I can't get this to work with my ATI vid card" - then over thanksgiving break last year, this old *** desktop (after I was using a HP pavillion dv9000 laptop as a desktop at school), I put Ubuntu Gutsy on it so it could be a good server, since I figured it couldn't do anything else.

Then everything came to life when I installed Gutsy.  I'm very happy dual booting so I can boot windows for games (WINE just don't cut it for me), and I've used it every day since I installed Ubuntu, I'm not thrilled with Hardy, it has a slight stability problem compared to Gutsy, IMO, but this old tank that kept crashing in windows now runs fine with Ubuntu, even runs faster and cooler.

Though, unfortunately, my 5 year old rig is going bye-bye, as when school starts this year, I'm going to build a new computer.  I can't deny the inevitable, this 5 year old computer's has been used every day for the last 5 years, pieces of the motherboard broke off it, the fans are uncontrollable.  It had a good life, now it's limping around...

It's a 3.0Ghz Hyperthreaded P4 with 1Gb DDR400, an Nvidia Geforce 7600 GS, an SB Live!, a Linksys 10/100 family faster ethernet adapter, all planted on an Abit IC7-G motherboard with 200/120 hard drives and a 750 external.

Not sure what to do with it *sniffle sniffle* - I might leave it here as a server.

Ubuntu did indeed make my old desktop useful again.

Though when I build a new system, yeah Vista is going on it (I know many pple hate vista, but I haven't had problems with it, and word of mouth isn't enough for me) for gaming, but Ubuntu will still be my daily day-to-day operating system.
