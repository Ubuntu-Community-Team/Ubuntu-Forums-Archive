### Post by SubNetMask on 2009-04-01
Don't you guys have an introduce yourself section? anyway...

Well, I finally got a copy of Ubuntu to work on my PC, and I realized, I should have tried harder to get it to install before, its the best OS ever!

I am a hobbyist Programmer, and I used Dev C/C++ on windows vista, (only reason I had vista is because I have a gaming PC, so yeah games...) and Vista was ok for me, since I got a good PC, and I'm not a noob at Computers, I never got a virus, and even when it crashed (BSOD or just Stupid conflicts), I was always able to get it back up and running, Fix whatever driver problems or just OS problems I encountered.

But, it was a real hassle for Programming, half the time it would mess up the compiling, and it was just horrible.

But Linux(Ubuntu) is great for everything, first off I only had to install my Video Card driver, other than that, everything else was already set up. I'm running it from a CD, but just browsing around and Programming are wayy faster than any windows I've ever used.

It has all the tools I could possibly need, and its just amazing.

Most of what I do in programming is memory modification and algorithms, and GCC and GDB couldn't be more useful, in windows I actually had to write programs that do the same thing GDB does.

Now, everytime I boot into windows, I know what I'm missing out, and i can't bare to use it anymore, all I really use it for now is playing games, but as soon as I get another Hard Drive, I can transfer all my data, and take off windows, install Ubuntu, and Just use Wine or VMware to play games, ( I got a good enough PC, it'll work).

So basically, the point is, Ubuntu is now my favourite OS of all time!
