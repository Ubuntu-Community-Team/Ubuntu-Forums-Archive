### Post by jespdj on 2008-08-20
Good luck with Fedora.

I tried it, but didn't really like it. As far as I remember it was hard in Fedora to install codecs for non-free audio and video formats and other proprietary pieces of software, because Fedora has a more strict "it has to be 100% open source" philosophy than Ubuntu.

There isn't a big difference in how Ubuntu and Fedora work, but Ubuntu just seems to be a little bit more user friendly to me. Some things are more mature in Fedora though, such as Pulseaudio (it has been in Fedora longer than in Ubuntu).
