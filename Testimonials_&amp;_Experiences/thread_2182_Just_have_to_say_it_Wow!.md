### Post by HiddenWolf on 2004-10-26
This distro is exactly what I was looking for.

I have been constantly switching between Debian and Gentoo in the past.
I find Gentoo more intuitive, and the community is a lot friendlier, but it takes ages to set up, and if you like to mess around with software, and have to compile everything, that takes way too long in my case.

Debian is great, but it's slow, it's a chaos, and if you ever need anything, you'll have to figure out how to do it, since the 'ohh, that's not a problem, edit a config file, you'll be up and running easy' attitude is very obvious there.

Ubuntu got up easier and faster then I've ever managed with Debian, it has clear plans on how to move on, and it has a community, so I'm very enthousiastic.
