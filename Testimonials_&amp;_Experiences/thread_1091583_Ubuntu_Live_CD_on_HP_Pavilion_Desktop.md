### Post by ufusername on 2009-03-09
FWIW, here is what I had to do to get the Ubuntu live CD to run successfully on a HP Pavilion desktop A1010N.  It's an older but not quite obsolete desktop, so I was trying to determine if it was worthwhile to convert it to dual boot.

Display Problems:
To get Ubuntu Live CD to load on this machine, you need to read this thread [http://ubuntuforums.org/showthread.php?t=660715]("http://ubuntuforums.org/showthread.php?t=660715")

Ethernet Problem:
The guys over at LinuxQuestions.org knew how to get the Ethernet working.  Look for a post entitled *Driver for Realtek RTL8101L Fast Ethernet Controller* in the Hardware Forum.
