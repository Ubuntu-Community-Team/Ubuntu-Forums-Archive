### Post by DiscoNick on 2007-06-02
Hey all, its been a while since I posted here, but again I must thank the creators of Ubuntu for an outstanding job making my life easier.

I've been dreaming of creating a video site for a few years now, but was leery of using the Win tools such as Dreamweaver, GoLive and the other bloat. I finally decided to design the site using Ubuntu Edgy, and toward the end Feisty.

My site is also *hosted* on a dedicated Ubuntu Edgy server. Installing tools couldn't be easier using aptitude. I'm running ISP config + Apache 2, and MySQL 5/PHP 5. All design was done using Gedit, Gimp, Firefox + web dev toolbar and the framework is with Drupal 5.1. I did all of my work over a secure SFTP session built right into Gedit. No uploading here... just press save!

Anyway, thanks again. Feel free to use my testimonial anywhere to promote Ubuntu + Web Development.

And now, in case any are wondering if OSS pays off, the final product of all this effort: [http://shizzville.com](http://shizzville.com)

Thank you
Nick D'Amato
