### Post by N00b-un-2 on 2008-08-26
I've been playing with the idea of switching to Ubuntu for a while now.  I've been a microsoft user since the days of black and green CRT screens (remember playing Oregon Trail?) and I would consider myself to be anything but a novice computer user.  Well, the events that have led me to jump platforms almost entirely (still not completely ready to sever all ties with windows yet, maybe there is still a bit of nostalgia in that old blue screen of death... who knows)
But to begin with I purchased a Toshiba Laptop computer just a few weeks ago which came with the demonic and utterly retarded operating system, Windows Vista premium (premium-grade-bull-cr@p) which had me so frustrated within 20 minutes of using the computer that I formatted the hard drive and proceeded to attempt to roll back the OS to Windows XP pro, only to find that the older OS did not support my SATA hdd.  So after several hours of head scratching and searching the internet (on my fully functional Ubuntu partition I might add...) I finally figured out how to "slipstream" the proper SATA drivers into my copy of WinXP and proceeded to install the OS as normal.  Well, an hour and a half later (as opposed to the 10 or so minutes that Ubuntu took to install) I finally get windows XP up and running, but it's completely crippled by the fact that it DOESN'T HAVE A SINGLE WORKING DRIVER.  So more head scratching and Ubuntu based web surfing... find most of the proprietary drivers I need for my laptop on the Toshiba web site, although most of them were intended for use on Vista, but work allright on XP.
had to hand install every single driver to get rid of all the yellow dots in my hardware profile... After about ten reboots or so I've finally got a bare bones operational copy of XP installed on my computer.  woo hoo.
Now to address the fact that Windows does not come ready with any kind of productivity software (Office for example) or any kind of software for that matter... AND as soon as I boot the computer up it needs to download about three thousand security updates from the evil empire's mainframe which bogs down my internet connection for days.  And I needed to install my security software (antivirus, antispyware, firewall, etc.) so I'm already a couple days into the process of getting windows going on my brand new laptop (while Ubuntu was happily residing on my other partition ready to rock)
So finally I've got winXP up and running, secured and ready to go, although still needing to download a handful of updates virtually every time I boot up when I discover that Windows has removed my ability to boot into Ubuntu... GRRR!  Computer not recognizing my Ubuntu partition at all.  I think I fixed it by modifying my boot.ini file, or maybe i did it through Acronis Suite... don't remember.
So I've got my computer up and running for a couple weeks now, mostly still using windows.  Why change?  I've got my computer set up the way I want it, and it's fast, secure and easy to network with my desktop computer.  (well actually, XP is really difficult to set up a network properly with which is why I run Win2000pro server on my desktop, but that's another story altogether...)  everything is going good for a while until my windows XP installation becomes infected with a particularly stubborn and heinous virus known as Virtumonde. Another variant of the same virus infected my desktop computer about a month ago as well, which is one of the things that led me to install the much more secure albeit antiquated NT based win2K OS.
By now, I had lost all functionality of windows, and every attempt to remove the virus from my computer failed, leaving only one option: format and reinstall.  Meanwhile, my ubuntu partition was functioning perfectly, and I was even able to enlarge the size of my Linux partition and recover virtually all of my files from inoperable windows XP partition without infection or any problems thanks to Ubuntu's rock solid linux shell.
So after my WinXP was KIA by the virtumonde virus (which my expensive antivirus software didn't do a thing to...) I was forced to use Ubuntu as my primary OS for a while and I've discovered that aside from support of some games I play, Ubuntu does EVERYTHING that I need my computer to do and it does so better than windows ever could.
I'm not entirely ready to throw the baby out with the bathwater yet and do away with Windows completely, I've retained a small 10GB partition on my laptop with a barebones Windows XP installation, but from now on Ubuntu is my OS of choice.  I installed Wine on my machine and after figuring out that it works like a windows emulator and windows programs using Wine must be installed from within Ubuntu I was able to get Microsoft Office working.  I'm going to see about my recording software (Adobe Audition) and some games later on (Call Of Duty, Need for Speed...)  If I can get those programs running within Ubuntu, I can't really see any reason why I should continue to run Windows anymore.

I am sold!!!!:guitar: Rock On!
