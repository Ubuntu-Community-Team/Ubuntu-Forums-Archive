### Post by twirp on 2006-04-17
:) My PC works now...
Anywho,
My computer is from 1996, and it was starting to get out of date. (64 MB Ram, 16 MB VideoCard, 10 GB harddrive, Pentium II).
It had Windows 98 on it, until USB Flash drives came out, wireless drivers, and everything that you can't live without now-a-days basically.
And Windows 98 didn't support everything.
So we went out and bought Windows XP for alot of money.
And after installing it, we had wireless internet and could use the USB devices, but it would freeze whenever you tried running more that 2 applications.
So the computer sat and was untouched until, I got a package with Ubuntu CDs in the mail.
On the first try, Ubuntu ran just about as bad as Windows did.
And I was rather upset.
But I decided I wouldn't give up.
I downloaded other flavors of Linux (FC 1, Red Hat, Slackware, etc.) but none of them had what I wanted.
Upon doing some research, I learned that there were other desktop environments such as Xfce, FluxBox, and some others.
I found a live CD for Xfce and loved it because it didn't run as slow as Gnome or KDE.
But I wanted to use Ubuntu because it had what I wanted.
Someone told me that you can compile Xfce to work with Linux, and I was about to try that, but I needed to find a tutorial.
Upon Googleing Ubuntu and Xfce, I saw Xubuntu. :confused: 
I was confused at first because I knew there was Ubuntu, Kubuntu, and edubuntu, but I never heard of Xubuntu.
So I went to the wiki, and saw a way to install it.
I was reluctant at first because another forum said installing a different desktop environment wouldn't help.
But I tried anyways because I was more upset about having a (what used to be) expensive computer as a foot rest.
Upon installation, it ran!
and the mouse inside turned the wheel fast enough to run the applications I needed.
But after joining the forums, and looking through different posts, I found that it could be made faster!
The Ubuntu repositories (spelling?) had everything to turn a slow system into a fast one.
I found a post about OpenBox in GNOME.  One of my friends said that it's not fully Gnome's fault for going slow, the metacity was a main cause (supposivly fixed in 2.14 which I might try later).
So I followed the directions, and woolah it could run gnome.
I then saw something about fbpanel.
So when playing around with running openbox + nautilus + fbpanel.
the computer went even faster, and is useable (with GNOME, Xfce, and some other things).

So thanks Ubuntu and the people at UbuntuForums for helping me getting an old computer to work great.
Because now it can compete with my other computer (512 MB ram, Intel Celeron D, but Windows XP, until I convince my parents to run Ubuntu, or at least a dual-boot).

Now, what has happened to that one-hundred someodd Windows XP CD we bought?  It makes a great frisbee.:p

*No need to sell my soul*
