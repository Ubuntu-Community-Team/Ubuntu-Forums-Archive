### Post by Rick B. on 2011-09-20
My background since 1984 has been with PCs and midrange computers in many different capacities from operator to senior consultant. 

Just for fun I downloaded 11.04 and burned to a CD. I then loaded the live CD which did not load normally as in past releases. This is an issue. Once everything was loaded, I was able to explore the "new" Ubuntu desktop.

My conclusion is that I will stay with 10.10 Maverick indefinitely, which is running perfectly on both my Dell desktop 8400 and IBM T61 laptop. Natty 11.04 does not show any continuity from 10.10 and is so radically different that it is easy to understand why so many users complained about it, not only because it is so different but also because the learning curve for the average user from 10.10 to 11.04 would be formidable. Also, the forums have shown that there are many issues while or after installing 11.04.

It appears that Canonical did not solicit considerable input from the 1.4 million Ubuntu forum community members before making such a radical change in the Ubuntu interface. It's like they decided they were going to try to shove it down our throats without even a whimper from the user community. This sounds a lot like Microsoft's standard behavior. Very sad, very sad indeed!

> **Rick B. said:**

Good luck with that; support runs out mid-way through next year.

> Natty 11.04 does not show any continuity from 10.10 and is so radically different that it is easy to understand why so many users complained about it, not only because it is so different but also because the learning curve for the average user from 10.10 to 11.04 would be formidable.

Really? You absolutely sure about that?

My girlfriend finds her way around Unity without my help (I just had to tell her to push her mouse against the left edge to show the launcher). My girlfriend is no computer whizkid either - she has medically-diagnosed memory problems and needs my help whenever she wants to attach a file to an e-mail.

She doesn't have any trouble with classic Gnome or Unity.

Neither do I, although I'm an intermediate-advanced user.

The reality in computing is that things change, and they change quickly. Linux just changes quicker than Windows.

If you look at Mac OS X compared to Mac OS 9, that was a bigger change. Not only did the UI almost totally change, but the underlying system did too. Mac OS 9 users seemed to cope just fine.

> It appears that Canonical did not solicit considerable input from the 1.4 million Ubuntu forum community members before making such a radical change in the Ubuntu interface. It's like they decided they were going to try to shove it down our throats without even a whimper from the user community. This sounds a lot like Microsoft's standard behavior. Very sad, very sad indeed!

Canonical and Microsoft are both businesses. They don't need permission from the community before acting. The old Gnome 2 user interface was being discontinued anyway from its upstream developers, so all Gnome users will have to have a change at some point in time. At least Unity retains the notifications and indicators from previous Ubuntu releases. Gnome 3's user interface is an even bigger departure.

Canonical did, however, accept community contributions with code.

The practical advice I can give you is NOT to stick with 10.10 forever, because it will go End Of Life fairly soon. Either upgrade to 11.04 and use the Classic Gnome option (which is still a dead-end), or get used to Unity which you might like if you use it for a few days, or install Ubuntu 11.10 and install the Gnome Fallback option which works quite a lot like Gnome 2.

Sticking with an obsolete Ubuntu and not being able to get security updates or new programs is definitely not the best idea for you.
