### Post by Ms_Angel_D on 2009-05-06
I installed it in Virtualbox, but to be honest with you I can't see any reason for me to run out and buy the OS. After using Ubuntu and being able to so completely customize my desktop, not even the so called eye candy of windows 7 makes it look good.

My Current rig came with Vista home premium, I bought it just before I discovered Ubuntu, so I did use Vista for a while. I can say this of Win7 The UAC doesn't seem half as annoying as it first did in vista.

Just to get a feel of what it might be like to run it I decided to full screen it while I played with it, and to be honest I really just missed Ubuntu. I don't like the way I can simply right-click on a file and click run as administrator without even being asked for a password. Also what a pain it was to have to search the web the proper software required to make codecs available to the OS. The New taskbar, is ok but I can do twice is much with my gnome panels. I really see no reason to be excited about this version of windows (or any other version for that matter).

### Post by VoodooLoveDog on 2009-05-06
I installed Win7 RC yesterday. It's not bad. I think both installations (Ubuntu and Windows7) are about the same in terms of complexity. On the same HW, Ubuntu is faster using ext3 and much much faster using ext4. 

Looks as if Windows 7 needs quite a bit of memory too. I loaded both on a machine with 1GB of RAM. Aero is quite sluggish, where as, Compiz does quite well even with lots of eye candy. I personally think part of the problem is the immature drivers for Win7. 

Windows 7 also didn't recognize the sound and I had to install a WinXP 64-bit driver for it. It worked but wasn't great. 

Just like every other Win OS, it's best to wait 'til they've released a service pack for it.

I like the interface on Ubuntu better than Windows 7. I was never a fan of Aero, but that's subjective.
