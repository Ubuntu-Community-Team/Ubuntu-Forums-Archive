### Post by brim4brim on 2006-09-03
So having installed OSes (win & Lin) before I had experience in this area.  I've also installed previous versions of Ubuntu but dapper is by far the most advanced I've seen any Linux distribution and is ready for general users in my opinion and I honestly don't see how hardware manufacturers have a leg to stand on, they should be providing this OS with drivers in .deb format.

I showed Ubuntu to my brother who has tried Linux in the past and came to the conclusion that it was useless.  He was using Ubuntu for about 10 minutes before he decided he found it better than OSX.  Personally I can say I find it easier to use than Windows (when everything is going right* (when things go wrong they are generally hardware driver related **shakes fist**)).

Installation
Right so I have a partition setup for Ubuntu (as I duel boot) so I put in dapper (that was shipped to me free) and it boots the live CD.  I look around for a few seconds before I find the install icon **oops** then I double click it.  The hardest thing in the installation was partitioning, other than that it is as simple as being able to pick where you live on a map (because that's pretty much the only other thing you need to do during this install).  Partitioning is the same as it always has been in Windows and Linux, stressful.  Basically Ubuntu is easy to install if you don't want to duel boot but when you do, you have to partition up correctly so I deleted the last version of ubuntu that I installed to try out and just told it to use the largest contiguous free space or something similar (it was this easy for me because I've gone through the pain before :biggrin: ).  So it reboots and Linux is there.

Initial Impressions
Everything is where I want it to be and everything is named intelligently.  It's so easy even a child could do it (although with computers this should probably be so easy even an adult could do it).  Synaptic and everything works, hardware all setup correctly.  Most of the applications I want are there out of the box (Office, email, web browser, some simple games, chat client etc..)

The Problems ](*,) 
So after a while, I feel like it's a little sluggish so I check out my graphics card driver and it's mesa or similar.  In other words, ATI's drivers have not worked properly off a fresh install.  I have had this problem in every Linux distro and it is ATI's fault.  Basically I have a laptop X600 mobility card which they don't like to admit they built and only recently supported in Linux.  I find an Ubuntu Dapper howto and follow it and it works =D> .  However the problem is I had to use the terminal.  If I wasn't used to this, talk about the stress that would have caused me.  Anyway basically ATI, get your act together, there are average, non-technical people using Ubuntu (and that want to) and you can't to half-arsed things like this anymore.  Also I don't want to go through this, I just want it to work.

/ignore paragraph if not Irish
Next hardware problem is one I wouldn't have if Eircom weren't crippling Irish communication infrastructure but I won't get into that here.

My modem doesn't work out of the box because it's a win modem and conexant couldn't be bothered releasing a Linux driver (same as ATI, get your bloody act together you incompetent fools, swear to god I'm checking hardware works in Linux before I buy my next computer).  I look around and find linuxant.com (who developed drivers for Conexant modems even though they didn't need to :confused: as to why Conexant aren't doing this).  Anyway I run a scanmodem tool from the terminal (and later find out there is a GUI tool for windows) and I read the files it generates and find the correct driver from the website (which is in .deb format **happy** and install it).  

Now for the problem.  Conexant are Muppet's (and by Muppet's).  Due to the fact that this is a win modem and they have proprietary software drivers (patents blah blah blah), drivers can't just be released for Linux.  This means Linuxant have to charge $20 (an admittedly small fee but I'm a student) for the drivers to get full usage (as opposed to the 1.5KB/S I get currently) as they have development costs that they incur making the drivers work.  I still think Conexant/Dell owe me $20 :wink: .  One thing that deserves mentioning here is the .deb I load (thanks to Ubuntu) can be double clicked to install the driver.  I just grant it permission to install and enter my administrator password (for security reasons obviously).

Everything else I use worked out of the box (even my Microsoft mouse :-\" ).  I've not checked my infared port but I don't actually use it.  I know my wireless works flawlessly as I've used it on another persons connection.  So far these are the only two things I've had any issue with.

Conclusion (at last)
Ubuntu Dapper is pretty much the perfect OS in every respect but hardware support.  The one area they can't perfect on their own.  There are hardware manufacturers making drivers for Linux and I recommend that people check out the web for Linux compatible laptops/desktops before buying so that they don't run into hardware problems and so hardware manufacturers making Linux drivers are supported for their efforts :!: .

My final conclusion is that I've had success and am ultimately happy with Ubuntu Dapper release and recommend it to anyone.  In fact, I'm so impressed that when Windows Vista releases, I'll be using Ubuntu and not upgrading.  I'll also be re-editing my grub list so that Ubuntu is on top.  I will then be buying an Xbox 360 because I'm a gamer to play Windows games (thank you Ms for releasing a console so that I don't have to use your OS =D> ).  If you've gotten this far, thank for reading my wall of text.
