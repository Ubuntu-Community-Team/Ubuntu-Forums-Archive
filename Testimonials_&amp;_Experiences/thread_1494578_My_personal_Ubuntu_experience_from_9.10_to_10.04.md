### Post by ALF102 on 2010-05-27
Hi there, I've been using Ubuntu for almost a year now and I want to  post my personal experience with Ubuntu after almost a year of using it.
                                                    [B]How the idea began
[/B]       It all started a year ago, at that  time I was still using my Windows XP as my main OS. No, it was not a  genuine Windows XP, I've realised mine was a pirated one long time ago.  Yeah, piracy in my country is too common. Tthats because most of the  people here can't afford to buy a desktop or laptop with a genuine  Windows, its too expensive. Now then, back to my main point, back then I  was playing around with some hax that can deceive the Microsoft from  noticing whether my Windows is pirated or not. After using it, I was  able to fully update my Windows XP SP2 to SP3. After it restarted it  self, it crashed and end up in a loop. I was in total shock after that,  all my files, my precious settings, my valueble information lost in a  single night.
        
        A week later, I send the laptop for  reset and it costed me RM50 (Ringgit Malaysia) to get my laptop up and  running again. It was a horrible experience, because of that event I  swear to myself that I will always look for Open Source from now on  rather than taking the initiative of using hax or piracy. At that time I  am not aware of the existence Ubuntu or any other distro out there. But  I do know the existence of Linux, its just I don't know it was free. I  only know there is two OS out there namely Windows and Mac.

         One day I was lying down and try to think how to make sure that  particular event won't happen again. After lotsa thinking I got one word  hovering above my head which is obviously "Linux". But I still don't  know Linux is free, so I went googling some stuff and to my suprise I  found out Linux is free! So I went to Wiki to find out more about about  Linux and that's how I stumble upon the Word "Ubuntu". On that night I  started downloading the .iso file of Ubuntu 9.10 Karmic Koala. At that  particular time, I also print out the Ubuntu 9.10 technical guide from  somewhere I found at Ubuntu officials.

[CENTER][B]The  Installation Process

[/B][LEFT]        Two days  after that, the .iso file is finally finished downloaded. At that time, I  try to avoid installing using a LiveCD but that is a mistake that I  will never forget. Since I tryoto avoid using LiveCD, I use Wubi  instead. Of course the installation was rather smooth and easy, but  that's not where the problem started. After using Ubuntu 3-4 times, GRUB  fails me and of course I was a total beginner in this matter. Not  knowing how to solve my problem I simply reinstall Ubuntu still using  Wubi. But the same thing happen again. Then it just occur to me that I  really need to use a LiveCD.

        After spending a gruesome  hours I finally was able to get Ubuntu running smoothly with no  interruption or bugs, something like "out of the box" thing. I was  really awed by the desktop environent, the function and all the stuff in  Ubuntu. It was really satisfying because I made it work by my own  power. 

[CENTER][B]Aftermath

[/B][LEFT]        After a two days of my Ubuntu 9.10 installation, I  was still captivated by the working environment. It was a complete  different experience than using Windows. I was able to change or  modified almost anything. Even the colour of the line for the graph in  the System Monitor. But it as I was about to try to listen to some  songs, I was a bit surprise that I need to install the Restricted Extras  from the Synaptic Package Manager. But when I think back about stuff  like licensing for windows and etc, I noticed why it was called *Restricted*  Package in the first place. One of best thing is that all of the  software included are Open Source which is what I'm aiming for. Not bad  having Firefox and GIMP as a start. Afte that, more sofware comes in to  suit my needs.

[CENTER][B]Lucid Lynx is knocking on  the door

[/B][LEFT]        For several months I've  been using Ubuntu 9.10 Karmic Koala, I receive news about the upcoming  Ubuntu 10.04 Lucid Lynx.  But I don't intend to upgrade my version of  Ubuntu because I'm playing it safe. I somehow got a hunch that if I  upgrade my Ubuntu I will face a similar problem like the one I first  install my Ubuntu, so I just wait for week. It turns out my instinct was  correct as the new version of Ubuntu has a serious bug in GRUB, I was  really lucky for not upgrading it yet. So I wait a few days later to  allow the problem to be solve.

[CENTER][B]Upgrading

[/B][LEFT]Now after waiting few days, I finally able  to upgrade my Ubuntu to the latest version. It wasn't really that long  and its rather much easier than installing from a LiveCD. After  rebooting, I'm able to access Ubuntu 10.04 Lucid Lynx and again Ubuntu  never cease to make me feel awed. At that time my Ubuntu experience has  been taken to a whole new level. Also to my suprise, the upgrades works  perfectly fine. No problem or whatsoever. I am really impressed.

[CENTER][B]Conclusions

[/B][LEFT]        So  yeah, it was a really good experience to me after using Ubuntu, I learn a  lot of new things like searching for alternatives open source software  rather than using the paid ones. I have to say it wasn't  really a  smooth experience as I've encountered some problems here and there but  that makes it a whole lot worth it. So here's my say to Ubuntu, you guys  are amazing.
