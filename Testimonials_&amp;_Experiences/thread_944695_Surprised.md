### Post by esters on 2008-10-11
After doing some distro hopping ( FreeBSD 7.0, Frugalware 0.9 ) today I decided to give Ubuntu 8.04 a try.

Actually I used Ubuntu since 5.04, but after the release of 6.06 I switched to Debian, because I wanted to go deeper in the distro world.

Primarily I use Arch Linux with GNOME but I don't notice any difference between my "customized" Arch setup and default Ubuntu in terms of speed and memory consumption, the mayor difference is that in Arch I would need to spend more time than on Ubuntu to get +/- the same.

I've decided to go the no-terminal way of doing things, just to see how "user-friendly" it really is.

As for the bad side - After the restricted drivers manager installed nvidia drivers and I upgraded my system, restarted, Desktop effects were enabled and that was killing my card ( nVidia GeForce 6150 ), because I didn't share enough memory and the whole screen drawing process took minutes, even if I tried to switch between tabs in Appearance it couldn't. So I rebooted, shared maximum amount of RAM that I'm allowed to, booted into Ubuntu and disabled those effects.

Second thing - Just for testing, I tried to watch videos @ youtube, I know that flash isn't installed by default, but there was no from Firefox about missing plugins. The only solution was to click on the link to download a compressed package of plugin from adobe, which would raise another question - What can I do with it ?

Looking forward to see Interpid :)
