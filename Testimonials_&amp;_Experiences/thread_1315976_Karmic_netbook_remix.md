### Post by emastro on 2009-11-05
I must say I'm impressed with Karmic, I've used Jaunty netbook remix on my Compaq/HP mini 700 for some time now and I was terribly annoyed by the fact that at each kernel upgrade I lost sound and had to manually rebuild and reinstall the Alsa drivers.

I decided to wipe the netbook and try Kubuntu netbook remix first (I use Kubuntu on my desktop). Big mistake. Networking didn't work, lspci saw the wireless interface but I spent a day trying to make it work and finally had to give up.

For some reason the Ubuntu installer crashed on boot - I put it on three different USB sticks and never got further than the bootloader shell; so I went back to the old 9.04 installer, did a wipe and full install and then upgraded to 9.10. Magic! Even sound works!

The only really irritating "feature" is that now clicking the home screen top-left icon makes it forget the previous status and return to displaying the favorites, which isn't necessarily where one wants to go back - and if there's a way to alter this behaviour, I haven't been able to find it. But it's really a minor annoyance, and the rest seems to work just fine
