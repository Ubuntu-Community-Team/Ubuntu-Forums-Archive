### Post by Rotarychainsaw on 2006-02-16
So, I've been using Ubuntu since before Warty was official. I've done re-installs of each release just for the heck of it. The thing is though... I always used my 1ghz laptop for linux. I was pleased, Ubuntu was very usable. 

Then the urge to get XGL started taking over me. The laptops puny geforce2 couldn't handle it at the time (although I heard they fixed that already). So I made some room on my good computer.

My original plan was to install linux on a partition far away from windows. Like windows is on IDE and this was SATA. I was going to switch in bios where to boot first and not use grub at all. That didn't work though.

Installer was nice. I like the VGA option that permanently sets the framebuffer for the console. Somehow managed to get the partitions set up right. All the automatic options wanted to erase a whole drive. Everything got installed and I was amazed.

Ubuntu runs like a champ! I always liked it, but on a computer with 2x the ram, 2x the cpu power, and a hard drive that spins at a normal speed it is blazing. Compared to that laptop its like night and day. I am now seriously thinking about using Ubuntu as the main OS and keeping XP for Civ4 lol.

The moral of the story. Put ubuntu on a fast computer! I think it may be like OSX in a way. The more hardware you throw at it, the happier it is.

PS- XGL is running beautifully.
