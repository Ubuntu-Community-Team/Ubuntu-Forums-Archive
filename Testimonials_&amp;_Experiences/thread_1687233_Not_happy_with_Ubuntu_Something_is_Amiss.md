### Post by false truths on 2011-02-13
Before getting into other distributions, I feel that someone should ask if you have tried going with a complete clean install of Ubuntu. With every upgrade to a new version, a lot of code gets left behind to build up over time until eventually you have a mass of code sort of resemblant of the Windows registry just sitting up there, taking up space, and bogging down the entire system. I highly suggest you try backing up all your files (possibly your entire /home folder), wiping your partition clean and starting with a fresh install that's free of the buildup of upgrading versions.

If you're really interested in looking into other distributions instead, though, Arch Linux and Debian would probably be the best places to start.
