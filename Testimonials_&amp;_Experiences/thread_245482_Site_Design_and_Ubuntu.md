### Post by DiscoNick on 2006-08-27
I decided that I'd design my latest site using Ubuntu instead of Dreamweaver and Windows. The idea was to put this distro to the test, and see how fluently I could put it all together.

Well, a few weeks later the first beta is available. I gotta say, Unbuntu rocks! My workflow is much quicker using the desktop capabilities and applications of Linux (multi-desktop is awesome).

If you're curious, here's the site: [http://www.imagegrotto.com](http://www.imagegrotto.com)

It is a free image hosting site that will accept almost any image format, and convert it to JPG. You can even upload a ZIP archive of images, and it will create a slide show. All done using open source! (It is even hosted on Linux and uses ImageMagick as its core).

### Post by DiscoNick on 2006-09-03
