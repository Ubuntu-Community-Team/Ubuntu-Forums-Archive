### Post by arunb on 2010-10-24
Hi,

I have been using Ubuntu 10.04 soon after it was released on my old Toshiba Tecra laptop. The installation (no dual booting) was such a great success that I started to use the system for my office work, which is mostly software development.

Since some of my applications run on Windows I had to install Windows XP using VirtualBox. I find this to be a better choice.

I just bought a Acer Aspire 5745  that came pre-installed with Linpus Moblin. I had the choice of buying the Aspire 5745G version but I decided to get the linux version instead as I want to use Ubuntu as the primary operating system.

I couldn't get a backup of the Linpus Moblin OS, I suppose I can always download it from the internet.

I installed Ubuntu 10.04 using a bootable USB pendrive. 

There was some problem about the broadcom wireless driver, but I managed to sort this out fairly quickly, apart from this there has been no other problem.

Now everything works excellently except for a minor issue, the network manager applet icon shows the red exclamation mark even when connected to a wireless network. (I think this too has been fixed in the 10.10 release).

Apart from this minor issue there has been no other problem.

Ubuntu has definitely come a long way since 7.10. I find that the support for hardware is much better, the system has become more user-friendly with the usage of user-interfaces rather than command line.

Over these many months Ubuntu has given me great pleasure and I have found the experience to be extremely enjoyable. 

Thank you very much for making this happen.

ar

Agreed.  Dual booting introduces its own set of problems.  Also from my experience it seems that WinXP actually boots faster and runs better as a guest OS in a VM than it does when booting from its own partition.  Plus, you can keep the VM running and simply toggle back and forth between Linux and Windows.  The main thing is that you have enough RAM to support the both the host OS and the guest.
