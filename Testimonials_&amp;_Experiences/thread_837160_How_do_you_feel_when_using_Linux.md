### Post by wdaniels on 2008-06-22
> **Delenir said:**
> I feel like I'm missing out or that I'm a second-class citizen in the computer world. Things that everyone's enjoying in the Windows world may take years to reach Linux

I think it just depends on how sensitive you are to the latest "fashions" (for want of a better word). There's nothing technologically interesting that exists for Windows but not Linux, and probably a number of things the opposite. Being more interested personally in the software technology itself than the latest DRM-crippled media content, or drivers for the latest gadget etc. it doesn't bother me to be "missing out" on all that stuff. It's a bit like when I first changed to 64bit and found there was no (proper) flash support. "How can I use the web without flash?" I wondered, because flash is everywhere! I soon discovered that all this really meant was no advertisements :D The odd little thing that needed flash (like YouTube) I just lived without and was no less happy (and probably more productive) for it.

But that's not to say it's wrong to want to have all those sorts of things, or to be able to shop for gadgets without spending hours trying to discover which chipset it uses and if it has a decent Linux driver etc. As somebody else said, Linux is not for everyone. But when I first started using Linux full-time, I always used to notice the things that were easier on Windows and developed a somewhat irrational idea that life with Windows is always easier. Now when I have to use Windows for some reason I notice that this is not necessarily the case. Everything doesn't "just work" automatically because it's Windows and that's what it's designed for. I'm not sure how I came to forget about that.

So now I tend to notice all the things that are so much simpler and easier with Linux. I used to think that it was a pain to have to remember (or search for) the names of packages for things, but now when I use Windows I realise that I still have to remember the name of the program, find it's website, download it, click all the way through the installer, delete all the crap it puts on the desktop, the quick launch thing, the startup program that only checks for updates and advertises at you, the stuff it shoved secretly into the Run key in the registry, the IE toolbar it installed...thankfully this nonsense can never happen to Linux :D I don't have to defragment my disk every other week, I don't have to run a virus scanner, I'm not tempted to download half a zillion different emoticon packs because they probably wouldn't work anyway, and so on...

So, no, I don't feel like I'm missing out any longer, I feel like I'm in some secret, special world, where everything's in it's place and under control :D
