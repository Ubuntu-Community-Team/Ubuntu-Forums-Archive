### Post by thenes on 2008-03-31
The tread "Bye bye Linux :wink:" inspired me to write an opposite testimonial.

Since 2005, I have been playing a little bit with terminal in os x for a couple of years, and also always trying the latest versions of Ubuntu on live CD.

Last week I made the move to install Ubuntu on my Intel Mac Mini, dual booting with os x. Everything has gone all right. Ubuntu runs smoothly on my mac. The only problem I have had was with the hibernation mode, but it did not take me long to find out that I should use suspend in stead of hibernation, as that puts my mac to sleep in a gentle manner (not causing it to freeze like hibernation does).

I am even able to connect to the VPN-server of my University using Ubuntu. This does not work in os x 10.5 (though it did work in os x 10.4).

I am sorry that the one who wrote the other tread had so different experiences with his move to linux. For me, the move has been very pleasant.

### Post by armandh on 2008-03-31
the experiences are how one finds them.
I was lucky for my first Ubuntu, it all just worked.
admittedly it was all fairly standard hardware.
so why do some have such difficulty?

bad down load and/or bad burn
simply,  if your hardware is not stable enough to get a good CD,
then don't expect a good install.

couldn't get that video card/wifi/sound/printer to run with the live disk?
don't expect the install to be better. there may or may not be CL fixes.
some manufacturers are not yet, or not fully supporting linux
some are no longer in business. some bought out.
they are not interested in your use of old equipment 
they have a new one to sell to you.

sorry: ubuntu will not make a P-I run like a P4
please read the requirements
I find that a 800 mhz P-III with 256 meg mem to be the minimum for me

lastly don't blame ubuntu for software not compiled for linux
some may never be native linux. if it is a must have ... dual boot.

I find Ubuntu superb for desktops  
I no longer have to purchase an OS and AV just to give away a computer.
