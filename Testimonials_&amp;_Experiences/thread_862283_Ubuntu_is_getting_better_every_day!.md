### Post by dlublink on 2008-07-17
Hello,

I purchased a used HP evo n6000 laptop yesterday. I inserted the Ubuntu 8.04 CD and installed Ubuntu.

20 minutes later, the system was up and running fully functional. I did not need to configure *anything*. At one point, Ubuntu tried to install a prioprietary driver, that did not work, one boot the screen was jumpy at next boot the driver was off and it was working fine.

Congratulations to all those who have contributed to Ubuntu and the wider Linux community, great progress has been made and Ubuntu is becoming a really great operating system. In this case, it was far easier installing Ubuntu on the machine (20 minutes) than it was to install Windows XP ( 2 hours ).

I am currently recommending Ubuntu to anyone who will hear me out and am distributing Ubuntu CDs to anyone who might install it.

Keep up the good work!

David
