### Post by ventrical on 2011-05-01
I just wanted to make a quick note about the sleekness of Ubuntu overall. I had discovered that having an extra user account before an online upgrade from ubuntu 10.10 to 11.04 is a real good option just in case somthing goes wrong with your install. 

  I had an extra account and it enabled me to reconfigure and eliminate compiz settings in the original 10.10 install. I was able to get into Natty using the (generic recovery) option in the GRUB menu. So, not only do I have my original account (set to Ubuntu Classic) but I can switch between user where the seocndary user will employ Unity in a seamless fashion.

I also discovered that the compiz settings for Unity are pretty well set up to begin with and it can do some pretty neat tricks with the right hotkeys and up and along the sidebar.

Great work Canonical!:)

So there is all the wizardy of Ubuntu 10.10 with the novelty of Unity in 11.04!! at the flick of a switch.
