### Post by RobsterUK on 2006-09-13
I've been using Ubuntu for just over a week, and I love it. People say that Linux is all about the command line, but I've only had to use it a few times, mostly getting my windows drives mounted.

The best thing is that when I burn a CD I can do other things at the same time. It sounds stupid when I think about it, but on Windows the system pretty much ground to a halt and there certainly wasn't any hope of listening to mp3s. Today I burned a CD using Gnomebaker and I actually forgot it was running until the CD ejected.

I haven't missed windows at all, I thought I would due to being out of my comfort zone. It actually feels like a real drag when I have to boot into it to delete some files to make more space for my Ubuntu parition.
