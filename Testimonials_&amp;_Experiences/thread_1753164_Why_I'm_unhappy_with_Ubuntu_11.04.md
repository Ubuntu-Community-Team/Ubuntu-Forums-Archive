### Post by k999 on 2011-05-08
I upgraded to Ubuntu 11.04, and I'm pretty unhappy. Is there a downgrade option? Here's a couple of the reasons I don't like it:

1. Opening terminals on multiple desktops is a pain. I have to find the one I have, click Ctrl-Shift-n and move the new terminal to where I want it.

2. It's unstable (probably because it's new). My battery ran out, and my laptop suspended. When I started it up again, the screen was all screwy, so I had to reboot. When I was done rebooting, the screen resolution had changed to something useless. Which brings me to the next issue...

3. It's difficult to find stuff. Especially the first time, but even after that things are in impractical places. I've eventually found the link to "Places", which requires that I unfocus all other windows that are open, which is a pain. I finally found my monitor resolution settings by searching for "display" ("screen" and "desktop" didn't help). Makes me think the terminal window is more useful than ever. (Btw, I changed the resolution, and that made my desktop all buggy... Has this been tested much? Doesn't seem like it.)

4. It's slow. When I'm running a few applications, the feel of the desktop is much more unresponsive than 10.10 was. Changing desktops (which I do a lot) takes a lot longer than I'm used to.

I've only used it for a few days, but I'm already pretty dissatisfied. Given that this was an update with a major change like this, I think it was a really bad idea to change the upgrade path to make it more intrusive than it's usually been. Ubuntu should have waited until it was more tested and stable until they made the upgrade question a popup!

I'll look into ways to downgrade or change the desktop back to what I'm more used to.
