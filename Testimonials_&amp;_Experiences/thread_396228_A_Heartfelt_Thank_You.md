### Post by AlanRogers on 2007-03-29
Having read a number of threads recently stating that Ubuntu is not all it's cracked up to be, I thought I'd offer my recent experience which may be of some use to people, particularly those new to anything other than Windows. It'll be a long post, so you may wish to get yourself a drink before sitting down to read it.

Like so many others, I've been a Windows user for ever. Sure, I'd heard of Linux and Unix but never used them, for a variety of reasons, the apparent elitest/snobbish nature of those users I did know being in the top three. I'd also seen Macs but, short of trouble-shooting internet connectivity on them, I couldn't be bothered to learn the differences; Windows did all I needed it to do and I understood it.

Then along came the hype of Windows Vista, Microsoft's first new operating system in more than five years, together with [the annoucement]("http://support.microsoft.com/gp/lifesupsps#Windows") that support for all bar XP SP2 and 2003 Server would be stopped. Being a cynical old git, I read this to be Microsoft forcefully creating a market for Vista. I'm sure others could argue this point but, in my head, the facts stack up and I'm quite disappointed in Microsoft for doing it this way.

So, not having a machine that was capable of running Vista and not willing to shell out for one, plus having a number of friends and customers in the same boat, I needed to explore alternatives. I still wasn't prepared to invest in a Mac, so Linux was the logical choice, using old computer equipment that I had sitting around. To research an alternative, learn new skills and not invest too much (actually any) cash in doing so was a very attractive proposition.

I asked around, seeking the opinion of those I knew or suspected to have had some experience with Linux. The consensus was to try Mandriva, Fedora and/or Ubunta. So I downloaded and burned the ISOs but also read their web sites. My first choice was Ubuntu, because this forum appeared to be dynamic, personable and helpful. Now, three months in, I still haven't tried Fedora or Mandriva, though I have installed Ubuntu, Xubuntu and Kubuntu several times on different machines, learning more each time.

My journey into using Linux has not been rosey; I've given up trying to get wireless adapters to work, I've completely destroyed the MBR on a double-drive install and rendered Gigs of data inaccessible (since recovered using Linux tools), I've completely fouled importing my contacts and calendar into Evolution, and the list goes on. some challenges have been conquered, others have been worked around and still more remain a mystery.

Alongside this forum, the single most useful resource I've found to date is [Ayisu's web site]("http://www.psychocats.net/ubuntu/") which contains detailed HowTos on a variety of issues that beginners will almost certainly have to face. It seems only right to advertise his efforts here, so that others may visit and perhaps avoid being referred there somewhat more tersely. If you're new and haven't read his pages, which appear to be regularly revisited and revised, you're making life unneccessarily difficult for yourself.

However, I've learned tons and continue to learn, from this forum, web searches, investing in books (now there's a old concept) and generally trying it out. The upshot is that I have a machine which is totally Ubuntu and is now my principle machine. Sure, I still use Windows, on another machine, but I run a company supporting Microsoft products, so I need to be able to replicate errors, and some programs have yet to have a functionally comparable equivalent which runs under Linux, nor will they run well under Wine. I haven't tried Cedega yet.

So, to those who would post here to detract from Linux, particularly Ubuntu and its derivatives, I understand your frustrations and accept your reasoning, where offered. However, for me, Linux is a viable alternative for what I want it to do, though not necessarily for all that I want to do. This is a simple but powerful distinction. To those who post in reply, suggesting that the detractors get involved more, I agree the principle but it isn't always immediately apparent how and many of those experiencing the problems, like myself, wouldn't perhaps have the necessary skillset to assist. I could well be wrong; please someone respond on this point.

Using a dangerous religious analogy, I understand and accept that there alternatives out there which are equally viable though not necessarily directly comparable to what I currently believe to be acceptable or normal. I have not had some born-again epiphany, nor will I suddenly become a blinkered zealot; I'm not enlightened, merely better informed.

GNU/Linux is just an operating system, and a bloody good one at that. Ubuntu as a flavour works well for me and has been relatively easy to pick up as a transition from Windows but it's (just) a(nother) tool in my box. Sometimes, I need Windows, so I'll use it. The rest of the time, I'll use Ubuntu and learn more, in an effort to give back to these fora more than I've yet taken. 

In closing, I reiterate my thanks to those that maintain HowTos, be they here on the Ubuntu sites or on their own, and those that have (and will yet) take the time to answer my questions.
