### Post by Lacrimstein on 2007-10-19
This summer, I got really really bored, and I decided to install Linux and see what it's like. I have read some things about UNIX and from that I judged it (and its derivatives) to be a cool system. So I googled "free linux" and was taken to the Ubuntu website. I downloaded the iso, burned it, and booted from it. It was kind of sluggish (because I have only 256mb ram), but not that bad. So I installed it. There, I was pleasantly surprised: Only several dialogs showed up, asking straightforward questions, and after that the installation started and I went to eat (Unlike in Windows setup, where it asks for user input in random locations throughout the process). The installation took less than an hour. I was very pleased with the system, especially how customizable it was. I, as any noob, had several problems, especially my printer, but all of them were solved on the same day that I posted it on this forum (Ubuntu community is GREAT!!!). Every day I learned something new about the system, and I loved it even more. About a week ago I was forced to boot the Windows partition again - my dad asked me to write a windows-specific program for him. And it was a mess! It took 3 minutes to start up, 5 more to become usable, and it was so slow and just... unattractive. Thats when I decided I will never use windows on my home computer again. Finally, yesterday I downloaded and installed Gutsy - I personally expected it to be sluggish on my crappy hardware, but it ran faster than Feisty, even with maximum desktop effects turned on!!! So I want to thank the developers of this wonderful OS for making it, the wonderful community, which I am proud to be a part of, for helping me through my "noob times" and taking me in.
