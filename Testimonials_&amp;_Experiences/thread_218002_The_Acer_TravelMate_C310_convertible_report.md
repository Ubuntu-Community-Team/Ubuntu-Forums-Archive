### Post by prizrak on 2006-07-18
I just got a brand new TravelMate C310, don't remember the extended model number but it's the one with 512MB RAM. As soon as I got it Dapper was loaded. (Note: I'm a seasoned user and had Ubuntu since the first release). The installation was smooth and painless the setup was a bit rocky tho. 

THE TABLET:
The tablet didn't work out of the box I had to semi follow a how-to on the forum (not linking to it because it only gives direction but the setup required alot of thought). The problem was that wacom-tools were not installed by default and after they were the xorg.conf had to be modified otherwise the internal mouse as well as the tablet would not click. After that was done however it works like a charm aside from having right click capability and the pointer not being exactly where the digitizer is (Windows tends to have the same issue so it's not a Linux problem). 

BLUETOOTH:
This particular laptop has on board bluetooth however GNOME lacks any tools to actually test whether it is operational. The bluetooth status light is not on so it can be assumed it is not working. 

SD CARD READER:
To my surprise worked out of the box.

WIRELESS:
Same as above with Network-Manager, default network tool works as well but not with WPA networks. 

InfraRed:
Not tested due to lack of devices that use it.

Touchpad:
100% operational OOTB.

Video:
Had issues with the 'nv' driver but works well with the proprietary nVidia driver. 

HIBERNATE:
100% Operational OOTB.

SUSPEND: 
100% Operational OOTB.

CPU scaling works very well.

NOTE: The i686 kernel does NOT work with this CPU despite it being a Pentium M the system was slowed down to a crawl. 

FUNCTIONS:
The volume control buttons do not work and none of the special extra buttons are recognized by the OS (attempted to map shortcuts to them). 

Overall:
It is a very nice machine and does support Ubuntu pretty well. The only real nuisance is not being able to use the volume control. They should have a hardware control for it anyway.
