### Post by Idefix82 on 2008-11-08
In my opinion one of the most powerful applications under Ubuntu is gedit. You could show them a couple of plugins (for example with the external tools plugin you can do all sorts of fun stuff); show them how many languages it supports for syntax highlighting and tell them that it's easy to create your own language highlighting rules; show them how you can connect to an ssh or ftp server via the main menu, then access the remote folder directly in gedit thus allowing you to, say, edit your homepage directly on the server; and so on.

The other cool thing is the numebr of useful command line applications. For example, for me to copy all images into a different folder and then resize the copies to get thumbnails for my home page is one line. To upload both the original folder and the thumbnail folder onto my web space is another line. To send an email to somebody with a couple of attachments is also one line. I think to do all of this would take me less than a minute, while resizing 20 pictures with a graphics software can be a real pain.
