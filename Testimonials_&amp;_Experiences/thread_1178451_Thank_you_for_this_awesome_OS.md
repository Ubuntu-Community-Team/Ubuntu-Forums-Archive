### Post by nmehta6 on 2009-06-10
A little update...

More I find out about what tools are available in the Linux world more I am amazed what I missed out on.

I have been a developer writing C#.NET and ASP.NET applications using Visual Studio of course.

I found out about MonoDevelop and Mono framework. I can tell everyone that from 5 years of using Visual Studio this IDE and framework are really nice. There is such nice integration with testing tools like NUnit and web servers like Apache. I am deploying my Web sites and web services to Apache. It is so simple and intuitive.

I am a believer in 'What works'. I don't need to bash Microsoft for anything and it does work for some people but it feels like 'More of the same'.

I showed Ubuntu to my dad, mom and sister (novice users) and they switched right away because it is easy to learn:).
