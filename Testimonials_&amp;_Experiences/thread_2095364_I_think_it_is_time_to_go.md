### Post by gertcher on 2012-12-16
I am (was) very keen to get on with Ubuntu. I have not been in favour of M$ for a few years, and tried various flavours of Linux. The current one is Ubuntu 12.10. Overall I am impressed with Ubuntu, parts of it work quicker and better than Windows, but bits of it fail to perform professionally, such as:
Mouse pad control is very difficult
Failure to save a file on a network computer as a MS.doc file. This is a Libra failure.
Can only get my 'N' capable network laptop to receive a max of 56 Mb/s in stead of 120 Mb/s since a recent system update.
The lack of capitalisation in Thunderbird (I know it's not Ubuntu's fault).

There are several features of Ubuntu I don't like, I know it is possible to change them. I don't have the time to configure Ubuntu to perform they way I want it. I am a supporter of Ubuntu not an enthusiast. The point is, whether or not it is a failure of Ubuntu, Thunderbird, or Libra, they all are in the same boat.

I think my experiences are similar to many others that have tried Ubuntu and probably not persevered for as long as I have. Anyone that has been immersed with Windows (myself since DOS) for a long time will expect another operating system to work in much the same way, including the file system, and for me the file system syntax.

I am extremely grateful for all the help I have received from this forum, I have also learnt a lot and been able to give a little back to others. Reluctantly I will be going back to Windows as soon as I get a suitable system disc.  I will be back as soon as I feel Ubuntu has developed sufficiently for me to work with it, without having to work at it.

Regards  Greg
