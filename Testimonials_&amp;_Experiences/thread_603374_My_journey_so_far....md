### Post by Mikey_MW on 2007-11-05
Well, I came across a couple of negative comments about fluxbuntu (fair enough, everyone is welcome to their own views) but I wanted to share my own experiences.

Ubuntu 6.06 - This was where it all started. I don't know if cannonical posted cds before this version, but the offer of a free trial brought me to the dark side :) Made an unstable Athlon XP PC stable.

Xubuntu 6.10 - Hmmm... placed this on an old PIII Dell laptop - which doesn't have a cd drive. The ability to install stuff via ethernet modem really helped. Fast as well.

Kubuntu 7.04 - Installed this on relatives' computers, and have converted them to the church of (k)ubuntu. I don't have to fix their computers any more which rocks :guitar:

Fluxbuntu (version unknown) - I'm using an old toshi laptop 233mhz 64mb ram, 4gb hd and a weird **2.5MB** Graphics, Sound, Optical Drive. Actually only fluxbuntu would run on this laptop. All other distros x11 would poop themselves. 

All in all, a great journey. I'm holding off on ubuntu 7.10 for a while, trialling it on a spare machine. We just need to iron out a few kinks from this version and it should be good.

So how do I buntu? Xsane and Gimp for backing up and fixing the old family album. I know some say that Gimp isn't professional quality software - but I've backed up and repaired 1500+ photos. 

Open Office - thesis work (I can trust linux to not lose that 150 pages of work :)

Well, thats my $0.02. Enjoy your buntu journey!

### Post by mivo on 2007-11-07
The late beta of Gutsy had caused my old desktop to freeze randomly, so I had gone back to Feisty and then hesitated to upgrade when Gutsy was released (I needed the machine for work, so was feeling uneasy about experiments).

When my new desktop box arrived, I had some issues getting Ubuntu installed, which I had expected because of the Linux-unfriendly Abit AB9 Pro board (but you don't tell your employer you would rather have a different board if they just bought it, 3 gigs of memiry, the CPU and some other stuff for you), but it eventually worked. Actually, it was fairly simple once I had figured out what boot options I had to use. Gutsy has been running very well since, even with many Compiz-Fusion plugins enabled and various other modifications such as the very good and fun Avant Window Navigator.

The old desktop that had issues with the Gutsy beta got upgraded to 7.10 recently too (going to put the Hardy alpha on it closer to the Debian package import freeze) and I have not noticed any freezes anymore, either. So, that's another system where Gutsy lives happily on and works well (it's an ATI system) though I have not customized it much. Just the stock install plus a few extra packages and Compiz Fusion enabled.

Open Office ... it is too "heavy" for me. I prefer something "lighter" like Abiword, which I use for word processing. (Well, actually, most of the time I use Vim, but I don't want to start a religious war! ;)). Luckily, I don't need MS Office compatibility or any advanced abilities for my job.
