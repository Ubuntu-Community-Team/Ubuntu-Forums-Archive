### Post by Fidelio on 2007-03-14
I've been pretty impressed with the installation. Downloaded the ISO, ran the live CD no problems, installed with no problems.
Automatically set up, with no input from me,access to both my windows network and the internet.
Also, my printer, camera and mp3 player worked straight away with no tweaking.
That's pretty damn good I'd say.
Took a bit of tweaking to get mp3s to play, stream realplayer audio, and still not quite there with streaming video.
One or two things not quite there yet, especially Java and some performance issues with I think are related to my video card. But if I can iron these out, and I'm sure I will, I'll be more than happy.
