### Post by WTBCompPro on 2010-08-07
I got to say I have long been a fan of the ideas behind GNU/Linux. Freedom Rights and just the idea the user has control. I like that. Of course like most I am from a windows using background, and it just keeps getting worse. I have been using Linux on my Sony VAIO notebook since 9.10 Karmic Kola. I have then since installed Lucid on it and finally on my desktop. My only complaint could be that the flash player does not work in x64. For that reason I installed the x86 on my home PC and am only able to allocate 3GB of my 8GB of ram. Who needs it though Ubuntu is extremely light weight normally running just my web browser and media player my Windows XP x64 install would use 2GB of RAM 2GB of virtual memory(what is that) and I disabled the paging file trying to speed up my disk access. Ubuntu has only ever reached 1.5GB of memory usage(no swap partition) and that is with a Windows installation running in Virtual Window. The Primary reason I had to quit using windows though is I recently installed a 1 TB hard drive on it, and it just couldn't do it. My Explorer started freezing for 3 minutes at a time randomly when I accessed files on it. I found that booting into a Ubuntu live CD and then booting windows my the problem would temporarily go away. That was the final straw it had to be done. Free Software you can't be it got to join. Love Ubuntu although a challenging learning experience, it has been more rewarding and led to the most stable computing environment I have ever know.

How about them boot times! (I rarely even catch a glimpse of the loading screen)
