### Post by t0p on 2009-10-21
Just a few comments to make about your OP:

The title of this thread implies that your machine is mission-critical.  So why would you do this experimentation on such a vital machine?  If you want to try a different set up, you can do it on a different machine, or a separate partition if you don't own a second computer.

I'm confused about the "disappearance" of your wireless network manager.  Is it still installed on your computer, or has it somehow "uninstalled itself" (meaning: did you accidently uninstall it?)  You can find out if it's still installed by typing into a terminal the command

```
which gtkwifi
```

If it's still installed, you'll get a response something on the lines of

```
/usr/bin/gtkwifi
```

If it's still there, it may need reconfiguring.  If it's gone, a simple reinstallation will solve the problem.  Either way, I don't understand why you're messing with other network managers when gtkwifi always did the trick for you before.  Why change a winning formula?  And if you decide to change that winning formula anyway, why complain here when it doesn't win any more?  Ask for help and advice, sure: we're a helpful bunch here.  But complain?  About something you did to yourself?  Nuh uh!

Finally, a suggestion.  You said that Ubuntu worked fine for you before you messed with the desktop manager.  So why don't you try reinstalling Ubuntu so it's at that state of perfection that you'd achieved?  You *know* Ubuntu works fine when it's set up like that.  Much better than jumping into the unknown by installing some other distro you've never tried before (you mentioned trying "PCLinuxOS or something") or getting a pirated version of a proprietary OS through bittorrent (which is something you implied).  Or you could just use the Windows that you already have installed.  You're obviously dissatisfied with Windows (or why install Linux at all?) but if Ubuntu is as disappointing as you suggest now... well, "better the devil you know"... right?

Oh yeah, one more last thing: Ubuntu 9.10 is due out in a little over a week.  And, as you said, maybe it'll solve all your problems.  Then again, maybe it won't.  Are you sure you want to take that chance?  After all, it *will* cost you quite a bit of money.  Oh no, wait - Ubuntu's free isn't it?  That's right.  And you don't even have to steal it...
