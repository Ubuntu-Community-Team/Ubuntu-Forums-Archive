### Post by 3pinner on 2008-03-08
After about 3 months of using Ubuntu,I can officialy say I'm a convert.
I really like the whole setup, and at this point I do not see any reason to go back to Windows.
Please don't get me wrong. I'm not anti-Windows at all. Learned computers on it. Got a Windows 2000 MCSE a few years ago. My wife still uses it (for now :wink:   )
But I have found that everything I NEED to do on a computer is in Ubuntu. And what I was missing, I easily found, and it was no big deal to install once I got the hang of it.
I like & respect the "free" mentality as well; and makes me wish I knew a bit more about programming to be able to contribute more to the community.

My wallet also likes the "free" mentality! All the hundreds of dollars worth of software to do what-ever in windows, has a Linux version for the most part,( or if not a Linux version, at least a program that will do the same thing - usually for free) and I find I can do just about everything here as I did there.

And I admit there are a couple Windows related things I had to bring with me, LIghtzone for one - a terrific photo editing program that is much simpler than GIMP (And I like GIMP a lot)
There were a couple of other windows based programs that I could not live without, and can run them in WINE.

Overall- this has so far been a very good experience.

 Quirky sometimes - but that's because I make the common mistake of assuming that Linux is going to act like windows.
There is stuff that still needs to be worked on, a good PDF reader that I can print from. Evince won't print for me, but 99 percent of the other software included in the Ubuntu Package works very well.

Would I recommend changing from Windows?

Absolutely. 
Try it out. It is different, but "so what" and it works!

O.K. ............. i'll turn in my MCSE card........:lolflag:

For PDF's you could try xpdf.  Either look for it in Synaptic, or in a terminal, type:
```
sudo apt-get install xpdf-reader
```
Until you set that to be your preferred app, you will have to right click your file and choose to open with xpdf.  I can't guarantee it will work for you, but it's worth a try.  Does your printer work with other things?
