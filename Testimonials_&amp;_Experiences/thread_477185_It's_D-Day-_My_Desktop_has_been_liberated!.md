### Post by mac71 on 2007-06-18
Hi  guys, 
I just wanted to say, that I've only had a pc for about 14 months ( its second hand and quite low spec, I've since  increased the hard drive and RAM but thats it), since then I've learnt a fair bit about OS and a bit about hardware.
I discovered Ubuntu around October last year and went for a dual boot install straight away.
Almost over night, i was spending more time in Ubuntu and less and less in Windows. Ubuntu felt more secure and comfortable(once I got used to the app names(Oh, and it was/is definately more stable)), I was also becoming accustomed to using the terminal- which I don't mine saying was a bit daunting at first.
Anyhow, I found that I only seemed to use Windows to access my All-In-One scanner  to scan some pics or docs. Which was quite rare, and the camera has been replaced with a nice shinny new digital one that digikam "just works" with(its a 450MB install on windows!!). And I've ditched the AIO. So, no need for windows!
I performed a fresh install of Ubuntu and wiped Windows entirely from my hard drive.
I feel liberated! Much faster start up times, no more reboots, just because an application has been installed. No more super slow computer, because my anti-virus is automatically updating, or performing a scan. I could go on, and on.
Anyway, many thanks to Ubuntu , for the enlightenment and the liberation.

mac71
