> 580 MB ram taken? is something running in background? or is this just desktop environment that take up so much?
That is a misleading figure. Linux uses unused RAM for automatic caching, to speed up your machine. So, when you see 580Mb RAM used, it's much less than that. The automatic caching is automatically cancelled when the system needs the RAM.

Read all about it in [Linux Ate my RAM!]("http://www.linuxatemyram.com/")
