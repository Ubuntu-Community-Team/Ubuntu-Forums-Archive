### Post by chewyfood on 2010-08-14
Disclaimer: I still love Windows and have it on my main laptop.  I am completely new to Linux (but I do have a high level of technical expertise), so if there is some hidden option that I am missing to enable certain functionalities I complain about below, please let me know.


I had Windows 7 on my netbook for the longest time, but was not overly enthused with the poor performance of Windows on it.  I was aware I could install Windows XP on my netbook (heck, it came with it), but ever since I have adopted Windows 7 I could never look back thanks to all the niceties in the operating system.  Therefore I decided to give Ubuntu a go (I've had a burning desire to run it based on all the good reviews it has gotten, saying it has made Linux a much more user-friendly operating system.  Jeez I remember the days I had a friend create me a Slackware server for my home and how painful that was to setup).

My netbook is an Asus EeePC 1000HE (Intel Atom N280 1.66GHz, 1GB ram, 160GB SATA hard drive, Intel G945M, Bluetooth, Wireless BGN, webcam, built in microphone, 6 cell battery).

**Regarding netbook hardware:**
Ubuntu had no problems getting everything working, not a single driver issue.  Bluetooth with my phone worked, audio recorder picked up the built-in mic, wireless and wired connection both work, video of course works, video-out works, keyboard and mouse work.  I can't say I ran into any issues.

**Regarding Ubuntu default computing software (office, chat, etc.):**
Considering what a netbook is intended for (not everyday computing), I think the software provided is great for the general internet user.  I prefer Chrome over FireFox, but I downloaded Chromium (for the flash capabilities) with relative ease.  I don't have any complaints with the software, although a few more little games installed by default would be nice :) (minesweeper for example).  Everything I could need is there, audio player, video player, text editor, office suite, webcam software, audio recorder, internet browser...I can't think of any piece of software I need that wasn't there, well except for Adobe flash in the browser.  I found it a little annoying that a) i had to download flash with firefox, b) if i wanted chrome I had to download Chromium AND run it with a flag to enable plugins...definitely not user friendly especially for an internet full of flash content (youtube???).
It is worth nothing that there are certain pieces of software I "have" to have on my Windows computer, especially my work computer since it is a Windows XP machine (since XP doesn't do some things that 7 does...like Aero-snapping) but I don't see the value in making a strong request for having on Ubuntu by default.  Most of them are geeky, while others functionalities are already built into Ubuntu (e.g. a chat program, a program to have a hotkey to make a window appear on top of all other windows).

**Regarding built in software to control hardware (bluetooth, wireless, etc.):**
My biggest complaint is the wireless connection utility.  Yes, it is built beautifully - click the wireless icon, get a list of networks and click to connect - and it works great, but it just lacks certain niceties of the Windows based wireless utility.  Namely, information about the networks listed (connection type b/g/n/etc., security type wpa/wep/etc., strength rating poor/good/weak), and more importantly, auto-connect to secure networks without prompting me to click ok on the password (yes it stores the password in a profile, but it still asks me to connect with the password).
No complaints with the utilities for bluetooth, microphone, or webcam.
Being a hardcore Windows 7 users I am not overly impressed with the way Ubuntu handles multiple monitors.  I feel like I am back in the Windows XP days of multiple monitors.  Windows 7 changed all of that with the WinKey-P hotkey, which pops up a window letting you choose the first monitor, the second monitor, extended monitor, or duplicated monitors.  Soooooo easy!  But Ubuntu was much more difficult and cumbersome, besides problematic (read about my adventures in video playback with multiple monitors here: [http://ubuntuforums.org/showthread.php?t=1552689](http://ubuntuforums.org/showthread.php?t=1552689)).  Yes the computer detected the second monitor AOK and switching monitors or extending the desktop was very easy, but it was not as user friendly as Windows 7 makes it.

Similarly I was not enthused with the way removable media is handled.  Yes it detects it fine and reads all the content too, but it would be great, like Windows (XP even) to have a icon on the gray bar which would let me control the removable device (especially for safely removing it).  I don't feel like opening the file explorer to eject the media.

**Regarding Ubuntu itself:**
Coming from Windows 7 on the netbook, I am impressed with the startup and shut down time.
I am also pleased (not impressed, though) with the operation of it.  I say "not impressed" because while things certainly open faster than Windows on the netbook, I was expecting it to be lightning fast - that was just what I believed going into Ubuntu.
I really don't like being prompted for my password for some mundane tasks.  Yes, Vista and Windows 7 have this (UAC), but they make it rather simplistic to disable the frequency of being asked for permission.  And yes, I know that you can change this in Ubuntu too, but I understand that this task is certainly not for someone who isn't the computer literate type (editing some configuration file or something?).  I just think that if Ubuntu is to be considered a viable desktop option there should be a user-friendly way to allow users control when Ubuntu asks for a password (when you unlock your computer? yes/no, when you install updates? yes/no, when you install software from the software center?  yes/no, etc.)

**Regarding Ubuntu setup:**
Very good experience.  I formatted and installed from scratch via USB flash (didnt really care for a dual boot).  Had no problems.

There are some things I would like to see added to the setup process coming from a Windows user.  Things like:
-A prompt asking you about Linux updates whether you want auto-install, or auto-download with prompt, or no updates at all  (similar to Windows updates options).
-Security settings (with the password prompt options I mentioned above)
-Network setup (Windows asks you about connecting to a wireless network on first boot after install)


Not much else I have on my mind about it.  Considering my extent of use (netbook internet user) I don't think I will ever run into many of the problems or deficiencies more technical users will experience.  All in all, a great experience and I will keep Ubuntu on my netbook :).

### Post by murderslastcrow on 2010-08-14
I'm glad to hear of your good experience! Due to legal and ethical issues, certain codecs and plugins, which are closed source, are not included with Ubuntu. This includes Flash.

However, the easiest way to install this and have it enabled in all applications, as Flash is an ISV with official Ubuntu support, is to install Flash through the Software Center. Of course, if you want codecs and other typically installed closed or legally iffy software, you can install the ubuntu-restricted-extras package from the Software Center. This will take care of everything outside of DVD playback for you, which would require one extra step.

The installer for the next version of Ubuntu will include an option to install these closed drivers/codecs/plugins if it's necessary for a better user experience. This way Ubuntu can remain completely open source, but still offer people who want these options a simpler way to install them.

Outside of that, it seems you had a fairly positive experience. Also, wicd is available as an alternative wireless network manager that offers the information you're referring to. However, it hasn't been included by default for some time, as it was regarded to be too complex, and the code was more heavy than necessary. For most people, a simpler yet fully functional interface is more useful/easy on the eyes.

Also, if you right click the network icon there are many options and dialogs that would present you with this extra information, and then some.

Thanks for your thoughtful outline- I hope you continue to enjoy Ubuntu! Feel free to ask us any questions about applications, customizations, or anything else you're curious about. :D Glad to have you!

### Post by cariboo on 2010-08-14
I'd like to address a couple of your points. 

Flash isn't installed on Windows by default either, you have to download it too. I agree it's easier to do on Windows, but that is more the fault of Adobe than anything else. Installing the ubuntu-restricted-extras meta package install java, flash, mscorefonts and most of the codecs to play most types of audio/video media. 

I use chromium-browser as my default browser, I've never had to enable anything in order to use flash.

As for must have programs, depending on your needs, there are over 40,000 packages in the repositories, many of them are libraries, but there are quite a few programs that aren't available on Windows.

As for network-manager, I guess it depends on what type of information you are looking for, as long as my wireless works the way it should, I don't persoanlly care about extra inforamtion. The thing about using a linux variant, is that it is amazingly customizable, if a program doesn't do what you need, you can either use something else from the repositories, or create it your self.

Your suggestion about removable media is a valid one, I too don't really like having to open the file manager to safely remove a device. The only thing I can say in defense is that Ubuntu works on a 6 month release cycle, If you find something is missing, you can always request that a feature be added to the next release.

Most people that come from Windows complain about having to enter passwords to do any administrative tasks, that is because the design of Linux is totally different from Windows. Linux was designed from the ground up to be a multi-user system, so each user originally only had access to the files in their own directory, any actions on files in directories other than the home directory need administrator access. Windows is changing to be more in line with Linux and OSX.

Because of Ubuntu's 6 month release cycle, the only updates you will get after a version is released are security updates. The update manger is set by default to inform you of available updates once a week, it is up to you whether you do the updates or not.

You do have some valid points, but remember Windows has been around for 25 years, Ubuntu's first release was in 2004. The other thing to keep in mind is that things are done differently in Linux, I would suggest you devote just as much time learning to use a Linux variant as you did learning Windows. It becomes easier and and the way things are done do make sense, once you know what you are doing.

The next release of UNE will be 10/10/10, the user interface has been completely redesigned, I think you will be quite happy with the changes that have been made.
