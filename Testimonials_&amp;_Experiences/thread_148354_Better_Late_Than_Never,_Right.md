### Post by Trisa on 2006-03-21
I have been a Linux user for about 2 years now, but I have been using Ubuntu for only a year, so I guess I can still post my experience with Ubuntu Linux here.:D

I first started using Linux when I got curious about it. I was pretty happy with Windows XP; I never had any problems with it, and spyware and viruses were never an issue. My first distribution was Red Hat (can't remember the version number right now). It was a difficult experience for me. I remember my sound card wasn't working, and I wasn't able to access the internet from it. I uninstalled Red Hat and went back to Windows for a while.

So where does Ubuntu come in? I was using Suse 9.1 when I gained interest in Linux again. I can't remember exactly where on the web, but I read an article about Ubuntu and how it was a young distro. I tried it when Hoary Hedgehog was released. It was quite wonderful on my computer. Everything that didn't work before did, and then some. For there, I've been a Linux user ever since.

Currently, I'm using Dapper on my IBM Thinkpad R50e, and it's very nice so far. Granted there are still some small problems, but that's to be expected. I still have a partition of Windows XP when Cedega doesn't cut it (DirectX games such as AIRRADE -AIR- and Pocket Kanon & AIR), but I use Ubuntu more often than Windows anyway. Linux has gotten a lot easier to use than before, so I'm very glad for that.
