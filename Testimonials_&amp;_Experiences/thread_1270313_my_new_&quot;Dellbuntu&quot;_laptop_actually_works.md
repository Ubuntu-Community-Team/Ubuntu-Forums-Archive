### Post by armandh on 2009-09-20
Dellbuntu is good BUT.....on my mini9
I needed to dual boot with XP for some win only software.
XP wants to be the first active partition
the Dellbuntu restore disk does not install, rather it wipes and copies.
nor did it support compiz [possible conflict with the Dell desktop]
so I created a ntsf partition, first, active, about 50% of the drive.
installed XP
then installed OOTB Ubuntu to the "largest unpartitioned space"
add the [www.medibuntu.org](www.medibuntu.org) stuff, 1 line edit for audio, and adobe flash 
all is well
I do not know if dell has improved the update situation but in the past,
there have been concerns about less than timely security updates.
all things considered dellbuntu is good to go but [IMHO] limited
in the long run those with greater needs will want the real deal.
