### Post by anomie on 2010-05-14
I installed 10.04 Netbook Edition on my ASUS MK90H (yes, that's the goofy Disney netbook, intended to be marketed to children). I am happy to say - with all respect - that this is the first Ubuntu release I actually *like* and will be using regularly. 

[list]
[*] The netbook-focused UI is wonderful. It really does make intelligent use of 1024x600 screen real estate. 
[*] The GUI package manager actually works. (i.e. My system is not inexplicably borked after a couple updates.) 
[*] The default installation did not enable all sorts of unnecessary services (although I shut off cupsd and avahi-daemon post-install). 
[*] The "System Testing" applet, which can be run before installing anything, is fantastic. 
[*] The revolting brown theme has finally been dumped, in favor of something more visually pleasing. Yes, I know that's subjective, but this is my review. ;) 
[*] Did I mention the netbook UI rocks? Very, very fond of the design. 
[/list]

Great work, folks!
