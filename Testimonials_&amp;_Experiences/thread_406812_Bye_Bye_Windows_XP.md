### Post by feravolo on 2007-04-14
Just to let people  know that Ubuntu can run great on just about anything worth spending twenty dollars on:

WINBOOK: Celeron 533 and 192MB of RAM 6 GB HD (CD, 2 PCMCIA, 2 USB )

Ubuntu running just fine on this old notebook computer and should run even on slower CPU's if you have the 192MB RAM.  We use this computer to demonstrate to people how Ubuntu can give old computers a new lease on life. 

Firefox, Open Office, Remote Desktop/VNC  Clients all run just fine and I am able to access the "real computers" back at the office without any problems while I am on the road.

NOTE: Wireless PCMICA cards require that you  installation of that same .deb package and the Windows Device Drivers off of the CD that was included withe the wireless card.

 Since; this machine is so old ("built for 98") it doesn't have a built in LAN connection, we need to use a PCMCIA LAN card to connect it using a cable. All we had to do is plug it and then startup the system and we had a wired connection too.

I'd like to thank the whole of the Ubuntu project team(s) for making a superb operating system, one which will be with me for a lifetime!

The dock like bar at bottom is from this guide

 Let's begin with the **bottom panel**. Our aim is to have a panel which  contains only application launchers. The first step is to remove all the other  elements from the panel. In default ubuntu installation there are *Recycle  bin*, *Taskbar*, *desktop switcher* and *show desktop*  -button. These are added later to the top panel so don't worry. You won't lose  them forever! You can remove these elements by clicking right mouse-button on  them and selecting *"Remove from panel"* from pop-up menu. Now you have  empty gray panel. Next click right mouse-button on the empty panel and select  *"Properties"*. Change the Size of the panel to **50** and remove all  the ticks form the selectionboxes (See [_[COLOR=#0000ff]screenshot[/COLOR]_]("http://www.taimila.com/images/osx/panel_general.jpg")). Then  change to Background tab and select *"Solid color"*. Change the Color to  white, if it's not already, and move *"Style"* slider like in this [_[COLOR=#0000ff]screenshot[/COLOR]_]("http://www.taimila.com/images/osx/panel_background.jpg"). Now the  panel is ready. Next we just add icons (application launchers) to it. You can  simply drag and drop the icons from the desktop. Another way is to click right  mouse-button on the panel and select *"Add to panel..."*. Select *"Custom  application launcher"* from the window and fill *Name*, *Command*  and select some beatiful icon. You can also write a short comment about the  program, if you will. (See [_[COLOR=#0000ff]screenshot[/COLOR]_]("http://www.taimila.com/images/osx/create_launcher.jpg")) 

The top bar e.g application is on 0 so its see through

So all in all its no theme just small changes
