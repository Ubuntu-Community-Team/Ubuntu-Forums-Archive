### Post by jp1991 on 2012-03-04
First let me start by saying thank you to the Ubuntu community for amount of help and knowledge that is available. About 14 days ago a few things in my home were fried during a thunderstorm. 1 thing it fried was my main desktop PC running Win7. My first thought was to get the replacement parts and get it back going. Then I decided that I should wait and have my home checked by electricians and file a insurance claim if it all totaled up to enough $. I mean I can get by because I have other computers i can use. My wife has a desktop and my daughter has a laptop, both running Windows. Then I have an IBM laptop running Linux Mint and another laptop running MEPIS. The problem was that I really like using a desktop when I'm at home. So I decided to use some older parts and build me a desktop to use while we sort all the insurance stuff out. I have a lot of files for different Linux ditros and figured I'd go with Ubuntu 10.04 LTS. Now here's the best part. I've had zero problems so far with this PC. I've gotten it updated and installed the recommended drivers with no issues. Anything that I have had any kind of issue with ended up being my doing and was easily fixed with help of members of these forums (thank you again). I'm actually considering installing Ubuntu on my new PC when I get the replacement parts. 

So, in a nutshell, my Ubuntu experience has been great, and it's all because of the people involved in it's creation and the members here.
