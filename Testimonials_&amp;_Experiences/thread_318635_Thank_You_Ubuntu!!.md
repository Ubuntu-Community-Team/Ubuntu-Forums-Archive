### Post by 200mg on 2006-12-14
I have been a windows user since 3.1.  Windows is a fine OS, user friendly and mainstream, there is nothing wrong with that.  I have always wondered about linux and even tried a couple of different distros through the years.  SuSe was my fav for quite some time.  But i have to give kudos to the Ubuntu team for making a distro that is extremely user friendly besides missing some key stuff.(no traceroute??)  I find myself only turning to my xp machine for gaming.  Everything runs so much faster it's unbelievable.  Ubuntu with fluxbox is exactly how I want a computer to act and look, In windows you would have to skin windows juicing memory and so forth.  Ubuntu with flux is streamlined and because you're not using certain parts of the windows manager, the unused parts actually aren't using memory..huh imagine that?

     I believe ubuntu is the first step in the eventual trend toward open source.  You can see it (open source) penetrating windows now with firefox.  Eventually the masses will realize it goes beyond browsers.  I guess MS will just have to depend on Zune and Xbox sales in the future.

3.  everything runs faster
4.  gimp > photoshop
5.  IT"S FREE!!!!!
