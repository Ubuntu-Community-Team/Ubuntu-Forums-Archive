For quite a while I was happy running 6.10 then 7.04 -- and of course I have the BCM 4318 Anti-Christ wireless chipset on my Dell Inspiron B130. So I had to hack the install with Ndiswrapper to get wireless working etc.

This is a huge step forward for useability ----  Great job:guitar:
