### Post by KaoriKandiez on 2009-12-08
First off, I love doing things myself. I've always wanted an operating system that made things less of a hassle, but didn't try to think for me. I didn't want an OS bogged down with a bunch of crap applications you have to buy in 30 days or they expire. I wanted something spiffy and unique, aesthetically customizable, and above all else open source. People say open source software is garbage, but Linux distributions such as Ubuntu prove this theory wrong. 

The system I've been using for Linux is nothing special:

-2.8ghz Processor Pentium 4
-6200 A-LE 512mb 
-1 GB RAM
-40 Gig HD

Sure, it was nice when I got it (about 5-6 years ago, still works fine today), somewhere around the early age of the Dual Core era, but it's worth probably 300 $ in today's standards. However, one thing I noticed is that it runs Karmic Koala WAY better then XP. No freezing, no lagging, pretty stable, and handles multiple tasks with ease (single core, mind you). Ubuntu is obviously the better OS in terms of performance when compared to XP. 

Let's examine the recommended requirements for both XP Pro and Ubuntu.

XP Pro:

[LIST]
[*]Pentium 233-megahertz (MHz) processor or faster **(300 MHz is recommended)**
[*]At least 64 megabytes (MB) of **RAM (128 MB is recommended**)
[*]At least 1.5 gigabytes (GB) of available space on the hard disk
[*]Video adapter and monitor with Super VGA (800 x 600) or higher resolution
[/LIST]
Ubuntu:

[LIST]
[*]700 MHz x86 processor 
[*]384 MB of system memory (RAM) 
[*]8 GB of disk space 
[*]Graphics card capable of 1024x768 resolution
[/LIST]
Okay, so apparently Ubuntu requires more processing power, RAM, disk space, and graphics. *"Oh! That makes Windows better, right?!"* 

**WRONG!**

I had to run programs like Advanced System Care (really good program for windows users btw) almost bi-weekly, defrag every month or two, use ctrl alt del to get rid of frozen processes (ugh), format annually, and still didn't run to my expectations. I've had Ubuntu for a while now and I don't deal with any of that! Stupid windows.

Now I take care of all my PC's, and this one is no exception, but I've literally not had to do a freaking thing with Ubuntu. I know, defragging isn't necessary on Linux, thank god for ext4. Granted, all I have to occasionally do is empty my trash, clean up the apt cache or some unused dependencies. XP I had to get programs to do all that for me, and it wasn't just sudo apt-get install <apptosolveprobs>, it was searching over the net for 5 minutes for a reliable program, downloading it, installing, cleaning up after it, etc. The terminal/software center is really great for programs. And running a deb package/registering a key file isn't that hard too (if for some reason you can't find an app in the software center).

And security? No more running crappy, resource hogging applications like Norton anti-virus. I mean I used AVG but it's nice knowing I don't need it in Ubuntu. 

As for Aesthetics, I love how Ubuntu runs everything under one theme. Firefox, Pidgin, etc all use one theme. I have the Sickness Black theme installed and it looks great, especially with everything matching. Windows had themes for everything: Xfire, MSN, Firefox, Windows itself, any other program. And making a general theme in correlation to all your program's themes without using Windows blinds or some fake garbage was quite challenging. Ubuntu puts everything into a simple and eye-pleasing feel for all the applications. And adding screenies or docks or eye candy decorations on Ubuntu is like putting blue onto black, rather then blue onto purple/red/green. Also I love the way it displays pidgin alerts/current music tracks/etc. Very cool, simple, convenient popup.

I've always enjoyed Linux and always will. But, now onto my personal problems with Ubuntu. I'm not trying to troll, but be generally honest here. 

Despite the hardware on this PC, I've wanted to run a number of games and applications that I really needed. I like Adobe Photoshop CS4, it's in all honesty the best image editing software out there for a Graphical/Site developer like myself. No hating on gimp, but it just doesn't add up to the latest releases of Photoshop. It's neat software, but Photoshop has a wider variety of features. I was actually able to get Adobe Photoshop CS4 extended running very well on Ubuntu, by throwing in some DLL files and getting an earlier version of Wine (1.1.7). This ties into my complaint, stay with me here.

I'm also a gamer, despite the 6 series I'm using on this PC, I play a variety of games on here. It runs games like Rakion, GTA Vice City, Wolfenstein, Sauerbraten, Halo, etc just fine. My higher end compy is what I use for games like L4D2 and COD MW2, etc. 

However, being aware that a higher-end ATI would run much much better then a lower-end nvidea on Linux, I didn't expect the windows games I wanted to play to even start up. It did, but it runs halo like garbage, and the Rakion controls are off, and the graphics lag terribly even on the latest Wine releases. That's another thing, installing 2 versions of wine is so annoying (needed an earlier version for photoshop, remember?), because you have to compile them (zzzz) and use different prefixes and it's just a giant pain. So I haven't quite gotten those games to work that well. A positive thing I noticed though was that I get much better Ping/MS ingame. But I don't think my little Nvidea can handle running a game through wine on linux, and I can't afford a better card for it atm. Hell, a better all around system would probably improve this situation quite a bit. And I am not blaming Ubuntu for this (as these programs aren't even DEVELOPED for linux), but it does set me back a bit. See, the main games I played /programs I used with ease on windows XP are MUCH more of a hassle to use on Ubuntu at the moment. It would just rock my world if I could get these programs/games to run similarly/better then they do on XP. 

I'll probably add more to this later, as I'm tired and going to bed. Hope my testimonial/2 cents was useful and thank you to everyone working on and further developing Ubuntu :p
