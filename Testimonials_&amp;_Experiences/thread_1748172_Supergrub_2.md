### Post by stoogiebuncho on 2011-05-03
Not Ubuntu related, necessarily.  

Recently my parent's computer had a boot problem and would only display the dreaded grub> prompt.

No problem, I thought, I've reinstalled Grub using a LiveCD a couple of times, it's easy.  I found an old ubuntu disc and went on google just to be sure I remembered all the commands to find and assign the correct place to install grub.

Then I stumbled across [Super Grub Disk 2]("http://www.bootproblems.com/super-grub2-disk/").  It sounded intriguing, so I downloaded it and burned it.  

Put it in my parents computer, booted it.  It pops up within seconds and ask what I want it to do.  I ask it to detect the operating systems.  Two seconds later I have a list of all the linux kernels and the Windows installation.  I ask it to boot the most recent linux kernel, and bam, I'm in Ubuntu, and all I have to do to to reinstall grub is a simple grub-install dev/sda

Just thought I'd share, as this tool makes an admittedly easy task even easier.
