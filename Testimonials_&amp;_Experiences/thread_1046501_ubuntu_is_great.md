### Post by weisel on 2009-01-21
I've been on windows like forever started reading about Ubuntu(Linux) decided well i am going to give it a try. I used Wubi to install and found out i really like Ubuntu. After couple weeks decided i would do a clean install and uninstall Wubi.Got to be the best move i ever made just fantastic. Only other thing its like starting over learning a new system but, there are loads of information on this forum to help you. Windows i don't use unless i have to, only  if i have to use my printer Canon MF 4150 Laser. Other than that my system has been flawless everything just works no junk to install like Windows junk. Ubuntu 8.10 has a very clean desktop which is really nice. I would definitely buy a computer with ubuntu installed with no windows i would just have to find a printer that works.Love this system and thanks to the developers who came up with this idea great job.
