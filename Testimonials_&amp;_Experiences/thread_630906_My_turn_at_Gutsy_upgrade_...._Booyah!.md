### Post by Geekkit on 2007-12-03
I'll try to be short and sweet. Went from 7.04 to 7.10 without a hitch on work computer. It's a desktop computer with an onboard Intel graphics card. The difference being that compiz actually works (in 7.04 it didn't).

I did notice one funny behavior in 7.10 that is carried over from 7.04 and that it my mouse pointer icons are a mixture of two different schemes (i.e., mouse cursor over the text field is the default but moving icons on desktop or corners for resizing uses a different set of icons like green or something).

But other than that, the printer works, USB, CD-ROM, sound, etc., etc. I'm going to try a clean install on home computer after I've backed up all my crap first though - my home system is a little more touch and go since the H/W is quite old and I'm not sure what will happen.

On another note I found ObexFTP and ObexFTP Front end which allow me access to my Nokia phone's memory card which is quite cool. No more diddling fstab - I'm always nervous about that.

So this is 6 months of me now using Ubuntu as a (I know some of you hate this term but I'll say it anyways) Windows replacement. My personal experience is that Ubuntu is faster than XP (a lot of this being due to anti-virus software in XP having to watch over your every move which slows down performance considerably), more secure, less maintenance (gawd I love apt-get, synaptic, and the updater!!!), and personally more fun to configure (can change bloody well anything I want in it which is fun sometimes).

:)
