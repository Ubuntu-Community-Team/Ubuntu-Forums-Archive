I think I got the gist of your message, despite the broken English.  So here are my thoughts on what I think you said.  Sorry if I misunderstood.

1.  Have you checked for and filed bug reports on each of these issues?  The developers *RARELY* read these forums, so complaining here doesn't help.

2.  Did all of this work on Jaunty?  Did you do an upgrade or a fresh install of Karmic?  I would recommend a fresh install, just to be sure that something didn't go wrong during the upgrade.

3.  I know I've said this before, but it bears repeating: a lot of people are complaining about Karmic, but my computer works great in Karmic.  Why?  Because I tested Karmic for a long time to make sure that it would work for me.  The bugs that I filed during Alpha 3 and 4 have been resolved.  If someone doesn't test on your particular hardware, there's no way to know if it will work.  So *please* help by testing and filing bugs.

This is a largely volunteer community.  Remember that Ubuntu is free, so if you want to complain you have to help out as well.  If you file bugs early enough, then the devs will most likely be able to fix them prior to release, and they will be happy to do so...they just have to know what to do.

4.  If you are unable to volunteer to test and file bugs to get your hardware working, then you can always talk to Canonical directly.  Their support services are quite reasonable.  Depending on what you need it will probably cost less than buying a full copy of Windows 7.
