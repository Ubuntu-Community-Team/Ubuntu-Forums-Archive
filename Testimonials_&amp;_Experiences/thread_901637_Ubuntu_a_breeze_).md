### Post by Jaxster on 2008-08-26
Although i have used many distro's I have yet to really use one as a sole OS so i thought it was time since I had got fed up with xp and personally cannot stand vista so I thought I would try Ubuntu as I had previoulsy installed a verison on an old dual boot but only ever used it to develop C code.

As I expected a much nicer installation than the windows counterparts it allows much easier customization and in my case after a few tweaks I could do a direct install from HD since my cd-drive decided it wasn't going to read cd-r,

After installing all the software I would ever use and installing any drivers that I would need I find the whole experience much more pleasant and only regret not fully using linux earlier on.
