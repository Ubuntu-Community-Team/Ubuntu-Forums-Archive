### Post by keithpeter on 2009-08-11
Hello All

Just thought I'd record a few data points here to reduce the 'sample bias' (forums get used when people encounter problems). 

Asus Pundit (Athlon Dual Core 4600, 2.4GHz, GeForce 6150 graphics with 128Mb memory leeched from the 1Gb RAM) just works with Jaunty using the recommended restricted drivers. Only thing I had to do was try a few mixer settings to get the sound recorder working. 8.10 and 8.04 worked nicely as well but I wanted OpenOffice 3 that wasn't beta.

I also have Jaunty on an Asus EeePC 701, Asus EeePC 1000 SSD with no big issues (sometimes Firefox 'greys out' on long pages on the 1000, I think it might be the slower SSD). Love the Network Manager and wifi that just connects.

I've had most Ubuntu versions from 5.04 on an old Dell L400 P3 256Mb Ram laptop (12 inch, but poor battery life and runs hot) with no issues apart from fan control on 5.10. I'm using a minimal install with SLIM and LXDE on that machine now and its usable. Didn't put the network manager on, I just use wvdial with my Mobile web modem.

I should say I use a separate home partition on the Pundit and on the Asus 1000 so I upgrade by doing a clean install on the small OS partition as / and set the mount point for the larger partition as /home.

Here is hoping for a good solid 10.4 LTS (if it is the next LTS) so I have even fewer excuses to fiddle and have to just get work done! 

Isn't it amazing that you can get all the software you need to do most tasks for nothing?

Cheers
