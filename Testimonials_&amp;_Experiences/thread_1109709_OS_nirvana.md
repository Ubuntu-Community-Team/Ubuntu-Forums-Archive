### Post by nightrider.36 on 2009-03-29
I've been operating personal computers since the days of the VIC-20. I've used PCs, DOS, Windows, Macs, Amigas, you name it and I've probably operated it.  Ubuntu 9.04, although in beta is the best operating system I've ever used. I know that everyone's got their opinion about what's important in an OS but mine now rests in Ubuntu.

I'm a writer and a graphic artist. I do a little Perl scripting but not enough to consider myself a programmer.  Everything just works so smoothly on my computer now.  The fact that I can do professional level work using free tools is fantastic.

I finished tweaking my os's configuration and I feel like I've reached OS nirvana and at present have no need or desire to return to the Windows world.

### Post by nightrider.36 on 2009-03-29
> **JazonEsti said:**
> nightrider.36,

What program do you use in your graphics work? I am not a graphics artist, but I do meddle with pictures from time to time. Thanks!

It depends on what I'm working on. For general graphics manipulation and photo editing I use _GIMP_. Photoshop's still the graphic artist's warhammer but GIMP is an effective substitute.

For *3D modeling* and *rendering* I use _Blender_ and _POVray_ (rendering engine).

*Desktop publishing* I use _Scribus_. It's not Pagemaker but again, a great substitute.

_Inkscape_ for *illustration*.

I really like _ImageMagick_ because I can work it in to perl scripts and because I can make a lot of graphic adjustments from a command line.

I use _Picasa_ to manage graphics and make simple color adjustments.

Here's a great resource link to open applications that can be used in Linux.
[http://www.linfo.org/software_artists.html](http://www.linfo.org/software_artists.html)

See? Now that's what cool about the Ubuntu distro. If the beta completely trashed my computer, as Windows betas have, I would just reformat and reinstall the stable version that I have, 8.10.  It takes me about 30 minutes from start to finish compared to Windows which used to take me several hours because of all the patches.

I don't keep anything critical on the computer anymore so that I can rebuild it quickly. I use Gmail for email, Google docs for docs and spreadsheets and everything else can be reinstalled quickly from the synaptic manager.  I'm not seeing the reason for large hard drives anymore. Music mp3s go on DVDs now. It takes some getting used to doing things like this but it works out.
