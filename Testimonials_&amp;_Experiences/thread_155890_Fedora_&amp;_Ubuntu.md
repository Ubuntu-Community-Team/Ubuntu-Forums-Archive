### Post by motionsiren on 2006-04-05
Im a FreeBSD guy, mostly, and today I had to refresh a file server. I decided to go with GNU/Linux as it's file system (ext3) has better history and toolset for data recovery and seeming as how my needs are generic, i just want a fast and reliable OS that can update itself (rather do the several menaul steps in FreeBSD to accomplish the same thing.

I got ready to instal Fedora and was completely turned off at the install method, interface and options.... I could of got through it, and moved our data back in and get the ball rolling again, but i just did 'feel right' and i want to feel right, as im responsible for this machine and it's services.

All of this frustration was ended with a simple

```

boot: server

```

a nice, minimal install + package managment to get all the rest. I felt as if i were in a OpenBSD system.... the bareness of it.

and i really, really, appreciate that.

Thanks guys.

BTW, I've been running you as a desktop since 4, but today was my first dedicated server install. flawless.

Your 'just works' approach is working. You can count on me for donating to the project.
