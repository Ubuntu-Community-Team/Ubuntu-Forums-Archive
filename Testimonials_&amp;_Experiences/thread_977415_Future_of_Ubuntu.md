Hi :)

Comparing Vista's boot speed with Ubuntu? I can do that :) Because i had (!) Vista Ultimate on my machine. Genuine. The machine in question is by now way outdated but does a very decent job for me, still.

So, Vista. No password, i'm the owner and one of only two users on this machine (the other being my boyfriend who needs no extra account, we have no secrets) so it booted straight to, well, usability.

Took some 45 seconds from pressing the power button for the desktop and all icons to be present, HOWEVER another good 1:30 minutes before  i could actually DO something! Meaning if i clicked, say, the "Firefox" icon right after the desktop appeared, Firefox would not open for another one-and-half minute because Vista just wasn't done booting yet, it kept rummaging on the HDD searching for it's bits and pieces. My AV was "Avast".

Now, Ubuntu. 8.04.1 Hardy, i'm sticking to it. Compiz with all bells and whistles, Bluetooth dongle present, AWN. This takes an eternal 1:18 minutes for the desktop, icons and AWN to be present (Desktop comes first, then icons, lastly AWN but all within 4-5 seconds), again with auto-login. 

AND I CAN USE IT RIGHT AWAY!! The moment AWN shows up, all HDD activity stops and boot is completed, clicking "Firefox" will open it in a few seconds. 

So for me, Ubuntu clearly is the faster of the two. 

My Ubuntu is currently using 430 MB of my RAM. That is with 8 Firefox windows (not tabs!) open and one instance of KMPlayer, plus having uTorrent in Wine running in the background. Windows Vista would, all by itself (and the AV) happily gobble up 650-750 MB. Still, i have paid money for that RAM so i want my OS to use it, because that's what it's for. If Ubuntu would use all of the 2 GB and therefor never page anything to HDD i'm happy (and indeed my swap is kind of unused - 23 MB all day long, i don't know what it put there but it never changes at all regardless what i do with the machine, i checked that before). 

Advanced install options? Somewhat agree. I, too, never use Evolution and Transmission, however i also am not restricted in HDD space (at least not to the point i'd have to worry about those programs). See, Windows Vista demands (!) 15 GB upon install (it won't install if you got less!) and after it is installed it actually uses about 6-7 GB. Which is HUGE, and that's plain Windows. My Ubuntu right now is 5.6 GB and that is with a mile-long application list :) But i guess you could also install Ubuntu on a 1 GB drive...... basic system with GUI, Firefox and T-Bird. Heck, it even runs right off the CD, now try THAT with Vista :)

New Theme? Have you done that with Vista or XP? There (XP at least, i think Vista it's not possible at all?) it requires some rather obscure system hacking with modified dll files etc to change the theme. In Ubuntu you download some, drag-and-drop them into the themes manager, done. How easy do you want?? And what's wrong with Ubuntu's brown themes, don't you like chocolate? I've changed mine (like many do) to a black one, but i keep enough of Ubuntu's "human" style because it feels VERY comfortable, as opposed to XP's blue, silver, green or Vista's Aero. The Ubuntu people have thrown together a bunch of colours that are very pleasant to the eyes, a very important factor if you have to spend the whole day in front of a computer for a living! 

Vista's boot loader? Why? It doesn't work with Linux, where as Ubuntu's boot loader (GRUB) works fine with Windows. Now ask yourself who should allow who?? Hint: The selfish ones have an office in Redmond :)
