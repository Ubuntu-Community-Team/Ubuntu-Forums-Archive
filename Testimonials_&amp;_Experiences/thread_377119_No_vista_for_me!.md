### Post by Iceni on 2007-03-05
Hi everybody, and thanks for the help I've received in the short time I've been a member here. Here's my success story:

I call myself an advanced user. In the current generation of msn/myspace users I'm happy to chat on irc and not try to meet my soulmate on the internet. I've been using microsoft systems my whole life, from dos to windows and into xp. However, with the recent vista release things really got downhill. WinXP was okai with a strong computer and a good security setup, but vista really bummed me out. So I looked around a bit. My last real look at linux was a redhat distro back in '99, and I figured it should have matured a bit by now. A couple of friends are linux users so I could get help if I needed. 

Ubuntu was my choice, so I installed dapper. Had heaps of reinstalls because grub refused to install itself and everytime I installed my computer would still boot windows and only windows. Solved this by pulling out all my harddrives and installing ubuntu on a fresh sata drive. I was impressed. Sure, it didn't look like much, but it was fast, efficient and easy to work with. The usual nvidia problems hit me, but I worked around them and got the drivers to work pretty flawlessly. 

The usual linux newbie stuff worked like a charm. Beryl got a lot better after I upgraded to edgy - flawless update, btw - and the latest drivers. I can read all my ntfs disks, I can run most of my windows programs with wine and I found myself a nice theme that screams professional. No facy crap I'll get bored from in a week. However, if anyone is still with me, I belive this is the main weakness of ubuntu as a windows competitor. The standard interface is not inviting enough. There are a million themes, but as a newbie one have to read a lot on gtk/metacity/emerald/compiz/whatnot to even get to the point where the desktop can even compete with the professional, efficient windows-interface. I hear mandrive does this well, but as usual I have to edit things to run the live cd, so I didnt bother because I'm happy with my ubuntu now. 

So, this was another success story in the end. I still have a few things I want to sort out, mainly setting up dual-boot to play a few games that won't run well within ubuntu, but I've gotten everything else to work so there is no doubt I'll fix that when I get home and have some time. 

Thanks ubuntu team;)

Oh btw - opera is for linux! Bye bye firefox!
