### Post by spliffeh on 2010-12-28
- Ubuntu 10.10 fails to install in VMware.. something about the iso being invalid.. meh.. the iso is fine.

- So I start over and download the v10.04 iso and try a 'real' install.. no more messing around... I need Linux now....

- The Wubi installer decides to force 64 bit on me.. grrr... I wanted 32 bit for a reason. But it automatically starts downloading 64 bit and I have to wait another hour. Joy.

- Too many hard drives (I have about 6) causes Wubi to have a mental breakdown. It immediately starts spamming error pop-ups after clicking the install icon (apparently too many hard drives confuses it). I understand this problem is a few years old... and still isn't fixed.  Lovely.

- After resolving the above errors, Wubi fails to install on a NTFS RAID 0 hardware array (where my Windows 7 lives). I was told it would work. Argh. Had to install on a 3rd drive. Fun.

- Once installed Ubuntu 10.04 fails to recognize my ntfs hardware raid 0 array, it cannot be mounted. What driver do i need? Argh.

- Ubuntu has no sound on an Asus M4A87TD EVO motherboard. it detects the onboard sound, everything looks fine, but there is no sound. Ubuntu irc channel is clueless, no help there. Nobody is replying to my forum post either. Sigh.

Man it's been a rough day.... /end of rant (sorry had to vent!)
