### Post by makeupaname on 2008-07-27
<<Rant Warning>>

Yesterday, I decided to install Vista on my dual-core 64 bit AMD HP computer. I started at about noon and by 10:30 PM(10 hours later!!) the computer was still configuring and downloading stuff. If you tried to run the web browser or another program while it was working you would click and then it would sit there for a good five minutes before it would start to load the program. Finally about 11:00pm it seemed to be finished. I started I.E. and began to surf. During this whole thing the harddrive is thrashing away like it's calculating trajectories for NASA or something and the cooling fan would go on high every few minutes. After about 20 minutes of browsing, the webpages are taking like 2 or 3 minutes to load with the harddrive light on solid. 

I started thinking to myself. "Why did I do this? This was stupid. Why would anybody subject themselves to this?"

I got out my Ubuntu 8.04 disk. Put in in the drive, and restarted the machine. Spent a few minutes entering the necessary information and then went away and watched TV for 10 or 15 minutes. Came back, took the disk out, closed the CDROM drive, hit the restart button in ubuntu. Then I went into hardware drivers to load the nvidia driver, restarted again and I was up and running. Maybe a grand total of 20 or 25 minutes to a stable operating system. I surfed a while. Then I started the upgrade manager and went to bed.

Got up this morning, rebooted the machine once more and here we are. The harddrive light is off. The fan is off. Oops, the harddrive light just blipped. This is the way the computer should run. I started out my career as a system administrator on a PDP-11 computer. This HP is a monster by comparison. We supported, I don't know how many users on the PDP and Microsoft writes software that this HP can barely keep up with. My feeling is that it's a deal between Microsoft and it's partners, particularly it's hardware partners. Microsoft writes software that will barely run on the current hardware, the hardware manufacturers make better hardware, the consumer buys new hardware because he cant work the way it is currently and he is going crazy. Then Microsoft makes even more bloated software. The consumer buys the new Microsoft software because of some bell and whistle. And the cycle starts over.

If you go to Microsoft Forums, people are just happy to buy this software and fight with it. It's like a club or something. "Oh boy, I managed to get the <insert name of device here> working. I downloaded this patch, and talked to MS support, and ... blah blah blah.

The operating system is a tool. That's all. The computer is a tool. If a hammer had a handle that's too short or the head is stuck on crooked so that you can't hammer nails in straight, then you throw it away or take it back to the store and get one that's not broken. I realize that software is a lot more complicated than a hammer, I was a programmer for the State of Wyoming for awhile. But at the point that you release the software to the users, then the software should be pretty much complete and working unless the users are on a beta testing program or something and then everybody should understand that means the software is not right yet. 

To those who say, you aren't running the fastest hardware available, I say "Come on, we're talking about the operating system!" If I was running some kind of high powered ray tracing program or something I could understand needing a specially built graphics work station or something. But we are talking about the operating system. The most common denominator part of the computer software system. 

So why did I load Vista on in the first place? I don't know, I guess I forgot how much trouble it was. 

muan
