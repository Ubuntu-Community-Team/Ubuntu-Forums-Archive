### Post by lukeblack on 2007-08-25
Hey guys,

I know there are a lot of these threads, but if anyone is interested..

I used Windows for a long time, and like most of the people here I never really liked it. I'm sure I don't have to go over the reasons!

So I switched to Ubuntu about 8 months ago, and have been using it as my primary distribution since then. However, as a web designer it has been pretty difficult. I was forced to dual-boot into windows to use Adobe Photoshop and Flash when I was working, but that meant that other tasks had to wait until I booted back into Ubuntu (word processing, email, calendar, even browsing). I'm finally giving up Linux now, and buying an iMac so my workflow isn't so disrupted; although it feels like I'm making another compromise, because OS X still isn't my operating system of choice!

Basically, I really hope that sometime in the future running Adobe products on Linux becomes a reality. For the time being, I'll have to depart! Miss you guys :)
