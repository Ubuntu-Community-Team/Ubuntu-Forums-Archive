### Post by Wazeem on 2007-12-22
Well some of you may remember my (rant/giveup/silly) thread. Since then I have constantly been using ubuntu and well I thought I needed to give a more recent testimonial.

After 3 weeks of messing around with linux i finally ironed out all of the issues with compatibility and learning how to use the operating system. Now I am quite confident in the command line terminal.

First i really missed the MS Office 2003 suite but I am warming up to open office day by day (and it is my choice for making documents in now). The only gripe with this is that not everyone has OO so in those situations I am forced to work with MS Office (having to boot into XP). This isn't a problem with ubuntu itself so it doesn't make me think any less of the OS.

The best thing about ubuntu in my view is that it runs so smoothly on my dated machine (only 256 mb of ram) while at the same time running with maximum graphical effects. I suppose many things contribute to this, coded pretty nicely, less spyware (or any) and the biggest of all not having the need to run a memory hogging virus protection suite. Applications load pretty quickly and everything works as it should (software side).

Hardware side of the set up is not so sweet. Since I was pretty new to the OS I had a rough time setting up my wireless card. However thats just the set-up, once past the set-up there is little to complain about.

Feature wise I really like the OS. I love the concept of repos and installing software from a single command in the terminal. I would not have believed this unless I saw it. Being a windows user the concept was too great. This is the main feature I talk about when trying to impress friends about ubuntu.

Conclusion: Ubuntu is a neat OS providing you get past the set-up alive :P. Ok, maybe im exaggerating on the set-up issues. The actually install of the OS was the quickest I have ever done.

I am still learning and still need to learn how to work with Wine, but i'm assuming from this point it only gets better.
