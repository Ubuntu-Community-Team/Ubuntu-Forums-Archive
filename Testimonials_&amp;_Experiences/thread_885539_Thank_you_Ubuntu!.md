### Post by lennartack on 2008-08-10
Hi all!

It's been some years now since I switched from Windows to Ubuntu, so I think it's time to write my Testimonial :).

A few years ago I was a silly Windows user who knew there was someting like linux, made for pure computer geeks. I didn't even know about Ubuntu then. But like everyone I had problems with Windows and I was a mac fan though I couldn't afford one :P. Well I could maybe but I didn't buy one because it was expensive :P.

I don't know how I found out about Ubuntu anymore, but I do know I was surprised how good everyting worked. I mean the applications, not the hardware support ;). And that was still Breezy! I had lots of problems with my hardware, but instead of flaming linux is only for geeks I just geekified myself through the last years. That's the way :D. I even used some other distributions like Gentoo for some time. I actually like recompiling my kernel but it just takes too long :P. So I'm back with Ubuntu for a year or so.

Since the first day I've been using Ubuntu I've read lots of Howto's on this forum and the Ubuntu Wiki (And later Ubuntu Community documentation), and have had a lot of help on this forum and others. Thank you all for that!

After all those days one thing's for sure: I will never use Windows again. Maybe some other free distribution some time, but never Windows.. Unless it becomes free ofcourse. Free, as in free bear.

That's a nice ending :). Thanks again!

Lennart

### Post by Ethyrdude on 2008-08-10
I recently switched my computer from Windows XP to Ubuntu but I'm not new to Linux, my former computer (now my wife's) has Debian on it.  When I built my new computer in the fall, I was thinking (OK, maybe not) that I would use Vista as this was going to be a gaming machine.  I snagged a copy of XP 64 bit just to see if there were going to be any problems running a 64 bit OS, thinking that if I was going to go Vista, I might as well go 64, but best to try first, just in case.

Well, I just don't see any justification for going to Vista.  I also don't see any big need for a 64 bit desktop OS right now either as most of the stuff I had running was under 32 bit emulation.  I do have 3 Gs of Ram and had heard that 32 Bit Windows could act flaky (more than normal) with it.

Long story....making shorter

On a whim, I downloaded the iso for Ubuntu, I was hoping for a business card download, but the whole cd didn't take that long to grab, burn to DVD, ouch, no blanks CDs to be found, and away I went.

This was a nice easy quick install but I did run into a glitch.  I have an ide hdd that I use for back up only and a sata drive which I wanted to use.  I set the sata drive up as my Ubuntu drive and when I rebooted, I was prompted for the CD because my sata install was not recognized.  I could have messed around for a couple of hours with grub to set things straight but I found it much easier just to yank the connectors off the ide drive and reinstall, half an hour later all was good and my ide drive has been reconnected, to mount only when I need to back up data.  (I also do copy this over to cd from time to time but not as religiously as I should, I feel that having duplicate data on two hdds should be good enough, whatever I really can't afford to lose has already been archived many times on CD already.)
