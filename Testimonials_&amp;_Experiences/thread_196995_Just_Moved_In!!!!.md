### Post by Drobbins on 2006-06-15
Hello All, 

I have been a Windows user for as long as i can remember, i got my first computer when windows 3.1 was released then moved to 95 didnt bother with 98 and then jumped right into XP.

Over the past few months the amount of Viruses, Spywear and other junk the has appeared on the winows machine's is getting out of control not to mention the performance decrease of the system after a month's use which usually results in a full format. 

After browsing around the web looking for alternative operating systems i stumbeld across Ubuntu and noticed it was a linux Distro i immediatly thought that it was going to be much harder operating system than windows with lots of command line and hard to remember commands, how wrong i was. I downloaded the live CD played around for half hour or so and installed. 

The first thing that i noticed was the amount of time it took to install, i could have popped to the shop and back and it was done this was great once it was done i set about looking at the networking settings to get this beast online and to my amazement everything was already configured, how was this possible setup never asked me for such information good ol' ubuntu so next was to get some programmes so i looked around the forums here and found Automatrix which i installed and also the package manager in Ubuntu none of these required much of the hard command i origionally thought came with linux. 

As a round up of my initial thoughts are **great** i dont think i will move back to Windows in a great hurry Ubuntu is a great platform with a great community. 

/Drobbins
