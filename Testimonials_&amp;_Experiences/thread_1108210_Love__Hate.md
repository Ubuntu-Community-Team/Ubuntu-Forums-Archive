### Post by Methuselah on 2009-04-05
If you want to have a peaceful ubuntu experience I'd recommend you research the hardware you buy.
Better yet, you can buy a machine with Ubuntu instead of windows pre-installed.

Some hardware manus just don't make drivers for linux or release enough information for drivers to be written by others.
Due to windows current market dominance even the most obscure hardware will provide at least windows only drivers.
This might change as realities change but for now it's a sad but true fact.

For example, I find that logitech's recent webcams have become quite linux friendly. 
So when I need a webcam I vote with my dollars and buy a logitech that I know will work.
I don't buy something unknown or unsupported and then get in a huff when it doesn't work.

Internal phone modems are really problematic because the hardware is really stripped down.
Most of the functionality is actually executed on the CPU by the drivers and these are usually provided for windows only.
Check this site to see if it'll be of any help: [http://www.linuxant.com/drivers/](http://www.linuxant.com/drivers/) 
It depends on what kind of internal modem you have.

Anyway, I like Ubuntu and I am interested in using it so I support manufacturers who'll support my aims.
However, if your priority is that you want to be able to buy any PC hardware out there and have it work with the OS you're using, 
you can save yourself the wait for Jaunty and install 32 bit windows right now.
