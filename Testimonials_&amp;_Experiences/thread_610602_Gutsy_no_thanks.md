### Post by Junihausen on 2007-11-12
Sorry, I have to say that, but if you have a running feisty install - stay. If you're a noob and have a running windows system - don't install ubuntu.

I tried Ubuntu the first time with feisty and it took me weeks to find replacements for my everyday apps, which where as satisfying as the ones I knew, and I had more application crashs and more often a frozen system than on windows. Things like apps which won't start after once crashed until you delete the settings folder - simple didn't know such things under windows, but under ubuntu there seems to be no P2P software where this not happens once.

But I had to find out they can do more - lot's more.
When I switched to Gutsy I had to find out, that apps which ran perfectly well on feisty suddenly had bugs under gutsy. Why do they put buggy versions of miro media player and azureus bittorrent client in their repositories, while there are functioning versions in the repositories of the issuers of these programs???
Oh and fileroller the archive-manager produced damaged archives more than once, and could at least give a hint, how long the packing will take.

And once I accidently installed a GDM theme with a broken PNG and when Ubuntu couldn't start GDM on startup the only option it gave was a "shall I retry?" - so you get stuck in a loop. So safeboot and editing the config file with vi - that's nice for a windows noob.

Sorry, but I spent more time with broken apps and system crashs on ubuntu than I ever did on windows and that's not the way it should be if you want people to switch to ubuntu.
