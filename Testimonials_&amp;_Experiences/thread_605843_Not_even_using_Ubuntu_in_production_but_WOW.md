### Post by Blue Sassley on 2007-11-07
WOW how I love Ubuntu! I don't even use it very much at all... not even on my production computer but WOW do I love it! I started using Ubuntu about 3 weeks ago when I setup a Myth TV backend server on Ubuntu 7.04 on one of my old computers I almost never use. Everything worked perfectly... all drivers for the hardware just worked didn't have to do anything special at all, installing MythTV was a breeze and works better then what people are saying with my current hardware configuration. But then I deiced to try a Windoze based PVR solution on the same box to test some features it had... that was my mistake! First off Windows doesn't have any of the drivers for 40% of the hardware in my system, which I laugh really hard at because its all "Microsoft tested and approved hardware" . So the first thing I do is piss away 4 hours hunting down drivers and trying to install them... finding out that almost all of them don't even support the version of Windows I'm trying to use (Windows 2000 Pro) Second thing after getting most of the important drivers working I install the PVR software which just crashes and doesn't seem to work well as its based on Microsoft .NET framework (which I think is crap) As I sit here and think about all the time in my short life that I have lived (as I am only 24) and think of all the time I've spent training on Microsoft products (getting my MCP in server administration and others) I just think about wanting Ubuntu even more. I probably have learned more about Linux in this 3 weeks then I have ever learned in the past because it seems to be so easy to find the answers I'm looking for because of this GREAT Community.

 Anyway I really want to thank all the people that are making this GREAT OS possible and I hope that one day it will be able to take a large share of the OS market place and give Microsoft a huge run for there money.

Thanks for reading,
Bradley
