### Post by SonicSteve on 2006-07-15
Hi folks,

After some pain and tinkering my Linux box is now setup!! Most of the pain is rooted in the fact that I know very little about the linux/unix command prompt, file system, well everything. It set it up on a dual p3 733 msi 694 board, 320mb ram,32mb nvida440mx, 1 10gb hdd and 1 6.4gb hdd. It's running very impressively, not that hardware is impressive but Ubuntu is running well regardless.

It's not setup perfectly, but I have my samba shares, my mp3 files playing in the background mounted as a permanent mount. I think that this box may become my working pc and the family will get the Windows XP pro pc. The next step is to use the credentials file to hide my usename and password. 

I've learned so much about the command prompt already that I'm already starting to feel alot more at ease. I find it scary to not know the nuts and bolts behind my OS. It's taken me some time to get used to things but, it's pretty good and all indicators seem to point to ubuntu getting even better.

I love the fact that I won't have to put out over $100 to pick up another copy of windows. Yeah!!=D> =D>
