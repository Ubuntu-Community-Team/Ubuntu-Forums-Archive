### Post by 3rdalbum on 2011-09-20
If she doesn't like Unity, it's ridiculously easy to switch to Classic Gnome in Ubuntu 11.04 - it's just a popup menu away on the login screen.

If you want the latest and greatest, Ubuntu 11.10 will be out officially next month. There's still a few annoying Unity bugs that haven't been sorted out yet, but you can install Gnome Fallback on 11.10 to get something very similar to the old Gnome interface. Or use Unity 2D which is preinstalled and doesn't seem as buggy :-)
