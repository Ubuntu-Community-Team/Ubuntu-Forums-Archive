### Post by elebrin on 2005-11-07
well, a lot of others are posting their initial experences here and it looks like fun.

I have used Windows since win95.  I still use it at work and for Guild Wars.

I installed redhat 8, and got it working (2.4 series kernel) but gnome was slow as hell and KDE just sucked.  I installed redhat 9 (can't remeber the kernel version...) and gnome was better.  I installed XFCE and set up a really cool desktop envirnment.  I quickly became an XFCE diehard and I am very happy that they are adding it as a full distrobution.  I have had it installed since before it was available in the repo's (I did it from source).  I also used fedora core 1 and 2, but when I bought a tv tuner it never worked correctly with either.  All this was over a period of 2ish years.

After warty came out (it seems like quite a long time ago) I installed it and experenced gnome in a good configuration.  I also experenced my tv tuner working for the first time under linux (redhat, even with bttv and tvtime set up correctly, refused to let sound work correctly.  I ALWAYS had sound issues with redhat).  I was thrilled... I no longer had to boot to windows.  I remember one point typing 'uptime' and seeing four weeks! It isn't just that I could leave the computer up this long, but that I could do everything I needed to with linux and not need to reboot to windows.

I have installed each new fresh, with a partition at /home I can install and have a system working within two hours.  With Breezy the base install took 17 minutes (!!!) and the first big update wasn't bad.  Installing all the extras I use takes about an hour over my connection, and the rest goes for any new configuration I need to do.  

I have since moved from XFCE to FVWM (yep, I'm moving backwards with window managers, but hey, I like a simple desktop).  I can play JKII and Jedi Academy perfectly under wine.  I use gedit/jgrasp for programming.  My computer can play games, watch TV, watch any format of video, listen to any format of music, compile c/c++/java and run bash/perl/python scripts.  I have set up an apache server with little effort. I use remote access with SSH to access data without having to carry floppy disks or CDs. Under windows, I would not be able to do all of this.

I could not be happier with this distrobution, it has enabled be to do more with my computer.
