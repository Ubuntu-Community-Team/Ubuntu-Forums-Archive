Check the bios and see if I can point it at the other partition. How do I get into the bios? Can't be that hard.

There are several sites which explain how to configure GRUB to install, upgrade or repair, from disk to disk, assuming you already have a working linux install. [FAIL] With each passing hour of this battle I got more frustrated and less carefull. Redhat has instructions how to do it but it only works with Fedora. Hundreds of sites have helpful advice on how to proceed, all of which begins 'Open an Xterm'. Bit hard to do in XP. There are instructions how to set up server for a network install. First you config your Other linux machine like this... Hmmm. Seems you still need some other boot device. No floppy on this baby. No mem stick. No cdrom. Try to drive GRUB again, manually. Slightly better results, but still no install. Ok, let's go back to XP and look up more stuff. Reboot and - no menu. INSERT SYSTEM DISK.

If I shut down now, is there a GRUB there or not? How would I find out from within XP? No way of knowing.

"No operating system found. Insert system disk." Try to find GRUB, [FAIL] It isn't there. Nor is NTLDR. This is great. I've just bricked my friends perfectly functional laptop.
