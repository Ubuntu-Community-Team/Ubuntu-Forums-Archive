### Post by zero244 on 2007-10-19
I did a semi fresh install of Gutsy on a test machine that has a separate partition for the home folder.
Gutsy is a slicker install, most everything is a few clicks away. It pretty much caught all my hardware. A little tweaking here and there.
The problem is Im not sure I want to convert this machine which is running Edgy + XGL+ Beryl.
Everything is running perfectly.
Even though my home folder is intact on the test machine it is going to take some tweaking and program installations to get things back they way they were.
Gutsy is a improvement over Feisty but I might wait till I have to do a fresh install rather than upgrade for a few extra features.
I will go ahead and finish the tweaking on the test machine and continue to use Gutsy on it.
But this machine my main unit.....which is running Edgy...........I think I will leave it alone for right now.
For New users I would recommend Gutsy, it does simplify some things and the repositories seem to be functioning well.
If your upgrading expect some tweak time to get things the way you had them.
