### Post by gslug79 on 2008-05-02
I have used Ubuntu on my desktop box since Feisty, but have only now got round to joining the forum. Apart from one unexplained incident where I lost the Gnome panels on upgrading to Gusty, everything has been fine. The desktop was painfully slow with XP, but Ubuntu made it usable once more.

My original intention was to use the desktop as mainly a file server and keep XP on the laptop, but I was so impressed with Ubuntu that I switched that too. On acquiring an EEE, I kept Xandros for a few weeks, but couldn't resist putting Xubuntu on that too. Finally the file server. I built this in January, initially running Kubuntu Gusty (It is attached to the TV for sofa surfing), but just couldn't bring myself to like KDE so switched to Ubuntu. As I have an all Linux set-up I use NFS for the shares and sync the EEE to a directory on the other boxes using Unison.

I have now upgraded 3 of the 4 rigs to Hardy an apart from a few Firefox issues, I remain impressed.

Thanks to Canonical for a great OS – one that is rapidly turning me into a Geek!:wink:
