### Post by isee on 2009-11-20
Hi all

Just thought I would report on a Gateway laptop upgrade.

The install process froze at the end of Getting New Packages.  Waited 40 mins then canceled.  The Update Manager icon was in my top menu bar, saying 600 and some odd upgrades available.  Restarted Update Manager and clicked on the Upgrade icon and the process went through to completion.

Now my bitch.  And let me walk through this.  Rebooted the computer, and my themes and background changed.  Some ugly grey theme.  Also my System dropdown icon disappeared.  Basically the menu system changed.  Not so really bad but I wasn't prepared for that.

I was really upset about the grey theme.  Like just because I upgraded I was now forced to spend time to change MY theme back to what it was before.  I don't know if I didn't read the fine print, but I don't think I would have consented to having my user settings changed.  And with the menu system change, I was thinking No No, I'm just going to reinstall Jaunty.

So I rebooted, and the menu system changed back to what I am familiar with.  And the theme, although it changed from that grey, is now brown, not what I had originally.

I should mention that the original post-MS OS install on this computer was Xubuntu, but because my sound wouldn't work I changed to the Ubuntu desktop through Synaptic.

In the next upgrade, don't change the appearance of my desktop.  So thats what I have to say.
