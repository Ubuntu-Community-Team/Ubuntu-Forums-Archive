### Post by SingPraises on 2008-07-01
I just started with Ubuntu Hardy a few days ago.  It was mostly the price and security problems of Windows XP that made me want to switch. I love the whole Linux community and open source philosophy.  I chose Ubuntu because it gave the option of installing and uninstalling Linux like any other windows program so I could try it out.  
     I'm amazed how easily Ubuntu installed.  It didn't take much effort to get everything working.  I haven't noticed any hardware issues, though this laptop isn't attached to a printer, nor does it contain any fancy graphics or sound cards.  My external hard drive was recognized immediately. I'm amazed how quickly I can use the internet.  My e-mail loads so much faster than it ever did in Windows.  As a digital scrap-booker, my e-mail tends to be very graphics heavy. Firefox used to be very slow, despite having a fast internet connection. 
     The only thing keeping me from using Linux 100% is that Adobe Photoshop Elements 4 doesn't work with Wine.  If I can get Gimp to work for my needs, I'll be a full time Linux user.  Linux makes me want to learn more about computers so I can someday contribute to this community. I hope more people will realize that Linux has evolved to the point where you don't need advanced computer skills to use it.

### Post by Geekkit on 2008-07-02
> **hyper_ch said:**
> for adobe photoshop elements... you could run it in a virtual windows (vmware / vbox).

This is a good answer. A version of VMWare for Linux is free and quite easy to set up and get working. I've done this before and as long as you're not running games, you have what you need. Plus, windows has been put in its place: inside of a safe little sandbox where it can't do anything stupid.

To the OP: About the only couple of things that I found that I can't do in GIMP that Photoshop does for me are:

- Pantone Colors
- HDR
- Layer effects

For Pantone colors, I don't use them but if I did I would be stuck (don't know if this will ever be resolved since the Pantone people will not likely allow offer FOSS solutions). For HDR, I actually use qtpfsGUI which is actually a KDE app that does HDR photo manipulation including tone mapping effects which are quite nice. The layer effects can be taken care of if you have the patience to follow tutorials and make use of some of the filter effects in GIMP. I'm sure the next version of GIMP will offer layer effects. Actually, I just looked at the newest (unstable) release of GIMP (2.5) and it now incorporates (finally!!) Generic Graphics Library (GEGL) which will make it much easier to offer the layer effects via "non-destructive editing". Watch for new versions of GIMP.
