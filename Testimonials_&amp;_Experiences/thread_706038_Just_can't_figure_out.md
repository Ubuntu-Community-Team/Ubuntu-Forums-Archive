### Post by zolookas on 2008-02-24
You should try [k3b]("http://k3b.plainblack.com/screenshots"), it's the best burning choice in linux unless you are burning hd-dvd or blu-ray discs. It's a bit hard if you are used to wizards. Easier to use would be [brasero]("http://www.gnome.org/projects/brasero/screenshots.html") (it will also be included in ubuntu 8.04 by default). If you are burning HD-DVD or blu-ray discs, you will have to use [nero for linux ]("http://www.nero.com/eng/linux3.html")(it costs $25). 
You can find guides how to install photoshop 7/CS/CS2 (CS3 is not supported yet) in ubuntu. Like this: [http://news.softpedia.com/news/Install-Photoshop-CS2-on-Your-Ubuntu-PC-77260.shtml](http://news.softpedia.com/news/Install-Photoshop-CS2-on-Your-Ubuntu-PC-77260.shtml)

### Post by steveneddy on 2008-02-24
When I came to Linux, I was using PhotoShop a LOT for big and small jobs, and once I learned to use the GIMP, I found that GIMP did everything I needed.

I'm not a professional photo editor or layout artist, but it works well.

I came to Linux for the adventure and stayed because of the reliability, stability and dependability.

Linux is getting better, but if it is going to acquire a larger user base is to gave a great looking GUI out of the box. I don't think that Compiz-Fusion needs to be enabled, but the default window manager needs a look with reflection and color that will "impress" a new user.

People only like shiny stuff. Someone will buy a POS car if it is shiny. Even if you tell them about the unreliability and other issues that make it unreasonable to purchase. 

But the dirty and a little dented auto with proven reliability will be overlooked.

Make it pretty, they will come.

I would prefer that **they** stay away so that development focuses on reliability and stability instead of aesthetics and instability.
