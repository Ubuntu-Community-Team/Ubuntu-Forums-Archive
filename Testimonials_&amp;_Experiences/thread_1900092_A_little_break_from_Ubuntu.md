### Post by Myrddin Emrys on 2011-12-25
> **t-mark said:**
> I really hope that thing will be thrown out (or more adjustable) in next version and I am waiting for it, because I was very satisfied with Ubuntu for several years and I hope that affection will continue. 


The 'side starter bar' (Unity) will be used in the next version as well, but it's only the default desktop, and very easy to change. If you're otherwise very satisfied with Ubuntu, it would be worth checking out the alternatives. Three popular desktops - Xfce, LXDE and KDE - can each be installed with a single command:

[FONT=Courier New]sudo apt-get install xfce4
sudo apt-get install lxde
sudo apt-get install kde-full[/FONT]

You can then select which you want to use in the login settings. You might also be interested in the new Mate desktop, a third party project to continue work on the Gnome 2 desktop you are already familiar with:

[http://mate-desktop.org/](http://mate-desktop.org/)

It's nearly as easy to install as the others, though it's not currently an official part of Ubuntu (so use at your own risk!):

[http://wiki.mate-desktop.org/doku.php/download](http://wiki.mate-desktop.org/doku.php/download)
