### Post by rev_b on 2006-11-17
I always wanted to run a Linux based system, but I never found a distro that worked so well as (K)Ubuntu.

I find the concept of open source software very apealing, it's very nice to see what people of a comunity can do for each other without just thinking about business and profit and so on. In that point of view, the quality and support of Ubuntu is nothing short of amazing.

IMHO, it beats windows hands down. I now only use Ubuntu as my OS. I can do everything I did in windows, some times even better. I already used openoffice, Opera and Gimp in windows, so the transition was more than smooth. The installation is incredibly fast, and using a live CD is very useful - you can get a preview what works or not in your instalation. To my surprise everything worked, even the network was properly configured, my scanner worked just fine! - two things Vista RC1 couldn't make out of the box!!!! instalation is almost foolproof, and very, very fast. It  even detected the Vista loader on another HDD and configured grub  alowing me to boot to another hard drive, if needed!

Of course some searching is needed before everything works 100%. About the codecs, not a big deal, you have to install some in windows to watch some multimedia files, don't you? Graphic card configuration is rather simple, but ubuntu (and other linux distros) badly need a graphical interface to install and configure video drivers. The average "human being" doesn't know how to boot into a command line, install drivers and edit your xorg.conf file in order to get OpenGl and dual monitor working. I did bother to learn, but for most people this is a major drawback in configuring Linux.

Linux has an excelent memory management, it really takes advantage of my 2 Gb of Ram. It never goes to the swapfile. In windows, you can have all the RAM in the world, if you don't disable swap (and some programs don't run without it) it always uses it to some extent, slowing things down.

Anyway, after installing the system, it takes about 20 minutes to add those few things missing... just impressive. You can download tons of excelent free quality software with synaptic/apt-get. Openoffice 2.0.x is fantastic, and I can do everything I did in MS office, sometimes even better - I prefer Calc to Excel, believe it or not. The system boots to a gnome desktop in 30 seconds!! In the same PC, Windows XP takes twice the time! 

Games? Not a problem. Not much time to play anyway, but I must say I was surprised to see how much quality games have native linux versions. I have installed Quake 3, Doom 3, Quake 4, UT2004, Serious Sam TSE, MOHAA, America's army, and of course, Tux racer :mrgreen:

To my surprise, after codec's installed, everything about multimedia is MUCH smoother and straightforward than in windows. With 2 monitors, I get video aceleration in both, and you can't do that with windows!!! DVD plays wonderfully, every video format works nicely with Totem, I can use my Pinnacle TV card, and I can see TV and a DVD movie at the same time in each screen!! - try to do that with the mighty windows - I can move mp3 to my Creative Muvo, I can use my Canon DV camera with a firewire conection     , make and edit videos, and everything faster and more stable than in windows. 

So, despite I like and share the ideology behind Linux and open source software, I ended up using exclusively Ubuntu because it is just plain BETTER than windows. It's faster, just as stable (since windows 2000 stability isn't an issue anymore, let's face it), much more foolproof, it has a ton of high quality apps that do even better than comercial software - you need a thing like partition magic to do something more complicated with you partitions in windows. With Ubuntu, just sudo apt-get gparted, and there you have a powerful partition manager. And so on, and so on.

The quality and stability of the system and produtivity software makes me wonder about the need of companies paying huge money for licencing operating systems and office suites that add nothing or very few in features to completly free software.

So use Ubuntu. It's really that good. Share it and let people know about it and try it. Many will stick with it - I did e-mail all my contacts with a short note and a link to [www.ubuntu.com](www.ubuntu.com) - talk about spam...:D

Once more, I just want to thank everyone behind Ubuntu development and release. Great job.
