[FONT=Bookman Old Style, serif][SIZE=1]   Before giving my glowing evaluation of Ubuntu, I need to quickly provide a little background information.
[/SIZE][/FONT]
[FONT=Bookman Old Style, serif][SIZE=1]   I switched to Linux a year and half ago.  I was so fed up with Microsoft I decided to try out Linux.  First I tried Debian[/SIZE][/FONT][FONT=Bookman Old Style, serif][SIZE=1], but it was too difficult to use (I am almost computer illiterate). Then I tried[/SIZE][/FONT][FONT=Bookman Old Style, serif][SIZE=1] Mandrava, but found it to be a little lacking.  Finally I tried Ubuntu . I have been  using  it ever since and love it! 
[/SIZE][/FONT]
[FONT=Bookman Old Style, serif][SIZE=1]   I have only had one problem using the system, but it is not a fault with Ubuntu. The problem is with Open Office. Trying to type Chinese is difficult and I have not yet figured out if it is possible to use a vertical layout.  Other than using Chinese, Open Office is superior to Microsoft Word in almost every way. The only problem I have had with Ubuntu itself  was when trying to install 9.10 with a c.d. It did not work, so I had to spend two days upgrading from 8.04 over the internet! Thanks to a post on one of Ubuntu's fora I believe the c.d. problem is solved if I need to re-format in the future.[/SIZE][/FONT]
 [FONT=Bookman Old Style, serif][SIZE=1]   Ubuntu is a wonderful system to use. It is superior to Microsoft in every aspect and better than the other Linux systems I briefly tried. It is user-friendly, even for hopeless computer users like me, versatile and convenient to install. Microsoft requires four c.d.s to install what is on one Ubuntu disc and still does not have any hardware drivers! I do not understand why anyone uses Microsoft. Ubuntu's ability to display hundreds of languages is fantastic. If the ability to use multiple languages could be improved Ubuntu would be perfect.
[/SIZE][/FONT]

### Post by XubuRoxMySox on 2010-02-11
Here's another li'l trick if a LiveCD fails that would save you some time upgrading...

Use the *minimal* CD, *then* upgrade to the current version. That way you're only upgrading the system - and from there you can use Synaptic to install "Ubuntu-desktop" or whatever else. One minimal LiveCD - even from an older version - is all you ever need to install a fully up-to-date system!

-Robin

### Post by davidryderuk on 2010-02-11
>  The problem is with Open Office. Trying to type Chinese is difficult and I have not yet figured out if it is possible to use a vertical layout.

1. Go to Tools > Options

2. In the window that appears go to Language options > Languages

3. Click on "Enabled for Asian languages"

4. Close the window, then go to Format > Page

5. In the window that appears change the Text direction to "Right to left (vertical)"

Hope this helps.

Thanks for the post.
