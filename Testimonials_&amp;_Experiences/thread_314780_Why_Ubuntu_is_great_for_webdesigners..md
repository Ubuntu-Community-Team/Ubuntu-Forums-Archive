GIMP and Bluefish are more than adequate for most web designers if you run KDE rather than Gnome Quanta+ is a decent alternative to Bluefish. VIM is an awesome editor if you want a minimalist approach. I would also suggest html/xml-lint for validation checking of your code in Bluefish. We have come to this point where the non free flash-plugin is a problem for free software as lots of sites are relying on it to transmit media. We have the [GNASH]("http://www.gnu.org/software/gnash/") Player which to be fair is coming on nicely, but has a long way to go before it can be an effective replacement.

I don't mean any dis-respect to any of you and I'm glad you see Ubuntu as a better Desktop. Macromedia flash programming pays me my wages so I can work on open source ;-), but at home I'll stick to apache, php, and bluefish.
