### Post by Little Bit on 2009-07-24
My disappearing movie player was a video driver issue (Intel - notorious, according to stuff I have read here), but I fixed it!! From just reading about "xorg" here on the forums.

I've already discovered how **the command line is a powerful secret weapon.** Two single lines of text, then omygosh! Scrolling text (faster than I could read), hard drive whirring and spinning, my computer obediently busy making all the changes it was supposed to (it wasn't using the Intel driver by default - now it does). Reboot, and bingo!

I never would have dared use the command line in Vista (does Vista even *have* a CLI?). But in Ubuntu the computer doesn't scare me with "*are you really sure you want to do this*" messages that make me wonder if I'm about to cause a catastrophic collapse of the space-time continuum, lol. 

Now I've done something really geeky! I never knew I had it in me, lol!

More a fangirl now than ever,
