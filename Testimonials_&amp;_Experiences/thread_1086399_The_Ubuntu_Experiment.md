### Post by MarblePanther on 2009-03-04
My first suggestion is upgrade that RAM (if the mobo supports it) to as much as it will take.
EDIT:Nevermind, as you mentioned above ^^
If you can't, then you are going to have to use the alternate install disc (which uses a text-based installer: [http://www.ubuntu.com/getubuntu/downloadmirrors#alternate](http://www.ubuntu.com/getubuntu/downloadmirrors#alternate)
because the graphical installer on the regular livecd needs more RAM then you have to even load up.

I would suggest using Openbox as the desktop environment for that computer (If Xubuntu is too slow for it).
