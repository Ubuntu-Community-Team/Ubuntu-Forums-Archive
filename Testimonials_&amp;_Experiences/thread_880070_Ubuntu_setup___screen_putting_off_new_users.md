### Post by 3rdalbum on 2008-08-05
Ubuntu's installer could use some love. There were two problems with the installer originally, when resizing partitions:

1. The slider for choosing the amount by which to resize, didn't say whether you were selecting the new size for Ubuntu, or the new size for Windows. This has been fixed.

2. The user must choose mount points - what each new partition is (whether it's /, /home, swap, whatever). It was months after I switched to Ubuntu before I knew what a mount point is, and I don't believe there's even a textual description of what /, /home, and swap are, or whether all of them are needed, in the installer.

I would suggest allowing the user to choose between two partition layouts, like so:

  O  2 partitions: OS+user (/) and swap - Easiest
  O  3 partitions: OS partition (/), user data (/home), and swap - Flexible
  O  Set partitions and mount points manually - Advanced

Upon choosing one of the first two options, a screen similar to the "resize" screen appears. You can drag the borders of each of the partitions to choose the sizes of each of them. They are already labelled with "OS+user (/)", "swap", "OS (/)", "User (/home)" as necessary.  The defaults would be fairly sane.

The last option goes to the current manual partitioning screen.
