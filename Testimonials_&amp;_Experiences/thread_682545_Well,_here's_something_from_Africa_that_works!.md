### Post by johann mahne on 2008-01-30
...and it works as well as windows.
I had no worries with any of my hardware!

I have a dual boot with windows 2000.

The two biggest issues for me as a newbie starting up was:
Moving from DOS commands and navigation to linux ones.
Moving from understanding windows system files to Linux ones.

 I feel that the next version of UBUNTU should concentrate on these two issues as soon as the CD auto install starts.
  I got stuck with the dual boot partitions because I did not understand that Linux uses different file systems for 
it's partitions.
  I also struggled to edit files, because I was unsure of navigation and edit commands.

 Even a person installing windows needs to be able to perform administrative duties,so I do not think that Ubuntu is that complicated to install, it's only very different.

I recommend these websites for Newbies: freeOS and Ubuntu forum sites, as well as computer hope.
