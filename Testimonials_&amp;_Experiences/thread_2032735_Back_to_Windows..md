### Post by flemur13013 on 2012-07-24
For many years I programmed scientific/engineering/graphics stuff under UNIX, which had poor-looking GUIs (X/Motif) and crummy and massively disorganized fonts.  Those "features" were retained in Linux.  

Although the basic OS design is superior to Windows, Linux sucks. Linux applications suck -  buggy, with ugly inefficient GUIs and terrible documentation - and the port of Unix from Sun (etc) platforms to PCs to create Linux was poorly done.  

Every update to 12.04 produces obnoxious errors that are a PITA to fix, if they're fixable.  My last ubuntu update, since I won't be using linux anymore, completely screwed the icons (missing or wrong) and broke all the programs which control "themes".  My last update to Arch linux broke X.     Great stuff!
