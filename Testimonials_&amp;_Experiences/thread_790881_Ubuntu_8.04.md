### Post by Rucas on 2008-05-11
Well i have read a load more negative posts than positive ones here about the latest Ubuntu, i for one will be posting a good one...
No it was not all trouble free and actually initially i run a Network Upgrade on my laptop wich made 8.04 very crappy. A bit of backing up and a fresh install later, i must say that Ubuntu 8.04 is running pretty well for me. Im using it on a Toshiba Satelite A30 2.6ghz Intel Celleron (crappy, but oh well), shared 512Meg Ram (shared with the normal Intel graphic chips).
Prior to doing a fresh install, i must say that Ubuntu 8.04 failled even to start up at times, and some weird issues occured. Now its running with no problems.
Overall i have not experienced any major issues with it. So thank you for the Ubuntu!!! I really appreciate all the work and taught you guys put into this!!!
