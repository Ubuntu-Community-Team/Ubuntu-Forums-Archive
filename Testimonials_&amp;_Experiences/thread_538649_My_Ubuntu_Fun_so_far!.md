### Post by atrapine on 2007-08-30
Well I am not a noob to the Linux world been using it off an on sinse 97, And have been almost exclusively a KDE user apart from one dabble with Gnome 1.4
My Distro's I have used most have been Redhat,SuSe,Mandrake and for the most part have had fun using them.
Last time i tried Ubuntu was dapper, And I was less than impressed it wasn't Really bad just felt unfinished? Well after getting a bit board with SuSe 10.2 I decided to bite the bullet an see if much had changed with Feisty?
Well i am more than impressed it is very well done. Everything is a breeze to configure for the first time user. It is a big credit to the hard work people have been putting into this OS. And i can see why it it is getting so much press time at present.
One small last thing that Gnome needs is a really good Widget engine. If this last thing could be addressed in later releases I would say Ubuntu would  Be giving MS something to really worry about.  lol.
