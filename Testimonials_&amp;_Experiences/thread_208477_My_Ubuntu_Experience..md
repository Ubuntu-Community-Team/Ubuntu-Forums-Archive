### Post by FenrisAbraxas on 2006-07-03
Well first of all i've been a SuSE user for a couple of year until Gentoo installed in my main computer :P.

Gentoo was a pain in the arse to install BUT it was a great learning experience i managed to learn all the location of important configuration files, lerned to use lspci the right way, chroot and so many other things.

After that a friend of mine sold me an old laptop because it was to slow to run XP (it's a portege 7200 with no floppy, no cdrom) and i told him that comp could be usefull so i got the laptop with no removable media.

Following some guides testing grub for NT and learning even more about the flexibility of Linux and open source apps, i managed to boot up the ubuntu installer.

The installation went smooth and clean, detected all the hardware but it used to have a terrible performance and i was like "hmmm what's wrong, IceWM shouldnt be this slow".

Used to Gentoo i went to a console and do this:
```
sudo -s -H
vi /etc/rc.conf

```
To change from gdm to xdm but... 8-[ OMG!! rc.conf wasn't there! :confused: :confused:

Then i rememberd that i didn't compiled my kernel and got a x86 generic one. So i recompiled my kernel tweaking the CPU settings. 

Perfect, got better performance but there are lot's of unused processes loading that i don't need all the time like CUPS or anacron so found a great post in this forums about that, got better performance and more knowledge of my system.

Now i have a fairly good Jurassic Comp just to show off the power of the penguin! And i can say that Ubuntu can be VERY easy to install but to get a nice performance you have a lot of tweaking to do.

Just to note this, in a high end computer you probably won't notice the slowdown of a generic 686 kernel without proper personalisation. But do you really notice it in old hardware, where Linux is pretty good at :).

I keep with Gentoo as my #1 Distro but Ubuntu isn't bad at all in fact took the 2nd place in my GNU/Linux Distros.

I'll finish saying that Ubuntu is going the right way, great job and keep up the good work!.

Greetings from México!
