### Post by txrambler on 2007-08-26
I installed Ubuntu 7.04 on two separate machines this weekend. The first was a PIII, 512MB RAM, 40GB HD. I was surprised at how smooth and seamless it was. I was also surprised that the two (easily resolved) problems I had had with 6.06 were nowhere to be seen -- namely, NIC drivers and audio codecs for playback (the latter problem, btw, solved with easyubuntu...something newbies might want to check out!). I noticed it was quick and painless and decided on the other machine to do a timed install.

The second machine was a Dell GX240: P4 1.7Ghz, 512mb RAM, 20GB HD. I booted to the cd, and 28 minutes after the machine was turned on was surfing the Internet with zero problems. A certain Redmond software company will take more time than that just to format the hard drive!

Once again, the Dell install was seamless, all drivers are installed correctly. One of the biggest critiques I have heard mentioned of Linux was that it was not an operating system "Grandma could use". While there's certainly a learning curve to Linux, I would say that there is MUCH LESS of one in using Ubuntu 7.04 than in using Redmond's latest offering.

I finally have an O/S I can feel comfortable using on builds. And since I am working to try to get computers to underprivileged children, saving money and system resources on the operating system is ALWAYS a good thing!

### Post by freebeer on 2007-08-26
Your experience echos mine.  I did a dual boot install of both Win2k and 7.04 recently.  It took me roughly 6 hours to install Win2k (includes installation disks for video, etc. but not printer).  The basic Win2k installation was about 2 hours to format the disk (of course it had to format the whole thing - like nothing else would exist on the machine) and get into a running position.  The rest of the time was updating the machine to apply all the fixes, rebooting between fixes, getting more fixes, etc.  Note that I didn't install any printer drivers or any application software yet (MS Office, etc.).

The 7.04 install took about 30 minutes including having to resize the NTFS partition, repartition the drive for Ubuntu, install the OS, and a single reboot.  At this point I had a fully working installation that recognized and worked with all my hardware and I had several applications (OpenOffice, etc.) already installed.  And it recognized my network printer.  5? more minutes to install all the upgrades that Synaptic said were available and I was done.
