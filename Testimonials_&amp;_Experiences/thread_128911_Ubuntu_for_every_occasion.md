### Post by Minyaliel on 2006-02-12
Wow. 

I know I've already posted here a couple of times before, but wow. I just installed xubuntu. I've always been curious about the possibilities offered in Ubuntu when it comes to diferent desktops. Until five minutes ago, I had something like four different desktop systems on my laptop (!!) just to test them. My latest addition was xfce, and well, I'd say I've tried all the major things now. I've been comparing them now, almost unconciously, as I've been using them and have come to the conclusion that... well, simply put that my Ubuntu will never start boring me again *laughs*. Xubuntu is so fast and simple that it is perfect for school or just quickly opening up a document to edit it because it is so fast, it takes no time at all to boot it and it still looks great. I've just removed my Gnome, which I hadn't used since December, but there are some apps for that desktop that are great, and it isn't as slow as KDE, although it doesn't look as great as xfce either, in my opinion, however much you configure it, and it is considerably slower on my puter than xfce. KDE looks great and behaves great (it even overcame the multimedia troubles I experienced in Gnome) but takes ages to load... But again, I wouldn't survive without amaroK, the one program I use non- stop through the day (well, almost). So, KDE's good for sessions when you've got time to wait, like during weekend boredom etc., or for showing off ;) It never fails to impress Windows users... I love Ubuntu. It's the best thing I've ever accidentally stumbled across (I did!) and I'll never quit using it. And now that I know I can have it look any way I want, determine boot speed and even choose what Ubuntu flavour I want determined on occasion.... oh yeah.
