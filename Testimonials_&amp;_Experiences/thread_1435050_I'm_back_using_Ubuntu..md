### Post by apple+ on 2010-03-21
Hey all,

I used Ubuntu for the first time,I would say about last July,getting my first decent pc,finding Windows xp wasn't great with it and having a friend who uses Ubuntu and linux install in on my computer.After maybe 3 months of that I went to Mac,which has been the best experience for a OS and I believe OS X is the best,till today,when I was looking for a free virtual machine for running windows 7 pro eventually on my Mac for games and ran across Virtual Box,I have used Virtual Box before,being introduced to it by my friend who got me to use ubuntu in the first place.Ok,now on to today,I was digging around my software drawer looking for a copy of Windows xp I could stick on it for the time being and ran into my old ubuntu CD's,including Ubuntu 9.04 and I decided to load it up and boy,did I miss Ubuntu!!I now remember what got me away from windows to eventually use Mac and now that I got Ubuntu in a VM I'm really glad I put it on.I now am considering doing a full partition/install when 10.04 comes fully out so I don't have to format my HD to upgrade.Sorry for the long story,I'm just excited to be using Ubuntu and and happy I remembered my password for this forum. ;)

Thanks for reading. :D
