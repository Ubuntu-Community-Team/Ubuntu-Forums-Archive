### Post by WaeV on 2008-11-18
Well, I got arch installed and have certainly learned a lot.  I tried XFCE to start, but it wasn't for me.  One of my biggest surprises was when I ran the command "startx" instead of "xinit /usr/bin/gnome-session" to start gnome, and encountered the vanilla X window system.  It was odd, using a system that looked straight out of the early 90s.  I could clearly see the advantages over a straight terminal, though; three terminal windows and the ability to run graphical applications must have been amazing to those used to working with command-line programs such as nano.  Kinda like using compiz after years of XP's GUI. :P
