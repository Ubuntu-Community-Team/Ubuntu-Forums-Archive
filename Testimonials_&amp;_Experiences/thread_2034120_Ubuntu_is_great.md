### Post by paulc101 on 2012-07-27
Ubuntu is great. I like the unity interface and love the software centre, makes it really easy to find great software.

Some things I think Ubuntu could improve on are;

1. Network drive mapping - having to use the archaic method of editing the fstab to map your network drives is outdated. This should simply be point and click at this stage.

2. File Association - You should be able to browse for an application to open your files with, not just pick from a predetermined list. Not user friendly at all. 

One thing I loved about installing Ubuntu is that I could click the twitter links and open firefox - seriously you guys should just put an 'open firefox' button on the install page - because once it was open I could browse anywhere. I even set my youtube profile to html5 and was able to watch videos during my install. Better than watching a progress bar!
