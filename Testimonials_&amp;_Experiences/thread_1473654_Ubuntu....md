### Post by User3k on 2010-05-05
I have been with Ubuntu since the start. I actually gave a review of it on another forum after I tried it. I told people to keep an eye on this one. The first release was great and I saw Ubuntu going far.

But even then I kept bouncing back and fourth between Windows and Linux in general. I don't stay with just one distro either. I like to bounce around myself but Ubuntu and Xubuntu are my favorites.

I have it installed on both mine and my daughters computer. I am using Virtual box, which the 3D support is a huge and excellent step forward. Wine, though not without issues at times, has also helped. I plan on getting one game installed and I am hoping the fps has gotten better since the last development release.

When I install any OS, at least the ones I will be using as my main OS, I like to have the basics set up, or at least be able to set them up easily. Basics=Multi-Media Support, etc. I also like to have the basic amount of programs like media players, photo programs, internet, etc, set up. Then I can take it from there. I would rather install what I want rather then go through and take out a bunch of programs then install.

I also want to add that many people are complaining that Ubuntu is getting to easy, too much like Windows. I really disagree on this. I have used Windows since Win 3. I really don't see them becoming more like Windows at all. I do see them becoming more user friendly for the average user, which is a great thing that can and will only benefit all of Linux. If more of the "average  user," uses Ubuntu then the demand for software and hardware manufacturers to support Linux will be greater and if they want to make money on this rather then loose money then they will have to keep the end user happy with what they want.

I don't see it as a open source sin ether if Canonical want's to make money. Making money for hard work is not and never will be an issue with me. Screwing the people for a buck, like MS does, that does bother me.

So All my experiences with Ubuntu are positive. I only use XP for one game at the moment which will change once I get it set up with Wine. I believe security updates, etc for XP will end in 2014 and I refuse to install my copy of Vista again or buy Windows 7. I would do this no matter what but Ubuntu is making this change so easy and great.

Ubuntu is not perfect, no OS is, but they are by far the best choice out there right now. This doesn't mean I hate other distro's, I still do my fair share of distro hopping. However Ubuntu has a permanent spot on my hard drive while I use other hd's for the distro hopping.

I also want to add that setting up a home network and setting up remote access to other computers on the network has never been easier to do. Saves so much time using Ubuntu/Gnome that I have time to do other things as well as learn other things about Linux and helping others as much as I can.

My daughters computers, she is on Myspace a lot, uses the webcam, etc. Ubuntu is working out perfectly for her as well. That is a major plus.

I could go on but I don't want to use up all the space on the forum ;-) lol.

Ubuntu team, Canonical... Excellent work and thank you.

### Post by murderslastcrow on 2010-05-05
I totally agree- it's not becoming more Windows or Mac-like. It's just bettering and simplifying the Gnome and KDE and programs that were always there, and improving every new release. That's how open source works- that's why Apple uses so much of it, and their kernel itself is entirely open source.

I agree that there's no problem with Canonical trying to make a little extra money, and if they can do that by offering tons of online storage with Ubuntu One, a good music selection through the Ubuntu One Music store, and with commissions on people eventually buying Adobe CS through the Software Center, I don't see a problem with it.

Those same people were going to buy music on iTunes and strip the DRM, buy Adobe CS and run it in Wine or VirtualBox, run Steam in Wine anyway. I think it's great that we have native and open source resources for all of this, and we owe it to Canonical to support them in their endeavors for what they've given us. Doesn't mean we need to pay them money, we just need to support them in including things that are good for them AND us, which they have done, and they're doing a great job, and I can hardly see what they could so to open Ubuntu One and the Software Center up to us beyond opening the encryption, which would be unfair to them and cost them money.

I think it would be cool to have Ubuntu One link up to a personal server, though, but still, they're being as good as they can about these awesome new features. This is what open source was supposed to be- something that could make money, where everyone's offerings differ a bit and there's healthy competition. We don't want the colors and package management to be the only major difference between our distros, do we?

Anyway, I'm preaching to the choir, here. :3 I also love this latest release, and my friends and family quite like it, too. Let's keep making Ubuntu just a little bit better, eh?
