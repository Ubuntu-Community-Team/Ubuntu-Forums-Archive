### Post by ajgreeny on 2012-05-24
A few people, including me, have had a problem installing from the live CD with what seems to be the slideshow crashing.

In both ubuntu and xubuntu 12.04 my desktop installation from live CD would get to the final part of the installation procedure, where the slideshow is about to start, and the whole thing would simply crash out with a spinning cursor, but nothing happening.  This was repeatable 4 or 5 times in both OSs, so I eventually used the alternate install CD for both of those with no problem.

I then tried lubunrtu 12.04 from live CD, but before starting the installation I removed **ubiquity-slideshow-lubuntu**, having seen a post suggesting that would help.  The installation proceeded without a problem.

If this is your situation with ubuntu it may be worth running ```
sudo apt-get remove ubiquity-slideshow-ubuntu
``` to try the same thing.
