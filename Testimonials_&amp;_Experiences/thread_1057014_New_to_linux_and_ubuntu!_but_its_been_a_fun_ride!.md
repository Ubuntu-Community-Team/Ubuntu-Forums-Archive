### Post by Rumbl3 on 2009-02-01
Well, I figured i'd post a little something in here. New to the forums and linux etc.

I first gained interest in linux because all the times i've read how it's superior in the server world. So i set out on a mission to somewhat learn a little bit first before i just went off trying to setup a game server on it.

I've learned some but now i've gotten the bug. That I love linux enough that i use it 90% of the time now only time i'm in windows anymore is when i'm gaming on a game that isn't native or don't run on wine very well (left 4 dead). 

It's awesome though all the good tools that come with it for doing anything i can image. Don't have to spend 100's if not 1000's for a bunch of programs. I mean yeah windows has a lot of free stuff out there on the web but imo from my little bit of experience i've had lately linux stuff just works so much better and smoother.

It's pretty funny tho lately i been talking about it at work. Most of my co-workers are computer enthuist and gamers. They all act afraid to give linux a shot. I mean yeah if your a gamer i think you kinda need a dual boot machine for the time being if u want the new hot games to play. But other then that linux don't seem to hard to use and it's just so much better.

Anyway i'll shut up now some of my friends have gained interest in me adventures in linux using it to run my game server and doing most of my computing in it.
