### Post by bobvfr on 2013-02-03
I downloaded Ubuntu a month or so ago using the windows installer, then after a week of trying to get the wireless network card to work, gave up and un-installed.

Being stubborn I tried again this weekend, I have to say it does seem like there an awful lot of crossed wired postings on these forums, and not being a Unix user I really struggled to sort out the wheat from the chaff.

I guess people who know what they are talking about, forget that others don't and sometimes basic communication is forgotten.

Anyway, I had already figured out it was the drivers that hadn't been installed and eventually found a wiki with the right advice, downloaded the right drivers and it worked.

As mentioned Unix is newish to me, I used to run a FirstClass mail server and in doing so used NT server to do my DNS etc (Yes I know) so not really a computer newbie but I do forget stuff sometimes and had no need to play with Unix.

So Ubunto, I like it, some oddities, for example the srcoll bar in Firefox is OK but in other programs it is a pain, but I am sure I will get used to it.

At the moment I am dual booting with XP, but once I get my legacy files sorted I hope to ditch Windows.

As a domestic I am interested on a couple of areas:

 I use an HTC HD2 mobile, not a big user of apps, to be honest I could get by with just Google maps, but would like to see some integration between my phone and PC, so really looking forward to running Ubuntu phone, if anyone nows if there is a trail version out there, let me know. Still running it with Windows mobile 6.5 and Sense.

I am considering buying a basic desktop to use as a "Media center", hooked up to the TV via HDMI to play CD's, MP3's DVD's, and also to use it as a server for the families files etc, just need to find the right programs to use, can't say I like Rhythmbox much but maybe I need to play about with it a bit more.

I am sure I will have hundreds of silly questions over the next few weeks, and for the un-initiated the VFR part of the user name is the type of motorbike I ride. Got to admit I find bikes much more fun than computers ;)


Bob
