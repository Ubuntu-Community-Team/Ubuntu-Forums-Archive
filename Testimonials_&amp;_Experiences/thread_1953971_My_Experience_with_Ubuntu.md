### Post by serg2k9 on 2012-04-07
To start, I am 22 years old college student and have been using Windows since I was the tender age of eight. I know my way around windows like my own home, and it is basically a part of life. I always knew about other operating systems during my life time, but i always had a meddle curiosity about Linux. 

After I recently turned 22 years I became more interested about Linux and how it was open source and fast with a vibrant community. I "tested" out Linux on a virtual machine in Windows, needless to say i felt unimpressed at the time. However, I felt this way because of how overwhelmed i was with the amount of commands, shortcuts, and structures it has. I quickly then closed the VM and kept it as a itch on the back i didn't want to scratch, but shrug off. Less then frequent, I would boot the VM to see if i would have the will to learn, slowly and patiently i did. The vastness, openness, and greatness that Linux would unravel slowly to me. 

Months later of on and off relationships with the OS, i finally decided with version 11.10, i would install it as my main OS on an old laptop that used to run a horrid copy of Vista. The install went better than expected, originally was dual booted then deleted the Windows partition. Of course, for some unknown reason, and whenever the laptop would have problems, such as Gnome or etcetera i can lookup on-line. Fixes are as simply a command away from terminal, to starting with commands in Grub, or running a script at startup to make my life easier. Often, i would spent days and nights trying to fix a problem that a possible solution did not work. After sometime i felt it, with some experience, was time to install Ubuntu 11.10 on my main computer. Again, Install was easy to dual boot with windows and was flawless. Had problems along the way such as GFX card accelerating 3D, but solved it with the communities helpfulness. 

Anyway, in short, learning a completely new OS was a challenge that taught me to be patient, and carefully examine problems throughly and clearly.
