### Post by philfromns on 2007-05-06
Two months ago my Windows xp crashed for what was to be the last time, I had downloaded Ubuntu months earlier meaning to try it. So, I figured, now is the hour. I was mighty impressed and fell in love with it right away. Took some time to get used to. Then I installed Beryl and that was it, I never looked back. Do I miss Windows? Nope. So far I've tried other distros including PCLinuxos, Suse,and  Knoppix but I've found Ubuntu to be the best in my opinion. A couple of friends saw what I was running and they too want to install Ubuntu. Made me a believer!
I'm running a pentium 4, 266 with a gig of memory and a Nvidia Geforce 6200 graphics card.
