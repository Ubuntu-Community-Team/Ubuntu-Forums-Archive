### Post by p.i.m.p on 2006-09-03
Here is my experience with Ubuntu :

Having tried a variety of linux distros over a period of time, i've found that somehow or the other my linux box is unable to provide all of the functionality of my windows box. Somehow, there was always some random quirk with every distro that made me fall back to windows for my computing needs. The distros that I have tried over a period of time : -

Redhat
Mandrake
Fedora
Suse
Debian

The last one that I tried was fedora and though I am reasonably experienced with linux I simply could not get it to detect and run my Pinnacle PCTV card, which has been the single most troublesome piece of hardware that I have ever used! Then I read about Ubuntu. It's taken the linux world by storm and in the past few months has become the single most used distro on desktops! [source : DistroWatch]

Googling for further information, I found that you can order free Ubuntu Cd's. [http://shipit.ubuntu.com](http://shipit.ubuntu.com)

I registered and ordered a pack of five ubuntu cd's which were delivered to me within the five weeks promised :) The next part was installing ubuntu. The cd booted up to a live linux environment based on Gnome and double clicking the shortcut on the desktop started the install. The first time around I created a partition of 5 gb as I was gonna format it after playing around with ubuntu for a bit. The installation took all of five minutes as compared to Windows XP which takes about half an hour on my box. I rebooted and booted ubuntu up.

Impressions :
The boot splash screen looks great! However, the grub menu could also do with some colours to liven it up. This can easily be accomplished post install however which is what I have done.

Boot time is comparable to windows if you take into account that Windows makes you wait for some time on startup while it loads all the programs in startup.

The default human theme for gnome is gorgeous although i thought that the default drive icons could use a bit of work. I have since then changed the theme to one resembling Mac OS/X

Performance is great! Much better than Windows and a stable environment for downloading 24/7

After spending a month with Ubuntu, I have completely deleted Windows from my hard disk! Ubuntu works great for me, it detects every single piece of hardware I have and aside from the Nvidia drivers (which were installed with ease using apt-get), I've never had to install a hardware driver. I've resolved never to use Windows again. I mean, look at what I get with Ubuntu it looks better, performs better, is more stable and I can even play Windows games on it using Cedega. :)
