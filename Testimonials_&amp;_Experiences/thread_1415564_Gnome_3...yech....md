Why exactly should it be on the bottom? My way of thinking is that it should be on the top, because currently Gnome's application menu is in the top-left corner. The Activities button is the equivalent of the application menu, so it should be in the top left.

I'm disturbed by the seeming lack of customization options. I can't figure out how to set an application to stick to the applications box. I can't figure out how to get the full list to sort by category like Gnome Shell used to do. There are no right-click menus either.

And you can't add existing Gnome panel applets to the new panel or the shell. It's a very spartan environment so far. And the whole shebang doesn't work without 3D acceleration which is useless for those poor souls with unsupported ATI cards, ancient Nvidia cards, VIA and SiS graphics.
