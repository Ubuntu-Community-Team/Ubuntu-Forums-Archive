### Post by Mrandersonjr on 2010-11-22
I first used ubuntu when it was 8.04 (hardy). Once jaunty came around, it felt chunky, and I felt like ubuntu was controlling everything. I searched for other distros, and tried many. I stopped at Arch. Arch to me is the perfect distro, rolling release, no bloat, you control everything. Ubuntu is a good introduction, but it is not without it's problems. If I were to give a first time linux user a distro to use it would be Pclinuxos. Ubuntu now has become a joke, with the unity interface, and it's 6 month release cycle. So, if you feel like you outgrew ubuntu and are ready for something else, try Arch Linux.
