### Post by Old_Grey_Wolf on 2011-01-17
You can download a 10.10 iso using torrent which is very fast. Your speed will vary depending on the seeders at the time you use torrent. I've gotten 7Mb/s downloads speeds, or about 15 minutes to download the iso.  Upgrading from 10.04 to 10.10 will use the main Ubuntu server that is slower. The main server will not keep up with your Verizon data rate. I remember getting 200 to 300 Kb/s from the main server. An upgrade can download almost as much data as the new iso.
