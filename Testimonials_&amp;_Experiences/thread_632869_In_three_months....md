### Post by Scott_fakename on 2007-12-05
In the two or three months that i've been using Ubuntu, I have been able to become more proficient with Linux than I was with WIndows, which I've been using since 98 was newish. I'd been using microsoft almost all my computer owning life.

Linux, if you're willing and able to use command lines, which I prefer in many ways to GUIs (faster for some things, but I'm no command line hardliner by any stretch), you can do many times what you can accomplish, especially if you figure out how to use even history.

I'm still going to have to keep the windows partition, but I think I'm going to restructure my HDD from my current configuration (25gb winXP, 10gb Linux, 50gb Data files, mostly TV shows) to switch the sizes of ubuntu and windows.

My only caveat is that linux can't handle the TV recorder I use, and I'm a college student so I can't just go out buying video cards willy nilly. So I'm stuck doing a lot of things in windows. Also I use a dialup modem at home and ubuntu can't handle it, but I hear Gutsy may be able to.

Don't get me wrong... Windows is an ok system, and if it werent for windows, I'd have been forced to use Apple in those days. And Apple is just as evil as microsoft, but they're smaller so everybody cheers for them.

Anyway, if you read that entire post, I salute your persistance.

--Scott

### Post by matthewcraig on 2007-12-08
Great to hear about your success, Scott.  Things are changing for the better, too, so hang in there.  There's hope your TV recorder will pick up the support soon.  You can add to the effort by emailing the customer service department (not the technical support department!) of the hardware manufacturer and let them know you made the switch to Ubuntu.  Maybe after enough people tell them the same thing, then they will commit resources to producing Linux drivers themselves.  You can also let them know about the project that was announced this year where kernel hackers will write drivers for no-cost, as long as the company gives them the hardware specifications.  Smaller hardware firms might be interested in such an offer.  PM me for the link to that project, if you are interested.
