### Post by ilgaar on 2010-08-06
After quite a some time of experimentation on Ubuntu, I have discovered some interesting things.

1. There are certain applications you should be aware of and consider the risks, when you are going to install them and use them.

2. Compiling  software from source is the most reliable way to get the best native binary code for your desired application, but not always you can get a good result.

3. Setting up or configuring a network connection sometime can be really hard, specially if you are going to connect to some Windows Server, or try to establish a VPN connection, since most of the protocols are owned by Microsoft.

I'm using Ubuntu 10.04 or lucid with a Gnome environment, on an old Dell Inspiron 6000. Well from my own experience, I should admit that It sounds like I have brought the poor machine back to life, and finally I can really do something with it. With a powerful Kernel and so many programing options, plus abundant of applications as always, I should say that I am really impressed. Ubuntu has gone far beyond any other operating system out there, that I have seen, used and experienced. But like any other operating system it has some minor issues. 

For example the default applications that are installed on the system might not be a very good application to use, or even they might cause some annoying problems. And some times you might not find the application which you could use to satisfy your needs. But there are plenty of software just in your desktops Software Center, which you can probably find what you are looking for, or replace an undesired one with a better alternative. And of course there are applications out there you could find on the web and make use of.

From my own experience, you should be aware of two software in particular these days, which might give you hours of headache. One is Evolution, and the other is Firefox. Since they are installed by default, after some times of using them you will experience continues crashes. Specially Evolution that has various bugs. Firefox has also become slower and buggier as it has advanced through its version numbers.

After days of frustration, I finally decided to break the taboo, and stop using both of them, so I began to hunt for a better browser and a better email client software. After many weeks of testing various browsers and email clients I think I have finally found what should I use. Did you say Chrome? No, I'm afraid despite the great propaganda by Google about its increasing usage, I must admit that after a long test, for me it was another buggy browser causing Ubuntu to crash. But finally I have found the most stable browser, Opera! Wait a second! Is that open source? As far as I know, it is not. But with all the great features and most importantly speed and stability I cannot simply ignore it. I really recommend using Opera, just give it a shut and see what happens.

From the very start day that I installed Ubuntu, I used Evolution frequently, and Ubuntu was crashing every 2 hours or so. At first I didn't know the reason, but gradually I realized when I use Evolution more often Ubuntu crashes more often. For whatever reason it was, I decided that enough is enough and I should use Thunderbird. So far it is working just fine and hasn't caused any problems.

There are times when you would need to set a connection or configure an existing one to connect to a Windows server. This could really exhaust all your energy, so I recommend to schedule it for a day when you are absolutely free. My experience is that I was trying to connect to a Windows server via a VPN connection that uses L2TP protocol, which is a Microsoft protocol. After hours of reading different articles, tutorials and installing and testing different application I finally got it to work. Though it took me 2 days to solve the problem, but I have to admit that compared to other environments, specially Windows, It was much easier since there was a lot of available information on the web to use.

Installing a software by just double clicking is great isn't it? Well that is a phrase you could hear from a windows user. It's true that almost any application these days can be installed by a few clicks in Ubuntu, but sometimes you might not find what are you looking for. Sometime you need to download some software from web in order to use it according to your needs. The best way to install a third party software is to download the source and later compile it in your native environment. But be aware, you should have at least a basic knowledge of programing and some skills to use the black-box(Terminal). Unfortunately not everyone out there appreciate the advantages of open source, and sometime when you find the sources that you are looking for, might not work. I've recently compiled a few application, but only one of them was truly supported in a Linux environment, which was i2p a proxy software. All the others because of the poor support of the companies for Linux were merely a figure! I hope one day the world will understand the power of open source and start to appreciate it as it should.
