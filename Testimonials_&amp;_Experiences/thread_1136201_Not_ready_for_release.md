### Post by Old_Grey_Wolf on 2009-04-26
> **mk4umha said:**
> Anyone know how to even get this installed? I can't even find the Key to add to the key list to get the old driver. Worthless wiki IMO.

Get the PGP key from this address: [http://keyserver.ubuntu.com:11371/pks/lookup?op=get&search=0xCE90D8983E731F79](http://keyserver.ubuntu.com:11371/pks/lookup?op=get&search=0xCE90D8983E731F79)

Copy the text that begins with "-----BEGIN PGP PUBLIC KEY BLOCK-----" and ends with "-----END PGP PUBLIC KEY BLOCK-----".

Save the text in a text file.

Go to System > Administration > Software Sources and open the Authentication tab.

Click on Import Key File and select the file you just saved.
