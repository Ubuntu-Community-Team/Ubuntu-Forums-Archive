### Post by beno1990 on 2008-10-18
I have been a Linux user, using mostly Fedora, since I was 13 years old. I started out on Red Hat and then moved to Fedora Core 1 with its first release.

I enjoyed the OS and I stuck with it, but it was lacking in a few areas. Eventually, I got fed up of Windows on my one remaining Windows station and dual booted it with Ubuntu and Windows XP. I enjoyed using Ubuntu on there, I found its core desktop features easy to use (some of which were *easier* to use than on Windows), but at the same time it gave me the same capabilities I'd come to expect as a power user on other Linux distributions.

During the middle of August I was about to set up a server to host a website from my bedroom. I considered several operating systems, including multiple Linux distributions, BSD, and Solaris, before eventually deciding to use Ubuntu server edition 8.04. I'm glad I made that decision, because setting up the core features was so easy, and setting up the web server, mail server, and other server software was far easier than my experience with other server OSs, namely Windows server 2003. After the almost seamless initial setup, I found administering the system and customizing it using the package management system far easier than on Windows, likewise, compiling the IRC server daemon I wanted to use was far easier than compiling from source on Windows.

In addition to its ease of use, I've had almost no downtime on my server system, while most of the Windows servers I've worked with have progressively slowed over a period of a few days and ultimately needed rebooting, showing again that Ubuntu's ease of use doesn't come at any other cost.

After my old laptop hit the fritz (it fell apart almost literally, funny how a small crack can eventually cause the whole thing to crack up and fall apart), I had to get myself a new laptop. I'd decided by now that I didn't want to use another Windows computer in my home again, so I bought a Dell Inspiron 1525 with Ubuntu pre-loaded. Other than my slight confusion with them sending me a CD with the Windows drivers for the laptop's webcam, it's worked amazingly, and probably beyond my expectations.

Linux and Ubuntu still isn't perfect, just as no OS is, however, and I'm hoping I can help contribute in the future as a developer.

But apart from that, I'd just like to say a big thanks to everyone who's worked to make Linux and Ubuntu possible. :D
