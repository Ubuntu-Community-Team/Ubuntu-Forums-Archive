### Post by Baldrick_NZ on 2012-01-26
I've just applied the new kernel, and it's fixed some of my bigger personal niggles, particularly around the launcher dock.

You can now alter the icon size, meaning you can: 
a/ add more icons to the dock
b/ make the dock smaller for larger screen monitors
c/ gives some control over the dock back to the end user

Ofcourse, more control over the dock can be obtained by installing the latest version of 'Ubuntu Tweak', which I notice is not in the repos. Odd.

There is now a Bluetooth icon included on the taskbar where you can toggle on/off. Default is off. :-)

Ubuntu 12.04, IMHO,  is definitely shaping up to be the best release yet!
