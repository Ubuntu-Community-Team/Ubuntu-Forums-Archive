### Post by csacpt on 2011-02-07
Well been running Ubuntu for about two months now and since I have done so I am definitely not disappointed. Actually switching from MS Office to Open Office started the process. Then Firefox instead of IE. First install of Ubuntu was 10.10 on my Dell 1720. I somehow managed to waste the Vista on it in the process which was no big deal since it was a standby machine anyway. Anyway, the more I played with it the better I liked it. Still no Windows on that machine except for an XP VM that I needed  VPN for work. Honestly feels good to have Windows in a cage so to speak. LOL

As soon as I can learn how to setup Openswan or something for the VPN even that probably won't see much use. I do have some specialized hardware, cameras and ballistic chronograph which only has Windows drivers which are not generic. Anyway I do not plan to ever go back. 

The biggest plus for me is the people here and other places in the open source community, willing to help and just good to be around. Many thanks to all who make this possible and with a little time maybe I can return the favor. Think I will stick around for a bit.
