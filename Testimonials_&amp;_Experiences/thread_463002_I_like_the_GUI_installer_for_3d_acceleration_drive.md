### Post by vadania on 2007-06-03
Very often, after installing a new Linux distro, I did some configuration, install some apps and then installing proprietary graphics drivers. 
Installing graphics was a pain because I had to stop X, assign rights to some directories, read a tome of documentation and then I could install the drivers. The installation itself was most often errorless, but when I tried to start X again - ups!! X was broken beyond my ability to repair it.
As much as I like to work in the console, sometimes I really need GUI, so I had to reinstall Linux.

With Ubuntu 7.04 installing 3d acceleration graphic driver was a breeze! I had no errors and I didn't even need to stop X and use the console for the change. Since installed graphics drivers 3 months ago I had no problem with them.
