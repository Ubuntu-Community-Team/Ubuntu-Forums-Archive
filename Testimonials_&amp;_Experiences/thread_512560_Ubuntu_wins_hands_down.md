6) some firefox extensions wont work properly. but i guess thats hardly to do with ubuntu developers. im just ranting here.

### Post by Warren Watts on 2007-07-29
> **praveenmarkandu said:**
> I still prefer package managers like Smart and URPMI compared to apt-get. they just seem a lot smarter than apt-get when i try to search for something im not sure what the exact name is.

If you aren't sure what the name of the file is you are looking for, you can use apt-get's GUI interface (the Synaptic Package Manager).  It's under System | Administration | Synaptic Package Manager.  
Start it, then click the search button and put in a partial name it will find everything it matches.

For example, you can enter FTP into the search, and it will list all the FTP related packages and libraries it finds.

Hope this helps!
