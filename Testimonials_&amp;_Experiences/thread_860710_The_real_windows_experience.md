### Post by michaelkahl on 2008-07-15
So I'm tired of reading the troll posters on here, so I thought I'd post some true experiences as of late.
As some of you may know, if you read my intro post, I run Ubuntu at work for my main machine.  I recently got a laptop at work, and installed XP on it so I could work with SMS and Visio 2007.  I have a nice setup, I leave the laptop setting on top of my cabinets connected to my ubuntu box(ethernet).  I remote in when I need something from Visio or need to use the SMS client.  I share internet through my ubuntu box, joined the laptop to our domain and got everything running that I needed.  Here's the testimonial part.
I loaded XP, btw this is a Gateway M285-E tablet, and had several drivers that needed to be installed.  What really annoys me is when the ethernet device isn't supported, but after a quick visit to gateways site to downlaod drivers I realized what a mess there driver packages are.
For some drivers I did simply go to the manufacturer, but I wasn't sure of all the chipset model numbers on everything.  I knew I had an Intell Pro 1000 ethernet, but which one?  I did grab the wireless chipset utility from Intel, that was easy to do.
Once I got all the drivers taken care of, it was time to run updates.  I made my life simple and went strait to SP3, which installed with little problem.  Then I ran updates again, and windows updates listed updates that I needed for SP2????:confused:??? I have SP3 lol....but anyway.
I finished up updates, and no I didn't run SP2 updates, then came the task of installing all the software.
I was downloading from this website, that website, getting software from our share drive, installing Office 2007, Visio 2007, SMS, and the list goes on.  In total it was a one day setup, with a little spilled over into the second day.  (this isn't fully to blame on windows or a slow machine, I do have work to accomplish during the day too).
With Ubuntu I do the following,
Install
Update
Map network drives, and set mount points
install evolution-exchange plugin & setup email
install keepassx
install wine, with pronto instant messenger
install our xerox wcm20i printer (not difficult at all)
Thats about it...there really isn't much to it.  I know I have the laptop setup for Visio/SMS but thats only because I'm on a project atm using Visio.  In reality I could just remote into one of our servers for SMS functionality and if I wanted to use one of our lab pc's for visio, but I'd rather sit at my desk instead of going to another office/room.
Setting up Windows on that laptop reminded me of several things that I dislike about it.
1.  I'm frantically installing software from the start to protect against viruses/spyware.
2.  Hunting down drivers is a pain in the rear.
3.  Windows doesn't care what the focus of your mouse is, it just throws new windows in your face when it feels like it.
4.  Defraggin, cleaning the registry
5.  Poor use of memory and swap or page filing
6.  The ugly Luna theme
7.  IE (all versions)
I can't completely hate, so here are three points I give MS credit on.
1.  Wireless, I can get wireless working in linux, but not as well as windows does.
2.  Office 2007, esp Outlook 2007, I just like it alot.
3.  Xbox 360, not OS related, but especially after last night I see that they are doing everything right.  Netflix, FINAL FANTASY XIII, XNA, and so on.
So there it is, a long post about my true experience with XP and Ubuntu.
