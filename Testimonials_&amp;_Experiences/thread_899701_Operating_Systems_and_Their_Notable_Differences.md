### Post by Unanimated on 2008-08-24
Okay, I'm bored. Here's my personal positives and negatives of each OS that I've used in my life (Windows XP, Windows Vista, Mac OS X Tiger, Mac OS X Leopard, Ubuntu 8.04 LTS).

Windows XP:
Although it has numerous problems like viruses, hacking, and the god-awful Internet Explorer, I still managed to find good in it. I thought it was a decent platform for browsing the Web and occasionally playing a few games. It found it never really ate up my 787MB processor, until I started running those numerous antivirus and firewall programs that I had installed. That's my main gripe about Windows - it can be a little too in-your-face at times and it's viruses.

Windows Vista: (Note: Don't take a serious opinion to this part - I've probably used Vista for about 30 minutes in all.)
My grandpa recently got a computer that had Vista installed on it, so I've used it for about an hour and a half. I noticed a lot of problems, more so than XP, and I've read a lot about it on the Internet. I've read that it's like one big spyware program, submitting anything you do to Microsoft. Which I actually really doubt. It still has some of the main problems of XP, only without the partial stability. It did eat up his processor, it was extremely in-my-face about the simplest things ("Do you really want to close Internet Explorer?"), but the GUI looks nice. I downloaded a theme for Windows XP that made everything look like Vista, it just didn't have some of the special effects or the modifications to Windows Explorer.

Mac OS X Tiger:
I didn't really like this one. I thought it crashed too often, and Firefox really got on my nerves. For instance, if I had Meebo in one tab, MP3Tunes in another, and then I was just browsing around the internet, you can only control one Flash thing at a time. I would have been controlling MP3Tunes, then a friend would IM me in Meebo. MP3Tunes would have the keyboard, and I couldn't send anything to my friend in Meebo. I got it to swap the keyboard, but I really don't remember how. It got to that I had to have a different Safari window open whenever I wanted to use another Flash app. I just really hated it. The environment was nice, however.

Mac OS X Leopard:
This one was much better than Tiger. It was easier to use for me, and I think anyone who has no experience with a computer can easily pick it up and start using it. I found the display to be nice, the dock to be handy and nicely customizable, and the included Boot Camp is something that's great. However, I don't like how you have to hit File-->Quit to really close out of something. After a while of using Leopard, I noticed the computer really started to slow down unless I closed a program using File-->Quit. Personally, that's not the most user-friendly thing in the world.

Ubuntu 8.04 LTS:
This is my favorite out of the bunch. GNOME is very nice and easy to use, Compiz looks nice, I love all the included and available programs, and I think the repos is by far the easiest installation process for a program ever. I love the open source nature of Ubuntu - I love how everything is so customizable because of it. The things I did notice though, Compiz slows down my computer a lot, NVIDIA has really crappy support for it, and sometimes it crashes a lot. However, I can easily see myself using Ubuntu in general for years to come.

My times of experience:
Windows XP - Own a copy
Windows Vista - On my grandpa's computer
Mac OS X Tiger - On my other grandpa's Mac
Mac OS X Leopard - On my cousin's Mac she got for college
Ubuntu 8.04 LTS - Own it

Personally, I think that someone should go with whatever OS they like the best and what works for them. What made me switch from Windows to Ubuntu was that I was bored of Windows and wanted something completely different. So, I switched, and I love it.

### Post by 3rdalbum on 2008-08-25
What processor was in your Dad's computer?

I unpacked and set up a new computer that has Vista. It had a Pentium D processor (weak!) and two gigabytes of RAM, and I didn't notice any slowness after I removed HP's crapware. The CPU monitor occasionally jumped to 20-30% when idle but only for a second, and didn't seem to jump when the computer was already under load.

Mac OS's way of not requiring open programs to have an open window is just a different way of working. A Mac user quits their programs by pressing Command-Q, but a Windows or Linux user quits them mostly by the close box. There are good reasons for and against both methods.
