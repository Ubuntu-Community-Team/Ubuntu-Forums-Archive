### Post by DavidVanCan on 2007-07-10
Just had to share my positive experience. I had an an aging Pentium system that I decided to upgrade. Got a Core2Duo E6420 CPU and an ASRock Dual Series 4CoreDual-VSTA motherboard. The nice thing about this motherboard is that it supports either an AGP Video card or a PCI Express card. I had a legacy AGP card that I wanted to use to keep the upgrade price down. It also supports both SATA drives and up to 4 IDE drives. Again, I was able to use my existing IDE hard drives and DVD drives to save money but there is also a future upgrade path.
   Anyhow, I took my old computer apart and reassembled the new new one using a mixture of new components and existing components. The first piece of good news was that it powered up and I was able to access the BIOS settings on my first try. Then I booted from the Ubuntu 7.04 amd64 install disk. Again good news - the installation process went flawlessly. The only glitch occurred when I rebooted to start up off the hard drive. I had to make one change to the BIOS settings before I could boot from the hard drive. This had nothing to do with the Linux installer. Once I made this change I booted straight into 64 bit Linux. All devices are working properly and I am enjoying the snappy new performance.
   I am very, very happy that the upgrade went so smoothly. Ubuntu is just getting better and better.

Edit: Well, I've been running 64-bit Ubuntu for about a week now and have all the features that I wanted working including WINE, DVD playback, Flash in Firefox and the LAMP server. The only thing that isn't working as well as I had hoped is Google Earth. It's not getting along with my ATI video card and the ATI restricted drivers. Google Earth runs but it is emulating OpenGL in software and is therefore pretty slow. I think this is as much an issue with the Google Earth people as it is an issue with Ubuntu. At any rate, I don't doubt that one day Google Earth will run as well as everything else does either because the drivers get fixed or I buy a nice nVidia graphics card.
