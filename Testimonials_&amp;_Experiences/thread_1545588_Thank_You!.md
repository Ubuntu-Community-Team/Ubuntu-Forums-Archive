### Post by Maverick_Meerkat on 2010-08-04
Greetings all,

Let me preface this by saying that I have been a computer enthusiast since before the days of the Apple II and the Commodore 64.  So I remember when data was stored on audio cassettes. That pre-dates the invention of floppy disks. 

PC's have come along way since then. Like most people and for many years, I felt compelled to use Windows. Sure, it was glitchy, inconvenient, expensive, and required far too much maintenance, but what choice did I have? Buy a slick but overpriced Mac, which until recently didn't run Windows programs and traditionally had a pathetic selection of software? Some choice!

Being a bit of a techno-geek, I knew Linux existed, but I didn't know much about it. Nobody I knew used Linux. That was strictly for hard-core uber-geeks and what little I'd heard about Linux was rather intimidating. This coming from a Comp TIA certified repair tech with a background in programming. LOL.

Anyway, I finally got fed up with Windows and decided to take the plunge. Being the methodical, obsessive, perfectionist that I am, I did a little research before making the leap. When I considered all the available flavors of Linux, one in particular stood out. Ubuntu!

Why Ubuntu? Why did that ring a bell? Awhile back, I googled &#8220;Commodore&#8221; and run across this site that advertised an &#8220;all-in-one Commodore 64bit Phoenix&#8221;. I remembered that it offered Ubuntu as an option for the pre-installed OS! 

After seeing that, I'd been meaning to check out Ubuntu, but it just slipped my mind. I'd never heard of Ubuntu prior to viewing that advertisement, but the mere fact it was now somehow tied to this quasi-pseudo-neuvo-Commodore peeked my curiosity. Then I recalled something else: Sheldon had mentioned Ubuntu in this last season of The Big Bang Theory. Well, if it's good enough for Sheldon Cooper... 

So I did some market research and according to the numbers, half of all Linux users have chosen Ubuntu. In my mind, that meant it must be mature, well-developed, fairly stable, easy to use, and had no shortage of available software. A little more digging turned up some more interesting facts. 

1. Via WINE, Ubuntu was capable of running some (if not most) software designed for windows and do it without using VMware or emulation! 
2. It was possible to update/sync my 80 gig iPod with native software!
3. OpenOffice would provide access to all my M$ Office/Works docs, databases & spreadsheets!
4. Ubuntu was free!

Well, that cinched it!  At that point, I had no further reservations.

The installation process had one minor hiccup, but I half expected that. Seeing as I was using a five year old Toshiba laptop. In fact, I was expecting multiple issues. I also expected the need to install a small mountain of hardware drivers. Much to my surprise, installation was unexpectedly painless and required virtually no time at all (compared with the 90 min installation time required for XP Pro). 

When the install finished and Ubuntu booted, I was stunned to discover EVERYTHING was configured and functional! No video, sound, ethernet/wireless modem, or touchpad drivers needed to be installed!
 
It took a whopping 30 minutes of playing around with Ubuntu to convince me to scrap Windows! I was completely blown away. The more I fiddled with it, the more I fell in love. 

I can't believe how customizable everything is! With a little programming or scripting knowledge, there's not much you can't change in Ubuntu. 

That's something I've always hated about Windows. I'd have to hack resource files, .exe, and .dll files to remove all that darn branding. Additionally, I would have to spend hours after a fresh install to clean and optimize the system.

I have a custom XP Pro install that is exceptionally clean, but I'd still need to disable all those unnecessary services that load at start up and run in the background, optimize all the performance related settings, customize all the user settings, install drivers and desirable software, then use third party software to change all the hidden system settings. Not to mention searching the file system to delete all those useless help files, read me files, and .txt files.

With Ubuntu, there's not a great deal of branding, but I'm proud to keep it! As for optimization, there's not much to be done. What little there is to do, is a snap. As for maintenance, there's not much to do there either. There's no darn Windows type registry to muck things up. Temp files, caches, and removed packages are quick and easy to deal with and no more defraging the file system. As for security, there's no need for MULTIPLE suites of antivirus software. 

Don't even get me started on Ubuntu's superior performance. The boot time is blazingly fast! Internet speed is fantastic. Shut down is almost instantaneous. Ubuntu's ultra-low demand on system resources and it's minuscule memory footprint are amazing. 

My custom, cleaned, and optimized version of XP Pro required 200MB of RAM when idle immediately after boot. Ubuntu requires only 90MB! As for disk space, forget about it. XP Pro needs 1.5 &#8211; 2.5 gigs (depending on pagefile size) just for the OS. By the time I install all the software I need, it eats up 10 gigs. Ubuntu with all the software I require, only 2.6 gigs! 

Ok, so there was a short period of adjustment and shallow learning curve, but it certainly wasn't intimidating as I expected from a Linux based OS! As for the Terminal/CLI... Traditionally, I've hated them (once again, this coming from a guy with a background in programming), but with Ubuntu it's really growing on me. There are times when I actually prefer the Terminal to any GUI. Besides, it looks so cool! When someone sees you using it, they think you're some sort of techno-guru or hacker.

But seriously, the CLI in Windows isn't very functional. With Ubuntu, the Terminal is useful, powerful, and quite user friendly. 

I've been using Ubuntu, intensively, for three weeks now and I'm a convert. I don't see why anyone would want to use Windows. Ubuntu is clearly the superior choice. For an average user Ubuntu is just as easy, if not easier, to use. From a tech guy's point of view, Ubuntu is safer for the average user and far more flexible, powerful, and efficient for advanced users.

I'm tempted to say, I wish that I would have jumped on the band wagon along time ago, but on the other hand... I'm sort of glad to be a Johny-come-lately, because it has given Ubuntu time to mature and grow into the polished product it is today. I'm not easily impressed, but the Ubuntu team has done an outstanding job! =D>

I'm not much of a &#8220;joiner&#8221; nor am I typically an evangelist, but Ubuntu makes me want to contribute to the cause. It makes me want to get involved, to spread the word, and to enlighten others. Generally, I'm not one to give testimonials for any product or endorse anything, but lately I've been singing the praises of Ubuntu to all my techno-friends. 

Of course, the down side is: if everybody starts using Ubuntu rather than Windows, I'll have to find a new profession.

At [http://brainstorm.ubuntu.com/](http://brainstorm.ubuntu.com/) someone proposed an idea that, for those of us who have little else to offer, there should be some method of making financial donations as a means of showing our support for the ongoing development of Ubuntu and to express our gratitude for the continual and outstanding work being done by the Ubuntu team. Personally, I think that's a very good suggestion. 

Anyway, I just want to express my gratitude to everyone (upstream and downstream) who has contributed in any way to the development of Ubuntu and to those who develop all the wonderful (third party or otherwise) software that makes Ubuntu useful. That includes every person who uses Ubuntu, as well. If there wasn't a demand for it...

Thank you all for such an outstanding OS.
