### Post by ringding on 2009-10-21
Just thought I would join the Forum and sing my praises about Ubuntu.

As a Windows Server 2003 user we were looking at ways to deploy something quickly and effectively which gives us room to expand quickly without cost overheads. A friend suggested Ubuntu Server.

We decided on Ubuntu desktop due to the added advantage of RDP via TightVNC. 

We have never used ANY linux based system before (but I am a Perl programmer) so this was a BIG step.

I started on Friday afternoon last week. It has since took me 6 simple re-installs to get everything right as uninstalls seem almost impossible and getting Apache configured with Perl wasn't that simple until I had my head around it. Don't know why as it seemed easy looking back :)

However, now Apache is working with Perl, MySQL works and everything is stable and we made 10 backups of the OS for installing on 8 replacement servers.

So here we are 3 working days on and I wrote a neat Perl script to pop on a USB key which does all the settings on the machine including setting its new IP, Hostname, configures MySQL databases, formats a backup disk and mounts it plus much more. All pretty difficult on a windows box.

So whats my opinion? Superb. Today we had 6 working servers all deployed by midday.

Well done Ubuntu. Can't wait to become part of the Ubuntu and Linux community.
