Problems with windows

2. Whenever i log in now i get told to install windows genuine advantage confirmation software, i have the official disc with me right here and i paid my money, i don't see why i should have to proove this by clogging my computer with software that does sweet fa unless your using an illegal copy, which i am not.
3. I use alot of programs that are open source or work on linux anyway, apart from microsoft office and my video games that only distract me from working anyway i felt using a system that was susceptible to virus' (of which i've had many) due to familiarity was idiotic.
 
And so i sent off for a free printed version of kubuntu which arrived sooner than expected and it installed in 17 minutes! after a quick play around i was hooked, i admit i was abit freaked out but i decided that anything new is going to require some time to learn what does what. Unfortunately when i set off to install the programs that i wanted and to get em working i encountered a few problems.
 
Problems with Kubuntu
1. the first program i installed was the latest version of firefox which worked beautifully and i was pleased with how easy it was to get working using the software installation manager that came built in, however when i tried to remove Konquerer the existing browser it uninstalled according to the software manager but it was still on my internet programs list and still continued to work, now when programs in windows did this i'd just delete it from the program files. I had no idea where to look in linux and as such felt it was better to just live with it.
2. The second program i installed was Thunderbird, well i say installed it never came up in the internet programs list, it would'nt start up from the console (i may have been using the wrong command... :) ) and again i had no hope of finding it anywhere.
3. My dad who was cynical but curious as to how good linux was asked me to burn a music cd for him, as a comparison he burnt the same disc using Nero on windows, the linux disc was very picky as to what it would and would'nt work on, since my dad listens to cd's in the car 90% of the time and it worked on his car stereo it did'nt matter much, but it was rather annoying why it would'nt work on a normal cd player.
4. I am a fan of realplayer as a program as it enables me to download flash video's from the internet so i can put them on my phone or whatever, instead of using the software manager tool i decided to try installing via a pre downloaded package that i'd got before i installed kubuntu, yet again it installed but refused to show up anywhere or work via the console.
 
Eventually my computer completely refused to boot either windows or Kubuntu, although my dad insists linux is to blame i believe that my machine is frankly archaic and as such it was unfair to test kubuntu on it, any of the problems described above could well of occured due to faulty hard drives or my own ignorance.
 
When i get my new computer i intend to try kubuntu again because i strongly believe it's a good system that just requires a little bit of know how and determination to get it going succesfully.

Actually, that wouldn't explain anything at all. Behind the scenes, disc burning is done by a single program. No matter if you use K3B, Brasero, Gnomebaker etc the actual burning is done by the same program.

All CDs burnt in Linux conform to the "book standards". However, there are no standards for the quality of the media itself; it's much more likely that the disc you used was inferior to begin with.

Linux can read and play WMA through the use of the "w32codecs" (or "w64codecs") package, and flash videos automatically download to the /tmp directory when they load up on any web page. So you don't need Realplayer.

### Post by lykwydchykyn on 2009-05-13
I've not tried removing konqueror, but I wouldn't advise it.  It's not the ONLY file browser (dolphin is actually the default), but it isn't hurting anything being there, and it's tied in to a lot of things in KDE.

As a long-time KDE user, I can tell you that I'm not a fan of the default KDE setup on Kubuntu.  I generally select a variety of other applications from GNOME or XFCE when I don't care for the KDE alternative.  For example, adept/kpackagekit/kpackage have never done it for me, I use synaptic or aptitude for package management.  

As for thunderbird, I hate to say "works for me", but it does, so I cannot say why it didn't work for you.  Sometimes it takes a few minutes for the menu to update, especially on older systems.  In any case, I rarely bother with menus anymore, you can just hit alt-f2 and type "thunderbird" to launch it.  krunner (the dialog that comes up with alt-f2) is pretty intelligent and does a good job of finding programs based on keywords too (for instance, if I hit alt-f2 and type "mail" thunderbird comes up as a choice).

I've never had much trouble with k3b, in fact it's one of the 'killer apps' for me in KDE/Linux.  You might want to test a few more CD's before passing final judgement.  Who knows, though?  Maybe your burner has some "special sauce" that only the bundled Windows software takes advantage of.  Just speculation.

### Post by 3startuna on 2009-05-14
couple things

1. If your computer is older run Ubuntu with the regular Gnome. 

I have kDE on my computer but i tend to like the simplicity of gnome more than the pretty and IMHO more restrictive GUI of KDE. Give Gnome a try, its simpler and runs faster. I only use KDE to watch movies. And most KDE programs such as kopete and amarok works in gnome.

2. Downloading videos from the web. Unlike windows, when you watch a video on the web from a site like youtube, ubuntu downloads the video automatically to /tmp directory. simply open it and drag it somewhere else to save it and watch it later, no other software needed.Otherwise youtube ripper is a good program to use

2b. Konqueor - Its built into KDE as far as In know could be wrong so i dont think you can uninstall it. Maybe you can but im not sure. I would keep it just not use it.

3. Install the right plugins:

Some audio formats like wma doesnt work with ubuntu unless you install the w32codecs package. That was difficult to find as its no longer available in the repository. But there are sites where you can still get them. Secondly in default i.e. fresh install ubuntu doesnt play mp3s but it will download the right pluggins to do it, if you want to play them automatically. You will need to do this before making music cds.

Install Java

4 what exactly is wrong with your computer? try running live CD and re installing the grub boot loader if its not "starting up" Linux unlike windows is very sennsative to hardware changes, for example if you installed linux then deleted a partition without reconfiguring grub it wont boot up it will throw you an error message, or it will go past grub but show a "missing file system" error, these are all very easy things to fix.

Any problem your encountering is probably from a bad install but its nothing that isnt impossible to fix or really that difficult. 

5. Maybe you need to upgrade some hardware. Linux will try to run on an older computer and on older hardware, IMHO it does a good job at this but it will begin to show signs of weakness like the music skipping videos loading slowly etc, upgrading your hardware will make a very big difference.

It takes a bit of time to get used to for a new user, but once your patient it will change how you use a computer forever. You learn alot from running linux.
