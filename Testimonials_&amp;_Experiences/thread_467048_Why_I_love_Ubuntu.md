### Post by AlanR8 on 2007-06-07
If you know what you want, try to find it in Adept Package Manager, the automatic download and install feature in Ubuntu.

If you find software that you want elsewhere, look for a .deb file to download. Once saved, find it, Right click and take the option "Install with Package manager". 

Easy life!
