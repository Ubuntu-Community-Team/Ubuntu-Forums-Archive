### Post by newbie2 on 2005-11-11
"As a Debian user since the installation of Sarge became easy, I finally decided to try this "Ubuntu" Linux, which is built on top of Debian, that I'd heard so much about.

Now, Debian has become easy to install, but it helps to know a few things about your hardware, such as your monitor's vertiacl and horizontal refresh rate, and whether you need NVidia drivers. Ubuntu did all this for me, and didn't ask one single difficult question. The only thing anyone would possibly need to know is how to partition their hard drive if there is already a part that has data on it that they want to save. If an entire hard drive can be devoted to Ubuntu, I assume you just click through (I saved my home partition so partitioned manually - it's the same as done on Debian). Ubuntu has taken the stock Sarge installation script and smartened up the exact parts that would tend to be stumbling blocks for new users. Great job!

I've been using it for a day, and enjoying checking out the latest versions of Quanta (web development) OpenOffice2, etc. Everything works great. To add kde just install Kubuntu instead or do what I did - install the kubuntu-desktop package. I'd suggest installing the firestarter firewall app too; it's very easy and works well - it's interesting to see just who's trying to do what out there. Lots of Ms-sql-m entries and the like ;-)  "
