### Post by dv3500ea on 2010-06-24
I don't think so.
Linux is actually compatible with a huge range of hardware, though struggles with some newer stuff because manufacturers refuse to create open source drivers.
Most people would agree that Linux is successful because it is free and open source. It has many other advantages such as security but these mostly come from the freedom involved - anyone can improve Linux.

### Post by Penguin Guy on 2010-06-24
No, what you're looking at isn't incompatibility - it's merely a lower-level approach to computing. It means you see things in a more 'complete' way (it doesn't hide stuff form you, like file extensions - '[FONT="Courier New"]file**.txt**[/FONT]'). In fact, since Linux uses a lower-level approach, Windows is (in some cases) incompatible with Linux, but *not* the other way round.
