### Post by jprophet420 on 2008-03-15
A little bit about myself first... Trying to make a career transition into PC repair (self employed style) from actually working in a factory producing microprocessors. I have always been interested in Linux since I had heard about it in 1997 while I was going to school for a degree in electronics engineering technology. I have used many distos since then, but only as a hobby untill maybe a year ago. I have used a few fedora distros, slax, knoppix, and I had even tried the Dapper drake live cd.

Last year I got 'serious' and went to the best place for free knowledge, the public library. I had a 'free' pc to work with, Intel 1ghz processor, 20 gig hdd, 64 mb ram, and a library card. I started out with slax popcorn because not much would run on the limited memory. I was able to upgrade to 512mb and installed fedora 7. It was quite a good experience. I started learning to work with the command line, scripting etc. Everything was more complicated than windows but not hard to learn at all. The rig was not good enough to game on so I learned how to do everything one would do on a winslows box with linux except gaming. I build web pages, got all the multimedia working etc etc.

This Xmas I was able to upgrade to a new PC, a refurbished emachines. I immediately switched out the power supply and added a second HDD for ease of dual booting.

Boy was I wrong. The machine came with a copy of spyware infested Vista home basic. It is the worst OS I have ever used. It has a bug that when you install it the boot files get written to any drive available that has a windows recognizable file system. After much work I was able to figure out how to dual boot Vista/Fedora 8. However, Fedora 8 was crapola also. I installed the 64 bit version and it would not resolve dependencies. It updated to 9 beta and I got a black screen. Re-installed and dependencies would not resolve.

I had been hearing a lot about ubuntu by then. A friend IRL had been using it and one of my gaming friends was using it to play WoW. I decided to give it a try.

To my delight, the OS was on a live CD. This is important to me because one of the nice things about live distros is that if you can get them to read/write to the infamous ntfs file system you can remove viruses and spyware from peoples machines much more easily than you can from within windows. The only reason I even dual booted vista or even tried it was becasue I need to be able to work with it for customers.

I used the disk as a live distro to check it out and it was 100% compatible with my machine. Sound, video, and netwroking all started right up with no propmting. I had always used slax/knoppix and it seemed with every other machine one of them needed manual eth0 configuration.

Then I decided to install live from the cd. My brother had installed live knoppix before but that of course isn't the same thing (not to knock knoppix, its still one of my favorites). The ease of installation was amazing. It recognized all of the drives on the computer immediately and I was able to use the 'quick' install. No problems at all. It installed quickly and was much easier than Fedora or Windows hands down. 

When I arrived at my desktop I was able to get flash working immediately in Firefox. I installed Opera as it is my preferred browser but was unable to get flash to work. A quick trip to these forums (via google of course) was able to solve that problem.

Now with that problem solved I decided I needed some tunes. The mp3 codec autoinstalled with propmts. That made my day. Another quick trip to these forums and I had the advice to get my multimedia apps up to date. 

With very little work I was able to get my ntfs filesystem mounted via command line. I then realized this can be acomplished by clicking only, but even tho ubuntu is 'easy' I always like to know the command line equivilant.

The one thing that I had mixed feelings about was the pre-installed sudo. I like having my root priviliges as # also. I had gone afk for some of the install so i didnt know if i had been prompted for a root pw or not. I was able to set the pass and had then had root and sudo access. This was much easier than setting root during install then setting sudo later. Again I was impressed.

I was able to install wine with no dependency problems. Same with all the multimedia stuff. The forums are helpful. The OS is compact and tidy. So far its the best OS I've used.
