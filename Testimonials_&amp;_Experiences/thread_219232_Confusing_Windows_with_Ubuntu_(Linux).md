### Post by wahman143 on 2006-07-19
Hi all,
I know I'm relatively new to the boards here, but I've been trying to make it a point to help out as much as possible (gotta give back to the community!) since I've jumped aboard.  I've been using Gentoo Linux for about 2 years now, and while I still use that as my main desktop, I've grown to love (K)ubuntu on my work laptop and the server install as my server (duh!).

Anyway, while assisting the last few days, it's kinda ruffled my feathers to see so many people complaining about how things "don't work like in Windows...", comments that try to equate Ubuntu (or linux) to Windows in any way, or how this distro is not "n00b friendly".

1.  Linux is NOT Windows - never has been, never will be.  There is no way to equate the two, and you are foolish if you are under that impression.  There are things that are similar, but they are two different beasts.
2.  Things DO "just work" in Linux if you take time to read before installing and also take time to read up on what you are getting into.  While I think Ubuntu is one of the easiest distros that I've ever come across, I still find myself reading the Wiki and the Docs to make sure I'm not doing something wrong.
3.  Not N00b Friendly?  Are you kidding?  You sure you didn't download OpenBSD or something?  IMHO, the stark reality of it is this - if Ubuntu is not easy enough for you, then maybe you should revisit your reasons for being around.  

I don't mean this as a bash to the flocks of people who are switching, nor do I mean it as a deterrent to folks interested in doing so - I guess my point is this - they are two different operating systems and it's almost like learning a new language...there is a curve, and it will take time...and things MAY not work exactly as planned initially.

I really like this community and I hope I can stick around for a while...to learn and help others learn.  I think Linux is the greatest thing since sliced bread, and I'd like to show others that as well.  However, just remember...

LINUX IS NOT WINDOWS :lol:

Cheers,
Wah
