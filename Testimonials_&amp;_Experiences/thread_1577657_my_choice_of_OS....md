### Post by blwizard on 2010-09-19
Only thing that I wanted that Ubuntu couldn't do is games, but now that I've got a PS3, that's no longer a problem. 

It's been years since I started to feel that Linux was the only choice of OS for me (except for handling games of course). Eclipse, netbeans, python, Java, latex, shell, weka, mallet and vim are all the things I need, and they are easier to set up and to use on Linux than on Windows. I once tried Windows xp on my girlfriend's old laptop for a bit of assignment work and that experience was awful. It's hard to put in words how awful it was, I just felt the way of getting things done on windows was just not as straightforward, too many annoying clicks and restarting and stuff. I had to restart the system after installing adobe reader, I was like wtf? I'd already known windows can restart a lot (used to be a long time windows user myself), but actually experiencing those annoying restarts after I'd been using linux for so long was so different from just knowing it if you get what I mean. The lack of a real terminal program on Windows is really a pain in the **** if you're used to do things the *nix way. Windows is just not elegant and good enough.

Anyway, starting using Linux 10 years ago was the best choice I made and I couldn't feel better when trying to compare Linux to Windows. Ubuntu makes it even easier for me, less time wasted on setting things up and very good community support. Two thumbs up.
