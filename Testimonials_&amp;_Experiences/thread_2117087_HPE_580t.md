### Post by pfeiffep on 2013-02-17
After breathing new life into and old Dell laptop, I decided to install on my desktop!:D 
[LIST]
[*]Intel Core i7 3.2 Ghz
[*]12 Gb memory
[*]SSD 300 Gb (Win7 64 bit)
[*]HD  1 Tb
[*]AMD Radeon 6450 graphics
[/LIST]
12.10 installation was really smooth (on the 1 Tb internal hd), but I noticed it was only 32 bit since I installed from a DVD included with Ubuntu Unleashed (2013 ed). I downloaded the 64 bit version and installed it (wiping out the 32 bit version). 

Had a minor issue with 'About this computer' recognizing graphics which was easily remedied on Ask Ubuntu 
[http://askubuntu.com/questions/257062/does-version-12-10-support-amd-graphics-hd-6450]("http://askubuntu.com/questions/257062/does-version-12-10-support-amd-graphics-hd-6450")

Now everything is working as expected - and one unexpected surprise - after setting bios to boot from second physical drive grub provides the option to boot Win7 in addition to Ubuntu.

I'm one happy camper)):P
