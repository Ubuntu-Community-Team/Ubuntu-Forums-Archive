### Post by los_pepes on 2007-01-29
Howdy yall, first post...

I've been in the industry for several years, and have used every flavor of Windows, many distros of UNIX (Solaris, HPUX, Aix, etc) as well as some mainframe stuff, so I'm not really a newbie. As far as Linux, I've done a lot of web admin with Slackware, Fedora/Red Hat, and a little Debian. 

With the recent release of Vista, I, like many others on here, have simply decided that I'm just plain done with Windows... it's more of a feeling than a reason. Microsoft has done a lot of things right over the years (I personally like the .NET studio), but they've also alienated a lot of users and come off like some serious tool-boxes. So last week I wiped XP pro off of my Dell 600m laptop, and threw in an Ubuntu disk.

Wow!

I really, really, REALLY, like this OS; at the rate it's going, in a couple years I believe it will be the first serious threat to the Micro$oft monopoly. Apple (OSX) makes some great stuff, but let's be serious: a nice Apple system costs A LOT of money, and there is really no way around that. I can't get over how fast this OS has evolved. It used to take a lot of time and effort to get all of the hardware in a vanilla desktop to work correctly with Linux, and now I throw in a disk, and ALL of the hardware in my LAPTOP (bluetooth, mousepad, wireless, screen) works perfectly! I think I had to edit one lousy file to get my internal wireless working- piece of cake!

The GUI package manager is the bomb, and the apt-get command utility is what steered me away from the Red Hat .rpm setup towards Debian/Ubuntu-it really couldn't be any easier to install packages. 

Anyway, just saying hi, hope to be a regular.

-Los Pepes
