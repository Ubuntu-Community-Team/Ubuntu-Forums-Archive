### Post by got awesome? on 2007-12-09
I just nuked my first installation of Ubuntu this week. I had a good time setting it all up and eventually had NX, Samba, Apache, MySQL, PHP and a few other misc apps all running nicely. I then had a bit of a hard time setting up my graphics card (ATI) and was struggling with a very long boot time for which no amount of usplash.conf editing seemed to help. Anyway, I kinda wanted to do a clean install and set it all up again, just to get a bit more of a feel for it.

Which I have done! I now have a new install of Ubuntu, I got my graphics card working first this time (and it works well, much faster than it did previously with the Ubuntu driver, just had to do a minor xorg.conf edit), I now have Compiz working (woaah!!), NX Server (really good remote desktop program, much faster than VNC), and I'm working on the web server stuff tonight and Samba next.

I'm quite comfortable in the terminal now. I don't know all the commands yet, but a quick google search generally turns up the syntax, and I'm remembering more and more as I use them.

I will retain a copy of XP on my other machine (although I'm clean installing, and dual booting another Ubuntu on it), but I have a lot of Windows programs that I'll need to find equivalents for in Ubuntu (which I doubt will be difficulty) and all my precious games, for which unfortunately no substitute exists just yet, but that's OK for the moment, and here's hoping.

All in all I'm very impressed so far and eager to learn more, and I'd like to say thanks to everyone on this forum for being so helpful! :mrgreen:
