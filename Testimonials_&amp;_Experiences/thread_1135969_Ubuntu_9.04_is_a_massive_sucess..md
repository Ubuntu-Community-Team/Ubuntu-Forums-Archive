### Post by eilios on 2009-04-24
While the changes are not colossal, the small things really add up to a large improvement.

First, the boot up. Without ext4, I can boot up in 17 seconds including the BIOS 3 second thing. That is just great, it beats vista, and xp. Once you boot up, you are greeted with an amazing login screen. It's just breathtaking, it looks that amazing. The interface changes are great as well, administration is much better IMO, and then there is Ubuntu itself. It's so easy to customize, I set up accounts for my family and configured them to their preference, as well as a test to show off to my friends(it looks exactly like mac os x). Overall, Ubuntu is an extremely pleasant experience, even though there is a bit of tweaking to get it running, once you get it running you barely need to do anything. It just works, and that's more then you can say with Windows. You have more time to tweak, install programs, and just use your computer, not maintain it. If you are reading this from windows/mac os x/BeOS/ReactOS/BSD/Solaris/whatever you're running, you should really consider switching. It's definately something I would hate to be without.
