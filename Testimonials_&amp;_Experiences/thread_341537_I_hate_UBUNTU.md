### Post by 3rdalbum on 2007-01-20
> **pandisv said:**
> I don't know anything about Windows, but how on earth is this possible? When the kernel crashes, it crashes because of a bug in itself ... Surely if you could make the kernel "do something when it crashes", you might as well make it not crash in the first place.

Not true - you can program the kernel so that it jumps to a part of the code when unexpected things happen. This is how Linux can print out the words "kernel panic" and generate an "oops" report when it crashes. This is also how Mac OS X can display a picture on the screen when a panic happens, and how Windows can blue-screen or restart the computer when it crashes.
