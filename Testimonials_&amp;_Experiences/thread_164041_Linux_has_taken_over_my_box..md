### Post by last_one on 2006-04-22
I've been playin a bit with linux, first with damn small linux, then eventually working my way up to fedora core.  Unfortuneately I found problems with both of them. DSL did what I wanted it to while I was still mostly a windows person, letting me get a little more involved in the linux world.  After seeing what I could do with linux I resized my partitions and downloaded fedora core 4.  Although FC4 would be great for a business person, I'm neither a programer or a businessman. (Fedora Core 5 really ruined the fedora series.)

I then stumbled on ubuntu.  I formatted the fedora core installation and installed ubuntu.  At first I thought grub would work like fedora's grub setup (I was wrong) and I almost lost my windows partition.  After resetting my computer by installing a boot menu I was ready to try again.  Install was annoying as ever but it finally worked.

Ubuntu is the easiest OS to use by far in the linux world.  All the programs I need and all the dependencies I could ever have to search for are in the install program menu.  I no longer have to search for dependencies in 5 disks or hunt them down on the web.  It still took a little getting used to after comming off of fedora core but I love it now.
