### Post by Blue1k on 2005-10-13
Well I was using Ubuntu pre-breezy for a while now and was far very impressed, so I tried out the new Kubuntu.

Here are my griefs so far:

Can't access my Windows partition (set up properly in fstab)
Won't properly mount my USB stick
Adept sometimes takes a few times to load properly.

To be fair, it was just released so there may be some bugs to sort but first impressions are the most important ;) 
Other than these bugs I have found I have to tip my hat to the absolute beatiful design the Kubuntu team has produced. :)

edit: I figured out the NTFS problem. Fstab directory was not created properly during install
