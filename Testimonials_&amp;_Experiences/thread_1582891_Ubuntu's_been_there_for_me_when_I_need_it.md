### Post by Ocarina654 on 2010-09-27
I've been a Windows user for most of my short 21 years, but sometime back in High School a friend introduced me to Linux via a Knoppix live CD.

Long story short, I found Ubuntu and since then it's been my goto help for any computer issues.

The most recent one being a reinstall of Windows XP on the HD I've been dual-booting XP and 10.04 from recently.  This caused some problems with grub and my dual booting scheme.
(In case anyone's curious  [THIS]("https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows#Recovery Using the Ubuntu Desktop/Live CD (RECOMMENDED)") is what I used to fix my problem).

Once again, making a long story short, because of the *Excellent* documentation and Ubuntu's wonderful Live CD I was able to overcome the problem and am once again happily using Ubuntu 10.04 for my daily computing tasks.

So in other words, thanks Ubuntu and all who've made it possible throughout the years!  I doubt anything else will ever be able to take me away from Ubuntu because it's always been so good to me.

Thank you!
