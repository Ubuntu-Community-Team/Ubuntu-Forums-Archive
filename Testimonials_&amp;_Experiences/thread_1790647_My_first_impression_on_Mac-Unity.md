### Post by rin67630 on 2011-06-25
I updated to 11.04 a month ago.
Unity was not stable a the beginning, had trouble with the second screen that displayed nothing (GMA-X3100)
I switched back to gnome and immediately felt better, a lot of things were much more stable.
After some time I gave Unity another try and it seems now much more stable.
I was using the skin "Dust", which does not show well on Unity since the menus are displayed in the same color for background and foreground.

Some other skins like Radiance do not harmonize well either.

It seems that Unity aims now more at the Mac than at Windows, Isn't it?
The dock, the menu in the top-bar, the Mac is back... When Apple discontinues the Mac to concentrate on their gadget business, Ubuntu will be ready to take over... :p 

It takes some time to adapt to the new concepts, but it seems to be usable. Not a too bad start for that new baby...
Laszlo
