### Post by sudodus on 2012-11-27
I started with Ubuntu 7.10 way back in 2008, and upgraded to 8.04 LTS rather soon. I stayed with it until the next LTS version, Ubuntu 10.04, and now I have upgraded to 12.04 LTS for my main computer.

It took months to check and/or fix everything to work in the new environment, but now I seldom boot into 10.04, so I would say the migration is complete.

I'm very happy with 12.04, but I don't run the vanilla flavour (Unity) very often. I have installed 'the other desktop environments' too:

kubuntu-desktop
lubuntu-desktop
xubuntu-desktop

and I find myself running 'Lubuntu' most of the time because it is fast on my aging computers. This way I have several tools in addition to the standard ones delivered with the Lubuntu iso file.

     -o-

I have also made a portable Lubuntu + xubuntu-desktop system, installed to a USB3 pen drive (3 to get fast flash hardware but booting via USB2 ports). The hardware detection is improved with the built-in drivers, so it gives me a fully up-to-date and tweaked installed system in my pocket. I have also cloned it to a hard drive in an old laptop and to an SSD in an eSATA box, where it is also performs well in most computers.

Updating via USB is extremely slow, so I update the portable system via the HDD or SSD and clone it to the USB drive (using dd, the 'disk destroyer' that we are discouraged to write about nowadays). I recommend Clonezilla for cloning.
