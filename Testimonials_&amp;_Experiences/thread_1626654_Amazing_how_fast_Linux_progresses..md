### Post by kaldor on 2010-11-20
A friend of mine has a very old laptop. It was "Designed for Windows 98" and her brother installed a fresh copy of Windows XP on it last year. Within a few months, the usual Windows issues started to happen:

- Slow downs
- Malware
- Stuff not working as it should

Her brother took the "I'm not touching that old piece of ****" approach, so I had to fix it. She wanted Linux like my laptop. No distro worked except Fedora. Fedora was a mess and it frustrated her. 

Then Karmic came out: It almost worked. She has been using this for over a year, and it's been giving her a load of annoyances:

- Audio not working
- ATI Graphics card not working
- Flash useless FPS
- etc

She's going away for 6 months (or more) and I decided to upgrade her to Lucid since it'll be supported for a long time. I was amazed that EVERYTHING worked out of the box and no issues from Karmic were present.

9.04 = Wouldn't boot
9.10 = Lots of issues
10.04 = Instant compositing and wireless.

The only thing I had to do with the installation was remove some things such as Ubuntu One and such to speed it up. It's overall a good result and I'm amazed that it's progressed so fast.

Her 12 year old laptop finally has life again. The only thing she misses from Windows is Windows Live Messenger. Emesene is as close as it gets, but it's still not the same.

Considering that all she does is use a web browser and IM applications, Linux works. She's even learned to use apt-get to install applications, which I'm a bit surprised at. 

She's nowhere near a geek, but she's getting the handle on the OS very well. She says she prefers the GNOME 2-panel layout to Windows because it's "less cluttered".

I'm a Linux user myself (No Windows since early-mid 2008), but it always amazes me how much can be fixed in a few short months.
