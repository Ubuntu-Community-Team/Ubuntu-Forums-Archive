### Post by Riffer on 2008-08-11
1: Depends on a few things, age of machine, what you're using it for etc.  I use Ubuntu 8.04 and it works well for me.  LinuxMint, OpenSuse also work well.  If you Have an older machine DreamLinux is good.

2:  My understanding is Knoppix is used primarily from the CD, and whaile it can be installed its not easy to.  Unlike Ubuntu which has probably the easiest install around.

3: I believe that there are to types of versions for Ubuntu, Desktop and Server.  With each version there are 2 types of install disks, LiveCD and alternate text install.  You would want the Desktop LiveCD and test drive Ubuntu.  If everything works you know you have a very good chance that it will work if you install.

4:  As for a comparison, thats hard to do because everyone has their own preferences.  Every OS does things a bit differently from the others.  Having said that, Ubuntu and the Gnome desktop (IMO) offers sensible security along with an easily and highly configurable desktop. And with a multitude of applets and other programs you can make your desktop exactly the way YOU want it.

5:  As others have said you can dual boot, which is a great way to try out Ubuntu.  I did this when I first started and after awhile I found that I never used windows so I got rid of it.

Switching to any new OS will require a learning curve, you will get frustrated, you will break things.  And thats OK, its part of the fun.  I would though make sure that I backed up my sensitive files, either on there own separate partition or on a usb stick (or the like).

Hope that helps, Good Luck.
