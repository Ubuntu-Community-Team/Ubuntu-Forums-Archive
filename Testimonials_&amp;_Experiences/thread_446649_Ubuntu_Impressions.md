### Post by Altarbo on 2007-05-17
I finished installing Ubuntu about four days ago and wanted to share my experience.  Partly so that new users could read it and say, "So that's Ubuntu, huh?  I'm now a more knowledgeable, powerful person," but mainly I just needed to write my thoughts out of my head.

I have a spare computer, and it's been slowly dying.  It's old, so any time one of my family members (or friends who don't have internet) want to download music illegally, pirate software, or try out some shady Korean game; I ask them to do so on that computer.  This has progressively added more and more spyware and viruses to the computer, and a few weeks ago I decided I was going to have to wipe the harddrive and reinstall everything.

The only windows installer I had was for 2000.  I wasn't too keen on dropping $100 to get another copy of Windows XP.  The place I bought the computer from had gone out of business.  And I had heard a lot interesting things about OSS operating systems.  I spent a few days doing research, and decided that the ones I wanted to try were: AROS (amiga replacement OS), DragonflyBSD, FreeBSD, and Slackware.

I actually like AROS a lot.  Later in the summer if I have time, I might try and set up my computer to dual boot with AROS.  The problem was that it was only source-compatible for x86 (I think that's the name for the Intel processor family for PC) computers.  Very little new work was underway and I didn't want to have to buy new hardware to use obscure, no-warranty, no-support software from 1995.  

Then I tried the different *nixes.  Dragonfly sounded good, but the documentation I found was very technical.  I got x going and had a gui for my desktop, but no sound, or printers, or anything, and the amount of work it took to get a gui made me hesitant to try and go any further.  So then I tried Desktop BSD and Slackware.  These were way easier, but I still felt like too technical for me.  I have never used Unix before in my life, and all the docs were talking about using the commandline like it was their first language.

So then I started looking at Linux distros for people who do not have Unix knowledge.  This led me to Ubuntu.  I downloaded the LiveCD, liked the way it looked, and installed it.  I have to say, I really like what I've seen so far.

It was very easy to  set up my printer, and the drivers for everything else didn't even need setting up.  I then went to the add/remove programs section to take off all the software I didn't need (my harddrive is small and does not have room for waste.) and was shocked to see how easy it was to download most of the packages I needed directly from the Add/Remove menu.

After I installed all of the software I wanted from here, I started looking for the packages to install Links (browser) and LyX.  The LyX package was dependent on like thirty other packages, so that took a while to install.  It was much friendlier than Slackware though.  In Slack, whenever you download a package, it just lets you install whatever, lets you run it, and then devastates your machine.  The Ubuntu package manager, would let me download any package; but if it was dependant, it would say so along with the name of what it was dependant on.

After I had all the software I wanted, I started creating launchers for things I use the most on the top panel.  I then looked through my preferences (don't remember the actual name) and found a simple graphical tool to change fonts for GNOME.  I increased the size of the fonts to 12 and turned off the blur.  I have to say I really enjoy GNOME.  I do not need the ability to program every aspect of my desktop; yes, that's freedom, but in my hands it's just the freedom to crash the machine.  I do need something that can launch applications in a way that is fast and easy for me.  That is GNOME.

I don't think I'd reccomend Ubuntu to someone with no computer prior knowledge, but I would reccommend it to anyone with a decent understanding of computers.  The only software it's lacking are Photoshop and Games (I know it has some, but Windows is far ahead.).  I think I really prefer Ubuntu for everything else.  If anyone is reading this and thinking about trying Linux, I'd reccomend you start downloading the Ubuntu LiveCD right now; you can try Ubuntu without altering any files or setting on your computer, and if you like it, you can install from the CD.  If anyone who's helped with the project reads this I'd like to thank you for creating this; without it, I'd be shelling out $100 for weaker software because I'm too green for *nix.
