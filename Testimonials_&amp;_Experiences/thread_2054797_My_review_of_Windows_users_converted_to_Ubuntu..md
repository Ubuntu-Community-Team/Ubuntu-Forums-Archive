### Post by ruinairas on 2012-09-08
Okay. Today I will be voicing out some of the most COMMON problems I have seen with my own two eyes that lead potential Ubuntu users right back to Windows faster than they came. I will be talking as though I were these type of users.

-------------------------------------

So, I hear that this Ubuntu is a fast reliable operating system free from virus's and such. I suppose I should give it a try especially since my trustful friend strongly recommends it.

Okay, so now I managed to install Ubuntu..Wow neat this looks really good. I never knew Linux has gone this far. I almost like this better than Windows already.

Lets take a look at the software center. Wow, most of the apps I could ever use are here, that is good to know. Wait, there is one that I can't find..Why isn't Gimp 2.8 on here? Oh wait I see I haven't updated my system yet. (Updates the system)

Well, it's still not here. I thought all my packages were updated automatically? I should google this.

Oh I see, I must add a PPA, this looks easy. I'm not much of the CMD type so I'll follow this GUI instructions. I go to Software Center>Edit Sources>Other Sources and click add. Well, jeez that was easy.

Now I search again, well...it's still Gimp 2.6...whats up with this? Oh wait, the guide says I need to go to my update manager and click "check"...wait..that is a bit silly..eh, oh well I guess it works.

Oh nice, I see now here is Gimp 2.8. Awesome! I'm really digging this.

My friend said that I can run some of my games on Linux using wine, if I can't find it on Ubuntu. Well, lets see here. I go to winehq.org to find a game on the platinum list.

Oh great! The sims 3 is on there! I follow the instructions and get the game installed and running, but wait...Why is the unity menu and the top bar still showing? Thats fine I suppose. I guess I can understand the possible issues here. We are after all running a game for windows on linux. This type of thing is expected. I'll just close out of this once the game loads.

Ahh, what the heck what happend to my window indicators..I can't close, maximize, or minimize any of my windows..they aren't there!?!? I should call up my tech friend..

Okay, so he says that the GTK-WINDOW-DECORATOR has crashed..whatever that is. He gave me some simple instructions and we are back in progress.

I'll try to avoid using wine, that wasn't a very good experience. I'll go ahead and give a few open source games a try..Lets see..oh supertuxkart appears to be popular..and is a clone of one of my favorite games...lets install it!

Okay, it's installed lets play it...wait...whats with this..the unity menu is still there along with the top bar...okay? wtf is going on here..Maybe it's because I haven't restarted my computer after the previous crash.

Okay, Something really weird happend while restarting. Text was appearing all over the splash screen. At least it still booted, it wasn't fatal.

I try to launch the game again, to run into the same issue again. I also seem to have a extremely oversensitive touchpad response. What the heck?!?! 

After I manage to close the game the window decorator has crashed again...Okay, this is definitely not as stable as people have claimed this to be.

I think I'll just call up my tech friend and have him install Windows. I really want to use Ubuntu, I like the concept, but it looks way to advanced for me. I'm certain there is a easy solution to my problems, but for me I can't be fighting the computer like this. I just can't trust a system that has problems almost as soon as I started using it. My workflow requires more than just web browsing and talking on Facebook.

---------------------------

Ubuntu and Linux in general have many benefits over Windows, however, the problems with Ubuntu take that advantage and throws it out the Window. Lets open our minds more and try to think of ways to share that glory with the "casual users" too! :)

I'm sorry Ubuntu users, we just have to face these facts. The many problems with Ubuntu (Even if we don't run into them because it's not our workflow) are still there, and it's very unattractive to the targeted audience.

The two most common ones are oversensitive touchpad in fullscreen apps, and anything that requires changing the current resolution causes the Unity menu and top bar to show. (That is if you are using a computer with a higher resolution than 800x600).***

The most common response I get is "WELL GO TO WINDOWS, or DUAL BOOT". Which is just silly since I ***want ***to use Ubuntu, this type of attitude will just scare your current users away.
