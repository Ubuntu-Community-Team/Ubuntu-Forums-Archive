### Post by breadbin on 2008-06-04
Hiya, I've been dual booting with windows for a year or so but recently tried a solo boot of hardy heron that I got on a magazine. After getting the modem working I was delighted. thats the main thing I had trouble with before. i used to use mandriva 2007 and the 64 bit drivers drove me nuts. especially with wine.

The thing that got my goat today was messing around with Nvidia drivers. I could not get Quake 4 working. I had the ubuntu package installed and then installed the .run file from nvidia to update them. read somewhere that should stick to ubuntu packages so uninstalled the Nvidia.run drivers and replaced them with the ubuntu package and the hassle I had trying to get the desktop back was unreal. as it turns out i reinstalled the nvidia.run drivers after and surprisingly quake 4 started working again so i'm still here, hopefully for the long haul. 

The thing i like about using Linux coming from a windows user, is the satisfaction I get when I fix something. theres loads of help out there and usually its an easy fix, sometimes its the straw that breaks the camels back though.

I really like hardy heron now and hope to keep using it. the fact I have the internet connection working is massive. 

just my rant for the day:)
