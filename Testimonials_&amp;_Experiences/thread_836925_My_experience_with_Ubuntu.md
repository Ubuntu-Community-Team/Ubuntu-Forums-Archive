### Post by rossjman1 on 2008-06-22
I have been using Linux on and off for a few years now, so let me start by explaining my background with Linux. I started using Linux back in 2001. My reason for switching to Linux verses my Windows 98se system was because I heard it was much faster and more stable. So I started my Linux adventure with Knoppix Live CD and after a few days I did a HDD install. Among other things, my sound and printer were not supported. From Knoppix I moved to Debian with the same results. From Debian I moved to Ubuntu, which only had been out for a few months. I still had the same issues with sound and printing, so that ended my Linux adventure on my old computer from 1999.
The computer I'm using now is a Lenovo(IBM) Thinkpad T60. After using Windows XP on it for 6 months or so, I started dual booting Windows and Ubuntu, my first time doing such a thing. Installation was a breeze compared with my past Linux (and Windows) experiences. It took some uber hax (dpkg-reconfigure) to get my ATI Mobility Radeon card to get a decent display, from there I installed the proprietary drivers and started messing around with the unstable beryl. This was all well and good, but I had no real reason to use Ubuntu over Windows. Windows supported more of my hardware and applications, so there was just no point to boot into Ubuntu over Windows. Well, this brings us to my current experience with Ubuntu (8.04 LTS - do they even do those weird names anymore eg. Breezy Badger?).
I decided to install Ubuntu again on a whim, because my XP was being slow and clunky (for no reason, no virii, spyware, all defraged, etc). I popped in the live cd for Ubuntu version 7.10, and installed (graphically!) and updated through the Update Manager. Sound, video, EVERYTHING!... just worked. Ubuntu recognized my wireless card and network, so all I had to do was type the password. I found something good in Ubuntu that I have never seen firsthand in a Linux distro (previous versions of Ubuntu included), my mp3s played without editing my sources.list and installing some libraries. My second test was to burn a cd. Oh! now I have to install the codecs? WTF. Why doesnt Ubuntu need the mp3 codecs to play the files, just needs them to burn mp3s to a cd?
I remember I had Beryl running on this laptop a couple years earlier, and Beryl, while not so stable, was running decently on my computer. I liked the desktop cube ;). Now that Beryl has merged with Compiz and now that Compiz is included by default with the 8.04 release of Ubuntu, I figured that it must be better and more stable than before. So, I try to enable it in the Appearance window, and a helpful dialog pops up telling me that I have to install the proprietary ATI drivers. This was a pain in the *** a couple of years ago, but it's a breeze now (this release should have been called Breezy Badger). I turned on full effects, and changed the settings to my liking. Enabling the desktop cube does nothing, and now I cant change virtual desktops. To this day I still have not figured it out, but it's not a big deal since I have duel monitors and rarely use the virtual desktops.
Next to set up is duel monitors. My primary monitor is the one on my notebook, the other is a somewhat old NEC flatscreen. In Windows, setting this up was easy, but the maximum resolution was 800x600. Before I could do this in Ubuntu, I needed to search the forums. It appeared that I needed to install ATI Catalyst Control Center. After installing, I got my duel monitors set up and with a better resolution than in Windows.
I am very happy with my switch to Ubuntu so far, I can do almost everything that I did in Windows in Ubuntu. The one thing that Ubuntu has some catching up to do is gaming. Luckily, I am not a big PC gamer. I basically only played 2 games in Windows: Wolfenstein: Enemy Territory and SimCity 4. I have the Linux ET version running, but setting up SimCity 4 in Wine is very confusing, and I needed to follow a guide to set it up (had to manually make some registry entries). I was not expecting to write as much as I did, so I'm sorry to all the readers who were not expecting to read as much as you did. :)

Now to a couple simple suggestions: 
1) If I choose to use the proprietary ATI drivers, please automatically install the Catalyst Control Center, or at least tell me I should.
2) Don't include so many applications. Do we really need 3 different graphical shells, 2 different email applications, and 2 different cd burning applications. Or at least allow a "clean install" option that only includes GNOME, and the System Configuration applications?
