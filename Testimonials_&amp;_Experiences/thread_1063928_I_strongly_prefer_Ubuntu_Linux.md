### Post by glenngds2007 on 2009-02-08
I have two identical computers, built them my self so I know they are identical. One uses XP and One uses Ubuntu Linux.  The XP computer takes about 30 seconds to boot up to a desktop and so does the Linux computer. The difference is that the Linux computer is immediately usable after the desktop comes up, that is you can start programs and open the apps menu and so on. The XP computer takes about an extra minute before you can do the same.  That alone is reason enough to choose Linux and not XP. XP is much faster than Vista all the way around and Ubuntu Linux is somewhat faster than XP.:rolleyes:  Ubuntu Linux does not get gunked up the way XP does either.:KS

Another reason to not like windows(any version) is that on a PC that is used to any at all, it gets all gunked up.  To access 6 different IM networks in windows most people have 6 different IM programs running,  To access the same 6 IM networks in Linux you have one program to handle it all.
