### Post by nbotticelli on 2009-04-12
I recently attended and presented at the FOSSVT (Free Open Source Software in Vermont) conference in Fairlee, Vermont. It was a conference concentrated around using FOSS in education.

I am the network lab supervisor for a small elementary school in Southern Vermont and last summer created a summer camp where 5th and 6th grade students learned how to tear apart and build computers from scratch and then install and use Edubuntu. 

I basically presented on my experiences with the camp and using F/LOSS in my computer lab and in the school and how much of a boon to creativity it has become to be able to utilize open source in the school.

Overall the conference was really great and there were some FOSS "stars" there including the people who created the Sugar platform for the OLPC xo laptop which is now separated from the xo and can be run straight from a USB stick and practically run on any computer. It was really an amazing experience to be able to speak with these people and get to meet them in real life in my on state!

More information can be found about the conference on wwww.ncose.org (National Center for Open Source in Education) about the conference and hopefully soon there will be "podcast" recordings of all the workshops that went on.

It is very exciting to see schools and educators from all over the state (a rural state too!) start to use and get excited about using open source!
