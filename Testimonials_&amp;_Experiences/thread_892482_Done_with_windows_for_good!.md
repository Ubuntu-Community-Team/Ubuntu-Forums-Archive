### Post by helwitch on 2008-08-17
The last three things I ever booted windows for were:
[LIST]
[*]when I needed internet access through my cell phone via USB. This is SOLVED now, thanks to the usb-rndis-lite program from the synce folks!
[*]when I needed to run a windows mobile exe file to extract the cab from it, for installing to my windows mobile phone. Even when I was booted into windows and could 'properly' connect the phone via activesync, my preference was still to just extract the cab, save it, copy it to the phone, and run from the phone. This way I'd have it later. This is now SOLVED, thanks to activesync 3.7, which installs under WINE.
[*]when I need to update ms money. I don't do anything complicated, just basic stuff. I don't need the latest tax data, so I don't need MS Money 2008. Wine AppsDB says 2004 runs fine in WINE, so as soon as I find my 2004 install CD, I will NEVER NEED TO BOOT WINDOWS AGAIN, at least not foreseeably. I'm gonna keep it on my system, just in case of school stuff or something.
[/LIST]
I'm really looking forward to being completely done with windows, and happy I found solutions to my last few problems.
