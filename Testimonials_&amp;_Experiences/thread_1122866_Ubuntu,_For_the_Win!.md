### Post by tkelito on 2009-04-11
So I just want to say I love this community, I have been using Linux since version 6.06 (the only Ubuntu release so far that has been released behind schedule).  I absolutely love Linux and wish I could use it in my day to day.

Everyone operating system has problems, and that is not limited to Ubuntu, Fedora, Windows (Xp, Vista, 7), or Mac OS X for that matter.  All these OS's are going to have hardware compatibilities, it's a fact of life.  Because of this fact this forum sells me because it has all the answers and one of the best sets of free resources for computer help you will find on the internet, hands down.

Computers are my life, I deal mainly with Windows on a daily basis (Xp, Vista, Server 2003, etc) and a bit of Mac OS X for my job.  I have to be competent in running Windows and Mac to do my job.  When I come home, Linux has become my outlet, my time away from Windows and Mac.

I like to argue the point that there is no operating system that can do anything another can't, some just do certain aspects a bit better than others.

To be honest I am really not sure where I am going with this post, but I do want to say one thing -

This is one of the most professional forums, filled with some of the brightest minds in computing, I read through forum in awe on a daily basis with some of the knowledge that you all possess.  I first off want to thank all of you who contribute and help the community.  And secondly I just want to say to all the users who weren't happy with Linux for whatever reason, I give you credit for trying Linux.  You can't learn anything unless you try it for yourself.

Thanks everyone, happy computing!

-tkelito
