### Post by Chasehead on 2013-03-21
So I was trying to fix a printer issue for a friend of mine today using Ubuntu 12.04. It didn't take that long to figure out what it was, but the whole thing could have been avoided by one simple fix. What had happened is that the printer was printing 16 pages on to one page. This is an option that can be found in the printer settings menu:

[IMG]http://i45.tinypic.com/wgxahj.jpg[/IMG]

This was most likely a result of her using the scroll wheel (or trackpad scroll) to view the options below when the drop-down menu got in the way and changed it's settings on her. This is such an easy and unnoticeable mistake to make and I know I've gotten frustrated myself doing the same thing.

The scroll feature should only catch onto the menu when it has been clicked and activated. I think this is something that should be changed universally.
