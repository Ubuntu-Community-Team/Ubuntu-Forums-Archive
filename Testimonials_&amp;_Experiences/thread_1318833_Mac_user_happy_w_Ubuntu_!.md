### Post by natgab on 2009-11-07
I'm a long time Mac user, who is so far happy with his switch to Ubuntu.

I have a Power Mac w/ dual 1.8GHz G5s as my main Linux box.  I decided to change to Linux after Snow Leopard came out and left behind my still pretty powerful computer (and even the Quad dual-core G5s). I also do not like the fact that Apple still only makes two extremes: All-in-one that does not let you alter it or the very expensive Mac Pro. If they had some smaller Mac Pro w/ slower chip I would have saved up and bought that instead. :|

I actually intend to end up building a PC box (ironic );) later for my main computer to have the most common Ubuntu version.  So while I save money to make a real nice one, I have already loaded Ubuntu on my Power Mac so that I can get used to the differences and to learn how to use the different apps.

I've been running it for the last 2 weeks exclusively (no Mac OS X) and have it working as fast and reliable as it was under Leopard. So far I have only done web/email and some basic graphic work. But it all seems to work easy and make sense.

I also have to say that I am pleasantly surprised that my system running perfectly even though it is a PPC Mac, and a dual chip 64-bit version at that !  And I have Debian running on my little G3 iMac.
