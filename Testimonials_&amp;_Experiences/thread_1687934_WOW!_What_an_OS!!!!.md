### Post by mbunds on 2011-02-14
After more than 30 years of drinking Microsoft's kool-aid, I just installed 10.10 on a couple of older machines, one HP/Compaq EVO 530 compact, and one HP e-pc 42. These machines now outrun my W7 Ultimate Quad core!!!!! And what an outstanding desktop, my hat is really off to the community!

Some advice for people installing this OS:

1. Burn your .iso at your SLOWEST speed; the ubuntu installer leaves NOTHING to chance, and it is VERY sensitive to corrupted installer files and packages. I burned 4 duds which failed to install ubuntu, all the while screaming about "what junk ubuntu is" before I took advice from these forums and burned a good install disk ata a slow speed.

2. If using an LCD monitor with its VGA ports, be sure to use a VGA cable with ALL of the pins present; the ubuntu screens manager cannot detect your monitor if you use some of the VGA cables that do not present all 15 pins, and it will not present all of your monitor's possible resolutions if it can't detect it.

3. More than one try sometimes spells success when installing. My e-pc suffered from every possible install problem you can think of; would not boot from cd-rom, install failure from external cd-rom, install failure from USB drive, etc. Never give up! I discovered that, in this case, Windows 7 Ultimate appeared to create a valid bootable image on the USB drive, but only when I used an XP machine did it produce a usable image.

This OS is fun and blazing fast, and the Gnome Desktop is the most well-rounded of the big 3. KDE looks ok, but it just doesn't seem as polished for some reason.

Thanks to all of you who made this possible, and I hope to be a contributer someday!

### Post by tlcstat on 2011-02-17
Greetings,
> 2. If using an LCD monitor with its VGA ports, be sure to use a VGA cable with ALL of the pins present; the ubuntu screens manager cannot detect your monitor if you use some of the VGA cables that do not present all 15 pins, and it will not present all of your monitor's possible resolutions if it can't detect it. 

Since this post gets into older vga and cable pins I just thought you would like to know that a full pin cable may not even work. On a lot of the older systems the manufacturer would switch/crossover the pin-out on a couple of pins just to have a proprietary system. Compaq and HP were bad about this. The monitor would only work with the appropriate card and cable. Its simple! They wanted a lock on replacement parts. The same is true with power supplies. You couldn't use an after market power supply because of cross-over pin-out. The confusion never ends!
