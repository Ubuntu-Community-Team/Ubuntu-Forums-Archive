### Post by divgradcurl on 2006-09-04
I've been dabbling in Linux for about 10 years or so just for fun. But now I think I'll be spending most of my time in it.
Ubuntu is a very easy distribution to install and setup but for a few problems...
1) Getting my (soon to be replaced) Speedtouch 330 'modem' was a bit tricky.
2) Getting wmv, etc support on AMD64 was also a challenge.
But with the good HOWTO's in these forums the above were soon fixed.
That makes me 99.999% happy and 99.9999% free from windows xp (I still need it for my PODXT and some degree course software - QEMU isn't quite up to the job yet).
Ubuntu runs very smoothly and I think Firefox and Thunderbird (my faves) integrate better in Gnome than KDE (I did try Edgy Eft for a while, but too many niggles). And it's great that you can run the odd KDE app in Gnome with little trouble.
So my only real problem now is deciding on what to use Ubuntu for. With it being based on Debian, there is so much software to choose from.
I don't know 'where I want to go today' ;-) but I know that wherever I go it will be fast, functional and free.
Hopefully I'll even be able to help a few people out with my limited knowledge.

### Post by No Whammies on 2006-09-04
I've gotta agree with the original poster, especially on point #2.  i was in a bit of a quandary for my first month of Ubuntu usage.  I really liked the setup of Gnome, but there were some KDE apps that I refused to do without.  I tried loading both DEs, but that got annoying REAL fast, having to switch between the two.  Then, a post here brought to my attention that most KDE apps (like my faves K3B, AMarok and a couple others) work well in Gnome.  I gave it a try, saw that this wasn't some ol' booty chatter and rejoiced.  When I had to reinstall later (lightning storm that  did some damage to the hard drive and power - fixed, and made me glad I periodically back my stuff up on discs) I remembered my lessons and installed regular Ubuntu, my KDE faves, some extra themes I really liked and some other incidentals (you know, codecs, FreeLoader, Amule and   other stuff that I thought was nifty).  

Side note:  I was a loyal Winamp user for about 10 years, no joke.  I thought it was the best music program around, and it was...  For Windows.  Amarok outclasses Winamp in every possible way, not the least of which is the fact that I don't have to 'obtain' user keys to use the program after losing my own while housecleaning  (note:  memorize IMMEDIATELY after writing the code down to avoid embarassment and anguish!)

All in all, I kicked Windows to the curb because I was frustrated with the system overall.  I was just tired of using Windows.  I tried Ubuntu with the full expectation of growing MORE frustrated with Linux and appreciating Windows.  Guess what?  I find myself more disgusted with Windows on a daily basis while working with ubuntu.  Even the recent hiccup with the bad update didn't slow me down much (I LIKE CLI and don't mind using it). 

To bring this MUCH longer than intended speech to an end, I am loving the open-source world and am just ecstatic that Ubuntu was my first experience with a new OS (not counting that Apple 2E back in Junior High).
