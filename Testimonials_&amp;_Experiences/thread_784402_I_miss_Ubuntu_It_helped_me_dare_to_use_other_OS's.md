### Post by nenyalorien on 2008-05-06
[COLOR="DarkOrchid"]I first used Ubuntu Linux back in 2006, when I first got DSL connection. 

This may seem sappy and overly dramatic, but it had been my experience with Ubuntu, you see. Please bear with me. :)

Because of Ubuntu Linux, I was able to migrate from the Windows mentality of sticking to a system because it was "safe". Linux still had the stigma of being a "geek's system" even back then, in my country. 

But when I tried Ubuntu, I discovered how fun it was to try other OS's. I was happy with Ubuntu Edgy Eft, so I tried PC Linux OS. Not good enough for me. I seriously got turned off with the crazy Merlin and dragon hung on the moon graphics on some message/system dialogue boxes.

Though my computer at that time never could handle Feisty Fawn, I was able to really appreciate the system then. I think I would have appreciated it more with a higher resolution, but sadly, my CRT monitor could only handle the 800x600 (?) setting..

Eventually I had to leave my hometown, so I left my desktop, but my hard drives were sent over to me. Ubuntu is still in one of those. :) 

After Ubuntu, I was able to dare use a Mac, and I've been on it ever since. But it's been months now that I am having this deep ***_ache_*** to go back to Ubuntu. Now that I've seen screenshots and read about the new functions of Hardy Heron.....


I am now praying [-o< for a desktop to stick Ubuntu into, and eventually another Mac, so I can turn this system into a dual-boot OS X/Ubuntu Linux machine.\\:D/ [/COLOR]
