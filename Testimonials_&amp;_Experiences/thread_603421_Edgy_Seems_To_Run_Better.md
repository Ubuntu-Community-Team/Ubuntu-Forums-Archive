### Post by Vince4Amy on 2007-11-05
The machine I'm running Kubuntu on now (Not My Primary machine), has the following specification:

1GHz Intel Pentium III 
512Mb Memory
16Mb Shared Graphics Memory
40Gb Hard Disk Drive

This machine is now a Kubuntu only machine, but however for some reason Feisty & Gutsy just don't seem to run on it as stable as Edgy for example I experience the following problems with Feisty:

Problems with X
Frequent System Slowdowns
The Kernel Error which does the emergency park on hard disk drives.

Gutsy:

Lock-ups when logging off.
Shutdown problems (Shutdown Process hangs the machine sometimes).
Very slow performance or locking up when installing .deb files.

I don't know why some of the above happens, but when I run 6.10 none of this happens to me. I will not ditch Kubuntu for Windows I will continue using 6.10, but I just find it odd how a newer OS which is improved upon the previous version seems to actually have more problems than the previous release. Also to note none of these were upgrade installs, they were all fresh installed from the CD media. 

I won't bother upgrading to the newer releases on this machine, but I'm curious on why some of these things happen. I also get the logoff/shutdown error on my other machines too.
