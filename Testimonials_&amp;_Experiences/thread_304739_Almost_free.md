### Post by huntermix on 2006-11-22
I wanted to write my testimonial since the first day I installed ubuntu. I really liked what I saw but i thought it would be better for the community if i had give the distro some testing time to see the good/bad things about it.
I downloaded the Edgy release on october 26th and I must say this is EXCELLENT!
After I installed Edgy on my notebook I decided it would be the only OS that I would use there. The install was much simpler than with Windows, and the vaio even comes with recovery disks, but you have to keep switching disks and after the install and some things just don't work.
When using Windows on the vaio everything seemed slow, now with ubuntu not only looks better but it works better.
The notebook was a FE690 and Ubuntu Edgy Eft detected all the drivers (both cores,wireless card wih wep working, audio, volume up/down buttons, usb, network), i just had to install nvidia drivers and I got a fully working station, in much less time than with any other install. Only thing that wasn't detected where the FN buttons but I got smartdimmer to set the brightness so no worries for me :)
I had tried to use Linux as my main OS before(redhat,suse,kororaa,gentoo,freebsd,freespire,debian and so on) because I agree with the open source ideology but I always ended up going to the dark side after a few days. That hasn't happened with Ubuntu because I chose to use it because a simple reason: IT IS BETTER! The whole philosophy of open source and freedom came included! The other distros I had tried where good too but not for working. I really liked gentoo because I could learn a lot, now i prefer that things &#8220;just works&#8221; but if they don't THEN i can start editing text files and feeling like in swordfish, no problem.
Now I also have installed Edgy on my desktop along with an XP dual boot.
Ubuntu still need some work to be PERFECT but I think it's going in the right direction (at least with edgy). Let's see what feisty brings us!
THANK YOU FOR THIS EXCELLENT DISTRO TO ALL WHO WORKED ON UBUNTU!

P.S: I wrote this in Open Office, took the picture and ubuntu detected the camera to download the pictures, then used gthumb to see them and finally gimp did the conversion to a lower quality format. This may seem trivial, but i can keep doing things just like i did in other OS but free!
