### Post by circolo on 2008-05-28
Very, very new to Linux/Ubuntu.

But I'd been toying with the idea of installing Ubuntu on one of the spare PC's at home, so I'd already downloaded the ISO, and created the CD.

Anyway, the main home Laptop, (a Dell 1520 Inspiron) running Windows Vista, decided to corrupt it's partition table/file system. (To be honest I'm not sure which, as the fault wasn't properly diagnosed when Vista tried to recover this). In the end I ended up with a Laptop that simply refused to boot.

I have a WD my book. So using the live CD with Ubuntu, I booted up from there. Ubuntu, could see the Windows Vista Hard Drive, but didn't recognise it's partitions in the Ubuntu Partition Manager program. However, this was good enough. As I could see all the files I was afraid I might lose, if I needed to flatten the Vista install and re-install the system, which was looking very probable.

I plugged the WD my book in to the USB port on the Laptop, and Ubuntu recognised this, with no issues. Great! This mean't for the next hour or so I copied over all the files I wanted to retain from the poorly hard-drive to the WD my book drive. And I could then safely full re-partition the drive, using the Ubuntu program, and re-format it for a clean install of Vista, whilst having everything I needed on the WD my book.

So it's "thankyou" to Ubuntu for saving the day, and showing up Microsofts poor quality again.

And y'know, I'd be serious in looking at having Ubuntu as my main os if only I could:

Have Wireless access using the DELL wireless card,
Have the ability to play the Windows compatible games, (like gta, roller coaster Tycoon, etc) in Ubuntu.

That'd do it for me. So where do I go from here?
