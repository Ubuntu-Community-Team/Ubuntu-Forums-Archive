### Post by DirtDawg on 2007-02-05
Wow! For anyone who hasn't heard of it, [Xara Xtreme]("http://www.xaraxtreme.org/") is a vector illustration program that costs 80 bucks for a Windows version, but is 100% open-source and free for Linux. Now, I love Adobe Illustrator, but Xara is so well designed I can't help but think it's superior in a lot of ways (the copy of Illustrator I own is 10, so who knows, maybe it's improved with the "CS" series). In particular, the tools seems more intuitive to use in that there's no ctrl+MB1 or alt+space+MB2. And tasks such as gradient editing, once one gets used to a different system, is well thought out and a breeze to use.

I rarely gush about software like this, but I have personally found Inkscape somewhat unstable and ill-suited for my needs so Adobe Illustrator has always been one of those reasons to boot into Windows on occasion. But Xara is so well designed and seemingly stable that I think I may start booting into Linux to use it instead of Illustrator!

Possible drawbacks? I'm not sure if advanced options like equivalents to gradient meshes exist. Also, it looks as though the developers took a time machine back to 1986 to hire a graphic designer for the splash screen.

So, I'm here to say, if you're an artist who's found yourself wanting from your Inkscape experiences, try Xara.
