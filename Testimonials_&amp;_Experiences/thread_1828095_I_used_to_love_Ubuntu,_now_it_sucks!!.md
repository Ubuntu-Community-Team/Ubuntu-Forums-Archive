### Post by Dragonbite on 2011-08-18
You know what's funny about people leaving because of Unity? Chances are they'll go to another distribution, only to find that one move to Gnome 3!

A good thing about them is they are new, which means development should be going at a rapid pace and all of the "tweaks" and articles in OMG!Ubuntu make it sound like Unity is getting better and better.

I'm wishing I had a system able to handle 3D Unity or Gnome-shell. Unfortunately I end up going into fallback mode in both.

I've had mixed feeling with what I have tried (Unity 2D, Gnome Shell fallback), and have tried moving the Xfce.

So far it seems my preferred system is Fedora running KDE.  I've tried openSUSE and KDE but for some reason openSUSE just doesn't like my systems.  Also Fedora makes things easy because it finds my wireless (used to find my video but that's a lost cause).

I don't see Unity as a sole reason to leave. I mean, "fallback mode" in 11.04 is, um,... basically the Gnome everybody is used to before Unity came around.  Eventually there will be no distributions offering a Gnome that isn't Gnome 3 so there is little incentive to jump ship.

That said, why not keep with a familiar system (Ubuntu) but with a different Desktop environment (Xfce, LXDE, KDE, etc)?

Now, if Ubuntu One and the Software Store were to work with Kubuntu, then that would change which distro I run KDE on top of.
