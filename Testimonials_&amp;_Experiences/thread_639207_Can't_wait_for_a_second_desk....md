### Post by suziequzie on 2007-12-12
Hello. I've been dual booting Kubuntu and Win2K for awhile now. I've learned a lot (thanks largely to this forum, and to the many errors I've made that I had to correct) and really want to have a desktop computer that is dedicated solely to linux. I need to keep windows running on the other partition for my boyfriend (some programs he prefers to run natively in windows rather than through virtualization).  

Now, I've always wanted a second computer. Either I'm on the computer, and he's on the PS2, or vice versa. But if we both wanted to surf the net at the same time, we weren't able to.  I'd need a second computer... And now I have it.
[U][B]
NEVER THROW A STILL-FUNCTIONING PART AWAY![/B][/U]

Between upgrading mine and my sister's computers over the past 2 years, I realized I had accumulated all the parts I needed to build a second box.  And I have. I unplugged all the peripherals and chords from my main box, switched boxes on the desk, and installed Ubuntu on the new/old computer.  All I need now, is a mouse, a keyboard, speakers, and room. I plan on hitting the boxing day sales to see about getting these parts, and a small computer workstation.  I already have a router and 50' of cable. This box is only going to run Linux, specifially Ubuntu, and maybe BSD. 

I have tested this Frankenstein machine out, and it runs Ubuntu beautifully. No problems with any of my hardware, but I knew there wouldn't be: the computers these parts came from had already had linux on them before I swapped these parts out. I just can't wait to get it all set up and personalized! Tweaking the OS to be *Juuuuust* the way I like it is part of the fun. That old machine feels new again. [SIZE=1][COLOR=Red]Except for the noisy power supply... which I will eventually replace...[/COLOR][/SIZE]


Frankenstein's Machine: 
AMD XP 2200
40 GB HD
256 Ram
Nvidia 5200 128MB 8X AGP
Onboard sound and Lan
Asus  Mobo.
OS: Ubuntu Inside. And it flies!
