### Post by Godspell on 2011-01-18
I started using Ubuntu about a year and 2 months ago.
I started off with Karmic 64bit, then switched to 32bit because I felt as a beginner it'd be harder to resolve issues on the former.
Then, I upgraded to Lucid 32bit and until about 3 months ago, I switched to 64bit.
I did it because I read a message on the forum "x86 64bit Users" that 64bit is now very stable and support is also widely available. It was right btw because I've not encountered any major problem.

My experience with Ubuntu has been a wonderful ride. I'm an IT student at the University and I only have a laptop. There were quite a few problems but most of them were resolved with the help of Ubuntu Community. I still have issues like no VOIP on GoogleTalk Empathy, Spotify soundcard, headset etc. but for all these, I use Windows 7.
I will admit it would be nice to have Ubuntu on my laptop fault free and I already have made threads about these issues but in all honesty, I can put up with these.
I can do my Uni work and everyday computing on Ubuntu pretty well.
And as a Uni student, I'm still forced to use Windows especially for Visual Studio 2010.

The particular positive experience I have is the understanding of Opensource and it's view of computing. I happened to read about Ubuntu's origins and from then on, Linux and Unix history. I was pretty much blown away on how it all started (and shocked for my lack of knowledge in this area).
Before using Ubuntu, I always thought writing applications is very hard and only smart people can do so but later on, attempting to resolve the issues on it i.e. personalising the OS I would say, made me realise it's not that complicated but doable.
Also the fact that ANY software you want to use is in Software Centre or Repo for FREE is awesome. On top of that, I really enjoy using this forum because of its interactivity.

Overall, I was quite inspired and it sort of changed my mindset about computing and even my career goals. Now, I'm practising C++ whenever I'm free and would very much like to become an Opensource developer. I'm also planning to develop my final year project on Ubuntu.

I think it's safe to say I'm definitely sticking around. It's already my main OS, anyway. I feel quite independent to use it; there's no virus or spyware issue, no bsod, no anti-piracy message and best of all, hardly any freezing.

So yes, I've been using Ubuntu and I like it very much :D
Hope this is (somewhat) useful.

Regards,
GS
