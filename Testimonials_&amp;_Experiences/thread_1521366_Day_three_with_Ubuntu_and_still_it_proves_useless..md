### Post by TBABill on 2010-06-30
Not to be an insult, but you do have to learn how the OS works before judging that it may be the issue. No it is not currently configured perfectly for your machine...Ubuntu is designed for those who wish to not use proprietary drivers. They are available if you wish to use them, but by default the system does not install them for you. It's in their philosophy. Other distros include everything and may be your cup of tea since you are dealing with time issues and a young child consuming your time.

My immediate suggestion is to install either Ultimate Edition (it's just Ubuntu with everything for video and about anything else already installed, but you may still need to install your driver from Synaptic after installing the OS) or give Mint 9 a shot. Or if it doesn't have to be Ubuntu based, try PCLinuxOS....really does just work.

If you just want to use Ubuntu and have it work for you, use Synaptic to install the appropriate driver for your video card. You may need to search the forums and Google is a definite friend. But get anything you can from the repos so you won't have error issues later during updates or upgrades.

Second, install all the DVD lib files. Can't recall the exact file name, but you can NOT play DVD's without it and a couple others perhaps. Again, philosophy of Ubuntu, laws, etc.

Third, learn configuration techniques and search the forums for your answers. Most times someone else has faced and fixed similar issues. Use the threads marked [SOLVED] first because, obviously, the answer was found.

Fourth, give folks here a chance to help. Everyone has an input and is willing to invest part of their personal time to help. Be grateful for that help and appreciative that this OS is not as you described, assuming you learn enough to configure it properly. You'll learn to get your video driver set correctly, CPU scaling working to help with heat issues, sound and DVD and other video configuration, etc. It's not hard, but it is different. 

Lastly, it's not Windows. It is unique and requires learning. Did anyone know Windows configuration on their first install or use? Obviously not. Security software, network configuration, drivers and other issues are not always easy in Windows, but they SEEM easy because you are used to them. Give it a chance and ask lots of questions. In a very short time it becomes easy and normal.

Depends on the video card, but yes, Linux isn't the best choice for watching movies.

I find this hard to believe. Ubuntu is perfectly able to access Windows shares. As for the opposite direction - Samba is a bit trickier to configure, but you can set up OpenSSH server instead and use FileZilla.

> XBMC cant access my shared files on PC
XBMC uses SMB. If your Windows (or Samba) shares are properly configured, including user access permissions, XBMC will be able to access them.

What's wrong with Mplayer or VLC?

Now that's true.

> Complete lack of support
And that's not true. People receive a lot of support on this forum. It's often way harder to find a solution for some obscure Windows problem.
