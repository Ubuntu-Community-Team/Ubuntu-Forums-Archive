### Post by Welly Wu on 2012-12-02
On my System76 Lemur Ultra Thin (lemu4) notebook PC, I tried Ubuntu 12.04.1 64 bit LTS Precise Pangolin and Ubuntu 12.10 64 bit Quantal Quetzal and I have to say that Precise Pangolin was much better. It's a lot more stable and reliable. I have no problems with Linux kernel 3.2.0-34-generic whatsoever. It does not cause kernel panics or random system lock ups for my Intel 3rd Generation "Ivy Bridge" CPU or Intel HD Graphics 4000 integrated GPU either. Precise Pangolin is much faster than Quantal Quetzal too. It's much snappier and more responsive to me. I also found out that a lot of popular software packages are targeted for Precise Pangolin rather than Quantal Quetzal. ArmorForge, MyUnity, SplashTop Streamer, Steam for Linux, and some Ubuntu Forums member's custom Novell AppArmor profiles will only work with Precise Pangolin and that's not an exhaustive list at all. I'm sure there are many more software packages that are built for Precise Pangolin. I also found that Quantal Quetzal frequently crashed and broke compatibility with various software applications or packages. I had problems with the package catalogue that needed to be repaired. The other problem that I noticed about Quantal Quetzal is that the Ubuntu Unity Web Apps Desktop Integration frequently crashed with Google Chromium. It was not stable at all. This was one of the big features of Quantal Quetzal. Sadly, it is not ready for prime time. Overall, I have to say that LTS is the way to go for me. I need long term service and support more than new features that are often buggy and prone to crashes. I'm glad I learned this hard lesson and I re-installed Ubuntu 12.04.1 64 bit LTS from scratch. Now, I know what I will be missing and it does not bother me at all.
