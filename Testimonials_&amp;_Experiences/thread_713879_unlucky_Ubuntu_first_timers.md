### Post by steveneddy on 2008-03-03
The data is there if one cares to research before install.

Most people trying an Ubuntu install for the first time have never install any OS before or partitioned a hard drive. This is all new to them and they are trying to sort out problems that they have no previous knowledge of.

Unfortunately people are people and they will jump into things before they are ready.

Maybe we need a huge first time user link that talks about the beginning skills that are necessary to install on various hardware and would give links to websites that address Linux hardware compatibility, maybe even listed by computer brand and model.

This is just a reality of the Linux world and it will probably be the same for a while.
