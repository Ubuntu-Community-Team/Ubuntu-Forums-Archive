### Post by ShootEmUp on 2010-12-15
This is my third ubuntu install and I must say that ubuntu is the most user friendly linux (beats windows too if you have to install windows manually) EVER!
So to the story, I have a emachines T3642 stock model (2.6MHz, 1GB RAM, D-link DWA-130 wireless, etc.)
Vista just wasn't cutting it anymore, so I decided to try Ubuntu
Ubuntu is set up on a 10GB partition, with 2GB of swap, It is faster, lighter, stronger, more stable than Vista.
Only had one issue with it, wireless didn't work, fixed with ndiswrapper ([heres how]("http://ubuntuforums.org/showthread.php?t=1645273")).
Ubuntu has great games, Quake Live, Spring RTS engine, Openarena, Assault Cube, and many more.
I don't know how I lived so long without Ubuntu, windows was just a crutch. Ubuntu has been here for 6 years now, and hopefully many more, LONG LIVE UBUNTU! :KS:KS:KS:KS:KS
btw dual-boot because I still have a shitload of windows games.
