### Post by pete04 on 2012-05-17
Been working with 12.04 now since it went Beta in April. I can honestly say, now, that it's the best system I've ever used, hands down.

I started with Ubuntu as a way to breath life into aging computers more than two years ago, but now I don't want to use a computer without it. Now that I'm finally running it on new hardware, it's just breathtaking. Unity is the best it's been and I started this week using Gnome shell with the panel applet extension, and it's perfect.  There's so much to like in both, it's really hard to choose. The awesome thing is that I have the choice.

I'm in awe of the developers and the community that make this possible. =D>
