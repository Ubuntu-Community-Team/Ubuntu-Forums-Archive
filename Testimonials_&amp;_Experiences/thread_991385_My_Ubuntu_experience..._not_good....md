### Post by zekica on 2008-11-24
I have been using Linux for eight years, and Ubuntu for four. There was a problem you mentioned with new ext3 partition, and that you were unable to write to it. This is how it should work - normal users on unix like systems have write access only to their home folder by default, and this is why it is more difficult to create malware for Linux.

You could do that without opening terminal, only using nautilus or any other file manager, then selecting Properties from popup menu, and going to Permissions tab. The only thing you need to do is to run it as root - pressing Alt+F2 and typing: "gksudo nautilus" (without quotes).

The same thing can be said about XP where you need to type for example: "control userpasswords2" in Run dialog, if you want to setup automatic login on a system with more than one account.

All OS-es have their advantages and disadvantages, but you cannot only look at one's problems and overlook others'.

Copy a command by selecting the text and pressing Ctrl-c or Ctrl-x, hen open Terminal-


OK... there is a conflict in your requirements. 1st you say you need everything zoomed to 16:9 aspect ration. From this I understand you want cropping (either from top & bottom for original 4:3 aspects or left & right for original 2.33:1). But, doing this you will be left with no black borders at the bottom to display the subtitles bellow the movie (actually you would get some space since your monitor is more likely 16:10 and not 16:9 but... it's very little).

Anyway... enough ranting... :) Mplayer has pan&scan functionality bound to w & e keys. So... once your movie loads you can hit w/e keys to zoom in / out until you hit the required screen fill (excellent for wide movies). If you want to subtitles bellow the movies... :) there's a little trick for that too. Do
```
mplayer -vf expand=0:-75:0:0 -sub subfile.ext movie.divx
```
This will expand the movie by 75 pixels bellow and place the subtitles at the bottom of the movie. This will "move" the movie a bit up on the screen so you would get a top part with the movie and a bottom part with the subtitles. You can add this in your mplayer config file (google for it, you'll find it) so it is automatic each time you start mplayer (even the GUI one). Note that, unlike many windows players, from mplayer you can dynamically adjust sub-delays, av-delays, contrast, colors, light, etc. all while playing the movie. True, if you have perfect rips... this is not needed. But... let's face it... how many are perfect? Refer to the manual page for mplayer to check on the font settings for subtitles and, in general, all the options you have with mplayer. (Note... there are various UIs for mplayer).

You can also use VLC to crop any movie (with a nice menu) to the desired size. Just right click on the movie (while running) and select crop. However, on VLC, I am not sure on how to add subtitles bellow the movie. It's on the TODO list since 2004 but it does not look like it's done.
