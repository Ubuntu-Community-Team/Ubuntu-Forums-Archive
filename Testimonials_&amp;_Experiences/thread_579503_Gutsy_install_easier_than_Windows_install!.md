### Post by KillerSponge on 2007-10-18
I just got my new notebook today (Asus S37S). At first, I wanted it to dual-boot with Windows (for games) and Ubuntu (Gutsy).

Problem: the Windows installer wouldn't recognize my SATA harddrive without extra drivers... Which had to be loaded from a FLOPPY DRIVE. Yes, a floppy drive. Those things noone uses anymore. Of course my notebook doesn't have a floppydrive. So my options were to buy a usb floppydrive, or slipstream SATA drivers onto my XP install cd. After spending two hours en 4 wasted cd's, I had enough. I decided to just install Gutsy. 

Most flawless install ever. It recognized every piece of hardware, worked right out of the box. And people complain that Linux is hard? :P

### Post by markonhismobile on 2007-10-19
I'd have to second that Gutsy installs are easier then windows. Last night instead of upgrading my notebook from Feisty I did a clean install of Gutsy. I'd already aticipated doing these so when I installed Feisty I setup a seperate home partition.

After about 30 minutes the system had re-installed and when I rebooted all my desktop was exactly as it used to be! I launched Firefox and all my settings where still intact, same for Thunderbird once I grabbed it from the repo's. After spending a few minutes in synaptic tagging all the apps I had previously I was practically back to where I was before, except now I had a fully working Compiz Fusion without a messy sources.list (the reason I did a clean re-install) :-) All in I think I was only working on the laptop for a couple of hours,which wasn't helped by the slow downloads on the packages I needed to retrieve!

Now compare this to a windows install which would have taken at least 45-60 mnutes to complete, and then having to spend a few hours installing all my apps and drivers!
