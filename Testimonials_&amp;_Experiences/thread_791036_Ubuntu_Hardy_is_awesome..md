### Post by stchman on 2008-05-11
To all the people that sayd Hardy is the worst, Hardy is the best.  I have Hardy installed on 2 laptops and 2 desktops.

Compiz works once I enable the restricted drivers.

The 2 laptops are ATI video card equipped and Compiz works awesome.

There are a few quirks.

VLC to be default DVD player takes a little work to do.
No logoff sound(no biggie).
Missing OO 2.4 package for Java.
64 bit Java plugin.


Everything else works awesome.  I like how the keyring manager does not make you install the keyring manager.

All in all kudos to Ubuntu for a great release.
