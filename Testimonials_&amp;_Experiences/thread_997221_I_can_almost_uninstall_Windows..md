### Post by ftaylor on 2008-11-29
I started using Ubuntu when Gutsy Gibbon (7.10) was released. I am a computer scientist; programmer / developer / systems designer / everything else. 

I had never used any kind of Linux OS prior to Ubuntu. I have to say that I am massively impressed and I am now at the stage where I can almost get rid of Windows. I have managed to replace almost everything (I only keep Windows for photoshop now). 

It is **so** easy to get things done using Ubuntu. 

I don't have to contend with complicated, deep menus in order to complete a simple operation. Instead, I can ```
sudo gedit /etc/apt/apt.conf
``` and ```
synclient TouchpadOff=1
```.

I don't have to waste time trawling the internet for installers; I can go onto Google, search for "Ubuntu *perform task X*" and I will find a package in no time; which I can obtain incredibly easily with my trusty ```
sudo apt-get install xxxxx
```.

Thanks to you.

Finbarr
