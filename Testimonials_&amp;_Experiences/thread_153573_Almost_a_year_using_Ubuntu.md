### Post by woland on 2006-04-01
About the 20-something this April it would be a year since I switched to Ubuntu.

At first I was afraid to switch without the safety net, besides my HDD was a mess. Just to backup all the data I had I needed too much dvd's and time. Another issue was that if I wold need to reboot each time there is something I still cant do with Ubuntu, I'll probably stay with windows. Then my sister bought her new PC and it removed all my doubts. I had 160GB for backups and a windoze PC to fall back to if something goes wrong. 
I removed all the windows partitions except the one with the windows xp itself and installed Ubuntu Linux 5.04 (Hoary). I used it for two months, had some troubles a all newbies have, tried Fedora for a week or so, and went back Ubuntu. 
Since then I booted windows about 5 times in about 10 months. I did cheat since I've been occasionally using windows in VmWare since my sister moved out last November, taking her computer with her.
Two months ago a friend of mine installed xgl in Dapper and convinced me to do the same. During the install I broke my system, so I used this opportunity to completely repartition my HDD, and by doing this got rid of my windows partition, that I wasn't using anyway.

I'm quite happy using Ubuntu, and I definitely don't want to come back to windoze. Too bad I that couldn't make the glx work...
