### Post by rain4ever on 2008-09-23
I've been running ubuntu for a little over a year now. Mostly in virtual machines, because of wireless problems. I did eventually get wireless to run on one computer, just an older pc, not a laptop. Took me about a week and a half to get that working. So I only used virtual machines for my laptop. Loved Ubuntu but didn't have the patience. Recently I heard that the new linux kernel would support my little atheros wifi problem.I also found out that alpha five of ubuntu 8.10 had that kernel. So I but the 8.04 into my laptop, plugged my laptop to the Internet, and waited a few hours for it to update. Many problems I have had with 8.04 had been fixed with an unstable version of 8.10. I've been using the hell out of my laptop lately, and am happy to learn more and more about Ubuntu. 

I first learned about linux by a teacher but he made it sound like it was all command line stuff that I could never figure out. I got interested in it when I was complaining to a friend that I had to fix another friends windows xp computer because it had a bunch of virus's. I may have complained that my computer also crashed because I was running way too many processes for windows to handle (again). I was telling him that I was going to try and save up some money to buy a mac. I had tried a mac before, and disliked it, but not as much I hate windows. So he gave me a linux for human beings live cd.

I must have run at least 30 different linux distributions, but I came back to two that I liked, I hated the rest. I liked openSuse and Ubuntu. After a while openSuse felt too much like windows even though I could mess around with it a bit, it still felt restrictive. So I stayed with Ubuntu what was givin to me at the start. A couple of my friends liked Ubuntu as well but the process to get there games on it seemed too much for them. Now, though, people are straying more and more from windows. 

Thats seems great, but the general public isn't really educated. Most people like I was think there are only two operating systems windows and mac. Anyways I'm glad I found linux, and I'm to the point that I have my grandmother wanting to try it out. anyways thats just my experience. 

One more thing I would like to add about those that want to bash linux. Linux is not windows. It wasn't meant to be windows. People didn't switch because they were happy with windows. They switched because they didn't like it. If you like your windows then good keep your windows, we won't bash windows users like many windows users bash linux users, though most of us can't hardly be called users because we are part of the community. We don't want people to hate windows we are just spreading knowledge, and if people don't like what we have to teach them, then fine.

I'm sorry if I started a bit of a rant. Anyways I am not going to go on and say that "Ubuntu Rocks" just that Ubuntu works for me.
