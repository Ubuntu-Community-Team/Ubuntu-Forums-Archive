### Post by kherring7383 on 2012-10-23
Personally I'm a Mint user but have considered moving back to Ubuntu for a while. This last weekend I spent several hours getting it installed and had to deal with the Nvidia driver issue. Once I updated the driver, I was able to get a decent startup splash screen and good resolution. 

To see what other issues I'd find, I started to install my usual apps and over the next few hours had to research and update some  PPAs but in the end all of my apps seem to work now. I did notice an error message that pops up every now and then referring to evolution and the calendar. Since evolution might break the unity Desktop, I didn't try to remove it. I did however find an evolution package update at []("http://www.ubuntuupdates.org/package/core/quantal/main/base/evolution") and installed it and haven't seen any errors yet. Hopefully there will be some patches over the next few weeks and 12.10 will become more stable. I've been using Linux for several years now and it never fails, every time a new release comes out there are tons of problems but over time the community fixes them and they get updated. So for all you noobs, just stick with it. This is normal and will get better over time. At least it's not like Windows and having to depend on Microsoft to fix the problems as they see fit and post service packs and updates on their schedule.

### Post by Mark Phelps on 2012-10-23
Those of us who have been around Linux for a few years have learned, in some cases the hard way, that it takes a while for a new version to work through the "bugs".  

Case in point with 12.10 is the rash of posts we've been seeing about the AMD restricted drivers not working.

It's generally better, since there is no way to do a "rollback" from the new version to an older version, to either install in a separate location, or wait a few weeks, monitoring the forums, to see what problems get reported, and which ones get fixed.

I've only been using 12.10 a couple of days, and while it IS slower than 12.04 (on the same hardware), I've not seen any stability problems so far.
