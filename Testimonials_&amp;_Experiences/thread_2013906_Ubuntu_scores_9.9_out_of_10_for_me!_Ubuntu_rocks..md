### Post by goodbye-windows(tm) on 2012-07-01
Ubuntu is great!!! With the help I've gotten from this forum and google searches on the internet, I've managed to overcome all my technical problems,......with the exception of 1 issue (   [http://ubuntuforums.org/showthread.php?t=2003151](http://ubuntuforums.org/showthread.php?t=2003151)   ). 

I now manage 4 ubuntu systems, 2 of them are boat anchors using xubuntu, and 2 are very modern certified ububtu capable systems running Unity.  All of the systems I manage are for family members.  A fifth ubuntu system will be coming online soon,  by another family member. 

I admit booting into windows once in the last year, but it was by accident as it is my only system that still has Windows installed on it. But, I only booted up in windows because windows is the default during boot up and I didn't select Ubuntu because I was distracted.

My daughter became very addicted to a school supplied Mac a few years back, but she likes Unity because it's quite Mac like.  Her school supplied Mac laptop came with premium software, that probably cost many hundreds of dollars, if not more, and she is adapting to Unity use quite nicely.

So far, Ubuntu use hasn't saved us any money though, as all the systems I manage previously had windows installed on them-so we paid for windows anyway...... 

When I bought the 2 Dell systems last year, I was really sort PO'd, they refused to sell a computer without Windows installed. So, we paid for Windows anyway (both systems are Ubuntu only), Windows was wiped when I installed Ubuntu. The next time I buy a computer, it WILL NOT be a Dell (unless they change their policy of forcing me to pay for Microsoft software).

Our family has one person who works for Microsoft, so we all can get software for the PC quite cheaply. Last year at Christmas, there were no Microsoft software cd's given as gifts....for the first time in years!!!!!!!!!!

I haven't used windows for more than a year now.

I'm curious though....What percentage of all computer users use a Linux system, and are we still a BIG minority as a group?

_***I want to thank all who volunteer to support the Linux movement***_-Ubuntu is wonderful. I especially like the open sourced software availability and this forum. Not having to worry about hidden security issues with free software in Windows is a BIG advantage to me. No firewall, no virus protection needed, very little time spent maintaining the system and the faster boot and shut down times are all very nice adornments--icing on the cake!!

Regards,

Art


PS:I'm slowly learning the value of the command line/terminal mode.  I used to think of it as a curse, but now I realize it's a blessing!!! I'm actively and intentionally learning about terminal mode commands now instead of looking for ways to do the same function with GUI apps.
