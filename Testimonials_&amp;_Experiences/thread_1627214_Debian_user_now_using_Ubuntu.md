### Post by vharishankar on 2010-11-25
> **johntaylor1887 said:**
> That's not part of the main branch of Debian. It's a specialized port.

It's not even just a port, it's a different operating system with the Debian-specific components loaded on top and trying to achieve integration with the BSD base kernel (for example the GNU toolkit is used instead of the BSD toolkit).

However, it is uniquely Debian in one sense that Debian describes itself as an Operating System not as a Linux distribution and it's not identified solely with the Linux kernel. E.g. the incomplete GNU/Hurd. 

In that sense I'd say the Debian GNU/kFreeBSD is as much Debian as Debian GNU/Linux. Because the philosophy of Debian is what defines it uniquely amidst hundreds of Linux-specific distributions.

Debian is more than the Linux distribution Debian GNU/Linux, but I agree that GNU/Linux represents its main focus.
