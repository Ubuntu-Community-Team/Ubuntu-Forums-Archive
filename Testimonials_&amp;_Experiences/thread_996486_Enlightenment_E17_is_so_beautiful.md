I built it from SVN, and have a deb package available for you guys to try.
[http://myubuntu.dreamhosters.com/download/myE17-deb/](http://myubuntu.dreamhosters.com/download/myE17-deb/)

Enlightenment E17 does not have a Trayer like Gnome and KDE do, so I recommend you guys install a trayer.
apt://trayer

Then run the trayer by:
trayer --edge bottom --align right --widthtype request --transparent true --alpha 255

Then you might need to add these to the Startup items:
gnome-settings-daemon, nm-applet & trayer

Tips:
To use Gnome File Browser in E17
nautilus --no-desktop
