### Post by smartsmokey on 2008-06-09
I've had a top of the line Thinkpad sitting around for over a year. I wasn't able to do anything with it, seeing that the security settings were straight out of NSA. Anyway, the other day I was thinking "hey I wonder if I can install Linux on it instead." Surfed around, saw that Ubuntu was good, downloaded Hardy Heron on a different computer, burned the iso onto a disc, and shazam! It was good to go. I'm so friggin impressed with this. I'll never use windows or mac again. I've been able to install everything I could want just with add/remove programs. I've got the desktop enhancement thing with the cube, video editing, I don't even know what else. It's great. I had no trouble getting my ATI card running, my audio, everything. The only problem I had was with my built-in wifi...but that was because I had the actual/physical switch on the front of the computer that controls wifi to the "off" position. I love Ubuntu. 
    The only things I'd like to see are a Rosetta Stone type of program for learning languages and a guitar amp modeler that actually works. Oh, voice recognition too. 
  I feel like I owe Ubuntu something.
