### Post by uRock on 2010-07-07
I have had a few problems with Lucid, but most of them have cleared up via updates. Maintaining Lucid is fairly simple, run update manager once a week.
I have a simple update app that does everything with a double-click and an entry of a password.
```
cat > update
echo This app will bring your system up to date!
sleep 3
sudo apt-get update && sudo apt-get upgrade
echo Your system is now up to date!
sleep 3
Goodbye!
sleep 3
exit
[Ctrl+d]
chmod +x update
```Then I right-click the desktop and click to create launcher. See screenshot to see what to insert into the fields on the launcher.
