### Post by armandh on 2007-07-27
after two successful solo installs I was ready to go for it on a machine with xp OS dual boot
first I reduced the partition with partition majic already installed in xp,  leaving 12GB unpartitioned.
then  booted from the live disk and installed Ubuntu,  selecting use all the unpartitioned space option.
Ubuntu installed the mbr loader and menu choices
I also went back and reduced the xp partition by 2 GB for a fat32 file partition to share/transfer stuff. 
so far so good!!!
