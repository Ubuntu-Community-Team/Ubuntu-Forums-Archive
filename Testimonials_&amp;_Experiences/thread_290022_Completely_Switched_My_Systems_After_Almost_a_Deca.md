### Post by iamnafets on 2006-10-31
Been on the computer basically since I started walking (thats an exagerration), but my dad started me programming when I was 8.  So that has to count for something.  Anyways, been dabbling in programming, 3d, everything computer for quite a long time, but never made the foray into Linux due to hardware issues...

Took the plunge a week or two ago and downloaded the live CD (not quite the commitment it used to take), tried it out and loved it.  I had a lot of issues with wireless cards and video drivers, but ubuntu takes the cake with these forums.  That search function really does help :).  Been learning to use the command line, getting used to VI, and doing some programming in QT and Glade on the side.  I'm hoping to get involved in some project soon, but I guess this is a big thanks to everyone on the ubuntu project for making it easy for me to switch over and giving me a boost into the deeper stuff of linux.  

Anyways, the gist is...2 computers switched from Windows XP to Ubuntu (dual boot, but never use except for my MP3 player, which I'm working on porting over), soon enough the server at my parents' house, and eventually, the world!  Thanks a ton for all you guys do, Ubuntu rocks!
