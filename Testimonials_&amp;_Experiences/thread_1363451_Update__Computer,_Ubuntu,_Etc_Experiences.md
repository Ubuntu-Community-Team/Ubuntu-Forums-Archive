### Post by 3rdalbum on 2009-12-29
That was a bit of an odd testimonial.

Most (over 50%) wireless cards do work out-of-the-box on Ubuntu these days, so you're unlucky not to have one of those (apparently). Many others can be made to work.

Maybe the copy of Ubuntu you got from the library is outdated? You should always use the latest version, which is currently 9.10. If you request your own free CD at [https://shipit.ubuntu.com](https://shipit.ubuntu.com), you can try Ubuntu again. And if your wireless card doesn't work on the latest version of Ubuntu, then about $10 will get you a new one on eBay that will work with Ubuntu. (check the model number online to see if it will work BEFORE buying it).
