### Post by Methuselah on 2008-08-10
> **bailewen said:**
> Classic.

This is actually the first time I have ever gotten Ubuntu (or any distro for that matter) to work smoothly enough to keep me from going back to Windows. Vista was the last straw for me but I have "broken" Ubuntu way to many times to not sympathize with some people going back. I have to say that Ubuntu is *still* not ready for prime time. 

It's getting there. Damn close. **The main problems mostly have to do with the commitment to avoiding including any non-free packages with the basic distribution.** 


But the the bolded part is an absolutely essential commitment! 
It's not Ubuntu that makes software non-free it's the people who write that software.
Most proprietary software, even when cost free, grants limited non-transferable usage rights.
Since Ubuntu *IS* free/liberating (you can download it, copy it, give it away, derive a new OS from it, install as many times as you want to as many machines as you want) 
it cannot include, by default, software that users have no implicit right to redistribute or change.
Otherwise, everyone who exercised their rights under the Ubuntu license would be infringing the copyright of some included component.

Not surprisingly, if you install windows from scratch your video card will be recognized as basic VGA/SVGA until you download/install a driver from ATI/NVidia.
You won't be able to watch youtube videos until you manually install flash.
You obviously won't have skype until you download and install it as well.
Ubuntu is not unique in respecting copyrights but I'd argue that it makes it far easier to install some proprietary components than windows does.

Needless to say, I disagree with your assertion that Ubuntu is not ready for 'prime time'.
Linux is doing quite well on EEEPCs despite the included Xandros being quite a bit more basic than Ubuntu.
However, I do agree that despite Ubuntu's making OS installation easier than just about anything out there, many people are not up to setting it up.
I do completely disagree with the your implication that proprietary software issue is something Ubuntu is stubbornly refusing to remedy.
Ubuntu does not write the restrictive proprietary licenses.
In any case, the true remedy is pre-installation by OEMs because if people had to regularly install windows Ubuntu.com would be down everyday. :)
