### Post by MCVenom on 2010-05-01
Ok, you obviously do not know how software management works on Ubuntu. Here's a quick primer :p:

*A repository is basically just a server somewhere that hosts programs for a Linux distribution. When you use the Ubuntu Software Center, your computer asks the repositories for a list of programs, and displays it for you. When you install a program from USC, it download the program from the repos and installs it.*  There are four important repos in Ubuntu:


[LIST]
[*]**main** - Everything in here is free & open-source. Things in here get official bugfixes and security updates from Canonical, Ubuntu's parent company.
[*]**restricted** - Everything in here is closed-source. Canonical does what they can, but they don't really promise anything. This repo really only has things like drivers and whatnot.
[*]**universe** - Like in main, everything in this repo is free and open-source, and this is where you'll find the bulk of software available for Ubuntu. Canonical doesn't provide bug fixes and security updates for software in here, but hard-working volunteer package maintainers do.
[*]**multiverse** - Everything here is closed source/proprietary, unmaintained, and some things may even be illegal to install from here. Things like codecs are found here. This is not a repo where you'll find many programs perse, and you really should avoid it if you can.
[/LIST]
Now, as mentioned earlier, USC gets it's software from these repos. There's also a 'partners' repo, but all you'll find in there is a couple Adobe programs.

> **outlaw45k said:**

Well, as shown from our lesson on repos, **there's not yet any commercial/for-pay software repos!** ***Linux*** ***is a system that's always used free software above all else, and that's where the problem come in -- there is no system in place yet to charge for software through repositories.*** *Right now the Ubuntu devs are trying to find the best way to put a system in place for commercial software, so it's just a matter and when and what they come up with.* :p
