### Post by 3rdalbum on 2009-09-24
> **cmcanulty said:**
> but it sure would be nice if Ubuntu could just recognize all USB devices like windows and MAC do.

They don't. Windows depends on you installing a driver written by the hardware's manufacturer. Windows contains only a couple of standard drivers. Mac OS X relies on the same thing. Mac OS X supports almost no webcams, scanners, digital still cameras or MP3 players compared to Linux.

Mac OS X users must carefully check for compatibility before buying any hardware, just as Linux users should do.

> That should be a primary issue way before fancier desktops etc.

Open-source is about people working on what they want to work on, and what they can work on. The people behind Compiz either don't have any Linux-incompatible hardware, or they don't want to work on drivers, or they don't have the necessary knowledge to reverse-engineer hardware support.

I'd rather have nice desktop software than support for hardware that I can't even buy in my part of the world or will never again encounter.

Not sure about your Zip drive problem; those things were plug 'n' play back in 2005 when I last used them. SD card readers are generally the same; is it plugged in through USB?

Please post the output of the following commands:

```
lspci
lsusb
```
