### Post by 3Miro on 2011-03-29
In short fallback is Gnome 3 - Classic (which works exactly like Gnome 2).

Gnome 3 comes with two modes:

- Gnome-Shell: default developed by Gnome people.

- Gnome-Classic: essentially Gnome 2, however, Gnome panel and Metacity will no longer be supported (updated or fixed) so this is really only a fallback mode. It will be phased out eventually.


Gnome-Unity: default in Ubuntu 11.04, is Gnome 2 with Compiz+Unity pluing. Unity is only a dock and menu for compiz. I think eventually Ubuntu will have Gnome 3 + Compiz + Unity, but for now it is Gnome 2.

Ubuntu comes with Gnome-Unity by default, you should be able to easily install Gnome-Shell from the Software Center.

Your options if you don't like Shell and Unity:

- Use fallback mode, but this should be a temporary solution as fallback will be phased out.

- Disable Unity from Compiz, then you probably need a panel. You can either use Gnome-panel (see above for lack of support) or put AWN or something of the sort.

- Switch to XFCE, the new 4.8 version is about as close as you can get to Gnome 2 without having Gnome 2. You get almost all features that you are used to and some new ones.

- Switch to KDE/LXDE which are rather different from Gnome.

Now if we can put something like this (probably better explained) in a nice visible place for people to read, hopefully we can put an end to thins Unity vs Gnome 3 FUD.
