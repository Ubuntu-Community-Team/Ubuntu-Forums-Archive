### Post by Lloydb39 on 2013-01-23
My first computer, in 1984, had the CP/M OS. All command driven. Years later, moved to DOS, and learned its commands, too. If I needed to do something special, I wrote a program in Basic or dBase (which I miss for its simplicity). Got aboard GUI with Window 95. Found Linux about four years ago and was intrigued so I gave it a try. Got frustrated and posted a rant, to my regret, which got dozens of flaming replies. But kept trying and I now have three desktops and four laptops running Vista, Windows 7, Max OS, Xubuntu and two flavors of Ubuntu. A few months ago I turned my Windows 7 desktop over to the wife and built a computer which I loaded with 12.04 and 2D Unity. I don't have the beefs everyone else does with Unity and, with the help of the forums, I have everything running that I need. The only exception is that I still use Quicken on the wife's machine because I haven't found a Linux substitute. Despite having much command line experience, I am a long way from mastering the terminal, but I get by. All in all, it is an inexpensive and efficient way to get things done and the people who make it work should be congratulated.
