### Post by tmonte on 2007-07-15
Hi everyone,

I just wanted to share my very positive experience with Ubuntu over the past couple days.  I'm a 10+ year IT guy who built the early part of my career in a Windows world, yet I've worked on systems we've built with SuSe, RedHat and some machines running VMWare ESX (modded RH kernel).  I've since crossed to the management side of things, but my roots are still technical.

Personally, I've run Windows almost exclusively.  Since the late 90's I've dabbled in various Linux distros (RH 5.2, 6.1, 7.0, Mandrake 6.0, 7.0, FC5, etc), but I've never been able to get them to do everything I could in Windows with the same relative level of ease.  The problem has always stemmed from my love of cutting edge hardware, not that Linux couldn't necessarily handle the functions I wanted to perform.  With my 6-12 month hardware upgrade cycles, I just couldn't get there with out Windows.

This changed a day ago when I decided to give Feisty a shot dual-booting with Vista on my ASUS A8Js laptop.  This machine is about 5 months old, and while it's not exactly cutting edge, I thought for sure I'd be killing myself to get it to work.  I was wrong.

The base install went down just fine, and it seemed most everything worked right out of the box.  Wireless and the card reader worked immediately.  I installed the nvidia glx package (Go 7700), then tweaked the xorg.conf file to get the new driver working, which only took me a few minutes.  I was very happy to see how smooth that went after recalling the challenges in the past with sketchy installation routines.  WoW is about the only game I play regularly, so the next step was to load up Wine and get WoW running.  I copied the WoW installation over from the NTFS partition, tweaked the .wtf file to use OGL, and away I went.  I then learned that the audio wasn't actually working beyond the logon, which was odd.   A little poking through the how-to docs got that going by adding a couple option lines to the ALSA modprobe file.   

The real kicker for me is that in comparing Ubuntu to Vista with the same quality settings in WoW, both running a 98.xx version driver, Ubuntu is ahead by about 20 fps on average.  I am very impressed, especially considering WoW's "preferred" acceleration is under D3D.  Overall, the system feels snappier and seems smoother.  Don't get me wrong though, I am a huge fan of WinXP and while I can't compare performance between the two directly on this machine, I was running XP Pro on it in the past and performance under Feisty is about on par with XP.  Maybe Vista will catch up with driver updates, but I really think all the extra bloat makes it clunky.  (By the way, this machine is running a 2GHz Core 2, 2GB RAM, 120GB 5400RPM disk and the GeForce Go 7700 512MB, so it's no slouch.)

Thanks for producing such a quality distro, and thanks to the community for all the work that goes into knowledge sharing, documentation and support. 

Time to go load Ubuntu on my new file server/media machine...  :D



Tom
