### Post by Mr*Wiggy on 2006-10-31
Hello Folks,

I have been a MS Windows user since version 3.1 (1992) and before that it was Atari, Commodore etc. I use a PC mainly for recreation - surfing the Web, Ebay, Email and games. Plus a bit of serious stuff like online-banking and word processing. I have tried using various Linux Distros in the past but gave after a few hours. 

Then recently I found Ubuntu. I was struck right away by the differences in theme and style but especially the package management system, which is the best I have seen. I upgraded to 6.10 as soon as it was released.

like most Newbies, I have had many problems setting it up the way I want it. But I persevered and spent hours scanning this forum and the web for answers. Now it is working really well and I haven't booted Windows for the last 4 days. I guess I'm hooked :)

All I need now is some really good, commercial-quality,  RTS games for Linux and I will ditch MS forever.

Eric
