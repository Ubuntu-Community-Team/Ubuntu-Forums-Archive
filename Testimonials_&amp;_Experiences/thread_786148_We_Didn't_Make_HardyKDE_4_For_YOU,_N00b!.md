### Post by princess_daisy on 2008-05-07
Hi all --

I have been using (K)Ubuntu for a little over a year now but still consider myself pretty much a beginner.  I thought people might be interested in a relative newbie's experience installing Kubuntu Hardy with KDE 4 today (after much backing up).

**Pros:**
[LIST]
[*]It is very pretty.  The oversized icons don't bother me as much as they have some commenters.
[*]Installation went off without a hiccup.
[*]Firefox 3b5 seems pretty stable so far.
[*]Things are not as slow as I would have expected, given how slow Ubuntu 7.10 ran on this computer (a several-year-old Dell Latitude). (Basically, if I tried to run Firefox and do anything else, especially OpenOffice, it would go on strike.)
[/LIST]

**Cons:**
[LIST]
[*]The handling of KickOff (the application launcher) isn't very intuitive.  It took me a while to figure out how to add items to the Favorites list.
[*]I fell victim to [this crash]("http://ubuntuforums.org/showthread.php?t=746778") and am thus going panel-less for a while.
[*]I'm hoping more widgets/plasmoids/doohickeys/whatchamacallits come along -- specifically one for hard drive temperature and one for weather forecasts.
[*]Speaking of Plasma: Ctrl-F12, which brings all the various plasmoids to the front, can't work without taking a second to clear my desktop image.  I imagine on a faster computer this would be a far smoother transition.
[*]Adept (is there not a new package manager for KDE 4?) is... not trouble-free, shall we say.  If I ask it to grab too many packages at once, it will stall while downloading a certain package at, say, 28%, then start downloading the same package over again without clearing the stall, with the result that I have to cancel installation.  Obviously I've been able to install what I need so far, but this is going to get tricky when I start trying to grab LaTeX packages.
[*]Apparently there's no way to have a different image for each desktop, which I was able to do in Kubuntu 7.04.  Sob!  This was one of my favorite aspects of KDE -- that I could have one desktop/workspace for work, one for goofing off, etc.  As things are configured now there doesn't seem any way of distinguishing between desktops/workspaces (I currently have four).  Am I missing something?
[/LIST]

All in all, it's going pretty smoothly, considering how this release really should be considered still in beta.  And my desktop is more attractive.  The usability is not as high as I would like, though.

I went from Ubuntu to Kubuntu with Hardy and have found that KDE3 works very well for me. I cannot think of anything I miss here, and so I have not felt the need to try Hardy/KDE4. It will be exiting to see how the development goes though. I guess Kubuntu will have KDE4 by default in one of the next releases? My main worry with KDE4 for now is that, based on the testimonials, it seems to suffer a little bit of the Vista syndrome - to many visual effects and eye candy slows it down and only allows it to be used on newer hardware.
