### Post by _simon_ on 2007-05-04
Bit of info first up, I've been using Linux for about a year and half now, started out on Ubuntu but currently using openSUSE.

My better half's father has recently purchased a new machine, in a conversation to my other half he was talking about his frustration in finding decent antivirus software as the machine did not come with any. My other half mentioned that she used Linux and that viruses weren't an issue. 

Now, this got her father interested and he brought his machine over the same day so that I could set him up with a dual boot system (Windows XP MCE + Ubuntu).

**_THE GOOD_**

I've got his machine fully setup: sound, ATI drivers, dial-up internet (he's a bit behind the times). Windows partions were auto mounted, even the built in card reader works!
[B]
_THE BAD_[/B]

The installer, whilst a vast improvement over the 6.10 installer, still went wrong somewhere and left me looking at a "unable to find boot device" message. To fix this I had to run gparted BEFORE the installer, delete the partitions it had made and re-create them. 

If this was her dad installing Ubuntu then he wouldn't have had a clue what to do.

**_THE UGLY_**

Dial-up modem. Probably says it all really.

I've never used dial-up with Linux, so this was new to me. 

I have to say, I very nearly gave up, but I took a break and then gave it another bash.

It has taken me a good few hours to get this up and running and then I had to get my wallet out to purchase the full driver licence :(

_**CONCLUSIONS**_

More needs to be done in the way of dial-up. I know this is a pretty much outdated, dying technology, but people do still use it. Perhaps a menu link to a dial-up wizard that would detect the modem chipset and display the drivers from Linuxant for testing the connection and a link to purchase the full licensed versions. Gnome-PPP should also be installed via this wizard and a desktop shortcut or panel icon created to launch it.

I'm not sure what went wrong with the Ubuntu installer but I guess this means there are still a few problems with it. I did try the text based installer as well and have to say, setting up the keyboard was a bit long winded! That didn't work either by the way, it wasn't until I recreated the partitions that the installer made the first time round that the install was successful.

I'm impressed with the restricted driver manager, installing the ATI drivers were a breeze and also impressed that you can now download the plugins that firefox says are required, without having to jump through hoops of burning fire!
