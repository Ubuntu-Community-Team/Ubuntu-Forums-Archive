### Post by simonfiction on 2007-08-22
Hi,

Just wanted to be able to say somewhere well done on this amazing OS. I'm literally blown away by the entire thing and how far it's managed to come with little to no support from hardware vendors. I'm still getting to grips with a few things but overall it's far superior to windows in most ways. To top it off I installed Compiz last night and found myself drooling.

There are a few issues or annoyances I've had to deal with but when I look them up on the forums there always seems to be a solution or a fix being developed for the next release. One example is having to manually edit the xorg.conf, although I've read recently there will be a GUI for this in the next release.

I can't help but feel Ubuntu is so close to being a viable alternative for non tech users and I really hope it will be. It's nice to have the command line control but I'd love to see more functionality brought to the GUI and not just at the command line. I feel this is the only way that Ubuntu will really make it.
