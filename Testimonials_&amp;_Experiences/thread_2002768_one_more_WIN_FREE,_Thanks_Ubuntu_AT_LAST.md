### Post by armandh on 2012-06-13
my problem fully embracing Ubuntu has always been win only software
20 years of data in Intuit quickbooks has always been a deal killer when it came to totally dumping MS.

NOW one more computer is NOT dual boot, rather Linux only [almost]
Virtual Box for Ubuntu 12.04 just works.
and XP therein [the almost part] runs the quickbooks. 
all this on a 4 YO dell mini-9 [Inspiron 910] netbook
and surprisingly snappy [seems no worse than the boot of xp and AV]

no serious lag. it is way better than win-7 + AV and QB 
I know, tried it. even the 32 bit 7-starter and avast ran slow and hot

since support for XP tanks in 2014 it is nice to know of this Linux solution for older hardware.
running a dual boot of an OS with known exploits is problematic.

MS even offers a free XP down load for 7 professional's virtual environment. something one needs with legacy [real mode] software
in that case it was an e-mail client with 11,000 saved e-mails in a proprietary form. [and no virtual mode solution]

but for this Ubuntu & virtual box I am using a licensed XP-SP3 copy [this is a business use]

**PS**
just like xp on limited video memory it would get bogged down if given too much to do at once
soooo
I first unchecked ALL the [XP] start up services [ini config] save the 5 or so essential ones that will not un check [it ran better]
and upped the video memory allocated to the boxed xp [about half of the 128Mb]
much much better 
now better than not laggy;  smooth!
