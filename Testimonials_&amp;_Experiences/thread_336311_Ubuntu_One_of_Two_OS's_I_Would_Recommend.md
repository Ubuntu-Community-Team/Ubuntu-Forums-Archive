### Post by geber22 on 2007-01-11
I've used a lot of different operating systems in my life, and I have to say only two strike me as good.  Ubuntu and Mac OS X.  

I've used 

Fedora Core III, IV, V, tried to install 6 (Don't even go there) Fedora has gone downhill it's a shame because at one point it showed promise, but it seems to be getting worse not better.

Solaris at work (stinks except for server usage and too expensive),

Windows 95 - XP Using Windows is like dying a slow death.  When you first get the machine you think hey this isn't so bad, then every day it gets a little slower, a little more clogged, until finally the machine crawls and I mean crawls.  Not to mention any piece of software you need will cost you an arm and a leg.  Who knows maybe I've got a thousand viruses on the thing, not like theirs any good way to tell.

Ubuntu and Mac OS X are both very clean, and take into account the user experience.  My machine doesn't start dying a slow death either, bonus!!!  

Obviously somebody tried using these OS's before releasing them upon the general public.  I love Ubuntu and it loves me that is obvious.  Now OS X is also a great system, but the difference is one is free.  Anyway thanks for putting out a quality product guys.

GO UBUNTU!!!
