### Post by Alex Libman on 2009-12-17
I've had a MUCH easier time installing (pirated) Windows 7 on my old Dell Dimension 4600 than Ubuntu.  Windows just installs the needed drivers and everything just works by itself.

With Ubuntu I needed to figure out that the latest version of the NVIDIA driver doesn't work, and I have to use 173 - I can figure out how to edit my xorg.conf and so forth to fix a system where the screen doesn't work, but the average user cannot.  Then I had to get [a specially compiled fix for XUL-based programs]("https://launchpad.net/%7Efirefox-smooth-scaling/+archive/ppa") (ex. Firefox).  Now I can't upgrade to Lucid Lynx because of an NVIDIA-related bug, which makes it more difficult to install the latest Linux software on an Ubuntu system, while all Microsoft's betas work great as much as a year before they're released.  And the frames per second I'm getting with Linux is still much less than Windows...  :(

Don't get me wrong, Linux is a great server OS (except *BSD / MINIX 3 have a much freer license), but it still has a long way to go before it can compete with the big boys on the average user's desktop.
