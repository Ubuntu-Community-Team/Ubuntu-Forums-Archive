### Post by 3rdalbum on 2011-04-26
> **naikvarda said:**
> Please tell me something more about the natty.

If you want to open a file with a specific program that's in your launcher, you can just drag the file to the icon in the launcher. As you do this, the programs that CAN'T accept the file will dim, leaving only the programs that CAN. Very cool.

If you need to drop a file onto the window of a running program, hover the file on the icon in the launcher, and then the program's windows will "expose" (scale) so you can continue dragging the file to the part of the program's window that it needs to go to.

You can now hit Alt-F2 and type in an apturl to install software, for instance: "apt:gimp". Using Alt-F2 and typing the name of a program you want to run is the most efficient way of launching programs on Unity.

If you want to browse programs of a specific category, right-click on the Applications lense in the launcher, and then you can select the category you want to browse. You can do the same thing with the Files lense, for types of files you want to browse.

Is that enough information for you so far? :-)
