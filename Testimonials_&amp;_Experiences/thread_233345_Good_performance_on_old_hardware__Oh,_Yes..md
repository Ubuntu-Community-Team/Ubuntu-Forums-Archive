### Post by MikeMLP on 2006-08-10
I was working on a major research project tonight, when I realized I was opening a a rather large number of tabs.  So I decided to take a screenshot.  The photo is not doctored - I'm using Firefox 2.0 Beta.  Besides Gaim, Firefox, and Writer, which are visible, I also have 3 more Firefox windows open on other desktops for over 200 tabs in all, a couple terminal windows, Evolution, and Synaptic.  Performance seems the same as it was on first boot.  And the XGL keeps cranking away, on a system that will not be able to run Aero Glass on Windows Vista because it does not meet the minimum system requirements.  Go figure.

Thank you Ubuntu / Open Source for a fast, bloatware-free environment.  It is a power-user's dream.
