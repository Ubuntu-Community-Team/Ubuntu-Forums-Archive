### Post by anthonyvo on 2011-01-03
Hi,

I've been around computers for a while (writing BASIC on Apple IIe's in grade school then using Unix in college - pine for email, etc., etc.) I'm not a true admin or programmer, but I sure as heck can figure things out and follow directions.

In any case, recently my living room PC (running Win XP Pro SP2) crashed because of a corrupted logon file. I couldn't boot into safe mode or get a command prompt. That was the last straw (I'd been through everything from Windows 3.1, 95, 98, me, XP, 2000 [probably the best of the bunch,] and Vista.)

I decided install Ubuntu desktop (although I was torn between Ubuntu and the new Linux Mint.) So far, it's been only one day and I love it! Getting used to using the Terminal again (seems ol' school,) but other than un-complicating my thinking in my approach to an OS, it looks like it will be staying!

I look forward to learning and sharing here. It seems every Google search for anything Ubuntu points here.

Thanks!
-Anthony
