Some complaints/suggestions:

Users should be able to add folders, files, and applications to the launcher bar by right clicking on the launcher bar itself.

There should be a launcher for the old hierarchical view of applications (in case people want to use it).

When you view the applications from the applications button on the launcher bar, you should be able to right click on the application icons and have the same options you have when clicking on any application icon (copy, move, make link etc) as well as an add to launcher bar option.

Users should be able to move the launcher bar to bottom, top, right.

Nothing users want to get to should be more than two clicks away (drop down menus make this possible). As it is now, some preferences are four or more clicks away.

I'll stick with Unity for now and see if it grows on me, but I already see things I want to change.

This may sound obvious, but it has happened to me many times: check what the sound is set to in the application that you are using to play music.
