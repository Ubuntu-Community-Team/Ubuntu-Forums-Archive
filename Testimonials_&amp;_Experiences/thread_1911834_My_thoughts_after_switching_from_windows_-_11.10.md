### Post by wolf_rt on 2012-01-19
This post is about my experience of switching to Linux (Ubuntu 11.10) after getting sick of Windows.

Firstly, thanks to all involved in making this alternative available!

I chose to switch to Linux after getting angry over Microsoft's plans to attempt to 'lock' UEFI MB's to windows... I know there not going to pull it off, but this is my protest... I also had no plans to upgrade to Windows 8, as it sounds like it wants to turn my PC into a overgrown iPad... Stuff that!

So after searching for a distro, I settled on Ubuntu, it was a toss up with Mint, but I thought the Ubuntu homepage was better... :-)

So I made a USB stick... 'Live' worked, but was tragically slow, and i decided to give Linux a fair go I would have to do a HDD install... Really glad I did, the live version doesn't do Ubuntu justice at all.

I chose to have dual boot, as I'm a sometimes gamer, so I re-installed windows (cause it needed doing anyway... AGAIN....) and loaded up the Ubuntu CD I burned... no dice... something wrong with the CD... so back to windows and made a DVD-R.. this 'picked up' ok. But when I selected 'Install' it hung on a blank screen.
Luckily my partner had a PC right next to me, and a quick google showed me I had to press 'Tab' during boot for options... So I checked a box called 'no modeset' as instructed, and away it went.

After that shaky start the install was flawless, I especially appreciated the fact that the install continued while it was waiting for my input.
It connected to my wireless network during the install!!! (my new windows install is still not working with the network) and I pressed 'Back' to check the 'download propriety drivers' check-box after the network was connected... sweet as.

And booted straight into a user friendly desktop, where everything already worked! How novel... No spending 3 hours DL'ing and installing chipset drivers and re-booting 30 times ect...    (I only recently upgraded to W7 from XP and I only did that because every 6 months, I would have to set aside an ENTIRE weekend to reinstall XP and get it up to speed, at least W7 only takes about 4 hours..lol)

the only dramas that I was presented with was the screen resolution (expected as I use an old 21in CRT monitor that will go up to 2048-1536)
And the mouse sensitivity, as I have a 3500dpi razor mouse, a quick google once again turned up easy to follow instructions to fix this. A small script to adjust the sensitivity that is run on startup, no drama...(windows also has this problem, and I prefer the small script to having to install a 'Driver' that sits in the corner and does nothing)

So I set about installing all my favourite programs... no dramas here, either with command line instructions or using the 'Software centre'

This is when I found that I had 'broken' Flash... Just a Black box where videos should be. No problem I thought, probably got a bit excited installing programs from the command line... I'll reinstall. (since it had only taken about an hour up to this point)
So I did... Only to find 'Flash' broken again after installing GIMP and Audacity and surfing for about 10 minutes...
Hmmm... so I deleted and re installed Flash... problem solved.. for about 5 minutes.
I did get this fixed for a few days, by doing a 'complete uninstall' and running a script I found to remove all traces of Flash, but it keeps rearing its head when I install new programs, or for no reason at all... no idea on this one.

Since then I have been playing around and getting everything set up just right.. This is THE most functional desktop I have ever used... really good stuff!

Every question I have had, has been quickly answered by a quick search of google or this forum.

I would recommend Ubuntu to anyone... I have found it far more user friendly than windows, and a smaller learning curve than swapping from XP to Windows 7.

Thanks heaps.

P.S. Also getting my PC to network with the other PC's in the house (one windows 7 one XPsp3) was straight forward... not easy, but straight forward... compared to getting the Windows 7 pc's to work with the XP pc.. it was a piece of cake, no black magic involved at all)
my plan is now to install Ubuntu on the XP media centre PC!
