### Post by lento_ on 2012-05-08
This is the number one reason why I am put off Unity. It's not a unity-specific problem though, but one which most (all that I've tried) docks seem to have.

I don't get the love for docks at the moment. OK, they're pretty, but window switching with them is cumbersome. It involves mousing over and waiting, or making an extra click. Tools like expose or alt-tab can help, but if you've got lots of windows open then it can get slow or unusable.

When I'm at work it is common for me to have many windows open, almost always involving multiple instances of the same programme. A window switcher on a panel may not be pretty, but it is the quickest way to easily identify the windows which are open and to switch to the desired one in a single click.

Unity has many other problems, but the lack of window switcher is the main one which is putting me off using it. If I had to use it then the first thing I'd do is to trying and install a panel, which seems a bit of a shame in a system which attempts to replace panels with something better.

Unfortunately many other different operating systems and distributions seem to be going down the same root of using some kind of dock, so it will become increasingly harder to avoid.
