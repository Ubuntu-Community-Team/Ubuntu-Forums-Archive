### Post by Poo-buntu on 2010-12-02
I've found that the Ubuntu operating system is not so great for me. Sure, it works for basic functions like web-browsing, checking e-mail, a little word processing, etc. But outside of that, you're pretty limited to what you can do. For example:

You cannot install iTunes on Ubuntu. Maybe you can trick Ubuntu by installing it with "WINE", but it's half-assed at best. Maybe it will work, maybe it won't. It seems like nobody has a solid solution for this. Why? I don't want stupid RhythmBox or Banshee. I want iTunes!

I love Satellite Radio. Can't stream it with Firefox on Ubuntu. The only browser it works with is Internet Explorer. The end.

Did somebody give you a new game to play on your PC for Christmas? Will it work with Ubuntu? Will anything bought off the shelf at a Best Buy or somewhere else run on Ubuntu? Who knows? You're rolling the dice.

I think that at the end of the day, nobody should have to Google-search how to install and run a program on their operating system. Nobody should have to open up a DOS-like terminal and type in a bunch of text to MAYBE get something to work. I want my PC to work for me, not the other way around.

I applaud the idea of open-source software. I love Open Office, Firefox (even with it's limitations), and all the other great stuff that's out there. But I have to revert back to Windows after experimenting with Ubuntu. Am I alone on this?
