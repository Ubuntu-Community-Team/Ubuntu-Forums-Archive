### Post by Wolfghar on 2008-02-18
With my recent success installing Ubuntu on my dual head ATI desktop system I decided to give it a try on my laptop.

My desktop system took quite extensive tweaking to get the ATI drivers/dual head working (well worth it though :) ).  I figured my laptop would require some added effort as well...

The system in question...

Acer Aspire 5100 (AMD Turion 64 X2 TL50, 1.6 Ghz)
1 GB DDR2 ram
ATI Radeon Express 1100 graphics
802.11 b/g wireless

The laptop had been running Windows XP MCE quite happily since I bought it.

I went ahead and booted from the Ubuntu CD and was surprised to see my wireless adapter was recognized.  So I went ahead and started the install and wiped out the XP partition for a new swap and / partition...  (of course I imaged the xp partition just in case)

So I installed and booted up into Ubuntu (my splash screen doesn't show for some reason, minor glitch...)  Otherwise everything worked straight away, restricted drivers were automagically installed and at the correct resolution for my screen, wireless networking, wired networking, audio...  All worked after installing with no need for installing additional drivers!!!

That was the easiest installation I've ever done...  

Ubuntu!!!

Wolfghar
