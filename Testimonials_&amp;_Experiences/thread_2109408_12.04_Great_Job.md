### Post by steveneddy on 2013-01-27
10.04 was getting to the end of it's life - and the HD crashed in our laptop. So I thought we would try Mint. 

We tried - four times. Wireless issues, lock ups - slow - various performance related issues. So - DL Ubuntu 12.04.

Install quick and easy - just like I remembered.

Wireless - a snap with the Broadcom drivers.

HPLIP from HP - installed without a hitch.

Updated - no issues.

This is how Linux should be - not a hacking session on a supposed LTS - Ubuntu has the real LTS - and it runs great.

Thanks for another great release!

### Post by steveneddy on 2013-01-28
> **rewyllys said:**
> Did you try Mint 13 (Maya), which is based on Ubuntu 12.04?  

With respect to the issues you mention, Maya should have behaved essentially identically like 12.04.

We did attempt installing Maya. The system locked up when installing wireless drivers or when updating. Hard lock ups with no input control function,

We tried various work arounds - installing kernal headers and trying the driver from Broadcom directly. Very pretty - but there were continued issues.

Also when we first installed there was no GUI - really? - so run xorg-conf and move the file around. This was unacceptable for an LTS.

So we installed Maya for the third time with no updates - plugged into Ethernet and DLed 12.04 from Ubuntu.

No issues - wireless installed no probs - GUI worked out of the box - updates did not lock up the machine. Flash works out of the box. I can play mp3's and other music.

Ubuntu gets the thumbs up here - I would expect Mint to do a better job - but these are the reasons why I continue to use Ubuntu - it just works.

I don't have time for hacking anymore - I have a day job - I build guitar tube amplifiers and play in a band - so there is no time for funny business. 

That's my story and I'm sticking to it.
