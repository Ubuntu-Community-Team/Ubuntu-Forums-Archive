### Post by Tamlynmac on 2010-03-03

How can that be???? We've (my family & I) been Ubutnu only for >3 years and amazingly have had no real issues. We did see some required apps changes along the way, but nothing that wasn't easily altered.

We have found the OS extremely stable and with just a few common sense rules we've never had problems upgrading. Always use a spare HDD to test upgrades on all our systems prior to installing on primary HDD's. We only use Linux compatible hardware and when purchasing said hardware notify the seller that if it doesn't work - it will be returned. I've preached these solutions many times in this forum.

Thanks for sharing and hope your Ubuntu journey continues to be a rewarding one.
