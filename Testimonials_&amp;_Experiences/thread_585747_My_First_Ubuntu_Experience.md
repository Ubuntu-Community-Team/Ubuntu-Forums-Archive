### Post by miwaypet on 2007-10-21
I am new to this forum and Ubuntu. I installed Ubuntu 7.10 on my sony vaio vgn-fs980 this past Friday. I was a little more than worried. I had meant only to partition my hard drive and install it, but I screwed up that and wound up partitioning and formatting the drive for ubuntu. Glad I did. This is the best version yet! I have just recently finished the linux class offered at my tech college. I still don't have a full grip (or is it grep) on the command line and syntax, but I went for it.

No probs with install. Wireless worked ! Used the tools that I could download and install with package manager and add/remove to config linksys wireless card driver. Took some doing but found the info needed to get my dvd player to read the encrypt dvd's. In short I am impressed. I lost nothing with the removal of vista. I got the download through school. Vista worked well. Some probs, not many headaches. But my vaio's disks where getting more and more noisy. And my free space was slowly disappearing. Now my notebook runs cooler and much more quietly. Have a prob with memory stick light staying on. 

I prefer opera browser. Got flash for it through firefox by doing to desktop tower defence and then installing the plugins for it in firefox. It piped straight to opera, no probs. More importantly, no terminal. Only used that for the libdvdcss install.

So all that is left is to see if I can get docs to work between open office and ms office. 

That's all. Glad your here. Glad to be a part of the community. Thanks.
