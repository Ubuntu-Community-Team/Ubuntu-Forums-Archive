### Post by kashey_be on 2009-07-07
I don't know about motherboards but it deal well with half dead hard drives.
In my old machine I had a 20GB drive that already had some bad blocks.
Obviously windows was impossible to run at that point since it would crash very often.. however after installing ubuntu it knew exactly how to deal with it.
Ubuntu scanned the drive for bad blocks, and avoided using the areas it marked as bad.
What's even more interesting is that no immediate danger was hovering over the boot sector since every linux distro copies the super blocks multiple times, therefor even if the MBR is harmed, you can still boot.
amazing, don't you think?
