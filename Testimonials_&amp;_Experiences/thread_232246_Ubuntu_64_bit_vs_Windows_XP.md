### Post by gannic on 2006-08-08
I'll keep this short...I mainly use my PC for work(writing) and multimedia and was VERY skeptical. Basically XP took about 2-3 mins to boot on a high spec machine, partly due to all the antivirus business, and the WGA tool was getting on my nerves - I thought it was time to see what the alternatives were. I decided to try Ubuntu instead of reinstalling XP for the 5th time. It is still early on (day 5), but here are my thoughts.

Installing the Ubuntu OS was the easiest OS install I have yet seen, about a tenth of the time it takes to install XP. I had some issues with my wireless card that the occasional PC for internet user would have been in severe trouble with - unless they wanted to learn. However they may also have had problems with Windows in the same way - but installing wireless is generally fairly plain sailing in Windows (in my experience), and this was ,I thought,a sign of things to come.

I was completely wrong. Once the above problem was solved it was plain sailing again. Once automatix was installed I was flying. Hardly any command line work, just following the odd instruction here and there. My PC running ubuntu now has all the things I need on it - and is running much faster than it has ever been on windows. In addition, my portable media player is now running rockbox and is better than ever for it (the number of extra options are astounding).

All in all, being completely new to all this, I am gobsmacked - if you dont use your PC to play PC games (which can be achieved in Ubuntu but is certainly a lot harder) it just seems like there is no contest between Windows and Ubuntu. I may have got lucky with my hardware (which all works perfectly), but Ubuntu seems to be solid as a rock - it just works. Its just an opinion, but I have not needed to boot windows since I installed it, and I am not sure I ever will.

If you are getting a new or additional disk for windows, do yourself a favour and try Ubuntu first.

Congratulations to every one involved in making this system what it is.
