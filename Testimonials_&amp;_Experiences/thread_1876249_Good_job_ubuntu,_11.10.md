### Post by kensum on 2011-11-05
I returned to ubuntu when the direction of gnome was set. Having been  away from ubuntu for some time now, but wanting to use and test unity, I  returned with an install of 11.04. At the same time I did an install of  the 2nd alpha for 11.10. I have had no problems with either of them. I  have gotten rid of 11.04 as 11.10 is much improved. As for the unity de,  I have gotten used to it but there was several issues that I did not  feel at all comfortable with. The first is the lack of being able to  customize the desktop to suit the users needs (change can be painful),  which appears to be the main concern of most users. The other issue I  found is Dash. No matter how I try, I just can't seem to see a reason  for or a use for the dash as it is implemented. I believe Conical has  done a very good job putting this version together and they need to be  commended for going outside the box.

With that being said, I felt  the need to find out what gnome3 was about. I do not like the idea of  adding or using the gnome3 shell in ubuntu as it is designed to use  unity, so I turned to Sabayon 7 using gnome3. Having used sabayon in the  past, there were no surprises just as with ubuntu, but after using  ubuntu and sabayon gnome3 now, I have to say the gnome3 shell is much  more comfortable to use and it has a dash that is much more usable. Both  of these desktops will continue to be improved and I believe will  continue to be the desktop of choice in the future. None of us like  change, and I will miss the old gnome, but will continue to use the new  and watch it develop into something I can't be without. Until the next  change.... With that being said, I highly recommend you give unity or  gnome3 a try.

### Post by 3rdalbum on 2011-11-06
Gnome 3 Shell coexists with Unity in 11.10. I have both installed. It's really easy, and not dangerous at all; just install the gnome-shell package, log out, and then click on the cog icon on the login screen and change it to "Gnome".

When you log in, you've got Gnome Shell.

You can change it back to Unity at any time.

Gnome 3 is quite good, and its shell is very slick. However I prefer the extra space available on Unity, and I like the easy searching (Alt-F2 and start typing) and drag 'n' drop files onto Launcher icons. I certainly don't hold anything against Gnome 3 Shell, it is an excellent option.

I think Unity's shell will probably be the better one in the end. The development of it seems much quicker, and it's at such a good state even though it's very young compared to Gnome 3 Shell.

### Post by apollothethird on 2011-11-06
I like Unity.  I started using it in 11.04.  The unity in 11.04 was better and took less steps to start applications.  I just click on the "More Apps" icon, then click on my application and it was up and running.

The 11.10 edition would be great, but they added two clicks for starting and application.  Since I start hundreds of applications daily (some twice) the two extra clicks turn into thousands... double the work.

At present with 11.10 I have to first click Dash Home, then Click More Apps, then Click the option to remove the last search string, then click my application.

The recent applications area is great.  But you have to clear the last search string to see your favorite applications.  I've been trying to find a way to put the "More Apps" lens back on the launcher and remove the last search string.  Every time the update manager notifies me there's an update I rush to install it in hope that those two areas would be resolved.  I'm sure it'll be a matter of time before the developers notice this oversight and fix it.

Many people are noticing they have to do more mouse movements and clicks to start applications in Unity.  But a number of them might not be able to describe so clearly the problem.  Hopefully someone close to the developers will pass this important information on.

Other than that and the fact that there doesn't seem to be a way to stop applications from opening up in full screen, I'm enchanted with 11.10 also.  I never run applications in full screen.  So most of the time when I start an application I have to do three or for more clicks and mouse drags to get it out of full screen and get it workable.

When you consider the size monitors comes these days and the conveniences of having more than application in view I can't understand why so much design is placed in forcing applications to be full screen.  I see so many people struggling to find the application they want because they run everything full screen and have problems with the convenience of drag and drop.  The problem is compounded with the fact the Unity's launcher doesn't get you an option of selecting which of a program group you can pull up.  It's either all or nothing.  I'm hoping these components will also soon be addressed.

When I get time I'm going to try to network with the developers to help insure these flaws are taking into consideration for the next release.
