### Post by dg_vieira on 2010-12-15
My Ubuntu experience- grab some :popcorn: is going to be a long story Lol

 

 Well I would like to start of by saying wow, ubuntu has gone threw some major changes.
 The first time I heard about ubuntu and tried it was when it was a version 6.06 LTS and they sent me a copy to test out. I was very interested to continue to use it but I had problems with my drivers and sound cards not being detected and so on. So a few months went by and I checked back to ubuntu and it was then at edition 9.04. tried it but continued to have the problems so I left it to the side again. Waited a few months later and found out about ubuntu 10.04, tried it out and wiped out my old xp install.
 

  Updated to 10.10, and ever since 10.04 to 10.10 ubuntu has been everything I looked for in an OS.  I never seen a more perfect OS then linux, I like how it installed all my drivers on its own and was ready to use in a matter of minutes. I have been a windows guy ever since I was 7, now im 20 and love linux and am never turning back to windows. I made the change this past october and been enjoying linux ever since. I wish that I had made the change way earlier. Linux is very easy to learn.
 

  I like the fact that I can test out different distro versions in virtual box. I love how easy it is to install a program. And for the fact that I don't need to surf the internet hours searching for programs and drivers. I find ubuntu software center very useful, as it has everything some one can possibly want all in one place. Another thing I love about linux is, that I was able to bring some really old computers back to life that had xp on it. They were taking about 10 minutes to boot up and like 10 minutes to just open up a program, even after a clean install of xp, those computers still lagged, I guess because they had little memory. So I learned about “lubuntu”. Installed it to those old pc's. And man what an outcome. They boot up in seconds and everything was super fast. I like the fact that with linux, virus are a thing of the past. Now I can get what I need to get done fast and safe. Thanks to linux
 

 I was very disappointed at the posts of people saying they didn't like linux and were going back to xp. It kind of bothered me alittle. But what can you say everyone has there own opinions. To any new comer that reads this know that, everything you can do in windows, you can do in linux, just need to find an alternative program that will do what you need it to do.
 

 I work with mostly laptop/desktop repairs, so I tell everyone about how great linux is since clients usually ask whats the best OS for there computers. I point to linux. Even my dad, mom, bro and grandma loved linux after I introduced it to them :D
 

 I just see it like this from now on. Anything windows can do, LINUX can do BETTER.
 Windows is past, linux is future. IMO linux should come installed on the new computers manufactured and companies should just ditch windows over all. Since linux is the more superior OS :D
 

 After using linux and open source programs, I have found myself very interested in learning how to write programs and how to create web pages. Im actually looking for where to start so I can get to learning those things in linux. Linux opens doors for some great things and education.
 

 I would definitely like to get involved in the ubuntu community here & this ubuntu movement thats going on, and help as much as I can to help ubuntu grow and be the best. I will be hitting up these forums with some little questions I have about linux and setting up surveillance cam's and also live broadcasting in linux.
 

 I just want to say a BIGGG thanks to the developers of linux, and the ubuntu community. And just everyone that has contributed to make linux possible. Im very thankful from the bottom of my heart. You guys rock :D
 

 

 Sorry I made this so long, its just that linux is amazing. I can go on forever about how linux impacted my life :)
