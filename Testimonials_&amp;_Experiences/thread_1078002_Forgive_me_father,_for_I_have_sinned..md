### Post by wolfen69 on 2009-02-22
i have done the unthinkable and installed windows xp. :shock:

in virtualbox! yeah, i gave in because of a game. virtual pinball to be specific. i missed not being able to play it. it is hands down, the best pinball simulation (MAME) anywhere.

i finally figured out the nuances of virtualbox, like getting full screen, using a shared folder between linux and the guest OS, and having pen drives mount. if you were at my computer running my xp in virtualbox, you would think it was a regular install. you absolutely cannot tell the difference. if you ask me, it's even better, because xp boots up in 5 seconds! wow.

it took about 2 hours of researching to get vbox running the way i want, but man is it sweet. vbox even has 3D acceleration now!

don't think for a minute that i'm going back to windows as a crutch, it's just that this game is too good to stay away from anymore. i need it, having grown up playing pinball. it's in my blood. 

now my life is complete since i now have the one thing linux can't give me. but in reality, linux *is* giving me this, only because i can now play it *within* ubuntu. and if you pick seamless mode in vbox, you *dont even see* the xp part and just opens the xp app you want, and it opens up on the linux desktop. too cool for words. who needs wine anyway?

i have attached a couple screenshots of the xp desktop and pinball in action. i have about 150 tables to choose from. they are all authentic reproductions.

### Post by wolfen69 on 2009-02-23
> **loell said:**
> i've given up shared folder via vbox settings, i just settled for samba, did you ever figured out how  to share folders w/o resorting to samba?

yes, it's actually very easy. make a shared folder in linux.  (anywhere you want, and you don't have to make it shared, just make a folder) then open virtualbox, go to the settings for the guest os, go to "shared" folders or whatever, pick the path to it, start up the guest (windows) OS, start a command prompt, and do ```
net use x: \\vboxsvr\name_of_shared_folder
``` Enter.

next time you start windows the shared folder/network will show up in My Computer. download the virtualbox pdf how-to if unsure. that's how i learned it.
