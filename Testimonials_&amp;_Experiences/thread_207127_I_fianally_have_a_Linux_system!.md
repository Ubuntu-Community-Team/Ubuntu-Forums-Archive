### Post by Village Idiot on 2006-07-01
Hey Folks!....I've been using Ubuntu 6.06 for about 3 weeks now and heres the skinny. I installed it on a HP ze4547 laptop and it flies compared to Winblows XP! Its really nice to be able to easily configure a Linux system using a GUI but the command line is also easy to use thanks to great fourms like this one! Seriously, ANYONE can get this up and running to their liking without pulling their hair out! I now plan to build another computer and the next version of Ubuntu will be installed. From what I've read this OS is a lot more secure and I don't have to have a myrid of tools to clean out the nasties that lurk on the net, more time to enjoy this great OS. Since I have a good router and its set up right I don't even need a firewall. One of nicest features is the fact that there are thousands of apps and they are all free. Uninstalling and installing software is a breeze. This is by far the easiest Linux, in my opinion. Thanks to Ubuntu I fianally have a Linux OS. I've done many changes to make this my Ubuntu.
     I would like to thank everyone involved to make this happen! Thank You!
