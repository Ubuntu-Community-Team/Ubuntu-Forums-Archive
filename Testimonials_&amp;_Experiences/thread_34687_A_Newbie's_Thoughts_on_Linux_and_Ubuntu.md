### Post by ComPro on 2005-05-16
I have been a Windows user for many years now and I do a lot of work in Windows ranging from web surfing to graphic work to programming. Only recently have I decided to dive into the world of Linux.

Last summer circa July, I purchased a brand new computer with the intentions of installing Mandrake on it, seeing that Mandrake was supposed to be the most simple Linux distribution out there, perfect for Linux 'n00bs' such as myself. As luck would have it, Mandrake did not want to format my hard drive, so I was stuck installing Windows 2000 on my machine.

After a while, I felt like trying Linux again, but rather than installing it on my actual hardware, I installed it on a VMware. This time, the distribution was Fedora. It installed, but to be quite honest, I was very unimpressed. There were a number of things that did not work as they should have, and it was near impossible to figure them out.

After that, a friend of mine recommended Debian Linux to me, so I downloaded the ISO's and tried it out. It installed without a hitch, which was very nice for me, not knowing anything about Linux, and only a couple of things failed to work, which I was able to straighten out with a couple of phone calls to my friend. I began to like Debian so much that I installed it on my laptop. It was pretty good, but I never really did get the feel for it, for whatever reason. There were still some quirks, both on the VMware installation and the real installation, and I decided to reformat my laptop with Windows again.

Somewhat recently, I stumbled across the Ubuntu Live CD. I downloaded it and tried to run it, but it failed to boot on every machine I tried it on. I did not like Ubuntu at all after that experience and I vowed never to install it even in a virtual machine. That changed, however. With nothing to do and another taste for Linux experimentation, I downloaded the Ubuntu installer and ran it on a VMware. To my surprise, it was only one CD, and most distributions come with a mandatory 3-CD download. I really enjoyed that.

The installer seemed a bit slow, slower than most installations I had attempted, but that was only after the initial reboot as it was configuring everything. To be perfectly honest, I did not pay much attention to the installer. I just let it run, and I was quite happy that it did not require any user intervention.

After it had finished, Gnome loaded and I was on my way. To my surprise, everything worked. I had some updates, so I installed them. No problems. My biggest issue is with sound. My sound never seems to work in Linux without having to configure this and download that and go through a seemingly endless cycle of commands. Ubuntu had it working on the first boot. That definitely got points in my book.

I decided to take advantage of the apt-get tool, which I grew to love in my Debian experimentations, to get XMMS. It is because of apt alone that I am thrilled that Ubuntu is built off of Debian. XMMS installed with no problems, but it did not play an MP3 file. After a quick sound system switch in the XMMS preferences, I was jamming. I have run into that problem in the past, but it never was that easy of a fix. Once again, I give plus points to Ubuntu.

The only thing that I did not appreciate was the lack of other multimedia support. I was told that Ubuntu could not play any movies by default. I decided to try it out with a short MPG clip. Sure enough, it would not play. I figured that I could apt-get mplayer, but that didn't work. After doing some research, I found an Unofficial Ubuntu starter guide that took me through a series of commands to be able to install Mplayer and the various codecs. After about twenty minutes of that, Mplayer was installed. It wouldn't play the file, though.

I decided to give it a rest for a little while, but then I came back to it and decided to check the preferences. I switched the sound system in the Mplayer configuration to the same setting I chose in XMMS, and it worked. Ubuntu on my VMware can now do everything that I would want it to do, for testing purposes.

All in all, Ubuntu has been my most pleasurable Linux experience. Everything works straight out of the box, with the exception of multimedia applications, as it does not seem to be part of Ubuntu by default. If Ubuntu can have XMMS and Mplayer installed by default in later releases, I believe Ubuntu would be an A+ distribution.

-ComPro
