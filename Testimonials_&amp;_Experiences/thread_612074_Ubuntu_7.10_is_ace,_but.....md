### Post by DrMega on 2007-11-13
I've been using 7.10 (Gutsy?) for a couple of days now, and have to say it is ace. It must be said that although I've had Ubuntu on my home machines for over a year now, I am still mostly a Windows user, and like GUIs, and am by no means a Linux expert.

I can't fault Ubuntu 7.10. Everything just work, within an hour I had it all installed and configured up with surprisingly, absolutely no visits to the terminal window, even with the multimedia codecs and proprietary hardware drivers installed. It looks great and works great.

BUT, and this is trivial, flicking about the net, it seems most people agree that Xine is a better multimedia engine that GStreamer (that's certainly my experience), and the codecs package (ubuntu-restricted-extras) installs the Xine versions of the various codecs, why is Totem-GStreamer still the default media player I wonder.

I know this is truly trivial, as it takes all of a minute to swap to Totem-Xine, and it can be done through the GUI, but I can imagine an impatient Windows user experimenting for the first time dismissing at top OS because its default media player is not great.
