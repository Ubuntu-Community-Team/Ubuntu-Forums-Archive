### Post by ragrim on 2007-08-26
i just thought id add my 2 cents here, ill make it brief...

Ive been trying various Linux distros over the past 4 years and they are interesting and show promise, but i have gone back to windows every time, learning curve is just way too steep...

And then came Ubuntu, picked it up from a magazine cover booted the live CD installed the system in about 15 minutes and i was off and running, discovered apt-get and i was over the moon!

Now ive been using ubuntu for about 2 months, ive setup an ubuntu server as my gateway at work (100 or so users and 5 servers) and im soon implementing a mail server running on Ubuntu server. im even thinking about moving all our workstations to Ubuntu over the next 12-24 months since im seriously pissed at vista!

So from what ive experianced, good and bad im confident Ubuntu is the distro thats going to bring Linux OS on par with windows in the next few years, the support for Ubuntu is incredible, the learning curve is almost on par with windows (for general using) and the motivation behind the developers is awesome! 

So in short, Thanks Ubuntu and to the Ubuntu community and developers, thanks to everyone helping daily to get my system running. its strange, i find myself daily trying to give something back to the community for the first time ever, windows ive used for 5 years and not once have i felt the need to beta test or help in anyway.

Anyway, thats my experiance, just wanted to share it :)
