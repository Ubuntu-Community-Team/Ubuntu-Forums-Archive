### Post by isgie on 2010-03-14
Windows 7 looked pretty nice so I thought it would be a good idea to dual boot it with xp. I decided to get rid of my ubuntu partition to give windows 7 more space. I thought this was going to be the end of my linux experience. I used windows 7 for a few weeks and then installed karmic back. Whenever I boot up my computer, I go right to karmic. I still like ubuntu so much more than windows.
____________
I have been using ubuntu since August 2009. I started out with xubuntu because I was running an old computer at the time. Once I upgraded my hardware, I switched to ubuntu. I was amazed at how much linux could do. I was pretty shocked considering linux is not backed up by a large company. I never had issues with ubuntu jaunty. It ran perfectly on my old computer. I was also able to successfully activate compiz. I then wondered if the other linux distros out there were any better. I did a lot of experimenting and ended up going right back to jaunty. It starts up fast, runs perfectly on my hardware, recognizes my printer, and makes installing drivers for my graphics card easy. 

The update for karmic eventually came. I was excited to see what would be knew. I updated the day it came out. It took a while because the ubuntu servers were being killed by all of the upgraders but the wait was completely worth it. The new wallpaper and theme adjustments were great. Plus, I now had upgraded programs. At the time I did not know how to upgrade firefox on jaunty to 3.5.

I am still using karmic now. I have had some problems where I had to reinstall but that was mainly my fault for experimenting too much. When I use ubuntu for average purposes, I never have problems and that is enough for me.

I will never get rid of windows even though I prefer ubuntu but that is because I need itunes for my ipod touch.
