### Post by garetheld on 2012-04-29
It was made abundantly clear years ago that while other flavours of ubuntu would exist with other GUIs (kubuntu using KDE for example), the main ubuntu distribution would always use gnome.  With the release of the new LTS distribution of ubuntu i decided it was time to upgrade the OS on my laptop.  WHAT A MISTAKE!!!.  Instead of one of the most widely used GUIs in existence, I've now got a joke called unity to deal with.  It looks more like a mac than ubuntu and if I wanted a mac, I'd buy a mac.  It also seems to be very buggy.  After about 15 minutes of trying to use it, it crashed completely!  Ubuntu actually crashed on me!  If I wanted an ugly, cumbersome, buggy operating system that fundamentally doesn't work, I'd use windows.  Unity should be on a separate distro (something like unubuntu) for people who want to play with it.  I can see some possible uses, eg for use with touch screens.  It is not however suitable for a normal computer.  It's a lot slower than gnome and basic things are missing like the application menu.

Right now I'm trying to wipe unity and install gnome however I am having problems doing this, not least with the time it's taking.  This is the first time ubuntu has taken longer to install than windows!  Does anybody know a quick way of installing 12.04LTS.  I mean of course with gnome.  Unity simply doesn't count as a UI and as far as I am concerned the installation process is not complete until it has a working UI.

If there is no other way of doing this, can someone give me a few pointers as to creating a distro as I'd rather waste my time ripping unity out and putting gnome in once, than every time I have to install an OS.

### Post by Art1745 on 2012-04-30
I do not like Unity.  Neither do I like sudo codes.  I used dos and jcl eons ago; whenever I can, I go GUI.

To switch to Gnome, I open the Software Center and search on Gnome.  Gnome desktop utilities comes up almost immediately, and I install it.

When I re-boot there is a little gear (asterisk?) in the upper right corner of the login block.  I click on it and have my choice of several desktop environments.  I choose Gnome Classic.
