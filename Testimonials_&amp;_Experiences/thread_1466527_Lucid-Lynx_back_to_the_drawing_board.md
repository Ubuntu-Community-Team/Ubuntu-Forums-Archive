### Post by 3rdalbum on 2010-04-30
I'm sure you've seen my post in this forum.

WHY didn't you help test Ubuntu 10.04 while it was still in development? If you'd reported those bugs a month ago they could well have been fixed by now.

Ubuntu 10.04 is very stable on the hardware of people who have been testing it. That's no accident - hardware-specific bugs have been reported and fixed a long time ago. If you're able to test the pre-release version at the first beta and file a few bug reports with the "apport-bug" command, then you really should - it makes Ubuntu better for all of us.

It's still not too late. Please file your bugs with 'apport-bug'. For instance, with your Rhythmbox bug, run Rhythmbox, cause the buggy behaviour, go to the terminal and type:

```
apport-bug rhythmbox
```

Data about the current state of your system will be collected and attached with your bug report, assuming it's not already a duplicate (Launchpad will help you check for that).
