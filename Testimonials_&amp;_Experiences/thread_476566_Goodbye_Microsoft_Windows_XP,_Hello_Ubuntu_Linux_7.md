### Post by Jimbo2184 on 2007-06-17
Hi guys.

I've been using Ubuntu now for the best part of 5 months, in that time I've used Dapper Drake, Edgy Eft, and now Feisty Fawn. 

In that time Ubuntu improves with each and every release. I admit I was a little scared about upgrading the Ubuntu software and did backups just in case but I need not have been so paranoid because it always worked first time. 

I started with a modest 10GB partition and dual booted my computer alongside Windows XP Service Pack 2. My computer (for those who want to know my specs) is a Sony VAIO PCG-K115B Laptop. Its 3 years old now but the machine just keeps working, its a great travel companion as well. 10GB was a little cramped, I got everything I wanted in 3GB of software and was happy but my ever expanding collection of MP3's was getting to be a pain.

I eventually swapped out the original 40GB disk and added a much faster 80GB disk (with 8MB cache to replace the 1MB cache in the old drive) into my laptop, and formatted the drive as ReiserFS 3 for Ubuntu 6.10. Its been successfully upgraded to 7.04 now and I have never looked back. I've thrown away my Windows XP disk, I never want to return to a world of spyware, malware, slowdowns, unpredictable crashes and be told how to use my computer by Microsoft.

I've installed and removed a lot of software on my Ubuntu machine, and I am amazed that the system will not need defragmentation - ever. Linux is a very smart operating system in this respect, 5 months on and its still running well oiled, slick, efficient and smart. Windows on the same hardware slowed down to a crawl in 5 months of heavy use, Ubuntu doesn't it still runs beautifully out of the box some time later.

I also love the stability of the operating system, if X11 crashes for instance a simple change in runlevels is normally enough to get things back on track. Another thing I now cannot live without is the fantastic X11 session manager - I can leave all of my work, log out, and when I log back in all my applications reload flawlessly. Beryl and AIGLX give me that Vista(ish) look and I love my wobbly windows lol.

I'm a simple user, I burn CD's with K3B, amarok is the greatest music player on the planet, Firefox has to be without doubt the best browser, Evolution is the best email client and so the list goes on. 

Linux really does give me freedom, and I know with my now rebranded laptop (Ubuntu case stickers) I can switch this PC on, it works when I want it to and shutsdown gracefully when I'm done every time - now thats an operating system to be proud of! Well done guys for making a top-notch eXPerience for the little guys without much cash.
