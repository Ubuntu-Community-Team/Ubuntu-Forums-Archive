### Post by Hobble on 2011-01-11
Hi, this is my first post on this board.  I've switched to Ubuntu 10.10 about 2 weeks ago.   I was planning to try out Ubuntu for well over a year, I was just putting off getting an external hard drive so I kept dealing with Windows. Well, Christmas my girlfriend got my a 1TB external, and the first thing that came to my head was to get Windows off of my computer as fast as possible. Since the switch I ran into a few issues that I quickly resolved with the help of this board, and now I'm content with my computer for the first time. Ubuntu allows my computer to handle things effortlessly that would cripple it on Windows, I honestly didn't know truly know what my computer was capable of until I ditched Windows.   Every time I log on now it's just exciting to see what awesome features I can discover.

### Post by johnnybgoode83 on 2011-01-13
I made the switch to Ubuntu a year and a half ago as an escape from Vista and I havent looked back.  It started as booting into a live USB and learning it without making changes to my computer (i love the fact it is so lightweight it can run comfortably off a USB pen).  I have also tried other distros such as Opensuse, Fedora, Kubuntu and Mint and found Ubuntu to be the best for me.  I have had very few problems with the transition and any issues I did find were easily fixed by googling my question and getting help from the Ubuntu community who are much more helpful than windows users.  All in all Ubuntu has exceeded all of my expectations and I can see myself using it for many years to come
