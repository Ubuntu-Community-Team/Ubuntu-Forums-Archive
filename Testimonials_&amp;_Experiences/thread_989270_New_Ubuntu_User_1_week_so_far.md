### Post by MercilessMing on 2008-11-21
Not to get into the why I decided on trying Ubuntu/Linux Again (tried LIVE cds before but never installed)

My history with Linux (in order Slackware in the 90s/Fedora during it initial release/ubuntu ) is always played with it for about a day every few years to see where it has gone, for about 10 years now. My biggest complaint was the lack of some standard to install drivers and having to do EVERYTHING command line.

Well I installed Ubuntu 8.04 64bit last weekend and loving it.  Everything I did in Windows I can do now, (I no longer play games on PC just consoles so...), by career I am a ASP.NET programmer but Installed Apache, PHP and MySql and do all home code playing in this now instead of ASP.NET.  Ubuntu has put the fun in computers again for me.  My Rig is 4gigs Ram, Nforce 3 business MCP mobo, Nvidia 7200 128meg gs Refurb for 9$ (giving my dad the real vid card for gaming), Amd X2 4600 64bit 939pin, and using integrated Sound card.

I had no initial install problems, and with Mostly Synaptic and "Add/Remove" and Repository additions got everything I wanted installed and got USHARE working as well and used very little Command line to tweak what I needed to tweak as I learn.


Ubuntu and/or Linux has come so far from just 2 years ago when I last tried Linux. I am so far a Linux user now for home use and tell all my friends who don't Have to run windows to use ubuntu.  I think in so many ways as far as "Usability" Ubuntu has at least matched Windows Now.

I look forward to learning more from Fellow Ubuntu users here and hopefully becoming someone who can help.
