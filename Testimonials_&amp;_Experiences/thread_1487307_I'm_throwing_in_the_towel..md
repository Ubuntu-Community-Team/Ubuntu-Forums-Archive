### Post by 3rdalbum on 2010-05-19
If I were you I'd still check my RAM. Run Memtest86+ all night, or install Phoronix Test Suite and run their memory benchmarks; if the benchmark consistently causes a crash, then your RAM is bad.

Also, open up the side of your computer. Toward the top-right of the motherboard is where the RAM sits. Unlatch the sides of the RAM slot and the RAM will come out. Now, if the word "Kingston" is on the RAM stick, then you've got RAM that is either faulty or will quickly develop problems.

I've never personally had a faulty stick of RAM that WASN'T made by Kingston.
