### Post by lifve on 2008-05-10
I'm new with linux.
I just installed Ubuntu 8.04 LTS.
on Acer 5030.
I think it works fine.
at least I'm not using confusing windows anymore.
my laptop works fine, and the sound is better than windows, more configuration applicable.
early, I have trouble with my Huawei E220 modem. but it solved after I read post in here, thanks for Vodafone Mobile Connect Driver.

My opinion Ubuntu can replace apps that I used to use in Windows.
Totem (with some plugin installed) can read any my anime file.
Banshee can replace foobar
Transmission for utorrent
OpenOffice for MSOffice
GIMP for Photoshop (just for basic image)
Evolution for Thunderbird.
WINE for other Windows I need.

For me that apps enough.
And I happy see them works well. ^_^
I hope Ubuntu getting better and better.

...the only complain is the restricted ATI driver doesn't work well.
but whatever, without it my anime still show smoothly.

...goodbye Windows.
