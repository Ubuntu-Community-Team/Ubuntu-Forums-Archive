### Post by 3rdalbum on 2011-05-05
It helps if you can test new versions and report bugs, as far before the release as you can. If nobody has reported the problem during alpha or beta stages, it's quite understandable that the problem would make it to release date.

Especially for Ubuntu-developed software such as Software Center and Unity; I've submitted bugs for those projects and have had them addressed very quickly.

And a tip: Use the "ubuntu-bug" command rather than manually going to Launchpad. ubuntu-bug attaches relevant system information to your bug report and so it will be more likely that the bug will be investigated.

I'm actually not having any problems with Natty. It's working fine for me here, so your experience is certainly not true for everybody.
