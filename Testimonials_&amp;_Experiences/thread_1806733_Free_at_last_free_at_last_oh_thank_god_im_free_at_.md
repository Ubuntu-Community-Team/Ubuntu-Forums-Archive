### Post by 3rdalbum on 2011-07-20
It's funny you should mention binary installers (.bin files) and viruses in the same post - because the process that you did to add execute permission to the .bin file is one of the things designed to keep viruses out! If the user hasn't added execute permission to a malicious program distributed as a plain binary, then it simply won't be able to run.
