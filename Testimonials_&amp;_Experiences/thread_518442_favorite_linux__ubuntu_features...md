### Post by HermanAB on 2007-08-05
The best feature of Linux over Windows is security and immunity to crapware.  The maintenance effort required to keep Linux running is much less than with Windows.

The other advantage is the seamless blend from desktop to server.  If you need a FTP or a HTTP server, just start the thing - you don't need to go out and buy a different OS for several hundred dollars and install the whole darned thing from scratch just to get that.

### Post by Alterax on 2007-08-14
The features I enjoy most are:

Stability:  These are a pain to break once they are configured.  If it breaks, it took some effort.

Redundancy:  Configuration settings are not placed into a monolithic system such as the infamous "Registry".  Therefore corruption of one configuration is less likely to knock out the entire system.

Efficiency:  Efficient use of hardware resources do not require me to purchase the latest and greatest hardware to get a butt-kicking machine.  Also, a large software repository with the apt package management system permit me to easily install most software with a single command rather than hunting down dependencies.

Security:  The OS is designed for a multiuser environment from the ground up.  Windows was designed for a single user environment (and is working away from that).  This gives a CLEAR distinction between users and administrators, making it difficult for users to get privileges that they shouldn't have--a technique that viruses and "bad users" exploit.  This clear distinction makes it harder for one user to thrash a system or accidentally install something dangerous.
