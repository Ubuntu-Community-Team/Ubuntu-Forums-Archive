### Post by jperez on 2006-08-21
First off, Ubuntu is the future!

Secondly, here is my story with all Windows OS's, aside from Win3.1.

Well, I've been using Windows since the days of Win3.11 for Workgroups.  That was probably the only stable Windows I had ever used.  Windows 95 came along...the BSOD was the thing I feared the most since it ALWAYS meant something bad and at that time, I was still learning the OS.  When Windows 98FE came out, I was rather pleased...until the bugs started to come out.  Eventually Windows 98SE came out and that was better, but not so much.  In fact, the computer in which I am typing this has a combo of Win98SE and Ubuntu, but no need to worry, I'm using Ubuntu as I type.

Let us stop in Win98SE for a second.  This computer that I am using is a Gateway 433c with an Intel Celeron 433MHz CPU, 128MB RAM (Upgraded from 64MB), and a 20GB HDD that I had laying around.  I have about 5 20GB HDD's...I stock up on these whenever someone doesn't want them anymore.  Well, this kind of computer can run Win98SE very fast, in which it is still doing, but when your computer starts giving you the BSOD when you click on a donwload link in Firefox (downloading MediaWiki Software), then it's time to switch.

The very first Linux I ever tried was Pogo Linux some years back.  I think it was back in 2001 or 2002.  I used it for a while but never really got into it.  At that time, I was a Windows-fan and was running WinME...the WORST mistake Microsoft ever developed!  Enter Ubuntu:

Now, I tried Ubuntu back during Hoary Hedgehog.  I learned about it from a church member that uses it himself.  I installed it on an AMD Anthlon 600MHz PC with a 20GB HDD, 256MB RAM and a Soundblaster Live card.  Well, the two things that didn't work were the Ethernet card and the SoundBlaster Live.  Needless to say, I scarped that went back to Windows XP.  Now, I like Windows a lot due to that type of programs available for it, but stability wise...not my thing.

When I got my laptop, I installed Hoary Hedgehog on it an everything worked, except for the well-known system studdering.  This happens on some Inspirons, like my Inspiron 8100.  So, I was given advice to upgrade to Breezy and then to Dapper.  Didn't fix the problem.  I went backi to Windows XP Home on my laptop.  At least on my laptop, XP Home runs well, so I'm happy about that.

Well, after the BSOD happening with the Win98SE machine, one I'm using right now, I decided to take one more crack at Dapper right off instead of Hoary and then upgrading.  I already have the Dapper Alternate CD burned anyway from upgrading my laptop.  Well, it's running on this computer now and I'm happy!  The only thing is that it runs a bit slow at times.  Running at 433Mhz with only 128MB seems pretty foolish, but hey, it still runs pretty fast.

I've used just about every Windows OS made, except Vista as it offers nothing more then XP, IMHO.  Win3.1/Win95/Win98FE-SE/WinNT/WinME/Win2000/WinXP will never offer what Ubuntu can.  The only thing I need to do is learn Linux/*nix more.  I'm going to have to start learning it from the command-line just like I did DOS v6.22.  I'm sure in time, I will get it.  It's just harder to learn anything as you get older and have ADD.  Thanks for listening to my rant on Windows and my pleasure with Linux and Ubuntu.

Jesse~

### Post by bikeboy on 2006-08-21
Nice story.

Might I suggest you try using Xubuntu on the older slow computer rather than gnome, it should be a lot quicker. Best thing is, you don't need re-install or anything like that. Since you're not afraid of the command-line, type ```
sudo aptitude install xubuntu-desktop
```

Then logout, select session, xfce. You can still login with gnome as well, but you get to try out the lightweight xfce desktop. If you want to reverse the xfce install type ```
sudo aptitude remove xubuntu-desktop
```
