### Post by OutOfReach on 2008-06-18
Hi All!
    
    I absolutely LOVE Ubuntu and Linux! I've been using Windows for about 6 or more years, including Windows 95, 98, 2000, and XP. And Ubuntu really turned my world upside down (In a good way, if you know what I mean ;) ).
I've been using Ubuntu for a little more than a month now and I've already gotten used to the filesystem directories, and even some basic terminal commands! 

   I only wish that I would've actually known about Ubuntu/Linux earlier... Its so awesome! Ive even started making some simple command-line Python programs. And to think, I used to be a Windows fanboy :oops: . And IMMEDIATELY after I upgraded to Ubuntu 8.04 I noticed an amazingly drastic change in performance. Ubuntu ran a (Excuse my language) hell-of-a-lot faster than my previous Windows XP installation ever did!

I can get school reports/powerpoints done in OpenOffice, produce my music in Ardour and LMMS, Browse the web with Firefox, IM with Pidgin, Listen to my iPod with Banshee, edit my images with GIMP, write programs with Eclipse, and the list goes on, Ubuntu has it all! :guitar: 

Im on a Dell XPS M140 that I bought in '05, and Ubuntu recognizes everything, like it was made for my machine! It recognizes my wireless mouse, my printer, all my USB drives, speakers, my MIDI keyboard, and my iPod. I love it.

Sorry this is such a long rant, but Im just so excited of what Linux has to offer.

Anyways, Im honored to be using Linux and I hope to become part of the community! :)

Linux for life! :popcorn:
