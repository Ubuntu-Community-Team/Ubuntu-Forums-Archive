### Post by jedispork on 2009-08-31
I finally installed Ubuntu to see what all the fuss is about.  I am very impressed. A operating system other than windows that works.  I did have some issues with dual booting vista.  I had to use a program that edits the windows boot loader because grub just didn't work for me even after consulting the help thread here.

After I have my dual boot going I download amarok. Everything looks great but for some reason it won't play mp3 files?  even after i install the restricted package they won't play. So I do some more research and find out that amarok requires some additional packages to play mp3 files.  Why can't the program prompt you that you need additional setup to play mp3 files?  Seriously don't you think most people have at least a few mp3's? Now once I have that all setup and listening to my music under ubuntu I start contemplating making a total switch. 

I start to notice issues with sites like youtube and playing videos. The default selection for the open source flash plugin was just not up to the task.  Another problem solved by just installing the official adobe plugin.

Cleansing my pc of windows sounds so cool. I think a lot of people use linux just because it makes them feel elite that they are not using windows.  Then whoops run into another problem.  Trying to switch sound over to my headphone amp seems to have broken everything.  I can no longer get sound from my internal card, just a crackling noise when sound plays. I come back here and find a thread about sound issues. I attempt to reinstall the sound packages following directions in a thread and it completely messes up my system. 

At this point I'm not frustrated but starting to realize ubuntu is not going to work well on my desktop.  Ironically I still like windows for some of its free applications. Imgburn is one of them and I don't think its matched by any linux app.  I also run a media server and important programs like anydvd, and many others just don't work well in linux.  Right now anydvd is the only option for decrypting bluray movies.  

Even being a very casual gamer trying to get everything going under linux would be frustrating.  I really want to use ubuntu but I just don't see much point in switching over just to do some web browsing and then going back to windows to use programs that I need . 

I don't like that its a windows world but thats the way it is.  The malware problem on windows is totally unacceptable but I've been able to get along by using avira which does live virus/spyware all for free. Even now I'm still contemplating a switch but the effort to find compatible programs for linux feels futile. The operating system is given away for free yet still not enough people are using it to get mainstream support from software writers.

Thanks for reading. Again I want to say I am really impressed with Ubuntu and do plan to install it on secondary computers or annoying relatives that beep breaking their windows machines.

These two issues are related, you see neither the latest version of Firefox or Pidgin are mission critical updates.
Usually the packaging team does not deal with so called "latest and greatest" software, usually the software included with the system and the software in the repositories is the last known stable version of said application.
This means yes your software is still secure but if the software has a small update with one or two patches here and there it wont be available.
The latest firefox and pidgin are not mission critical updates so they are not officially listed.
Ubuntu has an odd way of maintaining its repositories, for some this is fine but if you want "latest and greatest" then maybe Ubuntu is not for you.
You may want to try other distributions alongside of Ubuntu, things like Mandriva, openSUSE, Mint, whatever might suit you better.
