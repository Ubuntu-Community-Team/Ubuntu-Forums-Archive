### Post by ptn107 on 2007-08-29
My curiosity with Linux in general started about a year ago.  At that time I knew nothing about the concept of open source and free software.  You can say that I've been a slave to Windows since the good old days, using every version from 3.11 to XP Professional.  Well after searching the net for a few days reading up on the various distributions I decided to go for it.  The very first one I downloaded was Ubuntu 6.10 and I've been hooked since.  I used Ubuntu for practically everything except some games which I could only get to run in Windows at the time.  It wasn't until Feb of this year that I ditched Windows completely from my desktop, laptop, and my home server \ media center.  The release of Feisty in April sealed the deal for me, I wasn't ever going back.  In the recent months I've tried to help out, so I joined up with the bug squad and documentation team for Ubuntu.  I look forward to getting even more involved in the future if possible.  Kudos as well to all the hard work the community has put into this OS.

Desktop -> Ubuntu 7.10, 8.04 (alpha 1)
Laptop -> Ubuntu 7.10
Home Server \ Media Center -> Ubuntu 6.06.1
