### Post by Mark Phelps on 2011-11-10
Arguing Windows vs. Ubuntu in terms of ease of customization is fruitless -- as Windows 7 has a very good forum, and there are tutorials that deal with customization.

But basically, you are correct in your assessment that Unity is harder for the novice user to customize than older versions of Ubuntu.  And, given Canonincal's insistence of Unity as being the path forward, I believe that is by design.

As to the fragility of 11.10, I'm going to break from the pack and agree with you there, as well.  I've been messing around with Ubuntu versions since 7.10, and have done a LOT of customizing, especially with different desktops and themes, and with 11.10, this was the first time that my desktop crashes simply because I opened a configuration app (ccsm).  To me, that indicates a desktop, or app, that really is too fragile for widespread distribution.  But, as I said, that is my opinion.

The standard "guru" response here is -- quit complaining! You can always choose another desktop (like Xubuntu), or another Distro (like Mint) but that just sidesteps the real issue -- that Unity should not have been forced on us as the default desktop before it was really ready for that.

Years ago, we got serious, heated disagreements as to which desktop was the best -- Gnome or KDE. Now, with Unity, we're seeing the same thing with Unity vs. Gnome.  But in this case, Canonical appears to be taking sides with Unity.

And that last situation, again IMHO, is going to drive away a lot of existing users and new users.
