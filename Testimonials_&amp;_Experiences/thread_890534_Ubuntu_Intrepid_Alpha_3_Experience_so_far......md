### Post by nickdbliss on 2008-08-15
I have been testing and using intrepid ever since its first alpha release. I am trying it on two laptops one is hp nx6325 and other is hp nc8430 and two desktop pcs. 
Mostly i use my laptop nx6325. So i installed alpha 3 release few days back and my experience with it is just superb. 

No problems with graphics, wireless, sound and other things that i faced in hardy and previous releases. 

The default graphic driver is just so good u need not to install restricted drivers. It gave me fps of 1100 and DRI enabled out of box. Compiz worked like a charm. 

With wireless i had to install ndiswrapper n when clicked on connect it actually connected with my WPA enabled network. In hardy i had a lot of headache doing it. 

There isnt any problem with sound when u have headphones connected like it used to give earlier. I could hear sound from laptop speakers n headphones in previous releases.

But still it has some bugs. Like the bug of usplash. It shows red n white stripes during startup. Keyboard n mouse stops to work during login screen. And some others which i have reported on bugs.launchpad.net

Since it is still an alpha release, i bet when it will be released finally in october, the way we look at linux will change  forever. 

Hats off to Ubuntu Team. Cheers.
