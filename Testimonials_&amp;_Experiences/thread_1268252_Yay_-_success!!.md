### Post by Rob Maddison on 2009-09-16
I thought I'd share success as well as problems.

I've done battle with Terminal over the last couple of days and have finally given myself a shiny new /home partition :-)

I did this all by my little self with no help from the forum (well direct help anyway). The main reason for doing it with no help was the challenge and every rotten blog or instructions page saying it's easy. It wasn't 'easy' for me but then I'm not used to terminal or Linux.

I got lost yesterday and had done the last line in /etc/fstab wrong too so nothing worked. Result: reinstall

I got it wrong today and got lost again (today was classic noob - I ended up copying the /home from the live CD into /newhome as I was still in the Live CD boot wasn't I? :-D) Result: frustration/reinstall

Tonight with the brand new reinstalled system it worked, but not without a terminal boot to remove the extra fstab which wasn't working right again (to many spaces), and a recovery mode boot to add the chown and chmod instructions as per Psychocats version, and redo the fstab instruction.

I now know two things:
1. I know squat and have to learn quickly if I want things to work better
2. Back ups are useful!

If you're having problems now it's likely somone else has had the problem already and very likely they're not dead because of it. It's not the end of the world just persevere with reading up, even if that means rebooting into live cd to access t'interweb thingy. 

Thanks for the forums and everyone that helps noobies like me.

Rob
Northampton
100% Ubuntu for one week.
