### Post by magneze on 2009-08-17
My non-computer literate sister is now a big fan of Linux! Her aging laptop was dying under the sheer weight of XP and it was doing that standard massively slow boot up which afflicts all Windows installs after a few months use.

I installed Xubuntu 9.04 and a bunch of apps to replace what she had and she's really happy with it. Everything works well and much faster than before.

Biggest problem I had was transferring her iTunes music over but even that was possible eventually. I wish the MP3 codecs were installed by default though as all the music ended up as Ogg Vorbis when I imported the music using Banshee which can't be stuck onto an iPod. Still need to get that all converted to MP3.

9.04 is really a big step if users which previously would only touch Windows can be transferred like this

Things which needed improving during the install:
* WiFi card wasn't supported OOTB and needed some hacking about
* MP3 codec wasn't installed by default

Finally, Apple really need to think about iTunes on Linux or they're going to lose customers - Amazon MP3 is going to clean up in the Linux market!
