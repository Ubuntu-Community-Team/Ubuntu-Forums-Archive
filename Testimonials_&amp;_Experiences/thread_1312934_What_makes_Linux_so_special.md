### Post by rahilm on 2009-11-03
Hello..
I am a fairly new Ubuntu user (If you consider 9 months as new). I found it in a magazine DVD last year and wanted to try it. (WARNING: THIS IS A VERY LONG STORY, SO READ ON IF YOU HAVE THE TIME) So i installed it via Wubii because someone on psychocats/ubuntu told me to do so and tried to understand this new realm. I had to say that I hated it more than anything (I was using 8.04 at that time). The new GUI, overuse of the command line for doing things that were done automatically in windows (like mounting drives) etc really was irritating at that time. I uninstalled it within a week and decided never to use it again. But fate had some other plans. It seems like my PC was infected by virus (Win32:Vitro google it and you will realise how serious it is) I somehow sneaked into windows and installed 8.10 via Wubii.. and then i was forced to use ubuntu for one and a half month (2 reasons: i didn't knew how to format my drive, and i was too lazy to call an IT guy to do it).

After that I decided to risk it and formatted my hard disk. REInstalled Windows but created a new partition for ubuntu and have rarely gone back to windows(games!!) since then. I upgraded to 9.04 which is my current release and I am planning to upgrade to Karmic withing 2-3 months.

The problem is (I think it is a problem) I don't feel like going back to windows. Never has a feeling come down that I rather use windows for this and that. Windows is still there and my brother uses it regularly but I don't. All my friends use Windows , I don't.  I have been using Windows for over 8 years by now, nothing that i remember compels me to look back.

So what makes Linux so special (ubuntu in particular). Is it the openness? is it the secure architecture? is it the lack of spyware and other malware? or is it because everyday there is something new to do .. something new to learn about your computer. I feel like Windows was hiding my computer from me. I now actually like the command line, the new GUI, all those unnecessary commands to mount drives, etc. Some of my friends think that ubuntu has brainwashed me. into liking it.

What do you think..? What makes linux so special?? Over Windows, Over Mac, over BSDs and other stuff

### Post by tripolitan on 2009-11-03
The reasons that make Linux special for one may not be the same reasons for others.

I simply decided that I have had enough of Windows (back in 1999) the rest is very similar to your experience. I have discovered and learned more about computers in less than a year than I have learned in the previous 7 years (yes, I'm old, started with DOS on a 8MHz XT).

The BSDs are very similar to Linux but I saw no reason to drop Linux in favor of another platform. I find MAC to be similar to Linux as well and runs on esthetically-pleasing hardware but pound for pound/chip for chip, it is not worth what Apple is charging + MAC lags Linux in features.

Now I run several servers/desktops running on "ancient" hardware: Ubuntu 8.04 LTS semi-local server (Apache+MySQL+Drupal+Icecast+Squid+SAMBA+FTP+ssh) on an obsolete Celeron 733MHz/512RAM. Karmic on first-generation P4 1.4GHz/256RAM !! 8.04 LTS on an old home-made dual Athlon MP 2000+ 1.5GB RAM server and my workstation is a deprecated IBM P4 3GHz/2G RAM running LTS with 3D. (+ 2 obsolete NOKIA N800 internet tablets 128MB RAM! running MAEMO).

As you can see, Linux does a whole lot more, comfortably and securely, for a whole lot less.

### Post by running_rabbit07 on 2009-11-04
What makes Linux so special to me? 

No Anti-virus constantly slowing down my machine and wanting my opinion as to what to do with suspected intrusions. 

If you stay a little behind on the distro releases, you will have a very stable system that needs little to no maintenance.

If you need a new program to, let's say, tune your router, you will find one in Synaptic that should work with your router.

When you ask people for help with a Windows machine, they hold their hand out for money, when you ask a knowledgeable friend for help with Linux, they say ooh, can I help?

When others try to talk down to you for using Linux, tell them if they haven't used it enough to try and make it work, then they do not know what they are talking about.

Ubuntu is fun to learn and even more fun to break.
