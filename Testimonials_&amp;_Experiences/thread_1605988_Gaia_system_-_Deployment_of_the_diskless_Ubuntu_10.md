### Post by kangarooks on 2010-10-26
Hi

I have a nice story to tell. I on my own managed to create a system that gives all of my uni's computer lab computers (around 200 of them) an Ubuntu desktop 8)

Full story is on my blog &#8594; [http://pleasanthacking.com/2010/09/29/gaia-system/]("http://pleasanthacking.com/2010/09/29/gaia-system/")

Till now all of my uni was 100% windows shop when dealing with desktops, but from now on, it has a chance to also boot into the Linux desktops.

I created Gaia system from scratch and by hand, without currently used automated systems for diskless booting, since they looked all too clumsy for me :P It's only Ubuntu server 10.04 for VM server and Ubuntu desktop 10.04 with some modifications as a source for all desktop systems.

[IMG]http://pleasanthacking.files.wordpress.com/2010/09/image137.jpg?w=300&h=225[/IMG]

All of the deployment was performed by me installing prepared by me vm into one of school's servers, and helping set proper stuff on school's dhcp, and give an entry for pxelinux and that were all changes made on school's systems. After these small changes all control is transfered to my system, only thing left to do was switching the desktop itself into a 'boot from net' mode.

More stuff about my Gaia system is on [my blog pleasanthacking.com]("http://pleasanthacking.com/2010/09/29/gaia-system/") I haven't made my mind on this, but also I think I will place on my blog my BSc thesis which I created around my project, so you could get into more detailed description of how this whole system worked. 

It all wouldn't happen without your help, you good people of Ubuntu dev team, Ubuntu forums and irc. Thanks once again for support and making Ubuntu as awesome as it currently is =D>
