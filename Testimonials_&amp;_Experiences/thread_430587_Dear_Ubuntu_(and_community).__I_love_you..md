### Post by kpel on 2007-05-02
Since the Beginner Talk forum is usually a place for newb help requests I figured I'd change the pace a bit.  I've been a Windows power-user for years and finally took the dive and tried out Linux about 3 weeks ago.  I settled on Ubuntu primarily because of the community.

I have to say, the whole experience has been positive.  After 'unlearning' all the Windows stuff and then learning the basic Linux things I find myself already comfortable with the OS.  I still know far less about Linux than I do about Windows and yet I feel confident in my ability to solve any problems I have in Ubuntu, in part because of the ease-of-use and to a greater extent because of the excellent support that is out there (both of Linux and Ubuntu-specific).  The transition for me was a bit easier because I'd already been running open-source or alternative programs in Windows for some time now, so most of them already have Linux builds available.

The thing that really 'sold' me though was the compatibility of the system.  Heh, go figure-- since it's still a struggle getting most manufacturers to provide a Linux driver.  The LiveCD had me with full internet access **and** sound.  I could list all the ways the install is better than Windows, but suffice it to say that it's better in probably every way.

I'll admit that there are a few things that I need to get straightened out.  I didn't know if I had 5.1 sound support, so I decided to check it while Amarok was running (with my entire CD collection re-ripped to .ogg)  I'd been using whatever the base sound setup was and finally decided to look into it.  And that's when I realized that the only reason my surrounds weren't playing was because the individual volume controls for the surrounds were turned all the way down.  Full 5.1 sound and not a single tweak needed on my part (except for turning the control sliders on).  *gasp!*  That very thing happened right before I started this post--it was what prompted me to.

I found out yesterday that Linux does in fact recognize the side mouse butto**n** (Logitech G5) as I was playing around with World of Padman and it captured the click for binding.  I just need to get it working in Opera now.


My only true problem now is that I'm too busy customizing Ubuntu to boot into XP and game.  I wish I'd never stumbled onto the -look.org sites.  I have a bad habit of messing with everything from the GRUB boot screen to the on-screen fonts.  Everything is nice and pretty with a fully functional Beryl+Emerald and the entire system running as it is right now (Pidgin, Opera, Amarok) is still 100 MB easier on my RAM than my heavily trimmed and pruned XP install *without* anything running.

I am a convert.  I would have deleted XP and never gone back to Windows if I'd been able to move all my games over too, but alas, it is not to be (yet).  It's probably for the best, though.  If I'd found out I could play the same games in Linux I'd probably have geeked-out so hard I would have dropped dead.  :-D 

So thank you to the Ubuntu team for putting together a great distro that 'just works'.  Thank you to the Linux community for putting together such great software, and for being so incredibly helpful.


Now to go mess with all the different docks I've found so that I can get rid of my lower panel.8-)
