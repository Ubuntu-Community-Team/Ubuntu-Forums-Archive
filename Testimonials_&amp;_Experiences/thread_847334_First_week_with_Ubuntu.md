### Post by wbutchart on 2008-07-02
Im a complete Linux novice, i dont really know what it means.

I downloaded ubuntu because a friend who works in computing told me it is the best option.  I got to that stage because windows is getting worse and i want off the sinking boat early.

So to the install, i downloaded the live CD - didnt work i need the text files version.  I then downloaded that, ran the install and partitioned the hard drive - something went wrong and it failed.  Took the PC with me to church (had to do youth work) left the install disk running for a second time, partitioned it again (no idea what i was doing) and this time it worked, not only did i have ubuntu but the parition thing was so obviously simple a notice like me could partition it and keep XP on the hard disk as a option.

My views of ubuntu are so far - very impressed.  It starts quick, does what i ask and apart from some extensive loading has been really good.

I love the add/remove program feature it makes it easy to do things.  I may have a lot to learn when it comes to linux but from experience i would say that the average user could now download and use it no experience required (i managed!).

Ubuntu seems to be making that which was once tech guys only into something that is a good option to the public.
