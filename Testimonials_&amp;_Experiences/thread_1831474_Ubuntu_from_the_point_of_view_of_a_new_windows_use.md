### Post by nothingspecial on 2011-08-23
This is just how I feel after using windows for about 45 minutes, I know that all operating systems have their advantages and disadvantages. I also know that I am a rare (becoming less rare) breed of user who came to computing through linux and thought that because of this, my experience was worth posting. I've seen it from the other point of view times too numerous to mention.

Some background....

My first computer came with Ubuntu 7.04 installed on it (a second hand gift). I have acquired other computers since then and booted windows for an hour or so. After a while I realise I'm more comfortable with linux so I have always completely nuked windows. Now, my son has an iPad and my wife is about to embark on a college course that requires Windows. I have tried windows in a VM but it's too slow, ok for messing about with I suppose but not for actual use/work.

So today I installed windows 7 to a partition. The installation went fine and I have fixed grub, no problem.

My speakers don't work, my usb turntable doesn't work, my printer doesn't work. Ubuntu just uses them as soon as it's installed. I haven't tried my wireless keyboard, sansa clip, camera or other stuff that just works with Ubuntu yet. 

Ok, I realise the whole point of installing windows was to use one specialised bit of software and one specialised piece of hardware so I can forgive the turntable, but a printer and some speakers???????

Then there is the fact that having only ever used linux I always format external drives, sticks etc ext3/4. Windows doesn't know they exist. I know that read/write support for ntfs was only included by default in Ubuntu from 7.10 but that was 4 years ago. Why doesn't windows repay the favour?

Then there is the software included with windows 7 home premium. I have a media player, a browser, a note app, a paint app, solitaire and err........ that's it????

When I install Ubuntu, ootb I get all sorts of fantastic software ready to go.

I haven't even tried to start networking the thing yet but I know I'm not looking forward to it.

So thank you Ubuntu and all the free software projects that have allowed this operating system to become what it is. From the point of view of a long time Ubuntu/linux user and a brand new Windows user, I'd say it's Windows that isn't ready for the desktop..........

.......I also realise that I am just experiencing what many first time Ubuntu users face from the opposite perspective so excuse me while I go and find an alternative to mpd, radiotray, get_iplayer, etc etc.......

...... and I've no idea how I'll manage without bash and all the cli utilities that make linux easy. I suppose I'd better go ahead and join a windows forum.

### Post by XubuRoxMySox on 2011-08-23
You're definitely the exception to the general rule!

Most of us come from Windows because when we bought our 'puter it just came with Windows - and a bunch of software thrown in by the manufacturer. A "bare bones" Windows installation doesn't come with all that extra stuff (Adobe Acrobat, anti-virus, etc) and almost no drivers! The OEMs put all that stuff in there. Windows by itself is not that big... it just "requires" all that extra bloatware in order to do what X/K/Ubuntu does right out of the wrapper (for a lot of folks anyway) - drivers and all!

The other thing that used to bother me about Windows was that only only cure for "Windows rot" (a geek's term for that gradual but apparently inevitable slllowwwwwdowwwwwwnn of Windows over time) is a new installation of Windows - which wipes the drive clean and then you have to get and add all those drivers and bloatware again to have a usable system.

With Linux I can mess with it 'til it breaks, and if I can't fix it, a re-install is done in a few minutes *without* losing all my saved bookmarks, e-mail folders, schoolwork, music, etc. That can't be done with Windows either.

I'm hoping I won't run into a college course that *requires* Windows, but if I do I'll either use it in a virtual environment (I already paid for the license when I bought this 'puter) or just use one of the school 'puters.
