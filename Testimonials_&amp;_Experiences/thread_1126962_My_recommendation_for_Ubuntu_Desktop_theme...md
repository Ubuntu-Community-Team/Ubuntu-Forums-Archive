### Post by JoaoMachado on 2009-04-16
Actually, what I did with my Desktop was just use existing themes and mix and match with Icons, Themes and Window Decorations etc.

First, make sure you have gnome-art, if not download it using Synaptic.

Second, download Screenlets also in Synaptic, this is the app for the widgets. (If you like vintage clock, download the clock theme from [gnome-look]("http://gnome-look.org/content/show.php/Screenlets+Vintage+Clock?content=85030") ) Note: in order for this Clock theme to work, you must install the theme in the global directory "/usr/share/screenlets/Clock/themes/".

Fourth, Change the Default Theme, from Human, to ClearLooks then Click on "Customise"; For  Controls= Clearlooks, for Colors=default, for Window Border=New Wave, for Icons=Human.

If you do not have the New Wave window border, you can download the New Wave Theme using Gnome-Art from the web.

Fifth, the Background is just a Hunter green color of you choice using the "Change Desktop Background" by right clicking on the Desktop.
My settings are as follows; Hue:148, Saturation:54, Value:30, Red:35, Green:75, Blue:54, Color Name: #234B36

And the last step is to set your top and bottom panels to about 50% Transparent/Opaque by Right clicking on the Panel>Properties>Background Tab and moving the slider to the middle.

Enjoy,

Joao
