### Post by laptoppana on 2008-08-24
I am a very new to Ubuntu, after few days experience with new Ubuntu and my panasonic, now all things seam working OK+

Laptop configuration:
- 1,2 GHz, 512 MB RAM, 40 GB
- WinXP (11GB) and Ubuntu (4GB) on C driver
- 25GB on D driver for data

After install Ubuntu:
Hardware:
- Video: working out of the box
- wireless: working out of the box, with XP, I spend half day to have a right config! With Unbuntu, it does install it self, great!
- keyboard: working out of the box, this is a wonderfull thing because the key board is mess in windowXP, a lot of key is in wrong place, like ()$@:;... 
- touchmouse: working out of the box
- Hotkey: have to install pcc_acpi, this is a direct and correct solution for that: [http://ubuntuforums.org/showthread.php?t=501766&page=2](http://ubuntuforums.org/showthread.php?t=501766&page=2), thanks!. However I found that when I push Fn+F3 (switch monitor), the system will hang, so do not touch it!
- Sound: This took me more time to make it work  , and I found this help me a lot: [http://ubuntuforums.org/showthread.php?t=363365&page=3](http://ubuntuforums.org/showthread.php?t=363365&page=3), thanks! Some note to sound problem: some people found that their sound card work when they untick External Amplifier in Alsa mixer, but with my case, my sound only work when I TICK that option. This is quite a trick!
- Battery: I got 4h30 - 5hour, similar when I run XP, thought a little bit less when I run Ubuntu, may be because of Compiz?
- SD card: not work or I have no idea how to make it work
- CPU speed step: 600 - 800-900-1200 MHz

Software:
- Supspend: work out of the box
- Hibernate: still not work, it hang after I push that function, any people known how to make it work?
- Compiz: Fantastic! You should try it! And cube work event with my lowend video card, with memory share about 20MB! The system only go slow if I enable transparent in cube, however I found that is not so extractive.
- Proxy: it work with my company ISA proxy, this is suprice me a lot! 
- ThunderBird email: still not work if I have to connect Internet through proxy. This is still some limit from Thunderbird I think
- Wine: Office 2003 work great, ACT not work, Blackberry Desktop not work

This is my first time to join to Linux/Ubuntu world, but I feel I will stick to it more! Great work! And FREE!! THANKS!.
