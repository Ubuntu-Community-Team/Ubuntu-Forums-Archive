### Post by vrangforestillinger on 2008-04-26
Once upon a time, after many years of Windows 95/98/XP, i tried out Linux. I did not feel at home. I had problems with the simplest things, like installing programs. I had to manually edit xorg.conf to get the resolution i wanted. But after some time, it grew on me. 

Now, I just realized, its the other way around:

I wanted to show a friend a file I had on my computer. Since Im running openssh-server to be able to get to my files remotly, I tried to find some kind of ssh- or scp-client on his windows-laptop (XP). But I found none. My friend never needed ssh, fair enough. (And it seems there was none installed by default. At least I could not find one.)

So we went to install one. First I had to search the internet. WinSCP seemed to be a popular one. So we downloaded that one. Before we started the install, I scanned the file for virus. You never know in Windows.. 

So the install started. "License" blablabla, "Install directory" blabla, "shortcut on the desktop?" blabla.

Then suddenly, *red alert*, "this program WinSCP-install is trying something fishy on your computer, will you allow it?" My friend is a routined windows/antivirus user and quickly chose the right answer. The install proceeded, and was finally finished. But it was not over yet.

I tried to connect to my computer, then again, *red alert*. We had to allow the program to connect to the internet. Then finally, after having to search the net, download an installfile, scan for virus, answer a ton of questions, allow this and allow that. I could connect to my computer.

Man how I missed the linux-terminal, where all this would be done in one command. (Two if a client had to be installed.) 

Now, Im in for an 8.04-install..
