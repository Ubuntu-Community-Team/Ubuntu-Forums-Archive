### Post by Jim D v2.0 on 2009-08-16
I just had an amazing experience that I would like to share.

Last week I had some hardware problems: either the CPU or the memory was acting flaky, and according to Ubuntu 9.04, memtest86+ it was my memory. I figured that it was time to do some hardware upgrades (most of my core hardware was 4+ years old).

I ordered a new MB, CPU, Graphics Card, Memory and 2 new HDDs. I got everything assembled and set it up to boot from the old IDE drive that had Ubuntu already installed.

Without any problems, the “new” computer booted right into Ubuntu, recognized all of the hardware changes and worked like a charm. I thought that at the very least I would need to re-install the OS.

Somehow, I suspect that given MS's DRM policy and activation features, that if I had been using windows this would not have been possible.

Major thumbs up to Ubuntu and GNU/Linux.
