### Post by 23meg on 2008-06-11
> **Keiji said:**
> I already do update packages in the package manager. How is this any different, then?

The updates that you install in a stable release (in other words, a released, final version of Ubuntu, as opposed to a version that's still being developed) [consist of high-impact bug fixes and security fixes]("https://wiki.ubuntu.com/StableReleaseUpdates"). The updates that are available in a development version are entirely different: they're building blocks that are stacked on top of each other that lead to the final release, and contain new upstream versions of packages, or Ubuntu-specific modifications, or both. They can introduce drastic changes, break things, go crazy, in a way that stable release updates can't.
