### Post by mje1708 on 2010-03-26
Hi

My advice is don't give up and don't despair.

I was fairly new on Ubuntu up to a few weeks ago. I also had issues when I first started but with the help of the forums here I have resolved them.So take each of your problems in turn and resolve them one by one. Post the details of your setup and the exact issue you are having and I am sure somebody can advise you.

It is inevitable that after using Windows you will find it takes a little time to get used to another operating system. I was the same. However now after using Ubuntu for just a few weeks, my Windows (although still installed) is never booted up. Ubuntu just seems to run my PC so much better.

I run a pretty dated setup which Windows ran ok but everything was slow. On Ubuntu it is like having a new computer.

As far as programs go I can't say that there is a like-for-like replacement for every single windows prog but so far I have had no problems for my requirements.

Internet browsing is faster and easier and multi-tasking is effortless - on Windows more than one or two progs open at same time slowed everything down to make it annoying on my system. And I didn't want to throw away a perfectly servicable PC simply cause windows needed it.

So stick it out for a few weeks - and see how it goes. And if Linux doesn't work for you go back to Windows - no big deal!

Good luck :)
