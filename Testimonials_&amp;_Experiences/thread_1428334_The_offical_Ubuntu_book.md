While not exactly light reading, Mark G. Sobell's [A Practical Guide to Linux Commands, Editors and Shell Programming]("http://www.amazon.com/Practical-Guide-Commands-Editors-Programming/dp/0131478230") features a fantastic reference section for using the Terminal in both Mac and Linux, as well as a concise yet informative history regarding Linux. Also, I highly recommend reading the section on operating system virtualization. I found it to be an excellent tool for explaining to family and friends how they can legally install Windows in order to continue using specific programs for work/etc within Ubuntu. 

(An anecdotal case in point for the aforementioned scenario would be my mother, a mechanical engineer, who needs specific Microsoft programs for work. On a positive note, because of Mark G. Sobell's book, she is seriously considering installing Windows in Virtualbox for work and switching to Ubuntu for everyday needs :) )
