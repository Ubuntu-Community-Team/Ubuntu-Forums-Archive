### Post by Hellweek on 2011-12-20
I wish to thank the many people on this forum for the help I have received in configuring my Ubuntu server.  I found answers to all my questions in these forums and never once had to ask a single question.  This just shows the level of commitment that this community has to helping each other.

Setting up my Linux machine was an experience, I must have scratched reloaded the system 10 times before I finely was happy with the configuration.  Yes, I found some of the information more cryptic then it needs to be, and it seems to me that sometimes people give overly complex answers to simple solutions.  I was happy in some ways about this as it forced me to understand the software and the os at a greater level then would have been achieved with better documentation.

Take for example the program psad, this program was exactly what I needed for real time blocking of people attempting to access my system.  Looking over these forums and Google I found so much mis-leading and incorrect information on how to set this obscure program up correctly.  After spending 2 days on it, I decided that I would use this as a programming lesson and roll my own program to do the same thing.  Yes, I decided that it would be easier to develop a program then to try and figure out the cryptic and many time misleading information.

I am happy I did, in the process I learned AWK, BASH scripting, Perl, and was able to setup and configure a mySql server.  In the end the frustrating experience of that one program caused me to become intimate with my system.  I now understand how Services work, what a daemon is and how to write one.

I have learned more about the file system layout then I ever thought I would need to know.  I have come to fall in love with the command line again.

In all, what started out as me playing with this OS has lead me into moving my source code repository over to the machine, move my music collection over and install forked-daap.
Has caused me to give up my little NAS device and create a Software Raid array and store all my stuff on the Ubuntu server.  I am finding that I am now spending more time on the Ubuntu server running a remote X session then on what I consider my primary machine.

I suspect at some point I will transition over to Linux.  

Again, I thank everyone here for the help that was provided, even if you never once had me ask a question.  The wealth of information allowed me to be self supporting.
