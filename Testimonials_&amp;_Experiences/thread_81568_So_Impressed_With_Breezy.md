### Post by ecobuntu on 2005-10-24
I think it's all about what you prefer.  I like KDE but I like GNOME more.  KDE has amazing eye candy but my CPU can't handle it.
Overall I am impressed with Ubuntu.  I migrate here from Debian because I wanted newer applications and didn't want to run Sid.  Sid scares me.  It seems to be a pretty good distro.  
Also someone mentioned earlier that KDE felt bloated and buggy, I would have to agree at least on the bloated part.  I do recommend checking out KDE 3.5, it's pretty amazing.  
Also if you want an even lighter weight environment try XFCE-4

sudo apt-get install xubuntu-desktop

It's a pretty neat little enivornment.

### Post by Nequeo on 2005-10-24
I kinda doubt Gnome will ever move away from C. I don't know any Gnome developers, but I know many people who develop for Gnome and they all seem rather disparaging about C++, preferring to work with Java when they need to do OO stuff. That being said, you can use C++ with GTK+ if you want. But Python is probably easier for simple applications. Take a look at the tutorials at [PyGTK]("http://www.pygtk.org"). You'll need to learn Python first, but Python is easy to pick up if you already know another language.
