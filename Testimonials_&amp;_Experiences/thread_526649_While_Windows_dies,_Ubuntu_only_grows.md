### Post by cunawarit on 2007-08-16
I certainly see Ubuntu doing good things, improving with every release, becoming a better desktop OS, gaining media attention, increasing its user base, attaining near cult status amongst Linux distributions, etc...

But is Windows dying? Is Vista a failure? I really don't see either, many of the bigger corporate have not adopted Vista yet. But is that a surprise? Resolutely no, many of them are still using Windows 2000, and very old versions of Unix too. Smaller places are slowly adopting Vista, my office right now has nearly 50% of its desktops running Vista, and every new desktop is ordered with Vista Business. There was a small learning curve, some budget went on migrating, but overall it went incredibly well and Vista is working great. My main development machine is running Vista and I haven't had any serious issues. 

It hasn't been perfect, the security isn't handled with anywhere near the same elegance as Ubuntu, ASP.NET Development Server appears to run slower, and at least one person in the office doesn't like the changes much... But the benefits are good too, the interface is much improved, IIS7 is very nice, there are a couple of useful Gadgets, apps load much faster, the system appears more responsive, and lastly it just plain looks good compared to XP which had aged very badly.
