### Post by borneux on 2009-03-01
thank you all for the welcome...

A little update :
I tried updating my openoffice from version 2.4 (default in ubuntu) to 3.0 and have some problems. Openoffice would crash all the time and start the document recovery. After perusing this forum and some other website, I found the root of the problem.

It appears that openoffice has a bug when transferring data from version 2 to version 3. So I have to rename the .openoffice2 folder and delete the .openoffice one to get it to work. Openoffice will then create a new user.

Hopefully ooo 3.0.1 doesn't have the same problem and if it does, I hope it'll fixed soon.
