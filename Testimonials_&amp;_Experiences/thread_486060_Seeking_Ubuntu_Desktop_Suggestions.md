### Post by benanzo on 2007-06-27
**Media Player:**
I use both the Ubuntu defaults - Rhythmbox and Totem with all the gstreamer codecs.  VLC is a must-have as well.

**Web Authoring:**
nvu/kompozer or bluefish.  Both are excellent.

**Terminal**
The default gnome-terminal is great.

**Virtualization:**
VMWare is owning the virt market right now.  KVM, Xen and QEMU are coming along nicely, but VMWare is just awesome.

**DVD Backup/Burner:**
To rip/transcode dvds to another format like xvid or ogg I use dvd:rip.  It's probably the best ripper/transcoder in existence on any platform.  For burning I use GnomeBaker.

**Window Manager:**
The Beryl version in the Ubuntu repos is more than stable enough to run daily.  It is great.

Good Luck!
