### Post by dyltman on 2010-03-21
:popcorn:

For a while now I've been convincing my mother (who is a developer for a programming-language called agda) that ubuntu is better then a mac, we've been debating for quite a while.

However my mother could not install agda on her macbook and has gone on about it for months. One day (today actually) when we were talking my mother asked me if I couldn't attempt to install agda on ubuntu. I said sure and looked up on the internet how, all you had to do was to install the package agda-mode (sudo apt-get install agda-mode) and everything installed by itself. So now she might want me to install ubuntu on her macbook.
