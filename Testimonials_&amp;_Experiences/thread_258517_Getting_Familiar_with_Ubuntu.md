### Post by hassan_saeed on 2006-09-16
I selected Ubuntu because i did not have any money to buy any other OS. I could use pirated cds but i always felt guilt for doing thas.It was too much. I thought i CANNOT use any pirated thing. I cannot be a thief for every thing i want to do on my computer.

I heard about Ubuntu. I liked its idea, i like the co-operation of the polite people on these forums.It gave me encouragement to install it as a dual boot with ofcourse windows as my bother still uses windows on the computer.

At first it was very difficult. And it was difficult mainly becasue i did not have a modem that was supported by Linux.
So i was stuck. Any packages that i downloaded from internet, i could not install them. I simply could not install any thing.

So the only activity i could do on ubuntu was to use its office suite and calculator and clicking here and there and shutting it down.

But the things were different when i bought a modem that was supported by Linux (ubuntu) now i could goto internet directly while using ubuntu. I could download packages using package mannager plus 'add remove' tool.

I got familiar with "permissions" ; "home folders" ; "sudo" ;copying and executing in the terminal the commands shown in different web pages.

I made many mistakes even corrupted (i dont know how) my installation but intalled it again. 

Time passed and now i really feel i can use ubuntu for my own work. I can check my email using evolution. I can buy audio cds and copy them in the .ogg format. 

A great change and power was the installation of "AUTOMATIX"
it is a great tool for beginers. Although i have not decided i will or will not use automatix to enable the mp3 support for my Linux media player but i know it can enable it easily.

With "Anjuta" which i was able to install using AUTOMATIX i can now do simple computer programming as it is my Hobby

In short after i became familiar with ubuntu,it started serving me.
And i want to serve it too.

I believe as the number of users will increase linux will automatically become important. More user friendly and more reliable.


I WISH UBUNTU and OPEN SOURCE REGIME a REVOLUTIONARY SUCCESS!!
