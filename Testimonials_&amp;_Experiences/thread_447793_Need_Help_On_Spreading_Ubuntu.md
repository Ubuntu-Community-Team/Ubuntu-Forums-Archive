### Post by aysiu on 2007-05-18
I'd say plant seeds and don't push it too hard.

Pushing Ubuntu on people who don't want it leads to:
1. Resentment about being "sold" something (yes, even if it's free)
2. The perception that you are a Linux zealot
3. Expectations that Ubuntu is a drop-in replacement for Windows and a panacea for all computer problems, which ultimately leads to disappointment and a return to Windows

Start small.

Start by introducing them to open source Windows programs:
[http://www.opensourcewindows.org/](http://www.opensourcewindows.org/)

Wean them slowly off proprietary applications. Introduce Thunderbird, Firefox, OpenOffice, GIMP, Audacity, FileZilla, Notepad++, Frostwire... whatever open source Windows programs you think might appeal to them. Once they're used to using open source and start associating open source with quality and trustworthiness, then you can start talking about Ubuntu a bit more, maybe show the live CD in action.

**Do not oversell Ubuntu and/or desktop Linux**. If people think it's magic, they'll be disappointed when they don't get magic. It's better to undersell it and have people by surprised by how much it *can* do.

### Post by kiddo on 2007-05-21
might or might not work for you, just stating my personal tactic: I stopped all technical support if it is not Ubuntu. I promise unlimited free technical support to friends using ubuntu.

If it is windows, I refuse to enhance the monopoly. I refuse to install pirated software. If they must use that, they will go fetch it themselves, not with my help.

Pretty drastic I know. But if I don't do that:
[LIST]
[*]I won't feel good inside
[*]I will keep wasting time fixing their proprietary/pirated software
[*]Nothing will change; why use ubuntu when "they can get all the software for free anyway"?
[/LIST]

The same reason why I decided to boycott MSN and ICQ and use exclusively Jabber. I made [an entire website]("http://jabber.ecchi.ca") dedicated to explain why I did that (in French).

Edit: About the burned CDs: they are fine, but having official, pressed CDs ([available for free, delivered to your door]("http://shipit.ubuntu.com") from shipit) helps a long way in making ubuntu look more professional and stand out in front of those "Windows XP Corporate" CD-Rs ;)
