### Post by AusIV4 on 2006-12-02
I use KDE because I found it to be more configurable than Gnome, and I like the feel of it's programs better than those in Gnome (Amarok is awesome).

The day I switched (my roommate had been bugging me to for a while) was the day I tried to make a shortcut to lock my screen of [Win+L], as it was on my windows box. I found that gnome would not allow the windows key to be used in conjunction with other keys for shortcuts - and as far as I'm concerned that's the only point in the windows key. That was the first thing that got me moving towards KDE. I'd always found Gnome rather unintuitive, but KDE fit like a glove.

There may have been improvements made to gnome since then, but I'm quite happy with the KDE software I've become accustomed to, so I doubt I'll be going back any time soon.

As my own little testimonial, I'm setting up my (currently) Windows laptop to run Kubuntu this afternoon. There are still a few programs I can't escape, so it will be a dual boot, but I don't figure I'll boot windows so much as once a month.
