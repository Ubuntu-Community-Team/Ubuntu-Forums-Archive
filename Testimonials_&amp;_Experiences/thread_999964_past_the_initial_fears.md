### Post by Joeb454 on 2008-12-03
Congrats :) I find 2 useful features of apt are

1) Tab complete - this *always* helps you find things, like correct package names
2) ```
apt-cache search $query
``` where $query is what you want to search for :)

Try them, you'll see what I mean, though sometimes it's better to run ```
apt-cache search $query | less
```
