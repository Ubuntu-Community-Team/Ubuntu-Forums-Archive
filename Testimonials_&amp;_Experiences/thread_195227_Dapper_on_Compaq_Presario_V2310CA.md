### Post by oskie on 2006-06-12
I for one am thoroughly impressed with Dapper Drake. This is the first flavour of Ubuntu that I have been able to install on notebook. My only disappointment was having to use partition magic in order to allow for a dual boot. The graphical installer failed to properly partition my harddisk (I am a relative newbie so perhaps a more experienced Linux user could overcome this obstacle). Still, having previously used Fedora 3 to 5 and SuSE 9.0 to 10.1, I am now an Ubuntu convert. What really blew me away was the out of the box hardware detection: my media card reader works, my screen resolution was automatically set on the first boot:, my volume control buttons work and my webcam lights up (haven't tested it yet though). None of these things have ever worked with any previous Linux distribution on this notebook before. No 3D on my ATI card nor wireless connection with my Broadcom  but that was to be expected. Kudos to the Ubuntu community.:KS
