### Post by xir_ on 2007-06-06
Well after using the livecd and trying to partition and old hard drive that i had converted to usb i somehow managed to mess up vista vistas booting process.

so i thought id give ubuntu a go full time rather than reinstalling vista again (4th time).

i installed it and its been an interesting process, the things that i loved were
[LIST]
[LIST]
[*]Connecting to my wireless network instantly (something even vista cant do)
[*]firefox and open office working seamlessly
[*]clean and easy control of many features
[*]drivers for nvidia found and installed instantly
[*]Beryl visual effects rock
[/LIST]


things i didnt like
[LIST]
[*]codecs that have to be manually installed, easy to do, but an unexpected none the less (i do understand the reasoning)
[*]various bugs i've yet to fix, like my mic doesn't work. and my dvd drive doenst read data dvd's
[*]VLC is not preinstalled, or no real WMP alternative (i know this is a third party issue, but just putting it out there).
[*]Requires a lot of customisation to get things working perfect (better than vista mind you)
[*]no tutorial on use of the command line (which i actually like)
[*]No msi installation type program, not a great problem because im good with computers.
[/LIST]

Things i would like to see
[LIST]
[*]MSI type installation package
[*]A live cd compiler (i think fedora got one recently), because i dont want to have to install everything again and i have a dozen friends who want to join the movement.
[*]better tv tuner support (should be done by manufacturer, but they suck)
[*]small fixes for the mic and dvd drive
[*]preinstalled compz controll panel (seeing as they have merged with beryl)
[/LIST]


My overall impression is the i love ubuntu to bits. Its fast, my battery live has been more than doubled using  it (with 1 swappyness my hard-drive rarely ever spins up). I like how you are treated like someone with a little grey matter when it comes to doing things. The comunity is fabulous. 

For me the way i compare this to a windows installation is that to change things in windows you have to download a program to do it for you. with ubuntu just hit the command line and you can do it urself

I shall be recommending this to as many people as i can

to summerise, im a convert
