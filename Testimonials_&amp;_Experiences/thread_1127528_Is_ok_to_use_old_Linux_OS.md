### Post by snowpine on 2009-04-16
Sure, it is fine as long as your friend both:
1. Doesn't care about security (old version of Ubuntu get no security updates)
2. Needs only the applications on the install CD (since the repositories are closed) OR knows how to install applications using other methods

But why not use an up to date distro designed specifically for older hardware, like Xubuntu, Crunchbang, Puppy, DSL, SliTaz, etc? In my experience, the lighter distros actually get lighter and faster over time as they improve.
