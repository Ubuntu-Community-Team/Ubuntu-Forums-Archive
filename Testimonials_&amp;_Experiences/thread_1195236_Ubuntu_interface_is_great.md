### Post by DeusExM1 on 2009-06-23
I like the KDE style but dislike the start menu. While its nice that its at the bottom right, its way too much like windows. You have to click through what seems to be so many options just to get to a program or two. Gnome is quicker in that respect.

Despite that i tried 3 times already to use KDE, because i really do like the style they got going. (was gonna live with the start menu, and thought the default kde programs were alright). But for whatever reason, some things never worked right for me. 

Volume control was screwed up and you had to take your time to configure it each time.
Dialog boxes were always screwed up! Always. Every single time a box popped out, it was usually too short and half the text was hidden.
Also, watching online videos was a disaster for some reason. If the window was maximized, the video would get very choppy. But if it was small, then it would flow alright. 

There are more things to go on with but You get the point. All these little things make or break the system. I just could not put up with these constant annoyances so reverted back to gnome. (Although i do think Gnome could learn a few things from KDE style wise).

my .02
