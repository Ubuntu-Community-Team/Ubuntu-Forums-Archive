### Post by 3rdalbum on 2012-03-09
> **armandh said:**
> 
I miss the 3 easy pieces of Gnome
apps, places and systems
where one could find what one needed.

thus I keep going back to 10.10

I'll try again after April

The old-school Gnome 2 layout will not appear by default in any new version of Ubuntu. Gnome 2 is dead. Hoping that suddenly Ubuntu will switch back to Gnome 2 is an unrealistic hope.

As has been mentioned about a thousand times already on this forum and elsewhere on the internet, you can install Gnome Session Fallback on 11.10 to get the old layout on top of Gnome 3. This won't be around forever, but it will buy you some time. However Unity has been around for nearly a year and a half - should probably have been enough time for the transition already.

> the transition from XP to 7 was easier

7 is basically XP with slightly improved security and a bit of extra hardware support. It's like saying the transition from Ubuntu 10.04 to 10.04.1 was easier.

The jump between Windows 7 and Windows 8 is massive. Unity gives you the indicators, file manager, software selection, theme and Software Center from previous versions, with a windowing interface (you know, overlappable windows, menus etc). Windows 8 gives you the Windows Phone 7 experience on desktop computers, and the old Windows 7 interface and software runs on a different desktop to the new Windows 8 programs. The new Windows 8 programs all run full-screen, there are no overlappable windows, there's no taskbar... a real mess.

If you see Windows 8 in action you might appreciate how smooth a transition it was from Gnome 2 to Unity. Apart from the global menu, Launcher and Dash, you find that the Unity experience is basically what we've been using all along.

I'd invite you to start using Unity in 11.10 (or in 12.04). This time, forget that Gnome 2 existed. This is a new user interface and there are some nifty features that really save time and effort when you get to grips with them.
