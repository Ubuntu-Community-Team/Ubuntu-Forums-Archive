### Post by sretenter on 2009-11-05
[COLOR=Black]My experience

[/COLOR][COLOR=Black]Positive:
- my wireless internet worked flawlessly
- install time was tiny and a very nice experience (the slide shows etc)
- start-up/shut-down very quick (i have an athlon 4200)
- love the icons
- love the new software centre (being able to click install and then not having to click any dialogues)
- overall feels more polished
- language, keyboard setup etc set correctly
- like the weather indicator
- like the way when you drag a window to the side of the screen it slides into the other, very smooth
- like the new desktop wallpaper
- like how the menu for preferences and administration have been simplified
- like the new network applet (e.g. blue tooth)
- my screen resolution was set up correctly first time

[/COLOR][COLOR=Black]Negative: (in order of annoyance/show stopping)
- sound. I have an m-audio audiophile 2496 which doesn't work. Should work out of box. This is a show-stopper. I don't want to have to set it up.

- it didn't help me set up my nvidia graphics driver (i have a standard 8400). I had to manually update software sources and such. It should have told me if I want to enable the graphics driver for a better experience or at least installed its open format one. After manually clicking update and then selecting the 184 driver it set-up fine. This one wasn't show-stopper as it did correctly set my native resolution for my dell 24" monitor.

- when trying to resize a window, the space in which the pointer changes to resize is *very* small, thus having to be almost pixel perfect on that hot spot. unlike windows which the hot spot is much larger.

- I have a microsoft intellipoint explorer 2.0 (wireless) & while this did work, two of the side buttons I use frequently are not in use. A simple dialogue asking what mouse you have would suffice for this and then having you be able to set what buttons do what (or even automatic back/froward)

- when resizing windows it shoes a translucent blue boarder for where the window will resize to. I'd prefer it to resize the window in real-time like windows.

- i hate having to enter my password in each time for many things. If I am going to change some particular setting then that's my choice, no one else is on the computer, nor does anyone else access it. A simple 'remember my password for this particular task your trying to perform' box would suffice.

- no easy way to select 'always login' from login screen, thus having to go into advanced preferences to set it (I expect some users would be put off by this alone)

- where is delete? If I right click on a file I expect to see 'delete' selection, this is very odd & confusing.

- no welcome screen explaining what things do what, for example a nice flash welcome screen giving you a quick tutorial on where to find programs, how to go on the internet etc would be very good & welcoming for new users.

- dictionary dock applet leaves shadow on the desktop (using 'normal' desktop effects) after defining a word to view its description (ruins ubuntu polished experience)

- no desktop icons such as my computer/my hard disk etc. and when using neutalis file explorer, it can get confusing trying to get to 'home' where it displays hard drives etc. 


So overall I'm very impressed with ubuntu and if those negatives were rectified I would seriously consider telling many friends about it and happily installing it on their PC's, however until those negatives are addressed I can't tell others *yet*.


Thanks for all your hard work, it is vastly improved.
[/COLOR]

> **sretenter said:**
> [COLOR=Black]- no desktop icons such as my computer/my hard disk etc. and when using neutalis file explorer, it can get confusing trying to get to 'home' where it displays hard drives etc. [/COLOR]

Ubuntu likes the clean, empty look - has done so since for ever. Personally, I prefer the vanilla gnome default with computer, home, trash icons on the desktop. So - here is the highly unintuitive way of getting those icons onto the desktop. And, yes, this could be done better, but isn't. But that's not Ubuntu's fault. That's an old gnome design decision.

Here goes:

System > Preferences > Main Menu. Make sure 'Applications' in the left pane is highlighted, and tick the box next to 'System Tools' in the centre pane. And that's just to get the configuration editor in the Applications menu. :roll:

Now close the Menu editor and go to Applications > System Tools > Configuration Editor. In CE, go to Apps > Nautilus > Desktop and tick whichever icons you want visible.

Whew!

Or quicker method: Alt-F2 > "gconf-editor" in command field and click "Run". Now follow last sentence from above.
