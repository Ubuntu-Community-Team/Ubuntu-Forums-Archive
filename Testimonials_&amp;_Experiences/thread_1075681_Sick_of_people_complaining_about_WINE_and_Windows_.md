### Post by stchman on 2009-02-20
Hello all, I need to vent about this.

I grow very weary about people constantly complaining that WINE does not run their Windows apps.

People seem to judge Ubuntu solely on it's ability to run Windows apps.  WINE is such a small part of the Ubuntu experience.

If I traded in my Prius hybrid for a 1 ton dually pickup woudl I expect the same gas mileage?  NO.  Just like folks who remove Windows in favor of Linux but expect it to do be the same as Windows.  Not good judgement.

Do the Apple folks sit around and bitch that OS X does not run Windows apps flawlessly.

I look at it this way, if WINE runs your Windows app great.  If it does not run then tough.

How many native Linux apps does Windows run?  Can I install the .deb file for Geany or K3b and it will work on Windows?  NO!!!

I am sick of people thinking Ubuntu is a free Windows that does not crash or get viruses.

If you are an extreme hardcore gamer you need either XP or Vista.  I game, I have a dual boot so I can play the newer games.  I don't expect Ubuntu to run FEAR or Crysis.  I use Ubuntu for everything besides gaming.

If you are so reliant on proprietary Windows apps then stick with Windows.

Just my rant.

### Post by Vince4Amy on 2009-02-21
WINE is just awesome. I can run most of my games at exact or better performance than Windows. I really respect the project and what the team are doing, yet a lot of users really don't like them. 

People need to be thankful that anything runs on it all! For me WINE has been near enough smooth sailing from 2006+ when I started using it again.

On Windows it's not so easy to play a game in Window mode if the game doesn't have the option for that.
