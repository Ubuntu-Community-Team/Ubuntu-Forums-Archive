### Post by BlueSpecial on 2010-11-29
Hello all,

new in the forum (go easy on me ;))

been wanting to try a Linux distro for a long time and as I got a spare laptop (I can't ditch Windows (yet! :))), I thought I would try Ubuntu 10.10.

The installation put me some problems... as I'm new to Linux (all my work is done in Windows at my company :( ) I couldn't get the thing to work... all it gave me was a black screen... no logo, no install screens, nothing...

I didn't give up! :p

After net consulting and some fiddling, I could run the beast by adding acpi=off noapic nolapic to the command string... it worked like a charm.

Installed NVIDIA drivers and system updates: it was so damn easy! Also the Flash plugin for Firefox all there in the Software Center, no hassle just a click! Great!

I also learned how to make changes to GRUB command line permanently (don't laugh it's so newbie I know :D) and let me tell you that I possibly learned a lot more of how a OS works during this weekend than with all my Win experience... 

Just wanted to share with you my wonderful Ubuntu experience so far.

Regards!
