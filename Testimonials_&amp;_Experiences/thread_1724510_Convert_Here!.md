### Post by Mr Stoozer on 2011-04-08
Hello to all at Ubuntu Forums :)

Let me start by saying that i grew up only knowing & ever using Windows and drooling over OS X (to this day i still haven't tried OS X). I'd never heard of Linux as it were, especially Ubuntu until a few of years ago. After the disappointing, stale & expensive Windows 7 came out, i started looking into alternative OS'. Ubuntu was the one everyone recommended - "It works fine", "it looks great", "it's stable". So i decided to install Ubuntu 9.04 alongside Windows; just incase things got hairy.
My first impression was that it was simple and ugly. I also couldn't get some of my hardware to work so i headed to my old friend Google and embarked on a quest of reading endless documentation, entering terminal commands, constant reboots and whole OS re-installations after i inevitably broke it.
I had failed my quest, and in frustration booted into good ol' Windows and deleted the Ubu partition. (To those who are taking note, you'll know what happened upon my next reboot -- good times!)
I didn't bother with ubu again until a few weeks ago. I carried on reading a few websites and some basic Ubuntu documentation however.
I installed 10.10 and thought it was much better OS (though a little bloated for me). This in turn lead me to install a stripped down Ubu based distro, Elementary Jupiter. Looking at how that worked, i attempted to 'build' my system from the Ubu server. I now have my desktop exactly as i want it with all the functionality and looks i could ever want. i don't think i've booted into Windows for a week now and i'm loving my experiences.

There has been a really steep learning curve for me. As i said i've always used Windows and things just work differently. The biggest put-off for me initially was having to use the terminal for *everything*. Even if there was a GUI, people would give you terminal commands instead. But that's Linux, and i'm fine with that now; in-fact, after setting up the server, i now find the terminal a much quicker way of executing tasks (+ it makes you feel super-geeky!) If you'd asked me that a couple of month ago and i would've laughed in your face!

OK, so what do i love so much?
The customisability - Every single aspect of the OS can be modified. I've spent so many hours tweaking this server build i can't tell you. And it's satisfying.
The community - i've found the Ubuntu community to be an invaluable resource for information. Without it, i would have completely given up on 9.04. So thank you to all who make it so great.
Stability - I feel confident i could leave this thing on forever. I've had no major dramas in this area, in-fact, some of my "Windows _only_" apps run **BETTER** on Wine! Unbelievable i know but its true!

Finally, to Windows users who are in the same boat as i was when i first started using Ubuntu/*nix. Stick with it. It doesn't work entirely the same as Windows and there is a definite learning curve. It'll feel foreign but that's the way it is. Install in a VM first and get to know it before diving in and installing to a partition. Above all, make sure you're online!

Thanks for reading. :KS
