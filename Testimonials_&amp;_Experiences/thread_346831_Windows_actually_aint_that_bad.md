### Post by the_darkside_986 on 2007-01-31
I hate having to boot into Windows. It's such an unpleasant experience. I want to play my graphics and CPU intensive game on my fairly modern but cheap system but Windows is always there to hog all the resources and bring down the framerate. If I could play it in Linux, I would launch a simple little failsafe x-terminal in order to have minimum background junk. I cannot think of any way to do something similar in Windows :confused: Shutting down all my extra background processes does not help too much either.

What really annoys me in Windows is having to be Administrator to run certain programs. Sometimes "Run as" works sometimes it doesn't. Sometimes I want to launch an instance of explorer as Admin but I can't. I tried doing it from cmd.exe prompt with RUNAS command but that doesn't work for anything except "dir" and "chdir" (completely useless to do those as Admin). In Linux, I just do "sudo nautilus." Windows fails to present a healthly balance between functionality and user-friendliness. 

In WinXP SP2, I never run anti-virus. I don't need to because I rarely do anything risky. I don't think Elder Scrolls is going to damage my system. I used to run things as Admin all the time but I quickly learned my lesson. No trouble since then. 

Microsoft is just a spineless business like other spineless businesses in the world. But I just wanted to point out real reasons why I can't stand Windows (not M$ or moral issues). Ubuntu is awesome. I wish I could run all my games on it natively. I don't know of any open-source implementations of Elder Scrolls 3/4 or GTA for *nix.
