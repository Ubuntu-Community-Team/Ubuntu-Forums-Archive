### Post by billgoldberg on 2008-04-26
Ubuntu 8.04 must be the best ubuntu distro I've used.

All my hardware was detected out of the box (not with 100% with gutsy).

I didn't need to modify xorg.conf to be able to run compiz fusion (gutsy had the no composite error).

The installation was even faster.

I did a clean gutsy install last month and this clean hardy install runs a lot faster than gutsy ever did (might be because I've used the reiserfs file system this time).

The firefox ubuntu startpage now looks good and with a google search embedded I can now use it permanently.

And to top it off some better software choices.

I wrote this because I wanted to thank everyone who worked on the OS.
