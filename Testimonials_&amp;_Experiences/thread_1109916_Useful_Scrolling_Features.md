### Post by Miljet on 2009-03-29
I'd like to point out a couple of neat scrolling features for those who haven't discovered them already.

Almost all of the beginner's bash guides suggest piping commands that produce long output through the less command. For example:
```
cat /boot/grub/menu.list | less
```
Although this is a very useful command, it is not needed if you use the terminal emulator built into Gnome. You can use the scroll wheel on your mouse to scroll through output.

And I especially love the ability to scroll a window without shifting focus to the window. Often when browsing these forums, I will run across a bit of code that I want to try. I can open a small terminal window in the corner of the screen to test the code. If I need to read more of the forum post, just move the curser over the forum window and use the scroll wheel to scroll without clicking to shift focus.
