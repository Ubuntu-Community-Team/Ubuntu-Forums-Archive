### Post by Thingymebob on 2008-11-15
For the first time since Dapper I have clean installed (with the alt CD) rather than dist upgrade.

How simple can it get? Such a huge difference to how I remember the pain of a dapper install or any Windows install Ever. Everything I need just works (ATI Video, Atheros wifi, still no obscure Ali m502 webcam though).

I could have let my 5 year old do this without worry.

**Congrats to everyone behind Ibex a first class effort.**

My system runs better than ever with a fresh install and I may always do this in future or at least every other release, A fresh install every 6 months is a bit too regular for me.

Maintaining my Home partition was a breeze although It would have been nice if the installer would have looked in here and asked if I wanted to recreate all the users it found. As it was, I was prevented from recreating the users as the home folder already existed. So had to rename the existing Home folder, recreate user, move docs and stuff as the user, then remove original home folder as su. Should have been a lot simpler.
