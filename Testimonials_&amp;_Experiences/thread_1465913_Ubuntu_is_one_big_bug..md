The equivalent machine bought pre-built from a store would cost anywhere from 800$ - 1000$ easy.  

Anyway, my point is that open source OS's, whether that be Ubuntu, Redhat, Gentoo, Arch or any other variant, have a learning curve.  Dont give up on just because they take a while to get used to and you might have to re-learn a few things.  In the end there are SOOO many benefits to using an OS such as Ubuntu.  Just give it some time and patience.

### Post by cariboo on 2010-04-30
You really only need to use GUFW to set firewall rules, if you aren't running any servers ie: web server, email server, file server plus more, you don't need to add any rules, the default rules work quite well. To enable the firewall rules, just press Alt-F2 and type:

```
gksu ufw enable
```

Unfortunately to check the rules, you must open a terminal and type:

```
sudo iptables -L
```

To learn more about securing your Ubuntu installation have a look at bodhi.zazen's [Security sticky]("http://ubuntuforums.org/showthread.php?t=510812").
