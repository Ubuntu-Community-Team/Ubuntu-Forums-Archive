### Post by wongpk on 2009-07-12
Hi all! Thanks for the helpful forum I am very satisfy with Ubuntu now :)

I did take the chance to switch years ago, I am a graphic student, doing mostly on Print, I was with XP then to Mac end of 2006. But I did follow Ubuntu news and version from time to time. Since 8.10 announced, I can't wait to try it and now I am stuck with it.

Now I turned myself from print design to web design. Well, mostly on coding the web only. When I need design or photo retouch, GIMP exact Photoshop clone if keyboard shortcut re-config. But then I did print design as freelance from time to time, so Mac is still with me :P

More or less 98% of my time I am with Ubuntu now. Simple, straight to the point operating system. I was wishing to a Vista interface + Mac smoothness + Gnome desktop. Now I got it all in Ubuntu :)

Installation was way fast this time. First I encounter Ubuntu was 6.06 LTS, but the CD was bitten off by my dogs (sorry :P ) So I requested a 8.10 CD, everything was pre-config, even my telcom USB 3G broadband need no driver, super impressive. Beside uninstall I did not install anything else.

Definitely recommend it to all my not-gamer-not-print-designer to use :)
