### Post by thewzd on 2008-08-03
So, this is me. An IT professional for 25+yrs, using all of the operating systems that have been put in front of me during my career, including Linux-based systems, and I found myself installing Ubuntu on a PC that's sitting around doing nothing, and I've got a little bit of spare time to "see what happens". No risk, no chance to lose any data.

So I downloaded the Ubuntu ISO, made a LiveCD, and away we go!

First thoughts: Wow! So simple.
Second thoughts: Really? It's that easy?
Third thoughts: OMG. It's working!

OK, so I'm a technical software developer, web developer, web host, freelance consultant, etc... but that doesn't change the events that take place during an Operating System installation... other than I could worry more because I know how bad things can go during installation routines.

BUT... installing Ubuntu was just so easy! I've never seen such a painless O/S install in my life! Congratulations to all Ubuntu developers involved.

I think that within 20 minutes from starting with my LiveCD, I'd got internet access with Firefox and I'd got a whole load of Community (free) software installed on the PC that was ready to use. Plus of course, automatic updates for the security patches.

Most of what a general user needs to get going comes as the base install. Things like web browser, email client, document processing, games, music and video players and system tools too. But clicking the Add/Remove Software option just made life so simple! You no longer have to search all day for software, it gets listed right in front of you, and it's free!

Using the interface is so simple. It's been well thought out. You'll get used to it right away. Intuitive, I think is the word we should use here.

Some things to think about from the 'moving over' point of view is that you should check the format of your old data files, and be sure that there's a program that can access your files so that you can update them in your new Linux desktop. An example being Microsoft (MS) Office. What version of software were your files stored as. Can your replacement office software (like OpenOffice) read/write to those files. So there is a little homework on your part to check some basics.  Another thing to get used to, is the FileSystem in Linux is different. I didn't say difficult, just different. It takes just a few minutes, and you'll understand. And for those bits of software that you can not find a replacement for, there is Windows Emulator. The most commonly used is called WINE, and it lets you run (most, but not all) Windows software as though it was installed in a Windows O/S.

So, overall, I'm happy. Very happy. Happy to the point that I've installed Ubuntu on 3 of my 4 PC's and I'm preparing the complete cross over for my Windows XP Software Development laptop to Ubuntu too!

And in my chats with non-technical friends, I've already got 3 of them asking me to get rid of their installed Windows Vista replace it with Ubuntu. I think that will be happening a lot.

From my experience... Ubuntu's easy to install, easy to use, very stable (v8.04), costs nothing, and has community support from international IT experts.

So I ask myself "Why oh why did I wait so long to go to a Linux Desktop?"

I'm sure there are many more success stories, go and look for them, but this is just mine. If you're thinking about doing it... backup your data files first, and get started. You will not be disappointed.

Oh: One more thing... the LiveCD that you use to install Ubuntu... it has a 'TRY ME' option where you can load the O/S from the CD without messing up any of your existing setup.  It's a great way to 'see what you get' before you actually do the full installation. It was 'try it' option that made me realise Ubuntu was for me!
