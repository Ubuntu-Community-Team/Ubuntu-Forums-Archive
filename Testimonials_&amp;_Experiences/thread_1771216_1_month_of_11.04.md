### Post by ulukay on 2011-05-30
i've upgraded my server(which acts as my mainpc) to 11.04, had lots of problems. e.g. ubuntu uses known-incompatible libs as dependencies for programs -> [https://bugs.launchpad.net/ubuntu/+source/twisted/+bug/778322](https://bugs.launchpad.net/ubuntu/+source/twisted/+bug/778322)

i immediately switched all 3 of my users (including myself) to unity. worked fine for a week. some small glitches (extremely slow performance when many windows are open, graphic corruption after unlocking the desktop (goes away after a few seconds))
all 3 users are always logged in, only 1 user is active. we are switching with fast-user-switching. worked fine with gnome ... doesn't work reliable with unity.

- lost a locked session one time
- got 2 system freezes when using fast-user-switching. 

as i mentioned earlier, i had NO freezes or crashes for 3 years now, after i used unity i got 2 in one month.

i'll switch back to gnome2. too bad since i liked the workflow in unity very much, but i'm using linux for stability, not eyecandy-3d ********.
