### Post by LinuxFox on 2008-11-05
I've been using Ubuntu for months, and a Linux Live CD a little before it.  I first learned about Linux a little bit from the Internet.  Later on I talked to a friend, and we had a good discussion about Linux.  Some time after that, I checked out Knoppix, since it ran off the CD I gave it a shot.  After using it for a day, I fell in love with the Linux OS, and had fun doing things on it.

One day I watched a Linux Journal podcast and it was showcasing Wubi.  After learning about Wubi and Ubuntu, I downloaded Ubuntu 8.04 and installed it using Wubi (which was even on the CD).  After installing, I was welcomed to a nice clean desktop screen with no icons.  I played around with a few games, and I surfed the web, but that's about it.

Then I decided to go deeper and try different things.  I was able to make a theme that matched the color to one of my favorite wallpapers.  I even compiled a few small games, learned a little bit about using the terminal to get packages, I even learned how to run certain packages.  I learned quite a bit within the past few months.  I find myself using Ubuntu even more than Windows XP.  I still use XP for Windows games, my music player, and watching DVDs (I rather use a legal DVD player, thank you), but I find myself doing everything else on Ubuntu.

I've learned a bit, and I'm still learning more.  Which is cool, I learned about Xubuntu yesterday and I thought it was cool to check it out inside Ubuntu.  With it using a light desktop, I tried the Xubuntu Live CD on my mom's old 467Mhz Computer with 256MB, It ran smoothly.  It did slow down because of the CD drive's speed, but when it didn't it ran nicely.

Overall, I've been impressed with Ubuntu and Linux in general.  This is just my opinion, but if you disagree I can understand.  Just wanted to share my experience with Linux and Ubuntu. 8)
