### Post by Burakumin on 2008-10-09
My friend introduced me to Ubuntu. He came over and installed Kubuntu and full KDE after not knowing what to do in Gnome and I found things quite laggy compared to XP. I liked how he made KDE extremely fast but KDE died and I couldn't go to the desktop. 

     Out of sort of knowing what to do to install stuff I did 'sudo apt-get install ubuntu-desktop' as a whim to maybe re-install the desktop to get things working again. After a bunch of text and a reboot I was back in Gnome. Eh I decided to download and burn the ISO with Gnome and it worked but it was the beta by accident. Well it seems to work fine.

     I found Gnome to be quite likable compared to KDE in a lot of instances and I am now learning it. I honestly didn't know how unstable KDE was. Sure it was fast and pretty but useless to me if it is going to keep dieing. I downloaded and reinstalled regular Ubuntu beta and I find I like it a whole lot more than KDE. Kudos to those who developed such a great product. Discover-ability of features is so nice. I love how different from Windows it is. It is a serious joy to use compared to the mundane XP. I find my webcam looks like garbage compared but it is a nominal price to pay.

     I re-installed Nero 3 and Alias Autodesk and a few other applications and removed Braseo, compiz, Gnome support for Firefox, the useless game suite, and a few other stuff I could only dream of removing in XP. After reading a few walkthroughs I got it quite speedy. I even forced myself to get used to the brown theme. I find myself using it more than my Mac. Looks like the PC may make a comeback! If only Photoshop worked...

     Not all is good. I hated wine with a purple passion. I cant find a good DVD menu program, Sony movie making applications are Windows only, my Ageia Physx card acts kind of screwey, my webcam has a nasty orange tint to it, some of the help files are incomplete or non-existant so I have to rely on Google, the 64 bit version of Ubuntu doesn't work (at least for me), and I got a peanut shell lodged in my keyboard. Not sure if that counts against Ubuntu but if I had Windows I wouldn't have been so relaxed to eat peanuts and would be busy fixing issues.:lolflag:

### Post by zero244 on 2008-10-10
Look into running Windows in a virtual machine.
I installed vmware server and created a Windows XP VM.
I now run Vmware Player using the VM I created with Server.
Once you have the VM created and running you can archive and compress it and run it on any machine.
You can take a copy of the Windows VM to another computer and run it with different hardware.
I dont line Wine either.......using a VM is much easier and convenient. You can have a full copy of Windows running in a Window on top of Linux.
Good luck
