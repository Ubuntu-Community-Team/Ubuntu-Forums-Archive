### Post by dolphinsonar on 2006-08-29
I have been using a windows operating system since I first started using a computer (barring that green screen in kindergarten with the little turtle on it).  I had always felt a certain limitation using the windows platform, I would try to do certain things, and the operating system would say, "NO!  You can not do that!"  I began to wonder who was operating whom?  In addition, the viruses that windows is so famous for catching wiped out my entire photo library--so a huge chunk of my memories are now nothing but scrambled ones and zeros.

I discovered GNU & Open Source software a few years ago, and I embarked on a laborious quest to only run only Free Software on my windows box, vowing never to be subjected to the flaws of proprietary software again.

The Sisyphusian task of getting windows to accept all the programs that are diametrically opposed to its corporate interests (see the Halloween documents at <a href="http://www.catb.org/~esr/halloween/">http://www.catb.org/~esr/halloween/</a>) turned out to be more difficult that I had initially anticipated.  For obvious reasons it is quite difficult to get an all free software system to work seamlessly when windows is controlling the terms of the debate.  It is quite a lot like trying to win an argument with an angry talk show host, when they are the ones who control the volume!  So I resigned myself to stop "arm wrestling with the table" and to figure out how to do this GNU/Linux operating system thing.

I figured out what a "distro" was, realized that each one was customized for a certain type of user, found out that you could experiment with different distros using a "boot disk" and discovered that Ubuntu was the GNU/Linux distro that was made to be the most user friendly and universally adaptable, and based on Debian, another (more complicated) distro that I found attractive.  I really wanted to go with Debian at first, but figured I should play it safe, learn the "command line" (which is the way you can go behind the scenes and configure GNU/Linux systems and learn computer programmer knowledge) then perhaps switch to Debian later on.

Now I run Ubuntu.  I am liberated.

My work is important to me, I am a reporter and writer.  Using windows was limiting, somewhat like riding a bicycle on a treadmill.  I am still learning about my new operating system and I have already learned a few command line functions.  The cool thing about Ubuntu is that you can take it easy and just use it for what you need, or you can step up to the challenge and learn more complicated operations that increase your overall understanding of computer programming (without the student loans).  It is a great operating system, easy to install, and gives you the oh so liberating option of deleting windows forever.  I win!

I had one funny glitch when I installed with the screen resolution on my widescreen laptop, but I used this site, to troubleshoot it and in twenty minutes had the problem solved.  I have my email, web browsing, office programs and a few games set up, and everything is running smooth.  And damned if it isn't a handsome looking interface.

I would just like to take a minute to thank the developers, and all of the contributers in the community that helped to put this together, I think it truly is a service to humanity.  Hopefully this type of open exchange of information can aid in ridding the world of secrecy, war and corruption, and truly usher in a new epoch that embodies the African spirit of Ubuntu.
