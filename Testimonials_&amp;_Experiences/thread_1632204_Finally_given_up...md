### Post by jshepherd on 2010-11-27
I have an old (P4) PC at home which had no OS on it. Specs are not too bad; P4 2.3Ghz, 1gig ram and 120gig HDD. It actually came with XP installed but I wanted a Linux box for the kids to use ie, all they need to use without going down the purchasing of additional MS software route. Now I tried Ubuntu 10.04 but had to revert to 9.04 due to Ralink driver problems with USB wireless adapter but really wanted an up to date OS. Had a go with Ubuntu 10.10, no joy. Btw, it was only the wireless issue I was having with Ubuntu, I love Ubuntu, I have it installed on my Dell laptop and both mine and my wifes netbooks (full gnome desktop versions not netbook edition).But to be honest I tried PCLinuxOS on the desktop and it works flawlessly, everything worked right from the 1st boot, even the sound card sounds better. The only thing I don't like is the KDE desktop but my children don't seem to mind. I'm still loyal to Ubuntu because I think it's still a more polished distro and if the driver issue can be resolved in future I'm certain to use on the desktop but PCLOS does seem to be a great all rounder. Has anyone else found this? Of course I could go out and buy a new wireless adapter with a different chipset but that just defeats the object.
