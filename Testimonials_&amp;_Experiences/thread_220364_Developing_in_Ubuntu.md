### Post by ramonklown on 2006-07-21
Well I can say that as a desktop and now as a development environment ubuntu has fulfilled my needs. I don't use windows at all, I develop aplications in Java, PHP, C++, etc. And the more I use it, the more I find it usefull, I guess that's what an OS should be all about. The more programs I install the less problems I have in my life aside from other OSes. 

Great programs are available for ubuntu specially ones like streamtuner which I didn't have on a windows machine or anjuta for c++ development. libraries are no problem at all with sypnatic, resolves dependencies just like Gentoo use to do (that was my last OS). And the reason why I moved out of Gentoo is because I wanted an OS that's friendly and scalable at the same time, that's not an easy task, but with ubuntu on the frontend and dapper/debian on the backend I guess they've come to just about the right spot.

One complaint I have about Ubuntu is the horrible Eclipse that comes from apt, I think that it should be for testing only. The come around was the JBoss IDE that has a robust interface for Eclipse and does a nice job on instability.

I still need to test pgadmin on Ubuntu and I'm sure that it won't be any headaches, I hope. I've received my cd's home last night and I stuck one of those stickers on my car, too bad is small, I wanted a bigger one transparent so I could glue it on my back glass. But it's good anyways at least I'm spreading the word and making the free cd shipment worth it. I only had nautilius freeze once in my hands making it the most stable linux distro I've seen till date, at the sametime scalable.

When I went to gentoo I wanted stability, what I got was speed till a certain point. Because if I had installed on it what I got installed in ubuntu I'm sure the speed would of gone to redhat ages.

Only complaint is about java, why java is so damn slow on linux in general! Sun should be chamed about the stability and speed in linux in general, it's like run everywhere but they forgot to mention that runs slow as hell on linux for the windows executives.
