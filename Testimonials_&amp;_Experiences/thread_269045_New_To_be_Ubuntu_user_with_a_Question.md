### Post by statikeffeck on 2006-10-01
Ubuntu automatically sets up the dual boot thing beautifully. I had run Windows XP with two 250GB SATA drives for a few weeks. Disk 1 is the C: drive, and boots into windows. Disk 2 is the X: drive and an empty 20GB partition for linux.
When I decided it was time to install Ubuntu, I booted from the Live CD, split my 20GB free partition into a 19GB and a 1GB for the linux swap. It installed without a hitch, and upon boot up it now automatically allows me to boot in Win XP or Ubuntu!
