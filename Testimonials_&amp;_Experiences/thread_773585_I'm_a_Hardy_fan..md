### Post by natrixgli on 2008-04-29
Hardy has been my friend since Alpha6 and I absolutely love it. I can't believe how much Ubuntu has grown up since I first tried Hoary! 

I didn't start using Ubuntu fully until 6.06, at which time I started using it on servers. I started using it on my desktop as of Feisty. I was using Slax which I had (despite Tomas' warnings) installed on my HDD and loved it due to it's ludicrous speed. I installed Ubuntu because I decided printing would be nice.

I have experienced only 2 issues, the crash in Firefox when going from one flash video to another, which I can resolve by opening flash videos in a new tab. The other issue was that briefly I could not use my Pinnacle pcHDTV tuner, which has been resolved.


Among my favorite items:

- Speed seems much improved.
- Looks more tidy and polished.
- Deskbar is more stable - what a relief!
- Gnome 2.22 is rad. Love GVFS is way better than Gnome-VFS!
- Cool new apps like Brasero, Vinagre.
- Firefox 3 is fab, thanks for including it!

I'm guessing that the next couple of releases will be pretty fabulous too. I can't wait to see more improvements with gvfs-fuse implementation. 

FYI I also have 15 employees now using Gutsy/Hardy (started switching earlier this year) and they're all loving it too, though some of them still have some minor issues here and there. Not problems with the software, though.. (PEBKAC) Most of them are older, with minimal computer skills. 


Anyhow, I am totally satisfied with Hardy, it's the best yet!



Cheerio,


-n8

### Post by natrixgli on 2008-04-30
> **Lantesh said:**
> Hey it's always good news hearing a success story.  I'm curious though.  With 15 employees plus yourself do you have any sort of official support that you purchased through Ubuntu?  I'm just wondering how that works with small business, or perhaps Ubuntu only offers service to larger enterprise type setups.

This is actually my 2nd job where Ubuntu has played a significant desktop role. At my previous employer, we used LTSP, and accessed a few windows apps (MSIE & Quickbooks) using rdesktop with SeamlessRDP.

We are a nonprofit agency with 200+ users, by the end of this year we expect to have about 30-40% switched over. We only use community support, and I pretty much handle the whole ball of wax.

I created a support site, complete with a KB, forums, ticketing system, and we have a weekly class called "geek of the week" where users can come and learn stuff and ask questions. Most of them are older adults with limited computer experience. So the switch to Ubuntu has not been without it's challenges but most of them are actually doing quite well. (A handful have even asked about getting Ubuntu on their home computers.)

Additionally, we run a few recreation centers for older adults, and many of their public computers all run Xubuntu.

We do rely on a few Windows apps, for which we are currently using Citrix, but we're in the process of evaluating NomachineNX as a replacement since they offer per server licensing as opposed to Citrix's per user licensing. 

In addition to Ubuntu desktops, we have 5 Ubuntu servers. One for web, one running Zimbra Collaboration Suite, one Samba File Server which also runs Openfire on an Ubuntu VM, one Samba Print Server, and one that runs 3 Windows Server 2k3 virtual machines with various apps, MSSQL databases, etc. 

We are very big on FOSS, all windows desktops use Pidgin/Firefox. Other stuff we use daily is Joomla, Zimbra, Bacula, Timetrex, TrixboxCE, HylaFAX, etc. 

The beauty of FOSS is that when we have an idea, we go with it. Virtual machines make it extra easy.


Cheers,

-n8
