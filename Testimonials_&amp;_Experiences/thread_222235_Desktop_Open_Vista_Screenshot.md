### Post by catlett on 2006-07-24
I don't think your screen shot uploaded. Did you take a screen shot and try to upload it in png format?
If you did then if your site is like the site I use (photobucket) it will not upload.
To get around this problem with things that are formatted png, open the file in gimp. Select "file" to get the menu to drop down. Select "save as". A window will open that gives you options for saving. Select "browse for extensions". That will open a menu with possible file formats for saving. Select "jpeg". It will give you another prompt. Say yes.
Then go to your upload site, select the jpeg image and it will upload.
You can post the image in your post if your upload site gives you an "image" url. In photobucket there is an img line. I just copy paste that url line (with the [img] tags and all) into the text box.When you post the thread, the image will be part of your post.
Just a tip, even though you didn't ask for it. But you desktop looks nice it would be a shame that a file extension kept it from being posted properly
