### Post by Luxx on 2012-05-12
I've been watching the last few releases hoping Ubuntu would take a path that wouldn't be so drastically different from what I have come to love about it. My favourite OS of all time has been Ubuntu 10.10, Maverick Meerkat. After using Ubuntu for 5 yrs I felt it had finally come to the point where it was ready for the average user and could still satisfy the custom-tinkering a Linux user would expect. Everything was working and I have everything exactly the way I want it; the menus, themes, colours, icons, panels, graphics settings, additional programs etc. Everything I want is at my fingertips in the most strategically ergonomic setup possible for the way I use my computer. This was possible with Ubuntu and Gnome 2. 

I cannot achieve this with Gnome 3 or Unity. Gnome has been around for quite a while. Long enough for significant tweaks and shortcuts to make it into the designs of the final package. But some of the most useful designs have been dropped in Gnome 3. Granted, there are some new features that are nice, but the maturity factor isn't there. I have been spoiled rotten by Ubuntu up to version 10.10 and now I'll have to start behaving like an actual Linux user again.

My OS is no longer supported and I can no longer keep my present environment by simply upgrading and I don't want to have to do a fresh install. This is the worst time for re-installing all my programs and files and customising my environment again. I may be able to prolong the move with gnome-session-fallback. Is that still available? I am wondering if it  would be sufficient to let me wrap up some projects and search for a new OS while having an up-to-date system?

I won't be warming up to Unity and it doesn't look like Ubuntu is going in the same direction I am. I'm looking for another flavour. I guess it's about time. I wish I could take the Ubuntu community with me.

### Post by Myrddin Emrys on 2012-05-13
> **Luxx said:**
> Everything was working and I have everything exactly the way I want it; the menus, themes, colours, icons, panels, graphics settings, additional programs etc. Everything I want is at my fingertips in the most strategically ergonomic setup possible for the way I use my computer. This was possible with Ubuntu and Gnome 2.

Luckily it's still possible with 12.04 if you spend 5 minutes installing MATE!:

[http://www.howtogeek.com/110052/how-to-install-the-mate-desktop-go-back-to-gnome-2-on-ubuntu/](http://www.howtogeek.com/110052/how-to-install-the-mate-desktop-go-back-to-gnome-2-on-ubuntu/)

[http://www.webupd8.org/2012/04/mate-desktop-12-released-install-it-in.html](http://www.webupd8.org/2012/04/mate-desktop-12-released-install-it-in.html)

[http://mate-desktop.org/install/#ubuntu](http://mate-desktop.org/install/#ubuntu)

Some other excellent alternative desktops have already been mentioned in this thread (I especially like Xfce), but none come as close to Gnome 2 as MATE, because it **is** Gnome 2.

You may have some minor tweaking to do to re-recreate the 10.x experience exactly. You may have to add 'MATE' to 'OnlyShowIn' in /etc/xdg/autostart/gnome-settings-daemon.desktop (see second link above) to ensure themes are applied correctly. You can install the mateconf-editor package if you want to (e.g.) remove the default desktop icons, and revert to traditional (rather than 'overlay') scrollbars using this tweak:

[http://sites.google.com/site/pedrorainho/blog/disableoverlayscrollbaronubuntu](http://sites.google.com/site/pedrorainho/blog/disableoverlayscrollbaronubuntu)

You might also want to select the core MATE applications as the defaults for various tasks; they have been renamed from the Gnome versions to avoid conflicts with a concurrent Gnome 3 installation:

[http://mate-desktop.org/applications/](http://mate-desktop.org/applications/)

Check the font settings, too - the defaults may not be ideal, and can look better on many systems with anti-aliasing, subpixel smoothing and slight hinting enabled.
