### Post by Off Shore on 2011-08-03
Well, as a newcomer to the Forum I thought it appropriate to use this post to say hallo as well as to pass on my impressions of ubuntu so far.

My experience of Ubuntu is varied. 

First time user was version 9.10. Very good with no negative comments at all.

Second version tried was 10.04. Again a rock solid distro, easy to install and highly configurable. However, the Plymouth splash screen was not all it should have been
and gave a very poor first impression of the distro..

Third Version tried was 10.10. Not quite as good an experience. Notable for problems with Plymouth and also much less stable than other versions I have tried.

Fourth version was a change to Xubuntu 11.04 because of problems Id found with 10.10. Interesting distro but lots of small problems as well as not so small ones too.
Plymouth was a problem again and remarkably, the distro is let down badly by ongoing problems with the shutdown command. Sometimes it would shutdown and sometimes it would go to a reboot.
Now Im on my fifth trial with Ubuntu 11.04.
The GUI is a marked departure from the desktop metaphor Ive been acustomed to and I expect it will take some time to get used to.  (Especially at my advanced age!) Plymouth is still a problem and I now wonder why on earth Ubuntu persists with something that more people than I cant get working correctly. Its a blight on a good system.
I am also finding it a bit difficult to get used to my windows disappearing when I minimise them!
Hopefully I will get used to this distro before I move on to the next version of Ubuntu.

All in all I have to say I do enjoy exploring Ubuntu but there are a number of niggles which do detract from the user experience.
However, Im in it for the long haul :)
