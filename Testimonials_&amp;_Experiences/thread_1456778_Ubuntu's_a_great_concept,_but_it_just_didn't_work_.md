### Post by GodLikeCreature on 2010-04-19
I think most of us will agree that it takes some courage to come here for the first time and ask.  Beginnings are difficult, new users are not convinced they will be getting any help at all, and feel intimidated.

I personally don't like what I see here.  There are far too many negative responses for any standard user to feel like they want to try again.  Yes, the OP blamed it on Ubuntu and that was probably unfair, but he's made the point that he's not experienced, so I think he's got the right to make a wrong assumption?

To the OP: I recommend you try one of the "safe" ways to use Ubuntu.  

- In my experience, the easiest way is to plainly boot from the LiveCD.  Aside from being the best way to learn if your hardware is natively supported by Ubuntu, it is completely safe, as it makes absolutely no changes on your hard drive.  

- If you like what you see, you can move on to a "proper" installation, using Wubi.  In order to do so, simply insert the LiveCD while you are in the middle of your Windows XP session.  A simple menu will popup, and you will see an option for "installing inside Windows".  Unless you have your drive encrypted (which will make it fail), this is a safe and easy way to test Ubuntu in a dual setup.  If you get tired of it, all you need to do is uninstall it, just like you would with any other Windows application.

- Before you actually commit to a dual drive setup, or worse yet, partitioning, I recommend you install Ubuntu in a USB pendrive.  All you need is an 8GB (minimum) USB drive and your LiveCD.  Just need to boot from the live CD and install on your USB drive.  The best thing about this setup is that you can take your Ubuntu with you and boot on different PCs, VERY convenient! **WARNING: Make sure you unplug your local hard drive(s) first!**

All the options above should allow you to use/test Ubuntu with minimum change to your computer setup, if any.  I recommend you try those out and get a feel of Ubuntu before you move on to more drastic installation configurations.  If you make a little effort to try Ubuntu, you will be glad you did not long from now.  I believe us Linux users have very little, if anything, to envy from Apple users.
