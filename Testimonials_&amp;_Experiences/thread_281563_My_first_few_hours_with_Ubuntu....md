### Post by dmorri18 on 2006-10-21
First, some background on myself. I have absolutely no experience with any form of Linux although I've always wanted to. Every article I've ever read stated it was troublesome to install...hardly any hardware supported..etc etc. Years went by and I started thinking of building a file server...so I googled best os for file server and Ubuntu popped up with some others...I read a little about it and it sounded interesting so...I dowloaded the .iso file. Here is how things went...frankly I am very un-impressed...

1. File download went quickly and without a hitch! Frankly it went so smoothly I couldn't believe it actually downloaded! Had to check several times to ensure file was there!

2. Followed detailed instructions on Ubuntu website on what software to download so I can burn .iso file to CD. Instructions clear, concise and easy to follow. I don't actually believe this is all I have to do! Am very skeptical of whole process and frankly I anticipate another CD coaster to add to my collection!

3. CD burning software installed without a hitch! Now I'm positive I'm doing something wrong...I'm almost certain it will come back to haunt me!

4. Burned .iso to CD. I can only imagine what is happening to my system since the software Ubuntu told me to download was not only easy to use, but burned the CD in no time at all! This isn't going like any of my Windows installs...frankly I'm getting scared!

5. Inserted CD and restarted system...am anticipating blue screen of death along with hours of re-formatting my drive...if it even recognizes any of them!

6. Ubuntu team obviously toying with me...system started no problem and after a nice scrolling list of items it was loading...I got to a desktop...after 2 minutes of staring at desktop, I gave up on the error screen I was sure would be popping up any minute. I'm sure Ubuntu is just unable to display the complex menu of problems that even now are destroying my BIOS!

7. Decided I wasn't going to let Ubuntu lull me into a false sense of security..if it was going to trash my system...I would have it do it from an install...not from the CD!

8. Doubled clicked the "Install" icon on desktop and went to get glass of water to throw on computer for when the fire would start...

9. Drank glass of water while waiting for fire...oddly enough...Unbutu FLEW through the install proccess without so much as even a click, snort, gurgle or cough...am beyond scared now! Expecting masked individual to appear on monitor telling me I have to saw through my ankle to use this OS!

10. I am sitting here without anything to do! Still waiting patiently for masked individual to make appearance. While waiting...downloaded all updates Unbutu was kind enough to say I needed...set up Email...browsed Help files...changed some desktop settings...now really really really scared!!

11. Screen very peaceful looking...sort of like coffee and cream...icons remind me of OS X...I miss my Mac! CAUTION! Ubuntu trying to lull me into trusting the OS!!!!...everything pretty self explanatory, browser works fine...email easy to set up...Ubuntu recognizes all of my hardware, drives, thumb drive, CD and DVD drives, my printer...my network, it's all there!...Now I know I am in for it!!!!

12. Final observations...

frankly, this whole process was so easy I am genuinely surprised. I have had absolutely NO problems with anything! This install took...maybe...15 minutes from start to finish? Whereas my Windows install took what...an hour or more? I think this whole thing...Ubuntu...Linux..etc is really cool!! It's so easy to install software! Honestly...I can't say enough good things about it. 

I hope I didn't offend anyone with my tongue in cheek comments above...I really was so surprised at how everything went...I couldn't believe it would actuall work! :-k  I'm looking forward to learning all about Ubuntu and what it can do! Thanks to everyone responsible for it's developement!!!!


Have a good day,

Dave M. :-({|=
