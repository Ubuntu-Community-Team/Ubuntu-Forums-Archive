### Post by Carborundum on 2012-02-23
A few days ago I bought a new computer, which is intended as a birthday gift to my sister. Unfortunately, I wasn't able to talk her into abandoning Windows entirely, instead having to settle for a dual-boot. This being the first time I have actually installed Windows and Ubuntu at the same time (rather than replacing Windows with Ubuntu, or installing Ubuntu a while after having installed Windows), I thought it would be interesting to make a note of which operating system was easier to install and set up. Here is the outcome:

Both operating systems installed without without a hitch and both installations took approximately the same amount of time, with only a couple of minutes in favor of Ubuntu.

Once installed, one of the operating systems unfortunately did not support either the WLAN or the LAN adapter out of the box, instead requiring third party drivers for both. This operating system was not Ubuntu. ;)

Once both OSes could connect to the internet, it was time to update them. Ubuntu had ~300 updates waiting to be installed, while the corresponding figure for Windows was ~100. Updating Ubuntu took about 15 minutes and one reboot. Updating Windows has so far taken a little over 7 hours and a half-dozen or so reboots, and it's still not done (mostly because Windows Update keeps finding new things it wants to update).

Suffice it to say that this experience has served as a reminder of why I left Windows for Linux in the first place. :D
