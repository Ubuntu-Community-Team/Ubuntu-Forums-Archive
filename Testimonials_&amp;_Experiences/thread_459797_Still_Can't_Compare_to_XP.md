### Post by picpak on 2007-11-17
If it helps anybody, this is what I do to burn audio CDs:

1) First I install Graveman, either through Synaptic or with
```
sudo apt-get graveman
```

2) Then I downgrade my kernel, because the latest one won't work with my CD burner. I download the file here:

[linux-image-2.6.20-11-generic_2.6.20-11.18_i386.deb](http://mirror.linux.org.mt/mirror/ubuntu/pool/mai n/l/linux-source-2.6.20/linux-image-2.6.20-11-generic_2.6.20-11.18_i386.deb)

and double click to install it.

3) Since the Ubuntu version of sox isn't compiled with MP3 support, I download an older version of it (from Edgy):

[sox_12.17.9-1_i386.deb](http://mirrors.kernel.org/ubuntu/pool/universe/s/sox/sox_12.17.9-1_i386.deb)

and install it by running
```
sudo dpkg -i sox_12.17.9-1_i386.deb
```

Of course, then the update manager wants to update it, so I create a file called /etc/apt/preferences and place in it the following:

```
Package: sox
Pin: version 12.*
Pin-Priority: 999
```

(I realize this can be done with Synaptic, but it didn't work for me.)

4) I save the file and exit. I open up Graveman, and to make sure it worked I go to File>Preferences>External Programs>Scan again for external programs (The light for sox should turn green).

I realize this is more work than practically anybody needs to do, but it's always worked for me.
