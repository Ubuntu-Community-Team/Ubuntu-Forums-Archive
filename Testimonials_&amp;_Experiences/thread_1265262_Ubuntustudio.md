### Post by UBUminJ on 2009-09-13
When I was searching for software to cleanup background noises in digital audio, I accidentally found Ubuntustudio.
After installation and a restart, I found startup screen and lot of other things have changed.
While I personally like the design very much, I think there should be a notice before installation that such things will change. Otherwise some people who spent a lot of work to design their desktop will have some unpleasant surprise.
Nevertheless, I like it.

### Post by canoemoose on 2009-09-13
However, without wanting to appear to be trolling, you could also argue that you shouldn't install any package without knowing what it is.  All the Kubuntu packages are in the repositories, meaning that, should you install them, you'd be presented with a Kubuntu rather than Ubuntu installation on reboot. The same goes for Xubuntu.  All the Ubuntu Studio packages I found do say they contain things for Ubuntu Studio; for example, the ubuntustudio-desktop package is described as follows:
[QUOTE=Synaptic Package Manager]Ubuntu Studio is a multimedia creation flavor of Ubuntu for the
Linux audio, video, and graphic enthusiast or professional.

A collection of applications that comprises the Ubuntu Studio
Desktop System.[/QUOTE]
Do you not feel that this provides enough of a warning?  I suppose the description could be altered to something more like the kubuntu-desktop package's description:
[QUOTE=Synaptic Package Manager]This package depends on all of the packages in the Kubuntu desktop system.
Installing this package will include the default Kubuntu KDE installation.
It is safe to remove this package if some of the desktop system packages are
not desired.[/QUOTE]
