### Post by sd401k on 2009-02-11
Hello everyone, I have been a Ubuntu user since August 08 and I have to say, Good job developers! A few months ago I installed 8.04 LTS Server on two machines at work that needed a serious upgrade (Windows 2000 Server). All I can say is WOW!. It is so nice to have servers that I don't have to mess with (Windows 2000 server coming with a web browser is a bad idea LOL). As for working from the command line, it's worth it to learn. When you only have a terminal displayed, people tend to keep their hands off LOL. These machines (one has KnowledgeTree DMS, the other has Citadel and Maintenance Assistant) have been stone reliable. Not one second of downtime yet. Easy install and setup and it just flatout WORKS. As we are a very small machine shop, and with money being as hard to find as it is, we need software like this. Long Live Open Source! Oh yeah, don't be afraid to give this a go in a production environment, if I can do it, anybody can and the community is very helpful. Looking forward to Jaunty guys. Thanks again.
