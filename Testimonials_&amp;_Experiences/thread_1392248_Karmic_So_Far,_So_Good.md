### Post by JSeymour on 2010-01-27
All I can say is I'm glad I didn't know 9.10 was such a disaster 
[*] before I installed it on this not-so-new Dell PowerEdge 1600SC server, and, in addition to installing the server distro on it, added all the desktop/workstation/development stuff to it, too.  I also added a sound card, a more capable graphics adapter (operating a dual-head display in "big desktop" mode), a two-port USB adapter and swapped-out the CD-ROM drive for a DVD writer. Oddly enough, despite pushing the envelope, as I so-obviously have, it seems to be working, well... all-but-flawlessly, near as I can tell. (/me knocks on wood.)

Only problems I've had so far:
[LIST]
[*]Linux's drivers didn't like the first DVD writer I bought. It was a fairly recent Pioneer model. Swapped-out for a more mature Sanyo drive and no more problem.
[*]This Dell's BIOS didn't like the first upgrade graphics adapter I tried. Turned out to be a well-known problem with 1600SCs. Swapped it for a different, newer adapter--all's well.
[*]Even with the new graphics adapter, graphics performance is not what I would've liked to have seen. Probably a result of being limited by PCI buss bandwidth.
[*]JPilot messed with my data a bit.  Can't blame that on 9.10.
[/LIST]

That's it. Been running it since about Christmas.  Other than as noted above, the system has worked like a charm. I've been pleasantly surprised. (A good thing, too, because I have to baby-sit networks, servers, desktops, you-name-it all day for a living. Last thing I need is the same grief at home.)

So, personally, my thanks to Ubuntu's developers and all the contributors to the distro.  A job well-done, from my perspective!

Jim

[*] The "such a disaster" comment refers to the many comments and threads I've seen where people complain that 9.10 was "too cutting edge" and "not sufficiently vetted" and so-on.
