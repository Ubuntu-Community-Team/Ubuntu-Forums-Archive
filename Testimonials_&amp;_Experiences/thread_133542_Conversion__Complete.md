### Post by LordBug on 2006-02-20
After spending several weeks on testing Ubuntu on a secondary PC, I finally decided to see how good of support I had on my main PC (Athlon 64/nForce 4/nVidia 6200/SATA).  I snagged the Ubuntu LiveCD and fired it up, only to have it lock up when trying to load up Gnome.  Hmm...  Let's try Kubuntu.  Same thing.  Hmm...  IRC time!  IRC suggestions, sadly, didn't provide the solution.  However, it gave me some ideas (thanks to Combatjuan and nadaR(?) for the help).  A bit of duct-tape engineering later, I had both LiveCDs working perfectly.  The problem?  The stock NV driver weren't working on my 6200 card.  The solution?  Flipping to a terminal before X finished loading (and locking) to install nVidia drivers manually and re-starting GDM.

The LiveCD hummed along like a champ.  Everything was working, even my USB multi-card reader (this actually surprised me).  With only limited RAM space to install test programs, I did what I could and was quite pleased with the results.

With that all done, I flipped out the LiveCD for an install CD.  Bye-bye Windows.  Still had some snags along the way, and I still do unfortunately.  Right now, the system is 99% finished.  I've gotten my Firefox and Thunderbird profiles moved over (no lost links or e-mails, yay!), full media support, and various games working well.  I've only managed minimal tweaking on the desktop so far, mainly just changing theme elements and putting in a new background.  I fully expect I'll do more later :)

Attaching a shrunk pic of the desktop.  The full size should be in the Gallery.  I'll have to figure out how to get the real image attached later.

### Post by LordBug on 2006-02-27
I've beaten out the large lumps now, and things are really humming along.  I did end up losing a few non-vital e-mails to a few confused mail filters.  I've since corrected that so it won't happen again.  I had USB issues, that ultimately traced back to 2 quirky USB devices:  a USB to serial adapter for my UPS, and a PS1 joypad adapter.  I pulled the joypad adapter off, as I don't need it for a while anyway (and I likely can't make it work in Linux).  The UPS I flipped to serial only and then spent 3 days getting monitoring to work correctly (NUT is a pain).  Samba is now setup, so filesharing is working nicely.  I had, and fixed (gogo forum searching!), an issue with slow CD ripping speeds.  Various tweaking has been done for speed.
