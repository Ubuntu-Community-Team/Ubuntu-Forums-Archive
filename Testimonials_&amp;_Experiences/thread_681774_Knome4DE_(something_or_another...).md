### Post by ibuclaw on 2008-01-29
Hi, I am using the latest installment of Ubuntu (Gutsy) and have been look at the KDE project for some time, and wanting to try out the new KDE4 package on the ppa server (See Kubuntu site for details). They recommend to use the xserver-xephyr, which allows an another xserver to run from within the current xserver you are using.
so I typed what they said to type in the console, but only to either get a "white noise" box or an execution error.

So after a bit of playing I executed this code in a file:

Xephyr :11; /usr/lib/kde4/bin/startkde

Of which had a weird effect... What happened was that kde4 started on top of my gnome desktop, and blended together in transparency in what I can only describe as a "merge" of the two desktops...

I will send screenshots as soon as I upload them...
