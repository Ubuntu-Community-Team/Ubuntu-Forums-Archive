### Post by mindlessly self indulgent on 2008-11-18
Okay, so I've still been mostly on my Windows partition because I need to get stuff sorted out using that, but I've been on Ubuntu every chance that I get.

I think this OS is a good one.  I'm pretty advanced when it comes to Windows, so sometimes I get frustrated when I can't figure stuff out in Ubuntu right away, but learning is always a good thing :)

I have a Toshiba Qosmio g25-av513, and it's working flawlessly for me so far.

Also, I have to say that I love the whole free software philosophy.

Anyway, that's my experience with Ubuntu thus far.  Once I have my torrents on Windows all downloaded and all of my files there backed up, I'm sure I'll be using Ubuntu predominately. 

I was so nervous to make the jump, but now I'm glad that I did!
