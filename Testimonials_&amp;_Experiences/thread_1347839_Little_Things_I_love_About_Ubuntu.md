### Post by The_Pirate_King on 2009-12-07
I like those things too! 


And in response to your dislikes: 

1. Try entering "sudo -i".  It will make every command you enter root so you don't have to type "sudo" every time.  Keep in mind that running commands as root can be dangerous, so make sure you know what you're doing.  The terminal is a REALLY powerful tool and can potentially do just about anything, plus learning how to use the terminal is a great technical skill.  However, if you absolutely must, there are almost always ways to avoid using it. 

2. Try a program called "playonlinux", it configures WINE to support a wide variety of apps/games so you don't have to.
