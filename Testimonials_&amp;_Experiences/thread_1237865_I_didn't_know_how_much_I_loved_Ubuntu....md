### Post by jfinner1 on 2009-08-11
...until I lost it. This is my story. I know it's really long, but I felt the need to share.

I've had Ubuntu on my computer for almost a year now, and I've loved it since the start. I admit that I was a bit lost and confused at the beginning, but I asked questions, and I tried things out, and eventually I figured out what I was doing. The reason I switched to Ubuntu in the first place was that my Windows OS had done a total crash on me, and a friend suggested installing a Linux OS instead of buying a new copy of Windows. I was immediately impress when Ubuntu gave me the option to recover my data from my crash OS, so I was able to regain all the data I thought that I'd lost. I never in my life thought that I'd go back to XP.

But then I got married. My husband and I shared the computer, and he wanted to play a few games, that would only run on Windows. So I set aside a small amount of space and dual booted. I asked him to never get online with Windows, because Ubuntu's protection is so much better. And things were ok.

Then I started school, and one of my requirements was to run a Windows OS with Office 2007. So I figure that the dual boot will still work. The only problem is, now I have to get online with XP. So I install an antivirus, and I run checks obsessively. 

Class started on a Tuesday. On the following Friday, my computer crashed. No virus that I can see, and nothing that appears to be actually wrong. After a few reboots and a disc scan, we're running again. For the day. On Saturday, I had a full crash, with no ability to log in to Windows. I could still access my Windows files from Linux, and Linux is still working great. A friend suggests that I reinstall windows. So I back up all my data, do 4 hours worth of file transfers to my mother's 11yr old computer, via a 4gb USB drive, and attempt to reinstall Windows on the existing Windows partition.

But something messes up. Windows overwrites Ubuntu. At first, my thought is that I'm glad I backed up ALL my data. My next thought is that it's not a big deal, since I can just re-partition my hard drive and reinstall Ubuntu. So I go to re-partition. But I can't. gParted claims there's bad sectors on my hard drive. Never mind that there weren't there until Windows crashed. There's nothing I can do.

So I try to make do with Windows. I even do my best to make Windows look like Ubuntu, with things like Rocket dock. A few weeks go by. Still, I miss my Ubuntu. Windows still isn't running quite right, but I blame it on a "failing Hard drive". 

Then, last Thursday, Windows starts giving me more problems then normal. By Friday, Office claims that it's not installed every time I try to open it. And by Sunday, it's crashed again. I'm at my breaking point. I do my homework on my mother's old '98, trying to figure out what to do. Then, a miracle! My husband's friend just bought a new laptop because the wireless card in his old one failed. The old one is running Vista, with the full Office suite, and he's giving it away! My husband scoops it up for me.

And my poor laptop, what am I to do with it? It keeps crashing, and bad sectors must mean that the hard drive is failing. But I still need the internet to upload files for my classes. Maybe I could use it as a way to transfer files to the internet. So I drag it out, and decide that if it's going to be online a lot, I should probably put Ubuntu on. I mean, it IS safer. 

So I wipe the hard drive, and start to install Ubuntu. But first I decide to run a disc check. And it comes up with, nothing!! I'm not sure how that happened, but hey, maybe my hard drive isn't failing! Excitedly, I start and finish the Ubuntu install.

The first time I start it up, when I hear that Jungle music, I actually cry. I missed Ubuntu. For the first time in weeks I feel like I have my computer again. I spent about an hour setting up my theme, my dock, my special effects. The whole time, I couldn't help grinning like a maniac. I have my computer back. I have Ubuntu back. What more do I need?

Since I've done the reinstall, I have had no problems with my computer. No crashes, no glitches. It runs faster then it has all month, even though I probably have more stuff running at one time, since I can now keep my messenger, my music player, my bit-torrent client, and browser running without crashing the computer. I don't care what the reason is, but I am never sticking another copy of Windows in this computer. I have the other computer to do what I need to for class, and my husband can use that to play his games. But this is what my computer wants, and I'm never switching away from Ubuntu again!

### Post by Martje_001 on 2009-08-12
Hey, who doesn't cry when they hear the start-up sound of Ubuntu? C'mon guys, be honest! :P

Next time Windows 'overwrites' Ubuntu (i.e. You're not able to choose Ubuntu when starting), try the following:

1. Boot from a live-cd
2. Go to a terminal
3. Type
```
sudo grub
```
4. Type
```
find /boot/grub/stage1
```
5. The output will be something like:
> **(hd[COLOR="DarkRed"]0[/COLOR],[COLOR="DarkGreen"]4[/COLOR])**
6. Type
> **root (hd[COLOR="DarkRed"]0[/COLOR],[COLOR="DarkGreen"]4[/COLOR])**
7. Type
> **setup (hd[COLOR="DarkRed"]0[/COLOR])**
8. Type
```
quit
```
9. Reboot and select Ubuntu :)

**[COLOR="Red"]I don't know if this will work with Ubuntu 9.10, since we're moving to Grub 2![/COLOR]**
