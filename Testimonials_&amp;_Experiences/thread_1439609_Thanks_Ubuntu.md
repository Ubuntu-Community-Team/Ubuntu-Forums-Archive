### Post by Tim_J on 2010-03-26
This is just a huge thank you to Ubuntu team and comunity as well as outside program developers & helpers.

I am tying on an Atheros connected Acer Aspire 150, 160 gig hard drive 1 gig ram, which I bought used with Win 7 Beta on it.
The beta seemed way better and quicker than any previous XP or Vista that I've used, but I had no intention of paying what MS wants for a licence.

Since this netbook was not to be my main computer and I've toyed a bit with linux in the past, I was not too afraid to go on a hunt for the perfect linux distro for it.

I found a link to the ISO to install Linpus lite and tried that. I must say, I can see Linpus being a pretty good start for alot of people... but I wasn't impressed with the proprietary nature or the lack of keeping more current with <Fedora releases.

I tried Chromium, which impressed me, but only as much as one can be impressed by a cloud OS. No dis-respect to the Chromium team, but I just felt that this little contraption could do way more and to stop at a cloud OS, was a waste.

Next came knoppix chopped down for netbooks version. Excellent distro, but I wanted more everyday people apps and eye candy without installing it all myself.

Enter, Ubuntu.

I tried Ubuntu several years ago when it was not very old. I liked it pretty good then, but had a hell of a time trying to get it to work with all the proprietary network and peripheral stuff that I was trying to use on it. Furthermore, I had been trying to run it on underpowered hardware, so it seemed horribly slow.

I decided (and I'm so happy I did), to give Ubuntu another shot and try it on this netbook.
It installed so easily from USB stick (while running on battery and had lots of battery left when done).
I was able to get my wifi working with very little trouble, and it actually connects before my desktop is done loading.
My Bell mobile 3G USB turbo stick is working perfectly with almost no effort.
Webcam seems fine in the built in app, but froze up during configuration in aMSN (still not sure I didn't do something dumb... will check again)
I installed Skype, aMSN, VLC media player and a few other goodies so far and everything is working great.

I am totally impressed with how far Ubuntu has come with this UNR. It boots really quite fast for an OS with so much in it and just feels good all around.
I am actually so happy with it, that it may be time to ditch the old XP on my desktop.
Then again, I am so impressed with this little netbook, that I haven't even looked at the desktop in a while.

Kudos to all involved.
