### Post by Dullstar on 2009-10-18
So, for all these years, I am afraid of Linux, due to a poor quality distro on a Pentium II several years back.  Then, I get back into an old game called, "Lemmings."  And guess what my operating system (Windows) did?

*crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash crash -  well, you get the idea.*

Of course, it requires DOSBox to run these days, so that's good because I can still use it.  But in customization, a file extension was used that my Windows installation had hated since the day it came from the store.  **** HP!  **** Micro$oft!

So, getting sick of my complaining, my brother downloads Wubi, and installs Ubuntu on my computer.  I started looking at it, and it was good.  I still used Windows a lot.  But then, I got back from my vacation to Disney World, desperate for some Linux!  So there that was for a while.  Then back to Windows for some games...  But still waiting to be ready to boot back into Linux.  I'm psyched about the concept of distro hopping, too!  I'm ready to jump in any moment!  Plus, some of my games run BETTER in WINE than they do in Windows.  What is wrong with this picture?  NOTHING!  Linux for the win!  **** Micro$oft Windows, all hail LINUX!!!

I have gotten so accustomed to Linux that I ditched MS Paint and Paint Shop Pro in favor of the GIMP (I actually found that the bug that "bugged" me the most was fixed when I installed it for my dad!).  I feel that the choice of Firefox as a default browser is a good way to ease newbies into Linux, as well, due to the fact that many people use it, and I'm accustomed to the browser.  I was vastly impressed by the software that came with Ubuntu, and want to see if there is a way to get GNOME on Windows, for the occasions that I have to use it (ICK, Windows interface!).  

UNetBootin is a very useful utility, I find, as well.  I prefer that a lot to install CD's.  I just install distro/system utility of my choice to flash drive, and YAY!

That is my awesome Linux experience in a nutshell.  Eventually, I will report what other distros are like!
