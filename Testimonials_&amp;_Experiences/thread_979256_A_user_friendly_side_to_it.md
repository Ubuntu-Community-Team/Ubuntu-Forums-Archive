Many Linux OS that I have tried are not as user friendly as Ubuntu. Some may claim that it is not but I am only a few clicks away from searching for an application to install. The other side of the story is installing applications with WINE. However if you can find an application that is available for Ubuntu then you are almost set. Its true, Ubuntu isn't the best for gaming but it does have potential if anyone would ever be willing to invest in it.
