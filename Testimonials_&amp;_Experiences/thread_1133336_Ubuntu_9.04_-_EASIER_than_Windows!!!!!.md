### Post by chrismoore on 2009-04-22
I've always been a faithful IT worker and computer nerd and I've always had this bizarre fascination.  Microsoft makes a fine line of products, but at work I'm surrounded by Windows all the time. Except for a brief couple of years during the hayday of OS/2 in the mid 90's, I've always always had Windows at work and school.  In 1996 I tried a copy of Slackware Linux that I purchased for $5.00 on CD and after a week of trying to set up I gave up.  Every few years I'd try again with some other distribution and be unhappy - something wouldn't work, or it just didn't do it as well as Windows or Macintosh.  I am one of those that users that demands luxury and functionality, and I truly believe you can have it both ways.

After my wife got her masters degree she decided to get a shiny new (and pink) Sony Vaio laptop for herself.  She let me have her old Toshiba Satellite A105 to mess around with.  

To install Windows XP I had to jump through so many hoops: 

Install Windows. 
Reboot. 
Install video drivers. 
Reboot. 
Install sound drivers. 
Reboot. 
Install wireless drivers. 
Reboot. 
Install modem drivers. 
Reboot. 
Download/install OpenOffice, virus checker, Nero (packaged with computer), reboot as needed. 

It took hours!

To install Ubuntu: 

Download and burn
Install. 
Reboot. 

That is IT!!  Everything was preconfigured at the end of the install! Enter the wireless key and I'm online and ready to get the mail. Office suite is already loaded up, CD burning applications are already there. 

The only weakness is that it gets messed up when it suspends.  The only way I can get it to come out of it is to remove the battery and put it back in. 

Ubuntu 9.04 is a great experience.  All the cool desktop effects are predone and it looks good.  Looking good is just as important as working good!  To have this much software installed it seems like it's performing very fast compared to Windows.  

I shall enjoy feeding my desire to explore other things.  To quote Captain Picard - "Let's see what's out there." :)

It isn't the real cdrecord. It's actually a symlink to wodim.
(To see it type 
```

cdrecord

```
if it says
```

wodim: No tracks specified. Need at least one.
Usage: wodim [options] track1...trackn

Use	wodim -help
to get a list of valid options.

Use	wodim blank=help
to get a list of valid blanking options.

Use	wodim dev=b,t,l driveropts=help -checkdrive
to get a list of drive specific options.

Use	wodim dev=help
to get a list of possible SCSI transport specifiers.

```
You got wodim.
Also if typing 
```

cdrecord --version

```
shows
```

Cdrecord-yelling-line-to-tell-frontends-to-use-it-like-version 2.01.01a03-dvd 
Wodim 1.1.9
Copyright (C) 2006 Cdrkit suite contributors
Based on works from Joerg Schilling, Copyright (C) 1995-2006, J. Schilling

```
This is a sure sign that you got wodim).
