### Post by themusicalduck on 2010-11-30
I had a quick look through some of your previous posts.

It looks like you installed with Wubi and it also looks like you've been trying to install Word using Wine.

I'm not really surprised you aren't enjoying Ubuntu much. Wine can be a pain to use and Wubi is somewhat unreliable.

When I first used Ubuntu I had real headaches trying to use Wine, but I rely much less on it now and Ubuntu seems so much better. I've also had bad experiences with Wubi getting broken and so never use it if I can avoid it.

If you really can't face leaving your Windows apps behind, then I'd recommend running Ubuntu in a virtual machine for now too. Otherwise I'd really try to get used to using OpenOffice instead, and also install Ubuntu properly to its own partition. Of course, you could also install Windows in a virtual machine inside Ubuntu and run some Windows programs in there (if your PC can handle it).
