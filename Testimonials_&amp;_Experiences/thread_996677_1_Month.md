### Post by scottuss on 2008-11-29
After about 1 month of use of Intrepid now I can say that I feel no major benefits of upgrading from Hardy (clean install upgrade). There are a few minor benefits and a minor disadvantage.

The minor benefits are the new theme, tabbed browsing in Nautilus, general package updates and a few other little things you tend not to notice on a daily basis. 

The disadvantage? Solving the little bugs (all seem to be done now) and the fact that although the Kernel boots quicker, Gnome seems to be a bit slower after log in.

Other than that, I'm fairly happy! (I have to say I've had more issues with my Intrepid server install than desktop, but that's a different matter)

So, my advice to those still on Hardy? If it ain't broke, don't fix it! (yet) unless you want some minor new stuff, as you ain't gonna get anything majorly ground breaking, I don't feel.


Just my opinions here guys, what do you all think?

### Post by lotacus on 2008-11-30
I'm happy with it because the new kernel has support for my Gateway laptop's hardware, such as the wifi 8247b chipset (though signal strength is considerably lower than the Windows) and some other hardware. I'm disappointed however in the release because i've noticed that boot time, shut down is terribly slower for me than it was in previous releases, the ui is notibly slower than before, app launching speed hasn't improved, almost as if Ubuntu doesn't keep anything in memory which is a shame because it uses somewhere around 800MB out of 4GB and has never touched the swap file.

I also don't see much innovation over previous releases. GNOME still looks the same as it always has, would be nice to see some innovative changes with the GUI.

Since compiz was actually enabled by default, I thought they would have fixed the video bug that is apparent when compiz is enabled, but no one has. They could have a least had compiz shut down once it detects a video being played.

HDMI out works to some degree, but the sound doesnt it's garbled, and that's a shame. Would have also been nice to add a report feature that doesn't require you to create an account just to report bugs or feature requests, that is so annoying and a speed bump to get to the final destination.

I do like the OS though, just think it still needs polishing but oh well! What more can one expect for free right?
