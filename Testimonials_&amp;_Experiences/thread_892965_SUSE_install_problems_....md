### Post by david_lynch on 2008-08-17
You might want to ask at the opensuse forums. When you download the iso, there should be a place on the download page you can find the md5 checksums for the images. All of the ftp download sites should have a file called MD5SUMS in the same directory as the iso files.

BTW suse 11.0 has some nice features, and if you stick with kde 3.5, it's nice and stable. But I'm telling you, gnome on suse 11.0 was so buggy that it drove me to switch my work computer to ubuntu hardy, after having been on suse since 2004. :frown:
