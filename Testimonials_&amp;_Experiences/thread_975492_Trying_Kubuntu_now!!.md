I was tracking Ubuntu intrepid since it  was announced but i must say, it didn't impress me when it comes to new desktop this time. So i thought to try Kubuntu for a change although i love gnome better. Anyways so i just installed kubuntu couple hrs back and so far it works great for me...

Sound: Pass
Wireless: Pass
Graphics: Pass
Function keys: Pass
Design: Pass, yet it ll take time to get used to it ;)
