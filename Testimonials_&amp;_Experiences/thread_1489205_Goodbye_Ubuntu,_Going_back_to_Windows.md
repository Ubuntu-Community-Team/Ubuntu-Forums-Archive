### Post by calisun on 2010-05-20
I have been using Ubuntu for several years already, and with every release there are always things that no longer work that worked before. But with me upgrading to 10.04, was the last straw.

Two thing that I use every day no longer work.

1) My acecad tablet now has major issues that prevents me from using it normally. It was a really big pain just to get it to work on previous versions, but now, no luck with getting it to work normally.

2) Kivio, a program that I use every day is no longer supported and I can't use it at all.
I know that there are similar products to Kivio, but I have several hundred kivio files saved that I am no longer able to open.

I know that there will be postings here about fixes for this and fixes for that. My issue is that why things that worked before, no longer work? Why does it have to be such a hassle to to find a fix and once the fix has been identified, it is a big hassle trying to get it fixed.  Once issues have been identified and fix has been posted here, why is it not being fixed on the system level so we can receive it with System Update?

I guess I am just tired of fixing issues, tired of finding out that something does not work any more and spending hours searching internet for a fix only to find out that this particular fix does not work for me and having to search again.

I guess I came to realize that I no longer care what system I use, I just want it to work.

I came to realize that no matter how bad Windows is, I never had as many issues with it as I do with linux.
