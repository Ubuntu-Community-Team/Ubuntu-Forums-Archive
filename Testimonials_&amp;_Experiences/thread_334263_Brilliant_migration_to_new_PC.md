### Post by gozzerd on 2007-01-08
I received a new hand-me-down computer for Christmas. (I never seem to have enough money to buy a new one) I've been running a multi-boot set-up for a long time, and so it was now time to try to get my hard drives to run in a new box. One of the partitions is XP Home. The new box is a wonderful dual P3 1ghz machine, with a gig of memory.  My old one was a single p3 with 512mb. I knew Windows XP Home would give me only one chip, but I thought it would boot easily. Nope. 

I had to re-install with my original cd, which is the origiinal release, because some needed files were corrupted. Then I figured i would use the online updater to get back up to speed. I tried to run update, but its not OK with either Firefox or Opera, you have to  use Internet Explorer. I had IE7, but it  won't run in a pre-SP2 environment. So I downloaded IE6. It won't run when IE7 is installed. IE7 had no uninstall listing.  Finally I remembered an SP2 cd I made to recover from an earlier disaster. So I upgraded off of an old home-made update cd to the point where IE7 would work, then updated online. 

Finally, two and a half or three hours later (or was it four?), my Windows partition would boot. Great.

Then I held my breath and tried to boot Ubuntu.  I was one of those floating in the ozone for a week or so  after the script for upgrading from dapper to edgy didn't quite work as intended. I have about 300 xorg.conf.xxxxxxxxx  sitting in my /etc/X11 folder still as a memento. 

I just sat there breathlessly with my fingers crossed and watched it boot. Everything looked normal. I got to the desktop and checked system monitor. Both cpu's were running! splitting the load perfectly evenly! As a user, I would never have known the hardware was any different! It recognized everything and ran perfectly without a hiccup.  The boot didn't even take longer while it tried to find new drivers. There were  no "wait while I install new hardware" wizards,  or anything.  It just ran. Perfectly.  I think it was one of the most brilliant things I have ever seen. 

NICELY DONE UBUNTU TEAM!!!!!
