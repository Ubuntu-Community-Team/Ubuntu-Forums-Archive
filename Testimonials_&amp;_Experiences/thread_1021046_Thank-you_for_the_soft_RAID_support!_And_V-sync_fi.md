### Post by hotweiss on 2008-12-24
Not to long ago I built a computer with soft RAID and I was only able to take advantage of it using Windows Vista. Setting up DMRAID by myself in Ubuntu was too tough for me and highly discouraging. Now comes Ubuntu 8.10 (alternate CD) with soft RAID support that installed flawlessly without the need of any esoteric commands. The speed difference between using RAID 0 and one hard drive is quite big, so I'm very happy to be using it. I hope that soft raid support will be integrated into Ubiquity in time for Ubuntu 9.04.

V sync in Compiz finally works! Mind you to get it working I had to set my screens correct refresh level in the Compiz settings menu to 60. No more watching Youtube videos with horrible horizontal tearing.

The only minus that I find when setting-up Ubuntu is the poor ACPI support for notebooks, but that is not the Ubuntu team's fault.

Now all I need to know is where I can buy Canonical stock. 

Thanks for all the hard work and the fantastic product. Merry Christmas.
