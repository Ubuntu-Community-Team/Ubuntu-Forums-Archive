### Post by windowless on 2009-09-08
Hi all just thought I would share my ubuntu experience.Wow thanks connical for a great product.Up until two moths ago I was a distro hopper mainly using windows xp.A year ago I set up a freenas box and forgot about it.The thing has been going eversince.Well anyway
my harddrive got very noisy so I used clonzilla to clone windows to a new drive no problem.Then I got thinking why do I only use linux tools(gparted being a fav) and not desktop.So time to bite the bullet and use linux only no dual boot to fall back on.So I looked around and the ubuntu community and vast repository swayed me.Everything has worked great and very fast on my modest machine. My kids love it as do I.Anything that did not work out of the box I found solutions on the forum.So thanks everyone.As an experiment I borrowed a fast machine with windows 7rc on it to see what the kids thought.I thought they would go for the eye candy.Not so the machine stayed on for half hour before they were back on ubuntu.They found windows slow and not intuitive very interesting.They made  me dual boot ubuntu with kubuntu for a flashy desktop.Anyway thats my rant I hope if anyone is thinking of trying ubuntu give it a go I found giving up windows great an I have learnt so mach more about computers so thanks again
