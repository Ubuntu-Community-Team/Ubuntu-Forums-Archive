### Post by blastus on 2005-11-29
[QUOTE=jdong]Linux simply doesn't see the kind of day-to-day deterioration that plagues Windows, for one reason or another...[/QUOTE]

I think one of the reasons is that most Windows software jacks the registry to run itself (and other software which it installs without asking) upon boot-up. And this is *legitimate software*. For example, even if you only install the Creative Audigy drivers (and not the entire Creative Audigy suite) on Windows XP, it jacks the registry to run unnecessary crap upon boot-up like windvdpatch and updreg.

Just a simple thing as putting in a DVD movie in the drive in Windows can install scumware on your machine if you are not alert. Some DVD movies are distributed with PCFriendly's InterActual player which nags you to install itself everytime you insert the DVD into the drive. The InterActual player is really spyware. And let's not forget about Sony's DRM rootkit distributed with some Sony BMG music CDs. Even if you install Microsoft Office, it may install all kinds of garbage to run upon boot-up that you will never need like ctfmon. And we all know how easy it is for software to jack Internet Explorer.

Forget about spyware. Just think about how much Windows software is distributed with adware, promotional offers, and all kinds of stuff that gets installed when you install the software. Of course, you can usually uninstall software but we all know that they all leave stuff behind. All this is also why there is a ton of software (the overwhelming majority is rogue software BTW) that claims to clean up the Windows registry, fix registry errors, clean up files on your hard drive etc...

I call software that jacks your machine, *jackware*. All spyware and adware is jackware but not all jackware is considered malware. It's no wonder that the cumulative effect of installing so much jackware deteriorates Windows after so long.
