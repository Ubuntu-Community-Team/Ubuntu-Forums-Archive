### Post by pretty_whistle on 2012-02-04
Why I prefer it:

-I can minimize and restore a window by using its button on the panel.
-Multiple workspaces have a thumbnail type of thing which tells me I have something on another workspace.  Unity's was blank and that posed a problem-if I put work aside for a few days later, I might forget open windows are there.
-The desktop icon's names have a background, making it easy to read regardless of the color of the wallpaper I'm using.
-More room to place things-a panel on top and a dock at the bottom.

One last thing:

It's swell that a user can choose from different desktops to suit their needs, likes and dislikes.

I did get used to Unity but I "prefer" this one.  :)

### Post by Trapper on 2012-02-06
I just want to add in here that there's a valid bug in U 12.04 Alpha 2 that can make an install fail. Near the end of the install the installer downloads some files. The flashplugin and microsoft fonts apparently are among those files. Unfortunately, the server they are on suffers from overload from time to time. If the installer fails to download those files it terminates the install rather than just skipping these non-critical files and allowing them to be downloaded later. Your choice is to try again later or simply forget about doing the install.

I've been a victim of this bug several times today. I am probably just going to forget about 12.04 until it's stable.
