### Post by geirknappen on 2012-11-23
I decided to install Xubuntu 12.04.1 yesterday. After some customization: moved top panel down, changed to humanity icons, moving the "close window " button from right to left in titlebar, making panel transparent, and so on... The experience of the desktop environment is pretty much identical to the one I had with ubuntu 10.04. The amount of resources used is almost identical as well. Xubuntu 12.04.1 has ubuntu softwarecenter, and this one is in my opinion better than the one in ubuntu 10.04.

Had no problem with installing nvidia proprietary drivers. Had to reinstall one broken package after install (cable Internet connection and synaptic packagemanager). Had to install b 43 wireless manually. The only thing that is not working now (as far as I know) is that the dvd is not detected when I insert it in the dvd tray (but I had the same problem with ubuntu 10.04). Haven't found any threads that explains how to solve this. Maybe this means it is not so easy to solve.
