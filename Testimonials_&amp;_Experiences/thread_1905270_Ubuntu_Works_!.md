### Post by bayouoldguy on 2012-01-06
Successful Fix: Thanks Ubuntu for having a OS that you can fix!. I was having a problem on my old Compaq desktop running 10.04.3 LTS. The active kernel was 2.6.32-32-386, & trying to use the "update manager" I received an error message : "some of the packages could not be retrieved from the server(s), do you want to continue ignoring these packages ?". I tried to click NO & the system went to: "Install" anyway. The update manager kept going around & around back to the download menue & repeating the action. I could not exit the update manager without a hard shutdown. Upon reboot & going back to update manager encountered the same problem, several more attempts. 
I was about to post for help, but on the next reboot, I thought of a boot into "safe mode" & found & accessed the "Fix Broken Packages" option. The system went thru the fix process & about an hour later , I had an updated & working Ubuntu 10.04.3 LTS with all the latest packages,& with update manager working !.

 Just thought I would let everyone know most of this "stuff" really works !.
Thanks Ubuntu Team!. ........ "G"
