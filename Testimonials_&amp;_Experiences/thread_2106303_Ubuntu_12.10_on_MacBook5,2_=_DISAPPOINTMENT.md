### Post by es_uomikim on 2013-01-18
I used Ubuntu for some time, then moved to Mac OS X and recently was thinking of moving to Ubuntu. I installed 64bit Ubuntu 12.04 on Intel Core 2 Duo MacBook5,2 and... after few days moved back to Mac OS X. Why?

- Ubuntu used only one of the CPUs, while Intel Core 2 Duo has 2 CPU's
- iSight (build-in cam) was not working (I didn't install Mac firmware - my fault, but I'd like it just to work!)
- build-in mic was not working
- batteries were used sightly less efficient
- secure swap was not working - I had to use traditional swap
- in the end "sound volume" tray icon disappeared and it was impossible to get it back (...and that was too much for me)

For all those problems I reviewed multiple threads on several Ubuntu forums, tried multiple ways to fix it (with re-installing Ubuntu). And: I'm not a total n00b: I used Debian, Ubuntu and Gentoo in the past.

Sorry but Ubuntu for MacBook5,2 really sucks - I must say it loud!

I'd really prefer to use linux on my Mac - I don't like unconfigurableness of this OS, the tons of preinstalled junk etc., but I wasted a week for configuring Ubuntu with no effect. I don't want to waste a month or more. If Ubuntu will get fixed I'd be happy to install it, but as for now it is not worth it... Sorry.
