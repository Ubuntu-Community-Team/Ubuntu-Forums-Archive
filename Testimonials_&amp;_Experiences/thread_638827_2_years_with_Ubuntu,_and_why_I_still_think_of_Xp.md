### Post by dev3 on 2007-12-12
Hi,
I have been an on-off user of Ubuntu for the past two years, and I would like to talk of my experiences.
I have been coding for the past 10 years, and have become a Windows Expert...somehow, I stuck to the Windows side, hardly touching Linux, till a good friend gave me a set of CD's of 5.10...I used that to install Linux on my then PC...I remember I had some problems installing it, and problems with hardware, most notably my NIC not working. Because of this, I couldn't update, nor could find any solution to my problems. Due to some personal problems, I didn't have much time for the next year, to play around with Linux or Ubuntu, and I seriously started using Linux only with the beta version of 7.04

I was blown away...It installed in 30 mins...with no problems...except one. I was installing it on my laptop, and I was far too afraid while partitioning...so I used a Linux system rescue CD to partition my HD, since i knew how to use it properly...anyway..It was a breeze using it...most of my hardware was working perfectly...All the softwares I needed were already installed, so I just loved it.
(anyone having even an iota of experience installing Xp knows it will takes hours to get a system to working condition).

I managed to screw my system badly twice, but thankfully I had backups, so I didn't have much of problems..

let me say here that I work in the field of GIS, so I have to use Windows platform for work purposes, and this affair with Ubuntu is for my own good
:)

So What do I like about Ubuntu??

**1] Installation Proces**s...Very easy, very n00b friendly, amazingly short time before you have a working desktop
**2] Most Things Just work**...Every Ipod I connected, every camera, any USB device I attach, works...
**3]Recognizes fileformats** Have a pdf file? It directly opens....Have a RAW file from your camera? Ubuntu can open it...Have some ancient file without any extension? More often than not, you can open and view it
**4]No Malware** I cannot overstate it's importance...I can plug in any pen drive..and I can see the virus, and it does nothing:KS
**5]Softwares with no bloat and essential functions** I cringe whenever I have to use most of windows programs, coz they are cluttered with useless features, unlike most programs on Linux

But then Why do I sometimes crave for Xp? Here is why:
**1] Instalation** Even though synaptic is an excellent piece of work, it is sometimes useless..when you have to install some obscure software, whose tarball you download from Sourceforge..It also requires an always on high speed connection, which is not so common here in India. Personally i feel that .deb files are the best way to install stuff..but then you can't update it.
[SIZE="2"]*Why can't we make a sort of deb file..which will install the software on my comp...and then add it to the synaptic list or something??*[/SIZE]
**2] Hardware support** I know it's the manufacturers fault for not releasing the specs...but the end result, ie that particular piece of hardware not working, is a major no-no when it comes to using Ubuntu or Linux.
**3]Function over form** what I mean by this, is that we choose to use more difficult tools to achieve the same things. For example...we tend to overuse the terminal, when the same could be achieved by point and click...This tends to dissuade n00bs, and when we claim it is more efficient and faster, it isolates us even further
**4]expected Functionalities Missing** There are many functionalities that I was used to as an Xp user, but I can't do them in Ubuntu, or I find that there is a roundabout way of doing it. For ex:Suppose I want to format a pen-drive. I can do it either via the terminal, or by using gparted. In windows, I could do this, just by clicking on the format menu in 'My computer'...Now I really do think that the windows method is far better in this regard
**5]Things breaking in New versions** I have often seen that things working in one edition of Ubuntu break in the other. I have a compaq laptop, and it working almost perfectly in 7.04, but lots of hardware had problems in 7.10..for example my media buttons did not work properly. there was a problem with the  touch pad, as well as with the microphone. I don't understand, how things working in one version, break in the next version...Isn't that more of a Microsoft thing??  :)


Having said all this, I have full faith in the community, and I know linux and the FOSS movement will get better and stronger
