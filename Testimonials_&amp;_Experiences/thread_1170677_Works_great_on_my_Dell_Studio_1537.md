### Post by misterfamilyguy on 2009-05-26
Hello Ubuntu community!
I just wanted to post a testimonial about my great and fun experience with Ubuntu Juanty distro. I have been a windows user since 1995 and just this year got rid of all my desktops and got a laptop in favor of portability. I bought a Dell Studio 1537 refurbished unit with built in webcam, microphone, backlit keyboard, 802.11n wifi, and pretty much maxed out options except video (I couldn't get a unit with everything PLUS the ATI card). Initially I was using Vista Home Premium 64 bit version and all was well until I got an error when trying to install some trivial software. I researched that and found the fix but got an error while doing the fix, researched that and found a fix but got an error doing the fix, and so on about 5 times. I came to the conclusion that I would need to reinstall Vista but was pretty angry since I only had the laptop for about 2 months. I tried Ubuntu before Jaunty came out and couldn't get it working right on any of my desktops, but read that since Dell ships it with some laptops, everything should work out of the box. So, I tried it and yes everything works great. The built in webcam and microphone, backlit keyboard, the quicklaunch media buttons, even the function key on the keyboard (which didn't work great under Vista). So, for anyone else thinking about trying it on their Dell Studio laptop, I highly recommend it. Currently I use the system for everything but playing video intensive games, flash games work great. It's a fun challenge learning all the different things about Linux as well, I want to thank everyone that have worked on this and that are working on it.
