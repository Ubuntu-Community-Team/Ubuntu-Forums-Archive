### Post by shaggy999 on 2007-08-30
I've been working with Fiesty Fawn for about 3 months now and just thought I would share my experience.

First off, my previous experience with computers. I've been using computers since Windows 3.11 & DOS. Basically, I came into the world of computers towards the end of the DOS era when it was right at its height, but Win95 was looming close in the future. I spent quite a bit of time using a DOS editor to change my autoexec.bat and config.sys files so that Wing Commander III, Stonekeep, or Under a Killing Moon would run w/ CD support, audio etc. Ah... the good 'ol days of setting your IRQ. Since then I've always been using Windows all the way through XP. I've become a pretty good power user on the Windows platform and eventually had an interest in programming... mainly learning Visual Basic and Java. During the years I've been frustrated many times with the stability of the Windows platform, but lived with it because there was no other option (besides the mac... ew)

I first learned about Linux in my teen years... around 98 or so and I tried out a distribution (which may have been Caldera linux or something). At the time I was on a 56k modem and nothing important to me worked (most importantly, internet) and on top of that I learned to my horror that games (my main reason for using a computer) were not big on linux.

I stayed with Windows and eventually went off to college. At the University I was exposed to other types of systems. The CS department used Sun machines so I quickly had to get used to the command line. Mostly what I was able to do was switch directories... create files... compile java source code... the basics. But I ran into a lot of people that ran Linux as their web server and quickly realized that if you wanted a stable, reliable platform you needed to run Linux. But the "geek" factor kept me away. I didn't fully understand how linux worked, although I appreciated some of the features that I knew about... such as how you could organize partitions on your hard drive and mount them anywhere in the file system and I liked the idea of a system built with security in mind from the beginning.

At the same time, I landed a job as a technical support technician and had a lot of experience working in a networked environment with mixed systems (Windows, Mac, Unix) and once in awhile had access to the awesome server room with racks upon racks of equipment. "Linux runs on these..." I thought. Cool. I saw the movie "Revolution OS" about the rise of Linux and was already using some open source resources (openoffice, ogg vorbis encoding, etc). A co-worker kept telling me about how cool FLAC was.

So around 2004 I decided to jump into linux again and tried Fedora Core and once again I felt it wasn't ready. This time the problem was wireless support. I could easily get internet access through the ethernet port on my laptop, but I had gotten used to a wireless setup. In addition apps were hard to find, configure, and install. User Interface issues also made it difficult to access things that were basic in Windows. I went back to Windows. I tried using a Knoppix CD, but performance of that was slow and I couldn't see myself using a CD-based boot system.

Now I'm here running Ubuntu. I've been using XP up until now, but have become very frustrated with having a fresh copy of windows installed with all the latest drivers and NOTHING ELSE and the sysem is still incredibly unstable with massive propagating errors resulting in BSOD or extremely slow performance. I had some minor experience helping people out with their Vista systems and was completely turned off with MS's new-fangled OS. It wasn't an option to me.

So once again I turned to linux. I searched the web for a good distro and started hearing about this "linux for human beings" and I got very interested. Is it possible...?

YES!!!

When I popped in the ubuntu CD and booted into the live operating system and discovered that EVERYTHING in my system was recognized and supported w/ ZERO configuration (including wireless) I was psyched. I dual-booted for about 2 months until I had begun to find my way.

Since then I've been blown away by some amazing features..... that even Windows doesn't have or have only become available in Vista. Here's an overview of some of the things that have blown me away:

1) Stability & Security -- Linux is rock solid
2) Usability -- The UI in Ubuntu is light-years beyond any other linux distribution I've ever used. Everything is put together the way a windows user would expect it to be.
3) Wow - Linux not only tells me how long I have left on my batteries but also, when charging, how much time is left before my batteries are charged! w00t!
4) Wow - When copying/moving files in the graphical manager you get a much more reliable estimate of how long the operation will take.
5) Wow -- With only a little bit of configuration I had support for all multimedia formats -- something lacking for me in earlier distributions)
6) apt-get update; apt-get upgrade; apt-get dist-upgrade.... Need I say more?
7) An amazing packaging system.... I drooled at the thought of writing a shell script that auto-downloads all my preferred applications and configs them from a fresh install of ubuntu. Try doing THAT windows.
8) xgl/compiz - Eventhough this is alpha software I find it highly usable and am willing to deal with minor quirks here and there as this amazing software is developed. Who cares about Aero and needing DX10 cards?
9) wine.... Wow. This stuff actually works! For those legacy apps for which there is no alternative, this deals with that final 1%
10) I can now finally call myself a linux geek. :)

So, here I am.... and no more dual-boot. I'm all ubuntu baby! This, of course, means no games for me... but that's ok, because this is a laptop and not a serious gaming system. But I do have a few games that either run natively (doom3) or through wine (diablo). I drool at the pictures of Bioshock and Crysis, but my laptop can't run it. If I built a gaming machine I'd use windows for that.... but this is not a gaming machine.

I am very, very happy with Ubuntu and thank all the developers for their hard work to make this software available to me. I am now looking into writing applications in C/C++ using cross-platform libraries... I'd like to develop for Linux.... it needs some games! :) For me, this is the year for Desktop Linux!
