### Post by Jongleur on 2007-08-22
I've now been using Ubuntu Dapper for a couple of weeks. It's the first distro that I have felt able to stick with, having tried several others in the past.
(Suse, Mandrake, Knoppix etc.,....I'm nothing if not persistent.) Now approaching my seventies, I've been computing for nearly thirty years - XZ81, Sinclair Spectrum, various Ataris, Acorn Archimedes, and eventually, (and very reluctantly,) Windows 98, XP and Vista. The sharper-eyed of you will have noticed that none of these computers has needed any input via a command line. This has been the greatest difference between my former computing and using Linux.

But it's OK. I'm really enjoying the experience and I haven't found it too difficult to get to grips with, but I suspect that anyone who is just used to running Windows straight out of the box would have a pretty hard time when things go wrong, despite the help given on this Forum.  (What help would they get from PC World?????!)

Strangely enough, I haven't felt the urge to boot into Windows for the past few days ................

### Post by yuri_rage on 2007-08-26
I'm a convert as well!  I've been using Feisty for over a month now, and I haven't looked back at all.  Ubuntu is a great distribution.  I messed with Red Hat and Slackware in the late 90s, but the lack of inter-operability with MS apps (for work) forced me to use the "other" OS.  After all those years of entrapment, I decided to see what Linux had to offer these days.  IMPRESSED is all I have to say!  I switched over completely and quickly...no dual boot for me!

I've got compiz-fusion installed, and I love it!  Sure, it's eye candy galore, but a few of the features (Expo, Cube, switching) really make it a useful, productivity increasing WM.  Kudos to the folks over at opencompositing.org!

A couple of minor complaints:
AMD64 is a pain.  I installed the 32 bit version this week after a lot of frustration.  When "ubuntu amd64" started prefacing EVERY internet search, I decided to give up on it (for now).

The "intermittent sound" bug is quite irritating.  I think I solved it (for my machine).  I have another post to that effect in the "Comprehensive solution" thread.

Otherwise, keep up the good work, Ubuntu!
