Hiya,


  I tried Mandrake 8.0 a few years ago.. and either I wasnt ready for Linux or Linux wasnt ready for me. I got a bug up my butt to try it again and did some research. I originally decided upon Ubuntu but upon giving it some more thought I wanted to try SuSe, full 5 disk install because I thought that having everything already installed for me would help me migrate from Windows XP. I was wrong.. the SUSE install was a mess and I spent over 100 hours trying to sort it all out. I had critical errors that made it pretty much unusable to me. So time to reformat and try plan A, Ubuntu. The install went well with only one minor hiccup that I was able to fix in a few minutes. I cannot beleive how much faster and smoother everything is compaired to Windows. I cannot even stand the thought of having to boot into Windows anymore.. and I have only been a Linux user for a day. My swap file has not been touched once ( 1 gig RAM ) and I no longer have that annoying hard drive grinding noise and long wait for my desktop to load. I have a lot of hardware that is not exactly Linux freindly, ATI video card on LCD monitor and Broadcom wireless network card plus others. I was able to get these devices easily working using the Ubuntu Forums. I just wanted to thank everyone who made it possible for a nOOb like me to install and operate Linux. Many people have dedicated countless hours of their lives to refine Linux to what it is today. If the progress keeps up at the current rate then one day the evil Windows empire will self destruct.
