### Post by jhanus10 on 2010-09-15
Well first off, Thank you to the Developers of Ubuntu!

My history I started using computers 28 years ago and was introduced to slackware 3.0 as my first intro to Linux at the time things were really complicated back then especially with Xwindows setup. My knowledge of MS Windows starting at 3.0 started growing so ended up in that direction eventually became certified as a Comptia A+. During the years I had always hoped that one day Linux would be an alternative to Windows for a not so technical minded user. Tried Redhat with each new release up to time they switched to Fedora Core, tried Mandrake till they switched to Mandriva also tried others many that I have forgotten the names. 2 years ago I started hearing about the Ubuntu Project and became curious so starting with version 8.x i tried each new release using a scenario if I was someone who had no clue what I was doing can I setup this OS and get everything working as I would a Windows Desktop. Well with the release of 10.04 I can honestly say that yes you I can. All of my network at home which consists of 3 desktops and 3 laptops I have converted them all to Ubuntu 10.04. Everything I did in Windows I can fully do in Ubuntu and even games running with wine. WOW, EVE, Age Of Conan, Aion are some to list. 

I have now started to implement support to my clients that would like to try switching to Ubuntu from Windows and would hope to make it spread like wild fire! I look forward to 10.10 and future projects!

Love you guys! Keep up the awesome work and I would like to offer my help in any way I can.

John
