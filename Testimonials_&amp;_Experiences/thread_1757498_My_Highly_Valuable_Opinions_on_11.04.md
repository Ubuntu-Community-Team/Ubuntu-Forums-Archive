### Post by Quit / on 2011-05-13
Hello Everyone,

This is my first post here and I'm pretty new to Linux (been using it a sum total of one month). I've been around the block a few times OS wise, having no particular loyalty to Windows, Mac OS or and particular flavor of Linux. I'm a fairly basic user and I'd love to give you my feedback on the new update for Ubuntu. I'm sure you're riveted to the spot!

First impressions:
It's a lot less demanding than the last version. The OS seems to understand when it's downloading an app/program/exe (what do we call them on here?) rather than making me compile something, or forcing me to terrify myself with arcane terminal commands.

Software centre seems to be coming together nicely, I understand it's still a relatively new idea.

I'm a bit of a fan of universal menus, so I was pleased to see that Ubuntu 11.04 is making a start to using them. I think I managed a bit of tweaking in the last version with something called GlobalMenu. However, I'm not a fan of having the menu hidden from me. I have to travel up to the top to **see** my options. Feels odd using my finger to do something my eyes are usually quite good at. Nothing is being obscured by having a menu on display all the time, so I don't know why it hides. Maybe it's shy?. As far as I can tell you can't change this in any settings anywhere. It's making difficult for me to get to grips with what software can do on this platform as it's all pretty new to me.

Launcher is a step in the right direction, but I seem to be battling to keep my desktop clutter free. Launcher insists on staying put on the left-hand-side, which is annoying but made even more so by the fact that icons like to deposit themselves on the left-hand-side of the desktop too. So, I have Steam in my launcher and a Steam icon on my desktop next to each other, which is ugly. If I move my steam desktop into another folder (I always have one called Desk Tidy which acts as my own Home folder) then it removes the icon from the launcher. The icon from the launcher should be able to dynamically re-direct itself once it's 'host app' has been moved. 

So, a good start, although a little flakey in places and unnecessarily stubborn in others.

I like a stable OS, me, which is why I decided to try Linux over Windows (also Ubuntu is free, so, yay) so it was a little disappointing that Ubuntu restarts itself occasionally. This has been very rare (maybe three or four times a week) and I can't seem to spot a pattern so I'm sorry I can't provide any more feedback than that.

I'm impressed with the new office suite, as the previous one (Open Office?) seemed to enjoy telling me it had saved my work, while not actually saving my work.

I know we're all free children of the earth here who are doing it for ourselves and fighting The Man, but .doc and .xls are really the default extensions for documents and spreadsheets out there in the world. If I don't remember to change the file format when saving I can (and have) sent myself files that are utterly useless when I open them at work. Can the new office suite not remember what file format I selected last?

Also, and this is a personal preference, I hardly ever, ever maximise windows (not even when I'm using Windows) so having these magnetic desktop borders is really getting up my goat. If there could be a preference to have a less pro-active desktop that would be great.

Having said I don't maximise windows, I do love a bit of full screen video, and 11.04 seems to handle that much better. I don't know if I was just missing some line of code or something, but services like BBC iPlayer and 4OD would usually freeze picture under certain circumstances but 11.04 has been smooth so far. Phew!

Big love and hugs for all the free goodies so far. I'm sure writing an OS is horribly complex so well done for doing it for the love of it. I think I'll be sticking around with Ubuntu for a while (I can't afford another Mac right now) so cheers to us, I'm sure we will be great together.

Matt.
