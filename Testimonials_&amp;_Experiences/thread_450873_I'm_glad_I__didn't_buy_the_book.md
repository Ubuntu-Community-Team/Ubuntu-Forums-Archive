### Post by EAL on 2007-05-21
I'm glad I didn't buy the book because I didn't need it!  This was not my first time giving Linux a try, I think the first time I tried it was Redhat and that was before the install would automatically recognize and install drivers for the NiC card.  Some years later, I tried yet again - this time it was back when USB was new ground and it was hell getting my PC to see my digital camera.  Not to mention font problems on web browsers and all that kind of stuff.

All that is ancient history now.  I set aside an entire weeked to transision to Ubuntu.  It turned out it took longer to install it than it did to get everything working properly.  There were no real hitches, and I think I can say an Ubuntu install is easier than a Windows install.  

The few very minor things I did run into (like being boggled over why I couldn't bittorrent more than one file at a time) were solved with a simple google search.  

-Eric
