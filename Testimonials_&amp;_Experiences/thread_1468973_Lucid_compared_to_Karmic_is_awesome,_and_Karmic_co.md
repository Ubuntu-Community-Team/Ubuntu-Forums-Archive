### Post by murderslastcrow on 2010-05-01
Too many times I hear people complaining about the closed drivers they didn't understand how to easily install in System/Administration/Hardware Drivers, having to install Flash/codecs in the restricted extras package, but they forget just how convenient this all really is.

Broadcom, NVidia, ATi- they all actually PROVIDE their drivers, even if they're not in the kernel. Compare this to OSX and Windows, where it's all on CDs or on websites in a list and an install utility. It's so streamlined, it's hardly something I'd call laborious.

Every release has its remote, small issues with certain computers and certain people, but this is to be expected when your user base is running on every type of hardware available for running a desktop OS on. Seriously, Ubuntu supports a lot more hardware than any individual version of Windows.

And sometimes, those of us who've used Linux for a while don't realize just how good we have it. Ubuntu in particular has the Ubuntu Software Center, Gnome and KDE are the chunkiest DEs we have and they run well on most computers bought since XP came out, we don't worry about viruses, we don't worry about slow downs, we don't worry about someone telling us how to use our computers, we don't worry at all beyond those of us who need to use niche Windows programs, it seems.

Then we dual-boot or use a VM. Big whoop.

When you notice how much Linux CAN DO, as opposed to focusing and blowing up the few cases of things it can't do (which is easy, since there are so few things it can't do), you realize just how awesome it is what you have here.

With Lucid Lynx, we have a slick, socially integrated, easy, simple, open, extensible, portable, powerful Operating System at our fingertips, for free. I don't want to sound biased here, because OS X is great, Windows 7 is great, we shouldn't focus on what they can't do all the time either, but...

I think Ubuntu 10.04 may be the best operating system to date. And this is coming from a very skeptical, critical, and thorough person. I hope it all keeps getting better, and if history is any reminder, I'm sure I'll be thinking Lucid is nothing much compared to 12.04.

Keep up the good work everyone! :D
