### Post by omar8 on 2008-06-21
I LOVE Ubuntu! It is just so amazing. I began using it about a year ago but eventually removed it and installed Vista (I was doing Coursework and I did not want to risk anything and decided that I was probably safer going with Office on a windows PC).
I do not understand why so many people say that Ubuntu needs a new look when personally, i think it looks fantastic - although, the system fonts (under appearance i think) should be set to 8px not 10px as it makes it look a lot nicer and is still easily readable.

My one problem - a problem which is with all Linux - is that the file system is a bit difficult, for example with windows all programs are installed to a specific location like C:\Program Files\blah which makes it easy to make shortcuts etc, while in Linux i just type a "command" a hope for the best :P I suppose this would be far too difficult to correct as it seems to be a character of the OS. I think that quite a lot of people would prefer to be able to manage their programs through nautilus than having to use the "add/remove" application all the time.

My final point is that at first I did not think that Ubuntu actually did that much and that opensuse or fedora running gnome would be the same thing, but I have to say, after trying them each for a couple days I missed all the small things with Ubuntu which makes it so many peoples favorite Linux.

I am using Hardy, and it only took 20 minutes to get set up, the "restricted drivers" makes it so much easier to set the system up. Its amazing how well Ubuntu works!:KS

(one point, why does the Ubuntu forums look a lot nicer in Windows than in Ubuntu - the font in ubuntu is a serif font which just looks weird)

Welcome and glad it went great for you. I've been so glad to be using Ubuntu myself because of its ease of use. It's not perfect but the advantages far outweigh the limitations.

To answer some of your questions:
shortcuts in Ubuntu (Linux/Unix) are called symbolic links. Nautilus (the equivalent to windows explorer from windows) can do this without having to type commands. In Nautilus (to get a Nautilus window go to the desktop menu bar at the top, click on the Places menu and select the Home Folder menu item), select a file and press Ctrl-M or go to the Edit Menu of Nautilus's menu bar and select the Make Link menu item.

Additionally if you want Nautilus to function much the way windows explorer does, see this link:
[http://ubuntuforums.org/showthread.php?t=600012](http://ubuntuforums.org/showthread.php?t=600012)

As for the Ubuntu forums. I'm afraid that's a font issue with .... Firefox. Do this:
1.) sudo apt-get install msttcorefonts
This will install the M$ core fonts such as Arial. You may need to enable the multiverse for that (example here: [https://help.ubuntu.com/community/Repositories/Ubuntu](https://help.ubuntu.com/community/Repositories/Ubuntu) )
2.) Once #1 is completed, start up Firefox, go to the Edit menu of Firefox's menu bar at the top and select the preferences menu item ... you get a dialog.
3.) Choose the content tab (I think this is still there in FF3), and change the default font to Arial. Your fonts should look better.

Some other common tricks are:
How to get the trash can on your desktop:
[http://www.manast.com/2008/03/13/show-computer-trash-icon-on-desktop-in-ubuntu/](http://www.manast.com/2008/03/13/show-computer-trash-icon-on-desktop-in-ubuntu/)

How to view MS Office 2007 documents (i.e., docx files) in Ubuntu:
[http://www.ubuntugeek.com/ubuntu-tip-how-to-openview-docx-files-in-openoffice.html](http://www.ubuntugeek.com/ubuntu-tip-how-to-openview-docx-files-in-openoffice.html)

Adding Nautilus Actions:
[http://dilationtime.wordpress.com/2008/06/11/nautilus-actions-empower-the-context-menu/](http://dilationtime.wordpress.com/2008/06/11/nautilus-actions-empower-the-context-menu/)

And how to add documents for your context menu for templates (i.e., the right click context menu in nautilus for creating new documents)
[http://ubuntuforums.org/showthread.php?t=241677](http://ubuntuforums.org/showthread.php?t=241677)

Hope this helps

### Post by Jonny0204 on 2008-06-21
Ubuntu is great, as too is linux in general tbh.

As for your file system problem, I was under the impression that it is infact easier in ubuntu than windows due to the fact that you can use a command (assuming you can spell correctly, something I always do wrong which bugs me :( lol)
For example, the command in the launcher for pidgin say, is simply just 'pidgin.'

Furthermore, when you install any standard application, it is most likely to be in the /etc folder, with the preferences relevant to your user in the hidden files in, for example, /home/omar8. So IMO its easier, but just becasue i see the fact i know how to do things like that on windows is because its all i ever knew until i was 16.

As for why this website looks better on windows than on ubuntu its probably because the page uses a default font css and in windows the fonts are nicer. You can change this in ubuntu though, all you need to do is add a folder in your homefolder called '.fonts' and then drop the font files off the internet into this. They should work straight away.

*note, incase you weren't aware, that the . in front will make the folder hidden
