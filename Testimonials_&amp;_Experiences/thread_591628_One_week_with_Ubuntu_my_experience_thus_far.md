### Post by Kymus on 2007-10-25
*This article was written in the hope of encouraging newcomers to make the switch. I hope that by reading about my experience as a new user, others will become warmer to the idea of trying Ubuntu. This may be freely distributed and linked to so long as the author is given credit. Thanks ;)*

One week ago, I decided I was ready to try out Ubuntu, but in reality, my journey didn't begin there exactly. Let's backtrack some.

My first time using Linux was Mandrake version 10; waaay back in 2004. I used it for a little while (dual booting a lot, actually), but then something happened (forget what) and I had to go back to using just Windows. My second journey was going to be a year ago.. I was ready to dive into Linux again because for so long I've been sick of Windows... so sick.. of the bugs, the errors, the viruses, the not booting, the praying it would boot every time I saw the splash screen (kid you not!).  Finally I decided that it was time to make a switch.  I tried.... but shortly after I'd hit “install” on the Ubuntu live CD, it'd just freeze up. I'm not sure where I posted this problem (certainly not the Ubuntu Forums), but it was never resolved. I had a few friends look at it; one is a Linux sys admin... and even he was like “uh....”. Yeah, funky stuff.

I always had a sneaking suspicion that it was the motherboard.... I just did. Well, a year later this motherboard dies and I set out to find a new one. After making my decision, and then getting the board and installing it, I was concerned... I knew I wanted to install Ubuntu again, but would it work? It should, but... what if I had sound or internet issues that were not fixable? Well, no sound + no internet = no computer, as far as I am concerned (I hate sitting in silence, and I am bored without the internet... yeah, ADD). I popped in the Live DVD, and watched as it poked along more... then more... and finally the desktop loaded! I was pleased. Very pleased.

So now we begin my journey. A year later and I still decided to use Ubuntu. Why? Well, I had one friend that used SuSE, and another who uses Gentoo. Both suggested I use either so that way if I needed help, I could just contact one of them. I considered it for a while, but ultimately I decided to stick with Ubuntu. I made this decision based on Ubuntu's popularity (which meant I'd be seeing new articles based around it more often) and its reputation for being so user friendly.

 I had zero issues with sound, video, or internet upon installation. Before I installed everything, I used an Amazon.com gift certificate I had (rewards from my Amazon.com Visa card kick ***!) to buy myself the Ubuntu Linux Bible, figuring that I'm going to need some help because 3 years ago I was clueless when it came to understanding bash (I knew “ls” that was about it :P) and the Linux file system as well.. This book has largely collected dust since its purchase. Oh it's a good book; certainly. But, every problem or question I've had thus far has been solved from either a google search or a post on the Ubuntu forums. This says that so long as someone is willing to ask, and willing to try, they will find a swift solution to what ails them. Honestly, I've found it easier to get help for Ubuntu than for Windows!

Thanks to the package managers, installation of most items was a breeze. But then there's this talk of the Synaptic Package Manager and talk about how it was “more advanced”... ooooh scary. But I found that it's not that hard ;). I would say that within 24 hours, I was starting to be able to figure things out for myself, and when something wouldn't work, I was able to think of an alternative. I was even beginning to understand the terminal; which is not something I knew much about.

I have found that thus far every program I've found to replace the programs I would always use in Windows has largely been better than what I used to use. From my music player (amarok) to my bittorrent client (kTorrent) to something as tiny as    my gmail notifier (checkGmail); each one has dazzled me with delight as its form and function greatly surpasses its windows alternative.  This is not only a relief, but a blessing. This is one of the biggest reasons why I now use Ubuntu as my primary operating system and I foresee very, very few boots to windows in the future. In a single week, I am now poking around and doing things almost as if GNOME was the desktop environment I've used all along.

I remember I actually was all set up and running the day before the new release of version 7.10 Gutsy Gibbon. I broke the 2-weeks to 1-month rule of waiting to install a new version because I was just itching to try out Gutsy and I heard a lot of good things about it. As silly as it sounds, it kinda felt like Christmas.... in October. That brings me to another point: peace of mind. Ubuntu may not be perfect, but it's not windows. When I install updates to my system, I really am not concerned about my computer going skitzo. I never installed windows updates because windows has a history of instability (windows me, anyone? Patches to fix patches? Yeah...). 

Originally, I wanted to install the K Desktop Environment because it's visually similar to windows with a start menu. I thought about it, and I decided not to, because I'm so sick of windows that the last thing I need is something that looks similar to it. When I first saw GNOME, I was pleased. I never liked the start menu much in windows anyway, and GNOME's menus – how they are designed, with such organization – is so much better. This is a wonderful alternative for anyone who doesn't like start menus :P.

Overall, my experience has been absolutely fantastic. I am very pleased with my experience thus far and I am slowly trying to convince my friends to switch as well. Linux seems scary at first; the idea of leaving behind so many old programs that you used on a daily basis for something new. Then there's the issues of understanding how to install things, how to use the terminal, troubleshooting... It can seem overwhelming at first, but after the first 24 hours, everything is down hill from there. As time passes, I am finding less and less reasons to need to boot into windows, and I am certain that this will only increase in a month, in a year, and onward..
