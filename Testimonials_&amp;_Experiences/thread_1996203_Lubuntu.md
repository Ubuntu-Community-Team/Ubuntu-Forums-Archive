### Post by hardyfan on 2012-06-04
My experience is Lubuntu 12.04 ins't just for antique machines. OK the desktop interface is basic, but on my Acer Travelmate 4530 with 2 GIG RAM, boots in 32 seconds and so fleet of foot, I love it.

The basic Chrome browser won't upload photos to eBay, and Abiword while adequate doesn't allow password protection of a document.  Hence, I ditched chrome and loaded Firefox, I also dumped Abiword and Gnumeric, replace with Libre Writer and Libre Calc.  I've added my own wallpaper, apart from that I'm running as is.

I think for the average man on the street, this is a basic but wonderful distro.  I surf the net, download and look at photos and create basic office docs.

Well done, sometimes the simplest things in life are the best.

### Post by ugm6hr on 2012-06-04
I agree... I've just moved my Dell Mini netbook (Atom-based + 1GB RAM + 8GB SSD HD) from Linux Mint 12 (11.10 based + Gnome 3) to Ubuntu 12.04 to Lubuntu 12.04.
MInt 12 was certainly usable. Ubuntu 12.04 was notably slow by my standards. Lubuntu is so much faster than both. And faster than my recollection of Xubuntu from years gone by.
The only issues I have with Lubuntu:
1. No GUI search - catfish is on hand - but strange it's not a default.
2. No "Expose" -like (as in Mac OSX) feature to review open application windows. Unfortunately, skippy (which used to add the feature, apparently) is no longer in the repository.
3. No GUI screenshot app - although "scrot -s" works well. I'm think of adding something similar to the application launcher with a .desktop file (if that works).
4. No scripts can be run from the file browser. I might actually add thunar, just for this purpose.
Some of the things I found surprisingly available:
1. samba / ftp access from file browser.
2. It's just faster to do anything.
While my use has been on a netbook (i.e. equivalent to an "older" computer) - it does have the capacity to run any of the -ubuntu variants.
