### Post by uberdonkey5 on 2009-12-08
so.. I was really worried about installing karmic, since it had taken me a while to get hardy just how I'd liked it. I have dual boot, with a copy of vista and xp. I was going to use xp, but had problems installing on my computer.. screwed up my hard disk somehow. Anyway, I had to install vista eventually...in total I used 5 disks for drivers, office etc.

Then it came to Karmic..

and it was SO quick. Not only was the installation incredibly quick, but the the file copying, start up of programs, boot up and boot down is incredible. Its like one second to shut down. I am seriously stunned. Its like I am using a supercomputer to run a tiny program.

I seriously think the creators of Karmic have made a LEAP forwards. It is finally ready for the average user. I had forgotten to install the medibuntu reps when I went to listen to some mp3, and it found and installed the correct files for me.

Finally I can say to my fellow windows users 'try this' without having to worry that I will have to hold their hand and get around numerous hardware problems and bugs. OK.. so there are bugs, but these are minor and already it seems many of these are sorted. (I needed bibliography in open-office, which crashes when you try to use it..  this just meant installing a file that was missing).

Seriously, I was one of the ones doubting Karmic, but I'd like to thank all the developers. It IS the best thing out there (and yes I have seen windows 7). Also, they have finally made some cool themes as standard.

:D
