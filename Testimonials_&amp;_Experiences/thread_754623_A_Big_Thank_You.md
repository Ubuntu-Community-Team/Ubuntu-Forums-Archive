### Post by Igniteflow on 2008-04-14
Not sure if this is the right place to post this, but after using Ubuntu for about six months now I'm finally feeling comfortable with it and seeing what a great thing Linux is.  For any new comers or "noobs" reading this, don't be discouraged when you have problems, it all gets easier.  Not only because you will learn, but also because Linux is continually improving with each release and becoming easier to use and work with.  

I feel very lucky to have this community to come to and ask for help when I encounter problems, so as the title says, a big thank you to all of you out there for spreading the love.

If any of you are interested in the story of how GNU/Linux was formed check out this documentary [http://video.google.com/videoplay?docid=7707585592627775409](http://video.google.com/videoplay?docid=7707585592627775409) called Revolution OS.

Peace
