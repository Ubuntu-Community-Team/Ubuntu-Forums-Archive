Hi all

Altough  this forum is more based on discussion and solving various problems, I want to point this topic and my opinions somewhere else.

Its not a flame-war 

I love more GNOME then KDE from the first sight, so have chosen Ubuntu to be my linux distro on desktop.

I tried to install OpenSuse 11.2 on my test, but was unable to enable mp3 playback, install properly my ati driver, and other stuff.

The migration to ubuntu was the best move that I have made.
Why?
No problems with installation, I got right after boot prompt in opensuse 11.2 menu back screen
No problem installing my ati drivers, sound or mp3 playback
No problem with enabling connection to my wifi router via Airlive USB wl-1600 usb
No problems with my hardware at all
No problems with making my desktop as I like it to be.
Conky, Ubuntu-tweak, Cairo-dock, additional themes, gnome-do, wine, openoffice, dropbox, rhytmbox, totem --> all this coll software made my ubuntu beautif, effective and satisfied.

Also I want to use this special opportunity to wish all ubuntu users, developers, admins a nice new year 2010 and keep up the excellent work, cause good isnt enough and Im awaiting the new Ubuntu 10.04 release which will surely kick ***. :P
