### Post by taylonr on 2007-01-01
I successfully set up my desktop which is wired to my router.  I've been using it for about a week.  That install was really smooth and the only thing I couldn't get to work was my printer from my XP box, but I finally got it to work.

So I thought I'd try to put it on my notebook.  Only problem is that my CD drive is REALLY screwed on my laptop.  The typical install was too intensive to run from the CD.

I found the "alternate CD"  THANK YOU!  The text worked great.

Then I connected my USB wireless card, and to my surprise, when I entered the WEP key EVERYTHING WORKED!

I have tried FC5 and FC6 and could not get this card to work.

I have Ubuntu to thank.

No real problems in this post, just a post to all the developers to say "Keep up the good work."
