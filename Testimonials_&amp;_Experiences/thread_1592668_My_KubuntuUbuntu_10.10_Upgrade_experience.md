### Post by slooksterpsv on 2010-10-10
What a total nightmare, and I know it's not Ubuntu's fault, it's mine. Here's the whole story:
So I installed Ubuntu 10.04 x64 went to Lubuntu 10.04 x64 then to Kubuntu 10.04 x64 all without reformatting, removing directories, what not. Ok So Kubuntu I was starting to like it, things were working after installing kde-full. Then Kubuntu 10.10 came out, I'm like OMG Dist-upgrade now! This is where the issues started.
My whole machine slowed down to the pace of a 33mhz running Windows XP. I beared with it, did dishes, got in the tub, etc. So I rebooted, and had tons of problems with Kubuntu 10.10, what a nightmare, the sound wouldn't switch to the default audio so I couldn't turn it up or down, it was slow, very slow to start up. Just bad. So I thought, ok I'll burn the Ubuntu ISO and install it. No, K3B burns the ISO at like 1x and makes the Ubuntu cd slow to load (unetbootin would hang wouldn't work). So I rebooted off the cd and it took forever to load, I mean 10 min. to get to the installer. So i told it to install, it hung in the middle of it, so I waited and waited - nothing happened after 15 min.

Fine I say, so I reboot, start in the graphics mode, remove all directories, except home (that has my data in it), and start reinstalling - I don't feel like transferring 120GB back again. So it's installing currently on Configuring target system, it froze on configuring hardware last time. So now I'm waiting to see how this fairs out.

Don't switch 3 distro environments (U,to L, to K without a reformat and expect a smooth upgrade.). So far, frustrating, but my Gnome is almost home where it belongs. I tried KDE, I tried to like it, but it's not for me.


EDIT: So I gave Kubuntu one more go, I installed Ubuntu 10.10 64-bit doing the above, after installing Ubuntu 10.10 I installed kubuntu-desktop (didn't remove gnome) and kept gdm as my DM. Oh my gosh what a difference, I really like KDE, it works *cries* it works and it's perfect, Kubuntu you're perfect I love you. Gnome I still love you too, but KDE is a new addition to the family.

### Post by craig10x on 2010-10-15
Good Luck and let us know how it goes for you :)

By the way, i got all the codecs simply by doing two things:  

1) Kubuntu Restricted Extras (which gives you everything including microsoft fonts)

2) and then went to the Medibuntu site and copied and pasted the command for just the encrypted dvd codec (the only thing the kubuntu restricted extras doesn't give you) that will put the deb file for it in your home folder and then you just "click and let it install"...

You can have the installer add codecs during installation, but the package it adds is not quite as complete as Kubuntu Restricted Extras so i would go that route if you haven't yet installed it...

I think you will be amazed at how nice KPackageKit works now (and the way it looks also)...While perhaps not quite as polished as ubuntu gnome, it definitely has been given some love (as compared to the older versions)...
I like it enough not to return to mint kde....that's how impressed i am with it...
