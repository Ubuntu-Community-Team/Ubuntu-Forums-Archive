### Post by julianb on 2010-05-26
I've been using Ubuntu a lot lately. But it took getting a new PC to make me realize how easy things are when you install Ubuntu on a new machine.

You get a system that doesn't need a few non-free programs demanding attention and resources just to be secure.
 
I think Win7 is a big step forward for Microsoft, but when you put it up against Ubuntu you notice some really annoying issues with it.
