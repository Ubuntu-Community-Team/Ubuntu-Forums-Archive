### Post by ronw01 on 2008-05-03
I have Ubuntu 8.04 up and running on wireless internet and I am extremely pleased. I want to thank everyone that was involved with the development and refinement of this release. All of the problems I had with previous versions that kept me from using Ubuntu Linux and Xandros as a real operating system have been resolved. I have been doodling with Linux for about 3 years but not getting very far primarily because I could not make a wireless connection to the internet. I have finally got beyond that barrier and it is gratifying. I did connect xandros to the internet a couple of years ago using an ndiswrapper that I found on a Linux site. But then found out a few months later that if I wanted to continue using it, it would cost me $60. I am quite willing to pay for software I need but in this case it would only work with the old "B" protocol and not "G". I decided to pass as "g" was then the standard and "B" was really going backwards.
The only comment I would make on 8.04 is that it takes longer than XP to boot. However, for anyone that is looking for a replacement for XP this is it.
