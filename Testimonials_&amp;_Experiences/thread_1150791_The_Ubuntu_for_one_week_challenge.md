### Post by waspbr on 2009-05-10
I guess "challenge" is a very fortunate choice of words, using a linux install without any prior knowledge cold-turkey is going to be a challenge. 

What I often see is that some people that have been using windows for some many years, decide to simply wipe it off  and replace with ubuntu, while that is fine. I find it comparable to learning how to ride a bike without the support wheels. of course it is possible to do it but you are probably going to fall a couple of times. And this may lead to some frustration. 

when I first started, I could not afford to give up on windows complitely, there were some very specific programs that I was using that were only available in windows (like Catia or solidWorks). So I had to dual boot. WhatI found was that whenever I couldn't do something in ubuntu, I would just go to windows and get it done. As I slowly started discovering things on ubuntu, then I didn't need to boot into windows as often. Now I only use windows for gaming and for very specific programs or hardware (e.g. my scanner is still not supported), being pragmatic helps.

So good luck on your challenge, but beware that it may turn out to be a rather challenging week.

### Post by lazyart on 2009-05-11
I agree that a week would only have the user longing to go back to Windows.

When I migrated it was over the course of 4-6 weeks.  I was lucky to have a spare machine.  Not nearly as powered as my Windows box, but I just wanted to try it out.  Hadn't really thought that I would stay.  This was Dapper Drake, just a couple weeks before Edgy Eft was to be released.

I spent some time poking around, kicking the tires.  30 minutes to an hour or so, then go back to the familiar confines of XP.  Little by little that hour became two or three as I learned about Synaptic and fiddled with the terminal.

Eventually I realized I needed more power and built myself a new machine for Ubuntu.  It has since been rebuilt, and just yesterday I did a ho-hum upgrade to Jaunty.  Everything worked, outside of VMWare Server, which was a 30 second ordeal running /usr/bin/vmware-config.pl ... which is standard after most kernel and OS updates.

A dual-core Pentium with 6 GB of RAM, 250 GB RAID 1, Debian in a VM running Apache for my website, Ubuntu Server in another VM as a mail server (Zimbra).  And yes, a third VM running Windows XP.  Required for my job.


After only a week you'll do just enough to get by and will still have all your Windows habits.  It really needs to be a month.  Good luck.
