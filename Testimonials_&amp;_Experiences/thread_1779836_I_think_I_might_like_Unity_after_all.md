### Post by wolfen69 on 2011-06-11
Just got a new (used) laptop, and decided what the heck, why not put unity on it, and give it a serious try. I have unity installed in vbox on my desktop already, but havn't spent much time with it. Anyway, after tweaking it and using it for a couple of hours, It really started to grow on me and made sense.

Some people complain about the way you open apps, but it's no big deal when I can put my 6 most used apps in the dock. And if I want to open something else, I merely just start typing in the name of the app, and bingo it's there.

I found it to be really smooth and bug free, and overall find it to be easy to use. Maybe it wasn't such a bad thing for canonical to reinvent the desktop after all. And just think, it's only going to get better as time goes by. Great job devs and contributors!

### Post by marl30 on 2011-06-12
I think what really gave Unity a bad name was mainly the large amount of bugs it had when it was released. I gave it a shot but all the bugs forced me to continue using Kubuntu which was much more stable. I tried Unity again recently and and it ran nicely and free of the earlier bugs.  

Had to do a fresh install today after messing around with the latest kernel which messed up 3d acceleration for my video card so bad that no fix resolved it. I just thought, what the heck, why not just install Ubuntu instead of Kubuntu. 

I've been fooling around Unity since and enabled the panel to show all apps; added the show desktop icon to the panel, then added a quick list menu to the home folder icon. Everything is working just how I wanting it. :D Nautilus is moving ok. Unity is starting a lot faster than it did last time I tried it. I think I can actually live with it this time.
