### Post by sbilstein on 2008-02-08
I manage to break my system constantly.
I'm not used to so many options, so much free software that is so powerful.  I heard that I could use several different hacks for my wireless card to improve its performance. Rather than to sticking to what works, or following the howto's exactly I began to expirement. Eventually, I had to do a fresh install.
The other day I recompiled my kernel to get an extra feature I didn't need that was to be included in a video driver patch only a few months away. Now my x is almost unuseable. The hours I did have it working until I got feature happy were excellent.
I tried to install different free versions of java just to see what its like. Now I can't run any applets.
When I was back on Vista all I used my computer for was homework and email.
Now on linux I have trouble getting to my homework and email because I'm so busy upgrading, repairing, installing, tweaking, spinning cubes on compiz, and otherwise breaking my system.

Ubuntu isn't unstable because of design flaws, it's unstable because it made me curious about software again, and how my computer runs, and I've been pushing it to the limit ever since I switched in october. When I was on window, there was no desire to mess around, the most tweaking I did was disabling all the shitty features that were built in.
Thanks for the heartaches Ubuntu; It makes my time on the computer much more enjoyable.
