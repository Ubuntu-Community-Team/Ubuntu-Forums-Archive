### Post by cdahmedeh on 2009-05-13
Hello,

My story is a pretty long one. I was always interested in Linux, but was worried about the compatibility with Windows Software. I have tried a few Live CD's and watch some YouTube video's of Compiz and that's pretty much all I knew about it. I told once myself "the day Wine will support every software on Windows, I will install Linux"; that two years ago. I was mostly a gamer, but PC Games we're getting uninteresting, all of them FPS or Strategy games. I was sticking with my old collection of games and was starting to get bored of Windows XP. 

I decide to install Windows Vista, which came preinstalled with my laptop. I've enjoyed the new interface, but the performance problems made me return to XP very quickly. Everything seemed OK, surfed the internet, played occasional games and so on. 

This is where it all changes, one night for absolutely no reason, I've decided to try give Linux a try. I knew that a Live CD wouldn't cut it, it needs to be on my hard drive alone, so I can see what it feels like to live with it.  The next day, I take a look at the compatibility of some of my 'old' games with Wine and it all looked fine. I was quite happy to see that. I've backed up all of my files to my external hard drive, and download a copy of Ubuntu 8.10. This was in December 2008. I didn't expect to stick with Linux forever. Being quite experienced with computers in general, I wasn't worried about the command line and other problems.

First thing I've noticed was how easy the installation was. Even without any knowledge, it seemed straight forward to setup partitions manually and so on. When Linux booted up, right away I've noticed some innovations that Windows still doesn't have yet (a progress bar in the boot up process). I've already noticed that Linux was much more secure, knowing that I had to type my username. I still clearly remember, when I was told that I could install some nVidia drivers which I did (worked fine) and everything else was just working so well. For the first time in my life, I didn't have to touch any registry, play with .dlls or any other Windows time wasting things. I was impressed by the amount of packages that existed in the repository. After that, I've learned how to use the command line properly, thanks to Google and the forums. Everything has been good now. 

Things that Windows never has got right, Linux has done so well :
-Synaptic Package Manager : My favorite application in Ubuntu. Everytime I need an application, my reflex is to open it up and type in what I'm looking for and within seconds I have the application in my menu.
-GNOME. I've always imagined an ideal interface in my head way before I knew about Linux. Strangely, GNOME looks exactly like it. Everything is placed neatly and properly, applications are kept simple but flexible. Everything just looks right and clean.
-Stability : I don't to need to worry about the number of applications I'm running anymore. My 3GB of RAM that was made useful in Vista is now empty room in Ubuntu. I could be runny 20 applications and enjoy the fact that when I open system monitor, only 500 megs of it is being used. Nothing hangs, nothing crashes, it all just runs. Even when an applications crashes, it is done nicely and cleanly. The Windows turns gray, open up the terminal and kill the process. In Windows, it meant that I have to restart my computer to get rid of the application.
-Driver support : I've found that Linux has better integration of drivers in the OS (maybe the number of hardware supported is less, but quality is better than quantity and it's not Linux fault's, it's the manufacturers). My fingerprint reader works directly in the system, I don't see any ugly Windows appearing for my login or anything with the companies name and logo of the fingerprint reader. It integrated in a subtle way. 
-Speed : Everything in Linux seems to run faster, even the applications crash faster ! Boot up speed is insane even on mechanical hard drives. Firefox starts much faster than it even did in Windows XP. 
-My favourite part : Control. Linux allows to have full control of my computer. I don't need to rely on someone else. Need to get my sixaxis supported, compile my own driver ! Want to change the startup screen, takes a few seconds ! Even when something isn't in the GUI, I know I can change it in the command line. All it takes is some reasearch. 

In the end, I know that I will never ever use Windows again. The freedom is great : everything is FREE and ready to be installed at your disposal. You know that you have the top of the line software, everything is fixed and updated every day in Linux. Linux is constantly changed, new kernel, new programs and so on, but the quality control and the 6 months release cycle makes sure I can take advantage of those without breaking my computer. 

Linux has really changed the way I use my computer, but after more than 5 months with it, I needed to talk about it. Thanks for making this possible, and thanks to all the developers ! There is nothing better than the feeling of turning on your computer, seeing the Command Line scrolling knowing what's going on in your computer. Few seconds later, your computer is ready to use, free of viruses, instability and commercial ideology !
