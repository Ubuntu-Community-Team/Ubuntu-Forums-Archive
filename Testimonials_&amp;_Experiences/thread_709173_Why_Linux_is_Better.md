### Post by aysiu on 2008-02-27
I think that link is terrible. Linux evangelism is counter-productive and sets unrealistic expectations for new users. Linux is not a drop-in replacement for Windows, and it is not Windows-without-problems.

Even though desktop Linux does not currently have malware targeted at it, as soon as a critical mass of users migrates over, we will get plenty of trojans and other malware exploiting social engineering, which completely bypasses the security in place. If you can trick the user, permissions mean nothing.

And desktop Linux (unlike server Linux) is *not* stabler than Windows. In fact, I've had far more unrecoverable crashes in Ubuntu than in XP. I still prefer Ubuntu to XP, despite the occasional inexplicable freeze, but I'm not going to lie to new users and tell them you'll never have to do a forced shutdown in desktop Linux. For more details, check out [this Google search](http://www.google.com/search?hl=en&q=site%3Aubuntuforums.org+freezes&btnG=Google+Search).

The list of people who should not use desktop Linux should be at the top of the page, not the bottom, unless we want a lot more "Ubuntu sucks. I'm going back to Windows!" threads in the Ubuntu Testimonials and Experiences subforum.
