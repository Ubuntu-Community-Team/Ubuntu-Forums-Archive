### Post by Meelad on 2011-10-17
I have been using Ubuntu since 9.04.. This for the me (so far) the most impressive one of the bunch, although it looks A LOT like MacOS (not that I'm complaining)..

After 3 days of using Ubuntu 11.10, this is my first take on it..

**1- Design:** Unity is beautiful! I really like it. People always like customizability, BUT if the Mac users buy their uncustomizable launcher and top bar for $1000+ then I have no reason to complain about getting mine for free.. And did I already mention that it looks beautiful?? Here is a snapshot with a few changes I made (shown in the pic)..



Nautilus looks much sexier now.. I like the space I get when I maximize a window.. The trick that first appeared on Windows 7 to automatically resize your window when dragging it to the edge of the screen is really practical.. I like the way the launcher gets out of the way when I need the space (let's show Apple how it's done for once!).. I still have my wobbly windows, (haven't tried to get the cube thing working yet), and the app menu (even though needs time to get used to) look pretty beautiful (and Mac-ish!).

There are still things that can be improved and worked on, but I really like what I'm seeing in this department so far..

**Score: 7.5/10**

**2- Performance:** I'm running Ubuntu 11.10 on a Q8400 Quadcore 2.66 Ghz processor with 4GB RAM (only 3.2 GB is recognized by Ubuntu), so I am not quite the one who can give a good review in this department, but on my computer Ubuntu 11.10's performance is great! The Software Center still takes 4-5 seconds to open though..

I noticed one more thing here but I'm not sure if I'm right about it.. It just seems to me that Ubuntu 11.10 divides the load much more equally among the processor cores than Ubuntu 10.10. I check the system monitor from time to time, and before, I noticed that two cores for example are used to 75% while the other two are at 3 or 5%.. Now whenever I open the system monitor I see all cores at pretty much same usage percentage (20% or 60% for example).. I'm not sure if there IS actually a difference between Ubuntu 11.10 and the previous versions in CPU usage but if there is, I do think that the new way is better in handling CPU load..

**Score: 8/10**

**3- Stability:** It crashed once when I played with Compiz, but I got it back running in 10 minutes and have been more careful when playing with Compiz since then. Didn't crash again and so far it looks rock solid for me. Definitely more stable than the 10.10 I had before it.

However I have been using it for mere 3 days, so I still can't give a final judgment on it in this department..

**Primary score: 7.5/10**

**4- Ease of use:** Although I haven't been using Ubuntu for an awfully long time (mere  2+ years), it still was a bit difficult for me to switch from the simple  trademark triple menu that gnome 2 got us used to.. It was also difficult for me to look down and not see the taskbar where it ALWAYS used to be.. However, I'm finding  my way through in Unity way quicker than I expected.. The launchbar (believe it or not) looks already easier for me to switch windows than the old taskbar (which was always FULL for me with open windows that all windows appear as (...) in the taskbar because there isn't enough room to show any part of the window title, so there wasn't a clear and quick way for me to remember which is which unless I memorize their locations and use the click and see method).. Unity is forcing me now to use a new way, that looks in fact much easier than the old one.

The apps menu is a bit more difficult to get used to, but I'm already getting used to that too, although some adjustments can definitely improve its accessibility and ease of use (including removing the "available for download" section that is ALWAYS shown when browsing your apps)..

Unity is very easy to use, however the only challenge that faces it is that people are used to another setup, and will need time till everybody is used to it.. I think the Mac users will find it much easier to work with than former Ubuntu users, but it have a lot of the Mac feel in it..

**Score: 7/10**

**Overall:** I'm sticking with Ubuntu 11.10, and unless something really disastrous happens in the following days, I don't see me going back to an earlier version of ubuntu..

**Overall score: 7.5**

[COLOR=Red]**Important note:**[/COLOR] I did NOT upgrade my 10.10 to 11.10, because Upgrade in Ubuntu = trouble.. I did a clean install after I backuped my files and removed Ubuntu 10.10 completely. And I think that's one of the reasons why I'm not having a lot of problems with it so far..

### Post by Meelad on 2011-10-17
> **anton706 said:**
> It happened to me too after the crash all that was left from unity was a menu in the top, How you restored unity?

Log in using Unity 2D or in the guest account or try to get access to the terminal in any way possible and run the following command:

```
unity --reset
```log out and in again using Unity..

If it doesn't work try the following command instead

```
unity --replace
```which is what I ran actually, and log in again.. Most probably that could solve your problem..

The problem is caused by trying to enable plugins in Compiz that need some other plugins to be disabled when those disabled plugins are needed by Unity, which disables Unity all together. That's why you need to reset Unity's settings to get it to work again..

Be careful in the future when making modifications in Compiz that involves enabling and disabling plugins..
