### Post by Lin24 on 2010-12-27
Hi ubuntu team (or whatever),

i would like to thank you for your work,

As a user of Windows 7, my computer went down because of a stupid malware (ie : Security Tool).

The huge problem was that i couldn't access anymore to windows, neither from normal boot, safe boot, restoration... and so on. It happened on Xmas (Roflmao Happy xmas :KS) and i spent my time until now trying to repair this %@!, using every way possible, windows repair boot cd, rescue cd from MANY antivirus companies, Ultimate Boot and so on... 

Unable to reach my windows log and having no backup cd or windows 7 installation disk, i did really think that my computer became useless :frown:...

Until, i saw the BitDefender Rescue CD : Based on a Xubuntu boot, i could run internet and run some applications... I just was "WTF", because until now i was thinking that all my problems were coming either from virus or failing materials, but then i realised that the main problem was : Windows 7 itself which became corrupted because of Security Tool.

After some googling, i came to the conclusion that ST corrupted my windows registry, making windows 7 unstable... But how can i repair the registry if i cannot even log on windows?!

Then the solution came up naturally : I downloaded and burned Ubuntu, installed it on multi-boot, i downloaded Wine, i ran some registry repair, then... ALL PROBLEM SOLVED \\:D/, i can run friggin windows 7 again...

Yet, i wasted 30 hours on this problem that would not have happened if windows was not that flawed (How can you repair this if you can't even run safe mode? if you don't have another computer next to you?).

And as i was searching for a antivirus tool for Ubuntu, i saw that it was not useful because there are only (according to wikipedia) 40 virus on linux, VS 60 000 (!) on windows...

As for now, i will run ubuntu most of the time, but i still need Windows 7 since most applications are compatible only with, what a pity.

That's all, again i would like to thanks Ubuntu team for creating such a safe and reliable work. I will now sleep peacefully and enjoy my remaining holidays.

I wish you a happy new year.

Truly, Lin24
