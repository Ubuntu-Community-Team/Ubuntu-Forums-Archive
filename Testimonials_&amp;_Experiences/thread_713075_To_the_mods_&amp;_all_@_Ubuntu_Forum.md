### Post by freebeer on 2008-03-02
Nice write-up.

My experience parallels yours. The first distro I tried to install was Fedora.  It did install, but there were all kinds of niggly little things that wouldn't work.  Mouse, sound, video, etc.  That wasn't really Fedora's fault, it was mine as I was a complete noob to linux and just getting my head around the new environment was daunting.  Then I heard about Ubuntu and decided to give that a try. (6.06) The install went very well and most things worked well for me.  I'm sure the stuff I learned by using Fedora helped a lot since I was slightly more familiar with the linux environment and had a few commands under my belt and kind of knew my way around the file system  (or probably more accurately, I learned how to search google more productively.) ;)

Subsequent installs of Ubuntu on other machines (7.04 then 7.10) went even smoother.  It's certainly a great distro and it meets all my needs for an OS (and meets needs that I didn't even know I had!) :D

The Ubuntu community is what seals the deal. Ubuntu (and the community around it) has made computing fun again.
