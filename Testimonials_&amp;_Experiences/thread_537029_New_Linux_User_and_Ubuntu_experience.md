### Post by Chief_Darkcloud on 2007-08-28
I want to share my experience with Ubuntu which has been very positive. I am a disgruntled ex- windows user tired of crashes, viruses, problems and decided to try Linux. I played around with Mandriva, Red Hat, and Ubuntu and think Ubuntu is the nicest of them all. It runs smooth and fast and have yet to experience any crashes/lockups. I like the feature where you can get software and updates just by selecting and clicking apply. Within minutes I had XMMS, KDE, and some other programs auto install for me. The built in CD burning ability is much nicer than Windows. I liked it so much I dumped Windows on both my home computers and will dive into learning as much about the OS. After a few months of trying it I will probally migrate my business computers to this OS. 

Only things I would like to see improved
1. A more user friendly disk partition manager during installation. 
2. Fix problem with some ATI cards using desktop effects. Out of my 2 systems the older ATI card works well with this feature and my newest higher end card does not. 
3. Have media players already included support MP3's. I had to get XMMS since the rhytymbox would not play them. 

Otherwise an excellent system and one that is very user friendly for a Linux newbie like me.
