### Post by CDL-WarChilde on 2011-02-07
So my main lappy took a header (with my help, need to replace mobo, dc jack replacement went awry).  So I have this old Panasonic Toughbook CF-51.  I've used Ubuntu since daffy duck, or so, anways, I installed v10.04 from the live CD.  Performance wasn't too shabby, but I thought it could be better, so I found a minimal install guide.  I have noticed a performance difference, not a huge one but enough to know that the minimal install, I like better.  

So far, this Pent M 1.7GHz, 512MB RAM, 20GB HD has been chugging along with no issues.  For what I need a laptop for, Ubuntu has been filling the bill.

### Post by CDL-WarChilde on 2011-02-08
> **mips said:**
> You don't have to use Gnome. You could do a minimal install and use LXDE or even Openbox+tint2 etc etc

Quite right, but I wanted to see how performance was with just a minimal install and gnome.  Particulary given the fact that I really didn't use everything that comes with the Ubuntu liveCD install.  My main reason for staying with gnome was familiarity.  I tried a distro with LXDE, Mint (Isadora), and was pretty impressed with performance, but, as with any relationship, I just had to try one more time with ubuntu+gnome.

> **cariboo907 said:**
> I found that using the gnome-desktop-environment, from the repositories, is much faster than the Ubuntu-desktop, all the enhancements that Canonical have added really slows gnome down.

As I stated before, the difference wasn't earth shattering, but it was definitely noticeable.  Gnome has been peforming quite well as a basic install.

My other reason for performing the minimal install is that I come across more hardware that isn't 4GB RAM, dual/quad core CPU and 300GB hard drives.  Most things that I come across are comparable to the specs of this laptop.  I really think that for the majority of people, a basic OS with web browsers and word processing software is sufficient for most people.  I also like the fact that most Linux distros now automatically support quite a bit of hardware, out of the box.  I remember trying to install Mandrake with a 3dfx video card...........ugh.
