### Post by lowracer on 2006-11-10
...and sells his iMac on eBay.

Been a unix user off and on since about 1984.  Bought my first Mac in 1989 and stuck with the platform through the years.  Finally OS X came out and that's Unix under the hood, used it for a few years then started thinking, so what's OS X got that Linux doesn't have?  Heard about Ubuntu a year ago, tried it on my iMac, it just didn't work.  Put it on a shelf.  Year later, tried again, it worked.  Tried it for 3 weeks, hey this does pretty much everything OS X does, and some things even better.

Was getting sick up and fed with iPhoto.  I have 12,000 (or is it 14,000) photos, and iPhoto always claimed it could scroll through these with ease. Well it never did and every year I sent in my $79 hoping the next version had fixed the bugs, and been disappointed that the bugs were still there or fixed but new ones in their place.  Then the OS costs $129 every couple of years just to add on a new feature like Spotlight or Expose or something.  Is it worth it, I ask myself? To keep sending all this money for so little in return?

Back to Ubuntu, tried DigiKam and F-Spot, hey these can replace iPhoto easily, and I get to organize the directories my own way, not the iPhoto way.  The scrolling is fast.

All my other favorites are here, Firefox, email, MP3-player (AmaroK - MUCH better than iTunes!).  This could work.  Mounted up my Mac partition, brought in some photoshop files, hey look at that, it reads those too!  Hooked up my camera, recognized & downloaded photos like a champ.  USB stick, check, mounted and opened the folder for me to use.  Firewire drive w/mac hfs file system, mounted and using with no problem. Canon scanner that never worked right on OS X.  Scanned with no complaints, no wrestling with drivers or badly written Canon software apps.  Openoffice.org reads all the other MS Office junk I have lying around too.  Now for the final bit.  Did some research and found out that UNREAL TOURNAMENT 2004 is on Linux too!  OK where do I sign up?

Went on to the internet, busted out the Mastercard. Bought a whole basket full of PC parts.  Put them together over two nights.  Intel D975 Mobo, Core 2 Duo, Nice Lian Li aluminum case (hey I was used to the mac's excellent case design, couldn't skimp on the aesthetics), nice 20" wide Samsung LCD.  Built PCs before, they never work right when you turn them on. Always something you have forgotten to hook up or hooked in wrong. Maybe some smoke and buzzing if it's real bad.

Put the Edgy LiveCD in the tray, gritted my teeth, hit the power button, and what do you know?  It all worked!  Installer came right up.  Got the system installed, tweaked the xorg.conf file to get the full screen resolution, and everything was there, recognized all my hardware, video, sound, network etc.  Copied in the files from my Mac and I'm good to go. 

What a great experience.  I'd like to say to any and all Ubuntu developers out there and Linux kernel developers THANKS!  This is awesome.  You guys should be proud to have made such a system for us all to use.  :cool:

### Post by 3rdalbum on 2007-01-19
Testimonial coming right up!

I'd only ever had Macintoshes, since about 1991. Loved the classic Mac OS. Disliked Windows due to its difficulty of use - I still believe that Windows 98 was a less stable operating system than Mac OS 8.5 or even OS 9 as long as you'd done a clean install of the Mac OS. I had used Windows a bit at my high school, but never owned a PC.

Then Mac OS X came out. My university switched to it. I hated it - the Dock was an attempt to roll the control strip, application switcher, and "programs tabbed folder" in one. I remember at the time, one of the Mac magazines compared OS X to a car, where all the internal storage compartments have been removed in favour of the driver "dragging everything to the floor". I still call the dock the "floor", even now!

It was out of our price range to buy a new Mac to run OS X, and I knew from experience at university that our iMac would run OS X like a dog (besides, we didn't really have enough hard disk space). So we stuck to OS 9, and slowly became unable to run the latest programs.

This also meant that we couldn't go to the latest websites, because they used features that only the latest web browsers had, and the latest web browsers were only available for OS X.

Ever since my school days, I'd always been interested in playing around with Linux or BSD; I don't know why. I'd never acted on it because I didn't have broadband. One day though, after we DID get broadband, I bit the bullet and did some research about running Linux on a Mac. Only a couple of distributions were available that did it, and Ubuntu won out amongst them in my eyes because:

1. I could try out the Live CD before installing it
2. It looked quite user-friendly
3. They would ship CDs to me for free!

I'll cut the story short: I found that my computer became usable again with Ubuntu, the user-interface was quite agreeable and there were enough UI storage compartments for me to use. It wasn't fast, but the latest versions would still run on it (unlike OS X). And I loved not having to pay for software.

Linux on PowerPC does have some limitations though, due to the proprietary world we live in, so I eventually bought a cheap PC in May last year and immediately installed Ubuntu onto it. It runs beautifully.

Almost exactly a year ago, I had installed Ubuntu on my iMac G3. To this day, Ubuntu is still on that machine, and I occasionally use it. In fact, I'm writing an Ubuntu derivative specially for obseleted Macintoshes (looking for community involvement!)
