### Post by stevenwscott on 2007-11-16
First I should start with the accolades: Outstanding job, really. There's not another Linux distro out there that's quite as "granny-friendly" as Ubuntu, and I am indeed impressed. From the driver detection and inclusion to the ease of use, I think you folks have done a superb job. 
Now the whine: However, who is scripting your X configuration gens? My god man, I've had to do more tweaking and poking with my X config under Ubuntu than any other distro, and more than not, FIX what Ubuntu has determined to be a valid config. 7.10 seems to be worse in this department, and BTW, the config generated for my dual-head monitors precluded actually trying to use them as a single screen as one would typically do. It wasn't until I removed the refresh-rates from the MODES section that it actually started working. You've also got some funky issue on the "configuring apt" part of the install - methink it's because it's trying to use an ethernet adapter before the adapter is actually configured. I downed the adapter to get it to continue. 
  All in all, I'm still very impressed, just wondering why X seems to have received less attention than other areas, particularily in a desktop distro.

SWS
