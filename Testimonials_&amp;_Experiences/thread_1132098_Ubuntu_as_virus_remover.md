### Post by lazyart on 2009-04-21
I've long used a live CD as a way to retrieve files from an unbootable (or unusable or compromised) Windows machine.  Today however there was a different use.

A Windows machine running F-Prot was able to detect a virus but unable to remove it.  I used another free antivirus that I could run a boot-time scan.  It unfortunately did not even recognize the files as infected.

So I booted up via my USB key (great inclusion into Intrepid was the ability to make bootable USB media) and into Ubuntu.  Mounted the disk, located the problem files and renamed them.  Booted into Windows to verify there were no issues and then deleted them.  Computer is clean, people are happy and I'm $75 richer.

You can hate Windows all you want, but it pays my mortgage.  Thanks Ubuntu!
