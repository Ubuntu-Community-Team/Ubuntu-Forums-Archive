### Post by user sam on 2009-08-28
First off, I love Linux. I adore Ubuntu. I can say with confidence, if not objectivity, that it is the best operating system that I ever used. It could safely be said that I am completely nuts, even obsessed with this finest of operating systems. And it is for this reason that I must separate myself from Linux for the time being.
It seems like I've been using linux operating systems for years while it has only been maybe six months. It is the ultimate tool, and I have misused it. I write this on a borrowed computer that sits next to my desk. My personal computer has been wrecked as a result of my stupidity, and I don't know if I will ever see it run again. 
I couldn't stop. First I was using OS X. I got bored with that and installed several linux systems in virtual-box. I expanded OS X with all the options I could find and nearly destroyed my system. After a while, it occurred to me that it would be more convenient. To install linux to my hard-drive. After I did this, I was happy for about five minutes. Linux isn't enough. I will have BSD as well!
So I split my drive and installed FreeBSD. It didn't boot. After about five tries, I reinstalled ubuntu, and then tried again with BSD. I blew everything up again. I tried using the os x startup disk to fix my broken system, but it was powerless to fix such foreign file types and then boot from them. After a last ditch effort to save my system, a disk is stuck in the drive and I can't get it out. The computer can no longer boot from the cd drive or the hard drive and I am completely screwed.
maybe no one will even see this post, but that's ok. I just want to say that someday I may have the self control to successfully use Ubuntu, but until then my geeky passion is merely sleeping, awaiting the day I can tame it.
I know, I am so overly melodramatic.
