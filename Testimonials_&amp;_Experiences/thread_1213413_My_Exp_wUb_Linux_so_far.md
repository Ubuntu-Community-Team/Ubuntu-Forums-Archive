### Post by Ms_Angel_D on 2009-07-16
a mini iso is a minimal install, it only only installs the basics required to have a system running. Where as the server, installs well server stuff, and unless your run web servers & mail servers and such you don't need them and they are adding to your system load. 

You can do the minimal install then sudo apt-get install ubuntu-desktop for gnome and your system should run just fine.

[https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)
