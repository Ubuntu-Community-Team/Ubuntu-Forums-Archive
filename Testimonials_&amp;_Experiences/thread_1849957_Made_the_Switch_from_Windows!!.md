Im not totally done with Ubuntu, BUT those two issues are big ones for me.

What kind of graphics card are you running? And are you using the proprietary drivers for it? If you're running an ATI/AMD Radeon, then there have been issues with the proprietary driver accelerating stuff like videos, Flash, and other stuff like that. That's more ATI's fault with their Linux drivers and not necessarily Ubuntu's or Linux's. I run the open-source driver on my netbook (which runs an AMD Radeon HD... high-end single-core netbook) and get good video on both my VLC and my streaming Flash videos (YouTube, etc.)

NVidia, I hear, runs pretty alright with videos and other 2D-accelerated stuff on both their open-source and proprietary drivers. However, Unity and other 3D-accelerated stuff tends to run slower on the open-source nouveau drivers than on the proprietary ones. I would check those and see if that helps.

I only know that because I have 2 computers: A desktop with NVidia HD graphics, and a netbook with ATI HD graphics.


**Back to the subject at hand:**

I started using Ubuntu at the beginning of the year with Ubuntu 10.10, mostly just testing it out on a persistent liveUSB. I did a permanent installation when Ubuntu 11.04 and Unity came out, right when I had to reformat and reinstall Windows XP Pro anyway. So there was no better time for me to jump into Ubuntu :D I just formatted my hard drive and partitioned it accordingly, and I now have a dual-boot of Windows XP and Ubuntu 11.04.

I prefer to run the Ubuntu Classic (GNOME 2.32) desktop environment since it's very customizable. I can make it look and act like Windows or look and act like Mac OS X very easily with just a few panel apps, panel placements, panel app arangements, a dock app (Docky, AWN, Cairo, etc.), global app menus, etc. I love how I can customize GNOME 2.32 to look and act any way I want. It's sad that development and support for GNOME 2.32 has ended in favor of smartphone-like GNOME 3. 

But Unity... I actually like Unity's interface. I can add and remove apps from the launcher in pretty much an identical fashion that I can with Windows 7, and arrange them how I want. But, it sucks that Unity is not very customizable. I can't even change the icons for each app launcher like I can with most of the apps on my AWN. Also, the panel content can't be rearranged nor customized either. There are a couple things I don't like about Unity, all of a minor nature, mainly because I got hooked on GNOME 2.32 (and right at the end of its run too). Sure, I mainly use GNOME 2.32, but I do play with Unity when I'm in the mood, especially since Unity **is** the future of Ubuntu. I like Unity's interface for what it does, and I can certainly live with it very well despite its lack of customization.

Now will I make a permanent switch to Ubuntu and leave Windows totally in the dust? No. As long as at least Windows 7 is still supported, I don't see myself leaving Windows completely. I still use it for a few programs and for a number of games (MMORPGs and others) that don't play well or not at all on Wine. I also use it to connect to my XBox 360 as well. Needless to say, my desktop computer is my true multi-purpose machine. The same goes for my netbook with keeping Windows 7 around. It's my mobile internet, and it actually plays some MMORPGs very well despite being a netbook (high-end one too). In addition, I use it to connect to Windows networks, which I still have yet to figure out on Ubuntu. And when I take it over to a friends house and my mother's house, their WiFi networks play with Windows just fine, but connect very poorly with Ubuntu (I blame their ISP's for that, not Ubuntu). I still have **some** use for Windows, so I'll be dual-booting Windows and Ubuntu for a while.

If it's any consolation, I use Ubuntu a lot more than I use Windows for all kinds of things.
