### Post by gnuvistawouldbecool on 2008-09-16
This is my experience with Ubuntu.  It was the first linux system I came across, though I had heard of it many times before then.  A friend said it wasn't as good as others, such as Mandriva, and even now, I agree, that is the case - in some areas.  Other areas I find Ubuntu is better.  With that said I did take a while until I installed it.  I then installed Kubuntu (7.10), which was OK, but it wasn't as good as I would have liked, due to some strange reasons for some things not working with it.  Then I tried to install standard Ubuntu 7.10, but pressed the wrong button, causing it to fail at the install and the boot loader was promptly giving error codes.  So then I installed Mandriva, and didn't look back until a few months later.

It was then that I realised that if I was going to get good with Linux, I should probably use a more popular one in cas I would ever need to help anyone with it, so I installed KDE 4 8.04.  Which I must say I find to be a bit rubbish, really.  I used that and the standard Kubuntu in both a proper install and Wubi, but didn't really like them that much, though by then I was familiar with KDE and so liked it anyway.

Then I decided to download 8.10 alpha 4, which for an unstable release I find surprisingly good.  So I use that, and it is better than Mandriva in terms of hardware support, except for the sound volume buttons, for some strange reason.  That said I prefer *(Mandriva's large choice of packages to install)* the way it looks in general, though Ubuntu's GNOME is at least on a par with Mandriva's.

So all in all, I would recommend either, but with this forum (for support) in mind as well, would recommend Ubuntu for a first time user.
