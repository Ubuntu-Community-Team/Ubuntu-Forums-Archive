### Post by Hedgehog1 on 2011-04-01
I installed the beta of 11.04 on my testing system.  It is an HP laptop (C50) with Nvidia on board and a 1.7 gig processor.

So far in my testing, the only thing that didn't work on the first try is the 'cheese webcam booth'.

Audio works, all the special laptop function keys work.  Nvidia driver hums right along. Wireless is flawless.  Sleep works, too.

I know there are still lots of little bugs to fix, but for a beta, Natty is looking very good.

The install is still a little tricky.  I had to install with the 'download updates during install' unchecked to finally get it to complete.  After first reboot, the loading of the Nvidia driver leaves Unity non functional for the first boot.

By the second reboot with the Nvidia drivers loaded and the 150 updates loaded with package manager, things are running nicely.

I still do not suggest that inexperienced user load Natty at this time; it is the first beta and only appropriate for more experienced Ubuntu users.

However, based on what I am seeing, the formal release of Natty will deliver a very usable distribution!

[SIZE="2"]**And Unity? After 30 minutes you forget it's there.**[/SIZE]  It's very functional, and a little time is all it takes to settle into it (now that it is working as intended).  Please don't judge Unity until you have used the correctly functioning version for a bit.

### Post by 3Miro on 2011-04-01
Yes, I had some trouble installing the beta from both the regular and alternative install. Once there, it wasn't very stable. Overall this is the most unstable Ubuntu beta that I have seen (count the last 3 releases). It is usable, but I do not recommend it for regular use.

Notes on Unity:
- Unity is nothing more or less than a dock on the side, new menu and a top panel allowing for the File menu of maximized windows as well as notification area and sound settings. The rest is compiz.
- Install CCSM to do setup compiz as you would normally do.
- The Dock now has a very nice auto-hide feature. I for one have not seen this before, it stays on normally, but hides if you try to move a window on top of it (or you maximize a window). Basically, it is there if you need it and it moves if it is  in your way.
- The menu now shows categories (Accessories, Network, Multimedia) if you right-click on it. Other than that, it still  gives you 50/50 installed/not-yet-installed applications. I wish it were at least 66/33 (two rows installed one row uninstalled).
- The menu also supports all sorts of searches.
- The top panel is somewhat buggy. While it allows for full-screen applications to get much more of your screen, it shows and hides the menus in a rather buggy fashion.
- Other than that, I had to change the window buttons the standard way (select Clearlooks, if you want, you can then customize to get back the colors and decorations, but with the right-buttons)

Things that I have not yet figured out:
- Opening new window in Firefox starts it in full-screen. It can probably be fixed via CCSM.
- There is a nice dock applet for changing workspaces, I wish I could change them by just scrolling over it.
- Some applications will not start form a dock launcher (like Geany). Probably some bug.

With a few more features, I could call Unity a nice fully functional system (and fix also fix the bugs).
