### Post by murderslastcrow on 2010-08-04
Go with Xfce4 (Xubuntu)- it's basically Gnome, but lighter. It can also use a lot of Gnome applets with Xfapplet. It can run comfortably on 128 MB of RAM, so it should be quite nice on 256 MB of RAM. Also, if that computer has a video processor with more than a few megabytes on it, you can probably get compositing running for a dock/compiz/fades/shadows, and so on. Xcompmgr is extremely light and can provide simple fades, shadows, and other niceties even on very old computers. Compiz isn't the only option.

Also, if you're really concerned with making it super light, I'd install Xubuntu and then install something like fluxbox inside of that. You can get nice round corners in FluxBox and it's one of the very lightest window managers around. Of course, it has a lot of good themes, including a KDE4 based theme, so it's very versatile and worth taking a look at.

Also, e17 is amazingly full featured and has been reported to run on 64 MB of RAM.
