### Post by AusIV4 on 2007-04-08
I've been trying for some time to convince my mother to try Ubuntu, but she's one of the more computer illiterate people around, and she's just starting to get used to XP, so she's not willing to make a switch.

About a week ago, her computer died. It would load Windows, but once it got to the desktop it would freeze up completely. I had suspected for some time that her hard drive was a bit questionable, so that was the first thing I looked at. I popped in an Ubuntu (dapper) live CD and rebooted. It got to the "Detecting Hardware" phase, and froze up.

I powered down, unplugged the hard drive, and booted it back up. It got past the Detecting Hardware phase, so I powered down again and took out the hard drive and put it in an external case I happened to have lying around. I plugged in the disk and was able to mount both the installation partition and the Compaq restore partition, and both seemed to be intact. I went to my local computer store and picked up another disk. I popped it into her computer, hooked it up, and booted to the live CD.

Next, I used 'dd' to copy her old drive to the new one. It took about 5 hours (it was an 80 GB drive), but when it was done, Windows booted right up - good as new (or at least, as good as any 18 month old Windows install).

So, while I'm not exactly thrilled that my mother is still using Windows, I'm glad Ubuntu was there to help keep things up and running.
