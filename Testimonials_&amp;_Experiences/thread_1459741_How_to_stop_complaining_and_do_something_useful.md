### Post by clegends on 2010-04-21
Hi folks, just gotta say I'm loving lucid, best release yet. I've got a fresh install of 10.04 netbook-remix with all updates on my eeepc 901, upgraded to 2gb of ram with an upgraded 64gb internal SD. Go ubuntu!

Points I really like:
1. Interface - great job with ambience theme, and minimilistic system tray icons, looks very cool
2. Overall speed and improvement - working beaut, boots great, and is fast.
3. Stability - still in beta, and already quite stable. I'm really impressed
4. Ubuntu One store - great concept
5. Social networking integration - also great concept
6. Minimal bugs, nothing show stopping, before the release date.
7. Hibernation and suspend work near-perfectly - it is a beta, etc.

What I don't like:
1. No configuration options for notify-osd - not a biggie, can always disable it if it really bothers me
2. slight lag on wake up from suspend before I can use mouse or keyboard - this is just nitpicking, really.
3. Default UNR interface... not my cup of tea, but great job on it anyway. Simple to log into gnome ui instead. Very slick.
3. All the complaining on these forums

All up I think this is a winner, and looks, feels, and is, very flash. Unlike past releases, I've really got to think hard about areas to improve. Keep up the great work folks!

As far as all the complaining on these forums... get a life people!!! Really, it's as if you own Ubuntu or something... yes it's a community project, to an extent, but it's owned and run by Canonical, overlooked by a team of people, and managed by one person. I personally think they are doing a great job, and staying true to open source ideals. But even if I didn't there are far more useful things I could be doing than pissing and moaning about it on these forums.

Eeebuntu is a classic example. They decided to branch off and create another distro, which they are doing... There are numerous other examples. If Ubuntu really bothers you that much, and you're not willing to help fix it, you have every option to leave.

For myself, I think the recent changes are great, and a step in the right direction. There has been a lot of complaints about the default buttons on the left... it's really easy to fix. Open up Preferences>Appearance>Themes and choose a different theme... there are only 3 with the buttons on the left. 

For those more adventurous, do some searches on the web for different window manager themes. Personally I use Emerald, as I find it more stable with compiz than metacity. Cripes, you can even install emerald, and the buttons are on the right to begin with. It will probably take you less than 20 minutes!

For those who don't like the decision to remove some of the icons in the menu, open up gconf-editor, go to desktop>gnome>interface and tick 'menus_have_icons'. Wow... was that easy, or what?

Seriously, if you don't like Ubuntu, there are hundreds of distros out there... most of them free. There are even distros that help you make your own distro. I think Ubuntu is currently a great distro, which is why I use it... but there's nothing to stop me using something else, or creating something on my own. 

So hats of to the Ubuntu team, keep up the great work!
