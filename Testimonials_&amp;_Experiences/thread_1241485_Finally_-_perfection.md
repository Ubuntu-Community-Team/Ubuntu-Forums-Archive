### Post by steveneddy on 2009-08-16
Having used Ubuntu since Warty and fighting xorg.cong, Nvidia drivers, Beryl and Gutsy, I think we are finally getting close to perfection.

I thought i would stick with Hardy due to the fact that it is an LTS, but recently upgraded to Intrepid (back up /home and totally reinstall) and I think that all of my issues are gone.

The Synaptic touchpad finally works like it is supposed to.

My Bluetooth mouse hooks up every time, no hassles.

The machine sleeps AND wakes up! Even with Compiz turned on.

Sound works. 

Compiz finally good enough to leave it running all the time.

If one of these things were OK in Hardy, there was another that would be tweaky. Get the tweaky one fixed and another one would go haywire.

I would like to try out the newer versions but this is my work machine and I cannot afford for it to go down of have to fix stuff just to work.

I can't wait for the next LTS. I believe that we have finally created something that is in reality a very strong contender for a world class operating system that can compete. I mean, really compete.

I need to get it home and see how it integrates on the network and how the  HPLIP work with the All-in-one in my SOHO. But that is weeks away and I'll cover that when it happens.

Regular and long time Ubuntu users should be proud of the work that has happened in creating this product. All of the posts and threads hammering out issues just to get things to work right together.

Just perfect. 

To whom it may concern - thank you.

### Post by Marflus on 2009-08-16
I use 9.04 and have had no major problems that I haven't been able to overcome, and certainly out of the box there were no difficulties. I had the same experience when I used 8.04, too. If you're lucky enough to (or have planned to) have compatible hardware, Ubuntu is a fantastic OS.

Perfection is an extreme word, though. While it's a good distribution that I use as my main OS, there are plenty of things it can improve on. It's brilliant that it does everything you need it to, but I'd not go so far as to describe it as perfect - indeed will any OS ever achieve perfection? I doubt it greatly.
