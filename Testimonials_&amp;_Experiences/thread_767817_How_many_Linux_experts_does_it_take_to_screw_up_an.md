But as a research scientist, I can tell you that linux is a far better platform for developing scientific software (in most cases).  In addition, there is a lot of scientific software available on linux because it is an offshoot of unix, which was around long before Windows or Mac OS existed and continues to be used today. Why should we switch to another platform that is (1) more difficult to develop for, (2) has far fewer scientific libraries and tools available, and (3) costs more?

Furthermore, many scientists work at universities or other locations that have Microsoft site licenses, so not using their software is hardly sticking it to them.  Additional Windows installs earn Microsoft very little (if any) in those cases.
