### Post by radagast2 on 2006-06-26
I just installed a dual boot of Windows XP and Ubuntu 6.06 from scratch; since it was the first time I'd installed either of them (though I had installed Debian 3.0 and many prior iterations of Windows), I figured it'd be fun to track the time it took me to install both operating systems.  

Long story short, Windows took me twice as long (2 hours) as Ubuntu (1 hour) to install and get basic functionality.  While Ubuntu's base install left me with a fully functioning system (all my hardware was working, etc.), Windows XP didn't recognize either my network card or sound card (so I had to manually add the drivers).  In the first three hours of use I also got multiple ads from software in Windows XP, but have yet to see a single software-based ad in Ubuntu.  I'm amazed by how easy Ubuntu was to install, and love how everything just works right out of the "box".

I've written a full comparison (including data tables) of the two installs here: [http://rhosgobel.blogspot.com/2006/06/installing-ubuntu-comparison-of-ubuntu.html](http://rhosgobel.blogspot.com/2006/06/installing-ubuntu-comparison-of-ubuntu.html)
