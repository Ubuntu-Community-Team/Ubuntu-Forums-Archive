### Post by muzikayise on 2010-09-06
[COLOR=black][FONT=Verdana]I just moved onto to Ubuntu this past weekend, i was going to reformat my laptop by using my recovery discs (Vista Business) so i can go back to day one, again. but i decided to give Ubuntu a chance.[/FONT][/COLOR]
 
[COLOR=black][FONT=Verdana]I&#8217;ve always wanted to make the switch to Ubuntu but I&#8217;ve was reluctant to do so because I&#8217;m a .NET developer therefore everything i do is on the windows platform. also back then i only had one laptop but now that i have two laptops at least then on the other laptop i can still use that for development.[/FONT][/COLOR]
 
 

[COLOR=black][FONT=Verdana]I must say I&#8217;m very impressed with Ubuntu, it runs so well on my laptop its way faster than Vista. I&#8217;ve got the following specs:[/FONT][/COLOR]
[LIST]
[*][FONT=Verdana]Toshiba Satellite Pro[/FONT]
[*][FONT=Verdana]Intel core 2 duo 2.4GH[/FONT]
[*][FONT=Verdana]4 GB RAM (only 32bit processor therefore only using 3GB)[/FONT]
[*][FONT=Verdana]250 GB HDD[/FONT]
[*][FONT=Verdana]DVD RW[/FONT]
[*][FONT=Verdana]256 MB Graphics card[/FONT]
[/LIST][COLOR=black][FONT=Verdana]When i had Vista it would use at least 43% of RAM and it took some time to boot but once i had logged in it worked fine, i didn&#8217;t have too many problems with it, i was able to develop using visual studio 2003, 2005, 2008 & 2010. as i said it worked well for me that&#8217;s why i didn&#8217;t see a need for me to move. but we all know that it was a doggy OS and therefore that&#8217;s why i was so keen to try Ubuntu.[/FONT][/COLOR]
 
[COLOR=black][FONT=Verdana]With Ubuntu installed now, i amazed that it only uses 16% (460MB) of RAM, which means when i run Windows 7 as virtual machine I&#8217;ll be able to allocate about 2GB of RAM to the Win7 VM and I&#8217;ll be able to run visual studio 2010 ok. i haven&#8217;t tried this yet, but i think it will work fine.[/FONT][/COLOR]
 
 

[COLOR=black][FONT=Verdana]Summary[/FONT][/COLOR]
[LIST]
[*][FONT=Verdana]Ubuntu is simple and easy to install[/FONT]
[*][FONT=Verdana]it boots very fast[/FONT]
[*][FONT=Verdana]it updates easily[/FONT]
[*][FONT=Verdana]installing additional software works well and easy to use[/FONT]
[*][FONT=Verdana]email is a dream - basically it works just like my outlook 2007 did[/FONT]
[*][FONT=Verdana]i like the 4 desktop thing, or tho this is not new but it's new to me and its awesome[/FONT]
[/LIST][COLOR=black][FONT=Verdana]I cannot find any fault with Ubuntu, it&#8217;s really a great OS and i thank the Ubuntu team for their dedication and hard work, also thank the Linux and Open Source guys for their work.[/FONT][/COLOR]
 
[COLOR=black][FONT=Verdana]The name Ubuntu is perfect for the OS, because I'm Zulu therefore i know what "Ubuntu" means in our culture. and in Zulu we have a saying that goes "umuntu, umuntu ngabantu", which means: a person is person because of other persons (people) and the same can be said about this OS i think. Ubuntu OS is what it is because of the Linux peolpe, open source and everyone that has made it possible.[/FONT][/COLOR]
 
[COLOR=black][FONT=Verdana]Well done guys :D[/FONT][/COLOR]
