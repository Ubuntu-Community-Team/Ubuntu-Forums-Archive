### Post by jkugler on 2008-05-30
After trying it out, Hardy is not working out for me.  Mainly due to these two bugs show-stopper bugs:

[https://bugs.launchpad.net/ubuntu/+source/evolution-exchange/+bug/231220](https://bugs.launchpad.net/ubuntu/+source/evolution-exchange/+bug/231220)

[https://bugs.launchpad.net/ubuntu/+source/vpnc/+bug/228365](https://bugs.launchpad.net/ubuntu/+source/vpnc/+bug/228365)

And this somewhat annoying one:

[https://bugs.launchpad.net/ubuntu/+source/network-manager/+bug/228357](https://bugs.launchpad.net/ubuntu/+source/network-manager/+bug/228357)

Corporate uses Exchange and a Cisco VPN.  I have to be able to use both, and neither were working in Hardy (Exchange in this case being the Evolution Exchange back-end).

I'm back up on Gutsy, my VPN connection has been up for over 12 minutes (what a concept!) and my Evolution session is still checking e-mail from an Exchange server just fine.

We'll see what happens when Intrepid comes out.  Hopefully they'll resolve these bugs soon since Hardy is an LTS release.
