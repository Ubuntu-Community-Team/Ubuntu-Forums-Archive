### Post by ventrical on 2011-02-26
> **frequent said:**
> How do i make a disk like that i have no clue but i would love to do it.:(

1. Have Ubuntu 10.04 or Lubuntu ISO ready.
2. Remove all standard harddrives from system.
3. Insert pendrive in USB port.
4. Boot system with Ubuntu distro.
5. Choose <install to disk> option. Ubuntu will recognize your pen drive as an virtual harddrive. Follow installation steps <easy>.
6. Remove CD and boot from pendrive. Voila' make sure to update through DSL cable so you can get wireless drivers.
7*note* If you have an older machine you  can boot to USB by using the PLOP bootloader that can be installed on Floppy or CD.

 The secret is to remove the hard drive!!!!!!!!!!!!!  The pendrive becomes the default hardrive and can be inserted in practically any PC (95% of them) without any damage to settings .. etc..
