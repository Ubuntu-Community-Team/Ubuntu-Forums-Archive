### Post by linuxsudo on 2008-05-31
This version of Ubuntu is very good in comparison to the previous versions I've used and I've even decided to give Gnome another try because I always used KDE. This version has allowed me to use Gnome again and like it and I don't think I'll switch back to KDE again for some time.

I've found this version to be very stable above all else and for some reason my hardware seems to be perform better than it did with previous versions, especially graphics card performance.

I use both Windows Vista & Ubuntu and I've been using Windows since 3.1 and Ubuntu since 5.10. I use Windows for games and Windows only applications (I won't bash Windows because I believe it is a good OS despite some of the opinions which I've read on this forum) and I use Ubuntu for Java programming. 

I've also done the following things to make my Ubuntu experience more enjoyable such as:

Changing the system colours to blue
Installing a blue usplash theme
Removing some preinstalled applications such as pidgin
Removing Firefox 3 & Installing Firefox 2 (Sorry But I don't think FF3 is quite stable enough yet)
