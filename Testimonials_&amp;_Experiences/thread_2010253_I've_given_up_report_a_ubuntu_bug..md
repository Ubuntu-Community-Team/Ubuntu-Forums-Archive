### Post by cariboo on 2012-06-26
Generally if it isn't a specific package you've found a bug in:

```
ubuntu-bug linux
```

is the way to go, be sure to include as much information as possible, as the majority of unanswered bugs don't contain enough information to determine what the problem is.

Another thing you have to have is patience, many bugs aren't specific to Ubuntu, so you are dependent on upstream to fix them. For example I submitted a bug in gnome-network-tools 3 years ago, and the dev responsible finally fixed it just 3 months ago.
