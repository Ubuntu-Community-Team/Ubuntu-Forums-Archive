I'm with PriceChild on this one.  while RPM's will work on debian distributions using alien, they're not guaranteed to do so.  If you get if from the Ubuntu repositories (meaning, using add-remove or Synaptic) then it won't break your system because it's been tested to work on this system by a team of advanced users who care that Ubuntu works well.

It's not recomended for novice users to have to cope with the problems that RPM's (through alien or whatever else) can cause.  The only thing I'll disagree with PriceChild, is that debs are way easier than "from source" ... I'm assuming you haven't compiled in Unix environments before.  It's very easy for some of us, but not for everyone.
