### Post by kyp4 on 2007-12-03
I am a long time computer enthusiast who just recently got into Linux and Ubuntu. I learned to love computers from my dad who is a software engineer. I remember using DOS and Windows 3.1 back in the day and I started to program when I was around 12 back in '96. Like many people I started with BASIC (QBASIC) and then moved on to C and C++. I was just so amazed with what you could do with computers if you knew how. Naturally I went into computer engineering in college and graduated in April '07 from Wright State. I now work as a software engineer for a defense contractor and it was there that I was introduced to Ubuntu, which is what I have on my machine there.

I had always wanted to get into Linux as I love the idea of open source and realized, after some exposure to it and OS design in school, that it was simply a better OS than Windows. But I was always reluctant to try it because I thought it was difficult to install and had limited hardware support (especially since I had a laptop which, as I understood, was not very well supported). But in August '07 I built a new desktop computer and when I setup the hard drive I left a partition for Linux. A few months later I finally got around to installing Linux so I downloaded the latest Ubuntu (Gutsy), burned it, and off I went.

I was delighted to find how easy it was to install, I daresay it is easier to install than Windows. Within an hour I was up and running with dual boot into XP, which was done automatically. It automatically recognized my windows partition (r/w access) and all of my hardware, even the multimedia keys on my keyboard worked! I was amazed at all the software available (for free) and was pleased to find that I could do almost everything I could do in Windows in Ubuntu. I can use it to maintain my iPod with gtkpod and to put programs on my TI calculator with TiLP. There are even open source programs for the stuff I do with music production (Ardour and Hydrogen). The only thing I still use Windows for is gaming (which is unfortunately necessary) but other than that I am running Ubuntu. And now that run Linux I intend to begin contributing to some open source projects in my free time.

The point of all my ranting is that I couldn't be happier with Ubuntu and I HIGHLY recommend it to anyone curious about making the switch to Linux. If nothing else you can download and burn it and run it from the CD to try it out. I am now an avid open source advocate and am trying to convince as many people as I can to make the switch, telling them (truthfully) how easy it is to install and use. I think and hope that one day open source will overtake Microsoft and I think that Ubuntu is a giant leap in that direction.
