### Post by sambhogi on 2011-04-30
I've been using Unity (running off a USB key - I never install new releases until several weeks have passed) for a couple of days now. Yes, it is different. Yes, I miss the Gnome menus, and I am concerned about the comparative lack of configurability though so far have had no reason to configure anything.

So far though, I haven't heard any compelling reasons for the intense hate for Unity. It appears mostly to be an irrational reaction to change. 

When I use OS-X, as I occasionally have to, I too get annoyed at how I am forced to use search, or open a new finder window, to find an application, for example. But that's just habit. Over time, it's become less annoying.

Much depends on the search technology and the organization of search returns. If this is fast and intelligent, the new interface will work well. Indeed, the search technology in Unity appears to be more efficient than in previous Ubuntu releases.

Since there seems to be a move away from directory-based organization, some type of file-tagging would be very useful in the long run (unless it's possible now and I haven't discovered it).

Anyway, I am being open-minded about Unity. Hopefully there will be some changes made to desktop Unity to accommodate some of the concerns that have been expressed about it (dual screens, etc.).
