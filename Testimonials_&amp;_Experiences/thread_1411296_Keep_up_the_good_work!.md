### Post by anoop999 on 2010-02-19
It seems illogical for fundamental mainstream software such as an operating system or word processor, used by millions of people, to be controlled by a monopoly corporation.

Ubuntu works great for me. (Unfortunately I also have to keep a Windows partition for certain software). The main problems are not of Ubuntu's making:

- Problems with flash. However I rarely use flash and I think many websites would be easier to use without it.

- Problems with wifi or other hardware drivers. I searched the internet and eventually found a line of code which activiated my wifi card.

- Problems with Word documents. ODF is the best word processor format because it is well documented, ISO approved, easy to parse and relatively compact. It saves on storage space and bandwidth but requires extra computing power than binary formats, but computers are powerful nowadays so this is not a problem. Microsoft Office should provide full ODF support, but it currently does not do so. I now use LaTeX or LyX wherever I can, which produces compact HTML and PDF output and can cope with very large documents.

- Problems with spreadsheets. OpenOffice can now run many Microsoft Office VBA macros. OpenOffice is slower than Excel with very large spreadsheets, but such large amounts of data would be better stored in a database and analysed with database or statistical software. This requires a change in the way we think about manipulating data using a computer, but is safer and more versatile.

Things I like about Linux and Ubuntu:
- Free, open source
- Live CD, so you can try before you install
- Powerful, accessible command line
- Many programming languages installed by default - you are actively encouraged to write your own scripts and programs to tell the computer what to do, rather than have it tell you what to do
- Software repositories and package management
- Security and lack of virus vulnerability
- Fast file system with no need to de-fragment
- Hardware support (for supported devices) is excellent, with no need to download drivers

My wish-list for further improvements:
- Make it easier to revert to a previous version of an updated package in case it causes a regression
- Support more types of hardware
- Liaise with manufacturers to sell new computers with Ubuntu pre-installed, or compatible systems with a blank hard drive
