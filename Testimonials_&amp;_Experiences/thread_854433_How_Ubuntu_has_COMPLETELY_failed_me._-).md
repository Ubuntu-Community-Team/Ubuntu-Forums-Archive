Hello all, I am making the switch back to XP for some very good reasons.

1.  WINE will not play 100% of ALL my Windows games and software flawlessly.
2.  The GUI is just way too customizable.
3.  The ability to choose 1000's of software titles in the repositories is a huge negative.
4.  I cannot run my favorite AV and AS scanners.
5.  I want to run defrag.
6.  IE 7 is the best web browser out there and it has tabs.
7.  Choice is a terrible thing.
8.  Microsoft says Linux has violated 230+ of their patents.
9.  Linux should be a free Windows.
10.  I think corrupt registries are awesome.
11.  Ubuntu installs way too fast and it's updates don't take about an hour to install.
12.  How can it be a good OS, it only takes up 1/4 the space of Vista?
13.  Security in an OS is way overrated.
14.  Having an office suite, PDF reader, email client, web browser, music player, full featured image editor, C/C++ compiler, PERL interpreter, data compression tool, CD/DVD burning app, and multi protocol IM clien installed by default cannot be good.  You are supposed to scour the web and illegally install your friend's copies to be good.

Another reason I was pushed away from Ubuntu is that when I installed it my PC my car all of the sudden would not start.  Ubuntu has now infected my transportation and that is a big no no.
