### Post by dewdrop_world on 2011-01-19
A friend of mine is setting up a new box to mirror my Ubuntu configuration (running audio software for a multimedia dance piece). Had some issues installing a real-time kernel and Jack2, but it has a happy ending.


> (The RT PPA) was FUBAR anyway - I got some message about using a previous version of it.  When I went to install linux-realtime, I got a SLEW of error messages, and then "Installed".  Rebooted OK, and went to add JACK2, and got some message about how the package system was scrambled, and how it was probably some 3rd party repository screwing me.


BUT WONDER OF WONDERS!  It went on to tell me what to do, and when I typed in the command it cleaned everything up, reinstalled linux-realtime, and then I was good to go!  OS software that just didn't just vomit and die - I almost fell off my chair!  Then JACK2 went fine.



I should note, he is extremely demanding and gets quite annoyed when software chokes (and routinely complains to me about the "unusability" of Linux). He was impressed. That says a lot.


So, whoever worked on making apt more robust, and even *helpful in case of failures* (!), big up! Great work!
