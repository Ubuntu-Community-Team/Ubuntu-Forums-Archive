### Post by blackgibson on 2006-03-17
I've been a Linux user in one way, shape, or form since 1994 or so. I first downloaded Slack onto 40 floppies to get the most I could out of my 486sx33. 
It really bothered me that every ounce of preformance wasn't being eeked out of my computer. They sure cost enough, why cripple it with bad software? Okay, Ill fess up. it was that and Afterstep looked SO much cooler than Windows 3.1 ;) 

I have Always looked to run the for distro with the most up to date ( and usably stable ) software with package managment above and beyond Slack's. While I've been known to mung up my system with a bleeding edge package or 2, I've never been one to run devel distros, and Debian stable being as mouldy as it is ( and for good reason, don't get me wrong ) My only Debian experiences were with offshoots like Corel and Stormix. Both soured me on Debian baised distros for a long time. This lead me into a what seemed to be a never ending cycle of switching between Redhat/Fedora and Mandrake. Both fine systems.. especally if all you do is use vendor packaged packages. Otherwise dependancy hell tends to like to rear its ugly head. At least it did for me far more often then I wanted. We actually still have a  Fedora Core 2 install at work on a nat/print/file server. Mostly cause it gets left alone

I gave Gentoo a shot, and while I totally respect and even admire the "compile it all yourself " approach it takes, the effort is not worth the results for me.  I even gave Slackware another shot. I can't stand its package managment, but by this time distro bloat was getting to be an issue for me. Gone were the days that I diddnt mind a thousand little programs not relivant to my needs on my PC  . I'm pretty much over that kind of exploring.. pretty much ;)

So Ubuntu was starting to get a buzz build around it. I let Warty pass and gave Hoary a shot. The ONLY problem I had was getting used to sudo instead  of using su. Extra attention to the desktop , APT, and up to date, biyearly releases, Gnome default.. wow...

My only real complaint is not being able to upgrade the to the next distro VIA CD ( yea, i know i can apt-get it, but i prefer haivng a disk in my hand ). Unless the direction of Ubuntu changes drastically, I don't forsee changing distros. Thanks devs. Thanks Mark.
