End of life for Ubuntu 10.10 (Maverick Meerkat)

Today I turned on my computers at home and at work to be presented with an alert telling me that the operating system on them, Ubuntu 10.10 “Maverick Meerkat”, has reached its end of life...

10.04, “Lucid Lynx”, was an LTS (long term support) release which used the old Gnome 2 interface. It’s still supported for another year, until April 2013.

10.10, “Maverick Meerkat” was a normal six-monthly release which used the Gnome 2 interface. Support has just expired for this version.

...Back in 2010 I went through my usual two upgrades...I had expected to upgrade to 11.04, but then Unity happened.

...That’s my real gripe. So please, Canonical, if you decide to make any more significant changes that may stop people upgrading by failing to work with their not-very-exotic hardware, at least do it in the version immediately following an LTS release.

### Post by viperdvman on 2012-06-03
Hey, I was able to get a Mac OS X style interface in GNOME Shell just like I had done in Ubuntu Classic (GNOME 2.32). It took using a number of GNOME Shell Extensions *(including some used in Linux Deepin)*, the AWN Dock *(or Cairo or Docky if you prefer)*, and some fitting GNOME Shell, Window, and GTK themes.

Granted, GNOME Shell and even GNOME Classic aren't as customizable as GNOME 2 was. And my Mac OS X example was just something I was able to do in GNOME Shell. Now for making a Windows-style interface, that might be quite a bit harder. Even the Mint GNOME Shell Extentions found in Linux Mint "Lisa" couldn't completely eliminate the top bar.

In all actuality, Ubuntu 11.04 is the last Ubuntu running GNOME 2 (just use the Ubuntu Classic session instead of Unity). But Natty is set to die this October. And then Lucid is set to die in April 2013, thus putting the final nail in Ubuntu's GNOME 2 coffin.

Really, the only distros left running GNOME 2 **that I have seen** are Fuduntu, Red Hat Enterprise Linux and other distros based directly on it (CentOS and Scientific Linux being two of them), and Debian "Squeeze" and most distros based directly on it (Snow Linux and SolusOS being two of them).

Yes, it's really a time of big changes to the most popular DE in Debian and Ubuntu. It sucks that something so good had to come to an end.
