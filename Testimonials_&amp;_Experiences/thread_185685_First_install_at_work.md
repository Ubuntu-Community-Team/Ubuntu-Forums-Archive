### Post by jethro10 on 2006-06-01
An old PC went tits up and popped a memory chip. Was too little memory left to run Windows, so I rebuilt it with Dapper and a lighter windows manager. Put on rdesktop and turned it into a thin client for the factory, we use a lot of thin clients for the shop floor.
So got the first one in via the back door. As its recovered an old PC, people are interested more than I thought. "Oh! that's that linux thingy then"

A good start from an I.T. person who only started on linux in march this year.
