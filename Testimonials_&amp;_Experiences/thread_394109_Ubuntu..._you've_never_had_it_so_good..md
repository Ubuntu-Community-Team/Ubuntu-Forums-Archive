### Post by RKCole on 2007-03-26
NicoleM.

I actually stumbled upon Ubuntu back in 2005.  I had always been afraid to try Linux as it was something new, but I am an adventurous type, so I took the jump with Ubuntu Breezy Badger (v5.10).  With each new version things became more and more better and amazing.  The reason why I went with Ubuntu is because of the community support and belief that Linux should be accessible.  They have screen readers, a screen magnifier, and much more for people with disabilities.  I did not see this much support and help with any other distro.  Sure...other distros like Fedora Core had the magnifier application installed, but it was never made known that "If you have a disability such as vision, we have software to help you."  This is one of the main reasons I chose and stuck with Ubuntu.

As far as advanced features...as far as I can see Ubuntu has more than enough things for the beginner, intermediate, and expert users/developers/businesses/etc.  The neat thing about Linux is that (typically) if something is not in the repositories, and you cannot find a Debian package (.deb) file to install a software program, you can utilize one of the strengths of the open source community and actually download the source code, compile it, and run the program.  It's as easy as finding the source code for your program of choice and compiling it using simple commands. (Note that you must have build-essential and necessary libraries installed, but that's as simple as typing in **sudo apt-get install build-essential**.  Not to mention that you can download the server edition if you are a business/Web developer and you can host your own Web sites using it.

It's all just amazing to me.

I hope I helped to answer your question.

Take care.
