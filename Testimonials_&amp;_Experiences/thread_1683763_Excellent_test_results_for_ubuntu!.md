WINDOWS XP TEST START

---------------------------------------------------

RESULTS

BOOT TIME: Ubuntu faster by 12.8 seconds

STARTING GOOGLE CHROME AND LOADING A SHORTCUT WEBPAGE FULLY:
WITH OS BOOT TIME: ubuntu 51.4 seconds faster
Without OS BOOT TIME: ubuntu 38.6 seconds faster

---------------------------------------------------------------
This test was performed on a core 2 duo proccessor, 4gb ram and 20gb/640gb free space on OS drive/HARD DRIVE computer.
This test includes GCSS which is a user preference in ubuntu, windows xp GCSS will increase login time and therefore slow the user down.
The reason for non-GCSS on windows xp is because u close it if u need it and it wastes time logging in.
The reason for GCSS on Ubuntu is because if u don't need it, it doesnt slow down login and can switch to different workspace and go back.
