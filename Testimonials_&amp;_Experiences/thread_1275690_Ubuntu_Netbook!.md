### Post by Beyecixramd on 2009-09-26
Hey there!

I've been in the Linux world for about a year ago, and a half since i deleted Windows completely and I'm using Linux as my main OS.

I started out with Guadalinex, which is a known spanish distro, i didn't know what Linux was, but all the informative crap in the cd, like "this cd is provided for you free of charge" and that stuff, really made me interested. I got it in a fair, don't ask where, because I don't remember. I used Linux Mint too, and Fedora, Debian, Mandriva, and some other distros.

But i never liked Ubuntu, because it was too popular, and i never liked it. I always thought "Switching from the most used OS to the most used distro is useless" and i still think that. I'm on Arch Linux right now. Anyway, the point is that Ubuntu has improved a lot. I've been testing out alphas and betas, and evenstable releases, on virtual machines. This last one, (I'm talking about the Ubuntu Netbook Remix Alpha 6) is great! I never imagined such a cool interface for a small device like a ntebook, good work.

But anyway, I would like to suggest a new default style in Firefox, just because the netbooks always have small screens, and space is the most important thing in these little devices. The more screen you have to read webs, the faster you can navigate.
