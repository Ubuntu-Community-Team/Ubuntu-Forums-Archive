### Post by perdition79 on 2007-11-24
You are doing a great thing, getting students to try Linux. You're trying to turn computers into tools again, rather than the mere Internet appliances that they've become in the past decade.  If I were a teacher, I'd do the exact same thing.  If I had an extra $20, I'd send you a gift card to buy some blank CD-R discs. (Sorry, but I'm flat broke.)

Back in '95, when I was in my high-school computer programming class, the teacher insisted upon an all unix lab.  We wrote programs for fortran and C, and used vi as the text editor.  We didn't even get to use X windows until the end of the school year.  The computer lab was made up of surplus IBM boxes from 1988, hooked up to a brand new $2500 server.  We all learned how to write code, and the whole lab was about the price of two new Windows boxes.

With the easy to use gnome desktop, today's students, in schools anywhere in the world, could use computer labs made up of 10-year-old computers with 256 megs of ram.  Most students are taught web browsing and Office-style applications these days, which are laughingly referred to as "computer skills." Open-source programs could encourage students to get back into playing with source code again; learn how to make computers work, rather than simply learning how to use them.
