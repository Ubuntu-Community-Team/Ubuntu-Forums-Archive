Great! Welcome to Ubuntu, and Ubuntu Forums. Don't worry; you'll get used to the Terminal in no time. If you have any problems, feel free to post them here.

Just some pointers for you:

If you are used to Arial, Verdana, Comic Sans Ms etc., there is a way to install them. Simply run this command in Terminal:

```
sudo apt-get install msttcorefonts
```Then, to use them straight away:

```
sudo fc-cache -fv
```That should work!

And, a shorter way to run a program, click "ALT+F2" and select the program from the list. This can be useful in some circumstances such as upgrading ubuntu etc.

Also, if you have Windows programs, you can still install them! Simply go to Applications -> Add/Remove and where you see at the top "Cannonical Sponsored applications" (or something like that!) change it to All available software. Then search for wine. You should see "Windows Compatibility layer" or something similar. Tick the Check, and click apply. Then, when installed, simply double-click an exe file and install it, or run it! Hope this isn't too complicated for you ;)

PS. To see if your windows application is supported by "WINE", check out [http://appdb.winehq.com](http://appdb.winehq.com) .

Hope you have an excellent experience with ubuntu!
