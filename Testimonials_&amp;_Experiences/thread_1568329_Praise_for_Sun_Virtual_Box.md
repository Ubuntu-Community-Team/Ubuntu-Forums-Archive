### Post by Tony Flury on 2010-09-05
I just wanted to recommend Sun/Oracles Virtual Box.

I now have a full Windows XP installation running as a guest inside a Ubuntu 10.04 host system, including USB, network and CD/ROM drive etc.

This windows XP installation was installed from scratch, and the only slight wobble was needing to create an ISO of the windows XP Disk as for some reason the windows installation process could not see the cd-rom.

My Windows XP guest system now has iTunes running completely successfully, so I can sync up my iPhone and iPod.

Next installation - my Sat Nav management application.

Well done Sun/Oracle - fantastic piece of kit.

### Post by KdotJ on 2010-09-05

It's great to be able to test stuff out like this, different configurations.
