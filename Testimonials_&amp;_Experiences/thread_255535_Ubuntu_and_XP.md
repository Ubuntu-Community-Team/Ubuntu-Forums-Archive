Yes, unfortunately I find I have to boot into Windows too when doing DVD authoring, but then I stupidly bought a video digitiser that only works with Windows anyway.

If you want to give DVD authoring a shot, I recommend trying DVDstyler. Just add this to your sources.list:

```
deb http://dvdstyler.sourceforge.net/repository/ubuntu/ dapper main
```

Then do an apt-get install dvdstyler.
