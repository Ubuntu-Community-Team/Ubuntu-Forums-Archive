If this isn't the right place to put this, someone please move it.

I've just deleted the entire Ubuntu partion of my HD (6GB on an 8GB Drive). I've reinstalled XP Pro, as there's no space for Vista even if my HD's R/W speed could keep up with doing the install in sometime less than 12 hours. So I won't be hanging around the forums for a little while. But that doesn't mean I'm leaving Linux forever. 

From the 2 weeks that I've used it , I like it alot. The open source concept is great, I love how helpful the community is, for basic use it worked as well as Windows (Though I hate KDE, Gnome FTW!). Compiz's graphical effects are VERY nice.

The only reason I uninstalled is just one word: "Games". All the 3D apps I try to run in WINE all fail miserably, keyboard doesnt work, screen goes white, stuff like that. But I'll be re-installing Ubuntu when I get my new HD to replace my fizzled Vista HD, I'm hoping to get 250GB and it's replacing an 80GB. So there will be ample room for Linux and Windows to run on the same computer. I'll keep using Linux for different stuff, like when I'm compiling the games I make, so I can support more than one OS.

I just wanted to thank the Ubuntu community for the great support I received while I was here, and to let you know I will be returning.
