### Post by mahy on 2006-11-02
I'd like to share my experience with user-friendliness of Linux. Just a coupla hours ago i beefed up my wifi security at home from WPA-TKIP to WPA-AES. I was doing it on a windows machine, because that was the only weak link in the chain (didn't support AES). Ok, so i configured it in the router's website and the connection dropped. With app. 5 mouseclicks i changed the settings to AES and it was up again.

And i thought to myself "wow, this was pretty good, i wonder how Linux will perform". To my utter astonishment, NO ADJUSTMENT WAS NECCESSARY! I just executed a script using wpasupplicant to connect, and it worked! Yipee! Long live Linux!
