### Post by JayKay3000 on 2009-07-27
Hello. I'll try not to write an essay but i have some wkd music on so hmm. :popcorn:

I just wanted to express how much I love Ubuntu with the gnome desktop environment and what changed my mind.

_Intro_

I'll admit I always thought linux in any form was rubbish, i'd used kde in uni and the version we had to use was all command line and to tell the truth I hate that, dont get me wrong, command line in linux is extremely powerful. I just like my gui.

Anyway. I was playing a game on windows and being blown away by how good a game mod was compared to the actual game and thinking that maybe open source is not that bad. I think the air of the poor mans choice is disappearing from my mind because there is so much open source in this day and age is pretty damn good.

So a few weeks later I tried openoffice.org still under windows and sent a long e-mail to a friend raving on how blown away I was and why i'd never buy ms office ever again and why no average joe ever need to buy ms office. So I was suddenly an open source convert? Well. My friend then put linux ubuntu with the gnome desktop on his laptop, I dismissed it as being rubbish till he brought his laptop over and i had a go and could not believe my eyes compared to types of things we were made to do on the kde environment (yes it had a gui but we were not allowed to touch it) and from what i've read up kde environment looks pretty good. But one step at a time.

OK. so a few months passed and I dunno. I think it was Friday I decided to download and boot ubuntu within windows using wubi. I used it, fell head over heals in love with it and Saturday decided to give it a popper install, sure i had some problems mainly being with the new style of boot loader then understanding ownership rights 'root owns this file' ...WHY! :) but i figured around that and got the boot loader sorted, I still had some remnants of vista kicking around which caused the problems + it did not cross my mind that ubuntu would install the linux boot loader.

_Why do I like it? Why should you like it?_

So it's free, but free is not really enough. Ubuntu with the gnome environment is great, it's so easy to use  as i'm a windows user. Things are in different places of course, but i'm used to firefox as i use that as my main windows browser and I am used to openoffice so I was soon getting used to the differences.

So far i've had zero hardware problems with any major components being reject and it happily found my n vidia gfx driver as soon as i'd fired it up. I played a movie, it found the codecs, I played an audio file and sure enough it found those too such as mp4a for example. So no need to download a codec pack like in windows. I was predicting nothing would work in ubuntu and that my pc would catch fire. 

There are lots of other things that make you think 'oh, thats actually been thought out for me, the user'

Application wise i've found software for basic movie editing, ubuntu comes with good graphics software and openoffice for writing documents and such, i can create html files, use mysql, run virtual machines, watch films, listen to music, brows the net and in effect do everything apart from play major video games. You might think i'd snuff it for that but the os overall is great. I like the look and feel and of course some of the 3d desktop effects are pretty neat and make using the os just a little more fun.

So maybe the applications are not quite as good as some of the applications you can pay for and use on windows. But when something overall this good is free it's pretty hard to make a massive complaints list.

The point is I can easily function in a non windows world. But i'm going to keep windows because I don't want to confine myself to any one os. But i'm writing this from the side of ubuntu and I only really load windows when I feel like gaming. 

_Conclusion_

In my perfect world bubble I would like to be able to walk into a computer store as an average joe, pick a pc I liked or laptop then be given 3 os choices, OSX(Mac), Windows and Linux. (those are the ones I see as being the 'big 3' but maybe others) I would then like the sales guy to try and give me the merits and pitfalls of all and for me to maybe go away and crunch the information and make up my own mind about the os I want to come with my new pc.

If you managed to read my essay then

THE END):P


Now if anyone has any tips or problems I should keep an eye out for using ubuntu with the gnome desktop. I've read that viruses are few and far between or any helpfull tips or comments then feel free to post away

### Post by Arthur_D on 2009-07-30
I can say that you're taking this the right way.
With that attitude, the command line will probably quickly become your friend. Not just because it's faster to do some tasks with it, but also because it gives you even more control over your system.
You might want to look at [http://www.pixelbeat.org/cmdline.html](http://www.pixelbeat.org/cmdline.html) for a list of commands. There are lots of commands I and probably you will never use, but some of them are incredibly valuable in certain situations, such as *sudo fdisk -l* (shows disk partition sizes and types).

Though, I have to warn you: any command starting with *rm*, *rmdir* or something along those lines you have to be 100% sure what it will do. *rm *and *rmdir *are used for deleting files and folders; run with caution.

Enjoy Ubuntu, and if you have questions, just do a Google search on the subject or post your questions in the forums. :)
