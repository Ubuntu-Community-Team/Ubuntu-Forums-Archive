### Post by selahlynch on 2008-12-09
I switched my personal laptop from Windows to Linux because I enjoy learning about computers and I wanted to try something new.  Also, I use Linux for some things at work, so I was inspired to learn more about Linux.

I installed Ubuntu 8.04 with a Gnome desktop on my Gateway MX3652 laptop.

I started out extremely frusterated by it... I came very close to switching back to Windows. 

Some things that frusterated me:
-it didn't automount my MP3 player
-it would not play a dvd... then when i installed codecs that allowed a dvd to play the default video player did not support menus, and had various bugs
-there was an annoying system beep that beeped very often
-something about my touchpad would cause the desktop to shift to a second desktop
-the sound didnt work!!
-bugs in the file browser, when resizing columns

I have since sorted most of these things out.  I learned that it is very important that I get comfortable searching in forums and that I get comfortable downloading and installing new software codecs and patches and other packages.

Ubuntu Linux is very cool in some ways.  I love that you are encouraged to be involved in the development.  I love how you can customize your setup in so many ways.  I'm amazed that all this was created and it is 100% free to use.

It was, however very frustrating to get started with. Its still frustrating how many bugs I run into with fundamental things like the file browser, or the video player.  Its also frustrating that I'm not clear on how to get involved reporting and fixing these bugs.
