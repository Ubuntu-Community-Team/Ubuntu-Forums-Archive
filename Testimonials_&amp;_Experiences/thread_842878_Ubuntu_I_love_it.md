### Post by Bastaard on 2008-06-27
After trying to switch many times, I'm finally 100% Ubuntu without even using a virtual machine for Windows XP, an OS that I still like a whole lot. Even though I've had many problems with hardware and software, after some months of using Ubuntu I'm pretty sure I'll be using it for a long long time. 

There's still many things lacking, many things not working as well as I would hope. But the ease of software install (even when I have to compile stuff myself), the availability of incredible amounts of great software, the configurability, the shinyness of themes and compiz, the great community, the freeness, seeing things improve on a daily basis, being part of the software idealism, it just makes me want to say:

I love Ubuntu.
