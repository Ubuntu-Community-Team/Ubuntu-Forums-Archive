[*]Windows power users are more likely to play Windows commercial games instead of (like Jane and Joe) playing console games. [*]Windows power users are more likely to install and troubleshoot Linux themselves instead of having a Linux-knowledgeable friend or family member install and troubleshoot Linux for them.[/list] None of these problem types have anything to do with Linux being difficult to use.
