### Post by dirkvic on 2007-12-15
I just wanted to thank the developers for making edubuntu. And the community for helping me set it up.

It was a hassle getting the display-drivers. I found these in this forum.

The wired network connection still doesn't work. But I got the wireless up and running using ndiswrapper. The included package didn't work out of the box. I had to manually remove it and build the source provided from ndiswrapper.org. But now it works :).

I have been using it for some days now and will probably favour it compared to windows. I think the main reason for this is the same as the people reverting back to windows keep mentioning; The ease of installing and removing programs. And once it is installed it just works. \\:D/

I set it up for my children aged 2 and 4. And the included educational software and the available games were a great success. My boy instantly got the hang of it.

Anyway, THANKYOU! =D>
