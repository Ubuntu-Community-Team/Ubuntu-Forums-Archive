### Post by Bedrock on 2008-10-10
Hi,

I am a complete newbie with Linux and I am just stumbling around trying things out just now. I have to say that my early impressions are very good indeed, I am well impressed. I used to keep a ubuntu disc lying around for times when my windows machine wouldn't let me do anything with viruses etc and was surprised when I came across an article on Zdnet ( [http://blogs.zdnet.com/hardware/?p=599](http://blogs.zdnet.com/hardware/?p=599) ) where it was claimed that setting up dual booting would be easy. I then couldn't resist the temptation to try it and frankly I was amazed at just how simple a procedure it was.

With the ubuntu installed, I ran into my first problem “where is the firewall hiding”? It wasn't in the Applications menu or any other menu that I could see and I finally found it lurking in the un-installed packages, after I found the package manager that is and a whole stack of other interesting functions were lurking in there too. A very concise and rounded set of applications are now at my disposal and I am very pleased with them too (even though I haven't a clue what some of them are for, it will be fun finding out). One problem I am finding so far is with viewing web pages with 360° QuickTime panoramas but that is a problem not confined to linux as I believe windows vista machines don't display them either, so guess that I will have to start using flash and java viewers to display my hobby stuff.

So, that's my early impressions, very easy to install even for the technically challenged like myself and a good set of pre-installed applications with more readily available. Perhaps a step by step walk through for the newbies would help a little i.e. if firestarter isn't in your apps, it will be in the package manager etc, Apart from that, I have no real problems to report so far, but I am sure I will get stuck at some point, just keep in mind I am not a techy person, so talk with wee short words and I will catch up eventually ;)

> **renzokuken said:**
> 
.....quick side note though. I think you'll find that a firewall IS installed by default, known as **iptables**. Firestarter is simply a GUI for iptables to allow configuration avoiding the command line.

ubuntu does NOT have a firewall. it is not needed in most circumstances.
from [here]("https://wiki.ubuntu.com/UbuntuFirewall"): "Ubuntu currently **does not** have an integrated firewall in its base installation. The tools that are available to create a firewall are largely based on GUI applications and/or designed for advanced users. Additionally, existing tools also do not provide package integration so that when a network daemon is installed, users have to determine on their own how to integrate the application with the firewall."

you learn something new everyday, huh?
