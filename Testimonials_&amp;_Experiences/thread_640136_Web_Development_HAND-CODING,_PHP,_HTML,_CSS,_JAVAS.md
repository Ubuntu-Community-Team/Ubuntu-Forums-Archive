### Post by p.herewini on 2007-12-13
I realise this subject may have been discussed before but I'm specifically targeting   other users who may also hand-code PHP, HTML, CSS and Javascript.

I recently converted from Windows to Ubuntu when it was in the Edgy stage if I remember correctly and sampled many different applications. I'm now using Gutsy. Really all I require for web development is a **text-editor which can syntax-highlight PHP and HTML in the same document** and an FTP program.

-- FTP --
Filezilla has always satisfied me for FTP, although I wasn't 100% happy with it until Gutsy showed certain bug fixes, now it works the same as it did in Windows.

-- Text editor --
I started sampling everything I could find but settled on Geany, for its ability to open files mounted with Samba and the ability to syntax-highlight HTML and PHP in the same document. They also added javascript syntax-highlighting in Gutsy. **The problem I now have is when I am typing rules in CSS and I hit [space] after typing [:] Geany inserts a [tab] instead of a [space].** I have tried different options but can't seem to stop it, and this problem only came about after upgrading to Gutsy.

e.g.
```

background-color:	#FFF;
/* a [tab] is entered ^ after colon when pressing [space] */

/* it should just enter a space like this: */
background-color: #FFF;

```

Very frustrating, because I have to back-space then space again to remove the tab and I write these rules all day. I like Geany the best, apart from this new problem.

Some other software I use which some people may find useful (all available from add/remove programs):
[LIST]
[*]Glipper (clipboard tool)
[*]Turkey (generates random filler text)
[*]Agave (generates colour-schemes)
[*]The Gimp (opensource photoshop alternative)
[*]KRuler (screen ruler)
[*]Gcolor2 (colour picker)
[/LIST]

### Post by Geekkit on 2008-08-07
Do 2 things:

1) In Geany, go to Edit -> Preferences -> Files Tab, click the checkbox labeled replace tabs by space
2) Whenever you tab, tab, tab, away within your document, save the document and immediately all tabs are replaced with spaces.

It's a bit of a quirk (tabs not replaced with spaces until you save) but maybe they call it a feature. ;)

Either way, it's my editor of choice for all coding. That and a handy shell script, ant build file, or make file.
