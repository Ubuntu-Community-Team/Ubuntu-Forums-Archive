### Post by mat2432 on 2013-02-17
Hi, 

I have been passing hours trying to install Ubuntu 12.10 on a very old PC as a second PC for my home. I have already used Ubuntu 10 on another PC for years without a glitch and was very impressed by Ubuntu OS. 

So for that other PC I thought installing the latest version of Ubuntu was a good idea. WRONG!

The biggest problem was with the graphic. Boot and everything turns black. Reinstall, re-update, now can see the log gizmo but after login black hole again. 

Then install NVidia package. Ok now no black hole, but the menu bar is nowhere to be found, so I cannot start a damn application (don't tell me you have put a charm stupid bar that cannot pop up half the time ala Windows.....).

So I decide to search around for my good old Ubuntu 10 installation CD. And boom, no hassle, no problem, no error, no crash, and in a jiff I'm ready to type this really deserved complain about how bad Ubuntu 12.10 seems to be.

Now note that it is not just for releasing some steam that posting this new thread, but also for helping other folks like me who might entice to try the latest version of this OS that proved to be excellent in the past and to let the developer know that they should not scrap the basic (working fine with NVIDIA graphic) just to follow the (smaller, less performant is better) latest trend.

Thanks, 

M
