### Post by solwic on 2008-06-18
I've been using Ubuntu on and off since Dapper Drake was still wearing diapers.  I understand why Ubuntu is free, why the fact that it is free matters, and how being free affects the end user (software issues, driver compatibility, more technical knowledge required of the end user, etc.).  But until today, I didn't truly understand how important it is that Ubuntu is (and stays) free.

A friend has a couple of old network servers sitting in his garage.  They're pretty old, but work, and if nothing else make for good, free cases to add new hardware to.  He offered them to me today, free of charge, just to make room in his garage for those other, not so important things like the car and lawn mower and such.  

Normally a free computer brings me mixed emotions.  Yes, it's free (woohoo!), but I've also got to find a way to make it run, which usually means chasing down an unused Windows key to get XP up and running.  But today when he told me I could have these towers, my very first thought was, "Hey...I can run Ubuntu Server on them...for nothing!  Yippee!"

I know this isn't a startling revelation to my fellow Ubuntu users, and I'm sure others have already experienced similar situations.  I just wanted to post this story so I could thank not just the developers of Ubuntu (thank you!), but also this wonderful community (thank you!).  I know that if I hit any snags setting up Ubuntu Server (something I've never done before) I'm only a few clicks away from either the solution (if it's already been posted) or help finding it.

You guys rock, Ubuntu rocks, and free, open-source software rocks.  

\m/ Ubuntu!  :)
