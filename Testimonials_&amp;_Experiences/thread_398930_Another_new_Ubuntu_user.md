### Post by spacedogg on 2007-04-01
I was curious about Linux last year so after an article in MaximumPC magazine about Ubuntu I figured I would give it a try. I installed 5.10 Breezy Badger and while it was easy to install and use, I could not get my wireless card to work, so I gave up and re-installed Windows. 

Flip forward a year, 

I'm taking a Unix/Linux class in college and low and behold, they suggest we use Ubuntu to complete our course work. Edgy Eft is much improved over Breezy Badger, but it's still super easy to use. I'm not a gnome fan, so I installed Kubuntu and I'm not looking back. I've only beed using it for 3 weeks, had to reinstall (which i found out later I didn't need to do as long as I make back-up copies of files that I change :) ) 4 or 5 times because of ATI driver issues, but I'm super impressed. I'll probably continue to use windows though, because all my games don't work with wine, but I'll definitely continue to use it on a regular basis.

A quick rundown of what I've learned to do so far:

Install packages through the command line and the package manager. (I like the package manager better)

Installed the newest ATI drivers for my X850XT PE, and got direct rendering working. It plays Warcraft3: The Frozen Throne beautifully. 

Installed audio and video codecs from the restricted repositories, which I had to enable. 

There's a lot that I know how to do now, but that's only a drop in the bucket compared to what can be done.

I do have a few complaints though:

1. Driver support for up to date hardware. It seems to lacking. I understand that it's not necessarily a Linux issue as some companies won't release the documentation needed to write new drivers. I had to remove my X-Fi sound card and use an old Audigy card that was in my kids computer to get sound to work. Also my ATI TV Wonder 650 has no driver support either. If I could get Myth TV or any other TV recording program to work with Ubuntu and my 650, I'd NEVER use Windows again.

2. Wireless support. Again, a driver issue and not a Linux issue per se, but still annoying. ndiswrapper does a good job filling in here, but I'd like it to "just work" :) .

Other than that, I'm happy to be using something new. It's fun, exciting, and gives me a new reason to use my computer.
