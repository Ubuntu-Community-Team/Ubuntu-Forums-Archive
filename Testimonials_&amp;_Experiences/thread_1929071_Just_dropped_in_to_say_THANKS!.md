### Post by Noobuntu Guy on 2012-02-21
I just finished installing Ubuntu 11.10 on my EeePC. WOW! The install went great. I had WiFi off the bat, my network printer (HP 6500 e709n) and network storage (WD MyBook World) were recognized and easy to install (The printer was easier than the Windows install). All my media is recognized and plays on the provided apps. New apps were easy to find and install. In short, this thing is great for anyone with an ounce of computer operations skills. 
 
I pulled down the Cairo Dock and installed it to show my wife the Mac OS X theme (She's an iAnything lover) but I went back to the standard dock because it seemed like the animation in the OS X theme was taking up too much of the EeePC's limited resources. On that note, Jupiter is a great app for the EeePCs.
 
Today I got my first chance to use the Ubuntu One storage system with my Android phone. All this is great.
 
So, no issues, I'm sure I'll have questions once I get through the basics. This thing is just fun to play with for now.
