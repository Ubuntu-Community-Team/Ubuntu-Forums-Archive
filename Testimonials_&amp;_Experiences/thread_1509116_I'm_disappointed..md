### Post by JKyleOKC on 2010-06-14
> **stalkingwolf said:**
> I have an old everex laptop that I can only run 8.04 on because that is the only one that i can manually configure the S3 graphics card in.

I havent been able to locate the spell to do it in anything since.You might try this: copy /etc/X11/xorg.conf to a flash drive. Note that the "X" is uppercase but everything else in the path is lowercase. Then try running a newer version from a live CD, getting a command-line login by pressing Ctrl-alt-F2, and copying xorg.conf back from the flash drive to /etc with the command "sudo cp /path/to/flash/drive/xorg.conf /etc/X11/xorg.conf" (you may have to do "sudo mkdir /etc/X11" first). This should retain the S3 configuration and you should then be able to do "sudo /etc/init.d/gdm start" to get a normal login screen.

If this works from the live CD, you can then update your 8.04 to any later version and repeat the drill.

The newer versions have eliminated the xorg.conf file and depend on other methods for automagically configuring the video, but the system will still respond to xorg.conf data if the file exists. I have no idea how long this will continue, though...
