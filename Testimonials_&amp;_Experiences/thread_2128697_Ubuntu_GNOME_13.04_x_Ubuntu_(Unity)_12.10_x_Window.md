### Post by H9D1uAehOP on 2013-03-23
[FONT=arial][SIZE=2][B]Ubuntu GNOME 13.04, my experiences
[/B]
It's still a daily build, but yeah, very stable and working well. For me, Ubuntu GNOME rocks and I'm loving this work. Everything I wanted = the Ubuntu power + the GNOME3 amazing interface. Everything working flawlessy and blazing fast. I tested UGR 12.10 too, but 13.04 is way faster.

Still no bugs, very good graphics performance and all my apps are working. GNOME 3 is so easy and intuitive, fast and beautiful. You find what you search in seconds, Empathy and Evolution integration are perfect and non-intrusive. 

Default icon pack is ok, native apps are fully optimized for GNOME 3 so it looks awesome. Chromium default theme doesn't look good, but with [GNOME 3 theme for Chromium]("https://chrome.google.com/webstore/detail/adwaita-gnome-3/oojbknijfmdmidgcgchmojbildmbdamm?hl=pt-BR") you'll get an unified design, beautiful like GNOME 3.

Five stars

[B]Ubuntu (Unity) 12.10, my experiences

[/B]Unity is beautiful, you can find everything with speed and no difficult. Works well but Ubuntu GNOME is faster, at least for me. Window organisation and Unity bar is fine, lens works good. 

For me, Unity is good. Just good, it's just not too beautiful like GNOME 3 and the speed isn't the same. I experienced some crashes with Unity that will be solved with Unity Next and Mir. When I was using Ubuntu GNOME then I come back to Ubuntu Unity, I was like "Where is GNOME 3?", because Unity isn't *catchy *like GNOME 3. And I think GNOME 3 is easier than Unity, when you use GNOME3 you understand the interface within minutes and in few minutes you already know how to launch apps, search and setup basic interface options like wallpaper. Unity isn't difficult but for me, it doesn't bring the best experience for an end-user. 

I used to love Unity but when I installed GNOME 3, I found my favourite desktop environment. I'm not saying that Unity sucks but for me, GNOME 3 is better. Both are good and Ubuntu should have GNOME 3 installed by default, with the option to choose between GNOME 3 and Unity at the first run.

Four stars.
[B]
Windows 8, my experiences

[/B]I'm not a Microsoft hater but Windows 8 is the weirdest operating system ever made. Metro apps? Sucks. They always run full screen, no integration with desktop apps. If you run Internet Explorer Metro and some desktop apps, you'll know what I'm talking about. Switching between these apps is awful.

That home screen, oh God... these tiles, they're so minimalist that makes they ugly. That mix between Metro and desktop icons. It makes me feel like I'm using a giant smartphone with Windows Phone. You easily get tired of that big Segoe UI Light font. 

WTF they made metro apps for Windows? Was it needed? It looks stupids, every single app running full screen, like a smartphone! And you can't use they side by side, because one of the two apps get very a very small space and you can't even adjust it!

And they're bringing the same experience to servers. When I heard it, I thought it was a joke. Seriously. Metro apps sucks on desktop, on server they're even worser.

Two stars. Because I'm forced to use it.

### Post by Roasted on 2013-03-26
Thanks for the insight. I'm a recent Gnome Shell adoptee after spending several weeks on each of the major desktop environments to see if there were any environments out there I would like more than Unity. I tried each for 2 weeks with an unbiased open mind and Gnome is what I ended up with. I have to agree, the Gnome interface is just downright beautiful. It's easy to use and easy to customize. Its level of customizability may not be as in depth as KDE, but Gnome does bring a level of polish that most other environments fall significantly short on.

Have you personally found a compelling reason to use Ubuntu Gnome over Ubuntu with Gnome 3 PPA? I have Ubuntu and Gnome 3 PPA installed because I feel like it gives me more of an overall package. That may be partially due to Ubuntu Gnome being relatively new, even though both Ubuntu and Gnome are far from new, I felt more confident using a regular Ubuntu install + Gnome 3 added on... at least with my work systems. Ubuntu Gnome I'd install at home without a second guess.

I do look forward to 13.04, and I also look forward to Gnome 3.8. I understand Gnome 3.8 is supposed to come out soon... one schedule said as early as tomorrow, but of course most things are released when they're ready, so maybe we'll have to see. :)

Glad you're liking Gnome 3 though. I wish more people would use it for more than 30 seconds before passing judgment. I find it very rare to find a user who dislikes Gnome 3 that used it for an extended period of time to get a real unbiased feel for it before coming to that conclusion. I was very vocal against Gnome 3's path but eventually I saw what they were doing and saw how capable it is. I'm considerably happier now that I'm using it full time both at home and at work.
