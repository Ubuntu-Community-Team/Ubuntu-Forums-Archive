### Post by Tamlynmac on 2010-12-13
> ventrical
I think it the rare exception to compare 10.10 Maveric with Windows  Vista. Maveric has worked awesomely for me on all my derelict PCs and.  as far as i am concerned, it is untouchable, by both Vista and 7!

The concept of comparing two totally different OS's on a system designed for only one, is questionable at best.

Comparisons, should be done under a controlled environment with specific  benchmarks and minimum interaction between the OS's being compared and  humans. True comparisons require systems that are designed specifically  for each OS. Those individuals who try Ubuntu and use their Windows  systems as a comparison, are in effect ignoring the fact that system  design is critical to OS functionality.

I've observed many a thread that complained about the need to replace a  system component when migrating to Ubuntu. This suggests (to me) that  some users don't understand the need for system compatibility and they  assume that all OS's should work with all systems. Additionally, since Ubuntu is not a  Windows clone and a learning curve is required - (IMHO) no comparison  should be attempted until an investment has been made in said curve.

It's been my observation, that many user comparisons imply that the  results are to be expected universally and that all users will share the  exact same experience. I'd recommend to those who make comparisons,  that they do so recognizing that others may not experience the same  results. Unless, they have the exact same system - setup and configured  identically. Even then, one must question opinions based on user  perceptions and expectations. For example: what one user might perceive  as a critical flaw, another may view as nothing but a slight annoyance  or having no impact.

Just my $0.02
