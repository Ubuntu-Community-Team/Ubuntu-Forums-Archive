### Post by Kolin S. Murray on 2006-11-30
Hi all,

I've been so fully enjoying all my learning about Linux and Ubuntu lately. I recently got an opportunity to to try it out on my old laptop - it was dying of a hardware problem and I wiped it clean of Microsloth and installed Ubuntu as a troubleshooting measure. Certainly my problem proved to be hardware.

My new laptop (a hand me down Dell 630m from my brother) will be arriving in a few weeks and I can't wait to pull XP completely off and install Ubuntu. 

We've had computers in our house as long as I can remember: Our first hard drive was a 20MB hardcard. I fondly remember playing ancient DOS games and knowing my way around the command line a bit. Then we got brainwashed so many years ago and I feel ashamed that I've been using Windows for so long. I'm glad that all you folks who made computers powerful, configurable, useful, and fun to use are all still around doing the same stuff.

I've wanted to post something in this forum for a while and I finally found the right area!

Go Linux!
