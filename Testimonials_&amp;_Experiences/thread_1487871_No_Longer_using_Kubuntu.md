I stopped using kubuntu 10.04 also.
I can't use an audio cd in a dual drive system (this is a known bug on bugs.kde.org).
One person having the same problem resorted to unpluggin one of his dvd drives in order to get an audio cd to load using dolphin.
I even tried suse 11.1 and can't load an audio cd in /dev/scd0 or get Amarok to play an audio cd.
It's clearly a problem with kde 4.x since gnome doesn't have this problem.
Dolphin will not load an audio cd in /dev/scd0 for me.
Amarok will not play an audio cd.
I can't continue burning multisession DVD-Rs using k3b.
With ubuntu 10.04. I am able to load an audio cd in both dvd drives and I can play an audio cd using rhythmbox.
I'm a kde girl so I can't use gnome when I've been a kde fan since 7.10.
I never had these problems in kde 3.x.
I'm burning kubuntu trinity 3.5x right now.
These same bugs still existing in previous versions of kde 4.x
Maybe I'll try back in about six months.
On another note, I install linux on old computers that I donate to kids.
I can't install kubuntu 10.04 if it doesn't work right.
I'll just install kubuntu using kde 3.x or puppy linux.
