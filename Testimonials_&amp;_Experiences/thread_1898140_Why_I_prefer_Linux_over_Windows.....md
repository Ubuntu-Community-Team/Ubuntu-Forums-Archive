### Post by wazupwiop on 2011-12-20
Hey Guys,

     I am writing this thread to sum up my thoughts about Ubuntu and Linux in general.  I highly recommend reading this if you are on the edge of taking the plunge into the Linux world.

     I will begin my thoughts stating my background.  I started using Ubuntu after 8.04 was released.  It was just an experiment.  I am 16 now, and I started using Ubuntu when I was 13 years old.  My dad works for Microsoft, so I have had the opportunity to try out many different Windows OS's over the last few years along with Linux OS's.

     Over the last 4 years, I have tried Puppy Linux, Linux Mint, CentOs, Ubuntu, Ubuntu Studio, Xubuntu, and a few some other os's that are not coming to mind atm.  I have been primarily a Ubuntu user for the last two years, but before switching to Ubuntu, I used Windows XP.

     Windows XP is a great operating system.  It is relatively stable now, and it is compatible with most software and hardware available.  One reason that I switched from XP to Ubuntu is that the processes in XP were hogging up more memory and processing power as the world's technology became more advanced.  One advantage that Windows OS's have in comparison to Windows is gaming capabilities, but gaming in Linux is not impossible.

     That said, Linux operating systems can do anything that Windows can do, it may just take a command or a button to get it to do what you want.  Linux is more customizable that Windows in my opinion.  You can customize your computer to be your average desktop, a web server, a media center, etc.  When you download most Linux OS's, it is like downloading a shopping cart.  You have what you need to get started with your shopping, but it is your responsibility to load it up with the things that you want.  With Windows, you get a half-loaded shopping cart, and you fill it up the rest of the way.

     To a new Linux user who is used to Windows, many things that are apparent to me, may not be so apparent to a new Linux user.  You basically have to relearn an OS.  This goes for many different Linux systems as well.  Red Hat based systems are different from Debian based ones.  Each OS has its advantages, but I like Xubuntu at the moment.

     With any Linux OS, you have to install codecs and restricted extras if you want to play certain audio files and DVD's.  I do this out of habit when I install any OS, but this may not be so apparent to you if you aren't used to Linux.  There is a learning curve, but after using Ubuntu for 3 years, I find that I like Ubuntu more than I ever liked XP.

     I have had my share of problems among the different OS's, but I have yet to find a problem that I could not fix.  All problems in Linux can be fixed by searching on the forums and talking to people with similar issues.  I have experienced less issues as Canonical has released updated versions of Ubuntu.

     One thing that I like about Linux in general is that everything is free.  You can pay the $100+ dollars to get MS office, or you can download Libre Office for Linux.  I use Linux as my everyday OS.  I do all of my school work on Linux, and I play flash online games, java games, etc.  I find that for most every-day applications, Linux out-performs Windows.

     That said, Windows does have the upper hand on Linux due to Netflix, modern gaming, and some specialized uses.  While Ubuntu can be used to watch Netflix, play WoW, and record and mix high-quality audio, some of these functions are harder in Linux than in Windows.  However, many companies are beginning to develop programs for Linux instead of only Windows.  In the next 5 years, I see Linux being supported much more than it is now, and I think Microsoft will have a tough challenge keeping users loyal to Windows as people begin seeking other options.

     Basically, for most every-day computing (text editing, surfing the web, simple recording), Linux wins out almost every time.  Windows may be better for you if you play a lot of games like TF2, WoW, etc. or use your computer for specialized functions like CAD editing or some vague job-related process.  Linux could probably run those functions through Wine, but I would recommend getting some help for that.  I prefer Linux for everyday use because I find it easy-to-use and very customizable to suit my needs and tastes.  I still dual-boot with Windows XP, but my Windows partition hasn't seen the light of day in weeks.

     I prefer Linux over Windows because it is free, easy-to-use (after you learn the OS, which isn't that hard), and Linux runs on older hardware, which is great for keeping computers running longer.  If you are trying to decide whether to take the plunge, try using the LiveCD.  It will run a slightly slower than the installed OS, but you can check for hardware compatibility and see if you like the OS or not.  It takes very little time to install the OS and updates (usually around 2 hours or so on my old laptop).  I can comfortably recommend using Linux to just about anyone.  It works well and runs great for me.  If you don't like an OS, you can always try out another one for free until you find what you like.

Sorry but like the previous poster, I have to disagree... Not only are video card drivers (esp. ATI/AMD) not as good as Windows, but DirectX is a far superior graphics API than OpenGL. When Doom 3 was released years ago, the opposite was true and DirectX was lagging behind, but a lot has changed in over 10 years.

If a developer had to port a game to Linux, they'd either have to use OpenGL sacrificing some quality or program it in both graphics APIs (in which case the Linux/Mac versions would have worse graphics than the Windows version). This is a lot of extra work which is the reason games aren't ported to Mac+ Linux any more as the market share is too small for that amount of work.

The reason Linux & Mac don't use DirectX is because Microsoft owns it(patent/copyright), so we're stuck with OpenGL, the main advantage of which is being open-source.

This isn't an issue for indie games because they don't use cutting edge graphics anyway which is why you can see Minecraft and Angry Birds on Iphones, Android, and Linux, but not high end commercial ones.
