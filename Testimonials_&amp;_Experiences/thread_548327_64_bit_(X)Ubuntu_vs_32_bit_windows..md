### Post by Jouke74 on 2007-09-11
Some experiences from a starting linux user in dual boot.

Reasons for me to go to linux were that I needed to run a 64 bit system to do some statistical analyses. After some distro hopping I stuck to (X)Ubuntu (can't really pick the best of the two). As I need windows for other statistical software I currently dual boot. I sometimes hop from Xubuntu to Ubuntu and vice versa. Here are some of the experiences putting Ubuntu linux and windows head to head. Is linux ready for the desktop? I think so.. Every point is getting + or - if one is better / worse, or = when equal. Now let's see:

**32bit/64 bit**
Getting 64 bit windows costs me a new Xp / Vista license, in linux I can choose freely to install any two, therefore linux + , windows -.

**OS installation**
If you know what to do, linux +, windows -.

**Hardware support**
With company drivers linux - windows +.
Without company drivers linux + windows -.
End result, as is on my system =

**System stability**
On bootup linux sometimes fails to start, windows always goes fine. However I fear that the start of failure in linux is related to windows making NTFS journal errors on my harddrive. Or, the FGLRX driver messes things up. For now linux = windows.

**General speed**
Ubuntu - = Windows Xp +.
Xubuntu + Windows Xp -.

**Program stability**
I run a limited number of programs in Linux and Windows and they crash rarely. Main offenders : SPSS under windows, Abiword under linux. With windows I need to reboot afterwards and send some stuff to MS (-), with linux I can basically continue (+). On the other hand from the programs which I use, the windows ones are generally a bit more stable and evolved and linux ones are often still improving / under developement. Thus windows = linux.

**Updates**
Ubuntu updates the whole OS twice a year and that takes time (-). 
(Yes I know I could have taken the LTS version, but that one failed to run). Windows XP is out of developmentl (-) hence it also does not take time. Windows security updates sort of matches the frequency of linux recommended updates and security updates. Both use quite a lot of bandwith, so again (=).  

**Security** 
Given what I have seen Windows(-) and Linux (+). If I surf a lot on the internet the risk of downloading malware is large in windows XP. 

**Multi Media**
If I would need to look for each codec under windows knowing what I know about Linux now, they are also equal =. Given that linux is not installing patented codecs by default and windows is, this comparison is not appropriate in my opinion. 

**Getting programs**
I do not have unlimited cash, thus linux + windows -...

**Support**
From the companies behind it: linux - and windows -
From the communities around it : linux + windows +

End result :
5 categories score equal, in the other 5 linux is better.

I think there will be a trend in Linux getting more and more market share, and this is proven by the point that Dell and probably Levono soon are going to sell laptops with Ubuntu preinstalled.

A big thank you to all free software developers.
