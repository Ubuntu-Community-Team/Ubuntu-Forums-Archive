### Post by malleus74 on 2008-11-05
I'll have to say that my Ubuntu experience has been up and down. I'm very impressed with how far it's come, though. 

What I'd do is use the newest live cd and see how everything works for you. If all your hardware works out of the box, then I'd consider doing a dual boot.

There's a few alterations you'll want to do after you install Ubuntu that'll make the experience easier for you. There's plenty of guides on the forum on how to do it so I'll just quickly overview why I did it:

1. Install medibuntu / the restricted software to get your multimedia working. I'd also install VLC because it has a very good gui and plays almost everything.

2. Add document templates to nautilus so you can 'right click' and create a blank documents for the basic file types.

3. Add scripts to nautilus to increase functionality. There's several good sites on the net with the scripts ready to unzip. In my view it's always better to have the ability even if you don't use it.

4. Add no script and adblock to Firefox to secure it. You can always temporarily allow /permanently allow any site you like. While you're at it browse youtube, etc, and make sure your flash is working... and try to open a pdf file in firefox and make sure that's ok. If not, install adobe flash and reader.

5. backup, backup, backup. You can at least copy your home directory so that you keep your settings.

6. BEWARE updates, especially **kernel ** updates. Backup before you allow any updates... see no#5 :)
