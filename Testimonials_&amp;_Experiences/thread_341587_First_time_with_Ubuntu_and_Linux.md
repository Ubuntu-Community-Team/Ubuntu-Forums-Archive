### Post by OneEyedMan on 2007-01-18
After 10+ years of being fed on only Microsoft Windows OS computers, I finally decided a while back to give this... linux-thingy... a try. I had heard horror stories of hardware conflicts, impenetrable technobabble, hostile fans, insanely difficult and arcane installations, so I was naturally rather skittish. After all, I did software and hardware repairs on Windows for a living, and the Windows OS had been literally hammered into my brain.

So with no small amount of nervousness, I gave Ubuntu a try.

Oh. My. God. It works! It all works! The network detected right away, the dual-boot worked like a breeze, the install walked me through the repartitioning, it was as easy as installing Windows XP! Maybe easier!

Here I had been worried about whether I was going to set the damn house on fire, and instead I was setting up Evolution and updating the system not 5 minutes after the install finished.

I'm far from being a skilled linux user. Installing programs is still very unnatural to me. the terminal still is more than a bit intimidating. But I'm not afraid of linux anymore. I've rotated my spare computer through Zenwalk, Freespire, Mandriva, Symphony, Elive, and Puppy, having a blast every time. And I never would have even thought of doign that if my first experience with linux hadn't been so welcoming and positive.

You guys have put something really remarkable together. You should be proud.:KS 

-J
