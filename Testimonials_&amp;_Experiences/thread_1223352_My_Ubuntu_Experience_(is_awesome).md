### Post by Dullstar on 2009-07-26
When I came to Ubuntu a few months ago, I was completely blown away.

It was awesome.  Ubuntu doesn't crash.  Windows does.
For some reason, I get a better wireless signal on Ubuntu than I do on Windows, if the number of bars can be trusted.

I like how easy it is to configure the clock in Ubuntu.  In Windows, anything other than changing the time and time zone takes a rocket scientist.  What if I like using 24 hour time just for the heck of it?  How do I enable it?  In Ubuntu, it took just like, a minute or two of experimenting to find the settings, but as for Windows, all I know about it is that it can be done.  Actually, now that I think of it, I can't remember what the default time setting with the installation was.  Maybe it was already set up that way.  Either way, I know how.

Has anyone noticed that in Windows, for the weather stuff in Windows you can install crap, live without it (which I generally do), or have crap install itself without your knowledge?  HATE WeatherBug.  If I wanted weather alerts and stuff like that, maybe I would have downloaded and installed WeatherBug.  However, WeatherBug must of drive-by downloaded or something, because I have no idea how it ever got on to the computer in the first place.  While I don't really care for weather statistics or stuff, if it can be enabled easily and be really nice, what the heck, I'll enable it.  Now I can't imaging the GNOME desktop environment without it easily at all.

Things just make sense in Ubuntu.  So far I have not yet found that "Awkward Thing" in Ubuntu, although I have found it in both Windows and Mac OS Classic (same awkward thing).  I will give Mac OS Classic credit for being older, but it seems weird to me that Shut Down is under "Special."  They changed that in recent versions, though.  In Windows, you have to go to start to shut down.  Weird.  Namewise, I like the file browser's name a lot better.  File Browser just tells you what it does.  Windows Explorer is not as descriptive.  By the way, if I know where I'm going in the file browser, than wouldn't I be browsing the files, not "exploring" them?

The terminal actually has a purpose in Ubuntu.  I like how it is usable, unlike in Windows, where all it does is use a CLI instead of a GUI.  Boring!  The terminal is easy to use in Ubuntu (don't know about other distributions, I've never used them), as well, and not completely needed except for a few things.  However, the average user will probably not need the terminal at all except maybe for compatibility issues.  In Ubuntu, the CLI (Terminal) can do advanced functions.  In Windows, the CLI just does the same thing as the file browser, without a GUI.  Do versions of Windows after XP even have a CLI?

I love how the System Monitor replaces the task manager by making it functional yet user-friendly, unlike Windows, which had a task manager that had two versions:  the user friendly but very buggy and cruddy version, and the not so user friendly and functional version.

How about customization?  I was able to make a theme on my own simply by combining elements from the other themes.  Easy!  Simple to save as well.  On Windows, you have to patch it to use a non-Microsoft theme.  Why?  I have no idea.  Also, if you want a login screen with an older interface, there is on obvious way to do it.  In Ubuntu, it's simple to change the login screen.

Boot times/shut down times:  a biggie, in my opinion.  I like how Ubuntu shows you startup/shutdown progress.  It's also faster.  Windows runs like a slug in both categories compared to Ubuntu.  Also, the "booting" screen looks a heck of a lot better than on Windows.  On Windows XP, the booting screen was ugly!  I'm not sure how much the quality compares on Ubuntu's, but if it's low quality just like on Windows, the problem was thwarted by using enough brains to make it small so it isn't easily noticeable.

Support:  Ubuntu wins.  These forums.  Windows = Microsoft Website = crappy support.

The most advanced word processor I know of that comes with Windows is WordPad, which isn't very good.  Either way, I'd download OpenOffice anyways, but still!  What's the point of the Micro$oft Office trail version on some installations (at least it came with the computer when I got it).

Eye Candy:  Ubuntu wins.  Better looking default font set, better effects available (does Windows even have effects?), desktop switcher without installing anything (for Windows its a "Powertoy" or something like that), wobbly windows, the option to install Compiz-config to enable effects like the Desktop Cube.

Been using Ubuntu for a few months...  and I'm loving it.
