### Post by marcele on 2007-10-30
The Good:
1. Printing support (with network printer discovery) much improved.
2. Restricted driver manager is sweet.
3. Eye candy is a nice touch.
4. NTFS read/write support rocks.

The Bad:
1. The installer hangs for over 5 minutes spitting out some cryptic error if you don't have a floppy drive installed.
2. The kernel still flashes some cryptic error that it can't access some memory portion on my laptop (I guess this is fixed in the  Linux Kernel v2.6.23 though ... it only took a year to fix :( )
3. BulletZProofX is far from BulletProof.
4. Firefox crashes all the time (on my laptop with Ati video card .. and on my desktop with nvidia 6600 video card (both with restricted drivers installed)).
5. Had many hard locks only fixable with the Reset button on my computer).
6. Firefox seems sluggish (I don't think that compiz and the scroll wheel  like each other)
7. Wireless on my broadcom barely works and drops out all the time (with restricted driver installed).
8. There needs to be a better review process for getting applications in the Ubuntu repository. There is a lot of different software in the repos that doesn't even work after its installed. (If it doesn't work .. then it shouldn't be in the repos!)

The 6 month release schedule is totally unrealistic for software as complex as an operating system. I've tried all the Ubuntu releases Gutsy seems to be wildly unstable.

I'm totally fine with having to scour the forums and apply fixes, workarounds etc. (I'm a linux sysadmin .. I love the console) .. but even after searching trying to fix everything .. its still not even half as stable as Vista .. :(

The typical response is well "Its free don't bitch" .. or "Report all the bugs in the bug tracker" or "Try another OS". There are already thousands of bug reports .. (some as old as a few years) ... There needs to be a fundamental change in the development process or Ubuntu is never going to make it.  Eye candy is nice .. but I need stability to actually get my work done.
