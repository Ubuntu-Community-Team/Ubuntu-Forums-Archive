Hi!

This is my first post, I've been reading the boards for a long time and I've managed to scrape by when having some problems doing just that.

However this time I felt I should jump in the discussion. 

Too much of my favorite software has been going the same "blingy-bloated" route these last years and I'm finding myself repeatedly having to keep looking for substitutes.

I'll keep things short, no one likes walls of text, so I won't get into many details just those I consider most relevant. This isn't a rant but more a vent to the Ubuntu developers from a long time user of their distro, now already an ex-user.

Unity sucks and Ubuntu is getting bloated with useless stuff. I do seriously know you want to deliver an operating system which computer illiterate people can use but than can only go to some extent. If doing that you ruin the experience for a great part of your user base that should really tell you how bad of a move it was. 

I don't want to go on a treasure hunt every time I need an application, I don't want to have a desktop tablet PC look-alike. I've got a desktop PC I want a menu, I want things organized. Not all thrown into some tab.

I also know that I can revert to the classic look but that is just for a limited time. 10.04 also is going to "soon" be dropped so having considered all options I've looked for other distros. Nor I want to waste time having to install other DEs after a full upgrade. Things should just be fine out of the box.

So long Ubuntu. It has been a nice run, with a sad goodbye.
