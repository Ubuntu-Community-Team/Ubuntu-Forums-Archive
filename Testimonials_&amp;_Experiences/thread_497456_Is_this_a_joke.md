### Post by ianfav on 2007-07-10
Well I thought I would try the latest and greatest Ubuntu 7.04 seeing you linux people are always going on about it.  Windows is this windows is that, Microsoft is that blah blah blah...
At least Microsoft have the brains to support a couple of the oldest interfaces to a computer namely a keyboard and a mouse.  Imagine my surprise when I booted from the Ubuntu CD and it did not even pickup my keyboard or mouse even though the initial start boot menu did.
Honestly how can you call it an viable OS when it doesn't even support keyboards or mice which have been around for years (USB wireless).
Really it can't be that hard, why bother offering a desktop OS if it doesn't even support the simplest of hardware.  You cannot expect for users to take this operating system seriously when there are problems such as this.  Rather than bagging other OS'es (that by the way work) maybe the people who put these linux offerings together should ensure they work easily for the end user without them needing to know the OS intimately to get it working before offering it as a viable alternative to Windows...
I actully thought it was a joke has nobody would be stupid enough to release a OS that doesn't support a keyboard and mouse imagine my surprise when I checked the forum only to discover that this is a common occurence.
Sorry if this is strongly worded but honestly......!!
