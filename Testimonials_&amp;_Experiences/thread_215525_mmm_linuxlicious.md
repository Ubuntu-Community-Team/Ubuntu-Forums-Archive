### Post by NonaWeisbaum on 2006-07-14
I stumbled upon (literally, using the FireFox extension) [this page]("http://www.getgnulinux.org/switch_to_linux/linux_from_windows/") and decided to download the cd and check it out. I've wanted to try Linux for a long time now but I've been kinda put off by the partitioning and whatnot so the running the OS from the cd before installing it was pretty rad. I'd read a few pages about how Linux isn't Windows and compatability can be an issue with newer hardware. I built this machine in April with the newest hardware I could afford, so I figured upon start up everything would be all buggered up.
Oh how wonderfully wrong I was. I am not sure that it does actually recognize everything but it did run from the cd beautifully and without incident. After a few minutes of using it I decided to install it on my smaller HD.
So far, so good. 
In your face Windows monopoly.. in your face.
I am really excited about this.
I can get Bluefish now!! A few weeks ago I was searching for a free html editor. I came across Bluefish and decided that I must have it. I went to the download page and there was no version available for Windows. I had to laugh. I thought to myself "This is how Linux users must feel."
