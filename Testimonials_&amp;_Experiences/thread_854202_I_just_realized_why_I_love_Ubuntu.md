### Post by estyles on 2008-07-09
My biggest annoyance with Windows recently was the lack of a sudo command.  But I found one.  It's not nearly as easy, but it exists - it's just hidden.

For sudo nautilus - Start->Run 'runas /user:administrator "explorer c:"'
For a root terminal - Start->Run 'runas /user:administrator cmd'

Now if only there was a Windows command for "don't get viruses" minus symantec/whatever.
