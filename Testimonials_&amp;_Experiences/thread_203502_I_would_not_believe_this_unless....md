### Post by rtcary on 2006-06-25
I would believe this unless it happened to me:

I run Linux on a couple of home servers (Red Hat), but I wanted to see what the desktop version of Ubuntu looks like.  My software development business is in Windows and my wife uses a Mac notebook, so desktop Linux is new to me.

To try it out, I stacked the deck pretty high against Ubuntu: I have an old Toshiba Win98 notebook that my grand-daughter uses when she visits and it has PCMCIA cards for Ethernet and Wireless.  I downloaded the Desktop 6.06 and tried it out before doing an install; I was impressed.  The look and feel were like the Mac; no longer the appearance of the GUI being an after thought.

When I tried the install several times it hung part way through,s so I downloaded the "Alternate" version.  The install went so smoothly it is hard to believe!  The wireless and Ethernet worked immediately.

Yes, the Desk Top install needs some work, but everything is there!

What is needed is some generic hardware (e.g. notebooks and desktops) that is reasonably priced and is *not* junk so one can install Ubuntu and have a good general purpose computer.
