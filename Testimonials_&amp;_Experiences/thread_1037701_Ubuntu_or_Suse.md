### Post by jbysmith on 2009-01-12
I've never had much luck with EnvyNG myself. I used to use it back in the Feisty Fawn area, but once Ubuntu got into the 8's, it's never worked properly for me.  Not sure if user error or just my hardware.

I can't comment on the ATI procedure, but nVidia's is pretty simple.  Need to hit their site and download the driver of course; might need some packages as well.  (I compiled my own kernel so didn't need it the last time, but I *think* you need the kernel headers if I recall, along with LibC6-Dev and a few others.. don't quote me.)

The procedure itself was easy.  But like I said, this is nVidia, so check ATI for the appropriate instructions.

First, I get into a console via Ctrl-Alt-F1. (Not a terminal via your desktop, as the next command is..)

Kill GDM, the Gnome display manager.   sudo /etc/init.d/gdm stop

If KDE, you'll want to kill kdm instead.  Same difference. XFCE and the like, /shrug I've no idea.

Then, just for laughs, I make a copy of my configuration.  sudo cp /etc/X11/xorg.conf /home/jensmith/xorg.conf.backup, as an example.  If things explode, I have this to fall back on.

After that, run the installer itself.   sudo sh ./NVidia...... the filename will vary.

The installer asks if you want it to modify your X config, I say yes.

Sudo reboot, enjoy Compiz in all its 3D goodness. (Supposedly you can just do a gdm restart, but meh old Winders habits die hard I suppose.. I prefer a clean reboot just to be sure.)

If the driver has a problem, Ubuntu will pick up on that and at worst, get you into a graphical failsafe mode of sorts so you can at least figure out what the issue is.  (Worst case, copy back the X config.)

That's pretty much it; I do take a few extra steps to blacklist the default nForce2 GART driver as the one with the display driver works better, also enable sideband addressing and all that stuff for every last drop of performance, but that's purely optional, and realistically probably doesn't make much of a difference unless you're the type who likes to kick up the CPU voltage a half percent just because you can, etc.
