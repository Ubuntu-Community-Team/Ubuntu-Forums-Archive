### Post by 3rdalbum on 2007-12-14
Why not virtualise Ubuntu? Use Virtualbox (it's free) or something similar to create a virtual Ubuntu drive on your desktop, and run Ubuntu in a window on Windows.

Virtualbox will automatically set up the required stuff so that you can access the internet using the connection Windows is using. Ubuntu will think that it's accessing the internet through Ethernet, but in actual fact the other end of the connection is Virtualbox, and Virtualbox is passing the internet requests through to your real USB-based modem.

USB host support is also available on Virtualbox, so you can try once-and-for-all to get your printer running, and if not you can at least save to a shared directory and then print through Windows.

Sure, it's not quite the same as running directly on the hardware, but it would be a good interim step for you, and it won't mess up anything on your Windows system. Check out [www.virtualbox.org](www.virtualbox.org).
