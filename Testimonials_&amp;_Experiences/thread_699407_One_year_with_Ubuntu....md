### Post by Flymo on 2008-02-19
Greetings,

We could not agree more, and by a coincidence, this is coming to you from an ACER Aspire 4315 (as mentioned above) purchased  in Bendigo, VIC from "The Good Guys" for $481 after $89 rebate and $8 cash discount....:)

It is very good indeed, but not flawless.

**Good points:**
Faster than expected, very agile indeed so long as we don't open a ton of windows at once.  The RAM is only 512M, shared with the graphics so that's reasonable.  (Plan is to add a 1G stick)
OpenGL also works better than expected.  The 3D Linux games rock, mostly.  Planet Penguin Racer is excellent, as are the Tron and Marble games, and the stunning Amoeba 3D demo works fine. GL-117 flight sim is good to about stage 2 graphics - mostly at about 35-40 fps,  it will run higher stages at lower fps.
X-Plane won't fly, but Plane-Maker is first rate.  We only have half the minimum required RAM for X-Plane, so no surprises there.

**Less good points:**
There's a reboot problem in our batch - all the units tested at our dealer had it.  They boot just fine, but hang on reboot.  Acer are allegedly working on this.... <sigh> 
TV-Out is dead.  Struggling to find a solution in the mire of the intel X3100 driver/configuration issues. It recognises that S-Video is connected in the logfile, but pretty much nothing else.
When on battery power the WiFi needs a reboot to work again if the system suspends it - bound to be a command-line fix for this.
The hard drive is not the fastest, but it manages well so long as you don't fill up the RAM and start using the swap too much.

**Other info:**
We shrank the boot partition and installed a second Ubuntu GG 7.10 in a new partition and it too rocks!  Only thing we spotted that did not work just as well as the pre-installed version was the Atheros  WiFi (not found).  The pre-installed Ubuntu uses ndiswrapper to good effect.  
Then we upgraded the second installation to UbuntuStudio. Gradually getting it working right, and learning the** many** packages.  Real-Time Linux!  Whooo.....

**Conclusion:**
Excellent value if you can live with the promise of a fix for the TV-Out and the reboot thing. The latter cannot be any more than a snafu at Acer, since (guess what?) the GG installation that we did lacks a reboot problem.   But can we find anyone educated at Acer who will understand the implication of this? Sadly not.  ](*,)

A pity, since it's a cracking little beast, and potentially hugely successful for Acer and Ubuntu.

All the best, Ben

**PS** there's another boot anomaly - as supplied it first tries to do a PXE  boot over a network because the BIOS boot order has it top of the list.  Takes a while to give up and continue normally, so we moved it down the list - boots MUCH faster now!:guitar:
