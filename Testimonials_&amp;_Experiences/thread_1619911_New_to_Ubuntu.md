### Post by sambooka on 2010-11-12
Hello All,

I have been using Linux for a number of years, but never used Ubuntu. I cut my teeth on Fedora and Mandriva long ago, switched to OpenSuse and ran that for a couple years I guess.

About a month ago I decided to try Ubuntu ( 10.10 x64 ) on my laptop ( dual Win7 and Linux) and wow! My wireless was up all on its own; it was waiting for passkey on first login. No compiling and installing drivers. And the Additional Drivers app loaded the latest Nvidia driver for my card, desktop effects were up and I was good-to-go! Point is I was very impressed out of the box. So much so I loaded it up on my PC as well. That went equally smooth.

I did switch from KDE to Gnome this go around. Running the Cairo-Dock on the bottom and Gnome panel at the top. Nice set up.

Anyway, I just wanted to say hello and voice my experience to date with Ubuntu. All grins!

Sambooka

### Post by 3Miro on 2010-11-12
Wireless drivers were a nice change for 10.10, in earlier versions you had to use the additional drivers like for the Nvidia card. Ubuntu is much nicer with respect to all the things that it does for you automatically. Also, don't get too comfortable with the current Gnome setup, it will change dramatically in 11.04. You are catching Gnome right when it is about to make a jump from Gnome 2 to Gnome 3, if you are KDE user you will remember KDE 3.5 -> 4.0.
