### Post by lavinog on 2010-10-14
Hi all,
I have been out of the loop with keeping up with Maverick other than keeping my KVM install up to date.

I normally wait a week or two before upgrading to the newest release, but I decided that to do a fresh install on my son's computer since it was still running Karmic.

The new installer is great.  I normally use the alternate install cd to avoid the overhead of loading the live environment, but I figured I should give it a try, and man, am I impressed on how smooth the install went.

The first thing I noticed was that the cd didn't seek as much.  The optical drive was a 10 year old cdrw drive, so I was expecting a much longer load time.  In the past, the live cds would seem to read from random parts of the disk.  I am not sure if ureadahead had anything to do with this or not, but good job.

The installer itself was a great improvement.  Allowing me to answer all of the install questions while the system simultaneously partitioned the drive and install the core files had me really impressed.
