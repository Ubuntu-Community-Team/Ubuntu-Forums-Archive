### Post by joebodo on 2009-07-27
Most of the applications are for my job as a developer.

Some of the key applications have minor to major issues. But, as an MS hater - I'm glad that I finally was able to make the full-time switch.

The good
**1. Firefox**
Initially had some flash issues due to making sun java the default alternative - but that seems to have worked itself out.
**2. Eclipse**
Made the mistake of starting with the repos edition and trying to upgrade - went horribly wrong. Installed from an eclipse download and it's solid.
**3. Prism** - flawless
**4. VM Player**

The bad
**5. Skype**
Audio cuts out every 10 minutes or so - requires a restart. I stay on Skype normally 4-8 hours a day. I'm hoping google voice can replace this app. Really sad that Skype has dropped the ball on linux support.
**6. Miro**
Stops working every 30 minutes or so. Several processes must be manually killed and restarted. This app seems to get worse with every release.
**7. Remote Desktop**
Unable to drag the window around for some odd reason - need to focus another window then it's all good again. If minimized for some period of time, the application stops responding and must be -9 killed.
**8. VPN**
Some damn secureclient thing that is windows only - so need to run on a VM and use lots of tunnels.
**9. Webex**
This is a pretty big one - there's some claims that this works out-of-the box - but not for me. I did have it working in 8.04 - but I think I was using a 32 bit kernel at the time.

Extras
**Multi-monitor** using my new nVidia card is great. Better than MS.
**Compiz** - works great on the nVidia - not so great on my replaced ATI.
**Copy and paste** - you never know what your going to get. -- frustrating
