### Post by doowgof on 2007-11-24
I wiped off windows completely and joined the family of Ubuntu. When I first got Ubuntu installed I realised I can use wireless straight away without any setting. OK, fair enough, I thought. Then after some time I needed to do some printing, so I plugged the USB cable of my HP 1010 laser pinter into my toshiba laptop, and after about 5 secs it can just print! You know both XP and Vista wouldn't recognise it without installing the driver provided by HP? Fantastic! And then the most impressive bit comes, just 5 mins ago I turned off the light cos my girlfriend was asleep, I thought the light from the screen was too bright therefore I'd like to turn it down. And I pressed the key Fn (function key) and F6 (it's the key combination of toshiba to turn the screen brightness down), and it just worked!!! You know if I don't install the particular software from toshiba Windows will not recognise this key combination at all??!! I was SHOCKED!

### Post by gadgetgirl02 on 2007-11-25
My favourite things about Ubuntu are:

* When the OS installs, it also installs all the applications just about anyone would need, like the OpenOffice suite. I especially like that it does this in less time than it takes Windows just to install the OS alone.

* The Add/Remove Software utility. It's nice having a vetted, wide-ranging list of things I might use that can get installed on my distro. No, it doesn't cover everything, but it gets you a long way there -- certainly enough to please any general application user.
