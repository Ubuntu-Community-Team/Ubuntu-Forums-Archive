### Post by 13blue on 2008-06-28
I understand your pain. The reason linux seems so difficult to install is that it can be all things to all people. Ubuntu is special because it aims to take out a lot of the frustration by limiting a number of choices. 

I think your frustration may be a stemming from a skewed point of view.

1. Download and run a Knoppix CD and look at all the software choices. If you were installing all of that software to use, you'd have to configure it as well. Ubuntu limits those by giving one choice per function. If you'd like to use another application you can install it.

2. Today's Linux distributions are so much more streamlined than ever before. This can be seen by installing a older version by going through text mode installation, WOW is it hard to figure out how to partition a drive without a gui representing it.

3. Even after installing Windows, if you don't have a standard system you have to install the Graphics Drivers, Network Card, and peripherals you might have, like printer, cameras, and scanners. And I doubt you'll get a Microsoft employee to answer the phone within 10 minutes. Which, if you had been less insulting and more detailed you'd probably have an answer by now like the commenter above is trying to help with.

4. All these distributions are FREE. If you want someone else to make your choices for you then pay them $125 to $600. If you want to do it free then you'll have to pay for it with time.
Like the old sign says "Cheap, Good, Fast; pick 2" If you want it cheap and good then it is going to take time.

Thank you for listening, I used to be like you until I became appreciative of the work these folks were doing for me.
