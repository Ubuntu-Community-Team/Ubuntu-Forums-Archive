### Post by centipod on 2013-02-25
**Greetings!**
 
 I have for some time wanted to do a blog post about my Xubuntu experience and what I did after install but since I don't have a blog I will post it here, hoping my testimonial will reach the masses!

First of all I should tell you about my computers. One stationary on which I mostly use Windows because I need Microsoft office for work and Lightroom for photo editing. The other is a HP Mini 210 (2 GB RAM) from 2011 with Xubuntu 12.10 and Windows 7 Starter. After that I spent the first year changing my linux distro more often than my underwear.;) Among the distros that I actually used for a while was Ubuntu, Mint, Pinguy and also Xubuntu.
 
 Let me make a short list over the things I was looking for in the distros:
 -Lightweight and snappy
 -Smart interface, I like some kind of expo feature
 -Low screen estate usage
 -After a while I also discovered that I like having a taskbar
 
 
When I had been changing back and forth between the distros I found that I liked Xubuntu 12.04 the best even though it didn't tick all the boxes (I'll get back to a workaround on this). Ubuntu and Pinguy wasn't snappy enough. Mint LXDE was OK, but a bit buggy in the notification bar and I didn't like that Firefox was Mint flavored and not vanilla.
 
 **So now to my current Xubuntu 12.10 setup and what I did after installing.**
 
 **Synapse** &#8211; Try it!
 This great little program lets you launch programs with lightning speed by simply hitting a shortcut (ctrl+space) and start typing the name. Often you get the program you are looking for suggested after only two letters. Just to hit enter to launch!
[[IMG]http://s17.postimage.org/m9zitmnyj/Synapse.jpg[/IMG]]("http://postimage.org/image/m9zitmnyj/")

 **Skippy-xd** &#8211; Expose/expo mode in XFCE
  Install this little program and bind a keyboard shortcut to it and get a zoomed out view over all open unminimized windows and click on the one you want to activate! I also added a launcher to this program in the launcher panel at the very top. That way I can still reach it just using the touch pad in a layed back lazy browsing position. [COLOR=#1155cc][FONT=Arial][SIZE=3]_[http://code.google.com/p/skippy-xd/downloads/list](http://code.google.com/p/skippy-xd/downloads/list)_[/SIZE][/FONT][/COLOR]
[[IMG]http://s10.postimage.org/k0qfb4bqd/skippy.jpg[/IMG]]("http://postimage.org/image/k0qfb4bqd/")

 **Move****d**** top panel to bottom ****and launcher panel to left side**
 Though I kinda like the look of having the panel at the top it gets a bit cluttered when using Firefox whose tabs also are at the top.
[[IMG]http://s1.postimage.org/7w6xj3zor/desktop.jpg[/IMG]]("http://postimage.org/image/7w6xj3zor/")

 **Install Firefox addon 'Hide Caption Titlebar Plus**
 I like the way Firefox looks in Ubuntu when the titlebar is hidden. Somehow this doesn't happen in Xubuntu but this addon makes it so. Frees up some screen estate when surfing. Sure you can hit F11 to surf full screen but  I find the tabs, address bar and searchbar are not as easy to reach then so I usually only use it when I'm reading a longer text.
[[IMG]http://s10.postimage.org/nwukfmhyd/firefox.jpg[/IMG]]("http://postimage.org/image/nwukfmhyd/")

 **Plank** &#8211; lightweight dock
 I had this in 12.04 but I mostly used it as a launcher so now I use the standard launcher panel. But if you are a fan of docks, give it a try!
 
**Jupiter** &#8211; For power saving
 I notice about 30-60 minutes extra battery life with this application (as reported when hovering the battery indicator). One annoying thing is that it has popups that don't have a time out. The stay until you click on them. They can be disabled by with root acces rename or remove the file '/usr/lib/jupiter/scripts/notify' (a tip from somewhere online)
 
**Libre Office**
 I rather use libre office than abiword so I installed it and removed the latter
 
**Gimp 2.8** &#8211; Single window interface is great!
 I really like the new Gimp because of the single window interface. You have to enable it manually in the Windows menu though.
 
**Pinta** &#8211; Keep it simple!
 I have often found Gimp too advanced and comlicated for simple actions like cropping a picture and resizing it when making a presentation. Pinta does the trick!
 
**Spotify**
 Can be installed by following the instructions on this page: [https://www.spotify.com/se/download/previews/](https://www.spotify.com/se/download/previews/)
 
**Clementine**
 I tried a lot of media players and found Clementine to be my flavor!
 
**Other** programs I like to have:
 VLC
 Skype
 

 **So whats left to fix?**
 I have recently been experimenting with themes and mouse cursors. I like dark themes but find that they don't always work very nice in different applications.  And custom mouse cursors only work in some applications. In the panels you get the default cursor. So now I have settled for Blackbird theme in Window Manager and graybird in Appearance and a ridiculously large orange mouse pointer for Firefox :) (from xfce-look.org) I would also like to be able to use my computers Fn media keys to control media players. Maybe if I need it enough I'll have another look at the workarounds people have posted.
 
So thats all folks! Hope you found it interesting reading my post and that I managed to give some of you a few tips! Finally I must thank all the people developing this solid distribution and the people who make those small very useful applications that you always wanted!
 
 All the best to you!
 centipod
