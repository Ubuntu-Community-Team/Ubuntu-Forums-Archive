### Post by Unterseeboot_234 on 2006-11-26
Just a follow-up after using this new machine. It is my first factory-built machine in a long time. Made in Germany -- PRO-Tec, I've got a gut feeling they configured optionally, which I would eventually have accomplished had I made a built my own white box. I was reluctant on their choice of nVidia PCIe graphics card. After all, that card anticipates future software to manipulate HD video and that software is still vaporware.

Automatrix on 64-bit has a lot fewer choices, but it discovered the nVidia and installed an updated control panel. Before that, I got locked-up whenever the machine went into screen-saver. That works fine now, however, I don't have my favorite screensaver -- the Matrix with the figures in the background. Shame, that screen made visitors go "whoa!".

There doesn't appear to be any choice about the screen's color-depth or refresh rate. Screen res set at 1060x900 barely gives an extra 4 cm of desktop. That puzzles me. I remember Unix desktops from school that were so tall, they had scrollbars and navigation windows.

Evidentially, there are hidden files that get cached. The first 10 minutes after the computer was all updated, the mouse was bringing up blank window directories BUT it never did a M$Windows lockdown.

So, this machine takes my breath away. Loads FreeDOS off the 2gig partition in less than 1 second. Loads ubuntu in less than a minute. Really, the DSL-modem is the slowest part. 

I've got to learn about 32-bit emulation, most importantly because that is the only way to get some .pdf docs off the web (plug-in for the browser). That can wait. I like java and was surprised there is no plug-in for the 64-bit applets in a browser.

Thanks, ubuntu. The best part is this forum. Real human data for humans.
