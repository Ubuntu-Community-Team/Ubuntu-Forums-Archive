### Post by MontyMan on 2011-09-27
After dabbling in Redhat / Fedora / CentOS on and off for five years now, I have discovered Ubuntu and am having amazing success with it!  A few details:

I work mostly with Windows just because that's mostly what my clients use.  But two weeks ago, I found an Ubuntu installation disk and decided to try it live.  I was so impressed with its performance on my home computer that I decided to clear out some space and install it multi-boot style.  That machine is a Compaq about 5 years old with AMD and about 1GB.

The thing that impresses me the most about a distro is if I can install and just hit the ground running.  Very little changes required to have basic office capabilities, browsing, and other utilities.  And what decisions I do have to make during the install, and subsequent package management, should be very simple.  Although I am a programmer and I'm capable of using the command line, I want to see Linux distros mature to the point that they can be managed easily by normal individuals.  I am very pleased with Ubuntu!

Of course, as most users would, I had the Ubuntu installation repartition my drive, reducing the size of the XP partition which houses precious data and  is not sufficiently backed up.  Good thing the installation and re-partitioning performed flawlessly.  All the hardware on that computer, including a Netgear Wi-Fi card I had plugged in, was automatically recognized and appropriately connected.

Package management is amazing.  Firefox, complete with Java and Flash support, is quick and easy!  I never thought I'd say that about a Linux distro.  I have used the package manager to install Lynx, the Terminal Server Client, and some games.  Very impressive!   I am still testing the office applications but they are working so far. 

At the office I had an old Dell Dimension with an 800 MHz P3 and less than a gig of RAM, with Windows 2000 (!) and a 40GB HDD.  This is a secondary computer, not my main one, but I wondered how it would perform with Ubuntu installed.  I performed the same type of dual-boot install, in order to get an idea how the performance looks on such a throwback machine.  I am impressed with the performance, even on this 12-year-old hardware.  When I first installed, I got a message that said the graphics hardware was not advanced enough for all the latest user experience, and that I would be using an alternative instead.  That was a one-time message and I haven't looked back.

To those who have made Ubuntu possible: thanks and good work!
