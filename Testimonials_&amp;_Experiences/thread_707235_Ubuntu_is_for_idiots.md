### Post by TheIdiot on 2008-02-25
Hello Ubuntu community.

I'm an IT student who's been using Windows all his life. I made the switch to Ubuntu last night because I was simply bored. I realised I had no reason to use Windows anymore - I don't play games anymore and didn't really require any Windows exclusive software for Uni purposes. I didn't really "hate" Windows, it was just time to move on since I felt I've learned all I've wanted to learn for Windows XP.

So 24 hours after the installation onto my 2003 laptop, my thoughts? Ubuntu is for idiots... AND I LOVE IT! The ease-of-use for an experienced Windows user makes this OS incredible. Here's a list of why I'm already falling in love with Ubuntu after coming from Windows XP:
[LIST]
[*]Huge and helpful community means quick and straight-forward answers and reviews/preference of software
[*]Many CLI commands to GUI tasks are explained on these forums or via google, meaning a quicker alternative to doing things
[*]No virii/spyware/adware means no multiple resource hogging software and hours spent on scanning
[*]Closed/stealthed ports by default means greater safety (dw, I've read the thread with members arguing about whether I need clamAV/Firestarter so I'm not being compeltely ignorant/complacent) 
[*]No registry means no registry cleaner needed, ease of installation and removal of software with no traces
[*]Central repository for most software ensures quality, reliability, compatibility and safety
[*]OSS means I no longer have to rely on potentially dangerous warez cracked by Joe Blow to get things done and I could customize source code and recompile to my liking
[*]Pre-installed software means I can get most tasks done. Pre-installed Windows software only allows me to draw stick figures
[*]Everything in Ubuntu is handled in Ubuntu, meaning I don't have to google for missing dlls from untrusted sources in order to run software from a company I've never heard of. No junk is left behind that I need extra software to scan from. I can do a full clean from the command line and Synaptic to remove obsolete/orphaned/residual pacakges.
[*]medibuntu and ubuntu-restricted-extras mean I don't have to google to find missing media codecs
[*]The way Ubuntu is built allows me to multi-task more efficiently and with less lag than Windows. Compared to "68 million Ways to Make Windows Faster," I've run into only 10-20 for Ubuntu so far. As somebody posted in some site I googled, "Wanting to make Ubuntu faster is like feeding NOS to a Ferrari." 
[*]Ubuntu allows greater eye-candy with compiz fusion and emerald.  (I'm actually a minimalist, but at least there's the choice!)
[*] Scripting means greater customization (I don't know how to script yet, but I'm going through linuxcommand.org for the basics soon!)
[*] Comparing sudo[/LIST]

In summary, to get Ubuntu running for the average idiot, all Ubuntu needs is w32codecs/libdvdcss2/ubuntu-restricted-extras, and it's set. This operating system is safe, reliable, quick and easy to use right out of the box. Oh I forgot to mention that Ubuntu comes at a great cost... OOPS NO IT DOESN'T. IT'S FREE :-D

But besides your everyday idiot, Ubuntu also has the potential for the curious to undergo continuous learning from the wide range of online sources created and maintained by a huge worldwide community.

I'm just surprised at how impatient some people can be. I've read some other testimonials on this forum, and it even made someone as new to Ubuntu as me laugh. I remember one person posting something along the lines of "Sigh, so Evolution doesn't work. So I downloaded a Thunderbird tar.gz file from a random site I googled. I then double-clicked and sigh, it wasn't an msi file, it was like a zip. So I extracted the file and sigh, there was no exe to run Thunderbird in the extracted folder. Ubuntu sucks. It can't even perform something simple like an install." lol If people wanted an OS that acted like Windows, why not just choose another Windows?

A proper question now. For a desktop environment, what can Windows do that Ubuntu can't? Right now I've only been noticing things that Ubuntu can do that Windows can't.
