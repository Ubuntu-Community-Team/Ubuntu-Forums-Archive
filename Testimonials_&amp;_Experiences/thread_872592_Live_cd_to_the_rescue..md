### Post by Monstroxus on 2008-07-28
I work in a hotel (sales) and was notified that a pc in the executive lounge didn't want to boot XP. That pc is solely used by guests to browse the net, that's it. The IT guy was on leave, so I had a look... indeed XP didn't continue to run after the initial boot up process. I happened that I had just burned (another ISO) that day (for a install for a relative) so I used that one to boot, and after a few minutes, that pc could be used to browse the net. Even better, this way, none of the guests could mess around and install strange software/chat/whatever virusses etc - since its linux and a live cd, so no install possible at all. I just left a little note notifying the guests not to switch off the pc and that browsing was for non-flash sites. (as the live cd doesn't have flash installed) At least this way, so I thought, the IT guy didn't have to come all the way to work on his day off. (and I can't make the decision to install linux, that's not up to me) Sadly I saw him a few hours later, reinstalling XP. I told him, why not install linux, nobody can mess that up without being root? Anyway, he want to stuck with XP, but asked if he could keep the copy. He has some fears that guests might want to open windows specific files. I told him, shouldn't be any problem... at least you can save money for the licenses. But for the time being he still wanted to install XP (he's simply afraid the management will bugger him)... anyway... I was just glad I could help out this way for at least a few hours duration.
