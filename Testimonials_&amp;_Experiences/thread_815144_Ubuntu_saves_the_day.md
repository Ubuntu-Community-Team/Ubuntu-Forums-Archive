### Post by webcoded612 on 2008-06-01
I've been using Linux for a few weeks (this time...I was around when Edgy was still broken, but left for a while) now and the one thing that frustrates me is the (lack of?) 3D support.  I can't play any games through Wine, and emulators don't do 3D rendering.  

I thought I could get past that; focus on more important things than video games, but that doesn't cut it.  I want to be able to play video games, and Linux should let me do that.  

But it doesn't.

So I wiped my HH install and installed Windows XP.  Within 30 minutes (no exaggeration) my PC was lagging, Skype and Explorer (not IE...Windows Explorer) were crashing left and right.  It was a mess.  I had to reboot three times just so I could properly disconnect my external HDD so that Linux would recognize it.  

I left the Windows install (thinking it was a glitch with the new round of updates from M$) and decided to install HH again, resizing the partition this time instead of the full install I'd done before.  Turns out not only does Ubuntu not support 3D rendering on my system, it also cannot resize a simple NTFS partition.  

But that's not the point.  I remember times when I would complain about Ubuntu, always ending with something like "I know it's not my hardware, because it runs fine in Windows".  It's odd to me that, this time, the converse was true.  Nothing was running right in Windows, but (except for partition resizing and 3D video) it all works more or less OK in Ubuntu.  

So Linux is the only OS on my system right now.  I'm going to put up a post about using gparted to repartition and install Windows for those games.  Hopefully the community will help.  :)

Moral of the post:  Ubuntu is (very) slowly becoming my go-to OS.  Once we get some 3D support, I'll be happy to grab my video camera and make a video of me burning my XP CD and CD Key to show all you guys.  :P

Thank you, Ubuntu (and the developers, and most of the community) for making this great OS available!  :)
