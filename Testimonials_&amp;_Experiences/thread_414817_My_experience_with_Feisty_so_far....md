### Post by Orval on 2007-04-20
I have been using (K)Ubuntu since 6.10 and it looks like Feisty is the most stable on my computer of all versions I have tried. =D> And I think Feisty is the most beautiful to look at (especially with Beryl up and running).

I encountered 2 bugs so far: in OpenOffice I have no longer a save or open file dialog. It does not matter if I click the "use OO dialogs" option or not. And the icons and menu items are invisible until I hoover my mouse over them. I have installed the Tango icons for oo. The second issue I can live with, but the first one is a showstopper. I need the Save as option and I need to be able to save new documents.

Then there is a strange issue that is not a bug but that surprises me. In the prior versions of Ubuntu (and Suse)  my harddrives were addressed as hda and hdb. Now they suddenly are sda and sdb. :confused: Well. It does not matter, because everything works.

I am still amazed that Beryl works so great on my old ATI radeon 9200 with the open source drivers and AIXGL. It is so smooth, even when playing a video file.

It's good to be in love. :lol:
