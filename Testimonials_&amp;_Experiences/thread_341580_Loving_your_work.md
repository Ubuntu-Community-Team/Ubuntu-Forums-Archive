### Post by 13thMonkey on 2007-01-18
A week ago my main PC (Athlon XP) was a gaming machine, connected to an old AMD Sempron box that I connected to via PCAW and ran ICS via a USB ADSL modem, a web development server which also hosted a couple of lightly used sites that I didn't want to host externally, Skype, MSN and a torrent client. Also connected to that via a cheap hub was an old P200 Windows 95 box that's occasionally used by *very* inexperienced people to check email, surf the web and make me have to reinstall everything once a month ](*,)  - but I digress.

I've been using Windows for over ten years but before that I was subjected to the delights of Sun OS which I never got the hang of as a casual user and quite frankly put me off anything ending in *nix* for life. However, my Sempron XP box finally crashed once too often so last week I decided enough was enough!

After spending some quality time with Mr Google looking at the various distros, I decided that Ubuntu was just what I needed. In the past ten years I've become a lot more computer savvy so I wanted something I could learn about and control but I also wanted something that would be easy to install and be pointy-clicky for when I was on the verge of throwing the computer out of the window.

To cut a long story short, Edgy was simple to install and recognised everything out of the box, apart from my USB ADSL modem - but I knew that was going to be a problem anyway thanks to Mr Google. I followed an excellent [guide ]("http://www.linux-usb.org/SpeedTouch/ubuntu/index.html") but still couldn't get online. A few internet searches later and I figured it out though. And better still I began to *understand* what was happening =D>  That's me though - as I've got more of a clue, all the guides I've used have worked (with a very easy tweak here and there sometimes).

It took me two days, but my server is now running everything I need it to. I can connect using FreeNX to my server from XP (and it's a lot faster than PCAW), access all my shared directories and hard drives from XP using samba, chat to my friends on MSN and Skype and my websites are all up (virtual servers are so much easier on Linux :)) etc. I still need to setup an FTP server and decide what to do with one of my NTFS drives but since it's going to involve burning a lot of DVDs either way, I'm going to look away now.... :-# 

I kept the dual boot to my server's XP install in case things screwed up and was glad of it when I was trying to install my modem (saved me fiddling around the back of the computer trying to swap the modem at least!)  but now I have a bit more of a clue I have no use for it. My next PC will be Vista because I play lots of games but everything else will run Ubuntu - I don't fancy having to reinstall XP on what used to be the Win95 box (to be replaced by my shiny new Ubuntu PC) everytime somebody actually clicks here on one of the flash 'click here you've won a prize if you download this' banners :rolleyes: And I'd quite like my server to not need rebooting everytime it loses the net connection/thinks I've looked at it in the wrong kind of way.

Anyway, I guess the point I'm trying to make is that I've had a lot of fun installing Ubuntu and setting things up and I'm going to have even more fun learning more about it. (That sounds a bit sad, but in my defence I've never enjoyed installing XP [-( ) And if I can do it then **it's not all that hard**. :-D  No way could I have done it without the various howtos and posts on the internet though, so thanks to all. 

Cheers,
Monkey (in danger of becoming a Linux bigot #-o )
