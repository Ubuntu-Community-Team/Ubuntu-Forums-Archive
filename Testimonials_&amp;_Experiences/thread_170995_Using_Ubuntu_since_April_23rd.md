### Post by Toe on 2006-05-05
and I'm very happy with it.

About two months ago I stumbled upon a discussion about various Linux distributions in a forum I frequent. One poster was very enthusiastic about Ubuntu, so I spent some time surfing the website und trying the Live CD.

Before that my only Linux experience had been with Knoppix.

My main reason for choosing Ubuntu over other distributions was its ease of use (although I'm quite an expert with Windows-systems I was still a bit intimidated by Linux) and the philosophy behind it ("humanity towards others").

I was quite impressed by Ubuntu, especially by the easy way of installing new programs and updates.

It took some time until I decided to install it, though.
Mainly because I needed to find out if software equivalent to that I was used to existed for Linux.
e.g.:
-a tool to enable the power-saving mode for my Athlon CPU
-Emule-Client
-a way to access my university's servers and download lecture notes
-a tray-icon-program to show the temperature values reported by all hardware sensors
-good audio- and video-players with support for lots of codecs
-CD- und DVD-burning program
-a good chat software for ICQ, AIM and MSN

The fact that I could keep playing UT2004 also encouraged me to give Linux a try.

First I chose Breezy. I shouldn't have started installing it late at night, though. Expecting quick results I was a bit frustrated by driver problems, missing MP3-support and the fact that I didn't know most of the basic terminal commands.

So I found myself going back to Windows the next morning.

But strangely I kept visiting the Ubuntu website and browsing the Dapper development forum.
Two weeks ago I decided to give it another try and installed the latest Dapper beta.
This time I was more relaxed when installing, knew more about the Linux basics and one by one solved all problems I encountered (webcam drivers, TV card, mounting my second HDD, installing codecs,...).

My system still has some minor problems (freezes whenever 3D graphics are displayed - I'm using the latest fglrx driver, Radeon 9000 Pro) - I'm going to report the problems in the Dapper forum.

But all in all I'm very pleased with Ubuntu's stability, the available software, the great support and the fact that it's free (my budget is quite low).

I'd like to thank Mr. Shuttleworth and every sponsor and developer out there!
