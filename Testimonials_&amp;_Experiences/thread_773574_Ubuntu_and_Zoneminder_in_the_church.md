### Post by theorganloft on 2008-04-29
Not only did I upgrade my MythBuntu and Ubuntu Studio to 8.04, I install Ubuntu and Zoneminder at my church saving them thousands!
Ubuntu and Debian run the server and the console/configuration workstation. 16 analog cameras, a Pro-video 16 port capture, and centralized power make for a powerful surveillance system thanks for the stability of Ubuntu!
