### Post by deuchar1 on 2007-03-29
Just found this forum and thought I'd give my input!

I'm a professional web developer and recently installed Edgy 6.10 on my home system. I was sick of problems with UNIX vs Windows character encoding issues and so forth, and as 99% of our client servers are UNIX based I decided I'd be as well working in a system like Linux. 

Ubuntu was recommended to me by my brother who uses it himself. I opted for an XP dual boot. I had only one problem with installation which was an inability to resize the NTFS partition. I eventually resolved this by disabling Windows' page file which was creating an unmovable block right at the end of my disk, meaning there was no clear space to slot in new partitions other than in the gap between. After disabling I ran DIRMS to defrag my system and then the installation worked fine.

Best decision I've made. The ability to grab software simply by typing a line in the terminal is brilliant. And the array of open source software is getting more and more impressive every day. I remember the days when Windows ruled the roost in my field simply because the software simply didn't exist for other operating systems than Macs. 

Now I run NVU/Screem/GIMP/BlueFish for all my design/development needs and rarely switch to Windows for anything. In fact, after discovering how to mount my Windows partition (opted for read-only to be safer) I find myself in XP about once a month to see if it still works, just!

One criticism I will make, albeit a minor one, is that Ubuntu relies too heavily on the system being online. I didn't have broadband in my new flat when I first installed and so couldn't auto-grab programs - I was manually downloading packages and found they can be a real pain in the neck when you have to manually grab all dependencies, the dependencies' dependencies, and so on.....    But nevertheless the day I got online I had a download fest and found hundreds of handy software and tools. 

You had me from Planet Penguin Racer ;)
