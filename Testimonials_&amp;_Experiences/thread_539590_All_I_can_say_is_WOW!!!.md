### Post by mitchlinux on 2007-08-31
I've been using XP for a long time, and I have tweaked it a lot. I formated mt home pc and installed Ubuntu and CompizFusion, and all I can say is WOW!! My wife is not tech savvy so she doesn't note the difference. 

As for the CompizFusion, at first I thought to myself "why would I need 4desktops for?" but Im actually using it a lot and I love rotating the cube.

Everytime I go to my work pc, which still has WinXP, it seems it so slow. It has double the memory of my home pc and my home pc is way faster. It seems Ubuntu is not a memory hog.

I would switch my whole office to Ubuntu but there is a program that I really need and runs on .NET, it's called Calyx Point, so I haven't switch my office PC's yet.

Anyways this Ubuntu is very very very good, and the best of all, it's free!

### Post by dark_harmonics on 2007-09-01
Its really easy to get virtualbox running fullscreen. Im actually typing on a computer that has that setup right now. You just need to "suggest" the correct screen resolution to xp after installing it in the VM. Then you make sure that "full screen legacy support" option is checked in your compiz fusion -> workarounds. Its so flawless that my boss doesnt even know im running ubuntu when im at work! 

When he steps away from my desk i switch my cube back to my much faster ubuntu screen:)
