### Post by locosway on 2011-10-14
I've been using Linux since the late 90's. I remember the excitement when a new distro would come out and I could buy it in a box at the computer store. Redhat, Suse, and *BSD were all you could buy, but there really wasn't any alternatives at that time.

When I got DSL that opened up new options, such as Debian which I ran for a long time. Gentoo came around and offered my new AMD64 machine some very needed 64bit support. And then Arch popped up and it was over to Arch, which was very nice, not having to compile all my own packages for my AMD64.

Then, there was this magically distro called Ubuntu. It promised everything, and kept most of it's promises. This was great! A Debian system that kept the packages more current. I fell in love, like many people.

Well, KDE ended up perverting their desktop, and off to Gnome it was. I never cared for Gnome because KDE was so much more polished, with it's app integrations. Now that Redcarpet was dead, Gnome wasn't so bad. Ubuntu and Gnome was a nice variation, and everything worked wonderfully.

Enter Ubuntu 11, and the horror that is Unity. More and more packages are unstable, missing dependencies, and unusable interfaces. Ubuntu destroyed a good thing, with their urgency to move to something newer that really is horrible to use on a laptop. It's not like Windows or OSX, instead it's kind of a mix, leaving the user to guess how it works.

Well, the good news is Ubuntu Classic was available, until last night. Now I'm stuck with Unity, unresolved deps, and a host of other issues. So, it pains me to say this, especially as a Linux Administrator, that I feel Ubuntu really missed the mark, and it's time to move on.

It's been a fun ride, hopefully this message isn't disregarded.
