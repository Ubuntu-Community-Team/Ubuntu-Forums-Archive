### Post by bythelake on 2010-05-29
I just wanted to say "thank you" to Ubuntu and the support community for an awesome product.

After years of wrestling with Windows XP and countless malware/virus issues, not to mention the performance loss due to antivirus software, I thought I'd try Linux again.  After some brief research, Ubuntu seemed to be the right choice.

A few months back, I started with 9.10 and moved to 10.04 when it was launched.  After wrestling a bit with some ATI video driver issues, Ubuntu 10.04 is all I've used for 2 months now.

It's wonderfully quick-booting and performs very well.  It seems that any software I've needed has been available from the Ubuntu Software Center, so I've not needed to revert to Windows for anything so far.

Great job, team.  Keep up the great work.  After 2 months, I don't miss Windows XP one bit, and didn't need to spend for a Windows 7 upgrade.

-Kevin
