### Post by wrycatcher on 2006-12-15
Yes, Ubuntu is now officially a verb! (in my lexicon anyway)  I could Ubuntu all day long actually.  I enjoy exploring the high configurability of the platform, and sampling the abundant applications that are available.  Everything from high-end, feature-rich mainstream apps to the obscure and minimalistic command line tools.

In these early days of my Ubuntu experience, I've been pleasantly surprised at some of the applications available which are on par with the Windows equivalents.  There are some holes which need to be filling still IMHO, but I'm willing to live with a few of those, provided they aren't gaping or overly numerous.  There doesn't seem to be a limit to what is possible, either.  Just because it isn't there now, doesn't mean it can't be done.  If there's a need, someone will come up with a solution.  Necessity being the mother of invention, you know.

Ubuntu (and the forums) have also served a philosophical purpose.  It has raised the issues of properietary file formats and open-source software (including device drivers), issues to which I'd previously been fairly ignorant.  This seems like a very touchy subject from what I've read.  For my part, I seem more aligned to the open-formats camp.  I don't really feel strongly about (freely available) binaries vs. open-source code, though.  I'm still trying to figure out why using binaries is so evil.  Maybe it's a fundamental issue of control and collaborative spirit.

Despite all the manual tweaks I've made to various parts of the system, and numerous config and  resource updates, it's been real joy to Ubuntu.  Refreshing.  I plan to craft my Ubuntu into my own work of art.   Really, that's how it feels, like I'm creating something unique and almost living.  Windows was functional and certainly capable as a desktop, but it was like ordering food off a fixed menu, while Ubuntu is more *a la carte*.   I love the freedom, the control, the choices.  I've already been talking up Ubuntu to friends and I'm rapidly becoming a Linux evangelist.  Whenever I see a place where it makes sense for someone to use Linux over Windows, I'm going to recommend Ubuntu and likely get it set up for them on their machine(s), complete with a tutorial on how to do all the things they were previously doing via Windows.  Many times it's purely email, word processing, and web browsing.  I mean, that's it!  And they are enduring painfully slow performance on a low-end machine with Windows.  So, I will help to enlighten them a little...

Long live Linux.  Long live Ubuntu.
