### Post by Wizgot on 2011-08-11
I just recently installed Ubuntu 11.04 alongside Windows vista on my laptop to test it out. I had done this because Vista was becoming unbearable for me. Whenever I would boot up my machine I would have to wait ages before I would see my desktop. After that, I would have to wait while background programs loaded. It was such a hassle and I needed a break. I had known about Ubuntu for a long time and always wanted to try it out. I downloaded the iso image and installed it alongside windows with Wubi. After restarting my computer and booting into Ubuntu I only had to wait 1 minute for it to load fully. You may think "Hey, Ubuntu only takes 10 seconds to load" not for my case but  I don't care when it used to take upwards of 5 minutes for me to start windows.

I'm still testing it out and doing miscellaneous things here and there. I find unity to be ok and after installing Compiz fusion and auto hiding the sidebar find it superb. I am trying to use Ubuntu for all my needs (not many when all I do is use my laptop for Internet surfing and torrents) and it works seamlessly. I was even able to get a hard drive that had stopped showing up on windows to work again. I really like this OS.

### Post by sanderd17 on 2011-08-11
Great to see your good installations. I installed my first Ubuntu 3 years ago, Ubuntu 8.04. And it worked good, but not perfect.

Since then, Ubuntu has got some ups and downs. The 10.10 release is my overall favourite (the most stable OS I've ever seen) and 11.04 is one of the worst in stability (for my hardware anyway), but good looking.

As for desktop environments, I'm now on the default Mint look (Mint is a derivation of Ubuntu, a bit more windows like look), but my favourite DE is Gnome-Shell. The only problem is that it isn't officially supported in Ubuntu yet (it should be in 2 months) so my previous Gnome-Shell environment broke with an Ubuntu update.

Overall, I find Gnome-Shell more stable, more intuitive and better customisable than Gnome 2.x (classic look) or Unity.

Anyway, welcome to Ubuntu
