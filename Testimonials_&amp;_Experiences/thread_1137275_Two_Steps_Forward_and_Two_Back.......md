### Post by Methuselah on 2009-04-25
3D acceleration is still a bit hit and miss.
I've found that the free drivers are stable but don't always support the latest cards.
The proprietary drivers do support the latest cards but tend to be buggy.
Intel graphics is the best for linux support but they do not make high performance discrete graphics cards (yet!).

However, to put things in perspective it really is quite remarkable that Ubuntu will run on most hardware even from a live CD without the user configuring any drivers.
We have been spoiled and therefore expect the perfection that seems so near; but from many installations of windows XP and windows 20003 server I know that what Ubuntu currently does is fantastic, if imperfect.
