### Post by Geekkit on 2009-04-30
Seriously, this is the first version of Ubuntu that I've had no problems with and installing it so quickly after a release (so still steaming from coming out of the oven). The last version gave me kernel lock ups. About the only (minor) problem I have is when I turn on compositing in either Compiz or Metacity, certain operations such as window resizing, restoring, and switching between windows causes the entire desktop to freeze for several seconds.

The link:
[http://www.linuxinsight.com/your-ati-radeon-very-slow-on-xorg-x-server-1.3.html](http://www.linuxinsight.com/your-ati-radeon-very-slow-on-xorg-x-server-1.3.html)

Suggested a solution but it didn't solve it for me. I found a forum post (here) that suggested that the it was a bug and that the bug was submitted but a link to the bug wasn't posted. But I mean not running Compiz most certainly won't kill me (or rip my face off).

Subtleties I've noticed:
- Just seems to be faster (FF, boot, bringing up applications, menus displaying, etc)
- Notices in the top right ... interesting
- Clicking on a file (or multiple files) in Nautilus now brings those files to the forefront. Remember that the old behaviour was to open them up behind Nautilus because it was assumed that Nautilus was going to keep focus? Not saying that this is a plus or a minus ... it just is. It's also like other OSes so maybe this is a good thing
- Ctrl-Alt-Backspace disabled ... but disabling the disabling is easy enuff!
- Blacklist is now called Blacklist.config (for turning off that annoying speaker ... actually that is something they should really get rid of. It's so ... 1990s)
- The installation process is ridiculously simple - seriously, try screwing that up
- Synaptic does real time updating for the searches, very cool
- Janitor ... hehehe.... maybe next version they'll make an Oscar the grouch character sweeping away all your unused packages, software
- "Start up applications" ... nice ... I know it's sessions and stuff but the naming makes it so much more ... human.
- Desktop background dissolve ... nice
EDIT: also the screen shots in Synaptic ... nice


Kind of looks like they added some cosmetic things as well as true performance enhancements. Anyway, it's pretty cool IMO. I'm still getting over how easy and quick it is to do a clean install.

Question: for sharing, where does one see where all shares are? There use to be something under System->Administration but they moved it.
