### Post by SonicSteve on 2007-01-23
I work with computers, I fix em I sell em, I even buy boards with bad capacitors so I can replace them and sell them if they restore well.

I had installed ubuntu on an ECS group mainbaord, K7S5A just a run of the mill Socket462 board with SIS chips on it. 
I had another  board ECS K7VMM2 or 3,  that worked in every way except the sound chip was flakey. I decided to use it because it had a cpu that was soldered to the mainboard and the K7S5A had a chip that could be upgraded. So since I can't sell the K7VMM board I decided to use it and free up the processor in the K7S5A board to sell it instead.

So in preparation for the change over I backed up my home folder and saved a copy of my FSTAB file. I then changed all the hardware and turned on the machine, I missed the boot up F8 option and Ubuntu started to boot up. I thought Hey lets see what happens, after all I was resigned to the fact that I would have to reinstall anyway. 
[SIZE="5"]
[COLOR="SandyBrown"]To my amazement it booted up perfectly[/COLOR][/SIZE]

[SIZE="6"][COLOR="Blue"][COLOR="Red"]Windows would never have done this. It would have freaked out with a BSOD, I know because I've tried it[/COLOR][/COLOR].[/SIZE]
[SIZE="4"][COLOR="Blue"]
I'm very impressed, I don't know if this would be the result in all cases but this is quite cool. [/COLOR][/SIZE]

### Post by Grey on 2007-01-23
I was really amazed by that the first time I tried it too.  (I'm less amazed now, since I no longer consider myself a Windows user, and expect such behaviour now).

My story was a little more dramatic though.  I was dual booting at the time, and a new linux user.  I finally got the cash to do an upgrade.  I was upgrading from an ECS K7S5A (popular board), with an Athlon XP 1800+ to an ASUS A8V Deluxe with an Athlon 64 3500+.

When I booted the computer to test it, Windows tried to boot, as it was the default OS.  It got partway through, and died a horrible death.  Just for laughs, I decided to try Ubuntu.  I was mighty shocked when it booted up just fine.  And that same OS was installed on my computer for the next year.  When it was finally removed in favour of 64 bit Ubuntu.  It NEVER died, and was always just as stable as the day I installed it.
