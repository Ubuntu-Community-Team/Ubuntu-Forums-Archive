### Post by KamikazeNY on 2009-05-14
Hey everyone. I wanted to stop by and introduce myself. I just made the switch from Vista to Jaunty last week. I went with the x64 distribution and installed to a RAID-1 array. In Vista I was using my mobo's fakeraid but with Ubuntu I disabled it and used Linux's software raid instead. So far I've managed to work out pretty much all my issues and everything's running like a top. 

Except for streaming to my stupid Apple Airport Express. I did see that PulseAudio 0.9.15 supports streaming to the Airport via the raop module, and I did get it to kinda work, but the audio is choppy. But at least I have a project now. ;)

I've been lightly using Unix and Linux servers at work for a few years (no desktop though) so I already knew the basic ins and outs. I'm a software engineer specializing in Java and J2EE development so if anyone has any questions, particularly related to my specialties, please let me know and I'll be happy to help to the best of my abilities.
