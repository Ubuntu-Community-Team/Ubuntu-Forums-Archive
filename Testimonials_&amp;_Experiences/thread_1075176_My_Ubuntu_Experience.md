### Post by BrendanMark on 2009-02-21
After a few teething probelms with other Linux distros (openSUSE & Mandriva) - where I had to turn the video card down to 800 x 600 in XP to get display - I installed Ubuntu 8.04 and have been working fine since. It even seemed to help settle openSUSE down so that I now have triple-boot machine with Ubuntu, openSUSE 11.0 and XP.

I installed 8.04 as that was the one with long term support, so I figured it would work with my 'old' hardware. Now I'm eyeing the three 8.10 Intrepid Ibis DVDs I've picked up in Linux mags with some curiosity. Should I wait for 9.04 or keep the Ubuntu install as my 'safe' Linux one and flog SUSE to death on a regular basis?

Getting connected and firewalled with secure email was a breeze. I've been using Unix as a system admin and systems programmer for almost 20 years, so the only thing I found annoying was the lack of immediate root access, but that's easy enough to get over and get used to using sudo.

And the wealth of great software available is breathtaking. I've seen some MS stuff, but have to say something like Eclipse is sooo much nicer on Linux!
