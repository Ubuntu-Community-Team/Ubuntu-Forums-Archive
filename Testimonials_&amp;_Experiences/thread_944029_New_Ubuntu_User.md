### Post by blitzo on 2008-10-10
I have been using Linux for several years.  I tried Ubuntu awhile back and well I liked what it was doing for the Linux community but it just didn't appeal to me.  Most of my experience is with Red Hat systems.  I was having some severe issues on a machine with Fedora and I needed something to install quick.  I installed the latest Ubuntu release as a temporary solution until I fix my problem.  So far I am really impressed.  I got everything working with no major headaches and I saved a lot of time.  I think I just might keep this install around.
