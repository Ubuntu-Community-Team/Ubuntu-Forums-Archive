### Post by kjb34 on 2008-09-05
Last year I got an old computer from my uncle. All it needed was a harddrive so I ordered one from Geeks.com but come to find out I got a SATA when I needed an IDE drive. SO instaed of sending it back I decided to build my own computer. SO I wnet back to Geeks and ordered everything and then I had to decide waht my OS was going to be. I didn't want to pay for Wimdows so that left Linux. My brother recommended Fedora but I wasn't sure so I went to Linux.org. After reading up on the various distros and visiting Distrowatch I decided on Ubuntu 7.04. I ordered a live Cd and was ready to go. Well when I first got my mobo it dead so I sent it back got another one so then I pop in the live cd doesn't work get another and dvd drive dies. During this time after doing some research I decided to get the alt install cd. I downloaded the Gutsy alt install and when my new drive arrived I install 7.10 with no problems. When I got the internet installed it worked perfectly. When it was time to buy a printer I researched the printers to find out which models were compatible with Linux and bought a compatible model and no problems getting it working.
Everything was going wel until I decided to upgrade my memory. My mobo had erractic since I got but I didn't really pay attention to it. Anyway as soon I install the memory the mobo crashes. I send it back they replace it and I connect itand it's dead too. Send it back I talk with tech support and ask them if they check the mobos berfore shipping them out they say no, so I ask if they would please check out mine before shipping it out. Mind while I'm sending these mobos back I haven't paid anything because the warranty and also they ended upgrading my mobo and cpu( to 64 bit chip). Finally I get a mobo that works but now my harddrive got messed up so I try to reinstall 7.10(32 bit) no luck.I try the 64 bit 8.04 (live and alt-install) no dice. I end installing 7.04(32 bit) but it doesn't access the internet, so I try 7.10 (32 bit) again I get it work and the internet is working again. So I upgrade to 8.04(32 bit) and it doesn't boot properly I find that I can boot up the old kernel and I use that. I searched the forums here and I found what was causing the problem and I get it working perfectly. Then I decide to try 64 bit 8.04 again with little success, then I try the same thing I did with the 32 bit and it works but it gets hung up with partitioning my hard drive. So I now I have screwed up my harddrive and repeat everything I did before and reinstall 8.04 (32 bit) but this time I set up a partition to put the 64 bit 8.04 on. I try it again but modify the procedure I used for 32 bit and it installs easily. 
Now let me say in spite of my troubles I enjoy using Ubuntu. 8.04 is been very easy to use and I found that it was so much easier to get flash and DVD playback working than in 7.10. My wife enjoys using Ubuntu because for our needs it does everything that we need. I have been to get my sister and mother to use Linux and my nieces and nephews don't even the notice any differences between Linux and Windows.
