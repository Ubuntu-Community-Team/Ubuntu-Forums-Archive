### Post by automaton26 on 2009-11-02
I'm a Kubuntu x64 user (for the last 12 months) who has just moved to a clean install of Karmic 9.10.

The installation process was quick and simple, and it's immediately noticeable that usability has improved:

[LIST]
[*]Encrypted /home option during installation
[*]Automatic writable access to my existing Vista partition
[*]Almost instant Shutdown, compared to Vista x64
[*]Network Connections widget, when configured with a static IP address
[*]Hardware Drivers, for my ATI Radeon HD 4850
[*]Dolphin context menus, for compressing/expanding archive files
[*]Simple installer for latest Firefox 3.5
[*]MPlayer "scaletempo" Audio Filter is available (-af=scaletempo)
[/LIST]

Many of those previously needed manual configuration but, in Karmic, **they just work**.

Overall I'm impressed so far, and my huge thanks go to everyone who contributes to all the component parts.

The *average* user really does have a choice of operating system on the PC these days.

### Post by cariboo on 2009-11-02
I would suggest that for someone like you with a lot of customizing, an upgrade may not work properly. You would have to comment out any non-standard repositories, remove any non-standard libraries and uninstall any programs installed from a ppa, and make sure the system is up-to-date, before even attempting the update.
