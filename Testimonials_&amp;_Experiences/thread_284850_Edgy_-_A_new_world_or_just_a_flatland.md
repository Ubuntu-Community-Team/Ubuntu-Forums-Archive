### Post by gruffy-06 on 2006-10-26
Last night, I had upgraded Ubuntu from 6.06 LTS to 6.10. All had worked flawlessly!

6.10 now seems to boot up faster and the sounds have changed a little. GNOME loads faster too. And now there are over 20,000 software packages, all in one distribution! The system is now easier to configure and all of it looks more consolidated.

The only discrepancy with the "Edgy Eft" is that it is still essentially the same OS in some areas. Users of the "Dapper Drake" still have long term support, but overall, Ubuntu is such a great success!
