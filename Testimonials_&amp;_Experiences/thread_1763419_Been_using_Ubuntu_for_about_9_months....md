### Post by larekare on 2011-05-20
I used Windows, all the way from 95 to XP. I read Vista was ****, heard 7 was back to XP. I had been keen on using linux for a while but hadn't taken the step. Then about 9 months ago I took the step on one of my computers. I found, that 10.4 was much easier than I expected and it felt REALLY good to use compared to Windows (I've tried Mac a few times and I tend to get confused, frustrated and irritated by errors -- not to mention bollocked by proprietarieness). 

Anyhow, I was using 10.4 and 10.10 no problems...I admit there was a learning curve, and getting used to different applications with smaller user groups was a bit hard (but I imagine starting to use windows XP for the first time without ever having used windows before would involve quite a bit of googling too). And even though I sometimes have compatibility, so does the windows users. 

Also, as I'm getting to know the OS a bit more, it is easier to find out how to fix these issues. 

Then came 11.4. I was really excited, so it was a big let-down that the Unity interface started out so buggy (though it seems to be getting better and better as time passes). But after reading around I understand why they released it even though it was flawed. They want more people to use Ubuntu, and I want that too so I guess I'm not so let-down know that I understand why. I think Unity is a pretty OK interface...I mean it hasn't changed much of my workflow (When I want to start a new program, instead of clicking in the top bar, I know click in the Unity bar on the left, and instead of going through the menus I just search...and switching between programs I just use alt-tab). 

So I'm still really happy with using Ubuntu. It still feels good and works about as well as Windows, better on some aspects (printing, security, lightweightedness, cool software, feeling, open-source, etc) and worse in others (games, opening docx with LibreOffice and some typical work-software does not support Linux + Wine is a hassle...). 

I realise that Ubuntu is a work in progress, and it is progressing fast and well. So I have one suggestion to make it better:

The Ubuntu Software Centre (which absolutely fantastic btw) has a 'paid app' category (which is also pretty nice (I'm playing Darwinia while waiting for Adobe software support;p)). But the problem is that I have to fill out a long form each time I want to buy something. You need the option (I'm saying option so you can make both lazy people and security-minded people happy) for a click-and-buy option or a click-confirm-buy with a credit card on file. This is an absolute must if you are going to have more apps there as I was already annoyed by not being able to buy something (spend money that benefits Ubuntu/Canonical and proves profitablity etc.) in 5 mins. Then I had to catch the bus and didnt have time to fill out and spent my money on beer instead. I mean come on! The whole point of a shop is to make it as EASY as possible TO SPEND MONEY. And I cannot see any rational reason for why that is not included when my Ubuntu One account works without me even having to accept or put in a password or anything. 


Otherwise I am really thankful and happy for this OS.:)
