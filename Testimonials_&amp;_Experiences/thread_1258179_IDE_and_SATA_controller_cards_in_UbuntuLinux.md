### Post by MartyBuntu on 2009-09-04
I gave my friend a VIA based controller card because the motherboard we used (budget Gigabyte board) had only one PATA port and we needed a permanent hook-up for a DVD-ROM drive, a CD-RW drive and an IDE hard drive.

Of course, he wanted to stick with Windows, but hey, that's his decision. The only problem was that Windows wouldn't recognise the controller card OOTB and we needed to get files off before proceeding with the XP installation.

No problem, fired up Ubuntu 8.10 Live CD, read the drive, transfered files to USB flash drive. Everything ran smoothly.

Once again, Linux OOTB hardware support amazes me.:)
