### Post by markosjal on 2009-03-20
After months of thinking about abandoning Microsoft Products completely, I finally did it. My move was inspired by an increasing number of hard disk write errors on all drives in my system . Because this has been with both USB and IDE drive, I do not believe what I see in the windows Event Viewer that  all of my drives are failing at the same time., More than likely, Microsoft is forcing obsolescence on windows 2000 through updates, and I do believe this to be a Microsoft tactic that they have used in the past! This is Microsoft's way of trying to make me believe I need new hardware and a new OS, most probably Windows Vista. Move over Windows, here come Ubuntu and it is as ready as ever!

Herein I will detail my experiences thus far, what I miss from Windows, and what I praise in Ubuntu.

I (thought) I found the most painless way to migrate to Ubuntu 8.10 (who cares about Ubuntu names as they only serve to confuse) . What I found is that the install did not install as planned in the end. 

About my install. 
I installed a second internal physical hard disk, and planned to boot depending on BIOS selection. What I found after attempting this was that Ubuntu installed GRUB onto my windows drive without asking me nor informing me. Ultimately it was my plan to remove the Windows disk completely, now with my install of Ubuntu I am dependent on that drive. If either drive fails I lose my install of Ubuntu. Why does it install the Grub loader on the Windows drive when I select to install on a new drive? Who knows.  I can not boot directly to the Ubuntu disk, and it appears I will never be able to remove the windows drive from my system, without reinstalling Ubuntu. 

After all was installed
I was surprised to find full read and write support for my NTFS formatted Windows 2000 System disk, after installing the NTFS support. I have full access to it and can open most files with a click (or two). I installed Wine. I became accustomed to an old program called Windows Time Code Calculator. I was surprised that when I navigated to my Windows disk and clicked the .exe for Windows time Code Calculator , that “Open with  Wine” was an option., and this ran perfectly .

3 days exclusively on Ubuntu, with no issues, 
until I tried to install support for My ATI All In Wonder Rage Pro which now seems to be stuck in low graphics mode, and still has no TV support. I have installed both an HP inkjet USB printer and a Samsung Laser that is connected via LPR/LPD port to my NAS. As for Windows Networking to my NAS , I sometimes see that the directory is unavailable, but a second try usually yields the expected result. My NAS is Linux Based but apparently only supports Windows (SMB) file sharing. What has astonished me was the data transparency across platforms, and how all files and disks seem to remain usable in one form or another including even some older formats like Autodesk Animator FLI files. I also attempted an install of Skype for Ubuntu, but it fails to connect to audio, so voice calls are impossible.

Installing Programs
For the most part, I have found most Ubuntu Program installs have gone off without a hitch including things like Hamachi, which I can not live without. I have also installed aMule which when directed to my existing eMule folders on my Windows Disk, Happily continued my downloads as if they had started there. Even switching between Windows (eMule) and Ubuntu (aMule) my downloads continue with whichever OS! I installed Skype for Ubuntu, as found on the Skype Web site, as mentioned above there are sound issues. I am not happy with the version of OpenOffice 2.4 as OpenOffice lists version 3.X on their web site. I am surprised that this latest version is not available in Synaptic Package Manager, and I do fear messing it up by installing a newer version, that could go bad. I praise the Gnome RDP program, as on my windows machine I used three programs to do what I do from that single program.  I was also impressed to find VirtualBox in Ubuntu, and using this one can easily set up any Guest OS, or Virtualize an Ubuntu server.

Multi-Lingual Support
When I want to change the OS and Most programs between Spanish and English , all I need to do is log out , select the Language and log back in. Not to mention I know I can Install other Desktop interfaces such as Xubuntu and Kubuntu. I  find this feature quite impressive, however I have not found an easy way to switch between keyboard mappings, and ALT codes such as ALT164 do not seem to map to the characters as I have become accustomed to in Windows. It seems there is most probably a solution for this , but as yet I have not found it. I am a person that for the most part has two keyboards on  a system, one in English , and one in Spanish, however this limitation does not accommodate my needs well  for the keyboard at least. 

All in all
I would say that Ubuntu 8.10 is ready for mass deployment in an office environment. It is way better than windows where users may share a computer ( that use different languages) or just for a single user that needs multi- lingual support. Most of the common office components work very well, and look much like what windows users are accustomed to. However for US based Excel users I did find one thing that takes a lot of getting used to, and that is the international separator format used in OpenOffice. In other words when one is used to “=Concatenate(A1,A2,A3)” this becomes “=Concatenate(A1;A2;A3)” which for a long time USA based Excel user can be difficult. I think that OpenOffice needs to conform better to the regional settings.

What I miss most
Adobe Photoshop!

What I miss secondly
Trillian Messenger, however I did read a post where users are running this via Wine. I can not image this however because it always seemed to slow down my Windows machine with my assorted 6 IM accounts. My contacts were neatly arranged into groups on Trillian, and now my contacts are a ClusterFrack on Pidgin.

What I would like to see
Video transcoding tools
Video Editing Tools
Better Image editing tools (Photoshop)
