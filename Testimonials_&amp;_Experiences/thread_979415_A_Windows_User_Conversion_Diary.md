### Post by Irihapeti on 2008-11-13
Re setting up SCIM:
Make sure you have "enable support to enter complex characters" enabled in System -> Administration -> Language Support

Then open a terminal and enter the command
```
locale|grep 'LANG='
```
That will tell you what locale the computer is set to, such as en_US.UTF-8  I gather that English locales need to be .UTF-8 in order to work, but I think most are in Ubuntu.

Then enter the command
```
im-switch -z <your locale> -s scim-bridge
```
Note that you don't need sudo.

After that, restart the x-server by pressing control-alt-backspace, or you can reboot if you really want to.

You'll also need to install some scim libraries - scim-m17n or scim-hangul - for inputting Korean.

I hope that's of some help.

Actually Mr. Gates is long-gone from this computer. There wasn't much available room on her HDD and I didn't want to be juggling files between external storage devices, so I asked her if I could just backup her personal files onto an existing separate partition (immediately accessible from Linux Mint, of course :D) and just wipe the Windows drive.

Regarding remote desktop: In the "viewee" computer, find "Remote Desktop" somewhere in preferences or admin (I'm at work so I'm not sure exactly where it is). From there, click "Allow other users to view and control my desktop", then specify other settings like a password and whether you want the "viewee" to be prompted when you're trying to remote control.

After that, it's very helpeful to have a dynamic DNS service set up so the end-user doesn't need to find their external IP and tell you what it is. You can use a service like the one at [www.dyndns.org](www.dyndns.org) for this. That way, you can simply type yourfriendsname.dyndns.biz as the IP for remote desktop viewing.

From the "viewer" computer, simply run "vinagre", and Remote Desktop Viewing will show up. From there, click "Connect" and insert your friend's external IP or dynamic dns host address. If you want to remote control from a Windows desktop like I do from work, you can use the free VNC viewer.

Let me know if you need any more help with this, or search for one of the many online guides about this topic :)
