### Post by Groundswell on 2009-11-10
I've been using ubuntu for a few years now, sometimes as dual boot, sometimes as my only OS. The only reason i have any interest.. let me try a different word, "patience" for windows is the same thing we've always heard, games. It's a shame so many game publishers/coders won't make the transition to linux, cause from my experience, it's the only step left for linux to take!
 
about 6 months ago i tried to install xp again, cause i missed playin left 4 dead with my buddies (it works on linux/wine, but not with hdr enabled, with hdr off it looks pretty terrible.) the cd was too scratched up, so i couldn't install it. last month, in the anticipation for left4dead 2, and avp3 in the future, i decided to try windows 7. I just want to share my rediculous experiences so far.
 
Since i built this machine, linux distributions of all shapes and sizes have ran just beautifully for me. other than a few fixable hiccups with using sli, everything has ran as intended. so cheers to penguin heads.
 
Right off the bat, win7 took about 5 hours to install. The slow installation coupled with troubleshooting a freezing and non resumable installation process was an absolute headache. In the end, I had to disconnect all my hardware except for one of my graphics cards, and disable my usb controller via bios to get it to successfully install.:???:
 
Once installed, I've been dealing with the all to common bsod. This has been happening from a multitude of reasons, and some were addressable by running the machine in verifier mode, and tracking down bad drivers. However I'm at the point where it won't crash in verifier mode, so all of the crashes happen in regular running mode, and the memory dumps point to random files/drivers.
 
After searching the web all over, trying community help, I've come to a wall. I'm about ready to return the product, but i figured i'de try one last thing.... microsoft tech support! [-o<
 
 But when i called microsoft, i started wondering why i would ever pay 200 dollars for this.

The gentleman, god bless his heart, was a moron, ha. He was web searching for a debugger tool while using desktop control, and was clicking on ads stating "FREE! Windows Debugger Tool! 100% FREE!" I immediately grabbed the mouse and close it on him, he didn't like this.

I also copied all my mini-dumps to my desktop for him, so he'de have access to them, and he couldn't figure out why notepad was displaying strange characters, I had to instruct him to the microsoft debugging tool i already had installed.

His idea of troubleshooting was waiting for the blue screen to happen, I had to inform him that if it were to happen while he was connected to my pc, the desktop sharing process would be haulted and he wouldn't be able to even see it. He couldn't comprehend this, he said he'de be able to see it pop up, I continued to explain that the bsod is a system halt, and at the moment it happens, no further system processes are made except a script instructing the formation of the memory dump, then restart command. HE STILL.. didn't understand.

He went further on to disable system hardware, including my network controller, and then couldn't understand why he couldn't see my desktop.

This is all what really happened, and i'm dumbfounded by it. absolutely in dumbfounded. Beyond all this, He basicly randomly clicked around for an hour, checking things again and again, and repeatedly said things like, "ok, i'm clicking your start button now, see that?" "I've opened device manager now, see that?" to make it sound like he was actually going somewhere with it.
 
In the end he set my computer with nothing enabled, no networking, no video cards, no sound, and told me use the computer to see if the bsod happens..... I said,, so you want me to play solitare for a few hours? maybe write up a journal in notepad? wtf am i supposed to do with this machine!? #-o
 
Right now i'm waiting for some level 2 tech support dude to call me, i'm def not getting my hopes up and i have the microsoft product return number sittin on my desk. I'm just warning anyone out there thinking about getting this pile of crap, be ready for some headaches ):P
