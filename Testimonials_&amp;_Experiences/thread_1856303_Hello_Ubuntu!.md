### Post by zephonic on 2011-10-08
Installing Ubuntu 11.04 was a quick and painless experience. Very impressed, all the more because I have been hearing that installing Linux can be a bit of a crap shoot. Even more impressive is that certain functions of this Asus laptop do no work with the pre-installed WinVista64, but they DO work with Ubuntu! :D

I am a Mac user myself, but we had this laptop lying around and I decided to satisfy my curiosity and see what Ubuntu was all about. I used the Wubi installer and it could not have been more straightforward and easy. No additional drivers to install, everything worked from go, and the software center took care of a few updates that made the GPU work with the newer U11 GUI.

I guess people in the server business already knew this, but it was a bit of an eye-opener for me to learn that an *_open-source_* OS can perform on par with (or better than!) commercial releases.

I have shared my first impressions here: 

[http://mennodeboer.wordpress.com/2011/10/08/hello-linux/](http://mennodeboer.wordpress.com/2011/10/08/hello-linux/)


Thank you, Ubuntu!

I'm glad to see you're enjoying Ubuntu. I was a Mac user too, but got a bit tired of having to buy new computers every few years to keep up with the OS and hardware requirements for running new Mac software. You know; you want to buy an iPod, you need the latest iTunes - which only runs on the latest OS - which only runs on a better computer than what you've got.

Just thought I'd correct you on a technicality. "Freeware" is closed-source software that is free to use, and might have restrictions on how you're allowed to use it. Reverse-engineering or modification is prohibited. "Open-source" allows viewing and modification of source code, and most forms of open-source software don't have usage restrictions either.

Ubuntu is actually open-source.

I don't mind whether people use the term 'freeware' to describe open-source software, but some people get a little upset by it.
