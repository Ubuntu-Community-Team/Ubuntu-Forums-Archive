### Post by ntman68 on 2007-05-10
I know this forum contains mostly problems and requests for assistance, but I would like to say that I have been using 7.04 for the past several days and it has been an absolute pleasure.

I am a complete Linux n00b, but have 15 years experience with Windows networks. I grabbed some old HP desktop that was lying around, popped in the CD, clicked "install" and voila! Simplest OS install I have done since DOS. I really enjoy the included apps and have been able to use them with nearly all of the MS file formats. I am even able to perform nearly all the functions in SharePoint without a problem.. an impressive feat to be sure.

I still have a long way to go, however, as applications installs are still baffling for me, and I have no idea how to monkey with drivers and such. Luckily everything loaded "out of the box" for me on this HP, but if I wasn't that lucky, I'm not sure I could have figured it out. I anticipate those things will get better over time, however. I really appreciate the great message board and grass-roots support forums. That is tremendously helpful for the uninitiated.

Again, very pleasantly surprised indeed with my Ubuntu experience and look forward to using it more and more in the future.
