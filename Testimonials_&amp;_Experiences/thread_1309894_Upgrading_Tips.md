### Post by psidrum on 2009-11-01
Do not upgrade your system right away after a new version comes out, if your system is working fine, then stick with it, wait until a few months has gone by so all the bugs/fixes for the new upgrade have been fix, and then upgrade.

this is my first time using Ubuntu, i started out with Jaunty and it was great, but upgrading broke a couple of stuffi n my system, Compiz doesnt work now so i cant have a 3d desktop/ acceleration, its slower when 3d desktop is not working, and Firefox stopped working, doesnt even load

so dont upgrade right away if your system is working

### Post by running_rabbit07 on 2009-11-01
For best results, do not upgrade, but **do a clean install**. 

If you need to do an upgrade, because you have a lot op programs that you do not want to reinstall, download, burn, and insert the** 9.10 alternate installer** ISO while your system is up and running. A window will open asking if you want to upgrade. I upgraded this way with no problems. It would be interesting to know if anyone used this method and did have problems.

### Post by cariboo on 2009-11-01
If you do plan to do an upgrade, the first thing to do is remove any programs installed via ppa's as they may have newer libraries than the version you are upgrading to, you can always reinstall them after the upgrade is finished. Next comment out the ppa's, then make sure your distribution is totally up-to-date. 

After doing all of the above you should be ready to do an upgrade without any problems.
