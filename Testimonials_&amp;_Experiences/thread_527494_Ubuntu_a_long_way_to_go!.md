### Post by jfinkels on 2007-08-17
> **Coolbreeze! said:**
> hiya.

im still surprised you say Ubuntu is a breeze to use, maybe you've had no hardware issues like me and got a newer computer.  I've used spectrums, amigas, macs and pc oses and its still feels like any other Linux os i've used.

Let us know what wireless network card you have, and we can help you get drivers if necessary (or search the forums or look at [http://doc.gwos.org](http://doc.gwos.org) ).
> 
2.Haven't a clue how to install programs or utilities with the terminal, seems a bit techy for me. What are deb or apt files which im not to sure about.

If you don't know how to install programs from the terminal, then don't! Ubuntu makes it easy to install software using the Synaptic Package Manager (in "System > Administration > Synaptic Package Manager"). A .deb file is a package that contains files that can be installed on your system. APT is the underlying program that Ubuntu uses to install .deb packages (Syn*apt*ic, *apt*itude, *apt*-get). Read the link in my signature about installing software, and the psychocats tutorial.

Again, I can't emphasize enough that you just need to learn how to utilize the full potential of your new system (if you have the time and energy).
> 
3. When I drag and drop to floppy disk it don't seem to save it

You need to make sure to unmount your disk before ejecting it, because Linux writes files to the disk when it unmounts the disk.
> 
4. In some cases it says you need permission to write to hard disk.

Read the link in my profile about 'root' & 'sudo'. *nix systems use straightforward permissions systems. Each file has read, write, and execute permissions for the owner of the file, the group that the file belongs to, and all other users. 
> 
5. Maybe I'm not as techy as most ubuntu users.

Not true...many, many users are not computer scientists - that's the point of Ubuntu, "Linux for human beings." We are doing our best to make a usable system, but, once again, it is difficult for people to learn a new system. Stick with it and you will be pleased.
> 
what you reckon? how can you help?

cheers 

mark

:)

For each problem you encounter, first search the forums, second make a new thread clearly and concisely describing your problem, third search Google or one of the other sites in my signature.
