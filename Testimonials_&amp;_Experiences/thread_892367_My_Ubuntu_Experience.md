### Post by TransplantedCanuck on 2008-08-17
DISCLAIMER: I forget stuff, if I feel I missed a point, I'll reply and add it later. Don't go all hairy gorilla on me if I missed a point (pro or con). 



I want to post about my adventures with Linux somewhere here with several goals in mind.

1. Venting my frustration
2. The pathetic hope somebody will [read my thread and help]("http://ubuntuforums.org/showthread.php?p=5435179#post5435179")
3. Complimenting Ubuntu on winning over the wife despite the nightmare Ubuntu has been/still is

I'm not going to touch on problems I ran into with compatibility (my TV card).

About me:
I've studied Windows, Novell and Cisco administration, but never learned linux. I haven't managed to land an admin job yet, I moved to a foreign country and learned the language and took the first job I could get (the only company from hundreds that even bothered to respond).

I'm a gamer and a happy windows user. I know its crap, and I know why. But I work around it well enough, and I had Vista installed for 6 months (vista = 64-bit driver support) without antivirus because I forgot, and still had nothing. (firefox, no-script, adblock and smart browsing).

About 3 months ago, the harddrive in my wife's computer started to die. I thought "why not install Linux". It'll be useful for me to learn, is supposedly really easy now-adays and my mom (who I helped to install it 2 years ago)loves it. When we installed it for my mom, the only problem we had was her wireless. Know issue. O-K.

So we slapped a brand new 500GB HD in her computer, I shove in the Ubuntu CD I had lieing around and install it. Silly me, its Ubuntu 6, so after doing 2 version upgrades, things are off to the races.... install pidgin for MSN/ICQ, install some other software that came reccomended.

But wait... theres no sound. Sound Blaster Audigy, nothing that obscure. Sure its creative but...
After 3 days of googling and smashing my face on the desk I found this thread: [http://ubuntuforums.org/showthread.php?t=205449&highlight=audigy+sound](http://ubuntuforums.org/showthread.php?t=205449&highlight=audigy+sound)

1-down.... countless to go. We tried about a dozen music programs, Amarok was ugly, a few others were unimpressive. Quod Libet was great, except for a few bizarre quirks that drove the woman insane. She's using Banshee now, and it still has a few problems, many I'd call "thats so obvious I can't believe its not in there". (e.g. Refresh library option).

The wife is a hobby photographer. She was in love with Adobe Lightroom, but mostly used XNview. I tried to install the Linux version of XNview in Ubuntu, but it must be for a different distro, because despite using google and the search function here, I couldn't find any way to get it installed (except in Virtual Box on XP). Nothing comes close to Lightroom, but we're ok with that. Gkam gets the job done. What is frustrating is the lack of programs that let us delete all the metadata for photos to make them smaller for emailing to friends. Theres plenty of auto-taggers and mass-commenters and whatever else that we don't see a point for, but no simple "wipe the stuff out" options.

Virtual Box: Where Ubuntu isn't enough, we run windows in a box. Makes sense right? At the very least because some of the scientific software she uses only runs in windows. Well, its not quite that easy. Everytime Ubuntu prompts for updates (more on updates later), it breaks Virtual box for the next few weeks (Linux-headers or some crap). Solution: Reboot into the older kernel. Well, what was the point of updating then!? Theres also random times where Virtual box simply won't open, no error message, the process simply runs in the task manager thing, but never opens on the desktop anywhere.

Updates: The auto-prompt-to-update feature, needs to be turned off. Windows updates are notoriously pointless/unreliable. But I've never had one totally trash my system or the software I use on it. Service packs: yes, normal updates: no.

Ubuntu updates however, regularly break software we use (pidgin: once Virtual Box: every-time). We also are the proud owners of a "buggered to hell" Ubuntu system, after installing updates. Half the administration tasks won't open (see my thread for details), programs don't work, strange errors occur and it generally makes me want to reinstall windows (she won't let me though). I've made several threads, posted info in several others and tried about 4 different places. Nobody seems to be able to help with this problem. Go-Ubuntu!

Sadly, after the nightmare it was to get Ubuntu working for us in the first place, I'm loathe to wipe it out and start over (who knows what will go wrong this time!!). But that appears to be the only option, except my free-time is somewhat small, and taking several days dedicated to trying to restore some working order to the system is not high on my "things I want to do" list.



Now to the good: 
- Despite all of the above frustration, bizarre happenings and everything else, despite the fact that my wife has to get me to install everything and I'm constantly bombarded with a stream of frustration as common/simple tasks in most programs (refresh library in banshee!?) seem to be the first thing to be forgotten by the developers of linux software, despite the fact that Ubuntu seems to do SOMETHING aggravating every other day. She's hooked. You must have done something right.

But theres a long way to go before you'll get deeper market penetration. Not every user is as patient as my wife, nor as generally competent. And not every user has a raving computer geek waiting in the wings to fix things on a daily/weekly basis. (or after every update).
Not every user who is competent has the patience to spend 5 hours googling/forum searching/thread bumping for problems every week.
