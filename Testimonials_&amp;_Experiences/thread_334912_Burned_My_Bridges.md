### Post by deejoshy on 2007-01-09
I suddenly became so annoyed with the total cost of upgrading to Vista (software plus memory chips) that I decided to take the Linux plunge.  I never even heard of Ubuntu until one week ago.  Now I am addicted. 

I decided not to go for the dual boot strategy.  Instead I backed up the files I wanted and wiped the disk clean with a 6.06 install.   It went totally smooth.  Didn't take long, either.  Yes, there are new things to learn but I like learning so it's no problem.  Still haven't figured out how to play a DVD movie (missing a codec) but I'm reading up now on packages and how to install them and am learning the shell to, which I like very much.  It's like getting the keys to the kingdom.  I'm still immersed in my Ubuntu Linux book.

This website is awesome.   The community is super helpful.  You guys and gals are all on to something fantastic.  

My roommate, who has an XP machine, thinks I have done something really, really scary and daring.   Isn't that hilarious?  

I love the way I don't have to worry much about viruses because of the root protection.  I never knew this.  Makes me wonder why I ever settled for less for so long.

It's amazing how my old machine suddenly feels new again.   It actually runs better than it ever did.   

Well, that's it.  I'm sure I will run into problems here and there but I know where to come for help.   :D 

Davito
Washington, DC

### Post by 3rdalbum on 2007-01-09
For your DVDs, you should look online for "libdvdcss", preferably as a Debian package. I think Automatix gets it for you as well.

Then install GXine, MPlayer, or VLC from the repositories. If you're using Kubuntu, then Kaffeine will work too.

### Post by seijuro on 2007-01-09
all you need to do for dvds is install libdvdread3 package and run the script /usr/share/doc/libdvdread3/examples/install-css.sh which downloads and installs libdvdcss and ofcourse have a player that plays dvds (xine, mplayer, etc.)
