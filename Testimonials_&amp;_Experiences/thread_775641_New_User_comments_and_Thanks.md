### Post by optique on 2008-04-30
Last night I installed Ubuntu 8.04 for the first time. What a nice experience!

The install was great. The desktop was great, and the initially installed software was impressive.

My integrated NIC was automatically discovered. My display worked first time, and I was able to install the Nvidia drivers with a few clicks.

I was able to play music cd's, watch a video, copy stuff off a flash drive, etc.

If I had to pick one thing that most impressed me, it was the easy installation of software. 

I just want to give a big thanks to the folks that made this happen.
