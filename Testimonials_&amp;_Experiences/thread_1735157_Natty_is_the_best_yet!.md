### Post by drewesq on 2011-04-21
OK, so I have been using the Natty Beta now on my eee pc 1008HA for a few days and thought I would share my experiences

The unity launcher is great, you get full screen web browsing as it has the dodge-windows-hide-feature, it is smooth and surprisingly unbuggy. You can customise what goes on there and the size of the icons. They also have right click jump list feature like win 7.

The integration of the menu bar into the panel for full screen windows works really nicely too. I also used to find the panel a bit buggy on 10.10 NBR, but it is much better on Natty.

I also have to say that everything worked, but I upgraded from 10.10. 

2 finger scrolling is now supported which it wasn't in 10.10

Small thing but I really like the controls on the panel, they all work very smoothly and I like the blue indicator when you get new emails IMs!

The system settings menu addition is great on the power icon on the panel.

I also have to say that banshee is very user friendly as are the dash and application menu.

All in all, the beta is very stable, I haven't had any crashes yet. The whole experience of Unity is much better than when I started using 10.10 NBR, not much to improve IMO.

Verdict: If you haven't tried Unity yet then you're in for a pleasant surprise and if you have been using 10.10 and waiting to upgrade you're in for a treat!!

):P

PS. I am sure battery life has improved and Natty seems to be less processor intensive.
