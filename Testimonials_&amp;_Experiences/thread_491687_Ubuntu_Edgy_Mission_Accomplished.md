### Post by zero244 on 2007-07-03
Ive been using Ubuntu since early January. The only two Ubuntu distros Ive tried is Edgy and Feisty. Ive had many problems with beryl and compiz. All my crashes were related to them.
I don't think Ive had a single crash or freeze up just running regular Gnome.
I have finally got a solid install of Ubuntu with Beryl running full bore.
All my beryl problems were related to a bad Nvidia card which I sent back for replacement today.
I had a new 6200 card sitting around for backup. After I installed the new card all my beryl problems vanished. Not a single problem since. I am running beryl version .2.0 with XGL.
The past six months I have only had to install Ubuntu 3 times and one of those times it was because I wanted to try a fresh install of Feisty.
I went back to Edgy because it seems to run very stable on this machine.
So Ive had some ups and downs, but I weathered it and am still standing. lol
Unfortunately I have to keep XP around for my MP3 players and scanner and gaming.
But I am really glad Ubuntu has worked out because it is overall a better OS, and less problematic.
I am no command line expert but I can get myself out of a few jams if the need arises.
Ive learned a lot about Linux and about computers in general.
My computer is running so good right now, I really don't have to tweak or do experimental stuff.
So until my motherboard dies I should be good for a while.
So if you want to get away from MS just keep plugging away with Linux you will eventually get to that sweet spot of success.
Its so nice not to have to worry about activation, viruses etc. 
When you look at the Linux OS and other free software you realize what a good job all the open source programmers do.
