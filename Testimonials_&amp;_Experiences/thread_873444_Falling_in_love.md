### Post by ingeva on 2008-07-29
This is not a support request.
Instead I'd like to share some of my experiences with Ubuntu so far.

Today is Tuesday, I started installing Ubuntu on Saturday. After 3 days I feel like I've used Ubuntu forever. Getting rid of old Windows (XP) will evidently make life a whole lot easier, and the only regret I have is "Why didn't I do this before?"

There are challenges, of course. I had to re-install because I met a brick wall with configuring Apache (See the "brick wall" threads in this forum) but after that most worked as a dream.

Evolution seems to be a little primitive. For instance I didn't find any way to change ports for my SMTP servers. I downloaded Mozilla Thunderbird and it has the necessary features.

Copying old messages from Outlook Express was not easy. I chose to move the most important messages to "inbox" and mail them as an attachment. I plan to "fool" my server inbox by copying the file to it so I can download all messages as regular mail.

Also the address book is clumsy. The nearest you get to a compatible format is csv, but exporting the various folders from Windows' address book is a slow procedure because it won't export from all folders in one go.

I set up Samba (Also a breeze!) so I could get easy access to my old files while working with the new computer (preferable to dualboot IMHO). However, Ubuntu can also access NTFS files...:)

Some programs are different from what you're used to. There are other keys to click, and for instance "Backspace" is ignored by FireFox, so unless there's a secret button somewhere going back to the previous site can only be done with a mouse click.
Same with gedit:  If there's an easy way to change tabs, it's certainly well hidden.

To the new user: Don't be frustrated by some start-up difficulties. Linus WILL make your computational life a whole lot easier and less troublesome. There are always someone to help in the Forum (I got all the help I needed in a very short time!)

In short: I love you Ubuntu!  Will you marry me?  :KS

### Post by hyper_ch on 2008-07-29
OE --> Thunderbird

do this

(in windows)

(1) Install TB

(2) Run TB

(3) Go to Extras (I think it's Options in english) and there you have an import assistant

(4) import messages, addressbook and settings into TB

Now you can copy the thunderbird windows profile folder over to your linux one...

in Windows it's like C:\Documents and Settings\[User Name]\Application Data\Thunderbird\Profiles\
and on linux it's /home/user/.thunderbird/profiles   (I think)

Basically just copy it over incl. the profiles.ini
