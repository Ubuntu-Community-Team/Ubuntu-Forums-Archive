### Post by apocalypsxx on 2008-03-13
Well after trying to configure Apache & php after downloading using
sudo apt-get install etc...(which I have to say worked fine) I used the synaptic manager to download & install for me..
Within 5 mins I had it all up & running & php working fine.

[http://localhost/phpinfo.php](http://localhost/phpinfo.php) :) showed all the system info as it should.

So if any other new guys out there want to get these packages running with minimum
headache I would recommend using
System>Synaptic package manager..for now untill you become more conversant with Ubuntu & linux in general..

BTW thnx to the guys who did help me before in a previous post..The problem wasn`t with your advice but my lack of understanding...:lolflag:

*One Happy Ubuntu user!*:)
