### Post by WinterWeaver on 2007-10-20
Right, so I'm not exactly new, but I've not yet added my wee post here.

very quick history...
I've used the following systems as I grew up... Dos, Win 3.1, Win 95, Win NT, Win ME, Win 2000, Win XP.
Throughout this period, I tried whichever was the popular flavour of Linux at the time. I always liked Linux, but could never really do what I wanted, and for me personally was a bit too complicated to use (I thought I knew everything cause I knew Windows so well  :P ).

So ... last year, I was talking about Open Source Software with some friends (I was using Open Office in Windows XP at the time) and they mentioned that they have a Open Source CD with all sorts of other open source stuff on it.
It was a Ubuntu CD (though I didn't know at the time I could boot from it and install ubuntu) --> I ended up downloading Dapper.

I've been upgrading on every new release... Dapper >> Edgy >> Feisty >> Gutsy, and I have to say... WOW.

Ubuntu was the first Linux distribution which made my transition to Linux easy (relatively). Now with the latest install, Gutsy, I'm looking back, and I'm amazed at how fast progress is going on this Distro and Linux in general (compared to other OS's)

I am aware that there are many out there, that don't think Gutsy is enough of a improvement from feisty, but personally, I have to say that it's definately the best one yet! Everything, including Compiz setup etc was a DREAM!!!

The only problem I have atm, is the mouse dissapearing every now and then when within a busy window (ie. when firefox loads a web-page), but I can live with that for now. Still, I'm amazed at how everything works so wonderfully out of the box!.

So BIG Thanks to the Developers for your hard work!!
Hopefully I'll be able to contribute more than just answering irc and forum questions in the future (learning python and zope/plone atm :P )

Regards

WW
