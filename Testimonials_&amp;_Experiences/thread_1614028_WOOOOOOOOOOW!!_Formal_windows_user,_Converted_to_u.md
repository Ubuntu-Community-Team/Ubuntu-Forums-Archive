### Post by P4man on 2010-11-05
Good to hear. Now burn an ubuntu iso to a cd or USB stick, and do a proper install instead of wubi. You will thank me later. Or maybe you wont, but you will curse later for not having done it :p. 

Wubi relies on windows file system, its a virtual filesystem on your ntfs drive. That makes installing easy but it makes the setup very fragile and dependant on windows, and wubi generally often causes problems (and believe it or not, costs some performance). If you ever run in to trouble, and you will, having wubi rather than a native install makes things so much harder to fix as well.

Wubi is a good idea for anyone wanting to try it out fast without having to partition, but dont use it for everyday computing.
