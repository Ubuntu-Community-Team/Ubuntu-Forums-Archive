### Post by trobn76 on 2008-09-02
I downloaded Ubuntu 8.04 64-bit and then burned it to a DVD and ran the installer on my USB drive (regular guided install). As soon as I rebooted and changed my boot device priority back to 1.) Floppy, 2.) USB drive, 3.) CD drive it booted as normal with no problems. 
Can't say that it seems much faster to run it on a USB drive than to run it on a regular HDD, but I haven't really compared so...

This is my build:

Asus P5K mobo
Intel Core2Dou E8500 wolfdale 
4Gig of RAM
4GB Kingston DataTraveler USB drive
Sapphire ATI Radeon 3870 HD
