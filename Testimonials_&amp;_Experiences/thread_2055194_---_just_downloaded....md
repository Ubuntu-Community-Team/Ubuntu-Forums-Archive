"Wubi" stands for "**W**indows **Ub**untu **I**nstaller" and puts Ubuntu into a file on the Windows NTFS filesystem. Not the same as a "bare metal" install, and should always be specified here because the 2 can vary greatly as to how problems are addressed at times.

About the "freezing", did you check the download? Though if I recall correctly, Windows may already do that for you automatically, not sure though. 

Another point to consider, is to burn the installation media at the lowest possible speed to reduce any chance of errors. And did you test your hardware with the live cd or just install immediately? If your hardware is old (you mention "Using an older Windows XP computer") you may need to consider a lighter flavour of Ubuntu like Lubuntu or even another more suitable distro for you hardware.

Hope this gives you some points to consider and helps you out in the longer run.
