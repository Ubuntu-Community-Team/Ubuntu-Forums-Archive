### Post by Myrddin Emrys on 2012-02-15
If you still have that 11.10 installation around, try MATE:

[http://mate-desktop.org/](http://mate-desktop.org/)
[http://wiki.mate-desktop.org/download#ubuntu](http://wiki.mate-desktop.org/download#ubuntu)

It's the Gnome 2 desktop you are using right now, forked and re-named so that development can continue independently. You can install it from the developers' repository  in a few minutes with just three commands:

sudo add-apt-repository "deb [http://packages.mate-desktop.org/repo/ubuntu](http://packages.mate-desktop.org/repo/ubuntu) oneiric main"
sudo apt-get install mate-archive-keyring
sudo apt-get install mate-core

### Post by kohoutek1 on 2012-02-18
About a month ago, I switched to GNOME Shell. Before that, I was using KDE 4.8. 

I'm an old, left-handed, top-toward-bottom looking Mac OS X user, and what I'm after is a DE that most closely simulates the OS X look n feel. As a Mac user, I don't need all the baroque, "look we can do this" bling of Unity. 

I just need a panel on top because that's where I naturally start looking, a tidy place to store my favorite apps, preferably away from where I execute OS functions, which would probably be the bottom, and -- most importantly -- all functions within one mouse click away from the desktop. 

By default, KDE is not like that, though after some configuring, I was able to get it close. I've found that GNOME shell is far closer to that Mac standard than Unity is. GNOME Shell is certainly elegant and simple in concept.
