### Post by lancest on 2010-08-16
I prefer the new Light theme work that to be included in 10.10. Running it now in Lucid and loving it. 

The new Light theme is much better looking than Radiance- and Ambiance is improved. Keep up the good work Canonical! 
Wow- I could never believe OSX would look this good.

For example Nautilus has a slider and and disks are better organized below the file system on left pane. *Lots of other improvements like a better looking sound file preview graphic.*

All I did for such Lightness glory was: 

sudo add-apt-repository ppa:murrine-daily/ppa 
sudo apt-get update && sudo apt-get upgrade

Only saw one drawback so far: Open Office Calc tabs are a bit small. A work in progress.
Edit: Some of the select functions don't highlight well in Nautilus panel and Evolution.
Anyway no problem for me.
