### Post by SEisch on 2011-10-16
Since January I have been using Ubuntu 10.10 32 bit as my main operating system.  Now that I have gotten used to it, I like it.  I find it do be very user friendly to people like me, who don't know much about software or programming.  I haven't had to download many programs since installing.  The default software packages cover most everything I need.
If Ubuntu still has a 32 bit version for their next long term support version, I am definitely going to give it a try.
If they stop making 32 bit versions, then I 'll just switch to my #2 favorite OS Puppy Linux.
It is the only other system I've found to be easy to install and use.  Mostly because you don't really need to install it.  Works fine off a CD.
All and all, I have to say good job Canonical.  I'm sticking with 10.10 until it doesn't work anymore.
