### Post by ad_267 on 2008-08-24
Welcome to Ubuntu!

That's brave doing a complete format. I still have a windows installation lying around on this machine somewhere but haven't used it in months.

Just be aware that Ubuntu isn't for everyone and there are still issues with support for some hardware as a lot of companies aren't that bothered about providing Linux drivers. That said, Ubuntu works great for me and many other people without any problems.

Make sure you read up on installing applications from the repositories as that's something that might be a bit new. Have fun!
