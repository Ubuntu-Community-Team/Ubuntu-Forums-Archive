### Post by asgard_command on 2007-10-09
I,ve been a long time user of open source software, Firefox, Gimp OOo. So it seemed natural to more to a Linux system as soon as possible. I tried a few years ago but pre broadband was just too slow to download but now it's great how quick you can download a whole distro and extra apps.

I tried the Ubuntu live cd and was so impressed I took a small partition from windows and went for a full install.
I should say that my laptop came with Vista pre installed and unlike many I have been very happy with it other than a few small apps not being compatible.

The Ubuntu install is amazingly smooth, I couldn't believe how much worked out of the box. I needed some extra little trick to get wpa security working for my wireless but everything else seemed to work fine. All the main programs I was already using where installed for me.

I would love to dump vista and turn whole pc over to Ubuntu there are just a few things stopping me. I don't like dual booting and really hope I can sort everything out before I get tired of it.

I thought the cube desktop was great but that seems to have stopped working no idea why. (minor issue no doubt will get better soon)

My sony mem stick reader doesn't work, I can live with this though I did use it loads. (have heard this may be fixed in Gutsy)

I have a Humax hard drive PVR that I had a small utility under windows to transfer tv from it to the pc. I have found a linux version but it's source code install and despite following instructions hasn't worked.
In also need to find a nice basic demux program and mpeg editor for the tv programs I used to use PVAStrumento and Cuttermaran on windows, but I haven't found good alternatives yet.

Despite reading up and installing every video player and codec I have seen recommended I can't play wmv files properly.

I tried to run one program under wine a cd based specialist encyclopedia type database but it didn't run.


I am still learning and hope to get to grips with my minor problems so I can stick with supporting such great opensource projects.
