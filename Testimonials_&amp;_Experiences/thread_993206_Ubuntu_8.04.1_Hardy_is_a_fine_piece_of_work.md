I'm sure that all in this thread understand this, but I'll point this out to newer users who are reading here.

The point releases (.1, .2) are not newer versions of Hardy. 8.04 is continually updated. The point releases are just a download point that includes all of the updates up to the "point". This alleviates a lot of time applying updates if you happen to install from the original Hardy disk.
