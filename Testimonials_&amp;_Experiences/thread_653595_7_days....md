### Post by fullmetalgerbil on 2007-12-30
I installed Ubuntu (7.10) a week ago and I never want to use Windows again. Sure theres been some snags, but thanks to this forum and the Ubuntu community I've gotten all the help I need towards a smooth running system.
 I've only owned my own computer for the past year and a half and though I've wanted to run a Linux OS for about a year and three months now I didn't think I knew enough to handle it. Plus, all my friends are die hard Windows users and they tried to deter me by saying Linux was really difficult and everything had to be run by command line. Finally though I worked up enough courage, and got sick enough of Microsoft, to migrate to Ubuntu. And its been a hundred times easier, from install to working around a few minor quirks, to everyday use, than I ever could have imagined. Yet, at the same time just from working around those quirks and reading onboard information in the Ubuntu Help center plus posts here at the forum as well as assorted Linux blogs, I feel I've actually learned more about my OS in a week than I knew about Windows in the year and a half I used it. Not that I'm still not living in n00bville when it comes to Linux, but using Windows left me more of a spectator and with Ubuntu I just feel more engaged in a sort of D.I.Y. personal project.
Also, I'm proud to be running an open source OS because I agree with the philosophy and now I'm doing something more than just ideologically supporting it. The fact that now I've been given the opportunity to have a platform whereby one day I might be able to contribute to the Linux community makes me feel really good. I'm a long ways from having skills like that but I look forward to it.:)
