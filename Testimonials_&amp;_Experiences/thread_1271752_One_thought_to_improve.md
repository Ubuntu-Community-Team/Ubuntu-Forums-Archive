### Post by 3rdalbum on 2009-09-21
The problem is not with the forums, nor with Ubuntu. The problem is that many users are trying to install Microsoft Office in Wine, and running up against a known bug in the Office program that prevents it from installing.

Still more users attempt to install DirectX in Wine, and run up against another Microsoft bug that causes blank screens.

Unfortunately Microsoft does not have a public bugtracker, and does not advertise a way for users to report bugs. I know, I've tried before now!*  So the best option that many people have is to go to the Ubuntu forums and let other people know about the problems with Microsoft software, so others can work around it. Because we can't fix Windows, because it's not open-source.

* I attempted to report a bug to Microsoft whereby filenames containing question marks and other special characters on an NTFS-formatted disk could not be accessed in Windows. Microsoft's own NTFS specification document says that those characters can be used, and ntfs-3g has no trouble storing those files and accessing them on Linux. Windows does not conform to the specification and I wanted to report this to Microsoft; but there is no public bugtracker nor any e-mail address nor phone number for bug reporting.
