### Post by Lateralis on 2011-05-05
Sharing files: Use a data partition formatted as NTFS to share documents between both Windows and Ubuntu.  Both OSs can read the NTFS drive and make modifications as necessary.  

If you want to share browser bookmarks, e-mails etc, yeah that can be more problematic.  But soft links and various import/export tricks can help you there.  

Playing games: This is always going to be a problem so long as game makers only consider building games for Windows.  In order for game manufacturers to take Linux gamers seriously there needs to be a serious number of Linux gamers.  But no gamer will give up their games so its a bit of a viscous circle.  There are a few clients and programs that you can try though.  Wine does work with varied levels of success with a lot of games (Check out [WineHQ]("http://www.winehq.org/")).  You could also consider using Crossover Games by CodeWeavers and PlayOnLinux.  There are also *some* games out there that have been built with Linux in mind.  

Print server: Never used one.  I print on a network and that works flawlessly.  But never used a print server.  I doubt though that you're the only Linux person using a print server though - perhaps you should try making a specific thread on that very specific topic?  


In truth, the only way you'll feel comfortable in Linux is if you jump in and immerse yourself.  The more you use Linux the more you will get used to it and the easier finding solutions will be.  At first it will take you a little longer to do things than in Windows, as you've probably only ever used Windows and Linux is alien.  But in time everything will be second nature.
