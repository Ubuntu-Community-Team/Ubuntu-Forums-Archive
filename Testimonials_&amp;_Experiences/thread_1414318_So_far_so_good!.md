### Post by MSich on 2010-02-23
I just thought I'd drop a line to say how pleased I am with Ubuntu.  I was a Fedora user for 2 years and recently (after a very painful upgrade) decided it was time to try something different.  I'm glad I did.
 
I jumped right into the Karmic install. (Actually, if I would have read through the forums first, I probably would have installed Jaunty instead.) But, I've had zero problems with Karmic and I really like it.
 
I haven't had to post for help, which is a good sign, I guess.  But the forums really helped me with getting my printer setup (user error, go figure).
 
Since then I've been reading the forums a lot and learning a lot.
 
I'm certainly not a power user, or a tinkerer.  Just a normal daily user, OO, Kino, FF, email, and Vbox with XP guest to run something my son needs for school, and Rhapsody to-go.
 
Anyway, thanks, and I'm sure I'll have to ask for help someday, but I know I'll find the answer here.
