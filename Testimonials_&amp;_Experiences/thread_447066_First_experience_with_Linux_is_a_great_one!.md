### Post by Bohlio on 2007-05-17
I'm new to Linux and I decided to Install Ubuntu Fiesty Fawn a couple days ago. Previously I had been running Windows XP Media Center Edition and I thought it was just fine. Not wanting to give up XP all together yet, I looked into dual booting both operating systems. I found an easy guide for partitioning the hard drive correctly with the Ubuntu CD and it went without a hitch. The setup was quick and easy. Next, I had to get my wireless network working, that's where I ran into some problems. I couldn't figure out how to make it work! I figured out that the card I was using needed ndiswrapper to make it work. After some searching, I found a guide from these forums and made it work. Now, I use Ubuntu for regular tasks like word processing, listening to music, and browsing the internet. All I use the XP partition for is gaming and watching TV. I never really felt the need to record TV so if I could find a simple program to just watch TV under Ubuntu, that would be great. All I'd use windows for then is playing C&C3, Until of course, I find an easy way to set it up in Ubuntu.

Short version... Ubuntu is incredibly easy to install, these forums are a major help for getting things up and running correctly, and It is a great operating system that I don't even have to pay for. I love it :)
