### Post by geirknappen on 2012-05-23
I have been using ubuntu 10.04 for a while now. I didn't plan to try ubuntu 12.04 until the point release 12.04.1, but i was curious. I downloaded the iso and put it on the startup disc creator. When I tried to boot from this usb i got an error, but I googled and found a solution [1]("http://alexsleat.co.uk/2010/11/27/how-to-fix-unknown-keyword-in-configuration-file-ubuntu-usb-boot/")


Then I installed ubuntu 12.04 from this usb to another  usb (32 gb). Then I had to fix broadcom wireless problem [2]("http://ubuntuforums.org/showthread.php?t=1974006"). I tested it for some minutes. I didn't get how I would handle  many  open documents at the same time in unity, and couldn't see any hide all windows button. No option to put the unity bar at the bottom. 

For some reason I got the feeling that the programs where taking a while to start. Much slower than ubuntu 10.04. I thought that might be because I was running ubuntu from the usb. I decided to install ubuntu on the computer hard drive. I did the system update thing, and continued testing, but I still got the feeling it was slow. I started the software center, but it froze, and I tried to find the force-quit thing. A lot of stuff to get used to.

Perhaps I have done something wrong, I thought. Then I remebered that I had not yet installed the recommended nvidia driver, and decided to do that. When I restarted, unity wouldn't work and the screen went black. I had also chosen: "do not require password to login". What to do now?

I decided to install ubuntu 10.04 back to my computer harddrive (and thats the only thing I got. That way I avoid seen the grub menu thing). I like 10.04.4. I don't know much about computers and programming, but I think it is interesting to read system updates when they come. There has been a lot of security updates, they don't come that often now. Perhaps ubuntu 10.04.4 has become quite secure. I don't have anything secret on my computer that I can think of, but secure computer is probably a good thing anyway. I don't know what I will do in the future. In April 2013 I probably should find some replacement for 10.04
