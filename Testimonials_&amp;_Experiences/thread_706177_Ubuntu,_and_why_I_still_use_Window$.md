### Post by masque7 on 2008-02-24
(Please don't shoot me down, I'm not 100% sure if this thread belongs here, feel free to move it. :))

I am in **no way** intentionally comparing *ubuntu to XP. I suppose you could say the purpose of my thread is to gain more knowledge, and see if anyone else is in the same position as me, and to share my perspective. 

I grew up using Windows machines… Why do I still use Windows (XP)?

I’d say post  98 era, this is when I started to experience the security issues of Windows. How the average user would be threatened by malicious codes, virii, spyware, etc. Was I aware of any other alternative? As many of us will answer, no I wasn’t…

I'll spare you the details, the reasons I still use Windows are:

[LIST]
[*]Specific software: Photoshop, Dreamweaver, Flash, DVD/AVI software/converters, online TV/streamer applications (Sopcast, TVAnts, etc). 

[*]Games: I’m not really much of a gamer, I don’t have the latest hardware. Wine is great, but quite honestly I’d rather just play my games in maximum FPS, and easy installation, etc.

[*]Sound/hardware: I’ll put my hands up and admit I don’t understand what I’m doing when it comes to scripts and .asound files to enable 5.1 sound in all applications/duplicate sound into the other speakers, etc.
[/LIST]

I haven’t suffered from security issues with Windows for  a few years now, thanks to open source software, and my own knowledge. Sure, I still get infected, but I haven’t had any major problems. It does annoy me that with all the time, hassle, and money spent in the past  to protect myself, I could have spent, learnt, and used that time to learn something else. Having to spend loads to get the latest anti-virus and firewalls (unaware that free software existed), our computers repaired, all the ‘computer repairmen’ just thriving  off the lies, deceit, and corruption that M$ had created. I sometimes think M$ intentionally do this to us. I think they leave holes in security for business purposes -  as I’ve gotten older by a few years, I’ve been able to see that. This is why I use Ubuntu. Sure, my dual-boot and virtual machines allow me to have the best of both worlds, it’s just a shame I keep going back to XP for the aforementioned reasons. I’m just glad I don’t support M$ or any company asking me to pay for anything ;)

I love how it’s not just an O/S, it’s a whole ideology, it’s a community that supports itself. 

Big thanks to all the developers, and the community itself, I’m glad to be a part of it. So I’ll ask you all, any suggestions?

### Post by mrsteveman1 on 2008-02-24
I use vista on this laptop (and gutsy and hardy separately), and the majority of my Linux problems have to do with interfaces and behavior.

I'm very used to the windows explorer for managing files and for some things it works better than nautilus or konqueror or dolphin. When you have to manage hundreds of thousands of files on remote SMB shares quickly, explorer seems to work the best. KDE4 shows some real promise though.

My other problem is partly psychological, i fly through interfaces very quickly (yea i have ADD :D ). I find that, at least for the moment, i can use the windows interface easier than i can use gnome or kde. OS X is even worse for me and feels like using the mouse at the end of a 10 foot pole. 

I tend to not have much problem with applications, i took photoshop classes in college and can just as easily use GIMP given a few minutes to get used to where things are. I recently used GIMP to develop some graphics for a friends website and had no problems at all.

The sound stuff is about to change a lot in Ubuntu, both the kde and gnome versions are implementing new sound subsystems which should make things a lot easier for users. 

I would like to point out that you give Microsoft a lot of credit where simple incompetence explains things perfectly. They simply aren't that good at managing an entire operating system and have proven to be incompetent on a number of occasions. The original Internet explorer was purchased by MS and then subsequently ruined. Windows XP, when released, came with every port open, no firewall, and had highly vulnerable network services listening on ports 135-139. The services running behind these ports would happily allow an attacker to compromise the machine remotely, and Microsoft didn't do this intentionally they just have no idea how to manage an entire operating system and gave no thought to security before making horrible decisions.

### Post by Mr_Tricorder on 2008-02-29
I have a desktop and a laptop.  They used to both be dual-boot systems, but now I only have ubuntu 8.04 alpha on my desktop and Windows XP MCE on my laptop (this was due to a series of unfortunate events involving a hard drive failure on my desktop and a botched installation of ubuntu 7.10 on my laptop, resulting in a full system restore).

Except for the inability to play several of my games that won't run well in wine or on my laptop's hardware (damn Intel graphics chipset), this works out nicely.  I'm very pleased with the way open source alternative apps have been developing, like GIMP, Open Office, vlc, rhythmbox, etc. and I use Virtualbox to run any Windows programs that I can't get to work in wine (except for games, but I think they're working on a way to enable accelerated graphics, which would make me very happy).  I haven't had a single sound problem with Linux for over a year now (although I don't have a 5.1 sound system, so I can't speak for your issue on that).  It's really amazing to see the progress Linux has made over the last few years.  Most of the real issues holding Linux back from being a serious rival to Windows and OS X are because several companies, especially gaming companies, don't want to port their software to Linux, but that has been slowly and steadily changing.

I need to keep Windows on my laptop anyway because the wireless network on my college campus will only work for Windows and OS X, plus it's better than using Virtualbox for Windows-only software since I have a lot more storage space on my laptop's hard drive than on my virtual hard drive for Windows on my desktop.
