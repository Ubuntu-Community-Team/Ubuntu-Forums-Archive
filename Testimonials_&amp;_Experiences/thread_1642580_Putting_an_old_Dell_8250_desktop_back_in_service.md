### Post by ricky58 on 2010-12-10
Hello All, 

I am new to the Linux OS and its various distros, but have been thinking about diving in and trying this OS. Admittedly, it has taken awhile to sift through the many variants to find an OS that would work and provide the amenities I wanted (Office, etc.). My objective was to revive this old Dell with only free software and, I feel I have succeeded. The system in question was an old Dell 8250 desktop with Pentium 4 processor 2.40GHz; 512MB ram 2.39 GHz, 112GB internal hard drive and a Netgear wireless card. The OS was slow and unstable.

After doing a little research, I settled on Xbuntu 10.10 for the operating system since it appears to have the OS stability I wanted and the out of the box install. Since I was going to donate the finished computer to a senior&#8217;s center, I thoroughly purged the hard drive using disk purge at the DOD level of purging. That process took 4 hours to do.

I downloaded Xbuntu onto another computer and did the md5sum check to make sure I received the whole file. Everything checked well, so I burned the file image to disk using the windows program for burning a cd. For some reason that method did not work and would not boot the Dell. So, I used another program called ISO Recorder [COLOR=red]http://isorecorder.alexfeinman.com/isorecorder.htm[/COLOR]. Burned a new cd and it worked like a charm.

After answering a few questions and telling it use the entire disk everything went like clockwork. All of the drivers needed were there which was nice. Even the Netgear wireless card works better with Xbuntu than it did with MS Windows XP. I also downloaded the 157 updates to insure security and stability.

Later after checking thins out, I added Office (not MS) and a couple of card games and Shotwell a picture organizer.

I have an HP DV5000 laptop I may get rid of Windows XP and put Ubuntu on it. 

So far, I love what I see. Now just to test it out for bit before I donate it.
