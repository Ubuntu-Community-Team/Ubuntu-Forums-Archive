### Post by polo_step on 2005-10-14
I'm on the 5.10 "Live" CD right now, booted on my normal Ubuntu computer.

Remarkably, everything seems not only to work, but it looks better than in the previous version and a lot of the little earlier annoyances aren't here.

On the recommendations of users here when I did the last install, I TRIED THE LIVE CD FIRST to see if there were any gross problems.  I downloaded it and checked the MD5 and did a slow burn-and-test on my new (non-Linux) notebook.  

So far, so good.  I'm going to keep my ear to the ground on this and if nothing bad pops up, I'll do an update to me regular system, or a clean install of 5.10 on a new drive in a few weeks.

Man, this looks so much cleaner in Firefox!  Not all the crappy little undersized fonts that I always had to correct...great.
