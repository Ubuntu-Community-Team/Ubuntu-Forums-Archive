### Post by Myrddin Emrys on 2012-10-24

> One thing I hate about Windows is when I install software, a whole bunch of other software that I didn't ask for tries to sneek in and install itself...I installed GIMP on my Windows partition and the install wizard asked me to install 3 or 4 more pieces of software.

This is a problem with many Windows packages, but it shouldn't be an issue with GIMP. The official site links to these Windows installers, which are clean:

[http://gimp-win.sourceforge.net/](http://gimp-win.sourceforge.net/)

You will need GTK (included in the installers) and can optionally install Postscript and Python support, etc., but there's nothing sneaky here. If you got GIMP from somewhere else, it may have been bundled with adware etc. by some unscrupulous site - see for example this report:

[http://www.bleepingcomputer.com/forums/topic385092.html](http://www.bleepingcomputer.com/forums/topic385092.html)
