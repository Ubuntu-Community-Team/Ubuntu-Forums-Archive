### Post by hamsteroid on 2007-11-10
Hello to Everyone!

This is my first post at Ubuntu forums and I would like to say hello and hope all is well.

I got a hold of ubuntu 7.10 (dvd iso) last weekend and quickly installed the standard gnome desktop.

Here is some feedback from what I think of it so far:

My specs:
2.8Ghz P-D820 Dell
1Gb ram,
Setup / and /home and /swap - a total of 16Gb HD space for Ubuntu.
7600GT Nvidia videocard
SB16 Live
1Mb BB Internet
1280x1024 lcd standard dell monitor
Dual boot XP Pro and Ubuntu.

Previous dists:
Redhat 8,9, Mandrivia 2006.1 / 2007.1

So far:
For me the Ubuntu installation went very smooth with absolutely so glitches. During install it detected my internet BB connection which was very nice. After installation was complete it automatically reminded me of updates - which I thought was another nice touch. Finally, ubuntu pointed me to the restricted nvidia driver which installed afterwards. This ease of setup contrasts to all my previous installs where I always needed to edit the x-org file manually.
I have an external usb 500Gb Seagate FreeAgent harddrive which was also detected flawlessly and was able to access all my music media files. Only issue with this is that the drive sometimes is inaccessible if the drive powers down into sleep mode. Ubuntu is unable to wake it up.
All my windows files are accessible over the /mnt directories (automatically mounted - nice) I use a 2Gb windows fat partition to write files to the windows side.
I still have to go through the system yet but the system has been very stable so far - able to boot up and shutdown cleanly everytime so far.

All in all a very good experience so far. Thumbs up!
