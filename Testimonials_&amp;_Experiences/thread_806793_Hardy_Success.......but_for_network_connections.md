### Post by groeswenphil on 2008-05-25
Well, as soon as the system loaded, two little pop-up doohdads appeared in the top right corner of the screen. One said that there were updates available and did I want to install them. The other said that there were restricted drivers available for my hardware....and I think it mentioned Nvidid. It asked me to click a box to enable them, so I did.
Then it told me that the changes would be effective after a re-start....so I did....and when it came back, the screen was set to 1280x1024....and I left it there because the screen looks gorgeous.

I read that people were having difficulty with Nvidia cards, but my installation simply detected my hardware, downloaded the drivers, asked if I wanted to use them and .....well it just did it.

Hope that helps.
