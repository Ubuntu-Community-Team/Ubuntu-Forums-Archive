### Post by bodhi.zazen on 2010-07-02
Moved to T&E and locked.

I understand, it is sometimes helpful to vent frustration.

I had a similar experience with 10.04 , I was anticipating a more stable release, considering it was a LTS.

I purchased a new computer for my children, the old one died, and wanted to simply install Ubuntu (Edubuntu actually) as was on the old, and transfer the data.

I ran into a bug in Ubuntu that was reported during the testing (alpha) release and it is still not fixed.

I had problems with Debian. Fedora 13 worked.

So I understand your frustration, but, you also need to consider the "entire package".

In Windows , the hard ware manufacturers build hardware in conjunction with Windows and when you purchase a new computer, you get Windows + manufacturer specific drivers.

Try purchasing a generic Windows installation disk and see how many problems you have with a Windows install. You would need to track down and install the drivers yourself.

Now on the Linux side, if the hardware manufacturers had the same commitment to Ubuntu, you would have a similar experience, a working OS out of the box.

If you search, you can find several computers that come with Ubuntu (or other versions) of Linux pre-installed, and they work out of the box as advertised.

Honestly, it if frustrating, but the "blame" for your woes is shared between Ubuntu, hardware manufacturers, and yourself (you for not doing your home work or not purchasing a system with Ubuntu pre installed).

I suggest you stop purchasing systems with Windows pre-installed and instead purchase systems with Linux pre-installed.

Examples :

[http://content.dell.com/us/en/enterprise/linux.aspx](http://content.dell.com/us/en/enterprise/linux.aspx)

[http://www.system76.com/](http://www.system76.com/)

[http://www.amazon.com/Acer-Aspire-R3600-Desktop-LINUX/dp/B002POMUUM/ref=sr_1_3?ie=UTF8&s=pc&qid=1278085857&sr=1-3](http://www.amazon.com/Acer-Aspire-R3600-Desktop-LINUX/dp/B002POMUUM/ref=sr_1_3?ie=UTF8&s=pc&qid=1278085857&sr=1-3)

If you do so you will have a better experience and support the migration from Windows -> Linux.

If not, you need to do your home work before you buy your hardware. Check any of the various hardware compatibility list.
