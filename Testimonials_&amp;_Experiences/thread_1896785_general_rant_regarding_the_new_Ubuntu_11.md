### Post by rebeltaz on 2011-12-17
I am not an early adopter of technology. My motto is "if it ain't broke, don't fix it." I am still using Ubuntu 10.04 LTS on all of my systems. I like it.

Having said that, I was building my girlfriend a new computer, so I decided to load 11.10 to see what all the hubbub was about. Minchia! Where to start? 

First, why did the designers/programmers decide that laptop and desktop computers needed to look like tablets and phones? Most home computers are not touchscreen, so why design the operating system as if they were?

I know you can install gnome-shell (I think it is) to get the old look back, but why not offer that choice on installation? So the user is forced to accept the "new and improved" version. (Isn't that a weird saying - "new and improved"? Which is it - new or improved. It can't be both!)

Then, if you want to adjust font size you have to install a separate utility (I refuse to call it an app!). 

Menus are hidden by default - still haven't figured out how to disable that. 

No more grouped programs neatly in a "start"-type menu - instead we get a "launcher bar." I want to run a program, not launch a space ship. Locating a program is a pain. Yes I know I can type it in the Search bar, but doesn't the defeat the whole point of a GUI? If I'm going to type programs names to run them, I'll just drop into terminal and do it from there. At least there I feel at home!

I mean, surely I cannot be the only one who DESPISES the look and feel of this new iteration of Ubuntu/Gnome3? 

I guess my systems will just have to remain outdated.

### Post by 3Miro on 2011-12-17
There is no Ubuntu 11. There is Ubuntu 11.04 and 11.10, which stands for dates: April 2011 and October 2011.

The old interface of Gnome 2 was killed by the people who were developing it. The Gnome project decided to make a new interface of Gnome-shell. Then Canonical (i.e. Ubuntu) decided to make their own interface called Unity. Both Gnome-shell and Unity are quite a bit different from what you are used to in Ubuntu 10.04. There is also the Gnome-session-fallback interface which is the closes to Gnome 2, but not quite there.

Basically Ubuntu 11.10 comes with Gnome 3 and the Unity interface. You can optionally install Gnome-shell (which I think is even more different) or Gnome-session-fallback.

Unity looks like Mac, but since I have not really used a Mac, I cannot say if it functions like a Mac. Try it on your own, some people like it. Other people like Gnome-shell. Then there are people that really miss the Gnome 2 interface and many of those are moving to XFCE. Then there is also LXDE and KDE.
