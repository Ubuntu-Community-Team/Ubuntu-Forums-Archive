### Post by Devport on 2011-11-07
I was disappointed by recent Ubuntu / GNOME updates and wanted to be able to do dist upgrades without having to cope with constant UI changes of decreasing quality (IMHO). I understand that Ubuntu wants to unify appearance for use on different modern devices but this can be done without reinventing the whole desktop ( i.e. this can be achieved easily on KDE by using Plasma and it is actively being worked on ).

So I switched to Kubuntu and it is really beatiful. It brings back the joy. The transition went smooth and due to the wonderful qtcurve theme the apps which require gtk look well integrated. As a long time GNOME user I can say that KDE is an adequate replacement for GNOME. Furthermore KDE is improving day by day - among other things I am looking forward to the final release of the Calligra Office Suite.

Now I can enjoy smooth upgrades again.
