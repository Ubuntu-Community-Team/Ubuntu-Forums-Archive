### Post by zero244 on 2008-09-14
Ive been using Ubuntu for over a year and half and I am pretty impressed.
I don't run Beryl much anymore only because I run Windows XP in a Vmware Box and it slows me down too much.
The One Big Plus I noticed about Ubuntu is very little fragmentation.
My Ubuntu partition is 40 gigs with 30 gigs of data on it. After a year and half I looked at the sectors and found less than 200 fragmented files out of several thousand.
Windows after this amount of time would of had thousands of fragmented files.
This system has not bogged down it runs just as fast today as it did 18 months ago when I installed it.
It really is a great OS........I can run Windows XP in a box all day and Ubuntu doesn't miss a beat. I can have both OS's running idle at the same time and the CPU usage is only 3 to 5 percent.
Vista uses 25 plus percent sitting idle.
This all started when a kid gave me a Dapper CD that I loaded in live mode.
Ive been running Linux ever since.
I found running Windows XP in Vmware Player more functional than running it in Vmware Server.
Player handles USB better and its easier to switch between screens.
When Vmware finally comes out with a stable release of Workstation that fully supports 3D I will pick up a copy.
I have a few apps I run in XP that arent available yet in Linux. 
To my way of thinking its easier to run Windows apps in a Vmware Machine than setting up wine etc.
Just my two cents.........still satisfied after 18 months, I guess I am a full convert now.
