### Post by gabo.cr on 2010-01-19
> Now it's not all sunshine and rainbows on Ubuntu though. It wont let me see what level the ink is at. In Ubuntus defense, normally in windows it will tell me the ink is low or completely out even though I'm able to print another 15-20 pages.

To check the ink level you can use "escputil".

```
sudo apt-get install escputil
```

Then use:

```
escputil --help
```

To see the instructions.
