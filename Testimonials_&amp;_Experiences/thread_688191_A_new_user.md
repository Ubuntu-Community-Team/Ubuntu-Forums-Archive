### Post by p_quarles on 2008-02-05
Hello and welcome. Glad your first experience with Linux has been a good one. :)

If you're not fond of dual-booting, but still need Windows for some things, you might eventually want to look into virtualization. Applications such as VMWare and Virtualbox will allow you to run Windows inside a virtual machine on your Ubuntu installation. It can't do everything, and it requires a machine with decent specs to do this, but it's a great solution if there are just one or two Windows apps that you can't do without.
