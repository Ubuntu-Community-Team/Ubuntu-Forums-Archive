### Post by farlander762 on 2008-06-27
I was a little bored one afternoon so I started reading the case studies in the /home/[my name]/examples directory.  I read the one about the Howard County Library and decided to contact them to see if it was really as great as stated.  I omitted the contact information.

Here were my comments/questions:

> 
Date: Sunday, June 15 2008 04:30 PM
Subject: Attn: Amy De Groff regarding Ubuntu Linux

Hello,

I read a case study about your library system converting to Ubuntu Linux that came with the Ubuntu software. I am just curious to know how well the implementation (really) went and if you are still satisfied with your decision to change from Windows.

Of course, any supplier of a good or service will always hail their product as the greatest ever, so I just want to know if it has been as good as advertised. Have you upgraded to 8.04 (Hardy Heron) yet?

I have been using Ubuntu for about 18 months now off and on and have decided that Hardy is going to mostly replace Windows on my own computers. (If I could just watch 'Lost' on ABC's site it would replace Windows.)

Thank you, 


Here was the first reply I received:

> 
Date:	 Mon, 16 Jun 2008 20:00:50 -0400 (EDT)
Subject:	Re: Attn: Amy De Groff regarding Ubuntu Linux

Thank you for contacting us. I have forwarded your request to Ms. De Groff for her consideration. I myself was so impressed by Ubuntu when the Library first implemented it that I formatted my own personal Dell Inspiron laptop (which used to run Windows XP) and installed Ubuntu on it. I've been very happy with the change and have found open source software to replace most programs that I used when working with Windows.

Thank you for using the Howard County Library. If I can assist you further, please contact me.

Daniel Curry
Assistant Circulation Supervisor
Howard County Library - Miller Branch

Then Ms. De Groff replied:

> Date:	 Thu, 26 Jun 2008 09:13:49 -0400 (EDT)
Subject:	Hardy Heron

Good morning. Thank you for your note to Howard County Library, dated June 15.

I am pleased you read the case study of Howard County Library with regards to Ubuntu.

We are very happy with Ubuntu and delighted by 8.04. We migrated slowly, by first adding to existing Windows machines the products we'd use on Ubuntu (Firefox, OpenOffice,Meebo) and THEN switching the operating system. It allowed our staff and customers to get used to the software applications with the operating system they already knew -- and when we did make the switch -- for many it was seamless.

The advantage for us with Ubuntu is the security and the control we enjoy - we can update all computers from one location (and managing 500 computers from one spot is a nice feature.) While this would of course be feasible in a Windows environment, it would involve licensing MORE software on top of the Windows licenses - with our use of open source - the licensing cost is 0.

The firm we've worked closely with, [[COLOR="Blue"]OpenSense Solutions[/COLOR]]("http://open-sense.com/") -- has been a tremendous help and a wonderful technical partner.

As for ABC, I would encourage you to get in touch with ABC; my staff here are in frequent (weekly) contact via email with Major League Baseball -- whose site is also not standards compliant and "requires" certain applications to run -- these type of requirements frustrate me as I sense they do you! I try hard to email sites that place these restrictions on users - In a few cases (Mary Kay for example) they have made changes! So, you never know!

Regards and enjoy Ubuntu!

Amy


Amy Begg De Groff

Howard County Library


Howard County Library is committed to using open source solutions [[COLOR="Blue"]whenever possible[/COLOR]]("http://hclibrary.org/index.php?page=97").


It seems that they really are quite pleased with Ubuntu.  I know that most of my experiences have been pleasant.
