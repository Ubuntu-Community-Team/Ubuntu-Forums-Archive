Hey Guys I just wanted to say that I love ubuntu, well that is the server edition. I have used the desktop version and its really great but for what I need I stick to Windows and Mac. But anyway I LOVE Ubuntu Server!!!! My company was always using some crap server Os's like Windows Server and Mac OS X server, which would always pose a security risk, and never worked the way we needed it too. Then I proposed the Idea to my boss. He was like "free?!?!?!" 

While we have been wasting buckets of money on Windows Server and Mac OS Server to run our web server, Files, Backups, and e-mail. And Every time something went wrong we lost everything and even though it was backed up, the servers were giving a hard time to restore from it. But then we switched to Ubuntu Server. 

Which handles everything well and we got a computer specialist to try to get into the system and mess it up and he couldn't after we secured Ubuntu Server. Also, not to mention how much faster our web server and File servers are. 

Also our business consists of about 150-200 people always trying to access the server and we need something reliable and Ubuntu Server works great!! Any ways guys I just wanted to tell you guys about my awesome experience with Ubuntu Server :) 

If you guys need any help configuring Ubuntu Server you can contact me by personal message. Or definitely post your question on the forums! The forums are great and you can get any questions answered. BTW thank you all the people on the forums who helped me with setting up Ubuntu Server.
