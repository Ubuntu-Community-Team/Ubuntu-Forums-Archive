### Post by wolfen69 on 2008-04-02
i recently bought a 10 year old compaq laptop for $25 US dollars which came with win98. it only had 32mb ram, so i maxed it out to 160 and bought a wireless card. it is now an $82 laptop. the best part:

**im running Gutsy 7.10 on it!** i installed the server edition and didnt install any server components during install. which left me with little more than the kernal after  install was finished. i then built it up with a base install of xfce4. i had a desktop but not much else. to make a long story short, i slowly built it up with some apps and networking components and have a sweet little laptop for under $100. i even got the wifi going. plus, the volume and contrast/brightness buttons on the laptop even work. wow! on a newer lappie this wouldnt be news, but 10 yrs old? 

i have done many installs of ubuntu, but never thought you could get a modern distro on such an old machine. sure, the ram is almost maxed out (160) but heck, anything is better than win98. it was a great learning experience too, since most things have to be set up via the terminal.

long live ubuntu! :guitar:

btw, april is my 1 year anniversary of going to linux full time. woo hoo!
