### Post by etherealremnant on 2006-12-16
The more I use Ubuntu, the more I love it.

I am one of the types that checks distrowatch every day looking for something new to install... but I always find myself coming back to Ubuntu.

Now that I've discovered the netinst image and the ubuntu-desktop, kubuntu-desktop, and xubuntu-desktop containers for apt... wow.

I prefer netinst ISOs to downloading an entire operating system CD because I find I don't use all the packages included on the ISOs and I can write the netinst image to my USB key so I'm not wasting CDs or DVDs either.

I did find that there was no way I was going to get the OS to install if I selected Ubuntu, Kubuntu, or Xubuntu after the base install so I went the backdoor route and ran apt-get install kubuntu-desktop and tried it out. Decided I didn't like Kubuntu so I ran apt-get autoremove kubuntu-desktop and ran apt-get install ubuntu-desktop. Worked flawlessly after a reboot. Then I decided that since I've never used Xubuntu before, I'd just install and run that. This time I was impatient and just ran apt-get install Xubuntu. I figured it would kill the install and I'd just have to reinstall but the only thing I've noticed so far is that the Xubuntu splash is still my old Ubuntu one and I still got to keep my Gnome apps. Win-win situation, really.

Try that with any other OS. Seriously. It won't happen. Linux is the only possibility for something like this to happen.

Oh, and big thumbs up for including nvidia-glx in the main repositories now! Makes installation much easier in the end... though its the outdated version and I'll have to grab the new version if I want aiGLX.

Anyway, Ubuntu is officially good for anything. All the apps I want to run work perfectly, nothing is ever broken out of the box that I've found (contrary to the SUSE, Fedora, and Mandriva worlds I was used to), and I love the free software philosophy. It never ceases to amaze me how so much of the software in linux blows away the non-free equivalents. Like Audacity, for example. Audacity has the tools of professional editing software but doesn't take the same system resources or crash (*cough* Sony Soundforge) and it doesn't take a rocket scientist to figure out how to use (*cough* Wavelab).

Now the hard part: converting my friends and family over. Maybe once they see Beryl in action... :twisted:

I wish KDE would get their act together though... I used to vastly prefer KDE to Gnome... then I got Ubuntu and actually gave Gnome a chance. The only thing I ever install from the KDE side is K3b. I haven't found anything better. I find Gnome to be much lighter overall, more organized... I guess more mature is the word I'm looking for. Once again, another thanks to Ubuntu for opening up new doors in that arena.

Note: Any ranting/flaming about me bashing KDE will be purposefully skipped over so don't waste your breath. :)
