### Post by N00b-un-2 on 2011-08-27
I'm sure that this topic has already been beaten to death elsewhere and I'm also pretty sure that my opinion is of the minority, but it is just that -- my opinion.  Not a statement of fact.
I've tried Unity on Ubuntu and I gave Gnome 3 a whirl on Fedora and to be completely honest, I absolutely detest both.  I'm perfectly happy with Gnome 2 the way I've been configuring it since I switched to using Linux (menu bar on top, dock down at the bottom, window buttons on the left -- essentially like OSX).  I don't want to change.  I also don't want to switch from using Ubuntu.  I've done my share of distro hopping in the past couple years, but I always find myself coming back to Ubuntu.  If Gnome 2 is not available in the next version of Ubuntu either I will not be updating or I'll simply switch to another distro that does have Gnome 2.  I'm fully aware that this is only going to be a temporary fix, since eventually everyone will be adopting Gnome 3.  I'm pretty upset about the fact that Gnome 3 doesn't support a "classic" mode the way that Ubuntu has Unity and standard Gnome.  I think that both the Unity camp and the Gnome 3 camp are too wrapped up in trying to reinvent the wheel and create a unique identity for Linux.  They're both taking major steps backwards in terms of flexibility and customizability and blatantly ignoring the tried and true desktop innovations that have been in use in Linux as well as Windows and OSX.  If every single other desktop environment in the world puts the dock at the bottom (Docky/AWN/Cairodock on Linux, the Dock on OSX and the Windows 7 "super bar") why in the world do both the Gnome 3 developers and the Unity developers insist on putting the dock on the left and completely removing the ability move the dock or customize the task bar?  Even Windows and OSX allow you to modify the taskbar to some extent.  It's my opinion that gnome 3 while having a few innovative new features is a massive step backwards in usability.  At least Unity can be disabled and the standard Gnome desktop can be used (for now anyway).  Unity started out as a netbook interface and in my opinion never should have been forced onto the desktop.  Give me back my gnome panel and let me move the dock where I want to put it and we'll talk.  Until then I pretty much hate both Gnome 3 and Unity.
