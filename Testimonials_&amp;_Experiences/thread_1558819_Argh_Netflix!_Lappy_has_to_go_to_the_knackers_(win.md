### Post by cadenwhitaker on 2010-08-22
Couple of things, I'm still using ubuntu for my "vm experience" where I have a native OS that automatically boots into a VM of ubuntu on OS startup. This is pure heaven. I can reinstall that native OS as many times as I like and my little ubuntu VM (with all my bookmarks, settings, saved passwords, important stuff) stays rock solid.

When I burned ubuntu on my laptop six months ago the experience was more mixed. Netflix support isn't ubuntu/linux's fault, I get that, but it's still a deal breaker for me.

The only other thing I can say is ubuntu is great, but there is too much reliance in the linux comunity for solving problems through the terminal window. I can't call my parents and tell them that linux is so easy, just "sudo apt-get install yourspecialprogram".  That's something I knew about getting into the community, but it is something we the linux users will have to work through if there is ever going to be a competive linux distro out there.

For a perfect example, check out how you'd go about mounting a windows share permanently, [https://help.ubuntu.com/community/MountWindowsSharesPermanently](https://help.ubuntu.com/community/MountWindowsSharesPermanently) COME ON GUYS WORK WITH ME HERE ;)!!!

Stay cool.
