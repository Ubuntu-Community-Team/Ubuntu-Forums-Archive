### Post by richrock on 2008-06-25
Hi,

I've been using Ubuntu since 5.10, and this has mostly been on my laptop(s).  I've recently just redone my desktop PC with 8.04, and completely ditched windows.  Why?  Because I like the challenge, and also that I've comfortably used Linux on a laptop for a few years now.

It's not to say it'll be without it's challenges, as far as hardware goes.  That's all part of the *fun* of computing, I thought.  Wasn't the early days dominated by people trying things out?  No plug and play then.  If I have to rewrite a script, change some obscure .conf file, so be it.  At least I am free.

And yes, although I have to use WinXP and Dreamweaver at work, I develop websites using an open source CMS, code in PHP and occasionally submit the odd bug report.  Getting into Linux in general makes we want to learn more about the system, it's core code.  So I'm trying to start learning C++, although it's going a little slow, especially when you have a series of big deadlines at work, and a girlfriend who craves attention!!

I'll probably blog about how the change goes, because it's interesting how much we have seemingly come to rely on the dross from Redmond.  Don't get me wrong, I cut my teeth on MS-DOS, Windows 3.1 and onwards. XP is a good system, but I just have seething resentment for Vista (I also manage the IT in our company, and the two graphic designers are using Vista :( )

So Ubuntu it is.  And long may it continue.

Rich
