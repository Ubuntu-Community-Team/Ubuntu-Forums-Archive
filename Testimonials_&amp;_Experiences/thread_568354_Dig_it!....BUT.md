### Post by Triniti on 2007-10-05
So far Ubuntu has been able to offer me everything I need... except for one tiny little program called DecoTech3030.
I love the speed, I'm running a duel boot system with Vista, and I have 2Gig Memory, a decent graphics card, and an OK CPU. 
Ubuntu is fantastic when it comes to speed and ease of use. Startup is super fast, and I find it easy to get around in. Still coming to grips with terminal commands, but I like the sense of achievement when something works.
I work from home, designing kitchens on the above mentioned program. I got it to work "perfectly" through Wine, but even though the program works, the screen is so dark that you cannot do any work on it. 
I would like to extend a challenge to some experienced Ubuntu users to try to get this program to work. Unfortunately there are no alternatives in Linux for this program, and my entire company uses it. In short... I'm stuck with it.

Anyone wanting a go at it can download a free demo from the link below.
[http://www.decotechsoftware.com/dds/free_demos/index.html](http://www.decotechsoftware.com/dds/free_demos/index.html) 
It's really an easy program to use, and I would love to see something like this for Ubuntu some time...

Well done Ubuntu, what a fantastic OS!
