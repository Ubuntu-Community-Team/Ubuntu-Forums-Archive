### Post by L815 on 2009-08-17
I've been using Linux for a while, but it has always been on the same familiar hardware. 

Well, recently I installed Xubuntu on my parents machine (not very powerful), and they were in need of a new wireless card. I had no issues with the old one, under Xubuntu, just that it was really old and the range was really short. 

So, I went out and bought a random card (Netgear), completely forgetting if there were drivers available for Xubuntu. It came with a cd, and I feared the worst, but I had decided to give it a shot anyway.

Later... I plug it in, boot up & magically the previously saved connection (from the old card) was automatically connecting without any of my assistance. I was so surprised at how easy and convenient it has become for Linux. 

I was prepared at hand with the driver cd, reboots, and a USB with the wireless key in case it came with its own manager (fearing I would have to install XP again). An anticipated 15-20min installation turned out to be a 2-5 min installation.

Anyway, just came to release my excitement.

PS: My parents have been using Xubuntu for about 1-2months. The only complaint I have heard from them is that there isnt't 'Spider Solitaire', haha.
