### Post by Roger Allott on 2009-12-24
On all the versions of Ubuntu I've come across, both Gnome Office and OpenOffice.org are installed automatically. I'm wondering why, and whether anyone here might be able to tell me.

For example, is there any task that can be done in Abiword that can't be done in OO's Writer? Or is there any task that can be done in Gnumeric that can't be done in OO's scalc?

Some aspects of Gnome Office do not have an equivalent in OpenOffice, such as gnucash I think, so why not just install those bits of Gnome Office and leave aside Abiword and Gnumeric?

Or, for people with PCs of limited resources, offer them the option of Gnome Office **or** OpenOffice?

There are perhaps many other examples of application redundancy in Ubuntu, but the Office applications seem to me to be the most obvious.

### Post by 3rdalbum on 2009-12-25
Before Ubuntu, all the Linux distributions attempted to install as many programs as possible. You'd get three different web browsers, five text editors, two image editors and a multitude of useless programs like icon editors.

Ubuntu came along with only one program per task.

The only double-ups are with text editors - 'vi' and 'nano' and 'Gedit'; web browsers 'w3m' and Firefox; and image viewers 'Gimp', 'F-Spot' and 'Eye of Gnome'. The first two redundancies are not really important because Nano and Vi are not visible in the install, and W3M isn't visible in the install. Ubuntu is going to remove Gimp from 10.04, and make Eye Of Gnome invisible (or remove it).

I have NEVER seen Gnome Office in a default Ubuntu install:

```
chris@chris-desktop:~$ abiword
The program 'abiword' is currently not installed.  You can install it by typing:
sudo apt-get install abiword
abiword: command not found

```

You must have installed from a remastered disc.
