### Post by wribeiro on 2013-02-05
Ubuntu 12.10 is not a stable version and users should be notified about this before download it. Recommended version should be 12.04, that really works.
It's not easy to convince people to substitute Windows OS and when things go wrong, there is a big rejection.

For me, for example, none of the 3 video drivers work on my NVidia Graphics, after kernel update to 3.7, and Unity simply do not start.
Furthermore, Unity integration with many programs (including LibreOffice suite) is a complete disaster.
Because of this I was forced to change the UI to LXDE, which solved several problems, but created others.

In short: after 4 months of suffering, I can only downgrade to 12.04, which will give me a lot of work and disappointment.
