### Post by MKyros on 2007-05-09
When I first heard about Linux, I didn't think very much of it. As a matter of fact, I thought it was dumb and useless. They don't have very many games, and it wasn't popular. Of course, me being in my early teens, I was ignorant and prejudiced. I now see with unclouded eyes of how great Linux is, and how pathetic that Windows fails in comparison.

My first impression was when installing Ubuntu (Dapper) was, "damn, that was fast." I was shocked that I was able to get the operating system up and running in just around 15-20 minutes, when it takes my Windows installation a couple of hours to do (format and the actual installing). 

My second impression was how easy it was to install software on this OS, when I had previously heard how hard it would be. The Synaptic Package Manager was a very valuable piece of software, and it brought all the software I would need for college and whatnot, for free!

My third impression was that setting up a network with my WinXP laptop took only minutes, instead of hours like I had to previously. The guide for Samba was ridiculously easy to find, and it was very easy to understand it as well; one google search was enough. You can't do that with Windows for some reason. Any kind of guide to help setup a Windows network was hard for me to understand. It wasn't until I read the Samba guide that I learned so much about Networking. 

But that's where the good things stop.

My fourth impression was that installing drivers for my ATI graphics card was a pain in the butt. However, I understand that ATI drivers is closed source, so it makes it difficult for programmers to develop an operating system that would be able to easily, and efficiently handle the drivers without problems, like for nVidia, I assume. 

Fifth would be that running my favorite games on Linux would also be a pain in the butt. I like to be able to get my software running without any problems or downloading, whatsoever. I didn't have much of a problem setting up Wine, but the game doesn't run stable on it. I understand this problem as well. It's all business, companies go where they can make a lot of money, and sadly, Windows is where the money is at. 

The sound is also a problem. I like to hear good, clear music. I had some trouble setting up OSS (I used that because ALSA lagged CSS :F), but it wasn't enough. 

Conclusion, Linux is great, it's as simple as that, but I must install WinXP back on my computer for the games and ease of it's use. But Linux has impressed me enough that I will later on setup my machine to dual boot both Linux and Windows. From now on, I'm going to keep a closer eye on everything that happens to this operating system. This OS has tons of potential, much more than what Windows can ever hope for; and I, for one, I hope to one day take the full plunge.

P.S. The community is great to :) .

### Post by maniacmusician on 2007-05-09
A few notes;

- Why install Dapper? The latest version is Feisty Fawn (7.04). Dapper was two releases ago

- Nvidia doesn't have open source drivers, but their closed source drivers are very good. ATI's are not.

- It's true that most of the games that you played on Windows probably won't work very well on Linux...but Linux does have a few titles that run very well on it. [http://appdb.winehq.org/](http://appdb.winehq.org/) <---- a list of games and apps that work well in WINE. 

There are also games that have native Linux versions. Examples are Quake, Doom, Unreal Tournament, sauerbraten, alien arena, etc. There are probably more, but I wouldn't know, I'm not a big gamer.

-  The sound has always worked for me by default. I guess that would probably be ALSA. You should also try different versions of Ubuntu. There is Kubuntu (uses KDE), Xubuntu (uses XFCE), and a few others. I'd really only recommend Ubuntu and Kubuntu to beginners. Xubuntu is also great after you have a little experience.

edit: and if you really want to be able to play games on Linux, you really should just get an nVidia card. it's tough sometimes to get 3D acceleration on the ATI cards. The GeForce 7600GT runs around $120 these days, that's not too bad for a card of its caliber.
