### Post by PongGod on 2006-07-11
Despite being a long-time Windows user and software developer, I've been intrigued by all the Linux buzz, but a bit reluctant to spend a lot of time trying to learn some new stuff that I may or may not get a lot of mileage out of.  But, when a co-worker of mine raved about Ubuntu having the easiest installation of any Linux distribution he's come across, I thought I'd give it a try.

Having got my free CDs in the mail, I set out to install it on an old PC that had been gathering dust in the corner.  As promised, the installation was extremely straight-forward and was finished without any complications in 30-40 minutes.  The user interface was intuitive enough that I could do all the basic stuff without having to resort to a manual.  So far, so good.  Now I have a backup machine that I can use next time my Windows box gives me any trouble.

Well, it wasn't long before trouble arrived.  When my USB flash drive came out of the washing machine (D'OH!), I wasn't very optimistic about salvaging the many valuable files it contained.  I stuck it in the USB port on my Windows box, and sure enough, the system only recognized it as "Unknown device".  So, I tried putting it in an older (USB 1.0) port and the results were even worse - the system locked up completely!  But, it was even worse than that - when I tried rebooting my system, it would hang, even in Safe Mode!  So, not only is my flash drive gone, I've got to restore my hard disk from a backup image... wonderful.

Then I remembered a friend of mine once telling me of a crashed hard disk he had of which he was only able to recover the data by booting up with a Knoppix CD.  So, I thought maybe there was a slim chance my new Ubuntu system would be able to read the flash drive.  I stuck it in the USB port and VOILA... there are all my files!  I immediately copied them all over to another flash drive and life is good once again.

What an awesome product Ubuntu is! I have a feeling I'll be using this system much more than I originally planned.

- Robert -
