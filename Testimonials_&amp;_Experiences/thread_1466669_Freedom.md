### Post by dbigers on 2010-04-30
[SIZE=2]Why it took me this long to get to Linux I don't know. All I know is that I am loving it. I have been involved with computers in some form since the C64. Then I migrated to DOS machines. Purchased a Mac in 95, back when the PowerPC Macs first debuted. I did that for 3D rendering and graphics which I still enjoy.
 
A couple of years later I switched back to PC's and started using Windows again. Mainly for the decreased cost of rendering speed. Windows memory management was terrible until XP came out though. If a program crashed (and 3D programs do crash from time to time) it usually brought the whole system down. XP solved that for me quite nicely. So for the past 8-9 years I have been quite content with Windows. 
 
My current PC is a quad Q6600 running Vista Home Premium. Vista really doesnt bother me as it does a lot of other people. It works, there are some annoyances. The UAC which is actually a good thing compared to XP when security is considered is loathed by many.
 
For the last year I have been getting to know Blender. I mainly use Lightwave, but Blender has some amazing features and is constantly worked on and getting up to date. The new 2.5 version is what really brought me to Linux I guess. I had been downloading new builds from Graphicall. Being alpha software, there are still plenty of bugs, some showstoppers. But it has been cool to check out where it is headed. I kept noticing that you could the most up to date build on Linux. Seems Linux is what the developers use. Windows builds and other platforms usually come later once someone is gracious enough to upload a build. 
 
So I started looking into Linux and eventually that led me here. So glad I did. I love Ubuntu. Liberation comes in many forms. As a computer user, liberation means I can use software that I want and how I want. I dont mean steal either. I can change anything and customise any part of my computing experience. I can find any number of educational software for the kids and I can rest assured when installing that they are safe to install because of the repository system. I can surf the web more securely. 
 
To be honest I have never gotten a virus or even picked up spyware on any Windows machine I have ever owned. But I have encountered and am asked to clean this stuff off of most of the computers of my family members. It gets old. Especially spyware, cause it usally opens up the computer and everything else pours in afterwards.
 
Once I get better versed with Ubuntu I want to get my family members to let me switch them over. Personally, I did a dual boot setup. I downloaded an ISO for Karmic Koala earlier this week. Burned the disk and started from there. At first I just installed from Windows using Wubi. I was satisfied enough with that to start. It is somewhat empowering to be able to boot into more than one OS. After a couple of days of testing with Wine I found most every Windows program I rely on, works either flawlessly or good enough for how often I use them.
 
So, at that point I did a real dual boot installation, including creating a real partition. Reinstalled 9.10. during this time I didn't even notice that 10.04 was about to be released. The very next day Lucid was released. I was going to try and upgrade 9.10. But it was taking forever to get the upgrades and packages. Being release day I am sure the servers and mirrors were being hammered. I canceled that. Then I downloaded the ISO for 10.04 and burned a new disc. That download was quick. From there I did a clean install and am now happily using 10.04.
 
On a side note. I have read that a lot of users were happy to see a new theme, citing the oldness of Humanity. In the couple of days that I used Karmic, I really liked the humanity theme. I dont particularly care for the new theme. But I have since reinstalled the Human theme from a Debian package on Launchpad.
 
The other thing I would like to mention is that both installs went flawlessly. Ubuntu/Linux found all my hardware. This PC is 1.5 years old and it picked up everything. Only annoyance was the audio did not work at first. It did actually, but as has been documented by others, it was muted. 
 
So in closing I wanted to thank everyone on the Ubuntu team, the people who post on these forums which has helped me a lot in the past couple of days, and Linus for his original vision in creating Linux. Life is full of choices, computing should be the same. Now it is for me. I will still keep Vista installed for a while, just in case. But for now my focus is on Linux/Ubuntu. 
[/SIZE]
