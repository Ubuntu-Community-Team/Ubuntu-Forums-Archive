### Post by BlaineM on 2008-01-16
So this is not the typical goodbye Ubuntu (I need help thread).

I wanted to post this goodbye to say thanks for all of the great support and responses that I have received over the last two years of using Ubuntu for the desktop.  I recently purchased a Macbook, and am giving OS X a round... I like it so far.

I tried to run the server version of Ubuntu for the little bit of programming that I do, but just didnt care for the debian type of control.  I got hooked on CentOS and just havent found anything that I like as much for the server.

Thanks for the great community that is built here, and I hope that things continue to grow and momentum continues to build in a positive direction for Open Source Software.

Sincerely,

Blaine Motsinger
