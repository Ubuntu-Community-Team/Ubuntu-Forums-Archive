### Post by xavier_r on 2007-07-27
Dude I tried this, and succeeded, and its awesome...
Believe me you gotta try this out...

Just do…

**$ sudo apt-get install w3m fbida**
Here, fbida, is our image-viewing software.. w3m, is our browser…

Then after installation is complete… Do Ctrl-Alt-F1...
Run w3m, and it will start showing images…

### Post by 3rdalbum on 2007-07-28
You can do the same thing with Links 2:

```
sudo apt-get install links2 gpm
links2 -g www.microsoft.com
```

Links 2 does it better though, as it supports some Javascript, and you can use the mouse! (as long as gpm is installed).
