### Post by mikemcfarlane on 2007-06-02
OK, disclaimer to start, I like Win XP, and as week old user of Ubuntu I am still in the first flushes of enthusiasm for all things Linux.

But, the Windows installation process sucks!

I had a fully operation and updated Ubuntu system in half a day. One CD, one broadband connection. Everything integrated, everything sorted.

I've started this morning reinstalling Win 2000 on my parents laptop (a winmodem scuppered my Ubuntu installation, they don't want broadband and I'm not tech enough to hack the winmodem stuff). So, to start, one Win installation CD. Install. OK so far. Now connect to the internet, go to various places and download drivers, install a large folder of drivers, reboot each time! Install service pack. Reboot. Install updates. Reboot. Install AV, firewall etc. Reboot. OK, now I've got an OS. Oh, I've still got a pile of applications to install! It just sucks. 

And Win looks crap!

Just letting off steam.

Mike

PS yes, I know, comparing a seven year old OS to a state of the art isn't clever, but XP and Vista aren't that much better in the installation process. Maybe a few less reboots:(

### Post by bromix on 2007-06-04
I have several friends and family members who ooh and ahh at my Ubuntu with XGL/Beryl all tweaked out, and talk about how neat my pc is, but still won't switch.  Yet, they still call me all the time when they windoze install gets all gummed up, froze up, and well, you name it.   I found a tool that at least makes a windoze install a little more tolerable.  It's at [http://www.autopatcher.com](http://www.autopatcher.com).   I keep a current copy of AutopatcherXP in my CD case.  

Lifehacker summed this up well as follows
     Windows only: AutoPatcher is a freeware Windows Update alternative that gives you more control over how you update your PC.

Rather than requiring an internet connection every time you need to update a PC, AutoPatcher can be burned to a CD and used on any computer - which comes as a godsend for any resident PC-geek of the family, especially when you've updated a PC over a 56k modem. The great part about AutoPatcher is that not only does it dole out the regular updates, but it also automates installation of several tools not generally included by Windows Update, like Windows PowerToys.

The only downside to AutoPatcher is that you can't automate future updates, meaning that you would have to check back monthly to get the "Update" releases. If you're fine with that, then go with AutoPatcher all the way. If you don't want to worry about checking back with AutoPatcher for updates, the regular Windows Update tool should remain turned on and updating. However, you should still keep AutoPatcher in mind any time you've re-installed Windows, as it provides a much easier and arguably secure way to update a freshly installed, unpatched version of Windows.
