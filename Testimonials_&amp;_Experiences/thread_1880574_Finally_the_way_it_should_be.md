### Post by rjs987 on 2011-11-13
Since early 2008 I've been running mostly all Linux at home. Soon after that I was able to switch over to Linux as my primary OS at work with a Windows vm. Windows is the standard at work so this was a very non-standard set up there, but I wasn't alone since many of the developers were doing the same. I support the desktop workstations so I saw this as a way to expand my support to those developers. Last year I was forced by the brainwashed M$ devotees to make Windows my primary OS and was "allowed" to have a Linux vm if I must. Well, that didn't work so well. You see, running Windows 7 as a vm on Linux works as well as running Windows 7 alone. At least that was my experience. We use a number of programs that are very M$ centric and simply will not work on anything else. We also use a number of programs that have Linux equivalents and I like the Linux version better. Besides, I was able to use Linux programs to provide expanded support beyond what I was expecting by recovering some data in files that no Windows program was able to recover or read.

We lease our computers on the desktops and mine was up to replace the lease this month. I promoted the idea that if I was running Linux as my primary and Windows 7 in a vm that when my workstation is replaced it would take me no more than 30 minutes to be back up and running on a new computer instead of taking all day to rebuild and reinstall all the programs in Windows on a new machine (true statement since all new workstations are loaded with a fresh image before they are deployed, and then any needed programs are added. Very time consuming. With a Windows 7 vm I can, and have, install Linux and vbox within 30 minutes. Copy the vm over and be back in production just like that. My boss agreed and I am guessing she informed my team leader (one of the M$ brainwashed devotees). He subsequently announced to the rest of the team that "he" decided our team can use any OS we want as long as we have a Windows 7 vm for the production programs we need to use. I guess I wore him down with a years worth of comments about issues on Windows 7 that I didn't need to put up with. What really got through to him was that as he was trying to get his new computer set up he was having all kinds of problems and ended up taking TWO days to get his new computer running right.

Anyway, I am now running 11.10 at work with Unity. I have 11.04 at home and up to a few months ago was running classic gnome. I decided at that time to try Unity and, though it was very different than what I was used to, it is growing on me. Now I am really OK with it. I did install ccsm and made a few mandatory changes. Like shrinking the oversized icons on the launcher and making the launcher stay put when I maximize a program. At work I started out with Unity and made the same changes using ccsm. I have a 24" screen and find I like having the Unity bars across the top and down the left side and the Windows 7 task bar across the bottom. I use vbox in seamless mode all the time and this works out really well for me. I actually like the improvements to Unity in 11.10 over Unity in 11.04.
