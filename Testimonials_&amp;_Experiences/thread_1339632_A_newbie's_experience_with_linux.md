### Post by HolyDiver24 on 2009-11-27
Well, about a year ago I made [a thread where I complained about the laggy feeling of linux's GUI]("http://ubuntuforums.org/showthread.php?t=989697"). I was using Xubuntu at the time. I returned to using Windows XP soon after I made that thread, but a few months later I tried Arch Linux. With improved Nvidia drivers the GUI felt much better so I returned to using Linux almost exclusively. Over the last half a year or so I learned a lot using Arch Linux and gradually came to use dwm and terminal applications in almost every instance where I could. 

However, recently I had to boot into Windows to do some video editing (kdenlive kept crashing) and somewhere along the line I found myself playing Quake 4. Immediately I noticed how amazingly smooth the graphics were. I couldn't believe my eyes.  All I'd read about linux (its memory management, design principles etc.) had led me to believe that Linux was fundamentally more efficient than Windows. So, I thought that if I ran a game like Quake 4 on Linux in its own X session without a window manager and the bare minimum of background processes that it would be blazing fast&#8212;certainly much faster than using bloated Windows XP with all its ram-hogging services and whatnot. Yet my eyes told me a different story. I had just played to three quarters through the game on my linux partition, and I was regretting it because the graphics were so much smoother and everything was faster on XP. I didn't go as far as to compare frame rates, but it felt like the difference between 30FPS with occasional stutters and 50-60FPS with zero stuttering.

So then I started using Windows XP again and I found the GUI to simply feel more solid. Scrolling, resizing, flicking through menus, highlighting text and dragging windows all happen instantaneously in XP. I changed the GTK theme in Windows to Clearlooks and opened up Pidgin. It just feels faster. Windows Explorer feels faster than any file manager I've used in Linux (pcmanfm, rox, xfe etc.) Even Firefox feels faster (particularly scrolling). The differences I'm talking about are small, and probably don't affect functionality, but it still irks me that they even appear to exist.

I'm fully willing to accept that it's just my mind playing tricks on me, but I still want to see if anyone else has the same experience as myself. Currently, I'm back to using XP. In about 6 months I'll probably find myself curious about the world of Linux once more and will wander back. I just hope that when that happens, whatever's causing me to perceive these differences (be it problems with drivers, GTK, X, or my eyes/head) is fixed.
