### Post by Riffer on 2008-10-11
Glad you like it.  Its a very rare occasion that you need the terminal for installing a program.  If its not in the repos then you could probably find a .deb which will install easily.  If there is no .deb its either a cutting edge release in which you need to worry about stability or there is probably 3 or 4 other programs with similar features that you could use instead.

While many are unfamiliar with using the terminal its a very powerful feature.  If nothing else it makes troubleshooting and fixing a down machine easier.  It also is easier to help people online using the terminal.  Instead of long winded explanations on how to find something in the GUI, you can share a command line with someone that can be cut and pasted into their terminal.  It has saved me on more then one occasion.

Having said that pretty well all the things that you can do with the command line has a GUI equivalent.
