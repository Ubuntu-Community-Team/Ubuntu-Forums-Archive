### Post by judge jankum on 2009-11-12
I'm a newbie, I've been using 9.04 for just a couple of months. And yes I've had "bugs" well sorta"
Each time I've had a bug,  I've installed a different version (older sometimes) of the program giving me problems and fixed it... While Other people use the default install of the same program flawless.... What seems to work on one system dosn't always work on another...But in Windows it's the same, some people get the freeze up or blue screen while some work good...
 Ubuntu is not (so it seems) a one size fits all system out of the box. But in Ubuntu you can match program versions and work arounds to your system specs to make just about any installation flawless....
The best system I've ever seen, and I started with Windows 3.11.....
