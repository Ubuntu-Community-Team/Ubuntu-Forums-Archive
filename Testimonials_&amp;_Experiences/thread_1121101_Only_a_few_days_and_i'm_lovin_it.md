### Post by wolfen69 on 2009-04-09
if you have the system resources, (i would recommend at least 2gb ram) you might want to consider using virtualbox to run windows. i've tried it (for testing purposes) and it ran beautifully. just install guest additions and you can run it full screen just like a regular install. you can't even tell you are running it within linux. saves time too, having to reboot. xp (virtual) loaded in 7 seconds.
