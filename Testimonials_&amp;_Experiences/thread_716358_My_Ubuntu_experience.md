### Post by Envergure on 2008-03-05
Windows users thinking of switching should read this.


In September 2006 I installed Ubuntu 6.04 on my XP machine, and I hated it.  Nothing worked and it was horribly slow, so I went back to XP.
In October 2007 I got sick of working with XP.  It was impossible to develop on.  So I installed Ubuntu 7.10 on it.  This time I loved it.  Everything worked flawlessly, and it was wonderfully responsive.

_The good:_
Ubuntu 7.10 is incredibly responsive.  It draws menus, etc. instantly, while XP tended to take a second or two.  It came with lots of awesome software, and it's very easy to develop on.  OpenGL runs faster in Ubuntu than Windows, too.  Software updating is quick and easy (you'll have to update something about once a week).  Ubuntu also boots a little faster, and there isn't that sluggish period between when the Start menu and the backdrop have loaded and when you can actually do anything.

_The bad:_
No commercial games.  There's a Linux version of most of the Unreal games, but that's about it.  With Wine, Steam will *run,* but not *install.*  There are lots of free games, though, and DOSBox is lots of fun.  Also, not all of the software is truely awesome, in fact some of it is pretty dismal, or still in development.  I can see Window computers on my family's network, but they can't see me.

_Software:_
I thought you might be interested in what I think of some individual applications.
Firefox:  Does its job pretty well.  Faster than IE at loading pages, but Flash is slower in Ubuntu than XP.
LMMS:  Not bad.  LMMS aims to be a free alternative to FL Studio.  It's lots of fun, and you really can write some good music with it.  I can't get it to update, though.
GIMP:  Awsome.  GIMP a fre alternative to Photoshop.  It's very powerful and easy to use, but it feels clumsy because it always opens like four windows.
Wine:  Results vary.  It runs Windows programs at near-native speeds but has trouble with DirectX apps.  My biggest peeve with Wine is that it doesn't do MSIs.  Nevertheless, I'm very impressed by the developers.
DOSBox:  Lots of fun.  DOSBox is designed to run DOS games, which can be downloaded from [www.dosgames.com](www.dosgames.com).  It does so perfectly, and it nearly makes up for the lack of real Linux games.
VLC:  VLC can run more sound and video formats than I've ever heard of, and it does so flawlessly, and without any extra libraries.
OpenOffice:  Depends what you're using it for.  OpenOffice has a word processor (awesome), a sildeshow editor (mediocre), a spreadsheet program (never used it), a "drawing" program (never used it) and a database utility (never used it).  It's good for doing homework and stuff, though, so I'm OK with it.
Compiz-Fusion:  This is the single most awesome thing that ever existed.  You can turn your desktop into a cube, or have the windows go jiggly, or any number of things.  It's just a silly toy, though, nothing useful.

_Is Ubuntu useful, or is it just a toy for geeks?_
People do make a living using Ubuntu, so I guess it is useful.  I think what a lot of them so is they start a free software project and make money from advertising on their websites.

_In conclusion:_
For my purposes, Ubuntu is definitely better than XP.  I would recommend it to anyone who wants to get into software development, anyone who hates the visual style of XP, and certainly to anyone with Vista on a machine that can't run it.

_A note:_
If you've ever noticed the "command prompt" in Windows and thought, "why would anyone want *that?"* then I'll tell you now that you will absolutely need to learn some basic text commands to get the most out of Ubuntu.  There are lots of Web sites about it, though, so don't panic!

_EDIT:_
Almost forgot--  Ubuntu has incredible support.  I haven't found a question yet that the people on these forums couldn't answer.
Oh, and one more caveat:  After I installed Ubuntu I had to update almost *everything,* which came to nearly 150MB, but if you already downloaded the CD image I guess that's not so bad.
