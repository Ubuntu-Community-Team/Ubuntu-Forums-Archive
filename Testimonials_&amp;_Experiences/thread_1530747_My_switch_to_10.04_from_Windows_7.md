### Post by user1397 on 2010-07-14
So I've been using ubuntu on and off since breezy (5.10), and it's always been the same deal... I get fed up with some issue I get on linux so I switch to windows for a few months, then windows does something incredibly frustrating so I switch back to linux and so forth.

This time though, I'm fairly confident my switch to linux is for good.

About a week ago I started experiencing weird freezes and lockups in windows 7 where everything was unresponsive, and the only solution was hard rebooting.  I thought it might be windows itself, but I installed ubuntu on a small partition and the same thing would happen.

I decided to just wipe the drive and install 10.04 from scratch and see if the problem persisted, and finally I actually got some sort of error message which told me it had something to do with my graphics card.  (I hadn't gotten the error message on windows, in fact I had gotten NO error message on windows whatsoever, and to be fair I only booted into my small ubuntu partition once where it just froze but gave no error message for some reason.)

Anyway, I tried dusting my computer (especially the graphics card), but the problem persisted.  I tried using the CPU VGA port but then my monitor wouldn't even detect my computer.  So I finally just removed my graphics card and used the CPU VGA port, and everything works fine now.

Sure, I can't play graphics-intensive games, but I don't really want to play any commercial games anymore, so it's perfect.

I think I'll install windows 7 in virtualbox just in case I need internet explorer or microsoft office for whatever reason (school/work), but I will remain happy with my ubuntu for all intents and purposes.
