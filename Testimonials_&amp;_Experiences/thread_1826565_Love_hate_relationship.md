### Post by Mikeriggin on 2011-08-16
Well after many years of reading about how great Linux is, I finally made the plunge and installed Ubuntu 11.04. I like the operating system very much, but have many problems with it. It hangs up constantly and I have to reboot my machine using the power button because the entire system becomes unresponsive. My latest adventure was trying to install virtualbox. I was finally successful installing the program (required 3 restarts) but that's as far as I can get. As I go through the process to set up a new virtual machine, there are many things that make no sense to me as a casual 15 year computer user. Many of the terms are new to me and make no sense. The end result is that I get a message that says I was unsuccessful because .... some technical jargon that makes no sense to me. I really need to be able to run a few windows only programs on this machine.. I understand that 11.04 is new and supposedly easier to use. I know that many many people have worked very hard on this project for a very long time. I also know that I am not a programmer and have no desire to be. I am afraid that I may have to be a slave to Bill Gates for the rest of my life! I'm not giving up yet, but could you people maybe think about making this thing easier for people to use that aren't computer programmers?
