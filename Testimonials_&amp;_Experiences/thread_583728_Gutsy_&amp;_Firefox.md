### Post by Phil Airtime on 2007-10-20
Alternatively, you can go to Applications - Add/Remove Software, make sure "All available applications" is selected in the top-right hand corner of the box, then enter "restricted" into the search. Select to install "Ubuntu restricted extras" and Flash, along with a lot of other useful closed-source bits and bobs (like fonts and Java) will be installed.
