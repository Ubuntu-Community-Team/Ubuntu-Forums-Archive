### Post by nuxl33 on 2012-01-17
First of all this probably isn't the right place to put this, so if it needs to be deleted or whatever go for it. I just need to get this off my chest.

Second please don't take this as me trashing your hard work if you are involved in Ubuntu in way shape or form.  I love Ubuntu for what it is, and for what it brings to the table for some folks. 


I've been around linux for 10 years now, and I haven't used it as a desktop in probably 6 years. 

I always kind of kept my ear to the ground to keep up with what was going on in the linux as a desktop world, so when I got fed up with windows it was natural for me to give ubuntu a shot.

Great distro!  Especially on unity I have no idea why anyone would bad mouth that, it's simply the best desktop I've ever used. 

The one and only grip I have is where is my ubuntu?

I can start with how root is locked out by default. I get why it's locked out, and that's actually quite genius. 

Yes I'm very aware that it can be enabled just with sudo passwd.

Is it really that hard to include that in the installer though? 

I think you can kind of see where I'm going with this.

Can I not be given a choice during installation to select either nano or vim as the default text editor? I loathe nano with every ounce of my being. 

I guess what I'm asking is will there ever be a fork of ubuntu geared towards  the linux admin? I can run debian all day long, but I'd rather use ubuntu.... 

Again great work and please don't think I'm bad mouthing your work.

### Post by cariboo on 2012-01-18
Ubuntu as it is now, is setup to be easy for a new user to install and use. The more experienced users are expected to know how to set it up to suit them. Ubuntu being a gnome based desktop, uses what the gnome developers have provided. 

For years Gnome has been removing administration tools, and with the advent of Gnome 3, have pretty well removed everything that was of any use from the default install. 

There are other developers stepping up to the plate, creating tools to change themes and icons, and other administration applets. For experienced users, Ubuntu has become more of a DIY distro, if you want anything other than the default installation.

BTW:

```
sudo -i
```

allows you to do anything that logging in as root does.

So, once upon a time, installing Linux meant you booted to an installer, got asked a zillion questions up front about whether you wanted foobaz or bozbar installed, as if you knew what they were and why, and then eventually after everything was said and done, maybe you got a desktop and got to explore all these nice things.  I remember that, took me 2 hours to install debian and when it was over, X11 wouldn't start because I answered  a question wrong about the mouse.  

Well, then there's this brilliant revolution where someone says "let's give people a ready-to-go desktop with the best programs, and make the installer ask as few questions as possible to make it brain-dead simple for new users to get to a working desktop, then they can change the configuration later when they know what they want."  Ubuntu grabs this idea and runs with it.  And here we are.
