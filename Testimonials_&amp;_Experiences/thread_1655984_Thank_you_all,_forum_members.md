### Post by rg4w on 2010-12-30
@CP2:

I do most of my work on a Dell Vostro 1400 - picked it up through Craig's list for $300, a value unlike anything I was ever able to find in the Mac world. :)  The Vostro has an Intel Core 2 Duo with 2 GM RAM and an Nvidia card for which the vendor provides an excellent driver for Linux.  In fact, I was speaking at a Mac developer conference last month and I had less trouble adjusting the projector's resolution than any of the speakers before me using their Macs.   And Google Earth just flies on that machine, even smoother than on my 17" Macbook Pro.  Nice as a laptop, and the video card works great with the 1600x1200 Dell monitor I found through Craig's List for $50 bucks. Man, I love discovering good hardware bargains.

I also have a very modest netbook I use when traveling: EeePC 901. The keyboard is a bit cramped for a touch-typist, but the humble Atom processor runs Ubuntu Netbook Remix 10.10 surprisingly well.  I even use OpenOffice and Google Earth on that machine and while it's not the snappiest experience it does okay.

I have a Lenovo IdeaCenter Q110 I use as an experimental cloud server in my office, also with an Atom processor but with a nice enough graphics card that I've enjoyed running Ubuntu 10.10 on it attached to a 20" monitor (though most of the time I manage it through VNC).

With all that fun going on I decided to resurrect an old HP with a crappy Celeron and only 384 MB RAM, and it's held up okay running Xubuntu for the few spidering experiments I run on it (though it's so RAM-strapped I'll probably switch it to Puppy Linux soon).

All of these installations have generally gone without a hitch, save for a few stupid user errors I made along the way.

So now I've  had enough good experiences with Ubuntu on modest hardware that I'm shopping for a new 17" laptop to use as my main system. The biggest problem I've had in picking one is something that only a long-time Mac fan can appreciate:  with Apple you have only a relatively few models to choose from and the components don't vary much among them, but since Ubuntu runs on darn near anything I have the whole world to choose from, and it's almost overwhelming. :)
