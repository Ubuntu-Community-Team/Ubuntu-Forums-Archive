### Post by SeanBlader on 2009-11-19
I'd been open to Linux for a while, actually purchased Red Hat 5 I think, maybe it was version 4, back when Red Hat was new and gave it a good look for a day, when I ended up giving up because of driver support or something, probably a resolution issue or something.

Then during the Feisty Fawn cycle a good friend of mine suggested I try out this new distro, Ubuntu. I trust this guy not to lead me astray, but still I asked about driver support and usability. He assured me that those were really great in Ubuntu and that I could just load up a live CD to try it out without changing my system, and that he'd even help me get it setup if there were any hangups.

So I figured out how to write and ISO (my first) from the work system (more bandwidth) and went on vacation. My last 2 days I had an opportunity to try it out, so I put in the CD and started up. Everything looked great on my old Inspiron 8500, it was faster, and easy to use, and had more apps than were on there previously, very cool. But no wireless. Broadcom. So we hung out one Saturday, started the install with 3 partitions for /, /home and swap, and went out to lunch. When we got back we spent about an hour getting ndiswrapper setup and used a couple of sneakernet based usb file transfer's from his laptop to mine, but in the end we got on the wireless and were good to go.

I've been with it ever since, with the exception of the brief windows laptop on the hp tx2000 tablet, which ubuntu didn't support well yet, I've since replaced that as well for another Dell laptop, on which everything works brilliantly on Karmic.

The end of the story though is much better. I've recently setup a LAMP server at home to replace the WAMP server, and just tonight I was on SSH to my Pre from my laptop, and using the command line on the Pre I was having it do a scp from the file server to itself. I'm really enjoying getting into the next level of Linux and doing some of the more advanced tricks, and doing those tricks gets better and better the more linux systems you have near you. Including the Palm Pre, I now have 3 regular systems that I use, and my home workstation that's dual boot, so I can still play the newest 3d games under windows 7, while still having access to Ubuntu when I need it on the same system. Overall, having a blast!
