### Post by seanc7 on 2007-11-15
I've tried a few versions of Ubuntu, Kubuntu and Xubuntu over the last 2 years and I always had to go back because something wasn't working right or I needed something that was only in Windows with no good Linux alternate.

But this time, I'm not having this problem.

Need to access AES Encrypted ZIP files? Install 7Zip and Q7Z for a GUI from the Add/Remove menu option.

Need to play WMA, WMV, etc, etc? Load the restricted drivers update from Add/Remove
.
Want a virus scanner because you work in .DOC and .XLS files all day? Install ClamAV from Add/Remove.

Want a firewall GUI? Install Firestarter through the Add/Remove.

Things have porgressed so much since v6.10 when I last tried Ubuntu.

I am very impressed and have had no reason to consider removing it!

My install, on a laptop, was simple, no issues except figuring out how to get my external monitor to do wide-screen mode properly. A little reading in the forums and I fixed it.

I love the fact that when my system starts up it takes half the time of XP (I used a stop watch). When the logon screen appears, the hard drive stops. Everything is loaded. Unlike XP where it kept thrashing away for another minute or two loading more stuff.

I'm very impressed with Ubuntu! Kudos to the team!

Sean
