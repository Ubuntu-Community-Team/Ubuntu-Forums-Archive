### Post by dawiba on 2008-12-08
Hello All

I've been tryng Ubuntu 8.10 for about a month now. It's installed on an old Dell Latitude D505 after trying out the live cd first. Installation was pretty painless. I partitioned the hard drive 50/50 between Windows XP and Ubuntu. All the laptop hardware worked as it was supposed to and as it had been under Windows. If anything, the wireless connection is more stable than under Windows.

Outside of that, I hooked up my ipod, camera, printer and scanner and all were recognised and I was able to use them. I had tried these while using the live cd, also without any problems. Following some help in the forums (10 things to do after installing Ubuntu.....), I downloaded some extra bits and pieces to allow me to use different file types and this all seems to work fine as well.

I've asked a couple of questions on the forums and got plenty of quick and helpful responses. Many thanks for that. I find the forums to be among the most newbie friendly I've encountered.

I use a Mac and my wife uses Windows Vista on her new laptop (which is why Ubuntu got installed on this old one). I find using Ubuntu fairly comfortable coming from the Mac and our kids have had no problems at all adapting to it. One bonus is that I use the Mac for music and I'll be able to use it solely for that purpose from now on.

So far, a very positive experience in terms of using the software and the support available through these forums.
