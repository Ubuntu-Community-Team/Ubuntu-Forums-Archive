### Post by QuazyPat on 2008-01-16
First off. I just saw that the site looks for similar thread titles when you enter your own. That's awesome. Anywayz.

Hello, all.

Figured I'd share how my experience w/ Ubuntu (7.10) has been going so far.

I decided to ditch XP the other day because Windows finally pissed me off to the point that I wanted it gone. Dumb errors, random crashes, the whole lot. That and it was doing really mean things to my HDD. So, my choices were:
 - Upgrade to Vista
 - Install a distro of Linux
 - Downgrade to Windows 98 (which I love)

I chose Linux. So I did some research on which distro would work for me and which one is most popular. I found Ubuntu and it seemed pretty cool . . . that I liked the name ^_^. So I burned the CD, ran off the Live CD for a bit, decided I liked it, wiped my HDD through Gparted 'cuz I really wanted to stick it to Windows and installed. The one thing that really got me was the fact that it just worked. A lot more useful features (like Gparted, Synaptc Package Manager, etc) that really made it more convenient. I booted up knowing that I was going to have to use the command line a lot more (hardly used it in Windows except maybe for ipconfig or dxdiag) so I looked up a bunch of basic commands like editing or renaming restricted files. How to install/update via the Terminal. All the basics. And after I felt a lot better about using it. 

Linux: 1 | Windows: 0

The first problem I encountered was my system randomly freezing. So I checked the forums, found a bunch of threads where people had the same problem and did as people suggested (turning acpi off in the boot params). And now I don't freeze any more. A simple way to fix a problem.

Linux: 2 | Windows: 0

And that was basically my last problem, too. I thought at first I'd miss playing games, but I realized I don't play as many games on PC any more as I thought I did. I hadn't touched WoW, GW, CSS or anything in a while. Been playing on my Wii and 360 a lot. Plus, I can install AA 2.5 which is better than the crappy 2.8.2 on Windows =) All I really used my computer for any more was talking w/ my friends, homework, whether it be an essay or programming, and just surfing the Internet. And since Ubuntu has FireFox ('cuz I can't stand IE), Pidgin and a word processor pre installed, I tackled all three things w/o having to install any additional software. 

Linux: 3 | Windows: 0

Overall, I'm really liking Linux. I'm learning to use the CL which will prove very helpful and I have a more stable and secure OS to use. I doubt I'll go back to Windows any time soon. Now if only I can get my school to switch . . .

Ciao. Cyaz 'round.
