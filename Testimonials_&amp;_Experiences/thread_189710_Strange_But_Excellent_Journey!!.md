### Post by deweese on 2006-06-05
I wanted to run a computer with Linux on it.  I decided to buy a Dell Server
Pentium 3 850mhz. 300SC Poweredge on Ebay.com just for having something inexpensive to learn on.  This came with Fedora Core 4 preinstalled from the seller.  When I got the computer, I tried it out.  I had trouble connecting online.
The way Fedora was installed, I would have to configure it myself
instead of the computer automatically doing it (DMCA).
     I decided to wipe out Fedora and go with Ubuntu.  I made it a dual boot with Windows at first(NT) but Windows crashed all the time so I deleted it and ran
Breezy but I still had the partition for Windows.  I decided the other day to do a fresh installation of Dapper because I was running out of hd space due to the partition I had with NT previously.
     I downloaded and burned Dapper Drake on a cd for installation.  What I found was that I had burned the Live CD when I thought I was burning the installation cd.  The labeling was changed from Breezy.  It used to be
"live cd or install", now it's "desktop or alternate install".
     I tried to install from the Live cd but it crashed.  I just put Breezy
back on and then upgraded to Dapper via download.  I was happy
because of all the new disc space.  Those updates can take some space.
     In Synaptic I saw there was a Kubuntu package.  I installed it.
I didn't realize it would change my system from Ubuntu to Kubuntu.
It runs just like Ubuntu but with the K and nice blue colors.  I also decided to
download and install from Synaptic a 3D Desktop.  I didn't really know what it was when I put it on, but it sounded good.  This program is really awesome
in my opinion which is why I am testifying!
     The 3D Desktop allows me to choose between a Gnome Desktop and a KDE
desktop on startup.  I just click session, choose Gnome or KDE and log in.
With the new wallpapers, this Dapper is a great looking os.  I am loving it.
     The only thing I wish for is that somehow I lost my new Dapper icons for computer and recycle bin and it reverted back to the Breezy Icons.  It happened in some of the installing.  
     Try out the 3d desktop and have a good day.

### Post by deweese on 2006-06-09
I think that the icons changed on my desktop because I went to Theme under preferences.  I didn't realize it but if you change the theme, it may change icons.  Therefore I now do not think Automatix changed my icons. I apologize for suggesting it did.

I took Kubuntu off my system so that it would be ubuntu again.  Once that happened, I wanted to switch back and forth from gnome to kde and it just wasn't the same.  My desktop on kde was missing many things.  I also had removed Konqueror.  I no longer was able to switch to many different desktop
wallpaper with a click.

When I wanted to go into synaptic to put kubuntu back and root too, it was no longer available.  I checked the repositories under Administration and then software, it would not show anything.  It would just crash everytime.

The system was broken.  What I did was do a fresh installation of Dapper Drake from the alternate install disc this time.  That was a pleasant experience.  It was just like installing breezy but it seemed faster.
Now I am back up with a fresh Dapper Drake installation.  I think if I want the full kde desktop as well as gnome, I'd have to install kubuntu on top of it again.  I think it would be cool if it would run as Ubuntu and give a choice of gnome or kde but it seems it has to be kubuntu to give that choice.
When I uninstalled kubuntu, when I shut down the computer said ubuntu.
When I restarted, it still came on as kubuntu.

On the bright side, the swiftfox browser runs super fast!  Dapper Drake and
Swiftfox loads websites quicker than Windows.  I proved this to my next door neighbor.  There was a site that was taking about 30 seconds for xp to load on a pentium 4.  Ubuntu Dapper Drake with Swiftfox loaded it in less than a second.  Pentium 3 with Dapper wasted and blew away a pentium 4 with Windows xp.
