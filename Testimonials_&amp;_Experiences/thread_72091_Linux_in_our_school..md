### Post by A-star on 2005-10-05
In the evening school I'm following, we have a course called "Netwerk software" where we learn now about linux.

We are using some book dated from 2003 (not my choice) which includes some outdated version of Suse.
This version doesn't want to be installed on Virtual PC -> we have to use this because we are not allowed to setup a dual boot.

By accident I had a copy of the Ubuntu Install cd with me which we all used to install ubuntu.
The teacher and the other students were amazed how easy it was to install.

We are now all happy Ubuntu users, and learning more every day.
