For a few weeks ago I desided to try Ubuntu cause I was bored,
so I formated an empty NTFS drive and installed Ubuntu (one of the easiest installations I have ever seen).
And that was the last time I saw Windows running on my comp.

I have had many annoing problems (and still have), but once I have figured it out, it works 10 times better (and resource friendly too). All the programs I use do what i want it to do unlike most windows programs that caused me daily irritation (and system crash from time to time).

So big tanks to the Ubuntu team, and the wonderful comunity :)
