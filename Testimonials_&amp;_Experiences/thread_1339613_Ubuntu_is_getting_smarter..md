### Post by thoffland on 2009-11-27
I just had to post a quick thank you to the devs of ubuntu... truly a great system! 

I've been using ubuntu since Hoary, and I've always liked it. But when people ask me about linux versus windows, I always say that there are things that Windows can do that Linux cannot. Today, I discovered just the opposite. 

My system is a dual boot with Ubuntu 9.10 and a recent upgrade from Vista to Win7. I'm on vacation and have a verizon broadband card for internet. It was working fine in Windows 2 days ago, but for some reason yesterday I couldn't browse the net! I still don't know what the problem is. I can ping any address, but the browser cant find it. Long story short, I decided to boot to Ubuntu. To my surprise it immediately found the usb adapter, asked me what the provider was, and the next thing I know I'm surfing the web again! 

So thank you to the developers for making such a great OS!! :guitar:
