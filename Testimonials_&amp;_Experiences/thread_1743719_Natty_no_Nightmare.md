### Post by ventrical on 2011-05-01
Iv'e seen the light !!

  After spending many years with Windows XP I have become naturally paranoid about malware and so I learned a little trick back several years ago about making a 'limited' account under Windows XP. 

The thing is that my paranoia came with me when I took up Ubuntu/Linux immersion and I made an EXTRA User_Account under Maveric 10.10 !!!!! before the natty upgrade.

 Now, after using generic recovery option at GRUB boot and a few tweaks in reduced graphics mode I not only have my *upgraded* 10.10 to 11.04 natty but also my clean Natty install and I am learning now that the Unity sidebar is in fact a sort of *compiz* in an of itself and with some tweaking and etc it can do some nice tricks.


So.. if there is a lesson to be learned today from all of this - MAKE an EXTRA user account BEFORE you upgrade online to natty ...because it WILL create a backdoor for you to use if you run into trouble.
