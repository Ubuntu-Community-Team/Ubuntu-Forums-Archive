### Post by f37u5g0d on 2008-04-17
I started dual booting xp and Ubuntu Linux about 6 months ago and I have loved ubuntu from the very beginning.  It is a joy to tinker with because it is so rewarding!  Fully functional Compiz-Fusion and the kiba-dock just for starters were both well worth the time getting them to work in the first place.
I am now considering completely removing the XP partition (I only have a 40g drive and is it getting cramped).  I came to this conclusion when I realized I hadn't logged onto xp in over a month.  I went through the start menu and realized the only program that I actually use xp for is itunes for my ipod.  If linux had the same kind of ipod support that itunes offers I would switch today.  The problem is in the file formats.  GTKpod doesn't offer AAC conversion or support of any kind so if I switch I would have to completely re-copy my entire (30g) ipod library.
I just don't understand how open office can cover every type of text, spreadsheet, math, and presentation format on the planet but multimedia formats are so difficult to work with.
I love ubuntu and have been reading a lot of Mac vs. Windows vs. Linux articles on the net lately.  I noticed that basically the only argument that anybody really has on Linux is that linux has a steep learning curve and often things don't work "out of the box".  I think what everybody forgets though is that 90% of linux flavors are 100% free.   Not to mention the endless open source software options on synaptics.  Linux has surpassed windows in every possible aspect.  I was using compiz before Vista's new Aero was available.  Compiz is better looking than Aero, provides all of the features of Aero plus about 20 more (that are even cooler) and it was free.  Linux updates all of its own software automatically (for free).  Linux doesn't fragment it's hard drive, it is more stable, offers better ram managment and its FREE!  I'll stop there because I could go on all day.
Basically all your paying for with a mac or windows os is hardware compatibility which in reality is the decision of the third party hardware provider.  They could just as easily write drivers for linux and then microsoft and mac literally would have NOTHING over linux.
