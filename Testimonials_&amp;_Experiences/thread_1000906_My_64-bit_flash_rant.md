### Post by lswb on 2008-12-03
Adobe just released their 64bit flash version within the last few weeks, if you installed it in Gutsy before that, it was the 32bit version. Sure most people install it without trouble, but there are many many ways it can go wrong as evidenced by the hundreds (maybe thousands?) of posts on this forum alone. 

If you ever installed a newer version (than available in the repositories) from Adobe in an attempt to get rid of bugs, it is easily possible to get the lib file installed in the wrong directory or get a symlink pointing to the wrong file. IME this is the main cause of trouble with installation problems.
