### Post by adza on 2008-02-18
Well, it's been well over a year now, thought it finally time to post into this forum. As of Oct 06, i was the proud owner of a flash 'new' 64 bit desktop PC and an old Sun Ultra 60 workstation collecting dust in the spare room... Things, to say the least, have changed... i now have linux on the desktop and it's never run better, i have 6.06 on my server and it's stable stable stable!! And, i now have a 'sneaky' partition on my work laptop that the sysadmins know nothing about!! But seriously, the reason that i'm hooked on linux is the learning curve i've taken in the last 16 months. I've gone from an 'advanced' windows user (now i know how funny that sounds), to still a rank n00b at real computing... but actually beginning to hack (it the good sense) properly.. i've learnt php, mysql and starting perl, written my own website (granted - it's crap but hey, it was always about the learning the functionality) and now am starting to learn about some penetration testing... now while this may sound sinister, i've already submitted the first report to my company on the first hole found in our online presence, needless to say turning some heads. So, in answer to the question 'is linux better than windows?', i can only answer the following.... 

There is no comparison whatsoever, unfortunately i think that the question is a moot point. Is linux a sensational desktop environment? Yes. Can new users use a linux desktop environment? Yes. But, i think the comparison is inept, it's about market segments really, it's like trying to ask if OSX will ever take over the desktop environment? No it won't, because SJ knows the segment he's pitching at, i think the linux people know that as well. After all, don't all us linux user take just a little bit of enjoyment out of commenting to people, 'well i'm not really sure about vista, i installed linux.. .etc etc'.. (i know i do - i will never buy vista). I think that ubuntu is not for humans, ubuntu is for enthusiasts. After all, isn't everyone proud of their ferrari?

thanks for permitting me my indulgence :)
