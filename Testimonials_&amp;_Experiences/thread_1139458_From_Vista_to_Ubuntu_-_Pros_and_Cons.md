### Post by ConfusedCartman on 2009-04-27
Using Wubi, I have installed the latest distribution of Ubuntu side-by-side with Vista. I have tried all major operating systems sans Linux, and I've heard good things, so I decided to give the most popular version a try.

**First Impressions**
First thing I noticed was the differences. Ubuntu does things so much differently than Windows, it's taking me a while to adjust. Initially, I missed the Quick Launch and Windows' default Time/Date location. Howver, once I discovered the power of Panels (which is genius), I quickly customized it to my liking. I retained some of the differences (the Main Menu still resides at the top left, the trash icon is best in the bottom right, and I like the Fast User Switch applet in the top right as well), but some things I moved (time/date and volume adjust are in the bottom right, I created a sort of "quick launch" of my favorite programs in the bottom left, and I moved the virtual desktop applet to the top left, immediate left of the Main Menu). What struck me hardest was the customizability of it all - it was incredibly easy to make Ubuntu what I wanted it to be. It very much felt like the Firefox of operating systems.

**Pros**
I love the interface. Panels is genius, and I love the out-of-the-box virtual desktops. It makes my workflow much simpler and quicker without crowding the screen. I also love how easy it was to get up and running - drivers were either already installed or, in the case of my proprietary nVidia driver, detected by the updater and after verification by me, downloaded and installed. All of this **without** going to the nVidia site. Unlike Windows, Ubuntu just *works*. That's one of the most compelling things about it - I don't have to worry about drivers, viruses, defragmentation, or network security - I just do what I do on a daily basis, and Ubuntu takes care of most of it without almost any work from me.

**Cons**
There's only two things I have to complain about. First is the barrier of entry - it's just not easy enough to switch from Windows to Linux. It took me a full two days to learn how Linux works, with "Help" assisting me along -  I just wish there was some sort of built-in "convert's guide" i that would assist me in understanding the differences between Windows and Linux and explain why the differences exist. I am absolutely sure something like that exists, but I think Ubuntu should point it out to new users automatically. Second (and a much more specific issue), I wish there was a way to add the Wine System Tray as an applet - I use uTorrent, and I don't plan on switching, so I'm sad to see that it resides in an extra window. I've instead opted to keep uTorrent open at all times in another virtual desktop, but the System Tray still displays regardless of uTorrent being open as a window or not, so I have demoted that to another virtual desktop as well. 

**Conclusion**
Ubuntu's a keeper. Even with it being a whole new beast to learn, I love the interface and the customizability. I won't be uninstalling Vista by any means (I am currently tethered to certain programs and I don't want to teach myself to use alternatives), but I'll only head into Vista once or twice a day to sync my iPod or do some site work (I run LBPCentral.com and I have a ton of related files on Vista). Otherwise, I'll be in Ubuntu, learning more as I go. :)

What I've started doing for my clients is converting their Windows install to a VMX file, wiping their computers and installing Xubuntu.  (I like it better because it has a much smaller Memory footprint than Ubuntu).  From there, I install VMWare Server and restore their Windows install from the VMX.  I then make sure that all Data is stored on a separate partition from all programs.  Here's the advantages that I get from running this setup:

The Host OS (Xubuntu) is immune from Viruses and Attacks.
The VMWare Server allows me to create Snapshots of the installation, which I save to the attached USB drive.  If Windows becomes corrupted (which happens a lot), or Infected (which happens a lot), I can easily restore an older snapshot of Windows , while retaining all the current Data.  It works A LOT better than Windows Restore Points.

The only downside is that the VM instance of Vista or XP can't run full 3D Games, which is not an issue for most of my clients.

If they're the type that likes to "mess around" with the system by installing things, I also install a separate VM of either Kubuntu, Mandriva or Ubuntu (and sometimes Suse).  Otherwise, I show them how to use Web Browsing, Emailing, Instant Messaging and other common functions in Xubuntu to try to start moving them over.  Most will use their Linux Installs at least for web browsing, and some don't use it at all, they still just use their Windows installations, but at least I don't have to worry about them losing anyone to Viruses, Malware, Corruption and data loss anymore.
