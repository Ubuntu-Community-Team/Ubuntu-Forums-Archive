### Post by SneakyWho_am_i on 2008-06-01

- in Windows 3, the desktop was a desktop and the filesystem was a filesystem. Very straightforward, no mind-bending grandfather paradoxes in the filesystem.
- In Windows 3, in the filesystem, you can distinguish between directories, associated filetypes and dissociated filetypes (and executables) at a glance. In Windows 95, the icons are almost completely random, and directories don't even exist any more (instead you have "folders" which can be any number of different things with the same icon)
- In Windows 3, you started programs by double-clicking them. Every time. In Windows 95 sometimes you single-click and other times you double click or use voodoo.
- in Windows 3, to exit to DOS, you select "Exit to DOS" .. In Windows 95, you have to select "Restart the Computer", and then hold down F5 during the boot sequence, and then do some other thing.... Really, horrible and painful and arduous.
- In Windows 3, you install drivers by putting in a floppy disk and running a program. In Windows 95, you basically can't install drivers. At all. Ever. Also, if you upgrade, all your devices stop working.
- Windows 95 takes 95 minutes to boot, Windows 3 only took 3 minutes.

Windows 98 was a little better when I got my hands on it, and later Windows 2000 would rock my socks, but that's another story.
