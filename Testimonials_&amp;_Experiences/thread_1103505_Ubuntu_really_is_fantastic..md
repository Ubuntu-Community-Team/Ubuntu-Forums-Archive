### Post by SonicSteve on 2009-03-22
I've been using Linux on my desktop now for roughly 2 years. At first it took me a while to become comfortable with the differences. Actually the first time I tried it I banged my head against a wall many times. Understand that to feel comfortable with Linux I needed to become familiar with the more advanced aspects of Linux just as I was with Windows for years before.

As with most Linux users I have started learning other distributions. This came about partly due to curiosity and also because I need to install library management software at my local church. Because of this I have been trying various distributions to which have been recommended by the software documentation.

So far I've tried Debian, OpenSuse, and Fedora along with Ubuntu. There have been other versions as well but those are big ones that many would recognize. 

So what is my point?

After trying many different versions I've come to really appreciate Ubuntu. Everything from adding and removing applications, networking, sources management.(and on, and on, and on) If you've tried any of these other versions of Gnu/Linux you will probably know what I'm talking about. 

Fedora's add/remove apps gui is nowhere near as friendly. 
Sources management in many versions is not as nice.
Setting a video card in other versions is not as nice
Debian was actually quite a shock. It lacked many of the extra things that Ubuntu has. 
Fedora had more than Debian but not like Ubuntu.
OpenSuse was quite nice and in fact had just as many extras but being a bit different required some adjustment.

Understand that I'm not bashing other versions but instead trying to point out how great Ubuntu really is. I know there are many great things about those other versions, and there are many versions still to try. I'm sure I'll have many people disagree with me.

[SIZE=3]In the end I just want to say,[/SIZE]
[SIZE=4]
Thank you Ubuntu Developers, your work is superb and you deserve to be told.[/SIZE]
