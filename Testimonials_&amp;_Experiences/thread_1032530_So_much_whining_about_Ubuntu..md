### Post by stchman on 2009-01-06
Sorry I have to just vent.  I am growing tired of the whining about Ubuntu.  I have Ubuntu installed on (5) different computers including (2) laptops.

One of the laptops is an Acer Aspire running Intrepid, the other (4) PCs run Ubuntu Hardy.  Three of the PCs including one of the laptops run 64 bit Hardy.  I have also installed Hardy on my friend's (2) PCs with not much issue.

Am I just "lucky", I think not?  There was nothing that kept me from checking the Ubuntu Hardware Compatibility List(HCL).  When I purchased my laptop(s) I checked to see if they had Linux friendly hardware before I purchased them.  I checked [www.openprinting.org](www.openprinting.org) to see if my printers work with Linux.

I chose Nvidia and Intel hardware as they are Linux friendly.  I refuse to buy Broadcom, Lexmark, etc. until they fully support Linux.

I also find it amusing when flaks tell me how difficult it is to install Ubuntu when they purchased their PC with Windows pre-installed.  Funny so their sample of one if a great indication.

Here is my Ubuntu install:

Boot up with LiveCD
Select Install
Select Partitioner
Select other install choices like time zone and keyboard
Wait for installer to finish
Reboot into Ubuntu
Install updates
Reboot into Ubuntu
Activate Restricted driver if Ubuntu tells me to
Reboot into Ubuntu
Go to Synaptic and install my favorite software

This is roughly a 45 min process.

Here is my typical Windows install:

Boot with Windows CD
Select install
Partition hdd as I want
Do pre-install
Reboot
Do post install
Reboot
Get ethernet card up and running with driver CD
Install AV
Install AS
Reboot
Install chipset driver from website
Reboot
Install video card driver from website
Reboot
Install Windows updates
Reboot
Surf to numerous website to install text editor, Zip, CD burning, Office apps, etc.

This process takes roughly (3) hours.

Now Vista is a more modern OS so more drivers are present, but Vista updates take FOREVER.

So which is easier?

My Rant.
