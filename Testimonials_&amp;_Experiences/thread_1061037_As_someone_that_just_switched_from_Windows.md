### Post by cptrohn on 2009-02-05
I have to say that I am very,very pleased with the linux world so far, and MORE than pleased with Ubuntu/Kubuntu.... 

It's a little tough at the very start getting used to how things work (and the thought of opening a terminal and actually typing in the codes scared me at first, but now I love it)

I had alot of problems with Windows (98, XP and Vista) always crashing, not doing what I wanted it do to (and not being able to get it to do what I wanted it do)... Since then I have installed a version of Linux on 3 different computers here, I drug my old desktop out of mothballs and tried puppy linux on it to start with, it was brought that old Win 98 machine right back into life and relevancy again.. It works like a dream now, actually better than it EVER worked with 98 on it (it was constantly crashing all the time, it's ran for about a month now without a restart with no problems at all)  Then it was Ubuntu on the XP machine, everything "just worked) after install and updates and a restart  the broadcom wifi, the whole works... then it went on to the newest laptop that had Vista pre-loaded.. the ONLY problem I had was with the Atheros AR242x wifi... but that was up and running in no time thanks to the linux backports I learned here..... the support of this community is 2nd to none.... it's awesome..  I then added the KDE4 desktop and I really enjoy that as well... The only thing that I regret was buying a Zune instead of an Ipod now.... but thats ok because thanks to this board I found out about Cowon and their amazing pmp's and have a D2 on order now...

So if anybody is reading this forum thinking about making the switch as a longtime person that used windows and switched over with basically no knowledge of anything about linux this has been a pretty easy process all-in-all and I don't think I have ever been happier using a computer than I have been in the last month or so of using Linux distros.. I wish I had done this years ago (I actually looked at computer with linux pre-installed and the salesman at Fry's told me that there wasn't much software for linux.... lying SOB... LOL)  All 3 of the computers I am soley running linux on now are working better than they ever have with Windows on them, and it's great not being restricted like I was with Windows for so long.... I know now that I will never switch back to a Windows system... and wonder why I put up with it for so long and all the problems I had over the years....


So thanks Ubuntu and the more experienced folks on this forum that helped me out so much...  it's great and very much appreciated!!   I'm one happy user right now.
