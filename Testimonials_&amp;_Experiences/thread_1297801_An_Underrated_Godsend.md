### Post by Verenos on 2009-10-22
I've been an Ubuntu user since the Gusty Gibbon days, and let me tell you from my experience ground up.

I was really fustrated with it at first (Mostly since I was in an unknown environment with no previous knowledge on UNIX* based systems.) I stuck with it and held myself together by dual booting two operating systems (Pre-Wubi era).

I got really into using it more after the release of Hardy Heron. Everything seemed like new experience and I wanted to dig deeper into the whole and see what else I can pull up. I started using compiz as my default windows manager in this release, and was simply blown away. Also because I could run a full 3D Window Manager on a computer that had virutally no decent specs at a reasonable speed.

This intrepid Ibex release I use now gave me problems with wireless chipsets, my Bluetooth Dongle not turning off, and with the usage of the Xbox Game Controller as a pad. However this release became my favorite boasting a good speed for my netbooks with custom kernels. Also this is my favorite release for typing,  saving, and testing programming languages.

The Juanty Jackalope is a very great release, However i'm not using it due to the fact it won't run on my netbooks at reasonable speeds without having a strong amount of flickering involved. So I installed this on my desktop dual booting it with Windows, and use it there. However not too often, since I take my netbooks instead since i'm on the go often.

Now for a conclusion. This System is a freakin' godsend!!!! It's sooooooo awesome once you get the hang of it. I use it everyday for intensive materials that require precision and it meets my tasks and duties with virtually no need for the Windows Operating System. The only time I use Windows is when there's a mandatory program that I need or anything that won't run in WINE. This is only a quick rough draft of what I experienced, there's certainly more to say. I'm not going to however, I already made this one long post. Why doesn't the whole planet Ubuntu often? 

THANK YOU FOR ALLOWING TO JOIN THE FORUMS AND SHARE MY EXPERIENCES.
