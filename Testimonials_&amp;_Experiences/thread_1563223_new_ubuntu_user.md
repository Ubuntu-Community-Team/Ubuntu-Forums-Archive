### Post by cookr35 on 2010-08-28
Well, it just popped into my head one day that I would do a search on linux. After looking at several decided to use wubi to try ubuntu 10.04. This was in June. Used it for about six weeks and decided to install to a seperate partition (never have messed with these before). Shrank my windows partition from windows xp, then booted into wubi and burned a live-cd(this was just easier to do in ubuntu) and installed. After I tested to make sure the new install worked, uninstalled wubi. All there is to really say for me is everything so far has worked exactly as it was supposed to. Rarely can I say that about windows anymore. I mean NO problems at all. Now I'm installing a few cli programs to play with, irssi and links2 are really fun to play with. Also trying to do most things in terminal to learn the commands and of course reading these forums. Only going to keep winxp for some old games I still play that won't run under wine.
