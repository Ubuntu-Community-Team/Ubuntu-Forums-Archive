### Post by DrMega on 2008-02-05
I'm sorry I'm not really an expert myself, but I have encountered some things that might help you.

Firstly, your sound issue. You can control the volume on various sound channels using alsamixer. Go to the terminal and run:

```
alsamixer
```

It raises a fairly basic but intuitive GUI. I'm not sure if it will solve the problem you are experiencing but it is worth a look.

As for the screen dimming out, is this power saving mode kicking in (ie does it happen at regular intervals like a few minutes with no keyboard or mouse input)? Power saving can be turned off via the Preferences menu. Also check your screensaver settings.

One thing I've found that I don't like is that whereas in Windows, the screensaver/power saving money knows not to kick in when you are running graphical stuff (like videos), but the same doesn't seem to be true of Ubnuntu. For my media center setup I turned off all power saving and screen saver stuff but I'm not sure if that would be appropriate with a laptop.
