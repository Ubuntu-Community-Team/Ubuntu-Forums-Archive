### Post by larrys4227 on 2009-11-12
First post ... hopefully in the right place.  No rants, no problems ... I'm just impressed with Ubuntu.

Alittle history: From roughly 1996-2003 or so, I was a pure Linux(Slackware) admin/user and had multiple machines running desktops and servers. My frontline machine to the world was a simple P90. When I took it down for good, it had an uptime around 2 years. I loved that little machine!! Behind it I had my own LAN with web/mail/ftp and spent alot of time playing and configuring my systems.

Got away from it around 2003 or 2004, and simply used a windows machine for my day to day PC needs.  Nothing fancy happening, just day-to-day internet needs.

Recently, both my WinXP laptops have experienced one problem or another. It feels like  the last couple of months I've spent way more time fixing their problems then doing other things in life I like to do. I've wanted another Linux machine for awhile .... Windows finally forced me to get on with it.

2 days ago I took one of the laptops (Dell Vistro1500) and loaded Ubunto 9.10. To say I'm impressed, is an understatement.  From partitioning for dual boot to the actual install was flawless. It recognized all of my hardware ... and while a few things needed a helping hand (Broadcom wireless), it was breeze.  The only thing that wont work is my Lexmark All-in-one, but I knew going in that it wasn't supported.

So far, anything I've thrown at this system I've been able to get to work easily.  Video, music, image manipulation, flash drives, data cards, etc. I like the software center too ... anything I've needed is right there, and if it doesnt work, I get rid of it and try something else.  So far, something in the list has always worked. Ubuntu is fast and so far very stable.

Granted, I'm only using Ubuntu as a desktop .... probably the only other thing I'll get running is Samba, but even then I dont do much pc to pc stuff.  All I'm looking for is a stable environment to do what I like to do ... Ubuntu is definately going to do that, and I can stop wasting my time on these XP machines.  Geesh, enough already!!

Enough of the soap box ... just wanted to let ya'all know that I'm impressed!  Linux desktops have come a long way since my old days of Slackware and KDE.

A short-cut to a terminal was the first thing put on the desktop!! Now, if I can only remember all the command line stuff I used to know .... :-)

Great product people!! Have a great day!  Larry (Florida-USA)
