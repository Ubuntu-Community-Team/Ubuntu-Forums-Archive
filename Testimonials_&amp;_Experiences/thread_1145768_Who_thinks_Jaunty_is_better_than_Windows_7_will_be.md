### Post by evermooingcow on 2009-05-02
ANY Linux based OS is better than Windows.
Only thing going for Windows is third party support.

Windows may be easy to use but thats until something fails. It is very difficult to troubleshoot. Windows is one giant package from everything starting with the kernel to the GUI and with a single giant config file - the registry.

This design is a serious flaw. Not only is it completely non-customizable, when one piece fails it can propagate to the rest of the system and there really isn't a way to fix something like that. Its just too much to manage. Its similarly easy for malware to break things too because if any of the many integrated apps (whether is be the GUI or the web browser) has a security hole it can possibly expose the core of the system.
