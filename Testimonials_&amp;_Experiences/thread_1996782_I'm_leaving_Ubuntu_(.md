### Post by ScottDeagan on 2012-06-04
I have been (what I would consider) a "hard core" Ubuntu user and evangelist for over 5 years now (I buy Ubuntu cups, t-shirts, mouse-mats and attended the last release party). Unfortunately, I have returned to Windows :(. The reasons why I have returned to Windows are as follows:

1. Window movement intermittently becomes laggy/jerky.
2. Cannot run secondary monitor without tearing (movies, dragging windows).
3. QuakeLive is jerky/laggy.

I'm not harshing out on Linux, I love Linux (and will continue to use it server-side/in VMs), and I despise Microsoft's business practises. However, the above mentioned 3 points have pushed me over the edge.

I have spent hours (weeks) trying to resolve the above issues. Unfortunately for me, I was unable to satisfactorily solve any of the above (i.e. I have tried setting "composite" to true in my xorg.conf, tried using the nouveau drivers, tried every "vsync" setting combination (in CCSM and the nVidia settings), changing mouse polling setting, using old proprietary nVidia drivers, using beta nVidia drivers, using xrandr etc etc etc).

It is with a deep and heavy heart that I have re-installed Windows 7 and am using it as my main OS :(. I will keep my eye on Wayland and will venture back into the wonderful world of Linux when Wayland is released. I'll also boot into Ubuntu every now and then to run updates and see if my problems are solved.
