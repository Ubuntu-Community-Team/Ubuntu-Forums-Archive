### Post by wolfen69 on 2007-11-13
i have had as many as 5 OS's on my PC. use a program like gparted to create the partitions. do the installations in this order: windows-mepis-ubuntu.i always install ubuntu last because it does a great job of seeing other OS's and getting them in GRUB. you should have no problems. make sure to create a swap partition also.(3 gigs will do)

good luck and have fun.
