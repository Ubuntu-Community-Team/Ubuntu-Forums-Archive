### Post by RagnarIV on 2012-01-28
I've used Ubuntu since my freshman year of college, back in 2005. So I'm no stranger to implementing work arounds and fixing my own issues. However I've reached the ends of my patience.

I'm sitting here on my ipad (I know, I know an apple product...) because after 12+ hours of work I am fed up.

Last night I upgraded my cpu/motherboard and ram. It was long overdue. I then did clean installs of Windows 7 and finally said goodbye to 10.10 and decided to upgrade to 11.10. I was expecting Grub to overwrite the windows bootloader and it did, however it did not detect the windows 7 bootloader, leaving me unable to switch back to my fresh windows install.

No worries, repairing a bootloader is pretty simple, except now my keyboard and mouse aren't working.. an issue I haven't had since 10.04 which was later fixed in 10.10 (bluetooth issues). 

So I dig out a USB keyboard, and try to run updates. Bam! 70 kb/sec downloads. I do some searching and 11.10 has issues with the realtek 8111E gigabit ethernet controller. Since when did we regress so far as to not have basic hardware support for common components?

So I find an alternative driver and the issue is resolved. Except now my Bluetooth keyboard and mouse are no longer working (still working on my laptop running 10.10!!!).

Now I enable my second monitor in the Nvidia Xserver config tool and all I'm getting is a white screen on my second monitor.

I just can't deal with it. Sacrificing hardware support, stability and ease of use for some crappy desktop environment is idiotic. I won't even begin to get into the issues I had with Unity.

So thats it I'm done with Ubuntu until they can demonstrate that have fixed the issues with this OS.
