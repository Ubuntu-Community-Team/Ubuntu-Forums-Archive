### Post by joe_lace on 2006-06-23
Ubuntu has been everything I hoped it would be and more.  Much of the reason for that has been these forums.  They have made it easy for me to find the help I need to get things up and running.  Most of the information I needed was already answered on this forum in threads that I was able to find easily using the search function of the forum.  Some of the things I was able to do seamlessly because of this forum are:

Install 3D card drivers,
Fix screen resolution issues,
Share Files and Printers from ubuntu to windows XP,
and lastly,
I was able to use [this]("http://www.ubuntuforums.org/showthread.php?t=84275") thread to allow myself to use a Virtual Machine image created using qemu with VMware player.  Now I am enjoying the best of both worlds. It works swimmingly and I couldn't be happier.  

Here's to hoping that future days will be as happy as these first three have been.
