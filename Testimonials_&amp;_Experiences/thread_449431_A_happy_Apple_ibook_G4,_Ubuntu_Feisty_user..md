### Post by tcrroadie on 2007-05-20
I just wanted to share my joy and happiness on my successful adventure installing Ubuntu Feisty Fawn on my Apple ibook G4.  

Last week my nerves just could not stand not having Linux on my Apple ibook anymore.  One year ago I did try out a live cd of Ubuntu Dapper on my ibook (for about 5 minutes) but just didn't feel the need at the time to go through all of the work installing it.  But lately I have been having so much fun using Ubuntu Edgy on my desktop system with Beryl that I had to give installing some form of Linux on my ibook a go. 

So I did some research on a good Linux distribution for ppc to start with and came across Yellow Dog Linux developed by [Terrasoftsolutions]("http://www.terrasoftsolutions.com/").  So I downloaded and burned the latest release of Yellow Dog 5.0.1 to a disk and reinstalled Mac OSX so I could create an extra partition to due boot.  Installation of Yellow Dog on my ibook was very easy, similar to installing Ubuntu.  I ended up only using Yellow dog on my ibook for about 3 days.

**Things I liked about Yellow Dog**.  

1.  Easy installation.

2.  Fairly easy to get my wireless g network card working. 

3.  Good documentation on Terrasoftsolutions web site. 

4.  Easy to use package manager (YUM).

5.  E17 is preinstalled. 

**Things I disliked about Yellow Dog**.

1.  Feels unfinished.

2.  Hardware detection is flaky.  Yellow Dog choose fbdev as the video driver for my graphics card.  My ibook has an ATI FireGL Mobility T2e chipset.  This is a no brainer, Yellow Dog should have choose one of the open drivers "ati" or "radeon".  I had to manually edit my xorg.conf file to set my video driver to radeon.  

3.  Version of Gnome used is very old, 2.14.  After using Gnome 2.16 and 2.18 for a while, 2.14 feels very slow.  

4.  Small software repository.  

**Now we will get to my experience so far using Ubuntu Feisty on my ibook.** 

Installation of Feisty is typical fair, easy.  It felt very good having apt back again as my package manager.  Apt is by far the best package manager on Linux at the moment.  It is fast and easy to use.  
[B]
Things I like about Ubuntu Feisty on my ibook.[/B]

1.  Configuration of my wireless card was incredibly simple thanks to bcm43xx-fwcutter being included in the Feisty repositories.  Took me less than 3 minutes to get my wireless card working. 

2.  Hardware detection during installation is A+++.  Ubuntu set up my video card properly by choosing the open driver "ati".  Video performance has been great so far. 

4.  Gnome 2.18 flies on my ibook.  Love it. 

5.  Software repositories contain some of my favorite applications such as mplayer. 

6.  Thanks to the repository [medibuntu]("http://medibuntu.sos-sts.com/index.php"), I can play my mp3 music files and watch videos using the media player VLC.  

7.  Stable.  Just works. 

**Things I don't like about Ubuntu Fiesty on my ibook.**

1.  No Flash support.  

2.  I can't think of much else to say here.  


I'd like to thank all of the developers that continue to work hard on porting Ubuntu to the ppc platform.  Without you I would not be able to continue my joy in using and learning more about this fine operating system on my Apple ibook.

I'm also including a screenshot of my custom Gnome desktop for your personal enjoyment.
