Well, I gave the "LinuxCAD 2009 for Windows 7 demo" a try, and I wasn't too impressed...
I did try it on XP instead of 7 as it is advertised to support, so bare that in mind while you read this.

The interface is a bit funky as it's a little like GIMP where you have toolbars floating in other windows and a main drawing window, but if you could get used to that it would be fine.

Despite it being advertised as using all of the same commands as AutoCAD, zoom was one command I noticed that simply did not work by typing it in – if there's one, there are probably others.  There was zoom functionality, but it was buried in the menus and the program was running so horribly slow that I couldn't actually tell if it was zooming or not, so I ended the task since it appeared to have locked up with the attempt.

Personally, if I were to need this, I'd contact the maker of the program and give them my horrible experience under Windows.  I'd then attempt to get a demo or trial version for Ubuntu (which I didn't see on the website) running to test that before paying a dime for this.  But that's just been my experience, if you test it under Windows 7 you may find it works perfectly, or you may purchase it and have it work flawlessly on Ubuntu.
