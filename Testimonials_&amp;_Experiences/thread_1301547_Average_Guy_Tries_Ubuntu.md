### Post by ancientpaint on 2009-10-26
I am an average windows user with very little computer knowledge - a web browse, AIM, maybe an mp3 kinda guy.  I  installed Ubuntu on my machine in order to access files when my windows computer caught a virus.  Thanks to ubuntu, I still have access to my archive of irreplaceable home photos and videos.  After saving some data I deleted windows and now only use ubuntu.  I want to respond to critics who claim one must be a computer expert to run ubuntu. This is just false.  Sure, there is a learning curve as with any OS or new experience in life but Ubuntu is very easy to use.  Learning the sturtuce of the file system took 10 mins. Thats it.  I even installed a varity of programs using the add/remove function...even a program that can copy online radio streams.  I love it.  I caught the linux meme and now am I am trying to convert all my firends, lol.  I just want to say thanks to everyone that has worked hard on this wonderful project.   Is there a place to donate?
