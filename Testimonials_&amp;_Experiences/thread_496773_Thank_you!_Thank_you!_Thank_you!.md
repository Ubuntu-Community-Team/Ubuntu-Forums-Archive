### Post by joelkm on 2007-07-09
Hello there, my name is Joel. 

I have been a windows user for a long time.  A few months ago I purchased a snazzy fully loaded Dell Precision M90 laptop.  This bad boy came loaded with 4 gigs of ram, 2 ghz dual core, 512 quadro graphics card, not to mention Windows XP.  Honestly all was great for the first couple days but then the typical error messages, blue screens, and lack luster performance that we have grown so used to started appearing.  After countless re installations, Vista appeared. Whoopee! My savior! The future operating system! Well after a clean install I came to this conclusion, "What Filth". My great new laptop ran like a decade old machine!  There must be an alternative!

There is..it's called Ubuntu! To tell you the truth I have always had a deep fear for any Linux based OS.  I mean, only hackers use Linux right!? Wrong! Installation was a snap, and 3 days into it I am in love! I am so impressed with how brilliant Ubuntu functions. There are all these incredibly cool features, I especially love how Ubuntu will update everything from your repositories automatically. I mean...thats just so cool! That sure beats surfing the web for hours looking for individual program updates. I installed Compiz Fusion and everything looks so cool!  Wait..and everything is free! OMG I think I'm going to have an aneurism!

Thank you Ubuntu!
