### Post by KBGT on 2008-09-03
Thank you to everyone involved in making Ubuntu a great OS.

I've had all the Windows releases only to find myself buying the latest anti virus to keep my system bug free and up to date. Only to get infected again with something or another.

Lets do a scam disk, or a disk de-frag, or clean this up. Finally doing the old f-disk and a new install to get it up and going again.

I was a the book store and came upon Redhat 8.0 with an install disk. Crossed my fingers and had it Dual-Booting with windows. Wow, I was an official Dual Booter now. This was cool, and most stuff worked except for the multimedia stuff, but after a fwe hundred RPM's I got some to work.

I was hooked. I tried Mandrake, another Redhat, and Suse.

Then I became an RPM junkie. I had them all over the place. The desktop, folders and the second hard drive. Just one more and it will work, only to find that this RPM is already installed. No! No! No! it says I need this RPM to make it work.

Then comes UBUNTU. Well why not, I've tried everything else. 

Within a few hours everything is working. I felt cheated. This can't be. I'm watching a DVD for the first time on this thing and playing mp3's.

Thanks to the documentation on the UBUNTU website it was a painless experience. I've even used the terminal a few times with success.

Thank's to everyone for a completely functional bug free computer.
