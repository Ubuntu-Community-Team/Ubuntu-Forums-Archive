As far as doing updates perhaps I shouldn't recommend that. It is just my personal preference.
If you do updates daily or as they come in, there is a risk to some people that something or some program might break.
If your willing to sort through the list of updates and just do kernel updates and specific security patches I would think that would be less risky to the integrity of your system and programs.

As far as leaving your /home folder intact and installing over it.
Many of your program and system settings will remain, making setting things up less work.
Your music and data files will remain untouched.
Any programs that Ubuntu doesnt install by default will need to be reinstalled.

Lets say you have gdesklets installed, when you go to install it again on the new install the icons that you were using will still be in the gdesklets folder.
So you can create a new launcher and just drag and drop the icon launchers onto the new launcher.
Your evolution mail box will remain as well.
You might want to export your Firefox bookmarks and save them then import them back.

Bottom line: Doing updates is a personal choice. If my machine is running rock solid and my programs are behaving as they should. 
I leave it alone regarding updates.

Linux is a great OS but you have to use some restraint. This machine is very stable, but if I turn on one too many features in Compiz it becomes very unstable.
