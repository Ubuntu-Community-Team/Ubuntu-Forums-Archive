### Post by Jefferson Heard on 2007-02-01
We recently built a visualization wall -- 16 projectors in a 4x4 array using an nVidia Quadro Plex and Matrox DualHead2Gos from each channel.  Ubuntu had no trouble with this whatsoever, and we are now running Edgy Eft, with the nVidia drivers and the latest X.org at 15 megapixels.  Tried other distros that were supposed to be more "production" friendly, and Ubuntu was the most reliable setup for this.
