### Post by armandh on 2011-09-05
there always seems to be some missing piece of the puzzle

this weeks experience
I have been a happy Ubuntu user since 7.04
sadly I have been stuck with an XP dual boot due to the must have, mission critical and win only Intuit Quick Books.

with the XP home showing its age along with the early P4 Dell it came in,
**if** I am going to spend the time reinstalling all my stuff, why not upgrade.

twice the width [64 bit]
twice the speed + some [3.4Ghz]
eight times the processor threads
and twelve times the memory

Wowzer the stuff opens fast
shrank the 7 partition, got lazy and put in mint
OK, so it is really Ubuntu with all the after market parts.

and discovered that my proprietary e-mail client in XP was a no go to any win after xp
[much less to Ubuntu]
worse still all 11K of my old e-mails were in an unusable file format unless one likes reading them in order in a spread sheet

the customer service wizzes suggested e-mailing them to my self.
I had a faster P4 that I was thinking about using. it was already Ubuntu/XP dual boot 
so it got the old e-mail client and file
talk about getting away from windows, I cant even get away from xp
at least I can find the old e-mails now.

NOW here is the **GOOD** part. I mounted a searchable image of the old drive in another new partition in the new box and yanked the drive from the old box to store in case I had missed saving any thing. I was wondering what to do with the rest when I spotted a drive with Ubuntu 10.10 still on it.

plunged it in and BINGO a fully working computer, quite snappy for an old P4 with 512 Meg of memory. already all decked out with all the medibuntu aftermarket parts it was WAY EASY !!!

Ubuntu just works GREAT on the old 1.6 Ghz P4
and, not surprisingly, so does mint on an i7
and so does Ubuntu 10.10 on the xp e-mail dual boot box [2.4 Ghz P4]

getting all the old printers/scanners to work in 7 was hard.
sort of a reversal of fortune as only one printer was stubborn with Ubuntu.
