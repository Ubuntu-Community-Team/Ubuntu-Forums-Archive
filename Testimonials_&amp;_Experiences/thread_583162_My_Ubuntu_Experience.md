### Post by Humph on 2007-10-20
Apologies for the novella-length post!

I installed Gutsy on my desktop machine this week, and with the exception of having to start the Live CD in safe mode to get the install to not hang at 15%, I've had almost no problems at all.

My on-board sound card on my nForce 2 motherboard wouldn't work in World of Warcraft since the voice chat patch, so I installed a spare SoundBlaster I had kicking around. It works okay in XP, but not in Ubuntu. The sound card works, but WoW really doesn't want to deal with it, and just locks up halfway through connecting to realm. So I looked around on Amazon. There was only one sound card that supported Linux, and most of the cards were ~£50. Anyway, I bought the Linux-friendly card and it arrived just in time for my 7.10 install, which was a nice bonus. Needless to say the sound card was detected and installed without recourse to the driver disc. Oh, and it works great in WoW - which, incidentally, runs much smoother under Gutsy than it ever did under Feisty. And the cost of this uber sound card? **£2.80**

All my hardware was detected and installed correctly, including my scanner. My printer wasn't detected, but as it is on a LAN print server, I really would not have expected it to be found!

My laptop install of Gutsy was similarly straightforward. The only sticking point has been getting wireless working. Rather annoying to have to start XP to turn wireless on and off.

Both of these installs took around 25 minutes each, and required one restart each.

This morning I spent from 06:00 to 10:00 re-installing XP on a Sony Vaio for a client. Four hours, and *13* restarts to install Windows, Windows Update and Sony Update. So 12.5% of that install time was waiting for restarts to complete!

What I found most annoying about the XP install was the amount of time spent watching Windows "Preparing" to do things. I wouldn't be at all surprised if another hour of my time hadn't been spent watching installers effectively doing nothing.

Now if only I could get Visual Basic 6.0 to work in Ubuntu...
