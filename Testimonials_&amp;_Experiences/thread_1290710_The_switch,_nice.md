### Post by dyingsun on 2009-10-13
Yay, new convert!

As you get more experienced with it, you'll wonder how you ever got anything done without it.

As for gaming, WINE is great for some things, but you might want to install Virtualbox with Windoze running inside it to allow you to run the games that dont work on WINE.

Word of advice: go to Synaptic and install  nautilus-open-terminal . It allows you to drop to the command line from any Nautilus folder, and drops you in that particular directory. It makes life soooo much easier when youre setting up and installing stuff.

Also, try this for setting up Flash:  [Installing 64 bit Flash on Linux]("http://usefulsoftwaregamesandknowledge.blogspot.com/2009/09/install-and-run-64-bit-flash-on-linux.html")

Good luck, and bookmark these forums! Don't know what I'd do without this bunch!
