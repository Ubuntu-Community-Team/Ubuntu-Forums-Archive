### Post by reiki on 2007-06-10
Ah yes... the ties that bind and apron strings finally cut and all that happy horse hockey. I am FINALLY not going to have to boot to WinXP any more.

Oh I've been an Ubuntu user since August of 2005.... but I had to keep an XP install on hand for just 2 items.

#1 - My Logitech Harmony remote. It connects via USB for programming and updates and this only works in Windows. Logitech's site has been claiming "Linux support coming soon!" for about 2 years. VMWare server running an XP VM did not handle the fact that the device would connect and disconnect during the update process.  VMWare Workstation apparenlt handles USB better, but.... close to $200 just to program my remote? I don't think so.

#2 - I use a CPAP machine at night ... keeps me breathing.... which of course keeps me alive. The machine has a smart card that holds data on pressure requirements and a bunch of other stuff. The manufacturer flat out refuses to release enough information to allow the card to be accessed by linux EVEN THOUGH there are drivers for the smart card reader that would probably work. The manufacturer uses a windows-only program that puts the data into an MS SQL database. In order to access my data I had to reboot to windows once a week to download the data.

VIrtualBox to the RESCUE! It handles the USB thing perfectly and both of the aforementioned devices run through their paces in a XP VM without issue. 

I could tell you about how everything ELSE I used to do in windows, I now do in Ubuntu, but that'd probably be boring because by now LOTS of people are doing the same thing. :)  But these 2 items and the fact that VirtualBox on Ubuntu makes it possible.... well.... I'm free at last.

Thank you, Ubuntu!
