### Post by Irihapeti on 2010-05-13
I had Karmic on a similar machine - Toshiba Satellite A10 with 256MB of RAM and a 30GB hard drive.

A full install of Ubuntu or even Xubuntu was struggling. It felt like wading through treacle. Instead, I installed a command-line system from the alternate CD and then added Openbox, tint2 (panel), and various other programs. That was a while ago and I don't remember what all of them were, but I do know that I installed Seamonkey instead of Firefox because it uses a bit less memory, and I was able to run OpenOffice - the version from openoffice.org, not the one from the repositories. Gpicview was the image viewer. I don't think I had any music-player, but if you really want to, you can install commandline ones.

I was also running conky so as to keep an eye on memory and so on, and I seem to recall that without any programs running, there was about 80MB or so of free memory. Occasionally when I was running something there would be a bit of swap space used, but for the most part it managed on RAM alone.

I think I did something similar with Lucid during the development phase, but I tried so many different things during that time that it's hard to remember now.
