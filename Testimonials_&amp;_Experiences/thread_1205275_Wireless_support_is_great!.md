### Post by fela on 2009-07-05
My mum got a laptop a few months ago, it had a dud, unactivated winblowsucks XP without any drivers (not even graphics drivers). Try as I might none of the drivers available on the net for the ethernet OR wireless would work, so I thought - why not install Linux on it? Sure enough, when I booted up the livecd for Ubuntu 8.10, the drivers for EVERYTHING including hardware graphics acceleration, wireless, ethernet, suspend/hibernate etc. were supported AUTOMATICALLY off the LIVECD!!! This is on a IBM Thinkpad T42. So, from my experience the driver support for some laptops is BETTER in ubuntu than it is in winblowsucks!

So don't bash Linux/Ubuntu for lack of driver support - every OS suffers from it.
