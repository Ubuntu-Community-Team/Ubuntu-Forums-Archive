### Post by Arcuru on 2007-09-23
I'd tried Ubuntu once before, but never really got used to it (mainly because the computer I had it on had no internet access :( ).  Anyways, a couple days ago I decided I wanted to try it again, so I set up my laptop to dual boot with XP.

After using Ubuntu for a couple hours, I realized just how much easier it was and, whether it was a good idea or not, completely removed XP with the intention of not going back.  The first impression was great and I haven't regretted it yet :)

A couple things I love
 - Amarok > iTunes
 - Installing and removing applications is far easier than XP

The few things I'm still having problems with*
 - My wireless card is installed and working, but I haven't yet been able to connect to my universities wireless network (a "hidden" network with WEP encryption)
  - I installed my NVIDIA graphics card with ENVY, but every time I reboot I have to change the screen resolution manually ( it says it can't overwrite the backup config file)

EDIT:  The tips below helped me solve them, so no more problems :)

I haven't tried to install my printer yet so I don't know how that will go (Dell Photo 926)
