### Post by vhunterd on 2007-11-04
Hello everyone, I'd like to start off by thanking everyone for the large amount of information that this forum has given me. From getting my wifi card to work correctly to getting the correct drivers installed for both of my machines, I must say that I would not have gotten as far without this community. I continue to run Ubuntu versions on both of my machines. Thank you everyone.

I had originally decided to go with a linux distro because I had a required Linux class for my degree (Computer Programming Major), at my local community college. The first distro I decided to install was OpenSuSe 10.2. 

I must say that even though I don't use OpenSuSe anymore, although, I could never quite get things to work right, it was a good experience for me. The problems I had are the reason I decided to change to Ubuntu (or perhaps it was my boss constantly telling me how great it was, at work). OpenSuSe 10.2 simply did not like my video cards, I spent so much time simply just trying to get xserver to start, I almost lost interest in a linux based distro altogether. YOU never quite worked either, I could barely ever download updates (yet I could still "see" them in the package update listing) and when I did, it always screwed up either xserver or my video card driver. I usually ended up having to edit my xorg file manually because Sax2 never seemed to work with any driver (VESA, VGA, NV, NVIDIA all never worked). I remember, I even crashed my system by typing man woman into a terminal session (why this crashed my system, I will never know).

When I first installed Ubuntu (7.0.4), I was actually amazed that everything worked (with exception to my wifi card which got fixed {not much} later). I was actually able to install all my apps from the included package manager or the apt-get utility. I have to say, this was the main reason I decided to stick with Ubuntu, and use it for most things over windows (I only have it installed for some class programs that I can't use WINE for). I have to say that I enjoy using Ubuntu much more than windows. Funny enough, I actually found out that my new wifi card drivers would work with WPA2 based encryption in Ubuntu, for some odd ball reason, ALL my windows machines required a patch not found in the regular updates to enable WPA2 to be used. It seems like every day I've used ubuntu, I've learned something different. And I like that, alot.

As I've been creating a game recently and needed to use functions/macros from the ncurses library package, I've found that linux is really great for development as well, at least, for the programs I create. I love using Anjuta over my old windows IDE.

Overall, I must say that I'm going to be a linux user for a long time. I have to say I was tempted to go with OSX, because, it's somewhat freeBSD based, but I like the open source way of life a bit better =). Perhaps I may try other distros in the future, just to see what they offer, but, I have to say Ubuntu will most likely always be my #1 choice when it comes to an OS.
