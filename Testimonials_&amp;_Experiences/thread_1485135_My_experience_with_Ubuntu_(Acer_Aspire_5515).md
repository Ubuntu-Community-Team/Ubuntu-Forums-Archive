### Post by Stigmata13 on 2010-05-16
From the first time I heard about Linux I always wanted to try it. So about half a year ago, I decided to give it a go. But knowing little about it, I ran into a little trouble. I'm not a heavy gamer, but there are a few games I like to play (such as Diablo II), and after installing Ubuntu 9.10, I found that my computer's performance playing games was very poor.

   So this ultimately led to me running back to windows. Linux became something I was proud to have, but it was rarely used. But recently, I decided to look into it further.

  After looking through many guides and posts, I found that my graphics chip, radeon mobile x1250, was put on legacy by ati, which meant that all the newer releases used open source drivers. I've grown to be a fan of open source, but when It comes to drivers, I still believe that it's best to use the ones on the manufactuers site. Anyway, I found that if I downgraded to version 8.04, the LTS, I would have much better graphics support. The installation went smoothly, but fresh out of the box, I found that the wireless didn't work. After doing some research, I found that installing the backports would fix this problem, and it did.

  So I updated, and installed wine, and gave the game a go. I found that according to the fps, it was running **even better** than windows! I was very happy about this. So I started playing for a while, and noticed something. I would get random lag spikes, about every 30 seconds or so. I figured it had something to do with my wireless, so I looked around, and that gave me an idea.

  I found that you could use windows xp drivers in linux using a special tool. So I downloaded that tool, and downloaded the drivers, and gave it a go. And to my surprise, the game started running smoothly!

  So with my game fix, I was confident I could leave the world of windows behind for good. Except, there was one thing that was bothering me. After installing firefox 3.6, I noticed that flash was slow and buggy, and would even crash firefox if put into full screen. I tried various things, but none seemed to work, until I got the idea to try a different browser. I downloaded Google Chrome (I've been a long time Firefox fanboy, but I figured I'd give it a try.) So I went to youtube.com, and opened a video of one of my favorite artists, and to my surprise, it was running smooth. Fullscreen would work, although a little laggy (but with a 1.6ghz cpu, I can't really complain!)

  Anyway my time using linux has been amazing. I'm learning more every day, and I'm proud to be free of windows, and all problems windows causes! 

(And by the way, if you happen to use this model of laptop, I highly suggest you try 8.04 if you're not already. The linux drivers from ATI's site perform far better than even the proprietary drivers Ubuntu installs for you!)
