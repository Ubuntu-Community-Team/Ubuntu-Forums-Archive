### Post by eksasol on 2010-03-22
Well, not charity per say and not a really review of Ubuntu, but I want to share an experience I had with Ubuntu, probably more to do with the linux philosophy as a whole.

Couple years ago I worked at Goodwill for a short time and there were lots of old computers and other computer peripherals that were donated (printer, scanner, digital cameras), we get hordes of them. I told the store they can't reuse/resell the old PC because of the Windows license issue, unless the harddrive was wiped, so the computers were being sent off else where. Testing the printers and scanners would have been a pain on Windows since one would have  to go download printer drivers online for all these printers. (Remember the only computer around with a proper license of Windows for the store to use is the manager's, and its Windows XP anyway which doesn't really have any drivers bundled like Vista or 7.) These peripherals were being discarded (into the dumpster) because there wasn't an effective way to test them.

So after a proposal, I took the best PC that was donated, wiped out XP in it and installed Ubuntu. Then set it up as a testing station for the peripherals. What's great is any printers or scanners that are plugged in gets detected immediately. For the purpose of reselling these things, no need for advance printing features, just to see if the thing can print. USB wifi cards? Plug it in and it gets detected immediately. So the other workers doesn't really need technical skill to test them.

Beside the obvious benefit of linux in this scenario, really from my experience Ubuntu does support quite a lot of hardwares out of the box which is why its so useful for the tasks I mentioned above. Notice the fact I mentioned having to go and download individual drivers for each printer if Windows XP was instead used referred to the problem with closed sources. Athough you could argue the other way around that its harder for newbie to install drivers in linux. I don't find that to be true anymore because of the repository system, just that some of these hardware companies don't want to support linux.

Good thing with me, I've yet failed to get a device that I bought to work in Ubuntu that also works on Windows. However, not so true with others so I can sympathize. So I think hardware compatibilities is one of top priority if Ubuntu want to win over more users, though I know the Ubuntu team and the developing community knows that.

### Post by julianb on 2010-03-22
I've been installing Ubuntu on low-cost machines that are used in Haiti. 

As you might imagine, people in Haiti want: an OS that is in French, which works on low cost hardware, and is reliable / does not get a lot of viruses. Ubuntu works out great. Don't have to deal with software that tries to determine whether the OS is pirated or not... don't have to deal with an OS that can't change language (all of the windows installs people have there are English-only).
