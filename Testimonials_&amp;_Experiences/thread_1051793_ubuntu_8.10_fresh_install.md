### Post by Bs0 on 2009-01-27
Always had window OS on all the machines I owned. I tried mandrake once but couldn't connect to the net, so I gave it up (yes,after 30 mins).

Recently I got a laptop given to me but the discs were lost and XP was not booting. I thought lets try Unbutu (someone told me how well build it was)?

I'll be honest i didn't expect much from it. I downloaded the ISO image from my xp rig, burned it and whacked the coaster in the laptop.
I went with the install "full wipe of windows", grining in anticipation of failure, how many hardware devices is this going to recognize? this is going to be fun!!

anyway it installed asked me my name, some details and what? no error yet?
the display is working? surely some stuff went astray, i pluged in the network cable laughing inside.What the...?. it detected my network and config itself ? I can see shared folder on my rig ! WTF is going on here? Clicked 2 buttons and network was runing!

Wait, this isn't possible,lets try to get on the net.
swaped chair got in frt of XP rig started ICS looked at petty computer animation, yawned twice and then clicked nope I dont want a set up disc.
a small hop to the laptop clicked firefox... speechless, google search page staring at me.

One of the main reason not to use a linux distro beside never being able to set up an internet connection successfully(in australia I had ASDN back then), was the lack of accountancy package with all bells and whistles for my location.
Anyway by now I am thinking "Is there anything this can't do?" couple of google search ( herr.. yes on my new freshly installed Ubuntu} on how to run windows apps under linux quickly brought me to the WINE home page.

Downloaded wine browsed install MYOB CD clicked open with WIne and BANG it worked..!I mean It installed just like if it was XP runing..!

only prob I had was an error message on my Xp rig (after reboot) That it couldn't detect my modem any more. ICS changed the setting from never dial a connection TO always dial my default connection, just checked option back, voila!

No need to say how impressed I am. I know ziltch about linux and everything is working FLAWLESSLY. No doubt that when XP goes dead on my main rig I will install Ubuntu on it and without fear.

To whohever made this ubuntu thing... Thanks Guys,
