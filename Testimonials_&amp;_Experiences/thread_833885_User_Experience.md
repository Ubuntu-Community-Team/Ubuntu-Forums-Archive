### Post by MeanStreak on 2008-06-19
Firstly, a big thank you for the entire Ubuntu development community. I switched from Windows XP to dual-boot XP/Ubuntu with Feisty Fawn and have been using it as my predominant OS ever since. The key advantages are increased security, faster system performance and faster boot time.

I thought it would be useful to provide some user feedback. 

I consider myself of average computer proficiency, I know my way around an OS but only have a rudimentary knowledge of using a command shell. Despite protestations to the contrary, the learning curve for a Windows XP user to Ubuntu is not trivial. I made 8 forum posts about various problems before achieving similar functionality to my Windows box. 

Note that some of the following issues are not specifically 'Ubuntu problems', but are included as they are part of my 'Ubuntu experience'.

My gripes:

* Printing - I cannot print using my Dell 700 Photo printer. I worked through the Red Hat LexMark600 install instructions concerning the issue. The issue is not yet resolved.  

* Amarok - Amarok does not support my iPod Classic 80GB (yes, Apple is the reason I know and yes there are workarounds but none are tried and true and none are trivial). My solution is to revert to RhythmBox. Consequently I've lost all my Amarok playlists. 

* Skype - Skype audio fails from time to time. Skype cannot remember my audio settings (using a USB headset to send/receive) from session to session. (I tried Ekiga softphone as an alternative but the requirement of configuring NAT from Symmetric to Cone without even providing even a link to online instructions about at Step 5 of the 'Install Druid' was woeful, I'm sorry.) 

Skype and Amarok are third-party applications, leaving printing
as the only native-Ubuntu problem, which in the grand scheme of things is trivial. 

Thanks again for an excellent product.
