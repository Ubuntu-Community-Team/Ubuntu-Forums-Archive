### Post by kojakk00 on 2008-05-06
I am a linux beginner. I was very disappointed with Gutsy and even stopped using Ubuntu for a while, well I almost did.  I was still using Gutsy on my PC at work, mainly for C++ development. But on my other PCs I switched back to Windows XP.  Hardy has changed all that. I am very happy again with Ubuntu. Here is a quick summary.

My first move was to install Ubuntu 8.04 from scratch (on a clean hard drive) on my Dell Latitude A510 laptop. I did that last week. After the installation I waited until the updates icon appeared, installed the updates and started playing. No problems whatsoever. I particularly liked the improved user friendliness. Unlike in the previous versions of Ubuntu (or any other Linux distro I have tried) here I could install Adobe Flash and Java in Firefox 3b5 instantly, just by pressing a button, like in Windows. The same goes for VPN. I haven't tried wireless yet.

Encouraged by this, I then put the 64-bit version of 8.04 on the desktop PC I built myself with an Intel Core Duo, matrox millennium G550 and two LCD 1600x1200 monitors. Again I did a full clean install. I was expecting problems because I had previously had a very hard time getting the two monitors to work in Gutsy. But I was very pleasantly surprised with Hardy's performance! I had the two monitors working in minutes, with the right resolution and scan frequency.  And I didn't even have to think _at all_ about that dreadful xorg.conf thingy, which cost me quite a few sleepless nights in the past. Everything could be done from the Ubuntu GUI. I couldn't get the Java plug-in to work in Firefox on the 64-bit version of 8.04 but I didn't have that working in Gutsy either. So I am not complaining.

The next thing I did was to run an upgrade from Gutsy to Hardy (again the 64-bit version) on my Intel Core Quad PC at work with four 500GB hard drives and a fairly newish Matrox Parhelia card. I did it this morning and so far, not a slightest problem! I didn't even notice when that nice portrait of a heron with a red cap appeared on the desktop. 

If your are new to linux, you will probably find Ubuntu 8.04 by far the easiest linux distro to use.
