### Post by tbg58 on 2005-12-01
I've been a Windows server sysadmin for nearly 10 years, having started out in server administration as a Novell CNE back in 1994.  I heard of Linux that far back and even tried to install Slackware a few times and got it up and running, but never fooled around with Linux enough to really get familiar with it.

A couple of years ago I built a Fedora box and started going through some tutorials and other materials, and about the same time at work we started virtualizing Windows boxes with VMWare, which uses RedHat as the host platform.  We had already deployed a lot of Linux servers, administered by a separate group.  Now the team I'm part of had a stake in some boxes that ran Linux, so I had to get to know it.

After having Fedora on an old 200Mhz K5 Acer system the company gave me an old desktop I loaded FC4 on to mess around with at work.  I also got an old demo tower server that had a 1.2Gig Celeron and a SCSI drive that they let me have to take home.   After the long installation of Fedora, which had started to become at least familiar, I began reading in some of the trade rags about Ubuntu, which some folks were raving about, so I visited the web site and downloaded the ISO for Breezy 5.10. 

I installed it on the old Celeron server at home, and I was really impressed.  Fedora's GUI install was nice, but I got inconsistent results.  Ubuntu Breezy on a single CD went through a spare but efficient text-based install that was easy to follow and quick to finish, and when it booted up the attractive aesthetics of Ubuntu's artwork and the general feel of the GUI began to woo me.

The disabled root account puzzled me at first, but I quickly figured out how to sudo, then configured Apache, PHP and MYSQL and began to play with the system.  A lucky find of 2GB of RAM for the old box gave me hardware that was effective (overkill!) to put Ubuntu through its paces, and Ubuntu began to grow on me.

The visual aesthetic of Ubuntu reflects an underlying ethos that has been very winsome and attractive, quite a contrast from the almost peevish "feel" of some of the other distros I tried to get my fingers around earlier.  Ubuntu is a fairly straightforward implementation of GNU/Linux, and shouldn't be that different as a total experiential package than other distros, yet here I was, a skeptical Windows guy getting giddy over it.

I think one of the things that makes it hard for folks to make the jump from Windows to Linux is analagous to graduate study in any discipline.  For the first couple of semesters one goes through classes struggling to get by in unfamiliar territory where even the terminology is unfamiliar, but at some point you've been exposed to enough literature and other material to reach a critical mass intellectually.  Better schools are aware of this and have introductory courses that teach the terminology and grammar that are necessary for interaction with the discipline -- the prolegomena or "first words" of the discipline.

My experience of Linux has been going through literature that is either too dumbed-down or just a bit beyond my reach, but I found some sources (free ones!) that helped to bridge the gap just about the same time I found Ubuntu.  I think the confluence of these two factors have at least contributed to what has been something like a religious conversion.  I was never unfavorably predisposed toward Linux; it was just unfamiliar territory.  And Ubuntu came along and was like meeting a new friend who you feel like you've known for a long time. 

And so I've become not only a convert, but an evangelist.  I'm sharing Ubuntu CDs with associates at work in the desktop support group -- I've set them up shell accounts on my Ubuntu server and put together an internal web site with the best and most attainable tutorials and ebooks I have found during my journey to Linux, so that their journey might be as fruitful as mine has been.

I still make my living as a Windows sysadmin, and will probably always have some Windows around -- it's a different tool for a different purpose and a different target user.  I've known that Linux is a good tool for a long time, but Ubuntu has made switching to Linux a homecoming as well as an adventure.

The journey continues!  Thanks for making it a little more cozy!

--The Bald Guy
