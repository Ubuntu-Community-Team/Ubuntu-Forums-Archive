### Post by freebeer on 2011-07-16
Welcome to Ubuntu and this forum - a truly great resource for discovering the OS.

Yes, the repository of free software is a great thing to discover.  One little note that might help you:  When hunting for an application, it's usually best to hunt in the repositories first.  These are often vetted by Ubuntu to work with Ubuntu (so things like dependencies are already checked out for you).

Not all Linuxes are the same - some are based on Debian (such as Ubuntu) and some are based on other "branches".  Being aware of what "family" of Linux you're working with will help you should you need to stray from the repositories.  (ie you want .deb installers and not .rpm)

That's a pretty general statement but hopefully that little bit of orientation will help you from frustration down the road.
