### Post by papercuts on 2007-04-23
I have been using ubuntu (and any linux in general) for two months now. In the beginning everything was a bit scary because I wasn't used to some core things (i.e. love your xorg.conf!). But now I have a better grasp, I understand directions enough to adopt them to my particular case. 

I criticize many things about linux but on the whole ubuntu is for me THE operating system and I can't see any reason to leave it. I came to use Linux mostly through open source from the Web Design envirnment.

I heard about Open Source before but I wasn't really grasping how that benefits an average person, until I began my own website. All I had to do was pay 20 Euros a year to my provider and everything else that I needed was provided for free by hardworking enthusiast giving their work to the public. I discovered Wordpress this way as a powerful blogging tool. then I learned all the cool plugins you can use with word press. Some others are working to translate Wordpress and all the plugins so I can use it in projects that don't use English (and I make use of that too).  Others still answered my idiotic questions and helped me get going. The result: I have now several web projects, one my own blog, another a 200 member restricted alumni site, and another a serious history website for the community.  All this costs me yearly 30 Euros. And my service provider uses Linux servers, which helps to cut costs for me there too.

So what about Ubuntu? I came to realize how cool it was to use software that answered all your needs (because as long as there is need, some programmer is working on an app for it). I also liked adding stuff easily with plugins that fit your apps nicely. I thought Hell why not do that with an operating system? 

This brought me to Ubuntu and Linux where I have a stable and good looking desktop and I can go to synaptic and just pick and install any program on the list without thinking about the costs or legal issues. I hope as I learn more I can come back and answer questions on the forums because as a non-programmer and poor student that is all I can do. :)
