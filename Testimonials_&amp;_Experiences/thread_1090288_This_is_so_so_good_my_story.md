### Post by argonaut747 on 2009-03-08
Newbie - I think Ubuntu is excellent!!
I've had a copy of Mandrake 6.5 on me shelf for years. When I first installed Linux all those years ago I found it far to daunting and re-installed Windows 95, (that's how long ago it was). I've dabbles with a couple of versions which were CD bootable but didn't do much with that either.
A couple of weeks ago I thought 'Do I really want to do my on-line banking within an OS that is so open to hacking?' the answer was NO so I downloaded Ubuntu. Yesterday I installed it onto a spare PC I have (AND 2600+, 1.5 gig RAM, 120Gb HD).
The install was a dream - it took about 10mins (how fast is that!!). I then tackled the internet connection issue which I had always failed at in the past (the real reason I abandoned and went back to Windows).
I have a Dell XPS 720 connected to which is a Sagen router supplied by SKY. I hunted out an RJ45 cable and plugged it all in. Ubuntu detected the MAC address but still the internet wouldn't connect and I couldn't see my DELL PC.
After a little head scratching I thought I'd failed again (I'm not very patient). Then in dawned on me - when I last used the AMD machine I turned off the LAN in the BIOS. I went in and set it to 'Active'. Boot up Umbutu and EVERYTHING worked automatically!!
So it was totally me being a thicky. Umbutu is EXCELLENT. It looks nice, it's fast, and it's doing exactly what I wanted it to do.
I am so impressed with this.
Will I move from Windows? No. For all the bad press it is still very good any easy to use. Will I continue to use Linux? Yes. It's too good to ignore.
I like the fact that I have a choice.
The only downside is I now have 2 of everything to run bot systems. Duel boot? Not a chance. Like I said I'm too impatient.
I feel like Captain Kurk surrounded by all this technology - loving it!!

### Post by presence1960 on 2009-03-08
Dual boot isn't as hard as you think to set up. You can just set up the partitions using gparted. See this: [http://www.psychocats.net/ubuntu/partitioning](http://www.psychocats.net/ubuntu/partitioning)

Once partitions are set up you can proceed with the install. When the Live CD starts the partitioner choose "manual" and it will allow you to choose which partitions to install Linux to.
