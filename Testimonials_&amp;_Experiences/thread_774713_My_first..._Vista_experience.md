### Post by Jon Monreal on 2008-04-29
Vista does have its annoyances and its quirks. But it seems to be pretty stable as XP was, and it seems to have less bugs than when XP originally came out (at the time I dual-booted it with Gentoo).

What I really wish Microsoft had done was make a release that was a lot like the established XP, with everything that the users liked, and just added to it more features. I would have been happy to install this alongside Ubuntu for some purposes. The sad thing is it seems that Microsoft may throw out Vista and concentrate on Windows 7.

There are some things about Vista that really upset me however such as exclusive DirectX 10 and Games For Windows games that only work with it. The fact that Halo 2 was so easily cracked to work on Windows XP proves that the reason that they made it only for Vista was not for their claimed better features but instead to get people to buy Vista if they won't buy an Xbox.

### Post by jbysmith on 2008-04-29
> **Jon Monreal said:**
> The fact that Halo 2 was so easily cracked to work on Windows XP proves that the reason that they made it only for Vista was not for their claimed better features but instead to get people to buy Vista if they won't buy an Xbox.

You called that one right; there's no reason DirectX 10 couldn't have been made for XP.

I tried Vista Ultimate for a little while.  

Granted, there are a few things that I do like about it.  The new file Explorer is actually rather nice and well thought-out, and has a lot of little things that I wish other file managers could do.  The eye-candy isn't bad, it's no Compiz but it's a nice start I suppose. The file copy/move system is a lot better at handling problems that come up during the copy.

That's pretty much where the love ends from me.

My major gripe is the performance.  Before any Vista fanboys jump out and cry "upgrade!" my system is reasonably current; well within the recommendations. Not state of the art, but I've played Crysis on medium-high details with a comfortable framerate. It's not horrible by any means, but noticeably slower than XP was.  Plus the memory usage is just a joke.  After optimizing services and whatnot (I killed off a lot), sitting idle on the desktop, plus security apps, plus Aero/DWM, Windows Search (that one I like), and a few other resident utilities, it's pushing 600MB easily. That's after the initial startup.. stuff.. that it does where it pushes a gigabyte for a few minutes.  I can get near identical out of XP, minus Aero of course but some "workalikes". I'm using around half that.  And it's a lot faster too. And *everything* works.

The new sound system is just awful, not just including the whole Creative Labs driver fiasco.  (Anyone with an Audigy and Vista knows what I mean)

UAC is a poor knockoff at best of Sudo, and easily beaten.  This is assuming the user doesnt disable it out of annoyance first.

The list goes on and on.  SP1 helped somewhat, but a few minor improvements and eye candy aside, it's a poor successor to XP.  Nowhere near the disaster than WinMe was though, at least it *is* reasonably stable.  Don't think it's crashed on me minus the earlier versions of drivers, assuming a driver is available in the first place.  

I'm pretty much done with Windows myself, but I really hope that MS learned from this train wreck and make Windows 7 what Vista should have been in the first place.
