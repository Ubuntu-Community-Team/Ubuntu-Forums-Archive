### Post by jaithehulk on 2012-09-08
Hi,

I have a Asus K53S series laptop with 4gb RAM,nvidia 610m,bluetooth and wireless. 
I have installed 12.04 a few weeks ago.
Everything is working fine

1.Battery life awesome.
2.Tata Photon+ data card is detected(very rare) :guitar:.
3.All Fn+function keys are working.:D
4.Manually enabled hibernate is working ;)

 There are a few glitches:-
1.Bluetooth(detects devices but file transfer failure).
2.Samsung s3 still not detected.

All in all experience is very good.
Hope this helps everyone with the same laptop series.

### Post by 3rdalbum on 2012-09-11
> **jaithehulk said:**
> 
 There are a few glitches:-
1.Bluetooth(detects devices but file transfer failure).

I assume you're doing file transfer to the Galaxy S3. If not, and you're trying to transfer files from an iPhone, then there's your problem - you can only Bluetooth from iPhone to iPhone, not to or from any other kind of phone.

Bluetooth file transfers kinda suck - they're slow! A better option would be to set up a Samba share on the computer and use File Expert and wifi on the phone to browse/ add files to the share. Wifi is a lot faster if you can't use USB, although I bet you probably just need a program on Ubuntu to access the contents of your phone.
