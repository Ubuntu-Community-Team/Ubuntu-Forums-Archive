Either way, I have nothing but positive things to say about Ubuntu, Linux, and the extremely helpful Ubuntu community.  I think it's funny that, often when I'm searching for a solution for a Linux-related problem, the answer ends up coming right from these forums (from a Google search result).  I guess that goes to show how helpful everyone here actually is.

This is now the fourth time I've installed Linux, and I'm actually keeping it this time.  I plan on moving all my music and documents to my linux partition later this week, so I'll only have a Windows partition for compatibility purposes.

Ubuntu really is surprisingly easy to use.  The only part that took time is the initial setup (especially with my widescreen monitor, although with the release of 915resolution, that wasn't a big deal at all).  Ubuntu detected everything else, and although the file structure took some time to get used to, I see now that it's much better than Windows ever was.

Thanks everyone (especially the Ubuntu developers and support guys on the forums).  I love you all, in a completely heterosexual way.
