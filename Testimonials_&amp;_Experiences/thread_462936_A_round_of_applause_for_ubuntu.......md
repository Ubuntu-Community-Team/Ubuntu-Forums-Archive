### Post by techno-mole on 2007-06-03
i just thought i would post a thread to show my appreciation for ubuntu.
i first tried ubuntu when edgy was the latest version (not that long ago) but due to my old hard drive giving up i had to wait for a new one, which i used to install fiesty on.

i must say that fiesty is alot easier to get on with (was for me) things that were a little bit of a problem in edgy are easy as with fiesty, for example i could never get the networking to work right (i have to network with 2 other pc's running xp) but in fiesty it just worked and was quicker to set up than it is with windows.
ive had a few issues (nothing major) i cant install wine, dont know why but i havent found a guide that will get it working on my system, but i only wanted it for games anyway, so im not bothered, especially when ive got doom 3 and never winter nights installed, so i can play through those.
ive had a bit of a learning experience with using different programs for various things, and ive found that generally the programs ive used do the job better than the windows versions do.
so all in all i think ubuntu is a very good and for me its very stable operating system and it is a viable alternative to windows.
the games issue is more to do with companies that seem to only be interested in taking peoples money and not with making their games available to users of other than windows operating systems.
cheers from one happy ubuntu user.

### Post by techno-mole on 2007-06-04
i did try some other distro's when i first got into linux and after a while i found ubuntu the easiest to get on with.
now i can get it all installed and set up how i want in about 3 hours, it used to take that long just to download all the updates for xp, on average it takes about 3/4 hours less to set up and install ubuntu than it did with xp.
there are improvements to be made though, although its not really down to ubuntu its down to companies who make stuff like blue tooth adapters and other bluetooth devices, and companies that produce other types of hardware.
its funny because the only real problems i have with ubuntu are not really anything to do with ubuntu or linux, its the companies that make things and dont consider the needs of us linux users, and games companies that dont consider us linux users.
i think that given time and alot of moaning on the part of linux users then maybe these companies will start to listen and maybe produce more in the way of linux drivers and games, and that way we wont have to worry about installing things like wine and cedega.
