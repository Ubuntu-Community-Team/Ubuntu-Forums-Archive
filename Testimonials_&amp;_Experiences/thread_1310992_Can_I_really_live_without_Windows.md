### Post by YosefKaro on 2009-11-02
For several reasons, I have decided to try to make a complete break from Windows and to try to go it with ubuntu alone...but I have my concerns.  For one is my internet connection.  I use a cellular USB modem and ubuntu doesn't provide any support for them.  The manufactures themselves provide very little support and information for linux users.  In order for me to get on the internet, I have to first of all download 5 deb files.  In order for me to download them, I have to have internet which forces me to have to keep rebooting back into windows.  Last night, I decided to make the complete break away from windows, going all the way to completely installing Karmic on my entire hard drive.  Then to my consternation, when I went to install the deb's to get my internet set up, I discovered that I was missing one of the dependencies.  That meant having to go back and re-install windows again just so I could download that missing dependency.  I like ubuntu, I think it is great but I feel like a David confronting a Goliath.  With windows, everything is so easy because everyone is making it easy for everyone else.  New technologies, applications and hardware make everything plug-and-play for windows while linux users have to bend over backwards to find work-arounds to make them work.  So I wonder if I can make this clean break from windows...I'm giving it a try again having re-installed Karmic on my entire hard drive.  However, it seems to be that something needs to change in order for ubuntu to be more appealing to the masses: not everyone appreciates the 'challenging' aspects of ubuntu.

-Yos

### Post by steveneddy on 2009-11-02
If your only/main connection is a cellular connection (I assume you are in the US) then go somewhere where there is free internet and DL files you need there.

Almost everyone dual boots when they first start using Ubuntu. It's OK.

Personally, I use a Virtual Machine (VirtualBox) in Ubuntu to run Windows Vista and 7. I can actually have a device that is tethered to the laptop, run Vista in a VM, have an internet connection in Vista and not in Ubuntu. So maybe getting the hardware working in a VM running Windows could make the transition better for you.

But since you are beginning, maybe it is best if you simply dual boot for a while.

But do whatever keeps you productive for your needs.

FYI - I use Ubuntu 100% of the time. I rarely fire up Windows in a VM. I just accepted the fact that Ubuntu is my OS and I just figure out how to get things done with Ubuntu. Sometimes you just have to jump in with both feet and see if you can swim.
