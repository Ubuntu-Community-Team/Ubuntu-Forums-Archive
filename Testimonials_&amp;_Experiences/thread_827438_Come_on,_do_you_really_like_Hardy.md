### Post by beniwtv on 2008-06-27
Another redundant "Yes" here.

I absolutely love Hardy. It's the *first* painless Linux install I had. No kidding. My laptop is a HP Pavilion dv6106 eu, for the curious, AMD64 Turion+ Nvidia GFGo 6150.

With previous versions, I could not boot the live CD. Not even in save graphics mode. I searched and searched but nobody seemed to have the same problem as me. (It would not boot X: both nv and vesa would hard lock my laptop). On Hardy, nv works as expected, yay! And jockey installed the nvidia driver just fine!

Then there was a problem with 64 bit not booting on this laptop (related to HAL). Fixed with Hardy now. Also, for some reason, the kernel would hard lock when in console mode. Works in Hardy.

Then, my WiFi. On the last versions, I had to use ndiswrapper to get my broadcom to work (I hate them!). The native bcm43xx driver worked, but was not reliable with this card (this was even mentioned on the driver's home page). With Hardy, the new b43 came. The card works now! Yay!

So, every feature of this laptop, yes every feature, works now like it would in Windows. And it just takes me 5 minutes to set-up, and no command line!

I'm also very happy with all the default software. Since Hardy, I do not need to install alternatives to built-in Ubuntu software. Although, install software I need not coming with Ubuntu. 

I admit that it took me time to adjust to the default software, since I needed changing my habits a lot. But at the end, it all pays out, since it just works beautifully! I think often software (flame-)wars (or Win/Linux) come down to personal habits / preference, but not to one software being superior to another.

Cheers,
