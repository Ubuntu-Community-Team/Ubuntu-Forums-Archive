### Post by Dawnshadow on 2006-08-18
I set my computer to dual-boot Ubuntu Linux and Windows XP about 3 weeks ago. It's a Compaq, about three years old, and was a pretty nice system when I got it-- processor's a little over 2 GHz, upgraded the RAM to 1 GB, decently big 160GB hard drive, and I added a low-end Nivdia graphics card that was on clearance for $50 a year ago. 

So far, I've been in Ubuntu almost the whole time, only going to Windows to play some games that WINE doesn't do, add new albums and/or MP3s in iTunes (I've grown to like its method of sorting music) or to use Photoshop Elements, which I've been using for a while and prefer over the GIMP. Ubuntu is a little faster than Windows, even with my new RAM, it seems to be more reliable, the UI (I think) is better now that I have everything set up how I want it, and I like not having to worry about spyware scans or viruses.

The only issues I have are with wireless and that I still haven't got my Wacom tablet working quite right. (I know there's a tutorial out there on how to fiddle with settings to make it work, but I'm not quite comfortable with that much fiddling at this point and I don't care much for GIMP, anyway. Although I have been using it as my main pointing device in Windows since I got it, I can live with a mouse.) In fact, if it weren't for the wireless issues-- I can't figure out "ndiswrapper," either, although I still haven't done much tutorial hunting yet-- it would be on my (older) laptop, too; it would probably run a lot faster.

(Edit: I forgot half the post. >_< So I'm adding it now.)

The one thing that I love about Ubuntu over all else-- and any other distribution that uses it or something like it-- is the Add/Remove interface and Synaptic. Hundreds of programs, installable in a single click, no long and lousy interfaces, no cost, no hoops to jump through. Installing the core programs I use in Windows (Firefox, AVG, Spybot, Ad-Aware, Trillian, iTunes, Photoshop Elements, Open Office) after using Synaptic was a pain in the... ear, not to mention the four programs I (and most college students) use most (Firefox, Open Office, music player, omni-IM client) come pre-installed in Ubuntu.

I'm about this close to copying a bunch of Live CDs and sneaking them into the housing office for move-in this year, and I'll never have to deal with a virus while trying to get final papers done again. Thanks. :)


and bam, there are your programs.

i like to do this whenever i reinstall; all i need to do is unzip all the settings folders into .home and my desktop and installation is exactly the same as before :)
