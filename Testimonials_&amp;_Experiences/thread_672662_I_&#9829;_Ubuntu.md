### Post by FM1 on 2008-01-19
When I built this desktop, I knew it would run Ubuntu, so I bought my hardware with an eye toward "known Linux-friendly". My old hardware will be retired soon (tax return time is coming), but today, just for fun and because it was too cold to go outside, I decided to see what would happen if I connected two monitors to my GeForce 6200. I still had the little adapter-thingie (like my technical term?) that had come with the card, so I connected my old 17" LCD to that and plugged it in beside the 19" widescreen. I booted up, actually expecting X to have a little fit, but it didn't. The second monitor didn't display my desktop, but fixing that was as simple as running nvidia-settings and telling it to use TwinView. I thought the different resolutions (1440x900 and 1280x1024) might be problematic with a spanned desktop...nope! Though I don't really *need* dual displays, I'm keeping both of them because I like the extra space. Besides, my (Windows-only) husband came in here and looked at me like I was some kind of über geek-genius. I'm not even *close*, but....shhhh! ;)

My whole experience with Ubuntu has been like that. I first decided to try Ubuntu when I read about an OS called "Warty Warthog"; partly because it looked different with the brown color scheme, but also because it was a way a noob could "use" the venerated Debian without knowing how to...actually install and configure Debian. At the time, I was still very new to Linux, dual-booting XP and Mandrake, and I added Warty as a third option. Other than the fact that I had trouble getting the sound working, Warty was pretty forgiving even though I didn't really know what I was doing. I used Warty, Hoary, Breezy, Dapper, Edgy, Feisty and now Gutsy, and it just keeps getting easier. I do check hardware for Linux-friendliness before I buy it, and it's great to bring something home, connect it and have it "just work". When my husband borrowed my digital camera, I had to find the driver disc for him. When I bought it, I just plugged it in and it worked. I was the worst kind of switcher--a computer control freak and a Windows power user--but I wouldn't go back to Windows now for anything. My Gnome desktop looks and behaves exactly the way I want it, and I love having all of my software update automatically without "update available" windows popping up all over the screen. I get a new OS every six months if I want, so I don't get bored. I don't need antivirus, or a firewall, and I don't have obnoxious software that insists it should run on startup, or run something in the system tray, unnecessarily using *my* resources. Windows is okay and I really have nothing against it, I just don't like it nearly as much as Linux, and especially Ubuntu.

I've tried several different distros and my laptop does run openSUSE, just for something different, but my first choice for a day-to-day OS is always Ubuntu. The OS is excellent, and if I do run into something I don't know how to do, or break something in my efforts to make it do what I want (I'm practically a professional at breaking X...hehe), a search of these forums usually gets me the answer. If not, I can post and ask, and as long as I provide the necessary information and ask nicely, I'll get the help I need because the community here is awesome. Sorry for the long ramble, BTW, and if you read all the way down here, thanks. If you helped someone with Ubuntu, or Linux of any flavor, or if you're a dev, then *double* thanks!

P.S. I did once install Debian Sarge, just to prove to myself that I could, but went right back to Ubuntu because I like it better. :D
