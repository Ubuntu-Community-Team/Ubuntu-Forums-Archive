### Post by Old_Grey_Wolf on 2008-06-21
I have been using Linux distros like Fedora and Ubuntu for a few years. The switch to Linux was a gradual process where I used Microsoft Windows less and less. I bought a new laptop a couple of weeks ago. It came with Vista. I booted it up and set up Vista. The Aero interface was not enabled by default so I enabled it. Then I decided I didn't like the anti-virus program than came with Vista so I replaced it with one that has a Linux version. I soon discovered that the firewall only protected against inbound traffic and did nothing for outbound traffic. I learned that the Vista firewall could be configured for outbound traffic, but that configuration was not done from the normal firewall configuration window. It had to be configured another way, and it was not user friendly to say the least. So, guess what, I replaced it with free firewall software. I needed word processing and spreadsheets, so on goes OpenOffice, another Linux compatible program. And I need a graphic editor, so here comes GIMP, yes, another Linux compatible software. I didn't want IE7, so on goes Firefox. Etc., etc., etc. By the time I was done there wasn't much of Microsoft left. If I could get rid of WGA and DRM <sarcasm> I might actually use it </sarcasm>. 

I still think that the OS you use should be the one that is right for you.

BUT I'M FREE TO CHOOSE !!!
