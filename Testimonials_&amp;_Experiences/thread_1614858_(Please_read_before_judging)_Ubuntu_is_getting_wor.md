### Post by hittingray on 2010-11-06
I've been an Ubuntu user all the way back since about 5.04. I loved all the versions upto 9.04, 9.10 was good as well but the CD drive bug really annoyed me. 

Since 10.04, Canonical is disappointing me more and more through each and every version. It's becoming more and more like a mainstream OS (design wise), and a lot more like OS X. First, in 10.04, the close/maximize etc. buttons got moved to the /left/, like which OS? Hmm I wonder, and then 10.10 Unity was introduced, even more like OS X, with it's dock style switcher, like OS X and where File, Edit, View etc. toolbar sits, up the top, in the top panel, even more like OS X. And what next? 11.04 sets Unity as the default desktop. I loved Ubuntu the way it was, not the way it is. I really hope they take this into consideration.

Not only myself, but many of my friends have tweaked their installations to have it the way it was. I hated the button address bar, I hate Unity, I hate the window options buttons on the left side, and I hate the way Ubuntu is turning into.

TLDR: Ubuntu is getting worse and worse through it's design in each and every version.
