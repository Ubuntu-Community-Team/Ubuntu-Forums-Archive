### Post by superyounan1 on 2007-06-09
I recently got a nice new 2gb jump drive and I thought I'd hunt around for some portable apps... I usually downloaded portable VLC onto school computer whenever I wanted to watch something.

Turns out the portable apps class of software has become very rich and mature, I found a great site called [http://portableapps.com/](http://portableapps.com/) that offered a free bundle of free software such as OpenOffice, VLC, Firefox, Thunderbid, Filezilla, ETC., all the great open source programs we all know and love, AND it came with a cool launcher that sits in yoru task bar.

Here's what I'm getting at.. I thought it would be great if all these apps were compiled in some sort of universal binary so that they'd run on all major platforms, Windows, Nix, and OSX, then I remembered that my Ubuntu comes with wine...

I tried the jump drive out, the laucher app works! the programs work! (for the most part). I even watched some star trek on the portable WINDOWS version of VLC using wine, needless to say I was impressed. Sure the colors were a bit off, but if you think about it, thats very impressive.

Portable firefox worked, so did Pidgin.. openoffice was a bit buggy, but I have to say, overall it was great.

Thanks ubuntu, thanks wine, for impressing me again.

Although I'd still love to see 3-in-one universal binary portable apps, that would be even better.
