Windows Vista was being slow (REALLY?!) and I got fed up with it. I had installed Ubuntu before in a WUBI installation but it was impacting it's performance because it's on the same NTFS partition, it's all fragmented, and all the other things. I decided that I should install Ubuntu again, because when I had before, it was a very enjoyable experience. It froze at the install sequence, so I just hard rebooted my computer. Everything was fine in Windows. So a few days later I decided to try it again, I waited a while, thing was that I hadn't given it enough time to thing about the partition size I wanted (Stupid, I know) and everything worked fine! So now I have ubuntu installed, alongside with Vista. So far it's great, and the IRC channels have been alot of help..

Thanks devs, for making a great OS.
