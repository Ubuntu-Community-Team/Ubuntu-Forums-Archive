### Post by theasprint on 2010-12-24
Hi,

Well I only know that the netbook edition is actually purposely equipped  with all the Buttons etc on the left. This is so as to give more ease  as navigating in the normal Ubuntu menus in the Desktop edition may  appear very small and cramped in the netbook screen.
*But you actually sound that you've used the netbook edition before, so I  don't really know what you're frustrating about. It seems almost the  same to me in the netbook editions.

About the system hangs too much thing, 

About the system hangs thing, maybe you should check what processes are running that takes up lots of your RAM?

Run this in terminal to see a list of processes. (Even hidden ones)
```
top
```

Edit: And also run this for the memory:
```
free -m
```
