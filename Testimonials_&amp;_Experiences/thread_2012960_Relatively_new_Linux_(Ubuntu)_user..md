### Post by stevecook on 2012-06-30
Hello folks. Steve Cook here, a relatively new Linux (Ubuntu) user.

I have been using Ubuntu now for about  three months. The first two and a half months of which was with Ubuntu 10.4 which, although, it was a bit buggy and froze on me periodically, I really enjoyed using. I initially went with 10.4 because I was running a crappy Celeron and a friend told me that 10.4 would run on a Celeron. I have since bought a second hand Pentium D and so I switched over to 12.4  two weeks ago and I can say that I personally find the Unity shell to be very unhelpful. This is not because it's visually different because, I am bound to say, I think that 12.4 Unity looks absolutely fantastic. It is also not because of a lack of "out-of-the-box" functionality. In terms of that, it's a vast improvement on 10.4 as it has picked up virtually all my peripherals on install and saved me a lot of work setting stuff up.  For me, the main problem with Unity is the appalling dashboard navigation system. The amount of mouse clicks required to just get get a simple job done is really silly.  

Thankfully, though, I quickly discovered the "Gnome Classic" option on startup and have reverted to that. Apart from one or two minor irritations (for example you have to write a script  to reinstate the capacity to right-click-create launchers on the Gnome-Classic desktop), the Gnome Classic interface works great. I should also say that 12.4 appears to be ultra stable on my machine. No freezes of any kind.

So now I have the best of all worlds. I have the much improved underlying functionality of 12.4 where everything just gets picked up a lot smoother plus I have the Classic-Gnome interface with works so efficiently for me.

And best of all....

**IT'S FREE!**

I love this OS and will NEVER go back to MS
