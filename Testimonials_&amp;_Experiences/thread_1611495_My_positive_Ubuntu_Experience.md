### Post by marl30 on 2010-11-01
I first came across Ubuntu after finding out about Virtualbox. I had Windows 7 installed and I found that I could use Virtualbox to install other OS without dual-booting. So, I went to YouTube and checked up on how to run different OSes such as OSX in Virtualbox. That is how I actually came to find out about Ubuntu. Someone at YouTube had this really cool OS that he said was Linux. He had customized it and had Virtualbox running with Windows 7 and Mac OSx both as guess OSes. I had no clue that a Linux system could look so good, let alone run application in the same manner as Windows. Well, that was largely due to the perception I had of Linux. I use to think Linux was nothing but command lines, and was an OS that only Geeks and System Administrators could use.  
 
 When I saw how user-friendly it was, I immediately did a search and found the Ubuntu site. I downloaded 9.10 and installed it in Virtualbox. I was definitely loving it, but thought I might dual boot it at some point in time. Weeks went by, and I still didn't dual boot it, though I had seen different online tutorials on how to create a partition without formatting my Windows installation. I guess I was wondering if this Linux/Ubuntu thing was just a fad that I might soon get over. Besides that, I didn't exactly warm to the looks of 9.11. I didn’t think it was ugly though... just looked weird to me. However, one weekend I logged into Ubuntu and it told me that an OS upgrade was available. Immediately I did the upgrade. After my computer restarted, I was in for a huge surprise. The upgraded version of Ubuntu had looked really pleasing on the eyes... I was completely sold at that point. I immediately Downloaded 10.04, partition my hard drive and did a dual boot.

After installing Ubuntu 10.04… everything worked out of the box. I had no issues whatsoever. The next couple of days I found myself spending all of my time using Ubuntu rather than Windows 7. I loved the fact that I didn’t have to wait for long boot ups or shutdowns. Ubuntu ran superfast all the time, and even my experience when browsing the internet seemed a lot smoother than with Windows. The other thing… I was one of the lucky ones whose favorite games all worked 100% perfect in Wine, so this gave me fewer reasons to log into Windows. I found myself logging into Windows every few days, which was just to update the antivirus and other updates. Using Ubuntu then going back to Windows felt weird, as I always thought of Windows 7 as being reasonably fast, but after using Ubuntu and then comparing the speeds, it was like a race between a race horse and a camel… Ubuntu being the race horse.

Two weeks after dual booting Ubuntu with Windows 7, I thought, what’s the use of keeping this large Windows partition when I prefer using Ubuntu over Windows?  So I deleted my Windows partition and plunged into the open source world of Ubuntu and Linux. I found out that Linux can do everything that Windows can do, in some cases, even better. However, I kept Xp installed in Virtualbox because of certain MS applications that I need for school, and so I can also update my phone with Windows until Wine begins supporting the software for my phone. It's now over six months later. I’ve since upgraded from 10.04 to 10.10. I'm loving it even more than I did 10.04. I can proudly say... I have no need to look back! I hope to be using Ubuntu, if all goes well, for years to come, even though I quite frequently toy around with other distros. So far, none of them have compelled me to give up Ubuntu.
