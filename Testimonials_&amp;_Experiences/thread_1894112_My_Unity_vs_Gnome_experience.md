### Post by 73ckn797 on 2011-12-11
This is not a rant against Unity but a comment on what I have found to be a solution that I like. It will likely be moved to Recurring Discussions. I tried to start this there but do not have permission.

I personally do not hate or despise Unity. I just do not want to use it or have it available as an option. Wanting to eliminate Unity I found this link: [http://linux-software-news-tutorials.blogspot.com/2011/10/ubuntu-1110-oneiric-remove-unity-and.html](http://linux-software-news-tutorials.blogspot.com/2011/10/ubuntu-1110-oneiric-remove-unity-and.html) . Following the applicable recommended commands (already had enabled gnome-session-fallback) I have removed the Unity DE and am now running Gnome shell only in Gnome Classic mode. I was already using Gnome Classic so these things only eliminated Unity. I did find 2 files related to Unity still showing to be installed in Synaptic but easily removed those also. Currently I am running 11.10 in a dual boot with 10.04. I am using 11.10 as a platform to test and be prepared for the next LTS 12.04.

SO. If you desire to completely remove Unity, follow the link.
