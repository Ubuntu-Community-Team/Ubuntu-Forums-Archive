### Post by dachapa on 2008-11-01
This is whilst experimenting with Ubuntu Server. I know comparing a Server build to a desktop build is rather unfair and uncanny, however, that is what I have been exposed to.

While taking part in the Management Information Systems class, I have more and more often come across things such as Open Source software, and its advantages. However, unlike many who experience an Open-Source Operative System, I stick with Windows. Now, I can not say I have tried running an Open OS, however, there are things that keep me from switching, or at least trying it. I cannot say for example, that throughout my life I have been fully satisfied with the Windows OS, but these are some of the reasons I stick with it.

1.Stick with what you know. I have been running Windows since Windows 95, and while I did not use every version of Windows available, I can believe the familiarity with them make them so easy to use. From Windows 95, I then moved to Windows 98, what I believe has been the best version of Windows. I stuck with 98 until XP came out, and I bought my first XP computer. At this time, I am using my XP Desktop, and also my Windows Vista laptop. Now Vista is a different story, but I can honestly say that from 95-XP, the interphase was so similar that every time I upgraded I did not have to learn where things were.

2.Windows XP does not crash as often as people want you to think. Yes, they still crash, but the amounts of time they do is not so unbearable as some Linux-aficionados will have you believe. In reality, all you have to do is make sure you do not try to open every single application possible at the same time. While I have no experience with Linux, I would be willing to say that a four year old computer would crash if 20 applications tried to start at the same time.

3.Software compatibility issues. I find this to be a huge disadvantage of Open-Source OS. Yes, they do have software that lets you install and operate Windows-based programs, but how well do they run? From different testimonies I have read from different people, the programs tend to run slower, and based only on simulation. If you are a gamer, I want to see you try your favorite game on a Linux computer, it will probably not run was well as a Windows computer with at least minimum requirements.

4.No real customer support. Okay, there is the On-line Open Source community and forums available for help, but fixing problems can be tedious, and require knowledge of specific software terms. However, Microsoft offers both an on-line, and a phone customer service. While it may not be so high quality, it does help specially if you rather have someone guide you with their voice than to have to read yourself. 

5.Windows is just easier to install and get it ready. It may take more time than installing a Ubuntu server, but I think Windows's visual interphase is simpler to follow than a “input and get lucky” command-line. For the average person, it is much easier to see things, than to have to research and ask for detailed information on how to install their OS.

I am sure there are more disadvantages for using a Open-Source OS, however, these are the most important to me. While yes, Linux for example may be better, I still think Windows is not so bad as people make it seem. Okay, updating every day gets tedious, but at least I know they are improving everyday. Just try to figure on your own which is best for you, and don't let others make you use something you are not comfortable with.

### Post by Rhubarb on 2008-11-01
You'd be right in saying that you're a bit biased there.

Ubuntu server is not windows, and it's certainly not for desktop use.
They just can't be compared because they're incredibly different in appearance, for use, and for the target audience.

It's true, many people stick with what they know, but as I'm sure you've noticed, finding your way around vista can be quite daunting the first time you use vista (from xp).

If you tried to start up 20 applications in Ubuntu at the same time (even on a 4 year old computer), sure it'd take a bit of hard drive thrashing, but all 20 applications would actually start up.

Software compatibility issues with Ubuntu can be a big deal with many new-comers to Linux.  While it is possible to use wine to run some windows applications (and games), usually there's no point.
We have many many many many alternatives to most applications in windows.
So most of the time we don't actually have a need to run windows apps.
(I run Portal and steam in wine, runs fine for me - some applications run a lot faster in wine than they do in their native windows setting).

There is certainly real customer support.  Canonical has support, and so do many other independent companies around the world.
And as you say, yes, there is always the forums here, and your favorite search engine.

Have you installed windows from a retail disc recently?
Have you any idea how bad the install screen for xp is?
Then there's drivers to install in windows, then there's an absolutely useless desktop you've got after windows is installed.  - There's no useful apps installed that are of any use.
Also, when installing the Ubuntu **Desktop** you'll have an actual Ubuntu desktop loaded up, no command line to be seen, you can use firefox to surf / play some games / do what ever you like while installing from a nice gui.

Indeed, it's best to use what you're comfortable with.
But, by all means go out and try what you're uncomfortable with, after all, it's free.

When friends of mine have difficulties with using their windows computers, I install Ubuntu on for them.
It simply is amazing how little problems I get from people that have switched from windows to Ubuntu.

For those people that consider themselves "windows power users", I find have the most trouble grasping Ubuntu.

Please have some fun, and download an Ubuntu Desktop iso.  You can try it out without having to install it from the CD.

I think you may be surprised about how easy it all is.

Here's a good article you may want to read up on:
[http://linux.oneandoneis2.org/LNW.htm](http://linux.oneandoneis2.org/LNW.htm)

And please give Ubuntu Desktop a go, it's free and it's fun :D
