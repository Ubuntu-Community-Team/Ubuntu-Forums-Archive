### Post by cifdtruckie on 2008-04-07
I recently built myself a new computer, and as I had been playing around with Ubuntu on my old system, decided to install it as the primary OS.  On first startup after the install, Ubuntu recognized all of my devices (the ONLY thing I had to do was run Envy for my nvidia vid card).  I was up and running in minutes, and any other progs I needed required using the ridiculously simple apt-get or package manager! :)

A few days ago, I decided that I missed gaming, so decided to install XP also, solely for playing games.  The install took about 5 times as long and automatically over-wrote the linux boot loader (would it really be that hard for M$ to program it to ask before doing that???).  It didn't recognize my network card, vid card, or sound card.  Thank heavens I could actually find my network card driver disk so I could get online and download the other drivers!  Of course, feeling ridiculously insecure while surfing, I also had to dl and install AVG and Spybot S&D, which meant going to their websites, manually downloading, then installing, registering, updating, etc.  Having only gone to these sites, when I ran Spybot right away, I was quite surprised to find that I already had 4 separate instances of ad-ware in my system!!  Then, of course there is that pesky auto-updater, which instead of the polite little orange square that lights up in Ubuntu, has that shield icon that is constantly begging me to restart my computer!! :mad:

Besides playing games or using other commercial apps that don't quite have a perfect counterpart for Linux, I can't imagine what makes Windows a "better" OS for anyone!!
