### Post by Jadd on 2007-11-07
So, this year, I've been keeping all the things I've noticed about Ubuntu, good or bad, in a tomboy note. The remarks concern Feisty and Dapper.  I'm wondering what I should do with them, and decided on posting them here:

My Ubuntu experience
Here are my remarks and notes about my Ubuntu experience:

    * Installation bug (the installation window would only display the title bar, couldn't see the progress bar or the status messages. Concerns Feisty Live CD)
    * Autorun CD has a spelling mistake (on windows)
    * Universe and multiverse are enabled by default (multiverse should display a warning when enabled, just like the codecs)
    * Ubuntu is not in the dictionary (even in Gutsy! I just checked! Now when I type Ubuntu in a tomboy note, it underlines it in red)
    * No boot-admin (why not? a GUI for editing /boot/grub/menu.lst is needed)
    * Updates with nothing selected is a 45MB download (I think it downloads linux kernel if I click Install Updates)
    * Totem plug-in doesn't display any errors (when codec is not installed for example, whereas totem in a seperate window does)
    * Rythmbox doesn't display any errors
    * Network settings required reboot
    * Network icon should show network activity
    * Partitions shouldn't appear on desktop (I've got 12, so this really annoys me, and there's no way to enable or disable them!)
    * Cannot browse windows network (Gutsy seems to have fixed this problem)
    * updating xscreensaver changed my theme (on Feisty)
    * Totem plugin doesn't work with Opera
    * Amarok wiki-lyrics don't work (Gutsy seems to have fixed this problem)
    * When installing abuse from add/remove, the sound package should be installed as well
    * when installing adonthell from add/remove, the adonthell-data package should be installed as well
    * if I click on a menu item in a certain workspace, that application should open in that workspace, even if I changed workspaces while it was loading (the way workspaces are managed deserves an entire post!)
    * onBoard doesn't do anything
    * Keyring Manager's help is empty (even in Gutsy)

Good stuff:

    * Ntfs accepted
    * Location button
    * Ask for codec
    * Printer detected on network
