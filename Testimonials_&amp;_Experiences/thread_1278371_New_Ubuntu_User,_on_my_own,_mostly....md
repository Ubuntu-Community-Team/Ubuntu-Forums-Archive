It is not a different program to Firefox, it is simply a script that you run in the terminal that updates your existing firefox (3.0.2 I think it is that comes with Ubuntu) to the latest stable build (3.5.3). It just means that you don't need to bother with shiretoko, which needs loads of updates, and isn't as stable (and has a weird name and horrid icon!). 

If you've managed to get hold of shiretoko then you must have added it's PPA's to your source list. If you were able to do this then I strongly reccomend that you give chromium a try, it has flash support now and is very stable, and is absolutely fast as lightning. Check out this link for info [https://launchpad.net/~chromium-daily/+archive/ppa](https://launchpad.net/~chromium-daily/+archive/ppa)

Have fun!
