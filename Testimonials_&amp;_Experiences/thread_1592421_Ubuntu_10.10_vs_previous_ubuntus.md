### Post by geirknappen on 2010-10-10
I have only used ubuntu since 9.04... Have tested ix86 maverick meerkat for about an hour now. I have the feeling it is as solid as ubuntu 10.04.1. The differences are:

* the live cd and installer is more user-friendly (my impatient friend who downloaded and installed the 64 bit 10.04.1 few days ago, can now more easily replace it with the 32 bit 10.10 [he had some problems with running software on the 64 bit 10.04.1])

* the hardware tester is more thorough (very good!)

*the software center looks better, and it shows history of installed software + it seems like all deb packages are opened in the software center.


Differences between 10.04.1 and 10.10 vs older ubuntus:

*both the 10.04.1 and the 10.10 default software is not so solid. For example: if I try to open a zip file, I get an error message, but if I download and install peazip, then it works. The default archive manager of ubuntu 9.04 would open zip files.

* The boot screen on my computer looks bad in both 10.04.1 and 10.10, maybe it is caused because I have several ubuntu versions installed on one hard-drive (triple boot with 9.04, 10.04.1 and 10.10)

My conclusion is: 10.10 is the one to use, for now at least. Time to delete 9.04 and 10.04.1 and make room for other linux distros.
