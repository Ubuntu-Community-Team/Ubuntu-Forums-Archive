### Post by Calash on 2007-06-26
I have been a regular Ubuntu user for about 6 months now, having started in Edgy and upgraded to Feisty.

To start, here was my previous environment before.

Dell XPS Gen5 - 3gb ram, 3 partitions, 2 XP and 1 Vista Beta
Custom AMD 2ghz PC running Fedora Core 5
iBook Clamshell (1999) running OS 10.3

Quite the spread of Operating systems :).  The Fedora box was acting as a server for MP3s and a PVR running MythTV.  My main complaint was that I had to jump through hoops every kernel update...reinstall video, Capture, LIRC...pain in the butt.

So, my Vista Beta partition was getting no use, and with Vista being released it was badly out of date.  Our Unix guys at work suggested Ubuntu as a good desktop OS.  Now, I had tried Ubuntu several months back on a Virtual PC....ran horribly so I dumped it.

Well, around the same time I joined a shared post system called Zoints Synergy.  Well, links from that system had led me here to find this great forum.  Now, armed with both a support system and extra drive space I downloaded the Live CD and went to town.

It took me all of 20 minutes playing with the Live CD to want to click install.  A little bit later and my triple boot system now had Ubuntu on it.  I will admit that at first I did not use it that much, preferring XP as I knew what I was doing.  That is a HUGE benefit of doing a dual boot system, you can go back at any time.  All my gaming was still done on XP, as was all my web development.

However, and slowly at first, I started migrating over to Ubuntu.  I think what really made me spend more time was how smooth the updates went.  Well, that and how awesome Beryl looked and functioned.  Played around and got IM, Email, Gmail notifier, and FTP running without a hitch.  Gimp slowly began to take the place of Photoshop and I found more and more of my time spent in Ubuntu.

When Feisty came out I had just suffered through another kernel update on Fedora.  Never did I have to reload video drivers in Ubuntu, so I took a weekend and reloaded the PVR box to Feisty...not that I needed all weekend.  It took about 3 hours to get everything but LIRC running, and once I found the instructions for that I was ready to go.  The system has been rock solid ever since.

The iBook was next, and quite a challenge.  It still is not perfect, but it runs XUbuntu very quickly and cam do most things.  Once I get the Youtube issues resolved it will go to my GF's sister as her Internet system

I have not touched my XP partitions in months.  Upon finding Cedega and installing Wine all my gaming is done in Ubuntu using a Fluxbox front end (Minor frame rate increase, but worth it for me).  I have a XP VMWare machine for the few things I need Windows for (Time Repoting for work requires IE).

So far my experience has been great with Ubuntu.
