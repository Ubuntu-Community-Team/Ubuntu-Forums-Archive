### Post by Jonathan Métillon on 2008-03-08
Hi,

I'm still a die hard Ubuntu and GNOME fan but I've recently reinstalled my original Windows XP with Sony Vaio recovery discs because my computer use evolved from web development to desktop publishing and mobile online sales presentation (using GPRS/UMTS connection).

I really tried to transfer my habits to Gimp, Inkscape and Scribus, but I'm sad to report that after two trying hard two years I still feel a LOT more comfortable with Adobe Creative Suite, and particularly Photoshop, Illustrator and InDesign.

Their are a few other softwares I badly miss on Ubuntu. For example, Google Earth. I know there's an [official Linux port]("http://earth.google.com/tour/thanks-linux4.html") but the 3D rendering is really buggy with my standard [Intel GMA 950]("http://ubuntuforums.org/showthread.php?t=340518"). However it runs really well on XP. The OCR engine of [OmniPage]("http://www.nuance.com/omnipage/professional/") is a lot more capable than XSANE + GOCR and its GUI more user friendly, with its zone-based [SIZE=-1]multiple recognition modules. I also miss the Nokia Nseries PC Suite for my Nokia N95 8GB, and CyberLink PowerDirector which is the only software to natively handle [the .MOD files my JVC hdd camcorder produces.]("http://ubuntuforums.org/showthread.php?t=231202")

And I like my [fingerprint reader.]("http://ubuntuforums.org/showthread.php?t=222838") I would love it to be usable on Ubuntu when an app requires root privileges! But only XP makes use of it, thanks to the great set of tools from [/SIZE][[SIZE=-1]Upek's [/SIZE]]("http://www.upek.com/solutions/psql/default.asp")[SIZE=-1][Protector Suite QL.]("http://www.upek.com/solutions/psql/default.asp")

**BUT! This damn Windows is so sluggish and buggy, my Firefox is crashing for no reason, apps are taking ages to launch, and having ten apps opened just eat all up of my 1.5 GB RAM and takes tens of seconds to switch between apps!**

So, I will install Ubuntu again. Dual boot. But I also want to be able to use Windows XP from Ubuntu. That way, if I just need to do a quick task with a Windows app, I don't have to reboot the full system and I just fire a virtualization tool.

I posted [my questions about virtualization here.]("http://ubuntuforums.org/showthread.php?t=719025")
[/SIZE]
