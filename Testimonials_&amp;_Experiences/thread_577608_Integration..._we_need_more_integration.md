### Post by buellman on 2007-10-16
Hi,

one thing I really wonder about since I am using Linux is why some/more things aren't more integrated? Maybe I am just a user and no programmer but is it really so hard to integrate things better?

We have a global adressbook in Gnome but how many apps really use it? Gaim does it, why doesn't Gajim use it? Does Skype use it (I know that Skype is no native app for Gnome but as many ppl like to use it... :-))? Are there more apps that could use it?
We have opensync but how comes that there is no easy way for a two-way-sync between 2 PCs Epiphany-Bookmarks, Evolution-Contacts, Calendar?

Maybe there are more things to mention... but I would be happy if anyone could explain me at least the two mentioned points above.

To me it is so obvious that there is something missing and maybe I should write to the developer of for example Gajim instead of writing here but... hmmm... at the moment I would only like to share my thoughts.

The point I would like to make is that I like Integration and one thing that I see is: I think that for example the popularity of OSX comes in part from THAT sort of integration. And no: I don't want Ubuntu to become like OSX or any other OS (if I would like that I would buy myself an Apple-product).
On the other hand: wouldn't it be for example sexy to have some sort of ".ubuntu" or ".linux" like ".mac"? A place where you could store your pics, contacts and yes... maybe some important data because you maybe could trust (in some way) the owner of this .linux or .ubuntu because it would maybe be from Cannonical which you must trust (sort of) anyway because you use Ubuntu. All from one hand... ok... would be good if you could trust this hand :-) But I am sure you ppl understand what I mean.

What do you think about more integration? With the last release(s) and now Gutsy we got/get many eyecandy and so on. Maybe now it is time for some more... hmmm... integration of all the goodies we have with linux.

Thanks for reading :-)

Greets. Buellman
