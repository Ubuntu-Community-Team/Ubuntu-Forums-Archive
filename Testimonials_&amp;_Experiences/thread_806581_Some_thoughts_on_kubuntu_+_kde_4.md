Here's some things that I think should be fixd/added to kde 4 which will help improve it

Support for FTP, ssh, and other protocols should be integrated into dolphin. This might defeat dolphins purpouse of beeing a simple file manager but it wouød be more sensible to have one application handling these things

Tracker should be integrated into the start menu so that it's possible to search for files not just icons. 

Knetworkmanager should be replaced by wireless assistant. I couldn't get wireless to work with knetworkmanager whereas wireless asistant got it right the first time. The best thing would probably be to develop a new application (using the code from wlassistant) which can handle both wierd and wireless connections. 

The startup time of kate should be improved or kate should be replaced by a lighter text editor. It's so anoying when I have to wait a few seconds just to edit a small text file

Vi should be replaced by nano as the standard console editor as nano is easier to use.

Krunner should show console output when running terminal commands

Support for pausing and speed limit when copying a file with dolphin

Drag n drop support when working with icons so that one can drag an icon from the start menu and onto the desktop. As it is now one have to right click on the icon and choose "show on the desktop" which is unintuitive. Add a  possibillity to drag icons from the desktop and into a folder.

A shortcut to .kde4/autostart have to be present in the menu. It could be named startup or something simmilar
