### Post by Plank117 on 2005-11-05
For a long time I had never liked linux. That is, I never really had the chance to use it. I could VNC into a machine downstairs but it was running RedHat, so I never got a chance to use a good distro. A buddy of mine recommended Ubuntu, and I asked him if it was easy to set up a dual boot, as I didn't quite want to just "dive into the pond before it thawed." I wasn't quite sure of the maturity of the distro. OSS is like scotch, it needs to be aged. Ubuntu was easy to set up, and I never encountered the rumored intricacies of setting up a dual-boot. 
I had never really liked windows, as no matter how much I upgraded my computer it ran with all the speed and reliability of a Ford pickup with square wheels. 
Getting hardware to work in Ubuntu is an ongoing process, but I've made a good deal of progress. I suppose the amount of work that needs to be done to make it just perfect is due to it's customizability. I've finally found an OS that isn't made for the dumbest user but still runs stably. I have never liked windows, it's unreliable and slow. I've never liked Macs either, because they are the computer equivalents of playpens. Both are, actually, because they make it so hard to hard to play "Scotty" and fix the warp drive yourself. With Ubuntu I fix everything myself and program with impunity. I have found the holy grail of user experience.
