### Post by jon61484 on 2007-02-14
Currently at this moment I am on a Pentium 2 300mhz system with three 128mb PC100 SDRAM dimms, an 80GB Western Digital hard drive, a simple network card, and a NVidia Vanta AGP video card.

This system, under ubuntu (and downloading xubuntu while I type this) is downloading, as well as processing internet information faster than my other computer. Although generally it's application processing speed is much quicker, under ubuntu in comparison to it in windows, well, I'd rather run this.

The other system is a pentium 4 3 Ghz with 1GB of 800mhz DDR and a SCSI hard drive. It's interesting to think the 9 year old system is downloading at faster rates off the same connection, and generally browsing web files MUCH faster than the year old system is. Kudos...

also. was reading that xubuntu would be better for me because of my older system. so I wanted to download it. This is how easy it was. I went to the xubuntu page from ubuntu.com and clicked on "Get Xubuntu". Then I went down to the part of existing ubuntu users. Said something about using synaptic. I searhced under system, administrative tools just quessing. Found synaptic package manager. searched for xubuntu in the search field, and it came up with all the packages I was interested in. clicked the boxes, clicked download, and 5 minutes later, it's already installing and removing the applied software.

Neat. Not very confusing. I guess the next thing is to figure out the basic parameters and uses of the terminal. I've been doing MS-DOS and windows crap for so long I can repair any system but linux is new and much more fun. so anyhow, this stuff is great. I love it. Thanks.
