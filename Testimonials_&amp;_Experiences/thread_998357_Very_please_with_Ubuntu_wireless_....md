I have been very pleased with wifi support in Ubuntu (8.01LTS and 8.1) ... my wmp54g v. 4.1 cards that have been troublesome with several other distro's work upon install with no tweaking or headaches ... my dwa-142 usb card works perfectly using ndiswrapper ... installing the ndiswrapper gtk interface and tools made using ndiswrapper a no brainer ... Kudo's to all involved in wifi development!
