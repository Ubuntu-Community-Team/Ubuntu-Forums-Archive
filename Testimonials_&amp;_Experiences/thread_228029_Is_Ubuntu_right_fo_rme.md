### Post by jimbren on 2006-08-03
I think you'll find the learning curve on the a lot of linux programs to be much shorter than you think...Open Office, for example.  You'll be able to use the basic features right away, the more advanced stuff in less than an hour.

But I don't want to beat the horse to death...it sounds like you really need to be able to run some windows applications.  In order to avoid a dual boot set up, consider running VMware.  This will allow you to do an installation of Windows within Ubuntu, and install whatever software you need there.  Then when you need to use Photoshop or whatever, you just fire up your virtual instance of Windows and off you go.  

There's an excellent howto on installing and configuring VMware  [here]("http://ubuntuforums.org/showthread.php?t=183209").  Take a look at it and decide if you want to give it a go or not.  It is pretty easy and I think will do everything you need to do.
