### Post by MikeBrown on 2008-11-10
I originally tried out the Linux thing around the time of my "signup date" on my signature to try and find a good alternative for Windows so that I could play Steam games (Half Life 2 episode 1 had just come out I believe) and had not worked for me, so I was hesitant to try out 8.10, but I figured that since I was putting it on an old computer just to learn the basics of Ubuntu and Linux (I've been a Microsoft user for about 10 years now, I know I know but I'm working on fixing that problem) and I have been pleasantly surprised at how easy it is to get a PC that is 4 years old up and running with the new distro. 

For those that are curious, the PC is an old HP Pavilion a545c computer with an AMD 3200+ processor, 1 gig of RAM and an 80 GIG HDD. Nothing too special (and I can get more detailed information if you really want it). 

Then I realized that I could set up a Remote Connection in case I wanted to transfer files on that PC or run a program while I was away (again, not used to Linux where this comes pre-installed and is freeware), so I tried to set it up where I could run the Remote desktop off of my Vista laptop. 

I ended up just installing a small partition on the Laptop for Ubuntu. I was nervous about doing so since I had a problem setting up a dual-boot system the previous time I tried Ubuntu which ended up corrupting my Windows partition and only allowing the Ubuntu partition to boot. But again I must say that I am pleasantly surprised at the ease of install (since it was a very well put together guide) that "held my hand" thru the entire install and as far as I can tell has not over-written anything in the Windows partition. 

For those of you that are (still reading and) wondering "why not just install Ubuntu on the entire HDD of your laptop? Well, I still use Windows for gaming and because my last experience with completely installing Ubuntu on a computer for gaming did not end well, I really do not want to repeat the process. That, and I am not sure what is and is not compatible with Ubuntu in my setup (another problem I encountered last time). I actually would have no problem switching over completely to Ubuntu if I had another gaming rig that I can play on. 

All that being said, I have very much enjoyed coming back to Ubuntu. One of the major things I noticed was that my laptop is not flaming hot like usual, because I guess Vista likes to use up so much memory that your laptop can cook things on it when it's done, which is a nice change in Ubuntu. 

Also I have not encountered a single person on these forums that hasn't been less than nice. I love the community that surrounds this OS because if you have a problem, people are quick to help, not poke fun at you because you have a problem. 

I really enjoy this OS and would switch over completely if I wasn't being held back by my games or my previous experience with steam running in WINE.
