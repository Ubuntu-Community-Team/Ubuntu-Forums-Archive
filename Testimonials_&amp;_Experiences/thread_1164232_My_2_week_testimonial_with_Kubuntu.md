### Post by maflynn on 2009-05-19
I&#8217;ve noticed a number of threads here that basically say ubuntu awful or crap and those folks are going back to windows blah blah blah.  I figured I&#8217;d add my experience and review and $.02.

Linux and more specifically Ubuntu has surprised me in a number of ways. First its low system requirements, I can run this puppy on a machine with a lower end processor and only 512meg of ram.  I&#8217;m coming from a Mac where you need a minimum of 2 gig and to get decent performance you really need 4gig, especially if you plan on running multiple apps.  I&#8217;ve also loaded windows XP, Vista and win7 on my machine and the latter two are extreme resource hogs, which also impact overall performance.  XP isn&#8217;t that bad, but it suffers badly from security problems, Vista probably has that as well but I gave up on that OS and wiped it off my HD

My goal for installing/using Ubuntu was the feasibility of using Ubuntu in place of windows, I never thought that I&#8217;d stop using OSX as I have a large number of applications installed that I need.  Windows however is different as I only need that for my job.  A second reason was to do it to learn something new. 


**First Thoughts:**
I first installed Ubuntu 9.04 in VMware but because the vmtools was not updated for 9.04, I found it a bit frustrating, so I went with 8.10 and found the experience very pleasing as the OS was a stable platform to connect to my work.  I dumped the vmware option and instead created a partition to install Kubuntu 9.04.  As you all can attest to installation was a snap and getting going was much quicker then installing windows (slooooow install) or even OSX.

As I got the OS configured to access my work network, I noticed a trend to needing to use the CLI for many of the simple tasks that I&#8217;ve grown accustomed  to via the GUI interface. For instance installing/configuring/running Cisco VPN is all done via the CLI.  This isn&#8217;t a knock in of itself but the need of using the CLI has been a criticism of Linux as to a reason why it hasn&#8217;t seen wide spread adoption by home users.  For me, its no big deal and found it more efficient in a number of ways then the GUI &#8211; of course YMMV

As I continued to using it, I found that I&#8217;m in ubuntu more then OSX, I don&#8217;t think at the moment it will completely replace my need or desire to use OSX but I can easily see using ubuntu most of the time and in fact that is the case right now.  I do have a number of applications that I just don&#8217;t think there&#8217;s any Ubuntu alternatives.  For instance, I use Aperture from Apple.  While Adobe has a competing product I don&#8217;t think they offer it for the linux platform.  Additionally I have an iPhone which gets synced up on iTunes and a few other apps that are decidedly Macintosh.  Still I find myself booting into Kubuntu more often then not.

**What I don&#8217;t Like:**
No OS is perfect and to that end, there are things within [k]ubuntu that I don&#8217;t like.  I think there are a few bugs in kde 4.2.2 but when I downloaded 4.2.3, it made things even more unstable.  Konqueror is a bit flaky at times, though I like it better then firefox.  Hardware support for my video is a sore spot, as I cannot easily switch my MacBook Pro&#8217;s GPU to the low powered integrated graphics and so far I&#8217;ve not been able to put the laptop to sleep.  Well, I can put it to sleep, it just doesn&#8217;t wake up.  I&#8217;ve created a thread on this, but so far no answers have been provided to resolve the problem. I can say it isn&#8217;t related to the nVidia proprietary drivers because in installing a fresh version of kubuntu (got a new 500gig drive), and before using those drivers or updated the system at all, I attempted to sleep/wakup the computer but no dice.  Since then I&#8217;ve tried various setup configurations in my attempt to resolve this problem.

**What surprised me:**
Is this forum and the community support of Ubuntu, while I may have some issues (they&#8217;re not show stoppers), any questions or requests for help typically are answered here extremely quickly and an abundance of recommendations and suggestions are offered..  I look forward to my continued adoption of Ubuntu, and using this more and OSX less.  While I installed VMware Server and loaded a windows VM in Ubuntu, I asked myself Why, its original intended purpose was to replace windows and it did that with flying colors.

**To Summarize:** 
While my experience with Ubuntu is rather short, I have found it to be a great platform to get my work done, the future looks even better as updates to resolve bugs are released and my knowledge increases sufficiently to better interact, extend and manipulate the system.

While one size doesn&#8217;t fit all, and for some Ubuntu or even Linux isn&#8217;t a good fit, its safe to say that Ubuntu offers a lot to the average user especially given how badly MS has been mucking with windows, along with viruses and various other problems associated with Microsoft.
