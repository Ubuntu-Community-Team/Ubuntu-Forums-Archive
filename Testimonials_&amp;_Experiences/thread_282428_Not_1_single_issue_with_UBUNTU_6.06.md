### Post by mattcn81 on 2006-10-22
First of all a few quick specs:
FIC AM37 board
AMD Athlon XP 2400+
ATI Radeon 9200
WD 120gb Master
WD 40gb Slave
1gb cheapo RAM

So I've been running XP Pro for the last few years, before that 98, 95, etc. I've tired out SuSe, Mandrake, Redhat, etc. This was awhile ago, but none of them seemed to work well with my hardware. So last night I downloaded Ubuntu 6.06 and tossed it on a CD. Tried it and to my surprise not a single problem. So what the heck, let's install this sucker. I installed it to my slave drive, just in case. 15 minutes later I was rebooted into Ubuntu and downloading 187 updates (208mb) which took less than 20 minutes. Here I sit, most likely about to remove XP Pro and reinstall to my master. I haven't had a single problem, which in a way is scary, but that's the Windows user still in me. So to the Ubuntu team, thank you very much.

Matt

### Post by maniacmusician on 2006-10-22
matt, 

i'm glad you're so happy, but perhaps you're rushing a bit? Believe me, I would be elated to see wipe windows and install ubuntu, but i'd highly recommend that you use it for a little while first, and make sure it has everything you need. reinstalling windows is a much bigger pain than reinstalling ubuntu. Also, I would encourage you to mess around with the different derivates (Kubuntu with KDE, Xubuntu with XFCE, etc) before you make your decision to wipe windows. Like i said, it'd be great to have you go full-on ubuntu, but i wouldnt want you to realize later that it doesnt have something you need, and force you to reinstall windows. so just make an informed decision :)

and welcome to ubuntu!
