### Post by squashpup on 2008-06-24
Right now, I'm posting from an ancient 433mhz laptop with 196 MB ram.  I'm using Xubuntu Hardy. Out of the box, Xubuntu ran pretty well.  But now that I've installed LXDE, it positively smokes.  I recommend this setup for any old, decrepit computer you have lying around.  

In Synaptic, just add the following repo for Hardy: [http://ppa.launchpad.net/lxde/ubuntu](http://ppa.launchpad.net/lxde/ubuntu) hardy main

Reload, then, find and check LXDE.  Hit Apply, and you got it.

Log out, and choose LXDE from the login menu.  Login as usual, and you're in LXDE.

LXDE is nice, because it is not only fast, it is also customizable WITHOUT editing config files, unlike IceWM or Fluxbox.  

I'm also using Opera, which runs MUCH faster than Firefox.  Also, if you want your plugins to work with Opera, don't use the version in the repos...go directly to the Opera site and download that one.  

Pretty amazing that this decrepit, 11 year old computer will still work this well, and with a brand new, updated operating system.  Cool.

### Post by squashpup on 2008-06-24
Oh, and BTW, I haven't had any of the lockup problems that others are reporting.  If there was any computer that would do that, it is this one, too...with meager ram and a slow processor.  Xubuntu/LXDE keeps the resource usage to a minimum.  Right now, with Opera open, I'm using 97 megs out of 196 available.  CPU usage is right at 29% or so.  Uptime is currently 1 day, 1hour, and 48 minutes.  

I'm having no stability/lockup issues whatsoever.  Nice.

BTW.,.interesting how I came to be an Ubuntu user.  I started with Knoppix, and liked the idea of a live CD.  I got used to Debian, and eventually settled on Mepis, as it was based on Debian.  I often had problems updating and downloading new software, but I just thought that was the nature of Linux, being a total newbie.

With Version 6, Mepis switched from Debian to Ubuntu repositories.  I was worried about this, but it turned out to be the best thing Mepis could have done.  All the dependency and updating problems disappeared overnight.

With Mepis 7, they switched back to Debian repos.  I knew they weren't as stable/compatible as Ubuntu, and I was disappointed.  But, I gave it a try.  On the very first day, I started running into dependency issues again.  Plus, Mepis now consumed so much RAM, I couldn't run it efficiently on my less than cutting edge computer...the processor temperature kept creeping up in the 120-130F range.

So, being a KDE fan, I downloaded Kubuntu.  It is by far the most stable, usable Linux system I've ever used.  Updates and new program installations always worked flawlessly.  And, my CPU temps at idle were in the 99F range. 

I've used Ubuntu variants ever since.  In fact, I installed Xubuntu on several student computers at the school where I work. 

I'm very happy with Ubuntu/Kubuntu/Xubuntu and, unless they radically change it for the worse, it will always be my preferred system.
