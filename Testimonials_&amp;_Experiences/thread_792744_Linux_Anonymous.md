### Post by enoughsaid05 on 2008-05-18
I have been off Windows since last December. This makes it almost 5 months.

I was excited when I installed Ubuntu but nearly fainted when I realise that it doesn't work as well as I would expect from Windows.

I realise that I can configure my system via GUI.

More importantly, I have a good support for system via online forums. I didn't even call the Microsoft helpdesk when my Windows crashed because I know that in 9 out of 10 cases of crashes, a reboot is often the solution.

Think I can live comfortably without Windows.
