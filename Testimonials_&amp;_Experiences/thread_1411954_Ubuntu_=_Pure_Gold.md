### Post by Scarfnoogan on 2010-02-20
I recently decided to make the switch from XP to linux, mostly because I wanted to try something new, and it's nice to learn things.

I searched a bit and found ubuntu was supposed to be the most noob friendly distro, so I went with it.

I installed from the live CD but couldn't get past the ubuntu logo(nothing but coloured lines of gibberish on the screen) but I tried again, this time using the try before install option and installed from the desktop...Perfection! 0 hardware issues which kind of makes me want to buy a lottery ticket

the 3 main things I wanted to be able to do were
1. sync my iphone
2. back up Wii games
3. broadcast movies to my TV

after about 3-4 days I figured out how to get all these things working.

I wont bore anyone with the how as it's probably common knowledge, but if you'd like to know I will certainly tell.

I love the customization ability(changed the theme, icons, etc) and I'm looking forward to more learning....then it will be time to make the change on my daughter's computer


Anyway, just wanted to say Go Ubuntu!
