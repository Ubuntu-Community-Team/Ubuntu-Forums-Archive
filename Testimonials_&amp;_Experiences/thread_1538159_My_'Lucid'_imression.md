### Post by PC_load_letter on 2010-07-24
Well, I finally took the plunge yesterday, and I clean installed Lucid 64 bit on my Dell Vostro 1400 (Intel Core2, 4GB Ram, Nvidia 8500GS). I installed it from a release day image, installed NVIDIA proprietary driver, then updated. Here is what I think so far: 

- Over all snappiness: More or less as fast as Karmic that it replaced. Booting up is about the same or slightly faster, shutting down is much faster. 

- Plymoth is not working during boot, I don't see any boot splash. 

- Synaptic and specially the administrative password window look HIDEOUS, my god, so no one expects user to see this? For as nice as the new themes look, this inconsistency is unacceptable.

- Quite a few **VERY** annoying problems:
  - The "mouse pointer theme" bug is just awful. Basically the mouse pointer does not update if Compiz is active. Workaround involves doing it manually by editing a config file. Never had this problem before, although some say the bug was 1st reported in 2007.

  - Gnome-do (the version from the repos) crashes much more often than I'm used to, and when you use it for the first time, a miniature image of the desktop flashes for a fraction of a second!!!!????

  - Gnome-do transparency goes away with some wallpapers!!!???? These are wallpapers from the repos. I'm not sure why is that and all of this is while Compiz is on. The original wallpaper looks hideous by the way in 1440x900 res. 

  - The icons (original theme) on the top menu bar look nice individually, but the distances between them make them look some how ugly and it can not be easily changed. 

  - The touchpad controls (from the mouse controls) are ok, but the double click is not fast enough. There is a touchpad utility that I installed that enables fast double click, but it only works for a second before it gets overridden by the system's mouse prefs. 

Overall, it's an improvement, not huge, but ok. The minor agonizing problems make it less than stellar IMHO. What do you think?
