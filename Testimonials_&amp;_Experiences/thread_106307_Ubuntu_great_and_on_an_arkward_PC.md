### Post by chrisTGc on 2005-12-20
Hullo and just to say Ubuntu is real great.

This is something of a difficult PC for any OS,why?;

Early ACPI on this VIA chipset is known to have problems which is why the older Advanced Power Management is the BIOS default.Printer is installed on parallel port.S3 Savage4 graphics card from 1999 is based on the Savage 3D which had issues.Internet is cable modem on USB.This machine is a Bitza (bits of this and bits of that) and started life in 1997 running a 166MHz CPU and Windows 95.Upgraded bit by bit with basement bargain or used parts.Now runs a Celeron 800MHZ chip.

The above means some work even in Windows to get all working well.And then in Windows installing Anti Virus,Anti Spyware et all to keep it working.Windows crashed real bad after doing the latest bit of hardware upgrade.So tried Ubuntu GNU/Linux on hard disc install.

Installing Breezy is easy enough,partitions where there already and it worked good straight out of the box.

Most needed applications already there.Following threads its been easy enough to install the couple of extra's wanted.XMMS for MP3 support.A text file to initiate the firewall.We are up and running well,real well.

Thanks to developers,posts and the Ubuntu Community.

Best Wishes Chris.
