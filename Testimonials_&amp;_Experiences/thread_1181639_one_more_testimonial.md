haha I am kidding. It is a testimonial from a happy Ubuntu user :p

I didn't think that I will write this, but I can't stay aside and see how happy people around spreading their testimonials, so I decided to add something from myself too.

You can skip the text below and move forward into the last section to read my conclusions.
And take my &#8220;sorry&#8221; If it is hard to read, eng isn't my native:)

My experience with Windows as desktop? Starting from school, it is about 10-12 years ago I found windowzzz and it was my OS for so long time.

At my first job I was forced to work on a pc with Linux RedHat (appears it was 6.0). I thought that Linux is something very cool. But it wasn't. I had to use ICQ from terminal in text mode.  Using linux in this hard mode I was feeling myself like a hacker. 

That was all my experience with Linux as desktop. No need to say I didn't want to back and used Windows as my desktop all the time by working as linux server administrator.

I knew and I know  that Linux as server is a very genius and powerful work horse. I have been supporting  linux servers about 7 years by working as a server administrator at  a hosting company. I hate window servers...but not desktops, I continued to use Window as my desktop. 

I got Asus notebook with Windowz XP, it worked ok for me for the last 1.5 years until I started to get the blue screen of death and got tired of anti-viruses and bad performance because of them. My Windows become like a heap of trash. I had to clean  register, defrag disk space, install some programs to automate it. I was fed up. It was the time for changes. I so liked linux servers and dreamed to have a Linux as my desktop, but I was afraid because of my experience with redhad long long time ago.

I had a lot of experience with linux servers and I thought it could help. 

Ok, I found the most popular distro &#8211; Ubuntu, installed it. No, I wasn't happy right after that, I had some problems with it. My Nvidia didn't work, some network issues that got fixed in 9.04, but nothing serious. All the thing were fixed and I compiled my own kernel with just the options I need, never did it for notebooks, but after a few of tries I managed to get it in the right way.

So what I can say now after all?

1. I didn't have to look for all the needed applications after installation, I had everything: my lovely firefox, Ooffice, a player,  email client, messenger,  even a torrent client!!! Codes were downloaded automatically, video works fine without any addition installation.
Do I need to say that I had to look this all in the internet and install it on Windows?

2.the alternative applications (email client, messenger etc.)worked fine, though I used some other on windows. Migration was very very easy.

3.I added so many applets to my desktop and configured my bars as never it could be on Windows. I have all graphics from sensors and a lot of necessary indicators and hot keys! It is a magic. I am a hacker again :D
4.There are a lot of other things that I love on Ubuntu. 

I want to say thank you for all the community for this great distro. There are a lot of work to do, but in the near feature we will see how Linux conquer the world!

I can say I am too much happy. I can't believe in this magic.
