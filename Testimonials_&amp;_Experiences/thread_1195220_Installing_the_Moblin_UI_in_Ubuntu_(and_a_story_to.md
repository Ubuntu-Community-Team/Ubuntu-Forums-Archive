### Post by wowfunhappy on 2009-06-23
I've been using Xubuntu for over a year now. The old computer died about two weeks ago however, and the new computer I've ordered has arrived.

With a new computer comes new expectations that I have for it, however silly that may be. Xubuntu/Ubuntu has always been good, but it's not perfect. And I want to try to get as close to perfect as I possibly can with my new PC.

The main thing I care about is the GUI. I like nice, minimalistic GUI's. If there's a button I will never use, or a menu I will never open, it shouldn't exist as far as I'm concerned. I've been trying several distros over the past few days since I got my computer, but frankly, there is very little variation beatween the different UI's.

Then I heard of Moblin. And I watched the video. And I was amazed.

But of course their has to be a problem. Moblin is made for netbooks, and I am using a desktop.

But after looking at the specs and seeing that I had a supported processor (Intel Core 2), I figured it was worth a shot. It actually works, but the system is very laggy and unusable.

The problem, I think, is my graphics card. Actually, I have no idea why the OS is able to run with my graphics card at all, it shouldn't be possible. Moblin is made for an Intel graphics chipset; I have an ATI Radeon HD 4670.

So, essentially, my hardware isn't supported.

But I want that GUI. That GUI is what I imagine when I think of a "perfect" computer.





Since it's merely the UI I want and not the distro itself, is there any way I could use another distro and install Moblin's UI as my graphical environment? Ubuntu would be preferable, but any disto is fine, so long as it works. I'll even try out Linux From Scratch if I have to; the key thing is that there's no actual programming required, since I can't program (Unless you silly Quickbasic creations).

It there is truly no way to do it, then can anyone suggest some other Linux Distro's, or Operating Systems in general, that I should try?
