### Post by the_lighthouse on 2008-03-09
I recently decided to give Linux a try, partly due to a friend having done so previously as well as becoming frustrated with the speed at which Vista was running on my computer even though the specifications are well above the recommended ones. It didn't help reading about Microsoft and their latest tactics as well - internal emails over Vista and their "Vista Capable" logo. I have been using open source applications such as 7-zip, Pidgin (Gaim) and GIMP to name a few for a while already, so I figured why not give something new a try.

My first impressions were good - Ubuntu ran really well - it didn't feel as though the operating system was there holding up everything else, startup and shutdown was kick and apart from the graphics driver issue I had ... things worked.

I did have some issues to sort through though - the splash screen was missing (had to search forums to find out I had to change a resolution setting in a .conf file), I did have some trouble with the graphics drivers (on my latop (ATI card) it was far easier to solve than on the desktop (NVIDIA) - I needed to install Envy to get it to work as all the manual methods I tried ran into a lot of issues someone of my Linux experience level would not know how to solve), but eventually that got sorted. I had to disable the splash screen for my Nvidia card though in order to boot up and shutdown.

It would be nice if things like that weren't an issue in future versions though - Most beginners don't really want to run into such issues :)

I did run into a few bigger problems a few days later though - on the laptop everything seems to be fine - but on my desktop, I wasn't allowed to write to a partition, I had problems with my wireless USB adapter (wg111v2) I would be able to connect, but overtime the connection would drop to 0% (not a router issue) and I had to manually remove the usb adapter and reconnect it to get it to work - this would happen sometimes within a few minutes and other times over a far longer period. Reminds me of how frustrating it was to get wireless working in Vista ... Related to this (I assume) just recently when I tried to shutdown/restart the PC - I would get a message saying deactivating/disconnecting wlan0 or something like that - related to wireless adapter and it would just freeze.

I took advice from other users who had good experiences with Wicd (appart from removing the network manager and loosing connection to the net and being prevented from installing Wicd) all went well after that and I have yet to have the connection drop to 0% and I don't get that message and freeze on reboot/shutdown anymore.

I don't feel Ubuntu/Linux is quite ready for everyone at this stage, but it's really, really close. 

Hopefully things like black screens/being unable to boot, wireless issues and graphics driver issues can be sorted out in future releases.

I am booting into Linux far more than Vista and I can't get enough of the speed of using the desktop and opening documents, especially PDF's,  Compiz and the wobbly windows - I like to shake the windows around for fun :P and these effects run on a weakish laptop almost just as a powered desktopl - Things like this really put Vista to shame.

Keep the improvements coming and good work to all those involved.

### Post by jespdj on 2008-03-09
Welcome to Ubuntu!

Yours is a typical reaction to first-time Linux users. Remember that most hardware manufacturers are unfortunately unwilling / afraid to officially support Linux. Device drivers for many pieces of hardware are proprietary, and it's hard for Linux developers to write working drivers if they don't have to official specifications for how to control a piece of hardware.

But despite the fact that the Linux developers have this disadvantage, there's a surprising range of hardware that does work with Linux out-of-the-box or with a little tweaking.
> **the_lighthouse said:**
> I don't feel Ubuntu/Linux is quite ready for everyone at this stage, but it's really, really close.
Well, if everything is installed and those little problems are solved, then I think Ubuntu/Linux *is* ready for the average user.

Note that installing Windows isn't for everybody, just like installing Ubuntu might require a little bit more than average knowledge. I installed Windows Vista on a few computers. Unlike Ubuntu, Vista needs device drivers to be separately installed after installing the OS. On my laptop I had to find and install drivers for the webcam, wireless, sound, bluetooth etc. to get everything working, while Ubuntu automatically recognised all those devices.

Sure, there are little wrinkles here and there (such as no boot splash screen in the 64-bit version; I also had that), but Ubuntu is getting better and better at every release (I'm now trying out Hardy alpha 6, 64-bit and so far I like it a lot).
