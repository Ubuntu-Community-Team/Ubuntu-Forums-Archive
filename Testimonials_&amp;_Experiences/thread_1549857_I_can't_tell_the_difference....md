### Post by Old_Grey_Wolf on 2010-08-11
I have three grandchildren that live with me. They tend to dominate 3 of my computers. Of those 3 computers, 1 runs Windows 7 and the other 2 run Ubuntu. The 2 that run Ubuntu don't have the CPU, memory, Windows CDs/Licenses, or disk space for Dual Boot or Virtualization. The 1 Windows box could support Dual Boot or Virtualization; however, it is just convenient to leave it Windows only as it is used as a multimedia server. 

The grandchildren don't know what an OS is. They know that they use the computer with the Windows icon on the bottom left for Netflix. They can use any of the computers for everything else. 

On the Windows 7 box, the applications I have had to install; such as, Email, Web Browser, Office Suite, Image Editor, Video Player, etc., are FOSS. So the applications they use; such as, Thunderbird, Firefox, OpenOffice.org, GIMP, VLC, etc., are the same whether they are on a Linux or Windows machine. 

Most of the games they play on-line run on Linux or Windows. If they want to play something else, they have a Wii in the game room.
