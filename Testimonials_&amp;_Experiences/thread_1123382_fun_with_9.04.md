### Post by drifter0000 on 2009-04-12
I have been using either ubuntu or mint for several months now along with a dual boot of either winXP or Vista. Every time I have to use windows to do something I have come to really dislike it.
Right now I am using the beta of 9.04 and just love it.  When I first started in linux I couldnt get anything to work--sound, skype, dvd, BBC in firefox and all sorts of things---the community in the forums was almost always there to help me.  I remember trying Win Server 2008 and changing it to a desktop like vista, it worked much better and it was faster and used less resources than vista. I made a post on the windows server forums and I am still waiting for a responce, that was months ago.
Things that didnt get a quick response in the ubuntu or mint forums were fairly easy to find with google.
Everytime I cant do something in Linux I have to remind myself that I used windows since dos days and that I understood the OS fairly well.  Some days I would just go looking around in the system files to see what was there.  I spent a lot of money on $50.00 books about windows and read them all and used them constantly for reference.
I havn't done any of these things yet for Linux.  Which in my opinoin represents the normal desktop user trying out linux for the first time.

Linux is so much easier to use than windows its just silly.  A couple days ago I just got finished getting a program to work in 9.04 but python 2.6 kept crashing so i went to synaptic to remove it.  I chose to remove all.  What a big mistake that was, it removed half my system.  And I had been to lazy to back up my home folder yet or to learn how to recover from the live CD.  So I just reinstalled the whole OS.   Took about 20 minutes.  I could never have done that with windows, not to mention all the proprietary software I was using with drm passwords that I would have to ask the software owner for a new free password.

I have no idea how linux works or how the file system is set up or what does what and where.  Yet I still find it easy to use for all my needs.

I am trying to say that if I a just spent a little bit of time reading about how linux works and learn what just a few of the commands do,  I would be able to create and even more fabulous OS with Linux.
For those like me that havnt done that yet it is easy to limp along with help from the forums and use the OS.

I want to thank everyone in the forums for helping me and I promise to study more at some point.
