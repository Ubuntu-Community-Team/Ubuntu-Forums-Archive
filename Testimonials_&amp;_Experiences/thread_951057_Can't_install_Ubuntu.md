### Post by 3rdalbum on 2008-10-19
Did you try installing Ubuntu from the live environment? Some chipsets and BIOSes don't like to boot from a file on an NTFS partition. In theory, Wubi should always work, but in practice there are problems that are out of Wubi's control.

Boot up the Ubuntu CD, run the installer, and resize the Windows partition to create room for Ubuntu. Install Ubuntu this way, reply to this thread saying that you've done it and it's worked, and I'll give your post a "Thanks".
