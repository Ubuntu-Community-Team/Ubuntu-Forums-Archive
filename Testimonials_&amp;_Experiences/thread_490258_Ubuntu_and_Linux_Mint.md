### Post by orange2k on 2007-07-02
I use Ubuntu for over two years now, and boy, what a joy it is to use it!
But a few days ago I decided to give Linux Mint a try. I read a couple of interesting reviews of Mint and it intrigued me, so I thought it was something like Ubuntu on stereoids. I know that Mint is based on Ubuntu and that it uses Ubuntu repositories. I enjoyed installing it: I put in the live cd, a couple of minutes later, the beautifull and very elegant desktop was there. So then I configured my adsl modem and connected to the internet, turned on Amarok and started installing Mint on my hard drive. The beauty with Mint is that it comes with all neccessary audio and video drivers preinstalled, so everything works out-of-the-box. Nice...
After 40 minutes, Mint was installed, so I rebooted and started tweaking it, adding other programs etc. But here the troubles started: I realized that there is no add/remove programs like there is in Ubuntu. What a pity! There is a Linux Mint software portal to install different programs, but the number of programs available there is very limited. Plus when I installed K3b, it froze my computer. So I went to the Linux Mint forum and I realized that the Mint community isnt very large like the Ubuntu...

My Linux Mint trip wasnt very long: I reinstalled Ubuntu today and I wont be using anything else anytime soon. Not that Mint is a bad choice, its just not as ripe as Ubuntu. For instance the start menu in Mint: very nice and elegant at first glance, but not as efficient as Ubuntu. Another thing: I installed Opera in Mint, but it wouldnt show up in any menu - very annoying.  But I very much admire the work put into Mint and I also like the fact that it includes the neccessary multimedia codecs out of the box, Envy is also installed for the installation of grafic drivers, Thunderbird is the default e-mail client...

Conclusion: if you want a distro that comes with out-of-the box multimedia functionality, Mint might be the right choice. But if you want stability and large support community - stick with Ubuntu. Feisty Fawn is still my Nr 1 choice in the world of linux.
