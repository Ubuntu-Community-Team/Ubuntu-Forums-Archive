### Post by getmeoutofhere on 2008-05-11
I had terrible problems with Gutsy - had to re-install every month or so.  Finally, I switched to Linux Mint, Daryna.  Yes, parts of the system still seems to get corrupted every now and then, but it seems to be able to recover itself, which I couldn't get Gutsy to do.  So I have been running Linux Mint for some months, without any major problems.  Linux Mint also has the advantage that it's pretty much ready to go once you've installed it - so really, if you're not a geek, Linux Mint should be worth considering.  Having said that, I preferred Feisty Fawn to Daryna, though I can't get Feisty to run on my current laptop.  As for Hardy Heron - I think I'll just wait for the next version of Linux Mint.
