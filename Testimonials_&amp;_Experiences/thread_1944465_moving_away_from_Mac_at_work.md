### Post by sixtyten on 2012-03-21
I've been using a Mac at work (Mac Pro/Macbook) to do web development for the past 4 years or so. Previous to this I used Windows machines with the occasional experimentation with Linux. When I first got my hands on a computer it had DOS. I enjoyed learning the ins and outs of the computer. Figuring out how this thing works was part of the fun. 

What brought my back to Linux was that desire to get my hands dirty again. I miss knowing whats going on in the background. Sure the Mac makes things easy but I feel its getting TOO easy. Right now most of my work is done in Dreamweaver(Code view) and Photoshop. I like Dreamweaver because of the integrated remote options for uploading and managing the content. 

I've setup an Ubuntu 10.04 machine running Aptana and Filezilla which seems to replace my Dreamweaver needs. Aptana occasionally hiccups on the FTP side so I rely on Filezilla to do most of the work. I've installed GIMP but I probably will end up running Photoshop in Wine or through Virtual Box. So far I'm pleased with the setup and I think once I get in a habit of using this environment, everything will become second nature. I also want to use the terminal more often to use various commands to automate and customize processes. I like that I can use rsync, cron, grep, or create a bash script to fit a need. I know I can use the terminal on the Mac to commands as well but I find myself relying too heavily on the GUI interface when I'm on the Mac. I think Linux forces you to get comfortable with various commands more often than the Mac through installs and such. Why mess with the GUI when I can up a terminal and run apt-get install to install something. 

One other note.. I just pre-ordered a Raspberry Pi and saw a statement on a blog about the Pi which sums up my feelings on Linux and computers today...

"What’s great about Raspberry Pi is that it will hopefully help bring back some of that nostalgia, and get kids excited about understanding how technology works, rather than just using it. "

I got tired of just USING the technology and thats why I'm now using Linux.

I'll try and post another update after I've used this environment for a while.
