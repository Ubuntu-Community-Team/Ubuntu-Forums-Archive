### Post by ausmuso on 2007-06-14
I have just got Feisty running precisely the way I want it and I am impressed and happy!

	This has been my first fory into Ubuntu. I have been a regular Linux desktop user since 1998,
I have worked with Slackware, Caldera, RedHat,Mandrake/Mandriva and, particularly in recent years,
PuppyLinux en SuSE/OpenSuSE. My first SuSE distro was 5.3, my latest one is OpenSuSE 10.1.

	Mind you, SuSE 10.1 isn't a bad distro. It is stable, hardware recognition is very good, and most things
work well. Novell's package management tools are pretty useless but then I've been using Apt4rpm for
a long time, it's very good. The Novell takeover, unfortunately, hasn't been good for SuSE users,
there's been a lot of bad vibes, especially following their agreement with Microsoft. Some things
which used to be so easy to install - like proprietary NVidia drivers etc.- suddenly have become
much harder to get. Somehow Novell's ideas seem to be getting in the way of the true Linux spirit.

	Installing Ubuntu Feisty was like coming home. The easiest and smoothest install I've ever
done. Perfect hardware recognition. Grub correctly detected SuSE 10.1 plus the legacy Win98SE
on my other hard drive, all 3 systems now boot correctly.
	After some tweaking of fstab I can now write and read any file of my systems from
Ubuntu. The best thing, I reckon, is Ubuntu's package management and repositories. With
the universe and multiverse repositories enabled, and after adding Medibuntu there was
this whole slew of software, and it's so fast and easy to get. I've got Totem running just
about anything under Ubuntu, much better then I ever got either Kaffeine or MPlayer to run
under SuSE.
	Gnome took some getting used to, after all these years with KDE. Here I had much help from
O'Reilly's excellent book "Ubuntu Hacks" by Oxer, Rankin & Childers. It helped me get under the
hood of the Gnome desktop and tweak things my way. The only application I still don't like is
Evolution, but then there's nothing wrong with sticking with KMail.

	I guess this is just another way of saying Thank you Mark Shuttleworth and the whole
Ubuntu crew and community, I guess I'll stay with you for a while!

Ausmuso
