### Post by gatorbrit on 2009-11-24
I have an old HP scanner.  I wondered if it would work with ubuntu.   Windows vista and 7 no longer recognizes it.  So I plugged it in, opened up Xsane - and bingo - it all works no problem!!

Then I needed to export a word file (from wine) to a pdf.  I don't have the very expensive adobe acrobat software that I would need in windows, but instead I just installed cups-pdf and bingo - printing to pdf is easy.  Installation took a couple of minutes!!!!

With the single exception of itunes not working too well, ubuntu completely rocks.  Ubuntu really is just easier than windows.
:popcorn:
