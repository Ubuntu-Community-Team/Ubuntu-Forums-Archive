### Post by Thelasko on 2009-04-08
Last weekend I had to reinstall Windows XP Pro on a machine because it had a virus.  I didn't have the disk that came with the computer, but I did have a copy of XP Pro that was purchased separately.  I had all kinds of problems installing it.

[LIST]
[*]After I found the copy protection codes, it took well over an hour to install!
[*]Once the installation was finished, my monitor was stuck in low graphics mode.  Apparently, Windows doesn't ship with Intel graphics drivers!
[*]To make matters worse, none of the network adapters on the machine worked.  Apparently Windows didn't ship with the drivers for those too!
[*]My sound wouldn't work.  No drivers for the sound card either!
[/LIST]

I had no choice but to log on to my Ubuntu machine and download the drivers for the various devices and transfer them over to the Windows machine on a thumb drive.  Imagine if I didn't have a second computer!

Seriously, Ubuntu is **way easier** to install that Windows.  I don't care what anyone says.
