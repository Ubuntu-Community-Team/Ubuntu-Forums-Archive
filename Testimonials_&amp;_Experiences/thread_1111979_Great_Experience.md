I've been an ubuntu fanboy for a long time, but I acknowledge that a lot of the time (based on past experience) Windows completely us on drivers that 'just work'. Windows users in the same office have an A4 sheet of bullet point instructions to run through to install the drivers and add the printer, it took me about 20 seconds!

Well done Ubuntu Compatibility/Driver/Awesomeness Team :o)

### Post by fballem on 2009-03-31
> **tdawgf said:**
> You know reading all this makes me realize something. I really need to buy a printer. Kinko's is bleeding me dry.

Do your research - the initial cost of printers is cheap, but it's the supplies that are murder. Strangely enough, when I did the numbers, I ended up buying an HP Color LaserJet cp1518ni. List price is about $400 CAD. Uses four toner cartridges (refills are about $100/each). I have had the printer for two years, and only had to replace the black cartridge once - the colour ones are still good.

I do a reasonable amount of printing.

The printer works really well using the HPLIP software that is well supported in ubuntu. The challenge for me and inkjets is that the ink dries out at the worst possible times - so an individual ink cartridge might be cheaper, but I would go through a lot of them!

Just a thought.

### Post by 3rdalbum on 2009-04-01
I remember trying to install an office printer on Mac OS 9 and Windows 98. Mac OS 9 just saw that the printer was connected to a spare port on the network and could immediately start using it. Windows was a different matter - I had to install the Windows NT driver onto the office server, and then install the Windows 98 driver onto the client computer and tell it to point to the server. I don't know if Windows is still like that with network printers, but I sure hope it isn't.

When I bought my HP multifunction, I plugged it into my father's computer and it "just worked" on Ubuntu. Literally, plugged it in and got a notification that it was ready to use. Good start. I went into the CUPS setup on my computer to set up network printing, and my local CUPS immediately found the printer attached to my father's computer and all I had to do was click on "Ok".

And people claim that Ubuntu is hard...
