### Post by MaxGunnar on 2007-10-20
I like Ubuntu 7.10,  it was some work to get the Nvidia graphics to work as Nvidia promised it would but thanks to you guys and a few reloads it works great.   The GUI didn't take too long to figure out and getting it networked to three other PCs  running XP was easier than I'd thought it would be,  The network printer took less thirty seconds to load up and print a test page.   I can surf and OpenOffice is great.  I can copy and burn DVDs as well as play nearly any media files.  I haven't fooled around with the photo editor yet but I would guess it is excellent as well.  The only reason I don't switch the other three boxes over is the work applications my wife uses and the game applications my thirteen year old son plays all the time but he did mention he wanted to make a dual boot box.  I use a tray when I need to use XP which is less as less as I go on.   You guys have got a marketable product that you could make money from,  But fortunately the altruistic better angels of your nature shine thru  =D>

### Post by perdition79 on 2007-10-20
I like Ubuntu 7.10 as well, got to the point where I was sick of 5-minute win98 boot sequences on a vintage PC.  Just installed last week, running the system from a 60-gig USB external drive, and it still loads in under a minute.  I can still run Windows off the old 30-gig HD when I need to, but it's so slow. The browser is lightning fast compared to IE 7, and all my hardware conflicts no longer exist.  The word processor works well, spreadsheet is ok.   It was either $800 to upgrade for XP, or spend $65 on a new drive, and this is the better choice.  At least 20 people asked me what Ubuntu is like already this week, so I gave out a stack of live CD's.  Half of them want to switch.

256 MB DDR 333Mhz
Nvidia Nforce 64MB video card
MSI compact mainboard
