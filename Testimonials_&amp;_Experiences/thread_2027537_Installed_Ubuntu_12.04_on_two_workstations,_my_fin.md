### Post by Version Dependency on 2012-07-17
> **GeeLo said:**
> Linux was made to run on older legacy systems, if Canonical keeps "remembering that", I'll be using Ubuntu for a long time to come..

There will always be options for those of us running ancient hardware.  There is the minimal install and the server install...and from there you can install whatever lightweight desktop/window manager/apps you want.  

I have an old machine (about 12 years old) that I know won't run any of the new modern desktops like Unity or KDE4, so I decided to put Lubuntu on it...but believe it or not, the Lubuntu graphical installer would crash the computer.  So I grabbed a 32-bit Server CD and ran the installation, skipping installing all the server-related stuff, and once it was done (about 10 minutes), then I sudo apt-get install lubuntu-desktop...and bam, the thing runs like a champ...and is supported for the next five years! :D
