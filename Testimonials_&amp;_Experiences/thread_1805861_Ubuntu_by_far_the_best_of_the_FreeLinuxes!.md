### Post by nomko on 2011-07-21
First, can we stay **_ontopic_**. This is not the place to discuss whether Linux is free or non-free of costs.
 
Secondly: I don't think or believe that Ubuntu is the best free linux by far. Why? The user determines whether the Linux distro he/she is using is the best for him/her. Some find Debian too complicated to work with and love the user friendliness of Ubuntu while others dislike Ubuntu and stay with OpenSuse because they believe that OpenSuse offers just that which they need. Which distro is the best depends on the opinion of the independent users themselve. What 1 person consider as good quality, the other person may look at it like it is total trash. Claiming that Ubuntu is the best distro by far is a claim which in my humble opinion can't be justified fully unless the opinions of all Linux users are known. This kind of claims are only good to start an "uproar" and an unwanted discussion which eventually will end in bashing and trolling. 
 
It is a fact that Ubuntu is aiming for the group of users which likes the user friendliness of Ubuntu and the way most hardware devices work out-of-the-box. To start with Linux, Ubuntu can be a good choice. But also Ubuntu has some flaws. For instance: despite of using a seperate root account and password, the user itse;f can take over full control over his/her system by gaining root priviliges and entering his/her own password. This is done much better in Debian! Another flaw: the 6-months release schedule of Ubuntu which makes it more unstable. This is not really helpfull of bringing a very stable release on the market. Other distro's are performing much better when it comes to stability! 
 
Ofcourse, Ubuntu is easy-to-use and most hardware works out-of-the-box, but to each benefit, there's also a disadvantage...
