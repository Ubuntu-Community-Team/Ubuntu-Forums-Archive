### Post by rohan000 on 2007-07-16
I recently bought a laptop which came with Windows Vista and installed Ubuntu to dual boot.

In Vista I tried to play a .wav file that I recorded long ago in Windows 95 using Windows' own Sound Recorder program... and it wouldn't play! Then I tried playing the same file in Ubuntu, and it worked immediately. Windows can't even support files made by its own old versions!

Then I have an HP printer which won't work in Windows until I instal its own software which has to load at startup, slowing down the computer. But in Ubuntu as soon as I plugged the printer in, it was detected and installed automatically.

At first I just wanted to play around with Linux and thought that I would end up using Windows most of the time. But since installing Ubuntu I haven't needed Windows once!

The only thing I still find better in Windows is the fact that Skype for Linux doesn't support video... but there's nothing ubuntu can do about that, of course.
