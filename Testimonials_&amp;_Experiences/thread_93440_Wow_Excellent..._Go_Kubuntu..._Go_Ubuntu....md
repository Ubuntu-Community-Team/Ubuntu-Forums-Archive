### Post by Optimal Aurora on 2005-11-22
I have recently tried out Ubuntu 5.10, I was expecting it to crash my system like Fedora, Ubuntu 5.04 and Kubuntu 5.04 did... Heck I was expecting it to be like MS Windows x64 XP Pro was and install itself and not let me access anything...

But I was wrong... Before I had problems setting up my network and creating user accounts (it would create them but not let them log on remotely) and as a 64bit OS it (being 5.04) was the second worstest OS I had ever seen. 

Now with 5.10, I can set up the network and user accounts and they be able to log in. I really enjoy the bootsplash image at the beginning that tells you that its loading the modules and configuring the network. I like the fact that finally it has the LVM which was part of the reason I constantly used Fedora for... I like the fact that for a change everything I use servers and graphics and programming software all work... Thank you developers and fellow k/ubuntu users...

I do still have a complaint, its a small insignificant complaint... I am a AMD64-K8 user. I would like to use dchroot to get access to 32bit breezy but I am having some problems with it not registering that I installed the files on to the system. But that is okay because I really don't need flash player too much... And being that Pentium D is 64bit enabled Macromedia should make the Flash 64bit plugin soon...

I would like to apologize to the last time I was here I made a very nasty specticle of myself. Sorry...
