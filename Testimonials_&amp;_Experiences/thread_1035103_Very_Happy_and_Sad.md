### Post by Tony Flury on 2009-01-09
just a small correction : 

The WWW was nothing to do with Microsoft : Most Web servers run on Unix (even some of Microsofts Corps own servers used Unix up to a year or so ago). The first Web server was an Open Source tool written at Cern for a Unix system.

Nearly all of the first internet tools were Unix based (ftp, usenet, gopher, email, etc). Even the first PC browser (for MS Windows) was not even microsoft - it was an free Open source browser called Netscape.

In fact Microsoft dismissed the internet and Web browsing, and only came up with Internet Explorer 1.0 as an after thought when it had realised it had missed the boat.

Yes I do realise that most people's first (and only) experience of the internet is via Internet Explorer and hotmail - but that hides the decades of non-Microsoft, free Open Source effort that gave us the actual internet and the WWW.
