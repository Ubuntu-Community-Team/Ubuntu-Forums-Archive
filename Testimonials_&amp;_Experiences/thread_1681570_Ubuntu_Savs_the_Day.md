### Post by raw157 on 2011-02-04
I had a pretty big project due today in class, I walked into class and noticed the computer was running windows 98 (how sad), it had a wicked old version of power point on it. Needless to say, my version was not compatible with that version. Luckily, I had my flash drive with ubuntu 10.10 on it. So i loaded it like a live CD, and ran my presentation on that.

I'm so thankful for ubuntu and the live CD ability it has. Without it I'd surely be SOL and have failed the project. I owe ubuntu my life.
