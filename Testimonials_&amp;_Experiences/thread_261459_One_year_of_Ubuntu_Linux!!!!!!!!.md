### Post by PatrickMay16 on 2006-09-20
Today, it's a year ago since I first installed Ubuntu Linux on my computer. 
For the first week of using it, I felt alien in it, and not at home. So I would boot windows 2000 a bit. But by October, I was using it more than windows. And by november, I barely used windows 2000 any more.
Since then, Ubuntu became my main operating system. 

These days I only ever boot Windows 2000 if something needs to be tested or done, or I'm feeling nostalgic for the "old days" when I used windows. So I use windows 2000 for maybe two hours every two months. The whole rest of the time on the computer is with ubuntu.

Ubuntu is excellent. Without it, I probably would never have gotten used to Linux. 

I'd like to make a big thank you to the ubuntu community for your great support, a big thank you to the ubuntu developers for a great system, and a big thank you to the whole ubuntu company, for shipping free CDs. 

By the way, in the year of using Ubuntu on this computer (my main computer), I installed it once. Whenever a new version came out since then, I dist-upgraded, and never had any problems with doing that. So I have had Ubuntu installed since then, and haven't reinstalled it once.
That says something about the upgrading procedures of Ubuntu, I think.
