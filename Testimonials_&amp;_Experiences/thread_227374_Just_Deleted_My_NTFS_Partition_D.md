### Post by darrenm on 2006-08-01
Been a Linux user for years and years but I always had to keep WinXP on another drive for emergencies. 

Switched from Mandrake to Ubuntu just on the beta cycle of Dapper and I have been amazed how good of a distro it is.

The only things I needed Windows for were video editing, playing some games (just half-life mainly) and some car EPC software.

Wine runs half-life fine, Im dropping myself in the deep end with Kino, Cinelerra and the rest and I (unfortunately) have Windows installed on vmware just in case I need to use the EPC software until I can get it working in wine.

Tonight I got to the point where there wasnt any point having Windows there any more so I just run exclusively Ubuntu now.

In all honesty I really do despise Microsoft and its great to be able to vote with my feet. I even have a fully legit license but I value my privacy and data security.
