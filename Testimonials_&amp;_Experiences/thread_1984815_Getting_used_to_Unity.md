### Post by TBABill on 2012-05-22
Well the interesting this is to see Gnome "borrowing" from Unity. OS's tend to copy each other's advancements in some ways and it speaks volumes that Unity has brought some tools to the table that others can appreciate and duplicate. 
 
I'm not into the "year of" discussion because growth is usually natural and steady unless some life-altering event happens to revolutionize something (like the iPod and the Internet, for example). I do think there will be a steady flow of defectors year over year, but those numbers will probably never gain a giant piece of market share even over time without that something revolutionary to raise awareness, curiosity and perceived need.

### Post by 3rdalbum on 2012-05-25
"Well, you can have this computer with Windows 8, which looks ugly and works completely differently to what you're used to, and runs everything full-screen; or you can have this other computer with Ubuntu which looks nice and has lots of helpful features, plus is easier to use on a desktop computer than Windows 8".

A little bit of salesperson training, some computers running Ubuntu in major stores, and a slightly lower price for the Ubuntu models, and it would be the closest we'd ever have come to the Year Of The Linux Desktop.

No but seriously, Unity has really set a benchmark for computing in the future. No desktop user interface can touch Unity right now. Sure, Unity requires a little bit of re-learning; but then that was the complaint that MS-DOS users had against the Macintosh GUI back in 1984.
