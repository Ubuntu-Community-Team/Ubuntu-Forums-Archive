### Post by 3rdalbum on 2008-05-12
If you want a great challenging Tetris game for Ubuntu, you should install "Bastet":

```
sudo apt-get install bastet
```

It's terminal-based, so use the terminal to launch it:

```
bastet
```

The games work fine as a little bit of light distraction; they're not there for serious gamers. Most people find Rhythmbox to be fine for their needs. Songbird without the web browser component is like an LCD TV with 50cm of cabinet behind it - defeats the purpose.

Totem Movie Player also works fine here; not sure why it's crashing for you. Maybe you should file a bug report and attach one of the files that's crashing?

Remember, you've gotten used to all the programs you like on Windows. The Linux programs are made by people who want their own certain features. You could file "Feature Enhancements" into the bug trackers of the certain projects and the developers might implement it.
