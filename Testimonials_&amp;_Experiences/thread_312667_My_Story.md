I ported to Ubuntu from Windows XP some months ago and have replacements for every program I used to use:

iTunes -> Amarok
FTP    -> gFTP
IE     -> Firefox
MSN    -> GAIM
Dreamweaver -> Quanta
MS Word -> OpenOffice.org Word
Microsoft Visual Studio -> Mono ;-)

Its all there and designed much better for the 'intelligent' user with good features which will actually surprise the user as to how clever they are. The only thing a lot of native windows apps will do is surprise you when they crash and you lose all of your work!

So go ahead and install Ubuntu and be enlightened!
