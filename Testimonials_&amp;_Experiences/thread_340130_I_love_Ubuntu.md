### Post by Sunflower1970 on 2007-01-16
Every day, I love this OS more and more, which includes Kubuntu, and even Xubuntu (both which I've been playing around with on my older computer system)

In December, after reading more and more about Vista, I made a goal that in 2007 I would begin learning about Linux because I no longer wanted to use MS products, the "security" of Vista irked me. I figured I'd give myself the entire year of '07 to learn about it. I was going to take my time in doing a big switch like this. 

So, I began doing research--typing in google something like "user friendly linux" and up popped Ubuntu. I remember thinking it was a funny name, and pushed it aside, going instead and looking at the one distro I knew about, Red Hat and their Fedora Linux. But, I was still leery, and kept on looking. That's when I ran into distrowatch.com, and found reviews on Ubuntu, and noticed that it was at the #1 spot. It now piqued my interest, so I came to the site, and began to read. I was impressed with what was written on the main site, and also with the forums and all the help people were giving one another. It seemed very newbie-friendly, and that was what I was looking for

I downloaded the Ubuntu and Kubuntu .iso's to see what flavor I liked better. Then, December 30th. My main computer dies with Windows XP on it. AAUUGH! My crash course in Linux begins....I have a second, older computer which I never use, so I dusted it off, and hooked it up to the 'net and began to do some more reading on saving the info on my hard drive so I can do a clean install of XP (I wasn't ready yet to use Linux full-time). I used Ubuntu's liveCD, but I had also come upon Knoppix (before the crash) and used these discs to help rescue my data. I became quite enamored with Ubuntu, and more and more pissed off at XP. So much, in fact I was thinking of going cold turkey and not reinstalling XP...(in the end I chickened out and did install both...). 

I found an old hard drive not being used, hooked it up in the 9 year old computer (which was still sporting a very slow version of Windows 98 on it) and installed it. Between Ubuntu and Knoppix, the data on my new computer was saved, and I was able to do a clean install of Windows XP, (but I also put Ubuntu on the newer one as well). 

I'm amazed at how well the older computer runs now. It's still a bit slow, but much perkier with Ubuntu on it. (It's a Compaq Presario 5630 Pentium II, 400mhz, 384 MB RAM, 2 hard drives that are 15GB each (lol) ATI 3D Rage Pro graphics card) Both computers now have pretty much the same stuff on 'em, which, again, amazes me. And the newer one (Pentium 4 2.2 Ghz, 1.5 Gig RAM, 2 hard drives (370 Gigs) and nVidia Geforce3 Ti 500 graphics card) zips along faster than it ever did with XP. In fact, I did a test one day. I booted up Ubuntu on the old one, and XP on the newer one, and Ubuntu actually booted up *before* XP did. Again, amazement. 

There are still a few things here and there that I need to learn--setting up the printer, being able to hook both computers up to use the same cable internet connection, and I'm still learning about the command line. I'm not real comfortable with it, but moreso than I was a few years ago when I first encountered it with my brother who was using it after he was having some huge problems with Windows 98 ). I'm also in the process of upgrading the graphics cards of both computers--and I'm a bit nervous about that, since the older one has the ATI card and I'll be installing the old GeForce3 card in its place. (at least, I hope it'll work--crosses fingers)

I have not booted into Windows for roughly two weeks, now, preferring the speediness and a bit more security with Linux, and also loving--LOVING the customization of my desktop, from changing fonts, to all the themes, switching out the login screens and splash screens. 

And, all the help I've received. I've posted a bit, but I've been able to look up just about any problem I'm having here by doing a search, and usually find the answer I need to solve my problem. Something I've never been able to find with Windows. I really appreciate what's been done here, and I look forward to being able to rid myself someday completely of anything Microsoft (well, except the keyboards and mice. I still prefer those to other brands :mrgreen: )
