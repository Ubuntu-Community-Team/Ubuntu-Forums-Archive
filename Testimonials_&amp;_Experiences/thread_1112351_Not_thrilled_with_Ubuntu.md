### Post by 3rdalbum on 2009-03-31
Well, I know it's a lot of work to begin with, but you could find out what applications to remove and then customise a live CD with all the unnecessary packages already removed. Once that's done, install it to your hard disk and build up as you want.

This is incorrect.  Even though the installer can get packages from the repositories, a full-blown Ubuntu installation is not required.

One way to get a *CLI* installation, when using the [Ubuntu Minimal CD](https://help.ubuntu.com/community/Installation/MinimalCD), is to type (at the install menu screen)...

### Post by Therion on 2009-04-02

I never said a full blown install was *required*.

Further... The Canonical website for the Minimal Install CD says:> ***The Minimal CD downloads packages from online archives at installation time*** instead of providing them on the install CD itself. ***Downloading packages at install time reduces the size of the install CD*** to approximately 5 to 20MB depending on architecture (see below), as well as providing only the packages needed for installation.
