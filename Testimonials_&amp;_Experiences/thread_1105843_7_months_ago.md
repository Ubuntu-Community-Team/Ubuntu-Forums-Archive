### Post by lovinglinux on 2009-03-25
Hi,

A few months ago I was extremely unsatisfied with Windows and I decided to try Ubuntu for the last time. 

There were a couple of moments that I almost decided to go back to Windows, mainly because of issues with hardware and configuration stuff that I couldn't understand. But with the support of this community, a little bit of patience and determination to understand the difference between both systems, I managed to get everything working. Today I regret not switching before. Ubuntu has changed my computer experience completely. Additionally, my system is now more stable than ever and it requires little maintenance, something that was a nightmare when running Windows.

Since this is my post number one thousand and I'm using Ubuntu for 7 months, I thought would be nice to make this post a testimonial. I know it's silly to "brag" about number of posts, but this is kind of a special moment for me, like an anniversary. Ubuntu has changed my computer experience, but this community made me feel at home. I'm a happy and proud Ubuntu user and member of this forum.

So, if you are reading this while still considering switching to Ubuntu, I recommend you give it a try. There is a big chance that you won't regret it.
