### Post by Dragon-RD on 2008-11-11
First of all allow me to say I've been a Windows user since I started using PC's and using Ubuntu 8.10 is my first true experience with Linux.

I decided to make the switch, not because I was unhappy with Windows (I still keep a HD with Vista) but more out of curiosity. I had been reading more and more about Linux recently around the web and decided to give it an try. I expected it to be painful. It really isn't, to my shock the install was flawless, Ubuntu then detected the latest Nvidia drivers for me. I started searching around for the Realtek sound drivers, only to find Ubuntu comes equiped with them (I think, well it works :P ). Totally flawless, the only problem I ran into was monitor resolution (Which I made a post here about, due to my failure as a geek I thought I was using 8.04 rather than 8.10), I resolved the issue easily by doing a google search and manually editing xorg.conf. I then decided to set up my desktop, and with the help of Compiz-fusion I now have it looking exactly how I want it. The next thing I looked into was performance this probably shocked me the most 351MB of memory used. That is less than half of what Vista uses.

I can't believe how something this good can be free. I'd just like to thank the Ubuntu developers on doing a truly stunning job. Sorry if this was posted in the wrong forum but I had to get it off my chest. 

Superb work all round,

Dragon-RD

(Yes I'm a Baldur's Gate 2 fan, which might I add runs perfectly under WINE!)
