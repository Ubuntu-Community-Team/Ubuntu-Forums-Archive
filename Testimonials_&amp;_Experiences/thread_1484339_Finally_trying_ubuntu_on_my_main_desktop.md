### Post by screaminj3sus on 2010-05-15
Trying wubi for the first time and trying out ubuntu on my gaming desktop (2.6ghz amd quad, 4gb ram, 4870) Only game I really play lately is bc2 and im waiting till the new patch until i start playin again so why not :)

Install went perfectly, dont notice any hdd performance decrease with wubi (i did install it to its own partition so it shouldn't fragment at all) Everything worked out of the box except my creative audigy 4 sound, but I just had to unmute something in alsa. There was one horrific experience I had which was trying to transfer all my utorrent torrents over to ubuntu. I was initially using deluge and god that was a bad idea. Its file checking is horribly broken and everytime I restarted the program files that were previously 100% and seeding would be stuck checking and/or back at 0%. After an hour of trouble with that I used ubuntu's default transmission client which worked perfectly.

Only other small issue so far was Even though I have both my windows drive set to automount on startup rhythmbox likes to freak out and remove all music from my library so I have to wait for it to add it all again, fixed that by switching to banshee.

I am definitely liking it so far, boots faster then windows and is a little faster.
