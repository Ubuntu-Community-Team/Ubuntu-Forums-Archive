### Post by goaliefight on 2010-01-13
I hope the Ubuntu Testimonials & Experiences board is a good place to post rants - because I have a few of them I would like to submit (constructively) so that maybe others won't have the same experience I have had.

I've been working with Unix and Linux servers for 11 years. In the Fall of 2008 I switched to Ubuntu from Windows because of security concerns. Unix based systems have always been my passion, but only recently have they advanced in the desktop arena to a point where it was practical for me to switch.

I work on computers from 8-5 M-F. When I come home I don't want to screw around with a computer. I simply want to read my email, catch up on news, and watch videos. I also want everything to "just work". This is the reason I selected Ubuntu.

I enjoy tinkering with my laptop, but there comes a point when becomes tedious and frustrating. When I first started using Ubuntu (8.10) as my main OS I quickly became accustomed to fixing problems with the Ubuntu desktop. It was fun and challenging at first, but now it's just stupid. I can't rely on Ubuntu to be a stable platform for even the simplest tasks.

When 9.04 came out, I was relieved to find that many of my problems in 8.10 were resolved - only to discover that my old problems had been replaced by an entirely new set of problems. Despite these problems, I continued to stay the course and work around them in the name of freedom, security, and change. I patiently waited for 9.10 in the hopes that it would hammer out the rough edges in Ubuntu and make my life easier. Sadly, I was let down again.

Ubuntu 9.10 turned out to be more unstable than any other version I have tried to date. Still, determined to fight, I stayed with Ubuntu as my primary OS. For a month I held my hopes high that 9.10 would be patched and become more usable over time. Once again, I was let down.

Since then, I've moved back to 9.04. I have also tried the 10.04 alpha in a desperate attempt to resolve problems. Sadly, the 10.04 release has continued the downward trend in the Ubuntu desktop.

I've submitted bugs, read forums, and been all around the internet trying to get a stable Ubuntu system. I don't have the patience to deal with it anymore. I don't know what distro I am switching to, but it's obvious I won't get where I need to be with Ubuntu. 

I believe the 6 month release cycle is flawed and needs to be changed, or Ubuntu will continue to suffer. Release software when its ready, not when it's due.

tl;dr
Fedup with Ubuntu's quality

### Post by Ugluk on 2010-01-13

Windows firewall is on by default, and blocks all incoming connections, unless you run Home Network Wizard, and explicitly specify that you want your files/printers to be accssible from the network. Firewalls cannot be 'cracked' or bypassed because the TCP/IP drivers filter firewall from any unsanctioned outside information. To change firewall setting one need to have administrative access (which no user has in Vista).

Also, if you put your machine in LAN, you definitely want your folders to be accessible from LAN. Likewise, if you install Samba, it binds to all of your interfaces, including WAN, unless you unbind it explicitly, just like in Windows.

And finally, if you run trojans with administrative rights  (which is more hard to do in Windows, BTW), all the security measures are void, since root user ignores all the file permissions, can tamper with firewall, and bind to any port.

P.S. Most home users use hardware router for ICS, which completely solve virus problems, despite possible misconfiguration of client computers.
