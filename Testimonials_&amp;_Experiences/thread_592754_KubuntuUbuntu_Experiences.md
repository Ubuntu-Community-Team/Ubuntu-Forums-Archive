### Post by vp.dancer on 2007-10-26
Greetings, I've been using Kubuntu since the 6.06 release, this was the first ever distro I used without being angry & loosing my temper with. When I first downloaded 6.06 I was very impressed over the way it worked better & it seemed to me to be more stable than Mandriva that I had tried.

Kubuntu was excellent from the start, now obviously being new to the technology I was a bit confused at first because obviously **Linux is NOT Windows**. For the first few months of using Kubuntu there were some problems that I faced, I found my answers by some searching & in the end I started using Kubuntu full time.

I would never like compiling things if I had to & I never understood how to set up certain things or even the file structure that Linux uses. Now I don't even think of any of this being hard or difficult to use.

In the past year I'd say things have progressed very well & that I feel that the development has improved much faster in the past year than it has for a while & I'd just like to thank the Ubuntu team & all of it's users for making these possible.

I'd also like to state that the WINE project has come along wounders in just one year. When I first used WINE Last year I found the support for applications to be terrible & that everything I ran on it seemed unstable, not to even mention games. Now the WINE project runs all the games I like playing & it will run everything I need nice & stable (I Can use Linux for my web development as Dreamweaver, Fireworks, Frontpage, Flash etc work fine on WINE Now), definately a lot of improvement was made there.

There are many great points about switching to Kubuntu, there's a small learning curve but after using Kubuntu for a few months it's quite easy to take to using it full time. I still keep a Windows machine for testing purposes for the customers I have.

I like to demo Kubuntu to users & I offer it in one of my system build packages in my business (I Will report the sales soon). Anyway just a thanks to the Ubuntu team.
