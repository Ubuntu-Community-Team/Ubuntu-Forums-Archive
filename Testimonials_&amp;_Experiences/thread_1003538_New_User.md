### Post by montheponies on 2008-12-06
Hi,

Just thought I'd jot down a few thoughts and comments since installing ubuntu (8.04) for the first time a few weeks back.

I installed this onto the wife's Dell 1525 as the preloaded vista was really crippling the speed of the machine (she only uses it for browsing, webmail, itunes etc). 

Anyway, first off the installation went like a dream, with dual boot automatically taken care of (had to have my safety net of vista just in case it all went horribly wrong).

Excellent support for all the native features of the dell laptop meant that there was little time wasted buggering around. Only slight issue was with wireless which was a bit intermittent - one update later and it was cured :D

Some excellent tutorials on setting up samba and mounting my NAS shares, has meant that laterally we're using songbird as the itunes replacement. 

On that the level of support is excellent, but I'd still say there is a fairly steep divide between complete novices and ubuntu given the majority of support is around the CLI. But for me personally I quite enjoyed it (sad that I am).

On the useability front, once setup and working it's stable and easy - no greater testament to that is that my wife has been using the laptop without complaint - even moreso than when vista was the default OS.

In summary I'm really impressed with how flexible and powerful this OS is and can only thank the numerous developers for their time in putting together such an excellent product.

Cheers Guys :mrgreen:

### Post by solwic on 2008-12-06
> **blakjesus said:**
> I love to hear new Ubuntu success stories.

Although as far as the CLI goes, most people would rather give you commands through the CLI to give support because it is convenient and fast. Instead of telling you: go to A -> B -> C and click D then wait and click E the can tell you type "random CLI command" instead. The CLI is your friend. :) You will learn how usefull it is to have the GUI and the CLI to work to your advantage.

This is the first lesson I learned.  I used to bitch and whine about the command line, too.  "This is the 21st century," I'd say.  "We've evolved beyond black screens and white text," I'd say.  "It's called G U I people, learn it."  

All comments that not only made me look like a four year old, but also tried to hide the 'lost' feeling I had.  I'd used windows for 14 years; I didn't want to think that I really *didn't* know as much about computers as I thought I did.  So instead of adjusting to the culture shock by learning about the culture, I tried adjusting by bitching about it.

Didn't work, and it only took a few helpful, kind, patient people that were there in the beginning telling me to "type this in the terminal" to fix whatever problem I was having for me to see that CLI is an *amazing* thing, and that it is much, much, MUCH simpler than trying to determine what GUI a person has (Gnome, KDE, Xfce, etc) and then writing out a dozen or so steps...all the while hoping the person hasn't hidden their "system tools" menu under applications because they never used it, or that they haven't modified it in some other way.

Yeah, the command line is awesome.  What's even more awesome is learning some basic shell scripting (very easy to do, even with only a passing acquaintance with programming theory) and automating custom tasks.

So the moral of the story:  learn the CLI.  It is your friend; far more so than the GUI is.  :cool:
