### Post by jeff.sadowski on 2008-11-30
I don't like it. Time for me to try gnome again.
If I'm going to have to learn new ways of doing things I might as well use gnome.
Adding buttons to the menu bar at the bottom is no longer as easy as drag and drop a button.
I like the locking of things that was a cool idea you can set things how you like and lock them making it harder to make a mistake. Although some would get annoyed.
I'll play around a little more with kde4 but I really think kubuntu's kde4 move is going to push me away from the kde desktop altogether. It wasn't exactly kubuntu's fault given that kde itself made the changes. One thing they have to learn though on linux the most important app .... let me start again .... on any OS the most important app is the command line. By default you should be able to easily get to a command line. I found it but it was just not intuitive almost as important as putting development on the install cd and making it easy to select.
