### Post by mastablasta on 2011-01-31
There is  the Ubuntu manual project (free e-book). Have you read it already?
Ubuntu pocket guide ( another free e-book for a bit older Ubuntu).
LinuxCommand if you want to learn about commands (yet another free e-book): [http://linuxcommand.org/tlcl.php](http://linuxcommand.org/tlcl.php)
 
This to me is more than enough to start. for any specific issue there is:
Ubuntu documentation pages.
You can also use man pages in terminal to learn about command. to get t them type in terminal:
 
man [command]
 
 
exit by pressing "q"
