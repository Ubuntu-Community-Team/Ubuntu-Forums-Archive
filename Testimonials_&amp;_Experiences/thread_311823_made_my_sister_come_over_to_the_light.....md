### Post by scotty2hott2k on 2006-12-03
so my sisters/boyfriends pc at their house was built by a friend of their's, it had winxp home on, however one day it started saying it required activation etc, since i had an xp cd i had from when i used winblows (but no longer do due to using exclusively ubuntu) i took it round theirs to install a legal windows for them....how much hassle was that...

it couldn't detect any drives to install onto as they use a sata drive and winblows requires the 3rd party drivers to work, trouble is they don't have a floppy disk drive and the drivers are on floppy disk. i thought no problem, i will put them on cd and load them via that...goto install windows then it says basically it can't install drivers from a cd only floppy disc.

i gave up and installed ubuntu (dapper due to the wireless driver issues in edgy) on their machine, install was as smooth as ever.

they have since been using ubuntu for the last few months and she has said how its much better/easier to use (i setup all basic multimedia functionality for them).

just got to teach them how to do their ipods on it, then they are all set :)

any way, thanks ubuntu community for building such a great distro and i look forward to feisty.
