### Post by JoelKrause on 2010-12-05
I've been using Ubuntu on and off for a year or two now, and have settled into it numerous times, but have found myself needing Windows for school, and with my limited system, didn't have the resources to run a VM. I'm giving it another shot... Who knows, I might not work out, but if it doesn't, it's not because it's a dodgy OS, it's because it just wasn't working they way I wanted it to. 

I plan on doing some Open Source Development, and thought why not do it on an "open-source" operating system... So yuh.

I will be attempting to use Ubuntu as my main OS, but if it doesn't work out, take no offence to it!

I've heard wonderful things about Ubuntu Server and when I get the parts in for my new desktop, I'll be running it as a server, with of course Ubuntu.

The one thing I love about Ubuntu is how light it is on the system. With Windows my 2.5GB of RAM could be gone with only a few things open, on Ubuntu right now I have Chrome, Thunderbird, Skype, emesene, Python, Eclipse, VLC, and FileZilla open and only using 20.5% of my RAM! :D

I hope Ubuntu and I work out together, it seems really great!
