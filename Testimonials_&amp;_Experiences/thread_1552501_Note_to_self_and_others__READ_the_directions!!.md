### Post by M_Mynaardt on 2010-08-13
I tried installing Ubuntu late last night; not the best idea.  I wanted to install Ubuntu on my old Toshiba Satellite A100, which has always run painfully slow with Vista.  First attempt to install left my computer with no OS and Ubuntu did not fire up.

After I'd calmed down (a bit) I tried to reboot the computer with the Live CD I made; yes, I did follow some of the directions!

The installation worked the second time!  I tried the "Try Ubuntu" option and _then_ the full install.  Then everything worked fine.

I *think* the abundance of junk on Vista interfered with the installation the first time.  The second (successful) installation wen through steps that didn't happen the first time.

Anyway, my advice is **_[COLOR="Indigo"]read[/COLOR]_** the instructions, do ***not***try the full install of Ubuntu until _after_you use the "Try Ubuntu" option first.  At least on a Vista machine!

And I'd also like to say "Good Riddance Vista!"  :P

Ubuntu runs so fast on this A100 compared to Vista, it's hard to believe it's the same machine!
