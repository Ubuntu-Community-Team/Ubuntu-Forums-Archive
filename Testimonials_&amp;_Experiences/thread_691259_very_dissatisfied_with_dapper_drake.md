Obviously, everything will be sucked for you so I will make this short.  You only have permissions to write to your own $HOME directory even if you are the only user on the machine, the only you create during the installing process.  It doesn't mean you are system administrator or anything like that.  However, you have access to root privilege so you need to use sudo (from terminal) or gksudo (GUI apps) if you need to write anywhere outside your $HOME directory.

[https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)

### Post by Zill on 2008-02-08
drsmrgn3000:  Dapper Drake is the long-term support version of Ubuntu.  This is a rock-solid version that I have used happily since it came out.  The newer versions, such as Gutsy, are very good but *may* not yet be quite as stable as Dapper.

If you have just started using Ubuntu may I suggest that you "don't run before you can walk".  Install the system from the live CD, using all the defaults, and then have a look around to see what you have - lots of sofware will have been installed automatically.  Linux is not Windows and there is a lot to learn.  Often different applications are used to provide similar functionality to Windows programs but it takes time to learn about these.

If more software is required then it is very simple to install new applications - far simpler than Windows in fact.  Just play around with your system and expand it as you go along.  Don't forget that it took a long time to learn about Windows!  Once you have moved over to Linux you won't go back :-)

I've had Dapper installed via the live CD since it came out and had no problems whatsoever with partitioning my system into a dual boot machine.  I set the resize slider at 50% and let the automatic partitioner handle the rest.  I lost no XP data in the transition and  I've upgraded the Ubuntu side of the hard drive every time the OS has matured.  I went to Gutsy while it was still in beta and still had very few problems.  The problems I did have were solved with a little research and help on these boards.  The kicker to all this.  I knew absolutely nothing about Linux when I started this.  I still only know a handful command line prompts. But the time I spend in my Windows partition these days is relatively small compared to the time I spend in Ubuntu. My point here is that for people like me, with no real prior knowledge of Linux,  Ubuntu is an easy distro to get your head around. I'm not sure where the original poster went wrong with his installation, but I've had none of those experiences. I guess I went into it with the expectation it was going to be different than Windows and was willing to accept those differences. As far as I'm concerned, there's no real right or wrong OS to use, only the OS that is most comfortable for *you* to use.  My Ubuntu setup is faster with less bloat (everything I do in Ubuntu takes only 5GB--my XP partition is something like 20GB).  It's been well worth the small learning curve and I've learned a lot about other Linux distros in the process.
