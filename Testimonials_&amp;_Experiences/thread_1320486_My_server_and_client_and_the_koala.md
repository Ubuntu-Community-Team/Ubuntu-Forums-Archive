### Post by egyn on 2009-11-09
Ok, this is my story.

Our household has got three main computers. One server running Karmic Koala, one netbook also Karmic Koala and one laptop running Windows XP.

_Server_
I recently upgraded the server from 8.04 to 9.10 (64-bit) without major incidents. The server is used as a home theater media pc (HTPC), a HTTP-server and as a backup-server using rsync. The upgrade made the system fall back to analog sound but that was easily solved through the sound preferences where I enabled digital sound once again through the optical cable. I have not tested sound through the HDMI yet so I cannot account for that. 
My girlfriend really likes the fact that she can login to the server from a remote desktop and stream music and video upon demand. We frequently use spotify, rythmbox, svtplay.se and smbplayer for "media consumption". 

_Netbook_
The netbook is a Samsung NC10 which originally ran XP. I haven't used ubuntu on it before because I was afraid that my girlfriend would give complaints. Since she is the one most frequently using communication services such as Skype and MSN Messenger I was a bit worried only relying on linux. Anyway, since she has gotten experience with the ubuntu server I decided to install the latest release to see how things would progress. Now, a well three days into the process I am enjoying the system and she is becoming more and more used to it. However, we both struggle in our minds since it is not possible to switch to the "standard" desktop view. 

_Laptop_
An old acer laptop which runs XP since I've bought Photoshop before and would like to continue using it. Maybe possible to use via wine? I'll check that out right away after writing this letter.

**Summary**
I am very happy to use ubuntu within the household. It works because more and more software is able to integrate with other software and communicate using multiple protocols and open standards. Looking forward into participating in the future of open source software. Might also contribute in the future if I find the time.
