### Post by cariboo on 2009-11-16
I mistakenly bought a Windows only D-Link WUA-2430 usb wireless adapter a couple of months a go, I've tried to get it to work with various distros since, Arch, AntiX, Mepis, Intrepid and Jaunty. Today I had some time to try again, It took less than 5 minutes to get it working using ndiswrapper and the previous Windows driver, the current Windows driver won't work. I tried both drivers at various time and never had any luck on any of the versions I tried it on.

This is on an old ECS K7SEM MB with a 750Mhz Duron and 512Mb ram, The board has an SIS chipset, so I stuck an old Nvidia MX4000 video card in it. Thats the only change to the system I made, everything else worked right out of the box.
