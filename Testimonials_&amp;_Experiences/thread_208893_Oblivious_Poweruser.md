### Post by elemental666 on 2006-07-04
I never new I was a Windows power user until I fully switched to Linux.  I've been a programmer for IBM, independant web developer, 2nd and 3rd tier network support technician and a whole slew of other Help Desk type jobs, all in the Windows world.  I mean I knew my stuff, but when I was coding (not the web stuff) I didn't really have to do much to my machine as our techs rocked.  The other jobs don't require much power really.  So I never considered myself a power user.

I had messed about with linux before at different times but only for very specific purposes and not in production environments (less web servers).  I knew how to get around, write basic shell scripts and configure (but not install or do chroot or anything) mail, web, ftp and ssh server packages, but I still consider that farely basic knowledge (especially for IT professional level knowledge).  

Then I start reading all the horror stories in the news and seeing the special reports on "ciber crimes."  Followed up by the MPA/RAA and a smattering of FCC and software pattent hub bub and I got spooked.  I can't afford the software to do the jobs I do professionally.  To stay up to date and on the edge enough to be profitable I need the best available tools, and I had to get them however I could.  To be completely truthful, I always felt conflicted about that.  I'm sure there are thousands that would agree.  Anyway, this spurred the switch to Linux.  But not the I'll play with linux and continue to use windows switch.  I do not have any propietary software on any of my computers today.

As I'm learning how to accomplish everything in Linux, I'm beggining to realize that I might just have been a windows power user.  Its probably still the new, but I find myself dealing with topics that are considered advanced in most linux circles. I've been using Ubuntu for about 4 months, without windows for 3. While I have most of my tools figured out and working functionally, I don't claim mastery of any of them yet.  The funny thing is while I'm discovering that I was a power user in the windows world, I'm also learning how little control over my computing environment I really had.  I still say that Linux is more complex for the user, but I am now seeing why.  

With more freedom comes more responsability.  Its our job as users to make sure everything is setup like you want it and the options available to you in this platform are better in both quality and quantity.  Minus a few bells and whistles in the Propietary domain everything you need a computer to do for you can be done in Linux and is most likely more customizable.  Then when you realize you can actually edit the code for most of the software, you see that this platform is the only platform that puts YOU in sole command of your environment.  All friendliness aside, Linux profides all the functionality the avg user needs.  All it asks that you get to know it a little better than the upity Propietary guys.

I am once again excited about being an IT Pro.  I'm looking forward to moving into the Open sector.  I am running my Web Consultant jobs on a completely Open platform, as it is.  I should have done this years ago and I'll never look back, because at the end of the day, when you weigh what you do against who you are, Linux just makes more sense.

Shout outs to the Ubuntu developers for putting together a rockin distro, I can't wait to contribute (just gotta figure out the new platform ;) ).
