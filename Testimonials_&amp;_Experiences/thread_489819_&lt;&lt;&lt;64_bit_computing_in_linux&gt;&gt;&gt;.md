### Post by Rhubarb on 2007-07-01
Am running 64bit Ubuntu here.  The nvidia drivers are very quick and easy to get going (from the repositories).
I can get quake 4 running very nicely at an odd resolution of 1680x1050, compiling the sauerbraten FPS was very easy to do.
With a bit of fiddling you can even capture any openGL  game that you can see on the screen (using yukon + seom).
I haven't tried installing UT2004 yet, but I don't think it'd be very hard to install.
The Penumbra: Overture Demo was easy to install.
So was Quake 2 with OpenGL running in wine.

Using mencoder to make x264 videos is noticeibly faster, though for some reason I was unable to use mencoder to encode using xvid in 64bit (the devs may not have compiled it with all the options in the 64bit repository).

You have to mess around a little bit to get Flash working in Firefox aswell.
