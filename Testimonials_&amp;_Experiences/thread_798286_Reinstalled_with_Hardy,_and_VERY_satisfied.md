### Post by christianxxx on 2008-05-18
So I've been haunted with boot problems. As I wrote in a different thread, my computer started to lag and wouldn't boot properly after wrestling with UPnP. I don't know if there's a connection.

After a while, I just gave up, took backup of my /home and reinstalled with Hardy.
Admittedly, I was rather skeptical as I've read the many poor testimonials about Hardy, but I figured it couldn't be worse than a computer not able to boot. So I gave it a try. I'm already running Hardy server without a trace of problems, but as far as I can understand, most of the problems is related to the desktop environment.

The installation was clean and fast, no issues at all. I didn't expect the installation to be able to keep my /home which it didn't either, but as I said I backed it up, so no problem.
70 updated available at first boot was a little strange, being so new release, but a broadband connection makes that a breeze as well.
The part I dreaded was software installation, as I had quite a lot of software installed, and I knew it would take for ever to install everything I wanted/needed/thought would be cool. Again, thanks to broadband and a good movie, no hazzle.

Status quo? Not exactly. Bluetooth works better otb than before, new firefox is stable and nice, my Ralink-based wireless works perfectly otb (if I only knew when I installed the server... ). Flash works fine, and Pulse audio server is currently without problems.
After only 12 hours (too short, I know) the system seems very stable and responsive. I have had no issues at all. No irritations. 

I'm just satisfied and happy. I hate re-installations, but on a rare occasion, it can lead to something good.
But I'll see in few days. Maybe I'll post an update. :)
