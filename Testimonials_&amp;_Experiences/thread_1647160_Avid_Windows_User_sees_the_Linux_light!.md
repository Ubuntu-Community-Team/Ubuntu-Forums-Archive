### Post by ronnielsen1 on 2010-12-17
[http://appdb.winehq.org/objectManager.php?sClass=version&iId=2819](http://appdb.winehq.org/objectManager.php?sClass=version&iId=2819)

> **What works**
Full installation.  

Most things - front end selections, options, default aircraft, airports, sound, effects, Sidewinder joystick, ATC. 

Freeware aircraft - IFDG A321, MelJet 777, Ito Kazunori F-4, Kirk Olsson F-16 etc. 

UK VFR Photographic Scenery. 

**What does not**
Top menu (you get most settings from Start Menu anyway). 

Can't complete the Add-on Scenery process (still trying to load new airports). 

Tried AI traffic - WOAI and PIA and can't see anything yet (also working on this). 

Display hardware selections; stuck on monitor default - but OK at 1920x1080. 


It's rated gold for Ubuntu 10.10 so it should run
