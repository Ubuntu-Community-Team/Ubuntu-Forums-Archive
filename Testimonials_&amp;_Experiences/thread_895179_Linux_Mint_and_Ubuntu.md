### Post by Ryuzakii on 2008-08-20
I genuinely appreciate the time and effort put into Linux Mint, but it's not for me. I've tried each of its releases from 3.0 up to the current version but I always find myself coming back to Ubuntu because I encounter truly bizarre problems in Linux Mint that I've never had to deal with in Ubuntu (or any other distro, for that matter). 

In my most recent Linux Mint-related escapade, I fired up my computer several months ago and discovered that it suddenly wasn't able to boot from any of my hard drives. I decided after some testing that it was a problem with the motherboard and had it sent back to the manufacturer for service, forcing me to downgrade back to my old XP laptop in the mean time. :-x
 
When I got my mobo back a couple of months later and saw that I was still getting the same issue, I decided to try wiping my main drive and putting Ubuntu on it for the heck of it...and the problem went away. I'm still not sure how that all adds up, but I've got a working system and that's all that matters :)
