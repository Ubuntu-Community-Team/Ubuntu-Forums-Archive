### Post by Baldrick_NZ on 2012-03-29
+1 QIII here too.

I think what happens is that people usually learn Windows first, which doesn't do anything in a human-logical sense. We get used to it and it becomes the norm. Then, when we're introduced to Linux (in this case Ubuntu), we have to re-learn a different way. A way that is based more on human logic (the way we perceive things should happen).
