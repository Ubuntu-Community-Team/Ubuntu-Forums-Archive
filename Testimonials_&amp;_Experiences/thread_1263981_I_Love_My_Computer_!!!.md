### Post by AgentZ86 on 2009-09-11
I have been using Ubuntu for a few years now

I can only say that there is a process in which learning the way Linux feels and works is no different then the process of learning how Windows or Mac works.

I'm so glad I decided to do that, and challenge myself.

I really enjoy my computing, which I never really did with Windows.

I would probably have enjoyed Windows and kept using it if it were not for all the problems and freezes, lockups, viruses, spyware etc.

Although there are few things that I wanted to do on Windows I can live without it and am glad I did.

I remember the days of just spending hours tweeking things and trying to get Windows to work correctly to play a stupid game I wanted to play.

I see posts about Linux not being compatible with Windows and that they want games etc. I can't help but wonder if it could be said that Windows is not even compatible with Windows.
I mean every stinking game I ever played either cause some problem with the computer, or just didn't work right without locking up.

Even from fresh install which was totally annoying to re-install my whole operating system just to play a game and because people would recommend this from time to time and which never worked anyhow.

I'm just reflecting back on the years of enjoyment I've had with my Linux Computers and Servers etc.

And figured I would post since I've not visited the forums in a while

Anyhow my slogan would be that (Linux IS = Pure Computing Pleasure)

Happy Computing

### Post by Zoot7 on 2009-09-18
Ditto that, I've enjoyed my computing a huge amount more since i've been using Ubuntu as my main OS.

I had OpenSUSE forced upon me in college, back when I started writing programs in C. I started dabbling with it on my machine, and started messing with other Distros such as Fedora, Knoppix and Slackware, but settled on Ubuntu because of the support. Started using it back with Feitsy and haven't looked back since.
The only issues I've had are Video driver related (only with nVidia, not ATi mind you).

I still dual boot with XP for games, Home Recording software and some Engineering apps I need for both work and studies, but 90% of my PC usage these days is Ubuntu. Never will Windows be my main OS again.

The sheer freedom, versatility, and robustness of Linux are what makes it for me. Windows just can't compete with that.
The most recent thing that affirmed that for me was having Ubuntu boot up perfectly after swapping the hard drive out of one totally different machine into another.
