### Post by tribaal on 2007-07-23
Hi all

Just a little post to relate my experience this weekend:

I had to buy a shiny new laptop a few weeks ago, and decided to go for an Ubuntu 64bits installation. Yesterday I decided to install World of Warcraft on it, not because I'm such a big gamer, but just to have something to fix on my laptop (computers are a little boring when they work as intended, so I like to break stuff up or install improbable software to get my daily CLI fix).

I'm disapointed.

Apart from entering the wineHQ repositories for the wine amd64 debs, I didn't have to use my dear CLI a single time. Installing wine was a breeze, and the World of Warcraft installer works "out of the box", by double clicking the .exe in nautilus.
The only painful part was downloading all the patches (no setup required for their bittorrent client, save for the router ports forwarding), and the game runs like a charm.

Please devs, think of all the frustrated CLI geeks out there, make Ubuntu harder :)


- trib'

PS: Please please please, I don't want to start a flamewar on how I could use Gentoo or FreeBSD instead, I mean this as a funny post :)
