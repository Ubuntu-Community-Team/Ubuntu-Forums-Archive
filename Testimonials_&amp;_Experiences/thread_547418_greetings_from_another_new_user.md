### Post by Monstroxus on 2007-09-10
I really liked your story, its fun isn't it? To make it even more fun... install Kiba Dock. Its a very nifty program launcher... you can basically drag and drop icons of programs that you like to use, into a launcher. Which will sit at the bottom, top, left, right, anywhere you want, you can all change the settings in its configuration options. These icons can blink, jump, rotate, jump etc etc. Its all up to you!... install is easy... you can find the program in Synaptic... (add/remove... the extensive one, that's without the ratings/stars one) search for kiba... install have fun! (play around with the settings first so you will get rid of that ugly blue area around the icons) You must be running Compiz Fusion in order to run this program nicely. Its great fun and very helpful too!
