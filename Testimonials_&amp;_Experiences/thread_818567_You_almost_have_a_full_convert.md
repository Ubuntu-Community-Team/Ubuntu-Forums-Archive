### Post by Aglarond on 2008-06-04
I'm a long-time Gentoo user who's tried every Ubuntu release since 4.10. The installs usually last a week or two, then I'm back to Gentoo either because it's too much trouble to resolve dependencies of software I need that's not in the repositories or I hit some feature (bug?) that drives me up the wall. I usually install it a month or two after the release to let some of the issues work themselves out. Hardy's been a big exception, though.

I loaded it cleanly on an old Dell C600 and everything but my wireless connection worked (D-Link DWA-130). A quick load of ndiswrapper and the drivers off the cd fixed that and I was up and running. This is the first Ubuntu release where I didn't have to get an update ndiswrapper from the developers and build it all by hand. I loaded Compiz Fusion just to see if it would work on a machine this old and, surprisingly, it did. I didn't leave it on since I don't really have the resources to run it really well, but I was impressed that it worked at all.

Because that went so well I tried loading it on my IBM Thinkpad T42. Almost everything, including the wireless, worked on the first boot. The only exception was the Sprint EVDO card. I unloaded the airprime module and it started working. So, blacklisting fixed that. I thought I was having trouble with the sound, but it turns out it's just quiet. Compiz Fusion works great on this machine and installed with no issues at all. This is also the first release where I haven't had to fiddle with xorg.conf.

The only thing that doesn't work every time on both of them is the suspend option, but I never got that working properly under Gentoo either, so I can't really complain about that here.

I still haven't converted my desktop to Ubuntu and I don't see changing my servers at work from Gentoo, but in the meantime I definitely have a new laptop distro.

So, with all of the complaints I've read in here, I just wanted to say great job to the developers. This has been the first Ubuntu release to really impress me and it's the first one that'll actually stay on the machines I loaded it on.

-Ag
