### Post by Welly Wu on 2012-12-06
So, I decided to upgrade to Ubuntu 12.10 64 bit Quantal Quetzal tonight. I wanted the the cloud integration features like online accounts, Ubuntu Unity Web Apps Desktop Integration, Ubuntu One synchronization and sharing, Gwibber, Amazon searches, etc. It's okay so far. I like how everything is connected to the cloud securely and safely. It feels like my cloud accounts are more like my locally installed desktop software applications. It's definitely cool.

I feel that Ubuntu 12.10 64 bit is a bit slower than Ubuntu 12.04.1 64 bit LTS. It's not that much of a difference, but it is noticeable. I like the new Ubuntu Unity desktop environment. It's got a lot of new features like previews and there are new lenses that make it able to integrate local files with cloud files and accounts. It's really nice. I thought that these new cloud features and integration was just a gimmick, but it's for real and it's cool to have. I like it and it's growing on me.

Ubuntu is a good GNU/Linux distribution and I can easily see why so many people choose to use it. It makes GNU/Linux easy for humans to understand how to use. It's no longer something to be feared because it's too difficult to learn how to use.

I'm looking forward to the future of Ubuntu. Long live Ubuntu!
