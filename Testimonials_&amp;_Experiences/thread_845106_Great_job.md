### Post by wirepuller134 on 2008-06-30
In the past I had played with Linux but wasn't really impressed with it for desktop applications.  Then I heard about this project, after doing a bit of research I have to say I am impressed.  I took an older box and installed Ubuntu.  The entire install process took approximately 35 minutes.  Every driver was installed and it simply worked. We then proceeded to start a 2 week test for stability.  Ubuntu with every patch gets better and better.  With Wine we are running most every windows based application that we need to run.  We have replaced our Isa server with the server edition, with great results.  The only programs I cannot run on Ubuntu are rslogics500/5000 so fourth as it creates a safety concern.  So now 1 month later.....the entire network is running Ubuntu, save one laptop for programming.  Thank you everyone whom has contributed to this project, you just made my life a lot easier.  As my 12 year old put it for a home machine "I prefer my computer like this because everything just works".  Kudos and keep up the good work, your efforts are appreciated.
