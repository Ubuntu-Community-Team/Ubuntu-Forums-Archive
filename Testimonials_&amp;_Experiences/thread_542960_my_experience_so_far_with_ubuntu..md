### Post by terry_gardener on 2007-09-04
i have used ubuntu for about a month and loving it. 

I have a dual boot with windows vista Home Premium. 

I had never used or even seen linux before using ubuntu so am a complete new user. The install including the partitioning to leave windows alone was easy. 7 windows just follow the instructions, about 25 minutes later it was installed. so i started navigating through the menu's to fine out where things where and thought that this is well organised i seemed to be in a logical place and felt natural. icons started to popup in the system tray so hovered mouse over the icons and it said what it was.  installed the updates and then graphics drivers with envy. Yes there is a learning curve for this OS but it is very easy to use and you dont have to use the terminal with you dont want to. for some fix resolutions it tells you step by step what to type in the terminal so it is still very if you follow the instruction. 

i had been wanting to try linux for a long time but never got around to it but when i seem a video on youtube vista vs ubuntu i thought i would give it ago as i thought it looked really good. 

over the past month i have had a couple of problems and questions which have all beeen answered by the users of this forum and very very quickly indeed which was impressive. The problems that i have encountered are: 

1. Xserver not started. 
2. permissions problem with home folder 
3. missing title bars and panels icons not loading properly. 

number 1 happened when i installed the driver before doing all the updates and then updated and it updated the kernel and got this message. 

number 2 happened because of my fault as i changed the premissions to test something but didn't change them back afterwards and restarted got message. 

number 3 happens mainly when i have booted into windows and when i next access ubuntu it happens dont know why but easily fixed by right clicking the beryl icon and clicking the reload decorator, and reload panel icon on panel. 

i have had a good play about the system and in my opinion it is better than any windows os, very customisable. 

i am just a casual user really only use pc for internet, email music, movies, downloading etc etc, i dont play many games on pc. 

my sister had a problem with windows last night and she had to install windows after windows update corrupted the windows. i was round for a couple of hours helping. she started installing it about 6.30 and it finsihed installing and all updates about 8.45 she still had all the software/drivers to install including the essentials Anti virus and firewall. it updated with about 130 packages and needed about 6-7 restarts as it would not install everything at the same time and do one restart properly would have died if it did. 

to get a pc up and running with linux ubuntu is far easier than windows and takes no where near the length of time as well. 

i am going to stick with the dual boot for the moment because i use tversity to streams/transcodes videos to the ps3 there is so pose to be a linux version coming out when it reaches version 1.0, when released i'll properly go with ubuntu 100%. 

ubuntu starts up quicker as well timed it with vista home premium, time taken to get to desktop and load all system tray icons and software and at usable desktop state, both with autologins, times from pressing power button to desktop usable state 

ubuntu 1min 15 secs
vista 2min 10 secs 

sorry for such a long post but wanted to share my experience.
