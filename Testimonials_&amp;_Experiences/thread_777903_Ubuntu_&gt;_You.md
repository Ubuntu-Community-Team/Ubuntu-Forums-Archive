### Post by Exsecrabilus on 2008-05-01
When Ubuntu 8.04 Release Candidate came out, I decided to get it to try it out to familiarize my experience with Ubuntu 8.04.

At that same time, I had been running Ubuntu 7.10 i386, but I recently had learned that my laptop was capable of running AMD64 versions.

So I downloaded the .iso of 8.04 amd64 RC, burned it to a disc and did a fresh install. (Since I couldn't upgrade to an AMD64 version from an i386 version.)

I loved the desktop and everything, but I had some newb issues.

I installed *bcm43xx-fwcutter* to enable WLAN to connect to the Internet, however, was saddened to know that I couldn't fetch the firmware; it wouldn't let me.

So I booted into the Live CD of Ubuntu 7.10, and did all that so I could connect to the Internet.

I then went to Ubuntu Forums and posted for help. It was then I became aware that I had to use *b43-fwcutter*, not *bcm43xx-fwcutter*.

So I downloaded the .deb package from [http://packages.ubuntu.com/](http://packages.ubuntu.com/) and rebooted into 8.04.

I installed the package but was still disheartened because for some reason, I couldn't fetch *wl_apsta-3.130.20.0.o* from the application *Hardware Drivers*.

So I rebooted into my Live CD again and posted for help. I learned I had to open up Terminal and do a manual install. Hmph, what?

But whatever, I did it, and soon, I was connected to the Internet. I was happy.

Then it hit me: Firefox 3 Beta 5 wouldn't work. It won't load any text for any pages, and wouldn't download files properly. I tried Firefox 2, but that crashed.....like all the time.

After a few days I tried running Firefox 3 Beta 5 from the command-line (Terminal); it worked. And although it loaded pages, it still wouldn't download properly. But I didn't care, it loaded pages, and that's all I needed.

There's more. There were two annoying things about running Firefox from the command-line:
One was that because I open Firefox from the command line, it didn't recognize the default theme and it made the page look ugly.
Secondly, if I closed the Terminal I used to open Firefox with, Firefox closed too.

But I didn't care, I lived with it. The saddest part was, after a few days, I was told I wouldn't be able to upgrade to final release of Hardy Heron, since "if you install all the updates on a RC, you'll basically be running 8.04 final version."

I was a fool to believe that. After a few days I got sick and tired of this Firefox bug and a few other stuff.

So I downloaded the .iso file, burned it to a DVD, transported all my files to a USB stick, and did a clean install of Ubuntu 8.04 amd64.

And guess what? All my problems vanished. When I opened up Firefox, text loaded and everything "just worked."

Ubuntu is the best operating system I've ever run in my life. To me, it's even better than Mac OS X Leopard (claimed to be the most stable operating system in the world), since you can actually customize it, there are free applications, and the people here are nice.
