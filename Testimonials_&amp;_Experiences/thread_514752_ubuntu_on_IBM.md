### Post by bob-r on 2007-08-01
I recently installed Ubuntu 7.04 server edition as a LAMP server.

My Machine: IBM Eserver Xseries 330 (Type 8674-41X) Server

Hardware Specifications:

&#8226;	Dual Pentium III 1.4Ghz Processors
&#8226;	1,280 MB Memory (One 1GB and One 256MB Stick)
&#8226;	Intel PRO/1000 XT Network Adapter Card
&#8226;	IBM ServeRAID 4LX Ultra160 SCSI Adapter
&#8226;	Dual Hot-Swappable 36.4GB 15K RPM SCSI Hard Drives
&#8226;	Raid-1 Hard Drive Configuration
&#8226;	CD-ROM
&#8226;	Floppy Drive

Initially, the installation kept hanging due to problems accessing the CD-ROM Drive; an H-L Data Storage Model CRN-8245B. I replaced the drive with a Samsung Model SN-124 and received the same symptoms. Finally, I installed a Toshiba Model SD-R2002 drive and the installation completed successfully. Ironically, I re-installed the original CD-ROM drive after the installation and have had no problems accessing it.

I installed the Gnome desktop without problems. I&#8217;m no Linux programming genius; for now, I need a graphical unit interface to help me along! lol

I installed an HP Deskjet 3940 printer using the new printer wizard and was happy to see the exact driver for it was available. The printer installed effortlessly and works like a charm.

My ultimate goal is to host my personal website on the server. I&#8217;m looking forward to learning more about Ubuntu; so far, I&#8217;m impressed! Not to mention Ubuntu is in my price range; free! lol
