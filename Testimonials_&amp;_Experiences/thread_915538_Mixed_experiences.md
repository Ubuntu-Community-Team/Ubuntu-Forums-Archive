### Post by meepster on 2008-09-09
I am a law student, so the demands I place on my computer are pretty minimal.  All I need from it is a web browser, a word processor, and ... well, that's about it.  (well, there's all the MIDI music stuff I do in my spare time, but that's another story)

So, when it was time for a new computer, I decided to take the plunge and upgrade my old computer with a dual-boot Linux/XP installation.  I've always been a bit scared of Linux - it seemed to me that you needed to be very knowledgeable in order to be able to use it.  I am rather clueless about computers, and I tend to break computers just by looking at them, so I'd resisted.

Well, I am now spending most of my time in Linux rather than Windows.  The total tally of problems I'm having is probably higher in Linux than in Windows - in the 2 weeks I've used this computer, I managed to break the keyboard mapping somehow (got it fixed, thank God - it would have been very hard to write this message without using the letter "i"), I still cannot connect to my MIDI keyboard, and I still cannot run Java applets in Firefox.  On the other hand, in Windows I cannot connect my wireless network card to the school network, and the IT tech spent 2 hours on it with no success.  In Linux it works flawlessly.  

As a future intellectual property lawyer, I've always been interested in open source software, in a theoretical sort of way - it's great to see it working so well in real life.

### Post by lswest on 2008-09-10
If you're having trouble with firefox and Java, make sure you've installed sun-java6-jre, or one of the open-source versions.  (I think sun-java6-jre is in the ubuntu-restricted-extras package if you haven't installed that yet either).

Installing can be done from the terminal with:
```
sudo apt-get install sun-java6-jre #(or ubuntu-restricted-extras if you want mscorefonts and mp3 support, etc.)
``` or else use Add/Remove or Synaptic (under Accessories and System-->administration-->Synaptic Package Manager respectively).
