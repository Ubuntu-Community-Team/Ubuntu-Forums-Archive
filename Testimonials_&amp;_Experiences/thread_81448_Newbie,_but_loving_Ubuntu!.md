Hey all,

I'm the newbie who posted about the USB modem.  I have learned a lot about Ubuntu and linux since that first post and I love my experience!

Ubuntu has been the most fun I've had with a computer since I got my first Windows machine (in 2000, running Win98SE).  I've been having a great time configuring, learning the new way of doing things, reading the forums and using the update feature.  

What suprises me the most is the sheer number of different programs that are equal to or better than the Windows counterparts.  I like the cool little games provided with Ubuntu, and am looking forward to getting the Civ-styled game in the future.

I have booted into Windows once in the past three days, and that was to run a game that I wanted to play.  Other than that, I can accomplish everything I usually do under Windows with Ubuntu.  The only thing I wish would work is Trillian, but I'm using Gaim and adjusting just fine.  (The only reason I wish it would work is because I paid for it...  Yeah I know, paying for stuff, but I liked the program and wanted all of the plugins!)
