### Post by Bucky Ball on 2011-01-07
System->Administration->Synaptic Package Manager. Search for 'ubuntu-restricted-extras'. Install.

System->Administration->Additional Drivers. Do you have anything in there?

Only do this next section if you are using 10.04 LTS for now. Upgrading the kernel to 2.6.35-24 in 10.10 could break your system at the moment. If you are using 10.10 and that kernel already then go ahead with this section:

System->Administration->Update Manager. Check for updates and install if any. 

If you're up to this, Applications->Accessories->Terminal and type in:

```
sudo apt-get upgrade
```This will NOT upgrade the OS, only packages that need upgrading.
