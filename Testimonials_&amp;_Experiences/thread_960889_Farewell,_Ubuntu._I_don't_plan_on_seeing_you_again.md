### Post by POW R TOC H on 2008-10-27
I haven't been using Ubuntu for a few months now. I decided to replace it with another linux distro, so after a few tries, I settled on Zenwalk. Why? Well, let's set aside that Ubuntu is becoming bloated, so bloated in fact that it takes 3-4 seconds to run gedit (and yes, that is a lot). 10+ to run firefox. A minute or so to boot.
The real reason I started to dislike Ubuntu (not to say I hate it, but hearing the name makes me twitch) is because of all the idiots who act like they own the world for using it. They've used it for a week, and suddenly they know all there is to know about it, and feel free to insult windows users or talk about how 'geeky they are, and they admit it'. Furthermore, there are people who treat ubuntu as the only linux distro in the universe (like linux is ubuntu, not the other way around). I saw millions of pages that have articles like "[standard unix utility] tutorial for Ubuntu". Well guess what, cp/awk/gcc/apt/sed is NOT Ubuntu-only!

I don't like windows. I didn't like it even when I used it. But I still don't bash it, or treat others who use it like idiots. Things like [this]("http://www.fayerwayer.com/up/2007/04/bill_gates_ubuntu_mini.jpg") make me want to puke. Then I hear something like "Ubuntu is a way of life!!11!". No, idiot, it's an operating system. You're not a "geek" for using linux, you're just a linux user. 

There you go. I hate Ubuntu. It's a fine distro (a bit slow) but I can't stand the fanboyism around it.
