### Post by recluce on 2012-01-04
You may want to try Cinnamon. Most of the look and feel of Gnome 2, but based on Gnome 3. Very promising, even in its early alpha stages.

.deb files available here: [https://github.com/linuxmint/Cinnamon/downloads](https://github.com/linuxmint/Cinnamon/downloads)

These .debs should install on Oneiric, you need "cinnamon" and "cinnamon-session", current version is 1.1.3. You need a Gnome 3 base, so do not try this on Natty or earlier! 

Or you could try Mint 12 (Ubuntu-based) and choose from Gnome3/Gnome Shell, Gnome3/MGSE, Gnome Classic and MATE (Gnome 2 port). Basically everything except for Unity.

### Post by HeroOfCanton on 2012-01-06

I can only speak for myself, but as someone who knows their way around a computer Unity is awful. I am looking to run multiple workspaces while programming switching from editors to outputs to graphics programs to browsers, etc. I don't want to have to type in the search bar every time a program is needed. I don't want the OS telling me how to size and place my windows. Don't want giant buttons (more mouse movement required). Simple is only good when you're not doing complicated things.

For a new or "everyday" computer user Unity seems great. In fact, it's still on my media box hooked to the TV (Xubuntu on everything else). For browsing and basic day to day functions Unity is fine. Not so much for "power users".

I think you see so many complaints because most linux users are in-depth techs who value power over simplicity. Just my 2 cents.

Hello HeroOfCanton and all

I do that all the time and I'm using Unity2d (graphics card won't run Compiz). I tried Unity2d out for a month just to see what all the fuss was about, and within a week or so I had sort of forgotten about it.


> **HeroOfCanton said:**
> I don't want to have to type in the search bar every time a program is needed. 

I don't. I just add the common ones to Launcher and then I load the other ones as I need them and leave them running for the session. I have around 7 to 10 applications running over the four workspaces depending on what I am doing. I use 'hibernate' instead of shutting down so I can keep the workspace layouts, and I tend to use the keyboard short-cuts a lot for workspace and window switching. I don't like the global menu on a large monitor and I hope there will be an option in settings about this in future versions.

I'm also trying out Gnome Shell for a month to see what all the fuss is about there as well, Clutter runs fine on this box. Gnome classic (no effects) works really well.  I should say my 'normal' interface is dwm/dmenu, so I'm pretty odd anyway.

As others have said, there is *plenty of choice* both within the Ubuntu repositories and looking further afield at other distributions. **That is the great thing about GNU/Linux - you can shape the environment to your needs, rather than shaping yourself to what Cuppertino or Redmond think you need.** So just build something that works for you.
