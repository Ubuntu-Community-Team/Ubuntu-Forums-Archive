### Post by charlie. on 2006-04-24
Installing a Linux Intranet server in my office has been my goal for quite some time. I downloaded the Ubuntu 6.06 “Dapper Drake” Beta, dated 2006-04-20, installation CD. Here are my thoughts so far:

The Ubuntu server installer is very quick and easy. I particularly liked the option of Reiser as a file system. Additionally, the reasonable timeout and ability to rerun the DHCP broadcast is a very nice feature that I have not seen before. I plugged in a dud cable and retrying saved me.

After restarting and logging into the new server, one notices exactly what the installer actually installed: not very much. Yes, that's right, if you want it, you must install it. I like this idea. This is how a server operating system installer should behave.

First thing I did was install the openSSH server with 'sudo apt-get install openssh-server' so that I could go and stick the box in our cabinet and proceed to work remotely. This installation proceeded flawlessly, prompting for the CD as a installation source and starting the SSH server once completed. I immediately launched putty on my windows workstation and connected with no problems! (Putty even supports colours!)

I have yet to install Apache, MySql and PHP.  So far, Ubuntu impresses me more than any other Linux distribution.
