it's ok though. i can boot xubuntu and run mostly from ram. it's relatively smooth with a minor hicup every so often.. i even installed xmms so i can listen to cool idm music from shoutcast while i post this. i'm tickled that i actually installed a program without having a hard drive. 

amazing. i think i'll write a song about it on my air guitar.
