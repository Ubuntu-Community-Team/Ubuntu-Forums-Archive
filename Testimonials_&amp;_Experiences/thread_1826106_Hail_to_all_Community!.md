### Post by sanderd17 on 2011-08-16
Thanks for your kind words, but Ubuntu is certainly not the only good distro around. It is the easiest though.

And if I'm allowed to be negative, Linux will indeed win the overall game. OSX and Windows will go down rather fast, but it's not GNU/Linux that will win the game. It's going to be OS like Android/Linux or Meego/Linux or maybe even Chrome/Linux. The desktop computer will go down (they will still be around, but the number of other devices will be a lot bigger) and the little devices in our house will run on Linux.

This is now already the case. If you have a GPS like a Garmin or a TomTom, than there is a big chance that it runs on Linux. If you have a smartphone, apart from an iPhone or Windows phone, than the kernel is most likely Linux. If you have a router at home, it is probably running Linux ...

But you were talking about GNU/Linux distros I believe. 

In that view, I believe Unity has some design mistakes. It is usable, but not as great as it looks. The most things in Unity are difficult to adapt, and it uses a lot of unnecessary hiding. 

I've also used Ubuntu since 8.04, and I always returned to Ubuntu with Gnome, but I could not live with Unity. Although the difference between Gnome-Shell and Gnome2 is bigger than the difference between Unity and Gnome2, I do prefer Gnome-Shell.

When you first start Gnome-Shell, you have one button to get in the application-management screen, the activities button. Apart from that, you have some indicators like the volume, battery and network connection and a user menu where you can find the system settings and the buttons to log-out and put your computer off. You don't get to see your notifications (they can be seen on the activities screen). So the beginning is very minimal (but functional if you like to use hotkeys). 

But the big power of Gnome-Shell is the fact that you have a large amount of extensions (and that amount is still growing). You can add other applets to the main bar, you can add a dock to your normal screen, you can add the old gnome menu, you can move the notifications to where they were ...

Everything is written in JavaScript with CSS, so there is a huge amount of web developers that can develop themes or extensions for Gnome-Shell. And if you don't like one color of a theme, just go in the CSS file and adapt it.

Right now, I'm on an Arch installation with Gnome-Shell, and I don't think it will change any time soon.
