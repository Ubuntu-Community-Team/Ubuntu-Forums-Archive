### Post by Riffer on 2010-03-09
I've been using Ubuntu since 7.04 and I've always heard the horror stories about getting wi-fi to work.  So it was with some trepidation that I installed 9.10 on an old Seanix laptop, which in order for us to use it for what we wanted wi-fi had to work.  

Using a livecd and checking that everything was working, I checked the network manager app.  It had not only found my wi-fi network, it was dead easy to setup for wep security (a lot easier then XP) and after the install all settings remained.

That was 2 months ago and its still running rock solid, which again is something I can't say about XP.  With 2 other laptops running XP in the house, both have constant problems with drop-off and connection.

Well done Ubuntu.

P.S.  I've found 9.10 to be the best release yet and am looking forward to 10.04.
