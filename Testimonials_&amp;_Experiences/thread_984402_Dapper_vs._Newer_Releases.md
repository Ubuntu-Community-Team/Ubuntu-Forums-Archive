### Post by zero244 on 2008-11-16
I used Dapper for a couple of months just before Edgy came out. Ive been using Edgy on my main computer ever since.
I use Feisty on the other computer in the house mainly because of better wireless support than Edgy.
To be honest I dont plan on upgrading Edgy till I just have to.
One problem with Edgy there is no repository support anymore. Which can be a problem if you want to use some newer software.
I dont really worry about security updates since Linux is pretty secure right out of the box.
Its too bad Ubuntu cant keep the old repositories online. But I understand they can only support releases for so long and have to move on.
At any rate Ubuntu and Linux in general is a very good OS. Much more user friendly than years past.
There is nothing wrong with Dapper, if everything works to your liking keep on using it.
I was in my doctors office and he had a ultra-sound machine that was still using Windows 98.
