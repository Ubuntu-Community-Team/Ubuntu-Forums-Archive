I guess I didn't really address you're real concern in regards to having to start fresh. 

My main advise would be always create the partitions you feel are necessary to you. 
In my case /boot (since I multi boot) /home (since I always do clean installs) beside that I got a partition dedicated to vmware and iso's, then last but not least a backup partition. 

If I at any point do an fresh install that I feel should include the /boot and /home partitions for whatever reason I simply back them up on the /backup partition. Once I am done I place whatever kernels I need back to the /boot partition and any config files from /home that I find useful to my new home.
Of course you can likewise backup files from you're /(root) partition if the need be. 

All this takes is that you are prepared, then I find that I am able to do a clean install in something like 3 hours and be fully back where I left off, that includes coffee and cookies. 

I only have to do this every 6 moths or so I don't view it as any big deal. 

Hope it helps. :-)
