### Post by texas.chef94 on 2008-07-20
In an effort to avoid partition decisions and somewhat fearfull of gparted I choose the wubi installer option to load 8.04.
While it gave me total unrestricted use of ubuntu it had its own unique concerns which forum has had the patience to contend with my somewhat vague questions.
The beginner with clicking on install in this option really thinks they are home free as the only decision is password. It loads ubuntu as it would any other windows file. You go to control pane+add-remove and there it is with all other window programs. However you seasoned users realize windows is not a sharing format and is extremely biased as evidenced by by 0x7 partition questions and windows repeated focus of same at boot. I was fortunate enough to get help at User Group in Dallas which uninstalled the wubi ubuntu and reinstalled 8.04 with partitions.
Please excuse this length, but I just wanted the new user to be aware of the difference in the wubi way and the tried and true installation. Only way to go from my perspective. Thanks to Forum and thanks to Nort Texas User Group.
