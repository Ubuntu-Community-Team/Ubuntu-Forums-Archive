### Post by Welly Wu on 2012-01-22
I own an ASUS N61JV-X2 notebook PC which I purchased in late August 2010 from Amazon. I upgraded it to include Crucial 8 gigabytes of DDR3 1,066 MHz SDRAM and an Intel 2nd Generation MLC NAND FLASH 34nm X25-M 160 gigabyte Solid State Drive. I installed Ubuntu 11.10 64 bit. I finally got everything to work properly. I installed Ironhide since my notebook PC comes with a Nvidia GeForce GT 325M GPU and Optimus technology. I especially like the Unity 3D interface. It is so simple to use and it looks real pretty on the desktop. Basically, it lets the software packages and applications shine by getting out of the way literally. I also like the new ALT + TAB graphical switcher. It is very pretty as well. It allows me to quickly switch between running software applications and even select which window I want to see if I am running multiple windows within one software application. I think that is pretty darn handy. One of the best things that I like about Ubuntu is how stable and reliable it really is. I used to use Microsoft Windows 7 Ultimate 64 bit and the Reliability Monitor told me that I scored a 1 out of 10 which meant that my Windows operating system was unstable and unreliable. It cost me over one thousand dollars in terms of software applications that I bought just to make it function correctly. I had to buy every niche software application to make Windows 7 work properly that I literally ran out of money to buy more.

I am on a limited fixed income monthly budget. I just can not afford Microsoft products anymore. I am going to become a full-time graduate student pursuing a masters degree this fall 2012 semester so my expenses will be high and my budget will be very limited. My graduate school and degree program do not care which operating system I use so long as I turn in typed papers on time and I show up for quizzes and exams.

Ubuntu saved me a lot of time and money and it has been almost two weeks so far. Last night, I upgraded from Ubuntu 10.04.3 64 bit LTS to 11.10 64 bit in three easy upgrades without one single problem. I can not say that Microsoft Windows will allow me to do that. It cost me nothing except 90 minutes of my time. By upgrading to the latest version of Ubuntu, it solved all of my hardware and software problems.

Ubuntu 11.10 64 bit is extremely fast on my computer. I can cold boot in less than 10 seconds and shut down in less than 14 seconds. I can reboot my computer in less than 8 seconds. I setup a custom installation with LUKS/LVM full disk encryption using AES-CBC mode 256 bits with SHA-512 hash algorithm and the speed and performance of Ubuntu seems to be no different than running a plain vanilla installation without any encryption. It is really that fast! AES encryption is unbroken and no shortcut method has been found other than brute force method.

Finally, I love the fact that Ubuntu is so portable. I can install Ubuntu on virtually any computer and it will work with few issues. All of my data is secured by TrueCrypt using AES-XTS mode 256 bits with SHA-512 hash algorithm. I formatted my Intel SSD, Seagate FreeAgent Desk 1.5 terabyte external hard disk drive, and Kingston DataTraveler HyperX 128 gigabyte USB 3 thumb drive to the GNU/Linux /ext4 file system. I love the fact that very few people use GNU/Linux in the general population which means that my data is unreadable by their Windows or Mac operating systems.

Ubuntu is extremely secure. I studied for my CompTIA Security+, CISSP, and other vendor specific security IT certifications and I took a course in operating systems. I studied Ubuntu's architecture which is based largely on Debian Testing and I deem it to be secure by design and secure by default and secure in deployment. I like the fact that all of the key and non essential parts about Ubuntu are loosely decoupled which prevents that avenue of attack vectors and the usage of AppArmor along with the limited user privileges in the standard user profile. Canonical does an excellent job of keeping Ubuntu safe and secure to use for hundreds of millions of Ubuntu users worldwide and they proudly highlight some of its security features and mechanisms which do not require non technical people to understand how to use in order to use their Ubuntu computers. I feel a lot safer and more secure using Ubuntu than Microsoft Windows by a long shot. I just dramatically decreased my likelihood of being attacked successfully by a large margin by switching.

I will stick with the latest version of Ubuntu 64 bit GNU/Linux. I have no real need for Microsoft products anymore including Windows or Office. They are much too expensive for me to be able to afford until I land a full-time job in my chosen career path as an English professor.

Ubuntu is a joy to use and it is so simple to learn how to use it and develop real mastery and confidence in your technical skills. It is really not that hard to install and use at all.

Most people have not heard about Ubuntu or Linux in my hometown. I do not plan to evangelize Ubuntu because I do not want to attract attention to myself among my family members and friends, but I will praise it when it comes up for discussion.

For me, the best single thing about Ubuntu is that it is always available for free of charge. I get a very high quality operating system and software packages at absolutely no cost. It gets the job done for me regardless of which specific task that I need it to do for general purpose computing or entertainment needs.

Thanks for reading. If you have something to say in reply, then I would look forward to reading your replies.

The other GNU/Linux operating systems are nice such as Red Hat Fedora or OpenSUSE, but I choose to stick with Ubuntu because it is so easy and user friendly.

I plan to buy a System76 Bonobo Professional with the highest end hardware options available by April 31st, 2015. It will be a desktop replacement notebook PC running the latest version of Ubuntu 64 bit and I plan to spend up to $5,050.00 USD including shipping and a 3 year warranty. That's how much I love computers and Ubuntu!
