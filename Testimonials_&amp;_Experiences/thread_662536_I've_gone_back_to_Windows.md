### Post by fiddledd on 2008-01-09
Warning longish post!!!

Well I've used Ubuntu for over a year now and I've overcome all of the problems I've had and have a Ubuntu system that does all I want it to do. Unfortuneatly my Wife and Son can't overcome their problems with Ubuntu without my help, so they've gone back to Windows. These problems include playing games and hardware that I can't get to work (USB Webcams, USB Framegrabbers etc). Some games work fine under Wine, alot don't, the Hardware problems are hit and miss some things I've solved, some I can't. None of these problems are faults in Ubuntu. The Games developers can't make a profit from Linux, well maybe they can but they can make a hell of a lot more from Windows. The Hardware manufacturers don't much care about suporting Linux, again, there's no profit. IMHO no Linux based Desktop OS will ever become "Mainstream" untill the slogan "It just works" is true but I believe most Linux users don't care about that, they just want an OS that's free and works for them, which is what they have already with Ubuntu. So why have I gone baack to Windows as well?. I need to remain familar with Windows, with the security risks, the bugs etc and the best way to do that is to use Windows regularly, Dual booting and using Windows occaisonally isn't enough. I will still check these forums and still help if I can because these forums are one of the great things about Ubuntu. But, for the foreseeable  future I'll be using XP (LUA, Freeware Firewall, Freeware on demand Antivirus), I've never had a Virus and never been hacked (using computers since Commodore 16 more than 20 years ago), lets hope that don't change. :) (Sorry about the spelling mistakes I'm not awake properly and the spellcheck isn't working)
