### Post by aysiu on 2008-07-18
> **sink said:**
> All my hardware work out of the box while useing windows xp I need to install many drivers, why?
The drivers are all in the Linux kernel.

Windows leaves the hardware "detection" up to the hardware suppliers (who have to include a Windows driver CD the user has to have to install the drivers).

On the one hand, this makes Ubuntu a lot more convenient if you have Linux-compatible hardware.

On the other hand, some hardware is not Linux-friendly, and knowing that a hardware manufacturer-provided Windows driver will get things working even if they don't at first gives Windows users more flexibility in terms of what hardware they choose.
