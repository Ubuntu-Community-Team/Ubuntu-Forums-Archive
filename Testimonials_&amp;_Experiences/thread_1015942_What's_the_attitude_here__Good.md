### Post by capscrew on 2008-12-19
@RRCampbell12,

As to your last post:

> although I can go into Network Configuration and make what seem to be the necessary changes, the data is not saved.

This is currently a very hot topic.  There are a few things at play here.    Network Manager is in a state of flux.  The underlying code has changed with every release.  Gutsy, Hardy and Intrepid releases are similar but different in critical ways.  For a good article on the current state of affairs see:
[[COLOR="Blue"]http://www.arachnoid.com/linux/NetworkManager/index.html[/COLOR]]("http://www.arachnoid.com/linux/NetworkManager/index.html") 

Thanks to bab1 for posting this URL at:
[[COLOR="DarkSlateGray"]**http://ubuntuforums.org/showthread.php?t=1004752**[/COLOR]]("http://ubuntuforums.org/showthread.php?t=1004752")

As to your critique of this forum; it is what it is.  Being a public forum, all viewpoints are tolerated.

A far more important point (weakness) is the system itself.  There seems to be no easy way to search this forum's repository of questions or, more importantly, the solutions to common these reoccurring problems.  

Another factor that adds to the confusion is the Linux, and in particular Ubuntu, need for change.  Don't get me wrong, change is good, but I feel a new release every 6 months is too rapid to flesh out the major problems. Accordingly, I feel that the latest Ubuntu release is for learning and not viable production hosts.  I use Hardy (8.04) and in some instances Gutsy (7.10).

As you have indicated you want to learn Linux.  Remember that with Ubuntu this can be a moving target.  Gnome in particular has  many new initiatives.
