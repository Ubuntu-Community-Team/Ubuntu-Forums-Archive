### Post by jak on 2006-01-31
The last time I used Ubuntu was 4.10 - a while ago now! But I was impressed then how it exceeded my expectations of a Linux distro (I had previously only tried Fedora core 1) and it became my distro of choice. 
After managing to get a good number of friends to try out Linux, with just few liking how it works, my hard drive became corrupt and I went back to a Windows XP only machine.
I've just downloaded and installed Ubuntu 5.10 and WOW! It's come so far, everything that was a little 'niggle' before has gone! And out of the box support for my wireless card is MASSIVE bonus, the new ways of adding applications is brilliant - the main reason I kept using Windows as my default OS was because I couldn't live without WinAmp - but I've now got amaroK that can run on GNOME, that's no longer a reason to go back, and the loading screen is just very impressive!

Just one problem - the ATI/Radeon drivers don't seem to support my second graphics card, an ATI Radeon 9200SE PCI - just need to get my second monitor up n running and then I think it's Ubuntu from now on!
