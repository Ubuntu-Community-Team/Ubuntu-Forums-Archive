### Post by zero244 on 2009-03-05
Im running Ubuntu Hardy.......which is running flawlessly and stable.
I installed a newer version of Wine to run a Windows mp3 streaming program.
The program wouldn't run correctly in a earlier version of wine on Edgy.
This time it installed and was running in about 10 minutes and works great.
It even downloads the streams into my Music folder on my ext3 partition.

At this point I am very close to not needing Windows at all.
I'm looking forward to the day I can delete Windows forever.
