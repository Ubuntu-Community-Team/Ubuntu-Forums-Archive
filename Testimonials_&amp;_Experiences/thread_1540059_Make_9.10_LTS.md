### Post by 3rdalbum on 2010-07-27
1. This is a rant message, posted in a support forum. This should be in Testimonials.

2. Ubuntu is a meritocracy, not a democracy. Ubuntu does not need your approval to change the UI if it has been advised by somebody who has proven themselves in UI design.

3. If you don't like the changes, then use gnome-stracciatella-session to get something very close to upstream Gnome. This is more like ordinary upstream Gnome than Ubuntu 9.10 was. Ubuntu has always modified Gnome and Ubuntu 10.04 is a continuation of this. I suspect you haven't actually seen regular upstream Gnome before.

4. Ubuntu 9.10 is not an LTS, 10.04 is. LTS does not mean "old and crusty", it means "able to be supported for the full period", so there's no reason why minor UI changes such as those in Lucid should not be pushed into an LTS. They are minor, anyway. Rearranging three buttons and adding a text box to your IM status menu is hardly "unsupportable", and it's not a "trial" either - this is likely to be the final location of these controls.

5. 10.04 is very stable on my computers, and presumably stable on the computers used to test Lucid while it was in development. Try running some comprehensive RAM tests. RAM sticks can become very faulty, very quickly.
