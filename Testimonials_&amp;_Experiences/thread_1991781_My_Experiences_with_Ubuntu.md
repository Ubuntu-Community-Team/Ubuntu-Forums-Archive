### Post by poettone on 2012-05-31
Hello Everyone!


I decided about 3-4 months ago that I was going to give Ubuntu a shot when I ran across wubi for windows, which for the record I can say is the only machine that has ran anywhere near stable at all.. I've got a plethora of machines at my disposal (see machine list below) and I swear, I've not gotten one machine to run right, without various issues, from networking to sound. Each and every time I update, make any type of changes, I'm presented with something new that doesn't work right. I've tried installing Ubuntu on various Dell, HP, Toshia, Acer and IBM hardware to include desktops, workstations, and laptop models. 

Every single machine has issues of some sort and all I do is update when instructed. The machine I'm writing this on is yet another fresh installation and I can't get various things working on it, very frustrating to say the least. 

I've tried various versions from 11.04 to 12.04, I've tried Ubuntu Studio, LXDE, XFCE, Cinnamon, Mint, you name it, I've tried it and you know, it seems no matter what I do, how many drives in the system, how I partition it, whether I use lvm or not, crypt or not,  or what desktop I use or what application I install, something typically goes wrong!

I'm one step away from walking away from this environment all together and yes, I know some maybe thinking, "I bet he is a noob or something" but I assure you, I can set the machines up with the defaults and it either doesn't work right from the start or it breaks when I attempt to update anything. It's actually a sad thing that we can't even get the network manager to run correctly and it's one of major components of any machine.. We have to have INTERNET:), Right:)

I'm not hear to bash Ubuntu, it's developers or anyone else as I feel Ubuntu is the most advanced Linux desktop alive today and has had major improvements done since I last installed it so many long years ago.. But c'mon guys, we need to have the basic tools functional in a release. This is the main reason most users are scared of running Linux, they don't want all the issues or have simply heard of users having major issues and it makes them cry:)

Honestly, If I could get 1 of these machines running correctly, where I didn't have to worry so much about what the next update will break, or even get it working correctly from the start, I might switch away from windows completely. At this point however, it's simply not possible.. Trust me, I honestly want too, soo bad, but I can't afford the instability as I need one stable machine in my career and for now Windows is the only thing I can have any reliance on, sadly:(

Now on the other hand: Good Stuff:)

First of all I really love the power that is built into the Linux OS as far as server OS's are concerned. I love the various products for audio, video and photography on the desktop. These coupled with an array of tools for managing a system are things that are hard to live w/o these days. X has come a very long way over the last decade or so, although the same concept is there, it continues to be a very handy tool for various reasons including headless systems. The scripting capabilities of bash, the anacron and cron tools and the list goes on and on regarding the stable, rock solid features in the server environment. Now I can't really speak all that much for Ubuntu server, but I have worked with a large number of Debian servers in the past and being that it is built on top of it, I would also apply that to the Ubuntu release, but others may disagree. As far as the desktop environment goes, I was confused at first by the Unity desktop, but once I learned more about it, I actually started to enjoy it more than your traditional menu driven systems.  

I know I've spilled my guts about Ubuntu and sadly, it wasn't all good, but it's the truth and what I've experienced in the Ubuntu world. I would say at this point I'm going to continue to support Ubuntu and it's efforts to become a viable stable environment that end users can enjoy. I've actually talked a few home users into trying it out. One I may have to revert back to windows for, but hey, not everything is perfect right;).

I'm not one to be defeated easily however, so I will make yet another attempt at installation and pray things work and I can update without fear.

I leave you all with this and if I've sounded harsh on Ubuntu, please please understand that I'm being biased here in my own experiences with the versions mentioned above.

### Post by kurt18947 on 2012-05-31
I've not had a problem with Network Manager but there have been some machines that had a problem with it.  Those users downloaded and installed Wicd with good success.  If you choose to try that, download Wicd _before_ uninstalling Network Manager.    Ubuntu or another Linux distro is not a drop in replacement for Windows.  It has a different file system with permissions which can be frustrating for Windows users, different method of designating partitions/drives, no registry (big loss, that :rolleyes:).   

I'd be disappointed if you had problems with the T series Thinkpads.  I've installed Ubuntu and other distros on various Thinkpads with no issues at all. It appears that the most common complaint for new users/new machines is getting wireless networking to work.  There is a pretty good networking and wireless forum here.  Some time spent there searching can be pretty helpful.  The odds are good you're not the first person to encounter a problem.
