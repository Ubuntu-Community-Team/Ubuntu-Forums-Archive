### Post by b9anders on 2007-01-26
I have been a windows user all my life. Although I have always been a meddler, I have never meddled for the sake of curiosity, but simply to run smoother. Running a 1.4 ghtz/256 MB Ram laptop, those skills come in handy. 

Tired of XP lagging, I did a re-install to clean up the system and this time monitor systemetically anything that entered my system. I had some 12 different programs dedicated purely to showing my cpu use, monitoring ram, checking for spy- and ad-ware, cleaning the registry, keeping tab of start-up applications, including ones hidden inside windows applications. There wasn't a single application running on start-up, whether essential to windows or otherwise installed, that I hadn't assessed and decided whether to run or not, with lots of non-essential startup applications trimmed away. I kept nothing on my desktop but the dustbin, did away with unneccesary windows graphics and even compressed my background image for efficient usage. In short, this time my install was as well-trimmed and tight as you could ask for.

And yet within a month, after installing essential updates, drivers, filling up the harddrive again and having a few features like anti-virus, a chat client a low-performance file manager and an email notifier run on start-up, Windows XP again began running progessively slower, using up more than my RAM just for start-up applications and having to rely on harddrive for the rest.

------

I have been using Xubuntu for a mere six days, I have as many threads asking questions in the beginner's forum, yet already I am a convert. It's not just the speed - although xubuntu runs faster than anything I have tried since I ran an equally welltrimmed install of Windows 98 and it's an absolute breeze to be running a system that is actually tuned to the performance of my pc without having to sacrifice much of anything but some flashy colours. It's the straight-out-of-the-box functionality of the system that makes the difference. It took a few days to get a run of what software to use, but doing a re-install again yesterday, and two hours later I had everything I needed to cover all my needs simply through the basic installation and two easy to use installation programs in automatix and easyubuntu. And most importantly, it was so intuitive I could find my way around it immediately.

Except for the chat client, the same basic internet applications I used in windows: Opera for browsing, skype for talking. In gaim I have a chat client to cover my needs across multiple platforms, replacing trillian which ate up more resources anyway.
All the multimedia I need - a sensible music manager replacing Itunes, dvd playback, the same sound editor I use in windows (Audacity), the same image editor (the GIMP), the same generic video player (VLC), even the same image library in Picasa.
All the office applications I need, replacing microsoft office without a hitch.

And then there's the little things - Like volume control. What I searched a long long time for in windows freeware to replicate here comes built into the system, allowing my logitech z-2300 speakers to play better than ever. And all I really wanted was a simple treble adjustor, something windows couldn't give. Although I did come across one of the shortcomings of the GUI. Looking for a way to adjust the volume, I of course went into the settings manager and found nothing of use. But then remembered a 'volume control' launcher that I could set up in the top panel which worked a charm. Not exactly the natural place to look for it.
Or the sheer neatness of looking for new and exciting software in Synaptic - frankly a brilliant concept I would never have imagined even existed as a windows user.
The workgroups. I love them. Nothing more to say.

So what are the drawbacks?

I have yet to look into how to make the wireless connection work. The GUI offers no help at all, and I am a bit uneasy about having to work this out.

I miss a few programs like process explorer, which gave me unparralleled insight into the system and Total Commander which is just a brilliant file manager for windows. I realise there are similar file managers for linux, but as yet I haven't found one fully compatible with Exfce. You'll note these are programs I downloaded from elsewhere and (until recently in the case of process explorer) had nothing to do with microsoft anyway.

As a newbie linux meddler, I still find it frustrating that although linux fulfils all my basic needs, I still don't know my way around the system yet. And in this light, I wish the GUI was more developed so I could play around with the system and just learn through osmosis as I did with windows and not having to get into learning a new language as the heavy reliance on the command line requires once you want to meddle behind the scenes. But this is something I can take with a grain of salt because Xubuntu DOES fulfil all my basic needs. Meddling further is just for the sake of getting to know the system better and the best part is, since it runs so smoothly, I probably won't need to meddle so much anyway. And it's not like I haven't spent hours trying to work out a problem in Windows. And the support is better for Ubuntu.

I miss being able to run Pirates and Counter-strike 1.6. I realise this should be possible on Linux, but right now, probably due to my lack of familiarity with the command line, WINE is just greek to me. Those two games is the only thing that is keeping me dualbooting. Once I am confident I can use those two, it will be goodbye Windows and I doubt I will ever want to return. Looking at what microsoft users have in store with Windows Vista, I think I have chosen the right time to leave Microsoft behind as well.

And the best part? It's all freely available. It just feels better to use a product that has been offered freely in the interest of functionality than a product designed to sell.

I started with xubuntu out of curiosity and mild idealism, I didn't really expect it to satisfy my needs. After a bumby first couple of days,  I can happily say it has satisfied my needs beyond all expectations.
