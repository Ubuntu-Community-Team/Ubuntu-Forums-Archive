### Post by sunrex on 2009-08-28
I never did realize how much better Ubuntu was over every other OS/Distro out there.

Server 2003, svhost uses 100% of one core. Can't figure it out, throw that piece of **** out the window.

Arch linux. I liked it, it was a good stable server OS. Sadly, it wouldn't clear up ram after large FTP file transfers and I was stuck with 100% ram usage pretty much..

OpenSUSE. Spent all of one hour AFTER installing the OS trying to get SAMBA working, gave up after windows still refused to connect. Also having to put up with the .RPM packages was the biggest pain in the *** I have ever had to deal with... now I remember why I stopped using SUSE 9.2 pro after buying it.

Gentoo. Good distro, especially considering its compiled from scratch essentially.. but pacman confused me, but thats not the distro's fault...

CentOS... YUM sucks.

Ah, and theres ubuntu.

Installer no longer craps out on me in the partitioning like it used to!. Customizing your partitions has never been easier.

GUI out of box no issues, instant install of vuze and gnome and proftpd and c... whatever, you get the point.

Low memory footprint with full GUI running, no 100% ram usage for whatever reason.

EXT4 filesystem support. Did I mention opensuse doesn't have that?.

Remote Desktop works without using over half my CPU, scores instant points with me since teamviewer on windows uses over half my QUAD core cpu.

Meh. Point is, I like it ;).

Thank you ubuntu, loyal user since... 2003?. God, it's been a long time. I'm so sorry for ever leaving you :(.
