### Post by 3rdalbum on 2008-11-17
If you're having trouble with networking on Ubuntu, then Linux Mint is unlikely to solve your problem. They are built on the same base.

The problems you encountered are by no means normal, and for a quiet majority of users everything will work out-of-the-box. It looks like your wireless card is not compatible (or rather, the card's vendor has not assisted in writing Linux drivers for it).

Just like people adopting the Mac OS, or adopting Vista, you must make sure that your hardware is compatible before switching to Linux.

Be sure to try Ubuntu 9.04 when it is released, as the wireless card's vendor may have written Linux drivers by this time.

That's actually a myth :D  You can still get crapware, a misconfigured server can be a major security loophole, typing in commands some random guy in IRC tells you to do as a superuser (see the Malicious Command sticky on this forum), things like that.
