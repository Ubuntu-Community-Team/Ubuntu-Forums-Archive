Firstly any time you install a new program (or update) you will be asked to provide your password.  This is a security measure to prevent malicious software from being installed.  One of the reasons that Linux is more secure then Windows.

2:  Since Ubuntu is based on Debian the method of installing is to use what are called DEB files and you can tell these by the file extension .deb.  A little more info .. some Distros use RPM and others use YUM as methods of installation.  

3:  All programs that you install through Ubuntu (synaptic, add/remove etc.) is automatically updated.  Programs that you install via CLI are not.  Also any program that you have installed through Ubuntu has been tested to ensure that it won't break your system, when you go outside this system you run a risk.  And again for security reasons with any programs that you want to install through CLI you will have to change their "permissions".

4:  With the Sun Java version of 6.16 in Ubuntu, it hardly seems all that worth while to go through those hoops for version 6.17.  As I said before, programs that are found in Ubuntu may nor be bleeding edge, but they have been tested and have a reasonable insurance that it will work well with your system.
