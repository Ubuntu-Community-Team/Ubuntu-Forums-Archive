### Post by plurworldinc on 2009-10-21
After installing 9.10 I found it to be delightful. One of the first highlights I found was the new usplash and log in window, very polished and professional if I may say so.The wallpaper wasn't that bad to began with, but I have never been a big fan of the brown, or most of the boring wallpapers with the versions before. Now there is a variety of great wallpapers to pick from right out of the box.

   Now on a very small down note, the only real problem I had so far was the Login screen didn't give me all of the features I was use to and the Start-Up Manager didn't work the same which was a major problem for me. See I have become use to having a little more control over my system since I switched to Linux. I am hoping this will all be worked out and back to normal when the final release is out in a couple of day.

   Another feature which I am not sure everyone else thinks about it is the new Apt Store which I thank is a wonderful idea for all as long as they don;t get rid of Add/Remove all together. The new apt store has it's good side and it's bad side. The only really downside I noticed right off the bat was you can only download and install one program at a time. Now this may work fine for Window users, but I like the idea of being able to download everything at once. This is great when you are setting up your friends computers with Ubuntu and get them damn near everything they could use in the same time it takes to clean a Windows PC for basic use out of the box. Now the plus side of the store is a tricky one because with a store this will give software developers that would normally just write programs for Windows and Mac another profitable platform to right for and get paid from. I just thank it would be nice to sit at my Ubuntu laptop and play World of Warcraft or be able to run Sony Vegas Pro if it was available for Linux.

   Now I am not knocking the open source software we have to work with and love, but if I can use my Pidgin IM in Windows I would love to be able to do the same as run some of my highend Windows software with Linux.

   But as I was saying earlier this is one of the best distros to date. Everything damn near worked out of the box for my laptop, I got wifi, I got sound, my videos are playing smoothly, hell my network is wonderful. I am a very, very happy customer....:)

### Post by 3rdalbum on 2009-10-21
It's actually "Ubuntu Software Center" - it's not a store yet.

You can queue up multiple software downloads and installs, up to 20 actually. If you're setting up a machine for someone else I'd recommend just using Synaptic because you can go through and mark for installation everything you want, and then just let it get all the software!
