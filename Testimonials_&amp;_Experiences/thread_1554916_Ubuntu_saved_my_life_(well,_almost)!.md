### Post by Captain Chirac on 2010-08-17
I'm really, really unexperienced with Ubuntu. However, most of it is quite awesome. I tried Debian, but it kept giving me problems. I tried OpenSuse, but it was troublesome as well. However, Ubuntu was working as soon as I would install it.
Now, its main drawback compared to Windows is the network feature. Ubuntu, and probably Linux in general, is really, really bad when it comes to domestic networking. While Windows would pretty much immediately detect the domestic network automatically, listing the shared hard drives and such, Ubuntu can't do ****. I tried to use Samba and all, but its usage is way too complicated and typically too terminal-friendly for a new user like me.
Nothing is perfect, and Ubuntu only confirms the rule. It's damn great on any other point, but when it comes to Networking, Windows wins ultimately.

That was a rent. I'm really satisfied over Ubuntu, but this drawback just repulses me. No, I'm not going to format my hard drive and install Windows again. This would be blasphemous! But it's just quite frustrating to see such advanced software being that "unadvanced" when it comes to a such fundamental feature.

Regardless of all this, thanks to the Ubuntu team and community.

### Post by foldingstock on 2010-08-17
> Nothing is perfect, and Ubuntu only confirms the rule. It's damn great on any other point, but when it comes to Networking, Windows wins ultimately.

Considering that Cisco's IOS is heavily based on Linux and is used on most large enterprise network routers/switches, Linksys uses Linux and FreeBSD embedded kernels on most of their home network routers, D-Link uses Linux and BSD embedded kernels on their home network routers, etc, I think "Windows wins at Networking" is an invalid statement based on a single, personal experience. 

Ubuntu may not work good for you when put into a primarily Windows environment. On the other hand, Windows may not work good when put into a primarily Linux environment.

I am really, really satisfied with Ubuntu. I don't need to click a thousand times in order to install something or remove some option.

When it comes to the rest, however, Ubuntu wins fair and square. And having to manually move my *** and plug USB ports isn't going to prevent me from falling in love with Ubuntu!
