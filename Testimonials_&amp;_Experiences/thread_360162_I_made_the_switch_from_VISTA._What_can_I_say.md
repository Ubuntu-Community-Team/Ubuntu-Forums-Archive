### Post by Thrashers7989 on 2007-02-12
I have a load of horror stories from Windows Vista. It comes with one great feature: an operating system that has no more eye candy than XP, but for some reason, it runs twice as slow on my very fast and new laptop.

After waiting (no exaggeration) 8 minutes for Microsoft Word 2003 (cancel or allow) to open up, I decided enough is enough and I took another 3 minutes to open Firefox (cancel or allow).  I heard that a Windows downgrade from Vista to XP is too much of a pain in the neck, so I logged onto some Linux distribution sites (cancel or allow) and I saw a lot of good talk about Ubuntu. I basically decided I was sick of Microsoft all together.

I figured I needed a distribution that is good for both the novice and advanced user. Since Ubuntu is suitable for both, it will allow a more-than-novice user such as myself to learn more about the Linux operating system, and become advanced in my experience.

I downloaded Ubuntu (cancel or allow). 

The part I was dreading most was the partitioning. Let's face it, as long as this world is built for Windows, I'm going to need it to at least sit somewhere on my computer for when I need it (academic, work use). Partitioning was a breeze, took little time, and was so self-explanatory that my grandma, who can't even operate a clock radio, could do it.

So now GRUB allows me to choose Ubuntu or Vista, Ubuntu being the default.

It took me a couple days to set Ubuntu up exactly the way I wanted to (programs, eye candy, learning some key things, upgrading from Dapper to Edgy). I'm happy, settled, and ready to learn more and get deeper in not just Ubuntu, but the whole Linux environment.

There are a few things I miss about Vista:

The fact that I can no longer take a vacation while waiting for Vista to boot up.
Cancel or Allow
The fact that there will not be any viruses and trojans to keep me company.
The pipes screensaver
The obligation to use Windows Messenger (not to be mistaken with MSN messenger)

So, Ubuntu community, thank you for a great experience. I will carry on with the experience. Also, if you didn't get my "cancel or allow" reference, see _[this video]("http://www.youtube.com/watch?v=VKM1cAtAdtQ")_. It might not be advertising Linux, but it's oh so true.

[_[another Vista demo]("http://www.youtube.com/watch?v=oiG7KFDYkLI&mode=related&search=")_]

Sincerely,

Bryan

### Post by dragoonmac on 2007-02-15
I keep windows on my machine for 2 reasons, fixing other people's NTFS drives, and World of Warcraft. That being said I got a free copy of Vista and decided to try upgrading.

It has some nice new features, folder structure is more like linux with nice little home directories, the shell is more like nautilus with its added functionality, search has finally caught up to beagle, it has a SLED like start menu, and the voice recognition is the best I've ever seen and pretty painless to set up. 

Ubuntu is still better.

For Example: 
I can run all of Beryl's features on my laptops integrated intel915 chip. AIGLX compositing works flawlessly and fairly smooth even while doing other things. On Vista, because intel won't write a WDDM driver, I can't run the glass features (a definate motivator for me to upgrade) 
The sidebar still has nothing on GNOME's integrated features. Wireless networking is still Klunky at times. There is no persistant gksudo/kdesu, it asks for every change. Haven't even tried to install codecs on the sound engine yet. 
Still no viable alternatives to Tomboy, Bash and K3B (except nero... which isn't cheap).
Hmmmm... Not gonna roll back to XP, but I think linux us still going to be my primary OS.
