### Post by Philip Gray on 2012-02-10
Hi from South Africa. I purchase the Ubuntu-User and Linux-Magazine everytime when they come out. 

I have read the article, ‘Unity And Progress’, on the Unity Desktop by Joe Casad. This was an interview with Neil Patel. I also read the article
‘Caution: Fresh Paint’ on the GNOME 3 Desktop by Christoph Langner.  I found both articles informative. I have an NVidia graphics card in my PC 
at home and was pleasantly surprised to find that the Ubuntu 11.10 live cd ran on my home pc with the Unity Desktop (11.04 would not run). I 
tried out GNOME 3 Shell using the OpenSuSe 12.1 livecd from the Linux-Magazine issue 134.

I would like to say that I do not like either the Unity Desktop or the GNOME 3 Desktop. I did not receive a pleasant experience from either
one. I feel that both desktops are clunky.

My issues with the Unity Desktop are:
•    I cannot move the dashboard.
•    There are no Min, Max buttons in program windows when they are maximized. This 
     does not make any sense. There should be an option which gives the user the choice 
     to show or hide them.
•    Windows are displayed maximized by default. Except for office documents I do not 
     like to work with windows maximized. This is also an irritation.
•    The program windows do not have menus. I like menus. The lack of menus being 
     automatically displayed in MSOffice 200/2010 was one of the reasons I installed 
     Linux at home. I dislike the ribbon bar.
•    I cannot add icons (shortcuts/application launchers) to the desktop. I like using 
     icons since it speeds up program selection and execution.
•    There is no applications menu. This facility currently gives me the flexibility of 
     adding items where I want them, creating folders/groups as I want to. I do not 
     always agree with Ubuntu’s default locations.
•    I cannot add any items to the panels (bars) at the top and bottom of the screen
•    Synaptic package manager is missing from the Application pages. I did read 
     somewhere that you can add it by installing it from the Software Centre. This makes 
     no sense at all.
•    Each page shows the software installed and below this other software that is 
     available. This I find very irritating. I have not found how to remove this. I regard 
     this as unnecessary screen clutter. If I want to see what is available I can go to the 
     Software Centre.

I have similar issues with the GNOME 3 Shell. I think that Neil Patel is being a little presumptuous when he says that users who do not like 
Unity will hook in, sooner or later,  the users who want their old desktop back. I still have not decided which is the least offensive of the
two desktops.

I am currently on 10.04LTS and I was thinking of upgrading to 12.04LTS. I have some concerns and queries:
•    While Neil says that users will still be able to install GNOME 2. My concern is how 
      long will this option be available.
•    The GNOME 3 article mentions dconf-editor which can be used to display icons on 
      the desktop. Does Unity have a similar facility?
•    I now face an issue as to which alternative desktop to use. I do not want to use 
      either the Unity or the GNOME Shell desktops.
•    I feel that Canonical, with the Unity Desktop, is starting to do what Microsoft has 
      being doing for years, namely:

[LIST]
[*]  Deciding what is best for their users.
[*]  Pushing their ideas on their users and the community.
[*]  Ignoring what the community wants and how they feel.
[*]  Giving their users what they do not want.
[*]  Deciding how their users should work.
[/LIST]
I feel that the Unity and GNOME 3 Desktops are regressing not progressing.

The above points are my personal views. I have no intention of using either the Unity and the GNOME 3 Desktops in their current form.

People can call me old fashioned. I like to keep things simple (My first computer was a Commodore 64). 

I like Ubuntu very much but I do not like the direction Canonical is going with the desktop. I have found articles on Wikipedia on two other
desktop projects. Namely: Maté and Cinnamon. I went to the Cinnamon website ([http://cinnamon.linuxmint.com/](http://cinnamon.linuxmint.com/)) and had a look at their themes. 
Well I was impressed they all look professional and I found that they have a version for Ubuntu 11.10. Hopefully they will bring out
a version for 12.04LTS. If they do I will try it. They seem to have combined some aspects of GNOME 2 with GNOME 3. The items that impressed
me the most were:

[LIST]
[*]That you can add icons etc to the desktop.
[*]That there is a menu which looks like a combination of the ones in Gnome 2 and KDE4.
[*]That there is no annoying dashboard.
[*]That you can add items to the two desktop panels.
[/LIST]

I currently dual boot into Ubuntu and Windows XP. My Windows XP is set up to look like Windows 95/98. 

These comments are just my views. I will be interested to hear what others in the community think about this.

Regards
Philip

### Post by Version Dependency on 2012-02-10
> **Philip Gray said:**
> In  theory it basically gives me what I want namely icons on the desktop, a nice start menu, no annoying dashboard ability to add applets to the panel bars.

I'm confused.  First, you can put icons on the desktop in Unity (just like in Gnome, KDE, XFCE, LXDE, etc...).  At least, I can do this in 11.10 and 12.04.  But my mom uses 11.04, and I know I've seen her desktop littered with files and folders.

Second, there are a TON of indicators you can add to the panel: [full list here]("http://askubuntu.com/questions/30334/what-application-indicators-are-available").  If you notice, one of the indicators listed gives you the old style Gnome drop menu if you must have it.

Third, the annoying dash has it's uses especially with the third-party lenses that are available: [full list here]("http://askubuntu.com/questions/38772/what-lenses-for-unity-are-available").  Some of the more popular lenses would be the Youtube, Piratebay, Gwibber, and Reddit lenses.  

And finally, you hate using the dash to search for apps?  Add the most popular ones to the launcher, obviously.  And, if you want, you can [add category quick lists]("http://www.omgubuntu.co.uk/2012/02/category-unity-quicklists/") to the launcher (like Internet or Multimedia) where you right-click on the icon and a list of available apps appears.

The above are some of the reasons many of us love Unity.

Gnome 2 is actually no longer even included in Ubuntu, though there are  various ways of getting a somewhat similar experience. The 'fallback  mode + indicator applet' mentioned above is one; Xfce with customised  settings would be another. However, even without these there's no need  to worry about your system being 'stuffed'! Gnome 2 has been forked as 'MATE', with everything renamed so that MATE and Gnome 3 don't conflict. I use it with 11.10 (and it reportedly works with 12.04 Alpha). If you want to try it, ignore the websites that suggest using the Mint package (which can cause problems), and go instead to the developers' own repositories:

[http://wiki.mate-desktop.org/download#ubuntu](http://wiki.mate-desktop.org/download#ubuntu)
[http://mate-desktop.org/](http://mate-desktop.org/)

I'd suggest some experiments in VMs first, to see which solution suits you best before you upgrade.
