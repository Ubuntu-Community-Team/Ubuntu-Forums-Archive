### Post by jbysmith on 2009-04-07
As far as Ubuntu having bugs go, absolutely.  There is no one perfect OS with zero bugs in it. Doesn't exist.  Never has and never will.  

But on the plus side they do tend to get corrected in Linux a lot quicker than in Windows. (Shoot some Windows bugs have gone *years* before finally being fixed.. some not at all.)  Plus Linux in general is constantly evolving; stuff will break.  Canonical does a great job making sure stuff works good, but nobody's perfect; stuff can and does slip through but gets patched pretty quickly.  Windows on the other hand, bug patching aside, pretty much never changes for a few years till the next build.  (And then usually everything breaks, from drivers and software compatibility to your bank account.)  

As far as Vista's drive usage goes.. well that's Vista.  If you have say a gig or less of memory, it's going to definitely be swap-happy as it takes a lot more to run than XP.  It also does a lot behind the scenes.. between SuperFetch, the indexer, the swap file, and other things running, it can really pound the drive.  Tends to ease up after a while, usually the first few days are the worst.  Of course, disabling some stuff would help too if you're not using them.  As far as the menu, the All in One and such, got all the current drivers?  Might want to hit the manufacturer's site and make sure.  If your system is a bit light on processor power/memory (you didn't specify), might want to consider going with XP if you absolutely have to have Windows going.  Should also look into native alternatives to whatever software you're needing, there's quite a lot of good stuff out there.  Or, if it absolutely has to be Windows based, Wine or VirtualBox picks up the slack nicely too.
