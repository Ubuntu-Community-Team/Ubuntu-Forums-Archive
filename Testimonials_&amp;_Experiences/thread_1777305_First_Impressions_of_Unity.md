### Post by Jerry N on 2011-06-07
After playing with Mint 11 (I like it!) and watching the comments on Unity, I decided to add it to my test computer so I now have quintuple boot WinXP, Win7, Mint 10, Mint 11 and Ubuntu 11.04.  I have only a few hours of using Unity but I can certainly see why it can be frustrating to experienced gnome users - I keep wanting to right click on the upper panel and add the plain old menu system, same as I do with Mint.  I also keep wanting to just drag an icon onto the desktop.  I have seen a lot of bad mouthing of Windows on these forums over using the Start button to access the shutdown functions but now you get to the System Settings in Ubuntu through the Exit icon!  

So far, I don't think I am a fan of Unity but a lot of that might be because it is so much different from Gnome and Windows.

I haven't experienced Gnome 3 yet.

Jerry
