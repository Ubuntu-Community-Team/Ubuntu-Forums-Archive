### Post by MrDigital1 on 2006-10-17
I've been a Windows user since 3.1 , a Linux "tinkerer" for a few years and a more recent Ubuntu user (with the release of 6.06).  I'm a Gamer so I have a Windows XP box at home, I also have two Macs running OS X Tiger.

My Gaming Rig at home is as stable as a rock, but I've *never* used IE on it to go anywhere.  I will be upgrading my system to Vista probably soon after it comes out.  I will have a Ghost image of my system prior though :D

Spyware sucks though, enterprise versions of Antispyware do OK but not good enough yet.  I'd prefer to move the entire facility to Firefox but that's not really my call.

I love MAC OS X Tiger and will upgrade to Leopard when it comes out.  It's a great OS for getting things done, email etc.  I don't forsee changing my Macs to Ubuntu or any other linux anytime in the future.

I love Ubuntu.  I run it on my primary Work system (which we are a 100% Windows shop) but I use the awesome VMWare Workstation for Windows stuff and get done what needs to be done.  Linux in the workplace isn't for all businesses yet.  We use proprietary (no choice in the matter) software that the clients only run on Windows, no matter what I do to try it in Wine.  I forsee using Ubuntu for a long while and already advocate it to other people to at least try the Live CD to see how they like it.

Ubuntu is by far the easiest distro I've ever used for installing new software into it.  The Synaptic system is great and I figure if it's not there the average user probably doesn't need it.  Ubuntu is an OS I would put on my Mom's computer.
