### Post by eilios on 2009-02-01
While Windows constantly shells out security patches, and looks great, I found it to be a constant hindrance as I was forced to maintain Antivirus software, the AERO feel was a bit too tacky, and worst of all, setting it up was a pain. When I first tried Ubuntu, it was definately different.

When I started up, the installation was fun, quick, and relatively bug free. Windows users constantly complain that they want to try Linux, but the installs are too hard. At first, this was a major reason to not want to try it, but I figured if I used virtualbox there is no harm done, so I started up, in about 10 minutes the install was finished, I set up virtualbox, and installed. It was amazing, because setting it up was pretty fun, it was simple to use, and it walked people through it. Unfortunately, setting up drivers was a hassle at first. The terminal was a bit different from Batch commands I frequently used.

However, after a few hours of learning the interface and commands, and with great help from the friendly community, I was getting accustomed to it. It was a simple enough procedure to set up programs, installing became a breeze. I felt like catching up on some old school RPGs, so I downloaded ZSNES. It was extremely painless to install, and took about 5 minutes from the applications menu. From there it was as easy as typing a line of text into the terminal to start it up. I thought it would be a hassle to use the terminal, but after learning to use sudo, it was much easier then at first I thought.

Overall, I am very pleased with Ubuntu, it's not as user friendly as Windows, but it's a better OS overall, and has enough horsepower, plus all the customization I want. It has the eyecandy, and it's free; so I believe this was one of my better ventures.
