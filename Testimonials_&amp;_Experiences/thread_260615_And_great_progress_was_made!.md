### Post by Zyzzyx on 2006-09-19
Been just over a week now of having Ubuntu installed. Haven't really done too much, most of my system use is for basic web browsing (hooray Opera!). Tonight though, finally got somewhere.

Thought I'd try something different, a 'why not' install. So, I put VM-Ware Server on. Had a few difficulties on aborted installs, having to go delete everything, but got it. Was feeling great.

Then it backfired a bit. Tried installing Win2K on a VM system, install hangs at the *last* little bit. Tried again, same thing.  Oh well. Dug out my original (pre-SP1) XP disk.

Got XP installed on a VM system. It works! How wierd is this, to have XP running **IN** my Ubuntu system. But hey, its working! And decently smooth too (AthlonXP @ 2.2ghz, 1gb ram). Did a bit and tossed on Office 2003, one of the reasons I was having to boot back to XP, to get into my Outlook2003 occasionally. I've got WebAccess on my Exchange server, but without IE, it can't do everything. Managed to get Outlook2003 working, with an IMAP connection, instead of my usual direct Exchange setup.  

Hmm, this means that I do have IMAP set up right on my Exchange server. OK, time to try Evolution again.

Fired up Evolution, and aftef a fair bit of fiddling (again) with settings, ended up with a listing of my Inbox. WooHoo!!!  

But, I'm ending the evening on a negative. Got the IMAP going, but don't have SMTP outgoing working yet. Oh well, that'll come.

Still, this was a great fun night working in Ubuntu. And I'm really settling into the multiple desktops. I had VM-Ware on one, Evolution and stuff on another, Opera with lots of tabs and some terminals on another, and UbuntuForums with some terminals on the last. Toss in a few more things in each as well. I can see why I never thought of such a thing (multiple desktops) in Windows, I was never able to DO that much stuff, and still have a smoothly working system!  Oh, and to top it off, I've got Folding@Home running as well, taking up them spare CPU cycles.

So, a successful VM-Ware install, some more learning about the file system layout, and getting my email (partially) working.  Yeah, t'was a good night. And now its time for bed.



PS. One gripe so far with Ubuntu is that I haven't booted into XP and played Planetside since two Saturdays ago, when I installed Ubuntu. Been having too much fun without gaming!
