### Post by Kelvari on 2012-07-09
I first started using Ubuntu in the first half of 2007 on an OLD IBM Thinkpad that I had gotten second-hand. The original Windows install on it was so messed up I couldn't do anything with it, and the driver hell that I wound up going through trying to install it made me want to throw the poor thing out the window. 

I'd tried some other Linux distributions in the past - Fedora Core 3 and 4 - but had major issues with stability with them. Enter Ubuntu 7.04 (Feisty Fawn). The Gnome UI took some minor getting used to, but all-in-all, I found the transition to be mostly painless. There were some issues with Microsoft-only formats (.docx) at school, but those have been resolved with newer versions of both LibreOffice and Microsoft Office.

Now, two laptops and 8 releases later, I'm a dedicated Ubuntu-ite, running 12.04 on a Gateway MD2614U and Lubuntu 12.04 on an old desktop that's serving as a Minecraft server for a friend. With the pending release of Windows 8 - what seems to me to be a disaster in the making - I don't foresee myself returning to the Microsoft camp anytime soon.

### Post by Tamlynmac on 2012-07-09
> Kelvari
I first started using Ubuntu in the first half of 2007

My family and I also started in 2007 with 7.04 and have been Ubuntu "only" since that time. We don't use the new releases and when support runs out for 10.04 in 2013 we'll switch all our systems over to Xubuntu (it's running on one of our desktops now).

I can say without any hesitation that it's been a fun ride. With each new release came improvements and some challenges. But in the end, Ubuntu has proven to be quick, rock stable and intuitive. We learned to stay with the LTS's and to test each new release prior to commitment, assuring a successful transition. At present, we have 5 systems running Ubuntu and numerous other family & friends in the region account for another approx. 15 systems.

The one thing I don't do is concern myself with Windows or argue over the new Ubuntu releases. I don't invest any time bashing, commenting or even comparing Windows to Linux. It's been my experience that the best way to attract new users is to let them experience Linux for themselves. I generally have a stick and/or disk around with Xubuntu on it and show those that express any interest - how to try it. Staying away from comments regarding Windows and focused only on the benefits of Linux.

Good Luck & I hope you continue enjoying your OS'(s) of choice.
