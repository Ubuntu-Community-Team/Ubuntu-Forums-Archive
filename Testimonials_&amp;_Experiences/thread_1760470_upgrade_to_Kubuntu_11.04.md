### Post by el_koraco on 2011-05-17
Fresh install ftw. Possible problems - the 3.6.38 kernel has some power regression issues that effect some machines, like heat management that stops working after the computer goes into idle. Some Aetheros cards, using the ath9 and ath5 kernel driver can get you into trouble, but it can be remedied. Broadcom cards are using the new open source drivers, which is also a hit and miss situation. Fullscreen flash works only half reliably. Some GPU's are likely to run into trouble, but what's new. Other than that, Kubuntu 11.04 is incredibly fast and reliable, and the 4.6.2 and 4.6.3 KDE series rock the house. 

if you're gonna be upgrading no matter what (although i suggest you don't, but go for a clean install), make sure you disable any proprietary drivers and PPA's, and release holds you might have on packages.

### Post by SeijiSensei on 2011-05-19
I first installed 4.6 using the backports repository for 10.10.  On a laptop with an Intel 945 GM graphics adapter, it was unbearably slow until I turned off desktop effects.  On my desktop machine with an NVIDIA 9500 GT I have no performance issues whatsoever.

I've since upgraded fully to 11.04.  I noticed a few connectivity problems with my Linksys USB wireless adapter (RT2500 chip) that I solved by remaining with kernel 2.6.35-28 from my prior Maverick install.  (This isn't my first problem with the kernel drivers for this device; starting with 10.04, some kernels work fine while others have problems keeping the adapter connected.)
