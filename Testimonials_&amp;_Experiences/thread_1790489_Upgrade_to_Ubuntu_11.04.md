### Post by deserthowler on 2011-06-25
After over-thinking the project for a few months,:confused: I finally decided to upgrade from Ubuntu 10.10 to Ubuntu 11.04. 8-) After all the problems I read about, it went off without a hitch ... so far. [-o< So I have successfully upgraded my Lenove Thinkpad T60 9.10 --> 10.04 --> 10.10 --> 11.04.  :guitar:

So far Unity, Unity2D, and Gnome classic all seem to work; so does FVWM (the best window manager ever!!!).  Grub works, the grub background works, the login manager works, the login manager background works, the wallpaper works, WiFi works.  All of my customizations held over from 9.10 still work.

Easy process and I only held my breath for 2 hours.:rolleyes:

I bought the Lenovo used but checked the hardware specs before I bought it.

Just thought that newbies would like to know that it can be done.

Earl
