### Post by martinw89 on 2007-03-22
I'm very comfortable in a Linux environment, I've been using different distros for about 4 years now.  While I got stuck on Gentoo for a while because of the customization options and because it worked well after messing around, I've fallen in love with Ubuntu.  Everything works out of the box and in some cases better than Windows.  For example, because the drivers are already there ethernet and wireless access worked right away.  I did encounter a serious error though.  When I installed the nvidia drivers, "nvidia-glx-config enable" killed my X config.  But I just had to use a terminal to revert to the old config.  After reverting, "nvidia-xconfig" worked successfully.  I'm very pleased with package management and the community, all-in-all I'm just really pleased.

A  quick note though, Edgy Xubuntu and Kubuntu were not nearly as easy to get working straight away.  I'm sure this is partly because of the desktop environments themselves but it's one of the reasons it took me so long to really like Ubuntu (for some reason I didn't want to use a Gnome based distro, kind of stupid in retrospect)
