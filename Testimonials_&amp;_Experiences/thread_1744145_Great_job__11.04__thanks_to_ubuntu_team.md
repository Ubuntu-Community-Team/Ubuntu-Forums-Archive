### Post by JheregJAB on 2011-04-30
You would be right, if unity was not the default.

Ubuntu 11.04 was released as a finished, ready-for-the-general-audience product. However, it appears the general audience is having many problems with it. For an average linux user, the problems we are encountering are minor - we are used to things like this. Unfortunately, the point of Unity was to make Linux more user friendly for the non-linux user, the user who isn't going to be "educated" about what Unity is and requires for them to be able to use it, the user who frankly will only care if it works or not. If you log in with no knowledge of Ubuntu or Linux whatsoever, Unity is what you see, so that is what you have to judge 11.04 on, not some comparatively obscure setting most people won't even notice without being told its there or knowing they even have the option of looking.

Unity would be a big, positive-direction step for linux... if It did what it was supposed to. It does not, and saying that not using unity is a solution to unity, while congratulating the team that failed, is absurd.

Unity is not working right now, so as far as the now-target audience is concerned, 11.04 is not working. You have your opinion, and I will have mine. All I can say is that I hope no one gives up on Ubuntu because of this blunder.

Jacob
