### Post by rrich1974 on 2012-12-27
i hear you man! 
there are two philosophy that are in conflict with each other. 
as a general rule, if you want a stable OS you must keep it as light as possible, without unnecessary bling-blings. in the ent, as i told before, you don't stare at the desktop, you run some apps. the same thing is valid for windows too. 
on the other hand, you want to learn something about the core of the system, you want to customize, install and try all kind of programs, etc, etc. in this case, sooner or later, you will get some degree of instability. 
that is what i say: for the first case, you should install a stable release i.e. ubuntu 12.04 and if you want to play around you can also have a partition of around 10 GB to play with a cutting edge distribution i.e. ubuntu 12.10 in our case.
