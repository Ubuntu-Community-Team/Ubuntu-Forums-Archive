### Post by MoobyMuffin on 2008-04-15
I just wanted to stop by and say hello. I recently made the plunge and replaced Windows on both my wife's and my computer and I'm happy I did. I've got Gutsy running on my wife's PC and Hardy on mine and we've not had any issues with the conversion. My wife is still getting used to Ubuntu, but she's catching on fast. She's just happy that she can browse the web again!

I've been a Windows user and developer for a long time and I used to think Linux was not an option, and up until recently I used to think the only draw to Linux was the fact that it was free, which I'm quickly learning that is not the case. As someone said in another post of mine, "We use Linux because we like Linux." I'm quickly learning that's the truth :)

I'm still a n00b but catching on fast! Here's to hoping I can contribute one day soon!
