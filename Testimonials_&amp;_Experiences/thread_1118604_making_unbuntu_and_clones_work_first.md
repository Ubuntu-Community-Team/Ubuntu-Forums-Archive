### Post by 3rdalbum on 2009-04-07
Are you on 32-bit or 64-bit?!

If you're on 64-bit, then just install the 64-bit Flash Player, or the 32-bit "flashplugin-nonfree" package which also pulls in nspluginwrapper (used to run 32-bit plugins on a 64-bit browser).

Alternatively, I know you can download Flock as 32-bit and it will run 32-bit plugins. I know this because I accidentally downloaded the 32-bit edition.

If you're already on 32-bit, then any browser in the repositories, including the Firefox that ships with Ubuntu, will be 32-bit.
