### Post by Mazza558 on 2008-03-21
It was with some trepidation that I downloaded the beta of Hardy Heron earlier today. I burnt it to disk, rebooted, it detected my resolution straightaway. It installed, got to 96% and got stuck there. Oh dear. I gave up after 2 hours and restarted with a Gutsy CD in order to upgrade. I did so just now, and the overall experience is fantastic. 

- b43-fwcutter works just as well as Ndiswrapper now (it used to be awful on Gutsy - Ndiswrapper was the only option. I understand now however that to install it at the moment, you need to use the command line as the Restricted Manager method is broken.

- No need for XGL! Yay! Installation of the ATI drivers was even easier as I didn't need to install XGL seperately. Not using XGL might even mean that 3D apps like Google Earth work properly now (haven't checked yet). I'd like to add, however, that it is slightly slower than the XGL version, so keep that in mind.

-Brightness works! Amazing! Finally I can rest my eyes, as the default brightness (highest) is a tad too bright.

-Lots of little things like Tracker in the notification area telling you what it's doing.

-Firefox 3 Beta 4 is very nice, a massive improvement from FF2.

-The biggest noticeable bug at the moment is the font rendering in FF. The individual characters are far too close.

Overall, this will absolutely be the most stable, easy-to-use version of Ubuntu (or Linux distros in general) ever released. Good job, Devs! :)
