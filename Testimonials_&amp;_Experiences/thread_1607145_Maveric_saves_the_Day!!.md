### Post by ventrical on 2010-10-27
Thanks again Ubuntu team. I was in a really tough spot, trying to help a friend with an audio/visual problem. They needed to  record some childrens books in french (francais) for a student who was having difficulty learning the language. The Nokia recorder worked well but it was way too difficult to seperate each file and the audio wasn't good enough (at least so I thought).

   Some while back I had noticed how when with the mouse ,Ubuntu 10.4 and 10.10 would hover over a .wav file or .ogg file that the file would play imediately (which was a cool default). Anyways , I think the <Sound Recorder> was also a default install with Meerkat and it works just great !! I can record any music or voice file, save it, and use Brasero CD/DVD burner to make a working MP2 CD !!

  Iv'e racked my brains out trying to find somthing similar in Windows XP but to no avail.

Thanks again .. impressive!!
