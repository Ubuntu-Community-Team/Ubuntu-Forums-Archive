### Post by Dragonbite on 2008-10-20
The other night I was wearing my Ubuntu cap when I went to pick up Pizza we ordered. The guy behind the counter asked "You Ubuntu?"

He has tried Ubuntu but uses another distribution instead.

I thought it was interesting to run across another Linux user "off the street." That's where wearing something Linux, or especially Ubuntu, comes in handy. 

It is a testament of the smart marketing Canonical and Ubuntu are doing, that the Ubuntu logo is recognizable and that even if one doesn't use Ubuntu, most Linux users recognize the logo and the name.

I don't know if it is more recognized than Red Hat's shadowman or SUSE's Chameleon but considering it has had less time to become recognized and that Ubuntu is targeted to average consumers and not just server system administrators and computer geeks I think they've done a great job!
