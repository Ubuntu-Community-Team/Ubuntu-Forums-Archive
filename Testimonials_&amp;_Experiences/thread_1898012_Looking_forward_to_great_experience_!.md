I am not new to Ubuntu/*nix in general .. I got to know about it back in 2006 when I saw a friend of mine install SuSE xx on his laptop. I had never seen another OS and didn't really know what to make of it .. sure the work could be done and all but the fact that Windows isn't the only OS was news to me ( I was 16 years old then ) ...
 
I remember installing Ubuntu 7.04 on my desktop after partitioning the hard disk and remember how I got the audio/video working and the subsequent feeling. Since '06 I've had a dual boot with *buntu on it ( dabbled a lot with Kubuntu/Xubuntu?Edubuntu too ). When I was done with my college days, I got myself a laptop which is when my problems started. My habit of changing the OS more frequently than I would get my haircut would result in a broken system very often and soon I damaged my hard disk. Windows would refuse to detect the hard-disk ( which I no longer regret now ! ) and I would have trouble working with other versions of Ubuntu and Linux out there ..
Unity interface -- not comfortable
KDE - Nautilus would give problems with my 'damaged' hard disk
XFCE - Xubuntu live disk wouldn't install
other Linux - I had problems getting the wifi to work
 
Now I've put on 10.04 on the system and I'm loving the vanilla GNOME desktop - the file manager works smoothly, the applications dont have their GUIs broken ( as they had when I'd install XFCE over Kubuntu and other such permutations). I've disabled most updates so my system won't break after some package gets installed. My laptop should work for a LONG time without any hiccups now.
 
I know the post is too long and wordy. Bottomline - Hurray for 10.04 .. hope it doesn't run into problems on my laptop after April '13 !!:P:KS:lolflag:

### Post by stalkingwolf on 2011-12-22
There are several options that might make it possible to save your drive. It has been my experience that when a drive starts to get noisy it isnt far from belly up.  Unfortunately there has been no consistancy as to how long its life will be.  Ive had a couple that ran for years and a couple that lasted minutes.

Hirens boot disk has a tool on it that i have had good success with. It is the HDD regenerator.
It and UBCD have a couple formatting tools and there is gparted.   Remember any method of formatting wipes the drive.

There is a cable available on ebay under 15.00 that allows you to run any drive from usb. i have several.
