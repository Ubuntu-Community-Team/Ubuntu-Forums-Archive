### Post by Kilcaff on 2009-11-12
Hello,
 
I've been reading through these forums now since I first got my aging desktop pc to my new house which was about a week before the Koala was officially released up the Eucalyptus tree.
 
I've decided to give my thoughts on how I'm finding it.  I'm a windows user, I've tried them all from 3.1 to Vista, even ME!  But I'm no programming expert and my needs and demands have changed over time.  Currently I have XP on a desktop and Vista pre-installed on a laptop but I'll cover that in a bit.
 
So why start looking at Ubuntu?
 
I originally tried some form of Linux a good number of years ago, it meant quite a few floppies and confusing instructions so I gave it up.  I always liked the idea of Linux so when someone I knew recommended heading towards Ubuntu I was happy to look into it.
 
I was looking for a good alternative to Windows, given that I’d missed out on the introductory price for Windows 7 and that it had shot up to the usual ridiculous amount.
 
Windows has always been a wrestle to use with random performance lags, bloatware and security issues. I hate the way MS tries to force feed you their products, the amount of times it tried to install IE8 overtly and covertly was shocking.  It reminds me of those opt out boxes on most application forms now, in the case of non-essential software and updates it should be taken I don’t want it unless I say I do.
 
The idea that I can finally get a non-bloated OS with security that would support most hardware is too good not to explore.
 
So owing that I’m fairly impatient I downloaded the beta than rc and then proper release of 9.10.
 
Here are my impressions so far…
 
Installation.
 
No real trouble here.  I had some bother burning the discs but that’s the burners fault not the software.  It did what I expected it to and it was simple enough for me to follow without finding a tutorial.I had some issues with the partitioning as I didn’t realise you had to slide the bar along the bottom graph even when you’d clicked the top radio button for dual boot.It was quick as well.
 
First use (beta/rc)
 
I only had a bit of a play here to see what the hardware support was like.  It got my keyboard, mouse etc fine.  I could see the display so it was using my graphics card and the resolution was ok, not the HD output I wanted, but it was there.  This is only on my desktop so far, hooked up to a HD tv.
 
It failed to get my wireless card working though and it didn’t play mp3’s straight from the install.  I’m not far from my router so I plugged in a network cable for internet access and it installed the files to play mp3.
 
Given it was a beta I figured the wireless would be sorted for the release so I waited.

I then did a clean install the full release once it was came out.
 
I had no issue with it going on the pc, dual boots fine with xp and booted up very quickly.
 
I also downloaded the 64bit version for my laptop.
 
I find it pleasing to look at, easy to navigate through and generally use.  I’ve now downloaded a few added bits, get my 3d card showing the 3d chess and messed about changing backgrounds etc (the national geographic website has really good ones).
 
However I have had issues.  The wireless issue wasn’t at all solved with the final release.  I’ve got a thread going on problems with the bcm driver on my laptop, it just won’t find my network as it’s on a channel higher than 11.  Ok I can drop it down but it’s not really the point and it’s where it is so it doesn’t get interference from a neighbour’s router.The WPN311 card in the desktop doesn’t seem to be working at all but I’ve not started on that one yet.
 
My other main issue is that I really want Ubuntu to take control of the media centre but it won’t output the HD resolution which XP manages with the catalyst control centre.  I can’t use the CCC for linux as my card is too old so this is a real pain at the mo as it means too much pressing of the av button on the tv.  I’m hoping to get a tv card very soon and I’d love it to output to the HD channel.  Incidentally, any tv card recommendations?
 
I also find that firefox is slow at times, doesn’t like java or flash and now my music is starting to skip when I play it on rhythm box on the desktop pc.  However that same application got into my ipod straight off which is more than anything other than itunes manages on windows.  This is an issue that gets on my nerves and it seems to have only just manifested.  I’ve not been on the computer for a few days so not managed to have a crack at fixing it.
 
I know some of these issues can be worked through and these forums are fantastic but lets just say it took the shine off it for me.
 
But, before it sounds like I’m unhappy with Ubuntu, I’m not, far from it.  I’m mad, frustrated and a little disappointed at times but I’m also addicted.  I love the way you can install something, and then remove it with ease.  I hate the way it prompts me all the time for my admin password but I’m getting use to it and I love the speed the system has.
 
There are some things that seem over-complicated and technical but then I’m learning terminal commands all the time.
 
Haven’t tried gaming on it but then XP is still there so don’t feel the need to yet.  I’ve burnt cd’s easily enough and had a look at open office (which the Open University give to you on a cd as well for report writing, though I’ve heard mixed things about it).
 
I still dual boot with vista and xp and now that vista upgraded itself and promptly broke on my girlfriends laptop she’ll be using dual boot too…
 
For those interested I’m including my specs below.
 
Desktop:
300gb HD, 50gb HD, Connect 3d radeon x800 xt 256mb, Amd Newcastle athlon 64 3200 754pin, Asus k8v-se deluxe s754 via k8t800, Netgear WPN311 (I’m not a big fan of this card or software for it on windows or linux!), Win XP
 
Laptop:
HP 6735sAMD Turion X2 Dual Core RM-72 2.1Ghz, 4GB 800DDR2, 250GB 15.4&quot; DVD RW LightScribe Writer, WLAN, Webcam, Bluetooth 2.0+, Onboard ATI Radeon HD3200 Windows Vista Home Premium

### Post by Teamgeist on 2009-11-13
Concerning Flash in Firefox.

Since you're running 64-Bit Ubuntu you should use the 64-Bit Flash-Plugin from Adobe.

This is what you have to do.

First uninstall the 32-Bit Flash and the wrapper that makes it work on 64-Bit Ubuntu.

```
sudo aptitude purge flashplugin-installer flashplugin-nonfree nspluginwrapper
```

Then download the 64-Bit Flash-Plugin from Adobe:
[http://download.macromedia.com/pub/labs/flashplayer10/libflashplayer-10.0.32.18.linux-x86_64.so.tar.gz](http://download.macromedia.com/pub/labs/flashplayer10/libflashplayer-10.0.32.18.linux-x86_64.so.tar.gz)

Unpack it. You will get a file called libflashplayer.so

Now copy it to the .mozilla/plugins folder in your home directory.
The dot in front of the folder name means that it is a hidden folder.

So go to your home folder and press Ctrl+h to see all the hidden folders.

Navigate to the .mozilla folder and then to the Plugins folder (if it does not exist, just create the Plugins folder yourself).

Now just drag and drop the libflashplayer.so file into this folder.

Restart Firefox. Done.
