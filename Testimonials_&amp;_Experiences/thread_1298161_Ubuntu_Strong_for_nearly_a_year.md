### Post by Kraust on 2009-10-22
I have been using only Ubuntu (no Windows Partitions) for nearly a year now and I have to say that I have loved the experience overall. I have had minimal issues with compatibility or getting things done. I feel that for now with every new computer I'll get I'll make my Primary partition an Ubuntu one and only use Windows if I really want Gaming because that's all Windows provides for me at this moment.

*Advantages over the past year:*

*A Clean Workspace environment
*Dock Customization without the need of Processor intensive applications that run over Windows
*An application repository for everything I'd ever want
*Seamless updates via the update manager for all my packages
*Minimal compatibility issues]
*Having a Virus free experience.

*Disadvantages over the past year:*
*Issues with both Flash, Java and Video Playback (Primarily H264)
*Wine needing more intensive system reqs for programs than would be needed in Windows. Primarily games but also a Program I need for my Networking class that I'm having problems getting Linux version of.
*Having what is now a 9 year old PC

(Also it's helping me do better in School, as I'm looked to by one of my Professors as a person that "knows Linux" versus my other classmates who only use Windows.)
