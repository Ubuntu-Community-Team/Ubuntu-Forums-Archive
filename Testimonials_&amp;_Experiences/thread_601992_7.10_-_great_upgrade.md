### Post by cudds on 2007-11-03
So I got a dell vostro 1700 and put 7.04 on it. Ubuntu found all my drivers over than my nvidia card but that was just a quick download from their site.
Cool so everything was running just dandy but I could not get beryl or compiz to run right. Sometimes they would load but were really, really slow and selecting the cube would crash it etc. Oh well I thought, these technologies are developmental any way.

So today I had a bit of time so I thought I'd run through the upgrade. It only took about 30 minutes and my computer was mostly fully useable during that time.

On reboot I discovered that all the new fancy compiz gear ran first time! horray!

The only kink was that my audio driver didn't run but apparently this is a known bug for my PCI set and running a patch on it worked a treat.

So it looks like Ubuntu is going from strength to strength - every new flavour/version I try there's something to impress me!

Now - I just need to find that hack that stops my touchpad from working for a micro-second after a keystroke as I keep nudging it whilst typing and it's going all over the shop. I'm thinking this is built into windows as I've never had this problem before
