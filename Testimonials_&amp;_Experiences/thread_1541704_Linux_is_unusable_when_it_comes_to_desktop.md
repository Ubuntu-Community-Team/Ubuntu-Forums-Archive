### Post by prodigy_ on 2010-07-29
For me any flavor of Linux is unusable when it comes to desktop. Ubuntu 10.04 isn't really better or worse than the rest.

There are **always** issues with drivers. Since 2008 I tried to install fglrx about 10 times, using all possible methods and it always either failed outright or worked until next the next kernel upgrade. 

Software installation (and especially uninstallation) is still problematic. Repositories are just crutches. They alleviate the pain, but don't solve the actual problem. Once you need to uninstall something that wasn't installed with a package manager, you feel like you're in the stone age.

Upgrades to new releases never work correctly. Flash support on 64-bit is nonexistent. Compiz is so slow it hurts. And so further, and so on...

It's been this way forever. Of course there are some improvements here and there. For a simple laptop or a netbook, where you use only a Web browser, a mail client and OOo, Linux is probably the best choice if you can get it to work with your hardware. But for a media center or for a gaming rig, it's horrible. In this regard nothing has changed and nothing will change in the future. Linux will never be a serious desktop OS.
