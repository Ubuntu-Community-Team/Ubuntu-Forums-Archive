### Post by Matymoo on 2007-10-29
I stuck with Ubuntu for around a year. Edgy worked fine once I disabled IPV6 (after much frustration and searching) and required some messing around with the xorg.conf file to get decent screen resolution. 

Feisty... internet worked as soon as I booted for the first time. No need to turn off IPV6. Screen resolution still had to be manually configured and a new problem, the Nvidia driver caused all sorts of weird problems when using a 3D desktop (Beryl/compiz).

Gutsy is a failure. One good addition was the ability to increase my screen resolution without editing xorg!! Once again, IPV6 must be manually turned off. Nvidia driver must be manually installed. I was unable to successfully get a 3D desktop running. In addition to all this, I got errors whenever I attempted to do a system update. I tried many software sources to no avail. I was unable to install the flash plugin for firefox. If I clicked on the "install plugin" prompt, I immediately got a "not found" notification. I tried all the fixes I could find in the forums... didn't help. Uninstalled and went back to Feisty and had no probs. Tried 2 more gutsy installs.... no difference.

I am now using openSUSE 10.3.
I was presented with the option to turn off IPV6 during installation. (Ubuntu just enables it and it's up to you to trawl the forums to find out why your internet is not working and how to turn off IPV6). 

To install the correct Nvidia driver, there is a one click function on their website. 

[http://en.opensuse.org/NVIDIA](http://en.opensuse.org/NVIDIA)

To install Compiz, again there is a one click function again.

[http://en.opensuse.org/Compiz_Fusion](http://en.opensuse.org/Compiz_Fusion)

No need to mess around. One click and it's installed and working!!
It just worked!!!! (Isn't that ubuntu's slogan??)

Guys. I gave Ubuntu a go for almost 12 months. I was mostly impressed as a first time Linux user. Things were a bit tricky until Gutsy... when they became damn near impossible. 

I reluctantly gave PCLinuxOS and now OpenSUSE a go. In all honesty, if you are a first time Linux user, OpenSUSE would be a far better choice than Ubuntu. Sorry guys. I did try.
