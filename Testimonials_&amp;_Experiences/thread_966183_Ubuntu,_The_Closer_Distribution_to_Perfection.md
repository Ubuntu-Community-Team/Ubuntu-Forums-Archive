... And that's why there are only four suggestions to improve it, while GNOME will be another independent issue in another independent post.

    * An ability to boot to command line by:
          o Either pressing 'Esc' during the boot progress (recommended)
          o Or adding a "Command Line" option in the boot menu
          o Or both.

      With a screen like this:
      [http://www.linuxfuture.org/media/issue.png](http://www.linuxfuture.org/media/issue.png)
      Just like Slax
    * This means that mc (Midnight Commander) should be available
    * And there should by a way to import/export user profiles, to/from a GZip file, includes:
          o Documents
          o Email Data
          o Favorite Bookmarks
          o htdocs (Apache files)
          o Desktop
          o Application Configurations
          o Templates
          o RSS Feeds
          o Atom Feeds
          o Web history
          o Cookies
          o Recent Documents (links)

      These should be stored together in a one folder.
    * The last suggestion is to use GPerfection themes. Take a look:
