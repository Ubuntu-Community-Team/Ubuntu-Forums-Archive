### Post by dah7171 on 2008-10-17
I had to replace my previous computer earlier this year - my old computer's motherboard was burnt out.

I got a budget computer which had Linux installed. I wasn't too keen on it so I installed Windows XP but since I had installed it more than 3 times previously( on my old computer) I had to phone Microsoft to get a code to install the operating system which is ridiculous since I paid for it legitimately. Since I had Windows XP SP1 I had to download service pack 2 and loads of updates and Service Pack3 . It took ages. I had to install a program for my printer/scanner/photcopier and also for my wireless adapter and office program and photo editing program etc.

When Windows crashed on me recently the thought of going through this process again put me off trying to reinstall. I got a cdrom with Ubuntu 8.10 and installed Ubuntu. Between the install and setting up the system the way I wanted took only about an hour. Admittedly I had issues with playing DVDs because of DRM but I looked at the thread on Ubuntu forums, copied and pasted various commands in the Terminal and , hey presto, I can play DVDs. My wireless adapter and printer/scanner were immediately recognised so I did not have to install software for them.

Open Office is great and so are the photo editing programs. Everything seems so much faster in Linux. I will not go back to Windows. 

I will be 60 in just over a week and I am not a techie. If I can use Ubuntu Linux anyone can. One of my daughters can use it OK as can my 13 year old grandson.

I push Ubuntu Linux as much as I can and hope it has a great future
