### Post by tbzep on 2008-01-03
IIRC, there's two choices (I'm not very familiar with Vista).  

1. Burn a "mastered" cd, which waits until you are finished adding files then burns the whole thing....disk at once. The help file says this should be readable in all systems.

2. Burn as you go...burning each file or folder you drop in, as if it were a regular drive. The help file warns that this method isn't compatible with a lot of systems.  I think if you just click the ok's it defaults to this.
