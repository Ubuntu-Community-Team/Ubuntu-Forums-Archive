### Post by Hallvor on 2007-03-15
Ubuntu 6.06 was the first distro I ever tried. Something went wrong with my Windows XP partition, so I decided to install Linux. The install went smoothly, but I basically did the same mistake as many other newbies and expected a free version of Windows where most things were the same. Many small problems accumulated, and I did not expect such a learning curve. Problems turned into frustration, and I spent basically a whole weekend reinstalling Windows XP. Had to look all over my house for a driver for my wifi-card. Fortunately found it. Then browsing the web for a few other drivers and finally installing all the software I wanted.

Anyway, I decided to read a little more about linux. I came across these forums and decided to install Ubuntu 6.06 on an old 8 gb hard drive I found in a computer in my attic. So the installation went as painlessly as last time, but this time I was better prepared. So I took it step by step, read on the forums, asked questions and learned little by little. And since I double booted, I could always boot back to Windows if the learning curve was too steep. However, as I solved more and more problems, I did so less frequently.

Today I have a perfectly working OS. Since I love file sharing, my computer (amd 2500+ xp, 512 mb ram) pretty much runs 24/7. When I used Windows, the computer became slow after a couple of days and needed a reboot. Now, my computer can run for many days and it will still be fast. No need for a reboot at all.

I also installed Ubuntu 6.10 on my wife`s heavily bloated and spyware infected laptop with Windows Xp , and cut the bootup time from 5 minutes to 50 seconds.

I just want to thank for all the friendly help I have got on these forums. It was priceless for me as a newbie. Hope I can help others as I learn more in the months to come.
