### Post by Duskao on 2009-11-06
I have been reading all over the forums about how crappy/bad/$%*&^$ karmic is. I am here to disagree. I'm whole heartedly prepared for a flogging from the trolls out there.
My first point is this. It's Ubuntu 9.10/karmic and this not a long term support release, it has also only been out for about a week at this point so there are going to be lots of bugs that haven't been run into yet with the general public, if you want a really stable desktop use 8.04 LTS, most if not all bugs have been worked out of that release. 

Point #2. Most issues I have run into can be fixed with some tweaks here and there. This is the linux community we should be used to tweaking things the way we want them to be. 

Point #3. Ubuntu is free! We are given an amazing operating system where the possibilities are nearly endless of what we can do an accomplish with it. Canonical Ltd. had done a great job with Ubuntu and we as the linux community should be thanking them for releasing such high quality products free of charge (unless you want official support) instead of flaming them so regularly when things aren't working "perfectly" 1 week after release. Look at Windows Vista (which karmic has been compared to... seriously???) it was released in 2007 retail, thats almost 3 years ago and the operating system is still full of bugs. Not to mention all that goes with windows OS's (virus/malware/rootkits/tune up utilities/defragging ect.) I don't feel I need to go on about this.

Give the koala a chance at least, let a bit more time pass, maybe a month, let some more bugs get worked out before you start flaming it. If nothing happens after a decent amount of time then feel free to start being a critic. 

Thanks for the OS Canonical. You guys do a terrific job.
