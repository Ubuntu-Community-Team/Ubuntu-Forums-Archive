### Post by groundpounder on 2009-06-01

Forced (or coerced if you prefer) equipment upgrade is a very m$ thing to do.  I'm really disappointed to be leaving Ubuntu behind, but it just didn't meet my needs anymore.

So long all,

### Post by snowpine on 2009-06-05
When an Ubuntu release reaches "end of life" its repositories are moved to [http://old-releases.ubuntu.com/](http://old-releases.ubuntu.com/) . You can simply edit your /etc/apt/sources.list and change all of the URLs to start with old-releases. Obviously nothing new is added to these repos, but they are available if you need to install something for an old release.
