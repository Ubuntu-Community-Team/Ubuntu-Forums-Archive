### Post by Karm on 2009-01-01
I've tinkered around with various Linux distributions, but never stuck with one for any period of time. I just performed a dual-boot installation on my HP notebook (Pavilion dv6000 running Windows Vista) and I must say that I'm very impressed with Ubuntu.

The install was smooth and I've got Flash, Adobe reader, Java, and Netbeans all up-and-running quickly. There were no issues with the wireless connection either.

Thanks go out to all you Ubuntu developers and community members that contribute. I think I'm going to be using Ubuntu a lot more in the days ahead.

Cheers!
