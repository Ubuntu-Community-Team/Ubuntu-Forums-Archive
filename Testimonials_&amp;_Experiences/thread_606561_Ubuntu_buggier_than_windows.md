### Post by Ub1476 on 2007-11-08
Not to be rude, but you are new to Ubuntu after all. I think I reinstalled three times before I learned to use Ubuntu. Like, the gnome-settings-deamon error comes to me when I use apps which is in beta stage (like Gimmie), when I remove it/turn it off, everything works smooth again. Compiz-Fusion (the cube) is also in beta stage, so when you downloaded the compiz-config and changed the settings, it's not supposed to work 100% anyway:p. Installing is easy, add/remove features a 100% gui + 20.000+ apps. Extracting is also easy. Just right click on the file and choose "extract here". I don't know about the k3b error, BUT it is for KDE (kubuntu), so you might want to try Gnomebaker or Graveman. I haven't had any problems with k3b myself though..

Linux apps which isn't in the repos, usually have a .deb package. If they only come in source you have to open a terminal in the app folder and write "./configure && make && sudo make install", but all stable apps have a .deb package.
