### Post by lexpython on 2008-01-21
I'm an long-time Windows user, and recently I decided to install Ubuntu on my old 600 Mhz PC to try it out.  I set it up as a dual-boot with XP.
I'm pretty good with Windows, to the point where I occasionally hire out to help friends with their machines. I use a lot of free windows programs to keep my system clean and fast, such as "Tune XP", "Advanced Windows Care 2", "Auslogics Disc Defrag", and on and on.

The machine: 600Mhz Pentium III, ASUS motherboard, 680+ Megs of RAM.  Runs XP SP2 without any trouble.  Runs Photoshop CS1 without any problems other than speed.

So far, here's my Ubuntu cons & pros:
Cons:
1) It is arcane.  Some software is easy to install, some you have to use a terminal and research the installation, and then it doesn't always work, like XMMS for example.
2) It freezes.  I was not expecting that, but it freezes nearly every day, to the point I have to hard-shut-down my machine.  Ouch.  Songbird and Firefox seem to be the main culprits.
3) Linux in general, even to the websites, is difficult for the layman to figure out.  Just finding this "Ubuntu Testimonial" page took some digging, as a simple search turned up nothing.  This seems fairly universal amongst Linux pages.
4) "Easy" stuff, like Flash in Firefox, doesn't work, even after several tries with several plug-ins.

Pros:
1) Relatively easy to install, though even this took some research, and I'm still not sure what my "swap" disc is, though I assume Ubuntu stores stuff temporarily there.  I learned of the need for a "swap" drive upon encountering an error message in the install.  A non-geek would have stopped right there.
2) Ubuntu looks pretty good, and there's a lot of customization available.
3) It's a teeny bit faster than Windows within the OS, though not so much in Firefox.

The skinny:
There is no way I'm switching. I may run Ubuntu when I build a home server to manage the files on my 4 existing computers.
While it's certainly made some progress from the Gnome front-end I used when I worked for Sun Microsystems, it's still hard to use for the average non-programmer geek, without some serious research and maybe a gook Linux book.  While I'm certain that any answers I may need are out there, I don't have time to find them.  I need something that works well out of the box.

It was only after I installed Ubuntu 7.10 that I discovered Ubuntu Studio.  Maybe studio would be better for me, as I need music software like WinAmp.  XMMS had the arcane-install, and Songbird is crashy. The native juke is OK, but really basic.

If only my 600Mhz had a DVD reader or allowed me too boot from USB in the BIOS, I could install Ubuntu Studio, but alas, that is not the case.

Theres my $0.02.  Until Ubuntu is complete enough for the average user out of the box, it can only attract simple-needs users and mighty geeks.  I look forward to playing with it and learning when I have the time.  I already spend 60+ hours a week on computers, so it would be quite a commitment for me to learn Linux stuff.

Cheers!
