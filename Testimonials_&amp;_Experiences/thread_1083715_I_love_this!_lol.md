### Post by mdsmedia on 2009-03-17
I'm no security expert, and I've read many posts regarding Linux vs. Windows security.

Linux isn't immune to malware, but it is largely immune to Windows malware.

Linux can be infected, but the spread of that malware on Linux systems is far more problematic.

If you use Linux and install software from outside the repositories you are at a higher risk of infection than installing software from the repositories.

If you install software from outside the repositories which is FOSS software you are safer than if you install non-FOSS software.

If Linux's popularity has a direct causal relationship with infection, and the Web is a popular target of malware writers, why, given that Linux is dominant in web servers, are there not more (any?) infections of Linux systems?

Linux isn't immune, but Windows is a much easier target.
