### Post by davposs on 2011-11-09
I also have to confess I'm enjoying the clutter free look of gnome 3.
However I also seem to be having some usability issues.
It was running well until a few days ago when suddenly it started running extremely slowly. This is a fast machine and when I dropped back to Unity everything ran lightning fast again. Didn't find much help in the forums as to what the issue might be. Checked all the machine parameters, processes etc and all seemed ok.
I persisted and tried a range of things/workarounds. In the end I tried changing the theme to something different (picked Ambience) and then on logout/ login everything started running fine again. Wierd.
Anyway I thought I'd post this just in case someone else has a similar issue and this happens to help them
