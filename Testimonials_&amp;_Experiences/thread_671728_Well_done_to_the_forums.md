### Post by azimuth on 2008-01-19
I can sympathize with your slowness. I have a full blown Ubuntu 7.10 on a Pentium 200. It wasn't planned that way, it just sort of happened. The alternate install of Xubuntu Dapper turned out to not have the right module for my ethernet card. I stuck the Gutsy LiveCD in and Dang! it installed. I have everything working on that old computer now and have been changing out the windows manager and browser to speed it up. 

I found this site [http://www.psychocats.net/ubuntu/minimal#barebones](http://www.psychocats.net/ubuntu/minimal#barebones)  ,after the fact and would go the minimal install next time. A 10Mb basic CLI OS and 6 lines of commands to apt-get a usable GUI up and running. As far as I can see, it all comes out of the repositories and requires no compiling.... That's dirt simple compared to a full install just so I can replace and  uninstall a lot of it.

If I ain't careful, I may learn something yet!
