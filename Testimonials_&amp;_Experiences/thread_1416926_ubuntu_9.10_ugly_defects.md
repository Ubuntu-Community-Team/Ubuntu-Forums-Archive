### Post by 3rdalbum on 2010-02-27
> **opengl_cpp said:**
> Since I installed ubuntu 9.10 I faced some ridiculous problems that I've never seen in previous versions:

One of the worst problems comes from sound driver. There is a weird issue when I play a game. Most of the games don't event play any sound, others play sounds with a lot of killer noises.

Don't use OpenAL; it can cause a lot of these issues. I'm guessing you're playing OpenArena.

> the third one revolves around flash player, I have installed all three major flash players

Don't. Use ONE. Most people use Adobe Flash Player. If you are on 32-bit, just do 'sudo apt-get install flashplugin-nonfree'. If you're on 64-bit, download and install it from Adobe's website (the 64-bit version).

If you prefer freedom, then use Gnash, but don't expect it all to work because it's a reverse-engineered player.

> I wonder that I had no problem with my ubuntu 9.04. It was amazing. I hope that 10.4 is not the same product

I say this to everyone, but don't just 'hope' that 10.04 has fewer problems than 9.10. Help us test! 10.04 alpha 3 has just come out and if you update it every couple of hours you can report bugs on every problem you see.

Without bug reports, the Ubuntu developers don't know when things are not working on certain machines or certain setups. So download the Alpha 3 image, install it in a spare partition (or make a spare partition), keep it up to date and use Apport!
