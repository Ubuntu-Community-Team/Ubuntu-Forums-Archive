### Post by gringopig on 2009-11-07
After some time now, from 8.04 to 9.10, I have tried to persevere with Ubuntu thinking that at some point it could be an alternative to Windows.
Sadly, my experience has been negative.
My main issue with the operating system has been the shameful lack of support for my Highpoint RR2300 RAID card, on which I store all my media files.
Having no access to these files is a major failure. Highpoint are culpable too for not providing working drivers but I feel it is the Ubuntu project most at fault for not having native support for RAID cards of this type.
The need to be some sort of super-user with an in depth knowledge of arcane terminal commands also rankles; I need to use the operating system not understand how it works!
This, in my opinion, is why Ubuntu (and Linux of all variants) will never make it to the mainstream userbase.

So, back to lovely Windows. Triple booting XP x86, Vista x64 and Windows 7 x64.
Pure pleasure.
They work. The drivers work. All is well.

I used the only indispensable module of Ubuntu (GParted) to repartition my system drive and Ubuntu is gone for good.
