### Post by Melindrea on 2008-09-18
So, finally went for the reinstallation of my desktop (It's been needing it for a bit now - the last few months I've gotten blue screens of death, which never happened before). 

I remember reinstalling XP on it some years back, and needing to use the CD the computer came with (that restores the harddrive) to get my network card to work, and was fearing this. However, no need to fear. I just installed it, and I had internet. My new keyboard (an A4 tech with volume buttons and such) worked out of the box, with all buttons except the one labelled Favorites. My soundcard was worse. However, I'd already figured that out, and I had found a link to OpenSound, which has a very nice wizard for how to get it to work. And work, it does. =)

Things still to do: Flash, java, Skype, couple of other programs, but I seem to recall finding tutorials for how to easy set up most of those, so I don't fear it. Oh, and check how it plays DVDs.

Surfing is surfing, the amount of information downloaded when surfing the internet depends not on the operating system, but on the websites your visiting, or the browser your using.

You can customize Firefox to be more frugal with what is downloaded, you can also run someting like Dillo or even Links if you needing something lite for doing things like reading these forums. I actually install "links" first thing in case I need google on a new to me hardware configuration, that way if for some reason I get stuck in CLI mode I can google my way back to a desktop.

Dillo is a bit fancier than Links, it actually has a GUI, and can display pictures and things, it is incredibly lightweight and insanely fast, indeed I'm going to install it now and use it for everything that doesn't require the bells and whistles of Firefox.
