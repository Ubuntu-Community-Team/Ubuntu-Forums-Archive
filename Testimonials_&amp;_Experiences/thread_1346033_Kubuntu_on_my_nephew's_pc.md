### Post by Chame_Wizard on 2009-12-04
After checking the  Live CD for the 2nd time for defects/errors,I decided to clean install Karmic Koala again(I cancelled the last one when it was 80% #-o ).

Since there was a ext4 and a swap partition from the previous install,I have to removed both( i.e allocating the free space partition).

I was shocked how fast the install was(make ext4 and swap partitions,filling in important fields):full 20! minutes.
The funny thing is that this was on a Intel dual core pc(a Packard Bell pc :shock:).And faster than my AMD single core  64 bit pc with the same specs(they have 320 GiB HDD space).

After that,I did the things what I do on my own pc(fresh install):Upgrading the package manager,edit the software sources, installing the Nvidia drivers,make sure to use the FLOSS apps and of course customize the whole thing.

They(my younger nephew's) were totally surprised when I did ctrl+F1/switched the virtual desktops,giving them options to choose a lot of things.

So good thing that it's a dual boot,cause the Visblah Basic they have is screwing the pc.Although their connection to the Internet is lower than mine:1,5 Mib/s against my 25 Mib/s,there still a lot to learn.

CPU:Intel dual core E2160(1,8 Ghz)
