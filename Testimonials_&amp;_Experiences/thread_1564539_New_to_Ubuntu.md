### Post by rosslad on 2010-08-30
Hi all,

I came over to Ubuntu 3 weeks ago and just wanted to say what I'm thinking about it right now.

I've been a windows user since windows 95 onwards and although it gave me everything I needed and more to get all my computer needs done there has always been a couple of things that really bugged me about it.

1st was speed, with all my windows installations it felt like I spent more than half my computer time just waiting....
whether it was waiting for programs to load, waiting for internet pages to load (I switched to Firefox 2 years ago and although it was faster on the windows os I still waited for ages for it to even show signs of loading), or downloads.

2nd was the much unneeded (for my use anyway) extras installed along with it that in all my years I never needed.

about 3 weeks ago I was not in the mood as I was waiting for ages to do something but everything was taking too long... too many restarts later I said that's it I'm going to wipe my drive and install fresh again and see how it goes.... then I thought actually I'm going to give something else a try so I searched and came across Ubuntu.

My first impressions were "OH No!", then "hey cool" and then "this is brilliant"

Installed seamlessly, was easy to understand (for the most part) and boy was it fast!!
I can spend more time working and doing general computer / internet tasks instead of waiting.... and waiting. lol

Love transmission, downloads are peaking at my max, never seen it do that on windows even straight after a fresh install... god knows what other processes / resources were running in the back of windows to keep my speeds crawling.

Love the themes i can apply as well as the ease of installing programs so easily from the software centre... brilliant!

Overall my experience has been awesome and have been raving about it to anyone who will listen for the past couple weeks... but there are some things I still need to get used to:

Luckily I found most of what I need in the software centre and with a click I can install it but there have been times I want some program and get confused on how to install an external program that is not in the software centre, I am so used to .exe files and not a big reader, I am more a trial and error sort of person so I need to get used to how to easily install downloaded programs.

Another issue I have is that I have tried many of the CD /DVD writing programs in Ubuntu but the speeds are really slow when burning a disk... This along with copying files over to a usb / memory card are the only times I have to wait (for long periods) on Ubuntu... I hope this is something simple I can get sorted as windows (amazingly) was super fast with them tasks.

I haven't got around to looking at everything I can add to Ubuntu but I have looked for some particular programs and came up empty so far, Programs like Frontpage 2003/2007 or Expression and Xara Webstyle.
These are programs I used most often on my windows os and although I know there may be ways to run these on Ubuntu I would rather an open source / compatible alternative with the same basic functions.

But as you can see its nothing major.... all in all I am really happy with Ubuntu and plan to stick with it (hoping I can sort the CD / DVD & USB / SD card speed problem as quickly as possible).

I am having no other issues and besides those 3 points above I don't miss windows at all!

So Thanks for an awesome alternative and keep up the great work.

Thanks
Tony

### Post by takisan on 2010-08-30
Just to solve the running windows programs questions: there's a program named named WINE that runs windows progs on Ubuntu and other Linux / UNIX platforms. It's available in the software repositories, but they have a special repository (head to [http://www.winehq.org/](http://www.winehq.org/) for more info).

As for the CD / DVD burning speed, I have no idea; that's not my area of expertise, but somebody else can probably answer it.

Have fun with Ubuntu.

EDIT: Also, if WINE doesn't work, there's a little group of programs called Computer Emulators, and they work brilliantly for several purposes, because if it crashes, it doesn't crash the host system. The only cache with using Virtualbox (my personal favorite) is that you need to have a copy of Windows and an Install disc. WINE, however doesn't require a Windows Install Disc, and is also Open Source.

Emulating a Computer also has slight advantages with being able to mount and unmount disc images, and network hardware, but they are a bit sluggish on my systems unless I'm running DOS or Win 98SE.
