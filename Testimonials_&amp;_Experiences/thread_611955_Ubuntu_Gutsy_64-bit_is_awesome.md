### Post by MNICY on 2007-11-13
I first found Ubuntu because my computer was dead. Literly dead. Dead dead dead dead. The motherboard was fried. It was so bad. XP would not load. I dug out my old 98 and installed it. Oh my god, that is a painfull OS. I guess it was good back in the day, but compaired to XP.. :mad:
So, i thought, 'why not try linux'. So i did some reasearch and decided to try Ubuntu. Downloaded Ubuntu Edgy (this was about a month before Feisty was released) and put it on a CD.
I was able to use my computer for another 2 months before it stoped working. I was impressed.  
So anyway, i got a new computer (thank god). Not super, but works great. Of course, it came with Vista lol. Installed XP, but could not get the internet to work (apparently no drivers for XP.. only vista... lol, you dont hear that very often). So, i decided to try good old Ubuntu again (which i decided i liked when it was on my old PC). I downloaded 64 bit Ubuntu Feisty, installed it, and it was great. I messed around, broke the install, and reinstalled it easily (and did not have to activate it like in XP). i was able to easily recover my files before reformating the harddrive usign Live CD and a flash drive :D
I actually ended up reisntalling about 3 times.
Then, i decided to try Gutsy. Downloaded 64 bit Gutsy, and it works GREAT. All the problems with 64 bit Feisty are gone (like Flash in Firefox), and the OS is running amazingly.
Compiz was amazing. In feisty i had to look up guides, and i actually messed up  my system once because of trying to install.
But in Gutsy, all i had to do was install the compizconfig settigns manager, and it was up and running :D (i would suggest putting an option in the desktop effects section that will download the settings manager for you, as i did have a bit of trouble finding exactly how - the help file told me but i wish it was more strait forward. Put a warning if you must). 
All in all, Gutsy is amazing.
I got rid of my Windows partition, and now have just a full ubuntu computer :D
The only thing i have is a virtual machine that i use for a few programs i cant get to work in WINE. (by the way, i am very impressed with the ease to install programs like wine. Add/Remove is very nice for the adverage user, or if you know the name then 'sudo apt-get install' works great!

I have tried a few other linux distros, and they are ok. But i like Ubuntu's feel, and i think it will grow big.

By promoting Xbox so much, Microsoft is eventualy going to take away the biggest reason i think most people have with sticking with widnows. They will all switch to Macs. Some will switch to Linux.
