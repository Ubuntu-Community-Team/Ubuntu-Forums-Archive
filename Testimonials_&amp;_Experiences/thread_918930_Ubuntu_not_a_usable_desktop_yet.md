### Post by Toet on 2008-09-13
Now using Linux as my only operating system for some years. Used a lot of distro's by now. Started with Ubuntu, now maybe have ended with Ubuntu.

It's nice Ubuntu. Easy to install, almost everything you need, the rest installed within a click or two.

But to depend on an operating system is a different story. I like music a lot and I play music a lot from my computer. It is not nice when you are enjoying your music, and it suddenly stops or breaks. Or when at a party someone wants to show some YouTube, and it just crashes. Or when you are watching a DVD, and it just stops.

This happens too much in too many different ways. It should just work and go. The only distro I've tried and is stable is Slackware. I've tried Debian, Ubuntu and Slackware. I've tried other distro's, but only for a few days, so that's not a good try. Only if I have used a distro for a few months exclusively I can say something about it's reliability.

Debian Etch is quite ok, it is more stable than Ubuntu, but sometimes the implementations of packages is not ok for me. Simple things can be a real pain. It just doesn't work smoothly sometimes, packages are adjusted and some functionalities of the original version do not work. Furthermore gnome, in my point of view, is getting worse as it progresses, sadly. It is for me hard to understand that Ubuntu is build on the not stable version of Debian. Unstable Debian. To get the newest and most fancy versions of software. It's all there new for me to test, that's what is feels like.

It's ok to use Debian as a server, but it's not there to be used as an only desktop, I think.

I do understand to base Ubuntu on Debian, cause it's such a broad distro, a lot of maintainers available.

Ubuntu I liked with Dapper where everything seemed to work. Now with Hardy it's the next LTS, but I can not rely on it. I makes me a bit sad, because I fell in love with the concept and the community. I feel a bit heart broken now.

I will go now to Slackware with XFCE, waiting for KDE 4 to be ready to use, or maybe try to see if Solaris 10 can offer me the things I need.

I'm writing this not to burn down Ubuntu, cause it did gave me Linux, without it, I would not be using Linux at all. But it's not giving me a desktop I can use everyday, exclusively. Hope to come back once to find Ubuntu a reliable desktop.

### Post by AusIV4 on 2008-09-14
I'll try to give you a few points for improvement.

First, Flash tends to be unstable on 64 bit Ubuntu. I've never had any problems with 32 bit. On 64 bit, I've found that Flashblock is incredibly helpful. If you're only running the flash document you want to see, it's a lot less likely to crash than if you're running that plus several flash ads.

I use Amarok for music and Kaffeine for videos, but I think the key point here is the Xine engine. Rythmbox and Totem, which come with Ubuntu, default to the GStreamer engine, but I believe both can be changed to the Xine engine, which is more reliable in my experience. I use VLC for DVD playback, because it seems to have the best menu support.
