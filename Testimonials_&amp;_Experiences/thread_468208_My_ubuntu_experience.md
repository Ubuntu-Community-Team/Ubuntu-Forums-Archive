### Post by Epilonsama on 2007-06-08
I'm a 18 year old Computer Enthusiast. I have a liking of all kinds of Free/Open source Software, like Firefox or Open Office.  But i wasn't sure i could try a Linux distro cuz im not fond of messing with my system to much cuz If i messed up i wont be able to get a new PC.    I started using Live CD distro but it wasn't that good cuz it was kinda slow or didn't have full support of all my hardware.  One day I came across this Program that installs a Linux Distro like if it where a .exe program, so this program was Wubi and the Distro was Ubuntu.  At first i used an older version of Wubi but it didnt work, i was angry but i didn't give up and thank God a new version of Wubi come out, and let me tell ya it installed Ubuntu with out any problems.  When i started using Ubuntu i was like a kid in Disneyland cuz everything was new to me, I started messing around with Ubuntu until, I messed up big time trying to repair a screen resolution bug.  I reinstall Ubuntu using Wubi and i was able to fix the resolution problem but messing around one more time i messed up big time and this time i wanted to try Kubuntu and so i did, I installed Kubuntu to see how it was and for the moment I'm amassed of its looks, but i messed up again and made the kicker unusable.  But instead of uninstalling Kubuntu I was able to reinstall just the kicker saving me time and with a new mistake I'm learning how to use Linux so one day I can make a real partition and install Linux there.
