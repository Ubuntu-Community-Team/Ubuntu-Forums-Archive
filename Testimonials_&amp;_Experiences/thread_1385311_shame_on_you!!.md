### Post by cariboo on 2010-01-19
**Please keep it civil**

You aren't supposed to run Firestarter all the time, it is only a tool to help you set up the firewall, a combination of netfilter and iptables. Firestarter needs to be run as root, which is another security risk. The preferred tool now is ufw/gufw as firestarter hasn't been updated since 2005, the Debian maintainer adds bug fixes, but that is about it. 

The only thing watching Firestarter blocking attempted connections does is make you very paranoid.

If one of the only reason you installed Ubuntu, was because of Vista's UAC, you installed it for the wrong reason, as the security model in Linux is much tighter than Windows, and you need even more authorizations than Windows.

If you want to use something other than Windows, and you don't like Ubuntu, there are several other major distributions, I would suggest you try Fedora, OpenSUSE or Mandriva
