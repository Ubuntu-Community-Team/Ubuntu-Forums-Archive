### Post by Duncan J Murray on 2009-09-22
Just thought I'd pencil some thoughts as to why I think it's great:


AUTONOMY

The operating system is under your control.  It doesn't do anything unless you ask it to, it doesn't bother you with things that it thinks you should know about.  It simply boots up, and waits for your command.

Windows used to infuriate me when it would install things without me asking (some updates), allow software to add processes to start up (and not allow me to remove them eg quicktime, realplayer etc..).

SIMPLICITY

A clear elegant layout, and nothing more (is needed).  You have what you need to access, and there's no extra faff.

SPEED

It's quick at booting!  And once it gets to the desktop it doesn't spend 2 minutes further accessing your harddrive.

ELEGANTLY CONCEIVED

I think it's great the way the operating system feels as if it works as a whole.  From the basic user rights over files, to the synaptic package manager, and the seamless integration of wireless.  It's as if the operating system was planned, rather than have parts bolted on.

PHILOSOPHY

No more need be said.



I've written this in the hope that someone involved in ubuntu read's this and this spurs them on to continue the great work!

All best,

Duncan.
