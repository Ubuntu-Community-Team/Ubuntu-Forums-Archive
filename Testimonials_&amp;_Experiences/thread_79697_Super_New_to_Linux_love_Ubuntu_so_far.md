### Post by fezzik on 2005-10-20
I am about a new as they come. I have built and partitioned etc. several computers. I ended up getting frustrated trying to do the dual boot thing. I didn't want to pay for a resizer for one use and the one on the Ubuntu disk kept telling me that Windows hadn't been shutdown properly but I shut it down as properly as I could. I ended up killing the beast and jumping head long into the world of Ubuntu 5.1 Linux. It took some doing for me to learn where and how to use the terminals. I for real thought I was going to have to get into codes and stuff but it is just like putting commands into dos or something when I finally figured it out. I like Ubuntu. I have been curious about Linux for a few years but never had the courage to jump in. I tried live disks but you can't do a lot of modification to them so anything that wasn't working I couldn't tweak like my wifi card for one. I have a lot to learn but the only place I have windows now is on the restore disk that came with my computer. Bonus for me though I have been using Firefox and Thunderbird for a while for my web and email devices both at home and work. Also I started using Gimp a few months ago. Surprisingly I found I actually almost liked Windows XP when I was running Open Source programs on it. I figured since I like these so much why not go all the way. So far I am glad I did. It may have taken me 3 days to get my wireless card working but that was my fault it turned out to be easy but I was using the wrong driver. OOps Live and learn. Now I know. I am still using the preview so I am looking forward to the full version arriving at my door. I am glad to see such a supportive community. It has helped me make the somewhat intimidating transition into the world of Linux. Wish I could switch at work too but alas that is not my decision. Now if someone can help me get my touch pad to work like it used to with windows I would love you forever. I need my scrolling and want the screen tap to work like a click not like a middle click or half click or something wierd. Anyway so far Ubuntu Rocks. Looking for a good book or books hopefully not too long but won't mind if it is really helpful and still have to decide between program choices. Everyone said you won't have as many programs but the problem I have had as an in decisive person is which Linux, which system (kde, gnome, etc), and which programs do I want to use on it. It is a varitable Smorgasborg of Programs and possibilities. This is great. It would be nice to have a central informative source to go for linux questions and stuff even if it is just one for each distro. I like the page here and just about all questions are covered but you have to really search to find somethings or post again and then it really gets redundant for newbies like me that have tons of questions and are not afraid to experiment a little. 

This was the only Linux OS I tried out that my sound worked with no tweaking.  Soo Cool.
