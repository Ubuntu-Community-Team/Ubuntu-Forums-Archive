### Post by 3rdalbum on 2008-12-06
When I first used Windows, I wanted to test my new computer's speed. I decided to transcode all my MP3s to WMAs as a test.

I downloaded Windows Media Encoder, but it would not encode more than one file at a time. I ended off having to use WME's command-line interface to accomplish the task.

This doesn't mean that the only way you can batch-transcode MP3s to WMAs is via the command-line, it's just that it was the simplest and easiest way. The same goes for installing software on Ubuntu. If you can't bear to use the terminal at all, then there's Synaptic Package Manager. And if you can't bear to use the terminal at all, then you'd better hope nothing goes wrong with Windows; most of the solutions on the Microsoft Knowledgebase revolve around calling DLLs directly from the MS-DOS prompt!
