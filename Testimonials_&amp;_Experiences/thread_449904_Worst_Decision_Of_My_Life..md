### Post by Lucifiel on 2007-05-21
Okay, first off:

Ubuntu is another operating system, it's not Windows Alternate Version. It's not made to replace Windows. If you've heard of Mac, yep, Mac OS X and other o/ses for Mac are similar to Linux. :)

Kubuntu = Ubuntu + another desktop environment. Huh? Desktop environment? You say.

Try this guide, then. It's NOT completed yet but it explains a lot for users new to Ubuntu:

[http://ubuntuforums.org/showthread.php?t=448312](http://ubuntuforums.org/showthread.php?t=448312) 

Beryl is still in beta status. It might not run well for you and as is the case for plenty of beta programs, be warned that installing a beta software that uses and affects certain system files requires precaution. Thus, this might screw up your system files and possibly your entire installation. This is also the case for any o/s like Win XP, etc. (Been using Win since 3.1 and there've been plenty of times when I screwed up my entire Windows 3 hours after installing it and my family got pissed off because of that.) 

Wine only runs a certain variety of programs successfully. If you still need to use certain Windows programs, either:

a) find an equivalent Linux program. Still, what programs were you using in WinXP beforehand? I can see BF2 and some other games but what other programs did you use?  

or 

b) dual-boot Win XP with Ubuntu. For this, install WinXP first. Then, install Ubuntu.

Also, I'm sorry if any Ubuntu fanboys got too enthusiastic and tried to convince you that everything that ran under Windows could run well too in Ubuntu.
