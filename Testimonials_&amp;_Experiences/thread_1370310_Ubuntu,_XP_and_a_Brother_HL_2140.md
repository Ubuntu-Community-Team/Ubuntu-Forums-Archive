### Post by PaulClarke on 2010-01-02
Hello

This morning I bought a Brother HL 2140 laser printer for $99 (Australian). It only has a USB interface. I hooked it up to a D-Link DP-301U Printer Server that interfaces between the USB port on the printer and an ethernet input on a switch.

We have four Ubuntu PC and four Windows.  There is a dual boot so there is a total of seven computers  I first installed a XP.  It took about 20 minutes.  First you set up the connection to the printer box and then you install the printer using the disk supplied.  (16 separate screens in the instruction book)

I reboot into Ubuntu (9.10).  I took a chance and just when to printer window clicked the "new printer" did a search on a network printer.  It found the printerbox and then I installed the printer from the printer select option.  Total time less than 15 seconds.  I printed a page and all looks fine.

My only question is... How good are the drivers that come from the printer select list?

Brother have a downloadable driver.  Do I need to bother?

A real good news story

Must go and install it on the other XP /Vista machines.
