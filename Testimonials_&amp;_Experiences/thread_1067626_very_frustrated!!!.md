### Post by Guille Damke on 2009-02-12
Hi ccstud17,
Welcome to the forums. Don't get frustrated, remember that you are using a different OS than what you are used to have, don't expect Ubuntu to be a copy of Windows, any OS has pros/cons.
Thousands of packages/programs can be installed doing a few clicks from the repositories. You can use Synaptic Package Manager (System>Administration>Synaptic_package_manager). Once there, make sure you have enabled all the repositories, go to settings>repositories and check the boxes for Main, Universe, Restricted and Multiverse. Close this little window and click on "reload". Then you can just search for the package you need. A very usefull package, that installs Java and Flash among others is "ubuntu-restricted-extras", be aware that there may be some copyright issues in some countries.
Good luck, be patient and in a few days you'll be getting used to this Operating System, it is worth to gives it a try.

### Post by wolfen69 on 2009-02-12
the following assumes you are using ubuntu 8.10

first, go to system>administration>software sources and make sure everything is checked. then you can also go to the third party tab and check that also. close and reload when asked.

open terminal, and copy and paste into:
```
sudo wget http://www.medibuntu.org/sources.list.d/intrepid.list --output-document=/etc/apt/sources.list.d/medibuntu.list
```
then
```
sudo apt-get update && sudo apt-get install medibuntu-keyring && sudo apt-get update
```

now, EVERYTHING will be available to you in synaptic package manager. 1000's of packages. just take your time and be patient. once the light goes on in your head, you will be addicted. it can be a challange at first, but like anything else in life, it takes time to learn something new. remember, even with windows, no one is born with the knowledge of how to use an OS.

and PLEASE, ask any and all questions you may have. we are here to help you get over the initial hump. enjoy the ride.
