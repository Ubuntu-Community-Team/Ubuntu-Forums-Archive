### Post by rip_it on 2009-04-24
Well I'll be darned,  I have been waiting for this day, the day I can get rid of windows completely.

On 8.10 I could not use Logmein.com without a reloading error.  With Jaunty 9.04 I installed last night, logmein.com works great.  Apparently 8.10 didnt work with Java 6, only with 5 and that was sketchy.  Java 6 works great with Jaunty 9.04.  Everything loads faster overall. Bootup was faster, beats windows and 8.10.  I like the minimizing of the update manager, doesnt get in your way while your browsing.  Recommending this to my relatives and friends now.  I've gotten everything to install so far so good.  Vlc media player, firefox addons, havent installed Wine yet, thats my next test.  The video card doesnt seem to be recognized with the correct drivers (Nvidia GEforce 9400 GT).  Not getting the higher end effects.  Thats the only downfall im seeing.  Hoping the fix comes out in an update.  I installed it on a Dell Dimension 9100 Desktop.  2 Harddrives with Windows XP. The install was smooth.   Happy days.
