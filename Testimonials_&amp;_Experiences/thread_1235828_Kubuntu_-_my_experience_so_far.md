### Post by slooksterpsv on 2009-08-09
Well I'm a fan of Linux first of all, I love all the tweaking I can do with it - a lot more than Windows. So far, even though I've used Linux back and forth since I was 10 or 11 (starting with SuSE 7.1 I think), it's been an interesting experience.

I installed Kubuntu 9.04 just a few days ago, I tried it before, didn't like it, still thought there were major issues with memory leaks and allocations, but still I thought I would try it again - (granted its not Kubuntu per se, it's Flash, the flash plugin does leak a lot of memory).

I installed PC-BSD before Kubuntu and started to like it a lot, it had KDE 4.0 on it and seemed to run really well, the only caveat that prompted me to go to Kubuntu was my ATI Graphics card (a 4850 not being used to any potential on PC-BSD vs Full on Kubuntu, you choose). So I went ahead and reformatted my computer with Kubuntu. The install took only minutes really, it didn't take an hour or 30 mins or so, it was really really really quick. Upon booting up Kubuntu I grabbed my music from Vista copied it over (I reformat a lot so I don't want to lose it, if I can recover from one drive and not there other, lets make a copy).

Immediately after copying all my music I started Amarok, it asked me to download plugins so it could playback my music. Simple, it asked me, I didn't have to go to the termainal and sudo apt-get everything. That was one thing I didn't mind, but still it bothered me I had to do it everytime I needed something for an application. Once the plugins installed, I started it back up and started with my music.

Next I installed Firefox so I could have Flash support (have to watch YouTube and the likes). Got flash working, but no audio, found out the PCM was turned all the way down (I usually fix my audio settings on startup, but this time everything was going so smooth I forgot). Found out Konqueror could play flash, so I got the nsplugins for it and started using Konqueror a bit. Next I installed Wine, Opera, VirtualBox - getting my basics setup for what I play around with. I started using Wine and that's when I started to wonder if I really wanted to keep the OS. Wine would make XOrg run at 99% cpu usage (and a quad-core running slow cause its CPU usage is through the roof isn't good). So I figured, let's use VirtualBox and grab my XP CD so I could have XP in case I needed Windows apps. I did and got VirtualBox with XP running so I could use NetFlix.

I tried out other apps - like Kontacts, KOrganizer, KMail, etc. but I don't care for them, so I'm installing Evolution for my "Office 2003" type Mail program.

So far I'm impressed with Kubuntu, I may not buy Windows 7 for myself (get it for my family though) just because Kubuntu works really well, gaming is good, apps are running well, besides Flash being a memory hog (not releasing memory up too well), I love Kubuntu, and I keep going into Vista for updates and to check on a few items, but now I'm restarting into Kubuntu more often.

I LOVE KUBUNTU!!! WOOT!!!
