### Post by mdmunoz on 2006-09-02
I have been a Mac user for quite some time, and after participating in a lot of open source stuff going on with OS X, I figured I should try the OS (Linux) where a lot of the code for that open source software comes from.

I had a lot of trouble with my install, mainly because the first time I heard about Grub stage 2 not loading on intel macs was after I rebooted from the live CD (maybe a warning or something on the downloads page, please?), so I was kind of in limbo for a few days with a live cd and an empty hard drive.

That said, I eventually got it working, and ubuntu is awesome. I found that a lot of the functionality I got from my old open-source apps came with the install, and installing new software was mostly point-and-click.

I think ubuntu is superior to any version of windows I have ever used (and despite being a mac user, I have had more than my share of bad windows experience), and that claims that it is not ready for the public are unfounded (except in the case of intel macs, but that seems likely to be fixed very soon).

I am going to switch back to OS X because I love quartz, core audio/video/animation, spotlight, smart folders, webkit, and all of the power those features can offer to open source projects, as well as the seamless hardware integration, but there are no hard feelings. The world is better off because of Linux and ubuntu in particular, and I wish this community the best of luck forever and aye.
