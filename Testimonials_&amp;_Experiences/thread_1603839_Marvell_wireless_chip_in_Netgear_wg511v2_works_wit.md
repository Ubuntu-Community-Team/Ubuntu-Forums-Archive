### Post by ajgreeny on 2010-10-23
Hooray!!!

I have an old Acer Travelmate 2200 laptop using a Netgear WG511v2 pcmcia card, which until Maverick has resolutely refused to work with any of the WPA wireless security settings.

Using ndiswraper, via ndis-gtk and with the WinXP or Win2000 driver .INF file, it was fine using WEP, but I was not satisfied with that security level, so stuck with WPA for our other wireless laptops, and used wired connection for this old Acer.

I have just done a clean install of 10.10, and WPA & WPA2 security works with no problems at all, as far as I can see so far.

Incidentally, I could not keep Lucid on this Acer as it overheated to the point of getting too hot to handle, so have gone from Karmic to Maverick, missing Lucid, though I did try wireless with WPA for the few hours Lucid was on it, but had to give up that without success.  Now not only does the wireless work with WPA, but the laptop also remains cool again, as it should.

Well done to all who have made whatever changes have made this possible. Great work all round, and I'm extremely grateful.
