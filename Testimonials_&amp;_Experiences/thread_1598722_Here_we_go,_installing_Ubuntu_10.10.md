### Post by GeeLo on 2010-10-16
Hello all  ):P


I'm currently installing Ubuntu 10.10 on my old Pentium 3. This was the one that I had Ubuntu 9.10 on, and was using it as a Apache Web Server. I am using the Alternative install CD as I know this will be easier for this system to boot into the installer, and so far the install is going great. The hardware specs are very frugal on this system, and I know that I may have to weed the applications down a bit afterwards, and use a light Window Manager like Blackbox or Fluxbox.

Now I've been using ubuntu since 4.10, and for the most part, the distro has grown by leaps and bounds. I have helped out other users in the community on IRC (freenode) and I have also asked for help over the years. No matter how long I have been working in the many fields of IT (20+ years), there is always something new or to learn. 

Be back when the install is complete. :)
Geelo
