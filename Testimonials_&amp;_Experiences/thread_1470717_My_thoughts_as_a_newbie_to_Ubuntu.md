### Post by untmdsprt on 2010-05-03
First I have been using both Mac and Windows computers since 1989. Because of Win3.1, I HATE Windows with a passion. I've had to use them at work though, and glad I never bought one for home use. I am on my 3rd Mac, and still love it to this day.

After being fed up with iPhoto, and Time Machine, I decided to give Ubuntu a try. I have tried installing Gnome on Apple's X11, and it was an exercise in pulling out your hair waiting for everything to compile. Yes the Gimp is available for Mac, but it seemed to be lacking compared to the Linux version.

Here's what I've done since using Ubuntu:

You have to use the Desktop version if you want to dual boot. It's a lot easier to see what you're doing so you don't accidently erase your main OS. If you plan to single boot your computer, then the alternative one is good.

If you're expecting a graphical interface with the alternative installation, you have to have your computer hooked up to the internet. This one is good if your computer is not on the best side, but you have an old one you'd like to use. Xubuntu is a good one for old computers.

Ubuntu studio looks wonderful from what I've installed on my Mac. So much so that I want to build a PC just to put the studio version directly onto it. At least with this one I can use a regular keyboard and not have trackpad issues. I can also build one to what I want, and have the best studio computer. :)

There are a few annoyances though. First is how the hell do you disable the trackpad on a Macbook 2,1 when you're trying to type?!? It's too damn sensitive when I'm trying to type and I've had to get the cursor back on track more times than I care to count!! I've looked everywhere for step by step instructions to get this fixed. I've seen a few but it's always the person is using another Mac model than the one I have. :(

The other annoyance is trying to get my wireless keyboard to be paired. I'm using the newest 2 battery version from Apple. This one isn't a major annoyance, but would be nice to have to get rid of that trackpad issue.

Overall Ubuntu is a good addition to what I already have. If I wasn't tied to my iPhone for my music and apps needs, then I'd probably consider using Ubuntu more.

**UPDATE:** After using Ubuntu for a week now, I'm convinced that it doesn't suit my needs at this time. It is a great OS that does deserve a place in anyone's home, but I find that I'm still using my Macintosh apps more. When I get ready to buy a new computer, I will compare the cost of a new Mac vs building a PC. I certainly will compare processor to processor, RAM to RAM, and not consider price until the very end.
