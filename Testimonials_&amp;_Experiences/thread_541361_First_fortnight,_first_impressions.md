### Post by Adnams on 2007-09-02
As the title says, I've been running Feisty for a couple of weeks now.

I got into it having bought a PHP book earlier this year by Jono Bacon - who I understand is involved in the making of Ubuntu in some way - which had a Live CD along with it. Out of idle curiosity, I stuck it in the drive, rebooted and was pleasantly surprised with the result.

My wife recently bought a laptop, which meant her old tower (AMD 2500 Barton chip with 512Mb RAM) was surplus to requirements - not for long! Downloaded the latest version from this site, burnt the ISO to CD and within 20-30 minutes had it installed and running.

No configuration was necessary for it to use my broadband connection, which I thought I might need to fiddle with, and within a couple of minutes of booting up, a notification had popped up informing me of a substantial list of system or package updates that were available. Downloaded these and was impressed to see that a reboot wasn't needed on this occasion - something that always gets my goat with Windows.

I use Firefox all the time on Windows and it was good to see it pre-installed and ready to go. None of my hardware proved to be a problem - I ripped a few CDs to MP3, couldn't get them to play and thought my soundcard wasn't working at one stage, until a brief read on these forums showed I needed to install some additional codecs. Well, that and the fact that I had plugged my headphones into the on-board sound jacks, like I do on my main Windows machine, rather than into the card's jacks ...

The main delay was on the partitioning screen - I knew from prior reading it was necessary when installing Linux but I'd neither had nor needed to with all the installs of Windows I'd done over the years. I have two big problems with Ubuntu or Linux in general and this is the first one - I think the Ubuntu installer could or should explain more about the choices and hold the user's hand a bit more during the process. Maybe I'm just wanting too much here though - maybe there's only so much the process can be simplified. I ended up creating a dual-boot, which was not what I originally wanted - I've since read a bit more on this and other sites and have installed it again and was able to wipe the entire hard disk in the process.

My other bugbear has been with the one that gets all us Linux noobs - installing software. Doing so via the repositories is simplicity itself and an absolute joy - I mean installing third-party stuff downloaded manually as a tarball. I just about managed to install Astro Menace the other night by following the instructions on the creator's website, but I think I must have screwed up somewhere as it refuses to run when I select it from the Applications menu. What's more, it doesn't appear on any of the lists when I've tried to uninstall it.

Overall, I think Ubuntu is awesome - it runs smooth as silk on what is now an old machine with, by today's standards, a small amount of RAM. This is an impression further enhanced when you consider that it is completely free, rock solid and coded by a community that does it for the love of it rather than profit. Bring on Gutsy!
