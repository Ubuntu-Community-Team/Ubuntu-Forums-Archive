### Post by kio_http on 2009-11-13
I tested the very old release Kubuntu Hoary. I wanted to see the progress done over the years. Since Hoary was the first release ever of Kubuntu I chose this release. 

I ran it in Virtual Box 3 running on host Kubuntu Karmic.

The installer

Only alternate installer available. this wasn't a problem to me however since I generally prefer alternate cd's.

Speed:
It was quite fast in the Vm.

KDE:
This was what I spent ost time playing with. It came with KDE 3.4 and I was able to compare with my running karmic.

Boot:
The boot process had no splash screen.

Support and repositories as I expected were all dead however. But this did'nt matter as I just installed to have a look.
