### Post by Adoran on 2009-05-18
For Ubuntu and I, Its been a year now - Windows reloads 4, Ubuntu reloads 0. With windows threatening to stop supporting XP, it gave me the final nudge to shall reclaim that day a year back where I have to backup/format/reinstall/patch/restore backup! Except now I face Bug #1 - everyone is using Windows.

The world certainly doesn't make it easy but I was not without hope. The problem - my parents are bright people but have never come across, neither on the PCs they have bought nor in PCs in their work place, Linux. Only Windows. "I have a problem with Word" "I need to make a PowerPoint" their entire lexicon had been proprietarrily hijacked.  

All the potential was there to convert them, all I had to do was lead them willingly.

_How I kicked their habit _
Part 1: Installed three little programs all available for linux and two that are open source. 
[LIST]
[*]Firefox
[*]OpenOffice
[*]Picasa
[/LIST]

Part 2: Changed the default targets for the following file extensions.
[LIST]
[*]doc/docx
[*]xls/xlsx
[*]ppt / pptx
[*]jpeg / png
[*]browser
[/LIST]

Part 3: Explain to parents that I have changed the programs because they were (a) pirates and (b) potential security risk.

Part 4: Wait for serendipity. My niece is born. 
> Mum: "I want to store all my pictures and make sure their safe."
Me: "Mum, I'm afraid that NTFS is simply a black-box of ill-documented filth I wouldn't trust further than I can throw it"
Mum: "Huh?"
Me: "Lets switch to Linux"
Mum: "Whats that?"
Me: "An open source OS that is essentially bomb-proof"

Mum agrees on the condition that it does all the things shes used to.

Part 5: Install it. Show her.

> Mum: "Its got the same programs as windows"
Me: "No mum, switched them months ago for open source versions that do the same thing. Enjoy."

--Happy Ending, clap clap--

Seriously, thank you all! I doubt that anything short of the sexy Ubuntu interface would have caused an aesthetic repulsion. No drivers to configure. No security issues. One sweet, bomb-proof setup keeping irreplaceable files safe in situ and providing a fail-safe backup on my new home server.

Anybody else converted their family?

i have been using linux very casually on/off for the past several years, but only in the past year have i gotten serious about migrating over to it. the most compelling reason for that would be cost. about 99 % of what i want to do with a pc i can do for free with linux, whereas microsoft windows has spawned it's own economy. i also consider linux a more advanced operating system over all windows versions, and it's nature lends itself to efficiency. whereas windows would be the exact opposite, consuming more and more power/storage/etc. as the years progress (like i said, it's spawned an entire economy). 

as i sit here typing this on a $ 2400 dell laptop, had i no appetite for games like Crysis, NWN2, etc....i would have been completely happy with linux on a simple HP dv5 laptop. at times, it feels like i went out and bought an aston martin db9, just to drive it like a grandpa all the way home......
i think this is where a linux installation would have an advantage -- equivalent computing power with less guts at a lower cost.

overall, i do need both platforms to do what i want with my pc's....but both platforms are fairly powerful, and when you know what you're doing you'd be well served with either linux or windows.

Well, there is a fundamental issue here.  Ubuntu users, by nature, learn about how their system works, how to personalize it, and thus have a better understanding of how operating systems work.  If one has never been exposed to, much less used, an Ubuntu desktop system before, how would they know if they would prefer it to Windows XP or Vista?

My clinical psychologist wife, has invested anxiety and shed many tears over the years, learning how to use her past desktop computers and current laptop computer, all which come with the agonizing Windows operating system pre-installed.  Furthermore, she has learned to use Microsoft Frontpage to build her professional web site.  I confess, there are no programs that make changing and updating web pages so simple with no need for understanding anything about HTML codes.

So, I have a used Gateway E2100 desktop computer with UBUNTU and copies of my wife's document files installed, so she can play with it and take it for a few test runs.  No pressure from me, although I would love to rid our house of all-things-Windows.  We have five computers.  In addition to my wife's E2100 and Compaq Presario V2000 laptop, I have a Gateway E4100 desktop and HP Special Edition L2005US laptop.  Both of my computers are strictly UBUNTU Jaunty.

So, our own household may be Windows-free sometime soon.  I talk about Ubuntu with everyone I see quite a bit, but again the fundamental issue comes into play.  Without a person experiencing or seeing an UBUNTU desktop or knowing the differences in security and update processes between Windows and LINUX systems, how is one going to convert to that which they do not understand?  My adult children and my siblings all have bought computers with Windows installed. 

 Even my very computer-based son-in-law prefers Windows, not because it is necessarily a better system but because the expensive software he needs for multimedia projects demands a Windows-based system on which to run the programs.  Although, I doubt if he has every looked into the UBUNTU studio version, and all of the optional software that can be doanloaded for it.
