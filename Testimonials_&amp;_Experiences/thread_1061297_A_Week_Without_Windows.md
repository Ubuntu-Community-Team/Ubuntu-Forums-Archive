### Post by ubuntu27 on 2009-02-06
for ID3 Tags, you can use ["EasyTAG"]("http://easytag.sourceforge.net/"). [EasyTAG]("http://easytag.sourceforge.net") allows you to view, edit and write ID3 Tags.

It supports MP3, MP2, MP4/AAC, FLAC, Ogg Vorbis, MusePack, Monkey's Audio and WavPack files.


It is in the repositories so you can download it with Synaptic Package Manager or with the terminal:

```
sudo apt-get install [easytag]("apt://easytag")
```

From reading around the web, it seems that you have to install another package to get MP4 and AAC support.
The package you have to also install is **easytag-aac**

```
sudo apt-get install [easytag-aac]("apt://easytag-aac")
```

### Post by howlingmadhowie on 2009-02-08
if we haven't scared you off already bushido let me just say that we do all love each other really :)

the last few posts have been a good example of probably the strongest tension in the ubuntu community: the difference between free software and open source.

the free software people (i am one myself, just so you know) tend to regard the way we interact with computers and software as having an ethical dimension. for them we have a duty to use computers and software responsibly and to help others. non-free software removes your freedom to help and be helped and is therefore morally bad. the quintessential free software advocate is Richard Stallman.

open source people stress the power of the development method. for them, free software is probably better than non-free because the development method produces better code with less bugs. They do not mind using non-free software if they consider it better than any free counterpart. I can't think of a really famous example of an open source person. To the best of my knowledge, Linus Torvalds himself falls somewhere between the two extremes. 

the ubuntu community in particular has a lot of tension in this respect. this comes in part from the growth in popularity of ubuntu and the necessary inclusion of non-free software to meet mass market expectations (the flash plugin or nvidia graphics drivers, for example). 

so now you know what's going on here :)
