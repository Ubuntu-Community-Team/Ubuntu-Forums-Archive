### Post by chyptcrypt on 2012-06-11
I want to thank ubuntu, I just recently created a few flash drives that pretty much change personal computing altogether. 

I saw a lot of discouragement and confusion when i first researched it but it works great.

I used YUMI multi boot and made a persistent ubuntu 12.04 alongside ultimate boot cd.
YUMI also made a program to create the casper file for persistence and it's automatic. Add ubuntu first, then dont add the other boot os until you run the program to make it persistent, both programs are at pendrivelinux under yumi. very personalizable drive and with a persistent ubuntu 12

i also made a backtrack 5 persistent drive but it is by itself, and i love backtrack, i think i'll use it on my laptop when i install one there.  i can always keep a third flash drive for storing installed apps if i run out of room in the casper file.

over all, the bootable flashdrive with persistence environment changes personal computing entirely, bringing the physical locality of your /usr to the pebble sized drive in your pocket. plug it into any terminal and you are online.

thanks ubuntu, it won't be long before others take over the methodology and tie it to the cloud things.
