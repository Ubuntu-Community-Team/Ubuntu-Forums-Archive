### Post by Lord Illidan on 2006-01-11
For my A level Computing Project, I have been planning all along to use Linux to develop it. Thus I asked my teacher if I could install it on one of the school computers. She is really very understanding, though she wouldn't let me install it on one of the newer Dell pentium 4s they have because of some warranty.

No matter. I installed Kubuntu on an AMD Athlon 1800. All I needed to do was enter the network details and IP addresses, then resize the NTFS drive.

First off, I was impressed with the speed. It works fine on my 2ghz celeron, but on the Athlon, it runs smokin hot, beating XP on the Dells I daresay, and with all the bells and whistles of KDE, too. The same in the installation.  

Hardware detection seems to be ok, too.

Now, I have it with baghira, to ape the mac and impress my friends who lust for one...

Also, it is those little details they appreciate. Like Kate. I use Kate to write pascal programs. You know about folding... i.e.
```
Begin
[INDENT]writeln ('bla bla bla');
[/INDENT][INDENT]readln ();
[/INDENT]End;
```

becomes
```
-Begin
------------------------------------------------------
```

which can be expanded, if you like. Comes in handy with big programs. Also, it looks a damn sight better than Borland TP 7.

Also, Automatix impressed them. The idea that one program could install everything was well recieved...

Also, synaptic looks awesome from their persepective.. I think I might have to install Kubuntu on more machines.. ah well..
