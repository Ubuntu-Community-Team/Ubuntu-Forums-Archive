### Post by 50words on 2008-08-30
You may recall seeing [this thread]("http://ubuntuforums.org/showthread.php?t=890610") pop up just a couple of weeks ago. After using Windows again for two weeks, I am back to Ubuntu and happy for it.

The main reason I gave Windows another try was my bad luck working with video in Ubuntu. Unfortunately, it really did not work much better in Windows. Movie Maker only sort of works, won't output HD-quality videos (except for full DV), and insists on using the WMV format.

My wife's Macbook works a lot better for video editing, so although it takes me longer to put together a video if I have to do it when she is not around, I'll deal.

I also found myself using Xubuntu (I installed it on a 5GB partition) for about four or five days because I prefer Bluefish to any of the text editors I have tried in Windows. I write two blogs and do my own web design work, so I need a good text editor.

And while I miss being able to use my iPod Touch with my computer, it works even better with my wife's Macbook. Since I don't need to sync it often, I'll be happy sticking with that.

The main thing I could not do easily on Windows was access my office computer remotely. That computer runs Ubuntu and serves as a file server. I use Unison to sync files, but I could ont get Unison for Windows to work at all, so I gave up. Plus, while I am sure PuTTY works, I am not interested in commandline access. With Ubuntu, I can access my server easily and graphically (using Nautilus) any time I want to.

So I'm back using Ubuntu, and sticking with it.
