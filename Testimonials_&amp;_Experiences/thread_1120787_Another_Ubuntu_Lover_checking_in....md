### Post by MysticGold04 on 2009-04-09
I just want to say thanks to the developers and community for Ubuntu. I think it is a stellar distro.  I just want to share a recent install experience.  

Last night after performing a system backup of all my important files, docs, pictures, videos, address book, mail, etc.  I decided to install Hardy on my 4 year old Sony Vaio Desktop.  It is a P4 3.0ghz with 1.5GB of Ram and a 160GB hdd.  I have an older GeForce 6600 PCIe video card.  

The install went great... created the root partition, swap area and separate home partition.. great! Installing... username and password setup.. ok... complete... reboot... hey my username and password does not work... I flip over to my other machine, that is running Jaunty Beta and come to the forum to see if there is a way to reset my password with the livecd or something... then it dawns on me that I can boot to recovery mode.. Voila! problem solved!  First thing I do is install the Nvidia driver by enabling it.. Ok.. I'm half thinking I'm going to have an issue once it installs, but to my suprise I didn't. The display came up perfect! (on my other machine, I had to create a custom x11.conf file) Wow... then I proceeded to do light customization, install Thunderbird and restore my backed up mail... worked like a charm.  

I realize that everyone will not have this type of experience, but it is possible if you be patient and not give up trying to get any issues you may have solved.  I am no expert (yet) but I think the Ubuntu team definately has a winner here, I hope it just gets better than it is... Is that possible?? :mrgreen:
