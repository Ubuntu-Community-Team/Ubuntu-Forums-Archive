### Post by dashford on 2007-09-05
I have no formal computer training but enjoy messing around with new things . I like Ubuntu for the most part and could almost switch over completely except for a few major issues.  I cannot recommend it to a regular user who simply wants everything thing to work.  Those types of users I believe are the majority.  I would hate to work for Dell and try and explain how to use the system.  It has to be a nightmare.  I use the forums for everything and they are great.  I learned how to do everything from them.  I  don't think you can install the system without a Windows computer nearby to connect to the forums for help.  I dual boot and use Windows XP for my primary work.  I use Ubuntu for Internet and CD/DVD burning and ripping.

I have installed the system on several systems

Dell Latitude D505 laptop,1 gig ram, Intel internal wireless card,, dual boot with Windows XP pro (Primary computer)
Dell Latitude CSx laptop 128 ram, Dlink WNA-2330 PCMI card (Formally junk laptop too slow for Windows XP)
Emachine T1440 desktop, 256 ram, Buffalo wireless G USB, (Formally junk computer to slow for Windows XP)

The install on all computers was at least as easy as Windows XP and was easier each time.  Each computer is connected to a wireless router.  The wireless setup was pretty easy. The cards are connected to my home wireless router and office wireless.

 First is a list of things that work better than Windows.

1.	CD/DVD burning and ripping has never been easier.  I use K3b for all general burning including mp3 to standard CD format.  Grip for CD ripping and k9 copy for DVD backup copies.  To buy that kind of software for Windows would be almost $200.00.  Nero,Roxio, and other major programs of the like install so much crap. Then just when you think you bought the version that does it all you suddenly have to upgrade to super gold platinum edition for another $60.00.  Window Media has so much clutter built in that I can't figure out how to use it consistently.

2.	No anti virus.  Everything is so much faster.  With Windows XP you need about 1 gig of ram to run Norton, Outlook, and Office apps smoothly.  Vista needs about 2 gigs.

3. 	Stability.  Windows always seemed to slow down about 30 percent and weird errors start popping 6 months after a new install.  Ubuntu has stayed constant for a year. 

4.  	Connecting to Window shares and internal networking is really easy.
 
4.	Customization and Themes work better and don't mess things up.

Want I don't like and why I cannot recommend it to the average person.

1.	Printing is horrible and has caused me weeks of pain to the point of having to leave the system in the past.  I have learned to accept this major flaw.  Printing has always been one of the primary functions of a computer and just baffles me that a fully functional Office suite has been created that pretty much replicates Microsoft Office. Networking is pretty easy. And I can burn CDs and DVDs six ways to Sunday using 4 different drives internal and external that I found in a junk bin but I can print a one page from my Brother MFC 420cn.  I have looked up everything I could on the fourm, downloaded all the Brother Linux files but I never had any success.  I was able to network on to my work HP printer and that worked.  I tried a Sharp printer but it did not work either.  There has never been a printer that I could not connect to in Windows.  I cannot recommend the system to somebody with the caveat that they have to buy a compatible printer.  I cannot accept any excuse for that. Too many amazing functions are built into this software for printing not to be easy.

2.	Your average person would want embedded Flash, Windows, and Quick-time to work seamlessly.  For me I don't mind as much but when I let someone spend and hour on my computer getting the feel of the system they figure out pretty quick that some of their websites don't work as well.  That turns them them off and I have to try and explain

3.	Compiling tar files or something like that.  All that is hazy to me.  If is it is not a deb file I have no idea maybe there is a better way to install from a tar but so far it seems like way to much work.  I go to Synaptic for most of my stuff but every once in a while I download a tar then I am screwed.  

4.	Finding installed files.  I have installed all kinds of stuff but half of it does not go in to applications or anywhere.  It seems kind of hard to find them and often times I can't.

5. 	Terminal is ok and sudo bash is your friend but sometime it can get rough and average computer user won't want any of that.  

I think the system is almost there and think my problems should be fixed soon.  I hope.
