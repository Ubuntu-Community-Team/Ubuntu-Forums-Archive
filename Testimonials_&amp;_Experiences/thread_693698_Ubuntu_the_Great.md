### Post by Separ on 2008-02-11
I had absolutely no idea what linux was until last january when I was on my work experience with IT support people.

I first tried out SuSE, it ran fine for a while but gave up on me when I added a new hard drive and so i decided to wipe the partition and go back to XP. This was most likely easily sortable but I was still a noob at this time. Tried again but decided I hated it because the update manager is so crap ^^.

Went to try Kubuntu next as I decided from SuSE I liked the KDE interface on it over gnome but it really did not like any computer I put it on. Had a few problems and then it completely gave up the ghost and I had to wipe my system. I tried it a couple more times but still no luck. This was all with Feisty btw. And I should also mention that it was RC3 but at the time I didnt know any difference.

Then I thought I would give it one last go but I heard Ubuntu was a bit less buggy than Kubuntu and that I could switch between Gnome and KDE easily. I wanted to install it 2 or 3 days before the final Gutsy release but I decided to wait and get the new distro. Got the CD downloaded and burned the next day but it went to a black screen :(. Decided to try the alternate installer as a last hope and it worked perfectly, no interruptions. I found the alternate CD a lot easier to work with than the Live CD because it gives you more control over what happens.

And from then, I have had everything working out of the box and managed to install mostly all of my favorite windows games and it runs so much faster than the vista partition I have. I am so glad to have something that finally works.
