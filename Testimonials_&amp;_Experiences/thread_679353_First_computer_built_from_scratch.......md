### Post by Electraglider on 2008-01-26
....without first breaking it in with Windoze. :guitar:

New Computer Project

Components:
MB BIOSTAR TF560 A2+ NF560 AM2+/AM2 - Retail (Qty=1, Price=$79.99)
CPU AMD|A64 LE1620 2.4G AM2 90N R - Retail (Qty=1, Price=$47.00)
PSU APEVIA(ASP)|ATX-CW500WP4 RT - Retail (Qty=1, Price=$34.99)
VGA MSI NX7300LE-TD128EH 128MB R - Retail (Qty=1, Price=$29.99)
MEM 1Gx2|PATRIOT 240P PDC22G6400LLK - Retail (Qty=1, Price=$68.99)

All parts came from Newegg. I paid around $13.00 for shipping and there was a total of $65 worth of rebates. Which I have yet to recieve. Of course I just sent them in this morning.

Operating System:
Ubuntu X64 7.10 Gutsy

Got everything together and only had two bloopers. 
1.The video card wasn't completely seated but the two little LEDs on the Motherboard accurately indicated the failure.
2.Upon subsequent power up the BIOS didn't detect my harddrive. Upon closer inspection I discovered the cable had become unplugged. That was easy enough to fix.

The next thing was to go through all the BIOS settings and make sure everything was set correctly or at least was viewable. 

Now the fun part, installing the Operating system. Run the live CD first and see what does or does not work. 

Everything worked including the network. 

I guess all I  can say is that if anybody is looking for some hardware to build up a new box to run Ubuntu on you can't miss with the above combination of parts. 

I got the computer together on Friday and I spent Saturday playing around with the Desktop effects. I got the cube working, the raindrops and I started hunting around for some 3d games to install.

I installed and enabled the Nvidia restriced drivers and I haven't had a lick of problems. I'm not a total Linux newbie but I'm far from being a guru. I have dabbled with Mandriva but after having played around with that I decided to install Ubuntu and haven't looked back.

For anybody that believes Ubuntu is more difficult than windows, just try installing Windoze from scratch on a machine that has never had an operating system installed and the  hard drive never been formatted. Since I have some history with windows prior to Win95 by comparison Ubuntu is childs play.

This was one of the few times I have spent more time playing around with the operating system features and applications than trying to get everything working. I got the network printer all setup from the linux driver repositories. I imported my web browser favorites from another machine across my network so I didn't have to start all over from collecting websites. My wireless mouse worked without so much as a whimper. I even created and printed the envelopes I used to send in the rebate forms using Openoffice from my new computer. It's too bad there aren't more computers sold with Linux preinstalled. With the exception of installing the operating system I didn't do anything I wouldn't have had to do with a new computer preinstalled with Windoze.
