### Post by gtrtx on 2007-10-06
Hello all, 

I thought I might share my experience with Linux and Ubuntu as it's from the perspective of a long time Linux user.

I started using Linux back in 96(or maybe 97, not really sure). My first install was Debian. It came on a disk I got from a computer magazine. It installed fine, but I was presented with a plain console based login screen. I logged in hoping for great things. You see, the magazine had several screen shots of Enlightenment, which was all the rage back then.  I was then presented with a prompt and had absolutely no idea where to go from there. 

Luckily I had setup to dual boot, and had a working installation of Win95 on a separate partition. So, I would reboot, login to windows, dial up my internet connection and look for information. Well, this was very frustrating and time consuming. I never did get to see Enlightenment on that installation, much less a GUI of any kind. But, I wanted to run Linux and be l337(sic). 

I decided that I needed step by step instructions, so I went out and bought a book. The book was called "Linux Secrets", it was about 900 pages long and came with Slackware 3.0 . I spent a couple of days without sleep reading, installing and configuring Linux with the book as my guide. I learned alot about Linux and computers in general from that experience. 

I finally got things working the way I wanted them, sort of. With slackware, back then, there was alot of config files that had to be edited by hand. Many programs needed to be compiled from source(I remember compiling GIMP, that was an adventure in itself).

My modem was a winmodem, not compatible with Linux, so I went out and bought a new "hardware" modem. I got online, and went to a #linux channel on irc. Everybody laughed at me because my kernel was severely out of date, and my version of slack was old. I wasn't feeling very l337. Back then CD burners where rare(and very expensive) and downloading 700MB on dialup wasn't really an option. Luckily I found a store that carried the latest versions of slackware for about 10 bucks. 

But I stuck it all out, and carried on with Linux for several years. It was only Slackware for me. I installed it on several computers and I upgraded and setup new systems. I quit dual booting somewhere along the way.  As a matter of fact, I missed out on the whole windows98 era of MS, and much of the XP era as well. 

I tried other distros, but always came back to Slackware. It was alot of work sometimes getting things to work. OpenGL was near impossible. I had(still have actually) and TV tuner card that was a real bear to get working. I would say that linux and slackware changed me from a computer user into a computer geek. 

Anyways, in about 2004, I was really getting back into music(playing with bands and such). I didn't have time for all the computer stuff. I still wanted a computer to use, but I didn't want to fool with all the maintenance and  what not. So I did the unthinkable. I installed XP.

Now XP worked ok for me. I no longer cared about my l337ness. I could update and upgrade quite easily. New hardware installed easily. Well, not all of it. My tv tuner card gave me some problems, but I got it working without too much hassle. It basically worked. Not great, but did what I needed it to do for the most part. Mainly I use my computer for entertainment. Stuff like music, movies. I sometimes use it to make demos of song ideas I have. 

I was chugging along with XP and one day I was chatting with some fellows in IRC. We mostly talked about music and such but one day we got on the subject of computers. They told me they where using Linux. I explained that I used to use it, but just wanted the simplicity of XP for now. They told me about Kubuntu and suggested that I try a live cd. So I did. It took me less then 5 minutes to decide that this was what I was going to use. I just backed up all my data files to a spare computer(yeah, I have several), wiped the drive and installed Kubuntu. It did everything that I needed it to do without hassle. The TV card actually installed with LESS issues than I had with XP.  I was amazed as this was nothing like my experience with slackware. My first install was actually easier than any Windows install i've ever done. Everything just worked. 

That was Breezy, and  Dapper came out the next day so I upgraded without issue. I have always kept up with the latest releases, and each one surpasses my expectations. 

Since then, I've installed ubuntu, xubuntu and studio ubuntu. I've been impressed with them all. I just recently built a new system that has Feisty 64 bit on it. It will never have to endure a Windows install. I don't think that I will ever use another MS OS again. Because a 64 bit version of Ubuntu Studio(Gutsy) is going to be release soon,  I'm sure I will upgrade once again, and I'm quite confident I will have good results(the real time kernel in 32 bit Feisty made recording a breeze, so I'm sure a 64 bit RT kernel will be even better). 

Probably one the things I like most is that If I want to "geek-out" and compile my own kernel(which I have done), I can. But I don't have to. I can use the command line, but I don't have to. All my hardware works fine, but I can tweek it if I like. I love Compiz-Fusion, not only does it look extremely cool, but it provides added functionality to my desktop and there are several methods by which I can install it. 

This isn't to say I haven't had any issues. I have run into a couple problems here and there, but nothing that wasn't easily resolved using the the available resources(ie This Forum). I will say that I've had far fewer problems thusfar with Ubuntu compared to the couple of years that I spent using XP. 


Now, I must say, I don't regret my experience with Slackware. It probably taught more about computers then any other OS i've ever used. But as far as simplicity and ease of use, Ubuntu has it beat hands down(to be fair, I haven't tried the more recent versions of Slack, and probably won't). 

Anyways, I could probably ramble on for days about Ubuntu and Linux, but this post is already way too long....

thanks for letting me share,

sean
