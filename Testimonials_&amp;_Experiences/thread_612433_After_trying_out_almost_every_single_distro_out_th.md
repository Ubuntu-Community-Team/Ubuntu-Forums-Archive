### Post by hotweiss on 2007-11-13
My first Linux experience was Ubuntu 7.04 which was filled with me learning the ins and outs of Linux. Feisty had some problems (printing in my case), but rather than that it worked flawlessly. Then Gutsy came along and my system started freezing randomly for no obvious reason. So I gave up on Ubuntu and I went to Xubuntu. I really liked it. It was configurable and snappy. Then my xorg problem began with my DPI settings going haywire which required me to use a magnifying glass to see the fonts. After much researching I found a solution, but some of the fonts were still a bit odd. Then a problem with permissions when extracting a file from a rar archive was the last straw for me. So I gave up on Ubuntu all together. Tried PCLinuxOS, but it couldn't create a proper Grub set up for me and I was never able to boot up PCLinuxOS. Then I tried Fedora 8 a really nice distro that tried to use ATH5K drivers instead of the MadWifi drivers. It's a shame the ATH5K drivers didn't work since they support SuperG 108. None the less, no wifi no go for me. Then I tried Open Suse 10.3 which was even more buggy than Xubuntu. I tried some other Distros but if they did not support my Atheros chipset I could not really use them. Plus, I'm new to Linux so I don't want to spend hours on end trying to figure out how to load the Madwifi drivers in a specific Linux distro when the support community is relatively small. I was going to give Ubuntu  and Linux one more try by giving Kubuntu a try. And boy am I happy after installing Kubuntu. Everything worked out of the box except for my Bluetooth. I just had to add a Toshiba ACPI script to turn my Bluetooth on. And to my surprise Kubuntu has an actual Bluetooth manager that works!!! The KDE desktop is much nicer to the eye than the Gnome counterpart and it is a lot more stable. Although I miss having a session manager and some configuration options. I've even kept Konqueror as my primary web browser; although there are some quirks. Overall the whole Kubuntu experience is much nicer than the Ubuntu and Xubuntu experience. It seems that I'm a Linux user for life now.

PS-I wish that the mouse side buttons were enabled by default in Ubuntu...

### Post by Scheater5 on 2007-11-13
I know exactly how you feel my friend - and my experience with distros goes much farther a-field than even your's, and over a greater period of time.  Yet I still return to Kubuntu.  
And just while we're on the subject, Kubuntu has an even more extensive session manager than Gnome - check under Advanced in System Settings.  And you can use anything in Kubuntu that you used in Ubuntu - they're the same operating system with a different desktop manager and a different set of default applications.  They use the same repositories and you can use "apt-get" to snatch anything you like from your old Ubuntu setup - including Gnome itself - and add it to your new install.  
Welcome to Linux.  Enjoy.

### Post by Scheater5 on 2007-11-14
Under "Advanced" in System Settings you will find "Session Manager."  You can select "start from a saved session" which will add an icon to your k-menu "Save Session."  At any time if you click that icon, when you next log on you will find the same programs you had open at that point.
