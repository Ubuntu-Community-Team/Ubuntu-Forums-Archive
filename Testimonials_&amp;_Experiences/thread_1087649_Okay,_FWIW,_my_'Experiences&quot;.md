### Post by ufusername on 2009-03-05
In case anyone is interested, here is a summary of my experiences with Ubuntu.

My interest in Ubuntu was in running gnuCash for my small business.  While there is a Windows version of gnuCash, it tends to be buggier than the Linux version, and the bugs take longer to get fixed.  So, I set out to see if I could run gnuCash on its native turf...

My first experiment was with an old Dell Latitude with no CD drive and a whopping 128MB of memory.  Using Wubi, I got Ubuntu on the box, and it ran as well as the Windows 2000 that was already there.  And gnuCash ran well.

Buoyed by that success, I tried to run the live CD on a HP Pavilion desktop with 512 MB of memory (and a CD drive!).  The only way Ubuntu would run was in safe mode, and it was incapable of finding the ethernet adaptor even then.

I next tried Ubuntu on a Compaq Presario with 2G of memory.  The live CD ran fine on it, so I intrepidly set out to install.  And ran into the fact that this laptop has a hard drive so fragmented that the install partitioner can't make a dual boot install.  (If you're a glutton for punishment, there's a thread about this on the installation forum.)

So, 1 for 3 with Ubuntu.  Not the result I was hoping for.  I suppose that, with time and $$, the other two systems could possibly be made to run Ubuntu, but not being a computer hobby 'geekie' sort,  I'm not really that interested.  (I suppose a Wubi installation might work on the Compaq, I may get around to trying that someday.)

Anyway, maybe Mr. Gates and Mr. Jobs did provide some service in return for all the billions we gave them ;)

### Post by Ms_Angel_D on 2009-03-05
Wubi isn't the best way to go if your looking for a true Ubuntu experience, it might be good to check out the OS but that's about where it ends. The fact that it's running on NTFS in a virtual disk, provides lower performance, and means your subject to problems caused by the windows system. If you truly want a Ubuntu experience, then I suggest a real dual boot and not a Wubi install.
