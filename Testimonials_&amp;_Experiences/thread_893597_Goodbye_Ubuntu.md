### Post by BiscuitTin on 2008-08-18
Having spent several days tinckering and with Ubuntu following a recommendation and trying for hours to get certain things to work - and failing on most of them, I find that overall very disappointed by the experience.

To be fair, not all of the problems were with Ubuntu, but Linux support for hardware in general, but my impression is that Ubuntu is far less mature, or simply more broken, than some of the main stream Linux distros like Redhat, SuSE and Debian. In terms of installation and configuration, Linux in general still seems has a long way to go to catch up with Windows, but if you can get something to work, then it works very well.

I'm not going back to Windows just yet, as I'm currently giving another distro a try and so far finding it better behaved than Ubuntu. Regrettably though, for now this means goodbye Ubuntu.
