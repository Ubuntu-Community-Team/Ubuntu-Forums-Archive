### Post by ThinkBuntu on 2007-05-01
I've "tasted" quite a few flavors of distros since I began using Linux over OS X, using Zenwalk most heavily, and for now I've arrived at Ubuntu 7.04 Feisty. I think I'll stick with it for a while, because I appreciate the lower overhead resulting from fewer ordinary tasks that I usually have to do with other OSes.

Here's my take so far:

PROS
* Better included themes
* Decent performance
* Extremely robust distrubution: Even when I'm offline, I seldom get stuck without an error explaining why
* Fairly stable
* Network (wireless and otherwise): A+, far superior to any other

CONS
* I have found more stability issues than I did when I used Edgy (6.10)
* No compiz frontend. Even I have a hard time making sense of the options for compiz in gconf-editor, and my only front-end options are wobbly windows and a cube that doesn't work
* The default media player is in serious need of some love. I've switched to Banshee for now and am pleased.
* Performance seems a little slow, but then again I'm used to using Zenwalk with Xfce.

Anyway, good to see progress is being made! I'm hoping that 7.10 erases these stability issues for good. I'll be sticking with Ubuntu for the little things, like not having to restart my network if I booted without an ethernet connection and then plugged one in.
