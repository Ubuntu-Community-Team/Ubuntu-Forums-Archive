### Post by WheelsOfSteel on 2008-11-08
Kubuntu/Ubuntu

I did a fresh install of Kubuntu 8.10 last week and after a few issues I have a fully (apart from int. mic) working system.  I had been using Ubuntu and enjoyed the experience and had no major issues, just had to do a fresh install and fancied a change.

My first experience of kubuntu and the new KDE were very good.  There are a few issues with the GPU which have been mentioned in a variety of places &#8211; however, this isn't enough to deter me from staying with the KDE version for a while.  

Have to say, I do like the GUI and the integration of Konquerer.  Didn't like it intially, and even installed firefox, but now with it's quite likeable.  I love dolphins ftp (via ssh) integration.  Don't like the program start procedure, but that maybe because I was so use to the Gnome way in Ubuntu.  Love the top left corner mouse thing for presenting all the apps across the screen -  big virtual destop user so with the v.d. Hotkeys it;s a breeze to zip around the desktop.  Talking of desktop's not quite to grips with the Plasmoids things yet, but looks promising.  Trying to get some scrolling news tickers on the internal desktop areas, very much wip.

I have been a Linux user on and off since the relatively early days and over the years used a number of flavours.  In fact for quite a few years I have used it exclusively only having ms windows recently as it was pre-installed on the laptop I am using now, but of course I have dual-boot and am typing this on Kubuntu.

Event the wife uses Ubuntu, although she does complain that she can't use the napster client so...  I host my servers: music, web (LAMP), proxy all using Ubuntu so i'm not a casual user.  Having a computer science degree and being in the technology field, does help me to overcome some of the problems that exist, but they have been getting less and less over the years.

With  (Ubuntu) becoming more mainstream, going are the days of running it on low(er) spec hw.  This laptop is a HP Pavilion dv7-1025tx with 4GB of RAM which runs it well on the whole, though I still get intermittent freezes of up to 5 seconds when doing certain operations which I hope get fixed soon &#8211; imagine it's the gpu issue.  Playing blu-ray I need to drop out windows.

Anyway, just wanted to ramble and share my experience as a very content K/Ubuntu user.  And before I get flamed for using Kunbuntu on an Ubuntu forum, I do use Ubuntu on all my other systems apart from this new laptop. ..

PS &#8211; I have had a few 'glitches' getting this machine completely up and running (still have to use an external mic but I can live with that), so if anyone has one of these laptops and needs some help, do let me know.  After all, that's what this is all about and it works very well.
