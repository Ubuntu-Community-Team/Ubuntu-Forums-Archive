### Post by Kristofer Van Orton on 2009-08-31
Before I tried Ubuntu I'd never tried any kind of Linux OS. I've always been a Windows guy and never even really considered using it, and I've also had a lot of experience with OSX since that is what is at my school, but had no knowledge about Linux, hell I didn't even know what it looked like.
About a week ago or so I was looking into dual booting XP and Vista because some of my programs didn't work correctly using Vista. After a lot of thought and research, I figured it would be easier to just dual boot Ubuntu and Vista, and not have to deal with all the boot manager problems that come with XP/Vista dual booting, and I also wanted more expirience in Linux.
Since I got Ubuntu I don't usually use Windows anymore, although it nice to have there (I couldn't live without Adobe if you know what I mean), but for everyday use I stick to Ubuntu. I'm addicted...:lolflag:
