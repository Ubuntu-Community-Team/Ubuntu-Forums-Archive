### Post by canadianwriterman on 2008-05-22
I've been using Ubuntu since Breezy Badger. With Feisty I converted to using it full time and, with Gutsy I bought a laptop from Zareason with Ubuntu pre-installed. For all versions, I have installed late alphas and then did a fresh install with the final release.

Hardy was a little different. I did not install it until the final release. First I did an upgrade from Gutsy (the first time I've done it) and the upgrade went perfectly. However, found it a bit flaky under normal use, so I did a clean install. That didn't help... Hardy feels more like a beta version than a final release. I had all the typical issues like excessive CPU usage and crashing Firefox. The final straw was after a recent update to HAL, Hardy would no longer mount or even read my CD-ROM. THere are lots of others with that issue on the forums and, so far, no way to fix it. As I use my laptop for teaching, the CD issue is a show stopper.

So, I'm going to reinstall Gutsy and wait for the point release of Hardy. Now, my point is that I've never had those kinds of problems with a final release before. My feeling is that Hardy was rushed just to meet schedule and that it really should have been held back a while. I don't think this is going to help build confidence with the general public and the business community... at least those who jump on the new release right away.

Anyone else feel the same way?
