### Post by RichardG891 on 2009-04-28
Don't normally do this but I've been driven to hunting down the best thread to sing my praises on Jaunty, and this seems to be it.:razz: So here goes:

Booting:
Intrepid - boot freezes due to HP Pavillion/nForce chipset + kernel bug
Jaunty - fixed (at least on AC power) and MUCH faster boot time.

Power management:
Intrepid - broken (have to disable acpi to maintain sanity)
Jaunty - fixed, and hibernates and suspends - and, unlike Hardy, recovers from hibernation and suspend as well - bonus!

Pulse Audio:
Intrepid - broken, especially with Java and flash
Jaunty - All fixed

Open office:
Intrepid: another 6 months of v2.4.
Jaunty: 3.0! (OK, so it's the same as 2.4... nevermind)

Compiz:
Intrepid - pretty good, a few titlebar glitches
Jaunty - solid, and maybe a bit faster.

Video:
Intrepid: (nVidia) can't run two monitors without severe malfunction.
Jaunty: Gnome no longer attempts to take control - display manager even offers to start nVidia settings for you. Can run two monitors perfectly... with virtual desktops as well.
Can't speak for any type other than nVidia, though.

WiFi:
Intrepid: Couldn't get it to work, at all, no matter what.
Jaunty: Flawless, even on the live CD before installing it.

Theme:
Intrepid: Yup! Still orange.
Jaunty: browny kinda orange... nevermind, Mint 7 will be as nice as ever; maybe next time.

Many Many thanks for such a great release.
