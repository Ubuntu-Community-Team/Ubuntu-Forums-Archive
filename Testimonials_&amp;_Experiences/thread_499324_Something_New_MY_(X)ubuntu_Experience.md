### Post by Shin_Gouki2501 on 2007-07-12
Funny enough this day on a news side today there was a thread about gobuntu and i felt like writing there about my user experience. But i think here its a bit more apropriate. So lets start!
Im running ubuntu since April and had now to switch back because my GF needs her printer and scanner and i can get it to work ( neither sound..)

+ Samba
+ Free NX
+ FTP Server (although config was a burden..)
+ XFCE fast and stable( only my GF got it hangup 3 times .. i never was able to do this  but shell stayed working)
+ µ torrent runs fine with Wine
+OpenOffice runs great

- install with apt-get ist is great UNTIL u have to config something afterwards to get the app going..
- no Skype Video explain that to my GF :/, Skype Audio seems less good then in windows?
- mp3 ripp Programm ...slow.. kinda not working ( Ripper X or so ..)
- since  2 Days no Sound , it just happend, no change to system  ,just like that didn'T happend with windows on this pc...
- HP scanjet 2400 and HP deskjet 3500 are BOTH not working :/ (although i think u COULD get it to working with some tweaks in the cups config...)
But i dont want to config , if my printer is known then it should just "work" :)

- the xubuntu standard config just moves  files into hidden directory if u r.g. delete something from usb stick. To delete it u have to go to trash...

-its kinda strange ans very annoying that if u install an app via apt-get install it seems so simple but in MANY applications u have to configure somehting ALTHOUGH more then 90% of the users ( my guess) could just live very FINE with a WORKING default configuration

oh i missed a plus:

+ a very big plus is the ubuntu community , the forums here AND the IRC Channel are just incredebile u can get ( as far as i know) in the IRC Channel 24 h help.
Very rare i didn't get there an answer and even then i can post here. So i dont feel so much lost in the new Linux world :)

For the future:
Desktop effects might be nice for somepeople , i rather dont care. Working hardware drivers(printer ,SCANNER!!!) , easy W-LAN are more important IMO. Please make more working default configs!Of course Linux users like to tweak ( maybe because they are forced to ^^) but MANY , MANY people jsut want to "use" a computer and if they are used to the lazy "windows" manners.. it aint too easy to convince them ;)

wbr Shin Gouki

Also i like the idea that in near future i could install my ubuntu to a ( very fast) USB Stick and i can plug it just everywhere i want it to!
