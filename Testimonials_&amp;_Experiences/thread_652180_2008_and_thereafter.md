### Post by Fanless_Puppy_Fan on 2007-12-31
Quickbooks was what prevented my switchover to full-time linux for the last couple of years. I finally sat down to figure out the best solution, and came up with 2 different , but entirely adequate solutions.

1. Purchase crossover from codeweavers.com for $40 and you can run a host of windows programs natively on your linux desktop.

2. Install virtualbox and then install a windows 2000 or XP OS in the virtual box and run quickbooks from there.

I use method 2. You can then set up a shared folder where you store your qb data file and this makes for easier backups and portability. I prefer the windows 2000 pro virtual machine because it seems fast and does shares more easily.

I now have a dual boot and wish I would have chosen to use the whole disk for gutsy.
