### Post by billcauley on 2011-11-12
I'm very pleased with the upgrade to 11.10. Unity 2D is the best interface that I've ever used, and it was a total surprise, as it didn't work with the prior upgrade. Nothing beats doing things like loading programs with two keystrokes (Super + no.). I primarily use Thunderbird and Chrome with Google synchronization, and the LibreOffice suite covers just about everything else. When I typed in a web address with Dash and Chrome loaded the website, I was sold! Problems: I occasionally lose the keyboard, and need a restart while using keystroke combinations in Thunderbird; but as with most things in Ubuntu, a fix will be along shortly. I could go on and on; ...just sign me a "happy user of Ubuntu"!.
[also posted as a reply in another thread before I saw this forum]

UPDATE: 24 Nov 2011
I wrote the above after using Unity 2D for a few day, but knew all along that I missed some of customizations that I had with Gnome.  And then I discoverd Gnome Do.

So now I've gone to Gnome Classic and Gnome Do.  I still have the Super + spacebar access that's in Unity, but was able to have panels running again with the information at a glance that I'd gotten used to.  I was a Happy User before, but now I'm a Very Happy User.

Now if I can only get my microphone to work...
