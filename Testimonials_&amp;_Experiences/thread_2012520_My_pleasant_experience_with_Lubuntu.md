### Post by JRV on 2012-06-29
I installed Ubuntu 12.04 32 bit on an old P4 test system.  Using Unity2D video playback was jerky, so i decided to try Lubuntu.  I installed Lubuntu desktop with the command:```

sudo apt-get install lubuntu-desktop

```
Now the same videos play smoothly.  Lubuntu is without a doubt the way to go for older systems.

### Post by Lars Noodén on 2012-06-29
You can go even lighter if you skip the Desktop Environment completely, LXDE in your current case, and just use a Window Manager straight up.  They can be a little harder to customize, but there is still a lot of room for change.  Openbox is popular and you already have it with Lubuntu.  See the difference between FVWM and FVWM-crystal for a good example of how much they can be tweaked.  AFAIK, the smallest window manager in the repository is [Oroborus](http://www.oroborus.org/)
