### Post by rosvall on 2007-11-23
Hi guys.  This is so nice to post to here again. I started using Ubuntu when warty came out. Before Ubuntu I was using Gentoo and Fedora(redhat). 

At some point I installed windows and started developing software for it. with python,c and c#. I really don't know why. I just somehow started to think that open source/free software was just naive playground and nothing more. 

Few days ago I figured out how naive I was back then. I also got frustrated the fact that I was really using Windows as my development platform. Then at the very moment I decided that no way this can't go on. not anymore. Being pretty much sick and tired about the way how Microsoft works. I won't say they are "evil". Well I will. :D You just can't really trust anything when it comes to Windows and microsoft. No more. I'm gone.

I'm downloading Ubuntu at the moment. (i write this from debian machine). 

Why I'm writing this? I don't know. It is just so great to come back after a year. Feels like i'm finally there where I belong. using and developing open source software. I really missed Emacs ;) Just kidding. What I really missed was this great community. The spirit of free software/open source software.

Now I know. Let's start hacking. Let's make things rock! :)
