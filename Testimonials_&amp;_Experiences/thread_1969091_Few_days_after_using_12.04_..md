### Post by bbmak on 2012-04-29
I am a Lubuntu users, and so far my experience is good and fair.
-I notice that the monitor drivers are improved because before, 11.10, I still have to manually config my monitor resolution via org.conf, but in 12.10, it is automatic now:). 

However some improvements are still needed.

-I try to upgrade my Lubuntu from 11.10 -> 12.04 via the upgrade utility. However, it breaks. Luckily, I have back up all my files before upgrade. I can just format and re-install 12.04. Therefore, I hope Ubuntu teams can work on that process.

-I always keep a computer running windows because some of my softwares are required Windows to run such as turbotax. I know you can run virtualbox, but I do not like virtual environment. Therefore, I hope Ubuntu teams can work with WINE to allow Windows applications much easier to run on Linux environment.
