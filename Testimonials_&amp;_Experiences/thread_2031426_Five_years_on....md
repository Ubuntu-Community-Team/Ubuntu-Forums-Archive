### Post by Irihapeti on 2012-07-22
Five years ago, I found a dvd in a magazine which included an ISO image of Ubuntu 7.04 and I tried it out. I still remember being terrified that the installation was going to wreck my computer somehow. :)

It's been an interesting ride. For nearly two years, I was on dialup. At times, I tore my hair trying to solve problems, usually because I was trying to do something advanced and arcane. I think I managed to solve most of the problems. Anyway, I decided to stick with it.

For three years, I was Windows-free. A couple of years ago, I needed to install Windows again because I couldn't run something in Ubuntu. I've learned that it helps to think in terms of which OS works best in which set of circumstances, and I'm not interested in converting people. Nor do I care to criticise when someone leaves Ubuntu.

One interesting side-effect of using Ubuntu (and occasionally other Linux) has been a growth in my confidence when maintaining Windows. I don't mind tweaking things that previously I'd have been terrified of.

And I have learned about the value of backups/disk images. They've saved me from ruination more than once.

Is Ubuntu granny-friendly? This granny thinks so. :)

### Post by JeffThePCGeek on 2012-07-23
I was first introduced to Linux by my uncle when I was only 12 years old or so.  Even then, I was still highly interested with technology and he knew I loved computers.  He introduced me to Ubuntu 8.04.  I was amazed at Compiz mostly.  He burned me a disk and when I got home, I tried installing it and Grub failed to install correctly and had to get the Windows XP Boot Loader back on my computer.  I then didn't touch Ubuntu until 8.10 hit and I installed it via Wubi.  Since then I have been on and off using Linux.  Since then I have learned so much about Linux that I am now running my own Linux server webhosting company and I try to use Ubuntu on a daily basis now.  When Unity was introduced, I absolutely hated it.  Now that it has finally stabilized, I'm in love with it.

### Post by mickeelm on 2012-07-24
Five years for me as well, 7.04 was the first release I installed (or maybe 7.10, can't remember).

I have been dual booting with Windows for a while during that period though. At first, I just did so for "safety", didn't really want to let it go if Ubuntu should prove to suck, and then it was becuase my bank required an application that only ran on Windows.

With the bank situation changed, I now run Ubuntu only. Not that I hated Windows, I just like Ubuntu way better. Haven't tried any other distros yet and I don't plan to do so in the near future but that might happen one day.

The introduction of Unity was controversial for me as well, but now I've been comfortable with how to use it and I'm loving it. With the HUD, even sweeter.

Well, I'll keep running on Ubuntu for many years is my belief and anyone who hasn't tried should definitely give it a shot!
