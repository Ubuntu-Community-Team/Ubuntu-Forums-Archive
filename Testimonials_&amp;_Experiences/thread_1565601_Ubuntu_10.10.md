### Post by marl30 on 2010-09-17
It is now my first week of running 10.10. No issues so far. All the bugs I've encountered have mainly been third party stuff. I also noticed that some of these bugs have been fixed by the latest updates (and yes I did also report them). So far, the OS itself is very stable. This is surprising, with the amount of beta packages that are installed. Even the Nvidia driver I have installed is the latest beta. From my experience, Ubuntu seems to be moving in the right direction. 

My only issue is getting VMware to install properly with newer Kernel versions (I know this has nothing to do with Maverick, but the Linux kernel. Had the same issue with Lucid when I upgraded the kernel).

I didn't really have an issue with Lucid why I upgraded to Maverick. I just had to check it out for myself, to see what everyone was talking about. So far, I'm completely happy with Maverick. I can't wait for the release date! :D
