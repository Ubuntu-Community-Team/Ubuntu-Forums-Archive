### Post by Sirron on 2009-03-02
If you want ubuntu on your laptop without actually changing the partitions or risking altering the Windows installation, you could use a virtual machine.

For that I recommend [_VirtualBox_]("http://www.virtualbox.org"). In case you aren't familiar with the concept, a virtual machine runs like a program, within windows, but allows you to use another operating system at the same time. It will not affect your hard disks partitions, instead it will create a virtual hard disk, which will appear in windows as a file that you can simply delete if you no longer want it.
