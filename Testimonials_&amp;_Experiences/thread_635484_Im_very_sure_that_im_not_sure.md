### Post by 3rdalbum on 2007-12-10
The most difficult thing about the filesystem on Linux and Unix is finding out that there are only 3 places you need to know about:

/media - where all your other disks can be accessed from

/home - where all the users' home directories are

/tmp - Temporary directory - if disk space gets low, check in here.

Many new users say "Err! I can't find where Ubuntu puts executable files so I can add them to the menu!". That's where all the complaints about the filesystem come in. Look at the screencast in my signature, which tells you all you need to know about adding things to the menu. It's a PoP, really easy once you realise that it's different to Windows.
