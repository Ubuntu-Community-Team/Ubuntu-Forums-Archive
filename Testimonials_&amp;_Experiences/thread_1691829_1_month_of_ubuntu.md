### Post by robro on 2011-02-20
I have had ubuntu for about 1 month and it is by far the best thing I have ever downloaded, the install went flawless and fast too, about 45 minutes in total, had some problems at first (because of my ingeronce, not ubuntu :lol:), and the first thing i downloaded was "[teeworlds](http://www.teeworlds.com)", on windows it would lag so much it was unplayable, on ubuntu I was amazed at how smooth it ran, the next day I looked in the repos for some software, and i was yet again amazed at how many things were FREE to download, and about the 2nd week i was messing with the config(?) files manully and well it messed up ubuntu completly :lol: but with the help of the forums I was able to get it back with 1 simple command :), i made a short list of why ubuntu it better than windows:
```

dosn't automatically download updates every time i connect to the internet
fast boot up
can use it as soon as it boots up
has every thing needed for day to day use preinstalled
99% of applications are free and are native to Linux
alternative applications for almost everything you had on windows
what ever dosn't run natively to Linux theres wine
what ever dosn't run with wine theres Virtual Box
you can change the desktop background unlike in windows 7 starter
and it's all FREE

```
:lol: anyways i want to thank the ubuntu community for there help, and the developers of ubuntu, Nice job :)
