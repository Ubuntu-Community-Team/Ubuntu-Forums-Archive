### Post by aljoriz on 2009-09-12
After happily using ubuntu for some time and finding that it crashes.  It does happen but not as frequent as say Windows98.  

My initial flavor was Ubuntu, then changed to XUbuntu (after not being familiar) I went back to Ubuntu.  When I found out the following for XUBUNTU:
a.  the mouse right click is very useful to access most used apps and log off
b.  Scroll mouse wheel to hide apps

Is it just me or does majority of here switched from one flavor or another?
I formatted the drive when changing OS, took a bath and the installation was finished.  Yes, I know that you can add Xubuntu to a Ubuntu OS.

So in as short as 3weeks I have moved from XP>Windows7>Ubuntu>Xubuntu>Ubuntu> and finally XU again.  

I find Ubuntu and its various versions fun to use.  Trying to make a webcam work in ubuntu or finding a problem and fixing it via google and ubuntu forums; solving these give one a sense of satisfaction.

ubuntu may have flaws like crashing but that does mean its inferior to windows.  After all, what kind of OS allows you to touch the system file when the OS is running and no I don't advice touching the system file of ubuntu.

Ubuntu came to be what it is due developers, the user and finally the community.  The internet has changed how operating system works but in the same manner it changes how we use softwares. Seeking help has never been a problem with Ubuntu.  

The community made it a great open source OS!
