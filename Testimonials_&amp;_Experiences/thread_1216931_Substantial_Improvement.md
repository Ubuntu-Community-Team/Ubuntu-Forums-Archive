### Post by Aphotic on 2009-07-18
I have been a fairweather linux user for a good five or six years. It all started with my first boot into knoppix, ranging all the way up to compiling Gentoo (ick). I have always loved the idea and certain aspects of linux, but there have always been petty little nuisances which have driven me back to windows.

I think the Jaunty install I have on my Lenovo Ideapad S10e is the first linux install which has earned a right to stay and not be turned back to windows.

The Ubuntu community has provided fixes for the very short list of out of box issues I did have.

The ALSA Update script solved my internal microphone issue.
After screwing up GRUB trying to make a bootable USB drive (Now I know to use Unetbootin) a reinstall of 9.04 netbook remix fixed my lack of 3D support. (No idea what the issue was before)

My only complaints at the moment are how poorly my Intel GMA945 is performing with graphics. I know it is a weak chipset but it should still be able to handle Gridwars and some old WINE-run Win 95 Games. I have read this is related to rewrite of Intel drivers so I will just wait for the appropriate performance fix.

The netbook remix has some window issues which are minor but irritating. Desktop effects mess things up royally, some fullscreen apps cannot run fullscreen, and I've been having some odd little glitches with the menu manager. It will keep jumping to a certain menu item while I am in the middle of editing another area. Also, creating a new Menu Item will only create a copy of an existing one.

Despite all my experiences with Linux my knowledge of it is still strictly superficial, so here are a few things I think it would be neat to see developed.

-A Linux application whose role is similar to the windows application GMABooster (Google it) it allows software overclocking of common intel graphics chips.

-CPU Manipulation Software for Atom based processors. It would be nice to either Overclock or Underclock my netbook according to my performance or battery needs.

These very minor issues aside, I'm very pleased with Jaunty's performance. I can run Office as well as do anything I could do in windows before and more. (Especially at once, I have much lower resource usage now.)
