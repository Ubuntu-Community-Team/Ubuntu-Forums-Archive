### Post by wandalalakers on 2010-05-09
I was a Kubuntu 8.04 user and installed Kubuntu 10.04 two days ago.
I have to say that I'm really disappointed.

Here are the bugs that I've come across.

1. Dolphin crashes when deleting files and doesn't prompt with an "install debug packages" option. (kde 4.4.2)
2. Installed kde 4.4.3 via ubuntu ppa and the print screen doesn't work (I had to fix it manually)
3. k3b 1.91.0 can't continue multisession DVD-Rs. (kde 4.4.2 and 4.4.3)
4. rss-glx screensavers do not show up in the control panel when installed (this is a know kde4.x problem)
5. Firefox cannot backup bookmarks.html or bookmarks.json (kde 4.4.2 and 4.4.3)
6. Dolphin is very slow when deleting files. (kde 4.4.2 and 4.4.3)
Note: I installed kde 4.4.3 via ubuntu ppa to see if any of these problems would be fixed but they weren't.

I'm reinstalling 8.04.4 tonight.
I'll try 10.04.1 in two months to see if these bugs have been fixed.
If not, I guess I'll try again with 10.04.2.
