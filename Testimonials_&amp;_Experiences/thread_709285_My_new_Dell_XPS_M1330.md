### Post by curvedinfinity on 2008-02-27
I can hardly believe it, but I've been using Ubuntu for about two years now. I have to use Windows XP and Mac OSX regularly for business, but I simply feel most comfortable on Ubuntu now.

Just recently, however, my family decided to move my Ubuntu desktop to the living room to be used as an HTPC (which it works great as). This left a big gap in my life because the only computers I had available for my development and normal computer usage were a big Macbook Pro and a big HP laptop-monolith.

I decided it was time for a new computer -- one with Ubuntu. I wanted a laptop, so I finally could bring Ubuntu anywhere I went. I wanted to be able to play games. I wanted it to be pretty (I do a lot of creative work, and surrounding one's self with inspiring things is beneficial to the creative process). I also wanted everything to just work. These conditions meant that I had just a few good choices:
[LIST=1]
[*]Dell
[*]System76
[*]Zareason
[/LIST]

I spent a lot of time thinking about all of the different factors, and eventually purchased a Dell XPS M1330 n-series. I'm writing to you on it now. :)

I ended up going with the base 1.66 GHz processor, 1 GB RAM, GeForce 8400 GS, 120 GB HDD, normal screen, and 9-cell battery.

I just received it yesterday -- exactly 8 days after my purchase. Opening the box was a real treat; the XPS packaging is superb. Inside the box Dell shipped, there were two smaller boxes, each white with a pleasing gray graphic design with the XPS logo. One of the smaller boxes contained a very high quality black binder with documentation, a CD wallet, and a cleaning cloth (with embossed XPS logo pattern). The other box contained a black canvas magnetically clasped wallet, which contained the actual computer, battery already installed (and later I found, fully charged).

I turned it on, and, of course, it just worked. The rest of the short setup process is Ubuntu history.

Now that I've actually been using the computer, I took some time to try out different applications, to see if I had any issues with performance.

In general, the processor is more than fast enough for general desktop usage. I've taken the liberty of installing preload, which makes a noticeable difference in application start times, which further makes the whole system snappy.

Running desktop effects, most OpenGL applications are sluggish. Running some tests, I found that the 8400 GS is definitely fill-limited to under four 1280x800 fills per 60 Hz frame. Considering desktop effects generally need about two full-screen fills per frame (to render the background and then the windows on top), it was obviously the thing keeping my games from running well. After I turned the desktop effects off, all of the games I played ran great. I love playing Spring, so this was a relief for me. Turning off the effects every time I run a game is a worthwhile compromise for me.

The webcam works great out of the box with V4L2. The mics need to have the default input source set to the analog source in the Sound properties dialog, and you have to crank the digital, digital-1, digital-2, and input volumes.

The 9-cell battery is phenomenal. With an internet radio station going (meaning wifi and sound), desktop effects (meaning graphics card usage), 40% screen brightness, and normal internet browsing, I was on battery power for two hours, with 55% battery remaining. -- It would be easy to imagine this battery lasting for 7 hours while running economically.

That's it for my initial impressions of this great little laptop. To kind of sum it up, I had no surprises with the purchase and am very satisfied. I'd recommend this M1330 to anyone with similar needs in a new computer. :)
