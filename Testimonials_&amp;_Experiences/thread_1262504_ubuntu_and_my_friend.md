### Post by amicus_curious on 2009-09-10
I'm a Windows product developer for a very large software company, not Microsoft, and I periodically dabble with new versions of Linux just to see how its coming. I would say that Ubuntu Linux is as useful for most purposes that I have as is Windows itself. The lone exception for me, outside of the need to substitute Linux compatible applications for what one used with Windows, is the development environment where I use Visual Studio extensively. That is a rare kind of case however and would certainly not affect the vast majority of users of personal computers. 
 
I have noticed, however, that some things do need a deeper understanding of computers in general and do require some problem solving. The OP here touches on that problem, too, in that some savvy practitioner was required to guide the new user through an installation of a pretty common function, i.e. iPod management. In the office environment Linux installations may suffer somewhat from more complex access requirements for Exchange, SharePoint, and web meeting applications, but there would probably be paid IT staff to handle such things.
 
The new user was fortunate to have such help, but it is likely that 99% of potential users would not have access to such and would eventually conclude that Linux was no good.
