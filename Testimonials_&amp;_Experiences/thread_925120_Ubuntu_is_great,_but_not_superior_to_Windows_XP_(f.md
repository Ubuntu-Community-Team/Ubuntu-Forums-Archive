### Post by Duranxl on 2008-09-20
Hi.

Before i start, i'd like to make clear that i'm nowhere near pro-windows. These are some objective points.

Well, to start off i've been using ubuntu for almost 5 months now, and i've got to point where i see why people prefer linux (ubuntu) over windows.
Many people always told me how superior ubuntu is over windows, in every aspect.. but after using it everday for 5 months straight, i have to disagree on the **every aspect** part.

While i've got most of the stuff working on ubuntu, i still feel this OS hasn't matured like windows has.

Here are some points why I've come to this conclusion:

**1) Hibernation.**
After having read tons of pages about all the hibernation problems (_it seems im definitely not the only one_), I was unable to perform a successful hibernation on Ubuntu. I ended up installing s2disk, which works (*most of the time*)
Now even though all linux people always moan about windows having such a ineffective code etc, my XP install hibernates twice as fast as my ubuntu install, and resumes twice as fast too. 

**2) Wireless Networking.**
Even though out-of-the-box-support has improved a lot with the l latest ubuntu releases, there are still lots of problems with wireless connection.
This is one of those issues that should not be present on a so-called matured OS as Ubuntu.
Internet is probably one of the most important thing when it comes to computers.
Regularly Ubuntu fails to detect my network or just won't connect, when windows has 0 problems.

**3) Music Players** 
While everyone always complains about WMP being so bad (which it might be on some fronts), _none_ of the music players i tried on ubuntu have such an efficient library as WMP.
Now don't get me wrong, i'm not trying to point out that WMP is that good, but at least it updates the library on-the-fly, which, for some reason does not exist in any of the players i tried (?!)
Yes rhythmbox updates on startup, but WMP adds the song instantly when i paste it in my music folder.

**4) HD video support**
After trying most of the big video players on Ubuntu, and even after re-compiling Mplayer with the CoreAVC codec (which i use on XP as well), i'm still unable to play 1080p 5.1 videos on ubuntu as smoothly as on XP. I've heard this is due to hardware (gpu) support?
With HD becoming bigger everyday, i think this is a big issue as well.

**5) NVIDIA & Ubuntu**
While i do not have any experience with ATI cards, I do have experience with ubuntu on 3 NVIDIA systems
Well, none of them worked right out-of-the-box.
While the proprietary driver is a good thing, they didn't work on any of the 3 PCs i tried.
Of course problems like this could be hardware related..but the fact is that the average user will have a hard time getting it to work. 
All 3 PCs prompted me with low-gfx-mode upon reboot, even after xorg.conf changes.
(EnvyNG solved most of the problems, though)

**6) Dual Monitor Support**
This might be a part of what's above, but this is another example of an important feature that will most likely give you a headache.
While it might work out-of-the-box on some configurations, you will probably have a hard time.
I'm still unable to use my dual monitor exactly as i want to by clicking on a single button (like on windows).

**7) Stability**
Many people have always claimed this was one of Windows' big issues and one of linux' strong points.
E.g. "on ubuntu, when a program crashes, it will just be the program, not the OS itself".
Fact is, i've had firefox crash on ubuntu by opening a 50MB XML file, to the point where I couldn't do anything in ubuntu.
The Firefox window turned grey and none of the ubuntu taskbars or keyboard shortcuts worked. Not even ctrl+alt+f1.
Also, sending files via Emesene (IM client i use), will cause my whole system to become unresponsive, not just emesene.

**8 ) Bugs**
This one might sound weird to some of you as well.. but occasionally i have to deal with some pretty irritating bugs.
This includes the things i mentioned before like hibernation (even with s2disk) and wireless suddenly not working, but also HDDs that suddenly won't mount, icons disappearing from my taskbar (like WICD, battery meter) and more problems that should not occur on such a big OS.


All this together makes ubuntu feel very 'buggy' compared to windows...to the point where I need to reboot into windows to check if my router is having problems, because i just don't trust ubuntu when it says "Can't connect"

Any comments would be great (if it wasn't a too long read :p)

### Post by owend on 2009-03-24
Hibernate/suspend: I'm still with Hardy, which originally did neither. It does now!:D 

Open (in Gedit, as sudo) /usr/lib/hal/scripts/linux/hal-system-power-suspend-linux, navigate down to the line QUIRKS="" and activate everything, so the line reads QUIRKS=" --quirk-s3-bios --quirk-s3-mode --quirk-dpms-suspend --quirk-dpms-on --quirk-vbestate-restore --quirk-vbemode-restore --quirk-vga-mode3 --quirk-vbe-post --quirk-reset-brightness" (I don't think they're all needed, but I left them all in, belt and braces!)

Save it and it suspends perfectly, restores after the password in about ten seconds. It should also work for hibernate, but I don't use that.
