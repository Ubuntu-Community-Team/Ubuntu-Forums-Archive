### Post by adempewolff on 2009-08-13
I've been using Ubuntu at varying levels for more than 2 years now, and it has been my main OS with very rare boots into Windows since early this spring.  Although I've always liked Ubuntu, last night I had a moment where I realized that Ubuntu was fully meeting and exceeding my needs and that my current setup enabled me to be more productive than I had ever been before.

At that moment I had several firefox windows open each with ~8 tabs and various flash controls, java applets, etc within them, I had rhythmbox open playing music, I had evolution open, I had picassa 3 open, I had several terminals open for various tasks (troubleshooting, top, etc), I had gedit open with several tabs, I had a pdf viewer open, I had the gimp open preparing a passport photo, and I had virtualbox open running XP within which I was running bitpim to get stuff off my phone over bluetooth (usb bluetooth dongle is buggy with current kernel).  Ok, so maybe these are just all the programs that I used in a 20 minutes period but at any given point most all of them were open at once, and virtualbox was running the entire time.  To top it all off Compiz lets me organize and use all these windows very efficiently and with the level of eyecandy that I want, no more, no less.

Somewhere in the middle of all this multitasking I realized exactly how much I was doing at once and stopped myself as I was about to run top because I realized that everything was running fine--fast even--there was no need to run top.  This was a great feeling and one I had honestly never experienced before.  Windows (and windows programs... cough:office 2007coughadobepdfviewercough) had always hogged too many resources, and I had just barely gotten Ubuntu fully set up the way I wanted it (firefox profile in RAM, upgraded to 64bit for the flash plugin, large enough swap to hibernate, etc).  This 'moment" of course happened the day after I bought 2 2gb sticks of RAM from newegg... oh well, virtualbox will put it to good use!

All of this on top of the fact that I consider my hardware fairly modest as it is 2 years old:

Dell Latitude D630
Intel Core 2 Duo T7100 1.8ghz
2 x 1GB RAM (I was a fool and bought a DDR2 533 to match a DDR2 667, Ubuntu still runs great!)
NVIDIA Quadro NVS 135m
250GB WD 7200RPM HDD

Furthermore development on Ubuntu seems to just be getting better and better.  Problems I had 2 years ago when I used it as my main OS for a semester, had just disapeared when I installed intrepid this spring.  When I upgraded to Jaunty even more problems disappeared.  Sure I have problems left (afor mentioned bluetooth dongle, compatibility with university's WPA wireless, etc) but they no longer feel like insurmountable problems--just things I need to find a workaround for or figure out myself before they are included in the next release.

On top of this all everything is free and there is a great community for support.  Compare this to Microsoft where by paying for software you lock yourself into usage rights that make many tasks impossible.  Someday the big software companies are going to realize that punishing paying customers for their shitty business model that encourages piracy is going to be the end of them.  While they continue to live in this alternate world, Cannonical, Sun, and other companies that *get it* will continue to offer superb software and support to paying customers and offering similarly outstanding software for free to the community at large--which in turn helps by reporting bugs and contributing to fixes, a true symbiotic relationship.

I just hope that someday I'm able to give back to the open-source community half as much as it has given to me!
