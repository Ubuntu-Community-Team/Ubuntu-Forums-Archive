### Post by Welly Wu on 2012-12-03
I've been trying a bunch of desktop environments using Ubuntu 12.04.1 64 bit LTS and I have to say that I strongly prefer Ubuntu Unity desktop environment by far now. I did try Ubuntu 12.10 64 bit, but I found that it was not nearly as stable and I had a lot of problems with the package catalogue being corrupted when I removed software packages that I no longer needed. Ubuntu Unity is clean and modern and it is fast and responsive in Ubuntu 12.04.1 64 bit LTS. It's definitely usable right now and I can customize it further with the My Unity or Ubuntu Tweak software applications through PPAs. It does strongly resemble Apple Macintosh OS X although it definitely has been tweaked for GNU/Linux. I like the fact that it is a unique desktop environment specific to modern Ubuntu releases. It also makes my job as a PC technician much easier as I am responsible for providing help and support to my best friend who uses Ubuntu 12.04.1 64 bit LTS on his ASUS notebook PC. When I set the Unity launcher to auto-hide, it gets out of my way and it allows me to focus on the specific software application window exclusively which is quite nice. I like the fact that it is a complete user experience that does not abandon the desktop user like GNOME 3.x.

The other desktop environments are what they are. They are competing alternatives to Ubuntu Unity. I find that they rely on hot corners or side bars to get functions to appear which I don't like at all.

Canonical made a gutsy move by introducing and developing Ubuntu Unity. I think that their hard work has paid off quite nicely with Ubuntu 12.04.1 64 bit LTS as Ubuntu Unity is polished and stable. It has been controversial to say the least, but I don't want to get into that with this thread.

Ubuntu Unity has grown on me. It took a long while to come around to liking it, but I can't deny its strengths. It's the template by which Canonical is going to push their Ubuntu brand to more PC users worldwide.
