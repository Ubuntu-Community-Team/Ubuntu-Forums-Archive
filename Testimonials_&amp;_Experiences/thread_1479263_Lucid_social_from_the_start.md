### Post by Ender985 on 2010-05-10
I appreciate the effort Canonical has made pushing forward the concept of an internet-able operative system with Lucid. When I started using Ubuntu back with 8.04, the beauty of the apt-get command really struck me as a very positive thing. Suddenly, I was able to get whole new working pieces of software, only by writting a single line in the terminal. Seamless integration of the whole OS with the internet is a feature that should be pursued as far as possible, and Ubuntu is doing a great job on that front.

I'm aware that the "social from the start" idea started back with Karmic, but Lucid was set to be the version that would take the advantage of the fully developed social-oriented tools, namely Ubuntu One, and the OS integration with Gwibber, Empathy or Evolution.

However, imho there are several points that make this whole new Lucid experience not so rounded and smooth as it could be:
 
 - Ubuntu One is still completely unable to work behind a proxy. 

 - Gwibber/Empathy need to be easily customizable. I still haven't found a way to turn off the spell correction in Gwibber, or to change the font sizes of the chats in Empahty. One (or both?) of this programs ask me for access to the default keyring after every boot, and I can't find the way to grant them permanent access. Probably, some/most of these things can be done tweaking some obscure .conf file hidden somewhere in the system, but such very basic actions should be able to be changed from the GUI. (And I would appreciate if any of you know which obscure files I need to tweak and how to in order to solve these issues!)

 - The much hated "Envelope icon" still does NOT serve as a minimized-to-taskbar Evolution. Sure, you can start Evolution from there, but if you want to be notified when a new mail arrives to your inbox, you have to keep the application running in the desktop. Or use AllTray for that matter. Talk about missed opportunities here.

 - The Envelope icon and the (x)Username icons do serve simmilar goals, but hold split functionalities that should be better distributed or simply unified. So I can change my offline/online in Empathy status from the (x)Username icon, but if I want to actually launch Empathy, I have to go to the Envelope icon? And what has an envelope to do with a chat/microblogging program at all?

I think Lucid is a great step forward, but I was expecting more on the social front, given that it is to be our LTS for another 2 years. I hope my feedback helps making Ubuntu an even greater OS that it already is.
