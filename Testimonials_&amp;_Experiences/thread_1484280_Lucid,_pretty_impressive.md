Later, Seeker

And if they do break torrents, the mirrors all support resuming downloads as far as I can see. Head to your nearest terminal, cd to the directory you want to download to, and run wget -c <the url of the iso goes here>

You can end that process at any point with Ctrl+C, then when you go back and run wget -c again in the same place (just use the Up arrow to find the command if you don't use the terminal that often, or hit Ctrl+R and start typing the command; the rest will appear magically). It will happily start downloading again from where it was :)

You get to feel like a ninja, too. Isn't the terminal awesome?
