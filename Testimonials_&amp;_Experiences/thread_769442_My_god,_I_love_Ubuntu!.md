### Post by Northsider on 2008-04-26
So I tried to set up a dual boot today: Vista and openSUSE.  Ok, I like KDE so openSUSE is pretty neat to me.  I went through the entire installation for suse and rebooted.  I got the GRUB menu and had 4 listings for windows, none of which worked!  I tried to reinstall the vista boot menu with the restore disk, but that did not work and I kept booting into suse and could not access windows.

Ok, I thought, I have used ubuntu a lot before and the installation/GRUB is a lot better than this.  So I installed Kubuntu 8.04, rebooted, found Linux as well as Vista in the GRUB menu, booted to vista and voila!  I LOVE UBUNTU!  Keep up the great job!
