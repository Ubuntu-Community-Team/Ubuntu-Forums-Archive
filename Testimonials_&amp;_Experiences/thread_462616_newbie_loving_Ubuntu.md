### Post by caro on 2007-06-02
I made the switch to Ubuntu yesterday with a new laptop and a fresh copy of Fiesty Fawn after thinking about it for about a month.  Couldn't be happier!

Despite the problems some people experience, I've had no problems (yet -- I'm sure I will someday).  But within the first five minutes, I had the computer connected to my wireless network effortlessly.  Since then, I've made a few modifications thanks to the very helpful people here on the forums.  Just a few examples:
- updated screen resolution for widescreen laptop LCD
- installed fonts
- found Windows network at home
- installed printer
- able to connect to the office using Remote Desktop
- customized my desktop

The only problem I had was a GRUB error with the kernel updates, but since I had a brand new install, I just blew it away and started over.  I'll be more judicious with updates next time.

I imagine will Win2K desktop will just be fired up when I do my taxes or use the shared printer.

Great job Ubuntu team.  And thanks to the user community for being a great resource.
