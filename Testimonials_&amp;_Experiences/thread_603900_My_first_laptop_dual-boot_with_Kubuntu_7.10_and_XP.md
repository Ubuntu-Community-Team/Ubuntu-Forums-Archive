### Post by Rudi Völler on 2007-11-05
Hi all, 

Decided finally to install Kubuntu on my Dell Precision M65 laptop and here is my experience so far...

**The Installation**
Backed up my files, made sure I had a copy of the XP installation CD and defragmented the hard drive. Then i _upgraded my Bios_ to the lates version. I feel that this point is not emphasised enough in installation instructions for Linux distros in general since it can save you alot of troubles. Anyway, the installation (with the alternate cd) was completely troublefree, manual partitioning of the HD no problem and all major hardware was detected. Installed the restricted Nvidia drivers and only problem was my Canon Pixma MP460 printer/scanner/copier (and still is). I was very impressed with how easy the installation was since I have a home server running on Gentoo that was 10 times more difficult to install...

**The Configuration**
Setting up the repositories was a bit of a problem since the [http://de.archive.ubuntu.com/ubuntu](http://de.archive.ubuntu.com/ubuntu) server was empty for some reason, but I switched temporarily to the ones from Switzerland and everything was fine. I also had a bit of a problem figuring out if I needed any other repositories than the ones from ubuntu. Finally I decided to add the ones from Canonical and Medibuntu and be done with it. An official guide explaining which unofficial repositories are available and why you should need them would be nice. I know this is discussed briefly in the Ubuntu documentation, but I would like to see a more detailed explanation.
Installation of Compiz-Fusion was relatively easy, and everything works fairly well. Installation of Mp3 and windows media codecs was a bit of a pain, but no biggie. 
Playing movies from my samba file server (on the Gentoo box) was a bit strange, since I have to mount the shares in order to play them directly from the server. Any workarounds for this would be great... 
All in all, the configuration was relatively easy and I got the system pretty much set up the way I want it. 

**Overall**
The installation and working out the kinks has been easy for the most part. Im very impressed with my first Ubuntu experience and Im well on my way to become hooked :) 
It was a bit strange however to find no official documentation for the 7.10 version. I used the documentation for previous versions and it made really no difference but I think it looks better if you have documentation for the latest version. And Kopete crashing was not the best first experience a new Kubuntu user should have, but I got it sorted in no time. 

I give the install and config process a grade of _8/10_ and the Ubuntu developers have done a great job here! But I still dont think that my mother could have done this... :)
