I am very impressed with Ubuntu tonight. I had borked up some config files pretty bad so I decided to reinstall gutsy. Fortunately I made a separate home partition so not a big deal, I would just have to reinstall all my programs. Then I found this: [Howto create a list of installed packages]("http://ubuntuforums.org/showthread.php?t=261366"). 1 line on the command prompt, reinstall gutsy, 2 more lines on the command prompt, and I was done. All my programs, settings, everything was the same, save for the config files I messed up. Back in my Windows days a reinstall was something to dread and put off. Ubuntu just keeps making me say WOW! :biggrin:
