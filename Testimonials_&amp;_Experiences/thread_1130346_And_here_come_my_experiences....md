### Post by SR_ELPIRATA on 2009-04-19
I've been supporting many of my client's windows computers and my own and there comes a time when you start to wonder if there's a better OS in dealing with Spyware and viruses and the oh so wonderful blue screen of death.

When I met Linux the first time, Mandrake 7.0, I had run into lots of problems with hardware, but still liked the concept of knowing what is installed on my system and that kind of program description still exists today in Ubuntu.

After Mandrake I returned to windows for a couple of years untill a client of mine got a server without software, since windows as server would cost a lot I gave him the option of Linux. Then, a friend of mine was using OpenSuse and basically, as he knew a lot of linux, I had him doing the OS install and configuration. That was the first time I came to see Gnome, up untill then I was a KDE lover.

I got OpenSuse running on a VM and I kind of liked how it felt but still felt like too complicated for me to use it and therefore... I wouldnt recommend it for my friends nor clients. But I loved how Gnome worked there. That server is still working and have given NO problems whatsoever.

Anyways, another year went by but now I was giving Gnome more thought. Then I remember a gamer friend of mine that told me he used Ubuntu whenever he needed to do payments over the net, and the name stuck with me. But I never came to the site untill I learned of an article in a magazine that claimed they had made a Home Theather computer with a special edition of Ubuntu and some other parts.

So, I downloaded that special edition of Ubuntu and... it didnt work, some problems with the VM software (microsoft, figures). So, not wanting to give up, decided to visit the homepage, downloaded the iso (also for xubuntu and fluxbuntu) and when I saw 7.10 for the first time... I was hooked.

To me, visually, 7.10 was so clean looking and bright, I wanted more of it. As it had gnome I felt familiar with the interface but unlike OpenSuse, the 'control panel' was easier to access. It also found my net connection and I was in the internet easily. Installation was also a LOT different, Ive been carrying the OpenSuse CDs for some time, 5 CD's in total, whereas Ubuntu only uses 1.

Decided to dual boot and see how the system would behave. Then is when the troubles began. Had to manually set the network, but made it work. Had to configure the Samba, but made it work (thanks to these forums). Noticed that whenever problems surfaced... for the most part there were solutions for it. In terms of usability, for me, only 2 major points were in between going full blown Ubuntu and not: 5.1 audio and gaming.

My Abit NF7-S2 motherboard had superb 5.1 sound over windows, but it wasnt working in ubuntu. The gaming part I understood there was going to be troubles and that it might not work as easily as in windows is, so the breaking point was more audio than gaming. After many howtos.. sound would work only in 2ch mode, so I kept going dual booting.

Back then, I kept having CD issues where the CD wouldnt work at all if no CD was loaded when the system was started. Also noticed many hard drive crashes, mostly the windows one but at times the Ubuntu drive would fail as well.

Then 8.04 came, with pulseaudio, but it didnt work well either. Then I read on the pulseaudio site that the implementation done for 8.04 was poor, so I waited for 8.10. System stability kept being so and so.

When 8.10 arrived... there started the new tests. I read on a forum about how many of these sound chipsets could be set by just altering a text file, which I did, and voila! 5.1 sound was working.

Now, I didnt have an excuse to go Ubuntu only, so... I did it. Got rid of windows and went with 8.10. Still I was having stability problems and after some years of changing power supply, video card and even memory... finally thought that it was the hard drive, and in the end... it was the mobo.

A mobo change came, with new CPU and more RAM, and just plugged in the supposedly faulty drive, which is this one, and its working great. Now I just need to start from scratch, because this drive was mostly a test bed, I wanted to see what was out there and now that I know what I want (at least for now) I need to start over.

Also tested the 8.10 in 3 laptops and other PC systems and noticed how easy it has been detecting even the wireless cards, so I have been promoting the Ubuntu experience and those that have tested Ubuntu like it as well. Of course, some of them are not as advanced user as I am (and I think of me as a mediocre to noob user in Linux/Ubuntu) and as such, some have doubts and most of them still dualboot.

I know its just a matter of time that some of the stuff they miss a lot gets incorporated into the software we have available. To me, it has been a fascinating experience, and still is. I now hook every of my systems to whatever printer I come by, and see if it can work or not and each day I get surprised with them results.

I miss the gaming part though. I have been testing with cedega and wine, but there's always something that gets in the way. Even some 'native' games have some bugs. Is part of the process I know, and I'll keep testing these because I want Ubuntu to be better and if we give up... it wont.

I think that, for what most of the computer users do, Ubuntu is ready for the desktop. The vast majority of people I know, only write and print and get in the internet, and they all suffer from the security problems that windows PCs have.

I can only hope they will open their eyes some day and realize that Windows is not the only option. Who knows what they will do, but I do know what I'll do :)

Go Ubuntuuuuuu!!
