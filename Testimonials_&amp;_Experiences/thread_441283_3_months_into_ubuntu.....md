### Post by ginnie6 on 2007-05-12
we have 3 computers at home,,,soon to be 5 (BIG family!). out of the current three 2 are running ubuntu and the other is running xp. so in the last 3 months I've had to reload winodws once, run system restore 2x, and run the computer's restore program once. I've replaced the 10/100 card on it. Run the antivirus and spyware cleaners a gazillion times, and done tons of updates on it. suddenly for some reason I can no longer read files from it on the home network......the ubuntu computers are fine and can share files no problems. Now are you ready for the maintenance/upkeep I've done on the ubuntu computers? Basically none. Just a few updates here and there and I did upgrade the main one to feisty. while it has taken minutes out of my week to keep these 2 computers up and running I have spent HOURS keeping the windows computer running. It is the newest and fastest of the three and yet it is the slowest and most unreliable. If I could talk the teenagers into using ubuntu I would wipe windows off of it. Unfortunately they still need winodws for some school programs and getting them to use wine just isn't going to work.
