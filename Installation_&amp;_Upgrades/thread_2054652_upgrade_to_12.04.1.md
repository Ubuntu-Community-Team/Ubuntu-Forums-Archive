### Post by oldfred on 2012-09-08
It should work, but the upgrade is a big change and with some hardware there are issues. Good backups are important particularly of /home.

I would still download a liveCD of 12.04.1. It can be useful if you need to make repairs and you can test it to see if everything from liveCD works on your system.

After doing upgrades every version (every 6 mo) from 6.06 thru 9.04 and then a clean install of 9.10, I now only do clean installs. I had to do the clean install as I was converting from 32bit to 64bit, but clean install did an automatic houseclean and just worked. But good backups are even more important with a clean install. I also cheat with my clean installs and install to a new 25GB / partition, so I can go back and boot old system if I have major issues.
