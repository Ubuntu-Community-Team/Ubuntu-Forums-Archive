### Post by rsambuca on 2007-10-07
Is it possible for you to back up your /home to the external drive for now?  While you certainly can dual boot by installing in this order, it is a little bit more difficult and will likely take some configuring.  Windows likes to live in the first partition of the primary drive, so you would save yourself some headaches down the road if you reverse the order of things.
