### Post by solcott on 2008-04-18
It sounds like you already have Windows installed on the machine you are about to try ubuntu on, is this correct?

If it is you should take a look at this page before you try installing.
[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)

A lot of people won't say it, but the fact of the matter is almost everyone who tries setting up a dual boot for the first time loses EVERYTHING, so before you try this make backups of everything you are not prepared to lose.

Let me write that down again.

**MAKE BACKUPS OF EVERYTHING YOU ARE NOT PREPARED TO LOSE!** :-)

That page is a pretty good guide for doing the install.  Another option would be to wait six days for Ubuntu 8.04 ([**_New Features since Ubuntu 7.10_**]("http://www.ubuntu.com/testing/804rc#head-1265c643d5763372826b64cae74ee8fe10634f1f")) which seems to have an installer much friendlier to Windows users.

Good luck, let us know about anything else you need help with, and welcome to the Ubuntu forums. :-)
