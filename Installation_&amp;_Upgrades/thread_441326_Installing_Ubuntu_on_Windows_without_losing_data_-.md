### Post by orrorin on 2007-05-12
1. backup important data.
2. use windows disk manager to shrink your disk
3. create a new partition in the free space.
4. install ubuntu in free space

if you are dual booting with vista, ubuntu 7.04 will recognize vista. earlier versions don't do that you need to edit menu.lst for grub. for earlier versions of ubuntu, google "dual-boot ubuntu vista".
