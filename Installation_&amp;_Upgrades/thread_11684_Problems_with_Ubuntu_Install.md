### Post by Rancoras on 2005-01-18
If you browse the burned disc in windows, do you see only one file?  the iso file maybe?  If so, you need to burn it again, using the "burn image" option in your burning software.

Do you have free, unpartitioned space on your HDD?  If not, use partition magic to resize.  As always, back up your important data first.
