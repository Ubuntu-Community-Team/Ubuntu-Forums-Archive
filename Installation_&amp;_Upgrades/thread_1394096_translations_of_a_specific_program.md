### Post by raymondh on 2010-01-30
Try system > admin > language support.  You'll need to log out and log in for the changes (in windows and menus) to take effect.

Also, for directions/understanding, you could always try the 'man' (no quotes) command in terminal.

As an example,  if I want to learn more about blackjack, the game:

```
man blackjack
```
