### Post by grahammechanical on 2016-05-26
Did you install a version of Ubuntu that has reached end of life? If so, the repositories for end of life versions are closed and that could be the reason for these error messages.

It is possible to upgrade from an end of life release to a supported release but it is not simple and it will depend on how many versions you are from a supported release and there is a definite risk of the upgrade breaking in some way.

[https://wiki.ubuntu.com/Releases](https://wiki.ubuntu.com/Releases)

Regards.

### Post by howefield on 2016-05-27
Current versions won't fit a CD so you'll have to use a DVD or a USB memory stick to burn the image. Use this to install over the top of the existing 9.04 installation into the same partitions. If you select "something else" at the partitioning stage but not mark for formatting, your existing /home folder will be preserved.
