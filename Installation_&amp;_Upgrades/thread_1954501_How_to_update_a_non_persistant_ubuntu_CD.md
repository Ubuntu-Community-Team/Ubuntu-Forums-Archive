### Post by zombifier25 on 2012-04-08
You can use UCK (Ubuntu Customization Kit) and customize an existing Ubuntu CD the way you like it using UCK's package manager.
```
sudo apt-get install uck
```
After that, you have to rewrite the .iso to a CD.
