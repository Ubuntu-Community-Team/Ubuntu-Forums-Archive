### Post by Vladlenin5000 on 2014-10-19
Hi, welcome.

So you're not being able to boot from a (supposedly) bootable Windows installation pendrive?
If so, there are a few things you should keep in mind:
1. Booting from installation media is independent from the already installed OS, if any. You'd be doing the same exact procedure even in a brand new clean HDD or SSD;
2. Windows forums are probably a better source of information about how to create an boot from a bootable USB pendrive with Windows;
3. I know it can be done with Microsoft native tools for Windows 7/8; it *may* be possible to do the same for XP with third-party tools (it's up to you to research about it)

You DON'T want to install XP anyway... It's EoL (Enf of Life) and no longer supported. On top of that you mentioned that the factory installed OS was Windows 7 Starter -and- that the owner wants _that_ back. I don't understand why the XP are being equated...
