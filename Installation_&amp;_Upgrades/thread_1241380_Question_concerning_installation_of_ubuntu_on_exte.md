### Post by nhasian on 2009-08-16
The Linux kernel does a good job of autodetecting hardware on bootup, so there shouldnt be any major difficulties.  If you have proprietary display drivers it might give you some isues.  

external hard disks are the same as internal hard disks but with an external case and power supply.  So use it or lose it!  Running an OS from an external hard disk will be slower than an internal hard disk due to the bandwidth of USB.  At least until USB 3.0 comes out later this year ^_^
