### Post by Vaphell on 2010-08-20
in general all apps installed in windows are totally useless in linux. I can run few apps/games from their windows install directory but it's because they don't require tons of registry keys to work and i have a compatibility layer installed (called wine).

If you've decided to use linux, find native linux apps that do the thing you want. Using stuff via wine is not a hassle-free experience and should be used as a last option. If you got plenty of ram you can install fullblown windows in a virtual machine (virtualbox being one of them), which produces better results at a cost of performance penalty. Fancy stuff like 3d is a hit or miss.
