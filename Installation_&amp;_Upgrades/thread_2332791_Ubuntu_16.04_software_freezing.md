I got a solution to the problem at [bleeping computer]("http://www.bleepingcomputer.com/forums/t/622133/slowness-due-to-internal-hardware-breakdown/?p=4057039"). Problem resolved by following these instructions -  Tools->Options->LibreOffice->View menu, turn off "Use Anti-Aliasing"

Just downloaded updates and the problem is back worse. Clicked tools>options and I get a blank page. Restarted the PC and still had the problem. Turned PC off and then on resulted in the problem gone.
