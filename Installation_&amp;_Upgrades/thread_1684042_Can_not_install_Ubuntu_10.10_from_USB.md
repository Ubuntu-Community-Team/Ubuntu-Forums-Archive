### Post by PABlanche on 2011-02-08
My problem is solved, but I report it in order to help others.

I try to install Ubuntu 10.10 32bit on a Compaq nc8230 laptop from a USB drive

I generate the bootable USB drive from a Windows computer using LinuxLive USB Creator.
[http://www.linuxliveusb.com/](http://www.linuxliveusb.com/)

During the creation process, I formated the USB drive.

Ubuntu boots from the USB drive and work properly. BUT, when I ask for installing, it stopped at the second screen (first screen ask for language). The mouse cursor turns without end, and there is no activity on the computer (hard drive not turning, cpu fan at low speed). The only thing I can do is cancel the installation and boot from the USB drive.

The problem was solved by just changing the USB drive. I went from an Imation to a Lexar one (both 4GB). I re-dit all the installation operations and it works just fine.

If you know the technical reason, fell free to explain.
If you are stuck as I was, try simply changing the USB drive.

Happy Linux.
