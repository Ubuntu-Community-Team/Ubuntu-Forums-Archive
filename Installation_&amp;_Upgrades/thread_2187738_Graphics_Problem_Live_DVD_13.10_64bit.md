### Post by grahammechanical on 2013-11-13
The next time you load the live session try this

1) Press Enter at the first purple screen
2) Select the language and press Enter
3) At the text mode Try - Install screen press F6
4) Look for a drop down menu towards the bottom right of the screen
5) Select nomodeset and press Enter
6) Select ESC to go back to the text mode Try - Install screen
7) press Enter to select Try Ubuntu.

Regards.
