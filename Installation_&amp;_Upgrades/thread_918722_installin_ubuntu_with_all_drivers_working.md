### Post by tommcd on 2008-09-13
Since XP is no longer working, and since your father needs XP what I would do is:
 Reinstall Windows XP. Install it on a partition large enough for your needs and leave free space for Ubuntu, then install Ubuntu onto the remaining free space.

If you do not have a proper XP install CD, and only have a "restore" CD that came with your computer, then reinstall XP, and then install Ubuntu. Use Ubuntu's partition manager to shrink XP to free up enough space for Ubuntu.

Write back if you need more help. Since you need XP I would just reinstall it without worrying about virtualization programs.

IF you reinstall XP to the hard drive it should work (including all drivers) as it did before. 

Which do you have, a Windows XP CD along with a bunch of driver CDs; or a "restore" CD that reinstalls XP as it was originally installed on your system when you bought it?

Either way, reinstalling XP should have all your drivers working as they did before. I have reinstalled XP a few times and have not had a problem. Keep in mind that if you have updated drivers since you first bought your computer, then you will need to update those after reinstalling, but you would have to do that anyway if you installed XP inside VM ware.

But will all drivers{like scanner,printer...etc}work.

One more thing,
The drivers for peripheral devices like the printer and scanner will likely need to be reinstalled after you reinstall XP. The printer and scanner should have come with driver CDs which you can use to reinstall the drivers. If you no longer have the driver CDs you can go to the printer and scanner manufacturer's websites to download the drivers. It is probably a good idea to download the drivers from the manufacturer's websites anyway, since there are probably newer drivers available for the devices than the ones that came with the devices on the driver CDs.
