### Post by bcbc on 2012-03-29
You have to use the 'nomodeset' option to boot the first time. For Wubi's first boot see post #8 of this thread: [http://ubuntuforums.org/showthread.php?t=1613132](http://ubuntuforums.org/showthread.php?t=1613132)

After booting you should be offered the choice of installing the proprietary driver (it should appear as an app-indicator - top right) or you can run 'additional-drivers' yourself.

There is some indication that the default proprietary doesn't work properly for this card, but this link may be out of date: [http://askubuntu.com/questions/66201/ati-proprietary-drivers-not-working-with-an-ati-radeon-hd-3400](http://askubuntu.com/questions/66201/ati-proprietary-drivers-not-working-with-an-ati-radeon-hd-3400)
