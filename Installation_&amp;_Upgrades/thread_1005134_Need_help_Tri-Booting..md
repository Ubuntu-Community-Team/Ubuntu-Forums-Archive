### Post by samuraix51 on 2008-12-08
I've got a triple booting Laptop and 2 dual booting desktops.
Easyist way I've found.
1. Setup partitions how you want them.
2. Install Windows
3. Install 1st Linux system - be sure to select / use right partitions you set up for it
4. Install 2nd Linux system - be sure to select / use right partitoins you set up for it
5. configure grub to boot all three
6. should be good to go.
