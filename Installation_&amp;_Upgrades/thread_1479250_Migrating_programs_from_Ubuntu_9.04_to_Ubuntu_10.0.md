### Post by gadolinio on 2010-05-10
One method I know is opening system--> administration--> synaptic package manager, then "file" menu, then "save selections as...". In the appearing dialog, choose location and name of the file you're going to create (which contains the list of programs you have currently installed). MAKE SURE TO CHECK THE BOX "SAVE COMPLETE STATE, NOT ONLY CHANGES". Then save it. Back up that file you've just created, and paste it into the new system. Then open synaptic there, then "file" --> "read selections...", and select the copied file. That way, synaptic will download and then install every program you had in your previous system.
Good luck!
