### Post by darkod on 2010-01-25
As long as you have dvd writer and not only cd writer, Nero should be OK with that.

But this task is much easier in the free ImgBurn (google will find it for you). You open the program, select Burn Image to Disc, and that's it. It doesn't care whether you are burning on cd or dvd as long as the image can fit on the disc. There is no separate selection to make CD or DVD like in Nero.

PS. You can use that program to butn files and folders too, so in fact you don't need Nero unless you are using it for something else too. But this time you need the option Burn Image.
