Ubuntu 10.04 is the current long-term support version.

Since it is supported for much longer than the normal 6-month release the 

developers release "maintenance updates" for the live cd with updated 

packages. This is helpful in some cases, for example if you can't boot 

from the original live cd that was released initially because of a bug 

that was present or if you run an offline computer and want updated 

packages. So basically, if you already have 10.04 you don't need 10.04.2 

since they are both 10.04, but "snapshots" from different periods of time.

Which release did you try to upgrade from? 10.04?
