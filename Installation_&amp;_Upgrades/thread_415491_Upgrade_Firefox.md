### Post by Virgo53 on 2007-04-20
Go here:
[http://www.psychocats.net/ubuntu/firefox](http://www.psychocats.net/ubuntu/firefox)
(I upgraded using option 2 and then number 2 (automated install).
Do not remove Version 1.5 from your system but open Synpatiic
Package Manager and lock the current version of Firefox to prevent
updates for 1.5 from being presented. To check for updates in Firefox 
2.0.0.3 login as root from a terminal and type firefox
Go to the window help tab and check for updates. When finished,
close Firefox and restart it before visiting any web sites.
