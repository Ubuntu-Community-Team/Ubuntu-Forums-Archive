### Post by Quackers on 2010-12-06
It might be an idea to back everything up anyway!
It would also be an idea to boot from the live cd and "try ubuntu" and see how Maverick runs on your hardware.
Upgrading so many versions at once is likely to be troublesome and very time consuming! I would install fresh - after checking Maverick over first.
