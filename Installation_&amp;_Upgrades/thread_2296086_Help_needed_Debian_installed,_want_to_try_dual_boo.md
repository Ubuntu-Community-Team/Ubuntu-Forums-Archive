### Post by Bashing-om on 2015-09-23
james_l2; Hi ! Welcome to the forum.

If I may, as a 1st time installer, may I suggest that you take the easy way ?
Just boot the install medium and select " install ubuntu along side " . Let the install wizard do all the heavy lifting. It will take care of making the room for ubuntu and installing a default system. You only have to respond to the administrative details. Be aware however, in this default, grub's (bootloader) default install is to the MBR of the 1st hard drive and it will be that booting authority. 

Things can and do go wrong for a number of reasons - ranging from operator error to system problems - it is always good practice to maintain good backups.
