### Post by fballem on 2009-04-29
I don't normally participate in bash sessions - whether it's Windows users who bash Linux or Linux users who bash Windows. That being said ...

1/ If you want to run games, and that is the primary use for your computer, then at this time, Windows is probably the best choice.

2/ There are very few Windows programs that don't have an equally capable equivalent in Linux. For example, OpenOffice Writer can be used in place of Microsoft Word, OpenOffice Calc can be used in place of Microsoft Excel, .... I have not found a full replacement for Microsoft Visio and have made the decision to live without it. In addition, there are some challenges with Microsoft Access, but again, I can live without it.

3/ There are some Windows programs, such as Enterprise Architect from Sparx Systems, that do not have a Linux equivalent, but will run under a Windows Emulator, such as Wine. This is a satisfactory solution for my circumstances.

4/ If you installed ubuntu, or any other Linux distribution, with the expectation that it would work exactly like Windows and would run all Windows programs, then Linux is not for you.

5/ If you installed ubuntu, or any other Linux distribution with the expectation that it would work, but you will have to learn some new ways of doing things, then Linux may be exactly what you need.

6/ To return to Windows, start up your machine using the Windows CD and do a complete installation. You must start from the CD so that you can reformat the hard drive.

7/ If this doesn't work, then you should start your machine from the ubuntu start up CD and run it from the live CD. Install a program called GParted (it's in the repositories) and delete the partitions that are on your hard drive. Make sure that you apply the changes. When this is done, shutdown your machine, using the appropriate mechanism (top right corner of your screen, where it says 'Live User'). Then reboot from your Windows CD and see if you can now complete the installation.

8/ If this still doesn't work, then call Microsoft support - they can walk you through some additional processes to get the hard disk to format properly. I had to do that once when I was still experimenting to determine what I wanted to use.

9/ Millions of people are using Linux in various forms as their primary desktop. It is not junk, it's just different from Windows. Millions of people use Windows as their primary desktop. It is not junk, it's just different from Linux. What is appropriate for me may not be appropriate for you. For example, I don't play games, so the lack of game support in Linux is not an issue for me. It is clearly an issue for you.

10/ My primary desktop is ubuntu and I'm quite happy with it. Other machines in the house are Windows, and I'm quite happy with those as well.

Hope this helps,
