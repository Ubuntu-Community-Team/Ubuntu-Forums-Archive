### Post by Old_Grey_Wolf on 2010-01-21
During the Installation you can select a guided install. It will do the resize/repartitioning for you. If you are running XP then it should work well for you. However, I don't recommend that if you are running Vista. I don't know about Window 7.

Based on you question, it seems that you have done nothing to resize/partion the HD so far. If you have already created space for your Ubuntu installation, you need to explain what you have done. It depend on how you created an formatted it.

If you have proceeded further; then, here is some information that may help you. If it is unallocated space then select the option when installing Ubuntu to use the unallocated space. 
If you formatted the extra space in some way, you may need to use the manual install option when using the Live CD.

Answer back so people know what method you used to establish the space for the install so they can help you. I am about ready to retire for the night, so, best wishes.

Anyway, back up important files before you proceed with an install.
