Hi,

So I would like to know if you've used an Wubi install or the Original install.

The difference between the Wubi and Original install is that the Wubi install is executed with the Wubi.exe program for Windows. Also the Wubi install uses the Windows Bootloader, compared to the original install which uses the Grub2 Bootloader.

However, to provide better information to the users of the forum:

Can you go and boot from a LiveCD (that means Try Ubuntu without Changes), and then access the Internet, and then download the Bootinfoscript in the link in my signature below.
