### Post by scradock on 2008-05-05
As I understand it, if you have Ubuntu installed (Gutsy, let's say) and do a dist-upgrade to Hardy you should lose nothing.

If you re-install from a live-CD things are more problematic. Ideally you would need to save all your "data" files (anything you don't want to lose) on a separate partition, so that the installer can wipe-out and re-format the original Ubuntu partition but leave your data partition intact. Then you can mount the data partition in the new install and access it as before.

If you are talking about something more complex, or you don't understand what I'm talking about, ask a more detailed or specific question and I'll try to help....
