There are several ways to create a bootable USB drive. If you are on Ubuntu, I would recommend using the [USB Creator]("http://en.wikipedia.org/wiki/Ubuntu_Live_USB_creator") app. If you are on Windows, try [Unetbootin]("http://unetbootin.sourceforge.net/"). Both are fairly self-explanatory and easy to use.
