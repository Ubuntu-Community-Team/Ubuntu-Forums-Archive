### Post by ajgreeny on 2008-10-09
If you mean the installation Live CD (or even the alternate install CD) you need to set your bios to boot from the CD first.  when you start up the computer look for a message saying "Press DEL to enter setup" or something similar, and try doing that to get to the bios of your computer.

Look also at [http://www.psychocats.net/ubuntu/dualboot](http://www.psychocats.net/ubuntu/dualboot)
