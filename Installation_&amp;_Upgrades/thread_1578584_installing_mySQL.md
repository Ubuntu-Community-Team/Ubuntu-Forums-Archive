### Post by lovinglinux on 2010-09-21
Mot applications can be installed from the repositories. Is not recommended to download stuff from the web, so always check the repositories first, to see if the software is available. Mozilla is a safe source tho and I download from the myself when I want a version that is not in the repositories.

The "equivalent" of Windows .exe in Ubuntu is .deb. It installs with double click, just like in Windows. For more info on installation methods see [https://help.ubuntu.com/10.04/add-applications/C/index.html](https://help.ubuntu.com/10.04/add-applications/C/index.html)

For mySQL see the post above. For Firefox see the [[COLOR="Sienna"]**Installing Other Versions**[/COLOR]](http://firefox-tutorials.blogspot.com/2010/05/installing-other-versions.html) tutorial.
