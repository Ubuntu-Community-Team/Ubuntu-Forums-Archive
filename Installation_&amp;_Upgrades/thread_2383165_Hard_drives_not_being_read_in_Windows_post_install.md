### Post by yancek on 2018-01-22
Your HDD stops being read from UBuntu?  Do you see a message of any kind?  Could you post it?  You also neglected to mention where windows and Ubuntu are installed, are they both on the SSD or one on the SSD and one on the 1TB  drive?  Is the HDD partition you are having problems reading from a windows fileysystem or a Linux filesystem?  From which OS are you having trouble reading/accessing it?  If you are having problems accessing an ntfs partition from Ubuntu it is likely that you have left the system hibernated while using windows and then booting UBuntu.  If this isn't the situation, you will need to post more details.
