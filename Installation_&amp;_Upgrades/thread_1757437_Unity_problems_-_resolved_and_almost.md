### Post by Gstrazds on 2011-05-13
Hi There Just upgraded from 10.10 to 11.04 Unity

The other day my unity desktop self destructed; menus launchers all gone.
The folders and Icons where all that remained in the space.

The user defined workspace was also in the same condition and currently not fixable...

The Ubuntu Classic workspace was not effected.

You can change from Unity to Classic by clicking on the login box; proceed to log in almost; then look down at the bottom of the screen where you will see a drop down list 

- by clicking on that you get four choices; Unity, Classic, Classic -no effects, user defined
- gleaned from a fellow that did a how to with pictures.. lost the reference

The self destructed unity desktop was resolved by placing a control center icon on the classic desktop - opening the compiz settings in Unity - restoring defaults to regain all the menus and the side bar... Playing with contradictory options resulted in a I give up mode.. which is happening in the user defined version which I did not touch.

The second  problem is or was related to Video.. Nvidia Geforce 7150 in my case where;

 Should you have a bunch of windows open.. the one that is brought to the forefront goes blank - returns to visible after it is sent to the background...???? 

It seems to be related to ones choice of frame controls and frame borders. 

I like ambiance controls and ambiance borders; now it's Dust controls and Dust sand borders.

which so far is not causing the window blanking problem

The Unity Desktop uses Compiz with the defaults settings..
