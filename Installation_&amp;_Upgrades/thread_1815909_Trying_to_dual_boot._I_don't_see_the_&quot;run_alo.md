First and obvious question: is Windows 7 booting OK on that machine?

There are a few reasons why the installer might fail to see a pre-exisiting Windows installation. Let's get an overview of your system. Boot up with the live Ubuntu CD and select "try Ubuntu" so that you are taken to the live desktop. Ensure that you are connected to the internet and go to this site:

[http://bootinfoscript.sourceforge.net/](http://bootinfoscript.sourceforge.net/)

Download and run the boot info script according to the instructions there and post the contents of the RESULTS.txt file between [noparse]```
 and 
```[/noparse] tags for clarity - you use the the [img]http://ubuntuforums.org/images/editor/code.gif[/img] button on the message toolbar for this.

Although the script is designed to help investigate boot problems, it is useful in situations like this.
