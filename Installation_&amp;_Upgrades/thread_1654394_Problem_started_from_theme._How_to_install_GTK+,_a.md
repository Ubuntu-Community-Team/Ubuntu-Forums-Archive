### Post by daGrevis on 2010-12-28
Ok, I solve the problem.

If you can't simply drop archive of theme into *System -> Preferences -> Appearance -> Theme*, then unpack it, take folder of theme and move it to username/.themes (where username is your username (for example dagrevis), then open *Theme* (mentioned before) and there the theme would appear! If it don't, then you have a problem... xD
