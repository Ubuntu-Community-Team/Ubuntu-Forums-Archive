I've installed Ubuntu, and it works just fine.  However I now wish I'd installed the KDE desktop instead of Gnome.

---

### Post by ajmorris on 2006-12-21
No you do not have to re-install to do this. Just open your package manager (you should have synaptic) and scroll down to either KDE package or if it is there Kubuntu. You would have only had the option to install it in the initial installation process if your version had it with it such as enterprise.
Before you install KDE or Kubuntu, on your GDM login screen click "session" and make sure KDE is not there as an alternative.

### Post by bulldog on 2006-12-21
Just type in your terminal```
sudo aptitude install kubuntu-desktop
``` and after the install you can choose in your login screen which one you want to boot.
