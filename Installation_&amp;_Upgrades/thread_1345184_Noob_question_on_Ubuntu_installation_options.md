Go for dual boot. Trust me !!

Okays ... wubi runs from within XP and is useful for dipping your toe into ubuntu as it runs faster than trying it from the LiveCD, which runs really slow, because it needs to keep reading from the CD (Which is SLOW)

Side by Side, means you get the choice on boot, of either operating system - this can be really useful if you get a 'nasty' on your XP, as the Ubuntu area will be able to heal it for you (Gee, aren't we kind).

It also means that upgrades are a darn site easier to do.

The dually booted ubuntu will be able to 'see' your XP area - so documents, music, vids.. etc can all be accessed.
