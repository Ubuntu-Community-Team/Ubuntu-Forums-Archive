### Post by cotcot on 2006-09-05
Sure you can install (k)ubuntu on a stick. I have ubuntu on a 1 GB stick. The installation required 700 MB. 
Persistent mode means that when you log out your settings are kept. You can also install programmes with synaptic as you to an a hard disk. These will be on the stick too.
You can see a link to a howto to do this.
