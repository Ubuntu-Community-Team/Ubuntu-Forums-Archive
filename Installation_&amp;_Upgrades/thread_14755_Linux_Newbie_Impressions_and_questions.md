### Post by tdell on 2005-02-09
Well, I have finally decided to give Linux a try and chose Ubuntu as my first experience.

So far I am impressed.

What I have accomplished so far:

Successfully installed a dual boot with WindowsXP.
Network working great I can see and access all my computers.
ADSL internet working fine.
Upgraded the kernel.
Installed Firefox 1.0, Java 1,5, Macromedia Flash, Mplayer, Realplayer and Pan newsreader, no problems.
Modified the Desktop and themes to my liking.
Got my screen resolution close to where I want it with my ATI card (still room for improvement there)
Got all security updates.
Imported all my Favorites and documents from XP

One major problem I have not been able to get my print server (part of my SMC router) to work yet.

An annoyance, my machine will not power off like it does in Windows it just sits there at the black screen waiting fo me to push the power button (OK did I forget to say I am lazy?)


So, all in all I guess I did not do too badly for someone who has never used Linux before. I am sure there will be a few other things I might need to know in the future.

I rarely use Windows anymore (except to print) and hopefully it will be gone totally by the end of theis month.
