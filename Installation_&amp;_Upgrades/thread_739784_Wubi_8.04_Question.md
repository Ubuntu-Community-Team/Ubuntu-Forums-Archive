### Post by chewearn on 2008-03-31
If you want to install ubuntu in a separate partition, you should first resize the Windows partition to make some room.  Use a Windows based partition editor, e.g. Partition Magic.

Once you have some empty harddisk space, leave the space unallocated.  During ubuntu installation, choose the option to install in largest continuous empty space (or something like that).
