### Post by goveia on 2011-02-08
I figured it out my self. The desktop environment, all the settings, my documents, the desktop background are not part of the kernel.

And the kernel is not the entire operating system, its just a small but important part of it.

So  by using a different kernel I am not using a different operating  system, I am using the same operating system and this is therefore  why if I set my desktop background to picture X it remains the same  regardless of the kernel I am using.

And why if I change picture X to picture Y the other kernels are affected.
