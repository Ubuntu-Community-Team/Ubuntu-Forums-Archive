### Post by navneeth on 2006-10-04
There were some updates available for some files and the Linux kernel. The updates were downloaded and installed. Now, whenever I start my computer, I see two Linux kernels instead of one. Something like this...

Ubuntu kernel 2.6.15-27-386
Recovery Mode
Ubuntu kernel 2.6.15-26-386
Recovery Mode
XP

Is this normal? If it's not, how do I remove the older version?

---

### Post by Kateikyoushi on 2006-10-04
It is normal, the old kernel is kept as a backup just in case, might something go wrong with the new one.
You can remove the old with aptitude or synaptic and actually later it will be marked for uninstallation.
