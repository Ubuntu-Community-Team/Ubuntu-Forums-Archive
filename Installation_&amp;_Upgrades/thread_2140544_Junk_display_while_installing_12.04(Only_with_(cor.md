### Post by dsv47 on 2013-04-30
You are using a relatively new model of processor, so you may be having Linux kernel compatibility issues. The Intel Core i3 3220 CPU has caused screen display problems with Ubuntu running on other models of motherboards besides yours. It would be interesting to see if installing Ubuntu 13.04 instead of 12.04 solves your problem (because of the newer version of the kernel in 13.04).
