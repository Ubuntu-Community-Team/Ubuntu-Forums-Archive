### Post by bluexrider on 2011-12-09
you may need to set the BIOS to boot from the CD/DVD first.

to do that, start the laptop and as soon as you see the lettering on the screen hit (usually) F2, possibly F12, select the boot sequence from the BIOS menu and set the the first one to CD/DVD then F10 to save.

insert the Ubuntu disk and reboot
