### Post by gobstopper on 2009-04-11
The solutions seem to vary depending on which version of Ubuntu you are running and which version of VMWare server you want to install.

Personally I have not installed VMWare to a desktop version of Ubuntu, I have always used Ubuntu server (because I want the machine to have as much resource as possible to run the virtual machines). I have found the [Howtoforge](http://www.howtoforge.net/) web site and forums to be a great help. A number of users have posted articles containing the complete installation process for installing VMWare on a number of different Ubuntu platforms. Just enter "Ubuntu VMWare" into the search field and you'll find a number of solutions.
