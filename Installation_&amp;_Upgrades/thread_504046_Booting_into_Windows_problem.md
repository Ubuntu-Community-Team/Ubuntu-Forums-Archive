Hello 

I have installed ubuntu 7.04 onto a 2nd harg drive. The main hard drive is a SCSI containing win xp. I altered the BIOS boot order to be CD, then D, then C. However when I changed the boot order  to CD then C I found ubuntu on D was booting. Can someone advise me pleas?

David

### Post by bulldog on 2007-07-18
Grub is probably installed on the IDE hdd which is (hd0).
Your Sata is (hd1) and I think you have to edit the menu.lst to get your windows to work.
```
gksudo gedit /boot/grub/menu.lst
```
Add some  lines to your windows entry,
map (hd0) (hd1)
map (hd1) (hd0)
rootnoverify (hd1,0)
