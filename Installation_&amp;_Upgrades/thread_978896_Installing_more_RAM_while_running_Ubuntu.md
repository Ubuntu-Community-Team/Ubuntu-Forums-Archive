### Post by mikewhatever on 2008-11-11
It's probably a bad idea to put in more RAM while running any OS. Power the computer off first, then add more RAM. Nothing else is usually required.

### Post by Steve1961 on 2008-11-11
You shouldn't need to do anything.  Just install it and Ubuntu should recognise the upgrade.  Check to confirm in system monitor or cat /proc/meminfo
