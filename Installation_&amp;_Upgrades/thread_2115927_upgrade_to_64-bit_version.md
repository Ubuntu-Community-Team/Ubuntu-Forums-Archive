### Post by howefield on 2013-02-14
You'll need a fresh clean install, but you can install into the same partitions.

It won't be an upgrade, so ensure that you have saved everything you wouldn't like to lose before commencing.

---

### Post by Warren Hill on 2013-02-14
One more point to bear in mind is drivers.  If everything worked out of the box with Ubuntu you have nothing to worry about but if you had to install 3rd party drivers then check there is 64-bit support for these.
