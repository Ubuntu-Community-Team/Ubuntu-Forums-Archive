### Post by coffeecat on 2009-11-15
Point of information: you do not have a root password. The first created user account has administrative privileges and its password can be used to gain temporary root powers. Useful link:

[https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)

If you have a keyboard problem you might consider starting a new help thread with a different title in order to solve this.
