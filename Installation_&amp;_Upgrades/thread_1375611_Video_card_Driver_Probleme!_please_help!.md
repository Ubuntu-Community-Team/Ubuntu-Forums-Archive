### Post by Mark Phelps on 2010-01-08
It worked in 8.04 because, back then, there WAS an ATI driver for your card.  Since then, ATI has dropped support for your card (along with lots of others). so now, there is no ATI driver you can install.

If you're seeing a graphical desktop, you already have the default open source driver installed -- and that is all this is available now.
