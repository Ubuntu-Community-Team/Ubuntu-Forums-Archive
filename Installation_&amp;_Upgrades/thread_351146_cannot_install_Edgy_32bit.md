### Post by bscbrit on 2007-02-01
If you are using a CD that you made yourself, try burning the CD at a slower speed and then re-installing.

### Post by bscbrit on 2007-02-01
The point is not to burn at the maximum speed - there is a greater chance of a write error on the CD.  Write the CD at half maximum speed.  The time you lose burning it you will save during the installation.

### Post by bscbrit on 2007-02-01
It should be OK, but if you have bought cut price CDs, go down to 4x or even 2x.
