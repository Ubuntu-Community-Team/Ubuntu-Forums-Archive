### Post by Partyboi2 on 2008-09-29
To run the xubuntu live cd, you need 128 mb[]("http://en.wikipedia.org/wiki/RAM") to run or 192 to install. If you install using the alternate cd you will only requires  64 mb for the install process.[]("http://en.wikipedia.org/wiki/RAM")
Once it has been installed they recommend a min of 192mb but for better performance 256mb

> To run the xubuntu live cd, you need 128 mb[]("http://en.wikipedia.org/wiki/RAM") to run or 192 to install. If you install using the alternate cd you will only requires  64 mb for the install process.[]("http://en.wikipedia.org/wiki/RAM")

64mb does not currently work due to the following bug:
[https://bugs.launchpad.net/ubuntu/+source/debian-installer/+bug/202959](https://bugs.launchpad.net/ubuntu/+source/debian-installer/+bug/202959)

I would recommend 256mb of ram and 1.5gb of hard drive (plus space for your files of course).
