### Post by avtolle on 2008-07-21
Wubi is, for sake of simplicity, an .exe that installs Ubuntu "within Windows". Generally, a "virtual" hard drive is created on the HDD for Ubuntu to install and run on. It has proven safe for me, running under Vista. The Ubuntu runs "natively", and the one area of performance degradation is, for me, HDD access, but it is not bad. 

I will caution that a Wubi install might not be any better for you than what you've tried to date. Thus, if the problem installing involves, e.g., SATA drives, the problem will still be there. You might want to take a look at the Wubi Forum for information.
