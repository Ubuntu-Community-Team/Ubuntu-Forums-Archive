### Post by Spaceman9 on 2008-10-23
The upgrade should be listed in Update Manager when the 8.10 is posted on the Kubuntu web site. It should say: New distribution release '8.10' is available. Click Upgrade and follow the on-screen instructions.

You can upgrade online that way so you don't need to download a new .iso image and burn a cd.

Ignore everything I just said. I did a search for upgrading to 8.10 Kubuntu.

This is the way to upgrade to Kubuntu 8.10 [https://help.ubuntu.com/community/IntrepidUpgrades/Kubuntu](https://help.ubuntu.com/community/IntrepidUpgrades/Kubuntu)
