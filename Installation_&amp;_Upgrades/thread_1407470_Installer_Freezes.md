### Post by phillw on 2010-02-15
> **Mencarta said:**
> The MD5 hash is okay. When I tried to install it I waited 15 minutes and got the I/O Error: Error reading boot CD. Any suggestions?

We're deep into 'funnies' land here (and they're not called 'funnies' because they make you laugh).

Either the CD, or the CD drive is unhappy. At this point, you're going to need access to another computer. Boot that computer with the CD and see if you can get "Try without altering the computer" mode to work. That will pretty much rule out the CD.

If you haven't done so already, get a CD-Drive cleaning disk. (There is a cheat for laptops) and clean the drive.

Try a different brand of CD-R  (oddly enough, it is reported that el-cheapo CD-R's can work better than branded ones ... strange, but true)

Try burning the CD on a different system - Been there, seen that one.

Check that the cooling fan(s) and your motherboard is not full of fluff (especially on the power supply) Booting an OS from CD does put additional strain on your system, if the system gets too warm, or the power supply cannot cool itself sufficiently, then voltage drops will occur and cause errors.

Get a USB drive (1GB is big enough) and make a boot USB stick if your computer supports USB booting in BIOS - that completely removes problems with CD's...

It's like a never ending list, until you find the silly thing that is stopping it.

Cleaning the CD drive and checking for build up of dirt on cooling fans is an easy one to do.

The answer is there ... just a case of finding it.

Regards,

Phill.

On a laptop, when the drive pops open, so does the carriage with the LED assembly in it.

By using a cotton bud dampened in vodka, you can clean the LED's - just be real careful - the vodka won't hurt them, but poking around like rattling ashes in a bon-fire certainly will., just be real gentle, then use the dry side of the cotton bud to dry the lens assembly off.

If you break it, don't come a running to me - This is a 'cheat' and you have to be **real** gentle !!!
