### Post by Bucky Ball on 2013-10-09
Try installing ARandR from the repositories. It is a lightweight app that lets you set different resolutions for different monitors (among other things) via a GUI.

Once you have it installed, launch it and go Outputs>Your Monitor>Resolution. That easy, hopefully. ;)
