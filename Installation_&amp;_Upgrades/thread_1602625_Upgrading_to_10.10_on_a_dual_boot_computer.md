### Post by M_Mynaardt on 2010-10-24
Okay, I got it figured out.  I didn't have the update manager set properly.  I needed it to have the "Release Upgrade" option set to "Normal Releases" which I did not for some reason.  Just upgraded to 10.10 this morning.  So far so good.  Looks like I'll have to get used to a few new bits, but other than that, no problems so far!
