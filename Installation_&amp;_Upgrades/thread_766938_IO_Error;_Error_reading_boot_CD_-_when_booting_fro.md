I've downloaded "ubuntu-8.04-desktop-i386.iso" and burnt it using K3B. The sum checking in K3B passes without problem.

When I boot from this CD I get "I/O Error: Error reading boot CD" with the only option to restart the computer. This happens during "Loading Linux kernel" after starting to install. I get the same thing when checking the CD for errors (in the CD boot menu), i.e. it doesn't complete the check, the error appears before it's done.

### Post by Tmi on 2008-04-30
After following the advice of some here on the forums to burn the iso at the lowest possible speed it turned out that this was the solution.

So if you have the same problem, lower your burn speed.
