### Post by 505 on 2008-03-30
I upgraded a week ago from 7.10 to 8.04 and I already encountered some problems. In the beginning something went really bad, couldn't load the settings manager and no way to boot into any graphical environment (not even safe mode). Other annoyances were with camera's not mounting and dead keys on the keyboard where I don't want them.

Lesson of all that: if you are familiar enough with Linux/Ubuntu you can install it, but do know how to solve problems on you own, help is not always available for problems that started with a 'today's upgrade'. If you are unsure, wait for the final when all problems should be solved.
