### Post by dwingojr on 2007-06-01
I use VMWare Server console. It will allow you to run WinXP or any other Windows OS inside a window on your Ubuntu implementation (a "Virtual Machine"). Download it for free at [www.vmware.com](www.vmware.com). I prefer this because there are no broken boot loaders, you don't have to worry about the order of installation, and it works perfectly every time. I need stability in the enterprise, and this software has delivered! I have used some type of Virtual Machine since 2003, and this is very handy when I do need Windows. I have XP and Vista running "on top" of Ubuntu on my Dell D8xx. Good luck to you!
