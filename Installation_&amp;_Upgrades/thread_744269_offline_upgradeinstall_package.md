### Post by dstew on 2008-04-03
There is a service called [nonetdebs]("http://nonetdebs.homeip.net/") which might help. You upload to them a text file that shows the status of your system, and they assemble a list of links to the packages you need to update. You can download the packages to a CD or USB stick. It is only for Debian systems.
