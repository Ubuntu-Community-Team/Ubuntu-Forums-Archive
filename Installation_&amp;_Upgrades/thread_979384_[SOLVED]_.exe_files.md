### Post by lisati on 2008-11-11
.exe files are usually designed for MS-DOS or Windows, and won't run without help with Ubuntu.

If it's a "self-extracting archive", you can sometimes use the "unzip" command from the terminal. Otherwise you *might* be able to use it with the help with a program called Wine. See [https://help.ubuntu.com/community/Wine](https://help.ubuntu.com/community/Wine) for more information
