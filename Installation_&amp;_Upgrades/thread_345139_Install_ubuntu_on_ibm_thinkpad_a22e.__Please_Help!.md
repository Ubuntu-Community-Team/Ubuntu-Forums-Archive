### Post by AMT on 2007-01-23
Try downloading and using the Alternate install CD. Select text-based install at the boot screen (easy to use, don't worry). This will install Ubuntu like a normal OS instead of using the Live CD feature.

If that doesn't work, try Xubuntu.
