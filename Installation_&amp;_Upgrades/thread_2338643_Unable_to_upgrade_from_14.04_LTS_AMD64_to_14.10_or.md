### Post by ajgreeny on 2016-09-30
There is no point upgrading to 14.10 as that has lost all support and will get no more upgrades.  The repos for 14.10 have also moved to the EOL address.

If you really want to upgrade, I would suggest you clean install 16.04 or otherwise stick with 14.04 which has support till April 2019.

If you really prefer an online upgrade, you will only be able to go to version 16.04 and that option should be showing in your Software Sources ->Updates tab -> "Notify me of a new Ubuntu edition" where you need to choose "For long term support versions".
