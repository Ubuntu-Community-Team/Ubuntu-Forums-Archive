### Post by gordintoronto on 2010-04-05
The version you are looking for was released last week. It normally takes several months for a new version to make it into a new release of Ubuntu.

The new version is available now to people who are basically application developers. In other words, they understand how to compile a program from source code, they keep development libraries up to date, and a whole lot of other stuff which gets in the way if you just want to USE a computer.

One of the things those developers do is put applications into a PPA. Even that might take weeks. It's not a mechanical process.
