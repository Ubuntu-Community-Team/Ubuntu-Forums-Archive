### Post by meng on 2007-01-02
1GB won't be enough to install 6.06, unless you're talking about a server install (but who would want to do an Ubuntu server install onto portable storage?) I don't think even 2 GB is enough. 4 GB yes.

BUT if you mean you want to copy the CD files from the install disk to the portable storage and make it bootable, that is possible, if you want a more portable way of running the LiveCD. But that's not what most folks mean when you say "install".
