### Post by erginemr on 2008-03-12
If you own a legal copy of matlab, you can try installing WINE as the Windows emulator:
[http://www.winehq.org/](http://www.winehq.org/)

and run matlab via Wine. Matlab has been reported to work with it:
[http://appdb.winehq.org/objectManager.php?sClass=application&iId=49](http://appdb.winehq.org/objectManager.php?sClass=application&iId=49)

But also, apparently there is a Linux version for Matlab. 
[http://www.mathworks.com/products/matlab/requirements.html](http://www.mathworks.com/products/matlab/requirements.html)

You can try looking for its Linux version and registering it with the licence of your Windows copy.
