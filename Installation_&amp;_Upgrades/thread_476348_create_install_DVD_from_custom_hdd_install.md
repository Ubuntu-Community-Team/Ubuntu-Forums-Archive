### Post by IcarusR on 2007-06-17
Hi, In the past I have used Ghost, which is on Hirens Boot CD [http://www.hiren.info/pages/bootcd](http://www.hiren.info/pages/bootcd)
This is a rescue CD with LOADS of great tools. 
I ghost the drive to a file on an external drive. The idea being if I have a drive failure, or screw it up I can re-image the drive & be back to the same state as when the image was created with minimal effort.
I have to say I have not tried to re-image a drive yet as I do not have a spare 2.5 inch SATA drive.
Hirens CD is notoriously difficult to find in the net but here is a D/L link. I have not tried this link so can not vouch for it.
The other way that I used to use was to boot the PC from tomsrtbt [http://www.toms.net/rb/](http://www.toms.net/rb/) which is another usefull resorce. Then create a tarball of your install to a slave drive.
The resulting files from both methods above could then be burnt to DVD. To restore you would need to boot from the relevant distro.

I have not tried either methods completely & so can not give finite instructions on how to, or whether they work as desired, but should be a good starting point for you.
