### Post by samrat1004 on 2007-04-13
**Problem in Kickstart installation-->**
i want to configure my 15 laptops using kickstart with pxe boot
for that i make one of my laptop server and install kubuntu 6.06 LTS and followed the steps here include underneath:


1.i install  the following debian packages: 

tftpd-hpa 
dhcp3-server 
netkit-inetd 

2.Prepare my tftp server
Start my tftpd server via inetd with the following line in /etc/inetd.conf: 

tftp           dgram   udp     wait    root  /usr/sbin/in.tftpd /usr/sbin/in.tftpd -s /var/lib/tftpboot

sudo update-inetd --enable BOOT

and create the tftp directory /var/lib/tftpboot (if it hasn't already been created) 

On Ubuntu 6.06 Server (or if /etc/inetd.conf doesn't exist) i edit /etc/default/tftpd-hpa: 

RUN_DAEMON="yes"
OPTIONS="-l -s /var/lib/tftpboot"

3.Prepare my dhcp-server

i have a working dhcp server add the following lines to my group in /etc/dhcp3/dhcpd.conf file:

# dhcpd.conf
ddns-update-style             ad-hoc;

option subnet-mask            255.255.255.0;
option broadcast-address      192.168.10.255;
option routers                192.168.10.254;
option domain-name-servers    192.168.10.254;
option domain-name            "ltsp";   # You really should fix this
option option-128 code 128 = string;
option option-129 code 129 = text;


get-lease-hostnames           true;

next-server                   192.168.10.254;
option root-path              "192.168.10.254:/opt/ltsp/i386";

subnet 192.168.10.0 netmask 255.255.255.0 {
  range 192.168.10.100 192.168.10.199;
  filename "pxelinux.0";
  next-server          192.168.10.254;

}





4.Prepare ubuntu netboot
Get the files
Put the netboot files into the /tftpboot directory. 

lftp -c "open [http://archive.ubuntu.com/ubuntu/dists/dapper/main/installer-i386/current/images/;](http://archive.ubuntu.com/ubuntu/dists/dapper/main/installer-i386/current/images/;) mirror netboot/"
cp -a netboot/* /var/lib/tftpboot
# untar the pxeboot.tar.gz
cd /var/lib/tftpboot/
tar xfz netboot.tar.gz


5.Pxe bootloader config
Edit or patch pxelinux.cfg/default :

DISPLAY ubuntu-installer/i386/boot-screens/boot.txt

F1 ubuntu-installer/i386/boot-screens/f1.txt
F2 ubuntu-installer/i386/boot-screens/f2.txt
F3 ubuntu-installer/i386/boot-screens/f3.txt
F4 ubuntu-installer/i386/boot-screens/f4.txt
F5 ubuntu-installer/i386/boot-screens/f5.txt
F6 ubuntu-installer/i386/boot-screens/f6.txt
F7 ubuntu-installer/i386/boot-screens/f7.txt
F8 ubuntu-installer/i386/boot-screens/f8.txt
F9 ubuntu-installer/i386/boot-screens/f9.txt
F0 ubuntu-installer/i386/boot-screens/f10.txt

DEFAULT install

LABEL install
        kernel ubuntu-installer/i386/linux
        append vga=normal initrd=ubuntu-installer/i386/initrd.gz ramdisk_size=16417 or 14332 root=/dev/ram rw  -- ks=http://192.168.10.254/ubuntu/ks.cfg

PROMPT 1
TIMEOUT 0
