### Post by leon.lain.delysid on 2014-08-12
Hello! =)

- Launch the Software Updater
- Go to the options menu
- In the "Updates" tab, you should have an option for notifications, which you can switch from "LTS versions notifications" to "never"

This is how it works in 14.04. It might be a little different on 12.04 but not too much. I hope you can find it.
