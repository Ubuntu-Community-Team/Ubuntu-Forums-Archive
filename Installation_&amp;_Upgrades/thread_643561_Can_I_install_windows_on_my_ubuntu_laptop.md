### Post by themusicwave on 2007-12-17
Hello,

Yes you can have both operating systems.  You need to run what is called a daul boot system.

The easiest way to setup a daul boot in my opinion is to install windows first.  Then install Ubuntu, it will see Windows there and will setup a dual boot for you.

You'll want to have Ubuntu Shrink your windows partition so that you have a partition for each OS.  

Ubuntu will do the rest.

Now every time you boot you will be prompted to pick either Ubuntu or Windows.

Search for dual boot for more information.

Let me know i yo have any questions, I have bene running dual boots for quite awhile now.
