### Post by Topsiho on 2014-02-03
In an upgrade more data has to be downloaded than the size of the .iso, as the later, extra, installed programs have to be updated too. So it is quite normal that during an update twice or more data has to be downloaded. How to do an upgrade (or download of an .iso) on a dial-up connection, i wouldn't know. Possibly you could acquire a .iso of 13.10 some other way, and install that. And update, and install the other programs after that. The update (necessary as the .iso is some months old now) may take a considerable download too.

Other thing: Ubuntu 13.10 will be supported for 9 months only, and expires next July.

In April the LTS version 14.04 will be distributed, and will be supported for 5 years. Maybe it is advisable for you (dial up connection, not on line all the time), to wait for this 14.04 version.
