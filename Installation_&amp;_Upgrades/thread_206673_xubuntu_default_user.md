I'm unfamiliar with Xubuntu installation, and this looks strange, but try root as the user name. root is the administrator on Linux. You should immediately add a new user, since running as administrator is unsafe. When adding the user, give him permission to run administrative tasks. Then never again login as root. Read more here: [https://help.ubuntu.com/community/RootSudo]("https://help.ubuntu.com/community/RootSudo")
