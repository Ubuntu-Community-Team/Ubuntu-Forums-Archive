> I downloaded ubunto 9.04 to my IBM laptop, what  web address can detail the rest of the instructions of how to download the  recommended burning software and details of the rest of the process to use ubuntu.


Look here: [https://help.ubuntu.com/community/](https://help.ubuntu.com/community/)

If you are currently running Windows then find CDBurnerXP, a free program, and burn the iso to disk. Then simply boot computer with the new disk. Recommend burn disk at slowest speed and perform disk check prior to installing. You will also first need to check the download to verify it is good. That is accomplished by performing a "md5sum" check on the iso file. Look around, there are available programs for Windows that will allow you to do that.
