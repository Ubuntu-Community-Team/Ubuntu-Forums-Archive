### Post by jcolyn on 2011-01-03
Back when I was dual booting before going 100% Linux I would do a fresh install of Windows, update, install any additional software, then defrag. I would then shrink the windows partition using partition or disk manager (if I recall correct) by right-clicking My Computer. Be sure the free space created when shrinking is unformatted.

Once all of this was done I would then start the Linux install. If you only want to create a / and swap partition alongside the Windows partition then choose to install onto the free space. Don't choose to run side by side.

[SIZE=2][COLOR=Red]**Only do the below if you know what you are doing..............**[/COLOR][/SIZE]

If you want to create  /boot, /, and /home partitions then choose to manually partition. Windows will be listed as /dev/sda1. Click the free space and choose add. Create the partitions except the swap as primary.
