### Post by princeofthecity on 2011-04-14
i used unetbootin to create my bootable usb with 10.10 for netbook.  i am trying to install ubuntu on my brand new fresh out of the box Asus Eee PC 1018P but the problem is when i hit esc during the boot to get to bios to select boot from usb, the usb doesnt show up as an option.

i also tried hitting f2 and making it an option to boot from usb but that doesnt seem to work either.

has anyone had similar issues with booting from the usb?  could it be possible that my unetbootin created bootable usb sucks and i need to re-create it?

thanks all!

---

### Post by princeofthecity on 2011-04-16
solution:  i re-created the bootable usb with unetbootin using the same .iso as the first one.  works fine after disabling bootbooster in the bios setup menu and making sure usb was set to 2nd position in the boot order.  restarted and hit esc ONCE to enter boot screen.  my usb showed up and ubuntu booted normally.

thanks all who read this even though no replies.  maybe this will help some other n00b one day. ;)
