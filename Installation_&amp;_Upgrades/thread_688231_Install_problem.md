### Post by dabl on 2008-02-05
I prefer the Alternate Install CD, myself.

Your problem is a video display issue. On the Alternate CD, you have some options for setting different display modes (is it F6?). Choose "safe graphics" or "VGA 640 x 480" or "VESA" and keep trying until you find one that works.

Probably after you finish installing, and reboot, you're going to have this same problem again.  Search this forum for posts on your graphics chip, and you should be able to learn how to make it display and work correctly.  :)
