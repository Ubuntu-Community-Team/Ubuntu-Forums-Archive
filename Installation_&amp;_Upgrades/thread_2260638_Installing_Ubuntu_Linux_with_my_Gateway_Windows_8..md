### Post by ajgreeny on 2015-01-13
Have a good look through all the info at [https://help.ubuntu.com/community/UEFI](https://help.ubuntu.com/community/UEFI) as I am fairly sure this laptop will be using UEFI.
If you do not understand all of what is written there come back again and ask.

To answer your specific question is less easy as it depends what you mean by "problems involving the Windows boot manager".  After installing Ubuntu the machine will not normally use the windows bootloader any more; grub will take over and boot both Ubuntu and Windows.

For more details Isuggest you wait for more answers; I don't have Windows any more so have no personal experience of dual booting with Windows 8; XP was my last version.
