### Post by Rhubarb on 2007-12-24
I have noticed with some Dell notebooks (as an example) that the vista DVD will not install if there is something already on the hard drive (it just crashes).
To clear the hard drive, put in the ubuntu 7.10 live CD and boot from it (or use the gparted live cd).
Go to System --> Administration --> Partition Editor
Click on all the partitions and delete them, when finished you will be able to install vista.
