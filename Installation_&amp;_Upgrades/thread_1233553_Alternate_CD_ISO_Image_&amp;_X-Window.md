### Post by jaxxstorm on 2009-08-06
There are NO Gui elements in the Ubuntu server CD. You'll need internet access when to enable them.

What are you using the server for? You can use a standard Ubuntu CD and install things like Apache, SSH, Tomcat or any other server element. The only difference between the server CD and the Desktop CD is that the desktop CD has gui elements with it and the server CD has server packages bundled with it. They're the same thing otherwise.
