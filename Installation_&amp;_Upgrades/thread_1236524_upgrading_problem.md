### Post by phillw on 2009-08-10
I'd suggest using Synaptics Package Manager.

System ==> Administration ==> Synaptic Package Manager

If you type in iproute into it, it should show one as installed (green-box) - If you suspect any problems (as you do) - mark it for re-installation and apply.

This will both re-install it and check it is popping on the latest version in one go.

Hope that helps.
