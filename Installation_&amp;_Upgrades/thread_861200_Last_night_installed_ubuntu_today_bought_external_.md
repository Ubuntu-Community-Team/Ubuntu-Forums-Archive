### Post by logos34 on 2008-07-17
easiest way is to unplug the internal drive before installing ubuntu.  Then when you want to use linux just enter the Bios and toggle the boot disk to USB.  Add windows to ubuntu with [ntfs-config]("https://help.ubuntu.com/community/MountingWindowsPartitions/ThirdPartyNTFS3G#Ubuntu%208.04%20LTS%20%28Hardy%20Heron%29")
