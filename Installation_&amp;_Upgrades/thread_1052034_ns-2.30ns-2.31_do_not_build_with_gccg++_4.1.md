### Post by juanka on 2012-02-12
Easier way to keep the compiler versions:
First of all, follo instructions at:
  	 	 	 	 	 	    [COLOR=#000000][http://nsnam.isi.edu/nsnam/index.php/User_Information#Ubuntu_Installation_Guide](http://nsnam.isi.edu/nsnam/index.php/User_Information#Ubuntu_Installation_Guide) [/COLOR] 
  and then, when compiling, you would get error of type "deprecated ... string to char". Very easy. Look for the line of the file where the error happens and cast the error. I mean, substitute this:
  	 	 	 	 	 	     [COLOR=#000000]p = (char *) “string” [/COLOR] 
  [COLOR=#000000]where you see this:
 [/COLOR] 
  [COLOR=#000000]p = “string” 
[/COLOR]
[LEFT][COLOR=#000000]by the way, don't forget to lock the compiler versions anyway or you could get new error if the systems update them.
[/COLOR] [/LEFT]
