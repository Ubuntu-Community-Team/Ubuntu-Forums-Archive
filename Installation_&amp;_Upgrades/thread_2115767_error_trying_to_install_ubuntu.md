Wubi stands for windows, ubuntu installer, where ubuntu is installed 'within' windows, which is generally considered only a
trial install.
If your computer is very new and came with windows 8 preinstalled
you probably have the uefi bootloader as compared to the older
MBR [msdos}. UEFI is something i have very little experience with, i do know that whatever mode the windows bootloader is installed in, ubuntu has to be the same.
Some more info in this link.

[http://en.wikipedia.org/wiki/Unified_Extensible_Firmware_Interface](http://en.wikipedia.org/wiki/Unified_Extensible_Firmware_Interface)

A recent how to install using UEFI here.

[https://help.ubuntu.com/community/UEFI](https://help.ubuntu.com/community/UEFI)
