Have a look at:-
[http://ubuntuforums.org/showthread.php?p=8199471#post8199471](http://ubuntuforums.org/showthread.php?p=8199471#post8199471)

and you may get some more help.  As you say, you can stop using compiz, which is a shame, do a workaround with the driver which also has some other problems, or you can run in 16 bit colour, which appears to overcome the problem.

Let's hope this is sorted soon with an update.
