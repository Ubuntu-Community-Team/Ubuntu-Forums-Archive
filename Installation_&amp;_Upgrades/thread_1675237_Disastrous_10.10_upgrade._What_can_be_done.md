### Post by TBABill on 2011-01-25
First I would try the LiveCD from when you installed Kubuntu 10.04. Run that and mount the drive to see if you can recover your data files. Since you did a partial install the system may not run correctly, but the files that were not part of the system and those not being upgraded may well be intact for recovery. 
 
As far as getting it to boot, I don't know of any fixes when a mid-install termination takes place. Hopefully someone will come along soon with some ideas on that end. As far as I know a new install is the only fix. Much quicker too and if you get all your files safely off the drive it's probably the safest method of getting the system back.
