### Post by Bashing-om on 2012-11-02
short answer-> Yes
12.04 ===>Ubuntu Software Center->top taskbar->edit->Software Sources:
click on arrow at the Download from -window- ,choose "other", click select best server --wait while testing is conducted--click on choose server, enter your password, select close.

gets the faster responding mirror for you.
