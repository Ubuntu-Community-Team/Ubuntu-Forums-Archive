### Post by Shinbu-Otaku on 2008-01-08
there are 2 ways to do this, 1: go into the teminal and type *boot game* (i think) and it boots back into the ps3 XMB, or 2: put the live cd back in the PS3, go to system > administration > Partition Editor and delete or format the ubuntu partition and re-assign the space back to the ps3 side of it (if you don't want to have linux on there again that is, if you do want to install it again then obviously dont do this)

If the live cd doesnt work (im not too sure it does) then install GParted using the add/remove programs option on Ubuntu, as this is the partition editor.

hope that helps
