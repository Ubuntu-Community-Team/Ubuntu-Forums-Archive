### Post by awsumatt on 2016-10-14
Just update to the newest (nvidia-370) drivers. Go to the login screen, press **Ctrl**+**F1** and login then enter this into the console:
```
sudo add-apt-repository ppa:graphics-drivers/ppa
sudo apt update
sudo apt-get install nvidia-370
```
