### Post by deadflowr on 2014-11-16
Don't know which version of Ubuntu but you can set the update manager to never check for updates(not recommended)
In 14.04 open up the program Software and Updates, and go to the section Updates.
In Updates go to "automatically check for updates and" in the dropdown select Never.
From here on you would need to run updates manually >> see apt-get update.

If on an older version(specifically 12.04), the basics still apply, but the program Software and Updates does not exist, so instead open the Update Manager and in the bottom left corner, click on the settings button. Do the same as mentioned above.

As far as uninstalling the software center, I've never thought of doing that, simply because it's not that big a package.
You can simply unlock it from the launcher if it shows there and forget about it...
