### Post by mx32 on 2011-10-22
Finally I fixed the installation and Ubuntu 11.10 its running smooth.

I did enter the command:

```
sudo dpkg --configure -a
```

Followed instructions on screen. It showed some errors, so at the end I did run the same command again and this time no error was reported.

Hope this help others. :D
