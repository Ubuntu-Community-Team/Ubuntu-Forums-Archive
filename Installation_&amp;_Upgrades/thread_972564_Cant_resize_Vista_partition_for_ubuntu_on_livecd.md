### Post by pytheas22 on 2008-11-05
I'm not sure why the slider doesn't work--are you sure you're clicking in the right spot?  In any case, you could always choose manual partitioning, which will allow you to type in the exact size to which you want to decrease your Windows partition, rather than indicating it using the slider.

Also, in a worst case, you can always use [wubi]("http://wubi-installer.org/") to get around the partitioning.
