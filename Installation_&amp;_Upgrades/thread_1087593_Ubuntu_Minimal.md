### Post by bubwitmaingay on 2009-03-06
You can use eeeXubuntu also if you really loved to use Ubuntu for every machine that you have. Nonetheless, there are other distros available for eeePC especially the 2g, 4g or 8g surf.

Try eeePCLinuxOS (Fedora) or Pupeee (Slackware). The former takes time to boot - about 3 minutes, can be installed into the machine for 700MB and has all the apps needed (togo); but the latter only takes about 30 seconds to boot, for less than a hundred MB installation space and has also all the apps needed.
