### Post by ajgreeny on 2020-11-25
This has often happened to new installations in the past; I have seen it on my Xubuntu systems, both 18.04 and 20.04 if I remember correctly, but in both cases these error messages seemed to be false, as I could never find anything giving me further information on any of the system logs or dmesg.

In both those systems those errors stopped appearing after a very short time and a few system upgrades, so as long as the OS runs well after boot I would not worry too much.
