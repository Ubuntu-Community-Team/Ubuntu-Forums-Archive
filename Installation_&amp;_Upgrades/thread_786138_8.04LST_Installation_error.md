### Post by lemming465 on 2008-05-10
If you are doing a clean install I assume you are booting an Ubuntu 8.04 live CD.  If that scenario is giving errors, it's usually due to some defect in the burned CD. At the boot options, pick the integrity check.  If the CD doesn't pass, try burning a different one, perhaps at a different rate (usually slower), or with different media, or after cleaning the CD drive.
