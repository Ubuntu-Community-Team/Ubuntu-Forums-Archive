Hi!

Old system: Thunderbird 1.5.0.14ubu, Ubuntu 7.04
New system: Thunderbird 2.0.0.14, Ubuntu 8.04

Thunderbird installed by Synaptic.

I copied my old profile folder ~/.mozilla-thunderbird in my new home directory.

This should provide a smooth migration, including mails, addresses and settings.

WARNING!
If you want to migrate the same way check the foldernames inside of Thunderbird on your old system.
If you have created custom folders make sure their names do not end with a dot.
Otherwise you will find empty folders on your new system.
