### Post by skwillz on 2006-10-18
Can you tell us what type of computer you have?

Many computers that now ship with BIOS RAID are using software controllers. Unfortunately, I have found these to have limited support in Ubuntu.

But you could just as easily use the drives separately with the OS on one drive and data on the other.

I think your current WinXP setup would probably go kaput, though.. assuming it was installed when RAID created a single 320GB drive.
