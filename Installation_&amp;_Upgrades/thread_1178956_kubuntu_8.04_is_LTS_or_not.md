### Post by Bucky Ball on 2009-06-05
Pretty simple. Install Ubuntu 8.04 LTS, go to Synaptic Package manager, search for KDE and install. When you boot and login, go to 'sessions' and choose KDE. Make this your default session and you're good to go. You can also load

kubuntu-desktop

... from synaptics and that should give you the apps. But this will be sitting on Ubuntu 8.04 LTS; you are just using KDE desktop manager/environment, not a full  Kubuntu install as such.

You might like to try some other desktop environments while you're at it like Xfce, Openbox. There are many. Just choose the session you fancy from 'Sessions' at login. All my machines are 'hybrid' like this; I love it! Makes Linux/Ubuntu even more customisable.
