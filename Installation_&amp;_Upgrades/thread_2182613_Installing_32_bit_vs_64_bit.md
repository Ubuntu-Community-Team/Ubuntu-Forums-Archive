### Post by drmrgd on 2013-10-21
Not too clear on what you want to accomplish.  Are you saying that you want to replace the OS with 64-bit Ubuntu?  Discs themselves are not confined to word size; it's the CPU that considers this.  If you try to install 64-bit Ubuntu on a 32-bit system, you should get an error telling you that you can't do it. If you're just looking for USBs to back up to, then there's nothing to worry about.
