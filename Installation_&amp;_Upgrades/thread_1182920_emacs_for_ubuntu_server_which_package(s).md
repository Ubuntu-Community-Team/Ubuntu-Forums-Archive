### Post by unutbu on 2009-06-09
If you want emacs to run without an X server, you want to install one of the *-nox packages. ("nox" means "No X windows").

emacs21-nox is emacs version 21
emacs22-nox is emacs version 22
emacs-snapshot-nox is emacs version 23
