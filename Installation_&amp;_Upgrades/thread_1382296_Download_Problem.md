### Post by Techsnap on 2010-01-15
You're trying to run a Windows program on Ubuntu? Give WINE a try [http://www.winehq.org](http://www.winehq.org), WINE is not perfect nor is it a 100% drop in replacement for running all Windows applications so if it doesn't work but you really want to use the program but also Ubuntu I suggest you reboot.

[http://appdb.winehq.org](http://appdb.winehq.org), here you can see if your application is compatible, but you could try anyway even if it isn't listed. If it does work, post your experience there.
