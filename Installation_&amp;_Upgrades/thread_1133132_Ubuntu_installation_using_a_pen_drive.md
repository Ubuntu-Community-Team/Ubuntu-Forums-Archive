### Post by Old_Grey_Wolf on 2009-04-22
If you have Ubuntu 8.10 installed already, you can use the menu options to create a bootable USB for installing 9.04. Download the 9.04 ISO. Then use the menu option "System > Administration > Create a USB Startup Disk" to create a bootable 9.04 USB. Then reboot from the USB and install as you would from a Live CD.
