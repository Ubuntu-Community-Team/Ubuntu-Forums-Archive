I am using Ubuntu 16.04 LTS. This morning an auto update presented and I clicked to install it. Another box appeared that stated "will require installation of untrusted packages".
How does one determine what is being installed. I have an idea that this is connected to the Google Earth installed, but it doesn't state that anywhere.

---

### Post by Dennis N on 2020-01-25
PPAs can be deemed 'untrusted'. When adding a PPA to your sources, you will see, for example (from launchpad.net):
> You can update your system with unsupported packages from this **untrusted PPA** by adding ppa:alexlarsson/flatpak to your system's Software Sources
So that might be the case.
