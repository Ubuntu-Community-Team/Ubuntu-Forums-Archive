### Post by Lateralis on 2010-09-18
It varies.  With something like Arch (and I guess Gentoo) you do not get a graphical user interface and have to install one manually after installation.  Such distros take a lot of post-installation customisation and aren't recommended for the average PC user.  

With Fedora, Ubuntu, OpenSuSe and other such distros, they're configured to work "out of the box" for most hardware profiles.  For these distros you whack the CD in the drive and boot into a Live session to test it out and then install the distro from the Live session if you like it.  The exact details of how you install varies from distribution to distribution, but the three distros I mentioned all have very easy step-by-step installation guides to help you through the process.
