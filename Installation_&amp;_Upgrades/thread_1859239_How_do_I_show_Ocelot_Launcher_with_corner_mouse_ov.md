### Post by Hempsy on 2011-10-13
The only way I've found is to install "CompizConfig Settings Manager" from the software centre. Launch the program, scroll down to ubuntu unity plugin and click it. This contains the launcher behaviour settings. Good luck, I found that it's not the same as 11.04 even set to top left.
