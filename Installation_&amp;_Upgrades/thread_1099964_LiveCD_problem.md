### Post by Rocket2DMn on 2009-03-18
Hi, you might like to check out [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto) for more details.  I suggest burning at a slow speed like 4x, this helps prevent write errors.  If you still have problems, you may want to try a different batch of CD-Rs.
If you *really* can't get it to work, you can order a CD from ShipIt - [https://shipit.ubuntu.com/](https://shipit.ubuntu.com/) - thought it may take a few weeks to arrive.
