### Post by Polaris395 on 2012-06-16
Hi all!

I current have a small server running with Ubuntu 11.04. It has a DVD drive and a 2TB Seagate drive. I installed Natty Narwhal on the machine and got all my applications and going. 

Unfortunately, after some time of use, I needed to restart the machine. I did so, and I find that the machine will not boot! I pass through BIOS, but then it hangs for ~10 seconds and gives the message "No boot device found - press any key to restart!"

To fix this, I put in the installation CD that I used and selected the option to "Boot from first hard disk". Lo and behold, everything boots up A-ok! I took a look at fdisk -- only thing mounted is /dev/sda (my 2TB drive). 

I've restarted the machine to test different variations (presence of CD, different boot flags, ETC.) and the results are all the same. My hardware will not boot unless I select the option from the install CD to boot from the hard drive. 

Can anyone shed any light on how to fix this problem?!

Cheers, 
Polaris395!
