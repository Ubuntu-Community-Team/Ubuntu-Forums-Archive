### Post by markhepworth on 2009-05-30
That .img fie is for placing on a usb stick. If your netbook can boot from usb stick, then try this: [https://help.ubuntu.com/community/Installation/FromImgFiles](https://help.ubuntu.com/community/Installation/FromImgFiles)

If not, here is the link for burning an .iso file to cd: [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto) (if you go that route, you'll need to get a .iso fie instead of your .img)

Good luck.
