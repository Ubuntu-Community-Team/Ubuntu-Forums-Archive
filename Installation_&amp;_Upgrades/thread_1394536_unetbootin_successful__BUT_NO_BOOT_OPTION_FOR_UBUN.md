### Post by C.S.Cameron on 2010-02-02
Unetbootin installs a "virtual" Live CD to your C drive.
The first time you boot, you should be able to start Ubuntu and then install it to your hard drive from there. 
If you chose to dual boot Windows you can uninstall UNetbootin the next time you boot Windows.
