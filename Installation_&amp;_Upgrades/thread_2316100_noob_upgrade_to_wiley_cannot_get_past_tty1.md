### Post by egeezer on 2016-03-05
At the first blinking cursor, type in the name you used to set up your original (14.10) install.     When the next line (Password) appears, type in the password you set.  Nothing will appear (security measure).     At the next cursor, try ```
startx
```  That should give you the new GUI.
