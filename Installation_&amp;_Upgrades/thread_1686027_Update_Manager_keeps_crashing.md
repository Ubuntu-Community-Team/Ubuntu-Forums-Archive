### Post by Oliphant on 2011-02-11
I fired up my Ubuntu today and I saw a red circle with a white line in it telling me there is or was an error with the Update Manager.  

Each time I opened the Update Manager the window would pop up and the progress bar would read "building..." then would crash.  I restarted, tried again but still crashing.  

Ended up opening Terminal and doing

sudo apt-get update

Update Manager fired right up and did its thing like normal.  

Just figured I'd share in-case another noob like me came across it. :)
