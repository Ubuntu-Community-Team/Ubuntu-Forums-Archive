### Post by Rocket2DMn on 2008-02-10
Yup - [https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows](https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows)
If you don't like the option for vista in 1.1, go down to 1.2 using the LiveCD to overwrite the windows bootloader.
