### Post by Old_Grey_Wolf on 2012-03-09
Try using a live CD\DVD first to see if you can see the Fedora partition.

Then, install Ubuntu side-by-side with Fedora. The Fedora should show up in the filesystem. If not then mount the Fedora partition.
