### Post by coldraven on 2013-06-23
As SuperFreak says, you need to make a bootable thumb drive. I use Unetbootin but PenDrive should work fine.
[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

As for the CD, did you just copy the files to it? That won't work, you have to use the "Burn Image" option in your disc burner software to make it bootable.
That is, select "Burn Image" (or equivalent terminology) then choose the Ubuntu.iso file that you downloaded.
