### Post by Mark Phelps on 2010-03-01
Actually, if you're not very experienced with Linux, you should not force an install on an untried machine.

You should boot from the LiveCD to see how well it works, and how well it detects your hardware and installs the proper drivers.

Unlike with MS Windows, there is no Driver CD you insert to install the drivers.  So, any drivers that do not get installed during LiveCD mode, and result in hardware not working, are going to range from trivial to impossible to fix in the real installation.
