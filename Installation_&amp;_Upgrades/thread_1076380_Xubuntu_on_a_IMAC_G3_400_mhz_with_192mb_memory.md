### Post by MedianMajik on 2009-02-24
Were you using a Live cd?  I had to use the Alternative CD/text install.  Also, Gutsy didn't work for me.  I had to use 7.04 on my iMac G3 350mhz with 192mb ram.

After it boots up you can press "Tab" and type your installation command from the list that appears. I used powerpc-install
