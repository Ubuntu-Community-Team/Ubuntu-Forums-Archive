### Post by Pumalite on 2007-08-20
If you have Vista installed, shrink the partition with the Vista partitioner; then install Ubuntu in new created partition and install Ubuntu in MBR. It will give you a Menu to choose Vista or Ubuntu, Same for XP, except that you could use Gparted to do the partitioning.
