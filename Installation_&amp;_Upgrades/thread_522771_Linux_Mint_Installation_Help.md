### Post by DirtDawg on 2007-08-11
It's hard to tell from your post just how much experience you have so please excuse me if my suggestions are obvious. 

The Linux installer will not repartition the entire hard drive. Before you install Linux, you will need to repartition the Windows side first. Or reinstall Windows and give it a smaller partition. If Windows already occupies the entire hard drive, the install won't work. 

If that's not the problem, please explain the nature of the error you receive. That will help us help you.

Good Luck
