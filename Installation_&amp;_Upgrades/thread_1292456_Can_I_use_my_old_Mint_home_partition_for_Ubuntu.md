### Post by lindsay7 on 2009-10-15
I would think so. When you install form the live cd and choose the manual install option, you will see the partitions and how they are set up in terms or /, /home, swap.  You would pick the / partition to format and use as the root partition. You would use the /home partition and leave it unformatted. The swap partition will take care of it self.  If your old Mint /home is not compatible you should find out at this point in the install.
