### Post by p_quarles on 2008-01-13
Is this a SATA drive, by any chance? XP wasn't designed with native support for SATA disks. It's possible to load the SATA drivers using a floppy disk, and it's possible to remaster the XP installation disk to include the drivers (and other useful stuff like service pack 2). 

Anyway, this is more of a Windows issue, so you'll have better luck looking for further information on Windows-related sites. I can tell you, though, that there is a lot of documentation out there about how to do this.
