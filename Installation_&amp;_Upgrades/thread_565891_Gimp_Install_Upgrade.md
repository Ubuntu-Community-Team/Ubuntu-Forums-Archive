### Post by rsambuca on 2007-10-02
Ubuntu basically takes a snapshot of the Debian unstable repositories every six months.  The Ubuntu developers then update gnome, tweak it to make it as stable as they can, add in all of the Ubuntu goodness and then release it.  Basically, once a version has been release, it is updated for security fixes and minor upgrades, but new versions are generally not added in until the next version of Ubuntu.
