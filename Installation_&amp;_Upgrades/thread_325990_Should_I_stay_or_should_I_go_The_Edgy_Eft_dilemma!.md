### Post by Sef on 2006-12-26
Dapper is the Long Term Stable version.  Hence packages are older, but less likely to have problems.   

Edgy is the unstable version.  Hence the packages are newer, and more likely to have problems.  

A few questions to ask yourself about upgrading:

1) Is there some application you need that cannot install on Dapper?  

2) Do you have the time to fix Edgy if there are any problems with the install?

3) Do you want to fix Edgy problems, even if you have the time?

4) What is the reason you want to upgrade to Edgy?
