### Post by dino99 on 2012-11-05
you can:
- remove xorg.conf if it exist: ```
 sudo rm /etc/X11/xorg.conf
```
- purge then reinstall nvidia:
```
sudo apt-get purge nvidia-current nvidia-common
```
```
sudo apt-get install nvidia-current
```

then reboot
