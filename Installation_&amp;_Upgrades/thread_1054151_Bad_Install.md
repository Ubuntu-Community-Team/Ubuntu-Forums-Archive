### Post by cariboo on 2009-01-29
Depending on how old the previous version was, you have to upgrade in steps, for instance if you were to upgrade from 7.10, you would need to follow this upgrade path:

```
7.10-->8.04-->8.10
```

If you went directly from 7.10-->8.10 you are going to run into dependency problems.
