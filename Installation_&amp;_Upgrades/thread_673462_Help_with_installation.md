### Post by ajgreeny on 2008-01-20
Have a look at [http://www.psychocats.net/ubuntu/installing](http://www.psychocats.net/ubuntu/installing) which should tell you all you need to know.
Why do you think your CD is not bootable?  If you burned the iso file you downloaded as an image, not simply as a data file it should boot if your bios is set to boot from CD first.  For more info on burning iso files to cd see [http://www.psychocats.net/ubuntu/iso](http://www.psychocats.net/ubuntu/iso)
Don't worry about the little things that seem big hurdles.  We were all there at one time and soon enough you will feel quite at home with Ubuntu.
