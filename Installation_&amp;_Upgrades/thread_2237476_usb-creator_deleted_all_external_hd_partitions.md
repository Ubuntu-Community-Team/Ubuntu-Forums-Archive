### Post by sudodus on 2014-08-02
I think you are on the right track. Try first with ***Testdisk***, then with ***PhotoRec***. Both come from

[http://www.cgsecurity.org/](http://www.cgsecurity.org/)

Browsing that site you will find quite a lot of tips how to recover as much information as possible.

If you cannot restore the partition table and file system, the only alternative left is to restore the file content, but the file names are lost, unless stored inside the file data (which is the case sometimes).
