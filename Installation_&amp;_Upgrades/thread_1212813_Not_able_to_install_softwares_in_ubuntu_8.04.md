### Post by Partyboi2 on 2009-07-14
Hi, to upgrade ubuntu to 8.10 you need to use the [[COLOR=Blue]Alternate cd[/COLOR]]("http://noncdn.releases.ubuntu.com/jaunty/"), to install the software you want you can either manually download the deb and the dependencies from [[COLOR=Blue]here[/COLOR]]("http://packages.ubuntu.com/"). If you want to download the deb packages and all their dependencies automatically you can use a program called [[COLOR=Blue]Keryx[/COLOR]]("http://keryxproject.org/") and a good walk through tutorial can be found [[COLOR=Blue]here[/COLOR]]("http://crashsystems.net/2009/01/keryx-tutorial/").
