### Post by Growlor on 2011-05-01
This is a lot harder than it has to be due to all the changes, but here is the process to do this:
1) Open Firefox
2) Press the Alt key (or hover over the words "Firefox Web Browser in the top of your desktop) This will display the menu list.
3) Click on Bookmarks -> Show all bookmarks.
4) Make the menu that opens full screen and press the alt key or hover over the word "Library" (you can alternately not make it full screen and go to the very top of your desktop and hover over the words "Firefox web browser" - its really confusing because everyone who has been using PC based GUI's for a while will not expect the changes to be in the top of the entire desktop instead of the window that has the focus!)
5) Go to Import and backup -> Restore -> Choose file
6) Browse to your desired .json file backup and select it then press the "Open" button.

Hope this helps.
