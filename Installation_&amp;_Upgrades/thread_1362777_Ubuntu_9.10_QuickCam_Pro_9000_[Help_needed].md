### Post by chenxiaolong on 2009-12-23
Those files are for Windows only. The webcam should work without the CD. Just install the [COLOR="DarkOrchid"]cheese[/COLOR] package with Synaptic ([COLOR="SeaGreen"]System Menu > Administration > Synaptic Package Manager[/COLOR]). After you install it, you can open the program in [COLOR="SeaGreen"]Applications Menu > Sound & Video > Cheese Webcam Booth[/COLOR].
