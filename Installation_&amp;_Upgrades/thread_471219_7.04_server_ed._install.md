### Post by taurus on 2007-06-11
1.  Server version only comes with a prompt, no window manager whatsoever.  If you want to run Gnome, you need to install it.

```
sudo aptitude update
sudo aptitude install ubuntu-desktop gdm
sudo /etc/init.d/gdm start
```

2.  The root account is locked by default so if you need root privilege, use sudo instead.

[https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)

3.  If you don't have XP on your machine anymore, you can remove the entry in /boot/grub/menu.lst (near the end of the file).

```
sudo nano -Bw /boot/grub/menu.lst
```
Save it with <Ctrl>o and exit with <Ctrl>x.
