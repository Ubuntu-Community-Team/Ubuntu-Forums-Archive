### Post by el_profi on 2008-04-29
I wanted to install 8.04 over an existing 7.10 installation on a dual boot system with 2 ntfs and 4 ext3 partitions.
gparted was not able to detect the existing partitions at all and only offered to use the entire disk for the new installation.
I discovered that gparted was confused by the second ntfs partition which was a logical partition.
I backuped and deleted this partition with another tool and redefined it as a primary partition. 
Now gparted was able to detect all paritions correctly and the installation succeeded. Nevertheless very annoying !!!
