### Post by sikander3786 on 2010-11-15
Yes you can save those packages simply using APTonCD.

[http://aptoncd.sourceforge.net](http://aptoncd.sourceforge.net)

Or copy them from /var/cache/apt/archives to a safe location and put them in the same folder on your new install.
