### Post by Impavidus on 2016-12-20
If you want to uninstall anything using apt-get, you need the exact name of the package. It's case-sensitive, although I think always lower-case. But I haven't found any package on 16.04 named "appgrid" or "AppGrid".

How did you install the software? If from the software centre, that may have given you the title of the software, which may be slightly different from the actual package name. If you did not install the software using the package manager (apt-get, synaptic, software centre), you can't uninstall it that way either.
