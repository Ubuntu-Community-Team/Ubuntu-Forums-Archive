### Post by oldfred on 2022-01-24
Did you turn off Windows fast start up which sets hibernation flags on all NTFS partitions and may block ESP, so grub cannot install.
Or settings in UEFI may have a lock on updates to ESP and/or UEFI settings for security which are separate from UEFI Secure Boot setting.

What model HP?

You typically have to update UEFI, and SSD firmware even if new system.
And in UEFI change boot order once grub is installed. Grub uses efibootmgr to add new entry for "ubuntu", but HP seems to be the only one that resets order after first boot. But using UEFI to change order seems to work.
