Your BIOS seems configured correctly. Maybe your CD is not bootable.
1) Disable Quiet Mode in your BIOS. You need those error messages.
2) Try to boot another PC with your CD. 
     If you can't boot it, CD is bad, burn another CD.
     If you can, then your BIOS has a problem, you should contact/download and burn latest BIOS EPROM from the manufacturer
