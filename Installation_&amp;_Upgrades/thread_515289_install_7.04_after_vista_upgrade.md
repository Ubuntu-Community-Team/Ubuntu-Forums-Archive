### Post by merlinus on 2007-08-01
Use vista's built-in disk management tool to shrink its partition.

You may haved already done this, but delete temp and other unused files, backup important data, defrag several times, and if it exists in vista, set virtual paging drive to zero (you can set it back to original size after installing ubuntu).

Then shrink the partition to leave room for ubuntu.
