### Post by grahammechanical on 2014-06-14
Do you have another OS on that machine? If it is Windows 8 then do not use Wubi because it is not compatible with Windows 8. You could try this

1) at the first purple screen press Enter
2) at the next screen select the language (English is default) and press Enter
3) at the next screen with its options to Try Ubuntu and Install Ubuntu press F6
4) look for a menu at the bootom right of the screen and select nomodeset and press Enter
5) press ESC to get back to the Try - Install screen and make sure that Try is highlighted and press Enter.
