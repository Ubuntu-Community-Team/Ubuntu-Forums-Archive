### Post by egeezer on 2015-04-25
Your 14.04 64-bit is supported until 2019 - it is already an LTS release.  14.04.1 and 14.04.2 are updates of 14.04, the difference being that 14.04.2 is based on a more recent kernel, and if you install it as .2, the kernel will continue to be updated.  14.04 will retain the 3.13.0-xx kernel for its lifetime, and so will 14.04.1.

This is a more complete explanation:
[https://wiki.ubuntu.com/Kernel/LTSEnablementStack](https://wiki.ubuntu.com/Kernel/LTSEnablementStack)
