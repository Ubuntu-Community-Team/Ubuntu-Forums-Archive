You've provided no clear product or release details, nor details of your hardware.

I'll assume you're asking about Ubuntu Desktop, but you may mean Ubuntu Core, or Ubuntu Server; and it's best if you're clear with details so we're not guessing.  Ubuntu releases are *year.month* in format, ie. Ubuntu 22.04 LTS is the 2022-April release, Ubuntu 18.04 LTS is the 2018-April release, though a few *specialist* releases use the *year* format (*eg. Ubuntu Core 18 is the 2018 release*), which provides a clue as to when the product was put together, and what types of hardware it'll know about. 

Ubuntu LTS releases also come with kernel hardware stack options; so the ISO chosen to install can be handy (*ie. Ubuntu 20.04 LTS Desktop will install with a different kernel  (5.4) to Ubuntu 20.04.5 LTS* (5.15)) so it's best if you're precise with details.

You've also not ruled out procedural errors; did you [verify the ISO before writing to medi]("https://tutorials.ubuntu.com/tutorial/tutorial-how-to-verify-ubuntu#0")a? or the write to media (*this varies on release which you didn't provide so using whatever method applies to your unstated product*)
