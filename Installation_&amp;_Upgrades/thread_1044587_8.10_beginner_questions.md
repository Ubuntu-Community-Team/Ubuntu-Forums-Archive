### Post by cariboo on 2009-01-19
My fully configured Ubuntu desktop takes up 4Gb. If you want to create a usb device with a full Ubuntu install on it you must first install it on a computer, then use something like [Remastersys]("http://www.remastersys.klikit-linux.com") to create an iso that can be installed on a usb device or bootable DVD.

To create more empty hard drive space you have to defrag your hard drive at least twice and then use Vista's disk managment tools to resize your Vista partition.
