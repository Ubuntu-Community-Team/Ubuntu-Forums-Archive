### Post by darkod on 2010-06-22
Get a huge usb hdd next to you. Use the ubuntu cd and boot in live mode, Try Ubuntu option. In Places it should show you all partitions on all disks. It doesn't know what is what, so it will probably say 100GB filesystem, 200GB filesystem ,etc.

Click on the partition you need, or on one at a time, and it should mount it and open it. Copy all files needed to the usb hdd.

You don't even need to install ubuntu to read files on ntfs/fat32 partitions.
