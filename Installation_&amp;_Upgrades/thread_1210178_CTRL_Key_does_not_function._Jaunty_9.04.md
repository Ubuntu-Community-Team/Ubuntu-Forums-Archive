### Post by wetinwales on 2009-07-11
Hi All
Installed Jaunty on an AMD A8N-SL1 with GEFORCE 7800 etc.

The CTRL key does NOT function with SOME software eg:-

Thunderbird.

GIMP.. only works after page is maximised with F11. In all other pages it doesn't work.

File-System.. with certain windows CTRL does not operate.

Thus for example, trying to copy list of files to move or delete etc, this cannot be done other than one at a time.

With some software CTRL works fine.

Can anyone help?

Thanks
WiW

### Post by wetinwales on 2009-07-11
Well nobody seems to answer.
Meanwhile I've solved it.
<System Preferences <Windows <Movement Key <Choose 'Super'
make sure Ctrl isn't selected as the system thinks you are trying to move a window when you press CTRL.

My God! I solved a problem and posted the solution on the Forum. 
First time... and nobody's listening !!!!!
