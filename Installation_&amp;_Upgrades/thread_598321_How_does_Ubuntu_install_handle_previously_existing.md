### Post by raytard on 2007-10-31
It handles it very well actually. My last computer was win xp pro, win 2k pro, and win2k server, along with ubuntu, kubuntu, dsl, and fedora (i was looking for which distro I liked most). All of them ran fine and I never lost any of my windows documents. This is a rare occasion where vista makes something easier, it has the ability to repartition the free (unused space) on the harddrive. Just partition out a chunk of free space and install ubuntu on that free space, it will not harm any of the vista/xp files, so all your media, system files, programs, etc are intact, and ubuntu is on a separate partition.
