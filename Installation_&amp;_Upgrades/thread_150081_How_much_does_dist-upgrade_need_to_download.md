### Post by petervk on 2006-03-25
My guess is about 600mb. It's basically everything that is on the ubuntu dapper cd. If you download the [daily iso]("http://cdimage.ubuntulinux.org/daily/current/"), you are basically getting everything that would come in a dist-upgrade.

Oh, and if you run "sudo apt-get dist-upgrade" it will tell you how much you will need to download and then ask you if this is ok before procedding.
