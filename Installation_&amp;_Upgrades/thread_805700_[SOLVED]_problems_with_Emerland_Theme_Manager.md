### Post by barnex on 2008-05-24
Automatically downloading the themes in emerald is currently broken. To install themes manually, do not 'double-click' on them, but use the 'import theme' button in emerald. 
This worked for me, hope it helps you.

Press ALT+F2 and type "emerald --replace" without the quotes. This will turn on emerald. To enable it to start during system startup, go to "System --> Preferences --> Sessions" and add the same command under "startup programs". Hope this helps.
