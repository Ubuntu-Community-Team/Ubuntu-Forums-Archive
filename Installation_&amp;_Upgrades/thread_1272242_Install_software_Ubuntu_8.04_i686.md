Using dpkg to install packages 
cd directory 
sudo dpkg -i package_name.deb 
*Note: replace directory with the directory in which the package is stored and package_name with the filename of the package. 
Getting a list of recently installed packages. 
[COLOR=red]zcat -f /var/log/dpkg.log* | grep "\ install\ "[/COLOR] | sort (see Below) 
 Step 1 ->Installation commands 
apt-get install <package_name> 
apt-get build-dep <package_name>  
aptitude install <package_name> 
Step 2 -> Typical usage example 
apt-cache search Jdownloader 
apt-cache show torcs 
apt-cache policy torcs 
apt-get install torcs 
dpkg -L torcs | grep /usr/games/  
------> I would like to save it in Applications --Internet 
apt-cache search torcs 
Step 3 
Setting up apt-get to use a http-proxy 
export http_proxy=http://yourproxyaddress:proxyport 
(e.g) [http://jdownloader.org/download](http://jdownloader.org/download) 
Step 4 APT configuration file method 
gksudo gedit /etc/apt/apt.conf 
Acquire::http::Proxy "http://yourproxyaddress:proxyport";
