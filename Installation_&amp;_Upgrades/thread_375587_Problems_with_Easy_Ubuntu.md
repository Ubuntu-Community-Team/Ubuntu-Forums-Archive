### Post by tgm4883 on 2007-03-04
You might not think that this is much help, but just food for thought.

Using EasyUbuntu or Automatix2 (I used to use automatix2) can cause problems later on when upgrading (so I hear).  Everything that they install is installable from other sources (most easily using synaptic).  A repository is where the data is stored that Ubuntu needs to update, install programs and such.  The sources.list is the file where the location of these repositories is stored.  I use the one from here
[http://ubuntuguide.org/wiki/Ubuntu_Edgy](http://ubuntuguide.org/wiki/Ubuntu_Edgy)

It also gives instructions on how to install most of the other programs I like.  Check it out as it may offer you some help.  

As for your problem with EasyUbuntu, I cannot help with that as I do not use the program.
