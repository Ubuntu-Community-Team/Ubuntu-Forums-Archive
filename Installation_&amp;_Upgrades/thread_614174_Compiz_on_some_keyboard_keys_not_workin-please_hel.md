### Post by bulldog on 2007-11-15
If you want to reinstall,just boot the live cd and install
when you come to the partition part choose manual
Mount your / as / set it to format and set bootflag enable format ext3
mount swap as swap and format swap.

Just backup your data before you begin,it will erase all the data on those partitions.
