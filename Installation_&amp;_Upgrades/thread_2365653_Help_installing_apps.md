### Post by TheFu on 2017-07-09
Welcome to the forums.

Installing applications on Ubuntu is very different from Windows. A few primers:
[http://www.dedoimedo.com/computers/ultimate-linux-guide-for-windows-users.html](http://www.dedoimedo.com/computers/ultimate-linux-guide-for-windows-users.html)
[http://linux.oneandoneis2.org/LNW.htm](http://linux.oneandoneis2.org/LNW.htm)
[https://help.ubuntu.com/stable/ubuntu-help/](https://help.ubuntu.com/stable/ubuntu-help/)

If you read those in that order, many differences will be pointed out in the first 90 minutes or so. That will make some basic differences clear and save hours, days, weeks of frustration.

To get the best help here, please show exactly what you typed and exactly what the result was. Any errors should be posted too.  All of this is usually best done from a terminal window so the exact program, userid, directory and inputs are shown.  Then the errors will be shown on that same screen, which can be copy/pasted as text (avoid images!) here.
