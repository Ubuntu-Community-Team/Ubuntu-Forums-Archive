### Post by tuxxy on 2008-07-23
did you ruin the .BIN and install java?

```
sudo update-alternatives --config java
```

Run that in terminal and it will allow you to select which java your system should use, then its just a case of installing the java plugin to your mozilla/plugins folder.  To check you installed the plugin type

about : plugins into browser and it should list the java, I use supplied GCJ and it runs good for me.
