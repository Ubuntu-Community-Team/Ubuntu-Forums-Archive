### Post by dstew on 2008-05-26
It could be several things. First, the mirror could truly be unavailable, because everybody is downloading Ubuntu. Or, it could be a problem with the DNS server resolving the mirror name.

I think the warning about having an un-useable system is referring to your new installation, not your other partitions like Windows. It just means that the installer could not finish.

If it complains about all the mirrors, it could be a problem with the DNS server resolving the domain name of the mirror. Sometimes, the issue is with your router. If you can plug in directly to your modem, and bypass your router, it might work better.
