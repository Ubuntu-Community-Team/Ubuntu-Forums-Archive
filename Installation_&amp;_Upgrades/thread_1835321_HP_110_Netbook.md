### Post by Mark Phelps on 2011-08-29
I wouldn't try virtualization on that PC.  Netbooks are notoriously underpowered, with low speed processors and little memory -- both of which spell disaster for trying to use virtualization.

If you CAN boot from USB on the netbook, you should research Unetbootin to see how to make a USB stick containing an Ubuntu distro -- and then boot from that, using the Try Ubuntu option.

Do NOT force an install until after you try using Ubuntu for a while.  This will give you a chance to see how well it detects your hardware and loads the proper drivers.
