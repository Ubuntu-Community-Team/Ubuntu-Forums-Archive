### Post by Kareeser on 2009-04-29
As far as I know, lisati's suggestion creates an Ubuntu bootable "Live-USB" type of structure, so you can install Ubuntu on other computers using your USB key instead of a CD.

If you were inquiring about installing Ubuntu straight onto a USB key, you can go ahead by using a LiveCD to install it onto the USB key... I don't know how to put it more simply :)
