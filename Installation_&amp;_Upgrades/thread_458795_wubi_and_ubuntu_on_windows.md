### Post by aysiu on 2007-05-30
Well, you can *use* the live CD without partitioning the drive... that's why it's called a "live" CD. But if you want to install a dual boot with the live CD, then, yes, you will have to partition the drive.

Wubi allows you to install Ubuntu in a dual boot with Windows without having to repartition, though.
[http://www.cutlersoftware.com/ubuntusetup/wubi/en-US/index.html](http://www.cutlersoftware.com/ubuntusetup/wubi/en-US/index.html)
