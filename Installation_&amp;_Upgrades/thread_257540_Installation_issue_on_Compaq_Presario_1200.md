### Post by ajgreeny on 2006-09-14
I think you need more ram for the live CD install (192 minimum, I think)  Get a copy of the alternate installation CD and then when you have it installed and up and running (slowly) try xubuntu-desktop, as it is much better on older lower ram machines.
