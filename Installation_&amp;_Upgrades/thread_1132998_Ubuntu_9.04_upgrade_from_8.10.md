### Post by snowpine on 2009-04-22
Hi Anthony, you can upgrade to 9.04 right now if you like. Simply open a terminal and enter the following command:

```
gksu update-manager -d
```

Obviously there may be some minor changes between now and the final release sometime tomorrow.

However, if you will be without internet access for 2 weeks, and you need this computer for work or other important tasks, I highly recommend NOT upgrading to a brand new release. There are always bugs in any new release, and with no internet access, you will not be able to download any fixes.
