### Post by oldfred on 2013-08-23
If you go into UEFI menu and choose the kubuntu entry does it not boot. You do need secure boot off and UEFI on. But CSM also has to be off. Some systems, I think HPs only have a secure boot on or off setting and under CSM it trys to boot from efi partition and if not found tries MBR in BIOS/CSM mode. You may have to experiment as every vendor's UEFI is different and some vary even by model.

HP's also have a lot of efi repair/configuration files in efi partition. Boot-Repair automatically creates efi boot entries in its 25_custom so you can boot them. But you probably do not need most.
