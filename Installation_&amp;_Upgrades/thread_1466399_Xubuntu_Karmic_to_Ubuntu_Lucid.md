### Post by zvacet on 2010-04-30
You can switch from xfce to gnome following [this]("http://www.psychocats.net/ubuntu/puregnome") guide.After that 

```
sudo apt-get update && sudo apt-get upgrade
```

Then go to the synaptic and upgrade via network.
