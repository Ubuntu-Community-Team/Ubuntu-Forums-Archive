### Post by Bucky Ball on 2013-01-27
Welcome to the forums. Boot from the install USB and when at a desktop launch Gparted. Right click the Win partition to make sure it is unmounted then simply delete. You should have an icon on the desktop to install Xubuntu. 

You can let it do its thing automagically or select Something Else when you get to the partitioning section if you want more control over partitions and sizes. If you partition manually then you can delete the Win partition during that section of the install instead of previously.

I think there is also an option to 'Install over Win' but, again, you will have no control over partitions and sizes and get what Xubuntu gives you. This might not be what you want.
