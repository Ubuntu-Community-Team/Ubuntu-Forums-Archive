### Post by oldsoundguy on 2008-01-01
big difference between lappy and desktop.  HP lappy is very proprietary and built on specifications, whereas the desktops are built from purchased parts (I have an older one running 7.10 and it is an MSI motherboard.) Still has the HP splash screen when it boots, but in no way is an HP! (works fine).
As to ATI problems .. the answer is yes.  Only the generic drivers in Ubuntu seem to work, but it WILL, at times, install the "restricted drivers".  Crap shoot on that one.  And the install program "Envy" (3rd party program) just does not do it with ATI (tried it on two ATI equipped machines and in both cases, it did not see a 9800 card)  (nVidia works, however .. and VERY WELL .. right now have one Linux so equipped.)
