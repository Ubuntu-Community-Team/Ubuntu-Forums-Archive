I want to make sure this gets seen by anyone with a 64 bit system with RAID that has boot failures trying to install 9.10.

*AMD 64 system with serial RAID (2 500gb Samsung HDs for me)*

1.  Use the alternate install CD for AMD 64 !!!!!

2.  Choose F6 from install menu and choose nodmraid option.

3.  When asked if you want to activate RAID, choose **NO**.

4.  I used the "guided partition - use entire disk option.  It correctly identified my 2 500gb hard drives.

5.  Finish install normally and reboot.  Mine worked flawlessly doing this when no other install would allow my machine to boot up.

I do hope this helps even one other person, finding this answer took awhile and a lot of frustration.
