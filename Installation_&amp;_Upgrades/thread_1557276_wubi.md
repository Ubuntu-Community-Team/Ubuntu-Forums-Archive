It's better to download Ubuntu yourself (using bittorrent is fastest). 

Then you can burn it to a CD and run wubi off that - it's also useful as a recovery CD later. You can also boot the CD and "try without installing" to make sure Ubuntu runs ok on your machine.  

If you can't burn a CD, you can still run wubi off the downloaded .iso - just download wubi.exe and place it in the same folder as the .iso before running. Then everytime you reinstall it doesn't download the image again.
