### Post by Mercury_Alpha on 2009-04-21
If you installed the beta and have downloaded all the updates that have been released since then, you're running the RC. In a couple days, the final version of Jaunty will be "released," and you'll upgrade to just by downloading the finalized packages.

You only need "update-manager -d" when you're upgrading to a new release number. (The beta is still 9.04, it's just not the retail version.)
