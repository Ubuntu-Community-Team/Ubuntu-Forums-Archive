### Post by leaber on 2005-10-18
With the Live CD you can try Ubuntu without installing anything in your hard drive. Ubuntu and the programs in it, are loaded into memory from the CD (thats why its gonna be really slow! )
I would recommend you to try the live cd before installing Ubuntu, just to make sure all your hardware is compatible with ubuntu and you like it.

To do this, you first have to enable your PC to boot from the CD/DVD drive, this option is in the BIOS. After you enable it, just insert the live CD in the drive and try UBUNTU ! 

If you like it I can help you with the installation, but there is a lot of info in the forums and goolge about Ubuntu installation.
