### Post by maclarke on 2010-01-15
I'm running 9.04 from a USB stick.  So it seems to be doing all that 8.10 does. But, I have a dreaded defunct ATI card. But I can play youtube videos, CNN video, Is there any other check I can do to make sure the ATI card is working?

### Post by haribol707 on 2010-01-16
Prepare your USB in persistent mode (~1GB free space recommended), bootup ubuntu and install the restricted driver when prompted. If not, install the driver as you did with 8.10. If there are no problems, you should be fine with 9.04.
