### Post by lisati on 2007-09-10
I found that the "Live CD" worked better on my desktop than on my laptop (a Toshiba Satellite from the A100 range of models) and ended up using the "alternate" cd to install Ubuntu on my laptop.

You can download the alternate CD (which uses text based installer) from [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) and by checking the checkbox just below the "Start download" button. Unfortunately it probably won't give you the chance to "play" before you install, but since you have access to a desktop where the "Live CD" works you can always check it out there.
