Just a quick disclaimer: I have made a very similar post in the programming section -- it evolved out of a programming question that I had earlier.  My apologies in advance if it counts as a double-post.

My question is this: Can anyone refer me to a well written tutorial about the in, outs, and how-tos of installing software on an Ubuntu system? I certainly agree that there is a rather copious amount of literature and documentation about this subject, to be found in man files, websites, etc.  However, one problem that I am having is that many (if not most) of these articles and tutorials seem to assume that the reader has a great deal of prior knowledge about Ubuntu and Linux in general.  Often times, these articles only address one narrow aspect of the software installation process in general, or bog down in terminology-dense explanations while omitting basic, crucial concepts of how to install software on Ubuntu (or any Linux system).
