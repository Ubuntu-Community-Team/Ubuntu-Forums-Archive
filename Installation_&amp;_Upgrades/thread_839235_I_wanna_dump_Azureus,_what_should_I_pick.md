I second these two user's motions. I used to use Ktorrent but moved to Deluge a while back.

To make this decision simple, I'll propose the following:

If you're using Gnome; Xfce or just any environment where you're predominantly using GTK+, go with Deluge.

If you're using KDE or most of your apps are Qt, use Ktorrent.
