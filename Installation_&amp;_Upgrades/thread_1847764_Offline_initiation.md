### Post by plucky on 2011-09-21
> Is an internet connection required to install 11.04?

No

It will install the system if there is no internet connection.

> The graphical installer seems to suggest it does; 

It will try to set up a connection and ask if you want to download updates,but also gives you the option to not set up a network connection during the install.

### Post by sanderd17 on 2011-09-21
If you have an internet connection, it will offer to download extra languages, extra codecs and general updates.

If you want a more clear installer (for a long-time linux user), you can use the alternate install CD. This doesn't have a live environment, but a text-based installer.
