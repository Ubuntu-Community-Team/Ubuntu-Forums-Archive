### Post by mikewhatever on 2011-10-09
The process is rather simple, so that I doubt there is a guide.

1. Create some disk space for Windows, either unallocated space or an NTFS partition. Use Gparted from the Ubuntu live CD.

2. Install Windows.

3. Reinstall Grub from the Ubuntu live CD.
