1. Go to [http://www.ubuntu.com](http://www.ubuntu.com), download the operating system to the computer.Case example: Ubuntu-Netbook 10.04 i386
2. Insert blank CD-R into the DVD/CD-R drive
3. Locate for that downloaded file which is ubuntu-10.04-netbook-i386.iso (699.8mb)
4. Right-click iso file/write to disk
5. Disk to disk screen appears/select burn.
6. CD-R will register itself as Ubuntu-Netbook 10.4 i386 when completed. The total transferred file size is 1.4GB

1. Insert installation CD in the DVD/CD-R drive
2. Insert at least a 2GB blank USB drive to the computer.  The startup process deletes all file in the USB drive
3. Go to main menu/system/administration/startup disk creator
4. Make startup disk screen appears/select make startup disk
5. When complete, the total transferred file size is 827.3mb
