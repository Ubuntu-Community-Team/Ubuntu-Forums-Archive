### Post by goodfella on 2007-01-08
I have just recently put together a pc with a 64 bit amd processor and set up a windows xp pro x64 and ubuntu 6.06 AMD64 dual boot.  Typically people expect windows to have better hardware support than linux but in my case, it was the opposite.  I have had fewer problems with hardware under ubuntu than under windows xp pro x64.

Here is a list of the hardware which I either could not use or had to download special drivers for:

Linksys wmp54g (version 4):  had to download 64 bit driver and install
serial port pci card:  vendor only supplies 32 bit driver therefore will not work in x64 windows [see link]("http://www.microsoft.com/windowsxp/64bit/facts/benefit.mspx")
Microsoft Natural Ergonomic Keyboard 4000:  had to download driver and install.

Under linux my hardware worked natively.  All I had to do for my wireless card was to setup the /etc/network/interfaces file and my wireless network was up and running.  I am thinking that microsoft is spending more time on vista 64 bit support than xp.  Good thing I have my copy of xp 32 bit.  Guess I will have  to triple boot unless somebody has another solution.
