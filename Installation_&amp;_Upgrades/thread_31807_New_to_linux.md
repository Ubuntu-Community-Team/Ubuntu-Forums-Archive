### Post by Professor X on 2005-05-04
Welcome xjlx.

[The ubuntu guide](http://www.ubuntuguide.org) covers how to install multimedia codecs, streaming audio/video, and includes some great tips.  If you want to get a CLUE (command line user experience), I recommend checking out the guides at [the linux documentation project](http://en.tldp.org).  Another excellent tutorial is [rute](www.icon.co.za/~psheer/book/index.html.gz), but it may be a bit too advanced for you right now.  You will likely find yourself lost at times, but it is important to remember that linux will require you to get your hands dirty.

Cheers
