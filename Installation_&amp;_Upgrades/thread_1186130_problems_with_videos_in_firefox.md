### Post by Crunchy the Headcrab on 2009-06-13
You need flash to watch youtube videos.  If you are funning 32bit Linux, then you should be able to download from flashplayers website.  

If you are using 64bit Linux then go to [http://labs.adobe.com/downloads/flashplayer10.html](http://labs.adobe.com/downloads/flashplayer10.html) and download the 64bit alpha at the bottom of the page.  Create a folder in the .mozilla folder (found in /home) and name it plugins.  Then unzip the file from [http://labs.adobe.com/downloads/flashplayer10.html](http://labs.adobe.com/downloads/flashplayer10.html) into the plugins folder.

I'm a newb to Linux, so you might want to wait until a more knowledgeable member comes along, but the above worked for me.
