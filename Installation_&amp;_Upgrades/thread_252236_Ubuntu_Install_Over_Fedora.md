### Post by meng on 2006-09-06
There should be a point in the installation where you have different options to install (take up whole disk, take up free space, or manually partition). What you probably ought to do is manually partition, make sure that you are installing over the partition holding root (/) and format that partition (do not keep existing data).

BUT, if you have personal data in your /home, and you don't have a separate partition for /home, then all this data will be wiped.
