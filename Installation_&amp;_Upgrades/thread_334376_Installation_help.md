### Post by yager on 2007-01-08
I am not an expert on the LiveCD, but here are a few questions to help others.

Do you have the menus on the screen and can you start the terminal program from the menu?

If you can start the Terminal, check the directory for the Desktop to see if there are any files present.

```
cd Desktop
dir
```

if you want to start the install routine, you could execute it from the terminal window at this point or keep pushing forward on getting the icons back onto the desktop.

Hope this gets you going.
