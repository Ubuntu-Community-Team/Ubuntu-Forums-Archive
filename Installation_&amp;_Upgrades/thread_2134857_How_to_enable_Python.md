### Post by mchid2 on 2013-08-27
```
sudo apt-get install [COLOR=#000000]html2text[/COLOR]
```
and then . . .
```
sudo apt-get install python-[COLOR=#000000]html2text[/COLOR]
```

The next time you open a terminal, it should be clean. 8-)
