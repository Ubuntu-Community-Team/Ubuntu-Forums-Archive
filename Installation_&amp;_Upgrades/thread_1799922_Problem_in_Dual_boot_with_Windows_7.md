### Post by Blasphemist on 2011-07-08
That is a video issue message. Please see this sticky thread. [http://ubuntuforums.org/showthread.php?t=1743535](http://ubuntuforums.org/showthread.php?t=1743535)

The first 3 posts will help you make changes to resolve it. With 11.4 there is a change in the handoff of video modes during boot. You at least aren't having a problem running ubuntu but you'll need to make some changes in grub 2 to allow it to do display in a resolution it can handle at the time.
