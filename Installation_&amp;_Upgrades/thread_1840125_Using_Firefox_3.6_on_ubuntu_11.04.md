### Post by vasa1 on 2011-09-07
You can get it from here:
[http://www.mozilla.org/en-US/firefox/all-older.html](http://www.mozilla.org/en-US/firefox/all-older.html)
For example, there is this US version of 3.6.22:
-http://download.mozilla.org/?product=firefox-3.6.22&os=linux&lang=en-US-
and just above it is the British English version.

My guess is it will come as a compressed file (*.tar.bz2) which you can move to the directory of your choice in /home and then uncompress by right-clicking on it and choosing "Extract here".

You'll then have to ensure that you click on the exe that is present in the resulting directory rather than any Firefox anywhere else. You will also have problems with the profile unless you already know that.

If you are planning to use just 3.6.22, then uninstalling any other version may simplify things. Otherwise, you could consider starting specific versions with the help of the Profile Manager, though how exactly that's done, I'm not sure.
