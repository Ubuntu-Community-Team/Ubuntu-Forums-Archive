### Post by ed-koala on 2009-11-15
World of Warcraft had major install issues on Wrath of the Lich King not allowing to install from multiple disks.  The solution was to copy all the disks to one folder on the HD and install from there.

Definately check WineHQ for their database of compatible games.  If it's rated bronze or worse, you'll almost certainly have to use Windows to enjoy your game, for now.
