### Post by dabl on 2007-08-16
I would first suspect the CD as having a defect.  You should check your downloaded ISO file's md5 checksum against the value shown alongside the download file on the Ubuntu site or mirror where you got it.

You should burn the installation CD at 4X speed.

You didn't say whether you are using the Live CD or the Alternate.  I personally like the Alternate better because it gives more visibility and control over the process.

HTH  :)
