### Post by wingman1 on 2011-07-29
Okay so. When you boot up your PC you should get some sort of a textual menu aka. GRUB bootloader. there you have some ubuntu kernels eg. Ubuntu generic 2.36.1 or something like that. The first one the latest veersion installed. below it you have the recovery mode option.go there and you'll get another menu in wich you should go to the failsafeX option. This will run your ubuntu with low graphics and it worked for me.
