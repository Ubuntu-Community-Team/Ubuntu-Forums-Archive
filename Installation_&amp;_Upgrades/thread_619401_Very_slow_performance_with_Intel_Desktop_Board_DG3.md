### Post by mcTNT on 2007-11-21
Hi,

I have purchased 2 new computers with Intel Desktop Board DG33TL and i want to install Linux on it. First i try to install Debian Etch or Lenny but my nic and chipset not supported by Debian kernel.

Then i try to install Debian from Stable installer ISO image with a modification by Kenshi Muto [http://mirror.home-dn.net/d-i/](http://mirror.home-dn.net/d-i/). Setup complete fine. My nic detected. But after reboot system performance is very slow.

So i try to install Ubuntu 7.10 Server. Setup complete fine. My nic and dvd detected. And once again after reboot system performance is very slow. It takes about 10 minutes to start system. Every command that I am lunching takes some minutes. I can't find any warnings or errors in system logs.

Any suggestions?

---

### Post by mcTNT on 2007-11-22
Problem solved! I install Ubuntu AMD64 instead of i386.
