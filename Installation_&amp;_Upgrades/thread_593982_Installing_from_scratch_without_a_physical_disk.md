### Post by domain on 2007-10-27
Yes you can install Linux distr. without using DVD or CD :
1) Download the ISO file for the OS you want to install 
2) Mount it to a Virtual DVD-ROM, OR using a Virtual Machine Program ( like VirtualBox ) you can select to use an ISO image file for the installation.

BUT i tried to do so, with the new distr. Ubuntu 7.10 and it did not work out although it worked with Fedora 7.0 !
