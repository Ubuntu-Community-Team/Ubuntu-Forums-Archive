### Post by jespdj on 2007-11-27
What software are you using to put the downloaded ISO image on a CD?

Note that it does not work if you put the ISO on a data CD as a big file. You need software that knows how to write an ISO image to a CD.

See this: [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)
