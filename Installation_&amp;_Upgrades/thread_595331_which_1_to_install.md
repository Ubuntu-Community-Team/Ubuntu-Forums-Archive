### Post by ofb on 2007-10-28
178mb? Surely that's a typo? Perhaps 128mb?

Ubuntu and Kubuntu require at least 256mb to install. Xubuntu requires 128MB RAM to run but 192MB RAM to install.

I'd say get more RAM or look at different distros. For instance, Puppy Linux, which is about the smallest thing out there, or Damn Small Linux, or DSL-N, which is Damn Small Linux with many more features for a less stripped-down experience.

There's also Fluxbuntu, but since that project can't make a proper website lately, I'm less than encouraged that this is a good time to try their distro.

So, yeah - you don't meet the minimum specs for an official Ubuntu project. Go burn the LiveCDs of the lightweight distros mentioned and see if you like them.
