### Post by pololo on 2014-07-10
The easiest way to install Citrix Receiver is download it from the web of Citr[COLOR=#000066]__[/COLOR]ix [www.citrix.com](www.citrix.com), in the main page select "Download", at the left side under "Find Downloads", in "Select product" select "Citrix Receiver" (the products not are ordered alphabetically), and in "Select download type" select "Receiver for Linux", and click "Find".
Next select "Citrix for Linux 13.0". At the end of the new page, you must select your platform (for ARM processors, for 32-bits systems or for 64-bits systems).
Now you have two products for download (Citrix Receiver and USB Support Package; first install receiver) as tar, deb or rpm format; download the packages in your prefer format and install.
I installed Citrix Receiver downloading the .deb files and using GDebi; previously I installed GDebi Installer Package from the Ubuntu Software Center.
