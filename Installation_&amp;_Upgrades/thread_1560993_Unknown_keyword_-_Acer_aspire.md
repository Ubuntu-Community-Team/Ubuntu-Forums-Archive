### Post by techxcell on 2010-08-25
I'm afraid the "facility" did not configure the USB boot options properly. It happens a lot.

I suggest you use the tried and tested [UNETBOOTIN]("http://unetbootin.sourceforge.net") utility. Click on the name to download it. If on Vista or 7 run the [UNETBOOTIN]("http://unetbootin.sourceforge.net") with administrator privileges or it will not work properly.
