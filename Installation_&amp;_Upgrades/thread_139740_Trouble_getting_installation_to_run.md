### Post by Sutekh on 2006-03-05
[QUOTE=voodoo_chile]
I downloaded Ubuntu 5.10 and burnt the ISO Image as a bootable disc in Roxio Easy Cd Creator and then tried to use that to install Ubuntu.[/QUOTE]
I think that may be the problem.  The ISO must be burnt as a image, not a data disc not a bootable disc.

I don't know Roxio Easy CD Creator, but found instructions on the web for burning an ISO image

> 1. Insert a blank CD into your CD writer.
2. Launch Easy CD Creator.
3. In the File menu, select "Create CD From Disc Image."
4. Browse to the location where the ISO image was saved on your hard drive.
5. Click on the "Files of type:" dropdown box and select "ISO Image Files (*.ISO)."
6. Select the ISO image file.
7. At the CD Creation Setup dialog box, make sure the Write Method is set to Disc-At-Once and Close CD.
8. Click on OK.
9. Click on OK once again.
10. Your CD-ROM will be created.

Then proceed as you did before, when it boots from the CD it should take next to no time to get to the main screen

Then just press **Enter** to start installing Ubuntu
