### Post by ruibernardo on 2007-11-02
And you can't. 

You would have to upgrade to Edgy, then to Feisty and then to Gutsy, each time with lots of downloads (1 GB each time, to say the least) and upgrade time (hours?) plus the probability that each upgrade can break something. 3 upgrades is very risky and will give you certainly some headache.

If you do have a /home partition, install a fresh Gutsy, then install the programs you have installed on your Dapper. Type "dpkg --get-selections > installed.txt" to have an idea of what packages you have installed now.
