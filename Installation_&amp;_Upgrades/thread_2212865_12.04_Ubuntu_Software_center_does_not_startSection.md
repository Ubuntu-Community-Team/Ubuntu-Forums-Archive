### Post by Kirboosy on 2014-03-24
It most likely broke after you added the MediUbuntu Repository, right?

That repository is really old and doesn't work anymore. You should remove it from your system and see if that fixes the problem. [LibDVDCSS]("http://www.videolan.org/developers/libdvdcss.html") is the replacement package to MediUbuntu

You can view your sources under the following location
```
/etc/apt/sources.list
```

Hope that helps!
