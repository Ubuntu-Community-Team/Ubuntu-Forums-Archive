### Post by Catsworth on 2007-06-11
The CD that you might 'buy' will be exactly the same as what you downloaded.  Ubuntu doesn't give the initial user root privileges by default, you accomplish 'root' tasks by use of the 'sudo' command.

If you do a search of the forum you will find many explanations of why and how this works.
