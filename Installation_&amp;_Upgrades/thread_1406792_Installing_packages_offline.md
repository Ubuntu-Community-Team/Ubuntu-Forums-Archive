### Post by labinnsw on 2010-02-21
Using the Ubuntu disc, install ndisgtk, ndiswrapper-common and ndiswrapper-utils-x from Synaptic.

Starting from step 2 in the link, [URL="http://www.cyberciti.biz/faq/linux-ndiswrapper-wpa_supplicant-howto/"]
configure ndiswrapper using the windows driver for your wireless adaptor.[/URL]

With internet access now available, you can install the other packages normally.
