Did you verify the md5sums of the downloaded iso file and the install CD that you burned? If you don't know how to do verify md5sums, see this:
[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)
The md5sum check verifies that the downloaded iso and the CD you burn are not corrupted. I would first rule that out before looking into other things. The Lenovo laptops are well supported in linux.
Remember to burn the install CD at the slowest possible speed.
