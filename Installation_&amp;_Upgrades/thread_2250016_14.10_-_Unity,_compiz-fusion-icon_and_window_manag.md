This message probably correlates with the issues I face after logging in to Unity. After a few seconds the graphical system shows some confusing behaviour. Next to starting i.e. a file manager (thunar, files, nemo) I can work in the applications window but if I drag this window around there are "snapshots" every one or two mm (like the mouse traces you can activate - except the snapshots remain).
