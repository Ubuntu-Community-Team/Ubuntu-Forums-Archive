### Post by twtduck.ii on 2011-11-19
A couple questions:
1. Do you have Windows?
2. If so, what version (XP,Vista, 7)
3. Have you checked the disk for defects?
4. Have you tried reburning the disk at a slower speed to ensure accuracy?
If you don't have Windows, it's quite probably a problem with your wireless card. I know somebody where Windows didn't detect his wireless card but Ubuntu did, so it's probably not an issue with detecting the card if Windows sees it.
Also, try installing without the wifi and install Ubuntu-Restricted-Extras (from software center) and updates later. Ubuntu-Restricted-Extras are the packages for commonly used formats that cannot be installed directly from the cd (such as mp3 and flash).
Hope it helps!
