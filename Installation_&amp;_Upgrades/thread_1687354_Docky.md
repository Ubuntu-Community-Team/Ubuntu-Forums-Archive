### Post by jwcalla on 2011-02-13
To get rid of the bottom panel simply right-click on an empty portion of the panel and select "Delete This Panel".

To add the Window List to the top panel you can right-click on an empty portion of the panel and select "Add To Panel..." then select "Window List".  You may need to expand the size of the window list applet to be able to see the window titles.

Alternatively you can make the bottom panel auto-hide by selecting that option in the "Properties" on the right-click menu.  There is also an option in "Properties" to move the bottom panel to the top, if you'd like to have two top panels.
