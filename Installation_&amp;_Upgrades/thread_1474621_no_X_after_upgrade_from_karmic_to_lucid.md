This is how to get an xorg.conf back in 9.10 (or 10.04, I guess):

sudo Xorg -configure

That creates a file, xorg.conf.new in ~/ (the home dir of the user). 
Edit it to add whatever lines you need.<<<for me it genrated a conf with ALL options available to the driver which were commented out for ease of use <<<<<

Then copy to the correct location:

sudo cp /path-to-user-home/xorg.conf.new /etc/X11/xorg.conf

Reboot. If it still doesn't seem to be using the new settings, run:

sudo dpkg-reconfigure -phigh xserver-xorg

and reboot.
