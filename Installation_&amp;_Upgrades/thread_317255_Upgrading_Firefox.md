### Post by ciscosurfer on 2006-12-12
Which version of Ubuntu are you running?  And might I ask why you don't want to upgrade to 2.0 (many security and bug fixes were incorporated into the 2.0 version -- I would just upgrade to 2.0)

In Dapper, the repositories have version 1.5.8
In Edgy, the latest version in 2.0

Version 1.0.7 is very old.  I would strongly advise against using it.

If you want other versions that are older (but aren't supported through Ubuntu) you can search for them on the Mozilla site and download them from there.

If you want to upgrade to 2.0 (again, recommended) you can check this page if you don't want to download through the repos: [http://www.psychocats.net/ubuntu/firefox](http://www.psychocats.net/ubuntu/firefox)
