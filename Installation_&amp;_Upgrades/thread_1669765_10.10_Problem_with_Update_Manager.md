### Post by dino99 on 2011-01-18
have you checked your sources ? (use only genuine ubuntu repo)

open synaptic:

 on left side look for broken package(s)

remove/purge (right click) the package you want to remove

install gtkorphan, then use it (system admin orphan) by selecting ALL on the dialog bottom box (after synaptic closed)

into a terminal:
sudo dpkg-reconfigure -phigh -a
(could be long, dont stop it)
