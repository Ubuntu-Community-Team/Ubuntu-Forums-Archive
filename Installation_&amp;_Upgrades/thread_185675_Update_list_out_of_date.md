### Post by Jussi Kukkonen on 2006-06-01
Software versions are frozen when the operating system is released -- that's the point of having different releases and not just one constantly updating operating system (that would never be stable). Only security updates are allowed.

So the bad news is that Breezy won't be getting version 2.14 of gnome-power-manager. If you want it, you need to upgrade the OS (or install a newer version by hand).

The good news is that you won TMTIOP -- "The most TLAs in one paragraph"-competition.
