### Post by Tatty on 2008-04-17
If you installed it from the repositories then it should appear under Applications->Internet->Firestarter.

If it is not there then you can launch it from a terminal with ```
gksu firestarter
```

If you want to add it to your menu then right-clic on the menu and create a new menu item for it, with gksu firestarter as the command to launch.
