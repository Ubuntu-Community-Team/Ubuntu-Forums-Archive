### Post by Pawe_Jaworowski on 2014-08-17
Hello guys,

I have problem with intalling ubuntu from pendrive.

So, I have downloaded the newest version of ubuntu. I haved used two programs to install ubuntu on pendrive: Universal USB and LinuxLive USB. In both I had the same situation. Unetbooting do not work for me (do not respond during installing boot).
After that I go to the bios, choose UEFI as first boot option. Restart and I could see menu with 4 options (Try, Install ect.). In addition, before menu I get error, but it prints so fast that i can't read it. Error starts with "Can't" :P 
Typing any options creates blank screen. So, after typing Try or Install nothing happends, just blank screen.

Why it happends?

Thank you for any advice.

EDIT: I have found solution. The problem was with UEFI. I did some mess in my BIOS and I disabled UEFI and boot not from Flash USB, but directly from my pendrive. It works!
