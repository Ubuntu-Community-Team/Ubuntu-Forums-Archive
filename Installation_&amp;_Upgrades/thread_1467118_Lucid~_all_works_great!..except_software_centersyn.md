### Post by coffeecat on 2010-04-30
The slowness is probably down to the fact that the servers are being overloaded at the moment. Just be patient - it will settle down in a day or so - or change the download server in Software Sources. You may find that one of the mirrors is less congested than the main servers.

As far as VLC and ubuntu-restricted-extras not appearing, I don't know. If you managed to install them they should show up in Synaptic, but check this. In Synaptic, Settings > Repositories > Ubuntu Software tab. Make sure the universe and multiverse boxes are ticked. Then do a 'Reload'.

If VLC installed correctly, it will appear in the (gnome) menu under Applications > Sound and Video.

### Post by coffeecat on 2010-04-30
> **Rasa1111 said:**
> 
 patience i will have then. lol


But if you do get impatient :wink: , have a look at this:

[https://help.ubuntu.com/community/Repositories/Ubuntu#Download%20Server](https://help.ubuntu.com/community/Repositories/Ubuntu#Download%20Server)

If you click on the 'Select Best Server' it will ping all 350 and tell you which responds quickest. I was getting very slow download speeds after a fresh Lucid install this morning. My Software Sources download server was set to 'Server for UK' so I changed it to one of the other UK mirrors and immediately got speeds near the maximum of my broadband connection. It's worth doing - it spreads the load around.

By the way, once I'd changed the server I successfully installed both ubuntu-restricted extras and VLC so they're both still there.

Have fun!
