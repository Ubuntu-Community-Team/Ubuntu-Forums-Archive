### Post by jkoyle on 2007-10-18
I too was having serious issues trying to update today - timeouts, couldn't update sources, etc.  The default sources (in the US) are setup to use us Ubuntu's servers.

I manually went into Adept manager and configured the sources to use a *mirror*.  In my case I chose a mirror that was in the same state as myself.

After doing this I all the packages have downloaded in < 40 minutes and it's now installing all the debs.
