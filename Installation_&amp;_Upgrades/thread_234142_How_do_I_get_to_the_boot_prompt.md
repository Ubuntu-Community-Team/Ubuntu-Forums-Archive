### Post by A2A on 2006-08-11
Hi gary0usa,
I would go ahead and select "Start or Install Ubuntu"  and load the Live CD to the desktop.  From there click on Applications -> Terminal

Than you may type in "server" as instructed, keep in mind that the server once installed is completely text based, and has no GUI at all. You may also ckick on the "install" icon that you will see on your desktop.  A lot of users after installing the server version convert to the desktop version (has GUI), which is relatively easy to do via command  prompt. 

Welcome to the forums and good luck with the installation.
