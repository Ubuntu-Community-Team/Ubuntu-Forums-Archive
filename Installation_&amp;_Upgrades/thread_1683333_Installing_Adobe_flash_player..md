### Post by Kirboosy on 2011-02-07
Go [here]("http://www.adobe.com/products/flashplayer/")

Download the .apt package and allow it to install


OR

Open Synaptic Package Manager (System -->Admin --> Synaptic(

In the quick search box type "ubuntu-restricted-extras"

Mark for installation for the package that matches. Allow it to install any dependancies.

Hit Apply and let everything download/install

~Caboose
