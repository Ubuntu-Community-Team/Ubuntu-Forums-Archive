### Post by dbd on 2006-04-13
This isn't really an answer to your question, but it wil allow you to use opera.

There is a program called "klik" which lets you donwload programs as one single file, and then run the, just using that file. Just like for Macs. This is very useful if you are having problems installing a program, or simply don't want to install a program because you don't want the install making changes to your system.

Simply download and install klik from here:
[http://klik.atekon.de/](http://klik.atekon.de/)
Then you can download programs from there website, or by pressing Alt+F2 and typing klik:// followed by the name of the program,  so for opera you would use:
klik://opera
That'll download the single file version of opera, and put it on your desktop.
