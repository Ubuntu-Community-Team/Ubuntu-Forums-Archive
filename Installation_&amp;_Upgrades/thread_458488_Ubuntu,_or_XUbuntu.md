### Post by GrueTamer on 2007-05-29
With those specs, it's pretty much personal preference, since either should run (the processor MIGHT be a small problem, but I'm not sure).  I'd recommend installing Ubuntu, seeing if it works, well, and if it doesn't, you can install xubuntu in the same installation and can use either gnome (regular ubuntu) or xfce (xubuntu).  To install xubuntu, in the terminal, type

```
sudo apt-get install xubuntu-desktop
```

To boot into xfce, in gdm (the login screen), you need to change the session to xfce, or xubuntu, or whatever it's called.  This'll be under options in the default ubuntu gdm screen.

Or, you could be safe and just install xubuntu.  To install gnome on top of that if you want to try it, do

```
sudo apt-get install ubuntu-desktop
```

Xfce is basically a lower end version of gnome, with different menus (that's the simplest way I can explain it).  It can be pretty like gnome, it just uses less resources.  Since you can run either or, the choice is up to you as to which to install first.
