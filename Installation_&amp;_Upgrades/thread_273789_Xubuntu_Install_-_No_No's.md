### Post by Zerksis on 2006-10-08
This is posted in the hope that it will save someone time & effort in getting Xubuntu up and running.

These notes apply to the "*Alternate CD*" ***and*** when the *OEM install* option is used.

If you use the official Xubuntu 6.06.1 (Dapper Drake) .iso image;- 

*    xubuntu-6.06.1-alternate-i386.iso*      from

[http://cdimage.ubuntu.com/xubuntu/releases/6.06.1/release.1/](http://cdimage.ubuntu.com/xubuntu/releases/6.06.1/release.1/)

problems may be encountered on some machines where after installing you can only log-in to a command prompt, no Graphical user desktop.
  
This problem can be avoided if you select the first option listed.  Install in Text mode.

[COLOR=Red]*** Do Not Use OEM Mode***[/COLOR].

 
Should you have an install problem - first select the option *Check CD for Defects*
If the CD fails, then run an MD5 check on the .iso image.  This could save a long download.
Might just be a CD burn error. 

For Windows users you can get a nice easy MD5 check program from;-

[http://ww2.nero.com/nero6/eng/Nero_MD5_Verifier.html](http://ww2.nero.com/nero6/eng/Nero_MD5_Verifier.html)

Paste the MD5 for your iso image that you obtain from;-

[http://cdimage.ubuntu.com/xubuntu/releases/6.06.1/release.1/MD5SUMS](http://cdimage.ubuntu.com/xubuntu/releases/6.06.1/release.1/MD5SUMS)

browse to your download & check.

Other forum link    [http://www.ubuntuforums.org/showthread.php?t=272175](http://www.ubuntuforums.org/showthread.php?t=272175)

The OEM mode for Ubuntu (Gnome Desktop) has worked OK for me.

### Post by Herman on 2006-10-09
Hello,Zerksis, here's a little bit more info on OEM mode, in case it will be of any help to someone.
**Install in 'OEM' mode**. The (_O_riginal _E_quipment _M_anufacturer) installation is for computers that are being set up in a factory or a store with Ubuntu pre-installed.  During the OEM installation, you set an OEM password. You then log in with the username 'oem' and the password that you set. After you shut down, it will be set up to allow the new owner to set up their own new user account on boot-up.  For an Official OEM Installer Overview, [Click Here]("https://help.ubuntu.com/community/Ubuntu_OEM_Installer_Overview").
