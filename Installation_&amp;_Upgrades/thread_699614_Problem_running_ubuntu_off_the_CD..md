### Post by gpilkay on 2008-02-17
You may have a problem similar to my friend's desktop.  We had to flip through a few Live-CD's in order to get one that worked (which in his case was 7.04).  If this is a more modern computer I do recommend the 7.10 version as the first try, it was the only one that could boot to the Live-CD environment without trouble on my Dell.  I also use it in my Virtual Machine and my desktop, it seems to work fine.  Kobuntu is also good, as you've been told, and it may boot where others will not.  If this is a kernel problem, you can usually get better results with decent hardware by going a version up.

I carry about three different Live-CDs for system rescue for friends when I'm at school, does it show?

You didn't mention, did you do the media check before trying to boot for the first time?  You'd be surprised how often the CD has a defect that is unreadable till you do a real boot.
