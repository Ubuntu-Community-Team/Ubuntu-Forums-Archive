### Post by m4ng0 on 2005-02-06
If you have "Ubuntu install CD" but you can't boot it, you should read
install/README.sbm file (it's on the CD itself!).
It explains how you can build a boot floppy with Smart Boot Manager, so you can boot Ubuntu CD even with a BIOS which doesn't support direct boot from CD.
