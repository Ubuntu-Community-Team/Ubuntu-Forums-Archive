### Post by claracc on 2010-09-24
I think ubuntu remains in a hidden folder in windows and for some reason the uninstallation has not removed it.
This kind of folder will be named by .1000 or .trash1000 or something similar (I don't remember very well), and is in c:\. 
If certainly ubuntu has been uninstalled (it doesn't appears in programs installed in windows), I think you have to find the hidden folder where there is ubuntu and delete it. 
Perhaps you have to download some software as treesize in order to see the hidden ubuntu folder in windows.
