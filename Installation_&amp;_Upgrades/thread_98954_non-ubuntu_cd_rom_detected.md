### Post by bwog on 2005-12-05
That is the live CD for x86 processors like the celeron, so that is ok. 

I am not saying that this will solve the problem, but the first thing you do is to verify the iso. Did you do that?

The iso must be burned as an image. It is possible that this went wrong or that your CDplayer is having trouble with it.  What may help is to burn the CD slower. Did you try that? 

One more thing: if you used a rewritable CD for the image, try a normal CD instead.
