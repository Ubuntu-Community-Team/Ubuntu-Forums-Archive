### Post by Vladlenin5000 on 2015-07-21
Hi and welcome.

It has nothing to do with your internet connection so don't worry :p
What just happened is exactly what is supposed to happen when using an EoL release: Unsupported and servers' contents pertaining to that release where moved to archive.

Please install a supported release, either 14.04 LTS (Long Term Support, until 2019) or the current 15.04 (later you'll have to update to 15.10 and then to 16.04 LTS - release date April 2016 -).
