### Post by Tank Jr on 2011-11-14
Hi all,
Over the weekend I did a clean install of Ubuntu 11.10 32 bit on my desktop. I had my home on a separate partition, and so I only formatted the root partition. 
I did not want to use Unity, and so I moved to Gnome 3. I was pleasantly surprised that since I had Gnome 2 configured on my account, Gnome 3 picked up many of the settings out of the box. For example, I had the min/max/close buttons on Windows straight away; I was able to install compiz config settings manager, and pretty soon I had my rotating cube back.
I tried adding applets and launchers to the top panel, and found directly right clicking did not work on the panel. However, Alt+right click+left click gives me an option to add an application to the panel, and, if an app already exists at the location, I have an option to move it. I moved the clock from the center to the right.
So overall I am pleased with 11.10.
