### Post by javyer on 2005-10-14
Well CD-RW must be erased before burning a new iso (DVD-RW doesn't)... Gnome Cd burnner asks to erase the CD-RW when it's not empty, before burning an ISO file. 
But Gnome cd burnner seems to do a quick format, and sometimes, this format is not sufficient. 

Try to format the CD-RW with a burning tool like graveman or k3b. With them you can perform a complete CD-RW format (on 4X CD-RW this can take 30/40mn)

Try it ;)
