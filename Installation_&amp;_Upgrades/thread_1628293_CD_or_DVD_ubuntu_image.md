### Post by ajgreeny on 2010-11-22
I feel that packages are updated so quickly in the linux world that you might as well stick with a CD install image, rather than download a huge DVD image and then straight away update many of the packages meaning another huge download.

There is even some good reason, particularly if some time after the release of a ubuntu version, to install using the minimal instal CD and when installing choose the ubuntu-desktop meta-package to be added, which will I think give you a full ubuntu system, but with no further immediate updates following the install.

Personally I believe the CD image is the best compromise for ease of use, and for a smaller download image.
