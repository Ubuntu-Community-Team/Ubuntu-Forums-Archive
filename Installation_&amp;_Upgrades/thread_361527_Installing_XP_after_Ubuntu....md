### Post by tkjacobsen on 2007-02-14
Usually installing windows will remove grub from mbr. Read this [https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows](https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows) before you install windows and you should be fine. I haven't experienced any problems installing windows besides getting grub uninstalled.
