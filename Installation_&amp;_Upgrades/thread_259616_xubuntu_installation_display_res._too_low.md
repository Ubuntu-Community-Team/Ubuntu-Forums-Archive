I'm trying to install xubuntu 6.06.1 onto one of my older machines. (433 celeron, 160 ram, 4 gig HD)

When I try to run the install, I'm asked which language I'd like to use. I can select English from the list, but I'm not able to continue to the next step. The window is too tall and I'm unable to see the lower portion of the window. I tried resizing the window, but it doesn't help. Under Setting --> Display settings, the only options I have are 'default' and '320x240'.

Is there a way to adjust the resolution so that I can see all windows fully? The monitor I'm using is a Dell M780.

### Post by eclectic on 2006-09-17
I just went through the same thing with Ubuntu: maybe my solution will help.

When the first screen comes up (after loading the Live CD), you're given several options. Press F4 and change the screen resolution to at least 1024 by 768. Then change the primary menu to option #2 - Safe Graphics Mode - then start the CD. Your resolution will be better, so double-click on the 'Install' icon -- you should be fine from there.

Mine installed properly, leaving me in my monitor's optimum 1280 by 1024 native resolution.

Hope this works for you.
