### Post by Mr_JMM on 2012-11-21
Since you're not running an LTS version now I'd go for 12.10 and have the latest. But it depends on your system. If you have the bandwidth you may want to try 12.10 first and if you have problems that you feel are hardware realted go for 12.04. 12.04 is an LTS version and been around for 7 months now so many of the bugs will have been either resolved or solutions found. Especially since you have two drives now, testing is easier.

I've not installed onto an SSD but I personally haven't seen tht many issues with it.
