### Post by Herman on 2007-10-13
First, all you need is a swap area on your hard disk and your Live CD will work much faster, (like normal anyway).

You can make a swap area on your hard disk and pre-partition for your install as well if you want to, with a [GParted -- LiveCD]("http://gparted.sourceforge.net/"). Those are free and only a small download and are made to run on almost no RAM or swap area at all.

Or you can do as the previous posters advised and use the 'Alternate' CD to install with.

How well a computer can perform depends on not only the size of your RAM, but the speed of the RAM you have as well. Also L1 and L2 caching (which is another part of the memory a lot of people don't seem to be aware of), and the motherboard's cache controller all affect the performance of your computer. Adding more RAM will help a lot if you can afford it and your motherboard can support it though.
