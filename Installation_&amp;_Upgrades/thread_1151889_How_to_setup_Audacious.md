### Post by the.scarecrow on 2009-05-07
Perhaps you need to go into Synaptic Package Manager,
search for ubuntu-restricted-extras and mark the box for installation,
then click Apply.
this will install .mp3 support and lots more too.

note make sure you select the correct one as there are
ubuntu-restricted-extras
kbuntu-restricted-extras
and
xbuntu-restricted-extras
but you do say you are using Ubuntu.

Also I can thoroughly recommend Audacious as an audio player.

Hope this helps you out.
