### Post by Sef on 2006-05-07
The problems could be one of the following:

1) Bad CD

2) Bad Burn

3) Did you md5sum?

4) Incompatible hardware

Is there another computer you could try it on?  If not, get a live cd and see what happens.  If you don't have any problems, then something is wrong with the cd and not the hardware.
