### Post by coffeecat on 2010-11-10
As far as I can see, grub is set up just fine and there is a menu entry for Vista in the grub menu which should work. However, it is **seventeenth** (17th!) on the list and I guess it has disappeared off the bottom of the screen. What you need to do is when the grub menu appears on screen, use the &#8595; key to scroll down the list until you come to the Vista entry.

If I am right, what you need to do long-term is to uninstall about five old unused kernels from your system, after which the grub menu will be more manageable. If you want help with this, post back.
