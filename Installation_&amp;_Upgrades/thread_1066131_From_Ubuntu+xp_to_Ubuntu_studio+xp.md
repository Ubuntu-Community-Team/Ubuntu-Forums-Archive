### Post by cariboo on 2009-02-10
You don't have to reinstall, go to System-->Administration-->Synaptic Package Manager-->Edit-->Mark Package by Task and select the parts of Ubuntu studio you want to install from the menu.

Linux is Linux, no matter what distribution you are using.
