Hello. 
Yes, if you disable USB Legacy Support in the Bios you will be fine, but you will not able to use the mouse in the bios...

However ASRock have released some bios updates (after I noticed them the problem) that solved the problem. You can download it here: [http://www.asrock.com/MB/download.asp?Model=P67%20Pro3&o=BIOS](http://www.asrock.com/MB/download.asp?Model=P67%20Pro3&o=BIOS)
