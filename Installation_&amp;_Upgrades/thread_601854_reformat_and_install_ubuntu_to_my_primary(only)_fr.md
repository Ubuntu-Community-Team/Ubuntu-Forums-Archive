### Post by bluepowder7 on 2007-11-03
i always used the gnome partition editor to clean up and organize my new partitions, before running the install feature.

only problem i found is that SOMETIMES it complains about being unable to format or resize or setup certain partitions.  the easiest thing is probably to erase all the partitions, click apply. then set up a new partition or two, click apply. if you need to subdivide into more partitions, do that then click apply.  basically, once you click apply it'll try to do EVERYTHING you attempted - even if that wasn't what you finally decided on.  so, let's say you're trying to figure out where and how to make your partitions and what to format them to, and you make 4, detete 3, add 1, remove it, remove another 2, add 3, and so on - once you click apply it'll do it all step by step, like a macro.  irritating as hell.  draw out your partitions on paper, and use mouse clicks to only orgainze your final arrangement.
