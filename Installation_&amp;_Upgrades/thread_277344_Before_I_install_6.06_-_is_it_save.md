### Post by meng on 2006-10-14
I'm not certain that anyone can guarantee 100% safety for this, but most people don't experience problems, and I suspect that most problems are due to user error. That still leaves some installations that go pear-shaped regardless of due caution. I have not experienced any such problems personally.

It is highly recommended to defrag the Windows installation before installing Ubuntu. Multiple times even! If you have had this installation for 2 years, you may have some serious fragmentation there.

In summary, no one can give you a 100% guarantee with either the Desktop or the Alternate CD installation. If you really feel you cannot afford to hose your system, then don't install Ubuntu on that computer.
