Unfortunately you need DEBs, not RPMs. You will have to either download them

== OR == (recommended)

Add the OOo PPA as per [these instructions]("http://ubuntu-snippets.blogspot.com/2008/10/ppa-for-openofficeorg-30.html"). You can then install openoffice by using a simple apt-get install command. Adding the PPAs will also deliver updates to your system using the familiar update-manager. And there is an alternate installation method mentioned in the linked page as well.
