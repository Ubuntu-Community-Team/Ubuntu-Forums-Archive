### Post by Liviu-Theodor on 2008-11-26
On my system, the NVIDIA proprietary driver 177 does not work in 3D mode, does not work compiz, emerald, etc. with it. For other with the same problem: in [FONT="Courier New"]Application->Add/Remove...[/FONT], uninstall it, and install instead the driver 173.
