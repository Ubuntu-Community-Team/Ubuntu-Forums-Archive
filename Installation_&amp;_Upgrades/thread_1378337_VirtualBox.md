### Post by snowpine on 2010-01-11
Windows as a guest OS in Virtualbox won't work well for games (they can't "see" your graphics card). I therefore recommend a dual boot for gamers.

Lots of Virtualbox information here:
[http://ubuntuforums.org/forumdisplay.php?f=308](http://ubuntuforums.org/forumdisplay.php?f=308)
[http://www.virtualbox.org/manual/UserManual.html](http://www.virtualbox.org/manual/UserManual.html)

### Post by howefield on 2010-01-11
Are you thinking that installing virtualbox and creating an xp virtual machine will solve all your gaming needs ?

It won't.

Although there are many great games that run through linux / wine / playonlinux, vm, ect, it is not likely yet to give you what you want.

You'll probably still need your "real" windows install for a bit yet. 

Go to the virtualbox website and view some of the tutorial videos.

[http://www.virtualbox.org/wiki/VirtualBoxTV](http://www.virtualbox.org/wiki/VirtualBoxTV)
