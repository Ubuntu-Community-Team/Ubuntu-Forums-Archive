### Post by potatan on 2007-12-19
Hi,

I'm a relative noob, but I installed Ubuntu on my HP Pavilion laptop a couple of weeks ago. (it's a DV5000 series)

I already had XP pro loaded on there with two partitions, C: for Windows and D: for data

What I did was to transfer the data to my C: drive then use the whole of D for installing Ubuntu.

The best way that I found was to boot from the Live CD, but not do the install from there.  I believe that if you just choose the "Run Ubuntu from CD" (or something - second option I think - there may be a "Safe Graphics Mode" or something too which could help), you will get an Ubuntu system up and running.  Then you will find an icon on the desktop to perform the actual install.  Worked for me every time.

GRUB is the boot loader - the program that lets you choose which operating system to load once your installation is complete, in my case Windows XP or Ubuntu 7.10

Feisty Fawn/Gutsy Gibbon etc. are different versions of Ubuntu.  The current stable version is Gutsy Gibbon, also known as 7.10 (as it was released in October 2007 - 7.10).


Just a bit of advice - after using Windows by default for 20+ years, and having downloaded and tinkered with various Linux distributions in all that time, Ubuntu is finally the one that has made me consider making a total switchover for all the PCs in my home.  Good luck!

Cheers
potatan
