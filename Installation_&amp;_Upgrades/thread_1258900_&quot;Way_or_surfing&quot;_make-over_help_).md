### Post by starcraft.man on 2009-09-05
How did you install flash before? I'd recommend ya uninstall it by same means before proceeding, though sounds like it didn't install properly, probably wouldn't be a bother.

The preferred means is to use the installer from the repositories. See install guide in sig for a detailed explanation. To easily install flash, open a terminal (applications > accessories > terminal) and type in and return:

```
sudo apt-get install flashplugin-installer
```

Also,  you might want to install the ubuntu-restricted-extras package. For a better understanding, and a GUI means, please see sig like said.
