### Post by gpilkay on 2008-04-10
I doubt you can just install it from Windows, W98 hasn't been supported for a long time and most modern programs are quite a bit beyond its ability, plus the partitioning is not going to work right.  You'd have to boot from the CD to install.  Keep in mind that Server edition has no GUI.

And with the anger, it will quickly get you NO help and probably reported.  We are not paid in any way.  We are volunteers that do what we can to help those as we are able.

Good starting info would be:

Processor
Memory
Hard drive Space
Ethernet or not?  Some w98 boxes have no ethernet port.

If Ubuntu can not handle your system, you also have other options.  Damn Small Linux (that's its name, no joke, it's a silly title but a great product) works on very low resources, and Puppy Linux actually is billed as a great way to restore an old Windows 98 machine.
