### Post by vanadium on 2011-10-09
You cannot install Gnome 3 on Ubuntu 11.04 using apt-get, which is still based on gnome 2. You must have used an unsupported installation method, and this entrains a risk. In your case, this would mean a reinstall.

I suggest you locate the forthcoming Ubuntu 11.10, which is near release quality. 11.10 is based on gnome 3, and you can install gnome 3 shell using synaptic.
