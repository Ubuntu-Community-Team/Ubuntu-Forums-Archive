### Post by mmmmna on 2010-01-28
This is most likely decided by the settings in the BIOS. Review your manual for how to enter BIOS setting mode BEFORE the system boots. Usually, you press a key (F2, delete) right after the very FIRST screen shows on the computer (a few seconds after you turn it on). Then the BIOS setting screens (be careful) need to be researched until you see a boot order declaration. Adjust these.

Also possible is you pressing a different key (at about the same time as BIOS setup) to bring up a special dialog window where you can select what device is to provide the boot system. Again, read the manual for the system.
