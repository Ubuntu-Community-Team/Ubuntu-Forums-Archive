### Post by Beastlicker on 2009-12-27
Open up terminal and cd over to the location where the .exe is. Once there, run 


wine "insert .exe file here" (without the quotes)

After that, let the installer load up.

After it installs, go to the programs menu, find the wine section and run the program from there.
