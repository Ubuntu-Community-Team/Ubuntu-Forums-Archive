### Post by joker99 on 2011-12-31
I noticed that whenever I complete installation options wizard 
before the installer completes copying files, it hangs with
text "Ready when you are". 
Such parallel copying/filling up forms was introduced recently (a few releases back). Before it, I had no problems with installer. 

The workaround is to wait until installer copies all files and
then start filling up installation options.

### Post by digismack on 2012-01-11
This is a huge bug. This solution worked for me after Googling for hours and even trying multiple disks / variations of Ubuntu/Lubuntu/Xubuntu. The bug is even present on the 11.10 installer. Simply leaving the install screen on the time zone selection screen until it copies all files and says "ready when you are" is the only thing that worked.
