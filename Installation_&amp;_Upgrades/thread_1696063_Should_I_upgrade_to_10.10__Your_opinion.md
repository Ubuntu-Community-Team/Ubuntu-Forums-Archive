A word of caution:

[http://ubuntuforums.org/showthread.php?t=1622388]("http://ubuntuforums.org/showthread.php?t=1622388")

Understanding the new live installer/ubiquity: Release 10.10 (Maverick)

The new installer has some problems, it has changed quite a bit in this release compared to 10.04.  It is now being recommended by several members that the **"Install alongside existing operating system" option not be used.**  The danger is that this option will overwrite the *other* operating system.  It has been happening.  Read the thread listed at the beginning of this post.
