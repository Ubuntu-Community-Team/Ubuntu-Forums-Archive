### Post by byte69 on 2007-03-06
I do not think that is possible.   Because the ISO is a image of a bootable CD/DVD so once that is burned you have only that one image.   I don't believe you can burn say two or more cd ISO images one would over right the other if the software would even let you do it.   So No you can not burn multiple ISO images to a DVD.  But you could burn the iso files to DVD but they will not be bootable till you burn the actual ISO image to cd/dvd.
