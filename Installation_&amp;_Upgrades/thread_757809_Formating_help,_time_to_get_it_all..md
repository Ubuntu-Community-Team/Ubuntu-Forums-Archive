### Post by mikewhatever on 2008-04-17
Yes, since Ubuntu and XP reside on separate partitions, you can format the one for XP using Gparted live cd. Just backup the data you need from that partition first. If you are not sure which one to format, post the output of <sudo fdisk -l>.
[http://gparted.sourceforge.net/livecd.php](http://gparted.sourceforge.net/livecd.php)
