### Post by SeijiSensei on 2016-06-12
1) Install the "[LAMP]("https://help.ubuntu.com/community/ApacheMySQLPHP")" package - Apache, MySQL, and PHP

2) Download and unzip the [most recent version]("https://wordpress.org/download/") of WordPress into a directory below /var/www/html.

3) Navigate to [http://your.server.ip.addr/path/to/wordpress](http://your.server.ip.addr/path/to/wordpress)

For all of this you'll need root privileges using sudo.
