### Post by hhh on 2010-10-05
I run Ubuntu 10.04 with Compiz effects and RGBA transparency on a 30G partition with 1G ram and a 1.7MHz processor. It's a desktop with a motherboard Nvidia GeForce 6150. That system should be fine, but the easiest way to check is to burn a Live CD or transfer the Ubuntu ISO to a 1G or greater USB drive via Unetbootin and try it out live first. You might not be able to check out Compiz that way as you may need restricted drivers for that but you'll see if basic networking and such work.
