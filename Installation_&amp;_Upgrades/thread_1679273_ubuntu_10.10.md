### Post by ubername on 2011-01-31
> **viper1377 said:**
> is this the latest vlc for ubuntu 10.10 the codin for it plz ?

As I say, you are much better off using the Ubuntu software Centre.

Click on 'Applications' (Top Left of the screen) At the bottom of the menu you will see an option Called Ubuntu Software Centre. Click on that.

When the Ubuntu Software Centre window opens, at the top right of the window is a box to enter the name (or part of name) of what you are looking for. Type 'VLC' in that and it will show you what is available. Select 'VLC' and click 'Install'
