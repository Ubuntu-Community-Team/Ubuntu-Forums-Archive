### Post by Juffo-Wup on 2007-10-19
Well, I don't know if this will help you or not, but at least on my system I was left without window decorations. After I installed the package compiz-gnome (as I had on my Feisty install, before I upgraded) I regained them.

As for the workspaces, it seems to me that these require you to enable the desktop cube. You can install compizconfig-settings-manager if you have not done so already to enable it  (and any installed plugins) at will.

Since Beryl (as far as I know) was merged into Compiz-Fusion, perhaps you could find the old Beryl plugins there.

Hope this helps.
