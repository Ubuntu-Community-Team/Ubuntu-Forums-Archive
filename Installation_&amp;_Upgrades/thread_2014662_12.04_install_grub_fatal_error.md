### Post by darkod on 2012-07-02
/dev/sda is a disk and the MBR of that disk. /dev/sda1 is partition #1 on that disk.

Grub2 by default doesn't install on a partition (like sda1), it should be installed onto a MBR of a disk.

If you prefer the grub2 from 12.04 to run the show and you boot from sda, boot the 12.04 once, open terminal and execute:
```
sudo grub-install /dev/sda
```

That will install the grub2 from 12.04 onto the MBR of /dev/sda.
