### Post by gdesilva on 2020-08-03
Hi, I am running Ubuntu Studio 20.04 and can confirm that audio settings icon was on the top panel when I did the installation. If you do not have it for whatever reason, you can add Pulse Audio Volume Control app to the panel or run pavucontrol command from a terminal. If you click on the icon and select Audio Mixer option this will allow you to select the HDMI port for output.
