It's not difficult to install the latest LibreOffice via a .deb file download.  The only negative is that updates will not be automatically installed.  i've had pretty good luck with 3.5.x, the only negative I can recall is trying to open a password protected .odt file created in 3.5 in 3.3 or 3.4.  It didn't want to open.  Opening a file created in 3.3 or 3.4 in 3.5 hasn't been a problem.
