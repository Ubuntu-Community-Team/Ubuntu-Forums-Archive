### Post by christhemonkey on 2006-01-09
If anyone is wondering, i downloaded a program called sbootmgr from here;
[http://sunsite.lanet.lv/ftp/mirror/Slackware/rootdisks/]("http://sunsite.lanet.lv/ftp/mirror/Slackware/rootdisks/")
burnt it onto  a floppy using rawrite.exe 
(which is on the same site)
This then allowed me to select boot from a CD, which then installed ubuntu from the CD flawlesly.
