### Post by jpkotta on 2008-12-06
It's not a GUI program, nor does it have to be.  It is a typesetter for music.  You write a markup file (like HTML or LaTeX) and run lilypond on it, and it produces a .pdf.  If you were looking for something like OO.o Writer for sheet music, you will be disappointed.  Lilypond is for people who make sheet music often, and want it to look right.

Maybe there is some sort of front end for it, look around on Google.
