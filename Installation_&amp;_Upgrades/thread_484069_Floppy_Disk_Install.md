### Post by Milky The Brown Cow on 2007-06-25
Hey! a google of Ubuntu Floppy gave a result from some site that apparently gives you one free solution and then asks you to become a member, but here is the solution they gave.

"If it's the case where your PC does not support booting the ubuntu install CD then you can fix that by
usiing a boot floppy called smartbootmanager. It's free and you can get it at [http://btmgr.webframe.org/](http://btmgr.webframe.org/)

You load your install CD then boot up using the smartbootmanager floppy.  You will get a screen allowing you to select boot from CD."   ~  [http://www.experts-exchange.com/Operating_Systems/Linux/Linux_Setup/Q_21476200.html](http://www.experts-exchange.com/Operating_Systems/Linux/Linux_Setup/Q_21476200.html)

Hope this helps!
