### Post by sanderd17 on 2010-01-07
I suppose this isn't in the repostories because it's closed source and because it's new. You can go to the opera download website and just install the .deb package.

link to 10.10 beta 1:
[http://www.opera.com/browser/download/?os=linux-i386&ver=10.10b1&local=y]("http://www.opera.com/browser/download/?os=linux-i386&ver=10.10b1&local=y")

If you use an amd64 distribution you'll have to change the version.

once you downloaded it: double click and it should install. You won't get updates for this since it's not in the repostories.

Ow, wait a minute, it's a pre alpha version and it's not availiable for linux yet. You'll not be able to install the 10.5 version.
