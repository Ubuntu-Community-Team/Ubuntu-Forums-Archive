### Post by arubislander on 2012-02-01
Without knowing what program it is there's not really a common one-size-fits-all way of 'installing' a tar.gz archive.

Some archives just have to be unpacked into a folder under your home folder, others require an installer to be run after unpacking. If an installer needs to be run, some then live completely in your home folder, others scatter files all over the place...

### Post by drmrgd on 2012-02-01
I think an easy way to go about it is to open the .tar.gz file in Nautilus to unpack it where you'd like.  Then, once it's unpacked, have a look for a README or INSTALL text file, which will usually have the installation instructions.  You may have to do it by command line, and you may find that you need to manually install some packages on which the main one you want is dependent upon.
