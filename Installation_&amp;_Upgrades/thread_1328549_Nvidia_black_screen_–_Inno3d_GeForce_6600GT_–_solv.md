### Post by vohary on 2009-11-16
[COLOR=Black]I use Linux  Mandriva 2007. The graph[/COLOR][COLOR=Black]ics card worked fine.
I installed a new Ubuntu 9.10 and my graph[/COLOR][COLOR=Black]ics card [/COLOR][COLOR=Black]didn’t work. When [/COLOR][COLOR=Black]I rebooted after installation I [/COLOR][COLOR=Black]saw a black screen [/COLOR][COLOR=Black]only. I found a lot of solution[/COLOR][COLOR=Black]s and discussion[/COLOR][COLOR=Black]s [/COLOR][COLOR=Black]about this problem [/COLOR][COLOR=Black]on internet. My solution is [/COLOR][COLOR=Black]to replace the bios in graphics card (05.43.02.80.00 Inno3d) [/COLOR][COLOR=Black]with bios (05.43.02.46.00 Inno3D) - download from [www.mvktech.net]("http://www.mvktech.net"). [/COLOR][COLOR=Black]I did downgrade the bios! The card works fine in Madriva 2007 (driver 96) and Ubuntu 9.10 (driver 185).  I [/COLOR][COLOR=Black]tried to use 5.43.02.61.00 from Inno3D and it didn’t work. [/COLOR][COLOR=Black]I [/COLOR][COLOR=Black]tried to use 05.43.02.46.68 Leadtek- it worked but not [/COLOR][COLOR=Black]ideally.
I have read [/COLOR][COLOR=Black] this type (6600GT AGP) has problem with FastWrite – to use FastWrite disable in BIOS. [/COLOR][COLOR=Black]

PS: The [/COLOR][COLOR=Black]version of VideoBIOS is written in “/var/log/Xorg.0.log”

Good luck
