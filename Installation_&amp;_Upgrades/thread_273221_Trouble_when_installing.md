### Post by public_void on 2006-10-07
I'd check the CD your using is okay. On the Ubuntu boot menu select "Check CD for defects", this will check the integity of the disk. If any of the check fail reburn the disk. If you haven't checked the md5 them do so. See [http://www.etree.org/md5com.html](http://www.etree.org/md5com.html) to do it on Windows. If the check fails then you will have to download the ISO again. I would also burn the CD at a low speed to reduce the likelyhood of defects.
