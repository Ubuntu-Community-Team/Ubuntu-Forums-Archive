### Post by bruenig on 2007-01-02
> **bfr said:**
> I just finished booting Ubuntu.  It looks good so far, with the desktop and everything.  Do I need to run the program called "Install" on my desktop?   Or would I have to go through the whole setup process every time I turn on my computer if I don't run it (I'm guessing this is the case)?

The install button installs ubuntu to the hard drive. If you see the install button that means you are running off your cd drive and RAM with the live cd. So yes you do need to click install if you want it to be on your hard drive and run much faster. Also if you don't install it, all the applications and stuff you install will need to be reinstalled every time you boot.
