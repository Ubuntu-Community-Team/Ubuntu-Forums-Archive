### Post by darkod on 2009-12-05
Sorry you are having all these problems but this is very strange. You should try running the Try Ubuntu option first from the cd and that will show you if thee is any hardware incompatibility or something. If it doesn't want to boot, try burning another cd at low speed (not more than 8x) or even downloading another image, just to make sure the first one is not corrupted. You can also try putting on usb stick instead of cd and boot from that.
After the Try Ubuntu option is working fine, select the Install Ubuntu and it should be fine.
You also had previous problems even with accepting Enter on the keyboard, it just doesn't sound right.
Missing module might be a corrupted downloaded image. I've never seen that message myself.
