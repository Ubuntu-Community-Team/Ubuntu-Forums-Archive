### Post by toujours on 2005-10-30
If you want to use strictly Windows software, you need to use either Windows or Wine, a Windows emulator.

If you simply want MSN compatibility, try GAIM.

To see the contents of your existing windows partition, you need to "mount" it. I don't know if this can be done with the Live CD, but many people use this to see Windows partitions fron Linux. It's described in [www.ubuntuguide.org](www.ubuntuguide.org) along with a lot of other useful stuff.
