### Post by supererki on 2009-04-28
check man mount for mounting details.
but basically it should be like 
sudo mount [options] /what /where

so you should try 
sudo mount /dev/cdrom /mount/point or something

maybe mount -t iso9660 to be more specific
and make sure there is something to be mounted 
like disk in a cd drive.
all the best.
