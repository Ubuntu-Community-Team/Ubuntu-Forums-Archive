### Post by C.S.Cameron on 2010-06-14
Download the iso from here:

[http://www.ubuntu.com/desktop/get-ubuntu/download](http://www.ubuntu.com/desktop/get-ubuntu/download)

You can make a bootable USB in Windows using UNetbootin from here.

[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

Or you can burn the iso to CD and use the supplied usb-creator to make a bootable persistent USB. Or you can use it to install the full version to USB, (4GB min stick).
