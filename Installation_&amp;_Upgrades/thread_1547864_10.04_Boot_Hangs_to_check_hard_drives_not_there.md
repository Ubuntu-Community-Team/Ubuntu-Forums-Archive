### Post by jws957 on 2010-08-07
I just upgraded to 10.04 and it went very smoothly. Only problem is that this version now tries to check a couple of hard drives that are external and not attached to the system. They were set up some time ago and the boot will not proceed unless I manually enter "S" to skip. I realize there is probably a simple solution to this and would appreciate some advice. I have removed folders for these disks that were in /media/... but that didn't solve the problem.

---

### Post by ajgreeny on 2010-08-07
Let's see the output of ```
cat /etc/fstab
``` It is possible that those external disks are still listed in that file and the system is trying to check them.  A manual edit to either delete or comment out those lines may be all that is needed.

---

### Post by jws957 on 2010-08-07
That did it. Many thanks!
