### Post by Partyboi2 on 2009-02-03
[[COLOR=Blue]This[/COLOR]]("https://help.ubuntu.com/community/BurningIsoHowto") will show how to check the md5sum an burn the iso to disk. Make sure you burn the ubuntu iso file at a low speed like x4 or less. 
Once you have successfully burned the iso to a disk you will need to reboot with the disk you burned in the cdrom, making sure you have "boot from cdrom" set to first in your bios. All going well you will be greeted with a Ubuntu main menu.
