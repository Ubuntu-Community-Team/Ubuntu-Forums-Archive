### Post by rippon on 2007-04-01
I have been using Ubuntu since the first release. The progress the project has made over the years is amazing. One of the best things about Ubuntu is the ease of installation. However, I continually face the same annoying problem: there is no easy GRUB configuration during the installation setup.

Going through the normal language, keyboard, and manual partition setup is fine. But generally, there is either no or very little options for GRUB configuration. When Ubuntu tries to make it 'easy' for me, it always messes things up. For example, with 6.10, the only GRUB install option was my PATA drive, even though I was installing the distribution on my SATA drive--which Ubuntu recognized. I had to unplug all the other hard disks just so it would write GRUB onto the correct drive.

Again, today, I installed the 7.04 BETA. Yet again, the installation lacks adequate GRUB options. GRUB needs a full-featured setup section during the installation. It is too important to not have these options.

Sure, there are many other ways to accomplish this. But, I would really enjoy a more complete GRUB section in future releases. Not only for my benefit, but to strengthen Ubuntu as a distribution as well.

Dan
