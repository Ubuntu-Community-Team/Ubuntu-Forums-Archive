### Post by Rocket2DMn on 2007-07-20
When you download and burn a live cd, you need to follow some steps to be more confident the cd will work correctly.  
First, when you download the .iso image, you need to check the md5sum.
[HowTo]("https://help.ubuntu.com/community/HowToMD5SUM")
[Hashes]("https://help.ubuntu.com/community/UbuntuHashes") to check against
When that is all good, you should burn at a slow speed, like 4x, and use quality media if you can, not the cheap stuff.

Once you have this cd, you can try your install.  If it fails, which it sometimes does for people, you can use the alternate cd - it doesn't give you the live cd capacity but you can install using a text based installer.  
[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) - and check the box for Alternate CD at the bottom.
