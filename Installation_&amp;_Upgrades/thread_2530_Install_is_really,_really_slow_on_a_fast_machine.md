I solved my problem. After another setting change and another install attempt, and many, many hours I restarted and picked up the installation after a cold restart. I was prompted to use "aptitude", a recovery tool (kinda), I finished the install within 5 mins.

I ran a bunch of hdd diagnostic tools and memory diagnostic tools and everything seemed okay. Just wanted to post this because I'm sure that I will not be the only one with this problem. 

So far, minus one hitch, this system is really nice.
