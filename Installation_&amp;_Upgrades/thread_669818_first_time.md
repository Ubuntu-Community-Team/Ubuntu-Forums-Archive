### Post by Nebutron on 2008-01-17
> **hojiro said:**
> well i downloaded the last desktop version and burned it into the cd...

Okay,  if you can run the cd and pull up the ubuntu desktop without installing to the hard drive, you have the live CD.  Sometimes people who have trouble installing from the live CD have better luck using the "alternate installation CD."  To get this, you have to check the little box in the download window that says "alternate installation CD."  

The Alternate install CD will not give you the option of running the operating system from CD.  It is for install only.
