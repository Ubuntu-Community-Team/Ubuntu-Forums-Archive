### Post by drewdle on 2011-07-20
I don't expect you'd see much difference. My P4 2.8Ghz machine doesn't run any faster than my 1.86Ghz Core 2 Duo machine when it comes to just puttering around, although that is comparing two processors that are six years apart. I think you'd see a difference is you regularly do processor intensive work, so encoding audio and video, compressing images, working with huge compressed files, etc. 

If by "faster" you mean how snappy the interface is, I think that's more dependent on your system memory and your graphics card, especially seeing as though Unity uses graphics acceleration. 

Hope my two cents help. :D
