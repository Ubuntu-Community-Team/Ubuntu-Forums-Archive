### Post by More or Less on 2012-04-25
When I upgraded using the "gnome" (the ubuntu version), I didn't have any problems with losing software.  When I changed to the KDE Kubuntu look, I made a folder with shortcuts which launched the programs, but I couldn't see them listed.  For some reason Gimp's icon doesn't appear even though it was removed and reinstalled as a test.
 
So, you won't lose software either in Kubuntu, but it might look a little messy until you fix the shortcuts. For me, I use it to my advantage.  It gives me a chance to clean up the computer files I need and I can delete what I don't need.
 
And if you aren't interested in any of that, at installation it asks you if you want to keep the software or write over the partition.
