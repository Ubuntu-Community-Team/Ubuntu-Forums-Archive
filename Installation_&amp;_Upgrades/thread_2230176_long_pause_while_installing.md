### Post by grahammechanical on 2014-06-17
It is too early in the install process for Ubuntu to be installing. Have you set a location? Have you entered your name and a username and password. The install does not really begin until we enter that information.

What to do? Are you sure that you have clicked Install? The installer may be waiting for user interaction. That is the best that I can offer. I have never had that situation and I have done many installs. At least I cannot remember things getting stuck at that point. Try clicking Back.

If the partitioner is resizing a Windows partition, then that might take some time.
