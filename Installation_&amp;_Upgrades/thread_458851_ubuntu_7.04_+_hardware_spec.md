### Post by Kobalt on 2007-05-30
The LiveCD sometimes needs boot options in order to work. You can try this way : at the LiveCD boot screen press F6 and add at the end of the line, after a space, the following : > noapic nolapic
Then press Enter to boot.

If it still doesn't work then you should try the Alternate CD. If you need help on how to install Ubuntu with the Alternate CD see [this]("http://users.bigpond.net.au/hermanzone/").
