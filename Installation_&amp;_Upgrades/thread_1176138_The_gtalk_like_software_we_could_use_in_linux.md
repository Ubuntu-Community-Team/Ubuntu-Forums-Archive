Hi you Guys, thanks for Xiongqin, Lerosua, Junyi, Ziyan and Me(Yeqin Su). We could use gtalk like software in Linux.

U could communicate with your friends through this one with text and Voice Chat.

U could download this source code from this website:[HTML]http://code.google.com/p/talkmm/[/HTML]

Or, download the deb from here. At first, something you should know, download 3 package from here, libjingle_0.4.1-1_i386.part1.zip libjingle_0.4.1-1_i386.part2.zip talkmm_0.2-1_i386.deb, you must unzip the libjingle packages and rename them to *.part1.rar
 and *.part2.rar. Finally, unrar these package, you could get the libjingle_0.4.1-1_i386.deb.

U must install libjingle first.
