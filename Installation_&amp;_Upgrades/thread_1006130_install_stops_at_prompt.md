Groknet, it sounds like there's a problem with hardware drivers. You say it installed nicely onto an XP machine and not onto several Vista machines. What are the specs/brand/model of the computer you are trying to get Ubuntu running on?

Alternatively, you could try a different distribution e.g. SUSE, Mepis or PCLinuxOS.
