### Post by black veils on 2012-09-14
all i can say here is:

if you want recovery mode for that system, you access it from that system, not your new install. to access it you hold Shift when the computer starts booting.

i assume you used the new install to collect your files and configurations data from the other system? if not, you may as well do that, though it could have been done from a live environment. obviously i mean you should make backups of those files to a usb flash drive/online storage like dropbox/dvds.

due to the nature of partial upgrades, i would suggest you simply start from the beginning (fresh install), of course you can still try using some of your configurations data, for applications like a web browser and messenger.

and in future if you see an option for partial upgrade, ignore it, wait for 24 hours or so and see if that option appears.
