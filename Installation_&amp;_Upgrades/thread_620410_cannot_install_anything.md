### Post by rsambuca on 2007-11-22
It sounds like your software repositories are messed up for some reason.  To fix this, go to your top menu bar and select "System -> Administration -> Software Sources".  On the Ubuntu Software tab, make sure that all of the boxes are checked, and then press the "Download from" button.  Select "Other".  On the window that opens, press the "Select Best Server" button.  It will then test all of the repository mirror sites and pick the fastest one to your location.

You should then be able load your programs.
