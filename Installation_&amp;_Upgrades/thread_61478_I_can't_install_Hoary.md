### Post by macgyver2 on 2005-08-31
[QUOTE=okatido]Hi,
I am from Turkey. I have downloaded Hoary ISO file. But i don't know what i am going to do next. I prepared a CD with this ISO file but my system doesn't boot it. Can you please help me?[/QUOTE]
My first suggestion would be to go into your bios and check the order of the devices your computer looks at to find a bootable system.

In case you don't know how to get into the bios...you have to hit a certain key right when the computer is starting up, before it begins loading the operating system. The key varies between computers...I've seen F2, F8, F10, F12, Del. It's quite possible your computer tells you which key to hit to enter the bios setup (unfortunately some don't). Once you get into the bios, find the right option--usually named "boot order" or something like that--and make sure your CD drive is higher than your hard drive. Exit the bios (make sure you save the changes!) and try the Hoary CD again.

If that doesn't work we'll look into other possibilities.
