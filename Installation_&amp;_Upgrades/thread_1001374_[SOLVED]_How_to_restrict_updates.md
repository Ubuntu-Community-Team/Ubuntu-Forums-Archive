### Post by 2hot6ft2 on 2008-12-03
Sure it can. System>Administration>Synaptic Package Manager then Search for evolution then highlight it and click on "Packages" and mark the "Lock Version" option you may want to mark all of the ones for evolution or you can just remove it once and for all, your choice.

But you'll lose other nice features such as glipper if you remove all the evolution packages so I would suggest just locking the version then it wont update it.
