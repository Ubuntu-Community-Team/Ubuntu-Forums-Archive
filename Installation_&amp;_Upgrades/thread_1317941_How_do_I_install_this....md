The driver you have linked to would need to be compiled (which can be a messy and painful process).

What you can do instead is use ndisgtk to install the windows driver (you select the .inf file). The link to the Windows drivers is here: [http://ftp.dlink.ru/pub/Wireless/DWA-110/Drivers/20080415-DWA-110-S0025-WW.zip](http://ftp.dlink.ru/pub/Wireless/DWA-110/Drivers/20080415-DWA-110-S0025-WW.zip)
