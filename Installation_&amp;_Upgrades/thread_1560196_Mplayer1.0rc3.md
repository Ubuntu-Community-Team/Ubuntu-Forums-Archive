### Post by andrew.46 on 2010-08-24
Hi David,

If you are using Lucid Lynx you will be pleased to know that MPlayer 1.0 rc3 is available in the Ubuntu Repository and thus you do not need to download files directly from the MPlayer website. One way to install it is to search through Synaptic:

System --> Administration --> Synaptic Package Manager

and type MPlayer into the 'Quick Search' box and the install the file it finds. I am not sure what the 'Music Player' is either :). 
All the best,

Andrew
