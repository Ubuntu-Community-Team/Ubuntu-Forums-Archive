### Post by djbushido on 2008-11-25
on wine apps, i read that the installer works, but not office. I recommend installing openoffice. ```
sudo apt-get install openoffice
```
OpenOffice has support for microsoft formats.
Also, you can install openoffice 3, which has support for the new *.docx, etc. from this [http://news.softpedia.com/news/How-To-Install-OpenOffice-org-3-0-in-Ubuntu-8-10-96449.shtml]("http://news.softpedia.com/news/How-To-Install-OpenOffice-org-3-0-in-Ubuntu-8-10-96449.shtml")
Good luck!
