### Post by jerrylamos on 2008-11-01
What hardware are you running?  A bit hard for us to figure out without knowing where you are coming from.

Now I got a black screen on IBM Thinkpad R31 as do others using certain Intel graphics hardware, see Launchpad Bug #259385.

A quick thing to try is to do Applications Accessories Terminal
then enter
metacity --replace &

That only works if you can get the Gnome desktop up at all.  It's also only good until you reboot.

Another thing is if you are running at all is System Preferences Appearance Visual Effects then select None

Now I never switch users so I don't know what that does.  I default boot without a login since I'm the only one that uses this computer.
