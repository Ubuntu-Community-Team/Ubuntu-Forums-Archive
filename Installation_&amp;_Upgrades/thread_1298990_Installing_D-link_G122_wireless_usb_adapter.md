Linux is NOT Windows, meaning, you can't run MS Windows programs in Linux to install drivers.

You would do better doing a search in the Networking subforum using the model number of your D-link card.  There's a possibility that someone has already figured out how to make it work.

If not, do a search in that same subforum for NDISWrapper -- an alterative method that sometimes work using MS windows drivers.
