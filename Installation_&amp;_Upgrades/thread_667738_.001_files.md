ok i have about 13 or so part files all reading .001 .002 as the file name. how do i join all these back together to make on file?

---

### Post by jrib on 2008-01-14
```
cat foo.001 ... foo.013 > foo
```
replacing "..." with the missing files in between.  Order is important.  Alternatively, you can let the shell do the work for you:
```
cat foo.* > foo 
```
