So I was upgrading from 9.04 to 9.10 when my laptop overheated and shutdown near the end of the upgrade process. Now I can not boot into ubuntu. It starts to boot 9.04 but then fails. Is there someway that I could possibly resume the update from where it left off through the command prompt? Or does anyone else have any suggestions?

By the way I fixed the over heating problem with the laptop. The heatsink at the end of the heatpipe was clogged with dust so no air could flow. So I removed the dust and added some arctic silver to all of the chips that the heatpipe covers and now all is good (except ubuntu is still broken).
