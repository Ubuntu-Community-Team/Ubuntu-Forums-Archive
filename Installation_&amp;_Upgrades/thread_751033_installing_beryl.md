### Post by Partyboi2 on 2008-04-10
If you are using gusty you can go to System>Preferences>Appearance and on the visual effects tab check extra. This will give you visual effects. (You may need to install the restricted drivers for your video card) If you are wanting to config compiz fusion install  compizconfig-settings-manager package from synaptic or in a terminal
```
sudo apt-get install compizconfig-settings-manager
``` 
[here]("http://forlong.blogage.de/article/2007/8/29/How-to-set-up-Compiz-Fusion") is a good tutorial on compiz fusion
