### Post by pseudonym on 2006-09-27
Ubuntu, Xubuntu, Kubuntu - at base they are the same distro, they just use different desktop environments. Each flavour comes with different applications, but there is nothing stopping you from running any application 'normally' used by one flavour on another flavour. But it can be easier from a configuration point of view of you stick to those apps which your flavour is optimised for.

Xubuntu is much less resource-intensive than the other two, and I use it on my system which is more than capable of a running Gnome or KDE smoothly. I recommend it highly.

AFAIK they all use the same linux kernel, so hardware detection should be the same.

The Ubuntu Guide is a great resource. Its info is also applicable to Xubuntu.
Some other links which you will find very useful are the [Xubuntu homepage]("http://www.xubuntu.org/"), plus [here]("https://help.ubuntu.com/community/RestrictedFormats") and [here]("http://nongeeksight.blogspot.com/2006/07/multimedia-in-xubuntu-made-easy.html") for
setting up multimedia. 

You should probably also check out Automatix and Easy Ubuntu, which are scripts designed to automate the whole process of multimedia setup. I've never used them myself (I'm used to setting it all up manually) but apparently many people swear by them. You can search them on the Ubuntu forums for all the details.

HTH and welcome to Ubuntu!
