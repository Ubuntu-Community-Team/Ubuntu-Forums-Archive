### Post by garvinrick4 on 2009-11-13
Put Disc in CD tray run and see if it does not give you the option to run with WUBI.

Install with WUBI it will ask how much size you want Wubi folder to be. Up to 30 Gig.

Give them your password of choice twice and it should install in about 12 minutes or so.

It will be a folder in C: next to Users folder. Boots from Dos4grub. Can uninstall from package or from ADD and Remove programs in Windows.

C: drive will be /host in Ubuntu can bring files to Ubuntu from /host/users/your login name

Cannot bring files from Linux to Windows have to put on external drive and drag and drop while in windows.

Is a very good way to learn Ubuntu if you trash you just throw away and start over. Does not effect Windows. It is like an App in Windows, works very nice.

Have fun with linux, read Forums and learn. Plenty of .pdf files to download on Ubuntu.
