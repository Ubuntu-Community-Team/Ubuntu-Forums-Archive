### Post by tommcd on 2011-03-11
The recommended way to install graphics card drivers in Ubuntu is to go to: System-> Administration-> Additional Drivers. Then see if Ubuntu offers to instal the ati driver for you. See this: [https://help.ubuntu.com/community/BinaryDriverHowto/ATI](https://help.ubuntu.com/community/BinaryDriverHowto/ATI)
Also see this about the open source *radeon* driver: [https://help.ubuntu.com/community/RadeonDriver](https://help.ubuntu.com/community/RadeonDriver)
I have no personal experience with ati, since I have always used nvidia cards, which are still more linux friendly than ati. Hopefully the info I have provided will help.
