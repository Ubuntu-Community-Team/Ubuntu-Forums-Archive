### Post by loudnlownoma on 2008-04-13
I have been lucky so far with all of the dual-boots I have setup - Laptop had XP and Feisty, then XP and Gutsy - Desktop has Vista, Gutsy, and Hardy. Of course, it neeer hurts to be safe and have the back-up, but it has worked very well for me so far without trouble.
