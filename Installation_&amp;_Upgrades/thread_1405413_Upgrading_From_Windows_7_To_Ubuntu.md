### Post by crlang13 on 2010-02-12
> **RedSingularity said:**
> You mean when you install Ubuntu?  There will be an option while installing it.  This option will allow you to install Ubuntu alongside windows without deleting anything.

Exactly what Red said.  This is called duel booting.  Essentially, you're given the option to partition your computer into two hardrives, one for windows, one for ubuntu.  If you choose this option, each time you turn on the computer, you'll be asked to choose which operating system you want to use.

When you go to install ubuntu, the other option to partitioning (duel booting) is to wipe Windows all together and only have Ubuntu on your computer.  If you choose this option, Windows and everything on it will be deleted and then replaced with Ubuntu...  So you'll have to back everything up in order to move it to Ubuntu

If you choose to duel boot, back everything up just in case.  Most likely, things will not go wrong, but it's good to take the precaution.:D

### Post by crlang13 on 2010-02-12
OH, and to add to above.

If you simply want to try Ubuntu without installing it, you can boot off the live CD.

Basically, make the live CD [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) (to download Ubuntu) [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto) (on making the cd)

Then boot with the CD in, and, instead of selecting install from the list of options, choose to boot from the CD.  With this, no changes are made to your computer and you can have a fool around without having to install anything.
