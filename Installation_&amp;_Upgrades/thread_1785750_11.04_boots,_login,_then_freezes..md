### Post by mörgæs on 2011-06-18
There is some advice here, for example the boot options:
[http://ubuntuforums.org/showthread.php?t=1580857](http://ubuntuforums.org/showthread.php?t=1580857)

Yes, you can boot a live CD and get all your user files, if you want to reinstall 10.04. When the file manager is running, you can click 'file system' in the panel to the left. This will give access to your old home directory.
