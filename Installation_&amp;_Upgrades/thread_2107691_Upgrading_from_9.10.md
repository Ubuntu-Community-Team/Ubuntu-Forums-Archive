### Post by grahammechanical on 2013-01-22
Have you run the live session yet? What happened?

[http://www.ubuntu.com/download/help/install-desktop-latest]("http://www.ubuntu.com/download/help/install-desktop-latest")

The live session should let you run Gparted and you can write out information about your hard disk partitions. A screen shot posted here would be even better.

Basically, when you install Ubuntu you need to choose the Do Something Else options. And edit the partition settings to tell the installer to install Ubuntu into the partition that has 9.10 in it. I would like to be more specific but without greater detail about your hard disk partitions I might be leading you astray.
