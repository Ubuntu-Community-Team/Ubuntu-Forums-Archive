### Post by Partyboi2 on 2009-01-24
The program is in the Ubuntu repositories, open a terminal (Applications>Accessories>Terminal) and install by typing
```
sudo apt-get install geda
``` Or you can open Synaptic Package Manager and search for 'geda' and install.
