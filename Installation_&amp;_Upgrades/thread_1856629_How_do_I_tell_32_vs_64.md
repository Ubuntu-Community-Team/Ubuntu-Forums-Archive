### Post by Mazate on 2011-10-08
How do I tell if I have the 32 or 64 bit version of ubuntu installed?

---

### Post by oldos2er on 2011-10-08
```
uname -a
```

x86_64 GNU/Linux means a 64-bit kernel, i686 GNU/Linux means 32-bit.
