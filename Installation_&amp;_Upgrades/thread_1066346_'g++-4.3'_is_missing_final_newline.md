### Post by Partyboi2 on 2009-02-11
You could try what is suggested [[COLOR=Blue]here[/COLOR]]("https://answers.launchpad.net/ubuntu/+question/2591")
So open a terminal and
```
echo -en '\n' | sudo tee -a /var/lib/dpkg/info/g++-4.3.list

```
