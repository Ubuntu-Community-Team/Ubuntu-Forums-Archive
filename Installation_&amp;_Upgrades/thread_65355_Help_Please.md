### Post by mlomker on 2005-09-13
Laptops use something called a winmodem.  Basically it's a DSP that runs a piece of software that tells it how to be a modem (it's not a 'real' modem, perse).  There are drivers available for a variety of them but none are easy to get working/install.  This one of the user-unfriendly aspects of linux.

[Here is](http://www.linmodems.org/) a website that specializes in the topic.

If you wanted to buy a copy of Linspire or Xandros (instead of using Ubuntu) then it'd probably work out of the box.  I use to run Linspire and everything just works, but it costs money for that convenience!
