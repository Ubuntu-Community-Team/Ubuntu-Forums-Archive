My condolences...

Canon printers suck.  You can purchase a proprietary package called Turbo Print which supports some Canon printers.  It worked with our MP980, but that printer was such a POS (network printer, and you can only set it up while connected to a windows computer w/ a usb cable) always dropped off the network and was a super pain to get back on again.

I ended up ditching it and getting an HP 6500 wireless.  Fantastic reliable printer with excellent Linux support.

In a nutshell, you have three options- get Turboprint (I think it was about $40usd), which at best is a band-aid fix, go back to windows and happily print to your canon printer, or buy something with better Linux support.  

For the record- that HP was recently on sale at newegg for $100 w/ free shipping.  I paid $140 + tax at Costco.
