### Post by cdub772 on 2007-11-30
I was having similar problems, and found the issue to be I had a space and a period in my user name.

I changed my user name to start with a lowercase letter and only contain other lowercase letters, and this issue went away.

Also, make sure all USB/PCMIA devices are disconnected (I read that somewhere else).

Hope it works for you...

### Post by Revan256 on 2007-12-04
I had the exact same problem and here's how i fixed it. I had some confusion on what computer could burn what discs, so I ended up downloading the file onto one computer, using a jumpdrive with the ISO Image burning software on it to take it to a disc-burning computer and created it there. I think I might have lost some files in translation.

1-Make sure you download and burn the cd all on the same computer
2-Take into consideration the above mentioned minor details (about the usernames and such)
3-Take your time, don't rush and let everything load first. Touch nothing except the install icon and go to town.
