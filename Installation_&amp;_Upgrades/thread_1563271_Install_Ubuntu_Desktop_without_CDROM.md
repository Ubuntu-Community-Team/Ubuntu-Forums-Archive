### Post by Revolutionary101 on 2010-08-28
Yes you will be able to install a Desktop version of Ubuntu via a flash drive. However, not all computers support booting from USB. You will have to check your BIOS to see if you can boot from a USB drive. To check just look for boot priority and see if there is an option for USB boot. If you can't find it then your computer doesn't support booting from USB.
