I'd go dual-boot for sure.. that way your daughter can startup Windows anytime, since she uses it at work this may be needed.
[https://help.ubuntu.com/community/WindowsDualBoot]("https://help.ubuntu.com/community/WindowsDualBoot")
Don't worry too much about the partitioning, if you want to go dual boot the only thing you have to worry about it have Windows installed and during the installation of Ubuntu stay off the Windows NTFS-partition, at least don't format it. ;-)

List of distro's I can recommend to anyone in search for ease of setup, maintenance and use, in random order..
Ubuntu, [Kubuntu]("http://www.kubuntu.org/"), [Mint]("http://linuxmint.com/"), [PCLOS]("http://www.pclinuxos.com/"), [openSUSE]("http://www.opensuse.org/"), [Fedora]("http://fedoraproject.org/") , [Mandriva]("http://www.mandriva.com/"), [Zenwalk]("http://www.zenwalk.org/") etc.. etc..
