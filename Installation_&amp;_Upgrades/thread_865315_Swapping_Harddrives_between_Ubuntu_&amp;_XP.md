### Post by wpshooter on 2008-07-20
No, primarily it has to do with the drivers, etc. that would be required and installed by the O/S for that particular computer/motherboard & other components.  Please note that this is a somewhat simplistic explanation.

But before you would try to put that XP hard drive in that computer and try to get it running on that system by reinstalling XP on it, you would more than likely want to set the BIOS on the computer back to defaults and then customize various settings as applicable.
