### Post by Wim Sturkenboom on 2012-12-22
Don't use versions that are no longer supported.

In the Ubuntu family, try lubuntu or xubuntu; they will probably work smoother on your hardware than Ubuntu. Also, either use 12.04 or 12.10 (11.10 support ends in april 2013 so you have to upgrade).

And there are plenty other options if it does not have to be an ubuntu family member.
