### Post by jheady on 2007-05-05
I have been having a difficult time installing 7.04 on a new ASUS P1-AH2 (onboard NVIDIA GeForce 6150 video).  Using the live CD graphical install, the highest available resolution is 800x600.  The bottoms of the installation windows are not visible, and thus neither are the buttons to continue through the installation.  I have tried to run through xorg configuration, which shows higher resolutions are possible, however after running and restarting the x server, 800x600 is still the highest resolution available.  Does anyone have any suggestions?  Should I just use the text installer instead?  I would prefer to use the graphical installer if possible, but if it means re-burning a disc with updated drivers or configuration files, I'd rather not.  Thanks!

---

### Post by benn333 on 2007-05-06
Try pressing "Ctrl,Alt,+" a few times (the one on the numeric pad, not by the backspace key). This should cycle you through a couple other resolutions.

If that doesn't give you better options, moving the mouse to the edge of the screen should allow you to scroll around the rest of the desktop (it's partly displaying off-screen). It's not ideal, but it should get you through the install until you're able to install a driver for your graphics card.

---

### Post by jheady on 2007-05-06
Thanks for the advice ... I didn't know about cycling through resolutions.  The scrolling option did not work as I would have thought.  I just installed using the text installer and everything was fine.  Thanks again for the response.
