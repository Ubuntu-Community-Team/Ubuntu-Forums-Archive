---

### Post by Dennis N on 2022-04-30
Installing gnome-shell-extensions from the extensions website with Firefox is not possible with the snap version of Firefox. Use the **gnome-shell-extensions-manager** utility to install and manage extensions. It's in the Ubuntu repository and also available as a Flatpak.
