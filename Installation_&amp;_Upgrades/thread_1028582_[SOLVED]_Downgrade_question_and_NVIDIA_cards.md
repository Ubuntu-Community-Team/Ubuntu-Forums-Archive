### Post by sanford55 on 2009-01-02
Hi everyone.  I currently dual boot XP and Ubuntu.  I upgraded to 8.10.  However, I have an NVIDIA gforce graphics card.  This card is not supported so I was warned that a generic nv driver would be used.  I asked on the forum and it sounded like all that would be lost is 3D effects.  No big deal.  But I lost more:  I cannot get the 1440x900 resolution that works best with my Dell widescreen monitor.  None of the resolutions I tried in Ubuntu 8.10 are any good.  

So I want to go back to 8.04.  My question is whether there are any problems doing this (I do not see many posts about downgrading and possible problems with it) and whether Hardy will automatically start using the original graphics driver rather than the generic and inadequate one.  

Thanks for your help, Sanford

### Post by 67GTA on 2009-01-02
There is no way to downgrade. You can reinstall 8.04 and keep your home folder. When you boot for the first time, it should automatically use the nv driver. You can install the 3D nvidia driver at that point.
