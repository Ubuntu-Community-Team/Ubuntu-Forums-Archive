### Post by Odrodzona-Sarmacja on 2007-07-09
Setup?

When you run win program in wine better do it it in window mode (wine->config->graphics->"emulate a virtual desktop" option). If the window freezes, then you can close it without crashing ubuntu. When you mount a cd in ubuntu don't forget to mount it in wine too (wine->config->drives->add->show advanced->cd-rom->"set the path to the mount in linux"). If something doesn't work as advertised on wine, then you maybe need to search for "wine tools" and start installing parts of original windows system. When everything works perfectly in windowed wine it still doesn't mean it won't crash in full screen mode. That's it. I hope it helps :)
