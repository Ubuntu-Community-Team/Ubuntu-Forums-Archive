### Post by AndyCooll on 2007-09-22
AFAIA you can't, you'll need to download the server version. The server version is actually still a full CD, it just doesn't load into a live cd environment. 

As you say the server install itself is just a basic command line install. However IIRC this CD will still actually do a normal default desktop install by default, you have to actually type "server" at the start for it to actually do a server install.
