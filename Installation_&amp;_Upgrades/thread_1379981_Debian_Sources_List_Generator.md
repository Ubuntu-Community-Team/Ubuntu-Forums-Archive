Preseeded config-file generator? I'm not sure what you mean by that.
When you install Ubuntu it will just install all the files off the CD. You don't need to set up a sourses.list before hand.
With Debian, you can do a net install, or a "business card" install. This installs a minimal command line system using a much smaller install CD. You then set up your sources.list and apt-get the packages you want. In this way you can get a more customized system. 
Ubuntu has a similar "minimal install" CD:
[http://www.psychocats.net/ubuntu/minimal](http://www.psychocats.net/ubuntu/minimal)
