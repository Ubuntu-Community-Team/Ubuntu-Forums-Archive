### Post by sudodus on 2014-01-02
That is possible two ways.

The Windows way: Copy and paste by marking and hotkey or drop-down menu.

The linux way: Mark (only) by pressing the left mouse button and move till the text is marked. Paste at the cursor in the other window by clicking the middle button or wheel (or both keys at the same time if there is no middle button).

Both ways are possible between terminals as well as GUI editors (when different kinds and the same kinds).
