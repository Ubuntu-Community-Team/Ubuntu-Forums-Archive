### Post by Cheesemill on 2013-06-12
Downloads are here...
[http://www.ubuntu.com/download/desktop](http://www.ubuntu.com/download/desktop)

You should use the 64-bit version of either 12.04 or 13.04, it's up to you to decide which one.

12.04 is supported until 2017 but has slightly older versions of software included
13.04 is only supported until January but has the latest versions of all of the included software.

12.04 may be more stable as there has been more time to fix any bugs found since it was released.
13.04 has a newer kernel so it has better support for new hardware.

At the end of the day it's your choice, I would suggest downloading them both and trying them out from a USB stick before you decide which one you want to install.
