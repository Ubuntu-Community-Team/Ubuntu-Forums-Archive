### Post by jerome1232 on 2012-03-23
A .bin is like a .exe, you simply run it. Set it as executable (right click, properies, permisisions tab, allow executing file as a program)

Then double click it, click run, you may have to click run in terminal, if it's poorly written and exits with some message about not having permission, you may need to pop open a terminal, type "sudo " with a space after it and no quotes, then drag and drop the file into your terminal and press enter.
