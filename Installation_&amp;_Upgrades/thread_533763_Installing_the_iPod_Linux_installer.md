### Post by DirtDawg on 2007-08-24
Hi, I cannot help you with iPod Linux, but I can suggest you try [Rockbox]("http://www.rockbox.org/") instead. It's built using iPod Linux and there's plenty of documentation. If you carefully read through the [manual specified to your model]("http://www.rockbox.org/manual.shtml"), you shouldn't have any problems. It's a fairly easy process.

I've been using Rockbox for months now and i think it's the bee's knee's.

### Post by Paul133 on 2007-08-24
Note: Debian/Ubuntu users must install the libpng3 and libssl0.9.7 libraries (apt-get install libpng3 libssl0.9.7) before attempting to run the installer as it is not installed by default.

Did you install those libraries?

This seems to be instructions for installing iPodLinux on an iPod:
[http://ipodlinux.org/Installation#Starting_iPodLinux](http://ipodlinux.org/Installation#Starting_iPodLinux)

For the record, I don't use iPL myself, though I did get it working on a nano in the past. Good luck.
