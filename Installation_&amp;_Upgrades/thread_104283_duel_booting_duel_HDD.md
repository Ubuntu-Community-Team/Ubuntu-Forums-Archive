### Post by hod139 on 2005-12-15
I'm not sure if I understand your question.  If you want to have Windows on one hard drive and ubuntu on the other, that is possible and fine.  The ubuntu installer won't copy the data from the second hardrive to the Windows drive for you though, you will have to do that before installing ubuntu.  The ubuntu installer will (should :)) recognize Windows on the other drive when installing ubuntu onto the second, and set up the boat loader correctly.  The ubuntu wiki has many pages on howto install if this does not answer your question.  [https://wiki.ubuntu.com/Installation?highlight=%28install%29]("https://wiki.ubuntu.com/Installation?highlight=%28install%29")
