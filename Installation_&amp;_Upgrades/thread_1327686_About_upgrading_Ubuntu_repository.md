### Post by howefield on 2009-11-15
In the main, the software in the repository is not updated between Ubuntu releases, bar a few notable exceptions and security fixes.

If you want to keep some other piece of software up to date, you either need to find a .deb package, or maybe a repository maintained by the developers, or compile from source.
