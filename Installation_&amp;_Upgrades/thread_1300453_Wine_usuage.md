### Post by Mark Phelps on 2009-10-25
> **Camilia said:**
> Has anyone used wine to read CD written for windows?  I tried it and it didn't work. How do get it to work? 


Contrary to some assumptions, installing Wine does not install any new device drivers.  So, any hardware that did not work before the Wine install, will not suddenly work afterward.

I'm not aware of any CDs "written for Windows".  If, however, they are commercial CDs and expect to run an MS Windows program when they are inserted, THAT is a problem.

You will have to find out what that program is and see if it can be installed in Wine.  Suggest to check the site below to verify Wine compatibility for that program.  If it's not listed, or if it's rating is not Gold or Platinum, you're wasting your time trying to use Wine:

[http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true]("http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true")
