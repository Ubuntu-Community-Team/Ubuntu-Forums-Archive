### Post by gungaden on 2005-03-21
[QUOTE=lao_V]May seem like a silly question but how are you burning them on the CD? Normal files or as an ISO image??[/QUOTE]

A very good question..... if you did not burn the ISO file as an image, it will not boot and install. Most CD burning software needs to be specially configured to burn the ISO as an image.

Also note that most AMD processors need the i386 ISO rather than the "AMD64" ISO to install and run properly.
