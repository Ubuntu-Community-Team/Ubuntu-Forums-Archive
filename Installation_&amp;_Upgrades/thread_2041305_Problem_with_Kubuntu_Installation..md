### Post by mastablasta on 2012-08-14
it could be that wubi has some limitations or needs special settings in case of multiboot. I wouldn't know. one way to test this is to uninstall Ubuntu and try to install only Kubuntu. Make sure you image is good. 
 
a good method is to download image first, make sure it's good, add wubi.exe in same folder and then run it. that way it should use that good image for install.
 
if you plan on using Kubutnu it's good to remove some gnome stuff form it otherwise oyu might end up doubling some services. for example when i added Xubuntu desktop to Ubuntu i ended up with 2 network managers and 2 battery monitors. it was not necessary and waste of resources.
 
so how to remove Ubuntu and get only Kubuntu (KDE)? well quite easy:
[http://www.psychocats.net/ubuntu/purekubuntu](http://www.psychocats.net/ubuntu/purekubuntu)
