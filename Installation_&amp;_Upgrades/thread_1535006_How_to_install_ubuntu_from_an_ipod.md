### Post by varunendra on 2010-07-22
Please post some more details about where you want to install ubuntu, whether currently you have a working OS there or not, etc.


If your ipod can be detected by your pc/laptop's bios as an external USB storage device, then [unetbootin]("http://unetbootin.sourceforge.net/") can do the job for you, but first you'll need a working win/linux installation somewhere to use unetbootin (or any other method, for that matter)!
