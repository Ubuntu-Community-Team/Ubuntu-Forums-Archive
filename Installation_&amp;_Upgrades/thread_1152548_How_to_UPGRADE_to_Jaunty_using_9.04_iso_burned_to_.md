---

### Post by HyRax on 2009-05-07
You can only upgrade from CD if you downloaded the ALTERNATE INSTALL CD. The Live CD cannot perform upgrades.

If you have the Alternate disc, then it's just a matter of booting up your 8.10 installation, stick the CD in, wait for the prompt and hit the Upgrade button.
