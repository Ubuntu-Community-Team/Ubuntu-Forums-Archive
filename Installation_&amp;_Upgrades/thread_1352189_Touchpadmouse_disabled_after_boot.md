### Post by lemming465 on 2009-12-12
It's almost certainly some kind of kernel driver bug, and you aren't the only one seeing it.  If you can hold off until 10.04, it might be better next april.  Otherwise if the regular Ubuntu version desktop version is OK, feel free to run that.  However, if 9.10 goes south on you, you'd have to reinstall an older version such as 8.04 or 9.04; there is no easy way to downgrade.  Be cautious about your backups if you take the plunge with the newer version.
