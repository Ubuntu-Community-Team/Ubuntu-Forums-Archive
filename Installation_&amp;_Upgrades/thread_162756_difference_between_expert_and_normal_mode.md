Normal is more automatic: the machine decides the defaults and tweaks.  Expert is less automatic: you decide the defaults and tweaks.

### Post by 5-HT on 2006-04-19
If you are new to Linux, I would highly recommend sticking with the normal install.

If you are worrying about partitioning, the normal install method does allow you to manually edit your partition tables.

Also, if you choose do to an expert install, your user is not added to the sudoers list automatically (at least in Breezy).
