### Post by fantab on 2013-06-05
So you are NOT dual booting Windows and Linux?
That installer is only for 'UPGRADING' from previous Windows version to Win8. If you don't have any Windows already installed then that key/installer won't work. You will have do use Win8 DVD/USB to do a complete install and then use that 'product-key'. To do this you have to have NTFS Partition or 'Unallocated space' on your Hard Disk to be able to install Win8.

### Post by Mark Phelps on 2013-06-05
fantab is correct -- this comes directly from the page you linked:

> This PC must be running Windows 7, Windows Vista, Windows XP with Service Pack 3 (SP3), Windows 8 Release Preview, Windows 8 Consumer Preview, or Windows Developer Preview. (Windows 8 Setup checks that your PC meets all system requirements before it installs Windows 8.)
