### Post by Steve Zenone on 2008-02-27
According to [https://help.ubuntu.com/community/PortableDevices/iPhone](https://help.ubuntu.com/community/PortableDevices/iPhone) gtkpod 0.99.12 is needed to fix corruption on an iphone from gtkpod 0.99.10:
[INDENT]*"The database corruption from 0.99.10 can be fixed so tune loss is temporary - just re-write with 0.9.12"*.[/INDENT]
Have any of you had luck compiling gtkpod 0.99.12 on Ubuntu 7.10? I've continued running into errors and wanted to check with the community before I spin my wheels any longer on this.

Thank you in advance for any feedback and help. --steve

### Post by Steve Zenone on 2008-03-01
Instead of beating my head up against the wall to get gtkpod 0.99.12 compiled and working, I opted to use Amarok. I was a bit hesitant to use Amarok at first since I had been using gtkpod for a year+ for my Nano and was comfortable with it. However, Amarok, in my opinion, is superior. In addition, Amarok fixed the iphone database corruption caused by gtkpod 0.99.10.
