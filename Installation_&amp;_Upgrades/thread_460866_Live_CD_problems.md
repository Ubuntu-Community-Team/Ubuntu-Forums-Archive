### Post by rillip on 2007-06-01
The alternate CD does not allow for running without installing, which is why Moth_Wingthane said "also, if you want to try it without installing..."

I would recommend you try 6.06 or 6.10 if you're having issues with 7.04.  In theorey, it's always getting better, but in reality, some people have hardware recognition issues crop up from time to time.  However, you should be aware that some people have reported major issues trying to get suspend/hibernate to work on these versions, whereas 7.04 fixed them.  Something to think about since you mentioned it's a laptop.
