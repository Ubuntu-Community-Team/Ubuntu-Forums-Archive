### Post by Partyboi2 on 2009-08-13
Hi, you will need to download the alternate cd, you cannot use the live cd to upgrade.

### Post by Mark224 on 2009-08-13
Thanks.  I suspected this but I am puzzled why there is a need for two CDs.

### Post by Partyboi2 on 2009-08-13
The alternate cd is a text base installer, unlike the Live Desktop version you do not get the option to try before installing. 
The live cd is good if you want to check that Ubuntu runs ok on your machine.
The alternate cd is good if you are installing on low end spec machines or want to do a upgrade.
