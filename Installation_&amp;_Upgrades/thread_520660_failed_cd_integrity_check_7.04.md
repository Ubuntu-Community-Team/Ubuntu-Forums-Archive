### Post by tgm4883 on 2007-08-08
It's my understanding that some cd drives will always fail the CD no matter what (even is you used ship it).  I don't remember the reason for this (as I was using Fedora at the time, Fedora gives you a warning at the CD check screen about this).  I don't see why this couldn't apply in this case also.  With that being said, I have also had CD burners that would burn a CD which would only reliably work in the same burner.  In either case, as long as you verified the CD after burning, and burned at the slowest possible speed, you should be fine.  If you are doing a fresh install I don't see what it would hurt to try the install.
