### Post by grahammechanical on 2014-04-15
I have been using Trusty Tahr (14.04) for months and I also use Chromium. The 14.04 Software Centre does have the Adobe Flash plug-in Installer in it. Although it is for Mozilla, it will also work for Chromium. I have it installed on my Trusty Tahr.

The Adobe web site will not have a version for 14.04 because 14.04 has not yet been officially released. And also Adobe stopped upgrading its flash player for Linux the other year. So, it is doubtful if there is a latest Adobe Linux Flash Player version.  Oh, by the way, it is Chromium 34 that I am using. furthermore, Adobe Flash Player is proprietary software. It cannot be in the Software Centre for legal reasons. That is why it is only an Installer that is in the Software Centre.

There is a second thing we can do and that is to install Pepper Flash. This link explains it.
