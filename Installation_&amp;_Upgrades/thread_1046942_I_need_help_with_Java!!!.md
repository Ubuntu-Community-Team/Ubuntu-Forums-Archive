### Post by Partyboi2 on 2009-01-21
Open a terminal (Applications>Accessories>Terminal) and change to where the downloaded file is, so if its on your Desktop
```
cd ~/Desktop
```then 
```
sudo chmod +x jre-6u11-linux-i586.bin

```then
```
sudo ./jre-6u11-linux-i586.bin

```
