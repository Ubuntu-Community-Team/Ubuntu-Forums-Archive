### Post by boz on 2006-10-26
About two weeks ago I had decided that I was going to do a fresh install on my Compaq v6000 to eliminate a few minor annoyances.  Before doing so I had decided to try an upgrade to Edgy to see if it would break things.  It certainly did.  I couldn't even boot into rescue mode.  (The problem's not with the devs, it's with the proprietary hardware.  I just want to clarify that I'm not complaining.)

It was hard enough to get Dapper to work on this thing, so I would wait a couple of months before attempting a fresh install (not an upgrade).  At LEAST a couple of months.  Beware, also, that the Live CD tends to be misleading with this laptop.  On my first install of Ubuntu the Live CD ran flawlessly, but the kernel froze on boot up after the install.  (BTW, if you're using this computer and you still have problems with Dapper, add "noapic" to your boot parameters and remove "pci=off" if you're using it.)  Just thought I would share my experience.

In short, v6000 owners I would definitely not try to do an upgrade.  Do a fresh install if you want to use Edgy.  Granted, that was two weeks ago and I'm sure the devs have made leaps and bounds since then, but it's just safer.  It appears that some people are having upgrade problems, so I'm 90% sure anyone with this laptop will as well because it's still a problem under Dapper.
