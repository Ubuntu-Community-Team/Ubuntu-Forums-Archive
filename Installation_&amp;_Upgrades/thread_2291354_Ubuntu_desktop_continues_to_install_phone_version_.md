### Post by Bucky Ball on 2015-08-19
Welcome. Unless you are intending to test and report bugs, don't use 15.10. It is not released yet and not a good place to start unless you want an extreme learning curve. 

If you want support it is best to give as much information as you can. What happens when you attempt to install 15.04? What are the specs of your machine? 'No luck' is not enough. Please tell us exactly what is happening so we can help. See the link in my signature for the best ways to post for support.

Personally, I'd recommend 14.04 LTS as a good place to start. It is a long-term support release so supported until 2019. 15.04 until January 2016. If you have a very new machine 15.04 may be preferable, though (as in the last six months or so).
