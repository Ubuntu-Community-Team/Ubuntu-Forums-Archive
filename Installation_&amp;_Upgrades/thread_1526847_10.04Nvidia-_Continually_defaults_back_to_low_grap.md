Old post that maybe someone will stumble upon:

Be sure to have the nvidia module built for the kernel you want to use the driver with. Therefore you MUST have the headers for this appropriate kernel version installed and take a look for "nvidia.(k)o" or "nvidia-current.ko" or something similar in
```
/lib/modules/$(uname -r)/kernel/drivers/video/
```or

```
/lib/modules/$(uname -r)/updates/dkms/
```

to insert module
```
sudo modprobe nvidia
```Restart computer or just X (CRTL+ALT+BACKSPACE).
