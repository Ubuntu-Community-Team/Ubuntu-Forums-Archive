### Post by lemming465 on 2008-05-10
If an Ubuntu 7.10 live CD works and an 8.04 live CD doesn't, then you have exotic enough hardware to trip over an unpatched bug.  You might want to hold off on upgrading for a few months.  If neither version works, you probably have a hardware failure, not a software problem.  Try reseating the CPU and memory, or take it to a repair shop, or something.
