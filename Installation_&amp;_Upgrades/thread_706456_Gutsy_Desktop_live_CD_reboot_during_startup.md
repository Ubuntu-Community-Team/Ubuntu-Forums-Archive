### Post by dpiddy on 2008-02-24
I was trying to install Gutsy Desktop on a pretty vanilla white box (2.8GHz P4, VIA mobo) but the live system was rebooting during load. I removed quiet and splash from the boot line and saw something right before the reboot about attaching something security-wise.

I checked the BIOS settings and didn't see anything odd but loading my BIOS's default settings fixed it. Thought I'd pass that on.
