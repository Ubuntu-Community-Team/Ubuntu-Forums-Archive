### Post by mikewhatever on 2009-08-23
One possible reason, you've installed the server edition, which has no GUI.
If that's the case, you can either reinstall using the desktop cd, or add the GUI by installing ubuntu-desktop from CLI. If that's not the case, try this guide: [http://www.psychocats.net/ubuntu/nox](http://www.psychocats.net/ubuntu/nox).
