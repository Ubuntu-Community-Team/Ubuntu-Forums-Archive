### Post by tsiMental on 2006-06-15
This package accually works!! :D 

But you need to do one more thing to make it happen:

```
sudo -i
915resolution -l
```
make a note of the number in front of the 1024x768 32bit resolution and type this command:
915resolution <the number you just noted> 1280 768 32
for me this looked like:
```
915resolution 54 1280 768 32
```Then you could do a "915resolution -l" again to verify the change.

Now, I pressed [CTRL]+[ALT]+Backspace and ubuntu came back on with the correct resolution. ;) (Using the Live-CD still)


Remember, this change is not permanent, so you will either need to do this each time you boot or add the "915resolution <the number you just noted> 1280 768 32" command to your boot script, before the xserver is started.

Take a look at the README file of 915resolution for more information...
/usr/share/doc/915resolution/README.txt.gz
