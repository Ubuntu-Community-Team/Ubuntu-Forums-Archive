### Post by Mark Phelps on 2009-10-27
You can't install MS Windows programs directly in Ubuntu.  You have to first install the Wine or Crossover packages to set up an interface.

But, before you do that, go to the WineHQ site, look through the Application Compatibility database for the apps you want to install.  If they are not listed, or if their ratings aren't Gold or Platinum, you're wasting your time.
