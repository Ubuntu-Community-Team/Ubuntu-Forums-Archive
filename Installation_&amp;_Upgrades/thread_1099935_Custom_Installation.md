### Post by Rocket2DMn on 2009-03-18
You can do a minimal install - [https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)

IIRC, you are given the option not to install some stuff.  Also, if you do a server installation as suggested above, you can install the gnome-core metapackage instead of the ubuntu-desktop one.  That will only install the stuff needed for Gnome and not all the extra stuff that typically comes in a fresh installation.
