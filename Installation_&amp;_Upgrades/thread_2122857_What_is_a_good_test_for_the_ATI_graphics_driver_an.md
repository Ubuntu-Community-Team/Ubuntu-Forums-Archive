### Post by superdaveozzborn on 2013-03-29
Minetest is totally free, the name/login/password is only to allow you to return to the game at a later time, simply create a user name and password for the first login. after that you will need the same user name and password to return to the game and retain ownership of the structure and items such as chests and blocks. but this game is more dependent on the CPU because it uses low resolution graphics unless you add the 128bit or 256bit theme packs to it which are very nice. and be warned ITS VERY ADDICTING!! ;)
