### Post by ubfan1 on 2014-07-31
This is a UEFI machine right?  Are you trying to install on the internal disk, or an external disk?  Do you have a second disk (maybe SSD) in the machine?  You will eventually get to dealing with the Nvidia video issues, but that comes after a successful boot.  Are you trying to Dual Boot with Windows?  
Normally, you would first go to the UEFI settings/BIOS and turn off fast boot (so you can more easily hit the function keys like F2 or F12), and turn off raid or any other rapid start business.  Also adjust the boot order to make USB/CDROM before the hard disk.  Secure boot on Toshibas seems to work fine in grub if you are booting Ubuntu, but grub cannot boot Windows with secure boot on (but you can always get to it via the efi boot menu).  Then in Windows power settings, make sure that shutdown is really not sleep or hibernate.     Shrink the windows partition/filesystem from the Windows disk manager, and check the disk a few times.
Now you should be ready to boot the Ubuntu CD or USB, and try or install.
