### Post by cybergnome on 2010-11-18
The problem with your concept is that the CDs and DVDs used as live systems are read only.  They have to create a system RAM disk for read/write, and it limited by the amount of physical memory in a particular machine.  Performance is also accordingly degraded.  Otherwise, the answer is that it's possible, but probably not practical.
