### Post by deadflowr on 2016-09-26
All versions regardless will say 14.04.5.
But from the download page linked to already, just go down to the utopic stack (unsupported)
and the kernel to be installed will be 3.16 

If you want to know why they all end up 14.04.5 is because that is the latest version number all get to when the system is fully up-to-date.
That has nothing to do with the hardware stack.

A fresh install of Ubuntu 14.04.1 fully-updated will show as 14.04.5.

Simply put, even if you installed 14.04.2, by the time it got updated, it would turn into 14.04.5.

And a mini.iso is fully up-to-date during the install.
