### Post by Sims2789 on 2007-12-12
View the CD's contents (I don't know what this is in Mac OS, but it's "Explore" in Windows) and make sure it looks like a normal CD. If there's only one file on it and that file ends in iso then you didn't burn it correctly. I don't know how to burn an image in Mac OS, but someone else on the forums probably does. This was the mistake I made when going from Windows to Ubuntu.

Also, make sure you downloaded the alternate PowerPC image, not the alternate i386 one.

If you've both burned the CD correctly and have burned the correct CD, you might need to go into your laptop's BIOS and set the CD drive to compatibility mode. Again, I don't know how to do this on a Mac, but usually you enter your BIOS when you first turn your computer on by pressing/holding a specific button.
