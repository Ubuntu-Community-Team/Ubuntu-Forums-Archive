### Post by damis648 on 2008-05-03
To do a full install and dual-boot with windows, you have to boot the CD, so go into your bios and set the CD as the first boot device. Boot up with the cd in the drive, then select you language and choose "Install Ubuntu" and follow the on-screen directions.
