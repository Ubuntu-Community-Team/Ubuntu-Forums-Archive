### Post by grahammechanical on 2023-07-04
I doubt it. I have a laptop with the kind of features that your keyboard has. I purchased the machine with Ubuntu pre-installed. And all the keyboard functions worked. But when I installed another version of Ubuntu in another partition of the same machine the keyboard features stopped working. The machine's supplier installed the necessary drivers in the first installation.

To get all the keyboard functions working in the second install of Ubuntu I had to setup the supplier's software repository on the second install just like it had been set up in the first install and then download and install the drivers. Doing it was simple once I had worked it all out.

Does Asus have software repositories for Linux? Do they contain Linux hardware drivers for things like keyboards? The Linux developers provide most hardware drivers within the standard install process. You can test if Linux already has drivers for your keyboard by downloading the Ubuntu ISO image. Burning it to a USB memory stick and testing Ubuntu in the TRY UBUNTU mode. The standard functions of the keyboard should work as normal. It is the special functions that you should test.
