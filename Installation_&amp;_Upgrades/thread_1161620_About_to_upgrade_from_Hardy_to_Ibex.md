### Post by Partyboi2 on 2009-05-17
Hi, if your hardy system is very stable do you really need to upgrade? You are currently using a long term support version and when the next LTS comes out you will have the option to upgrade straight to the new one without having to install the releases in between.
If you do want to upgrade then you will need to open up Software Sources and under the "Update" tab make sure that "Release Upgrade" is set to "Normal"
Make sure you machine is up to date and then there should be an option to upgrade to the next release when you open update-manager.
