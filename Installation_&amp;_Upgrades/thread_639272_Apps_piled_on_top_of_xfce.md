### Post by d0nny2600 on 2007-12-13
Not really. Xfce will speed up the system yes - But whether that outweights the look, feel and user friendlyness of a more sophisticated Window Manager like Gnome or KDE is debateable. Try it out..If you like it keep it..If you don't like it scrap it! If you are really looking for performance upgrades consider installing some more memory! Gnome and KDE don't really require the lastest and greatest machines to run efficiently so you should be fine using them everyday if you don't like how Xfce integrates into your distro!
