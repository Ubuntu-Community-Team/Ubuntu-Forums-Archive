None of this makes sense.  :confused:  Merely running the Live CD should not initiate installation.  Nor should it shut down the computer.

First, your BIOS -- is it set to boot from the CD ROM drive first?

Second, your Live CD -- did you (a) check the md5sum on the downloaded ISO file, and (b) burn it at the slowest speed available for your optical drive (4X is best, 8X normally works, anything faster often does not work)?
