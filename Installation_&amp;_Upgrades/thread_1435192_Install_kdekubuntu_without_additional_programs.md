### Post by Herman on 2010-03-21
[Modest Specs]("http://www.psychocats.net/ubuntu/minimal")
See psychocats excellent web page about that, in the link above.

[Windows 98 with Gutsy Gibbon Low Memory Install]("http://members.iinet.net.au/%7Eherman546/p2.html")
Look at the web page in the next link (above), and see how you can install a minimal system with the Ubuntu, (or Kubuntu) 'Alternate' CD.
Notice that when you get to the end, you can install any desktop you like, gnome, KDE, XFCE, fluxbox, IceWM, whichever one(s) you like, so in your case just install KDE.

Another easy way to go about it would be to install [Ubuntu Rescue Remix]("http://ubuntu-rescue-remix.org/") instead and then add the KDE desktop to that and add your desired programs. Ubuntu Rescue remix comes with some great data recovery programs which you might not need but they won't be any weight to carry, you won't notice any darg on your system resources just because of those and you never know, maybe they will come in handy someday.

So now you know three ways to perform a minimal installation and you should be able to install whichever desktop you like.  :D
