### Post by ubuntu27 on 2007-09-10
Ubuntu LTS (Dapper Drake)? If I remember correctly, in that version you need another CD (Alternate CD or Install CD) to install.

What is your objective on Install Ubuntu 6 (LTS) ? Are you going to install a server? Do you really need the long term support? 
If not, then I recommend your to install the latest Ubuntu 7.04 (Fesity Fawn) 

You can download either [Alternate CD or Desktop CD here]("http://www.ubuntu.com/getubuntu/downloadmirrors")

**Alternate CD** = A CD or DVD specialized in (only) installing your Ubuntu Operating System. Has more advance option than Desktop CD.

**Desktop CD** = Previously called LiveCD, it let's you try out Ubuntu without installing to your hard drive. Current version also let's you install form this CD.
