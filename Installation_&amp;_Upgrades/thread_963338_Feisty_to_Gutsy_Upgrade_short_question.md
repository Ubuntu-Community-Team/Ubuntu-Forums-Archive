---

### Post by Partyboi2 on 2008-10-30
You won't be able to roll back, so best to backup your important stuff before upgrading. when you are upgrading it will ask you if you want to keep your modified files or to use the new ones. You can back these up when they ask to modify them. Or if you already know the files you want to kept you could back them up before the upgrade so that after the upgrade you can use them if you prefer.
