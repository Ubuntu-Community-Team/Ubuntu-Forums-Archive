### Post by kvonb on 2007-04-18
I have absolutely no idea what a VFD is, but you don't have to install anything on your hard disk.

Simply wait until tomorrow when the brand new version of Ubuntu is officially released, download, burn to CD, and boot.

It is a "LiveCD" which means that you will boot into a fully operational copy of Ubuntu 7.04, running from the CD and RAM.

Next use "Synaptic" to search for the software you want to try out, if it's not in there, search the Internet for it and try to find the .deb package version.

Simply click on the downloaded package to install it.

If it works for you, you could then proceed to install Ubuntu, if it doesn't work, simply press the reset button and eject the CD.

Nothing will have been changed on your computer, and you can continue to use it as before.
