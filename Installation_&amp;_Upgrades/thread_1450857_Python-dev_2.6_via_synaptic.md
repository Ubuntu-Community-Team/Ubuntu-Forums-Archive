### Post by ajgreeny on 2010-04-09
You open synaptic and search for python-dev.  If you are not aware of it, get to know synaptic, as it is just about the best package manager application alive, in my opinion.  I have never used the old Add/Remove nor the new Software Centre, but have always used synaptic.

You could also, of course, simply do the old fashioned ```
sudo apt-get install python-dev
```in terminal; quick and easy, and in spite of my comments about synaptic, it actually does exactly the same job as these other ways.
