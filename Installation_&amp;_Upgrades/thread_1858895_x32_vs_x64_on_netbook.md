### Post by mcduck on 2011-10-13
With the tasks you do with the netbook, and the amount of RAM you have, using 64-bit OS wouldn't really make much, if any, difference over 32-bit OS.

Such tasks simply don't benefit from the ability to use larger numbers or higher precision that 64-bit computing would give you. If you were encoding video, rendering 3D graphics, or doing some other highly CPU-intensive media-related or scientific tasks things would be a bit different, but I kind of doubt any sane person would even want to do such things on Atom-powered netbook... ;)

On the other hand running the 64-bit version doesn't have much downsides either, perhaps slightly better overall performance thanks for the larger 64-bit registries on the CPU, balanced by the slightly larger memory usage of a 64-bit system. I'd say I doubt you'll notice any difference between 32-bit and 64-bit OS, either one should perform pretty much the same for you.
