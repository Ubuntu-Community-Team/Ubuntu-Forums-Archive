### Post by pyros on 2007-06-05
you are correct, you cannot boot from the cd if you just opend the iso image and coppied the files to the disk. you have to have a cd burning application that can handle images. I am assuming that you are using windows, Nero will do this, as well as some versions of roxio's easy cd creator. If you have neither of these, try:
[http://infrarecorder.sourceforge.net/](http://infrarecorder.sourceforge.net/)
an opensource burner for windows. that is what I use.  You may find the wiki documentation for burning the iso image useful as well, it can be found here:
[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

iso is an international standards organization, but if I recall correctly, the name is not an acronym, but a greek word meaning equal.
