### Post by Rocket2DMn on 2008-09-09
64MB is not enough RAM to use the Ubuntu installer, I believe 128 is the minimum for the alternate install cd.  The LiveCD installer needs 256MB I think.
What you are basically asking for the is alternate install cd, but again, you don't have enough RAM with only 64MB.

Unless you upgrade the RAM (which for a computer that old, you shouldn't bother), you will probably want to try a smaller distribution, like Damn Small Linux (DSL) or Puppy Linux.  They are pretty minimal, but that's a pretty old computer...
