### Post by Cheesemill on 2009-11-23
No need to extract the files, just make sure you burn the file as a disc image, don't just burn the iso file onto a disk.

Then just reboot your machine with the CD in the drive and away you go (you may have to change settings in your BIOS if it doesn't automatically boot from the CD).

For more info check out the guides here [http://www.psychocats.net/ubuntu/index](http://www.psychocats.net/ubuntu/index)
