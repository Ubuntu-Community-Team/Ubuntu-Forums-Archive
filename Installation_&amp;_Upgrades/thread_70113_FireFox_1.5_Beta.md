### Post by GreyFox503 on 2005-09-28
I'm using it right now, and its really easy to do.  Download the .tar.gz here:

[http://download.mozilla.org/?product=firefox-1.5b1&os=linux&lang=en-US](http://download.mozilla.org/?product=firefox-1.5b1&os=linux&lang=en-US)

Then you unpack it to wherever you want.  It comes with the working program itself, no install or anything.  To use it you just run the "firefox" script in the directory.

Before you do this, though, I would back up your .mozilla directory which contains your bookmarks, extensions, settings, etc.

```
cp -a ~/.mozilla ~/.mozilla_backup
```

Upon its first run it will check your extensions for compatibility.  Almost surely some of them will not work with the beta.  That's ok though.  You can still use it and you should have your bookmarks too.  If you want to run the old firefox, just start it in the normal way.
