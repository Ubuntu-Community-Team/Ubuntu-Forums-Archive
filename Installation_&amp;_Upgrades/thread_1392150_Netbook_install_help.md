### Post by darkod on 2010-01-27
Yes, the installer will give you option to format whole disk and use it. It's called like "Erase and use entire disk" or similar.
Unless you manage to get an error message from the screen, we can't know why you can't use Try Ubuntu or Install Ubuntu options.

### Post by redkcir on 2010-01-27
I have an Aspire one and did an install on a SD card. I formatted the SD and installed ubuntu while the SD card was in a USB reader. I then changed my boot priority to look at the USB first. Now I just insert the SD and boot to what every OS I want.
