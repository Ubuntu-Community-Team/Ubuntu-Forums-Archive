### Post by bwhite82 on 2007-04-19
Try making the following changes to this section:

> ## ## End Default Options ##

title Ubuntu, kernel 2.6.20-15-generic
root (hd0,0)
kernel /boot/vmlinuz-2.6.20-15-generic root=UUID=9071da45-9147-4f20-8e6c-762462a1bf62 ro quiet splash noapic
initrd /boot/initrd.img-2.6.20-15-generic
quiet
savedefault

The above is what you have now, I'm thinking you need to change that "root=UUID" to something more like this:

> ## ## End Default Options ##

title Ubuntu, kernel 2.6.20-15-generic
root (hd0,0)
kernel /boot/vmlinuz-2.6.20-15-generic **root=/dev/sdaX** ro quiet splash noapic
initrd /boot/initrd.img-2.6.20-15-generic
quiet
savedefault

Where "sdaX" is whichever partition you have Ubuntu installed on.
