### Post by jim_p on 2007-07-29
Since Ctrl+Alt+Del works, the OS is still loading and is searches forever to find something,
lets say a device driver. Try pressing Esc or F2 (I don't remember which key ubuntu uses)
to go into verbose mode and locate when it stops.
Remove any unessesary hardware if so.
For instance, my pc stops at "Loading device drivers" when I have plugged in both my usb stick AND the
bluetooth dongle.
