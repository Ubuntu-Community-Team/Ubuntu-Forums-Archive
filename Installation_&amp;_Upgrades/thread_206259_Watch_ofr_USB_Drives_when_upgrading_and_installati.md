### Post by MicroFace on 2006-06-29
When upgrading from 5.x to Dapper Drake 6.06 I did not realize I had my USB Drive in my system.  The upgrade took so long on some steps that I just shut the machine off and hosed the whole process so I downloaded the desktop version and tried several times to load the system on my computer.
All with very very long delays. We are talking 2 days for an installation here.  Then I tested teh HArd drive with the Ultimate Boot CD and the hard drive was fine, but the installation was still taking forever.  Until just today I was about to remove the hard drive and put in another when i noticed that there was a USB drive in the back.  I pulled that out and everything went as fast as lightning.
So be very very very careful when upgrading and installing and make sure you have no USB drives attached to the system.
