### Post by mikewhatever on 2007-02-16
Similar problems with Live CDs are not unheard of, but there are several things you can do.
1. md5sum check [https://help.ubuntu.com/community/HowToMD5SUM?highlight=%28md5sum%29](https://help.ubuntu.com/community/HowToMD5SUM?highlight=%28md5sum%29)
2. reburning at lower speed [https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28iso%29](https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28iso%29)
3. running CD integrity check from the first screen.

Make sure there is enough RAM to run a Live CD session.
