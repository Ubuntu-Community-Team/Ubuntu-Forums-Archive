### Post by -Jeremy- on 2010-05-16
To change Gnome's shortcuts you can either go to:
System > Preferences > Keyboard Shortcuts

or

Open Gconf-Editor > apps > metacity > global_keybindings

Please note that there is a difference between Gnome shortcuts and Linux/UNIX shortcuts but I'm not sure where to find those (yet).
