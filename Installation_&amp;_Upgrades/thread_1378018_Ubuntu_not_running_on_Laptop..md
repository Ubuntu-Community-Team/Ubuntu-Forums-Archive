### Post by presence1960 on 2010-01-10
[https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)

With less than 384 MB of RAM you will not be able to use the Live CD for a graphical install. You will have to use the alternate text based installer. Get it [here]("http://www.ubuntu.com/getubuntu/downloadmirrors#alternate").
