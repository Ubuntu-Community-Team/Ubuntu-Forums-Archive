### Post by dethlock on 2008-10-28
While installing 8.04 via CD, I encountered the following problem.  The keyboard and mouse stop working right after selecting the language to use.  Obviously, installation is a bit more challenging without a keyboard or mouse.  To solve this problem, go into the BIOS and temporarily disable power management, and also turn off halt on all errors.
