I recently became a member of the Ubuntu Community:guitar:.  I installed the 9.04 version and it was love at first sight.  However, I couldn't get my sound to work on Youtube or any other site.  So I read somewhere that Internet Explorer is not good to download image files, it is recommended that you use FireFox.  So I gave it a shot and it fixed the sound problems.

SO IN SHORT, USE FIREFOX IN WINDOWS ([-X) TO DOWNLOAD THE UBUNTU IMAGE FILE.  THERE SHOULD BE NO SOUND PROBLEMS AFTER THAT.
