### Post by Bucky Ball on 2016-03-17
*Thread moved to **Installation & Upgrade**.*

Welcome. People will take into account you're a new comer and you will be better served here. 

Yes, I have some suggestions. Plug in an ethernet cable and your problem should go away. Or choose not to configure wireless during install. This is best as better also to not update or tick the box to install third-party drivers during install also. You can install those drivers later if you need them.

Also best to 'Try Ubuntu' from the live media first prior to installing. You are going to need to check whether you have Win installed in UEFI or BIOS mode and also let us know which version of Win. Depending on which, you MUST boot into Windows and switch off hibernation before you're going to be able install anything. You also need to create some free space to install Ubuntu to.

Frankly, if you only want to try Ubuntu and have a look around, why are you installing it? Or you're not? Just choose 'Try Ubuntu' when you eventually get there and that will give you a live session from the install media without changing anything on your hard drive.
