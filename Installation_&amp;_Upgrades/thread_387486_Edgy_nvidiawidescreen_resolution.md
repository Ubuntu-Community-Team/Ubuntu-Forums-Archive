### Post by bulldog on 2007-03-18
First of all you need to be sure your graphics card can use this resolution.

If it can,install the driver for your graphics.
```
sudo aptitude install nvidia-glx
```

Last step ```
sudo dpkg-reconfigure xserver-xorg
```
Pick the desired resolution and take a look at your xorg.conf.
```
gksudo gedit /etc/X11/xorg.conf
``` and take a look if the driver is listed proper as "nvidia" instead "nv"  if not change it manually and take a look if the right resolution appears in your monitor section.

Reboot and it should be fine.

### Post by bulldog on 2007-03-18
I have the following in my xorg.conf [have a 20 inch widescreen monitor]
```
Section "Monitor"
    Identifier     "Generic Monitor"
    HorizSync       28.0 - 84.0 <----
    VertRefresh     43.0 - 60.0 <----
    Option         "DPMS"
EndSection

Section "Device"
    Identifier     "nVidia Corporation NV40 [GeForce 6800 Ultra]"
    Driver         "nvidia"
EndSection

Section "Screen"
    Identifier     "Default Screen"
    Device         "nVidia Corporation NV40 [GeForce 6800 Ultra]"
    Monitor        "Generic Monitor"
    DefaultDepth    24
    Option         "AddARGBGLXVisuals" "True"
    SubSection     "Display"
    Modes      "1680x1050" "1440x900" "1152x864" "1024x768" "800x600" "640x480"
```
