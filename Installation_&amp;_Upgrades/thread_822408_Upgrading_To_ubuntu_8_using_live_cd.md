### Post by psycho5 on 2008-06-08
Just boot with the live cd, go to install

When it comes to partitioner, If you want to completely remove your ubuntu 7 and use ubuntu 8, then just format the exisiting ext3/ext2 partitions and use them as / and swap 

or / and swap and /home or however you want to set it up. 

That'll give you a clean install.

To setup a swap you need to go under format options and select swap.
