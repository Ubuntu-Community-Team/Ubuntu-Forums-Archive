### Post by oldfred on 2010-02-17
Is your music in your 8.04 install or in your /home? If so you will need to back it up anyway and should make a new partition for either /home or /music or /data if more than just music, so it does not get overwritten. This assumes you have space on your hard drive. A new install without having the /home data only needs to be 10-20GB and you can use your existing swap. If you have room you can create a new partition for root and just install leaving your existing for the time being.
