### Post by Impavidus on 2018-11-09
Download tools like wget or torrent clients should be able to resume the download from the point where is was interrupted. Web browsers can't, as far as I know. Once you have the .iso file, you can install Ubuntu without a network connection. I think upgrading works too, except for the packages not included on the live disk. And you may need a network connection to download proprietary drivers (if you need them) or restricted extras (if you want them).
