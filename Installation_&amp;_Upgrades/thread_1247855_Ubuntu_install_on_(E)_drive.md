Ubuntu specifically, and Linux in general, doesn't use .exe files -- those are used in MS Windows.

If Ubuntu installed correctly, you should be offered a menu at boot time that allows you to select Windows or Ubuntu.  Selecting Ubuntu will launch it to a desktop.  You don't need to go into any folders and click on any files.
