I'd recommend upgrading to Bionic (Ubuntu 18.04) which you can do online. Xenial is EOL in April next year. Bionic has a sufficient version of the package
[https://packages.ubuntu.com/search?keywords=fwupdate&searchon=names&suite=all&section=all](https://packages.ubuntu.com/search?keywords=fwupdate&searchon=names&suite=all&section=all)

You could even boot to the Bionic install CD/USB, install the fwupdate application in the live CD/USB environment, do the upgrade then reboot to the installed OS on your internal drive.
