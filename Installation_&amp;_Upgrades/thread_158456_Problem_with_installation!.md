### Post by taurus on 2006-04-11
Can you list the spec of your system?  Also, did you download the ISO image yourself or did you receive a free CD from ShipIt?  If you downloaded and burnt it yourself, make sure you check the integrity of the ISO image with either md5sum or sha1sum before you burn it and you need to burn it at a slow speed like 4x...  Otherwise, you can always try the LiveCD to see if it can find everything on your machine first before you install it (not with the LiveCD but with the install CD).
