### Post by sikander3786 on 2010-09-27
If your computer can run older version Ubuntu or derivative, it surely should run the new one as well. No need to go back to an older version.

Did you check quality of the image used to create Live USB? See here.

[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

You've checked on 2 different HDDs, I will recommend that you also check with some other Flash Drive. Or try installing from a CD if possible.
