### Post by aysiu on 2008-07-04
I'm sure there's a menu item for this, but I don't know where it is in Xubuntu, so press Alt-F2 and type ```
gksudo gdmsetup
``` enter your password, and then select to enable automatic login. I think it's under security or convenience, or something like that.
