Afaik, there is no 8.09, so you either have 8.04 or 8.10.

You can't upgrade directly from 8.10 to 9.10, but instead you must upgrade to each version in turn.  So, from 8.04, you must upgrade to 8.10, then to 9.04, then to 9.10.

This page gives some info: [https://help.ubuntu.com/community/UpgradeNotes](https://help.ubuntu.com/community/UpgradeNotes)

It is not necessary to reinstall.
