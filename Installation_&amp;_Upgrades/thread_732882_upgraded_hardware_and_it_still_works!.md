### Post by simmonsm on 2008-03-23
I've been running Ubuntu 7.10 for a while on an Abit AL8 with P4 3.2MHz CPU and 1Gb RAM for a few years now. I decided to upgrade to a Asus Maximus with a Core2 Duo E6550. Now, I know they are both LGA775 but after replacing and booting from my original Ubuntu disk I was AMAZED to find that everything is working as before (only quicker) The only minor change was to the network config and then /etc/init.d/networking restart. ! From experience, windoze would have bitched about just every little change and required a load of machine restarts. I'll probably install a 64-bit version anyway at some point.
