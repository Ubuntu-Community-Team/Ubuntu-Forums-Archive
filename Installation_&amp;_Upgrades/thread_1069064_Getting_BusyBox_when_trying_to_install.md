### Post by prshah on 2009-02-13
> **Hexifford said:**
> BusyBox comes up

If you get an initramfs (aka busy box) screen:

a) Check the CD
b) Burn CD at a lower speed
c) [Boot with the all_generic_ide kernel option]("http://ubuntuforums.org/showpost.php?p=5436513&postcount=7")
d) [Change the CD drive]("http://ubuntuforums.org/showthread.php?t=703228")
