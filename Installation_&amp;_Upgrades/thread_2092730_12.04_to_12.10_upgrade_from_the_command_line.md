### Post by darkod on 2012-12-08
First make sure it is fully updated. Then you can try opening the update manager with the -d option, like:
gksu update-manager -d

If that also doesn't offer 12.10, I think in terminal it should be:
sudo do-release-upgrade

Make sure you have a full backup of your data before you start. Upgrades can sometimes go wrong making the system unbootable.
