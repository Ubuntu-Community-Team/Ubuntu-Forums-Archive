### Post by lao_V on 2005-04-04
Are you using Kynaptic package manager? If so then you won't find a place to edit repositories at the moment cuz kynaptic is still in early development stage. There's two workarounds to this. Either edit the /etc/apt/sources.list manually or install synaptic package manager and do settings -> repositories
