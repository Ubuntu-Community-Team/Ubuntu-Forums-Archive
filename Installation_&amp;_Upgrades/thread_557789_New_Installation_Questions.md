### Post by Nebby on 2007-09-23
The best way to install it is download the LiveCD, which lets you use Ubuntu without actually installing it on your harddrive (it runs entirely from CD and ram). Once you've got that running, there's an install button on the desktop.
Once you hit that, Ubuntu has a partitioning wizard in the graphical install, which'll help you shrink your XP partition and make one for Ubuntu.
