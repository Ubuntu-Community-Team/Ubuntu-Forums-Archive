### Post by ubunterooster on 2010-11-06
My guess is that you are only receiving LTS versions

Go to system > administration > update manager > settings > updates > Release upgrade. Select "Normal releases"

---

### Post by Enigmapond on 2010-11-06
You have to change the settings to "normal releases" however, you will be better off doing a fresh install..also if your Lucid is running good...there's really no need to change it and you may encounter many problems along the way....
