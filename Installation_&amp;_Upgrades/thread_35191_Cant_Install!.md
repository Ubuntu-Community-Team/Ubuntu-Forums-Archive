### Post by bored2k on 2005-05-18
[QUOTE=Mofo]I just dragged it and burned, but the file was ubuntu-5.04-install-i386.iso
so i am not sure what happened, but it looks as if it was burned as an ISO.[/QUOTE]
 That's your problem. You don't just drag n drop an .ISO disc. ISO = Equal wich means you are making an exact copy of something [Ubuntu disc]. Whatever program you are using (that's NOT XP built in burner) select BURN IMAGE > DISC and point at the .iso. In nero it would be burn image [then click on all types of images because nero reads .nrg by default] and select the ubuntu.iso.
