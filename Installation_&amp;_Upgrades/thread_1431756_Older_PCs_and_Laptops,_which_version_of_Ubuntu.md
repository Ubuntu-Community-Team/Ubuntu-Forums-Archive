### Post by mikewhatever on 2010-03-17
1. Yes, absolutely.

2. It depends on the specs. Pentium4 is fine for any version, but if those PCs have much less then 512MB of RAM, I would recommend a more light weight distro. With 512MB you should be fine running Xubuntu, while Ubuntu and Kubuntu might be a bit slow. With 1GB or more, pick the version you like.

3. You probably meant an iso image for PC, and yes you can.
