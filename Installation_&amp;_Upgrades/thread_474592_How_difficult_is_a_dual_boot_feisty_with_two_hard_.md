### Post by confused57 on 2007-06-15
Here's a relatively current thread by someone who wanted to install Ubuntu on a separate hard drive from Windows:

[http://ubuntuforums.org/showthread.php?t=473962](http://ubuntuforums.org/showthread.php?t=473962)

For me personally, having Ubuntu and Window's on separate hard drives is less complicated, if you install grub to the Ubuntu drive's mbr...that way if Ubuntu fails to boot, then change your bios boot order to boot your Window's drive.
