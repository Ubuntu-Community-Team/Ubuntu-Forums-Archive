### Post by pepsi_max2k on 2007-04-22
From the live CD open Places > Home Folder. You should see a list of your old partitions on the left. Double click the one containing your old home / linux install, then use the icon on the desktop to open it, browse to /home and copy files elsewhere :)

**EDIT:** ok, places > home is for gnome ubuntu... should be similar ish in kubuntu, do you have a main computer icon on desktop? there should be a list of available partitions in there or another file browser somewhere...
