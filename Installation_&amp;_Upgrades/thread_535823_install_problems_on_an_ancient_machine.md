### Post by kostkon on 2007-08-27
The memory is too low! You could try installing it with the *alternate install CD*. I think you will have better results with *Xubuntu*.

If you could increase the memory at least to 128MB it would be perfect. You could even install *Ubuntu* (with the *alternate install CD* as already said) and get decent performance. I am saying this from experience. I have installed *Ubuntu* on a P200 with 128MB RAM and it ran slowly of course, but decently.

You could also try installing the *server edition* of *Ubuntu* and putting a light window manager like *Fluxbox* afterwards.

If you want to use the PC as a server you don't even need a GUI. Just install the *server edition* of *Ubuntu*.

Unfortunately, the *LiveCD* needs at least 256MB of RAM.
