### Post by Revolutionary101 on 2010-08-29
Ubuntu does install on any x86 chipset. For a rule of thumb if Windows can run on the chipset, Ubuntu will run with it. The only problem that you might run into is for your PCI cards (GPUs, Sound Cards etc.). That is why I suggest going to [this page]("https://wiki.ubuntu.com/HardwareSupport") to check what hardware is known to work with Ubuntu.
