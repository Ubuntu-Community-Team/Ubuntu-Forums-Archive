### Post by Idefix82 on 2008-11-22
If you want to upgrade using the update manager then you will probably either be able to jump from the current LTS release to the next LTS release or you will have to update hopping through all the versions.

On the other hand you always have the option of doing a clean install (and keeping your /home if it's on a separate partition) so with that you will be able to jump from any version to any other.
