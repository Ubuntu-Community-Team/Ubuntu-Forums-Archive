### Post by tea for one on 2018-08-20
> **kendor22 said:**
> I have downloaded  18-4 to dvd but cannot get it to open from my drive ,am i missing something.  Ken

Have you "burnt" the Ubuntu ISO to a DVD?

If you haven't, there are many guides on the internet such as [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

If you have successfully prepared the DVD:- 

You have to leave the DVD in the drive
Restart the PC
Check that your BIOS is set to load the DVD first

This should load the Ubuntu Live System from where you can explore the software and eventually install the OS.

There are also many guides for installation and I would advise a bit of light reading before proceeding. [https://help.ubuntu.com/lts/installation-guide/index.html](https://help.ubuntu.com/lts/installation-guide/index.html)

Good luck
