### Post by logos34 on 2007-12-10
mozilla-thunderbird is just a transitional package...you want thunderbird (2.0.0.8), that's the main one.  The rest are popular extensions like lightning (calendar), mozilla-imagezoom (magnifier), gnome support, etc.  I have both of the above installed plus thunderbrowse and one or two others.  If you want the very latest version (2.0.0.9) plus any other add-ons get it from the mozilla site...it'll run from a folder in your home directory.
