### Post by pizzach on 2005-11-15
Running Mac OS X, go to "Disk Utility" (it's in the Utilities folder which is in the Application folder.)  There is a MD5SUMS menu somewhere there.  Furthermore, burn the CD using Disk Utility.  There is another menu that says "Burn Image...." or something like that.


If you're using Mac OS 9 I think you would use the "Disk Copy" application.  Otherwise never tired to burn an image In Mac OS 9...
