### Post by upchucky on 2009-04-21
make sure you have the Live bootable version.

boot the pc with it, this means to boot but not do an actual install, if the pc boots up to a graphical display, then the actual install will work.

there is also an alternate live cd that can be used to boot if a particular machine has trouble booting to a graphical display.

there are instances where an actual install seems to fail even after proving the display worked just by booting with the live cd, when this happens there are a couple of files that need to be edited. this can be gone into later if needed.
