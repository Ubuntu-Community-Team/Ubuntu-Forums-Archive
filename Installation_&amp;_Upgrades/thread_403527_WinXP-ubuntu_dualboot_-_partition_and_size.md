### Post by mikewhatever on 2007-04-07
Windows will need one partition, presumably NTFS file system. The size depends on the HDD size, but the partition shouldn't be less then 5 BG (more is advisable).
Ubuntu needs at least two partitions (root - about 5-10 GB, swap - about 512 MB)  , but also can have three (root - about 2-3 BG,home - about 5-10 GB, swap -512 MB) 
Look here for more [http://psychocats.net/ubuntu/partitioning](http://psychocats.net/ubuntu/partitioning)
