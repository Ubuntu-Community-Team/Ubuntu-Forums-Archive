### Post by rehilliard on 2009-05-26
seriously, if the "bad partition" shows up as unformatted in Windows Disk Management (right-click My Computer-->Manage-->Disk Management), then when you install Ubuntu, just tell the installer to use all available freespace and it should find the unformatted partition. Just pay attention to the lower graphic on the disk partitioning screen. it'll show you how much is going to be allocated for ubuntu. Know how big the "bad partition" is going into the installation.
