### Post by zvacet on 2010-05-13
To be sure that your system is up-to-date type in terminal

```
sudo apt-get update && sudo apt-get upgrade
```

After that follow [this.]("http://www.ubuntu.com/getubuntu/upgrading#Upgrading%20Using%20the%20Alternate%20CD/DVD")
