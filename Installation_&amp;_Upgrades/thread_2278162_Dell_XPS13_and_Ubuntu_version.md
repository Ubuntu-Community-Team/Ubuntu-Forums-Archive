### Post by grahammechanical on 2015-05-15
Specifically speaking, for hardware compatibility it is always best to buy a machine with Ubuntu pre-installed. Generally speaking, for hardware compatibility the latest versions of Ubuntu are more likely to be compatible with hardware that was released since older versions of Ubuntu were released.

As regards Dell XPS13 Developer Edition it comes with Ubuntu 14.04 LTS
