### Post by howefield on 2009-06-26
You don't have to, but you can if you want.

The installer will help you make your choice during the disk partitioning stage.

If you decide to resize the windows partition and install Ubuntu, you will have the choice of operating system to boot into at boot up.

### Post by lisati on 2009-06-26
As howefield says, you can but you don't have to. The Ubuntu installer's partitioner has an option to use the entire hard disk, which will wipe absolutely everything.

It might be a good idea to try dual boot or Wubi first before committing yourself to wiping Windows.

Important: make a backup of all important data, and make sure you have a backup of any Windows recovery partition that might be present. The last thing you want to have happen is the need to reinstate Windows for whatever reason and discover that you can't, or you have permanently lost something important.
