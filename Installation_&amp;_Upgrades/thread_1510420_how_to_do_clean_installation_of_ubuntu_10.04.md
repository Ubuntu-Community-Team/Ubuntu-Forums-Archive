### Post by TenPlus1 on 2010-06-15
If you want to erase everything on your hard-drive and install Ubuntu 10.04 Lucid follow these steps:

1. Boot from the Live CD and select Install on Hard-drive
2. Select country & keyboard from screens that appear next
3. When partition screen appears select Use Entire Hard-Drive (easiest)
4. On next screen type user name, password and computer name
5. Install (grub will sort itself out from here)

Reboot and you'll have a fresh Ubuntu install...  

If you are using a graphics card like nvidia or ati then you will be prompter to install a Proprietary driver in the top right of the screen, other than that you might like to check out these sites for additional software/repositories:

[http://www.getdeb.net](http://www.getdeb.net)     and    [http://www/medibuntu.org](http://www/medibuntu.org)
