### Post by presence1960 on 2010-01-28
If the 20 GB is unpartitioned choose the use largest continuous free space option and ubuntu will create root & swap from the unallocated 20 GB.

If you wish to create your own partitions choose manual. If you choose install them side by side the partitioner will allow you to take space from windows- I don't think you want that option.
