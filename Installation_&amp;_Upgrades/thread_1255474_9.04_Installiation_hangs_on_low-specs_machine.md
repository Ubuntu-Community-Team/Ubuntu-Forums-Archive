### Post by MotleyPhule on 2009-09-01
I don't think the documentation lies exactly.  The minimal install it refers to is a barebones installation that gives you a command line and not much else.  You can then go on to install X from there, if your computer has enough ram.  If that's what you're interested in, then try following the instructions here: [http://www.psychocats.net/ubuntu/minimal](http://www.psychocats.net/ubuntu/minimal)

Are you using the right CD for this?  You need the alternate CD for that kind of installation.
