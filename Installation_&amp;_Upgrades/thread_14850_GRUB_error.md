According to the GRUB Manual at [http://www.gnu.org/software/grub/manual/grub.html#Troubleshooting](http://www.gnu.org/software/grub/manual/grub.html#Troubleshooting) error 3:
> 
Bad or corrupt data while decompressing file.
This error is returned if the run-length decompression code gets an internal error. This is usually from a corrupt file.
