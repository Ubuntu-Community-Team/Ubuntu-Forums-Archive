### Post by socket1 on 2009-07-26
For people having the Ubuntu 9.04 live cd or install not even start and crash to a BusyBox console I've read forums that said by using a USB key to install it it may work but I just plugged in a USB DVD Drive and it worked flawlessly compared to my regular IDE drive which gave me the issues before.

So if you have a USB cd/dvd drive and are having issues with ubuntu 9.04 crashing a busybox prompt before the livecd even starts then try installing it with the usb drive.

I used the x64 release for this I can't confirm that the 32bit version will have the same results.
