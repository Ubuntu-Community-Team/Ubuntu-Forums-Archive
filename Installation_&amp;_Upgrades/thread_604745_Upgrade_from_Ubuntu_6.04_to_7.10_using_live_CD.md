### Post by az on 2007-11-06
You cannot upgrade using the live cd.  The live cd is an all-or-nothing kind of install.

You also cannot upgrade from 6.06 to 7.10 directly.  You will need to upgrade to 7.04 first.

When the next LST release is out, you will be able to upgrade from 6.06 to it directly.
