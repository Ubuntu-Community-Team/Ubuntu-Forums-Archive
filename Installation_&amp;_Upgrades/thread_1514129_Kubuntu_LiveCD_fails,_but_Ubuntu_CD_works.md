### Post by ankspo71 on 2010-06-21
Hi,
I'm not sure why one would work and the other wouldn't, but it is possible to install Kubuntu from the alternate install cd, unless for some strange  reason that fails to boot for you too. The alternate install cd would be my recommendation. Keep your Ubuntu live cd though, because you can still use that for whenever you need to use a live cd.

[http://www.ubuntu.com/getubuntu/downloadmirrors#alternate](http://www.ubuntu.com/getubuntu/downloadmirrors#alternate)
The mirrors to download from are located towards the bottom of the page. After you select a download location, be sure to click on a "kubuntu" link, then choose an "alternate" iso to download. These don't have a live desktop or graphical installer, so it shouldn't fail to boot. Installation is pretty straight forward, except for maybe the partitioner which takes a little getting used to.
Hope this helps.
