### Post by Old_Grey_Wolf on 2011-01-07
It is best to back up your /home before doing an upgrade, just in case something doesn't work with your configuration.

You should update your current version before upgrading.

You can upgrade using the command 
```
sudo apt-get dist-upgrade
```
