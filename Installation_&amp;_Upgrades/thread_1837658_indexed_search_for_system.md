### Post by boblizar on 2011-09-02
tracker is a gnome indexed search program.  it quickly searches through files and indexes files to search quickly.  you can download it from the apt repo.  

alt + f2

run

gksu synaptic

then search for tracker

click tracker and install.

to edit the settings open a terminal and type "tracker" then hit tab a few times to get a list of the little programs it added....

alt + f2

run
tracker-preferences

then select show icon all the time.  and setup indexing to how you expect it to act, i tell it to take its time to index because i dont usually search for something in the first 5 minutes of my computer being on.
