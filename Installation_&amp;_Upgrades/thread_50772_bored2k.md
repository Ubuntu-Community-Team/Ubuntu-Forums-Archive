### Post by dataw0lf on 2005-07-21
sudo provides you 'root' level access via the /etc/sudoers file.  i.e., prepending any 'root' commands with sudo will do.

However, if you're intent on creating a 'real' root account, just type 'sudo passwd root' in a terminal and there you go.
