### Post by ballantony on 2011-11-17
You can't just copy it straight across, you need to use something like unetbootin [http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/) on a linux machine or pendrivelinux [http://www.pendrivelinux.com/](http://www.pendrivelinux.com/) if you are creating it on a windows machine.  
 
It's a good way to try different distros to see which is best.  (you could try lubuntu as well like @nothingspecial suggests)

### Post by TBABill on 2011-11-17
You can just install Unetbootin and then it's simple to make a bootable USB. I ran Lubuntu from a flash drive on a netbook with the Atom N450 processor and 1GB RAM and I ended up forgetting it was running from USB and not installed on the HDD. It ran so much better than Win7 starter. Only concern I had was backing up Windows so I could reinstall if something went wrong, wouldn't work later or if I just needed Windows for something.  I could dual-boot, no problem, but I wanted my Windows backed up to DVD and had no ability to do that since I didn't own an external DVD writer. For $25 it could be a life saver if you bork the install and overwrite something critical to your Windows install and need Windows later.
