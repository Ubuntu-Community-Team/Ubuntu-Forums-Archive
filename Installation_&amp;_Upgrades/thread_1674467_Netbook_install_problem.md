### Post by Quackers on 2011-01-24
It sounds like you are getting stuck at the "who are you?" page.
The usual cause is that the username has been started with a capital letter. No capital letters are permitted in that field.
Another cause can be that no password has been entered and/or confirmed.
Correct these fileds and the "forward" button should be usable.
