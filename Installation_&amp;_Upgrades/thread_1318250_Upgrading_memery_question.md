### Post by jheaton5 on 2009-11-07
> **kammce said:**
> Will I have to erase my entire hard drive in order to upgrade to 9.10. Also I am using the CD to upgrade.

If you are using a CD then you are not upgrading, you are doing a fresh install.  Yes, you will erase all of the partition you install ubuntu in.  It will not erase any other partition, e.g. Windows, unless you do something wrong.  If you want to upgrade from a previous ubuntu version use Udate Manager.
