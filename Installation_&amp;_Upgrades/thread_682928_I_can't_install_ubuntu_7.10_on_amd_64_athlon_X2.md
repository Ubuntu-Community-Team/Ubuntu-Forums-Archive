### Post by BuntuFreak on 2008-01-30
I had a similar problem on one of my desktops. Sometimes, the Ubuntu Live CD's "default" graphics driver does not play well with the graphics card in the computer. Try selecting "VESA" or "VGA" mode when booting off the live CD. That did the trick for me.

You could try the text based installation, of course. But that's so 19th century.:lolflag:
