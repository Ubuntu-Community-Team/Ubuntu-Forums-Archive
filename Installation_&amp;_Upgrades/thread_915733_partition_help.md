### Post by chronographer on 2008-09-10
Hi. Choose guided, then resize the existing partition to reduce its size by around 10-20GB. Make sure you have this much free space on the existing partition.

Alternately o the live cd you can manually resize the partition and then use 'existing free space" in the guided part of the installer.
