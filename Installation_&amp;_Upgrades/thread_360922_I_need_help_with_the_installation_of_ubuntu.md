### Post by phiphedog on 2007-02-13
Try booting with another bootable cd such as the windows cd and see if it that boots ok. If it does, I'd reburn the Ubuntu CD and try again. If it still doesn't boot with the windows CD then double check your boot priority in the BIOS and if it still fails then change CD/DVD drives.

### Post by mancel on 2007-02-13
I had the same problem..  I'd downloaded the ISO from a site other than the Ubuntu site so i went [>here<]("http://www.nullriver.com/index/products/winmd5sum") and downloaded winMd5sum to compare the checksum of the ISO i had with the correct checksum from [>this place<]("https://help.ubuntu.com/community/UbuntuHashes").  It turned out the ISO i had wasn't the right one because the checksums didn't compare.  Once i had an ISO that had the right checksum I burnt it onto a CD and it booted up fine.

Hope this helps.
