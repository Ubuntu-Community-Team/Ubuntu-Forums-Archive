Hi All, 

After uninstalling some of the applications installed under WINE, I still find the program entries listed under Applications > Wine > Programs. How can I remove these unwanted entries? I have looked under the wine directory and these are not listed under Programs or Windows directories therein.

### Post by frank95com on 2007-09-04
Just right click on application menu and then edit menus.
This opens alacarte, the menu editor. Just delete the programs' entries that you want to cancel and close the application.
All done!;)
