### Post by coldraven on 2013-11-05
If you double click the .tar.gz file the archive manager will open. Click "Extract" and choose where you want to put the folder.
Then open the folder and read the README text file by double clicking it. 
You will have to compile the program before you can use it. Try this, if you get errors you may need to install some packages, maybe "build-essential":
```
sudo apt-get install build-essential
```
See here for compiling:
[http://www.wikihow.com/Compile-a-Program-in-Linux](http://www.wikihow.com/Compile-a-Program-in-Linux)
