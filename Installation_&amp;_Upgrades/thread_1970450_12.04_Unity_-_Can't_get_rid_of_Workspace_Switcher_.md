### Post by HPA on 2012-05-01
> **ChoPnGo said:**
> How can I remove Workspace Switcher Button while still having multiple workspaces?
In ccsm go to "General Properties" and then to "Desktop Size" tab.
It's the last tab on the right.
Set all values to "1".
Then close ccsm and restart your computer.
After restart you will find that the Workspace Switcher Button is disappeared.

### Post by mclaborn on 2012-05-09
Here is the script I use that seems to work.

```
gconftool-2 --type int --set /apps/compiz-1/general/screen0/options/vsize 1
gconftool-2 --type int --set /apps/compiz-1/general/screen0/options/hsize 1
gconftool-2 --type int --set /apps/compiz-1/general/screen0/options/number_of_desktops 1
```

### Post by Darthfather on 2012-10-04
Here is my method: 
1. Requires compiz config with extra plugins
2. Open the main compiz window
2. Go to "*Desktop*"
3. Go to "*Ubuntu Unity Plugin*"
4. Go to "*Experimental*" tab
5. Scroll down and un-check "*Show "Expo Icon" in the launcher*"
6. That's it!
It worked for me. No need to reduce the number of your workspaces.
