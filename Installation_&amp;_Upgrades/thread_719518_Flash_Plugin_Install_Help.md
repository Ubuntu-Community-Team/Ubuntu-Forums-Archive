Found an interesting way to install flashplayer:

1) Download tar.gz
2) Terminal -- change name of extracted folder via nautilus
3) Change permissions in nautilus (shouldn't be able to, but can) RW make executable
4) Terminal Command:  install-flashplayer, Continue:
5) Gotta enter you browsers executable folder.

Worked for me!
