Do you mean installing Ubuntu in a Sun Sparc32 or an Sparc64 (a.k.a UltraSparc) box? No. At least not at the moment. Ubuntu supports IA32 (a.k.a Intel 386 through Pentium4 CPU family), IBM PowerPC on Apple's hardware (a.k.a Macintosh) and x86_64 (AMD 64, a.k.a Opteron and Athlon64) CPUs, the x86_64 distribution can be run in some Intel CPUs as well (but I don't recall which ones, I stay away of Intel's IA64 CPUs as a matter of principle ;-)).

Yet, there are alternatives. If you want to use GNU/Linux, you can use Debian (yes, Debian is *the* Universal Operating System, only [NetBSD]("http://www.netbsd.org/") runs on more hardware). If you want the Ubuntu experience but don't mind using a different OS, you can use [Nexenta]("http://www.gnusolaris.org/gswiki"), an [OpenSolaris]("http://www.opensolaris.org/") distro with an Ubuntu based GNU userland.
