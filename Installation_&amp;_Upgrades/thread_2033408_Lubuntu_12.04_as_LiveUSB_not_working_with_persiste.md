### Post by ajgreeny on 2012-07-26
I have never used the universal-usb-installer, but have used both the ubuntu startup-disk-creator and also unetbootin.  The startup-disk-creator is only available to run on ubuntu, but unetbootin can be run from Linux and Windows, so I would advise you to try one or other of those two applications and see if that gives you pertsistence.

I know it can be done as I used one of them (can't remember which) and get a persistent usb system of Lubuntu 12.04 with no problem.
