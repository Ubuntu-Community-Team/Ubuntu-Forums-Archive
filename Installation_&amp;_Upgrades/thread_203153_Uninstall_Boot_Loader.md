### Post by Herman on 2006-06-24
Okay, easy! Just make a GAG floppy disk or CD-RW first, (to install it from.) Then run the GAG floppy or CD, and look on figure 16 in this link [(*click here*]("http://users.bigpond.net.au/hermanzone/p12.htm")) and see where the button is called "**Save in [COLOR=Red]H[/COLOR]ard Disk**"?
Well after you have GAG set up by following the procedure described if figs 1 to 15, you just press your 'H' key and it will install GAG to your MBR and the first track of your hard disk from either your GAG floppy disk or from your GAG CD.
Let me know if that doesn't work and I'll test it to make sure nothing is wrong with those instructions, but I think that will be okay, I have tested those already quite a lot. maybe I need to emphasise more that you need to install it from CD or floppy disk.
Regards, Herman :D
