### Post by forestpixie on 2007-08-27
> The only problem is, I've spent ages rating all my music in Amarok and last time I changed distro., all these rating were lost

In assuming that Kubuntu and Ubuntu save files to similar places - all my Amarok data - collection dbase etc are in /home/kevin/.kde/share/apps/amarok - which is a hidden folder. Last time I reinstalled, before I had a separate home, I copied these beforehand and then replaced after install and all was well. Look in your home folder.

> If I keep my home folder the same when editing partitions and reinstalling Kubuntu, will this mean that I won't affect any of this data?

If home is on a separate partition,  when you reinstall mount it as /home but don't format and it'll use that as home without losing data - or at least it should - backup what you need to though :)

> Another problem I have is that I've saved quite a number of username/password combinations in Konquerer

I don't use Kubuntu so am not sure - but I would expect that there'll be a file with these - have a look in hidden folders in the same place as the amarok folder.
