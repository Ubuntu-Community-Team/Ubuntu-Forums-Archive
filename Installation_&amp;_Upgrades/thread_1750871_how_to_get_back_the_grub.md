### Post by oldfred on 2011-05-06
If you have the systems installed on different drives did you install the boot loaders to different drives. Then selecting the other drive in BIOS or one time boot key (f12 on mine) may work. I prefer to have different boot loaders on different drives and then set BIOS to boot grub as it gives a choice of Ubuntu or Windows. But you can also boot windows from BIOS in case of problems on Ubuntu drive.

If not you need to reinstall grub or grub2.
