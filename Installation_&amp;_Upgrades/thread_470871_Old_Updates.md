### Post by FyberOptic on 2007-06-11
I hope this hasn't been asked about or discussed before, but I didn't find anything related, so I thought I'd ask.

Basically, have Warty, Hoary, and Breezy stopped being supported?  I was running a backup machine which still had Hoary installed, and when I brought it up yesterday to do some things, I came to find out that the udpates server no longer had directories for these older distros (as you can see at [http://us.archive.ubuntu.com/ubuntu/dists/](http://us.archive.ubuntu.com/ubuntu/dists/)).  

I tried to force an upgrade to Dapper (the first oldest one available), but that totally broke a lot of stuff, including networking.  Luckily I had a Edgy text installer disc handy, which I was able to use as an APT source, and forced an upgrade to that.  I had to sort out a few small issues, but on the whole, that got things working again, luckily.  I did all the updates, and figured I wouldn't push my luck by trying an upgrade to Feisty.

I just don't understand why Ubuntu would seemingly stop supporting Hoary, when it's barely two years old (and Breezy which is less than that), yet I've been running a 2+ year old Debian on my main server (with regular updates) without issue.  Considering Ubuntu is boasted for its generally twice-yearly updates as opposed to Debian's two year periods, it doesn't really seem advantageous to have so many more versions if they're going to be unsupported so soon.
