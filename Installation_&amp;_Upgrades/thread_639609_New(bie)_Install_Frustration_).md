### Post by ubukool on 2007-12-13
Hi,

It really would be a lot easier if you defragged your windows and made some partition space on one drive for installing Ubuntu.  When I first used Ubuntu (Dapper Drake) I had a dual boot solution and there was no problem, Grub worked fine.  I know it can be a bit scary having to do things like repartitioning and terminal work, but generally if you follow the instructions everything is fine.  

I think it's got to be accepted the Ubuntu is going to be for the more computing-interested user, someone who doesn't just want to use Windows but to be a bit more adventurous with the sense of achievement that brings when you manage to achieve things and the sense of frustration you feel when you don't!  For me, Ubuntu is like a puzzle - you either want to work to get the solution and feel satisfied when you have triumphed, or you don't.  If you don't, there is always windows which 'just works' ( but sometimes it doesn't!!!).

I guess it depends on what kind of person you are and what you are looking for.
