### Post by BLaZuRE on 2012-04-14
Are you on a desktop or laptop?

If you are on a desktop, you may need to go into BIOS and change it so that your CD drive boots before your hard drive.

If you are on a laptop and are using an external cd/dvd drive, you may need to delay the boot by pressing keys to one of your motherboard's options (I forgot what I've done in the past, maybe BIOS).  That will let your dvd drive start loading the disk since it finally got some speed.

For any scenario, there are different recovery disks.  Some are used by Windows recovery programs and some are discs that run at boot.  Yours is probably the latter, but you could double check.
