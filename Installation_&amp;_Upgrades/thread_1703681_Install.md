### Post by mikewhatever on 2011-03-10
You could try installing from the external hard drive, but keep in mind, that the program that prepares the installation media will wipe everything on that hard drive.
Personally, I think it best to invest in a cheap 1-2GB flash stick and use it to install Ubuntu.
I'd also recommend testing before installing, and should you ever need to use Windows, do make sure you have a recovery strategy before replacing it with Ubuntu.
