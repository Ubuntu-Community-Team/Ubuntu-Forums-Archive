### Post by cariboo on 2009-02-27
Linux does not use drive letters, so if you are trying to mount D:\ it won't work. You should see the drive in Places-->Home Folder in the panel on the left, if you don't have a proper partition label it will show up as the drive size. Just double click the drive and a window will pop up asking you to enter the password you set when you installed Ubuntu.

You don't really need an anti virus program for Ubuntu, as of right now there are no linux viruses in the wild. The only reason I could see for using an anti virus program is if you are running a mail server. Linux is not Windows most of what you know about Windows doesn't work here. Most viruses are written for Windows, and windows executables don't run on Linux with out using some sort of software layer like Wine.
