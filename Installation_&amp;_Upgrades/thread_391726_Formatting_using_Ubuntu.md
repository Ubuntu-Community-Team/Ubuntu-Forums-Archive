### Post by ffxr on 2007-03-23
you are given the option to format, during the install process using the live cd.., its quite similar to the windows install in that regards, if you have experience of that..

you'll need to create a small (256-512mb) linux swap partition as well as your main one for ubuntu.. 

another thought: if you havent got another pc handy, it might be an idea to hold onto a windows option on the pc.. until your sure the install will complete ok..
