### Post by TigerCR1200 on 2009-08-08
The lack of directions for the install seems to be a short comings of Linden Lab not linux it self. 

However from where you are at you want to choose Extract, what you have is a compressed archive. Much like a zip file you would have seen on windows. When you press extract choose your home directory (this should be the default and show your username). Once you have it extracted locate the file README-linux.txt this will give your further instructions on what needs to be done. 

> From the readme:
The Second Life Linux client entirely runs out of the directory you have
unpacked it into - no installation step is required.

Run ./secondlife from the installation directory to start Second Life.
