### Post by HankB on 2017-09-13
That's pretty frustrating! It sounds like some kind of bug in the power management. I've seen that but don't recall how I resolved it. One thing I can think of is to go into the BIOS and see if there are options for power management. In the Intel world they're signified by S3, S5 and so on with S5 being a deeper sleep. I would try disabling deeper sleep. There might be other BIOS options for power management that might affect this.
