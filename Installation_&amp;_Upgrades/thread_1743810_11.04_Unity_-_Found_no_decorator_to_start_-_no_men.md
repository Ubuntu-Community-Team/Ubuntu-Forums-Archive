### Post by TaffyDownUnder on 2011-04-29
I've fixed it thanks to this post:
[http://ubuntuforums.org/showpost.php?p=10735368&postcount=2](http://ubuntuforums.org/showpost.php?p=10735368&postcount=2)

basically in a terminal:
cd ~
rm -rf .gnome .gnome2 .gconf .gconfd .metacity

sudo apt-get remove ubuntu-desktop

sudo apt-get install ubuntu-desktop

shtudown -r 0

Now when I login I have the unity interface.

### Post by wayra on 2011-04-30
If you're like me and came from kde with compiz, you might have to remove compiz completely to reset. Figured that out while checking out profiles in compiz (there was no "unity" profile in ccsm).

This is what did it for me (with nvidia-current drivers):

sudo apt-get remove compiz*

sudo apt-get install ubuntu-desktop

unity --reset

You might have to reinstall compizconfig-settings-manager as well..
