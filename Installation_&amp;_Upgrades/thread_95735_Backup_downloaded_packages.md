### Post by mark_g on 2005-11-27
Is it possible to backup the packages I've downloaded with Synaptic to a second hard drive, erase and reinstall Ubuntu and then reinstall the backed up packages, so I don't have to connect to the internet?

---

### Post by Xian on 2005-11-27
The packages you've downloaded are in /var/cache/apt/archives. You can back up that directory and then replace it for use on another installation.
