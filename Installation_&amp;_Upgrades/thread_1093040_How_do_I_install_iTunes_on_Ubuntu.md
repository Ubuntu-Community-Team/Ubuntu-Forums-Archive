### Post by mocoloco on 2009-03-11
Not trying to annoy but new users often unintentionally ask the wrong questions; one I've heard is "how do I install Windows Media Player?" because a user wants to view an avi file.  There are 3 things for ipod users coming over to Ubuntu: connecting the hardware, getting new files, and using any files you currently have.  Only the last one is of any complication.

1) Using the hardware
Your ipod will work great with Ubuntu.  Just plug it in and you'll be asked if you want to open Rythmbox.  You'll see it on the left hand side and you'll be able to copy music to and from your ipod.  My wife has an ipod, and happily uses it with her Ubuntu system.

2) Getting new music files
What you won't be able to do is buy music from itunes; Apple has left Linux users out in the cold on that one.  There are however lots of alternatives places to buy music; one I like is [Amazon's MP3 service]("http://www.amazon.com/MP3-Music-Download/b?ie=UTF8&node=163856011"), and they [directly support Ubuntu]("http://www.amazon.com/gp/dmusic/help/amd.html/ref=sv_dmusic_3").  
Of course ripping MP3s from CDs or getting them from free downloads etc will work fine.

3) Using your existing music/video files
So this is where your desire to run itunes becomes relevant. If you have existing DRM itunes files you won't be able to play them on Ubuntu without converting them to another format, or you have the option to try and get iTunes running under Wine or Crossover, but from what I hear it's difficult.  As my wife never bought music from Apple I don't have any experience with these two options but you can search these forums or google around on converting itunes DRM files or installing itunes under wine.

Hope that clears some things up.
