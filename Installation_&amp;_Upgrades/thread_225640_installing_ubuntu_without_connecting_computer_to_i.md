if you have a dvd burner you might want to download the dvd iso instead, it has all the packages from the main universe & multiverse repos. if you cant use the dvd just get the install cd, and install ubuntu then go into synaptic and select all the extra packages you want, then click on file **generate package download script** and take it to a computer running linux with internet access and run it and it should download the packages for you to put on cd to install later.
