### Post by luisromangz on 2008-01-08
You can use real partitions as hard disks for VMWare, but Windows isn't able to boot from there.

But have in mind that even if that were possible, you wouldn't be able to run your games using the virtual machine, as it doesn't provide 3D hardware acceleration of the kind need for these games.
