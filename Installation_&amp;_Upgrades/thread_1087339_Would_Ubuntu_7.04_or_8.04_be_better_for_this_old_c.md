### Post by snowpine on 2009-03-05
I do not recommend installing an older version of Ubuntu under any circumstances. 7.04 is no longer supported (no updates, bug fixes, security patches, repositories, etc) and 7.10's support will end next month.

A better choice is to install a distro that is under current development, but designed specifically for older hardware. If you're looking for something based on Ubuntu, Crunchbang 8.04 or 8.10 would be a good choice, but my #1 recommendation for that hardware would be Puppy linux. Good luck!
