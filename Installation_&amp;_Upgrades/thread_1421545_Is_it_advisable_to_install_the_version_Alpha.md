### Post by darkod on 2010-03-04
Not as a main OS. If the user will use another OS for the main work, you can have 10.04 Alpha to check it out.

Otherwise it's not really recommended. During the development cycle, even things that worked earlier can break down in an Alpha release. And they can work again in the final release.
