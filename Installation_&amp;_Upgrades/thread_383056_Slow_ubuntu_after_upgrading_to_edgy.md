I used the instructions from the [HOWTO: Install Edgy 6.10 from an .iso file (hd-media)]("http://ubuntuforums.org/showthread.php?t=316093") thread to upgrade my dapper installation to edgy. The installation itself took an inordinate amount of time so I'm leaning towards blaming the installation method. Any suggestions?
