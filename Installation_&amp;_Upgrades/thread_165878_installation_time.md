### Post by Ensnared on 2006-04-25
Depends on your hardware really. Slow computer = slow install ;)

I think it took roughly an hour on my Athlon 2600+ with 1GB RAM from I power on the computer and until I can log in for the first time on a full desktop install. It's been a while though so I can't really say for sure.

It takes longer to update the system and configure it to your preference, and that will also depend a lot on the speed of your internet connection.

### Post by Jagot on 2006-04-25
I have a Dell Inspiron 1300: 1.5 GHz Celeron-M, 256MB Ram.

I tend to reinstall quite a lot because I experiment quite a bit and it generally takes about 45 mins for a default install, then add another 15 mins to download updates.

If you're talking about a base install in terms of no GUI, like a server install, then it takes me about half an hour to get from inserting the install CD to the CLI.
