### Post by dstew on 2008-03-05
> I get the Xorg looking the exact same way as the posts do and it wont workMany of the things in xorg.conf may be specific to the machine it was created on. Using the xorg.conf seen in another post may not work. Usually, people are trying to point out just one or two items in the file that might help, but the more changes you make to xorg.conf, the more likley that it will break.

Instead of editing xorg.conf directly, have you tried ```
sudo dpkg-reconfigure xserver-xorg
```This will re-create an xorg.conf file tailored to your hardware. It will give you a choice of video drivers to install. If you see the name of your restricted driver there, you can select it. If not, try to install the package first```
sudo apt-get update	
sudo apt-get install linux-restricted-modules-generic restricted-manager
sudo apt-get install xorg-driver-fglrx
sudo depmod -a

```
