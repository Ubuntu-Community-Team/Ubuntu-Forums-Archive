### Post by ajgreeny on 2014-12-15
I am not sure I fully understand what you did, nor where you have got to at the moment.
The ubuntu server edition does not have any desktop environment, ie, no GUI to use; if you don't add one after installing, it only has a command-line interface. Why did you choose the server edition if you are not able to use the command line?

However, assuming you can login to the command line and you want to end up with a standard Ubuntu desktop with the unity DE, you can try running the command ```
sudo apt-get install ubuntu-desktop
```
