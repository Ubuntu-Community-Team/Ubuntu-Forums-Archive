### Post by Frogs Hair on 2010-12-04

### Post by Quackers on 2010-12-04
If you go to System > Admin > Software sources and then to the updates tab and look at the bottom right in the Release Upgrade box does it say Long Term Support Releases only? If it does click on it and select Normal releases and run Update Manager again.
