### Post by mikewhatever on 2009-10-06
The text based installer is usable, but not as user friendly as the regular Ubuntu installer. If you are not familiar with it, but want to try, I suggest practicing first in virtual box. The easiest partitioning option is to use the entire hdd. Here is a good site to illustrate the text installer in action-> [http://members.iinet.net/~herman546/index.html](http://members.iinet.net/~herman546/index.html).
