### Post by emmerc on 2008-09-08
I have a five years old Toshiba Satellite 6100 which CD-rom unit fried so I could not install from it. This machine does not reboots from usb units and I had not a network available to do an installation from it so this is what I did and you might do too:

1. Take your hard drive out of your machine and set it on a similar machine. It has not to be identical. I used an Acer Laptop.

2. Install ubuntu on it until the machine restarts on its own

3. Edit /etc/X11/xorg.conf to fit the other machine in video chip and screen then turn off the machine

4. Take your hard drive back to its original machine, start it and enjoy your freedom! 

I have done this at least two times so I know it really works! This is something I tried three times with WinXp and it was not possible.
