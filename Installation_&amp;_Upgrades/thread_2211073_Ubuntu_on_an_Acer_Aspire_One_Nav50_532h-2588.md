### Post by ajgreeny on 2014-03-14
I think you'll do a lot better with either Xubuntu or even Lubuntu which are both OSs with lower resource needs than Ubuntu with its unity desktop.

My personal vote would be for Xubuntu, which I use on a netbook with Atom cpu, 160GB HDD and 1GB ram, the same as that one, and which is fine; not the speediest in the world but very usable.

Xubuntu 14.04 is released in another 4 or 5 weeks and could be worth trying now as it already seems very stable, but if you are not comfortable with sorting out any problems that might still appear, go for 12.04 for now and then update later to the new 14.04 once any bugs have been overcome.
