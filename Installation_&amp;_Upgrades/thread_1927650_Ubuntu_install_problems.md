### Post by darkod on 2012-02-18
Not showing any partitions is usually a sign of raid meta data existing on the disk. Have you used this disk in raid before? Are you running raid now?

If you ARE NOT running raid, try to remove any meta data remains by booting into live mode, and executing in terminal:
sudo dmraid -E -r /dev/sda

DO NOT do this if you are running raid, it will mess up the array.

On another note, I would boot windows and remove the wubi install. If the meta data was the problem, you should have no problem installing proper dual boot after you remove the meta data. And a proper dual boot is always a better solution than wubi.
