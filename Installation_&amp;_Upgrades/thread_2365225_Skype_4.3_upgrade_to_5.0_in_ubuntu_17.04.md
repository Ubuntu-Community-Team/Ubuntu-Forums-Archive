### Post by ajgreeny on 2017-07-04
You do not, and can not, upgrade from skype-4.3 to skype-5.3; they are differently named packages and can both be on the same system without a problem.

However, skype-4.3 is about to stop working, or may already have done so, so you will need to download the new debian/ubuntu skype-for-Linux version from [https://go.skype.com/linux.deb](https://go.skype.com/linux.deb) and install that, most easily done by first installing gdebi.  Once that is installed you will see that a repository for updates of the package has been added to your software-sources and updates will happen automatically as for all other packages.
