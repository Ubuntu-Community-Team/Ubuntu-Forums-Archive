### Post by Partyboi2 on 2008-02-26
you can install Azureus from the ubuntu repo's  by making sure you have universe repo enabled. Software Sources (System.Admin>Software Sources) first tab tick all except source code if not already ticked.
Then in a terminal (Applications>Accessories>Terminal) 
```
sudo apt-get install azureus
``` this will install version 2.5.0 though. If you want the lastest download then go [here ]("http://azureus.sourceforge.net/howto_linux.php")for how to.
