> **Tattletale1 said:**
> Does anyone have info on how to upgrade to the latest version of Ubuntu from Ubuntu 8.10?  My Uncle Mihai just sent me an upgrade disk...

Welcome to the Forum!

Remember to BACK UP all your important DATA first. Run the CD/DVD your uncle has sent you and see if it is latest 11.10 or ask your uncle what version it is. If it is 11.10 then you don't have to download the latest version of ubuntu. Proceed with the disk you have.

If the disk your uncle has sent you is not 11.10 then you better **download the latest version of Ubuntu **and make a bootable LiveCD or LiveUSB as instructed [**here**]("http://www.ubuntu.com/download/ubuntu/download")  and do a Fresh install only. 


Have fun.
