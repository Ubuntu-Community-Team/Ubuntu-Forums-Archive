### Post by Mark Phelps on 2010-09-25
If you're asking, does the DVD provide more stuff to install (i.e., apps) than the CD, the answer is no.

The DVD comes with language packs.  As far as I know, the CD does not.

You still have to install apps AFTER installation by downloading them from the internet using Ubuntu options.  They don't come with the DVD.
