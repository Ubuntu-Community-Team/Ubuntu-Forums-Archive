### Post by gsrcrxsi on 2008-01-23
so im having an issue with the installer. first the specs

asus p5k3 MB
intel q6600
2 GB G.Skill DDR3 1066MHz
3x500 GB WD SATA
1x320 GB WD IDE

so my original plan was to install mythbuntu 7.10 as im using this computer for a mythtv server and a virtual machine to run windows (vmware) i plan to use the 320 GB drive for the OS and the 3 500 GB drives for storage (probably add another drive and raid it). the bios sees my drive no problem so i figured i was good to go. so i popped in the mythbuntu 7.10 disc and booted up. get to the partitioner and it only shows my 500 GB drives. hmm. so i go back and pop in the windows xp disc and a kubuntu 7.04 disc i had lying around to see if either of them will see the drive, and they both did. so i figured it might be a bug with the mythbuntu disc. so i downloaded the ubuntu 7.10 disc and the same thing happens. is this a known problem? does anyone know why its doing this? any help would be appreciated :)

### Post by gsrcrxsi on 2008-01-23
well it seems ive solved the problem lol. i had the jumper on my HDD in the master position (which is still technically correct) but i removed the jumper altogether and everything is fine now. installing as we speak.
