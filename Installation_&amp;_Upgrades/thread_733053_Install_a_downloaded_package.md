### Post by RealPSL on 2008-03-23
The link you provided is for the source code. Installing from source would involve extracting the files from the source which I am not sure you are inclined to do. The same site does provide precompiled binaries and I have provided that link below. If you choose to use the link below you will be prompted to install it automatically by firefox using the GDebi installer. Some people prefer installing from source because you can choose your own options which is one of the beauties of open source. Others like pre-compiled packages because it is a little easier and saves time but you sacrifice the peace of mind of knowing that you performed the operation yourself.

[http://launchpadlibrarian.net/6716221/pptp-linux_1.7.0-2ubuntu2_i386.deb](http://launchpadlibrarian.net/6716221/pptp-linux_1.7.0-2ubuntu2_i386.deb)
