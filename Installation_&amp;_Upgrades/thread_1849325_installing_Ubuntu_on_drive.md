### Post by 2F4U on 2011-09-24
Well, I don't know what software you installed to dual boot, but in general, doing a fresh installation of Ubuntu will try to replace the existing boot loader with Ubuntu's grub boot loader. This may cause problems. I guess it would correctly detect Windows, but I am not sure about the flash drive installation.
