### Post by sylvanogre on 2010-10-21
Tried a clean install of 10.10 (Maverick Meerkat) on  Optiplex GX240 with a USB keyboard.  Install would almost complete (mouse and keyboard to answer prompts during installation), then the installer would crash, or the squash.fs file system couldnt be mounted, or the kernel would panic, etc.  The error wasnt the same each time, and I tried several times.

The solution?  use a PS/2 keyboard!  (Not the best solution for this particular setup, but it will have to do, wont it!)

Dell Optiplex GX240
Pentium4 1.5GHz
512MB RAM
20.5 GB Hard Drive (erase and use all available space for installation)
Dell PS/2 mouse
Dell USB keyboard (model SK-8115 (install fails, see text above)
IBM PS/2 keyboard model KB-8923 (install succeeds, see text above)
Dell 17" monitor (using resolution detected during installation)
ubuntu-10.10-desktop-i386.iso burned to 4x CD at 2x speed
default install:  no commandline switches, no updates, no 3rd party options
