### Post by Declan Moriarty on 2008-07-06
After upgrading to Ubuntu 8.04 I login and the screen disappears and some dos text appears and the login screen comes up again.  When you login again everything is normal.  What is causing this and are there solutions to this?

---

### Post by Declan Moriarty on 2008-07-07
I have solved this one.  In the sessions menu of the login window (Using the plain option in order to increase accessibility! - Ha Ha Ha!!) I was able to set the session to GNOME rather than "Last".  This solved it.  Evidently the system forgot the type of desktop it had used before the upgrade so was confused.  It works now without having to re-login each time.
