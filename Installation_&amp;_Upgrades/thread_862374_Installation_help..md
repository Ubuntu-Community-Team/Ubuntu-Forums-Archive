### Post by Gallienus on 2008-07-17
It sounds to me like your laptop isn't currently set to boot up from a CD.

You can get into your computer's BIOS (Basic Input Output System) by pressing a key just after you switch on. The actual key varies -- for example when I switch on my computer, I see the message 'Press DEL to run Setup'.

When you have access to your BIOS settings, try to find something called 'Boot Sequence' or something similar and make sure that your computer will boot up from a CD first.

Hope this helps... :)
