### Post by Mark Phelps on 2010-10-12
It's generally easier to install MS Windows first because Ubuntu will detect that later, and even if it doesn't, it's a simple matter to start Ubuntu, run a terminal command, and update the boot menu to include the MS Windows OS.

To add it second, if you are unable to install from a Win7 DVD, you will have to make a Win7 USB installer.

I have no idea how to do that using Ubuntu, so you will need that expertise from someone who does know how.

IF you have an MS Windows machine, there are ways to make an Win7 USB installer using that.  In that case, suggest you go to sevenforums.com (a Win7 forum) as they have tutorials that will walk you through that process.
