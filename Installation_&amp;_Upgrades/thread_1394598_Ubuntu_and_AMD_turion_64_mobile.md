### Post by Revolutionary101 on 2010-01-30
Go to [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) and click on "Alternative download options, including Ubuntu installer for Windows" then select 64 bit and that will download the correct ISO for you.

That ISO should install Ubuntu on your laptop.
