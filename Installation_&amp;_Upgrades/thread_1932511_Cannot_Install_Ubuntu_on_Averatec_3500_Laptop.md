### Post by Paddy Landau on 2012-02-28
According to the specs, you have only 512Mb RAM. That is not enough for Ubuntu.

If you wish to stay with the Ubuntu family, try [Lubuntu]("https://wiki.ubuntu.com/Lubuntu"), designed for lower specification machines. If you are happy to move away from Ubuntu, there are a few other distros such as [Bodhi Linux]("http://bodhilinux.com/") and [Crunchbang]("http://crunchbanglinux.org/").
