### Post by ddave01 on 2007-10-24
I have just finished the version upgrade using Adept and setting the Repositories as required.

The upgrade worked very well on two  nvidia based amd 64Bit machines.

Two issues need attention:

**OPENOFFICE**
two packages were not installed and needed to be manually install openoffice.org and openoffice.org-base.  Installing these packages did NOT make openoffice useable.  It was necessary to remove ( i renamed it)  /home/user/.openoffice directory.  On restarting openoffice all is well.

**FIREFOX**
Same issues as above. Solved by  removing  (i renamed it)  /home/user/.mozilla-firefox directory. And letting firefox create a new directory.

On the whole the version upgrade was flawless and a very good experience.
