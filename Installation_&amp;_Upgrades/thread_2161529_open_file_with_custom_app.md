### Post by sudodus on 2013-07-11
In a terminal window you should be able to run

```
 program file
```

where 'program' could be ***blender*** and 'file' could be**[FONT=courier new] file1.blend[/FONT]**

Depending on the file browser you can create custom choices via right-clicking or the pull-down menus.
