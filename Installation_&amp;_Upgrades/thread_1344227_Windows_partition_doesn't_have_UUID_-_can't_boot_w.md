I hope you now realize that LVM is a part of Linux which could stand for Linux Volume Manager, rather than Logical Volume Manager. Its not a good idea to put other operating systems partitions in LVM as they wont be able to see them.

Sounds very bad, when I used to know about windows it always needed re installing, I hear its better these days. My opinion is a re install is needed for windows.


On a laptop you should be careful to plan your partitions layout and you should also not allow two operating systems to use it in two different ways, ie only use a shared partition in a format windows and linux like, say fat32, or maybe the DVD file system.


You should always know the layout of partitions its so easy to mess up with data loss if done wrong, especially with powerful tools like LVM its possible you would not notice for a long time.
