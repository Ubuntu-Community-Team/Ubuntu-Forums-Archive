### Post by spcwingo on 2010-01-04
You can try:

```
sudo apt-get install --reinstall gdm
```

then:

```
sudo dpkg-reconfigure --phigh xserver-xorg
```

finally:

```
sudo reboot
```

Hopefully that should at least get you a working xserver/gdm.

If that doesn't work, have you tried the command:

```
startx
```

after you login on the console?  Good luck and let me know how it turns out either way.
