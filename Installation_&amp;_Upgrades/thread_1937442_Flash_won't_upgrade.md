### Post by winh8r on 2012-03-09
Check to see if you can see the moving image and text here:

[http://kb2.adobe.com/cps/155/tn_15507.html]("http://kb2.adobe.com/cps/155/tn_15507.html")

and try viewing a video on youtube.

If these are working then you might need to clear your browser cache.

First though try to access the game by bypassing the cache as follows:

get to the loading screen where it tells you you need to update flash, then press ```
control + shift + R 
```(this will reload the page but without using the cache)

Sometimes just pressing ```
Control + F5 
```does the same thing.

Try these and if the game loads then you will need to clear you browser cache by going to```
 Edit > Preferences > Privacy
``` and select the "clear all current history" option. This will allow you to select what you want to clear.

Hope this helps.
