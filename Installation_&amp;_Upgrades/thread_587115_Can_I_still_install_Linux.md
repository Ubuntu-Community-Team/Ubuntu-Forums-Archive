### Post by Pandemic187 on 2007-10-22
Normally I would just have the regular old version of Windows installed on my laptop, but due to problems I was having with my DVD-RAM drive, HP has insisted I format my system and put their software on my hard drive before they will send me a replacement DVD-RAM drive. Because of this, I have three primary partitions on my hard drive, all for Windows. I tried to install Linux, but I discovered that only four primary partitions can be present on one hard drive, which means I cannot install Linux since it needs two partitions. Is there any way I can still install Linux?

---

### Post by dabl on 2007-10-22
Make the fourth primary partition an "Extended" partition type, and then make your 2 or 3 logical partitions for Linux within that. :)
