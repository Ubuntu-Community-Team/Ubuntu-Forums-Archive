### Post by Cannonade on 2007-03-08
I have also have a WD 250GB mybook and have installed ubuntu onto it.

Before you install ubuntu to it, make sure you eject the drive first before you continue. The big green circular light on the hard disc will not go off. This worked for me, I had the same problem initially.

I knew to do this from when I was attempting to install Sabayon, VERY unsuccessfully!

You might encounter other problems with grub when booting from this disc. If you do, please read my thread and post any answers if you have them!

[http://www.ubuntuforums.org/showthread.php?t=378450&highlight=error+18+250GB](http://www.ubuntuforums.org/showthread.php?t=378450&highlight=error+18+250GB)
