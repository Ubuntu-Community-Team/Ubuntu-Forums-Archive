Simply copy paste the whole /home folder onto an external drive, all your file should be there. For your Firefox Bookmarks, simply go to Bookmarks > Organize Bookmarks > Import and Backup (in Firefox).

Make sure that when the copy of your /home directory is done, that you can 'CTRL-h' (show hidden files) and see some folders beginning with a dot . (IE  .wine, .ssh  etc)

You should then have a good backup of your files.

Hope this helps
