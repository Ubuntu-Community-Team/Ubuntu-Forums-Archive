### Post by guiverc on 2022-04-29
[Ubuntu 22.04 LTS has been released]("https://fridge.ubuntu.com/2022/04/21/ubuntu-22-04-lts-jammy-jellyfish-released/") and is thus available for new installs.

The release announcement I provided a link for contains the text

> Users of Ubuntu 21.10 will soon be offered an automatic upgrade to  22.04. Users of 20.04 LTS will be offered the automatic upgrade when  22.04.1 LTS is released, which is scheduled for the 4th of August. For  further information about upgrading, see:

If you're after the date of Ubuntu 22.04.1 LTS' release, you can view the [release schedule]("https://discourse.ubuntu.com/t/jammy-jellyfish-release-schedule/23906/2") though do note the ISO release (*which is what the date really represents*) is for new installs of the product, the Ubuntu Release team usually open the *taps* for upgrades the following week if all looks good, or delay a week or too as necessary.

If you want more *mechanical understanding* of what occurs; an answer I [wrote here]("https://askubuntu.com/questions/1403610/22-04-is-suggested-on-ubuntus-website-but-not-in-the-repository/1403615#1403615") maybe helpful  (*that answer was written awhile ago now, and the meta file-contents may have changed and thus I'd **may **have used different wording if I wrote it today*).
