### Post by vanadium on 2017-11-03
To change the appearance of the top bar, you need to install and apply a shell theme. Before you can do that, however, you need to install and activate the "User themes" extension. When that extension is active, the dropdown next to "Shell" will allow you select a shell theme if any is installed.
