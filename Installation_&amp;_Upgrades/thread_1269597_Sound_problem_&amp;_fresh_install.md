### Post by stlsaint on 2009-09-18
a few options...1) do a backup of your /hom(u can use simple config tool suite out of synap) then fresh install and restore /home.
2) un-install vbox and see if that fixes sound...or try to boot into restore mode and repair broken packages.
3) explain your sound problem more in depth for troubleshooting here.

FYI...during installation you can choose to manually select your partition setup and at the partitioner screen you can make /home its own partition...that way later on should you choose to do a fresh install again...you will only have to format root(/) and install system files there but you wont have to touch your /home partition leaving all files, music, pics intact.
