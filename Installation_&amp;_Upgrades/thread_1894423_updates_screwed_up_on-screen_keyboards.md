### Post by twiistedkaos on 2011-12-12
I just updated to 11.10, I have a faulty hardware keyboard on this laptop so I am forced to use a onscreen keyboard for certain keys. After my update I've been noticing all onscreen keyboards have been acting glitchy. At first I thought it may have just been onboard but it isn't, it's all virtual keyboards. 

Here's what's happening. I click on a key with the mouse "sticks" as if it's using massive amounts of cpu and instead of it clicking the key once it's "holding" it for a random amount of time spitting out what should be "n" into "nnnnn"

Now you can see how annoying it would be to type in a forum post with this happening. I've tried all other virtual keyboards that are in the software manager and they ALL are doing this, which did not happen in 11.04

EDIT: It only seems to do it when I click with the touchpad, when clicking with the button this "lag" effect does not happen

EDIT2: It seems after reseating my touchpad connector everything is working fine. Odd coincidence.
