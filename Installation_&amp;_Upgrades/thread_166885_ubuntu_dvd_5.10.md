### Post by Jagot on 2006-04-27
It won't come with any compilers.  Ubuntu uses apt-get from the command line and also Synaptic as a graphical front-end.

To get the compilers you will need to do, from the command line:

sudo apt-get installer build-essential

You will also need to download any IDE's you want.

Unfortunately I think that downloading is the only way.
