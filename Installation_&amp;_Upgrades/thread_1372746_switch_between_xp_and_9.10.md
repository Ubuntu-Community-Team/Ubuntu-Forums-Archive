NO if they each are installed to a partition of their own. But if you run XP in virtual box from Ubuntu you will not have to reboot to run xp. But of course there are limitations. You will need enough RAM to run both at once. At minimum I would say 1 to 1.5 GB. You will be asked when setting up virtualbox how much RAM to allot to it.
