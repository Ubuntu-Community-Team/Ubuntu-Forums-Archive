### Post by logos34 on 2008-02-23
Grub is a bootloader, it doesn't do partitioning or installing.  You can create a grub boot floppy for an existing linux installation, but that's another story.

To install ubuntu from windows without a working cdrom, try [this howto]("https://wiki.ubuntu.com/Installation/FromWindows") (-->UNetbootin or manual netboot method, or the 'cd image' approach)
