### Post by CelticWarrior on 2020-10-31
There's some misunderstanding here.

Native resolution = the highest resolution the device is capable of reproducing. If it *[COLOR=#000000]keeps defaulting to 4k (3840x2160) screen resolution[/COLOR]*[COLOR=#000000] and you can see what's going on and not a blank/black screen, an "out of range"/"not supported" message or similar than that IS its native resolution.

You may want a lower resolution but that doesn't make it the native one. AFAIK it can't be changed for the server installer.[/COLOR]
