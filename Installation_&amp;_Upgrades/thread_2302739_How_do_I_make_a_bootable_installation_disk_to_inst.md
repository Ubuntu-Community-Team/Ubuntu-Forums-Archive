I am a bit confused. If there is no OS on the computer how were you able to download the Ubuntu ISO image? what old version of Ubuntu are you using?

The way I do it is to put a DVD disc in the DVD drive or USB memory stick in a USB port, then I load the file manager. I browse to the ISO image and select it, then I right click the ISO image and I select write to disk. A dialog will appear asking us to confirm the action. And the file manager does the rest.
