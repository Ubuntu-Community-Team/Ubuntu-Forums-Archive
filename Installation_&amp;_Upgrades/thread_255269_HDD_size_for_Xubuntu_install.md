### Post by go2dell on 2006-09-11
From the Xubuntu Dapper Release Notes:

> Recommended Minimum Requirements

To run the Desktop CD at least 128 megabytes of RAM is required. To use the installed system at least 64 megabytes of RAM is required but 128 is recommended. At least 1.4 gigabytes of disk space is required. 

You can read the rest here:
[https://wiki.ubuntu.com/XubuntuDapperReleaseNotes]("https://wiki.ubuntu.com/XubuntuDapperReleaseNotes").

### Post by go2dell on 2006-09-11
I've been running Xubuntu on a notebook with 64MB RAM and a 6GB HDD.  Slow, but not unbearably so.  The cost of upgrading to more is literally more than the value of the computer.

A 4.3GB drive will be plenty of room.  If you can boost to 128MB RAM, you'll probably be happiest, but 96MB would be fine if your needs are modest.  I would suggest using the Alternate Install (text-based) just to speed things up at installation time.
