### Post by ~!geek!~ on 2011-08-07
If you already tried booting from the media (cd or usbdrive) containing windows xp, There appears to be a problem with this media itself. 
If you are sure that it was not the case, you may try formatting the harddisk using gparted using a live cd or live usb containing ubuntu.
Now try again installing windows xp. 
If you are able to install the windose, you may go for ubuntu and install it alongside to get dual boot system.
