Wubi install (ubuntu inside windows) or ...  did you install Ubuntu in its' own partition.

If in it's own partition, you can edit /boot/grub/menu.lst to reflect your preference or, use synaptic to search for and download startupmanager.  In SUM (which is GUI-based and also writes tot he menu.lst) you can set defaults there.  

I have never used SUM for a wubi-install so be careful.

Another option is to press the space bar to stop the timer.
