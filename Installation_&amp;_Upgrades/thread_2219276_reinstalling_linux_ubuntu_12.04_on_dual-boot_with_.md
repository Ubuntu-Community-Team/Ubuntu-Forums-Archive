### Post by oldfred on 2014-04-23
I know several with Windows 8 have had issues with any of the auto install options. Windows 8 may revert to its always on hibernation or fast boot.
But if it just says Ubuntu and does not mention Windows do not use any auto install options.

Better to use Something Else and choose the same / (root) partition as you currently have. Same for /home if a separate partition, but do not check the format box.

You really should have a full image backup of Windows and a Windows repair Cd or flash drive.

       The vendor recovery DVDs are just an image of your drive as purchased. If you have housecleaned a lot of cruft normally included, run many updates with many reboots, and added software you may want a full back up.
