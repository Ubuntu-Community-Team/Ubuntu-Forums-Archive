### Post by oldfred on 2010-08-27
You do not have a bootable windows. You have a windows in sda6 but that is a logical partition. Did you delete another copy of windows?

Windows will only install in a logical partition if you have another install of windows in a primary partition and it then moves the boot files to the primary partition. You cannot normally boot a copy of windows in the extended. Windows will not let you repair it as it will not see it.

Easiest way is to copy your data and reformat to make a primary partition and reinstall. If you have to keep it there is a bit risky way to edit partition table directly converting sda6 to sda3 since it is at the end of the extended and you have another primary available.

Caljohnsmith using sfdisk to edit partition table from text file
[http://ubuntuforums.org/showthread.php?t=1036600](http://ubuntuforums.org/showthread.php?t=1036600)
