### Post by triplemaya on 2009-02-10
Solved

Like a previous problem, this resolved itself when I rebooted, so presumably the changes were not fully effective until then.

Further digging showed that it was the nvidia driver which caused the problem.  I was trying to get a geforce2 card to work with intrepid and there seem to be known problems with this combination - in the end I resorted to a new card which was geforce6 and everything displayed correctly.
