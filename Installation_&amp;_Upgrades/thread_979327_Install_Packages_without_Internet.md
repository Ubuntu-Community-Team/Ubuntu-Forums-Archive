### Post by Partyboi2 on 2008-11-12
.exe are windows file extension, ubuntu does not run them natively.  Look online for .deb packages for the programs you want to install, then download them and double click on them to install. Since you don't have internet you could use the Synaptic "[[COLOR=Blue]Generate Package Download Script[/COLOR]]("https://help.ubuntu.com/community/Synaptic/PackageDownloadScript")" or use [[COLOR=Blue]keryx[/COLOR]]("http://keryx.betaserver.org/") to install packages.
