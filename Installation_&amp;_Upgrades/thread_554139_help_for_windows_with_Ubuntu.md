### Post by mikeize on 2007-09-18
You can try a program called WUBI.  I use it on my laptop.  You will have to choose windows from the grub menu, otherwise I believe it will default to Ubuntu, just because of the order.  I don't know if you can change that or not.
Wubi is nice because you don't have to partition your disk-- it makes the linux 'partition' as just a file within windows.  Easy install, easy uninstall.  I've had minimal problems with it.  Especially just for testing (u)buntu, you should look into it.
