### Post by Rogue21121987 on 2005-10-20
Yes, it can be done easily.  First, you need to make sure that there is nothing on your second hard drive that you want to keep.  If there is, back it up - the easiest way to install onto a new hard drive is to wipe it clean.  Next, put the Ubuntu CD into your CD drive and boot your computer.  You should see an installer.  I can't remember the exact details of the installer (I'm at school), But it should be fairly straight forward.  Just remember that you want to install Ubuntu on hdb and put the GRUB boot loader on the Boot Record of hda.  The installer is pretty good, it picked up XP without a problem on my machine, and it will ask prompt you for these at some stage, I just can't remember at what stage it asks.  Just remember to back up everything on you second hard drive.  Give it a try, and if you have any problems I'll try to help.
