### Post by bummm on 2009-07-05
I was just wondering how to install software system-wide in Ubuntu?  So installed software will be available for all users; current users & future users. Will this be achieved by installing while logged into the root account?

---

### Post by taurus on 2009-07-05
Just use either System -> Administration -> Synaptic Package Manager or Applications -> Add/Remove.  No need to log in as root although you need to be able to use sudo/gksudo.
