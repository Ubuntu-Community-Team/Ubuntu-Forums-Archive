### Post by aysiu on 2007-11-22
A content management system is a complex system set up to allow anyone to add, edit, and delete content on a website--kind of like a blog, but more fully featured.

A webpage editor allows you to edit the full code of a webpage.

If it has to be point-and-click, try Kompozer:
[https://help.ubuntu.com/community/Kompozer](https://help.ubuntu.com/community/Kompozer)
