### Post by sub2007 on 2008-01-19
I'm pretty sure that Windows will replace your MBR. You should let it do that and then after doing that you will need to replace it with Grub again.

Instructions are here: [https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows](https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows)

I've used Super Grub Disk before after (I assume) Windows decided to break my GRUB and nothing was booting. It was very easy and steps you through it nicely.
