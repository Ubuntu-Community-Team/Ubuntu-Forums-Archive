### Post by Mark Phelps on 2010-03-30
> **laflair13 said:**
> So basically, I want to remove this kubuntu and put windows 7 back on my system so I can get all my window programs back.

Depends on what Win7 media you have for doing the reinstall.

If your machine came preinstalled with Win7, you will have either a recovery partition (if you didn't erase it installing Kubuntu) and/or a Restore CD -- provided by the PC vendor.

IF you restore using the vendor-provided stuff, you will put the machine back into the state is was when you first used it -- including any SW or other apps that the vendor preinstalled.

If you erased the recovery partition, or if you only have a restore CD that then expects to find a recovery partition, you will have to contact the vendor to get full restore media.  The restore CD will not be enough to completely restore the machine.

IF you have a Win7 Installation DVD, you should boot from an Ubuntu LiveCD first and remove all the partitions on the drive, and then boot from the Win7 DVD and just do an install.  But realize, this will NOT get your "windows programs back".  You will have to reinstall all of them from scratch.
