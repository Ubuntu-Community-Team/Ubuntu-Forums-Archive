### Post by Xian on 2005-09-24
[QUOTE=tehwa]I'm basically stuck at the point that I need to install Breezy to the root partition, although this would cause conflicts with my Hoary install (?).[/QUOTE]
As long as the Breezy root partition is not in the same location as your Hoary install then there should be no problems. Just be sure to actually create a new partition for Breezy from the free space on your HD during the installation by selecting the manual partitioning option.
