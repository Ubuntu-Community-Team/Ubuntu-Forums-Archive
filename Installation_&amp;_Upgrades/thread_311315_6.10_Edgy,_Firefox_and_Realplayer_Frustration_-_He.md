### Post by Dominicus on 2006-12-02
Well, I'm searching deep in my patience repository, but alas it's time to vent....here goes...

Fresh install of Edgy and want to hear Realplayer audio news feeds from [http://www.NPR.org](http://www.NPR.org)

Went Applications/Add-Remove... Realplayer not there :( 

Went to wiki/docs/unofficial Edgy install guide, added the Edgy commercial repository from Canonical (supposed to include Realplayer10)....after refresh, still no 'Realplay' entry found in Synaptic Package Manager.

Tried from command line:
sudo aptitude install realplay
->"package not found"!

Went with method#2 from wiki..download from Real website .bin file directly. Followed instructions to install from terminal window.  Bunch of files extract, populating my desktop, etc but installer later hangs without any Realplayer enabled.

Hours and lots sifting through several methods posted in ubuntuforums.  Finally came accross this one:

From terminal window type:
```
wget -c http://www.debian-multimedia.org/pool/main/r/realplay/realplayer_10.0.7-0.0_i386.deb

sudo dpkg -i realplayer_10.0.7-0.0_i386.deb
```

This installed Realplayer....wow happiness until I realized Firefox would not play .ram or other Realplayer audio/video streams from the browser.

Firefox keeps asking me what application it should use with these types of files.  The only option in pull-down was Media Player.  Realplayer not a listed option.  I could, however, copy the stream URL and paste into the Realplayer application to play.

Back to forums...endless talk about renaming plugin files nphelix.so, or making them executable (tried that to no avail).

I went to System/Preferences/Menu_Layout, clicked on "Sound and Video" section, then right-clicked on Realplayer10 icon and selected "Properties"  There I learned the directory tree this link follows to launch Realplayer.  Next time Firefox asked what application to use with .ram files, clicked "Browse..." and added "/usr/bin/Realplay"

Now whenever I click, the Realplayer application is launched and plays the stream. The download window also opens.  In some sites I get yet another browser window with ads/sponsor images...etc.  Less than desirable behaviour, and quite a frustrating experience with Ubuntu Edgy trying unsuccessfully to install a BASIC application/plugin.

Thank you soooo much for allowing me to VENT my frustration.  Now my question:[B]

How in the world can you enable Realplayer plugin in Firefox!???? [/B]](*,)
