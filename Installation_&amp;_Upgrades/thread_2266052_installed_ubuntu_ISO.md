### Post by sudodus on 2015-02-21
Clonezilla works in virtual machines too :-)

But it needs a target drive (virtual drive) of the same or larger size as the source drive, which might be a problem for you.

You can also use the One Button Installer, make a tarball of your tweaked system and install systems from that tarball.

[https://help.ubuntu.com/community/OBI](https://help.ubuntu.com/community/OBI)
