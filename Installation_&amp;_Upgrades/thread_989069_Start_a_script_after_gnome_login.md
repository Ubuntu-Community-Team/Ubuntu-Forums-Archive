### Post by Afkpuz on 2008-11-22
I don't know how to run a script that needs root privileges, but here's how to execute a regular bash script at each login.  You can save it anywhere really.

System>Preferences>Sessions

Click add, then point to your script.  Simple as that.  You can also use this to open programs at start up.
