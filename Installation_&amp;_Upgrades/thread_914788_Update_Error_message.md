### Post by Rocket2DMn on 2008-09-09
You have some entries for Edgy in your sources.list file which need to be commented out or removed.  Since Edgy is no longer supported, the repositories no longer exist.  You can place a # in front of each of those lines by editing your sources.list file
```
gksudo gedit /etc/apt/sources.list
```
Alternatively uncheck such entries from System->Administration->Software Sources (check each tab).
When you are finished, you need to update
```
sudo apt-get update
```
^ Software Sources will ask you to refresh, this is the same thing.
