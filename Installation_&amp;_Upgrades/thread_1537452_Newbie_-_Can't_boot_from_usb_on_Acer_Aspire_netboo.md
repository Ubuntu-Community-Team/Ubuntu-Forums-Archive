### Post by 00b00nt00 on 2010-07-23
Creating a bootable USB stick is easy-peasy within Ubuntu itself, which is rather a Catch-22 situation! However, if you can burn a live CD of Ubuntu and then run the live CD on another PC, you can use startup disk creator to make a bootable USB stick for your Aspire.
