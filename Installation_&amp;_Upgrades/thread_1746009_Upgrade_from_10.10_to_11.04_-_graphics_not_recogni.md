### Post by johnnybgoodzm on 2011-05-03
This has been solved by an update so most likely was a bug in the driver/gnome - I ran update manager - there were some gnome updates and a intel driver that looking promising - so into grub recovery mode - run X -configure copied the file to /etc/X11 and the problem is solved
