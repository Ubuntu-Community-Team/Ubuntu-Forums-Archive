### Post by Dennis N on 2009-12-22
XMMS is no longer available from the Ubuntu repository. The most similar program in the repository is audacious, which derives from XMMS. I use it and like it. Use the refugee skin to replace the default; also most classic WinAmp skins will work with it.  

Some people have installed XMMS from another source and got it working, but it requires some work:
