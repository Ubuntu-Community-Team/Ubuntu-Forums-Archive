### Post by forkandles on 2008-11-29
What is the content of your  /etc/hosts file?

For username max you should have something like this:


127.0.0.1	localhost
127.0.1.1	max-desktop

# The following lines are desirable for IPv6 capable hosts
::1     ip6-localhost ip6-loopback
fe00::0 ip6-localnet
ff00::0 ip6-mcastprefix
ff02::1 ip6-allnodes
ff02::2 ip6-allrouters
ff02::3 ip6-allhosts


You obviously have internet access. 
Do you have a firewall enabled such as guarddog, which may be incorrectly setup?

I assume that you are using Update Manager in 8.10?
Try using Synaptic Package Manager instead and see if the problem persists.

Also have a look at the last post on this thread.
[http://ubuntuforums.org/showthread.php?t=994051](http://ubuntuforums.org/showthread.php?t=994051)

Make sure all your repositories are enabled in Synaptic:
Settings > Repositories >  Ubuntu Software 
                Ditto              >  Third Party Software
                 Ditto             >  Updates
                 Ditto             >  Authentication  (3 keys should be entered there)

Click on Reload after making any changes.
