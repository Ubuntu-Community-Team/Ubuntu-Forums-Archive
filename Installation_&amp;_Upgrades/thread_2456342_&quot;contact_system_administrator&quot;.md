### Post by Impavidus on 2021-01-10
I don't see what you mean by your first and second sentence. But you can try running the upgrade again and hope it works better this time. If something doesn't work, we need precise error messages.

Usually, however, it's better to try a fresh install if a release upgrade isn't working well. Making a live dvd of 20.04 and starting fresh will be the fastest way out of your problem. Or even better and more modern, use a live usb, as they are faster, reusable and work on newer computers without dvd drive, unless your computer is so old that it doesn't boot from usb.

### Post by mastablasta on 2021-01-12
you can force install the upgrade.
 if it fails, you can download 20.04 and install that without formatting the disk. that should come close to an upgrade. 
if that fails, you can backup your files in home (actually you might want to do that before any upgrade attempt) and format the drive. 

but before going with this nuclear option you could also boot from drive and delete system files and hidden files in home, then install without format. i think that could also work. if ii creates new home, you just move the data files from old home to the new one.
