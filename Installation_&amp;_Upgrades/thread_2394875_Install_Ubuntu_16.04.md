### Post by yancek on 2018-06-22
There should be no problem burning an iso of Ubuntu 16.04 to a DVD from Ubuntu 14.04.  Use brasero or whatever other CD/DVD burning software you have.  Make sure you use the option to burn as an image and not just copy.  You should be able to upgrade directly form 14.04 to 16.04 if you want.

### Post by KBD47 on 2018-06-22
Yes, just download the Ubuntu 16.04 iso file and burn the image to your dvd using brasero or xfburn or whatever image burning app you like.
