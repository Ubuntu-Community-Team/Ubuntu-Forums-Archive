### Post by peterpan7191 on 2010-10-19
Plymouth freeze problem fixed

After rebooting into recovery mode and modifying the following file

"/etc/X11/xorg.conf"

and editing the line 
 Driver        "nvidia"
to 
 Driver        "nv"             (This loads the default drivers instead of nvidia drivers)

The system did boot further.

Later install nvidia latest graphic drivers from the nvidia website  ------- [http://www.nvidia.com/Download/index.aspx?lang=en-us](http://www.nvidia.com/Download/index.aspx?lang=en-us)

Things are fine.
