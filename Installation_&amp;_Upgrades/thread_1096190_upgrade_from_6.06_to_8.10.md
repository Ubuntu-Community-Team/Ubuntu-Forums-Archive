### Post by tommcd on 2009-03-14
> **bour1992 said:**
> 
Can you tell me how can I replace the 6.06 version with the 8.10?

You will need to do a clean install of 8.10 if you want the newest version of Ubuntu. 
Since 6.06 was a LTS (long term support) version, it is possible to do a dist-upgrade to the next LTS version, which is 8.04, and is still supported. See this:
[https://help.ubuntu.com/community/HardyUpgrades](https://help.ubuntu.com/community/HardyUpgrades)
