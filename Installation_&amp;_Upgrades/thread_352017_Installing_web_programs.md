### Post by Zimmer on 2007-02-02
[https://help.ubuntu.com/community/InstallingSoftware](https://help.ubuntu.com/community/InstallingSoftware)

Read that Very Carefully...

The best (and safest)  way to install software is from the Official Repositories Via Synaptic Package Manager..in Ubuntu, I believe Adept is the tool in Kubuntu (see the  link above)

Yes, you can download .deb  (Debian packages) files and install them 'by hand' but you may end up breaking the system.... so as you are new, best advice, leave it alone for the moment. I quote from the above link...
> Sometimes you might want to install a package which you have downloaded from a website, rather than from a software repository. These packages are called .deb files. Because they may have been created for a different Linux distribution, they may have unmet dependencies on Ubuntu and so may not be installable.
Synaptic package Manager provides a graphical interface for installing software, 
Or simply  Add/Remove programs from the K-Menu...
I believe the Opera browser may be available from a Universe Repositotry, so look up How to add more repositories if you cannot find it.
