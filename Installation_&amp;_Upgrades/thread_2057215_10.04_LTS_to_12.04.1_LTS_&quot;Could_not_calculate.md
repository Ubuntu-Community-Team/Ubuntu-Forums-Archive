### Post by plucky on 2012-09-13
> * Unofficial software packages not provided by Ubuntu

Do you have any third party PPA's or repositories that are still enabled?


e.g.**Hit [http://debian.slimdevices.com](http://debian.slimdevices.com) stable/main Packages**

Try disabling all the third party repositories and then also run ```
sudo apt-get dist-upgrade
``` to see if there are any updates still to be installed and not held back.

Then try the "update-manager -d" again.

Good Luck

### Post by Bucky Ball on 2012-09-13
You don't have the headroom? Guessing the partition you have 10.04 LTS on is almost full. Upgrade needs space to download and store a lot of stuff which is later deleted. If you have 10.04 on a 15Gb partition and are using 13Gb, say, you have little chance. I could be wrong but that's a stab. Had something similar awhile back with same message.

Best to uncheck everything except the main tab. Doubt that is the issue though.
