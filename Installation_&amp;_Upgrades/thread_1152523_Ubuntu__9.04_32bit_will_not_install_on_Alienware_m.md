### Post by HyRax on 2009-05-07
The LiveCD does not always agree with all systems out there, and interestingly it primarily affects brand name machines.

Download the Alternate install CD instead. This uses a far simpler (and faster) text-based install mode that will work on the most obscure of PC hardware.
