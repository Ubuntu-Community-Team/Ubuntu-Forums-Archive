### Post by lechien73 on 2010-07-20
You don't have to delete the Windows partition first, just boot from the Ubuntu CD and, when the installer prompts you, select to "Erase and use the entire disk", as in the attached screenshot.

You can usually accept the defaults for the size of the main partition and the size of the swap.
