### Post by Barticus on 2007-12-05
PLEASE NOTE: THESE INSTRUCTIONS ARE FOR A WIPE AND REINSTALL ONLY, ISSUING THIS COMMAND CAN CAUSE YOUR PC TO STOP BOOTING FROM THE HARD DRIVE. YOU HAVE BEEN WARNED.

If your laptop has a floppy drive, download a Windows 98SE boot diskette from this site, if you don't already have one;

[http://www.bootdisk.com/](http://www.bootdisk.com/)

It's been a long time, but I believe that you can also run this command from a 98 boot CD, if  you have one.

Then boot from the diskette.  If you have more than one hard drive (if it's a laptop, I doubt it, but for the sake of clarity......) make sure you are seeing the drive you wish to re-initialize.  Once that is done, from the C:\ prompt type FDISK /MBR.  This will reset your hard drive's Master Boot Record.  You should now be able to boot from your XP install CD and do a complete re-installation.
