### Post by jfl on 2008-01-11
[FONT="Comic Sans MS"]I have been running Dapper on one of the office machines for over a year.
A few days ago, being curious I downloaded 7.04 and burned a Live CD (at 24X).
Worked good, decided to install.
However, that box HAS TO WORK everyday of the week, so I decided to create a /home partition and a another ext3 to install 7.04, keeping the 6.06 untouched.
Having bad memories of upgrading from Breezy to Dapper, I was utterly surprised (and pleased), less than an hour later to have Feisty running, network, network printers, sound working perfect !!!
So, I figure "why not try Gutsy" it is so easy.
I download the file, burn the CD ... don't even get to the first screen.
Burn another CD slower (4X) ... same result.
So I did what I should have done first, "checksum" the file; didn't pass.
Download again, checksum OK, burn at 4X.
Fire it up get Gutsy running from the live CD ... reboot to install ... install hangs at 37% ... reboot ... same result !
*(Stupidity is doing the same thing over and expect different results)[/FONT]*
Reboot and use the Check CD in the boot screen: 3 bad files !
Reburn the CD ... check again: 1 bad file !!!
Burn at 2X ... flawless install, everything is good. Been running Gutsy for 2 days, nobody complaining,
All this to show that the install from the Live CD is not difficult if everything is checked step by step; install will also hang on a bad hard drive, it happened to me a year ago the drive crashed a few days later.
