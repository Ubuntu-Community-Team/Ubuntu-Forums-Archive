### Post by tiggsy on 2009-10-19
OK. For anyone else with this problem.

Reboot in terminal: 

```
sudo reboot
```

Then when it starts up again, 

```
sudo dpkg --configure -a
sudo apt-get dist-upgrade
```

and for me (after numerous previous attempts) worked.

Be careful if it asks you for a multiple choice decision NOT to choose "start another shell to examine the situation" or something like that.
