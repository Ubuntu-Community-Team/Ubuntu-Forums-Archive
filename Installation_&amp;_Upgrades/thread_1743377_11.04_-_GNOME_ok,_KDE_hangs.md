This works:
1. Start your kubuntu system and switch to any text console, for example <Ctrl><Alt><F1> and log in
2. Install ubuntu-desktop
```
sudo apt-get install ubuntu-desktop
```
3. Reboot and start Gnome session ("Ubuntu classic")
3. Go to "System Settings" in the "System" menu
4. Disable desktop effects
5. Logout and start KDE session

### Post by dalenkruse on 2011-04-30
There's a way to do this from the console without having to install ubuntu-desktop.  When your KDE session hangs, type <Ctrl>+<Alt>+F1.  This will get you to a console.

Log in, then type:
```
vi ~/.kde/share/config/kwinrc
```

In the [Compositing] section, change
```
Enabled=true
```
to
```
Enabled=false
```

If the Enabled statement isn't there, add
```
Enabled=false
```
to the [Compositing] section.

Then, restart.
