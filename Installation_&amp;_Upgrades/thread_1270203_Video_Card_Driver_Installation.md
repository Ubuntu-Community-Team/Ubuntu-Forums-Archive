### Post by Storm Rider on 2009-09-19
Why not just do System - Administration - Hardware Driver , and install it from there.  It's much easier and will guarantee you don't have to reinstall the Nvidia drivers after a kernel update.

---

### Post by Storm Rider on 2009-09-19
If you really want to use the downloaded file.  Open a terminal "Applications - Accesories - Terminal", change to the directory the file is saved in with the cd /home/yourname/directory-name, then run the installation file.

This is the hard way of doing things if your new.
