### Post by lykwydchykyn on 2010-02-01
If you just want to get files off the computer, and you have something to copy them onto (like a USB drive or network share), you don't need to install Ubuntu.  You can just mount the old Windows partition from the desktop on the live CD and copy them to wherever.

See this:
[http://www.howtogeek.com/howto/windows-vista/use-ubuntu-live-cd-to-backup-files-from-your-dead-windows-computer/](http://www.howtogeek.com/howto/windows-vista/use-ubuntu-live-cd-to-backup-files-from-your-dead-windows-computer/)

If do want to install, see this:

[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)
