### Post by ibutho on 2009-02-04
The best solution in my opinion is to install the ubuntu-restricted-extras package. This install flash and other proprietary plugins. To do this, run Synaptic or the Add/Remove program, enable all the repositories, update the sources, search for ubuntu-restricted-extras and select it for installation.
