Good luck with it, it's a fine idea.
A couple of things that might help:
The ultimate reference is probably the [Debian installer manual]("http://www.us.debian.org/releases/stable/i386/index.html.en"), particularly sections 6.3.2.5 and 7.2. You might not want to use [the older ubuntu desktop directions]("https://help.ubuntu.com/community/EncryptedFilesystemLVMOnFeistyAndDapper"), but they might help you understand how the pieces fit together.
