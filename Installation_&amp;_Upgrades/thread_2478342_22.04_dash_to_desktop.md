### Post by col48 on 2022-08-25
This link gives the core of the solution:
[URL="https://linuxconfig.org/how-to-create-desktop-shortcut-launcher-on-ubuntu-22-04-jammy-jellyfish-linux"]

For anything in /usr/share/applications, drag a copy from there using the file manager into the Desktop folder.  It will then immediately appear on the Desktop.
Then right-click on the desktop icon and click on Allow Launching.  The appearance and the caption of the desktop icon will change and clicking on it will then launch the program.
