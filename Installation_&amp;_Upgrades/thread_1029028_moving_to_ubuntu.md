### Post by crwmike on 2009-01-03
You can have both KDE and GNOME on the same install. To add GNOME to your existing Kubuntu, enter in the terminal...
```
sudo apt-get install ubuntu-desktop
```
Select your session at the login screen.
Your KDE apps will still be there.

If you are going to do a fresh install of Ubuntu, add the KDE apps by entering...```
sudo apt-get install kubuntu-desktop
```

I have Ubuntu on my laptop and run Kate often, I was able to start Kmail and K3b on my machine.(running GNOME).
