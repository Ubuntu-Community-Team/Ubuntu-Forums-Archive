### Post by MattBD on 2008-08-25
You'd have to get the separate Alternate installer CD (unless you have a DVD copy of Ubuntu, which would include that). This can be downloaded from the main Ubuntu website.
If that doesn't work, I'd probably try another distro or two. It may be that your computer simply doesn't play well with Ubuntu, in which case you might be better off with another distro. I recommend Linux Mint, PCLinuxOS, SimplyMEPIS or Mandriva as good alternatives. I'd also recommend you try the Knoppix live CD, it has some of the best hardware detection around so if any distro will work with it, it'll be Knoppix.
