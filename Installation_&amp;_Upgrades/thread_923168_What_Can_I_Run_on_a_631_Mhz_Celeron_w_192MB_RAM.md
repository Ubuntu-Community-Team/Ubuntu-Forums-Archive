### Post by robert shearer on 2008-09-18
Stick with Puppy (Dingo 4), Xubuntu might just hobble along (you will need to use the Alternate Install cd) but Ubuntu will be so slow as to be unusable(if it installs at all).

More Ram would make a big difference.

 384Mb plus to run Ubuntu and 512Mb would be good. 

An old machine like this probably takes sdram which can be got from our local Recycle centre here in New Zealand for $2 a 128Mb stick.(big boxes full of it!) so go look for some where you are.


From the Xubuntu download site is this comment....... Minimum system requirements

To run the Desktop CD (LiveCD + Install CD), you need 128 MB RAM to run or 192 MB RAM to install. The Alternate Install CD only requires you to have 64 MB RAM.

To install Xubuntu, you need 1.5 GB of free space on your hard disk.

Once installed, Xubuntu can run with 192 MB RAM, but it is strongly recommended to have at least 256 MB RAM.
