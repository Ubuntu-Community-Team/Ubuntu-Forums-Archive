### Post by rsambuca on 2007-11-15
> **Kouki said:**
> It's been doing it for over 24 hours now, so I think its an issue with my computer.  Someone would have fixed the server by now, whereas I've been at work and unable to fix mine.

How would I go about switching mirrors?

Go to "System -> Administration -> Software Sources".

On the first tab, click the "Download from:" button and select "Other".

Then press the button for "Select Best Server".

It will test every mirror and pick the fastest current one to you.
