### Post by TBABill on 2012-05-01
There are no longer normally any issues with 64 bit. 64 bit, on hardware capable and powerful enough to run it, has been tested to be faster than 32 bit. Phoronix tested 32 bit, 32 bit PAE and 64 bit to compare them side by side...64 bit won nearly across the board.
 
I would recommend you test it via live session before deciding, but with your hardware I would start with 64 bit because it should operate smoothly on that machine.
