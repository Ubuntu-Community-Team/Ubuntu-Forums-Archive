### Post by kleeman on 2005-06-25
Assuming you have the 386 kernel add the following lines to /boot/grub/grub.conf in Fedora 4 (where the similar lines are for the Fedora 4 kernel)


title           Ubuntu, kernel 2.6.10-5-386
root            (hd0,8)
kernel          /boot/vmlinuz-2.6.10-5-386 root=/dev/hda9 ro quiet splash
initrd          /boot/initrd.img-2.6.10-5-386
savedefault
boot

### Post by kleeman on 2005-06-25
OK everything looks OK in principle. So try the following: The line
 kernel /boot/vmlinuz-2.6.10-5-386 root=/dev/hda9 ro quiet splash
in /boot/grub/grub.conf
remove the splash and quiet options i.e. replace with
kernel /boot/vmlinuz-2.6.10-5-386 root=/dev/hda9 ro 

Second remove the lines:


savedefault
boot.

following this (they are absent in the Fedora boot)
