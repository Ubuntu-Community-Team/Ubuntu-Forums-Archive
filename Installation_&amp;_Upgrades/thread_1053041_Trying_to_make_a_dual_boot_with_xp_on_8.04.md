### Post by Therion on 2009-01-28
Is this a SATA hard drive you're talking about?  If so, and if your XP install disc is pre-Service Pack 1 (meaning SP-1 has not been included on the CD), you're going to run into issues with XP not recognizing a SATA hard drive.  It wasn't until SP-1 that SATA drivers became native to Windows XP.
