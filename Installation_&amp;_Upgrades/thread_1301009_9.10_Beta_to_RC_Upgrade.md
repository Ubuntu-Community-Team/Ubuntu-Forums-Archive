### Post by Cybie257 on 2009-10-25
The best way to do so is to go into a terminal
Applications-->Accessories-->Terminal
and type the following: 

1st : sudo aptitude update

2nd : sudo aptitude safe-upgrade

This will 'safely' upgrade your system and is recommended over using the update manager during Alpha/Beta/RC periods. 

There really is no reason to to it via ISO images, unless the computer is not connected to the Internet. Is this the case?
