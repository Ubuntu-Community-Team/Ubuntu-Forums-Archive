### Post by merlinus on 2007-08-01
Did you burn the cd as a disk image or a data disk?  Look to see the contents of the cd in windows -- if only one file, then it is the latter.

It needs to be burned at no more than 4x speed as a disk image.  Your burner software should have this option.

-merlin
