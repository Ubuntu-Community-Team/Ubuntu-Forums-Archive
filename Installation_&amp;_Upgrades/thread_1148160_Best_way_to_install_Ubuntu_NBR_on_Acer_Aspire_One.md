### Post by udippel on 2009-05-04
'Yes' is the answer to your questions. Put it on a stick, chose the first option, look at it if you like it. Then do the install by selecting the second options directly after start. It is a great success here, replacing the linpus. Yes, you can as well chose the layout between desktop and NBR.
Here (A110-8.9") everythings works okay, almost everything. On Linpus, mplayer was working fine, here it crashes badly. Also, taking videos with the webcam is close to impossible (1.5 frames per second). Lastly, NBR has a slightly higher consumption, so the battery lasts a tad less long.
