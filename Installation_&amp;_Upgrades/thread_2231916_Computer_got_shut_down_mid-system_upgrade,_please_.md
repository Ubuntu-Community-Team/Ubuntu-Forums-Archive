### Post by Vladlenin5000 on 2014-06-28
Hi, welcome.

Unfortunately both 12.10 and 13.04 are EoL. Not supported. Please read [http://ubuntuforums.org/showthread.php?t=2229730](http://ubuntuforums.org/showthread.php?t=2229730)
My suggestion is to boot from a current version - 14.04 - liveUSB or liveDVD, access the internal file system, backup everything you need (if you haven't done that already and yes, you definitely should backup whenever you try any risky operation) and then perform a fresh installation.
