### Post by confused57 on 2006-05-04
That should be plenty of disk space, you may want to do a search on partitioning to see how others have partitioned their hd's.
I had win98SE, which ran without reinstalling for about 5 years, and was crashing frequently.  You might want to reinstall win98SE, or if you have data you don't want to lose, just do a repair from the install disk.  You probably already know all of this; but thought I'd mention it.   

I don't want to assume anything, windows needs to be already installed before installing Breezy for dual-booting and Breezy can do all the necessary partitioning, as suggested by the link Sef gave you.  You partitioning will depend a lot on how much data you expect to be storing in the shared partition.
