### Post by cuberts on 2010-04-26
Hi, so I figured that I would help people today, by teaching you how to install google earth on Ubuntu.

Now it is a bit more challenging that just installing it on Windows, because on Linux, google earth is not default.
I decided to teach this because I thought that a lot of people might like google earth, because I do-especially the newest version.

Steps:
1. Open up a terminal
(Applications>accesories>terminal)
2. Type this code into the terminal
```
sudo apt-get install googleearth-package
```
3. After that is done, then you will have to type this code in the terminal
```
make-googleearth-package --force
```
4. Now when you do that it will give you a bunch of warnings, so just ignore those. 
5. When you have all of that, that will have unpacked the package, then you can go to your home file (places>home folder>)
6. Find the file with the box icon, which should say google earth.
7. From there it is pretty straight forward- just simply install it, and have fun!
