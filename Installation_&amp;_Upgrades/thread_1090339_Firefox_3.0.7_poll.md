### Post by Rallg on 2009-03-08
Here on the Ubuntu forums, when 3.0.7 was first available as a download, several users reported problems. The complaints varied from how Flash worked (or didn't on certain sites, to problems with the FF window appearance, to outright inability to launch FF due to supposed dependency problems.

However, there were not as many complaints as one would expect if 3.0.7 had a general problem. After waiting a day, I installed 3.0.7 without any difficulty. No site I visit has shown a problem, but then I do not visit many sites that might be popular among other users.

When FF updates, it checks add-ons for compatibility. Perhaps some add-ons passed the test, but should have failed? I use Adblock, Flashblock, UnMHT, and Ubuntu Firefox Modifications, all without difficulty. Flash and Java work. 32-bit system.
