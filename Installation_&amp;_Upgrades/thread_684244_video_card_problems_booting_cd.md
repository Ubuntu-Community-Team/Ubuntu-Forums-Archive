### Post by Rocket2DMn on 2008-02-01
A lot of people have problems getting the LiveCD to work due to video problems, so there is an alternate install cd which is just a text install without a live session, but it works very well.  You can download it at [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) by checking the box near the bottom for Alternate cd.  Once you install, if you have video problems, you can troubleshoot them rather easily at that time.
