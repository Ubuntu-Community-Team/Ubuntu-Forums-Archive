### Post by cariboo on 2009-04-23
You shouldn't have any problems with the upgrade. The only problem I can see is if you upgrade the file system to Ext4, you third party boot-manager may not have support for it yet.

---

### Post by Synthros on 2009-04-23
I agree with Cariboo.  It's been a long time since I've tried upgrading on a dual-boot system, but last time I did, I was able to get it done with no problems.  If I remember correctly, I was running SUSE and Windows XP, using Norton Boot Magic.  Performing the upgrade has no adverse effects on the Boot Magic configuration (and things are a lot more sophisticated now than they were at that time).  So, you *should* be okay.
