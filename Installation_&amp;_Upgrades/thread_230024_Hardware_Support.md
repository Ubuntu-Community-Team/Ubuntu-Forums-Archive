### Post by sitedesign on 2006-08-05
Your system probably won't show up there unless someone else has one and verifies it. However since Dell use differing hardware in the same model numbers it is difficult to know if it will be fully compatible.

The best thing to do is get a copy of the latest Ubuntu which is a live CD system, boot up and try it. If you like it then install it as a dual boot system to start with so that you can swap back to Windows if you have problems.

Then once you have got comfortable then reduce the size of the Windows partition to say 5GB and just stay in Ubuntu. You may as well leave the windows there since you have paid for it (pre-installed).

Hope that helps
