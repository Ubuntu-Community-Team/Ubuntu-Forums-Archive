### Post by kondor on 2005-07-27
When the install screen first appears, there is a menu that lists various Fkeys (F1, F2 etc.) that will show possible options to pass to the install. Read them all carefully BEFORE you hit "enter". 

You can enter "expert" install by typing "expert" at the initial screen prompt and hitting "enter".

Understand that you will have to know how to define your partitions and other various things concerning the Linux setup. If you want to learn Linux, that is a good way to begin. Jump in and break the box. You'll have to study - read the Man pages and various study papers on the Net. Just do a Google on "Linux instruction" or Linux. You shouldn't have any trouble.

Yes, the StartCom distro RAAM is multimedia oriented. I have it in but haven't had time to play with it yet. The dvd ISO is 2.1 Gigs.

Good luck!
