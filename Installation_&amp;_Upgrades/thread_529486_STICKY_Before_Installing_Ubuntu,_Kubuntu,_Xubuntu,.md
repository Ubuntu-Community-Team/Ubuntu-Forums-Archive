**steps before trying *buntu**
Update your BIOS to the newest version you can (which maybe years old,even when you get the last version)
Do a memory check
check your harddrive
get rid of any old linux partitions with delpart
Check system requirements and get the *buntu that your comp meets
Download *buntu and verify md5checksum
burn cd at slow rate 2x-8x (slower the better)
before install do the cd check that comes on the disk and make sure it passes
install
(write down any errors you get)
Try new cd rom media and repeat slow burn and cd checks, try another cdburning software like infrarecorder, try another cd-burning drive

If none of that works search through the bug list
[https://bugs.launchpad.net/ubuntu/](https://bugs.launchpad.net/ubuntu/)
