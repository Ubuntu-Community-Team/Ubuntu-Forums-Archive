### Post by oldfred on 2023-08-31
What brand/model system?
If less than 10 years old, it is UEFI. Vendors still call UEFI as BIOS in many cases.

If older than 10 years, you need to be installing a lightweight flavor as Ubuntu is really for newer systems with more resources.
What version of Ubuntu?
[https://ubuntu.com/download/flavours](https://ubuntu.com/download/flavours)
Light weight flavors:
Lubuntu, xubuntu, Ubuntu MATE, Budgie
