### Post by louieb on 2007-03-20
> **sethra_lavode said:**
> I have a 30 GB hard drive and 256 RAM ... if I keep both XP and Ubuntu on my laptop, will that slow it down significantly?
XP may slow down a bit.  It depends on how full the XP partition is. The native XP file system NTFS has to work harder if it get above 70-75% full.
> Should I create a partition just to make sure my files are safe?If you savvy enough go ahead and create a 500MB partition and format it as a Linux swap partition. The Live CD will find it and use it automatically. This will speed Linux up a bit.    
> Or should I back up all of my files first?Just popping in a live CD and booting to it will not touch your hard drive.  
Most of the time when data loss occurs its during the resizing of a partition to make room for Linux. Yes by all means back it up before starting a hard drive install.
