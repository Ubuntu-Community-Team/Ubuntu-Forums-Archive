### Post by magicfab on 2007-05-23
Wikipedia has a list of current 64-bit processor architectures:
[http://en.wikipedia.org/wiki/64_bit#Current_64-bit_microprocessor_architectures](http://en.wikipedia.org/wiki/64_bit#Current_64-bit_microprocessor_architectures)

The Intel Core 2 duo processors (which include the t5600) are part of the Intel64 family, found in that list.

Check this guide to learn about Intel's processor numbers meaning:
[http://www.intel.com/products/processor_number/index.htm](http://www.intel.com/products/processor_number/index.htm)

Sepcifically, this page lists the processor you mention as being Intel64:
[http://www.intel.com/products/processor_number/chart/core2duo.htm](http://www.intel.com/products/processor_number/chart/core2duo.htm)

I would suggestusing the Alternate edition CD, which is mostly guaranteed to work. 

Althoug I haven' t tried these, you may find the following resources helpful to install from USB stick or from Hard Drive, with floppies (maybe buy / borrow an external USB floppy drive ?):
[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)
[https://help.ubuntu.com/community/Installation/FromHardDriveWithFloppies](https://help.ubuntu.com/community/Installation/FromHardDriveWithFloppies)

Good luck! And nice machine BTW :)
