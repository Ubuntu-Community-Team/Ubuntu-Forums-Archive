### Post by ryanskyer on 2009-02-26
Not want to upgrade some packages, have to uncheck each time when updating. Any idea to remove those from the update manager?

---

### Post by Partyboi2 on 2009-02-27
If you don't want to upgrade a package you could lock the current version. Go to Synaptic and search and highlight the package you don't want upgraded and select Package>Lock Version.
