### Post by zvacet on 2010-03-28
> does it imply replacing the older packages with new without reinstalling the entire OS

Yes,that´s how it works.Replacing old packages with new version of package.You can upgrade package(s) or entire Ubuntu version.For example if you upgrade from Karmic to Lucid you are upgrading entire Ubuntu from one version to another.

You can not upgrade from 9.10amd to 9.10 i386 simply because that is not upgrade.That is just switch from one architecture to another.In this case proper name is fresh install,because you will replace amd for i386.
