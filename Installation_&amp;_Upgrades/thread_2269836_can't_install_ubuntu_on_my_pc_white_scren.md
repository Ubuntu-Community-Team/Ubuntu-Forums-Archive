### Post by DarkSapphire on 2015-03-18
I can understand your problem real well, because I had a problem like you had.  

1.  Did you check the md5 sum?  Here is the link to the hashes [https://help.ubuntu.com/community/UbuntuHashes](https://help.ubuntu.com/community/UbuntuHashes) and how to check md5 [https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

2.  What program did you use to install the Ubuntu installer on?  Unetbootin ect.

3.  You probably booted to the USB because of the white screen, but just in case, did you boot to the USB?  Make sure your hard drive is directly after the USB in the boot order.

If you checked the md5, used a installer program, and booted from the USB, do it again!  One time, I had to install Ubuntu ISO to the USB 4 times until it worked!
