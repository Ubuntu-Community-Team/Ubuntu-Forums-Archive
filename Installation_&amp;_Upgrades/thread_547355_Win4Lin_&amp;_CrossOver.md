### Post by sqlpython on 2007-09-09
I have been a CrossOver / WINE user for any of my few Win App needs.
However while using PCBSD I was introduced to Win4Lin. I had heard of it and knew of others who had used it but with mixed reviews. So, because doing four identical Kubuntu 7.04 amd64 installs on Dell Inspiron 1501 Laptops and the users were real Windoze converts I decided to give Win4Lin Pro a try.

 I may yet reinstall CrossOver Pro 5.xx as I have used it for years on Xandros installs.
By comparison Win4Lin runs faster and as it is a virtual enviroment will install and support many many more WinXP apps.

Win4Lin requires three parts
1. A Win4Lin Pro download
2. Updating some C libs per the on CD install Readme
3. A bootable copy of WinXP or Win2000

Installation is real easy and zooms on this amd64 x2 Laptop.
By install I allowed 15 gb disk space and 256 mb ram for the WinXP install.
This puts Windows where it belongs as a secondary OS in a Window on a Linux Desktop.
