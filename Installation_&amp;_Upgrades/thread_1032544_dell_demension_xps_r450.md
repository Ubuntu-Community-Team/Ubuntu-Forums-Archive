### Post by cariboo on 2009-01-06
It looks like you have a defective ram module. Unfortunately Ubuntu v.8.10 will not run on a computer with such low specifications, to run Intrepid, I would suggest a computer than can run Vista, at the least.

If you want to install linux on the computer you mentioned, I would suggset either [PuppyLinux]("http://www.puppylinux.org/"), or [DSL]("http://damnsmalllinux.org/").

I have installed both on the origional pentium processor based computers, and they are actually quite speedy.
