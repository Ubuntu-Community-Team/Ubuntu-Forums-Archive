### Post by kostkon on 2012-04-28
Don't try to upgrade using the ALT+F2 update-manager -d way, it's only used to upgrade to alpha or beta versions, not the final.

Just open the update manager and check for updates. Eventually you'll get the upgrade button, if not today, then tomorrow or in 2-3 days depending on the server.

Also make sure that the option *Show new distribution releases* in your software sources is set to LTS only (or normal) releases and not to never.

Hope that helps.
