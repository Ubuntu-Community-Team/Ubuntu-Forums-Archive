### Post by merlinus on 2007-05-16
From what I could see at the deepburner website, the software can make iso images, which is what you need to do with the .iso ubuntu file that you downloaded.

Make sure that this is what you instruct the software to do.  Then the cd will be bootable, al least on computers that are set up to boot from a cd drive.

Short of that, you might check to see if any friends or acquaintances have cd burning software that will let you do this.

Another option might be to download the alternative cd image, but that will not give you a GUI installer, just command line.

Good luck!

-merlin
