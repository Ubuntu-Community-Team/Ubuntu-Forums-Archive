### Post by kansasnoob on 2010-01-09
It might be interesting to see the output of:

```
sudo apt-get update
```

Just to see if there are any errors. Also:

```
sudo apt-get -f install
```

Just to see what it says, **if it recommends removing stuff I wouldn't do that just yet or you might end up not being able to boot -14 either!**

It might also be helpful to see the output of:

```
cat /etc/apt/sources.list
```

You could also go to Synaptic Package Manager and see if the Broken package filter shows anything.
