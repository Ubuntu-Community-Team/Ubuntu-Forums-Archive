### Post by tcrroadie on 2007-11-10
When using the live cd you will only be able to test video mode settings within the terminal.  

You will also only be able to adjust the screen modes when your PS3 is connected to your TV via HDMI or Component.  

If the TV you are using is a standard definition television (480), you will be disipointed in the screen realestate that is available on that TV.  If using any type of television as a display for the Playstaion 3, I would recommend a dispaly with a resolution no less than 720.  I have my PS3 hooked up to my 42" Plasma Samsung and am loving it.  My Plasma TV is 720.  Typing this post on it right now from my couch.  

Hope this helped.
