### Post by lykeion on 2010-03-24
To run windows games you need to install wine, just search for wine in Add/remove applications.

Wine has a homepage where you can find out if a specific program/game works with wine. 
AOE3 seems to work OK but needs a little fiddling with dlls to get it to work, see:

[http://appdb.winehq.org/objectManager.php?sClass=version&iId=3795](http://appdb.winehq.org/objectManager.php?sClass=version&iId=3795)

I think I have managed to get it working so it shouldn't be too hard, just read the instructions on the WineDB page carefully.
