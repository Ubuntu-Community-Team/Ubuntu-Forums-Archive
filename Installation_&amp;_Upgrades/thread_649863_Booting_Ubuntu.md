### Post by logos34 on 2007-12-25
If grub installed at all it's probably on the mbr of the ubuntu disk (but you're booting to vista drive).  Reboot and enter the Bios (hit F2, Tab, Del whatever key it uses)...change the ubuntu disk to first in the hard disk boot order
