### Post by Bucky Ball on 2010-09-16
Install Ubuntu. Go to Synaptics Package Manager and add the Studio packages (do a search for Ubuntu Studio in there). Studio is Ubuntu with packages for a/v added. Ubuntu Studio ***IS*** Ubuntu with some extras (which is why it comes on DVD, doesn't fit on CD).
