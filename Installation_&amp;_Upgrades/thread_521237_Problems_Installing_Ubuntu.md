### Post by nilfisk1 on 2007-08-09
As far as I know there is no way to install from windows, except on a virtual machine. What I would do is go to the bios and disable your hard drive, just to see what it boots then. If it boots the live cd then it's just a faulty setting somewhere in your bios. If it doesn't boot anything then I would consider reburning the live cd. 

Good luck.

### Post by splintercellguy on 2007-08-09
Go to your BIOS and make sure the boot order is set so that the CD goes first. If that is the case, perhaps you have burned the CD incorrectly? You must burn ISO files as an image, not as a new data session.
