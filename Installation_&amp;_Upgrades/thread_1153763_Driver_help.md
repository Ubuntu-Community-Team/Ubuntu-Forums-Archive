### Post by ad_267 on 2009-05-09
Why would you need a driver for a router? You should be able to just plug a network cable between the router and your laptop and have everything work fine. Or am I missing something here?

If you haven't yet configured your router, you can usually just connect over a web interface to change your router settings.

Most hardware comes with Windows drivers, which are pretty useless for Linux, but then most hardware is supported automatically in Linux without any configuration required so you should be able to just plug things in and have them work.

The only drivers I've had to install are my video card drivers, which aren't installed by default because they're proprietary drivers provided by nvidia. I just had to go to System - Administration - Hardware Drivers to enable them.
