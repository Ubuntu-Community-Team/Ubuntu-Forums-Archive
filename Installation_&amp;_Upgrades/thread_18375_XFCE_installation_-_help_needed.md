### Post by landotter on 2005-03-06
The default Gnome will run fine on such spec, if maybe a little laggy. XFCE will use fewer resources and be snappier, at the expense of some convenience like desktop icons, etc.

I have both on my box.

That IS a tiny drive by today's standards, so if you want to remove gnome and install xfce4, all you have to do is open up Synaptic (Computer/System Configuration/Synaptic) and select gnome for removal, and xfce4 for installation. 

Any reason you can't get a larger drive?
