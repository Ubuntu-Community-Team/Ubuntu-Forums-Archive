### Post by kebes on 2007-10-03
> **Rob500 said:**
> What's Kernel Freeze? :confused:

"[Kernel Freeze]("https://wiki.ubuntu.com/KernelFreeze")" just means that the developer team will stop changing the kernel after that point. The "kernel" is the central piece that runs the entire operating system.

Basically this just means that we are very close to the final release of Gutsy, where almost nothing more is going to be changed before the actual release. The idea is to "freeze" all the code and give testers a week or so to make sure that the system is fully functional and fully stable.

You can see the release schedule here:
[https://wiki.ubuntu.com/GutsyReleaseSchedule](https://wiki.ubuntu.com/GutsyReleaseSchedule)

The final release is October 18th, but assuming that no major bug is found, that release will be nearly identical to the current testing releases.
