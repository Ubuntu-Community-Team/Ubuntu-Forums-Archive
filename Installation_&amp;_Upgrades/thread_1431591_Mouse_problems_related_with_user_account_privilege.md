I installed Ubuntu 9.10 on four x86_64 pcs that were supposed to have the same hardware...

When I drag the mouse as administrator it works fine in all four computers. When I drag the mouse using a user account, it works only in one of the four machines... all other aspects of the mouse work fine. The installation method was basically the same for the four computers...

### Post by jcgp on 2010-03-17
Problem solved: I removed the user account, removed /home/user, created it again with the same name and using the same procedure as before, and this time it works fine...
