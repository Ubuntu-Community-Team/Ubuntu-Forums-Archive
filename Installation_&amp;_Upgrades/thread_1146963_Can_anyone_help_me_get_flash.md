### Post by murrie on 2009-05-03
Yeah,

Really simple.  Download and extract flash for jaunty 64 bit. You should see a file called libflashplayer.so.  If you do good, highlight the file and copy it to the clipboard with control-c.  Next bring up your home folder with nautilus and press control-h to unhide your .files and then go to .mozilla and in that create a plugins folder.  Paste (control-v) the libflashplayer.so into that folder and restart firefox, that's it.
