### Post by SickTwist on 2005-12-10
The automatix script might not be legal in the United States. This might not mean as much to an individual but it does present more concern for a company.

Stick with the default Ubuntu install to keep your company out of fuzzy legal issues. There is even an OEM option for the Breezy installer that does not require a username and password to be set during the install. These questions will be given to the first user to run the machine.
