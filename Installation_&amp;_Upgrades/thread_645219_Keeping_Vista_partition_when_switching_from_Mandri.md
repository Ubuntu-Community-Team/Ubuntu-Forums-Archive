### Post by jken146 on 2007-12-19
You're right about the partitioning: you want to choose the 'manual' option and replace the Mandriva partition with a Ubuntu one.  The swap can be left alone, and so can any partitions with windows or personal files on them.  Just specify a partition for / and tick the box for format.
