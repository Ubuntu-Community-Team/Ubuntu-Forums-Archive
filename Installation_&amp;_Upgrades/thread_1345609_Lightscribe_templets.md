### Post by hal10000 on 2009-12-04
I don't know anything about lightscribe, but in linux you usually have a hidden directory in your home folder which is related to an application you have installed.
Maybe there is a directory called /home/your_user_name/.lightscribe (the directory starts with a dot because it's usually hidden)

If such a directory (or with a similar name) exists, then look into this directory and with a little luck you can find out where you have to copy your templates to.

Maybe you should read the documentation (if any) that came with the package.
