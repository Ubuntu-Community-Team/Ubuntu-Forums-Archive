### Post by mörgæs on 2013-04-02
You will get the Xubuntu desktop corresponding to the Ubuntu you were already running. If you are on Ubuntu 12.04 you will get Xubuntu 12.04 in parallel so both are available when booting.

Remember that the long time support is different for Ubuntu and Xubuntu.
