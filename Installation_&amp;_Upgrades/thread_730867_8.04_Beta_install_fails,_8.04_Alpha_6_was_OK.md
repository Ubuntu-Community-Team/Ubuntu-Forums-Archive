8.04 Beta gets to 94% install then vanishes.  /boot is incomplete, there is no /boot/grub.
I have 8.04 Alpha 6 on another partition.  I copied over /boot/grub to the new Beta partition.  Then copied 
/boot/initrd.img-2.6.24.12-generic.bak
to the same name without the .bak.
Then added a new stanza to the existing GRUB menu.lst.
Now it boots.
Seems OK.  Of course I'm making an assumption that the only thing that failed was the GRUB install routine.  I don't know yet what is else may be broken.  Alpha 6 installed without this problem.
