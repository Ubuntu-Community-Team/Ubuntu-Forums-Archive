### Post by Mark Phelps on 2010-09-21
Before you ask ... I see using Google that your eMachines has a Radeon Xpress 200 video card/chip.  That means that there are NO ATI/AMD drivers that you can download and install that will work with current Ubuntu versions.  You already have the only working driver installed -- the open-source driver.

Just telling you this in case you're planning on looking around for ATI drivers.
