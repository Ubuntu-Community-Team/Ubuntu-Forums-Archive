### Post by AoSteve on 2009-06-28
I would imagine that one of two things happened.  

1:  The install was bum, and something happened during the process that wasn't caught by the software.   Try reinstalling and seeing if this helps.  Also, try 8.04 LTS from the ubuntu site as well.  

2:  A drive partition error or possibly a failure could have occured.  This is usually found by "Disk i/o error"  "System Halted"  which is defaulted by the BIOS when it doesn't find a bootable media.

One last thing to try, make sure that you've burned that ISO to the cd slowly.  4x or 8x usually does the trick.  I've had my fair share of disc failures because of burning at higher speeds.

I hope this helps out bud!
