### Post by shadowlab on 2007-11-10
I currently share a keyboard and mouse between 2 machines:

- machine #1 running XP Pro with RealVNC Server.

- machine #2 running Ubuntu 5.10 with a keyboard & mouse. I use x2vnc to share input with the XP box.


I want to perform a clean install of Ubuntu 7.10 on machine #2 to take advantage of the latest features, but after running the live CD on this box I was unable to locate x2vnc in the Synaptic Package Manager. Is x2vnc no longer supported?

[EDIT]
Answered my own question. Thanks for all the replies :-)

 I was able to find x2vnc after enabling  "Community-maintained Open Source software (universe)" from settings > repositories.
