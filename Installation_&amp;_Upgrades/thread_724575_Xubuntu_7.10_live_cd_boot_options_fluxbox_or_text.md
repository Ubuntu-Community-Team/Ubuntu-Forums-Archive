The Xubuntu Live CD won't have Fluxbox available as an option, it uses XFCE.  You could try Fluxbuntu if you're looking for a  Fluxbox-fronted Ubuntu derivative.

Also, with less than 256 MB of RAM, you'll need the Xubuntu alternate CD to install anything, even if it's just installing a command-line system.  The GUI installer requires at least 256 MB to run.
