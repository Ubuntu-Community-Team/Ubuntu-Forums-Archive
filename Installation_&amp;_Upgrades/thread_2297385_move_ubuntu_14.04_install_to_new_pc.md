### Post by coffeecat on 2015-10-03
Ubuntu 10.04 is long past end-of-life. The desktop version became eol in May 2013, the server version in April this year. There are no longer any updates, and more importantly, no security fixes. That installation is a security risk if you take it on the internet. Rather than bother with the hassle of cloning it to a new pc, you need to make a fresh installation of a currently supported version of Ubuntu. You are probably best off with 14.04.3, the latest LTS (long-term support). The current intermediate release, 15.04, only lasts until January next year, and the next short-term support version, 15.10, won't be released until later this month.

One thing you need to be aware of. You are using the old gnome2 desktop in Ubuntu 10.04 which has now been replaced with the very different Unity desktop in Ubuntu. Some like it; some don't. If you are attached to the old gnome2 layout, you can get something almost the same with the Mate desktop, which you can get by installing the Ubuntu Mate flavour. Do you need some links for downloads?

### Post by howefield on 2015-10-03
> **tmy said:**
> ..I meant to say 14.04..

[Clonezilla]("http://clonezilla.org/") would create an image of the existing disk which can then be cloned to the new pc, assuming that the new hard disk is at least as big as the old one. Clonezilla is run as a live CD or USB, one small drawback is that unfortunately it looks more complicated than it actually is.

There are other (free) solutions that do the same job and may have simpler interfaces. I think I have seen Macrium Reflect Free mentioned in these forums and it does support ext file systems.

Make sure that you have backups of your important data before going through any procedure to move your installation.

Personally, I'd take the opportunity to do a fresh install but that sounds not at all what you want to do :)
