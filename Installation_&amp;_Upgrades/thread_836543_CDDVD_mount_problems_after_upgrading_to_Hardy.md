### Post by denisliber on 2008-06-21
This is an edited version since I found the main reason for freezes:

Hello! I have upgraded to Hardy 2 days ago (from Gutsy)and everything went well... not everything actually :( 
When I login everything is working except the panels, both of the take something like 3 minutes to load. 
During the work if the system mounts/unmounts any device: (CD, DVD, USB-stick, camera, mp3 player) both panels freeze again for about 2 minutes. 
I have created another user account and there are NO freezes, tried to delete all gnome config files to resset -> NO result.

Can someone help me please....

Solved after no responses at all :(
1. Copied all needed configuration directories and files from my home to other location preserving the atributes (mc is a great thing)
2. Deleted my user and purged the directory
3. Created the same user with the same number and copied the config. files back.
4. Ready to go!
