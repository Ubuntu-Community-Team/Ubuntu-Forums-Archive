### Post by mcduck on 2009-06-04
Gimp is included by default, but there are many alternatives available if you want.

UFRaw is a decent RAW editor, available as both standalone program and Gimp plugin. In case you are into HDR photographing Qtpfsqui is a great tool.

Photoshop and Flash are Windows (or OS X) programs and not supposed to run in Linux, although you might have some luck running the Windows versions with Wine.
