### Post by snowpine on 2008-11-04
The SliTaz LiveCD is under 30mb, and I highly recommend it for a lot of reasons. It doesn't support ntfs out of the box, so you need to install the dos filesystem package and re-master the CD. Damn Small Linux (DSL) is another excellent choice.

Here is an excellent list of distros ranked according to the size of the Live CD: [http://ubuntuforums.org/showthread.php?t=575456](http://ubuntuforums.org/showthread.php?t=575456)
