### Post by snowpine on 2009-05-20
Yes; in the Synaptic Package Manager, do a search for "ubuntustudio". You'll see separate "metapackages" for the audio, video, graphics, etc. applications. Install all of the ubuntustudio packages for the full Ubuntu Studio experience. You can also install "linux-rt" if you want a real time kernel for low latency audio production (though not all users find it necessary; I personally found it a little buggy on my hardware).

Good luck!
