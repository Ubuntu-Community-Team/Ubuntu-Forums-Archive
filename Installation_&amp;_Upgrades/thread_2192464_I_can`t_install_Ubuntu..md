### Post by fantab on 2013-12-08
Where did you get Ubuntu CD from? Newer versions of Ubuntu do NOT fit on a CD, is it a DVD?
Download Ubuntu from [Here]("http://www.ubuntu.com/download/desktop"), preferably using [Bittorrent Here]("http://www.ubuntu.com/download/alternative-downloads").
Then run a [MD5Sum check]("https://help.ubuntu.com/community/HowToMD5SUM") on the downloaded ubuntu.iso. (If you have the .iso already then run on it as well, if sums don't match re-download).
Then [BURN]("http://www.ubuntu.com/download/desktop/burn-a-dvd-on-windows") the DVD.

When booting a DVD you must choose the DVD as your first boot device in BIOS/UEFI.

Did Windows came pre-installed your machines?
Dual-Booting with Windows8, especially if its booting in UEFI requires some additional steps on your part... Read more from link below:
[https://help.ubuntu.com/community/UEFI](https://help.ubuntu.com/community/UEFI)
[http://ubuntuforums.org/showthread.php?t=2147295&p=12657352#post12657352](http://ubuntuforums.org/showthread.php?t=2147295&p=12657352#post12657352)

Good Luck...
