### Post by Partyboi2 on 2009-05-01
Hi, you should be able to boot into your Ubuntu install and when you have logged in, insert the alternate cd in the cdrom/dvdrom and a box should pop up with the option to upgrade to Jaunty. If no option to upgrade appears then open a terminal and try
```
gksu "sh /cdrom/cdromupgrade"
```
