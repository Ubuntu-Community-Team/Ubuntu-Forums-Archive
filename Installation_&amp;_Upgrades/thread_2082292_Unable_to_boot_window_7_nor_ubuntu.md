### Post by oldfred on 2012-11-09
Welcome to the forums.

How are you booting, from hard drive or from USB flash drive?  Does f12 or some other key give a one time boot option?

Does f8 when booting into Windows get to a repair console? You have to be quick if booting from a menu.

If booting from hard drive nothing should have changed or did you do a full install to the flash drive? Full installs take more than 4GB or about 4.4GB as a minimum, but even then you cannot do updates. With 4GB you should have done the installer version which would not change Windows at all. What instructions did you follow?

Do you have a Windows repairCD or USB? If you do not have one, use any available friend with either the  same  32 or 64 bit system.
