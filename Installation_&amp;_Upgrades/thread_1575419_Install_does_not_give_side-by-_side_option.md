### Post by oldfred on 2010-09-15
Many windows computers use all four primary partitions. Window7 has a hidden 100MB boot/recovery & then the main install. Vendors add a recovery that is really the CD/DVD they did not give you (and you should make those DVD as backup) and often another utility partition. If so you have to decide to delete one, you can back it up. Some utility partition may have software tied to special keys, you have to check you documentation.
