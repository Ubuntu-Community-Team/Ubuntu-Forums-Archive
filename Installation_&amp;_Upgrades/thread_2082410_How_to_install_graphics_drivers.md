### Post by grahammechanical on 2012-11-09
When you installed Ubuntu did you tick the box to accept third party software? If you did and if you were installing Ubuntu 12.04 or 12.10, then the proprietary videos driver were installed and are being used.

Ubuntu comes with a default open source video driver called Nouveau. Otherwise we would not get a working operating system. So, in a sense we do not need to install graphics drivers.

If you are using 12.04 and I do not know that you are for you do not tell us, then go to System Settings and load up the Additional Drivers utility.

If you are using 12.10 then you have to look somewhere else. You open System Settings>Software Sources>Additional Drivers.

Regards.
