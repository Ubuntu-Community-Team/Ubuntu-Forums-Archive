### Post by Quackers on 2011-06-05
It's not always required where "fakeraid" is concerned but it is often necessary to manually delete the raid metadata from the drives.

!!! Please note that using this will destroy any raid array in use !!!

```
sudo dmraid -rE /dev/sda
sudo dmraid -rE /dev/sdb
## etc, etc, etc ##
```
Obviously you need to use your own drives' designations. The command must be run for all drives which were used in the raid array that you don't want to use any more.

EDIT
If your disks show in the raid utility as "non-raid" there should be no need to delete the raid volumes. They shouldn't exist any more.
