### Post by Ensnared on 2006-09-13
If you have no other problems reading CD's, my bet would be on a faulty disc (CD, that is). Did you try selecting "Test installation media" or whatever the menu option is once you boot from the CD?

I suggest checking the MD5 hash against your downloaded image to make sure it didn't get corrupted during download, and if the hash matches, try burning it again (I always use an RW media for this so I don't waste CD's - once it's installed I no longer need it anyway).
