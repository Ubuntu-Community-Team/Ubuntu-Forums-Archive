If I delete anything from it the goes from my home folder so I assume that if I delete it then I will lose everything. In the folder I have a folder "desktop". ---

### Post by howefield on 2019-04-21
If you have gnome tweaks installed, go to the Extensions tab and click on the preferences icon for the Desktop Icons extension and deselect as required.

---

### Post by deadflowr on 2019-04-21
Install gnome tweaks then go to Extensions and go to the desktop icons extension.
click on the cog icon and you can toggle the Home folder to not show.

Because gnome no longer allows the file manager to show desktop icons an extension was created to do this.
A couple of discussions on that here:
[https://discourse.ubuntu.com/t/no-desktop-icons-already-arrived-in-19-04/9295]("https://discourse.ubuntu.com/t/no-desktop-icons-already-arrived-in-19-04/9295")
[https://community.ubuntu.com/t/files-nautilus-v3-28-will-lose-the-desktop-icons-capability/3115]("https://community.ubuntu.com/t/files-nautilus-v3-28-will-lose-the-desktop-icons-capability/3115")
