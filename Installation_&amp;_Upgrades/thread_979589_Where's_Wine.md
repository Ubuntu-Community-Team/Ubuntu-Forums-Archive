### Post by dazvid on 2008-11-12
Should be able to pop into Applications -> Add/Remove
Enable "All available packages" and search for wine. I don't think you need to enable any more repositories for wine to install. In the search list it is listed as "Wine Microsoft Windows Compatibility Layer".

Hopefully that works for you.
