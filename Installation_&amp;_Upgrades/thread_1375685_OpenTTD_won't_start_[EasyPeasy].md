### Post by Prendi on 2010-01-08
All OpenTTD versions prior to the 1.0-series need the data files from either a bought version of Transport Tycoon or a free replacement. Did you manually install those?

If you did, open a Terminal (under Accessories in your Application menu) and enter "openttd". This should give you detailed error messages.
