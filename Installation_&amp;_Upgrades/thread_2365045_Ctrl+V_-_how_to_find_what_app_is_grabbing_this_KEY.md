### Post by JoelParke on 2017-07-01
I solved this when I discovered that my PageUp key on the keyboard was stuck.  This was clobbering the Shift+h, Ctrl+V and probably a few other keys.  
Problem solved, but it WAS PAINFUL!  Perhaps this will help someone else.
