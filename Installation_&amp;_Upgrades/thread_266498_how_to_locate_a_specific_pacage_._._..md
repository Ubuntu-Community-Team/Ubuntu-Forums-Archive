### Post by al_sharpe on 2006-09-28
Hello jansenh:

Linear is write, it is in the universe repository.
In your Synaptic Package Manager
Settings > Repositories 
Select 4th one down Ubuntu 6.06 LTS (Binary)
Community maintained (Universe)
Click add and close then Reload on the Menu bar
Now search for 915resolution and mark for installation 
then Apply on Menu bar

OOPS noticed you are using 5.10 so use it instead of 6.06
