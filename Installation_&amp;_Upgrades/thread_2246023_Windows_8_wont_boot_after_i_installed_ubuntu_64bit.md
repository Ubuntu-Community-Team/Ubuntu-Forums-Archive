### Post by oldfred on 2014-09-27
This has been updated with one more command. There is not a trusty version of Boot-Repair and you have to change tusty to saucy for Boot-Repair's version.

 Boot Repair -Also handles LVM, GPT, separate /boot and UEFI dual boot.:
[https://help.ubuntu.com/community/Boot-Repair](https://help.ubuntu.com/community/Boot-Repair)

Post link to the summary report from Boot-Repair.

UEFI and BIOS/CSM/Legacy are not compatible. Better to install both systems in one or the other. But even if both in different modes, you should be able to boot from UEFI menu. You may have to turn on/off UEFI mode or Legacy BIOS mode to match how you installed.

How you boot installer is how it installs. And do not reinstall Ubuntu unless you use something else. Boot-Repair can convert an Ubuntu BIOS install to UEFI if you installed in BIOS boot mode.
