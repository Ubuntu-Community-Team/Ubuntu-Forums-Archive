### Post by ajgreeny on 2012-07-30
I can only suggest that you boot to the live system then run ```
sudo apt-get remove ubiquity-slideshow-ubuntu
```in terminal and see if that gets you any further.

On some hardware there seems to be a bug which crashes the slideshow and stops the installation running.  If that does not help you may need to use the Alternate-install-CD
