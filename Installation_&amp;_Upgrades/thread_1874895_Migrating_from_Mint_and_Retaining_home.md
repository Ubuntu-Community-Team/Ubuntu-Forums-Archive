### Post by Hedgehog1 on 2011-11-04
The method I use is the same as you are setup for.

Select the 'something else' (a.k.a. manual) partitioning at Ubuntu install time.

set the current '/' partition to be '/' and check the format box.

Set the current '/home' partition to  be '/home' and DO NOT CHECK the format box.
