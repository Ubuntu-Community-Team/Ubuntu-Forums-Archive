### Post by arvindp3 on 2009-11-28
I have installed Shiretoko (firefox 3.5.5)  browser from Ubuntu repository using Synaptic manager.  Firefox 3.0.15 was installed with Ubuntu installation CD, and that is still installed.   Shiretoko is faster and better than 3.0.15,  but page zoom is not working properly in Shiretoko.  Specifically, CTRL ++  does not zoom pages proportionately.  See attached snapshot of Guardian page which shows misalignment of text and pictures once zoomed-in using CTRL ++.

Firefox portal help yielded a solution for the same problem for ver 3.0.15 -- i.e. start in safe mode to identify which add-ons/plug-ins may be causing this and selectively disable those.  This corrected problem of 3.0.15 on my PC.  

But the same solution is not working with Shiretoko -- i.e. even after starting it in safe mode and disabling all add-ons, the page does not zoom proportionately.   

Has anyone experienced this and any solutions known?
