### Post by boz on 2007-02-03
I was using Ubuntu 6.06 and I just upgraded to Xubuntu 6.10.  I must say that the combination of upgrading to 6.10 and using the Xfce server is *highly* satisfactory.  I installed the xubuntu-desktop environment before I installed Xubuntu 6.10, so I proved to myself that Xfce is faster than Gnome or KDE.  I can't say whether or not Xfce has any shortcomings yet or not, but no worries.  If I end up not liking it for some reason, I can "sudo apt-get install ubuntu-desktop."  If there are others out there who have less RAM than they desire, you should try xubuntu-desktop.  You can switch between Gnome and Xfce at the login window by clicking the Sessions button.
