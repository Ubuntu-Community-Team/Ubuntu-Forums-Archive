### Post by Martigen on 2008-03-13
The Ubuntu install kernel can see Windows LDM partitions. The Ubuntu installer, however, cannot -- so it's impossible to install Ubuntu on Windows 'Dynamic Disk' partitions even though the Linux kernel fully supports them, and indeed can happily boot and run Ubuntu on them -- you just can't get that far as the Ubuntu installer pretends they're not there.

Which is a) annoying as all buggery and b) completely bizarre that this simple functionality is missing.

The Alternate install CDs suffer the same problem. As does the latest 8.04 alpha.

Has anyone developed a work around? If there isn't one, or the installer can't be fixed, Ubuntu won't be able to find a home on my machine :/
