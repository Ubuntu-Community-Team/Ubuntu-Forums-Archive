### Post by JoshuaBranson on 2011-08-05
My Update Manager wants me to install "GNOME user's guide." Here's the catch. I'm using Xfce. Is there anyway to customize what Update Manager will update? ie: So that is won't want me to update/install GNOME user's guide?

### Post by kerry_s on 2011-08-05
Go into synaptic preferences, uncheck install recommends. I think it's the third 1 down on the first tab of preferences. 

This will cause apps to pull in only what is needed to run.
don't forget to reload.
