### Post by SR_ELPIRATA on 2009-03-04
Dont have an ATI board but soon will be. My question to you would be along the lines of why the main monitor goes to sleep. Have you tried changing the power settings and screensaver so that it never goes to sleep?

You can check these on System = Preferences, in screensaver and power management. Id set them both to never. If you want to save electricity just manually power down the monitor.

The other suggestion would be to have the main monitor set as secondary and leave the DVI as main. I have a feeling you love to see the HDMI one but if its going to sleep its because you dont use it much? And if you dont use it much... why is set as primary?
