### Post by Mark Phelps on 2010-09-21
Don't understand your question, but if you're asking how to install XP to an external drive, you're basically out of luck. MS does NOT want you to be able to carry around a portable installation of their OS products; therefore, unlike with Ubuntu, they do not provide a way to install their OSs to external drives or USB sticks.  

They do provide ways to install FROM external drives or USB sticks -- but that's an entirely different thing.
