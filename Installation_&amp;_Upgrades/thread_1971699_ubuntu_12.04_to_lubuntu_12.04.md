### Post by TBABill on 2012-05-02
```
sudo apt-get install lubuntu-desktop
```Logout, click the sprocket, select Lubuntu, enter password, enjoy :) No need to reinstall the whole OS just to change desktop environments. You can also install xubuntu or kubuntu desktops the same way.
