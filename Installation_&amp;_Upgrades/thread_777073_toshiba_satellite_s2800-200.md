Hello Peter,

It's a boot installation disk ("live cd") that already has grub installed. Boot from the cd drive to install xubuntu (it should come up with a grub menu and several options.)  If you can't boot from cd then check your bios settings.

[additional note:] Many laptops have a special key sequence in order to boot from cd, mine needs me to press "F8". You may need to check for that too.

Good Luck!
