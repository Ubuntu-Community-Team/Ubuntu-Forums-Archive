### Post by uRock on 2011-01-14
Hello and welcome to the forums,

I recommend Ubuntu 10.04 64bit. It is supported for 3 years, whereas the 10.10 release is only supported for 1.5 years.

The desktop version should work fine, but I recommend booting the LiveCD and testing it on the system before installing. To see how to do this, please go to [this link]("http://www.ubuntu.com/desktop/get-ubuntu/download") and at step three, click the **Show me how** button.

Please feel free to ask any questions you may have.
