### Post by spydeyrch on 2010-07-03
For your laptop, does it have an ethernet port on it? If so, have you tried connecting to the internet via wired ethernet cable, then running the install hardware program found under system -> admin? It should scan your hardware and find any needed drivers that are compatible with your hardware.

Give that a try first if you haven't already.

I usually do it like that and I find my wireless, GPU, and a few other hardware drivers that way. I have tried it on a plethora of different systems with ranging hardware specs and have not had any issues so far. Oh, and I was using 10.04 on them all. :D

-Spydey
