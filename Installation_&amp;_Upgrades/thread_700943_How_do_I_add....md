### Post by miriad on 2008-02-18
You can right click the menus on the bar and hit "Edit menus", then you can add a new item to the menu where you want it, name the item, and for "command" put the command to run NetBeans (Or browse for the NetBeans exacutable, and put the path to that in)

### Post by Partyboi2 on 2008-02-18
main menu (System>Preferences>Main Menu) on the left pane click on Programming, then on the right click on add item. Then create the launcher
Name = netbeans
command = netbeans
comment = (whatever you want)
you can also click on the icon to the left and changing the icon pic
after pressing ok it should appear under Applications>programming>netbeans
