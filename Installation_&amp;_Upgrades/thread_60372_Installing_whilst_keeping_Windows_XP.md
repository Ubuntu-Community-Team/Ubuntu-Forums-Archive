### Post by GavinX on 2005-08-27
You don't have to use expert mode to do this. Just boot the installation CD normally and install Ubuntu. What you need to make sure you do is to partition your drive and make a separate partition for Ubuntu. In this way, Windows will be safe. The Ubuntu disk can do this for you. (When you reach the partitioning section, it will tell you that Windows was found and ask if you want to overwrite it. Select "no" and it will allow you to repartition your drive).
