### Post by mamous on 2008-12-27
Sorry there where a problem in Pidgin 2.5.3 So they install new one to fix the problems It's 2.5.3a
I would like to tell how to update pidgin 2.4.1 to 2.5.2
[COLOR="Red"][SIZE="2"]**1-**[/SIZE][/COLOR] u have to download these three files
1- pidgin ([http://www.getdeb.net/download/3615/0](http://www.getdeb.net/download/3615/0)) (561.6 kB)
2- pidgin-data ([http://www.getdeb.net/download/3615/1](http://www.getdeb.net/download/3615/1)) (6.9 MB)
3- Libpurple0 ([http://www.getdeb.net/download/3615/2](http://www.getdeb.net/download/3615/2)) (1.5 MB)
[COLOR="Red"][SIZE="2"]**2-**[/SIZE][/COLOR] Go to the Synaptic package manager and search for pidgin.
Mark "Pidgin" and "pidgin-data" for removal, it will advise its removing a libpurple0 file as well.
[COLOR="Red"][SIZE="2"]**3-**[/SIZE][/COLOR] Install the pidgin-data file first (right click and open with the GDebi package installer)
Once pidgin-data is installed then right click on the new libpurple0 installer and install that, then do the same for pidgin.
[COLOR="Red"][SIZE="2"]**4-**[/SIZE][/COLOR] Start pidgin and it should be working fine.
