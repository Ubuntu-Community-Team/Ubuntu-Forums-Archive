### Post by fantab on 2012-12-23
Don't use NERO. Use [**UNETBOOTIN**]("http://unetbootin.sourceforge.net/"). Since you already downloaded the .iso you have to guide the Unetbootin to the downloaded .iso. Or you can use [Win32DiskImager]("https://wiki.ubuntu.com/Win32DiskImager"), the page says you can't write .iso to USB but it will. You have to manually enter ".iso" next to the file name in the dialog box before you begin writing.

Good Luck.
