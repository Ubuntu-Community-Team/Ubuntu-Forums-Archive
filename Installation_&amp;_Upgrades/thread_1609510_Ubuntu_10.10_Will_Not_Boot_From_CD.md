### Post by different_guy on 2010-10-30
Hello my dear Ubuntu mates.

I have problems installing Ubuntu 10.10. It will not boot from CD. I have never had this problem before. So, I was thinking that maybe there is some problem with BIOS but everything is just fine. First boot device is CD ROM. After that I have tried older versions of Ubuntu and some other operating systems and it boots without a problem. But when it comes to newest version, it just will not boot. I have downloaded Ubuntu 10.10, extracted files from ISO file and burned them to the CD and nothing. Then I tried the same thing except I burned only Ubuntu ISO file to the CD and again no success. Am I doing something wrong? Thank you in advance!

Cheers mates.

### Post by psusi on 2010-10-30
Yes, you are burning it wrong.  You need to burn the iso as an image file, not extract it, and not burn it as a normal file.
