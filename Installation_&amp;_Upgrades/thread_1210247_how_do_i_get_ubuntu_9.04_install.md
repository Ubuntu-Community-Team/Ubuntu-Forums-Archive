[SIZE=2][FONT=Arial]Hi,

nothing to be sorry for !!

Ensure you have the latest CD for Ubuntu, the bootable disk image (ISO) can be gotten from here

[http://releases.ubuntu.com/9.04/](http://releases.ubuntu.com/9.04/)

Unless you have non-standard computer, select the 1st option (Intel x86)

Burn the ISO image, boot with it and you will get the options to run it from CD without altering your system, or installing it.

If you are going to install - MAKE SURE YOU BACK UP YOUR DATA !!!

Hope the above helps,
