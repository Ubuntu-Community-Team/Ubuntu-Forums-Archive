### Post by stuporglue on 2006-01-08
Was this "winrar" file with the extension of .iso? If so, then you misunderstood what and iso file is for. An ISO is a disk image. You'll burn that entire ISO file to CD. 

Note that you must "burn file as image" or "burn image" or somethign similar. Burning the iso file as a file will not work! When you the cd is burned, you should see that same list of files on the CD.
