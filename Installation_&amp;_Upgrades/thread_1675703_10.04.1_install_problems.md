> **BearDrummer said:**
> the shell says dimension 4300 - the annoying box that hinges at the front.  The motherboard belongs to it.  I am not sure about the graphics card.

If you boot into windows, right click on My Computer, Properties, look around for Hardware or Devices - something like that it should tell you what graphics it has.
I see that some of these have an nvidia card, so I'd try booting from CD and supply the nomodeset option:
Press Spacebar when you see the little keyboard and person icon bottom centre, select language, then hit F6, down arrow to nomodeset, hit enter, ESC, hit enter on "Try without installing" to boot.

If you don't have the max memory 512MB - I wouldn't bother trying to run 10.04 Ubuntu. Something like xubuntu or lubuntu.

PS if that works and you still want to try wubi you have to supply nomodeset on the first wubi boot as described here: [http://ubuntuforums.org/showpost.php?p=10089820&postcount=8](http://ubuntuforums.org/showpost.php?p=10089820&postcount=8)
Then after installation is completed, you'll have to override again on the next boot as described here: [http://ubuntuforums.org/showpost.php?p=10069997&postcount=1](http://ubuntuforums.org/showpost.php?p=10069997&postcount=1)
