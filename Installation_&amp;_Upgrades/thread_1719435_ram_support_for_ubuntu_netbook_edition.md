### Post by kiddfroster on 2011-04-01
A quick peek at Asus' website shows that yours only supports 2GB of RAM. [http://www.asus.com/product.aspx?P_ID=hbAiLQA6LRG0qcNH](http://www.asus.com/product.aspx?P_ID=hbAiLQA6LRG0qcNH)

The reason for this is because the chipset in your computer only supports 2GB of RAM, so it's a limitation of the hardware, not by which operating system you use.
