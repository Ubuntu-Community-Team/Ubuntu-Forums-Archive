### Post by A Belgian on 2006-09-26
Hi,

I'm trying to install Ubuntu (6.06) on an old computer I have (a 6x86 at 100 Mhz). The Bios is set up so it will boot from CD-ROM first.

If I put in the Ubuntu-CD ROM, the computer says : [INDENT]
  Boot from ATAPI CD-ROM
  1. FD     System Type-(00)[/INDENT]

and ignores the CD-ROM and starts booting up from hard disk.

- The CD ROM is OK because I have already tried it with a ship-it CD ROM from Ubuntu (which works correctly on my other computer)
- If I put in another bootable CD-ROM (Win98 for instance) it boots up correctly from CD-ROM. So the BIOS boot options are set correctly I think.

What's the problem ?

Thanks for any help...

Erwin

### Post by whizbaby on 2006-09-26
Some drives don't like some particular CDs (stars are alligned). Try burning the same and/or different .iso image (e.g. alternate install) at slow speed.
