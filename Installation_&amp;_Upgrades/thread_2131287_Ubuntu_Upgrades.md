### Post by fantab on 2013-04-01
Download and install Ubuntu 12.04 LTS, which is supported until 2017. It is quite stable and I would recommend to a newb, like you. Older version of Ubuntu are not recommended unless there is a special need to do so. And DO NOT UPGRADE. Do a Clean Install.

Follow the instructions [HERE]("http://www.ubuntu.com/download/help/install-desktop-long-term-support").
