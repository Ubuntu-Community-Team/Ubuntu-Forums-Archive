### Post by ajgreeny on 2010-09-04
Check the iso file you used with its md5sum, andf then check the install media by hitting any key when you boot the live CD or usb, and go to "Check install CD" or some similar thing in the menu.

If both of those are OK you may need to use a boot option, so press F6 for those and try them one buy one.
