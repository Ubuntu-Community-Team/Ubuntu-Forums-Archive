### Post by Herman on 2006-01-15
I just tried out the latest GParted on its own live CD. Version 0.1 has just come out on 12th Jan. 
Previously I had trouble using GParted on the Ubuntu Live CD in my test computer because the computer only barely had enough RAM to run 'Breezy' live CD, so 'Breezy' plus GParted was very slow.
The Alpha 3    0.0.9- Gparted live CD didn't work with my Book PC's video chip.
It would work with my good computer, but I wasn't ready to do any partitioning work on that one right yet.

But now I have downloaded the latest GParted -livecd -01 and I have given it a test run and it works great in my old 'Book PC'! The display works fine and not having to support a large operating system, I still have plenty of RAM, so it really performs now!
I resized my new 'Breezy' (on ext3), to a smaller size, resized the extended partition it's in right to the end of the disk, and pasted a back-up copy of my whole installed partition there! 
This is really exciting, it will make it easier to do a lot of things we love to do. I couldn't wait to tell everyone! 
[http://gparted.sourceforge.net/livecd.php]("http://gparted.sourceforge.net/livecd.php")
Anyone who needs to copy, resize partitions and move them around or any other kind of partitioning work should give it a try, you'll love it! :D :D :D  (Herman)
