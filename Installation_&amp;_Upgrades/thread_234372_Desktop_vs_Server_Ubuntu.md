### Post by az on 2006-08-11
There is nothing wrong with installing the LAMP stack on top of the Desktop install.  LIkewise, you can install ubuntu-desktop and linux-386 on top of a server install.

Do not do this on a production system, though, since it flies in the face of best security practices.


But the packages do not conflict or anything.
