### Post by jualin on 2009-09-12
The flash plugin for chromium is not yet implemented by default due to some issues, but you can get around it by going to the terminal and typing > chromium-browser --enable-plugins
You can also right click on the icon for chromium and change the command to the one above. Hope this helps! 
P.S. [Check out this website]("http://webupd8.blogspot.com/2009/07/how-to-enable-flash-support-for.html")
