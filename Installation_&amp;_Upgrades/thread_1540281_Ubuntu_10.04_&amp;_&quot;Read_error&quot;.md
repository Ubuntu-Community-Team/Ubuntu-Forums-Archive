### Post by quixote on 2010-07-30
Hmm.  That does not look good.  My first guess would be a hard drive failure.  Hopefully, I'm wrong.  The thing about hard drive failures -- if that's what this is -- is that the more you try to use them, the worse the failure gets.  So the first thing to do, if you have any data on there that you need, is to see whether you can get it off to removable media.

Boot with a liveCD, and see whether it shows the hard drive on the desktop.  Have removable media large enough for your files already plugged in.  If you can, move what you need.  Once you have that, then we can try to troubleshoot what's going on.
