### Post by QIII on 2015-05-30
AMD brought the first CPUs with the 64bit instruction set to the consumer market, and it was called AMD64.

That is the name that stuck in the industry.

It is not to cause confusion.

You will also see it called x64 and x86-64.

AMD64 is a historical convention, nothing more.
