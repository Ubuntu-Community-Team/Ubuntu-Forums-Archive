### Post by dejitarob on 2007-03-17
Hm, what did you install that is not in the application menu? In the unlikely event that it is not there, you can try going to the console, attempting to guess at the first few letters of the program name, and then hit TAB. For example for audacity you could try typing this in the console: 'aud<TAB>' and it will autocomplete the rest of the command to run audacity.
