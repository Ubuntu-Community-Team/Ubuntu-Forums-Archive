### Post by P4man on 2010-10-30
You need an ISO file. You cant turn your installed ubuntu into a livecd just like that (some other tools can though, like remastersys).

If you have an ubuntu ISO, then you dont need startup disk creator. I assume it would work, but you can just burn it directly to CD or DVD using brasero or any dvd writing app. Its only for USB sticks with persistent storage that you really need that app.
