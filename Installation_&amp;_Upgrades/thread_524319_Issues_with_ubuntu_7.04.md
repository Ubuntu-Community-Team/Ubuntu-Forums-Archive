This is annoying--I blame it solely on Fedora 7. A lot of guys have had problems with this distro. (Of course, I'm not asking about how to fix FC-7.)

I've had good luck in the past with Ubuntu 6.06LTS  and now with 6.10,  currently on another machine. OK, I have this legacy config for the Fedora 7 desktop sys:

NF325-A7 Biostar nf3 socket 754 board

ADATA 512MB DDR-400 2.5
Liteon DVD-RW Lightscribe IDE 20X

OK, I guess my question is this: Will I have issues upgrading to Ubuntu level 7 as I did with Fedora 7? I can't find any page concerning issues--those not in 6.x, but unfortunately included in 7.04--especially for desktop PCs... 

The thing with Linux is that it's not that difficult to backup, scrap one's file system, reformat, and start over with another distro. Maybe I should go ahead and risk it.... Perhaps I shouldn't make such a big deal out of this. On the other hand, Fedora has taught me that it's not always best to upgrade in Linux. In the past, I've tended always upgrade software--virtually always, it was beneficial.

### Post by bulldog on 2007-08-14
If you use the live cd,you have to boot it first before you can install.
The install button is on the Desktop.
But you can have an experience how ubuntu gets along with your hardware.
Just take a look around and see if you encounter any problems with a live session.
If you are satisfied,you can install with one push on the install button.

If you experience trouble with your graphics,it's maybe better to install with the 'alternate cd' which uses a text based interface.
After the install you can install drivers for the graphics card.
