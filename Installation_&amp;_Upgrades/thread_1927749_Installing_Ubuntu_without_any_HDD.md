### Post by darkod on 2012-02-18
1. Try the usb stick in another computer. Can it boot fine?
2. Make sure you do select to boot from usb when trying the laptop again.

Apart from that, it should run fine even without HDD present. Sometimes video issues might make it not work properly, but it should at least be able to boot from the stick.

Also, some boards do not allow anything to run unless they see a boot flagged partition on the hdd. I hope it's not the case here.
