---

### Post by DalekClock on 2007-09-13
Try looking under Add/Remove for "GNOME Partition Editor" and install it to the live CD, you can open it under System > Administration. Use it to make a new partition for Ubuntu. Alternatively, if you've still got the Edgy partition and your not desperate to use the migration assistant, you could use the Update Manage to get Feisty.
