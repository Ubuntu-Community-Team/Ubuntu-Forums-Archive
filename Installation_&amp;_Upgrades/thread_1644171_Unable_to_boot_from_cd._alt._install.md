### Post by Quackers on 2010-12-12
If you can boot from the Live cd and get to the purple Ubuntu screen press any key at that stage and you should get another screen up. If you press F6 on that screen some boot options should appear on the right side. You may need to try all of them one at a time. The most often successful one is "nomodeset". I would try that one first. Hopefully one of these boot options will allow you to continue the installation.
