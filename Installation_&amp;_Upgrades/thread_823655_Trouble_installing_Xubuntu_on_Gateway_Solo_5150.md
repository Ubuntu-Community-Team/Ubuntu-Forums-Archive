### Post by snowpine on 2008-08-21
The minimum RAM requirements for Xubuntu 8.04 are 192mb using the Live CD and 128mb using the Alternate CD. There is also a Minimal Install that just installs the command line, then you can install a lightweight windows manager on top of that ([http://www.psychocats.net/ubuntu/minimal#barebones](http://www.psychocats.net/ubuntu/minimal#barebones)). It should work on 64mb, however, there is a bug that causes it to hang at 92% (generating locales). :(

You might have better luck with Puppy.
