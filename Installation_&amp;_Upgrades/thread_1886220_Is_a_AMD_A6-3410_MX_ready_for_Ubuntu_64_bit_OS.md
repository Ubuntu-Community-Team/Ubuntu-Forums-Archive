### Post by darkod on 2011-11-24
Ubuntu 64bit will work fine with any 64bit CPU, and most CPUs are 64bit since years ago.

However, I'm still not sure that the graphics integrated into the A6 is yet recognized correctly in Ubuntu. I have seen one person on the forum saying he has problems with A6-3510MX.

Try running the ubuntu cd in live mode first to see if it will load correctly.
