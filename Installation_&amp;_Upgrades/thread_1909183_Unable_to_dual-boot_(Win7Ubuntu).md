### Post by darkod on 2012-01-14
You can't boot like that from the image file.
What are you actually trying to achieve? Install ubuntu on your win7 laptop as dual boot?

You need to burn a cd from that image you downloaded, and you can boot the laptop with the cd and install on it.

You can burn it under windows or ubuntu, what ever is easier for you. Just make sure you burn the cd from image, not just burn the ISO file as a file onto the cd. Don't burn at a speed higher than 8x or 10x.

Try making the cd first and then try to boot the laptop with it selecting the Try Ubuntu option. It will run from the cd and you can check how it works on the laptop.

After you see it's running fine from the cd, we can talk about installing it.
