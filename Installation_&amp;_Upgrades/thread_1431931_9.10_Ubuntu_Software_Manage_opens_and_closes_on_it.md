### Post by bswanboat on 2010-03-17
9.10 Ubuntu Software Manage opens and closes on its own. When I click on the menu item "Ubuntu Software Center" the program opens then closes. 

I tried to open it from a terminal and received this message:
"(software-center:2543): GStreamer-WARNING **: adding type gulong multiple times
ERROR: Could not load classifier cascade /usr/share/opencv/haarcascades/haarcascade_frontalface_alt2.xml
Could not initialize GStreamer: Error re-scanning registry , child terminated by signal"

How do I repair this? Repairing broken packages did not help.

Bill

---

### Post by bswanboat on 2010-03-17
After doing a search on "adding type gulong multiple times" I found the solution.

sudo aptitude remove frei0r-plugins
