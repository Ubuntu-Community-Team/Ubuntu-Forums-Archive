### Post by cariboo on 2008-11-10
Depending on what you are using the computer for. I usually set my / partiton for between 7Gb and 20Gb. For a desktop installation 7Gb should give enough room so that you do not have to constantly worry about cleaning out archived packages and log files, while giving you enough space that /tmp isn't complaining about not having enough free space.
