Fifty may be a bit high but here is a link on some.

[http://xwinman.org/](http://xwinman.org/)

You can do a dpkg save to reinstall with, or just read.

Install dselect if you need to reload, the first command saves the list in home.

```
dpkg --get-selections > installed software
``````
dpkg --set-selections < installed-software
```Run this to bring up dselect to reinstall.

```
sudo dselect
```Any installs that were downloads from the web and not in the repos will need a download again,

I have made this list into a bash and loaded it in synaptic as well.

[http://ubuntuforums.org/showthread.php?t=261366](http://ubuntuforums.org/showthread.php?t=261366)
```
