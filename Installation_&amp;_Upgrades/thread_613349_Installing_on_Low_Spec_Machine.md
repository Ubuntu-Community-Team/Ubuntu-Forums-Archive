The "live" cd requires 256+ Mb ram though the "alternate" install disk will work well with less, but likely not with 50 Mb. With just 50 Mb ram you should go for puppy linux or DSL (damn small linux). If adding ram is an option, you could go for 256 Mb or as close as you can get and go for ubuntu. At 256 gnome or kde would be a bit slow, but xfce (Xubuntu) will move right along. I currently have an old PIII 500 laptop with 384 Mb ram running gutsy and handling gnome quite nicely.
