### Post by claracc on 2014-06-19
Perhaps this guide: [https://help.ubuntu.com/community/Ubuntu_OEM_Installer_Overview](https://help.ubuntu.com/community/Ubuntu_OEM_Installer_Overview) can help you.

Abot installation process, note: 

>To get to the OEM Installation menu you have to keep the Shift key pressed during boot as of Lucid Lynx 10.04.1. 

Then

>Once you are on the installer selection options menu hit the F4 key to select the OEM install mode. Then hit Enter.
