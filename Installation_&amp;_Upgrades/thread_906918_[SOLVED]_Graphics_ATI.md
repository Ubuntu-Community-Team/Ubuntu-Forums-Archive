### Post by pytheas22 on 2008-08-31
If you do an upgrade from 7.10 to 8.04, you shouldn't have to reinstall the graphics driver--although if something weird happens, you may.  If you do a clean install of 8.04 in the place of your 7.10 system, then you would need to reinstall the ATI driver.  Probably the easiest way to do it is to use [envy]("http://albertomilone.com/nvidia_scripts1.html"); give it a try and if it doesn't work, let us know and we can do it another way.
