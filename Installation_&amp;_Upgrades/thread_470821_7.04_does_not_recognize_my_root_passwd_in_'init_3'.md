### Post by taurus on 2007-06-11
If you really want to install nVidia by hand, press <Ctrl><Alt>F2 and log in with your username and password.  Then, shutdown X with

```
sudo /etc/init.d/gdm stop
```
Now, install nVidia driver for your graphic card and one it's done, restart X again with

```
sudo /etc/init.d/gdm start
```
On the other hand, why not install it the easy way, [http://www.psychocats.net/ubuntu/nvidia](http://www.psychocats.net/ubuntu/nvidia)?
