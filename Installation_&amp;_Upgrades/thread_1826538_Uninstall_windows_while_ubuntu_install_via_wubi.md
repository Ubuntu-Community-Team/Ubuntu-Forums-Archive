### Post by rreyes3713 on 2011-08-16
One solution possibility ...

since you have a Wubi installation, what about doing a "migration"?

Create a new partition on the hard drive. Migrate the Wubi installation to this new partition. Then from Ubuntu, remove the Window partition. 

If you are interested, I'll look for the post how to do this. 

Personally I would keep the Window installation. You never know what could happen during an Ubuntu update or you make some changes and something goes wrong. 

At least you have a "working" operating system. Just a thought.
