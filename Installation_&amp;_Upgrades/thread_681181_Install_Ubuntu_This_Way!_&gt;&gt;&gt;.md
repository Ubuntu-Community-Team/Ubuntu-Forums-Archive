Hi. I'm having several problems since I tried to install ubuntu BECAUSE OF THIS:
My CD burner wasn't working soo well, but I didn't notice; so I burnt the CD with the UBUNTU image, and everything seemed to be OK. 

When I was in the Live Session, installing ubuntu, it was about in 52% percent, when it gave me an error. The CD had 1 FILE WRONG ( I know because I checked the CD in the ubuntu menu) 

The thing is that ubuntu hides windows' parition and installs GRUB (an application to switch between OSs) the latest, so my installation could't reach to complete that stage. 

Now, I am stuck in ubuntu, but I found my way out, you have to make a GRUB diskette (google it). That will save me, according to the IT guy.

INSTALLATION I RECOMMEND:
DONT BURN A CD. Because it may have errors and not complete your installation.
If you are on windows use this: [https://wiki.ubuntu.com/InstallerForWindows](https://wiki.ubuntu.com/InstallerForWindows)
or find a way to install ubuntu from your OS. The whole idea is to use the ISO file mounted on a virtual drive, so that you can dodge any error from the CD.
