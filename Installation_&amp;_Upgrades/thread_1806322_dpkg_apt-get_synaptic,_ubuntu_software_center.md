### Post by snowpine on 2011-07-17
> **th00ht said:**
> I'm a bit confused by all the different installation tools that Ubunut offers. dpkg, apt-get synaptic, and ubunut software center. Why are there so many installation tools and which one should I use?

They are different "front ends" to the same process (dpkg) and you should use whichever you prefer. For a beginner used to GUI and uncomfortable with the terminal, Software Center is probably the easiest. If you are a terminal nerd like me then give apt-get a try. :)
