### Post by CardUJ on 2010-05-20
I had an issue with a new install of 10.4 that the nvidia driver could not detect the correct resolution. I was only offered basic resolution or with some messing about i could get to 1360x768....

I read a lot of forums, tried many things, but it seemed massively complicated for a new'ish monitor, card and OS...

It was resolved simply by booting up with the monitor connected via DVI rather than VGA. All the correct information was pulled through (it was previously not detecting my monitor model, as well as not offering the correct resolutions).

Once I had configured it as i needed, i saved the xorg.conf, reconnected the vga cable and it worked as it should. I hope this helps someone spend less time on it than i did.

For those keen to configure xorg etc... this is a copy of mine, if it helps... Now if i can find a simple way of getting itunes working in wine with 64-bit ubuntu, i will be done!
