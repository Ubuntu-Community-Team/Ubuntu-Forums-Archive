### Post by slambrick on 2007-11-20
Hi, I would think that the simplest and easiest way is to reformat your partitions when installing from the Livecd. This way you get a nice fresh start with no residual leftovers. Of course make sure you either have backed up any data to somewhere else or make sure your data is on a partition you don't reformat. Good Luck and Cheers.
