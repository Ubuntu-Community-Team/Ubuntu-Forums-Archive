### Post by Cheesehead on 2012-06-14
Since you plan on creating a USB-version, try Debian Live ([http://live.debian.net](http://live.debian.net)) as a way to create a custom live-boot environment.

It doesn't take your current desktop environment, it takes the lists of packages and custom files you specify and creates the new custome environment you specify. It also saves a lot of space.
