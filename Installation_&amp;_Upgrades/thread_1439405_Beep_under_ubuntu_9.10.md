### Post by dstew on 2010-03-26
Does it only beep when running Ubuntu? Does Ubuntu (or Windows) run OK when the beeping is going on? Sometimes the beep may be an alert to a motherboard configuration problem. There might be some LEDs on the board that give you an error code, or maybe the pattern of beeps. You might check the [Asus support forums]("http://vip.asus.com/forum/view.aspx?id=20091230164851875&board_id=1&model=P5N-T+Deluxe&page=1&SLanguage=en-us").
