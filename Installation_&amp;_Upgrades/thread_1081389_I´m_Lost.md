### Post by wpshooter on 2009-02-26
ALWAYS, the first thing you should do AFTER downloading and burning a Ubuntu CD, is to run the CD verification function that is found on the initial Ubuntu boot screen menu.  Do this BEFORE trying to either boot to the live CD or install from that CD.  Do this whether you did any kind of verification during the CD burn process or not.
