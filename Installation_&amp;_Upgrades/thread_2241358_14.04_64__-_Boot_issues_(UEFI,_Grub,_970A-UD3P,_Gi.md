### Post by oldfred on 2014-08-25
You have  a  full drive LVM install in sda with separate /boot & /efi partitions. The efi partition is for UEFI boot. And with UEFI you have gpt partitioned drive.
And you have a BIOS boot with with MBR(msdos) of 13.10 in sdb.

Which install are you trying to boot.
You do have to go into UEFI/BIOS and turn on UEFI to boot install in sda, and turn off UEFI and turn on BIOS/CSM to boot install in sdb.
Not sure if either is fully configured or not. I do not know AMD systems and whether they need boot parameters to boot.

LVM is another level of complexity, but does add some flexibility for experienced users. I do not use it.
