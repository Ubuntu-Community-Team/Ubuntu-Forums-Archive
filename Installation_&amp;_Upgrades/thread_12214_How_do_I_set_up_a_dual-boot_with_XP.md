[QUOTE=ZZZZZ]and does ubuntu makes a partition for me or do i have to do it by myself?[/QUOTE]
No ubuntu does not create partitions .You must have enough free unpartitioned space before you start. my suggestion is to defag Xp then you can use partition magic in windows to resize your existing partition and you will need at least 2gb to install Ubuntu . You can also use another linux distro like mandrake or  redhat to do the same. Once you have your free space boot from ubuntu cd and follow instructions make sure you save boot loader to MBR.
