### Post by Rocket2DMn on 2008-04-01
You should have the option to enter Safe Graphics mode when you start up the CD, you may have to hit F6 or something, I don't remember.  If you are just going to install, you can use the alternate install cd which doesn't have a live session, just an easy to use text based installer.  You can download it at [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) by checking the box at the bottom for alternate cd.
The LiveCD just doesn't have full support for video drivers, but your system will work after you install.  You will want to use the restricted drivers for your card, it should prompt you about that after you install, too.
