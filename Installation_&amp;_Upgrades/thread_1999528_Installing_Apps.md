### Post by cortman on 2012-06-08
Not sure why you would think that installing Gnome shell also downloads icon themes. I assume you are talking about the Faenza icon theme, which is available for download at the best website around for Gnome visual stuff- [gnome-look]("http://gnome-look.org/").
For instruction on how to install the icon theme once you have it, see [here]("http://cortman.wordpress.com/2012/02/25/how-to-install-themesicons-in-gnome-shell/").
