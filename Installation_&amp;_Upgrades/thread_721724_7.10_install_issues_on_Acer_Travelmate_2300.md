### Post by Pumalite on 2008-03-12
The kernel did not recognize your card and did not load the module, therefore:UNCLAIMED

'*-network:1 UNCLAIMED
description: Ethernet controller
product: [AirConn] INPROCOMM IPN 2220 Wireless LAN Adapter (rev 01)
vendor: Linksys, A Division of Cisco Systems
physical id: 4
bus info: pci@0000:02:04.0
version: 00
width: 32 bits
clock: 33MHz
capabilities: bus_master cap_list
configuration: latency=64 maxlatency=32 mingnt=32 '

Go to Adnistration>Software Sources and make sure 'backports' are enable.

Then, you might want to try this:
sudo apt-get install linux-backports-modules-generic

Reboot and post lshw -C network again.
