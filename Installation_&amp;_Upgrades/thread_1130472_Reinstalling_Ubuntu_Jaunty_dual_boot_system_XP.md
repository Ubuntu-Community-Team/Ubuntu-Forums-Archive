### Post by tommcd on 2009-04-20
Get yourself a Parted Magic live CD:
[http://partedmagic.com/](http://partedmagic.com/)
Boot up Parted Magic and reformat your Ubuntu partition. Then boot up the Jaunty CD and reinstall it.

I'm not sure why there is no option to reinstall, unless this is something new in Jaunty. I have reinstalled in previous versions of Ubuntu, and the option to install is always available. I have not tried to reinstall Jaunty though.
