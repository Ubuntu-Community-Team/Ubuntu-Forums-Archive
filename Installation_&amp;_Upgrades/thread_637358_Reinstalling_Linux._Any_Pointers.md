### Post by rsambuca on 2007-12-10
Mint is just built on an ubuntu base, so if compiz runs on Ubuntu with your specs, then it will run on Mint as well.  You can actually install Ubuntu, and then just add the mint repo, and you will be running mint.

If you feel confident with the manual partition option, when installing setup /home on its on partition. That way if in future you have to reinstall for any reason you will still have you /home and settings.
If not confident to manually partition then you can always move /home later
[http://www.psychocats.net/ubuntu/separatehome](http://www.psychocats.net/ubuntu/separatehome)
