### Post by Odrodzona-Sarmacja on 2008-02-04

### Post by diaa on 2008-02-04
yes, you are correct, most applications store their settings in your home directory in file or folders starting with a dot, for example firefox settings are stored in *.mozilla*, pidgin in *.purple* and so on.

so I guess you're on the right track
