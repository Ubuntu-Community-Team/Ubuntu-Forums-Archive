### Post by leonardo_neo on 2009-04-03
I can't say what you 'should' do, but I can say what I 'would' do if I were in you place.

You see, if you can run 8.10 in your computer, then you shouldn't have any problem with 9.04 (In terms of hardware requirements). The newer version is introducing ext4 file system, which is claimed to work fast. Beware though, even many experienced users are not showing interest to install ext4, as ext4 is having some issues presently (data loss, and some other issues). But still ext3 is the default file system in 9.04, so if you wish you can continue with ext3 only.

Now if I were in your position, I would first try to improve the RAM, because 256 Mb is the minimum requirement asked by Ubuntu. I guess you must be having DDR1 RAM, which is expensive now a days, but you should try getting it.

Second thing which I will do, is that I will upgrade to 9.04 for sure. A newer version usually have fixes and new features, so why not to get that? :)
