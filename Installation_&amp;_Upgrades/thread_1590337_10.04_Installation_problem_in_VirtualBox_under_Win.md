### Post by sikander3786 on 2010-10-07
Are you installing from CD-ROM? Did you download Ubuntu imgae? If so did you check the MD5SUM of the downloaded image?

[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

Also check the disc for defects. There is an option on the first menu where you choose "Try Ubuntu without making any changes".
