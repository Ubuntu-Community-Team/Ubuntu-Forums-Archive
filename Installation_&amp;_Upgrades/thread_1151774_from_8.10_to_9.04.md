### Post by HyRax on 2009-05-07
You can do a full upgrade using the ALTERNATE 9.04 install CD. You CANNOT do an upgrade with the 9.04 LIVE CD.

The upgrade does not have to access the Internet to finish, however it is well advised that you do provide some Internet access in order to gain access to the latest and greatest updates.

### Post by HyRax on 2009-05-07
You're probably using an Intel based gfx chipset - there are some known issues including black screens. You may be better off doing a complete clean install.

I had similar issues with my EeePC (using Intel gfx) and a clean install was the simplest way to alleviate them. My /home is in a separate partition, so installing over the top without formatting /home is trivial.
