If everything **works fine and does not show any error**, you could keep you 32-bit application. 64-bit arch processors can support both 32-bit and 64-bit Ubuntu versions.

I'd recommend though that you install the 64-bit version in your system.

Click on the link below to read more on 32-bit and 64-bit Ubuntu versions.

[https://help.ubuntu.com/community/32bit_and_64bit](https://help.ubuntu.com/community/32bit_and_64bit)
