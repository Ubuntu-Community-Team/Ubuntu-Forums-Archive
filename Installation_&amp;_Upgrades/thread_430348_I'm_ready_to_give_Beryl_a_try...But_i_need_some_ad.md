### Post by renzokuken on 2007-05-02
those bits at the bottom are the controls for nano. nano is a terminal based text editor. ^ represent the Control key

so to save the file press Ctrl+X for exit, it will then ask if you would like to save, press 'Y' for yes and then Enter to confirm.

all saved

your graifx card is an Intel 845 onboard chipset.

you could also do this in gedit which would be alot easier for a beginner.

in a terminal, type the following

```
gksudo gedit /etc/X11/xorg.conf
``` and then do the changes they say. if you need more help, post the contents of that file here and i can show you where to make the changes so you understand a bit better. knowing your way round an xorg.conf is a valuable linux skill
