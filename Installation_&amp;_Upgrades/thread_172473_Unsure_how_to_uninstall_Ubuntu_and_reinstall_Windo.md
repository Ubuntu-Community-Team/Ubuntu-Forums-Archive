### Post by scooper on 2006-05-08
It couldn't be much easier if you have a bootable XP CD or floppies.  You just install XP as if it's a blank machine and it will blow away Ubuntu.  The real trick is if you want to preserve the ability to boot both systems.

Either let the XP install automatically select a partition or choose the one with ubuntu and let it delete it and reformat it as NTFS.  The automatic selection, if it kicks in, will likely choose the one big partition it doesn't recognize as its own.

It's hard to give more precise instructions without looking at the install screens myself.

Good luck,
Steve
