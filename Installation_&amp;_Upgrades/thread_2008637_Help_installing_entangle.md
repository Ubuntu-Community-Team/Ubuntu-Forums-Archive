### Post by greyboar on 2012-08-20
Hi,
in order to install something from [www.getdeb.net](www.getdeb.net) you need first to add their repository. As described on the welcome site go to [www.getdeb.net/updates/Ubuntu/12.04/](www.getdeb.net/updates/Ubuntu/12.04/) and install the **getdeb** package or add the repository by hand. After I installed the package, I run ```
sudo apt-get update
``` and from  now on you can install **entangle** etc. with one click!
