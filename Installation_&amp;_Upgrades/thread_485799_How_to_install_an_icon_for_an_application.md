### Post by luvr on 2007-06-27
If you're trying to add an icon to, e.g., the top panel, so that you can start the application from the panel (like the *Firefox* icon), then simply:
[LIST]
[*]Right-click on the panel;
[*]Select *"Add to panel..."*
[*]Tick *"Custom Application Launcher"*
[/LIST]
In the *"Create Launcher"* window, you can tick *"Browse..."* to locate the executable file, and you can tick the *"No icon"* button to select an icon for the launcher. Also, type a *"Name"* for your launcher, and optionally a *"Comment"* that will pop up if you move your mouse pointer over the icon.
