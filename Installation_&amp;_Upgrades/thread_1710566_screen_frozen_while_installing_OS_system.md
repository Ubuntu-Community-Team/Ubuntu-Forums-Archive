Welcome to UF
 When using the reisub keys to shut down, with some keyboards you have to be sure to let off the alt & sys rq keys before letting up the "b" key, and sometimes you may need to try it twice. If that doesn't work, you may be forced to do a hard reboot. If so, be sure to hold the power button until it completely stops.

Also, what video card do you have?
