### Post by darkod on 2010-02-22
You probably only downloaded the ISO image. Create a cd from that ISO image and boot with the cd in the computer. If CDROM is before HDD in your BIOS it would boot from the cd where you will have option only to Try Ubuntu running from the cd, Install Ubuntu, etc.
