### Post by Therion on 2009-01-23
Any package updates performed while running a "Live" session will not be persistent.

You can create your own "LiveCD" by using something like **Remastersys** or you could download **Super Ubuntu** (the install images are updated monthly (or thereabouts)) since those install discs will have most of the updates on them already.

Free bonus link to Super Ubuntu:  [http://www.mininova.org/tor/2101216](http://www.mininova.org/tor/2101216)
