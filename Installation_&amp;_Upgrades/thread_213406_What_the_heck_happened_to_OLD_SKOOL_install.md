### Post by mogelhead on 2006-07-11
The CD you downloaded is both a Live CD and Install CD.

When you boot with it, it will start a linux desktop from the CD. Thats why it's slow.

Fortunately you can also use this CD to install ubuntu. See [this guide]("https://help.ubuntu.com/community/GraphicalInstall").
