### Post by 73ckn797 on 2009-01-16
> **BrentonEccles said:**
> If I install Ubuntu, does it completely wipe Microsoft Windows (in this case Vista) from my system? [please say yes ;p]


That, my friend, is entirely up to you.

Using the Live CD you could go to System/Administration/Partion Editor and reformat the hard drive for the new installation. You can allow Ubuntu to resize your drive and install on the newly created partition and run a dual boot system. Later, if you really want to get rid of Vista, you can still wipe the Vista partition and resize the Ubuntu partition to use the entire drive space.

Also read this: [https://help.ubuntu.com/community/](https://help.ubuntu.com/community/)

Read the sticky notes in the Absolute Beginners forum.
