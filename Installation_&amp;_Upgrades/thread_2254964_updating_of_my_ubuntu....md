### Post by grahammechanical on 2014-12-01
Ubuntu uses a utility called APT (Advanced Packaging Tool) to download, install software and that includes updating/upgrading the operating system. Not only can we run various APT commands in the terminal but we also can use different Graphical User Interface utilities but they all use the APT program to do the work.

[http://en.wikipedia.org/wiki/Advanced_Packaging_Tool](http://en.wikipedia.org/wiki/Advanced_Packaging_Tool)

Therefore, there is no alternative to updating Ubuntu if the method we are using is broken. We must fix the problem. We cannot help you unless you tell us the problem. 

Ubuntu usually gives error messages. These messages are important because they can be used to determine what is wrong. Once we get an idea of what is wrong we can give different suggestions as to what might fix the problem.

Do you install applications through the Ubuntu Software Centre? Or have you downloaded software from some web site and then installed it? Has the computer been switched off during the time the operating system was being updated? Are you using an unsupported version of Ubuntu? If you are not using Ubuntu 12.04 or 14.04 or 14.10, then you are using a version of Ubuntu that has reached End of Life and will no longer be updated in the usual way.

You see, there could all kinds of reasons why you are having this problem. We are having to guess. And that is causing a delay in you fixing the problem.
