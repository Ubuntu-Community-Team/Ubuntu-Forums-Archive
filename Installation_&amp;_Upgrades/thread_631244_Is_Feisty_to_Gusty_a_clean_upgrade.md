### Post by Topsiho on 2007-12-04
I did the upgrade, but will not do so again upgrading to Hardy.

A clean install is very much better, and easy, and faster then the upgrade, even if the upgrade takes only the pressing of a button (and answering some questions ***during the process that can take hours!!!***).

I downloaded some 1200 MB of files, which is two CDroms. And as I said: the whole process did take hours, where a clean install takes 20 minutes.

Of course: you have to download the .iso file (700 MB), burn it, check it, start it and only then you can do the install.
After that, and there is a disadvantage, you have to install all your special programs again.
Another disadvantage is that you have to really take care you do not wipe your home directories (you have the /home on a separate partition, do you? If not: upgrade!).
Anyway: first do a backup of all your personal and precious files.

If you have more than one computers you can use the CDrom to install Gutsy on all of them.

Topsiho

### Post by barney_1 on 2007-12-04
Wow, that's a lot of negativity. 

I say upgrade.  I've done it on a kubuntu system as well as an ubuntu system.  If you are using proprietary video drivers such as ati or nvidia (especially if installed using envy) you should uninstall those drivers before the upgrade and reinstall them afterward.

Other than that the upgrade was pretty successful for me.  It's such a pain to get settings back to where you want them after a clean install.  I think it's better to upgrade and chase the small number of issues that might pop-up as a result.

Good luck!
