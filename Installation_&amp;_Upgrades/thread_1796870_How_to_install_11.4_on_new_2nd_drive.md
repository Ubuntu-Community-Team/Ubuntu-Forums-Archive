### Post by jcobban on 2011-07-04
I picked up a used computer with WinXP on an 80GB drive.  I installed a 1TB drive in it.  I want to install Ubuntu on the 2nd drive, but the installer will only do that if I request manual partitioning, which requires more knowledge of how to organize the drive layout that I have.  In particular it complains that I have not allocated a swap partition, but I do not see any obvious option in the partitioning dialog to manually allocate a swap partition.

---

### Post by mikewhatever on 2011-07-04
You simply need to create a new partition about the size of RAM and select 'swap' as its type. You can also tell the installer to erase the entire disk, which would autopartition it.
