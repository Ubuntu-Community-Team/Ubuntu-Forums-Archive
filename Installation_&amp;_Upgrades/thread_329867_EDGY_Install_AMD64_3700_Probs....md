### Post by phossal on 2007-01-02
I recommend avoiding the 64bit Ubuntu, if you're using it. There isn't any advantage, but there are some associated headaches. Use the 386.iso. I had similar troubles with the splash screen with 64bit Ubuntu, among other things. Although my card wasn't properly configured "out of the box" with the 32bit install, there was significant improvement.

A moderator I respect, who also runs a 64bit platform, recently informed me it was possible to use the 386.iso. Up until a week ago, I had just resigned myself to being without certain features.
