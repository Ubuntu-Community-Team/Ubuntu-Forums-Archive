### Post by Herman on 2010-03-20
I think you should wait until Lucid is released in another month or so. 
Not everything works in Lucid yet if you want to use it as a normal operating system.
In my Lucid installation I can't start Open Office.org spreadsheet (it doesn't open), and every time I plug in a USB I need to mount it the old fashioned way, (command line).
Yesterday our gnome desktops went awal for a couple of hours.
I haven't been able to install Quantum GIS in mine yet either.

If you still want to try Lucid out it would be better to install it alongside your existing installation and dual boot two Ubuntus rather than upgrade.
