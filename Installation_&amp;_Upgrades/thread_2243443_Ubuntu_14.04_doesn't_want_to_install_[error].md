### Post by yancek on 2014-09-08
What are you trying to install, Ubuntu 14.04?  How did you put the Ubuntu iso on the flash drive?  You can't just copy it.  There is different software available to do that such as unetbootin or pendrivelinux which would create a bootable flash.  If you don't have anything else (any other operating system) on the drive, you could select erase disk as an option as long as that is the only hard drive.  Otherwise, you would need to make sure you get the correct drive.  Also, posting the error would be a good idea.
