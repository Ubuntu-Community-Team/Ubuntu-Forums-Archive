### Post by viper250 on 2010-02-22
try a cd iso (sometimes burning an iso to dvd can be a problem)
if the cd iso loads then the issue is probably the dvd iso errors.
If the dvd drive has a sata interface you may have to change the mode of the drive in the bios. even though it may be a sata drive bios has an ide mode emulation for both hard drive and cd/dvd drives.
but as you stated it loaded after second try and crashed when you clicked on an icon showing video anomolies. this is a kernal panic and is likely caused by the video mode! different  settings may need to be tried
