### Post by efflandt on 2012-07-27
How are you booting the install (CD or bootable iso on USB), and does that boot to a live system?  If it cannot even find that are, you pressing whatever hotkey the BIOS splash screen shows for selecting boot device (typically F12). 

Even if you set boot device order in CMOS setup to boot something else before hard drive, sometimes you still need to select removable devices during boot.
