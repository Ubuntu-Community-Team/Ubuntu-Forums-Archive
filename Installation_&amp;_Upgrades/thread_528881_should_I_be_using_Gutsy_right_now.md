### Post by kostkon on 2007-08-18
Yes, you will be able to go from a Tribe or beta to the final release. But remember, a *Tribe* version is an alpha version so expect various problems and bugs. Also, expect various breakages when updating from a current version to the next one. Your system will become fully stable only when you go from the *Release Candidate* to the *Final Release* version.
