### Post by jpkotta on 2008-12-18
[https://help.ubuntu.com/8.04/musicvideophotos/C/cdburning.html](https://help.ubuntu.com/8.04/musicvideophotos/C/cdburning.html)

I prefer k3b for burning optical disks.  AFAIK, all the GUI burning programs use the same backend stuff, so it's all about user interface and robustness.

If you just want to put 5 GB of data on a DVD instead of your HD or 7 CDs, this will work.  If you think you can transform 7 CDs into an equivalent DVD, you will probably be disappointed.
