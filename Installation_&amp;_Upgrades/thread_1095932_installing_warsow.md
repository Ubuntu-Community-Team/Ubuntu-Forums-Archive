### Post by Partyboi2 on 2009-03-14
Make sure you have the "multiverse" repo  enabled, you can do this by going to Software Sources (System>Admin>Software Sources) and ticking the box for "multiverse" if it is not already ticked on the first tab, then close and reload the package list. You can either search synaptic (System>Admin>Synaptic Package Manager) for warsow and install or you can install it from the terminal (Applications>Accessories>Terminal)
```
sudo apt-get install warsow
```
