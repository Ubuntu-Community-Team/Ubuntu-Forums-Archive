### Post by AndrewTheArt on 2008-04-05
If you want a user friendly and modern OS that will work with those requirements, I recommend Xubuntu.

To run the Desktop CD (LiveCD + Install CD), you need 128 MB RAM to run or 192 MB RAM to install. The Alternate Install CD only required you to have 64 MB RAM.

To install Xubuntu, you need 1.5 GB of free space on your hard disk.

Once installed, Xubuntu can run with 64 MB RAM, but it is strongly recommended to use at least 128 MB RAM. (That means you might want to upgrade the RAM if possible. If you can't, Xubuntu will still run at a tolerable speed with 64MB RAM)

Xubuntu can be downloaded from:
[http://www.xubuntu.org/get](http://www.xubuntu.org/get)
