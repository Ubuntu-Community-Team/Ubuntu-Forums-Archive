### Post by whych on 2010-01-05
You shouldn't need more than a 1 gig usb stick for the netbook remix. Have you updated to the latest nc10 bios? I found it messes up the live usb, so it's best to use the older bios.
Wubi should also work for the remix version, but I hated the netbook version and use the normal ubuntu when I want a debian distro (my main distro is suse).
You can also try eeebuntu - the interface is nicer than ubuntu remix or any of the others if you are a more power user.
Also, try playing with the boot options given by pressing F4 or F6 on the ubuntu boot screen.
