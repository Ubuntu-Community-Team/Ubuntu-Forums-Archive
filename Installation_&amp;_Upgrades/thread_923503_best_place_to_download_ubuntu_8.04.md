### Post by prshah on 2008-09-18

> 
i tried the ubuntu website but the file i downloaded was corrupt, i have tried lots of sites but all i have tried are corrupt


Try the "torrent" files. You will need a bittorrent client.

Get the ".torrent" file from [http://mirrors.gigenet.com/ubuntu/8.04.1/](http://mirrors.gigenet.com/ubuntu/8.04.1/)

For example: [http://mirrors.gigenet.com/ubuntu/8.04.1/ubuntu-8.04.1-desktop-i386.iso.torrent](http://mirrors.gigenet.com/ubuntu/8.04.1/ubuntu-8.04.1-desktop-i386.iso.torrent)

Save the torrent file, then open that file in a bittorrent client. For a more detailed explanation of torrent (including usage instructions), see [https://help.ubuntu.com/community/BitTorrent](https://help.ubuntu.com/community/BitTorrent)

Finally, remember to burn the ISO as an image; ie. don;t burn the ISO _file_ itself to the disk.
