### Post by avtolle on 2009-01-31
The 32 bit Ubuntu will work fine, although if you have RAM>4 gigs, the excess will not be accessed directly. The 64 bit will work on your Intel system; it is my understanding that as AMD was first with 64-bit, by convention the release for 64 bit processors are labeled AMD, but as the instruction set is standardized, it will work whether the processor is Intel or AMD.
