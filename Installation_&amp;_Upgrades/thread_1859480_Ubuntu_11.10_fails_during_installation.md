### Post by AlmightyHeretic on 2011-10-14
I'm assuming you're booting the liveCD in Try Ubuntu mode. When your desktop comes up open Computer. Then find your existing Ubuntu installation. It will not be under File System. File System is the liveCD folder. Instead it will be under "250gb Hard Disk: 125gb File System" or something like that. If you have a dual-boot installation with Windows and Ubuntu then it'll be the drive that doesn't contain your Windows installation. From there open the "home" folder. Then open your user folder. You should then see a list with all of your user folders such as Documents, Videos, Pictures... Your document should still be wherever you saved it. If you browse to the appropriate folder and it's not there then there's a chance that you did a clean install and wiped your data.
