### Post by drs305 on 2009-10-22
If you boot the LiveCd and use GParted you can erase them all if you wish, then set up your partitions or let Ubuntu do it for you. Unmount all the partitions, including swap, to delete the partitions. You will already be on the LiveCD so all you would have to do is click on the Install icon.

9.10 is coming out in a week if you would prefer to wait for the next release.
