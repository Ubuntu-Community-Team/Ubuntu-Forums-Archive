### Post by wrm1609 on 2007-03-09
Have just installed Ubuntu dual boot with Windows XP . Ubuntu works well - including first time set up of a wireless network card, and printer, that I could not get to work at all in the previous release. Since then, however, I have also added a wireless keyboard and mouse to my set-up. These work well in Ubuntu (and in XP), but do not work at bootloader (grub) time - selection of the boot options is, therefore, impossible. Is it correct to assume that if I want to dual boot between Ubuntu and XP, then my only choice is to revert to a traditional keyboard, or is there a way round this?

---

### Post by Kateikyoushi on 2007-03-09
Try to enable the USB keyboard support in BIOS using an old keyboard.
