### Post by zalberico on 2009-10-29
You're using a 32 bit OS at the moment (just because you happen to be) however, your chip is 64 bit and can run the 64 bit ubuntu.  The 'AMD' version is for all 64 bit x86 processors (both Intel AMD and others).  It is called amd64 simply because amd came up with the x86-64 instruction set architecture and named it that.  

So in short either the 32 or 64 bit will work.

Install the gnome ubuntu (regular ubuntu).  It'll be easier for you to find help.

Good luck!
