### Post by egeezer on 2015-09-19
That happened to me a while ago, but I was too busy to track it down so I just used The Great Debian Update Tweak: Synaptic Reload > Mark All Upgrades > Apply.  After a lot of times doing that, the Software Updater miraculously started working again.  This happened on 14.04 with kernel 3.13.0 as well as 3.16.0 - two different systems.  Sorry to be no help, but thought I'd let you know you weren't the only one! ;-)
