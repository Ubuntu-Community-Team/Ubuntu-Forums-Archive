### Post by dstew on 2008-04-04
> Is they a way to use synaptic to install packages wich are on the USB stick or the hard drive.There is a way. It is not using synaptic, since that program depends on apt-get, which depends on internet access. But, to get packages with dependencies, to download to a USB stick or CD, try the [nonetdebs service]("http://nonetdebs.homeip.net/"). You create a text file that contains the current status of the packages on your no-internet computer, and upload it to their web site when you get to an internet computer. They analyse the status, just like the Update Manager or Synaptic, and give you a list of links to the Debian packages to install. What is nice about it is it includes all the dependencies. Then, you down load the packages, copy then to a USB stick or CD, transfer then to your no-internet computer and install them.
