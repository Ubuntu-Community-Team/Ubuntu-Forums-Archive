### Post by QIII on 2018-05-03
Hello!

Wine, which allows some Windows applications to run on
Linux, does not list Legacy Family Tree 9.0, so you would be on your own trying to get it work that way.

As for virtualization:

VirtualBox may be the easiest solution for you.  You can either install it from the repo, which is limited without some proprietary bits, or from virtualbox.org.

That would still require the Windows installation media and  license key.  I think that is about $120US for the OEM version of Windows 10 Home.
