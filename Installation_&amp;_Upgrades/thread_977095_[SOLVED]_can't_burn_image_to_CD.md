### Post by Partyboi2 on 2008-11-11
Try using a different program to burn the iso, I use one called [[COLOR=Blue]imgburn[/COLOR]]("http://www.imgburn.com/") its a small program that has never failed me. Make sure that the iso you download md5sum matches. See [[COLOR=Blue]here[/COLOR]]("https://help.ubuntu.com/community/HowToMD5SUM"). Then burn the iso as and iso not data at x4 or less. If you requested a free cd of ubuntu from [[COLOR=Blue]shipit[/COLOR]]("https://shipit.ubuntu.com/") it will take about 4-8 weeks to arrive.
