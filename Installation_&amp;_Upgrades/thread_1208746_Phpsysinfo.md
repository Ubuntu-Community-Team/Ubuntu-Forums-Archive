### Post by Partyboi2 on 2009-07-10
Hi, you can open Synaptic Package Manager (System>Admin>Synaptic Package Manager) and search for it and install it or you can open a terminal (Applications>Accessories>Terminal) and install it with
```
sudo apt-get install phpsysinfo
```
