### Post by oldfred on 2012-04-15
Post output from to see if partition entries look ok:

sudo fdisk -lu

gparted will not show partitions if partition table has errors or it cannot mount NTFS partitions that need chkdsk.

Also was drive ever part of a RAID set or have you ever turned on RAID in BIOS. That can add settings to drive that confuses gparted.
