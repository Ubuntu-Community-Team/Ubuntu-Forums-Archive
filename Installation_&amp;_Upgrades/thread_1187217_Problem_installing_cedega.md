### Post by Mike'sHardLinux on 2009-06-14
Hi.

You downloaded the source code. You should look up how to install from source. The basic process is generally referred to as "./configure, make, make install". Installing from source can be a lot more work because it is up to you to make sure your computer meets all the dependencies, IOW, has all the needed software installed to compile the source.

If I were you, I'd search to see if there's already a specific tutorial for installing Cedega from source in Ubuntu, since there may already be one.

Good luck.
