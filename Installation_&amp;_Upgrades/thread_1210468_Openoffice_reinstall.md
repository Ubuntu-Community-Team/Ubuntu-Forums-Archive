### Post by unlimitedz on 2009-07-12
Never tried it, but I think you can go to "System-Administration-Software Sources" and then go to "Third Party Software".  From there you can enable the Ubuntu CD, or add a CDRom (at the bottom there's a button, make sure the CD is in the drive when you click it).  After you add the source, you should be able to use the package manager to add/remove software from it (you'll have to update the package listing).
