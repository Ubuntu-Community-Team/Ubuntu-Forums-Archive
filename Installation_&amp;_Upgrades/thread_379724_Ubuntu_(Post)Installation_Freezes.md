### Post by lhtown on 2007-03-08
Random crashes can be a bit tough to run down. It sounds like yours isn't too random though. Here are some ideas of things to try to narrow down the problem. (listed in no particular order

-You can try checking the disk you used to install to make sure it is error free. Also be aware that a flaky CD drive might introduce errors. As far as I know, those types of errors usually cause install problems though.

-You can check your systems memory from the live CD when you start it up, but select the option to check memory instead of to boot up.

-You just installed a new operating system. Did you mess with your system and possibly knock something loose? Try rechecking all internal connections and making sure add in cards and memory are seated properly and that cables aren't damaged and are correctly inserted. As an aside, you can clean the dust out of your system and make sure the fans are working properly.

-Make sure all cables are properly attached to the box.

-Did the live CD work OK? Does it crash? If not, you should be able to get a working installation. 

-You should probably check your disk and reinstall. Also, be aware that some computers handle certain brands and types of media better than others. Generally, to be safer, you can burn a CD-r instead of a CD-rw.

-Be aware that Dapper tends to be more stable than Edgy which is somewhat of an experimental release. That said, Edgy is a good release too.

-Related to the hardware issue, you might double check that you fan is working on your processor if you messed with the internals before installing.
