When you get to the stick figure/keyboard screen, hit enter, choose language, then choose keyboard layout if you want to with F3, but here's the important bit, use F6 and go through the boot options one at a time starting with **nomodeset**.

It is very likely that one of those options will get you to a live system desktop.
