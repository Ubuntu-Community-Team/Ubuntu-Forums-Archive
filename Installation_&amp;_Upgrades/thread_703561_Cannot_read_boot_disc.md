### Post by Existentialist on 2008-02-21
None of your hardware should cause the issue you are having.  I would check the md5 of the iso you downloaded to ensure it downloaded properly.  Or when you boot to the cd run the check cd for defect option which will check to see the cd burned properly.  If not, you probably need to turn the speed of the burner down when making the cd.  Being out of cds, you could try installing form a USB flash drive.  This link has more information on doing this:
[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)
