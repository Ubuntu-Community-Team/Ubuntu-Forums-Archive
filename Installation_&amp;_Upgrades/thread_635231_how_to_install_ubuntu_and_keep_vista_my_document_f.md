### Post by ajgreeny on 2007-12-08
So defrag your vista a couple of times to make sure it is properly defraged, then use the vista tools to reduce the vista partition as much as you can without messing it up totally.  Now install ubuntu on the free part of the drive, copy over the data files from the vista partition, and lastly delete the vista partition once you are happy that you've got all you want and need.  You may need to reinstall grub after removing vista but it's easily done with the ubuntu live cd.
