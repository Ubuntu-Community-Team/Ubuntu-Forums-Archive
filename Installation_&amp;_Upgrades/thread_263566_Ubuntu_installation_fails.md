### Post by Sef on 2006-09-23
> Are The Live CD and the one you download and burned by youself the same thing??? I have got them both, i have look into folders, it seems they are the same, but i may be wrong.

There are two ways to install Ubuntu (and Kubuntu and Xubuntu):

From a Live CD or from an alternate version.  Whether you get them from Ship-it or download and burn, they are the same. (Note: I think Ship-it might only send out a live cd now.)

The alternate version uses a text-based installer.  However, it is mostly just common sense to figure out or use the default.  The only real problem might be if you manually partition.  (Which can be a problem on the Live CD as well.)
