### Post by pointyblue on 2007-07-27
I've just installed Xubuntu from the[ alternate install iso]("http://cdimages.ubuntu.com/xubuntu/releases/feisty/release/xubuntu-7.04-alternate-i386.iso") on a 150 mhz computer with 190 mb ram and a 10 gb hd.

### Post by pointyblue on 2007-08-19
Solved!

sudo apt-get install linux-image-generic

updated the kernel and made everything work as it should.
