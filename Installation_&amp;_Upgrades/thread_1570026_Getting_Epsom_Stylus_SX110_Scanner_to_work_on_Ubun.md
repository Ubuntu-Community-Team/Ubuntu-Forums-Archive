Hi all.

I installed Ubuntu 10.04 as a replacement for winXP on my laptop (Dell Inspiron 6400) and everything is super fast and works from the word go!!

Except for my Epsom Stylus SX110 printer/scanner. The printer function works 100% but I am unable to use the scan function.

Simple Scanner gives the error message "No scanner detected" and I cannot find the scanner under the Preferences.

Can anyone give me advice Please?

Thank you

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Hi all.

I solved the problem and it seems to work for ALL Epson products. That is printers and scanners.

1. Install "Sane" from the Synaptic Package Manager.
2. Go to the Avasys website [http://avasys.jp/eng/linux_driver/](http://avasys.jp/eng/linux_driver/)
3. Download and install the correct driver for your version of ubuntu.
4. Download and install Imagescan from the avasys website.

All done!!! It works 100%
