### Post by Cybie257 on 2009-04-06
It kind of sounds like you are saying, "Is there a way I can get a brand new car without buying a new car".. 

But, I will start with this. What version of Firefox is installed on your computer? I'm thinking it should be 3.x.x as I think 8.10 had that as default, whereas 8.04 was 2.x.x..

If it's 3.x.x, it should update via System updates. If your computer is not setup to automatically download/notify of updates, you can invoke the update program by going to:

  System -> Administration -> Update Manager

This will download and (automatically) install any updates to software on your computer, including FireFox. 

Hope this helps for a start. Welcome to Ubuntu!! :)

-Cybie

### Post by Cybie257 on 2009-04-06
It will download AND install after you hit the Install Updates button on the update manager. You will see a list of all the available updates. 

When you load update manager, click the Check button to get the latest list of the updates and then select install updates. You'll see when you open it, it's really easy. :)
