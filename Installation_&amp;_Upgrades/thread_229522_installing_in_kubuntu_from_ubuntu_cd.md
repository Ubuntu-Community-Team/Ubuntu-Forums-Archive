### Post by aysiu on 2006-08-04
If you have the Ubuntu **Alternate CD** paste these commands in the terminal: ```
sudo apt-cdrom add
sudo aptitude update
sudo aptitude install gedit gcalctool
``` If you have the Ubuntu **Desktop CD**... well, it can't be done, then.
