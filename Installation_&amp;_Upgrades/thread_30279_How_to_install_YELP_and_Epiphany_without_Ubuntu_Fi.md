### Post by Burgundavia on 2005-04-28
If you want FF 1.0.3 get the version from Debian Unstable, not an autopackage. Autopackage currently doesn't interface with apt, although it may in the future. If you get that version, you will actually be able to keep yelp, as you are actually using a package manager for what it is good for, managing packages.
