### Post by Quackers on 2011-04-20
Ok. It may be that the installer is crashing.
Have you checked the md5sum of the downloaded .iso file?
[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

If that checks out ok you should check the cd for errors.
At the first purple screen (with the little man icon at the bottom) press any key. Choose your language then on the next screen you should see an option to check the disc for errors.
If it finds any errors at all the disc is no good.
The image needs to be burned again - try a slower speed.
