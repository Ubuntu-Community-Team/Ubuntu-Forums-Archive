### Post by logos34 on 2007-10-29
You files look fine.  Couldn't spot any obvious errors except:

-the line in fstab for your floppy is incomplete.  Edit.
[B]
gksudo gedit /etc/fstab[/B]

> /dev/[COLOR="Red"]fd0[/COLOR] /media/floppy0 auto rw,user,noauto 0 0

> What it seems to be is that the kernel isnt finding my drives, so it isn't able to load anything and just hangs at the loading bar.

There's a slight possiblity that gutsy and feisty are choking on mounting filesystems (libata driver confusion?)...just a guess.  You could try using UUID in place of 'dev/hda1':

**ls -l /dev/disk/by-uuid /dev/hda1**

then add it to fstab like the others.

Or else the kernels are hanging up on some other hardware detect issue, in which case you could try some of parameters listed here:
[https://help.ubuntu.com/community/BootOptions](https://help.ubuntu.com/community/BootOptions)

Someone else might have some suggestions.
