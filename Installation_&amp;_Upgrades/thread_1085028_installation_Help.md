### Post by Silvestro Fantacci on 2009-03-02
> **xCrusher said:**
> i don't need the install files, i already burned it to a disk.

There are two different installation CDs that can be used to install Ubuntu:

* Default desktop installation CD - runs a graphical installer
* Alternate desktop installation CD - runs a text-based installer

You are presumably using the first one.

What I would do first is to try once more with the CD you have burnt, this time selecting F4 "Safe graphics mode" before starting the installation.

If that does not solve the problem, I would burn and try the other CD.

Also note that your PC should have at least 384 MB RAM.
