### Post by snowpine on 2010-08-04
Hi Luis, the first thing I would try is to [burn a 10.04 Live CD]("https://help.ubuntu.com/community/BurningIsoHowto") (7.10 is no longer supported)) and use the "check for defects" option. Once you are sure you have a good burn, use the "Try Ubuntu with no changes to your computer" option for a while, make sure all of the hardware is supported. Only then would I attempt an install. 

If you post the hardware specs (especially the processor, RAM, and video card) it will help us help you. Good luck!
