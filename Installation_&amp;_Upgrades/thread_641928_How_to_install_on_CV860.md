### Post by windyweather on 2008-01-30
Once I had 512MB of memory on the system, since CD boot requires at least 384MB, it worked like a champ.

The screen size problem - size fixed to 800x600 and this being too small for install dialogs - did not show up with the CV860. The screen went to 1280x1024 to match my NEC 18" LCD monitor.
