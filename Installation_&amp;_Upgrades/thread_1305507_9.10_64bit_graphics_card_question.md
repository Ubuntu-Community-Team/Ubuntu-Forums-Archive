you can use envy

Installing Envy is very easy:
-> Open the terminal
-> Type &#8220;sudo apt-get install envyng-gtk&#8221; to install envy
-> Run envy from Applications>SystemTools menu
-> Select the Compatible and Recommended driver and then click on apply.
