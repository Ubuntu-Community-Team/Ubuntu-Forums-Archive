### Post by igknighted on 2006-10-08
The poster above is correct... During the install XP will erase the drive, but you have to specify it.  During the install it will ask you what partition you want to put XP on, just go one by one and delete every partion on the drive (follow the on screen instructions, it is very easy).  After you have deleted all the partitions, you will need to create a new one to use the whole drive.  Again, follow the onscreen instructions and accept the defaults Windows suggests.  After this, select the partition you just created and Windows will install.  Don't walk away, as there will be several wizards to click through though.  Best of luck!
