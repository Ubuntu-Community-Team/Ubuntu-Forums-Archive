### Post by por100pre1 on 2007-10-22
Type this in **Konsole** and press ENTER (if you don't find Konsole in the programs list press Alt+F2 and type konsole):

```
sudo aptitude install flashplugin-nonfree
```

For installing other programs use Adept Installer:

```
kdesu adept_installer
```
