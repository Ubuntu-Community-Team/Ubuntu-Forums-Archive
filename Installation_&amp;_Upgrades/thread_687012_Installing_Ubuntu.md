### Post by PmDematagoda on 2008-02-03
Unfortunately, the following step:-
> used winrar to unzip it, then burned it to a regular recordable disc 
is what caused the problem. 

You need to burn the ISO file itself, meaning to tell the CD burning software to burn an ISO to a CD, not as a data disc. The ISO also must remain intact with no changes done to it.

One more thing, it is highly recommended that you burn the Live CD at a speed of 4x or less since a speed higher than 4x could corrupt the Live CD.
