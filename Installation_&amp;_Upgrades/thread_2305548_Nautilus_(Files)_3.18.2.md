### Post by sudodus on 2015-12-07
Have you tried other file browsers. It is worth testing ***thunar*** and ***pcmanfm***. I have them installed alongside each other, and they do not interfere in my system. But if you remove nautilus, you may get problems, because I think it is part of your Ubuntu desktop.

If you want to play safe, you can test in a live system before you install.

***Edit:*** Another option is to use ***symbolic links*** for the locations you use often.
