### Post by GregBrannon on 2010-03-19
Eclipse is incredibly easy to install by itself.  Download the latest package appropriate for your hardware from the Eclipse project site, unpack the files into a directory to which you have full access (like \username\eclipse), and then run the eclipse program using the eclipse executable in the eclipse folder.
