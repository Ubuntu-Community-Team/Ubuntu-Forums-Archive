### Post by stoneybroke on 2008-02-15
Hi I had a similar problem to you I didn't get any error messages just a black screen after a lot of hair pulling the problem turned out to be NVIDIA and the LCD monitor Ubuntu needed the drivers installed for Nvidia first!!! and you can't do that. The only way I cured it was to set up Ubuntu 7.10 on a old monitor then install the drivers for Nvidia then change back to the LCD monitor

Using Amd 64 x 2 6400+

That might not solve the problem but it might be worth a try.
