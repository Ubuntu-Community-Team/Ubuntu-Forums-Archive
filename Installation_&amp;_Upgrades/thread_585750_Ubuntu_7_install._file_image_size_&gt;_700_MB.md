### Post by PacketCollision on 2007-10-21
Sounds like your download got corrupted.  The i386-desktop iso should be 695.8 MB (729608192 bytes).  The x86_64-desktop image is 696.8 MB (730691584 bytes).  Try downloading the iso again, and if you use FTP, remember to use BIN mode.  You can check your download by following the instructions here: [https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)
