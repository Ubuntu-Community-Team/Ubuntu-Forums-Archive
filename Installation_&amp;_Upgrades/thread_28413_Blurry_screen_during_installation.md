### Post by gizmospc on 2005-04-21
If you are able to get through the installation process, then follow this link (assuming your web access works) to get the proper VIA Linux driver.

[VIA Driver Websight](http://www.viaarena.com/Default.aspx?PageID=2) 

If you aren't able to get through installation, then I would try and find an inexpensive ATI or Nvidia card to get you by.  I suggest these because of their support of the Linux community, although I'm sure other card manufacturers also support Linux.  Ubuntu seems to have an easier time with the more mainstream video cards from my experience.
