### Post by Kalinda on 2007-03-08
There should be an Install icon on the desktop after you've selected the "Start or Install Ubuntu" from the menu on the initial screen.

After that it's really easy.. but you should have all your partitions worked out if you're keeping Windows or want to do anything fancy with your HD set up. I put my /home folder on its own partition, for example, with /home as the mount point.

There's probably instructions floating around here or on the official site for how to install it, you should have a look. It's really easy :)
