### Post by ubfan1 on 2016-12-30
Get into the UEFI settings and turn off fast boot, so you have a chance to hit a function key at power-on.  Now, there should be a function key (varies by machine, try F10, F12, F2, ESC, DEL, ... etc.) to find the boot menu, not the full setup.  This boot menu will give you a one time chance to boot something other than the default.

### Post by ubfan1 on 2016-12-30
At power on, there might be a note at the bottom of the post screen about which keys to press for setup, boot menu, etc.  You can pretty safely put the CDROM/DVD drive and USB before the hard disk.  That way, you can still boot Windows by default when no other device is present, but can still boot the USB and try Ubuntu.
