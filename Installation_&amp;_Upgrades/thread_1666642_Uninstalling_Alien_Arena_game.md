> **Maruthi.linux said:**
> 
but i now need to install and i dont know how to go about this... where that game installed and all

If you want to uninstall the game:
If you installed it from the Ubuntu repos, you can uninstall it via Synaptic or the software center.
If you installed it from source code, you will likely have to delete the directories the installer created. Most likely, the game will have installed to /usr/local/games/ or possibly /usr/local/. There will also be an executable launcher in /usr/bin/ or /usr/local/bin/.

Write back if you need more help.
FYI: If you ever install something from source code, it helps if you first find out how to uninstall it, in case you ever need to.
