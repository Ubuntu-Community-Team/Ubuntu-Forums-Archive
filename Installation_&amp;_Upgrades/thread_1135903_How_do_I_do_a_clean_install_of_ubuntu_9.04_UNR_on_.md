### Post by beanhead on 2009-04-24
Well you need to do more than download it to your flash drive. You need to install the usb image writer in synaptic packager. Just search image writer.
when you have that installed you will find it under accessories. Open it and browse for the Ubuntu Netbook Remix image you downloaded. Then select write to your usb key. Or usb stick. Then reboot hit F12 and boot from your usb stick. Then you can pick install that way have fun ! UNR IS GREAT

I have it running on an aspire one right now as im typing to you.
