### Post by canadianwriterman on 2010-09-21
Traditionally, I have done a clean install of Ubuntu with each new version. I reinstall all my files from a backup drive each time. I use Evolution's backup utility and Firefox's bookmark backup, which is great. However, there are some annoyances. For example, I lose all my remembered logins/passwords in Firefox and all the tag editing I did to song names from within Rhythmbox.

Is there a way to backup Firefox and Rhythmbox data. Would saving the "." files in the home directory achieve that? Or is there a Rhythmbox database file that I can save and reinstall after the clean install?

Thanks for your help.

### Post by lovinglinux on 2010-09-21
For Firefox you can use [FEBE]("https://addons.mozilla.org/en-US/firefox/addon/2109/") or simply copy the ~/.mozilla/firefox folder.

Firefox 4 will come with a Sync feature that will allow to sync bookmarks, passwords, tabs, history and settings, but you can install [the extension]("https://addons.mozilla.org/en-US/firefox/addon/10868/") in FF 3.6. See  [Firefox Backups](http://firefox-tutorials.blogspot.com/2010/05/backups.html) tutorial.
