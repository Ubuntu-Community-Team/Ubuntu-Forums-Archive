### Post by meng on 2006-12-12
Okay, to go back to the very basics, in order to have the install work you need:
1) Non-corrupted download of the ISO. Have you tried checking the md5sum?
2) Good burn. If in doubt (and if you have the spare media) go back and re-burn at lower speed
3) Good luck. Some systems just don't seem to get through the install from the LiveCD (which is what you've got). In that case, try again with the AlternateCD.

Your other option is to try the i386 version, which is what I have installed on my AMD64. I don't think you miss out on that much by using the 32-bit version. Others disagree, of course.
