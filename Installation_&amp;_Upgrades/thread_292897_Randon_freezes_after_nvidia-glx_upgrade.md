### Post by ulriks on 2006-11-12
Problem solved using tseliot's [Envy Script]("http://albertomilone.com/nvidia_scripts1.html")

and adding the following to the xorg.conf

```
Option "NvAGP" "0"
Option "RenderAccel" "Off"
Option "IgnoreDisplayDevices" "DFP,TV"
Option "NoRenderExtension" "Off"
Option "AllowGLXWithComposite" "Off"
```

As suggested in point 4 of the problems Section of tseliot's guide
