### Post by coyote on 2004-10-30
If you mkdir /home/yourusername/.mozilla/plugins

then install your multimedia players. i.e.  flashplayer, realplayer, mplayer, etc....
either hard copied or simlinked to the new directory

then install mozplugger

It will look in the new directory for installed plugins and Firefox will be able to open all related multimedia using the appropriate plugin.
