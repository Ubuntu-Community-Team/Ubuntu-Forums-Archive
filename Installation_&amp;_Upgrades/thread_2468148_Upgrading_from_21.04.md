### Post by grahammechanical on 2021-10-19
It will all depend on having Software & Updates set to notify you for any new version. A few days ago I upgraded an install of 20.10 to 21.04 and immediately I was invited to upgrade to 21.10. Software & Updates>Updates tab>Notify me of a new Ubuntu version>drop down menu>For any new version.
