### Post by markharding557 on 2007-12-26
upgrading will retain all your settings etc but it will install any default programs which you may have removed.
also occaisionaly upgrading can fail
the way i get around this is duelboot the new version on a different partition so i can take my time setting it up.
only when the new version is fully up and running would i remove the older one.
you can retian your settings etc by using a separate home partition
