### Post by logix2 on 2019-03-01
If you don't press any button when the boot starts, it automatically boots to "Try Ubuntu without installing". But you should still have an "Install Ubuntu" desktop shortcut that you can use to launch the Ubuntu installer. Like in the top left-hand side corner of [this screenshot]("https://i.imgur.com/0GMnDKA.png").
