 It's not. To install Ubuntu from an ISO simply set your bios (hit DELETE or F1 or any other key your computer may say at bootup IE: "Hit DEL to enter setup.") to boot from your cd-rom drive. Insert the Hoary cd and reboot. It should boot from the cd.

Another problem you may have is if you burnt the ISO directly. What I mean is, an ISO needs to be burnt with a burning program that supports an ISO. Simply putting the ISO file on the CD does not make it a Hoary CD. My personal windows preference for burning ISOs is Nero. They have a trial version that will do it for you on their homepage. If you need to re-birn it correctly, simply install Windows and Nero (I know, a lot of time!) and burn the disc. Then, reboot like described above. The Ubuntu installer should come up. Don't worry about formatting before hand, Ubuntu does all of that for you!
