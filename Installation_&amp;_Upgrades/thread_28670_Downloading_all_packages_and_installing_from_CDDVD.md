### Post by tonym on 2005-04-21
There are DVD images on the Ubuntu site that contain many packages  -  you could certainly download these and use for installs.

If you need EVERY package,  you could use debmirror to create a mirror of these parts of the archive you need and then back this up to DVD.  When you copy back to your home machine you would then add the disc archive to /etc/apt/sources.list.
