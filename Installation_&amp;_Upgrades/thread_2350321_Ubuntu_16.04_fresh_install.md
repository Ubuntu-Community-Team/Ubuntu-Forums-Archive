### Post by oldfred on 2017-01-23
If you have 2GB or more of RAM, the 64 bit version may be better. 
Few full systems in last 10 years are 32 bit, some tablets may be.

You can download the 64 bit version for free.
       Lots of info on USB boot for installing
[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)

If older system, gpu may not have enough horsepower to run full Ubuntu with Unity.
Lighter weight versions/flavors may then be better.

 [https://help.ubuntu.com/community/Lubuntu/InstallingLubuntu](https://help.ubuntu.com/community/Lubuntu/InstallingLubuntu) 
[http://xubuntu.org/](http://xubuntu.org/)
