### Post by grahammechanical on 2015-01-28
Ubuntu Touch has become Ubuntu for Devices and in a year or so it will simply be called Ubuntu. Well, that is the intention. In about 8 days time there should be an announcement about about the first Ubuntu phone by a Spanish company called BQ. And after that we are expecting another announcement of Ubuntu phones being released by a Chinese company called Meizu.

There is no such thing as "entry-level" Ubuntu Touch and it has never been the intention of the developers to provide ISO images of a Ubuntu phone OS for installation on each and every mobile device on sale. It was always the intention to provide a Ubuntu phone image for OEMs to pre-install on mobile devices for retail and that is about to come true.

As Ubuntu is Free and Open Source Software the code has been available for download but only for the 3 Nexus devices that were being used by the developers for testing the code. Even with one of those Nexus devices installing Ubuntu Touch has always been seen as an experiment.

Some community developers have worked to port the code to other devices. But those ports are not officially supported.

[https://wiki.ubuntu.com/Touch/Devices](https://wiki.ubuntu.com/Touch/Devices)

There is a porting guide but it is hopelessly out of date. A new guide is being worked on.

[https://wiki.ubuntu.com/Touch/DeprecatedPorting](https://wiki.ubuntu.com/Touch/DeprecatedPorting)
