Dear Community
I have now installed Ubuntu from the Live CD on two different laptops and I have notived that for both cases I got an error message at the Grub setup. Actually, the installation fails because of that. It fails even if I set to be (hd,x) to indicate which is the (grub)partition intended for Ubuntu.

The solution was simple. I installed openSuse first, their installation of Grub was working well. After that I could install Ubuntu from the Live Cd. 

I hope this can help someone.
