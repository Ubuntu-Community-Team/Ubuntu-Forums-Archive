### Post by zvacet on 2008-01-24
You have build-essential on CD.

```
sudo apt-cdrom add
```

```
sudo apt-get install build-essential
```

Or go to the system>administration>software sources>chek box with CD and close.Reload.Install build-essential.

**For network.**

Click on network manager icon>manual configuration<select your modem>properties>select your type of connection(dhcp or static)>close.In DNS tab you will find address.Delete it and add your nameservers.Go back to general tab and now tick your modem>window will pop up with changing network interfaces.When it´s finish close.

ppoeconf
