### Post by madtinkerer on 2008-04-26
Hi,

You can mount a cd image without a cd burner so you can access the files.  If you are using windows, try a program called daemon tools.  If you are in a Linux environment, try a program called gmount-iso.  Using these programs, you can access the information in an iso image and transfer it wherever you need.
