### Post by briandu on 2008-09-27
add the folowing to your sources:

goto /System/Admin/Sources and then goto 3rd Party Software.

goto Ubuntu Software and make sure 
tickbox for Proprietary Drivers..... is ticked and
tickbox for Software restricted ......is ticked
Add the following:

Type Binary
URI: [http://packages.medibuntu.org/](http://packages.medibuntu.org/)
/distribution: hardy
Components: free non-free

then /system/admin/update and nvidia will upgrade itself as well so you should be up to scratch.

Cheers
