Same here. Seems to be a bug:
[https://bugs.launchpad.net/ubuntu/+source/udisks/+bug/663815](https://bugs.launchpad.net/ubuntu/+source/udisks/+bug/663815)

Solution: umount/mount manually specifying the exec flag, or format drive as ext4.
