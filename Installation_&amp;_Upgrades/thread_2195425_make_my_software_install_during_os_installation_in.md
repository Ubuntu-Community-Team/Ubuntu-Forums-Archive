### Post by grahammechanical on 2013-12-24
For ease of use for ordinary people the Ubuntu install process installs a complete Linux distribution that will suit most ordinary users needs. We also have the Ubuntu Software Centre where we can, in one click, add and remove applications. So, for example, if we have no use for the small number of games that are default installed we can easily remove them. Ubuntu is intentionally designed not to give us a bewildering choice of install options.
