Hi,

BURG is an experimental project based on GRUB2. The menu system is completely redesigned and support configuration of screen layout and action using text file. I've uploaded a PPA binary package for ubuntu karmic, the address is:

```

deb http://ppa.launchpad.net/bean123ch/burg/ubuntu karmic main 
deb-src http://ppa.launchpad.net/bean123ch/burg/ubuntu karmic main 

```

Detail usage can be founded at:

[https://help.ubuntu.com/community/Burg](https://help.ubuntu.com/community/Burg)
