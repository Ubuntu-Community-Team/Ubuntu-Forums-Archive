### Post by LastDino on 2014-05-22
You need to use Bootable tool which has something like gparted. You can also use Ubuntu live season and install gparted on it to use for this purpose. 

You can't edit it by windows because it can't read ext4 and Ubuntu wont edit its own root partition when it is mounted. 

BUT before you do any of that, do take back-up.
