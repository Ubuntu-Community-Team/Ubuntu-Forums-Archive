Just put the Windows 98 CD in the optical drive and restart the computer. You'll need to boot off the Windows 98 CD the same way that you booted off the Ubuntu CD. (You may need to press the Escape key during boot to make the BIOS ask you which device to boot from). You will have the opportunity during the Windows 98 install to delete all of the partitions on the hard disk drive to use the entire drive for Windows if that is what you desire.

After the Windows installation has finished you will need to make sure that drivers for all of the hardware are installed. If some are missing you may need to find the drivers on the Internet.

Then run the Windows Update utility to download all of the security updates for Windows 98. This is really important if you plan to connect this computer to the Internet. You should also install an anti-virus scanner as well.

Even if you decide to run Windows, there is still a lot of great [Free Software]("http://www.gnu.org/philosophy/free-sw.html") that you can use like [Firefox]("http://www.mozilla.com/firefox/"), [Thunderbird]("http://www.mozilla.com/thunderbird/"), [AbiWord]("http://www.abisource.com/"), [7-Zip]("http://www.7-zip.org/"), [OpenOffice.org]("http://www.openoffice.org/") and [Gaim]("http://gaim.sourceforge.net/") to name a few projects.
