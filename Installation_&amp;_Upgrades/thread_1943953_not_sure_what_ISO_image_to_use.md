### Post by darkod on 2012-03-20
In the image name, there are basically two pieces of information to look for:
1. The "type" of the image, which can be: desktop, alternate or server.
The desktop and the alternate both install the same but the alternate has a text based installer and supports raid and lvm better. The server image is self explanatory.

2. The architecture type. i386 is for 32bit and amd64 is for 64bit. Don't get confused by the 'amd' word, it works on any 64bit CPU including Intel. AMD was the first to come up with 64bit cpu so the term amd64 remained.

You have all the ISOs here:
[http://releases.ubuntu.com/oneiric/](http://releases.ubuntu.com/oneiric/)
