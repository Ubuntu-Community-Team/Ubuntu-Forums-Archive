### Post by _simon_ on 2007-10-22
Check you can run both drives together first, I've heard of BIOS issues when trying to use a SATA and IDE HDD at the same time.

You'd be better off just using the SATA for it's quicker access times. If you want to keep /home separate you could just partition it.

Not sure how much you're paying for the 80GB SATA but 160GB aren't much more:
