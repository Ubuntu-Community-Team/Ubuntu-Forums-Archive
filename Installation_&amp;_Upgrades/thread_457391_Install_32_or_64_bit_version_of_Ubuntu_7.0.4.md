### Post by rsambuca on 2007-05-28
I have been installing both the 32bit and 64bit versions of ubuntu on my rig since Dapper.  With this last version (Feisty), I haven't ever used the 32bit system at all.  There is a little bit of effort to get some proprietory things working - namely flash and java with Firefox, but there are excellent How-to's, and it only takes an extra 10 - 15 minutes to set this up, and you only have to do it once.

I would recommend just going with the 64bit version, as you will inevitably run into a minor wrinkle whether you use 32 or 64 bit OS.
