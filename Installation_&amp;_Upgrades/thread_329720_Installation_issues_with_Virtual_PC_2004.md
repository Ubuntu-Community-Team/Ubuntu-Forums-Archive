### Post by Balaam's Miracle on 2007-01-02
Before i made the final switch to Ubuntu, i first test-drove it in a virtual machine for a couple of months.
I have tried installing it in Virtual PC, but that sucks so much, there was absolutely no way i could get it to work. I had the same results as you have.

Then i tried VMWare and that works great. Simply download VMware player from [here]("http://www.vmware.com/download/player/") and a ready-made Ubuntu VM from [here]("http://www.vmware.com/vmtn/appliances/directory/ubuntu.html") or choose one yourself from [here]("http://www.vmware.com/vmtn/appliances/directory/cat/45").
If you want to install Ubuntu yourself, you could download [this VM]("http://www.vmware.com/vmtn/appliances/directory/374"). 
Although it's designed to try out a Live CD without rebooting your PC, it might be possible to install Ubuntu (or any other OS) in the VM anyway, though i've never tested that.

I hope this helps.
