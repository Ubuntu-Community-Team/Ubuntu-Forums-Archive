### Post by TurkVU on 2008-10-15
Just building a new system with 2 SATA drives and having a problem getting the Ubuntu 8.04 install disk to find them. I tried the install CD several times and checked the connections, etc. But everytime the install gets to the Gparted window it comes up with no partitions detected.

The BIOS detects both drives just fine.

I was planning on making this system a dual-boot anyway, so my next step was to try installing Vista. It found the HD's and installed just fine.

I'm now ready to move on to getting Ubuntu running in the unallocated space on the first drive. Any ideas of how to get the install CD to recognize it??

---

### Post by TurkVU on 2008-10-16
Figured it out. Set the Access type in the BIOS to LBA and it found the drive.
