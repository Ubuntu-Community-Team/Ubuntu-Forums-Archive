### Post by howefield on 2009-10-23
> **jayanramesh said:**
> Is it possible to install ubuntu using Live CD If so please show me the way how to do.And could anyone tell me the difference between alternate cd and Live cd.

[https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall)

Download the iso file for the version you want and burn to cd, then boot your computer from it. You can either boot to a desktop and install from there or choose to install from the cd boot screen.

[http://www.ubuntu.com/GetUbuntu/download](http://www.ubuntu.com/GetUbuntu/download)

Click on "Text based &#8220;alternate installer&#8221; bottom right hand corner for a brief description and download links.

You'll also find a few relevant screencast videos which will help at

[http://screencasts.ubuntu.com/](http://screencasts.ubuntu.com/)
