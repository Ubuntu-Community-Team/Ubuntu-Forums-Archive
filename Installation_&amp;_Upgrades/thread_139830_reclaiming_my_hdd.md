### Post by andrewsawyer on 2006-03-04
This isn't really an Ubuntu question, however...

Start >> Control Panel >> Administrative Tools >> Computer Management >> Disk Management.

Do that from within Windows XP.  This will load up a program that will give you a view of all your hard disks.  You can then wipe out any partitions you like, even if Windows doesn't recognise the partition type.
