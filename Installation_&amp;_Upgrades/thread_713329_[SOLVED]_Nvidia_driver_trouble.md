### Post by Whiffle on 2008-03-02
Your xorg.conf isn't configured to use the nvidia driver, so you'll have to edit it:

```

sudo gedit /etc/X11/xorg.conf

```


Now, change this line:
```

Driver "nv"

```

to 
```

Driver "nvidia"

```

then save it, and try to modprobe again.  It should work this time (assuming you have the nvidia driver installed, which if you don't, try the envy link above)

### Post by Pumalite on 2008-03-02
8800GTS has questionable support. You can go to the Nvidia site and download the latest driver: 
[http://www.nvidia.com/Download/index.aspx?lang=en-us](http://www.nvidia.com/Download/index.aspx?lang=en-us)
Install it through a virtual Terminal:
Ctrl+Alt+F1
username
password
sudo /etc/init.d/gdm stop
sudo sh /path/to/driver/NVIDIA-Linux-x86xxxx.run
Accept the License
Let the driver compile the module into your kernel
Let the driver reconfigure your xorg.con file
Then 
sudo /etc/init.d/gdm start
Or:
startx
Make sure you have build-essential installed before trying this:
sudo aptitude install build-essential
