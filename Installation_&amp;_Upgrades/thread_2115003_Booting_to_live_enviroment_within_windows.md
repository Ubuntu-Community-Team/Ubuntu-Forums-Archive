### Post by C.S.Cameron on 2013-02-11
It is not a problem to put an Ubuntu virtual machine on a pendrive and run it from Vbox installed in Windows, (and vice-versa).

I think there is also a portable version of Vbox for windows.


Edit:

[http://www.vbox.me/](http://www.vbox.me/)

Be careful though, I recall running something like this a few years ago and it messed up the version of Vbox that was installed on the machine.
