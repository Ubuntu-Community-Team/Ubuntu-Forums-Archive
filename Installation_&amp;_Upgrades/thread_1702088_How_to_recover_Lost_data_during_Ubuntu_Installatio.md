### Post by Mark Phelps on 2011-03-07
If you've installed Ubuntu OVER WinXP, anything you do to the Ubuntu install (such as installing data recovery apps), is going to worsen your chance of recovering the data.

You would need to be able to remove the drive, connect it to another PC, and do the data recovery from there.

If you can do that, you have a couple of choices:
1) Use Linux apps to do the recovery -- testdisk and photorec come to mind
2) Use Windows apps to do the recovery -- GetDataBack and Recovery My Files come to mind.

My own experience has been that the second option has had a lot better luck recovering Windows files and paritions than the first -- but those are not free.  You can download trial versions from Runtime Software to see what they can find, but you have to buy them to do the actual data recovery.
