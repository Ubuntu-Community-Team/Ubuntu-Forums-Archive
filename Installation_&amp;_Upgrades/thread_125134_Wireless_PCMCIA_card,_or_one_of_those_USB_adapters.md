### Post by bernardfrancois on 2006-02-03
I bought a Cisco Aeronet 350 series PCMCIA card trough ebay. I didn't have to install anything, just put it in the PCMCIA slot and start my computer.

I chose that card because - on some compatibility list - it was the only card for which the manufacturer made drivers and utilities for Linux. But in fact I never installed those (but its still good so support manufacturers that have some Linux support, just to show hardware manufacturers that consumers care about Linux support).

[edit]
