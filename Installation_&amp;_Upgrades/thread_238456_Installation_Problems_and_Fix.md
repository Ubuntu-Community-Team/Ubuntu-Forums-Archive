### Post by Greenparot on 2006-08-17
I'm new to Linux.  I have been trying out all the different Linux OS distributions out there (Mandriva, SuSE, PCLinuxOS, etc..) and liked what I saw with Ubuntu.

I burned the live cd and booted from it.  The OS had all the features I needed and I loved the add/remove application capabilities for a Linux noob like myself.

When I did install Ubuntu I ran into errors a couple of times.  The install would hang/stall around 80% to 93%.  So obviously something was wrong.

After reading a howto on installing Ubuntu ([click here]("http://psychocats.net/ubuntu/installing.html")), I mounted the /windows on my Windows partition and made sure I had a swap partition.  After doing this, the install worked flawlessly.

I am now running Ubuntu off my 2nd HD partition and loving this OS.

I just wanted to post this in hopes of being a help to other Linux/Ubuntu newbies like myself.
