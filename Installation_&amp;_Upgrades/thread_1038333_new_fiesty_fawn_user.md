### Post by whoop on 2009-01-12
Feisty Fawn is outdated and not supported anymore (it was released beginning 2007).

If you are running a fresh install of feisty fawn you might want to consider reinstalling it with a more recent version (Hardy or Intrepid)

[https://wiki.ubuntu.com/Releases](https://wiki.ubuntu.com/Releases)

If you want and to upgrade your existing feisty, you are welcome to try. It will not be easy cause feisty is outdated and not an LTS release.
So to get the most recent install you will have to update to Gutsy, then Hardy and then Intrepid.

Once you have upgraded or reinstalled your linux to a recent version firefox 3 will be intalled by default.
