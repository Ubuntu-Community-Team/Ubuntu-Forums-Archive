### Post by Wim Sturkenboom on 2012-10-11
If this is a Win98 system, it's more than likely too old to run current Ubuntu. Not enough memory, slow processor etc. To make sure though, post the specs.

There is a program called [plop](http://www.plop.at/en/bootmanager/download.html). You can install it on a floppy (or CD), boot from it and it will allow you to boot from USB, even if the BIOS does not support booting from USB.
