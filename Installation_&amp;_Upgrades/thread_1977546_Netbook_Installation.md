### Post by sudodus on 2012-05-10
I suggest that you try some of the *ubuntu flavours live from a USB  boot drive before you start installing something.

- Xubuntu with the light desktop environment XFCE
- Lubuntu with the ultra-light desktop environment LXDE

As you probably know, there is a light version of KDE with Openbox available in Kubuntu. (I think both the standard desktop environment of Kubuntu and Ubuntu Unity are too heavy at least for not-quite-new netbooks.)
