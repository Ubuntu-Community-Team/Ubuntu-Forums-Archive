### Post by mikewhatever on 2009-10-05
There are several partitioning modes, but the ones relevant to you are guided and manual. The guided mode will ask you which partition to resize and how much space should be allocated to Ubuntu. The manual mode is where you partition yourself, thus having more control over which partitions are created and of what sizes. If you pick the latter, the file system is ext3. The installation partition will obviously be formatted during the installation.
