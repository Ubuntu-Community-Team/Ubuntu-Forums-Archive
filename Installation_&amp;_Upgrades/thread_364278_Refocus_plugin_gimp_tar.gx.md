### Post by compdata on 2007-12-19
all i had to do was to use apt-get install to install refocus.  

I ran
sudo apt-get install gimp-refocus
sudo apt-get install gimp-ufraw         (for ufraw raw conversion plugin). 

They seem to be working fine.
