### Post by gsmanners on 2010-01-03
I don't think Ubuntu automatically resolves third-party repository updates. You're going to need to check those yourself.

For example, if you used Medibuntu, check the following page:

[http://packages.medibuntu.org/](http://packages.medibuntu.org/)

Since they indicate support for karmic, you may then update that source. Make sure you follow any instructions they give as those may change from one release to the next.
