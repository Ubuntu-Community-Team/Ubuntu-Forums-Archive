### Post by 23meg on 2006-12-27
The partition manager will prompt you to create a Linux partition when installing. You don't have to set it up in advance.

You can read files on NTFS partitions. Writing is also supported, but write support isn't installed by default since it's still not regarded 100% safe and stable.
