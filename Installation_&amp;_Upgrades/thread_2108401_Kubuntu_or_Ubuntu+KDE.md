### Post by tartalo on 2013-01-24
If you intall KDE on top of regular Ubuntu you will have Ubuntu's software selection + KDE's software selection, so you will have two Window Managers, two File managers, two text editors and so on and so on.

If you install kubuntu-desktop on top of regular Ubuntu you will have even more duplicates (kubuntu-desktop includes KDE plus more).

I'd go for Kubuntu + Your stuff.
