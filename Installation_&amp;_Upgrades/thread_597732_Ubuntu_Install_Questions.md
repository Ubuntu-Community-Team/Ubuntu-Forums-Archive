1.  Feisty released last April so it's 7.04 while Gutsy released early this month so it's 7.10, the latest version.

2.  You can install GRUB to MBR (Master Boot Record) to boot both Ubuntu and XP.

3.  It can detect and install on SATA drive.

4.  Ubuntu comes with Gnome while Kubuntu comes with KDE but you can still install KDE and use it even if you use Ubuntu CD to install it, and vice versa.

5.  Look at #2.
