### Post by wjp.reg on 2006-10-29
Did you notice that the Automatix repos section duplicates the repos set out above it.

I would delete the Automatix repos START|END section altogether, save it, and then sudo apt-get update and everything should be right.

In case you weren't aware, you need to edit the sources.list with sudo, supervisory priviledges.

cheers
