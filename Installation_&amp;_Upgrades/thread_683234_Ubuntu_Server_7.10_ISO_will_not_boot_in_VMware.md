### Post by upfwnv03 on 2008-01-30
There is a free CD/DVD burner out called 'InfraRecorder'. I use it on my XP box. As Shazaam stated you'll want to make sure you burn the .iso as a 'CD Image' or 'Disc Image'. In InfraRecorder they list it as 'Burn Image' and the default dialog box opens to *.iso or *.cue type files. You can find infrarecorder at 

infrarecorder.sourceforge.net

### Post by upfwnv03 on 2008-01-30
I forgot one other tool. Try checking the MD5 sum. This will let you know if the iso is corrupted or not.
On Windows XP I use 'winMd5sum' from Nullriver Software. 

[www.nullriver.com/index/products/winmd5sum](www.nullriver.com/index/products/winmd5sum)

They list it under 'Open Source / Freeware'
