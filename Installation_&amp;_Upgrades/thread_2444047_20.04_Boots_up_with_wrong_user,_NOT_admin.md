### Post by CelticWarrior on 2020-05-24
Is your wife's account set to autologin without password? If so it's doing what it has been told to do. There's no "reboot as root", there's just reboot. Set as it should, i.e., no autologin, it'll reboot to the greeter where users choose their own account and login with their password.

In any case, this has nothing to do with Virtualbox or virtualizaton. "AMD-V disabled in BIOS" is exactly what it says, a BIOS/UEFI setting that should be enabled for virtualization.
