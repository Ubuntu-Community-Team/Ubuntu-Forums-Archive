### Post by mikewhatever on 2008-02-15
You might want to leave some free space for the Vista partition, so about 10 GB is fine for Ubuntu. (A regular Ubuntu installation only takes up 2.4 gb.) You can use the external drive and Vista partition for storage, since Ubuntu 7.10 can both read and write ntfs.
No, your Windows installation will not get chopped up. To save yourself time and trouble, follow one of the dual-boot guides.
