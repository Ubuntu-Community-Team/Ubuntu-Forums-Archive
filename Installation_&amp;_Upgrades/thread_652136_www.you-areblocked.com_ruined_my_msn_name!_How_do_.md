### Post by Herman on 2007-12-30
Firefox is the best web browser. 
Firefox for Windows is popular with Windows users for it's lack of Internet explorer vulnerabilites. 

To set Firefox to have more features, we can add 'plugins'. 
To add a 'plugin' to Firefox, you click 'Tools;, and then click 'Add-ons', (right down at the bottom of the drop-down menu).

Then in 'Add-ons', you'll see three icons: 'Extensions', 'Themes', and 'Languages'.

In the 'Extensions' window, you'll notice two hyperlinks in the right-hand bottom corner. One says 'Get Extensions', and the other one says 'Get  Ubuntu  Add-ons'.

I went into "Get Extensions", and then clicked the link in there titled 'Privacy and security and there is  a whole  16  pages of  software that has been made fro you to add to your Firefox Web Browser.  :)
I found one on page 2 that might help you a little, here is a link for it, It's called '[**WOT**]("https://addons.mozilla.org/en-US/firefox/addon/3456")'.
You can try that out if you like. There are probably a lot more in there and they're all probably good.
Here's another one that looks good, it's called [**iTrustPage**]("https://addons.mozilla.org/en-US/firefox/addon/3661"), and it looks like that would help you a lot too.
And here's another one! 
**[netcrafttoolbar]("https://addons.mozilla.org/en-US/firefox/addon/1326")**

All those will probably be a big help, but you'll still have to be more careful yourself of course. 
Take a good look around through there for yourself, I didn't go all the way through yet. Likely you'll find a few more.
You might see other things in there not related to phishing that you might like too. :)

Regards, Herman :)
