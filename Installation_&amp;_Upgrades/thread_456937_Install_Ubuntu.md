### Post by ep2011 on 2007-05-28
It's very simple to install a Dual boot system using the LiveCD. I've done it twice with no problems, and many other people do without problems as well. All you do is when you are up to the partioning part of the Install, choose either manual or have them resize the partion (I always like manual better). If you do manual, just make sure you have a / partion with at least 6 GB, and a swap partition of about 1 GB. (I always like to have a seperate /home with as much space as you can spare, as this makes reinstalls easy).

Go to [http://www.psychocats.net/ubuntu/partitioning](http://www.psychocats.net/ubuntu/partitioning) and [http://www.psychocats.net/ubuntu/installing](http://www.psychocats.net/ubuntu/installing) for more information.
