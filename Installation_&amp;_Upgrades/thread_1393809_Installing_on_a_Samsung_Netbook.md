### Post by darkod on 2010-01-30
And what is happening exactly when you select Install? I have installed both UNR 9.10 and Desktop 9.10 32bit on NC10 and it works fine.
Also, have you tried the usb stick on another computer just to make sure it's working fine and the installation process can start there?

Another thing. It is usually better to shrink the win7 partition using the windows disk management rather than ubuntu installer. If win7 is taking your whole hdd, without any unallocated space, you need to shrink it first to make unallocated space for ubuntu. And after the shrink boot win7 few times to do its disk checks.
