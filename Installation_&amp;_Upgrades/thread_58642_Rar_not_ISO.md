### Post by heimo on 2005-08-21
I believe you have file compression program associated to .iso file extensions and you're seeing rar or winrar icon on that file? You can open your regular burning software and browse to that file, be sure to burn from image instead of burning that file to disk. On some burning programs you just drag'n'drop it to list of files to burn and program will ask this question, on others - you'll select this from menu or at the beginning when you select what you're going to burn (data disk/music disk etc).
