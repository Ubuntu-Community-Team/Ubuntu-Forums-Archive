### Post by Mark Phelps on 2010-11-04
There is no System Restore capability built into Ubuntu as there is in MS Windows. Instead, we handle backup and restore using add-on products.

Personally, I've found Clonezilla to be a good tool for doing backups and restores -- but it does a whole drive or partition, it doesn't just do changes.

Other will recommend remastersys -- but I haven't used that.
