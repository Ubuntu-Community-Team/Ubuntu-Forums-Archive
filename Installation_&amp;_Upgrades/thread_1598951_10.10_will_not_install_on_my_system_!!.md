### Post by sikander3786 on 2010-10-17
First of all check the image for defects.

[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

Secondly, try to burn the disc at the lowest possible speeds.

Make sure that your CD-ROM Drive is in proper working condition. Can you boot some previously burnt discs with it?

Sometime master/slave jumper configuration on the drives also tend to cause problems like this.

My last shot would be aiming for a memtest. Bad RAM modules don't cause that error too often, but I have seen it happen at least twice.
