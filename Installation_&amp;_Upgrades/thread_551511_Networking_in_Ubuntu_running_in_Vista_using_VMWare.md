I'm trying to install Ubuntu 7.04 as a guest OS using VMWare in Vista Home Premium, on a Sony VAIO VGN-FZ11L.

Everything has installed absolutely fine, but I can't get internet access in Ubuntu. I've set up Ubuntu to go for Host-only networking and have shared by Wifi connection in Vista. But I can't connect to any websites in Ubuntu.

---

### Post by maybeway36 on 2007-09-15
Use [VirtualBox]("http://www.virtualbox.org") instead, it's more optimized for Linux VMs.

---

### Post by jim0203 on 2007-09-15
Thanks so much - have just run the Live CD in VirtualBox and networking works straight out of the box! Or straight out of the VirtualBox, if you will... :-)

For the record I had tried Parallels and VMWare and neither of these got the networking sorted, so I'd recommend to anyone else to use VirtualBox.
