Ultimate boot 503 or ver 5.0.3

This is a good OS boot utility CD ISO image. You can download this for free. It contains a number of utilities and function. For example you can boot DOS and go to the C drive to load up Windows 9X. You can load Gparted to edit the partitions. You can run Memtest86 to check your RAM. A later version of this compilation is being sold for money but this one is free.
