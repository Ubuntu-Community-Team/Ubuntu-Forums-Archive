### Post by drascus on 2007-01-23
Hello well I upgraded from dapper to Edgy. When I did my Wireless card stopped working. This was quite annoying because it took me forever to get it working in the first place under ndiswrapper. So here is my post for all of you who decided to upgrade and have the same problem. Well it seem thtat this upgrade did something or other to my ndis libraries it must of had to remove some or something. I am not that technical so here is how to fix it. Go to synaptic and make sure that you have all the utils packages installed and the ndis common package installed. When you have this done restart. All should work as it did before. After I did this I had all my network settings still there and everything was comming up roses. I hope this helps someone. 

Card :broadcom wireless air force one 54g 802.11g wireless Lan controller.
Laptop: Compaq  2000
