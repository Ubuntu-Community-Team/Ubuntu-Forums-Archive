### Post by fdrake on 2011-08-15
you can download the files in 1 computer and put them in a shared folder on the server. and make the other computer upgrade using the files contained on the "server pc" thought a predefined user-name account.

see link below on how to:
[https://help.ubuntu.com/community/Installation/LocalNet](https://help.ubuntu.com/community/Installation/LocalNet)

ps.  I do not suggest you to upgrade from a stable-long-term release(ubuntu 10) to an unstable-short term release(ubuntu 11). you will encounter in the future problems and complications in which none will be able to help with since it will not be supported.
if you're computer is doing fine like that there is no reason to upgrade unless you have serious problem with the hardware.
