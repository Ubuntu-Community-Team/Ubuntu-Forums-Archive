The combination of your BIOS/motherboard and the USB enclosure is not a happy one. Unfortunately, that happens a lot. For example, I have a Patriot USB stick that I use to boot several live CDs. Works perfectly on dozens of computers, except a Dell Vostro 1000, where it just doesn't boot.

Things to try:

- Use eSATA if your system has it
- If you use USB 3, try to use an USB 2 port instead
- try a different USB enclosure
- double-check BIOS settings
