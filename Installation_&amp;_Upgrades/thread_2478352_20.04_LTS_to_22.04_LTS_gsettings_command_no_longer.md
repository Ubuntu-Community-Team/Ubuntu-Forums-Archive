### Post by AlexHudghton on 2022-08-24
I am using this script to change the display background on a daily on-reboot schedule

#!/bin/bash

cd /home/alex/Wallpapers

DAY=`date +%j`

TYPE=".jpg"

BKG=$DAY$TYPE

gsettings set org.gnome.desktop.background picture-uri file:////home/alex/Wallpapers/"$BKG"


Since the upgrade the gsettings line does nothing. No errors reported.

Any clues appreciated

Alex

---

### Post by tea for one on 2022-08-24
Via dconf editor, there are now two settings:-

/org/gnome/desktop/background/picture-uri
/org/gnome/desktop/background/picture-uri-dark

It seems to depend on your choice of Light or Dark Style in Settings > Appearance.
