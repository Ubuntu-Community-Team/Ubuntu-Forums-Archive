### Post by Sef on 2006-01-11
Sounds like you need to set the BIOS to boot from the CD-ROM.  To do that, when your computer first starts up, it should say something like 'press X to enter set up (or BIOS.)  It will tell you X.  X usually is F2, delete, or ESC.  One there find the boot page and set it up like this:  Floppy, CD-ROM, HD.
