### Post by netrattler on 2005-06-22
[QUOTE=kona0197]By the way - how do I edit the names of things in my "start" menu or change the progams icons? Only other thoing I need to know is how to add things to the desktop - like my HDD for easy access. And oh yeah - is there a dock made for GNOME? And where can I set GNOME to turn off the monitor after 5 minutes? THANKS!!  :)[/QUOTE]

1. Editing things in your Applications menu can be done by a program called smeg once you install it, after it's installed it can be located at Applications -> System Tools -> Smeg Menu Editor.

2. Here is how to make a link on your desktop of the / directory. Right click on your desktop -> Click on Create Launcher -> Name put what you want -> In the Command: field type file:/// -> Type change to Link -> Icons click the browse button and select /usr/share/icons/gnome/48x48/devices/gnome-dev-harddisk.png -> Click Ok -> Click Ok -> Test your new link out and it should take you to the / directory. If you want another directory like your home directory than replace file:/// with file:///home/username

3. Not sure what you want by dock, do you just want to add programs to the panels or extra features like weather and the such.

4. Monitor Click on System -> Preferences -> Screensaver -> Click on the Advanced tab -> Change the Display Management to what you want.

Joe
