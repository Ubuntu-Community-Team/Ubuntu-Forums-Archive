### Post by Zafrusteria on 2008-01-02
That would probably be the Windows binary :-)

Have a look at KeypassX which you can install with Synaptic same program just ported to Linux. Or download a slightly more up to date version from 

[http://www.keepassx.org/](http://www.keepassx.org/)

Just get the .deb file and use the tool GDebi package installer once its downloaded. The GDebi installer should be in Applications-> system tools.
