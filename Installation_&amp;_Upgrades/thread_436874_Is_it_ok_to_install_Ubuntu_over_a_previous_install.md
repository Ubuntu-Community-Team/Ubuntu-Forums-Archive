Yup, you can easily pop in the Ubuntu live or alternate cds and then when you get to the partitioner just mount your root again and make sure right under it says (format and keep existing partition or such). Just a note but Ubuntu only really needs 6-8 GB of memory, I'd make a 32 GB partition for /home, then all your user data will be saved if you ever reinstall or have a problem.

I was gonna recommend ya just install the Ubuntu desktop but if you want to repartition and your getting a bit of instability from Kubuntu you may want to do a clean install just to be sure.

As for Ubuntu being more polished, I wouldn't say that its really just a different way to do things. I don't really think that wireless setups are THAT much easier on Ubuntu. They are really only seperated by gnome/kde and the preinstalled apps.
