### Post by ajgreeny on 2016-01-20
You can upgrade from either the current LTS (14.04) or the most recent version (15.10) to the next LTS version (16.04) so if you are certain that you will move to 16.04 soon after it's released it will make little difference which you choose now.

I have not used Lubuntu 15.10 so I'm unaware of the bugs that it may suffer from, but I do have Lubuntu 14.04 on an old netbook and it is very fast and stable, and I believe is an excellent OS for that type of hardware.  Your hardware would also be able to run Xubuntu if you like the XFCE desktop.  It is similar in general layout, but much more configurable in my opinion.
