What version of Ubuntu?

To rule out these issues, check the md5sum and check the cd/usb for defects.
[[COLOR="Red"]HowToMD5SUM[/COLOR]]("https://help.ubuntu.com/community/HowToMD5SUM")
[[COLOR="Red"]CDIntegrityCheck[/COLOR]]("https://help.ubuntu.com/community/Installation/CDIntegrityCheck?action=show&redirect=CDIntegrityCheck")

I'm not familiar with the program you used to create your usb, but Unetbootin is a really good one.
[[COLOR="Red"]Unetbootin[/COLOR]]("http://unetbootin.sourceforge.net/")

Also, when you are able to boot it, you should select "Try Ubuntu" to make sure it will run well with your hardware, before installing.
