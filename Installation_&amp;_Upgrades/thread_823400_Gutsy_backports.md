### Post by mxg01 on 2008-06-09
I have Gutsy on two machines and I've taken a couple of specific things from the backports repository but that was only because the Gutsy versions didn't do what I wanted. I enabled the backports repository, got what I needed and disabled them again. I don't see any reason to install backports unless you need something you can't get in your current version. So, I'd leave them alone. Gutsy is still getting security updates and will be supported for a while yet.
