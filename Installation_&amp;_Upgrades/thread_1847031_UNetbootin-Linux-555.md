I am running with Ubuntu 10.10 and was interested in checking out 11.04.

I downloaded the 11.04 ISO image and installed UNetbootin to create a bootable USB (8gig).

I downloaded the DEB version of UNetbootin. Installed fine. Created the bootable USB. Rebooted from USB and watched the "loader" move back and forth. The screen then went black and all HD and USB activity stopped.

I then thought, maybe a glitch in the DEB file. Downloaded the BIN version of UNetbootin. Short story. Same result.

Finally used the "Create Startup Disk" via 10.10's "System" menu. Ran like a charm. USB created and rebooted. I am using the "Try" version of 11.04 to post this message.

Has anyone else come across this problem?
