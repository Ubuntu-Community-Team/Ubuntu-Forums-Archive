### Post by az on 2007-04-16
Use the alternate installation cd.

But the alternate installer will take much longer to complete the job than the Desktop cd.  The desktop cd copies the installed system from the disk to the new system.  The alternate installer installs every single package one-by-one onto the new system.

It would be cool if there could be a console-only script that would do the same thing as the live cd installer, but not from within a live session.
