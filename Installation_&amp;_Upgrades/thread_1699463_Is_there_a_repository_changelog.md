### Post by grahamjenson on 2011-03-03
I was wondering if I could get changelogs of the repositories (when a package is added, removed, etc.). This would be useful to see how a repository evolves over time, and look at how packages change.
I figured that they should exist, but are probably not generally useful so are not publicly available.

I can get changelogs of individual packages (from [http://changelogs.ubuntu.com/](http://changelogs.ubuntu.com/)), but these do not include information like adding a package to update repos, only the proposed repos, probably because this is an internal process.

Also sometimes the versions are renamed, from what exists in the changelog to what exists in the repository. This is a disconnect that makes it difficult to follow what is happening.
