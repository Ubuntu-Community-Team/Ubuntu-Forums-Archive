### Post by starcraft.man on 2009-06-19
That's pretty strange. That drive must really be messed up to not show up in the partitioner. I'd recommend gparted live found [here]("http://gparted.sourceforge.net/download.php") Download the live stable release. If you need a guide, look at the documentation section under english html section and read a few pages.

Just burn it on a CD from a clean computer and then boot it, select defaults and then you'll be in a live partitioner. If it doesn't show up there, well that must be something more.

I'd also check the cables to the drive, and that it shows up in the bios if not already done. If you get any trouble, reply.
