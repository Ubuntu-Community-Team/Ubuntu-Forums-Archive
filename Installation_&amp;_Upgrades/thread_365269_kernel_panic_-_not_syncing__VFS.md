### Post by tgalati4 on 2007-02-22
Perhaps the power supply is weak.  It's the only component that has not been replaced and it would cause unexplained errors.  I assume the machine is several years old.  

Ubuntu doesn't work well with failing hardware.  With Windows, you couldn't tell if Windows was crashing or if the hardware was failing.  You would get blue screens either way.

Good luck!
