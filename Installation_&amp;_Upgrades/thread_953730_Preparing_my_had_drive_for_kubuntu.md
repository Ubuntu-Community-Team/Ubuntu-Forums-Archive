### Post by richard.e.morton on 2008-10-20
Hi There,

These days I would consider doing it a different way. 

When I dual boot I alsways find that the next thing I want/need to work on requires me to reboot into the other environment, cause of a bit of data, an application or some sort. 

I fixed this by installing Ubuntu/Kubuntu in a virtualised desktop with something like virtualbox, which is free. over time convert more and more apps into your preferred environment (hopefully Kubuntu), then next time you reinstall Windows just install it the other way around, Kubuntu as your prime environment with a virtialised Windows environment. Eventually hopefully you'll be able to do away with Windows altogether.
