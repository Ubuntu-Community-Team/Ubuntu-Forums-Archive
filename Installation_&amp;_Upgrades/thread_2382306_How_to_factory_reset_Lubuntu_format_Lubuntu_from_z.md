### Post by sudodus on 2018-01-11
1. You can make an OEM installation of Ubuntu. See this link,

[Ubuntu_OEM_Installer_Overview](https://help.ubuntu.com/community/Ubuntu_OEM_Installer_Overview)

It works with Lubuntu too (according to the instructions in the link above).

Edit: 

1. You can log in and do some tweaking (if you wish).

2. The OEM system puts an icon on the desktop 'Prepare for shipping ...'. It starts a program, that makes the final preparation start after rebooting. So when you are happy with the system, you run 'Prepare for shipping ...', and the system is ready for the buyer.

[hr][/hr]
2. Otherwise you can wipe the drive and then provide Lubuntu in a cheap USB pendrive, and instruct the buyer how to install it.
