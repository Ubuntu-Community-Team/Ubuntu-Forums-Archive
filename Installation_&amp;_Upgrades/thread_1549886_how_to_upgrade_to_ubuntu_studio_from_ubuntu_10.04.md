### Post by bergfly on 2010-08-10
Open your package manager - Synaptic and then do a search for ubuntustudio-desktop. Select this for install and click apply. This should install all the needed files. 

Quick note, this will change the appearance of the desktop a little and move some things around. Everything presently installed will still be there, but in a bit of a different place maybe.
