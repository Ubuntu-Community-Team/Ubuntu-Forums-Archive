### Post by ichigo on 2008-05-19
Boot to ubuntu and insert your CD into your drive. As far as I remember u will then be asked if you want to upgrade... And then, well, the rest should be obvious...:popcorn:


EDIT: However, if you wan a clean install instead of an update, you could install ubuntu from CD (as u would if ther was no Linux on the disk) and choose to overwrite your Linux partition. Be careful to choose the right one, because choosing the wrong one would probably mess up your whole system.

### Post by barnex on 2008-05-19
You might also want to make sure that "Release Upgrades" are set to "Normal releases" or "LTS releases", under the "Updates section" of the Software Sources program (System/Administration/Software Sources). If you are not prompted to update upon insertion of the CD, you can start Update Manager manually.
