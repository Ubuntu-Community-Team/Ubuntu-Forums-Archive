### Post by sofasurfa on 2007-04-24
The main difference I've noticed (I use Xubuntu) is that if you connect to a WPA wi-fi network using ndiswrapper then Edgy gives a rock solid 100% signal strength connection every time. My experience of Feisty is that successful connection is extremely rare. If a connection is successful then the signal strength is generally 58% and prone to dropping. This both with a static PC (lives under the TV) and a laptop which moves around to different locations in the house. This was bad enough for me to wipe the disks and go back to Edgy.

### Post by the8thstar on 2007-04-24
There is a bit of a comparison between Edgy -> Feisty and XP -> Vista, as we are talking about upgrades in both cases.

Generally speaking, the upgrade to Feisty is easier than to Vista, as the OS will not require you to have a "race car" computer with at least 2Gb of RAM and 5Ghz dual core processors to run smoothly.

There have been reports of  issues when upgrading of course, but nothing too dangerous. The best you can do is to reinstall the whole OS from a bootable CD to do a clean install. Feisty is more stable than Edgy, works better with my wi-fi chip and looks slickier in general.
