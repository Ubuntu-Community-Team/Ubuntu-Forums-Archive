### Post by louieb on 2007-08-16
Hello Daniell, Installing Linux on a computer is a little more involved than say installing  say MS office. For one thing Ubuntu is going to want  2 partitions.  1 for the operating system and a small 1 for swap. To get an idea of what the install will be like look at [Psychocats Ubuntu Linux Resources]("http://www.psychocats.net/ubuntu/index.php"). There is a nice walkthru of the install process there. 

Are you on AOL dialup? That may be a problem. Dialup in general is harder to configure than a broadband connection. AOL in particular, it has it own connection and sign in program.   Do a forum search on **AOL**. There is a program called **penggy** which is an AOL connection program. Don't know anything about it, never had to use it. 

Do you have a 2nd computer? Its always nice to have a computer with a working internet connection handy while setting up a PC to Dual Boot.  
Hope  it works out for you. Lou.
