### Post by ajgreeny on 2008-04-27
Live CD shouldn't ask for a user and password, but just out of interest try **ubuntu** as user name and leave the password blank.  On the multi-desktop live CDs produced by Linux Format magazine in the UK, ie with gnome, kde and xfce desktops installed the live CD can be booted into either desktop after loging out by doing exactly that; without entering the user name, it doesn't work if I remember correctly.  However if you are being asked for user and password at the first login of a liveCD, something must be wrong.  Try burning the iso image again or downloading again if the md5sum is not right.
