### Post by emptybox on 2009-04-19
Gnome is just the name of the desktop environment.

Gnome Power Manager is the program that deals with suspending and hibernating the computer, and shutting down the screen after a period of inactivity, among other things.
Not sure how it impacts on wireless networking though?

If it was a kernel upgrade that took place, you could try booting into the previous version, if it is still in Grub and see if the wireless works in that.

gnome-power-manager can be installed through Synaptic Package Manager (or the terminal).You could have a look in there and try uninstalling and then reinstalling it and see if it works then?
