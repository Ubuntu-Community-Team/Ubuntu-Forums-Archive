### Post by dstew on 2008-03-02
Do the Disk Integrity Test from the Live CD menu. If it is good, consider using the Alternate Install CD. It installs the same Ubunut desktop as the Live CD, but uses a text-based installer. Get it from the Download Ubuntu page, click the box below the Start Download button.
