You shouldn't need to install any particular drivers for your keyboard in linux; it should work out of the box. Any driver/software CD that comes with the keyboard will usually be for Windows and will not install and/or perform in Linux.

However, you should find most features (if not all) working just fine without the need for drivers.

Please post back if you are facing problems with the keyboard and/or specific features of the keyboard.
