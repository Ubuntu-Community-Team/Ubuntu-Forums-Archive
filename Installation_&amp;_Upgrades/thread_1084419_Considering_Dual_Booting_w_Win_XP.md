### Post by Trebaruna on 2009-03-02
Hard drive: depends on what you want. If you play lots of games, you may need more space than if you just use XP to browse the net. When I dual booted XP I used 10GB. The install takes up a lot less (especially if you use nLite), but games take up a lot, too. It's better to overestimate than to underestimate if you can at all miss the space.
RAM: If it ran XP before, there's no reason it shouldn't, now. 768MB isn't very luxurious, but it should run.
Dual boot speed: it's quite simple, really. You either load XP, or Ubuntu. When you're using the one, the other is not active at all. There is zero overhead involved.

After quickly skimming through the tutorial it looks like it will serve your needs.
Please realize that doing anything like this involves somewhat risky actions. Make sure you either don't care about, or have backups of the data on your machine. Everything should be fine, but better safe than sorry, right?
