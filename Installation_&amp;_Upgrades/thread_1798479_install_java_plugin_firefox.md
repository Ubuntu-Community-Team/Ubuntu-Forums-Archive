### Post by TBABill on 2011-07-06
If you want Sun Java's plugin, just go into software sources and enable the partner repository, then search for java plugin and install via Synaptic. I would imagine it also shows up in the Ubuntu Software Center, but I've never installed it that way. 
 
I make a point of always adding the Sun Java plugin because the open java plugin does not work as well in some of the sites I use.
