> **swasea-to said:**
> i cant even take a screenshot

Your Flash player is working correctly.

The Adobe Flash site explains that message: [http://www.macromedia.com/support/documentation/en/flashplayer/help/help06.html](http://www.macromedia.com/support/documentation/en/flashplayer/help/help06.html)

It is not specific to Edubuntu, the same message would appear in Windows too.

It does seem there is a bug in the version of the Flash player you are using that prevents the "Allow" button from working (see [https://support.mozilla.com/en-US/forum/1/429963](https://support.mozilla.com/en-US/forum/1/429963)). You should be able to change the maximum storage capacity by following these steps:

1) Right-click on the Flash game (you might need to do this quickly, before the message appears) and click on "Settings..."

2) Click on the 3rd tab at the bottom of the little dialog that appears, it will have an icon that looks like a folder.

3) The "Local storage" settings should appear, drag the slider all the way to the right, so the maximum storage is "Unlimited".

4) Click the Close button. You may need to refresh your browser so the new setting takes effect.
