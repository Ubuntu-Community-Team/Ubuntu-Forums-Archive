### Post by ruibernardo on 2007-10-17
Hi tech0007,

you can only encrypt the hard drive with a clean (new) install from a Alternate CD of Gutsy (alternate or server CD). The Live CD (desktop) don't have it, even with a fresh install.

When you select to encrypt your disk, the installer will delete every partitions on the disk, and then create encrypted partitions, so an upgrade isn't possible.
