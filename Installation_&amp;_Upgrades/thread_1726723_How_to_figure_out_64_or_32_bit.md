### Post by wojox on 2011-04-11
In a terminal:

```
uname -r
```

will tell you what you have installed. For a more stable edition try [Ubuntu 10.04.2 LTS (Lucid Lynx) Netbook]("http://releases.ubuntu.com/lucid/")
