### Post by Xian on 2005-03-12
[QUOTE=pugrit]how to install kde to ubuntu (warty) ?[/quote]
You don't need kubuntu to install KDE to Warty. Just open a terminal and:

$ sudo apt-get update
$ sudo apt-get install kde

That will take care of the installation.

[QUOTE=pugrit]Has ubuntu (hoary) already being shipped?id like to ask some.[/QUOTE]
No, you can order it from the [Shipit](http://shipit.ubuntulinux.org/) page, but it won't be sent out until April.
