If you want to have the latest all the time, then you have to pay the price: update and upgrade, sometimes reinstall every six months.

### Post by Partyboi2 on 2008-01-16
installing all the latest updates won't get you to the next release to do that you would need to either to a network install or install from a cd. 
For installing over the net
[http://www.ubuntu.com/getubuntu/upgrading](http://www.ubuntu.com/getubuntu/upgrading)
for reinstalling you can download the live cd
[http://releases.ubuntu.com/releases/7.10/](http://releases.ubuntu.com/releases/7.10/)
or you can upgrade without having to reinstall everything by using the alternate cd
[http://releases.ubuntu.com/releases/7.10/](http://releases.ubuntu.com/releases/7.10/)

But if you are happy with what you are using and are having no problems, I don't see the point of upgrading unless you are wanting all the lastest things, which often comes with a few headaches.
