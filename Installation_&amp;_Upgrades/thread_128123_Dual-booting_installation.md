Has anyone else expirienced this and can someone help me? So you already have a partition made to install Ubuntu on? Or did you mean you have one partition you need to divide up and install Ubuntu? (With Windows on it most likely?)
That's okay, just look at[ this website]("http://users.bigpond.net.au/hermanzone/"), and it might help you decide what you need to do. There are three different types of Ununtu installs illustrated there. One is for use if you have WIndows with a FAT32 filesystem, another is for NTFS, and another is for people who want to make an extra partition for their /home for Ubuntu. The FAT32 one is the easiest and also I think the best. That's if you have no partition already made. 
If you do have your partition already made, that's okay, just select 'manual partitioning', but you won't need to do any manual partitioning, it will show you a list of your partitions and you can pick the one to install Ubuntu on and after that it's easy. You will see from the illustrations I think.
As for the network part of it, it's a good idea to plug in your internet connection during the installation if you have one, so that your network will be automatically detected and configured by the installing software. For most people that's all they need to do, the rest is automatic. Unless you happen to have a winmodem maybe or some type of connection that Ubuntu doesn't recognise. If you do, I'm not sure, but I think you can set that up later on, so you can just skip that step in your install. Also just skip it if you aren't on the internet, obviously you are, or we wouldn't be reading your post. :-D (Herman)
