I can totally forgive the other issues/bugs I've experienced with Gutsy, but this one is debilitating. I work a lot with 2D/3D graphics, movie/sound editing, and run a few cpu/video intensive games... running at half my processor speed is simply NOT acceptable.

I understand that Ubuntu, like all distros, are a work in progress.. I know there are bound to be bugs and annoying issues with every new release.
