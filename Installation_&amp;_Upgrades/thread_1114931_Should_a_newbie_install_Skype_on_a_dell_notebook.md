### Post by issih on 2009-04-03
You should be fine..I'd vaguely reccomend adding the deb lines and the gpg key for the medibuntu repository (assuming that is the method you plan to use) via the software sources dialog, as that ought to stop you entering invalid data into the files.

All in all though, its pretty easy, and not that scary I promise.

Hope that helps

P.S. alternatively you can just download the .deb file from here

[http://www.skype.com/download/skype/linux/choose/](http://www.skype.com/download/skype/linux/choose/)

by clicking on the little ubuntu logo, then click on the downloaded file to install it.
