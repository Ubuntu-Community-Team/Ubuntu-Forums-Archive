### Post by PmDematagoda on 2008-01-09
If Ubuntu 7.04 works the way you want it to, then it is not absolutely necessary to upgrade it to Ubuntu 7.10 currently since Feisty still has a bit of support left. So if you like Ubuntu the way it is, keep it, also an upgrade has quite a chance of breaking the system so what you are taking is a risk as well. But when support for Ubuntu 7.04 does run out, then it may be time to really think either about an upgrade or a clean install of Ubuntu 7.10.
