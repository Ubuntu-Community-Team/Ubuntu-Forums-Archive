### Post by avtolle on 2006-08-21
Paulo: definitely download and burn as an iso image the Desktop CD to have a "Live" CD; the alternate install CD will install, there is no "live" option. Should you decide to install after trying it out, then you could pursue the Alternate Install CD for this purpose, but you are left with the Desktop CD to run as a Live CD.
