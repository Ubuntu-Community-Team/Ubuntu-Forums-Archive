### Post by mr_shedges on 2008-12-22
So spent ages searching the net looking for the easiest way to install ubuntu on a usb stick.

Try numerous and found that some just plain don't work others wont do what you ask and the install doesn't even start etc.

Well on a whim i tried to install from the alternative CD. Heay presto, it all worked.

All you need to do is select your USB drive for the install and grub. Bingo. I would reckomend a decent size USB. Installed was just over 2.5 gb. got a 8.1 gb stick so all is well.

Tried it on 6 machines and as long as you do not set up fancy screens like Beryl etc worked without a glitch an all.

Didn't see this posted anywhere when i searched for it, thought it mite be useful.:P
