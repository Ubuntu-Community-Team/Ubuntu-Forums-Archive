You need to be aware that if, after the "update", something no longer works properly, or you just don't like it, either way -- there is no way to "roll back" to what you have in place at present.

The "safest" thing to do is to back off your install using Clonezilla,  to an external drive.  At least that way, if the update proves to be more problems than you can handle, you will have a way to get your current setup back.

### Post by darkod on 2012-10-14
Unless there is some functionality that you need and can't be added to 12.04, nobody is making you UPGRADE to 12.10. (to diffrenciate from installing UPDATES on your system, UPGRADE to the next release is different)

The 12.10 will have 18 months of support after it comes out, while the current 12.04 is LTS and has 5 years of support from the date it was released.

So, unless 12.10 has something you really need, I wouldn't even consider upgrading. Just keep your 12.04 that you already customized the way you like it.

Also, one thing to note if you do decide to upgrade using Update Manager, it might be configured to inform you only about LTS releases. So you might need to change that option to Normal releases if you want it to offer the 12.10 when it comes out.

You can upgrade in two different ways.
You can either use the upgrade option in the update manager, or you can run a fresh install.
Each method has it benefits, and its drawbacks.
Using the upgrade through the upgrade manager option also upgrades the entire system you're running, including all upgrade-able programs and packages you've since installed. However this method, when it goes wrong, can leave your system completely useless(I use this method, and I've never had any serious issues).
Running a fresh install is the easier, less stressful method in that everything is installed through the disk, and any initial problems can, in most cases, be rectified by re-installing again. This method though only gives you a base install, and the setting, and programs you've installed will need to be re-installed(however you see fit).
The important thing to do when upgrading either way is to back up your important data(Documents, Videos,Pictures,etc,etc).

And as a personal opinion, if you decide to use the upgrade through the update manager, wait. The first weekend the version is release can be very, very very, slow.
