The best thing I can tell you is this: 0) Make sure you do an md5sum check before of your downloaded iso image. If you are windows, the easiest way to do this is from [here]("http://www.nero.com/enu/Nero_MD5_Verifier.html").
If the md5sum generated doesn't match with the original md5sum, then you must re-download because you are getting a corrupt file. 

* Do not uncompress the iso image!!!!

1) Assuming you have a clean iso image, proceed to burn it to your blank dvd/cd. Again, if you are a windows user and don't want to spend alot of money, go [here]("http://isorecorder.alexfeinman.com/isorecorder.htm").  

2) Change the boot order to cd first.
