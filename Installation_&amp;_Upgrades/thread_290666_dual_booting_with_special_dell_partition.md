### Post by jmanbeck on 2006-11-01
I think that's the Dell Diagnostics partition and there are some utilities in it that are run the first time you boot up your computer. On my Dell Latitude D810 it was about 60MB with a FAT16 file system.  If this is the same type of partition you have, then it's ok to delete it.  All of those utilities should be available on the resource CD that came with the notebook.  I recently set up a dual boot, deleted this partition and have had no problems.
