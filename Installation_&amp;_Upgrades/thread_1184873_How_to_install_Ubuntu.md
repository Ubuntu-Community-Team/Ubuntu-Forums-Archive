No, Ubuntu doesn't support any Windows games.

Some of them *can* be used with Wine, and some run even better than they do in Windows. On the other hand, many will not work at all, or require a lot of tweaking to run.

If you mainly use your computer for gaming you're pretty much stuck with using Windows.

(Of course there's the option of dualbooting so you can run the games in Windows and do everything else in Linux.)

### Post by Herman on 2009-06-11
> Of course there's the option of dualbooting so you can run the games in Windows and do everything else in Linux.:D I agree with mcduck, use Windows for your games and use Ubuntu for everything else. (Your personal files and any sensitive files).

Contrary to popular beliefs, no firewall can protect the average Windows user, because no matter how good the firewall is, the users most often allow any program access to the internet that asks for it without checking what the program is for. If they spent the time checking every program, it would take up all their time. They would have no time left for work or playing any games.
In Windows, you can install any software from any (unknown) programmer and then you never know if it contains malware or contains an as-yet-undiscovered virus until many months later. Meanwhile, your antivirus tells you that you are 'protected', because you are using their brand of antivirus, but then it discovers the virus six months or more after you installed the game the virus was found in.

Therefore, I think the Ubuntu way of doing things is better.
Ubuntu does come with a lot of free games, and there are probably hundreds you can install from the repositories. I installed Ubuntu for a neighbor across the street and I installed just about every game on offer and the list was as long as your arm. 

Ubuntu could still use more games, especially if they're good ones, but not just anyone can go and put any new game they like into the repos. 
In Ubuntu we install software and games through special installing software, (apt-get or dpkg), and we download the files from 'repositories' where only registered developers can upload software to. That's one of the reasons why Ubuntu is secure and free from malware and spyware and so on. We don't (usually) need any firewall because there are no rogue programs trying to call home. Some repos are very strict and exclusive and other repos are less strict, but the repos most of us use are perfectly safe.

Try installing some of the games available for Ubuntu from the repositories and see if you like them or not.

Regards, Herman :)

Also :
[B]1. back up your data. During partitioning and installation there is always a chance of data loss or corruption. It usually doesn't happen, BUT don't be the one caught with your pants down!
2. If you are going to resize a Windows partition to set up a dual boot always defrag Windows at least once or twice. Windows has a bad habit of throwing files all over the disk. This will remedy that and will make your resize operation safer.[/B]
