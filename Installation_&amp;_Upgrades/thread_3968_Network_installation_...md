### Post by ashley_v on 2004-11-11
[QUOTE=][/QUOTE]
 I don't think ubuntu has boot floppies like Debian, just the live cd and the other cd.  Not too many distros have support for that anymore.

It generally consist of you writing 2 or 3 images to floppy, booting, grabbing an address from your dhcp server or you isp's and then proceeds to download the installer from a mirror site.  The only distros I know you can do this with are Debian, OpenBSD, FreeBSD, NetBSD, and maybe a few others but it's slim to none.

If you want to know what Ubuntu feels like you could try the Debian method but you would have a Debian system, not Ubuntu.  Their pretty much the same in regard to the way system is set up.  Except debian has flavors like stable (woody) testing (sarge), and unstable (sid).

If I were you I'd just buy a single blank cd from the store and burn it (ubuntu).  Cost ya $1 US.

Good luck
