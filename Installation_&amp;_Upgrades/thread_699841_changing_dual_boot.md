### Post by bilge.tutak on 2008-02-17
I did not install Mythbuntu, but I am pretty sure it will offer to you to format the hard drive and install the mythbuntu (check with their installation). After installation, it will probably install a boot loader with Dapper and Mythbuntu also. Just be careful which drive you are selecting during mythbuntu installation.
Alternatively, if you want you can install mythtv directly from packages for your Dapper and test it there. Then you will not need a new installation.
