### Post by snowpine on 2008-09-09
You need 384mb of ram to install Ubuntu using the live CD, so I would recommend a ram upgrade. If that's not an option, you can try the Alternate Install CD (which requires 256mb) or a lighter-weight version such as Xubuntu. Good luck!
