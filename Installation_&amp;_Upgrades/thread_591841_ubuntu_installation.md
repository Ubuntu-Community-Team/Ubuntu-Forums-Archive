### Post by Pumalite on 2007-10-25
You have to download an iso from here:[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)
Then use this guide: [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)
Burn as 'Image' not as 'data'. Burn at 4x. Then boot from it. Make sure CD is first in boot order in BIOS
