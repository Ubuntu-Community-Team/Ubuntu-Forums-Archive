### Post by nocturn on 2005-01-04
[QUOTE=colpaeko]I was wondering: did anyone attempt to install KDE on an Ubuntu box? And if yes, how did you do it?[/QUOTE]

Yep (my wife is using this).
Just enable the universe repository and install the package 'kde'.

Commandline:
aptitude install kde

Log out, on the login screen, select a KDE session.

This gives you KDE 3.2.3
