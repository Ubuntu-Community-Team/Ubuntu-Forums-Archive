### Post by mikewhatever on 2008-01-28
Most package versions are not upgraded during the lifetime of each release. That said, you do get security an recommended updates which do not change the version of the package updated. One exceptions I can think of is Firefox, which gets upgraded by incorporating security updates. After installing 7.10 from CD, you'll get over 100 MB of updates.
It seems that the latest stable version (7.10) is the one to recommend, but if it does not work for some reason, try Feisty (7.04). They are quite similar feature wise, but Feisty's support ends in Oct 2008, and it has older software.
