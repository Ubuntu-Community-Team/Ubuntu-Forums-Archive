Lets say I choose to make a new installation in english. How do I change this installation to lets say danish (preserving the encoding in utf8, of course)?

---

### Post by moma on 2007-01-18
In GNOME, select the [COLOR="DarkOrchid"]System [/COLOR]-> [COLOR="DarkOrchid"]Administration[/COLOR] -> [COLOR="DarkOrchid"]Language support[/COLOR] menu. Then choose "Danish" from the "Supported Languages" list.  Press the [Apply] button and it will install Danish language packages for the Desktop, Openoffice.org, Firefox and Bash messages, with full dk.UTF8 support.

Select Danish as the "Default language" in the same dialog.

Press CNTR + ALT + BACKSPACE to restart your desktop -- og den vil snakke dansk
