### Post by Charles Hand on 2006-09-12
For what it's worth, I'd slip a Ubuntu CD into that puppy and see how it goes. You'll be able to see right away what hardware it detects and what it doesn't. If all looks good, click "install".

If something doesn't work right out of the box, you can search these forums for help. Note in particular the ndiswrapper package which many find helpful. In case your network device doesn't have linux drivers, you can use ndiswrapper to use the Windows driver for the device under Linux.

-Charlie

### Post by nrwilk on 2006-09-13
If you want to try an ubuntu distribution, I'd try Xubuntu right from the start.  XFCE is lighter than Gnome and KDE, and it's *much* better on older hardware.

The following distributions are just *some* designed to work well either on older hardware, or on very low system requirements:
[Damn Small Linux (DSL)]("http://www.damnsmalllinux.org/")
[Puppy Linux]("http://www.puppylinux.org/user/viewpage.php?page_id=1")
[KATE OS]("http://kateos.org/")
[Vector Linux]("http://www.vectorlinux.com/")
[Feather Linux]("http://featherlinux.berlios.de/")

DSL and Vector are both pretty big now.  Puppy is supposed to be good, but I've never heard of KATE OS.  Though, when I saw KATE OS, it made me want to check it out. :)

Hope you're able to get something running on it well.
