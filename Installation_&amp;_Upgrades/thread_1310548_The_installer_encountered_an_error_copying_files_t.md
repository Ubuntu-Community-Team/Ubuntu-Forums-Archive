### Post by mickcalcara on 2009-11-02
I was able to resolve this. I am fairly confident my dvd/cd drive was over heating during the install. I notice these failures happened when the CD drive was operating at full speed and was very warm to the touch. Also these errors occurred earlier on when the drive was operating ( ie. working with the Live CD before I launched an install ) My solution: I put a cold pack on the drive and quickly install have drive had been idle for some time.The installation completed without issue.
