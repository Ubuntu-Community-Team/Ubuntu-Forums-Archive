### Post by sikander3786 on 2010-11-12
The best way to test an Ubuntu version will be to burn a LiveCD, boot from it and test all your hardware. Download both 10.04 and 10.10 and see the difference yourself.

You can download from the official Ubuntu website directly or using a torrent.

This link might give you some tips on installation.

[https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall)

Don't forget to run a check on the downloaded image. Instructions here.

[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

Burn the disc at the lowest possible speed.

Once you test the Live CDs and decide which one you are going to install, we will be happy to help further.

### Post by Wtower on 2010-11-12
Version 10.10 is fine. You can reach it from ubuntu.com. You could also try to boot from a usb as described therein and check how it behaves on your system. I wouldn't expect any problems on average. 

Ubuntu recommends 64 bit, but to be totally honest I use 32-bit to avoid any compatibility issues. Besides, with pae kernel ubuntu can reach 8mb ram, which is something that ubuntu normally auto-detects upon installation or even configured later. I would recommend to go for 32 bit unless you really have memory demanding apps like video editing.
