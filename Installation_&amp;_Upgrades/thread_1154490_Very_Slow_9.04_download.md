### Post by lovinglinux on 2009-05-10
I don't think the problem is the ISP or the router. I have the same model btw.

When I installed Jaunty I had to download all my personal applications at the same time and the speed was incredibly slow. So I changed the source server and everything went well, pretty fast.

Open "System >> Administration >> Software Sources", click the "Download from" drop-down menu and select "Other", then hit "Select Best Server" or manually select a server near you.
