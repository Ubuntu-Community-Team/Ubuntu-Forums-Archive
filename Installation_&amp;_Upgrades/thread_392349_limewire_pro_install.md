### Post by rsambuca on 2007-03-24
The md5sum hashes are a method used to verify that the file you have contains no errors from the original.  Each file has a unique md5sum, so if you had errors during downloading the md5sums won't match.

To test the md5sum, in a linux terminal just type```
md5sum 'filename'
```The algorithim will then run and spit out the md5sum text string.  Compare yours to the known correct version.
