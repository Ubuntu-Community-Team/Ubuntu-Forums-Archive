### Post by ottosykora on 2012-01-02
1. establish wired connection

2.start take care of wireless

little bit of confused I admit, as missing firmware would be for the built in mobile phone device rather then the normal wlan module.

To use the mobile phone module in the 110, you need other software to be installed and firmware to be loaded, but this works only sometimes and is not reliable at all. Don't bother about it.

Just get online with wired connection, run update and all should be OK.

the 110 works with ubuntu wired and wireless out of the box, but if during the istallation for some reason no internet connection was made, then some things can be missing. Therefore proceed with the wired connection first!
