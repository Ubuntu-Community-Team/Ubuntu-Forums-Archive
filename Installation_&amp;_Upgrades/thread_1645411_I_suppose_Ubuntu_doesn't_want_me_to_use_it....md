### Post by sikander3786 on 2010-12-14
Welcome to the forums :-)

Yes I've got a very simple suggestion here. As soon as the computer starts booting from the USB drive, press any key to gain access to boot options page. Then press F6, enable 'nomodeset' and continue boot. It might solve your blank screen issue.

[https://help.ubuntu.com/community/LiveCDBootOptions#Main%20Page%20Options](https://help.ubuntu.com/community/LiveCDBootOptions#Main%20Page%20Options)
