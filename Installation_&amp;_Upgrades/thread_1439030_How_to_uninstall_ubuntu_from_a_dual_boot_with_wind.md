### Post by lindsay7 on 2010-03-25
You do not need to uninstall it. Just do a clean install using the partition or partitions that Ubuntu is now using.  Use the manual option during the install.  How is it now set up? Do have a separate parttion for /home and one for /?  If so just install the new version to your / partition and and leave the /home alone (do not format it).
