### Post by Arand on 2008-05-17
For burning the iso, you could use [Infrarecorder](http://infrarecorder.sourceforge.net/) (Free and Open source) or [ImgBurn](http://www.imgburn.com/) (freeware).

Ubuntu normally automatically installs the drivers you need, or asks you if you want to install them if they are "Non-Open" drivers.

As long as you get drivers for your network card working, and have an internet connection, Ubuntu should be able to connect to the internet automatically.
