### Post by mac9416 on 2009-10-04
> is it possible/feasible to customize a live version

There is a nice guide here for customizing live CDs: [https://help.ubuntu.com/community/LiveCDCustomization](https://help.ubuntu.com/community/LiveCDCustomization)

Also, I've heard about the Ubuntu Customization Kit: [http://uck.sourceforge.net/](http://uck.sourceforge.net/)

> so that some things are saved when you reboot?
You can use Ubuntu's USB-Creator to make a USB (and probably SD card) live install that saves settings/files when shut down. You can find it at System > Administration > Create a USB startup disk.

Hope that helps!
