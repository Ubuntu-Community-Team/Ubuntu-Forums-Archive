### Post by SamH on 2005-06-17
In the location window where you usually type URLs, instead type in **about:config** that will bring up a page of config settings for Firefox.  Scroll down to the general useraget.vendor setting.  It will read 1.0 or such.  Double click on the setting name, a dialog box will appear where you can change the 1.0 to 1.0.4.  That's all there is to it.

Also, you can type about: plugins to see a page telling you about all your installed plugins.

Hope this helps.
