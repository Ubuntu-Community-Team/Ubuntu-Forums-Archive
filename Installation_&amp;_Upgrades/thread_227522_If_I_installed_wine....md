### Post by amp_man on 2006-08-01
to use wine to run exes under nautilus, just left click on an exe, select "Open with", the click the little triangle at the bottom of the window that pops up, and type "wine". should work for any gui-based program, although command-line programs might be tricky (not entirely sure)
