### Post by Dennis N on 2014-09-19
Install the package **flashplugin-installer** from the Software Center or by using a package manager. flashplugin-installer will download and install the correct Adobe Flash Player package for you as it is installed.

### Post by claracc on 2014-09-19
In every ubuntu release, 12.04  and 14.04,  if you want to see flash videos you have to install adobe flash plugin which is proprietary software. Do as Dennis N points.

In order to see images in your e-mail you cannot use gedit software since it is software to see text, you will have to select image viewer in order to see images. Click on the image file in mail and use open with "image viewer" ( perhaps you will have to look for image viewer).
