### Post by baltadt on 2009-07-16
I am running a serial keyboard and had a similar problem.  When typing it would do two things: 1 it would lag on the response and 2 would input characters constantly until i hit back space enough then it would delete everything. turns out if was a hard drive malfunction.  Down load a program called kill disk put on a cd and run it on boot up.  This will completely revert your hard disk to oem factory setup...complete wipe.  Then reinstall Xubuntu. This worked for me and my computer actually runs faster now.  Hope this will help.
