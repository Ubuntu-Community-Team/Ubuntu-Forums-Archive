### Post by merlinus on 2007-09-03
You do not have enough RAM to install from the ubuntu live cd.  You can use the Alternate text-based one:

[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

Tick the box below the green Start Download text, then click the text to begin.

When you get to the partiioning menu, select Use Entire Disk, and ubuntu will replace xp, or vista.

Xubuntu might run better on your machine, so you might download that instead.

In any event, you can change from one to the other very easily once installed.

-merlin
