### Post by Impavidus on 2022-03-20
Groovy, or Ubuntu 20.10, only ever supported an upgrade to Hirsute, or Ubuntu 21.04. 20.10 reached end of life before 21.10 was released, so no upgrade path from 20.10 to 21.10 was ever created. And 21.04 reached end of life too, so that upgrade has been closed.

You can create a live disk of 21.10 and install it over 20.10 without formatting the 20.10 filesystem. That should upgrade your 20.10 directly to 21.10, keeping your installed software (if still available), settings (if still applicable) and documents. Make sure you've got backups anyway and be ready for a fresh install in case something goes wrong.

Ubuntu 22.04 is close to release now. If you feel adventurous, you can try that too.
