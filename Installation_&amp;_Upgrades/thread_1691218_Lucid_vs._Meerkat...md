### Post by laceration on 2011-02-19
Meerkat was a pointless upgrade.  It's major significance was the Shotwell photo program was added -- and this is a trifle.  All the updated libraries and programs only resulted in more kinks and annoying little problems.  Ubuntu has an update every 6 regardless if it is warranted or not.  If you can find out if Meerkat will resolve your problems, and I would guess that it probably won't, go for it. Otherwise the only compelling reason would be that it would be possible to upgrade to Natty without having to do a fresh install when it comes out.  Natty seems promising because of Unity.
