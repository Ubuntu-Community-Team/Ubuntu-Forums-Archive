### Post by 23meg on 2007-02-13
Depends on your requirements which way is best. If you require full compatibility, a virtual machine is the best way (still won't work with 3D acceleration, and some devices may not be utilized fully). If you need a way to run them in your current environment, without the high processor load of a virtual machine, you can try WINE, but only a small portion of Windows apps will work with it.
