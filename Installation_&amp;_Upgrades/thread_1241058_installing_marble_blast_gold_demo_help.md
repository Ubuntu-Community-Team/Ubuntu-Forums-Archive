### Post by Partyboi2 on 2009-08-16
Hi, open a terminal (Applications>Accessories>Terminal) and cd to where the downloaded MarbleBlastGoldDemo-1.4.1.sh.bin file is, so if its on your Desktop
```
cd ~/Desktop
```Make the file executable
```
chmod +x MarbleBlastGoldDemo-1.4.1.sh.bin
```then start the install process with
```
./MarbleBlastGoldDemo-1.4.1.sh.bin
```
