> **lse123 said:**
> how switch between install / uninstall _or_ install-new/see packages on linux hdd already installed ?
Switch between what? Synaptic has a really easy, self-explanatory interface, so just open it up and you will understand how to use it. Use the search function to find packages, right-click a package or left-click the checkbox next to it to see the install/uninstall/update options.
Installed packages have a green checkbox next to them and a version number in the "installed version" column, packages that are not installed have a white checkbox and no version listed in the "installed version" column.
