### Post by Jeaux on 2011-02-26
I have ran a Joomla site using Xampp off a Windows box.

You just install the "install" package for Joomla in your localhost w/e. Then browser to localhost in your browser.  Set up a DB and DB user via phpMyAdmin and give the right permissions.  Install Joomla.  Remove installation folder and that's it.
