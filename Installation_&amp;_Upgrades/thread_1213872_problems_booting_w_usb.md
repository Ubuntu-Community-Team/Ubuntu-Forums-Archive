### Post by Mighty_Joe on 2009-07-16
Booting the Live CD image on a USB should give you the option to try Ubuntu or install, just like the Live CD.
There's a lot of things that could be wrong, especially with new hardware.  It has been a while, but I remember having problems with unetbootin.  
Have you tried using the USB drive on another computer?  If it boots, then the problem is your hardware.  If it doesn't boot, the problem may be unetbootin.  
Do you have a CD drive?  Try the Live CD.
