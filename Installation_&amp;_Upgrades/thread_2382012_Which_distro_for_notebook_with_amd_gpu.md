Welcome to the forums.

Video drivers are not distro-specific, generally.  Did you try the "Additional Drivers" tool or google for a driver directly from AMD support?

Generally, it is best to use the package manager for drivers that are available (additional drivers) and avoid getting drivers directly from the GPU vendor.  Linux isn't Windows.
