### Post by Mark Phelps on 2011-01-05
Before you go to the trouble to install and configure Wine, you would do better to go to the following site and look up the ratings for the apps you want to run:

[http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true]("http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true")

If the app name you enter returns no results, or is rated Garbage, it won't work.  IF it's rated Bronze, some of the features won't work.  To fully use an app, you need to see a rating of Gold or Platinum.
