### Post by ushimitsudoki on 2009-01-02
Install Windows inside VirtualBox.

From there, you can run even run VirtualBox in "Seamless" mode, which puts the Windows apps right on the Ubuntu desktop.

This will (probably) work perfectly well for non-3D applications. 3D stuff is possible, but it is new in the latest release, and not full featured.

If you want Windows Games, you still need to dual boot. Else, virtualization is the answer.
