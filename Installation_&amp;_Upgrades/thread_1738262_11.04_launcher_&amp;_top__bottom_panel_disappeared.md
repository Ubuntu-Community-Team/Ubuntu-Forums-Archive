### Post by Corgan on 2011-06-17
Personally I've found this to be the result of the deep coupling of the installed applications (i.e. Gwibber, Evolution, Empathy) with gnome-panel. Removing them removes both.

In my case, reinstalling gnome-panel, and simply running it gets us back up, but the removed software is again reinstalled.
