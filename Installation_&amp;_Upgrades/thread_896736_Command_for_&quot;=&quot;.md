### Post by prshah on 2008-08-21
> **megauser said:**
> 
 I need to add the line "ACPI=OFF" but the problem is I can't find the combo to make the "=" on a Swedish keyboard layout

Try this; after typing in "ACPI", hold down Alt, and press (rapidly) 0061 _from the numberpad_, then release Alt.. has the "=" appeared? 

If you're on a laptop, you will have to enable numlock first and study the keys to understand where the numberpad numbers are. Or you can temporarily connect an external keyboard.
