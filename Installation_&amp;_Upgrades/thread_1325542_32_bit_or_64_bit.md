### Post by Axos on 2009-11-13
64-bit is only for people who actually need more than 3 GB of RAM. 64-bit doubles the size of all the pointers in a program. In programs which use a lot of pointers, this can dramatically increase the amount of memory they use. So with 64-bit you get less out of the memory you have. Of course, if you have more memory than you need, this doesn't matter.
