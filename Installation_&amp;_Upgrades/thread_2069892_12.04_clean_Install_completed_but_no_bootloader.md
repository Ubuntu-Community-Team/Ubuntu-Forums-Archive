### Post by oldfred on 2012-10-11
Is this an older IDE drive? And do you have jumpers set so it is primary bootable? Or if in cable select does BIOS let you choose it as a boot drive?

Some BIOS seem to promote flash to sda for some reason as  yours has done. Normally sda is the hard drive and sdb the flash drive and grub just installs to the MBR of sda.

Try Boot Repair. It can also run boot script as part of its BootInfo report but also gives extra info. And it may let you install grub to the hard drive if BIOS lets you. You can install into your liveUSB.

Boot Repair -Also handles LVM, GPT, separate /boot and UEFI dual boot.:
[https://help.ubuntu.com/community/Boot-Repair](https://help.ubuntu.com/community/Boot-Repair)
You can repair many boot issues with this or 'Create BootInfo' report (Other Options) & post the link it creates, so we can see your exact configuration and diagnose advanced problems.
