### Post by XubuRoxMySox on 2010-01-20
That hardware is more than sufficient for any of the 'buntus. I like [COLOR="Blue"]Xubuntu[/COLOR] on my older hardware for it's speed and ease, but I also like it for it's simple, newbie-friendly interface. [Crunchbang]("http://crunchbanglinux.org") (one of my favorites!) is less "familiar-looking" and has a little steeper learning curve (which is really fun actually). It's stark minimal appearance belies it's power and versatility.

The latest version of X/K/Ubuntu will boot and run *much* faster than the earlier versions! [COLOR="Blue"]Xubuntu[/COLOR] (9.10) does not ship with most of the software that has given some people fits (PulseAudio, for example - Xubuntu uses ALSA instead) and runs trouble-free on my 'puter after all the updates.

-Robin
