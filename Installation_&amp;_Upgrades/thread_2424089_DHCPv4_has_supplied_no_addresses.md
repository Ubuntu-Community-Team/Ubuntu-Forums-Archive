### Post by gdesilva on 2019-08-03
Not sure what screen you are referring to (a screenshot would have been useful) but here are a few thing to check for;

1. have you got the network cable plugged in correctly?
2. Is your router set to act as a dhcp server? By default most consumer grade routers are set to do so but they also provide the option of having it turned off. So check the router to make sure that it is set to serve dhcp addresses.
3. Is the cable from the PC is plugged into the router? If you are going through hub/switch make sure all those connections are working.
4. If you are using wifi, have you connected to the local wifi network?
