### Post by mannheim on 2006-10-30
I think that if you log out of a gnome session without first manually closing firefox, then firefox seems to think it has crashed, and will offer to restore your session next time you launch it.

Like the original poster, I would like to be able to disable this feature. For me, logging out while applications are still open is normal behavior: the apps should then behave as if you closed them normally. (Of course, an application like an editor or word-processor etc is different in this respect!)
