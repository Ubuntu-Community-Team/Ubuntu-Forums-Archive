### Post by Insightfill on 2007-09-18
Keep in mind that the live CD ("Desktop") will be DOG slow in that configuration, mostly due to the lack of RAM.  The live CD will at least let you know that it works, but not give you a good feel for the performance.  For that - you might as well use the alternate installation disk.

I've actually got a K6-450 laptop with 64MB of RAM that's tolerable in Windows 2000, but I'm tempted to install Xubuntu on it and see what happens.  Yes, it's under-spec, but it was right on the line with Win 2000 anyway.
