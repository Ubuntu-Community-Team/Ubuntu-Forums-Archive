### Post by Brandon Williams on 2009-07-16
adobe-flashplugin is in canonical's partner repository. Anyone running i386 ubuntu should be able to simply enable the partner repo (e.g. for jaunty, add 'deb [http://archive.canonical.com/ubuntu](http://archive.canonical.com/ubuntu) jaunty partner' to /etc/apt/source.list) and install the package. This should cover the majority of users.

Your tutorial should be helpful for users on other architectures, though.
