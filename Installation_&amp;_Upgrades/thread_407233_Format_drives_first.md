### Post by jglen490 on 2007-04-11
Unless the machine has Vista on it, I wouldn't format it.  Formatting with Windows will do nothing except put a Windows file system on the drive.  If it currently has Vista, you can use fdisk to delete partitions and then let Ubuntu set up its own.

You can run Ubuntu with the "live" option first, just to check it out.  Then you can load it to your hard drive later, if you like it.

Or, you juat take the jump and install straight from the CD - just make sure you have booted to the CD drive first and not the hard drive.  Just read and follow the installer as it does its thing and you should be fine, assuming you have no really strange hardware.  If you have an ATI or Nvidia video card, you may wantto get the latest driver from them, but to start, the stock drivers, while not pretty, will work.  Your ethernet or wifi connection may need some drive and configuration work, but a little research and some help will get all that going, too.

Good luck, have fun, and come back with questions.
