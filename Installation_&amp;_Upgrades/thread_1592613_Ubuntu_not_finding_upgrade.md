Today I downloaded the latest Wubi from Ubuntu's website. I thought it would be the 10.10 wubi, but it was giving me 10.04. I figured that I could just download it and then get the upgrade when it was done.

The only problem is that update manager isn't telling me anything about a new release. It gave me the usual stockpile of 100 updates for software (I just installed Ubuntu), but no new Ubuntu releases. I even tried sudo do-release-upgrade but to no avail.

So, does anyone know how to fix this?

### Post by BaffledMollusc on 2010-10-10
By default, if you are using a long term release (such as 10.04), the upgrade manager only tells you about other long term releases (which 10.10 isn't). 

See my post here for a solution: 

[http://ubuntuforums.org/showthread.php?t=1592529](http://ubuntuforums.org/showthread.php?t=1592529)
