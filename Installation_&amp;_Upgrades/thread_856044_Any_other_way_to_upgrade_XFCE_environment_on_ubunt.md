### Post by funkydan2 on 2008-07-12
You should be able to do this if you can get hold of (download) an Xubuntu Alternative install CD.

I'm not sure if the CD will be automatically added by Synaptic when you put it in or if you'll have to go Settings->Repositories in Synaptic to add the CD.  Then once the CD's been added, it should be a matter of using synaptic to install either the xubuntu-desktop metapackage or just the particular Xfce packages that you want.
