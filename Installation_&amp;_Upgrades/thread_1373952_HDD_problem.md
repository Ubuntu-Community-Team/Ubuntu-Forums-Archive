### Post by presence1960 on 2010-01-06
At boot from the Live CD when the menu appears hit F4 and choose safe graphics mode. For more info see here: [https://help.ubuntu.com/community/BootOptions](https://help.ubuntu.com/community/BootOptions)

If that does not work try the alternate text based install which BTW is way more powerful than the Live CD with more options such as RAID & LVM but just not as pretty as the Live CD. Get it [here]("http://www.ubuntu.com/getubuntu/downloadmirrors#alternate").

Your ATI vid card may be the culprit.
