### Post by rainwalker on 2009-08-05
That's because admin apps by default won't use the themes you have selected. To fix this you can do ```
sudo ln -s ~/.themes /root/.themes
sudo ln -s ~/.icons /root/.icons
sudo ln -s ~/.fonts /root/.fonts
```
