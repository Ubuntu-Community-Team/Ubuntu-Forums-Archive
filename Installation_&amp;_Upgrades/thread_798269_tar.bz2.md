### Post by Cato2 on 2008-05-18
You could install from the tarball but it would be **much** easier if you just install the Ubuntu package through Synaptic - just search for 'virtualbox'.  Or you can use apt-get as follows (haven't done this myself so have a look for a HOWTO if it doesn't work):```
apt-get install virtualbox-ose
```

This will mean downloading again but it's so much easier to do this it will save a lot of time.
