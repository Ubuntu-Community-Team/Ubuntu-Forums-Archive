### Post by Bliepo32 on 2008-01-02
It is possible to install Ubuntu without connection. In fact, one of my own computers had a broken wireless, and the install still went fine. So yes, you can install Ubuntu without a (working) internet connection. It goes exactly the same as a normal install.
