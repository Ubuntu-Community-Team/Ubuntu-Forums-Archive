### Post by wilee-nilee on 2012-05-09
Install the firefox addon flash aid restart and run it's wizard.

[https://addons.mozilla.org/en-US/firefox/addon/flash-aid/](https://addons.mozilla.org/en-US/firefox/addon/flash-aid/)

If you have not installed the restricted-extras look in the software center for the one with your desktop like ubuntu, kubuntu, xubuntu,or lubuntu. This will add other codecs and MS fonts and is actually the flash installer as well. Install this before you add the flash aid.
