### Post by tgalati4 on 2009-03-12
In a browser:

localhost:631

Add printer

Your printer should show up automatically.

Click through the dialog boxes.

Try printing from a few different applications.

In the second PC browser:

localhost:631

Select IPP printer.

Type in the name as it appears on the first printer.

Something like:  [http://yourmachine/printer/usb_laserjer1200](http://yourmachine/printer/usb_laserjer1200)

Print from 2nd machine.

Enjoy
