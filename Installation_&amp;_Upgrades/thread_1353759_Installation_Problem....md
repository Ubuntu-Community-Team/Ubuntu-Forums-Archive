### Post by darkod on 2009-12-13
I can't be sure about the error but you could try as alternative to use a usb stick. I think you can even create it from the cd itself. Let it load the Try Ubuntu option again, go in System-Administration and you have USB Startup disc creator there. I'm not sure if it uses the cd itself or you'll have to point it to the ISO on the hdd, but either case it should work.
Then boot with that stick and see how it goes. It has helped people with strange problems with the cd.
