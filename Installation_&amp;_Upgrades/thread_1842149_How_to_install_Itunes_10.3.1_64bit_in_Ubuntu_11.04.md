### Post by MARP1961 on 2011-09-11
To be perfectly honest with you, I have never heard of anyone getting iTunes to work correctly in Ubuntu using Wine. I have tried many times and often been able to install but never got it working properly. The iTunes Store certainly doesn't work. Apple simply want you to run iTunes on a Mac computer (even Windows computers seem to have problems with iTunes from time to time).

You have a straightforward choice:

Have a **Windows partition to run iTunes** and other Windows only programmes (Dual Boot) or **manage without iTunes** and use the Ubuntu default of **Banshee** which seems to manage iPods really well nowadays.

Just as an 'aside', has anyone ever managed to do anything useful with Wine apart from drink it?!
