### Post by tonymaro on 2008-01-30
I've just thrown together a new Tyan S3870 based 1U server.  Ubuntu 7.10 64 bit server fails to install claiming "no common cd-rom drive was detected".  I did some research and tried all the comments I could find regarding expert mode, BIOS changes, etc to no avail.

I finally ended up using an external USB DVD reader to do the installation.

Just an FYI for others experiencing the same problem.  :confused:

For reference, this machine uses a standard IDE connection to the drive.  The BIOS settings all look identical to every other IDE CD setup I've ever seen.  It is not a SATA drive.  It is however a slimline drive like you might find in a laptop since it's a 1U server.

I tried all suggestions I could find, including this thread:  [http://ubuntuforums.org/showthread.php?t=4447](http://ubuntuforums.org/showthread.php?t=4447)

It is _not_ a bad disc.  I've installed with this disc on three other servers this week with no problems.  In fact, it installed properly using this disc on the external USB reader.
