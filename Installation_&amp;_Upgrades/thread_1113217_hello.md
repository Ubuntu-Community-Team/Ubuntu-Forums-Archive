### Post by Partyboi2 on 2009-04-01
> my friend installed linux and he said to me not to install them because i have to make a format on my pc and everything is going to be deleted. also my pc will broke and it will be useless!Hi, there are a couple of methods you can use to install ubuntu, one of them is to its own partition which means your windows (assuming you are using windows) partition will be shrank down and and some extra partitions created for ubuntu to install onto. This can all be done by choosing that option at the partitioning stage of the ubuntu install and it will setup the partitions for you.. When ever you work on partitions it is advised to backup your important stuff in case something goes wrong which is not very often but a good practice to follow.
Ubuntu will not delete everything on the hard drive  unless you decide that you no longer want to use your current operating system and instead would like to install ubuntu to the whole hard drive.

If you go with the 2nd method that is to use wubi to install ubuntu inside windows which means that you will not have to worry about the fear of deleting your window operating system.
