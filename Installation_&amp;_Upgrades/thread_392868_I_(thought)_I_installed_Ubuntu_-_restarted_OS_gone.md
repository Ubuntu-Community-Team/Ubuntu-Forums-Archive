### Post by aysiu on 2007-03-25
Yes, you ran that all off the disk.

The disk is a "live CD," meaning that it creates an entirely functional operating system environment from the CD itself and your computer's RAM.

To actually install Ubuntu, click on the **Install** icon on the desktop. More details here:
[http://www.psychocats.net/ubuntu/installing](http://www.psychocats.net/ubuntu/installing)

**Back up everything important to you before setting up a dual boot**
