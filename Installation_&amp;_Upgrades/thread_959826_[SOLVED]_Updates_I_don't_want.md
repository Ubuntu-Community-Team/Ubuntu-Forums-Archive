I recently fixed the update manager(ubuntu hardy) and after updating, it keeps notifying me about an update for alien arena which I used to have but uninstalled. It's really annoying me. How do I stop it from showing updates I don't need?

### Post by Shazaam on 2008-10-26
To disable unwanted updates...
Open Synaptic, find said updates and highlight each of them. Go to Package (on the menu bar) and check off "Lock version" for each one.
