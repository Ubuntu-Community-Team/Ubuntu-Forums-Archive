### Post by skunkworks on 2006-11-11
Assuming you do have the space/extra partition set up to install another OS, there should be no problem with the /boot files, in most distributions, the installer will detect the other systems and add itself to the bootloader menu.  I tried Ubuntu 64 bit  for a few days, but like you said, it is hard to use as a desktop seeing as most of the ¨popular¨ things like flash do not work right or at all yet.  I try to refrain from using things like flash and sun's java anyway since those are proprietary programs.  The only reason I can see why you would want 64bit on the desktop right now is if you are a programmer who wants to take advantage of it.
