### Post by jasemfas on 2015-06-21
Hello everyone,

A few days ago I downloaded Ubuntu to install it on a MacBook Air, model A1465, using a USB stick, to be the only OS on the device.
I created the bootable USB installer, plugged it in and restarted the computer. Held down the alt button and chose the USB as the boot device. Selected the 'Install Ubuntu' choice and then chose to delete everything and run Ubuntu as a single boot. The Installation went smooth, no errors whatsoever and then it asked for the computer to be restarted. I click restart, and wait until it asks me to remove the USB stick and then click enter. I do so and the system reboots.
The MacBook sound effect plays, screen goes on, stays black for about 30 to 35 seconds then the flashing question mark folder appears. 

TL;DR
Installed Ubuntu from a USB stick on MacBook Air early 2014 as a single boot, deleting everything on the device while letting the installer deal with partitions division and got a flashing question mark folder after a succesful installation.
