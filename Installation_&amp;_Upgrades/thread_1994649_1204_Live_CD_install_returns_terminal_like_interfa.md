### Post by darkod on 2012-06-03
Actually if the live mode has an issue running, it's better not to upgrade online since there is big chance the upgraded installation will have the same issue. Until you find out what is bothering it, I wouldn't upgrade.

If you have nvidia, did you try booting into live mode with the nomodeset parameter?
[http://ubuntuforums.org/showpost.php?p=10069997&postcount=1](http://ubuntuforums.org/showpost.php?p=10069997&postcount=1)
