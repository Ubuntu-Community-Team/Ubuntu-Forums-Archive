### Post by Ginsu543 on 2009-12-11
It appears (according to Launchpad) that Skype has been removed from the Medibuntu repositories because it conflicts with the version being made available directly from [www.skype.com](www.skype.com). You can download Skype by going to their download page. They have a version available for Ubuntu 8.04 and two versions (32-bit and 64-bit) for Ubuntu 8.10+. Even though the file is called skype-ubuntu-intrepid_*.deb, I installed it on my Dell Mini 9 running Karmic and it seems to be working fine.
