### Post by Schalken on 2007-11-25
Chapter 4 of the Debian Installation Guide (Ubuntu being based on Debian) has some information for installing from the network, USB flash disk or floppy: [http://www.debian.org/releases/stable/i386/ch04.html.en](http://www.debian.org/releases/stable/i386/ch04.html.en)

Note that is for i386 and there is a specific installation manual for other archietectures.

If you can't find a way to install Ubuntu disklessly I recommend installing Debian instead, especially for a server.
