### Post by mk1w86 on 2008-12-07
Open the menu editor and find where you want the shortcut and then click New Item.  

Under application type select if you want it to run in a terminal or not.

In the command box type the command you normally use to launch the program.

Fill the name and comment box in as you wish, you can also choose an icon by clicking on the one to the left.:D
