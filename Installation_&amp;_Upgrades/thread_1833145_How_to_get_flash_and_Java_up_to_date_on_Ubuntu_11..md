### Post by oldos2er on 2011-08-25
If you already have them installed, ```
sudo apt-get update && sudo apt-get upgrade
``` will get you the latest versions in the repositories.

Things get a bit more complicated if you, for example, installed java using this method [https://sites.google.com/site/easylinuxtipsproject/java](https://sites.google.com/site/easylinuxtipsproject/java) , in which case you'd go through the same steps using the latest *.bin
