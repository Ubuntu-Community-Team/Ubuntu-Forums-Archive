I've successfully installed Ubuntu and Xubuntu 7.10 on an old desktop, and now decided to load it onto an older HP Pavilion xf125 laptop, but have had no luck. I'm hoping someone here can give me some suggestions (I am a Linux Newbie).

I've tried VGA safe mode, and vga=771, acpi=off, irqpoll, pci=noacpi, noapic, and nolapic in the boot line - nothing seems to make any difference at all. I've also tried both a Ubuntu and a Xubuntu Live CD, and with both it freezes at the same point.

Bios Version JA.M1.31
AMD Duron 1GHz Processor
512MB Ram
S3 Graphics TwisterK video display adapter
20GB hard drive with a bunch of free space

The laptop is running Windows XP now, and I was able to boot up with both Puppy Linux and Damn Small Linux CDs, so I don't think there is anything wrong with the hardware. To me, it smells like a problem with the video driver, but I'm at a loss where to go from here. 

Any help or suggestions would be greatly appreciated!
