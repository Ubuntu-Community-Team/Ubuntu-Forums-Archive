### Post by lemming465 on 2010-03-24
There a number of possible causes for this, unfortunately, and we don't yet have enough information to pick the right one.  

However, the most common problem is a bad CD-R burn.  There is an option on the initial boot menu to check the CD; what happens if you pick that?  If it fails or hangs, throw out that first CD and burn another copy, maybe at a slower speed.  Or try a different color of CD-R media; not all media works equally well in all drives.

A second, less likely possibility is bad memory: how old is the computer, and has it been successfully running other operating systems such as windows?  Another of the boot options is "memtest86+"; try running that for at least 20 minutes and see if your computer passes.
