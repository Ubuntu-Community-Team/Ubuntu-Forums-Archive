### Post by yancek on 2018-01-17
s pointed out above, downloading the UBuntu iso is the first step.  You then need to either burn it as an image to a DVD or use software specifically designed to create a bootable usb.  If that works as it should, you will then have a 'Live' Ubuntu which you can use either to do an actual install or use to test.  If you use it to test, you cannot save any changes as that is the way it is designed.  You would need to install it to a drive on the PC.
