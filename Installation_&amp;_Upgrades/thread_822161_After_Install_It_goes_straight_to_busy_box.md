### Post by avtolle on 2008-06-08
There are many people with the same problem, and there seem to be many solutions, not one of which works for everyone. So, search this forum for Busy Box and review the various threads/posts, and perhaps you will run across one that helps you. 

For those more knowledgeable, do you have a RAID setup, are you running SATA drives; these are but two of the things I've seen. Other issues I've seen range from a hard shutdown (using the power button to turn the computer off) to graphics cards which seem to cause a problem. Posting your system information may be helpful to the knowing ones.

Good luck.
