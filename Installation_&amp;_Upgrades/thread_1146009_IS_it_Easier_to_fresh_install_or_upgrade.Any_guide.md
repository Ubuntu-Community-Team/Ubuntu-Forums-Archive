### Post by snowpine on 2009-05-02
Hi Boo8W, Did your Dell come with Ubuntu 8.04 pre installed from the factory? If so, it is a special version of Ubuntu designed specifically for your mini, and 8.04 is the newest version. 8.04 is a "long term support" release that will be supported through 2011.

If you have a Mini 9, you can do a fresh install of 9.04 if you want. I am running 9.04 on my Mini 9 with no problems. If you have a Mini 10 or 12, I recommend sticking with the special Dell version due to problems with the graphics card.

ps If your only problem is that wireless doesn't work, I recommend trying to fix that problem before you give up and install something different. Check out the search feature. :)
