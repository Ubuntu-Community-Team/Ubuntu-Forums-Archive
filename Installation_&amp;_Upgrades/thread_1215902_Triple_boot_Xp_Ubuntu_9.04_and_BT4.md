to add an os has seemed hard to me so i have never done it. if you install the others first then install ubuntu 9.04 this process would be fairly easy. if you dont want to do that your going to have to install your new operating system, then use

and edit grub to your needs. [COLOR=Red]make sure[/COLOR] you make a copy of grub in case you screw it up.
