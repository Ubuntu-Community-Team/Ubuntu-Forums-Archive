After you've unzipped the file, you can open up a xterm and cd into the directory where the files have been unzipped to. Look for a README or an INSTALL file. It'll tell you what to do.

Usually, the steps are (to be executed in that directory):

./configure
make
make install

### Post by lindsay7 on 2009-03-27
FYI. I just downloaded this for you laptop from the Opera web site.

When you click on the download button there web site knows that you need the Linux version. When you click on that you can choose which version of Ubuntu you need (my case 8.10). I used Firefox to download and you can choose to save or open. If you choose open, it downloads and opens andy you just click install and you are done. the Application/internet menu will have the Opera icon listed and you can start from there or right click on it to add it to you desktop.  All done in about 2-3 minutes.
