### Post by aysiu on 2009-12-05
Go to [http://www.gnome-look.org](http://www.gnome-look.org)

Find a GTK, Metacity, or icon theme.

Download it. Do not extract it or right-click it or enter any terminal commands.

Go to System > Preferences > Appearance

Take the .tar.gz file as is and drag it to the Appearance window
