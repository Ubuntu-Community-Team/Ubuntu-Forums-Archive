### Post by warp99 on 2008-04-09
You can install one version of Ubuntu with multiple desktops since the base install for Ubuntu is the same for all of the versions (Ubuntu, Kubuntu, and Xubuntu). The only difference is the desktop (Gnome, KDE, and XFCE) 

As for installing multiple OSs you can install as many as your system can hold since the boot manager (Grub) will sense other operating systems when you install them and place them in the boot manager list.

Edit:

Read the wiki entry on disk partitioning:

[http://en.wikipedia.org/wiki/Disk_partitioning#PC_BIOS_partition_types](http://en.wikipedia.org/wiki/Disk_partitioning#PC_BIOS_partition_types)

Primary and Logical is the same thing.
