### Post by dabl on 2010-11-02
The Ubuntu OS needs maybe 4.5GB of space. Why do you think you need more than 30GB?

You didn't say which version of Windows.  If it is Vista or later, you need to (a) run defrag in Windows, and (b) use the Windows partition manager to shrink the Windows partition.  Once the Windows partition is shrunk, then you can use GParted to enlarge the Ubuntu partition.
