### Post by pcfan1234 on 2019-09-15
First, do you use USB drive or a DVD?
Apple uses UEFI.
The ISO images are hybrid, so they can be booted up with an old BIOS or an UEFI.
Make sure you USB or DVD is bootable (try another UEFI capable PC if you have one).

### Post by yancek on 2019-09-15
You will need to provide a lot more information if you want help.  THe best way to get that info is to go to the site below while you are booted into Ubuntu, either the installed system or the USB you used to install and downloading and running the boot repair software.
