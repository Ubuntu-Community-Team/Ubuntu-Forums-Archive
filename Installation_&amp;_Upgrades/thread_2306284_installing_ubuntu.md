### Post by sudodus on 2015-12-14
0. You should use a supported version, and 12.04 LTS is the oldest supported version.

1. I would suggest that you start trying the version 14.04.1 LTS, and the flavours Lubuntu, Xubuntu and Ubuntu MATE (and update/upgrade it to be up to date).

2. If you have problems with the graphics, I suggest that you try

- a proprietary driver in 14.04.1
- a light-weight community re-spin based on Ubuntu 12.04.1 LTS for example Bento, Bodhi, LXLE.

3. You can also try the newer version 15.10 which might work better with some hardware.

-o-

I suggest that you tell us about your computer and its hardware

- Brand name and model
- CPU
- RAM (size)
- graphics chip/card
- wifi chip/card

It makes is possible to give better advice.

You can also read these links:

[Old hardware brought back to life](http://ubuntuforums.org/showthread.php?t=2130640)

[Try Ubuntu (Kubuntu, Lubuntu, Xubuntu, ...) before installing it](http://ubuntuforums.org/showthread.php?t=2230389)
