### Post by Bartender on 2010-01-03
If you're SURE you won't miss Vista, and you've just installed Ubuntu recently so haven't spent a lot of time getting it just how you want, the simplest thing to do would be to make sure the optical drive is #1 boot device in BIOS.  Then toss your Ubuntu install CD in the optical drive, reboot the PC, and tell the Ubuntu partitioner to use the entire drive when it gets to the partitioning step.  Step 5 I think?

Easy as it gets.
