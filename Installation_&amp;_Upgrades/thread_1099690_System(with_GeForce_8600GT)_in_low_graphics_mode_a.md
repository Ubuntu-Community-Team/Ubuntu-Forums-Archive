### Post by cariboo on 2009-03-18
Make sure you are fully updated before installing the video drivers. Open a terminal and type:

```
sudo apt-get update && sudo apt-get dist-upgrade
```

Once you have made sure you are fully upgraded try installing the restricted drivers again. I would suggest trying the 180 drivers.

Jim

### Post by icephoenix on 2009-03-18
So apparently upgrade process didn't update my menu.lst and grub was loading old kernel, so nvidia module compilation was failing(as I understood), updating menu.lst and reinstalling nvidia-glx-180 fixed it all!
