### Post by deadflowr on 2024-02-01
She would have had to have signed up for pro, which is free for up to 5 machines.

I'd check in the terminal with
```
pro status
```

There is an option to signup/enable it in Software and Updates so probably where she enabled it.
It can be disabled.

You read up on it here: [https://ubuntu.com/pro](https://ubuntu.com/pro)

### Post by yancek on 2024-02-01
If the system is Xubuntu 20.04, updates support ended in April, 2023.  See the link below.  The Pro extended security maintenance gives security updates for up to 5 home computers for free for up to 10 years.   You might want to upgrade to a more current version.  Ubuntu and some of its derivatives give 5 years support but Xubuntu does not.

[https://xubuntu.org/release/20-04/](https://xubuntu.org/release/20-04/)
