### Post by habys on 2007-09-27
I was all geared up to ask a question, but found the solution while redoing and documenting my case, so here it is if it helps anyone:

Hardware:
Intel DQ965GF Board box retail (AA D41676-601)
BIOS Version: CO96510J.86A.5773.2007.0206.0046
Intel Core 2 Duo E6600
1GB DDR2 667MHz
SATA Western Digital Raptor 74GB Hard drive
SATA ASUS Quiet Track DVD-RW

Software:
Windows XP
Ubuntu 7.04 64bit


Problem:
After installing XP, then installing Ubuntu,  I get "Operating System Not Found" error message after reboot from setup.

Tried reinstalling grub via the livecd to no avail, when I decided to try updating the BIOS.
After installing CO96510J.86A.5967.2007.0919.2223 (or 5967 for short), and reinstalling 7.04, the error message disappears and I get grub!

So maybe this will help somebody! Thanks for this great forum.
