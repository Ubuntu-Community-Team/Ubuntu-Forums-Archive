### Post by Bucky Ball on 2009-10-15
For a start, 9.10 is not released yet and is still 'testing' and your post should be in the Karmic 'testing' forum. If you are looking at getting into Ubuntu 9.10 is not the best place to start (unless you are looking at diving head first into the learning curve and spending a lot of time in a terminal). 

I would install 8.04 LTS, pretty much rock solid. Once you get the swing of things you can change to the 'bleeding edge' release. Karmic is about breaking and fixing right now and not due out til the end of this month. I personally wait for six months at least after the release date but I need my machines to be ultra-stable. :)

ps: your graphic is MASSIVE! You might try attaching the pic instead or perhaps shrinking the file before posting. Good luck with it all and welcome.

pps: if you login is everything working okay? If so and if you haven't already, plug in an ethernet cable (essential even to get wireless up usually), get in there and get all updates (they will be offered automatically if you are online), reboot.
