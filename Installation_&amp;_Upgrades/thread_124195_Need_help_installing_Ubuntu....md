### Post by de_doughboy on 2006-02-01
Dear Friend, I think the answer to most of your queries above is yes. If your a wantAbe user then you should first down load the **GO LIVE CD**. By first burning a bootible Ubuntu CD. Then you will have all the tools needed to try out Linux and test your system before you commit a host machine to it. 

If your worried about your windows environment you had better start making backups. I recommend Acronis True Image v8 for windows. If you want to get around the partitioning menu commands simple use the escape [esc] key. The go back to the main menu until you see the tab for run a shell. You can jump to the shell as soon as the install menu start the select language script.

My advice is to get a cheap PC i.e. P1, (any speed will do,) and start learning Linux with it before committing to a newer notebook/desktop. I personally dislike the GRUB boot loader, and think it should be rewritten. There are other easier to use commercial boot loader tools that you can get to help with the  boot-init like 'osl2000' or 'Acronis Disk Director.' Just go to download.com and snag them.  

As with any free service you have to exercise a great deal of patience. Especially when downloading because it is not a perfect system. It took me five (5) times before I actually got a good iso-boot image of Breezy, but then I'm unlucky. :(  but I was able to get the **GO LIVE CD** to build on the first try, and it has worked on every PC I've tried it on. It's great if you just like to dabble. :) 

If your a bit more serious about using Linux then you might want to purchase a complete Library set from PC World. It's only $16 for a four volume set, and the mini-textbook inclusive is a great desk reference. \\:D/
