### Post by Impavidus on 2021-04-29
The upgrade from a supported Ubuntu release (if any) always goes to the next supported release, so from 20.04 it goes to 20.10. This will change when 20.10 reaches end of life in July. To get to 21.04 using the update manager, you have to go via 20.10. But the upgrade from 20.10 to 21.04 isn't available yet (at least, not using the update manager), as such upgrades are only enabled when they have been tested and confirmed to be reliable enough.
