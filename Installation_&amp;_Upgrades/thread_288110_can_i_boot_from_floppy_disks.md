### Post by fritz_monroe on 2006-10-29
Sure you can.  [Here]('https://help.ubuntu.com/community/Installation/FromHardDriveWithFloppies") is a How To for installing from hard drive.  If you boot off a regular DOS boot floppy, you can transfer the CD to the hard drive and follow the How To.  Or you could modify the boot floppy and have it point to the CD.  Since I'm not a Linus expert, I can't say how to do that.

I haven't installed Ubuntu via floppy, but I have done Mepis like this.

I should have looked a little more.  [Here's]("https://help.ubuntu.com/community/SmartBootManagerHowto") a how to for installing without a bootable CD drive.  And [here]("https://help.ubuntu.com/community/Installation") are some additional installation how to's.
