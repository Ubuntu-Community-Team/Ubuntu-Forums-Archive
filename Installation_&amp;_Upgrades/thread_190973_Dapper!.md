### Post by aristotlewilde on 2006-06-06
Ok, I have been dipping my big toe into Linux for the last 6 or 7 months now.  I had Ubuntu on an old machine, but could never get it to install properly on my main PC.  

As a result, I had my main PC dual booting Fedora Core and XP and I never really got into Fedora.  My Ubuntu machine sat in the closet unused.

This past weekend, I found about about Dapper Drake and was finally able to get Ubuntu installed properly on my main system.  2 days later, I used Automatix to properly configure my Nvidia card and download al the software I need.  I am loving it.  

If only I could get my X-Fi sound card working, I'd be set!!  I may even setup a second set of speakers to use onboard sound with Ubuntu.  (is it possible to have onboard set to be used by Linux while XP uses the x-fi?)

Anyway, I just had to say that Ubuntu rocks.  It is so much easier for a Linux novice to use than Mandriva or Fedora (the other 2 I've tried).
