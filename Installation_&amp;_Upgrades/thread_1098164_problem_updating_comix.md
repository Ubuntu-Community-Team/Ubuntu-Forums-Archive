### Post by taurus on 2009-03-16
Remove the old version from your machine first either with add/remove or synaptic.  Then, download the .deb to your desktop and double-click it to install it.  No need to download the source and build it yourself.

[http://www.getdeb.net/app/Comix](http://www.getdeb.net/app/Comix)
