Is your computer configured to boot up from the CD drive before the harddrive? You can check by putting the Windows install CD in and see if it boots up. If not change the appropriate settings in the BIOS.

Have you burnt the ISO to the cd correctly? (Have made this mistake myself on a couple of occasions :) ) See this link:

### Post by 3rdalbum on 2006-06-01
With Nero, in the StartSmart program, there will be an option to "Burn Disc Image" (it's under one of the buttons). Hit that and you can't go wrong. Don't do "Bootable CD", that's something completely different.

You should see a whole heap of files and folders on the CD when it's done. And I would recommend burning it at 8x.
