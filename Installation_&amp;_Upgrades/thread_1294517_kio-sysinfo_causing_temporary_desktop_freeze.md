### Post by oshunluvr on 2009-10-18
I don't know if this is the correct place to post this...

In the process of adding some packages to my system I noticed my desktop would freeze for 30-50 seconds when I attempted some actions. Namely,  opening the device notifier and selecting a device or typing a letter in the search bar of kmenu.

I had installed several packages but most seemed innocuous, like krusader and various archiving utilities like arj. The one system package was **kio-sysinfo 0.0~svn20090623-jaunty-ppa1**. Removing this and re-loggin in fixed the issue.

Since this is not a ubuntu or kde package I'm not going to bother reporting it to them. I just wanted to let others know not to use it.
