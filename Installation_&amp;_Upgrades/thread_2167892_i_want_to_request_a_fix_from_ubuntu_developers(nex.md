### Post by ian-weisser on 2013-08-15
"It doesn't work" or "It stopped working" or "It doesn't work well" are considered bugs.
If you think you have found a bug, then please report the bug using the proper venue.
See [https://help.ubuntu.com/community/ReportingBugs](https://help.ubuntu.com/community/ReportingBugs)

Developers generally ignore all bug complaints that don't have a real, triaged bug report open.
The bug report must be complete. Developers are not psychic, nor have access to your hardware to reproduce the problem.

Tips:
Help the bug triager with *all* information they need. Respond quickly and completely to any request for more information.
Use complete sentences, proper grammar, and appropriate punctuation. Many developers do not speak English as their native language - make communication easy for them.
Help test patches to fix the problem.
