### Post by kinngg on 2007-08-20
Yes but you would have to make a new partition for your new ubuntu
You can either do that with the installation process where it will guide you on where or which hard drive you want to partition or
you can install a program called partition magic on windows and make a ext2 or ext3 partition
During the ubuntu installation process it will guide you on which files you would like to transfer to your new ubuntu desktop. It doesn't matter if you chose anything after installation you can still access your local disk
And to answer your questions yes you can preserve your files as long as you don't format your vista partition
