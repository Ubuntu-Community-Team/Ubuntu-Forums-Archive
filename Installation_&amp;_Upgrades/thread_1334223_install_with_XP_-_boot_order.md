### Post by darkod on 2009-11-22
The option you are talking about, wubi install, will keep the windows bootloader but it will ADD ubuntu option to the menu. As far as I know you can not boot windows and then select to start ubuntu.
It will be installed inside windows similar to windows app, but you have to choose in windows bootloader.
And yes, windows will remain default so if nothing is touched it will boot into windows.
