### Post by az on 2005-09-01
What is the size of your harddrive?  A default install of ubuntu takes up just over two gigs.  If you need it to take up less, you can install with the

linux archive-copier/copy=false

option which will avoid cacheing about 300 megs worth of packages onto the harddrive for later use.  I use this when I want to install on a really small hard disk.

Hoary had a disk-admin tool which handled the mountpoints of disks that were not set up during the install.  It was unstable, so it was removed before Hoary got released.  It works fine in breezy.

If you are installing fresh, you can always add the mountpoint with the installer.  Just assisn the partition a lablel and tell the intstaller to keep existing data.

### Post by mcduck on 2005-09-01
Someone already mentioned how to get Disk- Computer- and Trashcan icons on desktop, but you can also get program icons there easily: just right-click program you want from Applications-menu, select 'Add To Panel' and then drag the icon from panel to your desktop.
You can also right-click on desktop and select 'Create Launcher' to make your own shortcuts..

What comes to removing stuff from default installation, I wanted to do same thing but there really wasn't anything to remove. I removed Evolution, as I do not need calendar and I like to read my mails with Thunderbird. That's pretty much everything.

Then I installed Bum to select running processes, but there wasn't much either. As I don't need username and password to connect to net I think I could have disabled PPP, and as my prosessor (AMD Athlon XP) doesn't support powernowd It's useless. Still,  I think disabling those would really make any difference in performance so in the end I left everything as it was.
