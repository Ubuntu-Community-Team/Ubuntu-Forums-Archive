### Post by ajgreeny on 2010-03-17
You can certainly use the mint menu in ubuntu, but you must make sure it is the right version, and it's not always been the equivalent mint menu to ubuntu version unfortunately.

I used it for a short time in jaunty, but you needed the mint menu version from the previous mint, if I remember correctly.  I decided that I didn't really like it so removed it anyway, but in my case it was the two packages
mintmenu_4.2.2_all.deb
mintsystem_6.2_all.deb
that were needed, which was from the mint version one behind jaunty.  Perhaps that is your problem also, so remove what you have got and try again having searched out the version needed.  There is plenty of info in this forum if you search, or search also on [www.uboontu.com](www.uboontu.com)
