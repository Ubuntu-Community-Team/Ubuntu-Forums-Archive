### Post by oli888 on 2006-05-21
Here is a guide on how to install Ubuntu if you are on Windows.

[LIST=1]
[*]Go to [http://www.ubuntu.com/download](http://www.ubuntu.com/download)

[*]Choose a mirror

[*]Choose an install CD. For most computers the "PC (Intel x86) install CD" is fine.

[*]Download the ISO file (it will take a while).

[*]Download and install ISO recorder from [http://isorecorder.alexfeinman.com/isorecorder.htm](http://isorecorder.alexfeinman.com/isorecorder.htm)

[*]Double-click the ISO file that you downloaded and it should open with ISO recorder.

[*]Follow the instructions to burn to a blank CD at choose the lowest speed you can get.

[*]Insert the CD into the computer you are installing Ubuntu on and before Windows.

[*]Before Windows loads up, it say press Delete (or another buton depending on your computer) to enter System Configuration (or something similar). Press it and choose to boot from the CD.

[*]Reboot when prompted an the Ubuntu installer should load up.

[*]When it says "boot:", just press enter to begin the installation

[*]Select the appropriate options during the installation using [Up] and [Down] to choose an option and [Enter] to select it.

[*]After a while the installation should restart adn it will install the packages.

[*]A login screen will be presented and you can begin using Ubuntu!

[/LIST]
