### Post by taurus on 2009-02-10
Is there any OS on that second PC?  You don't open program on the CD; instead, you boot your machine with the CD and install Ubuntu from it.  Therefore, make sure you burn the ISO file as an image file with your burner app, NOT a data file.  Then, go into the BIOS and make sure you have CD/DVD as the first bootable device.

When you boot from the CD, you have an option to use it before you install it if you want to give it a test run.  Remember, it would run a little slow since it has to access everything on the CD.

### Post by taurus on 2009-02-10
Do you have a separate/empty partition to install Ubuntu on?  If you have burnt it as an ISO image and change the BIOS to boot from the CD/DVD first, just restart your machine, making sure the disc is in the drive and off you go.

And if you plan to have the installer to resize your harddrive, making room to install Ubuntu, it's best if you boot into windows first and defrag your harddrive a couple of times first.
