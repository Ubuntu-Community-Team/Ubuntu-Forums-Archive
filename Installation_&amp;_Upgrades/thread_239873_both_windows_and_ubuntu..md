### Post by K.Mandla on 2006-08-19
Let me see. ... I did this once a very long time ago, so I'm a little foggy on it. ...

If I remember right, the installer program gives you the option of installing to one hard drive or the other. You'll have to know ahead of time if your windows drive is the master or slave device, and I believe master should show up as hda and slave as hdb (sorry if I'm getting too technical here).

You'll be able to install to the second drive, and the installer should give you the option of putting grub (the boot loader) on one drive or another. 

From that point, when you turn on the computer, you'll get a small menu that asks you if you want to start Windows or Ubuntu. You can edit that menu from Ubuntu to boot to one or another. ...

I think that's it. Does that answer your questions?

Edit: [Here]("https://help.ubuntu.com/community/WindowsDualBoot")'s a page that describes how to build a standard dual boot. That might help you some.
