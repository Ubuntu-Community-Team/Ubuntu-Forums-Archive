### Post by lidex on 2010-01-17
You have a wubi install, which means ubuntu was installed as a file in your windows partition. Not much you can do about that. One option is to purchase another HDD and install ubuntu stand-alone on it.

If there is anything to do from within windows, these links may help:
[http://wubi-installer.org/faq.php]("http://wubi-installer.org/faq.php")
