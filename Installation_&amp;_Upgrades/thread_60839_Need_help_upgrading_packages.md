### Post by DJ_Max on 2005-08-29
1.) This isn't going to be an option all the time, since Upgrading essential package A, may require upgrade minor package B & C. Besides, you won't get too many updates, unless it's for security with Ubuntu.

2.) They will come from different repositories, all you have to do is use Ubuntu Security Update Manager. Which will only update security fixes.

3.) What repositories are you using?

4.) Both APT & Synaptic will, Synaptic is just a GUI interface for APT.
