### Post by mithunc on 2012-05-24
Sorry to resurrect this, but I experienced the same problem today and yesterday. This is both on a machine running Precise and one with Lucid. The solution for Totem, in Lucid anyway (Totem version 2.30.2), is as follows.

Edit -> Preferences
Color Balance -> Reset to Defaults

Somewhere at some point it decided to change the "Hue" value to its lowest possible value. Problem solved.
