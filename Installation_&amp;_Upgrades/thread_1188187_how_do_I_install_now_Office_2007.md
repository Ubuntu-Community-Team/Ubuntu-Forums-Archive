### Post by Mark Phelps on 2009-06-15
You're going to have a tough time.  

CodeWeavers (the Wine folks) maintain a database of application compatibility with MS apps.  The link below is to their page for Microsoft Office 2007.  While the original version might work if you hack it enough, you can see from the page that once you upgrade to SP1, you're dead:

[http://www.codeweavers.com/compatibility/browse/group/?app_parent=1911;]("http://www.codeweavers.com/compatibility/browse/group/?app_parent=1911;")

### Post by Timtro on 2009-06-15
You're better off with Word 2000, or 2003, which are supported in Crossover Linux (which is sort of like a beefed up Wine, from the same makers).

[http://www.codeweavers.com/compatibility/browse/name/?sort](http://www.codeweavers.com/compatibility/browse/name/?sort)[app_name]=DESC

Good luck. My advice is to figure out how to do what you need without using too many Windows software. It's not easy, but pretty much anything that can be done well in Windows can be done well in Linux too. The only exceptions to this rule are because there is less commercial support for Linux, not that the OS is in any way limiting.
