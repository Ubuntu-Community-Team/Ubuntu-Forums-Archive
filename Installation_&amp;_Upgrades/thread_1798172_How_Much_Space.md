Firstly, that is normal...
4GB = 4000Mb, 1GiB = 1024Mb -> 4GB is not equal to 4GiB ... 4GB = 3,9GiB

Installing or a Live USB?

A Live USB version will fit on any 1GB USB, since it is made out of an ISO (650MB)

A full install will not be possible, You need more space.
