### Post by lvlo on 2009-06-10
Use this tutorial to add PPA for OpenOffice: [http://news.softpedia.com/news/How-To-Install-OpenOffice-org-3-0-in-Ubuntu-8-10-96449.shtml](http://news.softpedia.com/news/How-To-Install-OpenOffice-org-3-0-in-Ubuntu-8-10-96449.shtml)

This PPA contains OpenOffice 3.1 now.

**UPD:** don't forget to change "intrepid" to "hardy" in deb-line. It will looks like: ```
deb http://ppa.launchpad.net/openoffice-pkgs/ubuntu hardy main
```
