### Post by ijason on 2007-11-28
you can probably remove it by going into the system>admin>synaptic package manager. then search for java and it will tell you all the instances of it installed on your machine. you can select to remove them from there. 

you may then want to reboot and then re-install java through synaptic... search for 'java nonfree' to get the right one.
