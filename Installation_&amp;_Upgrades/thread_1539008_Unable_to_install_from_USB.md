### Post by dcollier on 2010-07-26
If you just copied the disto to the jump drive it may not work like you want it to.  There is a nice utility that you can use via windows or linux called LiLi (Linux Live usb creator) that will make a usb/jump drive bootable using any distro of your choice.  Give it a shot. Here is a link [http://www.linuxliveusb.com/](http://www.linuxliveusb.com/)
