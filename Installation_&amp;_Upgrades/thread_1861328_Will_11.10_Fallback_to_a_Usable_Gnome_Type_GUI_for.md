In 11.04, Unity3D was the default, fallback was Gnome2 "Ubuntu Classic" and Unity2D could be installed from the Software Center.  Both an Dist-Upgrade from 11.04 and a new fresh install to 11.10 result in the same as in relates to Gnome-

In 11.10, Unity3D is the Default, Unity2D is the fallback. Gnome3 is not there.  Gnome3 can be installed from the Software Center. Go to the search filter there and type in "gnome destop"  Select learn more. Scroll down and select what options you want installed with it. Press on the apply changes in options. Select install.

After the install completes, reboot.  At the login screen, at the upper right corner of the login dialog box, there is a white Ubuntu icon... Click on it with your mouse. A pulldown menu will appear with your desktop options. Select "Gnome Cliassic." That will load the Gnome panels that mimic what past users are used to...

Note that the System Menu is now under the Users Name (upper most-right), under System Settings... Just like in all the other desktop menus now.
