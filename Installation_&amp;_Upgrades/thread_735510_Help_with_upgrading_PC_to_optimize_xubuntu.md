### Post by Existentialist on 2008-03-26
The processor is still going to bottleneck the system somewhat, but upgrading the ram is probably the most cost effective option.  Having more ram will allow for more cache which eliminate having to read everything from the hard drive, which with the slower system bus, should make things feel faster.  Other than the ram, you can remove any unneeded services from running for a little better performance.
