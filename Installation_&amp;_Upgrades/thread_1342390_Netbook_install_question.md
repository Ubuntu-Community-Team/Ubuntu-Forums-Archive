### Post by cyberey66 on 2009-11-30
When you burn the CD use the slowest settings.  You don't know how many install issues are just a result of a bad burned CD.  Also, if you are using a CD, do the self check on the main screen.

In my opinion, I would just use a USB stick if you have one.  It eliminates the need for an external cd drive on netbooks.  I used it on my install on my gateway netbook.  You can make a bootable install cd in windows by just point and clicking using this program:

[http://www.linuxliveusb.com/](http://www.linuxliveusb.com/)

Just download the ISO of the version you want to install and then use the program to create a bootable usb key from it.

Good luck!

### Post by darkod on 2009-11-30
I haven't on that model, I have Samsung NC10 but what I can recommend is to download both Ubuntu Netbook Remix and Desktop 32bit images. Put one on a usb stick (or maybe you're planning to install with usb cd-rom) and use Try Ubuntu option. See how you like it.
Then put the other image on the stick and try it too.
I am saying this because there is little difference between UNR and Desktop 32bit, the desktop version will run just fine on a netbook too. On the other hand there is difference in the menu layout and it all depends on personal preference. Not knowing this, because I was new to Ubuntu too, I installed UNR straight away, and hated the menu compared to desktop version. So I ended up installing desktop 32bit over it 2 days later.

The Try Ubuntu option will also allow you to see if everything is working on the netbook as you expect. Try surfing, video, audio, etc. Before starting the install.
