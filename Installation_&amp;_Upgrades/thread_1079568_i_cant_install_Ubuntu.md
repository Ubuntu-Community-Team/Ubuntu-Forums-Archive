### Post by avtolle on 2009-02-24
Take a look at [https://help.ubuntu.com/community/BootOptions](https://help.ubuntu.com/community/BootOptions) regarding how to use boot options.

On 8.04, IIRC, there was a problem for some with recognition of SATA drives during installation. The boot option to use for installation was "all_generic_ide", which worked for many people. Good luck.
