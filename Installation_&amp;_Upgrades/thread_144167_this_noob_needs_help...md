### Post by Bartender on 2006-03-13
dus - 
If you're still out there, don't charge into this.  Read it thru, take your time, you'll notice that the command lines are different for Kubuntu than Ubuntu.  Don't start typing in the wrong ones.  Each command line must be typed in (or copy/pasted) then hit the Enter button.  You should see Ubuntu get some fairly small packages, then say Done and go back to a waiting command line.  Then type/paste the next command.  
In Ubuntu, after you've typed in the second command line (which downloads/installs the Automatix installer) then follow Arnie's steps to start Automatix.  Then pick which programs you want to download, then sit back and watch the fun.
