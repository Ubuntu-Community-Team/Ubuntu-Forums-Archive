### Post by xjcannonx on 2008-12-18
It is actually really easy
1.Download ubuntu 8.10 from the website
  1.a. Make sure you burn the downloaded iso at slowest speed on a cd-r preferably(but a cd-rw or even dvd-r/rw will work) burn it as disc-at-once or option similar to that...also it has to be burned as an iso, not data or whatever

2. pop it in your computer that you want to wipe clean.
  2.a. You have to set your computer to boot from cd drive first, usually hit one of the F buttons or esc/del, i dont know your particular case
  2.b. Once you get it to boot from the disc, ubuntu should come up and you should see an option to "check cd" make sure you do this to make sure the cd burned correctly.
  2.c. You can then go through the installpretty easily and when you get to partitioning you can select the first option i believe which is Guided - entire disk.

Dual boot is next but lets see if this can be knocked out first...
