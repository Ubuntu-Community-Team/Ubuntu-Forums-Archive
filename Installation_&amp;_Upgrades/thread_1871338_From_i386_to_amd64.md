### Post by nickleboyblue on 2011-10-28
Since the software is free, I would say your best bet is to do a fresh install WITHOUT updating... The idea is this:  Upgrading to the next version is great, but after about three or four versions, it starts working less and less and you end up using the latest desktop but with the older set of applications, not to mention config file problems, etc.

I would suggest you back everything up and install the latest version from cd.  That way, you know that pretty much everything will work out of the box without tweaking of fixing, and you can simply restore all of your data from backup.

Make sure you back your information up; it can't be recovered (easily) after wiping the hard drive.
