If you get a grub menu, then you have booted thru grub and boot script does not really help.

Have you tried nomodeset or any other boot parameters? 
What video do you have?

       How to set NOMODESET and other kernel boot options in grub2

When booting without splash or quiet & see the drivers being loaded, do you see any obvious errors. Where it stops is not usually the issue often checking battery whether you have one or not, but some driver not loading earlier in the process.
