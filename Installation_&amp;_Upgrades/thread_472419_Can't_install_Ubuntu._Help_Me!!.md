### Post by lemonwonder on 2007-06-13
Ok have you actually installed or just run the live cd? (I have win98)

Download the 7.04 for PC image, and burn to cd as ISO (not as data) at an extremely slow speed.

Then place it in the cd tray and re-start. Then when the title screen opens, press enter. You can walk away for 10 mins, come back and it should be loaded.

Click on install.

Then do everything by what you think up to the partition.

For partition, use the first option, and set the slider to 10gb at least (if 10gb isnt free, just do as much as possible) 

Let it partition, then you set in your username and password (dont forget password or user)
Set it to go, you should arrive at a summary screen. Select ok, it shouldnt delete files, i did mine today and it didnt, it just divided up my HD which is what we want.

When its loaded everything, restart your computer, select UBUNTU from grub and enjoy!

(To get into admin account for any reason, login to your user and go to "login options" under administration and edit under security)

> Minimum system requirements

To run the Desktop CD (LiveCD + Install CD), you need 128 MB RAM to run or 192 MB RAM to install. The Alternate Install CD only required you to have 64 MB RAM.

To install Xubuntu, you need 1.5 GB of free space on your hard disk.

Once installed, Xubuntu can run with 64 MB RAM, but it is strongly recommended to use at least 128 MB RAM.
