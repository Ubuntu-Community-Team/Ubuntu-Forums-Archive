### Post by yancek on 2016-05-06
If you were running Ubuntu 14.10, you would need to first update to the next release which is 15.04.  Neither are supported any longer (see the link below) so the repositories are no longer available to update/upgrade.  You can upgrade LTS versions from one to another (example 14.04 to 16.04) but if you are using a non-LTS release you need to go 14.10 to 15.04 to 15.10.

Just to verify, were you actually using 14.10 or was that a typo?  Do you have an UBuntu Live CD/DVD flash drive, or any other Linux system on DVD/flash drive?

[https://wiki.ubuntu.com/Releases](https://wiki.ubuntu.com/Releases)
