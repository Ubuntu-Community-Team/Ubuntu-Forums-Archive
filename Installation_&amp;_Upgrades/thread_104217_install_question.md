### Post by Rackerz on 2005-12-15
Just choose the partition Mandriva is on and install Ubuntu over that, making sure to format it first. You can dual boot and keep Windows. 

By Lilo i think your talking about the Lilo boot loader? If so, yes Ubuntu does use it but when installing it will ask you to install the 'GRUB' bootloader, you must find and select the 'Lilo' boot loader.
