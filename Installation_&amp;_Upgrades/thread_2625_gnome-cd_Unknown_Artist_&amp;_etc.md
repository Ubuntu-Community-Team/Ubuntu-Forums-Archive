### Post by emperor on 2004-10-29
Anyone know how to fix the CDDB access problem for gnome-cd? All I get is "Unknown" for every field. In contract, Sound Juicer seems to read the Album info OK. Looking at the Gnome Config using "gconif", a CDDB slave is setup to "freedb.freedb.org" port 888 which is idential to Fedora Core 2 which displays the CDDB info correctly. It seems like there are a lot of little things that just don't work right in Ubuntu just yet, mainly with "udevs" and automounting of cdrom/dvd media.[img]http://www.ubuntuforums.org/images/smilies/eusa_boohoo.gif[/img] I hope one of development guys are working some of the bugs out!
 
 As a work around, I downloaded XMMS and the cdplay plugin. It has ODDB support and it works correctly and the song titles are displayed!
