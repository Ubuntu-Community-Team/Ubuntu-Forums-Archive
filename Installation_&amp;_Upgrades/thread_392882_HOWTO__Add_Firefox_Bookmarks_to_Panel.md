### Post by lordnibbla on 2007-03-25
I don't really know if this is common knowledge but I do believe that I liked what this tutorial I made has done for me :D

Ok, first off I should say that I am using Ubuntu 6.10 Edgy although these steps could probably be applied to other linux distros.

**1.**Make a new folder at /home/usrname/favicon/.

**2.**Save all of the sites favicons to /home/usrname/favicon/ by going to [http://site.com/favicon.ico](http://site.com/favicon.ico) and change their names to their liking as you do so.

**3.**Goto Firefox and browse to a website you want to incorporate into your panel. Then drag the link onto the panel.

**4.**You should now see a generic icon for the link you provided on the panel. Right click this icon,properties. And then click on the picture of the icon in the menu.

**5.**Now type  /home/usrname/faviconinto the address bar here to see all of your icons  :yes: select and hit ok, close.
[[IMG]http://img86.imageshack.us/img86/1547/screenshot1of9.th.png[/IMG]](http://img86.imageshack.us/img86/1547/screenshot1of9.png)

**6.**To change icon size right click panel, properties.
[[IMG]http://img154.imageshack.us/img154/9404/screenshot6sk1.th.png[/IMG]](http://img154.imageshack.us/my.php?image=screenshot6sk1.png)

**7.**Now you repeat 2-4 for the other links you want to add, except for the fact that you dont need to type /home... in the bar because it is already in the dropdown menu!

**8.**Profit!

a pic of them in action:
[IMG]http://img170.imageshack.us/img170/3821/ssoy7.png[/IMG]

You can also put them on the top obviously.
The only problem I came across was that some of the icons were stubborn and required copying one layer in gimp and pasting that into a new ico replacing the old one.  If that makes any sense..
Hope that this was helpful.
