### Post by h_boise@yahoo.com on 2014-04-24
I installed the update of Photofilmstrip, version 2.0.0. It is not in the Ubuntu Software Center but can be downloaded as a Debian package from photofilmstrip.org. 

The new software did not work with on Ubunto 12.04 computer. The longest show it would process was 1:59 minutes long. That is minutes, not hours. It's about 25 images. I try to limit my shows to 20 minutes.

Luckily, the last version 1.4.0.1 is still in the Software Center. I removed 2.0.0 and reinstalled 1.4.0.1.

I have made this post as a caution to other users, not a request for help

I will stick with the older version until a newer one is available from the Software Center.

I do have a five year old machine. Maybe, an up-to-date machine will work.
