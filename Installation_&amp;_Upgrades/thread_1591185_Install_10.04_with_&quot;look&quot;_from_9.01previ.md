### Post by jon.reeve on 2011-03-02
Old post, I know, so you may have already solved this issue (or not?) but still, thought I'd contribute. 
Unless I'm mistaken, I think the bootup technologies for 10.04 changed dramatically from previous incarnations--plymouth instead of usplash, something having to do with xsplash, so I don't think it's possible to get the old theme back unless you want to try porting the old usplash theme to plymouth. 

That having been said, there are a ton of great plymouth themes that you can customize your box with. Check out [http://www.webupd8.org/2011/01/earth-sunrise-is-gorgeous-plymouth.html](http://www.webupd8.org/2011/01/earth-sunrise-is-gorgeous-plymouth.html)
