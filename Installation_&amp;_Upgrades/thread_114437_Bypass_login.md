### Post by byen on 2006-01-08
ok..since you said Ubuntu..im thinking it is gnome that you are using: Do this: In the menu.. go to System>Administration>Login Screen setup>General Tab>Automatic login>select username 

If you are using KDE-Desktop:
kmenu> System Administration>Login manager>convenience>enable auto login

hope that is what you were asking.Goodluck
