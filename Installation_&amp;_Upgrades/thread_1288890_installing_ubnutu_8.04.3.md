### Post by rreese6 on 2009-10-12
Normally if you are installing an Operating System and you have errors, you should work on those first.
could you post some of the errors you got?
One thing to be sure of is that the copy of Ubuntu doesn't have errors on the CD you burned. Boot up the LiveCD and select Check CD Media from the menu. if you have errors, then you need to make a new CD.
I suggest that you burn the CD at the slowest speed possible also, like 4x or 8x.
if the Check CD Media test passes, then we need to address the errors one at a time.
