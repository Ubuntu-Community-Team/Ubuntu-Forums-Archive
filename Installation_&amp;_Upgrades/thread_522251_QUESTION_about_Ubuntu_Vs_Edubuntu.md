### Post by kidders on 2007-08-11
Hi there,

For the mostpart, any Ubuntu variant can be turned into any *other* Ubuntu variant with little more than a few commands, so no matter which one you start with, you can still access all the same applications & features. Which one to choose is really only a question of how you want Ubuntu to feel, straight out of the box.

If you don't like Edubuntu, you can feel free to choose Kubuntu (or something else) instead, without losing any functionality. It's not quite the same as in the Windows world, where choosing a "home" version over a "pro" version, for example, makes real long-term differences to what your OS can do.

I hope that helps.
