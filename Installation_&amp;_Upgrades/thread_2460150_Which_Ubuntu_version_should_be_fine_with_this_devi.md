### Post by Impavidus on 2021-04-03
20.04, that's the current LTS version. Unless you really want the latest interim release (20.10; 21.04 will be released this month), but that's not really recommended for new users. There's no point in installing an older version.

There are different flavours though: Ubuntu, Kubuntu, Xubuntu, several others. Your computer is powerful enough to run each of them, so that's just about personal preference. Watch some screenshots, run some livedisks and pick the one you like most.

BTW, you do know that Ubuntu is not a drop-in replacement for Windows, right?
