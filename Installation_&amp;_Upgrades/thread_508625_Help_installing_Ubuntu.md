### Post by dabl on 2007-07-24
I would say first check the "md5 sum" of the ISO that you already downloaded. md5 checksums for the ISO files are on the Ubuntu site where you found your ISO -- make sure you get the one for your file.

If the md5 checksum is a match, then the problem is probably the burn process.  You should set the burning speed down to 4X when you burn this particular CD.

If the md5 checksum is not a match, on your downloaded ISO, then you need to try again with the download.
