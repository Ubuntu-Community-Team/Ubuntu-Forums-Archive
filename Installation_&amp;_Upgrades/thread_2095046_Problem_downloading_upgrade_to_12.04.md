### Post by rajhanschinmay on 2012-12-16
You can download the Ubuntu 12.04 CD.
Kindly make sure that the new version is only different in version name and not the nit image.
i.e. if u r using 32-bit then it is safe to download 32-bit Ubuntu 12.04 .iso image.
Then create a USB boot pen drive using usb disk creator option.

Then restart computer / laptop and boot from pen drive.

At one screen u wil get 3 options:
1) Upgrade 10.04 to 12.04
2) Erase everything and install 12.04
3) Something else.

You can select 1st or 3rd options.
Choose 3rd options only if u r knowing where is ur /, swap and /home on ur disk. If you know that then u can reassign them. Do not tick mark format. This means it will not format those drives and retain ur earlier files as they r.

Thank you.
