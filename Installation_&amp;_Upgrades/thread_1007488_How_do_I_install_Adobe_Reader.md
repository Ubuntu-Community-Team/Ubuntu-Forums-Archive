Acroread (Adobe Reader) is in the Medibuntu repositories. To get it...

First, install the Medibuntu repositories if you haven't already done so, and enable them in "Software Sources".

[https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)

Then go to Synaptic, search for, and install the mozilla acroread plugin. That will install not only the Firefox plugin, but the Adobe Reader 8 package as well.

You should then be all set.

Edit:

You'll notice that my link and ajgreeny's are the same for the Medibuntu repositories. Apparently I was composing at the time that he posted.
