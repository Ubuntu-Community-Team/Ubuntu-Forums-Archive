### Post by ian-weisser on 2014-07-14
Pretty much the same way you install lubuntu on an internet-connected machine.

You download the installer.
You create an install USB.
You boot the target system from the install USB.

After the install, the non-connected machine obviously won't receive updates. And the method of installing additional software is slightly different.
