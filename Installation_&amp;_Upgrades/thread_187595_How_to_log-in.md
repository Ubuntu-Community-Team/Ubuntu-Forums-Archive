### Post by strawberry on 2006-06-03
Solution:
Boot with install cd,
Choose 'Rescue a broken system'
Ask shell for proper partition (where new system installed) 
Type 'adduser *username*' then password twice 
Close the rescue process and reboot.
You can login the defined *'username'*
8)
