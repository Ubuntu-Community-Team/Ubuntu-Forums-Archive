Use the Ubuntu Live CD.  It will run Ubuntu 5.10 (Breezy)  off of the CD and not install it.   Be sure that your BIOS is set to boot from your CD (ROM/Writer) before it checks your hard drive. (Example of Boot Check:  1. Floppy 2. CD ROM/Writer 3. Hard drive.)

### Post by heimo on 2005-12-29
Something that must be said about running Live version (from CD) is that it's naturally quite much slower than running from hard disk. Booting up takes quite much longer, user interface is slower. But it's definitely very nice way to check what Linux desktops look like and it also serves the purpose of checking hardware support / how well your hardware is detected. If Live version works well, it'll most likely install on your system.

You can also install Ubuntu (or any other Linux distribution) on the same hard disk with Windows - this is called [dual booting]("https://wiki.ubuntu.com/WindowsDualBootHowTo").

There are many other linux distribution too which come with "Live CDs" - for example: [Suse]("http://www.novell.com/products/desktop/") and KDE version of Ubuntu, called [Kubuntu]("http://www.kubuntu.org/") Go ahead and try different versions before deciding. :) Personally I _really_ like Ubuntu and [Debian]("http://www.debian.org/").
