### Post by rufius on 2007-04-18
Well you can't "upgrade" per se. You have to completely reinstall the system. If you haven't got a seperate partition for /home then you should backup that folder as well. 

Distributions are completely different operating systems so you can't upgrade from one distribution to another, you can only upgrade from a distribution release (ie: Ubuntu 6.10) to a newer distribution release (ie: Ubuntu 7.04).
