### Post by Ub1476 on 2008-03-18
[Emesene]("http://emesene.org/") (probably in repos) is a very feature rich instant messenger apps written in Python, which has webcam support.

You can read the official doc about [Webcams]("https://help.ubuntu.com/community/Webcam") on Ubuntu for help.

Pidgin is a very lightweight IM which main feature is to support all available protocols, therefore webcam and such is not a feature in it. It has a nice plugin system though, which you can install through Synaptic.

Indeed, *Pidgin* does not support video. If you want to use your webcam with *MSN*, you can try *AMSN*. Also, *Kopete* has video support for some protocols (although I don't know for which of them).

Some other options would be *Skype* or *Ekiga*.
