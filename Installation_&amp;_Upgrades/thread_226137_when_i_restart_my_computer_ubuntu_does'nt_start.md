### Post by confused57 on 2006-07-30
Here is an excellent iso burning "HowTo":

[http://psychocats.net/ubuntu/iso.html](http://psychocats.net/ubuntu/iso.html)

Basically, you need to do a md5sum check of the iso to make sure it downloaded correctly, burn it "as an image"(not data) at the slowest speed your burning software allows, then make sure your bios is set to boot the CD drive before the hard drive.
