### Post by moonoo on 2010-08-17
The site has an installer ready for you to use:
GNU/Linux-x86 Installer  

[http://sourceforge.net/projects/tuxguitar/files/TuxGuitar/tuxguitar-1.2-linux-x86-installer.run](http://sourceforge.net/projects/tuxguitar/files/TuxGuitar/tuxguitar-1.2-linux-x86-installer.run)

Download it
open nautilus
find the place you downloaded the file to
right click and go to properties
select permissions
tick the "Allow executing file as program" box
close
double click

The Bitrock installer talks you through the rest

Happy Tabbing
