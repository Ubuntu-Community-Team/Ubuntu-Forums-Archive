### Post by SonicTheHedgehog on 2006-09-14
Is there a way that I can install Ubuntu to my Hard Drive, but not install GRUB.  That way when I turn my computer on it'll just go to Windows normally, but if I wanna use Ubuntu can I make a boot disk that I can use to boot the computer into Ubuntu withut installing GRUB on my HDD?

---

### Post by confused57 on 2006-09-14
You can use the Dapper alternate cd and install grub to floppy during install:
[http://www.ubuntuforums.org/showthread.php?t=188819](http://www.ubuntuforums.org/showthread.php?t=188819)

That way, you can insert the floppy to boot Ubuntu, without the floppy, your system boots to Windows.

The Desktop live cd doesn't give you the option of where to install grub, it's automatically installed to the mbr.
