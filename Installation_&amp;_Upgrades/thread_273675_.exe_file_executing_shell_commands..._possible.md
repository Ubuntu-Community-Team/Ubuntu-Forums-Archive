### Post by andiii on 2006-10-08
Create a textfile and put in:

#!/bin/sh
cp ~/backup /media/usb
exit 0


Of course you can put in more commands and lot more details..
Make the textfile executable (chmod a+x filename)
If you want to execute the script by just typing the script name the file has to be in a folder that is in your $PATH. I use ~/bin for all my scripts.
