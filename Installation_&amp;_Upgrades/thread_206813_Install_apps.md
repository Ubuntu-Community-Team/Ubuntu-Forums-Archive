Apt and Synaptic are your friends.  Apt is the command line tool for installing software that satisfies dependancy issues, and Synaptic is a GUI for Apt.  You can find Synaptic under the System --> Administration menu.  From there, you should be able to search for just about any software you are looking for, including nVidia and Java applications.
