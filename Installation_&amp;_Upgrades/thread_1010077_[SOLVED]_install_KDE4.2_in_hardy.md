### Post by tommcd on 2008-12-14
If KDE 4.2 is not available in the Ubuntu studio repositories, then your best bet would be install Kubuntu 8.10 or Ubuntu studio 8.10. Or dist-upgrade to Ubuntu studio 8.10. Trying to install KDE from the 8.10 repos in Hardy (8.04) would likely lead to a lot of dependency issues that could break things.
