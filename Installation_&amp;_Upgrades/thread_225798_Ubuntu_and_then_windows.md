### Post by orb9220 on 2006-07-30
For ubuntu to setup dual boot it needs to see all the drives so it knows how to write the grub.

The proper way to do dual boot is to have both drives hooked up then install xp first then install ubuntu.

There are many threads here on installing a dual boot and also editing grub menu. Do a search on grub menu or dual boot to understand the process.
