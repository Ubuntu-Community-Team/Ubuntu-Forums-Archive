### Post by metoo on 2005-11-26
He (and you) will not want this integrated graphics thing. It causes problems and is very slow. Go for a nvidia graphics card. Even a 6200 is much faster than the integrated one (but too slow for modern shooter games). NVidia is easier to install (3D mode) then ATI. That's why I recommend NVidia.

256MB only? Then I 2nd the advice to use 512MB. You can get it going with 256 but if you can, make it 512. Depends of the usage. 256MB RAM and a 64Bit CPU is not a balanced pair.
