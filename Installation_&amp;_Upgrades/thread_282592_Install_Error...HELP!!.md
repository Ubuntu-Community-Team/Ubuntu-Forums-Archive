### Post by Sef on 2006-10-23
> 1. make a partition with Partition Magic(Swap, win XP. ext2fs)

Partition Magic and Linux often don't do well together.  Instead use [GParted]("http://gparted.sourceforge.net/").  It has a graphical interface like P.M.

> My computer is Penttium III 400 Mhz, ram 192 MB

Live CD's X window boot is too slow.

I want...command line install

Ubuntu will be real slow on your computer.  Xubuntu would be better.  Best would be fluxbox or icewm; to install them, read [Installation/Low Memory Systems]("https://help.ubuntu.com/community/Installation/LowMemorySystems").
