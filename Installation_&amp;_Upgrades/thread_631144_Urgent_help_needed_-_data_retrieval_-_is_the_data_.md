### Post by sethvath on 2007-12-04
You might still be able to recover part of it using photorec [http://ubuntuforums.org/showthread.php?t=417761](http://ubuntuforums.org/showthread.php?t=417761)

I'm assuming there's still windows on the other partition, if so install the software and try to salvage it. There's many other data recovery tools for windows but most of them are not free, unlike photorec. Thats your best bet for now. 

To minimize overwriting any further data on your D drive, make sure you dont install anything to your D drive. If its only 1 hdd with 1 partition, remove the whole hdd and mount it on another system to do the recovery.
