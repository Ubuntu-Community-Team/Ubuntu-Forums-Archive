Remastersys or Relinix will do the job for you.

[https://debianhelp.wordpress.com/2011/10/12/relinux-a-new-fork-of-remastersys/](https://debianhelp.wordpress.com/2011/10/12/relinux-a-new-fork-of-remastersys/)
[]("http://download.tuxfamily.org/madbox/")
If you want a cd-r and not a dvd-r size, I would install Madbox Ubuntu disto, and install Remastersys on Madbox Ubuntu, and then install whatever packages you want..  Create your Remastersys dist. Test with Startup Disc Creator and a Flash Thumb Drive. Madbox is very tiny to begin with.. I've done it myself, and I have been able to keep it under 650mb in live installation disc remastersys iso.
