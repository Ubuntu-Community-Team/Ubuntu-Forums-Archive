### Post by louieb on 2010-05-26
Upgrading to 10.04 with the Live CD is not an option.  Due to space limitations the Live CD does not include the necessary software. 

You can use - as you have read - the alternate CD or a network upgrade.  Does not matter - in the end you will wind up with the same results.   

In the end you will still have to upgrade some packages over the network - Any software packages you have installed that are not on the Alternated CD - and  -  any packages updated since the alternated CD was released.

[https://help.ubuntu.com/community/LucidUpgrades](https://help.ubuntu.com/community/LucidUpgrades)
