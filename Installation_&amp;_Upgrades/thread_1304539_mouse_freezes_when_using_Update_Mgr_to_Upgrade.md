### Post by wanderingarticfox on 2009-10-29
OK Guys and Girls: I figured it out so here it is for others with old systems!
I'm running on an old 1.33 GHz CPU with 1 GB RAM.
The problem was a hang up in dropping old software from 9.04. The solution was to run the Computer Janitor to rid the system of the old files then to rerun the update manager. It took a while with a DSL hook-up and another while with the 1.33 GHz/1GB system but I do love the new wallpaper of 9.10. I hope this helps anyone with an older system. I have fired MS and plan to build a new system this spring Ubuntu or bust :)
