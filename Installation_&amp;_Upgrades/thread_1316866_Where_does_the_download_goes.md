### Post by Fire_Chief on 2009-11-06
For an inplace upgrade (not a clean install), you want to use the Alternate CD instead of the Desktop CD. See how-to steps here [http://www.ubuntulinux.org/getubuntu/upgrading]("http://www.ubuntulinux.org/getubuntu/upgrading")

32-bit torrent [http://releases.ubuntu.com/9.10/ubuntu-9.10-alternate-i386.iso.torrent]("http://releases.ubuntu.com/9.10/ubuntu-9.10-alternate-i386.iso.torrent")
