### Post by Tristan Carlson on 2009-06-15
I need to reinstall Ubuntu 9.04 from a Live CD; my first installation with the CD worked fine, but now after updating various programs are failing, and I'd like to get a fresh start, by the technique found here: [http://ubuntutip.googlepages.com/reinstall](http://ubuntutip.googlepages.com/reinstall). However, when I put the CD in the computer and turn it on, there's no option to boot from the CD. To clarify: I'm trying to reinstall Ubuntu 9.04 on a machine that's already running it, but it lacks an option to do so. So how does one exactly boot from a Live CD on a computer that's running Ubuntu? Thanks to anyone who can provide me with a solution.

---

### Post by merlinus on 2009-06-15
You need to look in your bios and make sure the cd boots before the hdd.  It usually is one of the F keys that you press right after startup, but it may be Del.

Perhaps googling for your specific model can help, if you no longer have the docs.
