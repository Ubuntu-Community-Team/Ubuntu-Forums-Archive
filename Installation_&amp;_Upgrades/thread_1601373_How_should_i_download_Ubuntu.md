### Post by P4man on 2010-10-20
Wubi=trouble. Avoid it, unless for some reason you really cant or dont want to touch your partitions. You'll run in to trouble when you want to resize partitions, if you screw up and need to fix your install, and each time you update or upgrade there is a ridiculous high chance it breaks the bootloader. There is a small performance hit too. Only advantage is that its really easy to install for windows users.

As for the 32 bit version. there are endless discussions on that. Basically, it usually doesnt matter. 64 bit is a tiny bit faster and supports more than 4GB ram which already is far more  than most ubuntu users ever need; downside, potentially some minor issues with flash playback and wifi drivers.
