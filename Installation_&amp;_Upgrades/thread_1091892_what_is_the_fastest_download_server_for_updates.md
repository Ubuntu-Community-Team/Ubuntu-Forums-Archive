### Post by Afkpuz on 2009-03-09
Well, assuming that you absolutely cannot just install kubuntu from disc, there is a built in tool for choosing the fastest server.  Goto System>Administration>Software Sources.  Click on the drop down box labeled "download from" and select other.  Then on the right, click "Select best server".  Let it run and it will choose the server with the best ping for you.
