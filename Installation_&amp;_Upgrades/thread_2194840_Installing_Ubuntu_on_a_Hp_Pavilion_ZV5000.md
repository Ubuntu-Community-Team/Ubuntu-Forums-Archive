### Post by Mark Phelps on 2013-12-21
> if i install ubuntu on this computer will i be able to use the built in media card reader

My own experience with card readers is different -- so I would say, probably NOT.  More than once, I've had to restort to using a USB-plugged card reader because the built-in card reader has not been detected by one Linux distro after another.

Also, with that old an ATI card, you will be relegated to using the default Radeon drivers -- as AMD hasn't made proprietary drivers for that card for years.  If the video isn't very good, sorry, that's all you're going to get.
