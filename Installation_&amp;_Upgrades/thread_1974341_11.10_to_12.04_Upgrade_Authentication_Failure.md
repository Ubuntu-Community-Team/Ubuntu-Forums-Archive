I'm having an issue on one of my computers currently running 11.10 that I want to upgrade to 12.04. The computer boots fine I type in the password and all is well. Updates work as well, but when I choose to upgrade to 12.04 and type in my password for authentication I get an error:authentication failed. 

I figured I somehow didn't have admin. rights, but I checked and the system swears I do. I have attempted to change the password by going into recovery mode and dropping to root using passwd no luck it would not overwrite my current password. I tried to change the password using the GUI system tool and my computer just sat there twirling away for 20 min with no results.

Has anyone else ever run into this problem? Any possible fixes?

### Post by zvacet on 2012-05-06
If you want to change password for user then in recovery mode

```
passwd username
```
