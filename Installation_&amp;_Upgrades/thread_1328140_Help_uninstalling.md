### Post by TheHimself on 2009-11-16
First let me tell you that you don't have to do this. You might want to keep it "just in case". For example it's possible that after an upgrade an application that you need stop working. Then you can use Vista until you find a solution for the problem.

But if you REALLY need to uninstall Vista and you have the two OS's on different partitions and you've installed Ubuntu after Vista then it's easy to get rid of Vista. Just format the partition in which Vista is or delete it. You can do this using gparted (You might need to install it first) :

```

sudo gparted

``` 

Be very careful about what you do.
