### Post by dgtldaydrmr on 2007-11-12
I would imagine that there was a recovery partition on your HDD and with the install you may have overwritten it with one of your installations. The "F10" option was a windows boot option (custom created by the HW manufacturer) that would boot off the "recovery" partition you may no longer have.

So..

1. You need to determine if you deleted your windows partition along with your recovery partition.
2. If so you may need to request a recovery disk from the manufacturer in order to get windows operational again.
3. Determine if you really need windows on your computer. (think about it :) do you really need it?)
