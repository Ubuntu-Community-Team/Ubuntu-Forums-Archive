### Post by ajgreeny on 2010-12-06
For some reason you have an unknown bootloader on sdb2, which is the swap partition of the internal hard disk

You can perhaps ignore the unknown bootloader on sda1 as this maybe syslinix in the live USB, but I think it is also surprising that this comment appears in the RESULTS.txt file.

What, if anything, did you ask the installer to do when grub was installed to the system?  It should have simply been left as the default (dev/sda) and not installed to any partition.
