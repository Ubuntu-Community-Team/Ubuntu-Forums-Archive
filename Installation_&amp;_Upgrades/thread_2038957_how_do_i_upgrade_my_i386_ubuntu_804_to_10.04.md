### Post by oldfred on 2012-08-07
I did upgrades from every version 6.06 thru 9.04. But I converted to 64bit with 9.10 and had to do a clean install. Now I only do clean installs.

If you have 32bit version, you cannot upgrade to 64bit. You have to do a clean install, but want to backup /home which has all your data & user settings, any system wide settings you maybe did that would be in /etc, and a list of installed applications to make it easy to reinstall.
