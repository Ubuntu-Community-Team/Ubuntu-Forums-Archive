### Post by darkod on 2012-07-15
If running 10.04 LTS you need to go into Software Sources and in the Updates tab select to prompt you only for Long Term Releases. That means from LTS to LTS.

Towards the end of August when 12.04.1 LTS is released, it will prompt you to update to it.

But before you upgrade, it's best to make a live cd with 12.04 and see if it loads the Try Ubuntu correctly. That can show you if there are any issues with your hardware.

Also, before you start any upgrade process it's best to have a complete backup of all your important data, since sometimes upgrades can go wrong.
