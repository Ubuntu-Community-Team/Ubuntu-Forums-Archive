Okay here is a bit of an odd problem I had. I have a Dual OS setup with my computer. Ubuntu 11.10. And Windows vista.

Now lets say I wanted to add another OS. OR even better, change one of the two OSes I have on my computer to another one. How could I do it? Would I just delete the Partition that one is on? Would I install the new on that? What and how could I go about it?

Thanks in advance.

---

### Post by IWantFroyo on 2012-12-28
You would use a LiveCD like GParted to erase the partitions the operating system you didn't want was occupying, and you would install the operating system you wanted to the free space. There's usually an "install to free space" option, and a partitioning tool is included with many Linux distributions.

### Post by offgridguy on 2012-12-28

start editing partitions.

situations like yours.
