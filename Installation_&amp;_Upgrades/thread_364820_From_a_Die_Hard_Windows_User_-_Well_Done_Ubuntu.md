### Post by acaby on 2007-02-18
I just wanted to leave this post to commend all those involved in this great OS. I have been using windows for so many years I would not care to remember. (The Old DOS) I stumbled on Ubuntu by accident while I was browsing and decided to give it a try. I downloaded the live disc for 6.10, burnt it and ran it and I have been hooked ever since. After 2 day's and only 3 GB on my HD I started deleting,Archiving, and transferring to CD's until I had about 8 GB's and I ran the install to HD from the Live CD.
This has been about a week and I have had very few problems getting everything to work on my 7 year old (Win98SE)(650MHz) Computer.
All of the main programs here I already used on windows i.e. Firefox, NVU, OpenOffice, etc. so this software is not new to me. (By The way I do have a Registered Microsoft Office) but I liked Open office better because of the ability to export to PDF.
The only problem I have had is trying to use WINE. It has been a real nightmare so I finally gave up and un installed it because it could never find the correct system file and every time I tried to install a program with it it would tell me that it could not write to C:/Program Files/*.* because I was out of disc space. (**Could be right**) only got about 3 G left in windows partition. 
The main reason I wanted to leave this post is because My Main use of my computer is for Internet Marketing and most of the good software that is used in this field is ONLY windows based. i.e. Link Cloakers, Website Submission Software, Web CEO, Etc.
With these restrictions I have no choice but to leave the windows partition on my HD. Wish I did not have to because this OS is Awesome and my heartfelt thanks to all those involved in making it happen.

To the moderators I apologize for rambling in this post. Just wanted everyone to know how I felt.
Arthur
