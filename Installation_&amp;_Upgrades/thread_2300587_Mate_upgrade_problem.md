Trying to upgrade Mate 1.8.0 to Trusty using these commands I do not know what to do from here?


 I have never figured out how to mark a post solved. Please help me with that. As to the other

problem in this post I was able to complete the job with synaptic. Thanks All. 

$ sudo apt-add-repository ppa:ubuntu-mate-dev/ppa

$ sudo apt-add-repository ppa:ubuntu-mate-dev/trusty-mate

$ sudo apt-get update

$ sudo apt-get upgrade

$ sudo apt-get install --no-install-recommends ubuntu-mate-core ubuntu-mate-desktop
