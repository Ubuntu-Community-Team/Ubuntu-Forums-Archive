 You DON'T burn .iso discs like that. You open up apps like Nero and select BURN IMAGE, then load up the .iso. You can't just paste the .iso file in the disc and click on "el torito bootable disc" on Nero or Alcohol/whatever. That is just not right.

Nero > Burn Image to disc > Select iso > next > x speed > burn.
Same with Alcohol or Roxio. [Don't use native windows xp burner..\
