---

### Post by BC59 on 2012-01-15
If you are using Firefox, open the home folder, press CTRL+H to see the hidden files and copy the folder .mozilla
In the new installation paste this folder to the same place -before opening the Firefox (because Firefox will create this folder when opening for the first time)
This way you are going to have exactly the same configuration you had before, with the same extensions, bookmarks etc.

### Post by Cavsfan on 2012-01-15
Another way to backup **bookmarks** is to click on bookmarks and then **Show all bookmarks** and there is an option to **import and export** your bookmarks.
The first option backs them up as a **.json** file which is easily imported once exported.

It works well.
