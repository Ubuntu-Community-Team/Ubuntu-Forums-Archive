### Post by utnubuuser on 2010-10-30
swap might be on... select the swap partition and then select swapoff and see if that helps.

You could also try launching gparted with root privileges and see if that gets you any further.
In a terminal type:> gksu gparted
(gsku is the sudo command for graphical applications in gnome)

PS. when installing on a laptop, your swap partition should be at least as large as the amount of ram installed on the machine, ( I usually allocate 1.25x the amount of ram).
