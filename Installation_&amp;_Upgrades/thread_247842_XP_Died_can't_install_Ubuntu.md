### Post by Metacarpal on 2006-08-31
There are also cases where setting the CD-ROM to the first boot device doesn't always quite work...

Computers often have a setting where you can choose your boot device manually.  Try hitting F12 when you first turn on the computer.  If that gives a list of boot devices there, select CD-ROM and see if it starts up.
