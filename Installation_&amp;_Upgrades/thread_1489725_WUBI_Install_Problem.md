### Post by cybrsaylr on 2010-05-21
FWIW I'm not a big fan of WUBI even though it looks very easy to setup.

The main reason is if Windows takes a dump on you, I believe you will also lose the ability to access Ubuntu. Correct me if I am wrong.

This is why I always preferred doing a dual boot setup. Dual booting is only a little bit more involved to setup but after it's done you will have two OSs for use on your PC independent of each other. If one fails you always have the other to use.
