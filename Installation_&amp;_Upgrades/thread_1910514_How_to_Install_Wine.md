### Post by Mark Phelps on 2012-01-17
You're probably installing Wine because you want to run some Windows apps.  In that case, before you try installing them, you can prevent wasting time and effort by using the second link that BlinkinCat posted to search for the apps and versions you want to use.  IF you get no response, or the rating isn't Gold or better, installing that particular app is likely going to be a waste of your time. Wine is not a replacement for MS Windows; instead, it's a kludge that only allows the execution of SOME Windows apps and games.
