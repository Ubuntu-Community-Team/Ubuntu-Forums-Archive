### Post by kyphi on 2007-12-28
Would you check the following setting in OpenOffice.org:

Tools -> Options -> Load/Save -> Microsoft Office - tick all entries in the screen on the right.  These should be ticked by default.

The standard installation of OpenOffice.org (Linux or Windows) should let you open all document formats.

Suggestion:  Record a .doc file to a USB drive while in XP then try to open it in XP to check that it works and then boot to Ubuntu and check it there.
