### Post by am7146 on 2008-05-09
I got it.  Turns out I had to turn on "Legacy USB" support in the BIOS to be able to boot from the USB CD drive.

And Heron is *much* cleaner after the clean install.  PowerTOP shows many fewer wakeups.  Graphics are more responsive.  Stability is better.  WiFi setup easier.  I just had a lot of crap in there.
