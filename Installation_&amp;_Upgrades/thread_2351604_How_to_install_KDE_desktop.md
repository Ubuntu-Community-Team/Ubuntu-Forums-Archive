> **satimis said:**
> I need Ubuntu with KDE desktop to test Kdenlive, not multi-desktop

In that case, I'd suggest Kubuntu 16.04 which is an LTS (Long Term Support) release.

> What are the difference; Kubuntu and Kubuntu Neon

As I understand it, from the Neon website... 

"KDE neon will provide users with more up-to-date packages of Qt in addition to cutting-edge KDE offerings."

so Neon is cutting edge as far as KDE is concerned whereas Kubuntu will give you fixed versions of the KDE stack but possibly more stable and predictable given you want to test a particular piece of software.

> KDE Neon User Edition 64bit and User LTS Edition 64bit

Well, again, you can read as well as I can, and from the website I infer that the KDE Neon User Edition 64bit is bang up to date with KDE software, ie cutting edge whereas the LTS version is the latest **officially released** software, possibly more "stable and predictable" than the non LTS version.
