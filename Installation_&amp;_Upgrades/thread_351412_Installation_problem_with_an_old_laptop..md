### Post by meng on 2007-02-01
Unless your notebook has the maximum 192 MB RAM (and even if it does, you're probably still out of luck) there's no way you're going to get Ubuntu installed on that laptop using the Live CD. I'd suggest the Alternate CD instead. Furthermore, if you install GNOME (Ubuntu) you will find it operates very slowly. Consider Xubuntu (with Xfce) at the very most, and you would really be much better off with Fluxbuntu (google for it). Or else consider another lightweight Linux distribution, there are several, Damn Small Linux and Puppy Linux are very highly recommended.
