### Post by wislon on 2007-02-09
I have been wondering about this myself (since there's a few implementations that mono won't do, e.g. battling with ActiveSync to write Windows Mobile applications). I haven't tried it, but I've heard of people who have run it inside a VMWare virtual machine with some reports of success. I am still trying to figure out how to get one of these VM systems to support multiple monitors.

I also came across this link:

[https://help.ubuntu.com/community/WindowsXPUnderQemuHowTo](https://help.ubuntu.com/community/WindowsXPUnderQemuHowTo)

Which will apparently let you run XP in a VM, and then let you do the equivalent of remote desktop into it (on the same box) in order to run your applications full-screen in Gnome or whatever you're using. Again, I haven't tried it, but it sounds like a workable solution too.

Good luck, and please post back if you decide to try this? I'm curious to find out how well it would work! :)
