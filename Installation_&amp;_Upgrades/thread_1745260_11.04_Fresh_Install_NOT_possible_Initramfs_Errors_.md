### Post by eks on 2011-05-01
You are using a SanDisk USB drive, right?

If so, you have three options:

[LIST=1]
[*] remove U3 launchpad from the USB drive on a windows machine with [SanDisk's tool]("http://u3.sandisk.com/launchpadremoval.htm"):
[*] use another USB drive from another brand or a SanDisk without U3 launchpad
[*] install 10.10 and upgrade right after boot.
[/LIST]

It sucks, but it seems at this time these are the only options. More about it here: [http://ubuntuforums.org/showthread.php?t=1743527](http://ubuntuforums.org/showthread.php?t=1743527)
