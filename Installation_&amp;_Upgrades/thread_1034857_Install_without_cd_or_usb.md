### Post by Maarek Stele on 2009-01-09
Try the mini install.  You load the alternate install onto the USB, about 58megs in size.  The rest is downloaded during the install.  You also get to choose what will be on the laptop.  Ubuntu, Xubuntu, Kubuntu, and or any server software.

[https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)

this is a download install, so it will take longer.
