### Post by drs305 on 2011-11-20
Remove these two lines, at the bottom of what you posted:
```

deb http natty main
deb-src http natty main
```

They are causing the error and the main and src main repositories are already incorporated earlier in the file.

The error message tells you it's line 59. Other than studying the approximate location, one way to quickly find the error is to open Gedit or text editor, enable line number display, and then look at Line 59. Another way, if there is only one error, is to open the file with Gedit already on line 59:

gksu gedit +59 /etc/apt/sources.list
