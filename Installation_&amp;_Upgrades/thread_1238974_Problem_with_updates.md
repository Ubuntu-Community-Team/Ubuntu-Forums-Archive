### Post by starcraft.man on 2009-08-13
Easy solution, pick another server. There are hundreds eh? Surely there is one close to India in the list. I can personally vouch for the Canadian server from savoirfairelinux.net, it's speedy fast for me and I always saturate my 5-6 megabit line. 

System > Admin > Software SOurces >Download From... Other .

You can select best server which will find best or simply pick one from the list as I usually do that is geographically close.
