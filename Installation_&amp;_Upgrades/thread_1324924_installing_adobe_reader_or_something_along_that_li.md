### Post by sgosnell on 2009-11-13
Evince should be installed.  It comes with the distro, and that's the easiest way to view .pdf files.  You should be able to double-click on .pdf files and have them open in Evince.  You can download the Adobe reader for Linux if you prefer it.  It's in the repositories, as 'acroread'.  You can get it through Synaptic package manager or via the command line with 'sudo apt-get install acroread'.

You can get to the torrent client via Applications/Internet/Transmission BitTorrent Client.
