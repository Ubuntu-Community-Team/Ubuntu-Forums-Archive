### Post by linuxsyst on 2011-02-13
Press ```
Alt+F2
``` and type ```
update-manager -d
``` this should show the option to upgrade to the latest alpha release of natty.

Through the terminal this can be done using the same command but you need use ```
sudo
``` or ```
sudo -s
``` to do this.
