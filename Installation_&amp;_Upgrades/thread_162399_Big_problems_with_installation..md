### Post by cgjones on 2006-04-19
The 7.8 MB of free space is a Windows thing. I believe that it has something to do with the master boot record or partition tables. I would leave it alone.

I would try defragmenting the hard drive through Windows, then try the Ubuntu install again. You want to make sure that all of Windows is as close to the beginning of the hard drive as possible before you try resizing.
