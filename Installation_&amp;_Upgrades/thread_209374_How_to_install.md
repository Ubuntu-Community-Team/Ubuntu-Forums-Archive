### Post by woedend on 2006-07-05
put the cd in the tray, then reboot the computer.
it should automatically load a usable ubuntu from the cd.  If it does not you must edit your bios settings to boot from the CD before the hard drive.  
Anyhow, once you get the cd up and running, you can use ubuntu as you wish, or install it.  There is a big icon on the desktop that says "Install".  Double click it and it will hold your hand the rest of the way.  HTH
