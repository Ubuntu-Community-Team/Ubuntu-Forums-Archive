### Post by Impavidus on 2018-03-23
18.04 will be released in April. The upgrade from 16.04 to 18.04 will be offered when 18.04.1 gets released, which I think will be in July. But if you really wish, you can force the upgrade before that.

The upgrade will mostly keep the packages you already have, just replace them with new versions. Different flavours of Ubuntu only differ in the set of packages that gets installed by default, so the release upgrade won't move you from one flavour to another.

I have to add that I'm not so familiar with Ubuntu Budgie. Once the number of flavours became more than 4, I lost track.
