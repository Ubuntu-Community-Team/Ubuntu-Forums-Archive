### Post by Marcus Derekus on 2008-11-16
Why must you install from network?

can't you just burn the iso to a cd and boot the pc off of cd and install ubuntu?

Or burn a cd boot into windows and use the cd to do wubi install.

Or if you for some reason cant burn a cd then download the iso onto the pc mount it and do a wubi install 

Finally if you cant burn a cd or boot off of one but you dont want a wubi install  

download unetbootin onto the pc as well as the ubuntu iso.
And use unetbootin to boot directly off of the iso
