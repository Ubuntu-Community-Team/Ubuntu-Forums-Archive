### Post by aysiu on 2007-01-23
Well, you would [install Xubuntu first](http://www.psychocats.net/ubuntu/xfce), then [remove Ubuntu](http://www.psychocats.net/ubuntu/purexfce), and finally [upgrade to Edgy](https://help.ubuntu.com/community/EdgyUpgrades).

For all the trouble, though, you might be better off just doing a clean install of Xubuntu Edgy after [making a separate /home partition](http://www.psychocats.net/ubuntu/separatehome).
