I wasn't able to reproduce the problem on my machine with the same FF version and add-ons. However, I would recommend using different add-ons. FlashGot is a well established add-on, developed by an experienced developer, but the other two have issues.

I am not sure if you are referring to [this YouTube Downloader]("https://addons.mozilla.org/en-US/firefox/addon/youtube-downloader/"), but that add-on is throwing errors in the console.

The Ant Video Downloader seems to work properly and has a huge user base. However, it has been developed since 2008 and it is still listed as *Experimental*. Due to the long history of development, I can only assume it hasn't been fully approved by Mozilla for some reason. Since it has been partially reviewed, you can assume it is safe to use, but it could be the cause of your performance problems.
