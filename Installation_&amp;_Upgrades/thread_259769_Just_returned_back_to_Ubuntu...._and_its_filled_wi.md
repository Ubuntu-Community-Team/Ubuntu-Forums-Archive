### Post by jtbl on 2006-09-17
For Flash to work correctly on 64 bit ubuntu you must use a 32 bit browser.  The firefox that comes with ubuntu by default is 64 bit and can only use an open source alternative, gnash, which doesnt have the level of compatibility that adobe flash has.  If you install swiftfox and swiftfox plugins with automatix for 64 bit ubuntu then you will get a 32 bit browser that uses adobe's flash.
