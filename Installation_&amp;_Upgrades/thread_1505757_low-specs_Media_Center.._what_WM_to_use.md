### Post by _Mark_ on 2010-06-10
Think you maybe pushing you're luck with those specs especially if you want to play HD.

If you had nvidia graphics you would be able to use VDPAU ( more info here [http://en.wikipedia.org/wiki/VDPAU](http://en.wikipedia.org/wiki/VDPAU)) basically it offloads a lot of the processing to the GPU similar to dxva in Windows. According to the wikipedia entry their is no ATI support for VDPAU

This link tells you what cards support it [http://www.mythtv.org/wiki/VDPAU](http://www.mythtv.org/wiki/VDPAU)

Even then I don't think you're CPU will cope

### Post by martinr on 2012-05-19
With your specs running a video should be a breeze. Try tuning your hardware first (use SiSoft Sandra 2002). FSB to PCI bus, caching, memory timing, HD UDMA mode optimizetion, etc. If all else fails, try installing hardware mpeg accelleration (a TV combo card for example).
