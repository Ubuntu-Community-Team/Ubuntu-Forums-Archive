### Post by Hedgehog1 on 2011-05-01
Right now no one I know has found a way to change the position of the launcher.

You can make the icons smaller so it is less intrusive:

1) Use Ubuntu Software Center and search for CCSM.  Install the Compiz Config Settings Manager application.

2) go to the control panel (now the last item in the 'power button' menu), and run Compiz Config Settings Manager

3) Select the unity plug in:

[IMG]http://img51.imageshack.us/img51/558/ntchangeiconssize.png[/IMG]

4) Adjust the launch panel icon size (and I like to make the top panel semi-transparent):

[IMG]http://img806.imageshack.us/img806/3707/unityccsm.png[/IMG]
