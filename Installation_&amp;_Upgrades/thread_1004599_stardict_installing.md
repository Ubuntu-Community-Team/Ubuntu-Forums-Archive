### Post by Partyboi2 on 2008-12-07
Stardict is in the ubuntu repo's you can install it by opening a terminal (Applications>Accessories>Terminal) and typing
```
sudo apt-get install stardict
``` If you are using intrepid (8.10) this will install version 3.0.1
