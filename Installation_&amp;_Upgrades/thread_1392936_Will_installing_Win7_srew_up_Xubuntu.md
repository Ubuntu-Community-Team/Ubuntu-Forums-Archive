### Post by audiomick on 2010-01-28
No, it will overwrite your grub. You will have to re-install grub.
If you have grub 2, that is, a fresh install of 9.10
[https://help.ubuntu.com/community/Grub2](https://help.ubuntu.com/community/Grub2)

If it is grub legacy, that is, an older Ubuntu or 9.10 updated from an older version
[https://help.ubuntu.com/community/GrubHowto](https://help.ubuntu.com/community/GrubHowto)
