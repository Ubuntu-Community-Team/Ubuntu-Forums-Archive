### Post by drs305 on 2012-06-10
Do you know the key to press to access your BIOS during power up? If not, you can probably find it online somewhere. You should be able to access it by repeatedly pressing the key as the computer powers up - often the computer will beep at the time the key should be pressed.

Otherwise, try holding down the SHIFT key as the computer boots to see if you can get the GRUB menu to display. If that doesn't work, try pressing the ESC key repeatedly.

If you can get the menu to display we may be able to change some of the kernel options to allow it to boot to the Desktop.
