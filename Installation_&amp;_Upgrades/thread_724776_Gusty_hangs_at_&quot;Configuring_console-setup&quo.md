### Post by jeeves on 2008-03-15
It may have just been luck, but I was able to get around this problem by allowing the Xubuntu installer to configure the network settings using DHCP. For some reason this seems more reliable when installing than the manual configuration with a static IP.

Maybe there is a thread somewhere on this, but I get the impression that you need to be online to get Xubuntu installed.

So I can't be sure what fixed the problem, but at least it's gone.
