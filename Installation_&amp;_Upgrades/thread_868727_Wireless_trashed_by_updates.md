### Post by Brws on 2008-07-24
Hi

Just got round to downloading the latest updates to Ubuntu 7.10, after several weeks inaction. On restarting, I found that my wireless connection had been trashed. The firmware (wl-apsta.o) for the bcm43xx driver was apparently now missing. I downloaded it on my Windows box, transferred & installed it to the Ubuntu setup, and all is now well. Hope this helps any other users who've hit the problem
