### Post by edwina on 2005-08-09
[QUOTE=tripletrouble42]Yeah, I have to reformat to partition, no? Cause I really dont want to have to go through that[/QUOTE]
 That depends. You can shrink certain partitions in order to make room for a new partition. This seems to be perfectly achievable with a Mac by using "parted" or something similar. The danger is that you may lose some files if it goes wrong, so make sure you back up and make sure that your disk is thoroughly defragmented first. There are plenty of Mac Linux users out there though, so have a sniff about on Google using "shrink partition linux" or something.
