### Post by xgtu on 2009-02-18
Solved it myself.  Had to start from scratch though.  I chose the wrong option on the partition screen: Guided LVM with encryption for entire drive. Though my data still appeared to be there, I could not figure out how to get to it.

Read the manual a little more carefully and viola, I partitioned multiple logical volumes (sized based on recommendations) in the ext3 file system format.
