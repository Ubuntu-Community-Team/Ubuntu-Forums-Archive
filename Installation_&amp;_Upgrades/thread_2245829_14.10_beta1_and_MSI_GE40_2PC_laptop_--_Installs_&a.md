### Post by brulaz on 2014-09-26
The new MSI laptop has the latest Intel i7 and chipset and Nvidia GTX 850M.
Wanted to leave Win8.1 and UEFI in place, so first turned off Secure Boot in Uefi-bios and Fast Start in Win.

As I've only used Debian over the last 10 years, I tried that first.
Debian Jessie Beta 1 install went ok, but there was an issue with firmware. I could repartition the HD, but after the install starting from GRUB2 would hang.
Tried daily Jessie, same prob.
Tried latest systemrescue 4.3.1 (Gentoo based) and it too woud hang after selecting a kernel to load at GRUB2

So tried Ubuntu 14.10 Beta 1 and there have been no problems at all, other than learning the quirks of a slightly diff OS.
Maybe it's the 3.16 kernel (Jessie is only 3.14), dunno. 

Thanks Ubuntu!
