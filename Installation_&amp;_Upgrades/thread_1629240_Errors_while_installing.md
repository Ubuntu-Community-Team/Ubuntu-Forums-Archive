See these tutorials for installing Ubuntu fom a usb drive:
[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)
[https://help.ubuntu.com/community/USB%20Installation%20Media](https://help.ubuntu.com/community/USB%20Installation%20Media)

I don't know if these will help you, but since nobody else has responded so far I thought I would post them.
Also, be sure to verify the *md5sum* of the Ubutnu iso image file that you downloaded to be sure it is not corrupted:
[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

If your computer can boot from a CD, I would consider burning the Ubuntu iso image to a CD and installing Ubuntu that way.
