### Post by xc1024 on 2008-11-22
sure you can do it...
there are 3 things you need:
1. ubu 8.10 livecd
2. usb stick
3. computer with bios supporting usb drive boot
just boot the livecd on computer with cd drive, go on settings, look for liveusb install, click, follow instructions.
after that you should have your usb drive bootable. now turn on the target computer, choose to boot from usb drive/removable media/whatever. install procedure just like in normal live cd.
good luck.
