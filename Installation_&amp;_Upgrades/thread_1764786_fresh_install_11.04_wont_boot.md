```

```
--------------------------------------------------------------------------------

#

# swap was on /dev/sda5 during installation
UUID=af9c2cbe-91f9-419c-9aee-760c6b50c0fd none            swap    sw              0       0

```

If you are still unable to boot, please boot off the LiveCD/Live USB, select TRY, and then:

In the Terminal (Menu to: Applications >> Accessories >> Terminal), please run these two commands:

```
sudo mount /dev/sda1 /mnt
```

```
gksudo gedit /mnt/etc/fstab
```

[COLOR="red"]And change this line:[/COLOR]

```
/dev/sda1       /               ext4    errors=remount-ro 0       1

```
[COLOR="red"]To this:[/COLOR]

```
UUID=d998362a-b293-4b65-aa1e-0b8ceabbe555  /   ext4    errors=remount-ro 0       1
```

Then save the file and exit the editor (ignore any error messages from the save)

Reboot.
