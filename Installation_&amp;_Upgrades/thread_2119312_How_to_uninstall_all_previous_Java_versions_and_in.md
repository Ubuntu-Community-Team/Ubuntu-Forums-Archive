### Post by QIII on 2013-02-23
See the Java Community Wiki link in my signature.

I recommend the method you will find under "Oracle Java 7", "Command Line Methods", "Using webupd8.org's strikingly simple method"

You don't need to worry about uninstalling previous versions if you don't want to.  That method will set Oracle Java 7 (latest update) as your alternative and will keep your Oracle Java updated as Oracle releases new updates.

When you are not using the Java browser plugin for class, I would disable it.  There are some significant security vulnerabilities.

Best of luck with your course!
