### Post by phillw on 2009-08-09
go into synaptic package manager, type in firefox.

The top one should read firefox, in the next column something like 3.5.3~hdg ......

As long as it has a green box, then the major updates fro firefox are being done automatically.

You can over-ride this and get beta (test) versions, but the ones it downloads are the canonical stable releases.

If you type in firefox - the 1st item should be just called 'firefox', with a description of 'meta package for the popular mozila browser'
