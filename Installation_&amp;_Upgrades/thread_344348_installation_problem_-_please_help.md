### Post by phlo15 on 2008-01-30
I know this is an old thread, but I thought I would post this in case it helps other users. 

I had this problem when I tried to change the root password while my system time was not up to date. (my sytem clock displayed the epoch time 'Wed Dec 31 18:00:00 CST 1969').

So, I just changed the password AFTER updating my system time and this solved the problem.
