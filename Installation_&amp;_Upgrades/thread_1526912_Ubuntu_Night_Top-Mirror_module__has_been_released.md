### Post by feige on 2010-07-08
Ubuntu night([http://ubuntu9.com](http://ubuntu9.com)) is a website of  service platform  co-established by Ubuntu fans to provide more agile and easy-use Ubuntu Linux.All are on the way of development.

The newest released module is TOP-MIRROR,which providing all available mirror sort by three ways: top-fast is by average download speed,top-near is by your geographical position( normally close to your country),top-stable is by download speed variance statics.

As network and mirror server status may change from time to time,we search and test the mirror by periodically,and show the latest information for you.

You can choose a top-fast mirror,or by you own preference, download its source list to directory "/etc/apt/sources.list.d/" or simply overwrite the file "/etc/apt/sources.list".We also recommend you to backup you original source list file before overwriting.
