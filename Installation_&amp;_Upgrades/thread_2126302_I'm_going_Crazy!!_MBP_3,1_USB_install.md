### Post by grahammechanical on 2013-03-16
You do not say what version of Ubuntu you are trying to install or whether the machine has Windows 8 on it. A machine that has Windows 8 pre-installed may also have secure boot enabled. The only Ubuntu versions at the moment that stand any chance of installing on a machine with secure boot enabled is 12.04.2 and 12.10. Both available from the Ubuntu download page. If you are trying to install a Ubuntu 12.04 or a 12.04.1 image then it will not install with secure boot enabled.
