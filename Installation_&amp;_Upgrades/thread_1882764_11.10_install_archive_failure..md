### Post by David006 on 2011-11-17
I found your issue elsewhere ..

*Fails to boot after upgrade, stuck on winbind package*
[http://askubuntu.com/questions/66606/fails-to-boot-after-upgrade-stuck-on-winbind-package](http://askubuntu.com/questions/66606/fails-to-boot-after-upgrade-stuck-on-winbind-package)


You need to do two things:
```
  [b]sudo dpkg --purge winbind
	
  sudo dpkg --configure -a[/b]
```

After you resolve this, and assuming you are using Samba (Windows support), you need to re-install 'winbind'.

You need one of these (if the first fails, then use the second).
```
  **sudo apt-get -f install**
```

```
  **sudo apt-get install winbind**
```
