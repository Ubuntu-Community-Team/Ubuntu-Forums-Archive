### Post by Spaceman9 on 2008-10-26
Boot Synaptic and click Settings then Repositories then Third-Party Software.

Put a check mark next to Cdrom with Ubuntu 8.04 Hardy Heron. If you're using a different version of Ubuntu then the nake of your version will be there instead of 8.04 Hardy Heron.
