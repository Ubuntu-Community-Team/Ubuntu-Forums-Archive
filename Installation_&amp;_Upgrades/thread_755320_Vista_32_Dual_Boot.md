### Post by Pumalite on 2008-04-14
Partition with Vista partitioner. Burn new CD. Follow this guide:
[https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28burn%29%7C%28](https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28burn%29%7C%28)
Do md5sum, burn at 4x or less, as image, not as data.

### Post by StefAndrew on 2008-04-14
Yeah, I've read that it's best to "shrink" the vista partition from within Vista.  It's much more reliable when shrinking the Vista partition so there is free space to be able to install Ubuntu.  What version of Ubuntu are you trying to install?  I've had problems with CD burns not working very well.  I was told to try burning at a slower rate and that seems to do the trick.  Never burn at max speed and it seems to be more reliable.

If you need help with how to shrink the partitions in Vista, just let me know.
