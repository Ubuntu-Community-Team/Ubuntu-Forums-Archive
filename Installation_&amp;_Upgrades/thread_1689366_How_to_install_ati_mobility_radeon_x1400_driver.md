### Post by Mark Phelps on 2011-02-17
If you're running an Ubuntu version newer than 8.10, there are NO drivers you can install that will work with your card and these newer Ubuntu versions.  AMD dropped restricted driver support for your card years ago.  The only working drivers now are the open-source drivers -- and they are installed automatically when you do initial setup of Ubuntu.

IF you try to force the installation of any other drives, you will only trash your display and have to go to a LOT of work to clean up the damage and get back to a working desktop.
