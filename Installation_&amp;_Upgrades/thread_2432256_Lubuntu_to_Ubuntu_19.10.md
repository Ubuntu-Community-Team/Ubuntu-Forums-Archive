### Post by mörgæs on 2019-11-30
If you are running Lubuntu 19.10 then the command ```
sudo apt install ubuntu-desktop
``` will add all missing software in order to get to Ubuntu 19.10.

Be aware that you will get a system dependent on Snap which tends to slow things down.

### Post by guiverc on 2019-11-30
You can add another desktop as @[mörgæs]("https://ubuntuforums.org/member.php?u=939075") suggested, however I get the feeling your Lubuntu is old (14.04 LTS maybe).

If that's the case, I'd suggest a re-install. You could re-install using 'something else' & not format your system (it'll note your software, erase system directories, install, then try and add back your additional packages (if available) without touching user files), BUT there is a huge jump from 14.04 (if that is what you were on) to 19.10, especially given you are changing desktops - the result may not be ideal (or as good as a clean install).

If you just want to add the GNOME (or Ubuntu) desktop, I'd for sure add it.  My own system was the other way around (it started as Ubuntu, and I added `lubuntu-desktop`, `xubuntu-desktop`.. to mine..  I select which I use for the session when I login.  Negatives with multiple desktop are a cluttered menu (more program options, eg. I have all Lubuntu's + Ubuntu's, + Xubuntu's + Ubuntu-MATE's.. in mine), using the wrong program on the wrong desktop works fine but can waste resources (esp. memory; not an issue if you have 4gb or more, but worth considering if you have less than 4gb in my opinion), plus larger upgrades (my upgrades including all desktops I have installed on my 'bloated' system)
