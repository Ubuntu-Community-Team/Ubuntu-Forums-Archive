### Post by Partyboi2 on 2009-01-14
You should be able to see Gimp Image editor under Applications>Graphics>GIMP Image editor, it is installed by default.
If it is not in the menu then try going to System>Pref>Main Menu and highlight "graphics" on the left and then on the right window tick "GIMP Image Editor" then check the menu (Applications>Graphics>GIMP Image editor) again.
