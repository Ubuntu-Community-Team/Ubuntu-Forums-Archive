### Post by Angela_C.Murphy on 2016-01-08
I have a Dell Inspiron 14R laptop. about 4.5 years old with Core I3 processor, which came with Windows 7 installed.  When it first came, we were in a hurry to leave on a trip, so I installed Ubuntu from the only medium I had - a 12.04 LTS Live CD, 32-bit.  The dual boot with Windows has worked well.

It is now time to upgrade Ubuntu.  I have learned some more about 64-bit vs 32-bit installations, and have decided that I would like to go to a 64-bit installation of 14.04.  Am I correct in assuming that I can't do this using the 12.04 Upgrade Manager?  In fact, don't I have to do a completely new installation of 14.04 from the 64-bit DVD?

### Post by Mark Phelps on 2016-01-08
You can not UPGRADE from 32-bit to 64-bit; thus, you are correct in presuming that you must do a clean install.
