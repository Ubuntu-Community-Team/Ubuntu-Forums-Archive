### Post by taurus on 2006-11-05
If you are happy with everything in Dapper, I don't see any point for upgrading to Edgy.  And if you really want to run Edgy, better do a fresh install instead of upgrading because some people run into some serious problems when they upgrade from Dapper to Edgy.  And remember, Dapper will be supported for 3 years so you would get your update and everything with Dapper.

### Post by aysiu on 2006-11-05
I see no compelling reason for you to switch.

But if you *do* upgrade, [make a full backup of your Dapper partition using PartImage](http://www.psychocats.net/ubuntu/partimage). That way, in case things do get screwed up, you have an easy way to restore things.
