### Post by ajgreeny on 2014-01-02
Provided you have the original vista DVD, or know someone with one for vista, and you have the original OEM Product Key (it may still be on a sticker beneath the machine) you should be able to reinstall vista without too much difficulty.

I think vista was the last MS OS that allowed that manner of installation; later versions require the correct original DVD, I think, and will not allow MS activation with the wrong one.
