### Post by aysiu on 2005-11-03
The live CD is like a test-drive.
You can try Ubuntu out **without harming your computer**.

The installer CD installs Ubuntu on your computer, erasing either all or part of your existing hard drive.

It's like the difference between a two-week free trial of Netflix and the month-by-month you-pay-for-it version. The live CD is no commitment. The installer is a big commitment.

Oh, except for Ubuntu, even if you commit, it's still free.

### Post by SickTwist on 2005-11-05
Both CDs feature the exact same programs. If you like how the live CD runs, the install CD will give you a similiar experience (but it will run much faster and be able to save your settings when the computer is off).

The live CD is also a great tool to use Ubuntu wherever you go (Internet cafes, friend's house, school, etc.) without altering computers that don't belong to you. Or you could use it to try to fix a computer that will not boot.

Live CDs are great! :D

Some live CDs offer the option to install to a hard drive but Ubuntu does not have that ability at this point.
