### Post by veruca on 2006-12-25
I was browsing the net looking for a quick way to install linux via a network install. I just inherited and IBM thinkpad 240x (no cd or floppy, 500mhz PIII, 130mb pc100 ram, and one pc network card) and with those stats and no external CD rom I had few options.

So I was looking and came across instlux which while "upgrade" your system from win2k/NT/XP with a 7 meg windows exe. You reboot the system and it gives you boot options and you boot into the network install which will auto detect everything. Its so easy and I recommend it to anyone that needs something simple if they have limited resources on installing linux. There are net install and cd install options for both SuSe and Ubuntu.

I'm using the notebook as a surveying tool for wireless networks when I install wireless networks and large square feet homes. But anyways...

The reason for posting this is I was surprised to not see any topics on this and thought that it would be a valuable tool for people like me.

[http://instlux.sf.net](http://instlux.sf.net)
