### Post by snowpine on 2008-08-29
1. Yes, you absolutely do not need the RT kernel for many purposes. It is however recommended if you are doing multitrack recording. I did find the RT kernel unstable on my computer, so I'm not using it any more (my needs are modest). Remember that you can choose which kernel to use from Grub, so you can switch between the RT and normal kernels at your leisure.

2. Ubuntu Studio IS Ubuntu, so anything you can do in Ubuntu, you can do in Ubuntu Studio, too!
