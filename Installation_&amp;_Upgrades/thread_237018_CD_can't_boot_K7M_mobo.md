### Post by ajgreeny on 2006-08-15
Have a look at smart boot manager which will boot from a floppy and give you the option to then boot from CD drive.  You will need to copy the disk image to floppy (in windows using rawrite?) and then just boot from floppy.  Good luck; if winXP can do it I'm sure ubuntu can too.
