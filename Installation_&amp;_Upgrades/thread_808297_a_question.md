### Post by Dan++ on 2008-05-26
There's a program called Cedega ([http://www.transgaming.com/](http://www.transgaming.com/)) which requires payment, which will allow you to play a lot of Games in Linux.

Alternatively there's WINE, a Windows Emulator, which theoretically can run any exe, I think, but in practise it doesn't always. More info is at [http://www.winehq.org/](http://www.winehq.org/)
