I've been reading loads about partitioning during my Ubuntu installation effort, and if I understand your setup fully then I’d partition the larger drive which would shrink the XP system part of the drive, leaving you room to install Ubuntu on the rest of the drive. Check out GParted when you start looking at making the partition, it might make things easier.
