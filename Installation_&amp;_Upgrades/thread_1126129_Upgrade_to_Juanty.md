### Post by marks_linux on 2009-04-15
Hit ALT-F2 then type in 'update-manager -d' without quotes.

Should get a message about a newer version being available. Select update and wait for he download, was about 300MB if I remember on my other 8.10 machine

Remember 9.04 is still in Beta (but you knew that already!)
