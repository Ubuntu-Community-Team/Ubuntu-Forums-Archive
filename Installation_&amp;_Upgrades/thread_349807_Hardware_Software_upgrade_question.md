### Post by Jussi Kukkonen on 2007-01-31
Almost certainly it will just work. I don't see big reasons to backup: nothing should be happening to your data and even if the new motherboard won't boot, you can always just move the disks to another machine. Admittedly, copying your home dir to a fat partition or a CD might make it more accessible from a windows machine if something like that would happen.

A bootable CD might be handy in case the OS on the hard disk doesn't boot for some reason, but personally I probably wouldn't bother...
