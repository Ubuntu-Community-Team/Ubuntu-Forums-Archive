### Post by 23meg on 2005-10-12
first of all, you check if it's in the repositories. you do this by launching Synaptic (which is in System / Administration), hitting "search", and typing the name of the program you're looking for. if it's there, right click its package name, and choose "mark for installation". before you do this, make sure you've enabled all repositories. you can find instructions on how to do this in the unofficial ubuntu guide. 

see if this much helps you..
