### Post by lechien73 on 2011-05-05
Hello & welcome to the forums,

If you install the ubuntu-restricted-extras package, then MP3 playback should work for you.

To do this, go to Applications -> Ubuntu Software Centre, type the word "restricted", and this should return the Ubuntu Restricted Extras package.

After this, it should work for you.
