### Post by avtolle on 2008-06-20
I wonder if the OP is running into the bug which has been the subject of a few threads on the Forum concerning the host name file. To check, go to System>>Administration>>Network; click on unlock, enter password; select hosts tab. There should be several lines in it, the one to look at is the second one, which in mine begins 127.0.1.1. After that should appear only PCNAME (the name of your computer on any network, etc.). If there is anything else after it, delete it so that it reads 127.0.1.1 PCNAME. Close, and try your update again. The first line should read something like this, BTW: 127.0.0.1 localhost.
