Not a biggie, but just a note:  When installing 11.04 I selected the upgrade option (from 10.04), and computer promised to keep all my files, bookmarks, etc. It didn't happen -- as far as I can tell it seemed to have completely wiped out all the pre-existing stuff and installed a fresh, clean white canvas.

Luckily I backed up everything before just in case. But still this is something folks should be aware of. If upgrading back up all files prior to making the change, or else risk losing it all as I almost did.
