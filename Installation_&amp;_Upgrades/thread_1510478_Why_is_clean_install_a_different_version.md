### Post by anantshri on 2010-06-15
features and other stuff are all provided by both clean install or upgrade. 


difference is 
upgrade let you keep your customized theme and don't apply its own theme. 
upgrade keep your current configuration and hence a lot of the changes specially visial changes are not so visible when we upgrade.
