### Post by sub2007 on 2008-01-05
As far as I know there are three main install methods:
1. Live CD - you download a Live CD, boot into the Ubuntu OS and then click Install on the Desktop and then install it into a partition you make
2. Alternate - a text based installer which you get by booting into the alternate CD and again install it into a partition
3. Wubi - where you run the installer from within Windows and it sets up a dual boot without partitioning

Just wondering which of the three you used? Or maybe even a different method?
