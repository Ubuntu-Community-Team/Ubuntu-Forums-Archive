### Post by Lord Landis on 2007-10-20
It's hard to say for sure.  I'm not surprised that Dell would do something like that - they have a tendency to install a lot of things on their systems.  Though in this case it's probably ndis wrapper for the WiFi or something like that...

Anyway, if you've got a restore CD (or partition), it can't hurt.  The worst that could happen is that 7.10 doesn't work and you need to redo the system with the restore discs (which would contain any additional programs/scripts Dell shipped).

Alternatively, ask Dell's tech support what they think.  After all, you paid for the system, they should be able to support it.
