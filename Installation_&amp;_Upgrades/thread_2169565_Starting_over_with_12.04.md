### Post by Mark Phelps on 2013-08-23
Regarding starting over from 12.04, when you upgraded to 12.10, that brought with it a new version of the X-server -- a version that some video drivers can't handle.  IF that was part of the problem, you need to know that 12.04.2 upgraded to the same X-server version; so when you go back, make sure it is to 12.04.1.

You can get that from the following link: [http://old-releases.ubuntu.com/releases/]("http://old-releases.ubuntu.com/releases/")
