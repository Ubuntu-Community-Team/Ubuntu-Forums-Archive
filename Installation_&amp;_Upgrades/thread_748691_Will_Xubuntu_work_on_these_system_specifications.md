### Post by Aearenda on 2008-04-07
'Speed' is so subjective - it depends on what you are used to. I used to be quite happy with the speed of my 1 GHz Pentium-M laptop until I started to use a Core 2 Duo desktop - then I began to think the laptop was slow. 

I think it would be 'ok for me' on that machine so long as you don't try to run too much at once. I can't say whether _you_ will think it is 'fast' or not. Most likely not. 

Debian would probably start up and run slightly faster, but you'd have more post-installation setup to do - there is an XFCE install CD for Debian, rather like Xubuntu. 

Running apps such as OpenOffice and Firefox will be much the same regardless of Ubuntu/Xubuntu/Debian, once they have started up - and starting up will depend on RAM availability and disk throughput. More RAM would be a good idea - at least 256Mb. Otherwise swapping to disk will slow everything down. 

Try it and see - all it costs is your time!
