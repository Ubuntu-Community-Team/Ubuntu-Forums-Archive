### Post by dbott67 on 2005-10-15
If you're talking about installing or updating software, you just have to choose one method.  Synaptic is the easiest for looking for & installing software, as well as updating.

Apt-get does exactly the same thing, but makes it much easier to provide directions for people.  For example, if you were asking how to get the 'Open Terminal' back in the context menu, I would tell you to just type in the following command at the terminal: 
```
sudo apt-get install nautilus-open-terminal
```
The nice part is that you can just cut & paste it into your terminal.

Hope that helps.
-Dave

### Post by 5-HT on 2005-10-15
As dbott67 mentioned, you can choose your favorite.

Easiest three ways to do an upgrade and not a clean install:

- Just put in the cd (when running) and it should auto run    asking you if you want to upgrade
- Using synaptic/apt/aptitude and downloading the packages
- Using synaptic/apt/aptitude and using the breezy cd
