### Post by Rhizoid on 2011-02-05
Hi Jillie,

You can install Ubuntu from a pendrive if you have one?  As you still have Windows installed you might find the tools at [http://www.pendrivelinux.com](http://www.pendrivelinux.com) easiest to use to create a LiveUSB instead of a LiveCD/DVD.

[http://www.pendrivelinux.com/universal-usb-installer-easy-as-1-2-3/](http://www.pendrivelinux.com/universal-usb-installer-easy-as-1-2-3/)

You'll need a USB pendrive with adequate storage for the .iso image you download, the .iso image itself and the Universal USB installer I've linked above.

Once you've created the USB it's just a matter of booting from it and choosing to install, rather than to try out Ubuntu.

CAUTION: Make sure you've backed up everything you want to keep before you do the install, once it's gone it's very hard to recover from!

Hope that helps,
