### Post by dstew on 2008-04-23
The upgrade will have a new kernel, so the old drivers and codecs will not work. You will also need to install new versions of your programs. Most Ubuntu programs have versions that will work with the new OS version, but it is not guaranteed, especially for Universe programs. I am not sure, but I think using the upgrade button on the Update Manager will try to get new versions of installed software for you, and preserve your /home folder, which contains your data and configuration files, but it is probably better just to back up your home folder, and do a fresh install, then use Synaptic to install new versions of your programs.
