### Post by yancek on 2014-10-10
> My hard drive is presently partitioned into one large partition. Is that the problem?

Yes.  No guarantee that will be the only problem.  If you have some method to resize/shrink the partition within xp, do that.  Otherwise you can use GParted which is on the installation medium.  Open a terminal and type:  sudo gparted to open it and then shrink the partition to create unallocated space.  Take a look at the tutorial on installing Ubuntu 14.04 below.  It is very detailed and has a lot of additional information as well as a link to another very detailed tutorial on using GParted.

 [http://www.dedoimedo.com/computers/ubuntu-14-04-install-guide.html](http://www.dedoimedo.com/computers/ubuntu-14-04-install-guide.html)
