In March/April, i wanna to try a Linux server, so i tried Ubuntu.
All ok, only that command lines not...
So how about add a GUI? seems by now this is not popular between users... So "i found" **Webmin**.
Is not a system GUI, but a Web based GUI (Runs using your web browser).
Now i was really going for it and... Uau! Lots of improvements and new add-ons!
I guess its a must for all (at least for beginners, like me).
