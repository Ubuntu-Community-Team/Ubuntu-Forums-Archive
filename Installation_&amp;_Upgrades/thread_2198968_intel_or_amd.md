### Post by QIII on 2014-01-11
AMD64 is a historical reference only.  AMD brought the first 64bit hardware to the consumer market, so it got called AMD64.  You will also find it called x86-64.

Anyway, don't worry about it.  It will work on your 64 bit Intel hardware.
