> **Duncfog said:**
> I have a Intel 915G/GV grantsdale chipset and want to upgrade the memory.  How do i find what memory my chipset supports

you mean upgrade the ram, right? simply open the computer and remove a stick. take it with you when you go to buy the new ram. and say your computer has 2 slots. and only using one, it's a safe bet you can add another stick of the same type of ram. or if both slots are used, you should be able to double the ram in both slots. for example 256x2=512 to 512x2=1024 (1gb).
