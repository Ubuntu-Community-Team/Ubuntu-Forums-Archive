### Post by ajgreeny on 2008-07-15
Did you burn the CD as an image or just copy the file to the blank CD?  If you did the latter, that is why you are having the problem.  Using brasero in your wubi install, or whatever you have in windows to burn CDs, make sure this time that you chose to burn CD image when you get to the first brasero screen.  I don't use windows any more so can't help with programs for that, but I'm sure they are all able to burn a CD image, though the windows own burning software may not.
