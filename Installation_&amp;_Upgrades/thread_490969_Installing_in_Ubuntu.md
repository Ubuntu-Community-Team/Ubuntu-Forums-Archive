### Post by aquavitae on 2007-07-03
I assume you're new to linux?  In linux you generally don't need to do the windows-style download-click install.  Instead use the package manager wich automatically downloads and installs the program from a repository on the internet.  In ubuntu the package manager is Synaptic (or Adept if you're using kubuntu).  It will be under your menus somewhere.  Run it, find firefox in the list, click install, find firefox in your menus, borwse internet.  Easy!
