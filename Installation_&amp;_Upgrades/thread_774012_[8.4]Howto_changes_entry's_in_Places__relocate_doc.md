### Post by eldragon on 2008-04-29
how i would do it:
just leave the Documents folder in your home empty
add the following line to the end of /etc/fstab
```

/mnt/partition_of_documents/   /home/username/Documents/   bind   0   0

```
remember to replace the target partition name ;)

this would make your documents folder point to that partition.

you could bind disk drives/folders/ whatever.

this could effectively give you for example, a same Documents folder for every user.
