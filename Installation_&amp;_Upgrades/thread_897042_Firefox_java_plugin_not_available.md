### Post by wd5gnr on 2008-08-23
I assume you are running AMD64. If this is the case, you can do one of two things. Either install OpenJava which works about 99% of the time (tip: stereo sound seems broken but if you can select mono sound, that seems to work) OR install 32-bit Firefox and the Java plug in for it.

You can read more about this in the x86_64 forum.
