### Post by Mark Phelps on 2012-04-23
OK, so what boots? Win7? Ubuntu? Neither?

Did you make the mistake of using the SLIDER to shrink the Win7 partition to install Ubuntu?  That can have the effect of corrupting the Win7 OS partition so it won't boot anymore.

And, if you did not bother to use the Win7 Backup feature to create and burn a Win7 Repair CD BEFORE you forced the Ubuntu install, you don't have a CD to use to repair the Win7 boot.

So, what do you want working again? Win7? Ubuntu?

If you have access to a working PC that has Internet access, then read through the link below to download and burn a Boot-Repair CD:

[http://ubuntuforums.org/showthread.php?t=1769482]("http://ubuntuforums.org/showthread.php?t=1769482")
