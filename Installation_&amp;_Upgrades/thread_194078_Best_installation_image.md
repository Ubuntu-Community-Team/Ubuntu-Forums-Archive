### Post by 23meg on 2006-06-10
"Desktop" is a live CD that also gives you a graphical install option while the OS is running in live CD mode. "Alternate" is the equivalent of the old "install" disc and has  a few more in-depth installation options.

### Post by aysiu on 2006-06-10
The Desktop CD is a live CD with a new graphical installer.
The Alternate Install CD is a text-based graphical installer.

Both CDs can install Ubuntu. One runs a live session, and it allows you to install from the live session.
