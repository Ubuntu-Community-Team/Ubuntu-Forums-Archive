### Post by darkod on 2013-01-07
Well, you are typing it wrong. If you are using locale keyboard, make sure the correct one is selected because the letters you are typing might be different otherwise.

For resetting your password using recovery mode:
[http://www.psychocats.net/ubuntu/resetpassword](http://www.psychocats.net/ubuntu/resetpassword)

### Post by mlentink on 2013-01-07
Please also be aware that passwords in ..x are case-sensitive, so check whether you have pressed caps lock. If it's a laptop you might also want to check whether Num Lock is activated, as this can interfere with the keys on some laptops.
