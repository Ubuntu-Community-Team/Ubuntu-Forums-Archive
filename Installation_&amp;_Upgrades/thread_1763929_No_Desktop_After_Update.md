I guess there is no one with a solution to this, so what I did to save my files was used the Ubuntu live cd to start terminal and mount the root.disk file in my c:/ubuntu/disks folder and retrieved the files i needed that way, then reinstalled Ubuntu. Kind of a crude way to do things but what ever works when your stuck I guess.
