### Post by aquavitae on 2008-06-20
What arguments are passed to the two kernels - i.e. do they differ. Also, have a look on the internet for bug reports for that kernel version - it may be something known, if not, file a report. It's probably best to try to get the new kernel working, but if you want to boot the old one by default, edit /boot/grub/menu.lst and change the default option. I can't remember exactly how to do this and I'm not on linux at the moment, but the file is well documented and it should be easy. IIRC its just a case of changing a number.
