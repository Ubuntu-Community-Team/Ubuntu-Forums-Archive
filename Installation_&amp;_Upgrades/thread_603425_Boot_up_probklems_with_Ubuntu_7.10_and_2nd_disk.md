### Post by bulldog on 2007-11-05
I think you should have a look at your fstab.
Your second disk is mounted in fstab,but your uuid is wrong i guess.
You can find the uuid by the next command in a terminal,
```
ls /dev/disk/by-uuid -lh  
```

Your second hdd should be connected to see this!!
So maybe you should connect both hdd's and try in recovery mode.
If this fail,you can use the live cd instead but the command will change into,
```
sudo vol_id -u <partition>
```

If you find the uuid,change it in the fstab and reboot.
You can edit the fstab by,
```
gksu gedit /etc/fstab
```

You can make a safety copy first to be sure
```
sudo cp /etc/fstab /etc/fstab_backup
```
