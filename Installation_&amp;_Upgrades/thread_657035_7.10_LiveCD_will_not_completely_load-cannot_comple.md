### Post by Fred_E _krugar on 2008-01-03
Ok I got everything working now. This is what you have to do.


Start live cd by choosing install ubuntu with safe graphics mode.

Once everything is installed go to Nvidia and download the drivers (envy will not work) [http://www.nvidia.com/object/linux_display_ia32_169.07.html](http://www.nvidia.com/object/linux_display_ia32_169.07.html) . Just download them to your desktop.

Once they are D/L  hit the Cntrl+Alt+F1 keys then you will need to log in.

When you are logged in you need  to shut down the x-server to do this type:
            sudo /etc/init.d/gdm stop

then you need to go to where the drivers were D/L too: 

              cd /home/yourusername/Desktop

Install the drivers like so :

              sudo sh NVIDIA-Linux-x86-169.07-pkg1.run

Accept license answer yes to all questions and let it run its course, once it is done restart x-server: 

               sudo gdm


I hope this helps you guys out, it took me two days to finally to get ubuntu back and running right with my new 8800GT. :)

### Post by Gibby82 on 2008-01-04
Ok good news! I have successfully installed the drivers and everything is running smoothly now. I tried the CLI method Fred_E_krugar outlined, but it didn't work due to me not having some libc stuff installed. Basically the installer would error out because of it. So, noticing Envy was mentioned I looked that up and installed it (which I believe installed the missing libc stuff). I then had it install the drivers and that was that. So, my final solution, for others:

**1.** Boot in Safe Graphics mode.
**2.** Install.
**3.** Reboot into the desktop. 
**4a.** Install all of the required files for the drivers (libc and whatever else) and run the  install program:
  a. CTRL ALT F1
  b. sudo /etc/init.d/gdm stop
  c. cd to the driver location (/home/yourusername/Desktop) For me it was just Desktop. 
  d.Install the drivers: sudo sh NVIDIA-Linux-x86-169.07-pkg1.run
  e.Accept license agreement etc.
  f. Once finished- restart x-server: sudo gdm
OR
**4b.** Download and install Envy:
  a. [http://albertomilone.com/nvidia_scripts1.html](http://albertomilone.com/nvidia_scripts1.html)
  b. Once installed follow the instructions found on the website for installing the drivers.
  c. Restart.
**5.** Enjoy your new video card!
