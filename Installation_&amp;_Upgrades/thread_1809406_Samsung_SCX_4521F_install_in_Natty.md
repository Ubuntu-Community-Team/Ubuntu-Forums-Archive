### Post by hodad on 2011-07-21
YAY!\\:D/

I actually solved the problem ON MY OWN!

All I did was go to the "Printer" icon under "System" in Unity, and DELETED the existing Samsung 4521F printer (which had a nasty red exclamation mark over it).

Then I just chose +ADD, and was asked (can't remember exact wording) if I wanted to install a new printer driver.  Then I navigated thru the choices for the 4521F, and the printer came back  online.

I'm so relieved that I didn't have to try fixing in the terminal, trying hundreds of lines of code to modify countless setup files in various locations to no avail! ( I did that a couple of years ago on the same printer; what a nightmare!).

Goes to show that you should exhaust the simplest methods first...
