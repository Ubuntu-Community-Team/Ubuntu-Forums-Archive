### Post by drs305 on 2009-03-29
Yes, you can still install ubuntu from the livecd. You can either make or select the partition during the installation or boot to the livecd Desktop and select System, Administration, Partition Editor (gparted) and create the partition(s) before you start. You will have to have a separate partition for ubuntu since windows is ntfs and ubuntu is formatted to ext, normally ext3.

In response to your PM about Xubuntu vs Ubuntu:

If you have the resources (a relatively modern computer with a normal amount of ram) I'd go with Ubuntu. Xubuntu is designed as a lighter weight OS but it has good features. 

You can install one and then install the other's "Desktop" via synaptic (the software manaager) to see what it is like, having access to the look of both.
