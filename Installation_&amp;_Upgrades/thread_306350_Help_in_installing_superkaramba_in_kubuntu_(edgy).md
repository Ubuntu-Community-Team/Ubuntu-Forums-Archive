### Post by deepwave on 2006-11-25
Run System > Adept Manager (Manage Packages)

In Adept from the Menu: Adept > Manage Repositories

Make sure you have the line or similar
deb [http://us.archive.ubuntu.com](http://us.archive.ubuntu.com) edgy universe

superkaramba is part of the universe packages.

Press Close, when done with edits.  (This can also be done in synaptic and manually editing /etc/apt/sources.lst)

Search for superkaramba.  Install, and voila done!

As for theme, superkaramba lets you find new themes or open existing ones.  Additionally you can browse [http://kde-look.org]("http://kde-look.org") for more superkaramba themes.
