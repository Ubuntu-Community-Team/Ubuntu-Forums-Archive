This problem has been documented. Try these links.

[https://wiki.ubuntu.com/PowerPCKnownIssues](https://wiki.ubuntu.com/PowerPCKnownIssues)

[https://bugs.launchpad.net/ubuntu/+source/linux/+bug/1066435](https://bugs.launchpad.net/ubuntu/+source/linux/+bug/1066435)
