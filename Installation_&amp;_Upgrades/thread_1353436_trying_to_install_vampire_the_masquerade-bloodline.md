### Post by Gr3ghammett on 2011-08-18
i've installed Star Wars Battlefront II which houses 4 install discs. What I've done is used Furious ISO to mount the disc, copy all the files to a folder on my desktop from each disc in separate folders, and when the installer asks me to switch discs in a message box, I point it to the .cab file it requests off the files on my computer from the copied disc it requests.  For me this has worked flawlessly when it comes to installations. You can also configure wine to say the cd drive of your computer should be the same as the one it should emulate or use for it's cd rom drive. I hope this helps. Please let me know if it works or if you need any additional help.
