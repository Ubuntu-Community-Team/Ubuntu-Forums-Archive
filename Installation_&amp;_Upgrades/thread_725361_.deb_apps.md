If you can find packages on [http://getdeb.net/](http://getdeb.net/) as **PurposeOfReason** suggested you can add that link to your source list.Type in terminal

```
gksudo gedit /etc/apt/sources.list
```

add line           deb [http://ubuntu.org.ua/](http://ubuntu.org.ua/) getdeb/

save and close

```
sudo apt-get update
```
