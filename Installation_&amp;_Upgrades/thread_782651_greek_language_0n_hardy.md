### Post by simosx on 2008-05-10
> **mastoul said:**
> Hello to everyone.Recently i install ubuntu 8.04 in Greek language.The problem is that some menus appear in greek and some in english!I made also an upgrade from 7.10 to 8.04 at my laptop and there the language is correct everywhere.What should i do?

You should click on System/Administration/Language support.
Once there, you will be prompted to add some packages. Click ok and you are done. These are the missing Greek translation packages.

The reason for this is that the CD does not fit all translation packages of all languages, so these are added with a step like this.
