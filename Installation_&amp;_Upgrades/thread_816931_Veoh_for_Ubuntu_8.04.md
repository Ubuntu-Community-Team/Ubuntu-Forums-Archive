### Post by Francewhoa on 2009-07-03
Veoh Player isn't available for Ubuntu yet. **In the meantime you can watch full Veoh videos with the 'Illimitux' plug-in for Firefox [https://addons.mozilla.org/en-US/firefox/addon/11037](https://addons.mozilla.org/en-US/firefox/addon/11037)**

Video tutorial on how to use this extension at [http://www.illimitux.net/faq.php](http://www.illimitux.net/faq.php)

Note: Illimitux is no longer working for Veoh & Megavideo since June 2009. The developers already know about it and are working on it [http://www.illimitux.net/contact.php](http://www.illimitux.net/contact.php)
