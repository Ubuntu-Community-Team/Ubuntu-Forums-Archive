### Post by adam814 on 2010-01-19
TOR definitely does have limitations.  If that won't work for you you'll need to hunt around for a solution.

You might be able to get away with an SSL proxy located somewhere else.  You *might* even find a free one.

Or if a few euros a month isn't a problem there are lots of companies that will rent you a shell on a machine somewhere in Europe.  Most people use them for IRC bots, but there's nothing to stop you from using them for a proxy server really (other than possibly a monthly bandwidth cap).
