### Post by Herman on 2012-04-13
The Ubuntu 'Alternate Installation CD' .iso file features two options called;

'guided - use entire disk and set up LVM', 
or 
'guided - use entire disk and set up encrypted LVM'.

The chance to select either of those begins at the start of the 'Partition Disks' stage of the installation, right after the installer sets itself up for your hardware.

---

### Post by darkod on 2012-04-13
To add, if you enter in the manual option you will also be able to configure LVM and this option gives you best control over the partitioning, especially if you want to use dual boot.
