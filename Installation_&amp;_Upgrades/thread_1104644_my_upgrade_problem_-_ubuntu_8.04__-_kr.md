### Post by Partyboi2 on 2009-03-25
The binary-lpia architecture is not available from then ubuntu repos. I would suggest commenting out those 3 ubuntu repos from your sources.list and keep the dell ones.

If you are not sure how to do this open a terminal (Applications>Accessories>Terminal) and post the output to
```
cat /etc/apt/sources.list
```

### Post by brianchidester on 2009-03-25
What system are you using? Also in your Software Sources did you check main universe restricted multiverse on the first tab? (those are Canonical -supported, Community-maintained, proprietary drivers, and restricted software I believe) If you did then you need to uncheck them and the errors will go away, the default repos in the Dell versions of Ubuntu don't have that same structure.
