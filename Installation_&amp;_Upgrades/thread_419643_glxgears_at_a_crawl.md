### Post by GeDaMo on 2007-04-23
You can stop the X server by witching to a console using Alt-F1, using
```
ps aux | grep X
```
to get the process ID of the X server then using
```
kill <process ID>
```
to kill it.

I don't recommend installing the driver form the Nvidia website however - it may cause problems with your X configuration.

Can you post your /etc/X11/xorg.conf ?
