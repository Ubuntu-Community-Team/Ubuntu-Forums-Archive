### Post by grahammechanical on 2021-04-09
That machine was originally sold to run either Windows XP or Windows Vista. Although the computer processor is 64 bit it might not be possible to run Ubuntu 20.04 on it. May be the machine does not have enough RAM or may be the video card is not powerful enough or does not have enough of its own video memory to run the graphical desktop environment of Ubuntu 20.04. Does the video card share memory with the motherboard?

Please provide details of the hardware. The CPU; the RAM; the video card. You may need to try a different flavour of Ubuntu, such as Lubuntu. Or even a different distribution of Linux. You need an operating system that can run on less powerful hardware. Ubuntu might not be suitable for that machine.
