### Post by ajgreeny on 2012-03-05
If you install Ubuntu and then run ```
sudo apt-get install xfce4
``` you will get just the basic xfce desktop environment but without all the additional stuff that you would get with xubuntu-desktop package.

I do not know how xfce will deal with global menus and the overlay-scrollbars that come with ubuntu 11.10, but there are ways to deal with that if you don't want them;  you can even stop them running or remove them from unity in 11.10 if you wish.

Unity 2d is certainly less resource hungry than unity 3d so you could probably run it successfully depending on the age of your computer, but it may be worth looking at cinnamon, though it is in the early stages of development at the moment, as unity is probably less configurable than some other alternatives.
[https://launchpad.net/~gwendal-lebihan-dev/+archive/cinnamon-stable](https://launchpad.net/~gwendal-lebihan-dev/+archive/cinnamon-stable)
