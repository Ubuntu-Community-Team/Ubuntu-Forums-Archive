### Post by deathjaguar on 2023-03-01
If anyone else runs into this problem, switch from Wayland to X11. To permanently do that:
Edit the [FONT=var(--ff-mono)]/etc/gdm3/custom.conf file and uncomment [/FONT][FONT=var(--ff-mono)]WaylandEnable=false[/FONT]
