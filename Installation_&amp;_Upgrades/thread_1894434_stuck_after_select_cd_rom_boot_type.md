### Post by Rex Bouwense on 2011-12-12
Did you run a md5sum on the download?

[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

Did you burn the ISO at the lowest possible speed?  Burning at higher speeds sometimes results in a bad burn.

Try burning the ISO to a CD.  Occasionally, burning to a CD works when burning to the DVD does not.
