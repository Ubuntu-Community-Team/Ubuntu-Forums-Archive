### Post by e79 on 2010-10-09
Ubuntu server is command-line only when you install it. If you really want a GUI, see here : [https://help.ubuntu.com/community/ServerGUI](https://help.ubuntu.com/community/ServerGUI) or try another distribution (Debian, Fedora etc)

But I would suggest to try with the command line as you WILL have to use it anyway at a certain point with ANY *NIX servers. Don't fear the CLI, it is your friend, and the Forum Community will always be here to help you in case you need it  :P .

Hope this helps
