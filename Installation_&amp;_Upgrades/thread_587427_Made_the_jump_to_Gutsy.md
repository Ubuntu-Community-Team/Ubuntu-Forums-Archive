### Post by hise0001 on 2007-10-22
Beryl and Compiz and merged back together as Compiz fusion (or something like that).

Do Administration->Restricted Drivers Manager to enable 3d features of the driver.

Do sudo apt-get install compizconfig-settings-manager to install the configuration manager.

Do Preferences->Appearance->Custom to configure the level of 3d functionality.

That's my real quick-n-dirty... hope it helps.
