### Post by Rocket2DMn on 2008-04-20
Hi andyhuli,
Are you able to access the BIOS in the first part of your computer's boot sequence.  You usually only have a second to hit the correct key to open the BIOS.  Common keys are F2, F8, F12, DEL, or ESC.  From the BIOS you can change the boot order to make the CD/DVD drive ahead of the hard disk.  You should then be able to install from the LiveCD.
What is different this time around than the first time you installed?
