### Post by tgalati4 on 2009-08-24
Rhythmbox has a built-in avahi share daemon.  Run rhythmbox on two diffent machines and you can play media files from each one.

Mediatomb also works.  I haven't had problems on a LAN but it's been spotty across the web and from Linux to Windows machines.
