### Post by linuxinstalledfromhdd on 2011-05-23
You could reset the changes you have made.  Create a live stick of Ubuntu on a USB drive.  Then make the changes again, boot from the newly created stick, and then compare the settings from within the live USB in terminal with the old system.  It would be a matter of changing them to whatever works in the new system and transferring those settings to the old system.
