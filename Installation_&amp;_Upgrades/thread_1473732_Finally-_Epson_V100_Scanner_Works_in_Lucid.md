### Post by pepage on 2010-05-05
It was really easy to get my Epson Perfection V100 PHOTO scanner to work in Lucid (10.04).

Just downloaded the 2 recommended deb packages from AVASYS Corp. and installed both these packages using the GDEBI command. That is it! Simple Scan now shows both my HP Laserjet 3200 and Epson V100 scanner.

I also added QCAD to my system. There is a problem with the package but this was fixed by manually adding QCAD to the Main Menu. I did purchase QCAD online but when asked if I wanted Windows or Linux, I replied Windows. Why?....Because I know it will work in the future i.e. not broken in the next release. I love Linux because I do not need an anti virus program but still feel the need to have a Windows backup. Ubuntu is getting better!
