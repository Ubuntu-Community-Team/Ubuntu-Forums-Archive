### Post by berbs on 2005-10-21
follow this link and install the 915resolution script on your machine (sudo make install). Open bootmisc.sh (/etc/init.d/bootmisc.sh) and add the following line after the first if block (about line 11)

915resolution 3c 1400 1050

It should work once you restart X

### Post by gelse on 2005-10-22
the problem is the ati-driver included.
i figured out that the included version (8.16.20) doesnt like the 1400x1050 resolution at all.
if you want to spend some work on it to get it to work, then i suggest using the _newest_ driver from ati (although the older driver 8.14.13 does work well, too, and solves some very special problems, i wouldnt recommend it, because getting it to work in the newer versions of ubuntu means a bit of C-coding within the modules...)
there are plenty howtos out there, but if you want me to help you, drop a line.
