### Post by RasterBurn on 2010-02-24
hello there, i have encountered the same problem, i have solved the same problem as well, the ubuntu repositories are missing a required file for an app called "python-soya" which is required for the game the missing file is libode0debian1 which can be downloaded from [http://packages.debian.org/lenny/libode0debian1](http://packages.debian.org/lenny/libode0debian1) and installed manually before installing balazar :)
