### Post by mouseboyx on 2007-12-03
KUbuntu and Ubuntu are exactly the same except for the software that comes with them. Don't worry its a common misconception.

Ubuntu uses the gnome desktop environment as default and Kubuntu uses the kde as default. you can run both on both.

Also you can only edit partitions from the live CD or the lowest run level when your drives are not mounted so, the Ubuntu live CD will work the best.
