### Post by bryanl on 2010-08-20
The best option is to use is to use [http://www.google.com/patents](http://www.google.com/patents) or one of the other patent search sites as they provide PDF versions of the original patents.

If you need to view USPTO images, then you can use the TIFF plugin.
[http://vinay.in/mozilla-tiff-plugin/](http://vinay.in/mozilla-tiff-plugin/)
The biggest problem I have with this is that the USPTO only shows one page at a time and the plugin doesn't support printing.
