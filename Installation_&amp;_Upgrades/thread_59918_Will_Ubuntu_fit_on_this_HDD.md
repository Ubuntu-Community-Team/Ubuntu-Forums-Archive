> I basically want to use this for internet/email/Instant messaging and stuff like that yes. for that ..this is more than enough...though you would want to install XFCE windows manager later..for faster desktop reflexes...

thats your choice....but you will want to have a decent %aside for swap (which is used when ram is low). For what you have.... 512mb-1gb sounds good!
