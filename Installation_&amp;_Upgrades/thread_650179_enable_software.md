### Post by alan34 on 2007-12-26

Click Settings - Repositories  make sure Propietary drivers for devices is ticked then close.

Then search for nvidia should be there. 

I would have a read of the packages listed if you have a newer graphics card maybe nvidia-glx-new
might be better.
