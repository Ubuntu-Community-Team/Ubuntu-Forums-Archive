### Post by zvacet on 2009-11-13
Check [md5sum]("https://help.ubuntu.com/community/HowToMD5SUM") and check [disc integrity.]("https://help.ubuntu.com/community/Installation/CDIntegrityCheck?action=show&redirect=CDIntegrityCheck") if md5sum is not correct download same iso eiyh torrent and point download to thr folder with existing iso.That way torrent will just remove bad files and replace them with good ones.After thar run md5sum again and if it is O.K. burn disc on lower possible speed and after that run disc integrity.When you are done and all is correct go for install.
