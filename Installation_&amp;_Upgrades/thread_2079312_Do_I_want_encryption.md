### Post by AlexDudko on 2012-11-01
No problem using Dropbox on an encrypted drive/partition.
But do you really need encryption?
Some benefit of encrypting /home partition is on laptops, then in case it is stolen nobody will get your private data from the drive. But you also need to set grub password and disable recovery mode boot option.
