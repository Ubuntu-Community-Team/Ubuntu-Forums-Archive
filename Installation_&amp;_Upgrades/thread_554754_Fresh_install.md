### Post by dabl on 2007-09-19
Make an Alternate Install CD?

[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

Under the green "Start Download" is a check box, which you can use to get the Alternate Install ISO.  It is a "character mode" installer -- don't worry, you don't need to know any bash commands to get through it.
