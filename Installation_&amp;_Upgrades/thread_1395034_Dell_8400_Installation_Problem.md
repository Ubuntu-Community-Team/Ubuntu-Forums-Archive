### Post by milton1 on 2010-01-31
This is not normal. There are several possibilities: bad burn, bad reader, bad RAM. Probably not a bad hdd, as it sounds like you are not even reaching the partitioner. Try selecting the "check disk for errors" option when you boot the cd.
