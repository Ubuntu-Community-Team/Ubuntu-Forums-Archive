### Post by Mark Phelps on 2011-10-24
Boot into Ubuntu. Install Grub Customizer.  You will have to Google for this as I'm not at my Linux box and don't have the ppa link.

But once that is installed, run it, select Preferences, and in the first tab, open the pulldown and scroll past the "position" entries until you find one like Windows 7 Loader on ....

Select that, Save, and reboot.

Win7 should now be the default.
