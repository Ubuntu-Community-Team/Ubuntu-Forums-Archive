### Post by snowpine on 2008-11-05
Hi there, I would first of all recommend installing Fluxbuntu from a CD instead of a DVD, burning it at a slow speed, and checking the md5 sums to make sure there were no burn errors. It would also be helpful if you told us more about your system specs. 

Finally, the Fluxbuntu installer has the reputation of being somewhat buggy. There is a "Fluxbuntu remix" that might be worth trying if all else fails (see my 2nd post): [http://ubuntuforums.org/showthread.php?t=954549](http://ubuntuforums.org/showthread.php?t=954549)
