### Post by KiLaHuRtZ on 2010-03-19
Well, you can change all instances of "karmic" to "lucid" in '/etc/apt/sources.list', install it and then change "lucid" back to "karmic".

Make changes, then...

```
sudo apt-get update
sudo apt-get install ubuntu-artwork
```

Revert changes, then...

```
sudo apt-get update
```

Just be sure you make a backup of your sources file first!
