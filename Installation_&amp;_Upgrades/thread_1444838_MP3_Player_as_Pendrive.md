### Post by Knight on 2010-04-02
Was finally able to do it using [http://www.linuxliveusb.com/]("http://www.linuxliveusb.com/"), really simple and easy to use. It doesn't seem to work on boot, but I suspect this PC doesn't support USB booting and will try on some other ones later. However it does install and the VM part of it is functional.
