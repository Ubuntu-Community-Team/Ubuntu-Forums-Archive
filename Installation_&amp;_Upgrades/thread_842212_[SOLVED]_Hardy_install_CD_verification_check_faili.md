### Post by Vivaldi Gloria on 2008-06-27
I don't know why the server cd fails for you. But you can try the mini cd:

[https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)

This is an about 10MB iso cd which downloads everything during installation. So if you have a decent net connection you can try it.
