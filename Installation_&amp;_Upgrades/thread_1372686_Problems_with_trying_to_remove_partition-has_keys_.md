### Post by oldfred on 2010-01-05
The key symbol is a lock because it is mounted. You cannot edit mounted partitions or the partitions your are working from. The best way is to use the liveCD you used to install or download a gparted liveCD. Even the liveCDs will use the swap it finds so you have to click on the swap and swapoff.
