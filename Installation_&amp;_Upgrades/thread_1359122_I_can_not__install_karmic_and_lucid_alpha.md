### Post by sanderj on 2009-12-19
On my Dell Studio 1535, 9.04 was OK, but 9.10 and Lucid don't boot or boot very slowly. See [https://bugs.launchpad.net/ubuntu/+source/linux/+bug/404899](https://bugs.launchpad.net/ubuntu/+source/linux/+bug/404899)

I don't know whether this is the same as the problem with your laptop, as your laptop is quite different from mine.

Advice:
1) turn off "quiet splash" in the boot options and then see at what stage the boot process stops.
2) use another computer to boot Karmic/Lucid and create a bootable USB stick, and see if that works in your Compaq laptop.
