This is very simple tutorial for newbies:

Install java6 sdk:
> sudo apt-get install sun-java6-jdk

Install netbeans5.5: 
Note: Follow the screen instruction, dowload the netbeans5.5 from the sun's website and place it into /tmp directory.
> sudo aptitude install netbeans5.5


Then you should able to run Netbeans5.5 from Application->Programming->NetBeans
