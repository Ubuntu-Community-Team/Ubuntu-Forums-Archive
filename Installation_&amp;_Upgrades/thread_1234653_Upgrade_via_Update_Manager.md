### Post by Jerriy on 2009-08-08
Wrong.

It can and will delete/replace lots of stuff. You have the option of preventing only some (but not all!!!) of the stuff that's gonna be deleted. 

Anyway from my personal experience: DONT upgrade Ubuntu via update manager! Make a clean break and **clean install** the new version!

I still am encountering problems as a result of me upgrading via "update manager"

But if you're stubborn and still want to upgrade via update manager then get ready for troubles [like these]("http://ubuntuforums.org/showthread.php?t=1212276") for example.

Don't say you've not been warned later!
