### Post by ammostro on 2010-05-02
I've been able to install Ubuntu 10.04 Netbook Remix from USB drive with no issue, selecting "USB something" in the menu that comes up hitting F12 during boot (when the Dell logo is displayed). If this doesn't work for you, try checking if "USB Emulation" is enabled in BIOS settings (you can check them hitting F2 at boot).
