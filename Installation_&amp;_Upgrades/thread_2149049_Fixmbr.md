### Post by kat47 on 2013-05-27
Hi i have just attempted to fix my master boot record after having problems earlier with an installation of ubuntu 12.04LTS.
I entered my windows7 using my repair disc(you can make one of these in windows 7) and attempted to fix mbr using the command prompt,it would not work.
So after a quick google i found the problem.
Again using the repair disc for windows 7,choose command prompt and type c:,then press enter,
then type bootrec.exe/fixmbr.
A message will be displayed saying its fixed the problem,not in those words though:) so just type exit and restart your machine and windows 7 should start:).
Just wanted to share this in case someone else has the same problem.
