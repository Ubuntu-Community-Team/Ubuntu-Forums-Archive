### Post by wilee-nilee on 2009-11-12
I doubt if anybody on here will tell you to go ahead and upgrade in the 1st place especially with encryption. Do yourself a favor back up your stuff off your computer then do a fresh install and set up the encryption on the fresh install. 9.10 has some new features which make a fresh install the best way to go. Also make sure your computer will run with a live cd and has no inherent problems that can't be fixed.
