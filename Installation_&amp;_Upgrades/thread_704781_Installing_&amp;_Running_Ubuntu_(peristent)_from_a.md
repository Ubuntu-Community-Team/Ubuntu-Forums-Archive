### Post by Zakhafr on 2008-02-22
The question has already been submitted many times, I know, but I recently found a post on PendriveLinux :

[http://www.pendrivelinux.com/2008/02/13/pendrivelinux-2008-install-from-windows/]("http://www.pendrivelinux.com/2008/02/13/pendrivelinux-2008-install-from-windows/")

That is a definition of what I call **easy** installation.

It is just 3 clicks :
-click 1 = download the file
-click 2 = uncompress the file to the USB key
-click 3 = run a command file to make the USB key bootable.

I can also guess it was made by persons who really understand why we would want a Linux on an USB key!
I mean the fine stuff is that you don't even have to format your key (assuming it is already in FAT32 format), even less to partition it. So your key will still be usable by other systems to exchange data without having space blocked by partitions. You know, Windows XP is very "basic", it does only handle the first partition of an USB stick, and only if this first partition is in FAT (16 or 32) format.
Persistence is handled on this installation through a "loopback" file. So, if you need more space for persistance, you just run the command to make a bigger file, no need to re-partition!

When do I use my USB Linux key? At work, during lunch break. Because we have Windows PC where we have no admin rigths. So when I want to run my personnal stuff and programs wihtout interfering with my office PC, I just start it from the USB stick.
It does also work perfectly on this office PC (Core Duo/ 4x3 Screen / basic sound & Video card) and automatically switch drivers on my personal laptop (Centrino / 16x10 wide screen / ATI card and advanced Intel sound system).


And it is... **Mandriva** !


Whereas the tutorials to install Unbuntu on an USB drive are very very long, and not easy at all.
They result in an USB stick been partitionned, so it is only partially usable to exchange files with Windows.
There is a post on PendriveLinux also, to install Ubuntu. I tried it and it produced an unstable installation :
-1st boot = OK
-2nd boot = a bunch of GP Faults at startup
-3rd boot = does not boot anymore !


On the French forum, one post out of ten is from people asking how to install Ubuntu on an USB Drive and boot from it (in persistent mode).
Here there are also some posts asking how to do that.



So, I do sincerely hope some people having skills with Ubuntu (that's not me, I'm a newbie), or even Canonical, would one day consider doing a really easy install on USB drives (such as the one I linked), because yes, there are situations where you want Linux on an USB drive!

When this will be made, I'll be very glad to try Ubuntu instead of Mandriva.


And who knows, maybe one day I'll decide to install it in place of Windows on my PC :)



*Keep up the good job!*
