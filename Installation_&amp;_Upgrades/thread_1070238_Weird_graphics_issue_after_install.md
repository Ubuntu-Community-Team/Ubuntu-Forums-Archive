### Post by nicoverbking on 2009-02-15
Did you install any drivers (like the nvidia drivers) after the install? That could be the cause.

I would insert the Live CD and reinstall Ubuntu. Make sure, before you install, that you checked the CD for defects. If after the install is done, the problem persists, I suggest you download the latest release (Ubuntu 8.10 Intrepid Ibex) and try that one.

OR

You might want to try Xubuntu, another variant of Ubuntu, that runs nicely on older machines, and might not suffer from the issue you have.

Take a look at : [http://www.xubuntu.org/](http://www.xubuntu.org/)
