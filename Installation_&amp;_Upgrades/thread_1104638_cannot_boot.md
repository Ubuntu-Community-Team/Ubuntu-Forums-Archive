if the disc turns out to be the culprit, then burn the iso at a slow speed, again.

### Post by Therion on 2009-03-23
Put the CD you burned in the drive and then open it up so you can see what's on it.

Do you see SEVERAL files and folders on the CD, or do you see only one file on the CD ending with .iso as the extension?

If you only see ONE file on the CD, you have not burned the CD as an "image" file, and your computer will not be able to boot from it.

See this tutorial, or post, if you need assistance:  [http://www.psychocats.net/ubuntu/iso](http://www.psychocats.net/ubuntu/iso)
