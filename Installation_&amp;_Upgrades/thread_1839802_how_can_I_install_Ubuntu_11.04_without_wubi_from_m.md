### Post by Mark Phelps on 2011-09-06

> how can I install Ubuntu 11.04 without wubi from my Windows XP?

What folks have basically told you is that you can't install Ubuntu using Windows XP.  

Instead, you have to create install media and boot from that -- selecting the option to use the entire drive.  That will overwrite your entire disk and replace XP with Ubuntu.
