### Post by howefield on 2009-12-13
Synaptic Package Manager is a program through which you can install the tahoma replacement font.

Go to System > Administration > Synaptic Package Manager and in the search field type tahoma. Click in the box to the left of ttf-tahoma-replacement and select "Mark for Installation"

Then apply, and apply again. Should install the file for you.
