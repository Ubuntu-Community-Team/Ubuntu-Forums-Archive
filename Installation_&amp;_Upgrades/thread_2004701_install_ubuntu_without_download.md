### Post by darkod on 2012-06-16
I don't know about the CD, but if you put the wubi.exe and the ISO in a same folder and execute the .exe, it should use the ISO instead of downloading over internet. But for that to work the wubi.exe and the ISo have to be the exact same version.

If you already have the CD, and wubi.exe is also on the CD, what you could do is:
1. Copy the wubi.exe from the CD into a folder.
2. Use some program and create an ISO from the CD and save it in the same folder.

That way you make sure the versions match. Start the wubi.exe and it should use the ISO.
