### Post by labinnsw on 2008-12-25
While completing the installation process, you will be given the option to format the entire drive or a portion of it. Just go ahead and start the installation process.

You can start the process by restarting your computer with the CD in the drive. Select boot from disk and follow the appropriate instructions on screen.
