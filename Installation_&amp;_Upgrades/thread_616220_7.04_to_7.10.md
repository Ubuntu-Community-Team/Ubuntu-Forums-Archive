### Post by PmDematagoda on 2007-11-18
Open the sources.list file for editing using:-

```
sudo gedit /etc/apt/sources.list
```

And then uncomment the repository addresses given in the error message by typing # in front of them. Save the file, do:-
```
sudo apt-get update

```
and then try upgrading again.
