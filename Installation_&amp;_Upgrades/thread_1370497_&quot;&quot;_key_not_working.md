### Post by Old_Grey_Wolf on 2010-01-02
Use the menu "System > Preferences > Keyboard". When the window opens select the "Layouts" tab. Select "+ Add". Then select the "Country" (Canada) then select the "Variants" that display the keyboard the way you want it. Then select "+ Add". 

Hope this helps. The instructions are for vanilla Ubuntu so you may have to hunt around the menus in NBR to find the equivalent.
