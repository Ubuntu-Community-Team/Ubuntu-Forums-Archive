### Post by slickymaster on 2015-01-16
You can't upgrade from 11.04 to 14.04 directly. Even though it's possible to update one version at a time, which in you case would involve upgrading from 11.04 to 11.10 to 12.04 to 14.04, there's always the possibility of problems and each one would involve downloading of packages and several hours of work.

So my advise to you is to consider a clean install of 14.04. **_Make sure everything is backed up_** before starting.

### Post by TheFu on 2015-01-16
Welcome to the forums.

Support for 11.04 ended in 2011. There is no easy (if any) upgrade path. Finding out-of-support repositories will be hard, if you did want to attempt the 11.04-->11.10-->12.04-->14.04 path.

Best to backup your programs, settings, and data, then install a fresh 14.04 LTS, and migrate your old programs, settings, and data over.

Folks like you (and I) should stay with LTS releases only. Do not install any intermediate versions 14.04, 16.04, 18.04 ... that's every 2 yrs.  Support for these is 5 yrs.

Many things have changed since that time. New way networking is handled, new major release of apache, new major release of Samba .... enough has changed that I would start fresh.

Sorry there isn't an easy answer.  Being on a supported OS matters just as much for linux as it does for other OSes.
