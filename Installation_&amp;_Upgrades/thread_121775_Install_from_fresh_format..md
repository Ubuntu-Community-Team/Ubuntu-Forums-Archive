### Post by Howcomes on 2006-01-26
Generally to get into the BIOS its one of the following keys: F1, DEL, ESC - There's ALWAYS a way to get into the bios, worst comes to worst try holding down those keys, or any keys for that matter, during boot to get a Keyboard init error. From there you may see a message indicating which key to use to get into the BIOS.
