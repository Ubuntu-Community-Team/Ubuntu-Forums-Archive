### Post by automaton26 on 2009-08-06
I strongly recommend that everyone always:

1) Uses md5sum (there is a Windows version) to check that the downloaded .iso file is *exactly* correct, before bothering to burn it to disk. That will remove one possible cause of errors. The numbers to check against are at [https://help.ubuntu.com/community/UbuntuHashes]("https://help.ubuntu.com/community/UbuntuHashes")

2) Uses the option in the burning software that will *verify* the data after it's been written to disk.

You can only be reasonably sure the disk is correct if you do **both** of these tasks.
