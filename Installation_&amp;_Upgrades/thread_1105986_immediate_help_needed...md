### Post by houcem on 2009-03-25
I had the same problem as you and it was related to memory modules. check the health of your modules by running a mem test 86+ ([http://www.memtest.org/download/2.11/memtest86+-2.11.exe.zip](http://www.memtest.org/download/2.11/memtest86+-2.11.exe.zip)) unpack it to usb key and boot from it if the test fails then you certainly have a memory problem.
