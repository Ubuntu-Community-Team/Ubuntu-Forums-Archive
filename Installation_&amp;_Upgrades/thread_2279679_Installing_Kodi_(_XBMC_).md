### Post by Vladlenin5000 on 2015-05-25
Kodi (or former XBMC) wasn't "designed" for netbooks because it requires a capable graphics systems. Your "Android box" has it whereas your netbook most likely hasn't.
You can try anyway: [http://kodi.wiki/view/HOW-TO:Install_Kodi_for_Linux](http://kodi.wiki/view/HOW-TO:Install_Kodi_for_Linux)
Once installed it can be started like any other program -or- like a standalone desktop environment. Just log out your normal Kubuntu session and before logging in again make sure you select Kodi instead of the default Kubuntu session and it'll go staright away to the Kodi interface. Assuming it works in your machine, that is.
