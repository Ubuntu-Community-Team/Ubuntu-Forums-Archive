### Post by Rocket2DMn on 2008-02-14
If you are currently using the LiveCD, then you should try using the alternate install cd which is a text based installer without a live session, but is very intuitive and easy to use.  This fixes most problems associated with using the LiveCD, which just doesn't work for everybody.
You can get the alternate install cd here by checking the box at the bottom for the alternate desktop cd - [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)
