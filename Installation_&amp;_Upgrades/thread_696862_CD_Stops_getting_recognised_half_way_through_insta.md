### Post by RedSquirrel on 2008-02-15
If you can't get through checking the CD for defects that's not a good sign. I'm not sure if it's your CD  drive that's causing the problem or if it's the Ubuntu installation CD that you have made.

Do other CDs work in that drive? If yes, I would recommend checking the md5 sum of the iso file you downloaded if you haven't done so already:

[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

If that passes, you could try burning again:

[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

Are you using the Desktop CD (LiveCD) to install? You might want to try the Alternate install CD instead.

You could also try another CD drive if you have one lying around.
