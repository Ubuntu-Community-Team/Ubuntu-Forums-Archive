### Post by coffeecat on 2012-07-22
Ubuntu 9.04 has been end-of-life and unsupported since October 2010 so connecting to the internet with a system which can have no security patches newer than then is not a good idea. Also, my guess is that you are wanting to use a mobile broadband USB dongle. 9.04 would have drivers for only some of the older ones, if at all.

Is there any reason for wanting to use 9.04? If you are able to use the current LTS, 12.04, you'll find support for many mobile broadband USB devices to be good and usb-modeswitch will already be installed.
