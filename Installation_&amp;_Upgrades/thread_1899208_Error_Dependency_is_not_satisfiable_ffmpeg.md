I am trying to install a DVD authoring tool 'ffdiaporama' but I get the error message - 

**Error: Dependency is not satisfiable: ffmpeg**

---

### Post by Paddy Landau on 2011-12-23
It means that the program you are trying to install depends on ffmpeg being installed.

To install it, open Ubuntu Software Centre, search for ffmpeg, and install it. Then try again.

If it still complains, it may require a specific version of ffmpeg. In that case, check the website of your DVD authoring tool to find out which version of ffmpeg it requires. The Ubuntu Software Centre will tell you which version is installed on your computer.
