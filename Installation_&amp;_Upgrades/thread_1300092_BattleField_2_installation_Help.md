### Post by Jim_in_Omaha on 2009-11-10
For what its worth. At this current moment I have BF2 running in the latest Wine 1.1.32 with 9.10 and punkbuster is working. 

What I did was follow the latest info in the Appdatabase in winehq.org to get it to work with minimal graphic anomalies, then to get Punkbuster to work I copied the pb folder from the demo of BF2 in wine to the fully installed 1.50 version of the BF2 after renaming it's pb folder to pb.old  

I'll keep testing this, also it is true if you set your video driver control panel to override the applications antialiasing settings and set it to 2x or more it clears up a considerable amount of graphics issues.

There is new activity in the winehq database for this game since 9.10 and 1.1.32

update: I did get one server to boot me after about 5 minutes... I'll keep messing with it.
