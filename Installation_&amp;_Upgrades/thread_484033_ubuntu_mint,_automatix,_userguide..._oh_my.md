### Post by canadianwriterman on 2007-06-25
Personally, I use Ubuntu proper. Many people are using Mint and liking it, because it contains most of the media codecs by default, which can save time if you need them. Automatix is not a derivative, though. It is a program containing scripts for installing codecs and a wide variety of other commonly-desired programs in Ubuntu. You install it like any other program into your Ubuntu OS. 

It is controversial and many here on the forums advise against using it due to the fact that it adds repositories to your system and because it may cause problems when updating your system. I have used Automatix for some time and have never had a problem, but I do a fresh install with each new version of Ubuntu, rather than an upgrade.
