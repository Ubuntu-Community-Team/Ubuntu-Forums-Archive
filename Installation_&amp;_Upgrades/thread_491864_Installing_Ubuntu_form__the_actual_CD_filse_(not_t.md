### Post by x64Jimbo on 2007-07-04
There is an installer file for installing from within Windows - it's still in very early testing phases, so I do not recommend it for beginners. 
How are you burning the CD? Have you tried burning the CD at low speed to avoid errors? Most of the time if the checksum is correct once downloaded, but errors appear on the CD, it's because the CD is burned too fast.
