### Post by bapoumba on 2007-06-06
[http://www.ubuntu.com/getubuntu/download]("http://www.ubuntu.com/getubuntu/download")
Choose the alternate version (click below the green "Start download" button) and run the text mode iinstall. You can install a *-desktop package afterward. The alternate install uses debian text-mode installer.
