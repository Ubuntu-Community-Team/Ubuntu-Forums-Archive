### Post by Coelocanth on 2007-02-26
1. Open CdBurnerXPpro
2. Click 'Create Data CD/DVD'
3. Put in a blank disc
4. Go to 'File'
5. Select 'Burn Disc From ISO File'
6. Browse to the ISO image.
7. Select 'Finalise Disc'.
8, Click 'Burn Disc'

*edit* These instructions are from their forums, so hopefully it works. Good luck!
