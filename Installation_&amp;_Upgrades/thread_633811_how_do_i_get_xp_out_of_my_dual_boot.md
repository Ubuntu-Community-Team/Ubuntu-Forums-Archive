### Post by aysiu on 2007-12-07
When you set up the dual boot did you go with the default installation options, or did you do an advanced Grub installation?

If you did a default installation, you can just unmount and delete the Windows partition, format it as Ext3, and then mount it as extra storage.

More details on the second part of that process here:
[http://www.psychocats.net/ubuntu/mountlinux](http://www.psychocats.net/ubuntu/mountlinux)
