### Post by Scotty1982 on 2007-10-17
I was having a lot of problems with installing updates for 7.10 installed from the downloaded cd image, here's how i got it to work:

Instead of installing 7.10 rc, install 7.04 (feisty fawn) then install all the updates for that, then upgrade it to 7.10. to upgrade, hit alt+F2 then type update-manager -d and click the icon to update to 7.10.

this method worked great for me. hopefully tomorrow's release will have the issue fixed. if not, this method should still work.
