### Post by louieb on 2009-07-13
The text based installer is great. -  Built by Debian, been around for years, pretty much bug free,  same questions as the live CD installer. (language,time zone,...)  Will set up dual boot - just like the live CD. 

The best walk-through I've found on how to use it [Illustrated Dual Boot Site]("http://members.iinet.net/%7Eherman546/index.html")

BTW the mini install CD (csabo2 post) uses the same text based installer as the alternated CD. The only diffrence it downloads packages from the net at install time.
