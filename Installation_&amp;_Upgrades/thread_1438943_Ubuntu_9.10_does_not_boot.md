### Post by quixote on 2010-03-26
I'm not sure what you mean:

1) You start from the CD, but it never gets any further than the text screen where one of the choices is to install.

or

2) You boot the CD normally, select the install button, but then it doesn't install.

If it's 1), I'd suspect a bad download or a bad burn.  Just try to download a new copy and burn it, maybe specifying the slowest speed this time instead of accepting "auto".

If it's 2), maybe you've forgotten to specify a mount point?  Although I don't think you need to do that unless you're manually partitioning, and it sounds like you aren't.
