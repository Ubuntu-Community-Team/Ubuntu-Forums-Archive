### Post by skyopensource on 2010-09-11
Hi Bargi,

Instead of installing the flash player using the install plug-in option in Firefox, try downloading and installing it separately from Adobe website: [http://get.adobe.com/flashplayer/](http://get.adobe.com/flashplayer/).

Under the "Select version to download" drop down list, select the version which suites to your Ubuntu. I have tried the "APT for Ubuntu 9.04+" as I have the Ubuntu version Ubuntu 10.04.1 LTS installed on my Notebook.

The flash player plug-in is working fine on my Firefox now.
