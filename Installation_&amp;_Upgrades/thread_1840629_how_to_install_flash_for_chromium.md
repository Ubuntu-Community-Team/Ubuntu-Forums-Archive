### Post by Enigmapond on 2011-09-07
Welcome to the forum. An easy way to accomplish this, I assume you have Firefox as well. Install the addon Flash-aid in FF. Run the script, use all the default settings and let it install the current flash. This should fix all your flash problems system-wide. It will also un-install anything that conflicts.

[https://addons.mozilla.org/en-US/firefox/addon/flash-aid/](https://addons.mozilla.org/en-US/firefox/addon/flash-aid/)

Cheers!
