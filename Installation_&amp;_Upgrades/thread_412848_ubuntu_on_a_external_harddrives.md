### Post by dfreer on 2007-04-18
5400 RPM is a standard laptop HD speed, saves you battery and it is quieter but the I/O is slower due to the low speed. If you want it to run completely from USB (e.g. portable), you'll need to get a 2.5" external HD, the 3.5" often require a seperate power cord to run.

Never installed OX10, but it and Ubuntu should run off the external with no problems at all (as long as you can boot from USB in BIOS, or have a boot manager on your primary HD that can).
