### Post by wilee-nilee on 2010-02-26
On the same HD you need a NTFS partition for both to read. If you need to shrinkW7 use it's disk manager, and use the onboard gparted on a live cd top resize the Ubuntu. 

You could also just use a external HD with a NTFS partition to do the same or a thumb drive.
