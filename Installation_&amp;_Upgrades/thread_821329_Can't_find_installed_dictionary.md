### Post by Partyboi2 on 2008-06-07
make sure you have opendict installed as well.
```
sudo apt-get install opendict
``` Once it is installed you can open opendict by Applications>Accessories>Dictionary opendict and under "Dictionaries" menu you will see the english-dutch dictionary you installed.
