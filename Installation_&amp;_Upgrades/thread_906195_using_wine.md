### Post by Paresh on 2008-12-08
I had the same problem with MS Reader loading and exiting after a second.

Trying to run it from terminal gave me an error log that indicated that msvcirt.dll was missing.

Note that this is not msvcrt.dll, which I had to download separately, but msvc**_i_**rt.dll, which is not even mentioned in any instructions I had read.

Once this was copied into the MS reader directory, it worked fine.
