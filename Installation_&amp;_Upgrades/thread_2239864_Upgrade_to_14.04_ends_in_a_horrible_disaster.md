### Post by grahammechanical on 2014-08-16
Want to get to a Linux command line? Go to Recovery mode and at the Recovery menu select Network to set up a network connection and then Root to get a root shell prompt. When finished with the commands type exit to get back to the recovery menu and then Resume to resume loading Ubuntu.
