### Post by darkod on 2012-12-06
It should be fine. The only thing might be the video driver depending what card you plan to use.
If the CPU is in fact one of the new AMD APUs with integrated Radeon HD, you will probably need to install the fglrx driver to get full functionality. It should still work by default, only that it uses a driver called radeon with more limited options.
For Nvidia you also need to do few steps to get a driver running. Very often nvidia won't even load by default, unlike ati. But the fix is easy, if it gets to that.
