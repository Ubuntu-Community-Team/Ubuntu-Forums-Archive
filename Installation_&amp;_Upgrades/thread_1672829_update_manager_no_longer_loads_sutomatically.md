### Post by sikander3786 on 2011-01-21
Might be you've got all the updates installed and no new updates were available lately...?

Anyways, go to Update Manager > Settings and make sure "Important Security Updates" and "Recommended Updates" are checked. Also, make sure Update Manager is configured to check for updates "Daily" and "Only notify about updates" is selected.
