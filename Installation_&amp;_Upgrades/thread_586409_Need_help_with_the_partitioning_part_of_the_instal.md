### Post by Jimlas53 on 2007-10-22
Hi Deletrius,
Take a look here:  [https://help.ubuntu.com/community/forum/installation/Partitioning?highlight=%28partitioning%29](https://help.ubuntu.com/community/forum/installation/Partitioning?highlight=%28partitioning%29)

for a short guide to setting up a new partition manually.
Yes, at a minimum you need a partition for /, which is the root directory (sort of like C: to Windows), a swap directory, and I would recommend a /home directory, which is where all your user stuff will go.  There are very good folks here on the forums, so ask and someone can help.

Hope this helps!
