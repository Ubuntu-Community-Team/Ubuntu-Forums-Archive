### Post by kmclar on 2008-04-20
I'm wondering if anyone else is seeing this?  With a fresh install of 8.04 RC (i386), I'm seeing vertical lines where the text is fuzzy, spaced every few inches across the screen.  Card and driver are nvidia.  This does not happen with 7.10, screen is sharp all the way across.  Solved!  I set the refresh rate from 50 to 51 under System>Preferences>Screen Resolution, and that fixed it!
