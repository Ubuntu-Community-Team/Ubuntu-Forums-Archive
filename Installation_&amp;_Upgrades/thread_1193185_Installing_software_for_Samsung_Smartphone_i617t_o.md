### Post by pytheas22 on 2009-06-21
The CD that came with your phone is not likely to work with Linux.  Hardware manufacturers rarely support Linux directly.  However, it may be possible to get it to work using a third-party application.

A quick Google search for "i617t ubuntu" doesn't turn up anything useful, which isn't a good sign, but if you could please connect the phone to your computer via USB, then open a terminal and post the output of this command:
```

lsusb
```

we might be able to find instructions on getting the device to connect to Ubuntu.
