### Post by eclectic on 2006-09-17
I just went through the same thing, but the solution is pretty straightforward.

As the previous posters said, when the first screen comes up, you're given several options. Press F4 and change the screen resolution to at least 1024 by 768. Then change the primary menu to option #2 - Safe Graphics Mode - then start the CD. Your resolution will be better, so double-click on the 'Install' icon -- you should be fine from there.

Mine installed properly, leaving me in my monitor's optimum 1280 by 1024 native resolution.

Hope this works for you.
