### Post by layers on 2010-10-19
Hello.

As far as I know, the Ubuntu CD is a Live CD - when you boot from it, the first option will be to try Ubuntu, without installing it (it shouldn't matter if it's a CD/DVD or a USB stick).

For the other problem - you not being able to boot up from a USB - there has to be an option in the BIOS settings. I think most computers now have it, but I would double check that, and also the order of boot-up devices - USB option should come before HDD boot.

### Post by drs305 on 2010-10-19
If you already have Grub 2 as your bootloader, you can also just boot the LiveCD ISO directly from the Grub menu. You don't even need to burn a CD if you don't want to. 

The ISO follows the standard Ubuntu conventions from what I remember, so the information in this thread applies:
[ISO Booting with Grub 2]("http://ubuntuforums.org/showthread.php?t=1549847")
