### Post by aysiu on 2006-11-24
No AmaroK in any Ubuntu version has MP3 support built in because Ubuntu does not include proprietary (closed source) codecs.

You can install MP3 suppport, though:
[https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)

I believe if you [**enable extra repositories**](http://www.psychocats.net/ubuntu/sources) and then try to play an MP3 in AmaroK, it'll prompt you to install the necessary codecs.
