### Post by ZeroYuy on 2007-11-07
> **gibsonman said:**
> Is there a simple way to install programs?

Either you can use Applications-> Add/remove.. Or, System-> Administration -> Synaptic package manager.

If you want to install programs not there, you will need to find a .deb file for it. Just sure deb file for -program- and you should come up with something.

The final, but not really simple way is to compile the source.

Edit: I forgot, the final way is sudo apt-get install -program- from the terminal. For instance,

sudo apt-get install ndisgtk

(Just did that randomly off hand)
