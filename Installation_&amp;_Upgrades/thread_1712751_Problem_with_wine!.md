### Post by Mark Phelps on 2011-03-23
You also need to be aware that Wine will NOT provide the same level of playing experience with Windows games as is available natively in MS Windows.  

Wine will only support DirectX9 -- and most recent games expect to be using v10 or v11.

So, you're going to get poorer performance in Wine.
