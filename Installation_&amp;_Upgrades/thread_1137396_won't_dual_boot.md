### Post by shload on 2009-04-26
Ok i figured out my problem.  I've got a second hard drive in the comp and though i installed on one hard drive ubuntu wanted to install grub on the other.  The option to change that comes at the very last step of the installation.

There's an "advanced" button to press before clicking the last button for install.  It is in the advanced option that i guess you can edit a couple grub options, like which hard drive to install to. 

Would have been nice for this kind of option to stand out a little more. ah well all fixed.
