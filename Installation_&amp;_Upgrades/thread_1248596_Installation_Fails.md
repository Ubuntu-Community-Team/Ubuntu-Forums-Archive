### Post by starcraft.man on 2009-08-24
Sounds like bad burn, do a cd integrity check and also check the iso against the hash. See here for details: [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

It is unlikely that a properly running cd would fail to run on two different PCs.
