### Post by Neo0351 on 2008-04-06
you can use envy, but since you already have the driver installed all you need to do is disable the nv driver
```
nano /etc/default/linux-restricted-modules-common
```

change the last line to like this

```
DISABLED_MODULES="nv"
```

### Post by sdennie on 2008-04-06
Ubuntu does some strange things with the nvidia drivers by default and so if you install the drivers from the nvidia page, when you next reboot, it can cause problems.  The suggestion to try envy is good but, if you decide not to go that route, doing something like:

```

dpkg -l | grep nvidia

```

And removing all the nvidia related packages will likely fix your problem as well.
