### Post by odelay on 2007-04-14
The problem has been resolved.  As linked to in this thread, and originally written by mikelito:

> cd /usr/X11R6/lib/X11/
sudo rm locale
sudo ln -s /usr/share/X11/XKeysymDB XKeysymDB
sudo ln -s /usr/share/X11/locale/ locale
Now go to your install directory and enter:
> ./Mathematica -NoSplashScreen
Should come up OK.
