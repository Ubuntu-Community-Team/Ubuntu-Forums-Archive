### Post by mingus on 2005-06-05
not an apt-get expert, but had a similar situation, perhaps this may help . . .

you are asked from which source for apt-get to install packages from.  be sure not to say network if you are not actually connected to network.  If you say "CD" it should complete package installation from the install CD (needs to be put back in drive) and then updates can be done later
