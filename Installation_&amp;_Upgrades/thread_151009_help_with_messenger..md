### Post by akiro.yamamoto on 2006-03-27
It's not exactly clear... but I assume you want to chat with a friend on the Yahoo network????
If your using the default Ubuntu install you should have Gaim installed.
Gaim is the GNOME messenger app. It supports AIM/AOL , MSN , Jabber/Google Talk and Yahoo Messenger.
Look for it under 
Applications >> Internet >> Gaim Instant Messenger
