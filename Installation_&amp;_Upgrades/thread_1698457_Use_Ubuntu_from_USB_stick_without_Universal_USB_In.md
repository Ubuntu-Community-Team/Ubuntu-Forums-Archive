### Post by usagiakumu on 2011-03-02
Ok I ran into a very weird issue with Universal USB Installer and Windows 7 x64, it would run until it came time to write said ISO to USB stick, and thats where things got sticky, it would crash out with some random error. Running in compatibility mode and Administrative mode don't solve this, so if you are like me the solution is simple:

Install Ubuntu via Wubi, then from the System Menu, choose to "create USB startup" and BAM works like a charm.
