### Post by tommcd on 2009-03-22
What version of Ubuntu are you using? If you are just trying to run the live CD to check it out, I would recommend trying the 8.10 32 bit live CD. If it still gives you trouble, try to boot with the "safe graphics mode" from the live CD menu.

How did you burn the CD? If you burned the CD from Windows I would recommend using Iso Recorder or Infra Recorder. 
[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)
And burn the CD at the slowest possible speed. But if you used the option "check CD for errors" from the live CD menu and it reported it was ok, then the disc is likely good. 
Can you post the errors you are getting when you try to run the live CD?
