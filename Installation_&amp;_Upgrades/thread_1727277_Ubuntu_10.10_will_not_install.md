Besides Lubuntu, which is an excellent option, you can install Ubuntu using the Ubuntu Minimal CD Image. Then add a basic window manager to get a graphical desktop. See this: [http://www.psychocats.net/ubuntu/minimal](http://www.psychocats.net/ubuntu/minimal)
