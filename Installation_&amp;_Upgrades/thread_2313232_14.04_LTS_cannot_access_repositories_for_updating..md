```

```

This will rebuild the cache.

### Post by grahammechanical on 2016-02-10
For those who do not know, if we install Ubuntu Restricted Extras from the Software Centre we will need to confirm our acceptance of the Microsoft End User Licence Agreement (EULA) for the installation of Microsoft TrueType fonts.

The dialog that appears may appear behind the software centre window. We will not see it unless we move the software centre window to one side. It will seem as if the installation has stalled.

This behaviour has caught me out a few times even though I know it is going to happen.

Regards.
