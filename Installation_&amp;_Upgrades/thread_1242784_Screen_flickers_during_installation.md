### Post by presence1960 on 2009-08-17
[boot options]("https://help.ubuntu.com/community/BootOptions") scroll down to F4 - you want safe graphics mode

if that does not work try the [alternate text based installer]("http://www.ubuntu.com/getubuntu/downloadmirrors#alternate")
