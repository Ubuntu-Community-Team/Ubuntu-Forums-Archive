### Post by saidinesh5 on 2009-07-16
i didnt exactly understand why he wants to boot into windows safemode to install ubuntu....
and you saying that he cannot boot using the CD..
this may be because :
1) the CD might be defective. So try checking it first on another PC.
2) the CD may not be the default boot device(and the windows drive still is active.) so try changing the "boot device order" in the BIOS(which you get after pressing F10 or Del or some other key immediately after you switch the PC on) and CD the first drive to be the CD drive.
