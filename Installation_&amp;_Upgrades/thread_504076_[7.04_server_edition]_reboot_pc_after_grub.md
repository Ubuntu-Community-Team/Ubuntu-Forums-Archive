### Post by oddish2211 on 2007-07-18
i just installed ubuntu server 7.04 on my old compaq pc,
it has no acpi, so i used the parameter "acpi=off" before the install.
i have installed ubuntu to hda1 and i setup grub on the MBR.
but now when i boot my fresh install for the first time, grub works fine but when i select the normal ubuntu and the recovery mode my pc gets rebooted.

it is a very old pc with these specs ;
amd k6-2 processor
160GB hd , 3 partitions , hda1 (ubuntu) hda2 (swap) hda3 (data, ~150GB)
cd/dvd player
2 WLAN cards 
ati rage videocard (dont know what sort of)

so i cant boot ubuntu, but i have the knoppix live cd, so i can edit files on the partition if i have to.

### Post by Pumalite on 2007-07-18
I'd bet is your card. Do you ever get to a command prompt? Try Crtl-Alt-F1, then dpkg-reconfigure xserver-xorg. Choose 'Generic Monitor' and 'vesa' as your driver. The rest; defaults are probably fine.
