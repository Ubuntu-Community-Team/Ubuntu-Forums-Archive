### Post by hbnmgr on 2012-07-09
In Synaptic Package Manager, select "SETTINGS" up at the top.

In "OTHER SOFTWARE" tab (must have disk in) and select "ADD VOLUME".

It will then read it, and make sure it is checked in the Other Software window.

Click CLOSE and RELOAD.  Look for what you want to install.
