### Post by Sef on 2008-09-16
> I want to reinstall hardy but I dont want to lose my home dir. Im in my live CD do i just install? Guided or manual? step by step from "prepare disk space" would be helpful thanks.

Do a manual install if you have a separate home partition.  The only partition you will need to delete and reformat is the root partition.  The swap will automatically be taken care of.  Just use the default file system (ext3.)  **Back up** your data first.  There are no guarantees that all will go 100% correct.
