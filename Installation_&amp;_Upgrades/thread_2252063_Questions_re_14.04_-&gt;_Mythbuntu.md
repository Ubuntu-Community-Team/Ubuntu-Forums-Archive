### Post by grahammechanical on 2014-11-09
There is a difference between MythTV and Mythbuntu.

MythTV is the program. Mythbuntu is an official flavour of Ubuntu that is purpose built to install Ubuntu with MythTV as the user interface. It will include all the bits and pieces that make up MythTV.

Mythbuntu is built on the latest versions of Ubuntu. It follows the release schedule of Ubuntu. So, we have Mythbuntu 14.04 but it is the underlying version of Ubuntu that is 14.04. MythTV will be at the development level that MythTV has reached by that time.

The packaging of MythTV for Ubuntu is a separate task from the development of Mythbuntu.

[http://www.mythbuntu.org/home](http://www.mythbuntu.org/home)

If you want to install MythTV on an existing install of Ubuntu, then install it from the software centre.
