### Post by invalid on 2005-11-18
Hi Dave,

From a fresh install, your root password is your initial users password - however, the root user does not exist for security reasons.

When it asks you for a root password, your user password will work here. You can always create a root user, if you feel better with this method, by
```
sudo passwd root
```and insert a new one here.

You can find more information on the sudo command if you are not too keen on it, in the Wiki if you need.

Hope this helps,
Cb

### Post by taurus on 2005-11-18
There is a root account on your system.  The problem is that the password for root has been scramble so if you want to log in as root, you need to assign a new password for it.  Are you sure you log in as the first user that you created?  Look in /etc/group to see if your login ID belongs to group adm!!!  You need to be in group adm if you want to use sudo...
