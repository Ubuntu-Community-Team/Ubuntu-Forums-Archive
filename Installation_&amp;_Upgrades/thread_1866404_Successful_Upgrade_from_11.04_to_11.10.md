### Post by Slim Odds on 2011-10-21
I just wanted to throw this post in so that it's not all failures being highlighted.

Smooth upgrade, system seems to be significantly faster (no empirical data to support this).

A note to some about upgrades in the past. Ubuntu does a pretty damn good job with this. I had a system that I installed 6.04 on and upgraded every release up to 10.04. That right, every six months. I never had any issues with the upgrades. The machine was acting as a home file/print server (though it was a desktop install).

Maybe I'm just the lucky one.
