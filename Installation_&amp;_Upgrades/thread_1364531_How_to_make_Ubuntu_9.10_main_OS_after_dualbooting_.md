### Post by raymondh on 2009-12-26
> **jhawk65 said:**
> 
I installed it via wubi installer and made a 11gb HD installation....I want to erase my Windows 7 Ultimate and make my Ubuntu installation my MAIN OS.


having installed Ubuntu thru wubi.....that means that Ubuntu "is a file inside windows".

If you erase windows, you erase all its' files .... Ubuntu included.

You can try to move Ubuntu outside windows.  Here's the wubiguide.  Sorry, I don't have any experience in this.

[https://wiki.ubuntu.com/WubiGuide#How](https://wiki.ubuntu.com/WubiGuide#How) do I migrate to a real partition, and/or get rid of Windows entirely?

Also in the wubiguide is how to make Ubuntu the default OS to boot.

[https://wiki.ubuntu.com/WubiGuide#How](https://wiki.ubuntu.com/WubiGuide#How) do I make Ubuntu the default boot option?

Lastly, this being a wubi-install .... the performance of Ubuntu will depend on the health of windows.  Keep win7 clean and defragged.
