### Post by Yoooder on 2007-06-20
I prefer the tutorial from pendrivelinux.com (just look for Ubuntu in the list towards the bottom of the page).  They break it down step-by-step and provide all the modified files you need.

The only note is the tutorial is setup for the 6.10 LiveCD, since the 7.04 LiveCD's persistence feature is broken.  Basically if you use 6.10 you can save your changes and manage it nearly like the full OS whereas using the 7.04 will behave like a CD where it will reset the settings at every reboot.
