### Post by linux4me on 2018-05-02
I tried the workaround described [here]("https://askubuntu.com/questions/961094/change-the-location-of-saved-screenshot-when-screenshot-taken-by-printscreen-but"), but finally just added Screenshot to my Favorites in Ubuntu. Now, if I only need a quick screenshot of the entire screen, I click the Print Screen button and go hunting in my ~/Pictures folder for the result, but if I want the whole menu from gnome-screenshot, I click the Screenshot icon in my Launcher.
