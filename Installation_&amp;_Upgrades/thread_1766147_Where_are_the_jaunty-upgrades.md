### Post by zvacet on 2011-05-24
Jaunty is not supported any more.That is a reason why you can not get updates.If you just started with Ubuntu and don't have any valuable files make fresh install of Natty.If you want you can upgrade Jaunty following [this.]("https://help.ubuntu.com/community/EOLUpgrades")Be aware that Karmic is not supported any more so you will have to continue upgrade to Lucid.This way is longer and time consuming,so it will be better to back up your files and make fresh install of Natty.
