### Post by Steve H on 2007-05-03
To be honest, there isn't much need for a virus scanner with Linux, unless it is an Email Server for a multiple OS network. There are Linux viruses but most never got out of the "Lab". So you can pretty much rest assured that you won't need one.

But if you still have your Windoze head on and you really would prefer a virus scanner then you would use the deb packages, Debian is the default packages for Ubuntu, and have the extension deb.

Hope that helps.
