### Post by darkod on 2012-01-07
With linux the problems are not that big when it remains without support as with windows, but you will probably miss out on a lot of software too. You would have to chase up the latest versions and install it yourself.

If you don't like Unity there is an option to use Gnome 3. That is little bit different from Gnome 2 but it's one option. I think you need to install the gnome 3 once, and after that at the login screen you can select whether you want to enter unity or gnome. It's even separate for the different users. One user can use unity and the other gnome, it doesn't need to be the same for all.
