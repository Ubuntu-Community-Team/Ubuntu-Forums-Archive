### Post by addicted_to_the_net on 2015-12-11
I posted this in the Ask Ubuntu site and thought I should put it here as well [http://askubuntu.com/questions/689068/kernel-panic-after-upgrade-to-ubuntu-15-10]("http://askubuntu.com/questions/689068/kernel-panic-after-upgrade-to-ubuntu-15-10")

I had to select the **wily-proposed** in software updates and then in a terminal:

 ```
   sudo apt-get update

    sudo apt-get upgrade
```

Rebooted and everything seems to be working fine now with Ubuntu 15.10 and the 4.2.0-21-generic kernel. Hope this helps someone!
