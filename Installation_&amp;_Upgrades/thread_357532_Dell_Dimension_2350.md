### Post by mpvano on 2007-05-10
Take a look at this bug report I filed about this issue - think it applies to similar machines as well (and I've definitely seen the problem with some Ubuntu releases on one of my own 2350s as well):

[https://bugs.launchpad.net/ubuntu/+source/usplash/+bug/109605](https://bugs.launchpad.net/ubuntu/+source/usplash/+bug/109605)

You can work around the problem as described there, but I've given up on 7,04 for now due to serious (and confirmed) kernel bugs - I'd wait until at least the first kernel update to install it if you need the machine working reliably. 7.04 looks good in many ways - I hope that the kernel problems are causing most of the trouble....
