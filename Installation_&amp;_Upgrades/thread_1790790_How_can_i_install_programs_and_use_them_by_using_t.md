### Post by bowens44 on 2011-06-25
That is a pretty broad question.
There is more then one way to install programs through the terminal. The easiest is to use the apt-get command. It really depends on the application and whether or not it's in the repositories.

To run an application , again it depends on the kind of application, but in most cases, just type the applications name and hit enter.

### Post by Rubi1200 on 2011-06-25
Hi and welcome to the forums Shadow5512 :)

This first link will give you some tips and tricks for using the terminal in general:
[https://help.ubuntu.com/community/UsingTheTerminal](https://help.ubuntu.com/community/UsingTheTerminal)

This link explains how to install software from the terminal:
[https://help.ubuntu.com/community/AptGet/Howto](https://help.ubuntu.com/community/AptGet/Howto)

Once something is installed, the general rule of thumb is that you can start the application in the terminal by simply invoking the name of the program e.g. firefox (remember that Linux is case sensitive).

If the program requires root privileges to run, the rule of thumb is to use sudo for non-graphical applications and gksudo for graphical programs.
