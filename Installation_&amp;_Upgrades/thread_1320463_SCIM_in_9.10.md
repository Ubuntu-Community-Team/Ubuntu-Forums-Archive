### Post by Irihapeti on 2009-11-09
The new default input method for 9.10 is iBus. You might like to try that and see if it does what you want. It comes with the m17n tables already installed. iBus settings are in the System -> Preferences menu (I think - I'm not in 9.10 at the moment).

However, if you want Scim, it needs to be installed. (iBus doesn't need to be removed.) You will need to install scim, scim-bridge-client-gtk and the language package you need. That can be done either through Synaptic or the terminal - your choice.

Details for setting up Scim in 9.10 are here: [http://ubuntuforums.org/showpost.php?p=8273690&postcount=3](http://ubuntuforums.org/showpost.php?p=8273690&postcount=3)

If you prefer using the terminal, you can use the method for earlier versions: [http://ubuntuforums.org/showpost.php?p=7956308&postcount=2](http://ubuntuforums.org/showpost.php?p=7956308&postcount=2)
