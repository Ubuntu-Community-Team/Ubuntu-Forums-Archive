### Post by taurus on 2006-07-06
If you want to install the 32bit, then you have to use the x86 version since AMD64 version is for 64bit!!!  Now, you have an option to download either the desktop (LiveCD and installer on the same CD), the alternate CD (an old fashion installer) or even the server version...

And Ubuntu is always free so don't worry about it.  The desktop will be supported for 3 years while the server will be for 5 years.
