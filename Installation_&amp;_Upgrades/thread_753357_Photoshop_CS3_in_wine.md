### Post by raashell on 2008-04-12
Up until recently, I ran Adobe CS3 Premium in a Virtual Box.  Everything ran smoothly, with the exception of Flash.  The reason that Flash didn't run as smoothly, was because Flash is a resource monster.  My computer is three years old, a 2ghz processer and 1.5 ram -- nothing special.

When you install Windows under Virtualbox, it asks you how much RAM do you want to give it?  I think I gave it 1gb and just had a few issues with Flash.  It wasn't uncommon for me to have Dreamweaver and Photoshop open at the same time with no problems.

The only reason I stopped using it was because I don't really need it. :)  I code most stuff in Emacs now and use the Gimp for photo editing.

One of the slow downs with virtualization was transferring files from my virtual windows to linux and vice versa.  You can't drag and drop or copy and paste between the two, you have to put files in a shared location to give and retrieve.
