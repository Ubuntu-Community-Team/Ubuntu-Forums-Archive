### Post by golovan on 2006-10-16
I've also experienced install hangup on a friends brand new pc today. The installer hanged on 15% several times. The solution for him was to use the option to manually edit partition table on the harddisk (use:/, /home, /swap), and then make sure both the partition list and the "use-for-root/home/swap" list on the next screen are the same (first / then /home and then swap). Maybe worth a try.
