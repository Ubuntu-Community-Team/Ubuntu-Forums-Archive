### Post by snowpine on 2013-02-13
It is recommended to install all updates, as they are security patches and bug fixes to the software you already have installed. (Not random new software you don't want/need.)

If it is a mission-critical system, then it is sensible to test the updates on an identical spare machine before applying the updates to your "production" computer.
