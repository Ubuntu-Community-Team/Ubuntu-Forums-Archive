### Post by MJDBatista on 2013-05-12
I could boot from a Ubuntu live CD (it seems the first one I used wasn't good) using "try Ubuntu" option and installed [Boot-repair]("https://help.ubuntu.com/community/Boot-Repair"). Then I've created a Bootinfo summary from which I understood my partitions where damaged. From the the "Try Ubuntu" session I've installed Ubuntu, creating the necessary partitions.
I've got my netbook back workinh properly.
