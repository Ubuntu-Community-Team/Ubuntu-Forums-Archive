### Post by winh8r on 2012-03-10
Alternative install CD's are generally used when the target machine either does not meet the specifications to run the LIVE cd or where the user requires a basic install which allows for further configuration of hardware etc in order for a fully functional desktop environment to be installed.


There are also minimal install cd's where the entire installation is very small and runs from the command line, the user then chooses which packages to install that are in keeping with either their requirements or the capabiliites of the machine


Hope this helps clear it up a bit.
