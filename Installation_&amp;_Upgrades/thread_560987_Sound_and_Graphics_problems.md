### Post by tony.morse on 2007-10-01
Ok after much messing about I've fixed these problems, so how did I do it...

Well I tried installing various drivers and so on as per the links mentioned, but in the end I discovered the problem was something else entirely.  So if you have a sound blaster Audigy and you get static instead of sound then this is what to do...

1. go to the sound options and make it display everything
2. select the switches tab
3. change the status of the digital/analogue switch

Thats it folks, by default it is set wrongly for the Audigy!!!
