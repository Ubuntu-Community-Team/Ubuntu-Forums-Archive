### Post by starcraft.man on 2009-07-08
Ok, at the boot screen (where it gives you the options to live or install) hit F4, and then select safe graphics mode. Then boot up live. This forces VESA, it should work. If not, you might want to check the media (i.e. run the cd check and do an md5 test on the ISO). Also, provide machine specs please with a reply if it persists.
