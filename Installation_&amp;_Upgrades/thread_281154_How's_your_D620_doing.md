### Post by cmaxo on 2006-10-30
I got this laptop a week ago and it is absolutely wonderful.  I used a linux rescue cd and ran qtparted to resize the windows partition and then installed Edgy.  

The wireless works right away, I installed NetworkManager though to  make things easier for going between all the wireless networks (school, home, work).

I have the C2D, WXGA+, Nvidia 110m, bluetooth, 9 cell w/ 6 cell media bay batteries, and the 100 GB 7200 rpm hard.

Standby works great if you follow the instructions posted by the testing team. [https://wiki.ubuntu.com/LaptopTestingTeam/DellLatitudeD620](https://wiki.ubuntu.com/LaptopTestingTeam/DellLatitudeD620)

As far as the screen brightness goes, I've had no problems.  This is the second laptop that I've owned and I don't notice any brightness issues.  At work I run a dual monitor setup through the port replicator but when I'm undocked I use the brightness keys on the keyboard and it looks really good.
