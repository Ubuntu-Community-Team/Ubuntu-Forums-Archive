### Post by Therion on 2009-06-16
That's a tough one.  I've seen this before and can only suggest you try a couple things.

1.  Wait.  My Karmic Koala install gave me a blank monitor for several seconds at one point during the install.  Just as I reached down for the Reset button things sprang to life and the install finished without issue.  What caused the monitor to blank-out like that I don't know, but if you haven't tried simply waiting it out for a few minutes, do so.  

2.  If still no joy, download the *Alternate Install CD* and try installing from that instead.
