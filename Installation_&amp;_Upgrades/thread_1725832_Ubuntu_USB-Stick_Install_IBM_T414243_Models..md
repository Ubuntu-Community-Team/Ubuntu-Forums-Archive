[FONT=Tahoma]This is for the IBM T41/42/43 Laptop Models.
But may work with other's aswell

[B]The USB-Stick Installation Freeze & not boot into Further Steps.

[/B]- Insert USB-Stick.
- Insert Any CD into the Case. (Needs any CD in the Case in order to boot to further mode on the USB)
- Boot From USB-Stick.
- And when you see the Ubuntu Screen it will continue to the installation steps.
- Once you enter the install Steps -> Take out the CD & continue installing from the USB.
- _So that Solves the Problem._. and the installation will be successful.

**Have Fun.!**
