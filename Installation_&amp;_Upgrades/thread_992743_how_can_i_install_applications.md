### Post by ndefontenay on 2008-11-25
Hi.

Also, you already have a bit torrent client installed after you've installed ubuntu.

It's called transmission. You should find it at the bottom of the internet group in the application menu.

If you want to install more programs, just go to add/remove programs.

You will have a huge list of apps you can pick, it will download and install for you. Vuze (old azureus) is part of it. You don't need to search for a name. Just search for torrent in the search bar. You should find a list of results.
