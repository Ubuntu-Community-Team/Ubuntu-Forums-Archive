### Post by bruenig on 2006-11-03
removing gnome-btdownload and ubuntu-desktop is no problem.

ubuntu-desktop is just a dummy package that calls all of the standard ubuntu apps when you install it. Uninstalling it is fine.

gnome-btdownload is the bittorrent client. If you are using bittorrent 5.0, then removing that shouldn't pose a problem either.
