### Post by oldfred on 2018-04-20
If Windows 10 pre-installed then it is UEFI and you need to install Ubuntu in UEFI boot mode.
How you boot install media is then how it installs, so boot flash drive in UEFI mode.

       Ubuntu UEFI install ISO
