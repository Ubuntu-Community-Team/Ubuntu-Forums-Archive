The first link indicates a Pentium M 755 (meaning that the second link for a 770 is irrelevant), which is a 32 bit processor without PAE. 

You can install Xubuntu 12.04.2 and use it as is or upgrade to a newer release using [Fake-PAE]("https://help.ubuntu.com/community/PAE").
