It's not needed to upgrade correctly, but I don't think you'll get all the new features.  For instance, the fast user switching applet is a new packages that's required by ubuntu-desktop, but I don't know or think it's required by anything else.  You can look at the differences in the requirements at packages.ubuntu.com and see what's changed.  There's no other way around it.  It would be nice if the had an ubuntu-desktop-new-features package that just had the new stuff, but I don't think they do.
