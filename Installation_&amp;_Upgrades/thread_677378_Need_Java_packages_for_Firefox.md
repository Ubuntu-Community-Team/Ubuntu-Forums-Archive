### Post by taurus on 2008-01-24
Probably best to download it from Sun's, the .bin file, from another computer, [http://java.sun.com/javase/downloads/?intcmp=1281](http://java.sun.com/javase/downloads/?intcmp=1281).  Unpack/Install it and then link the plugin to your ~/.mozilla/plugins or /usr/lib/firefox/plugins
