### Post by timetunnel on 2006-07-20
[LIST=1]
[*]Start Synaptic package manager, uninstall wine ;) 
[*]Go to [http://www.winehq.org/site/download-deb](http://www.winehq.org/site/download-deb)
[*]For older versions, read the bottom of that page
[*]Download the version you want, open Nautilus and double-click on the deb file. A deb-installer will start and install it.
[/LIST]
