### Post by draeath on 2008-01-12
He wants to install from a minimal CD that pulls everything from the mirrors as he goes. (usually the CD image would be 30-50mb in size, suitable for burning on physically smaller discs)

He's looking for the Ubuntu version of this: [http://www.debian.org/CD/netinst/](http://www.debian.org/CD/netinst/)


FOUND IT!
[https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)
