I don't recommend installing it to two separate HDDs for the record. Can cause trouble, it's much better in the long run IMO to put all your OSes on one HD and use others as simply storage discs. If you want to install both to the one with xp, see gparted documentation on their site ([link]("http://gparted.sourceforge.net/documentation.php")) and then use the live CD partition editor (system > admin > partition editor).
