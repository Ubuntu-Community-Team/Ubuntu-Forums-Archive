### Post by lindsay7 on 2009-05-24
wubi 9.04rc stands for release candidate. The regular 9.04 wubi should be out. I installed the rc version and there is very little difference with it vs. the standard version and the rc will update itself anyway.  From what I know the major difference between 8.10 and 9.04 is with ATI graphic cards. If you do not have one, there should be no problem. If you do it may work anyway. There is no way to tell until you install. I would go ahead and install 9.04 since sooner or later you are going to want to go to the new versions anyway. Since wubi puts Ubuntu under windows, if it does not install correctly you can delete it from the windows control panel and install 8.10.  Ubuntu installs in just a fraction of the time that windows does so you are not going to have to spend a lot of time putting 9.04 on and deleting if necessary... So just try 9.04 first.
