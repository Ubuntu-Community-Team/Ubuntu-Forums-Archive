### Post by dabl on 2011-05-14
Possibly the boot process, when detecting the keyboard, is not enabling the F keys.  You should have a "F Lock" key in the upper right corner (mine is US 105 key style).  In the boot process, as soon as you see the keyboard light flash, press the F Lock key, and then when the CD is booted, it should recognize the F keys.
