### Post by BenAshton24 on 2009-10-21
you wan't to install flashplugin-nonfree, if you have installed other packages related to flash then these might be conflicting with it... if so you can change this in about:config in firefox... or simply uninstall the other packages.

Hope this helps you,
Ben.
