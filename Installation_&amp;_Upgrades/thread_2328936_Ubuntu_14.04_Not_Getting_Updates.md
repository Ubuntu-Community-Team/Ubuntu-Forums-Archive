### Post by deadflowr on 2016-06-26
Look at the settings in Software and Updates. This is the name 14.04 + uses for the software sources.
Also, if you run a dash search for update manager, does it show a result for software updater?
Software updater is the new name for update manager in 14.04 +.

Or is it that when you do try to open and run it from the dash, or elsewhere it just acts like all is good?
