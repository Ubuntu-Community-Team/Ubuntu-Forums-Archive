### Post by tal007 on 2009-08-29
Before you attempt to install Ubuntu visit the Ubuntu site, read the installation requirements. If all these are met, then on with your mission.

First, make sure the iso image is transfered to your CD or DVD whatever you are using properly. Image burning is different from copying data on to your CD or DVD.

Second, make sure that your Hard drive has enough space on it. The space has to be unused, unoccupied by other operating system such Xp, Vista, .. etc.

Third, be patient. It takes some time for installation CD to bring up a screen on to your Desktop. It silently for some time examines your system and then after that if everything right it will present you a installation menu.

Give some time to see. If nothing happens, then you have to focus on some other things.
