### Post by ozhank on 2009-03-20
After several days of frustration, finally most things working from DVD with 8.10 from November last year. I have a Telstra Bigpond Sierra Air Card 880U, which I could not get configured to work with Network Manager. I used a wvdial.conf file as suggested in other posts, but with the entry telstra.bigpond and adding dns to test (check man wvdial.conf for information). Once connected, I could then update the packages. After restarting, Network Manager then detected the Air Card, which I then configured using the wizard, but changed the ipn to telstra.bigpond to reflect the account type I have. So far so good.
