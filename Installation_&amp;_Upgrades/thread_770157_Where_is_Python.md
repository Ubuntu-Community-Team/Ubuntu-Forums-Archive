### Post by ssam on 2008-04-27
python does not have a GUI (graphical user interface) that you launch.

it has an interactive mode that you start by opening a terminal and running the command
```
python
```
This is what you did when you got the >>> prompt.

if you want to learn to program there is a good sticky in the programming forum. [http://ubuntuforums.org/showthread.php?t=606009](http://ubuntuforums.org/showthread.php?t=606009)
