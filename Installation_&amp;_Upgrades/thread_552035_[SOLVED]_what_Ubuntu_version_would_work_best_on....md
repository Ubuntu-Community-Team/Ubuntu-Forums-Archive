### Post by manimal347 on 2008-02-04
No Ubuntu version will be passable. Perhaps you'll be able to use your computer reasonably if you do a base install with just Fluxbox and GTK 2.0 applications with no Gnome or KDE dependencies, but that will mean none of the Ubuntu experience, and lots of console applications. If you choose this, Debian Etch is probably a better choice, as base-installs are better documented for it. Even the CPU is a bit slow for XFCE, I think, despite how slim people claim the distribution and desktop environment is.

Try running the standard Xubuntu, but likely, you'll decide to spend some cash on another ram module (your laptop takes up to 512,) jumping to something like DSL or Puppy, or doing a base-install with a window manager like Fluxbox, Openbox, or well, there's got to be other really petite ones that are still updated yet not Ratpoison-absurd.
