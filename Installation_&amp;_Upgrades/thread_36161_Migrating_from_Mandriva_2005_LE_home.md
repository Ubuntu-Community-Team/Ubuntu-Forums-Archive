### Post by apollyonis on 2005-05-22
I was using Mandrake 10.1 Official before switching to Ubuntu and kept my /home partition. I wasn't using Gnome though, so I couldn't tell you whether it would keep any Gnome configs. All of my files were still accessible with no problems after I had kept the partition and switched, so I have no complaints.
