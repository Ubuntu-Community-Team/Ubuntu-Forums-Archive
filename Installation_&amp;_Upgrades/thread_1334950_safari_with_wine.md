### Post by julianb on 2009-11-23
In linux, my favorite browser is Google Chrome. Like Safari, it's fast and it is based on WebKit (most other open source browsers, like firefox, are based on Gecko). 

You can download a Chrome-for-linux installer (.deb file) here:
[http://dev.chromium.org/getting-involved/dev-channel](http://dev.chromium.org/getting-involved/dev-channel)

It's considered alpha software but I have experienced very few bugs. Far, far better than what you'd expect for an app running in Wine with a "bronze" rating.

Give the Chrome user interface a chance. It's different, but you can learn to like it if you don't already.
