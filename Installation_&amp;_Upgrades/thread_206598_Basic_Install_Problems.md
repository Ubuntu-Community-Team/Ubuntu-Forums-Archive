### Post by thingy on 2006-06-30
It could be that your computer is not configured to boot from CDs.

To change this, you would need to go into your computer's BIOS menu and change the boot order so that it tries booting from CD first, and then tries the hdd.

How you get into the system bios? Well it depends on your computer. Almost all computers, when they start up, *very* briefly show some information, such as "Press F1 to enter system bios" or system configuration.

So at that point if you hit the F1 key it would take you into the system bios.

The normal keys for accessing the bios are F1/F2/F10/F12 and the DEL key.

### Post by Brynster on 2006-06-30
Hello

Typically the instruction to boot from CD instead of HDD is in the motherboards BIOS 

To get into the bios usually involves the pressing of a specific key on the keyboard during the first few seconds of the boot process. The keys are usally either Delete, Backspace or one of the F keys, you'll need to check the motherboard handbook for specific instrustions. Or sometimes there is a prompt on screen during the initial boot process.

Once there navigate to the boot options for your motherboard and an set the first boot device to CD or Optical drive. save the settings and reboot with yuo live disc inserted and let it boot.
