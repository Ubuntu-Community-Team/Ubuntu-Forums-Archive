### Post by zanglang on 2007-07-26
I think for Gutsy related questions you would be better off asking in the Gutsy development subforum. ;)

1. It should be almost the same as Edgy to Feisty and so on, if you've done so before: you'll be prompted there has been a new version and if you'd like to upgrade. You don't have to reinstall nor reformat, all newer packages will be upgraded by apt, and you get to keep your partitions intact.

2. You can create a separate partition and set the mount point as /home.

Not sure about the rest, sorry. :)
