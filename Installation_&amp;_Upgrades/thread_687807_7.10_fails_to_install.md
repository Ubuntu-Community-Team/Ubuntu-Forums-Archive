### Post by Odrodzona-Sarmacja on 2008-02-04
The "/" partition needs minimum 4 Gb, so 5gb is very conservative specially as the temporary files will be on it too... I would assign about 9.5Gb of the first hd for root "/" partition.

The "swap" needs 256kbs, but even if folks give it entire 1gb, i still would assign 512kb for swap "/swap" partition on the end of the first hd.

The "/home" partition needs about 2 Gb, but 40 Gb for home "/home" partition would be logical, a very logical choice imho, specially if you plan to download something from internet, as these files will be stored in "/home/user/" directory :D

### Post by Odrodzona-Sarmacja on 2008-02-05
