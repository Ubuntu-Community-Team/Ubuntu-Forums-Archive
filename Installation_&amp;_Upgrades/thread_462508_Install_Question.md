### Post by merlinus on 2007-06-02
You will wipe out windows if you select "Use Entire Disk" (or something similar) when you get to the partitioning screen.

If you have enough disk space, you can select to install ubuntu in free space, and you will able to boot both it and windows.

And no, you cannot restore or install windows from an update disk, only from an original.

Good luck!
