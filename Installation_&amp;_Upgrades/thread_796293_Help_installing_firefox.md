### Post by hermes0710 on 2008-05-16

In the same app search for rar, zip and install them for compression utilities. You can also install 7-zip ([http://www.7-zip.org/download.html]("http://www.7-zip.org/download.html"))

### Post by Pumalite on 2008-05-16
You need tar if you are using Linux.
cd ~/Desktop
tar xvzf firefox-2.0.0.xx.tar.gz
cd firefox
./firefox
Make sure Firefox is closed when you are doing this.

### Post by AlanR8 on 2008-05-16
Briandr

Adept is a program that automatically downloads and installs programs to your machine. It is simple and easy, no need to go to sites and download .tar files etc if the file already exists in Adept. Adept "points" at the *buntu repositories or sources of software. 

Just open Adept, and in the search box type Firefox. A list will come up and all you need do is click the entry you want, eg firefox-2 that will install the current up to date stable version of Firefox, NOT the Beta 3 version that comes as default with Hardy. You can also uninstall programs here by right clicking the entry you want to remove and then selecting uninstall.
