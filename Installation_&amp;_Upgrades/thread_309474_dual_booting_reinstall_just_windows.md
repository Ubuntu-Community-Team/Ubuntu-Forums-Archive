### Post by Joe_CoT on 2006-11-29
1) Simply reinstall windows normally. Make sure you don't delete the ubuntu partition when installing Windows Xp
2) Afterwards, reinstall grub to access ubuntu from the boot menu. see this article (you'll want to read this **before** install Windows)

[Community Wiki How To: Recovering Ubuntu After Installing Windows](https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows)
