Next time please make sure you read over what you have written, as quite a lot of your post doesn't make any sense. I will interpret it as best I can. I am going to assume you meant that you are using Ubuntu 7.04 (Feisty Fawn).

To update firefox you can download the package from here:

[http://packages.ubuntu.com/feisty/i386/firefox/download](http://packages.ubuntu.com/feisty/i386/firefox/download)
