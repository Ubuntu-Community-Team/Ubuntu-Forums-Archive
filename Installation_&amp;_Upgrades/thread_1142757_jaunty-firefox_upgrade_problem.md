### Post by RwandaTim on 2009-04-30
I had a similar problem - could not view other tabs in Firefox and could not use mini search engine in top right of page.  I closed Firefox using File --> Quit.  Then I restarted Firefox.  While restarting, it brought up a dialogue box indicating that Firefox was updating addons to synchronize with my current version of Firefox.  Now no more problems.  Hope this helps.
