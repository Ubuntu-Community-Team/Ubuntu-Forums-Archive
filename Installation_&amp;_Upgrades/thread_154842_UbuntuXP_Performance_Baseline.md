### Post by Herman on 2006-04-04
If you want to really demonstrate the difference between Windows Xp and Ubuntu just try downloading a large map or any other large image and try scrolling around on it in Ubuntu and in Windows in the same computer.

I have a multi- booting computer with a 3 Ghz CPU and 512 Mb ram. I have a reasonably large .jpg map I downloaded, it's 3.2 Mb. (8363 x 7528 pixels).
In Ubuntu I can open it with 'Image Viewer', it takes around four seconds to open, but once it's open I can zoom in and pan around as fast as I can move my mouse, Ubuntu handles it like nothing at all.

Windows opens it faster with 'Windows Picture and Fax Viewer', but once it's open, the show's all over. Panning around is like cold molasses running uphill, a lot of freezing and thrashing goes on and the CPU fan sounds like the turbocharger on my CLR Mack towing three trailers up a hill.

I think that would be a typical type of comparison, Open Office .org takes a while to open in Ubuntu, and probabaly most other apps. Linux doesn't have the bad habit of loading stuff in the RAM all the time (tsrs: _T_erminate but _S_tay _R_esident in the RAM), so things take a few seconds to load, but once they are open, you have a lot more of the RAM available for use generally. 
We aren't comparing apples with apples. Poor old Windows goes down like a ton of bricks for my purposes, it's a loser.
