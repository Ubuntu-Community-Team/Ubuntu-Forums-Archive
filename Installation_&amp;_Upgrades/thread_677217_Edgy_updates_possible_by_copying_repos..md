### Post by ajgreeny on 2008-01-24
Get aptoncd from the repos and you can do exactly what you want; it copies and then adds the packages to the second machine's /var/cache/apt/archives folder so synaptic can then use them.  I use it often for syncing two machines without problems.  Aptoncd will also make an iso file which you can copy to a flash drive rather than mess around burning a CD each time.
