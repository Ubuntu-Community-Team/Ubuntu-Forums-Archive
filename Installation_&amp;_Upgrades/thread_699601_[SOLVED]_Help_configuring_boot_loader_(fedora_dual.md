### Post by danbyte on 2008-02-17
Thanks Guys,

I just change the Ubuntu section in /boot/grub/grub.conf to:


title Ubuntu (2.6.24.3)title Ubuntu (2.6.24.3)
        root (hd0,0)
        kernel /vmlinuz-2.6.24-3-generic ro root=/dev/sda4
        initrd /initrd.img-2.6.24-3-generic

Now I can boot ubuntu without problems.
