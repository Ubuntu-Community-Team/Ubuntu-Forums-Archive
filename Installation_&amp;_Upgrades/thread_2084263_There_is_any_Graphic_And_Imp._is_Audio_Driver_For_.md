### Post by Mark Phelps on 2012-11-15
Linux is NOT MS Windows -- you don't go out hunting down drivers for it.

Ubuntu scans the hardware, downloads the proper drivers, and installs them as part of initial setup.

If there are restricted drivers (non-open-source) available, they will be offered via Additional Drivers, after installation is done.
