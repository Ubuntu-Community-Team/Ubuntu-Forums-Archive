### Post by Craigus on 2007-11-30
> I am not sure if this is a known issue

If you think about it, this cannot possibly be due to a known issue with any OS you may be running, because your machine (for whatever reason) is not booting from the XP CD, ie before any OS is loaded.

This is either an incorrect setting in the BIOS, or a problem with the optical drive or the XP CD.

Does the optical drive appear in the BIOS drive list?

Are you sure that the optical drive is the first boot drive?

Will the machine boot from your Ubuntu CD ?
