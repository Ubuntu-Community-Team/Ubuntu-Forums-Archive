### Post by grahammechanical on 2013-08-24
Ubuntu 12.04 is a Long Term Support (LTS) version. You will not be notified of an upgrade to Ubuntu until the next Ubuntu LTS version is released and that will be towards the end of April 2014. I really do suggest that you wait until Ubuntu 2014 is released.

Here is why. When we use an LTS we can simpy upgrade to the next LTS but if we have an interim or standard release we have to upgrade to the next standard release and then the next one and so on.

So, you will have to upgrade to 12.10 and then to 13.04 to get up to date. It is a lot upgrading. But 13.04 only has nine months support and that is not enough to get you to 14.04. So, to keep using a supported version you will have to upgrade at some point to 13.10 and then to 14.04 to get the next LTS release.

To do it you open Software Sources in System Settings and open the Updates tab and change the setting for the panel Notify me of a new Ubuntu version from For Long Term Support Versions to For Any New Version and then check for updates and you will get a button that will invite you to upgrade to 12.10.

But remember, once you go to 12.10 you can only get to 14.04 by upgrading to 13.04 and then 13.10 and then 14.04. Or re-install with 14.04 when it is released. We have 8 months to wait for 14.04.

I can tell you this about Calligra Author, it is in the Ubuntu Software Centre of Ubuntu 13.10 as is Calligra Suite. I am testing the development version of 13.10 right now and it is in the Software Centre for installing on Ubuntu. It most probably is also in the Ubuntu 13.04 Software Centre as well.
