### Post by Greyowl750 on 2012-04-30
I had the same problem, try this...
start your machine set to boot  from your flashdrive.
when you get the error do this:
1) hit your "tab" key
2) type "live"
3) hit "enter"

it will load a live version of Ubuntu...from there you should be able to use the installer icon on your screen.

Good luck.
