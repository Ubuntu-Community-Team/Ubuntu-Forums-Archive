### Post by Mighty_Joe on 2009-06-01
I had problems using the USB Live CD Creator to create a bootable USB drive.  Probably has something to do with [this bug]("https://bugs.launchpad.net/ubuntu/+source/usb-creator/+bug/321544"). It probably wouldn't effect you if you used a small persistent partition (less that 2Gb).
I used the instructions [here]("http://beastie.cs.ua.edu/cs150/configuration.html") to install Ubuntu to a flash drive.  Works pretty well.
