### Post by kwrxxx on 2006-10-27
I downloaded and burned the 6.10 ISO on a newer DVD/CD writer and used the lowest settings which was 8x , but on my old dell 1.8 GHz machine CD-ROM numerous errors occurred like bad CD notice, freezing of the kernel and panics. I obtained a older CD writer and burned the ISO at 2x the install and live CD now work properly. This is something to look at if you have problems booting the 6.10 CD. BTW the 6.06 CD ISO that was burned using the DVD/CD writer worked flawlessly on the old Dells CD-ROM, unlike 6.10 edgy
