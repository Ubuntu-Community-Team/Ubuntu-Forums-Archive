### Post by blackened on 2009-01-28
Might just be a bad burn. If you downloaded it through Bittorrent, then try burning it at a really slow speed (2x-4x). If you downloaded via http or ftp, make sure you check the md5. If the downloaded data is corrupt, then every disk you burn will be bad. Once checked, burn it as I mentioned before: slowly.
