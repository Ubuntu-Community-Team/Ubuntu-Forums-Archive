### Post by howefield on 2015-03-23
Open up your address book and from the tools menu choose the export option, you can then import the resulting file back into the clean install.

---

### Post by ajgreeny on 2015-03-23
Everything for the configuration of TB is contained in a hidden **~/.thunderbird** folder in your home and within that the addressbook is all in a file called **abook.mab**.

Save that hidden folder and the abook.mab file and you have everything.

### Post by oldfred on 2015-03-23
I save the entire profile of both thunderbird and firefox.
When I was dual booting I copied profiles to a shared NTFS data partition so XP and Linux could use same profile. 
And I still copy entire profile to my laptop when I travel & copy it back to desktop when I return.

If you houseclean emails, it is not huge, but if you have lots of email with large attachments it may be a lot larger. I typically houseclean, save attachments I want separately and remove all old or very large emails. I also backup entire profiles periodically.
