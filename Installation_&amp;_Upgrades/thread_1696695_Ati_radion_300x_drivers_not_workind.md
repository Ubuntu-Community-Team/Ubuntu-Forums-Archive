### Post by Mark Phelps on 2011-02-28
If you're seeing a desktop, you already have the drivers installed.

AMD (who bought out ATI) dropped support for restricted Linux drivers for your card years ago.

The open-source drivers, which are installed by default during system setup, are the only ones available now.

There are no Linux drivers you can download to replace the ones you already have.
