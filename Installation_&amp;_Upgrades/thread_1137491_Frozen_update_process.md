### Post by cfchagas on 2009-04-28
After leaving the computer on for 3 days waiting for a reply, I decided to restart it. Should anything go wrong - e.g., no boot -, I'd just re-install version 8.10.
 
To my surprise, the system seemed to work fine on reentry. Update Manager showed there was an interrupted update and I had it restarted. It went on flawlessly all the way to the end. After cleanup and reboot, 9.04 is up and running.
