Finally, I have successfully installed Oracle 10g on my Ubuntu computer, just by using the .deb file "oracle-xe-universal_10.2.0. 1-1.0_i386.deb" downloaded from Oracle site: "http://www.oracle.com/technetwork/products/express-edition/downloads/102xelinsoft-102048.html". :)
