### Post by zvacet on 2008-09-30
You need to make space for Windows.That means if you have ubuntu on all your HDD you have to shrink Ubuntu.You can do it with [Gparted live CD.]("http://gparted.sourceforge.net/download.php") When you shrink Ubuntu you can leave unallocated space and on that space install Windows.Of course you will need boot Windows CD.After you install Windows you will probably want to reinstall grub.You can follow [this]("http://ubuntuforums.org/showthread.php?t=224351&highlight=reinstall+grub") guide.
