### Post by amohanty on 2005-07-18
To run any app:
1. Open up a terminal:
Applications->System Tools->Terminal
2. Type in the first few letters of the app and then press [TAB]. This will show you all the apps whose first name begins with the set of letters you just types, sort of like command completion.
3. Type in the full name and press enter.

To edit menus: look for smeg
