### Post by Baji on 2007-04-09
as a quick solution you should check you fstab for entries that dont use UUID=.....

replace them with the uuid, you can get those with the vol_id command

example:

in /etc/fstab:
```

/dev/hda1   /    ext3    defaults,errors=remount-ro 0       1
```

get UUID:
```
sudo vol_id /dev/sda1 | grep UUID
```
result:
```
ID_FS_UUID=ce643186-5f2c-4efd-9c0e-7d8f8519221f

```

replace in fstab
```

UUID=ce643186-5f2c-4efd-9c0e-7d8f8519221f   /    ext3    defaults,errors=remount-ro 0       1
```
