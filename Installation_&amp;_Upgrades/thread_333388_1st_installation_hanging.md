### Post by meng on 2007-01-07
You shouldn't need to pre-format or pre-partition your hard drive.
It's still possible that you have a badly burned CD.
In cases like this, I generally recommend using the Alternate CD rather than the Live/Desktop CD. However, this would mean that you won't get the chance to test drive Ubuntu before you install it.
