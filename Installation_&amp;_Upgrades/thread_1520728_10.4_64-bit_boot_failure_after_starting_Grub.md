### Post by dabl on 2010-06-30
> **twgray said:**
> 

Boot from (hd0,0) xfs
Starting up...



Are you running on an XFS root filesystem?  Do you have the kernel image in a separate (non-XFS) /boot partition?

Check the "Disadvantages" here:  [https://wiki.ubuntu.com/XFS](https://wiki.ubuntu.com/XFS)

Personally, I watched my XFS filesystem slowly deteriorate when I tried it as a root filesystem a couple years ago -- it lasted about 4 months before things got too flaky to tolerate.  It seems fine for data storage partitions, however.
