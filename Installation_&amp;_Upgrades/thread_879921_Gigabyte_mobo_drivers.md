### Post by Fire_Chief on 2008-08-04
Can you specify what components on the motherboard you need drivers for? Ubuntu should already have drivers for most of the basics (drive controller, chipset, graphics, network, sound). The configuration of one or two items may need to be tweaked to work properly but I'd be surprised if you actually needed to download drivers for the motherboard.

Regardless, you should first see what hardware Ubuntu is not recognizing. Then check with the manufacturer (Gigabyte) to see if the provide a linux driver for that component (i.e. on-board network card).
