### Post by Partyboi2 on 2009-05-11
Hi, looks like you need to install the libxml-libxml-perl package

```
sudo apt-get install libxml-libxml-perl
```There is also a older version of gcstar in the ubuntu repos if you would prefer all the dependencies taken care of
```
sudo apt-get install gcstar
```
