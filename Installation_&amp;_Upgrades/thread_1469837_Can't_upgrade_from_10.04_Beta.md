You are doing noting wrong, you just have a misconception. If you do all the updates to a "beta" version of the installation media, you have the release. The release candidate isnt a separate version. The version numbers dont change. 10.04 is still 10.04. A release is just the version with the bugs fixed to the point it is released. The bugs are fixed by the newest packages.
