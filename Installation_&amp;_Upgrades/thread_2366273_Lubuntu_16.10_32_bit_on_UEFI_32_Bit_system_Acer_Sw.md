### Post by Bucky Ball on 2017-07-15
Welcome. Wouldn't bother with 16.10. It reaches end-of-life in a week or two at which time you will need to upgrade to a supported release.

Would advise Lubuntu 16.04 LTS, a long-term support release with support until 2019. Try that and post again with full details of the system you are trying to install on, whether you are dual-booting or installing Ubuntu only, etc. and why you are using Rufus if you have issues. Good luck. ;)

(PS: You chose 32bit Lubuntu so presume you are using a 32bit system? If not, go 64bit.)
