### Post by tugboat on 2006-07-30
Update: I read some other topics and inferred that with the LiveCD running, if you leave the pc running during install without mucking about with it, the screensaver can try to execute.  I started the install over again, and click-and-dragged the window a few inches every 5 minutes or so, and the install went without a hitch.
