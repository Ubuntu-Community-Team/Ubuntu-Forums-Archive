### Post by taurus on 2007-11-22
You can install XFce4 (Xubuntu) from Gnome (Ubuntu) with

```
sudo apt-get update
sudo apt-get install xubuntu-desktop
```
Then, you can pick whichever window manager you want to use from the Sessions (bottom left) in the GUI login screen.
