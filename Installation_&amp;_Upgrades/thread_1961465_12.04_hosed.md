### Post by IWantFroyo on 2012-04-19
Hit Ctrl-Alt-F1 and sign in. Then, run:
```
sudo apt-get install lubuntu-desktop
```

This will install a different desktop (LXDE), which you can then sign into after rebooting. Hopefully you'll get your window manager and a panel.

If this doesn't work then I would hit Ctrl-Alt-F1 again, sign in, and then just use elinks, cp or ftp to move all your files off your computer. If you're using a standard email client then their should be a Mail or .mail folder in your ~/
I would plug in a USB drive and ```
cp -R Mail /media/nameofusb
```. You can use ls to determine what's in the folders, like the name of your USB.
