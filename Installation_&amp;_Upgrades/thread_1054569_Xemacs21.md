### Post by Partyboi2 on 2009-01-29
You can start it from the terminal by typing
```
 xemacs21
```
To add it to your Applications menu go to System>Pref>Main Menu and highlight the menu you want it under on the left then select  "New Item" and add
Name : xemacs21
Comamnd:  xemacs21
Comment: Leave blank or add what ever.
