### Post by snowpine on 2009-12-23
Hi gcmartin, it is recommended to perform a fresh reinstall every 6 months when a new Ubuntu is released. Having a separate /home partition will facilitate this... you can install the new release on the / partition while keeping user settings and documents intact on /home.
