### Post by dino99 on 2011-05-02
several ways to try:

- first reboot in recovery mode and select "repair packages": recovery mode is the second line from grub menu; if you dont see grub menu (which is the default on single OS system), hold "shift" key down at bios end process to see the grub menu

- if your system still able to boot, then open a terminal and run:
 sudo apt-get install ubuntu-desktop

that should reinstall all the removed packages, but i'm not sure, so run: 
 sudo apt-get update
 sudo apt-get install -f
 sudo dpkg --configure -a

- if none of the previous choices work, then better to make a clean install again
[http://ubuntuforums.org/showpost.php?p=10161428&postcount=2](http://ubuntuforums.org/showpost.php?p=10161428&postcount=2)
