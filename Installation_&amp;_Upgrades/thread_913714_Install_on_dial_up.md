### Post by Irihapeti on 2008-09-08
First of all, you will need to check that their modem is capable of working under Linux at a reasonable speed. There are instructions somewhere in the community documentation about how to do this. Or, depending on where you live, you might be able to pick up an external serial (not USB) modem that should work.

I've run Ubuntu for over a year now on a 56K dialup connection, so it can certainly be done. If I am doing large downloads, and that includes updates, I usually run a script that does the downloads at night and disconnects when finished. There's an option in Synaptic to create a download script instead of installing packages directly. I use it quite a lot. You can also use the script on another machine with a faster connection.

When it comes to upgrading, I wait until the live CD image is offered with a PC magazine. That also gives time for a lot of the early bugs to get ironed out. The downside is that there could be 200 MB of updates, but they can be spread out over several days.

Hope that helps.

Irihapeti
