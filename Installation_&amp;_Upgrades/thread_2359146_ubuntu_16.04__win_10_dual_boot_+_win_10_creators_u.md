### Post by westie457 on 2017-04-21
Hi
Connect your ubuntu drive. Boot your computer using the install media to Try-mode.
You should be able to check your files and back them up.
Also you might be able to fix everything with Boot-Repair (2nd option) available here. [https://help.ubuntu.com/community/Boot-Repair](https://help.ubuntu.com/community/Boot-Repair)

Before attempting any repairs click on 'Create Report', when this has finished you will be given a link to post for help. Post that link so we can offer suggestions .
