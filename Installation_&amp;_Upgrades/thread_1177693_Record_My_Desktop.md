### Post by cherva on 2009-06-04
Yes recordmydesktop is a command line tool. It won't show in the menus. There are two major GUIs for it and there are :
- gtk-recordmydesktop - witch is a GNOME based GUI
- qt-recordmydesktop - witch is a QT based (KDE) GUI.
When you install one of those two they will come up in the menus.
