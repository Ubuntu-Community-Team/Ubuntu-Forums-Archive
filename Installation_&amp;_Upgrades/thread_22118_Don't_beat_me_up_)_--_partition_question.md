Don't worry about the "boot" project option. I messed myself up in Nero using a similar option.

If the .iso is bootable, then the disc you burn with it will be bootable. What you have to do is burn the .iso in question as an image.

I ended up using a simple freeware program called [burnatonce](http://www.burnatonce.com/) to burn my Ubuntu .iso . It works quite well, and is far more straightforward than the multitudinous options offered by some burning suites.

Good luck!!
