### Post by ajgreeny on 2012-11-06
What version of hplip do you have installed?

I believe that the default hplip version in 11.10 (hplip 3.11.7) is too old for that printer, which looks as if it needs hplip 3.11.10 or later.  

The info is at [http://hplipopensource.com/hplip-web/models/photosmart/photosmart_5510_series.html](http://hplipopensource.com/hplip-web/models/photosmart/photosmart_5510_series.html)

You can get the later version from [http://hplipopensource.com/hplip-web/gethplip.html](http://hplipopensource.com/hplip-web/gethplip.html) 

How to install it at [http://hplipopensource.com/hplip-web/install/manual/distros/ubuntu.html](http://hplipopensource.com/hplip-web/install/manual/distros/ubuntu.html)
