### Post by 73ckn797 on 2008-09-21
Have you used the built-in software sources?

Go to: Applications, Add/remove and search for whatever.

Go to: System, Administration, Synaptic Package Manager and search for whatever.

Check the "Settings", "Repositories" and check all sources to be accessed.

Both of these will, once software is selected, install for you.
