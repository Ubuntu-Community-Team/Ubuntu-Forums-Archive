### Post by Mark Phelps on 2014-09-26
> is it possible to replace it with Windows 7 

Might NOT be ... possible, that is.  You need to see if there are Windows 7 drivers for your desktop -- BEFORE attempting to replace Win8.1 with Win7.  MS is discouraging OEMs from providing Win7 drivers for machines they sell preloaded with Win8.

Also, if what you really dislike about Win8.1 is the "tiles interface", then look into using one of the Start Menu utilities and booting to the Desktop, instead of the Start screen.  I've been doing this since Win8 came out (as I see NO VALUE in having a touchscreen interface on my desktop).  Check out the Windows 8 forum for details: eightforums.com.
