### Post by TenPlus1 on 2008-04-19
If you right-click on a space in the top panel and select Add to Panel from the menu, it will bring up a window containing add-on's from which you select Workspace Switcher then click Add...  This will make it appear on your top bar so that you can select your workspace, and by right-clicking on it you can setup how many desktops you want also...
