### Post by mikewhatever on 2007-02-26
Once you have it on CD, it's still too early for commands. You should boot your PC from that CD (restart with that CD in the CD-Rom drive). Then, it goes trough the process of loading up, and a live session starts. That means Ubuntu runs from RAM and the CD, no installation done yet. If everything works fine and you are ready and all, click the install icon on the desktop. All of it, including the installation and even commands, is nicely outlined here:
[http://psychocats.net/ubuntu/installing](http://psychocats.net/ubuntu/installing)
