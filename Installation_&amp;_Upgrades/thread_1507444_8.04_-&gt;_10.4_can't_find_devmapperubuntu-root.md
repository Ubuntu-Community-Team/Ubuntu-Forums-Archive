### Post by ronparent on 2010-06-13
If the system had an actual software raid there would not be a /dev/mapper/. That would be found only if dmraid discovered meta data on drives for a MB bios implemented raid set also known as fakeraid. I believe for a software raid you would be looking for something like /dev/md/.
