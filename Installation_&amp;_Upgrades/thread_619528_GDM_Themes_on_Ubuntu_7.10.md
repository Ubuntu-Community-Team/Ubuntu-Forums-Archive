### Post by Dark Hornet on 2007-11-21
Hello...from my experience, usually Ubuntu will use Metacity as the default theme...just download it, and then go to "System, Appearance", and select theme and click on Install---it will install the file that you downloaded.  If you want to use any theme that has a "gtk", or "gdm"  in it you need to download a special "engine", then you can use these.  Personally, I use Metacity.  I hope I have directed you to the correct places--if I am off a bit, sorry, but I am not in front of my Ubuntu box right now, I am at work.   Good Luck!
