### Post by deadflowr on 2022-03-26
There is no root login on Ubuntu.
Use sudo to run root level commands.
See: [https://help.ubuntu.com/community/RootSudo]("https://help.ubuntu.com/community/RootSudo")
for more on root and sudo.
