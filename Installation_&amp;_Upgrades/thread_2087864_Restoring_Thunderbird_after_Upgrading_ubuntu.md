### Post by Frogs Hair on 2012-11-24
Saving the .thunderbird works only if everything is identical. That would include Tbird version and add-ons. It is important to replace the folder prior to opening Tbird on the new installation.

Under Edit > Account Settings Sever Settings it is possible to get the information for each account and it is possible to browse the path to the profile folder which may be important if trying to import.
