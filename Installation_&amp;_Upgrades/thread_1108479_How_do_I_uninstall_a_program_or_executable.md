### Post by upchucky on 2009-03-27
assuming you used the package manager to install it go to,
systems--> administrator--> synaptic package manager

find the package in the list and mark it for removal  then remove it.

or alternately open a terminal, -->applications--->accessories--->terminal

and type

```
sudo apt-get remove xaimer
```
