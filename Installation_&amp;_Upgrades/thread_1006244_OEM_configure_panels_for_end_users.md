### Post by mk1w86 on 2008-12-09
You can run an OEM install of Ubuntu by pressing F4 at the boot screen of the installation disk.

The first time the machine boots it should login and let you change any settings you want and when it next boots it will start the setup wizard to let customers set up accounts and passwords etc. :)
