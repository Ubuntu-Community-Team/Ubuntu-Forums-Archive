The problem climaxed with the message: "The resize operation is impossible because of an unknown reason. it is impossible to resize the partition"!

I then rebooted 98SE with bootable floppy and did "fdisk" where I saw this: Partition 1, Status A, Type "NON-DOS!!!!!! My C: drive was non-dos!

So, remembering tht I was running a program called "GoBack". Which creates a special HD set up to record the system so that I can "revert back" to exactly the way my system was, if there were any problems with virus's or after loading in funky software. So I un-installed "GoBack on the 98SE drive, rechecked the partition with Fdisk and lo and behold I now had a partition type that was now DOS!

I rebooted Ubuntu and the rest ihistory as they say! :) 

Lastly, if you are running GoBack on Windows OS, uninstall it first!
