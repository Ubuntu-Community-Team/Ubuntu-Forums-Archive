### Post by Rubi1200 on 2011-09-17
You can resize the Wubi virtual disk following this guide:
[http://ubuntuforums.org/showthread.php?t=1625371](http://ubuntuforums.org/showthread.php?t=1625371)

At the same time, I recommend you also make a backup copy of the root.disk and keep it somewhere safe like an external medium.

You may also want to consider saving the files you are using in Ubuntu somewhere else as that will also save space.

The final option is to migrate the Wubi install to a dedicated partition, especially if you are using it regularly:
[http://ubuntuforums.org/showthread.php?t=1519354](http://ubuntuforums.org/showthread.php?t=1519354)
