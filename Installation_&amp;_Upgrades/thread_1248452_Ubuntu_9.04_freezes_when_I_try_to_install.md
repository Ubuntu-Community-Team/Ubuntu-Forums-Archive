### Post by presence1960 on 2009-08-24
> **savataje said:**
> Ubuntu 9,04 freezes at 73 percent what do i do please help!

1. [MD5SUM]("https://help.ubuntu.com/community/HowToMD5SUM") your iso.

2. Burn the iso as an image to CD at no more than 4x speed. if you have a CD/DVD burning program that won't allow you to choose speed see [here]("https://help.ubuntu.com/9.04/switching/installing-burning.html")

3. Boot the Live CD and choose "Check disk for defects" prior to installing.

4. Try the [alternate text based installer]("http://www.ubuntu.com/getubuntu/downloadmirrors#alternate")
