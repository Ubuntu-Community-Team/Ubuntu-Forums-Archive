### Post by Partyboi2 on 2008-07-25
If you have internet connection you can install the package by opening a terminal (Applications>Accessories>Terminal)
and typing
```
sudo apt-get install libssl-dev
```Another thing you could try is to install the deb by the terminal
If the deb file is on your Desktop open a terminal and type
```
cd ~/Desktop
``` then
```
sudo dpkg -i package
```*Change package to name of package.
eg
```
sudo dpkg -i libssl-dev_0.9.8g-4ubuntu3.3_i386.deb
```
