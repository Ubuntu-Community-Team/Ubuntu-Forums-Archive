### Post by woedend on 2006-11-08
make sure you have the libc-dev package installed(just search for libc in synaptic, im not sure what its exact name is).  Also, the kernel headers, gcc, g++(i think this all comes in build essential)

### Post by FrodoB on 2006-11-08
Actually g++ was not installed and installing it resolves the issue. 

sudo apt-get install g++

Thanks for sending me in the right direction.

FrodoB
