### Post by tommcd on 2009-09-22
Try importing the GPG key for the repositories. See this:
[http://ubuntuforums.org/showthread.php?t=1238623](http://ubuntuforums.org/showthread.php?t=1238623)
Be sure to replace the EF4186FE247510BE code in that example with the code in your error message.
Perhaps the GPG key server is just down, or a little slow. Try it again later.

If this does not work then try a different mirror.

And welcome to the Ubuntu forums!
