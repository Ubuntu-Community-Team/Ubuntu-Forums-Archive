### Post by bapoumba on 2016-05-15
The repositories you are talking about have long been disabled. medibuntu does not exist any more, all is in the ubuntu-restricted-extras now. For google, I'm not sure. I would fresh install, maybe with a lighter flavor such as lubuntu.

Edit to be thorough : the older release are here : [http://old-releases.ubuntu.com/releases/](http://old-releases.ubuntu.com/releases/), but I would not try to upgrade from 8.10.
