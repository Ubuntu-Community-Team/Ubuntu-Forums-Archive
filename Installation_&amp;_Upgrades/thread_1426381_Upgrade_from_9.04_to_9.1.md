### Post by zvacet on 2010-03-10
You didn't do anything wrong.If you are using Gnome and you do the upgrade you will still use Gnome (or any other desktop environment you had on previous version).Upgrade doesn't change desktop it just get you newer software.If you want to install KDE type in terminal

```
sudo apt-get install kubuntu-desktop
```

and if you want to remove Gnome and install KDE read [this.]("http://www.psychocats.net/ubuntu/purekde")
