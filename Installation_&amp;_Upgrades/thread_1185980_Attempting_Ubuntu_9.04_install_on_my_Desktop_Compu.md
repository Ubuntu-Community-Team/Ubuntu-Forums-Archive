### Post by nvikram on 2009-10-15
You can download the beta of Ubuntu 9.10 here [http://www.ubuntu.com/testing/karmic/beta](http://www.ubuntu.com/testing/karmic/beta)

If you want to install wubi, just burn the ISO to a CD, like you normally would, then pop it in WHEN windows is running, the Wubi AutoRun should come up on its own and then all you have to do is follow instructions.

On a side note, I believe you have to install some nvidia drivers AFTER installation for a dual-monitor setup to work.
