### Post by gabello on 2008-12-23
Hello, I was wondering if there is a way to tell Update Manger to stop offering updates for a given program or at least make it to no longer offer an existing upgrade?
I know I can add/choose what categories of updates I want to receive, but I don't know if I can remove a given software update from the list so the Update Manager no longer offer it.
thank you

---

### Post by Partyboi2 on 2008-12-23
You can lock a package version in synaptic (System>Admin>Synaptic) so that it won't get upgraded, all you need to do is highlight the package you don't want upgraded and select "package" then "lock version".
