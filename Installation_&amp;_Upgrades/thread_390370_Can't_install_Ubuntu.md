### Post by rsambuca on 2007-03-21
256 MB of Ram is getting a little bit tight to be running a live CD, since everything has to run out of ram.  If you are serious about installing onto your Hard Drive, I would recommend you use the xubuntu alternate CD as Xubuntu will work a little better with your specs, and the alternate CD will be easier to install with since you don't have to run the live CD environment.

You can get it [[COLOR="Red"]here.[/COLOR]]("http://www.xubuntu.org/get#edgy")
