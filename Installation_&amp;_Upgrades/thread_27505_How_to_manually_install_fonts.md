### Post by ssam on 2005-04-16
the easiest way as a user to make a folder in you home directory called .fonts 

the . at the front of the file name makes the folder hidden. so you will have to turn on the show hidden files option in the view menu (of nautilus).

then make a new folder called .fonts (if its not there already)

now drop you fonts into this folder.

these font will be avaible next time you open a program. you might need to log out and in to make the avaible to gnome.

sam
