### Post by snowpine on 2010-10-04
You can't skip releases. You need to go to 10.04 first, and then on to 10.10 (when it is released).

Some users prefer to do a complete reinstall of each new release, for that clean fresh feeling. :)

All of this is well documented: [https://help.ubuntu.com/community/UpgradeNotes](https://help.ubuntu.com/community/UpgradeNotes)
