### Post by engine on 2011-01-15
Start by trying ```
which vim
``` at the command-line; this should tell you where to find it. Once you know where it's hiding, you should be able to use System > Preferences > Main menu to add it to Applications > Programming (for example).

Always "should", because while these steps work for me I'm not a confident enough user to guarantee they'll work for you ;-}
