### Post by xavier_r on 2007-04-20
It took me 18hours on my 128kbps connection to get a feisty CD using wget
And after the download was over there was checksum error

And according to guides over the internet the iso should be downloaded once again...
That meant another 18hours of problem

No way...  i was gonna do that... :) 

So here's a solution i found...

i downloaded the torrent file...
i ran it and gave the save location to the iso file already downloaded
it asked me to continue last session
the bittorent downloader started and it progress bar went up till near the end of the file... and it stopped just a little bit before the end...
There!!! the problem was there...

Anyways, it just took me just 2 more hours and bittorent completed the file download...
And when I did the checksum again, there was no error :lolflag: 

I didnt had to redownload the file...
