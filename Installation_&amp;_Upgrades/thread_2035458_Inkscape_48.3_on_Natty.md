This is the [nightly build]("https://launchpad.net/%7Einkscape.dev/+archive/trunk") that will give you the latest version of Inkscape. 

This is the development version so it may have some bugs and you should use it at your own risk. 

The latest version seems to be 48.2 (you said 48.3... so I am assuming you want the unstable version) if you want to install the latest stable version on ubuntu you just need to [compile it]("http://wiki.inkscape.org/wiki/index.php/CompilingInkscape") from the [source tar ball]("http://downloads.sourceforge.net/inkscape/inkscape-0.48.2.tar.gz").
