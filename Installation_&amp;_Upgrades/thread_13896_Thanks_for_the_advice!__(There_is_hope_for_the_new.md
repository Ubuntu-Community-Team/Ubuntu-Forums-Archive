### Post by Chris in Texas on 2005-02-03
This is my first post, but by using the search function I was able to find out info on the older laptop installs.  I used Smart Boot Manager and it worked like a charm.  Changed the BIOS to boot from CD and I finally have Ubuntu running.

I have to say, this is probably the best Linux distro I have played with yet.  After XP crashed on my laptop (again) I had to wipe the hard drive and start fresh.  Couldn't reload XP, wouldn't work correctly so I tried Redhat.  It's not bad but couldn't make any of the good stuff work (USB, PCMCIA slots, etc...), tried Libranet (not exactly the friendliest for a noob like me, plus it doesn't flow well on my laptop), Knoppix (not as user friendly either), Suse was a nightmare for me and huge as well!!!

I was going to try Debian, which led me here.  Gotta say, this is a very nice system.  All the plug and plays actually plug and play.  I can use my USB drive and the screen resolution is great.  It has brought new life to my little PII 300Mhz laptop.  I thought the little knockaround laptop was bound for the scrap heap, luckily I decided to try one more distro before I gave up.  This one seems to be the one.  I still have to work my way through the learning curve, but it seems to be going well.  I wish the help screen helped a little more, but it's pretty user friendly for the uninformed like myself (new Linux user, in case you hadn't noticed).

Thanks again for all the advice that I picked up on the search option.  Tons of information on this forum if you take the time to search through the old threads.

Now I just need to get my wireless card and firewire card up and running, then I'll be set.  Ah to be able to surf the internet from my front porch once again while drinking coffee.  Almost there, just in time for all those cold fronts to hit Texas.

Thanks again and I'll be back with questions of my own soon.

Chris
