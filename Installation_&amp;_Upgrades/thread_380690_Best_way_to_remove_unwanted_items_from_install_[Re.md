### Post by aysiu on 2007-03-10
Go to System > Administration > Synaptic Package Manager, sort the items by what's installed (the green-filled box on the left column of the list of packages), then start marking for removal things you think you don't need. You'll be warned if any potential removals will also remove something you *do* need.
