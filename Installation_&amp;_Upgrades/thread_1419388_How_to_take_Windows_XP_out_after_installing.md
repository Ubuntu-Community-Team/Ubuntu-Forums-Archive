### Post by presence1960 on 2010-03-01
If you want to get rid of windows that is very easy. First back up any data you do not want to lose. Then reinstall ubuntu choosing the use entire hard disk option. This will wipe everything off the disk and set ubuntu up on the whole disk. After the installation is complete you can restore the data you backed up.
