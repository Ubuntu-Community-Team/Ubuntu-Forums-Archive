Some checks for you:

1.  [MD5]("https://help.ubuntu.com/community/HowToMD5SUM")

### Post by presence1960 on 2009-09-07
> **Devil's Psycologist said:**
> Any tips? Ideas? How long does it usually take?
First things first. Did you back up any data you don't want to lose to an external disk?

After downloading the Ubuntu iso did you [MD5SUM]("https://help.ubuntu.com/community/HowToMD5SUM")? The hashes must match exactly or the iso is corrupted.

Did you burn the iso as an image at a slow speed (4x-8x). If your burning software will not allow you to choose burn speed see here for [Infrarecorder]("http://infrarecorder.org/").

Boot the Live CD and choose check CD for defects. If it passes install Ubuntu.

At the prepare disk space screen choose Use entire disk. See pic below.

P.S. if you have a recovery partition did you make a Botable set of CDs/DVDs to restore your machine back to factory image if need be. Or make a backup image of the recovery partition. You don't want Vista now, but you will save yourself some trouble later should you need to restore the factory image at a later time.
