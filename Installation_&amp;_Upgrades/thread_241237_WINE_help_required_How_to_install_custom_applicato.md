### Post by atomkarinca on 2006-08-22
the folder is usually under /home/username/.wine (you should be seeing hidden files)
you can associate exe files with wine or you could just run wine explorer typing "winefile" on the terminal and run files on the wine explorer. the program files are under ~/.wine/Program Files folder and wine also should add a wine folder to the k menu.
