### Post by snowpine on 2008-09-25
Tell us more about the system: processor, ram, etc.

Pentium 3 with 384mb of ram is the minimum for the Ubuntu Live CD.
There is an alternate CD that uses a text-based installer and requires only 256mb of ram.

For 256mb of ram I would recommend Xubuntu which is a slimmer cousin of Ubuntu.

### Post by snowpine on 2008-09-25
You are in luck, there is a Linux distro called Damn Small Linux or DSL that is designed for older systems: [www.damnsmalllinux.org](www.damnsmalllinux.org)

It's not as "nice" as Ubuntu but it is quite functional and will teach you a lot about Linux. Hopefully someday you'll get a nicer laptop and you can come hang out with us. :)

ps DSL is not the only option; check out this mega-thread: [http://ubuntuforums.org/showthread.php?t=575456](http://ubuntuforums.org/showthread.php?t=575456) The distros are sorted by size of the installer, not system ram requirements, but there is often a correlation between the two.

Good luck!
