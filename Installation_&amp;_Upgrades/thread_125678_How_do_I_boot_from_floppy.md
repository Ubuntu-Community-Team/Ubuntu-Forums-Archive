### Post by lha on 2006-02-04
If you are using Windows, get [rawwrite]("http://uranus.it.swin.edu.au/~jn/linux/rawwrite.htm") and use it to write sbm.bin to a floppy. Sbm.bin is on the install cd, in folder /install.
