### Post by fantab on 2013-12-18
12.04 is LTS supported for 5 years, ie till 2017, while 13.10 is supported for only 9 months, ie upto July 2013.
14.04 LTS is due for release in April 2014.... which will be supported until 2019.... So the choice of LTS Ubuntu is more reasonable, esp for Servers.
However 13.10 supports the latest software technologies and hardware...

I would say stick with LTS only for servers. You can install any version of 'supported' version of Ubuntu. You can also install 14.04 LTS (development release, still in pre-alpha stage, expect breakage).
