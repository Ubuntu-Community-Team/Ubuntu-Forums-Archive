### Post by aurelieng on 2009-05-11
Sure, it's worth a try. You can install on top of your ubuntu installation it with the following line:
```
apt-get install kubuntu-desktop
```
If you don't like it, just ask your login manager to start a Gnome session and/or install ubuntu-desktop again with
```
apt-get install ubuntu-desktop
```
