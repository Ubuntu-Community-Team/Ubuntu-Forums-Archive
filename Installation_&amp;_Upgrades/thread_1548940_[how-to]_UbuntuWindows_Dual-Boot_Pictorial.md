### Post by Computer Guru on 2010-08-09
Hi,

EasyBCD 2.0 was recently released with improved support for GRUB2 and Ubuntu 10.x.. We've recently updated the picture-by-picture tutorial for dual-booting Ubuntu and Windows with the Windows boot manager controlling the boot process (instead of GRUB in the MBR) for those that prefer such a thing.

Just for reference, should anyone be in need of such a guide, the pictorial is located here: [http://neosmart.net/wiki/display/EBCD/Ubuntu](http://neosmart.net/wiki/display/EBCD/Ubuntu)

Please note that due to [a bug]() in the Ubuntu setup that prevents the installation of GRUB to the Ubuntu bootsector instead of the MBR, extra steps are required.
