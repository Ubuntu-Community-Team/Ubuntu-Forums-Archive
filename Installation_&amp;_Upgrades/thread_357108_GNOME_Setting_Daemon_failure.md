### Post by dbhunt on 2007-10-11
I encountered a similar "Gnome Setting Daemon failure" when trying to install Ubuntu Feisty 7.0.4 on my Dell Latitude C610 with 256 MB Ram.  The install CD would get to give me the error message but still make it to Live CD mode but then hang when clicked the "Install" icon.  

The problem in my case was not enough memory (I learned this from another post somewhere in the forums, but I can't find it).

The solution is to install the "Alternate Desktop CD" version of Ubuntu 7.0.4.  This can be found on the [main download page]("http://www.ubuntu.com/getubuntu/download") , but you have to check a small box at the bottom that says "Check here if you need the alternate desktop CD. This CD does not include the Live CD, instead it uses a text-based installer."

The Alternative version does not go into "Live CD" mode (a big memory hog) and instead just does an install of Ubuntu 7.0.4.  The install prompts are perfectly understandable in a text mode.
