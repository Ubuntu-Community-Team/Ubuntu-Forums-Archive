### Post by cavalier on 2006-08-20
I suspect it will, yes. If you're using the whole hard drive for Ubuntu, you'll be able to tell it to use the whole drive again on the reinstall and it'll come up clean for you. If you're using a portion of the drive, you'll have to do a little manual partitioning, but no more than you did when you first installed Ubuntu.

Good luck, it's a great little distro, and I think you're going about it the right way. It's so notorious for "just working" that you may find yourself not needing that reinstall.

### Post by K.Mandla on 2006-08-20
If you know you want to install fresh and don't want to wait for the desktop CD to do it's thing, you can install from the alternate CD. 

It's a text-based installation, which is less exciting, but as I understand it, it's also less finicky.

I would also suggest using a disk eraser like [Killdisk]("http://www.killdisk.com") to blank the drive before reinstalling. I've had problems where leftover digital crud interfered with installation and/or reboots.

If you know you want to start from scratch, blanking the drive is a good idea.

Make sure you copy off anything you might want. ;)
