### Post by rsambuca on 2007-05-12
It isn't really advisable to try and use an old 64-bit /home with a 32-bit OS as the programs use different libraries.  The bottom line is some things MAY work, but a lot of stuff will simply be borked if you do this due to different libraries and such.  I am not sure what you can't get to work, but if you are going to go back to a 32-bit system, I would recommend saving your needed documents and data files, but to do a clean re-install.

I have 64-bit system running with flash, java, nvu, skype, yada-yada, so If you need help trying to get that stuff working before you switch, let us know.  Nothing was very difficult at all and I am pretty much a newbie here - only started 8 months ago with linux.
