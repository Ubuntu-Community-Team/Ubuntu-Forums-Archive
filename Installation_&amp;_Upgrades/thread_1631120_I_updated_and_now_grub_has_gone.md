### Post by Swagman on 2010-11-26
There are some "unspoken rules" when using Wubi.

**Before installation** defrag your windows system or preferably install to another partition.

Wubi **Isn't** the best way to run Ubuntu... It quite often has issues after updating <--- Voila !!

You will probably have to uninstall Ubuntu from within windows an re-install.
