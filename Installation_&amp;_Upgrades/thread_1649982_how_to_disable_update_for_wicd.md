### Post by Zill on 2011-01-03
Same [wicd bug]("https://bugs.launchpad.net/ubuntu/+source/wicd/+bug/540070") here ;-)

I have disabled wicd updates by using Synaptic.  Simply open Synaptic, select wicd and then enable "Lock version" in the Package menu at the top of the screen.
