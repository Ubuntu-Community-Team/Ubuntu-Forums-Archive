### Post by McSnickered on 2010-05-01
I installed the Ubuntu Netbook Edition 10.04 on my Dell Mini 10V and ran into 3 issues:

1.  When I start up the netbook, Ubuntu does not take me to a login screen.  It just goes right into the desktop.  I've tried changing the user settings to ask for a password on login, but it does no good.  I'd be grateful for some suggestions - it's not a big problem, but I would never travel with it in this condition.

2.  Could not get wireless connection.  This turned out to be a known issue related to the Broadcom wireless card.  I was able to get this fixed using the following:

[http://www.ubuntumini.com/2010/04/broadcom-wireless-driver-fix-in-lucid.html](http://www.ubuntumini.com/2010/04/broadcom-wireless-driver-fix-in-lucid.html)

3.  After getting the wireless connection going, Ubuntu now asks me for a KeyRing Password each time I get the wireless connection started.  This is really annoying!

Other than those 3 issues, I'm really impressed with the speed!  But I'm surprised that I'd have the issues above with a Dell Mini since Ubuntu has worked with Dell for quite a while.
