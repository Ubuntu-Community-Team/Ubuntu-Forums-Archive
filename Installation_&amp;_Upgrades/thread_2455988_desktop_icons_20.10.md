### Post by grahammechanical on 2021-01-01
In my case it was the upgrade from 18.04 to 20.04 that substituted an app's desktop icon for a generic icon. The tip to allow launching fix the issue. Yes, the application does have an icon in the launcher but this was a Windows application running under WINE and the installation of it put a icon on the desktop which the upgrade broke.

For those who do not know. In Ubuntu with the Gnome 3 shell we can use the Tweaks utility to enable some Gnome shell extensions including one to have desktop icons. It should be enabled by default in Ubuntu.
