### Post by Gen2ly on 2008-04-06
The best-case scenario is if there is an external hard-drive or flash drive about to save any important files from the Ubuntu install on it, then use the Ubuntu LiveCD and combine the two Ubuntu partitions.  This will erase both your Ubuntu partitions so a re-install would be necessary.

Another option would be to use the other partition for the /home folder or perhaps /usr/bin.  We talked about it recently here:

[http://ubuntuforums.org/showthread.php?t=742805](http://ubuntuforums.org/showthread.php?t=742805)
