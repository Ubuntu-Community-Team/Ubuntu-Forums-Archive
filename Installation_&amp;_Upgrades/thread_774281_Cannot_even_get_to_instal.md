### Post by lemming465 on 2008-05-05
If you can get access to an ubuntu 8.04 i386 live desktop CD, you might want to try that instead of 7.10.  It's newer and might work better.  The text login dialog can usually be ignored as it should only be visible briefly until the X-windows GUI comes up.

If the CD is not working properly for you, be sure to run the memtest to make sure you don't have hardware memory issues and the cd-rom self-test to make sure it was burned successfully.
