### Post by shadowofblood on 2010-11-15
This is a new security check in 10.04.  It sees your CPU as one that uses the NX Bit setting.  Check your BIOS to see if you can enable it.

> NX stands for "No Execute" and is a setting used in newer AMD CPUs that  prevents code from running in data-only memory pages, which is a common  technique used in buffer-overflow attacks.
  
Intel's version is XD , or "eXecute Disable".

If your BIOS does not have the option to enable NX (or XD), then check to see if you have a BIOS update available.  If not, or if that doesn't fix the problem, you will have to contact the BIOS manufacturer.
