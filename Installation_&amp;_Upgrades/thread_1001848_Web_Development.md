### Post by jamesstansell on 2008-12-04
A lot of what you mentioned can be installed from the ubuntu repositories.  For example, you didn't mention the glassfish server, but it's even in there.

However, most of the java stuff can also be installed just as easily without using the ubuntu packages, especially if you set up a maven proxy server for jars reachable by maven.  Other things, like eclipse, seem to work better if downloaded directly, and the ubuntu package doesn't seem to give you much benefit anyway.
