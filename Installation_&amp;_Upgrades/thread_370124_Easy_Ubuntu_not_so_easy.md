### Post by dsimpson on 2007-02-26
You didn't say which version of Ubuntu you were using. I have Dapper and Automatix is no longer supporting that version, Easybuntu didn't work for me and I can't access it. The best option that I found if you are installing codecs and MSTTfonts is BUMPS, which you can find by going to Ubuntuforums welcome page and go to 3rd party projects, BUMPS is a script that works easily and painlessly. If you are just loading programs (free) then synaptic or aptitude or apt-get work best.
