### Post by sudodus on 2013-08-20
It is possible, but I don't think it is worthwhile trying, because USB sticks are **cheap and small**, and it is much easier to make one for each purpose.

Pendrivelinux claims that YUMI works for a lot of operating systems including Windows, and there are some detailed advice concerning Windows.

[http://www.pendrivelinux.com/yumi-multiboot-usb-creator/](http://www.pendrivelinux.com/yumi-multiboot-usb-creator/)
