### Post by tommcd on 2011-04-01
Ubuntu 9.10 will reach end of life at the end of April 2011. Rather than waste time with this, you should upgrade to (or better yet, do a clean install of) Ubuntu 10.04. Or do a clean install of Ubuntu 10.10, which is the latest version. Ubuntu 10.10 will also have the newest version of wine, which will be a benefit if you must try to run Windows programs on linux.
