### Post by stuporglue on 2005-10-19
This is a great boot manager. You install it to a floppy, the boot to the floppy. It finds all the CD drives and installed OSs, and lets you pick which one you want. 

Their pages seem to be having a bit of trouble, so just click the 3.7 folder and download the program (ie. the .exe file if you're on Windows).

I used it for a long time on a crusty old IBM that didn't boot from CD. 

[http://btmgr.sourceforge.net/](http://btmgr.sourceforge.net/)
