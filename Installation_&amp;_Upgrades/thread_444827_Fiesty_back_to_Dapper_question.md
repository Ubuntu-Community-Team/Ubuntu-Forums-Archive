### Post by jerrylamos on 2007-05-19
You have choices - Dapper, Edgy, Feisty.  I've seen tons of postings from people that run easily on one and not the other.  You takes your choice.

Feisty has some complex technical support for SATA and for SOME wireless cards, and SOME LCD monitors work better. Or not.  so Feisty might run better or might break.  

Most applications will run on any of them so there's no compelling reason to change levels unless the newer one has some advantage for you.  You can try with CD Live; if something doesn't work CD Live it's probably not going to work installed either.

Now I'm a glutton for punishment so on one of my systems I'm running whatever pieces of Gutsy are available.  Some things work, some are busted, no surprise since it's not supposed to come all together until October.
