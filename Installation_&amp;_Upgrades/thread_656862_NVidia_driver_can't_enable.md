### Post by ~LoKe on 2008-01-03
I've got an 8600GT and it worked out of the box.  Perhaps the drivers are still the issue.  Install Envy from [here](http://albertomilone.com/nvidia_scripts1.html) and run it in the terminal like so:
```
sudo envy -g
```
For the graphical version, or:
```
sudo envy -t
```
for the CLI version.

Then follow the steps.
