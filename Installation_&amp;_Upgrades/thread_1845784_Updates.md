### Post by coffeecat on 2011-09-17
You do have point releases with the LTS versions but not the intermediate versions. 10.04 is a LTS and the current ISO is 10.04-3 which is the third point release since 10.04 went final. The various ISOs for 10.04-3 were made up on 19,20 and 21 July this year, so they include all updates until then. If you install from a CD made from a 10.04-3 ISO, you will still have subsequent updates to apply. For 10.10 and 11.04, which are non-LTS releases, the images are, as MAFoElffen says, current for when they were released. There will be no updated versions of non-LTS ISOs.
