### Post by ajgreeny on 2017-03-25
No doubt Windows 8 is installed with UEFI, not BIOS, so you will need to install Ubuntu in the same manner to be able to dual boot the two OSs.

Start by reading [https://help.ubuntu.com/community/UEFI](https://help.ubuntu.com/community/UEFI) 
and for more details on specific makers machines and potential problems also see [http://ubuntuforums.org/showthread.php?t=2147295](http://ubuntuforums.org/showthread.php?t=2147295)
