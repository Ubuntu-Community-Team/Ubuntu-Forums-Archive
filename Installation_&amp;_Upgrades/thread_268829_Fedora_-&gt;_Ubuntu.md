### Post by meng on 2006-10-01
Kernel upgrade is different from changing distributions. Can you confirm which you are trying to achieve?
Fedora Core 5 uses 2.6.15 kernel
Ubuntu 6.06 uses 2.6.15 kernel

Fedora Core 6 (testing) uses 2.6.17 kernel
Edgy Eft uses 2.6.17 kernel

Now I'm not certain which sub-sub-sub-kernel version is being used (mine, 6.06/Dapper, uses 2.6.15-27) but a significant kernel upgrade is not going to be achieved by migrating from "stable" Fedora to "stable" Ubuntu.

Perhaps you could spell out in un-technical terms what you really want. What benefits are you looking for? Then we will know if you will get those benefits from a newer kernel, or by changing distributions.
