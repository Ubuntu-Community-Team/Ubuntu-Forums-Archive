### Post by elongated on 2007-06-24
Well, I managed to do a successful install. For some reason ubuntu dislikes my DVD-RAM multi drive. I removed the ribbon cable from the master and set the slave to master, and i did a successful install to 6.06 using the alternate installer. From dapper drake I upgraded to edgy, then to the latest, feisty fawn and now i have beryl running quite nicely. Have a nice day, folks! :)
