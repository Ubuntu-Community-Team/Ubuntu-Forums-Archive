### Post by ejket on 2007-10-28
I fixed my refresh rate by reonfiguring xorg:

```
# dpkg-reconfigure -phigh xserver-xorg
```

However, my problem was related to detecting the monitor.  If you can pick yours from the list, then you're better off than I was.  In any case, a reconfig should give you a new starting point if changes to xorg.conf failed to do the trick or went sour.
