### Post by oldos2er on 2008-10-20
Burn the ISO image to a CD as an image, not data. It's best to burn it slowly, 4x or less, to minimize the chance of errors occurring. See [http://psychocats.net/ubuntu/iso#burning](http://psychocats.net/ubuntu/iso#burning)
 Then you need to set your computer's BIOS to boot from CD.
