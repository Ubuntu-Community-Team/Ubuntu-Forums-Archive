### Post by heimo on 2005-08-12
Sounds like you have burned the iso file to the disc instead of creating disc of the iso file. What's the difference? ISO file itself contains whole image of the install disc with hundreds of files in it. It has to be burned with cd burning programs "burn from image" or similar option.
