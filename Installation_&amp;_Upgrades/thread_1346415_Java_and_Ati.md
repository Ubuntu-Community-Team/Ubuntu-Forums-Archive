### Post by tommcd on 2009-12-05
Ubuntu 9.10 is the current version of Ubuntu. New versions come out every 6 months in April and October (usually). The AMD64 means you installed the 64bit version, as opposed to the 32bit version.
For the quick and dirty way to install just about anything in Ubuntu, open a terminal (applications > accessories > terminal) and use this site as a guide:
[http://ubuntuguide.org/wiki/Ubuntu:Karmic](http://ubuntuguide.org/wiki/Ubuntu:Karmic)
If you are new to Ubuntu and linux, be sure to read all the introductory stuff first.
Specifically, to answer your question:
for java:
[http://ubuntuguide.org/wiki/Ubuntu:Karmic#Java_Runtime_Environment_.28JRE.29_for_Firefox_plug-in](http://ubuntuguide.org/wiki/Ubuntu:Karmic#Java_Runtime_Environment_.28JRE.29_for_Firefox_plug-in)
You can also install packages with the Synaptic Package Manager (system > administration > Synaptic Package Manager) if you don't like using the terminal.
and for ati drivers:
[http://ubuntuguide.org/wiki/Ubuntu:Karmic#Install_Latest_Nvidia.2FATI_drivers](http://ubuntuguide.org/wiki/Ubuntu:Karmic#Install_Latest_Nvidia.2FATI_drivers)
Note: It should say: system > administration > hardware drivers. Then reboot after the drivers are enabled.
Here is another great resource for getting started with Ubuntu:
[http://www.psychocats.net/ubuntu/](http://www.psychocats.net/ubuntu/)
Write back if you need more help with this stuff.

And welcome to the Ubuntu forums! Welcome to the cool side of computing!
