### Post by xeth_delta on 2008-02-11
There can be different causes for the problem. First of all, check the downloaded image MD5 sum and don't burn the installation cd at a speed higher than 4x. Mor information here: [https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28burn%29%7C%28]("https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28burn%29%7C%28")

If, after following the steps mentioned in the link, you still have problems, please try using the "alternate CD" to install. The latter should work most of the times, wher the desktop CD wil fail.
