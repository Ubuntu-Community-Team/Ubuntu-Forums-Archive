Yes, I have _always_ had some sort of problem with boot disks burned to CD-RW's instead of CD-R's same goes for DVD-RW instead of DVD-R.
I would recommend to burn boot disks of any OS at no more than 4x and only on CD-R.

The MD5 check is a good idea too. :)
