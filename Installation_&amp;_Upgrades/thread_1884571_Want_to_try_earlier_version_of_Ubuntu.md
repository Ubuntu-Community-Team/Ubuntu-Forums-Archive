### Post by snowpine on 2011-11-21
You don't need to wipe your drive; you can run 10.04 in "Live" mode from a CD or USB thumb drive.

If you decide to install then the installer has the usual options: use the whole hard disk, install side-by-side, or manual. Assuming you have backed up all files in 11.04 (and you aren't dual-booting with another operating system), you can just choose the entire drive option and let the installer do its thing. (If you have a custom partition layout then you can choose Manual.)
