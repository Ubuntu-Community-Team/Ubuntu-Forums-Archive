### Post by njsharp on 2011-04-13
When you run Update Manager, you can view details during both the update cache and the download modules phases.  Since 10.10, and now continued in 11.04 at beta 2, the format changed such that the repository being used for each file no longer appeared in the list during the download.  True, you can mouse over each item and a pop up will tell you this information FOR THAT FILE, but it does not update for each new file, so you would have to keep mousing over each new one to see that information, which is often not practical as they are coming too fast.

Being able to see the repository used is valuable in making sure that you are retrieving from preferred sites, especially if some are unmetered by your ISP.

I would like to see a choice of format for those details, perhaps via Configuration Editor/apps/update-manager.

Still true at 11.04 beta 2 is that Synaptic Package Manager presents the information in my preferred format if you click "Show for individual files".

Am I alone?  Is there already a work around?
