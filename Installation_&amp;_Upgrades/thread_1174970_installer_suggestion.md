### Post by Cowloon on 2009-05-31
The installer is certainly easier than other linux installers I've used. And, this is meant as a helpful suggestion, if any installer developers read this. I think it would be faster for everyone if the installers that come in an ISO installed entirely from the ISO, and then had you upgrade packages after the installation.

Because Ubuntu is so popular, I suspect. Upgrading has always been absurdly slow, compared to when I was using Debian. Recommending that people install entirely from the network is, I suspect, why it's so slow.

In fact, I stayed with 8.04 for so long, because I have never had a couple of days free to upgrade my operating system, when I need to be using my computer.

Unless I've misunderstood something, the alternate CD, still installs from the internet, even though it says it is for installing when you don't have an internet connection.

At least, for me, it prompts me and asks if I want to download from the internet, to which I naturally say "No" (that's why I'm using this ISO). But, it downloads from the internet anyway.

I tried unplugging the ethernet cable and restarting and it hangs after it gets to some package that I suspect it's trying to download from the network.
