### Post by 043087m135 on 2007-07-10
I wouldn't recommend trying to run Xubuntu on a 166mhz system with 64 megs of ram. I'd recommend Puppy. I run puppy on my step-mom's computer, its a 200mhz with 96 megs of ram. The thing runs like a beast, with wireless functionality and all the rest. DSL is a great boot/rescue disk but not anywhere near as functional as a traditional hard-disk resident installation of Puppy. For older systems than that try DeLi, Desktop Light linux which runs on many of the same concepts as Puppy. 

Good luck, hope this response isn't too late!
