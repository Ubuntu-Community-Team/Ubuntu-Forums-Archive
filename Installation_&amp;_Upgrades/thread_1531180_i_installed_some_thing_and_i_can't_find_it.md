### Post by cariboo on 2010-07-14
If it isn't under Applications->Sound & Video, you can start it by pressing Alt-F2 and typing:

```
kdenlive
```

You can alos add it to the menu yourself by right-clicking on Applications->Edit Menus.

The kdenlive executable is located in /usr/bin eg:

/usr/bin/kdenlive
