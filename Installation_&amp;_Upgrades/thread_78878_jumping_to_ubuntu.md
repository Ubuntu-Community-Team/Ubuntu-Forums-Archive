### Post by Riverside on 2005-10-19
Having been using Linux exclusively at both home and work since 2002, I would say that Ubuntu is definitely worth trying.  I started off in the Unix world with Solaris (an old Sun Ultra 5 running Solaris 7 which I was able to requisition at work, which I had fun and lots of trials and tribulations with until realising that its previous owner had been rather too liberal with the rm command whilst logged in as root - a clean installation of Solaris 8 resolved those issues!) in 2001, and then installed SuSE 7.3 on my home machine later that year.

SuSE was at the time probably the most "user friendly" of the various Linux distributions available, in particular in terms of ease of installation.  Its YaST installation tool, and in particular its SaX X server configuration tool, made installation of Linux possible for almost anyone, to the best of my knowledge the first time that had happened in the Linux world.  Of course, most users who decided to stick with Linux subsequently (myself included) took time to learn what is happening in a Linux system under the hood, and gained more knowledge as time went on.

Ubuntu seems to have inherited SuSE's mantle as an easy to install (the easiest I think) and maintain Linux distribution, particularly Breezy Badger.  I am about to install Breezy on my main machine which is used for both personal use and work, replacing SuSE 9.2.  Ubuntu's community feel is particularly exciting, something SuSE post their acquisition by Novell appears to have lost.  There is also significant resource and a team of excellent technical folks behind Ubuntu, which makes it a distribution which one can most likely rely on.
