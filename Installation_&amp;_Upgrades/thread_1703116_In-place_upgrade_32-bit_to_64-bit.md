### Post by 1clue on 2011-03-08
Not sure how I'd do that on Ubuntu.  It MIGHT be possible with something like Gentoo though, but I think the broken stuff would probably be several times more painful than just scraping it off and starting over, and it would definitely take a lot more time.

If I were to try that on any distro, I would first convert to dual-mode and then start explicitly installing 64-bit apps and libraries where possible.  Then finally remove 32-bit libraries and apps.

Seriously, much easier to just back your data up and start over.
