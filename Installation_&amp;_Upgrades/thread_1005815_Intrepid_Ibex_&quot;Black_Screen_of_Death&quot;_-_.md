Hey guys,

This is what I did to "fix" the black screen problem. ("fix" may be the wrong word, but it will do for now)

1. Clean install of 8.10 (Use "Safe Graphics Mode - F4)
2. Installed the usual updates. (no reboot)
3. Installed "Restricted" drivers. (I use an ATI grafix card)
4. THEN reboot.

I figured that 8.10 had difficulty installing because it did not have the current ATI drivers. Using "Safe Mode" ensured that I could at least have a GUI long enough to be able to download the drivers. Well, that's my guess, please correct me if I am wrong.

I have rebooted several times now and each time 8.10 has functioned just fine.

I chose a "Clean" install because I have had problems in the past with the online upgrades. If you want to try upgrading, let me know how you get on.

I hope this helps.
