Hello, as an ubuntu user since 8.04 I did a lot of upgrade and clean install. So I can say that this time the upgrade is a really messy one.

I've got a netbook and so I upgraded from the remix edition to unity. Well in theory. In practice both of the interfaces were running at the same time. Messy. 
Another example is the softwares like F-spot. So F-spot is not used anymore. very well. but after the upgrade F-spot is still here and there's the new soft. Now I have two soft doing the same thing. Maybe during the upgrade we should be able to choose what we want to keep or delete ?
After this upgrade my poor netbook looks and feels like Windows after a few month of use : slow and fat. 

In fact it looks like most of the hard work done by the developpers was aimed at 'clean' installation. Which is nice for newcomers but not for upgraders.
