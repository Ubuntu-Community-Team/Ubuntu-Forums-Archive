### Post by snowpine on 2009-07-16
I recommend an inexpensive 1gb USB flash drive.

Then, you can install Ubuntu Netbook Remix following these instructions:
[https://help.ubuntu.com/community/Installation/FromImgFiles](https://help.ubuntu.com/community/Installation/FromImgFiles)

Or "regular" Ubuntu using Unetbootin:
[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

I do not recommend using an external hard drive that already has data, because you run the risk of destroying that data if you make a mistake. :) A 1gb thumb drive is cheap and safer. Good luck!
