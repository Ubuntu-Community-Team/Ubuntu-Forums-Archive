...

libamrnb3

libavformat1d
libavutil-dev
libavutil1d

libswscale1d
mencoder
mplayer
mplayer-doc

These are all packages from the [Medibuntu](http://medibuntu.org/) repository.  Perhaps you need to add the medibuntu-keyring:
```
sudo apt-get update && sudo apt-get install medibuntu-keyring && sudo apt-get update
```
More info: [Medibuntu Community Ubuntu Documentation](https://help.ubuntu.com/community/Medibuntu)
