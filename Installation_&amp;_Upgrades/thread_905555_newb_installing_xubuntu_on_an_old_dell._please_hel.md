### Post by pytheas22 on 2008-08-30
You may want to try installing using the alternative CD--there's a box that you can check to request this when downloading the ISO from the Ubuntu site.  The alternative CD uses less memory, which may be the problem here (256 MB is really the bare minimum for a live CD).

Another option, since you have Windows on the computer, is to install using [wubi]("http://wubi-installer.org").
