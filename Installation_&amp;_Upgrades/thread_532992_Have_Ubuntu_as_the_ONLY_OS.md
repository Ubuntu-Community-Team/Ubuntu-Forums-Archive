### Post by aysiu on 2007-08-23
There's no minimum RAM requirement for dual booting. There is a minimum hard drive space requirement, though (I'd recommend at least 20 GB).

If you don't have a lot of RAM, though, make sure you use the Alternate CD and not the Desktop CD to set up the dual boot.

### Post by wickstopher on 2007-08-23
Well, for starters, dual-booting does not mean that both operating systems are running at the same time.  So you don't need to worry about "not having enough RAM" to run both operating systems from the same computer.  You just boot into one at a time, and if you were running windows before, you certainly have enough RAM to run Ubuntu.

But if you want to get rid of windows entirely, just back up all the documents that you want to save (photos, music, letters, etc.), and follow the instructions on the installation CD to reformat your hard drive and install Ubuntu.

go to /[http://www.ubuntu.com/getubuntu]("http://www.ubuntu.com/getubuntu").  The instructions are fairly simple.
