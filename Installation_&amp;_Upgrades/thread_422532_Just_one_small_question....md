### Post by Phineas on 2007-04-25

### Post by ttakun on 2007-04-25
First Question: Do you really want to install Ubuntu, or you just want to give it a try?
If you just want to try it without changing anything in your computer, just download the corresponding ISO file depending on your system. Burn it into a CD, and then re-start your computer allowing it to start from the CD/DVD drive.
If what you want is really to install Ubuntu on your computer, it's up to you. You can have both of them or only Ubuntu (but you will loose Windows, its programs, data .... You will to backup your data before). If you want to have the two operating systems, let's raise another question: Do you want to install Ubuntu on the same hard disk or you will install it on another HD?
