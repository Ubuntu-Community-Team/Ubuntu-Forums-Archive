### Post by Cygni on 2010-11-16
If you are unable to complete an update through "Update Manager" due to the message: Failed to fetch [http://archive.canonical.com/ubuntu/pool/partner/a/adobe-flashplugin/adobe-flashplugin_10.1.102.65-1maverick1_i386.deb](http://archive.canonical.com/ubuntu/pool/partner/a/adobe-flashplugin/adobe-flashplugin_10.1.102.65-1maverick1_i386.deb) 403  Forbidden  

I found a solution: Visit the Maverick package site for the Adobe plug-in at [https://launchpad.net/ubuntu/maverick/i386/adobe-flashplugin/10.1.102.65-1maverick1+really10.1.102.64](https://launchpad.net/ubuntu/maverick/i386/adobe-flashplugin/10.1.102.65-1maverick1+really10.1.102.64)  

Download the .deb file and then open "Software Center", which should give you an installation option for the download.  

After installation, open the "Update Manager", click "check", which should show one less package to install, then click "install updates" and it should complete the process.
