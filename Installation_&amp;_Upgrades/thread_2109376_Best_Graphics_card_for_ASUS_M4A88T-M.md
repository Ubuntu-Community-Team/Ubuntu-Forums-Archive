### Post by darkod on 2013-01-27
That board has PCI-Express x16 slot and almost any card can be installed. Most ATI/AMD cards work fine with ubuntu and should work.

I would suggest to select a model according to the price you want to pay, and then google if that model has problems with ubuntu. If the search doesn't find any info about big problems, you can buy it.
