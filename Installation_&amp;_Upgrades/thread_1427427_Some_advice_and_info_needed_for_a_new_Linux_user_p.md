### Post by jjcv on 2010-03-11
> **silkworm2.5 said:**
> 
How can I install the _Ubuntu Netbook remix 9.10_ onto the hard drive, without formatting or anything, while still leaving the original system intact? 

As I see it you have two options.

1.  Shrink the partition you currently have on the laptop and install Ubuntu next to it so that you can boot from either partition.  (You can later remove the ubuntu partition).   There is an option to do this when you install.

2.  Install Ubuntu on a large USB stick and when booting the laptop tell it to boot of the USB stick (USB drive).  

The best option in your case is problem 2).

There are lots of HOWTO's on installing ubuntu on a USB or second drive.
