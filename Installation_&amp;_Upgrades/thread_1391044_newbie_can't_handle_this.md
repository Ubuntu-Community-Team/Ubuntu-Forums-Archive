### Post by snowpine on 2010-01-26
"theli.free.fr" is not a default software source for Ubuntu. I also see it is for Dapper, which is an obsolete 2006 release of Ubuntu. You must have added this software source for some reason, can you explain? :) 

The easiest thing to do is:

```
gksu gedit /etc/apt/sources.list
```

to edit your software sources. Find the 2 lines that say "theli.free.fr" and type a # symbol at the beginning of these 2 lines (to comment them out so they're skipped). Save the changes and try your update again. Good luck!

### Post by earthpigg on 2010-01-26
```
E: Could not get lock /var/lib/dpkg/lock - open (11: Resource temporarily unavailable)
E: Unable to lock the administration directory (/var/lib/dpkg/), is another process using it?
```

that is ubuntu telling you not to have synaptic or software center or apt-get or any other package management tool open/running at the same time as another.

run sudo apt-get, **_or_** synaptic, _**or**_ software center... but ***never*** have more than one open/running at the same time.

:D

> now I'm updated.

your package manager has an updated list of packages, but you haven't yet upgraded the packages installed on your system.

> sudo apt-get update

is to update the package manager with a current list of packages and upgrades and whatnot.

> sudo apt-get upgrade

is to look at the updated list and upgrade your system to the latest versions available in the repos - security patches and bug fixes, mostly.
