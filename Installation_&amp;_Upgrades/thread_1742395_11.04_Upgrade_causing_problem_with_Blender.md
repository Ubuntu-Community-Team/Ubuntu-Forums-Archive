### Post by alphacrucis2 on 2011-05-01
> **Charlie1962 said:**
> Hi

With 10.10 i had no problems with blenders latest version, however, after upgrading to 11.04, Blender 2.57 is  in Chinese and Blender(Windowed) installing 2.4 version


Any help getting back to Blender 2.57 English version is greatly appreciated.

If you installed an updated version by compiling from source or just a binary installer then the package manager has no way to know that a newer version is installed. Hence it thinks that the one in the 11.04 repo is newer than what you have. This is always going to be a problem on an upgrade. One way around it is to to make sure you remove any packaged version using synaptic before compiling and or installing a newer version outside of the package system.
