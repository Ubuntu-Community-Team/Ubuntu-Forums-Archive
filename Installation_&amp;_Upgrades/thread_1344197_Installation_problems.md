### Post by darkod on 2009-12-02
If possible boot with the cd, select Try Ubuntu, and it will run from the cd. Download the script from my signature, and execute it, if for example on the desktop, with:

sudo bash ~/Desktop/boot_info_script*.sh

You need to enter that into terminal (Applications -> Accessories -> Terminal). It will create results.txt file. Post the content of that file here but after you copy the text here, select the whole text (just the content coming from results.txt) and put CODE tags, click the # button in this toolbar for writing posts. It makes it easier to read.

That will show the whole boot process and lots of info about the hard drive.

PS. In case you didn't know, the LiveCD should give you internet access so you can do all of the above in one go, while booted with Try Ubuntu, no need to restart or take results.txt file to another comp.
