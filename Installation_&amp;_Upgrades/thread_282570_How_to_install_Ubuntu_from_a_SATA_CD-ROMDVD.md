You have to burn the image to the disk not the file. You have simply placed the iso file on the disk not what is contained in the iso file. There is no operating system on the disk you burned, only an iso file. XP Standard Burning Software **does not** have this capability. To burn the IMAGE not the iso to a disk you will have to istall something to allow you to do this. Check out this link: [http://www.ezbsystems.com/ultraiso/](http://www.ezbsystems.com/ultraiso/)  It may be helpful. I'm not really sure since i cant install it on my linux system.
