### Post by snowpine on 2010-04-18
I always test a Live CD of a new release before I commit to an upgrade. If the Live CD does not work, I know it is not the right release for my hardware and I skip it. 9.04 will be fully supported through October 2010. Wait until 10.04 has a stable release and give a Live CD a try, see if you have the same problem. 10.04 will have a focus on stability so maybe it will be right for you.
