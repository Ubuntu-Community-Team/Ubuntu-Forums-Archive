### Post by sina253 on 2011-05-24
Hi,
 
I did the following procedure and the problem solved:
 
1 - Went to  [http://releases.ubuntu.com/](http://releases.ubuntu.com/)
2 - Select appropriate Ubuntu version and went to that folder

3 - Downloaded the wubi.exe file from that  
4 - Put wubi.exe file alongside the Ubuntu ISO file in my hard drive which I downloaded before
5 - removed the Ubuntu CD from CD drive
6 - Run wubi.exe
7 - Keep clicking "continue" on error which said "there is no CD in drive"
8 - Ubuntu extracted and installed successfully
