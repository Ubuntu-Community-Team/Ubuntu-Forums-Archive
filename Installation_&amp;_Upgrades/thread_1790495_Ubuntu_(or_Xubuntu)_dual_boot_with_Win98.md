### Post by TheBig3 on 2011-07-13
Finally got everything done and installed. Found a current anti-virus for Win98 ([Multi Virus Cleaner 2011]("http://www.viruskeeper.com/us/mvc.htm")). Went with Lubuntu 11.04, took forever to install from 6x cd-rom. Both OS's are operating smoothly now after much tweaking of both. GRUB found and identified the win98 partition and included it in the boot menu. I decided for almost a 50/50 split of the drive, 64GB for windows, 4GB linux swap, and the remaining for lubuntu. I still have some games that I enjoy that work on the DOS/95/98/ME systems but not on newer windows or on linux either, so that's why I wanted a windows system and to dual boot with it. 
Both now have office programs and VLC media players installed and running smoothly. Last thing I plan to do on the system is max out the ram to 768MB for the benefit of both OS's. Honestly I did not care that it was/is outdated or unsupported, it was something that I wanted to do.

BTW: I do have newer systems, this was simply my first build ever.
