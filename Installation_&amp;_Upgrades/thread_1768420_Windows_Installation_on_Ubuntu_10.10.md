### Post by Hedgehog1 on 2011-05-27
For your situation, I belive you will need to burn the Windows ISO to a DVD, and then boot from the DVD.

Please note that you need to use the 'Create DVD from ISO' option, as an ISO file is a whole CD or DVD compressed into a single file.  The 'Create DVD from ISO' decompresses and make a complete disk again.

If the disk you create has the ISO file just sitting on it, it will not work and will not boot.

I believe that your USB has the ISO file sitting on it - and that will not boot.
