### Post by confused57 on 2006-06-16
If you're trying to install from the iso, I don't know how to do that...but if you installed the server using the Xubuntu-alternate CD, you can:
```
sudo nano /etc/apt/sources.list
```

Make sure there's not a # in front of the CD-rom entry near the top of the list.
Then make sure there is a # in front of anything that looks like a url.
Ctrl+X, press "y", press "enter" to save. Exit

Then at the prompt:
```
sudo apt-get install xubuntu-desktop
```

I'm not sure if your system will allow this...but it's worth a try.
