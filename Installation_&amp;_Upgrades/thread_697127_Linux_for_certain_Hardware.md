### Post by oskar2000 on 2008-02-15
Try to get at least 256mb ram. Yes, you can get a linux distribution running with less, but I don't think it's necessary. If you are in a city there will be companies all around you that are dumping (literally) 1ghz+ machines, because it's not even worth selling them. 

I don't think that any of the few distributions we have that are specifically made for old pc's are all that great as a desktop system. I would install Just any distro that offers a minimal or net-install, and just install E17 or fluxbox. Maybe even XFCE if it can take it, and go from there.
Dillo as a webbrowser, Abiword for word processing and so on. Look at DSL and Puppy for the program selection.
