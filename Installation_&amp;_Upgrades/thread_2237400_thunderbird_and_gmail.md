### Post by m_duck on 2014-08-01
For programs (like Thunderbird) which can't use two-factor authentication, you'll need to set up an "App-specific password".

Go to your Google account settings and go to the security tab, then click settings next to app passwords. You can then make an app specific password with which to log into Thunderbird.

The nice thing is that these can be revoked individually, so if for example Thunderbird was on a computer you lost access to, you could revoke that password and it would no longer be able to access your account, whilst other programs you use would remain unaffected.
