### Post by grahammechanical on 2011-03-17
I do not know if this will help ...

I type in a terminal winefile.  This loads a Windows Explorer type file manager. If you have a disc in the CD drive you should easily be able to browse to the CD and see all the files on the CD including the install.exe or setup.exe that will install the program.

You should be able to do the same with the configure wine entry on the wine menu. click Add application and you get a Windows Open File type dialog box from which you can browse to the CD Drive. I hope this is what you are looking for.

Regards.
