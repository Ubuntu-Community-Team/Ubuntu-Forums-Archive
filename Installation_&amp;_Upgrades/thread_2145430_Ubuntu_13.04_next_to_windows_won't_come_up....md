### Post by fantab on 2013-05-15
[**Boot-Repair**]("https://help.ubuntu.com/community/Boot-Repair") tool/utility should solve it for you. You should boot you Ubuntu installation DVD/USB, "Try Ubuntu" when the desktop is loaded install it and run it. Boot-repair will scan for errors and offer "Recommended Repair", run it. If it doesn't then post the output of "Bootinfo Summary' that Boot-Repair generates.

Follow the above link for more detailed instructions and explainations.
