### Post by egeezer on 2014-12-26
Why your Try Without Installing runs so very slowly I don't know, but installed versions invariably run much faster than trials from either CD/DVD or USB.  So if you want to do the installation, boot from the USB and make the Install choice immediately.  That too may be slow from the USB, but it is not an accurate measure of how the installed version will run.
