### Post by Ponhovo on 2008-08-22
fixed =D

i killed all the processes that was holding back the upgrade until the upgrade aborts
so, i screwed thewhole system

so i went into recovery mode in the very first kernel i had 
and did went to root shell 
so:
dpkg --configure -a

and all locales stopped holding back
when i boot back to gnome, it was all working fine

thats it, for those who got the same issue
