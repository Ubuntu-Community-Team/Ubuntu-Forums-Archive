Hallo peeps,

Just wondering if anyone knows of a way to blacklist a particular update.

I've set up resumable VNC sessions in Feisty and it meant using the Edgy version of vnc4server.

So everytime I login the notification bubble tells me there's a newer version. Quite irritating but the worst is that when other updates come out I've sometimes forgotten to untick vnc4server which then gets updated and kills all VNC hopes.

So I want to tell the system that where vnc4server is concerned I don't need any updates. Anyone know how?

### Post by floke on 2007-06-04
Highlight the packages you want to lock in synaptic,  - then from the 'package' menu option select lock, and voila!
