### Post by bonzodog on 2006-05-01
No, it should not really be a problem, as linux always does hardware detection at each boot, and then it loads the appropriate modules for the hardware, so it allows for almost dynamic changing of hardware. The only snag you could ever expect to hit is if you changed your graphics card.
