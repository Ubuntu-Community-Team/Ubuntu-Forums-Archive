### Post by upchucky on 2008-10-06
it has picked the wrong settings for the graphics card, you will have to google that card or search the posts here to see if there are any tweaks to making that card play nice.

once you find the proper info, you will probably need to edit the xorg.conf file to set the right settings.

again search here for how to edit the file, and get a gui working.

there is a plain vanilla setting that may work enough to let you see a gui.

the vesa or nv driver I believe. along with a resolution setting of 1024x768, or 800x600 that may get a gui up for you, once you have a basic gui, then research will find info to set it up properly.
