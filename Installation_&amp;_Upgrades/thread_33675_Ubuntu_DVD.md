The ubuntu Hoary 5.04 DVDs were available from the unbuntu archives (see wiki for mirrors). It is available only available as a downloadable ISO. The 040705 releases are essentially the combination of all the final releases of the 5.04 Hoary Live, unbuntu, and Kubunutu in i386 and amd64 flavors. They will boot as a Live disk or  install ubuntu, Kubuntu or both. It contains all of the Main repository and some things form the restricted repositories all on one disk.
