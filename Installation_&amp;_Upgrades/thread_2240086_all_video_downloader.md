### Post by michal-klekowicki on 2014-09-02
I had the same problem with this strange and very annoying app.


Install "Synaptic Package Manager" and search for "AllVideoDownloader". There will be a package marked with a green square (possibly i386) - mark it for being removed completely and apply the changes.


I is a really strange package - when installed from the Ubuntu Software Center it triggers a secondary installation process of a different version of the program which is not available from the Ubuntu Software Center and not seen by it.


However the above method will remove it.
