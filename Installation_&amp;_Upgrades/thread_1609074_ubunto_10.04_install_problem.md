### Post by Quackers on 2010-10-30
Welcome to UF.
The 10.04 file is a .iso file. That means it isn't just burned to a disc, it must be burned as an image with something like ImgBurn (free program for Windows) or Brasero in Linux, to name two. This is the only way to make the disc bootable.
