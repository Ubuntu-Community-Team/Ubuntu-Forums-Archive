### Post by bliffle on 2007-06-17
What I did was to get a new 80gb 2.5" HDD ($80) for my Thinkpad T40, and an aluminum box ($10) with USB connector to hold the old 30gb Windows HDD. 

I pulled the old Win HDD and put it in the box. Installed the new HDD, put the Ubuntu 7.04 CD into the CD reader, and powered on. I interrupted the BIOS and entered setup to make sure the T40 would use the CD as a boot device. 

It booted up from the live Ubuntu CD. I had created it beforehand by torrenting the distro and burning to a CD.

Once Ubuntu boots up from the CD you can fool around with it, go on the internet, etc., right from that CD. pretty handy. That way you can look around and get the feel of it before deciding to install to your new HDD.

Then I went ahead and installed it - no problem. If I needed a file from the old Win HDD I just plugged that USB cable from the aluminum box in and went and got it.

I ran that way a couple days to accumulate a list of reasons to go back to the old Win system to get stuff that would have to be prepared to port to Ubuntu. About the only thing I found was my old Email stuff in Outlook Express. So one day I swapped the old Win HDD back into the T40, booted up old Windows (and waited, and waited....) then ported Outlook Express over to thunderbird and then exported the thunderbird mail files so i could transfer them easily to Tbird running on Ubuntu.

That done, swapped the Ubuntu 80gb HDD back in, captured the Tbird file off the Win HDD (safely back in it's box!), imported that to Tbird on Ubuntu.

So now i have all my old files, family pics, documents, etc., in that box where I can get it anytime, and I have the old Win system frozen in time in case I ever need it again, which I doubt.

I have 3 working Thinkpads here for various purposes. One is used only for burning CDs and DVDs and is still running the old Win setup simply because I've honed it into a smooth and reliable system (it NEVER connects to the internet). But it's comparatively slow, as I can see from trials I've run on Ubuntu, and someday I'll convert it to Ubuntu also.

I have several 2gb USB memory gizmos for transferring data among the 3 systems easily. I do not network for safety reasons.

The only known problem I have to solve is to get my USB plugin HDTV stick (Diamond HDTV100) working so I can watch broadcast HDTV on the T40 when traveling. I just spent about 12 hours trying to get the Ubuntu stuff setup, but haven't found the answer yet, so what I'm going to do is make a little 10gb partition for Win on the new HDD with a Ubuntu program, like gparted, install a runt win system and then the TV software for the HDTV stick. Eventually I'll get the Ubuntu working, even if I have to buy a new Hauppage DVR-950, which is the only HDTV I've got good linux instructions for.

I have no problem with the wireless adapter in the T40, but sometimes after a couple days use I have to re-boot the T40. Not a big problem.
