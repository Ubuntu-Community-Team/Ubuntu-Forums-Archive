### Post by fela on 2009-09-05
There's no limitation, even if you install a completely basic system without even a desktop environment, you can always install whatever you want later when the base system is installed. Think of it as a barrel of liquid rather than a lego tower: you can always add more liquid if you don't put enough in to start with, and you can take some out if you want.
