### Post by guiverc on 2021-09-26
You need to peruse google results when you use it; it offers up the legitimate site yes, but also a *fan* site, plus others too if you ask in certain non-english languages.

Ubuntu, Lubuntu, nor Canonical control that web site (.net) so we have no control over it.

If you're trying to find a legitimate Ubuntu *flavor* site in future, don't ask google - but go to ubuntu.com and search there - you get to [https://ubuntu.com/download/flavours](https://ubuntu.com/download/flavours) which will send you to the official *flavor* sites for all *flavors*, not just Lubuntu.

Also FYI:   If you were using *ublock origin* as an extension; you would have been warned that you were going to a *risky/fake* site.
