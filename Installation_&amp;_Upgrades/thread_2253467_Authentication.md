### Post by ajgreeny on 2014-11-20
If the system is set to not ask for a password and autologin to your username and the seller did not tell you what the password was, you can reset to a new user password by following carefully the info at [http://www.psychocats.net/ubuntu/resetpassword](http://www.psychocats.net/ubuntu/resetpassword)

The password you set there will be the one you use to authenticate when installing packages.
