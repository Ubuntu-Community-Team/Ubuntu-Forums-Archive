### Post by housam on 2008-04-02
you must have two partitions to install ubuntu. one for the root ( / ) ext3 format ( more than 10 GB ) and the other one is a swap ( 1-2 GB is OK ).
you can create them instead of one of your empty partitions.
you can use Gparted tool ( on the live CD ) system >> admins. >> partition editor .

during the installation , choose the manual way and assign these partitions manually.
