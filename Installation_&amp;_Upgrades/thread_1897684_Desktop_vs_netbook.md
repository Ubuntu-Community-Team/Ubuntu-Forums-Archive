### Post by hackb0y294 on 2011-12-19
The 32 bit version will run on 64 bit architecture, but not vice-versa. I wouldn't reccomend installing the 64 bit version of 10.10, as there were numerous dependancy problems with older versions. These can be solved, but it's a lot of work. The only caveat with the 32 bit version is that Ubuntu wouldn't be able to use more than 3.75 Gigabytes of RAM (only applicable if you have more than that, though for most people running 3.75 Gig should be more than enough for Ubuntu, as it's relatatively light on system resources). Regarding the netbook version, it really doesn't matter whether you install the netbook or desktop version in terms of compatibilty, but the netbook version gives you a custom user interface (among other things) designed for the small screens of netbooks.
