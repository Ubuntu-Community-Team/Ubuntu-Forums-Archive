### Post by ajgreeny on 2009-08-04
You will need to restore the windows MBR, the same as you would on a normal laptop or desktop, but in that situation you would normally use the windows install CD.  I don't know how you can do it without a CD drive, but I imagine it must be possible to use a flash drive to do the same thing.

Failing that I presume you must have some form of recovery partition or flash drive that came with the machine, to enable you to restore things to the original settings as you bought it, so it may be that you will end up backing up your files and reinstalling the windows software from scratch.
