---

### Post by neo_1in on 2009-05-11
You can use either. In desktop cd the option to install grub is there in "Advanced Options", just before the commencement of installation. Alternate cd also gives you the option. For anything above 512 MB of RAM, go for desktop cd.

---

### Post by Mark Phelps on 2009-05-11
I found I had problems with partitioning using the desktop CD.  When I retried with the alternate CD, there were no problems.  So, if you get into partitioning and it doesn't seem to recognize your current partition scheme, you could try the alternate CD.
