### Post by john134 on 2014-01-10
So, I fixed this one myself, almost instantly :)

Basically it was just

```
$ sudo apt-get remove linux-generic linux-headers-generic

HOLD BREATH...

$ sudo apt-get install -f

$ sudo apt-get install linux-generic linux-headers-generic

BREATHE!

```
