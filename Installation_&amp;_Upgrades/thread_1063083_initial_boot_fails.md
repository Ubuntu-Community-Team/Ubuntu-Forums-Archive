### Post by avtolle on 2009-02-07
Can you download the Xubuntu alternate install disk and use it instead? To install Ubuntu from the Live CD, 384mb RAM is the minimum requirement; to install Ubuntu from the alternate CD, 256 mb is the minimum requirement. For Xubuntu, IIRC, it takes 192mb RAM to install from its Live CD (although 256mb is recommended); and for the Xubuntu alternate, 64mb (I think) to install.
