### Post by jjcv on 2010-03-11
If they provide a ubuntu  netbean*.deb file then you can probably just run:

sudo apt-get install netbean*.deb once you have downloaded.

If it is something else then you probably should remove your current version.

sudo apt-get remove <software name>
