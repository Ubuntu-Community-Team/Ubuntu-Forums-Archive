### Post by spd106 on 2007-07-11
Hi Pat,

Squid is a good proxy solution. It supports transparent pass-through as well as redirection proxying. Installation is very easy as it's an officially supported package in the core Ubuntu repository. 

The squid-cgi package is in the universe repo, if you want to have a web based configuration utility rather than edit text files.
