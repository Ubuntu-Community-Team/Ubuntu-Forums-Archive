Short answer: no.

Longer answer: 
1) If you have a PowerPC based MAC and you have dual-boot Ubuntu-OS X on it, you could run OS X inside Ubuntu using Mac-on-Linux (very cool thing). I am guessing that this is not the case.

2) You can Google "OS X under virtualization" and find out that some people have been somewhat successful in running OS X in VMWare/Xen/QEMU/PearPC and other products like that under Linux. Several caveats: 1) It breaks OS X License terms (Licensed only on Apple Hardware), 2) I don't know about the reliability of these solutions at the moment, 3) you would need to get OS X Installation Disks anyway and 4) you would need some serious technical expertise to try this.
