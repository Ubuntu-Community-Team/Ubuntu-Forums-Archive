### Post by sayakb on 2008-08-14
Open a terminal and type in:
```
sudo apt-get install edubuntu-desktop
```
Standard edubuntu uses Gnome itself. So you would have the extra edubuntu apps installed instead.

If you use KDE:
```
sudo apt-get install edubuntu-desktop-kde
```
