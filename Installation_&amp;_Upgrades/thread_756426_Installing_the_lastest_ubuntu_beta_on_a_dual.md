### Post by Existentialist on 2008-04-18
You should be able to boot from the windows cd, install to whichever partition you would like, then when that install is complete, boot from the ubuntu cd and install to the other partition.  Ubuntu will see the windows install and automatically set it up in the boot menu after installing so you can pick which OS to boot.
