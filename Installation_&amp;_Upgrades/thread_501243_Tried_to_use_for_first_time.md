### Post by aysiu on 2007-07-15
256 MB should be enough to run the Ubuntu Desktop CD as live session. It should also be enough to run a Ubuntu that's already been installed.

It is not, however, enough to run the Desktop CD's live session **and** install Ubuntu at the same time.

I would advise using the Alternate CD's installer instead.
