### Post by linux_tech on 2009-06-03
What are the specs?  Depending on how much memory you have you may consider an alternate install,  or possibly an alternate install of a older release. I was able to install 7.04 using alternate install on IBM T21 w/ 128MB RAM and after setting up SWAP, it works ok.
[http://old-releases.ubuntu.com/releases/feisty/](http://old-releases.ubuntu.com/releases/feisty/)
[https://help.ubuntu.com/community/Installation/LowMemorySystems](https://help.ubuntu.com/community/Installation/LowMemorySystems)

You can also check out Xubuntu.
[http://www.xubuntu.org/](http://www.xubuntu.org/)

### Post by snowpine on 2009-06-03
If you have a working OS already installed, you can do a hard drive install using Unetbootin: [http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

However, you should post your specs to determine whether the computer is a good candidate for Ubuntu. I think you would be better off with a lightweight distro like Puppy.
