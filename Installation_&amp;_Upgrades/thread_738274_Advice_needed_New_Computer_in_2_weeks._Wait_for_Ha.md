### Post by ShodanjoDM on 2008-03-28
Just install Gutsy but then don't customize or tweak it too much, or installing unsupported softwares from external sources to minimize the risk of breaking when you upgrade. Oh and you might want to create a dedicated partition exclusively for your /home folder. That way, your files will stay safe if the upgrade fails.

As for myself, I've tweaked my Ubuntu Studio 7.10 too much now that I'm not sure it will upgrade smoothly. Might as well doing a clean install when 8.04 is finally released.
