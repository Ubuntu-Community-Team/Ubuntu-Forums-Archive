### Post by Kevbert on 2009-06-12
If you want to update to a newer version, you need to go to System-Admin-Software Sources and under the Updates Tab set for Normal releases (instead of long term releases). Now go to System-Admin-Update Manager and you should now be able to update to 8.10 (and then 9.04). Don't forget to backup everything first in case of problems. It's probably better to do a clean install of 9.04 and then add all your applications/data afterwards.

### Post by merlinus on 2009-06-12
Upgrading can always lead to problems.  I have found that doing a fresh install of new versions works best.

I have a separate /home partition which does not get changed during new or reinstalls.  You may wish to do the same for data, and it also has configurations and settings for most of your apps.
