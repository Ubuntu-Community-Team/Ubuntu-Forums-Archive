### Post by drpjkurian on 2010-02-23
Hi
If you have installed Ubuntu in a new partition, follow the following steps
1.Boot from Live CD
2.Select the option which reads "Open without any change to your computer"
3.Open partition editor from 'administration' and delete the partition which is mentioned as ext3
4.Install Ubuntu again

Hope you have understood the steps
Do not touch the partition which is mentioned as NTFS, they are windows
