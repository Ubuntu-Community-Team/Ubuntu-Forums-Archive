### Post by tgalati4 on 2007-12-08
Go into your computer BIOS and set the "Allow USB boot devices"

Then boot from your USB device.  It should get you into a "Live" version of Ubuntu Studio.

Play with it for a while.  If you like it.  Install it to a hard drive.

If you are looking for a USB-only solution, then try Damn Small Linux or Dynebolic.  They are better configured for running from USB and saving a "Nest" file with your data.

### Post by Drunky on 2007-12-08
I have a few suggestions.

A) You could burn the iso to a disk and then reboot the computer with the disk in and let the live-cd start up.  If you enjoy it then install it.  See [this]("http://ubuntuforums.org/showpost.php?p=3882020&postcount=6") post about which program is probably best to burn the iso image.

B) Upgrade from Ubuntu 7.10 Gusty to Ubuntu Studio 7.10.  Use [this]("https://help.ubuntu.com/community/UbuntuStudio/UpgradingFromGutsy") Ubuntu Help wiki page to upgrade.  I used this method and did not have any issues.



I do not have any experience with booting from USB but I do know that some "older" machines do not have this optiion.  Maybe a bios update would help *shrug*.
