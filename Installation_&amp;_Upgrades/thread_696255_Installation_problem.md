### Post by digitalslavery on 2008-02-13
Welcome to the family...

You need to select the drive/partition from the list that you want to use for the OS and click Edit, from the list of drop downs where it says the drive name /sda4 (or what ever yours says) change it to a / (forward slash) that is the root.

good luck!
