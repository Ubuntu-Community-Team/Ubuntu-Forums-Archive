### Post by Cheesemill on 2012-11-13
Try downloading using a bittorrent client, this way the download will automatically resume if there are any problems. Also using torrents the downloaded is checksummed so you can be sure that your downloaded file is 100% correct.

[https://help.ubuntu.com/community/BitTorrent](https://help.ubuntu.com/community/BitTorrent)
[http://releases.ubuntu.com/quantal/](http://releases.ubuntu.com/quantal/)
