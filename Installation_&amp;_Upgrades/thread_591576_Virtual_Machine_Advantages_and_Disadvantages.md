### Post by Shazaam on 2007-10-25
Advantages:
Small size ( I have a virtual multi-boot thats 12 gigs w/5 os's)
NO physical hard drive access (unless you enable it)
Almost zero hardware problems (see below)
When you are tired of it (or trash it beyond repair) just delete it (virtual machine).

Disadvantages:
Slim to no 3d support
Setting up networking can sometimes be a pain (but is solvable)
Sound support sometimes is a problem

A virtual machine is fun. You can tweak, install, uninstall, partition it to your hearts content without bothering the host system at all. As far as networking I have the virtual machines set up with "nat" that lets me surf the net from the vm.
