### Post by snek on 2010-08-31
Considering how much Ubuntu relies on PPA's these days I am a bit surprised to find that there's no easy way to determine if a PPA is still live, and if not, be asked to remove it.

The new VLC version, for example, was released under a new PPA and the c-korn one has stopped to exist. Now, when doing apt updates I always get a warning that one of my repo's doens't exist anymore, but it doesn't say which one. Now, I know it's the c-korn PPA, but if I didn't I would have to guess at which of my 20 or so repos is actually dead.

We need a better way of managing this!

Mind you I only do this command-line, if it already exists in Synaptic I am not aware of it. Either way we need to be able to do it command-line too..
