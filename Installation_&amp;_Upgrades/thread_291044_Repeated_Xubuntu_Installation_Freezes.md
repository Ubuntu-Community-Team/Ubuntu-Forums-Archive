### Post by K.Mandla on 2006-11-02
See if you can access the terminal screen for the installation process, and see what's holding things up. Try pressing a combination of ALT and F1, F2, F3 (I forget which one it is), etc., until you get a log screen for the installation.

If that doesn't give you any ideas, try installing a server and upgrading to the desktop of your choice. (*sudo aptitude install xubuntu-desktop*, I'm guessing.)

(I've had some quirky behavior with Xubuntu installations under Edgy. Unless you're willing to run the risk, I would recommend sticking with Dapper.)
