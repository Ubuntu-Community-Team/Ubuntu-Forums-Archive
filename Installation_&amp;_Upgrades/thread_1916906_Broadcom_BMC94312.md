### Post by KatDawg on 2012-01-29
OK...I hope this might just help someone else in the future.

I was installing openSUSE over Ubuntu because it ran a little slow in VMbox. (Im useing SUSE in school). After the install...I could not get the network cards to work WIFI or wired.

So I decided to reload Ubuntu. Now I have the same problem! So I dig in.

I was getting the missing Firmware message....

So I get on Ubuntu Documentation and read the WifiDocsDriverbmc43xx page and start going through the steps...no luck. I check other sources...which were all pretty much the same. After several reinstall of different versions..etc..etc. I start to thing that its a hardware problem.

I check the patch cable...reset the Bios...reset the wireless adapter.

Nothing.

Then just for grins I decided to just reset the hub. AND THAT WAS IT!

Strange how that caused the errors.
