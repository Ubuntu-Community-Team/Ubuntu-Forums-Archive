I updated lucid this morning without problem, but that point goes to the problem with the original post.

You should probably tell us what release you are using. I know error messages can be difficult to capture, but it can really help with troubleshooting. There are various log files which are now viewable in recent releases via System, Administration, Log File Viewer. Updates are often recorded in dpkg.log

Note: You can also add the release you are using in your profile, so if you forget to include that information in the post us 'helpers' can check beneath your avatar. You enter that information via the User CP, Your Profile, Edit Your Details.

Hope the error isn't causing too many problems. If you need help fixing any that arose from the update try to give us some more information and we'll help.
