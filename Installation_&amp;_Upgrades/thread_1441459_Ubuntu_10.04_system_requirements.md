### Post by zvacet on 2010-03-29
According to [this]("https://help.ubuntu.com/community/Installation/SystemRequirements") yes you can upgrade.But maybe you would like to switch to Xubuntu.If you want to do that firat remove ubuntu-desktop with 

```
sudo tasksel remove ubuntu-desktop
```

and install xubuntu-desktop

```
sudo tasksel install xubuntu-desktop
```
