### Post by Mark Phelps on 2010-10-17
And, in future, do NOT upgrade Wubi versions.  Wubi is intended for trial use and not for extended use.  The Release Notes for 10.10 clearly indicated that upgrading Wubi is a problem.

Even if you decide to move to a separate installation of 10.10, in future, be sure to read the Release Notes for issues BEFORE you do an upgrade.  Can save yourself a lot of grief and problems as a result.
