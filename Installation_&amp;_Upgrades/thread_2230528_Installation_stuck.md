Welcome to the Ubuntu Forums :-)

With Vista, your computer might be middle-age or old. This means that you might be better off with the medium-light desktop environment of ***Xubuntu*** rather than standard Ubuntu. Or there might be problems with the drivers for graphics or wifi.

Please specify your computer

- Brand name and model
- CPU
- RAM (size)
- graphics chip/card
- wifi chip/card

Also, please give the output of the command in post #2 to *LastDino*, if you can run a live session 'Try Ubuntu' and start a terminal window.
