[COLOR=Sienna]This is for all who have slow/no internet connection.[/COLOR]

[COLOR=Sienna][SIZE=4][FONT=Times New Roman]I have experienced that Installing software are _really _troublesome in Ubuntu when you have slow or no internet connection. 

When I studied more about Ubuntu I learned that there are *.deb packages which are found in Ubuntu software repository. So that one could download the *.deb files from others computer and bring it to his computer by using 'removable storage'. But it is often found that the *.deb file needs more file which are called 'dependencies'.

Thinking about this problem I relized that it could be solved by providing *.UIA (Ubuntu Installer Archives) files which could be regular archive-files with the *.deb file and its dependencies. If it is done then people can easily install the softwares by only downloading *.UIA files.
