### Post by AlephN on 2007-08-26
> 
Anyway, during the install attempt, the install screen is too large and goes off the bottom of my monitor.

Hi. I just installed Ubuntu 7.04 a few hours ago, never used Linux before that. I came across this problem as well. (The default resolution was 800 x 600, nothing higher was available) Figured there was supposed to be some kind of Next button hidden below the screen, so I tried to raise the window, but a lot of approaches didn't work. (e.g. normal click-dragging, resizing the window, clicking on the button at the top left corner of the window and selecting move) After exploring around, I eventually found a 'Window Properties' screen, which says that you can hold down Alt and click-drag to move windows. That did the trick.

This solution was certainly far from obvious for a new user like me, and the problem looks like something that will occur for many installations.
