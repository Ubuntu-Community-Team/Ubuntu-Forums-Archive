Click [http://ubuntuforums.org/private.php](http://ubuntuforums.org/private.php) (under the "User CP" link).

As for the ISO problem, the Ubuntu i386 ISO will fit on any 700 MB CD.  Do you have an older CD without this capacity (I remember there being 650MB ones when CD burners first became cheap enough for personal use, but I haven't seen anything smaller than 700 in a long time)?  Make sure you have a standard 700MB blank CD.

Also make sure you have proper software to burn ISO images to CD.  Windows XP by default (don't know about Vista) doesn't know how to create CD's from ISO images; if you're trying to use Windows' built-in software, it won't work...you might be able to burn the ISO file to a disk, but this isn't the same as creating a disk from the ISO file.  ISO recorder [http://isorecorder.alexfeinman.com/isorecorder.htm](http://isorecorder.alexfeinman.com/isorecorder.htm) is a free and easy way to create an Ubuntu CD in XP.
