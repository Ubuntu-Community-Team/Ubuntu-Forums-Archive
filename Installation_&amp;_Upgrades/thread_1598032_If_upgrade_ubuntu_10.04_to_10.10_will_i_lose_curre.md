Somethings will, I did noticed that things like Compiz will be lost.  And because this is a new version, you would need to install the appropriate version.  But, if you are talking about desktop wallpaper, colors, etc, those can be backed up, and this can be tricky, it can be done. just copy the hidden folders "./" on your home folder.

There is a catch with this though, some programs, in the new version, could remember your old settings which might conflict with the new system.  In this case, just delete the folder of the application creating the conflict.
