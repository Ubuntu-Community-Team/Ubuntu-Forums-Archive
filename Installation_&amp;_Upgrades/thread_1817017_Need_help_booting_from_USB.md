### Post by lmarmisa on 2011-08-02
Welcome to the forums. :)

I suppose that you have downloaded the iso image file **ubuntu-11.04-desktop-i386.iso**. Try to check its [md5sum]("https://help.ubuntu.com/community/UbuntuHashes"). It should be **8b1085bed498b82ef1485ef19074c281**. If you have problem in order to calculate the md5sum of a file in XP, I recommend the HashTab tool ( [http://implbits.com/HashTab.aspx](http://implbits.com/HashTab.aspx) ).

Once you have verified that your iso image is good, try to create your Ubuntu Live USB using the tool UNetbootin:

 [http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

Best regards,

Luis
