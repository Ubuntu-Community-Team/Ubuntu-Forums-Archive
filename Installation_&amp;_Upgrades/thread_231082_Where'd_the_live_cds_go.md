> I don't follow the ubuntu scene so I'm unsure why I can't find live cds for Dapper Drake. 
The live cd is now the Desktop cd.  It contains an installer to install ubuntu by copying the live cd session back onto a hard disk.

> **mevets said:**
> Are alt install versions the same? I feel in love with running ubuntu over vmware and wanted to upgrade up from breezy badger.

Use the update-notifier to upgrade.  Click on the "upgrade to new version" or whatever it's called thing.  It will work well for you - it tackles common pitfalls such as the de-installation of the ubuntu-desktop package.  Unless you have no network access - you cannot upgrade using the Desktop cd - you would use the alternate cd for that.
