### Post by mrhhug on 2010-09-25
you do not HAVE to reload your system get some imaging software and just image your old hard drive to your new.

if you still want to dual boot, install windows first (like before) then when you install ubuntu, i think it asks you if you want grub to have ubuntu to be primary boot, if not grub defaults are easy enough to change.

you do not HAVE to reload - the only real issue is usually connecting both hard drives at the same time - i invested in a SATA to USB years ago and used it regularly for fixing windows viruses, backup of data , hard drive imaging... these are great [IMG]http://www.asia.ru/images/target/photo/50922846/USB_to_IDE___SATA_Cable.jpg[/IMG] if you plan to fix hardware ever invest in one
