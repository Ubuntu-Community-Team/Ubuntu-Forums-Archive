### Post by CelticWarrior on 2020-02-10
Install a newer release like 19.10 that can be easily update to 20.04 LTS.
Make sure installation is in UEFI mode.
Install the Nvidia drivers as recommended in Additional drivers. If it doesn't boot to the desktop with the default open-source driver, add 'nomodeset' to the boot parameters and then install the Nvidia drivers.

If further troubleshooting is required, including transitioning from Unity to the current Gnome DE, it should be done from an up-to-date OS, not form something like 16.04 (support ends in 2021).
