### Post by dave9191 on 2005-05-17
(K)ubuntu is a really fast quick 1 CD install, so there is no harm in trying that out after you get rid of FC3. Its a really nice fast and smooth distro. My main reason for using it over FC, is that everything on the computer worked out of the box. Including my centrino wireless :D

You would still need to install the nVidia drivers, download codecs for the media players and probably config X. As for the wireless it depends what card you got. If you are currently using an ndiswrapper for the driver, you would probably still need to use that. But you never know till you try :)

### Post by az on 2005-05-17
"would i still need to configure the X to have dual/clone view and install video players and the nvidia drives?"
You will need to install the non-free codecs and stuff from online repositories.  You need to enable those repositories with synaptic (tha package manager)  Should take you five minutes.  Nvida drivers are included.  I do not know about Xinerama.


"would i still need to install ndiswrapper to have my wireless work and change my kernel to stk16?"

What is stk16?  You get a 2.6.10 kernel and ndiswrapper is included.  Many wireless drivers are included so your card may work out of the box.

Ubuntu is debian-based and is trying to become a wonderful comunity which is ideal for making a useable linux desktop.  It is also free as in beer and free as in liberty.
