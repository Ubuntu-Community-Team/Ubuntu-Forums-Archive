### Post by oldfred on 2019-12-13
My old system, did not show bootable USB flash drives under any USB setting in bootable devices.
Only by accident did I find that under hard drives was another boot option which was the flash drive.
Check hard drives and see if it has a sub-menu.

Also many SSD need firmware update.
