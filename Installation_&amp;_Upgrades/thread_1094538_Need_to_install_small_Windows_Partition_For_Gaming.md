### Post by Neo_The_User on 2009-03-12
Gparted, cfdisk, and fdisk are utilities you can use. There are many others but my favorite is cfdisk. You need to shrink the primary linux partition then make windows use free space. I recommend doing it the other way around though. Full install of windows first then install Ubuntu via livecd then move the slider. much easier that way for new users.
