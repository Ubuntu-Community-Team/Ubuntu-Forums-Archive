### Post by K.Mandla on 2006-08-19
You could give the desktop CD a shot, and see if it suits you. It will boot to a live environment and you can check things out. If you dig it, there's an installer built in.

If you know you want to install it, the alternate CD is generally a more straightforward way of making the change.

Sounds like you could use the 64-bit version, although I don't have a 64-bit machine, so I can't necessarily recommend it over the 32-bit.

And a lot of wifi cards will work automatically with Ubuntu, but ultimately it depends on your hardware. For most others there are options.
