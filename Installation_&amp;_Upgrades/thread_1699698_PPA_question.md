### Post by grahammechanical on 2011-03-04
If you do as An Sanct says you will see a list of all the PPAs that are being used on your system as other software sources. All you need to do is to un-tick each line. Note down the address so that you can restore the PPAs if you want to. If there are none listed there, then you have none.
