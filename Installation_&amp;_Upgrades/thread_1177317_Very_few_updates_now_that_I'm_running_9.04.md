### Post by Therion on 2009-06-03
Starting with Jaunty the Update Manager only notifies you* weekly* and that's IF there are non-security related updates available for downloading.  Security-related updates will still prompt you as soon as they're available for download but routine stuff is now checked for weekly.  

If you're doing *manual* checks for updates (as I do) and you're noticing very, very few being available all I can tell you is... So am I.
