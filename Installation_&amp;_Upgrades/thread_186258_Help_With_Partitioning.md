### Post by tonyr on 2006-06-02
If you are using Ubuntu(Gnome), try **gparted**.  I think it's under
System->Administration somewhere.  You can also use **qtparted**,
which is not installed by default, but can be used with either Ubuntu
or Kubuntu.
