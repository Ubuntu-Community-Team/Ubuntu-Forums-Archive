### Post by TheFu on 2021-11-07
True i386 CPUs haven't been supported for many years.  A fully supported i686 had support until after 18.04.
May want to try Debian, which has wider CPU support than Ubuntu.  I'm not saying that Ubuntu cannot/will not work, just that 32-bit doesn't have much life remaining in the Ubuntu world, so it probably isn't worth the effort on that platform.

There are other options for computers that are less expensive which will have better overall results, though spending **any** money for a curiosity might want to be avoided.  I suspect Lubuntu might have a 32-bit version.  [https://lubuntu.me/focal-released/](https://lubuntu.me/focal-released/) ... nope. They dropped support.  And their last supported 32-bit release is beyond EOL, so that isn't an option.
