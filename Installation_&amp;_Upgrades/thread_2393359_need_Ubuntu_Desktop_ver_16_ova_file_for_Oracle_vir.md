### Post by TheFu on 2018-06-02
Why not just grab the 16.04 ISO and install it yourself?  Takes 15 minutes. There are hundreds, maybe thousands, of youtube videos showing how.  The only real trick is to provide 1-2G of RAM and 128MB of video RAM to the VM.
Inside a VM, you probably want a lighter GUI, [https://www.ubuntu.com/download/flavours](https://www.ubuntu.com/download/flavours) has some options. The Mate, XFCE, and LXDE versions are commonly recommended.

After you do the install, you can "Export Appliance" to create an OVA.
