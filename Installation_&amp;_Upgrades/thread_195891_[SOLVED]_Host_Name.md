### Post by jimbren on 2006-06-13
The hostname is the name of your computer on the network.  It allows you to identify each computer in your network, if there is more than one.  It can be just about anything you like, though I think there might be a 32 character limit.  

I did my install on a laptop, and it didn't use my last name for the default install.  Instead it was laptop followed by some numeric characters.  My guess would be that it took your last name from whatever you put in when the installer asked you for your name.  Remember, it asks for either "your name" or "full name" (can't remember which), then your username.
