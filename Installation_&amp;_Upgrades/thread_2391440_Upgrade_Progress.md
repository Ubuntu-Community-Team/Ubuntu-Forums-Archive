### Post by Frogs Hair on 2018-05-09
How did you initiate the upgrade ? In the terminal there are specific stages including sources and package download , unpacking, and setup though there are packages setup as needed during the process. From the update manger GUI the steps are indicated the same way including the the cleanup phase at the end. The GUI has a arrow marked details which shows terminal activity. Don't use the computer during upgrade because it may freeze the process.
