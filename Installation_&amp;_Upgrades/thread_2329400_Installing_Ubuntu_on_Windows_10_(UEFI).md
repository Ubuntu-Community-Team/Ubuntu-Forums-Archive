### Post by oldfred on 2016-07-01
Unless a tablet with 32 bit UEFI, we have not seen systems that do not let you turn UEFI Secure boot off. But many systems call it "Windows" and "Other" and in notes somewhere you find if installing Windows 7 you have to use "Other" as 7 is not Secure boot.

Ubuntu will install with UEFI Secure Boot on.
