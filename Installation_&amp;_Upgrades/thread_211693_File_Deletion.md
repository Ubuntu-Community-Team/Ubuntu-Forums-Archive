### Post by scxtt on 2006-07-08
formatting a partition, where files already reside - will destroy those files ...

if you install ubuntu - over top of windows - windows is GONE, only to return if you REinstall it - but all files from a previous install of windows are GONE ... ubuntu won't sit overtop of windows just waiting to be removed ...

you can use (almost) any number of OSes - be it windows and ubutu or any other distro - as long as you give each of them their own (separate) partition ...
