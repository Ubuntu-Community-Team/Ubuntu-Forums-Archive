### Post by sekinto on 2009-04-06
Check the .iso files' MD5 hashes against the ones provided on the Ubuntu site to make sure there are no inconsistencies with the files. If there are I suggest using torrents, BitTorrent is much more reliable because files are downloaded in pieces and pieces are checked for inconsistencies.

If you confirmed you have a good .iso file then try buring it to optical media using CDBurnerXP on the lowest speed setting possible (probably 1x).

If all that fails you probably have a faulty drive.

### Post by Mark Phelps on 2009-04-06
Don't know what options you're selecting in Nero, but you should be choosing to burn from an image.  That is not the default selection.  The default will simply burn the ISO to a disk by copying that file to the disk.

You can download and use ImgBurn so at least you won't have to buy another CD/DVD burning program.
