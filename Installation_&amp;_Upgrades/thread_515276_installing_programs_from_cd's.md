### Post by RRS on 2007-08-01
Do you want to install Ubuntu on the same computer as Windows98 and dual boot or do you have another computer you plan to use?

You only need a seperate router if you want to have two different computers with internet access at the same time.

Using the assumption that the cd you're wondering about is the one that came with the Linksys eithernet card, you won't need it, Ubuntu should recognize and activate the card on it's own during the installation.

Simply have the computer connected physically the same as you have it for Win98 before begining the installation, it may ask if you'd like to confiqure the DHCP server, say yes.

Usually to test hardware it's best to run the live(Desktop) cd of Ubuntu before installing. If you can surf the web with FireFox from it you shouldn't have any problems after installing to your hardrive.

Hope this helps clarify things for you and welcome to Ubuntu :)
