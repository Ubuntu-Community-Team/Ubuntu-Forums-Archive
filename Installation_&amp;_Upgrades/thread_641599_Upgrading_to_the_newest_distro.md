### Post by jimbob on 2007-12-15
Anything is possible so no guarantees, but I have done it several times with no problems.

Export your Compiz settings to a file in your home directory so you can import them back again if things change.  Also save your /etc/X11/xorg.conf file to preserve your screen resolution just in case.
