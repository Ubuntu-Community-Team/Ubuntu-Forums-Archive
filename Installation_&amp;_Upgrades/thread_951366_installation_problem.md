### Post by Partyboi2 on 2008-10-18
You should be able to start grass from the terminal (Applications>Accessories>Terminal) then type
```
grass
``` if this starts grass then you know it is working. If you are wanting to add it to your Applications you can open up main menu (System>Preference>Main Menu) and choose a section you want it in by highlighting the section on the left and choosing "New Item" then add the details to the create launcher box that opens making sure that the "command" you add the command that launches it, which should be grass
