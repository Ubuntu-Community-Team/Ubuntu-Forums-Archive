### Post by sikander3786 on 2010-11-23
> dpkg: parse error, in file '/var/lib/dpkg/status' near line 1865 package 'initramfs-tools':
 field name `' must be followed by colon

This error suggests that the file name in line 1865 under /var/lib/dpkg/status should be followed by a colon i.e, :

You can edit that file by

```
gksudo gedit /var/lib/dpkg/status
```

Navigate to line 1865 and put a colon manually.

Save, close and try installing again.

Good Luck!
