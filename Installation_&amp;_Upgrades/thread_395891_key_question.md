### Post by Jose Catre-Vandis on 2007-03-28
It carries out the command after you have downloaded the file, saves typing it in on two lines.

It's called a pipe.

On UK keyboard it is the vertical two dashes  (upper case if you know what I mean) next to the SHIFT key on the left. So it looks like this:

   |
   |

OK?

### Post by yabbadabbadont on 2007-03-28
If you are talking about the pipe symbol, "|", it is above the backslash "\".  It is used to pipe (send) the output of the first command to the second command (the one after the symbol) as the standard input for the second command.
