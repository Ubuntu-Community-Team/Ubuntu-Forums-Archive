### Post by taurus on 2006-08-30
If you want to run an ftp server, you need to go with proftpd or vsftpd (I am sure there are others but I think those two are the most popular ftp servers running in Linux/Unix).  They are both available with apt-get/aptitude/synatic and you don't have to install both!  One should be good.  Then, just check out the docs from their websites to learn how to use it...
