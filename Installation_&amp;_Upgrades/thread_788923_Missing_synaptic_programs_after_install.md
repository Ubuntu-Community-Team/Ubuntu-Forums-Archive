### Post by DieB on 2008-05-10
not for all Applications (especially those from universe) there exist Menu entries, but they still exist.
Just start synaptic, check the package u installed select properties and take a look on the tab installed files. Files with */bin* inside their name are for normal executables. Pick the one you need, got to System-Preferences/Settings-Main Menu and add an entry for this application.

hope that helps
