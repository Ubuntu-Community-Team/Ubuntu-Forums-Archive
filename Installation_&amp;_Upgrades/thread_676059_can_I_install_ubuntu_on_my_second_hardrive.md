### Post by Shazaam on 2008-01-23
Here is a good guide for dual booting.....

[http://www.psychocats.net/ubuntu/installing](http://www.psychocats.net/ubuntu/installing)

What I have done is use one drive for Windows and the other drive for linux. On the linux drive I make an Extended partition taking up as much space as it can, which allows me to put as many logical partitions on it that I want. At one time I had 7 versions of linux installed at once on the Linux drive.
What a lot of people have started to do is use some kind of virtualization program (VMware, VirtualBox etc) to run multiple operating systems. Less chance of something trashing your system after an error (either software caused or by human mistakes). Uses up less space too. Might be a viable option for you. There is a section on the forums here for Virtualization that you might want to check out.
