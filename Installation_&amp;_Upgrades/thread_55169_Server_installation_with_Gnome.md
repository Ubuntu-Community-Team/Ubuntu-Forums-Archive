### Post by DJ_Max on 2005-08-07
A server install is for servers, no GUI stuff. Gnome will install a lot of needed software (you will end up doing a normal install). ubuntu-desktop is nothing but a meta-package, it basically grabs other packages.

Take a look here. [https://wiki.ubuntu.com/Installation/LowMemorySystems](https://wiki.ubuntu.com/Installation/LowMemorySystems)
