### Post by fng on 2004-11-08
```
$> sudo apt-get update
$> sudo apt-get install ubuntu-calendar
```

Now go to Computer -> Desktop Preferences -> Login Screen Setup
Click on the "Graphical Greeter Tab"
The theme should be there
Select it and save the settings

All done now :)
