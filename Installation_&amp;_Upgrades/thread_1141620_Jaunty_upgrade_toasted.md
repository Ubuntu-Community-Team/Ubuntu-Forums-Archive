### Post by catelee2u on 2009-04-28
FIXED!!!

I restarted in safe mode/repair mode from grub menu (I didn't notice it before)
Then told it to repair/reinstall packages when the window popped up to ask what to do next.
It took awhile checking and installing everything and then it started up as usual but lightning fast boot time. Hope Jaunty keeps so great, haven't tested it out yet for everything.

Hope it helps someone else.
