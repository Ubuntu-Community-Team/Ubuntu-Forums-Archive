### Post by darkod on 2012-05-24
If you have the ISO, you need to put the wubi.exe and the ISO file in a same folder. Then start the wubi.exe and it will use the ISO instead of downloading from the internet.

But the version of wubi.exe must be the same as the version of the ISO. If you are not sure about the wubi.exe you have, it's better to extract it from the ISO because it is also contained inside. That way it will definitely be the same version.
