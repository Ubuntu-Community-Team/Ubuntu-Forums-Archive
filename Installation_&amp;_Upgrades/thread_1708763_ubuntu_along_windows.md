### Post by TechSupportx86 on 2011-03-17
Well for starters the easiest way to dual-boot is to install Windows ***First*** so ubuntu installs the GRUB boot-loader over top of the windows loader.

Second option is to install XP on an empty partition (or better yet a spare drive) and then re-install GRUB, altho i have only read of people doing this, never done it myself (in other words, possible but it *can* be tricky)
