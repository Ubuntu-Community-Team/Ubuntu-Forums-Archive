### Post by Rocket2DMn on 2008-03-10
> **Jaubsmat said:**
> Many thanks for helping to clear that up.

Is there anything, however, that I should be aware of when selecting additional applications to install?

Not really.  Some programs like flash have been known to be difficult with 64 bit, but I hear it is getting better - there have certainly been fewer posts about it recently.  If you ever install drivers from a 3rd party source (like say, video card drivers from nvidia's website), that is when you need to pay attention.  If you ever have a question about a specific instance, post a new thread and somebody will help you with it quickly.

You will get better performance out of your beastly computer by using the 64 bit version :)
