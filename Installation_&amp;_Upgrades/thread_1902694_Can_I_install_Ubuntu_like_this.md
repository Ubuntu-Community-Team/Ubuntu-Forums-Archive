### Post by darkod on 2011-12-31
I am sorry but I don't understand your point. And you will keep using win7 only in safe mode?

If you plan to install proper dual boot, it doesn't matter if win7 is broken. It's better to shrink the win7 partition to make space for ubuntu (if you don't have unpartitioned space) from within win7 Disk Management. Not sure if you can do that in safe mode.

You can still shrink it using Gparted in ubuntu live mode.

If your computer can open a boot menu when starting up with F10 or F12, you might be able to boot from usb stick prepared for ubuntu install.
