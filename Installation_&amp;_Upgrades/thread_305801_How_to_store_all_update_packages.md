### Post by Phulerock on 2006-11-23
if you reinstall new Ubuntu, or install another machine, the updating take a long time if you have the slow internet connection. I have the easy way to store all update packages, you can upgrade many machines without using internet bandwidth.

Go to /var/cache/apt/archives
all these updating pack's in this dir'

save all of it and you can use it to another PC by re-copy to /~/archives folder and upgrade by Synaptic !!!
