### Post by Rocket2DMn on 2008-06-08
Ubuntu 6.06 Dapper Drake is a few years old and although it is a Long Term Support (LTS) release, since you are just getting started, I would suggest downloading and installing the latest version of Ubuntu, which is also an LTS release.  The latest version is 8.04 and is called Hardy Heron, you can get it here: [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

You should also have a look here - [https://help.ubuntu.com/community/ApacheMySQLPHP](https://help.ubuntu.com/community/ApacheMySQLPHP)

I'm not sure what the support for PHP6 is since it is not a stable release yet.  Also, you may get better support for LAMP in the Server Platforms forum - [http://ubuntuforums.org/forumdisplay.php?f=339](http://ubuntuforums.org/forumdisplay.php?f=339)

Good luck, and welcome to Ubuntu!
