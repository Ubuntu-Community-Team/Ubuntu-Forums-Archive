There are many way to get the source. You can get the mainline source from the git repository from kernel.org. This may be less useful for you and it may be missing some of the Ubuntu sauce.

[https://wiki.ubuntu.com/KernelTeam/GitKernelBuild](https://wiki.ubuntu.com/KernelTeam/GitKernelBuild)

You can also get tar files for specific kernels from kernel.org and that will also be missing some of the Ubuntu sauce.

[https://www.kernel.org/](https://www.kernel.org/)

You can use apt-get and this will contain any Ubuntu sauce (You'll need to have the source code sources enabled in /etc/apt/sources.list).

```
sudo apt-get source linux-image-$(uname -r)
```

or for a specific kernel

```
sudo apt-get source linux-image-3.13.0-32-generic
```

[https://help.ubuntu.com/community/Kernel/Compile](https://help.ubuntu.com/community/Kernel/Compile)

Kind regards
