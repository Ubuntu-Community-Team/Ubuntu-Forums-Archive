### Post by StarWarrior on 2009-04-24
The netbook version has a nice launcher with bigger icons.

[http://media.ubuntuusers.de/wiki/attachments/42/42/netbook-remix.png](http://media.ubuntuusers.de/wiki/attachments/42/42/netbook-remix.png)

You can install it to your current system by installing the package "ubuntu-netbook-remix" with synaptic.

You can then change to the netbook version in System -> Settings -> Change Desktop(? something like that, don't know the english name for it)

Apart from the different user interface, there is no difference to a normal desktop Ubuntu.
