### Post by mmasic on 2007-12-26
-> I tried to change screen resolution to 1280x1024@60 using 'System settings' and then everything started

I had the almost same problem.
I solved it with few lines in ~/.kde/share/config/startupconfig
My example:
...
# kcmrandrrc Display ApplyOnStartup false
kcmrandrrc_display_applyonstartup="true"
# kcmrandrrc [Screen0]
kcmrandrrc_screen0_width="1152"
kcmrandrrc_screen0_height="864"
kcmrandrrc_screen0_reflectx="false"
kcmrandrrc_screen0_reflecty="false"
kcmrandrrc_screen0_refresh="85"
kcmrandrrc_screen0_rotation="0"
