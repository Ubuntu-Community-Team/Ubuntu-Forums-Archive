### Post by Xian on 2006-03-15
[QUOTE=MNGS]If I put this install CD in, will it "see" Red Hat and ask me if I want to replace it...or should I just take the machine down the block to the computer shop and have them install it???[/QUOTE]

The installer by default will place ONLY Ubuntu on your hard drive, but it is more than capable of seeing other OS partitions. If you use the manual partitioner which is on the InstallCD you have opportunities to preserve, write over, ignore, or simply mount any current disk partitions. In this manner you may decide what to keep, mount, or discard.

Read [Partitioning and Mount Point Selection](http://archive.ubuntulinux.org/ubuntu/dists/breezy/main/installer-i386/current/doc/manual/en/ch06s03.html#di-partition)
