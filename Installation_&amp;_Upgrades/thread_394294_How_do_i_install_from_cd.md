### Post by tgunner on 2007-03-26
First off, go here and download the ISO file of the correct ubuntu for your computer. [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

If you need software to burn an ISO to a CD, use this software: [http://www.afterdawn.com/software/cdr_software/cdr_tools/imgburn.cfm](http://www.afterdawn.com/software/cdr_software/cdr_tools/imgburn.cfm)

Then, put the CD in your PC, and start it. Select the first option on the CD menu.

As long as you have 256 MB RAM, you should be at the desktop after a few minutes.

On the desktop, click the icon which says "Install"

Follow the steps and wait for it to install.

### Post by DR4545 on 2007-03-26
You'll want to go into your BIOS screen for your motherboard.  The vast majority of mobos on the market enter their BIOS by pressing delete while booting up.  From there, set your boot order to boot from CD first, then from the HDD.  Save and Exit from your BIOS, and it should boot from the CD.

If Delete doesn't work to enter your BIOS, other things to try are ESC, F1, F2, CTRL-ALT-ESC, CTRL-ALT-ENTER, CTRL-ALT-INS, CTRL-ALT-S
