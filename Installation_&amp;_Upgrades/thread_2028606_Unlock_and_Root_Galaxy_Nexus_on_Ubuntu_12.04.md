### Post by janz84 on 2012-07-23
Install Android SDK:
[https://help.ubuntu.com/community/AndroidSDK](https://help.ubuntu.com/community/AndroidSDK)

Then follow this guide:
[http://source.android.com/source/building-devices.html](http://source.android.com/source/building-devices.html)

With an unlocked bootloader you should be able to flash any rom you want and install cwm. Good Luck!
