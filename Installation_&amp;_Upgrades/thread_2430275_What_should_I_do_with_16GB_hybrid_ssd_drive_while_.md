### Post by jimmyrus on 2019-10-30
If the drive shows up as two different drives/partitions you should be able to install ubuntu to the 16gb partition and keep the 1tb for home. Boot will be faster and os will run faster. but I'd probably put things like /tmp and /var on the 1tb partition since those could get big. 16gb is enough for an install but updates may take some room. Personally I'd just buy a bigger ssd and use that drive for /home.
