### Post by mingus on 2005-05-31
Your start is with the Ubuntu documentation.  You should be familiar in general with the installation process, which includes handling your situation.  The Install Manual has a section on partitioning and dual-boot, and there is a dualboot howto in the wiki.  The Debian Install Manual elaborates.  Do the homework first, as there are decisions you need to make in advance and tasks you will need to be prepared for.  This is no different than if you were adding a second Windows installation to the disk.  It is really not that different, but unfortunately most users proceed before having the basics down, and soon regret that.

Far and away what is most important is that you have a backup of your system, or at least your important data files, before you begin.  Do not skip this step!  And however you make the backup, test that it is good.

Finally, understand how to recover Windows itself.  If you have XP or W2K and the install CD - or a recovery CD - this will probably do.  But don't find yourself unable to boot into anything and unable to restore Windows to be bootable.

Short cuts are paid for in the long run.

--mingus
