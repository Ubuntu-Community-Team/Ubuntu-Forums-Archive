### Post by Partyboi2 on 2008-04-07
You can install it by enabling universe repo under Software Sources (System>Admin>Software Sources) first tab and installing from synaptic or in a terminal
```
sudo apt-get install kdepim
```(Unless there is a reason you have, to need to compile it from source.)
