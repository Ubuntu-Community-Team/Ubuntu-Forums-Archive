### Post by dinub1 on 2007-10-05

> hey thanks for the info guys.
about that link i floowed it and i went on actions then brun image and i selected the extracted iso file an it jut explorers the file .
do i burn the zip file??

No. Follow the instructions on how to burn iso image onto disk from the Howto link which has been provided to you. You did not burn the iso image. The iso file is an archive file which contains all Ubuntu installation files and programs in a compact form. When you select the "burn iso image" option, the program extracts all components from the iso archive and burns/writes them to disk as discrete components and files. What you actually didwas to burn the iso file directly onto CD. It cannot work that way.
