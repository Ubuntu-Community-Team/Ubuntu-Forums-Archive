Hi.

Yesterday I upgraded from 8.04 to 8.10. After the upgrade I lost all contact with my network. I managed to solve the problem myself, it turned out that the /etc/network/interfaces file somehow had been overwritten so that there was no options for eth0. I updated the file manually and then it worked again.

This is the second PC that I upgrade, the first one went fine. The difference between those two was that the first one has static IP address and the second one, which gave the problem, gets its IP address dynamically through DHCP.

Has anybody else experienced this? Can it be some kind of bug?
