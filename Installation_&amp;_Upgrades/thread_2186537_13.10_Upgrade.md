### Post by fantab on 2013-11-07
Upgrade is necessary. 13.04 has support for only 9 months, and it ends in January 2014, after which you will be forced to upgrade.
If you don't like upgrading then you must install and stick with LTS [Long Term Support, 5years support] versions like, 12.04 and 14.04 (Due for release in April 2014).

If I were you I would upgrade my 13.04 to 13.10 and when 14.04 LTS releases I would install that.

[Upgrading to 13.10 from 13.04]("https://help.ubuntu.com/community/Upgrades").
