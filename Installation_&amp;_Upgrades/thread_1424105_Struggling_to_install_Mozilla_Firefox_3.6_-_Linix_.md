### Post by mkvnmtr on 2010-03-07
Go to the Ubuntuzilla site. When you read it you will find two commands that need to be pasted in the terminal. These commands will enable the repositories with the Mozilla build of Firefox. Then open synaptic package manager. Reload and put firefox in the search box. Look for moxilla-build-firefox and install it.
When you are asked for a password in the terminal just type your password. It will not show but press enter anyway.

So make sure the, uhm...shortcut (by the way *shortcut* is a windoze term, so uhm try not to use it ;)) is linking to the correct folder.
