### Post by grahammechanical on 2015-11-29
For your information. Installing Ubuntu works best if we have an internet connection. Which you do not have unless you can connect with ethernet. In this way drivers for WiFI can be downloaded during the installation process.

If you can connect by ethernet then you can go to Additional Drivers and that utility might offer a driver for those WiFi devices. So, you have to decide which problem you want help solving. Is it:

a) Getting a WiFi device working

or

b) Get the 14.04 live session to run and installing without an ethernet connection.
