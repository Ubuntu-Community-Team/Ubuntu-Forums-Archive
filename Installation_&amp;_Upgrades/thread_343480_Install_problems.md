### Post by taurus on 2007-01-21
Sometimes, the desktop version (LiveCD) has some technical difficulty with some graphic cards during boot.  Therefore, I think you should consider using the alternate CD.  Even though it's a text base installer, it's real easy to follow and you wouldn't miss anything that comes with the desktop CD.
