### Post by Mark Phelps on 2008-09-26
If you're not already using Vista, and even then, using Vista-specific versions of MS windows apps, there is no real value in installing Vista on your machine.

I've been using Vista daily for nearly two years -- so I know something about the learning curve in becoming comfortable with the Vista way of doing things.

I've been using Ubuntu since the early 7.04 days -- and I've become comfortable with the different ways of doing things using GNU/Linux apps.

I started experimenting with Ubuntu when my Vista machine crashed due to a driver "update" -- and wiped out a lot of my files.  That ticked me off bigtime and I used it as an excuse to "jump ship" and try something else.

It took a while to become comfortable with Ubuntu, not because the look and feel is different (after all, one windows-based UI is much like all the others), but because I switched over to using Linux apps to do my work.

Don't believe the hype about how "Wine" has solved all that by running your MS Windows apps in Linux.  Go to the Codeweavers site and look up the Windows apps you want to run.  Be sure to check the product version.  Don't be surprised if the ratings assigned to the products you want are Bronze or Garbage.  If that's the case, Wine will be a waste of time.

If you really don't want to leave MS Windows behind, then don't -- install Vista and live with its quirks.

But, if there's no compelling reason for you to learn the Vista way of doing things, then just go straight to Ubuntu.  You'll have to find new apps and learn how to use them, but hey, you had to do that when new to MS Windows as well.
