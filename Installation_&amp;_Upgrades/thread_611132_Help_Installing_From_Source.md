### Post by Shazaam on 2007-11-12
Depending on the file type (zip, gzip, tar, tar.gz, etc) you need to unpack the file. Once you have it unpacked there are three commands you have to remember:
!.   ```
./configure
```
2.  ```
make
```
3.  ```
make install
```

Go here for more help.....
