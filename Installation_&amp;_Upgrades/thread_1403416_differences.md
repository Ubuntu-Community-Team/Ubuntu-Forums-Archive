I've been wondering for a long time now. What is the difference between the daily and the daily-live versions on cdimage.ubuntu.com?

---

### Post by gmargo on 2010-02-10
"daily-live" contains a daily build of the **desktop** version, which is a "LiveCD" [(http://en.wikipedia.org/wiki/Live_CD]("http://en.wikipedia.org/wiki/Live_CD")) that can be used standalone or for installation.
"daily" contains a daily build of the **alternate** version, which is not a LiveCD; it's only for installation.

### Post by gmargo on 2010-02-11
Desktop version = graphical installation of desktop environment
Alternate version = text installation of desktop environment
Server version = text installation + blessedly no graphical nonsense.

I always install the server version myself; and install the graphical bits afterwards.  There doesn't seem to be a daily server build though.  But why use a daily build anyway, unless you're testing?
