### Post by ivanovnegro on 2011-05-20
First of all I would try a clean install rather as upgrading.
Second, burn a Live CD of the newest Ubuntu release and try it first to see if you dont get any issues with it, especially graphics drivers.
If you want to upgrade anyway after testing enough the Live CD go to the update manager, Settings-> set there to use the normal releases not only LTS, later will pop up the update manager with the new updates/upgrade to a newer install.
Backup all your data!!!
Because you are using 10.04 it will first upgrade to 10.10 and then to 11.04, could be a long process, it would be really easier just to reinstall after backing up your data and it could spend you time, pain and headaches.

Edit: Important, see the [release notes]("https://wiki.ubuntu.com/NattyNarwhal/ReleaseNotes")!
