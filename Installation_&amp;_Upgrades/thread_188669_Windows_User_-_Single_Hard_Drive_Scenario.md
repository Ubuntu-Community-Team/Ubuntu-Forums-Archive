### Post by rcarring on 2006-06-04
Most XP OEM built computers have one hard disk, partitioned into one or more drives. Some OEMs have a blind partition at the start of the drive for diags. This partition should not be deleted. If you give the installer permission to wipe the hard disk then you will lose this partition.

Some OEMS divide the disk into two drives: C: and D: with the dvd/cd rom on drive E:

If you have no data on D:, you can remove it, to create disk space, and then use the alternate cd to install Ubuntu.

To remove drive D: go to Computer Management in Control Panel and run the Disk Manager. Read the help file before doing anything so you know what to do.

Please follow the instructions posted elsewhere for installing Ubuntu. Print them out and read them.
