### Post by meng on 2007-01-09
You don't have write access with the Ubuntu LiveCD but you have read access, so perhaps you should copy rather than move the files. Of course, moving is "easier" in that you can see what's left that hasn't been moved yet, but I thought I'd point out the obvious in case you hadn't considered it.
