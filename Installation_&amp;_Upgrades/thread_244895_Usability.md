### Post by rwfaught on 2006-08-27
Hello Ubuntu community.  I have a few questions.  This is not intended to be a flammatory post, although surely it will invoke some heated responses.  I discovered Linux five years ago.  I spent hours getting my system to a point where it was usable.  Frustrating hours.  I was satisfied in the end, but I could not run the software I used most often, even under emulators that were not emulators, and eventually, I decided that Linux, as cool as it is, was not worth the time it takes to make things "just work". 

Recently, I decided to give it another go.  I installed Ubuntu, and realized quickly that I had forgotten so much about everything I had learned the first time around.  Installation is easy, and I had a workable system in well under an hour.  I didn't expect to get work done; everything I have read has made it clear that Adobe CS2 does not run on Linux.  But, for some reason I expected that this time I would not have to RTFM to get applications running.  I just wanted to try some things out. Quickly.  Unfortunately, I still can't just find an application online, download it, open the install, and have it do all the work for me.  Why? 

My problem isn't that I can't figure out how to install these applications.  My problem is that I don't have the time, or the patience if I had the time, to do it.  I wonder why it is still a problem.  Will this problem be fixed?  Am I missing something?  Isn't this a common complaint with newcomers to the OS? When can I expect an "it just works" solution to Linux?
