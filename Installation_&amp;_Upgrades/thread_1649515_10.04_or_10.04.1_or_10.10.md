### Post by sikander3786 on 2010-12-20
There is no direct path of upgrade from 8.04 > 10.10.

However there is a direct path of upgrade from 8.04 > 10.04.1.

You mentioned you had some problems with 10.04 Live CD but did you try the 10.04.1 Live CD? Probably, those problems might have been solved in the point release of Lucid i.e, 10.04.1. Test it!

8.04 wouldn't be supported after April 2011 on the desktop so you definitely need to upgrade or do a clean install with any of the current release then.

If 10.04.1 doesn't work for you, test 10.10 from a Live CD and if successful, do a clean install. Or upgrade from 8.04 > 10.04.1 > 10.10.

The problems with 10.04 might be resolvable. Can you please mention the problems you had with Lucid?
