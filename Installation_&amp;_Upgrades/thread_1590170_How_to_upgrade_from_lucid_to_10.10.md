### Post by sikander3786 on 2010-10-07
Alt + F2,

```
update-manager -d
```

Before upgrading, make sure that all the updates for Lucid have been installed. And you of course know that 10.10 is still RC.

---

### Post by Cavsfan on 2010-10-07
I did that method and it went flawlessly! I did not loose a single thing! :P
The only thing I would suggest is to go into software sources and untick everything except the top 2 sources.
Before I did that I kept getting errors. And it takes approximately 2 hours for the upgrade, but IMO well worth it!
