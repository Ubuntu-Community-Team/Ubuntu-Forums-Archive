### Post by jaylsi on 2009-11-30
You don't need to remove 3.5 in order to install 3.0.

1) Search firefox 3.0 download for linux
2) Download the zip file and install it to a separate location on your system
3) If needed, change the symbolic link /usr/bin/firefox to 3.0 version
