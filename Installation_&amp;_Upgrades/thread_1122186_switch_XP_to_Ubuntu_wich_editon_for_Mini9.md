### Post by tommcd on 2009-04-10
You can install Ubuntu 8.10 or Ubuntu 8.04 LTS. Ubuntu 9.04 will be out in less that 2 weeks and will be an option also. There is also the Ubuntu netbook remix, which you can upgrade to after installing Ubuntu or install directly. I would go with Ubuntu 8.10 for now.
Here are the guides for the Dell Mini 9 from the Ubuntu Wiki:
[https://help.ubuntu.com/community/DellMini9](https://help.ubuntu.com/community/DellMini9)
[https://help.ubuntu.com/community/DellMini9/Intrepid](https://help.ubuntu.com/community/DellMini9/Intrepid)
