### Post by snowpine on 2009-11-30
You should be able to transfer the hard drive from one computer to another no problem, with two exceptions:

1. Obviously you can't move an amd64 install to a 32-bit x86 computer
2. Best not to enable any restricted hardware drivers (like nvidia graphics cards) until the hard drive is in its final destination.

### Post by snowpine on 2009-11-30
Yes, Ubuntu installs are generic (with the exception of any restricted hardware drivers).

For example, I have Ubuntu installed to a USB thumb drive, which I use to boot several different computers. No problem.
