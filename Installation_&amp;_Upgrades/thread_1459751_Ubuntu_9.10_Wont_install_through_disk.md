### Post by presence1960 on 2010-04-21
Did you [MD5SUM]("https://help.ubuntu.com/community/HowToMD5SUM") the iso prior to burning as an image to CD? The hashes must be a perfect match or the iso is not good.

Did you burn the iso as an image at no more than 8x speed? Higher speeds are ok for data, music , etc., but not so good for a bootable image. If one bit of data is bad the image will not work properly.
