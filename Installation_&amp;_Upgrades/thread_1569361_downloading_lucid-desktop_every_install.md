### Post by bcbc on 2010-09-06
Just download the file yourself - bittorrent is the quickest method - and place it in the same folder as wubi.exe. Then wubi.exe will use it.

Or you could just burn the .iso to disk. Either method will prevent wubi from downloading each time. Just make sure the wubi.exe matches the .iso version (the current version of both is 10.04.1)
