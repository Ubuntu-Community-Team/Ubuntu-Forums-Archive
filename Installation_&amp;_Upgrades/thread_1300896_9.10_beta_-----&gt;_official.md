3 days from now, the servers will be tasked a lot with a lot of folks downloading Karmic.  My plan is to wait for about 3 weeks after before upgrading/trying out Karmic.  I also plan to use the torrents.

You can upgrade using the network.  Make sure your release upgrade setting (system > administration > software sources > update tab) is set to receive normal releases and not NEVER.

Make sure you check the release notes and upgrade notes for reference.  Here is the upgrade notes for  Jaunty which may give you early clues on what to do.

[http://www.ubuntu.com/getubuntu/upgrading](http://www.ubuntu.com/getubuntu/upgrading)

I have always prefered a fresh install.  For me, it gives me a chance to try the new version first in live session (burning a liveCD and selecting TRY UBUNTU WITHOUT ANY CHANGE) and see how the new version reacts/plays well with my current computer hardware.  I also do this because there is no known DOWNGRADE path except for a re-install.

Good luck.  Back-up your files first and run fsck.
