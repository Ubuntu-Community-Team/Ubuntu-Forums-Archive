### Post by Quackers on 2011-03-27
Welcome to UF.
Have you cheked the md5sum of the downloaded iso file?
[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

If that checks out, you should check the cd for errors. On booting from the cd if you get to the first purple screen press any key. Then choose language. On the next screen select the "check for errors" option. 
If your cd doesn't get to the first purple screen there is something wrong with the iso file or the burn to disc.
