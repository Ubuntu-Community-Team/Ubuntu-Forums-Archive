### Post by dstew on 2009-06-10
Do you want to use your server command line to get and send your personal email, or set up an email server that your users will access over the internet? If the former, you can use the [mutt email client]("http://www.mutt.org/"). If the latter, you need to set up the server, then your users can use Squirrelmail as a front-end webmail server. See [this page]("https://help.ubuntu.com/community/MailServer").
