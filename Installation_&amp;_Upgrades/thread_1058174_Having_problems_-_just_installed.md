### Post by cariboo on 2009-02-02
You don't have much space for a second OS, I just reinstalled Jaunty and it is taking up 3Gb of my root partition. You need to make more room. 

Defrag your Windows partition at least twice and then boot the LiveCD and use System-->Administration-->Partiton Editor to resize your Windows partition.

Windows programs will not work in Linux, There is a windows application layer called WIne that can be installed using Add/Remove Programs or System-->Aministration-->Synaptic Package Manager. Check [WineHQ]("http://www.winehq.org/") for program compatability.
