### Post by pak33m on 2006-06-08
If you have the install files on a USB drive you should be able to boot straight form it.  

If you know how to change the boot options in BIOS then you should be able to set to boot from the USB drive.  This all depends if your BIOS supports it.     

Go here: [http://www.weethet.nl/english/hardware_bootfromusbstick.php](http://www.weethet.nl/english/hardware_bootfromusbstick.php) and check out step 6.  It may or may not be what you need depending, again, on your BIOS.

Good luck and I hope this helps...
