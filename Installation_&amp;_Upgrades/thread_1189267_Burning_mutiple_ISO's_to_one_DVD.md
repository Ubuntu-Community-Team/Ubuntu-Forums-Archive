### Post by dstew on 2009-06-16
An .iso is a **disk image**, and contains a whole file system, so as far as I know, only one .iso can go on one disk. Possibly, one .iso can go into one partition. Possibly you could make separate partitions, with one .iso in each partition, but I don't know how to do that on a dvd, only a hard disk. And then, you would have to have a way to boot the separate partitions. Sounds like an interesting experiment.
