Looks like you need to run a HDD check. Open a terminal, and give the command```
sudo touch /forcefsck
``` Then reboot normally. Your computer should run an automatic harddisk check.

Post back any specific error messages if the problem does not clear up.

Note that there is a (tiny) possibility of data loss during the fsck run, so your do this at your own risk.
