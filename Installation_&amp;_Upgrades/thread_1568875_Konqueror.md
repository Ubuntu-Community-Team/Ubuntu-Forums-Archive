### Post by claracc on 2010-09-06
To use konqueror instead of nautilus, you can select system --> preferences --> main menu --> tools and there you select the file browser, using the command for running konqueror ( perhaps /usr/bin/konqueror, I aam not for sure because I use nautilus ).

By default, ubuntu has ufw firewall, to configure it you can install its gui : gufw.

About firestarter, this thread can help you [https://help.ubuntu.com/community/Firestarter](https://help.ubuntu.com/community/Firestarter)
