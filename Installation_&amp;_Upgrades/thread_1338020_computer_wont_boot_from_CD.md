### Post by Bartender on 2009-11-26
Quick way to check whether you copied the download (wrong) or converted it to a bootable CD (correct) is to fire up a Windows PC.  

Wait for Windows to come up.  Toss the CD in the optical drive, and Explore it.  If Windows sees one big file you did it wrong.  If it sees several folders and a handful of files, then we at least know you converted the download correctly and can move on to other ideas.
