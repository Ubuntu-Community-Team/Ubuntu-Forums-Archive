Whoa before taking that extreme route try the following one last time: 
In a terminal do the following:

cd /etc/apt
sudo mv sources.list sources.list.bak
sudo echo "" > sources.list
sudo touch sources.list
sudo apt-get update
sudo rm -rf sources.list
sudo mv sources.list.bak sources.list
sudo touch sources.list
sudo apt-get update
