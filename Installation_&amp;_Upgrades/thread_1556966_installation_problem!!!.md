### Post by TBABill on 2010-08-20
+1
 
Try running  Live (top option), which will boot from the CD and not actually install. This allows you to run the OS, although much slower, and see if all your hardware works with the OS. Verify your wireless device, sound and other items work properly to be confident you won't have compatibility issues. Some items may still require installation of a proprietary driver (usually video and wireless from my experience), but not always. Many drivers are supported directly via the kernel without you having to do any additional work for your devices to operate properly.
 
If it seems to run ok from the CD, then you can do as stated above and just double click "install Ubuntu" and follow the screen options to install.
