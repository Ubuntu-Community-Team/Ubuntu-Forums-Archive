### Post by cortman on 2012-06-14
+1 for the minimal CD. I use it myself for all my Ubuntu installations.

1. Depends on your DE- most will come with power managing programs.
2. Figure out which drivers you need by what doesn't work on your laptop, google, and install manually is how I've done it... There may be a better way but this helps keep things slim.
3. You can install the fonts just like any other font; right click on the file and select "install". Or you apt-get it [like this]("http://askubuntu.com/questions/24983/how-do-i-reinstall-default-fonts").
