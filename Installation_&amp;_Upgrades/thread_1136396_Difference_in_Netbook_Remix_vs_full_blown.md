### Post by enchantedsky on 2009-04-26
I installed Netbook remix on my 1000HE, and it's been a great experience so far. Programs make better use of 1000HE's small screen.

The only problems I've seen so far is when I tried adjusting Compiz-Fusion settings, it screws up the GUI. If you're really into Compiz and 3D desktop effects, it is better to install the regular Ubuntu (and yes, the 1000HE 950 Intel Graphics card can handle Compiz)

Also, when I muted the volume one day, there was a loud buzzing sound. However, when I tried it again yesterday, there was no loud buzzing sound. Perhaps with updates, they are actively correcting problems.

Anyway, you can actively switch from regular desktop to the Ubuntu remix interface, and vice versa.
