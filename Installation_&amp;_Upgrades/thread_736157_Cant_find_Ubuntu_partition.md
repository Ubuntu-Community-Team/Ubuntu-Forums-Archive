### Post by tubasoldier on 2008-03-26
Windows can not access linux partitions natively.

However there is a few different programs for windows that allow access to ext3 filesystems. Some solutions are as follows:

[http://www.diskinternals.com/linux-reader/](http://www.diskinternals.com/linux-reader/)  - Read Only Acess
[http://www.chrysocome.net/explore2fs](http://www.chrysocome.net/explore2fs)
[http://www.fs-driver.org/index.html](http://www.fs-driver.org/index.html)                - Read and Write Access. But I have always had issues with this one.
