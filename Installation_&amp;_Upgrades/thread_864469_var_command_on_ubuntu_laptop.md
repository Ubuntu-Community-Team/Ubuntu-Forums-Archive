### Post by ibutho on 2008-07-19
If you mean that you would like to run a terminal, go to Applications -> Accessories -> Terminal. Once the terminal emulator is running, you can enter any commands. You can also the access the terminal by doing CTRL-ALT-Fn (n is any number between 1 and 6). To go back to the gui, you would do CTRL-ALT-F7.
