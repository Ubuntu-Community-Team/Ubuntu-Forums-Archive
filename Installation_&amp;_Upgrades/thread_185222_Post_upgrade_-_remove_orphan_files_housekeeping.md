### Post by fsman on 2006-05-31
We are nearing release date so I thought that I would share a great utility for those are have been following the beta and upgrading

It´s gtkorphan

[http://www.marzocca.net/linux/gtkorphan.html]("http://www.marzocca.net/linux/gtkorphan.html")

Its a great tool to clean up all those lib´s or old programmes you don´t need anymore.

I´ve just used it to remove some all lib´s which have found their way into my system over the last few months of using the ´flights´. Now I have a clean system ready for tomorrow.

Actually I should run it post offical launch.

PS I followed this tool with an $apt-get dist-upgrade to ensure that I hadn´t accidentially removed files which were dependent on ubuntu-desktop
