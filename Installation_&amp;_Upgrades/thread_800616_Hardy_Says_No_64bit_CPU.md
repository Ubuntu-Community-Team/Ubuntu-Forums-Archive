### Post by sonofusion82 on 2008-05-20
Intel T2500 is a Core Duo, not Core2 Duo. [http://en.wikipedia.org/wiki/Intel_Core](http://en.wikipedia.org/wiki/Intel_Core)
It is a 32-bit only processor.

If you have MS Windows, to confirm, try this tool.
[http://www.grc.com/securable.htm](http://www.grc.com/securable.htm)

If it says no for 64-bit, then sorry, you are out of luck. Just stick with Ubuntu 32-bit. :)
