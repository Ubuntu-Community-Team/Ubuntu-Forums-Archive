### Post by Arby on 2007-03-27
Dapper is an LTS (Long Term Support) release. Feisty+1 refers to whatever ubuntu decide to name the release that follows feisty. There is some talk that Feisty +1 will be the next long term support release as opposed to the intervening Edgy and Feisty which were 'testing' releases. I think it's intended that there should be a smooth upgrade path from one LTS release to the next without having to go through the intervening releases. This is particularly aimed at people using Ubuntu in a corporate environment where they might only upgrade every 2-3 years when there is a new LTS release. Of course this doesn't mean that home users can't also do  that if they want to.

Hope that clarifies things
