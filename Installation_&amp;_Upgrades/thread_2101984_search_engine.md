### Post by houseworkshy on 2013-01-06
If in Firefox simply navigate to the bing homepage, then click "Edit" from the firefox menu, choose "preferances", this will open a box, in the "general" tab on that box, which is the first one, click "Use current page".  That's it.  
A plug in worth installing for Firefox is called "add to search bar".  I can't remember if one has to install it to get the feature I describe next. I put it in ages ago I forget what firefox looks like without it.  Certainly with it one can right click on any search box in any internet site and choose "add to search bar" which will add it to your list of search engines.
To use it look for a smaller text box, adjacent to the address bar,  which has a drop down v at the the beginning and a picture of binoculars at the end.  You can enter search terms in that and use the drop down to choose any search engine or site search you have added.  Clicking the binoculars starts the search.
