### Post by caillean on 2008-08-18
Hi,

to keep your settings for desktop environment and applications simply keep your home folder, if it's on a separate partition just mount it as /home during the new installation. If not, make a backup before installing, all the settings are in the hidden folders.

Concerning the software selection - I'm not sure about this. There's a tool called remastersys. It allows you to create a bootable CD of your personalized system. Never tried it myself, but I'm sure you'll find some how-to's on the web...

Hope this helps a bit,
