### Post by johnson.d on 2010-02-17
For adding the language, You can use the following steps:
1) System-> Administration-> Keyboard-> Layout
2) Add -> By Language -> Language -> Bulgarian -> Add
Now Bulgarian gets added to choice of languages

Now for Choosing the language using key combinations:
1) In the same window click layout options
2) Drop down "Key(s) to change layout" 
3) select the option "Both shift keys together" (There are many combinations that you can choose from, this is one that found convenient and working in all the apps that I have tested.)
4) Close the window
5) Close the main window

Now, in any application, including your browsers, you can press the two ( right and left ) shift keys together to toggle between English and Bulgarian. Some key combinations may not work because they may be captured by the application you are using.
