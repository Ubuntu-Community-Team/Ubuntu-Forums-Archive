### Post by daenoid on 2009-12-17
You would probably just setup a virtual machine inside of Ubuntu in order to run Windows 7. It's pretty straight forward so try: [http://www.virtualbox.org/](http://www.virtualbox.org/)

As for the second question, you could probably start up your virtual machine and have it running on one of the desktop workspaces. You probably won't get the extra compiz effects with the machine though, but the cube/vm should work.
