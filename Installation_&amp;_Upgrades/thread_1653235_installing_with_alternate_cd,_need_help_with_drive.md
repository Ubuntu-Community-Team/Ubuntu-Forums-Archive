---

### Post by Mark Phelps on 2010-12-27
If you get to a graphical desktop after the installation, you are using the default open-source Nvidia drivers and don't have to do anything.

But if you then want to install the restricted Nvidia drivers, check System --> Administration --> Additional Drivers.  IT should then offer you the Nividia drivers for you to install.
