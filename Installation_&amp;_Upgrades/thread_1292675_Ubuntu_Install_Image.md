### Post by tommcd on 2009-10-16
> **maheshkuruganti said:**
> Hi,
   Is there a way by which I can create a Install CD without Live CD using my Hard Disk. 
Do you want to burn a CD from the .iso image on your hard drive? If that is so, then it is simple. Simply right-click on the .iso image and choose "write to disc". Be sure to choose the slowest possible speed. And when you boot the CD on the other computer be sure to first choose the option "check disc for defects" to be sure the CD is good.
