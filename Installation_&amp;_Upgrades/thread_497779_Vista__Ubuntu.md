### Post by wpshooter on 2007-07-10
First of all I would not recommend that you install 5.04 (it is a very old version of Ubuntu).

You should instead download, burn and attempt to install either Dapper 6.06.1 or Edgy 6.10.

You normally do not install this O/S thru M/S windows, but once you have burn the ISO image of Ubuntu to a CD the CD is a bootable CD and it will boot into Ubuntu if you have your CD as the first bootable device on your system.

Good luck.

P. S. - Be carefully, it is not impossible to ruin your M/S Vista installation !!!

### Post by EvilMarshmallow on 2007-07-10
I object:  Ubuntu's not a hobby... it's a lifestyle! :lolflag:

Indeed, be very careful installing this stuff.  And since you're using a new Dell, beware: wireless networking may not work right out of the box because Dells are fond of Broadcom network chipsets.  There's a great tutorial here on ubuntuforums (Search for "Broadcom") on how to do that, once you've installed the system.

You'll probably want a wired network connection attached to your laptop until you successfully complete that tutorial... and be prepared, it's a big learning curve.  I've been at it since Christmas and I'm only now getting to the point that I'm comfortable with it.
