### Post by JRV on 2011-07-02
Look at your grub menu.

Count the number of lines down to your Windows option. (The first line is zero.)

Open a terminal and type the command:
```
sudo gedit /etc/default/grub
```

Change "GRUB_DEFAULT" to the value obtained above.

In the terminal run this command:
```
sudo update-grub
```

### Post by YesWeCan on 2011-07-02
The other thing you could do is to put the Windows menu entry as the first entry in the list.
In /etc/grub.d just rename 30_os-prober to have a number lower than 10_linux, such as 09_os-prober.

Run sudo update-grub afterwards.
