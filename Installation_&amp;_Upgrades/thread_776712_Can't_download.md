### Post by Ex-Cyber on 2008-04-30
The US Ubuntu server does seem to be having problems. You can change to a different server in the Software Sources dialog (the "Download from" option). I don't recommend doing this unless you are on broadband, though, as you would have to refresh the package index (a 4MB+ file).
