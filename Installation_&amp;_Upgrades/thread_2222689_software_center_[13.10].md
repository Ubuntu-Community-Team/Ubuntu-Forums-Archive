### Post by kc1di on 2014-05-07
it looks like you have a bad ppa installed. you need to remove it and then it should work.
it's this one > [http://ppa.launchpad.net/ubuntu-audi...-i386/Packages](http://ppa.launchpad.net/ubuntu-audi...-i386/Packages) 

use this command in the terminal to remove it. Changing the waterver/ppa to the one above. ```
sudo add-apt-repository --remove ppa:whatever/ppa
```
then run ```
sudo apt-get update && apt-get upgrade
```
