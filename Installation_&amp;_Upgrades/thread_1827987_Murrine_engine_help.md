Hello and Welcome

I think the Murrine engine is installed by default on 11.04 . You can check in the synaptic package manager with the search filter .  If it is not installed , right click , mark for installation , and select apply . If you choose to use the terminal .```
sudo apt-get install gtk2-engines-murrine
```
