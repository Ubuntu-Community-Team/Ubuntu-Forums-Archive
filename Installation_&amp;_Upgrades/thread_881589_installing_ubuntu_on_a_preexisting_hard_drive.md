### Post by mahela007 on 2008-08-06
My HD is already partitioned into 3 ntfs file systems.
How do I install ubuntu on one these partitions?

---

### Post by hermes0710 on 2008-08-06
You can't install ubuntu on ntfs. One of the partitions must be formatted (during the installation) in order to become linux partition. You just boot from the cd and you will be prompted to select in which hd partition you want ubuntu installed.

### Post by hermes0710 on 2008-08-06
