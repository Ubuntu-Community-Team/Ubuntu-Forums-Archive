Apart from having financial problems it is not clear what your Ubuntu problem is. Please explain what "live patch" is and how it relates to Firefox?

Are you expecting Firefox to be updated and you are not seeing it happen? On Ubuntu 22.04 Firefox is now a Snap package application. Every time we run Software Updater and the utility updates/upgrades some application/packages Software Updater will also automatically check for updates to snap packaged applications and install those updates. Firefox as a snap package will get updates without the user knowing it is being updated.
