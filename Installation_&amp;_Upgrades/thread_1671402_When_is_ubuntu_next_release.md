### Post by tommcd on 2011-01-20
Ubuntu releases new versions every 6 months, in April and October. The next version of Ubuntu, which will be 11.04, will be released in April 2011.
The version 11.04 means that it is released in 2011 in the 4th month, which is April. Likewise, Ubuntu 10.10 was released in the 10th month (October) of 2010.
[https://wiki.ubuntu.com/Releases](https://wiki.ubuntu.com/Releases)
