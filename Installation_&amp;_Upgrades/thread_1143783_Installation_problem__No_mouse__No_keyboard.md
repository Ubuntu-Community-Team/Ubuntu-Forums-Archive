### Post by Mark Phelps on 2009-04-30
Sorry, don't know WHY this is happening, and according to recent posts, this appears to be happening a lot.

But ... when it happened to me, using a USB mouse and keyboard, I did the following:
1) unplugged them from a hub and plugged them directly into the REAR USB ports on the machine
2) Enabled Legacy USB support in the BIOS (previously had only USB 2.0 support enabled).

Those fixed my problems.
