### Post by wolfear on 2006-12-20
You can install the gnome (ubuntu) then, using synaptic, select to install the KDE (kubuntu) desktop.
I like the features of kubuntu, but the package management with synaptic just is more appealing to me (personal thing) which is why I would do gnome first.

You can theoretically have all the different desktops installed and just switch between them.

Depending upon how much data you plan to have, you could to your main install on the smaller drive using the recommended defaults and then set up a larger partition scheme on the second drive and move your "home" to that.
Then you have all the space you need for application to install on the main drive without losing storage for your settings and personal data. I have older, very small HDs (16 gigs between 2 drives) so this is always a concern.
Added advantage is if something goes haywire on your primary HD, you can salvage without losing all your data.
