The server isntall is just a command line, It appears it installed correctly, It just didnt install the version that you wanted.

I never read the download page before now, The desktop cd says

> The desktop CD allows you to try Ubuntu without changing your computer at all, and at your option to install it permanently later. This type of CD is what most people will want to use. You will need at least 192MB of RAM to install from this CD 
what will happen with that cd is this
1. Download and burn
2. Start the computer with cd in drive and it will run a full gui direcltly from cd
3. Use the version running of the cd to make sure your hardware works and you like it
4. If you like it their is a option to install the full ver + gui to your hard drive so the cd isnt needed.


The server version is just the installer that will install the server edition without a gui. 

The alternate install should look just like the server version while installing but when done you have a gui.

I would reccomend downloading the desktop cd and using it, dont worry, you will be able to permanently install it if you choose.
