### Post by Mark Phelps on 2012-01-27
Need to know what graphics adapter you're using ...

To find out, if you're not getting graphics at all, you will need to boot from an Ubuntu desktop CD.

Once in Ubuntu, open a terminal and enter "lspci" (without the quotes).  Look for the line containing "VGA" -- and report the results back here.  That will list the make and model of your graphics chipset.
