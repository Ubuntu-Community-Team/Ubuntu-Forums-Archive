### Post by drs305 on 2012-06-10
Yes you can do it this way if desired. If you want to end up at 12.04 though that is a lot of upgrades, each of which could fail.

I'd recommend waiting for the 12.04.1 release to come out in a month or so. The LTS upgrade will then show up in Update Manager (LTS) and you can do one upgrade to the next LTS. 

Either way you go, I'd recommend making an image of your current OS prior to the upgrade so you can revert should things not work out to your liking.
