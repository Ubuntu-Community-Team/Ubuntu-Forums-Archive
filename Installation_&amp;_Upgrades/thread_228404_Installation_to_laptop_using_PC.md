### Post by rattlerviper on 2006-08-03
Check out this thread [http://www.ubuntuforums.org/showthread.php?t=226879]("http://www.ubuntuforums.org/showthread.php?t=226879")
I think what you really want to do is install via network if you have a network card.  It's as simple as plugging in your laptop to the network and using 3 floppy disks that you download, follow the directions and you will have Debian Sarge on  your laptop.  If you would like to install Ubuntu there is a method to do so utilizing the same 3 disks and a debootstrap to change the install over to Ubuntu.  The link to thos instruction is [https://help.ubuntu.com/community/Installation/WithFloppies]("https://help.ubuntu.com/community/Installation/WithFloppies")
If you are able to follow the instructions better than I you should have luck installing Ubuntu that way(I tried Xubuntu, but failed and just install debian without trying too much).

I can't make a "positive" statement that what you want to do with changing the disk drive back and forth will cause problems, but my stomach says that there probably would be some problems associated with it.  Hopefully someone will show up soon who knows more about that particular method!:wink:
