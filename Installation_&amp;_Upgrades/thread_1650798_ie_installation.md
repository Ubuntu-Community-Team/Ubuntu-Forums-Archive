### Post by jerenept on 2010-12-22
It's not really possible to install IE in WINE, because it is so integrated into Windows. Are you redirected to an "Browser Incompatible" page?

If so, changing the User Agent that Firefox identifies itself as may work. Try [This Firefox add-on]("https://addons.mozilla.org/en-US/firefox/addon/59/") to do that.
