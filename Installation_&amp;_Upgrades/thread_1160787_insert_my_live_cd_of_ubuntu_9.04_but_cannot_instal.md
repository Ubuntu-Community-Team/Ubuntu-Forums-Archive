### Post by zvacet on 2009-05-16
Did you check [md5sum]("https://help.ubuntu.com/community/HowToMD5SUM") and [Cd integrity?]("https://help.ubuntu.com/community/Installation/CDIntegrityCheck?action=show&redirect=CDIntegrityCheck") If you get any errors download same iso with torrent and point download to the folder where your existing iso is.Torrent will just check for bad files and replace them with good ones.After that run md5sum again and if it is O.K. burn iso to the disc and check disc integrity.If all goes well go for install.
