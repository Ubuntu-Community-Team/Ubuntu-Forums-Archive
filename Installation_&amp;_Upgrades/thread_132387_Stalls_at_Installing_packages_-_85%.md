### Post by richo132 on 2006-02-18
I am trying to install BB 5.10 on a Celeron 800 PC with 160Gb WD HD and Matrox G400max video card.
The install completes to first reboot without problems.
On restart it stalls while Installing Packages  - Preparing for Installation...
Runs through at fairly steady rate to 85% then progress apparently stops.
The hard drive light still flickers in a random pattern but after 2 hours no change shown to progress bar.
I have repeated the install several times and tried using a different HD (120Gb Seagate) and manual partitioning.
I would appreciate any help on how to diagnose and fix the problem.
TIA

### Post by richo132 on 2006-02-18
Problem was caused by SB AWE 32 sound card (non-plug&pray).
Installation now completed ok.
