### Post by towsonu2003 on 2006-01-28
it is much easier than how it looks. start with what wiki tells you **step by step**. Take notes as you go trhough the installation (what commands did you write, what was the output in general, any error messages etc.) I don't think you'll get stuck, but if you do: just ask in here after specifying what steps you did and where you got stuck (include error message if any). 

Basically, you will just be reading the wiki page from start to end once before installing. than, download firefox from mozilla and follow he steps one by one. 

And how it works *in theory* as it is described in details in the wiki: basically, you will be installing a dependent package, backing up your old firefox profile, extract the tar.gz file to a specific location (just like a winzip file), and transfering some settings from old profile to the new one. 

don't let the command line stuff scare you. you might even like working with command line (it makes me feel I know stuff, which is cool \\:D/ )
