### Post by yancek on 2017-07-14
If you have Ubuntu Mate installed then you have the Grub2 bootloader which you can use to boot the standard Ubuntu iso you want to install directly to another partition or free space on any drive.  The link below is to the official Ubuntu documentation and if you scroll about half way down the page they even have example entries you can put in the grub.cfg file of Ubuntu Mate.

[https://help.ubuntu.com/community/Grub2/ISOBoot](https://help.ubuntu.com/community/Grub2/ISOBoot)
