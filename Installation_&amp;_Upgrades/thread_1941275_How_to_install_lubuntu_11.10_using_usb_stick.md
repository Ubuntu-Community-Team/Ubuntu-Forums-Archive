### Post by SteveDee on 2012-03-15
> **Thebuntuway said:**
> ...so then, can someone tell me how to install these OS into usb stick and install it in my lame pc that now using ubuntu 11.10...

If you want to switch from Ubuntu to Lubuntu follow this: [http://psychocats.net/ubuntu/purelxde](http://psychocats.net/ubuntu/purelxde)

To make a USB stick, download the Lubuntu ISO from the download web page. Open Synaptic Package Manager (from System Tools menu) then search and install: use-creator-gtk

Now run Startup Disk Creator from the System Tools menu to make your USB stick.

### Post by C.S.Cameron on 2012-03-15
I prefer using Startup Disk Creator from the Live CD or usb-creator from the iso, to install to USB.
UNetbootin also works.
Set Bios to use the USB as first hard drive.
If computer will not boot USB you should be able to use Plop to boot it.
Boot the USB and follow instructions.
