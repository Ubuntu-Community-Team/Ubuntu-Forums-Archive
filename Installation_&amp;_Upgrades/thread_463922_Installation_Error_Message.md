### Post by Jussi Kukkonen on 2007-06-04
There seems to be a problem with either the CD or the drive used to install... Some things you could try:
* Check that the CD was correctly burned (does the checksum match)
* use the other CD-drive when installing (if this is possible, see BIOS settings)
* burn at a low speed
* use another CD-writer to burn the disc
* use a high quality disc (for some reason liveCDs seem to be picky)
