### Post by oldfred on 2010-09-30
I do not like writing into the system partition from a foreign system except emergency repairs, as that can lead to issues.

I keep a separate NTFS partition for windows data that I might want to share, including firefox & thunderbird profiles. Some windows users recommend a separate data partition (D: drive) just so when you have to reinstall windows you do not have to reinstall all you data.

For your immediate needs just copy the file to a USB drive (FAT32), CD or floppy as those devices can be read from either relatively easily.
