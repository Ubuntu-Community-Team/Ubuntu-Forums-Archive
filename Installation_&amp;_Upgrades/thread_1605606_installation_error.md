### Post by sikander3786 on 2010-10-25
It is always advised to check the MD5SUM of the downloaded image before buring it to disc.

[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

If sums match, burn the image to a disc at the lowest possible speeds. And check the disc for defects if you get errors like that.

In addition to disc, CD-drive, it might also be caused by HDD failure. I can see you mentioned you can install Fedora and Windows without any problems still if problem persists, check the HDD for bad sectors.

Which version are you trying to install? Doesn't seem to be the latest one. If not, grab 10.10 or 10.04 from here.

[http://www.ubuntu.com](http://www.ubuntu.com)

I will put my money on a bad image or defected disc.
