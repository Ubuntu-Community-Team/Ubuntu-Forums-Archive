### Post by sudodus on 2014-01-23
In order to get relevant advice, please specify your computer

- Brand name and model
- CPU
- RAM size
- graphics chip/card

- present version of Ubuntu: 12.04 LTS, 13.10 or some other version?
- and flavour: standard Ubuntu, Kubuntu, Lubuntu, Xubuntu?

I don't understand what you mean by 'I would like to have a friend use but i want to delete my profile'. Please explain!

Finally, it is 'always' possible to go ahead with a clean install. And if the computer boots via USB, good :-)

... but, do you want to ***backup*** any files from the present installed system? In that case, do it ***before*** you install the new system.
