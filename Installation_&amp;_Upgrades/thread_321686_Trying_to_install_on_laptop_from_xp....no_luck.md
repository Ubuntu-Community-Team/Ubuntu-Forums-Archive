### Post by ThrobbingBrain66 on 2006-12-19
First, I would suggest that you boot with the cd, and instead of selecting to install Ubuntu, that you check that the CD burned correctly (3rd or 4th option on the screen, IIRC).  If that turns up in an error, then you must re-burn the CD at a slower rate.

If the md5sum check returns normal, then you may have to download the 'Alternate Install' CD.  This is a simple text based install cd, but tends to work when the LiveCD doesn't.
