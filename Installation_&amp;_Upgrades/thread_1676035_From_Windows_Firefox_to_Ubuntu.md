Hey,

I was wondering if it was possible to move your firefox settings from Windows such as bookmarks etc over to Ubuntu?

Anyone know/have a good tutorial for this?

---

### Post by smo0th on 2011-01-26
Bookmarks->Organize bookmarks->Import and backup->Backup

This will generate a json file for you and you may import it in your ubuntu ff =)
