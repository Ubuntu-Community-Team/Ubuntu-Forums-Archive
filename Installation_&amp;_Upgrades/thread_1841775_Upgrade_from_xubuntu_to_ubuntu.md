### Post by saltmarshlamb on 2011-09-10
Well you'd need to upgrade xubuntu to the latest version and then install ubuntu-desktop and then remove xubuntu-desktop.

You might be better off backing up your data and doing a clean install over the top of the existing xubuntu install.

### Post by uRock on 2011-09-10
> **saltmarshlamb said:**

+1 I recently went though the method of uninstalling the xubuntu-desktop packaging and installing ubuntu-desktop and the process missed quite a few packages, which caused some problems. It'd be much faster and easier to run a clean install of ubuntu 11.04. Backups are a must, even with upgrading, and running 11.04 from the LiveCD/USB on your machine is a good thing to do. Running the LiveCD/USB will help determine if your hardware will be supported well on the newer version.
