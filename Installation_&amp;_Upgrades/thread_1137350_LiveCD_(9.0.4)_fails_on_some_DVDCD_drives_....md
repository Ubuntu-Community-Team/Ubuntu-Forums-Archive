1)  Both CDs boot successfully from both drives;

2)  Check disk and Test memory run successfully from both CDs on both drives;

3)  Try Ubuntu runs successfully from both CDs on the LG drive; I can run Firefox, connect to the Internet, look at files on hard drives, etc.  However,

4)  Try Ubuntu fails to run successfully on the SONY drive. When the freeze occurs, the ready light on the SONY drive is out, and the only way I can eject the CD is to system reset first.

5)  I've checked the SONY website, and as best I can determine, there are no firmware updates for the SONY drive.  The drive firmware is 1.60;

6)  Try Ubuntu 8.10, and 8.0.4, runs successfully on the SONY drive.

This seems like a bug in 9.0.4 to me.  Comments?
