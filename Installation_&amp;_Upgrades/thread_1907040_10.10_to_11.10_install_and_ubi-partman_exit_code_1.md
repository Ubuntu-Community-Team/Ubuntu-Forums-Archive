ubi-partman failed with exit code 10

I'm working my way through the Google results but would sure like some help/advice. I've tried F6 and checking nodmraid at first of install, but it didn't work; I may not have done it right, but it looked pretty simple.

### Post by Longstreet on 2012-01-10
NM. I tried to create a LiveUSB using Startup Disk Creator, as outlined on the Ubuntu Download page. It would run fine, but not install.

Found a site suggesting Unetbootin to create the LiveUSB instead. Used the .iso I'd already downloaded, and it worked like a charm. I don't know why it would make the difference, but it did.

Hope this might be helpful to anyone who might have a similar problem and stumble across this.
