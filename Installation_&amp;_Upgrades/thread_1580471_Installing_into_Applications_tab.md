---

### Post by lechien73 on 2010-09-23
Right click on the Applications menu, and choose **Edit Menus**

You can create new launchers, and new folders from there. Or were you asking how to write code to create a launcher in the Applications menu?

---

### Post by sikander3786 on 2010-09-23
Under System > Preferences > Main Menu select Programming folder and click New Item and add details/path to your software.
