### Post by ddrummond on 2008-03-14
I had this problem when I first installed too. I made it a lot harder than it really was. Click on System then Administration then Software Sources. In software sources make sure, under the tab "Ubuntu Software", all the boxes are checked. Then go to the Restricted Device Manager in System > Administration and make sure Nvidia Graphics is checked. It'll do everything automatically from there.
