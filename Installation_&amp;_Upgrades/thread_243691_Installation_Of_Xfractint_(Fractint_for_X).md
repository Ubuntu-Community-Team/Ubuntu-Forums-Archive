### Post by notrublw on 2006-08-25
Xfractint is the x-windows port of fractint, one of the best fractal generators around (see [http://spanky.triumf.ca/www/fractint/fractint.html](http://spanky.triumf.ca/www/fractint/fractint.html))

This may be blindingly obvious to some, but it took me a little trial and error to get it right.

Use the advanced tab on the Add/Remove applications dialog to find and install Xfractint.

The executable is called xfractint and will be located in /usr/bin.  When setting it up with the Alacarte Menu editor make sure to mark the start in terminal box.

You should now hopefull have a working copy of Xfractint--the fractal will be drawn in one widow and the commands will be issued in the other.  Some of the drawing commands are a little different from the dos/windows version--see
 
[http://spanky.triumf.ca/www/fractint/xfractint_port.html](http://spanky.triumf.ca/www/fractint/xfractint_port.html)

### Post by notrublw on 2006-09-22
I've figured something else out-the zoom box commands seem only to work in the 'minimized' drawing window that you see when you start up xfractint-but you can expand/zoom the window to look at it full or any sized and then go back to the minimized one to play with the zoom box.

By default files are saved in your home directory as *.gif files.  I think the Unisys patent on the lzw compression algorithm is expired so that should be safe for now.  Anyway you can use the GIMP to change .gif to .png.
