### Post by Sef on 2009-04-24
> 1. should i try 9.04 yet?

How is 8.10 working for you?  It is optional.  If everything is set up fine, then it is a harder decision.  Try the live cd and see what problems, if any, you have.  If you have problems, then upon install you will have those problems, if any exist.  

>  2. I have the home in a separate hard drive i know to not reformat the /home,,,,, but do i reformat the / or not??? 	

if you have a separate /home, then do not reformat it.   Just make sure the mount is set to /home.   You only need to reformat the root (/) partition; 8.10 will be erased and 9.04 will be put on in its place.
