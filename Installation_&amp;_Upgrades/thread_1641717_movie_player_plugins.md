### Post by sikander3786 on 2010-12-10
The complete all-in-one package of codecs with flash, open source java and some fonts as a bonus :-)

```
sudo apt-get install ubuntu-restricted-extras
```

For DVDs, you need a little more work to do.

[https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs](https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs)

### Post by tommcd on 2010-12-10
To play DVDs you need **libdvdcss2**. You can get libdvdcss2 from the **medibuntu** repository:
[https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)
After you add the medibuntu repo you can also get the windows codecs.
