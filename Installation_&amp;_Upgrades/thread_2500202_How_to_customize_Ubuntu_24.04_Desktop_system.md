### Post by Dennis N on 2024-08-26
On Ubuntu, with the Gnome desktop, most customization is done with gnome shell extensions. These improve functionality and change appearance. To begin with, you need  to install the gnome shell extension manager in order to find and install them. There are about a zillion web sites with "best" extensions, like:

[https://www.omgubuntu.co.uk/best-gnome-shell-extensions](https://www.omgubuntu.co.uk/best-gnome-shell-extensions)

Everyone has their personal favorites. F.Y.I., I have installed and am using these on Ubuntu 24.04:

All Windows
Apps Menu
Blur My Shell
Custom Accent Colors
Just Perfection
Media Controls
Net Speed Simplified
Places Status Indicator
Quick Setting Tweaker
Removable Drive Menu
User Themes
Workspace Indicator

Besides that, you can change the icon, cursor, gnome shell and application themes. For these changes, we rely on the gnome-tweaks utility.
