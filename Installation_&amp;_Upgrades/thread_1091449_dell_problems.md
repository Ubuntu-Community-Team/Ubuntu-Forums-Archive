### Post by avtolle on 2009-03-09
With 256mb RAM, you're going to need to use the Alternate Install iso burned to CD. The minimum system requirements to use the Live (Desktop) CD includes 384 mb RAM to allow the graphical installer to run.

The Alternate installer is text-based, thus doesn't need the extra overhead to run. Bluntly, I'd recommend Xubuntu (has a lighter DE) to you with 256mb, if you don't want to upgrade the RAM (which is always my first suggestion), as it will give you a bit better performance. Other lightweight window managers/desktops include openbox, LXDE (used on Knoppix), and there are other suggestions that I'm sure will be made.
