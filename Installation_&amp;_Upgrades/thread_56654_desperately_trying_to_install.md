### Post by PeP on 2005-08-13
ordering cds is a good idea, the installer provides you tools to format and set up partitions.
 You might prefer to do this in a more user friendly environment, there are good intuitive tools to do that with windows (like partition magic).

Just a tip, if you try to resize a partition, backup everything you need on it because if something fails you might lose all the contents, this is true both for windows expensive program and the free tools provided by ubuntu.

### Post by varunus on 2005-08-13
Easy CD Creator can burn ISO files to CD, as can Nero, as can many other programs.  An ISO file simply contains files.  Ubuntu will boot into an installer once you do this; the installer is all you'll need to get an Ubuntu system.

Here's a guide on how to partition your hard drive using the Ubuntu installer (if you want to dual-boot).
[http://www3.telus.net/linux4u/ubuntu.html](http://www3.telus.net/linux4u/ubuntu.html)

I think there are some free programs that can burn ISO files; look on download.com.  Make sure you choose to "burn as image".  You can't just drag the ISO file over, or else you'll end up with a coaster.
