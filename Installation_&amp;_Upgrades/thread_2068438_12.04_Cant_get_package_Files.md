That is because it is looking for a file that doesn't exist on the server.

Open a terminal and 

```
sudo rm /var/lib/apt/lists/* -vf
``` will remove all the list files and ```
sudo apt-get update
``` will reload them from the repositories.

```
sudo apt-get upgrade
``` will install any new updates if the commands produce no errors.

Post back if any errors on executing any of the above commands.

Good Luck
