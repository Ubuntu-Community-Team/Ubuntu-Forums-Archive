### Post by PMO6022 on 2005-08-09
You are in luck!  Umbrello is in the universe repository, so you can download and install it with apt-get (or synaptic).  

First, follow the directions here [http://ubuntuguide.org/#extrarepositories]("http://ubuntuguide.org/#extrarepositories") to enable the universe repo if you have not done so already.

Then simply type:```
sudo apt-get install umbrello
```  and there ya go!
