### Post by por100pre1 on 2007-11-18
Any task like formating, resizing partitions, adding a different boot loader, or other any similar tasks can be risky. Try setting up a [dual-boot configuration]("https://help.ubuntu.com/community/WindowsDualBoot"), **backup your Windows user files to removable media or external hard disk drive before proceeding.**
