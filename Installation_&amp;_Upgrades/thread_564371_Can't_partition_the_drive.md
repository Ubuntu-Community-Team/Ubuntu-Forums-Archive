### Post by logos34 on 2007-10-01
It could also be that there are hiberntion or 'unmovable' files at the end of the hard disk.  Try temporarily disabling hibernation and swap/virtual memory, and turn down system restore to the minimum.  Then run defrag a few times.  Try install again.
