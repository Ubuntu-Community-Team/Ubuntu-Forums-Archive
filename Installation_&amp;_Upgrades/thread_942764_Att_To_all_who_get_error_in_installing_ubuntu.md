I was facing errors like SQUASHFS cannot read code block. This is how I solved the issue. Download MD5 comparing tool and copy MD5 for your Ubuntu and compare to see if no error. If they are same proceed to next step; otherwise download another ISO file.

If passed, get your burner burn Image at lowest speed. I discarded for sometime my Roxio (I didn't knew how to set smallest speed) in favor of Infra record (freeware) and set lowest speed.

Then boot up with your CD
