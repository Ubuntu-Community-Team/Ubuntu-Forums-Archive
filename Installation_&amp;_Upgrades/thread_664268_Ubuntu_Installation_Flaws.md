### Post by brucewagner on 2008-01-11
These are some VERY IMPORTANT flaws in the installation process of Ubuntu.

I don't know where, or who, to submit them to.... So I'm posting them here.  I hope someone who can do something about them will read this.

(1)   On my new HP Pavilion computers with ViewSonic monitors, Ubuntu's low-res graphical Install program has a main dialog window that is (A) too tall, and (B) cannot be re-sized.

The result is....  the "Next" button does not appear on the screen at all...!

It was only with a lot of luck (and my own experience) that I figured out that the ONLY WAY I COULD INSTALL UBUNTU....  was to:   First, change the bottom Panel Properties to display on the Right side....  Then, change the top Panel Properties to display on the Left side....  And only then, was I finally able to drag the Install window up just enough to BARELY make visible to top EDGE of the "Next" button.    I couldn't READ the button...  I just had to guess, and assume, that it said "Next", or something like that...

This is a MAJOR flaw.   These are brand new HP computers and monitors.   The Install window needs to be not as tall, and it needs to be re-size-able.

(2)   When you get to the Disk Partitioning section....  It is my opinion that "Use entire disk" should be the default.

(3)   Also, "Resize" should be explained better.  What will it do with the existing (Windows) partition.  Also, which is which --- which partition percentage is Windows, and which is Ubuntu?   This information should be made very explicitly clear on that partitioning screen.

(4)   Finally, upon launching the completed installation for the first time...  there needs to be a simple utility that pops up...  which allows the user to install all of the the necessary third-party proprietary code for all things such as: every video codec, DVD playback code, flashplayer plugins, java applet plugins, etc., etc. (after an appropriate legal warning verifying that they live in a country where this is legal).  (much like the QuickStart utility, see [http://ubuntuforums.org/showthread.php?t=613462](http://ubuntuforums.org/showthread.php?t=613462) )

(5)  How can we make sure that these issues get addressed?
