> I just picked up a new Asus U50a laptop running Windows 7 64bit version. I want to dual boot Ubuntu and Windows 7.

Thanks in advance.
Figure out how much space you want to allocate to Ubuntu. Defrag windows at least once, I suggest twice. Then from windows disk management utility shrink the windows partition to create that amount of free space for Ubuntu. Then boot the Ubuntu live CD/USB and at the partitioner window choose use largest continuous free space to install Ubuntu onto that free space you created. 

**_I would back up all your data prior to doing anything._**
