### Post by grahammechanical on 2022-04-27
I think a little explanation is needed to clear up some confusion.

Startup Disk Creator will take a Ubuntu ISO image and create a bootable DVD disc or bootable USB memory stick that when booted will load a Ubuntu live session that can be used to install Ubuntu to a drive or to try Ubuntu without it being installed. 

> I went to the troubled computer and installed the startup disk created  by startup disk creator and booted up the machine.  It booted up but  wanted to install the system.  I thought that was odd given that I  assumed I had already done that and apparently I was wrong.

You created a startup disc (USB stick) with Ubuntu installed on it. You put the USB stick into another machine and so the Ubuntu session that booted offered a Try Ubuntu and an Install Ubuntu.

P.S. The Ubuntu.com download page now offers the latest Ubuntu LTS release which is Ubuntu 22.04. That is what you downloaded.
