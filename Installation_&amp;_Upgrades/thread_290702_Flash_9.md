### Post by aysiu on 2006-11-01
Go to the plugins directory and delete it?

Presumably you would know where the plugins directory is, since that's where you placed Flash 9, according to your first post.

Did you place it in your local plugins folder or the universal one?

**Local**:
/home/*username*/.mozilla/firefox/*gibberish*/plugins

**Universal**:
/usr/lib/firefox/plugins (if you're using Ubuntu's Firefox)

or

/opt/firefox/plugins (if you installed Mozilla's Firefox)
