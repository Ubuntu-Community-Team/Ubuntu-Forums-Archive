### Post by grahammechanical on 2015-06-15
Open System Settings and then Software Sources (now called Software & Updates) and open the Updates tab and make sure that the panel labelled "Notify me of a new Ubuntu version" is set to "For long term support version."

The next time you run Software Updater or Update Manager (names change over time) you should get a button to press that will start the upgrade process.
