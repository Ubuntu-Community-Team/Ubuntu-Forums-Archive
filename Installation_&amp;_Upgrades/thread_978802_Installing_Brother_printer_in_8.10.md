### Post by brucewestfall on 2008-11-11
After installing 8.10, my Brother MFC-665CW multifunction printer did not work.  I removed and reinstalled it with no success. 
(it is a wonderful machine with SUPER cheap cartridges from databaazar.com that has worked great using 8.04 and 7.whatever...)

I just got it working by removing the printer again and then carefully following the steps on the Brother website

[http://solutions.brother.com/linux/en_us/instruction_prn3.html](http://solutions.brother.com/linux/en_us/instruction_prn3.html)

Seems to be working fine now.

Do not install it by adding a printer under
(Main Menu)
System/Administration/Printing

The wrong driver seems to be installed by using that method.
