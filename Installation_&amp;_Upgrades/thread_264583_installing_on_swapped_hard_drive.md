### Post by dickohead on 2006-09-25
Use the first machine to install ubuntu to the second machines hard drive IF:
There are no MAJOR hardware differences ie: 
Both have NVidia cards or both have ATI or both have Intel Integrated etc.
Both are using AMD processors or both are using Intel processors.
Both are ATX or both are Mini-ATX, Micro-ATX etc (this one may not matter so much)
Both have one standard CD drive, or DVD drive.

If one has 2 cd drives and 2 hard drives, but the other has 1 hard drive and 1 cd drive - disable the hard drives in the first computer and one of the cd drives, then install, that way when you move the drive over it won't have created devices that don't exist.

You could always take the CD-ROM out of your primary machine and let your second machine use it to install then remove it?

Either way *should* work, but the second will be better.
