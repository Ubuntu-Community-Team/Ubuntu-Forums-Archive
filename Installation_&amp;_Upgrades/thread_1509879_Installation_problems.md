### Post by darkod on 2010-06-14
F8 menu? You hit F8 while booting windows for advanced options like starting in safe mode, etc. You shouldn't need to hit F8 to get the windows bootloader offering choice for windows and wubi.

On another note, it is far better to install ubuntu as full dual boot OS, on its own partition and its own filesystem, instead of installing wubi inside windows.
