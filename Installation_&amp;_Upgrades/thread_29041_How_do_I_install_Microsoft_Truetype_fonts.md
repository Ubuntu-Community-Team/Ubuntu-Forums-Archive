### Post by Bob D. on 2005-04-22
[http://ubuntuguide.org/#extrafonts]("http://ubuntuguide.org/#extrafonts")

You want the 'msttcorefonts' package. You can also use Synaptic rather than the command line.

Please take a look at the entire guide. It may help you before asking questions that have been asked & answered.

Of course, the forum search tool is a big help as well: [http://ubuntuforums.org/showthread.php?t=1261&highlight=ms+ttf]("http://ubuntuforums.org/showthread.php?t=1261&highlight=ms+ttf")

If you want to install fonts you already have you can create a .fonts (note the dot) folder in /home/<your home>. In Nautilus or the File Browser, enable 'Show Hidden Files' and you can drag and drop your fonts into that folder.
