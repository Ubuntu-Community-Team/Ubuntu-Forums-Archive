### Post by confused57 on 2008-03-01
You could also try a configfile entry to boot Ubuntu:
```
title  Ubuntu Gutsy
configfile (hd0,3)/boot/grub/menu.lst
```

[http://users.bigpond.net.au/hermanzone/p15.htm#1._Configfile](http://users.bigpond.net.au/hermanzone/p15.htm#1._Configfile)

or as Pumalite suggested, reinstall Ubuntu's grub to the mbr & add a configfile entry to boot Fedora...this may be the best method, e.g.:
```
title  Fedora
configfile (hd0,6)/boot/grub/grub.conf
```
