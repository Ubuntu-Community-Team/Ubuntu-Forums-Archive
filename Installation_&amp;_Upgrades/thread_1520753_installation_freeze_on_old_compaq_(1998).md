### Post by alterpinguin on 2010-06-30
you cannot run the Live-Version without enough memory.
The minimum for older ubuntu versions was 256MB main-memory
and newer version may have need the same minimum or even more.
To do an install for computers with less than this memory (128MB is enough to run the server version or a minimalistic x11-version) you need to use the server-install-iso(CD-version). 

Pls. think about the usage of a small linux-version, there are a lot (some even with newer linux-kernels 2.6x and some with older, smaller ones like 2.4x). There are some like puppy-linux or damn small linux, which will boot and run with computers less than 256MB.

And last, remeber there are old features deleted in the new linux-versions. For old mainboards/chipset/cpu you may need those old linux-version with the tweaks to run them.
