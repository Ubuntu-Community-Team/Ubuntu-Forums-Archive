### Post by baeksu on 2008-10-28
That's quite constricted specs you have there.

32MB ram will probably not be enough to run any sort of graphical interface. You will use at least 15-20MB just getting the system running, so I doubt that running X is going to be a very pleasing experience.

Nowadays, most routers or nas boxes sold come with similar specs (see for instance the [NSLU2]("http://en.wikipedia.org/wiki/NSLU2"), so if you "need" a box for a role on the network, you might set it up as a router/nas/mail server.

As for which distro, you might want to explore the list at [penguinppc.org](http://penguinppc.org/about/distributions.php).
