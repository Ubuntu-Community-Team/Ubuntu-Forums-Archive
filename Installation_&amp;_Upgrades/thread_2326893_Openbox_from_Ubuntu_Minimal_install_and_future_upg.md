### Post by fix-the-fdev-drive on 2016-06-05
Hi everyone. In my quest to find a low-memory-use operating system, I recently used the 32bit Ubuntu 16.04 minimal image and added software manually...choosing Openbox and minimal additional software after the initial installation. This has given me what I've searched for. I have everything working wonderfully, and the computer's RAM usage at idle is only +/- 70 MBs. :)  

I know that on official Ubuntu releases and derivatives, one can be notified when LTS releases are ready via the Update Notifier. Since I'm trying to keep memory usage down, I'm doing updates via the terminal only. So, my question is... 

When a new LTS version is released, will there by any type of notification displayed in the terminal regarding a LTS upgrade while running updates, or will my only notification be the official announcement from Ubuntu; thereby, I will know to upgrade because I've "heard or read" that the LTS has been released?

Thank you!

### Post by grahammechanical on 2016-06-05
We will not get another LTS release until end of April 2018. And who knows what Ubuntu will have developed into by that time. We only get notifications of new releases if we are using Ubuntu Software Updater/Update manager. But you do not have Ubuntu desktop installed.

I do not think that Apt is able to provide notifications of new Ubuntu releases. You would have to upgrade using a terminal command. Just like users of server editions do.

Regards.

### Post by howefield on 2016-06-06
> **fix-the-fdev-drive said:**
> When a new LTS version is released, will there by any type of notification displayed in the terminal regarding a LTS upgrade while running updates, or will my only notification be the official announcement from Ubuntu; thereby, I will know to upgrade because I've "heard or read" that the LTS has been released

Don't believe you will get notification via the terminal, although you will be able to check for the availability of an upgrade via the terminal.

You can pretty much set your clock to April every second year. Or possibly more sensibly, July every second year when the first point release of the LTS is done. 

Once the release schedule is known for an upcoming release you'll find it at, eg,  [https://wiki.ubuntu.com/XenialXerus/ReleaseSchedule](https://wiki.ubuntu.com/XenialXerus/ReleaseSchedule), that might be worth bookmarking, "XenialXerus" can be substituted with YakketyYak for the current development release and so on for the next releases.
