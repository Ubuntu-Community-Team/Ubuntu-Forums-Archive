### Post by Dutch70 on 2011-05-06
Hi & welcome to the Ubuntu Forums Haggzstar.

It is highly recommended that you get a usb stick, at least 1GB or a cd even if you have to get it from a friend. If you have problems, you may need this to fix it.

If you go with Wubi aka windows installer, you just download the .exe and install it like any other windows program. You can then uninstall it the same way or migrate it to it's own partition for a true dual boot. I recommend 10.04.2, it's the most stable & it's supported longer than any other current release. 
Pay attention to the warning about Grub.
