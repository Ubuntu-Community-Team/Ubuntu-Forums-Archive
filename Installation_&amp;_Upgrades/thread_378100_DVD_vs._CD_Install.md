### Post by bruenig on 2007-03-06
The install is the same, the dvd has more programs on the dvd allowing you to use the dvd as a software repository if you don't have an internet connection or if you have a really slow internet connection. If you have a fast internet connection, I would just go with the cd because software can be installed from the online repos which will be more up to date and easier.
